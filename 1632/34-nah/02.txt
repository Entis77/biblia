Ciągnie ſkáżyćiel przećiw tobie / o Niniwe! opátrz miejſcá obronne / wyglądáj ná drogę / zmocnij biodrá / á bárdzo umocnij śiłę twoję ;
Bo Pán odwróćił pychę Jákóbową / jáko pychę Izráelową / przeto / że ich wyniƺcżyli ſkáżyćiele / á látorośle ich popſowáli.
Tárcżá mocárzy jego cżerwoná / rycerſtwo jego ƺárłátem odźiáne / wozy jego jáko pochodnie gorejące iſkrzyć śię będą w dźień potykániá jego / á jodły ſtráƺne trząść śię będą.
Wozy po ulicách grzmieć y po rynku ſkrzypieć będą ; ná wejrzeniu będą jáko pochodnie / á jáko błyſkáwice biegáć będą.
Szykuje mocárzy ſwoich / ále jednák upádną w ƺyku ſwym ; poſpieƺy śię do murów / jákoby tám zgotowáná byłá obroná.
Brámy śię przy rzekách otworzą / á kośćiół śię rozpłynie ;
A Chuſáb pojmáná będąc záwiedźioná będźie / á ſłużebnice jey prowádźić ją będą / hucżąc jáko gołębicá á bijąc śię w pierśi ſwe.
A ácżkolwiek Niniwe było jáko ſádzáwká wód od pocżątku ſwego / wƺákże już ſámi ućiekáją ; á choć kto rzecże : Stójćie / ſtójćie! wƺákże śię nikt nie obejrzy.
Rozchwyććież ſrebro / rozchwyććież złoto / y niezmierne bogáctwá / y cokolwiek nájkoƺtowniejƺego ze wƺyſtkich klejnotów drogich.
Wyplundrowáne y wybráne będźie / owƺem / do ƺcżętu ſpuſtoƺone będźie ; ſerce śię rozpłynie / koláno o koláno tłuc śię będźie / y boleść ná wƺyſtkich biodrách będźie / á oblicżá wƺyſtkich pocżernieją.
Gdźież jeſt jáſkiniá lwów / y páſtwiſko lwiąt? gdźie chodźił lew / lew / mówię / y lwicá / á nie było nikogo / ktoby je przeſtráƺył.
Lew / który doſtátkiem chwytáł lwiętom ſwoim y záduƺáł dla lwic ſwoich / który nápełniáł łupem jáſkinie ſwoje / á obłowem łożyſká ſwoje.
Otom Ja przećiwko tobie / mówi Pán záſtępów / á popálę ná proch wozy twoje / á miecż pożre lwiętá twoje ; y wykorzenię z źiemi łup twój / á nie będźie więcey ſłyƺány głoſ poſłów twoich.
