Biádá miáſtu krwáwemu ; wƺyſtko kłámſtwá y łupieſtwá pełne jeſt / á źdźierſtwo z niego nie wychodźi.
Tám będźie ſłyƺáne trzáſkánie bicżów / y grzmot kół / y tupánie koni / y ſkákánie wozów.
Jezdny dobędźie błyƺcżącego miecżá ſwego / y lśniącego oƺcżepá ; tám będźie wiele pobitych y wielkie gromády trupów / ták / że nie będźie licżby trupów / á przez ćiáłá pobitych wálić śię będą ;
Dla wielkośći wƺetecżeńſtw nierządnicy rozkoƺney / która śię báwiłá cżárámi / á záprzedáwáłá narody wƺetecżeńſtwámi ſwemi / y ludźi guſłámi ſwemi.
Otom Ja przećiwko tobie / mówi Pán zaſtępów / y odkryję podołek twój ná twárz twoję / á okáżę národom ſproſność twoję / á króleſtwom háńbę twoję :
Y wyrzucę ná ćię obrzydliwośći y zelżę ćię / y wyſtáwię ćię zá dźiw /
Ták / że ktokolwiek ćię ujrzy / oddáli śię od ćiebie y rzecże : Zburzone jeſt Niniwe / któżby śię go użálił? Gdźieżbym ƺukał tych / którzyby ćię ćieƺyli?
Izáliś ty lepƺá / niż ludne miáſto No / które leżáło miedzy rzekámi / wodámi otocżone będąc ; którego wáłem było morze / y od morzá mur jego?
Murzyńſká źiemiá y Egipt były śiłą jego / y narody niezlicżone / Putcżycy y Lubimcżycy / były mu ná pomocy ;
Wƺákże y to w záprowádzenie y w pojmánie przyƺło ; málucżkie jego ná rogách wƺyſtkich ulic roztrącáno á o nájſłáwniejƺych jego loſy miotáno / náwet wƺyſcy przedniejśi jego okowáni ſą w pętá.
Tákżeć y ty opojoná będźieƺ / y ſkryć śię muśiƺ / y ty ƺukáć będźieƺ pomocy przećiwko nieprzyjáćielowi.
Wƺyſtkie twierdze twoje ſą jáko figowe drzewo z owocem ránnym / którym gdy kto zátrząśnie / záráz wpádáją w uſtá tego / co je jeść chce.
Oto lud twój ſą niewiáſtámi w pośród ćiebie ; nieprzyjáćiołom twoim ƺeroko otworzone będą bramy źiemi twojey / á ogień pożre záwory twoje.
Nácżerpáj ſobie wody do oblężeniá / zmácniáj twierdze twoje / wleź w błoto / y depcż glinę / opráwiwƺy cegielnicę ;
Tám ćię pożre ogień / wytnie ćię miecż / pożre ćię jáko chrząƺcże ; zgromádź śię jáko chrząƺcże / zbierz śię jáko ƺáráńcżá.
Rozmnożyłeś kupców twojich nád gwiázdy niebieſkie ; ále jáko chrząƺcże przypádáją y odlátują / ták y ći.
Pánowie twoi ſą jáko ƺáráńcżá / á hetmáni twoi jáko wielcy chrząƺcże / którzy śię kłádą obozámi ná płotách cżáſu źimná / jedno ſłońce weƺło / álić odlátują / y nie znáć miejſcá ich / gdźie byli.
Zdrzemią śię páſterze twoi / o królu Aſſyryjſki! leżeć będą ſłáwni twoi ; obfitość ludu twego będźie po górách / ále nie będźie / ktoby go zgromádźił.
Niemáƺ lekárſtwá ná ránę twoję / nieulecżoná jeſt plágá twojá ; wƺyſcy / którzy powieść o tobie uſłyƺą / kláſkáć będą rękomá nád tobą ; bo ná kogoż uſtáwicżnie nie przychodźiło okrućieńſtwo twoje?
