Biádá miáſtu krwáwemu ; wƺyſtko kłámſtwá y łupieſtwá pełne jeſt / á ździerſtwo z niego nie wychodzi.
Tám będzie ſłyƺáne trzáſkánie bicżów / y grzmot kół / y tupánie koni / y ſkákánie wozów.
Jezdny dobędzie błyƺcżącego miecżá ſwego / y lśniącego oƺcżepá ; tám będzie wiele pobitych y wielkie gromády trupów / ták / że nie będzie licżby trupów / á przez ćiáłá pobitych wálić śię będą ;
Dlá wielkośći wƺetecżeńſtw nierządnicy rozkoƺnej / którá śię báwiłá cżárámi / á záprzedáwáłá národy wƺetecżeńſtwámi ſwemi / y ludzi guſłámi ſwemi.
Otom Já przećiwko tobie / mówi Pán záſtępów / y odkryję podołek twój ná twárz twoję / á okáżę národom ſproſność twoję / á króleſtwom háńbę twoję :
Y wyrzucę ná ćię obrzydliwośći y zelżę ćię / y wyſtáwię ćię zá dziw /
Ták / że ktokolwiek ćię ujrzy / oddáli śię od ćiebie y rzecże : Zburzone jeſt Niniwe / któżby śię go użálił? Gdzieżbym ƺukáł tych / którzyby ćię ćieƺyli?
Izáliś ty lepƺá / niż ludne miáſto No / które leżáło między rzekámi / wodámi otocżone będąc ; którego wáłem było morze / y od morzá mur jego?
Murzyńſká ziemiá y Egipt były śiłą jego / y národy niezlicżone / Putcżycy y Lubimcżycy / były mu ná pomocy ;
Wƺákże y to w záprowádzenie y w pojmánie przyƺło ; málucżkie jego ná rogách wƺyſtkich ulic roztrącáno á o nájſłáwniejƺych jego loſy miotáno / náwet wƺyſcy przedniejśi jego okowáni ſą w pętá.
Tákżeć y ty opojoná będzieƺ / y ſkryć śię muśiƺ / y ty ƺukáć będzieƺ pomocy przećiwko nieprzyjáćielowi.
Wƺyſtkie twierdze twoje ſą jáko figowe drzewo z owocem ránnym / którym gdy kto zátrząśnie / záráz wpádáją w uſtá tego / co je jeść chce.
Oto lud twój ſą niewiáſtámi w pośród ćiebie ; nieprzyjáćiołom twoim ƺeroko otworzone będą brámy ziemi twojej / á ogień pożre záwory twoje.
Nácżerpáj ſobie wody do oblężeniá / zmácniáj twierdze twoje / wleź w błoto / y depcż glinę / opráwiwƺy cegielnicę ;
Tám ćię pożre ogień / wytnie ćię miecż / pożre ćię jáko chrząƺcże ; zgromádź śię jáko chrząƺcże / zbierz śię jáko ƺáráńcżá.
Rozmnożyłeś kupców twoich nád gwiázdy niebieſkie ; ále jáko chrząƺcże przypádáją y odlátują / ták y ći.
Pánowie twoi ſą jáko ƺáráńcżá / á hetmáni twoi jáko wielcy chrząƺcże / którzy śię kłádą obozámi ná płotách cżáſu zimná / jedno ſłońce weƺło / álić odlátują / y nie znáć miejſcá ich / gdzie byli.
Zdrzemią śię páſterze twoi / o królu Aſſyryjſki! leżeć będą ſłáwni twoi ; obfitość ludu twego będzie po górách / ále nie będzie / ktoby go zgromádził.
Niemáƺ lekárſtwá ná ránę twoję / nieulecżoná jeſt plágá twojá ; wƺyſcy / którzy powieść o tobie uſłyƺą / kláſkáć będą rękomá nád tobą ; bo ná kogoż uſtáwicżnie nie przychodziło okrućieńſtwo twoje?
