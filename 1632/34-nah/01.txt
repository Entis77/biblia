Brzemię Niniwy. Kśięgi widzeniá Náhumá Elkoſejcżyká.
Pán jeſt Bóg zápálcżywy y mśćiwy ; mśćiwy jeſt Pán á gniewliwy ; Pán / który śię mśći nád przećiwnikámi ſwymi / y chowá gniew przećiwko nieprzyjáćiołom ſwoim.
Pán nierychły do gniewu á wielkiej mocy / który winnego nie cżyni niewinnym ; w wichrze y w burzy jeſt drogá Páńſká / á obłok jeſt prochem nóg jego /
Który gromi morze y wyſuƺá je / y wƺyſtkie rzeki wyſuƺá ; przed nim Báſán y Kármel mdleje / á kwiát Libáńſki więdnie ;
Góry drżą przed nim / á págórki śię rozpływáją ; źiemiá gore od oblicżá jego / y okrąg źiemi y wƺyſcy mieƺkájący ná nim.
Przed rozgniewániem jego któż śię oſtoi? á kto śię ſtáwi przećiwko popędliwośći gniewu jego? Gniew jego śię wylewá jáko ogień / á ſkáły śię pádáją przed nim.
Dobry jeſt Pán / y pośilá w dźień ućiśnieniá / á zná tych / którzy ufáją w nim ;
Przetoż powodźią prędką koniec ucżyni miejſcu jego / á nieprzyjáćiół Bożych ćiemnośći gonić będą.
Cóż zámyśláćie przećiwko Pánu? onći koniec ucżyni / utrápienie drugi ráz nie powſtánie.
Bo ták jáko ćiernie ſplećieni á opojeni ſą jáko winem ; przetoż jáko śćierniſko ſuche do ƺcżętu pożárći będą.
Z ćiebie wyƺedł ten / który złe myśli przećiwko Pánu / rádcá złośliwy.
Ták mówi Pán : Być śię byli ſpokojnie záchowáli / zoſtáłoby ich było wiele / álboby tylko byli przerzedzeni / y byłoby ich to minęło / á nie trápiłbym ich był więcej / ták jákom ich trápił.
Ale teráz ſkruƺę járzmo jego / áby ná tobie nie leżáło / á związki twoje potárgám.
Bo przećiwko tobie / o Niniwcżyku! Pán przykázáł / że nie będźie więcej imieniá náśieniá twego ; z domu bogá twego wygłádzę ryte y láne obrázy / á gdy zniewáżony będźieƺ / grób ći zgotuję.
Oto ná tych górách nogi wdźięcżne poſelſtwo opowiádájącego / zwiáſtującego pokój. Obchodźże / o Judo! urocżyſte świętá twoje / oddáwáj śluby twoje ; bo śię więcej złośnik mimo ćię chodźić nie pokuśi / do ƺcżętu jeſt wygłádzony.
