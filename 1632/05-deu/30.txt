Gdy tedy przyjdą ná ćię wƺyſtkie te rzecży / błogoſłáwieńſtwo / y przeklęſtwo / którem przełożył przed ocży twoje / á wſpomniƺ ſobie w ſercu twem miedzy wƺyſtkimi narody / do których ćię wypędźi Pán / Bóg twój ;
Y náwróćiƺ śię do Páná / Bogá twego / á będźieƺ poſłuƺny głoſowi jego we wƺyſtkim / co ja rozkázuję tobie dźiś / ty / y ſynowie twoi / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey :
Tedy ćię wywiedźie Pán / Bóg twój / z więźieniá twego / y zmiłuje śię nád tobą / á przywróći y zgromádźi ćię ze wƺyſtkich narodów / do których ćię rozproƺył Pán / Bóg twój.
By też kto z twojich wygnány był y ná kráj świátá / z támtąd zgromádźi ćię Pán / Bóg twój / y z támtąd zbierze ćię ;
A przyprowádźi ćię Pán / Bóg twój / do źiemi / którą odźiedźicżyli ojcowie twoi / y ośiędźieƺ ją / y będźieć dobrze cżynił / y rozmnoży ćię nád przodki twoje.
Y obrzeże Pán / Bóg twój / ſerce twoje / y ſerce náśieniá twego / ábyś miłował Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey / ábyś mógł żyć.
Y obróći Pán / Bóg twój / wƺyſtkie te przeklęſtwá ná nieprzyjáćioły twoje / y ná te / którzy ćię nienáwidźili / y którzy ćię prześládowáli.
A ty náwróćiwƺy śię poſłuƺny będźieƺ głoſowi Páńſkimu / y cżynić będźieƺ wƺyſtkie przykázánia jego / które ja tobie dźiś rozkázuję.
Y poƺcżęśćić Pán / Bóg twój / w káżdej ſpráwie rąk twojich / w owocu żywotá twego / y w owocu bydłá twego / y w owocu źiemi twej / ku dobremu ; ábowiem obróći śię Pán / áby śię ćieƺył z ćiebie / cżyniąc ći dobrze / jáko śię ćieƺył z ojców twojich /
Będźieƺli poſłuƺnym głoſowi Páná / Bogá twego / przeſtrzegájąc przykázáń jego / y uſtáw jego / nápiſánych w kśięgách zakonu tego / náwrácájąc śię do Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey.
Abowiem przykazánie / które ja przykázuję tobie dźiś / nie jeſt przed tobą zákryte / áni dáleko odległe ;
Nie jeſt w niebie / żebyś rzekł : Któż nam wſtąpi do niebá / á wźiąwƺy je / przynieśie do nas / y opowie je nam / ábyſmy je pełnili?
Ani też jeſt zá morzem / żebyś rzekł : Któż nam popłynie zá morze / á wźiąwƺy je przynieśie / y opowie je nam / ábyſmy je pełnili.
Ale bárzo bliſko ćiebie jeſt to ſłowo / w uśćiech twojich y w ſercu twojim / ábyś je cżynił.
Otom położył przed ocży twoje dźiś żywot y dobre / tákże śmierć y złe.
Abowiem ja rozkázuję tobie dźiś / ábyś miłował Páná / Bogá twego / y chodźił drogámi jego / y przeſtrzegał przykázáń jego / y uſtáw jego y ſądów jego / ábyś żył / y rozmnożył śię / y żebyć błogoſłáwił Pán / Bóg twój / w źiemi / do którey idźieƺ / ábyś ją pośiádł.
Jeſli śię záś odwróći ſerce twoje / á nie uſłucháƺ / ále dáwƺy śię zwieść / będźieƺ śię kłániał bogom obcym / y będźieƺ im ſłużył :
Opowiádám wam dźiś / że pewnie poginiećie / áni przedłużyćie dni ná źiemi / do którey śię przepráwiáƺ przez Jordán / ábyśćie tám wƺedƺy / pośiedli ją.
Oświádcżám śię dźiś przećiwko wam / niebem y źiemią / żem żywot y śmierć przedłożył przed ocżymá twemi / błogoſłáwieńſtwo / y przeklęſtwo ; przetoż obierz żywot / ábyś żył / ty y naśienie twoje /
A miłował Páná / Bogá twego / y poſłuƺny był głoſowi jego / y przy nim trwał ; ábowiem on jeſt żywotem twoim / y przedłużeniem dni twojich / żebyś mieƺkał ná źiemi / o którą przyśiągł Pán ojcom twoim / Abráhámowi / Izáákowi / y Jákóbowi / iż im ją dáć miał.
