Gdy tedy przyjdą ná ćię wƺyſtkie te rzecży / błogoſłáwieńſtwo / y przeklęſtwo / którem przełożył przed ocży twoje / á wſpomniƺ ſobie w ſercu twem między wƺyſtkimi národy / do których ćię wypędzi Pán / Bóg twój ;
Y náwróćiƺ śię do Páná / Bogá twego / á będzieƺ poſłuƺny głoſowi jego we wƺyſtkiem / co já rozkázuję tobie dziś / ty / y ſynowie twoi / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej :
Tedy ćię wywiedzie Pán / Bóg twój / z więzieniá twego / y zmiłuje śię nád tobą / á przywróći y zgromádzi ćię ze wƺyſtkich národów / do których ćię rozproƺył Pán / Bóg twój.
By też kto z twoich wygnány był y ná kráj świátá / ſtámtąd zgromádzi ćię Pán / Bóg twój / y ſtámtąd zbierze ćię ;
A przyprowádzi ćię Pán / Bóg twój / do ziemi / którą odziedzicżyli ojcowie twoi / y ośiędzieƺ ją / y będzieć dobrze cżynił / y rozmnoży ćię nád przodki twoje.
Y obrzeże Pán / Bóg twój / ſerce twoje / y ſerce náśieniá twego / ábyś miłowáł Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej / ábyś mógł żyć.
Y obróći Pán / Bóg twój / wƺyſtkie te przeklęſtwá ná nieprzyjáćioły twoje / y ná te / którzy ćię nienáwidzili / y którzy ćię prześládowáli.
A ty náwróćiwƺy śię poſłuƺny będzieƺ głoſowi Páńſkimu / y cżynić będzieƺ wƺyſtkie przykázániá jego / które já tobie dziś rozkázuję.
Y poƺcżęśćić Pán / Bóg twój / w káżdej ſpráwie rąk twoich / w owocu żywotá twego / y w owocu bydłá twego / y w owocu ziemi twej / ku dobremu ; ábowiem obróći śię Pán / áby śię ćieƺył z ćiebie / cżyniąc ći dobrze / jáko śię ćieƺył z ojców twoich /
Będzieƺli poſłuƺnym głoſowi Páná / Bogá twego / przeſtrzegájąc przykázáń jego / y uſtáw jego / nápiſánych w kśięgách zákonu tego / náwrácájąc śię do Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej.
Abowiem przykázánie / które já przykázuję tobie dziś / nie jeſt przed tobą zákryte / áni dáleko odległe ;
Nie jeſt w niebie / żebyś rzekł : Któż nám wſtąpi do niebá / á wziąwƺy je / przynieśie do náſ / y opowie je nám / ábyśmy je pełnili?
Ani też jeſt zá morzem / żebyś rzekł : Któż nám popłynie zá morze / á wziąwƺy je przynieśie / y opowie je nám / ábyśmy je pełnili.
Ale bárdzo bliſko ćiebie jeſt to ſłowo / w uśćiech twoich y w ſercu twojim / ábyś je cżynił.
Otom położył przed ocży twoje dziś żywot y dobre / tákże śmierć y złe.
Abowiem já rozkázuję tobie dziś / ábyś miłowáł Páná / Bogá twego / y chodził drogámi jego / y przeſtrzegáł przykázáń jego / y uſtáw jego y ſądów jego / ábyś żył / y rozmnożył śię / y żebyć błogoſłáwił Pán / Bóg twój / w ziemi / do której idzieƺ / ábyś ją pośiádł.
Jeźli śię záś odwróći ſerce twoje / á nie uſłucháƺ / ále dáwƺy śię zwieść / będzieƺ śię kłániáł bogom obcym / y będzieƺ im ſłużył :
Opowiádám wám dziś / że pewnie poginiećie / áni przedłużyćie dni ná ziemi / do której śię przepráwiáƺ przez Jordán / ábyśćie tám wƺedłƺy / pośiedli ją.
Oświádcżám śię dziś przećiwko wám / niebem y ziemią / żem żywot y śmierć przedłożył przed ocżymá twemi / błogoſłáwieńſtwo / y przeklęſtwo ; przetoż obierz żywot / ábyś żył / ty y náśienie twoje /
A miłowáł Páná / Bogá twego / y poſłuƺny był głoſowi jego / y przy nim trwáł ; ábowiem on jeſt żywotem twoim / y przedłużeniem dni twoich / żebyś mieƺkáł ná ziemi / o którą przyśiągł Pán ojcom twoim / Abráhámowi / Izáákowi / y Jákóbowi / iż im ją dáć miáł.
