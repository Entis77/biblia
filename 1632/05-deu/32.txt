Słuchájćie niebioſá / á mówić będę ; niech ſłuchá y ziemiá wymowy uſt moich.
Kropić będzie jáko deƺcż náuká mojá / popłynie jáko roſá wymowá mojá / jáko drobny deƺcż ná ziołá / á jáko krople ná tráwę.
Abowiem imieniá Páńſkiego będę wzywáł ; dájćież wielmożność Bogu náƺemu /
Skále / której ſpráwy ſą doſkonáłe ; bo wƺyſtkie drogi jego ſą ſpráwiedliwe. Boć jeſt Bóg práwdziwy / á niemáƺ w nim niepráwośći / ſpráwiedliwy y práwy jeſt.
Pokázili mu śię / jákoby nie byli ſynámi jego / ſproſnośćią ſwoją / náród zły / á przewrotny.
Y tákże to oddáwáćie Pánu / ludu głupi y ƺálony? izáli nie on jeſt ojcem twoim / który ćię ſobie nábył? on ćię ucżynił / y ſtworzył ćię.
Wſpomnij ná dni dáwne / uwáżájćie látá káżdego wieku ; ſpytáj ojcá twego / á oznájmi tobie ; ſtárƺych twoich / á powiedząć.
Gdy rozdzielił dziedzictwá Nájwyżƺy národom / á rozſádził ſyny Adámowe / położył gránice národom według licżby ſynów Izráelſkich ;
Abowiem dziáłem Páńſkim jeſt lud jego / Jákób ƺnurem dziedzictwá jego.
Ználázł go w ziemi puſtej / y ná puƺcży wielkiej y ſtráƺnej ; obwodził go / ucżył go / ſtrzegł go / jáko źrenicy oká ſwego.
Jáko orzeł wybáwiá orlętá ſwoje / nád orlętámi ſwojimi látájąc / rozƺerzá ſkrzydłá ſwoje / bierze je y nośi je ná ſkrzydłách ſwoich :
Ták Pán ſám prowádził go / á żáden obcy bóg nie był z nim.
Wynióſł go ná wyſokie miejſcá ziemi / áby używáł urodzájów polnych / y ucżynił / áby ſſáł miód z ſkáły y oliwę z opoki twárdej ;
Aby jádł máſło od krów / y mleko od owiec / z tłuſtośćią jágniąt / y bárány odchowáne w Báſán / y kozły z tłuſtośćią ziárn pƺenicżnych / á pił cżerwone wino wyborne.
Y roztył Izráel / y wierzgáł ( otyłeś / ſtłuśćiáłeś / zgrubiáłeś / y opuśćiłeś Bogá / który ćię ucżynił / ) y lekce powáżył Bogá / opokę zbáwieniá ſwego.
Ku zápálcżywośći pobudzili go przez cudze bogi / y przez obrzydliwośći rozdráźnili go.
Ofiárowáli dyjábłom / nie Bogu / bogom / których nie ználi / nowym / którzy z bliſká przyƺli / których śię nie báli ojcowie wáśi.
Opoki / którá ćię ſpłodziłá / zápomniáłeś / zápomniáłeś Bogá / Stworzyćielá twego.
Co widząc Pán poruƺył śię gniewem ná ſyny ſwoje / y ná córki ſwoje.
Y rzekł : Skryję oblicże moje przed nimi / obácżę / jáki będzie koniec ich ; ábowiem ſą národem przewrotnym / ſynámi / w których wiáry niemáƺ.
Oni mię wzruƺyli ku zápálcżywośći przez to / co nie jeſt Bogiem ; pobudzili mię próżnośćiámi ſwemi. Já też do zázdrośći pobudzę je przez te / którzy nie ſą ludem / przez náród głupi rozdráźnię je.
Abowiem ogień rozpálił śię w popędliwośći mojej / y będzie gorzáł áż do nájgłębƺego piekłá / y pożre ziemię / y urodzáj jej / y wypáli grunty gór.
Zgromádzę ná nie złe rzecży / ſtrzáły moje ná nie wyſtrzelám.
Zniƺcżeją od głodu / y będą ſtráwieni gorącżką / y śmierćią gorzką ; zęby też beſtyi poślę ná nie / z jádem gádzin ziemſkich.
Zewnątrz ośieroći je miecż / á w pokojách będzie ſtrách / ták ná młodzieńcá / jáko ná pánnę / ná ſſącego pierśi / y ná mężá ſędziwego.
Rzekłem : Rozproƺę je po kątách / y wygubię między ludźmi pámiątkę ich.
Gdybym śię ná pychę nieprzyjáćiół nie oglądáł / by śię ſnáć nie podnieśli nieprzyjáćiele ich / á nie rzekli : Ręká náƺá wynioſłá / á nie Pán / ſpráwiłá to wƺyſtko.
Abowiem ten náród ginie przez rády ſwe / y niemá rozumu.
O by mądrymi byli! wyrozumieliby to / y oglądáliby śię ná oſtátnie rzecży ſwoje.
Jákoby ugániáł jeden z nich tyśiąc / á dwáj goniliby dzieśięć tyśięcy! gdyby ich Bóg / opoká ich / nie záprzedáł / y Pán ich nie wydáł :
Abowiem nie jáko ſkáłá náƺá ſkáłá ich / co nieprzyjáćiele náśi ſámi oſądzą.
Abowiem z máćicy Sodomſkiej máćicá ich / y z látorośli Gomorſkich jágody ich / jágody jádowite / groná ich gorzkie.
Jád ſmocży wino ich / y trućizną żmij okrutną.
Izáli to nie jeſt ſkryto u mnie / zápiecżętowáno w ſkárbie moim?
Mojáć jeſt pomſtá y nágrodá ; cżáſu ſwego powinie śię nogá ich / poniewáż bliſki jeſt dzień zginieniá ich / á ſpieƺą śię przyƺłe rzecży do nich.
Abowiem ſądzić będzie Pán lud ſwój / á ſług ſwoich użáli śię / gdy ujrzy / że uſtáłá śiłá / á iż ták pojmány / jáko zoſtáwiony / nic nie mogą.
Y rzecże : Gdzież bogowie ich? Oná opoká / w której ufáli?
Którzy tłuſtość ofiár ich jádáli / y pijáli wino ofiár ich mokrych? Niechże wſtáną / y rátują wáſ / y niecháj będą ućiecżką wáƺą.
Obácżćież teráz / żem Já / Jám jeſt ſám / á że niemáƺ bogów oprócż Mnie ; Já zábijám y ożywiám / zránię / y Já ulecżę / á niemáƺ ktoby z ręki mojej wyrwáł.
Abowiem podnioſę ku niebu rękę moję / y rzekę : Żyję Já ná wieki.
Jeźli záoſtrzę błyƺcżący śię miecż ſwój / y porwie ſąd ręká mojá / oddám pomſtę nieprzyjáćiołom moim / y tym / którzy mię nienáwidzą / nágrodzę.
Opoję ſtrzáły moje krwią / á miecż mój náje śię mięſá ze krwi pobitych y pojmánych / ſkoro zácżnę mśćić śię nád nieprzyjáćioły.
Weſelćie śię národowie z ludem jego ; ábowiem krwi ſług ſwoich pomśći śię / y pomſtę oddá nieprzyjáćiołom ſwoim / á miłośćiw będzie ziemi ſwojej / y ludowi ſwemu.
Tedy przyƺedł Mojżeƺ / y mówił wƺyſtkie ſłowá pieśni tej w uƺách ludu tego / on y Jozue / ſyn Nunów.
A gdy dokonáł Mojżeƺ mówić tych wƺyſtkich ſłów do wƺyſtkiego Izráelá /
Rzekł do nich : Przyłóżćie ſercá wáƺe do wƺyſtkich ſłów / któremi śię já oświádcżám przećiwko wám dziś / ábyśćie je przykázywáli ſynom wáƺym / żeby ſtrzegli y pełnili wƺyſtkie ſłowá zákonu tego ;
Abowiem to ſłowo niemá być próżne u wáſ / gdyż jeſt żywotem wáƺym ; y w tem ſłowie przedłużyćie dni ná ziemi / do której śię przepráwiáćie przez Jordán / ábyśćie ją ośiedli.
Y rzekł Pán do Mojżeƺá tegoż dniá / mówiąc :
Wſtąp ná tę górę Abárym / ná górę Nebo / którá jeſt w ziemi Moábſkiej przećiw Jerychu / á oglądáj ziemię Chánánejſką / którą Já dáwám ſynom Izráelſkim w ośiádłość.
Y umrzeƺ ná tej górze / ná którą wſtąpiƺ / y będzieƺ przyłącżony do ludu twego / jáko umárł Aáron / brát twój / ná górze Hor / y przyłącżon jeſt do ludu ſwego ;
Dlá tego żeśćie wyſtąpili przećiwko mnie w pośród ſynów Izráelſkich przy wodách w poſwárku w Kádeſ ná puƺcży Syn / y żeśćie mię nie poświęćili w pośród ſynów Izráelſkich.
Bo przed ſobą oglądáƺ ziemię ; ále tám nie wnijdzieƺ / do tej ziemi / którą Já dáję ſynom Izráelſkim.
