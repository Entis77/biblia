Słuchájćie niebioſá / á mówić będę ; niech ſłuchá y źiemiá wymowy uſt moich.
Kropić będźie jáko deƺcż náuká mojá / popłynie jáko roſá wymowá mojá / jáko drobny deƺcż ná źiołá / á jáko krople ná tráwę.
Abowiem imieniá Páńſkiego będę wzywáł ; dájćież wielmożność Bogu náƺemu /
Skále / którey ſpráwy ſą doſkonáłe ; bo wƺyſtkie drogi jego ſą ſpráwiedliwe. Boć jeſt Bóg práwdźiwy / á niemáƺ w nim niepráwośći / ſpráwiedliwy y práwy jeſt.
Pokáźili mu śię / jákoby nie byli ſynámi jego / ſproſnośćią ſwoją / náród zły / á przewrotny.
Y tákże to oddáwáćie Pánu / ludu głupi y ƺálony? izáli nie on jeſt ojcem twoim / który ćię ſobie nábył? on ćię ucżynił / y ſtworzył ćię.
Wſpomnij ná dni dáwne / uwáżájćie látá káżdego wieku ; ſpytáj ojcá twego / á oznájmi tobie ; ſtárƺych twoich / á powiedząć.
Gdy rozdźielił dźiedźictwá Nájwyżƺy národom / á rozſádźił ſyny Adámowe / położył gránice národom według licżby ſynów Izráelſkich ;
Abowiem dźiáłem Páńſkim jeſt lud jego / Jákób ƺnurem dźiedźictwá jego.
Ználázł go w źiemi puſtey / y ná puƺcży wielkiej y ſtráƺney ; obwodźił go / ucżył go / ſtrzegł go / jáko źrenicy oká ſwego.
Jáko orzeł wybáwiá orlętá ſwoje / nád orlętámi ſwojimi látájąc / rozƺerzá ſkrzydłá ſwoje / bierze je y nośi je ná ſkrzydłách ſwoich :
Ták Pán ſám prowádźił go / á żáden obcy bóg nie był z nim.
Wynióſł go ná wyſokie miejſcá źiemi / áby używáł urodzájów polnych / y ucżynił / áby ſſáł miód z ſkáły y oliwę z opoki twárdej ;
Aby jádł máſło od krów / y mleko od owiec / z tłuſtośćią jágniąt / y bárány odchowáne w Báſán / y kozły z tłuſtośćią źiárn pƺenicżnych / á pił cżerwone wino wyborne.
Y roztył Izráel / y wierzgáł ( otyłeś / ſtłuśćiáłeś / zgrubiáłeś / y opuśćiłeś Bogá / który ćię ucżynił / ) y lekce powáżył Bogá / opokę zbáwieniá ſwego.
Ku zápálcżywośći pobudźili go przez cudze bogi / y przez obrzydliwośći rozdráźnili go.
Ofiárowáli dyjábłom / nie Bogu / bogom / których nie ználi / nowym / którzy z bliſká przyƺli / których śię nie báli ojcowie wáśi.
Opoki / która ćię ſpłodźiłá / zápomniáłeś / zápomniáłeś Bogá / Stworzyćielá twego.
Co widząc Pán poruƺył śię gniewem ná ſyny ſwoje / y ná córki ſwoje.
Y rzekł : Skryję oblicże moje przed nimi / obácżę / jáki będźie koniec ich ; ábowiem ſą národem przewrotnym / ſynámi / w których wiáry niemáƺ.
Oni mię wzruƺyli ku zápálcżywośći przez to / co nie jeſt Bogiem ; pobudźili mię próżnośćiámi ſwemi. Ja też do zázdrośći pobudzę je przez te / którzy nie ſą ludem / przez náród głupi rozdráźnię je.
Abowiem ogień rozpálił śię w popędliwośći mojey / y będźie gorzáł áż do nájgłębƺego piekłá / y pożre źiemię / y urodzáj jey / y wypáli grunty gór.
Zgromádzę ná nie złe rzecży / ſtrzáły moje ná nie wyſtrzelám.
Zniƺcżeją od głodu / y będą ſtráwieni gorącżką / y śmierćią gorzką ; zęby też beſtyi poślę ná nie / z jádem gádźin źiemſkich.
Zewnątrz ośieroći je miecż / á w pokojách będźie ſtrách / ták ná młodźieńcá / jáko ná pánnę / ná ſſącego pierśi / y ná mężá ſędźiwego.
Rzekłem : Rozproƺę je po kątách / y wygubię między ludźmi pámiątkę ich.
Gdybym śię ná pychę nieprzyjáćiół nie oglądáł / by śię ſnáć nie podnieśli nieprzyjáćiele ich / á nie rzekli : Ręká náƺá wynioſłá / á nie Pán / ſpráwiłá to wƺyſtko.
Abowiem ten náród ginie przez rády ſwe / y niemá rozumu.
O by mądrymi byli! wyrozumieliby to / y oglądáliby śię ná oſtátnie rzecży ſwoje.
Jákoby ugániáł jeden z nich tyśiąc / á dwáj goniliby dźieśięć tyśięcy! gdyby ich Bóg / opoká ich / nie záprzedáł / y Pán ich nie wydáł :
Abowiem nie jáko ſkáłá náƺá ſkáłá ich / co nieprzyjáćiele náśi ſámi oſądzą.
Abowiem z máćicy Sodomſkiej máćicá ich / y z látorośli Gomorſkich jágody ich / jágody jádowite / groná ich gorzkie.
Jád ſmocży wino ich / y trućizną żmij okrutną.
Izáli to nie jeſt ſkryto u mnie / zápiecżętowáno w ſkárbie moim?
Mojáć jeſt pomſtá y nágrodá ; cżáſu ſwego powinie śię nogá ich / poniewáż bliſki jeſt dźień zginieniá ich / á ſpieƺą śię przyƺłe rzecży do nich.
Abowiem ſądźić będźie Pán lud ſwój / á ſług ſwoich użáli śię / gdy ujrzy / że uſtáłá śiłá / á iż ták pojmány / jáko zoſtáwiony / nic nie mogą.
Y rzecże : Gdźież bogowie ich? Oná opoká / w którey ufáli?
Którzy tłuſtość ofiár ich jádáli / y pijáli wino ofiár ich mokrych? Niechże wſtáną / y rátują was / y niecháj będą ućiecżką wáƺą.
Obácżćież teraz / żem Ja / Jám jeſt ſám / á że niemáƺ bogów oprócż Mnie ; Ja zábijám y ożywiám / zránię / y Ja ulecżę / á niemáƺ ktoby z ręki mojey wyrwáł.
Abowiem podnioſę ku niebu rękę moję / y rzekę : Żyję Ja ná wieki.
Jeſli záoſtrzę błyƺcżący śię miecż ſwój / y porwie ſąd ręká mojá / oddám pomſtę nieprzyjáćiołom moim / y tym / którzy mię nienáwidzą / nágrodzę.
Opoję ſtrzáły moje krwią / á miecż mój náje śię mięſá ze krwi pobitych y pojmánych / ſkoro zácżnę mśćić śię nád nieprzyjáćioły.
Weſelćie śię národowie z ludem jego ; ábowiem krwi ſług ſwoich pomśći śię / y pomſtę oddá nieprzyjáćiołom ſwoim / á miłośćiw będźie źiemi ſwojey / y ludowi ſwemu.
Tedy przyƺedł Mojżeƺ / y mówił wƺyſtkie ſłowá pieśni tey w uƺách ludu tego / on y Jozue / ſyn Nunów.
A gdy dokonáł Mojżeƺ mówić tych wƺyſtkich ſłów do wƺyſtkiego Izráelá /
Rzekł do nich : Przyłóżćie ſercá wáƺe do wƺyſtkich ſłów / któremi śię ja oświádcżám przećiwko wám dźiś / ábyśćie je przykázywáli ſynom wáƺym / żeby ſtrzegli y pełnili wƺyſtkie ſłowá zakonu tego ;
Abowiem to ſłowo niemá być próżne u was / gdyż jeſt żywotem wáƺym ; y w tem ſłowie przedłużyćie dni ná źiemi / do którey śię przepráwiáćie przez Jordán / ábyśćie ją ośiedli.
Y rzekł Pán do Mojżeƺá tegoż dniá / mówiąc :
Wſtąp ná tę górę Abárym / ná górę Nebo / która jeſt w źiemi Moábſkiej przećiw Jerychu / á oglądáj źiemię Chánánejſką / którą Ja dáwám ſynom Izráelſkim w ośiádłość.
Y umrzeƺ ná tey górze / ná którą wſtąpiƺ / y będźieƺ przyłącżony do ludu twego / jáko umárł Aáron / brát twój / ná górze Hor / y przyłącżon jeſt do ludu ſwego ;
Dla tego żeśćie wyſtąpili przećiwko mnie w pośród ſynów Izráelſkich przy wodách w poſwárku w Kádeſ ná puƺcży Syn / y żeśćie mię nie poświęćili w pośród ſynów Izráelſkich.
Bo przed ſobą oglądáƺ źiemię ; ále tám nie wnijdźieƺ / do tey źiemi / którą Ja dáję ſynom Izráelſkim.
