Jeſlibyś ujrzał wołu brátá twego / álbo owcę jego obłąkáną / nie mijájże ich / ále je koniecżnie odprowádź do brátá twego.
Choććiby nie był bliſko brát twój / ánibyś go znał / przećię zápędźiƺ je do domu ſwego / y będźie przy tobie / áżby tego ƺukał brát twój / y wróćiƺ mu je.
Toż ucżyniƺ oſłowi jego ; toż też ucżyniƺ ƺáćie jego ; toż też ucżyniƺ wƺelákiej rzecży zgubioney brátá twego / któráby mu zginęłá / jeſlibyś ją ználázł / nie mijájże jey.
Jeſlibyś ujrzał oſłá brátá twego / álbo wołu jego / że upádł ná drodze / nie mijájże ich / ále go záráz z nim podnieś.
Niech niebierze niewiáſtá ná śię ƺát męſkich / áni niech śię nie obłócży mąż w ƺátę niewieśćią ; ábowiem jeſt obrzydliwośćią Pánu / Bogu twemu / ktoby to cżynił.
Gdybyś tráfił gniázdo ptáƺe przed ſobą w drodze / ná jákiemkolwiek drzewie / álbo ná źiemi / á w nim ptáƺętá álbo jájká / á mátkáby śiedźiáłá ná ptáƺętách álbo ná jájkách / nie bierzże máćierzy z dźiećmi ;
Ale wolno puśćiwƺy mátkę / dźieći weźmieƺ ſobie / ábyć śię dobrze dźiáło / y żebyś przedłużył dni twojich.
Gdy też zbudujeƺ dom nowy / tedy ucżyniƺ blánki po kráju dáchu twego / ábyś nie przywiódł krwi ná dom twój / gdyby kto ſpádł z niego.
Nie pośiewáj winnicy twojey / różnem náśieniem / byś ſnadź nie ſplugáwił pożytku náśieniá / któreś śiał / y urodzáju winnicy.
Nie będźieƺ orał wołem y oſłem poſpołu.
Nie oblecżeƺ ƺáty utkáney z wełny y ze lnu poſpołu.
Pocżyniƺ ſobie ƺnurki ná cżterech rogách ƺáty twojey / którą śię odźiewáć będźieƺ.
Gdyby kto pojął żonę / á wƺedƺy do niej / miáłby ją w nienáwiśći ;
A dáłby przycżynę / áby mówiono o niej / y wprowádźiłby ná nię złą ſłáwę / mówiąc : Pojąłem tę żonę / á wƺedƺy do niej / nie ználázłem jey pánną :
Tedy weźmie oćiec dźiewecżki / y mátká jey / y przynioſą znáki pánieńſtwá dźiewecżki oney do ſtárƺych miáſtá onego do brámy ;
Y rzecże oćiec oney dźiewecżki do ſtárƺych : Córkę ſwoję dáłem mężowi temu zá żonę ; á on ją ma w nienáwiśći ;
A oto ſám dał przycżynę / áby mówiono o niej / powiádájąc : Nie ználázłem przy córce twojey pánieńſtwá : ále oto ſą znáki pánieńſtwá córki mojey. Tedy rozwiną ono prześćierádło przed ſtárƺymi miáſtá onego.
A wźiąwƺy ſtárśi miáſtá onego mężá ſkárzą go.
Y wezmą z niego winę ſto ſrebrników / które oddádzą ojcu oney dźiewecżki / przeto / że puśćił złą ſłáwę o pánnie Izráelſkiej / y będźie ją miał zá żonę / y nie będźie jey mógł opuśćić po wƺyſtkie dni ſwoje.
Ale byłoliby práwdźiwe to obwinienie jey / á nie ználázłyby śię znáki pánieńſtwá przy oney dźiewecżce :
Tedy wywiodą onę dźiewecżkę przed drzwi domu ojcá jey / y ukámionują ją ludźie miáſtá onego / y umrze : bo ucżyniłá háńbę w Izráelu / popełniwƺy nierząd w domu ojcá ſwego ; ták odejmieƺ złe z pośród śiebie.
Jeſliby kto ználeźion był / obcujący z niewiáſtą / májącą mężá / tedy niech umrą oboje / mąż / który obcował z cudzą żoną / y oná niewiáſtá ; ták odejmieƺ złe z Izráelá.
Byłáliby dźiewecżká pánná poślubioná mężowi / á tráfiłby ją kto w mieśćie / y obcowáłby z nią ;
Tedy wywiódƺy ono oboje przed brámę miáſtá / ukámionujećie je / y umrą ; dźiewecżkę przeto / iż nie wołáłá w mieśćie / á mężá przeto / iż zelżył żonę bliźniego ſwego : ták odejmieƺ złe z pośrzodku śiebie.
A jeſliby ná polu tráfił mąż dźiewecżkę poślubioną á porwáwƺy ją / zgwáłćiłby ją / tedy umrze mąż / który obcował z nią / ſám tylko.
Ale dźiewecżce nic nie ucżyniƺ ; dźiewecżká nie jeſt winná śmierći ; bo jáko gdyby kto powſtáwƺy przećiwko bliźniemu ſwemu / zámordował go / táká to ſpráwá.
Poniewáż ją ná polu tráfił / á gdy wołáłá dźiewecżká poślubioná / nie był / ktoby ją rátował.
Gdyby też kto / tráfiwƺy dźiewecżkę pánnę / która nie jeſt zá mąż zmówioną / porwał ją / y leżáłby z nią / á záſtánoby je :
Tedy da on mąż / który obcował z nią / ojcu dźiewecżki pięćdźieśiąt ſrebrników / y będźie mu zá żonę / przeto że ją zelżył / áni jey będźie mógł opuśćić po wƺyſtkie dni ſwoje.
Nie pojmie nikt żony ojcá ſwego / y nie odkryje podołká ojcá ſwego.
