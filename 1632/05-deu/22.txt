Jeźlibyś ujrzáł wołu brátá twego / álbo owcę jego obłąkáną / nie mijájże ich / ále je koniecżnie odprowádź do brátá twego.
Choććiby nie był bliſko brát twój / ánibyś go znáł / przećię zápędziƺ je do domu ſwego / y będzie przy tobie / áżby tego ƺukáł brát twój / y wróćiƺ mu je.
Toż ucżyniƺ oſłowi jego ; toż też ucżyniƺ ƺáćie jego ; toż też ucżyniƺ wƺelákiej rzecży zgubionej brátá twego / któráby mu zginęłá / jeźlibyś ją ználázł / nie mijájże jej.
Jeźlibyś ujrzáł oſłá brátá twego / álbo wołu jego / że upádł ná drodze / nie mijájże ich / ále go záráz z nim podnieś.
Niech niebierze niewiáſtá ná śię ƺát męſkich / áni niech śię nie obłócży mąż w ƺátę niewieśćią ; ábowiem jeſt obrzydliwośćią Pánu / Bogu twemu / ktoby to cżynił.
Gdybyś tráfił gniázdo ptáƺe przed ſobą w drodze / ná jákiemkolwiek drzewie / álbo ná ziemi / á w nim ptáƺętá álbo jájká / á mátkáby śiedziáłá ná ptáƺętách álbo ná jájkách / nie bierzże máćierzy z dziećmi ;
Ale wolno puśćiwƺy mátkę / dzieći weźmieƺ ſobie / ábyć śię dobrze dziáło / y żebyś przedłużył dni twoich.
Gdy też zbudujeƺ dom nowy / tedy ucżyniƺ blánki po kráju dáchu twego / ábyś nie przywiódł krwi ná dom twój / gdyby kto ſpádł z niego.
Nie pośiewáj winnicy twojej / różnem náśieniem / byś ſnáć nie ſplugáwił pożytku náśieniá / któreś śiáł / y urodzáju winnicy.
Nie będzieƺ oráł wołem y oſłem poſpołu.
Nie oblecżeƺ ƺáty utkánej z wełny y ze lnu poſpołu.
Pocżyniƺ ſobie ƺnurki ná cżterech rogách ƺáty twojej / którą śię odziewáć będzieƺ.
Gdyby kto pojął żonę / á wƺedłƺy do niej / miáłby ją w nienáwiśći ;
A dáłby przycżynę / áby mówiono o niej / y wprowádziłby ná nię złą ſłáwę / mówiąc : Pojąłem tę żonę / á wƺedłƺy do niej / nie ználázłem jej pánną :
Tedy weźmie ojćiec dziewecżki / y mátká jej / y przynioſą znáki pánieńſtwá dziewecżki onej do ſtárƺych miáſtá onego do brámy ;
Y rzecże ojćiec onej dziewecżki do ſtárƺych : Córkę ſwoję dáłem mężowi temu zá żonę ; á on ją má w nienáwiśći ;
A oto ſám dáł przycżynę / áby mówiono o niej / powiádájąc : Nie ználázłem przy córce twojej pánieńſtwá : ále oto ſą znáki pánieńſtwá córki mojej. Tedy rozwiną ono prześćierádło przed ſtárƺymi miáſtá onego.
A wziąwƺy ſtárśi miáſtá onego mężá ſkárzą go.
Y wezmą z niego winę ſto ſrebrników / które oddádzą ojcu onej dziewecżki / przeto / że puśćił złą ſłáwę o pánnie Izráelſkiej / y będzie ją miáł zá żonę / y nie będzie jej mógł opuśćić po wƺyſtkie dni ſwoje.
Ale byłoliby práwdziwe to obwinienie jej / á nie ználázłyby śię znáki pánieńſtwá przy onej dziewecżce :
Tedy wywiodą onę dziewecżkę przed drzwi domu ojcá jej / y ukámionują ją ludzie miáſtá onego / y umrze : bo ucżyniłá háńbę w Izráelu / popełniwƺy nierząd w domu ojcá ſwego ; ták odejmieƺ złe z pośród śiebie.
Jeźliby kto ználezion był / obcujący z niewiáſtą / májącą mężá / tedy niech umrą oboje / mąż / który obcowáł z cudzą żoną / y oná niewiáſtá ; ták odejmieƺ złe z Izráelá.
Byłáliby dziewecżká pánná poślubioná mężowi / á tráfiłby ją kto w mieśćie / y obcowáłby z nią ;
Tedy wywiódłƺy ono oboje przed brámę miáſtá / ukámionujećie je / y umrą ; dziewecżkę przeto / iż nie wołáłá w mieśćie / á mężá przeto / iż zelżył żonę bliźniego ſwego : ták odejmieƺ złe z pośrodku śiebie.
A jeźliby ná polu tráfił mąż dziewecżkę poślubioną á porwáwƺy ją / zgwáłćiłby ją / tedy umrze mąż / który obcowáł z nią / ſám tylko.
Ale dziewecżce nic nie ucżyniƺ ; dziewecżká nie jeſt winná śmierći ; bo jáko gdyby kto powſtáwƺy przećiwko bliźniemu ſwemu / zámordowáł go / táká to ſpráwá.
Poniewáż ją ná polu tráfił / á gdy wołáłá dziewecżká poślubioná / nie był / ktoby ją rátowáł.
Gdyby też kto / tráfiwƺy dziewecżkę pánnę / którá nie jeſt zá mąż zmówioną / porwáł ją / y leżáłby z nią / á záſtánoby je :
Tedy dá on mąż / który obcowáł z nią / ojcu dziewecżki pięćdzieśiąt ſrebrników / y będzie mu zá żonę / przeto że ją zelżył / áni jej będzie mógł opuśćić po wƺyſtkie dni ſwoje.
Nie pojmie nikt żony ojcá ſwego / y nie odkryje podołká ojcá ſwego.
