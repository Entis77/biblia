Jeźlibyś ujrzáł wołu brátá twego / álbo owcę jego obłąkáną / nie mijájże ich / ále je koniecżnie odprowádź do brátá twego.
Choććiby nie był bliſko brát twój / ánibyś go znáł / przećię zápędźiƺ je do domu ſwego / y będźie przy tobie / áżby tego ƺukáł brát twój / y wróćiƺ mu je.
Toż ucżyniƺ oſłowi jego ; toż też ucżyniƺ ƺáćie jego ; toż też ucżyniƺ wƺelákiej rzecży zgubionej brátá twego / któráby mu zginęłá / jeźlibyś ją ználázł / nie mijájże jej.
Jeźlibyś ujrzáł oſłá brátá twego / álbo wołu jego / że upádł ná drodze / nie mijájże ich / ále go záráz z nim podnieś.
Niech niebierze niewiáſtá ná śię ƺát męſkich / áni niech śię nie obłócży mąż w ƺátę niewieśćią ; ábowiem jeſt obrzydliwośćią Pánu / Bogu twemu / ktoby to cżynił.
Gdybyś tráfił gniázdo ptáƺe przed ſobą w drodze / ná jákiemkolwiek drzewie / álbo ná źiemi / á w nim ptáƺętá álbo jájká / á mátkáby śiedźiáłá ná ptáƺętách álbo ná jájkách / nie bierzże máćierzy z dźiećmi ;
Ale wolno puśćiwƺy mátkę / dźieći weźmieƺ ſobie / ábyć śię dobrze dźiáło / y żebyś przedłużył dni twoich.
Gdy też zbudujeƺ dom nowy / tedy ucżyniƺ blánki po kráju dáchu twego / ábyś nie przywiódł krwi ná dom twój / gdyby kto ſpádł z niego.
Nie pośiewáj winnicy twojej / różnem náśieniem / byś ſnáć nie ſplugáwił pożytku náśieniá / któreś śiáł / y urodzáju winnicy.
Nie będźieƺ oráł wołem y oſłem poſpołu.
Nie oblecżeƺ ƺáty utkánej z wełny y ze lnu poſpołu.
Pocżyniƺ ſobie ƺnurki ná cżterech rogách ƺáty twojej / którą śię odźiewáć będźieƺ.
Gdyby kto pojął żonę / á wƺedłƺy do niej / miáłby ją w nienáwiśći ;
A dáłby przycżynę / áby mówiono o niej / y wprowádźiłby ná nię złą ſłáwę / mówiąc : Pojąłem tę żonę / á wƺedłƺy do niej / nie ználázłem jej pánną :
Tedy weźmie ojćiec dźiewecżki / y mátká jej / y przynioſą znáki pánieńſtwá dźiewecżki onej do ſtárƺych miáſtá onego do brámy ;
Y rzecże ojćiec onej dźiewecżki do ſtárƺych : Córkę ſwoję dáłem mężowi temu zá żonę ; á on ją má w nienáwiśći ;
A oto ſám dáł przycżynę / áby mówiono o niej / powiádájąc : Nie ználázłem przy córce twojej pánieńſtwá : ále oto ſą znáki pánieńſtwá córki mojej. Tedy rozwiną ono prześćierádło przed ſtárƺymi miáſtá onego.
A wźiąwƺy ſtárśi miáſtá onego mężá ſkárzą go.
Y wezmą z niego winę ſto ſrebrników / które oddádzą ojcu onej dźiewecżki / przeto / że puśćił złą ſłáwę o pánnie Izráelſkiej / y będźie ją miáł zá żonę / y nie będźie jej mógł opuśćić po wƺyſtkie dni ſwoje.
Ale byłoliby práwdźiwe to obwinienie jej / á nie ználázłyby śię znáki pánieńſtwá przy onej dźiewecżce :
Tedy wywiodą onę dźiewecżkę przed drzwi domu ojcá jej / y ukámionują ją ludźie miáſtá onego / y umrze : bo ucżyniłá háńbę w Izráelu / popełniwƺy nierząd w domu ojcá ſwego ; ták odejmieƺ złe z pośród śiebie.
Jeźliby kto ználeźion był / obcujący z niewiáſtą / májącą mężá / tedy niech umrą oboje / mąż / który obcowáł z cudzą żoną / y oná niewiáſtá ; ták odejmieƺ złe z Izráelá.
Byłáliby dźiewecżká pánná poślubioná mężowi / á tráfiłby ją kto w mieśćie / y obcowáłby z nią ;
Tedy wywiódłƺy ono oboje przed brámę miáſtá / ukámionujećie je / y umrą ; dźiewecżkę przeto / iż nie wołáłá w mieśćie / á mężá przeto / iż zelżył żonę bliźniego ſwego : ták odejmieƺ złe z pośrodku śiebie.
A jeźliby ná polu tráfił mąż dźiewecżkę poślubioną á porwáwƺy ją / zgwáłćiłby ją / tedy umrze mąż / który obcowáł z nią / ſám tylko.
Ale dźiewecżce nic nie ucżyniƺ ; dźiewecżká nie jeſt winná śmierći ; bo jáko gdyby kto powſtáwƺy przećiwko bliźniemu ſwemu / zámordowáł go / táká to ſpráwá.
Poniewáż ją ná polu tráfił / á gdy wołáłá dźiewecżká poślubioná / nie był / ktoby ją rátowáł.
Gdyby też kto / tráfiwƺy dźiewecżkę pánnę / którá nie jeſt zá mąż zmówioną / porwáł ją / y leżáłby z nią / á záſtánoby je :
Tedy dá on mąż / który obcowáł z nią / ojcu dźiewecżki pięćdźieśiąt ſrebrników / y będźie mu zá żonę / przeto że ją zelżył / áni jej będźie mógł opuśćić po wƺyſtkie dni ſwoje.
Nie pojmie nikt żony ojcá ſwego / y nie odkryje podołká ojcá ſwego.
