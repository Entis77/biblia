Jeźli pilnie ſłucháć będzieƺ głoſu Páná / Bogá twego / ſtrzegąc y cżyniąc wƺyſtkie przykázániá jego / które já dziś przykázuję tobie / tedy ćię Pán / Bóg twój / wywyżƺy nád wƺyſtkie národy ziemi.
Y przyjdą ná ćię te wƺyſtkie błogoſłáwieńſtwá / y trzymáć śię ćiebie będą / jeźli będzieƺ poſłuƺnym głoſu Páná / Bogá twego.
Błogoſłáwiony będzieƺ w mieśćie / błogoſłáwiony będzieƺ y ná polu ;
Błogoſłáwiony owoc żywotá twego / y owoc ziemi twojej / y owoc bydłá twego / płód rogátego bydłá twego / y trzody drobnego bydłá twego.
Błogoſłáwiony koƺ twój / y dzieżá twojá.
Błogoſłáwiony będzieƺ wchodząc / błogoſłáwiony y wychodząc.
Spráwi Pán / że nieprzyjáćiele twoi / którzy powſtáwáją przećiwko tobie / będą poráżeni przed oblicżem twojim ; drogą jedną wyćiągną przećiwko tobie / á śiedmią dróg ućiekáć będą przed oblicżem twojim.
Przykáże Pán błogoſłáwieńſtwu ſwemu / áby z tobą było w ƺpiżárniách twoich / y we wƺyſtkiem / do cżegobyś śćiągnął rękę twoję / y będzieć błogoſłáwił w ziemi / którą Pán / Bóg twój dáwá tobie.
Wyſtáwi ćię Pán ſobie zá lud święty / jákoć przyśiągł / jeźli przeſtrzegáć będzieƺ przykázáń Páná / Bogá twego / y będzieƺ chodził drogámi jego.
Y obácżą wƺyſtkie národy ziemi / że imię Páńſkie wzywáne jeſt nád tobą / á będą śię ćiebie lękáć.
Y ſpráwi Pán / że będzieƺ obfitowáł w dobrem / w owocu żywotá twego / y w owocu bydłá twego / y w owocu polá twego / w ziemi / o którą przyśiągł Pán ojcom twoim / że ją tobie dá.
Otworzyć Pán ſkárb ſwój wyborny / niebioſá / áby wydáwáły deƺcż ziemi twojej cżáſu ſwego / y pobłogoſłáwi wƺelkiej ſpráwie rąk twoich / y będzieƺ pożycżáł wielom národom / á ſám nie będzieƺ pożycżáł.
Y ucżyni ćię Pán przedniejƺym / á nie pośledniejƺym ; y będzieƺ tylko wyżƺy / á nie będzieƺ niżƺy / jeźli poſłuƺnym będzieƺ przykázániom Páná / Bogá twego / które já dziś rozkázuję tobie / ábyś ich ſtrzegł / y cżynił je.
A nie uſtąpiƺ od żádnego ſłowá / które já wám przykázuję dziśiáj / áni ná práwo áni ná lewo / idąc zá bogámi cudzymi / ábyś im ſłużył.
Lecż jeźli poſłuƺny nie będzieƺ głoſu Páná / Bogá twego / ábyś ſtrzegł y cżynił wƺyſtkie przykázániá jego y uſtáwy jego / które já przykázuję tobie dziś / tedy przyjdą ná ćię wƺyſtkie te przeklęſtwá / y ogárną ćię.
Przeklętym będzieƺ w mieśćie / przeklętym y ná polu.
Przeklęty koƺ twój / y dzieżá twojá.
Przeklęty owoc żywotá twego / y owoc ziemi twojej / płód rogátego bydłá twego / y trzody drobnego bydłá twego.
Przeklętym będzieƺ wchodząc / przeklętym y wychodząc.
Y pośle Pán ná ćię przeklęſtwo / trwogę / y zgubę we wƺyſtkiem / do cżego śćiągnieƺ rękę twoję / y co cżynić będzieƺ ; áż ćię wygłádzi / y áż záginieƺ nágle dlá złośći ſpráw twoich / któremiś mię odſtąpił.
Przepuśći Pán ná ćię morowe powietrze / áż ćię wyniƺcży z ziemi / do której idzieƺ / ábyś ją pośiádł.
Uderzy ćię Pán ſuchotámi / y zimnicą / y gorącżką / y upáleniem / y miecżem / y ſuƺą / y rdzą / y będą ćię dogániáć / áż wyginieƺ.
Y będzie niebo twoje / które jeſt nád głową twoją / miedziánem / y ziemiá / którá jeſt pod tobą / żelázną.
Dá Pán zá deƺcż ziemi twojej proch y popiół / á ten z niebá pádáć będzie ná ćię / áż zniƺcżejeƺ.
Podá ćię Pán ná upádek przed nieprzyjáćioły twymi ; drogą jedną wynijdzieƺ przećiwko nim / á śiedmią dróg będzieƺ ućiekáł przed twárzą ich / y będzieƺ ku wzruƺeniu wƺyſtkim króleſtwom ziemi.
A będą trupy twoje pokármem wƺelkiemu ptáſtwu powietrznemu / y zwierzowi ziemſkiemu / á nie będzie / ktoby ich odpędził.
Zárázi ćię Pán wrzodem Egipſkim / y niemocą zádnicy / y kroſtámi / y świerzbem / á nie będzieƺ mógł być ulecżony.
Zárázi ćię Pán ƺáleńſtwem / y ślepotą / y zdrętwiáłośćią ſercá.
Y będzieƺ mácáł o południu / jáko mácá ślepy w ćiemnośći ; á nie będąć śię ƺcżęśćiły drogi twoje ; do tego też będzieƺ ućiśniony / y ƺárpány po wƺyſtkie dni / á nie będzie / ktoby ćię wybáwił.
Żonę ſobie poślubiƺ / á inƺy mąż z nią będzie ſpáł ; dom zbudujeƺ á mieƺkáć w nim nie będzieƺ ; winnicę náſádziƺ / á używáć jej nie będzieƺ.
Wołu twego zábiją przed ocżymá twemi / á nie będzieƺ go jádł ; oſłá twego porwą przed twárzą twoją / á nie wrócąć go ; trzody twoje podáne będą nieprzyjáćiołom twoim / á nie będzie / ktoby ćię rátowáł.
Synowie twoi / córki twoje / národowi innemu wydáne będą / á ocży twoje pátrząc ná to uſtáwáć będą dlá nich przez cáły dzień / á nie będzie śiły w ręce twojej.
Owoc ziemi twojej / y wƺyſtkę prácę twoję pożre náród / którego ty nie znáƺ / á nie będzieƺ jedno ućiśniony / y udręcżony po wƺyſtkie dni.
Y będzieƺ jáko ƺálony / widząc to ocżymá twemi / ná co pátrzeć muśiƺ.
Zárázi ćię Pán wrzodem złym ná kolánách y ná łyſtách / ták / że nie będzieƺ mógł być ulecżonym / od ſtopy nogi twej y áż do wierzchu głowy twojej.
Záwiedzie Pán ćiebie / y królá twego / którego poſtánowiƺ nád ſobą / do národu / któregoś nie znáł / ty y ojcowie twoi / gdzie będzieƺ ſłużył bogom obcym / drewnu y kámieniowi.
Y będzieƺ dziwowiſkiem / báśnią / y przyſłowiem u wƺyſtkich národów / do których ćię záwiedzie Pán.
Náśieniá wiele wynieśieƺ ná pole / á máło zbierzeƺ ; bo to pożre ƺáráńcżá.
Winnicę náſádziƺ y upráwiƺ / ále winá nie będzieƺ pił áni zbieráł ; bo je pożre robáctwo.
Oliwnego drzewá doſyć mieć będzieƺ po wƺyſtkich gránicách twoich / á oliwą mázáć śię nie będzieƺ ; bo opádáją oliwki twoje.
Synów y córek nápłodziƺ / ále nie będą twoje ; bo pójdą w niewolą.
Wƺyſtkie drzewá twoje / y owoc ziemi / ƺáráńcżá pożre.
Cudzoziemiec / który mieƺká w pośrodku ćiebie / urośnie nád ćię znácżnie ; ále ty wielce poniżonym będzieƺ.
On będzie pożycżáł tobie / á ty mu nie będzieƺ pożycżáł / on będzie przedniejƺy / á ty będzieƺ pośledniejƺy.
Y przyjdą ná ćię wƺyſtkie te przeklęſtwá / y będą ćię goniły / y ogárną ćię / áż zniƺcżejeƺ / poniewáżeś nie był poſłuƺny głoſowi Páná / Bogá twego / ániś chowáł przykázániá jego / y uſtáw jego / któreć przykázáł.
A będą te plági ná tobie y ná náśieniu twem / zá znák y zá cud áż ná wieki.
Dlá tego / żeś nie ſłużył Pánu / Bogu twemu / z ućiechą / y z weſelem ſercá / májąc wƺyſtkiego doſtátek.
Y będzieƺ nieprzyjáćiołom twoim / które ná ćię Pán pośle / ſłużył w głodzie / y w prágnieniu / y w nágośći / y w niedoſtátku wƺyſtkiego ; y włoży járzmo żelázne ná ƺyję twoję / áż ćię wniwecż obróći.
Przywiedzie Pán ná ćię náród z dáleká / od końcżyn ziemi / który przyleći jáko orzeł / náród / którego języká nie zrozumieƺ.
Náród ſrogi / który nie będzie miáł względu ná ſtárego áni śię nád dziećięćiem zmiłuje ;
Y pożre owoc bydłá twego / y owoc ziemi twojej / áż ćię zniƺcży ; y nie zoſtáwić zbożá / moƺcżu / áni świeżej oliwy / áni ſtád wołów twoich / áni trzód owiec twoich / áż ćię wygubi ;
Y oblęże ćię we wƺyſtkich brámách twoich / áż upádną mury twe wyſokie y obronne / w którycheś ty ufáł po wƺyſtkiej ziemi twojej ; oblęże ćię we wƺyſtkich brámách twoich / po wƺyſtkiej ziemi twojej / którą dá Pán / Bóg twój / tobie.
Y będzieƺ jádł płód żywotá twego / ćiáło ſynów twoich / y córek twoich / któreć dáł Pán / Bóg twój / w onem oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój.
Mąż pieƺcżotliwy między wámi / y w rozkoƺy wychowány / będzie zájrzáł brátu ſwemu / y włáſnej żonie ſwej / y oſtátkowi ſynów ſwych / który pozoſtánie.
Y nie udzieli żádnemu z nich z mięſá ſynów ſwych / które jeść będzie / przeto że mu nie zoſtáło nic inƺego w oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój we wƺyſtkich brámách twoich.
Pieƺcżotliwá między wámi / y w rozkoƺy wychowáná niewiáſtá / którá ledwie nogą ſwoją doſtępowáłá ziemi dlá pieƺcżoty y rozkoƺy / będzie zájrzáłá mężowi ſwemu włáſnemu y ſynowi ſwemu / y córce ſwojej.
Y łożyſká ſwego / które wychodzi z niej przy porodzeniu / y ſynów ſwych / które urodzi ; bo je poje w niedoſtátku wƺyſtkiego potájemnie w oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój w brámách twoich.
Jeźliże nie będzieƺ przeſtrzegáł / ábyś cżynił wƺyſtkie ſłowá zákonu tego / które nápiſáne ſą w kśięgách tych / żebyś śię báł tego imieniá chwálebnego y ſtráƺnego Páná / Bogá twego ;
Rozmnoży ná podziw Pán plági twoje / y plági náśieniá twego / plági wielkie y trwáłe / tákże choroby złe y długie ;
Y obróći ná ćię wƺyſtkie choroby Egipſkie / którycheś śię lękáł / y chwycą śię ćiebie.
Wƺeláką też niemoc / y wƺeláką plágę / którá nie jeſt nápiſáná w kśięgách zákonu tego / przywiedzie Pán ná ćię / áż ćię wytráći.
Y zoſtánie wáſ bárdzo máło / którzyśćie przedtem byli jáko gwiázdy niebieſkie przez mnóſtwo / poniewáżeś nie był poſłuƺny głoſowi Páná / Bogá twego /
Y ſtánie śię / że jáko śię rádowáł Pán nád wámi / dobrze wám cżyniąc y rozmnáżájąc wáſ / ták śię rádowáć będzie Pán nád wámi trácąc wáſ / y wygłádzájąc wáſ ; y będziećie wykorzenieni z ziemi / do której idziećie / ábyśćie ją pośiedli.
Y rozproƺy ćię Pán między wƺyſtkie národy / od końcżyn ziemi y áż do końcżyn ziemi ; támże ſłużyć będzieƺ bogom cudzym / którycheś nie znáł / ty y ojcowie twoi / drewnu y kámieniowi.
A między onymi národámi nie wytchnieƺ ſobie / áni będzie miáłá odpocżynku ſtopá nogi twojej ; dáć też Pán támże ſerce lękliwe / y ocży zemdlone / y myśl ſfráſowáną.
Y będzie żywot twój jákoby záwieƺony przed tobą / y będzieƺ śię lękáł w nocy y we dnie / y nie będzieƺ pewien żywotá twego.
Ráno rzecżeƺ : Któż mi dá wiecżór? á w wiecżór rzecżeƺ : Któż mi dá záránie? dlá trwogi ſercá twego / którą śię zátrwożyƺ / y dlá tego / ná co ocżymá twemi pátrzeć muśiƺ.
Y wróći ćię Pán do Egiptu w okrętách / drogą / o którejmći powiedziáł : Nie oglądáƺ jej więcej ; á tám záprzedáwáć śię będziećie nieprzyjáćiołom ſwoim zá niewolniki / y zá niewolnice / á nie będzie / ktoby wáſ kupił.
