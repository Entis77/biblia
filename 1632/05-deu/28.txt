Jeźli pilnie ſłucháć będźieƺ głoſu Páná / Bogá twego / ſtrzegąc y cżyniąc wƺyſtkie przykázániá jego / które já dźiś przykázuję tobie / tedy ćię Pán / Bóg twój / wywyżƺy nád wƺyſtkie národy źiemi.
Y przyjdą ná ćię te wƺyſtkie błogoſłáwieńſtwá / y trzymáć śię ćiebie będą / jeźli będźieƺ poſłuƺnym głoſu Páná / Bogá twego.
Błogoſłáwiony będźieƺ w mieśćie / błogoſłáwiony będźieƺ y ná polu ;
Błogoſłáwiony owoc żywotá twego / y owoc źiemi twojej / y owoc bydłá twego / płód rogátego bydłá twego / y trzody drobnego bydłá twego.
Błogoſłáwiony koƺ twój / y dźieżá twojá.
Błogoſłáwiony będźieƺ wchodząc / błogoſłáwiony y wychodząc.
Spráwi Pán / że nieprzyjáćiele twoi / którzy powſtáwáją przećiwko tobie / będą poráżeni przed oblicżem twojim ; drogą jedną wyćiągną przećiwko tobie / á śiedmią dróg ućiekáć będą przed oblicżem twojim.
Przykáże Pán błogoſłáwieńſtwu ſwemu / áby z tobą było w ƺpiżárniách twoich / y we wƺyſtkiem / do cżegobyś śćiągnął rękę twoję / y będźieć błogoſłáwił w źiemi / którą Pán / Bóg twój dáwá tobie.
Wyſtáwi ćię Pán ſobie zá lud święty / jákoć przyśiągł / jeźli przeſtrzegáć będźieƺ przykázáń Páná / Bogá twego / y będźieƺ chodźił drogámi jego.
Y obácżą wƺyſtkie národy źiemi / że imię Páńſkie wzywáne jeſt nád tobą / á będą śię ćiebie lękáć.
Y ſpráwi Pán / że będźieƺ obfitowáł w dobrem / w owocu żywotá twego / y w owocu bydłá twego / y w owocu polá twego / w źiemi / o którą przyśiągł Pán ojcom twoim / że ją tobie dá.
Otworzyć Pán ſkárb ſwój wyborny / niebioſá / áby wydáwáły deƺcż źiemi twojej cżáſu ſwego / y pobłogoſłáwi wƺelkiej ſpráwie rąk twoich / y będźieƺ pożycżáł wielom národom / á ſám nie będźieƺ pożycżáł.
Y ucżyni ćię Pán przedniejƺym / á nie pośledniejƺym ; y będźieƺ tylko wyżƺy / á nie będźieƺ niżƺy / jeźli poſłuƺnym będźieƺ przykázániom Páná / Bogá twego / które já dźiś rozkázuję tobie / ábyś ich ſtrzegł / y cżynił je.
A nie uſtąpiƺ od żádnego ſłowá / które já wám przykázuję dźiśiáj / áni ná práwo áni ná lewo / idąc zá bogámi cudzymi / ábyś im ſłużył.
Lecż jeźli poſłuƺny nie będźieƺ głoſu Páná / Bogá twego / ábyś ſtrzegł y cżynił wƺyſtkie przykázániá jego y uſtáwy jego / które já przykázuję tobie dźiś / tedy przyjdą ná ćię wƺyſtkie te przeklęſtwá / y ogárną ćię.
Przeklętym będźieƺ w mieśćie / przeklętym y ná polu.
Przeklęty koƺ twój / y dźieżá twojá.
Przeklęty owoc żywotá twego / y owoc źiemi twojej / płód rogátego bydłá twego / y trzody drobnego bydłá twego.
Przeklętym będźieƺ wchodząc / przeklętym y wychodząc.
Y pośle Pán ná ćię przeklęſtwo / trwogę / y zgubę we wƺyſtkiem / do cżego śćiągnieƺ rękę twoję / y co cżynić będźieƺ ; áż ćię wygłádźi / y áż záginieƺ nágle dlá złośći ſpráw twoich / któremiś mię odſtąpił.
Przepuśći Pán ná ćię morowe powietrze / áż ćię wyniƺcży z źiemi / do której idźieƺ / ábyś ją pośiádł.
Uderzy ćię Pán ſuchotámi / y źimnicą / y gorącżką / y upáleniem / y miecżem / y ſuƺą / y rdzą / y będą ćię dogániáć / áż wyginieƺ.
Y będźie niebo twoje / które jeſt nád głową twoją / miedźiánem / y źiemiá / którá jeſt pod tobą / żelázną.
Dá Pán zá deƺcż źiemi twojej proch y popiół / á ten z niebá pádáć będźie ná ćię / áż zniƺcżejeƺ.
Podá ćię Pán ná upádek przed nieprzyjáćioły twymi ; drogą jedną wynijdźieƺ przećiwko nim / á śiedmią dróg będźieƺ ućiekáł przed twárzą ich / y będźieƺ ku wzruƺeniu wƺyſtkim króleſtwom źiemi.
A będą trupy twoje pokármem wƺelkiemu ptáſtwu powietrznemu / y zwierzowi źiemſkiemu / á nie będźie / ktoby ich odpędźił.
Záráźi ćię Pán wrzodem Egipſkim / y niemocą zádnicy / y kroſtámi / y świerzbem / á nie będźieƺ mógł być ulecżony.
Záráźi ćię Pán ƺáleńſtwem / y ślepotą / y zdrętwiáłośćią ſercá.
Y będźieƺ mácáł o południu / jáko mácá ślepy w ćiemnośći ; á nie będąć śię ƺcżęśćiły drogi twoje ; do tego też będźieƺ ućiśniony / y ƺárpány po wƺyſtkie dni / á nie będźie / ktoby ćię wybáwił.
Żonę ſobie poślubiƺ / á inƺy mąż z nią będźie ſpáł ; dom zbudujeƺ á mieƺkáć w nim nie będźieƺ ; winnicę náſádźiƺ / á używáć jej nie będźieƺ.
Wołu twego zábiją przed ocżymá twemi / á nie będźieƺ go jádł ; oſłá twego porwą przed twárzą twoją / á nie wrócąć go ; trzody twoje podáne będą nieprzyjáćiołom twoim / á nie będźie / ktoby ćię rátowáł.
Synowie twoi / córki twoje / národowi innemu wydáne będą / á ocży twoje pátrząc ná to uſtáwáć będą dlá nich przez cáły dźień / á nie będźie śiły w ręce twojej.
Owoc źiemi twojej / y wƺyſtkę prácę twoję pożre náród / którego ty nie znáƺ / á nie będźieƺ jedno ućiśniony / y udręcżony po wƺyſtkie dni.
Y będźieƺ jáko ƺálony / widząc to ocżymá twemi / ná co pátrzeć muśiƺ.
Záráźi ćię Pán wrzodem złym ná kolánách y ná łyſtách / ták / że nie będźieƺ mógł być ulecżonym / od ſtopy nogi twej y áż do wierzchu głowy twojej.
Záwiedźie Pán ćiebie / y królá twego / którego poſtánowiƺ nád ſobą / do národu / któregoś nie znáł / ty y ojcowie twoi / gdźie będźieƺ ſłużył bogom obcym / drewnu y kámieniowi.
Y będźieƺ dźiwowiſkiem / báśnią / y przyſłowiem u wƺyſtkich národów / do których ćię záwiedźie Pán.
Náśieniá wiele wynieśieƺ ná pole / á máło zbierzeƺ ; bo to pożre ƺáráńcżá.
Winnicę náſádźiƺ y upráwiƺ / ále winá nie będźieƺ pił áni zbieráł ; bo je pożre robáctwo.
Oliwnego drzewá doſyć mieć będźieƺ po wƺyſtkich gránicách twoich / á oliwą mázáć śię nie będźieƺ ; bo opádáją oliwki twoje.
Synów y córek nápłodźiƺ / ále nie będą twoje ; bo pójdą w niewolą.
Wƺyſtkie drzewá twoje / y owoc źiemi / ƺáráńcżá pożre.
Cudzoźiemiec / który mieƺká w pośrodku ćiebie / urośnie nád ćię znácżnie ; ále ty wielce poniżonym będźieƺ.
On będźie pożycżáł tobie / á ty mu nie będźieƺ pożycżáł / on będźie przedniejƺy / á ty będźieƺ pośledniejƺy.
Y przyjdą ná ćię wƺyſtkie te przeklęſtwá / y będą ćię goniły / y ogárną ćię / áż zniƺcżejeƺ / poniewáżeś nie był poſłuƺny głoſowi Páná / Bogá twego / ániś chowáł przykázániá jego / y uſtáw jego / któreć przykázáł.
A będą te plági ná tobie y ná náśieniu twem / zá znák y zá cud áż ná wieki.
Dlá tego / żeś nie ſłużył Pánu / Bogu twemu / z ućiechą / y z weſelem ſercá / májąc wƺyſtkiego doſtátek.
Y będźieƺ nieprzyjáćiołom twoim / które ná ćię Pán pośle / ſłużył w głodźie / y w prágnieniu / y w nágośći / y w niedoſtátku wƺyſtkiego ; y włoży járzmo żelázne ná ƺyję twoję / áż ćię wniwecż obróći.
Przywiedźie Pán ná ćię náród z dáleká / od końcżyn źiemi / który przyleći jáko orzeł / náród / którego języká nie zrozumieƺ.
Náród ſrogi / który nie będźie miáł względu ná ſtárego áni śię nád dźiećięćiem zmiłuje ;
Y pożre owoc bydłá twego / y owoc źiemi twojej / áż ćię zniƺcży ; y nie zoſtáwić zbożá / moƺcżu / áni świeżej oliwy / áni ſtád wołów twoich / áni trzód owiec twoich / áż ćię wygubi ;
Y oblęże ćię we wƺyſtkich brámách twoich / áż upádną mury twe wyſokie y obronne / w którycheś ty ufáł po wƺyſtkiej źiemi twojej ; oblęże ćię we wƺyſtkich brámách twoich / po wƺyſtkiej źiemi twojej / którą dá Pán / Bóg twój / tobie.
Y będźieƺ jádł płód żywotá twego / ćiáło ſynów twoich / y córek twoich / któreć dáł Pán / Bóg twój / w onem oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój.
Mąż pieƺcżotliwy między wámi / y w rozkoƺy wychowány / będźie zájrzáł brátu ſwemu / y włáſnej żonie ſwej / y oſtátkowi ſynów ſwych / który pozoſtánie.
Y nie udźieli żádnemu z nich z mięſá ſynów ſwych / które jeść będźie / przeto że mu nie zoſtáło nic inƺego w oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój we wƺyſtkich brámách twoich.
Pieƺcżotliwá między wámi / y w rozkoƺy wychowáná niewiáſtá / którá ledwie nogą ſwoją doſtępowáłá źiemi dlá pieƺcżoty y rozkoƺy / będźie zájrzáłá mężowi ſwemu włáſnemu y ſynowi ſwemu / y córce ſwojej.
Y łożyſká ſwego / które wychodźi z niej przy porodzeniu / y ſynów ſwych / które urodźi ; bo je poje w niedoſtátku wƺyſtkiego potájemnie w oblężeniu y śćiśnieniu / którem ćię śćiśnie nieprzyjáćiel twój w brámách twoich.
Jeźliże nie będźieƺ przeſtrzegáł / ábyś cżynił wƺyſtkie ſłowá zákonu tego / które nápiſáne ſą w kśięgách tych / żebyś śię báł tego imieniá chwálebnego y ſtráƺnego Páná / Bogá twego ;
Rozmnoży ná podźiw Pán plági twoje / y plági náśieniá twego / plági wielkie y trwáłe / tákże choroby złe y długie ;
Y obróći ná ćię wƺyſtkie choroby Egipſkie / którycheś śię lękáł / y chwycą śię ćiebie.
Wƺeláką też niemoc / y wƺeláką plágę / którá nie jeſt nápiſáná w kśięgách zákonu tego / przywiedźie Pán ná ćię / áż ćię wytráći.
Y zoſtánie wáſ bárdzo máło / którzyśćie przedtem byli jáko gwiázdy niebieſkie przez mnóſtwo / poniewáżeś nie był poſłuƺny głoſowi Páná / Bogá twego /
Y ſtánie śię / że jáko śię rádowáł Pán nád wámi / dobrze wám cżyniąc y rozmnáżájąc wáſ / ták śię rádowáć będźie Pán nád wámi trácąc wáſ / y wygłádzájąc wáſ ; y będźiećie wykorzenieni z źiemi / do której idźiećie / ábyśćie ją pośiedli.
Y rozproƺy ćię Pán między wƺyſtkie národy / od końcżyn źiemi y áż do końcżyn źiemi ; támże ſłużyć będźieƺ bogom cudzym / którycheś nie znáł / ty y ojcowie twoi / drewnu y kámieniowi.
A między onymi národámi nie wytchnieƺ ſobie / áni będźie miáłá odpocżynku ſtopá nogi twojej ; dáć też Pán támże ſerce lękliwe / y ocży zemdlone / y myśl ſfráſowáną.
Y będźie żywot twój jákoby záwieƺony przed tobą / y będźieƺ śię lękáł w nocy y we dnie / y nie będźieƺ pewien żywotá twego.
Ráno rzecżeƺ : Któż mi dá wiecżór? á w wiecżór rzecżeƺ : Któż mi dá záránie? dlá trwogi ſercá twego / którą śię zátrwożyƺ / y dlá tego / ná co ocżymá twemi pátrzeć muśiƺ.
Y wróći ćię Pán do Egiptu w okrętách / drogą / o którejmći powiedźiáł : Nie oglądáƺ jej więcej ; á tám záprzedáwáć śię będźiećie nieprzyjáćiołom ſwoim zá niewolniki / y zá niewolnice / á nie będźie / ktoby wáſ kupił.
