Wƺyſtkie przykázániá / które já dziś przykázuję tobie / chowáć y cżynić będziećie / ábyśćie żyli / rozmnożeni byli / y weƺli / á ośiedli tę ziemię / o którą przyśiągł Pán ojcom wáƺym.
Miejże w pámięći wƺyſtkę drogę / którą ćię prowádził Pán / Bóg twój / przez te cżterdzieśći lát po puƺcży / áby ćię trápił / y doświádcżył / áby wiádomo było / co jeſt w ſercu twojim / jeźlibyś przeſtrzegáł przykázáń jego / cżyli nie.
Przetoż dręcżył ćię / y głodem ći dokucżáł ; potem kármił ćię mánną / którejś nieznáł / áni jej też ználi ojcowie twoi / áby ćię náucżył / iż nie ſámym chlebem żyje cżłowiek / ále tem wƺyſtkiem / co pochodzi z uſt Páńſkich / żyć będzie cżłowiek.
Szátá twojá nie wiotƺáłá ná tobie / y nogá twojá nie nápuchłá przez te cżterdzieśći lát.
Poznájże tedy w ſercu ſwem / że jáko ćwicży cżłowiek ſyná ſwego / ták Pán / Bóg twój / ćwicży ćiebie.
A przeſtrzegáj przykázáń Páná / Bogá twego / ábyś chodził drogámi jego / y báł śię go.
Abowiem Pán / Bóg twój / prowádzi ćię do ziemi tej dobrej / do ziemi / gdzie ſą potoki wód / źródłá y przepáśći wynikájące po dolinách y po górách ;
Do ziemi obfitej w pƺenicę / y w jęcżmień / y w wino / y w figi / y w jábłká gránátowe ; do ziemi hojnej w oliwę y w miód ;
Do ziemi / w której bez niedoſtátku jeść będzieƺ chleb / á ná żádnej rzecży ſchodzić ći nie będzie ; do ziemi / której kámienie ſą żelázo / á z jej gór miedź kopáć będzieƺ.
Gdy będzieƺ jádł / á nájeƺ śię / będzieƺ chwálił Páná / Bogá twego / zá ziemię wyborną / którą dáł tobie.
Strzeżże śię / byś ſnáć nie zápomniáł Páná / Bogá twego / á nie zániedbáł przykázáń jego / y ſądów jego / y uſtáw jego / które já dziś rozkázuję tobie ;
By ſnáć / gdy jeść będzieƺ do ſytośći / y piękne domy pobudujeƺ / á mieƺkáć w nich będzieƺ ;
Y gdyć śię wołów y owiec námnoży / ſrebrá tákże y złotá będzieƺ miáł doſyć / y wƺyſtko / co będzieƺ miáł / rozmnoży śię :
Nie podnioſło śię ſerce twoje / y zápomniáłbyś Páná / Bogá twego / który ćię wywiódł z ziemi Egipſkiej z domu niewoli ;
Który ćię przeprowádził przez tę puƺcżą wielką y ſtráƺną / gdzie były węże jádowite / y niedźwiádki ; y przez ziemię ſuchą / gdzie niemáƺ wody / y wywiódł ći wodę ze ſkáły twárdej ;
Który ćię kármił mánną ná puƺcży / której nie ználi ojcowie twoi / áby ćię trápił / y doświádcżáł ćię / ábyć ná oſtátek dobrze ucżynił.
Ani mów w ſercu ſwem : Moc mojá / y śiłá ręki mojej nábáwiłá mię tych dóbr ;
Ale pámiętáj ná Páná / Bogá twego ; bo on dodáwá tobie mocy ku nábywániu bogáctw / áby potwierdził przymierzá ſwego / które poprzyśiągł ojcom twoim / jáko to okázuje dzień dziśiejƺy.
Ale jeźliż cále zápomniƺ Páná / Bogá twego / á pójdzieƺ zá bogámi obcymi / y będzieƺ im ſłużył / y kłániáł śię im / oświádcżám śię przećiwko wám dziś / że koniecżnie zginiećie.
Jáko národy / które Pán wytrácá przed wámi / ták záginiećie dlá tego / iżeśćie poſłuƺnymi nie byli głoſu Páná / Bogá wáƺego.
