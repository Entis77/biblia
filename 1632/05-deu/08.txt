Wƺyſtkie przykázániá / które já dźiś przykázuję tobie / chowáć y cżynić będźiećie / ábyśćie żyli / rozmnożeni byli / y weƺli / á ośiedli tę źiemię / o którą przyśiągł Pán ojcom wáƺym.
Miejże w pámięći wƺyſtkę drogę / którą ćię prowádźił Pán / Bóg twój / przez te cżterdźieśći lát po puƺcży / áby ćię trápił / y doświádcżył / áby wiádomo było / co jeſt w ſercu twojim / jeźlibyś przeſtrzegáł przykázáń jego / cżyli nie.
Przetoż dręcżył ćię / y głodem ći dokucżáł ; potem kármił ćię mánną / którejś nieznáł / áni jej też ználi ojcowie twoi / áby ćię náucżył / iż nie ſámym chlebem żyje cżłowiek / ále tem wƺyſtkiem / co pochodźi z uſt Páńſkich / żyć będźie cżłowiek.
Szátá twojá nie wiotƺáłá ná tobie / y nogá twojá nie nápuchłá przez te cżterdźieśći lát.
Poznájże tedy w ſercu ſwem / że jáko ćwicży cżłowiek ſyná ſwego / ták Pán / Bóg twój / ćwicży ćiebie.
A przeſtrzegáj przykázáń Páná / Bogá twego / ábyś chodźił drogámi jego / y báł śię go.
Abowiem Pán / Bóg twój / prowádźi ćię do źiemi tej dobrej / do źiemi / gdźie ſą potoki wód / źródłá y przepáśći wynikájące po dolinách y po górách ;
Do źiemi obfitej w pƺenicę / y w jęcżmień / y w wino / y w figi / y w jábłká gránátowe ; do źiemi hojnej w oliwę y w miód ;
Do źiemi / w której bez niedoſtátku jeść będźieƺ chleb / á ná żádnej rzecży ſchodźić ći nie będźie ; do źiemi / której kámienie ſą żelázo / á z jej gór miedź kopáć będźieƺ.
Gdy będźieƺ jádł / á nájeƺ śię / będźieƺ chwálił Páná / Bogá twego / zá źiemię wyborną / którą dáł tobie.
Strzeżże śię / byś ſnáć nie zápomniáł Páná / Bogá twego / á nie zániedbáł przykázáń jego / y ſądów jego / y uſtáw jego / które já dźiś rozkázuję tobie ;
By ſnáć / gdy jeść będźieƺ do ſytośći / y piękne domy pobudujeƺ / á mieƺkáć w nich będźieƺ ;
Y gdyć śię wołów y owiec námnoży / ſrebrá tákże y złotá będźieƺ miáł doſyć / y wƺyſtko / co będźieƺ miáł / rozmnoży śię :
Nie podnioſło śię ſerce twoje / y zápomniáłbyś Páná / Bogá twego / który ćię wywiódł z źiemi Egipſkiej z domu niewoli ;
Który ćię przeprowádźił przez tę puƺcżą wielką y ſtráƺną / gdźie były węże jádowite / y niedźwiádki ; y przez źiemię ſuchą / gdźie niemáƺ wody / y wywiódł ći wodę ze ſkáły twárdej ;
Który ćię kármił mánną ná puƺcży / której nie ználi ojcowie twoi / áby ćię trápił / y doświádcżáł ćię / ábyć ná oſtátek dobrze ucżynił.
Ani mów w ſercu ſwem : Moc mojá / y śiłá ręki mojej nábáwiłá mię tych dóbr ;
Ale pámiętáj ná Páná / Bogá twego ; bo on dodáwá tobie mocy ku nábywániu bogáctw / áby potwierdźił przymierzá ſwego / które poprzyśiągł ojcom twoim / jáko to okázuje dźień dźiśiejƺy.
Ale jeźliż cále zápomniƺ Páná / Bogá twego / á pójdźieƺ zá bogámi obcymi / y będźieƺ im ſłużył / y kłániáł śię im / oświádcżám śię przećiwko wám dźiś / że koniecżnie zginiećie.
Jáko národy / które Pán wytrácá przed wámi / ták záginiećie dlá tego / iżeśćie poſłuƺnymi nie byli głoſu Páná / Bogá wáƺego.
