Gdyby pojął kto żonę / á ſtáłby śię jej máłżonkiem / á przydáłoby śię / żeby nie ználázłá łáſki w ocżách jego / przeto / że ználázł przy niej co ſproſnego / tedy jej nápiƺe liſt rozwodny y dá w rękę jej / á puśći ją z domu ſwego.
A gdyby wyƺłá z domu jego / á odƺedłƺy ƺłáby zá drugiego mężá ;
A májąc ją w nienáwiśći on mąż drugi / nápiſáłby jej liſt rozwodny / y dáłby w rękę jej y puśćiłby ją z domu ſwego ; álbo jeźliby też umárł on mąż drugi / który ją był pojął ſobie zá żonę :
Nie będzie mógł mąż jej pierwƺy / który ją był opuśćił / znowu jej pojąć / áby mu byłá żoną / gdyż jeſt ſplugáwioná ; ábowiem obrzydliwośćią to jeſt przed oblicżem Páńſkim. Przetoż nie dopuƺcżáj grzeƺyć ziemi / którą Pán / Bóg twój / dáwá tobie w dziedzictwo.
Gdyby śię kto świeżo ożenił / nie wynijdzie ná wojnę / áni náń włożoná będzie jáká prácá ; wolen będzie w domu ſwym przez cáły rok weſeląc śię z żoną ſwoją / którą pojął.
Nikt nie weźmie w záſtáwie zwierzchniego y ſpodniego kámieniá młyńſkiego ; bo tákowy jákoby duƺę bráł w záſtáwie.
Jeźliby kto był ználezion / coby ukrádł cżłowieká z bráći ſwej / ſynów Izráelſkich / á hándlowáłby nim / y ſprzedáł go / tedy umrze on złodziej / y odejmieƺ złe z pośrodku śiebie.
Strzeż śię zárázy trądu / á przeſtrzegáj pilnie / żebyś cżynił wƺyſtko / cżego wáſ náucżą kápłáni Lewitowie ; jákom im rozkázáł / przeſtrzegáć tego / y cżynić to będziećie.
Pámiętáj co ucżynił Pán / Bóg twój / Máryi w drodze / gdyśćie wyƺli z Egiptu.
Gdy pożycżyƺ cżego bliźniemu twemu / nie wchodźże do domu jego / ábyś wziął co w záſtáwie od niego.
Ale ná dworze zoſtánieƺ / á cżłowiek / któremuś pożycżył / wynieśie do ćiebie záſtáw przed dom.
A jeźliby on cżłowiek był ubogi / nie ukłádzieƺ śię z záſtáwem jego.
Bez omieƺkániá wróćiƺ mu on záſtáw / gdy ſłońce zájdzie / żeby leżáł ná odzieniu ſwem / y błogoſłáwił ći ; á będzieć to ſpráwiedliwośćią przed Pánem / Bogiem twoim.
Nie ucżyniƺ krzywdy nájemnikowi ubogiemu / y potrzebnemu z bráći twojej / álbo z cudzoziemców twoich / którzy ſą w ziemi twej / w brámách twoich.
Tegoż dniá dáƺ mu zápłátę jego przed zájśćiem ſłońcá / ábowiem ubogi jeſt / á z tego żywi duƺę ſwoję ; żeby nie wołáł przećiwko tobie do Páná / á zoſtáłby ná tobie grzech.
Nie umrą ojcowie zá ſyny / á ſynowie nie umrą zá ojce ; káżdy zá grzech ſwój umrze.
Nie wywróćiƺ ſądu przychodniowi / áni śieroćie / áni weźmieƺ w záſtáwie ƺáty wdowy ;
Ale pámiętáj / żeś niewolnikiem był w Egipćie / á iż ćię wybáwił Pán / Bóg twój / ſtámtąd ; dlá tegoż já przykázuję tobie / ábyś to cżynił.
Gdybyś żął zboże twoje ná roli twojej / á zápámiętáłbyś ſnopá ná polu / nie wrácáj śię / ábyś go wziął ; przychodniowi / śieroćie / y wdowie to będzie / ábyć błogoſłáwił Pán / Bóg twój / w káżdej ſpráwie rąk twoich.
Gdy będzieƺ obieráł oliwki twoje / nie oglądájże śię ná káżdą gáłązkę zá ſobą ; przychodniowi / śieroćie / y wdowie to będzie.
Gdy będzieƺ obieráł winnicę twoję / nie zbierájże gron pozoſtáłych zá tobą ; przychodniowi / śieroćie / y wdowie to będzie.
A pámiętáj / iżeś był niewolnikiem w ziemi Egipſkiej ; przetoż já tobie przykázuję / ábyś to cżynił.
