Przeſtrzegáj mieśiącá Abib / ábyś weń obchodził święto przejśćiá Pánu / Bogu twemu / poniewáż mieśiącá Abibá wywiódł ćię Pán / Bóg twój / z Egiptu w nocy.
A będzieƺ ofiárowáł ofiárę świętą przejśćiá Pánu / Bogu twemu / z owiec y z wołów / ná miejſcu / które obierze Pán / áby tám mieƺkáło imię jego.
Nie będzieƺ z nimi jádł kwáƺonego ; przez śiedm dni jeść będzieƺ z nimi przáśniki / chleb utrápieniá / ( boś z kwápieniem wyƺedł z ziemi Egipſkiej ) / ábyś pámiętáł dzień wyjśćiá twego z ziemi Egipſkiej po wƺyſtkie dni żywotá twego.
A nie pokáże śię u ćiebie kwáſ we wƺyſtkich gránicách twoich przez śiedm dni / y nie zoſtánie nic przez noc z mięſá / którebyś ofiárowáł w wiecżór pierwƺego dniá / áż do poránku.
Nie będzieƺ mógł ofiárowáć ofiáry świętá przejśćiá w któremkolwiek mieśćie twojim / z onych / które Pán / Bóg twój / dáwá tobie.
Ale ná miejſcu / które obierze Pán / Bóg twój / áby tám mieƺkáło imię jego / ofiárowáć będzieƺ ofiárę świętą przejśćiá w wiecżór przy zájśćiu ſłońcá / w ten cżáſ / któregoś wyƺedł z Egiptu.
To upiecżeƺ y jeść będzieƺ ná miejſcu / które obierze Pán / Bóg twój ; potem wróćiƺ śię ráno / y pójdzieƺ do námiotów ſwoich.
Przez ƺeść dni będzieƺ jádł przáśniki / ále w dzień śiódmy święto urocżyſte będzie Páná / Bogá twego ; nie będzieƺ cżynił weń żádnej roboty.
Siedm też tygodni odlicżyƺ ſobie ; gdy zápuśćiƺ śierp w zboże twoje / pocżnieƺ licżyć śiedm tygodni.
A potem będzieƺ obchodził święto tygodni Pánu / Bogu twemu ; z doſtátku ręki twej dobrowolnie dáƺ według tego / jáko ćię ubłogoſłáwił Pán / Bóg twój.
A będzieƺ śię weſelił przed Pánem / Bogiem twoim / ty y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y Lewitá / który jeſt w brámách twoich / y przychodzień / y śierotá / y wdowá / którzy będą w pośrodku ćiebie / ná miejſcu / które obierze Pán / Bóg twój / áby tám mieƺkáło imię jego.
A będzieƺ wſpomináł / żeś był niewolnikiem w Egipćie ; przetoż ſtrzedz y cżynić będzieƺ te uſtáwy.
Przytem święto kucżek obchodzić będzieƺ przez śiedm dni / gdy zbierzeƺ pożytki z bojewiſká twego / y z práſy twojej.
Y będzieƺ śię weſelił w święto urocżyſte twoje / ty / y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y Lewitá / y przychodzień / y śierotá / y wdowá / którzy będą w brámách twoich.
Przez śiedm dni będzieƺ obchodził święto urocżyſte Pánu / Bogu twemu / ná miejſcu / które obierze Pán / gdyć błogoſłáwić będzie Pán / Bóg twój / we wƺyſtkich urodzájách twoich / y we wƺelkiej prácy rąk twoich ; á ták będzieƺ weſół.
Trzy kroć do roku ukáże śię káżdy mężcżyzná twój przed Pánem / Bogiem twoim / ná miejſcu / które obierze / w święto przáśników / y w święto tygodni / y w święto kucżek / á nie ukáże śię nikt przed Pánem próżny.
Káżdy ukáże śię z dárem ręki ſwojej według błogoſłáwieńſtwá Páná / Bogá twego / które on dá tobie.
Sędzie y urzędniki poſtánowiƺ ſobie we wƺyſtkich brámách twoich / które Pán / Bóg twój / dá tobie w káżdem pokoleniu twojim / áby ſądzili lud ſądem ſpráwiedliwym.
Nie będzieƺ podwrácáł ſądu / áni będzieƺ miáł względu ná oſobę / y dáru bráć nie będzieƺ ; ábowiem dár záślepiá ocży mądrych / á podwrácá ſłowá ſpráwiedliwych.
Spráwiedliwośći / ſpráwiedliwośći náśládowáć będzieƺ / ábyś żył / y pośiádł ziemię / którą Pán / Bóg twój / dáwá tobie.
Nie ſádź ſobie gáju z żádnego drzewá przy ołtárzu Páná / Bogá twego / który ſobie zbudujeƺ ;
Ani ſobie wyſtáwiáj ſłupá / gdyż to má w nienáwiśći Pán / Bóg twój.
