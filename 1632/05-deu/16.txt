Przeſtrzegáj mieśiącá Abib / ábyś weń obchodźił święto przejśćiá Pánu / Bogu twemu / poniewáż mieśiącá Abibá wywiódł ćię Pán / Bóg twój / z Egiptu w nocy.
A będźieƺ ofiárował ofiárę świętą przejśćiá Pánu / Bogu twemu / z owiec y z wołów / ná miejſcu / które obierze Pán / áby tám mieƺkáło imię jego.
Nie będźieƺ z nimi jádł kwáƺonego ; przez śiedm dni jeść będźieƺ z nimi przáśniki / chleb utrapienia / ( boś z kwápieniem wyƺedł z źiemi Egipſkiej ) / ábyś pámiętał dźień wyjśćiá twego z źiemi Egipſkiej po wƺyſtkie dni żywotá twego.
A nie pokáże śię u ćiebie kwás we wƺyſtkich gránicách twojich przez śiedm dni / y nie zoſtánie nic przez noc z mięſá / którebyś ofiárował w wiecżór pierwƺego dniá / áż do poránku.
Nie będźieƺ mógł ofiárowáć ofiáry świętá przejśćiá w któremkolwiek mieśćie twojim / z onych / które Pán / Bóg twój / dáwá tobie.
Ale ná miejſcu / które obierze Pán / Bóg twój / áby tám mieƺkáło imię jego / ofiárowáć będźieƺ ofiárę świętą przejśćiá w wiecżór przy zájśćiu ſłońcá / w ten cżás / któregoś wyƺedł z Egiptu.
To upiecżeƺ y jeść będźieƺ ná miejſcu / które obierze Pán / Bóg twój ; potem wróćiƺ śię ráno / y pójdźieƺ do námiotów ſwojich.
Przez ƺeść dni będźieƺ jádł przáśniki / ále w dźień śiódmy święto urocżyſte będźie Páná / Bogá twego ; nie będźieƺ cżynił weń żádney roboty.
Siedm też tygodni odlicżyƺ ſobie ; gdy zápuśćiƺ śierp w zboże twoje / pocżnieƺ licżyć śiedm tygodni.
A potem będźieƺ obchodźił święto tygodni Pánu / Bogu twemu ; z doſtátku ręki twej dobrowolnie dáƺ według tego / jáko ćię ubłogoſłáwił Pán / Bóg twój.
A będźieƺ śię weſelił przed Pánem / Bogiem twoim / ty y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y Lewitá / który jeſt w brámách twojich / y przychodźień / y śierotá / y wdowá / którzy będą w pośrzodku ćiebie / ná miejſcu / które obierze Pán / Bóg twój / áby tám mieƺkáło imię jego.
A będźieƺ wſpominał / żeś był niewolnikiem w Egipćie ; przetoż ſtrzedz y cżynić będźieƺ te uſtáwy.
Przytem święto kucżek obchodźić będźieƺ przez śiedm dni / gdy zbierzeƺ pożytki z bojewiſká twego / y z práſy twojey.
Y będźieƺ śię weſelił w święto urocżyſte twoje / ty / y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y Lewitá / y przychodźień / y śierotá / y wdowá / którzy będą w brámách twojich.
Przez śiedm dni będźieƺ obchodźił święto urocżyſte Pánu / Bogu twemu / ná miejſcu / które obierze Pán / gdyć błogoſłáwić będźie Pán / Bóg twój / we wƺyſtkich urodzájách twojich / y we wƺelkiej prácy rąk twojich ; á ták będźieƺ weſół.
Trzy kroć do roku ukáże śię káżdy mężcżyzná twój przed Pánem / Bogiem twoim / ná miejſcu / które obierze / w święto przáśników / y w święto tygodni / y w święto kucżek / á nie ukáże śię nikt przed Pánem próżny.
Káżdy ukáże śię z dárem ręki ſwojey według błogoſłáwieńſtwá Páná / Bogá twego / które on da tobie.
Sędźie y urzędniki poſtánowiƺ ſobie we wƺyſtkich brámách twojich / które Pán / Bóg twój / da tobie w káżdym pokoleniu twojim / áby ſądźili lud ſądem ſpráwiedliwym.
Nie będźieƺ podwrácał ſądu / áni będźieƺ miał względu ná oſobę / y dáru bráć nie będźieƺ ; ábowiem dár záślepiá ocży mądrych / á podwrácá ſłowá ſpráwiedliwych.
Spráwiedliwośći / ſpráwiedliwośći náśládowáć będźieƺ / ábyś żył / y pośiádł źiemię / którą Pán / Bóg twój / dáwá tobie.
Nie ſádź ſobie gáju z żádnego drzewá przy ołtárzu Páná / Bogá twego / który ſobie zbudujeƺ ;
Ani ſobie wyſtáwiáj ſłupá / gdyż to ma w nienáwiśći Pán / Bóg twój.
