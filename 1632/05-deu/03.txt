Potem obróćiwƺy śię ƺliśmy drogą ku Báſán ; y wyćiągnął przećiwko nám Og / król Báſáńſki / ſám / y wƺyſtek lud jego / ku bitwie w Edrej.
Tedy rzekł Pán do mnie : Nie bój śię go ; bom go dáł w ręce twoje / y wƺyſtek lud jego / y ziemię jego / y ucżyniƺ mu / jákoś ucżynił Sehonowi / królowi Amorejſkiemu / który mieƺkáł w Heſebon.
Dáł tedy Pán / Bóg náƺ / w ręce náƺe y Ogá / królá Báſáńſkiego / y wƺyſtek lud jego / y poráziliśmy go / ták że nie zoſtáło po nim nikogo.
Wzięliśmy też wƺyſtkie miáſtá jego ná on cżáſ ; nie było miáſtá / któregobyśmy im nie wzięli / ƺeśćdzieśiąt miáſt / wƺyſtkę kráinę Argob króleſtwá Ogowego w Báſán.
Te wƺyſtkie miáſtá opátrzone były murámi wyſokiemi / brámámi / y záworámi / oprócż miáſtecżek murem nie obtocżonych bárdzo wiele.
Y ſpuſtoƺyliśmy je / jákośmy ucżynili Sehonowi / królowi Heſebońſkiemu / wytráćiwƺy wƺyſtkie miáſtá / męże / niewiáſty y dzieći.
Tylko wƺyſtkie bydłá y łupy z miáſt rozebráliśmy między śię.
Wzięliśmy też ná on cżáſ ziemię z ręki dwóch królów Amorejſkich / którá leży z tej ſtrony Jordánu / od rzeki Arnon áż do góry Hermon.
( Sydońcżycy zowią Hermon Szyryjon / á Amorejcżycy zowią go Sánir. )
Wƺyſtkie miáſtá w równinie / y wƺyſtko Gáláád / y wƺyſtko Báſán áż do Selchá / y Edrej / miáſtá króleſtwá Ogowego w Báſán.
Bo tylko ſám Og / król Báſáńſki pozoſtáł był z olbrzymów ; oto / łoże jego łoże żelázne / ázáż jeƺcże nie jeſt w Rábbát / ſynów Ammonowych? dziewięć łokći długość jego / á cżtery łokćie ƺerokość jego ná łokieć męſki.
Gdyśmy tedy ziemię tę pośiedli ná on cżáſ / od Aroer / które jeſt nád potokiem Arnon / y połowę góry Gáláád / y miáſtá jej dáłem Rubenitom / y Gádytom.
A oſtátek Gáláádu / y wƺyſtko Báſán / króleſtwá Ogowego / dáłem połowie pokoleniá Mánáſeſowego / wƺyſtkę kráinę Argob / y wƺyſtko Báſán / które zowią ziemią olbrzymów.
Jáir / ſyn Mánáſeſów / pośiádł wƺyſtkę kráinę Argob áż do gránicy Jeſſury y Mácháty ; przetoż názwáł ją od imieniá ſwego Báſán Háwot Jáir / áż do dziśiejƺego dniá.
Máchirowi záś dáłem Gáláád.
A Rubenitom y Gádytom dáłem kráinę od Gáláádu áż do potoku Arnon / pół potoku z pogránicżem / áż do potoku Jábok / gránicy ſynów Ammonowych ;
Przytem równinę y Jordán z pogránicżem od Cyneret áż do morzá puſtego / które jeſt morze ſłone / pod górą Fázgá ná wſchód ſłońcá.
Y rozkázáłem wám ná on cżáſ / mówiąc : Pán / Bóg wáƺ / dáł wám tę ziemię / ábyśćie ją dziedzicżnie pośiedli / á ták zbrojno pójdziećie przed bráćią wáƺą / ſyny Izráelſkimi / wƺyſcy mężowie duży.
Tylko żony wáƺe / y dziátki wáƺe / y bydłá wáƺe / ( gdyż wiem / że dobytku śiłá máćie / ) zoſtáną w mieśćiech wáƺych / którem wám dáł.
Aż dá odpocżynek Pán bráći wáƺej / jáko y wám / że y oni pośiędą ziemię onę / którą Pán / Bóg wáƺ / dáwá im zá Jordánem : tedy śię wróći káżdy do ośiádłośći ſwojej / którą wám dáł.
Jozuemu też przykázáłem ná on cżáſ / mówiąc : Ocży twoje widziáły wƺyſtko / co ucżynił Pán / Bóg wáƺ / onym dwom królom ; toż ucżyni Pán / wƺyſtkim króleſtwom / do których ty pójdzieƺ.
Nie bójćież śię ich / poniewáż Pán / Bóg wáƺ / jeſt / który wálcży zá wáſ.
Y prośiłem Páná ná on cżáſ / mówiąc :
Pánie Boże / tyś pocżął okázywáć ſłudze twemu wielkość twoję / y rękę twoję możną ; bo któż jeſt Bogiem ná niebie / álbo ná ziemi / któryby cżynić mógł według ſpráw twoich / y według możnośći twoich?
Niech przejdę proƺę / ábym oglądáł tę wyborną ziemię / którá jeſt zá Jordánem / górę onę wyborną y z Libánem.
Ale śię rozgniewáł Pán ná mię dlá wáſ / y nie wyſłucháł mię / y rzekł Pán do mnie : Doſyć máƺ / nie mówże już więcej do mnie o to.
Wſtąp ná wierzch góry Fázgá / á podnieś ocży ſwe ku záchodowi / y ku północy / y ku południowi / y ku wſchodowi / á przypátrz śię ocżymá ſwemi ; bo nie przejdzieƺ przez ten Jordán ;
Ale porucż to Jozuemu / y umocnij go / á potwierdź go ; ábowiem on pójdzie przed tym ludem / á on podá im w dziedzictwo ziemię / którą oglądáƺ.
A ták mieƺkáliśmy w dolinie przećiw Betfegor.
