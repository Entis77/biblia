Potem obróćiwƺy śię ƺliśmy drogą ku Báſán ; y wyćiągnął przećiwko nam Og / król Báſáńſki / ſám / y wƺyſtek lud jego / ku bitwie w Edrej.
Tedy rzekł Pán do mnie : Nie bój śię go ; bom go dał w ręce twoje / y wƺyſtek lud jego / y źiemię jego / y ucżyniƺ mu / jákoś ucżynił Sehonowi / królowi Amorejſkiemu / który mieƺkał w Heſebon.
Dał tedy Pán / Bóg náƺ / w ręce náƺe y Ogá / królá Báſáńſkiego / y wƺyſtek lud jego / y poráźiliśmy go / ták że nie zoſtáło po nim nikogo.
Wźięliśmy też wƺyſtkie miáſtá jego ná on cżás ; nie było miáſtá / któregobyſmy im nie wźięli / ƺeśćdźieśiąt miaſt / wƺyſtkę kráinę Argob króleſtwá Ogowego w Báſán.
Te wƺyſtkie miáſtá opátrzone były murámi wyſokiemi / brámámi / y záworámi / oprócż miáſtecżek murem nie obtocżonych bárzo wiele.
Y ſpuſtoƺyliśmy je / jákośmy ucżynili Sehonowi / królowi Heſebońſkiemu / wytráćiwƺy wƺyſtkie miáſtá / męże / niewiáſty y dźieći.
Tylko wƺyſtkie bydłá y łupy z miaſt rozebráliśmy miedzy śię.
Wźięliśmy też ná on cżás źiemię z ręki dwu królów Amorejſkich / która leży z tey ſtrony Jordánu / od rzeki Arnon áż do góry Hermon.
( Sydońcżycy zowią Hermon Szyryjon / á Amorejcżycy zowią go Sánir. )
Wƺyſtkie miáſtá w równinie / y wƺyſtko Gáláád / y wƺyſtko Báſán áż do Selchá / y Edrej / miáſtá króleſtwá Ogowego w Báſán.
Bo tylko ſám Og / król Báſáńſki pozoſtał był z olbrzymów ; oto / łoże jego łoże żelázne / ázaż jeƺcże nie jeſt w Rábbát / ſynów Ammonowych? dźiewięć łokći długość jego / á cżtery łokćie ƺerokość jego ná łokieć męſki.
Gdyśmy tedy źiemię tę pośiedli ná on cżás / od Aroer / które jeſt nád potokiem Arnon / y połowę góry Gáláád / y miáſtá jey dáłem Rubenitom / y Gádytom.
A oſtátek Gáláádu / y wƺyſtko Báſán / króleſtwá Ogowego / dáłem połowie pokolenia Mánáſeſowego / wƺyſtkę kráinę Argob / y wƺyſtko Báſán / które zowią źiemią olbrzymów.
Jáir / ſyn Mánáſeſów / pośiádł wƺyſtkę kráinę Argob áż do gránicy Jeſſury y Mácháty ; przetoż názwał ją od imieniá ſwego Báſán Háwot Jáir / áż do dźiśiejƺego dniá.
Máchirowi záś dáłem Gáláád.
A Rubenitom y Gádytom dáłem kráinę od Gáláádu áż do potoku Arnon / pół potoku z pogránicżem / áż do potoku Jábok / gránicy ſynów Ammonowych ;
Przytem równinę y Jordán z pogránicżem od Cyneret áż do morzá puſtego / które jeſt morze ſłone / pod górą Fázgá ná wſchód ſłońcá.
Y rozkazałem wam ná on cżás / mówiąc : Pán / Bóg wáƺ / dał wam tę źiemię / ábyśćie ją dźiedźicżnie pośiedli / á ták zbrojno pójdźiećie przed bráćią wáƺą / ſyny Izráelſkimi / wƺyſcy mężowie duży.
Tylko żony wáƺe / y dźiátki wáƺe / y bydłá wáƺe / ( gdyż wiem / że dobytku śiłá máćie / ) zoſtáną w mieśćiech wáƺych / którem wam dał.
Aż da odpocżynek Pán bráći wáƺej / jáko y wam / że y oni pośiędą źiemię onę / którą Pán / Bóg wáƺ / dáwá im zá Jordánem : tedy śię wróći káżdy do ośiádłośći ſwojey / którą wam dał.
Jozuemu też przykázáłem ná on cżás / mówiąc : Ocży twoje widźiáły wƺyſtko / co ucżynił Pán / Bóg wáƺ / onym dwom królom ; toż ucżyni Pán / wƺyſtkim króleſtwom / do których ty pójdźieƺ.
Nie bójćież śię ich / poniewáż Pán / Bóg wáƺ / jeſt / który wálcży zá was.
Y prośiłem Páná ná on cżás / mówiąc :
Pánie Boże / tyś pocżął okázywáć ſłudze twemu wielkość twoję / y rękę twoję możną ; bo któż jeſt Bogiem ná niebie / álbo ná źiemi / któryby cżynić mógł według ſpráw twojich / y według możnośći twojich?
Niech przejdę proƺę / ábym oglądał tę wyborną źiemię / która jeſt zá Jordánem / górę onę wyborną y z Libánem.
Ale śię rozgniewał Pán ná mię dla was / y nie wyſłuchał mię / y rzekł Pán do mnie : Doſyć máƺ / nie mówże już więcey do mnie o to.
Wſtąp ná wierzch góry Fázgá / á podnieś ocży ſwe ku záchodowi / y ku północy / y ku południowi / y ku wſchodowi / á przypátrz śię ocżymá ſwemi ; bo nie przejdźieƺ przez ten Jordán ;
Ale porucż to Jozuemu / y umocnij go / á potwierdź go ; ábowiem on pójdźie przed tym ludem / á on podá im w dźiedźictwo źiemię / którą oglądáƺ.
A ták mieƺkáliśmy w dolinie przećiw Betfegor.
