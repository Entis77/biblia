Potem obróćiwƺy śię / poƺliśmy ná puƺcżą / drogą ku morzu cżerwonemu / jáko mówił Pán do mnie / y krążyliśmy około góry Seir przez wiele dni.
Tedy rzekł Pán do mnie / mówiąc :
Doſyćieśćie już krążyli około tej góry / obróććież śię ku północy ;
A ludowi rozkáż mówiąc : Wy wnet pójdziećie przez gránice bráći wáƺej / ſynów Ezáwowych / którzy mieƺkáją w Seir / y będą śię wáſ báli ; ále śię y wy pilnie ſtrzeżćie.
Nie dráźnijćież ich ; ábowiem nie dám wám ziemi ich / áni ná ſtopę nogi / gdyżem w ośiádłość dáł Ezáwowi górę Seir.
Żywność będziećie kupowáli u nich zá pieniądze / y jeść będziećie ; tákże y wodę od nich kupowáć będziećie zá pieniądze / y pić będziećie.
Abowiem ći Pán / Bóg twój / błogoſłáwił we wƺelkiej ſpráwie rąk twoich / y wiedziáł drogę twoję ná tej puƺcży wielkiej ; już przez cżterdzieśći lát Pán / Bóg twój / był z tobą / nie ſchodziłoć ná nicżem.
Poƺliśmy tedy od bráći náƺej / ſynów Ezáwowych / mieƺkájących w Seir / drogą równą od Elát / y od Aſyjongáber / á náwróćiwƺy śię událiśmy śię drogą ku puƺcży Moábſkiej.
Y rzekł Pán do mnie : Nie náćieráj ná Moábcżyki / áni podnoś wojny przećiwko nim ; boć nie dám ziemi ich w ośiádłość / gdyżem ſynom Lotowym dáł Ar w dziedzictwo.
( Emitowie pierwej mieƺkáli w niej / lud potężny y wielki / y wyſokiego wzroſtu / jáko Enákimowie ;
Które też zá olbrzymy miáno / jáko Enákimy / á Moábcżykowie zwáli je Emim.
Tákże w Seir mieƺkáli Horejcżycy przedtem / które ſynowie Ezáwowi wygnáli / y wygłádzili je przed ſobą / á mieƺkáli ná miejſcu ich / jáko ucżynił Izráel w ziemi ośiádłośći ſwojej / którą im dáł Pán. )
Wſtáńćież tedy / á przepráwćie śię przez potok Záred ; y przepráwiliśmy śię przez potok Záred.
A cżáſu / któregośmy chodzili od Kádeſ Bárne / áżeśmy śię przepráwili przez potok Záred / było trzydzieśći y ośm lát / áż wyginął wƺyſtek on náród mężów wálecżnych z obozu / jáko im był przyśiągł Pán.
Abowiem ręká Páńſká byłá przećiwko nim ná wytrácenie ich z obozu / áż ich wygubiłá.
Y ſtáło śię / gdy wyginęli wƺyſcy mężowie wálecżni / y pomárli z pośrodku ludu /
Że rzekł Pán do mnie / mówiąc :
Ty dziś przejdzieƺ gránicę Moábſką Ar /
Y przyjdzieƺ bliſko ku ſynom Ammonowym ; nie náćierájże ná nie / áni podnoś wojny przećiwko nim ; boć nie dám ziemi ſynów Ammonowych w ośiádłość / poniewáż ſynom Lotowym dáłem ją w dziedzictwo.
( Tę też ziemię zá ziemię olbrzymów miáno ; ábowiem olbrzymowie mieƺkáli w niej przedtem / które Ammonitowie názywáli Zomzomim ;
Lud potężny / y wielki / y wyſokiego wzroſtu / jáko Enákimowie ; ále wygubił je Pán przed nimi / á Ammonitowie opánowáli je / y mieƺkáli ná miejſcu ich ;
Jáko ucżynił ſynom Ezáwowym mieƺkájącym w Seir / dlá których wytráćił Horejcżyki przed oblicżem ich ; y wygnáli je / y mieƺkáli ná miejſcách ich áż po dziś dzień.
Hewejcżyki tákże / którzy mieƺkáli w Aſerym áż do Gázy / Káftorytowie / którzy wyƺli z Káftor / wytráćili je / y mieƺkáli ná miejſcu ich. )
Wſtáwƺy tedy idźćie / á przepráwćie śię przez potok Arnon ; oto / Já dáwám w ręce twoje Sehoná / królá Heſebon Amorejcżyká / y ziemię jego / pocżnijże ją pośiádáć / á podnieś przećiwko niemu wojnę.
Dziśiáj pocżnę puƺcżáć ſtrách y bojáźń twoję ná ludzie / którzy ſą pod wƺyſtkiem niebem / że gdy uſłyƺą wieść o tobie / drżeć y lękáć śię będą przed tobą.
Tedym poſłáł poſły z puƺcży Kádemot do Sehoná / królá Heſebońſkiego / z poſelſtwem ſpokojnem / mówiąc :
Niech przejdę przez ziemię twoję ; drogą poſpolitą pójdę ; nie uchylę śię áni ná práwo áni ná lewo.
Żywnośći zá pieniądze ſprzedáƺ mi / ábym jádł ; wody tákże zá pieniądze dáƺ mi / ábym pił : tylko przejdę nogámi memi ;
Jáko mi ucżynili ſynowie Ezáwowi / którzy mieƺkáją w Seir / y Moábcżycy / którzy mieƺkáją w Ar / áżbyśmy śię przepráwili zá Jordán / do ziemi / którą Pán / Bóg náƺ / dáwá nám.
Ale niechćiáł Sehon / król Heſebońſki / pozwolić nám przejśćiá przez ziemię ſwoję : bo był zátwárdził Pán / Bóg twój / duchá jego / y ſtwierdził ſerce jego / áby go podáł w ręce twoje / jáko to dziś widziƺ.
Tedy rzekł Pán do mnie : Otom ći już pocżął podáwáć w moc Sehoná / y ziemię jego : pocżnijże ją pośiádáć / ábyś odziedzicżył ziemię jego.
Ruƺył śię tedy Sehon przećiwko nám / ſám y wƺyſtek lud jego / chcąc z námi zwieść bitwę w Jázá ;
Ale go nám podáł Pán / Bóg náƺ / w moc / y poráziliśmy go / y ſyny jego y wƺyſtek lud jego.
Y wzięliśmy wƺyſtkie miáſtá jego ná on cżáſ / y wniwecżeśmy obróćili wƺyſtkie miáſtá / mężcżyzny y niewiáſty / y dzieći / nie zoſtáwiwƺy z nich nikogo.
Tylkośmy bydło pobráli ſobie / y korzyśći z miáſt / którycheśmy dobyli.
Od Aroer / które leży nád brzegiem potoku Arnon / y od miáſtá / które jeſt w dolinie áż do Gáláád / nie było miáſtá / któreby śię nám nie podáło ; wƺyſtkie podáł nám Pán / Bóg náƺ.
Tylkoś do ziemi ſynów Ammonowych nie dochodził áni do żádnego miejſcá przyległego potokowi Jábok / áni do miáſt ná górách / áni do żádnych miejſc / których zákázáł Pán / Bóg náƺ.
