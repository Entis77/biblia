Teraz tedy / o Izráelu / ſłucháj uſtáw / y ſądów / których ja was ucżę / ábyśćie je cżyniąc żyli / á wƺedƺy pośiedli źiemię / którą Pán / Bóg ojców wáƺych / dáwá wam.
Nie przydáćie do ſłowá / które ja wam rozkázuję / áni ujmiećie z niego / ábyśćie ſtrzegli przykázáń Páná / Bogá wáƺego / które ja wam rozkázuję.
Ocży wáƺe widźiáły / co ucżynił Pán dla Báál fegorá / jáko káżdego mężá / który śię udáwał zá Báál fegorem / wytráćił Pán / Bóg twój / z pośrzodku was.
A wy / którzyśćie trwáli przy Pánu / Bogu wáƺym / żyjećie wƺyſcy áż do dniá dźiśiejƺego.
Pátrzćież / żem was náucżał uſtáw y ſądów / jáko mi rozkázał Pán / Bóg mój / ábyśćie ták cżynili w źiemi tey / do którey wchodźićie / ábyśćie ją dźiedźicżnie pośiedli.
Przeſtrzegájćież ich tedy / á cżyńćie je ; to bowiem jeſt mądrość wáƺá / y rozum wáƺ przed ocżymá národów ; którzy uſłyƺáwƺy o tych wƺyſtkich uſtáwách / rzeką : Tylko ten lud mądry / y rozumny / y náród wielki jeſt.
Abowiem któryż náród ták wielki jeſt / coby mu byli bogowie ták bliſcy / jáko Pán / Bóg náƺ / we wƺyſtkiem / ilekroć go wzywámy?
Albo któryż náród ták wielki / któryby miał uſtáwy y ſądy ták ſpráwiedliwe / jáko wƺyſtek ten zakon / który ja przedkłádám wam dźiś?
A wƺákże miej śię ná piecży / á ſtrzeż duƺy twej pilnie / byś ſnadź nie zápomniał ná te rzecży / które widźiáły ocży twoje y áby ſnadź nie odſtąpiły od ſercá twego po wƺyſtkie dni żywotá twego ; y oznájmiƺ je ſynom twym / y ſynom ſynów twojich.
Nie zápomináj / żeś onego dniá ſtał przed oblicżnośćią Páná / Bogá twego / ná Horeb / gdy mówił Pán do mnie : Zgromádź mi lud / ábym im powiedźiał ſłowá moje / których śię ucżyć będą / áby śię mnie báli po wƺyſtkie dni / pokąd żyć będą ná źiemi / y ſynów ſwojich náucżájąc tego.
Tedyśćie przyſtąpili / á ſtánęliśćie pod górą / á oná górá páłáłá ogniem áż do ſámego niebá / á byłá ná niej ćiemność / obłok y mgłá.
Y mówił Pán do was z pośród ogniá ; głos ſłów ſłyƺeliśćie / áleśćie podobieńſtwá żádnego nie widźieli oprócż głoſu.
Y oznájmił wam przymierze ſwoje / które rozkázał wam / ábyśćie je cżynili / to jeſt dźieśięć ſłów / które nápiſał ná dwu táblicách kámiennych.
Mnie też rozkázał Pán ná on cżás / ábym was náucżał uſtáw y ſądów / ábyśćie je cżynili w źiemi / do którey idźiećie / ábyśćie ją dźiedźicżnie pośiedli.
A ták ſtrzeżćie pilnie duƺ wáƺych / ( gdyżeśćie nie widźieli żádnego podobieńſtwá / dniá / którego mówił Pán do was ná Horebie z pośrzodku ogniá / )
Abyśćie śię ſnadź nie popſowáli / y nie cżynili ſobie rytego báłwáná / álbo obrázu wƺelkiego podobieńſtwá / kƺtáłtu mężcżyzny álbo niewiáſty ;
Albo kƺtáłtu jákiego zwierzęćiá / które jeſt ná źiemi / kƺtáłtu wƺelkiego ptáká ſkrzydláſtego / który látá po powietrzu ;
Kƺtáłtu wƺyſtkiego tego / co śię płázá po źiemi / kƺtáłtu wƺelkiej ryby / która jeſt w wodźie pod źiemią.
Ani podnoś ocżu ſwych ku niebu / żebyś widząc ſłońce y mieśiąc / y gwiázdy ze wƺyſtkiemi záſtępy niebieſkiemi / nie dał śię uwieść / ábyś śię im kłániał / y ſłużyłbyś im / poniewáż je oddał Pán / Bóg twój / ná poſługę wƺyſtkim ludźiom pod wƺyſtkiem niebem.
Aleć was wźiął Pán / y wywiódł was jáko z żeláznego piecá / z Egiptu / ábyśćie byli jego ludem dźiedźicżnym / jáko dźiś jeſteśćie.
A Pán rozgniewał śię ná mię dla was / á przyśiągł / że nie przejdę zá Jordán / áni wnijdę do oney wyborney źiemi / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo.
Bo ja umrę w tey źiemi / ja nie przejdę zá Jordán ; ále wy przejdźiećie / y dźiedźicżnie ośiądźiećie tę dobrą źiemię.
Strzeżćież śię / byśćie ſnadź nie zápomnieli przymierza Páná / Bogá wáƺego / które ucżynił z wámi / á nie cżynili ſobie rytego obrázu ná kƺtáłt jákiejkolwiek rzecży / jákoć rozkázał Pán / Bóg twój.
Abowiem Pán / Bóg twój / jeſt ogień tráwiący / Bóg záwiſny.
Gdy ſpłodźiƺ ſyny y wnuki / á zſtárzejećie śię w źiemi oney / jeſlibyśćie śię popſowáli / á cżynilibyśćie ſobie ryte báłwány ná kƺtáłt jákiejkolwiek rzecży / y ucżynilibyśćie co złego przed ocżymá Páná / Bogá wáƺego / dráźniąc go :
Biorę ná świádectwo przećiwko wam dźiś niebo y źiemię / iż pewnie w rychle wyginiećie z źiemi / do którey pójdźiećie przez Jordán / ábyśćie dźiedźicżnie ją pośiedli ; nie przedłużyćie dni wáƺych w niej / bo pewnie wytráceni będźiećie.
Y rozproƺy was Pán miedzy národy / á zoſtánie was máło miedzy národy / do których záprowádźi was Pán.
Támże będźiećie ſłużyli bogom / roboćie rąk ludzkich / drewnu y kámieniowi / które áni widzą / áni ſłyƺą / áni wącháją.
A wƺákże y tám jeſli ƺukáć będźieƺ Páná / Bogá twego / tedy znájdźieƺ / będźieƺli go ƺukał cáłem ſercem twojim / y cáłą duƺą ſwoją.
Gdy ućiſk przydźie ná ćię / á nájdą ćię te wƺyſtkie rzecży w oſtátnie dni / tedy / náwróćiƺli śię do Páná / Bogá twego / á poſłuƺny będźieƺ głoſowi jego /
( Abowiem Bóg miłośierny Pán / Bóg twój jeſt ) nie opuśći ćię / áni ćię ſkáźi / áni zápomni ná przymierze ojców twojich / które im przyśiągł.
Pytáj śię teraz ná dni ſtárodáwne / które były przed tobą / ode dniá / którego ſtworzył Bóg cżłowieká ná źiemi ; á od jednego końcá niebá áż do drugiego końcá niebá ſtáłáli śię kiedy rzecż podobná tey ták wielkiej? álbo ſłuchánoli kiedy co tákowego?
Słyƺáłli kiedy który náród głos Bogá mówiącego z pośrzodku ogniá / jákoś ty ſłyƺał / á żyw zoſtał?
Albo kuśiłli śię który bóg przyść á wźiąć ſobie náród z pośrzodku innego národu przez pokuſy / przez znáki / y przez cudá / y przez wojny / y przez mocną rękę / y przez rámię wyćiągnione / y w ſtráchách wielkich / według wƺyſtkiego / jáko ucżynił dla was Pán / Bóg wáƺ / w Egipćie przed ocżymá twemi?
Tobie to ukázáno / ábyś wiedźiał / iż Pán jeſt Bóg / á niemáƺ innego oprócż niego.
Dał ći z niebá ſłyƺeć głos ſwój / áby ćię wyćwicżył / á ná źiemi ukázał ći ogień ſwój wielki / á ſłowá jego ſłyƺáłeś z pośrzodku ogniá.
A iż umiłował ojce twoje / obrał náśienie ich po nich / y wywiódł ćię przed ſobą mocą ſwoją wielką z Egiptu /
Aby wygnał národy wielkie y możniejƺe nád ćię przed twárzą twoją / y wprowádźił ćię / á dał ći źiemię ich w dźiedźictwo / jáko to dźiś widźiƺ.
Wiedzże tedy dźiś / á wſpomnij ná to w ſercu twojim / iż Pán jeſt Bogiem ná niebie wyſoko / y ná źiemi niſko niemáƺ innego.
A ták przeſtrzegáj uſtáw jego / y przykázáń jego / które ja dźiś rozkázuję tobie / ábyć śię dobrze wodźiło / y ſynom twoim po tobie ; ábyś też przedłużył dni ná źiemi / którą Pán / Bóg twój / da tobie / po wƺyſtkie dni.
Tedy oddźielił Mojzeƺ trzy miáſtá z tey ſtrony Jordánu ná wſchód ſłońcá.
Aby tám ućiekał mężobójcá / któryby zábił bliźniego ſwego z nieobácżeniá / nie májąc go w nienáwiśći przedtem / áby ućiekƺy do jednego z tych miaſt / żyw zoſtał ;
Beſer ná puƺcży / w równinie w źiemi Rubenitów / y Rámot w Gáláád miedzy Gádyty / y Golám w Bázán miedzy Mánáſſytámi.
Tenći jeſt zakon / który przedłożył Mojzeƺ ſynom Izráelſkim.
Te ſą świádectwá / y uſtáwy / y ſądy / które mówił Mojzeƺ do ſynów Izráelſkich / gdy wyƺli z Egiptu ;
Z tey ſtrony Jordánu / w dolinie przećiw Betfegor / w źiemi Sehoná / królá Amorejſkiego / który mieƺkał w Heſebon / którego poráźił Mojzeƺ / y ſynowie Izráelſcy / gdy ƺli z Egiptu ;
Y ośiedli źiemię jego / y źiemię Ogá / królá Báſáńſkiego / dwu królów Amorejſkich / którzy byli z tey ſtrony Jordánu ná wſchód ſłońcá ;
Od Aroer / które jeſt nád brzegiem potoku Arnon / y áż do góry Syon / która jeſt Hermon ;
Y wƺyſtkie polá nád Jordánem ná wſchód ſłońcá / y áż do morzá puſtego / pod górą Fázgá.
