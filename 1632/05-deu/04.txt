Teráz tedy / o Izráelu / ſłucháj uſtáw / y ſądów / których já wáſ ucżę / ábyśćie je cżyniąc żyli / á wƺedłƺy pośiedli źiemię / którą Pán / Bóg ojców wáƺych / dáwá wám.
Nie przydáćie do ſłowá / które já wám rozkázuję / áni ujmiećie z niego / ábyśćie ſtrzegli przykázáń Páná / Bogá wáƺego / które já wám rozkázuję.
Ocży wáƺe widźiáły / co ucżynił Pán dlá Báál fegorá / jáko káżdego mężá / który śię udáwáł zá Báál fegorem / wytráćił Pán / Bóg twój / z pośrodku wáſ.
A wy / którzyśćie trwáli przy Pánu / Bogu wáƺym / żyjećie wƺyſcy áż do dniá dźiśiejƺego.
Pátrzćież / żem wáſ náucżáł uſtáw y ſądów / jáko mi rozkázáł Pán / Bóg mój / ábyśćie ták cżynili w źiemi tej / do której wchodźićie / ábyśćie ją dźiedźicżnie pośiedli.
Przeſtrzegájćież ich tedy / á cżyńćie je ; to bowiem jeſt mądrość wáƺá / y rozum wáƺ przed ocżymá národów ; którzy uſłyƺáwƺy o tych wƺyſtkich uſtáwách / rzeką : Tylko ten lud mądry / y rozumny / y náród wielki jeſt.
Abowiem któryż náród ták wielki jeſt / coby mu byli bogowie ták bliſcy / jáko Pán / Bóg náƺ / we wƺyſtkiem / ilekroć go wzywámy?
Albo któryż náród ták wielki / któryby miáł uſtáwy y ſądy ták ſpráwiedliwe / jáko wƺyſtek ten zákon / który já przedkłádám wám dźiś?
A wƺákże miej śię ná piecży / á ſtrzeż duƺy twej pilnie / byś ſnáć nie zápomniáł ná te rzecży / które widźiáły ocży twoje y áby ſnáć nie odſtąpiły od ſercá twego po wƺyſtkie dni żywotá twego ; y oznájmiƺ je ſynom twym / y ſynom ſynów twoich.
Nie zápomináj / żeś onego dniá ſtáł przed oblicżnośćią Páná / Bogá twego / ná Horeb / gdy mówił Pán do mnie : Zgromádź mi lud / ábym im powiedźiáł ſłowá moje / których śię ucżyć będą / áby śię mnie báli po wƺyſtkie dni / pokąd żyć będą ná źiemi / y ſynów ſwoich náucżájąc tego.
Tedyśćie przyſtąpili / á ſtánęliśćie pod górą / á oná górá páłáłá ogniem áż do ſámego niebá / á byłá ná niej ćiemność / obłok y mgłá.
Y mówił Pán do wáſ z pośród ogniá ; głoſ ſłów ſłyƺeliśćie / áleśćie podobieńſtwá żádnego nie widźieli oprócż głoſu.
Y oznájmił wám przymierze ſwoje / które rozkázáł wám / ábyśćie je cżynili / to jeſt dźieśięć ſłów / które nápiſáł ná dwóch táblicách kámiennych.
Mnie też rozkázáł Pán ná on cżáſ / ábym wáſ náucżáł uſtáw y ſądów / ábyśćie je cżynili w źiemi / do której idźiećie / ábyśćie ją dźiedźicżnie pośiedli.
A ták ſtrzeżćie pilnie duƺ wáƺych / ( gdyżeśćie nie widźieli żádnego podobieńſtwá / dniá / którego mówił Pán do wáſ ná Horebie z pośrodku ogniá / )
Abyśćie śię ſnáć nie popſowáli / y nie cżynili ſobie rytego báłwáná / álbo obrázu wƺelkiego podobieńſtwá / kƺtáłtu mężcżyzny álbo niewiáſty ;
Albo kƺtáłtu jákiego zwierzęćiá / które jeſt ná źiemi / kƺtáłtu wƺelkiego ptáká ſkrzydláſtego / który látá po powietrzu ;
Kƺtáłtu wƺyſtkiego tego / co śię płázá po źiemi / kƺtáłtu wƺelkiej ryby / którá jeſt w wodźie pod źiemią.
Ani podnoś ocżu ſwych ku niebu / żebyś widząc ſłońce y mieśiąc / y gwiázdy ze wƺyſtkiemi záſtępy niebieſkiemi / nie dáł śię uwieść / ábyś śię im kłániáł / y ſłużyłbyś im / poniewáż je oddáł Pán / Bóg twój / ná poſługę wƺyſtkim ludźiom pod wƺyſtkiem niebem.
Aleć wáſ wźiął Pán / y wywiódł wáſ jáko z żeláznego piecá / z Egiptu / ábyśćie byli jego ludem dźiedźicżnym / jáko dźiś jeſteśćie.
A Pán rozgniewáł śię ná mię dlá wáſ / á przyśiągł / że nie przejdę zá Jordán / áni wnijdę do onej wybornej źiemi / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo.
Bo já umrę w tej źiemi / já nie przejdę zá Jordán ; ále wy przejdźiećie / y dźiedźicżnie ośiądźiećie tę dobrą źiemię.
Strzeżćież śię / byśćie ſnáć nie zápomnieli przymierzá Páná / Bogá wáƺego / które ucżynił z wámi / á nie cżynili ſobie rytego obrázu ná kƺtáłt jákiejkolwiek rzecży / jákoć rozkázáł Pán / Bóg twój.
Abowiem Pán / Bóg twój / jeſt ogień tráwiący / Bóg záwiſny.
Gdy ſpłodźiƺ ſyny y wnuki / á zſtárzejećie śię w źiemi onej / jeźlibyśćie śię popſowáli / á cżynilibyśćie ſobie ryte báłwány ná kƺtáłt jákiejkolwiek rzecży / y ucżynilibyśćie co złego przed ocżymá Páná / Bogá wáƺego / dráźniąc go :
Biorę ná świádectwo przećiwko wám dźiś niebo y źiemię / iż pewnie w rychle wyginiećie z źiemi / do której pójdźiećie przez Jordán / ábyśćie dźiedźicżnie ją pośiedli ; nie przedłużyćie dni wáƺych w niej / bo pewnie wytráceni będźiećie.
Y rozproƺy wáſ Pán między národy / á zoſtánie wáſ máło między národy / do których záprowádźi wáſ Pán.
Támże będźiećie ſłużyli bogom / roboćie rąk ludzkich / drewnu y kámieniowi / które áni widzą / áni ſłyƺą / áni wącháją.
A wƺákże y tám jeźli ƺukáć będźieƺ Páná / Bogá twego / tedy znájdźieƺ / będźieƺli go ƺukáł cáłem ſercem twojim / y cáłą duƺą ſwoją.
Gdy ućiſk przyjdźie ná ćię / á nájdą ćię te wƺyſtkie rzecży w oſtátnie dni / tedy / náwróćiƺli śię do Páná / Bogá twego / á poſłuƺny będźieƺ głoſowi jego /
( Abowiem Bóg miłośierny Pán / Bóg twój jeſt ) nie opuśći ćię / áni ćię ſkáźi / áni zápomni ná przymierze ojców twoich / które im przyśiągł.
Pytáj śię teráz ná dni ſtárodáwne / które były przed tobą / ode dniá / którego ſtworzył Bóg cżłowieká ná źiemi ; á od jednego końcá niebá áż do drugiego końcá niebá ſtáłáli śię kiedy rzecż podobná tej ták wielkiej? álbo ſłuchánoli kiedy co tákowego?
Słyƺáłli kiedy który náród głoſ Bogá mówiącego z pośrodku ogniá / jákoś ty ſłyƺáł / á żyw zoſtáł?
Albo kuśiłli śię który bóg przyjść á wźiąć ſobie náród z pośrodku innego národu przez pokuſy / przez znáki / y przez cudá / y przez wojny / y przez mocną rękę / y przez rámię wyćiągnione / y w ſtráchách wielkich / według wƺyſtkiego / jáko ucżynił dlá wáſ Pán / Bóg wáƺ / w Egipćie przed ocżymá twemi?
Tobie to ukázáno / ábyś wiedźiáł / iż Pán jeſt Bóg / á niemáƺ innego oprócż niego.
Dáł ći z niebá ſłyƺeć głoſ ſwój / áby ćię wyćwicżył / á ná źiemi ukázáł ći ogień ſwój wielki / á ſłowá jego ſłyƺáłeś z pośrodku ogniá.
A iż umiłowáł ojce twoje / obráł náśienie ich po nich / y wywiódł ćię przed ſobą mocą ſwoją wielką z Egiptu /
Aby wygnáł národy wielkie y możniejƺe nád ćię przed twárzą twoją / y wprowádźił ćię / á dáł ći źiemię ich w dźiedźictwo / jáko to dźiś widźiƺ.
Wiedzże tedy dźiś / á wſpomnij ná to w ſercu twojim / iż Pán jeſt Bogiem ná niebie wyſoko / y ná źiemi niſko niemáƺ innego.
A ták przeſtrzegáj uſtáw jego / y przykázáń jego / które já dźiś rozkázuję tobie / ábyć śię dobrze wodźiło / y ſynom twoim po tobie ; ábyś też przedłużył dni ná źiemi / którą Pán / Bóg twój / dá tobie / po wƺyſtkie dni.
Tedy oddźielił Mojżeƺ trzy miáſtá z tej ſtrony Jordánu ná wſchód ſłońcá.
Aby tám ućiekáł mężobójcá / któryby zábił bliźniego ſwego z nieobácżeniá / nie májąc go w nienáwiśći przedtem / áby ućiekłƺy do jednego z tych miáſt / żyw zoſtáł ;
Beſer ná puƺcży / w równinie w źiemi Rubenitów / y Rámot w Gáláád między Gádyty / y Golám w Bázán między Mánáſſytámi.
Tenći jeſt zákon / który przedłożył Mojżeƺ ſynom Izráelſkim.
Te ſą świádectwá / y uſtáwy / y ſądy / które mówił Mojżeƺ do ſynów Izráelſkich / gdy wyƺli z Egiptu ;
Z tej ſtrony Jordánu / w dolinie przećiw Betfegor / w źiemi Sehoná / królá Amorejſkiego / który mieƺkáł w Heſebon / którego poráźił Mojżeƺ / y ſynowie Izráelſcy / gdy ƺli z Egiptu ;
Y ośiedli źiemię jego / y źiemię Ogá / królá Báſáńſkiego / dwóch królów Amorejſkich / którzy byli z tej ſtrony Jordánu ná wſchód ſłońcá ;
Od Aroer / które jeſt nád brzegiem potoku Arnon / y áż do góry Syon / którá jeſt Hermon ;
Y wƺyſtkie polá nád Jordánem ná wſchód ſłońcá / y áż do morzá puſtego / pod górą Fázgá.
