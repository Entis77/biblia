Teć ſą ſłowá / które mówił Mojżeƺ do wƺyſtkiego Izráelá przed Jordánem ná puƺcży / ná równinie przećiw morzu cżerwonemu / między Fárán / y między Tofel / y Lábán / y Háſerot / y Dyzáháb.
A jeſt jedenáśćie dni drogi od Horebu przez górę Seir áż do Kádeſ Bárne.
Y ſtáło śię cżterdzieſtego roku / jedenáſtego mieśiącá / pierwƺego dniá tegoż mieśiącá / że opowiedziáł Mojżeƺ ſynom Izráelſkim to wƺyſtko / co mu był rozkázáł Pán do nich.
Gdy poráził Sehoná / królá Amorejſkiego / który mieƺkáł w Heſebon / y Ogá / królá Báſáńſkiego / który mieƺkáł w Aſtárot w Edrej.
Przed Jordánem w ziemi Moábſkiej pocżął Mojżeƺ wykłádáć ten zákon / mówiąc :
Pán / Bóg náƺ / mówił do náſ ná górze Horeb / y rzekł : Doſyćieśćie mieƺkáli ná tej górze.
Obróććież śię / á ćiągnąc idźćie do góry Amorejſkiej / y ná wƺyſtkie miejſcá pogránicżne jej / ná równiny / ná góry / y ná doliny / y ku południowi / y ku brzegu morſkiemu / ku ziemi Chánánejſkiej / y do Libánu / áż do wielkiej rzeki / rzeki Eufráteſ.
Oto / dáłem wám tę ziemię ; wnijdźćież á pośiądźćie tę ziemię / o którą przyśiągł Pán ojcom wáƺym / Abráhámowi / Izáákowi y Jákóbowi / iż im ją dáć miáł / y náśieniu ich po nich.
Y rzekłem do wáſ ná on cżáſ / mówiąc : Nie mogę ſám nośić wáſ.
Pán / Bóg wáƺ / rozmnożył wáſ / á otośćie dziś jáko gwiázdy niebieſkie w mnóſtwie ſwojim.
Pán / Bóg ojców wáƺych / niech rozmnoży wáſ nád to / jákośćie teráz / tyśiąc kroć więcej ; y niech wáſ błogoſłáwi / jáko wám obiecáł.
Jákoż ſám znośić mám ćiężką prácę około wáſ / y brzemię wáƺe / y ſwáry wáƺe?
Obierzćie z śiebie męże mądre y umiejętne á doświádcżone w pokoleniách wáƺych / á poſtánowię je przełożonymi nád wámi.
Tedyśćie mi odpowiedzieli / y rzekliśćie : Dobrá rzecż / którąś rozkázáł ucżynić.
Y obráłem przedniejƺe z pokoleń wáƺych / męże mądre / y doświádcżone / y poſtánowiłem je przełożonymi nád wámi : tyśiącżniki / y ſetniki / y pięćdzieśiątniki / y dzieśiątniki / y rządce w pokoleniách wáƺych.
Y rozkázáłem ſędziom wáƺym ná on cżáſ / mówiąc : Wyſłuchywájćie ſpráw między bráćią wáƺą / y ſądźćie ſpráwiedliwie między mężem y między brátem jego / y między przychodniem jego.
Nie miejćie względu ná oſoby przy ſądzie ; ták máłego jáko wielkiego wyſłuchywájćie ; nie bójćie śię nikogo / ábowiem ſąd Boży jeſt ; á jeźliby co przytrudniejƺem było ná wáſ / odnieśiećie do mnie / á já tego przeſłuchám.
Y przykázáłem wám ná on cżáſ wƺyſtko / cośćie cżynić mieli.
Potem ruƺywƺy śię z Horeb przeƺliśmy wƺyſtkę onę puſtynią wielką y ſtráƺną / którąśćie widzieli / idąc drogą ku górze Amorejſkiej / jáko nám był rozkázáł Pán / Bóg náƺ / y przyƺliśmy áż do Kádeſ Bárne.
Y rzekłem do wáſ : Przyƺliśćie áż do góry Amorejcżyká / którego Pán / Bóg náƺ / dáwá nám.
Oto / podáwá Pán / Bóg twój / tobie tę ziemię ; idźże / á pośiądź ją / jákoć powiedziáł Pán / Bóg ojców twoich / nie bój śię áni śię lękáj.
Y przyƺliśćie do mnie wƺyſcy / á mówiliśćie : Poślijmy męże przed ſobą / którzyby nám przeƺpiegowáli ziemię / y dáli nám pewną ſpráwę / którąbyśmy drogą do niej wnijść mieli / y miáſtá / do którychbyśmy weƺli.
Co gdy mi śię podobáło / obráłem z wáſ dwánáśćie mężów / po jednym mężu z káżdego pokoleniá.
Którzy odƺedłƺy weƺli ná górę / á przyƺli áż do doliny Eſchol / y przeƺpiegowáli ziemię.
Nábráli też z ſobą owocu onej ziemi / y przynieśli do náſ / y dáli nám o tem ſpráwę / á powiedzieli : Dobrá jeſt ziemiá / którą nám dáwá Pán / Bóg náƺ.
Aleśćie nie chćieli iść : lecżeśćie byli odpornymi ſłowu Páná / Bogá wáƺego.
Y ƺemráliśćie w námiećiech wáƺych / mówiąc : Iż náſ Pán miáł w nienáwiśći / wywiódł náſ z ziemi Egipſkiej / áby náſ podáł w ręce Amorejcżyká / y wygubił náſ.
Dokądże iść mámy? Bráćiá náśi zepſowáli nám ſerce / mówiąc : Lud ten więkƺy y roślejƺy jeſt / niźli my / miáſtá wielkie / y wymurowáne áż ku niebu / á nád to / ſyny olbrzymów widzieliśmy tám.
Alem já mówił do wáſ : Nie lękájćie śię / áni śię ich bójćie.
Pán / Bóg wáƺ / który idzie przed wámi / wálcżyć będzie zá wáſ / ták jáko wám ucżynił w Egipćie przed ocżymá wáƺemi.
Tákże y ná puƺcży / gdzieś widziáł / jáko ćię nośił Pán / Bóg twój / jáko nośi ojćiec ſyná ſwego / w onej wƺyſtkiej drodze / którąśćie ƺli / áżeśćie przyƺli ná to miejſce.
Lecż áni ták uwierzyliśćie Pánu / Bogu wáƺemu.
Który chodził przed wámi w drodze / przepátrując wám miejſce do ſtánowieniá obozu / w nocy w ogniu / áby wám ukázáł drogę którąbyśćie iść mieli / we dnie záś w obłoku.
Y uſłyƺáł Pán głoſ ſłów wáƺych / á rozgniewáł śię bárdzo / y przyśiągł / mówiąc :
Záiſte żáden z ludu tego národu złego / nie oglądá onej ziemi dobrej / którąm przyśiągł dáć ojcom wáƺym ;
Oprócż Kálebá / ſyná Jefunowego / ten ją oglądá / y temu dám ziemię / którą deptáł / y ſynom jego / przeto że cále ƺedł zá Pánem.
Tákże y ná mię rozgniewáł śię Pán dlá wáſ / mówiąc : Y ty tám nie wnijdzieƺ.
Jozue / ſyn Nunów / któryć ſłuży / ten tám wnijdzie / tegoż utwierdzáj ; bo ją on w dziedzictwo podá Izráelowi.
Dziátki też wáƺe / o którycheśćie mówili / że będą podáne ná łup / tákże ſynowie wáśi / którzy dziś nie wiedzą dobrego áni złego / ći tám wnijdą y onym ją dám / á oni ją dziedzicżnie pośiędą ;
Ale wy obróćiwƺy śię / idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
A odpowiádájąc mówiliśćie do mnie : Zgrzeƺyliśmy Pánu ; pójdziemy / y wálcżyć będziemy według tego wƺyſtkiego / jáko nám rozkázáł Pán / Bóg náƺ ; y ubráł śię káżdy z wáſ we zbroje ſwoje / á chćieliśćie wnijść ná górę.
Lecż Pán rzekł do mnie : Powiedz im : Nie wſtępujćie / áni wálcżćie ; bom nie jeſt w pośrodku wáſ / ábyśćie nie byli poráżeni od nieprzyjáćiół wáƺych.
Co gdym wám opowiedziáł / nie ſłucháliśćie / áleśćie odporni byli ſłowu Páńſkimu / y hárdzieśćie poſtąpili ſobie / á weƺliśćie ná górę.
Tedy wyćiągnął Amorejcżyk / który mieƺkáł ná onej górze / przećiwko wám / y gonili wáſ / jáko zwykły pƺcżoły / y porázili wáſ w Seir áż do Hormá.
Y wróćiwƺy śię / płákáliśćie przed Pánem ; lecż nie wyſłucháł Pán głoſu wáƺego / á nie nákłonił uƺu ſwoich do wáſ.
Mieƺkáliśćie tedy w Kádeſ przez wiele dni według licżby dni / którycheśćie tám mieƺkáli.
