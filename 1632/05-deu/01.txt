Teć ſą ſłowá / które mówił Mojzeƺ do wƺyſtkiego Izráelá przed Jordánem ná puƺcży / ná równinie przećiw morzu cżerwonemu / miedzy Fárán / y miedzy Tofel / y Lábán / y Háſerot / y Dyzáháb.
A jeſt jedenaśćie dni drogi od Horebu przez górę Seir áż do Kádes Bárne.
Y ſtáło śię cżterdźieſtego roku / jedenáſtego mieśiącá / pierwƺego dniá tegoż mieśiącá / że opowiedźiał Mojzeƺ ſynom Izráelſkim to wƺyſtko / co mu był rozkázał Pán do nich.
Gdy poráźił Sehoná / królá Amorejſkiego / który mieƺkał w Heſebon / y Ogá / królá Báſáńſkiego / który mieƺkał w Aſtárot w Edrej.
Przed Jordánem w źiemi Moábſkiej pocżął Mojzeƺ wykłádáć ten zakon / mówiąc :
Pán / Bóg náƺ / mówił do nas ná górze Horeb / y rzekł : Doſyćieśćie mieƺkáli ná tey górze.
Obróććież śię / á ćiągnąc idźćie do góry Amorejſkiej / y ná wƺyſtkie miejſcá pogránicżne jey / ná równiny / ná góry / y ná doliny / y ku południowi / y ku brzegu morſkiemu / ku źiemi Chánánejſkiej / y do Libánu / áż do wielkiej rzeki / rzeki Eufrátes.
Oto / dáłem wam tę źiemię ; wnijdźćież á pośiądźćie tę źiemię / o którą przyśiągł Pán ojcom wáƺym / Abráhámowi / Izáákowi y Jákóbowi / iż im ją dáć miał / y náśieniu ich po nich.
Y rzekłem do was ná on cżás / mówiąc : Nie mogę ſám nośić was.
Pán / Bóg wáƺ / rozmnożył was / á otośćie dźiś jáko gwiázdy niebieſkie w mnóſtwie ſwojim.
Pán / Bóg ojców wáƺych / niech rozmnoży was nád to / jákośćie teraz / tyśiąc kroć więcey ; y niech was błogoſłáwi / jáko wam obiecał.
Jákoż ſám znośić mam ćiężką prácę około was / y brzemię wáƺe / y ſwáry wáƺe?
Obierzćie z śiebie męże mądre y umiejętne á doświádcżone w pokoleniách wáƺych / á poſtánowię je przełożonymi nád wámi.
Tedyśćie mi odpowiedźieli / y rzekliśćie : Dobrá rzecż / którąś rozkázał ucżynić.
Y obráłem przedniejƺe z pokoleń wáƺych / męże mądre / y doświádcżone / y poſtánowiłem je przełożonymi nád wámi : tyśiącżniki / y ſetniki / y pięćdźieśiątniki / y dźieśiątniki / y rządce w pokoleniách wáƺych.
Y rozkazałem ſędźiom wáƺym ná on cżás / mówiąc : Wyſłuchywájćie ſpráw miedzy bráćią wáƺą / y ſądźćie ſpráwiedliwie miedzy mężem y miedzy brátem jego / y miedzy przychodniem jego.
Nie miejćie względu ná oſoby przy ſądźie ; ták máłego jáko wielkiego wyſłuchywájćie ; nie bójćie śię nikogo / ábowiem ſąd Boży jeſt ; á jeſliby co przytrudniejƺem było ná was / odnieśiećie do mnie / á ja tego przeſłuchám.
Y przykázáłem wam ná on cżás wƺyſtko / cośćie cżynić mieli.
Potem ruƺywƺy śię z Horeb przeƺliśmy wƺyſtkę onę puſtynią wielką y ſtráƺną / którąśćie widźieli / idąc drogą ku górze Amorejſkiej / jáko nam był rozkázał Pán / Bóg náƺ / y przyƺliśmy áż do Kádes Bárne.
Y rzekłem do was : Przyƺliśćie áż do góry Amorejcżyká / którego Pán / Bóg náƺ / dáwá nam.
Oto / podáwá Pán / Bóg twój / tobie tę źiemię ; idźże / á pośiądź ją / jákoć powiedźiał Pán / Bóg ojców twojich / nie bój śię áni śię lękáj.
Y przyƺliśćie do mnie wƺyſcy / á mówiliśćie : Poślijmy męże przed ſobą / którzyby nam przeƺpiegowáli źiemię / y dáli nam pewną ſpráwę / którąbyſmy drogą do niej wnijść mieli / y miáſtá / do którychbyſmy weƺli.
Co gdy mi śię podobáło / obráłem z was dwánaśćie mężów / po jednym mężu z káżdego pokolenia.
Którzy odƺedƺy weƺli ná górę / á przyƺli áż do doliny Eſchol / y przeƺpiegowáli źiemię.
Nábráli też z ſobą owocu oney źiemi / y przynieśli do nas / y dáli nam o tym ſpráwę / á powiedźieli : Dobrá jeſt źiemiá / którą nam dáwá Pán / Bóg náƺ.
Aleśćie nie chćieli iść : lecżeśćie byli odpornymi ſłowu Páná / Bogá wáƺego.
Y ƺemráliśćie w námiećiech wáƺych / mówiąc : Iż nas Pán miał w nienáwiśći / wywiódł nas z źiemi Egipſkiej / áby nas podał w ręce Amorejcżyká / y wygubił nas.
Dokądże iść mamy? Bráćia náśi zepſowáli nam ſerce / mówiąc : Lud ten więkƺy y roślejƺy jeſt / niźli my / miáſtá wielkie / y wymurowáne áż ku niebu / á nád to / ſyny olbrzymów widźieliśmy tám.
Alem ja mówił do was : Nie lękájćie śię / áni śię ich bójćie.
Pán / Bóg wáƺ / który idźie przed wámi / wálcżyć będźie zá was / ták jáko wam ucżynił w Egipćie przed ocżymá wáƺemi.
Tákże y ná puƺcży / gdźieś widźiał / jáko ćię nośił Pán / Bóg twój / jáko nośi oćiec ſyná ſwego / w oney wƺyſtkiej drodze / którąśćie ƺli / áżeśćie przyƺli ná to miejſce.
Lecż áni ták uwierzyliśćie Pánu / Bogu wáƺemu.
Który chodźił przed wámi w drodze / przepátrując wam miejſce do ſtánowieniá obozu / w nocy w ogniu / áby wam ukázał drogę którąbyśćie iść mieli / we dnie záś w obłoku.
Y uſłyƺał Pán głos ſłów wáƺych / á rozgniewał śię bárzo / y przyśiągł / mówiąc :
Zájiſte żáden z ludu tego narodu złego / nie oglądá oney źiemi dobrej / którąm przyśiągł dáć ojcom wáƺym ;
Oprócż Kálebá / ſyná Jefunowego / ten ją oglądá / y temu dam źiemię / którą deptał / y ſynom jego / przeto że cále ƺedł zá Pánem.
Tákże y ná mię rozgniewał śię Pán dla was / mówiąc : Y ty tám nie wnijdźieƺ.
Jozue / ſyn Nunów / któryć ſłuży / ten tám wnijdźie / tegoż utwierdzáj ; bo ją on w dźiedźictwo podá Izráelowi.
Dźiátki też wáƺe / o którycheśćie mówili / że będą podáne ná łup / tákże ſynowie wáśi / którzy dźiś nie wiedzą dobrego áni złego / ći tám wnijdą y onym ją dam / á oni ją dźiedźicżnie pośiędą ;
Ale wy obróćiwƺy śię / idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
A odpowiádájąc mówiliśćie do mnie : Zgrzeƺyliśmy Pánu ; pójdźiemy / y wálcżyć będźiemy według tego wƺyſtkiego / jáko nam rozkázał Pán / Bóg náƺ ; y ubrał śię káżdy z was we zbroje ſwoje / á chćieliśćie wnijść ná górę.
Lecż Pán rzekł do mnie : Powiedz im : Nie wſtępujćie / áni wálcżćie ; bom nie jeſt w pośrzodku was / ábyśćie nie byli poráżeni od nieprzyjáćiół wáƺych.
Co gdym wam opowiedźiał / nie ſłucháliśćie / áleśćie odporni byli ſłowu Páńſkimu / y hárdźieśćie poſtąpili ſobie / á weƺliśćie ná górę.
Tedy wyćiągnął Amorejcżyk / który mieƺkał ná oney górze / przećiwko wam / y gonili was / jáko zwykły pƺcżoły / y poráźili was w Seir áż do Hormá.
Y wróćiwƺy śię / płákáliśćie przed Pánem ; lecż nie wyſłuchał Pán głoſu wáƺego / á nie nákłonił uƺu ſwojich do was.
Mieƺkáliśćie tedy w Kádes przez wiele dni według licżby dni / którycheśćie tám mieƺkáli.
