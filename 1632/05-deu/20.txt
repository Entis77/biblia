Gdy śię ruƺyƺ ná wojnę przećiw nieprzyjáćiołom twoim / á obácżyƺ konie / y wozy / y lud więkƺy nád ćię / nie bój śię ich ; bo Pán / Bóg twój / z tobą jeſt / który ćię wywiódł z ziemi Egipſkiej.
A gdy śię przybliżáć będziećie ku potykániu / wyſtąpi kápłán / y przemówi do ludu.
A rzecże do nich : Słucháj Izráelu / wy dziś ſtácżáćie bitwę z nieprzyjáćioły wáƺymi ; niechájże nie ſłábieje ſerce wáƺe / nie bójćie śię y nie trwożćie ſobą / áni śię ich lękájćie ;
Abowiem Pán / Bóg wáƺ / idzie z wámi / áby wálcżył zá wáſ z nieprzyjáćioły wáƺymi / á iżby wáſ wybáwił.
Tákże mówić będą hetmáni do ludu / y rzeką : Jeźliż kto jeſt / co zbudowáł dom nowy / á w nim nie pocżął mieƺkáć / niech idzie / á wróći śię do domu ſwego / by ſnáć nie zginął w bitwie / á kto inny pocżąłby w nim mieƺkáć.
Albo jeźli kto jeſt / co náſádził winnicę / á nie używáł owocu z niej / niech idzie / á wróći śię do domu ſwego / by ſnáć nie zginął w bitwie / á kto inny używáłby z niej.
Albo jeźli kto jeſt / co má poślubioną żonę / á jeƺcże jej nie pojął / niech idzie / á wróći śię do domu ſwego / by ſnáć nie zginął w bitwie / á kto inny nie pojął jej.
Nádto hetmáni mówić będą do ludu / y rzeką : Jeźli kto jeſt bojáźliwy / á lękliwego ſercá / niech idzie / á wróći śię do domu ſwego / áby nie pſowáł ſercá bráći ſwej / jáko jeſt zepſowáne ſerce jego.
A gdy przeſtáną hetmáni mówić do ludu / tedy poſtáwią rotmiſtrze wojſk przed ludem.
Gdy przyćiągnieƺ do jákiego miáſtá / ábyś go dobywáł / pocżęſtujeƺ je pokojem.
A jeźliże pokój ofiárowány przyjmie / y otworzyć brámy / tedy wƺyſtek lud ználeziony w nim będzie hołdowáł y ſłużył.
Ale jeźli nie ucżyni pokoju z tobą / lecż podnieśie przećiw tobie wojnę / oblężeƺ je.
A gdy je dá Pán / Bóg twój / w rękę twoję / tedy zábijeƺ w nim káżdego mężcżyznę oſtrzem miecżá.
Tylko niewiáſty / y dziátki / y bydłá / y wƺyſtko / co będzie w mieśćie / wƺyſtek łup jego obróćiƺ ſobie w korzyść / y będzieƺ jádł łupy nieprzyjáćiół twoich / które dá Pán / Bóg twój / tobie.
Ták ucżyniƺ wƺyſtkim miáſtom / dáleko odległym od ćiebie / które nie ſą z miáſt tych národów.
Ale z miáſt národów tych / które Pán / Bóg twój / podáwá tobie w dziedzictwo / żádnej duƺy żywić nie będzieƺ.
Lecż do ƺcżętu wytráćiƺ je / Hetejcżyká / Amorejcżyká / Chánánejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká / jákoć rozkázáł Pán / Bóg twój ;
Przeto żeby nie ucżyli wáſ cżynić według wƺyſtkich obrzydliwośći ſwoich / które cżynili bogom ſwym / y zgrzeƺylibyśćie przećiw Pánu / Bogu wáƺemu.
Gdy oblężeƺ jákie miáſto / á przez wiele dni dobywáć go będzieƺ / ábyś je wziął / nie pſuj drzewá jego / wyćinájąc je śiekierą / bo z niego jeść będzieƺ ; przetoż go nie wyćináj / ( bo ázáż ſą ludźmi drzewá polne? ) ábyś ich używáł do oblężeniá.
Tylko drzewá / które obácżyƺ / że owocu nierodzą / te pſowáć będzieƺ y wyćináć ; y budowáć będzieƺ báƺty przećiwko onemu miáſtu / które z tobą wálcży / póki go nie opánujeƺ.
