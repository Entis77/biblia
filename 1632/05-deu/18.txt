Nie będą mieli kápłáni Lewitowie / y wƺyſtko pokolenie Lewi / dźiáłu / áni dźiedźictwá z innym Izráelem : ogniſte ofiáry Páńſkie y dźiedźictwo jego jeść będą.
A dźiedźictwá nie będą mieli miedzy bráćią ſwoją ; Pán jeſt dźiedźictwem ich / jáko im powiedźiał.
A toć będźie práwo náleżące kápłánom od ludu / od ofiárujących ofiárę / bądź wołu / bądź owcę ; tedy oddádzą kápłánowi łopátkę / y cżeluśći y káłdun.
Pierwoćiny zbożá twego / winá twego / y oliwy twojey / tákże pierwoćiny wełny z owiec twojich oddáƺ mu.
Abowiem obrał go Pán / Bóg twój / ze wƺyſtkich pokoleń twojich / áby ſtał ku uſłudze w imieniu Páńſkim / on y ſynowie jego / po wƺyſtkie dni.
A gdyby przyƺedł Lewitá z któregokolwiek miáſtá twego / z cáłego Izráelá / gdźie przemieƺkiwá / á przyƺedłby z cáłej chęći duƺy ſwej ná miejſce / które ſobie obrał Pán :
Tedy ſłużyć będźie w imieniu Páná / Bogá ſwego / jáko wƺyſcy bráćia jego Lewitowie / którzy tám ſtoją przed oblicżnośćią Páńſką.
Cżęść równą z drugimi jeść będą / oprócż tego / co im náleżáło z dóbr ojców ich.
Gdy tedy wnijdźieƺ do źiemi / którą dáwá Pán / Bóg twój / tobie / nieucż śię cżynić według obrzydliwośći tych narodów.
Niech śię miedzy wámi nie znájduje / któryby przewodźił ſyná ſwego / álbo córkę ſwoję przez ogień ; tákże wieƺcżek / guślárz / y wróżek / y cżárownik.
Y cżárnokśiężnik / y ten / który ma ſpráwę z duchy złymi / y práktykárz / y wywiádujący śię cżego od umárłych.
Abowiem jeſt obrzydliwośćią Pánu káżdy / któryby to cżynił ; bo dla tych obrzydliwośći Pán / Bóg twój / wyrzucá te narody przed tobą.
Doſkonáłym będźieƺ przed Pánem / Bogiem twoim.
Abowiem národowie ći / które ty opánujeƺ / wieƺcżków y guślárzy ſłucháją ; ále tobie nie dopuƺcżá tego Pán / Bóg twój.
Proroká z pośrzodku ćiebie / z bráći twej / jákom ja jeſt / wzbudźi tobie Pán / Bóg twój ; onego ſłucháć będźiećie ;
Według tego wƺyſtkiego / jákoś żądał od Páná / Bogá twego / ná górze Horeb / w dźień zgromádzenia / mówiąc : Niech więcey nie ſłuchám głoſu Páná / Bogá mego / y ná ogień ten wielki niech nie pátrzę więcey / bym nie umárł.
Przetoż rzekł Pán do mnie : Dobrze mówili / co mówili.
Proroká im wzbudzę z pośrzodku bráći ich / jákoś ty jeſt / y włożę ſłowá moje w uſtá jego / y opowiádáć im będźie wƺyſtko / cokolwiek mu rozkáżę.
Y ſtánie śię / że ktobykolwiek nie był poſłuƺny ſłowom moim / które on mówić będźie w imię moje / Ja tego ƺukáć będę ná nim.
Wƺákże prorok / któryby ſobie hárdźie pocżął / mówiąc ſłowo w imieniu mojim / któregom mu mówić nie rozkázał y któryby mówił w imię bogów obcych / niech umrze prorok tákowy.
A jeſlibyś rzekł w ſercu ſwem : Jákoż rozeznámy to ſłowo / którego nie mówił Pán?
Jeſliby co mówił on prorok w imię Páńſkie / á nie ſtáłoby śię to / áni wypełniło / onoć to jeſt ſłowo / którego nie mówił Pán ; z hárdośći to mówił prorok on / nie bójże śię go.
