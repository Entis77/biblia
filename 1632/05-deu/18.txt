Nie będą mieli kápłáni Lewitowie / y wƺyſtko pokolenie Lewi / dziáłu / áni dziedzictwá z innym Izráelem : ogniſte ofiáry Páńſkie y dziedzictwo jego jeść będą.
A dziedzictwá nie będą mieli między bráćią ſwoją ; Pán jeſt dziedzictwem ich / jáko im powiedziáł.
A toć będzie práwo náleżące kápłánom od ludu / od ofiárujących ofiárę / bądź wołu / bądź owcę ; tedy oddádzą kápłánowi łopátkę / y cżeluśći y káłdun.
Pierwoćiny zbożá twego / winá twego / y oliwy twojej / tákże pierwoćiny wełny z owiec twoich oddáƺ mu.
Abowiem obráł go Pán / Bóg twój / ze wƺyſtkich pokoleń twoich / áby ſtáł ku uſłudze w imieniu Páńſkim / on y ſynowie jego / po wƺyſtkie dni.
A gdyby przyƺedł Lewitá z któregokolwiek miáſtá twego / z cáłego Izráelá / gdzie przemieƺkiwá / á przyƺedłby z cáłej chęći duƺy ſwej ná miejſce / które ſobie obráł Pán :
Tedy ſłużyć będzie w imieniu Páná / Bogá ſwego / jáko wƺyſcy bráćiá jego Lewitowie / którzy tám ſtoją przed oblicżnośćią Páńſką.
Cżęść równą z drugimi jeść będą / oprócż tego / co im náleżáło z dóbr ojców ich.
Gdy tedy wnijdzieƺ do ziemi / którą dáwá Pán / Bóg twój / tobie / nieucż śię cżynić według obrzydliwośći tych národów.
Niech śię między wámi nie znájduje / któryby przewodził ſyná ſwego / álbo córkę ſwoję przez ogień ; tákże wieƺcżek / guślárz / y wróżek / y cżárownik.
Y cżárnokśiężnik / y ten / który má ſpráwę z duchy złymi / y práktykárz / y wywiádujący śię cżego od umárłych.
Abowiem jeſt obrzydliwośćią Pánu káżdy / któryby to cżynił ; bo dlá tych obrzydliwośći Pán / Bóg twój / wyrzucá te národy przed tobą.
Doſkonáłym będzieƺ przed Pánem / Bogiem twoim.
Abowiem národowie ći / które ty opánujeƺ / wieƺcżków y guślárzy ſłucháją ; ále tobie nie dopuƺcżá tego Pán / Bóg twój.
Proroká z pośrodku ćiebie / z bráći twej / jákom já jeſt / wzbudzi tobie Pán / Bóg twój ; onego ſłucháć będziećie ;
Według tego wƺyſtkiego / jákoś żądáł od Páná / Bogá twego / ná górze Horeb / w dzień zgromádzeniá / mówiąc : Niech więcej nie ſłuchám głoſu Páná / Bogá mego / y ná ogień ten wielki niech nie pátrzę więcej / bym nie umárł.
Przetoż rzekł Pán do mnie : Dobrze mówili / co mówili.
Proroká im wzbudzę z pośrodku bráći ich / jákoś ty jeſt / y włożę ſłowá moje w uſtá jego / y opowiádáć im będzie wƺyſtko / cokolwiek mu rozkáżę.
Y ſtánie śię / że ktobykolwiek nie był poſłuƺny ſłowom moim / które on mówić będzie w imię moje / Já tego ƺukáć będę ná nim.
Wƺákże prorok / któryby ſobie hárdzie pocżął / mówiąc ſłowo w imieniu mojem / któregom mu mówić nie rozkázáł y któryby mówił w imię bogów obcych / niech umrze prorok tákowy.
A jeźlibyś rzekł w ſercu ſwem : Jákoż rozeznámy to ſłowo / którego nie mówił Pán?
Jeźliby co mówił on prorok w imię Páńſkie / á nie ſtáłoby śię to / áni wypełniło / onoć to jeſt ſłowo / którego nie mówił Pán ; z hárdośći to mówił prorok on / nie bójże śię go.
