A toć jeſt błogoſłáwieńſtwo / którem błogoſłáwił Mojżeƺ / mąż Boży / ſynom Izráelſkim przed śmierćią ſwoją.
Y rzekł : Pán z Synáj przyƺedł / y pojáwił śię im z Seiru / objáśnił śię z góry Fárán / á przyƺedł z dzieśięćiomá tyśiącámi świętych ; z práwicy jego zákon ogniſty dány im.
Záiſte on miłuje lud ; wƺyſcy święći jego ſą w rękách twych / y oni ſkupili śię do nogi twej / áby co pojęli z ſłów twoich.
Zákon podáł nám Mojżeƺ / dziedzictwo zebrániu Jákóbowemu.
Bo był królem w Izráelu / gdy śię zgromádzili przedniejśi z ludu / tákże pokoleniá Izráelſkie.
Niech żyje Ruben / á nie umierá ; á niech będzie mężów jego pocżet.
Potem też błogoſłáwił Judzie / y rzekł : Wyſłucháj Pánie głoſ Judy / á do ludu jego wprowádź go ; ręká jego będzie wálcżyłá záń / á ty go rátowáć będzieƺ przećiw nieprzyjáćiołom jego.
A do Lewiego rzekł : Tummim twoje / y Urim twoje było przy mężu świętym twoim / któregoś kuśił w Máſſá / y z którymeś miáł ſpór u wód Merybá.
Tenći to jeſt / który rzekł ojcu ſwemu / y mátce ſwej : Nie oglądám śię ná wáſ : y bráći ſwych nie znáł / y o ſynách ſwych nie wiedziáł ; ábowiem oni ſtrzegą ſłów twych / y przymierze twoje záchowują.
Będą ucżyć ſądów twoich Jákóbá / á zákonu twego Izráelá ; kłáść będą kádzenie pod nozdrze twoje / á cáłopálenie ná ołtárz twój.
Błogoſłáwże / Pánie / mocy jego / á ſpráwy rąk jego przyjmij wdzięcżnie ; zetrzyj biodrá nieprzyjáćiół jego / y tych którzy go nienáwidzą / áby powſtáć nie mogli.
A do Benjáminá rzekł : Ten jeſt nájmilƺy Pánu ; będzie mieƺkáł przy nim bezpiecżnie ; bronić go będzie ná káżdy dzień / á między rámiony jego przebywáć będzie.
Do Józefá też rzekł : Błogoſłáwioná od Páná ziemiá jego z nájlepƺych rzecży niebieſkich / z roſy / y z źródeł z ziemi wynikájących.
Y dlá rozkoƺnych urodzájów ſłonecżnych / tákże dlá rozkoƺnych doſtáłych urodzájów mieśięcżnych ;
Y dlá rozkoƺnych gór ſtárodáwnych / y dlá rozkoƺnych págórków wiecżnych ;
Tákże dlá rozkoƺnych owoców ziemi / y obfitośći jej / á dlá życżliwośći mieƺkájącego w krzu. Niech to błogoſłáwieńſtwo przyjdzie ná głowę Józefowę / y ná wierzch głowy Názárejcżyká między bráćią jego.
Jáko pierworodnego wołu ozdobá jego / á jáko rogi jednorożcowe rogi jego / temi národy zbodzie ná porząd áż do oſtátnich gránic ziemi ; á teć ſą dzieśięć tyśięcy Efráimitów / á te tyśiące Mánáſeſytów.
A do Zábuloná rzekł : Weſel śię Zábulon w wyjśćiu ſwem / á ty Iſáſchár w námiotách twoich.
Ludu ná górę przyzowią ; tám ofiárowáć będą ofiáry ſpráwiedliwośći / poniewáż obfitość morſką ſſáć będą / y zákryte ſkárby piáſku.
A do Gádá rzekł : Błogoſłáwiony / który rozmnáżá Gádá! jáko lew mieƺkáć będzie / á porwie rámię y głowę ;
Który ſobie upátrzył ná pocżątku mieƺkánie / á iż tám o dziále ſwoim przez zákonodáwcę ubezpiecżony jeſt ; przetoż pójdzie z kśiążęty ludu / ſpráwiedliwość Páńſką wykoná / y ſądy jego z Izráelem.
A do Dáná rzekł : Dán jáko ƺcżenię lwie wyſkákujące z Báſán.
A do Neftálimá rzekł : Neftáli / náſycony przyjáźni y pełny błogoſłáwieńſtwá Páńſkiego / záchód y południe opánujeƺ.
A do Aſerá rzekł : Błogoſłáwiony nád inne ſyny Aſer / będzie przyjemny bráćiom ſwoim / y omocży w oliwie nogę ſwoję.
Żelázo y miedź pod obuwiem twojim ; y póki będą trwáć dni twoje / ſłynąć będzie moc twojá.
Niemáƺ ták práwego / jáko Bóg / który jeździ po niebie ku rátunku twemu / y w wielmożnośći ſwej ná obłokách.
Mieƺkániem twojim Bóg wiecżny / á ze ſpodku rámioná wiecżnośći. Ten wyrzući przed tobą nieprzyjáćielá / á rzecżeć : Wytráć go ;
Aby mieƺkáł Izráel bezpiecżnie ſám / źródło Jákóbowe / w ziemi zbożá y winá / którego też niebioſá kropić będą roſą.
Błogoſłáwionyś ty / Izráelu! Któż podobny tobie? ludu záchowány przez Páná / który jeſt tárcżą rátunku twego / á miecżem zácnośći twojej. Przeto obłudnieć śię poddádzą nieprzyjáćiele twoi / á ty wynioſłość ich deptáć będzieƺ.
