A toć jeſt błogoſłáwieńſtwo / którem błogoſłáwił Mojżeƺ / mąż Boży / ſynom Izráelſkim przed śmierćią ſwoją.
Y rzekł : Pán z Synáj przyƺedł / y pojáwił śię im z Seiru / objáśnił śię z góry Fárán / á przyƺedł z dźieśięćiomá tyśiącámi świętych ; z práwicy jego zakon ogniſty dány im.
Záiſte on miłuje lud ; wƺyſcy święći jego ſą w rękách twych / y oni ſkupili śię do nogi twej / áby co pojęli z ſłów twoich.
Zakon podáł nám Mojżeƺ / dźiedźictwo zebrániu Jákóbowemu.
Bo był królem w Izráelu / gdy śię zgromádźili przedniejśi z ludu / tákże pokoleniá Izráelſkie.
Niech żyje Ruben / á nie umierá ; á niech będźie mężów jego pocżet.
Potem też błogoſłáwił Judźie / y rzekł : Wyſłucháj Pánie głoſ Judy / á do ludu jego wprowádź go ; ręká jego będźie wálcżyłá záń / á ty go rátowáć będźieƺ przećiw nieprzyjáćiołom jego.
A do Lewiego rzekł : Tummim twoje / y Urim twoje było przy mężu świętym twoim / któregoś kuśił w Máſſá / y z którymeś miał ſpór u wód Merybá.
Tenći to jeſt / który rzekł ojcu ſwemu / y mátce ſwej : Nie oglądám śię ná was : y bráći ſwych nie znáł / y o ſynách ſwych nie wiedźiáł ; ábowiem oni ſtrzegą ſłów twych / y przymierze twoje záchowują.
Będą ucżyć ſądów twoich Jákóbá / á zakonu twego Izráelá ; kłáść będą kádzenie pod nozdrze twoje / á cáłopálenie ná ołtárz twój.
Błogoſłáwże / Pánie / mocy jego / á ſpráwy rąk jego przyjmij wdźięcżnie ; zetrzyj biodrá nieprzyjáćiół jego / y tych którzy go nienáwidzą / áby powſtáć nie mogli.
A do Benjáminá rzekł : Ten jeſt nájmilƺy Pánu ; będźie mieƺkáł przy nim bezpiecżnie ; bronić go będźie ná káżdy dźień / á między rámiony jego przebywáć będźie.
Do Józefá też rzekł : Błogoſłáwioná od Páná źiemiá jego z nájlepƺych rzecży niebieſkich / z roſy / y z źródeł z źiemi wynikájących.
Y dla rozkoƺnych urodzájów ſłonecżnych / tákże dla rozkoƺnych doſtáłych urodzájów mieśięcżnych ;
Y dla rozkoƺnych gór ſtárodáwnych / y dla rozkoƺnych págórków wiecżnych ;
Tákże dla rozkoƺnych owoców źiemi / y obfitośći jey / á dla życżliwośći mieƺkájącego w krzu. Niech to błogoſłáwieńſtwo przyjdźie ná głowę Józefowę / y ná wierzch głowy Názárejcżyká między bráćią jego.
Jáko pierworodnego wołu ozdobá jego / á jáko rogi jednorożcowe rogi jego / temi národy zbodźie ná porząd áż do oſtátnich gránic źiemi ; á teć ſą dźieśięć tyśięcy Efráimitów / á te tyśiące Mánáſeſytów.
A do Zábuloná rzekł : Weſel śię Zábulon w wyjśćiu ſwem / á ty Iſáſchár w námiotách twoich.
Ludu ná górę przyzowią ; tám ofiárowáć będą ofiáry ſpráwiedliwośći / poniewáż obfitość morſką ſſáć będą / y zákryte ſkárby piáſku.
A do Gádá rzekł : Błogoſłáwiony / który rozmnáżá Gádá! jáko lew mieƺkáć będźie / á porwie rámię y głowę ;
Który ſobie upátrzył ná pocżątku mieƺkánie / á iż tám o dźiále ſwoim przez zakonodáwcę ubezpiecżony jeſt ; przetoż pójdźie z kśiążęty ludu / ſpráwiedliwość Páńſką wykoná / y ſądy jego z Izráelem.
A do Dáná rzekł : Dán jáko ƺcżenię lwie wyſkákujące z Báſán.
A do Neftálimá rzekł : Neftáli / náſycony przyjáźni y pełny błogoſłáwieńſtwá Páńſkiego / záchód y południe opánujeƺ.
A do Aſerá rzekł : Błogoſłáwiony nád inne ſyny Aſer / będźie przyjemny bráćiom ſwoim / y omocży w oliwie nogę ſwoję.
Żelázo y miedź pod obuwiem twojim ; y póki będą trwáć dni twoje / ſłynąć będźie moc twojá.
Niemáƺ ták práwego / jáko Bóg / który jeźdźi po niebie ku rátunku twemu / y w wielmożnośći ſwej ná obłokách.
Mieƺkániem twojim Bóg wiecżny / á ze ſpodku rámioná wiecżnośći. Ten wyrzući przed tobą nieprzyjáćielá / á rzecżeć : Wytráć go ;
Aby mieƺkáł Izráel bezpiecżnie ſám / źródło Jákóbowe / w źiemi zbożá y winá / którego też niebioſá kropić będą roſą.
Błogoſłáwionyś ty / Izráelu! Któż podobny tobie? ludu záchowány przez Páná / który jeſt tárcżą rátunku twego / á miecżem zácnośći twojey. Przeto obłudnieć śię poddádzą nieprzyjáćiele twoi / á ty wynioſłość ich deptáć będźieƺ.
