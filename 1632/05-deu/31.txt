Przyƺedłƺy tedy Mojżeƺ / opowiedźiáł te ſłowá wƺyſtkiemu Izráelowi /
Y rzekł do nich : Sto y dwádźieśćiá lát jeſt mi dźiś ; nie mogę więcej wychodźić y wchodźić ; á też mi Pán powiedźiáł : Nie przejdźieƺ zá ten Jordán.
Pán / Bóg twój / ſám pójdźie przed tobą ; on wytráći národy te przed tobą / y pośiędźieƺ je : Jozue / ten pójdźie przed tobą / jáko powiedźiáł Pán.
Y ucżyni im Pán / jáko ucżynił Sehonowi / y Ogowi / królom Amorejſkim / y źiemi ich / które wygłádźił.
A gdy je wám Pán podá / tedy im ucżynićie według káżdego przykázániá / jákom wám przykázáł.
Zmácniájćież śię / á mężnie ſobie pocżynájćie / nie bójćie śię / áni śię lękájćie twárzy ich ; ábowiem Pán / Bóg twój / on idźie z tobą / nie odſtąpi ćię / áni ćię opuśći.
Tedy przyzwáwƺy Mojżeƺ Jozuego / rzekł do niego przed ocżymá wƺyſtkiego Izráelá : Zmocnij śię / á mężnie ſobie pocżynáj ; bo ty wnijdźieƺ z ludem tym do źiemi / o którą przyśiągł Pán ojcom ich / że im ją dá / á ty im ją w dźiedźictwo podźieliƺ.
A Pán on pójdźie przed tobą / on będźie z tobą / nie odſtąpi ćię / áni ćię opuśći ; nie bójże śię / áni śię lękáj.
Tedy nápiſáł Mojżeƺ ten zákon y oddáł go kápłánom / ſynom Lewiego / którzy nośili ſkrzynię przymierzá Páńſkiego / y wƺyſtkim ſtárƺym Izráelſkim.
Przytem rozkázáł im Mojżeƺ / mówiąc : Ná pocżątku káżdego śiódmego roku / w pewny cżáſ roku odpuƺcżániá / w święto Kucżek ;
Gdy śię zejdźie wƺyſtek Izráel / áby śię pokázáł przed Pánem / Bogiem twoim / ná miejſcu / które ſobie obierze / obwołáƺ ten zákon przed wƺyſtkim Izráelem w uƺách ich ;
A zgromádźiƺ lud / męże / y niewiáſty / y dźiátki / y przychodnie twoje / którzy ſą w brámách twoich / áby ſłucháli / y áby śię ucżyli / y báli Páná / Bogá wáƺego / y przeſtrzegáli / áby cżynili wƺyſtkie ſłowá zákonu tego.
Synowie też ich / którzy jeƺcże nie wiedzą / niecháj ſłucháją / y niech śię ucżą báć Páná / Bogá wáƺego / po wƺyſtkie dni / których żyć będźiećie ná źiemi / do której idźiećie / przepráwiwƺy śię przez Jordán wnijdźiećie / ábyśćie ją pośiedli.
Zátem rzekł Pán do Mojżeƺá : Oto / przybliżyły śię dni śmierći twojej : przyzówże Jozuego / á ſtáńćie w námioćie zgromádzeniá / á przykáżę mu. Szedł tedy Mojżeƺ y Jozue / á ſtánęli w námioćie zgromádzeniá.
Tedy śię ukázáł Pán w námioćie w ſłupie obłokowym ; y ſtánął on ſłup obłokowy náde drzwiámi námiotu /
Y rzekł Pán do Mojżeƺá : Oto / ty záśnieƺ z ojcy twoimi ; á powſtáwƺy lud ten będźie cudzołożył / náśládując bogów cudzoźiemców tej źiemi / do której wchodźi / áby mieƺkáł w pośrodku niej y opuśći mię / y złámie przymierze moje / którem poſtánowił z nim.
Przetoż zápáli śię gniew mój przećiwko jemu onego dniá / á opuƺcżę je / y ſkryję oblicże moje przed nimi / á będźie ná pożárćie / y znájdą go wiele złych rzecży y utrápieniá ; y rzecże dniá onego : Azáż nie dlátego / iż Bogá mego niemáƺ w pośrodku mnie / ználázły mię te złe rzecży?
A Já kryjąc ſkryję oblicże moje w on dźień dlá wƺyſtkiego złego / które cżynili / udáwƺy śię zá bogi cudzymi.
Teráz tedy nápiƺćie ſobie tę pieśń / á náucż jej ſynów Izráelſkich ; włóż ją w uſtá ich / áby mi byłá tá pieśń świádectwem przećiwko ſynom Izráelowym.
Abowiem wprowádzę go do źiemi / o którąm przyśiągł ojcom jego / opływájącej mlekiem y miodem ; y będźie jádł / á náje śię / y utyje / á obróći śię do bogów cudzych / y ſłużyć im będźie / y rozdráźnią mię / y zrzucą przymierze moje /
Y ſtánie śię / gdy go znájdźie wiele złego y utrápieniá / że będźie tá pieśń przed nim ná świádectwo ; ábowiem nie przyjdźie w zápámiętánie w uśćiech náśieniá ich / gdyż wiem myśli jego / y co on jeƺcże dźiś cżynić będźie / pierwej niż go wprowádzę do źiemi / o którąm przyśiągł.
Y nápiſáł Mojżeƺ tę pieśń onegoż dniá / y náucżył jej ſynów Izráelſkich.
Potem przykázáł Jozuemu / ſynowi Nunowemu / y rzekł : Zmácniáj śię / y mężnie ſobie pocżynáj ; ábowiem ty wprowádźiƺ ſyny Izráelſkie do źiemi / o którąm im przyśiągł ; á Já będę z tobą.
Y ſtáło śię / gdy nápiſáł Mojżeƺ ſłowá zákonu tego w kśięgi / y dokońcżył ich /
Rozkázáł Mojżeƺ Lewitom / którzy nośili ſkrzynię przymierzá Páńſkiego / mówiąc :
Weźmijćie kśięgi zákonu tego y włóżćie je przy ſtronie do ſkrzyni przymierzá Páná / Bogá wáƺego / áby tám były przećiwko tobie ná świádectwo ;
Abowiem já znám upór twój / y kárk twój twárdy ; oto / póki já jeƺcże dźiś żyję z wámi / jeſteśćie odpornymi Pánu / jákoż dáleko więcej po śmierći mojej?
Zgromádźćież do mnie wƺyſtkie ſtárƺe pokoleń wáƺych / y przełożone wáƺe / żebym mówił w uƺách ich ſłowá te / y będę wzywáł ná świádectwo przećiwko nim niebo y źiemię /
Poniewáż wiem / że po śmierći mojej popſujećie śię bárdzo / á uſtąpićie z drogi / którąm wám rozkázáł ; przetoż przyjdźie ná wáſ to złe w oſtátnie dni / gdy będźiećie cżynić złe przed ocżymá Páńſkimi / dráźniąc go ſpráwámi rąk wáƺych.
Y mówił Mojżeƺ w uƺách wƺyſtkiego zgromádzeniá Izráelſkiego ſłowá tej pieśni / áż jej dokońcżył.
