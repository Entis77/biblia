Przyƺedƺy tedy Mojzeƺ / opowiedźiał te ſłowá wƺyſtkiemu Izráelowi /
Y rzekł do nich : Sto y dwádźieśćiá lat jeſt mi dźiś ; nie mogę więcey wychodźić y wchodźić ; á też mi Pán powiedźiał : Nie przejdźieƺ zá ten Jordán.
Pán / Bóg twój / ſám pójdźie przed tobą ; on wytráći národy te przed tobą / y pośiędźieƺ je : Jozue / ten pójdźie przed tobą / jáko powiedźiał Pán.
Y ucżyni im Pán / jáko ucżynił Sehonowi / y Ogowi / królom Amorejſkim / y źiemi ich / które wygłádźił.
A gdy je wam Pán podá / tedy im ucżynićie według káżdego przykázánia / jákom wam przykázał.
Zmácniájćież śię / á mężnie ſobie pocżynájćie / nie bójćie śię / áni śię lękájćie twárzy ich ; ábowiem Pán / Bóg twój / on idźie z tobą / nie odſtąpi ćię / áni ćię opuśći.
Tedy przyzwáwƺy Mojzeƺ Jozuego / rzekł do niego przed ocżymá wƺyſtkiego Izráelá : Zmocnij śię / á mężnie ſobie pocżynáj ; bo ty wnijdźieƺ z ludem tym do źiemi / o którą przyśiągł Pán ojcom ich / że im ją da / á ty im ją w dźiedźictwo podźieliƺ.
A Pán on pójdźie przed tobą / on będźie z tobą / nie odſtąpi ćię / áni ćię opuśći ; nie bójże śię / áni śię lękáj.
Tedy nápiſał Mojzeƺ ten zakon y oddał go kápłánom / ſynom Lewiego / którzy nośili ſkrzynię przymierza Páńſkiego / y wƺyſtkim ſtárƺym Izráelſkim.
Przytem rozkázał im Mojzeƺ / mówiąc : Ná pocżątku káżdego śiódmego roku / w pewny cżás roku odpuƺcżánia / w święto Kucżek ;
Gdy śię zejdźie wƺyſtek Izráel / áby śię pokázał przed Pánem / Bogiem twoim / ná miejſcu / które ſobie obierze / obwołáƺ ten zakon przed wƺyſtkim Izráelem w uƺách ich ;
A zgromádźiƺ lud / męże / y niewiáſty / y dźiátki / y przychodnie twoje / którzy ſą w brámách twojich / áby ſłucháli / y áby śię ucżyli / y báli Páná / Bogá wáƺego / y przeſtrzegáli / áby cżynili wƺyſtkie ſłowá zakonu tego.
Synowie też ich / którzy jeƺcże nie wiedzą / niecháj ſłucháją / y niech śię ucżą báć Páná / Bogá wáƺego / po wƺyſtkie dni / których żyć będźiećie ná źiemi / do którey idźiećie / przepráwiwƺy śię przez Jordán wnijdźiećie / ábyśćie ją pośiedli.
Zátem rzekł Pán do Mojżeƺá : Oto / przybliżyły śię dni śmierći twojey : przyzówże Jozuego / á ſtáńćie w námioćie zgromádzenia / á przykáżę mu. Szedł tedy Mojzeƺ y Jozue / á ſtánęli w námioćie zgromádzenia.
Tedy śię ukázał Pán w námioćie w ſłupie obłokowym ; y ſtánął on ſłup obłokowy náde drzwiámi namiotu /
Y rzekł Pán do Mojżeƺá : Oto / ty záśnieƺ z ojcy twoimi ; á powſtáwƺy lud ten będźie cudzołożył / náśládując bogów cudzoźiemców tey źiemi / do którey wchodźi / áby mieƺkał w pośrzodku niej y opuśći mię / y złámie przymierze moje / którem poſtánowił z nim.
Przetoż zápáli śię gniew mój przećiwko jemu onego dniá / á opuƺcżę je / y ſkryję oblicże moje przed nimi / á będźie ná pożárćie / y znájdą go wiele złych rzecży y utrapienia ; y rzecże dniá onego : Azaż nie dla tego / iż Bogá mego niemáƺ w pośrzodku mnie / ználázły mię te złe rzecży?
A Ja kryjąc ſkryję oblicże moje w on dźień dla wƺyſtkiego złego / które cżynili / udáwƺy śię zá bogi cudzymi.
Teraz tedy nápiƺćie ſobie tę pieśń / á náucż jey ſynów Izráelſkich ; włóż ją w uſtá ich / áby mi byłá tá pieśń świádectwem przećiwko ſynom Izráelowym.
Abowiem wprowádzę go do źiemi / o którąm przyśiągł ojcom jego / opływájącey mlekiem y miodem ; y będźie jádł / á náje śię / y utyje / á obróći śię do bogów cudzych / y ſłużyć im będźie / y rozdráźnią mię / y zrzucą przymierze moje /
Y ſtánie śię / gdy go znájdźie wiele złego y utrapienia / że będźie tá pieśń przed nim ná świádectwo ; ábowiem nie przydźie w zápámiętánie w uśćiech náśieniá ich / gdyż wiem myśli jego / y co on jeƺcże dźiś cżynić będźie / pierwej niż go wprowádzę do źiemi / o którąm przyśiągł.
Y nápiſał Mojzeƺ tę pieśń onegoż dniá / y náucżył jey ſynów Izráelſkich.
Potem przykázał Jozuemu / ſynowi Nunowemu / y rzekł : Zmácniáj śię / y mężnie ſobie pocżynáj ; ábowiem ty wprowádźiƺ ſyny Izráelſkie do źiemi / o którąm im przyśiągł ; á Ja będę z tobą.
Y ſtáło śię / gdy nápiſał Mojzeƺ ſłowá zakonu tego w kśięgi / y dokońcżył ich /
Rozkázał Mojzeƺ Lewitom / którzy nośili ſkrzynię przymierza Páńſkiego / mówiąc :
Weźmijćie kśięgi zakonu tego y włóżćie je przy ſtronie do ſkrzyni przymierza Páná / Bogá wáƺego / áby tám były przećiwko tobie ná świádectwo ;
Abowiem ja znam upór twój / y kárk twój twárdy ; oto / póki ja jeƺcże dźiś żyję z wámi / jeſteśćie odpornymi Pánu / jákoż dáleko więcey po śmierći mojey?
Zgromádźćież do mnie wƺyſtkie ſtárƺe pokoleń wáƺych / y przełożone wáƺe / żebym mówił w uƺách ich ſłowá te / y będę wzywał ná świádectwo przećiwko nim niebo y źiemię /
Poniewáż wiem / że po śmierći mojey popſujećie śię bárzo / á uſtąpićie z drogi / którąm wam rozkázał ; przetoż przydźie ná was to złe w oſtátnie dni / gdy będźiećie cżynić złe przed ocżymá Páńſkimi / dráźniąc go ſpráwámi rąk wáƺych.
Y mówił Mojzeƺ w uƺách wƺyſtkiego zgromádzenia Izráelſkiego ſłowá tey pieśni / áż jey dokońcżył.
