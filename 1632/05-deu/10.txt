Ná on cżáſ rzekł Pán do mnie : Wyćieƺ ſobie dwie táblice kámienne / podobne pierwƺym / y wſtąp do mnie ná górę ; ucżyń też ſobie ſkrzynię drzewiáną.
A nápiƺę ná táblicách onych ſłowá / które były ná táblicách pierwƺych / któreś ſtłukł / á włożyƺ je do ſkrzyni.
Ucżyniłem tedy ſkrzynię z drzewá ſyttym / y wyćioſáłem dwie táblice kámienne / podobne pierwƺym / y wſtąpiłem ná górę / májąc dwie táblice w rękách ſwych.
Y nápiſáł Pán ná onych táblicách / ták jáko był pierwej nápiſáł / dźieśięć ſłów / które mówił Pán do wáſ ná górze z pośrodku ogniá w dźień zgromádzeniá onego ; y dáł mi je Pán.
A obróćiwƺy śię zſtąpiłem z góry / y włożyłem one táblice do ſkrzyni / którąm był ucżynił / y były tám / jáko mi rozkázáł Pán.
Tedy ſynowie Izráelſcy ruƺyli śię od Beerot ſynów Jáhákonowych ku Meſerá / gdźie umárł Aáron / y támże jeſt pogrzebiony ; á odpráwowáł urząd kápłáńſki Eleázár / ſyn jego / ná miejſcu jego.
Stámtąd śię ruƺyli do Gádgád / á z Gádgád do Jotbátá / do źiemi ćiekących wód.
Onegoż cżáſu odłącżył Pán pokolenie Lewi do noƺeniá ſkrzyni przymierzá Páńſkiego / á iżby ſtáwáli przed oblicżem Páńſkim do uſługi jemu / y żeby błogoſłáwili w imieniu jego áż do dniá dźiśiejƺego.
Dlá tego nie miáło pokolenie Lewi dźiáłu / áni dźiedźictwá między bráćią ſwoją ; ábowiem Pán jeſt dźiedźictwem jego / jáko mu powiedźiáł Pán / Bóg twój.
A jám trwáł ná górze / jáko y przedtem / cżterdźieśći dni y cżterdźieśći nocy / y wyſłucháł mię Pán y onego rázu / że ćię nie chćiáł Pán wytráćić.
Potem rzekł Pán do mnie : Wſtáń / idź wprzód przed ludem tym / że wnijdą / á pośiędą źiemię / o którąm przyśiągł ojcom ich / że im ją dám.
A teráz / Izráelu / cżegoż Pán / Bóg twój / żądá od ćiebie? jedno ábyś śię báł Páná / Bogá twego / á chodźił we wƺyſtkich drogách jego ; ábyś go miłowáł y ſłużył Pánu / Bogu twemu / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej.
Strzegąc przykázáń Páńſkich y uſtáw jego / które já dźiś rozkázuję tobie / ábyć śię dobrze dźiáło.
Oto Páná / Bogá twego / ſą niebioſá / y niebioſá niebioſ / źiemiá y wƺyſtko / co ná niej.
Wƺákże tylko w ojcách twoich upodobáło śię Pánu / że je umiłowáł / y obráł náśienie ich po nich / to jeſt / wáſ ze wƺyſtkich národów / jáko śię to dźiś okázuje.
Przetoż obrzeżćie nieobrzeſkę ſercá wáƺego / y kárku wáƺego nie zátwárdzájćie więcej.
Abowiem Pán / Bóg wáƺ / jeſt Bogiem bogów / y Pánem pánów / Bóg wielki / możny / y ſtráƺny / który nie má względu ná oſoby / áni przyjmuje dárów ;
Który cżyni ſąd śieroćie y wdowie / á miłuje przychodniá / dáwájąc mu chleb y odźienie.
Miłujćież tedy y wy przychodniá ; bośćie przychodniámi byli w źiemi Egipſkiej.
Páná / Bogá twego / będźieƺ śię báł / jemu ſłużył / przy nim trwáł / y przez imię jego przyśięgáł.
Onći jeſt chwáłą twoją / y on Bogiem twoim / który ucżynił z tobą wielmożne / y ſtráƺne rzecży / które widźiáły ocży twoje.
W śiedmdźieśiąt duƺ zſtąpili ojcowie twoi do Egiptu / á teráz rozmnożył ćię Pán / Bóg twój / jáko gwiázdy niebieſkie / w mnóſtwo.
