Ná on cżás rzekł Pán do mnie : Wyćieƺ ſobie dwie táblice kámienne / podobne pierwƺym / y wſtąp do mnie ná górę ; ucżyń też ſobie ſkrzynię drzewiáną.
A nápiƺę ná táblicách onych ſłowá / które były ná táblicách pierwƺych / któreś ſtłukł / á włożyƺ je do ſkrzyni.
Ucżyniłem tedy ſkrzynię z drzewá ſyttym / y wyćioſáłem dwie táblice kámienne / podobne pierwƺym / y wſtąpiłem ná górę / májąc dwie táblice w rękách ſwych.
Y nápiſał Pán ná onych táblicách / ták jáko był pierwej nápiſał / dźieśięć ſłów / które mówił Pán do was ná górze z pośrzodku ogniá w dźień zgromádzenia onego ; y dał mi je Pán.
A obróćiwƺy śię zſtąpiłem z góry / y włożyłem one táblice do ſkrzyni / którąm był ucżynił / y były tám / jáko mi rozkázał Pán.
Tedy ſynowie Izráelſcy ruƺyli śię od Beerot ſynów Jáhakonowych ku Meſerá / gdźie umárł Aáron / y támże jeſt pogrzebiony ; á odpráwował urząd kápłáńſki Eleázár / ſyn jego / ná miejſcu jego.
Stámtąd śię ruƺyli do Gádgád / á z Gádgád do Jotbátá / do źiemi ćiekących wód.
Onegoż cżáſu odłącżył Pán pokolenie Lewi do noƺeniá ſkrzyni przymierza Páńſkiego / á iżby ſtáwáli przed oblicżem Páńſkim do uſługi jemu / y żeby błogoſłáwili w imieniu jego áż do dniá dźiśiejƺego.
Dla tego nie miáło pokolenie Lewi dźiáłu / áni dźiedźictwá miedzy bráćią ſwoją ; ábowiem Pán jeſt dźiedźictwem jego / jáko mu powiedźiał Pán / Bóg twój.
A jam trwał ná górze / jáko y przedtem / cżterdźieśći dni y cżterdźieśći nocy / y wyſłuchał mię Pán y onego rázu / że ćię nie chćiał Pán wytráćić.
Potem rzekł Pán do mnie : Wſtáń / idź wprzód przed ludem tym / że wnijdą / á pośiędą źiemię / o którąm przyśiągł ojcom ich / że im ją dam.
A teraz / Izráelu / cżegoż Pán / Bóg twój / żądá od ćiebie? jedno ábyś śię bał Páná / Bogá twego / á chodźił we wƺyſtkich drogách jego ; ábyś go miłował y ſłużył Pánu / Bogu twemu / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey.
Strzegąc przykázáń Páńſkich y uſtáw jego / które ja dźiś rozkázuję tobie / ábyć śię dobrze dźiáło.
Oto Páná / Bogá twego / ſą niebioſá / y niebioſá niebios / źiemiá y wƺyſtko / co ná niej.
Wƺákże tylko w ojcách twojich upodobáło śię Pánu / że je umiłował / y obrał náśienie ich po nich / to jeſt / was ze wƺyſtkich národów / jáko śię to dźiś okázuje.
Przetoż obrzeżćie nieobrzeſkę ſercá wáƺego / y kárku wáƺego nie zátwárdzájćie więcey.
Abowiem Pán / Bóg wáƺ / jeſt Bogiem bogów / y Pánem pánów / Bóg wielki / możny / y ſtráƺny / który nie ma względu ná oſoby / áni przyjmuje dárów ;
Który cżyni ſąd śieroćie y wdowie / á miłuje przychodniá / dáwájąc mu chleb y odźienie.
Miłujćież tedy y wy przychodniá ; bośćie przychodniámi byli w źiemi Egipſkiej.
Páná / Bogá twego / będźieƺ śię bał / jemu ſłużył / przy nim trwał / y przez imię jego przyśięgał.
Onći jeſt chwáłą twoją / y on Bogiem twoim / który ucżynił z tobą wielmożne / y ſtráƺne rzecży / które widźiáły ocży twoje.
W śiedmdźieśiąt duƺ zſtąpili ojcowie twoi do Egiptu / á teraz rozmnożył ćię Pán / Bóg twój / jáko gwiázdy niebieſkie / w mnóſtwo.
