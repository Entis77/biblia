Teć ſą ſłowá przymierzá / które przykázáł Pán Mojżeƺowi poſtánowić z ſynámi Izráelſkimi w ziemi Moábſkiej nád ono przymierze / które poſtánowił z nimi ná Horeb.
Zwołáł tedy Mojżeƺ wƺyſtkiego Izráelá / y mówił do nich : Wyśćie widzieli wƺyſtko / co cżynił Pán przed ocżymá wáƺemi w ziemi Egipſkiej Fáráonowi / y wƺyſtkim ſługom jego / y cáłej ziemi jego ;
Pokuſy wielkie / które widziáły ocży twoje / znáki / y cudá one wielkie.
Ale nie dáł wám Pán ſercá ku zrozumieniu / y ocżu ku widzeniu / áni uƺu ku ſłyƺeniu / áż do dniá dziśiejƺego.
Y prowádziłem wáſ przez cżterdzieśći lát po puƺcży ; nie wiotƺáły ƺáty wáƺe ná wáſ / y obuwie wáƺe nie wiotƺáło ná nogách wáƺych.
Chlebáśćie nie jedli / y winá / y mocnego nápoju nie piliśćie / ábyśćie wiedzieli / żem Já Pán / Bóg wáƺ.
A gdyśćie przyƺli ná to miejſce / wyćiągnął Sehon / król Heſebońſki / y Og / król Báſáńſki / przećiwko nám ku bitwie / y poráziliśmy je.
A wziąwƺy ziemię ich / podáliśmy ją w dziedzictwo Rubenitom / y Gádytom / y połowie pokoleniá Mánáſeſowego.
A ták przeſtrzegájćież ſłów przymierzá tego / y cżyńćie je / áby śię wám ƺcżęśliwie powodziło wƺyſtko / co cżynić będziećie.
Wy ſtoićie dziś wƺyſcy przed Pánem / Bogiem wáƺym / kśiążętá wáƺe w pokoleniách wáƺych / ſtárśi wáśi / y przełożeni wáśi / wƺyſcy mężowie Izráelſcy ;
Dziátki wáƺe / żony wáƺe / y przychodzień twój / który mieƺká w pośrodku obozu twego / od tego / który drwá rąbie / áż do tego / co wodę cżerpie ;
Abyś wƺedł w przymierze Páná / Bogá twego / y w przyśięgę jego / którą Pán / Bóg twój / ſtánowi z tobą dziś ;
Aby ćię ſobie dziś wyſtáwił zá lud / á ſám żeby tobie był zá Bogá / jákoć powiedziáł / y jáko przyśiągł ojcom twoim Abráhámowi / Izáákowi y Jákóbowi.
A nie tylko z wámi ſámymi já ſtánowię to przymierze / y tę przyśięgę ;
Ale z káżdym / który tu ſtoi dziś z námi przed Pánem / Bogiem náƺym / y z káżdym / którego tu niemáƺ z námi dziś.
Wy bowiem wiećie / jákośmy mieƺkáli w ziemi Egipſkiej / y jákośmy przeƺli przez pośrodek národów / któreśćie przeƺli ;
Y widzieliśćie obrzydliwośći ich / y báłwány ich / drewno y kámień / ſrebro y złoto / które ſą przy nich.
Niechże nie będzie między wámi mąż / álbo niewiáſtá / álbo dom / álbo pokolenie / któregoby śię dziś ſerce odwróćiło od Páná / Bogá náƺego / żeby ƺedł / á ſłużył bogom národów onych / by ſnáć nie był między wámi jáki korzeń / wydáwájący żółć y piołun ;
Bo ktoby ſłucháł ſłów przeklęſtwá tego / á błogoſłáwiłby ſobie w ſercu ſwem / mówiąc : Pokój mi będzie / choćiáż według zdániá ſercá mego chodzić będę / przydáwájąc pijáną do uprágnionej.
Ale Pán nie chce odpuśćić tákowemu ; lecż záráz zápáli śię gniew Páńſki / á zápálcżywość jego przećiwko mężowi temu / y przypádnie náń wƺyſtko przeklęſtwo / które jeſt nápiſáne w tych kśięgách / y wygłádzi Pán imię jego pod niebem ;
Y wyłącży go Pán ná złe jego ze wƺyſtkich pokoleń Izráelſkich / według káżdego przeklęſtwá przymierzá nápiſánego w kśięgách zákonu tego /
Ták / że rzecże náród potomny / ſynowie wáśi / którzy powſtáną po wáſ / y cudzoziemiec / który przyjdzie z ziemi dálekiej ( gdy obácżą plági ziemi tej / y niemocy jej / któremi ją zárázi Pán ;
Bo śiárká y ſól wypáli tę wƺyſtkę ziemię / że nie będzie ośiewáná / áni też rodzić będzie / y nie zrośnie ná niej żádná tráwá / ná kƺtáłt podwróceniá Sodomy y Gomorry / Adámy y Seboim / które podwróćił Pán w gniewie ſwym / y w popędliwośći ſwej. )
Ták rzeką wƺyſcy národowie : Przecżże ták Pán ucżynił tej ziemi? cóż to zá gniew tej wielkiej zápálcżywośći?
Y odpowiedzą : Przeto / że opuśćili przymierze Páná / Bogá ojców ſwych / które poſtánowił z nimi / gdy je wywiódł z ziemi Egipſkiej.
Abowiem poƺli / á ſłużyli bogom cudzym / y kłániáli śię im ; bogom / których oni nie ználi / y którzy im nic nie udzielili.
Przetoż zápálił śię Pán gniewem przećiwko tej ziemi / áby przywiódł ná nie wƺelkie przeklęſtwo nápiſáne w tych kśięgách.
Y wykorzenił je Pán z ziemi ich w gniewie y w zápálcżywośći / á w popędliwośći wielkiej / y wyrzućił je do ziemi obcej / jáko dziś widzićie.
Rzecży tájemne náleżą Pánu / Bogu náƺemu / á jáwne nám y ſynom náƺym áż ná wieki / ábyśmy cżynili wƺyſtkie ſłowá zákonu tego.
