Teć ſą ſłowá przymierzá / które przykázáł Pán Mojżeƺowi poſtánowić z ſynámi Izráelſkimi w źiemi Moábſkiej nád ono przymierze / które poſtánowił z nimi ná Horeb.
Zwołáł tedy Mojżeƺ wƺyſtkiego Izráelá / y mówił do nich : Wyśćie widźieli wƺyſtko / co cżynił Pán przed ocżymá wáƺemi w źiemi Egipſkiej Fáráonowi / y wƺyſtkim ſługom jego / y cáłej źiemi jego ;
Pokuſy wielkie / które widźiáły ocży twoje / znáki / y cudá one wielkie.
Ale nie dáł wám Pán ſercá ku zrozumieniu / y ocżu ku widzeniu / áni uƺu ku ſłyƺeniu / áż do dniá dźiśiejƺego.
Y prowádźiłem was przez cżterdźieśći lát po puƺcży ; nie wiotƺáły ƺáty wáƺe ná was / y obuwie wáƺe nie wiotƺáło ná nogách wáƺych.
Chlebáśćie nie jedli / y winá / y mocnego nápoju nie piliśćie / ábyśćie wiedźieli / żem Ja Pán / Bóg wáƺ.
A gdyśćie przyƺli ná to miejſce / wyćiągnął Sehon / król Heſebońſki / y Og / król Báſáńſki / przećiwko nám ku bitwie / y poráźiliśmy je.
A wźiąwƺy źiemię ich / podáliśmy ją w dźiedźictwo Rubenitom / y Gádytom / y połowie pokoleniá Mánáſeſowego.
A ták przeſtrzegájćież ſłów przymierzá tego / y cżyńćie je / áby śię wám ƺcżęśliwie powodźiło wƺyſtko / co cżynić będźiećie.
Wy ſtoićie dźiś wƺyſcy przed Pánem / Bogiem wáƺym / kśiążętá wáƺe w pokoleniách wáƺych / ſtárśi wáśi / y przełożeni wáśi / wƺyſcy mężowie Izráelſcy ;
Dźiátki wáƺe / żony wáƺe / y przychodźień twój / który mieƺká w pośrodku obozu twego / od tego / który drwá rąbie / áż do tego / co wodę cżerpie ;
Abyś wƺedł w przymierze Páná / Bogá twego / y w przyśięgę jego / którą Pán / Bóg twój / ſtánowi z tobą dźiś ;
Aby ćię ſobie dźiś wyſtáwił zá lud / á ſám żeby tobie był zá Bogá / jákoć powiedźiáł / y jáko przyśiągł ojcom twoim Abráhámowi / Izáákowi y Jákóbowi.
A nie tylko z wámi ſámymi ja ſtánowię to przymierze / y tę przyśięgę ;
Ale z káżdym / który tu ſtoi dźiś z námi przed Pánem / Bogiem náƺym / y z káżdym / którego tu niemáƺ z námi dźiś.
Wy bowiem wiećie / jákośmy mieƺkáli w źiemi Egipſkiej / y jákośmy przeƺli przez pośrodek národów / któreśćie przeƺli ;
Y widźieliśćie obrzydliwośći ich / y báłwány ich / drewno y kámień / ſrebro y złoto / które ſą przy nich.
Niechże nie będźie między wámi mąż / álbo niewiáſtá / álbo dom / álbo pokolenie / któregoby śię dźiś ſerce odwróćiło od Páná / Bogá náƺego / żeby ƺedł / á ſłużył bogom národów onych / by ſnáć nie był między wámi jáki korzeń / wydáwájący żółć y piołun ;
Bo ktoby ſłucháł ſłów przeklęſtwá tego / á błogoſłáwiłby ſobie w ſercu ſwem / mówiąc : Pokój mi będźie / choćiáż według zdániá ſercá mego chodźić będę / przydáwájąc pijáną do uprágnioney.
Ale Pán nie chce odpuśćić tákowemu ; lecż záráz zápáli śię gniew Páńſki / á zápálcżywość jego przećiwko mężowi temu / y przypádnie náń wƺyſtko przeklęſtwo / które jeſt nápiſáne w tych kśięgách / y wygłádźi Pán imię jego pod niebem ;
Y wyłącży go Pán ná złe jego ze wƺyſtkich pokoleń Izráelſkich / według káżdego przeklęſtwá przymierzá nápiſánego w kśięgách zakonu tego /
Ták / że rzecże náród potomny / ſynowie wáśi / którzy powſtáną po was / y cudzoźiemiec / który przyjdźie z źiemi dálekiej ( gdy obácżą plági źiemi tey / y niemocy jey / któremi ją záráźi Pán ;
Bo śiárká y ſól wypáli tę wƺyſtkę źiemię / że nie będźie ośiewáná / áni też rodźić będźie / y nie zrośnie ná niej żádná tráwá / ná kƺtáłt podwróceniá Sodomy y Gomorry / Adámy y Seboim / które podwróćił Pán w gniewie ſwym / y w popędliwośći ſwej. )
Ták rzeką wƺyſcy národowie : Przecżże ták Pán ucżynił tey źiemi? cóż to zá gniew tey wielkiej zápálcżywośći?
Y odpowiedzą : Przeto / że opuśćili przymierze Páná / Bogá ojców ſwych / które poſtánowił z nimi / gdy je wywiódł z źiemi Egipſkiej.
Abowiem poƺli / á ſłużyli bogom cudzym / y kłániáli śię im ; bogom / których oni nie ználi / y którzy im nic nie udźielili.
Przetoż zápálił śię Pán gniewem przećiwko tey źiemi / áby przywiódł ná nie wƺelkie przeklęſtwo nápiſáne w tych kśięgách.
Y wykorzenił je Pán z źiemi ich w gniewie y w zápálcżywośći / á w popędliwośći wielkiej / y wyrzućił je do źiemi obcey / jáko dźiś widźićie.
Rzecży tájemne náleżą Pánu / Bogu náƺemu / á jáwne nám y ſynom náƺym áż ná wieki / ábyśmy cżynili wƺyſtkie ſłowá zakonu tego.
