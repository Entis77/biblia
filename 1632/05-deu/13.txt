Gdyby powſtał miedzy wámi prorok / álbo ſny miewájący / y ukázáłćiby znák / álbo cud ;
Y ſtáłby śię on znák álbo cud / o którym ći powiedźiał / á rzekłby : Pójdźmy zá bogámi obcymi / których ty nie znáƺ / á ſłużmy im :
Nie uſłucháƺ ſłów proroká tego / áni tego / co ſny miewá / gdyż was doświádcżá Pán / Bóg wáƺ / áby wiedźiał / jeſli wy miłujećie Páná / Bogá wáƺego / ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej.
Páná / Bogá wáƺego / náśládujćie / y onego śię bójćie / á przykázáń jego ſtrzegąc / y głoſu jego ſłuchájąc / ſłużćie mu / y przy nim trwájćie.
Ale prorok on / álbo miewájący ſny / zábity będźie ; bo to mówił / cżemby was odwiódł od Páná Bogá wáƺego / ( który was wywiódł z źiemi Egipſkiej / y odkupił ćię z domu niewoli / ) áby ćię zráźił z drogi / którą przykázał tobie Pán / Bóg twój / żebyś nią chodźił ; á ták wykorzeniƺ to złe z pośrzodku śiebie.
Jeſliby ćię też zwodźił brát twój / ſyn mátki twojey / álbo ſyn twój / álbo córká twojá / álbo żoná łoná twego / álbo przyjáćiel twój / którybyć był miły / jáko duƺá twojá / potájemnie mówiąc : Pójdźmy / á ſłużmy bogom obcym / którycheś nie znał ty / áni ojcowie twoi ;
Z bogów tych národów / które około was ſą / którzy bliſko ſą ćiebie / álbo dáleko od ćiebie / od końcá źiemi / y áż do końcá źiemi :
Nie pozwoliƺ mu / áni go uſłucháƺ / áni mu folgowáć będźie oko twoje / áni śię zmiłujeƺ nád nim / áni go utáiƺ ;
Ale koniecżnie zábijeƺ go ; ręká twojá nájpierwƺá nád nim będźie / ná zábićie jego / á ręká wƺyſtkiego ludu potem.
Y ukámionujeƺ go áż ná śmierć / poniewáż chćiał ćię odwieść od Páná / Bogá twego / który ćię wywiódł z źiemi Egipſkiej / z domu niewoli ;
Aby wƺyſtek Izráel uſłyƺáwƺy / bał śię / á nie cżynił więcey nic podobnego rzecży tey bárzo złej miedzy wámi.
A jeſlibyś uſłyƺał / żeby w któremkolwiek mieśćie twojim / które Pán Bóg twój / dáwá tobie ku mieƺkániu / ktokolwiek rzekł :
Wyƺli niektórzy ludźie przewrotni z pośrzodku ćiebie / á podwiedli obywátele miáſtá ſwego mówiąc : Pójdźmy / á ſłużmy bogom obcym / których nie znáćie :
Tedy będźieƺ ƺukał y dochodźił / á pilnie śię wywiádował ; á jeſli to práwdá / á rzecż pewná / że śię ſtáłá obrzydliwość táká miedzy wámi.
Koniecżnie wytráćiƺ obywátele miáſtá onego oſtrzem miecżá / zgłádźiƺ je / y wƺyſtko co w nim / y bydło jego pobijeƺ oſtrzem miecżá.
A wƺyſtkie łupy jego zbierzeƺ w pośród ulicy jego / y ſpáliƺ ogniem miáſto y z onym wƺyſtkim łupem do ƺcżętu Pánu / Bogu twemu ; y będźie mogiłą wiecżną / á nie będźie budowáne więcey.
Y nie zoſtánie w rękách twojich nic z onych rzecży przeklętych / áby śię odwróćił Pán od gniewu popędliwośći ſwojey / á ucżynił z tobą miłośierdźie / y zmiłował śię nád tobą / á rozmnożył ćię / jáko przyśiągł ojcom twoim.
Gdy ſłucháć będźieƺ głoſu Páná / Bogá twego / záchowywując wƺyſtkie przykázánia jego / które ja dźiś przykázuję tobie / żebyś cżynił / co práwego jeſt przed ocżymá Páná / Bogá twego.
