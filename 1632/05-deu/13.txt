Gdyby powſtáł między wámi prorok / álbo ſny miewájący / y ukázáłćiby znák / álbo cud ;
Y ſtáłby śię on znák álbo cud / o którym ći powiedziáł / á rzekłby : Pójdźmy zá bogámi obcymi / których ty nie znáƺ / á ſłużmy im :
Nie uſłucháƺ ſłów proroká tego / áni tego / co ſny miewá / gdyż wáſ doświádcżá Pán / Bóg wáƺ / áby wiedziáł / jeźli wy miłujećie Páná / Bogá wáƺego / ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej.
Páná / Bogá wáƺego / náśládujćie / y onego śię bójćie / á przykázáń jego ſtrzegąc / y głoſu jego ſłuchájąc / ſłużćie mu / y przy nim trwájćie.
Ale prorok on / álbo miewájący ſny / zábity będzie ; bo to mówił / cżemby wáſ odwiódł od Páná Bogá wáƺego / ( który wáſ wywiódł z ziemi Egipſkiej / y odkupił ćię z domu niewoli / ) áby ćię zráził z drogi / którą przykázáł tobie Pán / Bóg twój / żebyś nią chodził ; á ták wykorzeniƺ to złe z pośrodku śiebie.
Jeźliby ćię też zwodził brát twój / ſyn mátki twojej / álbo ſyn twój / álbo córká twojá / álbo żoná łoná twego / álbo przyjáćiel twój / którybyć był miły / jáko duƺá twojá / potájemnie mówiąc : Pójdźmy / á ſłużmy bogom obcym / którycheś nie znáł ty / áni ojcowie twoi ;
Z bogów tych národów / które około wáſ ſą / którzy bliſko ſą ćiebie / álbo dáleko od ćiebie / od końcá ziemi / y áż do końcá ziemi :
Nie pozwoliƺ mu / áni go uſłucháƺ / áni mu folgowáć będzie oko twoje / áni śię zmiłujeƺ nád nim / áni go utáiƺ ;
Ale koniecżnie zábijeƺ go ; ręká twojá nájpierwƺá nád nim będzie / ná zábićie jego / á ręká wƺyſtkiego ludu potem.
Y ukámionujeƺ go áż ná śmierć / poniewáż chćiáł ćię odwieść od Páná / Bogá twego / który ćię wywiódł z ziemi Egipſkiej / z domu niewoli ;
Aby wƺyſtek Izráel uſłyƺáwƺy / báł śię / á nie cżynił więcej nic podobnego rzecży tej bárdzo złej między wámi.
A jeźlibyś uſłyƺáł / żeby w któremkolwiek mieśćie twojim / które Pán Bóg twój / dáwá tobie ku mieƺkániu / ktokolwiek rzekł :
Wyƺli niektórzy ludzie przewrotni z pośrodku ćiebie / á podwiedli obywátele miáſtá ſwego mówiąc : Pójdźmy / á ſłużmy bogom obcym / których nie znáćie :
Tedy będzieƺ ƺukáł y dochodził / á pilnie śię wywiádowáł ; á jeźli to práwdá / á rzecż pewná / że śię ſtáłá obrzydliwość táká między wámi.
Koniecżnie wytráćiƺ obywátele miáſtá onego oſtrzem miecżá / zgłádziƺ je / y wƺyſtko co w nim / y bydło jego pobijeƺ oſtrzem miecżá.
A wƺyſtkie łupy jego zbierzeƺ w pośród ulicy jego / y ſpáliƺ ogniem miáſto y z onym wƺyſtkim łupem do ƺcżętu Pánu / Bogu twemu ; y będzie mogiłą wiecżną / á nie będzie budowáne więcej.
Y nie zoſtánie w rękách twoich nic z onych rzecży przeklętych / áby śię odwróćił Pán od gniewu popędliwośći ſwojej / á ucżynił z tobą miłośierdzie / y zmiłowáł śię nád tobą / á rozmnożył ćię / jáko przyśiągł ojcom twoim.
Gdy ſłucháć będzieƺ głoſu Páná / Bogá twego / záchowywując wƺyſtkie przykázániá jego / które já dziś przykázuję tobie / żebyś cżynił / co práwego jeſt przed ocżymá Páná / Bogá twego.
