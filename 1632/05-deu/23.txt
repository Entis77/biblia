Nie wnijdzie wypukły / áni trzebieniec do zgromádzeniá Páńſkiego.
Ani wnijdzie niepocżćiwego łożá ſyn do zgromádzeniá Páńſkiego / y dzieśiąte pokolenie jego nie wnijdzie do zgromádzeniá Páńſkiego.
Nie wnijdzie też Ammonitá / y Moábcżyk do zgromádzeniá Páńſkiego / áni dzieśiąte pokolenie ich nie wnijdzie do zgromádzeniá Páńſkiego / áż ná wieki ;
Dlá tego / że wám nie zábieżeli z chlebem / áni z wodą w drodze / gdyśćie ƺli z Egiptu / á że nájęli zá zápłátę przećiw tobie Báláámá / ſyná Beorowego / z Pethor Meſopotámii Syryjſkiej / áby ćię przeklináł.
Acż nie chćiáł Pán / Bóg twój / wyſłucháć Báláámá / ále obróćił Pán / Bóg twój / tobie jego przeklęſtwo w błogoſłáwieńſtwo ; bo ćię umiłowáł Pán / Bóg twój.
Nie ƺukáj pokoju ich / áni dobrego ich / po wƺyſtkie dni twoje ná wieki.
Nie będzieƺ śię brzydził Idumejcżykiem / bo brátem twoim jeſt ; nie będzieƺ śię brzydził Egipcżykiem / boś był przychodniem w ziemi jego.
Synowie / którzy śię im zrodzą w trzećiem pokoleniu / wnijdą do zgromádzeniá Páńſkiego.
Gdy śię ruƺyƺ z wojſkiem náprzećiwko nieprzyjáćiołom twoim / ſtrzeż śię od wƺelkiej złej rzecży.
Jeźliby był między wámi kto / coby był niecżyſtym z przygody nocnej / wynijdzie zá obóz / á nie wróći śię do niego.
A ku wiecżorowi omyje śię wodą / á po zájśćiu ſłońcá wnijdzie do obozu.
Będziećie też mieli miejſce zá obozem / gdzie będziećie wychodzili ná potrzebę przyrodzoną ;
Y będzieƺ miáł rydlik między nácżyniem ſwojim ; á gdybyś chćiáł uśiąść dlá potrzeby / wykopieƺ nim dołek / á obróćiwƺy śię / zágrzebieƺ plugáſtwo twoje.
Abowiem Pán / Bóg twój / chodzi w pośrodku obozu twego / áby ćię wyrwáł / y podáł ći nieprzyjáćioły twoje ; przetoż niech będzie obóz twój święty / áby nie widziáł przy tobie ſproſnośći jákiej / dlá którejby śię odwróćił od ćiebie.
Nie wydáƺ ſługi pánu jego / któryby śię ućiekł do ćiebie od páná ſwego.
Z tobą będzie mieƺkáł w pośrodku ćiebie / ná miejſcu / które ſobie obierze w jednem z miáſt twoich / gdzieby mu śię podobáło ; nie ucżyniƺ mu gwáłtu.
Nie będzie nierządnicá z córek Izráelſkich / áni będzie nierządnik z ſynów Izráelſkich.
Nie wnoś zápłáty nierządnicy / áni zápłáty pſá w dom Páná / Bogá twego / zá jákikolwiek ślub ; bo obrzydliwośćią u Páná / Bogá twego / jeſt to oboje.
Nie dáƺ ná lichwę brátu twemu / áni pieniędzy / áni żywnośći / áni jákiejkolwiek rzecży / którą dáwáją ná lichwę.
Cudzoziemcowi ná lichwę dáwáć możeƺ ; ále brátu twemu ná lichwę nie dáƺ / ábyć błogoſłáwił Pán / Bóg twój / w káżdej ſpráwie / do której śćiągnieƺ rękę twoję w ziemi / do której wnijdzieƺ / ábyś ją pośiádł.
Jeźlibyś ślubowáł ślub Pánu / Bogu twemu / nie omieƺkiwájże oddáwáć go ; bo koniecżnie upomni śię go Pán / Bóg twój / od ćiebie / á będzie ná tobie grzech.
A jeźli nie będzieƺ ślubowáł / nie będzie ná tobie grzechu.
To co wynijdzie z uſt twoich / wypełniƺ / y ucżyniƺ / jákoś ślubowáł Pánu / Bogu twemu / dobrowolnie / coś wymówił uſty twemi.
Gdy wnijdzieƺ do winnicy bliźniego twego / jedz jágody / ileć śię podobá / áż do ſytośći ; ále do nácżyniá twego nie bierz.
Tákże gdy wnijdzieƺ między zboże bliźniego twego / tedy nárwieƺ kłoſów ręką twą ; ále śierpá nie zápuƺcżáj w zboże bliźniego twego.
