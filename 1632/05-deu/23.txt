Nie wnijdźie wypukły / áni trzebieniec do zgromádzenia Páńſkiego.
Ani wnijdźie niepocżćiwego łożá ſyn do zgromádzenia Páńſkiego / y dźieśiąte pokolenie jego nie wnijdźie do zgromádzenia Páńſkiego.
Nie wnijdźie też Ammonitá / y Moábcżyk do zgromádzenia Páńſkiego / áni dźieśiąte pokolenie ich nie wnijdźie do zgromádzenia Páńſkiego / áż ná wieki ;
Dla tego / że wam nie zábieżeli z chlebem / áni z wodą w drodze / gdyśćie ƺli z Egiptu / á że nájęli zá zápłátę przećiw tobie Báláámá / ſyná Beorowego / z Pethor Meſopotámii Syryjſkiej / áby ćię przeklinał.
Acż nie chćiał Pán / Bóg twój / wyſłucháć Báláámá / ále obróćił Pán / Bóg twój / tobie jego przeklęſtwo w błogoſłáwieńſtwo ; bo ćię umiłował Pán / Bóg twój.
Nie ƺukáj pokoju ich / áni dobrego ich / po wƺyſtkie dni twoje ná wieki.
Nie będźieƺ śię brzydźił Idumejcżykiem / bo brátem twoim jeſt ; nie będźieƺ śię brzydźił Egipcżykiem / boś był przychodniem w źiemi jego.
Synowie / którzy śię im zrodzą w trzećiem pokoleniu / wnijdą do zgromádzenia Páńſkiego.
Gdy śię ruƺyƺ z wojſkiem náprzećiwko nieprzyjáćiołom twoim / ſtrzeż śię od wƺelkiej złej rzecży.
Jeſliby był miedzy wámi kto / coby był niecżyſtym z przygody nocney / wynijdźie zá obóz / á nie wróći śię do niego.
A ku wiecżorowi omyje śię wodą / á po zájśćiu ſłońcá wnijdźie do obozu.
Będźiećie też mieli miejſce zá obozem / gdźie będźiećie wychodźili ná potrzebę przyrodzoną ;
Y będźieƺ miał rydlik miedzy nácżyniem ſwojim ; á gdybyś chćiał uśiąść dla potrzeby / wykopieƺ nim dołek / á obróćiwƺy śię / zágrzebieƺ plugáſtwo twoje.
Abowiem Pán / Bóg twój / chodźi w pośrzodku obozu twego / áby ćię wyrwał / y podał ći nieprzyjáćioły twoje ; przetoż niech będźie obóz twój święty / áby nie widźiał przy tobie ſproſnośći jákiej / dla którejby śię odwróćił od ćiebie.
Nie wydáƺ ſługi pánu jego / któryby śię ućiekł do ćiebie od páná ſwego.
Z tobą będźie mieƺkał w pośrzodku ćiebie / ná miejſcu / które ſobie obierze w jednem z miaſt twojich / gdźieby mu śię podobáło ; nie ucżyniƺ mu gwáłtu.
Nie będźie nierządnicá z córek Izráelſkich / áni będźie nierządnik z ſynów Izráelſkich.
Nie wnoś zápłáty nierządnicy / áni zápłáty pſá w dom Páná / Bogá twego / zá jákikolwiek ślub ; bo obrzydliwośćią u Páná / Bogá twego / jeſt to oboje.
Nie dáƺ ná lichwę brátu twemu / áni pieniędzy / áni żywnośći / áni jákiejkolwiek rzecży / którą dáwáją ná lichwę.
Cudzoźiemcowi ná lichwę dáwáć możeƺ ; ále brátu twemu ná lichwę nie dáƺ / ábyć błogoſłáwił Pán / Bóg twój / w káżdej ſpráwie / do którey śćiągnieƺ rękę twoję w źiemi / do którey wnijdźieƺ / ábyś ją pośiádł.
Jeſlibyś ślubował ślub Pánu / Bogu twemu / nie omieƺkiwájże oddáwáć go ; bo koniecżnie upomni śię go Pán / Bóg twój / od ćiebie / á będźie ná tobie grzech.
A jeſli nie będźieƺ ślubował / nie będźie ná tobie grzechu.
To co wynijdźie z uſt twojich / wypełniƺ / y ucżyniƺ / jákoś ślubował Pánu / Bogu twemu / dobrowolnie / coś wymówił uſty twemi.
Gdy wnijdźieƺ do winnicy bliźniego twego / jedz jágody / ileć śię podobá / áż do ſytośći ; ále do nacżynia twego nie bierz.
Tákże gdy wnijdźieƺ miedzy zboże bliźniego twego / tedy nárwieƺ kłoſów ręką twą ; ále śierpá nie zápuƺcżáj w zboże bliźniego twego.
