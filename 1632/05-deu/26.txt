Gdy wnijdzieƺ do ziemi / którą Pán / Bóg twój / dáwá tobie w dziedzictwo / á ośiędzieƺ ją / y mieƺkáć w niej będzieƺ :
Tedy weźmieƺ pierwoćiny ze wƺyſtkich owoców ziemi / które ofiárowáć będzieƺ z ziemi twojej / którą Pán / Bóg twój / dáwá tobie / y włożyƺ je w koƺ / á pójdzieƺ ná miejſce / któreby obráł Pán / Bóg twój / áby tám mieƺkáło imię jego.
A przyƺedłƺy do kápłáná / który ná ten cżáſ będzie / rzecżeƺ do niego : Wyznáwám dziś Pánu / Bogu twemu / żem wƺedł do ziemi / o którą przyśiągł Pán ojcom náƺym / że ją nám dáć miáł.
Tedy wziąwƺy kápłán koƺ z ręki twojej / poſtáwi go przed ołtárzem Páná / Bogá twego.
Támże odpowieƺ / y rzecżeƺ przed oblicżem Páná / Bogá twego : Ojćiec mój był ubogi Syryjcżyk / y zſtąpił do Egiptu á pielgrzymowáł tám w málucżkim pocżćie / y ſtáł śię tám w náród wielki / możny / y obfity.
A gdy śię z námi źle obchodzili Egipcżánie / trápiąc náſ / y kłádąc ná náſ niewolą ćiężką /
Tedyśmy wołáli do Páná / Bogá ojców náƺych / y wyſłucháł Pán głoſ náƺ / á wejrzáł ná utrápienie náƺe / y ná prácę náƺę / y ná ućiſk náƺ ;
Y wywiódł náſ Pán z Egiptu ręką możną / y rámieniem wyćiągnionem / y w ſtráchu wielkim / y w znákách / y w cudách ;
Y przyprowádził náſ ná to miejſce / á podáł nám tę ziemię / ziemię opływájącą mlekiem y miodem.
A teráz oto przynioſłem pierwoćiny z owoców ziemi / którąś mi dáł / o Pánie! y zoſtáwiƺ ono przed Pánem / Bogiem twoim / y pokłoniƺ śię przed Pánem / Bogiem twoim ;
Y będzieƺ śię weſelił ze wƺyſtkich dóbr / któreć dáł Pán / Bóg twój / y domowi twemu / ty y Lewitá / y przychodzień / który jeſt w pośrodku ćiebie.
A gdy z pełná oddáƺ dzieśięćinę wƺyſtkich dzieśięćin urodzájów twoich roku trzećiego / który jeſt rokiem dzieśięćin / á dáƺ Lewićie / przychodniowi / śieroćie / y wdowie / y będą jedli w brámách twoich / y nájedzą śię ;
Tedy rzecżeƺ przed oblicżem Páná / Bogá twego : Wynioſłem co jeſt poświęcone z domu ſwego / y dáłem też z tego Lewićie / y przychodniowi / śieroćie y wdowie / według wƺelkiego przykázániá twego / któreś mi przykázáł ; nie przeſtąpiłem przykázániá twego / ánim go zápomináł ;
Nie jádłem w ſmętku moim z niego / ánim odjął z niego ná poſpolite używánie / ánim też dáł z niego ná pogrzeb ; byłem poſłuƺny głoſowi Páná / Bogá mego / ucżyniłem wƺyſtko / coś mi przykázáł.
Spojrzyjże z mieƺkániá świętego twojego z niebá / á błogoſłáw ludowi twemu Izráelſkiemu y ziemi / którąś nám dáł / jákoś przyśiągł ojcom náƺym / ziemi opływájącej mlekiem y miodem.
Dziś Pán / Bóg twój / rozkázuje tobie / ábyś záchowáł te uſtáwy y ſądy ; przeſtrzegájże tedy / á cżyń je ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej.
Przy Pánuś śię dziś opowiedziáł / ábyć był zá Bogá / y żebyś chodził drogámi jego / y ſtrzegł uſtáw jego / y przykázáń jego / y ſądów jego / y żebyś poſłuƺny był głoſowi jego.
Pán śię też dziś opowiedziáł przy tobie / ábyś mu był zá lud oſobliwy / jáko mówił do ćiebie / y żebyś ſtrzegł wƺyſtkich przykázáń jego ;
Y żeby ćię wywyżƺył nád wƺyſtkie národy / które ucżynił w chwále y w ſłáwie / y w zácnośći / á żebyś był ludem świętym Pánu / Bogu twemu / jáko mówił.
