Gdy wnijdźieƺ do źiemi / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo / á ośiędźieƺ ją / y mieƺkáć w niej będźieƺ :
Tedy weźmieƺ pierwoćiny ze wƺyſtkich owoców źiemi / które ofiárowáć będźieƺ z źiemi twojey / którą Pán / Bóg twój / dáwá tobie / y włożyƺ je w koƺ / á pójdźieƺ ná miejſce / któreby obrał Pán / Bóg twój / áby tám mieƺkáło imię jego.
A przyƺedƺy do kápłáná / który ná ten cżás będźie / rzecżeƺ do niego : Wyznáwám dźiś Pánu / Bogu twemu / żem wƺedł do źiemi / o którą przyśiągł Pán ojcom náƺym / że ją nam dáć miał.
Tedy wźiąwƺy kápłán koƺ z ręki twojey / poſtáwi go przed ołtárzem Páná / Bogá twego.
Támże odpowieƺ / y rzecżeƺ przed oblicżem Páná / Bogá twego : Oćiec mój był ubogi Syryjcżyk / y zſtąpił do Egiptu á pielgrzymował tám w málucżkim pocżćie / y ſtał śię tám w náród wielki / możny / y obfity.
A gdy śię z námi źle obchodźili Egipcżánie / trápiąc nas / y kłádąc ná nas niewolą ćiężką /
Tedyśmy wołáli do Páná / Bogá ojców náƺych / y wyſłuchał Pán głos náƺ / á wejrzał ná utrápienie náƺe / y ná prácę náƺę / y ná ućiſk náƺ ;
Y wywiódł nas Pán z Egiptu ręką możną / y rámieniem wyćiągnionem / y w ſtráchu wielkim / y w znákách / y w cudách ;
Y przyprowádźił nas ná to miejſce / á podał nam tę źiemię / źiemię opływájącą mlekiem y miodem.
A teraz oto przynioſłem pierwoćiny z owoców źiemi / którąś mi dał / o Pánie! y zoſtáwiƺ ono przed Pánem / Bogiem twoim / y pokłoniƺ śię przed Pánem / Bogiem twoim ;
Y będźieƺ śię weſelił ze wƺyſtkich dóbr / któreć dał Pán / Bóg twój / y domowi twemu / ty y Lewitá / y przychodźień / który jeſt w pośrzodku ćiebie.
A gdy z pełna oddáƺ dźieśięćinę wƺyſtkich dźieśięćin urodzájów twojich roku trzećiego / który jeſt rokiem dźieśięćin / á dáƺ Lewićie / przychodniowi / śieroćie / y wdowie / y będą jedli w brámách twojich / y nájedzą śię ;
Tedy rzecżeƺ przed oblicżem Páná / Bogá twego : Wynioſłem co jeſt poświęcone z domu ſwego / y dáłem też z tego Lewićie / y przychodniowi / śieroćie y wdowie / według wƺelkiego przykázánia twego / któreś mi przykázał ; nie przeſtąpiłem przykázánia twego / ánim go zápominał ;
Nie jádłem w ſmętku moim z niego / ánim odjął z niego ná poſpolite używánie / ánim też dał z niego ná pogrzeb ; byłem poſłuƺny głoſowi Páná / Bogá mego / ucżyniłem wƺyſtko / coś mi przykázał.
Spojrzyjże z mieƺkánia świętego twojego z niebá / á błogoſłáw ludowi twemu Izráelſkiemu y źiemi / którąś nam dał / jákoś przyśiągł ojcom náƺym / źiemi opływájącey mlekiem y miodem.
Dźiś Pán / Bóg twój / rozkázuje tobie / ábyś záchował te uſtáwy y ſądy ; przeſtrzegájże tedy / á cżyń je ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey.
Przy Pánuś śię dźiś opowiedźiał / ábyć był zá Bogá / y żebyś chodźił drogámi jego / y ſtrzegł uſtáw jego / y przykázáń jego / y ſądów jego / y żebyś poſłuƺny był głoſowi jego.
Pán śię też dźiś opowiedźiał przy tobie / ábyś mu był zá lud oſobliwy / jáko mówił do ćiebie / y żebyś ſtrzegł wƺyſtkich przykázáń jego ;
Y żeby ćię wywyżƺył nád wƺyſtkie národy / które ucżynił w chwále y w ſłáwie / y w zácnośći / á żebyś był ludem świętym Pánu / Bogu twemu / jáko mówił.
