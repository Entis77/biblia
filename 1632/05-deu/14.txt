Synámi jeſteśćie Páná / Bogá wáƺego : nie będźiećie śię rzezáć / áni cżynić łyśiny miedzy ocżymá wáƺemi nád umárłym ;
Abowiemeś ludem świętym Pánu / Bogu twemu / y obrał ćię Pán / ábyś mu był zá lud oſobliwy ze wƺyſtkich narodów / którzy ſą ná oblicżu źiemi.
Nie będźieƺ jádał żádney obrzydliwośći.
Teć ſą zwierzętá / które jeść będźiećie : Woły / owce / y kozy /
Jeleniá / y ſárnę / y báwołu / y dźikiego kozłá / y jednorożcá / y łośiá / y kózkę ſkálną.
Y wƺelkie zwierzę / które ma rozdźielone kopyto / ták że śię ná dwá kopytá dźieli ſtopá jego / y które przeżuwá miedzy zwierzęty / jeść je będźiećie.
A wƺákże z tych / które przeżuwáją / y które rozdźielone kopytá máją / jeść nie będźiećie / wielbłądá / y zájącá / y króliká ; ábowiem choćiáż te przeżuwáją / ále kopyt rozdwojonych nie máją ; niecżyſte wam będą.
Tákże świniá / choć ma kopyto rozdwojone / ále iż nie przeżuwá / niecżyſtá wam będźie ; mięſá ich jeść nie będźiećie / y śćierwu śię ich nie dotkniećie.
To záś jeść będźiećie ze wƺyſtkich rzecży / które ſą w wodách ; cokolwiek ma łuſkę y ſkrzele / jeść będźiećie.
Ale wƺyſtkiego / co niemá ſkrzeli / áni łuſki / jeść nie będźiećie / niecżyſte wam będźie.
Wƺyſtko ptáſtwo cżyſte jeść będźiećie.
Te záśię ſą / których jeść nie będźiećie : Orłá / y gryfá / y morſkiego orłá.
Y ſokołá / y ſępá / y káni wedle rodzáju jey.
Ani żádnego kruká wedle rodzáju jego.
Ani ſtruśiá / áni ſowy / áni wodney káni / áni krogulcá według rodzáju jego.
Y rárogá / y lelká / y łábędźiá.
Y pelikáná / y porfiryjoná / y nurká.
Ani boćiáná áni ſojki według rodzáju ich / áni dudká áni nietoperzá.
Wƺelki też płáz ſkrzydláſty niecżyſty wam będźie / jeść go nie będźiećie.
Káżdego ptáká cżyſtego jeść będźiećie.
Nie będźiećie jeść żádney zdechliny ; przychodniowi / który jeſt w bramách twojich / dáƺ to / á jeść to będźie / álbo ſprzedáƺ cudzoźiemcowi ; boś ty lud święty Pánu / Bogu twemu / nie będźieƺ wárzył koźlęćiá w mleku mátki jego.
Ochotnie będźieƺ dáwał dźieśięćiny ze wƺyſtkiego urodzáju náśieniá twego / co śię urodźi ná polu ná káżdy rok.
A będźieƺ pożywał przed Pánem / Bogiem twoim / ná miejſcu / które obierze / żeby tám mieƺkáło imię jego / dźieśięćin zbożá twego / z winá twego / y z oliwy twojey / y z pierworodztwá wołów twojich / y z owiec twojich / ábyś śię ucżył báć Páná / Bogá twego / po wƺyſtkie dni.
A jeſliby dáleká byłá drogá ná ćię / żebyś tego tám donieść nie mógł / przeto / że odległe jeſt tobie ono miejſce / któreby obrał Pán / Bóg twój / ku mieƺkániu tám imieniowi ſwemu / gdyć błogoſłáwić będźie Pán / Bóg twój ;
Tedy to ſpieniężyƺ / á májąc záwiązáne pieniądze w rękách twojich / pójdźieƺ ná miejſce / które ſobie obierze Pán / Bóg twój.
Y zá one pieniądze nákupiƺ wƺyſtkiego / co śię upodobá duƺy twojey / wołów y owiec / y winá / y innego nápoju mocnego / y wƺyſtkiego / cżegoby pożądáłá duƺá twojá / á będźieƺ tám jádł przed Pánem / Bogiem twoim / y będźieƺ śię weſelił / ty y dom twój.
A Lewity / który mieƺká w bramách twojich / nie opuśćiƺ go / ponieważ niemá dźiáłu / áni dźiedźictwá z tobą.
A po wyśćiu káżdego trzećiego roku odłącżyƺ wƺyſtkie dźieśięćiny urodzáju twego onegoż roku / y złożyƺ je w bramách twojich.
Tedy przydźie Lewitá / który niemá dźiáłu y dźiedźictwá z tobą / y przychodźień / y śierotá / y wdowá / którzy ſą w bramách twojich / y będą jeść / y nájedzą śię / ábyć błogoſłáwił Pán / Bóg twój / w káżdej ſpráwie rąk twojich / którą cżynić będźieƺ.
