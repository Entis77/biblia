Synámi jeſteśćie Páná / Bogá wáƺego : nie będziećie śię rzezáć / áni cżynić łyśiny między ocżymá wáƺemi nád umárłym ;
Abowiemeś ludem świętym Pánu / Bogu twemu / y obráł ćię Pán / ábyś mu był zá lud oſobliwy ze wƺyſtkich národów / którzy ſą ná oblicżu ziemi.
Nie będzieƺ jádáł żádnej obrzydliwośći.
Teć ſą zwierzętá / które jeść będziećie : Woły / owce / y kozy /
Jeleniá / y ſárnę / y báwołu / y dzikiego kozłá / y jednorożcá / y łośiá / y kózkę ſkálną.
Y wƺelkie zwierzę / które má rozdzielone kopyto / ták że śię ná dwá kopytá dzieli ſtopá jego / y które przeżuwá między zwierzęty / jeść je będziećie.
A wƺákże z tych / które przeżuwáją / y które rozdzielone kopytá máją / jeść nie będziećie / wielbłądá / y zájącá / y króliká ; ábowiem choćiáż te przeżuwáją / ále kopyt rozdwojonych nie máją ; niecżyſte wám będą.
Tákże świniá / choć má kopyto rozdwojone / ále iż nie przeżuwá / niecżyſtá wám będzie ; mięſá ich jeść nie będziećie / y śćierwu śię ich nie dotkniećie.
To záś jeść będziećie ze wƺyſtkich rzecży / które ſą w wodách ; cokolwiek má łuſkę y ſkrzele / jeść będziećie.
Ale wƺyſtkiego / co niemá ſkrzeli / áni łuſki / jeść nie będziećie / niecżyſte wám będzie.
Wƺyſtko ptáſtwo cżyſte jeść będziećie.
Te záśię ſą / których jeść nie będziećie : Orłá / y gryfá / y morſkiego orłá.
Y ſokołá / y ſępá / y káni wedle rodzáju jej.
Ani żádnego kruká wedle rodzáju jego.
Ani ſtruśiá / áni ſowy / áni wodnej káni / áni krogulcá według rodzáju jego.
Y rárogá / y lelká / y łábędziá.
Y pelikáná / y porfiryjoná / y nurká.
Ani boćiáná áni ſojki według rodzáju ich / áni dudká áni nietoperzá.
Wƺelki też płáz ſkrzydláſty niecżyſty wám będzie / jeść go nie będziećie.
Káżdego ptáká cżyſtego jeść będziećie.
Nie będziećie jeść żádnej zdechliny ; przychodniowi / który jeſt w brámách twoich / dáƺ to / á jeść to będzie / álbo ſprzedáƺ cudzoziemcowi ; boś ty lud święty Pánu / Bogu twemu / nie będzieƺ wárzył koźlęćiá w mleku mátki jego.
Ochotnie będzieƺ dáwáł dzieśięćiny ze wƺyſtkiego urodzáju náśieniá twego / co śię urodzi ná polu ná káżdy rok.
A będzieƺ pożywáł przed Pánem / Bogiem twoim / ná miejſcu / które obierze / żeby tám mieƺkáło imię jego / dzieśięćin zbożá twego / z winá twego / y z oliwy twojej / y z pierworodztwá wołów twoich / y z owiec twoich / ábyś śię ucżył báć Páná / Bogá twego / po wƺyſtkie dni.
A jeźliby dáleká byłá drogá ná ćię / żebyś tego tám donieść nie mógł / przeto / że odległe jeſt tobie ono miejſce / któreby obráł Pán / Bóg twój / ku mieƺkániu tám imieniowi ſwemu / gdyć błogoſłáwić będzie Pán / Bóg twój ;
Tedy to ſpieniężyƺ / á májąc záwiązáne pieniądze w rękách twoich / pójdzieƺ ná miejſce / które ſobie obierze Pán / Bóg twój.
Y zá one pieniądze nákupiƺ wƺyſtkiego / co śię upodobá duƺy twojej / wołów y owiec / y winá / y innego nápoju mocnego / y wƺyſtkiego / cżegoby pożądáłá duƺá twojá / á będzieƺ tám jádł przed Pánem / Bogiem twoim / y będzieƺ śię weſelił / ty y dom twój.
A Lewity / który mieƺká w brámách twoich / nie opuśćiƺ go / poniewáż niemá dziáłu / áni dziedzictwá z tobą.
A po wyjśćiu káżdego trzećiego roku odłącżyƺ wƺyſtkie dzieśięćiny urodzáju twego onegoż roku / y złożyƺ je w brámách twoich.
Tedy przyjdzie Lewitá / który niemá dziáłu y dziedzictwá z tobą / y przychodzień / y śierotá / y wdowá / którzy ſą w brámách twoich / y będą jeść / y nájedzą śię / ábyć błogoſłáwił Pán / Bóg twój / w káżdej ſpráwie rąk twoich / którą cżynić będzieƺ.
