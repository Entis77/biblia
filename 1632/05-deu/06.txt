A teć ſą przykázániá / uſtáwy y ſądy / które rozkázáł Pán / Bóg wáƺ / ábym was náucżył / żebyśćie je cżynili w źiemi / do którey idźiećie / ábyśćie ją pośiedli ;
Żebyś śię báł Páná / Bogá twego / przeſtrzegájąc wƺyſtkich uſtáw jego y przykázáń jego / które ja przykázuję tobie / ty y ſyn twój / y ſyn ſyná twego / po wƺyſtkie dni żywotá ſwego / áby śię przedłużyły dni twoje.
A ták ſłucháj Izráelu / y ſtrzeż / ábyś ták cżynił / iżby śię tobie dobrze dźiáło / y żebyśćie śię rozmnożyli bárdzo ( jáko obiecáł Pán / Bóg ojców twoich / tobie / ) w źiemi opływájącey mlekiem y miodem.
Słuchájże Izráelu : Pán / Bóg náƺ / Pán jeden jeſt.
Będźieƺ tedy miłowáł Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey / y ze wƺyſtkiej śiły twojey ;
A będą te ſłowá / które ja dźiś rozkázuję tobie / w ſercu twojim ;
Y będźieƺ je cżęſto przypomináł ſynom twoim / y rozmáwiáł o nich śiedząc w domu twym / y będąc w drodze / y kłádąc śię y wſtáwájąc.
Y przywiążeƺ je zá znák ná rękách twoich / y będą jáko nácżelniki między ocżymá twemi.
Nápiƺeƺ je też ná podwojách domu twego / y ná brámách twoich.
A gdy ćię wprowádźi Pán / Bóg twój / do źiemi / o którą przyśiągł ojcom twoim Abráhámowi / Izáákowi / y Jákóbowi / że ją tobie dá / miáſtá wielkie y dobre / którycheś nie budowáł ;
Przytem domy pełne dóbr wƺelákich / którycheś nie náprzątáł / y ſtudnie wykopáne / którycheś nie kopáł / winnice y oliwnice / którycheś nie ſádźił / á będźieƺ jádł / y nájeƺ śię :
Strzeżże śię / ábyś nie zápomniáł Páná / który ćię wywiódł z źiemi Egipſkiej / z domu niewoli.
Páná / Bogá twego / báć śię będźieƺ / jemu ſłużyć / á przez imię jego przyśięgáć.
Nie udáćie śię zá bogámi obcymi / zá bogámi národów / które ſą około was.
( Abowiem Bóg záwiſny w miłośći Pán / Bóg twój / w pośrodku ćiebie / ) by śię ſnáć gniew nie zápálił Páná / Bogá twego / przećiw tobie / á wytráćiłby ćię z oblicżá źiemi.
Nie będźiećie kuśić Páná / Bogá wáƺego / jákośćie go kuśili w Máſſá.
Przeſtrzegáć będźiećie z pilnośćią przykázáń Páná / Bogá wáƺego / y świádectw jego y uſtáw jego / któreć przykázáł.
A cżyń to / co jeſt práwego y dobrego przed ocżymá Páńſkimi / ábyć śię dobrze dźiáło / ábyś wƺedłƺy pośiádł tę wyborną źiemię / o którą przyśiągł Pán ojcom twoim :
Żeby wygnáł wƺyſtkie nieprzyjáćioły twoje przed tobą / jáko mówił Pán.
A gdyby ćię nápotem ſpytáł ſyn twój / mówiąc : Cóż to zá świádectwá / y uſtáwy / y ſądy / które wám przykázáł Pán / Bóg náƺ?
Tedy powieƺ ſynowi twemu : Byliśmy niewolnikámi Fáráonowymi w Egipćie / y wywiódł nas z Egiptu Pán w ręce możney.
Y cżynił Pán znáki / y cudá wielkie / y ƺkodliwe w Egipćie nád Fáráonem / y nád wƺyſtkim domem jego / przed ocżymá náƺemi.
A wywiódł nas ſtámtąd / áby nas wprowádźił / y dáł nám tę źiemię / o którą przyśiągł ojcom náƺym.
A ták rozkázáł nám Pán / ábyśmy cżynili wƺyſtkie te uſtáwy / bojąc śię Páná / Bogá náƺego / żeby śię nám dobrze dźiáło po wƺyſtkie dni / áby nas przy żywoćie záchowáł / jáko y dźiś.
Y będźie to ſpráwiedliwośćią náƺą / gdy będźiemy ſtrzedz y cżynić te wƺyſtkie przykázániá przed Pánem / Bogiem náƺym / jáko nám przykázáł.
