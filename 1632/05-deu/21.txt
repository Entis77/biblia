Jeźliby ználeziony był zábity w ziemi / którą Pán / Bóg twój / dáwá tobie / ábyś ją pośiádł / leżący ná polu / á nie wiedziánoby / kto go zábił :
Tedy wynijdą ſtárśi twoi / y ſędziowie twoi / y rozmierzą ziemię áż do miáſt / które ſą w około onego zábitego.
A któreby miáſto bliżƺe było onego zábitego / tedy wezmą ſtárśi miáſtá onego jáłowicę z ſtádá / którą jeƺcże nie robiono / á którá nie ćiągnęłá w járzmie ;
Y záwiodą ſtárśi tego miáſtá onę jáłowicę w dolinę kámieniſtą / gdzie nigdy nie oráno / áni śiáno / y utną tám ƺyję jáłowicy w onej dolinie.
Zátem przyjdą kápłáni / ſynowie Lewiego ; bo je obráł Pán / Bóg twój / áby mu ſłużyli / y żeby błogoſłáwili w imieniu Páńſkim / á według uznániá ich końcżyć śię má káżdy ſpór / y káżdá ráná.
Tákże wƺyſcy ſtárśi miáſtá onego / które jeſt nájbliżƺe zábitego / umyją ręce ſwoje nád jáłowicą śćiętą w dolinie ;
Y oświádcżą śię / mówiąc : Ręce náƺe nie wyláły tej krwi / áni ocży náƺe ná to pátrzáły.
Ocżyść lud twój Izráelſki / któryś odkupił / Pánie / á nie pocżytáj krwi niewinnej w pośrodku ludu twego Izráelſkiego. Tedy ocżyƺcżeni będą od onej krwi /
A ty odejmieƺ krew niewinną z pośrodku śiebie / gdy ucżyniƺ / co práwego jeſt przed ocżymá Páńſkimi.
Gdy też wynijdzieƺ ná wojnę przećiw nieprzyjáćiołom twoim / á podáć je Pán / Bóg twój / w ręce twoje / y nábierzeƺ z nich więźniów ;
A obácżyƺ między więźniámi niewiáſtę nádobną / której rozmiłowáwƺy śię / wziąłbyś ją ſobie zá żonę :
Tedy ją wprowádziƺ do domu ſwego ; y ogoli głowę ſwoję / y obrzeże páznogćie ſwoje ;
A złożywƺy z śiebie ƺátę / w której jeſt pojmáná / zoſtánie w domu twym / á płákáć będzie ojcá ſwego / y mátki ſwojej przez cáły mieśiąc : á potem wnijdzieƺ do niej / y będzieƺ mężem jej / á oná będzie tobie zá żonę.
A jeźlibyć śię potem nie podobáłá / tedy ją wolno puśćiƺ / á żádną miárą nie ſprzedáƺ jej zá pieniądze / áni nią kupcżyć będzieƺ / poniewáżeś ją zelżył.
Gdyby też kto miáł dwie żony / jednę miłą / á drugą omierzłą / y národziłyby mu ſynów / miłá y omierzłá / á byłby ſyn pierworodny omierzłej ;
Tedy gdy ſtánowić będzie zá dziedzice ſyny ſwoje / dóbr ſwoich : nie będzie mógł dáć pierworodztwá ſynowi miłej przed ſynem pierworodnym onej omierzłej ;
Ale pierworodztwo ſynowi omierzłej przyzná / dáwƺy mu dwojáką cżęść wƺyſtkiego / co má ; poniewáż on jeſt pocżątkiem śiły jego / jego jeſt práwo pierworodztwá.
Jeźliby kto miáł ſyná ſwáwolnego / y krnąbrnego / któryby nie ſłucháł głoſu ojcá ſwego / y głoſu mátki ſwojej / á będąc ſtrofowány / nie uſłucháłby ich :
Tedy go wezmą ojćiec jego y mátká jego / y przywiodą go do ſtárƺych miáſtá ſwego / y do brámy ſądowej miejſcá onego /
Y rzeką do ſtárƺych miáſtá onego : Ten náƺ ſyn jeſt ſwáwolny y krnąbrny / niepoſłuƺny głoſu náƺemu / żárłok jeſt y pijánicá ;
Tedy ukámionują go wƺyſcy ludzie miáſtá onego / y umrze ; á ták odejmieƺ złe z pośrodku śiebie / co gdy wƺyſtek Izráel uſłyƺy / báć śię będą.
A gdyby kto popełnił grzech godny śmierći / á byłby ſkázány ná śmierć / á obwieśiłbyś go ná drzewie /
Nie zoſtánie przez noc trup jego ná drzewie / ále koniecżnie pochowáƺ go tegoż dniá ; bo przeklęſtwem Bożem jeſt ten / co wiśi. Przetoż nie będzieƺ plugáwił ziemi twojej / którą Pán / Bóg twój / dáwá tobie w dziedzictwo.
