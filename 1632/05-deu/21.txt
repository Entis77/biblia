Jeſliby ználeźiony był zábity w źiemi / którą Pán / Bóg twój / dáwá tobie / ábyś ją pośiádł / leżący ná polu / á nie wiedźiánoby / kto go zábił :
Tedy wynijdą ſtárśi twoi / y ſędźiowie twoi / y rozmierzą źiemię áż do miáſt / które ſą w około onego zábitego.
A któreby miáſto bliżƺe było onego zábitego / tedy wezmą ſtárśi miáſtá onego jáłowicę z ſtádá / którą jeƺcże nie robiono / á która nie ćiągnęłá w járzmie ;
Y záwiodą ſtárśi tego miáſtá onę jáłowicę w dolinę kámieniſtą / gdźie nigdy nie oráno / áni śiáno / y utną tám ƺyję jáłowicy w oney dolinie.
Zátem przyjdą kápłáni / ſynowie Lewiego ; bo je obráł Pán / Bóg twój / áby mu ſłużyli / y żeby błogoſłáwili w imieniu Páńſkim / á według uznániá ich końcżyć śię má káżdy ſpór / y káżdá ráná.
Tákże wƺyſcy ſtárśi miáſtá onego / które jeſt nájbliżƺe zábitego / umyją ręce ſwoje nád jáłowicą śćiętą w dolinie ;
Y oświádcżą śię / mówiąc : Ręce náƺe nie wyláły tey krwi / áni ocży náƺe ná to pátrzáły.
Ocżyść lud twój Izráelſki / któryś odkupił / Pánie / á nie pocżytáj krwi niewinney w pośrodku ludu twego Izráelſkiego. Tedy ocżyƺcżeni będą od oney krwi /
A ty odejmieƺ krew niewinną z pośrodku śiebie / gdy ucżyniƺ / co práwego jeſt przed ocżymá Páńſkimi.
Gdy też wynijdźieƺ ná wojnę przećiw nieprzyjáćiołom twoim / á podáć je Pán / Bóg twój / w ręce twoje / y nábierzeƺ z nich więźniów ;
A obácżyƺ między więźniámi niewiáſtę nádobną / którey rozmiłowáwƺy śię / wźiąłbyś ją ſobie zá żonę :
Tedy ją wprowádźiƺ do domu ſwego ; y ogoli głowę ſwoję / y obrzeże páznogćie ſwoje ;
A złożywƺy z śiebie ƺátę / w którey jeſt pojmáná / zoſtánie w domu twym / á płákáć będźie ojcá ſwego / y mátki ſwojey przez cáły mieśiąc : á potem wnijdźieƺ do niej / y będźieƺ mężem jey / á oná będźie tobie zá żonę.
A jeſlibyć śię potem nie podobáłá / tedy ją wolno puśćiƺ / á żádną miárą nie ſprzedáƺ jey zá pieniądze / áni nią kupcżyć będźieƺ / poniewáżeś ją zelżył.
Gdyby też kto miał dwie żony / jednę miłą / á drugą omierzłą / y národźiłyby mu ſynów / miłá y omierzłá / á byłby ſyn pierworodny omierzłej ;
Tedy gdy ſtánowić będźie zá dźiedźice ſyny ſwoje / dóbr ſwoich : nie będźie mógł dáć pierworodztwá ſynowi miłej przed ſynem pierworodnym oney omierzłej ;
Ale pierworodztwo ſynowi omierzłej przyzná / dáwƺy mu dwojáką cżęść wƺyſtkiego / co má ; poniewáż on jeſt pocżątkiem śiły jego / jego jeſt práwo pierworodztwá.
Jeſliby kto miał ſyná ſwáwolnego / y krnąbrnego / któryby nie ſłucháł głoſu ojcá ſwego / y głoſu mátki ſwojey / á będąc ſtrofowány / nie uſłucháłby ich :
Tedy go wezmą ojćiec jego y mátká jego / y przywiodą go do ſtárƺych miáſtá ſwego / y do brámy ſądowej miejſcá onego /
Y rzeką do ſtárƺych miáſtá onego : Ten náƺ ſyn jeſt ſwáwolny y krnąbrny / niepoſłuƺny głoſu náƺemu / żárłok jeſt y pijánicá ;
Tedy ukámionują go wƺyſcy ludźie miáſtá onego / y umrze ; á ták odejmieƺ złe z pośrodku śiebie / co gdy wƺyſtek Izráel uſłyƺy / báć śię będą.
A gdyby kto popełnił grzech godny śmierći / á byłby ſkázány ná śmierć / á obwieśiłbyś go ná drzewie /
Nie zoſtánie przez noc trup jego ná drzewie / ále koniecżnie pochowáƺ go tegoż dniá ; bo przeklęſtwem Bożem jeſt ten / co wiśi. Przetoż nie będźieƺ plugáwił źiemi twojey / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo.
