Słucháj / Izráelu! ty przejdzieƺ dziś Jordán / ábyś wƺedłƺy pośiádł národy więkƺe / y możniejƺe / niżeś ty / miáſtá wielkie / y wymurowáne áż pod niebo ;
Lud wielki / y wyſokiego wzroſtu ſyny Enákowe / które ty znáƺ / y o którycheś ſłyƺáł / gdy mówiono : Któż śię oſtoi przed ſyny Enákowymi?
Przetoż wiedz dziśiáj / że Pán / Bóg twój / który idzie przed tobą / jeſt ogień tráwiący : on je wytráći / y on je poniży przed tobą / że je wypędziƺ / á wygłádziƺ je prędko / jákoć to obiecáł Pán.
Nie mówże w ſercu ſwem / gdy je wypędzi Pán / Bóg twój / przed tobą / powiádájąc : Dlá ſpráwiedliwośći mojej wprowádził mię Pán / ábym pośiádł tę ziemię ; ále dlá niezbożnośći národów onych Pán wygnáł je przed twárzą twoją.
Nie dlá ſpráwiedliwośći twojej / áni dlá proſtośći ſercá twego ty wnijdzieƺ / ábyś pośiádł ziemię ich ; ále dlá niezbożnośći národów tych Pán / Bóg twój / wypędzá ich przed twárzą twoją / y áby utwierdził Pán ſłowo / które przyśiągł ojcom twoim Abráhámowi / Izáákowi / y Jákóbowi.
Wiedzże tedy / że nie dlá ſpráwiedliwośći twojej / Pán / Bóg twój / dáje tobie tę wyborną ziemię / ábyś ją pośiádł / gdyżeś ty lud twárdego kárku.
Pámiętáj / á nie zápomináj / żeś do gniewu pobudzáł Páná / Bogá twego / ná puƺcży ; od onego dniá / jákoś wyƺedł z ziemi Egipſkiej / áżeśćie przyƺli ná to miejſce / odpornymiśćie byli Pánu.
Tákże y przy górze Horeb pobudziliśćie do gniewu Páná / y rozgniewáł śię Pán ná wáſ / áby wáſ wygłádził.
Gdym wſtąpił ná górę / ábym wziął táblice kámienne / táblice przymierzá / które ucżynił Pán z wámi / y trwáłem ná górze cżterdzieśći dni / y cżterdzieśći nocy / chlebá nie jedząc / áni wody pijąc :
Tedy mi dáł Pán dwie táblice kámienne / nápiſáne pálcem Bożym / ná których te wƺyſtkie ſłowá były / które mówił Pán do wáſ ná górze z pośrodku ogniá / w dzień zgromádzeniá wáƺego.
A gdy wyƺło cżterdzieśći dni / y cżterdzieśći nocy / dáł mi Pán one dwie táblice kámienne / táblice przymierzá ;
Y rzekł Pán do mnie : Wſtáń / znijdź ſtąd rychło ; ábowiem śię popſowáł lud twój / któryś wywiódł z Egiptu ; odſtąpili prędko z drogi / którąm im rozkázáł / y ucżynili ſobie litego báłwáná.
Y rzekł Pán do mnie / mówiąc : Widziáłem ten lud / á oto / lud twárdego kárku jeſt.
Puść mię á wytrácę je / y wygłádzę imię ich pod niebem / á ćiebie ucżynię w náród możniejƺy / y więkƺy niżli ten.
Tedym śię obróćił / y zſtąpiłem z góry / ( á górá oná páłáłá ogniem / ) dwie táblice przymierzá nioſąc w obu rękách moich.
A gdym ujrzáł / iżeśćie zgrzeƺyli przećiw Pánu / Bogu wáƺemu / ucżyniwƺy ſobie ćielcá litego y odſtąpiliśćie byli prędko z drogi / którą wám rozkázáł Pán :
Tedy wziąwƺy já one dwie táblice / porzućiłem je z obu rąk moich / á ſtłukłem je przed ocżymá wáƺemi.
Potem upádłem przed Pánem jáko y pierwej / przez cżterdzieśći dni y cżterdzieśći nocy / chlebám nie jádł / y wodym nie pił dlá wáƺych wƺyſtkich grzechów / któremiśćie zgrzeƺyli / cżyniąc złość przed ocżymá Páńſkimi / y dráźniąc go.
Bom śię báł onej popędliwośći y gniewu / którym śię był zápálił Pán przećiwko wám / áby wáſ wytráćił ; y wyſłucháł mię Pán y tym rázem.
Ná Aároná też rozgniewáł śię był Pán bárdzo / chcąc go zátráćić ; tedym śię też zá Aáronem modlił tegoż cżáſu.
A grzech wáƺ / któryśćie byli ucżynili / to jeſt ćielcá / porwáłem / y ſpáliłem go ogniem / y ſkruƺyłem go / tłukąc dobrze / áż śię w proch obróćił / y wrzućiłem proch jego w potok / który płynął z onej góry.
Potem w Táberá y w Máſſá / w Kibrot Hátááwá pobudziliśćie do gniewu Páná.
A gdy wáſ Pán wyſłáł z Kádeſ Bárne / mówiąc : Idźćie / á pośiądźćie tę ziemię / którąm wám dáł / odporniśćie byli ſłowu Páná / Bogá wáƺego / ániśćie mu wierzyli / ániśćie uſłucháli głoſu jego /
Owá odporniśćie byli Pánu od dniá tego / jákom wáſ poznáł.
Dlá tego upádłƺy przed Pánem / cżterdzieśći dni y cżterdzieśći nocy leżáłem ; bo już był Pán rzekł / że wáſ wytráćić miáł.
Y modliłem śię Pánu / y rzekłem : Pánie Boże / nie zátrácájże ludu twego / y dziedzictwá twego / któreś odkupił wielmożnośćią twoją / któreś wywiódł z Egiptu ręką możną.
Wſpomnij ná ſłużebniki twoje / Abráhámá / Izááká / y Jákóbá ; nie pátrz ná zátwárdziáłość ludu tego / y ná niezbożność jego / áni ná grzech jego /
By ſnáć nie rzekli obywátele ziemi tej / z którejś náſ wywiódł : Przeto że nie mógł Pán wprowádzić ich do ziemi / którą im obiecáł / álbo iż je miáł w nienáwiśći / wywiódł je / áby je pobił ná puƺcży.
Abowiem oni ſą ludem twoim / y dziedzictwem twojim / któreś wywiódł w mocy twojej wielkiej / y w rámieniu twojim wyćiągnionem.
