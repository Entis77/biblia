Słucháj / Izráelu! ty przejdźieƺ dźiś Jordán / ábyś wƺedƺy pośiádł narody więkƺe / y możniejƺe / niżeś ty / miáſtá wielkie / y wymurowáne áż pod niebo ;
Lud wielki / y wyſokiego wzroſtu ſyny Enákowe / które ty znáƺ / y o którycheś ſłyƺał / gdy mówiono : Któż śię oſtoi przed ſyny Enákowymi?
Przetoż wiedz dźiśiáj / że Pán / Bóg twój / który idźie przed tobą / jeſt ogień tráwiący : on je wytráći / y on je poniży przed tobą / że je wypędźiƺ / á wygłádźiƺ je prędko / jákoć to obiecał Pán.
Nie mówże w ſercu ſwem / gdy je wypędźi Pán / Bóg twój / przed tobą / powiádájąc : Dla ſpráwiedliwośći mojey wprowádźił mię Pán / ábym pośiádł tę źiemię ; ále dla niezbożnośći narodów onych Pán wygnał je przed twárzą twoją.
Nie dla ſpráwiedliwośći twojey / áni dla proſtośći ſercá twego ty wnijdźieƺ / ábyś pośiádł źiemię ich ; ále dla niezbożnośći narodów tych Pán / Bóg twój / wypędzá ich przed twárzą twoją / y áby utwierdźił Pán ſłowo / które przyśiągł ojcom twoim Abráhámowi / Izáákowi / y Jákóbowi.
Wiedzże tedy / że nie dla ſpráwiedliwośći twojey / Pán / Bóg twój / dáje tobie tę wyborną źiemię / ábyś ją pośiádł / gdyżeś ty lud twárdego kárku.
Pámiętáj / á nie zápomináj / żeś do gniewu pobudzał Páná / Bogá twego / ná puƺcży ; od onego dniá / jákoś wyƺedł z źiemi Egipſkiej / áżeśćie przyƺli ná to miejſce / odpornymiśćie byli Pánu.
Tákże y przy górze Horeb pobudźiliśćie do gniewu Páná / y rozgniewał śię Pán ná was / áby was wygłádźił.
Gdym wſtąpił ná górę / ábym wźiął táblice kámienne / táblice przymierza / które ucżynił Pán z wámi / y trwáłem ná górze cżterdźieśći dni / y cżterdźieśći nocy / chlebá nie jedząc / áni wody pijąc :
Tedy mi dał Pán dwie táblice kámienne / nápiſáne pálcem Bożym / ná których te wƺyſtkie ſłowá były / które mówił Pán do was ná górze z pośrzodku ogniá / w dźień zgromádzenia wáƺego.
A gdy wyƺło cżterdźieśći dni / y cżterdźieśći nocy / dał mi Pán one dwie táblice kámienne / táblice przymierza ;
Y rzekł Pán do mnie : Wſtáń / znijdź ſtąd rychło ; ábowiem śię popſował lud twój / któryś wywiódł z Egiptu ; odſtąpili prędko z drogi / którąm im rozkázał / y ucżynili ſobie litego báłwáná.
Y rzekł Pán do mnie / mówiąc : Widźiałem ten lud / á oto / lud twárdego kárku jeſt.
Puść mię á wytrácę je / y wygłádzę imię ich pod niebem / á ćiebie ucżynię w naród możniejƺy / y więkƺy niżli ten.
Tedym śię obróćił / y zſtąpiłem z góry / ( á górá oná páłáłá ogniem / ) dwie táblice przymierza nioſąc w obu rękách mojich.
A gdym ujrzał / iżeśćie zgrzeƺyli przećiw Pánu / Bogu wáƺemu / ucżyniwƺy ſobie ćielcá litego y odſtąpiliśćie byli prędko z drogi / którą wam rozkázał Pán :
Tedy wźiąwƺy ja one dwie táblice / porzućiłem je z obu rąk mojich / á ſtłukłem je przed ocżymá wáƺemi.
Potem upádłem przed Pánem jáko y pierwej / przez cżterdźieśći dni y cżterdźieśći nocy / chlebám nie jádł / y wodym nie pił dla wáƺych wƺyſtkich grzechów / któremiśćie zgrzeƺyli / cżyniąc złość przed ocżymá Páńſkimi / y dráźniąc go.
Bom śię bał oney popędliwośći y gniewu / którym śię był zápálił Pán przećiwko wam / áby was wytráćił ; y wyſłuchał mię Pán y tym rázem.
Ná Aároná też rozgniewał śię był Pán bárzo / chcąc go zátráćić ; tedym śię też zá Aáronem modlił tegoż cżáſu.
A grzech wáƺ / któryśćie byli ucżynili / to jeſt ćielcá / porwáłem / y ſpáliłem go ogniem / y ſkruƺyłem go / tłukąc dobrze / áż śię w proch obróćił / y wrzućiłem proch jego w potok / który płynął z oney góry.
Potem w Táberá y w Máſſá / w Kibrot Hátááwá pobudźiliśćie do gniewu Páná.
A gdy was Pán wyſłał z Kádes Bárne / mówiąc : Idźćie / á pośiądźćie tę źiemię / którąm wam dał / odporniśćie byli ſłowu Páná / Bogá wáƺego / ániśćie mu wierzyli / ániśćie uſłucháli głoſu jego /
Owá odporniśćie byli Pánu od dniá tego / jákom was poznał.
Dla tego upádƺy przed Pánem / cżterdźieśći dni y cżterdźieśći nocy leżáłem ; bo już był Pán rzekł / że was wytráćić miał.
Y modliłem śię Pánu / y rzekłem : Pánie Boże / nie zátrácájże ludu twego / y dźiedźictwá twego / któreś odkupił wielmożnośćią twoją / któreś wywiódł z Egiptu ręką możną.
Wſpomnij ná ſłużebniki twoje / Abráhámá / Izááká / y Jákóbá ; nie pátrz ná zátwárdźiáłość ludu tego / y ná niezbożność jego / áni ná grzech jego /
By ſnadź nie rzekli obywátele źiemi tey / z którejś nas wywiódł : Przeto że nie mógł Pán wprowádźić ich do źiemi / którą im obiecał / álbo iż je miał w nienáwiśći / wywiódł je / áby je pobił ná puƺcży.
Abowiem oni ſą ludem twoim / y dźiedźictwem twojim / któreś wywiódł w mocy twojey wielkiej / y w rámieniu twojim wyćiągnionem.
