Jeźliby śię ſpór wƺcżął między ſtronámi / á przyƺliby do ſądu / áby je ſądzono : tedy uſpráwiedliwią ſpráwiedliwego / á niezbożnego oſądzą zá niezbożnego.
A jeźli godzien będzie bićiá niepobożny / tedy káże go położyć ſędziá / y káże go bić przed ſobą według miáry niepráwośći jego / pod licżbą.
Cżterdzieśći kroć uderzy go ; więcej nie przydá / by ſnáć / jeźliby mu nád to zádáno więcej plág / nie znikcżemniáł brát twój przed ocżymá twemi.
Nie záwiążeƺ gęby wołowi młócącemu.
Gdyby mieƺkáli bráćiá poſpołu / á zƺedłby jeden z nich / nie májąc ſyná / nie pójdzie żoná onego zmárłego zá mężá obcego / brát jego wnijdzie do niej / á pojmie ją ſobie zá żonę / á práwem powinowáctwá przyłącży ją ſobie.
A pierworodny ſyn / któregoby porodziłá / názwány będzie imieniem brátá jego zmárłego / áby nie było wygłádzone imię jego z Izráelá.
A jeźliby nie chćiáł on mąż pojąć brátowej ſwojej / tedy pójdzie brátowá jego do brámy przed ſtárƺe / y rzecże : Nie chce brát mężá mego wzbudzić brátu ſwemu imieniá w Izráelu / y nie chce mię práwem powinowáctwá ſobie przyłącżyć.
Tedy go przyzowią ſtárśi miáſtá onego / y będą mówić z nim ; á ſtánąwƺy / jeźli rzecże : Nie chcę jej pojąć /
Przyſtąpi brátowá jego do niego przed ocżymá ſtárƺych / á zzuje trzewik jego z nogi jego / y plunie ná twárz jego / á odpowiádájąc rzecże : Ták śię ſtánie mężowi / któryby nie budowáł domu brátá ſwego.
Y názywáć będą imię jego w Izráelá : Dom wyzutego.
Gdyby śię powádzili z ſobą dwáj jácy mężowie / á przybieżáłáby żoná jednego / áby wyrwáłá mężá ſwego z ręki onego / co go bije / á śćiągnąwƺy rękę ſwą / uchwyćiłáby go zá łono jego /
Utnieƺ jej rękę / y nie zlituje śię nád nią oko twoje.
Nie będzieƺ miáł w worku twoim różnych gwichtów / więkƺego y mniejƺego.
Nie będzieƺ też miáł w domu twoim dwojákiego korcá / więkƺego y mniejƺego.
Wágę ſłuƺną y ſpráwiedliwą mieć będzieƺ ; tákże korzec ſłuƺny y ſpráwiedliwy mieć będzieƺ / áby śię przedłużyły dni twoje ná ziemi / którą Pán / Bóg twój / dáwá tobie.
Abowiem obrzydliwośćią jeſt Pánu / Bogu twemu / káżdy / który to cżyni / wƺelki cżyniący niepráwość.
Pomnij ná to / coć ucżynił Amálek w drodze / kiedyśćie ƺli z Egiptu ;
Jákoć zábieżáł drogę / á pobił oſtátnie wojſká twego / wƺyſtkie mdłe / idące zá tobą / gdyś ty był utrudzony y ſprácowány / á nie báł śię Bogá.
Przetoż / gdyć dá odpocżynienie Pán / Bóg twój / od wƺyſtkich nieprzyjáćiół twoich w około / w ziemi / którą Pán / Bóg twój dáwá tobie w dziedzictwo / ábyś ją pośiádł / wygłádziƺ pámiątkę Amálekowę pod niebem. Nie zápominájże tego.
