Jeſliby śię ſpór wƺcżął miedzy ſtronámi / á przyƺliby do ſądu / áby je ſądzono : tedy uſpráwiedliwią ſpráwiedliwego / á niezbożnego oſądzą zá niezbożnego.
A jeſli godźien będźie bićiá niepobożny / tedy káże go położyć ſędźiá / y káże go bić przed ſobą według miáry niepráwośći jego / pod licżbą.
Cżterdźieśći kroć uderzy go ; więcey nie przydá / by ſnadź / jeſliby mu nád to zádáno więcey plág / nie znikcżemniał brát twój przed ocżymá twemi.
Nie záwiążeƺ gęby wołowi młócącemu.
Gdyby mieƺkáli bráćia poſpołu / á zƺedłby jeden z nich / nie májąc ſyná / nie pójdźie żoná onego zmárłego zá mężá obcego / brát jego wnijdźie do niej / á pojmie ją ſobie zá żonę / á práwem powinowáctwá przyłącży ją ſobie.
A pierworodny ſyn / któregoby porodźiłá / názwány będźie imieniem brátá jego zmárłego / áby nie było wygłádzone imię jego z Izráelá.
A jeſliby nie chćiał on mąż pojąć brátowej ſwojey / tedy pójdźie brátowá jego do bramy przed ſtárƺe / y rzecże : Nie chce brát mężá mego wzbudźić brátu ſwemu imienia w Izráelu / y nie chce mię práwem powinowáctwá ſobie przyłącżyć.
Tedy go przyzowią ſtárśi miáſtá onego / y będą mówić z nim ; á ſtánąwƺy / jeſli rzecże : Nie chcę jey pojąć /
Przyſtąpi brátowá jego do niego przed ocżymá ſtárƺych / á zzuje trzewik jego z nogi jego / y plunie ná twárz jego / á odpowiádájąc rzecże : Ták śię ſtánie mężowi / któryby nie budował domu brátá ſwego.
Y názywáć będą imię jego w Izráelá : Dom wyzutego.
Gdyby śię powádźili z ſobą dwáj jácy mężowie / á przybieżáłáby żoná jednego / áby wyrwáłá mężá ſwego z ręki onego / co go bije / á śćiągnąwƺy rękę ſwą / uchwyćiłáby go zá łono jego /
Utnieƺ jey rękę / y nie zlituje śię nád nią oko twoje.
Nie będźieƺ miał w worku twoim różnych gwichtów / więkƺego y mniejƺego.
Nie będźieƺ też miał w domu twoim dwojákiego korcá / więkƺego y mniejƺego.
Wágę ſłuƺną y ſpráwiedliwą mieć będźieƺ ; tákże korzec ſłuƺny y ſpráwiedliwy mieć będźieƺ / áby śię przedłużyły dni twoje ná źiemi / którą Pán / Bóg twój / dáwá tobie.
Abowiem obrzydliwośćią jeſt Pánu / Bogu twemu / káżdy / który to cżyni / wƺelki cżyniący niepráwość.
Pomnij ná to / coć ucżynił Amálek w drodze / kiedyśćie ƺli z Egiptu ;
Jákoć zábieżał drogę / á pobił oſtátnie wojſká twego / wƺyſtkie mdłe / idące zá tobą / gdyś ty był utrudzony y ſprácowány / á nie bał śię Bogá.
Przetoż / gdyć da odpocżynienie Pán / Bóg twój / od wƺyſtkich nieprzyjáćiół twojich w około / w źiemi / którą Pán / Bóg twój dáwá tobie w dźiedźictwo / ábyś ją pośiádł / wygłádźiƺ pámiątkę Amálekowę pod niebem. Nie zápominájże tego.
