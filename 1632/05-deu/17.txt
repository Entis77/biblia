Nie będźieƺ ofiárował Pánu / Bogu twemu / wołu / áni bydlęćiá / ná któremby byłá ſkázá / álbo jákákolwiek wádá / gdyż to jeſt obrzydliwośćią Pánu / Bogu twemu.
Jeſliby śię ználázł miedzy wámi w któremkolwiek mieśćie twem / które Pán / Bóg twój / dáwá tobie / mąż álbo niewiáſtá / któryby śię cżego złego dopuśćił przed ocżymá Páná / Bogá twego / przeſtępując przymierze jego /
A poƺedƺy / ſłużyłby obcym bogom / y kłániáłby śię im / álbo ſłońcu / álbo mieśiącowi / álbo wƺyſtkim záſtępom niebieſkim / cżegom nie przykázał ;
Y oznájmionoćby to / á uſłyƺáwƺy / wywiádowáłbyś śię pilno / á oto / byłáby práwdá / y rzecż pewná / że śię ſtáłá tá obrzydliwość w Izráelu :
Tedy wywiedźieƺ onego mężá / álbo onę niewiáſtę / którzy ucżynili tę złą rzecż / do bram twojich / mężá onego / álbo niewiáſtę / y ukámionujeƺ je / á pomrą.
W uśćiech dwu świádków / álbo trzech świádków zábity będźie / kto umrzeć ma ; niech nie umierá ná ſłowo świádká jednego.
Ręká świádków będźie ná nim nájpierwƺá / ná zábićie jego / á ręká wƺyſtkiego ludu potem ; á ták odejmieƺ złe z pośrzodku śiebie.
Byłoliby też co trudnego przy ſądźie / miedzy krwią á krwią / miedzy ſpráwą á ſpráwą / y miedzy ráną á ráną / y około poſwárków w bramách twojich / tedy wſtánieƺ / á pójdźieƺ ná miejſce / które obierze Pán / Bóg twój.
Y przyjdźieƺ do kápłanów Lewitów / y do ſędźiego / który będźie ná on cżás / y będźieƺ ich pytał / á oznájmiąć / jáko to oſądźić maƺ.
Y ucżyniƺ według tego / jákoć powiedzą z miejſcá tego / które obrał Pán / y poſtąpiƺ ſobie według wƺyſtkiego / jáko ćię náucżą.
Według ſłów zakonu / którego ćię náucżą / y według ſądu / któryć powiedzą / ſpráwowáć śię będźieƺ ; nie uſtąpiƺ od ſłowá / któreć powiedzą / áni ná práwo áni ná lewo.
A jeſliby kto ucżynił to z hárdośći / żeby nie chćiał uſłucháć kápłáná tám poſtánowionego ku ſłużbie przed Pánem / Bogiem twoim / álbo ſędźiego / niech umrze on mąż ; y odejmieƺ to złe z Izráelá /
Aby wƺyſtek lud uſłyƺáwƺy bał śię / á więcey hárdźie ſobie nie pocżynał.
Gdy wnijdźieƺ do źiemi / którą Pán / Bóg twój / dáwá tobie / á opánujeƺ ją / y będźieƺ mieƺkał w niej / y rzecżeƺ : Poſtánowię nád ſobą królá / jáko y wƺyſtkie narody okolicżne :
Tego tylko poſtánowiƺ nád ſobą królá / którego obierze Pán / Bóg twój ; z pośrzodku bráći twej poſtánowiƺ nád ſobą królá ; nie będźieƺ mógł przełożyć nád ſobą cudzoźiemcá / któryby nie był brátem twoim.
Tylko niech nie chowá wiele koni / áni náwrácá ludu do Egiptu / áby wiele koni nábył / ponieważ wam Pán rzekł : Nie wrácájćie śię záś tą drogą więcey.
Nie będźie też miał wiele żon / áby śię nie odwróćiło ſerce jego ; ſrebrá też y złotá niech názbyt wiele nie nábywá.
A gdy uśiędźie ná ſtolicy króleſtwá ſwego / przepiƺe ſobie ten powtórzony zakon w kśięgi od kápłanów Lewitów.
Y będźie go miał przy ſobie á będźie go cżytał po wƺyſtkie dni żywotá ſwego / áby śię ucżył báć Páná / Bogá ſwego / y przeſtrzegał wƺyſtkich ſłów zakonu tego / y uſtáw tych / á cżynił je ;
Aby śię nie wynośiło ſerce jego nád bráćią jego / y żeby śię nie unośiło od tego przykázánia ná práwo / áni ná lewo / áby długo żył ná króleſtwie ſwojim / on y ſynowie jego w pośrzodku Izráelá.
