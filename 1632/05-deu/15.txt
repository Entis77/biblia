Ná pocżątku káżdego śiódmego roku cżynić będźieƺ odpuƺcżenie.
A toć jeſt ſpoſób odpuƺcżániá / żeby odpuśćił káżdy / który pożycżył z ręki ſwej / to / cżego pożycżył bliźniemu ſwemu ; nie będźie wyćiągáł długu od bliźniego ſwego / y od brátá ſwego / poniewáż obwołáne jeſt odpuƺcżenie Páńſkie.
Od obcego wyćiągáć dług możeƺ : ále cobyś miáł u brátá twego / odpuśći mu ręká twojá :
Dlá tego / żeby śię nie ſtáł między wámi kto przez ćię ubogim / poniewáż hojnie błogoſłáwić tobie będźie Pán w źiemi / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo / ábyś ją pośiádł.
Tylko żebyś ſłuchájąc poſłuƺny był głoſowi Páná / Bogá twego / ábyś ſtrzegł / y cżynił káżde przykázánie to / które já przykázuję tobie dźiśiáj /
Abowiem Pán / Bóg twój / błogoſłáwić ćię będźie / jákoć obiecáł ; y będźieƺ pożycżáł wielu národom / á ſám u nikogo nie będźieƺ pożycżáł ; y będźieƺ pánowáł nád wielą národów / á one nád tobą pánowáć nie będą.
Gdyby był u ćiebie ubogi ktokolwiek z bráći twojej w któremkolwiek mieśćie twojim / w źiemi twojej / którą Pán / Bóg twój / dáwá tobie / nie zátwárdzáj ſercá ſwego / áni záwieráj ręki twej przed brátem twoim ubogim ;
Ale ƺcżodrze otworzyƺ mu rękę twoję / y ochotnie pożycżyƺ mu / ile będźie potrzebowáł y cżegoby mu niedoſtáwáło.
Strzeż śię / áby nie byłá jáká niepobożność w ſercu twojim / żebyś miáł rzec : Nádchodźi rok on śiódmy / rok odpuƺcżeniá / y ſurowobyś śię ſtáwił brátu twemu ubogiemu / ták / żebyś mu nie użycżył / á onby wołáł przećiwko tobie do Páná / y miáłbyś grzech ;
Ale ochotnie dáwáć mu będźieƺ / y nie będźie niechętne ſerce twoje / gdy mu dáwáć będźieƺ / ábowiem dlá tegoć pobłogoſłáwi Pán / Bóg twój / we wƺyſtkich ſpráwách twoich / y do cżegokolwiek śćiągnieƺ rękę twoję.
Boć śię nie przebierze ubogich w źiemi wáƺej ; dlá tegoć rozkázuję / mówiąc : ábyś ƺcżodrze otwieráł rękę twą brátu twemu / y nędznemu twemu / y ubogiemu twemu w źiemi twojej.
Jeźliby śię záprzedáł tobie brát twój / Żydowin álbo Żydówká / á ſłużyłćiby przez ƺeść lát / tedy śiódmego roku wypuśćiƺ go wolno od śiebie ;
A gdy go wolno puśćiƺ od śiebie / nie puśćiƺ go próżnego.
Szcżodrze go udárujeƺ z bydłá twego / y z gumná twego / y z práſy twojej ; z tego / w cżem ći pobłogoſłáwił Pán / Bóg twój / dáƺ mu.
Y wſpomniƺ ſobie / żeś był niewolnikiem w źiemi Egipſkiej / ſkąd ćię wykupił Pán / Bóg twój ; dlá tego já to dźiś tobie przykázuję.
Jeźliby też rzekł do ćiebie : Nie pójdę od ćiebie / przeto iż ćię umiłowáł / y dom twój / á iż śię má dobrze u ćiebie :
Tedy wźiąwƺy ƺydło / przekoleƺ ucho jego ná drzwiách / y będźie ſługą twoim ná wieki ; toż y ſłużebnicy ſwej ucżyniƺ.
Niech to nie będźie przykro w ocżách twoich / że go puśćiƺ wolno od śiebie ; bo dwojáką zápłátę nájemniká záſłużył u ćiebie przez ƺeść lát ; y błogoſłáwić będźie tobie Pán / Bóg twój / we wƺyſtkiem / co będźieƺ cżynił.
Wƺyſtkiego pierworodztwá / które śię urodźi z bydłá twego / álbo z trzody twojej / ſámce / poświęćiƺ Pánu / Bogu twemu ; nie będźieƺ robił pierworodnym krowy twojej / áni będźieƺ ſtrzygł pierworodnych owiec twoich.
Przed Pánem / Bogiem twoim / będźieƺ je jádł káżdego roku / ná miejſcu / które obierze Pán / ty y dom twój /
A jeźliby ná nim byłá wádá / żeby chrome / álbo ślepe / álbo z jákąkolwiek wádą złą było / nie będźieƺ go ofiárowáł Pánu / Bogu twemu.
W brámách twych jeść je będźieƺ / niecżyſty y cżyſty zárówno / jáko ſárnę y jáko jeleniá ;
Tylko krwi jego nie będźieƺ jádł ; ná źiemię wylejeƺ ją / jáko wodę.
