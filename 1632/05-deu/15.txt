Ná pocżątku káżdego śiódmego roku cżynić będźieƺ odpuƺcżenie.
A toć jeſt ſpoſób odpuƺcżánia / żeby odpuśćił káżdy / który pożycżył z ręki ſwej / to / cżego pożycżył bliźniemu ſwemu ; nie będźie wyćiągał długu od bliźniego ſwego / y od brátá ſwego / ponieważ obwołáne jeſt odpuƺcżenie Páńſkie.
Od obcego wyćiągáć dług możeƺ : ále cobyś miał u brátá twego / odpuśći mu ręká twojá :
Dla tego / żeby śię nie ſtał miedzy wámi kto przez ćię ubogim / ponieważ hojnie błogoſłáwić tobie będźie Pán w źiemi / którą Pán / Bóg twój / dáwá tobie w dźiedźictwo / ábyś ją pośiádł.
Tylko żebyś ſłuchájąc poſłuƺny był głoſowi Páná / Bogá twego / ábyś ſtrzegł / y cżynił káżde przykazánie to / które ja przykázuję tobie dźiśiáj /
Abowiem Pán / Bóg twój / błogoſłáwić ćię będźie / jákoć obiecał ; y będźieƺ pożycżał wielu národom / á ſám u nikogo nie będźieƺ pożycżał ; y będźieƺ pánował nád wielą narodów / á one nád tobą pánowáć nie będą.
Gdyby był u ćiebie ubogi ktokolwiek z bráći twojey w któremkolwiek mieśćie twojim / w źiemi twojey / którą Pán / Bóg twój / dáwá tobie / nie zátwárdzáj ſercá ſwego / áni záwieráj ręki twej przed brátem twoim ubogim ;
Ale ƺcżodrze otworzyƺ mu rękę twoję / y ochotnie pożycżyƺ mu / ile będźie potrzebował y cżegoby mu niedoſtáwáło.
Strzeż śię / áby nie byłá jáká niepobożność w ſercu twojim / żebyś miał rzec : Nádchodźi rok on śiódmy / rok odpuƺcżeniá / y ſurowobyś śię ſtáwił brátu twemu ubogiemu / ták / żebyś mu nie użycżył / á onby wołał przećiwko tobie do Páná / y miáłbyś grzech ;
Ale ochotnie dáwáć mu będźieƺ / y nie będźie niechętne ſerce twoje / gdy mu dáwáć będźieƺ / ábowiem dla tegoć pobłogoſłáwi Pán / Bóg twój / we wƺyſtkich ſpráwách twojich / y do cżegokolwiek śćiągnieƺ rękę twoję.
Boć śię nie przebierze ubogich w źiemi wáƺej ; dla tegoć rozkázuję / mówiąc : ábyś ƺcżodrze otwierał rękę twą brátu twemu / y nędznemu twemu / y ubogiemu twemu w źiemi twojey.
Jeſliby śię záprzedał tobie brát twój / Żydowin álbo Żydówká / á ſłużyłćiby przez ƺeść lat / tedy śiódmego roku wypuśćiƺ go wolno od śiebie ;
A gdy go wolno puśćiƺ od śiebie / nie puśćiƺ go próżnego.
Szcżodrze go udárujeƺ z bydłá twego / y z gumná twego / y z práſy twojey ; z tego / w cżym ći pobłogoſłáwił Pán / Bóg twój / dáƺ mu.
Y wſpomniƺ ſobie / żeś był niewolnikiem w źiemi Egipſkiej / ſkąd ćię wykupił Pán / Bóg twój ; dla tego ja to dźiś tobie przykázuję.
Jeſliby też rzekł do ćiebie : Nie pójdę od ćiebie / przeto iż ćię umiłował / y dom twój / á iż śię ma dobrze u ćiebie :
Tedy wźiąwƺy ƺydło / przekoleƺ ucho jego ná drzwiách / y będźie ſługą twoim ná wieki ; toż y ſłużebnicy ſwej ucżyniƺ.
Niech to nie będźie przykro w ocżách twojich / że go puśćiƺ wolno od śiebie ; bo dwojáką zápłátę nájemniká záſłużył u ćiebie przez ƺeść lat ; y błogoſłáwić będźie tobie Pán / Bóg twój / we wƺyſtkim / co będźieƺ cżynił.
Wƺyſtkiego pierworodztwá / które śię urodźi z bydłá twego / álbo z trzody twojey / ſámce / poświęćiƺ Pánu / Bogu twemu ; nie będźieƺ robił pierworodnym krowy twojey / áni będźieƺ ſtrzygł pierworodnych owiec twojich.
Przed Pánem / Bogiem twoim / będźieƺ je jádł káżdego roku / ná miejſcu / które obierze Pán / ty y dom twój /
A jeſliby ná nim byłá wádá / żeby chrome / álbo ślepe / álbo z jákąkolwiek wádą złą było / nie będźieƺ go ofiárował Pánu / Bogu twemu.
W bramách twych jeść je będźieƺ / niecżyſty y cżyſty zárówno / jáko ſárnę y jáko jeleniá ;
Tylko krwi jego nie będźieƺ jádł ; ná źiemię wylejeƺ ją / jáko wodę.
