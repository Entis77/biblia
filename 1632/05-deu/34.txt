Tedy wſtąpił Mojżeƺ z onych równin Moábſkich ná górę Nebo / ná wierzch págórká / który jeſt ná przećiwko Jerychu ; á ukázáł mu Pán wƺyſtkę ziemię od Gáláád áż do Dán ;
Y wƺyſtkę ziemię Neftálimowę / y ziemię Efráimowę / y Mánáſeſowę / y wƺyſtkę ziemię Judowę áż do morzá oſtátniego.
Y ſtronę południową / y równinę doliny Jerychá / miáſtá oſádzonego pálmámi / áż do Segor.
Tedy mu rzekł Pán : Táć jeſt ziemiá / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : Náśieniu twemu dám ją ; pokázáłem ją ocżom twoim / ále do niej nie wnijdzieƺ.
Y umárł tám Mojżeƺ / ſługá Páńſki / w ziemi Moábſkiej / według ſłowá Páńſkiego.
Y pogrzebł go Pán w dolinie / w ziemi Moábſkiej / przećiw Betfegorowi / á nie dowiedziáł śię nikt o grobie jego áż do dniá tego.
A Mojżeƺ miáł ſto y dwádzieśćiá lát / gdy umárł ; nie záćmiło śię oko jego / áni śię náruƺyłá cżerſtwość jego.
Y płákáli ſynowie Izráelſcy Mojżeƺá ná polách Moábſkich przez trzydzieśći dni. A gdy śię ſkońcżyły dni płácżu y żáłoby Mojżeƺowej :
Tedy Jozue / ſyn Nunów / nápełniony jeſt duchem mądrośći / bo był włożył Mojżeƺ ręce ſwoje náń : y poſłuƺni mu byli ſynowie Izráelſcy / á cżynili / jáko przykázáł Pán Mojżeƺowi.
Y nie powſtáł prorok więcej w Izráelu podobny Mojżeƺowi / któregoby ták znáł Pán / twárzą w twárz.
We wƺyſtkich znákách / y cudách / dlá których go był poſłáł Pán / áby je cżynił w ziemi Egipſkiej przed Fáráonem y przed wƺyſtkimi ſługámi jego / y przed wƺyſtką ziemią jego.
Tákże we wƺyſtkich ſpráwách ręki możnej / y we wƺyſtkich poſtráchách wielkich / które cżynił Mojżeƺ przed ocżymá wƺyſtkiego Izráelá.
