Tedy wſtąpił Mojzeƺ z onych równin Moábſkich ná górę Nebo / ná wierzch págórká / który jeſt ná przećiwko Jerychu ; á ukázał mu Pán wƺyſtkę źiemię od Gáláád áż do Dán ;
Y wƺyſtkę źiemię Neftálimowę / y źiemię Efráimowę / y Mánáſeſowę / y wƺyſtkę źiemię Judowę áż do morzá oſtátniego.
Y ſtronę południową / y równinę doliny Jerychá / miáſtá oſádzonego pálmámi / áż do Segor.
Tedy mu rzekł Pán : Táć jeſt źiemiá / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : Náśieniu twemu dam ją ; pokázáłem ją ocżom twoim / ále do niej nie wnijdźieƺ.
Y umárł tám Mojzeƺ / ſługá Páńſki / w źiemi Moábſkiej / według ſłowá Páńſkiego.
Y pogrzebł go Pán w dolinie / w źiemi Moábſkiej / przećiw Betfegorowi / á nie dowiedźiał śię nikt o grobie jego áż do dniá tego.
A Mojzeƺ miał ſto y dwádźieśćiá lat / gdy umárł ; nie záćmiło śię oko jego / áni śię náruƺyłá cżerſtwość jego.
Y płákáli ſynowie Izráelſcy Mojżeƺá ná polách Moábſkich przez trzydźieśći dni. A gdy śię ſkońcżyły dni płácżu y żáłoby Mojżeƺowej :
Tedy Jozue / ſyn Nunów / nápełniony jeſt duchem mądrośći / bo był włożył Mojzeƺ ręce ſwoje nań : y poſłuƺni mu byli ſynowie Izráelſcy / á cżynili / jáko przykázał Pán Mojżeƺowi.
Y nie powſtał prorok więcey w Izráelu podobny Mojżeƺowi / któregoby ták znał Pán / twárzą w twárz.
We wƺyſtkich znákách / y cudách / dla których go był poſłał Pán / áby je cżynił w źiemi Egipſkiej przed Fáráonem y przed wƺyſtkimi ſługámi jego / y przed wƺyſtką źiemią jego.
Tákże we wƺyſtkich ſpráwách ręki możney / y we wƺyſtkich poſtráchách wielkich / które cżynił Mojzeƺ przed ocżymá wƺyſtkiego Izráelá.
