Tedy záwołáwƺy Mojżeƺ wƺyſtkiego Izráelá ; mówił do nich : Słucháj Izráelu / uſtáw y ſądów / które já dźiś mówię w uƺy wáƺe ; náucżćie śię ich / á przeſtrzegájćie tego / ábyśćie je cżynili.
Pán / Bóg náƺ / ucżynił z námi przymierze ná górze Horeb.
Nie z ojcy náƺymi ucżynił Pán to przymierze / ále z námi / którzyśmy tu dźiś wƺyſcy żywi.
Twárzą w twárz mówił Pán z wámi ná górze / z pośrodku ogniá /
( A jám ſtáł między Pánem / y między wámi ná on cżáſ / ábym wám odnośił ſłowo Páńſkie ; bośćie śię báli ogniá / á nie wſtąpiliśćie ná górę ) y rzekł :
Jám jeſt Pán / Bóg twój / którym ćię wywiódł z źiemi Egipſkiej / z domu niewoli.
Nie będźieƺ miáł bogów innych przedemną.
Nie cżyń ſobie obrázu rytego / áni żádnego podobieńſtwá tych rzecży / które ſą ná niebie wzgórę / y które ná źiemi niſko / y które w wodách pod źiemią ;
Nie będźieƺ śię im kłániáł / áni ich chwálił : bom Já Pán / Bóg twój / Bóg záwiſny w miłośći / náwiedzájący niepráwość ojców nád ſyny do trzećiego y do cżwártego pokoleniá tych / którzy mię nienáwidzą ;
A cżyniący miłośierdźie nád tyśiącámi tych / którzy mię miłują / y ſtrzegą przykázáń moich.
Nie bierz imieniá Páná / Bogá twego / ná dáremno : bo śię będźie mśćił Pán nád tym / który imię jego ná dáremno bierze.
Przeſtrzegáj dniá ſobotniego / ábyś go święćił / jákoć rozkázáł Pán / Bóg twój.
Przez ƺeść dni będźieƺ robił / y wykonáƺ wƺeláką robotę twoję ;
Ale dniá śiódmego jeſt odpocżynienie Páná / Bogá twego ; nie cżyń żádnej roboty / ty y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y wół twój / y ośieł twój / y káżde bydlę twoje / y gość twój / który jeſt w brámách twoich / áby odpocżynął ſługá twój / y ſłużebnicá twojá / jáko y ty.
A pámiętáj / żeś był niewolnikiem w źiemi Egipſkiej / y wywiódł ćię Pán / Bóg twój / ſtámtąd ręką możną / y rámieniem wyćiągnionem ; przetoż ći przykázáł Pán / Bóg twój / ábyś obchodźił dźień ſobotni.
Cżćij ojcá twego y mátkę twoję / jákoć przykázáł Pán / Bóg twój / áby przedłużone były dni twoje / y żebyć śię dobrze dźiáło ná źiemi / którą Pán / Bóg twój / dá tobie.
Nie będźieƺ zábijáł.
Nie będźieƺ cudzołożył.
Nie będźieƺ krádł.
Nie będźieƺ mówił przećiw bliźniemu twemu świádectwá fáłƺywego.
Nie będźieƺ pożądáł żony bliźniego twego / áni będźieƺ pożądáł domu bliźniego twego / roli jego / y ſługi jego / y ſłużebnicy jego / wołu jego / y oſłá jego / y wƺyſtkich rzecży / które ſą bliźniego twego.
Teć ſłowá mówił Pán do wƺyſtkiego zgromádzeniá wáƺego ná górze z pośrodku ogniá / obłoku / y mgły / głoſem wielkim / á nic więcej nie przydáł / y nápiſáł je ná dwóch táblicách kámiennych / które mnie oddáł.
Y ſtáło śię / gdyśćie uſłyƺeli głoſ z pośrodku ćiemnośći ; gdy górá ogniem páłáłá / żeśćie przyſtąpili do mnie / wƺyſtkie kśiążętá pokoleń wáƺych / y ſtárśi wáśi /
Y mówiliśćie : Oto nám ukázáł Pán / Bóg náƺ / chwáłę ſwoję / y wielmożność ſwoję / á głoſ jego ſłyƺeliśmy / z pośrodku ogniá ; dźiś widźieliśmy / że Bóg mówił z cżłowiekiem / á cżłowiek żyw zoſtáł.
A ták teráz przecżże mámy pomrzeć? ábowiem náſ ten ogień wielki pożre ; jeźli jeƺcże ſłyƺeć będźiemy głoſ Páná / Bogá náƺego / pomrzemy.
Abowiem cóż jeſt wƺelkie ćiáło / áby ſłyƺáło głoſ Bogá żywiącego / mówiącego z pośrodku ogniá / jáko my / á żywo zoſtáło?
Idźże ty / á wyſłucháj wƺyſtkiego / co będźie mówił Pán / Bóg náƺ ; ty záś powieƺ nám wƺyſtko / co do ćiebie mówić będźie Pán / Bóg náƺ / á my ſłucháć y cżynić to będźiemy.
A uſłyƺáwƺy Pán głoſ ſłów wáƺych / gdyśćie mówili do mnie / rzekł mi Pán : Słyƺáłem głoſ ſłów ludu tego / które mówili do ćiebie ; dobrze wƺyſtko mówili / co mówili.
Ktoby im to dáł / żeby ſerce ich było tákie / áby śię mnie báli / y ſtrzegli wƺyſtkich przykázáń moich po wƺyſtkie dni / áby śię im dobrze dźiáło y ſynom ich ná wieki.
Idźże / á rzecż im : Wróććie śię do námiotów wáƺych.
A ty tu zoſtáń przy mnie / y opowiem tobie wƺyſtkie przykázániá / y uſtáwy / y ſądy / których ich náucżáć będźieƺ / áby je cżynili w źiemi / którą Já im dáwám / áby ją pośiedli.
Przetoż ſtrzeżćie / ábyśćie cżynili / jáko wám rozkázáł Pán / Bóg wáƺ / nie uchylájąc śię ná práwo áni ná lewo.
Wƺelką też drogą / którą wám przykázáł Pán / Bóg wáƺ / chodźić będźiećie / ábyśćie żyli / y dobrze śię wám dźiáło / y żebyśćie przedłużyli dni ſwoje ná źiemi / którą pośiądźiećie.
