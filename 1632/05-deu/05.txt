Tedy záwołáwƺy Mojzeƺ wƺyſtkiego Izráelá ; mówił do nich : Słucháj Izráelu / uſtáw y ſądów / które ja dźiś mówię w uƺy wáƺe ; náucżćie śię ich / á przeſtrzegájćie tego / ábyśćie je cżynili.
Pán / Bóg náƺ / ucżynił z námi przymierze ná górze Horeb.
Nie z ojcy náƺymi ucżynił Pán to przymierze / ále z námi / którzyśmy tu dźiś wƺyſcy żywi.
Twárzą w twárz mówił Pán z wámi ná górze / z pośrzodku ogniá /
( A jam ſtał miedzy Pánem / y miedzy wámi ná on cżás / ábym wam odnośił ſłowo Páńſkie ; bośćie śię báli ogniá / á nie wſtąpiliśćie ná górę ) y rzekł :
Jam jeſt Pán / Bóg twój / którym ćię wywiódł z źiemi Egipſkiej / z domu niewoli.
Nie będźieƺ miał bogów innych przedemną.
Nie cżyń ſobie obrázu rytego / áni żádnego podobieńſtwá tych rzecży / które ſą ná niebie wzgórę / y które ná źiemi niſko / y które w wodách pod źiemią ;
Nie będźieƺ śię im kłániał / áni ich chwálił : bom Ja Pán / Bóg twój / Bóg záwiſny w miłośći / náwiedzájący niepráwość ojców nád ſyny do trzećiego y do cżwártego pokolenia tych / którzy mię nienáwidzą ;
A cżyniący miłośierdźie nád tyśiącámi tych / którzy mię miłują / y ſtrzegą przykázáń mojich.
Nie bierz imienia Páná / Bogá twego / ná dáremno : bo śię będźie mśćił Pán nád tym / który imię jego ná dáremno bierze.
Przeſtrzegáj dniá ſobotniego / ábyś go święćił / jákoć rozkázał Pán / Bóg twój.
Przez ƺeść dni będźieƺ robił / y wykonáƺ wƺeláką robotę twoję ;
Ale dniá śiódmego jeſt odpocżynienie Páná / Bogá twego ; nie cżyń żádney roboty / ty y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y wół twój / y ośieł twój / y káżde bydlę twoje / y gość twój / który jeſt w bramách twojich / áby odpocżynął ſługá twój / y ſłużebnicá twojá / jáko y ty.
A pámiętáj / żeś był niewolnikiem w źiemi Egipſkiej / y wywiódł ćię Pán / Bóg twój / z támtąd ręką możną / y rámieniem wyćiągnionem ; przetoż ći przykázał Pán / Bóg twój / ábyś obchodźił dźień ſobotni.
Cżćij ojcá twego y mátkę twoję / jákoć przykázał Pán / Bóg twój / áby przedłużone były dni twoje / y żebyć śię dobrze dźiáło ná źiemi / którą Pán / Bóg twój / da tobie.
Nie będźieƺ zábijał.
Nie będźieƺ cudzołożył.
Nie będźieƺ krádł.
Nie będźieƺ mówił przećiw bliźniemu twemu świádectwá fáłƺywego.
Nie będźieƺ pożądał żony bliźniego twego / áni będźieƺ pożądał domu bliźniego twego / roli jego / y ſługi jego / y ſłużebnicy jego / wołu jego / y oſłá jego / y wƺyſtkich rzecży / które ſą bliźniego twego.
Teć ſłowá mówił Pán do wƺyſtkiego zgromádzenia wáƺego ná górze z pośrzodku ogniá / obłoku / y mgły / głoſem wielkim / á nic więcey nie przydał / y nápiſał je ná dwu táblicách kámiennych / które mnie oddał.
Y ſtáło śię / gdyśćie uſłyƺeli głos z pośrzodku ćiemnośći ; gdy górá ogniem páłáłá / żeśćie przyſtąpili do mnie / wƺyſtkie kśiążętá pokoleń wáƺych / y ſtárśi wáśi /
Y mówiliśćie : Oto nam ukázał Pán / Bóg náƺ / chwałę ſwoję / y wielmożność ſwoję / á głos jego ſłyƺeliśmy / z pośrzodku ogniá ; dźiś widźieliśmy / że Bóg mówił z cżłowiekiem / á cżłowiek żyw zoſtał.
A ták teraz przecżże mamy pomrzeć? ábowiem nas ten ogień wielki pożre ; jeſli jeƺcże ſłyƺeć będźiemy głos Páná / Bogá náƺego / pomrzemy.
Abowiem cóż jeſt wƺelkie ćiáło / áby ſłyƺáło głos Bogá żywiącego / mówiącego z pośrzodku ogniá / jáko my / á żywo zoſtáło?
Idźże ty / á wyſłucháj wƺyſtkiego / co będźie mówił Pán / Bóg náƺ ; ty záś powieƺ nam wƺyſtko / co do ćiebie mówić będźie Pán / Bóg náƺ / á my ſłucháć y cżynić to będźiemy.
A uſłyƺáwƺy Pán głos ſłów wáƺych / gdyśćie mówili do mnie / rzekł mi Pán : Słyƺáłem głos ſłów ludu tego / które mówili do ćiebie ; dobrze wƺyſtko mówili / co mówili.
Ktoby im to dał / żeby ſerce ich było tákie / áby śię mnie báli / y ſtrzegli wƺyſtkich przykázáń mojich po wƺyſtkie dni / áby śię im dobrze dźiáło y ſynom ich ná wieki.
Idźże / á rzecż im : Wróććie śię do námiotów wáƺych.
A ty tu zoſtáń przy mnie / y opowiem tobie wƺyſtkie przykázánia / y uſtáwy / y ſądy / których ich náucżáć będźieƺ / áby je cżynili w źiemi / którą Ja im dáwám / áby ją pośiedli.
Przetoż ſtrzeżćie / ábyśćie cżynili / jáko wam rozkázał Pán / Bóg wáƺ / nie uchylájąc śię ná práwo áni ná lewo.
Wƺelką też drogą / którą wam przykázał Pán / Bóg wáƺ / chodźić będźiećie / ábyśćie żyli / y dobrze śię wam dźiáło / y żebyśćie przedłużyli dni ſwoje ná źiemi / którą pośiądźiećie.
