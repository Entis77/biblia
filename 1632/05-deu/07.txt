Gdy ćię wprowádźi Pán / Bóg twój / do źiemi / do którey wchodźiƺ / ábyś ją pośiádł / y wytráći národów wiele przed twárzą twoją / Hetejcżyká / y Gergiezejcżyká / y Amorejcżyká / y Chánánejcżyká / y Ferezejcżyká / y Hewejcżyká y Jebuzejcżyká / śiedm národów / więkƺych / y możniejƺych / niżliś ty ;
A podá je Pán / Bóg twój / tobie / iż je poráźiƺ : tedy wytráćiƺ je do ƺcżątku / nie będźieƺ brał z nimi przymierza / áni śię zlitujeƺ nád nimi ;
Ani śię ſpowinowáćiƺ z nimi : córki ſwej nie dáƺ ſynowi jego / y córki jego nie weźmieƺ ſynowi ſwemu ;
Boby zwiodłá ſyná twego / żeby nie ƺedł zá mną / á ſłużyliby bogom cudzym ; zácżem zápáliłby śię gniew Páńſki przećiwko wam / á wytráćiłby was prędko.
Ale owƺem to im ucżyńćie : Ołtárze ich poobáláćie / á ſłupy ich pokruƺyćie / y gáje ich poświęcone wyrąbićie / á ryte ich báłwány ogniem popálićie :
Abowiemeś ty lud święty Pánu / Bogu twemu ; ćiebie obrał Pán / Bóg twój / ábyś mu był oſobliwym ludem ze wƺyſtkich národów / które ſą ná źiemi.
Nie przeto / że was więcey nád inne národy / przyłącżył śię Pán do was / y obrał was / gdyż was było nájmniej ze wƺyſtkich národów ;
Ale iż was umiłował Pán / y ſtrzedz chćiał oney przyśięgi / którą przyśiągł ojcom wáƺym / wywiódł was Pán ręką możną / y wykupił was z domu niewoli / z ręki Fáráoná / królá Egipſkiego.
A ták wiedz / że Pán / Bóg twój / jeſt Bóg / Bóg wierny / chowájący przymierze / y miłośierdźie tym / którzy go miłują y ſtrzegą przykázáń jego / do tyśiącżnego pokolenia ;
A oddájący tym / którzy go máją w nienáwiśći / káżdemu w twárz jego / áby go wytráćił ; nie omieƺká temu / który go ma w nienáwiśći / w twárz jego oddá mu.
Przetoż przeſtrzegáj przykázánia / y uſtáw / y ſądów / które ja dźiś rozkázuję tobie / ábyś je cżynił.
Y ſtánie śię / że jeſli ſłucháć ſądów tych / á przeſtrzegáć / y cżynić je będźiećie / tedy też dotrzymá Pán / Bóg twój / tobie przymierza / y miłośierdźiá / które poprzyśiągł ojcom twoim.
Y umiłuje ćię ; y ubłogoſłáwi ćię y rozmnoży ćię ; bo pobłogoſłáwi owocowi żywotá twego / y owocowi źiemi twojey / zbożu twojimu / y winu twojimu / y oliwie twojey / płodowi krów twojich / y trzodom owiec twojich w źiemi / o którą przyśiągł ojcom twoim / że ją da tobie.
Błogoſłáwionym będźieƺ nád wƺyſtkie národy ; nie będźie u ćiebie niepłodny / y niepłodná / áni miedzy bydłem twojim.
Oddáli też Pán od ćiebie káżdą niemoc / y wƺelkie choroby egipſkie złe / o których wieƺ ; nie dopuśći ich ná ćię / ále je przepuśći ná wƺyſtkie / którzy ćię nienáwidzą.
Y wytráćiƺ wƺyſtkie národy / które Pán / Bóg twój / podá tobie ; nie ſfolguje im oko twoje / áni będźieƺ chwálił bogów ich ; boćby to było śidłem.
Jeſlibyś rzekł w ſercu ſwem : Więkƺe ſą te národy niżli ja / jákoż je będę mógł wygnáć?
Nie bój śię ich ; ále pilnie pámiętáj ná to / co ucżynił Pán / Bóg twój / Fáráonowi / y wƺyſtkim Egipcżánom ;
Ná one kuƺeniá wielkie / które widźiáły ocży twoje / y ná znáki / ná cudá / y ná rękę możną / y ná rámię wyćiągnione / którem ćię wywiódł Pán / Bóg twój ; tákći ucżyni Pán / Bóg twój / wƺyſtkim národom / których śię ty twárzy boiƺ.
Nád to pośle Pán / Bóg twój / ná nie śierƺenie / áż wygubi oſtátki ich / y te / którzyby śię pokryli przed tobą.
Nie lękájże śię twárzy ich / ábowiem Pán / Bóg twój / jeſt w pośrzodku ćiebie / Bóg wielki y ſtráƺny.
Y wyniƺcży Pán / Bóg twój / národy one przed tobą / po lekku y po troƺe ;
Nie będźieƺ ich mógł wytráćić prędko / by śię ſnadź nie námnożyło przećiw tobie beſtyi polnych.
Y podá je tobie Pán / Bóg twój / y zetrze je ſtárćiem wielkiem / áż będą wyniƺcżeni. A podá króle ich w ręce twoje / y wygubiƺ imię ich pod niebem ; nie oſtoi śię żáden przed tobą / áż je wytráćiƺ.
Obrázy ryte bogów ich popáliƺ ogniem ; nie będźieƺ pożądał ſrebrá / áni złotá / które jeſt ná nich / ábyś je miał ſobie bráć / byś śię ſnadź nie uśidlił w nim / poniewáż to obrzydliwość jeſt Pánu / Bogu twemu.
Nie wnośże obrzydliwośći w dom twój / ábyś śię nie ſtał przeklęſtwem / jáko y oná ; wielce śię nią brzydźić będźieƺ / y bárzo ją ſobie obmierzyƺ / poniewáż przeklęſtwem jeſt.
