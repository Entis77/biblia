Gdy wytráći Pán / Bóg twój / národy / których ziemię Pán / Bóg twój / dáwá tobie / á pośiędzieƺ ją / y mieƺkáć będzieƺ w miáſtách ich / y w domách ich :
Trzy miáſtá odłącżyƺ ſobie w pośród ziemi twojej / którą Pán / Bóg twój dáwá tobie w ośiádłość ;
Wyproſtujeƺ ſobie drogę / y rozdzieliƺ ná trzy cżęśći gránice ziemi twojej / którąć w dziedzictwo podá Pán / Bóg twój / żeby tám ućiekáł káżdy mężobójcá.
A toć będzie práwo mężobójcy / który tám ućiecże / áby żyw zoſtáł / gdyby zábił bliźniego ſwego z nieobácżeniá / á niemiáłby go przedtem w nienáwiśći ;
Jáko gdyby kto ƺedł z bliźnim ſwym do láſu drwá rąbáć / á zánióſłby śię ręką ſwą z śiekierą / áby ućiął drzewo / á ſpádłáby śiekierá z toporzyſká / y tráfiłáby bliźniego jego / żeby umárł / ten ućiecże do jednego z tych miáſt / áby żyw zoſtáł ;
By ſnáć nie gonił powinny zábitego mężobójcę onego / gdyby śię rozpáliło ſerce jego / á dogoniwƺy go ná dálekiej drodze / nie zábił go / choćby nie był winien śmierći / poniewáż go nie miáł w nienáwiśći przedtem.
Przetoż já przykázuję tobie / mówiąc : Trzy miáſtá odłącżyƺ ſobie ;
A gdy rozƺerzy Pán / Bóg twój / gránice twoje / jáko przyśiągł ojcom twoim / y dáć wƺyſtkę ziemię / którą záślubił dáć ojcom twoim :
Jeźliże będzieƺ ſtrzegł wƺyſtkich tych przykázáń / y cżynił je / które já dziś rozkázuję tobie / żebyś miłowáł Páná / Bogá twego / y chodził drogámi jego po wƺyſtkie dni / tedy przydáƺ ſobie jeƺcże trzy miáſtá do onych trzech miáſt.
Aby nie byłá wylewáná krew niewinná w pośrodku ziemi twojej / którą Pán / Bóg twój / dáwá tobie w dziedzictwo / áby nie zoſtáłá ná tobie krew.
Ale jeźliby kto májąc w nienáwiśći bliźniego ſwego / á záſádziwƺy śię náń / powſtáłby przećiw jemu / y uderzyłby go śmiertelnie / áżby umárł / á ućiekł do jednego z tych miáſt :
Tedy poślą ſtárśi miáſtá onego / y porwą go ſtámtąd / y wydádzą go w ręce powinnemu zábitego / y umrze.
Nie ſfolguje mu oko twoje ; ále odejmieƺ krew niewinną z Izráelá / ábyć śię dobrze dziáło.
Nie będzieƺ przenośił gránicy bliźniego twego / którą záłożyli przodkowie w dziedzictwie twojim / które ośiędzieƺ w ziemi / którą Pán / Bóg twój / dáwá tobie w ośiádłość.
Nie powſtánie świádek jeden przećiwko cżłowiekowi w jákiejkolwiek niepráwośći / álbo w jákimkolwiek grzechu ze wƺyſtkich grzechów / któremiby kto zgrzeƺył ; w uśćiech dwóch świádków / álbo w uśćiech trzech świádków ſtánie káżde ſłowo.
Jeźliby powſtáł świádek kłámliwy przećiw komu / świádcżąc przećiw niemu / że odſtąpił od Bogá :
Tedy ſtáną oni dwáj mężowie / którzy máją ſpór / przed Pánem / przed kápłány / y przed ſędziámi / którzy będą ná ten cżáſ.
Y będą śię pilnie wywiádowáli ſędziowie ; á obácżąli / że świádek on jeſt świádkiem fáłƺywym / kłámſtwo powiádájąc ná brátá ſwego /
Ucżyńćie mu / jáko on myślił ucżynić brátu ſwemu ; y odejmieƺ złe z pośrodku śiebie ;
Aby drudzy uſłyƺáwƺy to / ulękli śię / y nie dopuƺcżáli śię więcej tákowej złośći cżynić w pośród ćiebie.
A nie ſfolguje oko twoje ; gárdło zá gárdło / oko zá oko / ząb zá ząb / ręká zá rękę / nogá zá nogę będzie.
