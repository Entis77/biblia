Te ſą uſtáwy y ſądy / których ſtrzedz będźiećie / ábyśćie je cżynili w źiemi / którą dáwá Pán / Bóg ojców twoich / tobie / żebyśćie ją dźiedźicżnie trzymáli po wƺyſtkie dni / w których żyć będźiećie ná źiemi.
Zburzyćie do ƺcżętu wƺyſtkie miejſcá / ná których ſłużyli národowie / które wy pośiądźiećie / bogom ſwoim / ná górách wyſokich / y ná págórkách / y pod káżdem drzewem źielonem.
Y porozwáláćie ołtárze ich / y połámiećie ſłupy ich / gáje też ich poświęcone ſpálićie ogniem / y báłwány bogów ich porąbiećie / á wygłádźićie imię ich z miejſcá onego.
Nie ucżynićie ták Pánu Bogu wáƺemu ;
Ale miejſcá / które obierze Pán / Bóg wáƺ / ze wƺyſtkich pokoleń wáƺych / áby tám wyſtáwił imię ſwoje / y mieƺkáł ná nim / będźiećie ƺukáć y do niego śię ſchádzáć.
Támże będźiećie przynośić cáłopáleniá wáƺe / y ofiáry wáƺe / y dźieśięćiny wáƺe / y ofiáry rąk wáƺych / y śluby wáƺe / y dobrowolne dáry wáƺe / tákże pierworodztwá krów wáƺych / y owiec wáƺych.
Y tám będźiećie jeść przed Pánem / Bogiem wáƺym ; y będźiećie śię weſelić we wƺyſtkiem / do cżego śćiągniećie ręce wáƺe / wy y domy wáƺe / w którychći pobłogoſłáwi Pán / Bóg twój.
Nie będźiećie cżynić według tego wƺyſtkiego / jáko my tu dźiś cżynimy / káżdy / co mu śię zdá dobrego w ocżách jego ;
Abowiemeśćie jeƺcże nie przyƺli do odpocżynieniá / y do dźiedźictwá / które Pán / Bóg twój / dáwá tobie.
Ale gdy przeƺedłƺy zá Jordán / mieƺkáć będźiećie w źiemi / którą Pán / Bóg wáƺ / dáwá wám dźiedźicżnie ośieść / y dá wám odpocżynek od wƺyſtkich nieprzyjáćiół wáƺych w około / á mieƺkáć będźiećie bezpiecżnie :
Tedy ná miejſce / które obierze ſobie Pán / Bóg wáƺ / áby tám mieƺkáło imię jego / tám znośić będźiećie wƺyſtko / co já wám rozkázuję / cáłopáleniá wáƺe / y ofiáry wáƺe / dźieśięćiny wáƺe / y ofiáry rąk wáƺych / y wƺyſtko / co przedniego jeſt w ślubiech wáƺych / które ślubowáć będźiećie Pánu ;
Y weſelić śię będźiećie przed Pánem / Bogiem wáƺym / wy / y ſynowie wáśi / y córki wáƺe / y ſłudzy wáśi / y ſłużebnice wáƺe / y Lewitá / który jeſt w brámách wáƺych / poniewáż niemá dźiáłu / áni dźiedźictwá z wámi.
Strzeżże śię / ábyś nie ofiárowáł cáłopáleniá twego ná káżdem miejſcu / gdźiebyć śię zdáło :
Ale tylko ná miejſcu / któreby obráł Pán / w któremkolwiek pokoleniu twojim / tám ofiárowáć będźieƺ cáłopáleniá twoje / y tám cżynić będźieƺ wƺyſtko / co já rozkázuję tobie.
A wƺákże / jeźli śię upodobá duƺy twojej / zábijeƺ ſobie / y będźieƺ jádł mięſo według błogoſłáwieńſtwá Páná / Bogá twego / które dá tobie we wƺyſtkich brámách twoich ; niecżyſty y cżyſty jeść je będźie / jáko ſárnę álbo jeleniá.
Krwi tylko jeść nie będźiećie / ná źiemię wylejećie ją / jáko wodę.
Nie będźieƺ mógł jeść w brámách twoich dźieśięćiny zbożá twego / y winá twego / y oliwy twojej / y pierworodztw krów twoich / y owiec twoich / y wƺyſtkich ślubów twych / którebyś ślubowáł / y dobrowolnych dárów twoich / tákże y ofiáry ręki twej.
Ale przed Pánem / Bogiem twoim / jeść je będźieƺ ná miejſcu / które obierze Pán / Bóg twój / ty y ſyn twój / y córká twojá / y ſługá twój / y ſłużebnicá twojá / y Lewitá / który jeſt w brámách twoich ; y będźieƺ śię weſelił przed Pánem / Bogiem twoim / we wƺyſtkich rzecżách / do których śćiągnieƺ ręce twoje.
A ſtrzeż śię / ábyś ſnáć nie opuƺcżáł Lewity po wƺyſtkie dni twoje w źiemi twojej.
Gdy rozƺerzy Pán / Bóg twój / gránicę twoję / jákoć powiedźiáł / y rzekłbyś : Będę jádł mięſo / przeto że pożądá duƺá twojá jeść mięſá ; według wƺyſtkiej żądośći duƺy twojej będźieƺ jádł mięſo.
A jeźliby dálekie było od ćiebie miejſce / które obierze Pán / Bóg twój / áby tám przebywáło imię jego / tedy zábijeƺ z wołów twoich / y z owiec twoich / któreć dá Pán / jákom ći rozkázáł / y będźieƺ jádł w brámách twoich według wƺyſtkiej żądośći duƺy twojej.
Ale jáko jedzą ſárnę y jeleniá / ták je jeść będźieƺ ; niecżyſty y cżyſty zárówno jeść je będą.
Tylko bądź ſtátecżnym / ábyś krwi nie jádáł / bo krew jeſt duƺá ; przetoż nie będźieƺ jádł duƺy z mięſem jej.
Nie jedzże jej / ná źiemię ją wylej jáko wodę.
Nie jedz jej / áby śię dobrze dźiáło tobie / y ſynom twoim po tobie / gdybyś cżynił / co dobrego jeſt przed ocżymá Páńſkimi.
Ale poświęcone rzecży twoje / które będźieƺ miáł / y śluby twoje / weźmieƺ y przynieśieƺ ná miejſce / które obierze Pán ;
Y będźieƺ ofiárowáł cáłopáleniá twoje / mięſo y krew / ná ołtárzu Páná / Bogá twego ; ále krew inƺych ofiár twoich wyláná będźie ná ołtárzu Páná / Bogá twego ; mięſo jednák jeść będźieƺ.
Przeſtrzegájże / á ſłucháj tych wƺyſtkich ſłów / które já przykázuję tobie / áby dobrze było tobie / y ſynom twoim po tobie / áż ná wieki / gdy cżynić będźieƺ to / co dobrego y práwego jeſt przed ocżymá Páná / Bogá twego.
Gdy wytráći Pán / Bóg twój / przed oblicżem twojim te národy / do których ty wnijdźieƺ / ábyś je pośiádł / y opánowáł je / y mieƺkáł w źiemi ich.
Strzeżże śię / ábyś śię nie uśidlił / idąc zá nimi / gdy wytráceni będą przed twárzą twoją ; áni śię też pytáj ná bogi ich / mówiąc : Jáko ći národowie ſłużyli bogom ſwoim / ták y já też ucżynię.
Nie ucżyniƺ ták Pánu / Bogu twemu ; bo wƺyſtko / cżem śię brzydźi Pán / y cżego nienáwidźi / cżynili bogom ſwoim ; tákże też y ſyny ſwoje / y córki ſwoje pálili ogniem bogom ſwoim.
Cokolwiek já wám rozkázuję / tego ſtrzedz będźiećie / ábyśćie cżynili ; nie przydáƺ nic do tego / áni też ujmieƺ z tego.
