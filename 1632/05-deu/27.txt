Tedy przykázáł Mojżeƺ y ſtárśi Izráelſcy ludowi / mówiąc : Strzeżćie wƺelkiego przykázániá / które já wám dziś przykázuję.
Y ſtánie śię / że którego dniá przejdziećie przez Jordán do ziemi / którąć Pán / Bóg twój / dáwá / wyſtáwiƺ ſobie kámienie wielkie / y potynkujeƺ je wápnem ;
Y nápiƺeƺ ná nich wƺyſtkie ſłowá zákonu tego / ſkoro przejdzieƺ / ábyś wƺedł do ziemi / którą Pán / Bóg twój / dáwá tobie / do ziemi opływájącej mlekiem y miodem / jákoć obiecáł Pán / Bóg ojców twoich.
Gdy tedy przejdziećie przez Jordán / wyſtáwićie te kámienie / o których já wám dziś rozkázuję / ná górze Hebál / y potynkujeƺ je wápnem.
Támże zbudujeƺ ołtárz Pánu / Bogu twemu / ołtárz z kámieniá ; nie będzieƺ ich ćioſáł żelázem.
Z kámieniá cáłego zbudujeƺ ołtárz Páná / Bogá ſwego / y ofiárowáć będzieƺ ná nim cáłopáleniá Pánu / Bogu twemu.
Tákże ofiárowáć będzieƺ ofiáry ſpokojne / y będzieƺ tám jádł / á będzieƺ śię weſelił przed oblicżem Páná / Bogá twego ;
Y nápiƺeƺ ná onych kámieniách wƺyſtkie ſłowá zákonu tego znácżnie y jáśnie.
Y rzekł Mojżeƺ y kápłáni Lewitowie do wƺyſtkiego Izráelá mówiąc : Pilnuj á ſłucháj / Izráelu! dziś ſtáłeś śię ludem Páná / Bogá twego.
Przetoż będzieƺ poſłuƺnym głoſowi Páná / Bogá twego / á będzieƺ cżynił przykázániá jego / y uſtáwy jego / które já dziś przykázuję tobie.
Tedy przykázáł Mojżeƺ ludowi dniá onego / mówiąc :
Ci ſtáną / áby błogoſłáwili ludowi ná górze Gáryzym / gdy przejdziećie przez Jordán : Symeon / y Lewi / y Judá / y Iſáſchár / y Józef / y Benjámin.
Ci záś ſtáną ku przeklinániu ná górze Hebál : Ruben / Gád / y Aſer / y Zábulon / Dán / y Neftáli.
Y oświádcżą śię Lewitowie / á mówić będą do wƺyſtkich mężów Izráelſkich głoſem wynioſłym :
Przeklęty cżłowiek / któryby cżynił obráz ryty / álbo odlewány / obrzydliwość Páńſką / robotę rąk rzemieślnicżych / choćby to poſtáwił ná miejſcu ſkrytem ; tedy odpowie wƺyſtek lud / y rzecże : Amen.
Przeklęty / kto lekce wáży ojcá ſwego / y mátkę ſwoję ; y rzecże wƺyſtek lud Amen.
Przeklęty / który przenośi gránicę bliźniego ſwego ; á rzecże wƺyſtek lud Amen.
Przeklęty / który zwodzi ślepego z drogi / áby błądził ; y rzecże wƺyſtek lud Amen.
Przeklęty / który wywrácá ſpráwiedliwość przychodniowi / śieroćie / y wdowie ; á rzecże wƺyſtek lud Amen.
Przeklęty / ktoby obcowáł z żoną ojcá ſwego / bo odkrył podołek ojcá ſwego ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcowáł z jákiemkolwiek bydlęćiem ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcowáł z śioſtrą ſwoją / córką ojcá ſwego álbo córką mátki ſwojej ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcowáł z świekrą ſwoją ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby zábił tájemnie bliźniego ſwego ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby bráł dáry / áby zábił cżłowieká / krew niewinną ; y rzecże wƺyſtek lud Amen.
Przeklęty / ktoby nie zoſtáł przy ſłowách zákonu tego / áby je cżynił ; y rzecże wƺyſtek lud Amen.
