Tedy przykázał Mojzeƺ y ſtárśi Izráelſcy ludowi / mówiąc : Strzeżćie wƺelkiego przykázánia / które ja wam dźiś przykázuję.
Y ſtánie śię / że którego dniá przejdźiećie przez Jordán do źiemi / którąć Pán / Bóg twój / dáwá / wyſtáwiƺ ſobie kámienie wielkie / y potynkujeƺ je wápnem ;
Y nápiƺeƺ ná nich wƺyſtkie ſłowá zakonu tego / ſkoro przejdźieƺ / ábyś wƺedł do źiemi / którą Pán / Bóg twój / dáwá tobie / do źiemi opływájącey mlekiem y miodem / jákoć obiecał Pán / Bóg ojców twojich.
Gdy tedy przejdźiećie przez Jordán / wyſtáwićie te kámienie / o których ja wam dźiś rozkázuję / ná górze Hebál / y potynkujeƺ je wápnem.
Támże zbudujeƺ ołtárz Pánu / Bogu twemu / ołtárz z kámieniá ; nie będźieƺ ich ćioſał żelázem.
Z kámieniá cáłego zbudujeƺ ołtárz Páná / Bogá ſwego / y ofiárowáć będźieƺ ná nim cáłopalenia Pánu / Bogu twemu.
Tákże ofiárowáć będźieƺ ofiáry ſpokojne / y będźieƺ tám jádł / á będźieƺ śię weſelił przed oblicżem Páná / Bogá twego ;
Y nápiƺeƺ ná onych kámieniách wƺyſtkie ſłowá zakonu tego znácżnie y jáśnie.
Y rzekł Mojzeƺ y kápłáni Lewitowie do wƺyſtkiego Izráelá mówiąc : Pilnuj á ſłucháj / Izráelu! dźiś ſtáłeś śię ludem Páná / Bogá twego.
Przetoż będźieƺ poſłuƺnym głoſowi Páná / Bogá twego / á będźieƺ cżynił przykázánia jego / y uſtáwy jego / które ja dźiś przykázuję tobie.
Tedy przykázał Mojzeƺ ludowi dniá onego / mówiąc :
Ci ſtáną / áby błogoſłáwili ludowi ná górze Gáryzym / gdy przejdźiećie przez Jordán : Symeon / y Lewi / y Judá / y Iſáſchár / y Józef / y Benjámin.
Ci záś ſtáną ku przeklinániu ná górze Hebál : Ruben / Gád / y Aſer / y Zábulon / Dán / y Neftáli.
Y oświádcżą śię Lewitowie / á mówić będą do wƺyſtkich mężów Izráelſkich głoſem wynioſłym :
Przeklęty cżłowiek / któryby cżynił obraz ryty / álbo odlewány / obrzydliwość Páńſką / robotę rąk rzemieślnicżych / choćby to poſtáwił ná miejſcu ſkrytem ; tedy odpowie wƺyſtek lud / y rzecże : Amen.
Przeklęty / kto lekce wáży ojcá ſwego / y mátkę ſwoję ; y rzecże wƺyſtek lud Amen.
Przeklęty / który przenośi gránicę bliźniego ſwego ; á rzecże wƺyſtek lud Amen.
Przeklęty / który zwodźi ślepego z drogi / áby błądźił ; y rzecże wƺyſtek lud Amen.
Przeklęty / który wywrácá ſpráwiedliwość przychodniowi / śieroćie / y wdowie ; á rzecże wƺyſtek lud Amen.
Przeklęty / ktoby obcował z żoną ojcá ſwego / bo odkrył podołek ojcá ſwego ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcował z jákiemkolwiek bydlęćiem ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcował z śioſtrą ſwoją / córką ojcá ſwego álbo córką mátki ſwojey ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby obcował z świekrą ſwoją ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby zábił tájemnie bliźniego ſwego ; y rzecże wƺyſtek lud Amen.
Przeklęty / któryby brał dáry / áby zábił cżłowieká / krew niewinną ; y rzecże wƺyſtek lud Amen.
Przeklęty / ktoby nie zoſtał przy ſłowách zakonu tego / áby je cżynił ; y rzecże wƺyſtek lud Amen.
