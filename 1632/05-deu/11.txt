Miłujże tedy Páná / Bogá twego / y przeſtrzegáj obrzędów jego / y uſtáw jego / y ſądów jego / y przykázáń jego / po wƺyſtkie dni.
A poznájćie dźiś ( bo nie mówię do ſynów wáƺych / którzy nie ználi / áni widźieli káránia Páná / Bogá wáƺego ; ) wielmożność jego / rękę jego mocną / y rámię jego wyćiągnione ;
Y cudá jego / y ſpráwy jego / które ucżynił w pośród Egiptu / Fáráonowi / królowi Egipſkiemu / y wƺyſtkiej źiemi jego ;
Y co ucżynił wojſku Egipſkiemu / koniom jego / y wozom jego ; który ſpráwił / że je okryły wody morzá cżerwonego / gdy was gonili / y wytráćił je Pán áż do dniá tego ;
Tákże co wam ucżynił ná puƺcży / áżeśćie przyƺli ná to miejſce ;
Y co ucżynił Dátánowi / y Abironowi / ſynom Elijábá / ſyná Rubenowego / jáko otworzywƺy źiemiá uſtá ſwe / pożárłá je / y domy ich / y námioty ich / y wƺyſtkę májętność / która byłá przy nich / w pośrzodku wƺyſtkiego Izráelá.
A ocży wáƺe widźiáły wƺyſtkie ſpráwy Páńſkie wielkie / które cżynił.
Strzeżćież tedy káżdego przykázánia / które ja przykázuję wam dźiś / ábyśćie śię zmocnili / á wƺedƺy pośiedli źiemię / do którey idźiećie / ábyśćie ją odźiedźicżyli ;
A żebyśćie dni ſwojich przedłużyli ná źiemi / o którą przyśiągł Pán ojcom wáƺym / że im ją da / y náśieniu ich / źiemię opływájącą mlekiem y miodem.
Abowiem źiemiá / do którey ty idźieƺ / ábyś ją pośiádł / nie jeſt jáko źiemiá Egipſká / z którejśćie wyƺli / w którey pośiáwƺy náśienie twoje / pokrápiáćeś muśiał z prácą nóg twojich / jáko ogród járzyny.
Ale źiemiá / do którey idźiećie / ábyśćie ją pośiedli / źiemiá jeſt májącá góry y doliny / która dżdżem niebieſkim odwilżáná bywá :
Ziemiá / o którey Pán / Bóg twój / piecżą ma / y ná którą záwżdy ocży Páná / Bogá twego / pátrzą od pocżątku roku / y áż do ſkońcżeniá jego.
A ták będźiećieli pilnie ſłucháli przykázáń mojich / które ja dźiś przykázuję wam / żebyśćie miłowáli Páná / Bogá wáƺego / y ſłużyli mu ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej :
Tedy dam deƺcż źiemi wáƺej cżáſu ſwego / ránny y późny / y będźieƺ zbierał zbożá twoje / y wino twoje / y oliwę twoję.
Dam też tráwę ná polách twojich / dla bydłá twojego / y będźieƺ jádł / á nájeƺ śię.
Strzeżćież śię / by ſnadź nie było zwiedźione ſerce wáƺe / ábyśćie odſtąpiwƺy nie ſłużyli bogom cudzym / y nie kłániáli śię im ;
Skądby śię zápálił gniew Páńſki przećiwko wam / y zámknąłby niebá / y nie byłoby dżdżu / ániby źiemiá wydáwáłá urodzáju ſwego / y zginęlibyśćie prędko z źiemi tey wyborney / którą Pán dáwá wam.
Przetoż złóżćie te ſłowá moje do ſercá wáƺego / y do umyſłu wáƺego / á uwiążćie je ná znák ná rękách wáƺych / y niech będą jáko nácżelniki miedzy ocżymá wáƺemi.
A náucżájćie ich ſynów wáƺych / rozmáwiájąc o nich / gdy uśiędźieƺ w domu twym / y gdy będźieƺ w drodze / y gdy śię ukłádźieƺ / y gdy wſtánieƺ.
Nápiƺeƺ je też ná podwojách domu twego / y ná brámách twojich.
Aby śię rozmnożyły dni wáƺe / y dni ſynów wáƺych ná źiemi / o którą przyśiągł Pán ojcom wáƺym / że ją im da / póki niebo nád źiemią trwáć będźie.
Bo jeſliż z pilnośćią ſtrzedz będźiećie káżdego przykázánia tego / które ja wam rozkázuję / ábyśćie je cżynili / żebyśćie miłowáli Páná / Bogá wáƺego / y chodźili we wƺyſtkich drogách jego / trwájąc przy nim :
Tedyć wypędźi Pán one wƺyſtkie národy przed wámi / y pośiądźiećie národy więkƺe / y możniejƺe / niżeśćie wy ſámi.
Wƺelkie miejſce / po którem deptáć będźie ſtopá nogi wáƺej / wáƺe będźie ; od puƺcży y Libánu / od rzeki / rzeki Eufrátes / áż do morzá oſtátniego będźie gránicá wáƺá.
Nie oſtoi śię żáden przed wámi ; lękánie wáƺe / y ſtrách wáƺ puśći Pán / Bóg wáƺ / ná oblicże wƺyſtkiej źiemi / którą będźiećie deptáć / jáko wam powiedźiał.
Oto / ja dźiś wam przedkłádám błogoſłáwieńſtwo / y przeklęſtwo ;
Błogoſłáwieńſtwo / będźiećieli poſłuƺni przykázániu Páná Bogá wáƺego / które ja przykázuję wam dźiś ;
A przeklęſtwo / jeſli poſłuƺni nie będźiećie przykázániom Páná Bogá wáƺego / y uſtąpićie z drogi / którą ja wam dźiś rozkázuję / udáwájąc śię zá bogámi obcymi / których nieznáćie /
A gdy ćię wprowádźi Pán / Bóg twój / do źiemi / do którey idźieƺ / ábyś ją pośiádł / tedy dáƺ błogoſłáwieńſtwo to ná górze Gáryzym / á przeklęſtwo ná górze Hebál.
Azaż nie ſą zá Jordánem / zá drogą ná záchód ſłońcá / w źiemi Chánánejcżyków / którzy w polách mieƺkáją / przećiwko Gálgál / przy równinie Morech?
Abowiem wy przejdźiećie zá Jordán / ábyśćie wƺedƺy odźiedźicżyli tę źiemię / którą wam dáwá Pán / Bóg wáƺ / á pośiądźiećie ją y mieƺkáć w niej będźiećie.
Pilnujćież tedy / ábyśćie cżynili wƺyſtkie uſtáwy y ſądy / które ja wam dźiś przedkłádám.
