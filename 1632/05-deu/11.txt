Miłujże tedy Páná / Bogá twego / y przeſtrzegáj obrzędów jego / y uſtáw jego / y ſądów jego / y przykázáń jego / po wƺyſtkie dni.
A poznájćie dziś ( bo nie mówię do ſynów wáƺych / którzy nie ználi / áni widzieli kárániá Páná / Bogá wáƺego ; ) wielmożność jego / rękę jego mocną / y rámię jego wyćiągnione ;
Y cudá jego / y ſpráwy jego / które ucżynił w pośród Egiptu / Fáráonowi / królowi Egipſkiemu / y wƺyſtkiej ziemi jego ;
Y co ucżynił wojſku Egipſkiemu / koniom jego / y wozom jego ; który ſpráwił / że je okryły wody morzá cżerwonego / gdy wáſ gonili / y wytráćił je Pán áż do dniá tego ;
Tákże co wám ucżynił ná puƺcży / áżeśćie przyƺli ná to miejſce ;
Y co ucżynił Dátánowi / y Abironowi / ſynom Elijábá / ſyná Rubenowego / jáko otworzywƺy ziemiá uſtá ſwe / pożárłá je / y domy ich / y námioty ich / y wƺyſtkę májętność / którá byłá przy nich / w pośrodku wƺyſtkiego Izráelá.
A ocży wáƺe widziáły wƺyſtkie ſpráwy Páńſkie wielkie / które cżynił.
Strzeżćież tedy káżdego przykázániá / które já przykázuję wám dziś / ábyśćie śię zmocnili / á wƺedłƺy pośiedli ziemię / do której idziećie / ábyśćie ją odziedzicżyli ;
A żebyśćie dni ſwoich przedłużyli ná ziemi / o którą przyśiągł Pán ojcom wáƺym / że im ją dá / y náśieniu ich / ziemię opływájącą mlekiem y miodem.
Abowiem ziemiá / do której ty idzieƺ / ábyś ją pośiádł / nie jeſt jáko ziemiá Egipſká / z którejśćie wyƺli / w której pośiáwƺy náśienie twoje / pokrápiáćeś muśiáł z prácą nóg twoich / jáko ogród járzyny.
Ale ziemiá / do której idziećie / ábyśćie ją pośiedli / ziemiá jeſt májącá góry y doliny / którá dżdżem niebieſkim odwilżáná bywá :
Ziemiá / o której Pán / Bóg twój / piecżą má / y ná którą záwżdy ocży Páná / Bogá twego / pátrzą od pocżątku roku / y áż do ſkońcżeniá jego.
A ták będziećieli pilnie ſłucháli przykázáń moich / które já dziś przykázuję wám / żebyśćie miłowáli Páná / Bogá wáƺego / y ſłużyli mu ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej :
Tedy dám deƺcż ziemi wáƺej cżáſu ſwego / ránny y późny / y będzieƺ zbieráł zbożá twoje / y wino twoje / y oliwę twoję.
Dám też tráwę ná polách twoich / dlá bydłá twojego / y będzieƺ jádł / á nájeƺ śię.
Strzeżćież śię / by ſnáć nie było zwiedzione ſerce wáƺe / ábyśćie odſtąpiwƺy nie ſłużyli bogom cudzym / y nie kłániáli śię im ;
Skądby śię zápálił gniew Páńſki przećiwko wám / y zámknąłby niebá / y nie byłoby dżdżu / ániby ziemiá wydáwáłá urodzáju ſwego / y zginęlibyśćie prędko z ziemi tej wybornej / którą Pán dáwá wám.
Przetoż złóżćie te ſłowá moje do ſercá wáƺego / y do umyſłu wáƺego / á uwiążćie je ná znák ná rękách wáƺych / y niech będą jáko nácżelniki między ocżymá wáƺemi.
A náucżájćie ich ſynów wáƺych / rozmáwiájąc o nich / gdy uśiędzieƺ w domu twym / y gdy będzieƺ w drodze / y gdy śię ukłádzieƺ / y gdy wſtánieƺ.
Nápiƺeƺ je też ná podwojách domu twego / y ná brámách twoich.
Aby śię rozmnożyły dni wáƺe / y dni ſynów wáƺych ná ziemi / o którą przyśiągł Pán ojcom wáƺym / że ją im dá / póki niebo nád ziemią trwáć będzie.
Bo jeźliż z pilnośćią ſtrzedz będziećie káżdego przykázániá tego / które já wám rozkázuję / ábyśćie je cżynili / żebyśćie miłowáli Páná / Bogá wáƺego / y chodzili we wƺyſtkich drogách jego / trwájąc przy nim :
Tedyć wypędzi Pán one wƺyſtkie národy przed wámi / y pośiądziećie národy więkƺe / y możniejƺe / niżeśćie wy ſámi.
Wƺelkie miejſce / po którem deptáć będzie ſtopá nogi wáƺej / wáƺe będzie ; od puƺcży y Libánu / od rzeki / rzeki Eufráteſ / áż do morzá oſtátniego będzie gránicá wáƺá.
Nie oſtoi śię żáden przed wámi ; lękánie wáƺe / y ſtrách wáƺ puśći Pán / Bóg wáƺ / ná oblicże wƺyſtkiej ziemi / którą będziećie deptáć / jáko wám powiedziáł.
Oto / já dziś wám przedkłádám błogoſłáwieńſtwo / y przeklęſtwo ;
Błogoſłáwieńſtwo / będziećieli poſłuƺni przykázániu Páná Bogá wáƺego / które já przykázuję wám dziś ;
A przeklęſtwo / jeźli poſłuƺni nie będziećie przykázániom Páná Bogá wáƺego / y uſtąpićie z drogi / którą já wám dziś rozkázuję / udáwájąc śię zá bogámi obcymi / których nieznáćie /
A gdy ćię wprowádzi Pán / Bóg twój / do ziemi / do której idzieƺ / ábyś ją pośiádł / tedy dáƺ błogoſłáwieńſtwo to ná górze Gáryzym / á przeklęſtwo ná górze Hebál.
Azáż nie ſą zá Jordánem / zá drogą ná záchód ſłońcá / w ziemi Chánánejcżyków / którzy w polách mieƺkáją / przećiwko Gálgál / przy równinie Morech?
Abowiem wy przejdziećie zá Jordán / ábyśćie wƺedłƺy odziedzicżyli tę ziemię / którą wám dáwá Pán / Bóg wáƺ / á pośiądziećie ją y mieƺkáć w niej będziećie.
Pilnujćież tedy / ábyśćie cżynili wƺyſtkie uſtáwy y ſądy / które já wám dziś przedkłádám.
