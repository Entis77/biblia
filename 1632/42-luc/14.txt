Y ſtáło śię / gdy wƺedł Jezuſ w dom niektórego przedniejƺego Fáryzeuƺá w ſábát / áby jádł chleb / że go oni podſtrzegáli.
A oto cżłowiek niektóry opuchły był przed nim.
A odpowiádájąc Jezuſ / rzekł do zákonników / y do Fáryzeuƺów / mówiąc : Godzili śię w ſábát uzdráwiáć?
A oni milcżeli. Tedy on ująwƺy go / uzdrowił y odpráwił.
A odpowiádájąc rzekł do nich : Któregoż z wáſ ośieł álbo wół wpádnie w ſtudnią / á nie wnet go wyćiągnie w dzień ſábátu?
Y nie mogli mu ná to odpowiedzieć.
Powiedziáł też y wezwánym podobieńſtwo / ( bácżąc / jáko przedniejƺe miejſcá obieráli / ) mówiąc do nich :
Gdybyś był od kogo wezwány ná weſele / nie śiádájże ná przedniejƺem miejſcu / by ſnáć zácniejƺy nád ćię nie był wezwány od niego ;
A przyƺedłƺy ten / który ćiebie y onego wezwáł / rzekłby tobie : Dáj temu miejſce : á tedybyś ze wſtydem pocżął śiedzieć ná pośledniem miejſcu.
Ale gdybyś był wezwány / ƺedłƺy / uśiądź ná pośledniem miejſcu ; á gdyby przyƺedł ten / który ćię wezwáł / rzekłby tobie : Przyjáćielu! pośiądź śię wyżej ; tedy będzieƺ miáł cżeść przed ſpółśiedzącymi z tobą.
Bo wƺelki / kto śię wywyżƺá / poniżony będzie / á kto śię poniżá / wywyżƺony będzie.
Mówił też y onemu / który go był wezwáł : Gdy ſpráwujeƺ obiád álbo wiecżerzę / nie wzywájże przyjáćiół twoich / áni bráći twoich / áni krewnych twoich / áni ſąśiádów bogátych / żeby ćię ſnáć y oni záśię nie wezwáli / á ſtáłáćby śię nágrodá.
Ale gdy ſpráwujeƺ ucżtę / wezwijże ubogich / ułomnych / chromych y ślepych /
A będzieƺ błogoſłáwionym ; boć nie máją cżem nágrodzić / áleć będzie nágrodzono przy zmártwychwſtániu ſpráwiedliwych.
A uſłyƺáwƺy to niektóry z ſpółśiedzących / rzekł mu : Błogoſłáwiony / który je chleb w króleſtwie Bożem.
A on mu rzekł : Cżłowiek niektóry ſpráwił wiecżerzę wielką y záprośił wielu ;
Y poſłáł ſługę ſwego w godzinę wiecżerzy / żeby rzekł záproƺonym : Pójdźćie! bo już wƺyſtko gotowe.
Y pocżęli śię wƺyſcy jednoſtájnie wymáwiáć. Pierwƺy mu rzekł : Kupiłem wieś / y muƺę iść á oglądáć ją / proƺę ćię / miej mię zá wymówionego.
A drugi rzekł : Kupiłem pięć járzm wołów / y idę / ábym ich doświádcżył : proƺę ćię / miej mię zá wymówionego.
A drugi rzekł : Żonęm pojął / á dlátego przyjść nie mogę.
A wróćiwƺy śię on ſługá / oznájmił to pánu ſwemu. Tedy śię goſpodárz rozgniewáwƺy / rzekł ſłudze ſwemu : Wynijdź prędko ná ulice y ná drogi miejſkie / á ubogie y ułomne y chrome y ślepe wprowádź tu.
Y rzekł ſługá : Pánie! ſtáło śię / jákoś rozkázáł / á jeƺcże miejſce jeſt.
Y rzekł Pán do ſługi : Wynijdź ná drogi y między opłotki / á przymuś wnijść / áby był nápełniony dom mój.
Abowiem powiádám wám / że żáden z onych mężów / którzy byli záproƺeni / nie ukuśi wiecżerzy mojej.
Y ƺedł z nim wielki lud ; á obróćiwƺy śię / rzekł do nich :
Jeźli kto idzie do mnie á nie má w nienáwiśći ojcá ſwego / y mátki / y żony / y dzieći / y bráći / y śióſtr / náwet y duƺy ſwojej / nie może być ucżniem moim.
A ktokolwiek nie nieśie krzyżá ſwego / á idzie zá mną / nie może być ucżniem moim.
Bo któż z wáſ jeſt / chcąc zbudowáć wieżę / áby pierwej uśiádłƺy / nie obráchowáł nákłádu / máli to / cżemby jej dokońcżył?
Aby ſnáć / gdyby záłożył fundáment / á dokońcżyć nie mógł / wƺyſcy / którzyby to widzieli / nie pocżęli śię náśmiewáć z niego /
Mówiąc : Ten cżłowiek pocżął budowáć / á nie mógł dokońcżyć.
Albo który król jádąc ná wojnę / potykáć śię z drugim królem / pierwej uśiádłƺy / nie rádzi śię / mógłliby śię w dzieśięć tyśięcy ſpotkáć z onym / który we dwádzieśćiá tyśięcy jedzie przećiwko niemu?
A jeźli nie / gdy on jeƺcże jeſt dáleko od niego / poſły wypráwiwƺy do niego / prośi o to / co náleży do pokoju.
Tákżeć y káżdy z wáſ / ktoby śię nie wyrzekł wƺyſtkich májętnośći ſwoich / nie może być ucżniem moim.
Dobráć jeſt ſól ; lecż jeźli ſól zwietrzeje / cżemże ją nápráwią?
Nie przygodzi śię áni do ziemi áni do gnoju / ále ją precż wyrzucáją. Kto má uƺy ku ſłuchániu / niecháj ſłuchá.
