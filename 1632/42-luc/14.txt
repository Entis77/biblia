Y ſtáło śię / gdy wƺedł JEzuſ w dom niektórego przedniejƺego Fáryzeuƺá w ſábát / áby jádł chleb / że go oni podſtrzegáli.
A oto cżłowiek niektóry opuchły był przed nim.
A odpowiádájąc JEzuſ / rzekł do zakonników / y do Fáryzeuƺów / mówiąc : Godźili śię w ſábát uzdráwiáć?
A oni milcżeli. Tedy on ująwƺy go / uzdrowił y odpráwił.
A odpowiádájąc rzekł do nich : Któregoż z was ośieł álbo wół wpádnie w ſtudnią / á nie wnet go wyćiągnie w dźień ſábátu?
Y nie mogli mu ná to odpowiedźieć.
Powiedźiáł też y wezwánym podobieńſtwo / ( bácżąc / jáko przedniejƺe miejſcá obieráli / ) mówiąc do nich :
Gdybyś był od kogo wezwány ná weſele / nie śiádájże ná przedniejƺem miejſcu / by ſnáć zácniejƺy nád ćię nie był wezwány od niego ;
A przyƺedłƺy ten / który ćiebie y onego wezwáł / rzekłby tobie : Dáj temu miejſce : á tedybyś ze wſtydem pocżął śiedźieć ná pośledniem miejſcu.
Ale gdybyś był wezwány / ƺedłƺy / uśiądź ná pośledniem miejſcu ; á gdyby przyƺedł ten / który ćię wezwáł / rzekłby tobie : Przyjáćielu! pośiądź śię wyżej ; tedy będźieƺ miał cżeść przed ſpółśiedzącymi z tobą.
Bo wƺelki / kto śię wywyżƺá / poniżony będźie / á kto śię poniżá / wywyżƺony będźie.
Mówił też y onemu / który go był wezwáł : Gdy ſpráwujeƺ obiád álbo wiecżerzę / nie wzywájże przyjáćiół twoich / áni bráći twoich / áni krewnych twoich / áni ſąśiádów bogátych / żeby ćię ſnáć y oni záśię nie wezwáli / á ſtáłáćby śię nágrodá.
Ale gdy ſpráwujeƺ ucżtę / wezwijże ubogich / ułomnych / chromych y ślepych /
A będźieƺ błogoſłáwionym ; boć nie máją cżem nágrodźić / áleć będźie nágrodzono przy zmartwychwſtániu ſpráwiedliwych.
A uſłyƺáwƺy to niektóry z ſpółśiedzących / rzekł mu : Błogoſłáwiony / który je chleb w króleſtwie Bożem.
A on mu rzekł : Cżłowiek niektóry ſpráwił wiecżerzę wielką y záprośił wielu ;
Y poſłáł ſługę ſwego w godźinę wiecżerzy / żeby rzekł záproƺonym : Pójdźćie! bo już wƺyſtko gotowe.
Y pocżęli śię wƺyſcy jednoſtájnie wymáwiáć. Pierwƺy mu rzekł : Kupiłem wieś / y muƺę iść á oglądáć ją / proƺę ćię / miej mię zá wymówionego.
A drugi rzekł : Kupiłem pięć járzm wołów / y idę / ábym ich doświádcżył : proƺę ćię / miej mię zá wymówionego.
A drugi rzekł : Żonęm pojął / á dlátego przyjść nie mogę.
A wróćiwƺy śię on ſługá / oznájmił to pánu ſwemu. Tedy śię goſpodárz rozgniewáwƺy / rzekł ſłudze ſwemu : Wynijdź prędko ná ulice y ná drogi miejſkie / á ubogie y ułomne y chrome y ślepe wprowádź tu.
Y rzekł ſługá : Pánie! ſtáło śię / jákoś rozkázáł / á jeƺcże miejſce jeſt.
Y rzekł Pán do ſługi : Wynijdź ná drogi y między opłotki / á przymuś wnijść / áby był nápełniony dom mój.
Abowiem powiádám wám / że żáden z onych mężów / którzy byli záproƺeni / nie ukuśi wiecżerzy mojey.
Y ƺedł z nim wielki lud ; á obróćiwƺy śię / rzekł do nich :
Jeſli kto idźie do mnie á nie má w nienáwiśći ojcá ſwego / y mátki / y żony / y dźieći / y bráći / y śióſtr / náwet y duƺy ſwojey / nie może być ucżniem moim.
A ktokolwiek nie nieśie krzyżá ſwego / á idźie zá mną / nie może być ucżniem moim.
Bo któż z was jeſt / chcąc zbudowáć wieżę / áby pierwej uśiádłƺy / nie obráchowáł nákłádu / máli to / cżemby jey dokońcżył?
Aby ſnáć / gdyby záłożył fundáment / á dokońcżyć nie mógł / wƺyſcy / którzyby to widźieli / nie pocżęli śię náśmiewáć z niego /
Mówiąc : Ten cżłowiek pocżął budowáć / á nie mógł dokońcżyć.
Albo który król jádąc ná wojnę / potykáć śię z drugim królem / pierwej uśiádłƺy / nie rádźi śię / mógłliby śię w dźieśięć tyśięcy ſpotkáć z onym / który we dwádźieśćiá tyśięcy jedźie przećiwko niemu?
A jeſli nie / gdy on jeƺcże jeſt dáleko od niego / poſły wypráwiwƺy do niego / prośi o to / co náleży do pokoju.
Tákżeć y káżdy z was / ktoby śię nie wyrzekł wƺyſtkich májętnośći ſwoich / nie może być ucżniem moim.
Dobráć jeſt ſól ; lecż jeſli ſól zwietrzeje / cżemże ją nápráwią?
Nie przygodźi śię áni do źiemi áni do gnoju / ále ją precż wyrzucáją. Kto má uƺy ku ſłuchániu / niecháj ſłuchá.
