Y ſtáło śię / gdy wƺedł JEzus w dom niektórego Przedniejƺego Fáryzeuƺá w Sábbát áby jadł chleb / że go oni podſtrzegáli.
A oto cżłowiek niektóry opuchły był przed nim.
A odpowiádájąc JEzus rzekł do Zakonników y do Fáryzeuƺów / mówiąc ; Godźili śię w Sábbát uzdrawiáć?
A oni milcżeli. Tedy <i>on</i> ująwƺy go / uzdrowił / y odpráwił.
A odpowiádájąc rzekł do nich : Któregoż z was ośieł álbo wół wpádnie w ſtudnią / á nie wnet go wyćiągnie w dźień Sábbátu?
Y nie mogli mu ná to odpowiedźieć.
Powiedźiał też y wezwánym podobieńſtwo ( bacżąc jáko przedniejƺe miejſcá obieráli ) mówiąc do nich :
Gdybyś był od kogo wezwány ná weſele / nie śiadajże ná przedniejƺym miejſcu ; by ſnadź zacniejƺy nád ćię nie był wezwány od niego.
A przyƺedƺy ten który ćiebie y onego wezwał / rżekłby tobie ; Daj temu miejſce : á tedybyś ze wſtydem pocżął śiedźieć ná poślednim miejſcu.
Ale gdybyś był wezwány / ƺedƺy uśiądź ná poślednim miejſcu : á gdyby przyƺedł ten który ćię wezwał / rzekłby tobie : Przyjaćielu / pośiądź śię wyżey. Tedy będźieƺ miał cżęść przed ſpół-śiedzącymi z tobą.
Bo wƺelki kto śię wywyżƺa / poniżony będźie : á kto śię poniża / wywyżƺony będźie.
Mówił też y onemu który go był wezwał : Gdy ſpráwujeƺ obiad álbo wiecżerzą / nie wzywajże przyjaćiół twojich / áni bráći twojey / áni krewnych twoijch / áni ſąśiádów bogátych / żeby ćię ſnadź y oni záśię nie wezwáli / á ſtáłáćby śię nadgrodá.
Ale gdy ſpráwujeƺ ucżtę / wezwiże ubogich / ułomnych / chromych / y ślepych :
A będźieƺ błogoſłáwionym : boć nie máją cżym nádgrodźić / áleć będźie nádgrodzono przy zmartwychwſtániu ſpráwiedliwych.
A uſłyƺawƺy to niektóry z ſpół-śiedzących / rzekł mu : Błogoſłáwiony który je chleb w króleſtwie Bożym.
A on mu rzekł : Cżłowiek niektóry ſpráwił wiecżerzą wielką / y záprośił wielu.
Y poſłał ſługę ſwego w godźinę wiecżerzy / żeby rzekł záproƺonym : Pódźćie / bo już wƺyſtko gotowo.
Y pocżęli śię wƺyſcy jednoſtájnie wymawiáć. Pierwƺy mu rzekł : Kupiłem wieś / y muƺę iść á oglądáć ją : proƺę ćię miey mię zá wymówionego.
A drugi rzekł : Kupiłem pięć jarzm wołów / y idę ábym ich doświadcżył : proƺę ćię miey mię zá wymówionego.
A drugi rzekł ; Żonęm pojął : á dla tego przyść nie mogę.
A wróćiwƺy śię on ſługá / oznájmił to Pánu ſwemu. Tedy śię goſpodarz rozgniewawƺy / rzekł ſłudze ſwemu : Wynidź prętko ná ulice / y ná drogi miejſkie ; á ubogie / y ułomne / y chrome / y ślepe / wprowadź tu.
Y rzekł ſługá : Pánie ſtáło śię jákoś rozkazał ; á jeƺcże miejſce jeſt.
Y rzekł Pan do ſługi : Wynidź ná drogi y <i>miedzy</i> opłotki / á przymuś wniść ; áby był nápełniony dom mój.
Abowiem powiádam wam / że żaden z onych mężów którzy byli záproƺeni / nie ukuśi wiecżerzy mojey.
Y ƺedł z nim wielki lud : á obróćiwƺy śię rzekł do nich ;
Jeſli kto idźie do mnie / á nie ma w nienawiśći ojcá ſwego / y mátki / y żony / y dźieći / y bráći / y śióſtr / náwet y duƺe ſwojey / nie może być ucżniem mojim.
A ktokolwiek nie nieśie krzyżá ſwego / á idźie zá mną / nie może być ucżniem mojim.
Bo któż z was jeſt chcąc zbudowáć wieżą / áby pierwey uśiadƺy / nie obráchował nákłádu / mali to cżymby jey dokońcżył.
Aby ſnadź gdyby záłożył fundáment / á dokońcżyć nie mógł / wƺyſcy którzyby to widźieli / nie pocżęli śię náśmiewáć z niego /
Mówiąc ; Ten cżłowiek pocżął budowáć / á nie mógł dokońcżyć.
Albo który Król jádąc ná wojnę potykáć śię z drugim Królem / pierwey uśiadƺy nierádźi śię / mógłliby śię w dźieśiąći tyśięcy potkáć z onym który we dwudźieſtu tyśięcy jedźie przećiwko niemu.
A jeſli nie ; Gdy on jeƺcże jeſt dáleko od niego / poſły wypráwiwƺy <i>do niego,</i> prośi o to co należy do pokoju.
Tákżeć y káżdy z was / ktoby śię nie wyrzekł wƺyſtkich májętnośći ſwojich / nie może być ucżniem mojim.
Dobrać jeſt ſól. Lecż jeſli ſól zwietrzeje / cżymże ją nápráwią?
Nie przygodźi śię áni do źiemie áni do gnoju / <i>ále</i> ją precż wyrzucáją. Kto ma uƺy ku ſłuchániu / niechaj ſłucha.
