Y ſtáło śię / gdy náń lud nálegáł / áby ſłucháł ſłowá Bożego / że on ſtáł podle jeźiorá Gienezáretſkiego.
Y ujrzáł dwie łodźi ſtojące przy jeźiorze ; ále rybitwi wyƺedłƺy z nich / płókáli śieći.
A wſtąpiwƺy w jedną z tych łodźi / która byłá Szymonowá / prośił go / áby málucżko odjecháł od brzegu ; á uśiádłƺy / ucżył on lud z oney łodźi.
A gdy przeſtáł mówić / rzekł do Szymoná : Zájedź ná głębią / á zápuśććie śieći wáƺe ku łowieniu.
A odpowiádájąc Szymon / rzekł mu : Miſtrzu! przez cáłą noc robiąc / niceśmy nie pojmáli / wƺákże ná ſłowo twoje zápuƺcżę śieć.
A gdy to ucżynili / zágárnęli ryb mnóſtwo wielkie / ták że śię rwáłá śieć ich.
Y ſkinęli ná towárzyƺe / którzy byli w drugiej łodźi / áby przybywƺy rátowáli ich ; y przybyli y nápełnili obie łodźi / áż śię zánurzáły.
Co widząc Szymon Piotr / przypádł do kolán JEzuſowych / mówiąc : Wynijdź odemnie ; bom jeſt cżłowiek grzeƺny / Pánie!
Abowiem go był ſtrách ogárnął / y wƺyſtkie / co z nim byli / z onego obłowu ryb / które byli zágárnęli.
Tákże y Jákóbá y Jáná / ſyny Zebedeuƺowe / którzy byli towárzyƺe Szymonowi. Y rzekł JEzuſ do Szymoná : Nie bój śię ; od tego cżáſu ludźi łowić będźieƺ.
A oni wyćiągnąwƺy łódź ná brzeg / wƺyſtko opuśćiwƺy / poƺli zá nim.
Y ſtáło śię / gdy był w niektórem mieśćie / że oto był tám mąż pełen trądu / który ujrzáwƺy JEzuſá / pádł ná twárz / y prośił go / mówiąc : Pánie! jeſli chceƺ / możeƺ mię ocżyśćić.
Tedy wyćiągnąwƺy JEzuſ rękę / dotknął śię go / mówiąc : Chcę / bądź ocżyƺcżony ; y záráz odƺedł trąd od niego.
Y przykázáł mu / áby tego nikomu nie powiádáł : ále rzekł : Idź / á ukáż śię kápłánowi / y ofiáruj zá ocżyƺcżenie twoje / ták jáko rozkázáł Mojżeƺ / ná świádectwo przećiwko nim.
Y rozchodźiłá śię tem więcey powieść o nim ; y ſchodźiły śię mnóſtwá wielkie / áby go ſłucháły y uzdrowieni byli od niego od niemocy ſwoich.
Ale on odchádzáł ná puſtynią / y modlił śię.
Y ſtáło śię dniá niektórego / że on náucżáł / á śiedźieli też tám y Fáryzeuƺowie y náucżyćiele Zakonu / którzy śię byli zeƺli ze wƺyſtkich miáſtecżek Gálilejſkich y Judzkich / y z Jeruzálemu ; á moc Páńſká przytomná byłá ku uzdráwiániu ich.
A oto mężowie nieśli ná łożu cżłowieká powietrzem ruƺonego / y ƺukáli / jákoby go wnieść y poſtáwić przed nim.
A gdy nie ználeſli / którędyby go wnieśli / dla ćiżby / wſtąpiwƺy ná dách / przez poſowę ſpuśćili go łożem w pośrodek przed JEzuſá.
Który ujrzáwƺy wiárę ich / rzekł mu : Cżłowiecże! odpuƺcżone ſą tobie grzechy twoje.
Tedy pocżęli myślić náucżeni w Piśmie y Fáryzeuƺowie / mówiąc : Któż to jeſt / co mówi bluźnierſtwá? Któż może odpuƺcżáć grzechy / tylko ſám Bóg.
Ale JEzuſ poznáwƺy myśli ich / odpowiádájąc rzekł do nich : Cóż myślićie w ſercách wáƺych?
Cóż jeſt łátwiejƺego / rzec : Odpuƺcżone ſą tobie grzechy twoje / cżyli rzec : Wſtáń á chodź?
Ale iżbyśćie wiedźieli / że Syn cżłowiecży má moc ná źiemi odpuƺcżáć grzechy / ( rzekł powietrzem ruƺonemu : ) Tobie mówię : Wſtáń / á wźiąwƺy ná śię łoże ſwoje / idź do domu twego.
A on zárázem wſtáwƺy przed nimi / wźiąwƺy ná śię to / ná cżem leżáł / ƺedł do domu ſwego / wielbiąc Bogá.
Y zdumieli śię wƺyſcy / y chwálili Bogá / y nápełnieni byli bojáźnią / mówiąc : Widźieliśmy dźiś dźiwne rzecży.
A potem wyƺedł y ujrzáł celniká / imieniem Lewiego / śiedzącego ná cle / y rzekł mu : Pójdź zá mną.
Y opuśćił wƺyſtko / á wſtáwƺy / ƺedł zá nim.
Y ſpráwił mu Lewi ucżtę wielką w domu ſwoim ; á było wielkie zgromádzenie celników y innych / którzy z nim zá ſtołem śiedźieli.
Tedy ƺemráli náucżeni w Piśmie y Fáryzeuƺowie / mówiąc do ucżniów jego : Przecż z celnikámi y z grzeƺnikámi jećie y pijećie?
A JEzuſ odpowiádájąc / rzekł do nich : Nie potrzebująć zdrowi lekárzá / ále ći / co śię źle máją ;
Nie przyƺedłem / wzywáć ſpráwiedliwych / ále grzeƺnych do pokuty.
A oni mu rzekli : Przecż ucżniowie Jánowi cżęſto poƺcżą y modlą śię / tákże y Fáryzejſcy / á twoi jedzą y piją?
A on im rzekł : Izáli możećie ucżynić / żeby ſynowie łożnicy máłżeńſkiej pośćili / póki z nimi jeſt oblubieniec?
Lecż przyjdą dni / gdy oblubieniec odjęty będźie od nich ; tedy w one dni pośćić będą.
Powiedźiáł im też podobieńſtwo : Iż żáden łáty z ƺáty nowej nie przypráwiá do ƺáty wiotchej ; bo inácżej to / co jeſt nowego / drze wiotche / á do wiotchego nie zgádzá śię łátá z nowego.
Y nikt nie leje winá nowego w ſtáre ſtátki ; bo inácżej wino młode rozſádźi ſtátki / y ſámo wyćiecże / y ſtátki śię popſują.
Ale młode wino má być wlewáne w ſtátki nowe : á ták oboje bywáją záchowáne.
A nikt / kto śię nápił ſtárego / nie záráz chce młodego ; ále mówi : Lepƺe jeſt ſtáre.
