Y ſtáło śię gdy nań lud nálegał / áby ſłuchał ſłowá Bożego / <i>że</i> on ſtał podle jeźiorá Genezáretſkiego.
Y ujrzał dwie łodźi ſtojące przy jeźierze : ále rybitwi wyƺedƺy z nich / płókáli śieći.
A wſtąpiwƺy w jednę z tych łodźi / która byłá Symonowá / prośił go áby málucżko odjáchał od brzegu : á uśiadƺy / ucżył on lud z oney łodźi.
A gdy przeſtał mówić / rzekł do Symoná ; Zájedź ná głębią / á zápuśććie śieći wáƺe ku łowieniu.
A odpowiádájąc Symon / rzekł mu ; Miſtrzu / przez cáłą noc robiąc / niceſmy nie pojimáli : wƺákże ná ſłowo twoje zápuƺcżę śieć.
A gdy to ucżynili / zágárnęli ryb mnóſtwo wielkie ; ták / że śię rwáłá śieć ich.
Y ſkinęli ná towárzyƺe / którzy byli w drugiey łodźi / áby przybywƺy rátowáli ich : y przybyli / y nápełnili obie łodźi áż śię zánurzáły.
Co widząc Symon Piotr / przypadł do kolan JEzuſowych / mówiąc ; Wynidź ode mnie / bom jeſt cżłowiek grzeƺny / PAnie.
Abowiem go był ſtrách ogárnął / y wƺyſtkie co z nim byli / z onego obłowu ryb które byli zágárnęli.
Tákże y Jákubá y Janá ſyny Zebedeuƺowe / którzy byli towárzyƺe Symonowi. Y rzekł JEzus do Symoná : Nie bój śię ; Od tego cżáſu ludźie łowić będźieƺ.
A oni wyćiągnąwƺy łódź ná brzeg / wƺyſtko opuśćiwƺy / poƺli zá nim.
Y ſtáło śię / gdy był w niektórym mieśćie / że oto <i>był tám</i> mąż pełen trędu : który ujrzawƺy JEzuſá / padł ná twarz / y prośił go mówiąc : PAnie / jeſli chceƺ możeƺ mię ocżyśćić.
Tedy wyćiągnąwƺy JEzus rękę / dotknął śię go / mówiąc ; Chcę / bądź ocżyśćiony. Y záraz odƺedł trąd od niego.
Y przykazał mu áby tego nikomu nie powiádał : ále <i>rzekł,</i> Idź / á ukaż śię Kápłanowi ; y ofiáruj zá ocżyśćienie twoje / ták jáko rozkazał Mojzeƺ ná świádectwo przećiwko nim.
Y rozchodźiłá śię tym więcey powieść o nim : y zchodźiły śię mnóſtwá wielkie áby go ſłucháły / y uzdrowieni byli od niego / od niemocy ſwojich.
Ale on odchadzał ná puſtynią / y modlił śię.
Y ſtáło śię dniá niektórego / że on náucżał : á śiedźieli też tám y Fáryzeuƺowie y Náucżyćiele Zakonu / którzy śię byli zeƺli ze wƺyſtkich miáſtecżek Gálilejſkich / y Judſkich / y z Jeruzalem : á moc PAńſká <i>przytomna</i> byłá ku uzdrawiániu ich.
A oto mężowie nieśli ná łożu cżłowieká powietrzem ruƺonego : y ƺukáli <i>jákoby</i> go wnieść y poſtáwić przed nim.
A gdy nie ználeźli którędyby go wnieśli dla ćiżby / wſtąpiwƺy ná dách / przez poſowę zpuśćili go z łożem w pośrzodek przed JEzuſá.
Który ujrzawƺy wiárę ich / rzekł mu : Cżłowiecże / odpuƺcżone ſą tobie grzechy twoje.
Tedy pocżęli myślić Náucżeni w piśmie y Fáryzeuƺowie / mówiąc : Któż to jeſt co mówi bluźnierſtwá? Któż może odpuƺcżáć grzechy / tylko ſam Bóg?
Ale JEzus poznawƺy myśli ich / odpowiádájąc rzekł do nich : Cóż myślićie w ſercách wáƺych.
Cóż jeſt łátwiejƺego rzec? Odpuƺcżone ſą tobie grzechy twoje? Cżyli rzec? Wſtań / á chodź.
Ale iżbyśćie wiedźieli / że Syn cżłowiecży ma moc ná źiemi odpuƺcżáć grzechy / ( rzekł powietrzem ruƺonemu ) Tobie mówię wſtań / á wźiąwƺy ná śię łoże ſwoje / idź do domu twego.
A on zárázem wſtawƺy przed nimi / wźiąwƺy ná śię to ná cżym leżał / ƺedł do domu ſwego wielbiąc Bogá.
Y zdumieli śię wƺyſcy / y chwalili Bogá : y nápełnieni byli bojáźnią / mówiąc : Widźieliſmy dźiś dźiwne rzecży.
A potym wyƺedł / y ujrzał Celniká imieniem Lewiego / śiedzącego ná cle / y rzekł mu : Pódź zá mną.
Y opuśćił wƺyſtko : á wſtawƺy / ƺedł zá nim.
Y ſpráwił mu Lewi ucżtę wielką w domu ſwojim : á było wielkie zgromádzenie Celników y innych którzy z nim zá ſtołem śiedźieli.
Tedy ƺemráli Náucżeni w piśmie y Fáryzeuƺowie / mówiąc do ucżniów jego : Przecż z Celnikámi y z grzeƺnikámi jećie y pijećie?
A JEzus odpowiádájąc / rzekł do nich : Nie potrzebująć zdrowi lekárzá / ále ći co śię źle máją.
Nie przyƺedłem wzywáć ſpráwiedliwych / ále grzeƺnych do pokuty.
A oni mu rzekli : Przecż Ucżniowie Janowi cżęſto poƺcżą y modlą śię : tákże y Fáryzejſcy : á twoji jedzą y piją?
A on im rzekł : Izali możećie ucżynić / żeby ſynowie łożnice máłżeńſkiey pośćili / póki z nimi jeſt oblubieniec?
Lecż przydą dni / gdy oblubieniec odjęty będźie od nich : tedy w one dni pośćić będą.
Powiedźiał im też podobieńſtwo : Iż żaden łáty z ƺáty nowey nie przyprawia do ƺáty wiotchey / bo inácżey / to co jeſt nowego / drże <i>wiotche</i> : á do wiotchego / nie zgadza śię łátá z nowego.
Y nikt nie leje winá nowego w ſtáre ſtátki : bo inácżey wino młode rozſádźi ſtátki / y ſámo wyćiecże / y ſtátki śię popſują.
Ale młode wino ma być wlewáne w ſtátki nowe : á ták oboje bywáją záchowáne.
A nikt / kto śię nápił ſtárego / nie zaraz chce młodego : ále mówi ; Lepƺe jeſt ſtáre.
