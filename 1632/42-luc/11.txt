Y ſtáło śię / gdy on był ná niektórym miejſcu modląc śię / że gdy przeſtał / rzekł do niego jeden z ucżniów jego ; PAnie / náucż nas modlić śię / ták jáko y Jan náucżył ucżnie ſwoje.
Y rzekł im ; Gdy śię modlićie / mówćie : Ojcże náƺ / któryś jeſt w niebieśiech. Święć śię Imię twoje. Przydź króleſtwo twoje.;Bądź wola twojá / jáko w niebie ták y ná źiemi.
Chlebá náƺego powƺedniego daj nam ná káżdy dźień.
Y odpuść nam grzechy náƺe : bo też y my odpuƺcżamy káżdemu winowájcy náƺemu. A nie w wódź nas ná pokuƺenie : Ale nas zbaw od złego.
Zátym rzekł do nich : Któż z was mieć będźie przyjaćielá / y pójdźie do niego o północy / y rzecże mu : Przyjaćielu / pożycż mi trzech chlebów.
Abowiem przyjaćiel mój przyƺedł z drogi do mnie / á nie mam co przedeń położyć.
A on będąc w domu / odpowiedźiałby mówiąc : Nie uprzykrzaj mi śię : już ſą drzwi zámknione / á dźiatki moje ſą ze mną w pokoju : nie mogę wſtáć ábymći dał.
Powiádam wam / Choćiażby mu nie dał wſtawƺy / przeto że jeſt przyjaćiel jego : wƺákże dla niewſtydliwego <i>nálegánia</i> jego / wſtawƺy da mu ile potrzebuje.
Y Jać wam powiádam : Prośćie / á będźie wam dano : ƺukajćie / á znajdźiećie : kołácćie / á będźie wam otworzono.
Káżdy bowiem kto prośi / bierze : á kto ƺuka / znájduje : á temu co kołáce / będźie otworzono.
A któryż jeſt zwas Oćiec / którego gdyby prośił ſyn o chleb / izali mu da kámień? Albo <i>prośiłliby</i> o rybę : izali mu miáſto ryby da wężá?
Albo prośiłliby o jáje : izali mu da niedźwiadká?
Ponieważ tedy wy będąc złymi / umiećie dáry dobre dawáć dźiećiom wáƺym : jákoż dáleko więcey Oćiec wáƺ niebieſki da Duchá Świętego tym / którzy go oń proƺą?
Tedy wyganiał dyabłá który był niemy. Y ſtáło śię gdy wyƺedł on dyabeł / przemówił niemy : y dźiwował śię lud.
Ale niektórzy z nich mówili : Przez Beelzebubá kśiążę dyabelſkie wygania dyabły.
Drudzy záśię kuƺąc <i>go</i> / żądáli známienia od niego z niebá.
Ale on widząc myśli ich / rzekł im : Káżde króleſtwo rozdźielone ſámo przećiwko ſobie / puſtoƺeje / á dom ná dom upada.
A jeſliżeć y ƺátan rozdźielony jeſt przećiwko ſobie / jákoż śię oſtoji króleſtwo jego? Abowiem powiádaćie / iż Ja przez Beelzebubá wyganiam dyabły.
A jeſliż Ja przez Beelzebubá wyganiam dyabły / ſynowie wáƺy przez kogoż wyganiáją? Przetoż oni będą ſędźiámi wáƺymi.
Ale jeſliż Ja pálcem Bożym wyganiam dyabły / zájiſteć przyƺło do was króleſtwo Boże.
Gdy mocarz uzbrojony ſtrzeże páłacu ſwego / w pokoju ſą májętnośći jego.
Ale gdy mocniejƺy nádeń nádƺedƺy zwyćięży go / odejmuje wƺyſtko oręże jego w którym ufał / á łupy jego rozdawa.
Kto nie jeſt ze mną / przećiwko mnie jeſt : á kto nie zbiera ze mną / roſpraƺa.
Gdy duch niecżyſty wychodźi od cżłowieká / przechadza śię po miejſcách ſuchych ƺukájąc odpocżynienia : á nie ználazƺy / mówi : Wrócę śię do domu mego z kądem wyƺedł.
A przyƺedƺy / znájduje umiećiony y ochędożony.
Tedy idźie y bierze z ſobą śiedm innych duchów gorƺych niżeli ſam / á wƺedƺy mieƺkáją tám : y bywáją <i>rzecży</i> oſtátnie cżłowieká onego gorƺe niżeli pierwƺe.
Y ſtáło śię gdy on to mówił / <i>że</i> wyniózƺy głos niektóra niewiáſtá z ludu / rzekłá mu : Błogoſłáwiony żywot który ćię nośił / y pierśi któreś sſał!
Ale on rzekł : Owƺem błogoſłáwieni <i>ſą,</i> którzy ſłucháją ſłowá Bożego / y ſtrzegą go.
A gdy śię lud gromádźił / pocżął mówić ; Rodzaj ten zły jeſt : známienia ƺuka / ále mu známię nie będźie dáne / tylko ono známię Jonaƺá Proroká.
Abowiem jáko Jonaƺ <i>był</i> zá známię Niniwcżykom : ták będźie y Syn cżłowiecży temu rodzájowi.
Królowa z Południá ſtánie ná ſądźie z mężámi rodzáju tego / y potępi je : bo przyjácháłá od końcżyn źiemie / áby ſłucháłá mądrośći Sálomonowey : á oto tu więcey niżeli Sálomon.
Mężowie Niniwitſcy ſtáną ná ſądźie z tym rodzájem / y potępią go : przeto / że pokutowáli ná kazánie Jonaƺowe : á oto tu więcey niżeli Jonaƺ.
A nikt świecę zápaliwƺy / nie ſtawia jey w ſkrytośći / áni pod korzec : ále ná świecżnik : áby ći którzy wchodzą / świátło widźieli.
Świecą ćiáłá jeſt oko. Jeſliby tedy oko twoje było ƺcżere / y ćiáło twoje wƺyſtko będźie jáſne : á jeſliby złe było / y ćiáło twoje ćiemne będźie.
Pátrzajże tedy / áby świátło które jeſt w tobie / nie było ćiemnośćią.
Jeſli tedy wƺyſtko ćiáło twoje jáſne będźie / nie májąc jákiey cżąſtki záćmioney : będźieć wƺyſtko <i>ták</i> jáſne / że ćię jáko świecá bláſkiem oświeći.
A gdy to mówił / prośił go niektóry Fáryzeuƺ áby jadł obiad u niego : Wƺedƺy tedy / uśiadł <i>zá ſtołem</i>.
A widząc to Fáryzeuƺ / dźiwował śię / że śię nie umył przed obiádem.
Y rzekł PAn do niego : Teraz wy Fáryzeuƺowie / to co jeſt zewnątrz kubká y miſy / ochędażaćie : ále to co jeſt wewnątrz w was / pełne jeſt drapieſtwá y złośći.
Szaleni / izaż ten który ucżynił to co jeſt zewnątrz / nie ucżynił też te° co jeſt wewnątrz?
Wƺákże y z tego co jeſt wewnątrz dawajćie jáłmużnę : á oto wƺyſtkie <i>rzecży</i> będą wam cżyſte.
Ale biádá wam Fáryzeuƺowie / że dawaćie dźieśięćinę z miętki y z ruty y z káżdego źiela : lecż opuƺcżaćie ſąd y miłość Bożą : teć rzecży trzebá było cżynić / á onych nie opuƺcżáć.
Biádá wam Fáryzeuƺowie / że miłujećie pierwƺe miejſcá w bóżnicách / y pozdrawiánia ná rynkách.
Biádá wám Náucżeni w piśmie y Fáryzeuƺowie obłudni / bośćie jáko groby których nie widáć / á ludźie którzy chodząc po nich / nie wiedzą <i>o nich</i>.
A odpowiádájąc niektóry z Zakonników / rzekł mu ; Náucżyćielu / to mówiąc y nas háńbiƺ.
A on rzekł : Y wam Zakonnikom / biádá : ábowiem obćiążaćie ludźie brzemiony nieznośnymi / á ſámi śię y jednym pálcem ſwojim tych brzemion nie dotykaćie.
Biádá wam że budujećie groby Proroków / á ojcowie wáƺy pozábijáli je.
Zájiſte / świadcżyćie iż śię kochaćie w ucżynkách ojców wáƺych : ábowiem oni je pozábijáli / á wy budujećie groby ich.
Dla tegoż też Mądrość Boża rzekłá ; Poślę do nich Proroki y Apoſtoły / á z nich <i>niektóre</i> zábijáć y prześládowáć będą :
Aby ƺukano od tego rodzáju krwie wƺyſtkich Proroków / która wylána jeſt od záłożenia świátá :
Od krwie Ablá / áż do krwie Zácháryaƺá / który zginął miedzy ołtarzem y kośćiołem. Zájiſte powiádam wam / będą jey ƺukáć od rodzáju tego.
Biádá wam Zakonnikom / bośćie wźięli klucż umiejętnośći : ſámiśćie nie weƺli / á tym którzy wniść chćieli / zábraniáliśćie.
A gdy im to mówił / pocżęli nań Náucżeni w piśmie y Fáryzeuƺowie bárzo náćieráć / y przycżynę mu dawáć do mówienia o wielu <i>rzecżách</i> :
Cżyhájąc nań / y ƺukájąc áby co uchwyćili z uſt jego / żeby go oſkárżyli.
