Y ſtáło śię / gdy on był ná niektórem miejſcu / modląc śię / że gdy przeſtáł / rzekł do niego jeden z ucżniów jego : Pánie! náucż náſ modlić śię / ták jáko y Ján náucżył ucżniów ſwoich.
Y rzekł im : Gdy śię modlićie / mówćie : Ojcże náƺ / któryś jeſt w niebieśiech! Święć śię imię twoje ; przyjdź króleſtwo twoje ; bądź wolá twojá / jáko w niebie ták y ná ziemi.
Chlebá náƺego powƺedniego dáj nám ná káżdy dzień.
Y odpuść nám grzechy náƺe ; bo też y my odpuƺcżámy káżdemu winowájcy náƺemu. A nie wwódź náſ ná pokuƺenie / ále náſ zbáw od złego.
Zátem rzekł do nich : Któż z wáſ mieć będzie przyjáćielá / y pójdzie do niego o północy y rzecże mu : Przyjáćielu! pożycż mi trzech chlebów ;
Abowiem przyjáćiel mój przyƺedł z drogi do mnie / á nie mám / co przedeń położyć.
A on będąc w domu / odpowiedziáłby mówiąc : Nie uprzykrzáj mi śię ; już ſą drzwi zámknięte / á dziátki moje ſą ze mną w pokoju ; nie mogę wſtáć / ábym ći dáł.
Powiádám wám : Choćiáżby mu nie dáł wſtáwƺy / przeto że jeſt przyjáćiel jego / wƺákże dlá niewſtydliwego nálegániá jego wſtáwƺy / dá mu / ile potrzebuje.
Y jáć wám powiádám : Prośćie / á będzie wám dáno ; ƺukájćie / á znájdziećie ; kołácżćie / á będzie wám otworzono.
Káżdy bowiem / kto prośi / bierze / á kto ƺuká / znájduje / á temu / co kołácże / będzie otworzono.
A któryż jeſt z wáſ ojćiec / którego gdyby prośił ſyn o chleb / izáli mu dá kámień? Albo prośiłby o rybę / izáli mu zámiáſt ryby dá wężá?
Albo prośiłliby o jáje / izáli mu dá niedźwiádká?
Poniewáż tedy wy / będąc złymi / umiećie dáry dobre dáwáć dziećiom wáƺym : jákoż dáleko więcej Ojćiec wáƺ niebieſki dá Duchá Świętego tym / którzy go oń proƺą?
Tedy wygániáł dyjábłá / który był niemy. Y ſtáło śię / gdy wyƺedł on dyjábeł / przemówił niemy ; y dziwowáł śię lud.
Ale niektórzy z nich mówili : Przez Beelzebubá / kśiążęćiá dyjábelſkiego / wygániá dyjábły.
Drudzy záśię kuƺąc go / żądáli známieniá od niego z niebá.
Ale on widząc myśli ich / rzekł im : Káżde króleſtwo rozdzielone ſámo przećiwko ſobie puſtoƺeje / á dom ná dom upádá.
A jeźliżeć y ƺátán rozdzielony jeſt przećiwko ſobie / jákoż śię oſtoi króleſtwo jego? ábowiem powiádáćie / iż já przez Beelzebubá wygániám dyjábły.
A jeźliż já przez Beelzebubá wygániám dyjábły / ſynowie wáśi przez kogoż wygániáją? Przetoż oni będą ſędziámi wáƺymi.
Ale jeźliż já pálcem Bożym wygániám dyjábły / záiſteć przyƺło do wáſ króleſtwo Boże.
Gdy mocárz uzbrojony ſtrzeże páłácu ſwego / w pokoju ſą májętnośći jego ;
Ale gdy mocniejƺy nádeń nádƺedłƺy / zwyćięży go / odejmuje wƺyſtko oręże jego / w którym ufáł / á łupy jego rozdáje.
Kto nie jeſt ze mną / przećiwko mnie jeſt ; á kto nie zbierá ze mną / rozpráƺá.
Gdy duch niecżyſty wychodzi od cżłowieká / przechádzá śię po miejſcách ſuchych / ƺukájąc odpocżnieniá / á nie ználázłƺy / mówi : Wrócę śię do domu mego / ſkądem wyƺedł.
A przyƺedłƺy / znájduje umiećiony y ochędożony.
Tedy idzie y bierze z ſobą śiedm innych duchów gorƺych niżeli ſám / á wƺedłƺy mieƺkáją tám / y bywáją rzecży oſtátnie cżłowieká onego gorƺe / niżeli pierwƺe.
Y ſtáło śię / gdy on to mówił / że wynióſłƺy głoſ niektórá niewiáſtá z ludu / rzekłá mu : Błogoſłáwiony żywot / który ćię nośił / y pierśi / któreś ſſáł!
Ale on rzekł : Owƺem błogoſłáwieni ſą / którzy ſłucháją ſłowá Bożego y ſtrzegą go.
A gdy śię lud gromádził / pocżął mówić : Rodzáj ten rodzáj zły jeſt ; známieniá ƺuká / ále mu známię nie będzie dáne / tylko ono známię Jonáƺá proroká.
Abowiem jáko Jonáƺ był zá známię Niniwcżykom / ták będzie y Syn cżłowiecży temu rodzájowi.
Królowá z południá ſtánie ná ſądzie z mężámi rodzáju tego / y potępi je ; bo przyjecháłá od końcżyn ziemi / áby ſłucháłá mądrośći Sálomonowej ; á oto tu więcej / niżeli Sálomon.
Mężowie Niniwiccy ſtáną ná ſądzie z tym rodzájem / y potępią go / przeto / że pokutowáli ná kázánie Jonáƺowe ; á oto tu więcej / niżeli Jonáƺ.
A nikt świecę zápáliwƺy / nie ſtáwiá jej w ſkrytośći / áni pod korzec / ále ná świecżnik / áby ći / którzy wchodzą / świátło widzieli.
Świecą ćiáłá jeſt oko ; jeźliby tedy oko twoje było ƺcżere / y ćiáło twoje wƺyſtko będzie jáſne : á jeźliby złe było / y ćiáło twoje ćiemne będzie.
Pátrzájże tedy / áby świátło / które jeſt w tobie / nie było ćiemnośćią.
Jeźli tedy wƺyſtko ćiáło twoje jáſne będzie / niemájąc jákiej cżąſtki záćmionej / będzieć wƺyſtko ták jáſne / że ćię jáko świecá bláſkiem oświeći.
A gdy to mówił / prośił go niektóry Fáryzeuƺ / áby jádł obiád u niego ; wƺedłƺy tedy / uśiádł zá ſtołem.
A widząc to Fáryzeuƺ / dziwowáł śię / że śię nie umył przed obiádem.
Y rzekł Pán do niego : Teráz wy / Fáryzeuƺowie! to / co jeſt zewnątrz kubká y miſy / ochędáżáćie / ále to / co jeſt wewnątrz w wáſ / pełne jeſt drápieſtwá y złośći.
Száleni! izáż ten / który ucżynił to / co jeſt zewnątrz / nie ucżynił też tego / co jeſt wewnątrz?
Wƺákże y z tego / co jeſt wewnątrz / dáwájćie jáłmużnę / á oto wƺyſtkie rzecży będą wám cżyſte.
Ale biádá wám / Fáryzeuƺowie! że dájećie dzieśięćinę z mięty / y z ruty / y z káżdego zielá / lecż opuƺcżáćie ſąd y miłość Bożą : teć rzecży trzebá cżynić / á onych nie opuƺcżáć.
Biádá wám / Fáryzeuƺowie! że miłujećie pierwƺe miejſcá w bóżnicách y pozdráwiániá ná rynkách.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! bośćie jáko groby / których nie widáć / á ludzie / którzy chodząc po nich / nie wiedzą o nich.
A odpowiádájąc niektóry z zákonników / rzekł mu : Náucżyćielu! to mówiąc y náſ háńbiƺ.
A on rzekł : Y wám zákonnikom biádá! ábowiem obćiążáćie ludzi brzemiony nieznośnemi / á ſámi śię y jednym pálcem ſwoim tych brzemion nie dotykáćie.
Biádá wám! że budujećie groby proroków / á ojcowie wáśi pozábijáli je.
Záiſte świádcżyćie / iż śię kocháćie w ucżynkách ojców wáƺych ; ábowiem oni je pozábijáli / á wy budujećie groby ich.
Dlátegoż też mądrość Bożá rzekłá : Poślę do nich proroki y Apoſtoły / á z nich niektóre zábijáć y prześládowáć będą ;
Aby ƺukáno od tego rodzáju krwi wƺyſtkich proroków / którá wyláná jeſt od záłożeniá świátá /
Od krwi Ablá áż do krwi Zácháryjáƺá / który zginął między ołtárzem / y kośćiołem ; záiſte powiádám wám / będą jej ƺukáć od rodzáju tego.
Biádá wám zákonnikom! bośćie wzięli klucż umiejętnośći ; ſámiśćie nie weƺli / á tym / którzy wnijść chćieli / zábrániáliśćie.
A gdy im to mówił / pocżęli náń náucżeni w Piśmie y Fáryzeuƺowie bárdzo náćieráć / y przycżynę mu dáwáć do mówieniá o wielu rzecżách :
Cżyhájąc náń y ƺukájąc / áby co uchwyćili z uſt jego / żeby go oſkárżyli.
