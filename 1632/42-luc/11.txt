Y ſtáło śię / gdy on był ná niektórem miejſcu / modląc śię / że gdy przeſtáł / rzekł do niego jeden z ucżniów jego : Pánie! náucż nas modlić śię / ták jáko y Ján náucżył ucżniów ſwoich.
Y rzekł im : Gdy śię modlićie / mówćie : Ojcże náƺ / któryś jeſt w niebieśiech! Święć śię imię twoje ; przyjdź króleſtwo twoje ; bądź wolá twojá / jáko w niebie ták y ná źiemi.
Chlebá náƺego powƺedniego dáj nám ná káżdy dźień.
Y odpuść nám grzechy náƺe ; bo też y my odpuƺcżámy káżdemu winowájcy náƺemu. A nie wwódź nas ná pokuƺenie / ále nas zbáw od złego.
Zátem rzekł do nich : Któż z was mieć będźie przyjáćielá / y pójdźie do niego o północy y rzecże mu : Przyjáćielu! pożycż mi trzech chlebów ;
Abowiem przyjáćiel mój przyƺedł z drogi do mnie / á nie mam / co przedeń położyć.
A on będąc w domu / odpowiedźiáłby mówiąc : Nie uprzykrzáj mi śię ; już ſą drzwi zámknięte / á dźiátki moje ſą ze mną w pokoju ; nie mogę wſtáć / ábym ći dáł.
Powiádám wám : Choćiáżby mu nie dáł wſtáwƺy / przeto że jeſt przyjáćiel jego / wƺákże dla niewſtydliwego nálegániá jego wſtáwƺy / dá mu / ile potrzebuje.
Y jáć wám powiádám : Prośćie / á będźie wám dáno ; ƺukájćie / á znájdźiećie ; kołácżćie / á będźie wám otworzono.
Káżdy bowiem / kto prośi / bierze / á kto ƺuká / znájduje / á temu / co kołácże / będźie otworzono.
A któryż jeſt z was ojćiec / którego gdyby prośił ſyn o chleb / izáli mu dá kámień? Albo prośiłby o rybę / izáli mu zámiáſt ryby dá wężá?
Albo prośiłliby o jáje / izáli mu dá niedźwiádká?
Poniewáż tedy wy / będąc złymi / umiećie dáry dobre dáwáć dźiećiom wáƺym : jákoż dáleko więcey Ojćiec wáƺ niebieſki dá Duchá Świętego tym / którzy go oń proƺą?
Tedy wygániáł dyjábłá / który był niemy. Y ſtáło śię / gdy wyƺedł on dyjábeł / przemówił niemy ; y dźiwowáł śię lud.
Ale niektórzy z nich mówili : Przez Beelzebubá / kśiążęćiá dyjábelſkiego / wygániá dyjábły.
Drudzy záśię kuƺąc go / żądáli známieniá od niego z niebá.
Ale on widząc myśli ich / rzekł im : Káżde króleſtwo rozdźielone ſámo przećiwko ſobie puſtoƺeje / á dom ná dom upádá.
A jeſliżeć y ƺátán rozdźielony jeſt przećiwko ſobie / jákoż śię oſtoi króleſtwo jego? ábowiem powiádáćie / iż ja przez Beelzebubá wygániám dyjábły.
A jeſliż ja przez Beelzebubá wygániám dyjábły / ſynowie wáśi przez kogoż wygániáją? Przetoż oni będą ſędźiámi wáƺymi.
Ale jeſliż ja pálcem Bożym wygániám dyjábły / záiſteć przyƺło do was króleſtwo Boże.
Gdy mocárz uzbrojony ſtrzeże páłácu ſwego / w pokoju ſą májętnośći jego ;
Ale gdy mocniejƺy nádeń nádƺedłƺy / zwyćięży go / odejmuje wƺyſtko oręże jego / w którym ufáł / á łupy jego rozdáje.
Kto nie jeſt ze mną / przećiwko mnie jeſt ; á kto nie zbierá ze mną / rozpráƺá.
Gdy duch niecżyſty wychodźi od cżłowieká / przechádzá śię po miejſcách ſuchych / ƺukájąc odpocżnieniá / á nie ználázłƺy / mówi : Wrócę śię do domu mego / ſkądem wyƺedł.
A przyƺedłƺy / znájduje umiećiony y ochędożony.
Tedy idźie y bierze z ſobą śiedm innych duchów gorƺych niżeli ſám / á wƺedłƺy mieƺkáją tám / y bywáją rzecży oſtátnie cżłowieká onego gorƺe / niżeli pierwƺe.
Y ſtáło śię / gdy on to mówił / że wynióſłƺy głoſ niektóra niewiáſtá z ludu / rzekłá mu : Błogoſłáwiony żywot / który ćię nośił / y pierśi / któreś ſſáł!
Ale on rzekł : Owƺem błogoſłáwieni ſą / którzy ſłucháją ſłowá Bożego y ſtrzegą go.
A gdy śię lud gromádźił / pocżął mówić : Rodzáj ten rodzáj zły jeſt ; známieniá ƺuká / ále mu známię nie będźie dáne / tylko ono známię Jonáƺá proroká.
Abowiem jáko Jonáƺ był zá známię Niniwcżykom / ták będźie y Syn cżłowiecży temu rodzájowi.
Królowá z południá ſtánie ná ſądźie z mężámi rodzáju tego / y potępi je ; bo przyjecháłá od końcżyn źiemi / áby ſłucháłá mądrośći Sálomonowej ; á oto tu więcey / niżeli Sálomon.
Mężowie Niniwiccy ſtáną ná ſądźie z tym rodzájem / y potępią go / przeto / że pokutowáli ná kázánie Jonáƺowe ; á oto tu więcey / niżeli Jonáƺ.
A nikt świecę zápáliwƺy / nie ſtáwiá jey w ſkrytośći / áni pod korzec / ále ná świecżnik / áby ći / którzy wchodzą / świátło widźieli.
Świecą ćiáłá jeſt oko ; jeſliby tedy oko twoje było ƺcżere / y ćiáło twoje wƺyſtko będźie jáſne : á jeſliby złe było / y ćiáło twoje ćiemne będźie.
Pátrzájże tedy / áby świátło / które jeſt w tobie / nie było ćiemnośćią.
Jeſli tedy wƺyſtko ćiáło twoje jáſne będźie / niemájąc jákiej cżąſtki záćmioney / będźieć wƺyſtko ták jáſne / że ćię jáko świecá bláſkiem oświeći.
A gdy to mówił / prośił go niektóry Fáryzeuƺ / áby jádł obiád u niego ; wƺedłƺy tedy / uśiádł zá ſtołem.
A widząc to Fáryzeuƺ / dźiwowáł śię / że śię nie umył przed obiádem.
Y rzekł Pán do niego : Teraz wy / Fáryzeuƺowie! to / co jeſt zewnątrz kubká y miſy / ochędáżáćie / ále to / co jeſt wewnątrz w was / pełne jeſt drápieſtwá y złośći.
Száleni! izáż ten / który ucżynił to / co jeſt zewnątrz / nie ucżynił też tego / co jeſt wewnątrz?
Wƺákże y z tego / co jeſt wewnątrz / dáwájćie jáłmużnę / á oto wƺyſtkie rzecży będą wám cżyſte.
Ale biádá wám / Fáryzeuƺowie! że dájećie dźieśięćinę z mięty / y z ruty / y z káżdego źielá / lecż opuƺcżáćie ſąd y miłość Bożą : teć rzecży trzebá cżynić / á onych nie opuƺcżáć.
Biádá wám / Fáryzeuƺowie! że miłujećie pierwƺe miejſcá w bóżnicách y pozdráwiániá ná rynkách.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! bośćie jáko groby / których nie widáć / á ludźie / którzy chodząc po nich / nie wiedzą o nich.
A odpowiádájąc niektóry z zakonników / rzekł mu : Náucżyćielu! to mówiąc y nas háńbiƺ.
A on rzekł : Y wám zakonnikom biádá! ábowiem obćiążáćie ludźi brzemiony nieznośnemi / á ſámi śię y jednym pálcem ſwoim tych brzemion nie dotykáćie.
Biádá wám! że budujećie groby proroków / á ojcowie wáśi pozábijáli je.
Záiſte świádcżyćie / iż śię kocháćie w ucżynkách ojców wáƺych ; ábowiem oni je pozábijáli / á wy budujećie groby ich.
Dlátegoż też mądrość Bożá rzekłá : Poślę do nich proroki y Apoſtoły / á z nich niektóre zábijáć y prześládowáć będą ;
Aby ƺukáno od tego rodzáju krwi wƺyſtkich proroków / która wyláná jeſt od záłożeniá świátá /
Od krwi Ablá áż do krwi Zácháryjáƺá / który zginął między ołtárzem / y kośćiołem ; záiſte powiádám wám / będą jey ƺukáć od rodzáju tego.
Biádá wám zakonnikom! bośćie wźięli klucż umiejętnośći ; ſámiśćie nie weƺli / á tym / którzy wnijść chćieli / zábrániáliśćie.
A gdy im to mówił / pocżęli náń náucżeni w Piśmie y Fáryzeuƺowie bárdzo náćieráć / y przycżynę mu dáwáć do mówieniá o wielu rzecżách :
Cżyhájąc náń y ƺukájąc / áby co uchwyćili z uſt jego / żeby go oſkárżyli.
