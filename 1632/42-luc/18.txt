Y powiedźiał im jeƺcże podobieńſtwo do tego <i>zmierzájące,</i> iż śię záwżdy potrzebá modlić / á nie uſtawáć /
Mówiąc ; Był niektóry Sędźia w jednym mieśćie / który śię Bogá nie bał / y cżłowieká śię nie wſtydźił.
Byłá też wdowá w tymże mieśćie / która przychodźiłá do niego / mówiąc ; Pomśći śię <i>krzywdy</i> mojey nád przećiwnikiem mojim.
Lecż on długo nie chćiał. Ale potym rzekł ſam w ſobie ; Acżći śię Bogá nie boję / y cżłowieká śię nie wſtydzę :
Wƺákże iż mi śię uprzykrza tá wdowá / pomƺcżę śię <i>krzywdy</i> jey / áby ná oſtátek przyƺedƺy / nie byłá mi ćiężką.
Rzekł tedy PAN : Słuchajćież co mówi nieſpráwiedliwy ſędźia.
A Bóg izali śię nie pomśći <i>krzywdy</i> wybránych ſwojich / wołájących do śiebie we dnie y w nocy / choćiaż im długo ćierpi?
Powiádam wam / iż śię pomśći <i>krzywdy</i> ich w rychle. Ale gdy przydźie Syn cżłowiecży / izali znajdźie wiárę ná źiemi?
Rzekł też y do niektórych którzy ufáli ſámi w ſobie / że byli ſpráwiedliwymi / á inƺych zá nic niemieli / to podobieńſtwo.
Dwoje ludźi wſtąpiło do Kośćiołá áby śię modlili : jeden Fáryzeuƺ / á drugi Celnik.
Fáryzeuƺ ſtánąwƺy / ták śię ſam u śiebie modlił : Dźiękuję tobie Boże / żem nie jeſt jáko inni ludźie / drapieżni / nieſpráwiedliwi / cudzołożnicy / álbo jáko y ten Celnik.
Poƺcżę dwá kroć w tydźień : dawam dźieśięćinę ze wƺyſtkiego co mam.
A Celnik ſtojąc z dáleká / nie chćiał podnieść y ocżu ſwych w niebo / ále śię bił w pierśi ſwoje mówiąc : Boże bądź miłośćiw mnie grzeƺnemu.
Powiádam wam / żeć ten odƺedł uſpráwiedliwionym do domu ſwego / więcey niżeli on. Abowiem kto śię wywyżƺa / będźie poniżony / á kto śię poniża / będźie wywyżƺony.
Przynoƺono też do niego y niemowiątká / áby śię ich dotykał : Co gdy widźieli ucżniowie / gromili je.
Ale JEzus zwoławƺy ich / rzekł : Dopuśććie dźiatkom przychodźić do mnie / á nie zábraniajćie im : ábowiem tákowych jeſt króleſtwo Boże.
Záprawdę powiádam wam : Ktobykolwiek nie przyjął króleſtwá Bożego jáko dźiećiątko / nie wnidźie do niego.
Y pytáło go niektóre Kśiążę / mówiąc : Náucżyćielu dobry / co cżyniąc odźiedźicżę żywot wiecżny?
Y rzekł mu JEzus : Przecż mię zowieƺ dobrym? Nikt nie <i>jeſt</i> dobry / tylko jeden / <i>to jeſt</i> Bóg :
Umieƺ przykazánia? Nie cudzołóƺ : Nie zábijaj : Nie krádni : Nie świadcż fałƺywie : Cżći Ojcá twego y mátkę twoję.
A on rzekł : Tegom wƺyſtkiego przeſtrzegał od młodośći mojey.
Co uſłyƺawƺy JEzus / rzekł mu : Jednegoć jeƺcże nie doſtawa : Wƺyſtko co maƺ przedaj / á rozdaj ubogim / á będźieƺ miał ſkarb w niebie : á przyƺedƺy náśláduj mię.
A on uſłyƺawƺy to / bárzo śię záſmućił : bo był náder bogáty.
A gdy go JEzus ujrzał bárzo záſmuconego / rzekł : Jákoż trudno ći co máją pieniądze / wnidą do króleſtwá Bożego!
Abowiem łátwiey jeſt wielbłądowi przejść przez ucho igielne / niż bogátemu wniść do króleſtwá Bożego.
Tedy rzekli ći którzy to ſłyƺeli : Y któż może być zbáwiony?
Ale on rzekł : Co jeſt nie można u ludźi / można jeſt u Bogá.
Y rzekł Piotr : Otoſmy my opuśćili wƺyſtko / á poƺliſmy zá tobą.
Tedy im on rzekł : Záprawdę powiádam wam / iż niemáƺ nikogo / coby opuśćił dom / álbo rodźice / álbo bráćią / álbo żonę / álbo dźieći / dla króleſtwá Bożego :
Aby nie wźiął dáleko więcey w tym cżáśie / á w przyƺłym wieku żywotá wiecżnego.
A wźiąwƺy z ſobą onych dwunaśćie / rzekł im. Oto wſtępujemy do Jeruzalem : á wypełni śię wƺyſtko co nápiſano przez Proroki o Synu cżłowiecżym.
Bo będźie wydány pogánom / y będźie náśmiewány / y zelżony / y uplwány :
A ubicżowawƺy zábiją go : ále dniá trzećiego zmartwychwſtánie.
Lecż oni z tego nic nie zrozumieli : y było to ſłowo zákryte przed nimi / y nie wiedźieli co mówiono.
Y ſtáło śię gdy śię on przybliżał do Jerychá / ślepy niektóry śiedźiał podle drogi / żebrząc.
A uſłyƺawƺy lud przechodzący / pytał coby to było?
Y powiedźiano mu / iż JEzus Názáreńſki tędy idźie.
Y záwołał mówiąc : JEzuśie Synu Dawidów zmiłuj śię náde mną!
Lecż ći co ƺli w przód / gromili go áby milcżał. Ale on tym więcey wołał : Synu Dawidów zmiłuj śię náde mną!
Záſtánowiwƺy śię tedy JEzus / kazał go przywieść do śiebie. A gdy śię przybliżył / pytał go / Mówiąc :
Co chceƺ ábym ći ucżynił? A on rzekł : PAnie / ábym przejrzał.
A JEzus mu rzekł : Przejrzy / wiárá twojá ćiebie uzdrowiłá.
Y zárázem przejrzał / y ƺedł zá nim wielbiąc Bogá. Co wƺyſtek lud widząc / dał chwałę Bogu.
