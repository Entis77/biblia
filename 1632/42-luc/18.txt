Y powiedziáł im jeƺcże podobieńſtwo do tego zmierzájące / iż śię záwƺe potrzebá modlić / á nie uſtáwáć /
Mówiąc : Był niektóry ſędziá w jednem mieśćie / który śię Bogá nie báł / y cżłowieká śię nie wſtydził.
Byłá też wdowá w temże mieśćie / którá przychodziłá do niego / mówiąc : Pomśćij śię krzywdy mojej nád przećiwnikiem moim.
Lecż on długo nie chćiáł. Ale potem rzekł ſám w ſobie : Acżći śię Bogá nie boję y cżłowieká śię nie wſtydzę /
Wƺákże iż mi śię uprzykrzá tá wdowá / pomƺcżę śię krzywdy jej / áby ná oſtátek przyƺedłƺy / nie byłá mi ćiężką.
Rzekł tedy Pán : Słuchájćież / co mówi nieſpráwiedliwy ſędziá.
A Bóg izáli śię nie pomśći krzywdy wybránych ſwoich / wołájących do śiebie we dnie y w nocy / choćiáż im długo ćierpi?
Powiádám wám / iż śię pomśći krzywdy ich w rychle. Ale gdy przyjdzie Syn cżłowiecży / izáli znájdzie wiárę ná ziemi?
Rzekł też y do niektórych / którzy ufáli ſámi w ſobie / że byli ſpráwiedliwymi / á inƺych zá nic nie mieli / to podobieńſtwo :
Dwoje ludzi wſtąpiło do kośćiołá / áby śię modlili / jeden Fáryzeuƺ á drugi celnik.
Fáryzeuƺ ſtánąwƺy / ták śię ſám u śiebie modlił : Dziękuję tobie / Boże! żem nie jeſt jáko inni ludzie / drápieżni / nieſpráwiedliwi / cudzołożnicy / álbo jáko y ten celnik.
Poƺcżę dwákroć w tydzień ; dáję dzieśięćinę ze wƺyſtkiego co mám.
A celnik ſtojąc z dáleká / nie chćiáł podnieść y ocżów ſwych w niebo / ále śię bił w pierśi ſwoje / mówiąc : Boże! bądź miłośćiw mnie grzeƺnemu.
Powiádám wám / żeć ten odƺedł uſpráwiedliwionym do domu ſwego / więcej niżeli on : ábowiem kto śię wywyżƺá / będzie poniżony / á kto śię poniżá / będzie wywyżƺony.
Przynoƺono też do niego y niemowlątká / áby śię ich dotykáł ; co gdy widzieli ucżniowie / gromili je.
Ale Jezuſ zwołáwƺy ich / rzekł : Dopuśććie dziátkom przychodzić do mnie / á nie zábrániájćie im ; ábowiem tákowych jeſt króleſtwo Boże.
Zápráwdę powiádám wám : Ktobykolwiek nie przyjął króleſtwá Bożego jáko dziećiątko / nie wnijdzie do niego.
Y pytáł go niektóry kśiążę / mówiąc : Náucżyćielu dobry! co cżyniąc odziedzicżę żywot wiecżny?
Y rzekł mu Jezuſ : Przecż mię zowieƺ dobrym? Nikt nie jeſt dobry / tylko jeden / to jeſt Bóg.
Umieƺ przykázániá? Nie cudzołóż / nie zábijáj / nie krádnij / nie świádcż fáłƺywie / cżćij ojcá twego y mátkę twoję.
A on rzekł : Tegom wƺyſtkiego przeſtrzegáł od młodośći mojej.
Co uſłyƺáwƺy Jezuſ / rzekł mu : Jednegoć jeƺcże nie doſtáje ; wƺyſtko / co máƺ / ſprzedáj / á rozdáj ubogim / á będzieƺ miáł ſkárb w niebie ; á przyƺedłƺy náśláduj mię.
A on uſłyƺáwƺy to / bárdzo śię záſmućił ; bo był náder bogáty.
A gdy go Jezuſ ujrzáł bárdzo záſmuconego / rzekł : Jákoż trudno ći / co máją pieniądze / wnijdą do króleſtwá Bożego!
Abowiem łátwiej jeſt wielbłądowi przejść przez ucho igielne / niż bogátemu wnijść do króleſtwá Bożego.
Tedy rzekli ći / którzy to ſłyƺeli : Y któż może być zbáwiony?
Ale on rzekł : Co jeſt nie możebne u ludzi / możebne jeſt u Bogá.
Y rzekł Piotr : Otośmy my opuśćili wƺyſtko / á poƺliśmy zá tobą.
Tedy im on rzekł : Zápráwdę powiádám wám / iż niemáƺ nikogo / coby opuśćił dom / álbo rodziców / álbo bráći / álbo żonę / álbo dzieći dlá króleſtwá Bożego /
Aby nie wziął dáleko więcej w tym cżáśie / á w przyƺłym wieku żywotá wiecżnego.
A wziąwƺy z ſobą onych dwunáſtu / rzekł im : Oto wſtępujemy do Jeruzálemu / á wypełni śię wƺyſtko / co nápiſáno przez proroki o Synu cżłowiecżym.
Bo będzie wydány pogánom / y będzie náśmiewány / y zelżony / y uplwány :
A ubicżowáwƺy zábiją go ; ále dniá trzećiego zmártwychwſtánie.
Lecż oni z tego nic nie zrozumieli / y było to ſłowo zákryte przed nimi / y nie wiedzieli / co mówiono.
Y ſtáło śię / gdy śię on przybliżáł do Jerychá / ślepy niektóry śiedziáł podle drogi / żebrząc.
A uſłyƺáwƺy lud przechodzący / pytáł / coby to było?
Y powiedziáno mu / iż Jezuſ Názáreńſki tędy idzie.
Y záwołáł / mówiąc : Jezuśie / Synu Dawidowy! zmiłuj śię nádemną.
Lecż ći co ƺli wprzód / gromili go / áby milcżáł. Ale on tem więcej wołáł : Synu Dawidowy! zmiłuj śię nádemną.
Záſtánowiwƺy śię tedy Jezuſ / kázáł go przywieść do śiebie ; á gdy śię przybliżył / pytáł go / mówiąc :
Co chceƺ / ábym ći ucżynił? A on rzekł : Pánie! ábym przejrzáł.
A Jezuſ mu rzekł : Przejrzyj / wiárá twojá ćiebie uzdrowiłá.
Y zárázem przejrzáł / y ƺedł zá nim / wielbiąc Bogá. Co wƺyſtek lud widząc / dáł chwáłę Bogu.
