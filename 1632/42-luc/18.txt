Y powiedźiáł im jeƺcże podobieńſtwo do tego zmierzájące / iż śię záwƺe potrzebá modlić / á nie uſtáwáć /
Mówiąc : Był niektóry ſędźiá w jednem mieśćie / który śię Bogá nie báł / y cżłowieká śię nie wſtydźił.
Byłá też wdowá w temże mieśćie / którá przychodźiłá do niego / mówiąc : Pomśćij śię krzywdy mojej nád przećiwnikiem moim.
Lecż on długo nie chćiáł. Ale potem rzekł ſám w ſobie : Acżći śię Bogá nie boję y cżłowieká śię nie wſtydzę /
Wƺákże iż mi śię uprzykrzá tá wdowá / pomƺcżę śię krzywdy jej / áby ná oſtátek przyƺedłƺy / nie byłá mi ćiężką.
Rzekł tedy Pán : Słuchájćież / co mówi nieſpráwiedliwy ſędźiá.
A Bóg izáli śię nie pomśći krzywdy wybránych ſwoich / wołájących do śiebie we dnie y w nocy / choćiáż im długo ćierpi?
Powiádám wám / iż śię pomśći krzywdy ich w rychle. Ale gdy przyjdźie Syn cżłowiecży / izáli znájdźie wiárę ná źiemi?
Rzekł też y do niektórych / którzy ufáli ſámi w ſobie / że byli ſpráwiedliwymi / á inƺych zá nic nie mieli / to podobieńſtwo :
Dwoje ludźi wſtąpiło do kośćiołá / áby śię modlili / jeden Fáryzeuƺ á drugi celnik.
Fáryzeuƺ ſtánąwƺy / ták śię ſám u śiebie modlił : Dźiękuję tobie / Boże! żem nie jeſt jáko inni ludźie / drápieżni / nieſpráwiedliwi / cudzołożnicy / álbo jáko y ten celnik.
Poƺcżę dwákroć w tydźień ; dáję dźieśięćinę ze wƺyſtkiego co mám.
A celnik ſtojąc z dáleká / nie chćiáł podnieść y ocżów ſwych w niebo / ále śię bił w pierśi ſwoje / mówiąc : Boże! bądź miłośćiw mnie grzeƺnemu.
Powiádám wám / żeć ten odƺedł uſpráwiedliwionym do domu ſwego / więcej niżeli on : ábowiem kto śię wywyżƺá / będźie poniżony / á kto śię poniżá / będźie wywyżƺony.
Przynoƺono też do niego y niemowlątká / áby śię ich dotykáł ; co gdy widźieli ucżniowie / gromili je.
Ale JEzuſ zwołáwƺy ich / rzekł : Dopuśććie dźiátkom przychodźić do mnie / á nie zábrániájćie im ; ábowiem tákowych jeſt króleſtwo Boże.
Zápráwdę powiádám wám : Ktobykolwiek nie przyjął króleſtwá Bożego jáko dźiećiątko / nie wnijdźie do niego.
Y pytáł go niektóry kśiążę / mówiąc : Náucżyćielu dobry! co cżyniąc odźiedźicżę żywot wiecżny?
Y rzekł mu JEzuſ : Przecż mię zowieƺ dobrym? Nikt nie jeſt dobry / tylko jeden / to jeſt Bóg.
Umieƺ przykázániá? Nie cudzołóż / nie zábijáj / nie krádnij / nie świádcż fáłƺywie / cżćij ojcá twego y mátkę twoję.
A on rzekł : Tegom wƺyſtkiego przeſtrzegáł od młodośći mojej.
Co uſłyƺáwƺy JEzuſ / rzekł mu : Jednegoć jeƺcże nie doſtáje ; wƺyſtko / co máƺ / ſprzedáj / á rozdáj ubogim / á będźieƺ miáł ſkárb w niebie ; á przyƺedłƺy náśláduj mię.
A on uſłyƺáwƺy to / bárdzo śię záſmućił ; bo był náder bogáty.
A gdy go JEzuſ ujrzáł bárdzo záſmuconego / rzekł : Jákoż trudno ći / co máją pieniądze / wnijdą do króleſtwá Bożego!
Abowiem łátwiej jeſt wielbłądowi przejść przez ucho igielne / niż bogátemu wnijść do króleſtwá Bożego.
Tedy rzekli ći / którzy to ſłyƺeli : Y któż może być zbáwiony?
Ale on rzekł : Co jeſt nie możebne u ludźi / możebne jeſt u Bogá.
Y rzekł Piotr : Otośmy my opuśćili wƺyſtko / á poƺliśmy zá tobą.
Tedy im on rzekł : Zápráwdę powiádám wám / iż niemáƺ nikogo / coby opuśćił dom / álbo rodźiców / álbo bráći / álbo żonę / álbo dźieći dlá króleſtwá Bożego /
Aby nie wźiął dáleko więcej w tym cżáśie / á w przyƺłym wieku żywotá wiecżnego.
A wźiąwƺy z ſobą onych dwunáſtu / rzekł im : Oto wſtępujemy do Jeruzálemu / á wypełni śię wƺyſtko / co nápiſáno przez proroki o Synu cżłowiecżym.
Bo będźie wydány pogánom / y będźie náśmiewány / y zelżony / y uplwány :
A ubicżowáwƺy zábiją go ; ále dniá trzećiego zmártwychwſtánie.
Lecż oni z tego nic nie zrozumieli / y było to ſłowo zákryte przed nimi / y nie wiedźieli / co mówiono.
Y ſtáło śię / gdy śię on przybliżáł do Jerychá / ślepy niektóry śiedźiáł podle drogi / żebrząc.
A uſłyƺáwƺy lud przechodzący / pytáł / coby to było?
Y powiedźiáno mu / iż JEzuſ Názáreńſki tędy idźie.
Y záwołáł / mówiąc : JEzuśie / Synu Dawidowy! zmiłuj śię nádemną.
Lecż ći co ƺli wprzód / gromili go / áby milcżáł. Ale on tem więcej wołáł : Synu Dawidowy! zmiłuj śię nádemną.
Záſtánowiwƺy śię tedy JEzuſ / kázáł go przywieść do śiebie ; á gdy śię przybliżył / pytáł go / mówiąc :
Co chceƺ / ábym ći ucżynił? A on rzekł : Pánie! ábym przejrzáł.
A JEzuſ mu rzekł : Przejrzyj / wiárá twojá ćiebie uzdrowiłá.
Y zárázem przejrzáł / y ƺedł zá nim / wielbiąc Bogá. Co wƺyſtek lud widząc / dáł chwáłę Bogu.
