A gdy dokońcżył wƺyſtkich mów ſwoich przed onym ludem / wƺedł do Kápernáum ;
A niektórego ſetniká ſługá źle śię májąc / już práwie miáł umrzeć / którego on ſobie bárdzo powáżáł.
Ten uſłyƺáwƺy o JEzuśie / poſłáł do niego ſtárƺych z Żydów / proƺąc go / áby przyƺedłƺy uzdrowił ſługę jego.
A oni przyƺedłƺy do JEzuſá / prośili go z pilnośćią / mówiąc : Godźien jeſt / ábyś mu to ucżynił ;
Abowiem miłuje náród náƺ / y on nám bóżnicę zbudowáł.
A ták JEzuſ ƺedł z nimi. Ale gdy niedáleko był od domu / poſłáł do niego on ſetnik przyjáćioły / mówiąc mu : Pánie! nie zádáwáj ſobie prácy ; bomći nie jeſt godźien / ábyś wƺedł pod dách mój.
Przetoż y ſámego śiebie nie miáłem zá godnego / ábym był miáł przyjść do ćiebie ; ále rzecż ſłowo / á będźie uzdrowiony ſługá mój.
Bomći y já cżłowiek pod mocą poſtánowiony / májący pod ſobą żołnierzy / y mówię temu : Idź / á idźie / á drugiemu : Przyjdź / á przychodźi / á ſłudze mojemu : Cżyń to / á cżyni.
Tedy uſłyƺáwƺy to JEzuſ / zádźiwił mu śię / y obróćiwƺy śię / rzekł do ludu / który zá nim ƺedł : Powiádám wám / żem áni w Izráelu ták wielkiej wiáry nie ználázł.
A wróćiwƺy śię do domu ći / którzy byli poſłáni / ználeźli ſługę / który śię źle miáł / zdrowego.
Y ſtáło śię názájutrz / że ƺedł do miáſtá / które zowią Náim / á ƺło z nim ucżniów jego wiele y lud wielki.
A gdy śię przybliżył do brámy miejſkiej / tedy oto wynoƺono umárłego / ſyná jedynego mátki ſwojej / á tá byłá wdową / á z nią ƺedł wielki lud miáſtá onego.
Którą ujrzáwƺy Pán użálił śię jej / y rzekł jej : Nie płácż!
Y przyſtąpiwƺy dotknął śię trumny / ( á ći / co nieśli / ſtánęli ) y rzekł : Młodźieńcże! tobie mówię / wſtáń.
Y uśiádł on / który był umárł / y pocżął mówić ; y oddáł go mátce jego.
Tedy wƺyſtkich ſtrách zdjął / á wielbili Bogá / mówiąc : Prorok wielki powſtáł między námi / á Bóg náwiedźił lud ſwój.
Y rozeƺłá śię o nim tá wieść po wƺyſtkiej Judzkiej źiemi / y po wƺyſtkiej okolicżnej kráinie.
Y oznájmili Jánowi ucżniowie jego o tem wƺyſtkiem. A wezwáwƺy dwóch niektórych z ucżniów ſwoich /
Poſłáł je do JEzuſá / mówiąc : Tyżeś jeſt ten / który má przyjść / cżyli inƺego cżekáć mámy?
A gdy przyƺli do niego mężowie oni / rzekli : Ján Chrzćićiel poſłáł náſ do ćiebie / mówiąc : Tyżeś jeſt ten / który má przyjść / cżyli inƺego cżekáć mámy?
A onejże godźiny wiele ich uzdrowił od chorób / od niemocy / y od duchów złych / y wiele ślepych wzrokiem dárowáł.
A odpowiádájąc JEzuſ / rzekł im : Szedłƺy oznájmijćie Jánowi / cośćie widźieli y ſłyƺeli / iż ślepi widzą / chromi chodzą / trędowáći biorą ocżyƺcżenie / głuśi ſłyƺą / umárli zmártwychwſtáją / á ubogim opowiádáná bywá Ewángielijá.
A błogoſłáwiony jeſt / ktoby śię nie zgorƺył ze mnie.
A gdy odeƺli poſłowie Jánowi / pocżął mówić do ludu o Jánie : Cośćie wyƺli ná puƺcżę widźieć? Izáli trzćinę chwiejącą śię od wiátru?
Ale cośćie wyƺli widźieć? Izáli cżłowieká w miękkie ƺáty oblecżonego? Oto ći / którzy w ƺátách koƺtownych y w rozkoƺy żyją / ſą w domách królewſkich.
Ale cośćie wyƺli widźieć? Izáli proroká? Záiſte powiádám wám / iż więcej niż proroká.
Tenći bowiem jeſt / o którym nápiſáno : Oto Já poſyłám Aniołá mego przed oblicżem twojim / który zgotuje drogę twoję przed tobą.
Abowiem powiádám wám : Więkƺego proroká z tych / którzy śię z niewiáſt rodzą / niemáƺ nád Jáná Chrzćićielá żádnego ; lecż kto nájmniejƺy jeſt w króleſtwie Bożem / więkƺy jeſt / niżeli on.
Tedy wƺyſtek lud ſłyƺąc to / y celnicy / wielbili Bogá / będąc ochrzcżeni chrztem Jánowym.
Ale Fáryzeuƺowie y zákonnicy pogárdźili rádą Bożą ſámi przećiwko ſobie / nie będąc ochrzcżeni od niego.
Y rzekł Pán : Komuż tedy przypodobám ludźi rodzáju tego / á komu ſą podobni?
Podobni ſą dźiećiom / które śiedzą ná rynku / á jedne ná drugie wołáją / mówiąc : Gráłyśmy wám ná piƺcżáłkách / á nie táńcowáliśćie ; śpiewáłyśmy żáłobne pieśni / á nie płákáliśćie.
Abowiem przyƺedł Ján Chrzćićiel / y chlebá nie jedząc y winá nie pijąc / á mówićie : Dyjábelſtwo má.
Przyƺedł Syn cżłowiecży jedząc y pijąc / á mówićie : Oto cżłowiek obżercá y pijánicá winá / przyjáćiel celników y grzeƺników.
Ale uſpráwiedliwioná jeſt mądrość od wƺyſtkich ſynów ſwoich.
Y prośił go niektóry z Fáryzeuƺów / áby z nim jádł. A ták wƺedłƺy w dom Fáryzeuƺów / uśiádł.
A oto niewiáſtá / którá byłá w mieśćie grzeƺná / dowiedźiáwƺy śię / iż śiedźi w domu Fáryzeuƺowym / przynioſłá álábáſtrowy ſłoik máśći ;
A ſtánąwƺy z tyłu u nóg jego / płácżąc pocżęłá łzámi polewáć nogi jego / á włoſámi głowy ſwojej ućieráłá / y cáłowáłá nogi jego / y máśćią mázáłá.
A widząc to Fáryzeuƺ / który go był wezwáł / rzekł ſám w ſobie / mówiąc : Być ten był prorokiem / wiedźiáłby / którá y jáká jeſt tá niewiáſtá / co śię go dotyká ; bo jeſt grzeƺnicá.
A odpowiádájąc JEzuſ / rzekł do niego : Szymonie! mám ći nieco powiedźieć / á on rzekł : Powiedz / Náucżyćielu!
Miáł niektóry lichwiárz dwóch dłużników ; jeden dłużen był pięć ſet groƺy / á drugi pięćdźieśiąt.
A gdy oni nie mieli cżem zápłáćić / odpuśćił obydwom. Powiedz tedy / któryż z nich bárdźiej go miłowáć będźie?
A odpowiádájąc Szymon / rzekł : Mniemám / iż ten / któremu więcej odpuśćił. A on mu rzekł : Dobrześ rozſądźił.
Y obróćiwƺy śię do niewiáſty / rzekł Szymonowi : Widźiƺ tę niewiáſtę? Wƺedłem do domu twego / nie dáłeś wody ná nogi moje ; ále tá łzámi poláłá nogi moje / y włoſámi głowy ſwej otárłá.
Nie pocáłowáłeś mię / ále tá jáko weƺłá / nie przeſtáłá cáłowáć nóg moich.
Nie pomázáłeś oliwą głowy mojej ; ále tá máśćią pomázáłá nogi moje.
Dlácżego / mówię tobie / odpuƺcżono jej wiele grzechów / gdyż wiele umiłowáłá ; á komu máło odpuƺcżono / máło miłuje.
A on jej rzekł : Odpuƺcżone ſą tobie grzechy.
Y pocżęli ſpółśiedzący mówić między ſobą : Któż jeſt ten / który y grzechy odpuƺcżá?
Y rzekł do niewiáſty : Wiárá twojá ćiebie zbáwiłá. Idźże w pokoju.
