A gdy dokońcżył wƺyſtkich mów ſwojich przed onym ludem wƺedł do Kápernáum.
A niektórego Setniká ſługá / źle śię májąc <i>już práwie</i> miał umrzeć / którego on ſobie bárzo poważał :
Ten uſłyƺawƺy o JEzuśie / poſłał do niego Stárƺe z Żydów / proƺąc go / áby przyƺedƺy uzdrowił ſługę jego.
A oni przyƺedƺy do JEzuſá prośili go z pilnośćią / mówiąc ; Godźien jeſt ábyś mu to ucżynił.
Abowiem miłuje naród náƺ / y on nam bóżnicę zbudował.
A ták JEzus ƺedł z nimi. Ale gdy nie dáleko był od domu / poſłał do niego on Setnik przyjaćioły / mówiąc mu : PAnie / nie zádawaj ſobie prace : ( Bomći nie jeſt godźien / ábyś wƺedł pod dách mój :
Przetoż y ſámego śiebie nie miałem zá godnego / ábym był miał przyść do ćiebie. ) Ale rzecż ſłowo / á będźie uzdrowiony ſługá mój.
Bomći y ja cżłowiek pod mocą poſtánowiony / májący pod ſobą żołnierze : y mówię temu ; Idź / á idźie : á drugiemu ; Przydź / á przychodźi : á ſłudze mojemu ; Cżyń to / á cżyni.
Tedy uſłyƺawƺy to JEzus / zádźiwił mu śię : y obróćiwƺy śię / rzekł do ludu który zá nim ƺedł ; Powiádam wam / żem áni w Izráelu ták wielkiey wiáry nie ználazł.
A wróćiwƺy śię do domu ći którzy byli poſłáni / ználeźli ſługę który śię źle miał / zdrowego.
Y ſtáło śię názájutrz / że ƺedł do miáſtá które zowią Najim ; á ƺło z nim ucżniów jego wiele / y lud wielki.
A gdy śię przybliżył do bramy miejſkiey / tedy oto wynoƺono umárłego / ſyná jedynego mátki ſwojey ; á tá <i>byłá</i> wdową : á z nią ƺedł wielki lud miáſtá onego.
Którą ujrzawƺy PAN / użalił śię jey / y rzekł jey ; Nie płácż.
Y przyſtąpiwƺy dotknął śię truny / ( á ći co nieśli / ſtánęli ) y rzekł ; Młodźieńcże / tobie mówię / wſtań.
Y uśiadł on który był umárł / y pocżął mówić ; y oddał go mátce jego.
Tedy wƺyſtkie ſtrách zdjął / á wielbili Bogá / mówiąc ; Prorok wielki powſtał miedzy námi / á Bóg náwiedźił lud ſwój.
Y rozeƺłá śię o nim tá wieść po wƺyſtkiey Judſkiey źiemi / y po wƺyſtkiey okolicżney krájinie.
Y oznájmili Janowi ucżniowie jego o tym wƺyſtkim. A wezwawƺy dwu niektórych z ucżniów ſwoich /
Poſłał do JEzuſá / mówiąc : Tyżeś jeſt ten który ma przyść? cżyli inƺego cżekáć mamy?
A gdy przyƺli do niego mężowie oni / rzekli ; Jan Chrzćićiel poſłał nas do ćiebie / mówiąc : Tyżeś jeſt ten który ma przyść? cżyli inƺego cżekáć mamy?
A onejże godźiny wiele ich uzdrowił od chorób / od niemocy / y od duchów złych / y wiele ślepych wzrokiem dárował.
A odpowiádájąc JEzus / rzekł im ; Sżedƺy oznájmićie Janowi cośćie widźieli y ſłyƺeli : iż ślepi widzą / chromi chodzą / trędowáći biorą ocżyśćienie / głuƺy ſłyƺą / umárli zmartwychwſtáją / á ubogim opowiádána bywa Ewángelia :
A błogoſłáwiony jeſt ktoby śię nie zgorƺył ze mnie.
A gdy odeƺli poſłowie Janowi / pocżął mówić do ludu o Janie : Cośćie wyƺli ná puƺcżę widźieć? <i>Izali</i> trzćinę chwiejącą śię od wiátru?
Ale cośćie wyƺli widźieć? <i>Izali</i> cżłowieká w miękkie ƺáty oblecżonego? Oto ći którzy w ƺátách koƺtownych y w rozkoƺy żyją / ſą w domách królewſkich.
Ale cośćie wyƺli widźieć? <i>Izali</i> Proroká? Zájiſte powiádam wam / iż więcey niż Proroká.
Tenći bowiem jeſt o którym nápiſano ; Oto Ja poſyłam Anjołá mego przed oblicżem twojim / który zgotuje drogę twoję przed tobą.
Abowiem powiádam wam : Więkƺego Proroká z tych / którzy śię z niewiaſt rodzą / niemáƺ nád Janá Chrzćićielá żadnego. Lecż kto mniejƺy jeſt w króleſtwie Bożym / więkƺy jeſt niżeli on.
Tedy wƺyſtek lud ſłyƺąc to y Celnicy / wielbili Bogá / będąc ochrzcżeni chrztem Janowym.
Ale Fáryzeuƺowie y Zakonnicy pogárdźili rádą Bożą ſámi przećiwko ſobie / nie będąc ochrzcżeni od niego.
Y rzekł PAN ; Komuż tedy przypodobam ludźie rodzáju tego? A komu ſą podobni?
Podobni ſą dźiećiom / które śiedzą ná rynku / á jedne ná drugie wołáją mówiąc ; Gráliſmy wam ná piƺcżałkách / á nie táńcowáliśćie śpiewáliſmy żałobne pieśni / á nie płákáliśćie.
Abowiem przyƺedł Jan Chrzćićiel / y chlebá nie jedząc / y winá nie pijąc / á mówićie : Dyabelſtwo ma.
Przyƺedł Syn cżłowiecży jedząc y pijąc / á mówićie ; Oto cżłowiek obżercá y pijánicá winá / przyjaćiel Celników y grzeƺników.
Ale uſpráwiedliwiona jeſt mądrość od wƺyſtkich ſynów ſwojich.
Y prośił go niektóry z Fáryzeuƺów / áby z nim jadł : á ták wƺedƺy w dom Fáryzeuƺów / uśiadł.
A oto niewiáſtá która byłá w mieśćie grzeƺna / dowiedźiawƺy śię iż śiedźi w domu Fáryzeuƺowym / przynioſłá álábáſtrowy ſłojek máśći.
A ſtánąwƺy z tyłu u nóg jego / płácżąc pocżęłá łzámi polewáć nogi jego ; á włoſámi głowy ſwojey ućieráłá / y cáłowáłá nogi jego / y máśćią mázáłá.
A widząc to Fáryzeuƺ który go był wezwał / rzekł ſam w ſobie / mówiąc ; Być ten był Prorokiem / wiedźiałby która y jáka jeſt tá niewiáſtá / co śię go dotyka : bo jeſt grzeƺnicá.
A odpowiádájąc JEzus / rzekł do niego ; Symonie / mamći nieco powiedźieć : A on rzekł ; Powiedz Náucżyćielu.
Miał niektóry lichwiarz dwu dłużników : jeden dłużen był pięć ſet groƺy / á drugi pięćdźieśiąt.
A gdy oni nie mieli cżym zápłáćić / odpuśćił obiemá. Powiedz tedy / któryż z nich bárźiey go miłowáć będźie?
A odpowiádájąc Symon / rzekł ; Mniemam iż ten któremu więcey odpuśćił. A on mu rzekł ; Dobrześ rozſądźił.
Y obróćiwƺy śię do niewiáſty / rzekł Symonowi ; Widźiƺ tę niewiáſtę? Wƺedłem do domu twego / niedałeś wody ná nogi moje : ále tá łzámi polałá nogi moje / y włoſámi głowy ſwey otárłá.
Nie pocáłowałeś mię : ále tá jáko weƺłá / nieprzeſtáłá cáłowáć nóg mojich.
Nie pomázałeś oliwą głowy mojey : ále tá máśćią pomázáłá nogi moje.
Dla cżego mówię tobie / odpuƺcżono jey wiele grzechów : gdyż wiele umiłowáłá : á komu máło odpuƺcżono / máło miłuje.
A on jey rzekł : Odpuƺcżone ſą tobie grzechy.
Y pocżęli ſpół-śiedzący mówić miedzy ſobą : Któż jeſt ten który y grzechy odpuƺcża?
Y rzekł do niewiáſty : Wiárá twojá ćiebie zbáwiłá. Idźże w pokoju.
