Poniewáż wiele śię ich podjęło / ſporządzić hiſtoryję o tych ſpráwách / o których my pewną wiádomość mámy ;
Ták jáko nám podáli ći / którzy od pocżątku ſámi widzieli / y ſługámi tego ſłowá byli :
Zdáło śię też y mnie / którym tego wƺyſtkiego z pocżątku pilnie doƺedł / tobie to porządnie wypiſáć / zácny Teofile!
Abyś poznáł pewność tych rzecży / których ćię náucżono.
Był zá dni Herodá / królá Judzkiego / kápłán niektóry / imieniem Zácháryjáƺ / z przemiány Abijáƺowej / á żoná jego byłá z córek Aáronowych / której imię było Elżbietá.
A byli oboje ſpráwiedliwymi przed oblicżem Bożem / chodząc we wƺyſtkich przykázániách y uſpráwiedliwieniách Páńſkich bez nágány.
Y nie mieli potomſtwá / przeto iż Elżbietá byłá niepłodná / á byli oboje podeƺłymi w látách ſwoich.
Stáło śię tedy / gdy odpráwiáł urząd kápłáńſki w porządku przemiány ſwojej przed Bogiem /
Że według zwycżáju urzędu kápłáńſkiego przypádł náń loſ / áby kádził / wƺedłƺy do kośćiołá Páńſkiego.
A wƺyſtko mnóſtwo ludu było ná dworze / modląc śię w godzinę kádzeniá.
Tedy mu śię pokázáł Anioł Páńſki / ſtojący po práwej ſtronie ołtárzá / ná którym kádzono.
Y zátrwożył śię Zácháryjáƺ ujrzáwƺy go / á bojáźń przypádłá náń.
Y rzekł do niego Anioł : Nie bój śię / Zácháryjáƺu! boć jeſt wyſłucháná modlitwá twojá / á Elżbietá / żoná twojá / urodzić ſyná / y názwieƺ imię jego Ján /
Z którego będzieƺ miáł rádość y weſele / y wiele ich rádowáć śię będą z národzeniá jego.
Abowiem będzie wielkim przed oblicżem Páńſkim ; winá y nápoju mocnego nie będzie pił / á Duchem Świętym będzie nápełniony záráz z żywotá mátki ſwojej.
A wielu z ſynów Izráelſkich obróći ku Pánu / Bogu ich.
Bo on pójdzie wprzód przed oblicżem jego w duchu y w mocy Elijáƺowej / áby obróćił ſercá ojców ku dziećiom / á odporne ku roztropnośći ſpráwiedliwych / áby ſporządził Pánu lud gotowy.
Y rzekł Zácháryjáƺ do Aniołá : Po cżemże to poznám? bom já jeſt ſtáry / á żoná mojá podeƺłá w látách ſwych.
A odpowiádájąc Anioł / rzekł mu : Jám jeſt Gábryjel / który ſtoję przed oblicżem Bożem / á poſłánym jeſt / ábym mówił do ćiebie / á iżbym ći to weſołe poſelſtwo odnióſł.
A oto oniemiejeƺ / á nie będzieƺ mógł mówić áż do onego dniá / którego śię to ſtánie / dlátego / żeś nie uwierzył ſłowom moim / które śię wypełnią cżáſu ſwego.
A lud ocżekiwáł Zácháryjáƺá ; y dziwowáli śię / że ták długo báwił w kośćiele.
A wyƺedłƺy nie mógł do nich mówić ; y poználi / że widzenie widziáł w kośćiele ; bo im przez znáki ukázywáł / y zoſtáł niemym.
Y ſtáło śię / gdy śię wypełniły dni poſługiwániá jego / odƺedł do domu ſwego.
A po onych dniách pocżęłá Elżbietá / żoná jego / y kryłá śię przez pięć mieśięcy / mówiąc :
Iż mi ták Pán ucżynił we dni / w które ná mię wejrzáł / áby odjął háńbę moję między ludźmi.
A w mieśiącu ƺóſtym poſłány jeſt Anioł Gábryjel od Bogá do miáſtá Gálilejſkiego / które zwáno Názáret /
Do Pánny poślubionej mężowi / któremu imię było Józef / z domu Dawidowego / á imię Pánny / Máryjá.
A wƺedłƺy Anioł do niej / rzekł : Bądź pozdrowioná / łáſką udárowáná / Pán jeſt z tobą ; błogoſłáwionáś ty między niewiáſtámi.
Ale oná ujrzáwƺy go / zátrwożyłá śię ná ſłowá jego / y myśliłá / jákieby to było pozdrowienie.
Y rzekł jej Anioł : Nie bój śię / Máryjo! ábowiem ználázłáś łáſkę u Bogá.
A oto pocżnieƺ w żywoćie / y porodziƺ ſyná / y názwieƺ imię jego Jezuſ.
Ten będzie wielki / á Synem Nájwyżƺego będzie názwány / y dá mu Pán Bóg ſtolicę Dawidá / ojcá jego ;
Y będzie królowáł nád domem Jákóbowym ná wieki / á króleſtwu jego nie będzie końcá.
Zátem Máryjá rzekłá do Aniołá : Jákoż to będzie / gdyż já mężá nie znám?
A odpowiádájąc Anioł / rzekł jej : Duch Święty zſtąpi ná ćię / á moc Nájwyżƺego záćieni ćię ; przetoż y to / co śię z ćiebie święte národzi / názwáne będzie Synem Bożym.
A oto Elżbietá / pokrewná twojá / y oná pocżęłá ſyná w ſtárośći ſwojej / á ten mieśiąc jeſt ƺóſty onej / którą názywáno niepłodną.
Bo nie będzie niemożne u Bogá żádne ſłowo.
Y rzekłá Máryjá : Oto ſłużebnicá Páńſká ; niechże mi śię ſtánie według ſłowá twego. Y odƺedł od niej Anioł.
Tedy wſtáwƺy Máryjá w onych dniách / poƺłá w górną kráinę z kwápieniem do miáſtá Judzkiego.
A wƺedłƺy w dom Zácháryjáƺowy / pozdrowiłá Elżbietę.
Y ſtáło śię / ſkoro uſłyƺáłá Elżbietá pozdrowienie Máryi / ſkocżyło niemowlątko w żywoćie jej / y nápełnioná jeſt Elżbietá Duchem Świętym.
Y záwołáłá głoſem wielkim / y rzekłá : Błogoſłáwionáś ty między niewiáſtámi / y błogoſłáwiony owoc żywotá twego!
A ſkądże mi to / iż przyƺłá mátká Páná mego do mnie?
Abowiem jáko doƺedł głoſ pozdrowieniá twego do uƺów moich / podſkocżyło od rádośći niemowlątko w żywoćie moim.
A błogoſłáwioná / którá uwierzyłá : gdyż śię wykonáją te rzecży / które jej ſą opowiedziáne od Páná.
Tedy rzekłá Máryjá : Wielbi duƺá mojá Páná ;
Y rozrádowáł śię duch mój w Bogu / zbáwićielu moim /
Iż wejrzáł ná uniżenie ſłużebnicy ſwojej ; ábowiem oto odtąd błogoſłáwioną mię zwáć będą wƺyſtkie národy.
Bo mi ucżynił wielkie rzecży ten / który mocny jeſt / y święte imię jego ;
Y którego miłośierdzie zoſtáje od národu do národu nád tymi / co śię go boją.
Dokázáł mocy rámieniem ſwojim / y rozproƺył pyƺne w myślách ſercá ich.
Śćiągnął mocárze z ſtolic ich / á wywyżƺył uniżone.
Łáknące nápełnił dobremi rzecżámi / á bogácże rozpuśćił próżne.
Przyjął Izráelá / ſługę ſwego / pámiętájąc ná miłośierdzie ſwoje.
Jáko mówił do ojców náƺych / do Abráhámá y náśieniá jego ná wieki.
Y zoſtáłá z nią Máryjá jákoby trzy mieśiące ; potem śię wróćiłá do domu ſwego.
A Elżbiećie wypełnił śię cżáſ / áby porodziłá / y porodziłá ſyná.
A uſłyƺáwƺy ſąśiedzi y pokrewni jej / iż Pán z nią ucżynił wielkie miłośierdzie ſwoje / rádowáli śię poſpołu z nią.
Y ſtáło śię / że óſmego dniá przyƺli / áby obrzezáli dziećiątko ; y názwáli je imieniem ojcá jego / Zácháryjáƺem.
Ale odpowiádájąc mátká jego rzekłá : Nie ták ; lecż názwány będzie Jánem.
Y rzekli do niej : Żádnego niemáƺ w rodzie twoim / coby go zwáno tem imieniem.
Y ſkinęli ná ojcá jego / jákoby go chćiáł názwáć.
A on kázáwƺy ſobie podáć táblicżkę / nápiſáł mówiąc : Ján jeſt imię jego. Y dziwowáli śię wƺyſcy.
A zárázem otworzyły śię uſtá jego / y język jego / y mówił / wielbiąc Bogá.
Y przyƺedł ſtrách ná wƺyſtkie ſąśiády ich / y po wƺyſtkiej górnej kráinie Judzkiej rozgłoƺone ſą wƺyſtkie te ſłowá.
Tedy wƺyſcy / którzy o tem ſłyƺeli / ſkłádáli to do ſercá ſwego / mówiąc : Cóż to wżdy zá dziećię będzie? Y byłá z nim ręká Páńſká.
A Zácháryjáƺ / ojćiec jego / nápełniony będąc Duchem Świętym / prorokowáł mówiąc :
Błogoſłáwiony niech będzie Pán / Bóg Izráelſki / iż náwiedził y ſpráwił odkupienie ludowi ſwojimu ;
Y wyſtáwił nám róg zbáwieniá w domu Dawidá / ſługi ſwego /
Ták jáko mówił przez uſtá świętych proroków ſwoich / którzy byli od wieku :
Iż im miáł dáć wybáwienie od nieprzyjáćiół náƺych y z ręki wƺyſtkich / którzy náſ nienáwidzili ;
Aby ucżynił miłośierdzie z ojcámi náƺymi / y wſpomniáł ná przymierze ſwoje święte /
Y ná przyśięgę / którą przyśiągł Abráhámowi / ojcu náƺemu / że nám to dáć miáł /
Iżbyśmy mu bez bojáźni / z ręki nieprzyjáćiół náƺych będąc wybáwieni / ſłużyli ;
W świętobliwośći y w ſpráwiedliwośći przed oblicżem jego / po wƺyſtkie dni żywotá náƺego.
A ty / dziećiątko! Prorokiem Nájwyżƺego názwáne będzieƺ ; bo pójdzieƺ wprzód przed oblicżem Páńſkim / ábyś gotowáł drogi jego /
A iżbyś dáł znájomość zbáwieniá ludowi jego przez odpuƺcżenie grzechów ich.
Przez wnętrznośći miłośierdziá Bogá náƺego / w których náwiedził náſ Wſchód z wyſokośći.
Aby śię ukázáł śiedzącym w ćiemnośći y w ćieniu śmierći ku wyproſtowániu nóg náƺych ná drogę pokoju.
A ono dziećiątko roſło / y umácniáło śię w duchu / y było ná puſtyniách áż do onego dniá / którego śię okázáło przed Izráelem.
