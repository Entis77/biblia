Poniewáż wiele śię ich podjęło / ſporządźić hiſtoryję o tych ſpráwách / o których my pewną wiádomość mámy ;
Ták jáko nám podáli ći / którzy od pocżątku ſámi widźieli / y ſługámi tego ſłowá byli :
Zdáło śię też y mnie / którym tego wƺyſtkiego z pocżątku pilnie doƺedł / tobie to porządnie wypiſáć / zácny Teofile!
Abyś poznał pewność tych rzecży / których ćię náucżono.
Był zá dni Herodá / królá Judzkiego / kápłán niektóry / imieniem Zácháryjáƺ / z przemiány Abijáƺowej / á żoná jego byłá z córek Aáronowych / którey imię było Elżbietá.
A byli oboje ſpráwiedliwymi przed oblicżem Bożem / chodząc we wƺyſtkich przykázániách y uſpráwiedliwieniách Páńſkich bez nágány.
Y nie mieli potomſtwá / przeto iż Elżbietá byłá niepłodná / á byli oboje podeƺłymi w látách ſwoich.
Stáło śię tedy / gdy odpráwiáł urząd kápłáńſki w porządku przemiány ſwojey przed Bogiem /
Że według zwycżáju urzędu kápłáńſkiego przypádł náń loſ / áby kádźił / wƺedłƺy do kośćiołá Páńſkiego.
A wƺyſtko mnóſtwo ludu było ná dworze / modląc śię w godźinę kádzeniá.
Tedy mu śię pokázáł Anioł Páńſki / ſtojący po práwej ſtronie ołtárzá / ná którym kádzono.
Y zátrwożył śię Zácháryjáƺ ujrzáwƺy go / á bojáźń przypádłá náń.
Y rzekł do niego Anioł : Nie bój śię / Zácháryjáƺu! boć jeſt wyſłucháná modlitwá twojá / á Elżbietá / żoná twojá / urodźić ſyná / y názwieƺ imię jego Ján /
Z którego będźieƺ miał rádość y weſele / y wiele ich rádowáć śię będą z národzeniá jego.
Abowiem będźie wielkim przed oblicżem Páńſkim ; winá y nápoju mocnego nie będźie pił / á Duchem Świętym będźie nápełniony záráz z żywotá mátki ſwojey.
A wielu z ſynów Izráelſkich obróći ku Pánu / Bogu ich.
Bo on pójdźie wprzód przed oblicżem jego w duchu y w mocy Elijáƺowej / áby obróćił ſercá ojców ku dźiećiom / á odporne ku roztropnośći ſpráwiedliwych / áby ſporządźił Pánu lud gotowy.
Y rzekł Zácháryjáƺ do Aniołá : Po cżemże to poznám? bom ja jeſt ſtáry / á żoná mojá podeƺłá w látách ſwych.
A odpowiádájąc Anioł / rzekł mu : Jám jeſt Gábryjel / który ſtoję przed oblicżem Bożem / á poſłánym jeſt / ábym mówił do ćiebie / á iżbym ći to weſołe poſelſtwo odnióſł.
A oto oniemiejeƺ / á nie będźieƺ mógł mówić áż do onego dniá / którego śię to ſtánie / dlátego / żeś nie uwierzył ſłowom moim / które śię wypełnią cżáſu ſwego.
A lud ocżekiwáł Zácháryjáƺá ; y dźiwowáli śię / że ták długo báwił w kośćiele.
A wyƺedłƺy nie mógł do nich mówić ; y poználi / że widzenie widźiáł w kośćiele ; bo im przez znáki ukázywáł / y zoſtáł niemym.
Y ſtáło śię / gdy śię wypełniły dni poſługiwániá jego / odƺedł do domu ſwego.
A po onych dniách pocżęłá Elżbietá / żoná jego / y kryłá śię przez pięć mieśięcy / mówiąc :
Iż mi ták Pán ucżynił we dni / w które ná mię wejrzáł / áby odjął háńbę moję między ludźmi.
A w mieśiącu ƺóſtym poſłány jeſt Anioł Gábryjel od Bogá do miáſtá Gálilejſkiego / które zwáno Názáret /
Do Pánny poślubioney mężowi / któremu imię było Józef / z domu Dawidowego / á imię Pánny / Máryjá.
A wƺedłƺy Anioł do niej / rzekł : Bądź pozdrowioná / łáſką udárowáná / Pán jeſt z tobą ; błogoſłáwionáś ty między niewiáſtámi.
Ale oná ujrzáwƺy go / zátrwożyłá śię ná ſłowá jego / y myśliłá / jákieby to było pozdrowienie.
Y rzekł jey Anioł : Nie bój śię / Máryjo! ábowiem ználázłáś łáſkę u Bogá.
A oto pocżnieƺ w żywoćie / y porodźiƺ ſyná / y názwieƺ imię jego JEzuſ.
Ten będźie wielki / á Synem Nájwyżƺego będźie názwány / y dá mu Pán Bóg ſtolicę Dawidá / ojcá jego ;
Y będźie królowáł nád domem Jákóbowym ná wieki / á króleſtwu jego nie będźie końcá.
Zátem Máryjá rzekłá do Aniołá : Jákoż to będźie / gdyż ja mężá nie znám?
A odpowiádájąc Anioł / rzekł jey : Duch Święty zſtąpi ná ćię / á moc Nájwyżƺego záćieni ćię ; przetoż y to / co śię z ćiebie święte národźi / názwáne będźie Synem Bożym.
A oto Elżbietá / pokrewná twojá / y oná pocżęłá ſyná w ſtárośći ſwojey / á ten mieśiąc jeſt ƺóſty oney / którą názywáno niepłodną.
Bo nie będźie niemożne u Bogá żádne ſłowo.
Y rzekłá Máryjá : Oto ſłużebnicá Páńſká ; niechże mi śię ſtánie według ſłowá twego. Y odƺedł od niej Anioł.
Tedy wſtáwƺy Máryjá w onych dniách / poƺłá w górną kráinę z kwápieniem do miáſtá Judzkiego.
A wƺedłƺy w dom Zácháryjáƺowy / pozdrowiłá Elżbietę.
Y ſtáło śię / ſkoro uſłyƺáłá Elżbietá pozdrowienie Máryi / ſkocżyło niemowlątko w żywoćie jey / y nápełnioná jeſt Elżbietá Duchem Świętym.
Y záwołáłá głoſem wielkim / y rzekłá : Błogoſłáwionáś ty między niewiáſtámi / y błogoſłáwiony owoc żywotá twego!
A ſkądże mi to / iż przyƺłá mátká Páná mego do mnie?
Abowiem jáko doƺedł głoſ pozdrowieniá twego do uƺów moich / podſkocżyło od rádośći niemowlątko w żywoćie moim.
A błogoſłáwioná / która uwierzyłá : gdyż śię wykonáją te rzecży / które jey ſą opowiedźiáne od Páná.
Tedy rzekłá Máryjá : Wielbi duƺá mojá Páná ;
Y rozrádowáł śię duch mój w Bogu / zbáwićielu moim /
Iż wejrzáł ná uniżenie ſłużebnicy ſwojey ; ábowiem oto odtąd błogoſłáwioną mię zwáć będą wƺyſtkie národy.
Bo mi ucżynił wielkie rzecży ten / który mocny jeſt / y święte imię jego ;
Y którego miłośierdźie zoſtáje od národu do národu nád tymi / co śię go boją.
Dokázáł mocy rámieniem ſwojim / y rozproƺył pyƺne w myślách ſercá ich.
Śćiągnął mocárze z ſtolic ich / á wywyżƺył uniżone.
Łáknące nápełnił dobremi rzecżámi / á bogácże rozpuśćił próżne.
Przyjął Izráelá / ſługę ſwego / pámiętájąc ná miłośierdźie ſwoje.
Jáko mówił do ojców náƺych / do Abráhámá y náśieniá jego ná wieki.
Y zoſtáłá z nią Máryjá jákoby trzy mieśiące ; potem śię wróćiłá do domu ſwego.
A Elżbiećie wypełnił śię cżáſ / áby porodźiłá / y porodźiłá ſyná.
A uſłyƺáwƺy ſąśiedźi y pokrewni jey / iż Pán z nią ucżynił wielkie miłośierdźie ſwoje / rádowáli śię poſpołu z nią.
Y ſtáło śię / że óſmego dniá przyƺli / áby obrzezáli dźiećiątko ; y názwáli je imieniem ojcá jego / Zácháryjáƺem.
Ale odpowiádájąc mátká jego rzekłá : Nie ták ; lecż názwány będźie Jánem.
Y rzekli do niej : Żádnego niemáƺ w rodźie twoim / coby go zwáno tem imieniem.
Y ſkinęli ná ojcá jego / jákoby go chćiáł názwáć.
A on kázáwƺy ſobie podáć táblicżkę / nápiſáł mówiąc : Ján jeſt imię jego. Y dźiwowáli śię wƺyſcy.
A zárázem otworzyły śię uſtá jego / y język jego / y mówił / wielbiąc Bogá.
Y przyƺedł ſtrách ná wƺyſtkie ſąśiády ich / y po wƺyſtkiej górney kráinie Judzkiej rozgłoƺone ſą wƺyſtkie te ſłowá.
Tedy wƺyſcy / którzy o tem ſłyƺeli / ſkłádáli to do ſercá ſwego / mówiąc : Cóż to wżdy zá dźiećię będźie? Y byłá z nim ręká Páńſká.
A Zácháryjáƺ / ojćiec jego / nápełniony będąc Duchem Świętym / prorokowáł mówiąc :
Błogoſłáwiony niech będźie Pán / Bóg Izráelſki / iż náwiedźił y ſpráwił odkupienie ludowi ſwojimu ;
Y wyſtáwił nám róg zbáwieniá w domu Dawidá / ſługi ſwego /
Ták jáko mówił przez uſtá świętych proroków ſwoich / którzy byli od wieku :
Iż im miał dáć wybáwienie od nieprzyjáćiół náƺych y z ręki wƺyſtkich / którzy nas nienáwidźili ;
Aby ucżynił miłośierdźie z ojcámi náƺymi / y wſpomniáł ná przymierze ſwoje święte /
Y ná przyśięgę / którą przyśiągł Abráhámowi / ojcu náƺemu / że nám to dáć miał /
Iżbyśmy mu bez bojáźni / z ręki nieprzyjáćiół náƺych będąc wybáwieni / ſłużyli ;
W świętobliwośći y w ſpráwiedliwośći przed oblicżem jego / po wƺyſtkie dni żywotá náƺego.
A ty / dźiećiątko! Prorokiem Nájwyżƺego názwáne będźieƺ ; bo pójdźieƺ wprzód przed oblicżem Páńſkim / ábyś gotowáł drogi jego /
A iżbyś dáł znájomość zbáwieniá ludowi jego przez odpuƺcżenie grzechów ich.
Przez wnętrznośći miłośierdźiá Bogá náƺego / w których náwiedźił nas Wſchód z wyſokośći.
Aby śię ukázáł śiedzącym w ćiemnośći y w ćieniu śmierći ku wyproſtowániu nóg náƺych ná drogę pokoju.
A ono dźiećiątko roſło / y umácniáło śię w duchu / y było ná puſtyniách áż do onego dniá / którego śię okázáło przed Izráelem.
