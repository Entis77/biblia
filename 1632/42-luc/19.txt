A JEzuſ wƺedłƺy / ƺedł przez Jerycho.
A oto mąż / którego zwáno imieniem Zácheuƺ / który był przełożony nád celnikámi / á ten był bogáty.
Y żądáł widźieć JEzuſá / coby zácż był ; lecż nie mógł przed ludem / bo był máłego wzroſtu.
A bieżáwƺy náprzód / wſtąpił ná drzewo leśnej figi / áby go ujrzáł ; bo támtędy iść miáł.
A gdy przyƺedł ná ono miejſce / ſpojrzáwƺy JEzuſ w górę / ujrzáł go / y rzekł do niego : Zácheuƺu! zſtąp prędko ná dół / ábowiem dźiś muƺę zoſtáć w domu twoim.
Y zſtąpił prędko y przyjął go z rádośćią.
A widząc to wƺyſcy / ƺemráli / mówiąc : U cżłowieká grzeƺnego goſpodą ſtánął.
A ſtánąwƺy Zácheuƺ / rzekł do Páná : Oto połowę májętnośći moich dám ubogim / Pánie! á jeźliżem kogo w cżem podƺedł / oddám w cżwórnáſób.
Y rzekł mu JEzuſ : Dźiś śię ſtáło zbáwienie domowi temu / dlátego / że y on jeſt ſynem Abráhámowym.
Bo przyƺedł Syn cżłowiecży / áby ƺukáł y záchowáł / co było zginęło.
Tedy gdy oni ſłucháli / mówiąc dálej powiedźiáł im podobieńſtwo / dlátego / że był bliſko od Jeruzálemu / á iż oni mniemáli / że śię wnet króleſtwo Boże objáwić miáło.
Rzekł tedy : Niektóry cżłowiek rodu zácnego jecháł w dáleką kráinę / áby ſobie wźiął króleſtwo / y záśię śię wróćił.
A záwołáwƺy dźieśięćiu ſług ſwoich / dáł im dźieśięć grzywien y rzekł do nich : Hándlujćie / áż przyjádę.
Lecż mieƺcżánie jego mieli go w nienáwiśći / y wypráwili zá nim poſelſtwo / mówiąc : Nie chcemy / áby ten królowáł nád námi.
Y ſtáło śię / gdy śię wróćił wźiąwƺy króleſtwo / że rozkázáł do śiebie záwołáć ſług onych / którym był dáł pieniądze / áby śię dowiedźiáł / co który hándlując zyſkáł.
Tedy przyƺedł pierwƺy / mówiąc : Pánie! grzywná twojá dźieśięć grzywien urobiłá.
Y rzekł mu : Dobrze / ſługo dobry! iżeś był nád máłem wiernym / miejże włádzę nád dźieśięćiomá miáſtámi.
Przyƺedł y drugi / mówiąc : Pánie! grzywná twojá pięć grzywien urobiłá.
Rzekł y temu : Y ty bądź nád pięćiomá miáſtámi.
A inƺy przyƺedł / mówiąc : Pánie! oto grzywná twojá / którąm miáł ſchowáną w chuſtce ;
Bom śię ćiebie báł / żeś jeſt cżłowiek ſrogi ; bierzeƺ / cżegoś nie położył / á żnieƺ / cżegoś nie śiáł.
Tedy mu rzekł : Z uſt twoich ſądzę ćię / zły ſługo! Wiedźiáłeś / żem já jeſt cżłowiek ſrogi / który biorę / cżegom nie położył / á żnę / cżegom nie śiáł.
Przecżżeś tedy nie dáł ſrebrá mego do lichwiárzy? á já przyƺedłƺy / wźiąłbym je był z lichwą.
Y rzekł tym / którzy tuż ſtáli : Weźmijćie od niego tę grzywnę / á dájćie temu / który má dźieśięć grzywien.
Y rzekli mu : Pánie! máć dźieśięć grzywien.
Zápráwdę powiádám wám / iż wƺelkiemu / który má / będźie dáne / á od tego / który nie má / y to / co má / będźie od niego odjęte.
Ale y nieprzyjáćioły moje / którzy nie chćieli / ábym królowáł nád nimi / przywiedźćie tu / á pobijćie przedemną.
A to powiedźiáwƺy / ƺedł wprzód / wſtępując do Jeruzálemu.
Y ſtáło śię / gdy śię przybliżył do Betfágie y Betánii / ku górze / którą zowią oliwną / poſłáł dwóch z ucżniów ſwoich /
Mówiąc : Idźćie do miáſtecżká / które jeſt przećiwko wám / do którego wƺedłƺy / znájdźiećie oślę uwiązáne / ná którem żáden cżłowiek nigdy nie śiedźiáł ; odwiązáwƺy je / przywiedźćie :
A jeźliby wáſ kto ſpytáł / przecż je odwiązujećie? ták mu powiećie : Przeto / że go Pán potrzebuje.
Odƺedłƺy tedy ći / którzy byli poſłáni / ználeźli / jáko im był powiedźiáł.
A gdy oni odwiązywáli ono oślę / rzekli pánowie jego do nich : Przecż odwiązujećie oślę?
A oni powiedźieli : Pán go potrzebuje.
Y przywiedli je do JEzuſá / á włożywƺy ƺáty ſwoje ná ono oślę / wſádźili JEzuſá ná nie.
A gdy on jecháł / ſłáli ƺáty ſwoje ná drodze.
A gdy śię już przybliżáł tám / gdźie śię ſpuƺcżáją z góry oliwnej / pocżęło wƺyſtko mnóſtwo ucżniów rádując śię chwálić Bogá głoſem wielkim ze wƺyſtkich cudów / które widźieli /
Mówiąc : Błogoſłáwiony król / który idźie w imieniu Páńſkim ; pokój ná niebie á chwáłá ná wyſokośćiách.
Ale niektórzy z Fáryzeuƺów z onego ludu rzekli do niego : Náucżyćielu! zgrom ucżnie twoje.
A on odpowiádájąc / rzekł im : Powiádám wám / jeźliby ći milcżeli / wnet kámienie wołáć będą.
A gdy śię przybliżył / ujrzáwƺy miáſto / płákáł nád nim / mówiąc :
O gdybyś poznáło y ty / á zwłáƺcżá w ten to dźień twój / co jeſt ku pokojowi twemu! lecż to teráz zákryte od ocżów twoich.
Abowiem przyjdą ná ćię dni / gdy ćię otocżą nieprzyjáćiele twoi wáłem / y oblęgą ćię / y śćiſną ćię zewƺąd ;
Y zrównáją ćię z źiemią / y dźieći twoje w tobie / á nie zoſtáwią w tobie kámieniá ná kámieniu / dlátego / żeś nie poznáło cżáſu náwiedzeniá twego.
A wƺedłƺy do kośćiołá / pocżął wygániáć te / którzy w nim ſprzedáwáli y kupowáli.
Mówiąc im : Nápiſáno : Dom mój dom modlitwy jeſt / á wyśćie go ucżynili jáſkinią zbójców.
Y ucżył ná káżdy dźień w kośćiele ; lecż przedniejśi kápłáni y náucżeni w Piśmie / y przedniejśi z ludu ƺukáli go ſtráćić ;
Ale nie ználeźli / coby mu ucżynili ; ábowiem wƺyſtek lud záwieƺáł śię ná nim / ſłuchájąc go.
