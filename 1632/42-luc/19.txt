A <i>JEzus</i> wƺedƺy / ƺedł przez Jerycho.
A oto mąż którego zwano imieniem Zácheuƺ / który był Przełożony nád Celnikámi : á ten był bogáty.
Y żądał widźieć JEzuſá / coby zacż był. Lecż nie mógł przed ludem : bo był máłego wzroſtu.
A bieżawƺy naprzód / wſtąpił ná drzewo leśney figi / áby go ujrzał : bo tám tędy iść miał.
A gdy przyƺedł ná ono miejſce / pojrzawƺy JEzus w górę / ujrzał go / y rzekł do niego ; Zácheuƺu / zſtąp prędko ná dół : ábowiem dźiś muƺę zoſtáć w domu twojim.
Y zſtąpił prędko : y przyjął go z rádośćią.
A widząc to wƺyſcy / ƺemráli mówiąc : U cżłowieká grzeƺnego goſpodą ſtánął.
A ſtánąwƺy Zácheuƺ / rzekł do PAná : Oto połowicę májętnośći mojich dawam ubogim / PAnie : á jeſliżem kogo w cżym podƺedł / oddawam w cżwór naſób.
Y rzekł mu JEzus : Dźiś śię ſtáło zbáwienie domowi temu ; dla tego / że y on jeſt ſynem Abráhámowym.
Bo przyƺedł Syn cżłowiecży / áby ƺukał y záchował co było zginęło.
Tedo gdy oni ſłucháli / mówiąc dáliey / powiedźiał im podobieńſtwo ; dla tego że był bliſko od Jeruzalem / á iż oni mniemáli / że śię wnet króleſtwo Boże objáwić miáło.
Rzekł tedy ; Niektóry cżłowiek rodu zacnego / jácháł w dáleką krájinę / áby ſobie wźiął króleſtwo / y záśię śię wróćił.
A záwoławƺy dźieśiąći ſług ſwojich / dał im dźieśięć grzywien / y rzekł do nich : Hándlujćie áż przyjádę.
Lecż mieƺcżánie jego mieli go w nienawiśći / y wypráwili zá nim poſelſtwo / mówiąc ; Nie chcemy áby ten królował nád námi.
Y ſtáło śię gdy śię wróćił / wźiąwƺy króleſtwo / że rozkazał do śiebie záwołáć ſług onych / którym był dał pieniądze ; áby śię dowiedźiał co który hándlując zyſkał?
Tedy przyƺedł pierwƺy / mówiąc : Pánie / grzywná twojá dźieśięć grzywien urobiłá.
Y rzekł mu ; Dobrze ſługo dobry / iżeś był nád máłem wiernym / miejże władzą nád dźieśiąćią miaſt.
Przyƺedł y drugi / mówiąc ; Pánie / grzywná twojá pięć grzywien urobiłá.
Rzekł y temu : Y ty bądź nád piąćią miaſt.
A inƺy przyƺedł / mówiąc ; Pánie / oto grzywná twojá którąm miał zchowáną w chuſtce.
Bom śię ćiebie bał / żeś jeſt cżłowiek ſrogi : bierzeƺ / cżegoś nie położył : á żnieƺ / cżegoś nie śiał.
Tedy mu rzekł : Z uſt twojich ſądzę ćię / zły ſługo : Wiedźiałeś żem ja jeſt cżłowiek ſrogi / który biorę cżegom nie położył / á żnę cżegom nie śiał?
Przecżżeś tedy nie dał śrebrá mego do lichwiarzów? A ja przyƺedƺy / wźiąłbym je był y z lichwą.
Y rzekł tym którzy tuż ſtali : Weźmićie od niego tę grzywnę / á dajćie temu który ma dźieśięć grzywien.
Y rzekli mu : Pánie / mać dźieśięć grzywien.
Záprawdę powiádam wam / iż wƺelkiemu który ma / będźie dano : á od tego który nie ma / y to co ma będźie od niego odjęto.
Ale y nieprzyjaćioły moje / którzy nie chćieli ábym królował nád nimi przywiedźćie tu / á pobijćie przede mną.
A to powiedźiawƺy / ƺedł w przód / wſtępując do Jeruzalem.
Y ſtáło śię gdy śię przybliżył do Betfágie y Betániey ku górze którą zowią Oliwną / poſłał dwu z ucżniów ſwojich /
Mówiąc ; Idźćie do miáſtecżká które jeſt przećiwko wam / do którego wƺedƺy / znajdźiećie oślę uwiązáne / ná którym żaden cżłowiek nigdy nie śiedźiał : odwiązawƺy je / przywiedźćie.
A jeſliby was kto zpytał : Przecż <i>je</i> odwięzujećie? Ták mu powiećie : Przeto / że go PAN potrzebuje.
Odƺedƺy tedy ći którzy byli poſłáni / ználeźli jáko im był powiedźiał.
A gdy oni odwięzowáli ono oślę / rzekli pánowie jego do nich : Przecż odwięzujećie oślę?
A oni powiedźieli : PAN go potrzebuje.
Y przywiedli je do JEzuſá : á włożywƺy ƺáty ſwoje ná ono oślę / wſádźili JEzuſá <i>ná nie</i>.
A gdy on jáchał / ſłáli ƺáty ſwoje ná drodze.
A gdy śię już przybliżał tám gdźie śię zpuƺcżáją z góry Oliwney / pocżęło wƺyſtko mnóſtwo ucżniów rádując śię chwalić Bogá głoſem wielkim / ze wƺyſtkich cudów które widźieli /
Mówiąc : Błogoſłáwiony Król / który idźie w Imieniu PAńſkim : pokój ná niebie / á chwałá ná wyſokośćiách!
Ale niektórzy z Fáryzeuƺów z onego ludu rzekli do niego : Náucżyćielu / zgrom ucżnie ſwoje.
A on odpowiádájąc rzekł im : Powiádam wam / jeſliby ći milcżeli / wnet kámienie wołáć będźie.
A gdy śię przybliżył / ujrzawƺy miáſto / płákał nád nim / Mówiąc :
O gdybyś poznáło y ty / á zwłaƺcżá w ten to dźień twój co jeſt ku pokojowi twemu! Lecż <i>to</i> teraz zákryto od ocżu twojich.
Abowiem przydą ná ćię dni / gdy ćię obtocżą nieprzyjaćiele twoji wáłem / y oblęgą ćię / y zćiſną ćię zewƺąd.
Y zrównáją ćię z źiemią y dźieći twoje w tobie / á nie zoſtáwią w tobie kámieniá ná kámieniu : dla tego żeś nie poznáło cżáſu náwiedzenia twego.
A wƺedƺy do Kośćiołá / pocżął wyganiáć te którzy w nim przedawáli y kupowáli /
Mówiąc im : Nápiſano / Dom mój / dom modlitwy jeſt / á wyśćie go ucżynili jáſkinią zbójców.
Y ucżył ná káżdy dźień w Kośćiele. Lecż Przedniejƺy Kápłani y Náucżeni w piśmie y przedniejƺy z ludu / ƺukáli go ſtráćić :
Ale nie ználeźli coby <i>mu</i> ucżynili. Abowiem wƺyſtek lud záwieƺał śię ná nim / ſłuchájąc <i>go</i>.
