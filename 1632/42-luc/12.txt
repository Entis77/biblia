Miedzy tym gdy śię zgromádźiło wiele tyśięcy ludu / ták iż jedni po drugich deptáli / pocżął mówić do ucżniów ſwojich ; Naprzód ſtrzeżćie śię od kwáſu Fáryzejſkiego / który jeſt obłudá.
Boć nie jeſt nic ſkrytego / coby objáwiono być nie miáło : áni tájemnego cżegoby śię dowiedźieć nie miano.
Przetoż / cośćie mówili w ćiemnośći / ná świetle ſłyƺano będźie : á cośćie w ucho ƺeptáli w komorách / obwołano będźie ná dáchách.
A mówię wam przyjaćiołom mojim ; Nie bójćie śię tych / którzy ćiáło zábijáją / á potym nie máją coby więcey ucżynili.
Ale wam okażę kogo śię bać maćie ; Bójćie śię tego / który gdy zábije / ma moc wrzućić do piekielnego ogniá. Zájiſte powiádam wam / tego śię bójćie.
Izali piąći wróblików nie przedáją zá dwá pieniążki? Wƺákże jeden z nich nie jeſt w zápámiętániu przed oblicżem Bożym.
Owƺem y włoſy głowy wáƺey wƺyſtkie ſą policżone. Przetoż śię nie bójćie : nád wiele wróblików wyśćie zacniejƺy.
A mówię wam ; Wƺelki któryby mię wyznał przed ludźmi / y Syn cżłowiecży wyzna go przed Anjoły Bożymi.
Ale ktoby śię mnie záprzał przed ludźmi / záprą śię go przed Anjoły Bożymi.
Y káżdemu ktoby mówił ſłowo przećiwko Synowi cżłowiecżemu / będźie mu odpuƺcżono : ále temu ktoby przećiwko Duchowi Świętemu bluźnił / nie będźie odpuƺcżono.
A gdy was będą wodźić do bóżnic / y do Przełożonych / y do Zwierzchnośći ; nie troƺcżćie śię jáko y cobyśćie ku obronie odpowiedźieć / ábo cobyśćie mówić mieli.
Abowiem Duch Święty náucży was oneyże godźiny / cobyśćie mówić mieli.
Y rzekł mu niektóry z ludu : Náucżyćielu / rzecż brátu memu / áby śię ze mną podźielił dźiedźictwem.
Ale mu on rzekł : Cżłowiecże / któż mię poſtánowił ſędźią ábo dźielcą miedzy wámi?
Y rzekł do nich : Pátrzćie / á ſtrzeżćie śię łákomſtwá : gdyż nie w tym / że kto ma obfite májętnośći / żywot jego zależy.
Y powiedźiał im podobieńſtwo / mówiąc : Niektórego cżłowieká bogátego pole / obfity <i>urodzaj</i> przynioſło :
Y rozmyślał ſam w ſobie / mówiąc : Cóż ucżynię? Gdyż nie mam gdźiebym zgromádźił urodzáje moje?
Y rzekł ; To ucżynię : Rozwálę gumná moje / á więkƺe pobuduję / y zgromádzę tam wƺyſtkie urodzáje moje y dobrá moje :
Y rzekę do duƺy mojey : Duƺo / maƺ wiele dóbr złożonych ná wiele lat : odpocżyńże / jedz / pij / bądź dobrey myśli.
Ale mu rzekł Bóg : O głupi! tey nocy upomną śię duƺe twojey od ćiebie : á to coś nágotował cżyjeż będźie?
Tákći <i>jeſt</i> który ſobie ſkárbi / á nie jeſt w Bogu bogáty.
Y rzekł do ucżniów ſwojich : Dla tego powiádam wam / nietroƺcżćie śię o żywot wáƺ / cobyśćie jedli : áni o ćiáło / cżymbyśćie śię przyodźiewáli.
Zacniejƺy jeſt żywot / niż pokarm : á ćiáło / niż odźienie.
Przypátrzćie śię krukom / iż nie śieją áni żną / y nie máją ſpiżárni / áni gumná : á <i>wżdy</i> je Bóg żywi : Cżymżeśćie wy zacniejƺy niż ptacy?
Y któż z was troſkliwie myśląc / może przydáć do wzroſtu ſwego łokieć jeden.
Ponieważ tedy y namniejƺy rzecży nie przemożećie / cżemuż śię o inne troƺcżećie.
Przypátrzćie śię liliam jáko roſła : nie prácują / áni przędą. A powiádam wam / <i>że</i> áni Sálomon we wƺyſtkiey ſławie ſwojey / nie był ták przyodźiány jáko jedná z tych.
A jeſliże trawę / która dźiś jeſt ná polu / á jutro będźie w piec wrzucona / Bóg ták przyodźiewa ; jákoż dáleko więcey was / o máłowierni!
Wy tedy nie pytajćie śię cobyśćie jeść ábo cobyśćie pić mieli ; áni wyſoko latajćie <i>myślámi wáƺymi</i> :
Abowiem tego wƺyſtkiego narodowie tego świátá ƺukáją. Aleć Oćiec wáƺ wie / że tego potrzebujećie.
Owƺem ƺukajćie króleſtwá Bożego / á to wƺyſtko będźie wam przydano.
Nie bój śię / o málucżkie ſtadko! ábowiem śię upodobáło Ojcu wáƺemu dáć wam króleſtwo.
Przedawajćie májętnośći wáƺe / dawajćie jáłmużnę. Gotujćie ſobie mięƺki / które nie wiotƺeją ; ſkárb / którego nie ubywa w niebieśiech : gdźie złodźiey przyſtępu niema / áni mól pſuje.
Bo gdźie jeſt ſkarb wáƺ / tám będźie y ſerce wáƺe.
Niech będą przepáſáne biodrá wáƺe / y świece zápalone.
A wy <i>bądźćie</i> podobni ludźiom ocżekawájącym PAná ſwego / áżby śię wróćił z weſelá ; żeby / gdyby przyƺedł á zákołátał / wnet mu otworzyli.
Błogoſłáwieni oni ſłudzy / które gdy przydźie Pan / cżujące znajdźie. Záprawdę powiádam wam / iż śię przepáƺe / á poſádźi ich zá ſtół / á przechadzájąc śię będźie im ſłużył.
A jeſliby przyƺedł o wtórey ſtraży ; y o trzećiey ſtraży przyƺedłliby / á tákby <i>je</i> ználazł / błogoſłáwieni ſą oni ſłudzy.
A to wiedzćie / iż gdyby wiedźiał goſpodarz / o którey godźinie złodźiey ma przyść / wżdyby cżuł / á nie dopuśćiłby podkopáć domu ſwego.
Przetoż y wy bądźćie gotowi : bo o tey godźinie / o którey śię nie ſpodźiewaćie / Syn cżłowiecży przydźie.
Y rzekł mu Piotr : PAnie / do nas-że mówiƺ to podobieńſtwo cżyli do wƺyſtkich?
A PAn rzekł ; Któryż tedy jeſt wierny ƺáfarz y roſtropny / którego Pan poſtánowi nád cżeládźią ſwoją / áby im ná cżás wydawał obrok náznácżony.
Błogoſłáwiony jeſt on ſługá / którego gdyby przyƺedł Pan jego / znajdźie że ták cżyni.
Záprawdę wam powiádam / że go nád wƺyſtkimi dobrámi ſwojimi poſtánowi.
Ale jeſliby rzekł on ſługá w ſercu ſwojim ; Odwłacża Pan mój z przyśćiem <i>ſwojim</i> : y pocżąłby bić ſługi y ſłużebnice / á jeść / pić / y opijáć śię :
Przydźie Pan ſługi onego / dniá którego śię nie ſpodźieje ; y godźiny / którey nie wzwie / y odłącży go ; á cżęść jego położy z niewiernymi.
Ten záśię ſługá któryby znał wolą Páná ſwego / á nie był gotowym / áni cżynił według woli jego / wielce będźie karány.
Ale który nie znał / á cżynił rzecży godne karánia / mniey plag odnieśie : <i>A</i> od káżdego komu wiele dano / wiele śię od niego upomináć będą ; á komu się wiele powierzono / więcey będą chćieć od niego.
Przyƺedłem ábym ogień puśćił ná źiemię / y cżegoż chcę / jeſli już gore?
Aleć mam być chrztem ochrzcżony : á jákom jeſt śćiśniony / póki śię <i>to</i> nie wykona?
Mniemaćie ábym przyƺedł pokój dawáć ná źiemię? Bynamniey / powiádam wam ; ále rozerwánie.
Abowiem od tego cżáſu będźie ich pięć w domu jednym rozerwánych ; trzej przećiwko dwiemá / á dwá przećiwko trzemá.
Powſtánie oćiec przećiwko ſynowi / á ſyn przećiwko ojcu : mátká przećiwko córce / á córká przećiwko mátce : świekrá przećiwko ſynowey ſwojey / á ſynowa przećiwko świekrze ſwojey.
Mówił też y do ludu : Gdy widźićie obłok wſchodzący od Zachodu / záraz mówićie ; Przychodźi gwałtowny deƺcż : y ták bywa.
A gdy wiátr wiejący od Południá / mówićie ; Gorąco będźie : y bywa <i>ták</i>.
Obłudnicy! poſtáwę niebá y źiemie rozeznawáć umiećie / á tego cżáſu jákoż nie poznawaćie?
Przecżże y ſámi przez śię nie ſądźićie co jeſt ſpráwiedliwego?
Gdy tedy idźieƺ z przećiwnikiem twojim przed Urząd / ſtárajże śię w drodze / ábyś był wolen : by ćię ſnadź nie poćiągnął przed Sędźiego / á Sędźiaby ćię podał ceklarzowi / á ceklarzby ćię wrzućił do więźienia.
Powiádamći ; Nie wynidźieƺ z támtąd / pókibyś nie oddał do oſtátniego pieniążká.
