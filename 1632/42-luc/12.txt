Między tem / gdy śię zgromádziło wiele tyśięcy ludu / ták iż jedni po drugich deptáli / pocżął mówić do ucżniów ſwoich : Náprzód ſtrzeżćie śię kwáſu Fáryzejſkiego / który jeſt obłudá.
Boć nie jeſt nic ſkrytego / coby objáwione być nie miáło / áni tájemnego cżegoby śię dowiedzieć nie miáno.
Przetoż / cośćie mówili w ćiemnośći / ná świetle ſłyƺáne będzie / á cośćie w ucho ƺeptáli w komorách / obwołáne będzie ná dáchách.
A mówię wám przyjáćiołom moim : Nie bójćie śię tych / którzy ćiáło zábijáją / á potem nie máją / coby więcej ucżynili.
Ale wám okáżę / kogo śię báć máćie : Bójćie śię tego / który / gdy zábije / má moc wrzućić do piekielnego ogniá ; záiſte powiádám wám / tego śię bójćie.
Izáli pięćiu wróblików nie ſprzedáją zá dwá pieniążki? Wƺákże jeden z nich nie jeſt w zápámiętániu przed oblicżem Bożem.
Owƺem y włoſy głowy wáƺej wƺyſtkie ſą policżone. Przetoż śię nie bójćie / nád wiele wróblików wyśćie zácniejśi.
A mówię wám : Wƺeláki / któryby mię wyznáł przed ludźmi / y Syn cżłowiecży wyzná go przed Anioły Bożymi.
Ale ktoby śię mię záprzáł przed ludźmi / záprę śię go przed Anioły Bożymi.
Y káżdemu / ktoby mówił ſłowo przećiwko Synowi cżłowiecżemu / będzie mu odpuƺcżone : ále temu / ktoby przećiwko Duchowi Świętemu bluźnił / nie będzie odpuƺcżone.
A gdy wáſ będą wodzić do bóżnic / y do przełożonych / y do zwierzchnośći / nie troƺcżćie śię / jáko y cobyśćie ku obronie odpowiedzieć / álbo cobyśćie mówić mieli.
Abowiem Duch Święty náucży wáſ onejże godziny / cobyśćie mówić mieli.
Y rzekł mu niektóry z ludu : Náucżyćielu! rzecż brátu memu / áby śię ze mną podzielił dziedzictwem.
Ale mu on rzekł : Cżłowiecże! któż mię poſtánowił ſędzią álbo dzielcą między wámi?
Y rzekł do nich : Pátrzćie / á ſtrzeżćie śię łákomſtwá / gdyż nie w tem / że kto má obfite májętnośći / żywot jego záleży.
Y powiedziáł im podobieńſtwo / mówiąc : Niektórego cżłowieká bogátego pole obfity urodzáj przynioſło.
Y rozmyśláł ſám w ſobie / mówiąc : Cóż ucżynię / gdyż nie mám / gdziebym zgromádził urodzáje moje?
Y rzekł : To ucżynię : Rozwálę gumná moje / á więkƺe pobuduję / y zgromádzę tám wƺyſtkie urodzáje moje y dobrá moje ;
Y rzekę do duƺy mojej : Duƺo! máƺ wiele dóbr złożonych ná wiele lát ; odpocżnijże / jedz / pij / bądź dobrej myśli.
Ale mu rzekł Bóg : O głupi / tej nocy upomną śię duƺy twojej od ćiebie / á to / coś nágotowáł / cżyjeż będzie?
Tákći jeſt / który ſobie ſkárbi / á nie jeſt w Bogu bogáty.
Y rzekł do ucżniów ſwoich : Dlátego powiádám wám / nie troƺcżćie śię o żywot wáƺ / cobyśćie jedli / áni o ćiáło / cżembyśćie śię przyodziewáli.
Zácniejƺy jeſt żywot / niż pokárm / á ćiáło / niż odzienie.
Przypátrzćie śię krukom / iż nie śieją áni żną / y nie máją ſpiżárni / áni gumná / á wżdy je Bóg żywi ; cżemżeśćie wy zácniejśi niż ptácy?
Y któż z wáſ troſkliwie myśląc / może przydáć do wzroſtu ſwego łokieć jeden?
Poniewáż tedy y nájmniejƺej rzecży nie przemożećie / cżemuż śię o inne troƺcżyćie?
Przypátrzćie śię lilijom / jáko roſną / nie prácują / áni przędą ; á powiádám wám / że áni Sálomon we wƺyſtkiej ſłáwie ſwojej nie był ták przyodziány / jáko jedná z tych.
A jeźliże tráwę / którá dziś jeſt ná polu / á jutro będzie w piec wrzuconá / Bóg ták przyodziewá / jákoż dáleko więcej wáſ / o máłowierni!
Wy tedy nie pytájćie śię / cobyśćie jeść / álbo cobyśćie pić mieli / áni wyſoko látájćie myślámi wáƺemi.
Abowiem tego wƺyſtkiego národy tego świátá ƺukáją ; áleć Ojćiec wáƺ wie / że tego potrzebujećie.
Owƺem ƺukájćie króleſtwá Bożego / á to wƺyſtko będzie wám przydáne.
Nie bój śię / o málucżkie ſtádko! ábowiem śię upodobáło Ojcu wáƺemu / dáć wám króleſtwo.
Sprzedáwájćie májętnośći wáƺe / á dáwájćie jáłmużnę ; gotujćie ſobie mieƺki / które nie wiotƺeją / ſkárb / którego nie ubywá w niebieśiech / gdzie złodziej przyſtępu nie má / áni mól pſuje.
Bo gdzie jeſt ſkárb wáƺ / tám będzie y ſerce wáƺe.
Niech będą przepáſáne biodrá wáƺe / y świece zápálone.
A wy bądźćie podobni ludziom ocżekującym páná ſwego / áżby śię wróćił z weſelá / żeby / gdyby przyƺedł á zákołátáł / wnet mu otworzyli.
Błogoſłáwieni oni ſłudzy / których / gdy przyjdzie pán / cżujących znájdzie ; zápráwdę powiádám wám / iż śię przepáƺe / á poſádzi ich zá ſtół / á przechádzájąc śię / będzie im ſłużył.
A jeźliby przyƺedł o wtórej ſtráży / y o trzećiej ſtráży przyƺedłliby / á tákby ich ználázł / błogoſłáwieni ſą oni ſłudzy.
A to wiedzćie / iżby / gdyby wiedziáł goſpodárz / o której godzinie złodziej má przyjść / wżdyby cżuł / á nie dopuśćiłby podkopáć domu ſwego.
Przetoż y wy bądźćie gotowi ; bo o tej godzinie / o której śię nie ſpodziewáćie / Syn cżłowiecży przyjdzie.
Y rzekł mu Piotr : Pánie! do náſże mówiƺ to podobieńſtwo / cżyli do wƺyſtkich?
A Pán rzekł : Któryż tedy jeſt wierny ƺáfárz y roztropny / którego Pán poſtánowi nád cżeládzią ſwoją / áby im ná cżáſ wydáwáł obrok náznácżony?
Błogoſłáwiony jeſt on ſługá / którego / gdyby przyƺedł pán jego / znájdzie / że ták cżyni ;
Zápráwdę wám powiádám / że go nád wƺyſtkiemi dobrámi ſwojimi poſtánowi.
Ale jeźliby rzekł on ſługá w ſercu ſwojim : Odwłácżá pán mój z przyjśćiem ſwojim / y pocżąłby bić ſługi y ſłużebnice / á jeść / pić y opijáć śię ;
Przyjdzie pán ſługi onego / dniá / którego śię nie ſpodzieje / y godziny / której nie wie / y odłącży go / á cżęść jego położy z niewiernymi.
Ten záśię ſługá / któryby znáł wolę páná ſwego / á nie był gotowym / áni cżynił według woli jego / wielce będzie kárány ;
Ale który nie znáł / á cżynił rzecży godne kárániá / mniej plág odnieśie ; á od káżdego / komu wiele dáno / wiele śię od niego upomináć będą : á komu wiele powierzono / więcej będą chćieć od niego.
Przyƺedłem / ábym ogień puśćił ná ziemię / y cżegoż chcę / jeźli już gore?
Aleć mám być chrztem ochrzcżony ; á jákom jeſt śćiśniony / póki śię to nie wykoná.
Mniemáćie / ábym przyƺedł / pokój dáwáć ná ziemię? Bynájmniej / powiádám wám / ále rozerwánie.
Abowiem od tego cżáſu będzie ich pięć w domu jednym rozerwánych / trzej przećiwko dwom / á dwáj przećiwko trzem.
Powſtánie ojćiec przećiwko ſynowi / á ſyn przećiwko ojcu / mátká przećiwko córce / á córká przećiwko mátce / świekrá przećiwko ſynowej ſwojej / á ſynowá przećiwko świekrze ſwojej.
Mówił też y do ludu : Gdy widzićie obłok wſchodzący od záchodu / záráz mówićie : Przychodzi gwáłtowny deƺcż ; y ták bywá.
A gdy wiátr wiejący od południá / mówićie : Gorąco będzie ; y bywá ták.
Obłudnicy! poſtáwę niebá y ziemi rozeznáwáć umiećie / á tego cżáſu jákoż nie poznáwáćie?
Przecżże y ſámi przez śię nie ſądzićie / co jeſt ſpráwiedliwego?
Gdy tedy idzieƺ z przećiwnikiem twoim przed urząd / ſtárájże śię w drodze / ábyś był wolen / by ćię ſnáć nie poćiągnął przed ſędziego / á ſędziáby ćię podáł ceklárzowi / á ceklárzby ćię wrzućił do więzieniá.
Powiádám ći : Nie wynijdzieƺ ſtámtąd / pókibyś nie oddáł do oſtátniego pieniążká.
