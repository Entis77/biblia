A zwoławƺy <i>JEzus</i> dwunáści ucżniów ſwojich / dał im moc y władzą nád wƺyſtkimi dyabły / y áby uzdrawiáli choroby.
Y rozeſłał je żeby kazáli króleſtwo Boże / y uzdrawiáli niemocne.
Tedy rzekł do nich ; Nie bierżćie nic ná drogę / áni laſki / áni tájſtry / áni chlebá / áni pieniędzy / áni dwu ſukien miejćie.
A do któregokolwiek domu wnidźiećie / támże zoſtańćie / y z támtąd wynidźćie.
A którzybykolwiek was nie przyjęli / wychodząc z miáſtá onego / y proch z nóg wáƺych otrząśnićie / ná świádectwo przećiwko nim.
Wyƺedƺy tedy obchodźili wƺyſtkie miáſtecżká / opowiádájąc Ewángelią / á wƺędźie <i>chore</i> uzdrawiájąc.
Y uſłyƺał Herod Tetrárchá o wƺyſtkim co śię dźiało od niego : y był wątpliwym / dla tego że niektórzy powiádáli / Iż Jan zmartwychwſtał.
A niektórzy záśię ; Iż śię Eliaƺ ukazał : á drudzy / iż Prorok jeden z onych ſtárych <i>zmartwych</i>wſtał.
Tedy rzekł Herod ; Janámći ja śćiął : Któż wżdy ten jeſt o którym ja tákie rzecży ſłyƺę? y prágnął go widźieć.
A wróćiwƺy śię Apoſtołowie / powiádáli mu cokolwiek cżynili. A on wźiąwƺy je z ſobą / uſtąpił oſobno ná miejſce puſte przy mieśćie / które zowią Betſájdá.
Cżego gdy śię lud dowiedźiał / ƺedł zá nim : á przyjąwƺy je / mówił im o króleſtwie Bożym ; á te którzy uzdrowienia potrzebowáli / uzdrawiał.
A gdy śię dźień pocżął ſkłaniáć <i>ku wiecżorowi,</i> przyſtąpiwƺy oni dwánaście / rzekli mu ; Rozpuść ten lud / áby odƺedƺy do miáſtecżek okolicżnych / y do wśi / y do goſpód / ználeźli żywność : boſmy tu ná miejſcu puſtym.
Ale on rzekł do nich : Dajćież wy im jeść. A oni powiedźieli ; Nie mamy więcey tylko pięć chlebów y dwie rybie : oprócż żebyſmy ƺli / á kupili ná ten wƺyſtek lud żywnośći.
Abowiem było mężów około piąći tyśięcy. Y rzekł do ucżniów ſwojich ; Rozkażćie im uśieść w káżdym rzędźie po pięćdźieśiąt.
Y ucżynili ták. Y uśiedli wƺyſcy.
A <i>on</i> wźiąwƺy onych pięć chlebów / y one dwie rybie / wejrzawƺy w niebo / błogoſłáwił im / y łamał / y dawał ucżniom áby kłádli przed on lud.
Y jedli / y náſyceni ſą wƺyſcy : y zebrano co im zbyło ułomków / dwánaśćie koƺów.
Y ſtáło śię gdy śię on ſam oſobno modlił / <i>że</i> z nim byli ucżniowie ; y pytał ich mówiąc : Kimże mię być powiádáją ludźie?
A oni odpowiádájąc rzekli ; Janem Chrzćićielem ; á drudzy Eliaƺem ; á drudzy <i>mówią</i> iż Prorok niektóry z onych ſtárych <i>zmartwych</i>wſtał.
Y rzekł im : A wy kim mię być powiádaćie? A odpowiádájąc Piotr / rzekł ; CHryſtuſem onym Bożym.
Ale on przygroźiwƺy <i>im,</i> rozkazał áby tego nikomu nie powiádáli :
Mówiąc ; że Syn cżłowiecży muśi wiele ćierpieć / á być odrzuconym od Stárƺych ludu / y od Przedniejƺych kápłanów / y od Náucżonych w piśmie / y być zábitym / á trzećiego dniá zmartwychwſtáć.
Y mówił do wƺyſtkich ; Kto chce zá mną iść / niech záprzy ſámego śiebie / á niech bierze krzyż ſwój ná káżdy dźień / y náśláduje mię.
Abowiem ktobykolwiek chćiał záchowáć duƺę ſwoję / ſtráći ją ; á ktobykolwiek ſtráćił duƺę ſwą dla mnie / ten ją záchowa.
Abowiem cóż pomoże cżłowiekowi / choćby wƺyſtek świát pozyſkał / gdyby ſámego śiebie ſtráćił álbo ſobie uƺkodźił.
Abowiem ktoby śię kolwiek wſtydźił zá mię y zá ſłowá moje / zá tego śię Syn cżłowiecży wſtydźić będźie / gdy przydźie w chwale ſwey / y w Ojcowſkiey / y świętych Anjołów.
Aleć wam powiádam prawdźiwie : Są niektórzy z tych co tu ſtoją / którzy nie ukuƺą śmierći / áż oglądáją króleſtwo Boże.
Y ſtáło śię po tych mowách / jákoby po ośmi dniách / że wźiąwƺy z ſobą Piotrá / y Janá y Jákubá / wſtąpił ná górę áby śię modlił.
A gdy śię modlił / ſtał śię inákƺy kƺtałt oblicżá jego ; y ƺáty jego <i>ſtały śię</i> białe y świetne.
A oto dwá mężowie rozmawiáli z nim / á ći byli Mojzeƺ y Eliaƺ.
Którzy pokazawƺy śię w ſławie / powiádáli o jego śmierći / którą miał podſtąpić w Jeruzalemie.
A Piotr / y ći którzy byli z nim / obćiążeni byli ſnem : á ocućiwƺy śię / ujrzeli chwałę jego / y onych dwu mężów którzy z nim ſtali.
Y ſtáło śię / gdy oni odeƺli od niego / rzekł Piotr do JEzuſá ; Miſtrzu / dobrze nam tu być : Przetoż ucżyńmy trzy namioty ; tobie jeden / y Mojzeƺowi jeden / y Eliaƺowi jeden : nie wiedząc co mówił.
A gdy on to mówił / ſtał śię obłok / y záćienił je : y bali śię gdy oni wchodźili w obłok.
Y ſtał śię głos z obłoku mówiący ; Ten jeſt Syn mój miły / tego ſłuchajćie.
A gdy śię ſtał on głos / ználeźiony jeſt ſam JEzus. A oni milcżeli / y niepowiádáli w one dni nikomu nic z tego co widźieli.
Y ſtáło śię názájutrz / gdy oni zſtąpili z góry / <i>że</i> mu lud wielki zábieżał.
A oto mąż z onego ludu záwołał mówiąc : Náucżyćielu / proƺę ćię wejrzy ná ſyná mego ; boć jedynego mam.
A oto duch <i>zły</i> popada go / á záraz krzycży / á on rozdźiera go śliniącego śię : á zá ledwie odchodźi od niego ſkruƺywƺy go.
Y prośiłem ucżniów twojich áby go wygnáli ; ále niemogli.
Tedy JEzus odpowiádájąc / rzekł : O rodzáju niewierny y przewrotny! Dokądże z wámi będę? Y <i>dokądże</i> was ćierpieć będę? Przywiedź tu ſyná twego.
A w tym gdy on przychodźił / rozdárł go dyabeł y roztárgał. Ale JEzus zgromił duchá niecżyſtego / y uzdrowił młodźieńcá / y oddał go ojcu jego.
Y zdumieli śię wƺyſcy nád wielmożnośćią Bożą. A gdy śię wƺyſcy dźiwowáli wƺyſtkim rzecżam które cżynił JEzus / rzekł do ucżniów ſwojich :
Składajćie wy do uƺu wáƺych ſłowá te : Abowiem Syn cżłowiecży ma być wydány w ręce ludzkie.
Lecż oni nie zrozumieli ſłowá tego / y było zákryte od nich / że go pojąć nie mogli ; y nie śmieli go pytáć o to ſłowo.
Y wƺcżęłá śię gadká miedzy nimi ; Ktoby z nich był nawiękƺym.
A JEzus widząc myśl ſercá ich / wźiąwƺy dźiećię / poſtáwił je podle śiebie.
Y rzekł im : Ktobykolwiek przyjął to dźiećiątko w Imieniu mojim / mnie przyjmuje ; á ktobykolwiek mnie przyjął / przyjmuje onego który mię poſłał. Abowiem kto jeſt namniejƺy miedzy wƺyſtkimi wámi / tenći będźie wielkim.
A Jan odpowiádájąc rzekł ; Miſtrzu / widźieliſmy niektórego w Imieniu twojim dyabły wyganiájącego / y zábraniáliſmy mu ; przeto / że <i>zá tobą</i> z námi nie chodźi.
Y rzekł do niego JEzus : Nie zábraniajćie <i>mu</i> : bo kto nie jeſt przećiwko nam / zá námi jeſt.
Y ſtáło śię / gdy śię wypełniły dni / áby był wźięty wzgórę / że śię on ná to udał / áby ƺedł do Jeruzalem.
Tedy poſłał poſły przed ſobą : Którzy idąc weƺli do miáſtecżká Sámárytáńſkiego / áby mu nágotowáli <i>goſpodę</i>.
Lecż oni go nieprzyjęli / przeto że oblicże jego było obrócone do Jeruzalem.
A widząc to ucżniowie jego Jákub y Jan / rzekli : PAnie / chceƺże iż rzecżemy áby ogień zſtąpił z niebá / y ſpalił je : jáko y Eliaƺ ucżynił?
Ale <i>JEzus</i> obróćiwƺy śię zgromił je / y rzekł : Nie wiećie wy jákiegośćie duchá.
Abowiem Syn cżłowiecży nie przyƺedł zátracáć duƺ ludzkich / ále záchowáć. Y ƺli do inƺego miáſtecżká.
Y ſtáło śię gdy oni ƺli / <i>że</i> w drodze rzekł niektóry do niego : Pójdę zá tobą gdźiekolwiek pójdźieƺ PAnie.
A JEzus mu rzekł : Liƺki máją jámy / y ptaƺkowie niebieſcy gniazdá : ále Syn cżłowiecży nie ma gdźieby głowę ſkłonił.
Y rzekł do drugiego : Pódź zá mną. Ale on rzekł : PAnie / dopuść mi pierwey odejść y pogrześć Ojcá mego.
Ale mu JEzus rzekł : Niechaj umárli grzebią umárłe ſwoje : á ty poƺedƺy / opowiádaj króleſtwo Boże.
Rzekł też y drugi : Pójdę zá tobą PAnie : ále mi pierwey dopuść pożegnáć śię z tymi którzy ſą w domu mojim.
Rzekł do niego JEzus : Żaden któryby przyłożył rękę ſwoję do pługá / á oglądałby śię ná zad / nie jeſt ſpoſobny do króleſtwá Bożego.
