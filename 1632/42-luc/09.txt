A zwołáwƺy Jezuſ dwunáſtu ucżniów ſwoich / dáł im moc y włádzę nád wƺyſtkimi dyjábły / y áby uzdráwiáli choroby.
Y rozeſłáł je / żeby kázáli króleſtwo Boże / y uzdráwiáli niemocne.
Tedy rzekł do nich : Nie bierzćie nic ná drogę / áni láſki / áni táiſtry / áni chlebá / áni pieniędzy / áni dwóch ſukien miejćie.
A do któregokolwiek domu wnijdziećie / támże zoſtáńćie / y ſtámtąd wynijdźćie.
A którzybykolwiek wáſ nie przyjęli / wychodząc z miáſtá onego / y proch z nóg wáƺych otrząśnijćie ná świádectwo przećiwko nim.
Wyƺedłƺy tedy / obchodzili wƺyſtkie miáſtecżká / opowiádájąc Ewángieliję / á wƺędzie chore uzdráwiájąc.
Y uſłyƺáł Herod / Tetrárchá / o wƺyſtkiem / co śię dziáło od niego / y był wątpliwym dlátego / że niektórzy powiádáli / iż Ján zmártwychwſtáł.
A niektórzy záśię : Iż śię Elijáƺ ukázáł ; á drudzy / iż prorok jeden z onych ſtárych zmártwychwſtáł.
Tedy rzekł Herod : Jánámći já śćiął ; któż wżdy ten jeſt / o którym já tákie rzecży ſłyƺę? y prágnął go widzieć.
A wróćiwƺy śię Apoſtołowie / powiádáli mu / cokolwiek cżynili. A on wziąwƺy je z ſobą / uſtąpił oſobno ná miejſce puſte przy mieśćie / które zowią Betſáidá.
Cżego gdy śię lud dowiedziáł / ƺedł zá nim ; á przyjąwƺy je / mówił im o króleſtwie Bożem ; á te / którzy uzdrowieniá potrzebowáli / uzdráwiáł.
A gdy śię dzień pocżął ſkłániáć ku wiecżorowi / przyſtąpiwƺy oni dwunáſtu / rzekli mu : Rozpuść ten lud / áby odƺedłƺy do miáſtecżek okolicżnych / y do wśi / y do goſpód / ználeźli żywność ; bośmy tu ná miejſcu puſtem.
Ale on rzekł do nich : Dájćież wy im jeść. A oni powiedzieli : Nie mámy więcej tylko pięć chlebów y dwie ryby / oprócż żebyśmy ƺli / á kupili / ná ten wƺyſtek lud żywnośći.
Abowiem było mężów około pięćiu tyśięcy. Y rzekł do ucżniów ſwoich : Rozkáżćie im uśiąść w káżdym rzędzie po pięćdzieśiąt.
Y ucżynili ták / y uśiedli wƺyſcy.
A on wziąwƺy onych pięć chlebów y one dwie ryby / wejrzáwƺy w niebo / błogoſłáwił im / y łámáł y dáwáł ucżniom / áby kłádli przed on lud.
Y jedli / y náſyceni ſą wƺyſcy ; y zebráno / co im zbyło z ułomków / dwánáśćie koƺów.
Y ſtáło śię / gdy śię on ſám oſobno modlił / że z nim byli ucżniowie ; y pytáł ich mówiąc : Kimże mię być powiádáją ludzie?
A oni odpowiádájąc rzekli : Jánem Chrzćićielem / á drudzy Elijáƺem / á drudzy mówią / iż prorok niektóry z onych ſtárych zmártwychwſtáł.
Y rzekł im : A wy kim mię być powiádáćie? A odpowiádájąc Piotr rzekł : Chryſtuſem / onym Bożym.
Ale on przygroziwƺy im / rozkázáł / áby tego nikomu nie powiádáli /
Mówiąc : Że Syn cżłowiecży muśi wiele ćierpieć / á być odrzuconym od ſtárƺych ludu y od przedniejƺych kápłánów y od náucżonych w Piśmie / y być zábitym / á trzećiego dniá zmártwychwſtáć.
Y mówił do wƺyſtkich : Kto chce zá mną iść / niech záprze ſámego śiebie / á niech bierze krzyż ſwój ná káżdy dzień / y náśláduje mię.
Abowiem ktobykolwiek chćiáł záchowáć duƺę ſwoję / ſtráći ją ; á ktobykolwiek ſtráćił duƺę ſwą dlá mnie / ten ją záchowá.
Abowiem cóż pomoże cżłowiekowi / choćby wƺyſtek świát pozyſkáł / gdyby ſámego śiebie ſtráćił / álbo ſobie uƺkodził.
Abowiem ktoby śię kolwiek wſtydził zá mię y zá ſłowá moje / zá tego śię Syn cżłowiecży wſtydzić będzie / gdy przyjdzie w chwále ſwej y w ojcowſkiej y świętych Aniołów.
Aleć wám powiádám práwdziwie : Są niektórzy z tych / co tu ſtoją / którzy nie ukuƺą śmierći / áż oglądáją króleſtwo Boże.
Y ſtáło śię po tych mowách / jákoby po ośmiu dniách / że wziąwƺy z ſobą Piotrá y Jáná y Jákóbá / wſtąpił ná górę / áby śię modlił.
A gdy śię modlił / ſtáł śię inákƺy kƺtáłt oblicżá jego / y ƺáty jego ſtáły śię biáłe y świetne.
A oto dwáj mężowie rozmáwiáli z nim / á ći byli Mojżeƺ y Elijáƺ ;
Którzy pokázáwƺy śię w ſłáwie / powiádáli o jego śmierći / którą miáł podſtąpić w Jeruzálemie.
A Piotr y ći / którzy byli z nim / obćiążeni byli ſnem / á ocućiwƺy śię / ujrzeli chwáłę jego y onych dwóch mężów / którzy z nim ſtáli.
Y ſtáło śię / gdy oni odeƺli od niego / rzekł Piotr do Jezuſá : Miſtrzu! dobrze nám tu być ; przetoż ucżyńmy trzy námioty / tobie jeden y Mojżeƺowi jeden y Elijáƺowi jeden ; nie wiedząc / co mówił.
A gdy on to mówił / ſtáł śię obłok / y záćienił je ; y báli śię / gdy oni wchodzili w obłok.
Y ſtáł śię głoſ z obłoku mówiący : Ten jeſt Syn mój miły / tego ſłuchájćie.
A gdy śię ſtáł on głoſ / ználeziony jeſt ſám Jezuſ. A oni milcżeli / y nie powiádáli w one dni nikomu nic z tego / co widzieli.
Y ſtáło śię názájutrz / gdy oni zſtąpili z góry / że mu lud wielki zábieżáł.
A oto mąż z onego ludu záwołáł / mówiąc : Náucżyćielu! proƺę ćię / wejrzyj ná ſyná mego ; boć jedynego mám.
A oto duch zły popádá go / á záráz krzycży / á on rozdzierá go / śliniącego śię / á záledwie odchodzi od niego / ſkruƺywƺy go.
Y prośiłem ucżniów twoich / áby go wygnáli ; ále nie mogli.
Tedy Jezuſ odpowiádájąc rzekł : O rodzáju niewierny y przewrotny! dokądże z wámi będę / y dokądże wáſ ćierpieć będę? Przywiedź tu ſyná twego.
A wtem / gdy on przychodził / rozdárł go dyjábeł y roztárgáł ; ále Jezuſ zgromił duchá niecżyſtego y uzdrowił młodzieńcá / y oddáł go ojcu jego.
Y zdumieli śię wƺyſcy nád wielmożnośćią Bożą. A gdy śię wƺyſcy dziwowáli wƺyſtkim rzecżom / które cżynił Jezuſ / rzekł do ucżniów ſwoich :
Skłádájćie wy do uƺów wáƺych ſłowá te : ábowiem Syn cżłowiecży má być wydány w ręce ludzkie.
Lecż oni nie zrozumieli ſłowá tego / y było zákryte od nich / że go pojąć nie mogli / y nie śmieli go pytáć o to ſłowo.
Y wƺcżęłá śię gádká między nimi / ktoby z nich był nájwiękƺym.
A Jezuſ widząc myśl ſercá ich / wziąwƺy dziećię / poſtáwił je podle śiebie /
Y rzekł im : Ktobykolwiek przyjął to dziećiątko w imieniu mojem / mnie przyjmuje ; á ktobykolwiek mnie przyjął / przyjmuje onego / który mię poſłáł : ábowiem kto jeſt nájmniejƺy między wƺyſtkimi wámi / tenći będzie wielkim.
A Ján odpowiádájąc / rzekł : Miſtrzu! widzieliśmy niektórego w imieniu twojim dyjábły wygániájącego / y zábrániáliśmy mu / przeto / że zá tobą z námi nie chodzi.
Y rzekł do niego Jezuſ : Nie zábrániájćie mu ; bo kto nie jeſt przećiwko nám / zá námi jeſt.
Y ſtáło śię / gdy śię wypełniły dni / áby był wzięty w górę / że śię on ná to udáł / áby ƺedł do Jeruzálemu.
Tedy poſłáł poſły przed ſobą ; którzy idąc weƺli do miáſtecżká Sámárytáńſkiego / áby mu nágotowáli goſpodę.
Lecż oni go nieprzyjęli / przeto / że oblicże jego obrócone było do Jeruzálemu.
A widząc to ucżniowie jego / Jákób y Ján / rzekli : Pánie! chceƺże iż rzecżemy / áby ogień zſtąpił z niebá / y ſpálił je / jáko y Elijáƺ ucżynił?
Ale Jezuſ obróćiwƺy śię / zgromił je / y rzekł : Nie wiećie wy jákiegośćie duchá.
Abowiem Syn cżłowiecży nie przyƺedł / zátrácáć duƺ ludzkich / ále záchowáć. Y ƺli do inƺego miáſtecżká.
Y ſtáło śię / gdy oni ƺli / że w drodze rzekł niektóry do niego : Pójdę zá tobą / gdziekolwiek pójdzieƺ / Pánie!
A Jezuſ mu rzekł : Liƺki máją jámy / y ptáƺki niebieſkie gniázdá ; ále Syn cżłowiecży nie má / gdzieby głowę ſkłonił.
Y rzekł do drugiego : Pójdź zá mną! Ale on rzekł : Pánie! dopuść mi pierwej odejść y pogrześć ojcá mego.
Ale mu Jezuſ rzekł : Niecháj umárli grzebią umárłych ſwoich ; á ty poƺedłƺy / opowiádáj króleſtwo Boże.
Rzekł też y drugi : Pójdę zá tobą / Pánie! ále mi pierwej dopuść pożegnáć śię z tymi / którzy ſą w domu moim.
Rzekł do niego Jezuſ : Żáden / któryby przyłożył rękę ſwoję do pługá / á oglądáłby śię názád / nie jeſt ſpoſobny do króleſtwá Bożego.
