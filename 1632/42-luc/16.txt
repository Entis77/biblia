Mówił też y do ucżniów ſwoich : Cżłowiek niektóry był bogáty / który miáł ƺáfárzá / á ten był odnieśiony do niego / jákoby rozpráƺáł dobrá jego.
A záwołáwƺy go / rzekł mu : Cóż to ſłyƺę o tobie? Oddáj licżbę z ƺáfárſtwá twego ; ábowiem już więcej nie będźieƺ mógł ƺáfowáć.
Y rzekł on ƺáfárz ſám w ſobie : Cóż ucżynię / gdyż pán mój odbierá odemnie ƺáfárſtwo? Kopáć nie mogę / żebráć śię wſtydzę.
Wiem / co ucżynię / że gdy będę złożony z ƺáfárſtwá / przyjmą mię do domów ſwoich.
Záwołáwƺy tedy do śiebie káżdego z dłużników páná ſwego rzekł pierwƺemu : Wieleś winien pánu memu?
A on rzekł : Sto báreł oliwy. Y rzekł mu : Weźmij zápiſ twój / á śiádłƺy prędko / nápiƺ pięćdźieśiąt.
Potem drugiemu rzekł : A tyś wiele winien? A on mu rzekł : Sto korcy pƺenicy. Y rzekł mu : Weźmij zápiſ twój / á nápiƺ ośmdźieśiąt.
Y pochwálił pán ƺáfárzá nieſpráwiedliwego / iż roztropnie ucżynił ; bo ſynowie tego świátá roztropniejśi ſą nád ſyny świátłośći w rodzáju ſwoim.
Y Jáć wám powiádám : Cżyńćie ſobie przyjáćioły z mámmony nieſpráwiedliwośći / áby / gdy uſtániećie / przyjęli wáſ do wiecżnych przybytków.
Kto wierny jeſt w máłem / y w wielu wiernym jeſt ; á kto w máłem nieſpráwiedliwy / y w wielu nieſpráwiedliwym jeſt.
Poniewáżeśćie tedy w mámmonie nieſpráwiedliwej wiernymi nie byli / práwdźiwego któż wám powierzy?
A jeźliśćie w cudzem wiernymi nie byli / któż wám dá / co wáƺego jeſt?
Żáden ſługá nie może dwom pánom ſłużyć / gdyż álbo jednego będźie miáł w nienáwiśći / á drugiego będźie miłowáł ; álbo śię jednego trzymáć będźie / á drugim pogárdźi ; nie możećie Bogu ſłużyć y mámmonie.
A ſłucháli tego wƺyſtkiego y Fáryzeuƺowie / którzy byli łákomi / y náśmiewáli śię z niego.
Y rzekł im : Wy jeſteśćie / którzy ſámi śiebie uſpráwiedliwiáćie przed ludźmi / ále Bóg zná ſercá wáƺe ; bo co jeſt u ludźi wynioſłego / obrzydliwośćią jeſt przed Bogiem.
Zákon y prorocy áż do Jáná ; á od tego cżáſu króleſtwo Boże opowiádáne bywá / á káżdy śię do niego gwáłtem ćiśnie.
Y łátwiej jeſt niebu y źiemi przeminąć / niżeli jednej kreſce zákonu upáść.
Wƺelki / który opuƺcżá żonę ſwoję á inną pojmuje / cudzołoży ; á kto od mężá opuƺcżoną pojmuje / cudzołoży.
A był niektóry cżłowiek bogáty / który śię obłocżył w ƺárłát y w biśiór / y używáł ná káżdy dźień hojnie.
Był też niektóry żebrák / imieniem Łázárz / który leżáł u wrót jego owrzodźiáły.
Prágnąc być náſycony z odrobin / które pádáły z ſtołu bogácżowego ; ále y pſy przychodząc lizáły wrzody jego.
Y ſtáło śię / że umárł on żebrák / y odnieśiony był od Aniołów ná łono Abráhámowe ; umárł też y bogácż / y pogrzebiony jeſt.
A będąc w piekle / podnióſłƺy ocży ſwe / gdy był w mękách / ujrzáł Abráhámá z dáleká / y Łázárzá ná łonie jego.
Tedy bogácż záwołáwƺy / rzekł : Ojcże Abráhámie! zmiłuj śię nádemną / á poślij Łázárzá / áby omocżył koniec pálcá ſwego w wodźie / á ochłodźił język mój / bo męki ćierpię w tym płomieniu.
Y rzekł Abráhám : Synu! wſpomnij / żeś ty odebráł dobre rzecży twoje zá żywotá twego / á Łázárz tákże złe ; á teráz on má poćiechę / á ty męki ćierpiƺ.
A nád to wƺyſtko między námi y wámi otchłáń wielká jeſt utwierdzoná / áby ći / którzy chcą ſtąd przejść do wáſ / nie mogli / áni owi ſtámtąd przejść do náſ.
A on rzekł : Proƺę ćię tedy / ojcże! ábyś go poſłáł do domu ojcá mego :
Abowiem mám pięć bráći / áby im świádectwo wydáł / żeby też y oni nie przyƺli ná to miejſce męki.
Y rzekł mu Abráhám : Májąć Mojżeƺá y proroków / niechże ich ſłucháją.
A on rzekł : Nie / ojcże Abráhámie! ále gdyby kto z umárłych ƺedł do nich / będą pokutowáć.
Y rzekł mu : Poniewáż Mojżeƺá y proroków nie ſłucháją / tedy / choćby też kto zmártwychwſtáł / nie uwierzą.
