Y rzekł do ucżniów : Nie możná / áby zgorƺeniá przyjść nie miáły ; ále biádá temu / przez którego przychodzą!
Lepiejby mu było / áby młyńſki kámień záwieƺony był ná ƺyi jego / y wrzucony był w morze / niżby jednego z tych máłych zgorƺyć miáł.
Miejćież śię ná piecży. A jeźliby zgrzeƺył przećiwko tobie brát twój / ſtrofuj go : á jeźliby śię upámiętáł / odpuść mu.
A choćby śiedm kroć ná dzień zgrzeƺył przećiwko tobie / y śiedm kroć przez dzień śię do ćiebie náwróćił / mówiąc : Żál mi tego ; odpuść mu.
Y rzekli Apoſtołowie Pánu : Przymnóż nám wiáry.
A Pán rzekł : Jeźlibyśćie mieli wiárę jáko ziárno gorcżycżne / á rzeklibyśćie temu drzewu leśnej figi : Wyrwij śię z korzeniá / á wſádź śię w morzu / uſłucháłoby wáſ.
Y któż z wáſ jeſt / coby miáł ſługę orzącego álbo páƺącego / który / gdyby śię wróćił / zárázby mu rzekł : Pójdź / á uśiądź zá ſtół?
Y owƺem / izáli mu nie rzecże : Nágotuj / cobym wiecżerzáł / á przepáſáwƺy śię / ſłuż mi / áż śię nájem y nápiję / á potem y ty jedz y pij?
Izáli dziękuje ſłudze onemu / iż ucżynił to / co mu było rozkázáno? Nie zdá mi śię.
Tákże y wy / gdy ucżynićie wƺyſtko / co wám rozkázáno / mówćie : Słudzy nieużytecżni jeſteśmy / bo cośmy byli powinni ucżynić / ucżyniliśmy.
Y ſtáło śię / gdy ƺedł do Jeruzálemu / że ƺedł pośrodkiem Sámáryi y Gálilei.
A gdy wchodził do niektórego miáſtecżká / zábieżáło mu dzieśięć mężów trędowátych / którzy ſtánęli z dáleká.
A ći podnióſłƺy głoſ ſwój / rzekli : Jezuśie / Náucżyćielu! zmiłuj śię nád námi.
Które on ujrzáwƺy / rzekł im : Szedłƺy okáżćie śię kápłánom. Y ſtáło śię / gdy ƺli / że ocżyƺcżeni ſą.
Ale jeden z nich ujrzáwƺy / że jeſt uzdrowiony / wróćił śię : wielkim głoſem chwáląc Bogá ;
Y pádł ná oblicże ſwoje u nóg jego / dziękując mu ; á ten był Sámárytánin.
A Jezuſ odpowiádájąc / rzekł : Záż nie dzieśięć jeſt ocżyƺcżonych / á dziewięć kędy?
Nie ználeźli śię / áby śię wróćili / y dáli chwáłę Bogu / jedno ten cudzoziemiec?
Y rzekł mu : Wſtáń / idź / wiárá twojá ćiebie uzdrowiłá.
A będąc pytány od Fáryzeuƺów / kiedy przyjdzie króleſtwo Boże / odpowiedziáł im y rzekł : Nie przyjdzieć króleſtwo Boże z poſtrzeżeniem ;
Ani rzeką : Oto tu / álbo oto tám jeſt : ábowiem oto króleſtwo Boże wewnątrz wáſ jeſt.
Y rzekł do ucżniów : Przyjdą dni / że będziećie żądáć / ábyśćie widzieli jeden dzień ze dni Syná cżłowiecżego / ále nie oglądáćie.
Y rzeką wám : Oto tu / álbo oto tám jeſt ; ále nie chodźćie / áni śię zá nimi udáwájćie.
Abowiem jáko błyſkáwicá / błyſkájąc śię od jednej ſtrony / którá jeſt pod niebem / áż do drugiej / którá jeſt pod niebem / świeći : ták będzie y Syn cżłowiecży w dzień ſwój.
Ale pierwej muśi wiele ućierpieć / y być odrzuconym od národu tego.
A jáko było zá dni Noego / ták będzie y zá dni Syná cżłowiecżego.
Jedli / pili / żenili śię y zá mąż wydáwáli áż do onego dniá / którego wƺedł Noe do korábiá / y przyƺedł potop / á wytráćił wƺyſtkie.
Tákże też jáko śię dziáło zá dni Lotowych / jedli / pili / kupowáli / ſprzedáwáli / ƺcżepili / budowáli.
Ale dniá tego / gdy wyƺedł Lot z Sodomy / ſpádł jáko deƺcż ogień z śiárką z niebá / y wytráćił wƺyſtkie.
Tákći też będzie w on dzień / którego śię Syn cżłowiecży objáwi.
Onegoż dniá / byłliby kto ná dáchu / á nácżyniá jego w domu / niech nie zſtępuje / áby je pobráł ; á kto ná roli / niech śię tákże nie wrácá do tego / co jeſt pozád.
Pámiętájćie ná żonę Lotową.
Ktobykolwiek chćiáł záchowáć duƺę ſwoję / ſtráći ją ; á ktoby ją kolwiek ſtráćił / ożywi ją.
Powiádám wám : Onej nocy będą dwáj ná jednem łożu ; jeden wzięty będzie / á drugi zoſtáwiony.
Dwie będą mleć z ſobą ; jedná wziętá będzie / á drugá zoſtáwioná.
Dwáj będą ná roli ; jeden będzie wzięty / á drugi zoſtáwiony.
A odpowiádájąc rzekli mu : Gdzież Pánie? A on im rzekł : Gdzie będzie śćierw / tám śię zgromádzą y orły.
