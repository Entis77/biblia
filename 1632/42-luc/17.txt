Y rzekł do ucżniów : Niemożna áby / zgorƺenia przyść nie miáły : Ale biádá temu / przez kogo przychodzą.
Lepieyby mu było / áby młyńſki kámień záwieƺony był ná ƺyjey jego / y wrzucony był w morze / niżby jednego z tych máłych zgorƺyć miał.
Miejćież śię ná piecży. A jeſliby zgrzeƺył przećiwko tobie brát twój / ſtrofuj go : á jeſliby śię upámiętał / odpuść mu.
A choćby śiedm kroć ná dźień zgrzeƺył przećiwko tobie / y śiedm kroć przez dźień śię do ćiebie náwróćił / mówiąc ; Żal mi tego : odpuść mu.
Y rzekli Apoſtołowie PANU : Przymnóż nam wiáry.
A PAn rzekł : Jeſlibyśćie mieli wiárę jáko źiárno gorcżycżne / á rzeklibyśćie temu drzewu leśney figi : Wyrwi śię z korzenia / á wſadź śię w morzu ; uſłucháłoby was.
Y któż z was jeſt coby miał ſługę orzącego / álbo páſącego / który gdyby śię wróćił / zárazby mu rzekł : Pódź / á uśiądź zá ſtół.
Y owƺem / izali mu nie rzecże? Nágotuj cobym wiecżerzał / á przepáſawƺy śię / ſłuż mi áż śię nájem y nápiję : A potym y ty jedz y pij.
Izali dźiękuje ſłudze onemu / iż ucżynił to co mu było rozkazano? Nie zda mi śię.
Tákże y wy gdy ucżynićie wƺyſtko co wam rozkazano / mówćie : Słudzy nie użytecżni jeſteſmy / bo coſmy byli powinni ucżynić / ucżyniliſmy.
Y ſtáło śię gdy ƺedł do Jeruzalem / że ƺedł pośrzodkiem Sámáryjey y Gálilejey.
A gdy wchodźił do niektórego miáſtecżká / zábieżáło mu dźieśięć mężów trędowátych / którzy ſtánęli z dáleká.
A ći podniózƺy głos <i>ſwój,</i> rzekli ; JEzuśie Náucżyćielu / zmiłuj śię nád námi.
Które <i>on</i> ujrzawƺy / rzekł im ; Szedƺy okażćie śię kápłanom. Y ſtáło śię gdy ƺli / że ocżyśćieni ſą.
Ale jeden z nich ujrzawƺy że jeſt uzdrowiony / wróćił śię / wielkim głoſem chwaląc Bogá.
Y padł ná oblicże <i>ſwoje</i> u nóg jego dźiękując mu ; á ten był Sámárytan.
A JEzus odpowiádájąc rzekł ; Zaż nie dźieśięć jeſt ocżyśćionych? á dźiewięć kędy?
Nie ználeźli śię áby śię wróćili y dáli chwałę Bogu / jedno ten cudzoźiemiec.
Y rzekł mu ; Wſtań / idź / wiárá twojá ćiebie uzdrowiłá.
A będąc pytány od Fáryzeuƺów / kiedy przydźie króleſtwo Boże? odpowiedźiał im / y rzekł ; Nie przydźieć króleſtwo Boże z poſtrzeżeniem.
Ani rzeką ; Oto tu álbo oto tám <i>jeſt</i> : ábowiem oto króleſtwo Boże jest miedzy wámi.
Y rzekł do ucżniów ; Przydą dni / że będźiećie żądáć ábyśćie widźieli jeden <i>dźień</i> ze dni Syná cżłowiecżego / ále nie oglądaćie.
Y rzeką wam ; Oto tu / álbo oto tám <i>jeſt</i> ; <i>ále</i> nie chodźćie / áni śię <i>zá nimi</i> udawajćie.
Abowiem jáko błyſkáwicá / błyſkájąc śię od <i>jedney</i> ſtrony która jeſt pod niebem / áż do <i>drugiey</i> która jeſt pod niebem / świeći ; ták będźie y Syn cżłowiecży w dźień ſwój.
Ale pierwey muśi wiele ućierpieć / y bydź odrzuconym od narodu tego.
A jáko było zá dni Noego : ták będźie y zá dni Syná cżłowiecżego.
Jedli / pili / żenili śię / y zá mąż wydawáli áż do onego dniá / którego wƺedł Noe do Korabiá / y przyƺedł potop / á wytráćił wƺyſtkie.
Tákże też jáko śię dźiało zá dni Lotowych / jedli / pili / kupowáli / przedawáli / ƺcżepili / budowáli.
Ale dniá tego gdy wyƺedł Lot z Sodomy / zpadł jáko deƺcż ogień z śiárką z niebá / y wytráćił wƺyſtkie.
Tákći też będźie w on dźień / którego śię Syn cżłowiecży objáwi.
Onegoż dniá / byłliby kto ná dáchu / á nacżynia jego w domu / niech nie zſtępuje áby je pobrał ; á kto ná roli / niech śię tákże nie wraca do tego co jeſt pozad.
Pámiętajćie ná żonę Lotowę.
Ktobykolwiek chćiał záchowáć duƺę ſwoję / ſtráći ją : á ktoby ją kolwiek ſtráćił / ożywi ją.
Powiádam wam ; Oney nocy będą dwá ná jednym łożu / jeden wźięty będźie / á drugi zoſtáwiony.
Dwie będą mleć z ſobą ; jedná wźięta będźie / á druga zoſtáwiona.
Dwá będą ná roli ; jeden będźie wźięty / á drugi zoſtáwiony.
A odpowiádájąc rzekli mu ; Gdźież PAnie? A on im rzekł : Gdźie będźie śćierw / tám śię zgromádzą y orłowie.
