Y rzekł do ucżniów : Nie możná / áby zgorƺeniá przyjść nie miáły ; ále biádá temu / przez którego przychodzą!
Lepiejby mu było / áby młyńſki kámień záwieƺony był ná ƺyi jego / y wrzucony był w morze / niżby jednego z tych máłych zgorƺyć miał.
Miejćież śię ná piecży. A jeſliby zgrzeƺył przećiwko tobie brát twój / ſtrofuj go : á jeſliby śię upámiętáł / odpuść mu.
A choćby śiedm kroć ná dźień zgrzeƺył przećiwko tobie / y śiedm kroć przez dźień śię do ćiebie náwróćił / mówiąc : Żál mi tego ; odpuść mu.
Y rzekli Apoſtołowie Pánu : Przymnóż nám wiáry.
A Pán rzekł : Jeſlibyśćie mieli wiárę jáko źiárno gorcżycżne / á rzeklibyśćie temu drzewu leśney figi : Wyrwij śię z korzeniá / á wſádź śię w morzu / uſłucháłoby was.
Y któż z was jeſt / coby miał ſługę orzącego álbo páƺącego / który / gdyby śię wróćił / zárázby mu rzekł : Pójdź / á uśiądź zá ſtół?
Y owƺem / izáli mu nie rzecże : Nágotuj / cobym wiecżerzáł / á przepáſáwƺy śię / ſłuż mi / áż śię nájem y nápiję / á potem y ty jedz y pij?
Izáli dźiękuje ſłudze onemu / iż ucżynił to / co mu było rozkázáno? Nie zdá mi śię.
Tákże y wy / gdy ucżynićie wƺyſtko / co wám rozkázáno / mówćie : Słudzy nieużytecżni jeſteśmy / bo cośmy byli powinni ucżynić / ucżyniliśmy.
Y ſtáło śię / gdy ƺedł do Jeruzálemu / że ƺedł pośrodkiem Sámáryi y Gálilei.
A gdy wchodźił do niektórego miáſtecżká / zábieżáło mu dźieśięć mężów trędowátych / którzy ſtánęli z dáleká.
A ći podnióſłƺy głoſ ſwój / rzekli : JEzuśie / Náucżyćielu! zmiłuj śię nád námi.
Które on ujrzáwƺy / rzekł im : Szedłƺy okáżćie śię kápłánom. Y ſtáło śię / gdy ƺli / że ocżyƺcżeni ſą.
Ale jeden z nich ujrzáwƺy / że jeſt uzdrowiony / wróćił śię : wielkim głoſem chwáląc Bogá ;
Y pádł ná oblicże ſwoje u nóg jego / dźiękując mu ; á ten był Sámárytánin.
A JEzuſ odpowiádájąc / rzekł : Záż nie dźieśięć jeſt ocżyƺcżonych / á dźiewięć kędy?
Nie ználeſli śię / áby śię wróćili / y dáli chwáłę Bogu / jedno ten cudzoźiemiec?
Y rzekł mu : Wſtáń / idź / wiárá twojá ćiebie uzdrowiłá.
A będąc pytány od Fáryzeuƺów / kiedy przyjdźie króleſtwo Boże / odpowiedźiáł im y rzekł : Nie przyjdźieć króleſtwo Boże z poſtrzeżeniem ;
Ani rzeką : Oto tu / álbo oto tám jeſt : ábowiem oto króleſtwo Boże wewnątrz was jeſt.
Y rzekł do ucżniów : Przyjdą dni / że będźiećie żądáć / ábyśćie widźieli jeden dźień ze dni Syná cżłowiecżego / ále nie oglądáćie.
Y rzeką wám : Oto tu / álbo oto tám jeſt ; ále nie chodźćie / áni śię zá nimi udáwájćie.
Abowiem jáko błyſkáwicá / błyſkájąc śię od jedney ſtrony / która jeſt pod niebem / áż do drugiej / która jeſt pod niebem / świeći : ták będźie y Syn cżłowiecży w dźień ſwój.
Ale pierwej muśi wiele ućierpieć / y być odrzuconym od národu tego.
A jáko było zá dni Noego / ták będźie y zá dni Syná cżłowiecżego.
Jedli / pili / żenili śię y zá mąż wydáwáli áż do onego dniá / którego wƺedł Noe do korábiá / y przyƺedł potop / á wytráćił wƺyſtkie.
Tákże też jáko śię dźiáło zá dni Lotowych / jedli / pili / kupowáli / ſprzedáwáli / ƺcżepili / budowáli.
Ale dniá tego / gdy wyƺedł Lot z Sodomy / ſpádł jáko deƺcż ogień z śiárką z niebá / y wytráćił wƺyſtkie.
Tákći też będźie w on dźień / którego śię Syn cżłowiecży objáwi.
Onegoż dniá / byłliby kto ná dáchu / á nácżyniá jego w domu / niech nie zſtępuje / áby je pobráł ; á kto ná roli / niech śię tákże nie wrácá do tego / co jeſt pozád.
Pámiętájćie ná żonę Lotową.
Ktobykolwiek chćiáł záchowáć duƺę ſwoję / ſtráći ją ; á ktoby ją kolwiek ſtráćił / ożywi ją.
Powiádám wám : Oney nocy będą dwáj ná jednem łożu ; jeden wźięty będźie / á drugi zoſtáwiony.
Dwie będą mleć z ſobą ; jedná wźiętá będźie / á drugá zoſtáwioná.
Dwáj będą ná roli ; jeden będźie wźięty / á drugi zoſtáwiony.
A odpowiádájąc rzekli mu : Gdźież Pánie? A on im rzekł : Gdźie będźie śćierw / tám śię zgromádzą y orły.
