A JEzus pełen będąc Duchá Świętego / wróćił śię od Jordanu / y pędzony jeſt w Duchu ná puƺcżą.
Y był przez cżterdźieśći dni kuƺony od dyabłá. A nie jadł nic przez one dni ; ále gdy śię te ſkońcżyły / potym łáknął.
Y rzekł mu Dyabeł ; Jeſliś jeſt Syn Boży / rzecż kámieniowi temu / áby śię ſtał chlebem.
Ale JEzus odpowiedźiał mu / mówiąc ; Nápiſano / iż nie ſámym chlebem żyć będźie cżłowiek / ále káżdym ſłowem Bożym.
Y wwiódł go dyabeł ná górę wyſoką / y pokazał mu wƺyſtkie króleſtwá świátá / we mgnieniu oká.
Y rzekł mu dyabeł : Damći tę wƺyſtkę moc y ſławę ich / bo mi jeſt dáná : á komu chcę / dawam ją.
A ták jeſli śię ukłoniƺ przedemną / będźie wƺyſtko twoje.
A odpowiádájąc JEzus / rzekł mu : Pódź precż ode mnie ƺátánie : ábowiem nápiſano : PAnu Bogu twemu kłaniáć śię będźieƺ / y jemu ſámemu ſłużyć będźieƺ.
Potym wiódł go do Jeruzalem / y poſtáwił go ná ganku kośćielnym / y rzekł mu : Jeſliś jeſt Syn Boży / zpuść śię z tąd ná dół.
Abowiem nápiſano : Że Anjołom ſwojim przykazał o tobie / áby ćię ſtrzegli.
A że ćię ná ręku nośić będą / byś ſnadź nie obráźił o kámień nogi twojey.
A odpowiádájąc JEzus / rzekł mu : Powiedźiano / Nie będźieƺ kuśił PAná Bogá twego.
A gdy dokońcżył wƺyſtkich pokus dyabeł / odſtąpił od niego / do cżáſu.
Y wróćił śię JEzus w mocy onego Duchá do Gálilejey : y rozeƺłá śię o nim wieść po wƺyſtkiey oney okolicżney krájinie.
A on náucżał w bóżnicách ich / y był ſławiony od wƺyſtkich.
Y przyƺedł do Názáretu gdźie był wychowány : y wƺedł według zwycżáju ſwego w dźień Sábbátu do bóżnice / y wſtał / áby cżytał.
Y podano mu Kśięgi Izájáƺá Proroká : A otworzywƺy Kśięgi / ználazł miejſce gdźie było nápiſano :
Duch PAńſki náde mną / przeto mię pomázał : ábym opowiádał Ewángelią ubogim ; poſłał mię ábym uzdrawiał ſkruƺone ná ſercu / ábym zwiáſtował pojimánym wyzwolenie / y ślepym przejrzenie / y ábym wypuśćił ućiśnione ná wolność.
Abym opowiádał rok PAńſki przyjemny.
A záwárƺy Kśięgę / y oddawƺy ją ſłudze / uśiadł ; á ocży wƺyſtkich w bóżnicy pilnie nań pátrzáły.
Y pocżął do nich mówić ; Dźiśći śię wypełniło to piſmo w uƺách wáƺych.
Y wƺyſcy mu dawáli świádectwo / y dźiwowáli śię wdźięcżnośći onych ſłów / które pochodźiły z uſt jego / y mówili ; Izaż ten nie jeſt ſyn Józefów?
Y rzekł do nich ; Pewnie mi rzecżećie onę przypowieść : Lekárzu / ulecż ſámego śiebie : Coſmy ſłyƺeli żeś ucżynił w Kápernáum / ucżyń y tu w ojcżyznie ſwojey.
Y rzekł <i>do nich</i> : Záprawdę wam powiádam / Żaden Prorok nie jeſt przyjemnym w Ojcżyznie ſwojey.
Aleć wam w prawdźie powiádam : <i>że</i> wiele wdów było zá dni Eliaƺowych w ludu Izráelſkim / gdy było zámknione niebo przez trzy látá y ƺeść mieśięcy / ták iż był wielki głód po wƺyſtkiey źiemi.
Wƺákże do żadney z nich nie był poſłány Eliaƺ / tylko do Sárepty miáſtá Sydońſkiego / do jedney wdowy.
Y wiele było trędowátych zá Elizeuƺá Proroká w ludu Izráelſkim ; wƺákże żaden z nich nie był ocżyśćiony / tylko Náámán Syryjcżyk.
Tedy wƺyſcy w bóżnicy gdy to ſłyƺeli / nápełnieni byli gniewem.
A wſtawƺy / wypchnęli go precż z miáſtá ; y wywiedli go ná wierzch góry / ná którey miáſto ich zbudowáne było / áby go z niey ná dół zrzućili.
Ale on przeƺedƺy przez pośrzodek ich / uƺedł.
Y zſtąpił do Kápernáum miáſtá Gálilejſkiego : á tám je náucżał w Sábbáty.
Y zdumiewáli śię nád náuką jego : bo byłá mocna mowá jego.
A w bóżnicy był cżłowiek / który miał duchá dyabłá niecżyſtego ; y záwołał głoſem wielkim /
Mówiąc : Ach! Cóż my z tobą mamy JEzuśie Názáreńſki? Przyƺedłeś ábyś nas wytráćił : Znam ćię ktoś jeſt / <i>żeś</i> on Święty Boży.
Y zgromił go JEzus / mówiąc ; Umilkni / á wynidź z niego. Tedy dyabeł porzućiwƺy go w pośrzodek / wyƺedł z niego nic mu nie záƺkodźiwƺy.
Y przyƺedł ſtrách ná wƺyſtkie : y rozmawiáli miedzy ſobą / mówiąc : Cóż to zá ſłowo : że z władzą y z mocą rozkázuje duchom niecżyſtym / á wychodzą?
Y rozeƺłá śię o nim wieść ná wƺyſtkie miejſcá okolicżney krájiny.
A <i>JEzus</i> wſtawƺy z bóżnice / wƺedł w dom Symonów / á świekrá Symonowá miáłá gorącżkę wielką ; y prośili go zá nią.
Tedy on ſtánąwƺy nád nią / zgromił gorącżkę / y opuśćiłá ją : á zárázem wſtawƺy / poſługowáłá im.
A gdy ſłońce záchodźiło / wƺyſcy którzy mieli chorujące ná rozmájite niemocy / przywodźili je do niego / á on ná káżdego z nich ręce włożywƺy / uzdrawiał je.
Ku temu wychodźili y dyabli z wielu ich / wołájąc y mówiąc : Tyś jeſt on CHryſtus Syn Boży. Ale <i>on</i> zgromiwƺy <i>je,</i> nie dopuƺcżał im mówić : bo wiedźieli iż on jeſt CHryſtus.
A gdy był dźień / wyƺedƺy ƺedł ná miejſce puſte. A lud go ƺukał / y przyƺli áż do niego ; y zátrzymawáli go / áby nie odchodźił od nich.
A on rzekł do nich ; Y innym miáſtom muƺę opowiádáć króleſtwo Boże / bom ná to poſłány.
Y kazał w bóżnicách Gálilejſkich.
