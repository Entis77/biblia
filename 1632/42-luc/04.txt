A Jezuſ pełen będąc Duchá Świętego / wróćił śię od Jordánu / y pędzony jeſt od Duchá ná puƺcżą.
Y był przez cżterdzieśći dni kuƺony od dyjábłá / á nie jádł nic przez one dni ; ále gdy śię te ſkońcżyły / potem łáknął.
Y rzekł mu dyjábeł : Jeźliś jeſt Syn Boży / rzecż kámieniowi temu / áby śię ſtáł chlebem.
Ale Jezuſ odpowiedziáł mu / mówiąc : Nápiſáno / iż nie ſámym chlebem żyć będzie cżłowiek / ále káżdem ſłowem Bożem.
Y wwiódł go dyjábeł ná górę wyſoką / y pokázáł mu wƺyſtkie króleſtwá świátá we mgnieniu oká.
Y rzekł mu dyjábeł : Dám ći tę wƺyſtkę moc y ſłáwę ich ; bo mi jeſt dáná / á komu chcę / dáwám ją.
A ták jeźli śię ukłoniƺ przedemną / będzie wƺyſtko twoje.
A odpowiádájąc Jezuſ rzekł mu : Pójdź precż odemnie / ƺátánie! ábowiem nápiſáno : Pánu / Bogu twemu / kłániáć śię będzieƺ / y jemu ſámemu ſłużyć będzieƺ.
Potem wiódł go do Jeruzálemu / y poſtáwił go ná gánku kośćielnym / y rzekł mu : Jeźliś jeſt Syn Boży / ſpuść śię ſtąd ná dół ;
Abowiem nápiſáno : Że Aniołom ſwoim przykázáł o tobie / áby ćię ſtrzegli /
A że ćię ná rękách nośić będą / byś ſnáć nie obráził o kámień nogi twojej.
A odpowiádájąc Jezuſ rzekł mu : Powiedziáno : Nie będzieƺ kuśił Páná / Bogá twego.
A gdy dokońcżył wƺyſtkich pokuſ dyjábeł / odſtąpił od niego do cżáſu.
Y wróćił śię Jezuſ w mocy onego Duchá do Gálilei. Y rozeƺłá śię o nim wieść po wƺyſtkiej onej okolicżnej kráinie.
A on náucżáł w bóżnicách ich / y był ſłáwiony od wƺyſtkich.
Y przyƺedł do Názáretu / gdzie był wychowány / y wƺedł według zwycżáju ſwego w dzień ſábátu do bóżnicy / y wſtáł / áby cżytáł.
Y podáno mu kśięgi Izájáƺá proroká ; á otworzywƺy kśięgi / ználázł miejſce / gdzie było nápiſáno :
Duch Páńſki nádemną ; przeto mię pomázáł / ábym opowiádáł Ewángieliją ubogim ; poſłáł mię / ábym uzdráwiáł ſkruƺone ná ſercu / ábym zwiáſtowáł pojmánym wyzwolenie / y ślepym przejrzenie / y ábym wypuśćił ućiśnione ná wolność ;
Abym opowiádáł rok Páńſki przyjemny.
A záwárłƺy kśięgę y oddáwƺy ją ſłudze / uśiádł ; á ocży wƺyſtkich w bóżnicy pilnie náń pátrzáły.
Y pocżął do nich mówić : Dziśći śię wypełniło to piſmo w uƺách wáƺych.
Y wƺyſcy mu dáwáli świádectwo / y dziwowáli śię wdzięcżnośći onych ſłów / które pochodziły z uſt jego / y mówili : Izáż ten nie jeſt ſyn Józefowy?
Y rzekł do nich : Pewnie mi rzecżećie onę przypowieść : Lekárzu! ulecż ſámego śiebie! Cośmy ſłyƺeli / żeś ucżynił w Kápernáum / ucżyń y tu w ojcżyźnie ſwojej.
Y rzekł do nich : Zápráwdę wám powiádám : Żáden prorok nie jeſt przyjemnym w ojcżyźnie ſwojej.
Aleć wám w práwdzie powiádám / że wiele wdów było zá dni Elijáƺowych w ludzie Izráelſkim / gdy było zámknione niebo przez trzy látá y ƺeść mieśięcy / ták iż był wielki głód po wƺyſtkiej ziemi ;
Wƺákże do żádnej z nich nie był poſłány Elijáƺ / tylko do Sárepty / miáſtá Sydońſkiego / do jednej wdowy.
Y wiele było trędowátych zá Elizeuƺá proroká / w ludzie Izráelſkim / wƺákże żáden z nich nie był ocżyƺcżony / tylko Náámán / Syryjcżyk.
Tedy wƺyſcy w bóżnicy / gdy to ſłyƺeli / nápełnieni byli gniewem ;
A wſtáwƺy / wypchnęli go precż z miáſtá / y wywiedli go ná wierzch góry / ná której miáſto ich zbudowáne było / áby go z niej ná dół zrzućili.
Ale on przeƺedłƺy przez pośrodek ich / uƺedł.
Y zſtąpił do Kápernáum / miáſtá Gálilejſkiego / á tám je náucżáł w ſábáty.
Y zdumiewáli śię nád náuką jego ; bo byłá mocná mowá jego.
A w bóżnicy był cżłowiek / który miáł duchá dyjábłá niecżyſtego / y záwołáł głoſem wielkim /
Mówiąc : Ach! Cóż my z tobą mámy / Jezuśie Názáreńſki? Przyƺedłeś / ábyś náſ wytráćił ; znám ćię / ktoś jeſt / żeś on Święty Boży.
Y zgromił go Jezuſ / mówiąc : Umilknij á wynijdź z niego. Tedy dyjábeł porzućiwƺy go w pośrodek / wyƺedł z niego / nic mu nie záƺkodziwƺy.
Y przyƺedł ſtrách ná wƺyſtkie / y rozmáwiáli między ſobą / mówiąc : Cóż to zá ſłowo / że z włádzą y z mocą rozkázuje duchom niecżyſtym / á wychodzą?
Y rozeƺłá śię o nim wieść ná wƺyſtkie miejſcá okolicżnej kráiny.
A Jezuſ wſtáwƺy / z bóżnicy wƺedł w dom Szymoná / á świekrá Szymonowá miáłá gorącżkę wielką ; y prośili go zá nią.
Tedy on ſtánąwƺy nád nią / zgromił gorącżkę / y opuśćiłá ją ; á zárázem wſtáwƺy / poſługiwáłá im.
A gdy ſłońce záchodziło / wƺyſcy / którzy mieli chorujące ná rozmáite niemocy / przywodzili je do niego / á on ná káżdego z nich ręce włożywƺy / uzdráwiáł je.
Ku temu wychodzili y dyjábli z wielu ich / wołájąc y mówiąc : Tyś jeſt on Chryſtuſ / Syn Boży ; ále on zgromiwƺy je / nie dopuƺcżáł im mówić ; bo wiedzieli / iż on jeſt Chryſtuſ.
A gdy był dzień / wyƺedłƺy / ƺedł ná miejſce puſte. A lud go ƺukáł / y przyƺli áż do niego / y zátrzymywáli go / áby nie odchodził od nich.
A on rzekł do nich : Y innym miáſtom muƺę opowiádáć króleſtwo Boże ; bom ná to poſłány.
Y kázáł w bóżnicách Gálilejſkich.
