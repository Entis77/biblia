A przybliżáło śię święto przáśników / które zowią wielkánocą.
Y ƺukáli przedniejśi kápłáni y náucżeni w Piśmie / jákoby go zábili ; ále śię báli ludu.
Y wſtąpił ƺátán w Judáƺá / którego zwáno Iƺkáryjotem / który był z licżby dwunáſtu.
Ten tedy odƺedłƺy / zmówił śię z przedniejƺymi kápłánámi y z przełożonymi kośćiołá / jákoby go im wydáł.
Y urádowáli śię / y umówili śię z nim / że mu chcą dáć pieniądze.
Y obiecáł / y ƺukáł ſpoſobnego cżáſu / áby go im wydáł bez rozruchu.
Tedy przyƺedł dźień przáśników / którego miáł báránek być zábity.
Y poſłáł Piotrá y Jáná / mówiąc : Poƺedłƺy nágotujćie nám báránká / ábyśmy jedli.
Ale oni mu rzekli : Gdźież chceƺ / ábyśmy go nágotowáli?
A on rzekł do nich : Oto gdy do miáſtá wchodźić będźiećie / ſpotká śię z wámi cżłowiek / nioſący dzbán wody ; idźćież zá nim do domu / do którego wnijdźie /
A rzecżćie goſpodárzowi domu onego : Kázáł ći powiedźieć náucżyćiel : Gdźie jeſt goſpodá / kędybym jádł báránká z ucżniámi moimi?
A on wám ukáże ſálę wielką uſłáną / támże nágotujćie.
Tedy odƺedłƺy ználeźli / jáko im był powiedźiáł / y nágotowáli báránká.
A gdy przyƺłá godźiná / uśiádł zá ſtół / y dwánáśćie Apoſtołów z nim.
Y rzekł do nich : Żądájąc żądáłem tego báránká jeść z wámi / pierwej niżbym ćierpiáł.
Boć wám powiádám / że go więcej jeść nie będę / áż śię wypełni w króleſtwie Bożem.
A wźiąwƺy kielich y podźiękowáwƺy / rzekł : Weźmijćie to á podźielćie między śię.
Abowiem powiádám wám / że nie będę pił z rodzáju winnej máćicy / áż przyjdźie króleſtwo Boże.
A wźiąwƺy chleb y podźiękowáwƺy / łámáł y dáł im / mówiąc : To jeſt ćiáło moje / które śię zá wáſ dáje ; to cżyńćie ná pámiątkę moję.
Tákże y kielich / gdy było po wiecżerzy / mówiąc : Ten kielich jeſt nowy teſtáment we krwi mojej / którá śię zá wáſ wylewá.
Ale oto ręká tego / co mię wydáje / zemną jeſt zá ſtołem.
Synći záiſte cżłowiecży idźie / ták jáko jeſt poſtánowione ; ále biádá cżłowiekowi temu / który go wydáje!
Tedy śię oni pocżęli między ſobą pytáć o tem / ktoby wżdy z nich był / coby to ucżynić miáł.
A wƺcżął śię też ſpór między nimi o tem / ktoby śię z nich zdáł być więkƺy.
Ale on im rzekł : Królowie národów pánują nád nimi / á którzy nád nimi moc máją / dobrodźiejámi názywáni bywáją.
Lecż wy nie ták : owƺem kto nájwiękƺy jeſt między wámi / niech będźie jáko nájmniejƺy / á kto jeſt wodzem / będźie jáko ten / co ſłuży.
Bo któryż więkƺy jeſt? Ten / co śiedźi / cżyli ten / co ſłuży? Izáli nie ten / co śiedźi? Alem já jeſt w pośrodku wáſ jáko ten / co ſłuży.
A wy jeſteśćie / którzyśćie wytrwáli przy mnie w pokuſách moich.
Y jáć wám ſporządzám jáko mi ſporządźił Ojćiec mój / króleſtwo /
Abyśćie jedli y pili zá ſtołem moim w króleſtwie mojem / y śiedźieli ná ſtolicách / ſądząc dwánáśćie pokoleń Izráelſkich.
Y rzekł Pán : Szymonie / Szymonie! oto ƺátán wyprośił wáſ / áby wáſ odwiewáł jáko pƺenicę /
Alem já prośił zá tobą / áby nie uſtáłá wiárá twojá ; á ty niekiedy náwróćiwƺy śię / utwierdzáj bráći twoich.
A on mu rzekł : Pánie! gotówem z tobą iść y do więźieniá y ná śmierć.
A on rzekł : Powiádám ći / Piotrze! nie zápieje dźiś kur / áż śię pierwej trzykroć záprzeƺ / że mię nie znáƺ.
Y rzekł im : Gdym wáſ poſyłáł bez mieƺká / y bez táiſtry / y bez butów / izáli wám cżego nie doſtáwáło? A oni rzekli : Nicżego.
Tedy im rzekł : Ale teráz kto má mieƺek / niech go weźmie / tákże y táiſtrę ; á kto nie má miecżá / niech ſprzedá ſuknię ſwoję á kupi go.
Abowiem powiádám wám / iż śię jeƺcże muśi to / co nápiſáno / wypełnić ná mnie / miánowićie : Y z złocżyńcámi policżony jeſt ; boć te rzecży / które świádcżą o mnie / koniec biorą.
Ale oni rzekli : Pánie! oto tu dwá miecże. A on im rzekł : Doſyć jeſt.
Y wyƺedłƺy ƺedł według zwycżáju ná górę Oliwną / á ƺli zá nim y ucżniowie jego.
A gdy przyƺedł ná miejſce / rzekł im : Módlćie śię / ábyśćie nie weƺli w pokuƺenie.
A ſám oddálił śię od nich / jákoby ná ćiśnienie kámieniem / á klęknąwƺy ná koláná / modlił śię /
Mówiąc : Ojcże! jeźli chceƺ / przenieś ten kielich odemnie ; wƺákże nie mojá wolá / lecż twojá niech śię ſtánie.
Y ukázáł mu śię Anioł z niebá / pośilájący go.
Ale będąc w boju / gorliwiej śię modlił / á był pot jego jáko krople krwi śćiekájące ná źiemię.
A wſtáwƺy od modlitwy / przyƺedł do ucżniów / y ználázł je śpiące od ſmutku.
Y rzekł im : Cóż śpićie? wſtáńćie / á módlćie śię / byśćie nie weƺli w pokuƺenie.
A gdy on jeƺcże mówił / oto zgrájá / y ten / którego zwáno Judáƺem / jeden ze dwunáſtu / ƺedł przed nimi / y przybliżył śię do JEzuſá / áby go pocáłowáł.
A JEzuſ mu rzekł : Judáƺu! pocáłowániem wydájeƺ Syná cżłowiecżego?
A widząc ći / którzy przy nim byli / co śię dźiáć miáło / rzekli mu : Pánie! mámyli bić miecżem?
Y uderzył jeden z nich ſługę nájwyżƺego kápłáná / y ućiął mu ucho práwe.
Ale JEzuſ odpowiádájąc / rzekł : Zániechájćie áż póty ; á dotknąwƺy śię uchá jego / uzdrowił go.
Y rzekł JEzus do onych którzy byli przyƺli przeciwko niemu / do Przedniejƺych kápłanów / y do Przełożonych Kośćiołá / y do Stárƺych; Wyƺliśćie jáko na zbójcę z miecżmi y z kijmi.
Gdym ná káżdy dźień bywáł z wámi w kośćiele / nie śćiągnęliśćie rąk ná mię ; ále táć jeſt oná godźiná wáƺá y moc ćiemnośći.
Pojmáwƺy go tedy / prowádźili go y przyprowádźili go w dom nájwyżƺego kápłáná / á Piotr ƺedł zá nim z dáleká.
A gdy oni rozniećili ogień w pośrodku dworu y weſpół uśiedli / uśiádł y Piotr między nimi.
A ujrzáwƺy go niektórá dźiewká u ogniá śiedzącego / y pilnie mu śię przypátrzywƺy / rzekłá : Y ten z nim był.
A on śię go záprzáł / mówiąc : Niewiáſto! Nie znám go.
A po máłej chwili ujrzáwƺy go drugi / rzekł : Y tyś jeſt z nich ; ále Piotr rzekł : Cżłowiecże! nie jeſtem.
A gdy wyƺłá jákoby jedná godźiná / ktoś inƺy twierdźił / mówiąc : Práwdźiwie y ten z nim był ; bo też jeſt Gálilejcżyk.
A Piotr rzekł : Cżłowiecże! nie wiem co mówiƺ ; á zátem záráz / gdy on jeƺcże mówił / kur zápiáł.
A Pán obróćiwƺy śię / ſpojrzáł ná Piotrá. Y wſpomniáł Piotr ná ſłowo Páńſkie / jáko mu był powiedźiáł : Iż pierwej niż kur zápieje / trzykroć śię mnie záprzeƺ.
A Piotr wyƺedłƺy precż / gorzko płákáł.
Lecż mężowie / którzy weſpół trzymáli JEzuſá / náśmiewáli śię z niego / bijąc go ;
A zákrywƺy go / bili twárz jego y pytáli go / mówiąc : Prorokuj / kto jeſt / co ćię uderzył.
Y wiele innych rzecży bluźniąc mówili przećiwko niemu.
A gdy był dźień / zeƺli śię ſtárśi z ludu y nájwyżśi kápłáni y náucżeni w Piśmie / á przywiedli go do rády ſwojej.
Mówiąc : Jeźliś ty jeſt CHryſtuſ / powiedz nám? Y rzekł im : Choćbym wám powiedźiáł / nie uwierzyćie.
A jeźlibym też o co pytáł / nie odpowiećie mi / áni mię wypuśćićie.
Od tego cżáſu będźie Syn cżłowiecży śiedźiáł ná práwicy mocy Bożej.
Y rzekli wƺyſcy : Tyś tedy jeſt on ſyn Boży? A on rzekł do nich : Wy powiádáćie / żem já jeſt.
A oni rzekli : Cóż jeƺcże potrzebujemy świádectwá? Wƺákieśmy ſámi ſłyƺeli z uſt jego.
