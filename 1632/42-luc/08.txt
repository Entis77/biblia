Y ſtáło śię potem / że on chodził po miáſtách y po miáſtecżkách káżąc y opowiádájąc króleſtwo Boże / á oni dwunáſtu byli z nim /
Y niektóre niewiáſty / które był uzdrowił od duchów złych y od niemocy ich / jáko Máryjá / którą zwáno Mágdáleną / z której było śiedm dyjábłów wyƺło ;
Y Joánná / żoná Chuzego / urzędniká Herodowego / y Zuzánná / y inƺych wiele / które mu ſłużyły z májętnośći ſwoich.
A gdy śię ſchodził wielki lud / y z różnych miáſt gárnęli śię do niego / rzekł przez podobieńſtwo ;
Wyƺedł rozśiewcá / áby rozśiewáł náśienie ſwoje ; á gdy on rozśiewáł / tedy jedno pádło podle drogi y podeptáne jeſt / á ptáki niebieſkie pozobáły je.
A drugie pádło ná opokę / á gdy weƺło / uſchło / przeto iż nie miáło wilgotnośći.
A drugie pádło między ćiernie ; ále ćiernie weſpół z nim wzroſły / y záduśiły je.
A drugie pádło ná ziemię dobrą / á gdy weƺło / przynioſło pożytek ſtokrotny. To mówiąc wołáł : Kto má uƺy ku ſłuchániu / niecháj ſłuchá!
Y pytáli go ucżniowie jego / mówiąc : Coby to było zá podobieńſtwo?
A on im rzekł : Wám dáno wiedzieć tájemnicę króleſtwá Bożego ; ále innym w podobieńſtwách / áby widząc nie widzieli / á ſłyƺąc nie rozumieli.
A to podobieńſtwo tákie jeſt : Náśienie jeſt ſłowo Boże.
A którzy podle drogi / ći ſą / którzy ſłucháją / zátem przychodzi dyjábeł / y wybierá ſłowo z ſercá ich / áby uwierzywƺy / nie byli zbáwieni.
A którzy ná opoce / ći ſą / którzy gdy ſłucháją / z rádośćią ſłowo przyjmują / ále ći korzeniá nie máją / ći do cżáſu wierzą / á cżáſu pokuſy odſtępują.
A które pádło między ćiernie / ći ſą / którzy ſłucháją ſłowá : ále odƺedłƺy / od piecżołowániá / y bogáctw / y rozkoƺy żywotá bywáją záduƺeni / y nie przynoƺą pożytku.
Ale które pádło ná ziemię dobrą / ći ſą / którzy w ſercu uprzejmem y dobrem ſłyƺáne ſłowo záchowują / y owoc przynoƺą w ćierpliwośći.
A żáden zápáliwƺy świecę / nie nákrywá jej nácżyniem / áni jej kłádzie pod łoże / ále ją ſtáwiá ná świecżniku / áby ći / którzy wchodzą / widzieli świátło.
Bo niemáƺ nic tájemnego / coby nie miáło być objáwiono ; y niemáƺ nic ſkrytego / cżegoby śię nie dowiedziáno / y coby ná jáw nie wyƺło.
Przetoż pátrzćie / jáko ſłucháćie : ábowiem kto má / temu będzie dáne / á kto niemá / y to / co mniemá / że má / będzie odjęte od niego.
Tedy przyƺli do niego mátká y bráćiá jego ; ále do niego przyſtąpić nie mogli dlá ludu.
Y dáno mu znáć / mówiąc : Mátká twojá y bráćiá twoi ſtoją przed domem / chcąc ćię widzieć.
A on odpowiádájąc / rzekł do nich : Mátká mojá y bráćiá moi ſą ći / którzy ſłowá Bożego ſłucháją y cżynią je.
Y ſtáło śię dniá jednego / że on wſtąpił w łódź y ucżniowie jego / y rzekł do nich : Przepráwmy śię ná drugą ſtronę jeziorá. Y puśćili śię.
A gdy płynęli / uſnął. Y przypádłá náwáłność wiátru ná jezioro / y łódź śię zálewáłá / ták że byli w niebezpiecżeńſtwie.
A przyſtąpiwƺy / obudzili go / mówiąc : Miſtrzu / miſtrzu! giniemy. A on ocknąwƺy śię / zgromił wiátr y wáły wodne / y uśmierzyły śię / y ſtáło śię ućiƺenie.
Tedy im rzekł : Gdzież jeſt wiárá wáƺá? A bojąc śię / dziwowáli śię / mówiąc jedni do drugich : Któż wżdy jeſt ten / że y wiátrom rozkázuje y wodom / á ſą mu poſłuƺne?
Y przewieźli śię do kráiny Gádáreńcżyków / którá jeſt przećiw Gálilei.
A gdy wſtąpił ná ziemię / zábieżáł mu mąż niektóry z onego miáſtá / co miáł dyjábły od niemáłego cżáſu / á nie obłócżył śię w ƺáty / y nie mieƺkáł w domu / tylko w grobách.
Ten ujrzáwƺy Jezuſá / zákrzyknął / y upádł przed nim á głoſem wielkim rzekł : Cóż já mám z tobą / Jezuśie / Synu Bogá nájwyżƺego? proƺę ćię / nie dręcż mię.
Abowiem rozkázáł onemu duchowi niecżyſtemu / áby wyƺedł z onego cżłowieká : bo od wielu cżáſów porywáł go ; á choćiáż go wiązáno łáńcuchámi y w pętách ſtrzeżono / jednák on porwáwƺy okowy / bywáł od dyjábłá ná puſtynię pędzony.
Y pytáł go Jezuſ / mówiąc : Co máƺ zá imię? A on rzekł : Wojſko ; ábowiem wiele dyjábłów wſtąpiło było weń.
Tedy go prośili / áby im nie rozkázywáł ſtámtąd odejść w przepáść.
A byłá tám trzodá wielká świni / którá śię páſłá ná górze / y prośili go / áby im dopuśćił wſtąpić w nie. Y dopuśćił im.
A wyƺedłƺy dyjábli z onego cżłowieká / weƺli w świnie ; y porwáłá śię oná trzodá pędem z przykrá do jeziorá / y utonęłá.
A widząc páſterze / co śię ſtáło / ućiekli ; á poƺedłƺy / oznájmili to w mieśćie y we wśiách.
Y wyƺli / áby oglądáli to / co śię ſtáło ; á przyƺedłƺy do Jezuſá / ználeźli cżłowieká onego / z którego wyƺli dyjábli / oblecżonego / przy dobrem bácżeniu śiedzącego u nóg Jezuſowych / y báli śię.
Opowiedzieli im tedy ći / którzy widzieli / jáko uzdrowiono tego / który był opętány.
Y prośiło go wƺyſtko mnóſtwo onej okolicżnej kráiny Gádáreńcżyków / áby odƺedł od nich ; ábowiem ich był wielki ſtrách ogárnął. A on wſtąpiwƺy w łódź / wróćił śię.
Y prośił go on mąż / z którego wyƺli dyjábli / áby był przy nim ; ále go Jezuſ odpráwił / mówiąc :
Wróć śię do domu twego / á opowiádáj / jákoć wielkie rzecży Bóg ucżynił. Y odƺedł / po wƺyſtkiem mieśćie opowiádájąc / jáko mu wielkie rzecży Jezuſ ucżynił.
Y ſtáło śię / gdy śię wróćił Jezuſ / że go przyjął lud ; ábowiem náń wƺyſcy ocżekiwáli.
A oto przyƺedł mąż imieniem Jáiruſ / á ten był przełożonym bóżnicy ; á przypádłƺy do nóg Jezuſowych / prośił go / áby wƺedł w dom jego.
Abowiem miáł córkę jedyną około dwunáſtu lát / którá już konáłá. ( A gdy on ƺedł / ćiſnął go lud. )
A niewiáſtá / którá płynienie krwi ćierpiáłá od lát dwunáſtu / y wynáłożyłá byłá ná lekárzy wƺyſtko ſwoje pożywienie / á nie mogłá być od nikogo ulecżoná /
Przyſtąpiwƺy z tyłu / dotknęłá śię podołká ƺáty jego / á zárázem śię záſtánowiło płynienie krwi jej.
Y rzekł Jezuſ : Któż jeſt / co śię mnie dotknął? á gdy śię wƺyſcy zápieráli / rzekł Piotr / y ći / którzy z nim byli : Miſtrzu! lud ćię ćiśnie y tłocży / á ty mówiƺ : Kto śię mnie dotknął?
Y rzekł Jezuſ : Dotknął śię mnie ktoś / bom poznáł / że moc odemnie wyƺłá.
A widząc oná niewiáſtá / że śię nie utáiłá / ze drżeniem przyſtąpiłá y upádłá przed nim / y dlácżego śię go dotknęłá / powiedziáłá mu przed wƺyſtkim ludem / y jáko záráz uzdrowioná byłá.
A on jej rzekł : Ufáj / córko! wiárá twojá ćiebie uzdrowiłá ; idźże w pokoju.
A gdy on to jeƺcże mówił / przyƺedł niektóry od przełożonego bóżnicy / powiádájąc mu : Iż umárłá córká twojá / nie trudź Náucżyćielá.
Ale Jezuſ uſłyƺáwƺy to / odpowiedziáł mu / mówiąc : Nie bój śię / tylko wierz / á będzie uzdrowioná.
A wƺedłƺy w dom / nie dopuśćił z ſobą wnijść nikomu / tylko Piotrowi / y Jákóbowi / y Jánowi / y ojcu y mátce onej dziewecżki.
A płákáli wƺyſcy / y nárzekáli nád nią. Ale on rzekł : Nie płácżćież! Nie umárłáć / ále śpi.
Y náśmiewáli śię z niego / wiedząc / iż byłá umárłá.
A on wygnáwƺy precż wƺyſtkich / y ująwƺy ją zá rękę / záwołáł / mówiąc : Dziewecżko / wſtáń!
Y wróćił śię duch jej ; y wſtáłá záráz ; y rozkázáł / áby jej jeść dáno.
Y zdumieli śię rodzice jej. A on im zákázáł / áby nikomu nie powiádáli tego / co śię było ſtáło.
