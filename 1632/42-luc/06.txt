Y ſtáło śię w drugi ſábát / że ƺedł Jezuſ przez zbożá ; y rwáli ucżniowie jego kłoſy / á rękámi wyćierájąc jedli.
Ale niektórzy z Fáryzeuƺów rzekli do nich : Przecżże cżynićie to / cżego śię nie godzi cżynić w ſábát?
A odpowiádájąc Jezuſ / rzekł do nich : Azáśćie tego nie cżytáli / co ucżynił Dawid / gdy łáknął ſám / y ći / którzy z nim byli?
Jáko wƺedł do domu Bożego / á wziął chleby pokłádne / y jádł / á dáł y tym / którzy z nim byli ; których śię nie godziło jeść / tylko ſámym kápłánom?
Y rzekł im : Syn cżłowiecży jeſtći Pánem y ſábátu.
Stáło śię tákże y w inƺy ſábát / że Jezuſ wƺedł do bóźnicy / y náucżáł ; y był tám cżłowiek / którego ręká práwá byłá uſchłá.
Y podſtrzegli go náucżeni w Piśmie y Fáryzeuƺowie / jeźliby w ſábát uzdráwiáł / áby ználeźli / o coby náń ſkárżyli.
Ale on wiedziáł myśli ich / y rzekł cżłowiekowi / który miáł rękę uſchłą : Wſtáń á ſtáń pośrodku. A on wſtáwƺy / ſtánął.
Rzekł tedy do nich Jezuſ : Spytám wáſ o jednę rzecż : Godzili śię w ſábáty dobrze cżynić / cżyli źle cżynić? Cżłowieká záchowáć / cżyli zátráćić?
A ſpojrzáwƺy w koło po wƺyſtkich / rzekł onemu cżłowiekowi : Wyćiągnij rękę twoję! á on ták ucżynił / y przywróconá jeſt do zdrowiá ręká jego / jáko y drugá.
Ale oni nápełnieni będąc ƺáleńſtwem / rozmáwiáli miedzy ſobą / coby ucżynić mieli Jezuſowi.
Y ſtáło śię w oneż dni / odƺedł ná górę / áby śię modlił ; y był tám przez noc ná modlitwie Bożej.
A gdy był dzień / zwołáł ucżniów ſwych y wybráł z nich dwánáśćie / które też názwáł Apoſtołámi :
Szymoná / którego też názwáł Piotrem / y Andrzejá brátá jego / Jákóbá / y Jáná / Filipá / y Bártłomiejá ;
Máteuƺá / y Tomáƺá / Jákóbá / ſyná Alfeuƺowego / y Szymoná / którego zowią Zeloteſ ;
Judáƺá / brátá Jákóbowego / y Judáƺá Iƺkáryjotę / który potem był zdrájcą.
A zſtąpiwƺy z nimi / ſtánął ná miejſcu polá równego / y gormádá ucżniów jego / y wielkie mnóſtwo ludu ze wƺyſtkiej Judzkiej ziemi / y z Jeruzálemu / y z kráju pomorſkiego / leżącego przy Tyrze y Sydonie / którzy byli przyƺli / áby go ſłucháli / y byli uzdrowieni od chorób ſwoich ;
Y ći / którzy byli trápieni od duchów niecżyſtych / byli uzdrowieni.
A wƺyſtek lud ƺukáł / jákoby śię go dotknąć ; ábowiem moc wychodziłá z niego / y uzdráwiáłá wƺyſtkich.
A on podnióſłƺy ocży ſwoje ná ucżniów / mówił : Błogoſłáwieni jeſteśćie wy / ubodzy! bo wáƺe jeſt króleſtwo Boże.
Błogoſłáwieni jeſteśćie / którzy teráz łákniećie ; bo będziećie náſyceni. Błogoſłáwieni jeſteśćie / którzy teráz płácżećie ; bo śię śmiáć będziećie.
Błogoſłáwieni będziećie / gdy wáſ ludzie nienáwidzieć będą / y gdy wyłącżą / y będą wáſ ſromoćić / y imię wáƺe wyrzucą jáko złe / dlá Syná cżłowiecżego.
Rádujćie śię dniá tego y weſelćie śię ; ábowiem oto zápłátá wáƺá jeſt obfitá w niebieśiech ; boć ták włáśnie prorokom cżynili ojcowie ich.
Ale biádá wám bogácżom! bo już máćie poćiechę wáƺę.
Biádá wám / którzyśćie náſyceni! ábowiem łáknąć będziećie. Biádá wám / którzy śię teráz śmiejećie! bo śię ſmućić y płákáć będziećie.
Biádá wám / gdyby dobrze o wáſ mówili wƺyſcy ludzie ; bo ták cżynili fáłƺywym prorokom ojcowie ich.
Ale wám powiádám / którzy ſłucháćie : Miłujćie nieprzyjáćioły wáƺe / cżyńćie dobrze tym / którzy wáſ máją w nienáwiśći.
Błogoſłáwćie tym / którzy wáſ przeklináją ; y módlćie śię zá tymi / którzy wám złość wyrządzáją.
Temu / któryby ćię uderzył w policżek / nádſtáw mu y drugiego : á temu / któryćby bráł płáƺcż / y ſukni nie zbrániáj ;
Y káżdemu / któryby ćię prośił / dáj / á temu / co twoje bierze / nie upomináj śię ;
Y cobyśćie chćieli / áby wám ludzie cżynili / ták y wy im cżyńćie.
Abowiem jeźli miłujećie te / którzy wáſ miłują / jákąż łáſkę máćie? ábowiem y grzeƺnicy toż włáśnie cżynią.
A jeźli dobrze cżynićie tym / którzy wám dobrze cżynią / jákąż łáſkę máćie? ábowiem toż y grzeƺnicy cżynią.
A jeźli pożycżáćie tym / od których śię ſpodziewáćie odebráć / jákąż łáſkę máćie? ábowiem y grzeƺnicy grzeƺnikom pożycżáją / áby záśię tyle odebráli.
Owƺem miłujćie nieprzyjáćioły wáƺe / y cżyńćie im dobrze / y pożycżájćie / nic śię ſtąd nie ſpodziewájąc / á będzie wielká zápłátá wáƺá / y będziećie ſynámi Nájwyżƺego ; ábowiem on dobrotliwy jeſt przećiw niewdzięcżnym y złym.
Przetoż bądźćie miłośierni / jáko y Ojćiec wáƺ miłośierny jeſt.
Nie ſądźćie / á nie bądziećie ſądzeni ; nie potępiájćie / á nie będziećie potępieni ; odpuƺcżájćie / á będzie wám odpuƺcżono.
Dáwájćie / á będzie wám dáno ; miárę dobrą / nátłocżoną / y potrzęśioną / y opływájącą dádzą ná łono wáƺe ; ábowiem tąż miárą / którą mierzyćie / będzie wám záś obmierzono.
Y powiedziáł im podobieńſtwo : Izáli może ślepy ślepego prowádzić? ázáż nie obádwáj w dół wpádną?
Nie jeſtći ucżeń nád miſtrzá ſwego ; lecż doſkonáły będzie káżdy / będzieli jáko miſtrz jego.
A cżemuż widziƺ źdźbło w oku brátá twego / á bálki / którá jeſt w oku twojim / nie bácżyƺ?
Albo jákoż możeƺ rzec brátu twemu : Bráćie! dopuść / iż wyjmę źdźbło / które jeſt w oku twojim / á ſám bálki / którá jeſt w oku twojim / nie widziƺ? Obłudniku! wyjmij pierwej bálkę z oká twego / á tedy przejrzyƺ / ábyś wyjął źdźbło / które jeſt w oku brátá twego.
Nie jeſt bowiem drzewo dobre / które przynośi owoc zły ; áni jeſt drzewo złe / które przynośi owoc dobry ;
Gdyż káżde drzewo z owocu włáſnego poznáne bywá ; boć nie zbieráją z ćierniá figów / áni z głogu zbieráją winnych gron.
Cżłowiek dobry z dobrego ſkárbu ſercá ſwego wynośi rzecży dobre / á zły cżłowiek ze złego ſkárbu ſercá ſwego wynośi rzecży złe ; ábowiem z obfitośći ſercá mówią uſtá jego.
Przecżże mię tedy zowiećie Pánie / Pánie! á nie cżynićie tego / co mówię?
Káżdy / który przychodzi do mnie / á ſłuchá ſłów moich / y cżyni je / pokáżę wám / komu jeſt podobnym.
Podobny jeſt cżłowiekowi dom budującemu / który kopáł y wykopáł głęboko / á záłożył grunt ná opoce ; á gdy przyƺłá powódź / otrąćiłá śię rzeká o on dom / ále nie mogłá go poruƺyć ; bo był záłożony ná opoce.
Ale który ſłuchá á nie cżyni / podobny jeſt cżłowiekowi / który zbudowáł dom ſwój ná ziemi bez gruntu ; o który śię otrąćiłá rzeká / y zárázem upádł / á był upádek domu onego wielki.
