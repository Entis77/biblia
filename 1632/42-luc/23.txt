Tedy powſtáwƺy wƺyſtko mnóſtwo ich / wiedli go do Piłátá.
Y pocżęli náń ſkárżyć / mówiąc : Tegośmy ználeźli / że odwrácá lud y zákázuje dáni dáwáć ceſárzowi / powiádájąc : Że on jeſt Chryſtuſem królem.
Y pytáł go Piłát / mówiąc : Tyżeś jeſt on król żydowſki? A on mu odpowiádájąc rzekł : Ty powiádáƺ.
Y rzekł Piłát do przedniejƺych kápłánów y do ludu : Żádnej winy nie znájduję w tym cżłowieku.
Lecż śię oni bárdziej śilili / mówiąc : Iż wzruƺá lud / ucżąc po wƺyſtkiej Judzkiej ziemi / pocżąwƺy od Gálilei áż dotąd.
Tedy Piłát uſłyƺáwƺy o Gálilei / pytáł / jeźliby był cżłowiekiem Gálilejſkim?
A gdy śię dowiedziáł / iż był z páńſtwá Herodowego / odeſłáł go do Herodá / który też w Jeruzálemie był w one dni.
A Herod ujrzáwƺy Jezuſá / urádowáł śię bárdzo ; bo go z dáwná prágnął widzieć / dlátego / iż wiele o nim ſłyƺáł / y ſpodziewáł śię / iż miáł ujrzeć jáki cud od niego ucżyniony.
Y pytáł go wielą ſłów ; ále mu on nic nie odpowiádáł.
A przedniejśi kápłáni y náucżeni w Piśmie ſtáli / potężnie ſkárżąc náń.
Ale wzgárdziwƺy nim Herod z żołnierſtwem ſwem y náśmiáwƺy śię z niego / oblekł go w ƺátę biáłą y odeſłáł go záś do Piłátá.
Y ſtáli śię ſobie przyjáćiołmi Piłát z Herodem onegoż to dniá ; bo ſobie byli przedtem nieprzyjáćiołmi.
A Piłát zwołáwƺy przedniejƺych kápłánów y przełożonych y ludu /
Rzekł do nich : Oddáliśćie mi tego cżłowieká / jákoby lud odwrácáł : á oto já przed wámi pytájąc go / żádnej winy nie ználázłem w tym cżłowieku z tego / co náń ſkárżyćie ;
Ale áni Herod / bom wáſ odeſłáł do niego / á oto nic mu śię godnego śmierći nie ſtáło ;
Przetoż ſkáráwƺy wypuƺcżę go.
A muśiáł im Piłát wypuƺcżáć jednego ná święto.
Tedy záwołáło ſpołem wƺyſtko mnóſtwo / mówiąc : Stráć tego á wypuść nám Bárábbáƺá!
Który był dlá niejákiego rozruchu w mieśćie ucżynionego / y dlá mężobójſtwá wſádzony do więzieniá.
Tedy Piłát znowu mówił / chcąc wypuśćić Jezuſá.
Ale oni przećię wołáli / mówiąc : Ukrzyżuj / ukrzyżuj go!
A on po trzećie rzekł do nich : Y cóż wżdy ten złego ucżynił? Żádnej winy śmierći nie ználázłem w nim ; przetoż ſkáráwƺy wypuƺcżę go.
A oni przećię nálegáli głoſy wielkimi / żądájąc / áby był ukrzyżowány ; y zmácniáły śię głoſy ich y przedniejƺych kápłánów.
A ták Piłát przyſądził / áby śię doſyć ſtáło żądnośći ich.
Y wypuśćił im onego / który był dlá rozruchu y mężobójſtwá wſádzony do więzieniá / o którego prośili ; ále Jezuſá podáł ná wolę ich.
Gdy go tedy wiedli / uchwyćiwƺy Szymoná niektórego Cyrenejcżyká / idącego z polá / włożyli náń krzyż / áby go nióſł zá Jezuſem.
Y ƺło zá nim wielkie mnóſtwo ludu y niewiáſt / które go płákáły y nárzekáły.
Ale Jezuſ obróćiwƺy śię do nich / rzekł : Córki Jeruzálemſkie! nie płácżćie nádemną / ále rácżej ſáme nád ſobą płácżćie y nád dziátkámi wáƺemi.
Abowiemći oto idą dni / których będą mówić : Błogoſłáwione niepłodne / y żywoty / które nie rodziły / y pierśi / które nie kármiły.
Tedy pocżną mówić górom : Pádnijćie ná náſ! á págórkom : Przykryjćie náſ!
Abowiem poniewáż śię to ná zielonem drzewie dzieje / á cóż będzie ná ſuchem?
Wiedzieni też byli y inni dwáj złocżyńcy / áby weſpół z nim ſtráceni byli.
A gdy przyƺli ná miejſce / które zowią trupich głów / tám go ukrzyżowáli / y onych złocżyńców / jednego po práwicy / á drugiego po lewicy.
Tedy Jezuſ rzekł : Ojcże! odpuść im : boć nie wiedzą / co cżynią. A rozdzieliwƺy ƺáty jego / loſ o nie miotáli.
Y ſtáł lud / przypátrując śię / á náśmiewáli śię z niego y przełożeni z nimi / mówiąc : Inƺych rátowáł / niechże rátuje ſámego śiebie / jeźliże on jeſt Chryſtuſ / on wybrány Boży.
Náśmiewáli śię też z niego y żołnierze / przyſtępując á ocet mu podáwájąc /
Y mówiąc : Jeźliś ty jeſt król Żydowſki / rátujże ſámego śiebie.
A był też y nápiſ nápiſány nád nim literámi Greckiemi y Łáćińſkiemi y Żydowſkiemi : Tenći jeſt on król Żydowſki.
A jeden z onych złocżyńców / którzy z nim wiśieli / urągáł mu / mówiąc : Jeźliżeś ty jeſt Chryſtuſ / rátujże śiebie y náſ.
A odpowiádájąc drugi / gromił go mówiąc : Y ty śię Bogá nie boiƺ / choćiáżeś jeſt w temże ſkázániu?
A myć záiſte ſpráwiedliwie ; ( bo godną zápłátę zá ucżynki náƺe bierzemy ; ) ále ten nic złego nie ucżynił.
Y rzekł do Jezuſá : Pánie! pomnij ná mnie / gdy przyjdzieƺ do króleſtwá twego.
A Jezuſ mu rzekł : Zápráwdę powiádám tobie / dziś zemną będzieƺ w ráju.
A było około ƺóſtej godziny / y ſtáłá śię ćiemność po wƺyſtkiej ziemi áż do godziny dziewiątej.
Y záćmiło śię ſłońce / á záſłoná kośćielná rozerwáłá śię w pół.
A Jezuſ záwołáwƺy głoſem wielkim / rzekł : Ojcże! w ręce twoje polecám duchá mojego ; á to rzekłƺy / ſkonáł.
A widząc ſetnik / co śię dziáło / chwálił Bogá / mówiąc : Zápráwdę cżłowiek to był ſpráwiedliwy.
Tákże y wƺyſtek lud / który śię był zƺedł ná to dziwowiſko / widząc / co śię dziáło / bijąc śię w pierśi ſwoje / wrácáł śię.
A znájomi jego wƺyſcy z dáleká ſtáli / y niewiáſty / które zá nim były przyƺły z Gálilei / przypátrując śię temu.
A oto mąż / imieniem Józef / który był ſenátorem / mąż dobry y ſpráwiedliwy /
Który był nie zezwolił ná rádę y ná ucżynek ich / z Arymátyi / miáſtá Judzkiego / który też ocżekiwáł króleſtwá Bożego ;
Ten przyƺedłƺy do Piłátá / prośił o ćiáło Jezuſowe.
Y zdjąwƺy je / obwinął je prześćierádłem á położył je w grobie w opoce wykowánym / w którym jeƺcże nikt nigdy nie był położony.
A był dzień przygotowániá / y ſábát náſtáwáł.
Poƺedłƺy też zá nim y niewiáſty / które były z nim przyƺły z Gálilei / oglądáły grób / y jáko było położone ćiáło jego.
A wróćiwƺy śię / nágotowáły wonnych rzecży y máśći ; ále w ſábát odpocżynęły według przykázániá.
