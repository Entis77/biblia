A práwie ná ten cżás byli przytomni niektórzy / oznájmując mu o Gálilejcżykách / których krew Piłat pomięƺał z ofiárámi ich.
A JEzus odpowiádájąc / rzekł im ; Mniemaćie że ći Gálilejcżycy nád wƺyſtkie inne Gálilejcżyki grzeƺniejƺymi byli / iż tákowe rzecży ućierpieli?
Bynamniey / mówię wam. Y owƺem / jeſli nie będźiećie pokutować / wƺyſcy tákże poginiećie.
Albo ośmnaśćie onych ná które upádłá wieża w Syloe / y pobiłá je ; mniemaćie żeby ći winniejƺymi byli nád wƺyſtkie ludźie mieƺkájące w Jeruzalem?
Bynamniey / mówię wam. Y owƺem jeſli pokutowáć nie będźiećie / wƺyſcy tákże poginiećie.
Y powiedźiał <i>im</i> to podobieńſtwo : <i>Cżłowiek</i> niektóry miał figowe drzewo wſádzone ná winnicy ſwojey ; á przyƺedƺy / ƺukał ná nim owocu / ále nie ználazł.
Tedy rzekł do winiarzá : Oto po trzy látá przychodzę ƺukájąc owocu ná tym drzewie figowym / ále nie znájduję. Wytniże je ; bo przecżże tę źiemię <i>próżno</i> záſtępuje?
Ale on odpowiádájąc / rzekł mu : Pánie / zániechaj go jeƺcże y ná ten rok / áż je okopię y obłożę gnojem.
Owa ſnadź przynieśie owoc : á jeſli nie / potym je wytnieƺ.
Y naucżał w jedney bóżnicy w Sábbát.
A oto byłá <i>tám</i> niewiáſtá / która miáłá duchá niemocy ośmnaśćie lat : á byłá ſkurcżona / ták / iż śię żadną miárą nie mogłá rozproſtowáć.
Tę gdy ujrzał JEzus / záwołał jey do śiebie / y rzekł jey : Niewiáſto / uwolnionaś od niemocy twojey.
Y włożył ná nię ręce. A zárázem rozprośćiłá śię / y chwaliłá Bogá.
Tedy odpowiádájąc Przełożony nád bóżnicą / który śię bárzo gniewał że JEzus w Sábbát uzdrawiał / rzekł do ludu : Sześć dni jeſt w które trzebá robić : w te tedy dni przychodząc lecżćie śię / á nie w dźień Sábbátu.
Ale mu odpowiedźiał PAn / y rzekł : Obłudniku! ázaż káżdy z was w Sábbát nie odwięzuje wołu ſwego / álbo oſłá ſwego od żłobu / á nie wiedźie żeby go nápojił.
A tá córká Abráhámowá / którą był związał ƺátan oto już ośmnaśćie lat / zaż nie miáłá bydź rozwiązána od tey zwiąſki w dźień Sábbátu?
A gdy on to mówił / záwſtydźili śię wƺyſcy przećiwnicy jego. Ale wƺyſtek lud rádował śię ze wƺyſtkich onych chwalebnych ſpraw / które śię dźiały od niego.
Zátym rzekł <i>JEzus</i> : Cżemuż podobne jeſt króleſtwo Boże? A do cżegoż je przypodobam?
Podobne jeſt źiárnu gorcżycżnemu / które wźiąwƺy cżłowiek / wrzućił do ogrodá ſwego / y roſło : y ſtáło śię drzewem wielkim / á ptaƺkowie niebieſcy cżynili ſobie gniazdá ná gáłęźiách jego.
Y rzekł znowu : Do cżegoż przypodobam króleſtwo Boże?
Podobne jeſt kwáſowi / który wźiąwƺy niewiáſtá / zákryłá go we trzy miáry mąki / áżby wƺyſtko zkwáśniáło.
Y chodźił po miáſtách y miáſtecżkách / náucżájąc / <i>á</i> idąc w drogę do Jeruzalem.
Y rzekł mu niektóry : PAnie / cżyli máło tych jeſt / którzy máją być zbáwieni? A on rzekł do nich :
Uśiłujćie ábyśćie weƺli przez ćiáſną bramę : Abowiem powiádam wam : Wiele ich będą chćieli wniść ále nie będą mogli.
Gdy wſtánie goſpodarz y zámknie drzwi / á pocżniećie ſtać przede drzwiámi y kołátáć we drzwi / mówiąc : PAnie / PAnie! otwórz nam : Tedy <i>on</i> odpowiádájąc rzecże wam : Nie znam was z kądeśćie.
Tedy pocżniećie mówić : Jadáliſmy przed tobą / y pijáli / y ucżyłeś ná ulicách náƺych.
A on rzecże : Powiádam wam / nie znam was z kądeśćie. Odſtąpćie ode mnie wƺyſcy / którzy cżynićie niepráwość.
Tám będźie płácż y zgrzytánie zębów / gdy ujrzyćie Abráhámá / Izááká / y Jákóbá / y wƺyſtkie Proroki w króleſtwie Bożym : á ſámych śiebie precż wyrzuconych.
Y przydą drudzy od Wſchodu y od Zachodu / y od Północy / y od Południá / á uśiędą zá ſtołem w króleſtwie Bożym.
A oto ſą oſtátni / którzy będą pierwƺymi ; á ſą pierwƺy / którzy będą oſtátnimi.
W onże dźień przyſtąpili niektórzy z Fáryzeuƺów / mówiąc mu ; Wynidź / á idź z tąd / bo ćię Herod chce zábić.
Y rzekł im ; Idźćie / á powiedzćie temu liſowi : Oto wyganiam dyabły / y uzdrawiam dźiś y jutro / á trzećiego dniá dokońcżenie wezmę.
Wƺákże muƺę dźiś y jutro y po jutrze odpráwowáć drogę : ábowiem niemoże bydź / áby miał Prorok zginąć / oprócż w Jeruzalem.
Jeruzalem / Jeruzalem / które zábijaƺ Proroki / á kámionujeƺ te którzy do ćiebie bywáją poſłáni! ile kroć chćiałem zgromádźić dźieći twoje / ták jáko kokoƺ <i>zgromadza</i> kurcżętá ſwoje pod ſkrzydłá / á nie chćieliśćie.
Otoż zoſtánie wam dom wáƺ puſty. A záprawdęć wam powiádam / że mię nie ujrzyćie / áż przydźie <i>cżás,</i> gdy rzecżećie ; Błogoſłáwiony który idźie w Imieniu PAńſkim.
