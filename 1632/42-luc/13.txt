A práwie nátencżáſ byli przytomni niektórzy / oznájmując mu o Gálilejcżykách / których krew Piłát pomięƺáł z ofiárámi ich.
A Jezuſ odpowiádájąc / rzekł im : Mniemáćie / że ći Gálilejcżycy nád wƺyſtkie inne Gálilejcżyki grzeƺniejƺymi byli / iż tákowe rzecży ućierpieli?
Bynájmniej / mówię wám ; y owƺem / jeźli nie będziećie pokutowáć / wƺyſcy tákże poginiećie.
Albo ośmnáśćie onych / ná które upádłá wieżá w Syloe y pobiłá je / mniemáćie / żeby ći winniejƺymi byli nád wƺyſtkie ludzie mieƺkájące w Jeruzálemie?
Bynájmniej / mówię wám : y owƺem / jeźli pokutowáć nie będziećie / wƺyſcy tákże poginiećie.
Y powiedziáł im to podobieńſtwo : Cżłowiek niektóry miáł figowe drzewo wſádzone ná winnicy ſwojej / á przyƺedłƺy / ƺukáł ná nim owocu / ále nie ználázł.
Tedy rzekł do winiárzá : Oto po trzy látá przychodzę / ƺukájąc owocu ná tem drzewie figowem / ále nie znájduję. Wytnijże je ; bo przecżże tę ziemię próżno záſtępuje?
Ale on odpowiádájąc rzekł mu : Pánie! zániecháj go jeƺcże y ná ten rok / áż je okopię y obłożę gnojem ;
Owá ſnáć przynieśie owoc / á jeźli nie / potem je wytnieƺ.
Y náucżáł w jednej bóżnicy w ſábát.
A oto byłá tám niewiáſtá / którá miáłá duchá niemocy ośmnáśćie lát / á byłá ſkurcżoná / ták / iż śię żádną miárą nie mogłá rozproſtowáć.
Tę gdy ujrzáł Jezuſ / záwołáł jej do śiebie y rzekł : Niewiáſto! uwolnionáś od niemocy twojej.
Y włożył ná nię ręce / á zárázem rozprośćiłá śię y chwáliłá Bogá.
Tedy odpowiádájąc przełożony nád bóżnicą / który śię bárdzo gniewáł / że Jezuſ w ſábát uzdráwiáł / rzekł do ludu : Sześć dni jeſt / w które trzebá robić ; w te tedy dni przychodząc / lecżćie śię / á nie w dzień ſábátu.
Ale mu odpowiedziáł Pán y rzekł : Obłudniku / ázáż káżdy z wáſ w ſábát nie odwiązuje wołu ſwego / álbo oſłá ſwego od żłobu / á nie wiedzie / żeby go nápoił?
A tá córká Abráhámowá / którą był związáł ƺátán oto już ośmnáśćie lát / záż nie miáłá być rozwiązáná od tej związki w dzień ſábátu?
A gdy on to mówił / záwſtydzili śię wƺyſcy przećiwnicy jego : ále wƺyſtek lud rádowáł śię ze wƺyſtkich onych chwálebnych ſpráw / które śię dziáły od niego.
Zátem rzekł Jezuſ : Komuż podobne jeſt króleſtwo Boże / á do cżegoż je przypodobám?
Podobne jeſt ziárnu gorcżycżnemu / które wziąwƺy cżłowiek / wrzućił do ogrodu ſwego ; y roſło y ſtáło śię drzewem wielkiem / á ptáƺki niebieſkie cżyniły ſobie gniázdá ná gáłęziách jego.
Y rzekł znowu : Do cżegoż przypodobám króleſtwo Boże?
Podobne jeſt kwáſowi / który wziąwƺy niewiáſtá / zákryłá go we trzy miáry mąki / áżby wƺyſtko ſkwáśniáło.
Y chodził po miáſtách y miáſtecżkách / náucżájąc á idąc w drogę do Jeruzálemu.
Y rzekł mu niektóry : Pánie! cżyli máło tych jeſt / którzy máją być zbáwieni? A on rzekł do nich :
Uśiłujćie / ábyśćie weƺli przez ćiáſną brámę ; ábowiem powiádám wám : Wiele ich będą chćieli wnijść / ále nie będą mogli.
Gdy wſtánie goſpodárz y zámknie drzwi / á pocżniećie ſtáć przede drzwiámi / y kołátáć we drzwi / mówiąc : Pánie / Pánie! otwórz nám / tedy on odpowiádájąc rzecże wám : Nie znám wáſ / ſkądeśćie.
Tedy pocżniećie mówić : Jádáliśmy przed tobą y pijáli / y ucżyłeś ná ulicách náƺych.
A on rzecże : Powiádám wám / nie znám wáſ / ſkądeśćie ; odſtąpćie odemnie wƺyſcy / którzy cżynićie niepráwość.
Tám będzie płácż y zgrzytánie zębów / gdy ujrzyćie Abráhámá / Izááká / y Jákóbá / y wƺyſtkie proroki w króleſtwie Bożem / á ſámych śiebie precż wyrzuconych.
Y przyjdą drudzy od wſchodu y od záchodu / y od północy / y od południá / á uśiądą zá ſtołem w króleſtwie Bożem.
A oto ſą oſtátni / którzy będą pierwƺymi / á ſą pierwśi / którzy będą oſtátnimi.
W onże dzień przyſtąpili niektórzy z Fáryzeuƺów / mówiąc mu : Wynijdź á idź ſtąd ; bo ćię Herod chce zábić.
Y rzekł im : Idźćie / á powiedzćie temu liſowi : Oto wygániám dyjábły / y uzdráwiám dziś y jutro / á trzećiego dniá dokońcżenie wezmę.
Wƺákże muƺę dziś y jutro y pojutrze odpráwiáć drogę : ábowiem niemoże być / áby miáł prorok zginąć / oprócż w Jeruzálemie.
Jeruzálem! Jeruzálem! które zábijáƺ proroki / á kámionujeƺ te / którzy do ćiebie bywáją poſłáni ; ilekroć chćiáłem zgromádzić dzieći twoje / ták jáko kokoƺ zgromádzá kurcżętá ſwoje pod ſkrzydłá / á nie chćieliśćie.
Otoż zoſtánie wám dom wáƺ puſty. A zápráwdęć wám powiádám / że mię nie ujrzyćie / áż przyjdzie cżáſ / gdy rzecżećie : Błogoſłáwiony / który idzie w imieniu Páńſkim.
