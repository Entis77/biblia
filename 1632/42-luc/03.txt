A roku piętnáſtego pánowániá Tyberyjuƺá Ceſárzá / gdy Poncki Piłát był ſtároſtą Judzkim / á Herod Tetrárchą Gálilejſkim / á Filip / brát jego / Tetrárchą Iturejſkim y kráiny Tráchonitſkiej / á Lizánijáƺ Tetrárchą Abileńſkim /
Zá nájwyżƺych kápłánów Annáƺá y Káifáƺá / ſtáło śię ſłowo Boże do Jáná / Zácháryjáƺowego ſyná / ná puƺcży.
Y przyƺedł do wƺyſtkiej kráiny leżącej około Jordánu / káżąc chrzeſt pokuty ná odpuƺcżenie grzechów.
Jáko nápiſáno w kśięgách proroctw Izájáƺá proroká / mówiącego : Głoſ wołájącego ná puƺcży ; gotujćie drogę Páńſką / proſte cżyńćie śćieżki jego.
Káżdy pádół będźie wypełniony / á káżdá górá y págórek będźie zniżony / y miejſcá krzywe wyproſtują śię / á oſtre drogi będą głádkiemi ;
Y oglądá wƺelkie ćiáło zbáwienie Boże.
Mówił tedy ludowi / który wychodźił / áby był ochrzcżony od niego : Rodzáju jáƺcżurcży! któż wám pokázáł / żebyśćie ućiekáli przed przyƺłym gniewem?
Przynośćież tedy owoce godne pokuty á nie pocżynájćie mówić ſámi w ſobie : Ojcá mámy Abráhámá ; ábowiem powiádám wám / że Bóg może y z tych kámieni wzbudźić dźieći Abráhámowi.
A już śiekierá do korzeniá drzew przyłożoná jeſt ; przetoż káżde drzewo / które nie przynośi owocu dobrego / bywá wyćięte y w ogień wrzucone.
Y pytáł go lud mówiąc : Cóż tedy cżynić będźiemy?
A on odpowiádájąc rzekł im : Kto má dwie ſuknie / niecháj udźieli temu / co niemá ; á kto má pokárm / niech tákże ucżyni.
Przyƺli też y celnicy / áby byli chrzcżeni / y rzekli do niego : Náucżyćielu! á my cóż cżynić będźiemy?
A on rzekł do nich : Nic więcej nie wyćiągájćie nád to / co wám poſtánowiono.
Pytáli go też y żołnierze / mówiąc : A my co cżynić będźiemy? Y rzekł do nich : Nikomu gwáłtu nie cżyńćie / y nikogo nie potwárzájćie / á przeſtáwájćie ná żołdźie wáƺym.
A gdy lud ocżekiwáł / y myślili wƺyſcy w ſercách ſwych o Jánie / jeźliby ſnáć on nie był CHryſtuſem /
Odpowiedźiáł Ján wƺyſtkim / mówiąc : Jáć wáſ chrzcżę wodą ; lecż idźie mocniejƺy nád mię / któremum nie jeſt godźien rozwiązáć rzemyká u bótów jego ; ten wáſ chrzćić będźie Duchem Świętym y ogniem.
Którego łopátá jeſt w ręku jego / á wycżyśći bojewiſko ſwoje / y zgromádźi pƺenicę do gumná ſwego / ále plewy ſpáli ogniem nieugáƺonym.
A ták wiele y innych rzecży nápominájąc / odpowiádáł ludowi.
A Herod / Tetrárchá / będąc ſtrofowány od niego dlá Herodyjády / żony Filipá / brátá jego / y dlá wƺyſtkich złych ſpráw / które cżynił Herod.
Przydáł y to náde wƺyſtko / iż wſádźił Jáná do więźieniá.
Y ſtáło śię / gdy był ochrzcżony wƺyſtek lud / y gdy JEzuſ był ochrzcżony / y modlił śię / że śię niebo otworzyło ;
Y zſtąpił náń Duch Święty w kƺtáłćie ćieleſnym jáko gołębicá / y ſtáł śię głoſ z niebá / mówiąc : Tyś jeſt on Syn mój miły ; w tobie mi śię upodobáło.
