Y ſtáło śię z onych dni dniá jednego / gdy ucżył lud w kośćiele y kázáł Ewángeliję / że nádeƺli przedniejśi kápłáni y náucżeni w Piśmie z ſtárƺymi /
Y rzekli do niego / mówiąc : Powiedz nám / którą mocą to cżyniƺ / álbo kto jeſt ten / coć dáł tę moc?
A on odpowiádájąc / rzekł do nich : Spytám y já wáſ o jednę rzecż / á powiedzćie mi :
Chrzeſt Jáná byłli z niebá / cżyli z ludźi?
A oni myślili ſámi w ſobie / mówiąc : Jeźli powiemy / z niebá / rzecże : Cżemużeśćie mu tedy nie wierzyli?
Jeźliż záśię rzecżemy / z ludźi / wƺyſtek lud ukámionuje náſ / poniewáż zá pewne máją / że Ján jeſt prorokiem.
Y odpowiedźieli / że nie wiedzą / ſkądby był.
A JEzuſ im rzekł : Y já wám nie powiem / którą mocą to cżynię.
Y pocżął do ludu mówić to podobieńſtwo : Cżłowiek niektóry náſádźił winnicę / y nájął ją winiárzom / y odjecháł precż ná cżáſ niemáły.
A cżáſu ſwego poſłáł ſługę do onych winiárzy / áby mu dáli z pożytku onej winnicy ; ále oni winiárze ubiwƺy go / odeſłáli próżnego.
Y poſłáł záśię drugiego ſługę ; ále oni y tego ubiwƺy y zelżywƺy / odeſłáli próżnego.
Y poſłáł záśię trzećiego ; ále oni y tego zrániwƺy / wyrzućili precż.
A ták rzekł Pán onej winnicy : Cóż ucżynię? poślę ſyná mego miłego / ſnáć / gdy tego ujrzą / záwſtydzą śię.
Ale winiárze ujrzáwƺy go / rzekli między ſobą / mówiąc : Tenći jeſt dźiedźic ; pójdźćie zábijmy go / áby náƺe było dźiedźictwo.
Y wypchnąwƺy go precż z winnicy / zábili. Cóż im tedy ucżyni Pán onej winnicy?
Przyjdźie á potráći one winiárze / á winnicę oddá innym. Co oni uſłyƺáwƺy / rzekli : Nie dáj tego Boże!
Lecż on ſpojrzáwƺy ná nie / rzekł : Cóż tedy jeſt ono / co nápiſáno : Kámień / który odrzućili budujący / ten śię ſtáł głową węgielną?
Wƺelki / który upádnie ná ten kámień / roztrąći śię / á ná kogoby on upádł / zetrze go.
Y ſtáráli śię przedniejśi kápłáni y náucżeni w Piśmie / jákoby náń ręce wrzućili onejże godźiny / ále śię ludu báli ; ábowiem poználi / iż przećiwko nim wyrzekł to podobieńſtwo.
A ták podſtrzegájąc go / poſłáli ƺpiegi / którzy zmyśláli / jákoby byli ſpráwiedliwymi / áby go podchwyćili w mowie jego / á potem áby go podáli zwierzchnośći y w moc ſtárośćiną.
Y pytáli go / mówiąc : Náucżyćielu! wiemy / że dobrze mówiƺ y ucżyƺ / áni przyjmujeƺ oſób ; ále drogi Bożej w práwdźie ucżyƺ.
Godźili śię nám dáć cżynƺ ceſárzowi / cżyli nie?
Ale on obácżywƺy chytrość ich / rzekł do nich : Cżemuż mię kuśićie?
Ukáżćie mi groƺ ; cżyj má obráz y nápiſ? A odpowiádájąc rzekli : Ceſárzá.
Zátem on im rzekł : Oddáwájćież tedy / co jeſt ceſárſkiego / ceſárzowi / á co jeſt Bożego / Bogu.
Y nie mogli go podchwyćić w mowie jego przed ludem / á zádźiwiwƺy śię odpowiedźi jego / umilknęli.
A przyƺedłƺy niektórzy z Sáduceuƺów / ( którzy przecżą / y mówią / iż nie máƺ zmártwychwſtániá ) pytáli go /
Mówiąc : Náucżyćielu! Mojżeƺ nám nápiſáł : Jeźliby komu brát umárł / májąc żonę / á umárłby bez dźiátek / áby brát jego pojął onę żonę / á wzbudźił náśienie brátu ſwemu.
Było tedy śiedm bráći / z których pierwƺy pojąwƺy żonę / umárł bez dźiátek.
Y pojął wtóry onę żonę / á umárł y ten bez dźiátek.
Potem ją pojął y trzeći / tákże y oni wƺyſcy śiedmiu / á nie zoſtáwiwƺy dźiátek / pomárli.
Po wƺyſtkich też umárłá y oná niewiáſtá.
Przetoż przy zmártwychwſtániu któregoż z nich oná będźie żoną / poniewáż śiedmiu ich miáło ją zá żonę?
Tedy odpowiádájąc / rzekł im JEzuſ : Synowie tego wieku żenią śię y zá mąż wydáją.
Ale ći / którzy godni ſą / áby doſtąpili onego wieku / y powſtáną od umárłych / áni śię żenić áni zá mąż dáwáć będą.
Abowiem umieráć więcej nie będą mogli ; bo będą równi Aniołom / będąc ſynámi Bożymi / gdyż ſą ſynámi zmártwychwſtániá.
A iż umárli zmártwychwſtáną / y Mojżeƺ pokázáł przy onym krzáku / gdy zowie Páná Bogá Bogiem Abráhámowym y Bogiem Izáákowym y Bogiem Jákóbowym.
A Bógći nie jeſt Bogiem umárłych / ále żywych ; bo jemu wƺyſcy żyją.
Tedy odpowiádájąc niektórzy z náucżonych w Piśmie / rzekli : Náucżyćielu! dobrześ powiedźiáł.
Y nie śmieli go więcej o nic pytáć.
Y rzekł do nich : Jákoż powiádáją / że CHryſtuſ jeſt ſynem Dawidowym?
A ſám Dawid mówi w kśięgách Pſálmów : Rzekł Pán Pánu memu : Siądź po práwicy mojej /
Aż położę nieprzyjáćioły twoje podnóżkiem nóg twoich.
Poniewáż go tedy Dawid názywá Pánem / y jákoż jeſt ſynem jego?
A gdy ſłucháł wƺyſtek lud / rzekł ucżniom ſwoim :
Strzeżćie śię náucżonych w Piśmie / którzy chcą chodźić w ƺátách długich / y miłują pozdráwiániá ná rynkách y pierwƺe ſtołki w bóżnicách / y pierwƺe miejſcá ná wiecżerzách ;
Którzy pożeráją domy wdów / á to pod pokrywką długich modlitw : ćić odnioſą ćiężƺy ſąd.
