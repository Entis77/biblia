A potem náznácżył Pán y drugich śiedmdźieśiąt / y rozeſłáł je po dwóch przed oblicżem ſwojim do káżdego miáſtá y miejſcá / do którego ſám przyjść miał.
Y mówił im : Żniwoć wpráwdźie wielkie / ále robotników máło : prośćież tedy Páná żniwá / áby wypchnął robotników ná żniwo ſwoje.
Idźćież : Oto ja was poſyłám jáko báránki w pośród wilków.
Nie nośćież mieƺká / áni táiſtry / áni obuwiá / y nikogo w drodze nie pozdráwiájćie ;
A do któregokolwiek domu wnijdźiećie / náprzód mówćie : Pokój temu domowi.
A jeſliby tám był który ſyn pokoju / odpocżnie nád nim pokój wáƺ ; á jeſliż nie / wróći śię do was.
A w tymże domu zoſtáńćie / jedząc y pijąc to / co máją ; ábowiem godźien jeſt robotnik zápłáty ſwojey : nie przechodźćie śię z domu do domu.
A do któregobyśćiekolwiek miáſtá weƺli / á przyjęliby was / jedzćie / co przed was położą :
Y uzdráwiájćie niemocnych / którzyby w nim byli / á mówćie im : Przybliżyło śię do was króleſtwo Boże.
A do któregobyśćiekolwiek miáſtá weƺli / á nieprzyjętoby was / wyƺedłƺy ná ulice jego / mówćie :
Y proch / który przylgnął do nas z miáſtá wáƺego / otrząſámy ná was ; wƺákże to wiedzćie / że śię do was przybliżyło króleſtwo Boże.
A mówię wám : Iż Sodomie w on dźień lżej będźie / niż onemu miáſtu.
Biádá tobie / Chorázynie! biádá tobie / Betſáido! bo gdyby śię były w Tyrze y w Sydonie te cudá ſtáły / które śię ſtáły w was / dáwnoby były w worze y w popiele śiedząc pokutowáły.
Dlátego Tyrowi y Sydonowi lżej będźie ná ſądźie / niżeli wám.
A ty / Kápernáum! któreś áż do niebá wywyżƺone / áż do piekłá ſtrącone będźieƺ.
Kto was ſłuchá / mnie ſłuchá : á kto wámi gárdźi / mną gárdźi ; á kto mną gárdźi / gárdźi onym / który mię poſłáł.
A ták wróćili śię oni śiedmdźieśiąt z weſelem / mówiąc : Pánie! y dyjáblić śię nám poddáwáją w imieniu twojim.
Tedy im rzekł : Widźiáłem ƺátáná / jáko błyſkáwicę z niebá ſpádájącego.
Oto wám dáję moc / ábyśćie deptáli po wężách y po niedźwiádkách y po wƺyſtkiej mocy nieprzyjáćielſkiej / á nic was nie uƺkodźi.
Wƺákże nie rádujćie śię z tego / iż śię wám duchy poddáwáją ; ále rácżej rádujćie śię / że imioná wáƺe nápiſáne ſą w niebieśiech.
Onejże godźiny rozrádowáł śię JEzuſ w duchu / y rzekł : Wyſłáwiám ćię / Ojcże / Pánie niebá y źiemi! żeś te rzecży zákrył przed mądrymi y roztropnymi / á objáwiłeś je niemowlątkom ; zápráwdę / Ojcże! że śię ták upodobáło tobie.
Wƺyſtkie rzecży dáne mi ſą od Ojcá mego / á nikt nie zná / kto jeſt Syn / tylko Ojćiec / y kto jeſt Ojćiec / tylko Syn / á komuby chćiáł Syn objáwić.
Tedy obróćiwƺy śię do ucżniów / rzekł im z oſobná : Błogoſłáwione ocży / które widzą / co wy widźićie.
Bo powiádám wám / iż wiele proroków y królów żądáli widźieć / co wy widźićie / ále nie widźieli ; y ſłyƺeć / co wy ſłyƺyćie / ále nie ſłyƺeli.
A oto niektóry zakonnik powſtáł / kuƺąc go y mówiąc : Náucżyćielu! co cżyniąc odźiedźicżę żywot wiecżny?
A on rzekł do niego : W zakonie co nápiſáno / jáko cżytáƺ?
A on odpowiádájąc rzekł : Będźieƺ miłowáł Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey / y ze wƺyſtkiej śiły twojey / y ze wƺyſtkiej myśli twojey ; á bliźniego twego / jáko ſámego śiebie.
Y rzekł mu : Dobrześ odpowiedźiáł ; to cżyń / á będźieƺ żył.
A on chcąc ſámego śiebie uſpráwiedliwić / rzekł do JEzuſá : Y któż jeſt mój bliźni?
Ale JEzuſ odpowiádájąc rzekł : Cżłowiek niektóry zſtępowáł z Jeruzálemu do Jerychá / y wpádł między zbójców / którzy złupiwƺy go y rány mu zádáwƺy / odeƺli / ná pół umárłego zoſtáwiwƺy.
Y przydáło śię / że kápłán niektóry ƺedł tąż drogą / á ujrzáwƺy go / pominął.
Tákże y Lewitá / doſtáwƺy śię ná ono miejſce / á przyƺedłƺy y ujrzáwƺy go / pominął.
Ale Sámárytánin niektóry jádąc / przyjecháł do niego / á ujrzáwƺy / użálił śię go.
A przyſtąpiwƺy záwiązáł rány jego / á náláwƺy oliwy y winá / y włożywƺy go ná bydlę ſwoje / wiódł go do goſpody / y miał ſtáránie o nim.
A názájutrz odjeżdżájąc / wyjął dwá groƺe / y dáł goſpodárzowi / mówiąc mu : Miej o nim ſtáránie / á cokolwiek nádto wynáłożyƺ / ja / gdy śię wrócę / oddám ći.
Któryż tedy z tych trzech zdá śię tobie bliźnim być onemu / co był wpádł między zbójców?
A on rzekł : Ten / który ucżynił miłośierdźie nád nim. Rzekł mu tedy JEzuſ : Idźże / y ty ucżyń tákże.
Y ſtáło śię / gdy oni ƺli / że on wƺedł do niektórego miáſtecżká / á niewiáſtá niektóra / imieniem Mártá / przyjęłá go do domu ſwego.
A tá miáłá śioſtrę / którą zwáno Máryją / która też uśiádłƺy u nóg JEzuſowych / ſłucháłá ſłów jego.
Ale Mártá roztárgnioná byłá około rozmáitey poſługi ; która przyſtąpiwƺy / rzekłá : Pánie! y nie dbáƺ / że śioſtrá mojá mnie ſámę zoſtáwiłá / ábym poſługiwáłá? Rzecżże jey / áby mi pomogłá.
A odpowiádájąc JEzuſ rzekł jey : Márto / Márto! troƺcżyƺ śię y kłopocżeƺ śię około wielu rzecży ;
Aleć jednego potrzebá. Lecż Máryjá dobrą cżąſtkę obráłá / która od niej odjętá nie będźie.
