A potem náznácżył Pán y drugich śiedmdźieśiąt / y rozeſłáł je po dwóch przed oblicżem ſwojim do káżdego miáſtá y miejſcá / do którego ſám przyjść miáł.
Y mówił im : Żniwoć wpráwdźie wielkie / ále robotników máło : prośćież tedy Páná żniwá / áby wypchnął robotników ná żniwo ſwoje.
Idźćież : Oto já wáſ poſyłám jáko báránki w pośród wilków.
Nie nośćież mieƺká / áni táiſtry / áni obuwiá / y nikogo w drodze nie pozdráwiájćie ;
A do któregokolwiek domu wnijdźiećie / náprzód mówćie : Pokój temu domowi.
A jeźliby tám był który ſyn pokoju / odpocżnie nád nim pokój wáƺ ; á jeźliż nie / wróći śię do wáſ.
A w tymże domu zoſtáńćie / jedząc y pijąc to / co máją ; ábowiem godźien jeſt robotnik zápłáty ſwojej : nie przechodźćie śię z domu do domu.
A do któregobyśćiekolwiek miáſtá weƺli / á przyjęliby wáſ / jedzćie / co przed wáſ położą :
Y uzdráwiájćie niemocnych / którzyby w nim byli / á mówćie im : Przybliżyło śię do wáſ króleſtwo Boże.
A do któregobyśćiekolwiek miáſtá weƺli / á nieprzyjętoby wáſ / wyƺedłƺy ná ulice jego / mówćie :
Y proch / który przylgnął do náſ z miáſtá wáƺego / otrząſámy ná wáſ ; wƺákże to wiedzćie / że śię do wáſ przybliżyło króleſtwo Boże.
A mówię wám : Iż Sodomie w on dźień lżej będźie / niż onemu miáſtu.
Biádá tobie / Chorázynie! biádá tobie / Betſáido! bo gdyby śię były w Tyrze y w Sydonie te cudá ſtáły / które śię ſtáły w wáſ / dáwnoby były w worze y w popiele śiedząc pokutowáły.
Dlátego Tyrowi y Sydonowi lżej będźie ná ſądźie / niżeli wám.
A ty / Kápernáum! któreś áż do niebá wywyżƺone / áż do piekłá ſtrącone będźieƺ.
Kto wáſ ſłuchá / mnie ſłuchá : á kto wámi gárdźi / mną gárdźi ; á kto mną gárdźi / gárdźi onym / który mię poſłáł.
A ták wróćili śię oni śiedmdźieśiąt z weſelem / mówiąc : Pánie! y dyjáblić śię nám poddáwáją w imieniu twojim.
Tedy im rzekł : Widźiáłem ƺátáná / jáko błyſkáwicę z niebá ſpádájącego.
Oto wám dáję moc / ábyśćie deptáli po wężách y po niedźwiádkách y po wƺyſtkiej mocy nieprzyjáćielſkiej / á nic wáſ nie uƺkodźi.
Wƺákże nie rádujćie śię z tego / iż śię wám duchy poddáwáją ; ále rácżej rádujćie śię / że imioná wáƺe nápiſáne ſą w niebieśiech.
Onejże godźiny rozrádowáł śię JEzuſ w duchu / y rzekł : Wyſłáwiám ćię / Ojcże / Pánie niebá y źiemi! żeś te rzecży zákrył przed mądrymi y roztropnymi / á objáwiłeś je niemowlątkom ; zápráwdę / Ojcże! że śię ták upodobáło tobie.
Wƺyſtkie rzecży dáne mi ſą od Ojcá mego / á nikt nie zná / kto jeſt Syn / tylko Ojćiec / y kto jeſt Ojćiec / tylko Syn / á komuby chćiáł Syn objáwić.
Tedy obróćiwƺy śię do ucżniów / rzekł im z oſobná : Błogoſłáwione ocży / które widzą / co wy widźićie.
Bo powiádám wám / iż wiele proroków y królów żądáli widźieć / co wy widźićie / ále nie widźieli ; y ſłyƺeć / co wy ſłyƺyćie / ále nie ſłyƺeli.
A oto niektóry zákonnik powſtáł / kuƺąc go y mówiąc : Náucżyćielu! co cżyniąc odźiedźicżę żywot wiecżny?
A on rzekł do niego : W zákonie co nápiſáno / jáko cżytáƺ?
A on odpowiádájąc rzekł : Będźieƺ miłowáł Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojej / y ze wƺyſtkiej śiły twojej / y ze wƺyſtkiej myśli twojej ; á bliźniego twego / jáko ſámego śiebie.
Y rzekł mu : Dobrześ odpowiedźiáł ; to cżyń / á będźieƺ żył.
A on chcąc ſámego śiebie uſpráwiedliwić / rzekł do JEzuſá : Y któż jeſt mój bliźni?
Ale JEzuſ odpowiádájąc rzekł : Cżłowiek niektóry zſtępowáł z Jeruzálemu do Jerychá / y wpádł między zbójców / którzy złupiwƺy go y rány mu zádáwƺy / odeƺli / ná pół umárłego zoſtáwiwƺy.
Y przydáło śię / że kápłán niektóry ƺedł tąż drogą / á ujrzáwƺy go / pominął.
Tákże y Lewitá / doſtáwƺy śię ná ono miejſce / á przyƺedłƺy y ujrzáwƺy go / pominął.
Ale Sámárytánin niektóry jádąc / przyjecháł do niego / á ujrzáwƺy / użálił śię go.
A przyſtąpiwƺy záwiązáł rány jego / á náláwƺy oliwy y winá / y włożywƺy go ná bydlę ſwoje / wiódł go do goſpody / y miáł ſtáránie o nim.
A názájutrz odjeżdżájąc / wyjął dwá groƺe / y dáł goſpodárzowi / mówiąc mu : Miej o nim ſtáránie / á cokolwiek nádto wynáłożyƺ / já / gdy śię wrócę / oddám ći.
Któryż tedy z tych trzech zdá śię tobie bliźnim być onemu / co był wpádł między zbójców?
A on rzekł : Ten / który ucżynił miłośierdźie nád nim. Rzekł mu tedy JEzuſ : Idźże / y ty ucżyń tákże.
Y ſtáło śię / gdy oni ƺli / że on wƺedł do niektórego miáſtecżká / á niewiáſtá niektórá / imieniem Mártá / przyjęłá go do domu ſwego.
A tá miáłá śioſtrę / którą zwáno Máryją / którá też uśiádłƺy u nóg JEzuſowych / ſłucháłá ſłów jego.
Ale Mártá roztárgnioná byłá około rozmáitej poſługi ; którá przyſtąpiwƺy / rzekłá : Pánie! y nie dbáƺ / że śioſtrá mojá mnie ſámę zoſtáwiłá / ábym poſługiwáłá? Rzecżże jej / áby mi pomogłá.
A odpowiádájąc JEzuſ rzekł jej : Márto / Márto! troƺcżyƺ śię y kłopocżeƺ śię około wielu rzecży ;
Aleć jednego potrzebá. Lecż Máryjá dobrą cżąſtkę obráłá / którá od niej odjętá nie będźie.
