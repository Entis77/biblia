Y przybliżáli śię do niego wƺyſcy Celnicy y grzeƺnicy / áby go ſłucháli.
Y ƺemráli Fáryzeuƺowie y Náucżeni w piśmie / mówiąc : Ten grzeƺniki przyjmuje / y je z nimi.
Y powiedźiał im to podobieńſtwo / mówiąc :
Któryż z was cżłowiek / gdyby miał ſto owiec / á ſtráćiłby jednę z nich : izali nie zoſtawia onych dźiewiąćidźieśiąt y dźiewiąći ná puƺcży / á nie idźie zá oną która zginęłá áżby ją ználazł?
A ználazƺy / kłádźie ją ná rámioná ſwoje rádując śię.
A przyƺedƺy do domu / zwoływa przyjaćiół y ſąśiad / mówiąc im : Rádujćie śię ze mną : bom ználazł owcę moję / która byłá zginęłá.
Powiádam wam / że táka będźie rádość w niebie nád jednym grzeƺnikiem pokutującym / więcey niż nád dźiewiąćiądźieśiąt y dźiewiąćią ſpráwiedliwych / którzy nie potrzebują pokuty.
Albo która niewiáſtá májąc dźieśięć groƺy / jeſliby ſtráćiłá groƺ jeden : izali nie zápala świece / y nie umiáta domu / á nie ƺuka z pilnośćią / áżby ználázłá?
A ználázƺy / zwoływa przyjaćiółek y ſąśiádek / mówiąc : Rádujćie śię ze mną : ábowiem ználázłám groƺ którym byłá ſtráćiłá.
Ták / powiádam wam / będźie rádość przed Anjoły Bożymi nád jednym grzeƺnikiem pokutującym.
Nad to rzekł : Cżłowiek niektóry miał dwu ſynów.
Y rzekł młodƺy z nich ojcu : Ojcże / daj mi dźiał májętnośći ná mię przypadájącey. Y rozdźielił im májętność.
A po niewielu dni / zebrawƺy wƺyſtko on młodƺy ſyn / odjáchał w dáleką krájinę / y roſproƺył tám májętność ſwoję żyjąc roſpuſtnie.
A gdy wƺyſtko potráćił / ſtał śię głód wielki w oney krájinie ; á on pocżął niedoſtátek ćierpieć.
A ták ƺedƺy / przyſtał do jednego mieƺcżániná oney krájiny ; który go poſłał do folwárku ſwego / áby páſł świnie.
Y żądał nápełnić brzuch ſwój młótem które jadáły świnie : ále mu ich nikt nie dawał.
Potym przyƺedƺy k ſobie / rzekł : O jáko wiele najemników ojcá mego máją doſyć chlebá : á ja od głodu ginę.
Wſtawƺy <i>tedy</i> pójdę do ojcá mego / y rzekę mu : Ojcże / zgrzeƺyłem przećiwko niebu y przed tobą :
Y nie godźienem więcey bydź názywány ſynem twojim / ucżyń mię jáko jednego z najemników twojich.
Tedy wſtawƺy / ƺedł do ojcá ſwego. A gdy on jeƺcże był opodal / ujrzał go oćiec jego ; y użaliwƺy śię <i>go,</i> przybieżał / á padƺy ná ƺyję jego / pocáłował go.
Y rzekł mu ſyn : Ojcże / zgrzeƺyłem przećiwko niebu y przed tobą ; y jużem nie jeſt godźien ábym był názywány ſynem twojim.
Rzekł tedy oćiec do ſług ſwojich ; Przynieśćie onę przednią ƺátę / á oblecżćie go / y dajćie pierśćień ná rękę jego / y obuwie ná nogi <i>jego</i>.
A przywiódƺy ono tłuſte ćielę / zábijćie ; á jedząc bądźmy weſeli.
Abowiem ten ſyn mój umárł był / á záśię ożył ; zginął był / y ználeźiony jeſt : y pocżęli śię weſelić.
Ale ſtárƺy ſyn jego był ná polu. A gdy przychodząc przybliżył śię ku domowi / uſłyƺał muzykę y tańce.
A záwoławƺy jednego z ſług / pytał coby to było?
A on mu powiedźiał ; Brát twój przyƺedł / y zábił oćiec twój ono tłuſte ćielę / iż go zdrowego doſtał.
Y rozgniewał śię á nie chćiał wniść. Ale oćiec jego wyƺedƺy prośił go.
A on odpowiádájąc rzekł ojcu : Oto przez ták wiele lat ſłużę tobie / á nigdym nie przeſtąpił przykazánia twego ; wƺákżeś mi nigdy nie dał koźlęćiá / ábym śię z przyjaćioły mojimi weſelił.
Ale gdy ten ſyn twój / który pożárł májętność twoję z wƺetecżnicámi / przyƺedł ; zábiłeś mu ono tłuſte ćielę.
A on mu rzekł : Synu / tyś záwżdy ze mną ; á wƺyſtkie <i>dobrá</i> moje / twoje ſą.
Lecż trzebá było weſelić śię y rádowáć / że ten brát twój umárł był / á záśię ożył ; y zginął był / á ználeźiony jeſt.
