Y przybliżáli śię do niego wƺyſcy celnicy y grzeƺnicy / áby go ſłucháli.
Y ƺemráli Fáryzeuƺowie y náucżeni w Piśmie / mówiąc : Ten grzeƺniki przyjmuje y je z nimi.
Y powiedźiáł im to podobieńſtwo / mówiąc :
Któryż z was cżłowiek / gdyby miał ſto owiec / á ſtráćiłby jednę z nich / izáli nie zoſtáwiá onych dźiewięćdźieśięćiu y dźiewięćiu ná puƺcży / á nie idźie zá oną / która zginęłá / áżby ją ználázł?
A ználázłƺy kłádźie ją ná rámioná ſwoje / rádując śię.
A przyƺedłƺy do domu / zwołuje przyjáćiół / y ſąśiádów / mówiąc im : Rádujćie śię ze mną ; bom ználázł owcę moję / która byłá zginęłá.
Powiádám wám / że táká będźie rádość w niebie nád jednym grzeƺnikiem pokutującym / więcey niż nád dźiewięćdźieśiąt y dźiewięćiu ſpráwiedliwych / którzy nie potrzebują pokuty.
Albo która niewiáſtá májąc dźieśięć groƺy / jeſliby ſtráćiłá groƺ jeden / izáli nie zápálá świecy / y nie umiátá domu / á nie ƺuká z pilnośćią / áżby ználázłá?
A ználázłƺy / zwołuje przyjáćiółek y ſąśiádek / mówiąc : Rádujćie śię ze mną ; ábowiem ználázłám groƺ / którym byłá ſtráćiłá.
Ták / powiádám wám / będźie rádość przed Anioły Bożymi nád jednym grzeƺnikiem pokutującym.
Nádto rzekł : Cżłowiek niektóry miał dwóch ſynów /
Y rzekł młodƺy z nich ojcu : Ojcże! dáj mi dźiáł májętnośći ná mię przypádájący. Y rozdźielił im májętność.
A po niewielu dniách / zebráwƺy wƺyſtko on młodƺy ſyn / odjecháł w dáleką kráinę / y rozproƺył tám májętność ſwoję / żyjąc rozpuſtnie.
A gdy wƺyſtko potráćił / ſtáł śię głód wielki w oney kráinie / á on pocżął niedoſtátek ćierpieć.
A ták ƺedłƺy / przyſtáł do jednego mieƺcżániná oney kráiny / który go poſłáł do folwárku ſwego / áby páſł świnie.
Y żądáł nápełnić brzuch ſwój młótem / które jádáły świnie ; ále mu nikt nie dáwáł.
Potem przyƺedłƺy do śiebie / rzekł : O jáko wiele nájemników ojcá mego máją doſyć chlebá / á ja od głodu ginę!
Wſtáwƺy tedy / pójdę do ojcá mego y rzekę mu : Ojcże! zgrzeƺyłem przećiwko niebu y przed tobą.
Y nie godźienem więcey być názywány ſynem twoim / ucżyń mię jáko jednego z nájemników twoich.
Tedy wſtáwƺy / ƺedł do ojcá ſwego. A gdy on jeƺcże był opodál / ujrzáł go ojćiec jego / y użáliwƺy śię go / przybieżáł / á pádłƺy ná ƺyję jego / pocáłowáł go.
Y rzekł mu ſyn : Ojcże! zgrzeƺyłem przećiwko niebu y przed tobą / y jużem nie jeſt godźien / ábym był názywány ſynem twoim.
Rzekł tedy ojćiec do ſług ſwoich : Przynieśćie onę przednią ƺátę / á oblecżćie go / y dájćie pierśćień ná rękę jego / y obuwie ná nogi jego.
A przywiódłƺy ono tłuſte ćielę / zábijćie / á jedząc bądźmy weſeli.
Abowiem ten ſyn mój umárł był / á záśię ożył ; zginął był / y ználeźiony jeſt ; y pocżęli śię weſelić.
Ale ſtárƺy ſyn jego był ná polu ; á gdy przychodząc przybliżył śię ku domowi / uſłyƺáł muzykę y táńce ;
A záwołáwƺy jednego z ſług / pytáł / coby to było.
A on mu powiedźiáł : Brát twój przyƺedł / y zábił ojćiec twój ono tłuſte ćielę / iż go zdrowego doſtáł.
Y rozgniewáł śię / á nie chćiáł wnijść ; ále ojćiec jego wyƺedłƺy prośił go.
A on odpowiádájąc / rzekł ojcu : Oto przez ták wiele lát ſłużę tobie / á nigdym nie przeſtąpił przykázániá twego ; wƺákżeś mi nigdy nie dáł koźlęćiá / ábym śię z przyjáćioły moimi weſelił.
Ale gdy ten ſyn twój / który pożárł májętność twoję z wƺetecżnicámi / przyƺedł / zábiłeś mu ono tłuſte ćielę.
A on mu rzekł ; Synu! tyś záwƺe ze mną / á wƺyſtkie dobrá moje twoje ſą.
Lecż trzebá było weſelić śię y rádowáć / że ten brát twój umárł był á záśię ożył / y zginął był á ználeźiony jeſt.
