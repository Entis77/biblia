Y ſtáło śię w one dni / że wyƺedł dekret od ceſárzá Auguſtá / áby popiſáno wƺyſtek świát.
A ten popiſ pierwƺy ſtáł śię / gdy Cyreneuƺ był ſtároſtą Syryjſkim.
Y ƺli wƺyſcy / áby popiſáni byli / káżdy do miáſtá ſwego.
Wſtąpił też y Józef z Gálilei z miáſtá Názáretu do źiemi Judzkiej / do miáſtá Dawidowego / które zowią Betlehem / ( przeto / iż on był z domu y z fámilii Dawidowej ; )
Aby był popiſány z Máryją / poślubioną ſobie máłżonką / którá byłá brzemienną.
Y ſtáło śię / gdy tám byli / wypełniły śię dni / áby porodźiłá.
Y porodźiłá ſyná ſwego pierworodnego ; á uwinęłá go w pieluƺki / y położyłá go w żłobie / przeto / iż miejſcá nie mieli w goſpodźie.
A byli páſterze w onej kráinie w polu nocujący y ſtráż nocną trzymájący nád ſtádem ſwojim.
A oto Anioł Páńſki ſtánął podle nich / á chwáłá Páńſká zewƺąd oświećiłá je / y báli śię bojáźnią wielką.
Y rzekł do nich Anioł : Nie bójćie śię ; bo oto zwiáſtuję wám rádość wielką / którá będźie wƺyſtkiemu ludowi :
Iż śię wám dźiś národźił zbáwićiel / który jeſt CHryſtuſ Pán / w mieśćie Dawidowem.
A to wám będźie zá znák : znájdźiećie niemowlątko uwinione w pieluƺki / leżące w żłobie.
A záráz z onym Aniołem przybyło mnóſtwo wojſk niebieſkich / chwálących Bogá y mówiących :
Chwáłá ná wyſokośćiách Bogu / á ná źiemi pokój / w ludźiách dobre upodobánie.
Y ſtáło śię / gdy odeƺli Aniołowie od nich do niebá / że oniż páſterze rzekli jedni do drugich : Pójdźmyż áż do Betlehemu / á oglądájmy tę rzecż / którá śię ſtáłá / którą nám Pán oznájmił.
A ták ſpieƺąc śię / przyƺli y ználeźli Máryją y Józefá / y ono niemowlątko leżące w żłobie.
Y ujrzáwƺy rozſłáwili to / co im było powiedźiáno o tem dźiećiątku.
A wƺyſcy / którzy ſłyƺeli / dźiwowáli śię temu / co im páſterze powiádáli.
Lecż Máryjá záchowywáłá wƺyſtkie te ſłowá / uwáżájąc je w ſercu ſwojim.
Y wróćili śię páſterze / wielbiąc y chwáląc Bogá ze wƺyſtkiego / co ſłyƺeli y widźieli / ták jáko im było powiedźiáno.
A gdy śię wypełniło ośm dni / áby obrzezáno ono dźiećiątko / tedy imię jego názwáne jeſt JEzuſ / którem było názwáne od Aniołá / pierwej niż śię w żywoćie pocżęło.
Gdy śię też wypełniły dni ocżyƺcżeniá jej według zákonu Mojżeƺowego / przynieśli go do Jeruzálemu / áby go ſtáwili Pánu /
( Ták jáko nápiſáno w zákonie Páńſkim : że wƺelki mężcżyzná / otwierájący żywot / świętym Pánu názwány będźie. )
A żeby oddáli ofiárę według tego / co powiedźiáno w zákonie Páńſkim / párę ſynogárlic / álbo dwoje gołąbiąt.
A oto był cżłowiek w Jeruzálemie / któremu imię było Symeon ; á ten cżłowiek był ſpráwiedliwy y bogobojny / ocżekujący poćiechy Izráelſkiej / á Duch Święty był nád nim.
Y obwieƺcżony był od Bogá przez Duchá Świętego / że nie miáł oglądáć śmierći / áżby pierwej oglądáł CHryſtuſá Páńſkiego.
Ten przyƺedł z nátchnieniá Duchá Świętego do kośćiołá ; á gdy rodźice wnośili dźiećiątko / JEzuſá / áby ucżynili według zwycżáju zákonnego przy nim.
Tedy on wźiąwƺy go ná ręce ſwoje / chwálił Bogá y mówił :
Teráz puƺcżáƺ ſługę twego / Pánie! według ſłowá twego / w pokoju :
Gdyż ocży moje oglądáły zbáwienie twoje /
Któreś zgotowáł przed oblicżem wƺyſtkich ludźi ;
Świátłość ku objáwieniu pogánom / á chwáłę ludu twego Izráelſkiego.
A ojćiec y mátká jego dźiwowáli śię temu / co powiádáno o nim.
Y błogoſłáwił im Symeon / y rzekł do Máryi / mátki jego : Oto ten położony jeſt ná upádek y ná powſtánie wielu ich w Izráelu / y ná znák / przećiw któremu mówić będą.
( Y twoję włáſną duƺę miecż przeniknie / ) áby myśli z wielu ſerc objáwione były.
A byłá Anná prorokini / córká Fánuelowá / z pokoleniá Aſſer / którá byłá bárdzo podeƺłá w látách / y żyłá śiedm lát z mężem od pánieńſtwá ſwego.
A tá byłá wdową / około ośmdźieśiąt y cżterech lát ; którá nie wychodźiłá z kośćiołá / w poſtách y w modlitwách ſłużąc Bogu w nocy y we dnie.
Tá też onejże godźiny nádƺedłƺy / wyznáwáłá Páná / y mówiłá o nim wƺyſtkim / którzy ocżekiwáli odkupieniá w Jeruzálemie.
A ták wykonáwƺy wƺyſtko według zákonu Páńſkiego / wróćili śię do Gálilei / do miáſtá ſwego Názáretu.
A dźiećiątko ono roſło / y umácniáło śię w Duchu / pełne będąc mądrośći / á łáſká Bożá byłá nád nim.
A rodźice jego chádzáli ná káżdy rok do Jeruzálemu ná święto wielkánocne.
A gdy już był we dwunáſtym roku / á oni wſtępowáli do Jeruzálemu według zwycżáju onego świętá ;
Y gdy ſkońcżyli one dni / á już śię wrácáli názád / zoſtáło dźiećię JEzuſ w Jeruzálemie / á tego nie wiedźiáł Józef y mátká jego.
Lecż mniemájąc / że jeſt w towárzyſtwie podróżnem / uƺli dźień drogi / y ƺukáli go między krewnymi y między znájomymi.
A gdy go nie ználeźli / wróćili śię do Jeruzálemu / ƺukájąc go /
Y ſtáło śię po trzech dniách / że go ználeźli śiedzącego w kośćiele w pośrodku doktorów / ſłuchájącego ich y pytájącego ich.
Y zdumiewáli śię wƺyſcy / którzy go ſłucháli / nád rozumem y nád odpowiedźiámi jego.
A ujrzáwƺy go rodźice / zdumieli śię. Y rzekłá do niego mátká jego : Synu! przecżżeś nám to ucżynił? Oto ojćiec twój y já z boleśćią ƺukáliśmy ćię.
Y rzekł do nich : Cóż jeſt / żeśćie mię ƺukáli? Izáliśćie nie wiedźieli / iż w tych rzecżách / które ſą Ojcá mego / já być muƺę?
Lecż oni nie zrozumieli tego ſłowá / które im mówił.
Y zſtąpił z nimi / y przyƺedł do Názáretu / á był im poddány. A mátká jego záchowywáłá wƺyſtkie te ſłowá w ſercu ſwojim.
A JEzuſ pomnáżáł śię w mądrośći / y we wzrośćie y w łáſce u Bogá y u ludźi.
