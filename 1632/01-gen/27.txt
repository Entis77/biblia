Y ſtáło śię / gdy śię zſtárzáł Izáák / y záćmiły śię ocży jego / ták / iż widźieć nie mógł : tedy wezwáł Ezáwá / ſyná ſwego ſtárƺego / y rzekł mu : Synu mój! á on odpowiedźiáł : Owom ja.
Y rzekł Izáák : Otom śię już zſtárzáł / á nie wiem dniá śmierći ſwej.
Przetoż teraz weźmij proƺę nácżyniá twoje / ſájdák twój / y łuk twój / á wynijdź w pole / y ułów mi zwierzynę.
Y nágotuj mi potráwy ſmácżne / w jákich śię kochám / y przynieś mi / á będę jádł / ábyć błogoſłáwiłá duƺá mojá / pierwej / niż umrę.
Ale Rebeká ſłyƺáłá / gdy to mówił Izáák do Ezáwá / ſyná ſwego ; tedy wyƺedł Ezáw ná pole / áby ułowił zwierzynę y przynióſł.
Y rzekłá Rebeká do Jákóbá / ſyná ſwego / mówiąc : Otom ſłyƺáłá / gdy ojćiec twój mówił do Ezáwá / brátá twego / y rzekł :
Przynieś mi co z obłowu / á nágotuj mi potráwy ſmácżne / ábym jádł / y błogoſłáwił ći przed oblicżem Páńſkim / pierwej / niż umrę.
A ták teraz ſynu mój / uſłucháj głoſu mego w tem / co ja rozkázuję tobie.
A ƺedłƺy do trzody / przynieś mi ſtámtąd dwoje koźląt dobrych / á nágotuję z nich potráwy ſmácżne ojcu twemu / jáko rád jádá.
Y zánieśieƺ ojcu twemu / á będźie jádł ; dla tego ábyć błogoſłáwił / pierwej niż umrze.
Tedy rzekł Jákób do Rebeki / mátki ſwej : Oto / Ezáw brát mój / cżłowiek koſmáty / á jám cżłowiek głádki ;
Jeſli mię pomácá ojćiec mój / á będźie rozumiáł / że z niego ƺydzę / przywiodę ná śię przeklęſtwo / á nie błogoſłáwieńſtwo.
Y rzekłá mu mátká jego : Ná mię niech będźie przeklęſtwo twoje / ſynu mój ; tylko uſłucháj głoſu mego / á ƺedłƺy / przynieś mi.
Tedy on ƺedłƺy wźiął / y przynióſł mátce ſwej ; y nágotowáłá mátká jego potráwy ſmácżne / jáko rád jádáł ojćiec jego.
Y wźiąwƺy Rebeká ƺáty Ezáwá / ſyná ſwego ſtárƺego / nájkoƺtowniejƺe / które miáłá u śiebie w domu / oblokłá w nie Jákóbá / ſyná ſwego młodƺego.
A ſkórkámi koźlęcemi obwinęłá ręce jego / y głádkość ƺyi jego.
Y dáłá chleb y potráwy ſmácżne / które nágotowáłá / w ręce Jákóbá ſyná ſwego.
A on wƺedłƺy do ojcá ſwego mówił : Ojcże mój! á on rzekł : Owom ja! Ktoś ty jeſt / ſynu mój?
Y rzekł Jákób do ojcá ſwego : Jám jeſt Ezáw / pierworodny twój. Ucżyniłem / jákoś mi rozkázáł ; wſtáń proƺę / śiądź / á jedz z obłowu mego / áby mi błogoſłáwiłá duƺá twojá.
Y rzekł Izáák do ſyná ſwego : Cóż to jeſt? Prędkoś to ználázł / ſynu mój? á on odpowiedźiáł : Spoſobił to Pán Bóg twój / że mi śię nágodźiło.
Zátem Izáák rzekł do Jákóbá : Przyſtąp ſám / ábym ćię pomácáł / ſynu mój / jeſliś ty jeſt ſyn mój Ezáw / cżyli nie.
Tedy przyſtąpił Jákób do Izááká / ojcá ſwego / który pomácáwƺy go / rzekł : Głoſ jeſt głoſ Jákóbów / ále ręce Ezáwowe.
Y nie poznał go ; ábowiem były ręce jego jáko ręce Ezáwá / brátá jego / koſmáte ; y błogoſłáwił mu.
Y rzekł : Tyżeś jeſt ſyn mój Ezáw? á on odpowiedźiáł : Ja.
Zátem rzekł : Podájże mi / żebym jádł z obłowu ſyná mego / ábyć błogoſłáwiłá duƺá mojá. Tedy mu podáł / y jádł. Przynióſł mu też winá / y pił.
Y rzekł mu Izáák / ojćiec jego : Przyſtąpże teraz á pocáłuj mię / ſynu mój.
Tedy przyſtąpiwƺy pocáłowáł go / á ſkoro pocżuł wonność ƺát jego / błogoſłáwił mu / mówiąc : Oto wonność ſyná mego / jáko wonność polá / któremu błogoſłáwił Pán.
Niechájżeć dá Bóg z roſy niebieſkiej / y z tłuſtośći źiemſkiej / y obfitość zbożá y winá.
Niecháj ći ſłużą ludźie / á niech ći śię kłániáją národowie. Bądź pánem bráći twojey / á niech ći śię kłániáją ſynowie mátki twojey ; którzyby ćię przeklináli / niech będą przeklętymi / á którzyby ćię błogoſłáwili / niech będą błogoſłáwionymi.
Y ſtáło śię / gdy przeſtáł Izáák błogoſłáwić Jákóbowi / y ledwie Jákób odƺedł od oblicżá Izááká / ojcá ſwego / tedy Ezáw brát jego / przyƺedł z łowu ſwego.
Który nágotowáwƺy potráwy ſmácżne / przynióſł je ojcu ſwemu / y mówił do ojcá ſwego : Wſtáńże ojcże mój / á jedz z obłowu ſyná twego / áby mi błogoſłáwiłá duƺá twojá.
Tedy mu rzekł Izáák / ojćiec jego : Któżeś ty? A on rzekł : Jám jeſt ſyn twój / pierworodny twój / Ezáw.
Y zląkł śię Izáák zlęknieniem bárdzo wielkiem / y rzekł : Któż to / á gdźie jeſt ten / co ułowił zwierzynę / y przynióſł mi? y jádłem ze wƺyſtkiego / pierwej / niżeś ty przyƺedł / y błogoſłáwiłem mu / y będźie błogoſłáwionym.
A uſłyƺáwƺy Ezáw ſłowá ojcá ſwego / záwołáł głoſem wielkiem / y był żáłośćią wielką zjęty / y rzekł ojcu ſwemu : Błogoſłáwże też y mnie / ojcże mój.
A on mu rzekł : Przyƺedł brát twój chytrze / y wźiął błogoſłáwieńſtwo twoje.
Tedy Ezáw rzekł : Słuƺnieć názwáno imię jego Jákób / podƺedł mię bowiem już dwá kroć ; pierworodztwo moje wźiął / á teraz oto odnióſł błogoſłáwieńſtwo moje. Y rzekł : Toś mi nie záchowáł błogoſłáwieńſtwá?
Odpowiedźiáł Izáák / y rzekł do Ezáwá : Otom go pánem poſtánowił nád tobą / y wƺyſtkich bráći jego dáłem mu zá ſługi / y zbożem / y winem opátrzyłem go ; á tobie cóż teraz mam ucżynić / ſynu mój?
Y rzekł Ezáw do ojcá ſwego : Izáli tylko jedno błogoſłáwieńſtwo máƺ / ojcże mój? Błogoſłáwże y mnie ; y jámći ſyn twój / ojcże mój. Y podnióſł Ezáw głoſ ſwój / á płákáł.
Y odpowiedźiáł Izáák / ojćiec jego / y rzekł mu : Oto w tłuſtośći źiemi będźie mieƺkánie twoje / y w rośie niebieſkiej z góry.
A z miecżá twego żyć będźieƺ / y brátu twemu będźieƺ ſłużył ; ále ſtánie śię / że y ty pánowáć będźieƺ / y zrzućiƺ járzmo jego z ƺyi twojey.
Przetoż nienáwidźiáł Ezáw Jákóbá dla błogoſłáwieńſtwá / którem mu błogoſłáwił ojćiec jego ; y mówił Ezáw w ſercu ſwem : Przybliżáją śię dni żáłoby ojcá mego / á zábiję Jákóbá / brátá mego.
Y oznájmiono Rebece ſłowá Ezáwá / ſyná jey ſtárƺego / która poſłáwƺy / wezwáłá Jákóbá / ſyná ſwego młodƺego / y rzekłá do niego : Oto Ezáw / brát twój / ćieƺy śię tem / iż ćię zábije.
Przetoż teraz / ſynu mój / uſłucháj głoſu mego / á wſtáwƺy / ućiecż do Lábáná / brátá mego / do Háránu /
Y pomieƺkáj z nim przez jáki cżáſ / áż ućichnie gniew brátá twego /
Aż śię odwróći zápálcżywość brátá twego od ćiebie / y zápomni tego / coś mu ucżynił ; potem ja poślę / á wezmę ćię ſtámtąd ; bo cżemuż mam was obydwóch poſtrádáć jednego dniá?
Y rzekłá Rebeká do Izááká : Obmierzł mi żywot mój dla córek Hetejſkich ; jeſliże y Jákób weźmie ſobie żonę z córek Hetejſkich / jákie ſą córki źiemi tey / cóż mi po żywoćie?
