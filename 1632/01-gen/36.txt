A teć ſą rodzáje Ezáwowe / który jeſt Edom.
Ezáw pojął żony ſwoje z córek Chánánejſkich : Adę / córkę Eloná / Hetejcżyká ; y Oolibámą / córkę Any / córki Sebeoná / Hewejcżyká ;
Y Báſemát / córkę Iſmáelowę / śioſtrę Nebájotowę.
Y urodźiłá Adá Ezáwowi Elifáſá / á Báſemát urodźiłá Rehuelá.
Oolibámá też urodźiłá Jehuſá / y Jelomá / y Korego. Ci ſą ſynowie Ezáwowi / którzy mu śię urodźili w źiemi Chánánejſkiej.
Y wźiął Ezáw żony ſwoje / y ſyny ſwoje / y córki ſwoje / y wƺyſtkie duƺe domu ſwego / y trzody ſwoje / y wƺyſtko bydło ſwoje / y wƺyſtkę májętność ſwoję / której był nábył w źiemi Chánánejſkiej / y odƺedł do źiemi inƺej od Jákóbá / brátá ſwego ;
Bo byłá májętność ich wielká / że nie mogli mieƺkáć poſpołu / y nie mogłá ich znieść źiemiá pielgrzymowániá ich / dlá mnóſtwá ſtád ich.
Y mieƺkáł Ezáw ná górze Seir / á ten Ezáw jeſt Edom.
A teć ſą pokoleniá Ezáwá / ojcá Edomcżyków / ná górze Seir.
Y te ſą imioná ſynów Ezáwowych : Elifáſ / ſyn Ady / żony Ezáwowej / Rehuel / ſyn Báſemáty / żony Ezáwowej.
Synowie záś Elifáſowi byli : Temán / Omár / Sefo / y Gáátán / y Kenáz.
A Támná byłá záłożnicá Elifáſá / ſyná Ezáwowego / y urodźiłá Elifáſowi Amáleká. Ci ſą ſynowie Ady / żony Ezáwowej.
Ci też ſą ſynowie Rehuelowi : Náhát y Zárá / Sámmá y Mezá ; ći byli ſynowie Báſemáty / żony Ezáwowej.
Ci záś byli ſynowie Oolibámy / córki Any / córki Sebeoná / żony Ezáwowej : y urodźiłá Ezáwowi Jehuſá / y Jelomá / y Korego.
Teć ſą kśiążętá z ſynów Ezáwowych / ſynowie Elifáſá pierworodnego Ezáwowego : Kśiążę Temán / kśiążę Omár / kśiążę Sefo / kśiążę Kenáz.
Kśiążę Kore / kśiążę Gáátám / kśiążę Amálek. Teć kśiążętá z Elifáſá poƺły / w źiemi Edomſkiej / ći ſą ſynowie z Ady.
Ci záś ſą ſynowie Rehuelá / ſyná Ezáwowego : Kśiążę Náhát / kśiążę Zárá / kśiążę Sámmá / kśiążę Mezá. Te kśiążętá poƺły z Rehuelá / w źiemi Edomſkiej / ći ſą ſynowie Báſemáty żony Ezáwowej.
Ci záś ſą ſynowie Oolibámy / żony Ezáwowej : Kśiążę Jehuſ / kśiążę Jelom / kśiążę Kore. Te kśiążętá poƺły z Oolibámy / córki Any / żony Ezáwowej.
Ci ſą ſynowie Ezáwowi / y te kśiążętá ich. Onże jeſt Edom.
Ci też ſą ſynowie Seirá Chorejcżyká / mieƺkájący w onej źiemi : Lotán / y Sobál / y Sebeon / y Háná.
Y Dyſſon / y Eſer / y Diſán ; teć ſą kśiążętá Chorejſkie / ſynowie Seirowi w źiemi Edomſkiej.
A ſynowie Lotánowi byli Chory y Hemán ; á śioſtrá Lotánowá Támná.
Synowie záś Sobálowi : Hálwán / y Mánáhát / y Hewál / Sefo / y Onám.
Synowie też Sebeonowi ći ſą : Ajá y Aná. Tenći to Aná / który wynálázł muły ná puƺcży / gdy páſł oſły Sebeoná / ojcá ſwego.
Dźieći záś Anowe te ſą : Dyſon / y Oolibámá / córká Anowá.
A ſynowie Dyſonowi : Hámdán / y Eſebán / y Jetrán / y Chárán.
A ſynowie Eſerowi ſą ći : Báláán / y Záwán / y Akán.
A záśię ſynowie Dyſánowi : Huſ y Arán.
Teć ſą kśiążętá Chorejſkie : Kśiążę Lotán / kśiążę Sobál / kśiążę Sebeon / kśiążę Aná /
Kśiążę Dyſon / kśiążę Eſer / kśiążę Dyſán. Te były kśiążętá Chorejſkie / według kśięſtw ich / w źiemi Seir.
Ci też byli królowie / którzy królowáli w źiemi Edomſkiej / pierwej niż królowáł król nád ſyny Izráelſkimi.
Królowáł tedy w Edom Belá / ſyn Beorów / á imię miáſtá jego Dynhábá.
Y umárł Belá / á królowáł miáſto niego Jobáb / ſyn Zeráchów z Boſry.
Y umárł Jobáb / á królowáł miáſto niego Chuſám / z źiemi Temáńſkiej.
Y umárł Chuſám / á królowáł miáſto niego Hádád / ſyn Bádádów / który poráźił Mádyjáńcżyki / ná polu Moábſkiem / á imię miáſtá jego Háwid.
Y umárł Hádád / á królowáł miáſto niego Sámlá z Máſreki.
Y umárł Sámlá / á królowáł miáſto niego Saul / z Rechobot u rzeki.
Y umárł Saul / á królowáł miáſto niego Bálánán / ſyn Achborów.
Y umárł Bálánán / ſyn Achborów / á królowáł miáſto niego Hádár / á imię miáſtá jego Páhu / á imię żony jego Mehetábel / córká Mátredy / córki Mezáábowej.
Teć ſą imioná kśiążąt Ezáwowych / według ich pokoleniá / y według miejſc ich / y imion ich : Kśiążę Támná / kśiążę Hálwá / kśiążę Jetet.
Kśiążę Oolibámá / kśiążę Elá / kśiążę Pynon.
Kśiążę Kenáz / kśiążę Temán / kśiążę Mábſár.
Kśiążę Mágdyjel / kśiążę Hyrám / te ſą kśiążętá Edomſkie / według mieƺkániá ich / w źiemi ośiádłośći ich. Ten jeſt Ezáw / ojćiec Edomcżyków.
