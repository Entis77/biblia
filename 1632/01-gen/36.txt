A teć ſą rodzáje Ezawowe / który <i>jeſt</i> Edom.
Ezaw pojął żony ſwoje z córek Chánánejſkich : Adę / córkę Eloná Chettejczyká ; y Oolibámę córkę Any / córki Sebeoná Hewejczyká :
Y Báſemát córkę Iſmáelowę / śioſtrę Nebájotowę.
Y urodźiłá Adá Ezawowi Elifáſá / á Báſemát urodźiłá Rehuelá.
Oolibámá też urodźiłá Jehuſá / y Jelomá / y Coráha : ći <i>ſą</i> Synowie Ezawowi / którzy mu śię urodźili w źiemi Chánánejſkiey.
Y wźiął Ezaw żony ſwoje / y ſyny ſwoje / y córki ſwoje / y wƺyſtkie duƺe domu ſwego / y trzody ſwoje / y wƺyſtko bydło ſwoje / y wƺyſtkę májętność ſwoję którey był nábył w źiemi Chánánejſkiey / y odƺedł do źiemie <i>inƺey</i> od Jákóbá brátá ſwego.
Bo byłá májętność ich wielka / że nie mogli mieƺkáć poſpołu / y nie mogłá ich znieść źiemiá pielgrzymowánia ich / dla <i>mnóſtwá</i> ſtad ich.
Y mieƺkał Ezaw ná górze Seir : á ten Ezaw jeſt Edom.
A teć ſą pokolenia Ezawá Ojcá Edomczyków / ná górze Seir.
Y te ſą imioná Synów Ezawowych : Elifás ſyn Ady żony Ezawowey / Rehuel Syn Báſemát żony Ezawowey.
Synowie záś Elifáſowi byli : Temán / Omár / Sefo / y Gáátan / y Kenáz.
A Támná byłá założnicá Elifáſá Syná Ezawowego / y urodźiłá Elifáſowi Amáleká. Ci <i>ſą</i> ſynowie Ady żony Ezawowey.
Ci też <i>ſą</i> Synowie Rehuelowi / Náhát / y Zárá / Sámmá / y Mezá : ći byli Synowie Báſemát żony Ezawowey.
Ci záś byli Synowie Oolibámy córki Any / córki Sebeoná / żony Ezawowey : y urodźiłá Ezawowi / Jehuſá / y Jelomá / y Korego.
Teć <i>ſą</i> Kśiążętá z Synów Ezawowych. Synowie Elifáſá pierworodnego Ezawowego : Kśiążę Temn / Kśiążę Omár / Kśiążę Sefo / Kśiążę Kenáz.
Kśiążę Kore / Kśiążę Gáátám / Kśiążę Amálek : Teć kśiążętá z Elifáſá <i>poƺły</i> w źiemi Edomſkiey / ći <i>ſą</i> ſynowie z Ady.
Ci záś <i>ſą</i> Synowie Rehuelá ſyná Ezawowego : Kśiążę Náhát / Kśiążę Zárá / Kśiążę Sámmá / Kśiążę Mezá : Te Kśiążętá <i>poƺły</i> z Rehuelá / w źiemi Edomſkiey / ći <i>ſą</i> Synowie Báſemát żony Ezawowey.
Ci záś <i>ſą</i> Synowie Oolibámy / żony Ezawowey : Kśiążę Jehus / Kśiążę Jelom / Kśiążę Kore : te Kśiążętá <i>poƺły</i> z Oolibámy córki Any / żony Ezawowey.
Ci <i>ſą</i> Synowie Ezawowi / y te Kśiążętá ich. Onże <i>jeſt</i> Edom.
Ci <i>też ſą</i> Synowie Seirá Chorejczyká / mieƺkájący w oney źiemi : Lothán / y Sobál / y Sebeon / y Háná.
Y Dyſſon / y Eſer / y Diſán ; teć <i>ſą</i> Kśiążętá Chorejſkie Synowie Seirowi w źiemi Edomſkiey.
A Synowie Lotánowi byli Chory y Hemán : á śioſtrá Lotánowá Támná.
Synowie záś Sobálowi : Hálwán / y Mánáhát / y Hewál / Sefo / y Onám.
Synowie też Sebeonowi ći <i>ſą</i> : Ajá / y Aná. Tenći to Aná który wynálazł muły ná puƺczy / gdy páſł oſły Sebeoná Ojcá ſwego.
Dźieći záś Anowe te <i>ſą</i> : Diſon / y Oolibámá / córká Anowá.
A ſynowie Diſonowi : Hámdán / y Eſebán / y Jetrán / y Chárán.
A ſynowie Eſerowi ſą ći : Báláán y Záwán / y Akán.
A záśię ſynowie Diſánowi Hus y Arán.
Teć ſą Kśiążętá Chorejſkie : Kśiążę Lotán / Kśiążę Sobál / Kśiążę Sebeon / Kśiążę Ana.
Kśiążę Diſon / Kśiążę Eſer / Kśiążę Diſán. Te <i>były</i> Kśiążętá Chorejſkie / według Kśięſtw ich w źiemi Seir.
Ci też <i>byli</i> Królowie którzy królowáli w źiemi Edomſkiey / pierwey niż królował król nád ſyny Izráelſkimi.
Królował tedy w Edom Belá ſyn Beorów / á imię miáſtá jego Dinhábá.
Y umárł Belá / á królował miáſto niego Jobáb ſyn Zeráchów z Boſry.
Y umárł Jobáb / á królował miáſto niego Chuſám z źiemie Temáńſkiey.
Y umárł Chuſám / á królował miáſto niego Hádád / ſyn Bádádów / który poráźił Mádyáńczyki ná polu Moábſkim / á imię miáſtá jego Háwid.
Y umárł Hádád / á królował miáſto niego Sámlá z Máſreki.
Y umárł Sámlá / á królował miáſto niego Saul z Rehobot u rzeki.
Y umárł Saul / á królował miáſto niego Bálánán ſyn Achborów.
Y umárł Bálánán ſyn Achborów / á królował miáſto niego Hádár / á imię miáſtá jego Páhu / á imię żony jego Mehetábel córká Mátredy / córki Mezáábowey.
Teć <i>ſą</i> imioná Kśiążąt Ezawowych / według ich pokolenia / y według miejſc ich / y imion ich : Kśiążę Támná / Kśiążę Hálwá / Kśiążę Jetet.
Kśiążę Oolibámá / Kśiążę Elá / Kśiążę Pinon.
Kśiążę Kenáz / Kśiążę Temán / Kśiążę Mábſár.
Kśiążę Mágdijel / Kśiążę Hirám. Te <i>ſą</i> Kśiążętá Edomſkie / według mieƺkánia ich / w źiemi ośiádłośći ich / ten <i>jeſt</i> Ezaw Oćiec Edomczyków.
