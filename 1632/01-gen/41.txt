Y ſtáło śię po wyjśćiu dwóch lát / że śię śniło Fáráonowi / jákoby ſtáł nád rzeką.
A oto z rzeki wychodziło śiedm krów / pięknych ná wejrzeniu y tłuſtych ná ćiele / które śię páſły ná łące.
Oto / też śiedm krów innych wychodziło zá niemi z rzeki / ƺpetnych ná wejrzeniu / y chudych ná ćiele / które ſtáły wedle krów pierwƺych nád brzegiem rzeki.
Y pożárły one krowy ƺpetne ná wejrzeniu y chude ná ćiele / śiedm krów pięknych ná wejrzeniu y tłuſtych ; zátem ocknął śię Fáráo.
A gdy uſnął / śniło mu śię powtóre ; á ono śiedm kłoſów wyráſtáło z jednego źdźbłá / pełnych y cudnych.
Oto / też śiedm kłoſów ćienkich y wyſuƺonych od wiátru wſchodniego wyráſtáło zá nimi.
Y pożárły te kłoſy ćienkie / śiedm onych kłoſów pięknych y zupełnych ; y ocknął śię Fáráo.
A toć był ſen. A gdy było ráno / ſtrwożony był duch jego ; y poſłáwƺy wezwáł wƺyſtkich wieƺcżków Egipſkich y wƺyſtkich mędrców jego / y opowiedziáł im Fáráo ſny ſwoje ; á nie było / ktoby je wyłożył Fáráonowi.
Zátem rzekł przełożony nád podcżáƺymi do Fáráoná / mówiąc : Grzechy moje já dziś przypominám ſobie.
Fáráo rozgniewáwƺy śię ná ſługi ſwe / dáł mię był pod ſtráż do domu hetmáná żołnierzów / mnie y przełożonego nád piekárzámi.
Tám śię nám śnił ſen jednejże nocy / mnie y jemu ; káżdemu według wykłádu ſnu jego śniło śię.
A był tám z námi młodzieniec Hebrejcżyk / ſługá hetmáná żołnierzów / któremuśmy powiedzieli / y wyłożył nám ſny náƺe / káżdemu według ſnu jego wyłożył.
Y ſtáło śię / że jáko nám wyłożył / ták było ; mię przywróćił król ná miejſce moje / á onego obwieśił.
Tedy poſłáwƺy Fáráo / wezwáł Józefá / y prędko go wyprowádzono z więzieniá ; który oſtrzygłƺy śię / y odmieniwƺy ƺáty ſwoje / przyƺedł do Fáráoná.
Y rzekł Fáráo do Józefá : Śnił mi śię ſen / á nie mám / ktoby mi go wyłożył ; álem já o tobie ſłyƺáł / gdy mówiono / że gdy uſłyƺyƺ ſen / umieƺ go wyłożyć.
Y odpowiedziáł Józef Fáráonowi / mówiąc : Oprócż mnie Bóg opowie rzecży ƺcżęśliwe Fáráonowi.
Tedy rzekł Fáráo do Józefá : Zdáło mi śię we śnie / jákobym ſtáł ná brzegu rzeki.
A oto z rzeki wychodziło śiedm krów tłuſtych ná ćiele / y pięknych ná wejrzeniu / á páſły śię ná łące.
Oto / záś wychodziło śiedm krów innych zá niemi / nędznych y ƺpetnych ná wejrzeniu / y chudych ná ćiele ; nie widziáłem we wƺyſtkiej ziemi Egipſkiej ták ƺpetnych.
Y pożárły krowy chude y ƺpetne śiedm krów pierwƺych tłuſtych.
A choć śię doſtáły do wnętrznośći ich / przećię nie było znáć / że śię doſtáły do wnętrznośći ich : bo ná wejrzeniu były ƺpetne / jáko y przedtem ; y ocknąłem śię.
Widziáłem záś we śnie / á oto / śiedm kłoſów wyráſtáło z jednego źdźbłá pełnych y pięknych.
Oto / też śiedm kłoſów ſuchych / ćienkich / y wyſchłych od wiátru wſchodniego / wyráſtáło zá nimi.
Y pożárły te kłoſy ćienkie śiedm onych kłoſów pięknych. Y powiedziáłem to wieƺcżkom ; ále nie było / ktoby mi wyłożył.
Tedy rzekł Józef do Fáráoná : Sen Fáráonów jedenże jeſt : co Bóg ucżyni / oznájmił Fáráonowi.
Siedm krów pięknych jeſt śiedm lát / á śiedm kłoſów cudnych / jeſt też śiedm lát ; ſen to jeden.
Siedm záś krów chudych y ƺpetnych / które wychodziły zá niemi / jeſt śiedm lát / á śiedm kłoſów cżcżych / y wyſchłych od wiátru wſchodniego / będzie śiedm lát głodnych.
A toć jeſt / com powiedziáł Fáráonowi ; co Bóg będzie cżynił / ukázáł Fáráonowi.
Oto / śiedm lát nádejdzie bárdzo obfitych we wƺyſtkiej ziemi Egipſkiej.
A po nich náſtąpi śiedm lát głodu / y w zápomnienie przyjdzie wƺyſtká oná obfitość w ziemi Egipſkiej / y wytráwi głód ziemię.
Ták / że niebędzie znáć w ziemi obfitośći onej dlá głodu przyƺłego : ábowiem ćiężki będzie bárdzo.
A iż śię po dwá kroć śnił ſen Fáráonowi / znácży / że to pewná rzecż od Bogá / y poſpieƺá Bóg wykonáć ją.
Przetoż teráz niech znájdzie Fáráo mężá rozumnego y mądrego / á przełoży go nád ziemią Egipſką.
Niech ták ucżyni Fáráo / á poſtánowi urzędniki nád ziemią y zbierze piątą cżęść urodzájów w ziemi Egipſkiej przez te śiedm lát obfitych.
Y niech zbieráją wƺeláką żywność lát dobrych náſtępujących / y zgromádzáją zbożá pod rękę Fáráonową / y żywność w mieśćiech niecháj chowáją.
A będzie oná żywność ná wychowánie ziemi ná śiedm lát głodu / które będą w ziemi Egipſkiej / áby nie niƺcżáłá ziemiá od głodu.
Y podobáło śię to Fáráonowi / y wƺyſtkim ſługom jego.
Y rzekł Fáráo do ſług ſwoich : Izáż znájdziemy podobnego mężowi temu / w którymby był Duch Boży?
Zátem rzekł Fáráo do Józefá : Poniewáż ći oznájmił Bóg to wƺyſtko / niemáƺ żádnego ták rozumnego y mądrego jáko ty.
Ty będzieƺ nád domem moim / á według rozkázániá uſt twoich ſpráwowáć śię będzie wƺyſtek lud mój : tylko ſtolicą więkƺy nád ćię będę.
Nád to rzekł Fáráo do Józefá : Oto / poſtánowiłem ćię nád wƺyſtką ziemią Egipſką.
Zdjął tedy Fáráo pierśćień ſwój z ręki ſwej / y dáł go ná rękę Józefową ; oblekł go też w ƺátę biśiorową / y włożył łáńcuch złoty ná ƺyję jego.
Y kázáł go wozić ná wtórym wozie ſwoim / á wołáno przed nim : Kłániájćie śię. Y przełożył go nád wƺyſtką ziemią Egipſką.
Zátem rzekł Fáráo do Józefá : Jám jeſt Fáráo / á bez twego pozwoleniá nie podnieśie żáden áni ręki / áni nogi ſwej / we wƺyſtkiej ziemi Egipſkiej.
Y názwáł Fáráo imię Józefowe / Sáfnát Páneách / á dáł mu Aſenátę / córkę Potyfárá / przełożonego Ońſkiego / zá żonę. Y wyjecháł Józef ná ziemię Egipſką.
A było Józefowi trzydzieśći lát / gdy ſtánął przed Fáráonem / królem Egipſkim ; y wyƺedłƺy Józef od oblicżá Fáráonowego / objecháł wƺyſtkę ziemię Egipſką.
Zrodziłá tedy ziemiá w onych śiedmiu látách urodzájnych obfićie.
Y zgromádził Józef wƺyſtkę żywność onych śiedmiu lát / którá byłá w ziemi Egipſkiej y ſkłádáł żywność w mieśćiech ; urodzáj polny káżdego miáſtá / który był około niego / ſkłádáł w nim.
Zácżem názgromádzáł Józef zbożá / jáko piáſku morſkiego bárdzo wiele / áż go zániecháno licżyć ; bo mu nie było licżby.
A Józefowi urodzili śię dwáj ſynowie / pierwej niż przyƺedł rok głodu / które mu urodziłá Aſenát / córká Potyfárá / przełożonego Ońſkiego.
Názwáł tedy Józef imię pierworodnego Mánáſeſ / mówiąc : Że mi dáł Bóg zápomnieć wƺelkiej prácy mojej / y wƺyſtkiego domu ojcá mego.
A imię drugiego názwáł Efráim / mówiąc : Iż mię rozmnożył Bóg w ziemi utrápieniá mego.
Tedy śię ſkońcżyło śiedm lát obfitośći / którá byłá w ziemi Egipſkiej.
Y pocżęło śiedm lát głodu náſtępowáć / jáko był przepowiedziáł Józef. Y był głód po wƺyſtkich kráinách : ále we wƺyſtkiej ziemi Egipſkiej był chleb.
Jednák potem śćiśnioná byłá głodem wƺyſtká ziemiá Egipſká / y wołáł lud do Fáráoná o chleb. Y rzekł Fáráo wƺyſtkim Egipcżánom : Idźćie do Józefá / á co wám rzecże ucżynićie.
Y był głód po wƺyſtkiej ziemi. Tedy otworzył Józef wƺyſtkie gumná / w których było zboże / y ſprzedáwáł Egipcżánom ; bo śię był głód zmocnił w ziemi Egipſkiej.
Y ze wƺyſtkiej ziemi przyjeżdżáno do Egiptu / kupowáć żywność od Józefá ; bo śię był zmocnił głód po wƺyſtkiej ziemi.
