Y ſtáło śię po wyjśćiu dwu lat / że śię śniło Fáráonowi / jákoby ſtał nád rzeką.
A oto z rzeki wychodźiło śiedm krów / pięknych ná wejrzeniu y tłuſtych ná ćiele / które śię páſły ná łące.
Oto / też śiedm krów innych wychodźiło zá niemi z rzeki / ƺpetnych ná wejrzeniu / y chudych ná ćiele / które ſtáły wedle krów pierwƺych nád brzegiem rzeki.
Y pożárły one krowy ƺpetne ná wejrzeniu y chude ná ćiele / śiedm krów pięknych ná wejrzeniu y tłuſtych ; zátem ocknął śię Fáráo.
A gdy uſnął / śniło mu śię powtóre ; á ono śiedm kłoſów wyráſtáło z jednego źdźbłá / pełnych y cudnych.
Oto / też śiedm kłoſów ćienkich y wyſuƺonych od wiátru wſchodniego wyráſtáło zá nimi.
Y pożárły te kłoſy ćienkie / śiedm onych kłoſów pięknych y zupełnych ; y ocknął śię Fáráo.
A toć był ſen. A gdy było ráno / ſtrwożony był duch jego ; y poſłáwƺy wezwał wƺyſtkich wieƺcżków Egipſkich y wƺyſtkich mędrców jego / y opowiedźiał im Fáráo ſny ſwoje ; á nie było / ktoby je wyłożył Fáráonowi.
Zátem rzekł przełożony nád podcżáƺymi do Fáráoná / mówiąc : Grzechy moje ja dźiś przypominam ſobie.
Fáráo rozgniewáwƺy śię ná ſługi ſwe / dał mię był pod ſtráż do domu hetmáná żołnierzów / mnie y przełożonego nád piekárzámi.
Tám śię nam śnił ſen jednejże nocy / mnie y jemu ; káżdemu według wykłádu ſnu jego śniło śię.
A był tám z námi młodźieniec Hebrejcżyk / ſługá hetmáná żołnierzów / któremuśmy powiedźieli / y wyłożył nam ſny náƺe / káżdemu według ſnu jego wyłożył.
Y ſtáło śię / że jáko nam wyłożył / ták było ; mię przywróćił król ná miejſce moje / á onego obwieśił.
Tedy poſłáwƺy Fáráo / wezwał Józefá / y prędko go wyprowádzono z więźieniá ; który oſtrzygƺy śię / y odmieniwƺy ƺáty ſwoje / przyƺedł do Fáráoná.
Y rzekł Fáráo do Józefá : Śnił mi śię ſen / á nie mam / ktoby mi go wyłożył ; álem ja o tobie ſłyƺał / gdy mówiono / że gdy uſłyƺyƺ ſen / umieƺ go wyłożyć.
Y odpowiedźiał Józef Fáráonowi / mówiąc : Oprócż mnie Bóg opowie rzecży ƺcżęśliwe Fáráonowi.
Tedy rzekł Fáráo do Józefá : Zdáło mi śię we śnie / jákobym ſtał ná brzegu rzeki.
A oto z rzeki wychodźiło śiedm krów tłuſtych ná ćiele / y pięknych ná wejrzeniu / á páſły śię ná łące.
Oto / záś wychodźiło śiedm krów innych zá niemi / nędznych y ƺpetnych ná wejrzeniu / y chudych ná ćiele ; nie widźiáłem we wƺyſtkiej źiemi Egipſkiej ták ƺpetnych.
Y pożárły krowy chude y ƺpetne śiedm krów pierwƺych tłuſtych.
A choć śię doſtáły do wnętrznośći ich / przećię nie było znáć / że śię doſtáły do wnętrznośći ich : bo ná wejrzeniu były ƺpetne / jáko y przedtem ; y ocknąłem śię.
Widźiáłem záś we śnie / á oto / śiedm kłoſów wyráſtáło z jednego źdźbłá pełnych y pięknych.
Oto / też śiedm kłoſów ſuchych / ćienkich / y wyſchłych od wiátru wſchodniego / wyráſtáło zá nimi.
Y pożárły te kłoſy ćienkie śiedm onych kłoſów pięknych. Y powiedźiáłem to wieƺcżkom ; ále nie było / ktoby mi wyłożył.
Tedy rzekł Józef do Fáráoná : Sen Fáráonów jedenże jeſt : co Bóg ucżyni / oznájmił Fáráonowi.
Siedm krów pięknych jeſt śiedm lat / á śiedm kłoſów cudnych / jeſt też śiedm lat ; ſen to jeden.
Siedm záś krów chudych y ƺpetnych / które wychodźiły zá niemi / jeſt śiedm lat / á śiedm kłoſów cżcżych / y wyſchłych od wiátru wſchodniego / będźie śiedm lat głodnych.
A toć jeſt / com powiedźiał Fáráonowi ; co Bóg będźie cżynił / ukázał Fáráonowi.
Oto / śiedm lat nádejdźie bárzo obfitych we wƺyſtkiej źiemi Egipſkiej.
A po nich náſtąpi śiedm lat głodu / y w zápomnienie przydźie wƺyſtká oná obfitość w źiemi Egipſkiej / y wytráwi głód źiemię.
Ták / że niebędźie znáć w źiemi obfitośći oney dla głodu przyƺłego : ábowiem ćiężki będźie bárzo.
A iż śię po dwá kroć śnił ſen Fáráonowi / znácży / że to pewná rzecż od Bogá / y poſpieƺá Bóg wykonáć ją.
Przetoż teraz niech znájdźie Fáráo mężá rozumnego y mądrego / á przełoży go nád źiemią Egipſką.
Niech ták ucżyni Fáráo / á poſtánowi urzędniki nád źiemią y zbierze piątą cżęść urodzájów w źiemi Egipſkiej przez te śiedm lat obfitych.
Y niech zbieráją wƺeláką żywność lat dobrych náſtępujących / y zgromádzáją zbożá pod rękę Fáráonową / y żywność w mieśćiech niecháj chowáją.
A będźie oná żywność ná wychowánie źiemi ná śiedm lat głodu / które będą w źiemi Egipſkiej / áby nie niƺcżáłá źiemiá od głodu.
Y podobáło śię to Fáráonowi / y wƺyſtkim ſługom jego.
Y rzekł Fáráo do ſług ſwojich : Izáż znájdźiemy podobnego mężowi temu / w którymby był Duch Boży?
Zátem rzekł Fáráo do Józefá : Poniewáż ći oznájmił Bóg to wƺyſtko / niemáƺ żádnego ták rozumnego y mądrego jáko ty.
Ty będźieƺ nád domem moim / á według rozkázánia uſt twojich ſpráwowáć śię będźie wƺyſtek lud mój : tylko ſtolicą więkƺy nád ćię będę.
Nád to rzekł Fáráo do Józefá : Oto / poſtánowiłem ćię nád wƺyſtką źiemią Egipſką.
Zdjął tedy Fáráo pierśćień ſwój z ręki ſwej / y dał go ná rękę Józefową ; oblekł go też w ƺátę biśiorową / y włożył łáńcuch złoty ná ƺyję jego.
Y kázał go woźić ná wtórym woźie ſwoim / á wołáno przed nim : Kłániájćie śię. Y przełożył go nád wƺyſtką źiemią Egipſką.
Zátem rzekł Fáráo do Józefá : Jam jeſt Fáráo / á bez twego pozwoleniá nie podnieśie żáden áni ręki / áni nogi ſwej / we wƺyſtkiej źiemi Egipſkiej.
Y názwał Fáráo imię Józefowe / Sáfnát Páneách / á dał mu Aſenátę / córkę Potyfárá / przełożonego Ońſkiego / zá żonę. Y wyjechał Józef ná źiemię Egipſką.
A było Józefowi trzydźieśći lat / gdy ſtánął przed Fáráonem / królem Egipſkim ; y wyƺedƺy Józef od oblicżá Fáráonowego / objechał wƺyſtkę źiemię Egipſką.
Zrodźiłá tedy źiemiá w onych śiedmiu látách urodzájnych obfićie.
Y zgromádźił Józef wƺyſtkę żywność onych śiedmiu lat / która byłá w źiemi Egipſkiej y ſkłádał żywność w mieśćiech ; urodzáj polny káżdego miáſtá / który był około niego / ſkłádał w nim.
Zácżem názgromádzał Józef zbożá / jáko piáſku morſkiego bárzo wiele / áż go zániecháno licżyć ; bo mu nie było licżby.
A Józefowi urodźili śię dwáj ſynowie / pierwej niż przyƺedł rok głodu / które mu urodźiłá Aſenát / córká Potyfárá / przełożonego Ońſkiego.
Názwał tedy Józef imię pierworodnego Mánáſes / mówiąc : Że mi dał Bóg zápomnieć wƺelkiej prácy mojey / y wƺyſtkiego domu ojcá mego.
A imię drugiego názwał Efráim / mówiąc : Iż mię rozmnożył Bóg w źiemi utrapienia mego.
Tedy śię ſkońcżyło śiedm lat obfitośći / która byłá w źiemi Egipſkiej.
Y pocżęło śiedm lat głodu náſtępowáć / jáko był przepowiedźiał Józef. Y był głód po wƺyſtkich kráinách : ále we wƺyſtkiej źiemi Egipſkiej był chleb.
Jednák potem śćiśnioná byłá głodem wƺyſtká źiemiá Egipſká / y wołał lud do Fáráoná o chleb. Y rzekł Fáráo wƺyſtkim Egipcżánom : Idźćie do Józefá / á co wam rzecże ucżynićie.
Y był głód po wƺyſtkiej źiemi. Tedy otworzył Józef wƺyſtkie gumná / w których było zboże / y ſprzedáwał Egipcżánom ; bo śię był głód zmocnił w źiemi Egipſkiej.
Y ze wƺyſtkiej źiemi przyjeżdżáno do Egiptu / kupowáć żywność od Józefá ; bo śię był zmocnił głód po wƺyſtkiej źiemi.
