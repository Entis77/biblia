Zátem upádł Józef ná twárz ojcá ſwego / y płákał nád nim / á cáłował go.
Y rozkázał Józef ſługom ſwym lekárzom / áby wonnemi máśćiámi námázáli ojcá jego ; y námázáli wonnemi máśćiámi lekárze Izráelá.
A gdy śię mázánia jego wypełniło cżterdźieśći dni / ( bo śię ták wypełniáją dni tych / którzy wonnemi máśćiámi mázáni bywáją ) tedy go płákáli Egipcżánie przez śiedmdźieśiąt dni.
A po wyjśćiu dni żáłoby jego rzekł Józef do ſług Fáráonowych / mówiąc : Jeſlim teraz ználázł łáſkę w ocżách wáƺych / powiedzćie proƺę Fáráonowi / mówiąc :
Oćiec mój poprzyśiągł mię mówiąc : Oto / ja umierám ; w grobie moim / którym ſobie wykopał w źiemi Chánánejſkiej / tám mię pogrzebieƺ ; á teraz niech jádę / proƺę / y pogrzebię ojcá mego / y záś śię wrócę.
Tedy rzekł Fáráo : Jedź á pogrzeb ojcá twego / jáko ćię poprzyśiągł.
Jechał tedy Józef / áby pogrzebł ojcá ſwego ; jecháli też z nim wƺyſcy ſłudzy Fáráonowi / tákże ſtárśi domu jego / y wƺyſcy ſtárśi źiemi Egipſkiej ;
Y wƺyſtek dom Józefów / y bráćia jego / y dom ojcá jego ; tylko dźiátki ſwoje / y owce ſwoje / y woły ſwoje zoſtáwili w źiemi Goſen.
Szły też z nim y wozy / y jezdni ; á był pocżet bárzo wielki.
Y przyjecháli áż ná pole Atád / które jeſt przy brodźie Jordáńſkim / y płákáli tám płácżem wielkim y bárzo ćiężkim ; y obchodźił Józef po ojcu ſwym żáłobę przez śiedm dni.
A ujrzáwƺy obywátele źiemi Chánánejſkiej żáłobę onę ná polu Atád / mówili : Żáłobá to ćiężká Egipcżánów ; przetoż názwano imię miejſcá onego Abel Micráim / które jeſt przy brodźie Jordáńſkim.
Ucżynili tedy z nim ſynowie jego / jáko im był rozkázał.
Y záwieſli go ſynowie jego do źiemi Chánánejſkiej / y pogrzebli go w jáſkini ná polu Máchpelá / którą Abráhám był kupił z rolą ná ośiádłość grobu / od Efroná Hetejcżyká / przećiwko Mámre.
Zátem śię wróćił Józef do Egiptu z bráćią ſwą / y ze wƺyſtkimi / którzy jeźdźili z nim ná pogrzeb ojcá jego / odpráwiwƺy pogrzeb ojcá ſwego.
A widząc bráćia Józefowi / że umárł oćiec ich / mówili : Podobno będźie nas miał w nienáwiśći Józef / y ſowićie oddá nam wƺyſtko złe / któreśmy mu ucżynili.
Wſkázáli tedy do Józefá / mówiąc : Oćiec twój rozkázał / pierwej niż umárł / mówiąc :
Ták powiedzćie Józefowi : Proƺę / odpuść teraz przeſtępſtwo bráći twej / y grzech ich / żeć złość wyrządźili ; proƺę odpuść teraz wyſtępek ſługom Bogá ojcá twego. Y płákał Józef / gdy to mówili do niego.
Y przyſtąpili bráćia jego / á upádƺy przed nim / mówili : Otośmy ſługámi twoimi.
Y rzekł do nich Józef : Nie bójćie śię : bo ázáżem ja wam zá Bogá?
Wyśćie złe myślili przećiwko mnie / ále Bóg obróćił to w dobre / chcąc ucżynić to / co śię dźiś dźieje / áby záchował ták wielki lud.
A przetoż niebójćie śię / ja żywić będę was y dźiátki wáƺe ; á ták ćieƺył je / y mówił z nimi łágodnie.
Y mieƺkał Józef w Egipćie / ſám y dom ojcá jego / á żył Józef ſto y dźieśięć lat.
Y oglądał Józef ſyny Efráimowe áż do trzećiego pokolenia. Synowie też Máchyrá / ſyná Mánáſeſowego / porodźili śię ná kolánách Józefowych.
Y rzekł Józef do bráći ſwej : Ja umrę / ále Bóg zápewnie náwiedźi was / y wyprowádźi was z źiemi tey do źiemi / o którą przyśiągł Abráhámowi / Izáákowi y Jákóbowi.
Y poprzyśiągł Józef ſyny Izráelowe / mówiąc : Gdy was náwiedźi Pán Bóg / wynieśćie też kośći moje ſtąd.
Y umárł Józef / májąc ſto y dźieśięć lat ; którego námázáwƺy wonnemi máśćiámi / włożono do trumny w Egipćie.
