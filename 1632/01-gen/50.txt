Zátem upádł Józef ná twárz ojcá ſwego / y płákáł nád nim / á cáłowáł go.
Y rozkázáł Józef ſługom ſwym lekárzom / áby wonnemi máśćiámi námázáli ojcá jego ; y námázáli wonnemi máśćiámi lekárze Izráelá.
A gdy śię mázániá jego wypełniło cżterdzieśći dni / ( bo śię ták wypełniáją dni tych / którzy wonnemi máśćiámi mázáni bywáją ) tedy go płákáli Egipcżánie przez śiedmdzieśiąt dni.
A po wyjśćiu dni żáłoby jego rzekł Józef do ſług Fáráonowych / mówiąc : Jeźlim teráz ználázł łáſkę w ocżách wáƺych / powiedzćie proƺę Fáráonowi / mówiąc :
Ojćiec mój poprzyśiągł mię mówiąc : Oto / já umierám ; w grobie moim / którym ſobie wykopáł w ziemi Chánánejſkiej / tám mię pogrzebieƺ ; á teráz niech jádę / proƺę / y pogrzebię ojcá mego / y záś śię wrócę.
Tedy rzekł Fáráo : Jedź á pogrzeb ojcá twego / jáko ćię poprzyśiągł.
Jecháł tedy Józef / áby pogrzebł ojcá ſwego ; jecháli też z nim wƺyſcy ſłudzy Fáráonowi / tákże ſtárśi domu jego / y wƺyſcy ſtárśi ziemi Egipſkiej ;
Y wƺyſtek dom Józefów / y bráćiá jego / y dom ojcá jego ; tylko dziátki ſwoje / y owce ſwoje / y woły ſwoje zoſtáwili w ziemi Goſen.
Szły też z nim y wozy / y jezdni ; á był pocżet bárdzo wielki.
Y przyjecháli áż ná pole Atád / które jeſt przy brodzie Jordáńſkim / y płákáli tám płácżem wielkim y bárdzo ćiężkim ; y obchodził Józef po ojcu ſwym żáłobę przez śiedm dni.
A ujrzáwƺy obywátele ziemi Chánánejſkiej żáłobę onę ná polu Atád / mówili : Żáłobá to ćiężká Egipcżánów ; przetoż názwáno imię miejſcá onego Abel Micráim / które jeſt przy brodzie Jordáńſkim.
Ucżynili tedy z nim ſynowie jego / jáko im był rozkázáł.
Y záwieźli go ſynowie jego do ziemi Chánánejſkiej / y pogrzebli go w jáſkini ná polu Máchpelá / którą Abráhám był kupił z rolą ná ośiádłość grobu / od Efroná Hetejcżyká / przećiwko Mámre.
Zátem śię wróćił Józef do Egiptu z bráćią ſwą / y ze wƺyſtkimi / którzy jeździli z nim ná pogrzeb ojcá jego / odpráwiwƺy pogrzeb ojcá ſwego.
A widząc bráćiá Józefowi / że umárł ojćiec ich / mówili : Podobno będzie náſ miáł w nienáwiśći Józef / y ſowićie oddá nám wƺyſtko złe / któreśmy mu ucżynili.
Wſkázáli tedy do Józefá / mówiąc : Ojćiec twój rozkázáł / pierwej niż umárł / mówiąc :
Ták powiedzćie Józefowi : Proƺę / odpuść teráz przeſtępſtwo bráći twej / y grzech ich / żeć złość wyrządzili ; proƺę odpuść teráz wyſtępek ſługom Bogá ojcá twego. Y płákáł Józef / gdy to mówili do niego.
Y przyſtąpili bráćiá jego / á upádłƺy przed nim / mówili : Otośmy ſługámi twoimi.
Y rzekł do nich Józef : Nie bójćie śię : bo ázáżem já wám zá Bogá?
Wyśćie złe myślili przećiwko mnie / ále Bóg obróćił to w dobre / chcąc ucżynić to / co śię dziś dzieje / áby záchowáł ták wielki lud.
A przetoż niebójćie śię / já żywić będę wáſ y dziátki wáƺe ; á ták ćieƺył je / y mówił z nimi łágodnie.
Y mieƺkáł Józef w Egipćie / ſám y dom ojcá jego / á żył Józef ſto y dzieśięć lát.
Y oglądáł Józef ſyny Efráimowe áż do trzećiego pokoleniá. Synowie też Máchyrá / ſyná Mánáſeſowego / porodzili śię ná kolánách Józefowych.
Y rzekł Józef do bráći ſwej : Já umrę / ále Bóg zápewnie náwiedzi wáſ / y wyprowádzi wáſ z ziemi tej do ziemi / o którą przyśiągł Abráhámowi / Izáákowi y Jákóbowi.
Y poprzyśiągł Józef ſyny Izráelowe / mówiąc : Gdy wáſ náwiedzi Pán Bóg / wynieśćie też kośći moje ſtąd.
Y umárł Józef / májąc ſto y dzieśięć lát ; którego námázáwƺy wonnemi máśćiámi / włożono do trumny w Egipćie.
