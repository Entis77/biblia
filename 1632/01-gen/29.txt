Tedy Jákób wſtáwƺy / poƺedł do źiemi / mieƺkájących ná wſchód ſłońcá.
Y ujrzał ſtudnią ná polu / y trzy ſtádá owiec leżących przy niej ; bo z oney ſtudni nápáwano ſtádá / á kámień wielki był ná wierzchu oney ſtudni.
Abowiem ſchodźiły śię tám wƺyſtkie ſtádá / y odwálono kámień z wierzchu ſtudni / á nápáwano ſtádá ; potem záś kłádźiono kámień ná wierzch ſtudni ná miejſce jego.
Tedy rzekł do nich Jákób : Bráćia moi / ſkądeśćie? y odpowiedźieli : Z Háránu jeſteśmy.
Y rzekł do nich Jákób : A znáćie Lábáná / ſyná Náchorowego? odpowiedźieli : Znamy.
Zátem rzekł do nich : A dobrze śię ma? á oni odpowiedźieli : Dobrze ; á oto / Ráchel córká jego idźie z ſtádem.
Tedy rzekł : Oto / jeƺcże doſyć dniá / y nie cżás zgániáć ſtádá ; nápójćież owce / á idźćie / popáśćie ich.
A oni odpowiedźieli : Nie możemy / áżby śię zebráły wƺyſtkie ſtádá / y odwálony był kámień z wierzchu ſtudni / ábyſmy nápoili ſtádá.
A gdy to jeƺcże mówił z nimi / Ráchel nádeƺłá z owcámi ojcá ſwego / bo je oná páſłá.
Y gdy ujrzał Jákób Ráchelę / córkę Lábáná / brátá mátki ſwojey / z owcámi Lábáná / brátá mátki ſwej : tedy przyſtąpił Jákób / y odwálił kámień z wierzchu ſtudni / á nápoił owce Lábáná / brátá mátki ſwojey.
Y pocáłował Jákób Ráchelę / y podnióſƺy głos ſwój płákał.
Y oznájmił Jákób Rácheli / że jeſt brátem ojcá jey / á iż jeſt ſynem Rebeki : á oná bieżáwƺy opowiedźiáłá to ojcu ſwemu.
A gdy uſłyƺał Lábán wieść o Jákóbie / ſynu śioſtry ſwojey / wybieżał przećiwko niemu / y obłápił go / á pocáłowáwƺy / wwiódł do domu ſwego. A on Lábánowi powiedźiał o wƺyſtkim.
Y rzekł mu Lábán : Záiſteś ty jeſt kość mojá / y ćiáło moje. Y mieƺkał u niego przez cáły mieśiąc.
Potem rzekł Lábán do Jákóbá : Izali / żeś mi brát / ſłużyć mi będźieƺ dármo? powiedz mi / jáką ma być zápłátá twojá.
A miał Lábán dwie córki : imię ſtárƺej Lijá / á imię młodƺej Ráchel.
Ale Lijá byłá chorych ocżu / á Ráchel záś pięknego oblicża / y wdźięcżná ná wejrzeniu.
Miłował tedy Jákób Ráchelę / y rzekł : Będęć ſłużył śiedm lat zá Ráchelę / córkę twoję młodƺą.
Odpowiedźiał Lábán : Lepiej że ją tobie dam / niźlibym ją miał dáć mężowi innemu ; mieƺkájże ze mną.
Y ſłużył Jákób zá Ráchelę śiedm lat / y zdał mu śię ten cżás jáko kilká dni / przeto / że ją miłował.
Potem rzekł Jákób do Lábáná : Dáj mi żonę moję / ponieważ śię wypełniły dni moje / ábym wƺedł do niej.
Tedy wezwáwƺy Lábán wƺyſtkich mężów miejſcá onego / ſpráwił ucżtę.
A gdy był wiecżór / wźiął Liją / córkę ſwoję / y wwiódł ją do niego / á Jákób wƺedł do niej.
Dał też Lábán y Zelfę dźiewkę ſwoję Lii / córce ſwej / zá ſłużebnicę.
A gdy było ráno / poznał Jákób / że to Lijá / y rzekł do Lábáná : Cóżeś mi to ucżynił? Izálim ja nie zá Ráchelę tobie ſłużył? cżemużeś mię tedy oƺukał?
Y odpowiedźiał Lábán : Nie jeſt to w zwycżáju u nas / áby miáno wydáwáć zá mąż młodƺą przed ſtárƺą.
Wytrwáj z tą tydźień / á dam ći y tę zá ſłużbę / którą mi będźieƺ ſłużył jeƺcże drugie śiedm lat.
Y ucżynił ták Jákób / y wypełnił z tą tydźień ; potem dał mu Lábán Ráchelę / córkę ſwoję / zá żonę.
Dał też Lábán Rácheli / córce ſwej / Bálę / dźiewkę ſwoję ; dał jey zá ſłużebnicę.
Tedy też wƺedł Jákób do Rácheli / y miłował Ráchelę bárdźiej niż Liją / á ſłużył mu jeƺcże drugie śiedm lat.
A widząc Pán / że nienáwidźiał Liję / otworzył żywot jey ; á Ráchel niepłodná byłá.
Tedy pocżąwƺy Lijá porodźiłá ſyná / y názwáłá imię jego Ruben / bo rzekłá : Zájiſte wejrzał Pán ná utrápienie moje ; á ták teraz miłowáć mię będźie mąż mój.
Y záśię pocżęłá / y porodźiłá ſyná / á rzekłá : Zájiſte uſłyƺał Pán / żem ja byłá w nienáwiśći / przetoż dał mi też y tego ; y názwáłá imię jego Symeon.
Potem jeƺcże pocżęłá / y porodźiłá ſyná / y rzekłá : Y tą rázą przyłącży śię mąż mój do mnie / bom mu urodźiłá trzech ſynów ; przetoż názwáłá imię jego Lewi.
Nád to jeƺcże pocżęłá / y porodźiłá ſyná / y rzekłá : Teraz już chwálić będę Páná ; przetoż názwáłá imię jego Judá / y przeſtáłá rodźić.
