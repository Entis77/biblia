Tedy Jákób wſtáwƺy / poƺedł do ziemi / mieƺkájących ná wſchód ſłońcá.
Y ujrzáł ſtudnią ná polu / y trzy ſtádá owiec leżących przy niej ; bo z onej ſtudni nápáwáno ſtádá / á kámień wielki był ná wierzchu onej ſtudni.
Abowiem ſchodziły śię tám wƺyſtkie ſtádá / y odwálono kámień z wierzchu ſtudni / á nápáwáno ſtádá ; potem záś kłádziono kámień ná wierzch ſtudni ná miejſce jego.
Tedy rzekł do nich Jákób : Bráćiá moi / ſkądeśćie? y odpowiedzieli : Z Háránu jeſteśmy.
Y rzekł do nich Jákób : A znáćie Lábáná / ſyná Náchorowego? odpowiedzieli : Známy.
Zátem rzekł do nich : A dobrze śię má? á oni odpowiedzieli : Dobrze ; á oto / Ráchel córká jego idzie z ſtádem.
Tedy rzekł : Oto / jeƺcże doſyć dniá / y nie cżáſ zgániáć ſtádá ; nápójćież owce / á idźćie / popáśćie ich.
A oni odpowiedzieli : Nie możemy / áżby śię zebráły wƺyſtkie ſtádá / y odwálony był kámień z wierzchu ſtudni / ábyśmy nápoili ſtádá.
A gdy to jeƺcże mówił z nimi / Ráchel nádeƺłá z owcámi ojcá ſwego / bo je oná páſłá.
Y gdy ujrzáł Jákób Ráchelę / córkę Lábáná / brátá mátki ſwojej / z owcámi Lábáná / brátá mátki ſwej : tedy przyſtąpił Jákób / y odwálił kámień z wierzchu ſtudni / á nápoił owce Lábáná / brátá mátki ſwojej.
Y pocáłowáł Jákób Ráchelę / y podnióſłƺy głoſ ſwój płákáł.
Y oznájmił Jákób Rácheli / że jeſt brátem ojcá jej / á iż jeſt ſynem Rebeki : á oná bieżáwƺy opowiedziáłá to ojcu ſwemu.
A gdy uſłyƺáł Lábán wieść o Jákóbie / ſynu śioſtry ſwojej / wybieżáł przećiwko niemu / y obłápił go / á pocáłowáwƺy / wwiódł do domu ſwego. A on Lábánowi powiedziáł o wƺyſtkiem.
Y rzekł mu Lábán : Záiſteś ty jeſt kość mojá / y ćiáło moje. Y mieƺkáł u niego przez cáły mieśiąc.
Potem rzekł Lábán do Jákóbá : Izáli / żeś mi brát / ſłużyć mi będzieƺ dármo? powiedz mi / jáką má być zápłátá twojá.
A miáł Lábán dwie córki : imię ſtárƺej Lijá / á imię młodƺej Ráchel.
Ale Lijá byłá chorych ocżu / á Ráchel záś pięknego oblicżá / y wdzięcżná ná wejrzeniu.
Miłowáł tedy Jákób Ráchelę / y rzekł : Będęć ſłużył śiedm lát zá Ráchelę / córkę twoję młodƺą.
Odpowiedziáł Lábán : Lepiej że ją tobie dám / niźlibym ją miáł dáć mężowi innemu ; mieƺkájże ze mną.
Y ſłużył Jákób zá Ráchelę śiedm lát / y zdáł mu śię ten cżáſ jáko kilká dni / przeto / że ją miłowáł.
Potem rzekł Jákób do Lábáná : Dáj mi żonę moję / poniewáż śię wypełniły dni moje / ábym wƺedł do niej.
Tedy wezwáwƺy Lábán wƺyſtkich mężów miejſcá onego / ſpráwił ucżtę.
A gdy był wiecżór / wziął Liją / córkę ſwoję / y wwiódł ją do niego / á Jákób wƺedł do niej.
Dáł też Lábán y Zelfę dziewkę ſwoję Lii / córce ſwej / zá ſłużebnicę.
A gdy było ráno / poznáł Jákób / że to Lijá / y rzekł do Lábáná : Cóżeś mi to ucżynił? Izálim já nie zá Ráchelę tobie ſłużył? cżemużeś mię tedy oƺukáł?
Y odpowiedziáł Lábán : Nie jeſt to w zwycżáju u náſ / áby miáno wydáwáć zá mąż młodƺą przed ſtárƺą.
Wytrwáj z tą tydzień / á dám ći y tę zá ſłużbę / którą mi będzieƺ ſłużył jeƺcże drugie śiedm lát.
Y ucżynił ták Jákób / y wypełnił z tą tydzień ; potem dáł mu Lábán Ráchelę / córkę ſwoję / zá żonę.
Dáł też Lábán Rácheli / córce ſwej / Bálę / dziewkę ſwoję ; dáł jej zá ſłużebnicę.
Tedy też wƺedł Jákób do Rácheli / y miłowáł Ráchelę bárdziej niż Liją / á ſłużył mu jeƺcże drugie śiedm lát.
A widząc Pán / że nienáwidziáł Liję / otworzył żywot jej ; á Ráchel niepłodná byłá.
Tedy pocżąwƺy Lijá porodziłá ſyná / y názwáłá imię jego Ruben / bo rzekłá : Záiſte wejrzáł Pán ná utrápienie moje ; á ták teráz miłowáć mię będzie mąż mój.
Y záśię pocżęłá / y porodziłá ſyná / á rzekłá : Záiſte uſłyƺáł Pán / żem já byłá w nienáwiśći / przetoż dáł mi też y tego ; y názwáłá imię jego Symeon.
Potem jeƺcże pocżęłá / y porodziłá ſyná / y rzekłá : Y tą rázą przyłącży śię mąż mój do mnie / bom mu urodziłá trzech ſynów ; przetoż názwáłá imię jego Lewi.
Nád to jeƺcże pocżęłá / y porodziłá ſyná / y rzekłá : Teráz już chwálić będę Páná ; przetoż názwáłá imię jego Judá / y przeſtáłá rodzić.
