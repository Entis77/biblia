To gdy śię ſtáło / dáno znáć Józefowi : Oto / ojćiec twój záchorzáł ; który wziąwƺy dwóch ſynów ſwoich z ſobą / Mánáſeſá y Efráimá / jecháł do niego.
Y powiedziáno Jákóbowi / mówiąc : Oto / ſyn twój Józef idzie do ćiebie. A Izráel pokrzepiwƺy śię / uśiádł ná łożu.
Y rzekł Jákób do Józefá : Bóg wƺechmogący ukázáł mi śię w Luzie / w ziemi Chánánejſkiej / y błogoſłáwił mi.
A mówił do mnie : Oto / já rozrodzę ćię / y rozmnożę ćię / y wywiodę z ćiebie wielki náród ; á dám ziemię tę náśieniu twemu po tobie w dziedzictwo wiecżne.
Przetoż teráz dwáj ſynowie twoi / którzyć śię urodzili w ziemi Egipſkiej / pierwej niżem já tu do ćiebie przyƺedł do Egiptu / moi ſą / Efráim y Mánáſeſ ; jáko Ruben y Symeon moi będą.
Ale dzieći twoje / które po tych ſpłodziƺ / twoje będą ; imieniem bráći ſwojej będą zwáni w ośiádłośćiách ſwych.
A gdym śię wrácáł z Pádán / umárłá mi Ráchel w ziemi Chánánejſkiej w drodze / gdym jeƺcże był jákoby ná milę od Efráty / y pogrzebłem ją tám przy drodze ku Efráćie ; á toć jeſt Betlehem.
A ujrzáwƺy Izráel ſyny Józefowe / rzekł : Cżyi to ſą?
Tedy odpowiedziáł Józef ojcu ſwemu : Synowie to moi / które mi tu dáł Bóg ; á on rzekł : Przywiedź je proƺę do mnie / ábym im błogoſłáwił.
A ocży Izráelowe oćiężáły były dlá ſtárośći / y nie mógł dojrzeć : y przywiódł je do niego / które Jákób pocáłowáł y obłápił.
Zátem rzekł Izráel do Józefá : Oglądáć więcej oblicżá twego nie ſpodziewáłem śię / á oto / dáł mi Bóg widzieć y náśienie twoje.
Tedy Józef odwiódł je od łoná jego / y pokłonił śię oblicżem ſwem áż do ziemi.
A wziąwƺy Józef obudwu / poſtáwił Efráimá po práwej ręce ſwojej / á po lewej Izráelowej ; á Mánáſeſá po lewej ręce ſwojej / á po práwej Izráelowej / y przywiódł je do niego.
A wyćiągnąwƺy Izráel práwicę ſwoję / włożył ją ná głowę Efráimá / który był młodƺy / lewicę záś ſwoję ná głowę Mánáſeſá / umyślnie przełożywƺy ręce ſwoje / choć Mánánſeſ był pierworodny.
Y błogoſłáwił Józefowi / mówiąc : Bóg / przed którego oblicżem chodzili ojcowie moi / Abráhám y Izáák / Bóg / który mię żywił od młodośći mojej áż do dniá tego ;
Anioł / który mię wyrwáł ze wƺyſtkiego złego / niech błogoſłáwi dziećiom tym / á niech będą názywáni od imieniá mego / y od imieniá ojców moich / Abráhámá y Izááká / á jáko ryby niech śię rozmnożą ná ziemi.
A obácżywƺy Józef / iż włożył ojćiec jego rękę práwą ſwoję ná głowę Efráimowę / nie miło mu było ; y ujął rękę ojcá ſwego / áby ją przenióſł z głowy Efráimowej ná głowę Mánáſeſowę.
Y rzekł Józef do ojcá ſwego : Nie ták / ojcże mój ; ábowiem ten jeſt pierworodny / włóżże práwicę ſwoję ná głowę jego.
Ale śię zbrániáł ojćiec jego / y rzekł : Wiemći ſynu miły / wiem ; y tenći śię ſtánie w lud wielki / tenći też urośnie ; á wƺákże brát jego młodƺy urośnie nádeń / á z náśieniá jego wyjdzie mnóſtwo národów.
Błogoſłáwił im tedy dniá onego / mówiąc : Przez ćię będzie błogoſłáwił Izráel / mówiąc : Niech ćię wyſtáwi Bóg jáko Efráimá / y jáko Mánáſeſá ; á ták przełożył Efráimá nád Mánáſeſá.
Potem rzekł Izráel do Józefá : Oto / já umierám / á Bóg będzie z wámi / y przywróći wáſ do ziemi ojców wáƺych.
Oto / já dáwám ći cżęść jednę mimo bráćią twoję / którejm nábył z ręki Amorejcżyków miecżem moim / y łukiem moim.
