To gdy śię ſtáło / dáno znáć Józefowi : Oto / ojćiec twój záchorzáł ; który wźiąwƺy dwóch ſynów ſwoich z ſobą / Mánáſeſá y Efráimá / jecháł do niego.
Y powiedźiáno Jákóbowi / mówiąc : Oto / ſyn twój Józef idźie do ćiebie. A Izráel pokrzepiwƺy śię / uśiádł ná łożu.
Y rzekł Jákób do Józefá : Bóg wƺechmogący ukázáł mi śię w Luźie / w źiemi Chánánejſkiej / y błogoſłáwił mi.
A mówił do mnie : Oto / ja rozrodzę ćię / y rozmnożę ćię / y wywiodę z ćiebie wielki náród ; á dám źiemię tę náśieniu twemu po tobie w dźiedźictwo wiecżne.
Przetoż teraz dwáj ſynowie twoi / którzyć śię urodźili w źiemi Egipſkiej / pierwej niżem ja tu do ćiebie przyƺedł do Egiptu / moi ſą / Efráim y Mánáſeſ ; jáko Ruben y Symeon moi będą.
Ale dźieći twoje / które po tych ſpłodźiƺ / twoje będą ; imieniem bráći ſwojey będą zwáni w ośiádłośćiách ſwych.
A gdym śię wrácáł z Pádán / umárłá mi Ráchel w źiemi Chánánejſkiej w drodze / gdym jeƺcże był jákoby ná milę od Efráty / y pogrzebłem ją tám przy drodze ku Efráćie ; á toć jeſt Betlehem.
A ujrzáwƺy Izráel ſyny Józefowe / rzekł : Cżyi to ſą?
Tedy odpowiedźiáł Józef ojcu ſwemu : Synowie to moi / które mi tu dáł Bóg ; á on rzekł : Przywiedź je proƺę do mnie / ábym im błogoſłáwił.
A ocży Izráelowe oćiężáły były dla ſtárośći / y nie mógł dojrzeć : y przywiódł je do niego / które Jákób pocáłowáł y obłápił.
Zátem rzekł Izráel do Józefá : Oglądáć więcey oblicżá twego nie ſpodźiewáłem śię / á oto / dáł mi Bóg widźieć y náśienie twoje.
Tedy Józef odwiódł je od łoná jego / y pokłonił śię oblicżem ſwem áż do źiemi.
A wźiąwƺy Józef obudwu / poſtáwił Efráimá po práwej ręce ſwojey / á po lewej Izráelowej ; á Mánáſeſá po lewej ręce ſwojey / á po práwej Izráelowej / y przywiódł je do niego.
A wyćiągnąwƺy Izráel práwicę ſwoję / włożył ją ná głowę Efráimá / który był młodƺy / lewicę záś ſwoję ná głowę Mánáſeſá / umyślnie przełożywƺy ręce ſwoje / choć Mánánſeſ był pierworodny.
Y błogoſłáwił Józefowi / mówiąc : Bóg / przed którego oblicżem chodźili ojcowie moi / Abráhám y Izáák / Bóg / który mię żywił od młodośći mojey áż do dniá tego ;
Anioł / który mię wyrwáł ze wƺyſtkiego złego / niech błogoſłáwi dźiećiom tym / á niech będą názywáni od imieniá mego / y od imieniá ojców moich / Abráhámá y Izááká / á jáko ryby niech śię rozmnożą ná źiemi.
A obácżywƺy Józef / iż włożył ojćiec jego rękę práwą ſwoję ná głowę Efráimowę / nie miło mu było ; y ujął rękę ojcá ſwego / áby ją przenióſł z głowy Efráimowej ná głowę Mánáſeſowę.
Y rzekł Józef do ojcá ſwego : Nie ták / ojcże mój ; ábowiem ten jeſt pierworodny / włóżże práwicę ſwoję ná głowę jego.
Ale śię zbrániáł ojćiec jego / y rzekł : Wiemći ſynu miły / wiem ; y tenći śię ſtánie w lud wielki / tenći też urośnie ; á wƺákże brát jego młodƺy urośnie nádeń / á z náśieniá jego wyjdźie mnóſtwo národów.
Błogoſłáwił im tedy dniá onego / mówiąc : Przez ćię będźie błogoſłáwił Izráel / mówiąc : Niech ćię wyſtáwi Bóg jáko Efráimá / y jáko Mánáſeſá ; á ták przełożył Efráimá nád Mánáſeſá.
Potem rzekł Izráel do Józefá : Oto / ja umierám / á Bóg będźie z wámi / y przywróći was do źiemi ojców wáƺych.
Oto / ja dáwám ći cżęść jednę mimo bráćią twoję / którejm nábył z ręki Amorejcżyków miecżem moim / y łukiem moim.
