To gdy śię ſtáło / kuśił Bóg Abráhámá / y rzekł do niego : Abráhámie! A on odpowiedziáł : Owom já.
Y rzekł Bóg : Weźmij teráz ſyná twego / jedynego twego / którego miłujeƺ / Izááká / á idź do ziemi Moryjá / y tám go ofiáruj ná ofiárę páloną / ná jednej górze / o którejć powiem.
Tedy wſtáwƺy Abráhám bárdzo ráno / ośiodłáł oſłá ſwego / y wziął dwóch ſług ſwoich z ſobą / y Izááká ſyná ſwego / á nárąbáwƺy drew ná ofiárę páloną / wſtáł y ƺedł ná miejſce / o którem mu Bóg powiedziáł.
A dniá trzećiego / podnióſłƺy Abráhám ocży ſwe / ujrzáł ono miejſce z dáleká.
Y rzekł Abráhám do ſług ſwoich : Zoſtáńćie wy tu z oſłem / á já z dziećięćiem pójdziemy áż do onąd / á odpráwiwƺy modlitwy / wróćimy śię do wáſ.
Wziął tedy Abráhám drwá ná ofiárę páloną / y włożył je ná Izááká / ſyná ſwego / á ſám wziął w rękę ſwoję ogień y miecż / y ƺli obáj poſpołu.
Y rzekł Izáák do Abráhámá / ojcá ſwego / mówiąc : Ojcże mój! A on odpowiedziáł : Owom já / ſynu mój. Y rzekł Izáák : Oto ogień y drwá / á gdzież báránek ná ofiárę páloną?
Odpowiedziáł Abráhám : Bóg ſobie obmyśli báránká ná ofiárę páloną / ſynu mój ; y ƺli obáj poſpołu.
A gdy przyƺli ná miejſce / o którem mu Bóg powiedziáł / zbudowáł tám Abráhám ołtárz / y ułożył drwá / á związáwƺy Izááká / ſyná ſwego / włożył go ná ołtárz ná drwá.
Y wyćiągnął Abráhám rękę ſwoję / y wziął miecż / áby zábił ſyná ſwego.
Lecż záwołáł náń Anioł Páńſki z niebá / y rzekł : Abráhámie! Abráhámie! A on rzekł : Owom já.
Y rzekł Anioł : Nie wyćiągáj ręki twej ná dziećię / y nie cżyń mu nic ; bom teráz doznáł / iż śię ty boiƺ Bogá / y nie ſfolgowáłeś ſynowi twemu / jedynemu twemu / dlá mnie.
A podnióſłƺy Abráhám ocży ſwe / ujrzáł / á oto bárán zá nim uwiązł w ćierniu zá rogi ſwoje ; á ƺedłƺy Abráhám / wziął báráná y ofiárowáł go ná ofiárę páloną / miáſto ſyná ſwego.
Y názwáł Abráhám imię miejſcá onego : Pán obmyśli ; ztądże po dziś dzień mówią : Ná górze Páńſkiej będzie obmyślono.
Tedy záwołáł Anioł Páńſki ná Abráhámá powtóre z niebá mówiąc :
Przez śiebie ſámego przyśiągłem / mówi Pán : Poniewáżeś to ucżynił / á nie zfolgowáłeś ſynowi twemu / jedynemu twemu ;
Błogoſłáwiąc błogoſłáwić ći będę / á rozmnáżájąc rozmnożę náśienie twoje / jáko gwiázdy niebieſkie / y jáko piáſek / który jeſt ná brzegu morſkim ; á odziedzicży náśienie twoje brámy nieprzyjáćiół ſwoich.
Y błogoſłáwione będą w náśieniu twojim wƺyſtkie národy ziemi / dlá tego / żeś uſłucháł głoſu mego.
Wróćił śię tedy Abráhám do ſług ſwych / y wſtáwƺy / przyƺli poſpołu do Beerſebá ; bo mieƺkáł Abráhám w Beerſebá.
Y ſtáło śię potem / iż oznájmiono Abráhámowi / mówiąc : Oto národziłá y Melchá ſynów Náchorowi / brátu twemu.
Huſá / pierworodnego ſwego / y Buzá / brátá jego / y Chemuelá / ojcá Arámcżyków.
Y Káſedá / y Káſáná / y Feldáſá / y Jedláfá / y Bátuelá.
A Bátuel ſpłodził Rebekę ; ośmioro tych dzieći urodziłá Melchá Náchorowi / brátu Abráhámowemu.
A záłożnicá jego / której imię Reumá / urodziłá też Tábę / y Gáhámá / y Táháſá / y Mááchá.
