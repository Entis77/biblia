To gdy śię ſtáło / kuśił Bóg Abráhámá / y rzekł do niego : Abráhámie! A on odpowiedźiáł : Owom ja.
Y rzekł Bóg : Weźmij teraz ſyná twego / jedynego twego / którego miłujeƺ / Izááká / á idź do źiemi Moryjá / y tám go ofiáruj ná ofiárę páloną / ná jedney górze / o którejć powiem.
Tedy wſtáwƺy Abráhám bárdzo ráno / ośiodłáł oſłá ſwego / y wźiął dwóch ſług ſwoich z ſobą / y Izááká ſyná ſwego / á nárąbáwƺy drew ná ofiárę páloną / wſtáł y ƺedł ná miejſce / o którem mu Bóg powiedźiáł.
A dniá trzećiego / podnióſłƺy Abráhám ocży ſwe / ujrzáł ono miejſce z dáleká.
Y rzekł Abráhám do ſług ſwoich : Zoſtáńćie wy tu z oſłem / á ja z dźiećięćiem pójdźiemy áż do onąd / á odpráwiwƺy modlitwy / wróćimy śię do was.
Wźiął tedy Abráhám drwá ná ofiárę páloną / y włożył je ná Izááká / ſyná ſwego / á ſám wźiął w rękę ſwoję ogień y miecż / y ƺli obáj poſpołu.
Y rzekł Izáák do Abráhámá / ojcá ſwego / mówiąc : Ojcże mój! A on odpowiedźiáł : Owom ja / ſynu mój. Y rzekł Izáák : Oto ogień y drwá / á gdźież báránek ná ofiárę páloną?
Odpowiedźiáł Abráhám : Bóg ſobie obmyśli báránká ná ofiárę páloną / ſynu mój ; y ƺli obáj poſpołu.
A gdy przyƺli ná miejſce / o którem mu Bóg powiedźiáł / zbudowáł tám Abráhám ołtárz / y ułożył drwá / á związáwƺy Izááká / ſyná ſwego / włożył go ná ołtárz ná drwá.
Y wyćiągnął Abráhám rękę ſwoję / y wźiął miecż / áby zábił ſyná ſwego.
Lecż záwołáł náń Anioł Páńſki z niebá / y rzekł : Abráhámie! Abráhámie! A on rzekł : Owom ja.
Y rzekł Anioł : Nie wyćiągáj ręki twej ná dźiećię / y nie cżyń mu nic ; bom teraz doznał / iż śię ty boiƺ Bogá / y nie ſfolgowáłeś ſynowi twemu / jedynemu twemu / dla mnie.
A podnióſłƺy Abráhám ocży ſwe / ujrzáł / á oto bárán zá nim uwiązł w ćierniu zá rogi ſwoje ; á ƺedłƺy Abráhám / wźiął báráná y ofiárowáł go ná ofiárę páloną / miáſto ſyná ſwego.
Y názwáł Abráhám imię miejſcá onego : Pán obmyśli ; ztądże po dźiś dźień mówią : Ná górze Páńſkiej będźie obmyślono.
Tedy záwołáł Anioł Páńſki ná Abráhámá powtóre z niebá mówiąc :
Przez śiebie ſámego przyśiągłem / mówi Pán : Poniewáżeś to ucżynił / á nie zfolgowáłeś ſynowi twemu / jedynemu twemu ;
Błogoſłáwiąc błogoſłáwić ći będę / á rozmnáżájąc rozmnożę náśienie twoje / jáko gwiázdy niebieſkie / y jáko piáſek / który jeſt ná brzegu morſkim ; á odźiedźicży náśienie twoje brámy nieprzyjáćiół ſwoich.
Y błogoſłáwione będą w náśieniu twojim wƺyſtkie národy źiemi / dla tego / żeś uſłucháł głoſu mego.
Wróćił śię tedy Abráhám do ſług ſwych / y wſtáwƺy / przyƺli poſpołu do Beerſebá ; bo mieƺkáł Abráhám w Beerſebá.
Y ſtáło śię potem / iż oznájmiono Abráhámowi / mówiąc : Oto národźiłá y Melchá ſynów Náchorowi / brátu twemu.
Huſá / pierworodnego ſwego / y Buzá / brátá jego / y Chemuelá / ojcá Arámcżyków.
Y Káſedá / y Káſáná / y Feldáſá / y Jedláfá / y Bátuelá.
A Bátuel ſpłodźił Rebekę ; ośmioro tych dźieći urodźiłá Melchá Náchorowi / brátu Abráhámowemu.
A záłożnicá jego / którey imię Reumá / urodźiłá też Tábę / y Gáhámá / y Táháſá / y Mááchá.
