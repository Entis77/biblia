To gdy śię ſtáło / kuśił Bóg Abráhámá / y rzekł do niego : Abráhámie / A on odpowiedźiał / Owom ja.
Y rzekł <i>Bóg,</i> Weźmi teraz ſyná twego / jedynego twego / którego miłujeƺ / Izááká : á idź do źiemie Moriáh / y tám go ofiáruj ná ofiárę paloną / ná jedney górze / o któreyć powiem.
Tedy wſtawƺy Abráhám bárzo ráno / ośiodłał oſłá ſwego / y wźiął dwu ſług ſwojich z ſobą / y Izááká Syná ſwego / á nárąbawƺy drew ná ofiárę paloną / wſtał y ƺedł ná miejſce o którym mu Bóg powiedźiał.
A dniá trzećiego / podnióżƺy Abráhám oczy ſwe / ujrzał ono miejſce z dáleká.
Y rzekł Abráhám do ſług ſwojich / zoſtańćie wy tu z oſłem / á ja dźiećięćiem pójdźiemy áż do onąd / á odpráwiwƺy modlitwy / wróćiemy śię do was.
Wźiął tedy Abráhám drwá ná ofiárę paloną / y włożył je ná Izááká Syná ſwego / <i>á ſám</i> wźiął w rękę ſwoję ogiė y miecz / y ƺli obá poſpołu.
Y rzekł Izáák do Abráhámá Ojcá ſwego / mówiąc / Ojcze mój? A <i>on</i> odpowiedźiał : Owom ja Synu mój. Y rzekł <i>Izáák,</i> Oto ogień y drwá / á gdźież báránek ná ofiárę paloną?
Odpowiedźiał Abráhám : Bóg ſobie obmyśli báránká ná ofiárę paloną Synu mój / y ƺli obá poſpołu.
A gdy przyƺli ná miejſce o którym mu Bóg powiedźiał / zbudował tám Abráhám ołtarz / y ułożył drwá á związawƺy Izááká ſyná ſwego / włożył go ná ołtarz ná drwá.
Y wyćiągnął Abráhám rękę ſwoję / y wźiął miecz / áby zábił Syná ſwego.
Lecz záwołał nań Anjoł Páńſki z Niebá / y rzekł : Abráhámie / Abráhámie. A <i>on</i> rzekł / Owom ja :
Y rzekł <i>Anjoł</i> Nie wyćiągaj ręki twey ná dźiećię / y nie czyń mu nic : bom teraz doznał / iż śię ty boiƺ Bogá / y nie zfolgowałeś Synowi twemu / jedynemu twemu / dla mnie.
A podnióżƺy Abráhám oczy ſwe / ujrzał á oto báran zá nim uwiązł w ćierniu zá rogi ſwoje : á ƺedƺy Abráhám / wźiął báráná / y ofiárował go ná ofiárę paloną / miáſto Syná ſwego :
Y názwał Abráhám imię miejſcá onego / Pán obmyśli : z tądże podźiś dźień mówią. Ná górze Páńſkiey będźie obmyślono.
Tedy záwołał Anjoł Páńſki ná Abráhámá powtóre z niebá / mówiąc.
Przez śiebie ſámego przyśiągłem / mówi Pan : Ponieważeś to uczynił / á nie z folgowałeś Synowi twemu / jedynemu twemu.
Błogoſłáwiąc błogoſłáwićći będę : á rozmnażájąc rozmnożę naśienie twoje / jáko gwiazdy niebieſkie / y jáko piaſek / który jeſt ná brzegu morſkim : á odźiedźiczy naśienie twoje bramy nieprzyjaćiół ſwojch.
Y błogoſłáwione będą w naśieniu twojem wƺyſtkie narody źiemie / dla tego żeś uſłuchał głoſu mego.
Wróćił śię tedy Abráhám do ſług ſwych / y wſtawƺy przyƺli poſpołu do Beerſebáh : bo mieƺkał Abráhám w Beerſebáh.
Y ſtáło śię potym / iż oznájmiono Abráhámowi mówiąc : Oto národźiłá y Melchá Synów Náchorowi brátu twemu.
Huſá pierworodnego ſwego / y Buſá brátá jego / y Chámuelá Ojcá Arámczyków.
Y Káſedá / y Káſáná / y Feldáſá / y Jedláfá / y Bátuelá.
A Bátuel zpłodźił Rebekę : ośmioro tych dźieći urodźiłá Melchá Nachorowi Brátu Abráhámowemu.
A założnicá jego / którey imię Reumah / urodźiłá też Tábę / y Gáhámá / y Tácháſá / y Mááchá.
