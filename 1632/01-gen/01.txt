Ná pocżątku ſtworzył Bóg niebo y ziemię.
A ziemiá byłá niekƺtáłtowná y próżná / y ćiemność byłá nád przepáśćią / á Duch Boży unáƺáł śię nád wodámi.
Y rzekł Bóg : Niech będzie świátłość ; y ſtáłá śię świátłość.
Y widziáł Bóg świátłość / że byłá dobrá ; y ucżynił Bóg rozdziáł między świátłośćią y między ćiemnośćią.
Y názwáł Bóg świátłość dniem á ćiemność názwáł nocą ; y ſtáł śię wiecżór / y ſtáł śię záránek / dzień pierwƺy.
Potem rzekł Bóg : Niech będzie rozpoſtárćie / w pośrodku wód / á niech dzieli wody od wód.
Y ucżynił Bóg rozpoſtárćie ; ucżynił też rozdziáł między wodámi / które ſą pod rozpoſtárćiem ; y między wodámi / które ſą nád rozpoſtárćiem ; y ſtáło śię ták.
Y názwáł Bóg rozpoſtárćie niebem. Y ſtáł śię wiecżór / y ſtáł śię záránek / dzień wtóry.
Y rzekł Bóg : Niech śię zbiorą wody / które ſą pod niebem / ná jedno miejſce / á niech śię okáże miejſce ſuche ; y ſtáło śię ták.
Y názwáł Bóg ſuche miejſce ziemią / á zebránie wód názwáł morzem.
Y widziáł Bóg / że to było dobre. Potem rzekł Bóg : Niech zrodzi ziemiá tráwę / ziele / wydáwájące náśienie / y drzewo rodzájne / cżyniące owoc / według rodzáju ſwego / któregoby náśienie było w nim ná ziemi ; y ſtáło śię ták.
Y zrodziłá ziemiá tráwę / ziele wydáwájące náśienie / według rodzáju ſwego / y drzewo cżyniące owoc / w którym náśienie jego / według rodzáju ſwego ; y widziáł Bóg / że to było dobre.
Y ſtáł śię wiecżór / y ſtáł śię záránek / dzień trzeći.
Y rzekł Bóg : Niech będą świátłá ná rozpoſtárćiu niebieſkiem / ku rozdzielániu dniá od nocy / á niech będą ná znáki / y pewne cżáſy / y dni / y látá.
Y niech będą zá świátłá ná rozpoſtárćiu niebá / áby świećiły nád ziemią ; y ſtáło śię ták.
Y ucżynił Bóg dwá świátłá wielkie : świátło więkƺe / áby rządziło dzień / á świátło mniejƺe / áby rządziło noc / y gwiázdy.
Y poſtáwił je Bóg ná rozpoſtárćiu niebá / áby świećiły nád ziemią.
Y żeby rządziły dzień y noc / y cżyniły rozdziáł między świátłośćią / y między ćiemnośćią ; y widziáł Bóg / że to było dobre.
Y ſtáł śię wiecżór / y ſtáł śię záránek / dzień cżwárty.
Y rzekł Bóg : Niech hojnie wywiodą wody płáz duƺy żywiącej ; á ptáſtwo niech látá nád ziemią / pod rozpoſtárćiem niebieſkiem.
Y ſtworzył Bóg wieloryby wielkie / y wƺelką duƺę żywiącą płázájącą śię / którą hojnie wywiodły wody / według rodzáju ich ; y wƺelkie ptáſtwo ſkrzydláſte / według rodzáju ich ; y widziáł Bóg / że to było dobre.
Błogoſłáwił im tedy Bóg / mówiąc : Rozrádzájćie śię / y rozmnáżájćie śię / á nápełniájćie wody morſkie ; y ptáſtwo niech śię rozmnáżá ná ziemi.
Y ſtáł śię wiecżór / y ſtáł śię záránek / dzień piąty.
Rzekł też Bóg : Niech wydá ziemiá duƺę żywiącą według rodzáju ſwego ; bydło y płáz / y zwierz ziemſki / według rodzáju ſwego ; y ſtáło śię ták.
Ucżynił tedy Bóg zwierz ziemſki według rodzáju ſwego ; y bydło według rodzáju ſwego ; y wƺelki płáz ziemſki według rodzáju ſwego ; y widziáł Bóg / że to było dobre.
Zátem rzekł Bóg : Ucżyńmy cżłowieká ná wyobráżenie náƺe / według podobieńſtwá náƺego ; á niech pánuje nád rybámi morſkiemi / y nád ptáſtwem niebieſkiem / y nád zwierzęty / y nád wƺyſtką ziemią / y nád wƺelkim płázem / płázájącym śię po ziemi.
Stworzył tedy Bóg cżłowieká ná wyobráżenie ſwoje ; ná wyobráżenie Boże ſtworzył go ; mężcżyznę y niewiáſtę ſtworzył je.
Y błogoſłáwił im Bóg / y rzekł do nich Bóg : Rozrádzájćie śię / y rozmnáżájćie śię / y nápełniájćie ziemię ; y cżyńćie ją ſobie poddáną ; y pánujćie nád rybámi morſkiemi / y nád ptáſtwem niebieſkiem / y nád wƺelkim zwierzem / który śię ruchá ná ziemi.
Y rzekł Bóg : Oto dáłem wám wƺelkie ziele / wydáwájące z śiebie náśienie / które jeſt ná oblicżu wƺyſtkiej ziemi ; y wƺelkie drzewo / ná którem jeſt owoc drzewá / wydáwájące z śiebie náśienie / będzie wám ku pokármowi.
Y wƺelkiemu zwierzowi ziemſkiemu / y wƺyſtkiemu ptáſtwu niebieſkiemu / y wƺelkiej rzecży ruchájącej śię ná ziemi / w której jeſt duƺá żywiącá ; wƺelká járzyná zielá będzie ku pokármowi ; y ſtáło śię ták.
Y widziáł Bóg wƺyſtko / co ucżynił / á oto było bárdzo dobre ; y ſtáł śię wiecżór / y ſtáł śię záránek / dzień ƺóſty.
