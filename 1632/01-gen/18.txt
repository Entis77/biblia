Potem ukázáł mu śię Pán w równinie Mámre / á on śiedźiáł we drzwiách námiotu ſwego / gdy był nájgorętƺy dźień.
A podnióſłƺy ocży ſwe / obácżył / á oto trzej mężowie ſtánęli przećiw niemu ; y ujrzáwƺy je / wybieżáł przećiwko nim ze drzwi námiotu / y pokłonił śię do źiemi.
Y rzekł : Pánie mój / jeſlim teraz ználázł łáſkę w ocżách twoich / nie mijáj / proƺę / ſługi ſwego.
Przynioſą trochę wody / á umyjećie nogi wáƺe / y odpocżniećie pod drzewem.
Y przynioſą kęſ chlebá / że pośilićie ſerce wáƺe ; potem odejdźiećie / dla tegośćie bowiem przyƺli do mnie ſługi ſwego. Tedy rzekli : Ták ucżyń / jákoś powiedźiáł.
Y poſpieƺył śię Abráhám do námiotu do Sáry / y rzekł : Spieƺ śię : rozcżyń trzy miárki mąki świátłej / á ucżyń podpłomyków.
Abráhám záś ƺedł do trzody / y wźiął ćielę młode y wyborne / y dáł je ſłudze / który śię poſpieƺył / y nágotowáł je.
Wźiął też máſłá y mleká / y ćielę / które był nágotowáł / y poſtáwił przed nie / á ſám ſtáł przy nich pod drzewem ; y jedli.
Y rzekli do niego : Gdźie jeſt Sárá / żoná twojá? á on odpowiedźiáł : Oto jeſt w námioćie.
Tedy rzekł Pán : Wrócę śię pewnie do ćiebie o tymże cżáśie w rok / á oto / będźie miáłá ſyná Sárá / żoná twojá ; á Sárá ſłucháłá u drzwi námiotu / które były zá nim.
A Abráhám y Sárá byli ſtárzy / y zeƺli w lećiech ; y przeſtáło było bywáć Sárze według zwycżáju niewiáſt.
Y rozśmiáłá śię Sárá ſámá w ſobie / mówiąc : Gdym śię zſtárzáłá / rozkoƺy záżywáć będę ; y pán mój zſtárzáł śię.
Zátem rzekł Pán do Abráhámá : Cżemu śię rozśmiáłá Sárá / mówiąc : Záż práwdźiwie porodzę / gdym śię zſtárzáłá? Izáli jeſt co trudnego u Páná?
O tymże cżáśie wrócę do ćiebie roku przyƺłego / á Sárá będźie miáłá ſyná.
Y záprzáłá śię Sárá / mówiąc : Nie śmiáłám śię ; bo śię báłá. A Pán rzekł : Nie mów ták ; boś śię śmiáłá.
Potem wſtáli ſtámtąd mężowie oni / y obróćili śię ku Sodomie ; á Abráhám ƺedł z nimi wyprowádzájąc je.
Tedy rzekł Pán : Izáli ja zátáję przed Abráhámem / co mam ucżynić?
Poniewáż Abráhám pewnie rozmnożon będźie w lud wielki y możny / á w nim będą ubłogoſłáwione wƺyſtkie národy źiemi.
Znám go bowiem ; przetoż przykáże ſynom ſwoim / y domowi ſwemu po ſobie / áby ſtrzegli drogi Páńſkiej / y cżynili ſpráwiedliwość y ſąd ; áby przywiódł Pán ná Abráhámá / co mu powiedźiáł.
Rzekł tedy Pán : Krzyk Sodomy y Gomorry / iż śię rozmnożył grzech ich / iż bárdzo oćiężáł ;
Zſtąpię teraz / á obácżę / jeſli śię według krzyku tego / który mię doƺedł / do końcá ſpráwują ; á jeſliż nie / ábym śię wżdy dowiedźiáł.
Y obróćili śię ſtámtąd mężowie / y poƺli do Sodomy ; lecż Abráhám jeƺcże ſtáł przed Pánem.
Y przyſtąpił Abráhám / y rzekł : Izáli też zátráćiƺ ſpráwiedliwego z niezbożnym?
Jeſli ſnáć będźie pięćdźieśiąt ſpráwiedliwych w tem mieśćie / izáli je wytráćiƺ / á nie przepuśćiƺ miejſcu temu dla pięćdźieśiąt ſpráwiedliwych / którzy w nim ſą?
Niech to nie będźie u ćiebie / ábyś ucżynić miał rzecż tákową / y zábił ſpráwiedliwego z niezbożnym / á żeby był ſpráwiedliwy / jáko niezbożny. Niech to nie będźie u ćiebie. Izáli Sędźiá wƺyſtkiej źiemi nie ucżyni ſpráwiedliwośći?
Tedy rzekł Pán : Jeſli znájdę w Sodomie pięćdźieśiąt ſpráwiedliwych w ſámem mieśćie / odpuƺcżę wƺyſtkiemu miejſcu dla nich.
A odpowiádájąc Abráhám rzekł : Otom teraz zácżął mówić do Páná mego / ácżem ja proch y popiół.
A jeſliby niedoſtáwáło do pięćiudźieśiąt ſpráwiedliwych / pięćiu / izáli wytráćiƺ dla tych pięćiu wƺyſtko miáſto? Y rzekł Pán : Nie wytrácę / jeſli tám znájdę cżterdźieſtu y pięćiu.
Ná to jeƺcże mówiąc do niego Abráhám rzekł : A jeſliby śię ich tám ználázło cżterdźieśći? y odpowiedźiáł : Nie ucżynię nic dla tych cżterdźieſtu.
Y rzekł Abráhám : Proƺę niech śię nie gniewá Pán mój / że jeƺcże mówić będę : A jeſliby śię ich tám ználázło trzydźieśći? odpowiedźiáł : Nie ucżynię / jeſliż tám znájdę trzydźieſtu.
Tedy jeƺcże rzekł Abráhám : Otom teraz zácżął mówić do Páná mego : A jeſliby śię ich tám ſnáć ználázło dwádźieśćiá? odpowiedźiáł Pán : Nie zátrácę y dla tych dwudźieſtu.
Nád to rzekł Abráhám : Proƺę niech śię nie gniewá Pán mój / że jeƺcże ráz tylko przemówię : A jeſliby śię ich tám ználázło dźieśięć? Tedy rzekł Pán : Nie wytrácę y dla tych dźieśięćiu.
Y poƺedł Pán ſkońcżywƺy rozmowę z Abráhámem ; á Abráhám wróćił śię do miejſcá ſwego.
