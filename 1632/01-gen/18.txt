Potem ukázáł mu śię Pán w równinie Mámre / á on śiedziáł we drzwiách námiotu ſwego / gdy był nájgorętƺy dzień.
A podnióſłƺy ocży ſwe / obácżył / á oto trzej mężowie ſtánęli przećiw niemu ; y ujrzáwƺy je / wybieżáł przećiwko nim ze drzwi námiotu / y pokłonił śię do ziemi.
Y rzekł : Pánie mój / jeźlim teráz ználázł łáſkę w ocżách twoich / nie mijáj / proƺę / ſługi ſwego.
Przynioſą trochę wody / á umyjećie nogi wáƺe / y odpocżniećie pod drzewem.
Y przynioſą kęſ chlebá / że pośilićie ſerce wáƺe ; potem odejdziećie / dlá tegośćie bowiem przyƺli do mnie ſługi ſwego. Tedy rzekli : Ták ucżyń / jákoś powiedziáł.
Y poſpieƺył śię Abráhám do námiotu do Sáry / y rzekł : Spieƺ śię : rozcżyń trzy miárki mąki świátłej / á ucżyń podpłomyków.
Abráhám záś ƺedł do trzody / y wziął ćielę młode y wyborne / y dáł je ſłudze / który śię poſpieƺył / y nágotowáł je.
Wziął też máſłá y mleká / y ćielę / które był nágotowáł / y poſtáwił przed nie / á ſám ſtáł przy nich pod drzewem ; y jedli.
Y rzekli do niego : Gdzie jeſt Sárá / żoná twojá? á on odpowiedziáł : Oto jeſt w námioćie.
Tedy rzekł Pán : Wrócę śię pewnie do ćiebie o tymże cżáśie w rok / á oto / będzie miáłá ſyná Sárá / żoná twojá ; á Sárá ſłucháłá u drzwi námiotu / które były zá nim.
A Abráhám y Sárá byli ſtárzy / y zeƺli w lećiech ; y przeſtáło było bywáć Sárze według zwycżáju niewiáſt.
Y rozśmiáłá śię Sárá ſámá w ſobie / mówiąc : Gdym śię zſtárzáłá / rozkoƺy záżywáć będę ; y pán mój zſtárzáł śię.
Zátem rzekł Pán do Abráhámá : Cżemu śię rozśmiáłá Sárá / mówiąc : Záż práwdziwie porodzę / gdym śię zſtárzáłá? Izáli jeſt co trudnego u Páná?
O tymże cżáśie wrócę do ćiebie roku przyƺłego / á Sárá będzie miáłá ſyná.
Y záprzáłá śię Sárá / mówiąc : Nie śmiáłám śię ; bo śię báłá. A Pán rzekł : Nie mów ták ; boś śię śmiáłá.
Potem wſtáli ſtámtąd mężowie oni / y obróćili śię ku Sodomie ; á Abráhám ƺedł z nimi wyprowádzájąc je.
Tedy rzekł Pán : Izáli já zátáję przed Abráhámem / co mám ucżynić?
Poniewáż Abráhám pewnie rozmnożon będzie w lud wielki y możny / á w nim będą ubłogoſłáwione wƺyſtkie národy ziemi.
Znám go bowiem ; przetoż przykáże ſynom ſwoim / y domowi ſwemu po ſobie / áby ſtrzegli drogi Páńſkiej / y cżynili ſpráwiedliwość y ſąd ; áby przywiódł Pán ná Abráhámá / co mu powiedziáł.
Rzekł tedy Pán : Krzyk Sodomy y Gomorry / iż śię rozmnożył grzech ich / iż bárdzo oćiężáł ;
Zſtąpię teráz / á obácżę / jeźli śię według krzyku tego / który mię doƺedł / do końcá ſpráwują ; á jeźliż nie / ábym śię wżdy dowiedziáł.
Y obróćili śię ſtámtąd mężowie / y poƺli do Sodomy ; lecż Abráhám jeƺcże ſtáł przed Pánem.
Y przyſtąpił Abráhám / y rzekł : Izáli też zátráćiƺ ſpráwiedliwego z niezbożnym?
Jeźli ſnáć będzie pięćdzieśiąt ſpráwiedliwych w tem mieśćie / izáli je wytráćiƺ / á nie przepuśćiƺ miejſcu temu dlá pięćdzieśiąt ſpráwiedliwych / którzy w nim ſą?
Niech to nie będzie u ćiebie / ábyś ucżynić miáł rzecż tákową / y zábił ſpráwiedliwego z niezbożnym / á żeby był ſpráwiedliwy / jáko niezbożny. Niech to nie będzie u ćiebie. Izáli Sędziá wƺyſtkiej ziemi nie ucżyni ſpráwiedliwośći?
Tedy rzekł Pán : Jeźli znájdę w Sodomie pięćdzieśiąt ſpráwiedliwych w ſámem mieśćie / odpuƺcżę wƺyſtkiemu miejſcu dlá nich.
A odpowiádájąc Abráhám rzekł : Otom teráz zácżął mówić do Páná mego / ácżem já proch y popiół.
A jeźliby niedoſtáwáło do pięćiudzieśiąt ſpráwiedliwych / pięćiu / izáli wytráćiƺ dlá tych pięćiu wƺyſtko miáſto? Y rzekł Pán : Nie wytrácę / jeźli tám znájdę cżterdzieſtu y pięćiu.
Ná to jeƺcże mówiąc do niego Abráhám rzekł : A jeźliby śię ich tám ználázło cżterdzieśći? y odpowiedziáł : Nie ucżynię nic dlá tych cżterdzieſtu.
Y rzekł Abráhám : Proƺę niech śię nie gniewá Pán mój / że jeƺcże mówić będę : A jeźliby śię ich tám ználázło trzydzieśći? odpowiedziáł : Nie ucżynię / jeźliż tám znájdę trzydzieſtu.
Tedy jeƺcże rzekł Abráhám : Otom teráz zácżął mówić do Páná mego : A jeźliby śię ich tám ſnáć ználázło dwádzieśćiá? odpowiedziáł Pán : Nie zátrácę y dlá tych dwudzieſtu.
Nád to rzekł Abráhám : Proƺę niech śię nie gniewá Pán mój / że jeƺcże ráz tylko przemówię : A jeźliby śię ich tám ználázło dzieśięć? Tedy rzekł Pán : Nie wytrácę y dlá tych dzieśięćiu.
Y poƺedł Pán ſkońcżywƺy rozmowę z Abráhámem ; á Abráhám wróćił śię do miejſcá ſwego.
