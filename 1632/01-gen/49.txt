Wezwáł tedy Jákób ſynów ſwoich y rzekł : Zbierzćie śię / á oznájmię wám / co má przyjść ná wáſ w oſtátnie dni.
Zbierzćie śię / y ſłuchájćie ſynowie Jákóbowi / á ſłuchájćie Izráelá / ojcá wáƺego.
Ruben pierworodny mój / tyś moc mojá / y pocżątek śiły mojej / zácny doſtojeńſtwem / y zácny męſtwem.
Spłynieƺ jáko wodá : nie będźieƺ zácnym / boś wſtąpił ná łoże ojcá twego / y ſplugáwiłeś łoże moje / y zginęło doſtojeńſtwo twoje.
Symeon y Lewi / bráćiá / nácżyniá niepráwośći miecże ich.
W rádę ich niecháj nie wchodźi duƺá mojá / á z zgromádzeniem ich niech śię nie jednocży ſłáwá mojá ; bo w zápálcżywośći ſwej zábili mężá / á w ſwej woli wywróćili mur.
Przeklętá zápálcżywość ich / iż uporná / y gniew ich / iż zátwárdźiáły. Rozdźielę je w Jákóbie / á rozproƺę je w Izráelu.
Judá / tyś jeſt / ćiebie chwálić będą bráćiá twoi ; ręká twojá będźie ná ƺyi nieprzyjáćiół twoich ; kłániáć śię tobie będą ſynowie ojcá twego.
Szcżenię lwie Judá / od łupu / ſynu mój / wróćiłeś śię ; ſkłonił śię y położył śię jáko lew / y jáko lwicá / á któż go obudźi?
Nie będźie odjęte ſceptrum od Judy / áni Zákonodáwcá od nóg jego / áż przyjdźie Szylo / y jemu będźie oddáne poſłuƺeńſtwo národów.
Uwiąże u winnej máćicy oślę ſwe / á u wybornej máćicy winnej oślątko oślicy ſwej ; omyje w winie ƺátę ſwoję / á we krwi jágód winnych odźienie ſwoje.
Cżerwieńƺe ocży jego nád wino / á bielƺe zęby jego nád mleko.
Zábulon ná brzegu morſkim mieƺkáć będźie / y przy porćie okrętów / á gránice jego áż do Sydonu.
Iſáƺár jáko ośieł kośćiſty / leżący między dwomá brzemiony.
Upátrzył pokój / że jeſt dobry / y źiemię / że piękná / náchylił rámię ſwe ku noƺeniu / dlá tegoż będźie hołd dáwáł.
Dán ſądźić będźie lud ſwój / jáko jedno z pokoleń Izráelſkich.
Dán będźie wężem ná drodze / żmiją ná śćieżce / kąſájąc pięty końſkie / że ſpádnie názád jeźdźiec jego.
Zbáwieniá twego ocżekiwám Pánie!
Gád od wojſká zwyćiężony będźie ; ále y on potem zwyćięży.
Z Aſerá tłuſty chleb jego / á on wydá rozkoƺy królewſkie.
Neftáli jáko łáni wypuƺcżoná / mówiąc piękne ſłowá.
Látorośl płodná Józef / látorośl wyráſtájącá nád źródłem / á látorośli jego rozchodzą śię po murze.
Acż gorzkośćią nápełnili go / y ſtrzeláli náń / á nienáwidźili go ſtrzelcy :
Jednák zoſtáł potężny łuk jego / á zmocniły śię rámioná rąk jego / w rękách mocnego Bogá Jákóbowego / ſkąd śię ſtáł páſterzem y opoką Izráelową.
Od Bogá ojcá twego / który ćię wſpomógł / y od Wƺechmogącego / któryć błogoſłáwił błogoſłáwieńſtwy niebieſkiemi z wyſoká / y błogoſłáwieńſtwy przepáśći leżącej głęboko / y błogoſłáwieńſtwy pierśi y żywotá.
Błogoſłáwieńſtwá ojcá twego mocniejƺe będą nád błogoſłáwieńſtwá przodków moich / áż do gránic págórków wiecżnych ; będą nád głową Józefową / y nád wierzchem głowy odłącżonego między bráćią ſwą.
Benjámin jáko wilk porywájący / poránu jeść będźie łup / á wiecżór będźie dźielił korzyść.
Teć wƺyſtkie ſą dwánáśćie pokoleniá Izráelſkie / y to / co im powiedźiáł ojćiec ich / y błogoſłáwił im ; káżdemu według błogoſłáwieńſtwá jego błogoſłáwił im.
A rozkázáł im / y rzekł do nich : Já będę przyłącżon do ludu mego ; pogrzebćież mię z ojcy moimi w jáſkini / którá jeſt ná polu Efroná Hetejcżyká ;
W jáſkini / którá jeſt ná polu Máchpelá / którá jeſt ná przećiwko Mámre w źiemi Chánánejſkiej / którą kupił Abráhám z rolą od Efroná Hetejcżyká / w ośiádłość grobu.
Tám pogrzebiono Abráhámá / y Sárę / żonę jego ; tám pogrzebiono Izááká / y Rebekę żonę jego ; támem też pogrzebł Liją.
A kupiono tę rolą y jáſkinią / którá ná niej / od ſynów Hetowych.
Tedy przeſtáwƺy Jákób mówić do ſynów ſwoich / złożył nogi ſwe ná łoże y umárł / y przyłącżon jeſt do ludu ſwego.
