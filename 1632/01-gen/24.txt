A Abráhám był ſtáry y podeƺły w lećiech / á Pán błogoſłáwił mu we wƺyſtkiem.
Tedy rzekł Abráhám do ſtárƺego ſługi ſwego w domu ſwym / który wƺyſtkiem rządźił / co miáł : Połóż / proƺę / rękę twoję pod biodro moje ;
A záprzyśięgnę ćię przez Páná / Bogá niebá / y Bogá źiemi / ábyś nie bráł żony ſynowi memu z córek Chánánejſkich / między któremi já mieƺkám ;
Ale pójdźieƺ do źiemi mojej / y do rodźiny mojej / á ſtámtąd weźmieƺ żonę Izáákowi / ſynowi memu.
Tedy mu rzekł ſługá : A jeźliby ſnáć nie chćiáłá niewiáſtá oná iść ze mną do tej źiemi / mámże odprowádźić ſyná twego do źiemi / z którejś ty wyƺedł?
Y rzekł mu Abráhám : Strzeż śię / ábyś tám záśię nie záprowádzáł ſyná mego.
Pán Bóg niebieſki / który mię wźiął z domu ojcá mego / y z źiemi rodźiny mojej / y który mówił ze mną / á który mi przyśiągł / mówiąc : Náśieniu twemu dám źiemię tę ; on pośle Aniołá ſwego przed oblicżem twojim / y weźmieƺ ſtámtąd żonę ſynowi memu.
A jeźliby niechćiáłá oná niewiáſtá iść z tobą / wolny będźieƺ od tego poprzyśiężeniá mego ; tylko ſyná mego nie záprowádzáj tám.
Podłożył tedy ſługá rękę ſwoję pod biodro Abráhámá / páná ſwego / y przyśiągł mu ná to.
Y wźiął on ſługá dźieśięć wielbłądów / z wielbłądów páná ſwego / y poƺedł ; bo wƺyſtkie dobrá páná ſwego miáł w rękách ſwych ; á wſtáwƺy puśćił śię do Arám Náháráim / do miáſtá Náchorowego.
Y poſtáwił wielbłądy przed miáſtem u ſtudni wody / pod wiecżór / tego cżáſu / którego zwykły niewiáſty wychodźić cżerpáć wodę.
Y rzekł : Pánie / Boże páná mego Abráhámá! Niech mię proƺę potká dźiś / cżego żądám / á ucżyń miłośierdźie z pánem moim Abráhámem.
Oto / já ſtoję u ſtudni / á córki obywáteli miáſtá tego wyjdą cżerpáć wodę ;
Pánienká tedy / do którejbym rzekł : Náchyl proƺę wiádrá twego / że śię nápiję / á onáby rzekłá : Pij / owƺem y wielbłądy twoje nápoję ; tá niech będźie / którąś zgotowáł ſłudze twemu Izáákowi ; á po tem poznám / żeś ucżynił miłośierdźie z pánem moim.
Y ſtáło śię / że pierwej niż przeſtáł mówić / oto / Rebeká wychodźiłá / którá śię urodźiłá Bátuelowi / ſynowi Melchy / żony Náchorá / brátá Abráhámowego / nioſąc wiádro ná rámieniu ſwem.
A dźiewecżká oná byłá bárdzo piękná ná wejrzeniu / pánná / á której mąż nieuznáł ; tá przyƺedłƺy do ſtudni / nápełniłá wiádro ſwe / y wrácáłá śię.
Tedy zábieżáł jej on ſługá / y rzekł : Dáj mi śię proƺę nápić trochę wody z wiádrá twego.
A oná rzekłá : Pij / pánie mój / y prędko złożyłá wiádro ſwe ná rękę ſwoję / y dáłá mu pić.
A gdy mu śię dáłá nápić / rzekłá : Y wielbłądom twoim nácżerpię / áż śię nápiją.
Y wyláłá prędko wodę z wiádrá ſwego w koryto / á bieżáwƺy jeƺcże do ſtudni cżerpáć / nácżerpáłá wƺyſtkim wielbłądom jego.
A on mąż zdumiewáł śię nád nią / uwáżájąc z milcżeniem / jeźli mu Pán zdárzył drogę jego / cżyli nie.
Y gdy śię nápiły wielbłądy / wyjął on mąż náuƺnicę złotą / którá wáżyłá pół ſyklá / y dwie mánele / y dáł ná ręce jej / które wáżyły dźieśięć ſyklów złotá.
Y rzekł : Cżyjáś ty córká / powiedz mi / proƺę? á jeźli w domu ojcá twego miejſce dlá náſ / gdźiebyśmy przenocowáli?
A oná mu rzekłá : Jeſtem córká Bátuelá / ſyná Melchy / którego urodźiłá Náchorowi.
Nád to rzekłá mu : Jeſt u náſ doſyć plew y páſtwy / y miejſce do przenocowániá.
Y pokłonił śię on cżłowiek y dáł chwáłę Pánu /
Y rzekł : Błogoſłáwiony Pán / Bóg páná mego Abráhámá / który nie oddálił miłośierdźiá ſwego y práwdy ſwojej od páná mojego / ábowiem gdym był w drodze / przyprowádźił mię Pán w dom bráći páná mego.
Bieżáłá tedy dźiewecżká / y oznájmiłá w domu mátki ſwej / jáko śię co ſtáło.
Y miáłá Rebeká brátá imieniem Lábáná ; y wybieżáł Lábán przećiwko onemu mężowi áż ku ſtudni.
Bo ujrzáwƺy náuƺnicę / y mánele ná ręku śioſtry ſwej / y uſłyƺáwƺy ſłowá Rebeki / śioſtry ſwej / mówiącej : Ták mówił do mnie ten mąż ; przyƺedł do onego mężá / á oto / on ſtáł przy wielbłądách u ſtudni.
Y rzekł do niego : Wnijdź błogoſłáwiony Páńſki ; przecżbyś ſtáł ná dworze / jużem já nágotowáł dom / y miejſce wielbłądom?
Tedy wƺedł mąż on w dom ; á Lábán rozśiodłáł wielbłądy / y dáł plew y páſtwy wielbłądom / y wody dlá umyćiá nóg jego / y nóg mężów onych / którzy z nim byli.
Y położył przedeń / coby jádł ; ále on rzekł : Nie będę jádł / áż pierwej odpráwię rzecż ſwoję. Tedy rzekł Lábán : Mówże.
Y rzekł : Jám jeſt ſługá Abráhámów ;
A Pán ubłogoſłáwił páná mego bárdzo / y ſtáł śię możnym ; bo mu nádáł owiec / y wołów / y ſrebrá / y złotá / y ſług / y ſłużebnic / y wielbłądów / y oſłów.
A urodźiłá Sárá / żoná páná mego / ſyná pánu memu / w ſtárośći jego / któremu dáł wƺyſtko / co má.
Y poprzyśiągł mię pán mój / mówiąc : Nie weźmieƺ żony ſynowi memu z córek Chánánejſkich / w których źiemi já mieƺkám ;
Ale do domu ojcá mego pójdźieƺ y do rodźiny mojej ; á weźmieƺ ſtámtąd żonę ſynowi mojemu.
Y rzekłem do páná mego : Nie pójdźie ſnáć tá niewiáſtá ze mną.
Tedy mi odpowiedźiáł : Pán / przed któregom já oblicżem chodźił / pośle Aniołá ſwego z tobą / y poƺcżęśći drogę twoję ; á weźmieƺ żonę ſynowi memu z rodźiny mojej / y z domu ojcá mego.
Tedy wolen będźieƺ od poprzyśiężeniá mego / gdy przyjdźieƺ do rodźiny mojej ; ále jeźlićby jej nie dáno / wolen będźieƺ od poprzyśiężeniá mego.
Przyƺedłem tedy dźiś do ſtudni / y rzekłem : Pánie / Boże páná mego Abráhámá / jeźliż ty teráz ƺcżęśćiƺ drogę moję / którą já idę :
Oto / já ſtoję u ſtudni wody ; niechájże pánienká / którá wynijdźie cżerpáć wodę / á gdybym jej rzekł : Dáj mi proƺę nápić śię trochę wody z wiádrá twego ;
A onáby rzekłá do mnie : Y ty pij / nácżerpám też y wielbłądom twoim : tá będźie żoną / którą zgotowáł Pán ſynowi páná mego.
Niżelim já tedy przeſtáł mówić w ſercu ſwem / oto / Rebeká wychodźiłá / nioſąc wiádro ſwe ná rámieniu ſwem / y przyƺłá do ſtudni / á cżerpáłá ; którejm rzekł : Dáj mi pić proƺę.
Oná tedy prędko złożywƺy wiádro z śiebie / rzekłá : Pij / owƺem y wielbłądy twoje nápoję. Y piłem ; nápoiłá też y wielbłądy.
Y pytáłem jej / mówiąc : Cżyjáś ty córká? y odpowiedźiáłá : Jeſtem córká Bátuelá / ſyná Náchorowego / którego mu urodźiłá Melchá / tedym włożył náuƺnice ná twárz jej / y mánele ná ręce jej.
Zátem pokłoniwƺy śię / dáłem chwáłę Pánu / y błogoſłáwiłem Pánu / Bogu páná mego Abráhámá / który mię prowádźił drogą práwą / ábym wźiął córkę brátá páná mego / ſynowi jego
Przetoż teráz / jeźli chcećie ucżynić miłośierdźie y práwdę z pánem moim / oznájmijćie mi ; á jeźli nie / powiedzćie mi też / żebym śię obróćił ná práwo álbo ná lewo.
Tedy odpowiedźiáł Lábán y Bátuel / mówiąc : Od Páná tá rzecż wyƺłá ; my tobie w nicżem przecżyć nie możemy.
Oto Rebeká przed tobą ; weźmij ją / á idź ; á niech będźie żoną ſyná páná twego / jáko rzekł Pán.
Y ſtáło śię / gdy uſłyƺáł ſługá Abráhámów ſłowá ich / pokłonił śię áż do źiemi Pánu.
Zátem wyjął ſługá on nácżyniá ſrebrne / y nácżyniá złote / y ƺáty / á oddáł je Rebece ; dáł też upominki drogie brátu jej / y mátce jej.
Jedli tedy y pili / on y mężowie / którzy z nim byli / y zoſtáli tám ná noc ; á ráno wſtáwƺy / rzekł : Puśććie mię do páná mego.
Y rzekł brát jej / y mátká jej : Niecháj pomieƺká z námi dźiewecżká dźień / álbo dźieśięć ; potem pójdźieƺ.
A on rzekł do nich : Niezátrzymywájćie mię / gdyż Pán poƺcżęśćił drogę moję / puśććie mię / ábym jecháł do páná mego.
Zátem rzekli : Záwołájmy dźiewecżki / á ſpytájmy / co ná to rzecże.
Tedy záwołáli Rebeki / y mówili do niej : Chceƺże jecháć z tym cżłowiekiem? A oná odpowiedźiáłá : Pojádę.
Y puśćili Rebekę śioſtrę ſwoję / z mámką jej / y ſługę Abráhámowego / z mężámi jego.
Tedy błogoſłáwili Rebece / mówiąc jej : Sioſtráś náƺá / rozmnóż śię w tyśiąc tyśięcy / á niech pośiądźie náśienie twoje brámy nieprzyjáćiół ſwych.
Tedy wſtáwƺy Rebeká z dźiewecżkámi ſwemi / y wśiádłƺy ná wielbłądy jecháły zá onym mężem ; y wźiął ſługá on Rebekę / y odjecháł.
A Izáák wrácáł śię z przechádzki od ſtudni / którą zowią Żywiącego y Widzącego mię ; bo mieƺkáł w źiemi południowej.
A wyƺedł był Izáák / dlá modlitwy ná pole pod wiecżór / y podnióſłƺy ocży ſwe / ujrzáł wielbłądy przychodzące.
Podnioſłá też y Rebeká ocży ſwe / y ujrzáłá Izááká / y zśiádłá z wielbłądá ;
Bo rzekłá do ſługi : Cóż on zá mąż / który idźie przez pole przećiwko nám? Y odpowiedźiáł ſługá : Ten jeſt pán mój. A oná wźiąwƺy ráńtuch nákryłá śię.
Y powiedźiáł on ſługá Izáákowi wƺyſtko / co ſpráwił.
Y wprowádźił ją Izáák do námiotu Sáry / mátki ſwojej ; y wźiął Rebekę / y byłá mu żoną / y miłowáł ją. Y ućieƺył śię Izáák po śmierći mátki ſwojej.
