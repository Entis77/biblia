A Abráhám był ſtáry y podeƺły w lećiech / á Pán błogoſłáwił mu we wƺyſtkiem.
Tedy rzekł Abráhám do ſtárƺego ſługi ſwego w domu ſwym / który wƺyſtkiem rządźił / co miał : Połóż / proƺę / rękę twoję pod biodro moje ;
A záprzyśięgnę ćię przez Páná / Bogá niebá / y Bogá źiemi / ábyś nie brał żony ſynowi memu z córek Chánánejſkich / miedzy któremi ja mieƺkám ;
Ale pójdźieƺ do źiemi mojey / y do rodźiny mojey / á ſtámtąd weźmieƺ żonę Izáákowi / ſynowi memu.
Tedy mu rzekł ſługá : A jeſliby ſnadź nie chćiáłá niewiáſtá oná iść ze mną do tey źiemi / mámże odprowádźić ſyná twego do źiemi / z którejś ty wyƺedł?
Y rzekł mu Abráhám : Strzeż śię / ábyś tám záśię nie záprowádzał ſyná mego.
Pán Bóg niebieſki / który mię wźiął z domu ojcá mego / y z źiemi rodźiny mojey / y który mówił ze mną / á który mi przyśiągł / mówiąc : Náśieniu twemu dam źiemię tę ; on pośle Aniołá ſwego przed oblicżem twojim / y weźmieƺ ſtámtąd żonę ſynowi memu.
A jeſliby niechćiáłá oná niewiáſtá iść z tobą / wolny będźieƺ od tego poprzyśiężeniá mego ; tylko ſyná mego nie záprowádzáj tám.
Podłożył tedy ſługá rękę ſwoję pod biodro Abráhámá / páná ſwego / y przyśiągł mu ná to.
Y wźiął on ſługá dźieśięć wielbłądów / z wielbłądów páná ſwego / y poƺedł ; bo wƺyſtkie dobrá páná ſwego miał w rękách ſwych ; á wſtáwƺy puśćił śię do Arám Náháráim / do miáſtá Náchorowego.
Y poſtáwił wielbłądy przed miáſtem u ſtudni wody / pod wiecżór / tego cżáſu / którego zwykły niewiáſty wychodźić cżerpáć wodę.
Y rzekł : Pánie / Boże páná mego Abráhámá! Niech mię proƺę potká dźiś / cżego żądám / á ucżyń miłośierdźie z pánem moim Abráhámem.
Oto / ja ſtoję u ſtudni / á córki obywáteli miáſtá tego wyjdą cżerpáć wodę ;
Pánienká tedy / do którejbym rzekł : Náchyl proƺę wiádrá twego / że śię nápiję / á onáby rzekłá : Pij / owƺem y wielbłądy twoje nápoję ; tá niech będźie / którąś zgotował ſłudze twemu Izáákowi ; á po tym poznam / żeś ucżynił miłośierdźie z pánem moim.
Y ſtáło śię / że pierwej niż przeſtał mówić / oto / Rebeká wychodźiłá / która śię urodźiłá Bátuelowi / ſynowi Melchy / żony Náchorá / brátá Abráhámowego / nioſąc wiádro ná rámieniu ſwem.
A dźiewecżká oná byłá bárzo piękná ná wejrzeniu / pánná / á którey mąż nieuznał ; tá przyƺedƺy do ſtudni / nápełniłá wiádro ſwe / y wrácáłá śię.
Tedy zábieżał jey on ſługá / y rzekł : Dáj mi śię proƺę nápić trochę wody z wiádrá twego.
A oná rzekłá : Pij / pánie mój / y prędko złożyłá wiádro ſwe ná rękę ſwoję / y dáłá mu pić.
A gdy mu śię dáłá nápić / rzekłá : Y wielbłądom twoim nácżerpię / áż śię nápiją.
Y wyláłá prędko wodę z wiádrá ſwego w koryto / á bieżáwƺy jeƺcże do ſtudni cżerpáć / nácżerpáłá wƺyſtkim wielbłądom jego.
A on mąż zdumiewał śię nád nią / uwáżájąc z milcżeniem / jeſli mu Pán zdárzył drogę jego / cżyli nie.
Y gdy śię nápiły wielbłądy / wyjął on mąż náuƺnicę złotą / która wáżyłá pół ſyklá / y dwie mánele / y dał ná ręce jey / które wáżyły dźieśięć ſyklów złotá.
Y rzekł : Cżyjáś ty córká / powiedz mi / proƺę? á jeſli w domu ojcá twego miejſce dla nas / gdźiebyſmy przenocowáli?
A oná mu rzekłá : Jeſtem córká Bátuelá / ſyná Melchy / którego urodźiłá Náchorowi.
Nád to rzekłá mu : Jeſt u nas doſyć plew y páſtwy / y miejſce do przenocowánia.
Y pokłonił śię on cżłowiek y dał chwałę Pánu /
Y rzekł : Błogoſłáwiony Pán / Bóg páná mego Abráhámá / który nie oddálił miłośierdźiá ſwego y prawdy ſwojey od páná mojego / ábowiem gdym był w drodze / przyprowádźił mię Pán w dom bráći páná mego.
Bieżáłá tedy dźiewecżká / y oznájmiłá w domu mátki ſwej / jáko śię co ſtáło.
Y miáłá Rebeká brátá imieniem Lábáná ; y wybieżał Lábán przećiwko onemu mężowi áż ku ſtudni.
Bo ujrzáwƺy náuƺnicę / y mánele ná ręku śioſtry ſwej / y uſłyƺáwƺy ſłowá Rebeki / śioſtry ſwej / mówiącey : Ták mówił do mnie ten mąż ; przyƺedł do onego mężá / á oto / on ſtał przy wielbłądách u ſtudni.
Y rzekł do niego : Wnijdź błogoſłáwiony Páńſki ; przecżbyś ſtał ná dworze / jużem ja nágotował dom / y miejſce wielbłądom?
Tedy wƺedł mąż on w dom ; á Lábán rozśiodłał wielbłądy / y dał plew y páſtwy wielbłądom / y wody dla umyćiá nóg jego / y nóg mężów onych / którzy z nim byli.
Y położył przedeń / coby jádł ; ále on rzekł : Nie będę jádł / áż pierwej odpráwię rzecż ſwoję. Tedy rzekł Lábán : Mówże.
Y rzekł : Jam jeſt ſługá Abráhámów ;
A Pán ubłogoſłáwił páná mego bárzo / y ſtał śię możnym ; bo mu nádał owiec / y wołów / y ſrebrá / y złotá / y ſług / y ſłużebnic / y wielbłądów / y oſłów.
A urodźiłá Sárá / żoná páná mego / ſyná pánu memu / w ſtárośći jego / któremu dał wƺyſtko / co ma.
Y poprzyśiągł mię pán mój / mówiąc : Nie weźmieƺ żony ſynowi memu z córek Chánánejſkich / w których źiemi ja mieƺkám ;
Ale do domu ojcá mego pójdźieƺ y do rodźiny mojey ; á weźmieƺ ſtámtąd żonę ſynowi mojemu.
Y rzekłem do páná mego : Nie pójdźie ſnadź tá niewiáſtá ze mną.
Tedy mi odpowiedźiał : Pán / przed któregom ja oblicżem chodźił / pośle Aniołá ſwego z tobą / y poƺcżęśći drogę twoję ; á weźmieƺ żonę ſynowi memu z rodźiny mojey / y z domu ojcá mego.
Tedy wolen będźieƺ od poprzyśiężeniá mego / gdy przyjdźieƺ do rodźiny mojey ; ále jeſlićby jey nie dáno / wolen będźieƺ od poprzyśiężeniá mego.
Przyƺedłem tedy dźiś do ſtudni / y rzekłem : Pánie / Boże páná mego Abráhámá / jeſliż ty teraz ƺcżęśćiƺ drogę moję / którą ja idę :
Oto / ja ſtoję u ſtudni wody ; niechájże pánienká / która wynijdźie cżerpáć wodę / á gdybym jey rzekł : Dáj mi proƺę nápić śię trochę wody z wiádrá twego ;
A onáby rzekłá do mnie : Y ty pij / nácżerpám też y wielbłądom twoim : tá będźie żoną / którą zgotował Pán ſynowi páná mego.
Niżelim ja tedy przeſtał mówić w ſercu ſwem / oto / Rebeká wychodźiłá / nioſąc wiádro ſwe ná rámieniu ſwem / y przyƺłá do ſtudni / á cżerpáłá ; którejm rzekł : Dáj mi pić proƺę.
Oná tedy prędko złożywƺy wiádro z śiebie / rzekłá : Pij / owƺem y wielbłądy twoje nápoję. Y piłem ; nápoiłá też y wielbłądy.
Y pytáłem jey / mówiąc : Cżyjáś ty córká? y odpowiedźiáłá : Jeſtem córká Bátuelá / ſyná Náchorowego / którego mu urodźiłá Melchá / tedym włożył náuƺnice ná twárz jey / y mánele ná ręce jey.
Zátem pokłoniwƺy śię / dáłem chwałę Pánu / y błogoſłáwiłem Pánu / Bogu páná mego Abráhámá / który mię prowádźił drogą práwą / ábym wźiął córkę brátá páná mego / ſynowi jego
Przetoż teraz / jeſli chcećie ucżynić miłośierdźie y práwdę z pánem moim / oznájmijćie mi ; á jeſli nie / powiedzćie mi też / żebym śię obróćił ná práwo álbo ná lewo.
Tedy odpowiedźiał Lábán y Bátuel / mówiąc : Od Páná tá rzecż wyƺłá ; my tobie w nicżem przecżyć nie możemy.
Oto Rebeká przed tobą ; weźmij ją / á idź ; á niech będźie żoną ſyná páná twego / jáko rzekł Pán.
Y ſtáło śię / gdy uſłyƺał ſługá Abráhámów ſłowá ich / pokłonił śię áż do źiemi Pánu.
Zátem wyjął ſługá on nácżyniá ſrebrne / y nácżyniá złote / y ƺáty / á oddał je Rebece ; dał też upominki drogie brátu jey / y mátce jey.
Jedli tedy y pili / on y mężowie / którzy z nim byli / y zoſtáli tám ná noc ; á ráno wſtáwƺy / rzekł : Puśććie mię do páná mego.
Y rzekł brát jey / y mátká jey : Niecháj pomieƺká z námi dźiewecżká dźień / álbo dźieśięć ; potem pójdźieƺ.
A on rzekł do nich : Niezátrzymywájćie mię / gdyż Pán poƺcżęśćił drogę moję / puśććie mię / ábym jechał do páná mego.
Zátem rzekli : Záwołájmy dźiewecżki / á ſpytájmy / co ná to rzecże.
Tedy záwołáli Rebeki / y mówili do niej : Chceƺże jecháć z tym cżłowiekiem? A oná odpowiedźiáłá : Pojádę.
Y puśćili Rebekę śioſtrę ſwoję / z mámką jey / y ſługę Abráhámowego / z mężámi jego.
Tedy błogoſłáwili Rebece / mówiąc jey : Sioſtráś náƺá / rozmnóż śię w tyśiąc tyśięcy / á niech pośiądźie náśienie twoje brámy nieprzyjáćiół ſwych.
Tedy wſtáwƺy Rebeká z dźiewecżkámi ſwemi / y wśiádƺy ná wielbłądy jecháły zá onym mężem ; y wźiął ſługá on Rebekę / y odjechał.
A Izáák wrácał śię z przechádzki od ſtudni / którą zowią Żywiącego y Widzącego mię ; bo mieƺkał w źiemi południowej.
A wyƺedł był Izáák / dla modlitwy ná pole pod wiecżór / y podnióſƺy ocży ſwe / ujrzał wielbłądy przychodzące.
Podnioſłá też y Rebeká ocży ſwe / y ujrzáłá Izááká / y zśiádłá z wielbłądá ;
Bo rzekłá do ſługi : Cóż on zá mąż / który idźie przez pole przećiwko nam? Y odpowiedźiał ſługá : Ten jeſt pán mój. A oná wźiąwƺy ráńtuch nákryłá śię.
Y powiedźiał on ſługá Izáákowi wƺyſtko / co ſpráwił.
Y wprowádźił ją Izáák do namiotu Sáry / mátki ſwojey ; y wźiął Rebekę / y byłá mu żoną / y miłował ją. Y ućieƺył śię Izáák po śmierći mátki ſwojey.
