A podnióſłƺy Jákób ocży ſwe ujrzáł / á oto / Ezáw idźie / á z nim cżtery ſtá mężów ; y rozdźielił dźieći / z oſobná Lii / y z oſobná Rácheli / y z oſobná dwu ſłużebnic.
Y poſtáwił ſłużebnice / y dźieći ich / ná przodku / á Liją / y ſyny jey / zá nimi / Ráchelę záś z Józefem ná oſtátku.
A ſám ƺedł przed nimi / y pokłonił śię áż do źiemi śiedm kroć / niż przyƺedł do brátá ſwego.
Y záƺedł mu drogę Ezáw / y obłápiwƺy go / upádł ná ƺyję jego / y cáłowáł go ; y płákáli.
Potem podnióſłƺy ( Ezáw ) ocży ſwe / ujrzáł żony y dźieći / y rzekł : A ći co zácż ſą / twoiż to? y odpowiedźiáł : Dźiátki to ſą / które Bóg dáł z łáſki ſłudze twemu.
Y przybliżyły śię ſłużebnice y ſynowie ich / á pokłoniły śię.
Przybliżyłá śię też y Lijá / y dźieći jey / y pokłonili śię ; á potem przybliżył śię Józef y Ráchel / y pokłonili śię.
Y rzekł Ezáw : A ów wƺyſtek hufiec ná co / z którymem śię ſpotkáł? Odpowiedźiáł Jákób : Abym ználázł łáſkę przed ocżymá páná mego.
Tedy rzekł Ezáw : Mam ja doſyć / bráćie miły / miej ty ſwoje.
Y rzekł Jákób : Nie ták będźie proƺę ; jeſlim teraz ználázł łáſkę w ocżách twoich / weźmij upominek mój z ręki mojey / przeto / iżem widźiáł oblicże twoje / jákobym widźiáł oblicże Boże / y łáſkáwieś mię przyjął ;
Przyjmijże proƺę dár mój / którym ći przynióſł / gdyż mię hojnie błogoſłáwił Bóg / á mam wƺyſtkiego doſyć. A ták uprośił go / że to przyjął.
Tedy rzekł Ezáw : Ruƺmy śię / á idźmy / á ja pójdę przed tobą.
Y odpowiedźiáł mu Jákób : Wie pán mój / że z ſobą mam dźiátki młode / y owce kotne / y krowy ćielne / które jeſlibym przegnáł dniá jednego / pozdycháją wƺyſtkie ſtádá.
Niech w przód proƺę jedźie pán mój przed ſługą ſwoim / á ja poprowádzę śię z lekká / jáko zdąży trzodá / która jeſt przedemną / y jáko nádążą dźieći / áż przyjdę do páná mego do Seir.
Tedy rzekł Ezáw : Niech wżdy zoſtáwię przy tobie cokolwiek ludu / który jeſt ze mną. A on odpowiedźiáł : A ná cóż to? byłem ználázł łáſkę w ocżách páná mego.
Y wróćił śię dniá onego Ezáw drogą ſwą do Seir.
A Jákób obróćił śię do Suchot / y zbudowáł ſobie dom / y dla ſtád ſwoich pocżynił obory / á dla tego názwáł imię miejſcá onego Suchot.
Y przyƺedł Jákób zdrowo do miáſtá Sychem / które było w źiemi Chánánejſkiej / gdy śię wróćił z Pádán Syryjſkiego / y położył śię przed miáſtem.
Y kupił cżęść polá / ná którem rozbił námiot ſwój / od ſynów Hemorá / ojcá Sychemowego / zá ſto jágniąt ;
A poſtáwił tám ołtárz / y názwáł go : Mocny Bóg Izráelſki.
