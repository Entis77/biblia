A podnióſƺy Jákób ocży ſwe ujrzał / á oto / Ezáw idźie / á z nim cżtery ſtá mężów ; y rozdźielił dźieći / z oſobná Lii / y z oſobná Rácheli / y z oſobná dwu ſłużebnic.
Y poſtáwił ſłużebnice / y dźieći ich / ná przodku / á Liją / y ſyny jey / zá nimi / Ráchelę záś z Józefem ná oſtátku.
A ſám ƺedł przed nimi / y pokłonił śię áż do źiemi śiedm kroć / niż przyƺedł do brátá ſwego.
Y záƺedł mu drogę Ezáw / y obłápiwƺy go / upádł ná ƺyję jego / y cáłował go ; y płákáli.
Potem podnióſƺy ( Ezáw ) ocży ſwe / ujrzał żony y dźieći / y rzekł : A ći co zácż ſą / twoiż to? y odpowiedźiał : Dźiátki to ſą / które Bóg dał z łáſki ſłudze twemu.
Y przybliżyły śię ſłużebnice y ſynowie ich / á pokłoniły śię.
Przybliżyłá śię też y Lijá / y dźieći jey / y pokłonili śię ; á potem przybliżył śię Józef y Ráchel / y pokłonili śię.
Y rzekł Ezáw : A ów wƺyſtek hufiec ná co / z którymem śię ſpotkał? Odpowiedźiał Jákób : Abym ználázł łáſkę przed ocżymá páná mego.
Tedy rzekł Ezáw : Mam ja doſyć / bráćie miły / miej ty ſwoje.
Y rzekł Jákób : Nie ták będźie proƺę ; jeſlim teraz ználázł łáſkę w ocżách twojich / weźmij upominek mój z ręki mojey / przeto / iżem widźiał oblicże twoje / jákobym widźiał oblicże Boże / y łáſkáwieś mię przyjął ;
Przyjmijże proƺę dar mój / którym ći przynióſł / gdyż mię hojnie błogoſłáwił Bóg / á mam wƺyſtkiego doſyć. A ták uprośił go / że to przyjął.
Tedy rzekł Ezáw : Ruƺmy śię / á idźmy / á ja pójdę przed tobą.
Y odpowiedźiał mu Jákób : Wie pán mój / że z ſobą mam dźiátki młode / y owce kotne / y krowy ćielne / które jeſlibym przegnał dniá jednego / pozdycháją wƺyſtkie ſtádá.
Niech w przód proƺę jedźie pán mój przed ſługą ſwoim / á ja poprowádzę śię z lekká / jáko zdąży trzodá / która jeſt przedemną / y jáko nádążą dźieći / áż przydę do páná mego do Seir.
Tedy rzekł Ezáw : Niech wżdy zoſtáwię przy tobie cokolwiek ludu / który jeſt ze mną. A on odpowiedźiał : A ná cóż to? byłem ználázł łáſkę w ocżách páná mego.
Y wróćił śię dniá onego Ezáw drogą ſwą do Seir.
A Jákób obróćił śię do Suchot / y zbudował ſobie dom / y dla ſtád ſwojich pocżynił obory / á dla tego názwał imię miejſcá onego Suchot.
Y przyƺedł Jákób zdrowo do miáſtá Sychem / które było w źiemi Chánánejſkiej / gdy śię wróćił z Pádán Syryjſkiego / y położył śię przed miáſtem.
Y kupił cżęść polá / ná którem rozbił námiot ſwój / od ſynów Hemorá / ojcá Sychemowego / zá ſto jágniąt ;
A poſtáwił tám ołtárz / y názwał go : Mocny Bóg Izráelſki.
