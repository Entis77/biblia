A podnióſłƺy Jákób ocży ſwe ujrzáł / á oto / Ezáw idzie / á z nim cżtery ſtá mężów ; y rozdzielił dzieći / z oſobná Lii / y z oſobná Rácheli / y z oſobná dwu ſłużebnic.
Y poſtáwił ſłużebnice / y dzieći ich / ná przodku / á Liją / y ſyny jej / zá nimi / Ráchelę záś z Józefem ná oſtátku.
A ſám ƺedł przed nimi / y pokłonił śię áż do ziemi śiedm kroć / niż przyƺedł do brátá ſwego.
Y záƺedł mu drogę Ezáw / y obłápiwƺy go / upádł ná ƺyję jego / y cáłowáł go ; y płákáli.
Potem podnióſłƺy ( Ezáw ) ocży ſwe / ujrzáł żony y dzieći / y rzekł : A ći co zácż ſą / twoiż to? y odpowiedziáł : Dziátki to ſą / które Bóg dáł z łáſki ſłudze twemu.
Y przybliżyły śię ſłużebnice y ſynowie ich / á pokłoniły śię.
Przybliżyłá śię też y Lijá / y dzieći jej / y pokłonili śię ; á potem przybliżył śię Józef y Ráchel / y pokłonili śię.
Y rzekł Ezáw : A ów wƺyſtek hufiec ná co / z którymem śię ſpotkáł? Odpowiedziáł Jákób : Abym ználázł łáſkę przed ocżymá páná mego.
Tedy rzekł Ezáw : Mám já doſyć / bráćie miły / miej ty ſwoje.
Y rzekł Jákób : Nie ták będzie proƺę ; jeźlim teráz ználázł łáſkę w ocżách twoich / weźmij upominek mój z ręki mojej / przeto / iżem widziáł oblicże twoje / jákobym widziáł oblicże Boże / y łáſkáwieś mię przyjął ;
Przyjmijże proƺę dár mój / którym ći przynióſł / gdyż mię hojnie błogoſłáwił Bóg / á mám wƺyſtkiego doſyć. A ták uprośił go / że to przyjął.
Tedy rzekł Ezáw : Ruƺmy śię / á idźmy / á já pójdę przed tobą.
Y odpowiedziáł mu Jákób : Wie pán mój / że z ſobą mám dziátki młode / y owce kotne / y krowy ćielne / które jeźlibym przegnáł dniá jednego / pozdycháją wƺyſtkie ſtádá.
Niech w przód proƺę jedzie pán mój przed ſługą ſwoim / á já poprowádzę śię z lekká / jáko zdąży trzodá / którá jeſt przedemną / y jáko nádążą dzieći / áż przyjdę do páná mego do Seir.
Tedy rzekł Ezáw : Niech wżdy zoſtáwię przy tobie cokolwiek ludu / który jeſt ze mną. A on odpowiedziáł : A ná cóż to? byłem ználázł łáſkę w ocżách páná mego.
Y wróćił śię dniá onego Ezáw drogą ſwą do Seir.
A Jákób obróćił śię do Suchot / y zbudowáł ſobie dom / y dlá ſtád ſwoich pocżynił obory / á dlá tego názwáł imię miejſcá onego Suchot.
Y przyƺedł Jákób zdrowo do miáſtá Sychem / które było w ziemi Chánánejſkiej / gdy śię wróćił z Pádán Syryjſkiego / y położył śię przed miáſtem.
Y kupił cżęść polá / ná którem rozbił námiot ſwój / od ſynów Hemorá / ojcá Sychemowego / zá ſto jágniąt ;
A poſtáwił tám ołtárz / y názwáł go : Mocny Bóg Izráelſki.
