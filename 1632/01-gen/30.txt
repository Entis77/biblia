A widząc Ráchel / że nierodźiłá Jákóbowi / zájrzáłá Ráchel śioſtrze ſwej / rzekłá do Jákóbá : Dáj mi ſyny / á jeſli nie dáƺ / umrę.
Zápálił śię tedy gniewem Jákób ná Ráchelę / y rzekł : Záżem ja Bóg / który záwśćiągnął płód żywotá twego?
A oná rzekłá : Oto ſłużebnicá mojá Bálá ; wnijdźże do niej / y porodźi ná kolánách mojich / á będę też miáłá ſyny z niej.
Y dáłá mu Bálę / ſłużebnicę ſwoję / zá żonę ; y wƺedł Jákób do niej.
Tedy pocżęłá Bálá / y urodźiłá Jákóbowi ſyná.
Y rzekłá Ráchel : Skázał zá mną Bóg / y uſłyƺał głos mój / á dał mi ſyná ; y dla tego názwáłá imię jego Dán.
Potem záś pocżąwƺy porodźiłá Bálá / ſłużebnicá Rácheli / drugiego ſyná Jákóbowi.
Tedy rzekłá Ráchel : Mężniem śię biedźiłá z śioſtrą moją / á przemogłám ; y názwáłá imię jego Neftáli.
A obácżywƺy Lijá / że przeſtáłá rodźić / wźięłá też Zelfę / ſłużebnicę ſwoję / y dáłá ją Jákóbowi zá żonę.
Y urodźiłá Zelfá / ſłużebnicá Lii / Jákóbowi ſyná.
Zátem Lijá rzekłá : Przyƺedł huf ; y názwáłá imię jego Gád.
Porodźiłá też Zelfá / ſłużebnicá Lii / drugiego ſyná Jákóbowi.
Y rzekłá Lijá : To ná ƺcżęśćie moje ; bo mię ƺcżęśliwą będą zwáły niewiáſty ; y názwáłá imię jego Aſer.
Y wyƺedł Ruben cżáſu żniwá pƺenicżnego / y ználázł pokrzyki ná polu / á przynióſł je do Lii / mátki ſwej ; y rzekłá Ráchel do Lii : Dáj mi też proƺę z pokrzyków ſyná twego.
A oná jey odpowiedźiáłá : A máłoż ná tym / żeś mi wźięłá mężá mego / iż też chceƺ wźiąć y pokrzyki ſyná mego? Tedy rzekłá Ráchel : Niechájże śpi z tobą tey nocy zá pokrzyki ſyná twego.
A gdy śię wrácał Jákób z polá pod wiecżór / wyƺłá Lijá przećiwko jemu / y rzekłá : Do mnie wnijdźieƺ / gdyżem ćię pewną zápłátą nájęłá ſobie pokrzykámi ſyná mego ; y ſpał z nią oney nocy.
Tedy wyſłuchał Bóg Liję ; y pocżęłá / y porodźiłá Jákóbowi ſyná piątego.
Y rzekłá Lijá : Oddał mi Bóg zápłátę moję / żem byłá dáłá ſłużebnicę moję mężowi mojemu ; y názwáłá imię jego Iſáƺár.
Potem pocżąwƺy jeƺcże Lijá / porodźiłá ƺóſtego ſyná Jákóbowi.
Y mówiłá Lijá : Obdárzył mię Bóg zácnym upominkiem ; już teraz będźie ze mną mieƺkał mąż mój / bom mu urodźiłá ƺeśćiu ſynów ; y názwáłá imię jego Zábulon.
Potem porodźiłá córkę / y názwáłá imię jey Dyná.
Wſpomniał też Bóg / ná Ráchelę / y wyſłuchał ją Bóg / á otworzył żywot jey.
Tedy pocżąwƺy porodźiłá ſyná / y rzekłá : Odjął Bóg zelżywość moję.
Y názwáłá imię jego Józef / mówiąc : Niech mi przydá Pán drugiego ſyná.
Y ſtáło śię / gdy porodźiłá Ráchel Józefá / mówił Jákób do Lábáná : Puść mię / ábym śię wróćił do miejſcá mego / y do źiemi mojey.
Dáj mi żony moje / y dźieći moje / zá którem ći ſłużył / że odejdę ; bo ty wieƺ poſługi moje / jákom ći ſłużył.
Y rzekł do niego Lábán : Proƺę / jeſlim ználázł łáſkę w ocżách twojich / zoſtáń ze mną ; bom doznał tego / że mi Pán dla ćiebie błogoſłáwił.
Y rzekł : Miánuj mi zápłátę twoję / á dam ći ją.
Tedy mu odpowiedźiał Jákób : Ty wieƺ / jákom ći ſłużył / y jáki był dobytek twój przy mnie.
Bo tá trochá / którąś miał przedemną / rozmnożyłá śię wielce ; y błogoſłáwił ći Pán ná przyśćie moję / á teraz kiedyż ja śię też ſtáráć będę o dom ſwój?
Y rzekł : Cóżći mam dáć? Y odpowiedźiał Jákób : Nie dáƺ mi nic ; ále jeſli to ucżyniƺ coć powiem / tedy śię wrócę / á będę páſł y ſtrzegł bydłá twego.
Przejdę dźiś przez wƺyſtkie trzody twoje / odłącżájąc z támtąd káżde bydlę pſtre y nákrápiáne / y káżde bydlę płowe miedzy owcámi / á nákrápiáne y pſtre miedzy kozámi ; to będźie zápłátą moją.
Y da świádectwo o mnie ſpráwiedliwość mojá ná potem / gdy przydźie do zápłáty mojey przed tobą ; wƺyſtko co nie będźie pſtre y nákrápiáne miedzy kozámi / á płowe miedzy owcámi / niech będźie zá krádźież pocżytáne przy mnie.
Tedy rzekł Lábán : Oby śię ſtáło według ſłowá twego!
Y odłącżył onegoż dniá kozły ſtrokáte / y nákrápiáne / y wƺyſtkie kozy pſtre / y nákrápiáne / y wƺyſtkie / co jáką biáłą odmiánę miáły / tákże y płowe miedzy owcámi / y oddał je do rąk ſynów ſwych.
Y odłącżył śię Lábán od Jákóbá / jákoby ná trzy dni drogi ; á Jákób páſł oſtátek owiec Lábánowych.
Nábrał tedy Jákób prętów źielonych topolowych / y láſkowych / y káƺtánowych / y obłupił miejſcámi ſkórę ich do biáłego / obnáżájąc biáłość / która ná prętách byłá.
Y nákłádł onych prętów / które był obłupił / do rynien y do koryt / gdźie láno wody ( gdy przychodźiły owce / áby piły ) nákłádł ich przećiwko owcom / áby pocżynáły / gdyby pić przychodźiły.
Y pocżynáły owce pátrząc ná one pręty / y rodźiły jágniętá ſtrokáte / pſtre y nákrápiáne.
Y odłącżył Jákób jágniętá / á ſtáwiał owce twárzą do jágniąt ſtrokátych / y do wƺyſtkich płowych w ſtádźie Lábánowem / á ſwoje ſtádá ſtáwiał oſobno / áni ich obrácał ku ſtádu Lábánowemu.
A gdy wƺyſtkich owiec co ráńƺych przypuƺcżánie bywáło / kłádł Jákób pręty przed ocży owiec w korytá / áby pocżynáły pátrząc ná pręty.
Lecż gdy późniejƺych owiec przypuƺcżánie było / nie kłádł ich ; y były późniejƺe Lábánowe / á ráńƺe Jákóbowe.
Y ták zbogáćił śię on cżłowiek bárzo / y miał owiec wiele / y ſłużebnic y ſług / y wielbłądów / y oſłów.
