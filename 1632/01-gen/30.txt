A widząc Ráchel / że nierodziłá Jákóbowi / zájrzáłá Ráchel śioſtrze ſwej / rzekłá do Jákóbá : Dáj mi ſyny / á jeźli nie dáƺ / umrę.
Zápálił śię tedy gniewem Jákób ná Ráchelę / y rzekł : Záżem já Bóg / który záwśćiągnął płód żywotá twego?
A oná rzekłá : Oto ſłużebnicá mojá Bálá ; wnijdźże do niej / y porodzi ná kolánách moich / á będę też miáłá ſyny z niej.
Y dáłá mu Bálę / ſłużebnicę ſwoję / zá żonę ; y wƺedł Jákób do niej.
Tedy pocżęłá Bálá / y urodziłá Jákóbowi ſyná.
Y rzekłá Ráchel : Skázáł zá mną Bóg / y uſłyƺáł głoſ mój / á dáł mi ſyná ; y dlá tego názwáłá imię jego Dán.
Potem záś pocżąwƺy porodziłá Bálá / ſłużebnicá Rácheli / drugiego ſyná Jákóbowi.
Tedy rzekłá Ráchel : Mężniem śię biedziłá z śioſtrą moją / á przemogłám ; y názwáłá imię jego Neftáli.
A obácżywƺy Lijá / że przeſtáłá rodzić / wzięłá też Zelfę / ſłużebnicę ſwoję / y dáłá ją Jákóbowi zá żonę.
Y urodziłá Zelfá / ſłużebnicá Lii / Jákóbowi ſyná.
Zátem Lijá rzekłá : Przyƺedł huf ; y názwáłá imię jego Gád.
Porodziłá też Zelfá / ſłużebnicá Lii / drugiego ſyná Jákóbowi.
Y rzekłá Lijá : To ná ƺcżęśćie moje ; bo mię ƺcżęśliwą będą zwáły niewiáſty ; y názwáłá imię jego Aſer.
Y wyƺedł Ruben cżáſu żniwá pƺenicżnego / y ználázł pokrzyki ná polu / á przynióſł je do Lii / mátki ſwej ; y rzekłá Ráchel do Lii : Dáj mi też proƺę z pokrzyków ſyná twego.
A oná jej odpowiedziáłá : A máłoż ná tem / żeś mi wzięłá mężá mego / iż też chceƺ wziąć y pokrzyki ſyná mego? Tedy rzekłá Ráchel : Niechájże śpi z tobą tej nocy zá pokrzyki ſyná twego.
A gdy śię wrácáł Jákób z polá pod wiecżór / wyƺłá Lijá przećiwko jemu / y rzekłá : Do mnie wnijdzieƺ / gdyżem ćię pewną zápłátą nájęłá ſobie pokrzykámi ſyná mego ; y ſpáł z nią onej nocy.
Tedy wyſłucháł Bóg Liję ; y pocżęłá / y porodziłá Jákóbowi ſyná piątego.
Y rzekłá Lijá : Oddáł mi Bóg zápłátę moję / żem byłá dáłá ſłużebnicę moję mężowi mojemu ; y názwáłá imię jego Iſáƺár.
Potem pocżąwƺy jeƺcże Lijá / porodziłá ƺóſtego ſyná Jákóbowi.
Y mówiłá Lijá : Obdárzył mię Bóg zácnym upominkiem ; już teráz będzie ze mną mieƺkáł mąż mój / bom mu urodziłá ƺeśćiu ſynów ; y názwáłá imię jego Zábulon.
Potem porodziłá córkę / y názwáłá imię jej Dyná.
Wſpomniáł też Bóg / ná Ráchelę / y wyſłucháł ją Bóg / á otworzył żywot jej.
Tedy pocżąwƺy porodziłá ſyná / y rzekłá : Odjął Bóg zelżywość moję.
Y názwáłá imię jego Józef / mówiąc : Niech mi przydá Pán drugiego ſyná.
Y ſtáło śię / gdy porodziłá Ráchel Józefá / mówił Jákób do Lábáná : Puść mię / ábym śię wróćił do miejſcá mego / y do ziemi mojej.
Dáj mi żony moje / y dzieći moje / zá którem ći ſłużył / że odejdę ; bo ty wieƺ poſługi moje / jákom ći ſłużył.
Y rzekł do niego Lábán : Proƺę / jeźlim ználázł łáſkę w ocżách twoich / zoſtáń ze mną ; bom doznáł tego / że mi Pán dlá ćiebie błogoſłáwił.
Y rzekł : Miánuj mi zápłátę twoję / á dám ći ją.
Tedy mu odpowiedziáł Jákób : Ty wieƺ / jákom ći ſłużył / y jáki był dobytek twój przy mnie.
Bo tá trochá / którąś miáł przedemną / rozmnożyłá śię wielce ; y błogoſłáwił ći Pán ná przyjśćie moję / á teráz kiedyż já śię też ſtáráć będę o dom ſwój?
Y rzekł : Cóżći mám dáć? Y odpowiedziáł Jákób : Nie dáƺ mi nic ; ále jeźli to ucżyniƺ coć powiem / tedy śię wrócę / á będę páſł y ſtrzegł bydłá twego.
Przejdę dziś przez wƺyſtkie trzody twoje / odłącżájąc ſtámtąd káżde bydlę pſtre y nákrápiáne / y káżde bydlę płowe między owcámi / á nákrápiáne y pſtre między kozámi ; to będzie zápłátą moją.
Y dá świádectwo o mnie ſpráwiedliwość mojá ná potem / gdy przyjdzie do zápłáty mojej przed tobą ; wƺyſtko co nie będzie pſtre y nákrápiáne między kozámi / á płowe między owcámi / niech będzie zá krádzież pocżytáne przy mnie.
Tedy rzekł Lábán : Oby śię ſtáło według ſłowá twego!
Y odłącżył onegoż dniá kozły ſtrokáte / y nákrápiáne / y wƺyſtkie kozy pſtre / y nákrápiáne / y wƺyſtkie / co jáką biáłą odmiánę miáły / tákże y płowe między owcámi / y oddáł je do rąk ſynów ſwych.
Y odłącżył śię Lábán od Jákóbá / jákoby ná trzy dni drogi ; á Jákób páſł oſtátek owiec Lábánowych.
Nábráł tedy Jákób prętów zielonych topolowych / y láſkowych / y káƺtánowych / y obłupił miejſcámi ſkórę ich do biáłego / obnáżájąc biáłość / którá ná prętách byłá.
Y nákłádł onych prętów / które był obłupił / do rynien y do koryt / gdzie láno wody ( gdy przychodziły owce / áby piły ) nákłádł ich przećiwko owcom / áby pocżynáły / gdyby pić przychodziły.
Y pocżynáły owce pátrząc ná one pręty / y rodziły jágniętá ſtrokáte / pſtre y nákrápiáne.
Y odłącżył Jákób jágniętá / á ſtáwiáł owce twárzą do jágniąt ſtrokátych / y do wƺyſtkich płowych w ſtádzie Lábánowem / á ſwoje ſtádá ſtáwiáł oſobno / áni ich obrácáł ku ſtádu Lábánowemu.
A gdy wƺyſtkich owiec co ráńƺych przypuƺcżánie bywáło / kłádł Jákób pręty przed ocży owiec w korytá / áby pocżynáły pátrząc ná pręty.
Lecż gdy późniejƺych owiec przypuƺcżánie było / nie kłádł ich ; y były późniejƺe Lábánowe / á ráńƺe Jákóbowe.
Y ták zbogáćił śię on cżłowiek bárdzo / y miáł owiec wiele / y ſłużebnic y ſług / y wielbłądów / y oſłów.
