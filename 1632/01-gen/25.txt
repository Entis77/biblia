Potem Abráhám pojął drugą żonę / której imię było Keturá.
Którá mu urodźiłá Zámrámá / y Jokſáná / y Mádáná / y Midyjáná / y Jeſboká / y Suáchá.
A Jokſán ſpłodźił Sábę / y Dedáná ; á ſynowie Dedánowi byli Aſurymowie y Letuſymowie / y Leumymowie.
Synowie záś Midyjánowi byli Hefá / y Hefer / y Henoch / y Abydá / y Eldáá ; wƺyſcy ći byli ſynowie Ketury.
Y dáł Abráhám wƺyſtko / co miáł / Izáákowi.
A ſynom záłożnic / które miáł Abráhám / dáł upominki ; y wypráwił je od Izááká ſyná ſwego / jeƺcże zá żywotá ſwego / ku wſchodowi do kráiny wſchodniej.
Teć ſą dni lát żywotá Abráhámowego / które przeżył / ſto y śiedmdźieśiąt / y pięć lát.
Y uſtáwájąc umárł Abráhám w ſtárośći dobrej / zeƺły w lećiech / y ſyty dni ; y przyłącżon jeſt do ludu ſwego.
Y pogrzebli go Izáák y Iſmáel / ſynowie jego / w jáſkini Máchpelá / ná polu Efroná / ſyná Sochárowego / Hetejcżyká / które było przećiwko Mámre ;
Ná polu / które był kupił Abráhám u ſynów Hetowych ; tám pogrzebiony jeſt Abráhám / y Sárá / żoná jego.
A po śmierći Abráhámowej błogoſłáwił Bóg Izáákowi / ſynowi jego / á Izáák mieƺkáł u ſtudni Żywiącego y Widzącego mię.
A teć ſą rodzáje Iſmáelá / ſyná Abráhámowego / którego urodźiłá Hágár / Egipcżánká / ſłużebnicá Sáry / Abráhámowi.
Y te ſą imioná ſynów Iſmáelowych w názwiſkách ich / według rodzájów ich : pierworodny Iſmáelów / Nebájot ; po nim Kedár / y Abdeel / y Mábſán.
Y Máſmá / y Dumá / y Máſá.
Hádár / y Temá / Jetur / Náfiſ y Kedmá.
Ci ſą ſynowie Iſmáelowi / y te imioná ich / według miáſtecżek ich / y zámków ich ; dwánáśćie kśiążąt w fámilijách ich.
A było lát żywotá Iſmáelowego / ſto lát / y trzydźieśći lát y śiedm lát / y zƺedł á umárł / y przyłącżon jeſt do ludu ſwego.
Y mieƺkáli od Hewilá áż do Sur / którá leży ná przećiwko Egiptowi / idąc do Aſyryi ; przed oblicżem wƺyſtkich bráći ſwych umárł.
Te záś ſą rodzáje Izááká ſyná Abráhámowego : Abráhám ſpłodźił Izááká.
A Izáák miáł cżterdźieśći lát / gdy ſobie pojął Rebekę / córkę Bátuelá Syryjcżyká / z kráiny Syryjſkiej / śioſtrę Lábáná / Syryjcżyká / zá żonę.
Tedy śię modlił Izáák Pánu zá żonę ſwą / iż byłá niepłodná ; y wyſłucháł go Pán / y pocżęłá Rebeká / żoná jego.
A gdy śię dźiátki trącáły w żywoćie jej / rzekłá : Jeźliż ták miáło być / dlá cżegożem pocżęłá? ƺłá tedy / áby śię pytáłá Páná.
Y rzekł jej Pán : Dwá národy ſą w żywoćie twoim / y dwojáki lud z żywotá twego rozdźieli śię / á jeden lud nád drugi lud możniejƺy będźie / y więkƺy będźie ſłużył mniejƺemu.
A gdy śię wypełniły dni jej / áby porodźiłá / oto bliźniętá były w żywoćie jej.
Y wyƺedł pierwƺy ſyn liſowáty / y wƺyſtek jáko ƺátá koſmáty ; y názwáli imię jego Ezáw.
A potem wyƺedł brát jego / ręką ſwą trzymájąc zá piętę Ezáwá / y názwáno imię jego Jákób ; á Izáákowi było ƺeśćdźieśiąt lát / gdy mu śię oni národźili.
A gdy uroſły one dźieći / Ezáw był mężem w myśliſtwie biegłym y rolnikiem / á Jákób był mąż proſty mieƺkájący w námiećiech.
Y miłowáł Izáák Ezáwá / iż jádáł z łowu jego ; Rebeká záś miłowáłá Jákóbá.
Y uwárzył ſobie Jákób potráwę / á ná ten cżáſ przyƺedł Ezáw z polá ſprácowány.
Tedy rzekł Ezáw do Jákóbá : Dáj mi jeść / proƺę ćię / z tej cżerwonej potráwy / bom śię ſprácowáł : á przetoż názwáno imię jego Edom.
Któremu rzekł Jákób : Przedájże mi dźiś pierworodztwo twoje.
Y rzekł Ezáw : Otom já bliſki śmierći / cóż mi po pierworodztwie?.
Y rzekł Jákób : Przyśiążże mi dźiś / y przyśiągł mu. Y ſprzedáł pierworodztwo ſwoje Jákóbowi.
Tedy Jákób dáł Ezáwowi chlebá / y potráwę z ſocżewicy / á on jádł y pił / á potem powſtáwƺy odƺedł ; y pogárdźił Ezáw pierworodztwem ſwojim.
