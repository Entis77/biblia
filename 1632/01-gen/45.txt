Tedy śię Józef nie mógł dálej wſtrzymáć przed wƺyſtkimi / którzy ſtáli przed nim / y záwołáł : Wyprowádźćie wƺyſtkie odemnie. Y nie zoſtáł nikt przy nim / gdy śię dáł poznáć Józef bráći ſwej.
Y podnióſł głoſ ſwój z płácżem ; co ſłyƺeli Egipcżánie / ſłyƺáł też dom Fáráonów.
Y rzekł Józef do bráći ſwej : Jámći jeſt Józef ; á żywże jeƺcże ojćiec mój? y nie mogli mu bráćiá jego odpowiedzieć / bo śię zlękli oblicżá jego.
Tedy rzekł Józef do bráći ſwej : Przyſtąpćie / proƺę / do mnie ; y przyſtąpili. Zátem rzekł : Jám jeſt Józef / brát wáƺ / któregośćie ſprzedáli do Egiptu.
Jednák teráz nie fráſujćie śię / áni trwożćie ſobą / żeśćie mię tu ſprzedáli ; boć dlá záchowániá żywotá wáƺego poſłáł mię Bóg przed wámi.
Bo już dwá látá głodu było ná ziemi / á jeƺcże pięć lát zoſtáje / których nie będą oráć áni żąć.
Poſłáł mię tedy Bóg przed wámi / ábym wáſ záchowáł oſtátek ná ziemi / á żebym wám dodáł żywnośći ná oſwobodzenie wielkie.
Teráz tedy nie wyśćie mię tu poſłáli / ále Bóg / który mię poſtánowił ojcem Fáráonowym / y pánem wƺyſtkiego domu jego / á pánującym nád wƺyſtką ziemią Egipſką.
Spieƺćież śię / á idźćie do ojcá mego / y mówćie do niego : Toć wſkázuje ſyn twój Józef : Ucżynił mię Bóg pánem wƺyſtkiego Egiptu / przyjedźże do mnie / á nie omieƺkáj.
Y będzieƺ mieƺkáł w ziemi Goſen ; á będzieƺ bliſko mnie / ty y ſynowie twoi / y ſynowie ſynów twoich / y trzody twoje / y woły twoje / y wƺyſtko / co máƺ.
A będę ćię tám żywił ; bo jeƺcże pięć lát głodu będzie / ábyś od niedoſtátku nie zginął / ty / y dom twój / y wƺyſtko / co máƺ.
A oto / ocży wáƺe widzą / y ocży brátá mego / Benjáminá / że uſtá moje mówią do wáſ.
Oznájmijćie też ojcu memu wƺyſtkę zácność moje w Egipćie / y wƺyſtko / cośćie widzieli ; ſpieƺćie śię tedy / á przyprowádźćie tu ojcá mojego.
Zátem pádł ná ƺyję Benjáminá / brátá ſwego / y płákáł ; Benjámin też płákáł ná ƺyi jego.
Y pocáłowáwƺy wƺyſtkę bráćią ſwoję / płákáł nád nimi ; á potem rozmáwiáli z nim bráćiá jego.
Y rozgłoƺono tę wieść w domu Fáráonowym / mówiąc : Przyjecháli bráćiá Józefowi ; y podobáło śię to w ocżách Fáráonowych / y w ocżách ſług jego.
Tedy rzekł Fáráo do Józefá : Powiedz bráći ſwej : Ucżyńćie ták : nákłádłƺy brzemion ná bydłá wáƺe / idźćie ; á wróććie śię do ziemi Chánánejſkiej ;
A wziąwƺy ojcá wáƺego / y cżeládź wáƺę / przyjedźćie do mnie ; y dám wám dobre miejſce w ziemi Egipſkiej / y będziećie używáć tłuſtośći ziemi.
Y rozkáż im mówiąc : To ucżyńćie : weźmijćie ſobie z ziemi Egipſkiej wozów / dlá dziátek wáƺych y dlá żon wáƺych / á wziąwƺy ojcá wáƺego przyjedźćie tu.
A oko wáƺe niech nie żáłuje ſprzętu wáƺego / gdyż dobro wƺyſtkiej ziemi Egipſkiej wáƺe będzie.
Ucżynili tedy ták ſynowie Izráelowi ; y dáł im Józef wozy według rozkázániá Fáráonowego ; dáł im też żywnośći ná drogę.
Dáł z onychże wƺyſtkich káżdemu odmienne ƺáty ; ále Benjáminowi dáł trzy ſtá ſrebrników / y pięćioro ƺát odmiennych.
Ojcu też ſwemu poſłáł te rzecży : dzieśięć oſłów / nioſących z nájlepƺych rzecży Egipſkich / y dzieśięć oślic / nioſących zboże / y chleb / y żywność ojcu jego ná drogę.
Puśćił tedy bráćią ſwą / y odjecháli / á mówił do nich : Nie wádźćie śię ná drodze.
Którzy wyjecháwƺy z Egiptu przyjecháli do ziemi Chánánejſkiej / do Jákóbá ojcá ſwego.
Y oznájmili mu / mówiąc : Jeƺcżeć żyw Józef / á onći jeſt pánem nád wƺyſtką ziemią Egipſką ; y zemdláło ſerce jego ; bo im nie wierzył.
Lecż oni powiedzieli mu wƺyſtkie ſłowá Józefowe / które mówił do nich. A ujrzáwƺy wozy / które poſłáł Józef / áby go ná nich przywieziono / tedy ożył duch Jákóbá / ojcá ich.
Y rzekł Izráel : Doſyć mám ná tem / gdy jeƺcże Józef / ſyn mój / żyje ; pójdę á oglądám go / pierwej niż umrę.
