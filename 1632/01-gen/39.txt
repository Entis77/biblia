Tedy Józef był záwiedzion do Egiptu ; y kupił go Potyfár / dworzánin Fáráonów / hetmán żołnierſtwá / mąż Egipcżánin / z ręki Iſmáelitów / którzy go tám byli záwiedli.
Y był Pán z Józefem / który był mężem ƺcżęśliwie poſtępującym / á mieƺkáł w domu páná ſwego Egipcżániná.
Y bácżył pán jego / że Pán był z nim / á iż wƺyſtko co on cżynił / Pán ƺcżęśćił w ręku jego.
Y ználázł Józef łáſkę w ocżách jego / y ſłużył mu ; y przełożył go nád domem ſwym / á podáł wƺyſtko co miáł / w ręce jego.
Y ſtáło śię / gdy go przełożył nád domem ſwym / y nád wƺyſtkiem / co miáł / błogoſłáwił Pán domowi Egipcżániná dlá Józefá ; y było błogoſłáwieńſtwo Páńſkie nád wƺyſtkiem / cokolwiek miáł w domu y ná polu.
Przetoż porucżył wƺyſtko / co miáł / w ręce Józefowe / y ni o cżem u śiebie nie wiedziáł / tylko o chlebie / którego pożywáł. A był Józef pięknej twárzy / y wdzięcżny ná wejrzeniu.
Y ſtáło śię potem / iż obróćiłá żoná páná jego ocży ſwoje ná Józefá / y rzekłá : Śpij ze mną.
Ale nie chćiáł : y rzekł do żony páná ſwego : Oto / pán mój nie wie / ták jáko já / co jeſt w domu jego ; bo wƺyſtko / co miáł / podáł w ręce moje.
Y niemáƺ nikogo w domu tym nád mię przedniejƺego / y nie wyjął mi nic z mocy / prócż ćiebie / przeto żeś ty jeſt żoná jego ; jákożbym tedy miáł ucżynić tę wielką złość / y grzeƺyć przećiwko Bogu?
Y ſtáło śię / gdy oná námáwiáłá Józefá ná káżdy dzień / á on jej nie zezwáláł / áby ſpáł z nią / álbo bywáł z nią :
Tedy dniá niektórego / gdy wƺedł do domu / dlá odpráwowániá prácy ſwej / á nie było tám z domowników nikogo w domu ;
Uchwyćiłá go zá ƺátę jego / mówiąc : Śpij ze mną. Ale on zoſtáwiwƺy ƺátę ſwoję w ręku jej / ućiekł / y wyƺedł precż.
A gdy oná obácżyłá / iż zoſtáwił ƺátę ſwoję w ręku jej / á ućiekł precż :
Tedy záwołáłá ná cżeládź domu ſwego / y rzekłá do nich / mówiąc : Wejćie / wprowádził pán do náſ mężá Hebrejcżyká / áby náſ zelżył ; ábowiem wƺedł do mnie / áby ze mną ſpáł / áżem wołáłá głoſem wielkim.
A gdy uſłyƺáł / żem wynioſłá głoſ mój / y záwołáłá / zoſtáwiwƺy ƺátę ſwoję u mnie / ućiekł / y wyƺedł precż.
Y zátrzymáłá jego ƺátę u śiebie / áż przyƺedł pán jego do domu ſwego ;
Y rzekłá do niego w te ſłowá / mówiąc : Wƺedł do mnie ſługá ten Hebrejcżyk / któregoś przywiódł do náſ / áby mię zelżył.
Ale gdym podnioſłá głoſ ſwój / y záwołáłá / tedy zoſtáwił ƺátę ſwą u mnie / y ućiekł precż.
Y ſtáło śię / gdy uſłyƺáł pán jego ſłowá żony ſwojej / które rzekłá do niego / mówiąc : Ták mi ucżynił ſługá twój / rozgniewáł śię bárdzo.
Y wziął pán Józefá / á dáł go do domu więzieniá / tám / gdzie więźnie królewſkie ſádzáno / y był tám w domu więzieniá.
A Pán był z Józefem / y ſkłoniwƺy ku niemu miłośierdzie / dáł mu łáſkę w ocżách przełożonego nád domem więzieniá.
Tedy przełożony nád domem więzieniá / podáł w moc Józefowi wƺyſtkie więźnie / którzy byli w domu więzieniá ; á wƺyſtko / co tám cżynić mieli / to on ſpráwowáł.
A przełożony nád domem więzieniá nie doglądáł tego / cżego mu śię powierzył / dlá tego iż Pán był z nim / á co on cżynił / to Pán ƺcżęśćił.
