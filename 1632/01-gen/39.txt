Tedy Józef był záwiedźion do Egiptu ; y kupił go Potyfár / dworzánin Fáráonów / hetmán żołnierſtwá / mąż Egipcżánin / z ręki Iſmáelitów / którzy go tám byli záwiedli.
Y był Pán z Józefem / który był mężem ƺcżęśliwie poſtępującym / á mieƺkał w domu páná ſwego Egipcżániná.
Y bácżył pán jego / że Pán był z nim / á iż wƺyſtko co on cżynił / Pán ƺcżęśćił w ręku jego.
Y ználázł Józef łáſkę w ocżách jego / y ſłużył mu ; y przełożył go nád domem ſwym / á podał wƺyſtko co miał / w ręce jego.
Y ſtáło śię / gdy go przełożył nád domem ſwym / y nád wƺyſtkiem / co miał / błogoſłáwił Pán domowi Egipcżániná dla Józefá ; y było błogoſłáwieńſtwo Páńſkie nád wƺyſtkiem / cokolwiek miał w domu y ná polu.
Przetoż porucżył wƺyſtko / co miał / w ręce Józefowe / y ni o cżym u śiebie nie wiedźiał / tylko o chlebie / którego pożywał. A był Józef piękney twárzy / y wdźięcżny ná wejrzeniu.
Y ſtáło śię potem / iż obróćiłá żoná páná jego ocży ſwoje ná Józefá / y rzekłá : Śpij ze mną.
Ale nie chćiał : y rzekł do żony páná ſwego : Oto / pán mój nie wie / ták jáko ja / co jeſt w domu jego ; bo wƺyſtko / co miał / podał w ręce moje.
Y niemáƺ nikogo w domu tym nád mię przedniejƺego / y nie wyjął mi nic z mocy / prócż ćiebie / przeto żeś ty jeſt żoná jego ; jákożbym tedy miał ucżynić tę wielką złość / y grzeƺyć przećiwko Bogu?
Y ſtáło śię / gdy oná námáwiáłá Józefá ná káżdy dźień / á on jey nie zezwálał / áby ſpał z nią / álbo bywał z nią :
Tedy dniá niektórego / gdy wƺedł do domu / dla odpráwowánia prácy ſwej / á nie było tám z domowników nikogo w domu ;
Uchwyćiłá go zá ƺátę jego / mówiąc : Śpij ze mną. Ale on zoſtáwiwƺy ƺátę ſwoję w ręku jey / ućiekł / y wyƺedł precż.
A gdy oná obácżyłá / iż zoſtáwił ƺátę ſwoję w ręku jey / á ućiekł precż :
Tedy záwołáłá ná cżeládź domu ſwego / y rzekłá do nich / mówiąc : Wejćie / wprowádźił pán do nas mężá Hebrejcżyká / áby nas zelżył ; ábowiem wƺedł do mnie / áby ze mną ſpał / áżem wołáłá głoſem wielkim.
A gdy uſłyƺał / żem wynioſłá głos mój / y záwołáłá / zoſtáwiwƺy ƺátę ſwoję u mnie / ućiekł / y wyƺedł precż.
Y zátrzymáłá jego ƺátę u śiebie / áż przyƺedł pán jego do domu ſwego ;
Y rzekłá do niego w te ſłowá / mówiąc : Wƺedł do mnie ſługá ten Hebrejcżyk / któregoś przywiódł do nas / áby mię zelżył.
Ale gdym podnioſłá głos ſwój / y záwołáłá / tedy zoſtáwił ƺátę ſwą u mnie / y ućiekł precż.
Y ſtáło śię / gdy uſłyƺał pán jego ſłowá żony ſwojey / które rzekłá do niego / mówiąc : Ták mi ucżynił ſługá twój / rozgniewał śię bárzo.
Y wźiął pán Józefá / á dał go do domu więźieniá / tám / gdźie więźnie królewſkie ſádzáno / y był tám w domu więźieniá.
A Pán był z Józefem / y ſkłoniwƺy ku niemu miłośierdźie / dał mu łáſkę w ocżách przełożonego nád domem więźieniá.
Tedy przełożony nád domem więźieniá / podał w moc Józefowi wƺyſtkie więźnie / którzy byli w domu więźieniá ; á wƺyſtko / co tám cżynić mieli / to on ſpráwował.
A przełożony nád domem więźieniá nie doglądał tego / cżego mu śię powierzył / dla tego iż Pán był z nim / á co on cżynił / to Pán ƺcżęśćił.
