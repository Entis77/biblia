Y przyƺli dwáj Aniołowie do Sodomy w wiecżór / á Lot śiedziáł w brámie Sodomſkiej. Gdy je tedy ujrzáł Lot / wſtáwƺy ƺedł przećiwko nim / y ſkłonił śię twárzą ku ziemi / y rzekł :
Oto proƺę pánowie moi / wſtąpćie teráz do domu ſługi ſwego / á bądźćie tu ná noc / y umyjćie nogi ſwe ; potem ráno wſtáwƺy pójdziećie w drogę wáƺę. Którzy odpowiedzieli : Bynájmniej ; ále ná ulicy będziemy nocowáli.
Ale on przymuƺáł ich bárdzo / iż śię ſkłonili do niego / y weƺli w dom jego ; zácżem ſpráwił im ucżtę / y nápiekł chlebá przáśnego / y jedli.
Lecż pierwej niż oni poƺli ſpáć / oto obywátele miáſtá / mężowie Sodomſcy / obſtąpili dom / od młodego áż do ſtárego / wƺyſtek lud zewƺąd.
Y wołáli ná Lotá / y rzekli mu : Gdzie ſą mężowie / którzy przyƺli do ćiebie w nocy? wywiedź je do náſ / ábyśmy je poználi.
Tedy wyƺedł do nich Lot ze drzwi / y zámknął drzwi zá ſobą.
Y rzekł : Nie cżyńćie proƺę / bráćiá moi / nic złego.
Oto teráz mám dwie córki / które nie poznáły mężá ; wywiodę je teráz do wáſ / á cżyńćie z niemi / co śię wám podobá / tylko mężom tym nic nie cżyńćie ; bo dlátego weƺli pod ćień dáchu mego.
A oni rzekli : Pójdźże tám ; y mówili : Ten ſám przyƺedł / áby tu gośćiem był / á miáłby náſ ſądzić? przetoż gorzej ucżynimy tobie / niż onym ; y cżynili gwáłt wielki mężowi onemu Lotowi / y przyſtąpili / áby drzwi wyłámáli.
Ale mężowie oni / wyćiągnąwƺy rękę ſwoję / wwiedli Lotá do śiebie w dom / y zámknęli drzwi.
A męże one / którzy byli u drzwi domu / pozáráżáli ślepotą / od nájmniejƺego / áż do nájwiękƺego ; ták / iż śię ſprácowáli / ƺukájąc drzwi.
Tedy rzekli mężowie oni do Lotá : Máƺli tu jeƺcże kogo / zięćiá / álbo ſyny twe / álbo córki twoje / y wƺyſtko / co máƺ w mieśćie / wyprowádź z miejſcá tego.
Skázimy bowiem to miejſce / przeto / że śię wzmógł krzyk ich przed Pánem / y poſłáł náſ Pán / ábyśmy je ſkázili.
Tedy wyƺedłƺy Lot mówił do zięćiów ſwoich / którzy mieli pojąć córki jego / y rzekł : Wſtáńćie / wynijdźćie z miejſcá tego / bo ſkázi Pán to miáſto ; ále śię zdáło w ocżách zięćiów jego / jákoby żártowáł.
A gdy weƺłá zorzá / przymuƺáli Aniołowie Lotá / mówiąc : Wſtáń / weźmij żonę twoję / y dwie córki twoje / które tu ſą / byś ſnáć nie zginął w niepráwośći miáſtá tego.
A gdy śię oćiągáł / ujęli mężowie oni rękę jego / y rękę żony jego / y rękę dwóch córek jego / ( ábowiem mu Pán folgowáł / )i wywiedli go / y poſtáwili go przed miáſtem.
Y gdy je wywiedli precż / rzekł jeden : Jeźli chceƺ / záchowáj duƺę twoję / á nie oglądáj śię názád / áni ſtáwáj ná tej wƺyſtkiej równinie ; uchodź ná górę / byś ſnáć nie zginął.
A Lot rzekł do nich : Nie ták / proƺę / pánowie moi ;
Oto teráz ználázł ſługá twój łáſkę w ocżách twoich / y okázáłeś obfićie miłośierdzie twoje / któreś ucżynił ze mną / záchowáwƺy duƺę moję ; áleć já nie będę mógł ujść ná tę górę / by mię ſnáć nie záchwyćiło to złe / y umárłbym.
Ale oto tu jeſt miáſto nie dáleko / do któregobym ućiekł / máłeć jeſt ; proƺę / niech tám ujdę / ( wƺák máłe jeſt / )á będzie żywá duƺá mojá.
Tedy rzekł do niego : Oto / y w tem wyſłucháłem ćię / ábym nie wywróćił miáſtá tego / o któremeś mówił.
Spieƺże śię á uchodź tám / bo nie będę mógł nic ucżynić / áż ty tám dojdzieƺ ; przetoż názwáne jeſt imię miáſtá onego Zoár.
Wtem ſłońce weƺło ná ziemię / á Lot wƺedł do Zoár.
Tedy Pán ſpuśćił jáko deƺcż ná Sodomę y ná Gomorrę śiárkę y ogień / od Páná z niebá.
Y wywróćił miáſtá one / y wƺyſtkę onę równinę / y wƺyſtkie obywátele miáſt onych / y urodzáje onej ziemi.
Y obejrzáłá śię żoná jego idąc zá nim / á obróćiłá śię w ſłup ſolny.
Wſtáwƺy tedy Abráhám ráno / poſpieƺył śię ná ono miejſce / kędy ſtáł przed Pánem.
Y ſpojrzáł ku Sodomie y Gomorze / y ku wƺyſtkiej ziemi onej równiny / y obácżył / á oto wychodził dym z onej ziemi / jáko dym z piecá.
A gdy wywrácáł Bóg miáſtá onej równiny / wſpomniáł Bóg ná Abráhámá y wybáwił Lotá z pośrodku wywróceniá / gdy wywrácáł one miáſtá / w których Lot mieƺkáł.
Potem wyƺedł Lot z Zoár / y mieƺkáł ná górze / y dwie córki jego z nim / ábowiem śię báł mieƺkáć w Zoár ; ále mieƺkáł w jáſkini / on y dwie córki jego.
Tedy rzekłá ſtárƺá do młodƺej : Ojćiec náƺ ſtáry jeſt á niemáƺ mężá ná ziemi / któryby wƺedł do náſ / według zwycżáju wƺyſtkiej ziemi.
Pójdź / upójmy ojcá náƺego winem / á śpijmy z nim / ábyśmy záchowáły z ojcá náƺego náśienie.
Dáły tedy pić ojcu ſwemu winá onej nocy. Y wƺedłƺy ſtárƺá ſpáłá z ojcem ſwym ; ále on nie cżuł áni kiedy śię ukłádłá / áni kiedy wſtáłá.
Y ſtáło śię názájutrz / że rzekłá ſtárƺá do młodƺej : Otom já ſpáłá przeƺłej nocy z ojcem ſwym / dájmyż mu pić winá jeƺcże tej nocy / y wnijdzieƺ / y będzieƺ ſpáłá z nim / á záchowámy z ojcá náƺego náśienie.
Tedy dáły pić y onej nocy ojcu ſwemu winá ; y przyƺedłƺy młodƺá ſpáłá z nim ; ále on nie cżuł / áni kiedy śię ukłádłá / áni kiedy wſtáłá.
A ták pocżęły obie córki Lotowe z ojcá ſwego.
Y urodziłá ſtárƺá ſyná / á názwáłá imię jego Moáb ; ten jeſt ojcem Moábitów / áż do dniá tego.
Młodƺá też urodziłá ſyná / y názwáłá imię jego Benámmi ; ten jeſt ojcem ſynów Ammonowych / áż do dniá tego.
