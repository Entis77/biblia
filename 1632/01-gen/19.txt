Y przyƺli dwá Anjołowie do Sodomy w wieczór : á Lot śiedźiał w bramie Sodomſkiey. <i>Gdy je</i> tedy ujrzał Lot / wſtawƺy ƺedł przećiwko nim / y ſkłonił śię twarzą ku źiemi / y rzekł.
Oto / proƺę Pánowie moji / wſtąpćie teraz do domu ſługi ſwego / á bądźćie tu ná noc / y umyjćie nogi ſwe : potym ráno wſtawƺy pójdźiećie w drogę wáƺę : którzy odpowiedźieli : By namniey : ále ná ulicy będźiemy nocowáli.
Ale on przymuƺał ich bárzo / iż śię ſkłonili do niego / y weƺli w dom jego : záczym ſpráwił im ucztę / y nápiekł <i>chlebá</i> przáſnego / y jedli.
Lecż pierwey niż oni poƺli ſpáć oto obywátele miáſtá / mężowie Sodomſcy / obſtąpili dom / od młodego áż do ſtárego wƺyſtek lud zewƺąd.
Y wołáli ná Lotá y rzekli mu / Gdźie <i>ſą</i> mężowie którzy przyƺli do ćiebie w nocy? wywiedź je do nas ábyſmy je poználi.
Tedy wyƺedł do nich Lot ze drzwi / y zámknął drzwi zá ſobą.
Y rzekł / Nie czyńćie proƺę bráćia moji <i>nic</i> złego.
Oto teraz mam dwie córce które niepoznáły mężá / wywiodę je teraz do was / á czyńćie z nimi co śię wam podoba / tylko mężom tym nic nie czyńćie / bo dla tego weƺli pod ćień dáchu mego.
A <i>oni</i> rzekli / Pódźże tám : y mówili : <i>Ten</i> ſam przyƺedł áby tu gośćiem był / á miałby nas ſądźić? przetoż gorzy uczynimy tobie / niż onym : y czynili gwałt wielki mężowi <i>onemu</i> Lotowi / y przyſtąpili áby drzwi wyłamáli.
Ale mężowie oni / wyćiągnąwƺy rękę ſwoję / w wiedli Lotá do śiebie w dom / y zámknęli drzwi.
A męże one którzy byli u drzwi domu / pozárażáli ślepotą / od namniejƺego / áż do nawiękƺego : <i>ták</i> yż śię ſprácowáli ƺukájąc drzwi.
Tedy rzekli mężowie oni do Lotá. Máƺli tu jeƺcze kogo / źięćiá álbo Syny twe / álbo Córki twoje / y wƺyſtko co maƺ w mieśćie / wyprowadź z miejſcá <i>tego</i>.
Zkáźimy bowiem to miejſce / przeto że śię wzmógł krzyk ich przed Pánem / y poſłał nas Pan / ábyſmy je zkáźili.
Tedy wyƺedƺy Lot mówił do źięćiów ſwojch / którzy mieli pojąć córki jego. Y rzekł : Wſtańćie / wynidźćie z miejſcá tego / bo zkáźi Pan <i>to</i> miáſto / ále śię zdáło w oczách źięćiów jego jákoby żártował.
A gdy weƺłá zorza / przymuƺáli Anjołowie Lotá / mówiąc : Wſtań / weźmi żonę twoję / y dwie córki twoje / które tu ſą : byś ſnadź nie zginął w niepráwośći miáſtá <i>tego</i>.
A gdy śię oćiągał / ujęli mężowie oni rękę jego / y rękę żony jego / y rękę dwu córek jego / ( ábowiem mu Pan folgował ) y wywiedli go / y poſtáwili go przed miáſtem.
Y gdy je wywiedli precz / rzekł <i>jeden</i> : Jeśli chceƺ / záchowaj duƺę twoję / á nie oglądaj śię ná zad / áni ſtawaj ná ty wƺyſtkiey równinie : uchodź ná górę / byś ſnadź nie zginął.
A Lot rzekł do nich : nie ták / proƺę / Pánowie moji :
Oto teraz ználazł ſługá twój łáſkę w oczách twojch / y okazałeś obfićie miłośierdźie twoje / któreś uczynił ze mną / záchowawƺy duƺę moję / áleć ja nie będę mógł ujść ná tę górę / by mię ſnadź / nie záchwyćiło to złe / y umárłbym.
Ale oto <i>tu jeſt</i> miáſto nie dáleko / do któregobym ućiekł : máłeć jeſt / proƺę / niech tám ujdę : wƺák máłe jeſt / á będźie żywá duƺá mojá.
Tedy rzekł do niego / Oto y w tem wyſłuchałem ćię ábym nie wywróćił miáſtá tego / o któremeś mówił.
Spieƺże śię / á uchodź tám / bo nie będę mógł nic uczynić / áż ty tám dojdźieƺ : przeto názwáne jeſt imię miáſtá onego Zoár.
Wtym ſłońce weƺło ná źiemię / á Lot wƺedł do Zoár.
Tedy Pan ſpuśćił jáko deƺcz ná Sodomę y ná Gomorrę śiárkę y ogień / od Páná z Niebá.
Y wywróćił miáſtá one / y wƺyſtkę onę równinę : y wƺyſtkie obywátele miaſt onych y urodzáie oney źiemie.
Y obejrzáłá śię żoná jego idąc zá nim / á obróćiłá śię w ſłup ſolny.
Wſtawƺy tedy Abráhám ráno / poſpieƺył śię ná ono miejſce / kędy ſtał przed Pánem.
Y pojrzał ku Sodomie y Gomorrze / y ku wƺyſtkiey źiemi oney równey / y obaczył / á oto wychodźił dym z oney źiemie jáko dym z piecá.
A gdy wywracał Bóg miáſtá oney równiny / wſpomniał Bóg ná Abráhámá / y wybáwił Lotá z pośrzodku wywrócenia / gdy wywracał one miáſtá w których mieƺkał Lot.
Potym wyƺedł Lot z Zoár / y mieƺkał ná górze / y dwie córce jego z nim : ábowiem śię bał mieƺkáć w Zoár / ále mieƺkał w jáſkini / on y dwie córki jego.
Tedy rzekłá ſtárƺa do młodƺey / Oćiec náƺ ſtáry <i>jeſt</i> á niemáƺ mężá ná źiemi / któryby wƺedł do nas / według zwyczáju wƺyſtkiey źiemie.
Pódź / upójmy Ojcá náƺego winem / á ſpimy z nim / ábyſmy záchowáły z Ojcá náƺego naśienie.
Dáły tedy pić Ojcu ſwemu winá oney nocy / y wƺedƺy ſtárƺa ſpáłá z Ojcem ſwym / ále on nie czuł áni kiedy śię ukłádłá / áni kiedy wſtáłá.
Y ſtáło śię názájutrz / że rzekłá ſtárƺa do młodƺey : Otom ja ſpáłá przeƺłey nocy z Ojcem ſwym / dajmyż mu pić winá jeƺcze tey nocy / y wnidźieƺ / y będźieƺ ſpáłá z nim / á záchowamy z Ojcá náƺego naśienie.
Tedy dáły pić y oney nocy Ojcu ſwemu winá / y przyƺedƺy młodƺa ſpáłá z nim / ále on nie czuł / áni kiedy śię ukłádłá / áni kiedy wſtáłá.
A ták poczęły obie córki Lotowe z Ojcá ſwego.
Y urodźiłá ſtárƺa Syná / á názwáłá imię jego Moab : ten <i>jeſt</i> Ojcem Moábitów / áż do dniá tego.
Młodƺa też urodźiłá Syná / y názwáłá imię jego Benámmi : <i>ten jeſt</i> Ojcem Synów Ammonowych / áż do dniá tego.
