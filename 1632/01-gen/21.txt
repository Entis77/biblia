A Pán náwiedził Sárę / jáko był rzekł : y ucżynił Pán Sárze / jáko był powiedziáł.
Bo pocżęłá y porodziłá Sárá Abráhámowi ſyná w ſtárośći jego / ná tenże cżáſ / który mu był Bóg przepowiedziáł.
Y názwáł Abráhám imię ſyná ſwego / który mu śię urodził / którego mu urodziłá Sárá / Izáák.
Y obrzezáł Abráhám Izááká / ſyná ſwego / gdy był w ośmiu dniách / jáko mu był rozkázáł Bóg.
A było Abráhámowi ſto lát / gdy mu śię urodził Izáák / ſyn jego.
Tedy rzekłá Sárá : Śmiech mi ucżynił Bóg ; ktokolwiek uſłyƺy / śmiáć śię będzie ze mną.
Y rzekłá : Któżby to był rzekł Abráhámowi / że Sárá będzie kármiłá pierśiámi ſyny? gdyżem urodziłá ſyná w ſtárośći jego.
Roſło tedy dziećię / y odſtáwione jeſt od pierśi ; y ucżynił Abráhám ucżtę wielką w dzień odſtáwieniá Izááká.
Potem ujrzáłá Sárá ſyná Hágáry / Egipcżánki / przeƺydzájącego / którego urodziłá Abráhámowi ;
Y rzekłá do Abráhámá : Wyrzuć tę ſłużebnicę / y ſyná jej ; ábowiem nie będzie dziedzicżył ſyn tej ſłużebnicy z ſynem mym Izáákiem.
Ale śię to bárdzo niepodobáło w ocżách Abráhámowych / dlá ſyná jego.
Tedy rzekł Bóg do Abráhámá : Niech to przykro nie będzie w ocżách twoich z ſtrony dziećięćiá / y z ſtrony ſłużebnicy twojej ; coćkolwiek rzecże Sárá / uſłucháj głoſu jej ; boć w Izááku názwáne będzie náśienie.
Wƺákże y ſyná ſłużebnicy rozmnożę w náród / przeto iż náśieniem twojim jeſt.
Wſtáł tedy Abráhám bárdzo ráno / á wziąwƺy chleb y łágiew wody / dáł Hágárze ; y włożywƺy to ná rámię jej / y z dziećięćiem / odpráwił ją ; którá poƺedłƺy błąkáłá śię po puƺcży Beerſebá.
A gdy nie ſtáło wody w łágwi / porzućiłá dziećię pod jednem drzewem ;
Y odƺedłƺy uśiádłá przećiw niemu / ták dáleko / jáko ná ſtrzeleniu z łuku ; bo mówiłá : Nie będę pátrzyłá ná śmierć dziećięćiá ; á śiedząc przećiw niemu / podnioſłá głoſ ſwój / y płákáłá.
Tedy uſłyƺáł Bóg głoſ dziećięcy / y záwołáł Anioł Boży ná Hágárę z niebá / y rzekł jej : Cóżći Hágáro? nie bój śię / boć uſłyƺáł Bóg głoſ dziećięcy z miejſcá / ná którem jeſt.
Wſtáń / weźmij dziećię / á ujmij je ręką ſwoją : bo w náród wielki rozmnożę je.
Otworzył tedy Bóg ocży jej / że ujrzáłá źródło wody ; á ƺedłƺy nápełniłá łágiew wodą / y dáłá pić dziećięćiu.
Y był Bóg z onem dziećięćiem / które uroſło / y mieƺkáło ná puƺcży / był z niego ſtrzelec dobry z łuku.
A mieƺkáł ná puƺcży Fárán ; y wzięłá mu mátká jego żonę z ziemi Egipſkiej.
Y ſtáło śię onegoż cżáſu / że rzekł Abimelech / y Fikol / hetmán wojſká jego / do Abráhámá mówiąc : Bóg z tobą we wƺyſtkiem / co ty cżyniƺ.
A ták teráz / przyśiąż mi przez Bogá / że mię w nicżem podchodzić nie będzieƺ / áni ſyná mego / áni wnuká mego ; ále według miłośierdziá / którem ucżynił z tobą / ucżyniƺ ze mną / y z ziemią / w którejś był przychodniem.
Tedy odpowiedziáł Abráhám : Já przyśięgnę.
Y przymáwiáł Abráhám Abimelechowi o ſtudnią wody / którą mu byli gwáłtem odjęli ſłudzy Abimelechowi.
Y rzekł Abimelech : Niewiem ktoby to ucżynił / náwet y tyś mi nie oznájmił / y jám nie ſłyƺáł o tem dopiero dziś.
Nábráł tedy Abráhám owiec y wołów / y dáł Abimelechowi / y ucżynili obá przymierze.
Y poſtáwił Abráhám śiedmioro owiec z ſtádá oſobno.
Tedy Abimelech rzekł do Abráhámá : Ná cóż to śiedmioro owiec / któreś poſtáwił oſobno?
A on odpowiedziáł : Iż te śiedm owiec weźmieƺ z rąk moich / áby mi były ná świádectwo / żem wykopáł tę ſtudnią.
Dlá tego názwáno miejſce ono Beerſebá ; ábowiem tám obáj przyśięgli.
A ták záwárli przymierze w Beerſebá. Potem wſtáwƺy Abimelech / y Fikol / hetmán wojſká jego / wróćili śię do ziemi Filiſtyńſkiej.
Y náſádził Abráhám drzewá w Beerſebá / y wzywáł tám imieniá Páná Bogá wiecżnego.
Y mieƺkáł Abráhám w ziemi Filiſtyńſkiej przez wiele dni.
