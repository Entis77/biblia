A Pán náwiedźił Sárę / jáko był rzekł : y ucżynił Pán Sárze / jáko był powiedźiáł.
Bo pocżęłá y porodźiłá Sárá Abráhámowi ſyná w ſtárośći jego / ná tenże cżáſ / który mu był Bóg przepowiedźiáł.
Y názwáł Abráhám imię ſyná ſwego / który mu śię urodźił / którego mu urodźiłá Sárá / Izáák.
Y obrzezáł Abráhám Izááká / ſyná ſwego / gdy był w ośmiu dniách / jáko mu był rozkázáł Bóg.
A było Abráhámowi ſto lát / gdy mu śię urodźił Izáák / ſyn jego.
Tedy rzekłá Sárá : Śmiech mi ucżynił Bóg ; ktokolwiek uſłyƺy / śmiáć śię będźie ze mną.
Y rzekłá : Któżby to był rzekł Abráhámowi / że Sárá będźie kármiłá pierśiámi ſyny? gdyżem urodźiłá ſyná w ſtárośći jego.
Roſło tedy dźiećię / y odſtáwione jeſt od pierśi ; y ucżynił Abráhám ucżtę wielką w dźień odſtáwieniá Izááká.
Potem ujrzáłá Sárá ſyná Hágáry / Egipcżánki / przeƺydzájącego / którego urodźiłá Abráhámowi ;
Y rzekłá do Abráhámá : Wyrzuć tę ſłużebnicę / y ſyná jej ; ábowiem nie będźie dźiedźicżył ſyn tej ſłużebnicy z ſynem mym Izáákiem.
Ale śię to bárdzo niepodobáło w ocżách Abráhámowych / dlá ſyná jego.
Tedy rzekł Bóg do Abráhámá : Niech to przykro nie będźie w ocżách twoich z ſtrony dźiećięćiá / y z ſtrony ſłużebnicy twojej ; coćkolwiek rzecże Sárá / uſłucháj głoſu jej ; boć w Izááku názwáne będźie náśienie.
Wƺákże y ſyná ſłużebnicy rozmnożę w náród / przeto iż náśieniem twojim jeſt.
Wſtáł tedy Abráhám bárdzo ráno / á wźiąwƺy chleb y łágiew wody / dáł Hágárze ; y włożywƺy to ná rámię jej / y z dźiećięćiem / odpráwił ją ; którá poƺedłƺy błąkáłá śię po puƺcży Beerſebá.
A gdy nie ſtáło wody w łágwi / porzućiłá dźiećię pod jednem drzewem ;
Y odƺedłƺy uśiádłá przećiw niemu / ták dáleko / jáko ná ſtrzeleniu z łuku ; bo mówiłá : Nie będę pátrzyłá ná śmierć dźiećięćiá ; á śiedząc przećiw niemu / podnioſłá głoſ ſwój / y płákáłá.
Tedy uſłyƺáł Bóg głoſ dźiećięcy / y záwołáł Anioł Boży ná Hágárę z niebá / y rzekł jej : Cóżći Hágáro? nie bój śię / boć uſłyƺáł Bóg głoſ dźiećięcy z miejſcá / ná którem jeſt.
Wſtáń / weźmij dźiećię / á ujmij je ręką ſwoją : bo w náród wielki rozmnożę je.
Otworzył tedy Bóg ocży jej / że ujrzáłá źródło wody ; á ƺedłƺy nápełniłá łágiew wodą / y dáłá pić dźiećięćiu.
Y był Bóg z onem dźiećięćiem / które uroſło / y mieƺkáło ná puƺcży / był z niego ſtrzelec dobry z łuku.
A mieƺkáł ná puƺcży Fárán ; y wźięłá mu mátká jego żonę z źiemi Egipſkiej.
Y ſtáło śię onegoż cżáſu / że rzekł Abimelech / y Fikol / hetmán wojſká jego / do Abráhámá mówiąc : Bóg z tobą we wƺyſtkiem / co ty cżyniƺ.
A ták teráz / przyśiąż mi przez Bogá / że mię w nicżem podchodźić nie będźieƺ / áni ſyná mego / áni wnuká mego ; ále według miłośierdźiá / którem ucżynił z tobą / ucżyniƺ ze mną / y z źiemią / w którejś był przychodniem.
Tedy odpowiedźiáł Abráhám : Já przyśięgnę.
Y przymáwiáł Abráhám Abimelechowi o ſtudnią wody / którą mu byli gwáłtem odjęli ſłudzy Abimelechowi.
Y rzekł Abimelech : Niewiem ktoby to ucżynił / náwet y tyś mi nie oznájmił / y jám nie ſłyƺáł o tem dopiero dźiś.
Nábráł tedy Abráhám owiec y wołów / y dáł Abimelechowi / y ucżynili obá przymierze.
Y poſtáwił Abráhám śiedmioro owiec z ſtádá oſobno.
Tedy Abimelech rzekł do Abráhámá : Ná cóż to śiedmioro owiec / któreś poſtáwił oſobno?
A on odpowiedźiáł : Iż te śiedm owiec weźmieƺ z rąk moich / áby mi były ná świádectwo / żem wykopáł tę ſtudnią.
Dlá tego názwáno miejſce ono Beerſebá ; ábowiem tám obáj przyśięgli.
A ták záwárli przymierze w Beerſebá. Potem wſtáwƺy Abimelech / y Fikol / hetmán wojſká jego / wróćili śię do źiemi Filiſtyńſkiej.
Y náſádźił Abráhám drzewá w Beerſebá / y wzywáł tám imieniá Páná Bogá wiecżnego.
Y mieƺkáł Abráhám w źiemi Filiſtyńſkiej przez wiele dni.
