A Pán náwiedźił Sárę / jáko był rzekł : y ucżynił Pán Sárze / jáko był powiedźiał.
Bo pocżęłá y porodźiłá Sárá Abráhámowi ſyná w ſtárośći jego / ná tenże cżás / który mu był Bóg przepowiedźiał.
Y názwał Abráhám imię ſyná ſwego / który mu śię urodźił / którego mu urodźiłá Sárá / Izáák.
Y obrzezał Abráhám Izááká / ſyná ſwego / gdy był w ośmiu dniách / jáko mu był rozkázał Bóg.
A było Abráhámowi ſto lat / gdy mu śię urodźił Izáák / ſyn jego.
Tedy rzekłá Sárá : Śmiech mi ucżynił Bóg ; ktokolwiek uſłyƺy / śmiáć śię będźie ze mną.
Y rzekłá : Któżby to był rzekł Abráhámowi / że Sárá będźie kármiłá pierśiámi ſyny? gdyżem urodźiłá ſyná w ſtárośći jego.
Roſło tedy dźiećię / y odſtáwione jeſt od pierśi ; y ucżynił Abráhám ucżtę wielką w dźień odſtáwieniá Izááká.
Potem ujrzáłá Sárá ſyná Hágáry / Egipcżánki / przeƺydzájącego / którego urodźiłá Abráhámowi ;
Y rzekłá do Abráhámá : Wyrzuć tę ſłużebnicę / y ſyná jey ; ábowiem nie będźie dźiedźicżył ſyn tey ſłużebnicy z ſynem mym Izáákiem.
Ale śię to bárzo niepodobáło w ocżách Abráhámowych / dla ſyná jego.
Tedy rzekł Bóg do Abráhámá : Niech to przykro nie będźie w ocżách twojich z ſtrony dźiećięćiá / y z ſtrony ſłużebnicy twojey ; coćkolwiek rzecże Sárá / uſłucháj głoſu jey ; boć w Izááku názwáne będźie naśienie.
Wƺákże y ſyná ſłużebnicy rozmnożę w naród / przeto iż náśieniem twojim jeſt.
Wſtał tedy Abráhám bárzo ráno / á wźiąwƺy chleb y łágiew wody / dał Hágárze ; y włożywƺy to ná rámię jey / y z dźiećięćiem / odpráwił ją ; która poƺedƺy błąkáłá śię po puƺcży Beerſebá.
A gdy nie ſtáło wody w łágwi / porzućiłá dźiećię pod jednem drzewem ;
Y odƺedƺy uśiádłá przećiw niemu / ták dáleko / jáko ná ſtrzeleniu z łuku ; bo mówiłá : Nie będę pátrzyłá ná śmierć dźiećięćiá ; á śiedząc przećiw niemu / podnioſłá głos ſwój / y płákáłá.
Tedy uſłyƺał Bóg głos dźiećięcy / y záwołał Anjoł Boży ná Hágárę z niebá / y rzekł jey : Cóżći Hágáro? nie bój śię / boć uſłyƺał Bóg głos dźiećięcy z miejſcá / ná którem jeſt.
Wſtáń / weźmij dźiećię / á ujmij je ręką ſwoją : bo w naród wielki rozmnożę je.
Otworzył tedy Bóg ocży jey / że ujrzáłá źródło wody ; á ƺedƺy nápełniłá łágiew wodą / y dáłá pić dźiećięćiu.
Y był Bóg z onem dźiećięćiem / które uroſło / y mieƺkáło ná puƺcży / był z niego ſtrzelec dobry z łuku.
A mieƺkał ná puƺcży Fárán ; y wźięłá mu mátká jego żonę z źiemi Egipſkiej.
Y ſtáło śię onegoż cżáſu / że rzekł Abimelech / y Fikol / hetmán wojſká jego / do Abráhámá mówiąc : Bóg z tobą we wƺyſtkim / co ty cżyniƺ.
A ták teraz / przyśiąż mi przez Bogá / że mię w nicżem podchodźić nie będźieƺ / áni ſyná mego / áni wnuká mego ; ále według miłośierdźiá / którem ucżynił z tobą / ucżyniƺ ze mną / y z źiemią / w którejś był przychodniem.
Tedy odpowiedźiał Abráhám : Ja przyśięgnę.
Y przymáwiał Abráhám Abimelechowi o ſtudnią wody / którą mu byli gwáłtem odjęli ſłudzy Abimelechowi.
Y rzekł Abimelech : Niewiem ktoby to ucżynił / náwet y tyś mi nie oznájmił / y jam nie ſłyƺał o tym dopiero dźiś.
Nábrał tedy Abráhám owiec y wołów / y dał Abimelechowi / y ucżynili obá przymierze.
Y poſtáwił Abráhám śiedmioro owiec z ſtádá oſobno.
Tedy Abimelech rzekł do Abráhámá : Ná cóż to śiedmioro owiec / któreś poſtáwił oſobno?
A on odpowiedźiał : Iż te śiedm owiec weźmieƺ z rąk mojich / áby mi były ná świádectwo / żem wykopał tę ſtudnią.
Dla tego názwano miejſce ono Beerſebá ; ábowiem tám obáj przyśięgli.
A ták záwárli przymierze w Beerſebá. Potem wſtáwƺy Abimelech / y Fikol / hetmán wojſká jego / wróćili śię do źiemi Filiſtyńſkiej.
Y náſádźił Abráhám drzewá w Beerſebá / y wzywał tám imienia Páná Bogá wiecżnego.
Y mieƺkał Abráhám w źiemi Filiſtyńſkiej przez wiele dni.
