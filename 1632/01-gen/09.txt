Y błogoſłáwił Bóg Noego / y ſyny jego / y rzekł im : Rozrádzájćie śię / y rozmnáżájćie śię / y nápełniájćie źiemię.
A ſtrách wáƺ y bojáźń wáƺá będźie nád wƺelkiem zwierzęćiem źiemi / y nád wƺyſtkiem ptáſtwem niebieſkiem / y nád wƺyſtkiem / co śię ruchá ná źiemi / y nád wƺyſtkiemi rybámi morſkiemi : w rękę wáƺę podáne ſą.
Wƺyſtko co śię ruchá / y co żyje / wám będźie ná pokárm / jáko járzynę źieloną / dáłem wám to wƺyſtko.
Wƺákże mięſá z duƺą jego / którá jeſt krew jego / jeść nie będźiećie.
A záiſte krwi wáƺej / duƺ wáƺych ƺukáć będę / z ręki káżdej beſtyi ƺukáć jej będę ; tákże z ręki cżłowiecżej / z ręki káżdego brátá jego będę ƺukáł duƺy cżłowiecżej.
Kto wyleje krew cżłowiecżą / przez cżłowieká krew jego wyláná będźie ; bo ná wyobráżenie Boże ucżynion jeſt cżłowiek.
A wy rozrádzájćie śię / y rozmnáżájćie śię / rozpłádzájćie śię ná źiemi / y mnóżćie śię ná niej.
Tedy rzekł Bóg do Noego / y do ſynów jego z nim / mówiąc :
A Já / oto Já ſtánowię przymierze moje z wámi / y z náśieniem wáƺem po wáſ.
Y z káżdą duƺą żywiącą / którá jeſt z wámi ; w ptáſtwie / w bydle / y w káżdem zwierzęćiu źiemi / które ſą z wámi / ze wƺyſtkich / co wyƺły z korábiá / áż do káżdego zwierzęćiá ná źiemi.
Y poſtánowię przymierze moje z wámi ; á nie będźie zátrácone więcej wƺelkie ćiáło wodámi potopu ; y nie będźie więcej potop ná ſkáżenie źiemi.
Tedy rzekł Bóg : To jeſt znák przymierzá / który Já dáwám między mną y między wámi / y między káżdą duƺą żywiącą / którá jeſt z wámi / w rodzáje wiecżne.
Łuk mój położyłem ná obłoku / który będźie ná znák przymierzá między mną / y między źiemią.
Y ſtánie śię / gdy wzbudzę ćiemny obłok nád źiemią / á ukáże śię łuk ná obłoku :
Że wſpomnę ná przymierze moje / które jeſt między mną y między wámi / y między káżdą duƺą żywiącą w káżdem ćiele ; y nie będą więcej wody ná potop / ku wytráceniu wƺelkiego ćiáłá.
Będźie tedy łuk on ná obłoku / y wejrzę náń / ábym wſpomniáł ná przymierze wiecżne / między Bogiem y między wƺelką duƺą żywiącą w káżdem ćiele / które jeſt ná źiemi.
Zátem rzekł Bóg do Noego : Tenći jeſt znák przymierzá / którem poſtánowił między mną / y między wƺelkiem ćiáłem / które jeſt ná źiemi.
A byli ſynowie Noego / którzy wyƺli z korábiá / Sem / y Chám / y Jáfet ; á Chám jeſt ojcem Chánáán.
Ci trzej ſą ſynowie Noego / przez które śię nápełniłá ludem wƺyſtká źiemiá.
Tedy Noe pocżął upráwiáć źiemię / y náſádźił winnicę.
Potem pił wino ; á upiwƺy śię / odkrył śię w námiećie ſwoim.
A ujrzáwƺy Chám / ojćiec Chánáánów / nágość ojcá ſwego / oznájmił to dwom bráćiom ſwoim ná dworze.
Tedy wźiąwƺy Sem y Jáfet ƺátę / á włożywƺy ją obá ná rámioná ſwe / ƺli wſpák / y zákryli nágość ojcá ſwego ; á oblicżá ich odwrócone były / że nágośći ojcá ſwego nie widźieli.
A ocućiwƺy śię Noe z winá ſwego / gdy śię dowiedźiáł / co mu ucżynił ſyn jego młodƺy / rzekł :
Przeklęty Chánáán / ſługą ſług bráći ſwojej będźie.
Rzekł też : Błogoſłáwiony Pán Bóg Semów / á niech będźie Chánáán ſługą ich.
Niech rozƺerzy Bóg Jáfetá / y niech mieƺká w námiećiech Semowych / á niech będźie Chánáán ſługą ich.
Y żył Noe po potopie trzy ſtá lát / y pięćdźieśiąt lát.
Y było wƺyſtkich dni Noego / dźiewięć ſet lát / y pięćdźieśiąt lát / y umárł.
