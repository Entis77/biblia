Potem gdy uſłyƺáł Jákób ſłowá ſynów Lábánowych mówiących : Pobráł Jákób wƺyſtko / co miał ojćiec náƺ / y z tego / co było ojcá náƺego / tey wƺyſtkiej zácnośći doſtáł.
Widźiáł też Jákób twárz Lábánową / á oto / nie był tákim przećiwko niemu / jáko przedtem.
Tedy rzekł Pán do Jákóbá : Wróć śię do źiemi ojców twoich / y do rodźiny twojey / á będę z tobą.
Przetoż poſłáł Jákób / y wyzwáł Ráchelę y Liję ná pole do trzody ſwojey.
Y rzekł im : Widzę ja twárz ojcá wáƺego / że nie jeſt tákim przećiwko mnie / jáko przedtem / lecż Bóg ojcá mego był ze mną.
Wy też ſáme wiećie / żem ze wƺyſtkich śił moich ſłużył ojcu wáƺemu ;
Ale ojćiec wáƺ oƺukáł mię / y odmienił zápłátę moję po dźieśięć kroć ; jednák nie dopuśćił mu Bóg / áby mi ƺkodźił.
Jeſli kiedy powiedźiáł : Pſtre będą zápłátą twoją / tedy rodźiły wƺyſtkie owce jágniętá pſtre ; á gdy záś mówił : Strokáte będą zápłátą twoją / tedy rodźiły wƺyſtkie owce jágniętá ſtrokáte.
Y odjął Bóg dobytek ojcá wáƺego / á dáł go mnie.
Stáło śię bowiem w ten cżáſ / gdy śię owce złącżáły / żem podnióſł ocży ſwe / y widźiáłem przez ſen / á oto / ſámcy złącżáły śię z owcámi ſtrokátemi / pſtremi / y biáło nákrápiánemi.
Tedy mi rzekł Anioł Boży we śnie : Jákóbie! A jám odpowiedźiáł : Owom ja.
Potem rzekł : Podnieś teraz ocży ſwe / á obácż wƺyſtkie ſámce złącżájące śię z owcámi ſtrokátemi / pſtremi / y biáło nákrápiánemi ; bom widźiáł wƺyſtko / coć Lábán ucżynił.
Jám Bóg Betel / gdźieś námázáł kámień / gdźieś mi poślubił ślub. Teraz tedy wſtáń / wynijdź z źiemi tey / á wróć śię do źiemi rodźiny twojey.
Tedy odpowiedźiáłá Ráchel y Lijá / y rzekły mu : Izáż jeƺcże mámy cżąſtkę jáką y dźiedźictwo w domu ojcá náƺego?
Izáżeśmy zá obce nie były pocżytáne u niego? Iż nas przedáł ; y miáłżeby jeƺcże do ƺcżętu zjeść májętność náƺę?
Abowiem wƺyſtko bogáctwo / które odjął Bóg ojcu náƺemu / náƺe jeſt / y ſynów náƺych ; przetoż teraz wƺyſtko ucżyń / coć Bóg rozkázáł.
Wſtáł tedy Jákób / y wſádźił ſyny ſwe / y żony ſwe ná wielbłądy.
Y zábráł wƺyſtkę trzodę ſwoję / y wƺyſtkę májętność ſwoję / którey był nábył / dobytek nábyćiá ſwego / którego był nábył w kráinie Syryjſkiej / áby śię wróćił do Izááká / ojcá ſwego / do źiemi Chánánejſkiej.
A Lábán odƺedł był ſtrzydz owce ſwoje : wtem ukrádłá Ráchel báłwány / które miał ojćiec jey.
Y wykrádł śię Jákób potájemnie od Lábáná Syryjcżyká / ták że mu nie oznájmił / iż ućiekáł.
Y ućiekł ſám ze wƺyſtkiem co miał / á wſtáwƺy przepráwił śię przez rzekę / y udáł śię ku górze Gáláád.
Y dáno znáć Lábánowi dniá trzećiego / że ućiekł Jákób.
Który wźiąwƺy bráćią ſwoję z ſobą / gonił go przez śiedm dni / y dośćignął go ná górze Gáláád.
Lecż przyƺedł Bóg do Lábáná Syryjcżyká we śnie oney nocy / y rzekł mu : Strzeż śię / ábyś nie mówił z Jákóbem nic przykrego.
Y dogonił Lábán Jákóbá / á Jákób już był námiot ſwój rozbił ná górze ; Lábán też rozbił námiot z bráćią ſwą ná onejże górze Gáláád.
Tedy Lábán rzekł do Jákóbá : Cóżeś ucżynił / żeś śię wykrádł potájemnie odemnie / á uwiodłeś córki moje / jákoby pojmáne miecżem?
Przecżżeś potájemnie ućiekł / á wykrádłeś śię odemnie / á nie oznájmiłeś mi / gdyżbym ćię był puśćił z rádośćią / y z pieśniámi / y z bębnem / y z hárfą?
Y nie dopuśćiłeś mi / ábym pocáłowáł ſyny moje / y córki moje? Záiſte głupieś ſobie pocżął.
Jeſt to w mocy ręki mojey / ucżynić wám co złego ; ále Bóg ojcá wáƺego przeƺłej nocy rzekł do mnie / mówiąc : Strzeż śię / ábyś z Jákóbem nie mówił nic przykrego.
A teraz gdyć śię chćiáło odejść / żeś wielce prágnął do domu ojcá twego / cżemużeś ukrádł bogi moje?
Y odpowiádájąc Jákób / rzekł do Lábáná : Iżem śię báł / y myśláłem / byś mi ſnáć nie wydárł córek twoich.
Lecż ten / u kogo znájdźieƺ bogi twoje / niech umrze ; przed bráćią náƺą poznájże / co twego u mnie / y weźmij ſobie ; á nie wiedźiáł Jákób / że je Ráchel ukrádłá.
Wƺedł tedy Lábán do námiotu Jákóbowego / y do námiotu Lii / y do námiotu obydwóch ſłużebnic / á nie ználázł ; á wyƺedłƺy z námiotu Lii wƺedł do námiotu Rácheli.
A Ráchel wźiąwƺy one báłwány włożyłá je pod śiodło wielbłądowe / y uśiádłá ná nich ; y zmácáł Lábán wƺyſtek námiot / á nie ználázł.
Tedy oná rzekłá do ojcá ſwego : Niech śię nie gniewá pán mój / że nie mogę powſtáć przed twárzą twoją / bo według zwycżáju niewiáſt przypádło ná mię ; y ƺukáł / á nie ználázł báłwánów.
Rozgniewáł śię tedy Jákób / y fukáł ná Lábáná ; á odpowiádájąc Jákób / rzekł do Lábáná : Cóż zá przeſtępſtwo moje / co zá grzech mój / żeś mię gonił zápáliwƺy śię?
Otoś zmácáł wƺyſtek ſprzęt mój : cóżeś ználázł ze wƺyſtkiego ſprzętu domu twego? połóż tu przed bráćią moją / y bráćią twoją / á niech rozſądzą między námi dwomá.
Już dwádźieśćiá lát mieƺkáłem z tobą ; owce twoje y kozy twoje nie pomiátáły / á báránów ſtádá twego nie jádáłem.
Rozƺárpánego od zwierzá nie przynioſłem ći / jám ƺkodę nágrádzáł ; z ręki mojey ƺukáłeś tego / co było ukrádźione we dnie / y co było ukrádźione w nocy.
Bywáło to / że we dnie trápiło mię gorąco / á mróz w nocy / ták / że odchádzáł ſen mój od ocżu moich.
Jużem ći dwádźieśćiá lát w domu twoim ſłużył ; cżternáśćie lát zá dwie córki twoje / á ƺeść lát zá bydło twoje ; á odmieniáłeś zápłátę moję po dźieśięć kroć.
Y by był Bóg ojcá mego / Bóg Abráhámá / y ſtrách Izááká / nie był przy mnie / pewniebyś mię był teraz próżnego puśćił ; ále ná utrápienie moje / y ná prácę rąk moich wejrzáł Bóg / y przeſtrzegáł ćię nocy przeƺłej.
Tedy odpowiedźiáł Lábán / y rzekł do Jákóbá : Córki te córki ſą moje / y ſynowie ći ſą ſynowie moi / y dobytek ten dobytek mój / y wƺyſtko co widźiƺ / moje jeſt ; á tym córkom moim / cóż dźiś ucżynię / álbo ſynom ich / które zrodźiły?
Pójdźże tedy / á ucżyńmy przymierze / ja y ty / á będźie świádectwo między mną / y między tobą.
Y wźiął Jákób kámień / á poſtáwił go ná znák.
Y rzekł Jákób do bráći ſwej : Názbierájćie kámieni ; którzy nánośili kámieni / y ucżynili kupę / y jedli tám ná oney kupie.
Y názwáł ją Lábán Jegár Sáhádutá / á Jákób ją názwáł Gáled.
Bo mówił Lábán : Kupá tá niech będźie świádkiem między mną y między tobą dźiśiáj ; przetoż Jákób názwáł imię jey Gáled /
Y Myſpá ; ábowiem rzekł Lábán : Niech upátruje Pán między mną y między tobą / gdy śię rozejdźiemy jeden od drugiego.
Jeſli będźieƺ trápił córki moje / y jeſli pojmieƺ żony nád córki moje / nie máƺ tu nikogo między námi ; bácż / że Bóg jeſt świádkiem między mną y między tobą.
Y rzekł nád to Lábán do Jákóbá : Oto / tá kupá kámieni / y oto / znák ten / którym poſtáwił między mną y między tobą.
Świádkiem tá kupá / y świádkiem ten znák będźie tego / iż ja do ćiebie nie pójdę dálej zá tę kupę / y ty też nie pójdźieƺ do mnie zá tę kupę / y zá ten znák / ná złe.
Bóg Abráhámów y Bóg Náchorów niecháj rozſądzą między námi / Bóg ojcá ich. Przyśiągł tedy Jákób przez ſtrách ojcá ſwego Izááká.
Y nábił Jákób bydłá ná górze / y wezwáł bráći ſwej ku jedzeniu chlebá. Tedy jedli chleb / y nocowáli ná oney górze.
Potem Lábán wſtáwƺy bárdzo ráno / pocáłowáł ſyny ſwoje y córki ſwe / y błogoſłáwił im ; á odƺedłƺy / wróćił śię Lábán ná miejſce ſwoje.
