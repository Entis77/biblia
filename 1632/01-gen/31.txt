Potem gdy uſłyƺáł Jákób ſłowá ſynów Lábánowych mówiących : Pobráł Jákób wƺyſtko / co miáł ojćiec náƺ / y z tego / co było ojcá náƺego / tej wƺyſtkiej zácnośći doſtáł.
Widziáł też Jákób twárz Lábánową / á oto / nie był tákim przećiwko niemu / jáko przedtem.
Tedy rzekł Pán do Jákóbá : Wróć śię do ziemi ojców twoich / y do rodziny twojej / á będę z tobą.
Przetoż poſłáł Jákób / y wyzwáł Ráchelę y Liję ná pole do trzody ſwojej.
Y rzekł im : Widzę já twárz ojcá wáƺego / że nie jeſt tákim przećiwko mnie / jáko przedtem / lecż Bóg ojcá mego był ze mną.
Wy też ſáme wiećie / żem ze wƺyſtkich śił moich ſłużył ojcu wáƺemu ;
Ale ojćiec wáƺ oƺukáł mię / y odmienił zápłátę moję po dzieśięć kroć ; jednák nie dopuśćił mu Bóg / áby mi ƺkodził.
Jeźli kiedy powiedziáł : Pſtre będą zápłátą twoją / tedy rodziły wƺyſtkie owce jágniętá pſtre ; á gdy záś mówił : Strokáte będą zápłátą twoją / tedy rodziły wƺyſtkie owce jágniętá ſtrokáte.
Y odjął Bóg dobytek ojcá wáƺego / á dáł go mnie.
Stáło śię bowiem w ten cżáſ / gdy śię owce złącżáły / żem podnióſł ocży ſwe / y widziáłem przez ſen / á oto / ſámcy złącżáły śię z owcámi ſtrokátemi / pſtremi / y biáło nákrápiánemi.
Tedy mi rzekł Anioł Boży we śnie : Jákóbie! A jám odpowiedziáł : Owom já.
Potem rzekł : Podnieś teráz ocży ſwe / á obácż wƺyſtkie ſámce złącżájące śię z owcámi ſtrokátemi / pſtremi / y biáło nákrápiánemi ; bom widziáł wƺyſtko / coć Lábán ucżynił.
Jám Bóg Betel / gdzieś námázáł kámień / gdzieś mi poślubił ślub. Teráz tedy wſtáń / wynijdź z ziemi tej / á wróć śię do ziemi rodziny twojej.
Tedy odpowiedziáłá Ráchel y Lijá / y rzekły mu : Izáż jeƺcże mámy cżąſtkę jáką y dziedzictwo w domu ojcá náƺego?
Izáżeśmy zá obce nie były pocżytáne u niego? Iż náſ przedáł ; y miáłżeby jeƺcże do ƺcżętu zjeść májętność náƺę?
Abowiem wƺyſtko bogáctwo / które odjął Bóg ojcu náƺemu / náƺe jeſt / y ſynów náƺych ; przetoż teráz wƺyſtko ucżyń / coć Bóg rozkázáł.
Wſtáł tedy Jákób / y wſádził ſyny ſwe / y żony ſwe ná wielbłądy.
Y zábráł wƺyſtkę trzodę ſwoję / y wƺyſtkę májętność ſwoję / której był nábył / dobytek nábyćiá ſwego / którego był nábył w kráinie Syryjſkiej / áby śię wróćił do Izááká / ojcá ſwego / do ziemi Chánánejſkiej.
A Lábán odƺedł był ſtrzydz owce ſwoje : wtem ukrádłá Ráchel báłwány / które miáł ojćiec jej.
Y wykrádł śię Jákób potájemnie od Lábáná Syryjcżyká / ták że mu nie oznájmił / iż ućiekáł.
Y ućiekł ſám ze wƺyſtkiem co miáł / á wſtáwƺy przepráwił śię przez rzekę / y udáł śię ku górze Gáláád.
Y dáno znáć Lábánowi dniá trzećiego / że ućiekł Jákób.
Który wziąwƺy bráćią ſwoję z ſobą / gonił go przez śiedm dni / y dośćignął go ná górze Gáláád.
Lecż przyƺedł Bóg do Lábáná Syryjcżyká we śnie onej nocy / y rzekł mu : Strzeż śię / ábyś nie mówił z Jákóbem nic przykrego.
Y dogonił Lábán Jákóbá / á Jákób już był námiot ſwój rozbił ná górze ; Lábán też rozbił námiot z bráćią ſwą ná onejże górze Gáláád.
Tedy Lábán rzekł do Jákóbá : Cóżeś ucżynił / żeś śię wykrádł potájemnie odemnie / á uwiodłeś córki moje / jákoby pojmáne miecżem?
Przecżżeś potájemnie ućiekł / á wykrádłeś śię odemnie / á nie oznájmiłeś mi / gdyżbym ćię był puśćił z rádośćią / y z pieśniámi / y z bębnem / y z hárfą?
Y nie dopuśćiłeś mi / ábym pocáłowáł ſyny moje / y córki moje? Záiſte głupieś ſobie pocżął.
Jeſt to w mocy ręki mojej / ucżynić wám co złego ; ále Bóg ojcá wáƺego przeƺłej nocy rzekł do mnie / mówiąc : Strzeż śię / ábyś z Jákóbem nie mówił nic przykrego.
A teráz gdyć śię chćiáło odejść / żeś wielce prágnął do domu ojcá twego / cżemużeś ukrádł bogi moje?
Y odpowiádájąc Jákób / rzekł do Lábáná : Iżem śię báł / y myśláłem / byś mi ſnáć nie wydárł córek twoich.
Lecż ten / u kogo znájdzieƺ bogi twoje / niech umrze ; przed bráćią náƺą poznájże / co twego u mnie / y weźmij ſobie ; á nie wiedziáł Jákób / że je Ráchel ukrádłá.
Wƺedł tedy Lábán do námiotu Jákóbowego / y do námiotu Lii / y do námiotu obydwóch ſłużebnic / á nie ználázł ; á wyƺedłƺy z námiotu Lii wƺedł do námiotu Rácheli.
A Ráchel wziąwƺy one báłwány włożyłá je pod śiodło wielbłądowe / y uśiádłá ná nich ; y zmácáł Lábán wƺyſtek námiot / á nie ználázł.
Tedy oná rzekłá do ojcá ſwego : Niech śię nie gniewá pán mój / że nie mogę powſtáć przed twárzą twoją / bo według zwycżáju niewiáſt przypádło ná mię ; y ƺukáł / á nie ználázł báłwánów.
Rozgniewáł śię tedy Jákób / y fukáł ná Lábáná ; á odpowiádájąc Jákób / rzekł do Lábáná : Cóż zá przeſtępſtwo moje / co zá grzech mój / żeś mię gonił zápáliwƺy śię?
Otoś zmácáł wƺyſtek ſprzęt mój : cóżeś ználázł ze wƺyſtkiego ſprzętu domu twego? połóż tu przed bráćią moją / y bráćią twoją / á niech rozſądzą między námi dwomá.
Już dwádzieśćiá lát mieƺkáłem z tobą ; owce twoje y kozy twoje nie pomiátáły / á báránów ſtádá twego nie jádáłem.
Rozƺárpánego od zwierzá nie przynioſłem ći / jám ƺkodę nágrádzáł ; z ręki mojej ƺukáłeś tego / co było ukrádzione we dnie / y co było ukrádzione w nocy.
Bywáło to / że we dnie trápiło mię gorąco / á mróz w nocy / ták / że odchádzáł ſen mój od ocżu moich.
Jużem ći dwádzieśćiá lát w domu twoim ſłużył ; cżternáśćie lát zá dwie córki twoje / á ƺeść lát zá bydło twoje ; á odmieniáłeś zápłátę moję po dzieśięć kroć.
Y by był Bóg ojcá mego / Bóg Abráhámá / y ſtrách Izááká / nie był przy mnie / pewniebyś mię był teráz próżnego puśćił ; ále ná utrápienie moje / y ná prácę rąk moich wejrzáł Bóg / y przeſtrzegáł ćię nocy przeƺłej.
Tedy odpowiedziáł Lábán / y rzekł do Jákóbá : Córki te córki ſą moje / y ſynowie ći ſą ſynowie moi / y dobytek ten dobytek mój / y wƺyſtko co widziƺ / moje jeſt ; á tym córkom moim / cóż dziś ucżynię / álbo ſynom ich / które zrodziły?
Pójdźże tedy / á ucżyńmy przymierze / já y ty / á będzie świádectwo między mną / y między tobą.
Y wziął Jákób kámień / á poſtáwił go ná znák.
Y rzekł Jákób do bráći ſwej : Názbierájćie kámieni ; którzy nánośili kámieni / y ucżynili kupę / y jedli tám ná onej kupie.
Y názwáł ją Lábán Jegár Sáhádutá / á Jákób ją názwáł Gáled.
Bo mówił Lábán : Kupá tá niech będzie świádkiem między mną y między tobą dziśiáj ; przetoż Jákób názwáł imię jej Gáled /
Y Myſpá ; ábowiem rzekł Lábán : Niech upátruje Pán między mną y między tobą / gdy śię rozejdziemy jeden od drugiego.
Jeźli będzieƺ trápił córki moje / y jeźli pojmieƺ żony nád córki moje / nie máƺ tu nikogo między námi ; bácż / że Bóg jeſt świádkiem między mną y między tobą.
Y rzekł nád to Lábán do Jákóbá : Oto / tá kupá kámieni / y oto / znák ten / którym poſtáwił między mną y między tobą.
Świádkiem tá kupá / y świádkiem ten znák będzie tego / iż já do ćiebie nie pójdę dálej zá tę kupę / y ty też nie pójdzieƺ do mnie zá tę kupę / y zá ten znák / ná złe.
Bóg Abráhámów y Bóg Náchorów niecháj rozſądzą między námi / Bóg ojcá ich. Przyśiągł tedy Jákób przez ſtrách ojcá ſwego Izááká.
Y nábił Jákób bydłá ná górze / y wezwáł bráći ſwej ku jedzeniu chlebá. Tedy jedli chleb / y nocowáli ná onej górze.
Potem Lábán wſtáwƺy bárdzo ráno / pocáłowáł ſyny ſwoje y córki ſwe / y błogoſłáwił im ; á odƺedłƺy / wróćił śię Lábán ná miejſce ſwoje.
