Rozkázáł tedy Józef temu / który był ſpráwcą domu jego / mówiąc : Nápełnij wory mężów tych zbożem / jáko mogą znieść / á włóż pieniądze káżdego ná wierzch woru jego.
Kubek też mój / kubek ſrebrny / włóż ná wierzch do woru młodƺego z pieniędzmi zá zboże jego ; y ucżynił według ſłów Józefowych / jáko mu rozkázáł.
A gdy było ráno / mężowie oni puƺcżeni ſą / ſámi y oſłowie ich.
Wyƺedłƺy tedy z miáſtá / gdy nie dáleko byli / rzekł Józef do tego / który był ſpráwcą domu jego : Wſtáń / goń te męże / á dogoniwƺy ich / mów do nich : Cżemuśćie oddáli złe zá dobre?
Azáż nie ten jeſt kubek / z którego pijá pán mój? y ázáż on pewnie nie zgádnie przezeń / jácyśćie wy? źleśćie ucżynili / cośćie ucżynili.
Dogoniwƺy ich tedy / mówił do nich te ſłowá.
Ale oni odpowiedźieli mu : Cżemu mówi pán mój tákowe ſłowá? nie dáj tego Boże / áby to ſłudzy twoi ucżynić mieli.
Oto pieniądze / któreśmy byli ználeźli ná wierzchu worów náƺych / odnieśliśmy záś do ćiebie z źiemi Chánánejſkiej ; á jákożbyśmy kráść mieli z domu páná twego ſrebro álbo złoto?
U któregoby to ználeźione z ſług twoich / niecháj umrze ; á my będźiemy páná mego niewolnikámi.
Tedy on rzekł : Niechże ták będźie / jáko mówićie ; jednák przy którym śię znájdźie kubek / ten niech będźie niewolnikiem / á wy będźiećie niewinnymi.
Prędko tedy káżdy złożył wór ſwój ná źiemię ; y rozwiązáli káżdy wór ſwój.
Y ƺukáł od ſtárƺego pocżąwƺy ; á u młodƺego przeſtáł ; y ználázł kubek w worze Benjáminowym.
Tedy oni rozdárli ƺáty ſwoje / y włożywƺy brzemię káżdy z nich ná oſłá ſwego / wróćili śię do miáſtá.
Przyƺedł tedy Judáſ / y bráćiá jego do domu Józefá / który tám jeƺcże był / y upádli przed oblicżem jego ná źiemię.
Y rzekł do nich Józef : Cóżeśćie to ucżynili? ázáśćie nie wiedźieli / że pewnie zgádnie mąż táki / jákim já jeſt?
Tedy odpowiedźiáł Judáſ : Cóż odpowiemy pánu memu / cóż rzecżemy? y jáko śię my uſpráwiedliwić mámy? Bóg ználázł niepráwość ſług twoich ; otośmy niewolnikámi páná mego / y my / y ten / w którego ręku ználeźiony jeſt kubek.
A on rzekł : Nie dáj Boże / ábym to ucżynić miáł! mąż / w którego ręku ználeźiony jeſt kubek / ten będźie niewolnikiem moim ; á wy jedźćie w pokoju do ojcá wáƺego.
Zátem przyſtąpił do niego Judáſ y rzekł : Słucháj mię pánie mój ; niecháj przemówi proƺę ſługá twój które ſłowo w uƺy páná mego / á niech śię nie zápálá gniew twój ná ſługę twego / gdyżeś ty jeſt jáko ſám Fáráo.
Pán mój pytáł ſług ſwoich mówiąc : Máćież ojcá álbo brátá?
Tedyśmy odpowiedźieli pánu ſwemu : Mámyć ojcá ſtárego / y chłopię w ſtárośći jego ſpłodzone máłe / á brát jego umárł / á zoſtáł ſám tylko po mátce ſwej / y ojćiec jego miłuje go.
Potem mówiłeś do náſ ſług ſwoich : Przywiedźćie go do mnie / ábym go oglądáł ocżymá memi :
Y mówiliśmy do páná mego : Nie będźie mogło chłopię opuśćić ojcá ſwego ; bo gdyby opuśćiło ojcá ſwego / umárłby.
Tedyś rzekł do ſług ſwoich : Jeźli nie przyjdźie brát wáƺ młodƺy z wámi / nie ujrzyćie więcej oblicżá mojego.
Y ſtáło śię / gdyśmy odeƺli do ſługi twego / ojcá mojego / y powiedźieliśmy mu te ſłowá páná mego :
Tedy rzekł ojćiec náƺ : Jedźćie znowu / á kupćie nám trochę żywnośći.
Y powiedźieliśmy : Nie możemy tám iść : lecż jeźli brát náƺ młodƺy będźie z námi / tedy pojedźiemy ; bo inácżej nie będźiemy mogli oglądáć oblicżá mężá onego / jeźli brát náƺ młodƺy nie będźie z námi.
Y rzekł ſługá twój / ojćiec mój / do náſ : Wy wiećie / że dwóch ſynów urodźiłá mi żoná mojá ;
Y wyƺedł jeden odemnie / y rzekłem : Záiſte od zwierzá rozdárty jeſt / y nie widźiáłem go do tych miáſt ;
A weźmiećieli y tego od oblicżá mego / á przypádnie náń śmierć / tedy doprowádźićie ſędźiwość moję z żáłośćią do grobu.
Przetoż teráz jeźlibym przyƺedł do ſługi twego / ojcá mojego / á dźiećięćiáby z námi nie było / ( poniewáż duƺá jego jeſt przywiązáná do duƺy jego ) /
Stánie śię / ſkoro ujrzy / iż dźiećięćiá nie będźie / że umrze ; á odprowádzą ſłudzy twoi ſędźiwość ſługi twego / ojcá náƺego / z żáłośćią do grobu.
Bo ſługá twój przyrzekł zá to dźiećię / gdy je bráł od ojcá ſwego / mówiąc : Jeźlić go záś nie przywiodę / tedy będę winien grzechu przećiw ojcu memu po wƺyſtkie dni.
Teráz tedy niech zoſtánie proƺę ſługá twój miáſto dźiećięćiá tego niewolnikiem páná mego / á dźiećię niech idźie z bráćią ſwoją.
Bo jákoż já mám wróćić śię do ojcá mego / gdy tego dźiećięćiá ze mną nie będźie? chybábym chćiáł pátrzyć ná żáłość / któráby przyƺłá ná ojcá mego.
