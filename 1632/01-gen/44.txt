Rozkázáł tedy Józef temu / który był ſpráwcą domu jego / mówiąc : Nápełnij wory mężów tych zbożem / jáko mogą znieść / á włóż pieniądze káżdego ná wierzch woru jego.
Kubek też mój / kubek ſrebrny / włóż ná wierzch do woru młodƺego z pieniędzmi zá zboże jego ; y ucżynił według ſłów Józefowych / jáko mu rozkázáł.
A gdy było ráno / mężowie oni puƺcżeni ſą / ſámi y oſłowie ich.
Wyƺedłƺy tedy z miáſtá / gdy nie dáleko byli / rzekł Józef do tego / który był ſpráwcą domu jego : Wſtáń / goń te męże / á dogoniwƺy ich / mów do nich : Cżemuśćie oddáli złe zá dobre?
Azáż nie ten jeſt kubek / z którego pijá pán mój? y ázáż on pewnie nie zgádnie przezeń / jácyśćie wy? źleśćie ucżynili / cośćie ucżynili.
Dogoniwƺy ich tedy / mówił do nich te ſłowá.
Ale oni odpowiedzieli mu : Cżemu mówi pán mój tákowe ſłowá? nie dáj tego Boże / áby to ſłudzy twoi ucżynić mieli.
Oto pieniądze / któreśmy byli ználeźli ná wierzchu worów náƺych / odnieśliśmy záś do ćiebie z ziemi Chánánejſkiej ; á jákożbyśmy kráść mieli z domu páná twego ſrebro álbo złoto?
U któregoby to ználezione z ſług twoich / niecháj umrze ; á my będziemy páná mego niewolnikámi.
Tedy on rzekł : Niechże ták będzie / jáko mówićie ; jednák przy którym śię znájdzie kubek / ten niech będzie niewolnikiem / á wy będziećie niewinnymi.
Prędko tedy káżdy złożył wór ſwój ná ziemię ; y rozwiązáli káżdy wór ſwój.
Y ƺukáł od ſtárƺego pocżąwƺy ; á u młodƺego przeſtáł ; y ználázł kubek w worze Benjáminowym.
Tedy oni rozdárli ƺáty ſwoje / y włożywƺy brzemię káżdy z nich ná oſłá ſwego / wróćili śię do miáſtá.
Przyƺedł tedy Judáſ / y bráćiá jego do domu Józefá / który tám jeƺcże był / y upádli przed oblicżem jego ná ziemię.
Y rzekł do nich Józef : Cóżeśćie to ucżynili? ázáśćie nie wiedzieli / że pewnie zgádnie mąż táki / jákim já jeſt?
Tedy odpowiedziáł Judáſ : Cóż odpowiemy pánu memu / cóż rzecżemy? y jáko śię my uſpráwiedliwić mámy? Bóg ználázł niepráwość ſług twoich ; otośmy niewolnikámi páná mego / y my / y ten / w którego ręku ználeziony jeſt kubek.
A on rzekł : Nie dáj Boże / ábym to ucżynić miáł! mąż / w którego ręku ználeziony jeſt kubek / ten będzie niewolnikiem moim ; á wy jedźćie w pokoju do ojcá wáƺego.
Zátem przyſtąpił do niego Judáſ y rzekł : Słucháj mię pánie mój ; niecháj przemówi proƺę ſługá twój które ſłowo w uƺy páná mego / á niech śię nie zápálá gniew twój ná ſługę twego / gdyżeś ty jeſt jáko ſám Fáráo.
Pán mój pytáł ſług ſwoich mówiąc : Máćież ojcá álbo brátá?
Tedyśmy odpowiedzieli pánu ſwemu : Mámyć ojcá ſtárego / y chłopię w ſtárośći jego ſpłodzone máłe / á brát jego umárł / á zoſtáł ſám tylko po mátce ſwej / y ojćiec jego miłuje go.
Potem mówiłeś do náſ ſług ſwoich : Przywiedźćie go do mnie / ábym go oglądáł ocżymá memi :
Y mówiliśmy do páná mego : Nie będzie mogło chłopię opuśćić ojcá ſwego ; bo gdyby opuśćiło ojcá ſwego / umárłby.
Tedyś rzekł do ſług ſwoich : Jeźli nie przyjdzie brát wáƺ młodƺy z wámi / nie ujrzyćie więcej oblicżá mojego.
Y ſtáło śię / gdyśmy odeƺli do ſługi twego / ojcá mojego / y powiedzieliśmy mu te ſłowá páná mego :
Tedy rzekł ojćiec náƺ : Jedźćie znowu / á kupćie nám trochę żywnośći.
Y powiedzieliśmy : Nie możemy tám iść : lecż jeźli brát náƺ młodƺy będzie z námi / tedy pojedziemy ; bo inácżej nie będziemy mogli oglądáć oblicżá mężá onego / jeźli brát náƺ młodƺy nie będzie z námi.
Y rzekł ſługá twój / ojćiec mój / do náſ : Wy wiećie / że dwóch ſynów urodziłá mi żoná mojá ;
Y wyƺedł jeden odemnie / y rzekłem : Záiſte od zwierzá rozdárty jeſt / y nie widziáłem go do tych miáſt ;
A weźmiećieli y tego od oblicżá mego / á przypádnie náń śmierć / tedy doprowádzićie ſędziwość moję z żáłośćią do grobu.
Przetoż teráz jeźlibym przyƺedł do ſługi twego / ojcá mojego / á dziećięćiáby z námi nie było / ( poniewáż duƺá jego jeſt przywiązáná do duƺy jego ) /
Stánie śię / ſkoro ujrzy / iż dziećięćiá nie będzie / że umrze ; á odprowádzą ſłudzy twoi ſędziwość ſługi twego / ojcá náƺego / z żáłośćią do grobu.
Bo ſługá twój przyrzekł zá to dziećię / gdy je bráł od ojcá ſwego / mówiąc : Jeźlić go záś nie przywiodę / tedy będę winien grzechu przećiw ojcu memu po wƺyſtkie dni.
Teráz tedy niech zoſtánie proƺę ſługá twój miáſto dziećięćiá tego niewolnikiem páná mego / á dziećię niech idzie z bráćią ſwoją.
Bo jákoż já mám wróćić śię do ojcá mego / gdy tego dziećięćiá ze mną nie będzie? chybábym chćiáł pátrzyć ná żáłość / któráby przyƺłá ná ojcá mego.
