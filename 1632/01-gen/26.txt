Potem był głód ná źiemi / po onym głodźie pierwƺym / który był zá dni Abráhámowych ; Y poƺedł Izáák do Abimelechá / królá Filiſtyńſkiego / do Gerár.
Bo mu śię był ukázał Pán y mówił : Nie zſtępuj do Egiptu / ále mieƺkáj w źiemi / o którey Ja powiem tobie.
Bądźże gośćiem w tey źiemi / á Ja będę z tobą / y będęć błogoſłáwił ; ábowiem tobie y náśieniu twemu dam te wƺyſtkie kráiny / y utwierdzę przyśięgę / którąm przyśiągł Abráhámowi / ojcu twemu.
Y rozmnożę náśienie twoje jáko gwiázdy niebieſkie / á dam náśieniu twemu wƺyſtkie te kráiny : A będą błogoſłáwione w náśieniu twojim wƺyſtkie národy źiemi.
Przeto / iż Abráhám był poſłuƺny głoſowi mojemu / á ſtrzegł poſtánowieniá mego / przykázáń mojich / uſtáw mojich y práw mojich.
Tedy Izáák mieƺkał w Gerár.
Y pytáli śię mężowie onego miejſcá o żonie jego ; á on powiedźiał : Sioſtrá to mojá ; bo śię bał mówić : Żoná to mojá ; by go ſnadź nie zábili mężowie miejſcá tego dla Rebeki / iż byłá piękná ná wejrzeniu.
Y ſtáło śię / gdy tám mieƺkał przez nie máło dni / że wyglądał Abimelech / król Filiſtyńſki / oknem / y ujrzał / że Izáák żártował z Rebeką / żoną ſwą.
Tedy przyzwał Abimelech do śiebie Izááká / y rzekł : Práwdźiwieć to żoná twojá ; cżemużeś powiádał / śioſtrá to mojá? Y odpowiedźiał mu Izáák : Iżem u śiebie mówił : Bym ſnadź nie umárł dla niej.
Y rzekł Abimelech : Cóżeś nam to ucżynił? bez máłá ktokolwiek z ludu nie ſpał z żoną twoją ; y przywiódłbyś był ná nas grzech.
Rozkázał tedy Abimelech wƺyſtkiemu ludowi mówiąc : Ktoby śię dotknął mężá tego / álbo żony jego / śmierćią umrze.
Tedy śiał Izáák w oney źiemi / y zebrał roku onego ſto korcy / ábowiem błogoſłáwił mu Pán.
Y zbogáćił śię on mąż / á im dálej / tym więcey wzmágał śię / áż uroſł wielce.
Y miał ſtádá owiec / y ſtádá wołów / y cżeládźi doſyć ; przetoż mu zájrzeli Filiſtyńcżycy.
Y wƺyſtkie ſtudnie / które byli wykopáli ſłudzy ojcá jego / zá dni Abráhámá / ojcá jego / záſypáli Filiſtyńcżycy / y nápełnili je źiemią.
Y rzekł Abimelech do Izááká : Odejdź od nas / ábowiemeś dáleko możniejƺy niż my.
Y odƺedł ſtámtąd Izáák / y rozbił námioty w dolinie Gerár / y mieƺkał tám.
Y kopał záśię Izáák ſtudnie wód / które byli wykopáli zá dni Abráhámá / ojcá jego / co je byli záſypáli Filiſtyni po śmierći Abráhámowej / y zwał je temiż imiony / któremi je był názwał oćiec jego.
Tedy kopáli ſłudzy Izáákowi w oney dolinie / y ználeſli tám ſtudnią wód żywych.
Lecż poſwárzyli śię páſterze Gerárſcy z páſterzámi Izáákowymi / mówiąc : Náƺá to wodá ; przeto názwał imię ſtudni oney / Heſek / iż śię ſwárzyli z nim o nię.
Potem wykopáli drugą ſtudnią / y ſwárzyli śię też o nię ; dla tegoż názwał imię jey Sydná.
Zátem przenióſł śię ſtámtąd / y wykopał drugą ſtudnią / o którą żádnego ſporu nie było ; y názwał imię jey Rechobot / y mówił : Oto / teraz rozƺerzył nas Pán / y urośliśmy ná źiemi.
Y wſtąpił ſtámtąd do Beerſeby.
Y ukázał mu śię Pán onejże nocy / mówiąc : Jam jeſt Bóg Abráhámá / ojcá twego / nie bój śię / bom Ja jeſt z tobą ; y będęć błogoſłáwił / y rozmnożę náśienie twoje / dla Abráhámá / ſługi mego.
Tedy tám zbudował ołtárz / y wzywał imieniá Páńſkiego / y rozbił tám námiot ſwój / támże też wykopáli ſłudzy Izáákowi ſtudnią.
Abimelech potem przyjechał do niego z Gerár / y Achuzát przyjáćiel jego / y Fikol / hetmán wojſká jego.
Do których rzekł Izáák : Przecżżeśćie przyjecháli do mnie / gdyż wy mnie máćie w nienáwiśći / y wypędźiliśćie mię od śiebie?
A oni odpowiedźieli : Obácżyliśmy to dobrze / że jeſt Pán z tobą / y rzekliśmy : Ucżyńmy teraz przyśięgę miedzy ſobą / miedzy námi y miedzy tobą / y poſtánowiemy przymierze z tobą ;
Abyś nam nic złego nie cżynił / jákośmy śię też ćiebie nie tykáli ; y jákośmyć tylko dobrze cżynili / á puśćiliśmy ćię w pokoju ; á tyś teraz błogoſłáwiony od Páná.
Tedy im ſpráwił ucżtę / á jedli y pili.
Potem wſtáwƺy bárzo ráno przyśięgli jeden drugiemu ; y wyprowádźił je Izáák / y odeƺli od niego w pokoju.
Y ſtáło śię onegoż dniá / przyƺli ſłudzy Izáákowi / y powiedźieli mu o ſtudni / którą wykopáli / mówiąc mu : Ználeſliśmy wodę.
Y názwał ją Sybá ; dla tegoż imię miáſtá onego jeſt názwáne Beerſebá áż do dniá dźiśiejƺego.
Potem Ezáw májąc cżterdźieśći lat / pojął ſobie zá żonę Judytę / córkę Beery Hetejcżyká / y Báſemát / córkę Eloná / Hetejcżyká.
Które śię bárzo náprzykrzáły Izáákowi y Rebece.
