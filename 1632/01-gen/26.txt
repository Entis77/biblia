Potem był głód ná ziemi / po onym głodzie pierwƺym / który był zá dni Abráhámowych ; Y poƺedł Izáák do Abimelechá / królá Filiſtyńſkiego / do Gerár.
Bo mu śię był ukázáł Pán y mówił : Nie zſtępuj do Egiptu / ále mieƺkáj w ziemi / o której Já powiem tobie.
Bądźże gośćiem w tej ziemi / á Já będę z tobą / y będęć błogoſłáwił ; ábowiem tobie y náśieniu twemu dám te wƺyſtkie kráiny / y utwierdzę przyśięgę / którąm przyśiągł Abráhámowi / ojcu twemu.
Y rozmnożę náśienie twoje jáko gwiázdy niebieſkie / á dám náśieniu twemu wƺyſtkie te kráiny : A będą błogoſłáwione w náśieniu twojim wƺyſtkie národy ziemi.
Przeto / iż Abráhám był poſłuƺny głoſowi mojemu / á ſtrzegł poſtánowieniá mego / przykázáń moich / uſtáw moich y práw moich.
Tedy Izáák mieƺkáł w Gerár.
Y pytáli śię mężowie onego miejſcá o żonie jego ; á on powiedziáł : Sioſtrá to mojá ; bo śię báł mówić : Żoná to mojá ; by go ſnáć nie zábili mężowie miejſcá tego dlá Rebeki / iż byłá piękná ná wejrzeniu.
Y ſtáło śię / gdy tám mieƺkáł przez nie máło dni / że wyglądáł Abimelech / król Filiſtyńſki / oknem / y ujrzáł / że Izáák żártowáł z Rebeką / żoną ſwą.
Tedy przyzwáł Abimelech do śiebie Izááká / y rzekł : Práwdziwieć to żoná twojá ; cżemużeś powiádáł / śioſtrá to mojá? Y odpowiedziáł mu Izáák : Iżem u śiebie mówił : Bym ſnáć nie umárł dlá niej.
Y rzekł Abimelech : Cóżeś nám to ucżynił? bez máłá ktokolwiek z ludu nie ſpáł z żoną twoją ; y przywiódłbyś był ná náſ grzech.
Rozkázáł tedy Abimelech wƺyſtkiemu ludowi mówiąc : Ktoby śię dotknął mężá tego / álbo żony jego / śmierćią umrze.
Tedy śiáł Izáák w onej ziemi / y zebráł roku onego ſto korcy / ábowiem błogoſłáwił mu Pán.
Y zbogáćił śię on mąż / á im dálej / tem więcej wzmágáł śię / áż uroſł wielce.
Y miáł ſtádá owiec / y ſtádá wołów / y cżeládzi doſyć ; przetoż mu zájrzeli Filiſtyńcżycy.
Y wƺyſtkie ſtudnie / które byli wykopáli ſłudzy ojcá jego / zá dni Abráhámá / ojcá jego / záſypáli Filiſtyńcżycy / y nápełnili je ziemią.
Y rzekł Abimelech do Izááká : Odejdź od náſ / ábowiemeś dáleko możniejƺy niż my.
Y odƺedł ſtámtąd Izáák / y rozbił námioty w dolinie Gerár / y mieƺkáł tám.
Y kopáł záśię Izáák ſtudnie wód / które byli wykopáli zá dni Abráhámá / ojcá jego / co je byli záſypáli Filiſtyni po śmierći Abráhámowej / y zwáł je temiż imiony / któremi je był názwáł ojćiec jego.
Tedy kopáli ſłudzy Izáákowi w onej dolinie / y ználeźli tám ſtudnią wód żywych.
Lecż poſwárzyli śię páſterze Gerárſcy z páſterzámi Izáákowymi / mówiąc : Náƺá to wodá ; przeto názwáł imię ſtudni onej / Heſek / iż śię ſwárzyli z nim o nię.
Potem wykopáli drugą ſtudnią / y ſwárzyli śię też o nię ; dlá tegoż názwáł imię jej Sydná.
Zátem przenióſł śię ſtámtąd / y wykopáł drugą ſtudnią / o którą żádnego ſporu nie było ; y názwáł imię jej Rechobot / y mówił : Oto / teráz rozƺerzył náſ Pán / y urośliśmy ná ziemi.
Y wſtąpił ſtámtąd do Beerſeby.
Y ukázáł mu śię Pán onejże nocy / mówiąc : Jám jeſt Bóg Abráhámá / ojcá twego / nie bój śię / bom Já jeſt z tobą ; y będęć błogoſłáwił / y rozmnożę náśienie twoje / dlá Abráhámá / ſługi mego.
Tedy tám zbudowáł ołtárz / y wzywáł imieniá Páńſkiego / y rozbił tám námiot ſwój / támże też wykopáli ſłudzy Izáákowi ſtudnią.
Abimelech potem przyjecháł do niego z Gerár / y Achuzát przyjáćiel jego / y Fikol / hetmán wojſká jego.
Do których rzekł Izáák : Przecżżeśćie przyjecháli do mnie / gdyż wy mnie máćie w nienáwiśći / y wypędziliśćie mię od śiebie?
A oni odpowiedzieli : Obácżyliśmy to dobrze / że jeſt Pán z tobą / y rzekliśmy : Ucżyńmy teráz przyśięgę między ſobą / między námi y między tobą / y poſtánowiemy przymierze z tobą ;
Abyś nám nic złego nie cżynił / jákośmy śię też ćiebie nie tykáli ; y jákośmyć tylko dobrze cżynili / á puśćiliśmy ćię w pokoju ; á tyś teráz błogoſłáwiony od Páná.
Tedy im ſpráwił ucżtę / á jedli y pili.
Potem wſtáwƺy bárdzo ráno przyśięgli jeden drugiemu ; y wyprowádził je Izáák / y odeƺli od niego w pokoju.
Y ſtáło śię onegoż dniá / przyƺli ſłudzy Izáákowi / y powiedzieli mu o ſtudni / którą wykopáli / mówiąc mu : Ználeźliśmy wodę.
Y názwáł ją Sybá ; dlá tegoż imię miáſtá onego jeſt názwáne Beerſebá áż do dniá dziśiejƺego.
Potem Ezáw májąc cżterdzieśći lát / pojął ſobie zá żonę Judytę / córkę Beery Hetejcżyká / y Báſemát / córkę Eloná / Hetejcżyká.
Które śię bárdzo náprzykrzáły Izáákowi y Rebece.
