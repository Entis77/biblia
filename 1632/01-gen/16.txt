Sáráj tedy / żoná Abrámowá / nie rodźiłá mu ; ále miáłá ſługę Egipcżánkę / której imię było Agár.
Y rzekłá Sáráj do Abrámá : Oto teráz zámknął mię Pán / ábym nie rodźiłá ; wnijdź / proƺę / do ſłużebnicy mojej / ázáli wżdy z niej będę miáłá dźiátki ; y uſłucháł Abrám głoſu Sárái.
Y wźięłá Sáráj / żoná Abrámowá / Agárę Egipcżánkę / ſłużebnicę ſwoję / po dźieśięćiu látách / jáko pocżął Abrám mieƺkáć w źiemi Chánánejſkiej ; y dáłá ją Abrámowi mężowi ſwemu zá żonę.
Tedy wƺedł do Agáry / y pocżęłá ; á widząc / że pocżęłá / wzgárdzoną byłá páni jej w ocżu jej.
Y rzekłá Sáráj do Abrámá : Krzywdy mojej tyś winien ; jámći dáłá ſłużebnicę moję ná łono twoje ; ále oná / widząc że pocżęłá / wzgárdźiłá mię w ocżách ſwych ; niech rozſądźi Pán między mną y między tobą.
Y rzekł Abrám do Sárái : Oto ſłużebnicá twojá w rękách twoich / cżyń z nią coć śię zdá nájlepƺego ; y trápiłá ją Sáráj / y ućiekłá od oblicżá jej.
Y ználázł ją Anioł Páńſki u źródłá wód ná puƺcży / nád źródłem / przy drodze Sur.
Y rzekł : Agáro / ſłużebnico Sárái / ſkąd idźieƺ? y dokąd idźieƺ? á oná odpowiedźiáłá : Od oblicżá Sárái / páni ſwej / já ućiekám.
Rzekł jej Anioł Páńſki : Wróć śię do páni ſwej / á ukorz śię pod ręce jej.
Rzekł jej záś Anioł Páńſki : Mnożąc rozmnożę náśienie twoje / iż nie będźie mogło być zlicżone przez mnóſtwo.
Potem jej rzekł Anioł Páńſki : Otoś ty pocżęłá / y porodźiƺ ſyná / á názowieƺ imię jego Iſmáel ; bo uſłyƺáł Pán utrápienie twoje.
Ten będźie ſrogim cżłowiekiem : ręká jego przećiwko wƺyſtkim / á ręká wƺyſtkich przećiwko jemu ; á przed oblicżem wƺyſtkiej bráći ſwej mieƺkáć będźie.
Y názwáłá imię Páná / który mówił do niej : Tyś Bóg widzący mię ; rzekłá bowiem : Izálim tu nie widźiáłá tyłu widzącego mię?
Przetoż názwáłá ſtudnią onę ſtudnią żywiącego / widzącego mię ; á táć jeſt między Kádeſ / y między Bárád.
Y urodźiłá Agár Abrámowi ſyná / y názwáł Abrám imię ſyná ſwego / którego urodźiłá Agár / Iſmáel.
A Abrám miáł ośmdźieśiąt lát / y ƺeść lát / gdy mu urodźiłá Agár Iſmáelá.
