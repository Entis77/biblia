A widząc Jákób / że było zboże w Egipćie / rzekł do ſynów ſwoich : Cżemuż śię oglądáćie jeden ná drugiego?
Y mówił im : Otom ſłyƺáł / że jeſt zboże w Egipćie. Jedźćież tám / á kupćie nám ſtámtąd / ábyśmy żywi byli / á nie pomárli.
Jecháło tedy dzieśięć bráći Józefowych kupowáć zboże / do Egiptu ;
Ale Benjáminá / brátá Józefowego / nie poſłáł Jákób z bráćią jego / bo mówił : By ſnáć nie przypádło náń co złego.
Y ƺli ſynowie Izráelowi poſpołu z innymi támże idącymi kupowáć zboże ; ábowiem był głód w ziemi Chánánejſkiej.
A Józef był przedniejƺym rządcą w onej ziemi ; onże ſprzedáwáł zbożá wƺyſtkiemu ludowi ziemi. A gdy przyƺli bráćiá Józefowi / kłániáli mu śię twárzą áż do ziemi.
A ujrzáwƺy Józef bráćią ſwą / poznáł je ; lecż ſtáwił śię im jáko obcy / y mówił do nich ſurowo / y rzekł do nich : Skądeśćie przyƺli? Y odpowiedzieli : Z ziemi Chánánejſkiej / ábyśmy nákupili żywnośći.
Tedy poznáł Józef bráćią ſwą ; ále go oni nie poználi.
Y wſpomniáł Józef ná ſny / które mu śię śniły o nich / y rzekł im : Szpiegowieśćie wy / á przyƺliśćie / ábyśćie przepátrzáli miejſcá nieobronne tej ziemi.
A oni mu odpowiedzieli : Nie ták / pánie mój ; ále ſłudzy twoi przyƺli / áby nákupili żywnośći.
Wƺyſcyśmy ſynowie jednego mężá ; ludzieśmy ƺcżerzy / á nie ſą ſłudzy twoi ƺpiegámi.
Ale on rzekł do nich : Nie ták / áleśćie nieobronne miejſcá tej ziemi przyƺli przepátrowáć.
Y rzekli : Dwánáśćie náſ bráći było ſług twoich / ſynów jednego mężá w ziemi Chánánejſkiej ; á oto / nájmłodƺy z ojcem náƺym teráz jeſt w domu / á jednego już niemáƺ.
Y rzekł im Józef : Toć jeſt com já wám powiedziáł / mówiąc : Szpiegowieśćie wy.
Przez to wáſ doświádcżę : żywie Fáráo / nie wynijdziećie ſtąd / áż mi tu przyjdzie brát wáƺ młodƺy.
Poślijćież jednego z wáſ / áby przywiódł brátá wáƺego / á wy w więzieniu będziećie / áżby były doświádcżone ſłowá wáƺe / jeſtli práwdá przy wáſ ; á jeźli nie / żywie Fáráo / żeśćie wy ƺpiegowie.
Tedy je dáł pod ſtráż do trzech dni.
Y mówił do nich Józef dniá trzećiego : Ucżyńćie ták / á żyć będziećie ; boć śię já boję Bogá.
Jeźliśćie ƺcżerzy / brát wáƺ jeden niech będzie okowány w więzieniu / gdzieśćie wy byli ; á wy jedźćie y odnieśćie zboże / ábyśćie odjęli głodowi domy wáƺe.
A brátá wáƺego młodƺego przywiedźćie do mnie / á ſpráwdzą śię ſłowá wáƺe / y nie pomrzećie. Y ucżynili ták.
Y mówili jeden do drugiego : Zápráwdęśmy zgrzeƺyli przećiwko brátu náƺemu ; bo widząc utrápienie duƺy jego / gdy śię nám modlił / nie wyſłucháliśmy go ; dlá tegoż przyƺedł ná náſ ten kłopot.
Odpowiedziáł im tedy Ruben / mówiąc : Izálim wám nie mówił temi ſłowy : Nie grzeƺćie przećiw pácholęćiu? á nieuſłucháliśćie ; otóż teráz krwi jego z rąk náƺych ƺukáją.
A oni nie wiedzieli / żeby rozumiáł Józef ; bo tłumácż był między nimi.
Odwróćiwƺy śię tedy od nich Józef / płákáł ; á obróćiwƺy śię do nich / mówił z nimi / y wziąwƺy od nich Symeoná / związáł go przed ocżymá ich.
Y rozkázáł Józef / áby nápełniono wory ich zbożem / y wrócono pieniądze ich káżdemu do woru jego / y żeby im dáno żywnośći ná drogę ; y ucżyniono ták.
Tedy oni włożywƺy zbożá ſwoje ná oſły ſwe / odjecháli ſtámtąd.
Y rozwiązáwƺy jeden z nich wór ſwój / áby dáł obrok oſłowi ſwemu w goſpodzie / ujrzáł pieniądze ſwoje / które były ná wierzchu w worze jego.
Y rzekł do bráći ſwej : Wrócono mi pieniądze moje / á oto / ſą w worze moim. Tedy im upádło ſerce / y zdumieli śię / jeden do drugiego mówiąc : Cóż nám to Bóg ucżynił?
Zátem przyƺli do Jákóbá / ojcá ſwego / do ziemi Chánánejſkiej / y powiedzieli mu wƺyſtko / co śię im przydáło / mówiąc :
Mówił z námi on mąż / pán onej ziemi / ſurowo / y udáł náſ zá ƺpiegi ziemi ;
A myśmy mu rzekli : Szcżerzyśmy / nie byliśmy ƺpiegámi :
Dwánáśćie náſ było bráći ſynów ojcá náƺego ; jednego już niemáƺ / á młodƺy teráz jeſt z ojcem náƺym w ziemi Chánánejſkiej.
Y mówił do náſ mąż on / pán onej ziemi : Po tem poznám / żeśćie ƺcżerzy ; brátá wáƺego jednego zoſtáwćie u mnie / á zboże dlá odjęćiá głodowi domów wáƺych weźmijćie á idźćie ;
Potem przywiedźćie brátá wáƺego młodƺego do mnie / ábym poznáł / żeśćie wy nie ƺpiegowie / ále ƺcżerzy ; tedy wám wrócę brátá wáƺego / á w tej ziemi hándlowáć będziećie.
Y ſtáło śię / gdy wypróżniáli wory ſwoje / á oto / káżdy ználázł węzeł pieniędzy ſwych w worze ſwoim ; á obácżywƺy węzły z pieniędzmi ſwymi / oni y ojćiec ich / polękáli śię.
Y rzekł im Jákób / ojćiec ich : Ośieroćiliśćie mię. Józefá niemáƺ / y Symeoná niemáƺ / á Benjáminá weźmiećie ; ná mię śię to wƺyſtko złe zwáliło.
Y rzekł Ruben do ojcá ſwego / mówiąc : Dwóch ſynów moich zábij / jeźlić go záś nie przywiodę ; dáj go do ręki mojej / á já go tobie przywrócę.
Ale on rzekł : Nie pójdzie ſyn mój z wámi / gdyż brát jego umárł / á on ſám tylko zoſtáł ; á jeźliby náń przypádło co złego ná drodze / którą pójdziećie / tedy doprowádzićie ſędziwość moję z żáłośćią do grobu.
