A widząc Jákób / że było zboże w Egipćie / rzekł do ſynów ſwojich : Cżemuż śię oglądáćie jeden ná drugiego?
Y mówił im : Otom ſłyƺał / że jeſt zboże w Egipćie. Jedźćież tám / á kupćie nam ſtámtąd / ábyſmy żywi byli / á nie pomárli.
Jecháło tedy dźieśięć bráći Józefowych kupowáć zboże / do Egiptu ;
Ale Benjáminá / brátá Józefowego / nie poſłał Jákób z bráćią jego / bo mówił : By ſnadź nie przypádło nań co złego.
Y ƺli ſynowie Izráelowi poſpołu z innymi támże idącymi kupowáć zboże ; ábowiem był głód w źiemi Chánánejſkiej.
A Józef był przedniejƺym rządcą w oney źiemi ; onże ſprzedáwał zbożá wƺyſtkiemu ludowi źiemi. A gdy przyƺli bráćia Józefowi / kłániáli mu śię twárzą áż do źiemi.
A ujrzáwƺy Józef bráćią ſwą / poznał je ; lecż ſtáwił śię im jáko obcy / y mówił do nich ſurowo / y rzekł do nich : Skądeśćie przyƺli? Y odpowiedźieli : Z źiemi Chánánejſkiej / ábyſmy nákupili żywnośći.
Tedy poznał Józef bráćią ſwą ; ále go oni nie poználi.
Y wſpomniał Józef ná ſny / które mu śię śniły o nich / y rzekł im : Szpiegowieśćie wy / á przyƺliśćie / ábyśćie przepátrzáli miejſcá nieobronne tey źiemi.
A oni mu odpowiedźieli : Nie ták / pánie mój ; ále ſłudzy twoi przyƺli / áby nákupili żywnośći.
Wƺyſcyśmy ſynowie jednego mężá ; ludźieśmy ƺcżerzy / á nie ſą ſłudzy twoi ƺpiegámi.
Ale on rzekł do nich : Nie ták / áleśćie nieobronne miejſcá tey źiemi przyƺli przepátrowáć.
Y rzekli : Dwánáśćie nas bráći było ſług twojich / ſynów jednego mężá w źiemi Chánánejſkiej ; á oto / nájmłodƺy z ojcem náƺym teraz jeſt w domu / á jednego już niemáƺ.
Y rzekł im Józef : Toć jeſt com ja wam powiedźiał / mówiąc : Szpiegowieśćie wy.
Przez to was doświádcżę : żywie Fáráo / nie wynijdźiećie ſtąd / áż mi tu przydźie brát wáƺ młodƺy.
Poślijćież jednego z was / áby przywiódł brátá wáƺego / á wy w więźieniu będźiećie / áżby były doświádcżone ſłowá wáƺe / jeſtli práwdá przy was ; á jeſli nie / żywie Fáráo / żeśćie wy ƺpiegowie.
Tedy je dał pod ſtráż do trzech dni.
Y mówił do nich Józef dniá trzećiego : Ucżyńćie ták / á żyć będźiećie ; boć śię ja boję Bogá.
Jeſliśćie ƺcżerzy / brát wáƺ jeden niech będźie okowány w więźieniu / gdźieśćie wy byli ; á wy jedźćie y odnieśćie zboże / ábyśćie odjęli głodowi domy wáƺe.
A brátá wáƺego młodƺego przywiedźćie do mnie / á ſpráwdzą śię ſłowá wáƺe / y nie pomrzećie. Y ucżynili ták.
Y mówili jeden do drugiego : Zápráwdęśmy zgrzeƺyli przećiwko brátu náƺemu ; bo widząc utrápienie duƺy jego / gdy śię nam modlił / nie wyſłucháliśmy go ; dla tegoż przyƺedł ná nas ten kłopot.
Odpowiedźiał im tedy Ruben / mówiąc : Izálim wam nie mówił temi ſłowy : Nie grzeƺćie przećiw pácholęćiu? á nieuſłucháliśćie ; otóż teraz krwi jego z rąk náƺych ƺukáją.
A oni nie wiedźieli / żeby rozumiał Józef ; bo tłumácż był miedzy nimi.
Odwróćiwƺy śię tedy od nich Józef / płákał ; á obróćiwƺy śię do nich / mówił z nimi / y wźiąwƺy od nich Symeoná / związał go przed ocżymá ich.
Y rozkázał Józef / áby nápełniono wory ich zbożem / y wrócono pieniądze ich káżdemu do woru jego / y żeby im dáno żywnośći ná drogę ; y ucżyniono ták.
Tedy oni włożywƺy zbożá ſwoje ná oſły ſwe / odjecháli ſtámtąd.
Y rozwiązáwƺy jeden z nich wór ſwój / áby dał obrok oſłowi ſwemu w goſpodźie / ujrzał pieniądze ſwoje / które były ná wierzchu w worze jego.
Y rzekł do bráći ſwej : Wrócono mi pieniądze moje / á oto / ſą w worze moim. Tedy im upádło ſerce / y zdumieli śię / jeden do drugiego mówiąc : Cóż nam to Bóg ucżynił?
Zátem przyƺli do Jákóbá / ojcá ſwego / do źiemi Chánánejſkiej / y powiedźieli mu wƺyſtko / co śię im przydáło / mówiąc :
Mówił z námi on mąż / pán oney źiemi / ſurowo / y udał nas zá ƺpiegi źiemi ;
A myśmy mu rzekli : Szcżerzyśmy / nie byliśmy ƺpiegámi :
Dwánáśćie nas było bráći ſynów ojcá náƺego ; jednego już niemáƺ / á młodƺy teraz jeſt z ojcem náƺym w źiemi Chánánejſkiej.
Y mówił do nas mąż on / pán oney źiemi : Po tym poznam / żeśćie ƺcżerzy ; brátá wáƺego jednego zoſtáwćie u mnie / á zboże dla odjęćiá głodowi domów wáƺych weźmijćie á idźćie ;
Potem przywiedźćie brátá wáƺego młodƺego do mnie / ábym poznał / żeśćie wy nie ƺpiegowie / ále ƺcżerzy ; tedy wam wrócę brátá wáƺego / á w tey źiemi hándlowáć będźiećie.
Y ſtáło śię / gdy wypróżniáli wory ſwoje / á oto / káżdy ználázł węzeł pieniędzy ſwych w worze ſwoim ; á obácżywƺy węzły z pieniędzmi ſwymi / oni y oćiec ich / polękáli śię.
Y rzekł im Jákób / oćiec ich : Ośieroćiliśćie mię. Józefá niemáƺ / y Symeoná niemáƺ / á Benjáminá weźmiećie ; ná mię śię to wƺyſtko złe zwáliło.
Y rzekł Ruben do ojcá ſwego / mówiąc : Dwóch ſynów mojich zábij / jeſlić go záś nie przywiodę ; dáj go do ręki mojey / á ja go tobie przywrócę.
Ale on rzekł : Nie pójdźie ſyn mój z wámi / gdyż brát jego umárł / á on ſám tylko zoſtał ; á jeſliby nań przypádło co złego ná drodze / którą pójdźiećie / tedy doprowádźićie ſędźiwość moję z żáłośćią do grobu.
