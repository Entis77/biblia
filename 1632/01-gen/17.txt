A gdy już było Abrámowi dziewięćdzieśiąt lát y dziewięć lát / ukázáł śię Pán Abrámowi / y rzekł do niego : Jám jeſt Bóg Wƺechmogący ; chodź przed oblicżem mojem / á bądź doſkonáły.
A ucżynię przymierze moje / między mną y między tobą / y rozmnożę ćię bárdzo obfićie.
Tedy upádł Abrám ná oblicże ſwoje / y rzekł do niego Bóg / mówiąc :
Jám jeſt / oto ſtánowię przymierze moje z tobą / y będzieƺ ojcem wielu národów.
Y nie będzie zwáne dálej imię twoje Abrám ; ále będzie imię twoje Abráhám ; ábowiem ojcem wielu národów poſtánowiłem ćię.
A rozmnożę ćię bárdzo / y rozkrzewię ćię w národy / y królowie z ćiebie wynijdą.
Y utwierdzę przymierze moje między mną / y między tobą / y między náśieniem twojim po tobie / w národziech ich umową wiecżną ; żebym ći był Bogiem y náśieniu twemu po tobie.
Dám też tobie / y náśieniu twemu po tobie ziemię / w której teráz jeſteś gośćiem ; wƺyſtkę ziemię Chánánejſką w ośiádłość wiecżną / y będę Bogiem ich.
Nád to rzekł Bóg Abráhámowi : Ty też przymierzá mego przeſtrzegáć będzieƺ / ty y náśienie twoje po tobie / w národziech ſwoich.
A toć jeſt przymierze moje / które záchowywáć będziećie / między mną / y między wámi / y między náśieniem twojim po tobie / áby był obrzezány między wámi káżdy mężcżyzná.
Obrzeżećie tedy ćiáło nieobrzeſki wáƺej ; á to będzie znákiem przymierzá między mną / y między wámi.
Syn ośmiu dni / będzie obrzezány między wámi káżdy mężcżyzná w národziech wáƺych / ták domá národzony / jáko y kupiony zá pieniądze / od jákiegożkolwiek cudzoziemcá / któryby nie był z náśieniá twego.
Koniecżnie obrzezány będzie / urodzony w domu twoim / y kupiony zá pieniądze twoje ; á będzie przymierze moje ná ćiele wáƺem / ná przymierze wiecżne.
A nie obrzezány mężcżyzná / któregoby nie było obrzezáne ćiáło nieobrzeſki jego / będzie wytráconá duƺá oná z ludu ſwego ; ábowiem zgwáłćił przymierze moje.
Potem rzekł Bóg do Abráhámá : Sárái / żony twojej / nie będzieƺ zwáł imieniá jej Sáráj / ále Sárá będzie imię jej.
Y będę jej błogoſłáwił / á dám ći z niej ſyná ; będę jej błogoſłáwił / y będzie rozmnożoná w národy / á królowie národów z niej wynijdą.
Tedy Abráhám pádł ná oblicże ſwoje / y rozśmiáł śię / á mówił w ſercu ſwem : Záż cżłowiekowi ſtoletniemu urodzi śię ſyn? y ázáż Sárá w dziewięćiudzieśiąt lát porodzi?
Y rzekł Abráhám do Bogá : O by tylko Iſmáel żył przed oblicżem twojim!
Y rzekł Bóg : Záiſte / Sárá / żoná twojá / urodzi tobie ſyná / y názowieƺ imię jego Izáák ; y utwierdzę przymierze moje z nim / umową wiecżną / y z náśieniem jego po nim.
O Iſmáelá też wyſłucháłem ćię ; oto / błogoſłáwiłem mu / y rozrodzę go / y rozmnożę go bárdzo wielce. Dwánáśćie kśiążąt ſpłodzi / y rozkrzewię go w náród wielki.
Ale przymierze moje utwierdzę z Izáákiem / którego tobie urodzi Sárá / o tym cżáśie w roku drugim.
A przeſtáwƺy mówić z nim / odƺedł Bóg od Abráhámá.
Tedy wziął Abráhám Iſmáelá / ſyná ſwego / y wƺyſtkie urodzone w domu ſwym / y wƺyſtkie kupione zá pieniądze / káżdego mężcżyznę / z mężów domu Abráhámowego / y obrzezáł ćiáło nieobrzeſki ich / onegożto dniá / jáko mówił z nim Bóg.
A Abráhámowi było dziewięćdzieśiąt lát y dziewięć / gdy obrzezáne było ćiáło nieobrzeſki jego.
A Iſmáelowi ſynowi jego było trzynáśćie lát / gdy obrzezáne było ćiáło nieobrzeſki jego.
Tegoż dniá obrzezány jeſt Abráhám / y Iſmáel / ſyn jego.
Y wƺyſcy mężowie domu jego / urodzeni w domu / y kupieni zá pieniądze od cudzoziemców / obrzezáni ſą z nim.
