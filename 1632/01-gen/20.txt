Y ruƺył śię ſtámtąd Abráhám do źiemi południowej / á mieƺkáł między Kádeſ y między Sur / y był gośćiem w Gerár.
Tám powiedźiáł Abráhám o Sárze / żonie ſwej : Sioſtrá mojá jeſt ; przetoż poſłáł Abimelech / król Geráry / y wźiął Sárę.
Ale Bóg przyƺedł do Abimelechá we śnie w nocy / y rzekł mu : Oto ty umrzeƺ dla niewiáſty / którąś wźiął / bo oná má mężá.
Ale Abimelech nie przybliżył śię był do niej / y rzekł : Pánie / izáli też lud ſpráwiedliwy zábijeƺ?
Azáż mi on ſám nie powiádáł / śioſtrá mojá jeſt? á oná też ſámá nie mówiłá / brát mój jeſt? w proſtośći ſercá mojego / y w niewinnośći rąk moich ucżyniłem to.
Tedy mu rzekł Bóg we śnie : Wiemći ja / żeś to w proſtośći ſercá ſwego ucżynił ; y dla tegom ćię záwśćiągnął / ábyś nie zgrzeƺył przećiwko mnie / y nie dopuśćiłem ći / ábyś śię jey dotknął.
Teraz tedy wróć żonę mężowi / bo prorokiem jeſt ; y będźie śię modlił zá ćię / á będźieƺ żył ; á jeſliż jey nie wróćiƺ / wiedz / iż śmierćią umrzeƺ / ty / y wƺyſtko / co twego jeſt.
Tedy Abimelech wſtáwƺy ráno / zwołáł wƺyſtkich ſług ſwoich / y opowiedźiáł im to wƺyſtko ; co uſłyƺáwƺy / polękáli śię mężowie oni bárdzo.
Potem wezwáł Abimelech Abráhámá / y rzekł mu : Coś nám ucżynił? á com zgrzeƺył przećiwko tobie? iżeś przywiódł ná mię y ná króleſtwo moje grzech wielki? ucżyniłeś mi / cżegoś cżynić nie miał.
Y rzekł powtóre Abimelech do Abráhámá : Cóżeś upátrowáł / żeś tę rzecż ucżynił?
Y odpowiedźiáł Abráhám : Myśliłem ſobie : Podobno niemáƺ bojáźni Bożej ná tem miejſcu / y zábiją mię dla żony mojey.
A wƺákże práwdźiwie śioſtrá mojá jeſt / córká ojcá mego / choć nie córká mátki mojey ; pojąłem ją zá żonę.
Y ſtáło śię / gdy mię wypráwił Bóg ná pielgrzymowánie z domu ojcá mego / żem rzekł do niej : To miłośierdźie twoje będźie / które ucżyniƺ ze mną : Ná káżdem miejſcu / do którego przyjdźiemy / powieƺ o mnie : Brát to mój jeſt.
Tedy nábráwƺy Abimelech owiec / y wołów / y ſług / y ſłużebnic / dáł Abráhámowi / y wróćił mu Sárę / żonę jego.
Y rzekł Abimelech : Oto źiemiá mojá przed oblicżem twojim ; gdźieć śię kolwiek podobá / mieƺkáj.
A do Sáry rzekł : Otom dáł tyśiąc ſrebrników brátu twemu / onći jeſt záſłoną ocżu twoich u wƺyſtkich / którzy ſą z tobą ; á tem wƺyſtkiem Sárá wyucżoná byłá.
Y modlił śię Abráhám Bogu / á uzdrowił Bóg Abimelechá / y żonę jego / y ſłużebnice jego / y rodźiły.
Záwárł bowiem był Pán cále káżdy żywot domu Abimelechowego dla Sáry / żony Abráhámowej.
