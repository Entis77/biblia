Y mieƺkał Jákób w źiemi / gdźie przychodniem był oćiec jego / w źiemi Chánánejſkiej.
Teć ſą pokolenia Jákóbowe : Józef gdy miał śiedmnaśćie lat / páſł z bráćią ſwoją trzody ( będąc pácholęćiem ) z ſynámi Báli / y z ſynámi Zelfy / żon ojcá ſwego ; y odnośił Józef ſłáwę ich złą do ojcá ich.
A Izráel miłował Józefá nád wƺyſtkie ſyny ſwe / iż mu śię był w ſtárośći jego urodźił / y ſpráwił mu ſuknią rozmáitych fárb.
A widząc bráćia jego / że go miłował oćiec ich nád wƺyſtkę bráćią jego / nienáwidźili go / y nie mogli nic łáſkáwie z nim mówić.
Y śnił śię Józefowi ſen / á gdy go powiedźiał bráći ſwej / tym go więcey mieli w nienáwiśći.
Bo rzekł do nich : Słuchájćie proƺę ſnu tego / który mi śię śnił.
Otośmy wiązáli ſnopy ná polu / á oto / wſtáwƺy ſnop mój ſtánął / á około niego ſtojące ſnopy wáƺe kłániáły śię ſnopowi mojemu.
Y odpowiedźieli mu bráćia jego : Izali królowáć będźieƺ nád námi? y pánowáć nam będźieƺ? ſtądże go jeƺcże mieli w więkƺej nienáwiśći / dla ſnów jego / y dla ſłów jego.
Śnił mu śię też jeƺcże drugi ſen / y powiedźiał go bráći ſwej / mówiąc : Oto mi śię znowu śnił ſen : A ono ſłońce y mieśiąc / y jedenaśćie gwiázd kłániáło mi śię.
Y powiedźiał ojcu ſwemu y bráći ſwej / y gromił go oćiec jego y mówił mu : Cóż to zá ſen / coć śię śnił? Izali przyjdźiemy / ja y mátká twojá z bráćią twoją / ábyśmyć śię kłániáli áż do źiemi.
Y nienáwidźili go bráćia jego ; ále oćiec jego pilnie uwáżał tę rzecż.
Y odeƺli bráćia jego / áby páśli trzody ojcá ſwego w Sychem.
Tedy rzekł Izráel do Józefá : Izali bráćia twoi nie páſą w Sychem? pójdźże / á poślę ćię do nich : á on odpowiedźiał : Otom ja.
Rzekł mu tedy : Idźże teraz / á dowiedz śię / jáko śię máją bráćia twoi / y co śię dźieje z trzodámi / y dáƺ mi znáć. Wyſłał go tedy z doliny Hebron / y przyƺedł do Sychem.
Y nádƺedł go niektóry mąż / á on śię błąkał po polu ; y pytał go mąż on mówiąc :
Cżegóż ƺukáƺ? A on odpowiedźiał : Bráći mojey ƺukám ; powiedz mi proƺę / gdźie oni páſą.
Tedy rzekł on cżłowiek : Odeƺli ſtąd ; bom ſłyƺał / gdy mówili : Pójdźmy do Dotáin. Y ƺedł Józef zá bráćią ſwoją / á ználázł je w Dotáin.
Y ujrzeli go z dáleká / á pierwej niż do nich przyƺedł / rádźili o nim / áby go zábili.
Y mówili jeden do drugiego : Onoż miſtrz on ſnów idźie.
Teraz tedy pójdźćie / á zábijmy go / y wrzućmy go w jáką ſtudnią / á rzecżemy : Zły go zwierz pożárł ; á ták obácżymy / ná co mu wynijdą ſny jego.
Co gdy uſłyƺał Ruben / chćiał go wybáwić z rąk ich / mówiąc : Nie zábijájmy go.
Nád to rzekł do nich Ruben : Nie wylewájćie krwi / ále wrzuććie go w tę ſtudnią / która jeſt ná puƺcży / á ręki nie śćiągájćie nań. A to mówił / áby go wybáwił z rąk ich / y powróćił go ojcu ſwemu.
Y ſtáło śię / gdy przyƺedł Józef do bráći ſwej / zewlekli go z ſukni jego / z ſukni rozmáitych fárb / którą miał ná ſobie.
A porwáwƺy go / wrzućili go w ſtudnią / która ſtudniá byłá cżcżá / y nie było w niej wody.
A uśiádƺy / áby jedli chleb / podnieśli ocży ſwe / y ujrzeli / á ono pocżet Iſmáelitów / idących z Gáláád ; á wielbłądy ich nioſły korzenie / y kádźidło / y myrrę / á ƺły / áby to zánioſły do Egiptu.
Tedy rzekł Judás do bráći ſwej : Cóż zá pożytek / choćbyſmy zábili brátá náƺego / y zátáili krwi jego?
Pójdźćie / á przedájmy go Iſmáelitom / á ręká náƺá niech nie będźie ná nim ; brát bowiem náƺ / y ćiáło náƺe jeſt ; y uſłucháli go bráćia jego.
A gdy mijáli oni mężowie / Mádyjáńſcy kupcy / tedy wyćiągnęli / y wyjęli Józefá z ſtudni / y ſprzedáli Józefá Iſmáelitom zá dwádźieśćiá ſrebrników / którzy záprowádźili Józefá do Egiptu.
Tedy śię wróćił Ruben do oney ſtudni / á oto / już nie było Józefá w ſtudni ; y rozdárł ƺáty ſwoje.
A wróćiwƺy śię do bráći ſwej / rzekł : Pácholęćiá niemáƺ / á ja dokąd? ja dokąd pójdę?
Tedy wźięli ſuknią Józefowe / y zábili kozłá / á umácżáli ſuknią we krwi.
Y poſłáli onę ſuknią rozmáitych fárb / áby ją zánieśiono do ojcá jego / y rzekli : Tęśmy ználeſli / poznájże teraz / jeſli to ſukniá ſyná twego / cżyli nie.
A poznáwƺy ją / rzekł : Sukniá jeſt ſyná mego ; zwierz zły pożárł go ; koniecżnie rozƺárpány jeſt Józef.
Tedy rozdárƺy Jákób ƺáty ſwe / włożył wór ná biodrá ſwoje / żáłując ſyná ſwego przez wiele dni.
Y zeƺli śię wƺyſcy ſynowie jego / y wƺyſtkie córki jego / áby go ćieƺyli / lecż nie dał śię ćieƺyć / ále mówił : Zápráwdę zſtąpię zá ſynem moim do grobu ; y płákał go oćiec jego.
A Mádyjáńcżycy ſprzedáli Józefá do Egiptu Potyfárowi / dworzáninowi Fáráonowemu / hetmánowi żołnierſtwá.
