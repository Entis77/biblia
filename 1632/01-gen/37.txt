Y mieƺkáł Jákób w ziemi / gdzie przychodniem był ojćiec jego / w ziemi Chánánejſkiej.
Teć ſą pokoleniá Jákóbowe : Józef gdy miáł śiedmnáśćie lát / páſł z bráćią ſwoją trzody ( będąc pácholęćiem ) z ſynámi Báli / y z ſynámi Zelfy / żon ojcá ſwego ; y odnośił Józef ſłáwę ich złą do ojcá ich.
A Izráel miłowáł Józefá nád wƺyſtkie ſyny ſwe / iż mu śię był w ſtárośći jego urodził / y ſpráwił mu ſuknią rozmáitych fárb.
A widząc bráćiá jego / że go miłowáł ojćiec ich nád wƺyſtkę bráćią jego / nienáwidzili go / y nie mogli nic łáſkáwie z nim mówić.
Y śnił śię Józefowi ſen / á gdy go powiedziáł bráći ſwej / tem go więcej mieli w nienáwiśći.
Bo rzekł do nich : Słuchájćie proƺę ſnu tego / który mi śię śnił.
Otośmy wiązáli ſnopy ná polu / á oto / wſtáwƺy ſnop mój ſtánął / á około niego ſtojące ſnopy wáƺe kłániáły śię ſnopowi mojemu.
Y odpowiedzieli mu bráćiá jego : Izáli królowáć będzieƺ nád námi? y pánowáć nám będzieƺ? ſtądże go jeƺcże mieli w więkƺej nienáwiśći / dlá ſnów jego / y dlá ſłów jego.
Śnił mu śię też jeƺcże drugi ſen / y powiedziáł go bráći ſwej / mówiąc : Oto mi śię znowu śnił ſen : A ono ſłońce y mieśiąc / y jedenáśćie gwiázd kłániáło mi śię.
Y powiedziáł ojcu ſwemu y bráći ſwej / y gromił go ojćiec jego y mówił mu : Cóż to zá ſen / coć śię śnił? Izáli przyjdziemy / já y mátká twojá z bráćią twoją / ábyśmyć śię kłániáli áż do ziemi.
Y nienáwidzili go bráćiá jego ; ále ojćiec jego pilnie uwáżáł tę rzecż.
Y odeƺli bráćiá jego / áby páśli trzody ojcá ſwego w Sychem.
Tedy rzekł Izráel do Józefá : Izáli bráćiá twoi nie páſą w Sychem? pójdźże / á poślę ćię do nich : á on odpowiedziáł : Otom já.
Rzekł mu tedy : Idźże teráz / á dowiedz śię / jáko śię máją bráćiá twoi / y co śię dzieje z trzodámi / y dáƺ mi znáć. Wyſłáł go tedy z doliny Hebron / y przyƺedł do Sychem.
Y nádƺedł go niektóry mąż / á on śię błąkáł po polu ; y pytáł go mąż on mówiąc :
Cżegóż ƺukáƺ? A on odpowiedziáł : Bráći mojej ƺukám ; powiedz mi proƺę / gdzie oni páſą.
Tedy rzekł on cżłowiek : Odeƺli ſtąd ; bom ſłyƺáł / gdy mówili : Pójdźmy do Dotáin. Y ƺedł Józef zá bráćią ſwoją / á ználázł je w Dotáin.
Y ujrzeli go z dáleká / á pierwej niż do nich przyƺedł / rádzili o nim / áby go zábili.
Y mówili jeden do drugiego : Onoż miſtrz on ſnów idzie.
Teráz tedy pójdźćie / á zábijmy go / y wrzućmy go w jáką ſtudnią / á rzecżemy : Zły go zwierz pożárł ; á ták obácżymy / ná co mu wynijdą ſny jego.
Co gdy uſłyƺáł Ruben / chćiáł go wybáwić z rąk ich / mówiąc : Nie zábijájmy go.
Nád to rzekł do nich Ruben : Nie wylewájćie krwi / ále wrzuććie go w tę ſtudnią / którá jeſt ná puƺcży / á ręki nie śćiągájćie náń. A to mówił / áby go wybáwił z rąk ich / y powróćił go ojcu ſwemu.
Y ſtáło śię / gdy przyƺedł Józef do bráći ſwej / zewlekli go z ſukni jego / z ſukni rozmáitych fárb / którą miáł ná ſobie.
A porwáwƺy go / wrzućili go w ſtudnią / którá ſtudniá byłá cżcżá / y nie było w niej wody.
A uśiádłƺy / áby jedli chleb / podnieśli ocży ſwe / y ujrzeli / á ono pocżet Iſmáelitów / idących z Gáláád ; á wielbłądy ich nioſły korzenie / y kádzidło / y myrrę / á ƺły / áby to zánioſły do Egiptu.
Tedy rzekł Judáſ do bráći ſwej : Cóż zá pożytek / choćbyśmy zábili brátá náƺego / y zátáili krwi jego?
Pójdźćie / á przedájmy go Iſmáelitom / á ręká náƺá niech nie będzie ná nim ; brát bowiem náƺ / y ćiáło náƺe jeſt ; y uſłucháli go bráćiá jego.
A gdy mijáli oni mężowie / Mádyjáńſcy kupcy / tedy wyćiągnęli / y wyjęli Józefá z ſtudni / y ſprzedáli Józefá Iſmáelitom zá dwádzieśćiá ſrebrników / którzy záprowádzili Józefá do Egiptu.
Tedy śię wróćił Ruben do onej ſtudni / á oto / już nie było Józefá w ſtudni ; y rozdárł ƺáty ſwoje.
A wróćiwƺy śię do bráći ſwej / rzekł : Pácholęćiá niemáƺ / á já dokąd? já dokąd pójdę?
Tedy wzięli ſuknią Józefowe / y zábili kozłá / á umácżáli ſuknią we krwi.
Y poſłáli onę ſuknią rozmáitych fárb / áby ją zánieśiono do ojcá jego / y rzekli : Tęśmy ználeźli / poznájże teráz / jeźli to ſukniá ſyná twego / cżyli nie.
A poznáwƺy ją / rzekł : Sukniá jeſt ſyná mego ; zwierz zły pożárł go ; koniecżnie rozƺárpány jeſt Józef.
Tedy rozdárłƺy Jákób ƺáty ſwe / włożył wór ná biodrá ſwoje / żáłując ſyná ſwego przez wiele dni.
Y zeƺli śię wƺyſcy ſynowie jego / y wƺyſtkie córki jego / áby go ćieƺyli / lecż nie dáł śię ćieƺyć / ále mówił : Zápráwdę zſtąpię zá ſynem moim do grobu ; y płákáł go ojćiec jego.
A Mádyjáńcżycy ſprzedáli Józefá do Egiptu Potyfárowi / dworzáninowi Fáráonowemu / hetmánowi żołnierſtwá.
