A Jákób też poƺedł w drogę ſwoję y potkáli śię z nim Aniołowie Boży.
Y rzekł Jákób ujrzáwƺy je : Obóz to Boży ; y názwáł imię miejſcá onego Máhánáim.
Potem poſłáł Jákób poſły przed ſobą do Ezáwá / brátá ſwego / do ziemi Seir / do kráiny Edomſkiej.
Y rozkázáł im mówiąc : Ták rzecżećie do páná mego Ezáwá : To mówi ſługá twój Jákób : U Lábáná byłem gośćiem / y mieƺkáłem z nim áż do tego cżáſu.
A mám woły y oſły / owce / y ſługi / y ſłużebnice / á poſyłám odpowiedzieć pánu memu / żebym ználázł łáſkę w ocżách twoich.
Y wróćili śię poſłowie do Jákóbá / mówiąc : Przyƺliśmy do brátá twego Ezáwá / który też idzie przećiwko tobie / á cżtery ſtá mężów z nim.
Y zląkł śię Jákób bárdzo á ſtrwożył śię ; y rozdzielił lud / który z nim był / y owce / y woły / y wielbłądy / ná dwá hufce ;
Y rzekł : Jeźliby przyƺedł Ezáw do jednego hufcá / á poráziłby go / tedy hufiec / który pozoſtánie / będzie záchowány.
Y rzekł Jákób : Boże ojcá mego Abráhámá / y Boże ojcá mego Izááká / Pánie / któryś do mnie rzekł : Wróć śię do ziemi twojej / y do rodziny twojej / á ucżynięć dobrze.
Mniejƺym jeſt niż wƺyſtkie zmiłowániá / y niż wƺyſtká práwdá / którąś ucżynił z ſługą ſwym. Abowiem tylko o láſce mojej przeƺedłem ten Jordán / á teráz mám dwá hufce.
Wyrwij mię proƺę z ręki brátá mego / z ręki Ezáwá ; boć śię go boję / by ſnáć przyƺedłƺy nie zábił mię / y mátki z ſynámi.
Wƺákeś rzekł : Dobrze cżyniąc będęć dobrze cżynił / á rozmnożę náśienie twoje jáko piáſek morſki / który zlicżon być nie może / dlá mnóſtwá.
Y przenocowáł tám onej nocy / y wziął z tego / co miáł przy ręku / upominek dlá Ezáwá / brátá ſwego.
To jeſt kóz dwieśćie / y kozłów dwádzieśćiá / owiec dwieśćie / y báránów dwádzieśćiá.
Wielbłądzic odchowujących młode / ze źrebięty ich / trzydzieśći / krów cżterdzieśći / y wołów dzieśięć / dwádzieśćiá oślic / y dzieśięć ośląt.
Y oddáł je w ręce ſług ſwoich / káżde ſtádo z oſobná / y rzekł do ſług ſwoich : Idźćie przedemną / á plác ucżyńćie między ſtádem á między ſtádem.
Y rozkázáł pierwƺemu / mówiąc : Gdy śię potká z tobą Ezáw / brát mój / á ſpytá ćię / mówiąc : Cżyjeś ty? y dokąd idzieƺ? á cżyje to ſtádo przed tobą?
Tedy powieƺ : Sługi twego Jákóbá jeſt to upominek / poſłány pánu memu Ezáwowi ; á oto / y ſám idzie zá námi.
Tákże też rozkázáł drugiemu / y trzećiemu / y wƺyſtkim idącym zá temi ſtády / mówiąc : Temiż ſłowy mówćie do Ezáwá / gdy go potkáćie.
Powiećie mu też : Oto / ſługá twój Jákób idzie zá námi ; mówił bowiem : Ubłágám oblicże jego upominkiem / który idzie przedemną / á potem ujrzę oblicże jego ; owá mię ſnáć w łáſkę przyjmie.
Y poƺedł w przód on upominek przed oblicżem jego / á ſám przenocowáł onej nocy z hufcem ſwoim.
Wſtáwƺy tedy onej nocy / wziął obie żony ſwe / y dwie ſłużebnice ſwoje / y jedenáśćie ſynów ſwoich / y przeƺedł przez bród Jábok /
A wziąwƺy je / przepráwił je przez tęż rzekę / y przeprowádził wƺyſtko / co miáł.
A tylko ſám Jákób zoſtáł.
A oto / biedził śię z nim mąż áż do wejśćiá zorzy ; który widząc / że go nie mógł przemódz / uderzył Jákóbá w ſtáw biodry jego / y wytrąćiłá śię z ſtáwu biodrá Jákóbowá / gdy śię z nim mocowáł.
Y rzekł : Puść mię / bo już wſchodzi zorzá. Y odpowiedziáł : Nie puƺcżę ćię / áż mi będzieƺ błogoſłáwił.
Tedy mu rzekł : Co zá imię twoje? Y odpowiedziáł : Jákób.
Y rzekł : Nie będzie názywáne więcej imię twoje Jákób / ále Izráel ; boś ſobie mężnie pocżynáł z Bogiem / y z ludźmi / y przemogłeś.
Y ſpytáł Jákób mówiąc : Oznájmij mi proƺę imię twoje ; á on odpowiedziáł : Cżemu śię pytáƺ o imieniu mojem? y támże mu błogoſłáwił.
Tedy názwáł Jákób imię miejſcá onego Fánuel / mówiąc : Iżem widziáł Bogá twárzą w twárz / á záchowáná jeſt duƺá mojá.
Y weƺło mu ſłońce / kiedy minął miejſce Fánuel ; á on uchrámowáł ná biodrę ſwoję.
Przetoż nie jádáją ſynowie Izráelſcy żyły ſkurcżonej / którá jeſt przy ſtáwie biodry / áż do dniá tego / iż był uderzył w ſtáw biodry Jákóbowej / y w żyłę ſkurcżoną.
