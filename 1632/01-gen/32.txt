A Jákób też poƺedł w drogę ſwoję y potkáli śię z nim Aniołowie Boży.
Y rzekł Jákób ujrzáwƺy je : Obóz to Boży ; y názwał imię miejſcá onego Máhánáim.
Potem poſłał Jákób poſły przed ſobą do Ezáwá / brátá ſwego / do źiemi Seir / do kráiny Edomſkiej.
Y rozkázał im mówiąc : Ták rzecżećie do páná mego Ezáwá : To mówi ſługá twój Jákób : U Lábáná byłem gośćiem / y mieƺkáłem z nim áż do tego cżáſu.
A mam woły y oſły / owce / y ſługi / y ſłużebnice / á poſyłám odpowiedźieć pánu memu / żebym ználázł łáſkę w ocżách twojich.
Y wróćili śię poſłowie do Jákóbá / mówiąc : Przyƺliśmy do brátá twego Ezáwá / który też idźie przećiwko tobie / á cżtery ſtá mężów z nim.
Y zląkł śię Jákób bárzo á ſtrwożył śię ; y rozdźielił lud / który z nim był / y owce / y woły / y wielbłądy / ná dwá hufce ;
Y rzekł : Jeſliby przyƺedł Ezáw do jednego hufcá / á poráźiłby go / tedy hufiec / który pozoſtánie / będźie záchowány.
Y rzekł Jákób : Boże ojcá mego Abráhámá / y Boże ojcá mego Izááká / Pánie / któryś do mnie rzekł : Wróć śię do źiemi twojey / y do rodźiny twojey / á ucżynięć dobrze.
Mniejƺym jeſt niż wƺyſtkie zmiłowánia / y niż wƺyſtká práwdá / którąś ucżynił z ſługą ſwym. Abowiem tylko o láſce mojey przeƺedłem ten Jordán / á teraz mam dwá hufce.
Wyrwij mię proƺę z ręki brátá mego / z ręki Ezáwá ; boć śię go boję / by ſnadź przyƺedƺy nie zábił mię / y mátki z ſynámi.
Wƺákeś rzekł : Dobrze cżyniąc będęć dobrze cżynił / á rozmnożę naśienie twoje jáko piáſek morſki / który zlicżon być nie może / dla mnóſtwá.
Y przenocował tám oney nocy / y wźiął z tego / co miał przy ręku / upominek dla Ezáwá / brátá ſwego.
To jeſt kóz dwieśćie / y kozłów dwádźieśćiá / owiec dwieśćie / y báránów dwádźieśćiá.
Wielbłądźic odchowujących młode / ze źrebięty ich / trzydźieśći / krów cżterdźieśći / y wołów dźieśięć / dwádźieśćiá oślic / y dźieśięć ośląt.
Y oddał je w ręce ſług ſwojich / káżde ſtádo z oſobná / y rzekł do ſług ſwojich : Idźćie przedemną / á plác ucżyńćie miedzy ſtádem á miedzy ſtádem.
Y rozkázał pierwƺemu / mówiąc : Gdy śię potká z tobą Ezáw / brát mój / á ſpytá ćię / mówiąc : Cżyjeś ty? y dokąd idźieƺ? á cżyje to ſtádo przed tobą?
Tedy powieƺ : Sługi twego Jákóbá jeſt to upominek / poſłány pánu memu Ezáwowi ; á oto / y ſám idźie zá námi.
Tákże też rozkázał drugiemu / y trzećiemu / y wƺyſtkim idącym zá temi ſtády / mówiąc : Temiż ſłowy mówćie do Ezáwá / gdy go potkáćie.
Powiećie mu też : Oto / ſługá twój Jákób idźie zá námi ; mówił bowiem : Ubłágám oblicże jego upominkiem / który idźie przedemną / á potem ujrzę oblicże jego ; owá mię ſnadź w łáſkę przyjmie.
Y poƺedł w przód on upominek przed oblicżem jego / á ſám przenocował oney nocy z hufcem ſwoim.
Wſtáwƺy tedy oney nocy / wźiął obie żony ſwe / y dwie ſłużebnice ſwoje / y jedenaśćie ſynów ſwojich / y przeƺedł przez bród Jábok /
A wźiąwƺy je / przepráwił je przez tęż rzekę / y przeprowádźił wƺyſtko / co miał.
A tylko ſám Jákób zoſtał.
A oto / biedźił śię z nim mąż áż do wejśćiá zorzy ; który widząc / że go nie mógł przemódz / uderzył Jákóbá w ſtáw biodry jego / y wytrąćiłá śię z ſtáwu biodrá Jákóbowá / gdy śię z nim mocował.
Y rzekł : Puść mię / bo już wſchodźi zorzá. Y odpowiedźiał : Nie puƺcżę ćię / áż mi będźieƺ błogoſłáwił.
Tedy mu rzekł : Co zá imię twoje? Y odpowiedźiał : Jákób.
Y rzekł : Nie będźie názywáne więcey imię twoje Jákób / ále Izráel ; boś ſobie mężnie pocżynał z Bogiem / y z ludźmi / y przemogłeś.
Y ſpytał Jákób mówiąc : Oznájmij mi proƺę imię twoje ; á on odpowiedźiał : Cżemu śię pytáƺ o imieniu mojim? y támże mu błogoſłáwił.
Tedy názwał Jákób imię miejſcá onego Fánuel / mówiąc : Iżem widźiał Bogá twárzą w twárz / á záchowáná jeſt duƺá mojá.
Y weƺło mu ſłońce / kiedy minął miejſce Fánuel ; á on uchrámował ná biodrę ſwoję.
Przetoż nie jádáją ſynowie Izráelſcy żyły ſkurcżoney / która jeſt przy ſtáwie biodry / áż do dniá tego / iż był uderzył w ſtáw biodry Jákóbowej / y w żyłę ſkurcżoną.
