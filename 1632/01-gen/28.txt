Tedy wezwał Izáák Jákóbá / y błogoſłáwił mu / á rozkázał mu / mówiąc : Nie pojmuj żony z córek Chánánejſkich.
Ale wſtáwƺy idź do kráiny Syryjſkiej / do domu Bátuelá / ojcá mátki twojey / á weźmij ſobie z támtąd żonę / z córek Lábáná / brátá mátki twojey.
A Bóg Wƺechmogący niech ći błogoſłáwi / á niech ćię rozrodźi y rozmnoży / ábyś był w mnóſtwo ludu ;
Y niech ći da błogoſłáwieńſtwo Abráhámowe / tobie y náśieniu twemu z tobą / ábyś odźiedźicżył źiemię pielgrzymſtwá twojego / którą dał Bóg Abráhámowi.
Y ták wyſłał Izáák Jákóbá / który ƺedł do kráiny Syryjſkiej / do Lábáná / ſyná Bátuelowego / Syryjcżyká / brátá Rebeki / mátki Jákóbowej y Ezáwowej.
A widząc Ezáw / iż błogoſłáwił Izáák Jákóbowi / y poſłał go do kráiny Syryjſkiej / áby ſobie pojął z támtąd żonę / á iż błogoſłáwiąc mu / przykázał / mówiąc : Nie weźmieƺ żony z córek Chánánejſkich ;
Y Jákób poſłuƺny był ojcu ſwemu y mátce ſwojey / y poƺedł do kráiny Syryjſkiej ;
Widząc też Ezáw / że śię nie podobáją córki Chánánejſkie w ocżách Izááká / ojcá jego :
Tedy ƺedł Ezáw do Iſmáelá / y pojął mimo inne żony ſwoje / Máhálátę córkę Iſmáelá / ſyná Abráhámowego / śioſtrę Nebájotowę / ſobie zá żonę.
A Jákób wyƺedƺy z Beerſebá / ƺedł do Háránu.
Y przyƺedł ná jedno miejſce / y nocował tám ( ábowiem już było záƺło ſłońce ) á wźiąwƺy jeden z kámieni miejſcá onego / podłożył pod głowę ſwoję / y ſpał ná temże miejſcu.
Y śniło mu śię / á ono drábiná ſtáłá ná źiemi / á wierzch jey dośięgał niebá ; á oto / Aniołowie Boży wſtępowáli y zſtępowáli po niej.
A Pán ſtał nád nią y rzekł : Jam jeſt Pán / Bóg Abráhámá / ojcá twego / y Bóg Izááká / źiemię tę / ná którey ty śpiƺ / tobie dam y náśieniu twojimu.
A będźie naśienie twoje jáko proch źiemi / y rozmnożyƺ śię ná záchód / y ná wſchód / y ná północy / y ná południe ; á będą ubłogoſłáwione w tobie wƺyſtkie narody źiemi y w náśieniu twojim.
A oto / Ja jeſtem z tobą y ſtrzedz ćię będę gdźiekolwiek pójdźieƺ / y przywrócę ćię do tey źiemi ; bo nie opuƺcżę ćię / áż ucżynię com ći rzekł.
Tedy gdy śię ocknął Jákób ze ſnu ſwego / rzekł : Zápráwdę Pán jeſt ná tym miejſcu / á jam nie wiedźiał.
Y zlęknąwƺy śię / rzekł : O jáko to ſtráƺne miejſce! nic tu nie jeſt innego jedno dom Boży / á tu bramá niebieſká.
Y wſtał Jákób bárzo ráno / á wźiąwƺy kámień / który był podłożył pod głowę ſwoję / poſtáwił go ná znák / y nálał oliwy ná wierzch jego.
Y názwał imię miejſcá onego Betel ; bo było przed tym imię miáſtá onego Luz.
Tedy ucżynił Jákób ślub / mówiąc : Jeſliż będźie Bóg ze mną / á ſtrzedz mię będźie ná tey drodze / którą ja idę / y da mi chleb ku jedzeniu / y odźienie ku obłócżeniu /
A wrócę śię w pokoju do domu ojcá mego : tedy będźie mi Pán zá Bogá.
A kámień ten / którym wyſtáwił ná znák / będźie domem Bożym / á ze wƺyſtkiego / co mi dáƺ / dźieśięćinę pewną oddáwáć ći będę.
