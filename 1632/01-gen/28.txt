Tedy wezwáł Izáák Jákóbá / y błogoſłáwił mu / á rozkázáł mu / mówiąc : Nie pojmuj żony z córek Chánánejſkich.
Ale wſtáwƺy idź do kráiny Syryjſkiej / do domu Bátuelá / ojcá mátki twojej / á weźmij ſobie ſtámtąd żonę / z córek Lábáná / brátá mátki twojej.
A Bóg Wƺechmogący niech ći błogoſłáwi / á niech ćię rozrodzi y rozmnoży / ábyś był w mnóſtwo ludu ;
Y niech ći dá błogoſłáwieńſtwo Abráhámowe / tobie y náśieniu twemu z tobą / ábyś odziedzicżył ziemię pielgrzymſtwá twojego / którą dáł Bóg Abráhámowi.
Y ták wyſłáł Izáák Jákóbá / który ƺedł do kráiny Syryjſkiej / do Lábáná / ſyná Bátuelowego / Syryjcżyká / brátá Rebeki / mátki Jákóbowej y Ezáwowej.
A widząc Ezáw / iż błogoſłáwił Izáák Jákóbowi / y poſłáł go do kráiny Syryjſkiej / áby ſobie pojął ſtámtąd żonę / á iż błogoſłáwiąc mu / przykázáł / mówiąc : Nie weźmieƺ żony z córek Chánánejſkich ;
Y Jákób poſłuƺny był ojcu ſwemu y mátce ſwojej / y poƺedł do kráiny Syryjſkiej ;
Widząc też Ezáw / że śię nie podobáją córki Chánánejſkie w ocżách Izááká / ojcá jego :
Tedy ƺedł Ezáw do Iſmáelá / y pojął mimo inne żony ſwoje / Máhálátę córkę Iſmáelá / ſyná Abráhámowego / śioſtrę Nebájotowę / ſobie zá żonę.
A Jákób wyƺedłƺy z Beerſebá / ƺedł do Háránu.
Y przyƺedł ná jedno miejſce / y nocowáł tám ( ábowiem już było záƺło ſłońce ) á wziąwƺy jeden z kámieni miejſcá onego / podłożył pod głowę ſwoję / y ſpáł ná temże miejſcu.
Y śniło mu śię / á ono drábiná ſtáłá ná ziemi / á wierzch jej dośięgáł niebá ; á oto / Aniołowie Boży wſtępowáli y zſtępowáli po niej.
A Pán ſtáł nád nią y rzekł : Jám jeſt Pán / Bóg Abráhámá / ojcá twego / y Bóg Izááká / ziemię tę / ná której ty śpiƺ / tobie dám y náśieniu twojimu.
A będzie náśienie twoje jáko proch ziemi / y rozmnożyƺ śię ná záchód / y ná wſchód / y ná północy / y ná południe ; á będą ubłogoſłáwione w tobie wƺyſtkie národy ziemi y w náśieniu twojim.
A oto / Já jeſtem z tobą y ſtrzedz ćię będę gdziekolwiek pójdzieƺ / y przywrócę ćię do tej ziemi ; bo nie opuƺcżę ćię / áż ucżynię com ći rzekł.
Tedy gdy śię ocknął Jákób ze ſnu ſwego / rzekł : Zápráwdę Pán jeſt ná tem miejſcu / á jám nie wiedziáł.
Y zlęknąwƺy śię / rzekł : O jáko to ſtráƺne miejſce! nic tu nie jeſt innego jedno dom Boży / á tu brámá niebieſká.
Y wſtáł Jákób bárdzo ráno / á wziąwƺy kámień / który był podłożył pod głowę ſwoję / poſtáwił go ná znák / y náláł oliwy ná wierzch jego.
Y názwáł imię miejſcá onego Betel ; bo było przedtem imię miáſtá onego Luz.
Tedy ucżynił Jákób ślub / mówiąc : Jeźliż będzie Bóg ze mną / á ſtrzedz mię będzie ná tej drodze / którą já idę / y dá mi chleb ku jedzeniu / y odzienie ku obłócżeniu /
A wrócę śię w pokoju do domu ojcá mego : tedy będzie mi Pán zá Bogá.
A kámień ten / którym wyſtáwił ná znák / będzie domem Bożym / á ze wƺyſtkiego / co mi dáƺ / dzieśięćinę pewną oddáwáć ći będę.
