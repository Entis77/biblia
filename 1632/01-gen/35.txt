Rzekł potem Bóg do Jákóbá : Wſtáń / wſtąp do Betelá / á mieƺkáj tám / y ucżyń tám ołtárz Bogu / któryć śię ukázáł / gdyś ućiekáł przed oblicżem Ezáwá / brátá twego.
Tedy rzekł Jákób do domowników ſwych / y do wƺyſtkich / którzy z nim byli : Odrzuććie bogi cudze / którzy w pośrodku wáſ ſą / á ocżyśććie śię / y odmieńćie ƺáty wáƺe.
A wſtáwƺy pójdźmy do Betelá / y ucżynię tám ołtárz Bogu / który mię wyſłucháł w dźień utrápieniá mego / y był ze mną w drodze / którąm chodźił.
Y oddáli Jákóbowi wƺyſtkie bogi cudze / które mieli / y náuƺnice / które były ná uƺách ich / y zákopáł je Jákób pod onym dębem / który był nie dáleko Sychem.
Y wyƺli ſtámtąd ; á ſtrách Boży pádł ná miáſtá / które były około nich / iż nie gonili ſynów Jákóbowych.
Przyƺedł tedy Jákób do Luzy / którá jeſt w źiemi Chánánejſkiej / tá jeſt Betel / ſám y wƺyſtek lud / który z nim był.
Y zbudowáł tám ołtárz / á názwáł miejſce ono El Betel ; bo mu śię tám był Bóg ukázáł / gdy ućiekáł przed oblicżem brátá ſwego.
Tedy umárłá Deborá / mámká Rebeki / y pogrzebioná jeſt przy Betel pod dębem / y názwáno imię onego miejſcá / Allon Báchut.
Y ukázáł śię Bóg znowu Jákóbowi / gdy śię wrácáł z Pádán Syryjſkiego / y błogoſłáwił mu.
Y rzekł mu Bóg : Imię twoje jeſt Jákób ; nie tylko będźie zwáne imię twoje ná potem Jákób / ále też Izráel będźie imię twoje ; y názwáł imię jego Izráel.
Y rzekł mu Bóg : Jám jeſt Bóg wƺechmogący / rozrádzáj śię / y rozmnáżáj śię ; náród / y mnóſtwo národów będźie z ćiebie / á królowie z biódr twoich wynijdą.
Y źiemię / którąm dáł Abráhámowi y Izáákowi / tobie ją dám / y náśieniu twemu po tobie dám tę źiemię.
Y odƺedł Bóg od niego z miejſcá / ná którem mówił z nim.
Zátem poſtáwił Jákób znák ná miejſcu onem / gdźie Bóg mówił z nim / á znák on był kámienny / y pokropił go pokropieniem / y poláł go oliwą.
Y názwáł Jákób imię miejſcá onego / gdźie Bóg z nim mówił / Betel.
Potem odeƺli z Betel ; y było jeƺcże jákoby milá drogi do Efráty / y rodźiłá Ráchel á ćiężkie rodzenie miáłá.
A gdy ćiężko prácowáłá przy rodzeniu / rzekłá bábá do niej : Nie bój śię ; bo y tego ſyná będźieƺ miáłá.
Y ſtáło śię / gdy wychodźiłá duƺá jej / ( bo támże umárłá ) názwáłá imię jego Ben Oni ; ále ojćiec jego názwáł go Benjámin.
A ták umárłá Ráchel / y pogrzebioná jeſt ná drodze ku Efráćie ; táć jeſt Betlehem.
Y poſtáwił Jákób znák nád grobem jej ; toć jeſt znák grobu Ráchelinego áż po dźiś dźień.
Y poƺedł ſtámtąd Izráel / y rozbił námiot ſwój zá wieżą Heder.
Stáło śię tedy / gdy mieƺkáł Izráel w onej kráinie / że ƺedł Ruben / y ſpáł z Bálą / záłożnicą ojcá ſwego / y uſłyƺáł to Izráel. A było ſynów Jákóbowych dwánáśćie.
Synowie Lii : pierworodny Jákóbów Ruben / y Symeon / y Lewi / y Judáſ / y Iſáƺár / y Zábulon.
Synowie Rácheli : Józef y Benjámin.
A ſynowie Báli / ſłużebnicy Ráchelinej : Dán y Neftáli.
Synowie też Zelfy / ſłużebnicy Lii : Gád y Aſer. Ci ſą ſynowie Jákóbowi / którzy mu śię urodźili w Pádánie Syryjſkim.
Y przyƺedł Jákób do Izááká / ojcá ſwego / do Mámre / do miáſtá Arbá / to jeſt Hebron / gdźie mieƺkáł Abráhám y Izáák.
A było dni Izáákowych ſto lát / y ośmdźieśiąt lát.
Y dokonáł Izáák / y umárł / y przyłącżony jeſt do ludu ſwego / ſtáry y pełen dni ; á pogrzebli go Ezáw / y Jákób / ſynowie jego.
