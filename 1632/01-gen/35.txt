Rzekł potem Bóg do Jákóbá : Wſtáń / wſtąp do Betelá / á mieƺkáj tám / y ucżyń tám ołtárz Bogu / któryć śię ukázał / gdyś ućiekał przed oblicżem Ezáwá / brátá twego.
Tedy rzekł Jákób do domowników ſwych / y do wƺyſtkich / którzy z nim byli : Odrzuććie bogi cudze / którzy w pośrzodku was ſą / á ocżyśććie śię / y odmieńćie ƺáty wáƺe.
A wſtáwƺy pójdźmy do Betelá / y ucżynię tám ołtárz Bogu / który mię wyſłuchał w dźień utrapienia mego / y był ze mną w drodze / którąm chodźił.
Y oddáli Jákóbowi wƺyſtkie bogi cudze / które mieli / y náuƺnice / które były ná uƺách ich / y zákopał je Jákób pod onym dębem / który był nie dáleko Sychem.
Y wyƺli z támtąd ; á ſtrách Boży pádł ná miáſtá / które były około nich / iż nie gonili ſynów Jákóbowych.
Przyƺedł tedy Jákób do Luzy / która jeſt w źiemi Chánánejſkiej / tá jeſt Betel / ſám y wƺyſtek lud / który z nim był.
Y zbudował tám ołtárz / á názwał miejſce ono El Betel ; bo mu śię tám był Bóg ukázał / gdy ućiekał przed oblicżem brátá ſwego.
Tedy umárłá Deborá / mámká Rebeki / y pogrzebioná jeſt przy Betel pod dębem / y názwano imię onego miejſcá / Allon Báchut.
Y ukázał śię Bóg znowu Jákóbowi / gdy śię wrácał z Pádán Syryjſkiego / y błogoſłáwił mu.
Y rzekł mu Bóg : Imię twoje jeſt Jákób ; nie tylko będźie zwáne imię twoje ná potem Jákób / ále też Izráel będźie imię twoje ; y názwał imię jego Izráel.
Y rzekł mu Bóg : Jam jeſt Bóg wƺechmogący / rozrádzáj śię / y rozmnáżáj śię ; naród / y mnóſtwo narodów będźie z ćiebie / á królowie z biódr twojich wynijdą.
Y źiemię / którąm dał Abráhámowi y Izáákowi / tobie ją dam / y náśieniu twemu po tobie dam tę źiemię.
Y odƺedł Bóg od niego z miejſcá / ná którem mówił z nim.
Zátem poſtáwił Jákób znák ná miejſcu onem / gdźie Bóg mówił z nim / á znák on był kámienny / y pokropił go pokropieniem / y polał go oliwą.
Y názwał Jákób imię miejſcá onego / gdźie Bóg z nim mówił / Betel.
Potem odeƺli z Betel ; y było jeƺcże jákoby milá drogi do Efráty / y rodźiłá Ráchel á ćiężkie rodzenie miáłá.
A gdy ćiężko prácowáłá przy rodzeniu / rzekłá bábá do niej : Nie bój śię ; bo y tego ſyná będźieƺ miáłá.
Y ſtáło śię / gdy wychodźiłá duƺá jey / ( bo támże umárłá ) názwáłá imię jego Ben Oni ; ále oćiec jego názwał go Benjámin.
A ták umárłá Ráchel / y pogrzebioná jeſt ná drodze ku Efráćie ; táć jeſt Betlehem.
Y poſtáwił Jákób znák nád grobem jey ; toć jeſt znák grobu Ráchelinego áż po dźiś dźień.
Y poƺedł z támtąd Izráel / y rozbił námiot ſwój zá wieżą Heder.
Stáło śię tedy / gdy mieƺkał Izráel w oney krájinie / że ƺedł Ruben / y ſpał z Bálą / záłożnicą ojcá ſwego / y uſłyƺał to Izráel. A było ſynów Jákóbowych dwánaśćie.
Synowie Lii : pierworodny Jákóbów Ruben / y Symeon / y Lewi / y Judás / y Iſáƺár / y Zábulon.
Synowie Rácheli : Józef y Benjámin.
A ſynowie Báli / ſłużebnicy Rácheliney : Dán y Neftáli.
Synowie też Zelfy / ſłużebnicy Lii : Gád y Aſer. Ci ſą ſynowie Jákóbowi / którzy mu śię urodźili w Pádánie Syryjſkim.
Y przyƺedł Jákób do Izááká / ojcá ſwego / do Mámre / do miáſtá Arbá / to jeſt Hebron / gdźie mieƺkał Abráhám y Izáák.
A było dni Izáákowych ſto lat / y ośmdźieśiąt lat.
Y dokonał Izáák / y umárł / y przyłącżony jeſt do ludu ſwego / ſtáry y pełen dni ; á pogrzebli go Ezáw / y Jákób / ſynowie jego.
