A głód wielki był w onej ziemi.
Y ſtáło śię / gdy ſtráwili onę żywność / którą byli przynieśli z Egiptu / że rzekł do nich ojćiec ich : Idźćie znowu / á kupćie nám cokolwiek żywnośći.
Y rzekł do niego Judáſ / mówiąc : Oświádcżájąc oświádcżył śię przećiwko nám ten mąż / mówiąc : Nie ujrzyćie oblicżá mojego / jeźli nie będzie brát wáƺ z wámi :
Jeźli tedy pośleƺ brátá náƺego z námi / pojedziemy y nákupiemyć żywnośći ;
Ale jeźli nie pośleƺ / nie pojedziemy ; bo on mąż mówił do náſ : Nie ujrzyćie twárzy mojej / jeźli nie będzie brátá wáƺego z wámi.
Tedy rzekł Izráel : Przecżżeśćie mi ták źle ucżynili / powiedziáwƺy temu mężowi / że jeƺcże máćie brátá?
Y rzekli : Pilnie śię pytáł on mąż o náſ / y o rodzinie náƺej / mówiąc : Żywże jeƺcże ojćiec wáƺ? máćieli jeƺcże którego brátá? Y odpowiedzieliśmy mu według pytániá jego ; cóżeśmy wiedzieli / że miáł mówić : Przywiedźćie mi tu brátá wáƺego?
Y rzekł Judáſ do Izráelá / ojcá ſwego : Poślij tego młodzieńcá ze mną / á wſtáwƺy pojedziemy / ábyśmy żyli á nie pomárli głodem / ták my / jáko y ty / y dziátecżki náƺe.
Já przyrzekám záń / z ręki mojej ƺukáj go ; jeźli go nie przywiodę do ćiebie / á nie ſtáwię go przed tobą / będęć winien grzechu po wƺyſtkie dni ;
Bo gdybyśmy byli nie omieƺkáli / jużbyśmy śię byli dwá kroć wróćili.
Tedy rzekł do nich Izráel / ojćiec ich : Jeźliże ták być muśi / ucżyńćież to ; nábierzćie nájlepƺych pożytków ziemi w nácżyniá wáƺe / á zánieśćie mężowi onemu w upominku : trochę bálſámu / y trochę miodu / y rzecży wonnych / y myrry / orzechów terebintowych / y migdáłów.
Pieniądze też dwoje weźmijćie do rąk wáƺych / á pieniądze przywrócone ná wierzchu worów wáƺych odnieśćie w ręce ſwojej ; ſnáć śię to omyłką ſtáło.
Ale y brátá wáƺego weźmijćie / á wſtáwƺy jedźćie znowu do mężá onego ;
A Bóg Wƺechmogący niech wám dá miłośierdzie przed oblicżem tego mężá / áby wám wypuśćił brátá wáƺego drugiego y Benjáminá ; á já jáko ośieroćiáły bez dziátek będę.
Tedy wziąwƺy oni mężowie on podárek / y dwoje pieniądze wziąwƺy w ręce ſwe / y Benjáminá / wſtáli / y jecháli do Egiptu / y ſtánęli przed Józefem.
A ujrzáwƺy Józef z nimi Benjáminá / rzekł do tego / który był ſpráwcą domu jego : Wprowádź te męże w dom / á zábij bydlę y nágotuj ; bo ze mną jeść będą mężowie ći w południe.
Y ucżynił on mąż / jáko mu rozkázáł Józef / á wprowádził on mąż one ludzie w dom Józefów.
Báli śię tedy mężowie oni / gdy byli wprowádzeni w dom Józefów / y mówili : Dlá onychći to pieniędzy / które pierwej włożono było do worów náƺych / wprowádzeni tu jeſteśmy / áby potwárz ná náſ zwáliwƺy tárgnął śię ná náſ / á pobráł w niewolą náſ y oſły náƺe.
A przyſtąpiwƺy do mężá tego / który był ſpráwcą domu Józefowego / mówili do niego we drzwiách domu.
Y rzekli : Słucháj mię / pánie mój! przyjecháliśmy byli pierwej kupowáć żywność.
Y ſtáło śię / gdyśmy przyjecháli do goſpody / y rozwiązáliśmy wory náƺe / oto / pieniądze káżdego były ná wierzchu woru jego / pieniądze náƺe / według wági ich / któreśmy záś przynieśli w rękách náƺych.
Pieniądze też drugie przynieśliśmy w rękách náƺych / ábyśmy nákupili żywnośći / á nie wiemy / kto włożył te pieniądze náƺe do worów náƺych.
A on rzekł : Pokój wám / nie bójćie śię ; Bóg wáƺ / y Bóg ojcá wáƺego dáł wám ſkárb do worów wáƺych : pieniądze wáƺe doƺły mię. Y wywiódł do nich Symeoná.
A przywiódłƺy on mąż one ludzie w dom Józefów / dáł im wody / y umyli nogi ſwe ; dáł też obrok oſłom ich.
Zátem nágotowáli podárek / niż przyƺedł Józef w południe ; ſłyƺeli bowiem / iż tám mieli jeść chleb.
A gdy wƺedł Józef w dom / przynieśli mu podárek / który mieli w rękách ſwych w domu onym / y kłániáli mu śię áż do ziemi.
Y pytáł ich / jákoby śię mieli / y rzekł : Zdrówże jeſt ojćiec wáƺ ſtáry / o którymeśćie mi powiádáli? Żywże jeƺcże?
A oni odpowiedzieli : Zdrówći ſługá twój / ojćiec náƺ / jeƺcżeć żyw. A ſchyliwƺy śię / pokłonili mu śię.
Tedy podnióſłƺy ocży ſwe / ujrzáł Benjáminá / brátá ſwego / ſyná mátki ſwej / y rzekł : Tenże jeſt brát wáƺ młodƺy / o którymeśćie mi powiádáli? Y rzekł mu : Bóg niech ći będzie miłośćiw / miły ſynu.
Y pokwápił śię Józef wynijść / bo śię były wzruƺyły wnętrznośći jego ku brátu ſwemu / y ƺukáł miejſcá / gdzieby płákáł / y wƺedłƺy do komory / płákáł tám.
Potem umywƺy twárz ſwoję / wyƺedł záśię / y wſtrzymáł śię / y rzekł : Kłádźćie chleb.
Y położono jemu oſobno / y onym oſobno / Egipcżánom też / którzy jedli z nim / oſobno ; bo nie mogą jeść Egipcżánie z Hebrejcżykámi chlebá / gdyż to jeſt obrzydliwośćią Egipcżánom.
Y uśiedli przed oblicżem jego / pierworodny według pierworodztwá ſwego / á młodƺy według młodośći ſwojej ; y dziwowáli śię mężowie oni pátrząc jeden ná drugiego.
Y biorąc potráwy przed ſobą dáwáł im ; á doſtáłá śię pięć kroć więkƺá cżęść Benjáminowi nád inne cżęśći onych wƺyſtkich ; y pili / á podpili ſobie z nim.
