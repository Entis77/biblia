A głód wielki był w oney źiemi.
Y ſtáło śię / gdy ſtráwili onę żywność / którą byli przynieśli z Egiptu / że rzekł do nich oćiec ich : Idźćie znowu / á kupćie nam cokolwiek żywnośći.
Y rzekł do niego Judás / mówiąc : Oświádcżájąc oświádcżył śię przećiwko nam ten mąż / mówiąc : Nie ujrzyćie oblicża mojego / jeſli nie będźie brát wáƺ z wámi :
Jeſli tedy pośleƺ brátá náƺego z námi / pojedźiemy y nákupiemyć żywnośći ;
Ale jeſli nie pośleƺ / nie pojedźiemy ; bo on mąż mówił do nas : Nie ujrzyćie twárzy mojey / jeſli nie będźie brátá wáƺego z wámi.
Tedy rzekł Izráel : Przecżżeśćie mi ták źle ucżynili / powiedźiawƺy temu mężowi / że jeƺcże máćie brátá?
Y rzekli : Pilnie śię pytał on mąż o nas / y o rodźinie náƺej / mówiąc : Żywże jeƺcże oćiec wáƺ? máćieli jeƺcże którego brátá? Y odpowiedźieliśmy mu według pytánia jego ; cóżeśmy wiedźieli / że miał mówić : Przywiedźćie mi tu brátá wáƺego?
Y rzekł Judás do Izráelá / ojcá ſwego : Poślij tego młodźieńcá ze mną / á wſtáwƺy pojedźiemy / ábyſmy żyli á nie pomárli głodem / ták my / jáko y ty / y dźiátecżki náƺe.
Ja przyrzekám záń / z ręki mojey ƺukáj go ; jeſli go nie przywiodę do ćiebie / á nie ſtáwię go przed tobą / będęć winien grzechu po wƺyſtkie dni ;
Bo gdybyſmy byli nie omieƺkáli / jużbyſmy śię byli dwá kroć wróćili.
Tedy rzekł do nich Izráel / oćiec ich : Jeſliże ták być muśi / ucżyńćież to ; nábierzćie nájlepƺych pożytków źiemi w nacżynia wáƺe / á zánieśćie mężowi onemu w upominku : trochę bálſámu / y trochę miodu / y rzecży wonnych / y myrry / orzechów terebintowych / y migdáłów.
Pieniądze też dwoje weźmijćie do rąk wáƺych / á pieniądze przywrócone ná wierzchu worów wáƺych odnieśćie w ręce ſwojey ; ſnadź śię to omyłką ſtáło.
Ale y brátá wáƺego weźmijćie / á wſtáwƺy jedźćie znowu do mężá onego ;
A Bóg Wƺechmogący niech wam da miłośierdźie przed oblicżem tego mężá / áby wam wypuśćił brátá wáƺego drugiego y Benjáminá ; á ja jáko ośieroćiáły bez dźiátek będę.
Tedy wźiąwƺy oni mężowie on podárek / y dwoje pieniądze wźiąwƺy w ręce ſwe / y Benjáminá / wſtáli / y jecháli do Egiptu / y ſtánęli przed Józefem.
A ujrzáwƺy Józef z nimi Benjáminá / rzekł do tego / który był ſpráwcą domu jego : Wprowádź te męże w dom / á zábij bydlę y nágotuj ; bo ze mną jeść będą mężowie ći w południe.
Y ucżynił on mąż / jáko mu rozkázał Józef / á wprowádźił on mąż one ludźie w dom Józefów.
Báli śię tedy mężowie oni / gdy byli wprowádzeni w dom Józefów / y mówili : Dla onychći to pieniędzy / które pierwej włożono było do worów náƺych / wprowádzeni tu jeſteśmy / áby potwárz ná nas zwáliwƺy tárgnął śię ná nas / á pobrał w niewolą nas y oſły náƺe.
A przyſtąpiwƺy do mężá tego / który był ſpráwcą domu Józefowego / mówili do niego we drzwiách domu.
Y rzekli : Słucháj mię / pánie mój! przyjecháliśmy byli pierwej kupowáć żywność.
Y ſtáło śię / gdyśmy przyjecháli do goſpody / y rozwiązáliśmy wory náƺe / oto / pieniądze káżdego były ná wierzchu woru jego / pieniądze náƺe / według wági ich / któreśmy záś przynieśli w rękách náƺych.
Pieniądze też drugie przynieśliśmy w rękách náƺych / ábyſmy nákupili żywnośći / á nie wiemy / kto włożył te pieniądze náƺe do worów náƺych.
A on rzekł : Pokój wam / nie bójćie śię ; Bóg wáƺ / y Bóg ojcá wáƺego dał wam ſkárb do worów wáƺych : pieniądze wáƺe doƺły mię. Y wywiódł do nich Symeoná.
A przywiódƺy on mąż one ludźie w dom Józefów / dał im wody / y umyli nogi ſwe ; dał też obrok oſłom ich.
Zátem nágotowáli podárek / niż przyƺedł Józef w południe ; ſłyƺeli bowiem / iż tám mieli jeść chleb.
A gdy wƺedł Józef w dom / przynieśli mu podárek / który mieli w rękách ſwych w domu onym / y kłániáli mu śię áż do źiemi.
Y pytał ich / jákoby śię mieli / y rzekł : Zdrówże jeſt oćiec wáƺ ſtáry / o którymeśćie mi powiádáli? Żywże jeƺcże?
A oni odpowiedźieli : Zdrówći ſługá twój / oćiec náƺ / jeƺcżeć żyw. A ſchyliwƺy śię / pokłonili mu śię.
Tedy podnióſƺy ocży ſwe / ujrzał Benjáminá / brátá ſwego / ſyná mátki ſwej / y rzekł : Tenże jeſt brát wáƺ młodƺy / o którymeśćie mi powiádáli? Y rzekł mu : Bóg niech ći będźie miłośćiw / miły ſynu.
Y pokwápił śię Józef wynijść / bo śię były wzruƺyły wnętrznośći jego ku brátu ſwemu / y ƺukał miejſcá / gdźieby płákał / y wƺedƺy do komory / płákał tám.
Potem umywƺy twárz ſwoję / wyƺedł záśię / y wſtrzymał śię / y rzekł : Kłádźćie chleb.
Y położono jemu oſobno / y onym oſobno / Egipcżánom też / którzy jedli z nim / oſobno ; bo nie mogą jeść Egipcżánie z Hebrejcżykámi chlebá / gdyż to jeſt obrzydliwośćią Egipcżánom.
Y uśiedli przed oblicżem jego / pierworodny według pierworodztwá ſwego / á młodƺy według młodośći ſwojey ; y dźiwowáli śię mężowie oni pátrząc jeden ná drugiego.
Y biorąc potráwy przed ſobą dáwał im ; á doſtáłá śię pięć kroć więkƺá cżęść Benjáminowi nád inne cżęśći onych wƺyſtkich ; y pili / á podpili ſobie z nim.
