A żyłá Sárá ſto lát / y dwádzieśćiá lát / y śiedm lát ; te ſą látá żywotá Sáry.
Y umárłá Sárá w mieśćie Arbá / które zowią Hebron / w ziemi Chánánejſkiej : y przyƺedł Abráhám / áby żáłowáł Sáry / y płákáł jej.
Potem wſtáł Abráhám od umárłego ſwego / y rzekł do ſynów Hetowych / mówiąc :
Gośćiem y przychodniem jeſtem u wáſ ; dájćież mi ośiádłość grobu między wámi / ábym pogrzebáł umárłego mego od twárzy mojej.
Tedy opowiedzieli ſynowie Hetowi Abráhámowi / mówiąc mu :
Słucháj náſ / pánie mój : Kśiążęćiem Bożym jeſteś ty w pośrodku náſ : w nájprzedniejƺych grobách náƺych pogrzeb umárłego twego ; żáden z náſ nie będzie bronił grobu ſwego tobie / ábyś nie miáł pogrzebáć umárłego twego.
Tedy wſtáwƺy Abráhám / pokłonił śię ludowi onej ziemi / to jeſt ſynom Hetowym / y rzekł do nich / mówiąc :
Jeźli śię wám podobá / ábym pogrzebáł umárłego mego od twárzy mojej / ſłuchájćież mię / á przycżyńćie śię zá mną / do Efroná / ſyná Sochárowego /
Aby mi uſtąpił jáſkini ſwojej Máchpelá / którą má ná końcu polá ſwego / zá ſłuƺne pieniądze ; niech mi ją ſpuśći przed wámi w ośiádłość grobu.
( A Efron śiedziáł w pośrodku ſynów Hetowych. ) Tedy odpowiedziáł Efron Hetejcżyk Abráhámowi / w przytomnośći ſynów Hetowych / przed wƺyſtkimi / którzy chodzili w brámę miáſtá jego / mówiąc :
Nie ták / pánie mój / ále ſłucháj mię : Pole to dám tobie y jáſkinią / którá jeſt ná nim / dáwám ją tobie ; przed ocżymá ſynów ludu mego / dáwám ją tobie / pogrzebże umárłego twego.
Tedy śię pokłonił Abráhám przed ludem onej ziemi ;
Y rzekł do Efroná / w przytomnośći ludu onej ziemi / mówiąc : Rácżej / jeźlić śię zdá / proƺę / ſłucháj mię : dám ći pieniądze zá pole / weźmijże je odemnie / á pogrzebię tám umárłego mego.
Y odpowiedziáł Efron Abráhámowi mówiąc mu :
Pánie mój / ſłucháj mię. Ziemiá tá ſtoi zá cżtery ſtá ſyklów ſrebrá ; ále cóż to jeſt między mną y między tobą? pogrzeb umárłego twego.
Y uſłucháł Abráhám Efroná ; y odwáżył Abráhám Efronowi ſrebro / jáko był rzekł w przytomnośći ſynów Hetowych / cżtery ſtá ſyklów ſrebrá / ták jáko ƺły między kupcámi.
Y doſtáło śię pole Efronowe ( które jeſt w Máchpelu przećiwko Mámre / pole y jáſkiniá / którá jeſt ná nim / y wƺyſtkie drzewá / które były ná polu / które były ná wƺyſtkich gránicách jego w około ).
Abráhámowi w ośiádłość przed ocżymá ſynów Hetowych / y wƺyſtkich / którzy wchodzili w brámę miáſtá onego.
A ták pogrzebáł Abráhám Sárę / żonę ſwoję / w jáſkini polá w Máchpelu przećiwko Mámre / to jeſt Hebron / w ziemi Chánánejſkiej.
Y oddáne jeſt pole y jáſkiniá / którá byłá ná nim / Abráhámowi w ośiádłość grobu / od ſynów Hetowych.
