Y ſtáło śię niektórego dniá / gdy przyƺli ſynowie Boży / áby ſtánęli przed Pánem / przyƺedł też ƺátán miedzy nich / áby ſtánął przed Pánem.
Tedy rzekł Pán do ƺátáná : Gdźieś był / ſkąd idźieƺ? Y odpowiedźiał ƺátán Pánu / á rzekł : Okrążáłem źiemię / y przechodźiłem śię po niej.
Zátem rzekł Pán do ƺátáná : Przypátrzyłżeś śię ſłudze memu Ijobowi / że mu niemáƺ równego ná źiemi? Mąż to doſkonáły y ƺcżery / bojący śię Bogá / á odſtępujący od złego / y który jeƺcże trwá w uprzejmośći ſwojey ; á tyś mię pobudźił przećiw niemu / ábym go niƺcżył bez przycżyny.
Y odpowiedźiał ƺátán Pánu / y rzekł : Skórę zá ſkórę / y wƺyſtko / co ma cżłowiek / da zá duƺę ſwoję ;
Ale śćiągnij tylko rękę twoję / á dotknij kośći jego / y ćiáłá jego / ujrzyƺ / jeſliżeć w ocży złorzecżyć nie będźie.
Tedy rzekł Pán do ƺátáná : Oto w ręce twojey jeſt ; wƺákże żywot jego záchowáj.
Wyƺedƺy tedy ƺátán od oblicża Páńſkiego / záráźił Ijobá wrzodem złym od ſtopy nogi jego áż do wierzchu głowy jego ;
Ták / że wźiął ſkorupę / áby śię nią ſkrobał ; y śiedźiał w popiele.
Y rzekłá mu żoná jego : A jeƺcżeż trwáƺ w uprzejmoſći twojey? Złorzecż Bogu / á umrzyj.
Y rzekł do niej : Ták włáśnie mówiƺ / jáko ƺálone niewiáſty máwiáją. Izali tylko dobre przyjmowáć będźiemy od Bogá / á złego przyjmowáć nie będźiemy? W tym wƺyſtkim nie zgrzeƺył Ijob uſty ſwemi.
A gdy uſłyƺeli trzej przyjáćiele Ijobowi to wƺyſtko utrápienie / które nań przypádło / przyƺli káżdy z miejſcá ſwego : Elifás Temáńcżyk / y Bildád Suhytcżyk / y Sofár Náámátcżyk ; bo śię byli námówili poſpołu / áby przyƺedƺy pożáłowáli go / y ćieƺyli go.
A podnióſƺy ocży ſwoje z dáleká / nie poználi go / y wynióſƺy głos ſwój płákáli / á rozdárƺy káżdy płáƺcż ſwój miotáli proch nád głowy ſwe ku niebu ;
Y śiedźieli z nim ná źiemi śiedm dni y śiedm nocy / á żáden do niego ſłowá nie przemówił ; bo widźieli / że śię gwáłtownie wzmágáłá boleść jego.
