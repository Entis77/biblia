Y ſtáło śię niektórego dniá / gdy przyƺli ſynowie Boży / áby ſtánęli przed Pánem / przyƺedł też ƺátán między nich / áby ſtánął przed Pánem.
Tedy rzekł Pán do ƺátáná : Gdzieś był / ſkąd idzieƺ? Y odpowiedziáł ƺátán Pánu / á rzekł : Okrążáłem ziemię / y przechodziłem śię po niej.
Zátem rzekł Pán do ƺátáná : Przypátrzyłżeś śię ſłudze memu Ijobowi / że mu niemáƺ równego ná ziemi? Mąż to doſkonáły y ƺcżery / bojący śię Bogá / á odſtępujący od złego / y który jeƺcże trwá w uprzejmośći ſwojej ; á tyś mię pobudził przećiw niemu / ábym go niƺcżył bez przycżyny.
Y odpowiedziáł ƺátán Pánu / y rzekł : Skórę zá ſkórę / y wƺyſtko / co má cżłowiek / dá zá duƺę ſwoję ;
Ale śćiągnij tylko rękę twoję / á dotknij kośći jego / y ćiáłá jego / ujrzyƺ / jeźliżeć w ocży złorzecżyć nie będzie.
Tedy rzekł Pán do ƺátáná : Oto w ręce twojej jeſt ; wƺákże żywot jego záchowáj.
Wƺedłƺy tedy ƺátán od oblicżá Páńſkiego / záráził Ijobá wrzodem złym od ſtopy nogi jego áż do wierzchu głowy jego ;
Ták / że wziął ſkorupę / áby śię nią ſkrobáł ; y śiedziáł w popiele.
Y rzekłá mu żoná jego : A jeƺcżeż trwáƺ w uprzejmoſći twojej? Złorzecż Bogu / á umrzyj.
Y rzekł do niej : Ták włáśnie mówiƺ / jáko ƺálone niewiáſty máwiáją. Izáli tylko dobre przyjmowáć będziemy od Bogá / á złego przyjmowáć nie będziemy? W tem wƺyſtkiem nie zgrzeƺył Ijob uſty ſwemi.
A gdy uſłyƺeli trzej przyjáćiele Ijobowi to wƺyſtko utrápienie / które náń przypádło / przyƺli káżdy z miejſcá ſwego : Elifáſ Temáńcżyk / y Bildád Suhytcżyk / y Sofár Náámátcżyk ; bo śię byli námówili poſpołu / áby przyƺedłƺy pożáłowáli go / y ćieƺyli go.
A podnióſłƺy ocży ſwoje z dáleká / nie poználi go / y wynióſłƺy głoſ ſwój płákáli / á rozdárłƺy káżdy płáƺcż ſwój miotáli proch nád głowy ſwe ku niebu ;
Y śiedzieli z nim ná ziemi śiedm dni y śiedm nocy / á żáden do niego ſłowá nie przemówił ; bo widzieli / że śię gwáłtownie wzmágáłá boleść jego.
