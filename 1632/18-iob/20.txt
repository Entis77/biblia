A odpowiádájąc Sofár Náámátcżyk rzekł :
Do tego mię myśli moje przywodzą / ábym odpowiedźiáł ; przetożem śię poſpieƺył.
Słyƺáłem mnie háńbiącą nágánę ; ále duch wyrozumieniá mego odpowie zá mię.
Izáż nie wieƺ / że to jeſt od wieku / od tego cżáſu / jáko poſtáwił Bóg cżłowieká ná źiemi?
Iż chwáłá niepobożnych krótká jeſt / á weſele obłudniká ná mgnienie oká?
By też wſtąpiłá áż do niebá hárdość jego / á obłoku śię dotknęłá głowá jego :
A wƺákże ná wieki zginie jáko gnój jego / á ći / którzy go widźieli / rzeką : Gdźież śię podźiáł?
Uleći jáko ſen / á nie znájdą go ; bo ućiecże / jáko widzenie nocne.
Oko / które go widźiáło / nie oglądá go więcej / y nie ujrzy go więcej miejſce jego.
Synowie jego będą śię korzyć ubogim ; bo ręce jego muƺą wrácáć / co wydárł.
Kośći jego nápełnione ſą grzechámi młodośći jego / á w prochu z nim leżeć będą.
A choć złość ſłodnieje w uſtách jego / y tái ją pod językiem ſwoim ;
Kochá śię w niej / á nie opuƺcżá jej / zátrzymywájąc ją w pośrodku podniebieniá ſwego :
Wƺákże pokárm jego we wnętrznośćiách jego odmieni śię ; żółćią pádálcową ſtánie śię w trzewách jego.
Bogáctwá / które pożárł / zwróći / á z brzuchá jego wyżenie je Bóg.
Głowę pádálcową ſſáć będźie ; zábije go język jáƺcżurcży.
Nie oglądá źródeł rzek / ſtrumieni mówię miodu y máſłá.
Wróći prácę cudzą / á nie záżyje jej ; y choć znowu nábędźie wielkich májętnośći / nie ućieƺy śię niemi.
Bo ubogich dręcżył y opuƺcżáł / złupił dom / którego nie budowáł ; przetoż nic ſpokojnego nie pocżuje w żywoćie ſwoim /
Ani rzecży ſwych wdźięcżnych nie będźie mógł zátrzymáć.
Nic nie zoſtánie z pokármów jego / áni śię rozmnoży dobro jego.
Choćby y názbyt miáł wƺyſtkiego / śćiśniony będźie ; wƺelká ręká trápiących oburzy śię náń.
Choć będźie miáł cżem nápełnić brzuch ſwój / przećież náń Bóg puśći popędliwość gniewu ſwego / którą jáko deƺcż ſpuśći náń / y ná pokármy jego.
Gdy ućiekáć będźie przed bronią żelázną / przebije go łuk hártowny.
Wyjętá będźie ſtrzáłá z ſájdáku wypuƺcżoná / á grot przeniknie żółć jego ; á gdy uchodźić będźie / ogárną go ſtráchy.
Wƺyſtkie nieƺcżęśćiá záſádźiły śię náń w tájemnych miejſcách jego / á pożre go ogień nierozdymány : pozoſtáły w przybytku jego utrápiony będźie.
Odkryją niebioſá złość jego / á źiemiá powſtánie przećiwko niemu.
Przenieśie śię urodzáj domu jego ; dobrá jego rozpłyną śię w dźień gniewu jego.
Tenći jeſt dźiáł cżłowieká niepobożnego od Bogá / to dźiedźictwo náznácżone mu od Bogá.
