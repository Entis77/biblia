A odpowiádájąc Sofár Náámátcżyk rzekł :
Do tego mię myśli moje przywodzą / ábym odpowiedziáł ; przetożem śię poſpieƺył.
Słyƺáłem mnie háńbiącą nágánę ; ále duch wyrozumieniá mego odpowie zá mię.
Izáż nie wieƺ / że to jeſt od wieku / od tego cżáſu / jáko poſtáwił Bóg cżłowieká ná ziemi?
Iż chwáłá niepobożnych krótká jeſt / á weſele obłudniká ná mgnienie oká?
By też wſtąpiłá áż do niebá hárdość jego / á obłoku śię dotknęłá głowá jego :
A wƺákże ná wieki zginie jáko gnój jego / á ći / którzy go widzieli / rzeką : Gdzież śię podziáł?
Uleći jáko ſen / á nie znájdą go ; bo ućiecże / jáko widzenie nocne.
Oko / które go widziáło / nie oglądá go więcej / y nie ujrzy go więcej miejſce jego.
Synowie jego będą śię korzyć ubogim ; bo ręce jego muƺą wrácáć / co wydárł.
Kośći jego nápełnione ſą grzechámi młodośći jego / á w prochu z nim leżeć będą.
A choć złość ſłodnieje w uſtách jego / y tái ją pod językiem ſwoim ;
Kochá śię w niej / á nie opuƺcżá jej / zátrzymywájąc ją w pośrodku podniebieniá ſwego :
Wƺákże pokárm jego we wnętrznośćiách jego odmieni śię ; żółćią pádálcową ſtánie śię w trzewách jego.
Bogáctwá / które pożárł / zwróći / á z brzuchá jego wyżenie je Bóg.
Głowę pádálcową ſſáć będzie ; zábije go język jáƺcżurcży.
Nie oglądá źródeł rzek / ſtrumieni mówię miodu y máſłá.
Wróći prácę cudzą / á nie záżyje jej ; y choć znowu nábędzie wielkich májętnośći / nie ućieƺy śię niemi.
Bo ubogich dręcżył y opuƺcżáł / złupił dom / którego nie budowáł ; przetoż nic ſpokojnego nie pocżuje w żywoćie ſwoim /
Ani rzecży ſwych wdzięcżnych nie będzie mógł zátrzymáć.
Nic nie zoſtánie z pokármów jego / áni śię rozmnoży dobro jego.
Choćby y názbyt miáł wƺyſtkiego / śćiśniony będzie ; wƺelká ręká trápiących oburzy śię náń.
Choć będzie miáł cżem nápełnić brzuch ſwój / przećież náń Bóg puśći popędliwość gniewu ſwego / którą jáko deƺcż ſpuśći náń / y ná pokármy jego.
Gdy ućiekáć będzie przed bronią żelázną / przebije go łuk hártowny.
Wyjętá będzie ſtrzáłá z ſájdáku wypuƺcżoná / á grot przeniknie żółć jego ; á gdy uchodzić będzie / ogárną go ſtráchy.
Wƺyſtkie nieƺcżęśćiá záſádziły śię náń w tájemnych miejſcách jego / á pożre go ogień nierozdymány : pozoſtáły w przybytku jego utrápiony będzie.
Odkryją niebioſá złość jego / á ziemiá powſtánie przećiwko niemu.
Przenieśie śię urodzáj domu jego ; dobrá jego rozpłyną śię w dzień gniewu jego.
Tenći jeſt dziáł cżłowieká niepobożnego od Bogá / to dziedzictwo náznácżone mu od Bogá.
