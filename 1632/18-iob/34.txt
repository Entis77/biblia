Nádto mówił Elihu / y rzekł :
Słuchájćież / mądrzy! mów mojich / á náucżeni poſłuchájćie mię.
Bo ucho ſłów doświádcżá / jáko podniebienie ſmákuje pokármu.
Obierzmy ſobie ſąd / á rozeznájmy miedzy ſobą / co jeſt dobrego.
Ponieważ Ijob rzekł : Jeſtem ſpráwiedliwym / á Bóg odrzućił ſpráwę moję :
Y mámże kłámáć / májąc ſpráwiedliwą? Boleſny jeſt poſtrzał mój bez przewinieniá.
Któryż jeſt mąż táki / jáko Ijob / coby pił pośmiewiſko jáko wodę?
A coby chodźił w towárzyſtwie cżyniących niepráwość ; y przeſtáwáłby z ludźmi niepobożnymi?
Bo powiedźiał : Nie pomoże cżłowiekowi / choćby śię podobał Bogu.
Przetoż mię ſłuchájćie / mężowie rozumni! Niech będźie dáleká niepobożność od Bogá / y niepráwość od Wƺechmocnego.
Bo on według ucżynku płáći cżłowiekowi / á według drogi jego káżdemu nágrádzá.
A zgołá Bóg przewrotnie nie cżyni / á Wƺechmocny nie podwrácá ſądu.
Któż go przełożył nád źiemią? á kto wyſtáwił cáły okrąg świátá?
Jeſliby obróćił przećiwko niemu ſerce ſwoje / á duchá jego / y dech jego do śiebie wźiął :
Zginęłoby wƺelkie ćiáło ſpołu / á cżłowiekby śię do prochu náwróćił.
Maƺli tedy rozum / ſłucháj tego / á przyjmuj w uƺy ſwe głos mowy mojey.
Azaż ten / który ma w nienáwiśći ſąd / pánowáć może? ázaż tego / który jeſt wielce ſpráwiedliwy / niepobożnym ucżyniƺ?
Záż potępiƺ tego / który może rzec królowi : O bezecny! á kśiążętom : O niepobożny!
Który nie ma względu ná oſoby kśiążąt / y nie wáży ſobie więcey bogácżá nád ubogiego ; bo oni wƺyſcy ſą cżynem rąk jego.
Nágle umieráją ; á o północy wzruƺony bywa naród / y przemijá / á mocárz znieśiony bywa bez ręki ludzkiej.
Ocży bowiem jego nád drogámi cżłowiecżemi / á on widźi wƺyſtkie kroki jego.
Niemáƺ ćiemnośći / áni ćieniá śmierći / kędyby śię ſkryli ći / którzy cżynią niepráwość.
Bo ná nikogo nie wkłádá więcey / ták / żeby miał wchodźić w ſąd z Bogiem.
Poćierá bárzo wiele mocárzów / á inƺych miáſto nich wyſtáwiá.
Przeto / iż zná ſpráwy ich / obrácá im dźień w noc / áby byli potárći.
Poráżá ich jáko niepobożnych ná miejſcu jáwnem.
Przeto / iż odſtąpili od niego / á żádnych dróg jego zrozumieć nie chćieli :
Aby przywiódł ná nich wołánie znędzniáłych / á pokázał / że wyſłuchuje wołánie ubogich.
Gdy on ſpráwi pokój / któż go wzruƺy? tákże / gdy ſkryje oblicże / któż go ujrzy? A to cżyni ták cáłemu národowi / jáko káżdemu cżłowiekowi /
Aby dálej nie pánował cżłowiek obłudny ná upádek ludzki.
Zápráwdę miáłbyś mówić do Bogá : Przepuść ; ponioſę / á nie będę śię wzbrániał.
Nádto jeſlibym cżego nie bácżył / ty mię náucż ; jeſlim niepráwość popełnił / nie ucżynię tego więcey.
Izali według zdánia twego będźieƺ płáćił / żeć śię to nie podobá / á żeś ty owo obrał / á nie on? Ale wieƺli co lepƺego / powiedz.
Mężowie rozumni toż rzeką ze mną / á cżłowiek mądry przypádnie ná to /
Że Ijob nie mówi mądrze / á ſłowá jego nie ſą roztropne.
Boże / Ojcże mój! niech będźie Ijob doſkonále doświádcżony / przeto / iż nam odpowiádá / jáko ludźiom złym.
Bo przeſtępſtwá przycżyniá do grzechu ſwego ; chlubi śię miedzy námi / y mówi bárzo wiele przećiwko Bogu.
