Nádto mówił Elihu / y rzekł :
Słuchájćież / mądrzy! mów moich / á náucżeni poſłuchájćie mię.
Bo ucho ſłów doświádcżá / jáko podniebienie ſmákuje pokármu.
Obierzmy ſobie ſąd / á rozeznájmy między ſobą / co jeſt dobrego.
Poniewáż Ijob rzekł : Jeſtem ſpráwiedliwym / á Bóg odrzućił ſpráwę moję :
Y mámże kłámáć / májąc ſpráwiedliwą? Boleſny jeſt poſtrzáł mój bez przewinieniá.
Któryż jeſt mąż táki / jáko Ijob / coby pił pośmiewiſko jáko wodę?
A coby chodźił w towárzyſtwie cżyniących niepráwość ; y przeſtáwáłby z ludźmi niepobożnymi?
Bo powiedźiáł : Nie pomoże cżłowiekowi / choćby śię podobáł Bogu.
Przetoż mię ſłuchájćie / mężowie rozumni! Niech będźie dáleká niepobożność od Bogá / y niepráwość od Wƺechmocnego.
Bo on według ucżynku płáći cżłowiekowi / á według drogi jego káżdemu nágrádzá.
A zgołá Bóg przewrotnie nie cżyni / á Wƺechmocny nie podwrácá ſądu.
Któż go przełożył nád źiemią? á kto wyſtáwił cáły okrąg świátá?
Jeźliby obróćił przećiwko niemu ſerce ſwoje / á duchá jego / y dech jego do śiebie wźiął :
Zginęłoby wƺelkie ćiáło ſpołu / á cżłowiekby śię do prochu náwróćił.
Máƺli tedy rozum / ſłucháj tego / á przyjmuj w uƺy ſwe głoſ mowy mojej.
Azáż ten / który má w nienáwiśći ſąd / pánowáć może? ázáż tego / który jeſt wielce ſpráwiedliwy / niepobożnym ucżyniƺ?
Záż potępiƺ tego / który może rzec królowi : O bezecny! á kśiążętom : O niepobożny!
Który nie má względu ná oſoby kśiążąt / y nie wáży ſobie więcej bogácżá nád ubogiego ; bo oni wƺyſcy ſą cżynem rąk jego.
Nágle umieráją ; á o północy wzruƺony bywá náród / y przemijá / á mocárz znieśiony bywá bez ręki ludzkiej.
Ocży bowiem jego nád drogámi cżłowiecżemi / á on widźi wƺyſtkie kroki jego.
Niemáƺ ćiemnośći / áni ćieniá śmierći / kędyby śię ſkryli ći / którzy cżynią niepráwość.
Bo ná nikogo nie wkłádá więcej / ták / żeby miáł wchodźić w ſąd z Bogiem.
Poćierá bárdzo wiele mocárzów / á inƺych miáſto nich wyſtáwiá.
Przeto / iż zná ſpráwy ich / obrácá im dźień w noc / áby byli potárći.
Poráżá ich jáko niepobożnych ná miejſcu jáwnem.
Przeto / iż odſtąpili od niego / á żádnych dróg jego zrozumieć nie chćieli :
Aby przywiódł ná nich wołánie znędzniáłych / á pokázáł / że wyſłuchuje wołánie ubogich.
Gdy on ſpráwi pokój / któż go wzruƺy? tákże / gdy ſkryje oblicże / któż go ujrzy? A to cżyni ták cáłemu národowi / jáko káżdemu cżłowiekowi /
Aby dálej nie pánowáł cżłowiek obłudny ná upádek ludzki.
Zápráwdę miáłbyś mówić do Bogá : Przepuść ; ponioſę / á nie będę śię wzbrániáł.
Nádto jeźlibym cżego nie bácżył / ty mię náucż ; jeźlim niepráwość popełnił / nie ucżynię tego więcej.
Izáli według zdániá twego będźieƺ płáćił / żeć śię to nie podobá / á żeś ty owo obráł / á nie on? Ale wieƺli co lepƺego / powiedz.
Mężowie rozumni toż rzeką ze mną / á cżłowiek mądry przypádnie ná to /
Że Ijob nie mówi mądrze / á ſłowá jego nie ſą roztropne.
Boże / Ojcże mój! niech będźie Ijob doſkonále doświádcżony / przeto / iż nám odpowiádá / jáko ludźiom złym.
Bo przeſtępſtwá przycżyniá do grzechu ſwego ; chlubi śię między námi / y mówi bárdzo wiele przećiwko Bogu.
