Y odpowiedziáł Sofár Náámátcżyk / y rzekł :
Izáż śię nie godzi ná wiele ſłów odpowiedzieć? Albo izáli mąż wielomowny będzie uſpráwiedliwiony?
Bedąż ná twoje plotki ludzie milcżeć? A gdy ty ſobie przeƺydzáƺ / ćiebie nikt nie záwſtydzi?
Abowiemeś powiedziáł : Cżyſtá jeſt náuká mojá / á jeſtem cżyſtym przed ocżymá twemi.
Ale gdyby Bóg chćiáł mówić / y otworzyć uſtá ſwoje przećiwko tobie :
Tedyćby objáwił tájemnice mądrośći / żeś dwá kroć więkƺe káránie nádto záſłużył ; przetoż uznáj / że ćię Bóg przebácżył dlá niepráwośći twojej.
Izáli tájemnice Boże wybádáƺ? álbo doſkonáłośći Wƺechmocnego dośćignieƺ?
Wyżƺe ſą niż niebioſá / cóż ucżyniƺ? Głębƺe niż piekło / jákoż poznáƺ?
Dłużƺá miárá ich / niż ziemiá / á ƺerƺá / niż morze.
Jeźli wypełni / álbo jeźli záwrze / álbo jeźli w jedno śćiśnie / któż go záwśćiągnie?
Abowiem on zná márność ludzką / y widzi niepráwość ; á nie miáłby tego bácżyć?
Cżłowiek nierozumny nábywá rozumu / choć śię jáko źrebię leśnego oſłá rodzi cżłowiek.
Jeźli ty przygotujeƺ ſerce twoje / á wyćiągnieƺ do niego ręce twoje ;
Jeźliż niepráwość jeſt w ręce twej oddál ją / á mieƺkáć nie dopuƺcżáj niepráwośći w przybytkách twoich ;
Tedy podnieśieƺ oblicże twoje bez zmázy / á będzieƺ ſtáły / y nie będzieƺ śię báł.
Abowiem zápomniƺ kłopotu / á jáko wody / które pominęły / wſpomináć go będzieƺ.
Y nád południe jáśniejƺy náſtánie cżáſ twój ; záćmiƺli śię / będzieƺ jáko záránek.
Y będzieƺ ufáł / májąc nádzieję / á jáko w okopách bezpiecżnie ſpáć będzieƺ.
Będzieƺ leżáł / á nikt ćię nie przeſtráƺy ; y uniżáć śię będą przed twárzą twoją wiele ich.
Ale ocży niepobożnych uſtáną y ućiecżká ich zginie / á nádziejá ich będzie jáko wyjśćie duƺy z cżłowieká.
