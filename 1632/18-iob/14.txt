Cżłowiek / národzony z niewiáſty / dni krótkich jeſt / y pełen kłopotów ;
Wyráſtá jáko kwiát / y bywá podćięty / á ućieká jáko ćień / y nie oſtoi śię.
Wƺákże y ná tákiego otwieráƺ ocży twoje / á przywodźiƺ mię do ſądu z ſobą.
Któż pokáże cżyſtego z niecżyſtego? Ani jeden ;
Gdyż zámierzone ſą dni jego / licżbá mieśięcy jego u ćiebie ; zámierzyłeś mu kreſ / którego nie może przeſtąpić.
Odſtąpże od niego / áż odpocżnie / áż przejdźie jáko nájemnicży dźień jego.
Abowiem y o drzewie jeſt nádźiejá / choć je wytną / że śię jeƺcże odmłodźi / á látorośl jego nie uſtánie.
Choć śię zſtárzeje w źiemi korzeń jego / y w prochu obumrze pień jego :
Wƺákże gdy ucżuje wilgotność / puśći śię / y rozpuśći gáłęźie / jáko ƺcżep młody.
Ale cżłowiek umierá / zemdlony będąc / á umárłƺy cżłowiek gdźież jeſt?
Jáko uchodzą wody z morzá / á rzeká opádá y wyſychá :
Ták cżłowiek / gdy śię ukłádźie / nie wſtánie więcej / á pokąd ſtoją niebá / nie ocući śię / áni będźie obudzony ze ſnu ſwego.
Obyżeś mię w grobie ukrył y utáił / áżby śię ućiƺył gniew twój / á iżbyś mi zámierzył kreſ / kędy chceƺ wſpomnieć ná mię!
Gdy umrze cżłowiek / izáli żyć będźie? Po wƺyſtkie dni wymierzonego cżáſu mego będę ocżekiwáł przyƺłej odmiány mojej.
Záwołáƺ / á já tobie odpowiem ; á ſpráw rąk twoich pożądáƺ.
Acżkolwiekeś teráz kroki moje oblicżył / áni odwłócżyƺ kárániá zá grzech mój.
Zápięcżętowáne jeſt w wiązánce przeſtępſtwo moje / á zgromádzáƺ niepráwośći moje.
Práwdźiwie jáko górá pádłƺy rozſypuje śię / á ſkáłá przenośi śię z miejſcá ſwego.
Jáko wodá wydrążá kámienie / á powodźią záláne bywá / co ſámo od śiebie rośnie z prochu źiemi : ták nádźieję ludzką w niwecż obrácáƺ.
Przemágáƺ go uſtáwicżnie / á on ſchodźi ; odmieniáƺ poſtáć jego / y wypuƺcżáƺ go.
Będąli zácni ſynowie jego / tego on nie wie ; jeźli też wzgárdzeni / on nie bácży.
Tylko ćiáło jego / póki żyw / boleje / á duƺá jego w nim kwili.
