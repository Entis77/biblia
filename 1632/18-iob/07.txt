Izáli cżáſ nie jeſt zámierzony cżłowiekowi ná ziemi? á jáko dni nájemnicże nie ſą dni jego?
Jáko ſługá prágnie ćieniá / á jáko nájemnik cżeká końcá prácy ſwojej :
Tákiem já práwem dziedzicżnem wziął mieśiące próżne / á nocy boleśne ſą mi náznácżone.
Ukłádęli śię / tedy mówię : Kiedyż wſtánę? á rychło pominie noc? y pełen bywám myśleniá áż do świtániá.
Oblecżone jeſt ćiáło moje w robáki y w plugáſtwo z prochu ; ſkórá mojá popádáłá śię / y rozśiádłá śię.
Dni moje prędƺe ſą / niż cżółnek tkácki / y ſtráwione ſą bez nádziei.
Wſpomnij / o Pánie! iż wiátrem jeſt żywot mój / nie wróći śię oko moje / áby widziáło dobre rzecży.
Ani mię oglądá oko / które mię widywáło ; ocży twoje obrócone będą ná mię / á mnie nie będzie.
Jáko niƺcżeje obłok y przemijá / ták zſtępujący do grobu nie wynijdzie ;
Nie wróći śię więcej do domu ſwego / áni go więcej pozná miejſce jego.
Przetoż já nie mogę záwśćiągnąć uſt moich ; mówić będę w utrápieniu duchá mego / będę rozmáwiáł w gorzkośći duƺy mojej.
Izáżem já jeſt morze / álbo wieloryb / żeś mię oſádził ſtráżą?
Gdym rzekł : Poćieƺy mię łoże moje / y ulży mi nárzekániá mego pośćiel mojá :
Tedy mię ſtráƺyƺ przez ſny / y przez widzeniá trwożyƺ mną.
A przetoż obráłá ſobie powieƺenie duƺá mojá / á śmierć rácżej / niż zoſtáć w kośćiách.
Sprzykrzyłem ſobie żywot / nie wiecżnie będę żyw. Zániechájże mię / bo márnośćią ſą dni moje.
Cóż jeſt cżłowiek / że go ták wielce wáżyƺ? á że przykłádáƺ ku niemu ſerce twoje?
A że go náwiedzáƺ ná káżdy záránek? y ná káżdą chwilę doświádcżáƺ go?
Pokądże śię nie odwróćiƺ odemnie? á nie zániecháƺ mię / áżbym przełknął ślinę moję?
Zgrzeƺyłem / cóż mám cżynić? o ſtróżu ludzki! cżemuś mię ſobie zá cel położył / ábym był ſám ſobie ćiężárem?
Przecż nie odejmieƺ przeſtępſtwá mego / y nie przepuśćiƺ niepráwośći mojej? Bo śię teráz w prochu położę / á choćbyś mię ƺukáł ráno / nie będzie mię.
