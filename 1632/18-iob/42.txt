Tedy odpowiedźiáł Ijob Pánu / y rzekł :
Wiem / że wƺyſtko możeƺ / y nie może być záhámowány zámyſł twój.
Któż jeſt ten / pytáƺ / który záćiemniá rádę Bożą nieumiejętnie? Dlátego przyznáję / żem nie zrozumiáł ; dźiwniejƺe ſą te rzecży / niżbym je mógł pojąć y zrozumieć.
Wyſłuchájże / proƺę / gdybym mówił ; á gdy śię będę pytáł / oznájmijże mi.
Przedtem tylko ucho ſłyƺáło o tobie ; ále teraz oko moje widźi ćię.
Przetoż żáłuję y pokutuję w prochu y w popiele.
A gdy odmówił Pán te ſłowá do Ijobá / rzekł Pán do Elifáſá Támáńcżyká : Rozpálił śię gniew mój przećiw tobie / y przećiw dwom przyjáćiołom twoim / żeśćie o mnie nie mówili ták przyſtojnie / jáko Ijob / ſługá mój.
Przetoż teraz / weźmijćie ſobie śiedm ćielców / y śiedm báránów / á idźćie do ſługi mego Ijobá / y ofiárujćie cáłopálenie zá śię ; á Ijob / ſługá mój / niech śię modli zá wámi ; bo oblicże jego przyjmę / ábym nie ucżynił z wámi według głupſtwá wáƺego ; bośćie nie mówili ták przyſtojnie o mnie / jáko Ijob / ſługá mój.
A ták odeƺli Elifáſ Temáńcżyk / y Bildád Suhitcżyk / y Sofár Náámátcżyk / y ucżynili / jáko im rozkázáł Pán ; y przyjął Pán oblicże Ijobowe.
Zátem Pán przywróćił to / co było pobráne Ijobowi / gdy śię modlił zá przyjáćiół ſwoich ; y rozmnożył Pán wƺyſtko / cokolwiek miał Ijob / w dwójnáſób.
Zeƺli śię tedy do niego wƺyſcy bráćia jego / y wƺyſtkie śioſtry jego / y inni wƺyſcy / którzy go przedtem ználi / y jedli z nim chleb w domu jego / á żáłując go ćieƺyli go z ſtrony wƺyſtkiego złego / które był Pán náń przywiódł ; y dáł mu káżdy z nich upominek jeden / y káżdy náuƺnicę złotą jednę.
A ták Pán błogoſłáwił oſtátnim cżáſom Ijobowym / więcey niż pocżątkom jego. Bo miał cżternáśćie tyśięcy owiec / y ƺeść tyśięcy wielbłądów / y tyśiąc járzm wołów / y tyśiąc oślic.
Miał też śiedm ſynów / y trzy córki.
Y dáł imię pierwƺej Jeminá / á imię drugiej Kietzyjá / á imię trzećiej Kierenháppuch.
A nie znájdowáły śię niewiáſty ták piękne / jáko córki Ijobowe / we wƺyſtkiej oney źiemi ; y dáł im ojćiec ich dźiedźictwo między bráćmi ich.
Potem Ijob żył ſto y cżterdźieśći lát / y oglądáł ſynów ſwych / y ſynów ſynów ſwoich / áż do cżwártego pokoleniá.
A umárł Ijob / będąc ſtárym y dni ſytym.
