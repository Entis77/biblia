Niemáƺ ták śmiáłego / coby go obudźił ; owƺem któż śię ſtáwi przed twárzą moją?
Któż mi co dáł / ábym mu oddáł? cokolwiek jeſt pod wƺyſtkiem niebem / moje jeſt.
Nie zámilcżę cżłonków jego / áni śilnej mocy jego / á grzecżnego kƺtáłtu jego.
Któż odkryje wierzch odźieniá jego? z dwoiſtemi wędźidłámi ſwemi któż przyſtąpi do niego?
Wrotá gęby jego któż otworzy? bo ſtrách około zębów jego.
Łuſki jego mocne jáko tárcże / bárdzo śćiśle ſpojone.
Jedná z drugą ták ſpojoná / że wiátr nie wchodźi między nie.
Jedná do drugiej przylgnęłá / ujęły śię / á nie dźielą śię.
Kichánie jego cżyni bláſk / á ocży jego ſą jáko powieki zorzy.
Z uſt jego lámpy wychodzą / á iſkry ogniſte wyrywáją śię.
Z nozdrzy jego wychodźi dym / jáko z gárncá wrzącego / álbo kotłá.
Dech jego węgle rozpálá / á płomień z uſt jego wychodźi.
W ƺyi jego przemieƺkuje moc / á boleść przed nim ućieká.
Sztuki ćiáłá jego ſpoiły śię / cáłowite ſą w nim / że śię nie poruƺá.
Serce jego twárde jáko kámień / ták twárde / jáko ƺtuká ſpodniego kámieniá młyńſkiego.
Gdy śię podnośi / drżą mocárze / á od ſtráchu ocżyƺcżáją śię.
Miecż / który go śięgá / nie oſtoi śię / áni drzewce / áni ſtrzáłá / áni páncerz.
Żelázo pocżytá ſobie zá plewę / á miedź zá drzewo zbótwiáłe.
Nie upłoƺy go ſtrzáłá / á jáko źdźbło ſą u niego kámienie z procy.
Strzelbę ſobie pocżytá jáko ſłomę / á pośmiewá śię z ƺermowániá włócżnią.
Pod nim ſą oſtre ſkorupy ; śćiele ſobie ná rzecżách oſtrych jáko ná błoćie.
Cżyni / że wre głębokość jáko gárniec / á że śię mąći morze jáko w moźdźierzu.
Zá ſobą jáſną śćieƺkę cżyni / ták / że śię zdáje / iż przepáść má śiwiznę.
Niemáƺ ná źiemi równego mu / który ták ſtworzony jeſt / że śię nicżego nie boi.
Wƺelką rzecż wyſoką lekce wáży ; on jeſt królem nád wƺyſtkiemi ſrogiemi zwierzętámi.
