Ucżyniłem przymierze z ocżymá ſwemi / ábym nie pomyślał o pánnie.
Bo cóż zá dźiał od Bogá z góry? á co zá dźiedźictwo Wƺechmocnego z wyſokośći?
Azaż nie nágotowáne zginienie złośnikom / á ſrogá pomſtá cżyniącym niepráwość?
Azaż on nie widźi dróg mojich / á wƺyſtkich kroków mojich nie licży?
Jeſlim chodźił w kłámſtwie / á ſpieƺyłá śię ná zdrádę nogá mojá :
Niech mię zwáży ná wádze ſpráwiedliwej / á niech Bóg pozná ƺcżerość moję.
Jeſliż uſtąpiłá nogá mojá z drogi / á zá ocżymá memi ƺłoli ſerce moje / y do rąk mojich jeſliż przylgnęłá jáká zmázá :
Tedy niechże ja śieję / á inƺy niech pożywá / á moje látorośle niech będą wykorzenione.
Jeſli zwiedźione jeſt ſerce moje do niewiáſty / y jeſlim cżyhał u drzwi przyjáćielá mego :
Niechájże mele innemu żoná mojá / á niecháj śię nád nią inni ſchyláją.
Boć to jeſt ſproſny wyſtępek / á niepráwość oſądzeniá godná /
Gdyż ten ogień áż do zátráceniá pożerá / á dochody moje wƺyſtkie wykorzenić może.
Jeſlim ſtronił od ſądu z ſługą moim / álbo z ſłużebnicą moją / gdy ze mną ſprzecżkę mieli /
( Bo cóżbym cżynił / gdyby powſtał Bóg? álbo gdyby pytał / cobym mu odpowiedźiał?
Izáż nie ten / który mię w żywoćie ucżynił / nie ucżynił też y onego? á nie onże nas ſám w żywoćie wykƺtáłtował? )
Jeſliżem odmówił ubogim / cżego chćieli / á ocży wdowy jeſliżem záſmućił ;
Jeſliżem jádł ƺtucżkę ſwoję ſám / á nie jádáłá y śierotá z niej ;
( Abowiem śierotá z młodośći mojey roſłá ze mną / jáko u ojcá ; á jákom wyƺedł z żywotá mátki mojey / byłem wdowie zá wodzá. )
Jeſliżem widźiał kogo ginącego dla tego / że ƺáty nie miał / á nie dáłem żebrákowi odźieniá ;
Jeſliże mi nie błogoſłáwiły biodrá jego / że śię wełną owiec mojich zágrzał ;
Jeſliżem podnióſł przećiwko śieroćie rękę ſwoję / gdym widźiał w bramie pomoc moję :
Tedy niech odpádnie łopátká mojá od plec ſwych / á rámię moje z ſtáwu ſwego niech wytrácone będźie.
Abowiem lękáłem śię ſkruƺeniá od Bogá / á przed jego zácnośćią nie mógłbym śię oſtáć.
Jeſlim pokłádał w złoćie nádźieję moję / á do bryły złotá máwiáłem : Tyś ufánie moje ;
Jeſlim śię weſelił z wielu bogáctw mojich / á iż wiele nábyłá ręká mojá ;
Jeſlim pátrzał ná świátłość ſłońcá / gdy świećiło / á ná mieśiąc / gdy wſpániáło chodźił ;
Y dáło śię uwieść potájemnie ſerce moje / á cáłowáły rękę moję uſtá moje :
Y toćby byłá niepráwość oſądzeniá godná ; bobym śię tym záprzał Bogá z wyſokośći.
Jeſliżem śię weſelił z upádku nienáwidzącego mię / á jeſlim śię ćieƺył / gdy mu śię źle powodźiło.
( Y owƺem nie dáłem zgrzeƺyć uſtom moim / ábym miał żądáć przeklęſtwá duƺy jego. )
Azaż nie máwiáli domownicy moi : Oby nam kto dał mięſá tego / nie możemy śię y nájeść?
Bo gość nie nocował ná dworze / á drzwi moje otwieráłem podróżnemu.
Jeſlim zákrywał / jáko ludźie zwykli / przeſtępſtwá moje / y chowáłem w ſkrytośći mojey niepráwość moję ;
Y choćbym był mógł potłumić zgráję wielką / jednák y nájpodlejƺy z domu uſtráƺył mię ; przetożem milcżał / y nie wychodźiłem ze drzwi.
Obym miał kogo / coby mię wyſłuchał ; ále oto ten jeſt znák mój / że Wƺechmogący ſám odpowie zá mię / y kśięgá / którą nápiſał przećiwnik mój.
Cżylibym jey ná rámieniu ſwojim nie nośił? á nie przywiązáłbym jey ſobie miáſto korony?
Licżbę kroków mojich oznájmiłbym mu ; jáko do kśiążęćiá przyſtąpiłbym do niego.
Jeſliż przećiw mnie źiemiá mojá wołáłá / á jeſliże z nią ſpołem zágony jey płákáły ;
Jeſliżem pożytków jey używał bez pieniędzy / y jeſlim do wzdychánia przywodźił dźierżáwców jey :
Miáſto pƺenicy niech wznijdźie oſet / á miáſto jęcżmieniá kąkol. Tu śię ſkońcżyły ſłowá Ijobowe.
