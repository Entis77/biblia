Ucżyniłem przymierze z ocżymá ſwemi / ábym nie pomyśláł o pánnie.
Bo cóż zá dziáł od Bogá z góry? á co zá dziedzictwo Wƺechmocnego z wyſokośći?
Azáż nie nágotowáne zginienie złośnikom / á ſrogá pomſtá cżyniącym niepráwość?
Azáż on nie widzi dróg moich / á wƺyſtkich kroków moich nie licży?
Jeźlim chodził w kłámſtwie / á ſpieƺyłá śię ná zdrádę nogá mojá :
Niech mię zwáży ná wádze ſpráwiedliwej / á niech Bóg pozná ƺcżerość moję.
Jeźliż uſtąpiłá nogá mojá z drogi / á zá ocżymá memi ƺłoli ſerce moje / y do rąk moich jeźliż przylgnęłá jáká zmázá :
Tedy niechże já śieję / á inƺy niech pożywá / á moje látorośle niech będą wykorzenione.
Jeźli zwiedzione jeſt ſerce moje do niewiáſty / y jeźlim cżyháł u drzwi przyjáćielá mego :
Niechájże mele innemu żoná mojá / á niecháj śię nád nią inni ſchyláją.
Boć to jeſt ſproſny wyſtępek / á niepráwość oſądzeniá godná /
Gdyż ten ogień áż do zátráceniá pożerá / á dochody moje wƺyſtkie wykorzenić może.
Jeźlim ſtronił od ſądu z ſługą moim / álbo z ſłużebnicą moją / gdy ze mną ſprzecżkę mieli /
( Bo cóżbym cżynił / gdyby powſtáł Bóg? álbo gdyby pytáł / cobym mu odpowiedziáł?
Izáż nie ten / który mię w żywoćie ucżynił / nie ucżynił też y onego? á nie onże náſ ſám w żywoćie wykƺtáłtowáł? )
Jeźliżem odmówił ubogim / cżego chćieli / á ocży wdowy jeźliżem záſmućił ;
Jeźliżem jádł ƺtucżkę ſwoję ſám / á nie jádáłá y śierotá z niej ;
( Abowiem śierotá z młodośći mojej roſłá ze mną / jáko u ojcá ; á jákom wyƺedł z żywotá mátki mojej / byłem wdowie zá wodzá. )
Jeźliżem widziáł kogo ginącego dlá tego / że ƺáty nie miáł / á nie dáłem żebrákowi odzieniá ;
Jeźliże mi nie błogoſłáwiły biodrá jego / że śię wełną owiec moich zágrzáł ;
Jeźliżem podnióſł przećiwko śieroćie rękę ſwoję / gdym widziáł w brámie pomoc moję :
Tedy niech odpádnie łopátká mojá od plec ſwych / á rámię moje z ſtáwu ſwego niech wytrácone będzie.
Abowiem lękáłem śię ſkruƺeniá od Bogá / á przed jego zácnośćią nie mógłbym śię oſtáć.
Jeźlim pokłádáł w złoćie nádzieję moję / á do bryły złotá máwiáłem : Tyś ufánie moje ;
Jeźlim śię weſelił z wielu bogáctw moich / á iż wiele nábyłá ręká mojá ;
Jeźlim pátrzáł ná świátłość ſłońcá / gdy świećiło / á ná mieśiąc / gdy wſpániáło chodził ;
Y dáło śię uwieść potájemnie ſerce moje / á cáłowáły rękę moję uſtá moje :
Y toćby byłá niepráwość oſądzeniá godná ; bobym śię tem záprzáł Bogá z wyſokośći.
Jeźliżem śię weſelił z upádku nienáwidzącego mię / á jeźlim śię ćieƺył / gdy mu śię źle powodziło.
( Y owƺem nie dáłem zgrzeƺyć uſtom moim / ábym miáł żądáć przeklęſtwá duƺy jego. )
Azáż nie máwiáli domownicy moi : Oby nám kto dáł mięſá tego / nie możemy śię y nájeść?
Bo gość nie nocowáł ná dworze / á drzwi moje otwieráłem podróżnemu.
Jeźlim zákrywáł / jáko ludzie zwykli / przeſtępſtwá moje / y chowáłem w ſkrytośći mojej niepráwość moję ;
Y choćbym był mógł potłumić zgráję wielką / jednák y nájpodlejƺy z domu uſtráƺył mię ; przetożem milcżáł / y nie wychodziłem ze drzwi.
Obym miáł kogo / coby mię wyſłucháł ; ále oto ten jeſt znák mój / że Wƺechmogący ſám odpowie zá mię / y kśięgá / którą nápiſáł przećiwnik mój.
Cżylibym jej ná rámieniu ſwojim nie nośił? á nie przywiązáłbym jej ſobie miáſto korony?
Licżbę kroków moich oznájmiłbym mu ; jáko do kśiążęćiá przyſtąpiłbym do niego.
Jeźliż przećiw mnie ziemiá mojá wołáłá / á jeźliże z nią ſpołem zágony jej płákáły ;
Jeźliżem pożytków jej używáł bez pieniędzy / y jeźlim do wzdychániá przywodził dzierżáwców jej :
Miáſto pƺenicy niech wznijdzie oſet / á miáſto jęcżmieniá kąkol. Tu śię ſkońcżyły ſłowá Ijobowe.
