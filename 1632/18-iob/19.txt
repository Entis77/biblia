A odpowiádájąc Ijob rzekł :
Dokądże trápić będźiećie duƺę moję / á náćieráć ná mię mowámi ſwemi?
Już dźieśięćkroć záwſtydźiliśćie mię / y nie wſtydże was / że śię ták zátwárdzáćie przećiwko mnie?
A niech ták będźie / żem zbłądźił ; przy mnie zoſtánie błąd mój.
A jeſli śię przećiw mnie wynośićie / á obwiniáćie mię poháńbieniem mojim /
Wiedzćież / żeć mię Bóg podwróćił / y śiećią ſwoją obtocżył mię.
Oto / wołámli o krzywdę / nie bywám wyſłuchány ; krzycżęli / niemáƺ ſądu.
Drogę moję zágrodźił / żebym przejść nie mógł / á ná śćieƺce mojey ćiemnośći położył.
Z ſłáwy mojey złupił mię / y zdjął koronę z głowy mojey.
Popſuł mię zewƺąd / ábym záginął / á wyrwał jáko drzewo nádźieję moję.
Nádto zápálił śię ná mię gniew jego / á policżył mię w pocżet nieprzyjáćiół ſwojich.
Przyƺły rázem hufy jego / y utorowáły przećiwko mnie drogę ſwoję / y obległy w około námiot mój.
Bráći mojich odemnie oddálił / á znájomi moi ſtronią odemnie.
Opuśćili mię bliſcy moi / á znájomi moi zápomnieli mię.
Komornicy domu mego / y ſłużebnice moje máją mię zá obcego / cudzoźiemcem ſtałem śię w ocżách ich.
Wołámli ná ſługę mego / nie ozywá mi śię / choćiáż go proƺę uſtámi memi.
Tchem moim brzydźi śię żoná mojá / choć proƺę przez ſynów żywotá mego.
Y nájlichśi pogárdzáją mną / á gdy powſtáję / urągáją mi.
Brzydzą śię mną wƺyſcy nájwierniejśi moi / á którychem umiłował / ſtáli mi śię przećiwnymi.
Do ſkóry mojey / jáko do ćiáłá mego przyſchłá kość mojá ; ſkórá tylko zoſtáłá około zębów mojich.
Zmiłujćie śię nádemną / zmiłujćie śię nádemną / wy przyjáćiele moi! bo ręká Bożá dotknęłá mię.
Cżemuż mię prześládujećie / jáko Bóg / á ćiáłá mego nie możećie śię náſyćić?
Oby teraz nápiſáne były ſłowá moje! oby je ná kśięgách wyryſowano!
Oby rylcem żeláznym y ołowiem ná wiecżną pámiątkę ná kámieniu wydrążone były!
Acżći ja wiem / iż Odkupićiel mój żyje / á iż w oſtátecżny dźień nád prochem ſtánie.
A choć tá ſkórá mojá roztocżoná będźie / przećież w ćiele mojim oglądám Bogá ;
Którego ja ſám oglądám / y ocży moje ujrzą go / á nie inny ; choć zniƺcżáły nerki moje we wnętrznośćiách mojich.
Przecżże nie mówićie : Cżemuż go prześládujemy? gdyż śię przy mnie znájduje grunt dobrej ſpráwy.
Ulęknijćie śię ſámi miecżá / bo pomſtá niepráwośći jeſt miecż ; á wiedzćie / że będźie ſąd.
