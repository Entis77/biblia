Nádto mówił Elihu / y rzekł :
Y mniemáƺ / żeś to z rozſądkiem rzekł : Spráwiedliwość mojá przechodźi Boſką?
Boś powiedźiáł : Cóż mi pomoże? á co wezmę zá pożytek / choćbym nie grzeƺył?
Ale já tobie dowodnie odpowiem / y towárzyƺom twoim z tobą.
Spojrzyj w niebo / á obácż ; przypátrz śię obłokom / jáko ſą wyżƺe nád ćię.
Jeźli zgrzeƺyƺ / cóż ucżyniƺ przećiwko niemu? á jeźliby były rozmnożone niepráwośći twoje / cóż mu ucżyniƺ?
Jeźlibyś był ſpráwiedliwym / cóż mu dáƺ? álbo cóż weźmie z ręki twojej?
Cżłowiekowi podobnemu tobie niezbożność twojá záƺkodźi / á ſynowi cżłowiecżemu pomoże ſpráwiedliwość twojá.
Z mnóſtwá ućiśnionych / którzy do tego przywiedźieni ſą ; áby nárzekáli y wołáli dlá rámieniá mocárzów /
Żáden nie mówi : Gdźież jeſt Bóg / ſtworzyćiel mój / choć on dáje śpiewánie y w nocy?
Choć náſ wyucżá nád bydlętá źiemſkie / á nád ptáſtwo niebieſkie cżyni náſ mędrƺymi.
Tedy wołájąli dlá hárdośći złych / on ich nie wyſłuchuje.
Bo obłudy nie wyſłuchá Bóg / á Wƺechmocny nie pátrzy ná nich.
Dopieroż nie wyſłuchá ćiebie / poniewáż mówiƺ : Nie widźiƺ tego ; oſądźże śię przed nim / á ocżekuj go /
Gdyż ćię jedno trochę náwiedźił gniew jego / jákoby nie wiedźiáł wielkośći grzechów twoich.
Przetoż Ijob próżno otwierá uſtá ſwe / á bez umiejętnośći rozmnáżá ſłowá ſwoje.
