Cżemuż od Wƺechmocnego nie ſą zákryte cżáſy? á którzy go znáją / nie widzą dni jego?
Niezbożni gránice przenoƺą / trzody zábieráją y páſą.
Oſłá śierotek zájmują / á wołu od wdowy w záſtáwie biorą.
Spycháją ubogich z drogi ; ſpólnie śię muƺą nędzni kryć ná źiemi.
Oto jáko leśne oſły w puƺcżách wychodzą ná robotę ſwoję / wſtáwájąc ráno ná łupieſtwo ; puſtyniá jeſt chlebem ich / y dźieći ich.
Ná polu ubogiego pożynáją zboże / á niepobożni z winnic zbieráją.
Nágich nocowáć przymuƺáją bez odźieniá / którzy śię nie máją cżym nákryć ná źimnie.
Powodźią gór zmácżáni bywáją / nie májąc mieƺkánia przytuláją śię do ſkáły.
Porywáją śierotkę od pierśi / á od ubogiego biorą záſtáw.
Nágiemu dopuƺcżáją chodźić bez odźieniá / á o głodźie chowáją tych / którzy ich ſnopy noƺą.
A ći / którzy miedzy murámi ich wyćiſkáją oliwę y práſy tłocżą / prágną.
Ludźie w mieśćie wzdycháją / á duƺe zábitych wołáją / á Bóg temu wſtrętu nie cżyni.
Cić to ſą / którzy śię ſprzećiwiáją świátłośći / á nie znáją dróg jey / áni ſtánęli ná śćieƺkách jey.
Rániucżko wſtáje mężobójcá / zábijá ubogiego y niedoſtátecżnego / á w nocy jeſt jáko złodźiej.
Oko cudzołożniká pilnuje zmierzku / mówiąc : Nie ujrzy mię nikt ; y zákrywá oblicże ſwe.
Podkopywáją w ćiemnośći domy / które ſobie náznácżyli / y nienáwidzą świátłá.
Ale záránek jeſt im jáko ćień śmierći ; jeſli ich kto pozná / przypádá ná nich ſtrách ćieniá śmierći.
Lekkimi ſą ná wodách ; przeklęty dźiał ich ná źiemi ; nie pátrzą ná drogę wolną.
Jáko ſuƺá y gorącość tráwią wody śnieżne / ták grób grzeƺników.
Zápominá go żywot mátki jego / á robák ſłodkość z niego cżuje ; niemáƺ więcey pámiątki jego / á niepráwość połámáná jeſt jáko drzewo.
Roztrącá niepłodną / która nierodźiłá / á wdowie nie cżyni dobrze.
Poćiągá też mocárzy możnośćią ſwoją : á gdy ná nich powſtał / zwątpili o żywoćie ſwoim.
Dáje mu Bóg / ná cżemby bezpiecżnie ſpolegáć mógł : wƺákże ocży jego pátrzą ná drogi ich.
Ná chwilę wywyżƺeni ſą / álić ich niemáƺ ; zniżeni y śćiśnieni będą jáko inni wƺyſcy / á jáko wierzch kłoſá śćięći będą.
A jeſli nie ták jeſt / gdźież jeſt ten / coby mi zádał kłámſtwo / á coby obróćił wniwecż ſłowá moje?
