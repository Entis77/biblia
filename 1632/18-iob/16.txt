Ale odpowiedziáł Ijob / y rzekł :
Słyƺáłem tákowych rzecży wiele ; przykrymi ćieƺyćielámi wy wƺyſcy jeſteśćie.
Y kiedyż będzie koniec tym próżnym ſłowom? álbo co ćię przymuƺá / że ták odpowiádáƺ?
Azáżbym já ták mówił / jáko wy / gdybyśćie wy byli ná miejſcu mojem? ázáżbym zbieráł przećiwko wám ſłowá / y kiwáłlibym nád wámi głową ſwoją?
Owƺembym wáſ pośiláł uſtámi memi / á ruchánie wárg moich ulżyłoby boleśći wáƺych.
Ale jeźli będę mówił / przećież śię nie ukoi boleść mojá ; á jeźli też przeſtánę / izáż odejdzie odemnie?
A teráz zemdlił mię ; ſpuſtoƺyłeś / o Boże! wƺyſtko zgromádzenie moje.
Pomárƺcżyłeś mię ná świádectwo / á znácżne ná mnie ſchudzenie moje ná twárzy mojej / jáwnie świádcży przećiwko mnie.
Popędliwość jego porwáłá mię / y wziął nienáwiść przećiwko mnie ; á zgrzytájąc ná mię zębámi ſwemi / jáko nieprzyjáćiel mój / byſtremi ocżymá ſwemi ſpojrzáł ná mię.
Rozdzieráją ná mię uſtá ſwe / y ſromotnie mię policżkowáli / zebráwƺy śię ſpołu przećiwko mnie.
Podáł mię Bóg przewrotnemu / á w ręce niepobożnych wydáł mię.
Byłem w pokoju / ále mię potárł ; á uchwyćiwƺy mię zá ƺyję moję / roztrzáſkáł mię / y wyſtáwił mię ſobie zá cel.
Ogárnęli mię ſtrzelcy jego ; rozćiął nerki moje / á nie przepuśćił / y rozláł ná ziemię żółć moję.
Zránił mię / ráną ná ránę ; rzućił śię ná mię / jáko olbrzym.
Uƺyłem wór ná zśiniáłą ſkórę moję / á oƺpećiłem prochem głowę moję.
Twárz mojá płácżem oƺpeconá / á ná powiekách moich jeſt ćień śmierći.
Choćiáż żádnego łupieſtwá niemáƺ w rękách moich / á modlitwá mojá jeſt cżyſtá. ( A jeźli nie ták / )
O ziemio! nie zákrywájże krwi mojej / á niech nie má miejſcá wołánie moje!
Otoć y teráz w niebie jeſt świádek mój / jeſt świádek mój ná wyſokośći.
O kráſomówcy moi / przyjáćiele moi! wylewá łzy do Bogá oko moje.
Oby śię godziło wieść ſpór cżłowiekowi z Bogiem / y jáko ſynowi cżłowiecżemu z bliźnim ſwym!
Bo látá zámierzone nádchodzą / á śćieƺką / którą śię nie wrócę / już idę.
