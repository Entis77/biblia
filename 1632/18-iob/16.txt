Ale odpowiedźiáł Ijob / y rzekł :
Słyƺáłem tákowych rzecży wiele ; przykrymi ćieƺyćielámi wy wƺyſcy jeſteśćie.
Y kiedyż będźie koniec tym próżnym ſłowom? álbo co ćię przymuƺá / że ták odpowiádáƺ?
Azáżbym ja ták mówił / jáko wy / gdybyśćie wy byli ná miejſcu mojem? ázáżbym zbieráł przećiwko wám ſłowá / y kiwáłlibym nád wámi głową ſwoją?
Owƺembym was pośiláł uſtámi memi / á ruchánie wárg moich ulżyłoby boleśći wáƺych.
Ale jeſli będę mówił / przećież śię nie ukoi boleść mojá ; á jeſli też przeſtánę / izáż odejdźie odemnie?
A teraz zemdlił mię ; ſpuſtoƺyłeś / o Boże! wƺyſtko zgromádzenie moje.
Pomárƺcżyłeś mię ná świádectwo / á znácżne ná mnie ſchudzenie moje ná twárzy mojey / jáwnie świádcży przećiwko mnie.
Popędliwość jego porwáłá mię / y wźiął nienáwiść przećiwko mnie ; á zgrzytájąc ná mię zębámi ſwemi / jáko nieprzyjáćiel mój / byſtremi ocżymá ſwemi ſpojrzáł ná mię.
Rozdźieráją ná mię uſtá ſwe / y ſromotnie mię policżkowáli / zebráwƺy śię ſpołu przećiwko mnie.
Podáł mię Bóg przewrotnemu / á w ręce niepobożnych wydáł mię.
Byłem w pokoju / ále mię potárł ; á uchwyćiwƺy mię zá ƺyję moję / roztrzáſkáł mię / y wyſtáwił mię ſobie zá cel.
Ogárnęli mię ſtrzelcy jego ; rozćiął nerki moje / á nie przepuśćił / y rozláł ná źiemię żółć moję.
Zránił mię / ráną ná ránę ; rzućił śię ná mię / jáko olbrzym.
Uƺyłem wór ná zśiniáłą ſkórę moję / á oƺpećiłem prochem głowę moję.
Twárz mojá płácżem oƺpeconá / á ná powiekách moich jeſt ćień śmierći.
Choćiáż żádnego łupieſtwá niemáƺ w rękách moich / á modlitwá mojá jeſt cżyſtá. ( A jeſli nie ták / )
O źiemio! nie zákrywájże krwi mojey / á niech nie má miejſcá wołánie moje!
Otoć y teraz w niebie jeſt świádek mój / jeſt świádek mój ná wyſokośći.
O kráſomówcy moi / przyjáćiele moi! wylewá łzy do Bogá oko moje.
Oby śię godźiło wieść ſpór cżłowiekowi z Bogiem / y jáko ſynowi cżłowiecżemu z bliźnim ſwym!
Bo látá zámierzone nádchodzą / á śćieƺką / którą śię nie wrócę / już idę.
