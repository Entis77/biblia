A gdy przeſtáli oni trzej mężowie odpowiádáć Ijobowi / przeto / że śię ſobie zdał być ſpráwiedliwym :
Tedy śię rozpálił gniewem Elihu / ſyn Báráchelá Buzytcżyká z rodu Syryjſkiego / przećiw Ijobowi śię rozpálił gniewem / iż uſpráwiedliwiał duƺę ſwoję / więcey niż Bogá.
Tákże przećiwko trzem przyjáćiołom jego rozpálił śię gniew jego / że nie ználázƺy odpowiedźi / przećię potępiáli Ijobá.
Bo Elihu ocżekiwał / jáko oni Ijobowi odpowiedzą / gdyż oni ſtárśi byli w látách niż on.
Ale widząc Elihu / że nie było odpowiedźi w uſtách onych trzech mężów / rozpálił śię w gniewie ſwoim.
Y odpowiedźiał Elihu / ſyn Báráchelá Buzytcżyká / y rzekł : Jam nájmłodƺy w látách / á wyśćie ſtárcy ; przetoż wſtydźiłem śię / y nie śmiáłem wam oznájmić zdánia ſwego.
Myśláłem : Długi wiek mówić będźie / á mnóſtwo lat náucży mądrośći.
Aleć duch / który jeſt w ludźiách / y nátchnienie Wƺechmogącego dáje rozum.
Zácni nie záwƺe mądrzy / á ſtárcy nie záwżdy rozumieją ſądu.
Przetoż mówię : ſłucháj mię ; ja też oznájmię zdánie ſwoje.
Otom ocżekiwał ſłów wáƺych / á przyſłuchiwáłem śię dowodom wáƺym / cżekájąc / áżbyśćie doƺli rzecży.
Y przypátrywáłem śię wam / á oto żáden z was Ijobá przekonáć nie mógł ; y nie maƺ miedzy wámi / ktoby odpowiedźiał ſłowom jego.
Ale ſnadź rzecżećie : Ználeſliśmy mądrość ; ſám go Bóg przekonywá / nie cżłowiek.
Acżći śię Ijob nie zemną wdał w rzecż / á ja mu też nie wáƺemi ſłowy odpowiem.
Polękáli śię / nie odpowiádáją dálej ; niedoſtáje im ſłów.
Cżekáłemći / ále nie mówią ; umilknęli / á nic więcey nie odpowiádáją.
Odpowiem ja też z mej ſtrony ; oznájmię ja też zdánie ſwoje.
Bom pełen ſłów ; ćiáſno we mnie duchowi żywotá mego.
Oto żywot mój jeſt jáko moƺcż bez oddechu / á jáko becżká nowá rozpękłby śię.
Będę tedy mówił / á wytchnę ſobie ; otworzę wárgi ſwe / y odpowiem.
Nie będę teraz miał względu ná żádną oſobę / á z cżłowiekiem bez tytułów mówić będę.
Bo nie umiem tytułowáć / by mię w rychle nie porwał ſtworzyćiel mój.
