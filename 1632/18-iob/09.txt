Y odpowiedźiał Ijob / á rzekł :
Práwdźiwieć wiem / że ták jeſt ; bo jákożby miał być uſpráwiedliwiony cżłowiek przed Bogiem?
Jeſliby śię z nim chćiał ſpieráć / nie odpowie mu z tyśiącá ná jednę rzecż.
Mądry jeſt ſercem / y mocny śiłą ; któż użył pokoju / ſtáwiwƺy śię mu upornie?
On przenośi góry / á nie wiedzą ludźie / kto je podwrácá w gniewie ſwym.
On wzruƺá źiemię z miejſcá ſwego / á ſłupy jey trzęſą śię.
Gdy on zákáże ſłońcu / nie wſchodźi ; y gwiázdy piecżętuje.
On ſám rozpośćierá niebioſá / y depcże po wáłách morſkich.
On ſpráwił wóz niebieſki z gwiázd / Oryjoná y Hyjády / y inne gwiázdy ſkryte ná południe.
On cżyni rzecży wielkie / á niewybádáne y dźiwne / którym niemáƺ licżby.
Oto / idźieli mimo mię / nie widzę go ; á przychodźili / nie bácżę go.
Oto gdy co porwie / któż go przymuśi / áby przywróćił? Albo któż mu rzecże : Cóż cżyniƺ?
Gdyby Bóg nie odwróćił gniewu ſwego / upádliby przed nim pomocnicy hárdźi.
Jákoż mu ja tedy odpowiem? Jákie ſłowá obiorę przećiwko niemu?
Któremu / choćiáżbym był ſpráwiedliwym / nie odpowiem ; owƺem śię ſędźiemu memu upokorzę.
Choćbym go wzywał / á onby mi śię ozwał / przećię nie wierzę / áby przypuśćił do uƺów głos mój :
Bo mię ſtárł w wichrze / y rozmnożył rány moje bez przycżyny ;
Nie dopuƺcżá mi odetchnąć / owƺem mię náſycá gorzkośćiámi.
Jeſli śię udám do mocy / oto on nájmocniejƺy ; á jeſli do ſądu / któż mię z nim ſprowádźi?
Jeſlibym śię uſpráwiedliwiał / uſtá moje potępią mię ; jeſlibym śię doſkonáłym cżynił / tedy mię przewrotnym być pokáże.
Choćiáżbym był doſkonáły / przećież ja tego do śiebie znáć nie będę ; ále dam nágánę żywotowi memu.
Jedno jeſt / dla cżegom to mówił : że ták doſkonáłego / jáko y niezbożnego on niƺcży ;
Jeſli bicżem nágle zábijá / z pokuƺeniá niewinnych náśmiewá śię ;
Ziemiá podáná bywá w ręce niezbożniká / oblicże ſędźiów jey zákrywá. A jeſliż nie on / któż tedy inny jeſt / co to cżyni?
Ale dni moje prędƺe były niż poſeł ; ućiekły / á nie widźiáły nic dobrego.
Przeminęły jáko prędkie łodźie / jáko orzeł lecący do żeru.
Jeſli rzekę : Zápomnę nárzekánia mego / zániechám gniewu ſwego / á pośilę śię :
Tedy śię lękám wƺyſtkich boleśći mojich / widząc / że mię z nich nie wypuśćiƺ.
Jeſlim ja niezbożny / przecżże prózno prácuję?
A choćbym śię umywał wodámi śnieżnemi / y ocżyśćiłbym mydłem ręce moje :
Wƺákże w dole zánurzyƺ mię / y brzydźić śię mną będą ƺáty moje.
Abowiem on nie jeſt cżłowiekiem jáko ja / ábym mu śmiał odpowiedźieć / álbo żebym z nim miał iść w práwo.
Bo nie máƺ miedzy námi rozjemcy / któryby mógł rozwieść ſpráwę náƺę.
Niech tylko zdejmie zemnie rózgę ſwoję / á ſtrách jego niech mię nie ſtráƺy ;
Tedy będę mówił / á nie będę śię go bał ; bom ja nie jeſt táki ſám u śiebie.
