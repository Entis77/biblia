Potem otworzył Ijob uſtá ſwoje / y złorzecżył dniowi ſwemu.
Y záwołáł Ijob / mówiąc :
Bodáj był zginął dźień / któregom śię urodźił! y noc / w którą rzecżono : Pocżął śię mężcżyzná!
Bodáj śię był on dźień obróćił w ćiemność! By śię był o nim nie pytáł Bóg z wyſokośći / y nie był oświecony świátłośćią!
Bodáj go byłá záćmiłá ćiemność y ćień śmierći! by go był ogárnął obłok / y uſtráƺyłá go gorącość dźienná!
Bodáj byłá noc onę ośiádłá ćiemność / áby nie ƺłá w licżbę dni rocżnych / y w licżbę mieśięcy nie przyƺłá!
Bodáj noc oná byłá ſámotná / á śpiewániá áby nie było w niej!
Bodáj ją byli przeklęli / którzy przeklináją dźień / którzy ſą gotowi / wzruƺáć płácż ſwój!
Bodáj śię były záćmiły gwiázdy przy zmierzkániu jej! á cżekájąc świátłá / áby śię go byłá nie docżekáłá / áni nie oglądáłá zorzy poránnej!
Iż nie záwárłá drzwi żywotá mego / á nie ſkryłá boleśći od ocżu moich.
Przecżżem w żywoćie nie umárł / álbo / gdym z żywotá wyƺedł / cżemum nie zginął?
Przecżże mię piáſtowáno ná kolánách? á przecżżem ſſáł pierśi?
Abowiembym teráz leżáł y odpocżywáł ; ſpáłbym y miáłbym pokój /
Z królmi y z rádcámi źiemi / którzy ſobie budowáli ná miejſcách puſtych ;
Albo z kśiążętámi / którzy mieli złoto / á nápełniáli domy ſwe ſrebrem /
Albo cżemum śię nie ſtáł jáko mártwy płód ſkryty? álbo jáko niemowlątká / które nie oglądáły świátłośći?
Tám niepobożni przeſtáwáją ſtráƺyć / y tám odpocżywáją zwątleni w śiłę.
Támże więźniowie ſobie wydycháją / á nie ſłyƺą głoſu trápiącego ich /
Máły y wielki tám ſobie ſą równi á niewolnik wolny od páná ſwego.
Przecż nędznemu dáná jeſt świátłość / á żywot tym / którzy ſą utrápionego duchá?
Którzy cżekáją śmierći / á nie przychodźi / choć jej pilniej ƺukáją niż ſkárbów ſkrytych ;
Którzyby śię z rádośćią weſelili / pląſájąc / gdyby ználeźli grób.
Przecż dáná jeſt świátłość mężowi / którego drogá ſkrytá jeſt / á którego Bóg ćiężkośćiámi ogárnął?
Abowiem kiedy mám jeść / wzdychánie moje przychodźi / á rozchodźi śię jáko wodá rycżenie moje ;
Bo ſtrách / któregom śię lękáł / przyƺedł ná mię / á cżegom śię obáwiáł / przydáło mi śię.
Nie byłem bezpiecżny / ánim śię uſpokoił / ánim odpocżywáł / á przećież ná mię przyƺłá trwogá.
