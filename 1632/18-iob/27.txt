Potem dálej Ijob prowádźił rzecż ſwoję / y rzekł :
Jáko żyje Bóg / który odrzućił ſąd mój / á Wƺechmocny / który gorzkośći nábáwił duƺy mojey :
Że póki ſtáje tchu we mnie / y duchá Bożego w nozdrzách mojich /
Nie będą mówiły wárgi moje niepráwośći / á język mój nie będźie powiádał zdrády.
Nie dáj Boże / żebym was miał uſpráwiedliwiáć ; póki dech we mnie / nie odſtąpię od niewinnośći mojey.
Spráwiedliwośći mojey trzymáć śię będę / á nie puƺcżę śię jey ; nie záwſtydźi mię ſerce moje / pókim żyw.
Nieprzyjáćiel mój będźie jáko niezbożnik / á który powſtáje przećiwko mnie / jáko złośnik.
Co bowiem zá nádźiejá jeſt obłudniká / który śię w łákomſtwie kocha / gdy Bóg wydrze duƺę jego.
Izali Bóg uſłyƺy wołánie jego / gdy nań ućiſk przydźie?
Izáż śię w Wƺechmocnym rozkochá? á będźie wzywał Bogá ná káżdy cżás?
Ucżę was / będąc w ręce Bożey / á jáko idę z Wƺechmocnym / nie táję.
Oto wy to wƺyſcy widźićie ; przecżże wżdy próżność mówićie?
Tenći jeſt dźiał cżłowieká bezbożnego u Bogá / á toć dźiedźictwo okrutnicy od Wƺechmocnego wezmą.
Jeſli śię rozmnożą ſynowie jego / pójdą pod miecż : á potomſtwo jego nie náſyći śię chlebá /
Którzy po nim zoſtáną w śmierći pogrzebieni będą / á wdowy jego nie będą go płákáły ;
Choćby ſrebrá názgromádzał jáko prochu / á náſpráwiał ƺát jáko błotá :
Tedy náſpráwiáć ich on / ále ſpráwiedliwy oblekáć je będźie / á ſrebro ono niewinny dźielić będźie.
Zbuduje dom ſwój jáko mól / á jáko ſtróż budę wyſtáwi.
Bogáty záśnie / á nie będźie pogrzebiony ; ſpojrzyli kto / álić go niemáƺ.
Záchwycą go ſtráchy jáko wody / w nocy go porwie wicher.
Pochwyći go wiátr wſchodni / á odejdźie ; bo wicher ruƺył go z miejſcá ſwego.
Toć Bóg nań dopuśći / á nie przepuśći mu / choć przed ręką jego prędko ućiekáć będźie.
Kláśnie káżdy nád nim rękomá ſwemi / y wyſyká go z miejſcá ſwego.
