Tákże y żony <i>bądźćie</i> poddáne mężom ſwojim : áby y ći którzy nie wierzą ſłowu / przez <i>pobożne</i> obcowánie żon / bez ſłowá byli pozyſkáni ;
Obacżywƺy cżyſte w bojáźni <i>Bożey</i> obcowánie wáƺe.
Których ochędoſtwo niech będźie / nie ono zwierzchne / w ſplećieniu włoſów / y obłożeniu śię złotem / álbo w ubierániu śię w ƺáty :
Ale on ſkryty ſerdecżny cżłowiek / zależący w nieſkáżeniu ćichego y ſpokojnego duchá / który jeſt przed oblicżem Bożym koƺtowny.
Abowiem ták niekiedy y one święte małżonki / które nádźieję miáły w Bogu / zdobiły śię ; będąc poddáne mężom ſwojim.
Jáko Sará byłá poſłuƺna Abráhámowi / názywájąc go pánem : którey wy zſtáłyśćie śię córkámi / gdy dobrze cżynićie / nie bojąc śię żadnego poſtráchu.
Tákże y wy mężowie mieƺkajćie z nimi umiejętnie ; á jáko mdlejƺemu nacżyniu niewieśćiemu oddawajćie ucżćiwość / jáko też ſpół-dźiedźicżkam łáſki żywotá : áby śię modlitwy wáƺe nie przerywáły.
A ná koniec / wƺyſcy <i>bądźćie</i> jednomyślni / ſpół-ćierpiący doległośći / bráterſtwo miłujący / miłośierni y dobrotliwi.
Nie oddawájąc złego zá złe / áni łájánia zá łájánie : lecż przećiwnym obycżájem dobrorzecżąc ; gdyż wiećie iż ná to powołáni jeſteśćie / ábyśćie błogoſłáwieńſtwo odźiedźicżyli.
Abowiem kto chce żywot miłowáć / y oglądáć dni dobre / niech pohámuje języká ſwego od złego / á uſtá jego niech nie mowią zdrády.
Niech śię odwroći od złego / á cżyni dobre : niech ƺuka pokoju y śćiga go.
Abowiem ocży PAńſkie <i>otworzone ſą</i> ná ſpráwiedliwe / á uƺy jego ku proźbie ich : Lecż oblicże PAńſkie przećiwko tym którzy cżynią złe rzecży.
Y ktoż jeſt coby wam złe ucżynił / jeſlibyśćie dobrego náśládowcámi byli?
Ale choćiabyśćie też ćierpieli dla ſpráwiedliwośći / błogoſłáwieni <i>jeſteśćie</i> : á ſtráchu ich nie lękajćie śię / áni trwożćie ſobą ; ále PANA Bogá poświęcajćie w ſercách wáƺych.
A <i>bądźćie</i> záwƺe gotowi ku dániu odpowiedźi káżdemu domagájącemu śię od was ráchunku o tey nádźiejey która w was jeſt / z ćichośćią y z bojáźnią / májąc ſumnienie dobre.
Aby <i>w tym</i> w cżym was pomawiáją jáko złocżyńce / záwſtydźili śię ći którzy nágánę dawáją wáƺemu dobremu obcowániu w CHryſtuśie.
Lepiey bowiem jeſt ábyśćie dobrze cżyniąc / jeſli śię ták podoba woli Bożey / ćierpieli / niżeli źle cżyniąc.
Bo y CHryſtus raz zá grzechy ćierpiał / ſpráwiedliwy zá nieſpráwiedliwe : áby nas przywiodł do Bogá / umartwiony będąc ćiáłem / ále ożywiony duchem.
Przez którego y tym duchom którzy ſą w więźieniu / przyƺedƺy kazał.
Którzy niekiedy niepoſłuƺni byli / gdy raz ocżekawáłá Bożá ćierpliwość zá dni Noego / kiedy Korab gotowano ; w którym máło ( to jeſt ośm ) duƺ záchowáne ſą w wodźie.
Cżego teraz Chrzeſt wzorem będąc / záchowywa nas ( nie zſkładánie ćieleſnego plugáſtwá / ále dopytánie śię ſumnienia dobrego u Bogá ) przez zmartwychwſtánie JEzuſá CHryſtuſá :
Który jeſt ná práwicy Bożey / ƺedƺy do niebá / podbiwƺy ſobie Anjoły / y zwierzchnośći / y mocy.
