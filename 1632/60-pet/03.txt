Tákże y żony bądźćie poddáne mężom ſwojim : áby y ći / ktorzy nie wierzą ſłowu / przez pobożne obcowánie żon / bez ſłowá byli pozyſkáni ;
Obácżywƺy cżyſte w bojáźni Bożey obcowánie wáƺe.
Ktorych ochędoſtwo niech będźie / nie ono zwierzchne / w ſplećieniu włoſow / y obłożeniu śię złotem / álbo w ubierániu śię w ƺáty :
Ale on ſkryty ſerdecżny cżłowiek / zależący w nieſkáżeniu ćichego y ſpokojnego duchá / ktory jeſt przed oblicżem Bożym koƺtowny.
Abowiem ták niekiedy y one święte małżonki / ktore nádźieję miáły w Bogu / zdobiły śię ; będąc poddáne mężom ſwojim.
Jáko Sará byłá poſłuƺna Abráhámowi / názywájąc go pánem : ktorej wy ſtáłyśćie śię corkámi / gdy dobrze cżynićie / nie bojąc śię żadnego poſtráchu.
Tákże y wy mężowie mieƺkájćie z nimi umiejętnie ; á jáko mdlejƺemu nácżyniu niewieśćiemu oddawajćie ucżćiwość / jáko też ſpoł-dźiedźicżkom łáſki żywotá : áby śię modlitwy wáƺe nie przerywáły.
A ná koniec / wƺyſcy <i>bądźćie</i> jednomyślni / ſpoł-ćierpiący doległośći / bráterſtwo miłujący / miłośierni y dobrotliwi.
Nie oddáwájąc złego zá złe / áni łájániá zá łájánie : lecż przećiwnym obycżájem dobrorzecżąc ; gdyż wiećie iż ná to powołáni jeſteśćie / ábyśćie błogoſłáwieńſtwo odźiedźicżyli.
Abowiem kto chce żywot miłowáć / y oglądáć dni dobre / niech pohámuje języká ſwego od złego / á uſtá jego niech nie mowią zdrády.
Niech śię odwroći od złego / á cżyni dobre : niech ƺuká pokoju y śćiga go.
Abowiem ocży PAńſkie <i>otworzone ſą</i> ná ſpráwiedliwe / á uƺy jego ku prośbie ich : Lecż oblicże PAńſkie przećiwko tym / ktorzy cżynią złe rzecży.
Y ktoż jeſt coby wám złe ucżynił / jeſlibyśćie dobrego náśládowcámi byli?
Ale choćiábyśćie też ćierpieli dla ſpráwiedliwośći / błogoſłáwieni jeſteśćie : á ſtráchu ich nie lękajćie śię / áni trwożćie ſobą ; ále PANA Bogá poświęcajćie w ſercách wáƺych.
A <i>bądźćie</i> záwƺe gotowi ku dániu odpowiedźi káżdemu domagájącemu śię od was ráchunku o tej nádźiejey ktora w was jeſt / z ćichośćią y z bojáźnią / májąc ſumnienie dobre.
Aby <i>w tym</i> w cżym was pomawiáją jáko złocżyńce / záwſtydźili śię ći ktorzy nágánę dawáją wáƺemu dobremu obcowániu w CHryſtuśie.
Lepiej bowiem jeſt ábyśćie dobrze cżyniąc / jeſli śię ták podoba woli Bożey / ćierpieli / niżeli źle cżyniąc.
Bo y CHryſtus raz zá grzechy ćierpiał / ſpráwiedliwy zá nieſpráwiedliwe : áby nas przywiodł do Bogá / umártwiony będąc ćiáłem / ále ożywiony duchem.
Przez ktorego y tym duchom ktorzy ſą w więźieniu / przyƺedƺy kazał.
Ktorzy niekiedy niepoſłuƺni byli / gdy raz ocżekiwáłá Bożá ćierpliwość zá dni Noego / kiedy Korab gotowano ; w ktorym máło ( to jeſt ośm ) duƺ záchowáne ſą w wodźie.
Cżego teráz Chrzeſt wzorem będąc / záchowywa nas ( nie zſkładánie ćieleſnego plugáſtwá / ále dopytánie śię ſumnieniá dobrego u Bogá ) przez zmartwychwſtánie JEzuſá CHryſtuſá :
Ktory jeſt ná práwicy Bożey / ƺedƺy do niebá / podbiwƺy ſobie Anjoły / y zwierzchnośći / y mocy.
