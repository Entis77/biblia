Po tych ſpráwách wielmożnym ucżynił król Aſweruſ Hámáná / ſyná Hámedátowego / Agágiejcżyká / y wywyżƺył go / y wyſtáwił ſtolicę jego nád wƺyſtkich kśiążąt / którzy byli przy nim.
A wƺyſcy ſłudzy królewſcy / którzy byli u brámy królewſkiej / kłániáli mu śię / y upádáli przed Hámánem : ábowiem ták był rozkázáł król o nim. Ale Márdocheuƺ nie kłániáł śię / áni upádáł przed nim.
Przetoż rzekli ſłudzy królewſcy / którzy byli w brámie królewſkiej / do Márdocheuƺá : Cżemuż ty przeſtępujeƺ rozkázánie królewſkie?
A gdy ták do niego ná káżdy dźień máwiáli / á nie uſłucháł ich / oznájmili to Hámánowi / chcąc widźieć / jeźli śię oſtoją ſłowá Márdocheuƺowe ; bo im był powiedźiáł / że był Żydem.
A widząc Hámán / iż śię Márdocheuƺ nie kłániáł / áni upádáł przed nim / nápełniony jeſt Hámán popędliwośćią.
Y miáł to ſobie zá rzecż lekką / tárgnąć śię ná ſámego Márdocheuƺá ; ( bo mu było oznájmiono / z którego ludu był Márdocheuƺ / ) przetoż śię ſtáráł Hámán / áby wytráćił wƺyſtkich Żydów / którzy byli po wƺyſtkiem króleſtwie Aſweruſowem / náród Márdocheuƺowy.
A ták mieśiącá pierwƺego ( ten jeſt mieśiąc Niſán ) roku dwunáſtego królá Aſweruſá rozkázáł Hámán miotáć Pur ( to jeſt loſ ) przed ſobą ode dniá do dniá / y od mieśiącá áż do mieśiącá dwunáſtego ; ( ten jeſt mieśiąc Adár. )
Bo był rzekł Hámán do królá Aſweruſá : Jeſt lud niektóry rozproƺony y rozſypány między ludem po wƺyſtkich kráinách króleſtwá twego / którego práwá różne ſą od práw wƺyſtkich národów / á práw królewſkich nie przeſtrzegáją ; przetoż nie jeſt pożytecżno królowi / zániecháć ich.
Jeźli śię tedy królowi zdá / niech będźie nápiſáno / áby byli wytráceni. A já dźieśięć tyśięcy tálentów ſrebrá odwáżę do rąk przełożonych nád tą prácą / áby je odnieśli do ſkárbu królewſkiego.
Tedy zdjął król pierśćień ſwój z ręki ſwej / y dáł go Hámánowi Agágiejcżykowi / ſynowi Hámedátowemu / nieprzyjáćielowi żydowſkiemu.
Y rzekł król do Hámáná : Srebroć to dáruję / y ten lud / ábyś z nim cżynił / coć śię podobá.
Przetoż przyzwáno piſárzy królewſkich mieśiącá pierwƺego / trzynáſtego dniá tegoż mieśiącá / y nápiſáno wƺyſtko / jáko był rozkázáł Hámán / do kśiążąt królewſkich / y do ſtároſtów / którzy byli nád káżdą kráiną / y do hetmánów káżdego národu / do káżdej kráiny według piſmá jej / y do káżdego národu według języká jego. Imieniem królá Aſweruſá nápiſáno / y zápiecżętowáno ſygnetem królewſkim.
Y rozeſłáno liſty przez poſłów do wƺyſtkich kráin królewſkich / áby wygłádzono / wymordowáno / y wytrácono wƺyſtkich Żydów / od młodego áż do ſtárcá / dźiátki y niewiáſty / dniá jednego / trzynáſtego dniá mieśiącá dwunáſtego / ( ten jeſt mieśiąc Adár / ) á korzyść ich áby rozchwycono.
A táć ſumá byłá tych liſtów / áby obwołáno po wƺyſtkich kráinách / y oznájmiono wƺyſtkim národom / żeby byli gotowi ná on dźień.
Tedy wyjecháli poſłowie ſpieƺno z rozkázániem królewſkiem ; przybito też wyrok w Suſán / w mieśćie ſtołecżnem / á król y Hámán śiedźieli pijąc ; ále miáſto Suſán było zátrwożone.
