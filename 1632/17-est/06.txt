Onej nocy król nie mogąc ſpáć / kázáł przynieść kśięgi hiſtoryi pámięći godnych / y kroniki ; y cżytáno je przed królem.
Y ználeźli nápiſáne / że oznájmił Márdocheuƺ zdrádę Bigtáná y Tereſá / dwóch komorników królewſkich z tych / którzy ſtrzegli progu / że ƺukáli śćiągnąć rękę ná królá Aſweruſá.
Tedy rzekł król : Jákiejż doſtąpił cżći y zácnośći Márdocheuƺ dlá tego? Ná co odpowiedźieli ſłudzy królewſcy / dworzánie jego : Nic zá to nie odnióſł.
Y rzekł król : Któż jeſt w śieni? ( á Hámán przyƺedł był do śieni zewnętrznej páłácu królewſkiego / chcąc mówić z królem / áby powieƺono Márdocheuƺá ná ƺubienicy / którą mu był nágotowáł. )
Tedy odpowiedźieli królowi ſłudzy jego : Oto Hámán ſtoi w śieni. Y rzekł król : Niech ſám wnijdźie.
Y wƺedł Hámán. Któremu król rzekł : Coby ucżynić mężowi temu / którego król chce ucżćić? ( á Hámán myślił w ſercu ſwem : Komużby chćiáł król ucżćiwość więkƺą wyrządźić nád mię? )
Y odpowiedźiáł Hámán królowi : Mężowi / którego król chce ucżćić /
Niech przynioſą ƺátę królewſką / w którą śię ubierá król / y przywiodą koniá / ná którym jeżdżá król / á niech włożą koronę królewſką ná głowę jego ;
A dáwƺy onę ƺátę y onego koniá do ręki którego z kśiążąt królewſkich / z kśiążąt nájprzedniejƺych / niech ubiorą mężá onego / którego król chce ucżćić / á niech go prowádzą ná koniu po ulicy miejſkiej / á niech wołáją przed nim : Ták śię má ſtáć mężowi / którego król chce ucżćić.
Tedy rzekł król do Hámáná : Spieƺ śię / weźmij ƺátę y koniá / jákoś powiedźiáł / á ucżyń ták Márdocheuƺowi Żydowi / który śiedźi w brámie królewſkiej / á nie opuƺcżáj nic z tego wƺyſtkiego / coś mówił.
Przetoż wźiąwƺy Hámán ƺátę y koniá / ubráł Márdocheuƺá / y prowádźił go ná koniu po ulicy miejſkiej / wołájąc przed nim : Ták śię má ſtáć mężowi / którego król chce ucżćić.
Wróćił śię potem Márdocheuƺ do brámy królewſkiej / á Hámán pokwápił śię do domu ſwego z żáłośćią / májąc głowę nákrytą.
Y powiedźiáł Hámán Zereśie / żonie ſwej / y wƺyſtkim przyjáćiołom ſwoim wƺyſtko / co mu śię przydáło. Y rzekli do niego mędrcy jego / y Zereſ / żoná jego : Poniewáż z národu żydowſkiego jeſt Márdocheuƺ / przed któregoś oblicżem pocżął upádáć / nie przemożeƺ go / ále pewnie upádnieƺ przed oblicżem jego.
A gdy oni jeƺcże mówili z nim / oto komornicy królewſcy przyƺli / á przymuśili Hámáná / áby ƺedł ná ucżtę / którą byłá Eſter ſpráwiłá.
