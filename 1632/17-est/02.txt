To gdy śię ſtáło / á uśmierzył śię gniew królá Aſweruſá / wſpomniał ná Wáſty / y ná to / co byłá ucżyniłá / y ná dekret / który był wydán przećiwko niej.
Y rzekli dworzánie królewſcy / ſłudzy jego : Niech poƺukáją królowi dźiewecżek / pánienek piękney urody ;
A niech poſtánowi król ſtároſtów po wƺyſtkich kráinách króleſtwá ſwego / którzyby zebráli wƺyſtkie dźiewecżki / pánienki piękney urody / do Suſán miáſtá ſtołecżnego / do domu biáłych głów / pod dozór Hegájá / komorniká królewſkiego / ſtróżá biáłych głów / á dáli im ochędoſtwá ich.
A pánienká / któráby śię upodobáłá w ocżách królewſkich / niech króluje miáſto Wáſty. Y podobáłá śię tá rzecż w ocżách królewſkich / y ucżynił ták.
A był Żyd w Suſán / w mieśćie ſtołecżnem / imieniem Márdocheuƺ / ſyn Jáirá / ſyná Symhy / ſyná Cyſowego / z pokolenia Benjáminowego.
A ten był przenieśiony z Jeruzálemu z innymi pojmánymi / którzy byli przenieśieni z Jechonijáƺem / królem Judzkim / których był záwiódł w niewolę Nábuchodonozor / król Bábilońſki.
Ten chował Hádáſſę / którą też zwano Eſter / córkę ſtryjá ſwego / przeto / iż nie miáłá ojcá / áni mátki ; á byłá pánienká piękney urody / y wdźięcżney twárzy / którą Márdocheuƺ po śmierći ojcá jey y mátki jey zá córkę przyjął.
A gdy śię rozgłośiło rozkázánie królewſkie / y wyrok jego / y gdy zgromádzono pánienek wiele do Suſán / miáſtá ſtołecżnego / pod dozór Hegájá / wźięto też y Eſterę do domu królewſkiego pod dozór Hegájá / ſtróżá biáłych głów.
Y podobáłá mu śię oná dźiewecżká / á ználázłá łáſkę w ocżách jego. Przetoż jey záráz kázał dáć ochędoſtwo jey / y dźiał jey / y śiedm pánienek nádobnych kázał jey dáć z domu królewſkiego ; nádto opátrzeniá jey y pánienek jey polepƺył w domu biáłogłowſkim.
Ale nie oznájmiłá Eſter ludu ſwego / áni rodźiny ſwej ; ábowiem jey był Márdocheuƺ przykázał / áby nie oznájmowáłá.
Ale Márdocheuƺ ná káżdy dźień przechádzał śię przed śienią domu biáłogłowſkiego / chcąc śię dowiedźieć / jákoby śię miáłá Eſter / y coby śię z nią dźiáło.
A gdy przychodźił pewny cżás káżdej pánny / áby weƺłá do królá Aſweruſá / gdy śię wypełniło przy niej wƺyſtko według práwá biáłych głów przez dwánáśćie mieśięcy ; ( bo śię ták wypełniáły dni ochędáżánia ich / máżąc śię przez ƺeść mieśięcy olejkiem z myrry / á przez drugie ƺeść mieśięcy rzecżámi wonnemi / y innem ochędoſtwem biáłogłowſkiem. )
Zátem pánná wchodźiłá do królá / á o cokolwiek rzekłá / to jey dáno / áby z tym poƺłá z domu biáłogłowſkiego áż do pokoju królewſkiego.
W wiecżór wchádzáłá / á ráno śię záś wrácáłá do drugiego domu biáłogłowſkiego pod ſtráż Sááſgázy / komorniká królewſkiego / ſtróżá záłożnic ; nie wchádzáłá więcey do królá / ále jeſli śię upodobáłá królowi / przyzywano jey z imieniá.
A gdy przyƺedł cżás pewny Eſterze / córce Abiháilá / ſtryjá Márdocheuƺowego / ( który ją był ſobie wźiął zá córkę / ) áby weƺłá do królá / nie żądáłá nicżego / tylko co jey rzekł Hegáj / komornik królewſki / ſtróż biáłych głów. Y miáłá Eſter łáſkę w ocżách wƺyſtkich / którzy ją widźieli.
A ták wźiętá jeſt Eſter do królá Aſweruſá / do domu jego królewſkiego / mieśiącá dźieśiątego / ( ten jeſt mieśiąc Tebet / ) roku śiódmego królowánia jego.
Y rozmiłował śię król Eſtery nád wƺyſtkie biáłe głowy / á miáłá łáſkę y miłość u niego nád wƺyſtkie pánny / ták / iż włożył koronę królewſką ná głowę jey / á ucżynił ją królową miáſto Wáſty.
Nádto ſpráwił król ucżtę wielką ná wƺyſtkich kśiążąt ſwojich / y ſług ſwojich / to jeſt ucżtę Eſtery / y dał odpocżynek kráinom / y rozdáwał dáry / ták jáko przyſtoi królowi.
A gdy powtóre zebráne były pánny / á Márdocheuƺ śiedźiał u brámy królewſkiej ;
( A Eſter nie oznájmiłá byłá národu ſwego / áni ludu ſwego / jáko jey był rozkázał Márdocheuƺ ; bo rozkázániu Márdocheuƺowemu doſyć cżyniłá Eſter / jáko gdy ją wychowywał u śiebie. )
W oneż dni / gdy Márdocheuƺ śiedźiał u brámy królewſkiej / rozgniewał śię Bigtán y Teres / dwáj komornicy królewſcy / z tych / którzy ſtrzegli progu / y ƺukáli jákoby śćiągnąć rękę ná królá Aſweruſá.
Cżego dowiedźiáwƺy śię Márdocheuƺ / oznájmił to królowej Eſterze / á Eſterá to oznájmiłá królowi imieniem Márdocheuƺowem.
A gdy śię tego dowiádywano / ználázło śię ták ; y powieƺono obu ná ƺubienicy / á nápiſano to w kśięgách kroniki przed królem.
