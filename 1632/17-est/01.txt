Y ſtáło śię zá dni Aſweruſá / ( który Aſweruſ królowáł od Indyi áż do Murzyńſkiej źiemi / nád ſtem y dwudźieſtą y śiedmią kráin. )
Że zá onych dni / gdy śiedźiáł król Aſweruſ ná ſtolicy króleſtwá ſwego / która byłá w Suſán / mieśćie ſtołecżnem /
Roku trzećiego królowániá ſwego ſpráwił u śiebie ucżtę ná wƺyſtkich kśiążąt ſwoich / y ſług ſwoich / ná hetmánów z Perſów y z Medów / ná przełożonych y ná ſtároſtów onych kráin /
Pokázując bogáctwá / y chwáłę króleſtwá ſwego / y zácność á ozdobę wielmożnośći ſwojey przez wiele dni / miánowićie przez ſto y ośmdźieśiąt dni.
( A gdy śię dokońcżyły dni one / ucżynił król ná wƺyſtek lud / co go kolwiek było w Suſán / w mieśćie ſtołecżnem / od nájwiękƺego áż do nájmniejƺego / ucżtę przez śiedm dni ná ſáli w ogrodźie przy páłácu królewſkim. )
Opony biáłe / źielone y hijácyntowe záwieƺono ná ƺnurách biśiorowych y ƺárłátnych / ná kolcách ſrebrnych y ná ſłupách mármurowych ; łożá złote y ſrebrne ná tle kryƺtáłowem / y mármurowem / y páryjowem / y ſochárowem.
A nápój dáwáno w nácżyniu złotem / á to w nácżyniu co ráz innem / y winá królewſkiego doſtátkiem / jáko przyſtáło ná królá.
Ale do pićiá / według uſtáwy / nikt nie przymuƺáł. Abowiem ták był rozkázáł król wƺyſtkim rządcom domu ſwego / áby cżynili według woli káżdego.
Wáſty też królowá ſpráwiłá ucżtę ná biáłegłowy w domu królewſkim królá Aſweruſá.
A dniá śiódmego / gdy ſobie król podweſelił winem / rzekł do Mechumáná / Byſytá / Hárboná / Bygtá / y Abágtá / Zetárá / y Chárcháſá / do śiedmiu komorników / którzy ſłużyli przed oblicżem królá Aſweruſá /
Aby przywiedli Wáſty królowę przed oblicże królewſkie w koronie królewſkiej / chcąc pokázáć národom y kśiążętom piękność jey ; bo bárdzo piękná byłá.
Ale nie chćiáłá królowá Wáſty przyjść ná rozkázánie królewſkie / opowiedźiáne przez komorników. Przetoż rozgniewáł śię król bárdzo / á gniew jego zápálił śię w nim.
Tedy rzekł król do mędrców / rozumiejących cżáſy : ( bo táki był zwycżáj przedkłádáć ſpráwy królewſkie wƺyſtkim biegłym w práwách y w ſądách ;
A nájbliżƺymi jego byli Chárſená / Setár / Admátá / Tárſyſ / Mereſ / Márſená / Memuchán / śiedm kśiążąt Perſkich y Medſkich / którzy pátrzáli ná oblicże królewſkie / y śiádáli ná pierwƺem miejſcu w króleſtwie. )
Co cżynić podług práwá z królową Wáſty / przeto / iż nie ucżyniłá rozkázániá królá Aſweruſá / opowiedźiánego przez komorników?
Tedy odpowiedźiáł Memuchán przed królem y kśiążętámi : Nie przećiwko królowi ſámemu wyſtąpiłá Wáſty królowá / ále przećiwko wƺyſtkim kśiążętom / y przećiwko wƺyſtkim národom / którzy ſą po wƺyſtkich kráinách królá Aſweruſá.
Abowiem gdy śię tá ſpráwá królowej donieśie do wƺyſtkich niewiáſt / zniewáżą ſobie mężów ſwoich w ocżách ſwoich / y rzeką : Król Aſweruſ rozkázáł przywieść Wáſty królowę przed oblicże ſwoje / á nie przyƺłá.
Owƺem dźiśiáj toż rzeką kśiężny Perſkie y Medſkie / ( które ſłyƺáły poſtępek królowej ) wƺyſtkim kśiążętom królewſkim / á będźie doſyć wzgárdy y wáśni.
Przetoż / jeśli śię zá dobre widźi królowi / niech wynijdźie wyrok królewſki od oblicżá jego / á niech będźie wpiſán między práwá Perſkie y Medſkie / których śię przeſtępowáć nie godźi : Że nie chćiáłá przyjść Wáſty przed oblicżność królá Aſweruſá / przetoż króleſtwo jey dá król inney / lepƺej niż oná.
A gdy uſłyƺą ten wyrok królewſki / który wydáƺ po wƺyſtkiem króleſtwie ſwojim / jáko wielkie jeſt / tedy wƺyſtkie żony będą wyrządzáły ucżćiwość máłżonkom ſwoim od wielkiego áż do máłego.
Y podobáłá śię tá rádá królowi y kśiążętom. Y ucżynił król według rády Memuchánowej ;
A rozeſłáł liſty do wƺyſtkich kráin królewſkich / do káżdej kráiny piſmem jey włáſnem / y do káżdego národu językiem jego / áby káżdy mąż był pánem w domu ſwoim. A to obwołáno językiem káżdego národu.
