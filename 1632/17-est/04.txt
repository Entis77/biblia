A Márdocheuƺ / dowiedźiáwƺy śię wƺyſtkiego / co śię było ſtáło / rozdárł ƺáty ſwe / y oblekł śię w wór / á poſypáwƺy śię popiołem / wyƺedł w pośród miáſtá / y wołáł głoſem wielkim y żáłośnym.
Y przyƺedł áż przed brámę królewſką ; bo śię nie godźiło wnijść w brámę królewſką oblecżonemu w wór.
W káżdej tákże kráinie y miejſcu / gdźiekolwiek rozkázánie królewſkie / y wyrok jego przyƺedł / byłá wielká żáłość między Żydámi / poſt / y płácż / y nárzekánie / á w worze / y ná popiele wiele ich leżáło.
Przetoż przyƺedłƺy pánny Eſtery / y komornicy jej / oznájmili jej to ; y záſmućiłá śię królowá bárdzo y poſłáłá ƺáty / áby oblecżono Márdocheuƺá / zdjąwƺy z niego wór jego. Ale ich on nie przyjął.
Tedy záwołáwƺy Eſterá Atáchá / jednego z komorników królewſkich / którego jej był dáł zá ſługę / rozkázáłá mu z ſtrony Márdocheuƺá / áby śię dowiedźiáł / co y przecżby to było.
Wyƺedł tedy Atách do Márdocheuƺá ná ulicę miejſką / którá byłá przed brámą królewſką ;
Y oznájmił mu Márdocheuƺ wƺyſtko / co mu śię przydáło / y o tej ſumie ſrebrá / którą obiecáł Hámán odwáżyć do ſkárbu królewſkiego przećiwko Żydom / áby byli wytráceni.
Nádto dáł mu przepiſ wyroku / który był przybity w Suſán ná wytrácenie ich / áby okázáł Eſterze / y oznájmił jej ; á żeby jej rozkázáł / áby ƺłá do królá / y prośiłá go / á przycżyniłá śię do niego zá ludem ſwoim.
Tedy przyƺedłƺy Atách oznájmił Eſterze ſłowá Márdocheuƺowe.
Y rzekłá Eſterá do Atáchá / wſkázując przezeń do Márdocheuƺá :
Wƺyſcy ſłudzy królewſcy / y lud kráin królewſkich wiedzą / że ktobykolwiek ( mąż álbo biáłá głowá ) wƺedł do królá do śieni wnętrznej / nie będąc wezwány / to práwo o nim jeſt / áby był zábity / oprócż ná kogoby wyćiągnął król ſceptr złoty / ten żyw zoſtánie. Alem já nie byłá wezwáná / ábym weƺłá do królá / już przez trzydźieśći dni.
A gdy oznájmiono Márdocheuƺowi ſłowá Eſtery /
Rzekł Márdocheuƺ / áby záśię powiedźiáno Eſterze : Nie mniemáj w umyśle twoim / ábyś záchowáná być miáłá w domu królewſkim mimo wƺyſtkich Żydów.
Abowiem / jeźli ty ták cále milcżeć będźieƺ ná ten cżáſ / ulżenie y wybáwienie przyjdźie Żydom ſkąd inąd / ále ty y dom ojcá twego zginiećie ; á któż wie / jeźliś nie dlá tego cżáſu doſtąpiłá króleſtwá?
Y rzekłá Eſterá / áby záśię oznájmiono Márdocheuƺowi :
Idź / zbierz wƺyſtkich Żydów / którzy śię znájdują w Suſán / á pośććie zá mię / á nie jedzćie áni pijćie przez trzy dni / w nocy áni we dnie. Já też / y pánny moje tákże / będę pośćiłá ; tedy wnijdę do królá / choć to nie według práwá / á jeźli zginę / niech zginę.
Tedy ƺedł Márdocheuƺ / y ucżynił wƺyſtko / co mu byłá rozkázáłá Eſterá.
