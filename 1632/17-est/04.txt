A Márdocheuƺ / dowiedźiáwƺy śię wƺyſtkiego / co śię było ſtáło / rozdárł ƺáty ſwe / y oblekł śię w wór / á poſypáwƺy śię popiołem / wyƺedł w pośród miáſtá / y wołał głoſem wielkim y żáłośnym.
Y przyƺedł áż przed brámę królewſką ; bo śię nie godźiło wnijść w brámę królewſką oblecżonemu w wór.
W káżdej tákże kráinie y miejſcu / gdźiekolwiek rozkázánie królewſkie / y wyrok jego przyƺedł / byłá wielká żáłość miedzy Żydámi / poſt / y płácż / y nárzekánie / á w worze / y ná popiele wiele ich leżáło.
Przetoż przyƺedƺy pánny Eſtery / y komornicy jey / oznájmili jey to ; y záſmućiłá śię królowá bárzo y poſłáłá ƺáty / áby oblecżono Márdocheuƺá / zdjąwƺy z niego wór jego. Ale ich on nie przyjął.
Tedy záwołáwƺy Eſterá Atáchá / jednego z komorników królewſkich / którego jey był dał zá ſługę / rozkázáłá mu z ſtrony Márdocheuƺá / áby śię dowiedźiał / co y przecżby to było.
Wyƺedł tedy Atách do Márdocheuƺá ná ulicę miejſką / która byłá przed brámą królewſką ;
Y oznájmił mu Márdocheuƺ wƺyſtko / co mu śię przydáło / y o tey ſumie ſrebrá / którą obiecał Hámán odwáżyć do ſkárbu królewſkiego przećiwko Żydom / áby byli wytráceni.
Nádto dał mu przepis wyroku / który był przybity w Suſán ná wytrácenie ich / áby okázał Eſterze / y oznájmił jey ; á żeby jey rozkázał / áby ƺłá do królá / y prośiłá go / á przycżyniłá śię do niego zá ludem ſwoim.
Tedy przyƺedƺy Atách oznájmił Eſterze ſłowá Márdocheuƺowe.
Y rzekłá Eſterá do Atáchá / wſkázując przezeń do Márdocheuƺá :
Wƺyſcy ſłudzy królewſcy / y lud kráin królewſkich wiedzą / że ktobykolwiek ( mąż álbo biáłá głowá ) wƺedł do królá do śieni wnętrzney / nie będąc wezwány / to práwo o nim jeſt / áby był zábity / oprócż ná kogoby wyćiągnął król ſceptr złoty / ten żyw zoſtánie. Alem ja nie byłá wezwáná / ábym weƺłá do królá / już przez trzydźieśći dni.
A gdy oznájmiono Márdocheuƺowi ſłowá Eſtery /
Rzekł Márdocheuƺ / áby záśię powiedźiáno Eſterze : Nie mniemáj w umyśle twoim / ábyś záchowáná być miáłá w domu królewſkim mimo wƺyſtkich Żydów.
Abowiem / jeſli ty ták cále milcżeć będźieƺ ná ten cżás / ulżenie y wybáwienie przydźie Żydom ſkąd inąd / ále ty y dom ojcá twego zginiećie ; á któż wie / jeſliś nie dla tego cżáſu doſtąpiłá króleſtwá?
Y rzekłá Eſterá / áby záśię oznájmiono Márdocheuƺowi :
Idź / zbierz wƺyſtkich Żydów / którzy śię znájdują w Suſán / á pośććie zá mię / á nie jedzćie áni pijćie przez trzy dni / w nocy áni we dnie. Ja też / y pánny moje tákże / będę pośćiłá ; tedy wnijdę do królá / choć to nie według práwá / á jeſli zginę / niech zginę.
Tedy ƺedł Márdocheuƺ / y ucżynił wƺyſtko / co mu byłá rozkázáłá Eſterá.
