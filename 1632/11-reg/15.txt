Roku tedy ośmnáſtego królowánia Jeroboámá / ſyná Nábátowego / królował Abijám nád Judą.
Trzy látá królował w Jeruzálemie ; á imię mátki jego było Mááchá / córká Abiſálomowá.
Ten chodźił we wƺyſtkich grzechách ojcá ſwego / które cżynił przed nim ; á nie było ſerce jego doſkonáłe przy Pánu / Bogu ſwoim / jáko ſerce Dawidá / ojcá jego.
Wƺákże dla Dawidá dał mu Pán / Bóg jego / pochodnią w Jeruzálemie / wzbudźiwƺy ſyná jego po nim / á utwierdźiwƺy Jeruzálem /
Przeto / że cżynił Dawid / co było dobrego w ocżách Páńſkich / á nie uchylał śię od wƺyſtkiego / co mu rozkázał / po wƺyſtkie dni żywotá ſwego / oprócż ſpráwy z Uryjáƺem Hetejcżykiem.
Y byłá wojná miedzy Roboámem / y miedzy Jeroboámem po wƺyſtkie dni żywotá jego.
A inƺe ſpráwy Abijámowe / y wƺyſtko / co cżynił / ázaż nie jeſt nápiſáne w kronikách o królách Judzkich / jáko y wojná miedzy Abijámem y miedzy Jeroboámem?
A gdy záſnął Abijám z ojcy ſwymi / pochowano go w mieśćie Dawidowem. Y królował Azá / ſyn jego / miáſto niego.
A ták roku dwudźieſtego Jeroboámá / królá Izráelſkiego / królował Azá nád Judą.
Cżterdźieśći lat y jeden królował w Jeruzálemie / á imię mátki jego było Mááchá / córká Abiſálomowá.
Y cżynił Azá / co dobrego było w ocżách Páńſkich / jáko Dawid oćiec jego.
Abowiem wytráćił Sodomcżyki z źiemi / y wyrzućił wƺyſtkie báłwány / których byli nácżynili ojcowie jego.
Nádto y Mááchę / mátkę ſwoję / zrzućił z pánowánia / bo byłá ſpráwiłá ſtráƺnego báłwáná w gáju ; przetoż porąbał Azá tego ſtráƺnego báłwáná jey / y ſpálił u potoku Cedron.
A choćiáż wyżyny nie były ſkáżone / jednák ſerce Azy było doſkonáłe przy Pánu po wƺyſtkie dni jego.
Y wnióſł rzecży poświęcone ojcá ſwego / y rzecży / które ſám poświęćił / do domu Páńſkiego / ſrebro / y złoto / y nacżynia.
Y byłá wojná miedzy Azą y miedzy Báázą / królem Izráelſkim / po wƺyſtkie dni ich.
Abowiem Báázá / król Izráelſki / wyćiągnął przećiw Judźie / á zbudował Rámę / áby nie dopuśćił wychodźić y wchodźić nikomu do Azy / królá Judzkiego.
Ale wźiąwƺy Azá wƺyſtko ſrebro y złoto / które było pozoſtáło w ſkárbiech domu Páńſkiego / y w ſkárbiech domu królewſkiego / dał je w ręce ſługom ſwoim ; y poſłał je król Azá do Benádádá / ſyná Tábremonowego / ſyná Hezyjonowego / królá Syryjſkiego / który mieƺkał w Dámáƺku / mówiąc :
Przymierze jeſt miedzy mną y miedzy tobą / miedzy ojcem moim y miedzy ojcem twoim ; otoć poſyłám dáry / ſrebro y złoto ; idźże / wzruƺ przymierze twoje z Báázą / królem Izráelſkim / áby odćiągnął odemnie.
Y uſłuchał Benádád królá Azy ; á poſłáwƺy hetmány z wojſki / które miał przećiw miáſtom Izráelſkim / zburzył Hijon y Dán y Abelbetmááchę / y wƺyſtko Cynnerot / y wƺyſtkę źiemię Neftálim.
Co gdy uſłyƺał Báázá / przeſtał budowáć Rámy / y mieƺkał w Terśie.
Tedy król Azá zebrał wƺyſtek lud Judzki / nikogo nie wyjmując ; á pobráli kámienie z Rámy y drzewo jego / z którego budował Báázá : á zbudował z niego król Azá Gábáá Benjáminowe / y Máſfá.
A inne wƺyſtkie ſpráwy Azy / y wƺyſtká moc jego / y cokolwiek cżynił / y miáſtá / które zbudował / ázaż to nie jeſt nápiſáne w kronikách o królách Judzkich? Ale cżáſu ſtárośći ſwej chorował ná nogi ſwoje.
Y záſnął Azá z ojcy ſwymi / á pochowány jeſt z nimi w mieśćie Dawidá / ojcá ſwego. A Jozáfát / ſyn jego / królował miáſto niego.
Ale Nádáb / ſyn Jeroboámá / náſtąpił ná króleſtwo Izráelſkie roku wtórego Azy / królá Judzkiego / y królował nád Izráelem dwá látá ;
Y cżynił złe przed ocżymá Páńſkimi / chodząc drogámi ojcá ſwego / y w grzechách jego / któremi do grzechu przywodźił Izráelá.
Y zbuntował śię przećiw niemu Báázá / ſyn Achyjáƺá / z domu Iſáſchár ; á poráźił go Báázá u Giebbeton / które było Filiſtyńſkie ; bo Nádáb ze wƺyſtkim Izráelem obległ był Giebbeton.
Y zábił go Báázá roku trzećiego Azy / królá Judzkiego / á ſám królował miáſto niego.
Y ſtáło śię / gdy pocżął królowáć / że wymordował wƺyſtek dom Jeroboámowy ; á nie zoſtáwił żádney duƺy z narodu Jeroboámowego / áż je wytráćił według ſłowá Páńſkiego / które był opowiedźiał przez ſługę ſwego Achyjáƺá Sylonitcżyká ;
A to dla grzechów Jeroboámowych / który grzeƺył / y który do grzechu przywiódł Izráelcżyki / y dla przeſtępſtwá / którem wzruƺył ku gniewu Páná / Bogá Izráelſkiego.
A inne ſpráwy Nádábowe / y wƺyſtko co cżynił / ázaż to nie jeſt nápiſáne w kronikách królów Izráelſkich?
Y byłá wojná miedzy Azą / y miedzy Báázą / królem Izráelſkim / po wƺyſtkie dni ich.
Roku trzećiego Azy / królá Judzkiego / królował Báázá / ſyn Achyjáƺá / nád wƺyſtkim Izráelem w Terśie przez dwádźieśćiá y cżtery lat.
Y cżynił złe przed Pánem / chodząc drogámi Jeroboámowemi / y w grzechu jego / którym do grzechu przywiódł Izráelity.
