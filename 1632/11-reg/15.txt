Roku tedy ośmnáſtego królowániá Jeroboámá / ſyná Nábátowego / królowáł Abijám nád Judą.
Trzy látá królowáł w Jeruzálemie ; á imię mátki jego było Mááchá / córká Abiſálomowá.
Ten chodźił we wƺyſtkich grzechách ojcá ſwego / które cżynił przed nim ; á nie było ſerce jego doſkonáłe przy Pánu / Bogu ſwoim / jáko ſerce Dawidá / ojcá jego.
Wƺákże dlá Dawidá dáł mu Pán / Bóg jego / pochodnią w Jeruzálemie / wzbudźiwƺy ſyná jego po nim / á utwierdźiwƺy Jeruzálem /
Przeto / że cżynił Dawid / co było dobrego w ocżách Páńſkich / á nie uchyláł śię od wƺyſtkiego / co mu rozkázáł / po wƺyſtkie dni żywotá ſwego / oprócż ſpráwy z Uryjáƺem Hetejcżykiem.
Y byłá wojná między Roboámem / y między Jeroboámem po wƺyſtkie dni żywotá jego.
A inƺe ſpráwy Abijámowe / y wƺyſtko / co cżynił / ázáż nie jeſt nápiſáne w kronikách o królách Judzkich / jáko y wojná między Abijámem y między Jeroboámem?
A gdy záſnął Abijám z ojcy ſwymi / pochowáno go w mieśćie Dawidowem. Y królowáł Azá / ſyn jego / miáſto niego.
A ták roku dwudźieſtego Jeroboámá / królá Izráelſkiego / królowáł Azá nád Judą.
Cżterdźieśći lát y jeden królowáł w Jeruzálemie / á imię mátki jego było Mááchá / córká Abiſálomowá.
Y cżynił Azá / co dobrego było w ocżách Páńſkich / jáko Dawid ojćiec jego.
Abowiem wytráćił Sodomcżyki z źiemi / y wyrzućił wƺyſtkie báłwány / których byli nácżynili ojcowie jego.
Nádto y Mááchę / mátkę ſwoję / zrzućił z pánowániá / bo byłá ſpráwiłá ſtráƺnego báłwáná w gáju ; przetoż porąbáł Azá tego ſtráƺnego báłwáná jej / y ſpálił u potoku Cedron.
A choćiáż wyżyny nie były ſkáżone / jednák ſerce Azy było doſkonáłe przy Pánu po wƺyſtkie dni jego.
Y wnióſł rzecży poświęcone ojcá ſwego / y rzecży / które ſám poświęćił / do domu Páńſkiego / ſrebro / y złoto / y nácżyniá.
Y byłá wojná między Azą y między Báázą / królem Izráelſkim / po wƺyſtkie dni ich.
Abowiem Báázá / król Izráelſki / wyćiągnął przećiw Judźie / á zbudowáł Rámę / áby nie dopuśćił wychodźić y wchodźić nikomu do Azy / królá Judzkiego.
Ale wźiąwƺy Azá wƺyſtko ſrebro y złoto / które było pozoſtáło w ſkárbiech domu Páńſkiego / y w ſkárbiech domu królewſkiego / dáł je w ręce ſługom ſwoim ; y poſłáł je król Azá do Benádádá / ſyná Tábremonowego / ſyná Hezyjonowego / królá Syryjſkiego / który mieƺkáł w Dámáƺku / mówiąc :
Przymierze jeſt między mną y między tobą / między ojcem moim y między ojcem twoim ; otoć poſyłám dáry / ſrebro y złoto ; idźże / wzruƺ przymierze twoje z Báázą / królem Izráelſkim / áby odćiągnął odemnie.
Y uſłucháł Benádád królá Azy ; á poſłáwƺy hetmány z wojſki / które miáł przećiw miáſtom Izráelſkim / zburzył Hijon y Dán y Abelbetmááchę / y wƺyſtko Cynnerot / y wƺyſtkę źiemię Neftálim.
Co gdy uſłyƺáł Báázá / przeſtáł budowáć Rámy / y mieƺkáł w Terśie.
Tedy król Azá zebráł wƺyſtek lud Judzki / nikogo nie wyjmując ; á pobráli kámienie z Rámy y drzewo jego / z którego budowáł Báázá : á zbudowáł z niego król Azá Gábáá Benjáminowe / y Máſfá.
A inne wƺyſtkie ſpráwy Azy / y wƺyſtká moc jego / y cokolwiek cżynił / y miáſtá / które zbudowáł / ázáż to nie jeſt nápiſáne w kronikách o królách Judzkich? Ale cżáſu ſtárośći ſwej chorowáł ná nogi ſwoje.
Y záſnął Azá z ojcy ſwymi / á pochowány jeſt z nimi w mieśćie Dawidá / ojcá ſwego. A Jozáfát / ſyn jego / królowáł miáſto niego.
Ale Nádáb / ſyn Jeroboámá / náſtąpił ná króleſtwo Izráelſkie roku wtórego Azy / królá Judzkiego / y królowáł nád Izráelem dwá látá ;
Y cżynił złe przed ocżymá Páńſkimi / chodząc drogámi ojcá ſwego / y w grzechách jego / któremi do grzechu przywodźił Izráelá.
Y zbuntowáł śię przećiw niemu Báázá / ſyn Ahyjáƺá / z domu Iſáſchár ; á poráźił go Báázá u Giebbeton / które było Filiſtyńſkie ; bo Nádáb ze wƺyſtkim Izráelem obległ był Giebbeton.
Y zábił go Báázá roku trzećiego Azy / królá Judzkiego / á ſám królowáł miáſto niego.
Y ſtáło śię / gdy pocżął królowáć / że wymordowáł wƺyſtek dom Jeroboámowy ; á nie zoſtáwił żádnej duƺy z národu Jeroboámowego / áż je wytráćił według ſłowá Páńſkiego / które był opowiedźiáł przez ſługę ſwego Achyjáƺá Sylonitcżyká ;
A to dlá grzechów Jeroboámowych / który grzeƺył / y który do grzechu przywiódł Izráelcżyki / y dlá przeſtępſtwá / którem wzruƺył ku gniewu Páná / Bogá Izráelſkiego.
A inne ſpráwy Nádábowe / y wƺyſtko co cżynił / ázáż to nie jeſt nápiſáne w kronikách królów Izráelſkich?
Y byłá wojná między Azą / y między Báázą / królem Izráelſkim / po wƺyſtkie dni ich.
Roku trzećiego Azy / królá Judzkiego / królowáł Báázá / ſyn Achyjáƺá / nád wƺyſtkim Izráelem w Terśie przez dwádźieśćiá y cżtery lát.
Y cżynił złe przed Pánem / chodząc drogámi Jeroboámowemi / y w grzechu jego / którym do grzechu przywiódł Izráelity.
