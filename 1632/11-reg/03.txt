Y ſpowinowáćił śię Sálomon z Fáráonem / królem Egipſkim ; bo pojął córkę Fáráonową / y przyprowádźił ją do miáſtá Dawidowego / áżby dobudowáł domu ſwego / y domu Páńſkiego / y muru Jeruzálemſkiego w około.
Wƺákże lud ofiárowáł po górách / przeto / że nie był jeƺcże zbudowány dom imieniowi Páńſkimu áż do onych dni.
Y miłowáł Sálomon Páná / chodząc w przykázániách Dawidá / ojcá ſwego ; tylko że ná górách ofiárowáł y kádźił.
Szedł tedy król do Gábáon / áby tám ofiárowáł ; bo tám byłá górá nájwiękƺá ; tyśiąc ofiár cáłopálonych ofiárowáł Sálomon ná onym ołtárzu.
Y ukázáł śię Pán w Gábáon Sálomonowi przez ſen w nocy / y rzekł Bóg : Proś cżego chceƺ / á dám ći.
Tedy rzekł Sálomon : Tyś ucżynił z ſługą twoim Dawidem / ojcem moim / miłośierdźie wielkie / gdyż chodźił przed tobą w práwdźie y w ſpráwiedliwośći / y w proſtośći ſercá ſtáł przy tobie ; y záchowáłeś mu to miłośierdźie wielkie / iżeś mu dáł ſyná / któryby śiedźiáł ná ſtolicy jego / jáko śię to dźiś okázuje.
A teraz / o Pánie Boże mój / tyś poſtánowił królem ſługę twego miáſto Dawidá / ojcá mego / á jám jeſt dźiećię máłe / y nie umiem wychodźić áni wchodźić.
A ſługá twój jeſt w pośrodku ludu twego / któryś obráł / ludu wielkiego / który nie może zlicżony áni poráchowány być przez mnóſtwo.
Przetoż dáj ſłudze twemu ſerce rozumne / áby ſądźił lud twój / y áby rozeznáwáł między dobrem y złem ; ábowiem któż może ſądźić ten lud twój ták wielki?
Y podobáło śię to Pánu / że żądáł Sálomon tey rzecży.
Tedy rzekł Bóg do niego : Dla tego / żeś o to prośił / á nieżądáłeś ſobie długich dni / ániś żądáł ſobie bogáctw / ániś prośił o wytrácenie nieprzyjáćiół twoich / áleś ſobie prośił o rozum dla rozeznániá ſądu :
Otożem ucżynił według ſłów twoich ; otom ći dáł ſerce mądre y rozumne / ták / iż żáden równy tobie nie był przed tobą / áni po tobie powſtánie równy tobie.
Do tego y to / cżegoś nie żądáł / dáłem ći / to jeſt bogáctwá y ſłáwę / ták áby nie było żádnego tobie równego między królmi po wƺyſtkie dni twoje.
A będźieƺli chodźił drogámi mojemi / ſtrzegąc wyroków moich / y rozkázániá mego / jáko chodźił Dawid / ojćiec twój / tedy przedłużę dni twoje.
A gdy śię ocućił Sálomon / zrozumiáł / że to był ſen. Y przyƺedł do Jeruzálemu / á ſtánąwƺy przed ſkrzynią przymierzá Páńſkiego / ſpráwowáł cáłopáleniá / y ofiárowáł ofiáry ſpokojne / ſpráwił też ucżtę ná wƺyſtkie ſługi ſwoje.
Tedy przyƺły dwie niewiáſty wƺetecżnice do królá / y ſtánęły przed nim.
Y rzekłá jedná z onych niewiáſt : Proƺę pánie mój / ja y tá niewiáſtá mieƺkámy w jednym domu / y porodźiłám u niej w tymże domu.
Y ſtáło śię dniá trzećiego po porodzeniu mojem / że porodźiłá y tá niewiáſtá ; y byłyśmy poſpołu / á nie było nikogo obcego z námi w domu / oprócż nas dwóch w tymże domu.
Y umárł ſyn tey niewiáſty w nocy / przeto / iż go byłá przyległá.
A wſtáwƺy o północy / wźięłá ſyná mego odemnie / gdy ſłużebnicá twojá ſpáłá / y położyłá go ná łonie ſwojim / á ſyná ſwego umárłego położyłá ná łonie mojem.
A gdym wſtáłá ráno / chcąc dáć ſſáć ſynowi memu / otom ználázłá umárłego ; któremu gdym śię ráno przypátrzyłá / á oto nie był ſyn mój / któregom porodźiłá.
Y rzekłá oná drugá niewiáſtá : Nie ták ; ále ſyn mój jeſt ten żywy / á ſyn twój ten umárły. Ale oná rzekłá : Nie ; ále ſyn twój jeſt ten umárły / á ſyn mój ten żywy. Y ták śię ſpieráły przed królem.
Y rzekł król : Tá mówi : Ten żywy jeſt ſyn mój / á ſyn twój ten umárły ; á tá záśię mówi : Nie ták ; ále ſyn twój ten umárły / á ſyn mój ten żywy.
Przetoż rzekł król : Przynieśćie mi miecż. Y przynieśiono miecż przed królá.
Tedy rzekł król : Rozetnijćie to żywe dźiećię ná dwoje / á dájćie połowę jedney / á połowę drugiej.
Ale niewiáſtá / którey był ten ſyn żywy / mówiłá do królá / ( bo śię były poruƺyły wnętrznośći jey nád ſynem jey / )i rzekłá : Proƺę / pánie mój / dájćie jey to dźiećię żywe / á żádnym ſpoſobem nie zábijájćie go. Ale drugá rzekłá : Niech nie będźie áni mnie / áni tobie / rozetnijćie je.
Tedy odpowiedźiáł król / y rzekł : Dájćież tey dźiećię żywe / á żádną miárą nie zábijájćie go ; táć jeſt mátká jego.
A uſłyƺáwƺy wƺyſtek lud Izráelſki ten ſąd / który oſądźił król / báli śię królá : ábowiem wiedźieli / że mądrość Bożá byłá w ſercu jego ku cżynieniu ſądu.
