A nie było przez trzy látá wojny między Syryjcżykámi y między Izráelcżykámi.
Y ſtáło śię roku trzećiego / że przyjecháł Jozáfát król Judzki / do królá Izráelſkiego.
Tedy rzekł król Izráelſki do ſług ſwoich : Nie wiećież / iż náƺe jeſt Rámot Gáláád? A my zániedbywámy odebráć go z ręki królá Syryjſkiego.
Przetoż rzekł do Jozáfátá : Poćiągnieƺże ze mną ná wojnę przećiwko Rámot Gáláád? Y rzekł Jozáfát do królá Izráelſkiego : Jákom já / ták y ty ; jáko lud mój / ták lud twój ; jáko konie moje / ták konie twoje.
Nádto rzekł Jozáfát do królá Izráelſkiego : Spytáj śię proƺę dziś ſłowá Páńſkiego.
A ták zebráł król Izráelſki proroków około cżterech ſet mężów / y rzekł do nich : Mámże ćiągnąć ná wojnę przećiwko Rámot Gáláád? cży zániecháć? Y odpowiedzieli mu : Ciągnij ; bo je Pán dá w ręce królewſkie.
Ale Jozáfát rzekł : Niemáƺże tu którego proroká Páńſkiego / żebyśmy śię go pytáli?
Y rzekł król Izráelſki do Jozáfátá : Jeſt jeƺcże mąż jeden / przez któregobyśmy śię mogli rádzić Páná ; ále go já nienáwidzę / bo mi nic dobrego nie prorokuje / jedno złe / Micheáƺ / ſyn Jemlá. Y rzekł Jozáfát : Niech ták nie mówi król.
A ták záwołáł król Izráelſki komorniká niektórego / y rzekł : Przywiedź tu rychło Micheáƺá / ſyná Jemlowego.
Między tem król Izráelſki / y Jozáfát / król Judzki / śiedzieli ná ſtolicách ſwoich / ubráni w ƺáty królewſkie / ná plácu u wrót brámy Sámáryjſkiej / á wƺyſcy prorocy prorokowáli przed nimi.
A Sedechyjáƺ / ſyn Chenááná / ſpráwił ſobie rogi żelázne / y rzekł : Ták mówi Pán : Temi będzieƺ bódł Syryjcżyki / áż je wyniƺcżyƺ.
Tákże wƺyſcy prorocy prorokowáli / mówiąc : Ciągnij do Rámot Gáláád / á będzieć śię ƺcżęśćiło ; ábowiem je podá Pán w ręce królewſkie.
Tedy poſeł / który chodził / áby przyzwáł Micheáƺá / rzekł do niego / mówiąc : Oto teráz ſłowá proroków jednemi uſty dobrze tuƺą królowi ; niechże będzie proƺę ſłowo twoje / jáko ſłowo jednego z nich / á mów dobre rzecży.
Y rzekł Micheáƺ : Jáko żywy Pán / że co mi kolwiek rzecże Pán / to mówić będę.
A gdy przyƺedł do królá / rzekł król do niego : Micheáƺu / mámyż ćiągnąć ná wojnę przećiw Rámot Gáláád / cżyli zániecháć? A on mu rzekł : Ciągnij / á będzieć śię ƺcżęśćiło ; ábowiem je podá Pán w ręce królewſkie.
Y rzekł do niego król : A wieleż ćię rázy mám przyśięgą obowiązáć / ábyś mi nie mówił jedno práwdę w imieniu Páńſkim?
Przetoż rzekł : Widziáłem wƺyſtek lud Izráelſki rozproƺony po górách jáko owce / które nie máją páſterzá ; bo rzekł Pán : Nie máją ći páná ; niech śię wróći káżdy do domu ſwego w pokoju.
Y rzekł król Izráelſki do Jozáfátá : Izáżem ći nie powiádáł / że mi nie miáł prorokowáć dobrego / ále złe?
A Micheáƺ rzekł : Słuchájże tedy ſłowá Páńſkiego : Widziáłem Páná śiedzącego ná ſtolicy ſwojej / y wƺyſtko wojſko niebieſkie ſtojące po práwicy jego / y po lewicy jego.
Y rzekł Pán : Kto zwiedzie Achábá / áby ƺedł á upádł w Rámot Gáláád? A gdy mówił jeden ták / á drugi inácżej :
Tedy wyſtąpił duch / y ſtánął przed Pánem / mówiąc : Já go zwiodę. A Pán mu rzekł : Przez cóż?
Odpowiedziáł : Wynijdę / á będę duchem kłámliwym w uſtách wƺyſtkich proroków jego. Y rzekł mu Pán : Zwiedzieƺ / y pewnie przemożeƺ. Idźże / á ucżyń ták.
Przetoż teráz oto dáł Pán duchá kłámliwego w uſtá tych wƺyſtkich proroków twoich / gdyż Pán wyrzekł przećiwko tobie złe.
Tedy przyſtąpiwƺy Sedechyjáƺ / ſyn Chenááná / uderzył Micheáƺá w policżek / mówiąc : Kiedyż odƺedł Duch Páńſki odemnie / áby z tobą mówił?
Y odpowiedziáł Micheáƺ : Oto ty ujrzyƺ dniá onego / kiedy wnijdzieƺ do nájſkrytƺej komory / ábyś śię ſkrył.
Y rzekł król Izráelſki : Weźmij Micheáƺá / á wiedź go do Amoná / ſtároſty miejſkiego / y do Joáſá / ſyná królewſkiego.
Y rzecżeƺ : Ták mówi król : Wſádźćie tego mężá do więzieniá ; á dáwájćie mu jeść chleb utrápieniá y wodę ućiſku / áż śię wrócę w pokoju.
Ale odpowiedziáł Micheáƺ : Jeźliże śię wróćiƺ w pokoju / tedyć nie mówił Pán przez mię. Nádto rzekł : Słuchájćież wƺyſcy ludzie.
A ták ćiągnął król Izráelſki y Jozáfát / król Judzki / do Rámot Gáláád.
Y rzekł król Izráelſki do Jozáfátá : Odmienię śię / gdy pójdę do bitwy ; ále ty ubierz śię w ƺáty twoje. Y odmienił śię król Izráelſki / á ƺedł ku bitwie.
A król Syryjſki rozkázáł był hetmánom / których było trzydzieśći y dwá nád wozámi jego / mówiąc : Nie potykájćie śię áni z máłym / áni z wielkim / tylko z ſámym królem Izráelſkim.
Y ſtáło śię / gdy ujrzeli Jozáfátá hetmáni / co byli nád wozámi / rzekli : Zápráwdę to król Izráelſki ; y obróćili śię przećiwko niemu / chcąc śię z nim potykáć ; ále Jozáfát záwołáł.
Wtem obácżywƺy hetmáni / co byli nád wozámi / że nie ten był król Izráelſki / odwróćili śię od niego.
Lecż mąż niektóry ſtrzelił z łuku ná niepewne / y poſtrzelił królá Izráelſkiego między nity y między páncerz ; który rzekł woźnicy ſwemu : Náwróć / á wywieź mię z wojſká ; bom jeſt rániony.
Y wzmogłá śię bitwá dniá onego / á król ſtáł ná wozie przećiw Syryjcżykom : potem umárł w wiecżór / á krew ćiekłá z rány jego ná wóz.
Tedy wołáł woźny w wojſku / gdy już ſłońce záchodziło / mówiąc : Wróć śię káżdy do miáſtá ſwego y káżdy do ziemi ſwojej.
A ták umárł król / á odwiezion jeſt do Sámáryi / y pochowáno go w Sámáryi.
A gdy umywáno wóz w ſádzáwce Sámáryjſkiej / lizáli pſy krew jego / tákże gdy umywáno zbroję jego : według ſłowá Páńſkiego / które był powiedziáł.
A inne ſpráwy Achábowe y wƺyſtko / co cżynił / y dom z kośći ſłoniowych / który zbudowáł / wƺyſtkie też miáſtá / które pobudowáł / ázáż to nie jeſt ſpiſáne w kronikách o królách Izráelſkich?
Y záſnął Acháb z ojcámi ſwymi / á królowáł Ochozyjáƺ / ſyn jego / miáſto niego.
A Jozáfát / ſyn Azy / pocżął królowáć nád Judą cżwártego roku zá pánowániá Achábá / królá Izráelſkiego.
A Jozáfát miáł trzydzieśći y pięć lát / gdy królowáć pocżął / á dwádzieśćiá y pięć lát królowáł w Jeruzálemie ; á imię mátki jego było Azubá / córká Sálájowá.
Y chodził po wƺyſtkiej drodze Azy / ojcá ſwego / á nie odchyláł śię od niej / cżyniąc to / co było dobrego przed ocżymá Páńſkimi.
Wƺákże iż wyżyn nie poburzyli / jeƺcże lud ofiárowáł y kádził po wyżynách.
Ucżynił też pokój Jozáfát z królem Izráelſkim.
A inne ſpráwy Jozáfátowe / y moc jego / której dokázywáł / y jáko wálcżył / ázáż to nie jeſt nápiſáne w kronikách królów Judzkich?
Ten wyplenił z ziemi oſtátek Sodomcżyków / którzy byli pozoſtáli zá dni Azy / ojcá jego.
Ná ten cżáſ nie było królá w Edomſkiej ziemi ; tylko ſtároſtá był miáſto królá.
Y náſpráwiáł Jozáfát okrętów ná morze / áby chodziły do Ofir po złoto. Ale nie doƺły ; bo śię rozbiły one okręty w Aſyjongáber.
Rzekł tákże był Ochozyjáƺ / ſyn Achábá / do Jozáfátá : Niech jádą ſłudzy moi z ſługámi twymi w okrętách. Ale niechćiáł Jozáfát.
Záſnął tedy Jozáfát z ojcámi ſwymi / y pochowány jeſt z ojcámi ſwymi w mieśćie Dawidá / ojcá ſwego ; á królowáł Jorám / ſyn jego / miáſto niego.
Ochozyjáƺ / ſyn Achábá / pocżął królowáć nád Izráelem w Sámáryi roku śiedmnáſtego Jozáfátá / królá Judzkiego / y królowáł nád Izráelem dwá látá.
Y cżynił złe przed ocżymá Páńſkimi / chodząc drogą ojcá ſwego / y drogą mátki ſwej / y drogą Jeroboámá / ſyná Nábátowego / który przywiódł do grzechów Izráelá.
Służył tákże Báálowi / á kłániáł mu śię / y pobudzáł do gniewu Páná / Bogá Izráelſkiego / według wƺyſtkiego / co cżynił ojćiec jego.
