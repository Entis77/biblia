Tedy król Sálomon rozmiłowáł śię niewiáſt obcych wiele : nie tylko córki Fáráonowej / ále y Moábitcżánek / Ammonitcżánek / Edomcżánek / Sydońcżánek / Hetejcżánek.
Z tych národów / o których powiedźiáł Pán ſynom Izráelſkim : Nie wchodźćie do nich / y one niech nie wchodzą do wáſ ; ábowiem nákłoniłyby ſerce wáƺe zá bogi ſwymi. Do tych przylgnął Sálomon miłośćią.
Ták iż miáł żon królowych śiedm ſet / á záłożnic trzy ſtá ; y odwróćiły żony jego ſerce jego.
Y ſtáło śię / gdy już był Sálomon ſtáry / że żony jego nákłoniły ſerce jego zá bogi cudzymi / ták iż nie było ſerce jego zupełne z Pánem / Bogiem jego / jáko ſerce Dawidá / ojcá jego.
Ale udáł śię Sálomon zá Aſtárotą / boginią Sydońſką / y zá Molochem / obrzydliwośćią Ammonitów.
Y ucżynił Sálomon / co śię nie podobáło Pánu / áni chodźił doſkonále zá Pánem / jáko Dawid / ojćiec jego.
Tedy zbudowáł Sálomon káplicę Chámoſowi / obrzydliwośći Moábſkiej / ná górze przećiw Jeruzálemowi y Molochowi / obrzydliwośći ſynów Ammonowych.
Y ták ucżynił wƺyſtkim żonom ſwym cudzoźiemkom / które kádźiły / y ofiárowáły bogom ſwoim.
Y rozgniewáł śię Pán ná Sálomoná / że śię odwróćiło ſerce jego od Páná / Bogá Izráelſkiego / który śię mu był ukázáł po dwá kroć.
Y zákázáł mu tego / áby nie chodźił zá bogi cudzymi ; wƺákże nie ſtrzegł tego / co Pán przykázáł.
Przetoż rzekł Pán do Sálomoná : Poniewáżeś śię tego dopuśćił / nie ſtrzegąc przymierzá mego / áni wyroków moich / którem ći przykázáł / pewnie oderwę króleſtwo od ćiebie / á dám je ſłudze twojimu /
Wƺákże zá dni twoich nie ucżynię tego dlá Dawidá / ojcá twego ; ále z ręki ſyná twego oderwę je.
Lecż wƺyſtkiego króleſtwá nie oderwę ; pokolenie jedno dám ſynowi twemu dlá Dawidá / ſługi mego / y dlá Jeruzálemu / którem obráł.
Przetoż wzbudźił Pán przećiwniká Sálomonowi / Adádá Edomcżyká z náśieniá królewſkiego / który był w Edom.
Abowiem ſtáło śię / gdy Dawid był w Edom / á Joáb / hetmán wojſká / wyjecháł / áby pochowáł pobite / y pobił wƺyſtkie mężcżyzny w Edom ;
( Bo tám ƺeść mieśięcy mieƺkáł Joáb ze wƺyſtkimi Izráelcżykámi / áż wytráćił wƺyſtkie mężcżyzny w Edom. )
Tedy ućiekł Adád ſám / y niektórzy mężowie Edomſcy z ſług ojcá jego z nim / áby ƺli do Egiptu ; á Adád był chłopcem nie wielkim.
Którzy wyƺedłƺy z Mádyján przyƺli do Fárán / á wźiąwƺy z ſobą niektóre męże z Fárán weƺli do Egiptu / do Fáráoná królá Egipſkiego / który mu dáł dom / y żywność mu náznácżył / dáł mu też y źiemię.
Y ználázł Adád wielką łáſkę w ocżách Fáráonowych / ták / iż mu dáł zá żonę śioſtrę żony ſwej / śioſtrę królowej Táfneſ.
Y porodźiłá mu śioſtrá Táfneſ Gienubátá / ſyná jego / którego odchowáłá Táfneſ w domu Fáráonowym. Y był Gienubát w domu Fáráonowym między ſyny Fáráonowymi.
A gdy uſłyƺáł Adád w Egipćie / iż záſnął Dawid z ojcy ſwymi / á iż umárł Joáb / hetmán wojſká / tedy rzekł Adád do Fáráoná : Puść mię / że pójdę do źiemi mojej.
Któremu odpowiedźiáł Fáráo : Cżegoż ći niedoſtáwá u mnie / że chceƺ iść do źiemi twojej? A on rzekł : Nicżego ; á wƺákże puść mię.
Wzbudźił tákże Bóg náń przećiwniká / Rezoná / ſyná Elijádowego / który był ućiekł od Adárezerá / królá Soby / páná ſwego.
A zebráwƺy do śiebie męże / był kśiążęćiem roty / gdy je Dawid mordowáł ; przetoż odƺedłƺy do Dámáƺku / mieƺkáli w nim / á królowáli nád Dámáƺkiem.
Y był przećiwnikiem Izráelowi po wƺyſtkie dni Sálomonowe / á to oprócż ƺkód / które mu cżynił Adád ; bo śię brzydźił Izráelem / gdy królowáł w Syryi.
Jeroboám też / ſyn Nábátá Efrátejcżyká z Sáredy / ( á imię mátki jego Serwá / którá byłá wdową / )ſługá Sálomonowy / podnióſł przećiwko królowi rękę.
A táć byłá przycżyná / dlá której podnióſł rękę ſwą przećiwko królowi / że Sálomon zbudowáwƺy Mello / zápráwił dźiurę w mieśćie Dawidá / ojcá ſwego.
A Jeroboám był mąż mocny y możny. Przetoż widząc Sálomon młodźieńcá / że był ſpráwny / poſtánowił go nád podátkámi wƺyſtkiemi domu Józefowego.
Y ſtáło śię tegoż cżáſu / gdy Jeroboám wyƺedł z Jeruzálemu / że go ználázł ná drodze Achyjáƺ Sylonitcżyk / prorok / będąc odźiány płáƺcżem nowym ; á tylko ſámi dwáj byli ná polu.
Tedy wźiąwƺy Achyjáƺ płáƺcż nowy / który miáł ná ſobie / rozdárł go ná dwánáśćie ƺtuk.
Y rzekł do Jeroboámá : Weźmij ſobie dźieśięć ƺtuk ; bo ták mówi Pán / Bóg Izráelſki : Oto Já oderwę króleſtwo z rąk Sálomonowych / á dám tobie dźieśięć pokoleń.
Jedno tylko pokolenie zoſtánie mu dlá ſługi mego Dawidá / y dlá miáſtá Jeruzálem / którem obráł ze wƺyſtkich pokoleń Izráelſkich ;
Przeto / że mię opuśćili / á kłániáli śię Aſtároćie / bogini Sydońſkiej / y Chámoſowi / bogu Moábſkiemu / y Molochowi / bogu ſynów Ammonowych / á nie chodźili drogámi mojemi / áby cżynili to / co śię mnie podobá / áni też ſtrzegli wyroków moich / y ſądów moich / jáko Dawid / ojćiec jego.
Wƺákże nie odejmę wƺyſtkiego króleſtwá z ręki jego / owƺem zoſtáwię go kśiążęćiem po wƺyſtkie dni żywotá jego dlá Dawidá / ſługi mego / któregom obráł / który ſtrzegł rozkázániá mego / y wyroków moich.
Ale wźiąwƺy króleſtwo z ręki ſyná jego / dám tobie z niego dźieśięć pokoleń :
A ſynowi jego dám pokolenie jedno / áby zoſtáłá pochodniá Dawidowi / ſłudze memu / po wƺyſtkie dni przedemną w mieśćie Jeruzálemie / którem ſobie obráł / áby tám przebywáło imię moje ;
A ćiebie wezmę / ábyś królowáł nád wƺyſtkiem / cżego żądá duƺá twojá / á będźieƺ królem nád Izráelem.
Przetoż jeźli będźieƺ ſłucháł wƺyſtkiego / coć przykáżę / á będźieƺ chodźił drogámi mojemi / cżyniąc to / co dobrego jeſt w ocżách moich / ſtrzegąc wyroków moich / y przykázáń moich / jáko cżynił Dawid / ſługá mój : tedy będę z tobą / á zbudujęć dom mocny / jákom zbudowáł Dawidowi / y podám ći Izráelá ;
Y trápić będę náśienie Dawidowe dlá tego ; á wƺákże nie po wƺyſtkie dni.
Przetoż Sálomon chćiáł zábić Jároboámá ; ále wſtáwƺy Jeroboám ućiekł do Egiptu / do Seſáká / królá Egipſkiego / y był w Egipćie áż do śmierći Sálomonowej.
A inne ſpráwy Sálomonowe / którekolwiek cżynił / y mądrość jego / izáli nie ſą wypiſáne w kśięgách ſpráw Sálomonowych?
A dni / których królowáł Sálomon w Jeruzálemie nád wƺyſtkim Izráelem / było cżterdźieśći lát.
Y záſnął Sálomon z ojcy ſwymi / á pogrzebiony jeſt w mieśćie Dawidá / ojcá ſwego ; y królowáł Roboám / ſyn jego / miáſto niego.
