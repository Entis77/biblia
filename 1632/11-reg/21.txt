Y ſtáło śię potem : Miáł Nábot Jezreelitá winnicę / którá byłá w Jezreelu podle páłácu Achábá / królá w Sámáryi.
Y rzekł Acháb do Nábotá / mówiąc : Dáj mi winnicę twoję / ábym miáł z niej ogród dlá járzyn / ábowiem bliſká jeſt domu mego ; á dám ći zá nię winnicę lepƺą / niżli tá jeſt ; álbo jeźlić śię zdá / dám ći pieniędzy / ile ſtoi.
Y odpowiedzáł Nábot Achábowi : Nie dáj tego Pánie / ábym ći miáł dáć dziedzictwo ojców moich.
Tedy przyƺedł Acháb do domu ſwego ſmutny y zágniewány dlá ſłowá / które mu rzekł Nábot Jezreelitá / mówiąc : Nie dám ći dziedzictwá ojców moich ; y ukłádł śię ná łożu ſwem / á odwróćił twárz ſwoję / y nie jádł chlebá.
Wtem przyƺedłƺy do niego Jezábelá / żoná jego / rzekłá mu : Przecżże duch twój ták ſmutny / że nie jeƺ chlebá?
Y odpowiedziáł jej : Przeto żem mówił z Nábotem Jezreelitą / y rzekłem mu : Dáj mi winnicę twoję zá pieniądze / álbo jeźli chceƺ / dám ći winnicę zá nię ; ále on odpowiedziáł : Nie dám ći winnicy mojej.
Tedy rzekłá do niego Jezábelá / żoná jego : Y tákże ty ſpráwujeƺ króleſtwo Izráelſkie? Wſtáń / jedz chleb / á bądź dobrej myśli ; já tobie dám winnicę Nábotá Jezreelity.
A ták nápiſáłá liſt imieniem Achábowem / który zápiecżętowáłá piecżęćią jego / y poſłáłá on liſt do ſtárƺych y do przedniejƺych / którzy byli w mieśćie jego / y mieƺkáli z Nábotem.
A nápiſáłá on liſt w ten ſpoſób : Zápowiedźćie poſt / á poſádźćie Nábotá między przedniejƺymi z ludu ;
Y poſtáwćie dwóch mężów przewrotnych przećiw niemu / którzyby przećiwko niemu świádcżyli / mówiąc : Złożecżyłeś Bogu y królowi ; potem wywiedźćie go / á ukámionujćie go / áby umárł.
Y ucżynili mężowie onego miáſtá ſtárśi y przedniejśi / którzy mieƺkáli w onem mieśćie jego / jáko byłá wſkázáłá do nich Jezábelá / według tego / jáko nápiſáno było w liśćie / który poſłáłá do nich.
Zápowiedzieli poſt / y poſádzili Nábotá między przedniejƺymi z ludu.
Potem przyƺli dwáj mężowie przewrotni / y uśiedli przećiw niemu / á świádcżyli przećiwko niemu oni mężowie przewrotni / to jeſt przećiw Nábotowi przed ludem / mówiąc : Złożecżył Nábot Bogu y królowi. Y wywiedli go zá miáſto / y ukámionowáli go / y umárł.
Y poſłáli do Jezábeli / mówiąc : Ukámionowán jeſt Nábot / y umárł.
Y ſtáło śię / gdy uſłyƺáłá Jezábelá / że ukámionowány był Nábot / á iż umárł / rzekłá Jezábelá do Achábá : Wſtáń / pośiądź winnicę Nábotá Jezreelity / któryć jej nie chćiáł dáć zá pieniądze ; ábowiem nie żyje Nábot / ále umárł.
A ták uſłyƺáwƺy Acháb / że umárł Nábot / wſtáł / á ƺedł do winnicy Nábotá Jezreelity / áby ją pośiádł.
Tedy śię ſtáło ſłowo Páńſkie do Elijáƺá Teſbity / mówiąc :
Wſtáń / idź przećiw Achábowi / królowi Izráelſkiemu / który jeſt w Sámáryi ; oto jeſt ná winnicy Nábotowej / do której ƺedł / áby ją pośiádł.
Y rzecżeƺ do niego / mówiąc : Ták mówi Pán : Azáś nie zábił y nie pośiádł? Powiedzże mu / mówiąc : Ták mówi Pán : Ták jáko pſy lizáli krew Nábotowę / ták też pewnie pſy będą lizáć krew twoję.
Y rzekł Acháb do Elijáƺá : A jużeś mię ználázł nieprzyjáćielu mój? A on odpowiedziáł : Ználázłem ; ábowiemeś śię záprzedáł / ábyś cżynił złość przed ocżymá Páńſkimi.
Oto Já przywiodę ná ćię złe / á odejmę potomki twe / y wytrácę z domu Achábowego / áż do nájmniejƺego ƺcżenięćiá / y więźniá / y opuƺcżonego w Izráelu.
A ucżynię z domem twoim / jáko z domem Jeroboámá / ſyná Nábátowego / y jáko z domem Báázy / ſyná Ahyjáƺowego / dlá rozdráźnieniá / któremeś mię do gniewu pobudził / y przywiodłeś do grzechu Izráelá.
Tákże y o Jezábeli rzekł Pán / mówiąc : Pſy zjedzą Jezábelę między murámi Jezreelſkimi.
Tego / który umrze z domu Achábowego w mieśćie / pſy zjedzą / á tego / który umrze ná polu / zjedzą ptáki powietrzne.
Abowiem nie był nikt jáko Acháb / któryby śiebie ſámego záprzedáł / áby cżynił złe przed ocżymá Páńſkimi ; bo go poduƺcżáłá Jezábelá / żoná jego.
Abowiem śię dopuśćił rzecży bárdzo obrzydłych / chodząc zá báłwánámi według wƺyſtkiego / jáko cżynili Amorejcżycy / których wygnáł Pán przed oblicżem ſynów Izráelſkich.
A gdy uſłyƺáł Acháb te ſłowá / rozdárł odzienie ſwoje / á włożywƺy wór ná ćiáło ſwoje / pośćił y leżáł w worze / á chodził po málucżku.
Y ſtáło śię ſłowo Páńſkie do Elijáƺá Teſbity / mówiąc :
Widziáłżeś / jáko śię upokorzył Acháb przed twárzą moją? Poniewáż śię tedy upokorzył przed twárzą moją / nie przywiodę tego złego zá dni jego ; ále zá dni ſyná jego przywiodę to złe ná dom jego.
