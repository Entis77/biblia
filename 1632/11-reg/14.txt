Tegoż cżáſu rozniemógł śię Abijás / ſyn Jeroboámá.
Y rzekł Jeroboám do żony ſwej : Wſtáń teraz / á odmień śię / áby nie poznáno / żeś ty żoną Jeroboámową / á idź do Sylo ; oto tám jeſt Achyjáƺ prorok / który mi powiedźiał / żem miał zoſtáć królem nád tym ludem.
A wźiąwƺy z ſobą dźieśięćioro chlebá / y plácków / y fáſkę miodu / idź do niego ; on ći oznájmi / co śię ſtánie dźiećięćiu.
Y ucżyniłá ták żoná Jeroboámowá / á wſtáwƺy poƺłá do Sylo / y weƺłá do domu Achyjáƺowego ; ále Achyjáƺ nie mógł już widźieć / bo mu były záƺły ocży dla ſtárośći jego.
A Pán rzekł do Achyjáƺá : Oto żoná Jeroboámowá wchodźi / áby śię od ćiebie cżego wywiedźiáłá o ſynu ſwym / przeto że choruje ; ále ták á ták rzecżeƺ jey / á ſtánie śię / gdy będźie wchodźiłá / zmyśli śię być inƺą.
Przetoż gdy uſłyƺał Achyjáƺ tupánie nóg jey / wchodzącey we drzwi / rzekł : Wnijdź żono Jeroboámowá ; przecż śię zmyśláƺ być inƺą? Jam bowiem ſrogim poſłem do ćiebie.
Idź / powiedz Jeroboámowi : Ták mówi Pán / Bóg Izráelſki : Poniewáżem ćię wywyżƺył z pośrzodku ludu / á poſtánowiłem ćię kśiążęćiem nád ludem moim Izráelſkim ;
Y oderwáłem króleſtwo od domu Dawidowego / á dáłem je tobie / tyś jednák nie był jáko ſługá mój Dawid / który ſtrzegł rozkázánia mego / y który chodźił zá mną cáłem ſercem ſwojim / to tylko cżyniąc / co jeſt dobrego w ocżách mojich ;
Aleś cżynił złe nád wƺyſtkie / którzy byli przed tobą ; ábowiem odƺedƺy / pocżyniłeś ſobie bogi cudze / y láne / ábyś mię pobudźił do gniewu / á mnieś zárzućił w tył ſobie :
Przetoż oto Ja przywiodę złe ná dom Jeroboámowy / y wytrácę z Jeroboámá áż do nájmniejƺego ƺcżenięćiá / więźniá / y opuƺcżonego w Izráelu / y wymiotę oſtátki domu Jeroboámowego / jáko wymiátáją gnój / áż do cżyſtá.
Tego / który z domu Jeroboámowego umrze w mieśćie / zjedzą pſy / á który umrze ná polu / zjedzą ptáki powietrzne / ponieważ Pán wyrzekł.
A ty wſtáwƺy idź do domu twego á gdy wchodźić będźieƺ do miáſtá / tedy umrze dźiećię.
Y będźie go płákał wƺyſtek Izráel / y pochowáją go ; bo ten ſám z domu Jeroboámowego wnijdźie do grobu / przeto iż śię ználázło o nim ſámym ſłowo dobre od Páná / Bogá Izráelſkiego w domu Jeroboámowym.
Wƺákże poſtánowi ſobie Pán królá nád Izráelem / który wykorzeni dom Jeroboámowy dniá tego ; á co mówię / wzbudźi? Y owƺem już wzbudźił.
Y uderzy Pán Izráelá / y záchwieje nim / jáko śię chwieje trzćiná ná wodách / á wykorzeni Izráelá z źiemi tey dobrej / którą dał ojcom ich / y rozproƺy je zá rzekę / przeto iż ſobie pocżynili gáje / wzruƺájąc Páná ku gniewu.
A ták wydá Izráelá dla grzechu Jeroboámowego / który grzeƺył / y który do grzechu przywiódł Izráelá.
Tedy wſtáłá żoná Jeroboámowá / y poƺłá / á przyƺłá do Terſá ; á gdy wſtępowáłá ná próg domu / umárło dźiećię.
Y pochowáli je / á płákał go wƺyſtek Izráel według ſłowá Páńſkiego / które opowiedźiał przez ſługę ſwego Achyjáƺá proroká.
A inne ſpráwy Jeroboámowe / jáko wálcżył / y jáko królował / oto ſpiſáne ſą w kronikách o królách Izráelſkich.
A dni / których królował Jeroboám / było dwádźieśćiá y dwá lat / y záſnął z ojcy ſwymi / á Nádáb / ſyn jego / królował miáſto niego.
Roboám też / ſyn Sálomoná / królował w Judźie. A było Roboámowi cżterdźieśći lat y jeden / gdy pocżął królowáć / á śiedmnaśćie lat królował w mieśćie Jeruzálemie / które Pán obrał ze wƺyſtkich pokoleń Izráelſkich / áby tám przebywáło imię jego. A imię mátki jego było Náámá / Ammonitká.
Y cżynił Judá złe przed Pánem / á wzruƺyli go ku gniewu grzechámi ſwemi / któremi grzeƺyli nád wƺyſtko / co cżynili ojcowie ich.
Abowiem y oni pobudowáli ſobie wyżyny / y ſłupy / y gáje ná káżdym págórku wyſokim / y pod káżdym drzewem źielonem.
Byli też y Sodomcżycy w oney źiemi / ſpráwujący śię według wƺyſtkich obrzydliwośći pogánów / które wyrzućił Pán od oblicżnośći ſynów Izráelſkich.
Y ſtáło śię roku piątego królowánia Roboámá / że wyćiągnął Seſák / król Egipſki / przećiw Jeruzálemowi.
Y pobrał ſkárby domu Páńſkiego / y ſkárby domu królewſkiego / wƺyſtko to pobrał ; wźiął też wƺyſtkie tárcże złote / które był ſpráwił Sálomon ;
Miáſto których król Roboám ſpráwił tárcże miedźiáne / y porucżył je przełożonym nád piechotą / którzy ſtrzegli drzwi domu królewſkiego.
A gdy wchodźił król do domu Páńſkiego / bráłá je piechotá / y záśię odnośiłá do komor ſwojich.
A inne ſpráwy Roboámowe / y wƺyſtko co cżynił / ázaż nie ſą nápiſáne w kronikách o królách Judzkich?
Y byłá wojná miedzy Roboámem y miedzy Jeroboámem po wƺyſtkie dni.
Y záſnął Roboám z ojcy ſwymi / á pochowány jeſt z nimi w mieśćie Dawidowem ; á imię mátki jego było Náámá / Ammonitká. Y królował Abijám / ſyn jego / miáſto niego.
