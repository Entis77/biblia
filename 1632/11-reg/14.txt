Tegoż cżáſu rozniemógł śię Abijáſ / ſyn Jeroboámá.
Y rzekł Jeroboám do żony ſwej : Wſtáń teráz / á odmień śię / áby nie poznáno / żeś ty żoną Jeroboámową / á idź do Sylo ; oto tám jeſt Achyjáƺ prorok / który mi powiedziáł / żem miáł zoſtáć królem nád tym ludem.
A wziąwƺy z ſobą dzieśięćioro chlebá / y plácków / y fáſkę miodu / idź do niego ; on ći oznájmi / co śię ſtánie dziećięćiu.
Y ucżyniłá ták żoná Jeroboámowá / á wſtáwƺy poƺłá do Sylo / y weƺłá do domu Achyjáƺowego ; ále Achyjáƺ nie mógł już widzieć / bo mu były záƺły ocży dlá ſtárośći jego.
A Pán rzekł do Achyjáƺá : Oto żoná Jeroboámowá wchodzi / áby śię od ćiebie cżego wywiedziáłá o ſynu ſwym / przeto że choruje ; ále ták á ták rzecżeƺ jej / á ſtánie śię / gdy będzie wchodziłá / zmyśli śię być inƺą.
Przetoż gdy uſłyƺáł Achyjáƺ tupánie nóg jej / wchodzącej we drzwi / rzekł : Wnijdź żono Jeroboámowá ; przecż śię zmyśláƺ być inƺą? Jám bowiem ſrogim poſłem do ćiebie.
Idź / powiedz Jeroboámowi : Ták mówi Pán / Bóg Izráelſki : Poniewáżem ćię wywyżƺył z pośrodku ludu / á poſtánowiłem ćię kśiążęćiem nád ludem moim Izráelſkim ;
Y oderwáłem króleſtwo od domu Dawidowego / á dáłem je tobie / tyś jednák nie był jáko ſługá mój Dawid / który ſtrzegł rozkázániá mego / y który chodził zá mną cáłem ſercem ſwojim / to tylko cżyniąc / co jeſt dobrego w ocżách moich ;
Aleś cżynił złe nád wƺyſtkie / którzy byli przed tobą ; ábowiem odƺedłƺy / pocżyniłeś ſobie bogi cudze / y láne / ábyś mię pobudził do gniewu / á mnieś zárzućił w tył ſobie :
Przetoż oto Já przywiodę złe ná dom Jeroboámowy / y wytrácę z Jeroboámá áż do nájmniejƺego ƺcżenięćiá / więźniá / y opuƺcżonego w Izráelu / y wymiotę oſtátki domu Jeroboámowego / jáko wymiátáją gnój / áż do cżyſtá.
Tego / który z domu Jeroboámowego umrze w mieśćie / zjedzą pſy / á który umrze ná polu / zjedzą ptáki powietrzne / poniewáż Pán wyrzekł.
A ty wſtáwƺy idź do domu twego á gdy wchodzić będzieƺ do miáſtá / tedy umrze dziećię.
Y będzie go płákáł wƺyſtek Izráel / y pochowáją go ; bo ten ſám z domu Jeroboámowego wnijdzie do grobu / przeto iż śię ználázło o nim ſámym ſłowo dobre od Páná / Bogá Izráelſkiego w domu Jeroboámowym.
Wƺákże poſtánowi ſobie Pán królá nád Izráelem / który wykorzeni dom Jeroboámowy dniá tego ; á co mówię / wzbudzi? Y owƺem już wzbudził.
Y uderzy Pán Izráelá / y záchwieje nim / jáko śię chwieje trzćiná ná wodách / á wykorzeni Izráelá z ziemi tej dobrej / którą dáł ojcom ich / y rozproƺy je zá rzekę / przeto iż ſobie pocżynili gáje / wzruƺájąc Páná ku gniewu.
A ták wydá Izráelá dlá grzechu Jeroboámowego / który grzeƺył / y który do grzechu przywiódł Izráelá.
Tedy wſtáłá żoná Jeroboámowá / y poƺłá / á przyƺłá do Terſá ; á gdy wſtępowáłá ná próg domu / umárło dziećię.
Y pochowáli je / á płákáł go wƺyſtek Izráel według ſłowá Páńſkiego / które opowiedziáł przez ſługę ſwego Achyjáƺá proroká.
A inne ſpráwy Jeroboámowe / jáko wálcżył / y jáko królowáł / oto ſpiſáne ſą w kronikách o królách Izráelſkich.
A dni / których królowáł Jeroboám / było dwádzieśćiá y dwá lát / y záſnął z ojcy ſwymi / á Nádáb / ſyn jego / królowáł miáſto niego.
Roboám też / ſyn Sálomoná / królowáł w Judzie. A było Roboámowi cżterdzieśći lát y jeden / gdy pocżął królowáć / á śiedmnáśćie lát królowáł w mieśćie Jeruzálemie / które Pán obráł ze wƺyſtkich pokoleń Izráelſkich / áby tám przebywáło imię jego. A imię mátki jego było Náámá / Ammonitká.
Y cżynił Judá złe przed Pánem / á wzruƺyli go ku gniewu grzechámi ſwemi / któremi grzeƺyli nád wƺyſtko / co cżynili ojcowie ich.
Abowiem y oni pobudowáli ſobie wyżyny / y ſłupy / y gáje ná káżdym págórku wyſokim / y pod káżdem drzewem zielonem.
Byli też y Sodomcżycy w onej ziemi / ſpráwujący śię według wƺyſtkich obrzydliwośći pogánów / które wyrzućił Pán od oblicżnośći ſynów Izráelſkich.
Y ſtáło śię roku piątego królowániá Roboámá / że wyćiągnął Seſák / król Egipſki / przećiw Jeruzálemowi.
Y pobráł ſkárby domu Páńſkiego / y ſkárby domu królewſkiego / wƺyſtko to pobráł ; wziął też wƺyſtkie tárcże złote / które był ſpráwił Sálomon ;
Miáſto których król Roboám ſpráwił tárcże miedziáne / y porucżył je przełożonym nád piechotą / którzy ſtrzegli drzwi domu królewſkiego.
A gdy wchodził król do domu Páńſkiego / bráłá je piechotá / y záśię odnośiłá do komor ſwoich.
A inne ſpráwy Roboámowe / y wƺyſtko co cżynił / ázáż nie ſą nápiſáne w kronikách o królách Judzkich?
Y byłá wojná między Roboámem y między Jeroboámem po wƺyſtkie dni.
Y záſnął Roboám z ojcy ſwymi / á pochowány jeſt z nimi w mieśćie Dawidowem ; á imię mátki jego było Náámá / Ammonitká. Y królowáł Abijám / ſyn jego / miáſto niego.
