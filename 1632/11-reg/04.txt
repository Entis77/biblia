A ták król Sálomon był królem nád wƺyſtkim Izráelem.
A teć ſą kśiążętá / które miał : Azáryjáƺ / ſyn Sádoká kápłáná.
Elichoref y Achijá / ſynowie Syſy / byli piſárzámi ; Jozáfát / ſyn Ahiludá / kánclerzem ;
A Bánájás / ſyn Jojády / był hetmánem ; Sádok záś y Abijátár kápłánámi.
A Azáryjáƺ / ſyn Nátáná / nád urzędnikámi / á Zábud / ſyn Nátáná / był kśiążęćiem / przyjáćielem królewſkim.
Ahiſár záś był przełożony nád domem / á Adonirám / ſyn Abdy / nád wybránym ludem.
Miał też Sálomon dwánáśćie przełożonych nád wƺyſtkim Izráelem / którzy dodáwáli żywnośći królowi y domowi jego. Káżdy z nich przez jeden mieśiąc w roku królowi żywnośći dodáwał.
A teć ſą imioná ich : Syn Hurá ná górze Efráim ;
Syn Dekárá w Mákás / y w Sálbim / y w Betſámes / y w Elon y Bethánán ;
Syn Heſedá w Arubot / który trzymał Socho y wƺyſtkę źiemię Chefer ;
Syn Abinádábá / który trzymał wƺyſtkie gránice Dor / á Táfet / córkę Sálomonową / miał zá żonę.
Bááná / ſyn Ahiludá / który trzymał Tánách y Mágieddo / y wƺyſtko Betſán / które jeſt podle Sártány pod Jezreelem / od Betſán áż do Abelmeholá / áż zá Jekmáám.
Syn Gáber w Rámot Gáláádſkiem / który trzymał wśi Jáirá / ſyná Mánáſeſowego / które leżą w Gáláád. Jemu też náleżáłá kráiná Argob / która jeſt w Báſán / ƺeśćdźieśiąt miaſt wielkich murowánych z záworámi miedźiánemi.
Achinádáb / ſyn Iddoná / w Máhánáim.
Achimáás w Neftálim / który też pojął Báſemátę / córkę Sálomonową / zá żonę.
Bááná / ſyn Huſái / w Aſer y w Alot.
Jozáfát / ſyn Páruáchá / w Iſáſchár.
Semej / ſyn Eli / w Benjámin.
Gáber / ſyn Ury / w źiemi Gáláád y w źiemi Sehoná królá Amorejſkiego / y Ogá królá Báſáńſkiego ; á ten ſám był rządcą oney źiemi.
Tedy Judá y Izráel będąc niezlicżeni jáko piáſek / który jeſt nád morzem w mnóſtwie / jedli / y pili / y weſelili śię.
A Sálomon pánował nád wƺyſtkiemi króleſtwy od rzeki áż do źiemi Filiſtyńſkiej / y áż do gránicy Egipſkiej. Y przynośili dáry / á ſłużyli Sálomonowi po wƺyſtkie dni żywotá jego.
A tenći był rozchód Sálomoná ná káżdy dźień / trzydźieśći korcy mąki biáłej / y ƺeśćdźieśiąt korcy inney mąki.
Dźieśięć wołów kármnych / y dwádźieśćiá wołów páſtewnych / y ſto owiec / oprócż jeleni y ſárn / y báwołów / y ptáſtwá kármnego.
Abowiem on pánował wƺędy z tey ſtrony rzeki od Táſfá áż do Gázy nád wƺyſtkimi królmi / którzy byli przed rzeką / á miał pokój ze wƺyſtkich ſtron w około.
Y mieƺkał Judá y Izráel bezpiecżnie / káżdy pod winną máćicą ſwoją / y pod figą ſwoją / od Dán áż do Beerſebá / po wƺyſtkie dni Sálomonowe.
Miał też Sálomon cżterdźieśći tyśięcy koni ná ſtániu do wozów ſwojich / á dwánáśćie tyśięcy jezdnych.
A ták podejmowáli oni przełożeni królá Sálomoná / y wƺyſtkie / którzy przychodźili do ſtołu królá Sálomoná / káżdy mieśiącá ſwego / nie dopuƺcżájąc / áby ná cżym ſchodźić miáło.
Jęcżmień tákże / y plewy dla koni y mułów / zwoźili ná to miejſce / gdźie był król / káżdy według tego / jáko mu poſtánowiono.
Nádto dał Bóg Sálomonowi mądrość y roztropność bárzo wielką / á przeſtronność ſercá / jáko piáſek / który jeſt ná brzegu morſkim.
Abowiem więkƺá byłá mądrość Sálomonowá / niżli mądrość wƺyſtkich národów wſchodnich / y niżli wƺelká mądrość Egipcżánów.
Owƺem mędrƺym był nád wƺyſtkie ludźie / áż y nád Etáná Ezráhytę / y nád Hemáná / y Chálkolá / y Dárdá / ſyny Máholowe ; á był ſłáwny u wƺyſtkich národów okolicżnych.
Nádto złożył trzy tyśiące przypowieśći / á pieśni jego było tyśiąc y pięć.
Rozpráwiał też o drzewách / pocżąwƺy od cedru / który jeſt ná Libánie / áż do hizopu / który wyráſtá z śćiány. Mówił też o zwierzętách y o ptákách / y o gádźinách / y o rybách.
Przetoż przychodźili ze wƺyſtkich národów ſłucháć mądrośći Sálomonowej / y od wƺyſtkich królów źiemi / którzy ſłyƺeli o mądrośći jego.
