A ták król Sálomon był królem nád wƺyſtkim Izráelem.
A teć ſą kśiążętá / które miáł : Azáryjáƺ / ſyn Sádoká kápłáná.
Elichoref y Achijá / ſynowie Syſy / byli piſárzámi ; Jozáfát / ſyn Ahiludá / kánclerzem ;
A Bánájáſ / ſyn Jojády / był hetmánem ; Sádok záś y Abijátár kápłánámi.
A Azáryjáƺ / ſyn Nátáná / nád urzędnikámi / á Zábud / ſyn Nátáná / był kśiążęćiem / przyjáćielem królewſkim.
Ahiſár záś był przełożony nád domem / á Adonirám / ſyn Abdy / nád wybránym ludem.
Miáł też Sálomon dwánáśćie przełożonych nád wƺyſtkim Izráelem / którzy dodáwáli żywnośći królowi y domowi jego. Káżdy z nich przez jeden mieśiąc w roku królowi żywnośći dodáwáł.
A teć ſą imioná ich : Syn Hurá ná górze Efráim ;
Syn Dekárá w Mákáſ / y w Sálbim / y w Betſámeſ / y w Elon y Bethánán ;
Syn Heſedá w Arubot / który trzymáł Socho y wƺyſtkę ziemię Chefer ;
Syn Abinádábá / który trzymáł wƺyſtkie gránice Dor / á Táfet / córkę Sálomonową / miáł zá żonę.
Bááná / ſyn Ahiludá / który trzymáł Tánách y Mágieddo / y wƺyſtko Betſán / które jeſt podle Sártány pod Jezreelem / od Betſán áż do Abelmeholá / áż zá Jekmáám.
Syn Gáber w Rámot Gáláádſkiem / który trzymáł wśi Jáirá / ſyná Mánáſeſowego / które leżą w Gáláád. Jemu też náleżáłá kráiná Argob / którá jeſt w Báſán / ƺeśćdzieśiąt miáſt wielkich murowánych z záworámi miedziánemi.
Achinádáb / ſyn Iddoná / w Máhánáim.
Achimááſ w Neftálim / który też pojął Báſemátę / córkę Sálomonową / zá żonę.
Bááná / ſyn Huſái / w Aſer y w Alot.
Jozáfát / ſyn Páruáchá / w Iſáſchár.
Semej / ſyn Eli / w Benjámin.
Gáber / ſyn Ury / w ziemi Gáláád y w ziemi Sehoná królá Amorejſkiego / y Ogá królá Báſáńſkiego ; á ten ſám był rządcą onej ziemi.
Tedy Judá y Izráel będąc niezlicżeni jáko piáſek / który jeſt nád morzem w mnóſtwie / jedli / y pili / y weſelili śię.
A Sálomon pánowáł nád wƺyſtkiemi króleſtwy od rzeki áż do ziemi Filiſtyńſkiej / y áż do gránicy Egipſkiej. Y przynośili dáry / á ſłużyli Sálomonowi po wƺyſtkie dni żywotá jego.
A tenći był rozchód Sálomoná ná káżdy dzień / trzydzieśći korcy mąki biáłej / y ƺeśćdzieśiąt korcy innej mąki.
Dzieśięć wołów kármnych / y dwádzieśćiá wołów páſtewnych / y ſto owiec / oprócż jeleni y ſárn / y báwołów / y ptáſtwá kármnego.
Abowiem on pánowáł wƺędy z tej ſtrony rzeki od Táſfá áż do Gázy nád wƺyſtkimi królmi / którzy byli przed rzeką / á miáł pokój ze wƺyſtkich ſtron w około.
Y mieƺkáł Judá y Izráel bezpiecżnie / káżdy pod winną máćicą ſwoją / y pod figą ſwoją / od Dán áż do Beerſebá / po wƺyſtkie dni Sálomonowe.
Miáł też Sálomon cżterdzieśći tyśięcy koni ná ſtániu do wozów ſwoich / á dwánáśćie tyśięcy jezdnych.
A ták podejmowáli oni przełożeni królá Sálomoná / y wƺyſtkie / którzy przychodzili do ſtołu królá Sálomoná / káżdy mieśiącá ſwego / nie dopuƺcżájąc / áby ná cżem ſchodzić miáło.
Jęcżmień tákże / y plewy dlá koni y mułów / zwozili ná to miejſce / gdzie był król / káżdy według tego / jáko mu poſtánowiono.
Nádto dáł Bóg Sálomonowi mądrość y roztropność bárdzo wielką / á przeſtronność ſercá / jáko piáſek / który jeſt ná brzegu morſkim.
Abowiem więkƺá byłá mądrość Sálomonowá / niżli mądrość wƺyſtkich národów wſchodnich / y niżli wƺelká mądrość Egipcżánów.
Owƺem mędrƺym był nád wƺyſtkie ludzie / áż y nád Etáná Ezráhytę / y nád Hemáná / y Chálkolá / y Dárdá / ſyny Máholowe ; á był ſłáwny u wƺyſtkich národów okolicżnych.
Nádto złożył trzy tyśiące przypowieśći / á pieśni jego było tyśiąc y pięć.
Rozpráwiáł też o drzewách / pocżąwƺy od cedru / który jeſt ná Libánie / áż do hizopu / który wyráſtá z śćiány. Mówił też o zwierzętách y o ptákách / y o gádzinách / y o rybách.
Przetoż przychodzili ze wƺyſtkich národów ſłucháć mądrośći Sálomonowej / y od wƺyſtkich królów ziemi / którzy ſłyƺeli o mądrośći jego.
