Tedy rzekł Elijáƺ Teſbitá / jeden z obywáteli Gáláádu / do Achábá : Jáko żywy Pán / Bóg Izráelſki / przed którego oblicżnośćią ſtoję / że nie będźie tych lat roſy / áni deƺcżu / jedno według ſłów uſt mojich.
Y ſtáło śię ſłowo Páńſkie do niego / mówiąc :
Odejdź ſtąd / á obróć śię ná wſchód ſłońcá / y ſkryj śię u potoku Cháryt / który jeſt przećiwko Jordánowi.
Y będźieƺ pił z potoku : á rozkazałem krukom / áby ćię tám żywili.
Y poƺedł / á ucżynił według ſłowá Páńſkiego / y przyƺedƺy uśiádł u potoku Cháryt / który był przećiwko Jordánowi.
A kruki przynośiły mu chleb y mięſo ráno / tákże chleb y mięſo w wiecżór ; á pił z potoku.
Lecż po wyjśćiu niektórych dni wyſechł on potok ; bo nie pádał deƺcż ná źiemię.
Y ſtáło śię ſłowo Páńſkie do niego / mówiąc :
Wſtáń / idź do Sárepty Sydońſkiej / y mieƺkáj tám ; otom tám rozkázał niewieśćie wdowie / áby ćię żywiłá.
Tedy on wſtáwƺy ƺedł do Sárepty / y przyƺedł do brámy miáſtá / á oto tám niewiáſtá wdowá zbieráłá drwá ; który záwołáwƺy jey / rzekł : Przynieś mi proƺę trochę wody w nácżyniu / ábym śię nápił.
A gdy oná ƺłá / áby przynioſłá / tedy ná nię záwołał / y rzekł : Przynieś mi też proƺę ƺtucżkę chlebá w ręce twojey.
Y odpowiedźiáłá : Jáko żywy Pán / Bóg twój / żeć niemám piecżonego chlebá / oprócż z gárść pełną mąki w gárncu / á trochę oliwy w báńce ; á oto zbierám trochę drewek / ábym ƺłá / y zgotowáłá to ſobie y ſynowi ſwemu / á zjádƺy to / ábyſmy pomárli.
Tedy rzekł do niej Elijáƺ : Nie bój śię. Idź / ucżyń jákoś rzekłá : wƺákże ucżyń mi z tego pierwej podpłomyk máły / y przynieś mi ; potem też ſobie y ſynowi ſwemu ucżyniƺ.
Abowiem ták powiedźiał Pán / Bóg Izráelſki : Mąká z gárncá tego nie będźie ſtráwioná / áni oliwy z tey báńki ubędźie / áż do dniá / gdy Pán ſpuśći deƺcż ná źiemię.
Y poƺłá / á ucżyniłá podług ſłowá Elijáƺowego / y jádłá oná y on / y wƺyſtká cżeládź jey / áż śię wypełniły te dni.
Nie byłá ſtráwioná mąká z onego gárncá / áni oliwy z báńki ubyło / według ſłowá Páńſkiego / które powiedźiał przez Elijáƺá.
Y ſtáło śię potem / że śię rozniemógł ſyn oney niewiáſty / páni domu onego / á byłá niemoc jego bárzo ćiężká / ták / że w nim tchu nie zoſtáło.
Przetoż rzekłá do Elijáƺá : Cóż mnie y tobie / mężu Boży? przyƺedłeś do mnie / ábyś przywiódƺy ná pámięć niepráwość moję / umorzył ſyná mego?
Y rzekł do niej : Dáj mi ſyná twego ; y wźiąwƺy go z łoná jey / wnióſł go ná ſálę / ná którey mieƺkał / y położył go ná łożu ſwojim.
Y wołał do Páná / á rzekł : Pánie / Boże mój / izali też utrápiƺ wdowę / u którey mieƺkám / iżeś zábił ſyná jey?
A rozćiągnąwƺy śię nád dźiećięćiem po trzy kroć / wołał do Páná / mówiąc : Pánie / Boże mój / niecháj śię proƺę wróći duƺá dźiećiątká tego w ćiáło jego.
Y wyſłuchał Pán głos Elijáƺowy : y wróćiłá śię duƺá dźiećięćiá w ćiáło jego / y ożyło.
Tedy wźiął Elijáƺ dźiećię / y znióſł je z ſáli do domu / á oddał go mátce jego / y rzekł Elijáƺ : Wej / ſyn twój żyje.
Y rzekłá niewiáſtá do Elijáƺá : Terazem poznáłá / iżeś jeſt mąż Boży / á ſłowo Páńſkie w uśćiech twojich jeſt práwdá.
