Tedy rzekł Elijáƺ Teſbitá / jeden z obywáteli Gáláádu / do Achábá : Jáko żywy Pán / Bóg Izráelſki / przed którego oblicżnośćią ſtoję / że nie będźie tych lát roſy / áni deƺcżu / jedno według ſłów uſt moich.
Y ſtáło śię ſłowo Páńſkie do niego / mówiąc :
Odejdź ſtąd / á obróć śię ná wſchód ſłońcá / y ſkryj śię u potoku Cháryt / który jeſt przećiwko Jordánowi.
Y będźieƺ pił z potoku : á rozkázáłem krukom / áby ćię tám żywili.
Y poƺedł / á ucżynił według ſłowá Páńſkiego / y przyƺedłƺy uśiádł u potoku Cháryt / który był przećiwko Jordánowi.
A kruki przynośiły mu chleb y mięſo ráno / tákże chleb y mięſo w wiecżór ; á pił z potoku.
Lecż po wyjśćiu niektórych dni wyſechł on potok ; bo nie pádáł deƺcż ná źiemię.
Y ſtáło śię ſłowo Páńſkie do niego / mówiąc :
Wſtáń / idź do Sárepty Sydońſkiej / y mieƺkáj tám ; otom tám rozkázáł niewieśćie wdowie / áby ćię żywiłá.
Tedy on wſtáwƺy ƺedł do Sárepty / y przyƺedł do brámy miáſtá / á oto tám niewiáſtá wdowá zbieráłá drwá ; który záwołáwƺy jej / rzekł : Przynieś mi proƺę trochę wody w nácżyniu / ábym śię nápił.
A gdy oná ƺłá / áby przynioſłá / tedy ná nię záwołáł / y rzekł : Przynieś mi też proƺę ƺtucżkę chlebá w ręce twojej.
Y odpowiedźiáłá : Jáko żywy Pán / Bóg twój / żeć niemám piecżonego chlebá / oprócż z gárść pełną mąki w gárncu / á trochę oliwy w báńce ; á oto zbierám trochę drewek / ábym ƺłá / y zgotowáłá to ſobie y ſynowi ſwemu / á zjádłƺy to / ábyśmy pomárli.
Tedy rzekł do niej Elijáƺ : Nie bój śię. Idź / ucżyń jákoś rzekłá : wƺákże ucżyń mi z tego pierwej podpłomyk máły / y przynieś mi ; potem też ſobie y ſynowi ſwemu ucżyniƺ.
Abowiem ták powiedźiáł Pán / Bóg Izráelſki : Mąká z gárncá tego nie będźie ſtráwioná / áni oliwy z tej báńki ubędźie / áż do dniá / gdy Pán ſpuśći deƺcż ná źiemię.
Y poƺłá / á ucżyniłá podług ſłowá Elijáƺowego / y jádłá oná y on / y wƺyſtká cżeládź jej / áż śię wypełniły te dni.
Nie byłá ſtráwioná mąká z onego gárncá / áni oliwy z báńki ubyło / według ſłowá Páńſkiego / które powiedźiáł przez Elijáƺá.
Y ſtáło śię potem / że śię rozniemógł ſyn onej niewiáſty / páni domu onego / á byłá niemoc jego bárdzo ćiężká / ták / że w nim tchu nie zoſtáło.
Przetoż rzekłá do Elijáƺá : Cóż mnie y tobie / mężu Boży? przyƺedłeś do mnie / ábyś przywiódłƺy ná pámięć niepráwość moję / umorzył ſyná mego?
Y rzekł do niej : Dáj mi ſyná twego ; y wźiąwƺy go z łoná jej / wnióſł go ná ſálę / ná której mieƺkáł / y położył go ná łożu ſwojim.
Y wołáł do Páná / á rzekł : Pánie / Boże mój / izáli też utrápiƺ wdowę / u której mieƺkám / iżeś zábił ſyná jej?
A rozćiągnąwƺy śię nád dźiećięćiem po trzy kroć / wołáł do Páná / mówiąc : Pánie / Boże mój / niecháj śię proƺę wróći duƺá dźiećiątká tego w ćiáło jego.
Y wyſłucháł Pán głoſ Elijáƺowy : y wróćiłá śię duƺá dźiećięćiá w ćiáło jego / y ożyło.
Tedy wźiął Elijáƺ dźiećię / y znióſł je z ſáli do domu / á oddáł go mátce jego / y rzekł Elijáƺ : Wej / ſyn twój żyje.
Y rzekłá niewiáſtá do Elijáƺá : Terázem poznáłá / iżeś jeſt mąż Boży / á ſłowo Páńſkie w uśćiech twoich jeſt práwdá.
