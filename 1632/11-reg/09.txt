Y ſtáło śię / gdy dokońcżył Sálomon budowánia domu Páńſkiego / y domu królewſkiego / y wƺyſtkiego / co żądał Sálomon y chćiał ucżynić /
Że śię Pán ukázał Sálomonowi powtóre / jáko mu śię był ukázał w Gábáon.
Y rzekł Pán do niego : Wyſłucháłem modlitwę twoję / y prośbę twoję / którąś śię modlił przedemną / á poświęćiłem ten dom / któryś zbudował / áby tám przebywáło imię moje áż ná wieki ; y będą tám ocży moje y ſerce moje po wƺyſtkie dni.
A jeſli ty będźieƺ chodźił przedemną / jáko chodźił Dawid / oćiec twój / w doſkonáłośći ſercá y w proſtośći / á będźieƺ śię ſpráwował według wƺyſtkiego / com ći przykázał / ſtrzegąc wyroków mojich y ſądów mojich :
Tedy utwierdzę ſtolicę króleſtwá twego nád Izráelem ná wieki / jákom powiedźiał Dawidowi / ojcu twemu / mówiąc : Nie będźie odjęty z narodu twego mąż z ſtolicy Izráelſkiej.
Ale jeſli śię názád odwróćićie wy y ſynowie wáśi ode mnie / á nie będźiećie ſtrzegli przykázáń mojich / y wyroków mojich / którem wam podał / ále odƺedƺy będźiećie ſłużyli bogom cudzym / y będźiećie śię im kłániáli :
Tedy wytrácę Izráelá z źiemi / którąm im dał / á dom / którym poświęćił imieniowi memu / odrzucę od oblicża mego / á będźie Izráel przypowieśćią y báśnią miedzy wƺyſtkimi národámi.
A ták y ten dom / który był ſłáwny / káżdemu mimo idącemu będźie ná podźiw y ná poświſtánie / y rzecże : Przecżże ták ucżynił Pán tey źiemi y temu domowi?
Tedy odpowiedzą : Przeto / iż opuśćili Páná / Bogá ſwego / który wywiódł ojce ich z źiemi Egipſkiej / á chwyćili śię bogów cudzych / y kłániáli śię im / á ſłużyli im : dla tegoż przywiódł Pán ná nie to wƺyſtko złe.
Y ſtáło śię po wyśćiu dwudźieſtu lat / w których zbudował Sálomon owe obá domy / dom Páńſki y dom królewſki.
Do cżego Hirám / król Tyrſki / nádał był Sálomonowi drzewá cedrowego / y drzewá jodłowego / y złotá / ile jedno chćiał : tedy też król Sálomon dał Hirámowi dwádźieśćiá miaſt w źiemi Gálilejſkiej.
Y wyjechał Hirám z Tyru / áby oglądał miáſtá / które mu dał Sálomon : ále mu śię niepodobáły.
Y rzekł : Cóż to zá miáſtá / któreś mi dał / bráćie mój? Y názwał je źiemią Chábul / áż do dniá tego.
Abowiem poſłał był Hirám królowi ſto y dwádźieśćiá tálentów złotá.
A przycżyná poboru / który był rozkázał wybieráć król Sálomon / byłá / áby zbudował dom Páńſki / y dom ſwój / y Mello / y mury Jeruzálemſkie / y Háſor / y Mágieddo / y Gázer.
Fáráo bowiem / król Egipſki / wyćiągnął był / y wźiął Gázer / y popálił je ogniem / á Chánánejcżyká / który mieƺkał w tym mieśćie / wymordował / á dał je zá poſág córce ſwej / żonie Sálomonowej.
A ták zbudował Sálomon Gázer y Betoron niżƺe ;
Przytem Báálát y Tádmor ná puƺcży w tejże źiemi /
Y wƺyſtkie miáſtá / w których miał ſkłády Sálomon / y miáſtá wozów / y miáſtá jezdnych / y wƺyſtko według żądośći Sálomonowej / cokolwiek chćiał budowáć w Jeruzálemie y ná Libánie / y we wƺyſtkiej źiemi páńſtwá ſwojego.
Wƺyſtek tákże lud / który był pozoſtał z Amorejcżyków / Hetejcżyków / Ferezejcżyków / Hewejcżyków / y Jebuzejcżyków / którzy nie byli z ſynów Izráelſkich :
To jeſt / ſyny ich / którzy byli pozoſtáli po nich w źiemi / których nie mogli ſynowie Izráelſcy wytráćić / ucżynił Sálomon hołdownikámi y niewolnikámi áż do dniá dźiśiejƺego.
Ale z ſynów Izráelſkich nie ucżynił Sálomon żádnego niewolnikiem ; jedno byli ludźmi rycerſkimi / y ſługámi jego / y kśiążęty / y hetmány jego / y przełożonymi nád wozámi jego / y nád jezdnymi jego.
Było tych przedniejƺych z przełożonych / którzy byli nád robotą Sálomonową / pięć ſet y pięćdźieśiąt / co byli nád ludźmi wykonywájącymi robotę.
Lecż córká Fáráonowá przeprowádźiłá śię z miáſtá Dawidowego do domu ſwego / który jey zbudował Sálomon. Tedy zbudował y Mello.
Y ofiárował Sálomon trzy kroć ná káżdy rok cáłopalenia á ſpokojne ofiáry ná ołtárzu / który był zbudował Pánu ; ále kádźił ná onym ołtárzu / który był przed Pánem / gdy dokonał domu.
Okrętów też nábudował król Sálomon w Aſyjongáber / które jeſt podle Elotu / nád brzegiem morzá cżerwonego / w źiemi Edomſkiej.
Y poſłał Hirám ná tychże okrętách ſługi ſwe / żeglárze świádome morzá / z ſługámi Sálomonowymi ;
Którzy przypłynąwƺy do Ofir / wźięli z támtąd złotá cżtery ſtá y dwádźieśćiá tálentów / y przywieſli je do królá Sálomoná.
