Y ſtáło śię / gdy dokońcżył Sálomon budowániá domu Páńſkiego / y domu królewſkiego / y wƺyſtkiego / co żądáł Sálomon y chćiáł ucżynić /
Że śię Pán ukázáł Sálomonowi powtóre / jáko mu śię był ukázáł w Gábáon.
Y rzekł Pán do niego : Wyſłucháłem modlitwę twoję / y prośbę twoję / którąś śię modlił przedemną / á poświęćiłem ten dom / któryś zbudowáł / áby tám przebywáło imię moje áż ná wieki ; y będą tám ocży moje y ſerce moje po wƺyſtkie dni.
A jeźli ty będzieƺ chodził przedemną / jáko chodził Dawid / ojćiec twój / w doſkonáłośći ſercá y w proſtośći / á będzieƺ śię ſpráwowáł według wƺyſtkiego / com ći przykázáł / ſtrzegąc wyroków moich y ſądów moich :
Tedy utwierdzę ſtolicę króleſtwá twego nád Izráelem ná wieki / jákom powiedziáł Dawidowi / ojcu twemu / mówiąc : Nie będzie odjęty z národu twego mąż z ſtolicy Izráelſkiej.
Ale jeźli śię názád odwróćićie wy y ſynowie wáśi ode mnie / á nie będziećie ſtrzegli przykázáń moich / y wyroków moich / którem wám podáł / ále odƺedłƺy będziećie ſłużyli bogom cudzym / y będziećie śię im kłániáli :
Tedy wytrácę Izráelá z ziemi / którąm im dáł / á dom / którym poświęćił imieniowi memu / odrzucę od oblicżá mego / á będzie Izráel przypowieśćią y báśnią między wƺyſtkimi národámi.
A ták y ten dom / który był ſłáwny / káżdemu mimo idącemu będzie ná podziw y ná poświſtánie / y rzecże : Przecżże ták ucżynił Pán tej ziemi y temu domowi?
Tedy odpowiedzą : Przeto / iż opuśćili Páná / Bogá ſwego / który wywiódł ojce ich z ziemi Egipſkiej / á chwyćili śię bogów cudzych / y kłániáli śię im / á ſłużyli im : dlá tegoż przywiódł Pán ná nie to wƺyſtko złe.
Y ſtáło śię po wyjśćiu dwudzieſtu lát / w których zbudowáł Sálomon owe obá domy / dom Páńſki y dom królewſki.
Do cżego Hirám / król Tyrſki / nádáł był Sálomonowi drzewá cedrowego / y drzewá jodłowego / y złotá / ile jedno chćiáł : tedy też król Sálomon dáł Hirámowi dwádzieśćiá miáſt w ziemi Gálilejſkiej.
Y wyjecháł Hirám z Tyru / áby oglądáł miáſtá / które mu dáł Sálomon : ále mu śię niepodobáły.
Y rzekł : Cóż to zá miáſtá / któreś mi dáł / bráćie mój? Y názwáł je ziemią Chábul / áż do dniá tego.
Abowiem poſłáł był Hirám królowi ſto y dwádzieśćiá tálentów złotá.
A przycżyná poboru / który był rozkázáł wybieráć król Sálomon / byłá / áby zbudowáł dom Páńſki / y dom ſwój / y Mello / y mury Jeruzálemſkie / y Háſor / y Mágieddo / y Gázer.
Fáráo bowiem / król Egipſki / wyćiągnął był / y wziął Gázer / y popálił je ogniem / á Chánánejcżyká / który mieƺkáł w tem mieśćie / wymordowáł / á dáł je zá poſág córce ſwej / żonie Sálomonowej.
A ták zbudowáł Sálomon Gázer y Betoron niżƺe ;
Przytem Báálát y Tádmor ná puƺcży w tejże ziemi /
Y wƺyſtkie miáſtá / w których miáł ſkłády Sálomon / y miáſtá wozów / y miáſtá jezdnych / y wƺyſtko według żądośći Sálomonowej / cokolwiek chćiáł budowáć w Jeruzálemie y ná Libánie / y we wƺyſtkiej ziemi páńſtwá ſwojego.
Wƺyſtek tákże lud / który był pozoſtáł z Amorejcżyków / Hetejcżyków / Ferezejcżyków / Hewejcżyków / y Jebuzejcżyków / którzy nie byli z ſynów Izráelſkich :
To jeſt / ſyny ich / którzy byli pozoſtáli po nich w ziemi / których nie mogli ſynowie Izráelſcy wytráćić / ucżynił Sálomon hołdownikámi y niewolnikámi áż do dniá dziśiejƺego.
Ale z ſynów Izráelſkich nie ucżynił Sálomon żádnego niewolnikiem ; jedno byli ludźmi rycerſkimi / y ſługámi jego / y kśiążęty / y hetmány jego / y przełożonymi nád wozámi jego / y nád jezdnymi jego.
Było tych przedniejƺych z przełożonych / którzy byli nád robotą Sálomonową / pięć ſet y pięćdzieśiąt / co byli nád ludźmi wykonywájącymi robotę.
Lecż córká Fáráonowá przeprowádziłá śię z miáſtá Dawidowego do domu ſwego / który jej zbudowáł Sálomon. Tedy zbudowáł y Mello.
Y ofiárowáł Sálomon trzy kroć ná káżdy rok cáłopáleniá á ſpokojne ofiáry ná ołtárzu / który był zbudowáł Pánu ; ále kádził ná onym ołtárzu / który był przed Pánem / gdy dokonáł domu.
Okrętów też nábudowáł król Sálomon w Aſyjongáber / które jeſt podle Elotu / nád brzegiem morzá cżerwonego / w ziemi Edomſkiej.
Y poſłáł Hirám ná tychże okrętách ſługi ſwe / żeglárze świádome morzá / z ſługámi Sálomonowymi ;
Którzy przypłynąwƺy do Ofir / wzięli ſtámtąd złotá cżtery ſtá y dwádzieśćiá tálentów / y przywieźli je do królá Sálomoná.
