A gdy śię król Dawid zſtárzáł / y záƺedł w látá / chodź go odźiewáno ƺátámi / przećię śię zágrzáć nie mógł.
Y rzekli mu ſłudzy jego : Niech poƺukáją królowi / pánu náƺemu / młodej pánienki / któráby ſtáwáłá przed królem / y opátrowáłá go / á ſypiájąc ná łonie jego / żeby zágrzewáłá królá / páná náƺego.
Szukáli tedy pánienki piękney po wƺyſtkich gránicách Izráelſkich / y ználeſli Abiſág Sunámitkę / á przywiedli ją do królá.
A tá pánienká byłá bárdzo piękná / y opátrowáłá królá / y ſłużyłá mu ; ále jey król nie uznáł.
Lecż Adonijáƺ / ſyn Hággity / wynośił śię / mówiąc : Ja będę królowáł. Y náſpráwiáł ſobie wozów y jezdnych / y pięćdźieśiąt mężów / którzy biegáli przed nim.
A nie gromił go nigdy ojćiec jego / mówiąc : Przecżżeś to ucżynił? A był y ten bárdzo piękney urody / którego byłá porodźiłá Hággitá po Abſálomie.
A miał zmowę z Joábem / ſynem Sárwii / y z Abijátárem kápłánem / którzy pomágáli zá Adonijáƺem.
Ale Sádok kápłán / y Bánájáſ / ſyn Jojády / y Nátán prorok / y Semej / y Rehy y rycerſtwo Dawidowe / nie przeſtáwáli z Adonijáƺem.
Tedy nábił Adonijáƺ owiec / y wołów / y bydłá tłuſtego u kámieniá Zohelet / który był nád źródłem Rogiel / y wezwáł wƺyſtkich bráći ſwych / ſynów królewſkich / y wƺyſtkich mężów z Judá / ſług królewſkich.
Ale Nátáná proroká / y Bánájáſá / y innego rycerſtwá / áni Sálomoná / brátá ſwego / nie wezwáł.
Tedy rzekł Nátán do Betſáby / mátki Sálomonowej / mówiąc : A nie ſłyƺáłáś / iż króluje Adonijáƺ / ſyn Hággity / á Dawid / pán náƺ / nie wie o tem?
Przetoż teraz pójdź proƺę / dám ći rádę / á záchowáƺ zdrowie twoje y zdrowie ſyná twego Sálomoná.
Idź / á wnijdź do królá Dawidá / y mów do niego : Izáliś ty królu / pánie mój / nieprzyśiągł ſłużebnicy twojey / mówiąc : Sálomon / ſyn twój / będźie królowáł po mnie / á on będźie śiedźiáł ná ſtolicy mojey? Przecżże tedy króluje Adonijáƺ?
A gdy ty jeƺcże tám będźieƺ mówiłá z królem / ja przyjdę zá tobą / y dopełnię ſłów twoich.
A ták weƺłá Betſábá do królá ná pokój ; á król śię już był bárdzo zſtárzáł / á Abiſág Sunámitká poſługowáłá królowi.
Tedy náchyliwƺy śię Betſábá / pokłoniłá śię królowi / którey rzekł król : Cżego chceƺ?
A oná mu odpowiedźiáłá : Pánie mój / tyś przyśiągł przez Páná / Bogá twego / ſłużebnicy twojey / że Sálomon / ſyn twój / będźie królowáł po mnie / á on będźie śiedźiáł ná ſtolicy mojey.
A oto / już Adonijáƺ króluje / á ty teraz / królu / pánie mój / o tem nie wieƺ.
Abowiem nábił wołów y bydłá tłuſtego / y owiec bárdzo wiele / y wezwáł wƺyſtkich ſynów królewſkich / y Abijátárá kápłáná / y Joábá / hetmáná wojſká ; ále Sálomoná / ſługi twego / nie wezwáł.
Lecż ty królu / pánie mój / wieƺ / iż śię ocży wƺyſtkiego Izráelá oglądáją ná ćię / ábyś im oznájmił / kto będźie śiedźiáł ná ſtolicy królá / páná mego / po tobie.
Inácżej ſtánie śię / gdy záśnie król / pán mój / z ojcy ſwymi / że będźiemy ja y Sálomon / ſyn mój / jáko grzeƺnicy.
A oto / gdy oná jeƺcże mówiłá z królem / przyƺedł Nátán prorok.
Y opowiedźiáno to królowi / mówiąc : Oto Nátán / prorok ; który wƺedłƺy do królá / pokłonił śię królowi twárzą ſwą ku źiemi.
Rzekł zátem Nátán : Królu / pánie mój / záżeś ty rzekł : Adonijáƺ będźie królowáł po mnie / á on uśiędźie ná ſtolicy mojey?
Abowiem dźiś ƺedłƺy nábił wołów y bydłá tłuſtego / y owiec bárdzo wiele / y wezwáł wƺyſtkich ſynów królewſkich / y hetmánów wojſk / y Abijátárá kápłáná / á oto oni jedzą z nim y piją / y mówią : Niech żyje król Adonijáƺ!
Ale mnie / ſługi twego / y Sádoká kápłáná / y Bánájáſá / ſyná Jojádowego / y Sálomoná / ſługi twego / nie wezwáł.
Izáli od królá / páná mego / ſtáłá śię tá rzecż? á nie oznájmiłeś ſłudze twemu / kto má śiedźieć ná ſtolicy królá / páná mego / po nim?
Y odpowiedźiáł król Dawid / mówiąc : Záwołájćie do mnie Betſáby ; która przyƺedłƺy przed oblicżność królewſką / ſtánęłá przed królem.
Tedy przyśiągł król / mówiąc : Jáko żywy Pán / który wybáwił duƺę moję z káżdego ućiſku :
Iż jákom ći przyśiągł przez Páná / Bogá Izráelſkiego / mówiąc : Że Sálomon / ſyn twój / królowáć będźie po mnie / á on uśiędźie ná ſtolicy mojey miáſto mnie / ták dźiś ucżynię.
Y náchyliwƺy śię Betſábá twárzą ku źiemi / ukłoniłá śię królowi / y rzekłá : Niech żyje Dawid król / pán mój / ná wieki!
Zátem rzekł król Dawid : Záwołájćie do mnie Sádoká kápłáná / y Nátáná proroká / y Bánájáſá / ſyná Jojádowego. Y weƺli do królá.
Y rzekł im król : Weźmijćie z ſobą ſługi páná wáƺego / á wſádźćie Sálomoná / ſyná mego / ná mulicę moję / y prowádźćie go do Gihonu.
A tám go pomáże Sádok kápłán / y Nátán prorok zá królá nád Izráelem ; potem zátrąbićie w trąbę / á rzecżećie : Niech żyje król Sálomon!
Potem pójdźiećie zá nim ; á on przyƺedłƺy / śiądźie ná ſtolicy mojey / y będźie królowáł miáſto mnie ; bom mu ja rozkázáł / áby był wodzem nád Izráelem y nád Judą.
Y odpowiedźiáł Bánájáſ / ſyn Jojádá / królowi / mówiąc : Amen. Niech to ſtwierdźi Pán / Bóg królá / páná mego.
Jáko był Pán z królem / pánem moim / ták niech będźie z Sálomonem / á niecháj wywyżƺy ſtolicę jego nád ſtolicę Dawidá królá / páná mego.
A ták ƺedł Sádok kápłán / y Nátán prorok / y Bánájáſ / ſyn Jojádá / przytem Cheretcżycy / y Feletcżycy / y wſádźili Sálomoná ná mulicę królá Dawidá / á prowádźili go do Gihonu.
Tedy wźiął Sádok kápłán róg olejku z námiotu / y pomázáł Sálomoná. Potem trąbili w trąbę / y zákrzyknął wƺyſtek lud : Niech żyje król Sálomon!
Y ƺedł wƺyſtek lud zá nim. Tenże lud gráł ná piƺcżáłkách / weſeląc śię weſelem wielkiem / ták iż drżáłá źiemiá od głoſu ich.
Co gdy uſłyƺáł Adonijáƺ / y wƺyſcy wezwáni / którzy byli z nim / ( á już śię też byłá dokońcżyłá ucżtá / )ſłyƺąc też y Joáb głoſ trąby / rzekł : Cóż to zá krzyk miáſtá hucżącego?
A gdy on tego domáwiáł / oto Jonátán / ſyn Abijátárá kápłáná / przyƺedł. Któremu rzekł Adonijáƺ : Wnijdź ; boś ty mąż ſtátecżny / á powieƺ nám co dobrego.
Tedy odpowiedźiáł Jonátán / y rzekł Adonijáƺowi : Dawid król / pán náƺ / poſtánowił zápewne Sálomoná królem.
Abowiem poſłáł z nim król Sádoká kápłáná / y Nátáná proroká / y Bánájáſá / ſyná Jojádowego / do tego Cheretcżyki y Feletcżyki / którzy go wſádźili ná mulicę królewſką ;
Y pomázáli go Sádok kápłán / y Nátán prorok zá królá w Gihonie / y ƺli ſtámtąd weſeląc śię / ták / że zádrżáło miáſto ; tenći jeſt krzyk / któryśćie ſłyƺeli ;
Y już uśiádł Sálomon ná ſtolicy króleſtwá.
Nádto y ſłudzy królewſcy przyƺli / áby błogoſłáwili Dawidowi królowi / pánu náƺemu / mówiąc : Niech ſłáwniejƺe ucżyni Bóg imię Sálomonowe nád imię twoje / á niech wywyżƺy ſtolicę jego nád ſtolicę twoję. Y pokłonił śię król ná łożu ſwem.
Przytem ták rzekł król : Błogoſłáwiony Pán / Bóg Izráelſki / który dáł dźiś śiedzącego ná ſtolicy mojey / ná co pátrzą ocży moje.
Zlękli śię tedy / y wſtáli wƺyſcy wezwáni / którzy byli z Adonijáƺem / y poƺli káżdy w drogę ſwą.
Adonijáƺ tákże / bojąc śię Sálomoná / wſtáł / y poƺedł / á uchwyćił śię rogów ołtárzá.
Y oznájmiono Sálomonowi / mówiąc : Oto Adonijáƺ boi śię królá Sálomoná / á oto uchwyćił śię rogów ołtárzá / mówiąc : Niech mi dźiś przyśięże król Sálomon / że nie zábije ſługi ſwego miecżem.
Tedy rzekł Sálomon : Jeſli będźie mężem ſtátecżnym / nie ſpádnie y włoſ z niego ná źiemię ; ále jeſli śię w nim znájdźie co złego / pewnie umrze.
A ták poſłáł król Sálomon / y przywiedźiono go od ołtárzá ; á gdy przyƺedł / ukłonił śię królowi Sálomonowi / y rzekł mu Sálomon : Idź do domu twego.
