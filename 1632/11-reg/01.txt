A gdy śię król Dawid zſtárzáł / y záƺedł w látá / chodź go odziewáno ƺátámi / przećię śię zágrzáć nie mógł.
Y rzekli mu ſłudzy jego : Niech poƺukáją królowi / pánu náƺemu / młodej pánienki / któráby ſtáwáłá przed królem / y opátrowáłá go / á ſypiájąc ná łonie jego / żeby zágrzewáłá królá / páná náƺego.
Szukáli tedy pánienki pięknej po wƺyſtkich gránicách Izráelſkich / y ználeźli Abiſág Sunámitkę / á przywiedli ją do królá.
A tá pánienká byłá bárdzo piękná / y opátrowáłá królá / y ſłużyłá mu ; ále jej król nie uznáł.
Lecż Adonijáƺ / ſyn Hággity / wynośił śię / mówiąc : Já będę królowáł. Y náſpráwiáł ſobie wozów y jezdnych / y pięćdzieśiąt mężów / którzy biegáli przed nim.
A nie gromił go nigdy ojćiec jego / mówiąc : Przecżżeś to ucżynił? A był y ten bárdzo pięknej urody / którego byłá porodziłá Hággitá po Abſálomie.
A miáł zmowę z Joábem / ſynem Sárwii / y z Abijátárem kápłánem / którzy pomágáli zá Adonijáƺem.
Ale Sádok kápłán / y Bánájáſ / ſyn Jojády / y Nátán prorok / y Semej / y Rehy y rycerſtwo Dawidowe / nie przeſtáwáli z Adonijáƺem.
Tedy nábił Adonijáƺ owiec / y wołów / y bydłá tłuſtego u kámieniá Zohelet / który był nád źródłem Rogiel / y wezwáł wƺyſtkich bráći ſwych / ſynów królewſkich / y wƺyſtkich mężów z Judá / ſług królewſkich.
Ale Nátáná proroká / y Bánájáſá / y innego rycerſtwá / áni Sálomoná / brátá ſwego / nie wezwáł.
Tedy rzekł Nátán do Betſáby / mátki Sálomonowej / mówiąc : A nie ſłyƺáłáś / iż króluje Adonijáƺ / ſyn Hággity / á Dawid / pán náƺ / nie wie o tem?
Przetoż teráz pójdź proƺę / dám ći rádę / á záchowáƺ zdrowie twoje y zdrowie ſyná twego Sálomoná.
Idź / á wnijdź do królá Dawidá / y mów do niego : Izáliś ty królu / pánie mój / nieprzyśiągł ſłużebnicy twojej / mówiąc : Sálomon / ſyn twój / będzie królowáł po mnie / á on będzie śiedziáł ná ſtolicy mojej? Przecżże tedy króluje Adonijáƺ?
A gdy ty jeƺcże tám będzieƺ mówiłá z królem / já przyjdę zá tobą / y dopełnię ſłów twoich.
A ták weƺłá Betſábá do królá ná pokój ; á król śię już był bárdzo zſtárzáł / á Abiſág Sunámitká poſługowáłá królowi.
Tedy náchyliwƺy śię Betſábá / pokłoniłá śię królowi / której rzekł król : Cżego chceƺ?
A oná mu odpowiedziáłá : Pánie mój / tyś przyśiągł przez Páná / Bogá twego / ſłużebnicy twojej / że Sálomon / ſyn twój / będzie królowáł po mnie / á on będzie śiedziáł ná ſtolicy mojej.
A oto / już Adonijáƺ króluje / á ty teráz / królu / pánie mój / o tem nie wieƺ.
Abowiem nábił wołów y bydłá tłuſtego / y owiec bárdzo wiele / y wezwáł wƺyſtkich ſynów królewſkich / y Abijátárá kápłáná / y Joábá / hetmáná wojſká ; ále Sálomoná / ſługi twego / nie wezwáł.
Lecż ty królu / pánie mój / wieƺ / iż śię ocży wƺyſtkiego Izráelá oglądáją ná ćię / ábyś im oznájmił / kto będzie śiedziáł ná ſtolicy królá / páná mego / po tobie.
Inácżej ſtánie śię / gdy záśnie król / pán mój / z ojcy ſwymi / że będziemy já y Sálomon / ſyn mój / jáko grzeƺnicy.
A oto / gdy oná jeƺcże mówiłá z królem / przyƺedł Nátán prorok.
Y opowiedziáno to królowi / mówiąc : Oto Nátán / prorok ; który wƺedłƺy do królá / pokłonił śię królowi twárzą ſwą ku ziemi.
Rzekł zátem Nátán : Królu / pánie mój / záżeś ty rzekł : Adonijáƺ będzie królowáł po mnie / á on uśiędzie ná ſtolicy mojej?
Abowiem dziś ƺedłƺy nábił wołów y bydłá tłuſtego / y owiec bárdzo wiele / y wezwáł wƺyſtkich ſynów królewſkich / y hetmánów wojſk / y Abijátárá kápłáná / á oto oni jedzą z nim y piją / y mówią : Niech żyje król Adonijáƺ!
Ale mnie / ſługi twego / y Sádoká kápłáná / y Bánájáſá / ſyná Jojádowego / y Sálomoná / ſługi twego / nie wezwáł.
Izáli od królá / páná mego / ſtáłá śię tá rzecż? á nie oznájmiłeś ſłudze twemu / kto má śiedzieć ná ſtolicy królá / páná mego / po nim?
Y odpowiedziáł król Dawid / mówiąc : Záwołájćie do mnie Betſáby ; którá przyƺedłƺy przed oblicżność królewſką / ſtánęłá przed królem.
Tedy przyśiągł król / mówiąc : Jáko żywy Pán / który wybáwił duƺę moję z káżdego ućiſku :
Iż jákom ći przyśiągł przez Páná / Bogá Izráelſkiego / mówiąc : Że Sálomon / ſyn twój / królowáć będzie po mnie / á on uśiędzie ná ſtolicy mojej miáſto mnie / ták dziś ucżynię.
Y náchyliwƺy śię Betſábá twárzą ku ziemi / ukłoniłá śię królowi / y rzekłá : Niech żyje Dawid król / pán mój / ná wieki!
Zátem rzekł król Dawid : Záwołájćie do mnie Sádoká kápłáná / y Nátáná proroká / y Bánájáſá / ſyná Jojádowego. Y weƺli do królá.
Y rzekł im król : Weźmijćie z ſobą ſługi páná wáƺego / á wſádźćie Sálomoná / ſyná mego / ná mulicę moję / y prowádźćie go do Gihonu.
A tám go pomáże Sádok kápłán / y Nátán prorok zá królá nád Izráelem ; potem zátrąbićie w trąbę / á rzecżećie : Niech żyje król Sálomon!
Potem pójdziećie zá nim ; á on przyƺedłƺy / śiądzie ná ſtolicy mojej / y będzie królowáł miáſto mnie ; bom mu já rozkázáł / áby był wodzem nád Izráelem y nád Judą.
Y odpowiedziáł Bánájáſ / ſyn Jojádá / królowi / mówiąc : Amen. Niech to ſtwierdzi Pán / Bóg królá / páná mego.
Jáko był Pán z królem / pánem moim / ták niech będzie z Sálomonem / á niecháj wywyżƺy ſtolicę jego nád ſtolicę Dawidá królá / páná mego.
A ták ƺedł Sádok kápłán / y Nátán prorok / y Bánájáſ / ſyn Jojádá / przytem Cheretcżycy / y Feletcżycy / y wſádzili Sálomoná ná mulicę królá Dawidá / á prowádzili go do Gihonu.
Tedy wziął Sádok kápłán róg olejku z námiotu / y pomázáł Sálomoná. Potem trąbili w trąbę / y zákrzyknął wƺyſtek lud : Niech żyje król Sálomon!
Y ƺedł wƺyſtek lud zá nim. Tenże lud gráł ná piƺcżáłkách / weſeląc śię weſelem wielkiem / ták iż drżáłá ziemiá od głoſu ich.
Co gdy uſłyƺáł Adonijáƺ / y wƺyſcy wezwáni / którzy byli z nim / ( á już śię też byłá dokońcżyłá ucżtá / )ſłyƺąc też y Joáb głoſ trąby / rzekł : Cóż to zá krzyk miáſtá hucżącego?
A gdy on tego domáwiáł / oto Jonátán / ſyn Abijátárá kápłáná / przyƺedł. Któremu rzekł Adonijáƺ : Wnijdź ; boś ty mąż ſtátecżny / á powieƺ nám co dobrego.
Tedy odpowiedziáł Jonátán / y rzekł Adonijáƺowi : Dawid król / pán náƺ / poſtánowił zápewne Sálomoná królem.
Abowiem poſłáł z nim król Sádoká kápłáná / y Nátáná proroká / y Bánájáſá / ſyná Jojádowego / do tego Cheretcżyki y Feletcżyki / którzy go wſádzili ná mulicę królewſką ;
Y pomázáli go Sádok kápłán / y Nátán prorok zá królá w Gihonie / y ƺli ſtámtąd weſeląc śię / ták / że zádrżáło miáſto ; tenći jeſt krzyk / któryśćie ſłyƺeli ;
Y już uśiádł Sálomon ná ſtolicy króleſtwá.
Nádto y ſłudzy królewſcy przyƺli / áby błogoſłáwili Dawidowi królowi / pánu náƺemu / mówiąc : Niech ſłáwniejƺe ucżyni Bóg imię Sálomonowe nád imię twoje / á niech wywyżƺy ſtolicę jego nád ſtolicę twoję. Y pokłonił śię król ná łożu ſwem.
Przytem ták rzekł król : Błogoſłáwiony Pán / Bóg Izráelſki / który dáł dziś śiedzącego ná ſtolicy mojej / ná co pátrzą ocży moje.
Zlękli śię tedy / y wſtáli wƺyſcy wezwáni / którzy byli z Adonijáƺem / y poƺli káżdy w drogę ſwą.
Adonijáƺ tákże / bojąc śię Sálomoná / wſtáł / y poƺedł / á uchwyćił śię rogów ołtárzá.
Y oznájmiono Sálomonowi / mówiąc : Oto Adonijáƺ boi śię królá Sálomoná / á oto uchwyćił śię rogów ołtárzá / mówiąc : Niech mi dziś przyśięże król Sálomon / że nie zábije ſługi ſwego miecżem.
Tedy rzekł Sálomon : Jeźli będzie mężem ſtátecżnym / nie ſpádnie y włoſ z niego ná ziemię ; ále jeźli śię w nim znájdzie co złego / pewnie umrze.
A ták poſłáł król Sálomon / y przywiedziono go od ołtárzá ; á gdy przyƺedł / ukłonił śię królowi Sálomonowi / y rzekł mu Sálomon : Idź do domu twego.
