Y ſtáło śię ſłowo Páńſkie do Jehu / ſyná Hánánijego / przećiw Bááźie / mówiące :
Dla tego żem ćię wydźwignął z prochu / á poſtánowiłem ćię wodzem nád ludem moim Izráelſkim / á tyś chodźił drogámi Jeroboámowemi / y przywiodłeś do grzechu lud mój Izráelſki / pobudzájąc mię ku gniewu grzechámi ich :
Otoż ja wygłádzę potomki Báázy / y potomki domu jego / á ucżynię dom twój / jáko dom Jeroboámá / ſyná Nábátowego.
Tego / który z rodu Báázy umrze w mieśćie / zjedzą pſy / á tego / który umrze ná polu / zjedzą ptáki powietrzne.
A inne ſpráwy Báázy / y co cżynił / y moc jego / ázaż to nie jeſt nápiſano w kronikách królów Izráelſkich?
A gdy záſnął Báázá z ojcy ſwymi / pochowány jeſt w Terśie / y królował Elá / ſyn jego / miáſto niego.
A ták przez proroká Jehu / ſyná Hánánijego / ſtáło śię ſłowo Páńſkie przećiw Bááźie y przećiw domowi jego / y przećiw wƺyſtkiemu złemu / które cżynił przed oblicżem Páńſkim / wzruƺájąc go ku gniewu ſpráwámi rąk ſwojich / że ma być podobnym domowi Jeroboámowemu / y dla tego / że go zábił.
Roku dwudźieſtego y ƺóſtego Azy / królá Judzkiego / królował Elá / ſyn Báázy / nád Izráelem w Terśie dwá látá.
Y ſprzyśiągł śię przećiw niemu ſługá jego Zymry / hetmán nád połową wozów / gdy Elá w Terśie pijąc pijány był w domu Arſy / który był ſpráwcą domu królewſkiego w Terśie.
Wtem przypádł Zymry y ránił go / y zábił go roku dwudźieſtego y śiódmego Azy / królá Judzkiego / á królował miáſto niego.
A gdy już królował y śiedźiał ná ſtolicy jego / wymordował wƺyſtek dom Báázy / y powinne jego / y przyjáćioły jego ; nie zoſtáwił z niego y ƺcżeniątká.
A ták wygłádźił Zymry wƺyſtek dom Báázy według ſłowá Páńſkiego / które powiedźiał o Bááźie przez proroká Jehu.
Dla wƺyſtkich grzechów Báázy / y grzechów Eli / ſyná jego / którzy grzeƺyli / y którzy przywiedli do grzechu Izráelá / wzruƺájąc ku gniewu Páná / Bogá Izráelſkiego / próżnośćiámi ſwemi.
Ale inne ſpráwy Eli / y wƺyſtko co cżynił / izali nie nápiſáne w kronikách o królách Izráelſkich?
Roku dwudźieſtego y śiódmego Azy / królá Judzkiego / królował Zymry śiedm dni w Terśie / gdy lud obozem leżał u Giebbeton / które jeſt Filiſtyńſkie.
A gdy uſłyƺał lud leżący w oboźie tákową rzecż / iż Zymry ſprzyśiągƺy śię zábił królá : tedy wƺyſtek Izráel poſtánowili królem Amrego / który był hetmánem ná wojſkiem Izráelſkiem onegoż dniá w oboźie.
Przetoż odćiągnął Amry y wƺyſtek Izráel z nim od Giebbeton / á oblegli Terſę.
A gdy obácżył Zymry / iż wźięto miáſto / wƺedł ná páłác domu królewſkiego / y ſpálił śię ogniem z domem królewſkim / y umárł.
A to śię ſtáło dla grzechów jego / których śię dopuśćił / cżyniąc złe przed oblicżem Páńſkim / á chodząc drogą Jeroboámá / y w grzechách jego / których śię dopuƺcżał / do grzechu przywodząc Izráelá.
A inne ſpráwy Zymry y ſprzyśiężenie jego / które ucżynił / ázaż nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Tedy śię rozerwał lud Izráelſki ná dwie cżęśći ; połowá ludu ƺłá zá Tebni / ſynem Ginetowym / áby go ucżynili królem / á połowá ƺłá zá Amrym.
Ale przemógł lud / który przeſtáwał z Amrym / on lud / który zoſtáwał przy Tebni / ſynem Ginetowym ; y umárł Tebni / á królował Amry.
Roku trzydźieſtego y pierwƺego Azy / królá Judzkiego / królował Amry nád Izráelem dwánáśćie lat ; w Terśie królował ƺeść lat.
Y kupił górę Sámáryi od Semerá zá dwá tálenty ſrebrá / y pobudował ná oney górze / á názwał imię miáſtá / które zbudował imieniem Semerá / páná góry oney / Sámáryi.
Ale cżynił Amry złe przed ocżymá Páńſkimi / á dopuƺcżał śię rzecży gorƺych / niżeli wƺyſcy / którzy przed nim byli.
Abowiem chodźił wƺyſtkiemi drogámi Jeroboámá / ſyná Nábátowego / y w grzechu jego / którym przywiódł w grzechy Izráelá / wzruƺájąc ku gniewu Páná Bogá Izráelſkiego / próżnośćiámi ſwemi.
A inne ſpráwy Amrego / y wƺyſtko / co cżynił / y moc jego / którą pokázywał / ázaż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Amry z ojcy ſwymi / á pochowány jeſt w Sámáryi ; y królował Acháb / ſyn jego / miáſto niego.
Acháb tedy / ſyn Amrego / królowáć pocżął nád Izráelem roku trzydźieſtego y óſmego Azy / królá Judzkiego / á królował Acháb / ſyn Amrego / nád Izráelem w Sámáryi dwádźieśćiá y dwá lat.
Y cżynił Acháb / ſyn Amrego / złe przed ocżymá Páńſkimi nád wƺyſtkie / którzy byli przed nim.
Y ſtáło śię / nie májąc ná tym doſyć / iż chodźił w grzechách Jeroboámá / ſyná Nábátowego / że ſobie wźiął zá żonę Jezábelę / córkę Etbáálá / królá Sydońſkiego / á ƺedƺy ſłużył Báálowi / y kłániał mu śię.
Y wyſtáwił ołtárz Báálowi w domu Báálowym / który był zbudował w Sámáryi.
Do tego náſádźił Acháb gáj / á tym więcey wzruƺył ku gniewu Páná / Bogá Izráelſkiego / nád wƺyſtkie króle Izráelſkie / którzy byli przed nim.
Zá dni jego zbudował Hijel / Betelcżyk / miáſto Jerycho. Ná Abirámie / pierworodnym ſwoim / záłożył je / á ná Segubie / nájmłodƺym ſynu ſwym / wyſtáwił brámy jego / według ſłowá Páńſkiego / które powiedźiał przez Jozuego / ſyná Nunowego.
