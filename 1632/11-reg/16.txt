Y ſtáło śię ſłowo Páńſkie do Jehu / ſyná Hánánijego / przećiw Báázie / mówiące :
Dlá tego żem ćię wydźwignął z prochu / á poſtánowiłem ćię wodzem nád ludem moim Izráelſkim / á tyś chodził drogámi Jeroboámowemi / y przywiodłeś do grzechu lud mój Izráelſki / pobudzájąc mię ku gniewu grzechámi ich :
Otoż já wygłádzę potomki Báázy / y potomki domu jego / á ucżynię dom twój / jáko dom Jeroboámá / ſyná Nábátowego.
Tego / który z rodu Báázy umrze w mieśćie / zjedzą pſy / á tego / który umrze ná polu / zjedzą ptáki powietrzne.
A inne ſpráwy Báázy / y co cżynił / y moc jego / ázáż to nie jeſt nápiſáno w kronikách królów Izráelſkich?
A gdy záſnął Báázá z ojcy ſwymi / pochowány jeſt w Terśie / y królowáł Elá / ſyn jego / miáſto niego.
A ták przez proroká Jehu / ſyná Hánánijego / ſtáło śię ſłowo Páńſkie przećiw Báázie y przećiw domowi jego / y przećiw wƺyſtkiemu złemu / które cżynił przed oblicżem Páńſkim / wzruƺájąc go ku gniewu ſpráwámi rąk ſwoich / że má być podobnym domowi Jeroboámowemu / y dlá tego / że go zábił.
Roku dwudzieſtego y ƺóſtego Azy / królá Judzkiego / królowáł Elá / ſyn Báázy / nád Izráelem w Terśie dwá látá.
Y ſprzyśiągł śię przećiw niemu ſługá jego Zymry / hetmán nád połową wozów / gdy Elá w Terśie pijąc pijány był w domu Arſy / który był ſpráwcą domu królewſkiego w Terśie.
Wtem przypádł Zymry y ránił go / y zábił go roku dwudzieſtego y śiódmego Azy / królá Judzkiego / á królowáł miáſto niego.
A gdy już królowáł y śiedziáł ná ſtolicy jego / wymordowáł wƺyſtek dom Báázy / y powinne jego / y przyjáćioły jego ; nie zoſtáwił z niego y ƺcżeniątká.
A ták wygłádził Zymry wƺyſtek dom Báázy według ſłowá Páńſkiego / które powiedziáł o Báázie przez proroká Jehu.
Dlá wƺyſtkich grzechów Báázy / y grzechów Eli / ſyná jego / którzy grzeƺyli / y którzy przywiedli do grzechu Izráelá / wzruƺájąc ku gniewu Páná / Bogá Izráelſkiego / próżnośćiámi ſwemi.
Ale inne ſpráwy Eli / y wƺyſtko co cżynił / izáli nie nápiſáne w kronikách o królách Izráelſkich?
Roku dwudzieſtego y śiódmego Azy / królá Judzkiego / królowáł Zymry śiedm dni w Terśie / gdy lud obozem leżáł u Giebbeton / które jeſt Filiſtyńſkie.
A gdy uſłyƺáł lud leżący w obozie tákową rzecż / iż Zymry ſprzyśiągłƺy śię zábił królá : tedy wƺyſtek Izráel poſtánowili królem Amrego / który był hetmánem ná wojſkiem Izráelſkiem onegoż dniá w obozie.
Przetoż odćiągnął Amry y wƺyſtek Izráel z nim od Giebbeton / á oblegli Terſę.
A gdy obácżył Zymry / iż wzięto miáſto / wƺedł ná páłác domu królewſkiego / y ſpálił śię ogniem z domem królewſkim / y umárł.
A to śię ſtáło dlá grzechów jego / których śię dopuśćił / cżyniąc złe przed oblicżem Páńſkim / á chodząc drogą Jeroboámá / y w grzechách jego / których śię dopuƺcżáł / do grzechu przywodząc Izráelá.
A inne ſpráwy Zymry y ſprzyśiężenie jego / które ucżynił / ázáż nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Tedy śię rozerwáł lud Izráelſki ná dwie cżęśći ; połowá ludu ƺłá zá Tebni / ſynem Ginetowym / áby go ucżynili królem / á połowá ƺłá zá Amrym.
Ale przemógł lud / który przeſtáwáł z Amrym / on lud / który zoſtáwáł przy Tebni / ſynem Ginetowym ; y umárł Tebni / á królowáł Amry.
Roku trzydzieſtego y pierwƺego Azy / królá Judzkiego / królowáł Amry nád Izráelem dwánáśćie lát ; w Terśie królowáł ƺeść lát.
Y kupił górę Sámáryi od Semerá zá dwá tálenty ſrebrá / y pobudowáł ná onej górze / á názwáł imię miáſtá / które zbudowáł imieniem Semerá / páná góry onej / Sámáryi.
Ale cżynił Amry złe przed ocżymá Páńſkimi / á dopuƺcżáł śię rzecży gorƺych / niżeli wƺyſcy / którzy przed nim byli.
Abowiem chodził wƺyſtkiemi drogámi Jeroboámá / ſyná Nábátowego / y w grzechu jego / którym przywiódł w grzechy Izráelá / wzruƺájąc ku gniewu Páná Bogá Izráelſkiego / próżnośćiámi ſwemi.
A inne ſpráwy Amrego / y wƺyſtko / co cżynił / y moc jego / którą pokázywáł / ázáż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Amry z ojcy ſwymi / á pochowány jeſt w Sámáryi ; y królowáł Acháb / ſyn jego / miáſto niego.
Acháb tedy / ſyn Amrego / królowáć pocżął nád Izráelem roku trzydzieſtego y óſmego Azy / królá Judzkiego / á królowáł Acháb / ſyn Amrego / nád Izráelem w Sámáryi dwádzieśćiá y dwá lát.
Y cżynił Acháb / ſyn Amrego / złe przed ocżymá Páńſkimi nád wƺyſtkie / którzy byli przed nim.
Y ſtáło śię / nie májąc ná tem doſyć / iż chodził w grzechách Jeroboámá / ſyná Nábátowego / że ſobie wziął zá żonę Jezábelę / córkę Etbáálá / królá Sydońſkiego / á ƺedłƺy ſłużył Báálowi / y kłániáł mu śię.
Y wyſtáwił ołtárz Báálowi w domu Báálowym / który był zbudowáł w Sámáryi.
Do tego náſádził Acháb gáj / á tem więcej wzruƺył ku gniewu Páná / Bogá Izráelſkiego / nád wƺyſtkie króle Izráelſkie / którzy byli przed nim.
Zá dni jego zbudowáł Hijel / Betelcżyk / miáſto Jerycho. Ná Abirámie / pierworodnym ſwoim / záłożył je / á ná Segubie / nájmłodƺym ſynu ſwym / wyſtáwił brámy jego / według ſłowá Páńſkiego / które powiedziáł przez Jozuego / ſyná Nunowego.
