Tedy zebráł Sálomon ſtárƺe Izráelſkie / y wƺyſtkie celniejƺe z káżdego pokoleniá / y przedniejƺe z ojców ſynów Izráelſkich / do śiebie do Jeruzálemu / áby przenieśioná byłá ſkrzyniá przymierzá Páńſkiego z miáſtá Dawidowego / które jeſt Syon.
Y zeƺli śię do królá Sálomoná wƺyſcy mężowie Izráelſcy mięśiącá Wrześniá w urocżyſte święto ; á ten mieśiąc jeſt śiódmy.
A gdy śię zeƺli wƺyſcy ſtárśi Izráelſcy / wzięli kápłáni ſkrzynię.
Y przenieśli ſkrzynię Páńſką / y námiot zgromádzeniá / y wƺyſtkie nácżyniá święte / które były w námioćie / á przenieśli je kápłáni y Lewitowie.
Lecż król Sálomon / y wƺyſtko mnóſtwo Izráelſkie / które śię zeƺło do niego / z nim przed ſkrzynią ofiárowáli owce y woły / których nie licżono áni ráchowáno dlá mnóſtwá.
Wnieśli tedy kápłáni ſkrzynię przymierzá Páńſkiego ná miejſce jej do wnętrznego domu / do świątnicy świętych / pod ſkrzydłá Cherubinów.
Abowiem Cherubinowie mieli rozćiągnione ſkrzydłá nád miejſcem ſkrzyni / á okrywáli Cherubinowie ſkrzynię y drążki jej z wierzchu.
Y powyćiągáli one drążki / ták / że widáć było końce ich w świątnicy ná przodku świątnicy świętych ; ále nie widáć ich było zewnątrz ; y támże były áż do dniá tego.
Nic nie było w ſkrzyni tylko dwie táblice kámienne / które tám był ſchowáł Mojżeƺ ná Horebie / gdy ſtánowił przymierze Pán z ſynámi Izráelſkimi / gdy ƺli z ziemi Egipſkiej.
Y ſtáło śię / gdy wychodzili kápłáni z świątnicy / że obłok nápełnił dom Páńſki.
Ták iż śię nie mogli kápłáni oſtáć y ſłużyć dlá onego obłoku ; ábowiem nápełniłá byłá chwáłá Páńſká dom Páńſki.
Tedy rzekł Sálomon : Pán powiedziáł / iż miáł mieƺkáć we mgle.
Jużem zbudowáł dom ná mieƺkánie tobie / miejſce / ábyś tám przebywáł ná wieki.
Y obróćił król oblicże ſwoje / y błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu ; á wƺyſtko zgromádzenie Izráelſkie ſtáło.
Y rzekł : Błogoſłáwiony Pán / Bóg Izráelſki / który mówił uſty ſwemi do Dawidá / ojcá mego / y ſkutecżnie to wypełnił / mówiąc :
Ode dniá / któregom wywiódł lud mój Izráelſki z Egiptu / nie obráłem miáſtá ze wƺyſtkich pokoleń Izráelſkich ku zbudowániu domu / gdzieby przebywáło imię moje ; álem obráł Dawidá / áby był nád ludem moim Izráelſkim.
Poſtánowiłći był wpráwdzie w ſercu ſwem Dawid / ojćiec mój / zbudowáć dom imieniowi Páná / Bogá Izráelſkiego ;
Ale rzekł Pán do Dawidá / ojcá mego : Acżkolwiekeś poſtánowił w ſercu twem / zbudowáć dom imieniowi memu / y dobrześ ucżynił / żeś to umyślił w ſercu twojim :
Wƺákże ty nie będzieƺ budowáł tego domu ; ále ſyn twój / który wynijdzie z biódr twoich / ten zbuduje dom imieniowi memu.
A ták utwierdził Pán ſłowo ſwoje / które był powiedziáł. Bom já powſtáł miáſto Dawidá / ojcá mego / y uśiádłem ná ſtolicy Izráelſkiej / jáko był powiedziáł Pán / y zbudowáłem dom imieniowi Páná / Bogá Izráelſkiego.
Y náznácżyłem tám miejſce ſkrzyni / w której jeſt przymierze Páńſkie / które ucżynił z ojcy náƺymi / gdy je wywiódł z ziemi Egipſkiej.
Tedy ſtánął Sálomon przed ołtárzem Páńſkim / przed wƺyſtkiem zgromádzeniem Izráelſkiem / y wyćiągnął ręce ſwoje ku niebu /
Y rzekł : Pánie / Boże Izráelſki / niemáƺ tobie podobnego Bogá ná niebie wzgórę / áni ná ziemi niſko ; który chowáƺ umowę y miłośierdzie ſługom twym / którzy chodzą przed tobą cáłem ſercem ſwojim ;
Któryś ſpełnił ſłudze twemu Dawidowi / ojcu memu / coś mu powiedziáł ; coś mówił uſty ſwemi / toś ſkutecżnie wypełnił / jáko śię dniá tego pokázuje.
Przetoż teráz o Pánie / Boże Izráelſki / ziść ſłudze twemu Dawidowi / ojcu memu / coś mu powiedziáł / mówiąc : Nie będzie odjęty przed twárzą moją z národu twego mąż / któryby śiedziáł ná ſtolicy Izráelſkiej / jeźli tylko będą przeſtrzegáli ſynowie twoi drogi ſwej / chodząc przede mną / jákoś ty chodził przed oblicżnośćią moją.
Przetoż teráz / o Boże Izráelſki / niech będzie utwierdzone proƺę ſłowo twoje / któreś mówił do ſługi twego Dawidá / ojcá mego.
( Acżći wpráwdzie / izáli Bóg będzie mieƺkáł ná ziemi? Oto niebioſá / y niebá niebioſ / nie mogą ćię ogárnąć ; jákoż dáleko mniej ten dom / którym zbudowáł. )
A wejrzyj ná modlitwę ſługi twego / y ná prośbę jego / o Pánie Boże mój / wyſłucháj wołánie y modlitwę / którą śię dziś ſługá twój modli przed tobą.
A niech będą otworzone ocży twoje nád tym domem w nocy y we dnie / nád tem miejſcem / o któremeś powiedziáł : Tu będzie imię moje ; ábyś wyſłuchiwáł modlitwę / którą śię będzie modlił ſługá twój ná miejſcu tem.
Wyſłuchájże prośby ſługi twego / y ludu twego Izráelſkiego / który śię modlić będzie ná tem miejſcu. Ty wyſłucháj z miejſcá mieƺkániá twego / z niebá / á wyſłucháwƺy bądź miłośćiw.
Gdyby cżłowiek zgrzeƺył przećiwko bliźniemu ſwemu / á przywiódłby go do przyśięgi / ták / żeby przyśięgáć muśiáł / á przyƺłáby tá przyśięgá przed ołtárz twój w tym domu :
Ty wyſłucháj z niebá / á rozeznáj y rozſądź ſługi twoje / potępiájąc niezbożnego / y obrácájąc ſpráwy jego ná głowę jego / á uſpráwiedliwiájąc ſpráwiedliwego / oddáwájąc mu według ſpráwiedliwośći jego.
Gdyby był poráżony lud twój Izráelſki od nieprzyjáćielá / przeto iż zgrzeƺyli przećiw tobie / á náwróćiliby śię do ćiebie / wyznáwájąc imię twoje / á modląc śię / przepráƺáliby ćię w tym domu :
Ty wyſłucháj z niebá / y odpuść grzech ludowi twemu Izráelſkiemu / á przywróć je záśię do ziemi / którąś dáł ojcom ich.
Gdyby záwárte było niebo / á nie byłoby dżdżu / przeto że zgrzeƺyli przećiwko tobie / á modliliby śię ná tem miejſcu / wyznáwájąc imię twoje / á od grzechów ſwoich odwróćiliby śię / gdybyś je utrápił :
Ty wyſłucháj z niebá / á odpuść grzech ſług twoich / y ludu twego Izráelſkiego / náucżywƺy ich drogi práwej / po której chodzić máją / á dáj deƺcż ná ziemię twoję / którąś dáł w dziedzictwo ludowi twemu.
Byłliby głód ná ziemi / byłliby mor / ſuƺá / rdzá / ƺáráńcżá / jeźliby były chrząƺcże / jeźliby go śćiſnął nieprzyjáćiel jego w ziemi mieƺkániá jego / álbo jákákolwiek plágá / álbo jákákolwiek niemoc :
Wƺelką modlitwę / y wƺelką prośbę / którąby cżynił którykolwiek cżłowiek / álbo wƺyſtek lud twój Izráelſki / ktoby jedno uznáł ránę ſercá ſwego / y wyćiągnąłby ręce ſwe w domu tym :
Ty wyſłucháj z niebá / z miejſcá mieƺkániá twego / á odpuść / y ucżyń / y oddáj káżdemu według wƺelkich dróg jego / które znáƺ w ſercu jego ; bo ty / ty ſám znáƺ ſercá wƺyſtkich ſynów ludzkich ;
Aby śię ćiebie báli po wƺyſtkie dni / które żyć będą ná ziemi / którąś dáł ojcom náƺym.
Nádto y cudzoziemiec / który nie jeſt z ludu twego Izráelſkiego / przyjdzieli z ziemi dálekiej dlá imieniá twego ;
( Bo uſłyƺą o imieniu twojim wielkiem / y o ręce twojej możnej / y o rámieniu twojim wyćiągnionem / )przyjdzieli tedy / á będzie śię modlił w tym domu :
Ty wyſłucháj z niebá / z miejſcá mieƺkániá twego / á ucżyń wƺyſtko / o co záwołá do ćiebie on cudzoziemiec / áby poználi wƺyſcy národowie ziemſcy imię twoje / y báli śię ćiebie / jáko lud twój Izráelſki / á żeby wiedzieli / że imię twoje wzywáne jeſt nád tym domem / którym zbudowáł.
Gdyby wyƺedł lud twój ná wojnę przećiwko nieprzyjáćielowi ſwemu drogą / którą je pośleƺ / á modliliby śię Pánu / obróćiwƺy śię ku miáſtu / któreś obráł / y ku domowi / którym zbudowáł imieniowi twemu :
Wyſłuchájże z niebá modlitwę ich / y prośbę ich / á wykonáj ſąd ich.
Gdyby też zgrzeƺyli przećiwko tobie / ( bo niemáƺ cżłowieká / któryby nie grzeƺył / )á rozgniewáwƺy śię ná nie / podáłbyś je pod moc nieprzyjáćielowi / któryby je pojmáwƺy / záwiódł je w niewolę / do ziemi nieprzyjáćielſkiej / dálekiej álbo bliſkiej :
A upámiętáliby śię w ſercu ſwojim w onej ziemi / do której ſą záprowádzeni w niewolę / y náwróćiliby śię / á przepráƺáliby ćię w ziemi tych / którzy je pojmáli / mówiąc : Zgrzeƺyliśmy / y źleśmy ucżynili / niepobożnieśmy śię ſpráwowáli ;
A ták náwróćiliby śię do ćiebie z cáłego ſercá ſwego / y z cáłej duƺy ſwej / w ziemi nieprzyjáćiół ſwoich / którzy je pojmáli / á modliliby śię tobie / obróćiwƺy śię ku ziemi ſwej / którąś dáł ojcom ich / ku miáſtu / któreś obráł / y ku domowi / którym zbudowáł imieniowi twemu :
Wyſłuchájże tedy z niebá / z miejſcá mieƺkániá twego / modlitwę ich y prośbę ich / á wykonáj ſąd ich /
A bądź miłośćiw ludowi twemu / który przećiw tobie zgrzeƺył / y wƺyſtkim niepráwośćiom ich / któremi wyſtąpili przećiw tobie / á nákłoń ku nim miłośierdziá tych / którzy je pojmáli / áby śię zmiłowáli nád nimi ;
Poniewáż ſą ludem twoim / y dziedzictwem twojim / któreś wywiódł z Egiptu / z pośrodku piecá żeláznego.
Niech będą ocży twoje otwárte ná prośbę ſługi twego / y ná prośbę ludu twego Izráelſkiego / ábyś je wyſłucháł we wƺyſtkiem / o co ćię wzywáć będą.
Abowiemeś je ty ſobie odłącżył zá dziedzictwo ze wƺyſtkich národów ziemi / jákoś powiedziáł przez Mojżeƺá / ſługę twego / gdyś wywiódł ojce náƺe z Egiptu / o Pánie Boże!
Y ſtáło śię / gdy Sálomon modląc śię Pánu dokonáł wƺyſtkiej onej modlitwy y prośby / że wſtáł od ołtárzá Páńſkiego / á przeſtáł klęcżeć y podnośić rąk ſwoich ku niebu ;
A ſtojąc błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu wielkim głoſem / mówiąc :
Błogoſłáwiony Pán / który dáł odpocżnienie ludowi ſwemu Izráelſkiemu / według wƺyſtkiego / co powiedziáł ; nie chybiło żádne ſłowo ze wƺyſtkich ſłów jego dobrych / które mówił przez Mojżeƺá / ſługę ſwego.
Niechże będzie Pán / Bóg náƺ / z námi / jáko był z ojcy náƺymi ; niech náſ nie opuƺcżá / áni náſ odrzucá ;
Ale niech nákłoni ſerce náƺe ku ſobie / żebyśmy chodzili po wƺyſtkich drogách jego / ſtrzegąc rozkázániá jego / y wyroków jego / y ſądów jego / które przykázáł ojcom náƺym.
A niech będą te ſłowá moje / któremim śię modlił przed Pánem / bliſkie Páná Bogá náƺego we dnie y w nocy / áby wykonywáł ſąd ſługi ſwego / y ſąd ludu ſwego Izráelſkiego / ſądząc káżdą ſpráwę dniá ſwojego ;
Żeby poználi wƺyſcy národowie ziemſcy / iż Pán ſám jeſt Bogiem / á nikt inƺy.
Niechże tedy będzie ſerce wáƺe doſkonáłe ku Pánu / Bogu náƺemu / ábyśćie chodzili w wyrokách jego / á ſtrzegli przykázáń jego / jáko y dniá dziśiejƺego.
Tedy król / y wƺyſtek Izráel z nim / ſpráwowáli ofiáry przed Pánem.
Y ofiárowáł Sálomon ofiárę ſpokojną / którą ſpráwowáł Pánu / wołów dwádzieśćiá y dwá tyśiące / y owiec ſto y dwádzieśćiá tyśięcy. A ták poświęcáli dom Páńſki król y wƺyſcy ſynowie Izráelſcy.
Onegoż dniá poświęćił król pośrodek śieni / którá byłá przed domem Páńſkim ; bo tám ofiárowáł cáłopálenie / y ofiárę śniedną / y tłuſtośći ofiár ſpokojnych / przeto że ołtárz miedziány / który był przed Pánem / był máły / y nie mogły śię ná nim zmieśćić cáłopáleniá / y ofiáry śniedne / y tłuſtośći ofiár ſpokojnych.
A ták obchodził Sálomon ná on cżáſ święto zácne / y wƺyſtek Izráel z nim / zgromádzenie wielkie od wejśćiá do Emát áż do rzeki Egipſkiej / przed Pánem / Bogiem náƺym / przez śiedm dni y przez śiedm dni / to jeſt / przez cżternáśćie dni.
A dniá óſmego rozpuśćił lud ; którzy błogoſłáwiąc królowi / rozeƺli śię do przybytków ſwoich / weſeląc śię / y ćieƺąc śię w ſercu ſwem ze wƺyſtkiego dobrego / które ucżynił Pán Dawidowi / ſłudze ſwemu / y Izráelowi / ludowi ſwemu.
