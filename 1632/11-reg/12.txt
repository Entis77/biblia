Tedy jecháł Roboám do Sychem ; bo w Sychem zebráł śię był wƺyſtek Izráel / áby go poſtánowili królem.
Y ſtáło śię / gdy uſłyƺáł Jeroboám / ſyn Nábátá / który był jeƺcże w Egipćie ; ( bo był ućiekł przed królem Sálomonem / y mieƺkáł Jeroboám w Egipćie. )
Tedy poſłáli y wezwáli go. Przetoż przyƺedłƺy Jeroboám / y wƺyſtko zgromádzenie Izráelſkie / rzekli do Roboámá / mówiąc :
Ojćiec twój włożył ná nas járzmo ćiężkie ; ále ty teraz ulżyj nám niewoli ſrogiej ojcá twego / y járzmá jego ćiężkiego / które włożył ná nas / á będźiemyć ſłużyli.
Który im rzekł : Odejdźćie / á po trzech dniách wróććie śię do mnie. Y odƺedł lud.
Tedy wƺedł w rádę król Roboám z ſtárƺymi / którzy ſtáwáli przed Sálomonem / ojcem jego / zá żywotá jego / mówiąc : Co wy rádźićie / jákąbym miał dáć odpowiedź ludowi temu?
Którzy mu odpowiedźieli / mówiąc : Jeſli dźiś powolny będźieƺ ludowi temu / á poſłucháƺ ich / y dáƺ im odpowiedź / á będźieƺ mówił do nich ſłowá łágodne / będą ſługámi twymi po wƺyſtkie dni.
Ale on opuśćiwƺy rádę ſtárƺych / którą mu podáli / wƺedł w rádę z młodźieńcámi / którzy z nim wzrośli / á którzy ſtáwáli przed nim ;
Y rzekł do nich : A wy co rádźićie / ábyśmy odpowiedźieli ludowi temu / który rzekł do mnie / mówiąc : Ulżyj járzmá / które włożył ojćiec twój ná nas?
Tedy mu odpowiedźieli oni młodźieńcy / którzy z nim wzrośli / mówiąc :Ták odpowieƺ temu ludowi / którzy mówili do ćiebie / á rzekli : Ojćiec twój włożył ná nas járzmo ćiężkie / ále nám go ty ulżyj ; ták rzecżeƺ do nich : Nájmniejƺy pálec mój miężƺy jeſt niż biodrá ojcá mego.
Przetoż teraz ojćiec mój kłádł ná was járzmo ćiężkie / ále ja przydám do járzmá wáƺego ; ojćiec mój káráł was bicżykámi / ále ja was będę káráł korbácżámi.
Przyƺedł tedy Jeroboám / y wƺyſtek lud do Roboámá dniá trzećiego / jáko był rozkázáł król / mówiąc : Wróććie śię do mnie dniá trzećiego.
Y dáł ſrogą odpowiedź król ludowi / opuśćiwƺy rádę ſtárƺych / którą mu byli dáli /
A rzekł do nich według rády młodźieńców / mówiąc : Ojćiec mój obćiążáł was járzmem ćiężkiem / ále ja przydám do járzmá wáƺego ; ojćiec mój káráł was bicżykámi / ále ja was będę káráł korbácżámi.
Y nie uſłucháł król ludu ; bo byłá przycżyná od Páná / áby doſyć ucżynił ſłowu ſwemu / które był powiedźiáł Pán przez Achyjáƺá Sylonitcżyká do Jeroboámá / ſyná Nábátowego.
A gdy widźiáł wƺyſtek Izráel / że ich nie uſłucháł król / odpowiedźiáł lud królowi / ták mówiąc : Cóż my mámy zá dźiáł w Dawidźie? á co zá dźiedźictwo w ſynu Iſájowym? Idź do námiotów ſwych / o Izráelu / á ty Dawidźie opátrz teraz dom twój. Y rozeƺli śię Izráelcżycy do námiotów ſwoich.
A ták tylko nád ſynámi Izráelſkimi / którzy mieƺkáli w mieśćiech Judzkich / królowáł Roboám.
Y poſłáł król Roboám Adorámá / który był poborcą / y ukámionowáł go wƺyſtek Izráel / áż umárł ; przetoż król Roboám / wśiádłƺy co rychlej ná wóz / ućiekł do Jeruzálemu.
A ták odſtąpili Izráelcżycy od domu Dawidowego / áż do dniá tego.
Y ſtáło śię / gdy uſłyƺáł wƺyſtek Izráel / że śię wróćił Jeroboám / poſłáwƺy przyzwáli go do zgromádzeniá / y poſtánowili go królem nád wƺyſtkim Izráelem. Nie zoſtáło przy domu Dawidowym jedno ſámo pokolenie Judowe.
A przyjecháwƺy Roboám do Jeruzálemu / zebráł wƺyſtek dom Judowy / y pokolenie Benjáminowe / ſto y ośmdźieśiąt tyśięcy mężów przebránych ku bojowi / áby wálcżyli z domem Izráelſkim / áżeby przywrócone było króleſtwo Roboámowi / ſynowi Sálomonowemu.
Y ſtáło śię ſłowo Boże do Semejáƺá / mężá Bożego / mówiąc :
Powiedz Roboámowi / ſynowi Sálomonowemu / królowi Judzkiemu y wƺyſtkiemu domowi Judowemu y Benjáminowemu / y innemu ludowi / mówiąc :
Ták mówi Pán : Nie wychodźćie / áni wálcżćie z bráćią ſwoją / ſynmi Izráelſkimi ; wróććie śię káżdy do domu ſwego : ábowiem odemnie śię tá rzecż ſtáłá. Y uſłucháli rozkázániá Páńſkiego / á wróćili śię / áby odeƺli według ſłowá Páńſkiego.
Potem zbudowáł Jeroboám Sychem ná górze Efráim / y mieƺkáł w nim / á ſtámtąd wyƺedłƺy pobudowáł Fánuel.
Y rzekł Jeroboám w ſercu ſwem : Wnetby śię wróćiło króleſtwo do domu Dawidowego.
Gdyby chádzáł ten lud ſpráwowáć ofiáry do domu Páńſkiego do Jeruzálemu / y obróćiłoby śię ſerce ludu tego do páná ſwego / do Roboámá / królá Judzkiego / á zábiwƺy mię / wróćiliby śię do Roboámá / królá Judzkiego.
Przetoż nárádźiwƺy śię król / ucżynił dwóch ćielców złotych / y mówił do ludu : Doſyćieśćie śię náchodźili do Jeruzálemu ; oto bogowie twoi / o Izráelu / którzy ćię wywiedli z źiemi Egipſkiej.
Y poſtáwił jednego w Betel / á drugiego poſtáwił w Dán.
Y było to pobudką do grzechu / bo chádzáł lud do jednego z tych bogów áż do Dán /
Ucżynił też dom ná wyżynách / y poſtánowił kápłány niektóre z poſpólſtwá / którzy nie byli z ſynów Lewiego.
Nádto uſtánowił Jeroboám święto urocżyſte mieśiącá óſmego / piętnáſtego dniá tegoż mieśiącá / nákƺtáłt świętá / które obchodzono w Judźie / y ofiárowáł ná ołtárzu. Toż ucżynił w Betel / ofiárując ćielcom / które był ucżynił ; poſtánowił też kápłány w Betel ná wyżynách / które był pocżynił.
Y ſpráwowáł też ofiáry ná ołtárzu / który był ucżynił w Betel / piętnáſtego dniá mieśiącá óſmego / onegoż mieśiącá / który był wymyślił w ſercu ſwojim ; y ucżynił święto urocżyſte ſynom Izráelſkim / á przyſtąpił do ołtárzá / áby kádźił.
