Tedy jechał Roboám do Sychem ; bo w Sychem zebrał śię był wƺyſtek Izráel / áby go poſtánowili królem.
Y ſtáło śię / gdy uſłyƺał Jeroboám / ſyn Nábátá / który był jeƺcże w Egipćie ; ( bo był ućiekł przed królem Sálomonem / y mieƺkał Jeroboám w Egipćie. )
Tedy poſłáli y wezwáli go. Przetoż przyƺedƺy Jeroboám / y wƺyſtko zgromádzenie Izráelſkie / rzekli do Roboámá / mówiąc :
Oćiec twój włożył ná nas járzmo ćiężkie ; ále ty teraz ulżyj nam niewoli ſrogiej ojcá twego / y járzmá jego ćiężkiego / które włożył ná nas / á będźiemyć ſłużyli.
Który im rzekł : Odejdźćie / á po trzech dniách wróććie śię do mnie. Y odƺedł lud.
Tedy wƺedł w rádę król Roboám z ſtárƺymi / którzy ſtáwáli przed Sálomonem / ojcem jego / zá żywotá jego / mówiąc : Co wy rádźićie / jákąbym miał dáć odpowiedź ludowi temu?
Którzy mu odpowiedźieli / mówiąc : Jeſli dźiś powolny będźieƺ ludowi temu / á poſłucháƺ ich / y dáƺ im odpowiedź / á będźieƺ mówił do nich ſłowá łágodne / będą ſługámi twymi po wƺyſtkie dni.
Ale on opuśćiwƺy rádę ſtárƺych / którą mu podáli / wƺedł w rádę z młodźieńcámi / którzy z nim wzrośli / á którzy ſtáwáli przed nim ;
Y rzekł do nich : A wy co rádźićie / ábyſmy odpowiedźieli ludowi temu / który rzekł do mnie / mówiąc : Ulżyj járzmá / które włożył oćiec twój ná nas?
Tedy mu odpowiedźieli oni młodźieńcy / którzy z nim wzrośli / mówiąc :Ták odpowieƺ temu ludowi / którzy mówili do ćiebie / á rzekli : Oćiec twój włożył ná nas járzmo ćiężkie / ále nam go ty ulżyj ; ták rzecżeƺ do nich : Nájmniejƺy pálec mój miężƺy jeſt niż biodrá ojcá mego.
Przetoż teraz oćiec mój kłádł ná was járzmo ćiężkie / ále ja przydám do járzmá wáƺego ; oćiec mój kárał was bicżykámi / ále ja was będę kárał korbácżámi.
Przyƺedł tedy Jeroboám / y wƺyſtek lud do Roboámá dniá trzećiego / jáko był rozkázał król / mówiąc : Wróććie śię do mnie dniá trzećiego.
Y dał ſrogą odpowiedź król ludowi / opuśćiwƺy rádę ſtárƺych / którą mu byli dáli /
A rzekł do nich według rády młodźieńców / mówiąc : Oćiec mój obćiążał was járzmem ćiężkiem / ále ja przydám do járzmá wáƺego ; oćiec mój kárał was bicżykámi / ále ja was będę kárał korbácżámi.
Y nie uſłuchał król ludu ; bo byłá przycżyná od Páná / áby doſyć ucżynił ſłowu ſwemu / które był powiedźiał Pán przez Achyjáƺá Sylonitcżyká do Jeroboámá / ſyná Nábátowego.
A gdy widźiał wƺyſtek Izráel / że ich nie uſłuchał król / odpowiedźiał lud królowi / ták mówiąc : Cóż my mamy zá dźiał w Dawidźie? á co zá dźiedźictwo w ſynu Iſájowym? Idź do námiotów ſwych / o Izráelu / á ty Dawidźie opátrz teraz dom twój. Y rozeƺli śię Izráelcżycy do námiotów ſwojich.
A ták tylko nád ſynámi Izráelſkimi / którzy mieƺkáli w mieśćiech Judzkich / królował Roboám.
Y poſłał król Roboám Adorámá / który był poborcą / y ukámionował go wƺyſtek Izráel / áż umárł ; przetoż król Roboám / wśiádƺy co rychlej ná wóz / ućiekł do Jeruzálemu.
A ták odſtąpili Izráelcżycy od domu Dawidowego / áż do dniá tego.
Y ſtáło śię / gdy uſłyƺał wƺyſtek Izráel / że śię wróćił Jeroboám / poſłáwƺy przyzwáli go do zgromádzenia / y poſtánowili go królem nád wƺyſtkim Izráelem. Nie zoſtáło przy domu Dawidowym jedno ſámo pokolenie Judowe.
A przyjecháwƺy Roboám do Jeruzálemu / zebrał wƺyſtek dom Judowy / y pokolenie Benjáminowe / ſto y ośmdźieśiąt tyśięcy mężów przebránych ku bojowi / áby wálcżyli z domem Izráelſkim / áżeby przywrócone było króleſtwo Roboámowi / ſynowi Sálomonowemu.
Y ſtáło śię ſłowo Boże do Semejáƺá / mężá Bożego / mówiąc :
Powiedz Roboámowi / ſynowi Sálomonowemu / królowi Judzkiemu y wƺyſtkiemu domowi Judowemu y Benjáminowemu / y innemu ludowi / mówiąc :
Ták mówi Pán : Nie wychodźćie / áni wálcżćie z bráćią ſwoją / ſynmi Izráelſkimi ; wróććie śię káżdy do domu ſwego : ábowiem odemnie śię tá rzecż ſtáłá. Y uſłucháli rozkázánia Páńſkiego / á wróćili śię / áby odeƺli według ſłowá Páńſkiego.
Potem zbudował Jeroboám Sychem ná górze Efráim / y mieƺkał w nim / á z támtąd wyƺedƺy pobudował Fánuel.
Y rzekł Jeroboám w ſercu ſwem : Wnetby śię wróćiło króleſtwo do domu Dawidowego.
Gdyby chádzał ten lud ſpráwowáć ofiáry do domu Páńſkiego do Jeruzálemu / y obróćiłoby śię ſerce ludu tego do páná ſwego / do Roboámá / królá Judzkiego / á zábiwƺy mię / wróćiliby śię do Roboámá / królá Judzkiego.
Przetoż nárádźiwƺy śię król / ucżynił dwu ćielców złotych / y mówił do ludu : Doſyćieśćie śię náchodźili do Jeruzálemu ; oto bogowie twoi / o Izráelu / którzy ćię wywiedli z źiemi Egipſkiej.
Y poſtáwił jednego w Betel / á drugiego poſtáwił w Dán.
Y było to pobudką do grzechu / bo chádzał lud do jednego z tych bogów áż do Dán /
Ucżynił też dom ná wyżynách / y poſtánowił kápłány niektóre z poſpólſtwá / którzy nie byli z ſynów Lewiego.
Nádto uſtánowił Jeroboám święto urocżyſte mieśiącá óſmego / piętnáſtego dniá tegoż mieśiącá / nákƺtáłt świętá / które obchodzono w Judźie / y ofiárował ná ołtárzu. Toż ucżynił w Betel / ofiárując ćielcom / które był ucżynił ; poſtánowił też kápłány w Betel ná wyżynách / które był pocżynił.
Y ſpráwował też ofiáry ná ołtárzu / który był ucżynił w Betel / piętnáſtego dniá mieśiącá óſmego / onegoż mieśiącá / który był wymyślił w ſercu ſwojim ; y ucżynił święto urocżyſte ſynom Izráelſkim / á przyſtąpił do ołtárzá / áby kádźił.
