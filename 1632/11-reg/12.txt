Tedy jecháł Roboám do Sychem ; bo w Sychem zebráł śię był wƺyſtek Izráel / áby go poſtánowili królem.
Y ſtáło śię / gdy uſłyƺáł Jeroboám / ſyn Nábátá / który był jeƺcże w Egipćie ; ( bo był ućiekł przed królem Sálomonem / y mieƺkáł Jeroboám w Egipćie. )
Tedy poſłáli y wezwáli go. Przetoż przyƺedłƺy Jeroboám / y wƺyſtko zgromádzenie Izráelſkie / rzekli do Roboámá / mówiąc :
Ojćiec twój włożył ná náſ járzmo ćiężkie ; ále ty teráz ulżyj nám niewoli ſrogiej ojcá twego / y járzmá jego ćiężkiego / które włożył ná náſ / á będziemyć ſłużyli.
Który im rzekł : Odejdźćie / á po trzech dniách wróććie śię do mnie. Y odƺedł lud.
Tedy wƺedł w rádę król Roboám z ſtárƺymi / którzy ſtáwáli przed Sálomonem / ojcem jego / zá żywotá jego / mówiąc : Co wy rádzićie / jákąbym miáł dáć odpowiedź ludowi temu?
Którzy mu odpowiedzieli / mówiąc : Jeźli dziś powolny będzieƺ ludowi temu / á poſłucháƺ ich / y dáƺ im odpowiedź / á będzieƺ mówił do nich ſłowá łágodne / będą ſługámi twymi po wƺyſtkie dni.
Ale on opuśćiwƺy rádę ſtárƺych / którą mu podáli / wƺedł w rádę z młodzieńcámi / którzy z nim wzrośli / á którzy ſtáwáli przed nim ;
Y rzekł do nich : A wy co rádzićie / ábyśmy odpowiedzieli ludowi temu / który rzekł do mnie / mówiąc : Ulżyj járzmá / które włożył ojćiec twój ná náſ?
Tedy mu odpowiedzieli oni młodzieńcy / którzy z nim wzrośli / mówiąc :Ták odpowieƺ temu ludowi / którzy mówili do ćiebie / á rzekli : Ojćiec twój włożył ná náſ járzmo ćiężkie / ále nám go ty ulżyj ; ták rzecżeƺ do nich : Nájmniejƺy pálec mój miężƺy jeſt niż biodrá ojcá mego.
Przetoż teráz ojćiec mój kłádł ná wáſ járzmo ćiężkie / ále já przydám do járzmá wáƺego ; ojćiec mój káráł wáſ bicżykámi / ále já wáſ będę káráł korbácżámi.
Przyƺedł tedy Jeroboám / y wƺyſtek lud do Roboámá dniá trzećiego / jáko był rozkázáł król / mówiąc : Wróććie śię do mnie dniá trzećiego.
Y dáł ſrogą odpowiedź król ludowi / opuśćiwƺy rádę ſtárƺych / którą mu byli dáli /
A rzekł do nich według rády młodzieńców / mówiąc : Ojćiec mój obćiążáł wáſ járzmem ćiężkiem / ále já przydám do járzmá wáƺego ; ojćiec mój káráł wáſ bicżykámi / ále já wáſ będę káráł korbácżámi.
Y nie uſłucháł król ludu ; bo byłá przycżyná od Páná / áby doſyć ucżynił ſłowu ſwemu / które był powiedziáł Pán przez Achyjáƺá Sylonitcżyká do Jeroboámá / ſyná Nábátowego.
A gdy widziáł wƺyſtek Izráel / że ich nie uſłucháł król / odpowiedziáł lud królowi / ták mówiąc : Cóż my mámy zá dziáł w Dawidzie? á co zá dziedzictwo w ſynu Iſájowym? Idź do námiotów ſwych / o Izráelu / á ty Dawidzie opátrz teráz dom twój. Y rozeƺli śię Izráelcżycy do námiotów ſwoich.
A ták tylko nád ſynámi Izráelſkimi / którzy mieƺkáli w mieśćiech Judzkich / królowáł Roboám.
Y poſłáł król Roboám Adorámá / który był poborcą / y ukámionowáł go wƺyſtek Izráel / áż umárł ; przetoż król Roboám / wśiádłƺy co rychlej ná wóz / ućiekł do Jeruzálemu.
A ták odſtąpili Izráelcżycy od domu Dawidowego / áż do dniá tego.
Y ſtáło śię / gdy uſłyƺáł wƺyſtek Izráel / że śię wróćił Jeroboám / poſłáwƺy przyzwáli go do zgromádzeniá / y poſtánowili go królem nád wƺyſtkim Izráelem. Nie zoſtáło przy domu Dawidowym jedno ſámo pokolenie Judowe.
A przyjecháwƺy Roboám do Jeruzálemu / zebráł wƺyſtek dom Judowy / y pokolenie Benjáminowe / ſto y ośmdzieśiąt tyśięcy mężów przebránych ku bojowi / áby wálcżyli z domem Izráelſkim / áżeby przywrócone było króleſtwo Roboámowi / ſynowi Sálomonowemu.
Y ſtáło śię ſłowo Boże do Semejáƺá / mężá Bożego / mówiąc :
Powiedz Roboámowi / ſynowi Sálomonowemu / królowi Judzkiemu y wƺyſtkiemu domowi Judowemu y Benjáminowemu / y innemu ludowi / mówiąc :
Ták mówi Pán : Nie wychodźćie / áni wálcżćie z bráćią ſwoją / ſynmi Izráelſkimi ; wróććie śię káżdy do domu ſwego : ábowiem odemnie śię tá rzecż ſtáłá. Y uſłucháli rozkázániá Páńſkiego / á wróćili śię / áby odeƺli według ſłowá Páńſkiego.
Potem zbudowáł Jeroboám Sychem ná górze Efráim / y mieƺkáł w nim / á ſtámtąd wyƺedłƺy pobudowáł Fánuel.
Y rzekł Jeroboám w ſercu ſwem : Wnetby śię wróćiło króleſtwo do domu Dawidowego.
Gdyby chádzáł ten lud ſpráwowáć ofiáry do domu Páńſkiego do Jeruzálemu / y obróćiłoby śię ſerce ludu tego do páná ſwego / do Roboámá / królá Judzkiego / á zábiwƺy mię / wróćiliby śię do Roboámá / królá Judzkiego.
Przetoż nárádziwƺy śię król / ucżynił dwóch ćielców złotych / y mówił do ludu : Doſyćieśćie śię náchodzili do Jeruzálemu ; oto bogowie twoi / o Izráelu / którzy ćię wywiedli z ziemi Egipſkiej.
Y poſtáwił jednego w Betel / á drugiego poſtáwił w Dán.
Y było to pobudką do grzechu / bo chádzáł lud do jednego z tych bogów áż do Dán /
Ucżynił też dom ná wyżynách / y poſtánowił kápłány niektóre z poſpólſtwá / którzy nie byli z ſynów Lewiego.
Nádto uſtánowił Jeroboám święto urocżyſte mieśiącá óſmego / piętnáſtego dniá tegoż mieśiącá / nákƺtáłt świętá / które obchodzono w Judzie / y ofiárowáł ná ołtárzu. Toż ucżynił w Betel / ofiárując ćielcom / które był ucżynił ; poſtánowił też kápłány w Betel ná wyżynách / które był pocżynił.
Y ſpráwowáł też ofiáry ná ołtárzu / który był ucżynił w Betel / piętnáſtego dniá mieśiącá óſmego / onegoż mieśiącá / który był wymyślił w ſercu ſwojim ; y ucżynił święto urocżyſte ſynom Izráelſkim / á przyſtąpił do ołtárzá / áby kádził.
