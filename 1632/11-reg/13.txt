A oto mąż Boży przyƺedł z Judztwá z ſłowem Páńſkim do Betel / gdy Jeroboám ſtał u ołtárzá / áby kádźił.
Y záwołał przećiw ołtárzowi ſłowem Páńſkim / y rzekł : Ołtárzu / ołtárzu / ták mówi Pán : Oto ſyn národźi śię domowi Dawidowemu imieniem Jozyjáƺ / który będźie ofiárował ná tobie kápłány wyżyn / kádzące ná tobie / y kośći ludzkie popálą ná tobie.
Y dał mu znák dniá onegoż / mówiąc : Tenći jeſt znák / że to mówił Pán : Oto śię ołtárz rozpádnie / á wyſypie śię popiół / który jeſt ná nim.
A gdy uſłyƺał król Jeroboám ſłowo mężá Bożego / które obwoływał przećiw ołtárzowi w Betel / śćiągnął rękę ſwą od ołtárzá / mówiąc : Pojmájćie go. Y uſchłá ręká jego / którą był wyćiągnął przećiw niemu / á niemógł jey przyćiągnąć do śiebie.
Ołtárz śię też rozpádł / á wyſypał śię popiół z ołtárzá według znáku / który był dał mąż Boży ſłowem Páńſkim.
Przetoż odpowiádájąc król / rzekł do mężá Bożego : Proƺę ćię / proś oblicża Páná Bogá twego / á módl śię zá mną / áby śię wróćiłá ręká mojá do mnie. Y modlił śię mąż Boży oblicżu Páńſkimu / y wróćiłá śię ręká królewſká do niego / y byłá jáko pierwej.
Tedy rzekł król do mężá Bożego : Pójdź ze mną do domu / ábyś śię pośilił / á dam ći upominek.
Ale rzekł mąż Boży do królá : Byśmi dał połowę domu twego / nie pojádę z tobą / áni będę jádł chlebá / áni będę pił wody ná tym miejſcu.
Bo mi ták Pán rozkázał ſłowem ſwojim / mówiąc : Nie będźieƺ jádł chlebá / áni będźieƺ pił wody / áni śię wróćiƺ tą drogą / którąś przyƺedł.
Odƺedł tedy inƺą drogą / á nie wróćił śię tą drogą / którą był przyƺedł do Betel.
A prorok niejáki ſtáry mieƺkał w Betel / którego ſyn przyƺedƺy / opowiedźiał mu wƺyſtkę ſpráwę / którą był ucżynił onegoż dniá mąż Boży w Betel / y ſłowá / które mówił do królá / opowiedźieli ojcu ſwemu.
Y rzekł im oćiec ich : Którąż drogą poƺedł? Y pokázáli ſynowie jego drogę / którą poƺedł on mąż Boży / który był przyƺedł z Judztwá.
Zátem rzekł ſynom ſwym : Ośiodłájćie mi oſłá ; y ośiodłáli mu oſłá / y wśiádł nań /
Y jechał zá mężem Bożym / á ználázł go śiedzącego pod dębem / y rzekł mu : Tyżeś jeſt on mąż Boży / któryś przyƺedł z Judztwá? A on rzekł : Jeſtem.
Y rzekł do niego : Pójdź zemną do domu / żebyś jádł chleb.
Ale mu on rzekł : Nie mogę śię wróćić z tobą / áni iść z tobą / áni będę jádł chlebá / áni będę pił wody z tobą ná tym miejſcu ;
Bo śię ſtáłá do mnie mowá ſłowem Páńſkim : Nie będźieƺ tám jádł chlebá / áni pił wody / áni śię wróćiƺ idąc tą drogą / którąś ƺedł.
Któremu on odpowiedźiał : Y jam prorok jáko y ty ; Anjoł też rzekł do mnie ſłowem Páńſkim / mówiąc : Wróć go z ſobą do domu twego / áby jádł chleb / y pił wodę. Y ták ſkłámał przed nim.
Y wróćił śię z nim / á jádł chleb w domu jego / y pił wodę.
A gdy śiedźieli u ſtołu / ſtáło śię ſłowo Páńſkie do proroká / który go był wróćił ;
Y záwołał ná mężá Bożego / który był przyƺedł z Judztwá / mówiąc : Ták mówi Pán : Przeto żeś był odpornym uſtom Páńſkim / á nie ſtrzegłeś rozkázánia / któreć przykázał Pán / Bóg twój :
Aleś śię wróćił / y jádłeś chleb / á piłeś wodę ná miejſcu / o któremem ći był rzekł : Nie będźieƺ tám jádł chlebá / áni pił wody : nie będźie pochowány trup twój w grobie ojców twojich.
A ták gdy śię nájádł chlebá y nápił śię / ośiodłał oſłá prorokowi onemu / którego był wróćił.
A gdy odjechał / ſpotkał go lew w drodze / y zábił go. A trup jego był porzucony w drodze / á ośieł ſtał wedle niego / lew tákże ſtał podle trupá.
A oto mężowie mimo idący ujrzeli trupá porzuconego ná drodze y lwá ſtojącego podle niego : którzy przyƺedƺy powiedźieli to w mieśćie / w którym on ſtáry prorok mieƺkał.
Co gdy uſłyƺał on prorok / który go był wróćił z drogi / rzekł : Mąż Boży jeſt / który był odpornym uſtom Páńſkim ; przetoż podał go Pán lwowi / który go potárł / y zábił go według ſłowá Páńſkiego / które mu był powiedźiał.
Nádto rzekł do ſynów ſwojich / mówiąc : Ośiodłájćie mi oſłá. Y ośiodłáli.
A wyjecháwƺy ználázł trupá jego porzuconego ná drodze / á oſłá y lwá ſtojące przy trupie / ále nie jádł lew onego trupá / áni obráźił oſłá.
Tedy wźiął prorok trupá mężá Bożego / á włożywƺy go ná oſłá / przywiózł go ; y przyƺedł do miáſtá ſwego / áby płákał y pogrzebł go.
A położył trupá jego w grobie ſwoim / y płákáli go / mówiąc : Ach bráćie mój!
A pochowáwƺy go / rzekł do ſynów ſwojich : Gdy ja umrę / pochowájćie mię w tym grobie / w którym jeſt mąż Boży pochowány ; podle kośći jego połóżćie kośći moje /
Bo zápewne śię to ſtánie / co obwołał ſłowem Páńſkim / przećiw ołtárzowi / który jeſt w Betel / y przećiwko wƺyſtkim domom wyżyn / które ſą w mieśćiech Sámáryjſkich.
To gdy śię ſtáło / przećież śię nie odwróćił Jeroboám od drogi ſwej złej / ále znowu nácżynił z poſpolitego ludu kápłanów wyżyn ; kto jedno chćiał / poświęcał rękę jego / áby był kápłánem wyżyn.
Y byłá tá rzecż domowi Jeroboámowemu przycżyną do grzechu / áby był wykorzeniony y wygłádzony z źiemi.
