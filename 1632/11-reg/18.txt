Potem po wielu dniách / miánowićie po onym roku trzećim / ſtáło śię ſłowo Páńſkie do Elijáƺá / mówiąc : Idź / ukáż śię Achábowi ; bo ſpuƺcżę deƺcż ná źiemię.
Szedł tedy Elijáƺ / áby śię ukázał Achábowi ; á był głód gwáłtowny w Sámáryi.
Y záwołał Acháb Abdyjáƺá / który był ſpráwcą domu jego. ( A Abdyjáƺ śię bárzo Páná bał ;
Bo gdy mordowáłá Jezábel proroki Páńſkie / tedy wźiął Abdyjáƺ ſto proroków / y ſkrył ich po pięćdźieśiąt do jáſkini / y żywił je chlebem y wodą. )
Y rzekł Acháb do Abdyjáƺá : Idź przez źiemię do wƺyſtkich źródeł wód / y do wƺyſtkich potoków / ázá gdźie znájdźiemy tráwę / żebyſmy żywo záchowáli konie y muły / y żebyſmy nie zgubili bydłá.
Y rozdźielili ſobie źiemię / którą przejść mieli. Acháb ſám ƺedł jedną drogą / Abdyjáƺ też ƺedł drugą drogą oſobno.
A gdy Abdyjáƺ był w drodze / oto śię z nim Elijáƺ ſpotkał / który gdy go poznał / upádł ná oblicże ſwoje / y rzekł : A tyżeś jeſt pán mój Elijáƺ?
Y odpowiedźiał mu : Jam jeſt. Idź / powiedz pánu twemu : Oto Elijáƺ tu jeſt.
Do którego on rzekł : Cóżem zgrzeƺył / iż wydáwáƺ ſługę twego w ręce Achábowe / áby mię zábił?
Jáko żywy Pán / Bóg twój / że niemáƺ narodu / y króleſtwá / gdźieby nie poſłał Pán mój / áby ćię ƺukáno ; á gdy powiedźiáno / iż ćię niemáƺ / tedy obowiązał przyśięgą króleſtwá y narody / jáko ćię ználeść nie mogą.
A ty mi teraz mówiƺ : Idź / á powiedz pánu twemu : Oto Elijáƺ.
Y ſtáłoby śię / gdybym ja odƺedł od ćiebie / żeby ćię Duch Páńſki zánióſł / gdźiebym nie wiedźiał ; á ja ƺedƺy opowiedźiáłbym Achábowi / á gdyby ćię nie ználázł / zábiłby mię ; á ſługá twój boi śię Páná od dźiećińſtwá ſwego.
Azaż nie powiedźiáno pánu memu / com ucżynił / gdy mordowáłá Jezábel proroki Páńſkie? żem ſkrył z proroków Páńſkich ſto mężów / po pięćdźieśiąt mężów w jáſkini / y żywiłem je chlebem y wodą?
A ty teraz mówiƺ : Idź / powiedz pánu twemu : Oto Elijáƺ ; y zábije mię.
Y odpowiedźiał Elijáƺ : Jáko żywy Pán zaſtępów / przed którego oblicżnośćią ſtoję / że mu śię dźiś ukáżę.
A ták ƺedł Abdyjáƺ przećiw Achábowi / y oznájmił mu to. Przetoż ƺedł Acháb przećiw Elijáƺowi.
A ujrzáwƺy Acháb Elijáƺá / rzekł Acháb do niego : Azaż nie ty jeſteś / który cżyniƺ zámięƺánie w Izráelu?
Ná co mu odpowiedźiał : Nie jáć cżynię zámięƺánie w Izráelu / ále ty y dom ojcá twego / gdyż opuśćiwƺy rozkázánia Páńſkie náśládujećie Báálów.
Przetoż teraz poślij / á zbierz do mnie wƺyſtkiego Izráelá ná górę Kármel / y proroków Báálowych cżtery ſtá y pięćdźieśiąt / przytem proroków gájowych cżtery ſtá / którzy jádáją z ſtołu Jezábeli.
Poſłał tedy Acháb do wƺyſtkich ſynów Izráelſkich / y zebrał te proroki ná górę Kármel.
A przyſtąpiwƺy Elijáƺ do wƺyſtkiego ludu / rzekł : Y długoż będźiećie chrámáć ná obie ſtrony? Jeſli Pán jeſt Bogiem / idźćież zá nim ; á jeſli Báál / idźćież zá nim. Y nie odpowiedźiał mu lud y ſłowá.
Tedy rzekł Elijáƺ do ludu : Jam tylko ſám zoſtał prorok Páńſki ; á proroków Báálowych cżtery ſtá y pięćdźieśiąt mężów.
Niech nam dádzą dwu ćielców / á niech ſobie obiorą ćielcá jednego / á porąbią go ná ƺtuki / y włożą ná drwá ; ále ogniá niech nie podkłádáją : ja też przygotuję drugiego ćielcá / którego włożę ná drwá / á ogniá nie podłożę.
Potem wzywájćie imienia bogów wáƺych / á ja będę wzywał imienia Páńſkiego / á Bóg / który śię ozwie przez ogień / ten niech będźie Bogiem. Ná co odpowiádájąc wƺyſtek lud rzekł : Dobrześ powiedźiał.
Y rzekł Elijáƺ do proroków Báálowych : Obierzćie ſobie ćielcá jednego / á zgotujćie go pierwej / bo was jeſt więcey ; wzywájćież imienia bogów wáƺych / ále ogniá nie podkłádájćie.
A ták wźięli ćielcá / którego im dał / á zgotowáwƺy wzywáli imienia Báálowego od poránku áż do południá / mówiąc : O Báálu / wyſłucháj nas! Ale nie było głoſu / áni ktoby odpowiedźiał. Y ſkákáli koło ołtárzá / który byli ucżynili.
A gdy było południe / náśmiewał śię z nich Elijáƺ / mówiąc : Wołájćie więkƺym głoſem / ponieważ jeſt bóg ; tylko że śię álbo zámyślił / álbo jeſt zábáwny / álbo też jeſt w drodze ; álbo też śpi / ázá ocući.
A ták wołáli głoſem wielkim / y rzezáli śię według zwycżáju ſwego nożámi y włócżenkámi / áż śię krwią oblewáli.
Y ſtáło śię / gdy minęło południe / że prorokowáli áż do cżáſu ofiárowánia ofiáry śniedney ; ále nie było głoſu / áni ktoby odpowiedźiał / áni ktoby wyſłuchał.
Zátem rzekł Elijáƺ do wƺyſtkiego ludu : Przyſtąpćie do mnie. Y przyſtąpił wƺyſtek lud do niego. Tedy nápráwił ołtárz Páńſki / który był rozwálony.
Abowiem wźiął Elijáƺ dwánaśćie kámieni ; ( według licżby pokolenia ſynów Jákóbowych / do którego śię ſtáło ſłowo Páńſkie / mówiąc : Izráel będźie imię twoje. )
Y zbudował z tego kámieniá ołtárz w imię Páńſkie / á ucżynił około ołtárzá ƺeroki rów / coby mógł dwie miáry zbożá wyśiáć.
Potem ułożył drwá / y ná ƺtuki porąbał ćielcá / y kłádł go ná drwá.
Y rzekł : Nápełnijćie cżtery wiádrá wodą / á wylijćie ná cáłopálenie y ná drwá. Rzekł nádto : Powtórzćie / y powtórzyli ; rzekł jeƺcże : Ucżyńćie po trzećie / y ucżynili po trzećie /
Ták że płynęły wody około ołtárzá / áż y rów był nápełniony wodą.
Y ſtáło śię / gdy był cżás ſpráwowánia ofiáry śniedney / przyſtąpił Elijáƺ prorok / y rzekł : Pánie / Boże Abráhámá / Izááká / y Izráelá! dźiś niech poznáją / żeś ty jeſt Bogiem w Izráelu / á jam ſługá twój / á żem według ſłowá twego ucżynił to wƺyſtko.
Wyſłucháj mię Pánie / wyſłucháj mię / áby poznał ten lud / żeś ty Pánie jeſt Bogiem / gdybyś záś náwróćił ſercá ich.
Tedy ſpádł ogień Páńſki / y pożárł cáłopálenie / y drwá / y kámienie / y proch ; á wodę / która byłá w rowie / wyſuƺył.
Co gdy ujrzał wƺyſtek lud / upádli ná oblicże ſwe / y rzekli : Pán jeſt Bogiem / Pánći jeſt Bogiem.
Tedy rzekł Elijáƺ do nich : Pojmájćie proroki Báálowe / á żáden niech z nich nie uchodźi. Y pojmáno je. A ták odwiódł je Elijáƺ do potoku Cyſon / y támże je pobił.
Potem rzekł Elijáƺ do Achábá : Idź / jedz / á pij ; ábowiem oto ƺum dżdżu wielkiego.
Tedy ƺedł Acháb / áby jádł y pił ; á Elijáƺ wſtąpił ná wierzch Kármelu / y położył śię ná źiemię / á włożył twárz ſwoję miedzy koláná ſwoje.
Potem rzekł do ſługi ſwego : Idź teraz / á ſpojrzyj ku morzu. Który poƺedł / á ſpojrzáwƺy rzekł : Niemáƺ nic. Záśię rzekł : Idź / á wrácáj śię po śiedm kroć.
A zá śiódmym rázem rzekł : Oto obłok máły jáko dłoń cżłowiecżá wyſtępuje z morzá. Tedy on rzekł : Idź / á powiedz Achábowi : Záprzęgáj / á ujeżdżáj / áby ćię deƺcż nie záſtał.
Y ſtáło śię miedzy tym / że śię niebioſá obłokámi y wiátrem záćmiły / ſkąd był deƺcż wielki. A ták wśiádƺy Acháb / jechał do Jezreelá.
A ręká Páńſka byłá nád Elijáƺem ; y przepáſał biodrá ſwe / y bieżał przed Achábem / áż przyƺedł do Jezreelá.
