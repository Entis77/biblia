Potem po wielu dniách / miánowićie po onym roku trzećim / ſtáło śię ſłowo Páńſkie do Elijáƺá / mówiąc : Idź / ukáż śię Achábowi ; bo ſpuƺcżę deƺcż ná źiemię.
Szedł tedy Elijáƺ / áby śię ukázáł Achábowi ; á był głód gwáłtowny w Sámáryi.
Y záwołáł Acháb Abdyjáƺá / który był ſpráwcą domu jego. ( A Abdyjáƺ śię bárdzo Páná báł ;
Bo gdy mordowáłá Jezábel proroki Páńſkie / tedy wźiął Abdyjáƺ ſto proroków / y ſkrył ich po pięćdźieśiąt do jáſkini / y żywił je chlebem y wodą. )
Y rzekł Acháb do Abdyjáƺá : Idź przez źiemię do wƺyſtkich źródeł wód / y do wƺyſtkich potoków / ázá gdźie znájdźiemy tráwę / żebyśmy żywo záchowáli konie y muły / y żebyśmy nie zgubili bydłá.
Y rozdźielili ſobie źiemię / którą przejść mieli. Acháb ſám ƺedł jedną drogą / Abdyjáƺ też ƺedł drugą drogą oſobno.
A gdy Abdyjáƺ był w drodze / oto śię z nim Elijáƺ ſpotkáł / który gdy go poznáł / upádł ná oblicże ſwoje / y rzekł : A tyżeś jeſt pán mój Elijáƺ?
Y odpowiedźiáł mu : Jám jeſt. Idź / powiedz pánu twemu : Oto Elijáƺ tu jeſt.
Do którego on rzekł : Cóżem zgrzeƺył / iż wydáwáƺ ſługę twego w ręce Achábowe / áby mię zábił?
Jáko żywy Pán / Bóg twój / że niemáƺ národu / y króleſtwá / gdźieby nie poſłáł Pán mój / áby ćię ƺukáno ; á gdy powiedźiáno / iż ćię niemáƺ / tedy obowiązáł przyśięgą króleſtwá y národy / jáko ćię ználeść nie mogą.
A ty mi teráz mówiƺ : Idź / á powiedz pánu twemu : Oto Elijáƺ.
Y ſtáłoby śię / gdybym já odƺedł od ćiebie / żeby ćię Duch Páńſki zánióſł / gdźiebym nie wiedźiáł ; á já ƺedłƺy opowiedźiáłbym Achábowi / á gdyby ćię nie ználázł / zábiłby mię ; á ſługá twój boi śię Páná od dźiećińſtwá ſwego.
Azáż nie powiedźiáno pánu memu / com ucżynił / gdy mordowáłá Jezábel proroki Páńſkie? żem ſkrył z proroków Páńſkich ſto mężów / po pięćdźieśiąt mężów w jáſkini / y żywiłem je chlebem y wodą?
A ty teráz mówiƺ : Idź / powiedz pánu twemu : Oto Elijáƺ ; y zábije mię.
Y odpowiedźiáł Elijáƺ : Jáko żywy Pán záſtępów / przed którego oblicżnośćią ſtoję / że mu śię dźiś ukáżę.
A ták ƺedł Abdyjáƺ przećiw Achábowi / y oznájmił mu to. Przetoż ƺedł Acháb przećiw Elijáƺowi.
A ujrzáwƺy Acháb Elijáƺá / rzekł Acháb do niego : Azáż nie ty jeſteś / który cżyniƺ zámięƺánie w Izráelu?
Ná co mu odpowiedźiáł : Nie jáć cżynię zámięƺánie w Izráelu / ále ty y dom ojcá twego / gdyż opuśćiwƺy rozkázániá Páńſkie náśládujećie Báálów.
Przetoż teráz poślij / á zbierz do mnie wƺyſtkiego Izráelá ná górę Kármel / y proroków Báálowych cżtery ſtá y pięćdźieśiąt / przytem proroków gájowych cżtery ſtá / którzy jádáją z ſtołu Jezábeli.
Poſłáł tedy Acháb do wƺyſtkich ſynów Izráelſkich / y zebráł te proroki ná górę Kármel.
A przyſtąpiwƺy Elijáƺ do wƺyſtkiego ludu / rzekł : Y długoż będźiećie chrámáć ná obie ſtrony? Jeźli Pán jeſt Bogiem / idźćież zá nim ; á jeźli Báál / idźćież zá nim. Y nie odpowiedźiáł mu lud y ſłowá.
Tedy rzekł Elijáƺ do ludu : Jám tylko ſám zoſtáł prorok Páńſki ; á proroków Báálowych cżtery ſtá y pięćdźieśiąt mężów.
Niech nám dádzą dwóch ćielców / á niech ſobie obiorą ćielcá jednego / á porąbią go ná ƺtuki / y włożą ná drwá ; ále ogniá niech nie podkłádáją : já też przygotuję drugiego ćielcá / którego włożę ná drwá / á ogniá nie podłożę.
Potem wzywájćie imieniá bogów wáƺych / á já będę wzywáł imieniá Páńſkiego / á Bóg / który śię ozwie przez ogień / ten niech będźie Bogiem. Ná co odpowiádájąc wƺyſtek lud rzekł : Dobrześ powiedźiáł.
Y rzekł Elijáƺ do proroków Báálowych : Obierzćie ſobie ćielcá jednego / á zgotujćie go pierwej / bo wáſ jeſt więcej ; wzywájćież imieniá bogów wáƺych / ále ogniá nie podkłádájćie.
A ták wźięli ćielcá / którego im dáł / á zgotowáwƺy wzywáli imieniá Báálowego od poránku áż do południá / mówiąc : O Báálu / wyſłucháj náſ! Ale nie było głoſu / áni ktoby odpowiedźiáł. Y ſkákáli koło ołtárzá / który byli ucżynili.
A gdy było południe / náśmiewáł śię z nich Elijáƺ / mówiąc : Wołájćie więkƺym głoſem / poniewáż jeſt bóg ; tylko że śię álbo zámyślił / álbo jeſt zábáwny / álbo też jeſt w drodze ; álbo też śpi / ázá ocući.
A ták wołáli głoſem wielkim / y rzezáli śię według zwycżáju ſwego nożámi y włócżenkámi / áż śię krwią oblewáli.
Y ſtáło śię / gdy minęło południe / że prorokowáli áż do cżáſu ofiárowániá ofiáry śniednej ; ále nie było głoſu / áni ktoby odpowiedźiáł / áni ktoby wyſłucháł.
Zátem rzekł Elijáƺ do wƺyſtkiego ludu : Przyſtąpćie do mnie. Y przyſtąpił wƺyſtek lud do niego. Tedy nápráwił ołtárz Páńſki / który był rozwálony.
Abowiem wźiął Elijáƺ dwánáśćie kámieni ; ( według licżby pokoleniá ſynów Jákóbowych / do którego śię ſtáło ſłowo Páńſkie / mówiąc : Izráel będźie imię twoje. )
Y zbudowáł z tego kámieniá ołtárz w imię Páńſkie / á ucżynił około ołtárzá ƺeroki rów / coby mógł dwie miáry zbożá wyśiáć.
Potem ułożył drwá / y ná ƺtuki porąbáł ćielcá / y kłádł go ná drwá.
Y rzekł : Nápełnijćie cżtery wiádrá wodą / á wylijćie ná cáłopálenie y ná drwá. Rzekł nádto : Powtórzćie / y powtórzyli ; rzekł jeƺcże : Ucżyńćie po trzećie / y ucżynili po trzećie /
Ták że płynęły wody około ołtárzá / áż y rów był nápełniony wodą.
Y ſtáło śię / gdy był cżáſ ſpráwowániá ofiáry śniednej / przyſtąpił Elijáƺ prorok / y rzekł : Pánie / Boże Abráhámá / Izááká / y Izráelá! dźiś niech poznáją / żeś ty jeſt Bogiem w Izráelu / á jám ſługá twój / á żem według ſłowá twego ucżynił to wƺyſtko.
Wyſłucháj mię Pánie / wyſłucháj mię / áby poznáł ten lud / żeś ty Pánie jeſt Bogiem / gdybyś záś náwróćił ſercá ich.
Tedy ſpádł ogień Páńſki / y pożárł cáłopálenie / y drwá / y kámienie / y proch ; á wodę / którá byłá w rowie / wyſuƺył.
Co gdy ujrzáł wƺyſtek lud / upádli ná oblicże ſwe / y rzekli : Pán jeſt Bogiem / Pánći jeſt Bogiem.
Tedy rzekł Elijáƺ do nich : Pojmájćie proroki Báálowe / á żáden niech z nich nie uchodźi. Y pojmáno je. A ták odwiódł je Elijáƺ do potoku Cyſon / y támże je pobił.
Potem rzekł Elijáƺ do Achábá : Idź / jedz / á pij ; ábowiem oto ƺum dżdżu wielkiego.
Tedy ƺedł Acháb / áby jádł y pił ; á Elijáƺ wſtąpił ná wierzch Kármelu / y położył śię ná źiemię / á włożył twárz ſwoję między koláná ſwoje.
Potem rzekł do ſługi ſwego : Idź teráz / á ſpojrzyj ku morzu. Który poƺedł / á ſpojrzáwƺy rzekł : Niemáƺ nic. Záśię rzekł : Idź / á wrácáj śię po śiedm kroć.
A zá śiódmym rázem rzekł : Oto obłok máły jáko dłoń cżłowiecżá wyſtępuje z morzá. Tedy on rzekł : Idź / á powiedz Achábowi : Záprzęgáj / á ujeżdżáj / áby ćię deƺcż nie záſtáł.
Y ſtáło śię między tem / że śię niebioſá obłokámi y wiátrem záćmiły / ſkąd był deƺcż wielki. A ták wśiádłƺy Acháb / jecháł do Jezreelá.
A ręká Páńſká byłá nád Elijáƺem ; y przepáſáł biodrá ſwe / y bieżáł przed Achábem / áż przyƺedł do Jezreelá.
