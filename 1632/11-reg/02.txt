A gdy śię przybliżáł cżáſ śmierći Dawidowej / rozkázáł Sálomonowi / ſynowi ſwemu / mówiąc :
Ja idę w drogę wƺyſtkiej źiemi / á ty zmácniáj śię / y bądź mężem.
Záchowywáj uſtáwy Páná / Bogá twego / ábyś chodźił drogámi jego / y przeſtrzegáł wyroków jego / y przykázáń jego y ſądów jego / y świádectw jego / jáko nápiſáno w zakonie Mojżeƺowym / ábyć śię ƺcżęśćiło wƺyſtko / co ſpráwowáć będźieƺ / y we wƺyſtkiem / do cżego śię obróćiƺ :
Żeby utwierdźił Pán ſłowo ſwoje / które rzekł do mnie / mówiąc : Jeſli będą ſtrzedz ſynowie twoi drogi ſwej / chodząc przedemną w práwdźie / ze wƺyſtkiego ſercá ſwego / y ze wƺyſtkiej duƺy ſwojey / tedy nie będźie wytrácony po tobie mąż z ſtolicy Izráelſkiej.
A ty też wieƺ / co mi ucżynił Joáb ſyn Sárwii / co ucżynił dwom hetmánom wojſk Izráelſkich / Abnerowi / ſynowi Nerowemu / y Amáźie / ſynowi Jeterowemu / że je pozábijáł / á wyláł krew jáko ná wojnie cżáſu pokoju / y zmázáł krwią jáko ná wojnie páſ ſwój rycerſki / który miał ná biodrách ſwoich / y bóty ſwoje / które miał ná nogách ſwoich.
Ucżyniƺ tedy według mądrośći twojey / á nie dopuśćiƺ zejść ſędźiwośći jego w pokoju do grobu.
Ale nád ſynámi Bárſylái Gáláádcżyká użyjeƺ miłośierdźiá / á niech jádáją weſpół z inƺymi u ſtołu twego ; ábowiem oni tákimże ſpoſobem przyƺli do mnie / gdym ućiekáł przed Abſálomem / brátem twoim.
Oto też jeſt u ćiebie Semej / ſyn Giery / ſyná Jemini / z Báhurym / który mi też złorzecżył złorzecżeniem wielkiem w on dźień / gdym ƺedł do Máhánáim ; á wƺákże záƺedł mi drogę u Jordánu / y przyśiągłem mu przez Páná / mówiąc : Nie zámorduję ćię miecżem.
Teraz jednák nie przepuƺcżáj mu tego / á iżeś jeſt mężem mądrym / będźieƺ wiedźiáł / co mu máƺ ucżynić / ábyś wprowádźił ſędźiwość jego ze krwią do grobu.
Zátem záſnął Dawid z ojcy ſwoimi / á pogrzebiony jeſt w mieśćie Dawidowem.
A dni / których królowáł Dawid nád Izráelem / było cżterdźieśći lát. W Hebronie królowáł śiedm lát / á w Jeruzálemie królowáł trzydźieśći y trzy látá.
A ták Sálomon uśiádł ná ſtolicy Dawidá ojcá ſwego / y zmocniło śię bárdzo króleſtwo jego.
Tedy przyƺedł Adonijáƺ / ſyn Hággity / do Betſáby / mátki Sálomonowej / któremu oná rzekłá : A ſpokojneż jeſt przyjśćie twoje? A on odpowiedźiáł : Spokojne.
Nádto rzekł : Mam nieco mówić z tobą. A oná rzekłá : Mów.
Tedy rzekł : Ty wieƺ / iż moje było króleſtwo / á ná mię obróćili byli wƺyſcy Izráelcżycy twárz ſwoję / ábym królowáł ; ále przenieśione jeſt króleſtwo / y doſtáło śię brátu memu ; bo mu od Páná náznácżone było.
Przetoż ćię teraz proƺę o jednę rzecż / á nie odmáwiáj mi tego. A oná mu rzekłá : Mów.
Zátem on rzekł : Mów proƺę do Sálomoná królá / ( bo wiem / żeć nie odmówi / )áby mi dáł Abiſág Sunámitkę zá żonę.
Y odpowiedźiáłá Betſábá : Dobrze ; będę mówiłá o ćię z królem.
A ták ƺłá Betſábá do królá Sálomoná / áby z nim mówiłá zá Adonijáƺem ; y wſtáł król przećiwko niej / á pokłoniwƺy śię jey uśiádł ná ſtolicy ſwej ; kázáł też poſtáwić ſtolicę mátce ſwej / która śiádłá po práwicy jego.
Y rzekłá : Proƺę ćię o jednę máłą rzecż / nie odmáwiáj mi. Y odpowiedźiáł jey król : Proś mátko mojá ; ábowiem ći nie odmówię.
Tedy rzekłá : Niech będźie dáná Abiſág Sunámitká Adonijáƺowi / brátu twemu / zá żonę.
Lecż odpowiedźiáł król Sálomon / y rzekł mátce ſwojey : Przecżże prośiƺ o Abiſág Sunámitkę Adonijáƺowi? uproś mu y króleſtwo / ábowiem on jeſt brátem moim ſtárƺym nád mię / á má po ſobie Abijátárá kápłáná / y Joábá / ſyná Sárwii.
Y przyśiągł król Sálomon przez Páná / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / że przećiwko duƺy ſwej mówił Adonijáƺ te ſłowá.
A teraz jáko żywy Pán / który mię utwierdźił / y poſádźił ná ſtolicy Dawidá ojcá mojego / y który mi zbudowáł dom / jáko obiecáł / iż dźiś zábity będźie Adonijáƺ.
A ták poſłáł król Sálomon Bánájáſá / ſyná Jojádowego / który śię náń tárgnął / y zábił go.
A do Abijátárá kápłáná rzekł król : Idź do Anátot / do ośiádłośći twojey / ábowiemeś mężem śmierći ; wƺákże ćię dźiś nie zábiję / gdyżeś nośił ſkrzynię Páńſką przed Dawidem / ojcem moim / á iżeś to wƺyſtko ćierpiáł / cżem był trápiony ojćiec mój.
Y wyrzućił Sálomon Abijátárá / áby nie był kápłánem Páńſkim / żeby śię wypełniło ſłowo Páńſkie / które był wyrzekł nád domem Heli w Sylo.
Tá wieść gdy przyƺłá do Joábá / ( ábowiem Joáb przeſtáwáł z Adonijáƺem / choćiáż z Abſálomem nie przeſtáwáł / )tedy ućiekł Joáb do námiotu Páńſkiego / á uchwyćił śię rogów ołtárzá.
Y oznájmiono królowi Sálomonowi / że ućiekł Joáb do námiotu Páńſkiego / á że jeſt u ołtárzá : tedy poſłáł Sálomon Bánájáſá / ſyná Jojádowego / mówiąc : Idź / zábij go.
A przyƺedłƺy Bánájáſ do námiotu Páńſkiego / rzekł do niego : Ták mówi król : Wynijdź. Który odpowiedźiáł : Nie wyjdę / ále tu umrę. Y odnióſł to Bánájáſ królowi mówiąc : Ták mówił Joáb / y ták mi odpowiedźiáł.
Y rzekł mu król : Ucżyńże / jáko mówił / á zábij go / y pogrzeb go / á odejmieƺ krew niewinną / którą wyláł Joáb / odemnie y od domu ojcá mego.
A obróći Pán krew jego ná głowę jego : ábowiem tárgnął śię ná dwóch mężów ſpráwiedliwƺych y lepƺych niżli ſám / y zábił je miecżem / á ojćiec mój Dawid nie wiedźiáł o tem : Abnerá / ſyná Nerowego / hetmáná wojſká Izráelſkiego / y Amázę / ſyná Jeterowego / hetmáná wojſká Judzkiego.
A ták wróći śię krew ich ná głowę Joábowę / y ná głowę náśieniá jego ná wieki ; lecż Dawidowi y náśieniu jego / y domowi jego / y ſtolicy jego niech będźie pokój áż ná wieki od Páná.
Szedł tedy Bánájáſ / ſyn Jojádá / á rzućiwƺy śię náń zábił go / y pogrzebiony jeſt w domu ſwym ná puƺcży.
Y poſtánowił król Bánájáſá ſyná Jojádowego / miáſto niego nád wojſkiem / á Sádoká kápłáná poſtánowił król miáſto Abijátárá.
Potem poſłáł król / y przyzwáł Semejego / y rzekł mu : Zbuduj ſobie dom w Jeruzálemie / y mieƺkáj tám / á nie wychodź ſtámtąd nigdźie ;
Bo któregobyś dniá wyƺedł á przyƺedł zá potok Cedron / wiedz wiedząc / że pewnie umrzeƺ ; krew twojá będźie ná głowę twoję.
Tedy rzekł Semej do królá : Dobre jeſt to ſłowo ; jáko mówił król / pán mój / ták ucżyni ſługá twój / Y mieƺkáł Semej w Jeruzálemie przez wiele dni.
Y ſtáło śię po trzech lát / że ućiekli dwáj ſłudzy Semejemu do Achiſá ſyná Mááchy / królá Gietſkiego / y opowiedźiáno Semejemu / mówiąc : Oto ſłudzy twoi ſą w Giet.
Przetoż wſtáwƺy Semej / y ośiodłáwƺy oſłá ſwego / jecháł do Giet / do Achiſá / áby ƺukáł ſług ſwoich ; y wróćił śię Semej y przywiódł ſługi ſwe z Giet.
Y oznájmiono Sálomonowi / że był wyjecháł Semej z Jeruzálemu do Giet / y záśię śię wróćił.
Tedy poſłáł król / y wezwáł Semejego / y rzekł mu : Izálim ćię nie poprzyśiągł przez Páná / á nie oświádcżyłem śię przed tobą / mówiąc : Któregobyśkolwiek dniá gdźie wyƺedł / wiedz wiedząc / że zápewne umrzeƺ? Y mówiłeś do mnie : Dobre to ſłowo / którem ſłyƺáł.
Przecżżeś tedy nie ſtrzegł przyśięgi Páńſkiej y przykázániá / którem ći był przykázáł?
Nádto król rzekł do Semejego : Ty wieƺ wƺyſtko złe / którego świádome jeſt ſerce twoje / coś ucżynił Dawidowi / ojcu memu / y oddáł Pán złość twoję ná głowę twoję.
Ale król Sálomon błogoſłáwiony / ſtolicá Dawidowá będźie utwierdzoná przed Pánem áż ná wieki.
A ták rozkázáł król Bánájáſowi / ſynowi Jojádowemu / który wyƺedłƺy tárgnął śię náń / y zábił go. A ták utwierdzone jeſt króleſtwo w ręce Sálomonowej.
