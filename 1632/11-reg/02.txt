A gdy śię przybliżáł cżáſ śmierći Dawidowej / rozkázáł Sálomonowi / ſynowi ſwemu / mówiąc :
Já idę w drogę wƺyſtkiej ziemi / á ty zmácniáj śię / y bądź mężem.
Záchowywáj uſtáwy Páná / Bogá twego / ábyś chodził drogámi jego / y przeſtrzegáł wyroków jego / y przykázáń jego y ſądów jego / y świádectw jego / jáko nápiſáno w zákonie Mojżeƺowym / ábyć śię ƺcżęśćiło wƺyſtko / co ſpráwowáć będzieƺ / y we wƺyſtkiem / do cżego śię obróćiƺ :
Żeby utwierdził Pán ſłowo ſwoje / które rzekł do mnie / mówiąc : Jeźli będą ſtrzedz ſynowie twoi drogi ſwej / chodząc przedemną w práwdzie / ze wƺyſtkiego ſercá ſwego / y ze wƺyſtkiej duƺy ſwojej / tedy nie będzie wytrácony po tobie mąż z ſtolicy Izráelſkiej.
A ty też wieƺ / co mi ucżynił Joáb ſyn Sárwii / co ucżynił dwom hetmánom wojſk Izráelſkich / Abnerowi / ſynowi Nerowemu / y Amázie / ſynowi Jeterowemu / że je pozábijáł / á wyláł krew jáko ná wojnie cżáſu pokoju / y zmázáł krwią jáko ná wojnie páſ ſwój rycerſki / który miáł ná biodrách ſwoich / y bóty ſwoje / które miáł ná nogách ſwoich.
Ucżyniƺ tedy według mądrośći twojej / á nie dopuśćiƺ zejść ſędziwośći jego w pokoju do grobu.
Ale nád ſynámi Bárſylái Gáláádcżyká użyjeƺ miłośierdziá / á niech jádáją weſpół z inƺymi u ſtołu twego ; ábowiem oni tákimże ſpoſobem przyƺli do mnie / gdym ućiekáł przed Abſálomem / brátem twoim.
Oto też jeſt u ćiebie Semej / ſyn Giery / ſyná Jemini / z Báhurym / który mi też złorzecżył złorzecżeniem wielkiem w on dzień / gdym ƺedł do Máhánáim ; á wƺákże záƺedł mi drogę u Jordánu / y przyśiągłem mu przez Páná / mówiąc : Nie zámorduję ćię miecżem.
Teráz jednák nie przepuƺcżáj mu tego / á iżeś jeſt mężem mądrym / będzieƺ wiedziáł / co mu máƺ ucżynić / ábyś wprowádził ſędziwość jego ze krwią do grobu.
Zátem záſnął Dawid z ojcy ſwoimi / á pogrzebiony jeſt w mieśćie Dawidowem.
A dni / których królowáł Dawid nád Izráelem / było cżterdzieśći lát. W Hebronie królowáł śiedm lát / á w Jeruzálemie królowáł trzydzieśći y trzy látá.
A ták Sálomon uśiádł ná ſtolicy Dawidá ojcá ſwego / y zmocniło śię bárdzo króleſtwo jego.
Tedy przyƺedł Adonijáƺ / ſyn Hággity / do Betſáby / mátki Sálomonowej / któremu oná rzekłá : A ſpokojneż jeſt przyjśćie twoje? A on odpowiedziáł : Spokojne.
Nádto rzekł : Mám nieco mówić z tobą. A oná rzekłá : Mów.
Tedy rzekł : Ty wieƺ / iż moje było króleſtwo / á ná mię obróćili byli wƺyſcy Izráelcżycy twárz ſwoję / ábym królowáł ; ále przenieśione jeſt króleſtwo / y doſtáło śię brátu memu ; bo mu od Páná náznácżone było.
Przetoż ćię teráz proƺę o jednę rzecż / á nie odmáwiáj mi tego. A oná mu rzekłá : Mów.
Zátem on rzekł : Mów proƺę do Sálomoná królá / ( bo wiem / żeć nie odmówi / )áby mi dáł Abiſág Sunámitkę zá żonę.
Y odpowiedziáłá Betſábá : Dobrze ; będę mówiłá o ćię z królem.
A ták ƺłá Betſábá do królá Sálomoná / áby z nim mówiłá zá Adonijáƺem ; y wſtáł król przećiwko niej / á pokłoniwƺy śię jej uśiádł ná ſtolicy ſwej ; kázáł też poſtáwić ſtolicę mátce ſwej / którá śiádłá po práwicy jego.
Y rzekłá : Proƺę ćię o jednę máłą rzecż / nie odmáwiáj mi. Y odpowiedziáł jej król : Proś mátko mojá ; ábowiem ći nie odmówię.
Tedy rzekłá : Niech będzie dáná Abiſág Sunámitká Adonijáƺowi / brátu twemu / zá żonę.
Lecż odpowiedziáł król Sálomon / y rzekł mátce ſwojej : Przecżże prośiƺ o Abiſág Sunámitkę Adonijáƺowi? uproś mu y króleſtwo / ábowiem on jeſt brátem moim ſtárƺym nád mię / á má po ſobie Abijátárá kápłáná / y Joábá / ſyná Sárwii.
Y przyśiągł król Sálomon przez Páná / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / że przećiwko duƺy ſwej mówił Adonijáƺ te ſłowá.
A teráz jáko żywy Pán / który mię utwierdził / y poſádził ná ſtolicy Dawidá ojcá mojego / y który mi zbudowáł dom / jáko obiecáł / iż dziś zábity będzie Adonijáƺ.
A ták poſłáł król Sálomon Bánájáſá / ſyná Jojádowego / który śię náń tárgnął / y zábił go.
A do Abijátárá kápłáná rzekł król : Idź do Anátot / do ośiádłośći twojej / ábowiemeś mężem śmierći ; wƺákże ćię dziś nie zábiję / gdyżeś nośił ſkrzynię Páńſką przed Dawidem / ojcem moim / á iżeś to wƺyſtko ćierpiáł / cżem był trápiony ojćiec mój.
Y wyrzućił Sálomon Abijátárá / áby nie był kápłánem Páńſkim / żeby śię wypełniło ſłowo Páńſkie / które był wyrzekł nád domem Heli w Sylo.
Tá wieść gdy przyƺłá do Joábá / ( ábowiem Joáb przeſtáwáł z Adonijáƺem / choćiáż z Abſálomem nie przeſtáwáł / )tedy ućiekł Joáb do námiotu Páńſkiego / á uchwyćił śię rogów ołtárzá.
Y oznájmiono królowi Sálomonowi / że ućiekł Joáb do námiotu Páńſkiego / á że jeſt u ołtárzá : tedy poſłáł Sálomon Bánájáſá / ſyná Jojádowego / mówiąc : Idź / zábij go.
A przyƺedłƺy Bánájáſ do námiotu Páńſkiego / rzekł do niego : Ták mówi król : Wynijdź. Który odpowiedziáł : Nie wyjdę / ále tu umrę. Y odnióſł to Bánájáſ królowi mówiąc : Ták mówił Joáb / y ták mi odpowiedziáł.
Y rzekł mu król : Ucżyńże / jáko mówił / á zábij go / y pogrzeb go / á odejmieƺ krew niewinną / którą wyláł Joáb / odemnie y od domu ojcá mego.
A obróći Pán krew jego ná głowę jego : ábowiem tárgnął śię ná dwóch mężów ſpráwiedliwƺych y lepƺych niżli ſám / y zábił je miecżem / á ojćiec mój Dawid nie wiedziáł o tem : Abnerá / ſyná Nerowego / hetmáná wojſká Izráelſkiego / y Amázę / ſyná Jeterowego / hetmáná wojſká Judzkiego.
A ták wróći śię krew ich ná głowę Joábowę / y ná głowę náśieniá jego ná wieki ; lecż Dawidowi y náśieniu jego / y domowi jego / y ſtolicy jego niech będzie pokój áż ná wieki od Páná.
Szedł tedy Bánájáſ / ſyn Jojádá / á rzućiwƺy śię náń zábił go / y pogrzebiony jeſt w domu ſwym ná puƺcży.
Y poſtánowił król Bánájáſá ſyná Jojádowego / miáſto niego nád wojſkiem / á Sádoká kápłáná poſtánowił król miáſto Abijátárá.
Potem poſłáł król / y przyzwáł Semejego / y rzekł mu : Zbuduj ſobie dom w Jeruzálemie / y mieƺkáj tám / á nie wychodź ſtámtąd nigdzie ;
Bo któregobyś dniá wyƺedł á przyƺedł zá potok Cedron / wiedz wiedząc / że pewnie umrzeƺ ; krew twojá będzie ná głowę twoję.
Tedy rzekł Semej do królá : Dobre jeſt to ſłowo ; jáko mówił król / pán mój / ták ucżyni ſługá twój / Y mieƺkáł Semej w Jeruzálemie przez wiele dni.
Y ſtáło śię po trzech lát / że ućiekli dwáj ſłudzy Semejemu do Achiſá ſyná Mááchy / królá Gietſkiego / y opowiedziáno Semejemu / mówiąc : Oto ſłudzy twoi ſą w Giet.
Przetoż wſtáwƺy Semej / y ośiodłáwƺy oſłá ſwego / jecháł do Giet / do Achiſá / áby ƺukáł ſług ſwoich ; y wróćił śię Semej y przywiódł ſługi ſwe z Giet.
Y oznájmiono Sálomonowi / że był wyjecháł Semej z Jeruzálemu do Giet / y záśię śię wróćił.
Tedy poſłáł król / y wezwáł Semejego / y rzekł mu : Izálim ćię nie poprzyśiągł przez Páná / á nie oświádcżyłem śię przed tobą / mówiąc : Któregobyśkolwiek dniá gdzie wyƺedł / wiedz wiedząc / że zápewne umrzeƺ? Y mówiłeś do mnie : Dobre to ſłowo / którem ſłyƺáł.
Przecżżeś tedy nie ſtrzegł przyśięgi Páńſkiej y przykázániá / którem ći był przykázáł?
Nádto król rzekł do Semejego : Ty wieƺ wƺyſtko złe / którego świádome jeſt ſerce twoje / coś ucżynił Dawidowi / ojcu memu / y oddáł Pán złość twoję ná głowę twoję.
Ale król Sálomon błogoſłáwiony / ſtolicá Dawidowá będzie utwierdzoná przed Pánem áż ná wieki.
A ták rozkázáł król Bánájáſowi / ſynowi Jojádowemu / który wyƺedłƺy tárgnął śię náń / y zábił go. A ták utwierdzone jeſt króleſtwo w ręce Sálomonowej.
