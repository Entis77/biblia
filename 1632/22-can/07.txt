O jáko piękne ſą nogi twoje w trzewikách / o córko kśiążęcá! Opáſániá biódr twoich ſą jáko záwieƺeniá / ręką dobrego rzemieślniká urobione.
Pępek twój jáko cżáƺá okrągłá / która nie jeſt bez nápoju ; brzuch twój jeſt jáko bróg pƺenicy oſádzony lilijámi.
Obie pierśi twoje ſą jáko dwoje bliźniąt młodych ſárniąt.
Szyjá twojá jáko wieżá z kośći ſłoniowych ; ocży twoje jáko ſádzáwki w Heſebon podle brámy Bátrábin ; noſ twój jáko wieżá ná Libánie / która pátrzy ku Dámáƺkowi.
Głowá twojá ná tobie jáko Kármel / á włoſy głowy twojey jáko ƺárłát. Król widząc ćię byłby jáko przywiązány ná gánkách ſwoich.
O jákożeś piękná / y jáko wdźięcżná / o miłośći przerozkoƺná!
Ten twój wzroſt podobny jeſt pálmie / á pierśi twoje gronom.
Rzekłem : Wſtąpię ná pálmę / dośięgnę wierzchów jey. Niechájże mi tedy będą pierśi twoje jáko groná winne / á wonność nozdrzy twoich jáko jábłek wonnych ;
A uſtá twoje jáko wino wyborne / które ná proſt bárdzo mile płynie y ſpráwuje / że mówią wárgi śpiących.
Jám jeſt miłego mego / á do mnie jeſt żądzá jego.
Przyjdź / miły mój ; wyjdźiemy ná pole / á przenocujemy we wśiách.
Ráno wſtániemy do winnic ; oglądámy / jeſli kwitnie winná máćicá / jeſli śię záwiązują gronká / kwitnąli jábłká gránátowe ; tám ći oświádcżę miłośći moje.
Polne jábłuƺká wydáły wonność ſwoję / á przede drzwiámi náƺemi ſą wƺyſtkie owoce wdźięcżne / nowe y ſtáre / którem tobie / miły mój! záchowáłá.
