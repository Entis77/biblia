Jám jeſt jáko różá Sárońſká / á lilijá przy dolinách.
Jáko lilijá między ćierniem / ták przyjáćiółká mojá między pánnámi.
Jáko jábłoń między drzewem leśnem / ták miły mój między młodźieńcámi. Prágnęłám śiedźieć w ćieniu jego / y śiedzę ; bo owoc jego ſłodki jeſt uſtom moim.
Wprowádźił mię w dom winá / májąc zá chorągiew miłość przećiwko mnie.
Ocżerſtwijćie mię temi fláƺámi / pośilćie mię temi jábłkámi ; boć omdlewám od miłośći.
Lewicá jego pod głową moją / á práwicá jego obłápiá mię.
Poprzyśięgám wáſ / córki Jeruzálemſkie! przez ſárny y łánie polne / ábyśćie nie budźiły y nie przerywáły ſnu miłego mego / dokąd nie zechce.
Głoſ miłego mego! oto on idźie ſkácżąc po tych górách / á poſkákując po tych págórkách.
Miły mój podobny jeſt ſárnie / álbo młodemu jelonkowi ; oto on ſtoi zá śćiáną náƺą / wyglądá z okien / pátrzy przez kráty.
Ozwáł śię miły mój / á rzekł mi : Wſtáń / przyjáćiółko mojá! piękná mojá! á pójdź.
Abowiem oto minęłá źimá! deƺcż przeƺedł / y przeſtáł.
Kwiátki śię ukázują ná źiemi ; cżáſ śpiewániá przyƺedł / á głoſ ſynogárlicy ſłycháć w źiemi náƺej.
Figowe drzewo wypuśćiło niedojrzáłe figi ſwoje / á máćice winne rozkwitłe / wonią wydáły ; wſtáńże przyjáćiółko mojá / piękná mojá! á pójdź.
Gołębico mojá mieƺkájącá w rozpádlinách ſkálnych / w ſkrytośćiách przykrych! okáż mi oblicże twoje / niech uſłyƺę głoſ twój ; ábowiem głoſ twój wdźięcżny / á oblicże twoje pożądáne.
Połápćie nám liƺki / liƺki máłe / które pſują winnice ; poniewáż winnice náƺe kwitną.
Miły mój jeſt mój / á jám jeſt jego / który páśie między lilijámi ;
Ażby śię okázáł ten dźień / á ćienie przeminęły. Náwróć śię / bądź podobny / miły mój! ſárnie álbo jelonkowi młodemu ná górách Beter.
