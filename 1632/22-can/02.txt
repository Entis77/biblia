Jam jeſt jáko różá Sárońſká / á lilijá przy dolinách.
Jáko lilijá miedzy ćierniem / ták przyjáćiółká mojá miedzy pánnámi.
Jáko jábłoń miedzy drzewem leśnem / ták miły mój miedzy młodźieńcámi. Prágnęłám śiedźieć w ćieniu jego / y śiedzę ; bo owoc jego ſłodki jeſt uſtom moim.
Wprowádźił mię w dom winá / májąc zá chorągiew miłość przećiwko mnie.
Ocżerſtwijćie mię temi fláƺámi / pośilćie mię temi jábłkámi ; boć omdlewám od miłośći.
Lewicá jego pod głową moją / á práwicá jego obłápiá mię.
Poprzyśięgám was / córki Jeruzálemſkie! przez ſárny y łánie polne / ábyśćie nie budźiły y nie przerywáły ſnu miłego mego / dokąd nie zechce.
Głos miłego mego! oto on idźie ſkácżąc po tych górách / á poſkákując po tych págórkách.
Miły mój podobny jeſt ſárnie / álbo młodemu jelonkowi ; oto on ſtoi zá śćiáną náƺą / wyglądá z okien / pátrzy przez kráty.
Ozwał śię miły mój / á rzekł mi : Wſtáń / przyjáćiółko mojá! piękná mojá! á pójdź.
Abowiem oto minęłá źimá! deƺcż przeƺedł / y przeſtał.
Kwiátki śię ukázują ná źiemi ; cżás śpiewánia przyƺedł / á głos ſynogárlicy ſłycháć w źiemi náƺej.
Figowe drzewo wypuśćiło niedojrzáłe figi ſwoje / á máćice winne rozkwitłe / wonią wydáły ; wſtáńże przyjáćiółko mojá / piękná mojá! á pójdź.
Gołębico mojá mieƺkájącá w rozpádlinách ſkálnych / w ſkrytośćiách przykrych! okáż mi oblicże twoje / niech uſłyƺę głos twój ; ábowiem głos twój wdźięcżny / á oblicże twoje pożądáne.
Połápćie nam liƺki / liƺki máłe / które pſują winnice ; ponieważ winnice náƺe kwitną.
Miły mój jeſt mój / á jam jeſt jego / który páśie miedzy lilijámi ;
Ażby śię okázał ten dźień / á ćienie przeminęły. Náwróć śię / bądź podobny / miły mój! ſárnie álbo jelonkowi młodemu ná górách Beter.
