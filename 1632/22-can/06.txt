Miły mój wſtąpił do ogrodu ſwego miedzy zágonki źiół wonnych / áby páſł w ogrodách / y żeby zbierał lilije.
Jam jeſt miłego mego / á miły mój jeſt mój / który páśie miedzy lilijámi.
Pięknáś ty / przyjáćiółko mojá! jáko Terſá ; pięknáś / jáko Jeruzálem ; ogromná / jáko wojſko uƺykowáne.
( Odwróć ocży twoje odemnie / gdyż mię one ſrogim cżynią. ) Włoſy twoje ſą jáko ſtádá kóz / które wychodzą z Gáláád.
Zęby twoje ſą jáko ſtádo owiec / które wychodzą z kąpieli / z których káżda miewá po dwojgu / á niepłodney niemáƺ miedzy niemi.
Skronie twoje miedzy kędźierzámi twemi ſą jáko ƺtuká jábłká gránátowego.
Acżkolwiek jeſt ƺeśćdźieśiąt żon królewſkich / á ośmdźieśiąt záłożnic / á pánien bez licżby :
Wƺákże jednáż jeſt gołębicá mojá / uprzejmá mojá / jedynácżká u mátki ſwojey / bez zmázy u rodźićielki ſwojey. Ujrzáwƺy ją córki / błogoſłáwioną ją názwáły ; tákże y żony królewſkie y záłożnice / y chwáliły ją / mówiąc :
Któráż to jeſt / co śię pokázuje jáko zorzá / piękná jáko mieśiąc / cżyſtá jáko ſłońce / ogromná jáko wojſko uƺykowáne z chorągwiámi?
Zſtąpiłám do ogrodu orzechowego / ábym oglądáłá owoce roſnące w dolinách ; ábym obácżyłá / jeſli kwitną winne máćice / á wypuƺcżájąli pącżki jábłonie gránátowe.
Niżem śię dowiedźiáłá / duƺá mojá wſádźiłá mię ná wóz przedniejƺych z ludu mego.
Náwróć śię / náwróć śię / o Sulámitko! náwróć śię / náwróć śię / niech ná ćię pátrzymy. Cóż widźićie ná Sulámitce? Widźimy / jákoby hufy wojenne.
