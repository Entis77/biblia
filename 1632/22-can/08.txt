Obyżeś był jáko brátem moim / pożywájąc pierśi mátki mojey! ábym ćię ználázłƺy ná dworzu / pocáłowáłá ćię / á nie byłá wzgárdzoná.
Prowádźiłábym ćię / y wprowádźiłá do domu mátki mojey / gdźiebyś mię ucżył ; á jábym ći dáłá pić wino przypráwne y moƺcż z jábłek moich gránátowych.
Lewicá jego pod głową moją / á práwicą ſwoją obłápiá mię.
Poprzyśięgám was / córki Jeruzálemſkie! ábyśćie nie budźiły áni przerywáły ſnu miłego mego / dokąd nie zechce.
Któráż to jeſt / co wyſtępuje z puƺcży / podpárłƺy śię miłego ſwego? Pod jábłonią wzbudźiłám ćię / tám ćię pocżęłá mátká twojá / tám ćię pocżęłá rodźićielká twojá.
Przyłóż mię jáko piecżęć ná ſerce ſwoje / jáko ſygnet do rámieniá ſwego! ábowiem miłość mocná jeſt jáko śmierć / twárdá jáko grób záwiſtná miłość ; węgle jey jáko węgle ogniſte y jáko płomień gwáłtowny.
Wody wielkie nie mogłyby zágáśić tey miłośći / áni rzeki zátopić ; choćby kto wƺyſtkę májętność domu ſwego dáł zá tákową miłość / byłby pewnie wzgárdzony.
Mámy śioſtrę málucżką / która jeƺcże nie má pierśi. Cóż ucżynimy z śioſtrą náƺą w dźień / którego o niej mowá będźie?
Jeſliże jeſt murem / zbudujmyż ná niej páłác ſrebrny ; á jeſli jeſt drzwiámi / opráwmyż ją deƺcżkámi cedrowymi.
Jám jeſt mur / á pierśi moje jáko wieże. Wtencżáſ byłám przed ocżymá jego / jáko tá / która znájduje pokój.
Winnicę miał Sálomon w Báálhámon / którą winnicę nájął ſtróżom / áby káżdy przynośił zá owoc jey tyśiąc ſrebrników.
Ale winnicá mojá / którą mam / jeſt przedemną. Miej ſobie tyśiąc ſrebrników / Sálomonie / á dwieśćie ći / którzy ſtrzegą owocu jey.
O ty / która mieƺkáƺ w ogrodách! przyjáćiele ſłucháją głoſu twego ; ozwijże mi śię!
Poſpieƺ śię / miły mój! á bądź podobnym ſárnie / álbo młodemu jelonkowi ná górách źiół wonnych.
