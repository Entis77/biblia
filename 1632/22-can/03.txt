Ná łożu mojem w nocy ƺukáłám tego / którego miłuje duƺá mojá ; ƺukáłám go / álem go nie ználázłá.
Już tedy wſtánę / á obieżę miáſto ; po rynkách y po ulicách będę ƺukáć tego / którego miłuje duƺá mojá ; ƺukáłám go / álem go nie ználázłá.
Nátráfili mię ſtróżowie / którzy chodzili po mieśćie ; y ſpytáłám : Widzieliżeśćie tego / którego miłuje duƺá mojá?
A gdym málucżko odeƺłá od nich / zárázem ználázłá tego / którego miłuje duƺá mojá. Uchwyćiłám śię go / á nie puƺcżę go / áż go wprowádzę do domu mátki mojej / y do pokoju rodzićielki mojej.
Poprzyśięgám wáſ / córki Jeruzálemſkie! przez ſárny y łánie polne / ábyśćie nie budziły áni przerywáły ſnu miłego mojego / dokądby nie zechćiáł.
Któráż to jeſt / co wyſtępuje z puƺcży jáko ſłupy dymu / okurzoná będąc myrrą y kádzidłem drożƺem nád wƺeláki proƺek áptekárſki?
Oto łoże Sálomonowe / około którego ſtoi ƺeśćdzieśiąt mocárzów z mocárzów Izráelſkich.
Wƺyſcy ći trzymáją miecż / będąc wyćwicżeni do bitwy ; káżdy z nich má ſwój miecż przy boku ſwym dlá ſtráchu nocnego.
Páłác ſobie król Sálomon wyſtáwił z drzewá Libáńſkiego.
Słupy jego pocżynił ſrebrne / á pokłád jego złoty / podniebienie ƺárłátne / á wewnątrz uſłány jeſt miłośćią córek Jerozolimſkich.
Wynijdźćie / córki Syońſkie! á oglądájćie królá Sálomoná w koronie / którą go ukoronowáłá mátká jego w dzień zrękowin jego / y w dzień weſelá ſercá jego.
