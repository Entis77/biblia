Przyƺedłem do ogrodu mego / śioſtro mojá / oblubienico mojá! zbierám myrrę moję z rzecżámi wonnemi mojemi ; jem pláſtr mój z miodem moim / piję wino moje z mlekiem mojem. Jedzćież / przyjáćiele! pijćie / á pijćie doſtátkiem / mili moi!
Jáć śpię ; ále ſerce moje cżuje / y ſłyƺy głoſ miłego mego / kołácżącego y mówiącego : Otwórz mi / śioſtro mojá / przyjáćiółko mojá / gołębico mojá / uprzejmá mojá! ábowiem głowá mojá pełná jeſt roſy / á kędźierze moje kropli nocnych.
Y odpowiedźiáłám : Zewlekłám ſuknię moję / jákoż ją oblec mám? umyłám nogi moje / jákoż je zmázáć mám?
Miły mój śćiągnął rękę ſwoję dźiurą / á wnętrznośći moje wzruƺyły śię we mnie.
Y wſtáłám / ábym otworzyłá miłemu memu / á oto z rąk mych kápáłá myrrá / á z pálców moich myrrá ćiekącá ná rękojeść záwory.
Otworzyłám miłemu memu ; ále miły mój już był odƺedł y minął. Omdláłám byłá ná głoſ jego ; ƺukáłám go / álem go nie ználázłá ; wołáłám go / ále mi śię nieozwáł.
Nátráfili mię ſtróże / co chodzą po mieśćie ; ubili mię / zránili mię / wźięli y płáƺcż mój ze mnie ſtróże murów.
Poprzyśięgám wáſ / córki Jeruzálemſkie! Jeślibyśćie ználázły miłego mego / ábyśćie mu powiedźiáły / żem od miłośći záchorowáłá.
Cóż má miły twój nád innych miłych / o nájpiękniejƺá między niewiáſtámi? co má miły twój nád innych miłych / że náſ ták poprzyśięgáƺ?
Miły mój biáły y rumiány / y zácniejƺy nád innych dźieśięć tyśięcy.
Głowá jego jáko bryłá ƺcżerego złotá ; włoſy jego kędźierzáwe / cżárne jáko kruk ;
Ocży jego jáko gołębicy nád ſtrumieniámi wody / jáko umyte w mleku / ſtojące w mierze ſwojej ;
Policżki jego jáko zágonki źiół wonnych / jáko kwiátki wonnych rzecży ; wárgi jego jáko lilije wypuƺcżájące myrrę ćiekącą ;
Ręce jego jáko pierśćienie złote / oſádzone drogim kámieniem / hijácyntem ; brzuch jego jáko glánc kośći ſłoniowej / ſáfirem oſádzonej ;
Golenie jego jáko ſłupy mármurowe / poſtáwione ná podſtáwkách złotá wybornego ; oblicże jego jáko Libán / wyborne jáko cedry ;
Uſtá jego náder ſłodkie / á wƺyſtek jeſt pożądány. Tákić jeſt miły mój / y táki przyjáćiel mój / o córki Jeruzálemſkie!
Gdźież poƺedł miły twój / o nájpiękniejƺá między niewiáſtámi? Gdźie śię obróćił miły twój? á ƺukáć go będźiemy z tobą.
