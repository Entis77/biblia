Przyƺedłem do ogrodu mego / śioſtro mojá / oblubienico mojá! zbierám myrrę moję z rzecżámi wonnemi mojemi ; jem pláſtr mój z miodem moim / piję wino moje z mlekiem mojim. Jedzćież / przyjáćiele! pijćie / á pijćie doſtátkiem / mili moi!
Jáć śpię ; ále ſerce moje cżuje / y ſłyƺy głos miłego mego / kołácżącego y mówiącego : Otwórz mi / śioſtro mojá / przyjáćiółko mojá / gołębico mojá / uprzejmá mojá! ábowiem głowá mojá pełna jeſt roſy / á kędźierze moje kropli nocnych.
Y odpowiedźiáłám : Zewlekłám ſuknię moję / jákoż ją oblec mam? umyłám nogi moje / jákoż je zmázáć mam?
Miły mój śćiągnął rękę ſwoję dźiurą / á wnętrznośći moje wzruƺyły śię we mnie.
Y wſtáłám / ábym otworzyłá miłemu memu / á oto z rąk mych kápáłá myrrá / á z pálców mojich myrrá ćiekącá ná rękojeść záwory.
Otworzyłám miłemu memu ; ále miły mój już był odƺedł y minął. Omdláłám byłá ná głos jego ; ƺukáłám go / álem go nie ználázłá ; wołáłám go / ále mi śię nieozwał.
Nátráfili mię ſtróże / co chodzą po mieśćie ; ubili mię / zránili mię / wźięli y płáƺcż mój ze mnie ſtróże murów.
Poprzyśięgám was / córki Jeruzálemſkie! Jeślibyśćie ználázły miłego mego / ábyśćie mu powiedźiáły / żem od miłośći záchorowáłá.
Cóż ma miły twój nád innych miłych / o nájpiękniejƺá miedzy niewiáſtámi? co ma miły twój nád innych miłych / że nas ták poprzyśięgáƺ?
Miły mój biáły y rumiány / y zácniejƺy nád innych dźieśięć tyśięcy.
Głowá jego jáko bryłá ƺcżerego złotá ; włoſy jego kędźierzáwe / cżárne jáko kruk ;
Ocży jego jáko gołębicy nád ſtrumieniámi wody / jáko umyte w mleku / ſtojące w mierze ſwojey ;
Policżki jego jáko zágonki źiół wonnych / jáko kwiátki wonnych rzecży ; wárgi jego jáko lilije wypuƺcżájące myrrę ćiekącą ;
Ręce jego jáko pierśćienie złote / oſádzone drogim kámieniem / hijácyntem ; brzuch jego jáko glánc kośći ſłoniowej / ſáfirem oſádzoney ;
Golenie jego jáko ſłupy mármurowe / poſtáwione ná podſtáwkách złotá wybornego ; oblicże jego jáko Libán / wyborne jáko cedry ;
Uſtá jego náder ſłodkie / á wƺyſtek jeſt pożądány. Tákić jeſt miły mój / y táki przyjáćiel mój / o córki Jeruzálemſkie!
Gdźież poƺedł miły twój / o nájpiękniejƺá miedzy niewiáſtámi? Gdźie śię obróćił miły twój? á ƺukáć go będźiemy z tobą.
