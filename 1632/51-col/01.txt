Páweł / Apoſtoł JEzuſá CHryſtuſá przez wolę Bożą / y Tymoteuƺ brát /
Tym / którzy ſą w Kolośiech / świętym / y wiernym bráćiom w CHryſtuśie JEzuśie. Łáſká wám y pokój niech będźie od Bogá / Ojcá náƺego / y od Páná JEzuſá CHryſtuſá.
Dźiękujemy Bogu y Ojcu Páná náƺego JEzuſá CHryſtuſá / záwƺe modląc śię zá wáſ /
Uſłyƺáwƺy o wierze wáƺej w CHryſtuśie JEzuśie / y o miłośći przećiwko wƺyſtkim świętym /
Dlá nádźiei onej wám odłożonej w niebieśiech / o którejśćie przedtem ſłyƺeli przez ſłowo práwdy / to jeſt Ewángielii /
Którá przyƺłá do wáſ jáko y ná wƺyſtek świát / y przynośi owoc / jáko y u wáſ / od onego dniá / któregośćie uſłyƺeli y poználi łáſkę Bożą w práwdźie.
Jákośćie śię też náucżyli od Epáfry / miłego ſpółſługi náƺego / który jeſt wiernym ſługą CHryſtuſowym dlá wáſ ;
Który też oznájmił nám miłość wáƺę w duchu.
Dlátego y my od onego dniá / któregośmy to uſłyƺeli / nie przeſtájemy śię zá wáſ modlić y prośić / ábyśćie byli nápełnieni znájomośćią woli jego we wƺelkiej mądrośći y w wyrozumieniu duchownem.
Abyśćie chodźili przyſtojnie przed Pánem ku wƺelkiemu jego upodobániu / w káżdym ucżynku dobrym owoc przynoƺąc / y roſnąc w znájomośći Bożej /
Wƺelką mocą umocnieni będąc według chwálebnej mocy jego / ku wƺelkiej ćierpliwośći y nieſkwápliwośći z rádośćią /
Dźiękując Ojcu / który náſ godnymi ucżynił / ábyśmy byli ucżeſnikámi dźiedźictwá świętych w świátłośći ;
Który náſ wyrwáł z mocy ćiemnośći / y przenióſł do króleſtwá Syná ſwego miłego /
W którym mámy odkupienie przez krew jego / to jeſt / odpuƺcżenie grzechów ;
Który jeſt obrázem Bogá niewidźiálnego / y pierworodny wƺyſtkich rzecży ſtworzonych.
Abowiem przezeń ſtworzone ſą wƺyſtkie rzecży / które ſą ná niebie y ná źiemi / widźiálne y niewidźiálne / bądź trony / bądź páńſtwá / bądź kśięſtwá / bądź zwierzchnośći / wƺyſtko przezeń y dlá niego ſtworzone jeſt.
A on jeſt przed wƺyſtkimi / y wƺyſtko w nim ſtoi.
On też jeſt głową ćiáłá / to jeſt / kośćiołá / który jeſt pocżątkiem y pierworodnym z umárłych / áby on między wƺyſtkimi przodkowáł ;
Poniewáż śię upodobáło Ojcu / áby w nim wƺyſtká zupełność mieƺkáłá /
Y żeby przezeń z ſobą pojednáł wƺyſtko / ucżyniwƺy pokój przez krew krzyżá jego ; przezeń / mówię / ták to / co jeſt ná źiemi / jáko y to / co jeſt ná niebieśiech.
Y wáſ / którzyśćie niekiedy byli oddáleni y nieprzyjáćiele umyſłem w złośliwych ucżynkách / teráz też pojednáł.
Ciáłem włáſnem ſwojim przez śmierć / áby wáſ wyſtáwił świętymi y niepokálánemi y bez nágány przed oblicżem ſwojim ;
Jeźli tylko trwáćie w wierze ugruntowáni y utwierdzeni / á nieporuƺeni od nádźiei Ewángielii / którąśćie ſłyƺeli / którá jeſt kázáno wƺelkiemu ſtworzeniu / które jeſt pod niebem / której já Páweł ſtáłem śię ſługą ;
Który śię teráz ráduję w doległośćiách moich dlá wáſ / y dopełniám oſtátków ućiſków CHryſtuſowych ná ćiele mojem zá ćiáło jego / które jeſt kośćiół.
Któregom śię já ſtáł ſługą według dáru Bożego / który mi jeſt dány dlá wáſ / ábym wypełnił ſłowo Boże /
To jeſt / tájemnicę onę / którá byłá zákrytá od wieków y od rodzájów / ále teráz objáwioná jeſt świętym jego /
Którym chćiáł Bóg oznájmić / jákie jeſt bogáctwo tej tájemnicy chwálebnej między pogánámi / którá jeſt CHryſtuſ między wámi / nádźiejá oná chwáły ;
Którego my opowiádámy / nápominájąc káżdego cżłowieká / y ucżąc káżdego cżłowieká we wƺelkiej mądrośći / ábyśmy wyſtáwili káżdego cżłowieká doſkonáłym w CHryſtuśie JEzuśie ;
W cżem też prácuję / bojując według ſkutecżnej mocy jego / którá we mnie dźieło ſwoje potężnie ſpráwuje.
