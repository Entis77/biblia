Pánowie! ſpráwiedliwie y ſłuƺnie śię z ſługámi obchodźćie / wiedząc / iż y wy Páná máćie w niebieśiech.
W modlitwách trwájćie / cżując w nich z dziękowániem /
Modląc śię ſpołecżnie y zá námi / áby nám Bóg otworzył drzwi ſłowá / żebyśmy mówili o tájemnicy Chryſtuſowej / dlá której też jeſtem związány.
Abym ją objáwił / jáko mi śię godzi mówić.
Mądrze chodźćie przed obcymi / cżáſ odkupując.
Mowá wáƺá niech záwƺe będzie przyjemná / ſolą okráƺoná / ábyśćie wiedzieli / jákobyśćie káżdemu z oſobná odpowiedzieć mieli.
O wƺyſtkiem / co śię ze mną dzieje / oznájmi wám Tychykuſ / miły brát y wierny ſługá / y ſpółſługá w Pánu /
Któregom poſłáł do wáſ dlátego ſámego / áby śię wywiedziáł / co śię z wámi dzieje / y poćieƺył ſercá wáƺe ;
Z Onezymem / wiernym á miłym brátem / który jeſt z wáſ ; ći wám wƺyſtko oznájmią / co śię tu dzieje.
Pozdráwiá wáſ Aryſtárchuſ / ſpółwięzień mój / y Márek / śioſtrzeniec Bárnábáƺowy / ( o któremeśćie wzięli rozkázánie : Jeźliby do wáſ przyƺedł / przyjmijćież go. )
Y Jezuſ / którego zowią Juſtem / którzy ſą z obrzeſki. Ci tylko ſą pomocnikámi moimi w króleſtwie Bożem / którzy byli poćiechą moją.
Pozdráwiá wáſ Epáfráſ / który z wáſ jeſt ſługá Chryſtuſowy / który záwƺe bojuje zá wáſ w modlitwách / ábyśćie ſtáli doſkonáłymi y zupełnymi we wƺelkiej woli Bożej.
Bo mu dáję świádectwo / iż gorliwą miłość má przećiwko wám / y przećiwko tym / którzy ſą w Láodycei / y którzy ſą w Hijerápolu.
Pozdráwiá wáſ Łukáƺ / lekárz miły / tákże y Demáſ.
Pozdrówćie bráći / którzy ſą w Láodycei / y Nymfáſá / y zbór / który jeſt w domu jego.
A gdy ten liſt u wáſ przecżytány będzie / ſpráwćie to / áby też był w Láodyceńſkim zborze przecżytány ; á ten / który jeſt piſány z Láodycei / y wy też przecżytájćie /
A powiedzćie Archipowi : Pátrzáj ná to poſługiwánie / któreś przyjął w Pánu / ábyś je wypełnił.
Pozdrowienie ręką moją Páwłową. Pámiętájćie ná więzienie moje. Łáſká niech będzie z wámi. Amen.
