Abowiem chcę / ábyśćie wiedźieli / jáko wielką trudność mam o was y o tych / którzy ſą w Láodycei / y którzykolwiek nie widźieli oblicżá mego w ćiele ;
Aby poćieƺone były ſercá ich / będąc ſpojone miłośćią / á to ku wƺelkiemu bogáctwu zupełnego y pewnego wyrozumieniá / ku poznániu tájemnicy Bogá y Ojcá y CHryſtuſá /
W którym ſkryte ſą wƺyſtkie ſkárby mądrośći y umiejętnośći.
A toć mówię / áby was nikt fáłƺywemi dowodámi nie oƺukáł przez wyſtáwną mowę.
Bo ácżkolwiek nie jeſtem obecny ćiáłem / ále duchem jeſtem z wámi / rádując śię y widząc porządek wáƺ y utwierdzenie wiáry wáƺej w CHryſtuſá ;
Przetoż jákośćie przyjęli Páná JEzuſá CHryſtuſá / ták w nim chodźćie /
Będąc wkorzenieni y wybudowáni ná nim / y utwierdzeni w wierze / jákośćie śię náucżyli / obfitując w niej z dźiękowániem.
Pátrzćież / żeby was kto ſobie w korzyść nie obróćił przez filozofiję y przez próżne oƺukánie / ucżąc według uſtáwy ludzkiej / według żywiołów świátá / á nie według CHryſtuſá ;
Gdyż w nim mieƺká wƺyſtká zupełność bóſtwá ćieleśnie.
Y jeſteśćie w nim dopełnieni / który jeſt głową wƺelkiego kśięſtwá y zwierzchnośći :
W którym y obrzezáni jeſteśćie obrzezką nie ręką ucżynioną / to jeſt / zewlekłƺy ćiáło wƺyſtkich grzechów przez obrzezkę CHryſtuſową.
Pogrzebieni z nim będąc w chrzćie ; w którymeśćie też ſpołem z nim wzbudzeni przez wiárę / którą ſpráwuje Bóg / który go wzbudźił od umárłych.
Y was / gdyśćie byli umárłymi w grzechách y nieobrzeſce ćiáłá wáƺego / weſpół z nim ożywił / odpuśćiwƺy wám wƺyſtkie grzechy /
Zmázáwƺy on / który był przećiwko nám / cyrográf w uſtáwách záleżący / który nám był przećiwny / znióſł go z pośrodku / przybiwƺy go do krzyżá ;
Y złupiwƺy kśięſtwá y mocy / wiódł je ná podźiw / jáwnie tryumfując z nich ſám przez śię.
Niechájże was tedy nikt nie ſądźi dla pokármu / álbo dla nápoju ; álbo z ſtrony świętá / álbo nowiu mieśiącá / álbo ſábátów /
Które ſą ćieniem rzecży przyƺłych / ále práwdą jeſt ćiáło CHryſtuſowe.
Niecháj was nikt wygránego zákłádu nie oſądzá / który śię dobrowolnie w pokorę y w ſłużbę Aniołów / których nie widźiáł / wdáje / próżno śię nádymájąc z umyſłu ćiáłá ſwego.
A nie trzymájąc śię głowy CHryſtuſá / z którego wƺyſtko ćiáło przez ſtáwy y związániá pośiłek biorąc / y weſpół ſpojone będąc / rośnie wzroſtem Bożym.
Jeſliśćie tedy umárli z CHryſtuſem żywiołom świátá tego / przecżże śię / jákobyśćie jeƺcże żyli świátu / uſtáwámi báwićie?
Mówią niektórzy : Nie dotykáj śię / áni koƺtuj / áni rucháj ;
Co wƺyſtko káźi śię ſámem używániem / według przykázáń y náuk ludzkich ;
Które máją kƺtáłt mądrośći w nábożeńſtwie dobrowolnie obránem / y w pokorze / y w niefolgowániu ćiáłu ; wƺákże nie máją żádney wági / tylko do náſyceniá ćiáłá ſłużą.
