Abowiem chcę / ábyśćie wiedzieli / jáko wielką trudność mám o wáſ y o tych / którzy ſą w Láodycei / y którzykolwiek nie widzieli oblicżá mego w ćiele ;
Aby poćieƺone były ſercá ich / będąc ſpojone miłośćią / á to ku wƺelkiemu bogáctwu zupełnego y pewnego wyrozumieniá / ku poznániu tájemnicy Bogá y Ojcá y Chryſtuſá /
W którym ſkryte ſą wƺyſtkie ſkárby mądrośći y umiejętnośći.
A toć mówię / áby wáſ nikt fáłƺywemi dowodámi nie oƺukáł przez wyſtáwną mowę.
Bo ácżkolwiek nie jeſtem obecny ćiáłem / ále duchem jeſtem z wámi / rádując śię y widząc porządek wáƺ y utwierdzenie wiáry wáƺej w Chryſtuſá ;
Przetoż jákośćie przyjęli Páná Jezuſá Chryſtuſá / ták w nim chodźćie /
Będąc wkorzenieni y wybudowáni ná nim / y utwierdzeni w wierze / jákośćie śię náucżyli / obfitując w niej z dziękowániem.
Pátrzćież / żeby wáſ kto ſobie w korzyść nie obróćił przez filozofiję y przez próżne oƺukánie / ucżąc według uſtáwy ludzkiej / według żywiołów świátá / á nie według Chryſtuſá ;
Gdyż w nim mieƺká wƺyſtká zupełność bóſtwá ćieleśnie.
Y jeſteśćie w nim dopełnieni / który jeſt głową wƺelkiego kśięſtwá y zwierzchnośći :
W którym y obrzezáni jeſteśćie obrzezką nie ręką ucżynioną / to jeſt / zewlekłƺy ćiáło wƺyſtkich grzechów przez obrzezkę Chryſtuſową.
Pogrzebieni z nim będąc w chrzćie ; w którymeśćie też ſpołem z nim wzbudzeni przez wiárę / którą ſpráwuje Bóg / który go wzbudził od umárłych.
Y wáſ / gdyśćie byli umárłymi w grzechách y nieobrzeſce ćiáłá wáƺego / weſpół z nim ożywił / odpuśćiwƺy wám wƺyſtkie grzechy /
Zmázáwƺy on / który był przećiwko nám / cyrográf w uſtáwách záleżący / który nám był przećiwny / znióſł go z pośrodku / przybiwƺy go do krzyżá ;
Y złupiwƺy kśięſtwá y mocy / wiódł je ná podziw / jáwnie tryumfując z nich ſám przez śię.
Niechájże wáſ tedy nikt nie ſądzi dlá pokármu / álbo dlá nápoju ; álbo z ſtrony świętá / álbo nowiu mieśiącá / álbo ſábátów /
Które ſą ćieniem rzecży przyƺłych / ále práwdą jeſt ćiáło Chryſtuſowe.
Niecháj wáſ nikt wygránego zákłádu nie oſądzá / który śię dobrowolnie w pokorę y w ſłużbę Aniołów / których nie widziáł / wdáje / próżno śię nádymájąc z umyſłu ćiáłá ſwego.
A nie trzymájąc śię głowy Chryſtuſá / z którego wƺyſtko ćiáło przez ſtáwy y związániá pośiłek biorąc / y weſpół ſpojone będąc / rośnie wzroſtem Bożym.
Jeźliśćie tedy umárli z Chryſtuſem żywiołom świátá tego / przecżże śię / jákobyśćie jeƺcże żyli świátu / uſtáwámi báwićie?
Mówią niektórzy : Nie dotykáj śię / áni koƺtuj / áni rucháj ;
Co wƺyſtko kázi śię ſámem używániem / według przykázáń y náuk ludzkich ;
Które máją kƺtáłt mądrośći w nábożeńſtwie dobrowolnie obránem / y w pokorze / y w niefolgowániu ćiáłu ; wƺákże nie máją żádnej wági / tylko do náſyceniá ćiáłá ſłużą.
