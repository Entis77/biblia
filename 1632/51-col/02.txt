Abowiem chcę ábyśćie wiedźieli / jáko wielką trudność mam o was / y o tych którzy ſą w Láodycey ; y którzykolwiek nie widźieli oblicżá mego w ćiele :
Aby poćieƺone były ſercá ich / będąc ſpojone miłośćią ; á to ku wƺelkiemu bogáctwu zupełnego y pewnego wyrozumienia / ku poznániu tájemnice Bogá y Ojcá y CHryſtuſá :
W którym ſkryte ſą wƺyſtkie ſkárby mądrośći y umiejętnośći.
A toć mówię áby was nikt fałƺywymi dowodámi nie oƺukał / przez wyſtáwną mowę.
Bo ácżkolwiek nie jeſtem obecny ćiáłem / ále duchem jeſtem z wámi / rádując śię y widząc porządek wáƺ / y utwirdzenie wiáry wáƺey w CHryſtuſá.
Przetoż jákośćie przyjęli PANA JEzuſá CHryſtuſá / ták w nim chodźćie :
<i>Będąc</i> wkorzenieni y wybudowáni ná nim / y utwirdzeni w wierze ; jákośćie śię náucżyli / obfitując w niey z dźiękowániem.
Pátrzćież żeby was kto <i>ſobie</i> w korzyść nie obróćił przez Filoſofią / y przez prózne oƺukánie ; <i>ucżąc</i> według uſtáwy ludzkiey / według żywiołów świátá / á nie według CHryſtuſá.
Gdyż w nim mieƺka wƺyſtká zupełność bóſtwá ćieleśnie :
Y jeſteśćie w nim dopełnieni / który jeſt głową wƺelkiego kśięſtwá y zwierzchnośći.
W którym y obrzezáni jeſteśćie / obrzezką nie ręką ucżynioną ; <i>to jeſt,</i> zewlekƺy ćiáło wƺyſtkich grzechów przez obrzezkę CHryſtuſowę :
Pogrzebieni z nim będąc we chrzćie ; w którymeśćie też ſpołem z nim wzbudzeni przez wiárę / którą ſpráwuje Bóg / który go wzbudźił od umárłych.
Y was gdyśćie byli umárłymi w grzechách y w nieobrzezce ćiáłá wáƺego / weſpół z nim ożywił ; odpuśćiwƺy wam wƺyſtkie grzechy /
Zmázawƺy on który był przećiwko nam Cyrograf w uſtáwách <i>záleżący,</i> który nam był przećiwny / znióſł go z pośrzodku / przybiwƺy go do krzyżá :
Y złupiwƺy kśięſtwá y mocy / wiódł je ná podźiw / jáwnie tryumfując z nich ſam przez śię.
Niechajże was tedy nikt nie ſądźi dla pokármu / álbo dla napoju / álbo z ſtrony Świętá / álbo Nowiu mieśiącá / álbo Sábbátów :
Które ſą ćieniem rzecży przyƺłych ; ále <i>prawdą</i> jeſt ćiáło CHryſtuſowe.
Niechaj was nikt wygránego zákłádu nie oſądza / który śię dobrowolnie w pokorę y w ſłużbę Anjołów / których nie widźiał / wdawa / prózno śię nádymájąc z umyſłu ćiáłá ſwego.
A nie trzymájąc śię głowy <i>CHryſtuſá,</i> z którego wƺyſtko ćiáło przez ſtáwy y związánia pośiłek biorąc / y weſpół ſpojone będąc / roście wzroſtem Bożym.
Jeſliśćie tedy umárli z CHryſtuſem żywiołom świátá tego ; przecżże śię / jákobyśćie jeƺcże żyli świátu / uſtáwámi báwićie?
( <i>Mówią niektórzy</i> ; Nie dotykaj śię / áni koƺtuj / áni ruchaj :
Co wƺyſtko káźi śię ſámym używániem ) Według przykazań y náuk ludzkich ;
Które máją kƺtałt mądrośći w nabożeńſtwie dobrowolnie obránym / y w pokorze / y w niefolgowániu ćiáłu ; wƺákże nie máją żadney wagi / tylko do náſycenia ćiáłá ſłużą.
