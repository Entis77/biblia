A ták jeźliśćie powſtáli z Chryſtuſem / tego / co jeſt w górze / ƺukájćie / gdzie Chryſtuſ ná práwicy Bożej śiedzi ;
O tem / co jeſt w górze / myślćie / nie o tem / co jeſt ná ziemi.
Abowiemeśćie umárli / y żywot wáƺ ſkryty jeſt z Chryſtuſem w Bogu.
Ale gdy śię Chryſtuſ on żywot náƺ / pokáże / tedy y wy z nim okáżećie śię w chwále.
Umártwiájćież tedy cżłonki wáƺe / które ſą ná ziemi ; wƺetecżeńſtwo / niecżyſtość / námiętność / złą pożądliwość / y łákomſtwo / które jeſt báłwochwálſtwem /
Dlá których rzecży przychodzi gniew Boży ná ſyny odporne.
W którycheśćie y wy niekiedy chodzili / gdyśćie żyli w nich.
Lecż teráz złóżćie y wy to wƺyſtko / gniew / zápálcżywość / złość / bluźnierſtwo / y ſproſną mowę z uſt wáƺych.
Nie kłámćie jedni przećiwko drugim / gdyżeśćie zewlekli cżłowieká ſtárego z ucżynkámi jego /
A oblekliśćie nowego tego / który śię odnáwiá w znájomość / podług obrázu tego / który go ſtworzył.
Gdzie nie máƺ Greká y Żydá / obrzeſki y nieobrzeſki / cudzoziemcá y Tátárzyná / niewolniká y wolnego ; ále wƺyſtko y we wƺyſtkich Chryſtuſ.
Przetoż przyoblecżćie jáko wybráni Boży / święći / y umiłowáni / wnętrznośći miłośierdziá / dobrotliwość / pokorę / ćichość / ćierpliwość /
Znáƺájąc jedni drugich / y odpuƺcżájąc ſobie wzájemnie / jeźli má kto przećiw komu ſkárgę : jáko y Chryſtuſ odpuśćił wám / ták y wy.
A nádto wƺyſtko ( przyoblecżćie ) miłość / którá jeſt związką doſkonáłośći.
A pokój Boży niech rząd prowádzi w ſercách wáƺych / do któregośćie też powołáni w jedno ćiáło ; á bądźćie wdzięcżnymi.
Słowo Chryſtuſowe niecháj mieƺká w wáſ obfićie ze wƺelką mądrośćią / náucżájąc y nápominájąc ſámych śiebie przez pſálmy y hymny / y pieśni duchowne / wdzięcżnie śpiewájąc w ſercách wáƺych Pánu.
A wƺyſtko / cokolwiek cżynićie w ſłowie álbo w ucżynku / wƺyſtko cżyńćie w imieniu Páná Jezuſá / dziękując Bogu y Ojcu przezeń.
Żony! bądźćie poddáne mężom ſwym / ták jáko przyſtoi w Pánu.
Mężowie! miłujćie żony wáƺe / á nie bądźćie ſurowymi przećiwko nim.
Dziátki! poſłuƺne bądźćie rodzicom we wƺyſtkiem ; ábowiem śię to podobá Pánu.
Ojcowie! nie pobudzájćie do gniewu dzieći wáƺych / áby ſercá nie tráćiły.
Słudzy! poſłuƺni bądźćie we wƺyſtkiem pánom ćieleſnym / nie ſłużąc ná oko jáko ći / co śię ludziom podobáć chcą / ále w ƺcżerośći ſercá / bojąc śię Bogá.
A wƺyſtko / cokolwiek cżynićie / z duƺy cżyńćie / jáko Pánu / á nie ludziom.
Wiedząc / iż od Páná weźmiećie zápłátę dziedzictwá ; ábowiem Pánu Chryſtuſowi ſłużyćie.
A ten / co krzywdę cżyni / odnieśie zápłátę ukrzywdzeniá / á niemáƺći względu ná oſoby u Bogá.
