A ták jeſliśćie powſtáli z CHryſtuſem / tego / co jeſt w górze / ƺukájćie / gdźie CHryſtuſ ná práwicy Bożej śiedźi ;
O tem / co jeſt w górze / myślćie / nie o tem / co jeſt ná źiemi.
Abowiemeśćie umárli / y żywot wáƺ ſkryty jeſt z CHryſtuſem w Bogu.
Ale gdy śię CHryſtuſ on żywot náƺ / pokáże / tedy y wy z nim okáżećie śię w chwále.
Umártwiájćież tedy cżłonki wáƺe / które ſą ná źiemi ; wƺetecżeńſtwo / niecżyſtość / námiętność / złą pożądliwość / y łákomſtwo / które jeſt báłwochwálſtwem /
Dla których rzecży przychodźi gniew Boży ná ſyny odporne.
W którycheśćie y wy niekiedy chodźili / gdyśćie żyli w nich.
Lecż teraz złóżćie y wy to wƺyſtko / gniew / zápálcżywość / złość / bluźnierſtwo / y ſproſną mowę z uſt wáƺych.
Nie kłámćie jedni przećiwko drugim / gdyżeśćie zewlekli cżłowieká ſtárego z ucżynkámi jego /
A oblekliśćie nowego tego / który śię odnáwiá w znájomość / podług obrázu tego / który go ſtworzył.
Gdźie nie máƺ Greká y Żydá / obrzeſki y nieobrzeſki / cudzoźiemcá y Tátárzyná / niewolniká y wolnego ; ále wƺyſtko y we wƺyſtkich CHryſtuſ.
Przetoż przyoblecżćie jáko wybráni Boży / święći / y umiłowáni / wnętrznośći miłośierdźiá / dobrotliwość / pokorę / ćichość / ćierpliwość /
Znáƺájąc jedni drugich / y odpuƺcżájąc ſobie wzájemnie / jeſli má kto przećiw komu ſkárgę : jáko y CHryſtuſ odpuśćił wám / ták y wy.
A nádto wƺyſtko ( przyoblecżćie ) miłość / która jeſt związką doſkonáłośći.
A pokój Boży niech rząd prowádźi w ſercách wáƺych / do któregośćie też powołáni w jedno ćiáło ; á bądźćie wdźięcżnymi.
Słowo CHryſtuſowe niecháj mieƺká w was obfićie ze wƺelką mądrośćią / náucżájąc y nápominájąc ſámych śiebie przez pſálmy y hymny / y pieśni duchowne / wdźięcżnie śpiewájąc w ſercách wáƺych Pánu.
A wƺyſtko / cokolwiek cżynićie w ſłowie álbo w ucżynku / wƺyſtko cżyńćie w imieniu Páná JEzuſá / dźiękując Bogu y Ojcu przezeń.
Żony! bądźćie poddáne mężom ſwym / ták jáko przyſtoi w Pánu.
Mężowie! miłujćie żony wáƺe / á nie bądźćie ſurowymi przećiwko nim.
Dźiátki! poſłuƺne bądźćie rodźicom we wƺyſtkiem ; ábowiem śię to podobá Pánu.
Ojcowie! nie pobudzájćie do gniewu dźieći wáƺych / áby ſercá nie tráćiły.
Słudzy! poſłuƺni bądźćie we wƺyſtkiem pánom ćieleſnym / nie ſłużąc ná oko jáko ći / co śię ludźiom podobáć chcą / ále w ƺcżerośći ſercá / bojąc śię Bogá.
A wƺyſtko / cokolwiek cżynićie / z duƺy cżyńćie / jáko Pánu / á nie ludźiom.
Wiedząc / iż od Páná weźmiećie zápłátę dźiedźictwá ; ábowiem Pánu CHryſtuſowi ſłużyćie.
A ten / co krzywdę cżyni / odnieśie zápłátę ukrzywdzeniá / á niemáƺći względu ná oſoby u Bogá.
