Ná ſtráży ſwej ſtáć będę / y ſtánę ná báƺćie / wyglądájąc / ábym obácżył / co będźie Bóg mówił / cobym miał odpowiedźieć po kárániu mojim.
Tedy mi odpowiedźiał Pán / mówiąc : Nápiƺ widzenie / á nápiƺ rzetelnie ná táblicách / áby je prędko cżytelnik przecżytał /
Przeto / że jeƺcże do pewnego cżáſu odłożone jeſt widzenie / które wypowie ná ſkońcżeniu jego / á nie ſkłámie ; á jeſliby ná chwilę odwłácżał / ocżekuj nań ; boć zápewne przydźie / á nie omieƺká.
Oto kto ſobie hárdźie pocżyná / tego duƺá nie jeſt ƺcżerá w nim ; ále ſpráwiedliwy z wiáry ſwej żyć będźie.
Dopieroż cżłowiek opiły / przewrotny y hárdy nie oſtoi śię w mieƺkániu ſwojim / który rozƺerzá jáko piekło duƺę ſwoję / á jeſt jáko śmierć / która śię nie może náſyćić / choćby zgromádźił do śiebie wƺyſtkie národy / á zebrał do śiebie wƺyſtkich ludźi.
Izali ći wƺyſcy o nim przypowieśći nie ucżynią y wykłádów y gádánin o nim? mówiąc : Biádá temu / który rozmnáżá rzecży nie ſwoje / ( á dokądże? ) y obćiążá śię gęſtem błotem!
Izali nie powſtáną z prędká / którzy ćię kąſáć będą / y nie ocucą śię / którzy ćię ƺárpáć będą? y ſtánieƺ śię im łupem.
Bo iżeś ty złupił wiele národów / złupią ćię też wƺyſtkie oſtátki národów / dla krwi ludzkiej y dla gwáłtu ucżynionego źiemi y miáſtu y wƺyſtkim / którzy mieƺkáją w nim.
Biádá temu / który łákomie ƺuká zyſku ƺkárádnego domowi ſwemu / áby wyſtáwił wyſoko gniázdo ſwoje / á ták uƺedł z mocy złego!
Urádźiłeś háńbę domowi ſwemu / ábyś wytráćił wiele národów / á grzeƺył przećiwko duƺy ſwojey.
Abowiem kámień z muru wołáć będźie / y ſęk z drzewá wydá o tym świádectwo.
Biádá temu / który krwią buduje miáſto / á utwierdzá miáſtá niepráwośćią!
Azaż to nie jeſt od Páná zaſtępów / iż koło cżego ludźie prácują / to ogień ſkáźi / á nád cżym śię národy ſprácowáły / to nádáremno będźie?
Abowiem źiemiá będźie nápełnioná znájomośćią chwały Páńſkiej / jáko morze wody nápełniáją.
Biádá temu / który poi bliźniego ſwego / przyſtáwiájąc nácżyniá ſwego / ták áby go upoił / y nápátrzył śię nágośći jego!
Náſyćiƺ śię háńby dla ſłáwy ; pić będźieƺ y ty / á obnáżony będźieƺ ; obróći śię do ćiebie kielich práwicy Páńſkiej / y zwrócenie ſromotne przydźie ná ſłáwę twoję.
Bo ćię łupieſtwo Libánu okryje y ſpuſtoƺenie zwierząt / które ich ſtráƺyło : dla krwi ludzkiej y dla gwáłtu źiemi y miáſtá / y wƺyſtkich / którzy mieƺkáją w nim.
Cóż pomoże ryty obraz / że go wyrył rzemieślnik jego? álbo odlewány obraz y náucżyćiel kłámſtwá / że ufá rzemieślnik w roboćie ſwojey / cżyniąc báłwány nieme?
Biádá temu! który mówi drewnu : Ocuć śię / á kámieniowi niememu : Obudź śię! Tenże to ma ucżyć? Spojrzyj nań / powlecżonyć jeſt złotem y ſrebrem ; ále w nim niemáƺ zgołá żádnego duchá.
Pán jeſt w kośćiele świętobliwośći ſwojey ; umilknij przed oblicżem jego wƺyſtká źiemio!
