Ná ſtráży ſwej ſtáć będę / y ſtánę ná báƺćie / wyglądájąc / ábym obácżył / co będźie Bóg mówił / cobym miáł odpowiedźieć po kárániu mojem.
Tedy mi odpowiedźiáł Pán / mówiąc : Nápiƺ widzenie / á nápiƺ rzetelnie ná táblicách / áby je prędko cżytelnik przecżytáł /
Przeto / że jeƺcże do pewnego cżáſu odłożone jeſt widzenie / które wypowie ná ſkońcżeniu jego / á nie ſkłámie ; á jeźliby ná chwilę odwłácżáł / ocżekuj náń ; boć zápewne przyjdźie / á nie omieƺká.
Oto kto ſobie hárdźie pocżyná / tego duƺá nie jeſt ƺcżerá w nim ; ále ſpráwiedliwy z wiáry ſwej żyć będźie.
Dopieroż cżłowiek opiły / przewrotny y hárdy nie oſtoi śię w mieƺkániu ſwojim / który rozƺerzá jáko piekło duƺę ſwoję / á jeſt jáko śmierć / którá śię nie może náſyćić / choćby zgromádźił do śiebie wƺyſtkie národy / á zebráł do śiebie wƺyſtkich ludźi.
Izáli ći wƺyſcy o nim przypowieśći nie ucżynią y wykłádów y gádánin o nim? mówiąc : Biádá temu / który rozmnáżá rzecży nie ſwoje / ( á dokądże? ) y obćiążá śię gęſtem błotem!
Izáli nie powſtáną z prędká / którzy ćię kąſáć będą / y nie ocucą śię / którzy ćię ƺárpáć będą? y ſtánieƺ śię im łupem.
Bo iżeś ty złupił wiele národów / złupią ćię też wƺyſtkie oſtátki národów / dlá krwi ludzkiej y dlá gwáłtu ucżynionego źiemi y miáſtu y wƺyſtkim / którzy mieƺkáją w nim.
Biádá temu / który łákomie ƺuká zyſku ƺkárádnego domowi ſwemu / áby wyſtáwił wyſoko gniázdo ſwoje / á ták uƺedł z mocy złego!
Urádźiłeś háńbę domowi ſwemu / ábyś wytráćił wiele národów / á grzeƺył przećiwko duƺy ſwojej.
Abowiem kámień z muru wołáć będźie / y ſęk z drzewá wydá o tem świádectwo.
Biádá temu / który krwią buduje miáſto / á utwierdzá miáſtá niepráwośćią!
Azáż to nie jeſt od Páná záſtępów / iż koło cżego ludźie prácują / to ogień ſkáźi / á nád cżem śię národy ſprácowáły / to nádáremno będźie?
Abowiem źiemiá będźie nápełnioná znájomośćią chwáły Páńſkiej / jáko morze wody nápełniáją.
Biádá temu / który poi bliźniego ſwego / przyſtáwiájąc nácżyniá ſwego / ták áby go upoił / y nápátrzył śię nágośći jego!
Náſyćiƺ śię háńby dlá ſłáwy ; pić będźieƺ y ty / á obnáżony będźieƺ ; obróći śię do ćiebie kielich práwicy Páńſkiej / y zwrócenie ſromotne przyjdźie ná ſłáwę twoję.
Bo ćię łupieſtwo Libánu okryje y ſpuſtoƺenie zwierząt / które ich ſtráƺyło : dlá krwi ludzkiej y dlá gwáłtu źiemi y miáſtá / y wƺyſtkich / którzy mieƺkáją w nim.
Cóż pomoże ryty obráz / że go wyrył rzemieślnik jego? álbo odlewány obráz y náucżyćiel kłámſtwá / że ufá rzemieślnik w roboćie ſwojej / cżyniąc báłwány nieme?
Biádá temu! który mówi drewnu : Ocuć śię / á kámieniowi niememu : Obudź śię! Tenże to má ucżyć? Spojrzyj náń / powlecżonyć jeſt złotem y ſrebrem ; ále w nim niemáƺ zgołá żádnego duchá.
Pán jeſt w kośćiele świętobliwośći ſwojej ; umilknij przed oblicżem jego wƺyſtká źiemio!
