Brzemię / które widźiáł prorok Abákuk.
Dokądże wołáć będę / o Pánie! á nie wyſłucháƺ? Dokądże do ćiebie przed gwáłtem krzycżeć będę / á nie wybáwiƺ?
Przecżże dopuƺcżáƺ / ábym pátrzył ná niepráwość / y widźiáł bezpráwie / y zgubę / y gwáłt przećiwko ſobie? y przecż śię znájduje ten / który ſwár y niezgodę rozniecá?
Dlátego náruƺony bywá zákon / á práwu śię nigdy doſyć nie dźieje ; ábowiem niepobożny otácżá ſpráwiedliwego / dlátego wychodźi ſąd przewrotny.
Spojrzyjćie ná národy á obácżćie y dźiwujćie śię z zdumieniem / przeto / iż cżynię nieco zá dni wáƺych / o cżem gdy wám powiádáć będą / nie uwierzyćie.
Abowiem oto Já wzbudzę Cháldejcżyków / náród ſrogi y prędki / który ƺeroko pójdźie przez źiemię / áby pośiádł mieƺkániá cudze.
Stráƺny jeſt y ogromny ; od niego ſámego wynijdźie ſąd jego / y wywyżƺenie jego.
Konie jego prędƺe będą niż lámpárty / á ſrożƺe nád wilki wiecżorne ; ƺeroko rozćiągną śię jezdni jego / á jezdni jego z dáleká przyjdą / przylecą jáko orzeł ſpieƺący śię do żeru.
Káżdy z nich dlá łupieſtwá przyjdźie ; obrócą twárze ſwoje ná wſchód ſłońcá / á więźniów zgromádzą jáko piáſek.
Ten y z królów ƺydźić będźie / á kśiążętá będą ná pośmiech u niego ; ten też z káżdej twierdzy náśmiewáć śię będźie / á uſypáwƺy wáły weźmie ją.
Tedy śię odmieni duch jego / á wyſtąpi y przewini / myśląc / że tá moc jego jeſt bogá jego.
Izáliś ty nie jeſt od wieku / Pánie / Boże mój / Święty mój? myć nie pomrzemy ; o Pánie! poſtáwiłeś go ná ſąd ; ty / o ſkáło náƺá! ná káránieś go ugruntowáł.
Cżyſte ſą ocży twoje / ták / że ná złe pátrzyć y bázpráwiá widźieć nie mogą ; przecżżebyś miáł pátrzyć ná cżyniących przewrotność? Przecżżebyś miáł milcżeć / poniewáż niezbożnik pożerá ſpráwiedliwƺego niżeli ſám?
Miáłżebyś zániecháć ludźi jáko ryb morſkich / jáko płázu / który nie má páná?
Wƺyſtkie wędą wyćiągá / zágárniá je niewodem ſwoim / y zgromádzá je do śieći ſwoich ; dlátegoż śię weſeli y ráduje.
Przeto ofiáruje niewodowi ſwemu y kádźi śieći ſwojej ; ábowiem przez nie utył dźiáł jego / á pożywienie jego hojniejƺe.
Izáli dlátego będźie zápuƺcżáł niewód ſwój / á uſtáwicżnie zábijáł národy bez litośći?
