Brzemię / które widziáł prorok Abákuk.
Dokądże wołáć będę / o Pánie! á nie wyſłucháƺ? Dokądże do ćiebie przed gwáłtem krzycżeć będę / á nie wybáwiƺ?
Przecżże dopuƺcżáƺ / ábym pátrzył ná niepráwość / y widziáł bezpráwie / y zgubę / y gwáłt przećiwko ſobie? y przecż śię znájduje ten / który ſwár y niezgodę rozniecá?
Dlátego náruƺony bywá zákon / á práwu śię nigdy doſyć nie dzieje ; ábowiem niepobożny otácżá ſpráwiedliwego / dlátego wychodzi ſąd przewrotny.
Spojrzyjćie ná národy á obácżćie y dziwujćie śię z zdumieniem / przeto / iż cżynię nieco zá dni wáƺych / o cżem gdy wám powiádáć będą / nie uwierzyćie.
Abowiem oto Já wzbudzę Cháldejcżyków / náród ſrogi y prędki / który ƺeroko pójdzie przez ziemię / áby pośiádł mieƺkániá cudze.
Stráƺny jeſt y ogromny ; od niego ſámego wynijdzie ſąd jego / y wywyżƺenie jego.
Konie jego prędƺe będą niż lámpárty / á ſrożƺe nád wilki wiecżorne ; ƺeroko rozćiągną śię jezdni jego / á jezdni jego z dáleká przyjdą / przylecą jáko orzeł ſpieƺący śię do żeru.
Káżdy z nich dlá łupieſtwá przyjdzie ; obrócą twárze ſwoje ná wſchód ſłońcá / á więźniów zgromádzą jáko piáſek.
Ten y z królów ƺydzić będzie / á kśiążętá będą ná pośmiech u niego ; ten też z káżdej twierdzy náśmiewáć śię będzie / á uſypáwƺy wáły weźmie ją.
Tedy śię odmieni duch jego / á wyſtąpi y przewini / myśląc / że tá moc jego jeſt bogá jego.
Izáliś ty nie jeſt od wieku / Pánie / Boże mój / Święty mój? myć nie pomrzemy ; o Pánie! poſtáwiłeś go ná ſąd ; ty / o ſkáło náƺá! ná káránieś go ugruntowáł.
Cżyſte ſą ocży twoje / ták / że ná złe pátrzyć y bázpráwiá widzieć nie mogą ; przecżżebyś miáł pátrzyć ná cżyniących przewrotność? Przecżżebyś miáł milcżeć / poniewáż niezbożnik pożerá ſpráwiedliwƺego niżeli ſám?
Miáłżebyś zániecháć ludzi jáko ryb morſkich / jáko płázu / który nie má páná?
Wƺyſtkie wędą wyćiągá / zágárniá je niewodem ſwoim / y zgromádzá je do śieći ſwoich ; dlátegoż śię weſeli y ráduje.
Przeto ofiáruje niewodowi ſwemu y kádzi śieći ſwojej ; ábowiem przez nie utył dziáł jego / á pożywienie jego hojniejƺe.
Izáli dlátego będzie zápuƺcżáł niewód ſwój / á uſtáwicżnie zábijáł národy bez litośći?
