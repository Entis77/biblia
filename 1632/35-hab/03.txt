Modlitwá Abákuká proroká według rozmáitych pieśni złożoná.
O Pánie! uſłyƺáwƺy wyrok twój ulękłem śię. O Pánie! záchowáj ſpráwę twoję w pośrodku lát / y objáw ją w pośrodku lát ; w gniewie wſpomnij ná miłośierdzie.
Gdy Bóg ƺedł od południá / á Święty z góry Fárán / Selá! okryłá niebioſá ſłáwá jego / á chwáły jego ziemiá pełná byłá.
Jáſność jego byłá jáko świátłość / rogi były ná bokách jego / á tám byłá ſkrytá śiłá jego.
Przed oblicżem jego ƺedł mór / á węgle páłájące ƺły przed nogámi jego.
Stánął y rozmierzył ziemię / wejrzáł y rozproƺył národy ; ſkruƺone ſą góry wiecżne / y ſkłoniły śię págórki dáwne : drogi jego ſą wiecżne.
Widziáłem námioty Chuſán próżnośći poddáne / á opony ziemi Mádyjáńſkiej drżáły.
Izáli śię ná rzeki / o Pánie! izáli śię ná rzeki rozpálił gniew twój? Izáli ná morze rozgniewánie twoje / gdyś jecháł ná koniách twoich / y ná wozách twoich zbáwiennych?
Jáwnie odkryty jeſt łuk twój dlá przyśięgi pokoleniom wyrzecżonej / Selá!
Rozdzieliłeś rzeki ziemi : widziáły ćię góry y zádrżáły / powódź wód przeminęłá ; przepáść wydáłá głoſ ſwój / głębokość ręce ſwoje podnioſłá.
Słońce y mieśiąc záſtánowił śię w mieƺkániu ſwojim / przy jegoż świetle látáły ſtrzáły twe / y przy bláſku lśniącej śię włócżni twojej.
W zágniewániu podeptáłeś ziemię / w zápálcżywośći młóćiłeś pogán ;
Wyƺedłeś ná wybáwienie ludu ſwego / ná wybáwienie z pomázáńcem twoim ; przebiłeś głowę z domu niezbożniká / odkrywƺy grunt áż do ƺyi / Selá!
Potłukłeś kijmi jego głowę wśi jego / gdy śię burzyli jáko wicher / áby mię rozproƺyli ; weſelili śię / jákoby pożreć mieli ubogiego w ſkrytośći.
Jecháłeś przez morze ná koniách twoich / przez gromádę wód wielkich.
Gdym to ſłyƺáł / zátrząſnął śię brzuch mój! ná ten głoſ drżáły wárgi moje / zgniłość weƺłá w kośći moje / y wƺyſtekem śię trząſł / ſłyƺąc / że mám odpocżąć w dzień utrápieniá / gdy przyćiągnie ná ten lud nieprzyjáćiel / áby go przez wojnę wygłádził.
Choćby figowe drzewo nie zákwitnęło y nie było urodzáju ná winnicách / choćby y owoc oliwy pochybił / y role nie przynioſły pożytku / y z owcżárniby owce wybite były / á nie byłoby bydłá w oborách ;
Wƺákże śię já w Pánu weſelić będę / rozráduję śię w Bogu zbáwieniá mego.
Pánujący Pán jeſt śiłą moją / który cżyni nogi moje / jáko nogi łáni / y po miejſcách wyſokich poprowádzi mię. Przedniejƺemu nád śpiewákámi ná muzycżnem nácżyniu mojem.
