W on cżáſ uſłyƺáł Herod Tetrárchá / wieść o Jezuśie.
Y rzekł ſługom ſwoim : Tenći jeſt Ján Chrzćićiel ; on to zmártwychwſtáł / y dlátego śię cudá przezeń dzieją.
Abowiem Herod pojmáwƺy Jáná / związáł go był y wſádził do więzieniá dlá Herodyjády / żony Filipá / brátá ſwego.
Bo mu Ján mówił : Nie godzić śię jej mieć.
Ale gdy go on chćiáł zábić / báł śię ludu ; ábowiem go zá proroká mieli.
Gdy tedy obchodzono dzień národzeniá Herodowego / táńcowáłá córká Herodyjády w pośrodku gośći / y podobáłá śię Herodowi.
Skąd pod przyśięgą obiecáł jej dáć / cżegobykolwiek żądáłá.
A oná przedtem będąc nápráwioná od mátki ſwojej / rzekłá : Dáj mi tu ná miśie głowę Jáná Chrzćićielá.
Y záſmućił śię król ; ále dlá przyśięgi y dlá ſpółśiedzących kázáł jej dáć.
A poſłáwƺy kátá / śćiął Jáná w więzieniu.
Y przynieśiono głowę jego ná miśie / y oddáno dziewecżce / y odnioſłá ją mátce ſwojej.
A przyƺedłƺy ucżniowie jego wzięli ćiáło y pogrzebli je / á ƺedłƺy powiedzieli Jezuſowi.
To uſłyƺáwƺy Jezuſ / uſtąpił ſtámtąd w łodzi ná miejſce puſte oſobno ; á uſłyƺáwƺy lud / ƺli zá nim z miáſt pieƺo.
Wyƺedłƺy tedy Jezuſ ujrzáł wielki lud / y użálił śię ich / á uzdráwiáł chore ich.
A gdy nádchodził wiecżór / przyſtąpili do niego ucżniowie jego / mówiąc : Puſte jeſt to miejſce / á cżáſ już przeminął ; rozpuść ten lud / áby odƺedłƺy do miáſtecżek / kupili ſobie żywnośći.
A Jezuſ im rzekł : Nie potrzebá im odchodzić / dájćie wy im co jeść.
Ale mu oni rzekli : Nie mámy tu / tylko pięć chlebów y dwie ryby.
A on rzekł : Przynieśćie mi je tu.
Y rozkázáwƺy ludowi uśiąść ná tráwie / wziął onych pięć chlebów y dwie ryby / á wejrzáwƺy w niebo / błogoſłáwił / á łámiąc dáwáł ucżniom chleby / á ucżniowie ludowi.
Y jedli wƺyſcy á náſyceni byli ; y zebráli / co zbywáło ułomków / dwánáśćie koƺów pełnych.
A tych / którzy jedli / było około pięćiu tyśięcy mężów / oprócż niewiáſt y dziátek.
A wnetże przymuśił Jezuſ ucżniów ſwoich / áby wſtąpili w łódź / y uprzedzili go ná drugą ſtronę / áżby rozpuśćił lud.
A rozpuśćiwƺy lud / wſtąpił ná górę z oſobná / áby śię modlił ; á gdy był wiecżór / ſám tám był.
A łódź już w pośrodku morzá będąc / miotáná byłá od wáłów ; ábowiem był wiátr przećiwny.
Lecż o cżwártej ſtráży nocnej ƺedł do nich Jezuſ / chodząc po morzu.
A ujrzáwƺy go ucżniowie po morzu chodzącego / zátrwożyli śię / mówiąc : Obłudá to jeſt! y od bojáźni krzyknęli.
Lecż wnet rzekł do nich Jezuſ / mówiąc : Ufájćie! Jámći to jeſt ; nie bójćie śię.
A odpowiádájąc mu Piotr rzekł : Pánie! jeźliżeś ty jeſt / káż mi przyjść do ćiebie po wodzie.
A on rzekł : Pójdź! A Piotr / wyſtąpiwƺy z łodzi / ƺedł po wodzie / áby przyƺedł do Jezuſá ;
Ale widząc wiátr gwáłtowny / zląkł śię ; á gdy pocżął tonąć / zákrzyknął / mówiąc : Pánie / rátuj mię!
A Jezuſ záráz wyćiągnąwƺy rękę / uchwyćił go y rzekł mu : O máłowierny! przecżżeś wątpił?
A gdy oni wſtąpili w łódź / ućiƺył śię wiátr.
A ći / którzy byli w łodzi / przyſtąpiwƺy pokłonili mu śię / mówiąc : Práwdziwie jeſteś Synem Bożym.
Y przepráwiwƺy śię / przyƺli do ziemi Gienezáret.
A poznáwƺy go mężowie miejſcá onego / poſłáli do wƺyſtkiej onej okolicżnej kráiny ; y przynieśiono do niego wƺyſtkie / którzy śię źle mieli.
Y prośili go / áby śię tylko podołku ƺáty jego dotykáli ; á którzy śię kolwiek dotknęli / uzdrowieni ſą.
