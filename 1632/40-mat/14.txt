W on cżáſ uſłyƺáł Herod Tetrárchá / wieść o JEzuśie.
Y rzekł ſługom ſwoim : Tenći jeſt Ján Chrzćićiel ; on to zmartwychwſtáł / y dlátego śię cudá przezeń dźieją.
Abowiem Herod pojmáwƺy Jáná / związáł go był y wſádźił do więźieniá dla Herodyjády / żony Filipá / brátá ſwego.
Bo mu Ján mówił : Nie godźić śię jey mieć.
Ale gdy go on chćiáł zábić / báł śię ludu ; ábowiem go zá proroká mieli.
Gdy tedy obchodzono dźień národzeniá Herodowego / táńcowáłá córká Herodyjády w pośrodku gośći / y podobáłá śię Herodowi.
Skąd pod przyśięgą obiecáł jey dáć / cżegobykolwiek żądáłá.
A oná przedtem będąc nápráwioná od mátki ſwojey / rzekłá : Dáj mi tu ná miśie głowę Jáná Chrzćićielá.
Y záſmućił śię król ; ále dla przyśięgi y dla ſpółśiedzących kázáł jey dáć.
A poſłáwƺy kátá / śćiął Jáná w więźieniu.
Y przynieśiono głowę jego ná miśie / y oddáno dźiewecżce / y odnioſłá ją mátce ſwojey.
A przyƺedłƺy ucżniowie jego wźięli ćiáło y pogrzebli je / á ƺedłƺy powiedźieli JEzuſowi.
To uſłyƺáwƺy JEzuſ / uſtąpił ſtámtąd w łodźi ná miejſce puſte oſobno ; á uſłyƺáwƺy lud / ƺli zá nim z miáſt pieƺo.
Wyƺedłƺy tedy JEzuſ ujrzáł wielki lud / y użálił śię ich / á uzdráwiáł chore ich.
A gdy nádchodźił wiecżór / przyſtąpili do niego ucżniowie jego / mówiąc : Puſte jeſt to miejſce / á cżáſ już przeminął ; rozpuść ten lud / áby odƺedłƺy do miáſtecżek / kupili ſobie żywnośći.
A JEzuſ im rzekł : Nie potrzebá im odchodźić / dájćie wy im co jeść.
Ale mu oni rzekli : Nie mámy tu / tylko pięć chlebów y dwie ryby.
A on rzekł : Przynieśćie mi je tu.
Y rozkázáwƺy ludowi uśiąść ná tráwie / wźiął onych pięć chlebów y dwie ryby / á wejrzáwƺy w niebo / błogoſłáwił / á łámiąc dáwáł ucżniom chleby / á ucżniowie ludowi.
Y jedli wƺyſcy á náſyceni byli ; y zebráli / co zbywáło ułomków / dwánáśćie koƺów pełnych.
A tych / którzy jedli / było około pięćiu tyśięcy mężów / oprócż niewiáſt y dźiátek.
A wnetże przymuśił JEzuſ ucżniów ſwoich / áby wſtąpili w łódź / y uprzedźili go ná drugą ſtronę / áżby rozpuśćił lud.
A rozpuśćiwƺy lud / wſtąpił ná górę z oſobná / áby śię modlił ; á gdy był wiecżór / ſám tám był.
A łódź już w pośrodku morzá będąc / miotáná byłá od wáłów ; ábowiem był wiátr przećiwny.
Lecż o cżwártey ſtráży nocney ƺedł do nich JEzuſ / chodząc po morzu.
A ujrzáwƺy go ucżniowie po morzu chodzącego / zátrwożyli śię / mówiąc : Obłudá to jeſt! y od bojáźni krzyknęli.
Lecż wnet rzekł do nich JEzuſ / mówiąc : Ufájćie! Jámći to jeſt ; nie bójćie śię.
A odpowiádájąc mu Piotr rzekł : Pánie! jeſliżeś ty jeſt / káż mi przyjść do ćiebie po wodźie.
A on rzekł : Pójdź! A Piotr / wyſtąpiwƺy z łodźi / ƺedł po wodźie / áby przyƺedł do JEzuſá ;
Ale widząc wiátr gwáłtowny / zląkł śię ; á gdy pocżął tonąć / zákrzyknął / mówiąc : Pánie / rátuj mię!
A JEzuſ záráz wyćiągnąwƺy rękę / uchwyćił go y rzekł mu : O máłowierny! przecżżeś wątpił?
A gdy oni wſtąpili w łódź / ućiƺył śię wiátr.
A ći / którzy byli w łodźi / przyſtąpiwƺy pokłonili mu śię / mówiąc : Práwdźiwie jeſteś Synem Bożym.
Y przepráwiwƺy śię / przyƺli do źiemi Gienezáret.
A poznáwƺy go mężowie miejſcá onego / poſłáli do wƺyſtkiej oney okolicżney kráiny ; y przynieśiono do niego wƺyſtkie / którzy śię źle mieli.
Y prośili go / áby śię tylko podołku ƺáty jego dotykáli ; á którzy śię kolwiek dotknęli / uzdrowieni ſą.
