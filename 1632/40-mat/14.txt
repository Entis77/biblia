W on cżás uſłyƺał Herod Tetrárchá / wieść o JEzuśie.
Y rzekł ſługom ſwoim : Tenći jeſt Ján Chrzćićiel : on to zmartwychwſtał / y dla tego śię cudá przezeń dźieją.
Abowiem Herod pojmáwƺy Jáná / związał go był / y wſádźił do więźieniá dla Herodyjády żony Filipá brátá ſwego.
Bo mu Ján mówił : Nie godźić śię jey mieć.
Ale gdy go on chćiał zábić / bał śię ludu : ábowiem go zá proroká mieli.
Gdy tedy obchodzono dźień národzeniá Herodowego / táńcowáłá córká Herodyjády w pośrzodku <i>gośći</i> : Y podobáłá śię Herodowi.
Z kąd pod przyśięgą obiecał jey dáć cżegobykolwiek żądáłá.
A oná przed tym będąc nápráwioná od mátki ſwojey / rzekłá : Dáj mi tu ná miśie głowę Jáná Chrzćićielá.
Y záſmućił śię Król : ále dla przyśięgi / y dla ſpółśiedzących / kázał <i>jey</i> dáć.
A poſłáwƺy <i>kátá,</i> śćiął Jáná w więźieniu.
Y przynieśiono głowę jego ná miśie / a oddáno dźiewecżce : y odnioſłá ją mátce ſwojey.
A przyƺedƺy ucżniowie jego / wźięli ćiáło / y pogrzebli je : á ƺedƺy / powiedźieli JEzuſowi.
To uſłyƺáwƺy JEzus / uſtąpił z támtąd w łodźi ná miejſce puſte oſobno : á uſłyƺáwƺy lud / ƺli zá nim z miaſt pieƺo.
Wyƺedƺy tedy JEzus ujrzał wielki lud / y użálił śię ich / á uzdráwiał chore ich.
A gdy nádchodźił wiecżór / przyſtąpili do niego ucżniowie jego mówiąc : Puſte jeſt to miejſce / á cżás już przeminął : rozpuść ten lud / áby odƺedƺy do miáſtecżek / kupili ſobie żywnośći.
A JEzus im rzekł : Nie potrzebá im odchodźić / dájćie wy im co jeść.
Ale mu oni rzekli : Nie mamy tu tylko pięć chlebów / y dwie rybie.
A on rzekł : Przynieśćie mi je tu.
Y rozkázáwƺy ludowi uśiąść ná tráwie / wźiął onych pięć chlebów / y dwie rybie : á wejrzáwƺy w niebo / błogoſłáwił : á łámiąc / dáwał ucżniom chleby / á ucżniowie ludowi.
Y jedli wƺyſcy / á náſyceni byli : y zebráli co zbywáło ułomków dwánaśćie koƺów pełnych.
A tych którzy jedli / było około pięćiu tyśięcy mężów / oprócż niewiáſt y dźiátek.
A wnetże przymuśił JEzus ucżniów ſwojich / áby wſtąpili w łódź y uprzedźili go ná drugą ſtronę / áżby rozpuśćił lud.
A rozpuśćiwƺy lud / wſtąpił ná górę z oſobná / áby śię modlił. A gdy był wiecżór / ſám tám był.
A łódź już w pośrzodku morzá będąc / miotáná byłá od wáłów : ábowiem był wiátr przećiwny.
Lecż o cżwártey ſtráży nocney / ƺedł do nich JEzus / chodząc po morzu.
A ujrzáwƺy go ucżniowie po morzu chodzącego / zátrwożyli sobą / mówiąc : Obłudá to jeſt ; y od bojáźni krzyknęli.
Lecż wnet rzekł do nich JEzus / mówiąc : Ufájćie / Jámći to jeſt ; Nie bójćie śię.
A odpowiádájąc mu Piotr / rzekł ; Pánie / jeſliżeś ty jeſt / káż mi przyść do śiebie po wodźie.
A on rzekł ; Pójdź. A Piotr wyſtąpiwƺy z łodźi / ƺedł po wodźie / áby przyƺedł do JEzuſá.
Ale widząc wiátr gwáłtowny / zląkł śię. A gdy pocżął tonąć / zákrzyknął / mówiąc ; Pánie rátuj mię!
A JEzus záráz wyćiągnąwƺy rękę / uchwyćił go / y rzekł mu ; O Máłowierny! przecżżeś wątpił?
A gdy oni wſtąpili w łódź / ućiƺył śię wiátr.
A ći którzy byli w łodźi / przyſtąpiwƺy pokłonili mu śię mówiąc ; Práwdźiwie jeſteś Synem Bożym.
Y przepráwiwƺy śię / przyƺli do źiem. Gienezáret.
A poznáwƺy go mężowie miejſcá onego / poſłáli do wƺyſtkiej oney okolicżney kráiny : y przynieśiono do niego wƺyſtkie / którzy śię źle mieli.
Y prośili go / áby śię tylko podołku ƺáty jego dotykáli ; á którzy śię kolwiek dotknęli / uzdrowieni ſą.
