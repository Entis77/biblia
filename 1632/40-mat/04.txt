Tedy JEzus záwiedźiony jeſt ná puƺcżą od Duchá / áby był kuƺony od Dyabłá.
A gdy pośćił cżterdźieśći dni / y cżterdźieśći nocy / potym łáknął.
Y przyſtąpiwƺy do niego kuśićiel / rzekł ; Jeſliś jeſt Syn Boży / rzecż áby śię to kámienie ſtáło chlebem.
A on odpowiádájąc rzekł ; Nápiſano / Nie ſámym chlebem cżłowiek żyć będźie / ále káżdym ſłowem pochodzącym przez uſtá Boże.
Tedy go wźiął dyabeł do miáſtá świętego / y poſtáwił go ná ganku kośćielnym /
Y rzekł mu ; Jeſliś jeſt Syn Boży / zpuść śię ná dół : Abowiem nápiſano ; Iż Anjołom ſwojim przykazał o tobie / y będą ćię ná ręku nośili / ábyś ſnadź nie obráźił o kámień nogi ſwojey.
Rzekł mu JEzus ; Záśię nápiſano : Nie będźieƺ kuśił PAná Bogá twego.
Wźiął go záśię dyabeł ná górę bárzo wyſoką / y pokazał mu wƺyſtkie króleſtwá świátá y ſławę ich /
Y rzekł mu ; To wƺyſtko dam tobie / jeſli upadƺy pokłoniƺ mi śię.
Tedy mu rzekł JEzus : Pódź precż ƺátánie : ábowiem nápiſano ; PAnu Bogu twemu kłaniáć śię będźieƺ / y jemu ſámemu ſłużyć będźieƺ.
//Tedy go opuśćił dyabeł : á oto Anjołowie przyſtąpili / y ſłużyli mu.
A gdy uſłyƺał JEzus / iż Jan był podány <i>do więźienia,</i> wróćił śię do Gálilejey.
A opuśćiwƺy Názáret / przyƺedł y mieƺkał w Kápernáum / które jeſt nád morzem / w gránicách Zábulonowych y Neftálimowych.
Aby śię wypełniło co powiedźiano przez Izájaƺá Proroká / mówiącego :
Ziemiá Zábulonowá / y źiemiá Neftálimowá / <i>przy</i> drodze morſkiey zá Jordanem / Gálilea Pogánów :
Lud który śiedźiał w ćiemnośći / widźiał świátłość wielką ; á śiedzącym w krájinie y w ćieniu śmierći / weƺłá im świátłość.
Od onego cżáſu pocżął JEzus kazáć / y mówić ; Pokutujćie / Abowiem śię przybliżyło króleſtwo niebieſkie.
A gdy JEzus chodźił nád morzem Gálilejſkim / ujrzał dwu bráći / Symoná / którego zowią Piotrem / y Andrzejá brátá jego ; którzy zápuƺcżáli śieć w morze : ábowiem byli rybitwi.
Y rzekł im ; Pódźćie zá mną : á ucżynię was rybitwámi ludźi.
A oni záraz opuśćiwƺy śieći / ƺli zá nim.
A poſtąpiwƺy z támtąd / ujrzał drugich dwu bráći / Jákubá <i>ſyná</i> Zebedeuƺowego / y Janá brátá jego w łodźi / z Zebedeuƺem ojcem ich / oprawiájących śieći ſwoje ; y wezwał ich.
A oni wnetże opuśćiwƺy łódź y Ojcá ſwego / poƺli zá nim.
Y obchodźił JEzus wƺyſtkę Gálileją / ucżąc w bóżnicách ich / y każąc Ewángelią króleſtwá / á uzdrawiájąc wƺelką chorobę / y wƺelką niemoc miedzy ludem.
Y rozeƺłá śię wieść o nim po wƺyſtkiey Syryjey. Y przywodzono do niego wƺyſtkie źle śię májące / á rozmájitymi chorobámi y mękámi zjęte ; tákże y opętáne / y lunátyki / y powietrzem ruƺone : y uzdrawiał je.
A ƺedł zá nim lud wielki z Gálilejey / y z dźieśiąći miaſt / y z Jeruzalem / y z Judſkiey źiemie / y z Zá-Jordania.
