Tedy JEzus záwiedźiony jeſt ná puƺcżę od Duchá / áby był kuƺony od Dyjábłá.
A gdy pośćił cżterdźieśći dni / y cżterdźieśći nocy / potym łáknął.
Y przyſtąpiwƺy do niego kuśićiel / rzekł ; Jeſliś jeſt Syn Boży / rzecż áby śię to kámienie ſtáły chlebem.
A on odpowiádájąc rzekł ; Nápiſano / Nie ſámym chlebem cżłowiek żyć będźie / ále káżdym ſłowem pochodzącym przez uſtá Boże.
Tedy go wźiął dyjábeł do miáſtá świętego / y poſtáwił go ná gánku kośćielnym /
Y rzekł mu ; Jeſliś jeſt Syn Boży / ſpuść śię ná dół : Abowiem nápiſano ; Iż Aniołom ſwoim przykázał o tobie / y będą ćię ná ręku nośili / ábyś ſnadź nie obráźił o kámień nogi ſwojey.
Rzekł mu JEzus ; Záśię nápiſano : Nie będźieƺ kuśił PAná Bogá twego.
Wźiął go záśię dyjábeł ná górę bárzo wyſoką / y pokázał mu wƺyſtkie króleſtwá świátá y ſłáwę ich /
Y rzekł mu ; To wƺyſtko dam tobie / jeſli upádƺy pokłoniƺ mi śię.
Tedy mu rzekł JEzus : Pójdź precż ƺátánie : ábowiem nápiſano ; PAnu Bogu twemu kłániáć śię będźieƺ / y jemu ſámemu ſłużyć będźieƺ.
Tedy go opuśćił dyjábeł : á oto Aniołowie przyſtąpili / y ſłużyli mu.
A gdy uſłyƺał JEzus / iż Ján był podány <i>do więźieniá</i> / wróćił śię do Gálilei.
A opuśćiwƺy Názáret / przyƺedł y mieƺkał w Kápernáum / które jeſt nád morze / w gránicách Zábulonowych y Neftálimowych.
Aby śię wypełniło co powiedźiáno przez Izájáƺá Proroká / mówiącego :
Ziemiá Zábulonowá / y źiemiá Neftálimowá / <i>przy</i> drodze morſkiej zá Jordánem / Gálileá Pogánów :
Lud który śiedźiał w ćiemnośći / widźiał świátłość wielką ; á śiedzącym w kráinie y w ćieniu śmierći / weƺłá im świátłość.
Od onego cżáſu pocżął JEzus kázáć / y mówić ; Pokutujćie / ábowiem śię przybliżyło króleſtwo niebieſkie.
A gdy JEzus chodźił nád morzem Gálilejſkim / ujrzał dwu bráći / Szymoná / którego zowią Piotrem / y Andrzejá brátá jego ; którzy zápuƺcżáli śieć w morze : ábowiem byli rybitwi.
Y rzekł im ; Pójdźćie zá mną : á ucżynię was rybitwámi ludźi.
A oni záráz opuśćiwƺy śieći / ƺli zá nim.
A poſtąpiwƺy ſtámtąd / ujrzał drugich dwu bráći / Jákubá / <i>ſyná</i> Zebedeuƺowego / y Jáná / brátá jego w łodźi / z Zebedeuƺem ojcem ich / opráwiájących śieći ſwoje ; y wezwał ich.
A oni wnetże opuśćiwƺy łódź y ojcá ſwego / poƺli zá nim.
Y obchodźił JEzus wƺyſtkę Gálileją / ucżąc w bóżnicách ich / y káżąc Ewángielią króleſtwá / á uzdráwiájąc wƺelką chorobę / y wƺelką niemoc miedzy ludem.
Y rozeƺłá śię wieść o nim po wƺyſtkiej Syryi. Y przywodzono do niego wƺyſtkie źle śię májące / á rozmáitemi chorobámi y mękámi zjęte ; tákże y opętáne / y lunátyki / y powietrzem ruƺone : y uzdráwiał je.
A ƺedł zá nim lud wielki z Gálilei / y z dźieśiąći miaſt / y z Jeruzálem / y z Judzkiej źiemi / y z Zá-Jordánia.
