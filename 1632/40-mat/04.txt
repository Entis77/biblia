Tedy Jezuſ záwiedziony jeſt ná puƺcżę od Duchá / áby był kuƺony od dyjábłá.
A gdy pośćił cżterdzieśći dni y cżterdzieśći nocy / potem łáknął.
Y przyſtąpiwƺy do niego kuśićiel / rzekł : Jeźliś jeſt Syn Boży / rzecż / áby śię te kámienie ſtáły chlebem.
A on odpowiádájąc rzekł : Nápiſáno : Nie ſámym chlebem cżłowiek żyć będzie / ále káżdem ſłowem pochodzącem przez uſtá Boże.
Tedy go wziął dyjábeł do miáſtá świętego / y poſtáwił go ná gánku kośćielnym /
Y rzekł mu : Jeźliś jeſt Syn Boży / ſpuść śię ná dół / ábowiem nápiſáno : Iż Aniołom ſwoim przykázáł o tobie / y będą ćię ná rękách nośili / ábyś ſnáć nie obráził o kámień nogi ſwojej.
Rzekł mu Jezuſ : Záśię nápiſáno : Nie będzieƺ kuśił Páná / Bogá twego.
Wziął go záśię dyjábeł ná górę bárdzo wyſoką / y pokázáł mu wƺyſtkie króleſtwá świátá y ſłáwę ich /
Y rzekł mu : To wƺyſtko dám tobie / jeźli upádłƺy / pokłoniƺ mi śię.
Tedy mu rzekł Jezuſ : Pójdź precż / ƺátánie! ábowiem nápiſáno : Pánu Bogu twemu kłániáć śię będzieƺ / y jemu ſámemu ſłużyć będzieƺ.
Tedy go opuśćił dyjábeł / á oto / Aniołowie przyſtąpili y ſłużyli mu.
A gdy uſłyƺáł Jezuſ / iż Ján był podány do więzieniá / wróćił śię do Gálilei ;
A opuśćiwƺy Názáret / przyƺedł / y mieƺkáł w Kápernáum / które jeſt nád morzem w gránicách Zábulonowych y Neftálimowych ;
Aby śię wypełniło / co powiedziáno przez Izájáƺá proroká / mówiącego :
Ziemiá Zábulonowá y ziemiá Neftálimowá przy drodze morſkiej zá Jordánem / Gálileá pogánów ;
Lud / który śiedziáł w ćiemnośći / widziáł świátłość wielką / á śiedzącym w kráinie y w ćieniu śmierći weƺłá im świátłość.
Od onego cżáſu pocżął Jezuſ kázáć y mówić : Pokutujćie / ábowiem śię przybliżyło króleſtwo niebieſkie.
A gdy Jezuſ chodził nád morzem Gálilejſkiem / ujrzáł dwóch bráći : Szymoná / którego zowią Piotrem / y Andrzejá / brátá jego / którzy zápuƺcżáli śieć w morze ; ábowiem byli rybitwy.
Y rzekł im : Pójdźćie zá mną / á ucżynię wáſ rybitwámi ludzi.
A oni záráz opuśćiwƺy śieći / ƺli zá nim.
A poſtąpiwƺy ſtámtąd / ujrzáł drugich dwóch bráći / Jákóbá / ſyná Zebedeuƺowego / y Jáná / brátá jego / w łodzi z Zebedeuƺem / ojcem ich / popráwiájących śieći ſwoje / y wezwáł ich.
A oni wnetże opuśćiwƺy łódź y ojcá ſwego / poƺli zá nim.
Y obchodził Jezuſ wƺyſtkę Gálileją / ucżąc w bóżnicách ich / y káżąc Ewángieliją króleſtwá / á uzdráwiájąc wƺelką chorobę y wƺelką niemoc między ludem.
Y rozeƺłá śię wieść o nim po wƺyſtkiej Syryi ; y przywodzono do niego wƺyſtkie źle śię májące / á rozmáitemi chorobámi y mękámi zdjęte / tákże y opętáne / y lunátyki y powietrzem ruƺone ; y uzdráwiáł je.
A ƺedł zá nim lud wielki z Gálilei / z dzieśięćiu miáſt / y z Jeruzálemu / y z Judzkiej ziemi / y z Zá-Jordániá.
