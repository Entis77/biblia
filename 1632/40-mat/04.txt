Tedy JEzuſ záwiedźiony jeſt ná puƺcżę od Duchá / áby był kuƺony od dyjábłá.
A gdy pośćił cżterdźieśći dni y cżterdźieśći nocy / potem łáknął.
Y przyſtąpiwƺy do niego kuśićiel / rzekł : Jeſliś jeſt Syn Boży / rzecż / áby śię te kámienie ſtáły chlebem.
A on odpowiádájąc rzekł : Nápiſáno : Nie ſámym chlebem cżłowiek żyć będźie / ále káżdem ſłowem pochodzącem przez uſtá Boże.
Tedy go wźiął dyjábeł do miáſtá świętego / y poſtáwił go ná gánku kośćielnym /
Y rzekł mu : Jeſliś jeſt Syn Boży / ſpuść śię ná dół / ábowiem nápiſáno : Iż Aniołom ſwoim przykázáł o tobie / y będą ćię ná rękách nośili / ábyś ſnáć nie obráźił o kámień nogi ſwojey.
Rzekł mu JEzuſ : Záśię nápiſáno : Nie będźieƺ kuśił Páná / Bogá twego.
Wźiął go záśię dyjábeł ná górę bárdzo wyſoką / y pokázáł mu wƺyſtkie króleſtwá świátá y ſłáwę ich /
Y rzekł mu : To wƺyſtko dám tobie / jeſli upádłƺy / pokłoniƺ mi śię.
Tedy mu rzekł JEzuſ : Pójdź precż / ƺátánie! ábowiem nápiſáno : Pánu Bogu twemu kłániáć śię będźieƺ / y jemu ſámemu ſłużyć będźieƺ.
Tedy go opuśćił dyjábeł / á oto / Aniołowie przyſtąpili y ſłużyli mu.
A gdy uſłyƺáł JEzuſ / iż Ján był podány do więźieniá / wróćił śię do Gálilei ;
A opuśćiwƺy Názáret / przyƺedł / y mieƺkáł w Kápernáum / które jeſt nád morzem w gránicách Zábulonowych y Neftálimowych ;
Aby śię wypełniło / co powiedźiáno przez Izájáƺá proroká / mówiącego :
Ziemiá Zábulonowá y źiemiá Neftálimowá przy drodze morſkiej zá Jordánem / Gálileá pogánów ;
Lud / który śiedźiáł w ćiemnośći / widźiáł świátłość wielką / á śiedzącym w kráinie y w ćieniu śmierći weƺłá im świátłość.
Od onego cżáſu pocżął JEzuſ kázáć y mówić : Pokutujćie / ábowiem śię przybliżyło króleſtwo niebieſkie.
A gdy JEzuſ chodźił nád morzem Gálilejſkiem / ujrzáł dwóch bráći : Szymoná / którego zowią Piotrem / y Andrzejá / brátá jego / którzy zápuƺcżáli śieć w morze ; ábowiem byli rybitwy.
Y rzekł im : Pójdźćie zá mną / á ucżynię was rybitwámi ludźi.
A oni záráz opuśćiwƺy śieći / ƺli zá nim.
A poſtąpiwƺy ſtámtąd / ujrzáł drugich dwóch bráći / Jákóbá / ſyná Zebedeuƺowego / y Jáná / brátá jego / w łodźi z Zebedeuƺem / ojcem ich / popráwiájących śieći ſwoje / y wezwáł ich.
A oni wnetże opuśćiwƺy łódź y ojcá ſwego / poƺli zá nim.
Y obchodźił JEzuſ wƺyſtkę Gálileją / ucżąc w bóżnicách ich / y káżąc Ewángieliją króleſtwá / á uzdráwiájąc wƺelką chorobę y wƺelką niemoc między ludem.
Y rozeƺłá śię wieść o nim po wƺyſtkiej Syryi ; y przywodzono do niego wƺyſtkie źle śię májące / á rozmáitemi chorobámi y mękámi zdjęte / tákże y opętáne / y lunátyki y powietrzem ruƺone ; y uzdráwiáł je.
A ƺedł zá nim lud wielki z Gálilei / z dźieśięćiu miáſt / y z Jeruzálemu / y z Judzkiej źiemi / y z Zá-Jordániá.
