Tedy JEzuſ rzekł do ludu y do ucżniów ſwoich / mówiąc :
Ná ſtolicy Mojżeƺowej uśiedli náucżeni w Piśmie y Fáryzeuƺowie.
Przetoż wƺyſtkiego / cżegoby wám kolwiek rozkázáli przeſtrzegáć / przeſtrzegájćie y cżyńćie / ále według ucżynków ich nie cżyńćie ; ábowiem oni mówią / ále nie cżynią.
Bo wiążą brzemioná ćiężkie y nieznośne / y kłádą je ná rámioná ludzkie / lecż pálcem ſwoim niechcą ich ruƺyć.
A wƺyſtkie ucżynki ſwoje cżynią / áby byli widźiáni od ludźi / y rozƺerzáją brámy ſwoje / y rozpuƺcżáją podołki płáƺcżów ſwoich.
Nádto miłują pierwƺe miejſcá ná wiecżerzách / y pierwƺe ſtołki w bóżnicách.
Y pozdráwiániá ná rynkách / y áby je názywáli ludźie : Miſtrzu / miſtrzu!
Ale wy nie názywájćie śię miſtrzámi ; ábowiem jeden jeſt miſtrz wáƺ / CHryſtuſ ; ále wyśćie wƺyſcy bráćmi.
Y nikogo nie zówćie ojcem wáƺym ná źiemi ; ábowiem jeden jeſt Ojćiec wáƺ / który jeſt w niebieśiech.
A niecháj wáſ nie zowią miſtrzámi / gdyż jeden jeſt miſtrz wáƺ / CHryſtuſ.
Ale kto z wáſ nájwiękƺy jeſt / będźie ſługą wáƺym.
A ktoby śię wywyżƺáł / będźie poniżony ; á ktoby śię poniżáł / będźie wywyżƺony.
Lecż biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż zámykáćie króleſtwo niebieſkie przed ludźmi ; ábowiem tám ſámi nie wchodźićie / áni tym / którzyby wnijść chćieli / wchodźić nie dopuƺcżáćie.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż pożeráćie domy wdów / á to pod pokrywką długich modlitw / dlátego ćiężƺy ſąd odnieśiećie.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż obchodźićie morze y źiemię / ábyśćie ucżynili jednego nowego Żydá ; á gdy śię ſtánie / cżynićie go ſynem zátráceniá / dwákroć więcej niżeliśćie ſámi.
Biádá wám / wodzowie ślepi! którzy powiádáćie : Ktoby przyśiągł ná kośćiół / nic nie jeſt : ále ktoby przyśiągł ná złoto kośćielne / winien jeſt.
Głupi y ślepi! ábowiem cóż jeſt więkƺego / złoto cży kośćiół / który poświęcá złoto?
A ktoby przyśiągł ná ołtárz / nic nie jeſt ; lecż ktoby przyśiągł ná dár / który jeſt ná nim / winien jeſt.
Głupi y ślepi! ábowiem cóż więkƺego jeſt? dár / cżyli ołtárz / który poświęcá dár?
Kto tedy przyśięgá ná ołtárz / przyśięgá náń / y ná to wƺyſtko / co ná nim jeſt ;
A kto przyśięgá ná kośćiół / przyśięgá náń / y ná tego / który w nim mieƺká ;
Y kto przyśięgá ná niebo / przyśięgá ná ſtolicę Bożą / y ná tego / który śiedźi ná niej.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż dáwáćie dźieśięćinę z miętki y z ányżu y z kminu / á opuƺcżáćie powáżniejƺe rzecży w zákonie / ſąd y miłośierdźie y wiárę ; te rzecży mieliśćie cżynić / á onych nie opuƺcżáć.
Wodzowie ślepi! którzy przecedzáćie komárá / y wielbłądá połykáćie.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż ocżyƺcżáćie kubek z wierzchu y miſę / á wewnątrz pełne ſą drápieſtwá y zbytku.
Fáryzeuƺu ślepy! ocżyść pierwej to / co jeſt wewnątrz w kubku y w miśie / áby y to / co jeſt z wierzchu / cżyſtem było.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iżeśćie podobni grobom pobielánym / które śię zdádzą z wierzchu być cudne / ále wewnątrz pełne ſą kośći umárłych y wƺelákiej niecżyſtośći.
Tákże y wy z wierzchu zdáćie śię być ludźiom ſpráwiedliwi ; ále wewnątrz jeſteśćie pełni obłudy y niepráwośći.
Biádá wám / náucżeni w Piśmie y Fáryzeuƺowie obłudni! iż budujećie groby proroków / y zdobićie nágrobki ſpráwiedliwych /
Y mówićie : Byśmy byli zá dni ojców náƺych / nie bylibyśmy ucżeſnikámi ich we krwi proroków.
A ták świádcżyćie ſámi przećiwko ſobie / żeśćie ſynowie tych / którzy proroki pozábijáli.
Y wy też dopełniáćie miáry ojców wáƺych.
Wężowie! rodzáju jáƺcżurcży! y jákoż będźiećie mogli ujść przed ſądem ogniá piekielnego?
Przetoż oto já poſyłám do wáſ proroki / y mędrce / y náucżone w Piśmie / á z nich niektóre zábijećie y ukrzyżujećie / á niektóre z nich ubicżujećie w bóżnicách wáƺych / y będźiećie je prześládowáć od miáſtá do miáſtá ;
Aby przyƺłá ná wáſ wƺyſtká krew ſpráwiedliwá / wyláná ná źiemi / ode krwi Ablá ſpráwiedliwego áż do krwi Zácháryjáƺá / ſyná Báráchyjáƺowego / któregośćie zábili między kośćiołem y ołtárzem.
Zápráwdę powiádám wám : Przyjdźie to wƺyſtko ná ten náród.
Jeruzálem! Jeruzálem! które zábijáƺ proroki / y które kámionujeƺ te / którzy do ćiebie byli poſyłáni : ilekroć chćiáłem zgromádźić dźieći twoje / ták jáko zgromádzá kokoƺ kurcżętá ſwoje pod ſkrzydłá / á nie chćieliśćie?
Oto wám dom wáƺ puſty zoſtánie.
Abowiem powiádám wám / że mię nie ujrzyćie od tego cżáſu / áż rzecżećie : Błogoſłáwiony / który idźie w imieniu Páńſkim.
