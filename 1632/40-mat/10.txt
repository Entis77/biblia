A zwoławƺy dwunaśći ucżniów ſwojich / dał im moc nád duchy niecżyſtymi / áby je wyganiáli ; y <i>áby</i> uzdrawiáli wƺelką chorobę / y wƺelką niemoc.
A dwunaśći Apoſtołów te ſą imioná ; Pierwƺy Symon / którego zowią Piotr / y Andrzey brát jego / Jákub <i>ſyn</i> Zebedeuƺów / y Jan brát jego :
Filip y Bártłomiey / Thomaƺ / y Mátteuƺ on celnik / Jákub <i>ſyn</i> Alfeuƺów / y Lebbeuƺ názwány Táddeuƺ :
Symon Kánánejcżyk / y Judaƺ Iƺkáryot / który go też wydał.
Tych dwunaśći poſłał JEzus / rozkázując im / y mówiąc ; Ná drogę Pogánów nie záchodźćie / y do miáſtá Sámárytáńcżyków nie wchodźćie :
Ale rácżey idźćie do owiec / które zginęły z domu Izráelſkiego.
A idąc każćie / mówiąc ; Przybliżyło śię króleſtwo niebieſkie.
Chore uzdrawiajćie / trędowáte ocżyśćiajćie / umárłe wſkrzeƺajćie / dyabły wyganiajćie : dármośćie wźięli / dármo dawajćie.
Nie bierzćie z ſobą złotá áni śrebrá / áni miedźi w trzoſy wáƺe.
Ani tájſtry ná drogę / áni dwu ſukien / áni bótów / áni laſki : ábowiem godźien jeſt robotnik żywnośći ſwojey.
A do któregokolwiek miáſtá álbo miáſtecżká wnidźiećie / wywiádujćie śię ktoby w nim <i>tego</i> był godźien : y támże mieƺkajćie / póki nie wynidźiećie.
A wƺedƺy w dom / pozdrówćie go.
A jeſliby on dom tego był godny / niech nań przydźie pokój wáƺ : á jeſliby nie był godny / pokój wáƺ niech śię wróći do was.
A ktoby was nie przyjął áni ſłuchał ſłów wáƺych / wychodząc z domu álbo z miáſtá one° / otrząśnićie proch z nóg wáƺych.
Záprawdę wam powiádam ; Lżey będźie źiemi Sodomſkiey / y Gomorſkiey w dźień ſądny / niżeli miáſtu onemu.
Oto Ja was poſyłam jáko owce miedzy wilki : Bądźćież tedy roſtropnymi jáko wężowie / á ƺcżerymi jáko gołębice.
A ſtrzeżćie śię ludźi : Abowiem was będą wydawáć do Rády / y w Zgromádzeniách ſwojich was bicżowáć będą.
Tákże przed Stároſty y przed Króle wodzeni będźiećie dla mnie / ná świádectwo <i>przećiwko</i> nim y Pogánom.
Ale gdy was podádzą / nie troƺcżćie śię / jáko / y cobyśćie mówili : ábowiem wam dano będźie oneyże godźiny / cobyśćie mówili.
Bo nie wy jeſteśćie którzy mówićie / ále Duch Ojcá wáƺego / który mówi w was.
Y wyda brát brátá ná śmierć / y Oćiec ſyná / y powſtáną dźieći przećiwko rodźicom / y będą je zábijáć.
Y będźiećie w nienawiśći u wƺyſtkich dla Imienia mego : Ale kto wytrwa do końcá / ten będźie zbáwion.
A gdy was prześládowáć będą w tym mieśćie / ućiekajćie do drugiego. Bo záprawdę powiádam wam / że nie obejdźiećie miaſt Izráelſkich / áż przydźie Syn cżłowiecży.
Nie jeſtći ucżeń nád Miſtrzá / áni ſługá nád Páná ſwego.
Doſyć ucżniowi / áby był jáko Miſtrz jego / á ſługá jáko Pan jego : Jeſlić goſpodarzá Beelzebubem názywáli ; cżym więcey domowniki jego <i>názywáć będą</i>.
Przetoż nie bójćie śię ich : Abowiem nic nie jeſt ſkrytego / coby nie miáło bydź objáwiono ; y nic tájemnego / cżegoby śię dowiedźieć nie miano.
Co wam w ćiemnośći mówię / powiádajćie ná świátle : á co w ucho ſłyƺyćie / obwoływajćie ná dáchách.
A nie bójćie śię tych / którzy zábijáją ćiáło / lecż duƺe zábić nie mogą : Ale rácżey bójćie śię tego / który może y duƺę y ćiáło zátráćić w piekielnym ogniu.
Izali dwu wróblików zá pieniążek nie przedáją / á <i>wżdy</i> jeden z nich nie upádnie ná źiemię oprócż <i>woli</i> Ojcá wáƺego.
Náwet y włoſy wƺyſtkie ná głowie wáƺey policżone ſą.
Nie bójćie śię tedy : nád wiele wróblików wy zacniejƺymi jeſteśćie.
Wƺelki tedy któryby mię wyznał przed ludźmi / wyznam go Ja też przed Ojcem mojim / który jeſt w niebieśiech.
A ktoby śię mnie záprzał przed ludźmi / záprzę śię go y Ja przed Ojcem mojim / który jeſt w niebieśiech.
Nie mniemajćie żebym przyƺedł dawáć pokój ná źiemię : nie przyƺedłem dawáć pokoju / ále miecż.
Bom przyƺedł ábym rozerwánie ucżynił miedzy ſynem á ojcem jego / y miedzy córką á mátką jey / tákże miedzy ſynową y świekrą jey.
Y nieprzyjaćiołmi <i>będą</i> cżłowiekowi domownicy jego.
Kto miłuje Ojcá álbo mátkę nád mię / nie jeſt mię godźien : á kto miłuje ſyná álbo córkę nád mię / nie jeſt mię godźien.
A kto nie bierze krzyżá ſwego / y nie idźie zá mną / nie jeſt mię godźien.
Ktoby ználazł duƺę ſwoję / ztráći ją : á ktoby ztráćił duƺę ſwoję dla mnie / znajdźie ją.
Kto was przyjmuje / mnie przyjmuje : á kto mnie przyjmuje / przyjmuje tego który mię poſłał.
Kto przyjmuje Proroká w imieniu Proroká / zapłátę Proroká weźmie : á kto przyjmuje ſpráwiedliwego w imieniu ſpráwiedliwego / ſpráwiedliwego zapłátę weźmie.
Ktoby też nápojił jednego z tych to máłych tylko kubkiem źimney <i>wody</i> w imię ucżniá ; záprawdę powiádam wam / nie ztráći zapłáty ſwojey.
