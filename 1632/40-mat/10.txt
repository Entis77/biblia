A zwołáwƺy dwunáśći ucżniów ſwojich / dał im moc nád duchy niecżyſtymi / áby je wygániáli ; y áby uzdráwiáli wƺelką chorobę / y wƺelką niemoc.
A dwunáśći Apoſtołów te ſą imioná ; Pierwƺy Szymon / którego zowią Piotr / y Andrzej brát jego ; Jákób <i>ſyn</i> Zebedeuƺów / y Ján brát jego :
Filip y Bártłomiey / Thomáƺ / y Máteuƺ on celnik / Jákób ſyn Alfeuƺów / y Lebeuƺ názwány Tádeuƺ :
Szymon Kánánejcżyk / y Judáƺ Iƺkáryjot / który go też wydał.
Tych dwunáśći poſłał JEzus / rozkázując im / y mówiąc ; Ná drogę pogánów nie záchodźćie / y do miáſtá Sámárytáńcżyków nie wchodźćie :
Ale rácżey idźćie do owiec / które zginęły z domu Izráelſkiego.
A idąc káżćie / mówiąc ; Przybliżyło śię króleſtwo niebieſkie.
Chore uzdráwiájćie / trędowáte ocżyƺcżájćie / umárłe wſkrzeƺájćie / dyjábły wygániájćie : dármośćie wźięli / dármo dáwájćie.
Nie bierzćie z ſobą złotá áni ſrebrá / áni miedźi w trzoſy wáƺe.
Ani táiſtry ná drogę / áni dwu ſukien / áni butów / áni láſki : ábowiem godźien jeſt robotnik żywnośći ſwojey.
A do któregokolwiek miáſtá álbo miáſtecżká wnijdźiećie / wywiádujćie śię ktoby w nim <i>tego</i> był godźien : á támże mieƺkájćie / póki nie wynijdźiećie.
A wƺedƺy w dom / pozdrówćie go.
A jeſliby on dom tego był godny / niech nań przydźie pokój wáƺ : á jeſliby nie był godny / pokój wáƺ niech śię wróći do was.
A ktoby was nie przyjął áni ſłuchał ſłów wáƺych / wychodząc z domu álbo z miáſtá onego / otrząśnijćie proch z nóg wáƺych.
Zápráwdę wam powiádám ; Lżey będźie źiemi Sodomſkiey y Gomorſkiey w dźień ſądny / niżeli miáſtu onemu.
Oto Ja was poſyłám jáko owce miedzy wilki : bądźćież tedy roztropnymi jáko wężowie / á ƺcżerymi jáko gołębice.
A ſtrzeżćie śię ludźi : ábowiem was będą wydáwáć do rády / y w zgromádzeniách ſwojich was bicżowáć będą.
Tákże przed ſtároſty y przed króle wodzeni będźiećie dla mnie / ná świádectwo przećiwko nim y pogánom.
Ale gdy was podádzą / nie troƺcżćie śię / jáko / y cobyśćie mówili : ábowiem wam dáno będźie onejże godźiny / cobyśćie mówili.
Bo wy nie jeſteśćie którzy mówićie / ále duch Ojcá wáƺego / który mówi w was.
Y wydá brát brátá ná śmierć / y oćiec ſyná y powſtáną dźieći przećiwko rodźicom / y będą je zábijáć.
Y będźiećie w nienáwiśći u wƺyſtkich dla Imienia mego : Ale kto wytrwá do końcá / ten będźie zbáwion.
A gdy was prześládowáć będą w tym mieśćie / ućiekájćie do drugiego. Bo zápráwdę powiádám wam / że nie obejdźiećie miaſt Izráelſkich / áż przydźie Syn cżłowiecży.
Nie jeſtći ucżeń nád Miſtrzá / áni ſługá nád Páná ſwego.
Doſyć ucżniowi / áby był jáko Miſtrz jego / á ſługá jáko Pán jego : jeſlić goſpodárzá Beelzebubem názywáli / cżym więcey domowniki jego <i>názywáć będą</i>.
Przetoż niebójćie śię ich : ábowiem nic nie jeſt ſkrytego / coby nie miáło być objáwiono ; y nic tájemnego / cżegoby śię dowiedźieć nie miáno.
Co wam w ćiemnośći mówię / powiádájćie ná świátle ; á co w ucho ſłyƺyćie / obwoływájćie ná dáchách.
A nie bójćie śię tych / którzy zábijáją ćiáło / lecż duƺę zábić nie mogą : Ale rácżey bójćie śię tego / który może y duƺę y ćiáło zátráćić w piekielnym ogniu.
Izali dwu wróblików zá pieniążek nieprzedáją / á <i>wżdy</i> jeden z nich nie upádnie ná źiemię oprócż <i>woli</i> Ojcá wáƺego.
Náwet y włoſy wƺyſtkie ná głowie wáƺej policżone ſą.
Nie bójćie śię tedy : nád wiele wróblików wy zácniejƺymi jeſteśćie.
Wƺelki tedy któryby mię wyznał przed ludźmi / wyznam go Ja też przed Ojcem moim / który jeſt w niebieśiech.
A ktoby śię mnie zápárł przed ludźmi / záprę śię go y Ja przed Ojcem moim / który jeſt w niebieśiech.
Nie mniemájćie żebym przyƺedł dáwáć pokój ná źiemię : nie przyƺedłem dáwáć pokoju / ále miecż.
Bom przyƺedł ábym rozerwánie ucżynił miedzy ſynem á ojcem jego / y miedzy córką á mátką jey / tákże miedzy ſynową y świekrą jey.
Y nieprzyjáćiołmi <i>będą</i> cżłowiekowi domownicy jego.
Kto miłuje ojcá álbo mátkę nád mię / nie jeſt mię godźien : á kto miłuje ſyná álbo córkę nád mię / nie jeſt mię godźien.
A kto nie bierze krzyżá ſwego / y nie idźie zá mną / nie jeſt mię godźien.
Ktoby ználázł duƺę ſwoję / ſtráći ją : á ktoby ſtráćił duƺę ſwoję dla mnie / znájdźie ją.
Kto was przyjmuje / mnie przyjmuje : á kto mnie przyjmuje / przyjmuje tego który mię poſłał.
Kto przyjmuje proroká w imieniu proroká / zápłátę proroká weźmie : á kto przyjmuje ſpráwiedliwego w imieniu ſpráwiedliwego / ſpráwiedliwego zápłátę weźmie.
Ktoby też nápoił jednego z tych to máłych tylko kubkiem źimney <i>wody</i> w imię ucżniá ; zápráwdę powiádám wam / nie ſtráći zápłáty ſwojey.
