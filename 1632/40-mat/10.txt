A zwołáwƺy dwunáſtu ucżniów ſwoich / dáł im moc nád duchy niecżyſtymi / áby je wygániáli / y áby uzdráwiáli wƺelką chorobę / y wƺelką niemoc.
A dwunáſtu Apoſtołów te ſą imioná : Pierwƺy Szymon / którego zowią Piotr / y Andrzej / brát jego ; Jákób / ſyn Zebedeuƺá / y Ján / brát jego ;
Filip y Bártłomiej / Tomáƺ y Máteuƺ on celnik / Jákób / ſyn Alfeuƺá / y Lebeuƺ / názwány Tádeuƺ ;
Szymon Kánánejcżyk / y Judáƺ Iƺkáryjot / który go też wydáł.
Tych dwunáſtu poſłáł Jezuſ / rozkázując im y mówiąc : Ná drogę pogánów nie záchodźćie / y do miáſtá Sámárytáńcżyków nie wchodźćie ;
Ale rácżej idźćie do owiec / które zginęły z domu Izráelſkiego ;
A idąc káżćie / mówiąc : Przybliżyło śię króleſtwo niebieſkie.
Chore uzdráwiájćie / trędowáte ocżyƺcżájćie / umárłe wſkrzeƺájćie / dyjábły wygániájćie ; dármośćie wzięli / dármo dáwájćie.
Nie bierzćie z ſobą złotá / áni ſrebrá / áni miedzi w trzoſy wáƺe ;
Ani táiſtry ná drogę / áni dwóch ſukien / áni butów / áni láſki ; ábowiem godzien jeſt robotnik żywnośći ſwojej.
A do któregokolwiek miáſtá álbo miáſtecżká wnijdziećie / wywiádujćie śię / ktoby w nim tego był godzien / á támże mieƺkájćie / póki nie wynijdziećie ;
A wƺedłƺy w dom / pozdrówćie go.
A jeźliby on dom tego był godny / niech náń przyjdzie pokój wáƺ ; á jeźliby nie był godny / pokój wáƺ niech śię wróći do wáſ.
A ktoby wáſ nie przyjął / áni ſłucháł ſłów wáƺych / wychodząc z domu álbo z miáſtá onego / otrząśnijćie proch z nóg wáƺych.
Zápráwdę wám powiádám : Lżej będzie ziemi Sodomſkiej y Gomorſkiej w dzień ſądny / niżeli miáſtu onemu.
Oto Já wáſ poſyłám jáko owce między wilki ; bądźćież tedy roztropnymi jáko węże / á ƺcżerymi jáko gołębice /
A ſtrzeżćie śię ludzi ; ábowiem wáſ będą wydáwáć do rády / y w zgromádzeniách ſwoich wáſ bicżowáć będą.
Tákże przed ſtároſty y przed króle wodzeni będziećie dlá mnie / ná świádectwo przećiwko nim y pogánom.
Ale gdy wáſ podádzą / nie troƺcżćie śię / jáko y cobyśćie mówili ; ábowiem wám dáno będzie onejże godziny / cobyśćie mówili ;
Bo wy nie jeſteśćie / którzy mówićie / ále duch Ojcá wáƺego / który mówi w wáſ.
Y wydá brát brátá ná śmierć / y ojćiec ſyná / y powſtáną dzieći przećiwko rodzicom / y będą je zábijáć.
Y będziećie w nienáwiśći u wƺyſtkich dlá imieniá mego ; ále kto wytrwá do końcá / ten będzie zbáwion.
A gdy wáſ prześládowáć będą w tem mieśćie / ućiekájćie do drugiego ; bo zápráwdę powiádám wám / że nie obejdziećie miáſt Izráelſkich / áż przyjdzie Syn cżłowiecży.
Nie jeſtći ucżeń nád miſtrzá / áni ſługá nád Páná ſwego ;
Doſyć ucżniowi / áby był jáko miſtrz jego / á ſługá jáko Pán jego ; jeźlić goſpodárzá Beelzebubem názywáli / cżem więcej domowniki jego názywáć będą.
Przetoż nie bójćie śię ich ; ábowiem nic nie jeſt ſkrytego / coby nie miáło być objáwiono / y nic tájemnego / cżegoby śię dowiedzieć nie miáno.
Co wám w ćiemnośći mówię / powiádájćie ná świátle ; á co w ucho ſłyƺyćie / obwoływájćie ná dáchách ;
A nie bójćie śię tych / którzy zábijáją ćiáło / lecż duƺy zábić nie mogą ; ále rácżej bójćie śię tego / który może y duƺę y ćiáło zátráćić w piekielnym ogniu.
Izáli dwóch wróbelków zá pieniążek nie ſprzedáją / á wżdy jeden z nich nie upádnie ná ziemię oprócż woli Ojcá wáƺego.
Náwet y włoſy wƺyſtkie ná głowie wáƺej policżone ſą.
Nie bójćie śię tedy ; nád wiele wróbelków wy zácniejƺymi jeſteśćie.
Wƺelki tedy / któryby mię wyznáł przed ludźmi / wyznám go Já też przed Ojcem moim / który jeſt w niebieśiech ;
A ktoby śię mnie zápárł przed ludźmi / záprę śię go y Já przed Ojcem moim / który jeſt w niebieśiech.
Nie mniemájćie / żem przyƺedł dáwáć pokój ná ziemię ; nie przyƺedłem dáwáć pokoju / ále miecż.
Bom przyƺedł / ábym rozerwánie ucżynił między ſynem á ojcem jego / y między córką á mátką jej / tákże między ſynową y świekrą jej ;
Y nieprzyjáćiołmi będą cżłowiekowi domownicy jego.
Kto miłuje ojcá álbo mátkę nád mię / nie jeſt mię godzien ; á kto miłuje ſyná álbo córkę nád mię / nie jeſt mię godzien ;
A kto nie bierze krzyżá ſwego / y nie idzie zá mną / nie jeſt mię godzien.
Ktoby ználázł duƺę ſwoję / ſtráći ją ; á ktoby ſtráćił duƺę ſwoję dlá mnie / znájdzie ją.
Kto wáſ przyjmuje / mnie przyjmuje ; á kto mnie przyjmuje / przyjmuje tego / który mię poſłáł.
Kto przyjmuje proroká w imieniu proroká / zápłátę proroká weźmie ; á kto przyjmuje ſpráwiedliwego w imieniu ſpráwiedliwego / ſpráwiedliwego zápłátę weźmie.
Ktoby też nápoił jednego z tych to máłych tylko kubkiem zimnej wody w imię ucżniá / zápráwdę powiádám wám / nie ſtráći zápłáty ſwojej.
