Oney godźiny przyſtąpili ucżniowie do JEzuſá / mówiąc : Któż wżdy nájwiękƺy jeſt w króleſtwie niebieſkiem?
A záwołáwƺy JEzus dźiećięćiá / poſtáwił je w pośrzodku ich /
Y rzekł : Zápráwdę powiádám wam : Jeſli śię nie náwróćićie y nie ſtániećie śię jáko dźieći / żádnym ſpoſobem nie wnijdźiećie do króleſtwá niebieſkiego.
Kto śię tedy uniży jáko to dźiećię / tenći jeſt nájwiękƺym w króleſtwie niebieſkiem.
A ktoby przyjął jedno dźiećię tákie w imieniu mojim / mnie przyjmuje.
Ktoby záś zgorƺył jednego z tych máłych / którzy we mię wierzą / pożytecżniejby mu było / áby záwieƺony był kámień młyńſki ná ƺyi jego / á utopiony był w głębokośći morſkiej.
Biádá świátu dla zgorƺeniá! ábowiem muƺą zgorƺeniá przyść ; wƺákże biádá cżłowiekowi onemu / przez którego przychodźi zgorƺenie!
Przetoż jeſli ręká twojá álbo nogá twojá gorƺy ćię / odetnij ją y zárzuć od śiebie ; lepiej jeſt tobie / wnijść do żywotá chromym álbo ułomnym / niżeli dwie ręce álbo dwie nogi májąc / wrzuconym być do ogniá wiecżnego.
A jeſli ćię oko twoje gorƺy / wyłup je y zárzuć od śiebie ; lepiej jeſt tobie jednookim wnijść do żywotá / niżeli obá ocży májąc / być wrzuconym do ogniá piekielnego.
Pátrzájćież / ábyśćie nie gárdźili żádnym z tych málucżkich ; ábowiem wam powiádám / iż Aniołowie ich w niebieśiech záwƺe pátrzą ná oblicże Ojcá mojego / który jeſt w niebieśiech.
Przyƺedł bowiem Syn cżłowiecży / áby zbáwił to / co było zginęło.
Co śię wam zda? Gdyby który cżłowiek miał ſto owiec / á zábłąkáłáby śię jedná z nich / ázaż nie zoſtáwiá onych dźiewięćdźieśięćiu y dźiewięćiu / á poƺedƺy ná góry / nie ƺuká zbłąkáney?
A jeſli mu śię zdárzy / ználeźć ją / zápráwdę powiádám wam / że śię z niej bárdźiej ráduje / niż z onych dźiewięćdźieśięćiu y dźiewięćiu niezbłąkánych.
Ták nie jeſt wolá Ojcá wáƺego / który jeſt w niebieśiech / áby zginął jeden z tych málucżkich.
A jeſliby zgrzeƺył przećiwko tobie brát twój / idź / ſtrofuj go miedzy tobą y onym ſámym : jeſli ćię uſłuchá / pozyſkáłeś brátá twego.
Ale jeſli ćię nie uſłuchá / przybierz do śiebie jeƺcże jednego álbo dwu / áby w uśćiech dwu álbo trzech świádków ſtánęło káżde ſłowo.
A jeſliby ich nie uſłuchał / powiedz zborowi ; á jeſliby zboru nie uſłuchał / niechći będźie jáko pogánin y celnik.
Zápráwdę powiádám wam : Cobyśćiekolwiek związáli ná źiemi / będźie związáne y ná niebie ; á cobyśćie rozwiązáli ná źiemi ; będźie rozwiązáne y ná niebie.
Záśię powiádám wam : Iż gdyby śię z was dwáj zgodźili ná źiemi o wƺelką rzecż / o którąby prośili / ſtánie śię im od Ojcá mego / który jeſt w niebieśiech.
Abowiem gdźie ſą dwáj álbo trzej zgromádzeni w imię moje / támem jeſt w pośrzodku ich.
Tedy przyſtąpiwƺy do niego Piotr / rzekł : Pánie! wielekroć zgrzeƺy przećiwko mnie brát mój / á odpuƺcżę mu? cżyż áż do śiedmiu kroć?
Y rzekł mu JEzus : Nie mówięć áż do śiedmiu kroć / ále áż do śiedmdźieśiąt śiedmiu kroć.
Dla tego podobne jeſt króleſtwo niebieſkie cżłowiekowi królowi / który śię chćiał ráchowáć z ſługámi ſwymi.
A gdy śię pocżął ráchowáć / ſtáwiono mu jednego / który był winien dźieśięć tyśięcy tálentów.
A gdy niemiał ſkąd oddáć / kázał go pán jego záprzedáć / y żonę jego / y dźieći / y wƺyſtko / co miał / y dług oddáć.
Upádƺy tedy ſługá on / pokłonił mu śię / mówiąc : Pánie! miej ćierpliwość nádemną / á wƺyſtkoć oddam.
A użáliwƺy śię pán onego ſługi / uwolnił go / y dług mu odpuśćił.
A wyƺedƺy on ſługá / ználázł jednego z ſpółſług ſwojich / który mu był winien ſto groƺy ; á porwáwƺy go / duśił go / mówiąc : Oddáj mi / coś winien.
Przypádƺy tedy on ſpółſługá jego do nóg jego / prośił go / mówiąc : Miej ćierpliwość náde mną / á oddam ći wƺyſtko.
Lecż on nie chćiał / ále ƺedƺy wrzućił go do więźieniá / áżby oddał / co był winien.
Ujrzáwƺy tedy ſpółſłudzy jego / co śię ſtáło / záſmućili śię bárzo / á ƺedƺy oznájmili pánu ſwemu wƺyſtko / co śię ſtáło.
Tedy záwołáwƺy go pán jego / rzekł mu : Sługo zły! wƺyſtek on dług odpuśćiłem ći / żeś mię prośił ;
Azáżeś śię y ty nie miał zmiłowáć nád ſpółſługą twoim / jákom śię y ja zmiłował nád tobą?
A rozgniewáwƺy śię pán jego / podał go kátom / áżby oddał to wƺyſtko / co mu był winien.
Ták y Oćiec mój niebieſki ucżyni wam / jeſli nie odpuśćićie káżdy brátu ſwemu z ſerc wáƺych upádków ich.
