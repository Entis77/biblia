Oney godźiny / przyſtąpili ucżniowie do JEzuſá / mówiąc : Któż wżdy nawiękƺy jeſt w króleſtwie niebieſkim?
A záwoławƺy JEzus dźiećięćiá / poſtáwił je w pośrzodku ich /
Y rzekł : Záprawdę powiádam wam / Jeſli śię nie náwróćićie / y nie ſtániećie śię jáko dźieći / żadnym ſpoſobem nie wnidźiećie do króleſtwá niebieſkiego.
Kto śię tedy uniży jáko to dźiećię / tenći jeſt nawiękƺym w króleſtwie niebieſkim.
A ktoby przyjął jedno dźiećię tákie w Imieniu mojim / mnie przyjmuje.
Ktoby záś zgorƺył jednego z tych máłych którzy w mię wierzą / pożytecżniey by mu było / áby záwieƺony był kámień młyńſki ná ƺyjey jego / á utopiony był w głębokośći morſkiey.
Biádá świátu dla zgorƺenia ; ábowiem muƺą zgorƺenia przyść : wƺákże biádá cżłowiekowi onemu / przez którego przychodźi zgorƺenie.
Przetoż jeſli ręká twojá / álbo nogá twojá gorƺy ćię / odetni ją y zárzuć od śiebie : lepiey jeſt tobie wniść do żywotá chromym álbo ułomnym / niżeli dwie ręce álbo dwie nodze májąc / wrzuconym być do ogniá wiecżnego.
A jeſli ćię oko twoje gorƺy / wyłup je / y zárzuć od śiebie : lepiey jeſt tobie jednookim wniść do żywotá / niżeli obie ocży májąc / być wrzuconym do ogniá piekielnego.
Pátrzajćież ábyśćie nie gárdźili żadnym z tych málucżkich. Abowiem wam powiádam : Iż Anjołowie ich w niebieśiech záwƺe pátrzą ná oblicże Ojcá mojego / który jeſt w niebieśiech.
Przyƺedł bowiem Syn cżłowiecży áby zbáwił to co było zginęło.
Co śię wam zda? Gdyby który cżłowiek miał ſto owiec / á obłąkáłáby śię jedná z nich / ázaż nie zoſtawia onych dźiewiąćidźieśiąt y dźiewiąći / á poƺedƺy ná góry nie ƺuka obłąkáney?
A jeſli mu śię zdárzy ználeść ją / záprawdę powiádam wam / że śię z niey bárźiey ráduje niż z onych dźiewiąćidźieśiąt dźiewiąći nie obłąkánych.
Ták nie jeſt wola Ojcá wáƺego który jeſt w niebieśiech / áby zginął jeden z tych málucżkich.
A jeſliby zgrzeƺył przećiwko tobie brát twój : idź / ſtrofuj go miedzy tobą y onym ſámym : jeſli ćię uſłucha / pozyſkałeś brátá twego.
Ale jeſli ćię nie uſłucha / przybierz do śiebie jeƺcże jednego álbo dwu / áby w uśćiech dwu álbo trzech świádków / ſtánęło káżde ſłowo.
A jeſliby ich nie uſłuchał / powiedz Zborowi ; á jeſliby Zboru nie uſłuchał / niechći będźie jáko pogánin y celnik.
Záprawdę powiádam wam ; Cobyśćiekolwiek związáli ná źiemi / będźie związano y ná niebie ; á cobyśćie rozwiązáli ná źiemi / będźie rozwiązano y ná niebie.
Záśię powiádam wam ; Iż gdyby śię z was dwá zgodźili ná źiemi o wƺelką rzecż o którąby prośili / ſtánie śię im od Ojcá mego który jeſt w niebieśiech.
Abowiem gdźie ſą dwá álbo trzej zgromádzeni w Imię moje / támem jeſt w pośrzodku ich.
Tedy przyſtąpiwƺy do niego Piotr / rzekł ; PAnie wiele kroć zgrzeƺy przećiwko mnie brát mój / á odpuƺcżę mu? <i>cży</i> áż do śiedmi kroć?
<i>Y</i> rzekł mu JEzus : Nie mówięć áż do śiedmi kroć ; ále áż do śiedmidźieśiąt śiedmi kroć.
Dla tego / Podobne jeſt króleſtwo niebieſkie cżłowiekowi królowi / który śię chćiał ráchowáć z ſługámi ſwymi.
A gdy śię pocżął ráchowáć / ſtáwiono mu jednego / <i>który</i> był winien dźieśięć tyśięcy talentów.
A gdy niemiał <i>z kąd</i> oddáć / kazał go Pan jego záprzedáć / y żonę jego / y dźieći / y wƺyſtko co miał ; y <i>dług</i> oddáć.
Upadƺy tedy ſługá on / pokłonił mu śię / mówiąc ; Pánie / miey ćierpliwość náde mną / á wƺyſtkoć oddam.
A użaliwƺy śię Pan onego ſługi / uwolnił go ; y dług mu odpuśćił.
A wyƺedƺy on ſługá / ználazł jednego z ſpół-ſług ſwojich / który mu był winien ſto groƺy : á porwawƺy go / duśił go mówiąc ; Oddaj mi coś winien.
Przypadƺy tedy on ſpół-ſługá jego do nóg jego / prośił go / mówiąc ; Miey ćierpliwość náde mną / á oddamći wƺyſtko.
Lecż on nie chćiał : Ale ƺedƺy wrzućił go do więźienia áżby oddał co był winien.
Ujrzawƺy tedy ſpół-ſłudzy jego co śię ſtáło / záſmućili śię bárzo : á ƺedƺy oznájmili Pánu ſwemu / wƺyſtko co śię ſtáło.
Tedy záwoławƺy go Pan jego / rzekł mu : Sługo zły / wƺyſtek on dług odpuśćiłemći żeś mię prośił :
Azażeś śię y ty nie miał zmiłowáć nád ſpół-ſługą twojim / jákom śię y ja zmiłował nád tobą?
A rozgniewawƺy śię Pan jego / podał go kátom / áżby oddał to wƺyſtko co mu był winien.
Ták y Oćiec mój niebieſki ucżyni wam / jeſli nie odpuśćićie káżdy brátu ſwemu z ſerc wáƺych / upadków ich.
