Onej godziny przyſtąpili ucżniowie do Jezuſá / mówiąc : Któż wżdy nájwiękƺy jeſt w króleſtwie niebieſkiem?
A záwołáwƺy Jezuſ dziećięćiá / poſtáwił je w pośrodku ich /
Y rzekł : Zápráwdę powiádám wám : Jeźli śię nie náwróćićie y nie ſtániećie śię jáko dzieći / żádnym ſpoſobem nie wnijdziećie do króleſtwá niebieſkiego.
Kto śię tedy uniży jáko to dziećię / tenći jeſt nájwiękƺym w króleſtwie niebieſkiem.
A ktoby przyjął jedno dziećię tákie w imieniu mojem / mnie przyjmuje.
Ktoby záś zgorƺył jednego z tych máłych / którzy we mię wierzą / pożytecżniejby mu było / áby záwieƺony był kámień młyńſki ná ƺyi jego / á utopiony był w głębokośći morſkiej.
Biádá świátu dlá zgorƺeniá! ábowiem muƺą zgorƺeniá przyjść ; wƺákże biádá cżłowiekowi onemu / przez którego przychodzi zgorƺenie!
Przetoż jeźli ręká twojá álbo nogá twojá gorƺy ćię / odetnij ją y zárzuć od śiebie ; lepiej jeſt tobie / wnijść do żywotá chromym álbo ułomnym / niżeli dwie ręce álbo dwie nogi májąc / wrzuconym być do ogniá wiecżnego.
A jeźli ćię oko twoje gorƺy / wyłup je y zárzuć od śiebie ; lepiej jeſt tobie jednookim wnijść do żywotá / niżeli obá ocży májąc / być wrzuconym do ogniá piekielnego.
Pátrzájćież / ábyśćie nie gárdzili żádnym z tych málucżkich ; ábowiem wám powiádám / iż Aniołowie ich w niebieśiech záwƺe pátrzą ná oblicże Ojcá mojego / który jeſt w niebieśiech.
Przyƺedł bowiem Syn cżłowiecży / áby zbáwił to / co było zginęło.
Co śię wám zdá? Gdyby który cżłowiek miáł ſto owiec / á zábłąkáłáby śię jedná z nich / ázáż nie zoſtáwiá onych dziewięćdzieśięćiu y dziewięćiu / á poƺedłƺy ná góry / nie ƺuká zbłąkánej?
A jeźli mu śię zdárzy / ználeźć ją / zápráwdę powiádám wám / że śię z niej bárdziej ráduje / niż z onych dziewięćdzieśięćiu y dziewięćiu niezbłąkánych.
Ták nie jeſt wolá Ojcá wáƺego / który jeſt w niebieśiech / áby zginął jeden z tych málucżkich.
A jeźliby zgrzeƺył przećiwko tobie brát twój / idź / ſtrofuj go między tobą y onym ſámym : jeźli ćię uſłuchá / pozyſkáłeś brátá twego.
Ale jeźli ćię nie uſłuchá / przybierz do śiebie jeƺcże jednego álbo dwóch / áby w uśćiech dwóch álbo trzech świádków ſtánęło káżde ſłowo.
A jeźliby ich nie uſłucháł / powiedz zborowi ; á jeźliby zboru nie uſłucháł / niechći będzie jáko pogánin y celnik.
Zápráwdę powiádám wám : Cobyśćiekolwiek związáli ná ziemi / będzie związáne y ná niebie ; á cobyśćie rozwiązáli ná ziemi ; będzie rozwiązáne y ná niebie.
Záśię powiádám wám : Iż gdyby śię z wáſ dwáj zgodzili ná ziemi o wƺelką rzecż / o którąby prośili / ſtánie śię im od Ojcá mego / który jeſt w niebieśiech.
Abowiem gdzie ſą dwáj álbo trzej zgromádzeni w imię moje / támem jeſt w pośrodku ich.
Tedy przyſtąpiwƺy do niego Piotr / rzekł : Pánie! wielekroć zgrzeƺy przećiwko mnie brát mój / á odpuƺcżę mu? cżyż áż do śiedmiu kroć?
Y rzekł mu Jezuſ : Nie mówięć áż do śiedmiu kroć / ále áż do śiedmdzieśiąt śiedmiu kroć.
Dlátego podobne jeſt króleſtwo niebieſkie cżłowiekowi królowi / który śię chćiáł ráchowáć z ſługámi ſwymi.
A gdy śię pocżął ráchowáć / ſtáwiono mu jednego / który był winien dzieśięć tyśięcy tálentów.
A gdy niemiáł ſkąd oddáć / kázáł go pán jego záprzedáć / y żonę jego / y dzieći / y wƺyſtko / co miáł / y dług oddáć.
Upádłƺy tedy ſługá on / pokłonił mu śię / mówiąc : Pánie! miej ćierpliwość nádemną / á wƺyſtkoć oddám.
A użáliwƺy śię pán onego ſługi / uwolnił go / y dług mu odpuśćił.
A wyƺedłƺy on ſługá / ználázł jednego z ſpółſług ſwoich / który mu był winien ſto groƺy ; á porwáwƺy go / duśił go / mówiąc : Oddáj mi / coś winien.
Przypádłƺy tedy on ſpółſługá jego do nóg jego / prośił go / mówiąc : Miej ćierpliwość náde mną / á oddám ći wƺyſtko.
Lecż on nie chćiáł / ále ƺedłƺy wrzućił go do więzieniá / áżby oddáł / co był winien.
Ujrzáwƺy tedy ſpółſłudzy jego / co śię ſtáło / záſmućili śię bárdzo / á ƺedłƺy oznájmili pánu ſwemu wƺyſtko / co śię ſtáło.
Tedy záwołáwƺy go pán jego / rzekł mu : Sługo zły! wƺyſtek on dług odpuśćiłem ći / żeś mię prośił ;
Azáżeś śię y ty nie miáł zmiłowáć nád ſpółſługą twoim / jákom śię y já zmiłowáł nád tobą?
A rozgniewáwƺy śię pán jego / podáł go kátom / áżby oddáł to wƺyſtko / co mu był winien.
Ták y Ojćiec mój niebieſki ucżyni wám / jeźli nie odpuśćićie káżdy brátu ſwemu z ſerc wáƺych upádków ich.
