Abowiem podobne jeſt króleſtwo niebieſkie cżłowiekowi goſpodárzowi / który wyƺedł bárzo ráno nájmowáć robotników do winnicy ſwojey.
A zmówiwƺy śię z robotnikámi z groƺá ná dźień poſłał je do winnicy ſwojey.
A wyƺedƺy o trzećiej godźinie / ujrzał drugich / którzy ſtáli ná rynku próżnujący ;
Y rzekł im : Idźćie y wy do winnicy / á co będźie ſpráwiedliwego / dam wam.
A oni poƺli. Záśię wyƺedƺy o ƺóſtey y dźiewiątey godźinie / tákże ucżynił.
Potem o jedenáſtey godźinie wyƺedƺy / ználázł drugie / którzy ſtáli próżnujący / y rzekł im : Przecż tu ſtoićie cáły dźień próżnujący?
Rzekli mu : Iż nas nikt nie nájął ; y rzekł im : Idźćie y wy do winnicy / á co będźie ſpráwiedliwego / weźmiećie.
A gdy był wiecżór / rzekł pán winnicy ſpráwcy ſwemu : Záwołáj robotników / á oddáj im zápłátę / pocżąwƺy od oſtátnich áż do pierwƺych.
A gdy przyƺli oni / którzy o jedenáſtey godźinie byli nájęći / wźiął káżdy z nich po groƺu.
Przyƺedƺy też y pierwśi / mniemáli / że więcey wezmą ; ále wźięli y oni / káżdy z nich / po groƺu.
A wźiąwƺy / ƺemráli przećiwko goſpodárzowi /
Mówiąc : Ci oſtátni jednę godźinę robili / á ucżyniłeś je nam równymi / którzyśmy znośili ćiężár dniá y upálenie.
A on odpowiádájąc rzekł jednemu z nich : Przyjáćielu! nie cżynięć krzywdy ; ázaż śię nie z groƺá zmówił zemną?
Weźmij / co twojego jeſt / á idź ; chcę bowiem temu oſtátniemu dáć jáko y tobie.
Azaż mi śię nie godźi cżynić z mojim / co chcę? Cżyli oko twoje złośliwe jeſt / iżem ja jeſt dobry?
Tákći będą oſtátni pierwƺymi / á pierwśi oſtátnimi ; ábowiem wiele jeſt wezwánych / ále máło wybránych.
A wſtępując JEzus do Jeruzálemu / wźiął z ſobą dwánáśćie ucżniów ná oſobne miejſce w drodze / y rzekł im :
Oto wſtępujemy do Jeruzálemu / á Syn cżłowiecży będźie wydány przedniejƺym kápłánom y náucżonym w Piśmie / y oſądzą go ná śmierć.
Y wydádzą go pogánom ná pośmiewánie y ná ubicżowánie y ná ukrzyżowánie ; ále trzećiego dniá zmartwychwſtánie.
Tedy przyſtąpiłá do niego mátká ſynów Zebedeuƺowych z ſynámi ſwoimi / kłániájąc mu śię / y proƺąc nieco od niego.
A on jey rzekł ; Cżegóż chceƺ? Rzekłá mu : Rzecż / áby śiedźieli ći dwáj ſynowie moi / jeden po práwicy twojey á drugi po lewicy w króleſtwie twojim.
Ale JEzus odpowiádájąc rzekł : Nie wiećie / o co prośićie ; możećież pić kielich / który ja będę pił? y chrztem / którym śię ja chrzcżę / być ochrzcżeni? Rzekli mu : Możemy.
Tedy im rzekł : Kielichći mój pić będźiećie / y chrztem / którym śię ja chrzcżę / ochrzcżeni będźiećie ; ále śiedźieć po práwicy mojey y po lewicy mojey / nie jeſt mojá rzecż dáć wam / ále tym / którym jeſt zgotowano od Ojcá mojego.
A uſłyƺáwƺy to oni dźieśięćiu / rozgniewáli śię ná onych dwu bráći.
Ale JEzus zwołáwƺy ich / rzekł : Wiećie / iż kśiążętá národów pánują nád nimi / á którzy wielcy ſą / mocy dokázują nád nimi.
Lecż nie ták będźie miedzy wámi : ále ktobykolwiek miedzy wámi chćiał być wielkim / niech będźie ſługą wáƺym.
A ktobykolwiek miedzy wámi chćiał być pierwƺym / niech będźie ſługą wáƺym.
Jáko y Syn cżłowiecży nie przyƺedł / áby mu ſłużono / ále áby ſłużył / y áby dał duƺę ſwą ná okup zá wielu.
A gdy oni wychodźili z Jerychá / ƺedł zá nim wielki lud.
A oto dwáj ślepi / śiedzący przy drodze / uſłyƺáwƺy / iż JEzus przechodźił / záwołáli / mówiąc : Zmiłuj śię nád námi / Pánie / ſynu Dawidowy!
Ale on lud gromił ich / áby milcżeli ; lecż oni tym więcey wołáli / mówiąc : Zmiłuj śię nád námi / Pánie / ſynu Dawidowy!
A záſtánowiwƺy śię JEzus / záwołał ich y rzekł : Cóż chcećie / ábym wam ucżynił?
Rzekli mu : Pánie! áby były otworzone ocży náƺe.
A użáliwƺy śię ich JEzus / dotknął śię ocżu ich / á záráz przejrzáły ocży ich ; y ƺli zá nim.
