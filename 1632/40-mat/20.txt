Abowiem podobne jeſt króleſtwo niebieſkie cżłowiekowi goſpodarzowi / który wyƺedł bárzo ráno nájmowáć robotniki do winnice ſwojey.
A zmówiwƺy śię z robotnikámi z groƺá ná dźień / poſłał je do winnice ſwojey.
A wyƺedƺy o trzećiey godźinie / ujrzał drugie / którzy ſtali ná rynku próznujący /
Y rzekł im : Idźćie y wy do winnice ; á co będźie ſpráwiedliwego / dam wam.
A oni poƺli. Záśię wyƺedƺy o ƺóſtey y dźiewiątey godźinie / tákże ucżynił.
Potym o jedenaſtey godźinie wyƺedƺy / ználazł drugie którzy ſtali próznujący / y rzekł im : Przecż tu ſtojićie cáły dźień próznujący?
Rzekli mu : Iż nas nikt nie nájął : <i>Y</i> rzekł im ; Idźćie y wy do winnice : á co będźie ſpráwiedliwego / weźmiećie.
A gdy był wiecżór / rzekł Pan winnice ſprawcy ſwemu ; Záwołaj robotników / á oddaj im zapłátę pocżąwƺy od oſtátnich / áż do pierwƺych.
A gdy przyƺli oni którzy o jedenaſtey godźinie <i>byli nájęći,</i> wźiął káżdy z nich po groƺu.
Przyƺedƺy też y pierwƺy / mniemáli że więcey wezmą : ále wźięli y oni káżdy z nich po groƺu.
A wźiąwƺy / ƺemráli przećiwko goſpodarzowi /
Mówiąc : Ci oſtátni jednę godźinę robili / á ucżyniłeś je nam równymi / którzyſmy znośili ćiężar dniá / y upalenie.
A on odpowiádájąc / rzekł jednemu z nich : Przyjaćielu / nie cżynięć krzywdy : Azaś śię nie z groƺá zmówił ze mną?
Weźmi co twojego <i>jeſt,</i> á idź : chcę bowiem temu oſtátniemu dáć jáko y tobie.
Azaż mi śię nie godźi cżynić z mojim co chcę? Cżyli oko twoje złośliwe jeſt iżem ja jeſt dobry?
Tákći będą oſtátni pierwƺymi / á pierwƺy oſtátnimi. Abowiem / wiele jeſt wezwánych / ále máło wybránych.
A wſtępując JEzus do Jeruzalem / wźiął z ſobą dwánaśćie ucżniów ná oſobne miejſce w drodze / y rzekł im ;
Oto wſtępujemy do Jeruzalem / á Syn cżłowiecży będźie wydány Przedniejƺym kápłanom / y Náucżonym w piśmie / y oſądzą go ná śmierć.
Y wydádzą go pogánom ná pośmiewánie / y ná ubicżowánie / y ná ukrzyżowánie : ále trzećiego dniá zmartwychwſtánie.
Tedy przyſtąpiłá do niego mátká ſynów Zebedeuƺowych / z ſynámi ſwojimi kłaniájąc <i>mu</i> śię / y proƺąc nieco od niego.
A on jey rzekł ; Cżegóż chceƺ? Rzekłá mu ; Rzecż áby śiedźieli ći dwá ſynowie moji / jeden po práwicy twojey / á drugi po lewicy w króleſtwie twojim.
Ale JEzus odpowiádájąc / rzekł ; Nie wiećie o co prośićie. Możećież pić kielich który Ja będę pił? y chrztem / którym śię Ja chrzcżę być ochrzcżeni? Rzekli mu : Możemy.
Tedy im rzekł : Kielichći mój pić będźiećie / y chrztem którym śię ja chrzcżę ochrzcżeni będźiećie : ále śiedźieć po práwicy mojey y po lewicy mojey / nie jeſt mojá rzecż dáć <i>wam,</i> ále <i>tym</i> którym jeſt zgotowano od Ojcá mojego.
A uſłyƺawƺy <i>to</i> oni dźieśięć / rozgniewáli śię ná onych dwu bráći.
Ale JEzus zwoławƺy ich / rzekł : Wiećie iż Kśiążętá narodów pánują nád nimi / á którzy wielcy <i>ſą,</i> mocy dokázują nád nimi.
Lecż nie ták będźie miedzy wámi : ále ktobykolwiek miedzy wámi chćiał być wielkim / niech będźie ſługą wáƺym.
A ktobykolwiek miedzy wámi chćiał być pierwƺym / niech będźie ſługą wáƺym.
Jáko y Syn cżłowiecży nie przyƺedł áby mu ſłużono / ále áby ſłużył / y áby dał duƺę ſwą ná okup zá wielu.
A gdy <i>oni</i> wychodźili z Jerychá / ƺedł zá nim wielki lud.
A oto dwá ślepi śiedzący przy drodze / uſłyƺawƺy iż JEzus przechodźił / záwołáli / mówiąc : Zmiłuj śię nád námi PAnie / Synu Dawidów!
Ale on lud gromił ich / áby milcżeli. Lecż oni tym więcey wołáli / mówiąc : Zmiłuj śię nád námi PAnie / Synu Dawidów!
A záſtánowiwƺy śię JEzus / záwołał ich / y rzekł : Cóż chcećie ábym wam ucżynił?
Rzekli mu : PAnie / áby były otworzone ocży náƺe.
A użaliwƺy śię <i>ich</i> JEzus / dotknął śię ocżu ich : á záraz przejrzáły ocży ich. Y ƺli zá nim.
