Abowiem podobne jeſt króleſtwo niebieſkie cżłowiekowi goſpodárzowi / który wyƺedł bárdzo ráno nájmowáć robotników do winnicy ſwojej.
A zmówiwƺy śię z robotnikámi z groƺá ná dźień poſłáł je do winnicy ſwojej.
A wyƺedłƺy o trzećiej godźinie / ujrzáł drugich / którzy ſtáli ná rynku próżnujący ;
Y rzekł im : Idźćie y wy do winnicy / á co będźie ſpráwiedliwego / dám wám.
A oni poƺli. Záśię wyƺedłƺy o ƺóſtej y dźiewiątej godźinie / tákże ucżynił.
Potem o jedenáſtej godźinie wyƺedłƺy / ználázł drugie / którzy ſtáli próżnujący / y rzekł im : Przecż tu ſtoićie cáły dźień próżnujący?
Rzekli mu : Iż náſ nikt nie nájął ; y rzekł im : Idźćie y wy do winnicy / á co będźie ſpráwiedliwego / weźmiećie.
A gdy był wiecżór / rzekł pán winnicy ſpráwcy ſwemu : Záwołáj robotników / á oddáj im zápłátę / pocżąwƺy od oſtátnich áż do pierwƺych.
A gdy przyƺli oni / którzy o jedenáſtej godźinie byli nájęći / wźiął káżdy z nich po groƺu.
Przyƺedłƺy też y pierwśi / mniemáli / że więcej wezmą ; ále wźięli y oni / káżdy z nich / po groƺu.
A wźiąwƺy / ƺemráli przećiwko goſpodárzowi /
Mówiąc : Ci oſtátni jednę godźinę robili / á ucżyniłeś je nám równymi / którzyśmy znośili ćiężár dniá y upálenie.
A on odpowiádájąc rzekł jednemu z nich : Przyjáćielu! nie cżynięć krzywdy ; ázáż śię nie z groƺá zmówił zemną?
Weźmij / co twojego jeſt / á idź ; chcę bowiem temu oſtátniemu dáć jáko y tobie.
Azáż mi śię nie godźi cżynić z mojem / co chcę? Cżyli oko twoje złośliwe jeſt / iżem já jeſt dobry?
Tákći będą oſtátni pierwƺymi / á pierwśi oſtátnimi ; ábowiem wiele jeſt wezwánych / ále máło wybránych.
A wſtępując JEzuſ do Jeruzálemu / wźiął z ſobą dwánáśćie ucżniów ná oſobne miejſce w drodze / y rzekł im :
Oto wſtępujemy do Jeruzálemu / á Syn cżłowiecży będźie wydány przedniejƺym kápłánom y náucżonym w Piśmie / y oſądzą go ná śmierć.
Y wydádzą go pogánom ná pośmiewánie y ná ubicżowánie y ná ukrzyżowánie ; ále trzećiego dniá zmártwychwſtánie.
Tedy przyſtąpiłá do niego mátká ſynów Zebedeuƺowych z ſynámi ſwoimi / kłániájąc mu śię / y proƺąc nieco od niego.
A on jej rzekł ; Cżegóż chceƺ? Rzekłá mu : Rzecż / áby śiedźieli ći dwáj ſynowie moi / jeden po práwicy twojej á drugi po lewicy w króleſtwie twojim.
Ale JEzuſ odpowiádájąc rzekł : Nie wiećie / o co prośićie ; możećież pić kielich / który já będę pił? y chrztem / którym śię já chrzcżę / być ochrzcżeni? Rzekli mu : Możemy.
Tedy im rzekł : Kielichći mój pić będźiećie / y chrztem / którym śię já chrzcżę / ochrzcżeni będźiećie ; ále śiedźieć po práwicy mojej y po lewicy mojej / nie jeſt mojá rzecż dáć wám / ále tym / którym jeſt zgotowáno od Ojcá mojego.
A uſłyƺáwƺy to oni dźieśięćiu / rozgniewáli śię ná onych dwóch bráći.
Ale JEzuſ zwołáwƺy ich / rzekł : Wiećie / iż kśiążętá národów pánują nád nimi / á którzy wielcy ſą / mocy dokázują nád nimi.
Lecż nie ták będźie między wámi : ále ktobykolwiek między wámi chćiáł być wielkim / niech będźie ſługą wáƺym.
A ktobykolwiek między wámi chćiáł być pierwƺym / niech będźie ſługą wáƺym.
Jáko y Syn cżłowiecży nie przyƺedł / áby mu ſłużono / ále áby ſłużył / y áby dáł duƺę ſwą ná okup zá wielu.
A gdy oni wychodźili z Jerychá / ƺedł zá nim wielki lud.
A oto dwáj ślepi / śiedzący przy drodze / uſłyƺáwƺy / iż JEzuſ przechodźił / záwołáli / mówiąc : Zmiłuj śię nád námi / Pánie / ſynu Dawidowy!
Ale on lud gromił ich / áby milcżeli ; lecż oni tem więcej wołáli / mówiąc : Zmiłuj śię nád námi / Pánie / ſynu Dawidowy!
A záſtánowiwƺy śię JEzuſ / záwołáł ich y rzekł : Cóż chcećie / ábym wám ucżynił?
Rzekli mu : Pánie! áby były otworzone ocży náƺe.
A użáliwƺy śię ich JEzuſ / dotknął śię ocżu ich / á záráz przejrzáły ocży ich ; y ƺli zá nim.
