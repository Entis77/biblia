A gdy śię przybliżyli do Jeruzalem / y przyƺli do Betfágie do góry Oliwney / tedy JEzus poſłał dwu ucżniów /
Mówiąc im : Idźćie do miáſtecżká / które jeſt przećiwko wam / á záraz znajdźiećie Oślicę uwiązáną y Oślę z nią ; odwiążćieƺ <i>je,</i> á przywiedźćie do mnie.
A jeſliby wam co kto rzekł / powiedzćie / iż PAN ich potrzebuje : á zárazem puśći je.
A to śię wƺyſtko ſtáło / áby śię wypełniło co powiedźiano przez Proroká mówiącego ;
Powiedzćie Córce Syońſkiey : Oto Król twój idźie tobie ćichy / á śiedzący ná oślicy / y ná oślęćiu ſynu <i>oślice</i> pod járzmem będącey.
Szedƺy tedy ucżniowie / á ucżyniwƺy ták jáko im był rozkazał JEzus :
Przywiedli oślicę y oślę / y włożyli ná nie ƺáty ſwoje / y wſádźili go ná nie.
A wielki lud ſłáli ƺáty ſwoje ná drodze : á drudzy obćináli gáłązki z drzew / y ſłáli ná drodze.
A lud w przód y pozad idący / wołał mówiąc : Hoſánná Synowi Dawidowemu! błogoſłáwiony który idźie w Imieniu PAńſkim : Hoſánná ná wyſokośćiách!
A gdy on wjáchał do Jeruzalem / wzruƺyło śię wƺyſtko miáſto / mówiąc : Któż ten jeſt?
A lud mówił ; Tenći jeſt JEzus on Prorok z Názáretu Gálilejſkiego.
Tedy wƺedł JEzus do Kośćiołá Bożego / y wygnał wƺyſtkie przedawájące / y kupujące w kośćiele : á ſtoły tych co pieniędzmi hándlowáli / y ſtołki przedawájących gołębie poprzewracał /
Y rzekł im : Nápiſano : Dom mój dom modlitwy názwány będźie : áleśćie wy ucżynili z niego jáſkinią zbójców.
Tedy przyſtąpili do niego ślepi y chromi w kośćiele : y uzdrowił je.
A obacżywƺy przedniejƺy kápłani y Náucżeni w piśmie / cudá które cżynił / y dźieći wołájące w kośćiele / y mówiące : Hoſánná Synowi Dawidowemu / rozgniewáli śię.
Y rzekli mu : Słyƺyƺże co ći mówią? A JEzus im rzekł : y owƺem. Nigdyśćież nie cżytáli? iż z uſt niemowiątek y ſſących / wykonałeś chwałę.
A opuśćiwƺy je / wyƺedł z miáſtá do Betániey / y tám zoſtał.
A ráno wracájąc śię do miáſtá / łáknął :
Y ujrzawƺy jedno figowe drzewo przy drodze / przyƺedł do niego ; y nie ználazł nic ná nim / tylko ſámo liśćie / y rzekł mu ; Niechaj śię więcey z ćiebie owoc nie rodźi ná wieki. Y uſchło zárázem one figowe drzewo.
A ujrzawƺy to ucżniowie / dźiwowáli śię / mówiąc ; Jákoć prętko uſchło to figowe drzewo.
Tedy odpowiádájąc JEzus / rzekł im ; Záprawdę powiádam wam / Jeſlibyśćie mieli wiárę / á nie wątpilibyśćie : nie tylko to <i>co śię ſtáło</i> z figowym drzewem ucżynićie / ále gdybyśćie y tey górze rzekli ; Podnieś śię / á rzuć śię w morze : ſtánie śię.
Y wƺyſtko o cobyśćiekolwiek prośili w modlitwie wierząc / weźmiećie.
A gdy on przyƺedł do Kośćiołá / przyſtąpili do niego gdy ucżył / Przedniejƺy Kápłani y Stárƺy ludu / mówiąc ; Którąż mocą to cżyniƺ? á ktoć dał tę moc?
A odpowiádájąc JEzus / rzekł im ; ſpytam y Ja was o jednę rzecż / którą jeſli mi powiećie / y Ja wam powiem / którą mocą to cżynię.
Chrzeſt Janów z kąd był? z niebá / cżyli z ludźi? A oni myślili ſámi w ſobie / mówiąc ; Jeſli powiemy ; Z niebá : rzecże nam ; Cżemużeśćie mu tedy nie uwierzyli?
Jeſli záś powiemy ; Z ludźi : bojimy śię ludu ; bo Janá wƺyſcy máją zá Proroká.
A odpowiádájąc JEzuſowi / rzekli ; Nie wiemy. Rzekł im y on ; Y Ja wam nie powiem którą mocą to cżynię.
Ale cóż śię wam zda? Cżłowiek <i>niektóry</i> miał dwu ſynów : á przyſtąpiwƺy do pierwƺego / rzekł ; Synu / idź / rób dźiś ná winnicy mojey.
Ale on odpowiádájąc / rzekł : Nie chcę ; á potym obacżywƺy śię / poƺedł.
A przyſtąpiwƺy do drugiego / rzekł tákże : A on odpowiádájąc rzekł ; Ja <i>idę</i> Pánie : ále nie ƺedł.
Któryż z tych dwu ucżynił wolą Ojcowſką? Rzekli mu ; On pierwƺy. Rzekł im JEzus ; Záprawdę powiádam wam / Że was Celnicy y wƺetecżnice uprzedzáją do króleſtwá Bożego.
Abowiem przyƺedł do was Jan drogą ſpráwiedliwośći / á nie uwierzyliśćie mu ; ále Celnicy y wƺetecżnice uwierzyły mu : á wy widząc <i>to,</i> przećięśćie śię nie obacżyli / ábyśćie mu uwierzyli.
Drugiego podobieńſtwá ſłuchajćie ; Cżłowiek niektóry był goſpodarzem / który náſádźił winnicę / y płotem ją ogrodźił / y wkopał w niey práſę / y zbudował wieżę / y nájął ją winiárzom ; y odjáchał <i>precż</i>.
A gdy śię przybliżył cżás <i>odbieránia</i> pożytków / poſłał ſługi ſwoje do onych winiárzów / áby odebráli pożytki jey.
Ale winiárze pojimawƺy ſługi jego / jednego ubili / á drugiego zábili / á drugiego ukámionowáli.
Záśię poſłał inƺych ſług / więcey niż pierwƺych : Y tákże im ucżynili.
Ale náoſtátek poſłał do nich Syná ſwego / mówiąc ; Będą śię wſtydźić Syná mego.
Lecż winiárze ujrzawƺy onego Syná / rzekli miedzy ſobą ; Tenći jeſt dźiedźic : pódźćie / zábijmy go / á otrzymamy dźiedźictwo jego.
Tedy porwawƺy go / wyrzućili go precż z winnice / y zábili.
Gdy tedy Pan winnice przydźie / cóż ucżyni onym winiárzom?
Rzekli mu ; Złe / źle potráći / á winnicę najmie inƺym winiárzom / którzy mu oddawáć będą pożytki cżáſów ſwojich.
Rzekł im JEzus ; Nie cżytáliśćieƺ nigdy w piśmiech? Kámień który odrzućili budujący / ten śię ſtał głową węgiełną : od PAnáć śię to ſtáło / y dźiwne jeſt w ocżách náƺych.
Przetoż powiádam wam ; Iż od was odjęte będźie króleſtwo Boże / y będźie dáne narodowi cżyniącemu pożytki jego.
A ktoby padł ná ten kámień / roztrąći śię : á ná kogoby <i>on</i> upadł / zetrze go.
A uſłyƺawƺy przedniejƺy Kápłani y Fáryzeuƺowie podobieńſtwá jego / domyślili śię iż o nich mówił.
A chcąc go pojimáć / bali śię ludu : ponieważ go mieli zá Proroká.
