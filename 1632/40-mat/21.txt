A gdy śię przybliżyli do Jeruzálemu / y przyƺli do Betfágie / do góry oliwney / tedy JEzuſ poſłáł dwóch ucżniów /
Mówiąc im : Idźćie do miáſtecżká / które jeſt przećiwko wám / á záráz znájdźiećie oślicę uwiązáną y oślę z nią ; odwiążćież je á przywiedźćie do mnie.
A jeſliby wám co kto rzekł / powiedzćie / iż Pán ich potrzebuje ; á zárázem puśći je.
A to śię wƺyſtko ſtáło / áby śię wypełniło / co powiedźiáno przez proroká / mówiącego :
Powiedzćie córce Syońſkiej : Oto król twój idźie tobie ćichy / á śiedzący ná oślicy / y ná oślęćiu / ſynu oślicy pod járzmem będącey.
Szedłƺy tedy ucżniowie á ucżyniwƺy ták / jáko im był rozkázáł JEzuſ /
Przywiedli oślicę y oślę / y włożyli ná nie ƺáty ſwoje / y wſádźili go ná nie.
A wielki lud ſłáł ƺáty ſwoje ná drodze / á drudzy obćináli gáłązki z drzew / y ſłáli ná drodze.
A lud wprzód y pozád idący wołáł / mówiąc : Hoſánná ſynowi Dawidowemu! błogoſłáwiony / który idźie w imieniu Páńſkim / Hoſánná ná wyſokośćiách!
A gdy on wjecháł do Jeruzálemu / wzruƺyło śię wƺyſtko miáſto / mówiąc : Któż ten jeſt?
A lud mówił : Tenći jeſt JEzuſ / on prorok z Názáretu Gálilejſkiego.
Tedy wƺedł JEzuſ do kośćiołá Bożego / y wygnáł wƺyſtkie ſprzedáwájące y kupujące w kośćiele / á ſtoły tych / co pieniędzmi hándlowáli / y ſtołki ſprzedáwájących gołębie poprzewrácáł /
Y rzekł im : Nápiſáno : Dom mój dom modlitwy názwány będźie ; áleśćie wy ucżynili z niego jáſkinię zbójców.
Tedy przyſtąpili do niego ślepi y chromi w kośćiele / y uzdrowił je.
A obácżywƺy przedniejśi kápłáni y náucżeni w Piśmie cudá / które cżynił / y dźieći wołájące w kośćiele / y mówiące : Hoſánná ſynowi Dawidowemu : rozgniewáli śię.
Y rzekli mu : Słyƺyƺże / co ći mówią? A JEzuſ im rzekł : Y owƺem. Nigdyśćież nie cżytáli / iż z uſt niemowlątek y ſſących wykonáłeś chwáłę?
A opuśćiwƺy je / wyƺedł z miáſtá do Betánii / y tám zoſtáł ;
A ráno wrácájąc śię do miáſtá / łáknął.
Y ujrzáwƺy jedno figowe drzewo przy drodze / przyƺedł do niego / y nie ználázł nic ná nim / tylko ſáme liśćie / y rzekł mu : Niecháj śię więcey z ćiebie owoc nie rodźi ná wieki. Y uſchło zárázem one figowe drzewo.
A ujrzáwƺy to ucżniowie / dźiwowáli śię / mówiąc : Jákoć prędko uſchło to figowe drzewo!
Tedy odpowiádájąc JEzuſ / rzekł im : Zápráwdę powiádám wám : Jeſlibyśćie mieli wiárę / á nie wątpilibyśćie / nie tylko to / co śię ſtáło z figowem drzewem / ucżynićie / ále gdybyśćie y tey górze rzekli : Podnieś śię / á rzuć śię w morze / ſtánie śię.
Y wƺyſtko / o cobyśćiekolwiek prośili w modlitwie wierząc / weźmiećie.
A gdy on przyƺedł do kośćiołá / przyſtąpili do niego / gdy ucżył / przedniejśi kápłáni y ſtárśi ludu / mówiąc : Którąż mocą to cżyniƺ? á ktoć dáł tę moc?
A odpowiádájąc JEzuſ / rzekł im : Spytám y ja was o jednę rzecż / którą jeſli mi powiećie / y ja wám powiem / którą mocą to cżynię.
Chrzeſt Jáná ſkąd był? z niebá cżyli z ludźi? A oni myśleli ſámi w ſobie / mówiąc : Jeſli powiemy z niebá / rzecże nám : Cżemużeśćie mu tedy nie uwierzyli?
Jeſli záś powiemy z ludźi / boimy śię ludu ; bo Jáná wƺyſcy máją zá proroká.
A odpowiádájąc JEzuſowi rzekli : Nie wiemy. Rzekł im y on : Y ja wám nie powiem / którą mocą to cżynię.
Ale cóż śię wám zdá? Cżłowiek niektóry miał dwóch ſynów ; á przyſtąpiwƺy do pierwƺego / rzekł : Synu! idź / rób dźiś ná winnicy mojey.
Ale on odpowiádájąc rzekł : Nie chcę / á potem obácżywƺy śię / poƺedł.
A przyſtąpiwƺy do drugiego / rzekł tákże ; á on odpowiádájąc rzekł : Ja idę / pánie! ále nie ƺedł.
Któryż z tych dwóch ucżynił wolę ojcowſką? Rzekli mu : On pierwƺy. Rzekł im JEzuſ : Zápráwdę powiádám wám / że was celnicy y wƺetecżnice uprzedzáją do króleſtwá Bożego.
Abowiem przyƺedł do was Ján drogą ſpráwiedliwośći / á nie uwierzyliśćie mu / ále celnicy y wƺetecżnice uwierzyli mu ; á wy widząc to / przećież śię nie obácżyliśćie / ábyśćie mu uwierzyli.
Drugiego podobieńſtwá ſłuchájćie : Cżłowiek niektóry był goſpodárzem / który náſádźił winnicę / y płotem ją ogrodźił / y wkopáł w niej práſę / y zbudowáł wieżę / y nájął ją winiárzom / y odjecháł precż.
A gdy śię przybliżył cżáſ odbierániá pożytków / poſłáł ſługi ſwoje do onych winiárzy / áby odebráli pożytki jey.
Ale winiárze pojmáwƺy ſługi jego / jednego ubili / á drugiego zábili / á drugiego ukámionowáli.
Záśię poſłáł inƺych ſług / więcey niż pierwƺych ; y tákże im ucżynili.
Ale ná oſtátek poſłáł do nich ſyná ſwego / mówiąc : Będą śię wſtydźić ſyná mego.
Lecż winiárze / ujrzáwƺy onego ſyná / rzekli między ſobą : Tenći jeſt dźiedźic ; pójdźćie / zábijmy go / á otrzymámy dźiedźictwo jego.
Tedy porwáwƺy go / wyrzućili go precż z winnicy y zábili.
Gdy tedy pán winnicy przyjdźie / cóż ucżyni onym winiárzom?
Rzekli mu : Złe / źle potráći / á winnicę nájmie inƺym winiárzom / którzy mu oddáwáć będą pożytki cżáſów ſwoich.
Rzekł im JEzuſ : Nie cżytáliśćież nigdy w Piſmách : Kámień / który odrzućili budujący / ten śię ſtáł głową węgielną : od Pánáć śię to ſtáło / y dźiwne jeſt w ocżách náƺych?
Przetoż powiádám wám : Iż od was odjęte będźie króleſtwo Boże / y będźie dáne národowi cżyniącemu pożytki jego.
A ktoby pádł ná ten kámień / roztrąći śię / á ná kogoby on upádł / zetrze go.
A uſłyƺáwƺy przedniejśi kápłáni y Fáryzeuƺowie podobieńſtwá jego / domyślili śię / iż o nich mówił ;
A chcąc go pojmáć / báli śię ludu / poniewáż go mieli zá proroká.
