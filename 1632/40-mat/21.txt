A gdy śię przybliżyli do Jeruzálemu / y przyƺli do Betfágie / do góry oliwnej / tedy Jezuſ poſłáł dwóch ucżniów /
Mówiąc im : Idźćie do miáſtecżká / które jeſt przećiwko wám / á záráz znájdziećie oślicę uwiązáną y oślę z nią ; odwiążćież je á przywiedźćie do mnie.
A jeźliby wám co kto rzekł / powiedzćie / iż Pán ich potrzebuje ; á zárázem puśći je.
A to śię wƺyſtko ſtáło / áby śię wypełniło / co powiedziáno przez proroká / mówiącego :
Powiedzćie córce Syońſkiej : Oto król twój idzie tobie ćichy / á śiedzący ná oślicy / y ná oślęćiu / ſynu oślicy pod járzmem będącej.
Szedłƺy tedy ucżniowie á ucżyniwƺy ták / jáko im był rozkázáł Jezuſ /
Przywiedli oślicę y oślę / y włożyli ná nie ƺáty ſwoje / y wſádzili go ná nie.
A wielki lud ſłáł ƺáty ſwoje ná drodze / á drudzy obćináli gáłązki z drzew / y ſłáli ná drodze.
A lud wprzód y pozád idący wołáł / mówiąc : Hoſánná ſynowi Dawidowemu! błogoſłáwiony / który idzie w imieniu Páńſkim / Hoſánná ná wyſokośćiách!
A gdy on wjecháł do Jeruzálemu / wzruƺyło śię wƺyſtko miáſto / mówiąc : Któż ten jeſt?
A lud mówił : Tenći jeſt Jezuſ / on prorok z Názáretu Gálilejſkiego.
Tedy wƺedł Jezuſ do kośćiołá Bożego / y wygnáł wƺyſtkie ſprzedáwájące y kupujące w kośćiele / á ſtoły tych / co pieniędzmi hándlowáli / y ſtołki ſprzedáwájących gołębie poprzewrácáł /
Y rzekł im : Nápiſáno : Dom mój dom modlitwy názwány będzie ; áleśćie wy ucżynili z niego jáſkinię zbójców.
Tedy przyſtąpili do niego ślepi y chromi w kośćiele / y uzdrowił je.
A obácżywƺy przedniejśi kápłáni y náucżeni w Piśmie cudá / które cżynił / y dzieći wołájące w kośćiele / y mówiące : Hoſánná ſynowi Dawidowemu : rozgniewáli śię.
Y rzekli mu : Słyƺyƺże / co ći mówią? A Jezuſ im rzekł : Y owƺem. Nigdyśćież nie cżytáli / iż z uſt niemowlątek y ſſących wykonáłeś chwáłę?
A opuśćiwƺy je / wyƺedł z miáſtá do Betánii / y tám zoſtáł ;
A ráno wrácájąc śię do miáſtá / łáknął.
Y ujrzáwƺy jedno figowe drzewo przy drodze / przyƺedł do niego / y nie ználázł nic ná nim / tylko ſáme liśćie / y rzekł mu : Niecháj śię więcej z ćiebie owoc nie rodzi ná wieki. Y uſchło zárázem one figowe drzewo.
A ujrzáwƺy to ucżniowie / dziwowáli śię / mówiąc : Jákoć prędko uſchło to figowe drzewo!
Tedy odpowiádájąc Jezuſ / rzekł im : Zápráwdę powiádám wám : Jeźlibyśćie mieli wiárę / á nie wątpilibyśćie / nie tylko to / co śię ſtáło z figowem drzewem / ucżynićie / ále gdybyśćie y tej górze rzekli : Podnieś śię / á rzuć śię w morze / ſtánie śię.
Y wƺyſtko / o cobyśćiekolwiek prośili w modlitwie wierząc / weźmiećie.
A gdy on przyƺedł do kośćiołá / przyſtąpili do niego / gdy ucżył / przedniejśi kápłáni y ſtárśi ludu / mówiąc : Którąż mocą to cżyniƺ? á ktoć dáł tę moc?
A odpowiádájąc Jezuſ / rzekł im : Spytám y já wáſ o jednę rzecż / którą jeźli mi powiećie / y já wám powiem / którą mocą to cżynię.
Chrzeſt Jáná ſkąd był? z niebá cżyli z ludzi? A oni myśleli ſámi w ſobie / mówiąc : Jeźli powiemy z niebá / rzecże nám : Cżemużeśćie mu tedy nie uwierzyli?
Jeźli záś powiemy z ludzi / boimy śię ludu ; bo Jáná wƺyſcy máją zá proroká.
A odpowiádájąc Jezuſowi rzekli : Nie wiemy. Rzekł im y on : Y já wám nie powiem / którą mocą to cżynię.
Ale cóż śię wám zdá? Cżłowiek niektóry miáł dwóch ſynów ; á przyſtąpiwƺy do pierwƺego / rzekł : Synu! idź / rób dziś ná winnicy mojej.
Ale on odpowiádájąc rzekł : Nie chcę / á potem obácżywƺy śię / poƺedł.
A przyſtąpiwƺy do drugiego / rzekł tákże ; á on odpowiádájąc rzekł : Já idę / pánie! ále nie ƺedł.
Któryż z tych dwóch ucżynił wolę ojcowſką? Rzekli mu : On pierwƺy. Rzekł im Jezuſ : Zápráwdę powiádám wám / że wáſ celnicy y wƺetecżnice uprzedzáją do króleſtwá Bożego.
Abowiem przyƺedł do wáſ Ján drogą ſpráwiedliwośći / á nie uwierzyliśćie mu / ále celnicy y wƺetecżnice uwierzyli mu ; á wy widząc to / przećież śię nie obácżyliśćie / ábyśćie mu uwierzyli.
Drugiego podobieńſtwá ſłuchájćie : Cżłowiek niektóry był goſpodárzem / który náſádził winnicę / y płotem ją ogrodził / y wkopáł w niej práſę / y zbudowáł wieżę / y nájął ją winiárzom / y odjecháł precż.
A gdy śię przybliżył cżáſ odbierániá pożytków / poſłáł ſługi ſwoje do onych winiárzy / áby odebráli pożytki jej.
Ale winiárze pojmáwƺy ſługi jego / jednego ubili / á drugiego zábili / á drugiego ukámionowáli.
Záśię poſłáł inƺych ſług / więcej niż pierwƺych ; y tákże im ucżynili.
Ale ná oſtátek poſłáł do nich ſyná ſwego / mówiąc : Będą śię wſtydzić ſyná mego.
Lecż winiárze / ujrzáwƺy onego ſyná / rzekli między ſobą : Tenći jeſt dziedzic ; pójdźćie / zábijmy go / á otrzymámy dziedzictwo jego.
Tedy porwáwƺy go / wyrzućili go precż z winnicy y zábili.
Gdy tedy pán winnicy przyjdzie / cóż ucżyni onym winiárzom?
Rzekli mu : Złe / źle potráći / á winnicę nájmie inƺym winiárzom / którzy mu oddáwáć będą pożytki cżáſów ſwoich.
Rzekł im Jezuſ : Nie cżytáliśćież nigdy w Piſmách : Kámień / który odrzućili budujący / ten śię ſtáł głową węgielną : od Pánáć śię to ſtáło / y dziwne jeſt w ocżách náƺych?
Przetoż powiádám wám : Iż od wáſ odjęte będzie króleſtwo Boże / y będzie dáne národowi cżyniącemu pożytki jego.
A ktoby pádł ná ten kámień / roztrąći śię / á ná kogoby on upádł / zetrze go.
A uſłyƺáwƺy przedniejśi kápłáni y Fáryzeuƺowie podobieńſtwá jego / domyślili śię / iż o nich mówił ;
A chcąc go pojmáć / báli śię ludu / poniewáż go mieli zá proroká.
