A gdy było ráno / weƺli w rádę wƺyſcy Przedniejƺy Kápłani / y Stárƺy ludu przećiwko JEzuſowi / áby go zábili.
Y związawƺy go / wiedli y podáli go Pontſkiemu Piłatowi Stárośćie.
Tedy Judaƺ / który go był wydał / widząc iż był oſądzony / żáłując <i>tego,</i> wróćił trzydźieśći śrebrników Przedniejƺym kápłanom / y Stárƺym <i>ludu,</i>
Mówiąc ; Zgrzeƺyłem wydawƺy krew niewinną! A oni rzekli : Cóż nam do tego / ty ujrzyƺ.
A porzućiwƺy one śrebrniki w Kośćiele / odƺedł : á odƺedƺy / obieśił śię.
Ale przedniejƺy Kápłani / wźiąwƺy one śrebrniki / mówili : Nie godźi śię ich kłáść do ſkárbu / gdyż zapłátá jeſt krwie.
Y nárádźiwƺy śię / kupili zá nie rolą gárncarzowę ná pogrzeb gośćiom.
Dla tego oná rola názwána jeſt rolą krwie / áż do <i>dniá</i> dźiśiejƺego.
Tedy śię wypełniło / co powiedźiano przez Jeremiaƺá Proroká / mówiącego ; Y wźięli trzydźieśći śrebrników zapłátę oƺácowánego / który był oƺácowány od ſynów Izráelſkich :
Y dáli je zá rolą gárncarzowę / jáko mi poſtánowił PAN.
A JEzus ſtał przed Stároſtą. Y pytał go Stároſtá / mówiąc : Tyżeś jeſt on Król Żydowſki? A JEzus mu rzekł : Ty powiádaƺ.
A gdy nań ſkárżyli Przedniejƺy Kápłani y Stárƺy / nic nie odpowiedźiał.
Tedy mu rzekł Piłat : Nie ſłyƺyƺże jáko wiele przećiwko tobie świadcżą?
Lecż mu nie odpowiedźiał y ná jedno ſłowo : ták / iż śię Stároſtá bárzo dźiwował.
Ale ná Święto zwykł był Stároſtá wypuƺcżáć ludowi jednego więźniá / któregoby chćieli.
Y mieli ná ten cżás więźniá znácżnego / którego zwano Bárábbaƺ.
A gdy śię zebráli / rzekł do nich Piłat : Któregoƺ chcećie / ábym wam wypuśćił? Bárábbaƺá / cżyli JEzuſá / którego zowią CHryſtuſem?
Bo wiedźiał / iż go z nienawiśći wydáli.
A gdy on śiedźiał ná <i>ſądowey</i> ſtolicy / poſłáłá do niego żoná jego / mówiąc : Nie miey żadney ſpráwy z tym ſpráwiedliwym : bom wiele ućierpiáłá dźiś we ſnie dla niego.
Ale Przedniejƺy Kápłani y Stárƺy / námówili lud / áby prośili o Bárábbaƺá / á JEzuſá áby ſtráćili.
A odpowiádájąc Stároſtá / rzekł im : Którego chcećie ábym wam z tych dwu wypuśćił? A oni odpowiedźieli : Bárábbaƺá.
Rzekł im Piłat : Cóż tedy ucżynię z JEzuſem / którego zowią CHryſtuſem? Rzekli mu wƺyſcy : Niech będźie ukrzyżowány.
A Stároſtá rzekł : Cóż wżdy złego ucżynił? Ale oni tym bárźiey wołáli / mówiąc ; Niech będźie ukrzyżowány!
A widząc Piłat iż to nic nie pomagáło / ále <i>owƺem</i> śię więkƺy rozruch wƺcżynał / wźiąwƺy wodę / umył ręce przed ludem / mówiąc : Nie jeſtem ja winien krwie tego ſpráwiedliwego : wy ujrzyćie.
A odpowiádájąc wƺyſtek lud / rzekł : Krew jego ná nas y ná dźiatki náƺe.
Tedy im wypuśćił Bárábbaƺá : ále JEzuſá ubicżowawƺy / wydał <i>go</i> áby był ukrzyżowány.
Tedy żołnierze Stárośćini przywiódƺy JEzuſá ná ratuƺ / zebráli do niego wƺyſtkę rotę.
A zewlekƺy go / przyodźiali go płaƺcżem ƺárłatowym.
Y uplótƺy koronę z ćiernia / włożyli ná głowę jego ; y <i>dáli</i> trzćinę w práwą rękę jego : á upadájąc przed nim ná koláná / náśmiewáli śię z niego / mówiąc : Bądź pozdrowiony Królu Żydowſki.
A plując nań / wźięli onę trzćinę / y bili go w głowę jego.
A gdy śię z niego náśmiali / zewlekli go z onego płaƺcżá / y oblekli go w ƺátę jego ; y wiedli go áby był ukrzyżowány.
A wychodząc ználeźli cżłowieká Cyrenejcżyká / imieniem Symoná : tego przymuśili / áby nióſł krzyż jego.
A przyƺedƺy ná miejſce rzecżone Golgotá / które zowią miejſcem trupich głów :
Dali mu pić ocet z żółćią zmięƺány : á ſkoƺtowawƺy / nie chćiał pić.
A ukrzyżowawƺy go / rozdźielili ƺáty jego / y miotáli los : áby śię wypełniło co powiedźiano przez Proroká ; Rozdźielili ſobie ƺáty moje : á o odźienie moje los miotáli.
A śiedząc / ſtrzegli go tám.
Y przybili nád głową jego winę jego nápiſáną ; Ten jeſt JEzus Król Żydowſki.
Byli też ukrzyżowáni z nim dwá zbójcy : jeden po práwicy / á drugi po lewicy.
A ći którzy mimo chodźili / bluźnili go / chwiejąc głowámi ſwojimi /
Y mówiąc ; Ty co rozwalaƺ Kośćiół / á we trzech dniách budujeƺ <i>go,</i> rátuj ſámego śiebie ; Jeſliś jeſt Syn Boży / zſtąp z krzyżá.
Tákże y Przedniejƺy Kápłani z Náucżonymi w piśmie y z Stárƺymi / náśmiewájąc śię mówili ;
Inƺych rátował / á ſámego śiebie rátowáć nie może : Jeſliż jeſt Król Izráelſki / niech teraz zſtąpi z krzyżá / á uwierzymy mu.
Dufał w Bogu / niechże go teraz wybáwi / jeſli śię w nim kocha : Boć powiedźiał ; Jeſtem Synem Bożym.
Tákże też y zbójcy / którzy byli z nim ukrzyżowáni / urągáli mu.
A od ƺóſtey godźiny ſtáłá śię ćiemność po wƺyſtkiey źiemi / áż do dźiewiątey godźiny.
A około dźiewiątey godźiny / záwołał JEzus głoſem wielkim / mówiąc ; Eli / Eli / Lámá Sábáchtháni! to jeſt / Boże mój! Boże mój! cżemuś mię opuśćił?
Tedy niektórzy z tych co tám ſtali uſłyƺawƺy to / mówili : Eliaƺá ten woła.
A zárázem bieżawƺy jeden z nich / wźiął gębkę / y nápełnił ją octem ; á włożywƺy ná trzćinę / dał mu pić.
A drudzy mówili ; Zániechaj / pátrzajmy jeſli przydźie Eliaƺ / áby go wybáwił.
Ale JEzus záwoławƺy powtóre głoſem wielkim / oddał Duchá.
A oto zaſłoná kośćielna rozerwáłá śię ná dwoję / od wierzchu áż do dołu : y trzęſłá śię źiemiá : á ſkáły śię pádáły.
Y groby śię otwarzáły : á wiele ćiał świętych / którzy byli záſnęli / powſtáło.
A wyƺedƺy z grobów po <i>zmartwych</i> wſtániu jego / weƺli do miáſtá świętego : y pokazáli śię wielom.
Tedy Setnik y ći co z nim JEzuſá ſtrzegli / widząc trzęśienie źiemie / y to co śię dźiało / zlękli śię bárzo / mówiąc ; Prawdźiwieć ten był Synem Bożym.
A było tám wiele niewiaſt z dáleká śię przypátrujących ; które były przyƺły zá JEzuſem od Gálilejey / poſługując mu.
Miedzy którymi byłá Márya Mágdálená / y Márya mátká Jákubowá y Jozeſowá / y mátká ſynów Zebedeuƺowych.
A gdy był wiecżór / przyƺedł cżłowiek bogáty z Arymátyey / imieniem Józef ; który też był ucżniem JEzuſowym.
Ten przyƺedƺy do Piłatá / prośił o ćiáło JEzuſowe. Tedy Piłat rozkazał / áby <i>mu</i> było ono ćiáło oddáne.
A Józef wźiąwƺy ono ćiáło / uwinął je w cżyſte prześćierádło.
Y położył je w nowym grobie ſwojim / który był w opoce wykował : á przywáliwƺy do drzwi grobowych kámień wielki / odƺedł.
A byłá tám Márya Mágdálená / y druga Márya / które śiedźiáły przećiwko grobowi.
A drugie° dniá / który był <i>pierwƺy</i> po Przygotowániu / zgromádźili śię Przedniejƺy Kápłani y Fáryzeuƺowie do Piłatá.
Mówiąc ; Pánie / wſpomnieliſmy / iż on zwodźićiel powiedźiał gdy jeƺcże żyw był ; Po trzech dniách <i>zmartwych</i> wſtánę.
Rozkaż tedy obwárowáć grób / áż do dniá trzećiego : by ſnadź przyƺedƺy ucżniowie jego w nocy / nie ukrádli go / y nie powiedźieli ludowi ; Iż powſtał od umárłych : y będźie pośledni błąd gorƺy niż pierwƺy.
Rzekł im Piłat ; Maćie ſtraż / Idźćież / obwárujćie jáko umiećie.
A oni poƺedƺy / oſádźili grób ſtrażą / zápiecżętowawƺy kámień.
