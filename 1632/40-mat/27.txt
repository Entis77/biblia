A gdy było ráno / weƺli w rádę wƺyſcy przedniejśi kápłáni y ſtárśi ludu przećiwko JEzuſowi / áby go zábili ;
Y związáwƺy go / wiedli y podáli go Ponckiemu Piłátowi / ſtárośćie.
Tedy Judáƺ / który go był wydał / widząc / iż był oſądzony / żáłując tego / wróćił trzydźieśći ſrebrników przedniejƺym kápłánom y ſtárƺym ludu.
Mówiąc : Zgrzeƺyłem / wydáwƺy krew niewinną! A oni rzekli : Cóż nam do tego? ty ujrzyƺ!
A porzućiwƺy one ſrebrniki w kośćiele / odƺedł / á odƺedƺy powieśił śię.
Ale przedniejśi kápłáni wźiąwƺy one ſrebrniki / mówili : Nie godźi śię ich kłáść do ſkárbu kośćielnego / gdyż zápłátá jeſt krwi.
Y nárádźiwƺy śię / kupili zá nie rolę gárncárzową ná pogrzeb gośćiom.
Dla tego oná rolá názwáná jeſt rolą krwi / áż do dniá dźiśiejƺego.
Tedy śię wypełniło / co powiedźiáno przez Jeremijáƺá proroká / mówiącego : Y wźięli trzydźieśći ſrebrników / zápłátę oƺácowánego / który był oƺácowány od ſynów Izráelſkich ;
Y dáli je zá rolę gárncárzową / jáko mi poſtánowił Pán.
A JEzus ſtał przed ſtároſtą ; y pytał go ſtároſtá / mówiąc : Tyżeś jeſt on król Żydowſki? A JEzus mu rzekł : Ty powiádáƺ.
A gdy nań ſkárżyli przedniejśi kápłáni y ſtárśi / nic nie odpowiedźiał.
Tedy mu rzekł Piłát : Nie ſłyƺyƺże / jáko wiele przećiwko tobie świádcżą?
Lecż mu nie odpowiedźiał y ná jedno ſłowo / ták / iż śię ſtároſtá bárzo dźiwował.
Ale ná święto zwykł był ſtároſtá wypuƺcżáć ludowi jednego więźniá / któregoby chćieli.
Y mieli nátencżás więźniá znácżnego / którego zwano Bárábbáƺ.
A gdy śię zebráli / rzekł do nich Piłát : Któregoż chcećie / ábym wam wypuśćił? Bárábbáƺá / cżyli JEzuſá / którego zowią CHryſtuſem?
Bo wiedźiał / iż go z nienáwiśći wydáli.
A gdy on śiedźiał ná ſądowej ſtolicy / poſłáłá do niego żoná jego / mówiąc : Nie miej żádney ſpráwy z tym ſpráwiedliwym ; bom wiele ućierpiáłá dźiś we śnie dla niego.
Ale przedniejśi kápłáni y ſtárśi námówili lud / áby prośili o Bárábbáƺá / á JEzuſá áby ſtráćili.
A odpowiádájąc ſtároſtá / rzekł im : Którego chcećie ábym wam z tych dwu wypuśćił? á oni odpowiedźieli : Bárábbáƺá.
Rzekł im Piłát : Cóż tedy ucżynię z JEzuſem / którego zowią CHryſtuſem? Rzekli mu wƺyſcy : Niech będźie ukrzyżowány.
A ſtároſtá rzekł : Cóż wżdy złego ucżynił? Ale oni tym bárdźiej wołáli / mówiąc : Niech będźie ukrzyżowány!
A widząc Piłát / iż to nic nie pomágáło / ále owƺem śię więkƺy rozruch wƺcżynał / wźiąwƺy wodę / umył ręce przed ludem / mówiąc : Nie jeſtem ja winien krwi tego ſpráwiedliwego ; wy ujrzyćie.
A odpowiádájąc wƺyſtek lud / rzekł : Krew jego ná nas y ná dźiátki náƺe.
Tedy im wypuśćił Bárábbáƺá ; ále JEzuſá ubicżowáwƺy / wydał go / áby był ukrzyżowány.
Tedy żołnierze ſtárośćini przywiódƺy JEzuſá ná rátuƺ / zebráli do niego wƺyſtkę rotę ;
A zewlekƺy go / przyodźiáli go płáƺcżem ƺárłátowym ;
Y uplótƺy koronę z ćierniá / włożyli ná głowę jego / y dáli trzćinę w práwą rękę jego / á upádájąc przed nim ná koláná / náśmiewáli śię z niego / mówiąc : Bądź pozdrowiony / królu Żydowſki!
A plując nań / wźięli onę trzćinę / y bili go w głowę jego.
A gdy śię z niego náśmiáli / zewlekli go z onego płáƺcżá / y oblekli go w ƺátę jego / y wiedli go / áby był ukrzyżowány.
A wychodząc ználeſli cżłowieká Cyrenejcżyká / imieniem Szymon ; tego przymuśili / áby nióſł krzyż jego.
A przyƺedƺy ná miejſce rzecżone Golgotá / które zowią miejſcem trupich głów /
Dáli mu pić ocet z żółćią zmieƺány ; á ſkoƺtowáwƺy / nie chćiał pić.
A ukrzyżowáwƺy go / rozdźielili ƺáty jego / y miotáli los / áby śię wypełniło / co powiedźiáno przez proroká : Rozdźielili ſobie ƺáty moje / á o odźienie moje los miotáli.
A śiedząc / ſtrzegli go tám.
Y przybili nád głową jego winę jego nápiſáną : Ten jeſt JEzus / król Żydowſki.
Byli też ukrzyżowáni z nim dwáj zbójcy / jeden po práwicy / á drugi po lewicy.
A ći / którzy mimo chodźili / bluźnili go / chwiejąc głowámi ſwojimi /
Y mówiąc : Ty / co rozwáláƺ kośćiół / á w trzech dniách budujeƺ go / rátuj ſámego śiebie ; jeſliś jeſt Syn Boży / zſtąp z krzyżá.
Tákże y przedniejśi kápłáni z náucżonymi w Piśmie / y z ſtárƺymi / náśmiewájąc śię / mówili :
Inƺych rátował / á ſámego śiebie rátowáć nie może ; jeſliż jeſt król Izráelſki / niech teraz zſtąpi z krzyżá / á uwierzymy mu.
Dufał w Bogu / niechże go teraz wybáwi / jeſli śię w nim kocha ; boć powiedźiał : Jeſtem Synem Bożym.
Tákże też y zbójcy / którzy byli z nim ukrzyżowáni / urągáli mu.
A od ƺóſtey godźiny ſtáłá śię ćiemność po wƺyſtkiej źiemi áż do dźiewiątey godźiny.
A około dźiewiątey godźiny záwołał JEzus głoſem wielkim / mówiąc : Eli / Eli / Lámá Sábáchtáni! to jeſt / Boże mój! Boże mój! cżemuś mię opuśćił?
Tedy niektórzy z tych / co tám ſtáli / uſłyƺáwƺy to / mówili : Elijáƺá ten wołá.
A zárázem bieżáwƺy jeden z nich / wźiął gąbkę / y nápełnił ją octem / á włożywƺy ná trzćinę / dał mu pić.
A drudzy mówili : Zániecháj ; pátrzájmy / jeſli przydźie Elijáƺ / áby go wybáwił.
Ale JEzus záwołáwƺy powtóre głoſem wielkim / oddał duchá.
A oto záſłoná kośćielná rozerwáłá śię ná dwoje od wierzchu áż do dołu / y trzęſłá śię źiemiá / á ſkáły śię rozpádáły.
Y groby śię otwieráły / á wiele ćiał świętych / którzy byli záſnęli / powſtáło :
A wyƺedƺy z grobów po zmartwychwſtániu jego / weƺli do miáſtá świętego / y pokázáli śię wielom.
Tedy ſetnik y ći / co z nim JEzuſá ſtrzegli / widząc trzęśienie źiemi / y to / co śię dźiáło / zlękli śię bárzo / mówiąc : Práwdźiwieć ten był Synem Bożym.
A było tám wiele niewiáſt z dáleká śię przypátrujących / które były przyƺły zá JEzuſem od Gálilei / poſługując mu ;
Między któremi byłá Máryjá Mágdálená / y Máryjá / mátká Jákóbowá y Jozeſowá / y mátká ſynów Zebedeuƺowych.
A gdy był wiecżór / przyƺedł cżłowiek bogáty z Arymátyi / imieniem Józef / który też był ucżniem JEzuſowym.
Ten przyƺedƺy do Piłátá / prośił o ćiáło JEzuſowe. Tedy Piłát rozkázał / áby mu było ono ćiáło oddáne ;
A Józef wźiąwƺy ono ćiáło / uwinął je w cżyſte prześćierádło ;
Y położył je w nowym grobie ſwoim / który był w opoce wykował ; á przywáliwƺy do drzwi grobowych kámień wielki / odƺedł.
A byłá tám Máryjá Mágdálená / y drugá Máryjá / które śiedźiáły przećiwko grobowi.
A drugiego dniá / który był pierwƺy po przygotowániu / zgromádźili śię przedniejśi kápłáni y Fáryzeuƺowie do Piłátá.
Mówiąc : Pánie! wſpomnieliśmy / iż on zwodźićiel powiedźiał / gdy jeƺcże żyw był : Po trzech dniách zmartwychwſtánę.
Rozkáż tedy obwárowáć grób áż do dniá trzećiego / by ſnadź przyƺedƺy ucżniowie jego w nocy / nie ukrádli go / y nie powiedźieli ludowi / iż powſtał od umárłych ; y będźie pośledni błąd gorƺy niż pierwƺy.
Rzekł im Piłát : Máćie ſtráż / idźćież / obwárujćie / jáko umiećie.
A oni poƺedƺy / oſádźili grób ſtráżą / zápiecżętowáwƺy kámień.
