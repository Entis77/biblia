A gdy było ráno / weƺli w rádę wƺyſcy przedniejśi kápłáni y ſtárśi ludu przećiwko Jezuſowi / áby go zábili ;
Y związáwƺy go / wiedli y podáli go Ponckiemu Piłátowi / ſtárośćie.
Tedy Judáƺ / który go był wydáł / widząc / iż był oſądzony / żáłując tego / wróćił trzydzieśći ſrebrników przedniejƺym kápłánom y ſtárƺym ludu.
Mówiąc : Zgrzeƺyłem / wydáwƺy krew niewinną! A oni rzekli : Cóż nám do tego? ty ujrzyƺ!
A porzućiwƺy one ſrebrniki w kośćiele / odƺedł / á odƺedłƺy powieśił śię.
Ale przedniejśi kápłáni wziąwƺy one ſrebrniki / mówili : Nie godzi śię ich kłáść do ſkárbu kośćielnego / gdyż zápłátá jeſt krwi.
Y nárádziwƺy śię / kupili zá nie rolę gárncárzową ná pogrzeb gośćiom.
Dlátego oná rolá názwáná jeſt rolą krwi / áż do dniá dziśiejƺego.
Tedy śię wypełniło / co powiedziáno przez Jeremijáƺá proroká / mówiącego : Y wzięli trzydzieśći ſrebrników / zápłátę oƺácowánego / który był oƺácowány od ſynów Izráelſkich ;
Y dáli je zá rolę gárncárzową / jáko mi poſtánowił Pán.
A Jezuſ ſtáł przed ſtároſtą ; y pytáł go ſtároſtá / mówiąc : Tyżeś jeſt on król Żydowſki? A Jezuſ mu rzekł : Ty powiádáƺ.
A gdy náń ſkárżyli przedniejśi kápłáni y ſtárśi / nic nie odpowiedziáł.
Tedy mu rzekł Piłát : Nie ſłyƺyƺże / jáko wiele przećiwko tobie świádcżą?
Lecż mu nie odpowiedziáł y ná jedno ſłowo / ták / iż śię ſtároſtá bárdzo dziwowáł.
Ale ná święto zwykł był ſtároſtá wypuƺcżáć ludowi jednego więźniá / któregoby chćieli.
Y mieli nátencżáſ więźniá znácżnego / którego zwáno Bárábbáƺ.
A gdy śię zebráli / rzekł do nich Piłát : Któregoż chcećie / ábym wám wypuśćił? Bárábbáƺá / cżyli Jezuſá / którego zowią Chryſtuſem?
Bo wiedziáł / iż go z nienáwiśći wydáli.
A gdy on śiedziáł ná ſądowej ſtolicy / poſłáłá do niego żoná jego / mówiąc : Nie miej żádnej ſpráwy z tym ſpráwiedliwym ; bom wiele ućierpiáłá dziś we śnie dlá niego.
Ale przedniejśi kápłáni y ſtárśi námówili lud / áby prośili o Bárábbáƺá / á Jezuſá áby ſtráćili.
A odpowiádájąc ſtároſtá / rzekł im : Którego chcećie ábym wám z tych dwóch wypuśćił? á oni odpowiedzieli : Bárábbáƺá.
Rzekł im Piłát : Cóż tedy ucżynię z Jezuſem / którego zowią Chryſtuſem? Rzekli mu wƺyſcy : Niech będzie ukrzyżowány.
A ſtároſtá rzekł : Cóż wżdy złego ucżynił? Ale oni tem bárdziej wołáli / mówiąc : Niech będzie ukrzyżowány!
A widząc Piłát / iż to nic nie pomágáło / ále owƺem śię więkƺy rozruch wƺcżynáł / wziąwƺy wodę / umył ręce przed ludem / mówiąc : Nie jeſtem já winien krwi tego ſpráwiedliwego ; wy ujrzyćie.
A odpowiádájąc wƺyſtek lud / rzekł : Krew jego ná náſ y ná dziátki náƺe.
Tedy im wypuśćił Bárábbáƺá ; ále Jezuſá ubicżowáwƺy / wydáł go / áby był ukrzyżowány.
Tedy żołnierze ſtárośćini przywiódłƺy Jezuſá ná rátuƺ / zebráli do niego wƺyſtkę rotę ;
A zewlekłƺy go / przyodziáli go płáƺcżem ƺárłátowym ;
Y uplótłƺy koronę z ćierniá / włożyli ná głowę jego / y dáli trzćinę w práwą rękę jego / á upádájąc przed nim ná koláná / náśmiewáli śię z niego / mówiąc : Bądź pozdrowiony / królu Żydowſki!
A plując náń / wzięli onę trzćinę / y bili go w głowę jego.
A gdy śię z niego náśmiáli / zewlekli go z onego płáƺcżá / y oblekli go w ƺátę jego / y wiedli go / áby był ukrzyżowány.
A wychodząc ználeźli cżłowieká Cyrenejcżyká / imieniem Szymon ; tego przymuśili / áby nióſł krzyż jego.
A przyƺedłƺy ná miejſce rzecżone Golgotá / które zowią miejſcem trupich głów /
Dáli mu pić ocet z żółćią zmieƺány ; á ſkoƺtowáwƺy / nie chćiáł pić.
A ukrzyżowáwƺy go / rozdzielili ƺáty jego / y miotáli loſ / áby śię wypełniło / co powiedziáno przez proroká : Rozdzielili ſobie ƺáty moje / á o odzienie moje loſ miotáli.
A śiedząc / ſtrzegli go tám.
Y przybili nád głową jego winę jego nápiſáną : Ten jeſt Jezuſ / król Żydowſki.
Byli też ukrzyżowáni z nim dwáj zbójcy / jeden po práwicy / á drugi po lewicy.
A ći / którzy mimo chodzili / bluźnili go / chwiejąc głowámi ſwojimi /
Y mówiąc : Ty / co rozwáláƺ kośćiół / á w trzech dniách budujeƺ go / rátuj ſámego śiebie ; jeźliś jeſt Syn Boży / zſtąp z krzyżá.
Tákże y przedniejśi kápłáni z náucżonymi w Piśmie / y z ſtárƺymi / náśmiewájąc śię / mówili :
Inƺych rátowáł / á ſámego śiebie rátowáć nie może ; jeźliż jeſt król Izráelſki / niech teráz zſtąpi z krzyżá / á uwierzymy mu.
Dufáł w Bogu / niechże go teráz wybáwi / jeźli śię w nim kochá ; boć powiedziáł : Jeſtem Synem Bożym.
Tákże też y zbójcy / którzy byli z nim ukrzyżowáni / urągáli mu.
A od ƺóſtej godziny ſtáłá śię ćiemność po wƺyſtkiej ziemi áż do dziewiątej godziny.
A około dziewiątej godziny záwołáł Jezuſ głoſem wielkim / mówiąc : Eli / Eli / Lámá Sábáchtáni! to jeſt / Boże mój! Boże mój! cżemuś mię opuśćił?
Tedy niektórzy z tych / co tám ſtáli / uſłyƺáwƺy to / mówili : Elijáƺá ten wołá.
A zárázem bieżáwƺy jeden z nich / wziął gąbkę / y nápełnił ją octem / á włożywƺy ná trzćinę / dáł mu pić.
A drudzy mówili : Zániecháj ; pátrzájmy / jeźli przyjdzie Elijáƺ / áby go wybáwił.
Ale Jezuſ záwołáwƺy powtóre głoſem wielkim / oddáł duchá.
A oto záſłoná kośćielná rozerwáłá śię ná dwoje od wierzchu áż do dołu / y trzęſłá śię ziemiá / á ſkáły śię rozpádáły.
Y groby śię otwieráły / á wiele ćiáł świętych / którzy byli záſnęli / powſtáło :
A wyƺedłƺy z grobów po zmártwychwſtániu jego / weƺli do miáſtá świętego / y pokázáli śię wielom.
Tedy ſetnik y ći / co z nim Jezuſá ſtrzegli / widząc trzęśienie ziemi / y to / co śię dziáło / zlękli śię bárdzo / mówiąc : Práwdziwieć ten był Synem Bożym.
A było tám wiele niewiáſt z dáleká śię przypátrujących / które były przyƺły zá Jezuſem od Gálilei / poſługując mu ;
Między któremi byłá Máryjá Mágdálená / y Máryjá / mátká Jákóbowá y Jozeſowá / y mátká ſynów Zebedeuƺowych.
A gdy był wiecżór / przyƺedł cżłowiek bogáty z Arymátyi / imieniem Józef / który też był ucżniem Jezuſowym.
Ten przyƺedłƺy do Piłátá / prośił o ćiáło Jezuſowe. Tedy Piłát rozkázáł / áby mu było ono ćiáło oddáne ;
A Józef wziąwƺy ono ćiáło / uwinął je w cżyſte prześćierádło ;
Y położył je w nowym grobie ſwoim / który był w opoce wykowáł ; á przywáliwƺy do drzwi grobowych kámień wielki / odƺedł.
A byłá tám Máryjá Mágdálená / y drugá Máryjá / które śiedziáły przećiwko grobowi.
A drugiego dniá / który był pierwƺy po przygotowániu / zgromádzili śię przedniejśi kápłáni y Fáryzeuƺowie do Piłátá.
Mówiąc : Pánie! wſpomnieliśmy / iż on zwodzićiel powiedziáł / gdy jeƺcże żyw był : Po trzech dniách zmártwychwſtánę.
Rozkáż tedy obwárowáć grób áż do dniá trzećiego / by ſnáć przyƺedłƺy ucżniowie jego w nocy / nie ukrádli go / y nie powiedzieli ludowi / iż powſtáł od umárłych ; y będzie pośledni błąd gorƺy niż pierwƺy.
Rzekł im Piłát : Máćie ſtráż / idźćież / obwárujćie / jáko umiećie.
A oni poƺedłƺy / oſádzili grób ſtráżą / zápiecżętowáwƺy kámień.
