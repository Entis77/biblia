A gdy było ráno / weƺli w rádę wƺyſcy przedniejśi kápłáni y ſtárśi ludu przećiwko JEzuſowi / áby go zábili ;
Y związáwƺy go / wiedli y podáli go Ponckiemu Piłátowi / ſtárośćie.
Tedy Judáƺ / który go był wydáł / widząc / iż był oſądzony / żáłując tego / wróćił trzydźieśći ſrebrników przedniejƺym kápłánom y ſtárƺym ludu.
Mówiąc : Zgrzeƺyłem / wydáwƺy krew niewinną! A oni rzekli : Cóż nám do tego? ty ujrzyƺ!
A porzućiwƺy one ſrebrniki w kośćiele / odƺedł / á odƺedłƺy powieśił śię.
Ale przedniejśi kápłáni wźiąwƺy one ſrebrniki / mówili : Nie godźi śię ich kłáść do ſkárbu kośćielnego / gdyż zápłátá jeſt krwi.
Y nárádźiwƺy śię / kupili zá nie rolę gárncárzową ná pogrzeb gośćiom.
Dlátego oná rolá názwáná jeſt rolą krwi / áż do dniá dźiśiejƺego.
Tedy śię wypełniło / co powiedźiáno przez Jeremijáƺá proroká / mówiącego : Y wźięli trzydźieśći ſrebrników / zápłátę oƺácowánego / który był oƺácowány od ſynów Izráelſkich ;
Y dáli je zá rolę gárncárzową / jáko mi poſtánowił Pán.
A JEzuſ ſtáł przed ſtároſtą ; y pytáł go ſtároſtá / mówiąc : Tyżeś jeſt on król Żydowſki? A JEzuſ mu rzekł : Ty powiádáƺ.
A gdy náń ſkárżyli przedniejśi kápłáni y ſtárśi / nic nie odpowiedźiáł.
Tedy mu rzekł Piłát : Nie ſłyƺyƺże / jáko wiele przećiwko tobie świádcżą?
Lecż mu nie odpowiedźiáł y ná jedno ſłowo / ták / iż śię ſtároſtá bárdzo dźiwowáł.
Ale ná święto zwykł był ſtároſtá wypuƺcżáć ludowi jednego więźniá / któregoby chćieli.
Y mieli nátencżáſ więźniá znácżnego / którego zwáno Bárábbáƺ.
A gdy śię zebráli / rzekł do nich Piłát : Któregoż chcećie / ábym wám wypuśćił? Bárábbáƺá / cżyli JEzuſá / którego zowią CHryſtuſem?
Bo wiedźiáł / iż go z nienáwiśći wydáli.
A gdy on śiedźiáł ná ſądowej ſtolicy / poſłáłá do niego żoná jego / mówiąc : Nie miej żádney ſpráwy z tym ſpráwiedliwym ; bom wiele ućierpiáłá dźiś we śnie dla niego.
Ale przedniejśi kápłáni y ſtárśi námówili lud / áby prośili o Bárábbáƺá / á JEzuſá áby ſtráćili.
A odpowiádájąc ſtároſtá / rzekł im : Którego chcećie ábym wám z tych dwóch wypuśćił? á oni odpowiedźieli : Bárábbáƺá.
Rzekł im Piłát : Cóż tedy ucżynię z JEzuſem / którego zowią CHryſtuſem? Rzekli mu wƺyſcy : Niech będźie ukrzyżowány.
A ſtároſtá rzekł : Cóż wżdy złego ucżynił? Ale oni tem bárdźiej wołáli / mówiąc : Niech będźie ukrzyżowány!
A widząc Piłát / iż to nic nie pomágáło / ále owƺem śię więkƺy rozruch wƺcżynáł / wźiąwƺy wodę / umył ręce przed ludem / mówiąc : Nie jeſtem ja winien krwi tego ſpráwiedliwego ; wy ujrzyćie.
A odpowiádájąc wƺyſtek lud / rzekł : Krew jego ná nas y ná dźiátki náƺe.
Tedy im wypuśćił Bárábbáƺá ; ále JEzuſá ubicżowáwƺy / wydáł go / áby był ukrzyżowány.
Tedy żołnierze ſtárośćini przywiódłƺy JEzuſá ná rátuƺ / zebráli do niego wƺyſtkę rotę ;
A zewlekłƺy go / przyodźiáli go płáƺcżem ƺárłátowym ;
Y uplótłƺy koronę z ćierniá / włożyli ná głowę jego / y dáli trzćinę w práwą rękę jego / á upádájąc przed nim ná koláná / náśmiewáli śię z niego / mówiąc : Bądź pozdrowiony / królu Żydowſki!
A plując náń / wźięli onę trzćinę / y bili go w głowę jego.
A gdy śię z niego náśmiáli / zewlekli go z onego płáƺcżá / y oblekli go w ƺátę jego / y wiedli go / áby był ukrzyżowány.
A wychodząc ználeſli cżłowieká Cyrenejcżyká / imieniem Szymon ; tego przymuśili / áby nióſł krzyż jego.
A przyƺedłƺy ná miejſce rzecżone Golgotá / które zowią miejſcem trupich głów /
Dáli mu pić ocet z żółćią zmieƺány ; á ſkoƺtowáwƺy / nie chćiáł pić.
A ukrzyżowáwƺy go / rozdźielili ƺáty jego / y miotáli loſ / áby śię wypełniło / co powiedźiáno przez proroká : Rozdźielili ſobie ƺáty moje / á o odźienie moje loſ miotáli.
A śiedząc / ſtrzegli go tám.
Y przybili nád głową jego winę jego nápiſáną : Ten jeſt JEzuſ / król Żydowſki.
Byli też ukrzyżowáni z nim dwáj zbójcy / jeden po práwicy / á drugi po lewicy.
A ći / którzy mimo chodźili / bluźnili go / chwiejąc głowámi ſwojimi /
Y mówiąc : Ty / co rozwáláƺ kośćiół / á w trzech dniách budujeƺ go / rátuj ſámego śiebie ; jeſliś jeſt Syn Boży / zſtąp z krzyżá.
Tákże y przedniejśi kápłáni z náucżonymi w Piśmie / y z ſtárƺymi / náśmiewájąc śię / mówili :
Inƺych rátowáł / á ſámego śiebie rátowáć nie może ; jeſliż jeſt król Izráelſki / niech teraz zſtąpi z krzyżá / á uwierzymy mu.
Dufáł w Bogu / niechże go teraz wybáwi / jeſli śię w nim kochá ; boć powiedźiáł : Jeſtem Synem Bożym.
Tákże też y zbójcy / którzy byli z nim ukrzyżowáni / urągáli mu.
A od ƺóſtey godźiny ſtáłá śię ćiemność po wƺyſtkiej źiemi áż do dźiewiątey godźiny.
A około dźiewiątey godźiny záwołáł JEzuſ głoſem wielkim / mówiąc : Eli / Eli / Lámá Sábáchtáni! to jeſt / Boże mój! Boże mój! cżemuś mię opuśćił?
Tedy niektórzy z tych / co tám ſtáli / uſłyƺáwƺy to / mówili : Elijáƺá ten wołá.
A zárázem bieżáwƺy jeden z nich / wźiął gąbkę / y nápełnił ją octem / á włożywƺy ná trzćinę / dáł mu pić.
A drudzy mówili : Zániecháj ; pátrzájmy / jeſli przyjdźie Elijáƺ / áby go wybáwił.
Ale JEzuſ záwołáwƺy powtóre głoſem wielkim / oddáł duchá.
A oto záſłoná kośćielná rozerwáłá śię ná dwoje od wierzchu áż do dołu / y trzęſłá śię źiemiá / á ſkáły śię rozpádáły.
Y groby śię otwieráły / á wiele ćiáł świętych / którzy byli záſnęli / powſtáło :
A wyƺedłƺy z grobów po zmartwychwſtániu jego / weƺli do miáſtá świętego / y pokázáli śię wielom.
Tedy ſetnik y ći / co z nim JEzuſá ſtrzegli / widząc trzęśienie źiemi / y to / co śię dźiáło / zlękli śię bárdzo / mówiąc : Práwdźiwieć ten był Synem Bożym.
A było tám wiele niewiáſt z dáleká śię przypátrujących / które były przyƺły zá JEzuſem od Gálilei / poſługując mu ;
Między któremi byłá Máryjá Mágdálená / y Máryjá / mátká Jákóbowá y Jozeſowá / y mátká ſynów Zebedeuƺowych.
A gdy był wiecżór / przyƺedł cżłowiek bogáty z Arymátyi / imieniem Józef / który też był ucżniem JEzuſowym.
Ten przyƺedłƺy do Piłátá / prośił o ćiáło JEzuſowe. Tedy Piłát rozkázáł / áby mu było ono ćiáło oddáne ;
A Józef wźiąwƺy ono ćiáło / uwinął je w cżyſte prześćierádło ;
Y położył je w nowym grobie ſwoim / który był w opoce wykowáł ; á przywáliwƺy do drzwi grobowych kámień wielki / odƺedł.
A byłá tám Máryjá Mágdálená / y drugá Máryjá / które śiedźiáły przećiwko grobowi.
A drugiego dniá / który był pierwƺy po przygotowániu / zgromádźili śię przedniejśi kápłáni y Fáryzeuƺowie do Piłátá.
Mówiąc : Pánie! wſpomnieliśmy / iż on zwodźićiel powiedźiáł / gdy jeƺcże żyw był : Po trzech dniách zmartwychwſtánę.
Rozkáż tedy obwárowáć grób áż do dniá trzećiego / by ſnáć przyƺedłƺy ucżniowie jego w nocy / nie ukrádli go / y nie powiedźieli ludowi / iż powſtáł od umárłych ; y będźie pośledni błąd gorƺy niż pierwƺy.
Rzekł im Piłát : Máćie ſtráż / idźćież / obwárujćie / jáko umiećie.
A oni poƺedłƺy / oſádźili grób ſtráżą / zápiecżętowáwƺy kámień.
