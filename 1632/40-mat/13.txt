A dniá onego wyƺedƺy JEzus z domu / uśiadł nád morzem.
Y zebrał śię do niego wielki lud ; ták / iż wſtąpiwƺy w łódź / śiedźiał / á wƺyſtek lud ſtał ná brzegu.
Y mówił do nich wiele w podobieńſtwách <i>y</i> rzekł ; Oto wyƺedł rozśiewcá áby rozśiewał.
A gdy on rozśiewał / niektóre pádło podle drogi ; y przylećieli ptacy / á pozobáli je.
Drugie záśię pádło ná <i>miejſcá</i> opocżyſte / gdźie nie miáło wiele źiemie ; y wnet weƺło / iż nie miáło głębokośći źiemie.
Ale gdy ſłońce weƺło / wygorzáło : á iż nie miáło korzeniá / uſchło.
A drugie pádło miedzy ćiernie ; y wzroſło ćiernie / á záduśiło je.
A drugie pádło ná źiemię dobrą ; y wydáło pożytek / jedno ſetny / drugie ƺeśćdźieśiątny / á drugie trzydźieſtny.
Kto ma uƺy ku ſłuchániu / niechaj ſłucha.
Tedy przyſtąpiwƺy ucżniowie / rzekli mu ; Dla cżegoż im w podobieńſtwách mówiƺ?
A on odpowiádájąc / rzekł im ; Wam dano wiedźieć tájemnice króleſtwá niebieſkiego / ále onym nie dano.
Abowiem / kto ma / będźie mu dano / y obfitowáć będźie : ále kto nie ma / y to co ma / będźie od niego odjęto.
Dla tegoć im w podobieńſtwách mówię : iż widząc / nie widzą ; y ſłyƺąc / nie ſłyƺą / áni rozumieją.
Y pełni śię w nich proroctwo Izájaƺowe / które mówi ; Słuchem ſłucháć będźiećie / ále nie zrozumiećie : y widząc widźieć będźiećie / ále nie ujrzyćie.
Abowiem zátyło ſerce ludu tego / á uƺymá ćiężko ſłyƺeli / y ocży ſwe zámrużyli : żeby kiedy ocżymá nie widźieli / y uƺymá <i>nie</i> ſłyƺeli / á ſercem <i>nie</i> zrozumieli / y <i>nie</i> náwróćili śię / á uzdrowiłbym <i>je</i>.
Ale ocży wáƺe błogoſłáwione że widzą ; y uƺy wáƺe / że ſłyƺą.
Bo záprawdę powiádam wam / iż wiele proroków y ſpráwiedliwych / żądáli widźieć <i>to</i> co <i>wy</i> widźićie / ále nie widźieli ; y ſłyƺeć <i>to</i> co ſłyƺyćie / ále nie ſłyƺeli.
Wy tedy ſłuchajćie podobieńſtwá <i>onego</i> rozśiewce.
Gdy kto ſłucha ſłowá o tym króleſtwie á nie rozumie / przychodźi on Zły / y porywa to co wśiano w ſerce jego. Tenći jeſt on / który podle drogi pośiany <i>jeſt</i>.
A ná opocżyſtych <i>miejſcách</i> pośiany / ten jeſt który ſłucha ſłowá / y záraz je z rádośćią przyjmuje :
Ale nie ma korzenia w ſobie / lecż docżeſny jeſt : A gdy przychodźi ućiſk álbo prześládowánie dla ſłowá / wnet śię gorƺy.
A miedzy ćiernie pośiany ten jeſt / który ſłucha ſłowá ; ále piecżołowánie świátá tego / y omamienie bogactw záduƺa ſłowo ; y ſtawa śię bez pożytku.
A ná dobrey źiemi pośiany jeſt ten / który ſłucha ſłowá y rozumie : tenći pożytek przynośi / á przynośi jeden ſetny / drugi ƺeśćdźieśiątny / á drugi trzydźieſtny.
Drugie podobieńſtwo przełożył im / mówiąc ; Podobne jeſt króleſtwo niebieſkie cżłowiekowi rozśiewájącemu dobre naśienie ná roli ſwojey.
A gdy ludźie záſnęli / przyƺedł nieprzyjaćiel jego / y náśiał kąkolu miedzy pƺenicą / y odƺedł.
A gdy uroſłá trawá y pożytek przynioſłá / tedy śię pokazał y kąkol.
Tedy przyſtąpiwƺy ſłudzy goſpodárſcy / rzekli mu ; Pánie / Izaliś dobrego naśienia nie náśiał ná roli twojey? Z kądże tedy ma kąkol?
A on im rzekł ; Nieprzyjaćiel cżłowiek to ucżynił. Y rzekli ſłudzy do niego ; A chceƺże iż pójdźiemy á zbierzemy go?
A on rzekł ; Nie : byśćie ſnadź zbierájąc kąkol / nie wykorzenili záraz z nim <i>y</i> pƺenice.
Dopuśććie obojgu ſpołem rość áż do żniwá : A cżáſu żniwá rzekę żeńcom ; Zbierzćie pierwey kąkol / á zwiążćie go w ſnopki ku ſpaleniu : ále pƺenicę zgromadźćie do gumná mojego.
Inƺe podobieńſtwo przełożył im mówiąc ; Podobne jeſt króleſtwo niebieſkie źiárnu gorcżycżnemu / które wźiąwƺy cżłowiek wśiał ná roli ſwojey.
Które namniejƺeć jeſt ze wƺyſtkich naśion ; ále kiedy urośćie / nawiękƺe jeſt ze wƺyſtkich járzyn : y ſtawa śię drzewem / ták iż ptacy niebieſcy przylátując / gniazdá ſobie cżynią ná gáłązkách jego.
Inƺe podobieńſtwo powiedźiał im : Podobne jeſt króleſtwo niebieſkie kwáſowi / który wźiąwƺy niewiáſtá / zákryłá we trzy miáry mąki / áżby wƺyſtká zkwáśniáłá.
To wƺyſtko mówił JEzus w podobieńſtwách do ludu : á bez podobieńſtwá nie mówił do nich.
Aby śię wypełniło co powiedźiano przez Proroká mówiącego ; Otworzę w podobieńſtwách uſtá moje / wypowiem ſkryte rzecży od záłożenia świátá.
Tedy rozpuśćiwƺy on lud / przyƺedł JEzus do domu : y przyſtąpili do niego ucżniowie jego / mówiąc : Wyłóż nam podobieńſtwo o kąkolu oney roli.
A on odpowiádájąc / rzekł im : Ten który rozśiewa dobre naśienie / jeſt Syn cżłowiecży.
A rola jeſt świát : A dobre naśienie / ſą ſynowie króleſtwá : ále kąkol / ſą ſynowie onego złego.
Nieprzyjaćiel zaśię który go rozśiał / jeſtći dyabeł : á żniwo / jeſt dokonánie świátá : á żeńcy / ſą Anjołowie.
Jáko tedy zbieráją kąkol / á palą go ogniem : ták będźie przy dokonániu świátá tego.
Pośle Syn cżłowiecży Anjoły ſwoje / á oni zbiorą z króleſtwá jego wƺyſtkie zgorƺenia / y te którzy niepráwość cżynią :
Y wrzucą je w piec ogniſty : tám będźie płácż y zgrzytánie zębów.
Tedy ſpráwiedliwi łſnić śię będą jáko ſłońce w króleſtwie Ojcá ſwego. Kto ma uƺy ku ſłuchániu / niechaj ſłucha.
Záśię podobne jeſt króleſtwo niebieſkie ſkárbowi ſkrytemu w roli : który ználazƺy cżłowiek / ſkrył / y od rádośći <i>którą miał</i> z niego / odchodźi y wƺyſtko co ma przedawa / y kupuje onę rolą.
Záśię podobne jeſt króleſtwo niebieſkie cżłowiekowi kupcowi / ƺukájącemu pięknych pereł.
Który ználazƺy jednę perłę bárzo drogą / odƺedł / y poprzedał wƺyſtko co miał / y kupił ją.
Záśię podobne jeſt króleſtwo niebieſkie niewodowi zápuƺcżonemu w morze / y <i>ryby</i> wƺelkiego rodzáju zágarniájącemu.
Który gdy był pełen / wyćiągnęli <i>rybitwi</i> ná brzeg : á uśiadƺy wybieráli dobre <i>ryby</i> w nacżynia / á złe precż wyrzucáli.
Tákći będźie przy dokonániu świátá : wynidą Anjołowie / y wyłącżą złe z pośrzodku ſpráwiedliwych.
Y wrzucą je w piec ogniſty : tám będźie płácż y zgrzytánie zębów.
Rzekł im JEzus ; Wyrozumieliśćie to wƺyſtko? Rzekli mu ; Ták PAnie.
A on im rzekł : Przetoż káżdy Náucżony w piśmie wyćwicżony w króleſtwie niebieſkim / podobny jeſt cżłowiekowi goſpodarzowi / który wynośi z ſkárbu ſwego nowe y ſtáre rzecży.
Y ſtáło śię gdy JEzus dokońcżył tych podobieńſtw / puśćił śię z támtąd.
A przyƺedƺy do Ojcżyzny ſwojey / náucżał je w bóżnicey ich : ták / iż śię bárzo zdumiewáli / y mówili ; Z kądże temu tá mądrość / y tá moc?
Izaż ten nie jeſt on ſyn ćieśle? Izaż mátki jego nie zowią Márya : á bráćią jego Jákób / y Jozes / y Symon / y Judás?
A śioſtry jego izali wƺyſtkie u nas nie ſą? Z kądże tedy temu to wƺyſtko?
Y gorƺyli śię z niego. Ale JEzus rzekł im ; Nie jeſt Prorok beze cżći / tylko w Ojcżyznie ſwojey y w domu ſwojim.
Y nie ucżynił tám wiele cudów / dla niedowiárſtwá ich.
