A dniá onego wyƺedłƺy Jezuſ z domu / uśiádł nád morzem :
Y zebráł śię do niego wielki lud / ták / iż wſtąpiwƺy w łódź / śiedziáł / á wƺyſtek lud ſtáł ná brzegu.
Y mówił do nich wiele w podobieńſtwách y rzekł : Oto wyƺedł rozśiewcá / áby rozśiewáł ;
A gdy on rozśiewáł / niektóre pádło podle drogi ; y przylećiáły ptáki / á pozobáły je.
Drugie záśię pádło ná miejſce opocżyſte / gdzie nie miáło wiele ziemi ; y wnet weƺło / iż nie miáło głębokośći ziemi.
Ale gdy ſłońce weƺło / wygorzáło / á iż nie miáło korzeniá / uſchło.
A drugie pádło między ćiernie / y wzroſły ćiernie á záduśiły je.
A drugie pádło ná ziemię dobrą / y wydáło pożytek / jedno ſetny / drugie ƺeśćdzieśiątny / á drugie trzydzieſtny.
Kto má uƺy ku ſłuchániu / niecháj ſłuchá.
Tedy przyſtąpiwƺy ucżniowie / rzekli mu : Dlácżegoż im w podobieńſtwách mówiƺ?
A on odpowiádájąc / rzekł im : Wám dáno wiedzieć tájemnicę króleſtwá niebieſkiego / ále onym nie dáno ;
Abowiem kto má / będzie mu dáno / y obfitowáć będzie ; ále kto niemá / y to / co má / będzie od niego odjęto.
Dlátegoć im w podobieńſtwách mówię / iż widząc nie widzą / y ſłyƺąc nie ſłyƺą / áni rozumieją.
Y pełni śię w nich proroctwo Izájáƺowe / które mówi : Słuchem ſłucháć będziećie / ále nie zrozumiećie : y widząc widzieć będziećie / ále nie ujrzyćie ;
Abowiem zátyło ſerce ludu tego / á uƺymá ćiężko ſłyƺeli / y ocży ſwe zámrużyli / żeby kiedy ocżymá nie widzieli y uƺymá nie ſłyƺeli / á ſercem nie zrozumieli / y nie náwróćili śię / á uzdrowiłbym je.
Ale ocży wáƺe błogoſłáwione / że widzą / y uƺy wáƺe / że ſłyƺą ;
Bo zápráwdę powiádám wám / iż wiele proroków y ſpráwiedliwych żądáło widzieć to / co wy widzićie / ále nie widzieli / y ſłyƺeć to / co ſłyƺyćie / ále nie ſłyƺeli.
Wy tedy ſłuchájćie podobieńſtwá onego rozśiewcy.
Gdy kto ſłuchá ſłowá o tem króleſtwie / á nie rozumie / przychodzi on zły y porywá to / co wśiáno w ſerce jego ; tenći jeſt on / który podle drogi pośiány jeſt.
A ná opocżyſtych miejſcách pośiány / ten jeſt / który ſłuchá ſłowá y záráz je z rádośćią przyjmuje ;
Ale niemá korzeniá w ſobie / lecż docżeſny jeſt ; á gdy przychodzi ućiſk / álbo prześládowánie dlá ſłowá / wnet śię gorƺy.
A między ćiernie pośiány / ten jeſt / który ſłuchá ſłowá ; ále piecżołowánie świátá tego y omámienie bogáctw záduƺá ſłowo / y ſtáje śię bez pożytku.
A ná dobrej ziemi pośiány / jeſt ten / który ſłuchá ſłowá y rozumie / tenći pożytek przynośi ; á przynośi jeden ſetny / drugi ƺeśćdzieśiątny / á drugi trzydzieſtny.
Drugie podobieńſtwo przełożył im / mówiąc : Podobne jeſt króleſtwo niebieſkie cżłowiekowi / rozśiewájącemu dobre náśienie ná roli ſwojej.
A gdy ludzie záſnęli / przyƺedł nieprzyjáćiel jego / y náśiáł kąkolu między pƺenicą / y odƺedł.
A gdy uroſłá tráwá y pożytek przynioſłá / tedy śię pokázáł y kąkol.
Tedy przyſtąpiwƺy ſłudzy goſpodárſcy / rzekli mu : Pánie! izáliś dobrego náśieniá nie náśiáł ná roli twojej? Skądże tedy má kąkol?
A on im rzekł : Nieprzyjáćiel cżłowiek to ucżynił. Y rzekli ſłudzy do niego : A chceƺże / iż pójdziemy / á zbierzemy go?
A on rzekł : Nie! byśćie ſnáć zbierájąc kąkol / nie wykorzenili záráz z nim y pƺenicy.
Dopuśććie obojgu ſpołem rość áż do żniwá ; á cżáſu żniwá rzekę żeńcom : Zbierzćie pierwej kąkol á zwiążćie go w ſnopki ku ſpáleniu ; ále pƺenicę zgromádźćie do gumná mojego.
Inƺe podobieńſtwo przełożył im / mówiąc : Podobne jeſt króleſtwo niebieſkie ziárnu gorcżycżnemu / które wziąwƺy cżłowiek / wśiáł ná roli ſwojej.
Które nájmniejƺeć jeſt ze wƺyſtkich náśion ; ále kiedy urośnie / nájwiękƺe jeſt ze wƺyſtkich járzyn / y ſtáje śię drzewem / ták / iż ptáki niebieſkie przylátując / gniázdá ſobie cżynią ná gáłązkách jego.
Inƺe podobieńſtwo powiedziáł im : Podobne jeſt króleſtwo niebieſkie kwáſowi / który wziąwƺy niewiáſtá / zákryłá we trzy miáry mąki / áżby wƺyſtká ſkwáśniáłá.
To wƺyſtko mówił Jezuſ w podobieńſtwách do ludu / á bez podobieńſtwá nie mówił do nich ;
Aby śię wypełniło / co powiedziáno przez proroká mówiącego : Otworzę w podobieńſtwách uſtá moje / wypowiem ſkryte rzecży od záłożeniá świátá.
Tedy rozpuśćiwƺy on lud / przyƺedł Jezuſ do domu ; y przyſtąpili do niego ucżniowie jego / mówiąc : Wyłóż nám podobieńſtwo o kąkolu onej roli.
A on odpowiádájąc / rzekł im : Ten / który rozśiewá dobre náśienie / jeſt Syn cżłowiecży ;
A rolá jeſt świát / á dobre náśienie ſą ſynowie króleſtwá ; ále kąkol ſą ſynowie onego złego ;
Nieprzyjáćiel záśię / który go rozśiáł / jeſtći dyjábeł / á żniwo jeſt dokonánie świátá / á żeńcy ſą Aniołowie.
Jáko tedy zbieráją kąkol / á pálą go ogniem / ták będzie przy dokonániu świátá tego.
Pośle Syn cżłowiecży Anioły ſwoje / á oni zbiorą z króleſtwá jego wƺyſtkie zgorƺeniá / y te / którzy niepráwość cżynią ;
Y wrzucą je w piec ogniſty / tám będzie płácż y zgrzytánie zębów.
Tedy ſpráwiedliwi lśnić śię będą jáko ſłońce w króleſtwie Ojcá ſwego. Kto má uƺy ku ſłuchániu / niecháj ſłuchá.
Záśię podobne jeſt króleſtwo niebieſkie ſkárbowi ſkrytemu w roli / który ználázłƺy cżłowiek / ſkrył / y od rádośći / którą miáł z niego / odchodzi / y wƺyſtko / co má / ſprzedáje / y kupuje onę rolę.
Záśię podobne jeſt króleſtwo niebieſkie cżłowiekowi kupcowi / ƺukájącemu pięknych pereł ;
Który ználázłƺy jednę perłę bárdzo drogą / odƺedł / y poſprzedáwáł wƺyſtko / co miáł / y kupił ją.
Záśię podobne jeſt króleſtwo niebieſkie niewodowi zápuƺcżonemu w morze / y ryby wƺelkiego rodzáju zágárniájącemu.
Który gdy był pełen / wyćiągnęli rybitwi ná brzeg / á uśiádłƺy / wybieráli dobre ryby w nácżyniá / á złe precż wyrzucáli.
Tákći będzie przy dokonániu świátá ; wynijdą Aniołowie / y wyłącżą złe z pośrodku ſpráwiedliwych /
Y wrzucą je w piec ogniſty ; tám będzie płácż y zgrzytánie zębów.
Rzekł im Jezuſ : Wyrozumieliśćie to wƺyſtko? Rzekli mu : Ták / Pánie!
A on im rzekł : Przetoż káżdy náucżony w Piśmie / wyćwicżony w króleſtwie niebieſkiem / podobny jeſt cżłowiekowi goſpodárzowi / który wynośi z ſkárbu ſwego nowe y ſtáre rzecży.
Y ſtáło śię / gdy Jezuſ dokońcżył tych podobieńſtw / puśćił śię ſtámtąd.
A przyƺedłƺy do ojcżyzny ſwojej / náucżáł je w bóżnicy ich / ták iż śię bárdzo zdumiewáli y mówili : Skądże temu tá mądrość / y tá moc?
Izáż ten nie jeſt on ſyn ćieśli? Izáż mátki jego nie zowią Máryją / á bráćiá jego Jákób / y Jozeſ / y Szymon / y Judáſ?
A śioſtry jego izáli wƺyſtkie u náſ nie ſą? Skądże tedy temu to wƺyſtko?
Y gorƺyli śię z niego ; ále Jezuſ rzekł im : Nie jeſt prorok beze cżći / tylko w ojcżyźnie ſwojej y w domu ſwoim.
Y nie ucżynił tám wiele cudów dlá niedowiárſtwá ich.
