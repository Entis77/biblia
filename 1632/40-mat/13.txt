A dniá onego wyƺedƺy JEzus z domu / uśiádł nád morzem :
Y zebrał śię do niego wielki lud / ták / iż wſtąpiwƺy w łódź / śiedźiał / á wƺyſtek lud ſtał ná brzegu.
Y mówił do nich wiele w podobieńſtwách y rzekł : Oto wyƺedł rozśiewcá / áby rozśiewał ;
A gdy on rozśiewał / niektóre pádło podle drogi ; y przylećiáły ptáki / á pozobáły je.
Drugie záśię pádło ná miejſce opocżyſte / gdźie nie miáło wiele źiemi ; y wnet weƺło / iż nie miáło głębokośći źiemi.
Ale gdy ſłońce weƺło / wygorzáło / á iż nie miáło korzeniá / uſchło.
A drugie pádło miedzy ćiernie / y wzroſły ćiernie á záduśiły je.
A drugie pádło ná źiemię dobrą / y wydáło pożytek / jedno ſetny / drugie ƺeśćdźieśiątny / á drugie trzydźieſtny.
Kto ma uƺy ku ſłuchániu / niecháj ſłuchá.
Tedy przyſtąpiwƺy ucżniowie / rzekli mu : Dlácżegoż im w podobieńſtwách mówiƺ?
A on odpowiádájąc / rzekł im : Wam dáno wiedźieć tájemnicę króleſtwá niebieſkiego / ále onym nie dáno ;
Abowiem kto ma / będźie mu dáno / y obfitowáć będźie ; ále kto niemá / y to / co ma / będźie od niego odjęto.
Dlátegoć im w podobieńſtwách mówię / iż widząc nie widzą / y ſłyƺąc nie ſłyƺą / áni rozumieją.
Y pełni śię w nich proroctwo Izájáƺowe / które mówi : Słuchem ſłucháć będźiećie / ále nie zrozumiećie : y widząc widźieć będźiećie / ále nie ujrzyćie ;
Abowiem zátyło ſerce ludu tego / á uƺymá ćiężko ſłyƺeli / y ocży ſwe zámrużyli / żeby kiedy ocżymá nie widźieli y uƺymá nie ſłyƺeli / á ſercem nie zrozumieli / y nie náwróćili śię / á uzdrowiłbym je.
Ale ocży wáƺe błogoſłáwione / że widzą / y uƺy wáƺe / że ſłyƺą ;
Bo zápráwdę powiádám wam / iż wiele proroków y ſpráwiedliwych żądáło widźieć to / co wy widźićie / ále nie widźieli / y ſłyƺeć to / co ſłyƺyćie / ále nie ſłyƺeli.
Wy tedy ſłuchájćie podobieńſtwá onego rozśiewcy.
Gdy kto ſłuchá ſłowá o tym króleſtwie / á nie rozumie / przychodźi on zły y porywá to / co wśiáno w ſerce jego ; tenći jeſt on / który podle drogi pośiány jeſt.
A ná opocżyſtych miejſcách pośiány / ten jeſt / który ſłuchá ſłowá y záráz je z rádośćią przyjmuje ;
Ale niemá korzeniá w ſobie / lecż docżeſny jeſt ; á gdy przychodźi ućiſk / álbo prześládowánie dla ſłowá / wnet śię gorƺy.
A miedzy ćiernie pośiány / ten jeſt / który ſłuchá ſłowá ; ále piecżołowánie świátá tego y omámienie bogáctw záduƺá ſłowo / y ſtáje śię bez pożytku.
A ná dobrej źiemi pośiány / jeſt ten / który ſłuchá ſłowá y rozumie / tenći pożytek przynośi ; á przynośi jeden ſetny / drugi ƺeśćdźieśiątny / á drugi trzydźieſtny.
Drugie podobieńſtwo przełożył im / mówiąc : Podobne jeſt króleſtwo niebieſkie cżłowiekowi / rozśiewájącemu dobre náśienie ná roli ſwojey.
A gdy ludźie záſnęli / przyƺedł nieprzyjáćiel jego / y náśiał kąkolu miedzy pƺenicą / y odƺedł.
A gdy uroſłá tráwá y pożytek przynioſłá / tedy śię pokázał y kąkol.
Tedy przyſtąpiwƺy ſłudzy goſpodárſcy / rzekli mu : Pánie! izáliś dobrego náśieniá nie náśiał ná roli twojey? Skądże tedy ma kąkol?
A on im rzekł : Nieprzyjáćiel cżłowiek to ucżynił. Y rzekli ſłudzy do niego : A chceƺże / iż pójdźiemy / á zbierzemy go?
A on rzekł : Nie! byśćie ſnadź zbierájąc kąkol / nie wykorzenili záráz z nim y pƺenicy.
Dopuśććie obojgu ſpołem rość áż do żniwá ; á cżáſu żniwá rzekę żeńcom : Zbierzćie pierwej kąkol á zwiążćie go w ſnopki ku ſpáleniu ; ále pƺenicę zgromádźćie do gumná mojego.
Inƺe podobieńſtwo przełożył im / mówiąc : Podobne jeſt króleſtwo niebieſkie źiárnu gorcżycżnemu / które wźiąwƺy cżłowiek / wśiał ná roli ſwojey.
Które nájmniejƺeć jeſt ze wƺyſtkich náśion ; ále kiedy urośnie / nájwiękƺe jeſt ze wƺyſtkich járzyn / y ſtáje śię drzewem / ták / iż ptáki niebieſkie przylátując / gniázdá ſobie cżynią ná gáłązkách jego.
Inƺe podobieńſtwo powiedźiał im : Podobne jeſt króleſtwo niebieſkie kwáſowi / który wźiąwƺy niewiáſtá / zákryłá we trzy miáry mąki / áżby wƺyſtká ſkwáśniáłá.
To wƺyſtko mówił JEzus w podobieńſtwách do ludu / á bez podobieńſtwá nie mówił do nich ;
Aby śię wypełniło / co powiedźiáno przez proroká mówiącego : Otworzę w podobieńſtwách uſtá moje / wypowiem ſkryte rzecży od záłożeniá świátá.
Tedy rozpuśćiwƺy on lud / przyƺedł JEzus do domu ; y przyſtąpili do niego ucżniowie jego / mówiąc : Wyłóż nam podobieńſtwo o kąkolu oney roli.
A on odpowiádájąc / rzekł im : Ten / który rozśiewá dobre náśienie / jeſt Syn cżłowiecży ;
A rolá jeſt świát / á dobre náśienie ſą ſynowie króleſtwá ; ále kąkol ſą ſynowie onego złego ;
Nieprzyjáćiel záśię / który go rozśiał / jeſtći dyjábeł / á żniwo jeſt dokonánie świátá / á żeńcy ſą Aniołowie.
Jáko tedy zbieráją kąkol / á pálą go ogniem / ták będźie przy dokonániu świátá tego.
Pośle Syn cżłowiecży Anioły ſwoje / á oni zbiorą z króleſtwá jego wƺyſtkie zgorƺeniá / y te / którzy niepráwość cżynią ;
Y wrzucą je w piec ogniſty / tám będźie płácż y zgrzytánie zębów.
Tedy ſpráwiedliwi lśnić śię będą jáko ſłońce w króleſtwie Ojcá ſwego. Kto ma uƺy ku ſłuchániu / niecháj ſłuchá.
Záśię podobne jeſt króleſtwo niebieſkie ſkárbowi ſkrytemu w roli / który ználázƺy cżłowiek / ſkrył / y od rádośći / którą miał z niego / odchodźi / y wƺyſtko / co ma / ſprzedáje / y kupuje onę rolę.
Záśię podobne jeſt króleſtwo niebieſkie cżłowiekowi kupcowi / ƺukájącemu pięknych pereł ;
Który ználázƺy jednę perłę bárzo drogą / odƺedł / y poſprzedáwał wƺyſtko / co miał / y kupił ją.
Záśię podobne jeſt króleſtwo niebieſkie niewodowi zápuƺcżonemu w morze / y ryby wƺelkiego rodzáju zágárniájącemu.
Który gdy był pełen / wyćiągnęli rybitwi ná brzeg / á uśiádƺy / wybieráli dobre ryby w nácżyniá / á złe precż wyrzucáli.
Tákći będźie przy dokonániu świátá ; wynijdą Aniołowie / y wyłącżą złe z pośrzodku ſpráwiedliwych /
Y wrzucą je w piec ogniſty ; tám będźie płácż y zgrzytánie zębów.
Rzekł im JEzus : Wyrozumieliśćie to wƺyſtko? Rzekli mu : Ták / Pánie!
A on im rzekł : Przetoż káżdy náucżony w Piśmie / wyćwicżony w króleſtwie niebieſkiem / podobny jeſt cżłowiekowi goſpodárzowi / który wynośi z ſkárbu ſwego nowe y ſtáre rzecży.
Y ſtáło śię / gdy JEzus dokońcżył tych podobieńſtw / puśćił śię ſtámtąd.
A przyƺedƺy do ojcżyzny ſwojey / náucżał je w bóżnicy ich / ták iż śię bárzo zdumiewáli y mówili : Skądże temu tá mądrość / y tá moc?
Izáż ten nie jeſt on ſyn ćieśli? Izáż mátki jego nie zowią Máryją / á bráćia jego Jákób / y Jozes / y Szymon / y Judás?
A śioſtry jego izali wƺyſtkie u nas nie ſą? Skądże tedy temu to wƺyſtko?
Y gorƺyli śię z niego ; ále JEzus rzekł im : Nie jeſt prorok beze cżći / tylko w ojcżyźnie ſwojey y w domu ſwoim.
Y nie ucżynił tám wiele cudów dla niedowiárſtwá ich.
