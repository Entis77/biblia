Nie ſądźćie / ábyśćie nie byli ſądzeni ;
Abowiem jákim ſądem ſądzićie / tákim ſądzeni będziećie / y jáką miárą mierzyćie / táką wám odmierzono będzie.
A cżemuż widziƺ źdźbło w oku brátá twego / á bálki / którá jeſt w oku twojim / nie bácżyƺ?
Albo jákoż rzecżeƺ brátu twemu : Dopuść / iż wyjmę źdźbło z oká twego / á oto bálká jeſt w oku twojim.
Obłudniku! wyjmij pierwej bálkę z oká twego / tedy przejrzyƺ / ábyś wyjął źdźbło z oká brátá twego.
Nie dáwájćie świętego pſom / áni mieććie pereł wáƺych przed świnie / by ich ſnáć nie podeptáły nogámi ſwemi / y obróćiwƺy śię / nie rozƺárpáły wáſ.
Prośćie / á będzie wám dáno / ƺukájćie / á znájdziećie ; kołácżćie / á będzie wám otworzono.
Káżdy bowiem / kto prośi / bierze ; á kto ƺuká / znájduje ; á temu / co kołácże / będzie otworzono.
Y któryż z wáſ jeſt cżłowiek / którego prośiłliby ſyn jego o chleb / izáli mu dá kámień?
A prośiłliby o rybę / izáli mu dá wężá?
Jeźli wy tedy będąc złymi / umiećie dáry dobre dáwáć dziećiom wáƺym / cżemże więcej Ojćiec wáƺ / który jeſt w niebieśiech / dá rzecży dobre tym / którzy go proƺą.
Wƺyſtko tedy / cobyśćie chćieli / áby wám ludzie cżynili / ták y wy cżyńćie im ; tenći bowiem jeſt zákon y prorocy.
Wchodźćie przez ćiáſną brámę ; ábowiem przeſtronná jeſt brámá y ƺeroká drogá / którá prowádzi ná zátrácenie / á wiele ich jeſt / którzy przez nię wchodzą.
A ćiáſná jeſt brámá y wąſká drogá / którá prowádzi do żywotá ; á máło ich jeſt / którzy ją znájdują.
A ſtrzeżćie śię fáłƺywych proroków / którzy przychodzą do wáſ w odzieniu owcżem / ále wewnątrz ſą wilcy drápieżni.
Z owoców ich poznáćie je ; izáli zbieráją z ćierniá groná winne / álbo z oſtu figi?
Tákći wƺelkie drzewo dobre owoce dobre przynośi ; ále złe drzewo owoce złe przynośi.
Nie może dobre drzewo owoców złych przynośić / áni drzewo złe owoców dobrych przynośić.
Wƺelkie drzewo / które nie przynośi owocu dobrego / bywá wyćięte y w ogień wrzucone.
A ták z owoców ich poznáćie je.
Nie káżdy / który mi mówi : Pánie / Pánie! wnijdzie do króleſtwá niebieſkiego ; ále który cżyni wolę Ojcá mojego / który jeſt w niebieśiech.
Wiele ich rzecże mi dniá onego : Pánie / Pánie! izáżeśmy w imieniu twojim nie prorokowáli / y w imieniu twojim dyjábłów nie wygániáli / y w imieniu twojim wiele cudów nie cżynili?
A tedy im wyznám : Żem wáſ nigdy nie znáł ; odſtąpćie ode mnie / którzy cżynićie niepráwość.
Wƺelkiego tedy / który ſłuchá tych ſłów moich y cżyni je / przypodobám mężowi mądremu / który zbudowáł dom ſwój ná opoce ;
Y ſpádł gwáłtowny deƺcż / y przyƺłá powódź / y wiátry wiáły / y uderzyły ná on dom / ále nie upádł / bo był záłożony ná opoce.
A wƺelki / który ſłuchá tych ſłów moich / á nie cżyni ich / przypodobány będzie mężowi głupiemu / który zbudowáł dom ſwój ná piáſku ;
Y ſpádł deƺcż gwáłtowny / y przyƺłá powódź / y wiátry wiáły á uderzyły ná on dom / y upádł / á był wielki upádek jego.
Y ſtáło śię / gdy dokońcżył Jezuſ tych ſłów / że śię zdumiewáł lud nád náuką jego.
Abowiem je ucżył jáko moc májący / á nie jáko náucżeni w Piśmie.
