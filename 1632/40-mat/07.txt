Nie ſądźćie / ábyśćie nie byli ſądzeni.
Abowiem jákim ſądem ſądźićie / tákim ſądzeni będźiećie ; y jáką miárą mierzyćie / táką wam odmierzono będźie.
A cżemuż widźiƺ źdźbło w oku brátá twego / á bálki która jeſt w oku twojim nie bacżyƺ?
Albo jákoż rzecżeƺ brátu twemu ; Dopuść iż wyjmę źdźbło z oká twego : á oto bálká jeſt w oku twojim?
Obłudniku / wyjmi pierwey bálkę z oká twego : tedy przejrzyƺ / ábyś wyjął źdźbło z oká brátá twego.
Nie dawajćie świętego pſom ; áni miecćie pereł wáƺych przed świnie : by ich ſnadź nie podeptáły nogámi ſwymi ; y obróćiwƺy śię / nie rozƺárpáły was.
Prośćie / á będźie wam dano ; ƺukajćie / á znajdźiećie ; kołácćie / á będźie wam otworzono.
Káżdy bowiem kto prośi / bierze ; á kto ƺuka / znájduje ; á temu co kołáce / będźie otworzono.
Y któryż z was jeſt cżłowiek / którego prośiłliby ſyn jego o chleb / izali mu da kámień?
A prośiłliby o rybę / izali mu da wężá?
Jeſli wy tedy będąc złymi / umiećie dáry dobre dawáć dźiećiom wáƺym ; cżymże więcey Oćiec wáƺ który jeſt w niebieśiech / da rzecży dobre tym / którzy go proƺą.
Wƺyſtko tedy cobyśćie chćieli áby wam ludźie cżynili / ták y wy cżyńćie im : tenći bowiem jeſt Zakon y Prorocy.
Wchodźćie przez ćiáſną bramę : ábowiem przeſtronna jeſt bramá y ƺeroka drogá / która prowádźi ná zátrácenie ; á wiele ich jeſt którzy przez nię wchodzą.
A ćiáſna jeſt bramá y wąſka drogá która prowádźi do żywotá : á máło ich jeſt którzy ją znájdują.
A ſtrzeżćie śię fałƺywych Proroków / którzy przychodzą do was w odźieniu owcżym ; ále we wnątrz ſą wilcy drapieżni.
Z owoców ich poznaćie je : Izali zbieráją z ćiernia groná winne / álbo z oſtu figi?
Tákći wƺelkie drzewo dobre / owoce dobre przynośi : ále złe drzewo owoce złe przynośi.
Nie może dobre drzewo owoców złych przynośić ; áni drzewo złe owoców dobrych przynośić.
Wƺelkie drzewo które nie przynośi owocu dobrego / bywa wyćięte y w ogień wrzucone.
A ták z owoców ich poznaćie je.
Nie káżdy który mi mówi ; PAnie / PAnie / wnidźie do króleſtwá niebieſkiego : ále który cżyni wolą Ojcá mojego który jeſt w niebieśiech.
Wiele ich rzecże mi dniá onego ; PAnie / PAnie / Izażeſmy w Imieniu twojim nie prorokowáli / y w Imieniu twojim dyabłów nie wyganiáli / y w Imieniu twojim wiele cudów nie cżynili?
A tedy im wyznam ; Żem was nigdy nie znał : Odſtąpćie ode mnie którzy cżynićie niepráwość.
Wƺelkiego tedy / który ſłucha tych ſłów mojich y cżyni je / przypodobam mężowi mądremu / który zbudował dom ſwój ná opoce :
Y zpadł gwałtowny deƺcż / y przyƺłá powódź / y wiátry wiały / y uderzyły ná on dom ; ále nie upadł / bo był záłożony ná opoce.
A wƺelki który ſłucha tych ſłów mojich á nie cżyni ich / przypodobány będźie mężowi głupiemu / który zbudował dom ſwój ná piaſku :
Y zpadł deƺcż gwałtowny / y przyƺłá powódź / y wiátry wiały / á uderzyły ná on dom ; y upadł / á był wielki upadek jego.
Y ſtáło śię gdy dokońcżył JEzus tych ſłów / <i>że</i> śię zdumiewał lud nád náuką jego.
Abowiem je ucżył jáko moc májący / á nie jáko Náucżeni w piśmie.
