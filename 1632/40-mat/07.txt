Nie ſądźćie / ábyśćie nie byli ſądzeni ;
Abowiem jákim ſądem ſądźićie / tákim ſądzeni będźiećie / y jáką miárą mierzyćie / táką wam odmierzono będźie.
A cżemuż widźiƺ źdźbło w oku brátá twego / á bálki / która jeſt w oku twojim / nie bácżyƺ?
Albo jákoż rzecżeƺ brátu twemu : Dopuść / iż wyjmę źdźbło z oká twego / á oto bálká jeſt w oku twojim.
Obłudniku! wyjmij pierwej bálkę z oká twego / tedy przejrzyƺ / ábyś wyjął źdźbło z oká brátá twego.
Nie dáwájćie świętego pſom / áni mieććie pereł wáƺych przed świnie / by ich ſnadź nie podeptáły nogámi ſwemi / y obróćiwƺy śię / nie rozƺárpáły was.
Prośćie / á będźie wam dáno / ƺukájćie / á znájdźiećie ; kołácżćie / á będźie wam otworzono.
Káżdy bowiem / kto prośi / bierze ; á kto ƺuká / znájduje ; á temu / co kołácże / będźie otworzono.
Y któryż z was jeſt cżłowiek / którego prośiłliby ſyn jego o chleb / izali mu da kámień?
A prośiłliby o rybę / izali mu da wężá?
Jeſli wy tedy będąc złymi / umiećie dáry dobre dáwáć dźiećiom wáƺym / cżemże więcey Oćiec wáƺ / który jeſt w niebieśiech / da rzecży dobre tym / którzy go proƺą.
Wƺyſtko tedy / cobyśćie chćieli / áby wam ludźie cżynili / ták y wy cżyńćie im ; tenći bowiem jeſt zakon y prorocy.
Wchodźćie przez ćiáſną bramę ; ábowiem przeſtronná jeſt bramá y ƺeroká drogá / która prowádźi ná zátrácenie / á wiele ich jeſt / którzy przez nię wchodzą.
A ćiáſná jeſt bramá y wąſká drogá / która prowádźi do żywotá ; á máło ich jeſt / którzy ją znájdują.
A ſtrzeżćie śię fáłƺywych proroków / którzy przychodzą do was w odźieniu owcżem / ále wewnątrz ſą wilcy drápieżni.
Z owoców ich poznáćie je ; izali zbieráją z ćierniá groná winne / álbo z oſtu figi?
Tákći wƺelkie drzewo dobre/ owoce dobre przynośi ; ále złe drzewo owoce złe przynośi.
Nie może dobre drzewo owoców złych przynośić / áni drzewo złe owoców dobrych przynośić.
Wƺelkie drzewo / które nie przynośi owocu dobrego / bywa wyćięte y w ogień wrzucone.
A ták z owoców ich poznáćie je.
Nie káżdy / który mi mówi : Pánie / Pánie! wnijdźie do króleſtwá niebieſkiego ; ále który cżyni wolę Ojcá mojego / który jeſt w niebieśiech.
Wiele ich rzecże mi dniá onego : Pánie / Pánie! izáżeśmy w imieniu twojim nie prorokowáli / y w imieniu twojim dyjábłów nie wygániáli / y w imieniu twojim wiele cudów nie cżynili?
A tedy im wyznam : Żem was nigdy nie znał ; odſtąpćie ode mnie / którzy cżynićie niepráwość.
Wƺelkiego tedy / który ſłuchá tych ſłów mojich y cżyni je / przypodobám mężowi mądremu / który zbudował dom ſwój ná opoce ;
Y ſpádł gwáłtowny deƺcż / y przyƺłá powódź / y wiátry wiáły / y uderzyły ná on dom ; ále nie upádł / bo był záłożony ná opoce.
A wƺelki który ſłuchá tych ſłów mojich á nie cżyni ich / przypodobány będźie mężowi głupiemu / który zbudował dom ſwój ná piáſku ;
Y ſpádł deƺcż gwáłtowny / y przyƺłá powódź / y wiátry wiáły / á uderzyły ná on dom ; y upádł / á był wielki upádek jego.
Y ſtáło śię gdy dokońcżył JEzus tych ſłów / że śię zdumiewał lud nád náuką jego.
Abowiem je ucżył jáko moc májący / á nie jáko náucżeni w Piśmie.
