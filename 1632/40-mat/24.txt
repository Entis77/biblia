A wyƺedłƺy JEzuſ z kośćiołá ƺedł ; y przyſtąpili ucżniowie jego / áby mu ukázáli budowániá kośćielne.
Y rzekł im JEzuſ : Izáż nie widźićie tego wƺyſtkiego? Zápráwdę powiádám wám / nie zoſtánie tu kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedźiáł ná górze oliwney / przyſtąpili do niego ucżniowie oſobno / mówiąc : Powiedz nám / kiedy śię to ſtánie / y co zá znák przyjśćiá twego y dokonániá świátá?
Y odpowiádájąc JEzuſ / rzekł im : Pátrzćie / áby was kto nie zwiódł.
Abowiem wiele ich przyjdźie pod imieniem mojem / mówiąc : Jám jeſt CHryſtuſ ; y wiele ich zwiodą.
Y uſłyƺyćie wojny y wieśći o wojnách ; pátrzćież / ábyśćie ſobą nie trwożyli ; ábowiem muśi to wƺyſtko być / ále jeƺcże nie tu jeſt koniec.
Abowiem powſtánie náród przećiwko národowi / y króleſtwo przećiwko króleſtwu / y będą głody y mory y trzęśieniá źiemi miejſcámi.
Ale to wƺyſtko jeſt pocżątkiem boleśći.
Tedy was podádzą w udręcżenie / y będą was zábijáć / y będźiećie w nienáwiśći u wƺyſtkich národów dla imieniá mego.
A tedy wiele śię ich zgorƺy / á jedni drugich wydádzą / y jedni drugich nienáwidźieć będą.
Y wiele fáłƺywych proroków powſtánie / y zwiodą wielu.
A iż śię rozmnoży niepráwość / oźiębnie miłość wielu.
Ale kto wytrwá áż do końcá / ten zbáwion będźie.
Y będźie kázáná tá Ewángielijá króleſtwá po wƺyſtkim świećie / ná świádectwo wƺyſtkim národom. A tedyć przyjdźie koniec.
Przetoż gdy ujrzyćie obrzydliwość ſpuſtoƺeniá / opowiedźiáną przez Dánijelá proroká / ſtojącą ná miejſcu świętym / ( kto cżytá / niecháj uwáżá. )
Tedy ći / co będą w źiemi Judzkiej / niech ućiekáją ná góry ;
A kto ná dáchu / niecháj nie zſtępuje / áby co wźiął z domu ſwego ;
A kto ná roli / niech śię názád nie wrácá / áby wźiął ƺáty ſwe.
A biádá brzemiennym y pierśiámi kármiącym w one dni!
Przetoż módlćie śię / áby nie było ućiekánie wáƺe w źimie / álbo w ſábát.
Abowiem náoncżáſ będźie wielki ućiſk / jáki nie był od pocżątku świátá áż dotąd / áni potem będźie.
A gdyby nie były ſkrócone one dni / nie byłoby zbáwione żádne ćiáło ; ále dla wybránych będą ſkrócone one dni.
Tedy jeſliby wám kto rzekł : Oto tu jeſt CHryſtuſ / álbo tám / nie wierzćie.
Abowiem powſtáną fáłƺywi CHryſtuſowie / y fáłƺywi prorocy / y cżynić będą známioná wielkie / y cudá / ták / iżby zwiedli ( by możná ) y wybráne.
Otom wám przepowiedźiáł.
Jeſliby wám tedy rzekli : Oto ná puƺcży jeſt / nie wychodźćie ; oto w komorách / nie wierzćie.
Abowiem jáko błyſkáwicá wychodźi od wſchodu ſłońcá y ukázuje śię áż ná záchód / ták będźie y przyjśćie Syná cżłowiecżego.
Bo gdźiekolwiek będźie śćierw / tám śię zgromádzą y orły.
A záráz po utrápieniu onych dni ſłońce śię záćmi / á kśiężyc nie dá jáſnośći ſwojey / y gwiázdy będą pádáć z niebá / y mocy niebieſkie poruƺą śię.
Tedyć śię ukáże známię Syná cżłowiecżego ná niebie / á tedy będą nárzekáć wƺyſtkie pokoleniá źiemi / y ujrzą Syná cżłowiecżego / przychodzącego ná obłokách niebieſkich z mocą y z chwáłą wielką ;
Y pośle Anioły ſwoje z trąbą głoſu wielkiego / y zgromádzą wybráne jego od cżterech wiátrów / od krájów niebioſ áż do krájów ich.
A od drzewá figowego náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmłádzá y liśćie wypuƺcżá / poznájećie / iż bliſko jeſt láto.
Tákże y wy / gdy ujrzyćie to wƺyſtko / poznáwájćie / iż bliſko jeſt / á we drzwiách.
Zápráwdę powiádám wám / że nie przeminie ten wiek / áżby śię to wƺyſtko ſtáło.
Niebo y źiemiá przeminą / ále ſłowá moje nie przeminą.
A o onym dniu y godźinie nikt nie wie / áni Aniołowie niebieſcy / tylko ſám Ojćiec mój.
Ale jáko było zá dni Noego / ták będźie y przyjśćie Syná cżłowiecżego.
Abowiem jáko zá dni onych przed potopem jedli / y pili / y ożeniáli śię y zá mąż wydáwáli / áż do onego dniá / którego wƺedł Noe do korábiá /
Y nie ſpoſtrzegli śię / áż przyƺedł potop y zábráł wƺyſtkie : ták będźie y przyjśćie Syná cżłowiecżego.
Tedy będą dwáj ná roli ; jeden będźie wźięty á drugi zoſtáwiony ;
Dwie będą mleć we młynie ; jedná będźie wźiętá á drugá zoſtáwioná ;
Cżujćież tedy / poniewáż nie wiećie / którey godźiny Pán wáƺ przyjdźie.
A to wiedzćie / że / gdyby wiedźiáł goſpodárz / o którey ſtráży złodźiej má przyjść / wżdyby cżuł / y nie dáłby podkopáć domu ſwego.
Przetoż y wy bądźćie gotowi ; bo którey śię godźiny nie ſpodźiejećie / Syn cżłowiecży przyjdźie.
Któryż tedy jeſt ſługá wierny y roztropny / którego poſtánowił pán jego nád cżeládźią ſwoją / áby im dáwáł pokárm ná cżáſ ſłuƺny?
Błogoſłáwiony on ſługá / któregoby / przyƺedłƺy Pán jego / ználázł ták cżyniącego ;
Zápráwdę powiádám wám / że go nád wƺyſtkiemi dobrámi ſwemi poſtánowi.
A jeſliby rzekł on zły ſługá w ſercu ſwojim : Odwłácżá pán mój z przyjśćiem ſwojim ;
Y pocżąłby bić ſpółſługi / á jeść y pić z pijánicámi :
Przyjdźie pán ſługi onego / dniá / którego śię nie ſpodźieje / y godźiny / którey nie wie ;
Y odłącży go / á cżęść jego położy z obłudnikámi ; tám będźie płácż y zgrzytánie zębów.
