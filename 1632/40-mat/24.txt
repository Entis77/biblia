A wyƺedƺy JEzus z Kośćiołá / ƺedł : y przyſtąpili ucżniowie jego / áby mu ukazáli budowánia kośćielne.
Y rzekł im JEzus ; Izaż nie widźićie tego wƺyſtkiego? Záprawdę powiádam wam / nie zoſtánie tu kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedźiał ná górze Oliwney / przyſtąpili do niego ucżniowie oſobno / mówiąc : Powiedz nam kiedy śię to ſtánie? Y co zá znák przyśćia twego y dokonánia świátá.
Y odpowiádájąc JEzus / rzekł im ; Pátrzćie áby was kto nie zwiódł.
Abowiem wiele ich przydźie pod Imieniem mojim / mówiąc ; Jam jeſt CHryſtus : y wiele ich zwiodą.
Y uſłyƺyćie wojny y wieśći o wojnách : pátrzćież ábyśćie ſobą nie trwożyli ; ábowiem muśi to wƺyſtko być : ále jeƺcże nie <i>tu jeſt</i> koniec.
Abowiem powſtánie naród przećiwko narodowi / y króleſtwo przećiwko króleſtwu : y będą głody / y mory / y trzęśienia źiemie miejſcámi.
Ale to wƺyſtko jeſt pocżątkiem boleśći.
Tedy was podádzą w udręcżenie / y będą was zábijáć : y będźiećie w nienawiśći u wƺyſtkich narodów dla Imienia mego.
A tedy wiele śię ich zgorƺy : á jedni drugich wydádzą / y jedni drugich nienawidźieć będą.
Y wiele fałƺywych Proroków powſtánie ; y zwiodą wielu.
A iż śię rozmnoży niepráwość / oźiębnie miłość wielu.
Ale kto wytrwa áż do końcá / ten zbáwion będźie.
Y będźie kazána tá Ewángelia króleſtwá po wƺyſtkim świećie / ná świádectwo wƺyſtkim narodom. A tedyć przydźie koniec.
Przetoż / gdy ujrzyćie obrzydliwość ſpuſtoƺenia / opowiedźiáną przez Dánielá Proroká / ſtojącą ná miejſcu świętym : ( kto cżyta / niechaj uważa. )
Tedy ći co <i>będą</i> w źiemi Judſkiey / niech ućiekáją ná góry.
A kto ná dáchu / niechaj nie zſtępuje / áby co wźiął z domu ſwego.
A kto ná roli / niech śię ná zad nie wraca / áby wźiął ƺáty ſwe.
A biádá brzemiennym y pierśiámi karmiącym w one dni.
Przetoż módlćie śię / áby nie było ućiekánie wáƺe źimie / álbo w Sábbát.
Abowiem ná on cżás będźie wielki ućiſk / jáki nie był od pocżątku świátá áż do tąd / áni potym będźie.
A gdyby nie były ſkrócone one dni / nie byłoby zbáwione żadne ćiáło : ále dla wybránych / będą ſkrócone one dni.
Tedy jeſliby wam kto rzekł ; Oto tu jeſt CHryſtus / álbo tám : nie wierzćie.
Abowiem powſtáną fałƺywi CHryſtuſowie / y fałƺywi Prorocy / y cżynić będą známioná wielkie y cudá : ták / iżby zwiedli ( by można ) y wybráne.
Otom wam przepowiedźiał.
Jeſliby wam tedy rzekli : Oto ná puƺcży jeſt / nie wychodźćie : Oto w komorách / nie wierzćie.
Abowiem jáko błyſkáwicá wychodźi od Wſchodu ſłońcá / y ukázuje śię áż ná Zachód : ták będźie y przyśćie Syná cżłowiecżego.
Bo gdźiekolwiek będźie śćierw / tám śię zgromádzą y orłowie.
A záraz po utrapieniu onych dni / ſłońce śię záćmi / á kśiężyc nie da jáſnośći ſwojey / y gwiazdy będą pádáć z niebá / y mocy niebieſkie poruƺą śię.
Tedyć śię ukaże známię Syná cżłowiecżego ná niebie : á tedy będą nárzekáć wƺyſtkie pokolenia źiemie / y ujrzą Syná cżłowiecżego przychodzącego ná obłokách niebieſkich z mocą y z chwałą wielką :
Który pośle Anjoły ſwoje z trąbą głoſu wielkiego : y zgromádzą wybráne jego od cżterech wiátrów / od krájów niebios áż do krájów ich.
A od drzewá figowego náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmładza / y liśćie śię puƺcża / poznawaćie iż bliſko jeſt láto.
Tákże y wy gdy ujrzyćie to wƺyſtko / poznawajćie iż bliſko jeſt / <i>á</i> we drzwiách.
Záprawdę powiádam wam / że nie przeminie ten wiek / áżby śię to wƺyſtko ſtáło.
Niebo y źiemiá przeminą / ále ſłowá moje nie przeminą.
A o onym dniu y godźinie nikt nie wie / áni Anjołowie niebieſcy : tylko ſam Oćiec mój.
Ale jáko było zá dni Noego / ták będźie y przyśćie Syná cżłowiecżego.
Abowiem jáko zá dni onych przed potopem / jedli / y pili / y ożeniáli śię / y zá mąż wydawáli áż do onego dniá którego wƺedł Noe do Korabiá :
Y nie poſtrzegli śię / áż przyƺedł potop / y zábrał wƺyſtkie : Ták będźie y przyśćie Syná cżłowiecżego.
Tedy będą dwá ná roli : jeden będźie wźięty / á drugi zoſtáwiony.
Dwie będą mleć we młynie : jedná będźie wźięta / á druga zoſtáwiona.
Cżujćież tedy / ponieważ nie wiećie którey godźiny PAN wáƺ przydźie.
A to wiedzćie / że gdyby wiedźiał goſpodarz / o którey ſtraży złodźiey ma przyść / wżdyby cżuł / y nie dałby podkopáć domu ſwego.
Przetoż y wy bądźćie gotowi : bo którey śię godźiny nie ſpodźiejećie / Syn cżłowiecży przydźie.
Któryż tedy jeſt ſługá wierny y roſtropny / którego poſtánowił Pan jego nád cżeládźią ſwoją / áby im dawał pokarm ná cżás <i>słuƺny</i>.
Błogoſłáwiony on ſługá / któregoby / przyƺedƺy Pan jego / ználazł ták cżyniącego.
Záprawdę powiádam wam / że go nád wƺyſtkimi dobrámi ſwymi poſtánowi.
Ale jeſliby rzekł on zły ſługá w ſercu ſwojim ; Odwłacża Pan mój z przyśćiem <i>ſwojim</i> :
Y pocżąłby bić ſpół-ſługi / á jeść y pić z pijánicámi :
Przydźie Pan ſługi onego / dniá którego śię nieſpodźieje / y godźiny którey nie wzwie :
Y odłącży go : á cżęść jego położy z obłudnikámi : Tám będźie płácż y zgrzytánie zębów.
