A wyƺedłƺy Jezuſ z kośćiołá ƺedł ; y przyſtąpili ucżniowie jego / áby mu ukázáli budowániá kośćielne.
Y rzekł im Jezuſ : Izáż nie widzićie tego wƺyſtkiego? Zápráwdę powiádám wám / nie zoſtánie tu kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedziáł ná górze oliwnej / przyſtąpili do niego ucżniowie oſobno / mówiąc : Powiedz nám / kiedy śię to ſtánie / y co zá znák przyjśćiá twego y dokonániá świátá?
Y odpowiádájąc Jezuſ / rzekł im : Pátrzćie / áby wáſ kto nie zwiódł.
Abowiem wiele ich przyjdzie pod imieniem mojem / mówiąc : Jám jeſt Chryſtuſ ; y wiele ich zwiodą.
Y uſłyƺyćie wojny y wieśći o wojnách ; pátrzćież / ábyśćie ſobą nie trwożyli ; ábowiem muśi to wƺyſtko być / ále jeƺcże nie tu jeſt koniec.
Abowiem powſtánie náród przećiwko národowi / y króleſtwo przećiwko króleſtwu / y będą głody y mory y trzęśieniá ziemi miejſcámi.
Ale to wƺyſtko jeſt pocżątkiem boleśći.
Tedy wáſ podádzą w udręcżenie / y będą wáſ zábijáć / y będziećie w nienáwiśći u wƺyſtkich národów dlá imieniá mego.
A tedy wiele śię ich zgorƺy / á jedni drugich wydádzą / y jedni drugich nienáwidzieć będą.
Y wiele fáłƺywych proroków powſtánie / y zwiodą wielu.
A iż śię rozmnoży niepráwość / oziębnie miłość wielu.
Ale kto wytrwá áż do końcá / ten zbáwion będzie.
Y będzie kázáná tá Ewángielijá króleſtwá po wƺyſtkim świećie / ná świádectwo wƺyſtkim národom. A tedyć przyjdzie koniec.
Przetoż gdy ujrzyćie obrzydliwość ſpuſtoƺeniá / opowiedziáną przez Dánijelá proroká / ſtojącą ná miejſcu świętym / ( kto cżytá / niecháj uwáżá. )
Tedy ći / co będą w ziemi Judzkiej / niech ućiekáją ná góry ;
A kto ná dáchu / niecháj nie zſtępuje / áby co wziął z domu ſwego ;
A kto ná roli / niech śię názád nie wrácá / áby wziął ƺáty ſwe.
A biádá brzemiennym y pierśiámi kármiącym w one dni!
Przetoż módlćie śię / áby nie było ućiekánie wáƺe w zimie / álbo w ſábát.
Abowiem náoncżáſ będzie wielki ućiſk / jáki nie był od pocżątku świátá áż dotąd / áni potem będzie.
A gdyby nie były ſkrócone one dni / nie byłoby zbáwione żádne ćiáło ; ále dlá wybránych będą ſkrócone one dni.
Tedy jeźliby wám kto rzekł : Oto tu jeſt Chryſtuſ / álbo tám / nie wierzćie.
Abowiem powſtáną fáłƺywi Chryſtuſowie / y fáłƺywi prorocy / y cżynić będą známioná wielkie / y cudá / ták / iżby zwiedli ( by możná ) y wybráne.
Otom wám przepowiedziáł.
Jeźliby wám tedy rzekli : Oto ná puƺcży jeſt / nie wychodźćie ; oto w komorách / nie wierzćie.
Abowiem jáko błyſkáwicá wychodzi od wſchodu ſłońcá y ukázuje śię áż ná záchód / ták będzie y przyjśćie Syná cżłowiecżego.
Bo gdziekolwiek będzie śćierw / tám śię zgromádzą y orły.
A záráz po utrápieniu onych dni ſłońce śię záćmi / á kśiężyc nie dá jáſnośći ſwojej / y gwiázdy będą pádáć z niebá / y mocy niebieſkie poruƺą śię.
Tedyć śię ukáże známię Syná cżłowiecżego ná niebie / á tedy będą nárzekáć wƺyſtkie pokoleniá ziemi / y ujrzą Syná cżłowiecżego / przychodzącego ná obłokách niebieſkich z mocą y z chwáłą wielką ;
Y pośle Anioły ſwoje z trąbą głoſu wielkiego / y zgromádzą wybráne jego od cżterech wiátrów / od krájów niebioſ áż do krájów ich.
A od drzewá figowego náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmłádzá y liśćie wypuƺcżá / poznájećie / iż bliſko jeſt láto.
Tákże y wy / gdy ujrzyćie to wƺyſtko / poznáwájćie / iż bliſko jeſt / á we drzwiách.
Zápráwdę powiádám wám / że nie przeminie ten wiek / áżby śię to wƺyſtko ſtáło.
Niebo y ziemiá przeminą / ále ſłowá moje nie przeminą.
A o onym dniu y godzinie nikt nie wie / áni Aniołowie niebieſcy / tylko ſám Ojćiec mój.
Ale jáko było zá dni Noego / ták będzie y przyjśćie Syná cżłowiecżego.
Abowiem jáko zá dni onych przed potopem jedli / y pili / y ożeniáli śię y zá mąż wydáwáli / áż do onego dniá / którego wƺedł Noe do korábiá /
Y nie ſpoſtrzegli śię / áż przyƺedł potop y zábráł wƺyſtkie : ták będzie y przyjśćie Syná cżłowiecżego.
Tedy będą dwáj ná roli ; jeden będzie wzięty á drugi zoſtáwiony ;
Dwie będą mleć we młynie ; jedná będzie wziętá á drugá zoſtáwioná ;
Cżujćież tedy / poniewáż nie wiećie / której godziny Pán wáƺ przyjdzie.
A to wiedzćie / że / gdyby wiedziáł goſpodárz / o której ſtráży złodziej má przyjść / wżdyby cżuł / y nie dáłby podkopáć domu ſwego.
Przetoż y wy bądźćie gotowi ; bo której śię godziny nie ſpodziejećie / Syn cżłowiecży przyjdzie.
Któryż tedy jeſt ſługá wierny y roztropny / którego poſtánowił pán jego nád cżeládzią ſwoją / áby im dáwáł pokárm ná cżáſ ſłuƺny?
Błogoſłáwiony on ſługá / któregoby / przyƺedłƺy Pán jego / ználázł ták cżyniącego ;
Zápráwdę powiádám wám / że go nád wƺyſtkiemi dobrámi ſwemi poſtánowi.
A jeźliby rzekł on zły ſługá w ſercu ſwojim : Odwłácżá pán mój z przyjśćiem ſwojim ;
Y pocżąłby bić ſpółſługi / á jeść y pić z pijánicámi :
Przyjdzie pán ſługi onego / dniá / którego śię nie ſpodzieje / y godziny / której nie wie ;
Y odłącży go / á cżęść jego położy z obłudnikámi ; tám będzie płácż y zgrzytánie zębów.
