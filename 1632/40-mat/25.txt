Tedy podobne będźie króleſtwo niebieſkie dźieśiąći pánnam : które wźiąwƺy lámpy ſwoje / wyƺły przećiwko oblubieńcowi.
A było z nich pięć mądrych / á pięć głupich.
One głupie wźiąwƺy lámpy ſwoje / nie wźięły oleju z ſobą.
Lecż mądre wźięły oleju w nacżynia ſwoje z lámpámi ſwymi.
A gdy oblubieniec odwłacżał <i>z przyśćiem,</i> zdrzemáły śię wƺyſtkie / y poſnęły.
A o północy ſtał śię krzyk ; Oto oblubieniec idźie ! Wynidźćie przećiwko niemu!
Tedy wſtáły one wƺyſtkie pánny / y ochędożyły lámpy ſwoje.
Ale głupie rzekły do mądrych : Dajćie nam z Oleju wáƺego / boć lámpy náƺe gáſną.
Y odpowiedźiáły one mądre / mówiąc : <i>Nie damy</i> : By ſnadź nam y wam nie doſtáło / idźćie rácżey do przedawájących / á kupćie ſobie.
A gdy odeƺły kupowáć / przyƺedł oblubieniec : á te które były gotowe / weƺły z nim ná weſele : y zámknione ſą drzwi.
Lecż potym przyƺły y one drugie pánny / mówiąc : PAnie / PAnie / otwórz nam.
A on odpowiádájąc / rzekł : Záprawdę powiádam wam / nie znam was.
Cżujćież tedy : bo nie wiećie dniá áni godźiny / którey Syn cżłowiecży przydźie.
Abowiem jáko cżłowiek precż odjeżdżájący / zwołał ſług ſwojich / y oddał im dobrá ſwoje :
Y dał jednemu pięć talentów / á drugiemu dwá / á drugiemu jeden : káżdemu według przemożenia jego : y záraz precż odjáchał.
A poƺedƺy on który wźiął pięć talentów / robił nimi : y zyſkał drugą pięć talentów.
Tákże y on który <i>wźiął</i> dwá / zyſkał y ten drugie dwá.
Ale ten który wźiął jeden / odƺedƺy wykopał <i>dół</i> w źiemi / y ſkrył pieniądze páná ſwego.
A po długim cżáśie / przyƺedł pan onych ſług / y ráchował śię z nimi.
Tedy przyſtąpiwƺy on który był wźiął pięć talentów / przynióſł drugą pięć talentów / mówiąc ; Pánie / oddałeś mi pięć talentów / otom drugą pięć talentów zyſkał nimi.
Y rzekł mu Pan jego ; <i>To</i> dobrze ſługo dobry y wierny : nád máłem byłeś wiernym / nád wielem ćię poſtánowię : wnidź do rádośći páná twego.
A przyſtąpiwƺy y on który był dwá talenty wźiął / rzekł ; Pánie / oddałeś mi dwá talenty / otom drugie dwá talenty zyſkał nimi.
Rzekł mu Pan jego ; <i>To</i> dobrze ſługo dobry y wierny : gdyżeś był wierny nád máłem / nád wielem ćię poſtánowię : Wnidź do rádośći páná twego.
A przyſtąpiwƺy y ten który był wźiął jeden talent / rzekł ; Pánie / wiedźiałem żeś cżłowiek ſrogi / który żnieƺ gdźieś nie rozśiewał / y zbieraƺ gdźieś nie rozſypował.
Bojąc śię tedy / ƺedłem y ſkryłem talent twój w źiemię : oto maƺ co twego jeſt.
A odpowiádájąc pan jego / rzekł mu ; Sługo zły y gnuśny : wiedźiałeś iż żnę gdźiem nie rozśiewał / y zbieram gdźiem nie rozſypował :
Przetożeś miał pieniądze moje dáć tym / co pieniędzmi hándlują / á ja przyƺedƺy wźiąłbym był co jeſt mojego z lichwą.
Przetoż weźmićie od niego ten talent / á dajćie temu który ma dźieśięć talentów :
( Abowiem káżdemu który ma / będźie dano / y obfitowáć będźie : á od tego który nie ma / y to co ma / będźie od niego odjęto. )
A niepożytecżnego ſługę wrzuććie do onych ćiemnośći zewnętrznych. Tám będźie płácż y zgrzytánie zębów.
A gdy przydźie Syn cżłowiecży w chwale ſwojey / y wƺyſcy święći Anjołowie z nim / tedy uśiędźie ná ſtolicy chwały ſwojey.
Y będą zgromádzone przedeń wƺyſtkie narody / y odłącży je jedne od drugich / jáko páſterz odłącża owce od kozłów :
A poſtáwi / owce zájiſte po práwicy ſwojey / á kozły po lewicy.
Tedy rzecże Król tym którzy <i>będą</i> po práwicy jego ; Pódźćie błogoſłáwieni Ojcá mego ; odźiedźicżćie króleſtwo wam zgotowáne od záłożenia świátá.
Abowiem łáknąłem / á dáliśćie mi jeść ; prágnąłem / á dáliśćie mi pić ; byłem gośćiem / á przyjęliśćie mię :
Byłem nágim / á przyodźialiśćie mię ; byłem chorym / á náwiedźiliśćie mię ; byłem w więźieniu / á przychodźiliśćie do mnie.
Tedy mu odpowiedzą ſpráwiedliwi / mówiąc ; PAnie / Kiedyżeſmy ćię widźieli łáknącym / á nákarmiliſmy <i>ćię</i>? álbo prágnącym / á nápojiliſmy <i>ćię</i>?
Y kiedyſmy ćię widźieli gośćiem / á przyjęliſmy <i>ćię</i>? álbo nágim / á przyodźialiſmy ćię?
Albo kiedyſmy ćię widźieli chorym / álbo w więźieniu / á przychodźiliſmy do ćiebie?
A odpowiádájąc Król / rzecże im ; Záprawdę powiádam wam / cokolwiekeśćie ucżynili jednemu z tych bráći mojich namniejƺych / mnieśćie ucżynili.
Potym rzecże y tym którzy <i>będą</i> po lewicy ; Idźćie ode mnie przeklęći w ogień wiecżny / który zgotowány jeſt dyabłu y ánjołom jego.
Abowiem łáknąłem / á nie dáliśćie mi jeść ; prágnąłem / á nie dáliśćie mi pić :
Byłem gośćiem / á nie przyjęliśćie mię ; nágim / á nie przyodźialiśćie mię ; chorym y w więźieniu / á nie náwiedźiliśćie mię.
Tedy mu odpowiedzą y oni / mówiąc : PAnie / Kiedyſmy ćię widźieli łáknącym / álbo prágnącym / álbo gośćiem / álbo nágim / álbo chorym / álbo w więźieniu / á nie ſłużyliſmy tobie?
Tedy im odpowie / mówiąc ; Záprawdę powiádam wam / Cżegośćiekolwiek nie ucżynili jednemu z tych namniejƺych / y mnieśćie nie ucżynili.
Y pójdą ći ná męki wiecżne : ále ſpráwiedliwi do żywotá wiecżnego.
