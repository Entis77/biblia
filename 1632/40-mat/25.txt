Tedy podobne będźie króleſtwo niebieſkie dźieśięćiu pánnom / które wźiąwƺy lámpy ſwoje / wyƺły przećiwko oblubieńcowi.
A było z nich pięć mądrych / á pięć głupich.
One głupie wźiąwƺy lámpy ſwoje / nie wźięły oleju z ſobą.
Lecż mądre wźięły oleju w nácżyniá ſwoje z lámpámi ſwemi.
A gdy oblubieniec odwłácżáł z przyjśćiem / zdrzemáły śię wƺyſtkie y poſnęły.
A o północy ſtáł śię krzyk : Oto oblubieniec idźie ; wynijdźćie przećiwko niemu!
Tedy wſtáły one wƺyſtkie pánny / y ochędożyły lámpy ſwoje.
Ale głupie rzekły do mądrych : Dájćie nám z oleju wáƺego / boć lámpy náƺe gáſną.
Y odpowiedźiáły one mądre / mówiąc : Nie dámy / by ſnáć nám y wám niedoſtáło ; idźćie rácżej do ſprzedáwájących / á kupćie ſobie.
A gdy odeƺły kupowáć / przyƺedł oblubieniec ; á te / które były gotowe / weƺły z nim ná weſele ; y zámknione ſą drzwi.
Lecż potem przyƺły y one drugie pánny / mówiąc : Pánie / Pánie / otwórz nám!
A on odpowiádájąc / rzekł : Zápráwdę powiádám wám / nie znám wáſ.
Cżujćież tedy ; bo nie wiećie dniá áni godźiny / której Syn cżłowiecży przyjdźie.
Abowiem jáko cżłowiek precż odjeżdżájący zwołáł ſług ſwoich y oddáł im dobrá ſwoje ;
Y dáł jednemu pięć tálentów / á drugiemu dwá / á drugiemu jeden / káżdemu według przemożeniá jego / y záráz precż odjecháł.
A poƺedłƺy on / który wźiął pięć tálentów / robił niemi / y zyſkáł drugie pięć tálentów.
Tákże y on / który wźiął dwá / zyſkáł y ten drugie dwá.
Ale ten / który wźiął jeden / odƺedłƺy wykopáł dół w źiemi / y ſkrył pieniądze páná ſwego.
A po długim cżáśie przyƺedł pán onych ſług / y ráchowáł śię z nimi.
Tedy przyſtąpiwƺy on / który był wźiął pięć tálentów / przynióſł drugie pięć tálentów / mówiąc : Pánie! oddáłeś mi pięć tálentów / otom drugie pięć tálentów zyſkáł niemi.
Y rzekł mu pán jego : To dobrze / ſługo dobry y wierny! nád máłem byłeś wiernym / nád wielem ćię poſtánowię ; wnijdź do rádośći páná twego.
A przyſtąpiwƺy y on / który był dwá tálenty wźiął / rzekł : Pánie! oddáłeś mi dwá tálenty / otom drugie dwá tálenty zyſkáł niemi.
Rzekł mu pán jego : To dobrze / ſługo dobry y wierny! gdyżeś był wierny nád máłem / nád wielem ćię poſtánowię ; wnijdź do rádośći páná twego.
A przyſtąpiwƺy y ten / który był wźiął jeden tálent / rzekł : Pánie! wiedźiáłem / że jeſteś cżłowiek ſrogi / który żnieƺ / gdźieś nie rozśiewáł / y zbieráƺ / gdźieś nie rozſypywáł ;
Bojąc śię tedy / ƺedłem y ſkryłem tálent twój w źiemię ; oto máƺ / co twego jeſt.
A odpowiádájąc pán jego / rzekł mu : Sługo zły y gnuśny! wiedźiáłeś / iż żnę / gdźiem nie rozśiewáł / y zbierám / gdźiem nie rozſypywáł ;
Przetożeś miáł pieniądze moje dáć tym / co pieniędzmi hándlują / á já przyƺedłƺy / wźiąłbym był / co jeſt mojego / z lichwą.
Przetoż weźmijćie od niego ten tálent / á dájćie temu / który má dźieśięć tálentów.
( Abowiem káżdemu / który má / będźie dáno / y obfitowáć będźie ; á od tego / który niemá / y to / co má / będźie od niego odjęto. )
A niepożytecżnego ſługę wrzuććie do onych ćiemnośći zewnętrznych / tám będźie płácż y zgrzytánie zębów.
A gdy przyjdźie Syn cżłowiecży w chwále ſwojej / y wƺyſcy święći Aniołowie z nim / tedy uśiądźie ná ſtolicy chwáły ſwojej /
Y będą zgromádzone przedeń wƺyſtkie národy / y odłącży je / jedne od drugich / jáko páſterz odłącżá owce od kozłów.
A poſtáwi owce záiſte po práwicy ſwojej / á kozły po lewicy.
Tedy rzecże król tym / którzy będą po práwicy jego : Pójdźćie / błogoſłáwieni Ojcá mego! odźiedźicżćie króleſtwo wám zgotowáne od záłożeniá świátá.
Abowiem łáknąłem / á dáliśćie mi jeść ; prágnąłem / á dáliśćie mi pić ; byłem gośćiem / á przyjęliśćie mię ;
Byłem nágim / á przyodźiáliśćie mię ; byłem chorym / á náwiedźiliśćie mię ; byłem w więźieniu / á przychodźiliśćie do mnie.
Tedy mu odpowiedzą ſpráwiedliwi / mówiąc : Pánie! kiedyżeśmy ćię widźieli łáknącym / á nákármiliśmy ćię? álbo prágnącym / á nápoiliśmy ćię?
Y kiedyśmy ćię widźieli gośćiem / á przyjęliśmy ćię? álbo nágim / á przyodźiáliśmy ćię?
Albo kiedyśmy ćię widźieli chorym / álbo w więźieniu / á przychodźiliśmy do ćiebie?
A odpowiádájąc król / rzecże im : Zápráwdę powiádám wám / cokolwiekieśćie ucżynili jednemu z tych bráći moich nájmniejƺych / mnieśćie ucżynili.
Potem rzecże y tym / którzy będą po lewicy : Idźćie odemnie / przeklęći! w ogień wiecżny / który zgotowány jeſt dyjábłu y ániołom jego.
Abowiem łáknąłem / á nie dáliśćie mi jeść ; prágnąłem / á nie dáliśćie mi pić ;
Byłem gośćiem / á nie przyjęliśćie mię ; nágim / á nie przyodźiáliśćie mię ; chorym y w więźieniu / á nie náwiedźiliśćie mię.
Tedy mu odpowiedzą y oni / mówiąc : Pánie! kiedyśmy ćię widźieli łáknącym / álbo prágnącym / álbo gośćiem / álbo nágim / álbo chorym / álbo w więźieniu / á nie ſłużyliśmy tobie?
Tedy im odpowie / mówiąc : Zápráwdę powiádám wám / cżegośćiekolwiek nie ucżynili jednemu z tych nájmniejƺych / y mnieśćie nie ucżynili.
Y pójdą ći ná męki wiecżne ; ále ſpráwiedliwi do żywotá wiecżnego.
