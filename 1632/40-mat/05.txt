A JEzuſ widząc lud / wſtąpił ná górę ; á gdy uśiádł / przyſtąpili do niego ucżniowie jego.
A otworzywƺy uſtá ſwe / ucżył je / mówiąc :
Błogoſłáwieni ubodzy w duchu ; ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni / którzy śię ſmęcą ; ábowiem poćieƺeni będą.
Błogoſłáwieni ćiƺy ; ábowiem oni odźiedźicżą źiemię.
Błogoſłáwieni / którzy łákną y prágną ſpráwiedliwośći ; ábowiem oni náſyceni będą.
Błogoſłáwieni miłośierni : ábowiem oni miłośierdźiá doſtąpią.
Błogoſłáwieni cżyſtego ſercá ; ábowiem oni Bogá oglądáją.
Błogoſłáwieni pokój cżyniący ; ábowiem oni ſynámi Bożymi názwáni będą.
Błogoſłáwieni / którzy ćierpią prześládowánie dlá ſpráwiedliwośći ; ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni jeſteśćie / gdy wám złorzecżyć będą / y prześládowáć wáſ / y mówić wƺyſtko złe przećiwko wám / kłámájąc dlá mnie.
Rádujćie śię / y weſelćie śię ; ábowiem zápłátá wáƺá obfitá jeſt w niebieśiech ; ták bowiem prześládowáli proroki / którzy byli przed wámi.
Wy jeſteśćie ſól źiemi ; jeźli tedy ſól zwietrzeje / cżemże ſolić będą? Do nicżego śię już nie zgodźi / tylko áby byłá precż wyrzuconá y od ludźi podeptáná.
Wy jeſteśćie świátłość świátá / nie może śię miáſto ukryć ná górze leżące.
Ani zápáláją świecy / y ſtáwiáją jej pod korzec / ále ná świecżnik / y świeći wƺyſtkim / którzy ſą w domu.
Ták niecháj świeći świátłość wáƺá przed ludźmi / áby ucżynki wáƺe dobre widźieli / á chwálili ojcá wáƺego / który jeſt w niebieśiech.
Nie mniemájćie / ábym przyƺedł rozwiązywáć zákon álbo proroki ; nie przyƺedłem rozwiązywáć / ále wypełnić.
Zápráwdę bowiem powiádám wám : Aż przeminie niebo y źiemiá / jedná jotá álbo kreſká nie przeminie z zákonu / áżby śię wƺyſtko ſtáło.
Ktoby tedy rozwiązáł jedno z tych przykázáń nájmniejƺych / y ucżyłby ták ludźi / nájmniejƺym będźie názwány w króleſtwie niebieſkiem ; á ktobykolwiek cżynił y ucżył / ten będźie wielkim názwány w króleſtwie niebieſkiem.
Abowiem powiádám wám : Jeźli nie będźie obfitƺá ſpráwiedliwość wáƺá / niż náucżonych w Piśmie y Fáryzeuƺów / żádnym ſpoſobem nie wnijdźiećie do króleſtwá niebieſkiego.
Słyƺeliśćie / iż rzecżono ſtárym : Nie będźieƺ zábijáł ; á ktobykolwiek zábił / będźie winien ſądu ;
Ale já wám powiádám : Iż káżdy / kto śię gniewá ná brátá ſwego bez przycżyny / będźie winien ſądu ; á ktokolwiek rzecże brátu ſwemu : Ráchá ; będźie winien rády / á ktokolwiek rzecże : Błáźnie! będźie winien ogniá piekielnego.
A ták jeźlibyś ofiárowáł dár twój ná ołtárzu / á támbyś wſpomniáł / iż brát twój má co przećiwko tobie /
Zoſtáw tám dár twój przed ołtárzem / á odejdź / pierwej śię pojednáj z brátem twoim ; á potem przyƺedłƺy ofiáruj dár twój.
Zgódź śię z przećiwnikiem twoim rychło / pókiś jeſt z nim w drodze / by ćię ſnáć przećiwnik nie podáł ſędźiemu / á ſędźiáby ćię podáł ſłudze / y byłbyś wrzucony do więźieniá.
Zápráwdęć powiádám : Nie wynijdźieƺ ſtámtąd / pókibyś nie oddáł do oſtátniego pieniążká.
Słyƺeliśćie / iż rzecżono ſtárym : Nie będźieƺ cudzołożył ;
Aleć Já wám powiádám : Iż káżdy / który pátrzy ná niewiáſtę / áby jej pożądáł / już z nią cudzołóſtwo popełnił w ſercu ſwojim.
Jeźli ćię tedy oko twoje práwe gorƺy / wyłup je / á zárzuć od śiebie ; ábowiem pożytecżniej jeſt tobie / áby zginął jeden z cżłonków twoich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
A jeźliż ćię práwá ręká twojá gorƺy / odetnij ją / y zárzuć od śiebie ; ábowiem pożytecżniej jeſt tobie / áby zginął jeden z cżłonków twoich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
Záśię rzecżono : Ktobykolwiek opuśćił żonę ſwoję / niech jej dá liſt rozwodny ;
Ale Já wám powiádám : Ktobykolwiek opuśćił żonę ſwoję oprócż przycżyny cudzołóſtwá / przywodźi ją w cudzołóſtwo / á ktoby opuƺcżoną pojął / cudzołoży.
Słyƺeliśćie záśię / iż rzecżono ſtárym : Nie będźieƺ krzywo przyśięgáł / ále oddáƺ Pánu przyśięgi twoje ;
Ale Já wám powiádám / ábyśćie zgołá nie przyśięgáli / áni ná niebo / gdyż jeſt ſtolicą Bożą ;
Ani ná źiemię / gdyż jeſt podnóżkiem nóg jego ; áni ná Jeruzálem / gdyż jeſt miáſto wielkiego królá ;
Ani ná głowę twoję będźieƺ przyśięgáł / gdyż nie możeƺ jednego włoſá biáłym álbo cżárnym ucżynić.
Ale mowá wáƺá niech będźie : Ták / ták ; nie / nie ; á co więcej nádto jeſt / to od złego jeſt.
Słyƺeliśćie / iż rzecżono : Oko zá oko / á ząb zá ząb ;
Ale Já wám powiádám : Żebyśćie śię nie przećiwili złemu / ále ktoby ćię uderzył w práwy policżek twój / nádſtáw mu y drugi ;
Y temu / który śię z tobą chce práwowáć / á ſuknię twoję wźiąć / puść mu y płáƺcż ;
A ktoby ćię przymuƺáł iść milę jednę / idź z nim y dwie ;
Temu / co ćię prośi / dáj / á od tego / co chce u ćiebie pożycżyć / nie odwrácáj śię.
Słyƺeliśćie / iż rzecżono : Będźieƺ miłowáł bliźniego twego / á będźieƺ miáł w nienáwiśći nieprzyjáćielá twego ;
Aleć Já wám powiádám : Miłujćie nieprzyjáćioły wáƺe ; błogoſłáwćie tym / którzy wáſ przeklináją ; dobrze cżyńćie tym / którzy wáſ máją w nienáwiśći / y módlćie śię zá tymi / którzy wám złość wyrządzáją y prześládują wáſ ;
Abyśćie byli ſynámi Ojcá wáƺego / który jeſt w niebieśiech ; bo on to cżyni / że ſłońce jego wſchodźi ná złe y ná dobre / y deƺcż ſpuƺcżá ná ſpráwiedliwe y ná nieſpráwiedliwe /
Abowiem jeźli miłujećie te / którzy wáſ miłują / jákąż zápłátę máćie? ázáż y celnicy tego nie cżynią?
A jeźlibyśćie tylko bráći wáƺych pozdráwiáli / cóż oſobliwego cżynićie? ázáż y celnicy ták nie cżynią?
Bądźćież wy tedy doſkonáłymi / jáko y Ojćiec wáƺ / który jeſt w niebieśiech / doſkonáły jeſt.
