A <i>JEzus</i> widząc lud / wſtąpił ná górę : á gdy uśiadł / przyſtąpili do niego ucżniowie jego.
A otworzywƺy uſtá ſwe / ucżył je / mówiąc :
Błogoſłáwieni ubodzy w duchu : ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni którzy śię <i>ſmęcą</i> : ábowiem poćieƺeni będą.
Błogoſłáwieni ćiƺy : ábowiem oni odźiedźicżą źiemię.
Błogoſłáwieni którzy łákną y prágną ſpráwiedliwośći : ábowiem oni náſyceni będą.
Błogoſłáwieni miłośierni : ábowiem oni miłośierdźia doſtąpią.
Błogoſłáwieni cżyſtego ſercá : ábowiem oni Bogá oglądáją.
Błogoſłáwieni ſpokojni : ábowiem oni ſynámi Bożymi názwáni będą.
Błogoſłáwieni którzy ćierpią prześládowánie dla ſpráwiedliwośći : ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni będźiećie / gdy wam złorzecżyć będą y prześládowáć was / y mówić wƺyſtko złe przećiwko wam kłamájąc / dla mnie.
Rádujćie śię y weſelćie śię : ábowiem zapłátá wáƺá obfita jeſt w niebieśiech. Ták bowiem prześládowáli Proroki / którzy <i>byli</i> przed wámi.
Wy jeſteśćie ſól źiemie. Jeſli tedy ſól zwietrzeje / cżymże oſolona będźie? Do nicżego śię już nie zgodźi / tylko áby byłá precż wyrzucona / y od ludźi podeptána.
Wy jeſteśćie świátłość świátá : Nie może śię miáſto ukryć ná górze leżące.
Ani zápaláją świece / y ſtawiáją jey pod korzec / ále ná świecżnik : y świeći wƺyſtkim którzy <i>ſą</i> w domu.
Ták niechaj świeći świátłość wáƺá przed ludźmi : áby ucżynki wáƺe dobre widźieli / á chwalili Ojcá wáƺego / który jeſt w niebieśiech.
Nie mniemajćie ábym przyƺedł rozwięzowáć Zakon / álbo Proroki : Nie przyƺedłem rozwięzowáć / ále wypełnić.
Záprawdę bowiem powiádam wam : Aż przeminie niebo y źiemiá / jedno jotá álbo kreſká nie przeminie z Zakonu / áżby śię wƺyſtko ſtáło.
Ktoby tedy rozwiązał jedno z tych przykazań namniejƺych / y ucżyłby ták ludźi / namniejƺym będźie názwány w króleſtwie niebieſkim : A ktobykolwiek cżynił y ucżył / ten będźie wielkim názwány w króleſtwie niebieſkim.
Abowiem powiádam wam / jeſli nie będźie obfitƺa ſpráwiedliwość wáƺá niż Náucżonych w piśmie y Fáryzeuƺów / żadnym ſpoſobem nie wnidźiećie do króleſtwá niebieſkiego.
Słyƺeliśćie iż rzecżono ſtárym : Nie będźieƺ zábijał ; á ktobykolwiek zábił / będźie winien ſądu.
Ale Ja wam powiádam / iż káżdy kto śię gniewa ná brátá ſwego bez przycżyny / będźie winien ſądu : á ktokolwiek rzecże brátu ſwemu Ráchá / będźie winien rády : á ktokolwiek rzecże błaźnie / będźie winien ogniá piekielnego.
A ták jeſlibyś ofiárował dar twój ná Ołtarzu / á támbyś wſpomniał / iż brát twój ma co przećiwko tobie :
Zoſtaw tám dar twój przed ołtarzem / á odejdź : pierwey śię pojednaj z brátem twojim / á potym przyƺedƺy ofiáruj dar twój.
Zgódź śię z przećiwnikiem twojim rychło / pókiś jeſt z nim w drodze : by ćię ſnadź przećiwnik nie podał Sędźiemu / á Sędźiaby ćię podał ſłudze / y byłbyś wrzucony do więźienia.
Záprawdęć powiádam : Nie wynidźieƺ z támtąd / pókibyś nie oddał do oſtátniego pieniążká.
Słyƺeliśćie iż rzecżono ſtárym : Nie będźieƺ cudzołożył.
Aleć Ja wam powiádam : Iż káżdy który pátrzy ná niewiáſtę áby jey pożądał / już z nią cudzołóſtwo popełnił w ſercu ſwojim.
Jeſli ćię tedy oko twoje práwe gorƺy / wyłup je á zárzuć od śiebie : ábowiem pożytecżniey jeſt tobie áby zginął jeden z cżłonków twojich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
A jeſliże ćię práwa ręká twojá gorƺy / odetni ją y zárzuć od śiebie : ábowiem pożytecżniej jeſt tobie / áby zginął jeden z cżłonków twojich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
Záśię rzecżono : Ktobykolwiek opuśćił żonę ſwoję / niech jey da liſt rozwodny.
Ale Ja wam powiádam : Ktobykolwiek opuśćił żonę ſwoję oprócż przycżyny cudzołóſtwá / przywodźi ją w cudzołóſtwo : á ktoby opuƺcżoną pojął / cudzołoży.
Słyƺeliśćie záśię iż rzecżono ſtárym : Nie będźieƺ krzywo przyśięgał / ále oddaƺ PAnu przyśięgi twoje.
Ale Ja wam powiádam / ábyśćie zgołá nie przyśięgáli / áni ná niebo ; gdyż jeſt ſtolicą Bożą :
Ani ná źiemię ; gdyż jeſt podnóżkiem nóg jego : áni ná Jeruzalem ; gdyż jeſt miáſto wielkiego Królá :
Ani ná głowę twoję będźieƺ przyśięgał ; gdyż nie możeƺ jednego włoſá białym álbo cżarnym ucżynić.
Ale mowá wáƺá niech będźie : Ták / ták : Nie / nie : á co więcey nád to jeſt / <i>to</i> od złego jeſt.
Słyƺeliśćie iż rzecżono : Oko zá oko / á ząb zá ząb.
Ale Ja wam powiádam : Żebyśćie śię nie przećiwili złemu : ále ktoby ćię uderzył w práwy policżek twój / nádſtaw mu y drugiego.
Y temu który śię z tobą chce práwowáć / á ſuknią twoję wźiąć / puść mu y płaƺcż.
A ktoby ćię przymuƺał <i>iść</i> milę jednę / idź z nim <i>y</i> dwie.
Temu co ćię prośi / daj ; á od tego co chce u ćiebie pożycżyć / nie odwracaj śię.
Słyƺeliśćie iż rzecżono : Będźieƺ miłował bliźniego twego / á będźieƺ miał w nienawiśći nieprzyjaćielá twego.
Aleć Ja wam powiádam : Miłujćie nieprzyjaćioły wáƺe : błogoſławćie tym / którzy was przeklináją ; dobrze cżyńćie tym / którzy was máją w nienawiśći ; y módlćie śię zá tymi / którzy wam złość wyrządzáją / y prześládują was.
Abyśćie byli ſynámi Ojcá wáƺego który <i>jeſt</i> w niebieśiech : bo on to cżyni / że ſłońce jego wſchodźi ná złe y ná dobre ; y deƺcż zpuƺcża ná ſpráwiedliwe y ná nieſpráwiedliwe.
