A JEzuſ widząc lud / wſtąpił ná górę : á gdy uśiádł / przyſtąpili do niego ucżniowie jego.
A otworzywƺy uſtá ſwe / ucżył je / mówiąc :
Błogoſłáwieni ubodzy w duchu : ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni którzy śię ſmęcą : ábowiem poćieƺeni będą.
Błogoſłáwieni ćiƺy : ábowiem oni odźiedźicżą źiemię.
Błogoſłáwieni którzy łákną y prágną ſpráwiedliwośći : ábowiem oni náſyceni będą.
Błogoſłáwieni miłośierni : ábowiem oni miłośierdźiá doſtąpią.
Błogoſłáwieni cżyſtego ſercá : ábowiem oni Bogá oglądáją.
Błogoſłáwieni spokojni : ábowiem oni ſynámi Bożymi názwáni będą.
Błogoſłáwieni którzy ćierpią prześládowánie dla ſpráwiedliwośći : ábowiem ich jeſt króleſtwo niebieſkie.
Błogoſłáwieni będźiecie / gdy wám złorzecżyć będą y prześládowáć was / y mówić wƺyſtko złe przećiwko wám kłámájąc / dla mnie.
Rádujćie śię y weſelćie śię : ábowiem zápłátá wáƺá obfitá jeſt w niebieśiech. Ták bowiem prześládowáli Proroki / którzy <i>byli</i> przed wámi.
Wy jeſteśćie ſól źiemi. Jeſli tedy ſól zwietrzeje / cżemże oſolona będźie? Do nicżego śię już nie zgodźi / tylko áby byłá precż wyrzuconá / y od ludźi podeptáná.
Wy jeſteśćie świátłość świátá : nie może śię miáſto ukryć ná górze leżące.
Ani zápáláją świece / y ſtáwiáją jey pod korzec / ále ná świecżnik : y świeći wƺyſtkim którzy <i>ſą</i> w domu.
Ták niecháj świeći świátłość wáƺá przed ludźmi : áby ucżynki wáƺe dobre widźieli / á chwálili Ojcá wáƺego / który jeſt w niebieśiech.
Nie mniemájćie ábym przyƺedł rozwiązywáć zakon / álbo Proroki : nie przyƺedłem rozwiązywáć / ále wypełnić.
Zápráwdę bowiem powiádám wám : Aż przeminie niebo y źiemiá / jedná jotá álbo kreſká nie przeminie z zakonu / áżby śię wƺyſtko ſtáło.
Ktoby tedy rozwiązáł jedno z tych przykázáń nájmniejƺych / y ucżyłby ták ludźi / nájmniejƺym będźie názwány w króleſtwie niebieſkiem : á ktobykolwiek cżynił y ucżył / ten będźie wielkim názwány w króleſtwie niebieſkiem.
Abowiem powiádám wám / jeſli nie będźie obfitƺá ſpráwiedliwość wáƺá niż Náucżonych w piśmie y Fáryzeuƺów / żádnym ſpoſobem nie wnijdźiećie do króleſtwá niebieſkiego.
Słyƺeliśćie iż rzecżono ſtárym : Nie będźieƺ zábijáł ; á ktobykolwiek zábił / będźie winien ſądu.
Ale Ja wám powiádám / Iż káżdy kto śię gniewá ná brátá ſwego bez przycżyny / będźie winien ſądu : á ktokolwiek rzecże brátu ſwemu Ráchá / będźie winien rády : á ktokolwiek rzecże / błáźnie / będźie winien ogniá piekielnego.
A ták jeſlibyś ofiárowáł dár twój ná ołtárzu / á támbyś wſpomniáł / iż brát twój má co przećiwko tobie :
Zoſtáw tám dár twój przed ołtárzem / á odejdź : pierwej śię pojednáj z brátem twoim / á potem przyƺedłƺy ofiáruj dár twój.
Zgódź śię z przećiwnikiem twoim rychło / pókiś jeſt z nim w drodze : by ćię ſnáć przećiwnik nie podáł Sędźiemu / á Sędźiáby ćię podáł ſłudze / y byłbyś wrzucony do więźieniá.
Zápráwdęć powiádám : Nie wynijdźieƺ ztámtąd / pókibyś nie oddáł do oſtátniego pieniążká.
Słyƺeliśćie iż rzecżono ſtárym : Nie będźieƺ cudzołożył.
Aleć Ja wám powiádám : Iż káżdy który pátrzy ná niewiáſtę áby jey pożądáł / już z nią cudzołóſtwo popełnił w ſercu ſwojim.
Jeſli ćię tedy oko twoje práwe gorƺy / wyłup je á zárzuć od śiebie : ábowiem pożytecżniej jeſt tobie áby zginął jeden z cżłonków twoich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
A jeſliż ćię práwá ręká twojá gorƺy / odetnij ją y zárzuć od śiebie : ábowiem pożytecżniej jeſt tobie / áby zginął jeden z cżłonków twoich / á wƺyſtko ćiáło twoje nie było wrzucone do ogniá piekielnego.
Záśię rzecżono : Ktobykolwiek opuśćił żonę ſwoję / niech jey dá liſt rozwodny.
Ale Ja wám powiádám : Ktobykolwiek opuśćił żonę ſwoję oprócż przycżyny cudzołóſtwá / przywodźi ją w cudzołóſtwo : á ktoby opuƺcżoną pojął / cudzołoży.
Słyƺeliśćie záśię iż rzecżono ſtárym : Nie będźieƺ krzywo przyśięgáł / ále oddáƺ PAnu przyśięgi twoje.
Ale Ja wám powiádám / ábyśćie zgołá nie przyśięgáli / áni ná niebo ; gdyż jeſt ſtolicą Bożą :
Ani ná źiemię ; gdyż jeſt podnóżkiem nóg jego : áni ná Jeruzálem ; gdyż jeſt miáſto wielkiego Królá :
Ani ná głowę twoję będźieƺ przyśięgáł ; gdyż nie możeƺ jednego włoſá biáłym álbo cżárnym ucżynić.
Ale mowá wáƺá niech będźie : Ták / ták : nie / nie : á co więcey nád to jeſt / <i>to</i> od złego jeſt.
Słyƺeliśćie iż rzecżono : Oko zá oko / á ząb zá ząb.
Ale Ja wám powiádám : Żebyśćie śię nie przećiwili złemu : ále ktoby ćię uderzył w práwy policżek twój / nádſtáw mu y drugiego.
Y temu który śię z tobą chce práwowáć / á ſuknię twoję wźiąć / puść mu y płáƺcż.
A ktoby ćię przymuƺáł <i>iść</i> milę jednę / idź z nim y dwie.
Temu co ćię prośi / dáj ; á od tego co chce u ćiebie pożycżyć / nie odwrácáj śię.
Słyƺeliśćie iż rzecżono : Będźieƺ miłowáł bliźniego twego / á będźieƺ miał w nienáwiśći nieprzyjáćielá twego.
Aleć Ja wám powiádám : Miłujćie nieprzyjáćioły wáƺe : błogoſłáwćie tym / którzy was przeklináją ; dobrze cżyńćie tym / którzy was máją w nienáwiśći ; y módlćie śię zá tymi / którzy wám złość wyrządzáją / y prześládują was.
Abyśćie byli ſynámi Ojcá wáƺego który <i>jeſt</i> w niebieśiech : bo on to cżyni / że ſłońce jego wſchodźi ná złe y ná dobre ; y deƺcż ſpuƺcżá ná ſpráwiedliwe y ná nieſpráwiedliwe.
Abowiem jeſli miłujećie te którzy was miłują / jákąż zápłátę máćie ázáż y Celnicy tego nie cżynią?
A jeſlibyśćie tylko bráćią wáƺe pozdráwiáli / cóż oſobliwego cżynićie? ázáż y Celnicy ták nie cżynią?
Bądźćież wy tedy doſkonáłymi : jáko y Ojćiec wáƺ który jeſt w niebieśiech / doſkonáły jeſt.
