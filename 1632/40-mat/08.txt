A gdy zſtępowáł z góry / ƺedł zá nim lud wielki ;
A oto trędowáty przyƺedłƺy / pokłonił mu śię / mówiąc : Pánie! jeſli chceƺ / możeƺ mię ocżyśćić.
Y wyćiągnąwƺy JEzuſ rękę / dotknął śię go / mówiąc : Chcę / bądź ocżyƺcżony ; y záráz ocżyƺcżony jeſt trąd jego.
Tedy mu rzekł JEzuſ : Pátrz / ábyś nikomu nie powiádáł / ále idź / ukáż śię kápłánowi / y ofiáruj dár on / który przykázáł Mojżeƺ ná świádectwo przećiwko nim.
A gdy JEzuſ wƺedł do Kápernáum / przyƺedł do niego ſetnik / proƺąc go /
Y mówiąc : Pánie! ſługá mój leży w domu powietrzem ruƺony / y ćiężko śię trápi.
Y rzekł mu JEzuſ : Ja przyjdę y uzdrowię go.
A odpowiádájąc ſetnik rzekł : Pánie! nie jeſtem godźien / ábyś wƺedł pod dách mój ; ále tylko rzecż ſłowo / á będźie uzdrowiony ſługá mój.
Bomći y ja cżłowiek pod mocą innego / májący pod ſobą żołnierze ; y mówię temu : Idź / á idźie ; á drugiemu : Przyjdź / á przychodźi ; á ſłudze memu : Cżyń to / á cżyni.
A gdy to uſłyƺáł JEzuſ / zádźiwił śię / y rzekł tym / którzy ƺli zá nim : Zápráwdę powiádám wám : Anim w Izráelu ták wielkiej wiáry nie ználázł.
A powiádám wám : Iż wiele ich od wſchodu y od záchodu ſłońcá przyjdźie / á uśiądą zá ſtołem z Abráhámem y z Izáákiem y z Jákóbem w króleſtwie niebieſkiem.
Ale ſynowie króleſtwá będą wyrzuceni w ćiemnośći zewnętrzne / tám będźie płácż y zgrzytánie zębów.
Y rzekł JEzuſ ſetnikowi : Idź / á jákoś uwierzył / niech ći śię ſtánie ; y uzdrowiony jeſt ſługá jego onejże godźiny.
A gdy JEzuſ przyƺedł do domu Piotrowego / ujrzáł świekrę jego / leżącą ná łożu y májącą gorącżkę.
Y dotknął śię ręki jey / y opuśćiłá ją gorącżká ; y wſtáłá á poſługowáłá im.
A gdy był wiecżór / przywiedli do niego wiele opętánych : y wygániáł duchy ſłowem ; y wƺyſtkie / którzy śię źle mieli / uzdráwiáł ;
Aby śię wypełniło / co powiedźiáno przez Izájáƺá proroká / mówiącego : On niemocy náƺe ná śię wźiął / á choroby náƺe nośił.
A widząc JEzuſ wielki lud około śiebie / kázáł śię przepráwić ná drugą ſtronę morzá.
Tedy przyſtąpiwƺy niektóry z náucżonych w Piśmie / rzekł mu : Miſtrzu! pójdę zá tobą / gdźiekolwiek pójdźieƺ.
Y rzekł mu JEzuſ : Liƺki máją jámy / á ptáki niebieſkie gniázdá ; ále Syn cżłowiecży niemá / gdźieby głowę ſkłonił.
A drugi z ucżniów jego rzekł mu : Pánie! dopuść mi pierwej odejść y pogrześć ojcá mego ;
Ale mu JEzuſ rzekł : Pójdź zá mną / á niecháj umárli grzebią umárłe ſwoje.
A gdy on wſtąpił w łódź / wſtąpili zá nim y ucżniowie jego.
A oto śię wzruƺenie wielkie ſtáło ná morzu / ták iż śię łódź wáłámi okrywáłá ; á on ſpáł.
A przyſtąpiwƺy ucżniowie jego / obudźili go / mówiąc : Pánie! rátuj nas / giniemy.
Y rzekł do nich : Przecżżeśćie bojáźliwi? o máłowierni! Tedy wſtáwƺy / zgromił wiátry y morze / y ſtáło śię ućiƺenie wielkie.
A ludźie śię dźiwowáli / mówiąc : Jákiż to jeſt ten / że mu y wiátry y morze poſłuƺne ſą?
A gdy śię on przewiózł ná drugą ſtronę do kráiny Giergiezeńcżyków / zábieżeli mu dwáj opętáni z grobów wychodzący / bárdzo okrutni / ták iż nie mógł nikt przechodźić oną drogą.
A oto zákrzyknęli / mówiąc : Cóż my z tobą mámy / JEzuśie / Synu Boży? Przyƺedłeś tu przed cżáſem / dręcżyć nas?
Y byłá dáleko od nich trzodá wielká świni páſących śię.
Tedy go dyjábli prośili / mówiąc : Jeſli nas wygániáƺ / dopuść nám wnijść w trzodę tych świni.
Y rzekł im : Idźćie. A oni wyƺedłƺy / weƺli w onę trzodę świni / á oto porwáwƺy śię oná wƺyſtká trzodá świni / z przykrá wpádłá w morze / y pozdycháły w wodách.
Lecż páſterze ućiekli / á poƺedłƺy do miáſtá / opowiedźieli wƺyſtko / y to / co śię z onymi opętánymi ſtáło.
A oto wƺyſtko miáſto wyƺło przećiwko JEzuſowi / á ujrzáwƺy go prośili / áby z ich gránic odƺedł.
