A gdy zſtępował z góry / ƺedł zá nim lud wielki.
A oto trędowáty przyƺedƺy pokłonił mu śię / mówiąc ; PAnie / jeſli chceƺ / możeƺ mię ocżyśćić.
Y wyćiągnąwƺy JEzus rękę / dotknął śię go / mówiąc : Chcę / bądź ocżyśćiony : Y záraz ocżyśćiony jeſt trąd jego.
Tedy mu rzekł JEzus ; Pátrz ábyś nikomu nie powiádał : ále idź / ukaż śię Kápłanowi / y ofiáruj dar on który przykazał Mojzeƺ / ná świádectwo <i>przećiwko</i> nim.
A gdy JEzus wƺedł do Kápernáum / przyƺedł do niego Setnik proƺąc go /
Y mówiąc : PAnie / Sługá mój leży domá powietrzem ruƺony / y ćiężko śię trapi.
Y rzekł mu JEzus ; Ja przydę / y uzdrowię go.
A odpowiádájąc Setnik / rzekł ; PAnie / Nie jeſtem godźien ábyś wƺedł pod dách mój : ále tylko rzecż ſłowo / á będźie uzdrowiony ſługá mój.
Bomći y ja cżłowiek pod mocą <i>innego,</i> májący pod ſobą żołnierze : y mówię temu : Idź / á idźie ; á drugiemu : Przydź / á przychodźi ; á ſłudze memu : Cżyń to / á cżyni.
A gdy to uſłyƺał JEzus / zádźiwił śię / y rzekł tym którzy ƺli zá nim : Záprawdę powiádam wam ; Anim w Izráelu ták wielkiey wiáry nie ználazł.
A powiádam wam : Iż wiele ich od Wſchodu y od Zachodu <i>ſłońcá</i> przydźie / á uśiędą zá ſtołem z Abráhámem y z Izáákiem y z Jákóbem w króleſtwie niebieſkim.
Ale ſynowie króleſtwá będą wyrzuceni w ćiemnośći zewnętrzne ; tám będźie płácż y zgrzytánie zębów.
Y rzekł JEzus Setnikowi ; Idź / á jákoś uwierzył / niechći śię ſtánie. Y uzdrowiony jeſt ſługá jego onejże godźiny.
A gdy JEzus przyƺedł do domu Piotrowego / ujrzał Świekrę jego leżącą ná łożu / y májącą gorącżkę.
Y dotknął śię ręki jey / y opuśćiłá ją gorącżká : y wſtáłá / á poſługowáłá im.
A gdy był wiecżór / przywiedli do niego wiele opętánych : y wyganiał duchy ſłowem ; y wƺyſtkie / którzy śię źle mieli / uzdrawiał :
Aby śię wypełniło co powiedźiano przez Izájaƺá Proroká mówiącego ; On niemocy náƺe <i>ná śię</i> wźiął / á choroby <i>náƺe</i> nośił.
A widząc JEzus wielki lud około śiebie / kazał śię przepráwić ná drugą ſtronę <i>morzá</i>.
Tedy przyſtąpiwƺy niektóry z Náucżonych w piśmie / rzekł mu ; Miſtrzu / pójdę zá tobą gdźiekolwiek pójdźieƺ.
Y rzekł mu JEzus : Liƺki máją jámy / á ptacy niebieſcy gniazdá ; ále Syn cżłowiecży nie ma gdźieby głowę ſkłonił.
A drugi z ucżniów jego rzekł mu ; PAnie / dopuść mi pierwey odejść / y pogrześć ojcá mego.
Ale mu JEzus rzekł : Pódź zá mną ; á niechaj umárli grzebią umárłe ſwoje.
A gdy on wſtąpił w łódź / wſtąpili zá nim y ucżniowie jego.
A oto śię wzruƺenie wielkie ſtáło ná morzu / ták iż śię łódź wáłámi okrywáłá : á on ſpał.
A przyſtąpiwƺy ucżniowie jego / obudźili go / mówiąc ; PAnie / rátuj nas / giniemy.
Y rzekł do nich ; Przecżżeśćie bojáźliwi / o máłowierni? Tedy wſtawƺy / zgromił wiátry y morze : y ſtáło śię ućiƺenie wielkie.
A ludźie śię dźiwowáli / mówiąc ; Jákiż to jeſt ten / że mu y wiátry y morze poſłuƺne ſą?
A gdy śię on przewiózł ná drugą ſtronę do krájiny Gergezeńcżyków / zábieżeli mu dwá opętáni z grobów wychodzący / bárzo okrutni / ták iż nie mógł nikt przechodźić oną drogą.
A oto zákrzyknęli / mówiąc : Cóż my z tobą mamy JEzuśie Synu Boży? Przyƺedłeś tu przed cżáſem dręcżyć nas?
Y byłá dáleko od nich trzodá wielka świni / páſących śię.
Tedy go dyabli prośili / mówiąc ; Jeſli nas wyganiaƺ / dopuść nam wniść w trzodę tych świni.
Y rzekł im ; Idźćie. A oni wyƺedƺy / weƺli w onę trzodę świni : á oto porwawƺy śię oná wƺyſtká trzodá świni / zprzykrá wpádłá w morze / y pozdycháły w wodách.
Lecż páſterze ućiekli : á poƺedƺy do miáſtá / opowiedźieli wƺyſtko ; y to <i>co śię</i> z onymi opętanymi <i>ſtáło</i>.
A oto wƺyſtko miáſto wyƺło przećiwko JEzuſowi : á ujrzawƺy go / prośili áby z ich gránic odƺedł.
