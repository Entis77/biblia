Tedy przyſtąpili do JEzuſá z Jeruzálemu náucżeni w Piśmie y Fáryzeuƺowie / mówiąc :
Cżemu ucżniowie twoi przeſtępują uſtáwę ſtárƺych? ábowiem nie umywáją rąk ſwych / gdy máją jeść chleb.
A on odpowiádájąc / rzekł im : Cżemuż y wy przeſtępujećie przykazánie Boże dla uſtáwy wáƺej?
Abowiem Bóg przykázał / mówiąc : Cżćij ojcá twego y mátkę ; y ktoby złorzecżył ojcu álbo mátce / śmierćią niecháj umrze.
Ale wy powiádáćie : Ktoby rzekł ojcu álbo mátce : Dár / którykolwiek jeſt ode mnie / tobie pożytecżny będźie ; á nie ucżćiłby ojcá ſwego álbo mátki ſwojey / bez winy będźie.
Y wzruƺyliśćie przykázánia Boże dla uſtáwy wáƺej.
Obłudnicy! dobrze o was prorokował Izájáƺ / mówiąc :
Lud ten przybliżá śię do mnie uſty ſwemi / y wárgámi cżći mię ; ále ſerce ich dáleko jeſt odemnie.
Lecż prózno mię cżcżą / náucżájąc náuk / które ſą przykázánia ludzkie.
A záwołáwƺy do śiebie ludu / rzekł im : Słuchájćie á rozumiejćie.
Nie to / co wchodźi w uſtá / pokálá cżłowieká ; ále co wychodźi z uſt / to pokálá cżłowieká.
Tedy przyſtąpiwƺy ucżniowie jego / rzekli mu : Wieƺ / iż Fáryzeuƺowie / uſłyƺáwƺy tę mowę / zgorƺyli śię?
A on odpowiádájąc rzekł : Wƺelki ƺcżep / którego nie ƺcżepił Oćiec mój niebieſki / wykorzeniony będźie.
Zániechájćie ich ; ślepi ſą wodzowie ślepych / á ślepy jeſliby ślepego prowádźił / obádwá w dół wpádną.
A odpowiádájąc Piotr / rzekł mu : Wyłóż nam to podobieńſtwo.
Y rzekł JEzus : Jeƺcżeż y wy bezrozumni jeſteśćie?
Jeƺcżeż nie rozumiećie / iż wƺyſtko / co wchodźi w uſtá / w brzuch idźie / y do wychodu bywá wyrzucono?
Ale co z uſt pochodźi / z ſercá wychodźi / á toć pokálá cżłowieká.
Abowiem z ſercá wychodzą złe myśli / mężobójſtwá / cudzołóſtwá / wƺetecżeńſtwá / złodźiejſtwá / fáłƺywe świádectwá / bluźnierſtwá.
Toć jeſt / co pokálá cżłowieká : ále jeść nieumytemi rękomá / toć nie pokálá cżłowieká.
A wyƺedƺy JEzus ſtámtąd / uſtąpił w ſtrony Tyru y Sydonu.
A oto niewiáſtá Chánánejſká z onych gránic wyƺedƺy / wołáłá / mówiąc do niego : Zmiłuj śię nádemną Pánie / ſynu Dawidowy! córká mojá ćiężko bywá od dyjábłá dręcżoná.
A on jey nie odpowiedźiał y ſłowá. Tedy przyſtąpiwƺy ucżniowie jego / prośili go / mówiąc : Odpráw ją / boć wołá zá námi.
A on odpowiádájąc rzekł : Nie jeſtem poſłány tylko do owiec / które zginęły z domu Izráelſkiego.
Lecż oná przyſtąpiwƺy / pokłoniłá mu śię / mówiąc : Pánie / rátuj mię!
A on odpowiádájąc rzekł : Niedobrá jeſt / bráć chleb dźiećinny / á miotáć ƺcżeniętom.
A oná rzekłá : Ták jeſt / Pánie! á wƺákże y ƺcżeniętá jedzą odrobiny / które pádáją z ſtołu pánów ich.
Tedy odpowiádájąc JEzus rzekł jey : O niewiáſto! wielká jeſt wiárá twojá ; niecháj ći śię ſtánie / jáko chceƺ. Y uzdrowioná jeſt córká jey od onejże godźiny.
A JEzus poƺedƺy ſtámtąd / przyƺedł nád morze Gálilejſkie / á wſtąpiwƺy ná górę / śiedźiał tám.
Y przyƺedł do niego wielki lud / májąc z ſobą chrome / ślepe / nieme / ułomne y inƺych wiele / y kłádli je u nóg JEzuſowych / y uzdráwiał je /
Ták iż śię on lud dźiwował / widząc / że niemi mówią / ułomni uzdrowieni ſą / chromi chodzą / á ślepi widzą ; y wielbili Bogá Izráelſkiego.
Lecż JEzus zwołáwƺy ucżniów ſwojich / rzekł : Żál mi tego ludu ; ábowiem już trzy dni przy mnie trwáją / á nie máją / coby jedli / á nie chcę ich rozpuśćić głodnych / by ſnadź nie pomdleli ná drodze.
Tedy mu rzekli ucżniowie jego : Skądżebyſmy wźięli ták wiele chlebá ná tey puƺcży / ábyſmy ták wielki lud náſyćili?
Y rzekł im JEzus : Wieleż máćie chlebów? A oni rzekli : Siedm / y trochę rybek.
Tedy rozkázał ludowi / áby śiedli ná źiemi.
A wźiąwƺy one śiedm chlebów y one ryby / ucżyniwƺy dźięki / łámał y dał ucżniom ſwoim / á ucżniowie ludowi.
Y jedli wƺyſcy y náſyceni ſą / y zebráli / co zbyło ułomków / śiedm koƺów pełnych.
A było tych / którzy jedli / cżtery tyśiące mężów / oprócż niewiáſt y dźiátek.
Tedy rozpuśćiwƺy lud / wſtąpił w łódź / y przyƺedł ná gránice Mágdáláńſkie.
