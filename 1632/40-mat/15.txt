Tedy przyſtąpili do JEzuſá z Jeruzálemu náucżeni w Piśmie y Fáryzeuƺowie / mówiąc :
Cżemu ucżniowie twoi przeſtępują uſtáwę ſtárƺych? ábowiem nie umywáją rąk ſwych / gdy máją jeść chleb.
A on odpowiádájąc / rzekł im : Cżemuż y wy przeſtępujećie przykázánie Boże dla uſtáwy wáƺej?
Abowiem Bóg przykázáł / mówiąc : Cżćij ojcá twego y mátkę ; y ktoby złorzecżył ojcu álbo mátce / śmierćią niecháj umrze.
Ale wy powiádáćie : Ktoby rzekł ojcu álbo mátce : Dár / którykolwiek jeſt ode mnie / tobie pożytecżny będźie ; á nie ucżćiłby ojcá ſwego álbo mátki ſwojey / bez winy będźie.
Y wzruƺyliśćie przykázániá Boże dla uſtáwy wáƺej.
Obłudnicy! dobrze o was prorokowáł Izájáƺ / mówiąc :
Lud ten przybliżá śię do mnie uſty ſwemi / y wárgámi cżći mię ; ále ſerce ich dáleko jeſt odemnie.
Lecż próżno mię cżcżą / náucżájąc náuk / które ſą przykázániá ludzkie.
A záwołáwƺy do śiebie ludu / rzekł im : Słuchájćie á rozumiejćie.
Nie to / co wchodźi w uſtá / pokálá cżłowieká ; ále co wychodźi z uſt / to pokálá cżłowieká.
Tedy przyſtąpiwƺy ucżniowie jego / rzekli mu : Wieƺ / iż Fáryzeuƺowie / uſłyƺáwƺy tę mowę / zgorƺyli śię?
A on odpowiádájąc rzekł : Wƺelki ƺcżep / którego nie ƺcżepił Ojćiec mój niebieſki / wykorzeniony będźie.
Zániechájćie ich ; ślepi ſą wodzowie ślepych / á ślepy jeſliby ślepego prowádźił / obádwá w dół wpádną.
A odpowiádájąc Piotr / rzekł mu : Wyłóż nám to podobieńſtwo.
Y rzekł JEzuſ : Jeƺcżeż y wy bezrozumni jeſteśćie?
Jeƺcżeż nie rozumiećie / iż wƺyſtko / co wchodźi w uſtá / w brzuch idźie / y do wychodu bywá wyrzucono?
Ale co z uſt pochodźi / z ſercá wychodźi / á toć pokálá cżłowieká.
Abowiem z ſercá wychodzą złe myśli / mężobójſtwá / cudzołóſtwá / wƺetecżeńſtwá / złodźiejſtwá / fáłƺywe świádectwá / bluźnierſtwá.
Toć jeſt / co pokálá cżłowieká : ále jeść nieumytemi rękomá / toć nie pokálá cżłowieká.
A wyƺedłƺy JEzuſ ſtámtąd / uſtąpił w ſtrony Tyru y Sydonu.
A oto niewiáſtá Chánánejſká z onych gránic wyƺedłƺy / wołáłá / mówiąc do niego : Zmiłuj śię nádemną Pánie / ſynu Dawidowy! córká mojá ćiężko bywá od dyjábłá dręcżoná.
A on jey nie odpowiedźiáł y ſłowá. Tedy przyſtąpiwƺy ucżniowie jego / prośili go / mówiąc : Odpráw ją / boć wołá zá námi.
A on odpowiádájąc rzekł : Nie jeſtem poſłány tylko do owiec / które zginęły z domu Izráelſkiego.
Lecż oná przyſtąpiwƺy / pokłoniłá mu śię / mówiąc : Pánie / rátuj mię!
A on odpowiádájąc rzekł : Niedobrá jeſt / bráć chleb dźiećinny / á miotáć ƺcżeniętom.
A oná rzekłá : Ták jeſt / Pánie! á wƺákże y ƺcżeniętá jedzą odrobiny / które pádáją z ſtołu pánów ich.
Tedy odpowiádájąc JEzuſ rzekł jey : O niewiáſto! wielká jeſt wiárá twojá ; niecháj ći śię ſtánie / jáko chceƺ. Y uzdrowioná jeſt córká jey od onejże godźiny.
A JEzuſ poƺedłƺy ſtámtąd / przyƺedł nád morze Gálilejſkie / á wſtąpiwƺy ná górę / śiedźiáł tám.
Y przyƺedł do niego wielki lud / májąc z ſobą chrome / ślepe / nieme / ułomne y inƺych wiele / y kłádli je u nóg JEzuſowych / y uzdráwiáł je /
Ták iż śię on lud dźiwowáł / widząc / że niemi mówią / ułomni uzdrowieni ſą / chromi chodzą / á ślepi widzą ; y wielbili Bogá Izráelſkiego.
Lecż JEzuſ zwołáwƺy ucżniów ſwoich / rzekł : Żál mi tego ludu ; ábowiem już trzy dni przy mnie trwáją / á nie máją / coby jedli / á nie chcę ich rozpuśćić głodnych / by ſnáć nie pomdleli ná drodze.
Tedy mu rzekli ucżniowie jego : Skądżebyśmy wźięli ták wiele chlebá ná tey puƺcży / ábyśmy ták wielki lud náſyćili?
Y rzekł im JEzuſ : Wieleż máćie chlebów? A oni rzekli : Siedm / y trochę rybek.
Tedy rozkázáł ludowi / áby śiedli ná źiemi.
A wźiąwƺy one śiedm chlebów y one ryby / ucżyniwƺy dźięki / łámáł y dáł ucżniom ſwoim / á ucżniowie ludowi.
Y jedli wƺyſcy y náſyceni ſą / y zebráli / co zbyło ułomków / śiedm koƺów pełnych.
A było tych / którzy jedli / cżtery tyśiące mężów / oprócż niewiáſt y dźiátek.
Tedy rozpuśćiwƺy lud / wſtąpił w łódź / y przyƺedł ná gránice Mágdáláńſkie.
