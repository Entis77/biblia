Tedy przyſtąpili do JEzuſá z Jeruzalemu Náucżeni w piśmie / y Fáryzeuƺowie / mówiąc ;
Cżemu ucżniowie twoji przeſtępują uſtáwę ſtárƺych? Abowiem nie umywáją ręku ſwych / gdy máją jeść chleb.
A on odpowiádájąc / rzekł im ; Cżemuż y wy przeſtępujećie przykazánie Boże dla uſtáwy wáƺey?
Abowiem Bóg przykazał / mówiąc ; Cżći Ojcá twego y mátkę : y ktoby złorzecżył Ojcu álbo mátce / śmierćią niechaj umrze.
Ale wy powiádaćie ; Ktoby rzekł Ojcu álbo mátce ; Dar którykolwiek jeſt odemnie / tobie pożytecżny będźie : á nie ucżćiłby ojcá ſwego álbo mátki ſwojey / <i>bez winy będźie</i>.
Y wzruƺyliśćie przykazánia Boże dla uſtáwy wáƺey.
Obłudnicy / dobrze o was prorokował Izájaƺ / mówiąc ;
Lud ten przybliża śię do mnie uſty ſwymi / y wárgámi cżći mię : ále ſerce ich dáleko jeſt odemnie.
Lecż prózno mię cżcżą náucżájąc náuk <i>które ſą</i> przykazánia ludzkie.
A záwoławƺy do śiebie ludu / rzekł im ; Słuchajćie / á rozumiejćie.
Nie to co wchodźi w uſtá pokala cżłowieká / ále co wychodźi z uſt / to pokala cżłowieká.
Tedy przyſtąpiwƺy ucżniowie jego / rzekli mu ; Wieƺ iż Fáryzeuƺowie uſłyƺawƺy tę mowę / zgorƺyli śię?
A on odpowiádájąc / rzekł ; Wƺelki ƺcżep którego nie ƺcżepił Oćiec mój niebieſki / wykorzeniony będźie.
Zániechajćie ich : ślepi ſą wodzowie ślepych : á ślepy jeſliby ślepego prowádźił / obádwá w dół wpádną.
A odpowiádájąc Piotr / rzekł mu ; Wyłóż nam to podobieńſtwo.
Y rzekł JEzus ; Jeƺcżeż y wy bezrozumni jeſteśćie?
Jeƺcżeż nie rozumiećie / iż wƺyſtko co wchodźi w uſtá / w brzuch idźie / y do wychodu bywa wyrzucono?
Ale co z uſt pochodźi / z ſercá wychodźi / á toć pokala cżłowieká.
Abowiem z ſercá wychodzą złe myśli / mężobójſtwá / cudzołóſtwá / wƺetecżeńſtwá / złodźiejſtwá / fałƺywe świádectwá / bluźnierſtwá.
Toć jeſt / co pokala cżłowieká : ále jeść nieumytymi rękomá / toć nie pokala cżłowieká.
A wyƺedƺy JEzus z támtąd / uſtąpił w ſtrony Tyru y Sydonu.
A oto niewiáſtá Chánánejſka z onych gránic wyƺedƺy / wołáłá / mówiąc do niego ; Zmiłuj śię nádemną PAnie / Synu Dawidów! Córká mojá ćiężko bywa od dyabłá dręcżona.
A on jey nie odpowiedźiał <i>y</i> ſłowá : Tedy przyſtąpiwƺy ucżniowie jego / prośili go / mówiąc ; Odpraw ją / boć woła zá námi.
A on odpowiádájąc / rzekł ; Nie jeſtem poſłány tylko do owiec które zginęły z domu Izráelſkiego.
Lecż oná przyſtąpiwƺy / pokłoniłá mu śię mówiąc ; PAnie / rátuj mię!
A on odpowiádájąc / rzekł ; Nie dobra jeſt bráć chleb dźiećinny / á miotáć ƺcżeniętom.
A oná rzekłá ; Ták <i>jeſt</i> PAnie ; A wƺákże y ƺcżeniętá jedzą odrobiny / które pádáją z ſtołu Pánów ich.
Tedy odpowiádájąc JEzus / rzekł jey ; O niewiáſto! wielka jeſt wiárá twojá ; niechajći śię ſtánie jáko chceƺ. Y uzdrowiona jeſt córká jey od onejże godźiny.
A JEZUS poƺedƺy z támtąd / przyƺedł nád morze Gálilejſkie : á wſtąpiwƺy ná górę / śiedźiał tám.
Y przyƺedł do niego wielki lud / májąc z ſobą chrome / ślepe / nieme / ułomne / y inƺych wiele : y kłádli je u nóg JEzuſowych / y uzdrawiał je :
Ták iż śię on lud dźiwował / widząc że niemi mówią / ułomni uzdrowieni ſą / chromi chodzą / á ślepi widzą : y wielbili Bogá Izráelſkiego.
Lecż JEzus zwoławƺy ucżniów ſwojich / rzekł : Żal mi tego ludu : Abowiem już trzy dni przy mnie trwáją / á nie máją coby jedli : á nie chcę ich rozpuśćić głodnych / by ſnadź nie pomdleli ná drodze.
Tedy mu rzekli ucżniowie jego : Z kądżebyſmy wźięli ták wiele chlebá ná tey puƺcży / ábyſmy ták wielki lud náſyćili?
Y rzekł im JEzus : Wieleż maćie chlebów? A oni rzekli : Siedm / y trochę rybek.
Tedy rozkazał ludowi / áby śiedli ná źiemi.
A wźiąwƺy onę śiedm chlebów y one ryby / ucżyniwƺy dźięki / łamał / y dał ucżniom ſwojim : á ucżniowie ludowi.
Y jedli wƺyſcy / y náſyceni ſą : y zebráli co zbyło ułomków / śiedm koƺów pełnych.
A było tych którzy jedli / cżtery tyśiące mężów / oprócż niewiaſt y dźiatek.
Tedy rozpuśćiwƺy lud / wſtąpił w łódź : y przyƺedł ná gránice Mágdáláńſkie.
