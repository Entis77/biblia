A gdy śię JEzuſ národźił w Betlehemie Judskim zá dniá Herodá Królá / oto Mędrcy ze wſchodu ſłońcá przybyli do Jeruzálemu
Mówiąc ; Gdźież jeſt ten który śię národźił Król Żydowſki? Bośmy widźieli gwiázdę jego ná wſchód ſłońcá: y przyjecháliśmy / ábyśmy mu śię pokłonili.
Co gdy Król Herod uſłyƺáł / zátrwożył śię y wƺyſtko Jeruzálem z nim.
Przetoż zebráwƺy wƺyſtkie przedniejƺe Kápłány y Náucżyćiele ludu / dowiádywáł śię od nich / gdźieby śię miał CHryſtuſ národźić?
A oni mu rzekli : W Betlehemie Judskim : Bo ták nápiſáno przez Proroká :
Y ty Betlehemie źiemio Judská / żádną miárą nie jeſteś nájmniejƺe między Kśiążęty Judskimi : ábowiem z ćiebie wynijdźie Wódz / który rządźić będźie lud mój Izráelſki.
Tedy Herod wezwáwƺy potájemnie onych Mędrców / pilnie śię wywiádywáł od nich o cżáśie / którego śię gwiázdá ukázáłá.
A poſłáwƺy je do Betlehem / rzekł : Jecháwƺy / pilnie śię wywiádujćie o tem dźiećiątku ; á gdy znájdźiećie / oznájmijćie mi ábym y ja przyjecháwƺy / pokłonił mu śię.
Oni tedy wyſłucháwƺy Królá / poƺli : A oto oná gwiázdá / którą widźieli ná Wſchód <i>ſłońcá</i> / prowádźiłá je / áż przyƺedłƺy ſtánęłá nád <i>miejſcem</i> / gdźie było dźiećiątko.
A gdy ujrzeli onę gwiázdę / urádowáli śię rádośćią bárdzo wielką.
Y wƺedłƺy w dom / ználeſli dźiećiątko z Máryją mátką jego ; á upádłƺy / pokłonili mu śię : y otworzywƺy ſkárby ſwoje / ofiárowáli mu dáry / złoto y kádźidło y myrrę.
Lecż będąc upomnieni od Bogá we śnie / áby śię nie wrácáli do Herodá / inƺą drogą wróćili śię do kráiny ſwojey.
A gdy oni odeƺli / oto Anioł Páńſki ukázáł śię we śnie Józefowi / mówiąc : Wſtáwƺy / weźmi dźiećiątko y mátkę jego / á ućiecż do Egiptu / á bądź tám áż ći powiem. Abowiem Herod będźie ƺukáł dźiećiątká / áby je zátráćił.
Który wſtáwƺy w nocy / wźiął dźiećiątko y mátkę jego ; y uƺedł do Egiptu.
Y był tám áż do śmierći Herodowej : áby śię wypełniło / co powiedźiáno od Páná przez Proroká mówiącego : Z Egiptum wezwáł ſyná mego.
Tedy Herod ujrzáwƺy że był oƺukány od Mędrców / rozgniewáł śię bárdzo / á poſłáwƺy / pobił wƺyſtkie dźiátki / które były w Betlehemie y po wƺyſtkich gránicách jego / od dwu lát y niżej / według cżáſu / o którym śię był pilnie wywiedźiáł od Mędrców.
Tedy śię wypełniło co powiedźiáno przez Jeremijáƺá Proroká / mówiącego :
Głoſ w Rámie ſłyƺány jeſt / láment y płácż / y nárzekánie wielkie : Ráchel płácżącá ſynów ſwoich / y nie dáłá śię poćieƺyć ; przeto / że ich niemáƺ.
A gdy umárł Herod / oto Anioł Páńſki ukázáł śię we śnie Józefowi w Egipćie /
Mówiąc : Wſtáwƺy / weźmij dźiećiątko y mátkę jego / á idź do źiemi Izráelſkiej ; ábowiem pomárli ći / którzy ƺukáli duƺy dźiećięcey.
A on wſtáwƺy wźiął do śiebie dźiećiątko y mátkę jego / y przyƺedł do źiemi Izráelſkiej.
Lecż gdy uſłyƺáł iż Archeláuſ królowáł w Judzkiej źiemi / ná miejſcu Herodá ojcá ſwego / báł śię tám iść : ále nápomniány będąc od Bogá we śnie / uſtąpił w ſtrony Gálilejſkie.
A przyƺedłƺy mieƺkáł w mieśćie / które zowią Názáret : áby śię wypełniło co powiedźiáno przez Proroki : Iż Názárejcżykiem názwány będźie.
