A gdy śię JEzuſ národźił w Betlehemie Judzkiem zá dniá Herodá królá / oto mędrcy ze wſchodu ſłońcá przybyli do Jeruzálemu / mówiąc :
Gdźież jeſt ten / który śię národźił / król Żydowſki? Bośmy widźieli gwiázdę jego ná wſchód ſłońcá / y przyjecháliśmy / ábyśmy mu śię pokłonili.
Co gdy król Herod uſłyƺáł / zátrwożył śię / y wƺyſtko Jeruzálem z nim ;
Przetoż zebráwƺy wƺyſtkie przedniejƺe kápłány y náucżyćiele ludu / dowiádywáł śię od nich / gdźieby śię miáł CHryſtuſ národźić.
A oni mu rzekli : W Betlehemie Judzkiem : bo ták nápiſáno przez proroká :
Y ty Betlehemie / źiemio Judzká! żádną miárą nie jeſteś nájmniejƺe między kśiążęty Judzkimi ; ábowiem z ćiebie wynijdźie wódz / który rządźić będźie lud mój Izráelſki.
Tedy Herod wezwáwƺy potájemnie onych mędrców / pilnie śię wywiádywáł od nich o cżáśie / którego śię gwiázdá ukázáłá.
A poſłáwƺy je do Betlehemu / rzekł : Jecháwƺy / pilnie śię wywiádujćie o tem dźiećiątku ; á gdy znájdźiećie / oznájmijćie mi / ábym y já przyjecháwƺy / pokłonił mu śię.
Oni tedy / wyſłucháwƺy królá / poƺli ; á oto oná gwiázdá / którą widźieli ná wſchód ſłońcá / prowádźiłá je / áż przyƺedłƺy / ſtánęłá nád miejſcem / gdźie było dźiećiątko ;
A gdy ujrzeli onę gwiázdę / urádowáli śię rádośćią bárdzo wielką ;
Y wƺedłƺy w dom / ználeźli dźiećiątko z Máryją / mátką jego / á upádłƺy / pokłonili mu śię / y otworzywƺy ſkárby ſwoje / ofiárowáli mu dáry : złoto y kádźidło y myrrę.
Lecż będąc upomnieni od Bogá we śnie / áby śię nie wrácáli do Herodá / inƺą drogą wróćili śię do kráiny ſwojej.
A gdy oni odeƺli / oto Anioł Páńſki ukázáł śię we śnie Józefowi mówiąc : Wſtáwƺy / weźmij dźiećiątko y mátkę jego / á ućiecż do Egiptu / á bądź tám / áż ći powiem ; ábowiem Herod będźie ƺukáł dźiećiątká / áby je zátráćił.
Który wſtáwƺy / wźiął dźiećiątko y mátkę jego w nocy / y uƺedł do Egiptu ;
Y był tám áż do śmierći Herodowej / áby śię wypełniło / co powiedźiáno od Páná przez proroká / mówiącego : Z Egiptum wezwáł ſyná mego.
Tedy Herod ujrzáwƺy / że był oƺukány od mędrców / rozgniewáł śię bárdzo / á poſłáwƺy pobił wƺyſtkie dźiátki / które były w Betlehemie y po wƺyſtkich gránicách jego / od dwóch lát y niżej / według cżáſu / o którym śię był pilnie wywiedźiáł od mędrców.
Tedy śię wypełniło / co powiedźiáno przez Jeremijáƺá proroká / mówiącego :
Głoſ w Rámie ſłyƺány jeſt / láment / y płácż / y nárzekánie wielkie : Ráchel płácżącá ſynów ſwoich nie dáłá śię poćieƺyć / przeto / że ich niemáƺ.
A gdy umárł Herod / oto Anioł Páńſki ukázáł śię we śnie Józefowi w Egipćie /
Mówiąc : Wſtáwƺy / weźmij dźiećiątko y mátkę jego / á idź do źiemi Izráelſkiej ; ábowiem pomárli ći / którzy ƺukáli duƺy dźiećięcej.
A on wſtáwƺy / wźiął do śiebie dźiećiątko y mátkę jego / y przyƺedł do źiemi Izráelſkiej.
Lecż gdy uſłyƺáł / iż Archeláuſ królowáł w Judzkiej źiemi ná miejſcu Herodá / ojcá ſwego / báł śię tám iść ; ále nápomniány będąc od Bogá we śnie / uſtąpił w ſtrony Gálilejſkie ;
A przyƺedłƺy mieƺkáł w mieśćie / które zowią Názáret / áby śię wypełniło / co powiedźiáno przez proroki : Iż Názárejcżykiem názwány będźie.
