A gdy śię Jezuſ národził w Betlehemie Judzkiem zá dniá Herodá królá / oto mędrcy ze wſchodu ſłońcá przybyli do Jeruzálemu / mówiąc :
Gdzież jeſt ten / który śię národził / król Żydowſki? Bośmy widzieli gwiázdę jego ná wſchód ſłońcá / y przyjecháliśmy / ábyśmy mu śię pokłonili.
Co gdy król Herod uſłyƺáł / zátrwożył śię / y wƺyſtko Jeruzálem z nim ;
Przetoż zebráwƺy wƺyſtkie przedniejƺe kápłány y náucżyćiele ludu / dowiádywáł śię od nich / gdzieby śię miáł Chryſtuſ národzić.
A oni mu rzekli : W Betlehemie Judzkiem : bo ták nápiſáno przez proroká :
Y ty Betlehemie / ziemio Judzká! żádną miárą nie jeſteś nájmniejƺe między kśiążęty Judzkimi ; ábowiem z ćiebie wynijdzie wódz / który rządzić będzie lud mój Izráelſki.
Tedy Herod wezwáwƺy potájemnie onych mędrców / pilnie śię wywiádywáł od nich o cżáśie / którego śię gwiázdá ukázáłá.
A poſłáwƺy je do Betlehemu / rzekł : Jecháwƺy / pilnie śię wywiádujćie o tem dziećiątku ; á gdy znájdziećie / oznájmijćie mi / ábym y já przyjecháwƺy / pokłonił mu śię.
Oni tedy / wyſłucháwƺy królá / poƺli ; á oto oná gwiázdá / którą widzieli ná wſchód ſłońcá / prowádziłá je / áż przyƺedłƺy / ſtánęłá nád miejſcem / gdzie było dziećiątko ;
A gdy ujrzeli onę gwiázdę / urádowáli śię rádośćią bárdzo wielką ;
Y wƺedłƺy w dom / ználeźli dziećiątko z Máryją / mátką jego / á upádłƺy / pokłonili mu śię / y otworzywƺy ſkárby ſwoje / ofiárowáli mu dáry : złoto y kádzidło y myrrę.
Lecż będąc upomnieni od Bogá we śnie / áby śię nie wrácáli do Herodá / inƺą drogą wróćili śię do kráiny ſwojej.
A gdy oni odeƺli / oto Anioł Páńſki ukázáł śię we śnie Józefowi mówiąc : Wſtáwƺy / weźmij dziećiątko y mátkę jego / á ućiecż do Egiptu / á bądź tám / áż ći powiem ; ábowiem Herod będzie ƺukáł dziećiątká / áby je zátráćił.
Który wſtáwƺy / wziął dziećiątko y mátkę jego w nocy / y uƺedł do Egiptu ;
Y był tám áż do śmierći Herodowej / áby śię wypełniło / co powiedziáno od Páná przez proroká / mówiącego : Z Egiptum wezwáł ſyná mego.
Tedy Herod ujrzáwƺy / że był oƺukány od mędrców / rozgniewáł śię bárdzo / á poſłáwƺy pobił wƺyſtkie dziátki / które były w Betlehemie y po wƺyſtkich gránicách jego / od dwóch lát y niżej / według cżáſu / o którym śię był pilnie wywiedziáł od mędrców.
Tedy śię wypełniło / co powiedziáno przez Jeremijáƺá proroká / mówiącego :
Głoſ w Rámie ſłyƺány jeſt / láment / y płácż / y nárzekánie wielkie : Ráchel płácżącá ſynów ſwoich nie dáłá śię poćieƺyć / przeto / że ich niemáƺ.
A gdy umárł Herod / oto Anioł Páńſki ukázáł śię we śnie Józefowi w Egipćie /
Mówiąc : Wſtáwƺy / weźmij dziećiątko y mátkę jego / á idź do ziemi Izráelſkiej ; ábowiem pomárli ći / którzy ƺukáli duƺy dziećięcej.
A on wſtáwƺy / wziął do śiebie dziećiątko y mátkę jego / y przyƺedł do ziemi Izráelſkiej.
Lecż gdy uſłyƺáł / iż Archeláuſ królowáł w Judzkiej ziemi ná miejſcu Herodá / ojcá ſwego / báł śię tám iść ; ále nápomniány będąc od Bogá we śnie / uſtąpił w ſtrony Gálilejſkie ;
A przyƺedłƺy mieƺkáł w mieśćie / które zowią Názáret / áby śię wypełniło / co powiedziáno przez proroki : Iż Názárejcżykiem názwány będzie.
