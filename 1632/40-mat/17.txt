A po ƺeśći dniách / wźiął JEzus Piotrá / y Jákubá / y Janá brátá jego ; y wprowádźił je ná górę wyſoką oſobno.
Y przemieniony jeſt przed nimi : á rozjáśniło śię oblicże jego / jáko ſłońce ; á ƺáty jego ſtáły śię białe / jáko świátłość.
A oto ukazáli śię im Mojzeƺ y Eliaƺ / z nim rozmawiájący.
Y odpowiádájąc Piotr / rzekł do JEzuſá ; PAnie / dobrze nam tu bydź : Jeſli chceƺ / ucżynimy tu trzy namioty ; tobie jeden / y Mojżeƺowi jeden / y Eliaƺowi jeden.
<i>A</i> gdy on jeƺcże mówił / oto obłok jáſny záćienił je : A oto głos z obłoku mówiący ; Ten jeſt Syn mój miły / w którym mi śię upodobáło / tego ſłuchajćie.
To uſłyƺawƺy ucżniowie / upádli ná twarz ſwoję / y bali śię bárzo.
Tedy przyſtąpiwƺy JEzus / dotknął śię ich / y rzekł : Wſtańćie / á nie bójćie śię.
A oni podniózƺy ocży ſwoje / nikogo nie widźieli / tylko JEzuſá ſámego.
A gdy zſtępowáli z góry / przykazał im JEzus / mówiąc : Nikomu nie powiádajćie tego widzenia / áż Syn cżłowiecży zmartwychwſtánie.
Y pytáli go ucżniowie jego / mówiąc : Cóż tedy Náucżeni w piśmie powiádáją / że ma Eliaƺ pierwey przyść?
A JEzus odpowiádájąc / rzekł im : Eliaƺći pierwey przydźie / y nápráwi wƺyſtko :
Ale wam powiádam : Iż Eliaƺ już przyƺedł / wƺákże nie poználi go / ále ucżynili mu cokolwiek chćieli : tákći y Syn cżłowiecży ma ućierpieć od nich.
Tedy zrozumieli ucżniowie / że o Janie Chrzćićielu mówił do nich.
A gdy przyƺli do ludu / przyſtąpił do niego cżłowiek / y upadł przed nim ná koláná /
Y rzekł : PAnie / Zmiłuj śię nád ſynem mojim ; ábowiem lunátykiem jeſt / y ćiężko śię trapi : cżęſtokroć bowiem wpada w ogień / y cżęſtokroć w wodę.
Y przywiodłem go do ucżniów twojich / ále go nie mogli uzdrowić.
A odpowiádájąc JEzus / rzekł ; O rodzáju niewierny y przewrotny! Dokądże będę z wámi? Dokądże was będę ćierpiał? Przywiedźćie mi go ſám.
Y zgromił onego dyabłá JEzus : y wyƺedł od niego : y uzdrowiony jeſt on młodźieniec od onejże godźiny.
Tedy przyſtąpiwƺy ucżniowie do JEzuſá oſobno / rzekli mu : Cżemużeſmy go my wygnáć nie mogli?
Lecż JEzus rzekł do nich ; Dla niedowiárſtwá wáƺego. Záprawdę bowiem powiádam wam : Jeſli byśćie / májąc wiárę jáko źiárno gorcżycżne / rzekli tey górze : Przenieś śię z tąd ná ono miejſce ; tedy śię przenieśie : á nic niemożnego wam nie będźie.
Aleć ten rodzaj nie wychodźi tylko przez modlitwę / y <i>przez</i> poſt.
A gdy przebywáli w Gálilejey / rzekł do nich JEzus : Syn cżłowiecży będźie wydány w ręce ludzkie.
Y zábiją go : ále trzećiego dniá zmartwychwſtánie : y záſmućili śię bárzo.
A gdy przyƺli do Kápernáum / przyſtąpili do Piotrá ći którzy podátek wybieráli / y rzekli : Izali Náucżyćiel wáƺ nie dawa podátku?
Y rzekł : Dawa. A gdy wchodźił w dom / uprzedźił go JEzus / mówiąc : Cóżći śię zda Symonie? Królowie źiemſcy od kogoż biorą cło álbo cżynƺ? od ſynów ſwojich / cżyli od obcych?
Rzekł mu Piotr : Od obcych. Y rzekł mu JEzus : Toć tedy ſynowie ſą wolni.
Wƺákże ábyſmy ich nie zgorƺyli / ƺedƺy do morzá / zárzuć wędę / á tę rybę która napierwey uwięźnie / weźmi : á otworzywƺy gębę jey znajdźieƺ Státer : który wźiąwƺy / daj im zá mię y zá śię.
