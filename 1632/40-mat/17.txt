A po ƺeśćiu dniách wziął Jezuſ Piotrá y Jákóbá y Jáná / brátá jego / y wprowádził je ná górę wyſoką oſobno.
Y przemieniony jeſt przed nimi / á rozjáśniło śię oblicże jego / jáko ſłońce / á ƺáty jego ſtáły śię biáłe / jáko świátłość.
A oto ukázáli śię im Mojżeƺ y Elijáƺ / z nim rozmáwiájący.
Y odpowiádájąc Piotr / rzekł do Jezuſá : Pánie! dobrze nám tu być ; jeźli chceƺ / ucżynimy tu trzy námioty / tobie jeden / y Mojżeƺowi jeden / y Elijáƺowi jeden.
A gdy on jeƺcże mówił / oto obłok jáſny záćienił je ; á oto głoſ z obłoku mówiący : Ten jeſt Syn mój miły / w którym mi śię upodobáło / tego ſłuchájćie.
To uſłyƺáwƺy ucżniowie / upádli ná twárz ſwoję y báli śię bárdzo.
Tedy przyſtąpiwƺy Jezuſ dotknął śię ich y rzekł : Wſtáńćie / á nie bójćie śię.
A oni podnióſłƺy ocży ſwoje / nikogo nie widzieli / tylko Jezuſá ſámego.
A gdy zſtępowáli z góry / przykázáł im Jezuſ / mówiąc : Nikomu nie powiádájćie tego widzeniá / áż Syn cżłowiecży zmártwychwſtánie.
Y pytáli go ucżniowie jego / mówiąc : Cóż tedy náucżeni w Piśmie powiádáją / że má Elijáƺ pierwej przyjść?
A Jezuſ odpowiádájąc / rzekł im : Elijáƺći pierwej przyjdzie y nápráwi wƺyſtko ;
Ale wám powiádám : Iż Elijáƺ już przyƺedł / wƺákże nie poználi go / ále ucżynili mu / cokolwiek chćieli ; tákći y Syn cżłowiecży má ućierpieć od nich.
Tedy zrozumieli ucżniowie / że o Jánie Chrzćićielu mówił do nich.
A gdy przyƺli do ludu / przyſtąpił do niego cżłowiek / y upádł przed nim ná koláná /
Y rzekł : Pánie! zmiłuj śię nád ſynem moim : ábowiem lunátykiem jeſt / y ćiężko śię trápi ; cżęſtokroć bowiem wpádá w ogień / y cżęſtokroć w wodę.
Y przywiodłem go do ucżniów twoich / ále go nie mogli uzdrowić.
A odpowiádájąc Jezuſ / rzekł : O rodzáju niewierny y przewrotny! Dokądże będę z wámi? Dokądże wáſ będę ćierpiáł? przywiedźćie mi go ſám.
Y zgromił onego dyjábłá Jezuſ ; y wyƺedł od niego / y uzdrowiony jeſt on młodzieniec od onejże godziny.
Tedy przyſtąpiwƺy ucżniowie do Jezuſá oſobno / rzekli mu : Cżemużeśmy go my wygnáć nie mogli?
Lecż Jezuſ rzekł do nich : Dlá niedowiárſtwá wáƺego ; zápráwdę bowiem powiádám wám : Jeźlibyśćie / májąc wiárę jáko ziárno gorcżycżne / rzekli tej górze : Przenieś śię ſtąd ná ono miejſce / tedy śię przenieśie / á nic niemożebnego wám nie będzie.
Ale ten rodzáj nie wychodzi tylko przez modlitwę y przez poſt.
A gdy przebywáli w Gálilei / rzekł do nich Jezuſ : Syn cżłowiecży będzie wydány w ręce ludzkie ;
Y zábiją go / ále trzećiego dniá zmártwychwſtánie. Y záſmućili śię bárdzo.
A gdy przyƺli do Kápernáum / przyſtąpili do Piotrá ći / którzy podátek wybieráli / y rzekli : Izáli náucżyćiel wáƺ nie dáje podátku?
Y rzekł : Dáje. A gdy wchodził w dom / uprzedził go Jezuſ / mówiąc : Cóż ći śię zdá / Szymonie? Królowie ziemſcy od kogoż biorą cło álbo cżynƺ? od ſynów ſwoich / cżyli od obcych?
Rzekł mu Piotr : Od obcych. Y rzekł mu Jezuſ : Toć tedy ſynowie ſą wolni.
Wƺákże ábyśmy ich nie zgorƺyli / ƺedłƺy do morzá / zárzuć wędkę / á tę rybę / którá nájpierwej uwięźnie / weźmij / á otworzywƺy gębę jej / znájdzieƺ ſtáter / który wziąwƺy / dáj im zá mię y zá śię.
