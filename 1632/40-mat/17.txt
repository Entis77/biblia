A po ƺeśćiu dniách wźiął JEzus Piotrá y Jákóbá y Jáná / brátá jego / y wprowádźił je ná górę wyſoką oſobno.
Y przemieniony jeſt przed nimi / á rozjáśniło śię oblicże jego / jáko ſłońce / á ƺáty jego ſtáły śię biáłe / jáko świátłość.
A oto ukázáli śię im Mojzeƺ y Elijáƺ / z nim rozmáwiájący.
Y odpowiádájąc Piotr / rzekł do JEzuſá : Pánie! dobrze nam tu być ; jeſli chceƺ / ucżynimy tu trzy námioty / tobie jeden / y Mojżeƺowi jeden / y Elijáƺowi jeden.
A gdy on jeƺcże mówił / oto obłok jáſny záćienił je ; á oto głos z obłoku mówiący : Ten jeſt Syn mój miły / w którym mi śię upodobáło / tego ſłuchájćie.
To uſłyƺáwƺy ucżniowie / upádli ná twárz ſwoję y báli śię bárzo.
Tedy przyſtąpiwƺy JEzus dotknął śię ich y rzekł : Wſtáńćie / á nie bójćie śię.
A oni podnióſƺy ocży ſwoje / nikogo nie widźieli / tylko JEzuſá ſámego.
A gdy zſtępowáli z góry / przykázał im JEzus / mówiąc : Nikomu nie powiádájćie tego widzeniá / áż Syn cżłowiecży zmartwychwſtánie.
Y pytáli go ucżniowie jego / mówiąc : Cóż tedy náucżeni w Piśmie powiádáją / że ma Elijáƺ pierwej przyść?
A JEzus odpowiádájąc / rzekł im : Elijáƺći pierwej przydźie y nápráwi wƺyſtko ;
Ale wam powiádám : Iż Elijáƺ już przyƺedł / wƺákże nie poználi go / ále ucżynili mu / cokolwiek chćieli ; tákći y Syn cżłowiecży ma ućierpieć od nich.
Tedy zrozumieli ucżniowie / że o Jánie Chrzćićielu mówił do nich.
A gdy przyƺli do ludu / przyſtąpił do niego cżłowiek / y upádł przed nim ná koláná /
Y rzekł : Pánie! zmiłuj śię nád ſynem moim : ábowiem lunátykiem jeſt / y ćiężko śię trápi ; cżęſtokroć bowiem wpádá w ogień / y cżęſtokroć w wodę.
Y przywiodłem go do ucżniów twojich / ále go nie mogli uzdrowić.
A odpowiádájąc JEzus / rzekł : O rodzáju niewierny y przewrotny! Dokądże będę z wámi? Dokądże was będę ćierpiał? przywiedźćie mi go ſám.
Y zgromił onego dyjábłá JEzus ; y wyƺedł od niego / y uzdrowiony jeſt on młodźieniec od onejże godźiny.
Tedy przyſtąpiwƺy ucżniowie do JEzuſá oſobno / rzekli mu : Cżemużeśmy go my wygnáć nie mogli?
Lecż JEzus rzekł do nich : Dla niedowiárſtwá wáƺego ; zápráwdę bowiem powiádám wam : Jeſlibyśćie / májąc wiárę jáko źiárno gorcżycżne / rzekli tey górze : Przenieś śię ſtąd ná ono miejſce / tedy śię przenieśie / á nic niemożebnego wam nie będźie.
Ale ten rodzáj nie wychodźi tylko przez modlitwę y przez poſt.
A gdy przebywáli w Gálilei / rzekł do nich JEzus : Syn cżłowiecży będźie wydány w ręce ludzkie ;
Y zábiją go / ále trzećiego dniá zmartwychwſtánie. Y záſmućili śię bárzo.
A gdy przyƺli do Kápernáum / przyſtąpili do Piotrá ći / którzy podátek wybieráli / y rzekli : Izali náucżyćiel wáƺ nie dáje podátku?
Y rzekł : Dáje. A gdy wchodźił w dom / uprzedźił go JEzus / mówiąc : Cóż ći śię zda / Szymonie? Królowie źiemſcy od kogoż biorą cło álbo cżynƺ? od ſynów ſwojich / cżyli od obcych?
Rzekł mu Piotr : Od obcych. Y rzekł mu JEzus : Toć tedy ſynowie ſą wolni.
Wƺákże ábyſmy ich nie zgorƺyli / ƺedƺy do morzá / zárzuć wędkę / á tę rybę / która nájpierwej uwięźnie / weźmij / á otworzywƺy gębę jey / znájdźieƺ ſtáter / który wźiąwƺy / dáj im zá mię y zá śię.
