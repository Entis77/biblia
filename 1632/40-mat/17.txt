A po ƺeśćiu dniách wźiął JEzuſ Piotrá y Jákóbá y Jáná / brátá jego / y wprowádźił je ná górę wyſoką oſobno.
Y przemieniony jeſt przed nimi / á rozjáśniło śię oblicże jego / jáko ſłońce / á ƺáty jego ſtáły śię biáłe / jáko świátłość.
A oto ukázáli śię im Mojżeƺ y Elijáƺ / z nim rozmáwiájący.
Y odpowiádájąc Piotr / rzekł do JEzuſá : Pánie! dobrze nám tu być ; jeſli chceƺ / ucżynimy tu trzy námioty / tobie jeden / y Mojżeƺowi jeden / y Elijáƺowi jeden.
A gdy on jeƺcże mówił / oto obłok jáſny záćienił je ; á oto głoſ z obłoku mówiący : Ten jeſt Syn mój miły / w którym mi śię upodobáło / tego ſłuchájćie.
To uſłyƺáwƺy ucżniowie / upádli ná twárz ſwoję y báli śię bárdzo.
Tedy przyſtąpiwƺy JEzuſ dotknął śię ich y rzekł : Wſtáńćie / á nie bójćie śię.
A oni podnióſłƺy ocży ſwoje / nikogo nie widźieli / tylko JEzuſá ſámego.
A gdy zſtępowáli z góry / przykázáł im JEzuſ / mówiąc : Nikomu nie powiádájćie tego widzeniá / áż Syn cżłowiecży zmartwychwſtánie.
Y pytáli go ucżniowie jego / mówiąc : Cóż tedy náucżeni w Piśmie powiádáją / że má Elijáƺ pierwej przyjść?
A JEzuſ odpowiádájąc / rzekł im : Elijáƺći pierwej przyjdźie y nápráwi wƺyſtko ;
Ale wám powiádám : Iż Elijáƺ już przyƺedł / wƺákże nie poználi go / ále ucżynili mu / cokolwiek chćieli ; tákći y Syn cżłowiecży má ućierpieć od nich.
Tedy zrozumieli ucżniowie / że o Jánie Chrzćićielu mówił do nich.
A gdy przyƺli do ludu / przyſtąpił do niego cżłowiek / y upádł przed nim ná koláná /
Y rzekł : Pánie! zmiłuj śię nád ſynem moim : ábowiem lunátykiem jeſt / y ćiężko śię trápi ; cżęſtokroć bowiem wpádá w ogień / y cżęſtokroć w wodę.
Y przywiodłem go do ucżniów twoich / ále go nie mogli uzdrowić.
A odpowiádájąc JEzuſ / rzekł : O rodzáju niewierny y przewrotny! Dokądże będę z wámi? Dokądże was będę ćierpiáł? przywiedźćie mi go ſám.
Y zgromił onego dyjábłá JEzuſ ; y wyƺedł od niego / y uzdrowiony jeſt on młodźieniec od onejże godźiny.
Tedy przyſtąpiwƺy ucżniowie do JEzuſá oſobno / rzekli mu : Cżemużeśmy go my wygnáć nie mogli?
Lecż JEzuſ rzekł do nich : Dla niedowiárſtwá wáƺego ; zápráwdę bowiem powiádám wám : Jeſlibyśćie / májąc wiárę jáko źiárno gorcżycżne / rzekli tey górze : Przenieś śię ſtąd ná ono miejſce / tedy śię przenieśie / á nic niemożebnego wám nie będźie.
Ale ten rodzáj nie wychodźi tylko przez modlitwę y przez poſt.
A gdy przebywáli w Gálilei / rzekł do nich JEzuſ : Syn cżłowiecży będźie wydány w ręce ludzkie ;
Y zábiją go / ále trzećiego dniá zmartwychwſtánie. Y záſmućili śię bárdzo.
A gdy przyƺli do Kápernáum / przyſtąpili do Piotrá ći / którzy podátek wybieráli / y rzekli : Izáli náucżyćiel wáƺ nie dáje podátku?
Y rzekł : Dáje. A gdy wchodźił w dom / uprzedźił go JEzuſ / mówiąc : Cóż ći śię zdá / Szymonie? Królowie źiemſcy od kogoż biorą cło álbo cżynƺ? od ſynów ſwoich / cżyli od obcych?
Rzekł mu Piotr : Od obcych. Y rzekł mu JEzuſ : Toć tedy ſynowie ſą wolni.
Wƺákże ábyśmy ich nie zgorƺyli / ƺedłƺy do morzá / zárzuć wędkę / á tę rybę / która nájpierwej uwięźnie / weźmij / á otworzywƺy gębę jey / znájdźieƺ ſtáter / który wźiąwƺy / dáj im zá mię y zá śię.
