Strzeżćie śię / ábyśćie jáłmużny wáƺej nie cżynili przed ludźmi dlátego / ábyśćie byli widziáni od nich ; inácżej nie będziećie mieli zápłáty u Ojcá wáƺego / który jeſt w niebieśiech.
Przetoż / gdy cżyniƺ jáłmużnę / nie trąb przed ſobą / jáko obłudnicy cżynią w bóżnicách y ná ulicách / áby byli chwáleni od ludzi ; zápráwdę powiádám wám / odbieráją zápłátę ſwoję.
Ale ty gdy cżyniƺ jáłmużnę / niecháj nie wie lewicá twojá / co cżyni práwicá twojá.
Aby jáłmużná twojá byłá w ſkrytośći / á Ojćiec twój / który widzi / w ſkrytośći / ten ći jáwnie oddá.
A gdy śię modliƺ / nie bądź jáko obłudnicy ; ábowiem śię oni rádzi w bóżnicách y ná rogách ulic ſtojąc / modlą / áby byli widziáni od ludzi ; zápráwdę powiádám wám / iż odbieráją zápłátę ſwoję.
Ale ty / gdy śię modliƺ / wnijdź do komory ſwojej / á záwárłƺy drzwi ſwoje / módl śię Ojcu twemu / który jeſt w ſkrytośći ; á Ojćiec twój / który widzi w ſkrytośći / oddáć jáwnie.
A modląc śię / nie bądźćie wielomówni / jáko pogánie ; ábowiem oni mniemáją / że dlá ſwojej wielomównośći wyſłucháni będą.
Nie bądźćież tedy im podobni / gdyż wie Ojćiec wáƺ / cżego potrzebujećie / pierwej niżbyśćie wy go prośili.
Wy tedy ták śię módlćie ; Ojcże náƺ / któryś jeſt w niebieśiech! Święć śię imię twoje ;
Przyjdź króleſtwo twoje ; bądź wolá twojá jáko w niebie / ták y ná ziemi.
Chlebá náƺego powƺedniego dáj nám dziśiáj.
Y odpuść nám náƺe winy / jáko y my odpuƺcżámy náƺym winowájcom ;
Y nie wwódź náſ ná pokuƺenie / ále náſ zbáw ode złego ; ábowiem twoje jeſt króleſtwo y moc y chwáłá ná wieki. Amen.
Bo jeźli odpuśćićie ludziom upádki ich / odpuśći y wám Ojćiec wáƺ niebieſki ;
A jeźli nie odpuśćićie ludziom upádków ich / y Ojćiec wáƺ nie odpuśći wám upádków wáƺych.
A gdy pośćićie / nie bądźćież ſmętnej twárzy / jáko obłudnicy ; ƺpecą bowiem twárzy ſwoje / áby byli widziáni od ludzi / że poƺcżą ; zápráwdę powiádám wám / odbieráją zápłátę ſwoję.
Ale ty / gdy pośćiƺ / námáż głowę twoję / y umyj twárz twoję /
Abyś nie był widziány od ludzi / że pośćiƺ / ále od Ojcá twojego / który jeſt w ſkrytośći ; á Ojćiec twój / który widzi w ſkrytośći / oddáć jáwnie.
Nie ſkárbćie ſobie ſkárbów ná ziemi / gdzie mól y rdzá pſuje / y gdzie złodzieje podkopywują y krádną ;
Ale ſobie ſkárbćie ſkárby w niebie / gdzie áni mól áni rdzá pſuje / y gdzie złodzieje nie podkopywują / áni krádną.
Abowiem gdzie jeſt ſkárb wáƺ / tám jeſt y ſerce wáƺe.
Oko twoje jeſtći świecą ćiáłá twego ; jeźliby tedy oko twoje było ƺcżere / wƺyſtko ćiáło twoje jáſne będzie ;
Jeźliby záś oko twoje złe było / wƺyſtko ćiáło twoje ćiemne będzie ; jeźli tedy świátłość / którá jeſt w tobie / ćiemnośćią jeſt / ſámá ćiemność jákáż będzie?
Nikt nie może dwom pánom ſłużyć / gdyż álbo jednego będzie miáł w nienáwiśći / á drugiego będzie miłowáł ; álbo jednego trzymáć śię będzie / á drugim pogárdzi ; nie możećie Bogu ſłużyć y mámonie.
Dlátego powiádám wám : Nie troƺcżćie śię o żywot wáƺ / cobyśćie jedli / álbo cobyśćie pili / áni o ćiáło wáƺe / cżembyśćie śię odziewáli ; ázáż żywot nie jeſt zácniejƺy niż pokárm / y ćiáło niż odzienie?
Spojrzyjćie ná ptáki niebieſkie / iż nie śieją áni żną / áni zbieráją do gumien / á wżdy Ojćiec wáƺ niebieſki żywi je ; izáli wy nie jeſteśćie dáleko zácniejśi nád nie?
Y któż z wáſ troſkliwie myśląc / może przydáć do wzroſtu ſwego łokieć jeden?
A o odzienie przecżże śię troƺcżyćie? Przypátrzćie śię lilijom polnym / jáko roſną ; nie prácują / áni przędą.
A Já wám powiádám / iż áni Sálomon we wƺyſtkiej ſłáwie ſwojej nie był ták przyodziány / jáko jedná z tych.
Jeźli tedy tráwę polną / którá dziś jeſt / á jutro bywá w piec wrzuconá / Bóg ták przyodziewá / ázáż nie dáleko więcej wáſ! o máłowierni!
Nie troƺcżćie śię tedy / mówiąc : Cóż będziemy jeść? álbo co będziemy pić? álbo cżem śię będziemy przyodziewáć?
Boć tego wƺyſtkiego pogánie ƺukáją ; wieć bowiem Ojćiec wáƺ niebieſki / że tego wƺyſtkiego potrzebujećie.
Ale ƺukájćie náprzód króleſtwá Bożego / y ſpráwiedliwośći jego / á to wƺyſtko będzie wám przydáno.
Przetoż nie troƺcżćie śię o jutrzejƺy dzień : ábowiem jutrzejƺy dzień troſkáć śię będzie o ſwoje potrzeby. Doſyćći má dzień ná ſwojim utrápieniu.
