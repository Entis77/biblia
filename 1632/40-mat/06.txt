Strzeżćie śię ábyśćie jáłmużny wáƺej nie cżynili przed ludźmi / <i>dla tego</i> ábyśćie byli widźiáni od nich : inácżey nie będźiećie mieli zápłáty u Ojcá wáƺego / który jeſt w niebieśiech.
Przetoż / gdy cżyniƺ jáłmużnę / nie trąb przed ſobą jáko obłudnicy cżynią w bóżnicách y ná ulicách / áby byli chwáleni od ludźi : Zápráwdę powiádám wam / odbieráją zápłátę ſwoję.
Ale ty gdy cżyniƺ jáłmużnę ; niecháj nie wie lewicá twojá / co cżyni práwicá twojá:
Aby jáłmużná twojá byłá w ſkrytośći ; á Oćiec twój / który widźi w ſkrytośći / tenći jáwnie oddá.
A gdy śię modliƺ / nie bądź jáko obłudnicy : ábowiem śię oni rádźi w bóżnicách y ná rogách ulic ſtojąc modlą / áby byli widźiáni od ludźi : Zápráwdę powiádám wam / iż odbieráją zápłátę ſwoję.
Ale ty gdy śię modliƺ / wnijdź do komory ſwojey : á záwárƺy drzwi ſwoje / módl śię Ojcu twemu który jeſt w ſkrytośći : á Oćiec twój który widźi w ſkrytośći / oddáć jáwnie.
A modląc śię / nie bądźćie wielomówni jáko pogánie : ábowiem oni mniemáją / że dla ſwojey wielomównośći wyſłucháni będą.
Nie bądźćież tedy im podobni : gdyż wie Oćiec wáƺ cżego potrzebujećie / pierwej niżbyśćie wy go prośili.
Wy tedy ták śię módlćie : Ojcże náƺ / któryś jeſt w niebieśiech. Święć śię Imię twoje.
Przyjdź króleſtwo twoje. Bądź wolá twojá jáko w niebie / ták y ná źiemi.
Chlebá náƺego powƺedniego dáj nam dźiśiáj.
Y odpuść nam náƺe winy / jáko y my odpuƺcżámy náƺym winowájcom.
Y nie wwódź nas ná pokuƺenie ; Ale nas zbáw ode złego. Abowiem twoje jeſt króleſtwo / y moc / y chwałá ná wieki. Amen.
Bo jeſli odpuśćićie ludźiom upádki ich / odpuśći y wam on Oćiec wáƺ niebieſki.
Ale jeſli nie odpuśćićie ludźiom upádków ich / y Oćiec wáƺ nie odpuśći wam upádków wáƺych.
A gdy pośćićie / nie bądźćież ſmętney twárzy / jáko obłudnicy : ƺpecą bowiem twárzy ſwoje / áby byli widźiáni od ludźi że poƺcżą. Zápráwdę powiádám wam / odbieráją zápłátę ſwoję.
Ale ty gdy pośćiƺ / námáż głowę twoję / y umyj twárz twoję :
Abyś nie był widźiány od ludźi że pośćiƺ / ále od onego Ojcá twojego / który jeſt w ſkrytośći : á Oćiec twój który widźi w ſkrytośći / oddáć jáwnie.
Nie ſkárbćie ſobie ſkárbów ná źiemi / gdźie mól y rdzá pſuje / y gdźie złodźieje podkopywują y krádną.
Ale ſobie ſkárbćie ſkárby w niebie : gdźie áni mól áni rdzá pſuje ; y gdźie złodźieje nie podkopywują / áni krádną.
Abowiem gdźie jeſt ſkárb wáƺ / tám jeſt y ſerce wáƺe.
Oko twoje / jeſtći świecą ćiáłá twego : jeſliby tedy oko twoje było ƺcżere / wƺyſtko ćiáło twoje jáſne będźie.
Jeſliby záś oko twoje złe było / wƺyſtko ćiáło twoje ćiemne będźie. Jeſli tedy świátłość / która jeſt w tobie / ćiemnośćią jeſt / <i>ſámá</i> ćiemność jákáż <i>będźie</i>?
Nikt nie może dwiema Pánom ſłużyć : gdyż álbo jednego będźie miał w nienáwiśći / á drugiego będźie miłował : álbo jednego trzymáć śię będźie / á drugim pogárdźi. Nie możećie Bogu ſłużyć y Mámonie.
Dla tego powiádám wam : Nie troƺcżćie śię o żywot wáƺ / cobyśćie jedli / álbo cobyśćie pili ; áni o ćiáło wáƺe / cżembyśćie śię odźiewáli : ázaż żywot nie jeſt zácniejƺy niż pokárm / y ćiáło niż odźienie?
Pojrzyjćie ná ptáki niebieſkie / iż nie śieją áni żną / áni zbieráją do gumien / á wżdy Oćiec wáƺ niebieſki żywi je. Izali wy nie jeſteśćie dáleko zácniejśi nád nie?
Y któż z was troſkliwie myśląc może przydáć do wzroſtu ſwego łokieć jeden?
A o odźienie przecżże śię troƺcżyćie? Przypátrzćie śię lilijom polnym / jáko roſną ; nie prácują / áni przędą?
A Ja wam powiádám / iż áni Sálomon we wƺyſtkiej ſłáwie ſwojey nie był <i>ták</i> przyodźiány / jáko jedná z tych.
Jeſli tedy tráwę polną / która dźiś jeſt / á jutro bywa w piec wrzuconá / Bóg ták przyodźiewá / ázaż nie dáleko więcey was / o máłowierni?
Nie troƺcżćie śię tedy / mówiąc ; Cóż będźiemy jeść? álbo co będźiemy pić? álbo cżym śię będźiemy przyodźiewáć.
Boć tego wƺyſtkiego Pogánie ƺukáją : wieć bowiem Oćiec wáƺ niebieſki / że tego wƺyſtkiego potrzebujećie.
Ale ƺukájćie náprzód króleſtwá Bożego / y ſpráwiedliwośći jego / á to wƺyſtko będźie wam przydáno.
Przetoż / nie troƺcżćie śię o jutrzejƺy dźień : ábowiem jutrzejƺy dźień troſkáć śię będźie o ſwoje <i>potrzeby</i>. Doſyćći ma dźień ná ſwojim utrápieniu.
