Tedy rzekł nájwyżƺy kápłán : A tákże śię má tá rzecż?
A on rzekł : Mężowie bráćiá y ojcowie / ſłuchájćie! Bóg chwáły ukázáł śię ojcu náƺemu Abráhámowi / gdy był w Mezopotámii / przedtem niż mieƺkáł w Háránie.
Y rzekł do niego : Wynijdź z ziemi twojej y od twojej rodziny / y idź do ziemi / którąć ukáżę.
Tedy wyƺedłƺy z ziemi Cháldejſkiej / mieƺkáł w Háránie / á ſtámtąd / gdy umárł ojćiec jego / przenióſł go Bóg do ziemi tej / w której wy teráz mieƺkáćie.
Y nie dáł mu w niej dziedzictwá y ná ſtopę nogi / choć mu ją był obiecáł dáć w dzierżáwę / y náśieniu jego po nim / gdy jeƺcże nie miáł potomká.
Y mówił mu ták Bóg : Náśienie twoje będzie przychodniem w cudzej ziemi / y zniewolą je / y trápić je będą przez cżtery ſtá lát.
Ale ten náród / któremu ſłużyć będą / já będę ſądził / rzekł Bóg ; á potem wynijdą / y ſłużyć mi będą ná tem miejſcu.
Y dáł mu przymierze obrzezki ; y ták Abráhám ſpłodził Izááká / y obrzezáł go dniá óſmego / á Izáák Jákóbá / á Jákób dwunáſtu pátryjárchów.
A pátryjárchowie nienáwidząc Józefá / ſprzedáli go do Egiptu ; ále Bóg był z nim.
Y wyrwáł go ze wƺyſtkich jego ućiſków / á dáł mu łáſkę y mądrość przed Fáráonem / królem Egipſkim / który go poſtánowił kśiążęćiem nád Egiptem y nád wƺyſtkim domem ſwoim.
Potem przyƺedł głód ná wƺyſtkę ziemię Egipſką y Chánánejſką / y ućiſk wielki / y nie znájdowáli żywnośći ojcowie náśi.
A gdy uſłyƺáł Jákób / iż zbożá były w Egipćie / poſłáł ojców náƺych pierwƺy ráz.
A zá wtórym rázem poznány jeſt Józef od bráći ſwych / y objáwiony jeſt Fáráonowi náród Józefowy.
Tedy Józef poſłáwƺy / przyzwáł ojcá ſwego Jákóbá / y wƺyſtkę ſwoję rodzinę w śiedmdzieśiąt y pięćiu duƺách.
Y zſtąpił Jákób do Egiptu / y tám umárł on y ojcowie náśi.
Y przenieśieni ſą do Sychem / y położeni w grobie / który był kupił Abráhám zá pieniądze u ſynów Hemorá / ojcá Sychemowego.
A gdy śię przybliżył cżáſ obietnicy / o którą był przyśiągł Bóg Abráhámowi / rozrodził śię lud / y rozmnożył śię w Egipćie.
Aż náſtáł inny król / który nie znáł Józefá.
Ten podchodząc zdrádliwie náród náƺ / trápił ojców náƺych / ták / iż muśieli wymiátáć niemowlątká ſwoje / żeby śię nie rozkrzewiáły.
Pod ten cżáſ národził śię Mojżeƺ / á był kráſnym z dáru Bożego / którego chowáno przez trzy mieśiące w domu ojcá jego.
A gdy był wyrzucony / wzięłá go córká Fáráonowá y wychowáłá go ſobie zá ſyná.
Y wyćwicżony jeſt Mojżeƺ we wƺelkiej mądrośći Egipſkiej / á był możny w mowách y w ucżynkách.
A gdy mu było cżterdzieśći lát / przyƺło mu ná myśl / áby náwiedził bráći ſwych / ſyny Izráelſkie.
A ujrzáwƺy jednego ukrzywdzonego / ujął śię oń / y pomśćił śię krzywdy tego / który bezpráwie ćierpiáł / zábiwƺy Egipcżániná.
Abowiem mniemáł / że bráćiá jego rozumieją / że Bóg przez rękę jego dáje im wybáwienie ; lecż oni tego nie rozumieli.
A názájutrz pokázáł śię im / gdy śię z ſobą bili / y prowádził je do pokoju / mówiąc : Mężowie! bráćiáśćie ſobie ; przecżże śię ſpołem krzywdzićie?
Lecż ten / co krzywdził bliźniego / odegnáł go / mówiąc : Któż ćię poſtánowił kśiążęćiem y ſędzią nád námi?
Izáli mię ty chceƺ zábić / jákoś wcżoráj zábił Egipcżániná?
Y ućiekł Mojżeƺ zá temi ſłowy / y był przychodniem w ziemi Mádyjáńſkiej / gdzie ſpłodził dwóch ſynów.
A gdy śię wypełniło cżterdzieśći lát / ukázáł mu śię ná puƺcży góry Synáj Anioł Páńſki w płomieniu ogniſtym w krzáku.
A Mojżeƺ ujrzáwƺy / zádziwił śię onemu widzeniu ; á gdy przyſtąpił / áby śię temu przypátrzył / ſtáł śię do niego głoſ Páńſki :
Jám jeſt Bóg ojców twoich / Bóg Abráhámá / Bóg Izááká / y Bóg Jákóbá. A zádrżáwƺy Mojżeƺ / nie śmiáł śię przypátrywáć.
Y rzekł mu Pán : Zzuj obuwie z nóg twoich ; bo miejſce / ná którem ſtoiƺ / jeſt ziemiá świętá.
Widząc widziáłem utrápienie ludu mego / który jeſt w Egipćie / y ſłyƺáłem wzdychánie ich / á zſtąpiłem / żebym je wybáwił ; przetoż teráz chodź / poślę ćię do Egiptu.
Tego Mojżeƺá / którego śię byli záprzáli / mówiąc : Któż ćię poſtánowił kśiążęćiem y ſędzią? Tego Bóg kśiążęćiem y wybáwićielem poſłáł przez rękę Aniołá / który mu śię ukázáł w krzáku.
Y ten je wywiódł / cżyniąc cudá y známioná w ziemi Egipſkiej y ná morzu Cżerwonem y ná puƺcży / przez cżterdzieśći lát.
Tenći jeſt Mojżeƺ / który rzekł ſynom Izráelſkim : Proroká wám wzbudzi Pán / Bóg wáƺ / z bráći wáƺych / jáko mię / onego ſłucháć będziećie ;
Ten jeſt / który był w zgromádzeniu ná puƺcży z Aniołem / który mówił do niego ná górze Synáj / y z ojcámi náƺymi / który przyjął ſłowá Boże żywe / áby je nám podáł.
Któremu nie chćieli poſłuƺni być ojcowie náśi ; ále go odrzućili / y obróćili śię ſercy ſwemi do Egiptu.
Mówiąc do Aároná : Ucżyń nám bogi / którzyby ƺli przed námi ; ábowiem Mojżeƺowi onemu / który náſ wywiódł z ziemi Egipſkiej / nie wiemy co śię ſtáło.
Y ucżynili w one dni ćielcá / y ſpráwowáli ofiárę onemu báłwánowi / y weſelili śię w ſpráwách rąk ſwoich ;
Y odwróćił śię Bóg / y podáł je / áby ſłużyli wojſku niebieſkiemu / jáko nápiſáno jeſt w kśięgách prorockich : Záżeśćie mu bite / y inne ofiáry ofiárowáli ná puƺcży przez cżterdzieśći lát / domu Izráelſki?
Owƺem nośiliśćie námiot Molochá / y gwiázdę bogá wáƺego Remfáná / te obrázy / któreśćie ſobie ucżynili / ábyśćie śię im kłániáli ; przetoż wáſ záprowádzę zá Bábilon.
Námiot świádectwá mieli ojcowie náśi ná puƺcży / jáko był rozrządził ten / który powiedziáł Mojżeƺowi / áby go ucżynił według kƺtáłtu / który widziáł.
Który wziąwƺy ojcowie náśi / wnieśli z Jozuem tám / gdzie byłá ośiádłość pogánów / których Bóg wygnáł od oblicżnośći ojców náƺych / áż do dni Dawidowych ;
Który ználázł łáſkę przed oblicżem Bożem / y prośił / áby ználázł námiot Bogu Jákóbowemu.
A Sálomon zbudowáł mu dom.
Ale on Nájwyżƺy nie mieƺká w kośćiołách ręką ucżynionych / jáko prorok mówi :
Niebo jeſt ſtolicá mojá / á ziemiá podnóżek nóg moich. Cóż mi zá dom zbudujećie / mówi Pán / álbo które jeſt miejſce odpocżnieniá mego?
Izáli ręká mojá tego wƺyſtkiego nie ucżyniłá?
Ludzie twárdego kárku y nieobrzezánego ſercá y uƺów! wy śię záwżdy ſprzećiwiáćie Duchowi Świętemu jáko ojcowie wáśi / ták y wy.
Któregoż z proroków nie prześládowáli ojcowie wáśi / y niepozábijáli tych / którzy przedtem opowiádáli o przyjśćiu tego Spráwiedliwego / któregośćie wy śię teráz ſtáli zdrájcámi y mordercámi?
Którzyśćie wzięli zákon przez rozrządzenie Anielſkie / á nie ſtrzegliśćie go.
Tedy ſłuchájąc tego / pukáli śię w ſercách ſwych / y zgrzytáli náń zębámi.
A on będąc pełen Duchá Świętego / pátrząc pilnie w niebo / ujrzáł chwáłę Bożą / y Jezuſá ſtojącego po práwicy Bożej.
Y rzekł : Oto widzę niebioſá otworzone / y Syná cżłowiecżego ſtojącego po práwicy Bożej.
A oni krzyknąwƺy głoſem wielkim / zátulili uƺy ſwoje / y rzućili śię náń jednomyślnie.
A wypchnąwƺy go z miáſtá / kámionowáli ; á świádkowie złożyli ƺáty ſwoje u ſtóp młodzieńcá / którego zwáno Saul.
Y kámionowáli Szcżepáná / modlącego śię y mówiącego : Pánie Jezu! przyjmij duchá mojego!
A klęknąwƺy ná koláná / záwołáł głoſem wielkim : Pánie! nie pocżytáj im tego zá grzech! A to rzekłƺy / záſnął.
