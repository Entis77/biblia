A gdy zdrowo uƺli / dopiero poználi / iż onę wyſpę Melitą názywáno.
Ale on gruby lud pokázáł nám nie ládá ludzkość ; ábowiem zápáliwƺy ſtoſ drew / przyjęli náſ wƺyſtkich dlá deƺcżu pádájącego y dlá zimná.
A gdy Páweł nágárnął gromádę chroſtu y kłádł ná ogień / wyrwáwƺy śię żmijá z gorącá / przypięłá śię do ręki jego.
A gdy on lud gruby ujrzáł onę gádzinę wiƺącą u ręki jego / mówili jedni do drugich : Pewnie ten cżłowiek jeſt mężobójcą ; bo choć z morzá uƺedł / przećię mu pomſtá żywym być nie dopuśćiłá.
Lecż on otrząſnąwƺy onę gádzinę w ogień / nic złego nie ućierpiáł.
A oni cżekáli / żeby opuchł álbo nágle upádłƺy umárł ; á gdy tego długo cżekáli á widzieli / iż mu śię nic złego nie ſtáło / odmieniwƺy śię / mówili / że jeſt Bogiem.
A przy onych miejſcách miáł folwárki przedniejƺy onej wyſpy / imieniem Publijuƺ / który przyjąwƺy náſ / przez trzy dni przyjáćielſko podejmowáł.
Y ſtáło śię / że ojćiec onego Publijuƺá / májąc gorącżkę y biegunkę / leżáł ; do którego Páweł wƺedłƺy / modlił śię / á włożywƺy náń ręce / uzdrowił go.
To gdy śię ſtáło / tedy drudzy / którzy byli złożeni chorobámi ná onej wyſpie / przychodzili / y byli uzdrowieni ;
Którzy nám też wielką ucżćiwość wyrządzáli / á gdyśmy precż płynąć mieli / nákłádli nám / cżego było potrzebá.
A po trzech mieśiącách puśćiliśmy śię w okręćie Alekſándryjſkim / który zimowáł ná onej wyſpie / májącym zá herb Káſtorá y Pollukſá.
A przypłynąwƺy do Syrákuſ / zámieƺkáliśmy tám trzy dni.
A ſtámtąd płynąc kołem / przybyliśmy do Regijum / á po jednym dniu / gdy powſtáł wiátr południowy / wtórego dniá przypłynęliśmy do Puteolów.
Gdzie ználázłƺy bráći / uproƺeniśmy byli od nich / żebyśmy zámieƺkáli u nich przez śiedm dni ; á tákieśmy ƺli do Rzymu.
Stąd / gdy uſłyƺeli bráćiá o náſ / wyƺli przećiwko nám áż do rynku Appijuƺowego y do Trzech kárcżem ; których gdy Páweł ujrzáł / podziękowáwƺy Bogu / wziął śmiáłość.
A gdyśmy przyƺli do Rzymu / ſetnik oddáł więźnie hetmánowi wojſká ; ále Páwłowi dopuƺcżono / mieƺkáć oſobno z żołnierzem / który go ſtrzegł.
Y ſtáło śię po trzech dniách / że zwołáł Páweł przedniejƺych z Żydów ; á gdy śię zeƺli / rzekł do nich : Mężowie bráćiá! já nic nie ucżyniwƺy przećiwko ludowi y zwycżájom ojcżyſtym / będąc związány w Jeruzálemie / podánym jeſt w ręce Rzymián ;
Którzy wyſłucháwƺy mię / chćieli mię wypuśćić dlátego / że we mnie żádnej winy godnej śmierći nie było.
Lecż gdy śię temu ſprzećiwiáli Żydowie / muśiáłem ápelowáć do ceſárzá ; nie żebym miáł náród mój w cżem oſkárżáć.
Dlá tej tedy przycżyny zwołáłem wáſ / ábym śię z wámi ujrzáł y rozmówił ; ábowiem dlá nádziei ludu Izráelſkiego tym łáńcuchem jeſtem opáſány.
Lecż oni rzekli do niego : My ániśmy liſtów doſtáli o tobie z Judzkiej ziemi / áni kto z bráći przyƺedłƺy oznájmił / álbo mówił o tobie co złego.
Wƺákżebyśmy rádzi od ćiebie ſłyƺeli / co rozumieƺ ; ábowiem o tej ſekćie wiemy / iż wƺędzie przećiwko niej mówią.
A poſtánowiwƺy mu dzień / przyƺło ich do niego do goſpody niemáło / którym z oświádcżániem wykłádáł króleſtwo Boże / námáwiájąc ich do tych rzecży / które ſą o Jezuśie / z zákonu Mojżeƺowego y z proroków / od poránku áż do wiecżorá.
Tedy niektórzy uwierzyli temu / co mówił / á niektórzy nie uwierzyli.
A będąc niezgodnymi między ſobą / rozeƺli śię / gdy Páweł rzekł to jedno ſłowo : Iż dobrze Duch Święty powiedziáł przez Izájáƺá proroká / do ojców náƺych /
Mówiąc : Idź do tego ludu / á mów : Słuchem ſłucháć będziećie / ále nie zrozumiećie / á widząc widzieć będziećie / ále nic nie ujrzyćie ;
Abowiem zgrubiáło ſerce ludu tego / á ćiężko uƺymá ſłyƺeli y zámrużyli ocży ſwe / áby ſnáć ocżymá nie widzieli / á uƺymá nie ſłyƺeli / y ſercem nie zrozumieli / y nie náwróćili śię / á uzdrowiłbym je.
Niechże wám tedy wiádomo będzie / iż pogánom poſłáne jeſt to zbáwienie Boże / á oni ſłucháć będą.
A gdy to on rzekł / odeƺli Żydowie / májąc między ſobą wielki ſpór.
Y mieƺkáł Páweł przez cáłe dwá látá w nájemnej goſpodzie ſwojej / y przyjmowáł wƺyſtkich / którzy przychodzili do niego ;
Káżąc o króleſtwie Bożem / y ucżąc tych rzecży / które ſą o Pánu Jezuśie Chryſtuśie / ze wƺyſtkiem bezpiecżeńſtwem / bez przeƺkody.
