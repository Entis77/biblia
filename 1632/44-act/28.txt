A gdy zdrowo uƺli / dopiero poználi iż on wyſpey Melitą názywano.
Ale on gruby lud / pokazał nam nie ládá ludzkość : ábowiem zápaliwƺy ſtos drew / przyjęli nas wƺyſtkich / dla deƺcżu pádájącego / y dla źimná.
A gdy Páweł nágárnął gromádę chróſtu / y kładł ná ogień / wyrwawƺy śię źmijá z gorącá / przypięłá śię do ręki jego.
A gdy on lud gruby ujrzał onę gádźinę wiƺącą u ręki jego / mówili jedni do drugich : Pewnie ten cżłowiek jeſt mężobójcą ; bo choć z morzá uƺedł / <i>przećię</i> mu pomſtá żywym być nie dopuśćiłá.
Lecż on otrząſnąwƺy onę gádźinę w ogień / nic złego nie ućierpał.
A oni cżekáli żeby opuchł / álbo nagle upadƺy umárł. A gdy <i>tego</i> długo cżekáli / á widźieli iż mu śię nic złego nie ſtáło / odmieniwƺy śię / mówili że jeſt Bogiem.
A przy onych miejſcách miał folwárki Przedniejƺy onego wyſpu imieniem Publiuƺ : który przyjąwƺy nas / przez trzy dni przyjaćielſkie podejmował.
Y ſtáło śię że Oćiec onego Publiuƺá májąc gorącżkę y biegunkę / leżał : do którego Páweł wƺedƺy / modlił śię ; á włożywƺy nań ręce / uzdrowił go.
Co gdy śię ſtáło / tedy drudzy / którzy byli złożeni chorobámi ná onym wyſpie przychodźili / y byli uzdrowieni.
Którzy nam też wielką ucżćiwość wyrządzáli. A gdyſmy precż płynąć mieli / nákłádli <i>nam</i> cżego było potrzebá.
A po trzech mieśiącách / puśćiliſmy śię w okręćie Alexándryjſkim / który źimował ná onym wyſpie / májącym zá herb / Káſtorá y Polluxá.
A przypłynąwƺy do Syrákus / zmieƺkáliſmy <i>tám</i> trzy dni.
<i>A</i> z támtąd płynąc kołem / przybyliſmy do Regium. A po jednym dniu / gdy powſtał wiátr Południowy / wtórego dniá przypłynęliſmy do Puteolów.
Gdźie ználazƺy bráćią / uproƺeniſmy byli od nich / żebyſmy zmieƺkáli u nich przez śiedm dni : á tákeſmy ƺli do Rzymu.
Z tąd / gdy uſłyƺeli bráćia o nas / wyƺli przećiwko nam áż do rynku Appiuƺowego / y do Trzech kárczem : które gdy Páweł ujrzał / podźiękowawƺy Bogu / wźiął śmiáłość.
A gdyſmy przyƺli do Rzymu / Setnik oddał więźnie Hetmánowi wojſká : ále Páwłowi dopuƺcżono mieƺkáć oſobno z żołnierzem / który go ſtrzegł.
Y ſtáło śię po trzech dniách / że zwołał Páweł Przedniejƺych z Żydów ; á gdy śię zeƺli rzekł do nich : Mężowie bráćia / ja nic nie ucżyniwƺy przećiwko ludowi / y zwycżájom Ojcżyſtym / będąc związány w Jeruzalem / podánym jeſt w ręce Rzymiánów.
Którzy wyſłuchawƺy mię / chćieli mię wypuśćić ; dla tego / że we mnie żadney winy <i>godney</i> śmierći nie było.
Lecż gdy śię <i>temu</i> zprzećiwiáli Żydowie / muśiałem áppellowáć do Ceſárzá ; nie żebym miał naród mój w cżym oſkarżáć.
Dla tey tedy przycżyny zwołałem was / ábym śię z wámi ujrzał y rozmówił : ábowiem dla nádźieje ludu Izráelſkiego / tym łáńcuchem jeſtem opáſány.
Lecż oni rzekli do niego : My / ániſmy liſtów doſtáli o tobie z Judſkiey źiemie / áni kto z bráći przyƺedƺy oznájmił álbo mówił o tobie co złego.
Wƺákżebyſmy rádźi od ćiebie ſłyƺeli co rozumieƺ. Abowiem o tey ſekćie wiemy iż wƺędźie przećiwko niey mówią.
A poſtánowiwƺy mu dźień / przyƺło ich do niego do goſpody niemáło : którym z oświadcżániem wykładał króleſtwo Boże / námawiájąc ich do tych rzecży które ſą o JEzuśie / z zakonu Mojzeƺowego y z Proroków ; od poránku áż do wiecżorá.
Tedy niektórzy uwierzyli temu co mówił / á niektórzy nie uwierzyli.
A będąc niezgodnymi miedzy ſobą / rozeƺli śię / gdy Páweł rzekł <i>to</i> jedno ſłowo : Iż dobrze Duch Święty powiedźiał przez Izájaƺá Proroká do Ojców náƺych /
Mówiąc : Idź do tego ludu / á mów : Słuchem ſłucháć będźiećie / ále nie zrozumiećie ; á widząc widźieć będźiećie / ále nic nie ujrzyćie.
Abowiem zgrubiáło ſerce ludu tego / á ćiężko uƺymá ſłyƺeli / y zámrużyli ocży ſwe : áby ſnadź ocżymá nie widźieli / á uƺymá nie ſłyƺeli / y ſercem nie zrozumieli / y nie náwróćili śię / á uzdrowiłbym je.
Niechże wam tedy wiádomo będźie / iż Pogánom poſłáne jeſt to zbáwienie Boże / á oni ſłucháć będą.
A gdy to on rzekł / odeƺli Żydowie / májąc miedzy ſobą wielki ſpór.
Y mieƺkał Páweł przez cáłe dwie lećie w najemney goſpodźie ſwojey : y przyjmował wƺyſtkie którzy przychodźili do niego :
Każąc o króleſtwie Bożym / y ucżąc tych rzecży które ſą o PAnu JEzuśie CHryſtuśie / ze wƺelkim beſpiecżeńſtwem / bez przeƺkadzki.
