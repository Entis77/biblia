A gdy zdrowo uƺli / dopiero poználi / iż onę wyſpę Melitą názywáno.
Ale on gruby lud pokázáł nám nie ládá ludzkość ; ábowiem zápáliwƺy ſtoſ drew / przyjęli nas wƺyſtkich dla deƺcżu pádájącego y dla źimná.
A gdy Páweł nágárnął gromádę chroſtu y kłádł ná ogień / wyrwáwƺy śię żmijá z gorącá / przypięłá śię do ręki jego.
A gdy on lud gruby ujrzáł onę gádźinę wiƺącą u ręki jego / mówili jedni do drugich : Pewnie ten cżłowiek jeſt mężobójcą ; bo choć z morzá uƺedł / przećię mu pomſtá żywym być nie dopuśćiłá.
Lecż on otrząſnąwƺy onę gádźinę w ogień / nic złego nie ućierpiáł.
A oni cżekáli / żeby opuchł álbo nágle upádłƺy umárł ; á gdy tego długo cżekáli á widźieli / iż mu śię nic złego nie ſtáło / odmieniwƺy śię / mówili / że jeſt Bogiem.
A przy onych miejſcách miał folwárki przedniejƺy oney wyſpy / imieniem Publijuƺ / który przyjąwƺy nas / przez trzy dni przyjáćielſko podejmowáł.
Y ſtáło śię / że ojćiec onego Publijuƺá / májąc gorącżkę y biegunkę / leżáł ; do którego Páweł wƺedłƺy / modlił śię / á włożywƺy náń ręce / uzdrowił go.
To gdy śię ſtáło / tedy drudzy / którzy byli złożeni chorobámi ná oney wyſpie / przychodźili / y byli uzdrowieni ;
Którzy nám też wielką ucżćiwość wyrządzáli / á gdyśmy precż płynąć mieli / nákłádli nám / cżego było potrzebá.
A po trzech mieśiącách puśćiliśmy śię w okręćie Alekſándryjſkim / który źimowáł ná oney wyſpie / májącym zá herb Káſtorá y Pollukſá.
A przypłynąwƺy do Syrákuſ / zámieƺkáliśmy tám trzy dni.
A ſtámtąd płynąc kołem / przybyliśmy do Regijum / á po jednym dniu / gdy powſtáł wiátr południowy / wtórego dniá przypłynęliśmy do Puteolów.
Gdźie ználázłƺy bráći / uproƺeniśmy byli od nich / żebyśmy zámieƺkáli u nich przez śiedm dni ; á tákieśmy ƺli do Rzymu.
Stąd / gdy uſłyƺeli bráćia o nas / wyƺli przećiwko nám áż do rynku Appijuƺowego y do Trzech kárcżem ; których gdy Páweł ujrzáł / podźiękowáwƺy Bogu / wźiął śmiáłość.
A gdyśmy przyƺli do Rzymu / ſetnik oddáł więźnie hetmánowi wojſká ; ále Páwłowi dopuƺcżono / mieƺkáć oſobno z żołnierzem / który go ſtrzegł.
Y ſtáło śię po trzech dniách / że zwołáł Páweł przedniejƺych z Żydów ; á gdy śię zeƺli / rzekł do nich : Mężowie bráćia! ja nic nie ucżyniwƺy przećiwko ludowi y zwycżájom ojcżyſtym / będąc związány w Jeruzálemie / podánym jeſt w ręce Rzymián ;
Którzy wyſłucháwƺy mię / chćieli mię wypuśćić dlátego / że we mnie żádney winy godney śmierći nie było.
Lecż gdy śię temu ſprzećiwiáli Żydowie / muśiáłem ápelowáć do ceſárzá ; nie żebym miał náród mój w cżem oſkárżáć.
Dla tey tedy przycżyny zwołáłem was / ábym śię z wámi ujrzáł y rozmówił ; ábowiem dla nádźiei ludu Izráelſkiego tym łáńcuchem jeſtem opáſány.
Lecż oni rzekli do niego : My ániśmy liſtów doſtáli o tobie z Judzkiej źiemi / áni kto z bráći przyƺedłƺy oznájmił / álbo mówił o tobie co złego.
Wƺákżebyśmy rádźi od ćiebie ſłyƺeli / co rozumieƺ ; ábowiem o tey ſekćie wiemy / iż wƺędźie przećiwko niej mówią.
A poſtánowiwƺy mu dźień / przyƺło ich do niego do goſpody niemáło / którym z oświádcżániem wykłádáł króleſtwo Boże / námáwiájąc ich do tych rzecży / które ſą o JEzuśie / z zakonu Mojżeƺowego y z proroków / od poránku áż do wiecżorá.
Tedy niektórzy uwierzyli temu / co mówił / á niektórzy nie uwierzyli.
A będąc niezgodnymi między ſobą / rozeƺli śię / gdy Páweł rzekł to jedno ſłowo : Iż dobrze Duch Święty powiedźiáł przez Izájáƺá proroká / do ojców náƺych /
Mówiąc : Idź do tego ludu / á mów : Słuchem ſłucháć będźiećie / ále nie zrozumiećie / á widząc widźieć będźiećie / ále nic nie ujrzyćie ;
Abowiem zgrubiáło ſerce ludu tego / á ćiężko uƺymá ſłyƺeli y zámrużyli ocży ſwe / áby ſnáć ocżymá nie widźieli / á uƺymá nie ſłyƺeli / y ſercem nie zrozumieli / y nie náwróćili śię / á uzdrowiłbym je.
Niechże wám tedy wiádomo będźie / iż pogánom poſłáne jeſt to zbáwienie Boże / á oni ſłucháć będą.
A gdy to on rzekł / odeƺli Żydowie / májąc między ſobą wielki ſpór.
Y mieƺkáł Páweł przez cáłe dwá látá w nájemney goſpodźie ſwojey / y przyjmowáł wƺyſtkich / którzy przychodźili do niego ;
Káżąc o króleſtwie Bożem / y ucżąc tych rzecży / które ſą o Pánu JEzuśie CHryſtuśie / ze wƺyſtkiem bezpiecżeńſtwem / bez przeƺkody.
