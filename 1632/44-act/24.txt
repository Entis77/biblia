A po pięćiu dniách jecháł nájwyżƺy kápłán Anánijáƺ z ſtárƺymi y z Tertullem niejákim prokurátorem ; którzy ſtánęli przed ſtároſtą przećiwko Páwłowi.
A gdy był pozwány / pocżął náń ſkárżyć Tertulluſ / mówiąc :
Poniewáżeśmy wielkiego pokoju doſtąpili / y wiele śię dobrego temu národowi ſtáło przez twoję opátrzność / y záwƺe y wƺędy to ze wƺelákiem dźiękowániem przyznájemy / wielmożny Felikśie!
Ale żebym ćię długo nie báwił / proƺę / ábyś nas málucżko poſłucháł według zwykłej twojey ludzkośći.
Abowiemeśmy ználeſli tego mężá záráźliwego / y wƺcżynájącego rozruch między wƺyſtkimi Żydámi po wƺyſtkim świećie / y herƺtá tey ſekty Názárejcżyków.
Który śię też wáżył ſplugáwić kośćiół ; któregośmy też pojmáwƺy / według zakonu náƺego chćieli ſądźić.
Lecż przyƺedłƺy hetmán Lizyjáƺ z wielką mocą / wźiął go z rąk náƺych.
Rozkázáwƺy tym / którzy náń ſkárżą / iść do ćiebie / od którego śię ty ſám będźieƺ mógł / wywiádując śię / dowiedźieć tego wƺyſtkiego / o co my náń ſkárżymy.
Ná co śię zgodźili y Żydowie / mówiąc : Że śię ták rzecż má.
Tedy Páweł odpowiedźiáł / gdy náń ſtároſtá ſkinął / áby mówił : Od wielu lát wiedząc ćię być ſędźią tego národu / tem ochotniej dám ſpráwę o tem / co śię mnie dotycże.
Gdyż ty wiedźieć możeƺ / iż niemáƺ więcey dni tylko dwánáśćie / jákom ja przyƺedł do Jeruzálemu / ábym śię modlił.
Dotego áni mię ználeſli w kośćiele z kim gádájącego álbo buntującego lud / áni w bóżnicách / áni w mieśćie ;
Ani tego mogą dowieść / o co tu teraz ná mię ſkárżą.
To jednák przed tobą wyznáję / że według oney drogi / którą oni powiádáją być heretyctwem / ták ſłużę ojcżyſtemu Bogu / wierząc wƺyſtkiemu / cokolwiek nápiſáno w zakonie y w prorokách /
Májąc nádźieję w Bogu / że będźie / którego y oni cżekáją / zmartwychwſtánie y ſpráwiedliwych y nieſpráwiedliwych.
A ſám śię o to pilnie ſtárám / ábym záwƺe miał ſumienie bez obráżeniá przed Bogiem y przed ludźmi.
A po wielu látách przyƺedłem / ábym przynióſł jáłmużny národowi memu y ofiáry.
Ná tem ználeſli mię w kośćiele ocżyƺcżonego ( nie z ludem áni z rozruchem ) niektórzy Żydowie z Azyi.
Którzy też tu mieli ſtánąć przed tobą y ſkárżyć / jeſliby co mieli przećiwko mnie.
Albo niecháj ći ſámi powiedzą / jeſli we mnie ználeſli jáką niepráwość / gdym ſtáł przed rádą ;
Oprócż tego jednego głoſu / żem między nimi ſtojąc / záwołáł : Dla zmartwychwſtániá umárłych ja dźiś ſądzony bywám od was.
A uſłyƺáwƺy to Felikſ / odłożył ſpráwę ich / mówiąc : Gdy śię o tey drodze doſtátecżniej wywiem / kiedy tu hetmán Lizyjáƺ przyjedźie / rozeznám ſpráwy wáƺe.
Y rozkázáł ſetnikowi / áby ſtrzegł Páwłá / y pofolgowáł mu / y áby nie bronił żádnemu z przyjáćiół jego / poſługiwáć mu / álbo go náwiedzáć.
A po kilku dniách przyjecháwƺy Felikſ / z Druſyllą / żoną ſwoją / która byłá żydówká / kázáł záwołáć Páwłá / y ſłucháł go o wierze w CHryſtuſá.
A gdy on rzecż cżynił o ſpráwiedliwośći / y o powśćiągliwośći / y o przyƺłym ſądźie / uląkł śię Felikſ y odpowiedźiáł : Już teraz odejdź / á gdy cżáſ upátrzę / káżę ćię záwołáć.
A przy tem ſpodźiewáł śię / że mu Páweł miał dáć pieniądze / żeby go wypuśćił ; dlátego też tem cżęśćiej go wzywájąc do śiebie / rozmáwiáł z nim.
A po wyjśćiu dwóch lát miał po ſobie Felikſ námieſtniká / Porcyjuſá Feſtá ; á chcąc ſobie Felikſ łáſkę zjednáć u Żydów / zoſtáwił Páwłá w więźieniu.
