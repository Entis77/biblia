A po piąći dniách jáchał Nawyżƺy kápłan Anániáƺ z Stárƺymi / y z Tertullem niejákim Prokuratorem ; którzy ſtánęli przed Stároſtą przećiwko Páwłowi.
A gdy był pozwány / pocżął <i>nań</i> ſkárżyć Tertullus / mówiąc :
Ponieważeſmy wielkiego pokoju doſtąpili y wiele śię dobrego temu narodowi ſtáło przez twoję opátrzność ; y záwżdy y wƺędy to ze wƺelákim dźiękowániem przyznawamy / wielmożny Felixie.
Ale żebym ćię długo nie báwił / proƺę ábyś nas málucżko poſłuchał według <i>zwykłey</i> ſwojey ludzkośći.
Abowiemeſmy ználeźli tego mężá záráźliwego / y wƺcżynájącego rozruch <i>miedzy</i> wƺyſtkimi Żydámi po wƺyſtkim świećie / y herſtá tey Sekty Názárejcżyków.
Który śię też ważył zplugáwić kośćiół : któregoſmy też pojimawƺy / według Zakonu náƺego chćieli ſądźić.
Lecż przyƺedƺy Hetmán Lizyaƺ z wielką mocą / wźiął go z rąk náƺych.
Rozkazawƺy tym którzy nań ſkárżą / iść do ćiebie / od którego śię <i>ty</i> ſam będźieƺ mógł wywiádując śię dowiedźieć tego wƺyſtkiego / o co my nań ſkárżymy.
Ná co śię zgodźili y Żydowie mówiąc : Że śię ták rzecż ma.
Tedy Páweł odpowiedźiał / gdy nań Stároſtá ſkinął áby mówił : Od wielu lat wiedząc ćię bydź Sędźią tego Narodu / tym ochotniey dam ſpráwę o tym co śię mnie dotycże.
Gdyż ty wiedźieć możeƺ / iż niemáƺ więcey dni tylko dwánaśćie / jákom ja przyƺedł do Jeruzalem ábym śię modlił.
Do tego / áni mię ználeźli w kośćiele z kim gadájącego / álbo buntującego lud / áni w bóżnicách / áni w mieśćie.
Ani tego mogą dowieść / o co tu teraz ná mię ſkárżą.
To jednák przed tobą wyznawam / że według oney drogi / którą <i>oni</i> powiádáją być heretyctwem / ták ſłużę Ojcżyſtemu Bogu / wierząc wƺyſtkiemu cokolwiek nápiſano w Zakonie y w Prorocech.
Májąc nádźieję w Bogu / że będźie / którego y oni cżekáją / zmartwychwſtánie y ſpráwiedliwych y nieſpráwiedliwych.
A ſam śię o to pilnie ſtáram / ábym záwƺe miał ſumnienie bez obráżenia przed Bogiem y przed ludźmi.
A po wielu lat / przyƺedłem ábym przyniózł jáłmużny narodowi memu / y ofiáry.
Ná tym ználeźli mię w Kośćiele ocżyśćionego ( nie z ludem áni z rozruchem ) niektórzy Żydowie z Azyey :
Którzy też tu mieli ſtánąć przed tobą y ſkárżyć / jeſliby co mieli przećiwko mnie.
Albo niechaj ći ſámi powiedzą / jeſli we mnie ználeźli jáką niepráwość gdym ſtał przed Rádą :
Oprócż tego jednego głoſu / żem miedzy nimi ſtojąc záwołał : Dla zmartwychwſtánia umárłych / ja dźiś ſądzony bywam od was.
A uſłyƺawƺy to Felix / odłożył <i>ſpráwę</i> ich / mówiąc : Gdy śię o tey drodze doſtátecżniey wywiem kiedy tu Hetman Lizyaƺ przyjedźie / rozeznam ſpráwy wáƺe.
Y rozkazał Setnikowi áby ſtrzegł Páwłá / y pofolgował mu ; y áby nie bronił żadnemu z przyjáćiół jego poſługowáć mu / álbo go náwiedzáć.
A po kilku dni przyjáchawƺy Felix z Druſyllą żoną ſwoją / która byłá Żydówká / kazał záwołáć Páwłá ; y ſłuchał go o wierze w CHryſtuſá.
A gdy on rzecż cżynił o ſpráwiedliwośći / y o powśćiągliwośći / y o przyƺłym ſądźie ; ulękł śię Felix / y odpowiedźiał : Już teraz odejdź : á gdy cżás upátrzę / każę ćię záwołáć.
A przy tym ſpodźiewał śię że mu Páweł miał dáć pieniądze / żeby go wypuśćił : dla tego też tym cżęśćiey go wzywájąc <i>do śiebie,</i> rozmawiał z nim.
A po wyśćiu dwu lat / miał <i>po ſobie</i> Felix Namieſtniká / Porcyuſá Feſtá. A chcąc ſobie Felix łáſkę z jednáć u Żydów / zoſtáwił Páwłá w więźieniu.
