Mężowie bráćia y ojcowie! ſłuchájćie mojey / którą teraz do was cżynię / obrony.
A gdy uſłyƺeli / iż do nich rzecż cżynił żydowſkim językiem / tem śię bárdźiej ućiƺyli. Y rzekł :
Jámći jeſt mąż Żyd / urodzony w Tárśie Cylicyjſkim / lecż wychowány w mieśćie tem u nóg Gámálijelowych / wyćwicżony doſtátecżnie w zakonie ojcżyſtym / gorliwym będąc miłośnikiem Bożym / jáko wy wƺyſcy dźiś jeſteśćie.
Którym prześládowáł tę drogę áż ná śmierć / wiążąc y podáwájąc do więźieniá y męże y niewiáſty /
Jáko mi tego y nájwyżƺy kápłán jeſt świádkiem / y wƺyſcy ſtárśi / od których też liſt wźiąwƺy do bráći / jecháłem do Dámáƺku / ábym y te / którzy tám byli / związáne przywiódł do Jeruzálemu / áby byli káráni.
Y ſtáło śię / gdym jecháł y gdym śię przybliżáł do Dámáƺku o południu / że znágłá ogárnęłá mię świátłość wielká z niebá.
Y upádłem ná źiemię / á uſłyƺáłem głoſ mówiący do mnie : Saulu! Saulu! cżemu mię prześládujeƺ?
A jám odpowiedźiáł : Ktoś jeſt / Pánie? Y rzekł do mnie : Jám jeſt JEzuſ Názáreńſki / którego ty prześládujeƺ.
A ći / którzy byli ze mną / ácż widźieli świátłość y polękáli śię / ále głoſu nie ſłyƺeli onego / który ze mną mówił.
Y rzekłem : Cóż ucżynię / Pánie? A Pán rzekł do mnie : Wſtáń ; idź do Dámáƺku / á tám ći powiedzą o wƺyſtkiem / co poſtánowiono / ábyś ty ucżynił.
A gdym nie widźiáł przed jáſnośćią świátłośći oney / będąc prowádzony zá rękę od tych / co ze mną byli / przyƺedłem do Dámáƺku.
Tám niejáki Anánijáƺ / mąż pobożny według zakonu / májąc świádectwo od wƺyſtkich Żydów tám mieƺkájących /
Przyƺedłƺy do mnie y przyſtąpiwƺy / rzekł mi : Saulu bráćie / przejrzyj! A jám tejże godźiny wejrzáł náń.
A on rzekł : Bóg ojców náƺych obráł ćię / ábyś poznał wolę jego / á iżbyś oglądáł onego ſpráwiedliwego / y ſłucháł głoſu z uſt jego.
Abowiem mu będźieƺ świádkiem u wƺyſtkich ludźi / tego / coś widźiáł y ſłyƺáł.
Przetoż teraz cóż odwłácżáƺ? Wſtáń / á ochrzćij śię / á omyj grzechy twoje / wzywájąc imieniá Páńſkiego.
Y ſtáło śię potem / gdym śię wróćił do Jeruzálemu / á modliłem śię w kośćiele / żem był w záchwyceniu.
Y widźiáłem go mówiącego do śiebie : Spieƺ śię ; á wynijdź rychło z Jeruzálemu / poniewáż świádectwá twego nie przyjmą o mnie.
A jám rzekł : Pánie! onić wiedzą / żemći ja podáwáł do więźieniá / y bijáł w bóżnicách te / którzy wierzyli w ćię.
Y gdy wylewáno krew Szcżepáná / świádká twojego / jám też przy tem ſtáł / y zezwáláłem ná zábićie jego / y ſtrzegłem ƺát tych / którzy go zábijáli.
Y rzekł do mnie : Idźże ; boć ja ćię do pogán dáleko poślę.
A ſłucháli go áż do tego ſłowá / y podnieśli głoſ ſwój / mówiąc : Zgłádź z źiemi tákiego ; bo nie ſłuƺná / áby miał żyć.
A gdy oni wołáli / y miotáli ƺáty / y ćiſkáli proch ná powietrze /
Rozkázáł go hetmán wieść do obozu / y kázáł go bicżámi ſpróbowáć / żeby śię dowiedźiáł / dla którejby przycżyny náń ták wołáno.
A gdy go rozćiągniono / áby go bicżámi bito / rzekł Páweł do ſetniká / który tuż ſtáł : Izáli śię wám godźi cżłowieká Rzymiániná nieoſądzonego bicżámi bić?
Co uſłyƺáwƺy ſetnik / przyſtąpiwƺy do hetmáná / powiedźiáł mu / mówiąc : Pátrz / co cżyniƺ ; boć ten cżłowiek jeſt Rzymiáninem.
A przyſtąpiwƺy hetmán / rzekł mu : Powiedz mi / jeſliś ty jeſt Rzymiáninem? A on rzekł : Ták jeſt.
Y odpowiedźiáł hetmán : Jám zá wielką ſummę tego miejſkiego práwá doſtáł. A Páweł rzekł : A jám śię Rzymiáninem y urodźił.
A wnetże odſtąpili od niego ći / którzy go mieli wźiąć ná próby. Do tego y hetmán śię báł / dowiedźiáwƺy śię / że był Rzymiáninem / á iż go był kázáł związáć.
A ták názájutrz chcąc śię pewnie dowiedźieć tego / o coby był oſkárżony od Żydów / uwolnił go od onych związek / y rozkázáł śię zejść przedniejƺym kápłánom y wƺyſtkiej rádźie ich / á wywiódłƺy Páwłá / ſtáwił go przed nimi.
