Mężowie bráćiá y ojcowie! ſłuchájćie mojej / którą teráz do wáſ cżynię / obrony.
A gdy uſłyƺeli / iż do nich rzecż cżynił żydowſkim językiem / tem śię bárdziej ućiƺyli. Y rzekł :
Jámći jeſt mąż Żyd / urodzony w Tárśie Cylicyjſkim / lecż wychowány w mieśćie tem u nóg Gámálijelowych / wyćwicżony doſtátecżnie w zákonie ojcżyſtym / gorliwym będąc miłośnikiem Bożym / jáko wy wƺyſcy dziś jeſteśćie.
Którym prześládowáł tę drogę áż ná śmierć / wiążąc y podáwájąc do więzieniá y męże y niewiáſty /
Jáko mi tego y nájwyżƺy kápłán jeſt świádkiem / y wƺyſcy ſtárśi / od których też liſt wziąwƺy do bráći / jecháłem do Dámáƺku / ábym y te / którzy tám byli / związáne przywiódł do Jeruzálemu / áby byli káráni.
Y ſtáło śię / gdym jecháł y gdym śię przybliżáł do Dámáƺku o południu / że znágłá ogárnęłá mię świátłość wielká z niebá.
Y upádłem ná ziemię / á uſłyƺáłem głoſ mówiący do mnie : Saulu! Saulu! cżemu mię prześládujeƺ?
A jám odpowiedziáł : Ktoś jeſt / Pánie? Y rzekł do mnie : Jám jeſt Jezuſ Názáreńſki / którego ty prześládujeƺ.
A ći / którzy byli ze mną / ácż widzieli świátłość y polękáli śię / ále głoſu nie ſłyƺeli onego / który ze mną mówił.
Y rzekłem : Cóż ucżynię / Pánie? A Pán rzekł do mnie : Wſtáń ; idź do Dámáƺku / á tám ći powiedzą o wƺyſtkiem / co poſtánowiono / ábyś ty ucżynił.
A gdym nie widziáł przed jáſnośćią świátłośći onej / będąc prowádzony zá rękę od tych / co ze mną byli / przyƺedłem do Dámáƺku.
Tám niejáki Anánijáƺ / mąż pobożny według zákonu / májąc świádectwo od wƺyſtkich Żydów tám mieƺkájących /
Przyƺedłƺy do mnie y przyſtąpiwƺy / rzekł mi : Saulu bráćie / przejrzyj! A jám tejże godziny wejrzáł náń.
A on rzekł : Bóg ojców náƺych obráł ćię / ábyś poznáł wolę jego / á iżbyś oglądáł onego ſpráwiedliwego / y ſłucháł głoſu z uſt jego.
Abowiem mu będzieƺ świádkiem u wƺyſtkich ludzi / tego / coś widziáł y ſłyƺáł.
Przetoż teráz cóż odwłácżáƺ? Wſtáń / á ochrzćij śię / á omyj grzechy twoje / wzywájąc imieniá Páńſkiego.
Y ſtáło śię potem / gdym śię wróćił do Jeruzálemu / á modliłem śię w kośćiele / żem był w záchwyceniu.
Y widziáłem go mówiącego do śiebie : Spieƺ śię ; á wynijdź rychło z Jeruzálemu / poniewáż świádectwá twego nie przyjmą o mnie.
A jám rzekł : Pánie! onić wiedzą / żemći já podáwáł do więzieniá / y bijáł w bóżnicách te / którzy wierzyli w ćię.
Y gdy wylewáno krew Szcżepáná / świádká twojego / jám też przy tem ſtáł / y zezwáláłem ná zábićie jego / y ſtrzegłem ƺát tych / którzy go zábijáli.
Y rzekł do mnie : Idźże ; boć já ćię do pogán dáleko poślę.
A ſłucháli go áż do tego ſłowá / y podnieśli głoſ ſwój / mówiąc : Zgłádź z ziemi tákiego ; bo nie ſłuƺná / áby miáł żyć.
A gdy oni wołáli / y miotáli ƺáty / y ćiſkáli proch ná powietrze /
Rozkázáł go hetmán wieść do obozu / y kázáł go bicżámi ſpróbowáć / żeby śię dowiedziáł / dlá którejby przycżyny náń ták wołáno.
A gdy go rozćiągniono / áby go bicżámi bito / rzekł Páweł do ſetniká / który tuż ſtáł : Izáli śię wám godzi cżłowieká Rzymiániná nieoſądzonego bicżámi bić?
Co uſłyƺáwƺy ſetnik / przyſtąpiwƺy do hetmáná / powiedziáł mu / mówiąc : Pátrz / co cżyniƺ ; boć ten cżłowiek jeſt Rzymiáninem.
A przyſtąpiwƺy hetmán / rzekł mu : Powiedz mi / jeźliś ty jeſt Rzymiáninem? A on rzekł : Ták jeſt.
Y odpowiedziáł hetmán : Jám zá wielką ſummę tego miejſkiego práwá doſtáł. A Páweł rzekł : A jám śię Rzymiáninem y urodził.
A wnetże odſtąpili od niego ći / którzy go mieli wziąć ná próby. Do tego y hetmán śię báł / dowiedziáwƺy śię / że był Rzymiáninem / á iż go był kázáł związáć.
A ták názájutrz chcąc śię pewnie dowiedzieć tego / o coby był oſkárżony od Żydów / uwolnił go od onych związek / y rozkázáł śię zejść przedniejƺym kápłánom y wƺyſtkiej rádzie ich / á wywiódłƺy Páwłá / ſtáwił go przed nimi.
