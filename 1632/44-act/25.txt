Tedy Feſtuſ wjecháwƺy ná páńſtwo / po trzech dniách przyjecháł do Jeruzálemu z Cezáryi.
Y ſtáwili śię przed nim nájwyżƺy kápłán y przedniejśi z Żydów przećiwko Páwłowi / y prośili go /
Żądájąc łáſki przećiwko niemu / áby go kázáł przywieść do Jeruzálemu / ucżyniwƺy záſádzkę / áby go zábili ná drodze.
Ale Feſtuſ powiedźiáł : Iż Páweł jeſt pod ſtráżą w Cezáryi / á iż ſám tám w rychle pojedźie.
Którzy tedy / mówi / z was mogą / niechże z námi jádą ; á jeſli jeſt jáká winá w tym mężu / niechże náń ſkárżą.
A zámieƺkáwƺy u nich nie więcey tylko dźieśięć dni / jecháł do Cezáryi / á názájutrz uśiádłƺy ná ſądźie / kázáł Páwłá przywieść.
Który gdy przyƺedł / obſtąpili go ći / którzy byli przyƺli z Jeruzálemu Żydowie / przynoƺąc wiele y ćiężkich ſkárg przećiwko Páwłowi / których dowieść nie mogli ;
Gdyż on ſpráwę dáwáł o ſobie : Żem áni przećiwko zakonowi żydowſkiemu / áni przećiwko kośćiołowi / áni przećiwko ceſárzowi nic nie zgrzeƺył.
Ale Feſtuſ chcąc ſobie zjednáć łáſkę u Żydów / odpowiedźiáwƺy Páwłowi / rzekł : Chceƺże iść do Jeruzálemu / á tám o te rzecży ſądzony być przedemną?
Ale Páweł rzekł : Przed ſądem ceſárſkim ſtoję / gdźie mię ſądźić potrzebá : Żydówem w nicżem nie krzywdźił / jáko y ty lepiej wieƺ.
Bo jeſlim w cżem nie práwy y co godnego śmierći ucżynił / nie zbrániám śię umrzeć ; ále jeſli nie máƺ nic tákiego z tych rzecży / o które ná mię ſkárżą / nikt mię im wydáć nie może ; ápeluję do ceſárzá.
Tedy Feſtuſ rozmówiwƺy śię z rádą / odpowiedźiáł : Do ceſárzáś ápelowáł? do ceſárzá pójdźieƺ.
A gdy wyƺło kilká dni / król Agrypá y Bernice przyjecháli do Cezáryi / witáć Feſtá.
A gdy tám nie máło dni zámieƺkáli / Feſtuſ przełożył królowi ſpráwę Páwłową / mówiąc : Mąż niektóry zoſtáwiony jeſt od Felikſá w więźieniu.
Dla którego / gdym był w Jeruzálemie / ſtáwili śię przedemną przedniejśi kápłáni y ſtárśi żydowſcy / proƺąc o dekret przećiwko niemu.
Którymem odpowiedźiáł / że tego nie máją w zwycżáju Rzymiánie / áby którego cżłowieká mieli wydáć ná ſtrácenie / áżby pierwej oſkárżony miał przed ſobą te / co náń ſkárżą / y dánoby mu plác do odpowiedźi ná to / w cżem go obwiniáją.
Gdy śię tedy tu zeƺli / bez wƺelkiej odwłoki názájutrz záśiádłƺy ná ſądźie / kázáłem przywieść tego mężá.
Przećiw któremu ſtánąwƺy ći / co náń ſkárżyli / żádney winy nie przynieśli z tych / którychem śię ja ſpodźiewáł.
Lecż jákieś ſpory o ſwoich zábobonách mieli przećiwko niemu / y o niejákim JEzuśie umárłym / o którym Páweł twierdźił / że żyw jeſt.
Ja tedy wątpiąc o tem / o cżem ten ſpór był / rzekłem : Jeſliby chćiáł iść do Jeruzálemu / á tám o tem być ſądzony?
Lecż iż Páweł ápelowáł / áby záchowány był do Auguſtowego rozeznániá / rozkázáłem go chowáć / áżbym go poſłáł do ceſárzá.
Zátem Agrypá rzekł do Feſtá : Chćiáłbym ja tego cżłowieká ſłyƺeć. A on rzekł : Jutro go uſłyƺyƺ.
Názájutrz tedy / gdy przyƺedł Agrypá y Bernice z wielką okázáłośćią / y weƺli w dom ſądowy z hetmánámi y mężámi przedniejƺymi miáſtá onego / ná rozkázánie Feſtowe przywiedźiono Páwłá.
Y rzekł Feſtuſ : Królu Agrypo / y wƺyſcy mężowie / którzyśćie tu z námi! widźićie tego / o którego mię wƺyſtek lud żydowſki prośił / y w Jeruzálemie y tu wołájąc / że nie ſłuƺná / áby ten dłużej żyć miał.
A ja zrozumiáwƺy / że nie ucżynił nic śmierći godnego / á iż y on ſám ápelowáł do Auguſtá / ucżyniłem dekret / áby był poſłány.
O którym / cobym pánu pewnego piſáć miał / nie mam. Przetoż kázáłem go przed was przywieść / á nájwięcey przed ćię / królu Agrypo! ábym / po rozſądzeniu ſpráwy jego / miał co piſáć.
Bo mi śię nie ſłuƺná widźi / poſłáć więźniá / á tego / o co go obwiniáją / nie oznájmić.
