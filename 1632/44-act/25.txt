Tedy Feſtus wjáchawƺy ná páńſtwo / po trzech dniách przyjáchał do Jeruzalem z Cezáryey.
Y ſtáwili śię przed nim Nawyżƺy Kápłan y przedniejƺy z Żydów przećiwko Páwłowi ; y prośili go /
Żądájąc łáſki przećiwko niemu / áby go kazał przywieść do Jeruzalem : ucżyniwƺy záſadzkę / áby go zábili ná drodze.
Ale Feſtus powiedźiał ; Iż Páweł jeſt pod ſtráżą w Cezáryey / á iż ſam <i>tám</i> w rychle pojedźie.
Którzy tedy / mówi / z was mogą / niechże z námi jádą : <i>á</i> jeſli jeſt jáka <i>winá</i> w tym mężu / niechże nań ſkárżą.
A zmieƺkawƺy u nich <i>nie</i> więcey tylko dźieśięć dni / jáchał do Cezáryey : á názájutrz uśiadƺy ná Sądźie / kazał Páwłá przywieść.
Który gdy przyƺedł / obſtąpili <i>go</i> ći którzy byli przyƺli z Jeruzalem Żydowie / przynoƺąc wiele y ćiężkich ſkarg przećiwko Páwłowi / których dowieść nie mogli.
Gdyż on ſpráwę dawał <i>o ſobie,</i> Żem áni przećiwko Zakonowi Żydowſkiemu / áni przećiwko kośćiołowi / áni przećiwko Ceſárzowi nic nie zgrzeƺył.
Ale Feſtus chcąc ſobie zjednáć łáſkę u Żydów / odpowiedźiawƺy Páwłowi / rzekł : Chceƺże iść do Jeruzalem / á tám o te rzecży ſądzony być przede mną?
Ale Páweł rzekł : Przed Sądem Ceſárſkim ſtoję / gdźie mię ſądźić potrzebá : Żydówem ni w cżym nie ukrzywdźił / jáko y ty lepiey wieƺ.
Bo jeſlim <i>w cżym</i> niepraw / y co godne° śmierći ucżynił / nie zbraniam śię umrzeć. Ale jeſli niemáƺ nic tákiego z tych rzecży o które ná mię ſkárżą ; nikt mię im wydáć nie może : Appelluję do Ceſárzá.
Tedy Feſtus rozmówiwƺy śię z Rádą / odpowiedźiał : Do Ceſárzáś áppellował? Do Ceſárzá pójdźieƺ.
A gdy wyƺło kilká dni / Król Agryppá y Bernice przyjácháli do Cezáryey / witáć Feſtá.
A gdy tám niemáło dni zmieƺkáli / Feſtus przełożył Królowi ſpráwę Páwłowę / mówiąc : Mąż niektóry zoſtáwiony jeſt od Felixá w więźieniu.
Dla którego gdym był w Jeruzalem / ſtáwili śię przede mną Przedniejƺy Kápłani y Stárƺy Żydowſcy proƺąc o dekret przećiwko niemu.
Którymem odpowiedźiał / że tego nie máją w zwycżáju Rzymiánie / áby którego cżłowieká mieli wydáć ná ſtrácenie / áżby pierwey oſkárżony / miał przed ſobą te co nań ſkárżą ; y danoby mu plác do odpowiedźi ná to w cżym go winują.
Gdy śię tedy tu zeƺli / bez wƺelkiey odwłoki názájutrz záśiadƺy ná Sądźie / kazałem przywieść tego mężá.
Przećiw któremu ſtánąwƺy ći co <i>nań</i> ſkárżyli / żadney winy nie przynieśli z tych którychem śię ja ſpodźiewał.
Lecż jákieś ſpory o ſwojich Zaboboniech mieli przećiwko niemu / y o niejákim JEzuśie umárłym / o którym Páweł twierdźił że żywie.
Ja tedy wątpiąc o tym <i>o cżym</i> ten ſpór był / rzekłem ; Jeſliby chćiał iść do Jeruzalem / á tám o tym bydź ſądzony.
Lecż iż Páweł áppellował / áby záchowány był do Auguſtowego rozeznánia / rozkazałem go chowáć áżbym go poſłał do Ceſárzá.
Zátym Agryppá rzekł do Feſtá : Chćiałbym ja tego cżłowieká ſłyƺeć. A on rzekł : Jutro go uſłyƺyƺ.
Názájutrz tedy / gdy przyƺedł Agryppá y Bernice z wielką okazáłośćią / y weƺli w dom Sądowy / z Hetmánámi y mężámi przedniejƺymi miáſtá onego ; ná rozkazánie Feſtowe przywiedźiono Páwłá.
Y rzekł Feſtus : Królu Agryppo / y wƺyſcy mężowie którzyśćie tu z námi / widźićie tego o którego mię wƺyſtek lud Żydowſki prośił / y w Jeruzalem y tu wołájąc / Że nie ſłuƺna áby ten dłużey żyć miał.
A ja zrozumiawƺy że nie ucżynił nic śmierći godnego / á iż y on ſam áppellował do Auguſtá / ucżyniłem dekret áby był poſłány.
O którym cobym Pánu pewnego piſáć miał / nie mam. Przetoż kazałem go przed was przywieść / á nawięcey przed ćię Królu Agryppo / ábym po rozſądzeniu ſpráwy <i>jego</i> miał co piſáć.
Bo mi śię nie ſłuƺna widźi / poſłáć więźniá ; á tego / o co go winują / nie oznájmić.
