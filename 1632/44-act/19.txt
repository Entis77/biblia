Y ſtáło śię / gdy Apolloſ był w Korynćie / iż Páweł obƺedłƺy górne kráiny / przyƺedł do Efezu ; á ználázłƺy tám niektórych ucżniów /
Rzekł do nich : Izáliśćie wźięli Duchá Świętego / uwierzywƺy? A oni mu rzekli : Owƺemeśmy áni ſłyƺeli / jeźli jeſt Duch Święty.
Tedy rzekł do nich : W cóżeśćie tedy ochrzcżeni? A oni rzekli : W chrzeſt Jánowy.
Zátem rzekł Páweł : Jánći chrzćił chrztem pokuty / mówiąc ludowi / áby w onego / który miáł przyjść po nim / uwierzyli / to jeſt / w JEzuſá CHryſtuſá.
A uſłyƺáwƺy to / ochrzcżeni ſą w imię Páná JEzuſowe.
A gdy ná nie włożył Páweł ręce / zſtąpił ná nie Duch Święty / y mówili językámi y prorokowáli.
A było wƺyſtkich mężów około dwunáſtu.
A wƺedłƺy do bóżnicy / mówił bezpiecżnie przez trzy mieśiące / náucżájąc y námáwiájąc ich do króleſtwá Bożego.
A gdy śię niektórzy zátwárdźili á wierzyć nie chćieli / źle mówiąc o tej drodze Bożej przed mnóſtwem / odſtąpiwƺy od nich / odłącżył ucżnie / ná káżdy dźień ucżąc w ƺkole niektórego Tyránná.
A to śię dźiáło przez dwá látá / ták iż wƺyſcy / którzy mieƺkáli w Azyi / ſłucháli ſłowá Páná JEzuſowego / ták Żydowie / jáko y Grekowie.
A nie ládá cudá cżynił Bóg przez ręce Páwłowe ;
Ták iż ná chore przynoƺono chuſtki / álbo przepáſki od ćiáłá jego / y odchodźiły od nich choroby / y duchowie źli wychodźili z nich.
Tedy niektórzy z biegunów żydowſkich / którzy śię báwili záklinániem / wáżyli śię wzywáć imieniá Páná JEzuſowego nád tymi / którzy mieli duchy złe / mówiąc : Poprzyśięgámy wáſ przez JEzuſá / którego Páweł opowiádá.
A było ich śiedm ſynów jednego Żydá / imieniem Scewy / nájwyżƺego kápłáná / którzy to cżynili.
Tedy odpowiedźiáwƺy duch zły / rzekł : Znám JEzuſá / y wiem co Páweł ; ále wy cośćie zácż?
A rzućiwƺy śię ná nie cżłowiek on / w którym był duch zły / á opánowáwƺy je / zmocnił śię przećiwko nim / ták iż nádzy y zránieni wybiegli z onego domu.
Y było to wiádomo wƺyſtkim y Żydom y Grekom / którzy mieƺkáli w Efeźie ; y przypádł ſtrách ná nie wƺyſtkie / y było uwielbione imię Páná JEzuſowe.
A wiele tych / którzy uwierzyli / przychodźiło / wyznáwájąc y oznájmując ſpráwy ſwoje.
Y wiele z tych / którzy śię náukámi niepotrzebnemi páráli / znióſłƺy kśięgi / ſpálili je przed wƺyſtkimi / á obráchowáwƺy cenę ich / ználeźli tego pięćdźieśiąt tyśięcy ſrebrników.
Ták potężnie roſło ſłowo Páńſkie y zmácniáło śię.
A gdy śię to dokonáło / poſtánowił Páweł w duchu / áby przeƺedłƺy Mácedoniję y Acháję / ƺedł do Jeruzálemu / mówiąc : Iż potem / gdy tám będę / muƺę y Rzym widźieć.
A poſłáwƺy do Mácedonii dwóch z tych / którzy mu ſłużyli / Tymoteuƺá y Eráſtá / ſám do cżáſu zoſtáł w Azyi.
A pod on cżáſ ſtáł śię rozruch nie máły około drogi Bożej.
Abowiem niektóry złotnik / imieniem Demetryjuƺ / który robił kośćioły ſrebrne Dyjány / niemáły zyſk przywodźił rzemieślnikom ;
Które zgromádźiwƺy / y inne / którzy tákież rzemioſło robili / rzekł : Mężowie! wiećie / iż z tego rzemioſłá mámy doſtátki náƺe.
A widźićie y ſłyƺyćie / że nie tylko w Efeźie / ále máło nie po wƺyſtkiej Azyi ten Páweł námówił y odwróćił wielki lud / mówiąc : Że to nie ſą bogowie / którzy ſą rękámi ucżynieni.
Przetoż nám śię obáwiáć potrzebá / áby nie tylko rzemioſło náƺe w lekkie powáżenie nie przyƺło / ále áby y kośćiół wielkiej bogini Dyjány zá nic nie był pocżytány / á żeby nie przyƺło do ſkázy doſtojeńſtwo jej / którą wƺyſtká Azyjá y wƺyſtek świát chwáli.
A ſłuchájąc tego / y będąc pełni gniewu / krzyknęli / mówiąc : Wielká jeſt Dyjáná Efeſká!
Y było pełno po wƺyſtkiem mieśćie zámięƺániá / y wpádli jednomyślnie ná plác / porwáwƺy Gájá y Aryſtárchá / Mácedońcżyki / podróżne towárzyƺe Páwłowe.
A gdy Páweł chćiáł wnijść do poſpólſtwá / nie dopuśćili mu ucżniowie.
A niektórzy też z przedniejƺych mężów Azyjáckich / będąc mu przyjáćiołmi / poſłáwƺy do niego / prośili go / áby nie wychodźił ná plác.
Tedy jedni ták á drudzy inácżej wołáli ; ábowiem oná gromádá byłá zámięƺáná / á więcej ich nie wiedźiáło / dlácżego śię zbieżeli.
A z onej zgrái wywlekli Alekſándrá / którego popycháli Żydowie ; á Alekſánder ſkinąwƺy ręką / chćiáł dáć ſpráwę ludowi.
Ale gdy poználi / iż był Żydem / wƺcżął śię jednoſtájny głoſ od wƺyſtkich / jákoby przez dwie godźiny wołájących : Wielká jeſt Dyjáná Efeſká!
Tedy piſárz uśmierzywƺy onę zgráję / rzekł : Mężowie Efeſcy! y któryż jeſt cżłowiek / coby nie wiedźiáł / iż miáſto Efeſkie opiekuje śię kośćiołem wielkiej boginii Dyjány / y obrázem / który ſpádł od Jowiƺá?
A poniewáż śię temu nikt ſprzećiwić nie może / ſłuƺná / ábyśćie śię uſpokoili á nic ſkwápliwie nie cżynili.
Abowiemeśćie przywiedli tych mężów / którzy nie ſą áni świętokrádzcámi / áni bluźniercámi boginii wáƺej.
A jeźliż Demetryjuƺ y ći / którzy z nim ſą rzemieślnicy / máją co przećiw komu / wƺák bywá práwo / ſą też ſtároſtowie / niechże jedni drugich pozywáją.
Jeźli śię też o cżem inƺem pytáćie to śię może w porządnem zgromádzeniu odpráwić.
Abowiem trzebá śię obáwiáć / ábyśmy oſkárżeni nie byli o rozruch dźiśiejƺy / gdyż nie máƺ żádnej przycżyny / z którejbyśmy mogli dáć ſpráwę / żeśmy śię tu zbiegli. A to powiedźiáwƺy / rozpuśćił ono zgromádzenie.
