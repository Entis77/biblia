Y ſtáło śię gdy Apollos był w Korynćie / iż Páweł obƺedƺy górne krájiny przyƺedł do Efezu : A ználazƺy tám niektóre ucżnie /
Rzekł do nich : Izaliśćie wźięli Duchá Świętego uwierzywƺy? A oni mu rzekli : Owƺemeſmy áni ſłycháli jeſli jeſt Duch Święty.
Tedy rzekł do nich : W cóżeśćie tedy ochrzcżeni? A oni rzekli / W chrzeſt Janów.
Zátym rzekł Páweł : Janći chrzćił chrztem pokuty / mówiąc ludowi / áby w onego / który miał przyść po nim / uwierzyli / to jeſt / w JEzuſá CHryſtuſá.
A uſłyƺawƺy <i>to,</i> ochrzcżeni ſą w Imię PAná JEzuſowe.
A gdy ná nie włożył Páweł ręce / zſtąpił ná nie Duch Święty ; y mówili językámi y prorokowáli.
A było wƺyſtkich mężów około dwunaſtu.
A wƺedƺy do bóżnice / mówił beſpiecżnie przez trzy mieśiące / náucżájąc y námawiájąc ich do króleſtwá Bożego.
A gdy śię niektórzy zátwárdźili / á wierzyć nie chćieli / źle mówiąc o tey drodze <i>Bożey</i> przed mnóſtwem ; odſtąpiwƺy od nich odłącżył ucżnie / ná káżdy dźień ucżąc w ƺkole niektórego Tyránná.
A to śię dźiało przez dwie lećie : ták yż wƺyſcy którzy mieƺkáli w Azyey ſłucháli ſłowá PAná JEzuſowego / ták Żydowie jáko y Grekowie.
A nie ládá cudá cżynił Bóg przez ręce Páwłowe ;
Ták iż ná chore przynoƺono chuſtki álbo przepaſki od ćiáłá jego / y odchodźiły od nich choroby / y duchowie źli wychodźili z nich.
Tedy niektórzy z biegunów Żydowſkich / którzy śię báwili záklinániem / ważyli śię wzywáć Imienia PAná JEzuſowego nád tymi / którzy mieli duchy złe / mówiąc : Poprzyśięgamy was przez JEzuſá / którego Páweł opowiáda.
A było ich śiedm ſynów jednego Żydá <i>imieniem</i> Scewy Nawyżƺego kápłaná / którzy to cżynili.
Tedy odpowiedźiawƺy duch zły rzekł : Znam JEzuſá / y wiem co Páweł ; ále wy cośćie zacż?
A rzućiwƺy śię ná nie cżłowiek on w którym był duch zły / á opánowawƺy je / zmocnił śię przećiwko nim ; ták / iż nádzy y zránieni wybiegli z onego domu.
Y było to wiádomo wƺyſtkim y Żydom y Grekom / którzy mieƺkáli w Efeźie : y przypadł ſtrách ná nie wƺyſtkie / y było uwielbione Imię PAná JEzuſowe.
A wiele tych którzy uwierzyli przychodźiło / wyznawájąc y oznájmując ſpráwy ſwoje.
Y wiele z tych / którzy śię náukámi niepotrzebnymi paráli / zniózƺy kśięgi / ſpalili je przed wƺyſtkimi : A obráchowawƺy cenę ich / ználeſli <i>tego</i> pięćdźieśiąt tyśięcy śrebrników.
Ták potężnie roſło ſłowo PAńſkie / y zmacniáło śię.
A gdy śię to dokonáło / poſtánowił Páweł w duchu / áby przeƺedƺy Mácedonią y Acháią / ƺedł do Jeruzalem mówiąc : Iż potym gdy tám będę / muƺę y Rzym widźieć.
A poſławƺy do Mácedoniey dwu z tych którzy <i>mu</i> ſłużyli / Tymoteuƺá y Eráſtá ; ſam do cżáſu zoſtał w Azyey.
A pod on cżás ſtał śię rozruch niemáły około drogi <i>Bożey</i>.
Abowiem niektóry złotnik imieniem Demetryuƺ / który robił kośćioły śrebrne Dyány / niemáły zyſk przywodźił rzemieśnikom :
Które zgromádźiwƺy / y inne którzy tákież rzemieſło robili / rzekł : Mężowie / wiećie iż z tego rzemieſłá mamy doſtátki náƺe.
A widźićie y ſłyƺyćie / że nie tylko w Efeźie / ále máło nie po wƺyſtkiey Azyey ten Páweł námówił y odwróćił wielki lud mówiąc : Że <i>to</i> nie ſą bogowie / którzy ſą rękámi ucżynieni.
Przetoż nam śię obawiáć potrzebá / áby nie tylko rzemioſło náƺe w lekkie poważenie nie przyƺło / ále áby y kośćiół wielkiey bogini Dyány nizacż nie był pocżytány ; á żeby nie przyƺło do ſkázy doſtojeńſtwo jey / którą wƺyſtká Azya y wƺyſtek świát chwali.
A ſłuchájąc <i>tego</i> y będąc pełni gniewu / krzyknęli mówiąc ; Wielka jeſt Dyáná Efeſka.
Y było pełno po wƺyſtkim mieśćie zámięƺánia ; y wpádli jednomyślnie ná plác / porwawƺy Gájá / y Aryſtárchá Mácedońcżyki / podróżne towárzyƺe Páwłowe.
A gdy Páweł chćiał wyniść do poſpólſtwá / nie dopuśćili mu ucżniowie.
A niektórzy też z przedniejƺych mężów Azyjſkych / będąc mu przyjáćielmi / poſławƺy do niego / prośili go áby nie wychodźił ná plác.
Tedy jedni ták / á drudzy inácżey wołáli. Abowiem oná gromádá byłá zámięƺána ; á więcey ich nie wiedźiáło / dla cżego śię zbieżeli.
A z oney zgráje wywlekli Alexándrá / którego popycháli Żydowie. A Alexánder ſkinąwƺy ręką / chćiał dáć ſpráwę ludowi.
Ale gdy poználi iż był Żydem / wƺcżął śię jednoſtájny głos od wƺyſtkich / jákoby przez dwie godźinie wołájących : Wielka jeſt Dyáná Efeſka.
Tedy Piſarz uśmierzywƺy onę zgráję / rzekł : Mężowie Efeſcy / y któryż jeſt cżłowiek coby nie wiedźiał / iż miáſto Efeſkie opieka śię kośćiołem wielkiey bogini Dyány / y <i>obrázem</i> który zpadł od Jowiƺá.
A ponieważ śię temu nikt zprzećiwić nie może / ſłuƺna ábyśćie śię uſpokojili / á nic ſkwápliwie nie cżynili.
Abowiemeśćie przywiedli te męże / którzy nie ſą áni świętokrádźcámi / áni bluźniercámi bogini wáƺey.
A jeſliż Demetryuƺ y ći którzy z nim ſą rzemieśnicy / máją co przećiw komu / <i>wƺák</i> bywa práwo / ſą też Stároſtowie ; niechże jedni drugich pozywáją.
Jeſli śię też o cżym inƺym pytaćie to śię <i>może</i> w porządnym zgromádzeniu odpráwić.
Abowiem trzebá śię obawiáć / ábyſmy oſkárżeni nie byli o rozruch dźiśiejƺy : gdyż niemáƺ żadney przycżyny / z któreybyſmy mogli dáć ſpráwę / żeſmy śię tu zbiegli. A to powiedźiawƺy / rozpuśćił ono zgromádzenie.
