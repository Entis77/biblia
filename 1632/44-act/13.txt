A byli w Antyjochii we zborze / który tám był / niektórzy prorocy y náucżyćiele / jáko Bárnábáƺ y Symeon / którego zwáno Niger / y Lucyjuſ Cyrenejcżyk / y Mánáhen / który był wychowány z Herodem Tetrárchą / y Saul.
A gdy oni ſłużbę Páńſką jáwnie odpráwiáli y pośćili / rzekł im Duch Święty : Odłącżćie mi Bárnábáƺá y Saulá do tej ſpráwy / do którejm ich powołáł.
Tedy poƺcżąc y modląc śię / y wkłádájąc ná nie ręce / odpráwili je.
Oni tedy wyſłáni będąc od Duchá Świętego / przyƺli do Seleucyi / á ſtámtąd płynęli do Cypru.
A gdy byli w Sáláminie / opowiádáli ſłowo Boże w bóżnicách żydowſkich / á mieli z ſobą y Jáná do uſługi.
A przeƺedłƺy onę wyſpę áż do Páfu / ználeźli tám jákiegoś cżárnokśiężniká / fáłƺywego proroká / Żydá / któremu imię było Bárjezuſ.
Który był przy zácnym ſtárośćie / Sergijuƺu Páwle / mężu roztropnym. Ten przyzwáwƺy Bárnábáƺá y Saulá / prágnął ſłucháć ſłowá Bożego.
Lecż śię im ſprzećiwił Elimáſ / on cżárnokśiężnik / ( ábowiem śię ták wykłádá imię jego / ) ſtárájąc śię / jákoby ſtároſtę od wiáry odwróćił.
Tedy Saul / ( którego zowią y Páwłem / ) nápełniony będąc Duchá Świętego / á pilnie náń pátrząc /
Rzekł : O pełny wƺelkiej zdrády y wƺelkiej przewrotnośći / ſynu dyjábelſki / nieprzyjáćielu wƺelkiej ſpráwiedliwośći! nie przeſtánieƺże podwrácáć proſtych dróg Páńſkich?
A oto teráz ręká Páńſká nád tobą : y będzieƺ ślepym / nie widząc ſłońcá áż do cżáſu. A zárázem przypádłá náń chmurá y ćiemność / á błąkájąc śię / ƺukáł / ktoby go wiódł zá rękę.
Tedy widząc ſtároſtá / co śię ſtáło / uwierzył / zdumiewájąc śię nád náuką Páńſką.
A puśćiwƺy śię z Páfu Páweł y ći / którzy z nim byli / przyƺli do Pergi Pámfiliejſkiej. A Ján odƺedłƺy od nich / wróćił śię do Jeruzálemu.
A oni odƺedłƺy z Pergi / przyƺli do Antyjochii Piſydejſkiej / á wƺedłƺy do bóżnicy w dzień ſobotni / uśiedli.
A po przecżytániu zákonu y proroków / poſłáli do nich przełożeni bóżnicy / mówiąc : Mężowie bráćiá! máćieli wolę jákie nápominánie ucżynić do ludu / mówćie.
Tedy powſtáwƺy Páweł / á ręką ſkinąwƺy rzekł : Mężowie Izráelſcy / y którzy śię boićie Bogá! ſłuchájćie :
Bóg ludu tego Izráelſkiego wybráł ojców náƺych / y wywyżƺył lud / gdy byli przychodniámi w ziemi egipſkiej / y w rámieniu wyćiągnionem wywiódł je z niej.
Y przez cżáſ cżterdzieſtu lát znośił obycżáje ich ná puƺcży.
A wygłádziwƺy śiedm národów w ziemi Chánánejſkiej / loſem rozdzielił między nie onę ziemię ich.
A potem około cżteryſtá y pięćdzieśiąt lát dáwáł im ſędziów / áż do Sámuelá proroká.
A od onego cżáſu prośili o królá. Y dáł im Bóg Saulá / ſyná Cyſowego / mężá z pokoleniá Benjáminowego / przez lát cżterdzieśći.
A gdy go odrzućił / wzbudził im Dawidá zá królá / któremu też świádectwo wydáwájąc powiedziáł : Ználázłem Dawidá / ſyná Jeſſego / mężá według ſercá mego / który będzie cżynił wƺyſtkę wolę moję.
Z jegoż náśieniá Bóg według obietnicy wzbudził Izráelowi zbáwićielá Jezuſá.
Przed którego przyjśćiem kázáł Ján chrzeſt pokuty wƺyſtkiemu ludowi Izráelſkiemu.
A gdy Ján dokonáł biegu ſwego / rzekł : Kim mię być mniemáćie? Nie jeſtem já / ále oto idzie zá mną / u którego nóg obuwiá nie jeſtem godzien rozwiązáć.
Mężowie bráćiá / ſynowie národu Abráhámowego / y którzy śię między wámi Bogá boją! wámći ſłowo zbáwieniá tego poſłáne jeſt.
Abowiem ći / co mieƺkáją w Jeruzálemie / y przełożeni ich / nie znájąc tego Jezuſá y głoſów prorockich / które przez káżdy ſábát bywáją cżytáne ; wypełnili je / oſądziwƺy go.
A żádnej przycżyny śmierći w nim nie ználázłƺy / prośili Piłátá / áby był zábity.
A gdy wykonáli wƺyſtko / co o nim było nápiſáne / zdjąwƺy go z drzewá / włożyli go do grobu.
Ale go Bóg wzbudził od umárłych.
Który widziány jeſt przez wiele dni od tych / którzy z nim poſpołu przyƺli z Gálilei do Jeruzálemu / którzy ſą świádkámi jego przed ludem.
Y my wám opowiádámy tę obietnicę / którá śię ojcom ſtáłá / iż ją Bóg wypełnił nám / dziátkom ich / wzbudziwƺy Jezuſá.
Jáko też w Pſálmie wtórym nápiſáne jeſt : Syn mój jeſteś ty / jám ćiebie dziś ſpłodził.
A iż go wzbudził od umárłych / áby śię więcej nie wróćił do ſkáżeniá / ták powiedziáł : Dám wám święte dobrodziejſtwá Dawidowe wierne.
Przeto y indziej powiádá : Nie dáƺ Świętemu twemu widzieć ſkáżeniá.
Abowiemći Dawid zá wieku ſwego uſłużywƺy woli Bożej / záſnął y przyłącżony jeſt do ojców ſwoich / á widziáł ſkáżenie.
Lecż ten / którego Bóg wzbudził / nie widziáł ſkáżeniá.
Niechże wám tedy będzie wiádomo / mężowie bráćiá / iż śię wám przez tego opowiádá odpuƺcżenie grzechów :
Y od wƺyſtkiego od cżegośćie nie mogli być przez zákon Mojżeƺowy uſpráwiedliwieni / przez tego káżdy wierzący uſpráwiedliwiony bywá.
A przetoż pátrzćie / áby ná wáſ nie przyƺło to / co powiedziáno w prorokách :
Obácżćie wy wzgárdzićiele / y dziwujćie śię / á wniwecż śię obróććie ; bo já ſpráwuję ſpráwę zá dni wáƺych / ſpráwę / której nie wierzyćie / choćby wám kto o niej powiádáł.
A gdy oni wychodzili z bóżnicy żydowſkiej / prośili ich pogánie / áby y w drugi ſábát mówili do nich też ſłowá.
A po rozpuƺcżeniu zgromádzeniá / poƺło wiele Żydów / y nábożnych nowowierników zá Páwłem y Bárnábáƺem / którzy mówiąc do nich / rádzili im / áby trwáli w łáſce Bożej.
A w drugi ſábát / niemál wƺyſtko miáſto śię zgromádziło ná ſłuchánie ſłowá Bożego.
Tedy Żydowie widząc lud / nápełnieni ſą zázdrośćią / y ſprzećiwiáli śię temu / co Páweł powiádáł / mówiąc przećiwko temu y bluźniąc.
A Páweł y Bárnábáƺ / bezpiecżnie mówiąc / rzekli : Wámći nájpierwej miáło być opowiádáne ſłowo Boże ; ále poniewáż je odrzucáćie / á ſądzićie śię być niegodnymi żywotá wiecżnego / oto śię obrácámy do pogán.
Abowiem nám ták rozkázáł Pán / mówiąc : Położyłem ćię świátłośćią pogánom / ábyś był zbáwieniem áż do krájów ziemi.
A ſłyƺąc to pogánie / rádowáli śię / y wielbili ſłowo Páńſkie / y uwierzyli / ile ich kolwiek było ſporządzonych do żywotá wiecżnego.
Y roznośiło śię ſłowo Páńſkie po wƺyſtkiej onej kráinie.
A Żydowie poduƺcżáli niewiáſty nábożne y ucżćiwe / y przedniejƺe w mieśćie ; á wzbudzili prześládowánie przećiwko Páwłowi y przećiwko Bárnábáƺowi / y wygnáli je z gránic ſwoich.
A oni otrząſnąwƺy proch z nóg ſwoich ná nie / przyƺli do Ikonii.
A ucżniowie byli nápełnieni rádośći y Duchá Świętego.
