A byli w Antyjochii we zborze / który tám był / niektórzy prorocy y náucżyćiele / jáko Bárnábáƺ y Symeon / którego zwáno Niger / y Lucyjuſ Cyrenejcżyk / y Mánáhen / który był wychowány z Herodem Tetrárchą / y Saul.
A gdy oni ſłużbę Páńſką jáwnie odpráwiáli y pośćili / rzekł im Duch Święty : Odłącżćie mi Bárnábáƺá y Saulá do tey ſpráwy / do którejm ich powołáł.
Tedy poƺcżąc y modląc śię / y wkłádájąc ná nie ręce / odpráwili je.
Oni tedy wyſłáni będąc od Duchá Świętego / przyƺli do Seleucyi / á ſtámtąd płynęli do Cypru.
A gdy byli w Sáláminie / opowiádáli ſłowo Boże w bóżnicách żydowſkich / á mieli z ſobą y Jáná do uſługi.
A przeƺedłƺy onę wyſpę áż do Páfu / ználeſli tám jákiegoś cżárnokśiężniká / fáłƺywego proroká / Żydá / któremu imię było Bárjezuſ.
Który był przy zácnym ſtárośćie / Sergijuƺu Páwle / mężu roztropnym. Ten przyzwáwƺy Bárnábáƺá y Saulá / prágnął ſłucháć ſłowá Bożego.
Lecż śię im ſprzećiwił Elimáſ / on cżárnokśiężnik / ( ábowiem śię ták wykłádá imię jego / ) ſtárájąc śię / jákoby ſtároſtę od wiáry odwróćił.
Tedy Saul / ( którego zowią y Páwłem / ) nápełniony będąc Duchá Świętego / á pilnie náń pátrząc /
Rzekł : O pełny wƺelkiej zdrády y wƺelkiej przewrotnośći / ſynu dyjábelſki / nieprzyjáćielu wƺelkiej ſpráwiedliwośći! nie przeſtánieƺże podwrácáć proſtych dróg Páńſkich?
A oto teraz ręká Páńſká nád tobą : y będźieƺ ślepym / nie widząc ſłońcá áż do cżáſu. A zárázem przypádłá náń chmurá y ćiemność / á błąkájąc śię / ƺukáł / ktoby go wiódł zá rękę.
Tedy widząc ſtároſtá / co śię ſtáło / uwierzył / zdumiewájąc śię nád náuką Páńſką.
A puśćiwƺy śię z Páfu Páweł y ći / którzy z nim byli / przyƺli do Pergi Pámfiliejſkiej. A Ján odƺedłƺy od nich / wróćił śię do Jeruzálemu.
A oni odƺedłƺy z Pergi / przyƺli do Antyjochii Piſydejſkiej / á wƺedłƺy do bóżnicy w dźień ſobotni / uśiedli.
A po przecżytániu zakonu y proroków / poſłáli do nich przełożeni bóżnicy / mówiąc : Mężowie bráćia! máćieli wolę jákie nápominánie ucżynić do ludu / mówćie.
Tedy powſtáwƺy Páweł / á ręką ſkinąwƺy rzekł : Mężowie Izráelſcy / y którzy śię boićie Bogá! ſłuchájćie :
Bóg ludu tego Izráelſkiego wybráł ojców náƺych / y wywyżƺył lud / gdy byli przychodniámi w źiemi egipſkiej / y w rámieniu wyćiągnionem wywiódł je z niej.
Y przez cżáſ cżterdźieſtu lát znośił obycżáje ich ná puƺcży.
A wygłádźiwƺy śiedm národów w źiemi Chánánejſkiej / loſem rozdźielił między nie onę źiemię ich.
A potem około cżteryſtá y pięćdźieśiąt lát dáwáł im ſędźiów / áż do Sámuelá proroká.
A od onego cżáſu prośili o królá. Y dáł im Bóg Saulá / ſyná Cyſowego / mężá z pokoleniá Benjáminowego / przez lát cżterdźieśći.
A gdy go odrzućił / wzbudźił im Dawidá zá królá / któremu też świádectwo wydáwájąc powiedźiáł : Ználázłem Dawidá / ſyná Jeſſego / mężá według ſercá mego / który będźie cżynił wƺyſtkę wolę moję.
Z jegoż náśieniá Bóg według obietnicy wzbudźił Izráelowi zbáwićielá JEzuſá.
Przed którego przyjśćiem kázáł Ján chrzeſt pokuty wƺyſtkiemu ludowi Izráelſkiemu.
A gdy Ján dokonáł biegu ſwego / rzekł : Kim mię być mniemáćie? Nie jeſtem ja / ále oto idźie zá mną / u którego nóg obuwiá nie jeſtem godźien rozwiązáć.
Mężowie bráćia / ſynowie národu Abráhámowego / y którzy śię między wámi Bogá boją! wámći ſłowo zbáwieniá tego poſłáne jeſt.
Abowiem ći / co mieƺkáją w Jeruzálemie / y przełożeni ich / nie znájąc tego JEzuſá y głoſów prorockich / które przez káżdy ſábát bywáją cżytáne ; wypełnili je / oſądźiwƺy go.
A żádney przycżyny śmierći w nim nie ználázłƺy / prośili Piłátá / áby był zábity.
A gdy wykonáli wƺyſtko / co o nim było nápiſáne / zdjąwƺy go z drzewá / włożyli go do grobu.
Ale go Bóg wzbudźił od umárłych.
Który widźiány jeſt przez wiele dni od tych / którzy z nim poſpołu przyƺli z Gálilei do Jeruzálemu / którzy ſą świádkámi jego przed ludem.
Y my wám opowiádámy tę obietnicę / która śię ojcom ſtáłá / iż ją Bóg wypełnił nám / dźiátkom ich / wzbudźiwƺy JEzuſá.
Jáko też w Pſálmie wtórym nápiſáne jeſt : Syn mój jeſteś ty / jám ćiebie dźiś ſpłodźił.
A iż go wzbudźił od umárłych / áby śię więcey nie wróćił do ſkáżeniá / ták powiedźiáł : Dám wám święte dobrodźiejſtwá Dawidowe wierne.
Przeto y indźiej powiádá : Nie dáƺ Świętemu twemu widźieć ſkáżeniá.
Abowiemći Dawid zá wieku ſwego uſłużywƺy woli Bożej / záſnął y przyłącżony jeſt do ojców ſwoich / á widźiáł ſkáżenie.
Lecż ten / którego Bóg wzbudźił / nie widźiáł ſkáżeniá.
Niechże wám tedy będźie wiádomo / mężowie bráćia / iż śię wám przez tego opowiádá odpuƺcżenie grzechów :
Y od wƺyſtkiego od cżegośćie nie mogli być przez zakon Mojżeƺowy uſpráwiedliwieni / przez tego káżdy wierzący uſpráwiedliwiony bywá.
A przetoż pátrzćie / áby ná was nie przyƺło to / co powiedźiáno w prorokách :
Obácżćie wy wzgárdźićiele / y dźiwujćie śię / á wniwecż śię obróććie ; bo ja ſpráwuję ſpráwę zá dni wáƺych / ſpráwę / którey nie wierzyćie / choćby wám kto o niej powiádáł.
A gdy oni wychodźili z bóżnicy żydowſkiej / prośili ich pogánie / áby y w drugi ſábát mówili do nich też ſłowá.
A po rozpuƺcżeniu zgromádzeniá / poƺło wiele Żydów / y nábożnych nowowierników zá Páwłem y Bárnábáƺem / którzy mówiąc do nich / rádźili im / áby trwáli w łáſce Bożej.
A w drugi ſábát / niemál wƺyſtko miáſto śię zgromádźiło ná ſłuchánie ſłowá Bożego.
Tedy Żydowie widząc lud / nápełnieni ſą zázdrośćią / y ſprzećiwiáli śię temu / co Páweł powiádáł / mówiąc przećiwko temu y bluźniąc.
A Páweł y Bárnábáƺ / bezpiecżnie mówiąc / rzekli : Wámći nájpierwej miáło być opowiádáne ſłowo Boże ; ále poniewáż je odrzucáćie / á ſądźićie śię być niegodnymi żywotá wiecżnego / oto śię obrácámy do pogán.
Abowiem nám ták rozkázáł Pán / mówiąc : Położyłem ćię świátłośćią pogánom / ábyś był zbáwieniem áż do krájów źiemi.
A ſłyƺąc to pogánie / rádowáli śię / y wielbili ſłowo Páńſkie / y uwierzyli / ile ich kolwiek było ſporządzonych do żywotá wiecżnego.
Y roznośiło śię ſłowo Páńſkie po wƺyſtkiej oney kráinie.
A Żydowie poduƺcżáli niewiáſty nábożne y ucżćiwe / y przedniejƺe w mieśćie ; á wzbudźili prześládowánie przećiwko Páwłowi y przećiwko Bárnábáƺowi / y wygnáli je z gránic ſwoich.
A oni otrząſnąwƺy proch z nóg ſwoich ná nie / przyƺli do Ikonii.
A ucżniowie byli nápełnieni rádośći y Duchá Świętego.
