A gdy ſkázáno / żebyśmy płynęli do Włoch / oddáno y Páwłá / y niektóre inne więźnie ſetnikowi / imieniem Julijuƺowi / roty Auguſtowej.
Tedy wśiádłƺy w okręt Adrámiteńſki / májąc płynąć podle kráin Azyi / puśćiliśmy śię od brzegu / á był z námi Aryſtárchuſ / Mácedońcżyk z Teſáloniki.
A drugiego dniá przypłynęliśmy do Sydonu / kędy Julijuƺ ludzko śię Páwłowi ſtáwiwƺy / pozwolił mu iść do przyjáćiół / áby wcżáſu záżył.
A ſtámtąd śię puśćiwƺy / przypłynęliśmy pod Cypr / dlátego / że były wiátry przećiwne.
A przepłynąwƺy ono morze / które jeſt podle Cylicyi y Pámfilii / przybyliśmy do Miry / miáſtá Licyjſkiego.
A tám ſetnik ználázłƺy okręt Alekſándryjſki / który płynął do Włoch / wſádźił náſ weń.
A gdyśmy przez wiele dni z wolná płynęli / á záledwie przećiwko Knidowi przyjecháli / przeto / że nám wiátr nie dopuƺcżáł / popłynęliśmy pod Kretę podle Sálmonu.
A ledwie ją przeminąwƺy / przyƺliśmy ná miejſce niektóre / które zowią piękne porty / od którego bliſko było miáſto Láſeá.
A gdy cżáſ niemáły wyƺedł / y już było niebezpiecżne żeglowánie / przeto / iż już był y poſt przeminął / nápomináł je Páweł /
Mówiąc do nich : Mężowie! widzę já / iż z ukrzywdzeniem y z wielką ƺkodą nie tylko towárów y okrętu / ále też y duƺ náƺych będźie to żeglowánie.
Jednák ſetnik więcej ufáł ſpráwcy okrętu y ſternikowi / niż temu / co Páweł powiádáł.
A gdy nie było portu ſpoſobnego ku źimowániu / wiele ich rádę dáwáło / puśćić śię ſtámtąd / owáby jákożkolwiek mogli / przepráwiwƺy śię do Fenicyi / przeźimowáć u portu Kreteńſkiego / który leży między wiátrem południowym y záchodnim.
A gdy powionął wiátr z południá / mniemájąc / że ſwego przedśięwźięćiá dopięli / puśćiwƺy śię od brzegu / płynęli bliſko Krety.
Lecż nie długo potem uderzył ná nie wiátr gwáłtowny / który zowią Euroklidon.
A gdy był okręt porwány / á nie mógł śię oprzeć wiátrowi / puśćiwƺy śię płynęliśmy.
A gdyśmy pod niektórą máłą wyſepkę przypłynęli / którą zowią Kláudá / ledwieśmy mogli báćik zátrzymáć.
Który wćiągnąwƺy / rátunku używáli / podpáſáwƺy okręt / á bojąc śię / żeby nie wpádł ná hák / ſpuśćiwƺy żágle / ták płynęli.
A iż námi náwáłnośći bárdzo miotáły / názájutrz towáry wyrzućili.
A trzećiego dniá rękámi náƺemi okrętowe nácżyniá wyrzućiliśmy.
Lecż gdy śię áni ſłońce / áni gwiázdy przez wiele dni nie ukázáły / á náwáłność niemáłá nálegáłá / ná oſtátek odjętá byłá wƺyſtká nádźiejá / żebyśmy byli mogli być záchowáni.
A gdyśmy długo nie jedli / tedy Páweł ſtojąc w pośrodku ich rzekł : Mieliśćie zápráwdę / o mężowie! uſłucháwƺy mię / nie puƺcżáć śię od Krety / á ták ujść tej ſtráty y zguby.
Lecż y teráz nápominám wáſ / ábyśćie byli dobrej myśli ; boć nie zginie z wáſ żádná duƺá / oprócż okrętu.
Abowiem ſtánął przy mnie tej nocy Anioł Bogá tego / któregom já jeſt / y któremu ſłużę ;
Mówiąc : Nie bój śię / Páwle! muśiƺ ſtáwiony być przed ceſárzem / á oto dárowáł ćię Bóg wƺyſtkimi / którzy płyną z tobą.
Przetoż bądźćie dobrej myśli / mężowie! ábowiem wierzę Bogu / że ták będźie / jáko mi powiedźiáno.
A muśimy opáść ná niektórej wyſpie.
A gdy przyƺłá noc cżternáſtá / á myśmy śię błąkáli po morzu Adryjátyckiem około północy / zdáło śię żeglárzom / iż śię im okázywáłá niektórá kráiná.
Tedy ſpuśćiwƺy ƺnur z ołowiem / ználeźli głębiej dwádźieśćiá ſążni ; á málucżko odpłynąwƺy / záśię ſpuśćili ołów / y ználeźli piętnáśćie ſążni.
A bojąc śię / áby ſnáć ná miejſcá oſtre nie wpádli / zrzućiwƺy cżtery kotwice z ſteru / prágnęli / áby dźień był.
A gdy żeglárze myślili z okrętu ućiec / y ſpuśćili báćik ná morze / chcąc wrzkomo od przodku okrętu zárzucáć kotwice /
Rzekł Páweł ſetnikowi y żołnierzom : Jeźli ći nie zoſtáną w okręćie / wy záchowáni być nie możećie.
Tedy żołnierze obćięli powrozy u báćiká / y dopuśćili mu odpáść.
A międzytem niż śię rozedniáło / nápomináł Páweł wƺyſtkie / áby pokárm przyjęli / mówiąc : Dźiś temu cżternáſty dźień / jáko cżekájąc trwáćie bez pokármu / nic nie jedząc.
Dlátego proƺę wáſ / ábyśćie pokárm przyjęli ; bo to ſłuży ku záchowániu wáƺemu / gdyż żádnego z wáſ włoſ z głowy nie ſpádnie.
A to rzekłƺy y chleb wźiąwƺy / podźiękowáł Bogu przed wƺyſtkimi / y złámáwƺy pocżął jeść.
Zátem wƺyſcy będąc lepƺej myśli / y ſámi pokárm przyjmowáli.
A było náſ wƺyſtkich duƺ w okręćie dwieśćie śiedmdźieśiąt y ƺeść.
Będąc tem pokármem náſyceni / ulżenie cżynili okrętowi / wyrzucájąc zboże w morze.
A gdy był dźień / nie poználi źiemi ; wƺákże obácżyli niejáką odnogę májącą brzeg / do którego urádźili / jeźliby mogło być / przybić okręt.
A wyćiągnąwƺy kotwice / puśćili śię ná morze ; á rozpuśćiwƺy záwiáſy ſterowe y podnióſłƺy żágiel po wietrze / mieli śię do brzegu ;
Ale nápádłƺy ná miejſce / które miáło z obu ſtron morze / otrąćili okręt ; á przodek okrętu uwięznąwƺy / zoſtáł / nie ruƺájąc śię / lecż zád rozbijáł śię od gwáłtownych wáłów.
Tedy żołnierze rádźili / áby więźnie pozábijáli / iżby który wypłynąwƺy nie ućiekł.
Ale ſetnik chcąc záchowáć Páwłá / pohámowáł je od tego przedśięwźięćiá / y rozkázáł tym / którzy mogli pływáć / áby śię wprzód w morze puśćili y ná brzeg wyƺli ;
Inni záśię / niektórzy ná deſkách á niektórzy ná ƺtukách okrętu. Y ták śię ſtáło / że wƺyſcy zdrowo wyƺli ná źiemię.
