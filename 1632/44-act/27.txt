A gdy ſkazano / żebyſmy płynęli do Włoch / oddano y Páwłá / y niektóre inne więźnie / Setnikowi / imieniem Juliuƺowi / Roty Auguſtowey.
Tedy wśiadƺy w okręt Adrámiteńſki / májąc płynąć podle krájin Azyey / puśćiliſmy śię <i>od brzegu</i> : á był z námi Aryſtárchus Mácedońcżyk z Teſſáloniki.
A drugiego dniá przypłynęliſmy do Sydonu : kędy Juliuƺ ludzko śię Páwłowi ſtáwiwƺy / pozwolił <i>mu</i> iść do przyjaćiół áby wcżáſu záżył.
A z támtąd śię puśćiwƺy / przypłynęliſmy pod Cypr ; dla tego że były wiátry przećiwne.
A przepłynąwƺy ono morze które jeſt podle Cylicyey y Pámfiliey / przybyliſmy do Miry <i>miáſtá</i> Lycyjſkiego.
A tám Setnik ználazƺy okręt Alexándryjſki / który płynął do Włoch / wſádźił nas weń.
A gdyſmy przez wiele dni z wolná płynęli / y zá ledwie przećiwko Knidowi przyjácháli / przeto że nam wiátr nie dopuƺcżał / popłynęliſmy pod Kretę podle Sálmonu.
A ledwie ją przeminąwƺy / przyƺliſmy ná miejſce niektóre / które zowią Piękne Porty / od którego bliſko było miáſto Láſeá.
A gdy cżás niemáły wyƺedł / y już było niebeſpiecżne żeglowánie / przeto iż już był y poſt przeminął / nápominał <i>je</i> Páweł /
Mówiąc do nich : Mężowie / widzę ja iż z ukrzywdzeniem y z wielką ƺkodą nie tylko towárów y okrętu / ále też y duƺ náƺych będźie to żeglowánie.
Jednák Setnik więcey ufał ſprawcy <i>okrętu</i> y Sternikowi ániż temu co Páweł powiádał.
A gdy nie było portu ſpoſobnego ku źimowániu / wiele ich rádę dawáło puśćić śię z támtąd : owaby jákożkolwiek mogli przepráwiwƺy śię do Fenicyey / przeźimowáć u portu Kreteńſkiego / który leży miedzy wiátrem Południowym y Zachodniowym.
A gdy powionął wiátr z Południá / mniemájąc że ſwego przedśięwźięćia dopięli / puśćiwƺy śię od brzegu / płynęli bliſko Krety.
Lecż nie długo potym uderzył ná nie wiátr gwałtowny / który zowią Euroklydon.
A gdy był okręt porwány / á nie mógł śię oprzeć wiátrowi / puśćiwƺy śię płynęliſmy.
A gdyſmy pod niektóry máły wyſep przypłynęli / którą zowią Klaudá / ledwieſmy mogli báćik zátrzymáć.
Który wćiągnąwƺy / rátunku używáli / podpáſawƺy okręt / á bojąc śię żeby nie wpadł ná hak / ſpuśćiwƺy żagle / ták płynęli.
A iż námi náwáłnośći bárzo miotáły / názájutrz <i>towáry</i> wyrzućili.
A trzećiego <i>dniá</i> rękámi náƺymi okrętowe nacżynia wyrzućiliſmy.
Lecż gdy śię áni ſłońce / áni gwiázdy przez wiele dni nie ukazáły / á náwáłność niemáła nálegáłá / náoſtátek odjęta byłá wƺyſtká nádźiejá / żebyſmy byli mogli bydź záchowáni.
A gdyſmy długo nie jedli / tedy Páweł ſtojąc w pośrzodku ich / rzekł : Mieliśćie záprawdę / o mężowie! uſłuchawƺy mię / niepuƺcżáć śię od Krety / á <i>ták</i> ujść tey ſtráty y zguby.
Lecż y teraz nápominam was / ábyśćie byli dobrey myśli / boć nie zginie z was żadna duƺá / oprócż okrętu.
Abowiem ſtánął przy mnie tey nocy Anjoł Bogá <i>tego</i> któregom ja jeſt / y któremu ſłużę /
Mówiąc : Nie bój śię Páwle / muśiƺ ſtáwiony być przed Ceſárzem : á oto dárował ćię Bóg wƺyſtkimi / którzy płyną z tobą.
Przetoż bądźćie dobrey myśli mężowie. Abowiem wierzę Bogu / że ták będźie jáko mi powiedźiano.
A muśimy opáść ná niektórym wyſpie.
A gdy przyƺłá noc cżternaſta / á myſmy śię błąkáli po morzu Adriátyckim / około północy zdáło śię żeglarzom / iż śię im okázywáłá niektóra krájiná.
Tedy ſpuśćiwƺy ƺnur z ołowem / ználeźli <i>głębiey</i> dwádźieśćiá ſążeń : á málucżko odpłynąwƺy / záśię ſpuśćili ołów / y ználeźli piętnaśćie ſążeń.
A bojąc śię áby ſnadź ná miejſcá oſtre nie wpádli / zrzućiwƺy cżtery kotwice z ſtyru / prágnęli áby dźień był.
A gdy żeglarze myślili z okrętu ućiec / y ſpuśćili baćik ná morze / chcąc rzkomo od przodku okrętu zárzucáć kotwice /
Rzekł Páweł Setnikowi y żołnierzom : Jeſli ći nie zoſtáną w okręćie / wy záchowáni być nie możećie.
Tedy żołnierze obćięli powrozy u baćiká / y dopuśćili mu odpáść.
A miedzy tym niż śię rozedniáło / nápominał Páweł wƺyſtkie áby pokarm przyjęli / mówiąc : Dźiś <i>temu</i> cżternaſty dźień / jáko cżekájąc trwaćie bez pokármu / nic nie jedząc.
Dla tego proƺę was / ábyśćie pokarm przyjęli / bo to ſłuży ku záchowániu wáƺemu : gdyż żadnego z was włos z głowy nie ſpádnie.
A to rzekƺy y chleb wźiąwƺy / podźiękował Bogu przed wƺyſtkimi / y złamawƺy pocżął jeść.
Zátym wƺyſcy będąc lepƺey myśli / y ſámi pokármy przyjmowáli.
A było nas wƺyſtkich duƺ w okręćie dwie śćie śiedmdźieśiąt y ƺeść.
Będąc tedy pokármem náſyceni / ulżenie cżynili okrętowi / wyrzucájąc zboże w morze.
A gdy był dźień / nie poználi źiemie / wƺákże obacżyli niejáką odnogę májącą brzeg : do którego urádźili / jeſliby mogło bydź / przybić okręt.
A wyćiągnąwƺy kotwice / puśćili śię ná morze ; á roſpuśćiwƺy zawiáſy ſtyrowe / y podniózƺy żagiel po wietrze / mieli śię do brzegu.
Ale nápadƺy ná miejſce / które miáło z obu ſtron morze / otrąćili okręt : á przodek okrętu uwięznąwƺy zoſtał nie ruƺájąc śię / lecż zad rozbijał śię od gwałtownych wáłów.
Tedy żołnierze rádźili / áby więźnie pozábijáli / iżby który wypłynąwƺy nie ućiekł.
Ale Setnik chcąc záchowáć Páwłá / pohámował je od tego przedśięwźięćiá ; y rozkazał tym którzy mogli pływáć / áby śię w przód w morze puśćili / y ná brzeg wyƺli.
Inni záśię / niektórzy ná deſkách / á niektórzy ná ƺtukách okrętu. Y ták śię ſtáło / że wƺyſcy zdrowo wyƺli ná źiemię.
