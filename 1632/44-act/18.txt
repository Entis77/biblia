Potem Páweł odƺedłƺy z Aten / przyƺedł do Koryntu ;
A ználázłƺy niektórego Żydá / imieniem Akwilę / rodem z Pontu / który był świeżo z Włoch przyƺedł z Pryſcyllą / żoną ſwą / ( dlá tego iż był Kláudyjuƺ poſtánowił / áby wƺyſcy Żydowie z Rzymu wyƺli / ) przyƺedł do nich ;
A iż był tegoż rzemioſłá / mieƺkáł u nich y robił ; ábowiem rzemioſło ich było / robić námioty.
Tedy miewáł rozmowę w bóżnicy ná káżdy ſábát / y pozyſkiwáł y Żydy y Greki.
A gdy przyƺli z Mácedonii Syláſ y Tymoteuƺ / śćiśniony był w duchu Páweł / oświádcżájąc Żydom / że JEzuſ jeſt CHryſtuſem.
Lecż gdy śię oni ſprzećiwiáli y bluźnili / otrząſnąwƺy proch z ƺát / rzekł do nich : Krew wáƺá ná głowę wáƺę ; jám jeſt cżyſty / od tego cżáſu pójdę do pogán.
A odƺedłƺy ſtámtąd wƺedł do domu niejákiego cżłowieká / imieniem Juſtá / ſłużącego Bogu / którego dom był podle ſámej bóżnicy.
Lecż Kryſpuſ / przełożony bóżnicy / uwierzył Pánu ze wƺyſtkim domem ſwoim / y wiele z Koryntcżyków ſłuchájąc / uwierzyli / y ochrzcżeni ſą.
Zátem Pán rzekł Páwłowi w nocy w widzeniu : Nie bój śię / ále mów / á nie milcż.
Bom já jeſt z tobą / á żáden śię ná ćię nie tárgnie / ábyć miáł co złego ucżynić ; ábowiem já wielki lud mám w tem mieśćie.
Y mieƺkáł tám rok y ƺeść mieśięcy / náucżájąc u nich ſłowá Bożego.
A gdy Gálijo był ſtároſtą w Achái / powſtáli jednomyślnie Żydowie przećiwko Páwłowi / y przywiedli go do ſądu / mówiąc :
Ten námáwiá ludźi / áby przećiwko zákonowi Bogá chwálili.
A gdy Páweł miáł uſtá otworzyć / rzekł Gálijo do Żydów : O Żydowie! gdyby śię wám było jákie bezpráwie ſtáło / álbo jáká krzywdá / ſłuƺniebym wáſ znośił ;
Lecż jeźli jeſt jáká gádká o ſłowách y o imionách y o zákonie wáƺym / ſámi tego pátrzćie ; ábowiem já tego ſędźią być nie chcę.
Y odegnáł je od ſądowej ſtolicy.
Tedy porwáwƺy wƺyſcy Grekowie Soſtená / przełożonego bóżnicy / bili go przed ſądową ſtolicą / á Gálijo ná to nic nie dbáł.
A Páweł pomieƺkáwƺy tám jeƺcże przez nie máło dni / pożegnáwƺy śię z bráćmi / płynął do Syryi / á z nim Pryſcyllá y Akwiláſ / ogoliwƺy głowę w Kienchreách : bo był ucżynił ślub.
Zátem przyƺedł do Efezu y tám je zoſtáwił / á ſám wƺedłƺy do bóżnicy / miáł rozmowę z Żydámi.
A gdy go oni prośili / áby u nich przez dłużƺy cżáſ zámieƺkáł / nie zezwolił ;
Ale śię z nimi pożegnáwƺy / rzekł : Koniecżnie já muƺę święto nádchodzące w Jeruzálemie obchodźić ; lecż śię záśię do wáſ wrócę / będźieli wolá Bożá. Y puśćił śię z Efezu.
A gdy przyƺedł do Cezáryi / wſtąpiwƺy do Jeruzálemu á pozdrowiwƺy zbór / ƺedł do Antyjochyi.
Y zámieƺkáwƺy tám przez niektóry cżáſ / wyƺedł / obchodząc porządnie kráinę Gálátſką / y Frygiję / utwierdzájąc wƺyſtkich ucżniów.
A Żyd niektóry imieniem Apolloſ / rodem z Alekſándryi / mąż wymowny / przyƺedł do Efezu / będąc możnym w Piſmách.
Ten był wpráwiony w drogę Páńſką / á páłájąc w duchu / mówił y náucżáł pilnie o Pánu / wiedząc tylko o chrzćie Jánowym.
Ten pocżął bezpiecżnie mówić w bóżnicy. Którego uſłyƺáwƺy Akwiláſ y Pryſcyllá / przyjęli go do śiebie / y doſtátecżniej mu wyłożyli drogę Bożą.
A gdy chćiáł iść do Achái / nápomniáwƺy go bráćiá / piſáli do ucżniów / áby go przyjęli ; który gdy tám przyƺedł / wiele pomágáł tym / którzy uwierzyli z łáſki Bożej.
Abowiem potężnie Żydy przekonywáł / jáwnie tego dowodząc z Piſmá / iż JEzuſ jeſt CHryſtuſem.
