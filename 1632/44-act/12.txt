A pod onże cżás / udał śię ná to Herod Król / áby trapił niektóre ze Zboru.
Y zábił Jákubá brátá Janowego miecżem.
A widząc że śię to podobáło Żydom / umyślił pojimáć y Piotrá : ( á były dni Przáſników. )
Którego pojimawƺy podał do więźienia / porucżywƺy go ƺeſnaśći żołnierzom / áby go ſtrzegli ; chcąc go po Wielkieynocy wywieść ludowi.
Tedy ſtrzeżono Piotrá w więźieniu : á modlitwá uſtáwicżna dźiałá śię od Zboru do Bogá zá nim.
A gdy go już miał wywieść Herod / onejże nocy ſpał Piotr miedzy dwiemá żołnierzámi / związány dwiemá łáńcuchámi / á ſtróże przed drzwiámi ſtrzegli więźienia.
A oto Anjoł PAńſki przyſtąpił / á świátłość śię rozświećiłá w gmáchu ; á trąćiwƺy w bok Piotrá / obudźił go mówiąc : Wſtań rychło. Y opádły łáńcuchy z rąk <i>jego</i>.
Y rzekł Anjoł do niego : Opáƺ śię / á podwiąż obuwie twoje. Y ucżynił ták. Y rzekł mu : Odźiey śię w płaƺcż ſwój / á pódź zá mną.
Tedy wyƺedƺy <i>Piotr,</i> ƺedł zá nim ; á nie wiedźiał / że śię to dźiało po prawdźie / co śię dźiało przez Anjołá / lecż mniemał że widzenie widźiał.
A gdy minęli pierwƺą y wtórą ſtraż / przyƺli do bramy żelázney / która wiedźie do miáſtá ; á tá śię im ſámá przez śię otworzyłá. A wyƺedƺy przeƺli jednę ulicę / á zárázem odſtąpił Anjoł od niego.
Tedy Piotr przyƺedƺy do śiebie / rzekł : Teraz znam prawdźiwie / iż poſłał PAN Anjołá ſwego / y wyrwał mię z ręki Herodowey / y ze wƺyſtkiego ocżekawánia ludu Żydowſkiego.
A obacżywƺy <i>śię,</i> przyƺedł do domu Máryey / mátki Jánowey / którego názywano Márkiem / gdźie śię ich było wiele zgromádźiło / y modlili śię.
A gdy Piotr kołátał we drzwi u przyśionká / wyƺłá dźiewecżká imieniem Rode / áby poſłucháłá.
A poznawƺy głos Piotrów / od rádośći nie otworzyłá drzwi ; ále wbieżawƺy / oznájmiłá iż Piotr ſtoji u drzwi.
A oni rzekli do niey : Szálejeż. Wƺákże oná twirdźiłá iż śię ták rzecż má. A oni rzekli : Anjoł jego jeſt.
Ale Piotr nieprzeſtał kołátáć. A gdy otworzyli / ujrzeli go y zdumieli śię.
A zkinąwƺy ná nie ręką áby umilknęli / rozpowiedźiał im jáko go PAn wywiódł z więźienia / y rzekł : Oznájmićie to Jákubowi y bráći. A wyƺedƺy / ƺedł ná inne miejſce.
A gdy był dźień / ſtał śię rozruch niemáły miedzy żołnierzámi <i>o to,</i> coby śię z Piotrem ſtáło.
Lecż Herod / gdy śię o nim wywiádował á nie ználazł go / ucżyniwƺy ſąd o ſtróżách / kazał je ná ſtrácenie wywieść. A wyjáchawƺy z Judſkiey źiemie do Cezáryey / mieƺkał tám.
A ná ten cżás Herod / myślił o wojnie przećiwko Tyryjcżykom y Sydońcżykom. Ale oni jednomyślnie przyƺli do niego ; á námówiwƺy Bláſtá Podkomorzego królewſkiego / prośili o pokój ; dla tego / iż krájiná ich miáłá żywność <i>z dźierżaw</i> królewſkich.
A dniá pewnego / Herod oblekƺy śię w ƺátę królewſką y śiadƺy ná ſtolicy / ucżynił rzecż do nich.
A lud wołał : Głos Boży / á nie cżłowiecży.
A zárázem uderzył go Anjoł PAńſki / przeto / że nie dał chwały Bogu ; á będąc roztocżony od robáſtwá / zdechł.
A ſłowo PAńſkie rozraſtáło śię y rozmnażáło.
A Bárnábaƺ y Saul wróćili śię z Jeruzalem / wykonawƺy poſługę ; wźiąwƺy z ſobą y Janá / którego názywano Márkiem.
