A mąż niektóry imieniem Anániaƺ / z Sáfirą żoną ſwoją przedał májętność.
Y ujął <i>nieco</i> z onych pieniędzy / z wiádomośćią żony ſwojey : á przyniózƺy cżęść niejáką / położył u nóg Apoſtolſkich.
Y rzekł Piotr ; Anániaƺu / przecżże Szátan nápełnił ſerce twoje / ábyś zkłamał Duchowi Świętemu / y ujął z pieniędzy zá rolą?
Izali to coś miał nie twoje było ; á coś przedał / nie w twojey mocy zoſtawáło? Przecżżeś tę rzecż przypuśćił do ſercá twego? Nie zkłamałeś ludźiom / ále Bogu.
Tedy uſłyƺawƺy Anániaƺ te ſłowá / padƺy zdechł. Y przyƺedł ſtrách wielki ná wƺyſtkie / którzy to ſłyƺeli.
A wſtawƺy młodźieńcy / porwáli go ; á wyniózƺy / pogrzebli.
Y ſtáło śię po chwili / jákoby po trzech godźinách / że y żoná jego nie wiedząc co śię ſtáło / weƺłá.
Y rzekł jey Piotr : Powiedz mi / jeſliśćie zá tyle tę rolą przedáli? A oná rzekłá ; Ták jeſt / zá tyle.
A Piotr rzekł do niey : Przecżżeśćie śię z ſobą zmówili / ábyśćie kuśili Duchá PAńſkiego? Oto nogi tych / którzy pogrzebli mężá twego / u drzwi <i>ſą</i> ; y ćiebieć wynioſą.
Y pádłá záraz przed nogámi jego / y zdechłá. A wƺedƺy młodźieńcy / ználeſli ją umárłą ; á wyniózƺy / pogrzebli <i>ją</i> podle mężá jey.
Y przyƺedł ſtrách wielki ná wƺyſtek Zbór / y ná wƺyſtkie którzy to ſłyƺeli.
Lecż przez ręce Apoſtolſkie dźiało śię wiele známion y cudów miedzy ludem : ( A byli wƺyſcy jednomyślnie w przyśionku Sálomonowym ;
A z innych żaden nie śmiał śię do nich przyłącżyć ; ále lud wiele o nich trzymał :
Y owƺem przybywáło mnóſtwá wierzących PAnu / mężów y niewiaſt. )
Ták / że y ná ulice wynośili chore / y kłádli <i>je</i> ná pośćielách y łóżkách ; áby przynamniey ćień Piotrá przychodzącego záćienił niektóre z nich.
Zchodźiło śię też y mnóſtwo z okolicżnych miaſt do Jeruzalem / przynoƺąc chore y nágabáne od duchów niecżyſtych : á ći wƺyſcy byli uzdrowieni.
Tedy powſtawƺy Nawyżƺy Kápłan / y wƺyſcy którzy z nim byli / którzy byli z Sekty Sáduceuƺów / nápełnieni ſą zazdrośćią.
Y tárgnęli śię rękomá ná Apoſtoły / y podáli je do więźienia poſpolitego.
Ale Anjoł PAńſki w nocy otworzył drzwi u więźienia / á wywiódƺy je / rzekł :
Idźćież / á ſtáwiwƺy śię / mówćie do ludu w Kośćiele wƺyſtkie ſłowá tego żywotá.
Tedy <i>oni</i> uſłyƺawƺy <i>to,</i> weƺli ná świtániu do Kośćiołá / y ucżyli. A przyƺedƺy Nawyżƺy Kápłan y którzy z nim byli / zwołáli Rádę y wƺyſtkie Stárƺe ſynów Izráelſkich : y poſłáli do więźienia áby byli przywiedźieni.
A gdy ſłudzy przyƺli / nie ználeźli ich w więźieniu : co / wróćiwƺy śię / oznájmili / Mówiąc :
Więźienieć w prawdźie ználeźliſmy zámknione ze wƺelką pilnośćią / y ſtróże ná dworze przede drzwiámi ſtojące : lecż otworzywƺy / żadnegoſmy w nim nie ználeźli.
A gdy te ſłowá uſłyƺeli / y Nawyżƺy kápłan / y hetman kośćielny / y Przedniejƺy kápłani / wątpili o nich coby to było.
A przyƺedƺy ktoś / oznájmił im / mówiąc : Oto / mężowie któreśćie podáli do więźienia / ſtoją w Kośćiele / á ucżą lud.
Tedy poƺedł hetman z ſługámi / y przywiódł je bez gwałtu ; ( bo śię ludu bali / áby nie byli ukámionowáni. )
A przywiódƺy je / ſtáwili <i>je</i> przed Rádą. Y pytał ich Nawyżƺy Kápłan / Mówiąc :
Izaliſmy wam ſurowie nie zákazáli / ábyśćie w tym Imieniu nie ucżyli? A oto nápełniliśćie Jeruzalem náuką wáƺą / y chcećie ná nas wprowádźić krew cżłowieká tego.
Tedy odpowiádájąc Piotr y Apoſtołowie rzekli : Więcey trzebá ſłucháć Bogá / niż ludźi.
Bóg on Ojców náƺych wzbudźił JEzuſá / któregośćie wy zábili / záwieśiwƺy ná drzewie :
Tego Bóg zá Kśiążę y Zbáwićielá wywyżƺył práwicą ſwoją / áby dána byłá ludowi Izráelſkiemu pokutá y odpuƺcżenie grzechów.
A myſmy świádkámi jego w tym co mówimy : tákże y Duch Święty / którego dał Bóg tym którzy mu ſą poſłuƺni.
A oni to ſłyƺąc / pukáli śię / y rádźili o tym jákoby je zágłádźić.
Tedy powſtawƺy w Rádźie niektóry Fáryzeuƺ / imieniem Gámáliel Náucżyćiel zakonny / zacny u wƺyſtkiego ludu / rozkazał áby ná máłą chwilę precż wywiedźiono Apoſtoły :
Y rzekł do nich ; Mężowie Izráelſcy / miejćie śię ná bacżeniu z ſtrony tych ludźi / cobyśćie mieli cżynić.
Abowiem przed tymi dniámi powſtał był Teudás / udawájąc śię zá coś : do którego śię przywiązáło mężów w licżbie około cżterech ſet ; którego zábito : á wƺyſcy którzy z nim przeſtawáli / rozproƺeni ſą / y wniwecż śię obróćili.
Po nim powſtał Judás Gálilejcżyk zá dni popiſu / y uwiódł wiele ludu zá ſobą : <i>ále</i> y on zginął ; y wƺyſcy / którzy z nim przeſtawáli / rozproƺeni ſą.
Przetoż teraz powiádam wam / Dajćie pokój tym ludźiom / y zániechajćie ich : Abowiem jeſliżeć jeſt z ludźi tá rádá álbo tá ſpráwá / wniwecż śię obróći :
Ale jeſlić jeſt z Bogá / nie będźiećie mogli tego rozerwáć ; byśćie ſnadź y z Bogiem walcżącymi nie byli ználeźieni.
Y uſłucháli go. A záwoławƺy Apoſtołów / <i>y</i> ubiwƺy <i>je,</i> zákazáli / áby nie mówili w Imieniu JEzuſowym ; y wypuśćili je.
A ták oni ƺli od oblicżnośći oney Rády / rádując śię iż śię ſtáli godnymi odnośić zelżywość dla Imienia JEzuſowego.
Y nie przeſtawáli ná káżdy dźień w Kośćiele y po domách náucżáć / y opowiádáć JEzuſá CHryſtuſá.
