A mąż niektóry imieniem Anánijáƺ / z Sáfirą / żoną ſwoją / ſprzedáł májętność /
Y ujął nieco z onych pieniędzy z wiádomośćią żony ſwojej / á przynióſłƺy cżęść niejáką / położył u nóg Apoſtolſkich.
Y rzekł Piotr : Anánijáƺu! przecżże ƺátán nápełnił ſerce twoje / ábyś kłámáł Duchowi Świętemu / y ujął z pieniędzy zá rolę?
Izáli to / coś miáł / nie twoje było? á coś ſprzedáł / nie w twojej mocy zoſtáwáło? Przecżżeś tę rzecż przypuśćił do ſercá twego? Nie ſkłámáłeś ludźiom / ále Bogu.
Tedy uſłyƺáwƺy Anánijáƺ te ſłowá / pádł nieżywy. Y przyƺedł ſtrách wielki ná wƺyſtkich / którzy to ſłyƺeli.
A wſtáwƺy młodźieńcy / porwáli go / á wynióſłƺy pogrzebli.
Y ſtáło śię po chwili / jákoby po trzech godźinách / że y żoná jego nie wiedząc / co śię ſtáło / weƺłá.
Y rzekł jej Piotr : Powiedz mi / jeźliśćie zá tyle tę rolę ſprzedáli? A oná rzekłá : Ták jeſt / zá tyle.
A Piotr rzekł do niej : Przecżżeśćie śię z ſobą zmówili / ábyśćie kuśili Duchá Páńſkiego? Oto nogi tych / którzy pogrzebli mężá twego / u drzwi ſą / y ćiebieć wynioſą.
Y pádłá záráz przed nogámi jego nieżywá. A wƺedłƺy młodźieńcy / ználeźli ją umárłą / á wynióſłƺy pogrzebli ją podle mężá jej.
Y przyƺedł ſtrách wielki ná wƺyſtek zbór / y ná wƺyſtkich / którzy to ſłyƺeli.
Lecż przez ręce Apoſtolſkie dźiáło śię wiele známion y cudów między ludem / ( á byli wƺyſcy jednomyślnie w przyśionku Sálomonowym.
A z innych żáden nie śmiáł śię do nich przyłącżyć ; ále lud wiele o nich trzymáł.
Y owƺem przybywáło mnóſtwo wierzących Pánu / mężów y niewiáſt. )
Ták / że y ná ulice wynośili chorych / y kłádli je ná pośćielách y łóżkách / áby przynájmniej ćień Piotrá przychodzącego záćienił niektórych z nich.
Schodźiło śię też y mnóſtwo z okolicżnych miáſt do Jeruzálemu / przynoƺąc chorych y nágábánych od duchów niecżyſtych ; á ći wƺyſcy byli uzdrowieni.
Tedy powſtáwƺy nájwyżƺy kápłán / y wƺyſcy / którzy z nim byli / którzy byli z ſekty Sáduceuƺów / nápełnieni ſą zázdrośćią ;
Y tárgnęli śię rękomá ná Apoſtoły / y podáli je do więźieniá poſpolitego.
Ale Anioł Páńſki w nocy otworzył drzwi u więźieniá / á wywiódłƺy je rzekł :
Idźćież / á ſtáwiwƺy śię / mówćie do ludu w kośćiele wƺyſtkie ſłowá tego żywotá.
Tedy oni uſłyƺáwƺy to / weƺli ná świtániu do kośćiołá / y ucżyli. A przyƺedłƺy nájwyżƺy kápłán / y którzy z nim byli / zwołáli rádę / y wƺyſtkie ſtárƺe ſynów Izráelſkich / y poſłáli do więźieniá / áby byli przywiedźieni.
A gdy ſłudzy przyƺli / nie ználeźli ich w więźieniu / co wróćiwƺy śię / oznájmili / mówiąc :
Więźienieć wpráwdźie ználeźliśmy zámknione ze wƺelką pilnośćią / y ſtróże ná dworze przede drzwiámi ſtojące / lecż otworzywƺy / żádnegośmy w nim nie ználeźli.
A gdy te ſłowá uſłyƺeli / y nájwyżƺy kápłán y hetmán kośćielny y przedniejśi kápłáni / wątpili o nich / coby to było.
A przyƺedłƺy ktoś / oznájmił im / mówiąc : Oto mężowie / któreśćie podáli do więźieniá / ſtoją w kośćiele / á ucżą lud.
Tedy poƺedł hetmán z ſługámi y przywiódł je bez gwáłtu ; ( bo śię ludu báli / áby nie byli ukámionowáni. )
A przywiódłƺy je / ſtáwili je przed rádą ; y pytáł ich nájwyżƺy kápłán / mówiąc :
Izáliśmy wám ſurowo nie zákázáli / ábyśćie w tem imieniu nie ucżyli? A oto nápełniliśćie Jeruzálem náuką wáƺą / y chcećie ná náſ wprowádźić krew cżłowieká tego.
Tedy odpowiádájąc Piotr y Apoſtołowie / rzekli : Więcej trzebá ſłucháć Bogá / niż ludźi.
Bóg on ojców náƺych wzbudźił JEzuſá / któregośćie wy zábili / záwieśiwƺy ná drzewie.
Tego Bóg zá kśiążęćiá y zbáwićielá wywyżƺył práwicą ſwoją / áby dáná byłá ludowi Izráelſkiemu pokutá y odpuƺcżenie grzechów.
A myśmy świádkámi jego w tem / co mówimy / tákże y Duch Święty / którego dáł Bóg tym / którzy mu ſą poſłuƺni.
A oni to ſłyƺąc / pukáli śię y rádźili o tem / jákoby je zgłádźić.
Tedy powſtáwƺy w rádźie niektóry Fáryzeuƺ / imieniem Gámálijel / náucżyćiel zákonny / zácny u wƺyſtkiego ludu / rozkázáł / áby ná máłą chwilę precż wywiedźiono Apoſtoły ;
Y rzekł do nich : Mężowie Izráelſcy! miejćie śię ná bácżeniu z ſtrony tych ludźi / cobyśćie mieli cżynić.
Abowiem przed temi dniámi powſtáł był Teudáſ / udáwájąc śię zá coś / do którego śię przywiązáło mężów w licżbie około cżterech ſet ; którego zábito / á wƺyſcy / którzy z nim przeſtáwáli / rozproƺeni ſą / y wniwecż śię obróćili.
Po nim powſtáł Judáſ Gálilejcżyk zá dni popiſu / y uwiódł wiele ludu zá ſobą ; ále y on zginął / y wƺyſcy / którzy z nim przeſtáwáli / rozproƺeni ſą.
Przetoż teráz powiádám wám : Dájćie pokój tym ludźiom / y zániechájćie ich ; ábowiem jeźliżeć jeſt z ludźi tá rádá álbo tá ſpráwá / wniwecż śię obróći ;
Ale jeźlić jeſt z Bogá / nie będźiećie mogli tego rozerwáć / byśćie ſnáć y z Bogiem wálcżącymi nie byli ználeźieni.
Y uſłucháli go. A záwołáwƺy Apoſtołów y ubiwƺy je / zákázáli / áby nie mówili w imieniu JEzuſowem ; y wypuśćili je.
A ták oni ƺli od oblicżnośći onej rády / rádując śię / iż śię ſtáli godnymi odnośić zelżywość dlá imieniá JEzuſowego.
Y nie przeſtáwáli ná káżdy dźień w kośćiele y po domách náucżáć y opowiádáć JEzuſá CHryſtuſá.
