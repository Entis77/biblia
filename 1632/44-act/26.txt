Zátem Agrypá rzekł do Páwłá : Pozwáláć śię / ábyś mówił ſám od śiebie. Tedy Páweł wyćiągnąwƺy rękę / táką ſpráwę dáł :
Ná to wƺyſtko / z cżego mię obwiniáją Żydowie / królu Agrypo! pocżytám śię być zá ƺcżęśliwego / iż dziś mám odpowiádáć przed tobą.
A zwłáƺcżá żeś ty powiádom tych wƺyſtkich / które ſą między Żydámi / zwycżájów y ſporów ; przetoż ćię proƺę / żebyś mię ćierpliwie poſłucháł.
Co śię tedy tknie żywotá mego od młodośći / jáki był od pocżątku między národem moim w Jeruzálemie / wiedzą wƺyſcy Żydowie /
Będąc mi świádkámi z dáwná / ( gdyby świádectwo wydáć chćieli ) iż według nájdoſkonálƺej ſekty nábożeńſtwá náƺego żyłem / będąc Fáryzeuƺem.
A teráz o nádzieję onej obietnicy / ojcom od Bogá ucżynionej / ſtoję przed ſądem ;
Której dwánáśćie náƺych pokoleń uſtáwicżnie dniem y nocą ſłużąc Bogu / máją nádzieję doſtąpić ; o tę nádzieję ſkárżą ná mię Żydowie / o królu Agrypo!
Cóż? zá rzecż do wiáry niepodobną u śiebie ſądzićie / że Bóg umárłe wzbudzá?
Mnieć śię wpráwdzie ſámemu zdáło / żem był powinien przećiwko imieniowi Jezuſá Názáreńſkiego wiele przećiwnych rzecży cżynić.
Com też cżynił w Jeruzálemie / y wielem já świętych ſádzáł do więzieniá / wziąwƺy moc od przedniejƺych kápłánów ; á gdy mieli być zábijáni / wotowáłem przećiwko nim.
Y po wƺyſtkich bóżnicách cżęſtokroć je trápiąc / przymuƺáłem bluźnić / á náder wśćiekle przećiwko nim poſtępując / prześládowáłem je áż y do obcych miáſt.
Wcżem / gdym też do Dámáƺku jecháł / májąc włádzę y zlecenie od przedniejƺych kápłánów /
W południe / w drodze będąc / widziáłem / o królu! świátłość z niebá / jáśniejƺą nád jáſność ſłonecżną / którá oświećiłá mię / y tych / którzy jecháli ze mną.
A gdyśmy wƺyſcy upádli ná ziemię / uſłyƺáłem głoſ mówiący do śiebie / á mówiący Żydowſkim językiem : Saulu! Saulu! przecżże mię prześládujeƺ? trudno tobie przećiwko ośćieniowi wierzgáć.
A jám rzekł : Ktoś jeſt / Pánie? A on rzekł : Jám jeſt Jezuſ / którego ty prześládujeƺ.
Ale wſtáń / á ſtáń ná nogách twoich ; gdyżem ći śię dlátego pokázáł / ábym ćię ucżynił ſługą y świádkiem ták tych rzecży / któreś widziáł / jáko y innych / w których ći śię pokáżę.
Wyrywájąc ćię od tego ludu y od pogán / do których ćię teráz poſyłám /
Ku otworzeniu ocżu ich / áby śię náwróćili z ćiemnośći do świátłośći / á z mocy ƺátáńſkiej do Bogá / áby ták wzięli odpuƺcżenie grzechów / y dziáł między poświęconymi przez wiárę / którá jeſt w mię.
Przetoż / o królu Agrypo! nie byłem niepoſłuƺnym temu niebieſkiemu widzeniu.
Ale nájprzód tym / którzy ſą w Dámáƺku y w Jeruzálemie y w wƺyſtkiej kráinie Judzkiej / y pogánom opowiádáłem / áby pokutowáli y náwróćili śię do Bogá / cżyniąc ucżynki godne pokuty.
Dlá tych rzecży Żydowie w kośćiele mię pojmáwƺy / chćieli mię zábić.
Ale zá pomocą Bożą jeƺcże áż do dniá tego ſtoję / świádcżąc y máłemu y wielkiemu / nic nie mówiąc oprócż tego / co opowiedzieli prorocy y Mojżeƺ / że śię ſtáć miáło ;
To jeſt / iż Chryſtuſ miáł ćierpieć / á będąc pierwƺym z zmártwychwſtániá / opowiádáć miáł świátłość ludowi temu y pogánom.
To gdy on ku obronie ſwojej powiedziáł / rzekł Feſtuſ głoſem wielkim : Szálejeƺ Páwle! wielká náuká przywodzi ćię do ƺáleńſtwá.
Ale on rzekł : Nie ƺáleję / nájmożniejƺy Feśćie! áleć práwdziwe y zdrowe ſłowá powiádám.
Wie bowiem y król o tych rzecżách / przed którym bezpiecżnie mówię / gdyż nie tuƺę / áby co z tych rzecży u niego było tájno / poniewáż śię to nie w kąćie dziáło.
Wierzyƺ / królu Agrypo! prorokom? Wiem / iż wierzyƺ.
Zátem Agrypá rzekł do Páwłá : Máłobyś mnie nie námówił / żebym zoſtáł chrześćijáninem.
Ale Páweł rzekł : Życżyłbym od Bogá / áby y w máłu y w wielu / nie tylko ty / ále y wƺyſcy / którzy mię dziś ſłucháją / ſtáli śię tákimi / jákim y já jeſt / oprócż tych związek.
A gdy on to rzekł / wſtáł król / y ſtároſtá / y Bernice / y ći / którzy śiedzieli z nimi.
A uſtąpiwƺy ná ſtronę / rzekli jedni do drugich / mówiąc : Nic godnego śmierći álbo więzieniá nie cżyni ten cżłowiek.
Lecż Agrypá rzekł do Feſtá : Mógł ten cżłowiek być uwolniony / by był do ceſárzá nie ápelowáł.
