A w Cezáryi był mąż niektóry / imieniem Kornelijuƺ / ſetnik / z roty / którą zwáno Włoſką ;
Pobożny / y bojący śię Bogá ze wƺyſtkim domem ſwoim / y cżyniący jáłmużny wielkie ludowi.
A ten śię záwƺe Bogu modląc / widziáł jáwnie w widzeniu / jákoby o dziewiątej godzinie ná dzień / Aniołá Bożego / że wƺedł do niego y rzekł mu : Kornelijuƺu!
A on pilnie náń pátrząc / á przeſtráƺony będąc / rzekł : Cóż jeſt / Pánie? Y rzekł mu : Modlitwy twoje y jáłmużny twoje wſtąpiły ná pámięć przed oblicżność Bożą.
Przetoż teráz poślij męże do Joppy / á przyzwij Szymoná / którego zowią Piotrem.
Ten má goſpodę u niektórego Szymoná / gárbárzá / który má dom nád morzem ; tenći powie / cobyś miáł cżynić.
A gdy odƺedł Anioł / który mówił z Kornelijuƺem / záwołáwƺy dwóch ſług ſwoich / y żołnierzá pobożnego z tych / którzy przy nim uſtáwicżnie byli ;
A rozpowiedziáwƺy im wƺyſtko / poſłáł je do Joppy.
A názájutrz / gdy byli w drodze / á przybliżáli śię do miáſtá / wſtąpił Piotr ná dách / áby śię modlił / około godziny ƺóſtej.
A będąc łáknącym chćiáł jeść ; á gdy mu oni jeść gotowáli / przypádło náń záchwycenie.
Y ujrzáł niebo otworzone y zſtępujące ná śię nácżynie niejákie / jákoby prześćierádło wielkie / zá cżtery rogi uwiązáne y ſpuƺcżone ná ziemię ;
W którem były wƺelkie ziemſkie cżworonogie zwierzętá / y beſtyje / y gádziny y ptáſtwo niebieſkie.
Y ſtáł śię głoſ do niego : Wſtáń Piotrze! rzeż á jedz.
A Piotr rzekł : Żádną miárą / Pánie! gdyżem nigdy nie jádł nic poſpolitego álbo niecżyſtego.
Tedy záśię powtóre ſtáł śię głoſ do niego : Co Bóg ocżyśćił / ty nie miej tego zá niecżyſte.
A to śię ſtáło po trzykroć. Y wzięte jeſt záśię ono nácżynie do niebá.
A gdy Piotr ſám w ſobie wątpił coby to było zá widzenie które widziáł : tedy oto ći mężowie / którzy byli poſłáni od Kornelijuƺá / pytájący śię o dom Szymonowy / ſtáli przede drzwiámi ;
A záwołáwƺy / wywiádywáli śię / jeźliby tám Szymon / którego zowią Piotrem / goſpodę miáł.
A gdy Piotr myślił o onem widzeniu / rzekł mu Duch : Oto ćię trzej mężowie ƺukáją.
Przetoż wſtáwƺy / zſtąp / á idź z nimi / nic nie wątpiąc : bomći já je poſłáł.
Tedy Piotr zſtąpiwƺy do onych mężów / którzy od Kornelijuƺá do niego poſłáni byli / rzekł : Otom já jeſt / którego ƺukáćie. Cóż zá przycżyná / dlá którejśćie przyƺli?
A oni rzekli : Kornelijuƺ ſetnik / mąż ſpráwiedliwy y bojący śię Bogá y májący dobre świádectwo od wƺyſtkiego národu żydowſkiego / w widzeniu jeſt od Aniołá świętego nápomniony / áby ćię wezwáł w dom ſwój / y ſłucháł ſłów od ćiebie.
Tedy záwołáwƺy ich do domu / przyjął je do goſpody. A drugiego dniá Piotr ƺedł z nimi / y niektórzy z bráći z Joppy ƺli z nim.
A názájutrz weƺli do Cezáryi. A Kornelijuƺ cżekáł ich / wezwáwƺy powinowátych ſwoich y bliſkich przyjáćiół.
Y ſtáło śię / gdy wchodził Piotr / zábieżáwƺy mu Kornelijuƺ / przypádł do nóg jego / y pokłonił śię.
Ale go Piotr podnióſł / mówiąc : Wſtáń! y jámći też jeſt cżłowiek.
A rozmáwiájąc z nim / wƺedł / á ználázł wiele tych / którzy śię byli zeƺli.
Y rzekł do nich : Wy wiećie / że śię nie godzi mężowi Żydowinowi przyłącżáć álbo ſchádzáć z cudzoziemcem ; lecż mnie Bóg ukázáł / żebym żádnego cżłowieká nie názywáł poſpolitym álbo niecżyſtym.
Przetożem też niezbrániájąc śię przyƺedł / wezwány będąc ; pytám tedy / dlácżegośćie mię wezwáli?
A Kornelijuƺ rzekł : Od cżwártego dniá áż do tej godziny pośćiłem / á o dziewiątej godzinie modliłem śię w domu moim / á oto mąż niektóry ſtánął przedemną w odzieniu jáſnem /
Y rzekł : Kornelijuƺu! wyſłucháná jeſt modlitwá twojá / á jáłmużny twoje przyƺły ná pámięć przed oblicżność Bożą.
Przetoż poślij do Joppy / á przyzwij Szymoná / którego názywáją Piotrem ; ten má goſpodę w domu Szymoná / gárbárzá / nád morzem / który przyƺedłƺy / mówić z tobą będzie.
Záráz tedy poſłáłem do ćiebie / á tyś dobrze ucżynił / żeś przyƺedł. Teráz tedy jeſteśmy wƺyſcy przed oblicżem Bożem przytomni / ábyśmy ſłucháli wƺyſtkiego / coć rozkázáno od Bogá.
Tedy Piotr otworzywƺy uſtá / rzekł : Práwdziwie dochodzę tego / iż Bóg nie má względu ná oſoby ;
Ale w káżdym národzie / kto śię go boi á cżyni ſpráwiedliwość / jeſt mu przyjemnym.
A co śię tknie ſłowá / które poſłáł ſynom Izráelſkim / opowiádájąc pokój przez Jezuſá Chryſtuſá / który jeſt Pánem wƺyſtkiego /
Wy wiećie / co śię dziáło po wƺyſtkiem Żydoſtwie / pocżąwƺy od Gálilei / po chrzćie / który Ján opowiádáł ;
Jáko Jezuſá z Názáretu pomázáł Bóg Duchem Świętym y mocą / który chodził / cżyniąc dobrze y uzdráwiájąc wƺyſtkie opánowáne od dyjábłá ; ábowiem Bóg był z nim.
A myśmy świádkámi wƺyſtkiego tego / co cżynił w kráinie Judzkiej y w Jeruzálemie / którego zábili / záwieśiwƺy ná drzewie.
Tego Bóg wzbudził dniá trzećiego / y ſpráwił / żeby był objáwiony ;
Nie wƺyſtkiemu ludowi / ále świádkom / przedtem ſporządzonym od Bogá / nám / którzyśmy z nim jedli y pili po jego zmártwychwſtániu.
Y rozkázáł nám / ábyśmy kázáli ludowi / y świádcżyli / że on jeſt onym poſtánowionym od Bogá ſędzią żywych y umárłych.
Temu wƺyſcy prorocy świádectwo wydáją / iż przez imię jego odpuƺcżenie grzechów weźmie káżdy / co weń wierzy.
A gdy jeƺcże Piotr mówił te ſłowá / przypádł Duch Święty ná wƺyſtkie ſłuchájące tych ſłów.
Y zdumieli śię oni / którzy byli z obrzezániá wierzący / którzy byli z Piotrem przyƺli / że y ná pogány dár Duchá Świętego jeſt wylány.
Abowiem ſłyƺeli je mówiące językámi rozlicżnemi y wielbiące Bogá. Tedy odpowiedziáł Piotr :
Izáli kto może zábronić wody / żeby ći nie byli pochrzcżeni / którzy wzięli Duchá Świętego jáko y my?
Y rozkázáł je pochrzćić w imieniu Páńſkim. Y prośili go / áby u nich zoſtáł ná kilká dni.
