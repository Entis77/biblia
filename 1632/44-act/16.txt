Y przyƺedł do Derby y do Liſtry ; á oto tám był ucżeń niektóry / imieniem Tymoteuƺ / ſyn niektórey niewiáſty Żydówki wierney á ojcá Greká.
Temu świádectwo dáwáli bráćia / którzy byli w Liſtrze y w Ikonii.
Chćiáł tedy Páweł / áby ten z nim ƺedł / którego wźiąwƺy / obrzezáł dla Żydów / którzy byli ná onych miejſcách ; bo wƺyſcy wiedźieli / że ojćiec jego był Grekiem.
A gdy chodźili po miáſtách / podáwáli im ku chowániu uſtáwy / które były poſtánowione od Apoſtołów / y ſtárƺych / którzy byli w Jeruzálemie.
A ták śię zbory utwierdzáły w wierze / y przybywáło ich w licżbie ná káżdy dźień.
Tedy przeƺedłƺy Frygiję y Gálátſką kráinę / záwśćiągnieni będąc od Duchá Świętego / áby nie opowiádáli ſłowá Bożego w Azyi /
Przyƺedłƺy do Mizyi / kuśili śię iść do Bitynii / ále im Duch JEzuſowy nie dopuśćił.
Tedy minąwƺy Mizyję / zſtąpili do Troády.
Y pokázáło śię Páwłowi w nocy widzenie : Mąż niejáki Mácedońcżyk ſtáł / proƺąc go y mówiąc : Przepráw śię do Mácedonii á rátuj nas.
A ujrzáwƺy to widzenie / záráześmy śię ſtáráli o to / jákobyśmy śię puśćili do Mácedonii / będąc tego pewni / iż nas Pán powołáł / ábyśmy im kázáli Ewángieliję.
Puśćiwƺy śię tedy z Troády / proſtośmy bieżeli do Sámotrácyi / á názájutrz do Neápolu.
A ſtámtąd do Filipowá / które jeſt pierwƺe miáſto tey cżęśći Mácedonii nowo oſádzone ; y zoſtáliśmy w onem mieśćie przez kilká dni.
A w dźień ſábátu wyƺliśmy przed miáſto nád rzekę / gdźie zwykły bywáć modlitwy / á uśiádłƺy mówiliśmy do niewiáſt / które śię tám były zeƺły.
A niektóra niewiáſtá / imieniem Lidyjá / która ƺárłát ſprzedáwáłá w mieśćie Tyjátyrſkiem / Bogá śię bojącá / ſłucháłá ; którey Pán otworzył ſerce / áby pilnie ſłucháłá tego / co Páweł mówił.
A gdy śię ochrzćiłá y dom jey / prośiłá / mówiąc : Poniewáżeśćie mię oſądźili wierną być Pánu / wƺedłƺy do domu mego / mieƺkájćie ; y przymuśiłá nas.
Y ſtáło śię / gdyśmy ƺli ná modlitwę / iż niektóra dźiewecżká / co miáłá duchá wieƺcżego / zábieżáłá nám / á tá wielki zyſk pánom ſwoim przynośiłá / wróżąc.
Tá chodząc zá Páwłem y zá námi / wołáłá mówiąc : Ci ludźie ſługámi ſą Bogá nájwyżƺego / którzy nám opowiádáją drogę zbáwieniá.
A to cżyniłá przez wiele dni ; ále Páweł bolejąc nád tem y obróćiwƺy śię / rzekł onemu duchowi : Rozkázujęć w imieniu JEzuſá CHryſtuſá / ábyś wyƺedł od niej. Y wyƺedł onejże godźiny.
A widząc pánowie jey / iż zginęłá nádźiejá zyſku ich / pojmáwƺy Páwłá y Sylę / ćiągnęli je ná rynek przed urząd /
A ſtáwiwƺy je przed hetmány / rzekli : Ci ludźie cżynią zámięƺánie w mieśćie náƺem / będąc Żydámi :
Y opowiádáją zwycżáje / których śię nám nie godźi przyjmowáć áni záchowywáć / poniewáżeśmy Rzymiánie.
Y powſtáło poſpólſtwo przećiwko nim / á hetmáni rozdárłƺy ƺáty ich / kázáli je śiec rózgámi.
A gdy im wiele plág zádáli / wrzućili je do więźieniá przykázáwƺy ſtróżowi więźieniá / áby ich dobrze opátrzył.
Który wźiąwƺy tákie rozkázánie / wſádźił je do nájgłębƺego więźieniá / á nogi ich zámknął w kłodę.
A o północy Páweł y Syláſ modląc śię / chwálili Bogá pieśniámi / ták / że je ſłyƺeli więźniowie.
Y powſtáło z prędká wielkie trzęśienie źiemi / że śię poruƺyły grunty więźieniá / y zárázem śię otworzyły wƺyſtkie drzwi / y wƺyſtkich śię związki rozwiązáły.
A ocućiwƺy śię ſtróż więźieniá y ujrzáwƺy otworzone drzwi u więźieniá / dobył miecżá / chcąc śię ſám zábić / mniemájąc / iż więźniowie poućiekáli.
Lecż Páweł záwołáł głoſem wielkim / mówiąc : Nie cżyń ſobie nic złego : bośmy ſámi wƺyſcy.
A kázáwƺy záświećić / wpádł tám / á drżąc przypádł do nóg Páwłowi y Syli :
A wywiódłƺy je z więźieniá / rzekł : Pánowie! co mam cżynić / ábym był zbáwiony?
A oni rzekli : Wierz w Páná JEzuſá CHryſtuſá / á będźieƺ zbáwiony / ty y dom twój.
Y opowiádáli mu ſłowo Páńſkie / y wƺyſtkim / którzy byli w domu jego.
A wźiąwƺy je onejże godźiny w nocy / omył rány ich y ochrzćił śię záráz / on y wƺyſcy domownicy jego.
A wprowádźiwƺy je do domu ſwego / nágotowáł im ſtół / y weſelił śię ze wƺyſtkim domem ſwoim / uwierzywƺy Bogu.
A gdy był dźień / poſłáli hetmáni ſługi miejſkie / mówiąc : Wypuść one ludźie.
Y oznájmił ſtróż więźieniá te ſłowá Páwłowi / iż hetmáni poſłáli / ábyśćie byli wypuƺcżeni : teraz tedy wyƺedłƺy / idźćie w pokoju.
Ale im Páweł rzekł : Uśiekłƺy nas jáwnie rózgámi nie przekonánych / gdyżeśmy ſą ludźie Rzymiánie / wrzućili do więźieniá ; á teraz nas potájemnie wygániáją? Nic z tego ; ále ſámi niech przyjdą y wyprowádzą nas.
Tedy powiedźieli hetmánom ſłudzy miejſcy te ſłowá. Y zlękli śię / uſłyƺáwƺy / że byli Rzymiánie /
A przyƺedłƺy / przeprośili ich / á wywiódłƺy ich / prośili ich / áby wyƺli z miáſtá.
Wyƺedłƺy tedy z więźieniá / weƺli do Lidyi / á ujrzáwƺy bráći / poćieƺyli je y odeƺli.
