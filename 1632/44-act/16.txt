Y przyƺedł do Derby y do Liſtry ; á oto tám był ucżeń niektóry / imieniem Tymoteuƺ / ſyn niektórej niewiáſty Żydówki wiernej á ojcá Greká.
Temu świádectwo dáwáli bráćiá / którzy byli w Liſtrze y w Ikonii.
Chćiáł tedy Páweł / áby ten z nim ƺedł / którego wziąwƺy / obrzezáł dlá Żydów / którzy byli ná onych miejſcách ; bo wƺyſcy wiedzieli / że ojćiec jego był Grekiem.
A gdy chodzili po miáſtách / podáwáli im ku chowániu uſtáwy / które były poſtánowione od Apoſtołów / y ſtárƺych / którzy byli w Jeruzálemie.
A ták śię zbory utwierdzáły w wierze / y przybywáło ich w licżbie ná káżdy dzień.
Tedy przeƺedłƺy Frygiję y Gálátſką kráinę / záwśćiągnieni będąc od Duchá Świętego / áby nie opowiádáli ſłowá Bożego w Azyi /
Przyƺedłƺy do Mizyi / kuśili śię iść do Bitynii / ále im Duch Jezuſowy nie dopuśćił.
Tedy minąwƺy Mizyję / zſtąpili do Troády.
Y pokázáło śię Páwłowi w nocy widzenie : Mąż niejáki Mácedońcżyk ſtáł / proƺąc go y mówiąc : Przepráw śię do Mácedonii á rátuj náſ.
A ujrzáwƺy to widzenie / záráześmy śię ſtáráli o to / jákobyśmy śię puśćili do Mácedonii / będąc tego pewni / iż náſ Pán powołáł / ábyśmy im kázáli Ewángieliję.
Puśćiwƺy śię tedy z Troády / proſtośmy bieżeli do Sámotrácyi / á názájutrz do Neápolu.
A ſtámtąd do Filipowá / które jeſt pierwƺe miáſto tej cżęśći Mácedonii nowo oſádzone ; y zoſtáliśmy w onem mieśćie przez kilká dni.
A w dzień ſábátu wyƺliśmy przed miáſto nád rzekę / gdzie zwykły bywáć modlitwy / á uśiádłƺy mówiliśmy do niewiáſt / które śię tám były zeƺły.
A niektórá niewiáſtá / imieniem Lidyjá / którá ƺárłát ſprzedáwáłá w mieśćie Tyjátyrſkiem / Bogá śię bojącá / ſłucháłá ; której Pán otworzył ſerce / áby pilnie ſłucháłá tego / co Páweł mówił.
A gdy śię ochrzćiłá y dom jej / prośiłá / mówiąc : Poniewáżeśćie mię oſądzili wierną być Pánu / wƺedłƺy do domu mego / mieƺkájćie ; y przymuśiłá náſ.
Y ſtáło śię / gdyśmy ƺli ná modlitwę / iż niektórá dziewecżká / co miáłá duchá wieƺcżego / zábieżáłá nám / á tá wielki zyſk pánom ſwoim przynośiłá / wróżąc.
Tá chodząc zá Páwłem y zá námi / wołáłá mówiąc : Ci ludzie ſługámi ſą Bogá nájwyżƺego / którzy nám opowiádáją drogę zbáwieniá.
A to cżyniłá przez wiele dni ; ále Páweł bolejąc nád tem y obróćiwƺy śię / rzekł onemu duchowi : Rozkázujęć w imieniu Jezuſá Chryſtuſá / ábyś wyƺedł od niej. Y wyƺedł onejże godziny.
A widząc pánowie jej / iż zginęłá nádziejá zyſku ich / pojmáwƺy Páwłá y Sylę / ćiągnęli je ná rynek przed urząd /
A ſtáwiwƺy je przed hetmány / rzekli : Ci ludzie cżynią zámięƺánie w mieśćie náƺem / będąc Żydámi :
Y opowiádáją zwycżáje / których śię nám nie godzi przyjmowáć áni záchowywáć / poniewáżeśmy Rzymiánie.
Y powſtáło poſpólſtwo przećiwko nim / á hetmáni rozdárłƺy ƺáty ich / kázáli je śiec rózgámi.
A gdy im wiele plág zádáli / wrzućili je do więzieniá przykázáwƺy ſtróżowi więzieniá / áby ich dobrze opátrzył.
Który wziąwƺy tákie rozkázánie / wſádził je do nájgłębƺego więzieniá / á nogi ich zámknął w kłodę.
A o północy Páweł y Syláſ modląc śię / chwálili Bogá pieśniámi / ták / że je ſłyƺeli więźniowie.
Y powſtáło z prędká wielkie trzęśienie ziemi / że śię poruƺyły grunty więzieniá / y zárázem śię otworzyły wƺyſtkie drzwi / y wƺyſtkich śię związki rozwiązáły.
A ocućiwƺy śię ſtróż więzieniá y ujrzáwƺy otworzone drzwi u więzieniá / dobył miecżá / chcąc śię ſám zábić / mniemájąc / iż więźniowie poućiekáli.
Lecż Páweł záwołáł głoſem wielkim / mówiąc : Nie cżyń ſobie nic złego : bośmy ſámi wƺyſcy.
A kázáwƺy záświećić / wpádł tám / á drżąc przypádł do nóg Páwłowi y Syli :
A wywiódłƺy je z więzieniá / rzekł : Pánowie! co mám cżynić / ábym był zbáwiony?
A oni rzekli : Wierz w Páná Jezuſá Chryſtuſá / á będzieƺ zbáwiony / ty y dom twój.
Y opowiádáli mu ſłowo Páńſkie / y wƺyſtkim / którzy byli w domu jego.
A wziąwƺy je onejże godziny w nocy / omył rány ich y ochrzćił śię záráz / on y wƺyſcy domownicy jego.
A wprowádziwƺy je do domu ſwego / nágotowáł im ſtół / y weſelił śię ze wƺyſtkim domem ſwoim / uwierzywƺy Bogu.
A gdy był dzień / poſłáli hetmáni ſługi miejſkie / mówiąc : Wypuść one ludzie.
Y oznájmił ſtróż więzieniá te ſłowá Páwłowi / iż hetmáni poſłáli / ábyśćie byli wypuƺcżeni : teráz tedy wyƺedłƺy / idźćie w pokoju.
Ale im Páweł rzekł : Uśiekłƺy náſ jáwnie rózgámi nie przekonánych / gdyżeśmy ſą ludzie Rzymiánie / wrzućili do więzieniá ; á teráz náſ potájemnie wygániáją? Nic z tego ; ále ſámi niech przyjdą y wyprowádzą náſ.
Tedy powiedzieli hetmánom ſłudzy miejſcy te ſłowá. Y zlękli śię / uſłyƺáwƺy / że byli Rzymiánie /
A przyƺedłƺy / przeprośili ich / á wywiódłƺy ich / prośili ich / áby wyƺli z miáſtá.
Wyƺedłƺy tedy z więzieniá / weƺli do Lidyi / á ujrzáwƺy bráći / poćieƺyli je y odeƺli.
