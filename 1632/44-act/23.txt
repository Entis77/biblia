A Páweł pilnie pátrząc ná onę rádę rzekł : Mężowie bráćiá! já ze wƺyſtkiego ſumieniá dobrego chodziłem przed Bogiem áż do dniá tego.
Tedy Anánijáƺ / nájwyżƺy kápłán / rozkázáł go tym / którzy przy nim ſtáli / bić w gębę.
Tedy rzekł Páweł do niego : Uderzy ćię Bóg / śćiáno pobieláná! y ty śiedziƺ / ſądząc mię według zákonu / á rozkázujeƺ mię bić przećiwko zákonowi?
Zátem ći / którzy tám ſtáli / rzekli : Nájwyżƺemu kápłánowi Bożemu złorzecżyƺ?
A Páweł rzekł : Nie wiedziáłem / bráćiá! żeby był nájwyżƺym kápłánem ; bo nápiſáno : Kśiążęćiu ludu twego złorzecżyć nie będzieƺ.
A poznáwƺy Páweł / że ich jedná cżęść byłá Sáduceuƺów á drugá Fáryzeuƺów / záwołáł w onej rádzie : Mężowie bráćiá! jám jeſt Fáryzeuƺ / ſyn Fáryzeuƺá : o nádzieję y o powſtánie umárłych mię tu dziś ſądzą.
A gdy on to mówił / wƺcżął śię rozruch między Fáryzeuƺámi y Sáduceuƺámi / y rozerwáło śię ono mnóſtwo.
Abowiem Sáduceuƺowie mówią / iż niemáƺ zmártwychwſtániá / áni Aniołá / áni duchá ; ále Fáryzeuƺowie to oboje wyznáwáją.
Y wƺcżęło śię wołánie wielkie. A powſtáwƺy náucżeni w Piśmie z ſtrony Fáryzeuƺów / ſpieráli śię mówiąc : Niceśmy złego nie ználeźli w tym cżłowieku ; y jeźli mu co powiedziáł duch álbo Anioł / nie wálcżmyż z Bogiem.
A gdy śię wƺcżął wielki rozruch / obáwiájąc śię hetmán / áby Páwłá między ſobą nie rozƺárpáli / rozkázáł iść żołnierzom ná dół / á wydrzeć go z pośrodku ich / y odwieść do obozu.
A drugiej nocy ſtánąwƺy przy nim Pán / rzekł : Bądź dobrego ſercá / Páwle! ábowiem jákoś o mnie świádcżył w Jeruzálemie / ták muśiƺ świádcżyć y w Rzymie.
A gdy był dzień / zƺedłƺy śię niektórzy z Żydów / záwiązáli śię klątwą / mówiąc : Że nie mieli jeść áni pić / áżby Páwłá zábili.
A było ich więcej niż cżterdzieśći / którzy to przyśiężenie ucżynili.
Którzy przyƺedłƺy do przedniejƺych kápłánów / y do ſtárƺych / rzekli : Klątwąśmy śię záwiązáli / że nic nie ukuśimy / áżbyśmy Páwłá zábili.
Przetoż wy teráz dájćie znáć hetmánowi z pozwoleniem wƺyſtkiej rády / áby go jutro do wáſ wywiódł / jákobyśćie śię chćieli doſtátecżniej wywiedzieć o ſpráwách jego / á my / pierwej niż tu przyjdzie / gotowiśmy go zábić.
A gdy uſłyƺáł śioſtrzeniec Páwłá o tej záſádzce / przyƺedł / á wƺedłƺy do obozu / oznájmił to Páwłowi.
Tedy Páweł záwołáwƺy jednego z ſetników / rzekł : Záprowádź tego młodzieńcá do hetmáná / bo mu coś má powiedzieć.
A ták on wziąwƺy go / wiódł go do hetmáná y rzekł : Páweł więzień / záwołáwƺy mię / prośił / ábym tego młodzieńcá przywiódł do ćiebie / któryć má coś powiedzieć.
Tedy hetmán wziąwƺy go zá rękę / y uſtąpiwƺy ná ſtronę / wywiádywáł śię : Cóż to jeſt / co mi máƺ powiedzieć?
A on rzekł : Poſtánowili Żydowie prośić ćię / ábyś jutro wywiódł Páwłá przed rádę / jákoby śię chćieli co doſtátecżniejƺego wywiedzieć o nim.
Ale ty nie pozwáláj im tego ; bo śię náń náſádziło z nich więcej niż cżterdzieśći mężów / którzy śię klątwą záwiązáli / iż nie máją áni jeść áni pić / áżby go zábili ; y ſą już w pogotowiu / cżekájąc od ćiebie odpowiedzi.
Tedy hetmán odpráwił onego młodzieńcá / przykázáwƺy mu / áby tego przed nikim nie powiádáł / iż mu to oznájmił.
A záwołáwƺy dwóch niektórych z ſetników / rzekł : Nágotujćie dwieśćie żołnierzy / áby ƺli áż do Cezáryi ; do tego śiedmdzieśiąt jezdnych y dwieśćie drábántów ná trzećią godzinę w nocy ;
Nágotowáć też bydlętá / áby wſádziwƺy Páwłá ná nie / zdrowo go záprowádzono do Felikſá ſtároſty ;
Nápiſáwƺy liſt w ten ſpoſób :
Kláudyjuƺ Lizyjáƺ nájmożniejƺemu ſtárośćie Felikſowi zdrowiá życży.
Tego mężá pojmánego od Żydów / gdy już od nich miáł być zábity / przypádłƺy z rotą / odjąłem go / dowiedziáwƺy śię / iż jeſt Rzymiáninem.
A chcąc wiedzieć przycżynę / dlá której by náń ſkárżyli / wywiodłem go przed ich rádę ;
Y ználázłem / że náń ſkárżą o jákieś gádki z ſtrony zákonu ich / á że niemá żádnej winy / dlá którejby był godzien śmierći álbo więzieniá.
A gdy mi powiedziáno o záſádzce / którą mieli ucżynić Żydzi ná tego mężá / zárázem go poſłáł do ćiebie / opowiedziáwƺy też tym / co náń ſkárżyli / áby przed tobą mówili to / coby przećiwko niemu mieli. Miej śię dobrze.
Żołnierze tedy ták / jáko im było rozkázáno / wziąwƺy Páwłá / prowádzili go nocą do Antypátrydy.
A názájutrz / zoſtáwiwƺy jezdne / áby z nim jecháli / wróćili śię do obozu.
Którzy przyjecháwƺy do Cezáryi á oddáwƺy liſt ſtárośćie / ſtáwili przed nim y Páwłá.
A ſtároſtá liſt przecżytáwƺy / ſpytáł go / z którejby był kráiny / á zrozumiáwƺy / że był z Cylicyi /
Rzekł : Będę ćię ſłucháł / gdy też przybędą ći / którzy ná ćię ſkárżą. Y rozkázáł go ſtrzedz ná rátuƺu Herodowym.
