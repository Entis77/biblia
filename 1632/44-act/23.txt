A Páweł pilnie pátrząc ná onę rádę rzekł : Mężowie bráćia! ja ze wƺyſtkiego ſumieniá dobrego chodźiłem przed Bogiem áż do dniá tego.
Tedy Anánijáƺ / nájwyżƺy kápłán / rozkázáł go tym / którzy przy nim ſtáli / bić w gębę.
Tedy rzekł Páweł do niego : Uderzy ćię Bóg / śćiáno pobieláná! y ty śiedźiƺ / ſądząc mię według zakonu / á rozkázujeƺ mię bić przećiwko zakonowi?
Zátem ći / którzy tám ſtáli / rzekli : Nájwyżƺemu kápłánowi Bożemu złorzecżyƺ?
A Páweł rzekł : Nie wiedźiáłem / bráćia! żeby był nájwyżƺym kápłánem ; bo nápiſáno : Kśiążęćiu ludu twego złorzecżyć nie będźieƺ.
A poznáwƺy Páweł / że ich jedná cżęść byłá Sáduceuƺów á drugá Fáryzeuƺów / záwołáł w oney rádźie : Mężowie bráćia! jám jeſt Fáryzeuƺ / ſyn Fáryzeuƺá : o nádźieję y o powſtánie umárłych mię tu dźiś ſądzą.
A gdy on to mówił / wƺcżął śię rozruch między Fáryzeuƺámi y Sáduceuƺámi / y rozerwáło śię ono mnóſtwo.
Abowiem Sáduceuƺowie mówią / iż niemáƺ zmartwychwſtániá / áni Aniołá / áni duchá ; ále Fáryzeuƺowie to oboje wyznáwáją.
Y wƺcżęło śię wołánie wielkie. A powſtáwƺy náucżeni w Piśmie z ſtrony Fáryzeuƺów / ſpieráli śię mówiąc : Niceśmy złego nie ználeſli w tym cżłowieku ; y jeſli mu co powiedźiáł duch álbo Anioł / nie wálcżmyż z Bogiem.
A gdy śię wƺcżął wielki rozruch / obáwiájąc śię hetmán / áby Páwłá między ſobą nie rozƺárpáli / rozkázáł iść żołnierzom ná dół / á wydrzeć go z pośrodku ich / y odwieść do obozu.
A drugiej nocy ſtánąwƺy przy nim Pán / rzekł : Bądź dobrego ſercá / Páwle! ábowiem jákoś o mnie świádcżył w Jeruzálemie / ták muśiƺ świádcżyć y w Rzymie.
A gdy był dźień / zƺedłƺy śię niektórzy z Żydów / záwiązáli śię klątwą / mówiąc : Że nie mieli jeść áni pić / áżby Páwłá zábili.
A było ich więcey niż cżterdźieśći / którzy to przyśiężenie ucżynili.
Którzy przyƺedłƺy do przedniejƺych kápłánów / y do ſtárƺych / rzekli : Klątwąśmy śię záwiązáli / że nic nie ukuśimy / áżbyśmy Páwłá zábili.
Przetoż wy teraz dájćie znáć hetmánowi z pozwoleniem wƺyſtkiej rády / áby go jutro do was wywiódł / jákobyśćie śię chćieli doſtátecżniej wywiedźieć o ſpráwách jego / á my / pierwej niż tu przyjdźie / gotowiśmy go zábić.
A gdy uſłyƺáł śioſtrzeniec Páwłá o tey záſádzce / przyƺedł / á wƺedłƺy do obozu / oznájmił to Páwłowi.
Tedy Páweł záwołáwƺy jednego z ſetników / rzekł : Záprowádź tego młodźieńcá do hetmáná / bo mu coś má powiedźieć.
A ták on wźiąwƺy go / wiódł go do hetmáná y rzekł : Páweł więźień / záwołáwƺy mię / prośił / ábym tego młodźieńcá przywiódł do ćiebie / któryć má coś powiedźieć.
Tedy hetmán wźiąwƺy go zá rękę / y uſtąpiwƺy ná ſtronę / wywiádywáł śię : Cóż to jeſt / co mi máƺ powiedźieć?
A on rzekł : Poſtánowili Żydowie prośić ćię / ábyś jutro wywiódł Páwłá przed rádę / jákoby śię chćieli co doſtátecżniejƺego wywiedźieć o nim.
Ale ty nie pozwáláj im tego ; bo śię náń náſádźiło z nich więcey niż cżterdźieśći mężów / którzy śię klątwą záwiązáli / iż nie máją áni jeść áni pić / áżby go zábili ; y ſą już w pogotowiu / cżekájąc od ćiebie odpowiedźi.
Tedy hetmán odpráwił onego młodźieńcá / przykázáwƺy mu / áby tego przed nikim nie powiádáł / iż mu to oznájmił.
A záwołáwƺy dwóch niektórych z ſetników / rzekł : Nágotujćie dwieśćie żołnierzy / áby ƺli áż do Cezáryi ; do tego śiedmdźieśiąt jezdnych y dwieśćie drábántów ná trzećią godźinę w nocy ;
Nágotowáć też bydlętá / áby wſádźiwƺy Páwłá ná nie / zdrowo go záprowádzono do Felikſá ſtároſty ;
Nápiſáwƺy liſt w ten ſpoſób :
Kláudyjuƺ Lizyjáƺ nájmożniejƺemu ſtárośćie Felikſowi zdrowiá życży.
Tego mężá pojmánego od Żydów / gdy już od nich miał być zábity / przypádłƺy z rotą / odjąłem go / dowiedźiáwƺy śię / iż jeſt Rzymiáninem.
A chcąc wiedźieć przycżynę / dla którey by náń ſkárżyli / wywiodłem go przed ich rádę ;
Y ználázłem / że náń ſkárżą o jákieś gádki z ſtrony zakonu ich / á że niemá żádney winy / dla którejby był godźien śmierći álbo więźieniá.
A gdy mi powiedźiáno o záſádzce / którą mieli ucżynić Żydźi ná tego mężá / zárázem go poſłáł do ćiebie / opowiedźiáwƺy też tym / co náń ſkárżyli / áby przed tobą mówili to / coby przećiwko niemu mieli. Miej śię dobrze.
Żołnierze tedy ták / jáko im było rozkázáno / wźiąwƺy Páwłá / prowádźili go nocą do Antypátrydy.
A názájutrz / zoſtáwiwƺy jezdne / áby z nim jecháli / wróćili śię do obozu.
Którzy przyjecháwƺy do Cezáryi á oddáwƺy liſt ſtárośćie / ſtáwili przed nim y Páwłá.
A ſtároſtá liſt przecżytáwƺy / ſpytáł go / z którejby był kráiny / á zrozumiáwƺy / że był z Cylicyi /
Rzekł : Będę ćię ſłucháł / gdy też przybędą ći / którzy ná ćię ſkárżą. Y rozkázáł go ſtrzedz ná rátuƺu Herodowym.
