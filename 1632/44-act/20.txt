A gdy śię on rozruch ućiƺył / zwołáwƺy Páweł ucżniów / y z nimi śię pożegnáwƺy / wyƺedł ſtámtąd / áby ƺedł do Mácedonii.
A przeƺedłƺy one ſtrony / y nápomniáwƺy je ƺerokiemi ſłowy / przyƺedł do Grecyi.
A tám zámieƺkáwƺy przez trzy mieśiące / gdzie náń Żydowie záſádzkę ucżynili / gdy miáł płynąć do Syryi / umyślił śię powróćić przez Mácedoniję.
Y puśćił śię z nim áż do Azyi Sopáter / Bereeńcżyk / á z Teſálońcżyków Aryſtárchuſ y Sekunduſ y Gájuſ Derbejcżyk y Tymoteuƺ ;
A z Azyjátcżyków Tychykuſ y Trofimuſ / którzy wprzód poƺedłƺy / cżekáli náſ w Troádzie.
A my po dniách przáśników odpłynęliśmy z Filipowá / y przyƺliśmy do nich do Troády zá pięć dni / gdzieśmy zámieƺkáli śiedm dni.
Tedy pierwƺy dzień po ſábáćie / gdy śię ucżniowie zgromádzili ná łámánie chlebá / Páweł rozmáwiáł z nimi / májąc iść precż názájutrz / y przedłużył mowę áż do północy.
A było wiele lámp ná onej ſáli / gdzie byli zgromádzeni.
Tám śiedząc niektóry młodzieniec / imieniem Eutychuſ / w oknie / będąc ćiężkim ſnem zdjęty / gdy ták Páweł długo mówił / ſnem zmorzony pádł ná dół z trzećiego piętrá / y podnieśiony jeſt umárły.
A Páweł zſtąpiwƺy ná dół / przypádł náń / á ująwƺy go / rzekł : Nie trwożćie śię ; boć w nim jeſt duƺá jego.
A wſtąpiwƺy záśię / łámáł chleb / y jádł / y kázáł im długo áż do świtániá ; potem odƺedł precż.
Y przywiedli onego młodzieńcá żywego / y byli náder ućieƺeni.
A my przyƺedłƺy wprzód do okrętu / puśćiliśmy śię do Aſſonu / ábyśmy ſtámtąd wzięli Páwłá ; ábowiem ták był poſtánowił / májąc ſám pieƺo iść.
A gdy śię z námi zƺedł w Aſſonie / wziąwƺy go / przyjecháliśmy do Mityleny.
A ſtámtąd odpłynąwƺy / drugiego dniá przyƺliśmy przećiw Chyju / á trzećiego dniá przypłynęliśmy do Sámu / á pomieƺkáwƺy w Trogillu / názájutrz przyƺliśmy do Miletu.
Abowiem Páweł umyślił był minąć Efez / áby mu nie przyƺło cżáſu tráwić w Azyi / bo śię kwápił / jeźliby mu możná / áby ná dzień świątecżny był w Jeruzálemie.
Tedy z Miletu poſłáwƺy do Efezu / przyzwáł do śiebie ſtárƺych zborowych.
Którzy gdy do niego przyƺli / rzekł im : Wy wiećie od pierwƺego dniá / któregom przyƺedł do Azyi / jákom z wámi po wƺyſtek cżáſ był /
Służąc Pánu ze wƺelką uniżonośćią y z wielą łez y pokuſ / które ná mię przychádzáły z záſádzek żydowſkich.
Jákom śię nie ſchrániáł nicżego / coby było pożytecżne / ábym wám nie oznájmił / y nie ucżył wáſ jáwnie y po domách.
Świádectwo wydáwájąc y Żydom y Grekom o pokućie ku Bogu y o wierze w Páná náƺego Jezuſá Chryſtuſá.
A oto teráz já będąc związány duchem / idę do Jeruzálemu / nie wiedząc / co tám ná mię przyjść má.
Tylko że Duch Święty po miáſtách świádcży / powiádájąc / że mnie więzienie y ućiſki cżekáją.
Wƺákże já ná nic nie dbám / y nie jeſt mi ták drogá duƺá mojá / bym tylko bieg mój z rádośćią wykonáł / y poſługę / którąm wziął od Páná Jezuſá / ná oświádcżenie Ewángielii łáſki Bożej.
A teráz oto já wiem / że już więcej nie oglądáćie oblicżá mojego wy wƺyſcy / między którymim chodził / káżąc króleſtwo Boże.
Przetoż oświádcżám śię wám dniá dziśiejƺego / żem já jeſt cżyſty od krwi wƺyſtkich.
Abowiem nie chroniłem śię / żebym wám nie miáł oznájmić wƺelkiej rády Bożej.
Pilnujćież tedy ſámych śiebie / y wƺyſtkiej trzody / w której wáſ Duch Święty poſtánowił biſkupámi / ábyśćie páśli zbór Boży / którego nábył przez włáſną krew.
Boć já to wiem / że po odejśćiu mojem wnijdą między wáſ wilcy okrutni / którzy trzodzie folgowáć nie będą.
A z wáſ ſámych powſtáną mężowie / mówiący rzecży przewrotne / áby zá ſobą poćiągnęli ucżniów.
Przetoż cżujćie / pomnąc / żem przez trzy látá w nocy y we dnie nie przeſtáwáł nápomináć ze łzámi káżdego z wáſ.
A teráz / bráćiá! porucżám wáſ Bogu y ſłowu łáſki jego / który może pobudowáć y dáć wám dziedzictwo między wƺyſtkimi poświęconymi.
Srebrá álbo złotá / álbo ƺáty nie pożądáłem od nikogo.
Owƺem ſámi wiećie / że moim potrzebom / y tych / którzy ſą zemną / ſłużyły te ręce /
Wƺyſtkomći wám okázáł / iż ták prácując / muśimy podejmowáć ſłábe / á pámiętáć ná ſłowá Páná Jezuſowe / że on rzekł : Szcżęśliwƺá jeſt rzecż dáwáć / niżeli bráć.
A to powiedziáwƺy / klęknął ná koláná ſwoje / y modlił śię z nimi wƺyſtkimi.
Y ſtáł śię wielki płácż wƺyſtkich / á upádájąc ná ƺyję Páwłową / cáłowáli go ;
Smucąc śię bárdzo / nájwięcej tych ſłów / które im rzekł / że już więcej nie mieli oglądáć oblicżá jego. Y prowádzili go do okrętu.
