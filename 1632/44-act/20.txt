A gdy śię on rozruch ućiƺył / zwołáwƺy Páweł ucżniów / y z nimi śię pożegnáwƺy / wyƺedł ſtámtąd / áby ƺedł do Mácedonii.
A przeƺedłƺy one ſtrony / y nápomniáwƺy je ƺerokiemi ſłowy / przyƺedł do Grecyi.
A tám zámieƺkáwƺy przez trzy mieśiące / gdźie náń Żydowie záſádzkę ucżynili / gdy miał płynąć do Syryi / umyślił śię powróćić przez Mácedoniję.
Y puśćił śię z nim áż do Azyi Sopáter / Bereeńcżyk / á z Teſálońcżyków Aryſtárchuſ y Sekunduſ y Gájuſ Derbejcżyk y Tymoteuƺ ;
A z Azyjátcżyków Tychykuſ y Trofimuſ / którzy wprzód poƺedłƺy / cżekáli nas w Troádźie.
A my po dniách przáśników odpłynęliśmy z Filipowá / y przyƺliśmy do nich do Troády zá pięć dni / gdźieśmy zámieƺkáli śiedm dni.
Tedy pierwƺy dźień po ſábáćie / gdy śię ucżniowie zgromádźili ná łámánie chlebá / Páweł rozmáwiáł z nimi / májąc iść precż názájutrz / y przedłużył mowę áż do północy.
A było wiele lámp ná oney ſáli / gdźie byli zgromádzeni.
Tám śiedząc niektóry młodźieniec / imieniem Eutychuſ / w oknie / będąc ćiężkim ſnem zdjęty / gdy ták Páweł długo mówił / ſnem zmorzony pádł ná dół z trzećiego piętrá / y podnieśiony jeſt umárły.
A Páweł zſtąpiwƺy ná dół / przypádł náń / á ująwƺy go / rzekł : Nie trwożćie śię ; boć w nim jeſt duƺá jego.
A wſtąpiwƺy záśię / łámáł chleb / y jádł / y kázáł im długo áż do świtániá ; potem odƺedł precż.
Y przywiedli onego młodźieńcá żywego / y byli náder ućieƺeni.
A my przyƺedłƺy wprzód do okrętu / puśćiliśmy śię do Aſſonu / ábyśmy ſtámtąd wźięli Páwłá ; ábowiem ták był poſtánowił / májąc ſám pieƺo iść.
A gdy śię z námi zƺedł w Aſſonie / wźiąwƺy go / przyjecháliśmy do Mityleny.
A ſtámtąd odpłynąwƺy / drugiego dniá przyƺliśmy przećiw Chyju / á trzećiego dniá przypłynęliśmy do Sámu / á pomieƺkáwƺy w Trogillu / názájutrz przyƺliśmy do Miletu.
Abowiem Páweł umyślił był minąć Efez / áby mu nie przyƺło cżáſu tráwić w Azyi / bo śię kwápił / jeſliby mu możná / áby ná dźień świątecżny był w Jeruzálemie.
Tedy z Miletu poſłáwƺy do Efezu / przyzwáł do śiebie ſtárƺych zborowych.
Którzy gdy do niego przyƺli / rzekł im : Wy wiećie od pierwƺego dniá / któregom przyƺedł do Azyi / jákom z wámi po wƺyſtek cżáſ był /
Służąc Pánu ze wƺelką uniżonośćią y z wielą łez y pokuſ / które ná mię przychádzáły z záſádzek żydowſkich.
Jákom śię nie ſchrániáł nicżego / coby było pożytecżne / ábym wám nie oznájmił / y nie ucżył was jáwnie y po domách.
Świádectwo wydáwájąc y Żydom y Grekom o pokućie ku Bogu y o wierze w Páná náƺego JEzuſá CHryſtuſá.
A oto teraz ja będąc związány duchem / idę do Jeruzálemu / nie wiedząc / co tám ná mię przyjść má.
Tylko że Duch Święty po miáſtách świádcży / powiádájąc / że mnie więźienie y ućiſki cżekáją.
Wƺákże ja ná nic nie dbám / y nie jeſt mi ták drogá duƺá mojá / bym tylko bieg mój z rádośćią wykonáł / y poſługę / którąm wźiął od Páná JEzuſá / ná oświádcżenie Ewángielii łáſki Bożej.
A teraz oto ja wiem / że już więcey nie oglądáćie oblicżá mojego wy wƺyſcy / między którymim chodźił / káżąc króleſtwo Boże.
Przetoż oświádcżám śię wám dniá dźiśiejƺego / żem ja jeſt cżyſty od krwi wƺyſtkich.
Abowiem nie chroniłem śię / żebym wám nie miał oznájmić wƺelkiej rády Bożej.
Pilnujćież tedy ſámych śiebie / y wƺyſtkiej trzody / w którey was Duch Święty poſtánowił biſkupámi / ábyśćie páśli zbór Boży / którego nábył przez włáſną krew.
Boć ja to wiem / że po odejśćiu mojem wnijdą między was wilcy okrutni / którzy trzodźie folgowáć nie będą.
A z was ſámych powſtáną mężowie / mówiący rzecży przewrotne / áby zá ſobą poćiągnęli ucżniów.
Przetoż cżujćie / pomnąc / żem przez trzy látá w nocy y we dnie nie przeſtáwáł nápomináć ze łzámi káżdego z was.
A teraz / bráćia! porucżám was Bogu y ſłowu łáſki jego / który może pobudowáć y dáć wám dźiedźictwo między wƺyſtkimi poświęconymi.
Srebrá álbo złotá / álbo ƺáty nie pożądáłem od nikogo.
Owƺem ſámi wiećie / że moim potrzebom / y tych / którzy ſą zemną / ſłużyły te ręce /
Wƺyſtkomći wám okázáł / iż ták prácując / muśimy podejmowáć ſłábe / á pámiętáć ná ſłowá Páná JEzuſowe / że on rzekł : Szcżęśliwƺá jeſt rzecż dáwáć / niżeli bráć.
A to powiedźiáwƺy / klęknął ná koláná ſwoje / y modlił śię z nimi wƺyſtkimi.
Y ſtáł śię wielki płácż wƺyſtkich / á upádájąc ná ƺyję Páwłową / cáłowáli go ;
Smucąc śię bárdzo / nájwięcey tych ſłów / które im rzekł / że już więcey nie mieli oglądáć oblicżá jego. Y prowádźili go do okrętu.
