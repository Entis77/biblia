A Saul jeƺcże dychájąc groźbą y morderſtwem przećiwko ucżniom Páńſkim / przyƺedł do nájwyżƺego kápłáná /
Y prośił go o liſty do Dámáƺku do bóżnic / iż jeſliby tám ználázł tey drogi ( tą drogą idących / ) których mężów álbo niewiáſty / áby ich związáne przywiódł do Jeruzálemu.
A gdy jecháł / ſtáło śię / gdy śię przybliżáł do Dámáƺku / że z prędká oświećiłá go świátłość z niebá.
A pádłƺy ná źiemię / uſłyƺáł głoſ do śiebie mówiący : Saulu! Saulu! przecżże mię prześládujeƺ?
Tedy rzekł : Ktoś jeſt / Pánie? A Pán rzekł : Jám jeſt JEzuſ / którego ty prześládujeƺ ; trudno tobie przećiw ośćieniowi wierzgáć.
A Saul drżąc y bojąc śię / rzekł : Pánie! co chceƺ / ábym ja ucżynił? A Pán do niego : Wſtáń / á wnijdź do miáſtá / á tám ći powiedzą / cobyś ty miał cżynić.
A mężowie / którzy z nim byli w drodze / ſtánęli / zdumiáwƺy śię ; głoſ tylko ſłyƺąc / ále nikogo nie widząc.
Y wſtáł Saul z źiemi / á otworzywƺy ocży ſwoje / nikogo nie widźiáł. Tedy ująwƺy go zá rękę ; prowádźili go do Dámáƺku /
Kędy był trzy dni nie widząc / y nie jádł áni pił.
A był niektóry ucżeń w Dámáƺku / imieniem Anánijáƺ ; y rzekł Pán do niego w widzeniu : Anánijáƺu! A on rzekł : Otom ja / Pánie!
A Pán rzekł do niego : Wſtáń á idź ná ulicę / którą zowią proſtą / á ƺukáj w domu Judowym Saulá imieniem Tárſeńcżyká ; ábowiem oto śię modli.
Y widźiáł w widzeniu mężá / imieniem Anánijáƺá / wchodzącego / y rękę ná śię wkłádájącego / áby przejrzáł.
Y odpowiedźiáł Anánijáƺ : Pánie! ſłyƺáłem od wielu o tym mężu / jáko wiele złego cżynił świętym twoim w Jeruzálemie.
Y tu má moc od nájwyżƺych kápłánów / áby wiązáł wƺyſtkie wzywájące imieniá twego.
Y rzekł do niego Pán : Idźże ; ábowiem mi ten jeſt nácżyniem wybránem / áby nośił imię moje przed pogány y królmi y przed ſyny Izráelſkimi.
Abowiem ja mu ukáżę / jáko wiele muśi ćierpieć dla imieniá mego.
Y poƺedł Anánijáƺ / y wƺedł do onego domu / á włożywƺy náń ręce / rzekł : Saulu bráćie! Pán mię poſłáł / JEzuſ on / któryć śię ukázáł w drodze / którąś jecháł / ábyś przejrzáł / á był nápełniony Duchem Świętym.
Y zárázem ſpádły z ocżów jego jáko łuſki / y wnet przejrzáł / á wſtáwƺy ochrzcżony jeſt.
A wźiąwƺy pokárm / pośilił śię. Y był Saul z ucżniámi / którzy byli w Dámáƺku / kilká dni.
Y záráz kázáł w bóżnicách CHryſtuſá / że on jeſt Synem Bożym.
Y zdumiewáli śię wƺyſcy / którzy go ſłucháli / y mówili : Izáli to nie jeſt ten / który burzył w Jeruzálemie tych / którzy wzywáli imieniá tego? y tuć ná to przyƺedł / áby ich związáwƺy / wiódł do nájwyżƺych kápłánów?
A Saul tem więcey zmácniáł śię y záwſtydzáł Żydy / którzy mieƺkáli w Dámáƺku / dowodząc / iż ten jeſt CHryſtuſ.
A gdy przeƺło nie máło dni / urádźili Żydowie między ſobą / áby go zábili.
Ale śię dowiedźiáł Saul o záſádzce ich. Strzegli też brám we dnie y w nocy / áby go zábili.
Lecż ucżniowie wźiąwƺy go w nocy / ſpuśćili go po powroźie przez mur w koƺu.
A gdy przyƺedł Saul do Jeruzálemu / kuśił śię przyłącżyć do ucżniów ; ále śię go wƺyſcy báli / nie wierząc / áby był ucżniem.
Lecż Bárnábáƺ wźiąwƺy go / przywiódł go do Apoſtołów / y powiádáł im / jáko w drodze widźiáł Páná / á iż mówił do niego / y jáko w Dámáƺku bezpiecżnie mówił w imieniu JEzuſowem.
Y mieƺkáł z nimi w Jeruzálemie.
A bezpiecżnie ſobie pocżynájąc w imieniu Páná JEzuſowem / mówił y gádáł z Grekámi ; á oni śię ſtáráli / jákoby go zábić.
O cżem dowiedźiáwƺy śię bráćia / odprowádźili go do Cezáryi ; y odeſłáli go do Tárſu.
A ták zbory po wƺyſtkiej Judzkiej źiemi y Gálilei / y Sámáryi miáły pokój / budując śię / y chodząc w bojáźni Páńſkiej / á przez poćiechę Duchá Świętego rozmnáżáły śię.
Y ſtáło śię / gdy Piotr obchodźił wƺyſtkie / przyƺedł też do Świętych / którzy mieƺkáli w Liddźie.
Támże ználázł cżłowieká niektórego / imieniem Eneáƺá / od ośmiu lát ná łożu leżącego / który był powietrzem ruƺony.
Y rzekł mu Piotr : Eneáƺu! uzdráwiá ćię JEzuſ CHryſtuſ ; wſtáńże / á pośćiel ſobie. Y zárázem wſtáł.
A widźieli go wƺyſcy / którzy mieƺkáli w Liddźie y w Sáronie / którzy śię náwróćili do Páná.
A byłá w Joppie niektóra ucżennicá / imieniem Tábitá / która / wyłożywƺy / zowie śię Dorká ; tá byłá pełná dobrych ucżynków y jáłmużny / które cżyniłá.
Y ſtáło śię w one dni / że rozniemógłƺy śię / umárłá ; którą omywƺy / położyli ná ſáli.
A iż Liddá byłá bliſko Joppy / ucżniowie uſłyƺáwƺy / że tám jeſt Piotr / poſłáli do niego dwóch mężów / proƺąc go / áby śię nie lenił przyjść do nich.
Tedy wſtáwƺy Piotr / ƺedł z nimi ; á gdy przyƺedł / wprowádźili go ná ſálę / y obſtąpiły go wƺyſtkie wdowy / płácżąc y ukázując ſuknie y płáƺcże / które im Dorká robiłá / póki byłá z nimi.
A Piotr wygnáwƺy precż wƺyſtkie / klęknął ná koláná y modlił śię / á obróćiwƺy śię do onego ćiáłá / rzekł : Tábito / wſtáń! á oná otworzyłá ocży ſwoje / y ujrzáwƺy Piotrá / uśiádłá.
A on podáwƺy jey rękę / podnióſł ją / á záwołáwƺy świętych y wdów / ſtáwił ją żywą.
Y rozſłáwiło śię to po wƺyſtkiej Joppie / y wiele ich uwierzyło w Páná.
Y ſtáło śię / że przez wiele dni zoſtáł Piotr w Joppie u niejákiego Szymoná / gárbárzá.
