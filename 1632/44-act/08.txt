A Saul zezwolił ná zábićie jego. Y wƺcżęło śię onegoż cżáſu wielkie prześládowánie przećiwko zborowi / który był w Jeruzálemie / y rozproƺyli śię wƺyſcy po kráinách ziemi Judzkiej y Sámáryi / oprócż Apoſtołów.
Y pogrzebli Szcżepáná mężowie bogobojni / y ucżynili nád nim płácż wielki.
A Saul niƺcżył zbór / wchodząc w domy / á wywłócżájąc męże y niewiáſty / podáwáł je do więzieniá.
A ći / którzy byli rozproƺeni / chodzili opowiádájąc ſłowo Boże.
Lecż Filip záƺedłƺy do miáſtá Sámáryjſkiego / opowiádáł im Chryſtuſá.
A lud miáł wzgląd jednomyślnie ná to / co Filip mówił / ſłuchájąc y widząc cudá / które cżynił.
Abowiem duchy niecżyſte od wielu tych / którzy je mieli / wołájąc głoſem wielkim wychodziły / á wiele powietrzem ruƺonych y chromych uzdrowieni ſą.
Y ſtáłá śię rádość wielká w onem mieśćie.
A niektóry mąż / imieniem Szymon / był przedtem w onem mieśćie báwiący śię náuką cżárnokśięſką / y lud Sámáryjſki mámił / powiádájąc śię być cżymćiś wielkim.
Ná którego śię oglądáli wƺyſcy od nájmniejƺego áż do nájwiękƺego / mówiąc : Tenći jeſt oná moc Bożá wielká.
A oglądáli śię náń przeto / iż je od niemáłego cżáſu mámił cżárnokśięſtwy ſwemi.
A gdy uwierzyli Filipowi / opowiádájącemu króleſtwo Boże / y imię Jezuſá Chryſtuſá / chrzćili śię mężowie y niewiáſty.
Tedy y ſám Szymon uwierzył / á ochrzćiwƺy śię / trzymáł śię Filipá / á widząc cudá y mocy wielkie / które śię dziáły / zdumiewáł śię.
A uſłyƺáwƺy Apoſtołowie / którzy byli w Jeruzálemie / iż Sámáryjá przyjęłá ſłowo Boże / poſłáli do nich Piotrá y Jáná.
Którzy tám przyƺedłƺy / modlili śię zá nimi / áby przyjęli Duchá Świętego.
( Abowiem jeƺcże był ná żádnego z nich nie zſtąpił ; ále tylko pochrzcżeni byli w imię Páná Jezuſowe. )
Tedy ná nie wkłádáli ręce / á oni przyjmowáli Duchá Świętego.
A ujrzáwƺy Szymon / że przez wkłádánie rąk Apoſtolſkich był dáwány Duch Święty / przynióſł im pieniądze.
Mówiąc : Dájćie y mnie tę moc / áby ten / ná któregobymkolwiek ręce włożył / wziął Duchá Świętego.
Y rzekł mu Piotr : Pieniądze twoje niech z tobą będą ná zginienie / żeś mniemáł / żeby dár Boży miáł być zá pieniądze nábywány.
Niemáƺ w tej rzecży cżąſtki / áni loſu / gdy ſerce twoje nie jeſt proſte przed oblicżem Bożem.
Przetoż pokutuj z tej twojej złośći / á proś Bogá ; owáć ſnáć będzie odpuƺcżony ten zámyſł ſercá twego.
Abowiem ćię widzę być w gorzkośći żółći / y w związce niepráwośći.
Odpowiedziáwƺy tedy Szymon / rzekł : Módlćie śię wy zá mną Pánu / áby ná mię nic nie przyƺło z tych rzecży / któreśćie powiedzieli.
A ták oni oświádcżywƺy y opowiedziáwƺy ſłowo Páńſkie / wróćili śię do Jeruzálemu / y w wielu miáſtecżkách Sámárytáńſkich Ewángieliję opowiádáli.
Lecż Anioł Páńſki rzekł do Filipá / mówiąc : Wſtáń / á idź ku południowi ná drogę / którá od Jeruzálemu idzie ku Gázie / którá jeſt puſtá.
A on wſtáwƺy / ƺedł. A oto mąż Murzyn rzezániec / komornik królowej murzyńſkiej Kándáceſ / który był nád wƺyſtkiemi ſkárbámi jej / á przyjecháł był do Jeruzálemu / áby śię modlił ;
Y wrácáł śię / śiedząc ná wozie ſwoim / á cżytáł Izájáƺá proroká.
Y rzekł Duch Filipowi : Przyſtąp / á przyłącż śię do tego wozu.
A przybieżáwƺy Filip / uſłyƺáł go cżytájącego Izájáƺá proroká / y rzekł : Rozumieƺże / co cżytáƺ?
A on rzekł : Jákoż mogę rozumieć / jeźliby mi kto nie wyłożył? Y prośił Filipá / á wſtąpił y śiedziáł z nim.
A miejſce onego Piſmá / które cżytáł / to było : Jáko owcá ku zábićiu wiedziony jeſt / á jáko báránek niemy przed tym / który go ſtrzyże / ták nie otworzył uſt ſwoich ;
W uniżeniu jego / ſąd jego znieśiony jeſt / á rodzáj jego / któż wypowie? ábowiem znieśiony był z ziemi żywot jego.
A odpowiádájąc rzezániec Filipowi / rzekł : Proƺę ćię / o kim to prorok mówi? Sám o ſobie / cżyli o kim innym?
Tedy otworzywƺy Filip uſtá ſwe / á pocżąwƺy od tego Piſmá / opowiádáł mu Jezuſá.
A gdy jecháli drogą / przyjecháli nád jednę wodę. Tedy rzekł rzezániec : Otóż wodá! Cóż ná przeƺkodzie / ábym nie miáł być ochrzcżony?
Y rzekł Filip : Jeźliż wierzyƺ z cáłego ſercá / wolnoć. A on odpowiedziáwƺy / rzekł : Wierzę / iż Jezuſ Chryſtuſ jeſt Syn Boży.
Y kázáł ſtánąć wozowi ; y zſtąpili obádwáj w wodę / Filip y rzezániec / y ochrzćił go.
A gdy wyſtąpili z wody / porwáł Filipá Duch Páńſki / y nie widziáł go więcej rzezániec / ále jecháł drogą ſwoją / rádując śię.
A Filip áż w Azoćie jeſt ználeziony / á chodząc kázáł Ewángieliją po wƺyſtkich miáſtách / áż przyƺedł do Cezáryi.
