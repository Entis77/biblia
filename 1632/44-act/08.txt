A Saul zezwolił ná zábićie jego. Y wƺcżęło śię onegoż cżáſu wielkie prześládowánie przećiwko zborowi / który był w Jeruzálemie / y rozproƺyli śię wƺyſcy po kráinách źiemi Judzkiej y Sámáryi / oprócż Apoſtołów.
Y pogrzebli Szcżepáná mężowie bogobojni / y ucżynili nád nim płácż wielki.
A Saul niƺcżył zbór / wchodząc w domy / á wywłócżájąc męże y niewiáſty / podáwáł je do więźieniá.
A ći / którzy byli rozproƺeni / chodźili opowiádájąc ſłowo Boże.
Lecż Filip záƺedłƺy do miáſtá Sámáryjſkiego / opowiádáł im CHryſtuſá.
A lud miał wzgląd jednomyślnie ná to / co Filip mówił / ſłuchájąc y widząc cudá / które cżynił.
Abowiem duchy niecżyſte od wielu tych / którzy je mieli / wołájąc głoſem wielkim wychodźiły / á wiele powietrzem ruƺonych y chromych uzdrowieni ſą.
Y ſtáłá śię rádość wielká w onem mieśćie.
A niektóry mąż / imieniem Szymon / był przedtem w onem mieśćie báwiący śię náuką cżárnokśięſką / y lud Sámáryjſki mámił / powiádájąc śię być cżymćiś wielkim.
Ná którego śię oglądáli wƺyſcy od nájmniejƺego áż do nájwiękƺego / mówiąc : Tenći jeſt oná moc Bożá wielká.
A oglądáli śię náń przeto / iż je od niemáłego cżáſu mámił cżárnokśięſtwy ſwemi.
A gdy uwierzyli Filipowi / opowiádájącemu króleſtwo Boże / y imię JEzuſá CHryſtuſá / chrzćili śię mężowie y niewiáſty.
Tedy y ſám Szymon uwierzył / á ochrzćiwƺy śię / trzymáł śię Filipá / á widząc cudá y mocy wielkie / które śię dźiáły / zdumiewáł śię.
A uſłyƺáwƺy Apoſtołowie / którzy byli w Jeruzálemie / iż Sámáryjá przyjęłá ſłowo Boże / poſłáli do nich Piotrá y Jáná.
Którzy tám przyƺedłƺy / modlili śię zá nimi / áby przyjęli Duchá Świętego.
( Abowiem jeƺcże był ná żádnego z nich nie zſtąpił ; ále tylko pochrzcżeni byli w imię Páná JEzuſowe. )
Tedy ná nie wkłádáli ręce / á oni przyjmowáli Duchá Świętego.
A ujrzáwƺy Szymon / że przez wkłádánie rąk Apoſtolſkich był dáwány Duch Święty / przynióſł im pieniądze.
Mówiąc : Dájćie y mnie tę moc / áby ten / ná któregobymkolwiek ręce włożył / wźiął Duchá Świętego.
Y rzekł mu Piotr : Pieniądze twoje niech z tobą będą ná zginienie / żeś mniemáł / żeby dár Boży miał być zá pieniądze nábywány.
Niemáƺ w tey rzecży cżąſtki / áni loſu / gdy ſerce twoje nie jeſt proſte przed oblicżem Bożem.
Przetoż pokutuj z tey twojey złośći / á proś Bogá ; owáć ſnáć będźie odpuƺcżony ten zámyſł ſercá twego.
Abowiem ćię widzę być w gorzkośći żółći / y w związce niepráwośći.
Odpowiedźiáwƺy tedy Szymon / rzekł : Módlćie śię wy zá mną Pánu / áby ná mię nic nie przyƺło z tych rzecży / któreśćie powiedźieli.
A ták oni oświádcżywƺy y opowiedźiáwƺy ſłowo Páńſkie / wróćili śię do Jeruzálemu / y w wielu miáſtecżkách Sámárytáńſkich Ewángieliję opowiádáli.
Lecż Anioł Páńſki rzekł do Filipá / mówiąc : Wſtáń / á idź ku południowi ná drogę / która od Jeruzálemu idźie ku Gáźie / która jeſt puſtá.
A on wſtáwƺy / ƺedł. A oto mąż Murzyn rzezániec / komornik królowej murzyńſkiej Kándáceſ / który był nád wƺyſtkiemi ſkárbámi jey / á przyjecháł był do Jeruzálemu / áby śię modlił ;
Y wrácáł śię / śiedząc ná woźie ſwoim / á cżytáł Izájáƺá proroká.
Y rzekł Duch Filipowi : Przyſtąp / á przyłącż śię do tego wozu.
A przybieżáwƺy Filip / uſłyƺáł go cżytájącego Izájáƺá proroká / y rzekł : Rozumieƺże / co cżytáƺ?
A on rzekł : Jákoż mogę rozumieć / jeſliby mi kto nie wyłożył? Y prośił Filipá / á wſtąpił y śiedźiáł z nim.
A miejſce onego Piſmá / które cżytáł / to było : Jáko owcá ku zábićiu wiedźiony jeſt / á jáko báránek niemy przed tym / który go ſtrzyże / ták nie otworzył uſt ſwoich ;
W uniżeniu jego / ſąd jego znieśiony jeſt / á rodzáj jego / któż wypowie? ábowiem znieśiony był z źiemi żywot jego.
A odpowiádájąc rzezániec Filipowi / rzekł : Proƺę ćię / o kim to prorok mówi? Sám o ſobie / cżyli o kim innym?
Tedy otworzywƺy Filip uſtá ſwe / á pocżąwƺy od tego Piſmá / opowiádáł mu JEzuſá.
A gdy jecháli drogą / przyjecháli nád jednę wodę. Tedy rzekł rzezániec : Otóż wodá! Cóż ná przeƺkodźie / ábym nie miał być ochrzcżony?
Y rzekł Filip : Jeſliż wierzyƺ z cáłego ſercá / wolnoć. A on odpowiedźiáwƺy / rzekł : Wierzę / iż JEzuſ CHryſtuſ jeſt Syn Boży.
Y kázáł ſtánąć wozowi ; y zſtąpili obádwáj w wodę / Filip y rzezániec / y ochrzćił go.
A gdy wyſtąpili z wody / porwáł Filipá Duch Páńſki / y nie widźiáł go więcey rzezániec / ále jecháł drogą ſwoją / rádując śię.
A Filip áż w Azoćie jeſt ználeźiony / á chodząc kázáł Ewángieliją po wƺyſtkich miáſtách / áż przyƺedł do Cezáryi.
