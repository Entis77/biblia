Pierwƺe wpráwdzie kśięgi nápiſáłem / o Teofile! o wƺyſtkiem / co pocżął Jezuſ y cżynić y ucżyć.
Aż do dniá onego / którego dáwƺy rozkázánie Apoſtołom / które był przez Duchá Świętego obráł / wzięty jeſt w górę.
Którym też ſámego śiebie po męce ſwojej żywym ſtáwił w wielu niewątpliwych dowodách / przez cżterdzieśći dni ukázując śię im y mówiąc o króleſtwie Bożem.
A zgromádziwƺy je / przykázáł im / áby nie odchodzili z Jeruzálemu / ále iżby cżekáli obietnicy ojcowſkiej / o którejśćie mówić ſłyƺeli odemnie.
Abowiemći Ján chrzćił wodą ; ále wy będziećie ochrzcżeni Duchem Świętym po nie wielu tych dniách.
A ták oni zƺedłƺy śię / pytáli go mówiąc : Pánie! izáli w tym cżáśie nápráwiƺ króleſtwo Izráelſkie?
Lecż on rzekł do nich : Nie wáƺá rzecż jeſt / znáć cżáſy y chwile / które Ojćiec w ſwojej mocy położył.
Ale przyjmiećie moc Duchá Świętego / który przyjdzie ná wáſ ; y będziećie mi świádkámi y w Jeruzálemie / y we wƺyſtkiej Judzkiej ziemi / y w Sámáryi / áż do oſtátniego kráju ziemi.
A to rzekłƺy / gdy oni pátrzáli w górę / podnieśiony jeſt á obłok wziął go od ocżów ich.
A gdy zá nim do niebá idącym pilnie pátrzáli / oto dwáj mężowie ſtánęli przy nich w biáłem odzieniu /
Y rzekli : Mężowie Gálilejſcy! przecż ſtoićie / pátrząc w niebo? Ten Jezuſ / który w górę wzięty jeſt od wáſ do niebá / ták przyjdzie / jákośćie go widzieli idącego do niebá.
Tedy śię wróćili do Jeruzálemu od góry / którą zowią oliwną / którá jeſt bliſko Jeruzálemu / májąc drogi przez jeden ſábát.
A gdy weƺli / wſtąpili ná ſálę / gdzie mieƺkáli Piotr / y Jákób / y Ján / y Andrzej / y Filip / y Tomáƺ / Bártłomiej / y Máteuƺ / Jákób Alfeuƺowy / y Szymon Zeloteſ / y Judáſ Jákóbowy.
Ci wƺyſcy trwáli jednomyślnie ná modlitwie y prośbách / z żonámi y z Máryją / mátką Jezuſową / y z bráćmi jego.
A w oneż dni / powſtáwƺy Piotr w pośrodku ucżniów / rzekł : ( A był pocżet oſób weſpół zgromádzonych około ſtá y dwudzieſtu. )
Mężowie bráćiá! muśiáło śię wypełnić ono piſmo / które opowiedziáł Duch Święty przez uſtá Dawidowe o Judáƺu / który był wodzem tych / co pojmáli Jezuſá ;
Bo był policżony z námi / y doſtáł był cżąſtki tego uſługiwániá.
Tenći wpráwdzie otrzymáł rolę z zápłáty nieſpráwiedliwośći / á powieśiwƺy śię / rozpukł śię ná poły / y wypłynęły wƺyſtkie wnętrznośći jego.
Y było to jáwne wƺyſtkim mieƺkájącym w Jeruzálemie / ták iż názwáno onę rolę włáſnym ich językiem Akieldámá / to jeſt / rolá krwi.
Abowiem nápiſáno w kśięgách Pſálmów : Niecháj będzie mieƺkánie jego puſte / á niech nie będzie / ktoby w nim mieƺkáł / á biſkupſtwo jego niech weźmie inny.
Potrzebá tedy / áby jeden z tych mężów / którzy z námi bywáli po wƺyſtek cżáſ / który Pán Jezuſ przebywáł między námi /
Pocżąwƺy od chrztu Jánowego / áż do tego dniá / którego jeſt wzięty w górę od náſ / był z námi świádkiem zmártwychwſtániá jego.
Y poſtáwili dwóch : Józefá / którego zwáno Bárſábáƺem / którego też názywáno Juſtem / y Máćiejá.
A modląc śię mówili : Ty Pánie! który znáƺ ſercá wƺyſtkich / okáż z tych dwóch jednego / któregoś obráł ;
Aby przyjął cżąſtkę uſługiwániá tego / y Apoſtolſtwá / z którego wypádł Judáƺ / áby odƺedł ná miejſce ſwoje.
Y rzućili loſy ich. Y pádł loſ ná Máćiejá ; á przyłącżony jeſt ſpólnem zdániem do jedenáſtu Apoſtołów.
