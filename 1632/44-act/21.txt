A gdyśmy odjecháli / rozſtáwƺy śię z nimi / proſto jádąc / przyjecháliśmy do Kou / á názájutrz do Rodu / á ſtámtąd do Pátáry.
A tám ználázłƺy okręt / który miáł płynąć do Fenicyi / wśiádłƺy weń / jecháliśmy.
A gdy śię nám ukázáł Cypr / tedy zoſtáwiwƺy go po lewej ſtronie / płynęliśmy do Syryi / y przypłynęliśmy do Tyru ; ábowiem tám z okrętu towáry ſkłádáć miáno.
A ználázłƺy ucżniów / zámieƺkáliśmy tám śiedm dni ; którzy mówili Páwłowi przez duchá / áby nie chodził do Jeruzálemu.
A gdyśmy przemieƺkáli one dni / wyƺedłƺy / poƺliśmy / á wƺyſcy náſ prowádzili z żonámi y z dziátkámi áż zá miáſto / á klęknąwƺy ná koláná ná brzegu / modliliśmy śię.
A pożegnáwƺy śię jedni z drugimi / wſtąpiliśmy w okręt / á oni śię wróćili do domu.
A my odpráwiwƺy płynienie z Tyru / przypłynęliśmy do Ptolemáidy / á pozdrowiwƺy bráći / zámieƺkáliśmy u nich przez jeden dzień.
A názájutrz wyƺedłƺy Páweł y my / którzyśmy z nim byli / przyƺliśmy do Cezáryi / á wƺedłƺy w dom Filipá Ewángieliſty / który był jeden z onych śiedmiu / zoſtáliśmy u niego.
A ten miáł cżtery córki pánny / które prorokowáły.
A gdyśmy tám przez niemáło dni zámieƺkáli / przyƺedł z Judzkiej ziemi prorok niektóry / imieniem Agábuſ.
Ten przyƺedłƺy do náſ / y wziąwƺy páſ Páwłá á związáwƺy ſobie ręce y nogi / rzekł : To mówi Duch Święty : Mężá / którego jeſt ten páſ / ták zwiążą w Jeruzálemie Żydowie / y podádzą go w ręce pogánom.
A gdyśmy to uſłyƺeli / prośiliśmy y my y ći / którzy ná onem miejſcu byli / áby on nie chodził do Jeruzálemu.
Tedy odpowiedziáł Páweł : Cóż cżynićie płácżąc y ſerce mi pſując? Abowiem já nie tylko być związánym / ále y umrzeć jeſtem gotowy w Jeruzálemie dlá imieniá Páná Jezuſowego.
A gdy śię on nie dáł námówić / dáliśmy pokój / mówiąc : Niech śię ſtánie wolá Páńſká.
A po onych dniách / wziąwƺy rzecży ſwoje / ƺliśmy do Jeruzálemu.
A ƺli z námi y niektórzy ucżniowie z Cezáryi / wiodąc z ſobą tego / u któregośmy goſpodą ſtáć mieli / niejákiego Mnázoná Cypryjcżyká / ſtárego ucżniá.
A gdyśmy przyƺli do Jeruzálemu / wdzięcżnie náſ bráćiá przyjęli.
A názájutrz wƺedł z námi Páweł do Jákóbá / gdzie śię byli wƺyſcy ſtárśi zeƺli.
Które pozdrowiwƺy / rozpowiedziáł im wƺyſtko porządnie / co Bóg ucżynił między pogány przez uſługę jego.
Co oni uſłyƺáwƺy / chwálili Páná y rzekli mu : Widziƺ / bráćie! jáko jeſt wiele tyśięcy Żydów / którzy uwierzyli ; á ći wƺyſcy gorliwi ſą miłośnicy zákonu.
Ale o tobie wzięli ſpráwę / że odwodziƺ od Mojżeƺá wƺyſtkich tych Żydów / którzy ſą między pogány / mówiąc / że nie máją obrzezywáć dziátek / áni máją chodzić według uſtáw zákonnych.
Cóż tedy jeſt? Koniecżnieć śię muśi zejść lud ; bo uſłyƺą / żeś przyƺedł.
A przetoż cżyń to / coć mówimy ; Mámy tu cżterech mężów / którzy ná ſobie ślub máją.
Tych wziąwƺy do śiebie / ocżyść śię z nimi y ucżyń nákłád ná nie / áby ogolili głowy ; á poznáją wƺyſcy / że to / co o tobie ſłyƺeli / nic nie jeſt / ále że y ty ſám chodziƺ przeſtrzegájąc zákonu.
A o tych / którzy uwierzyli z pogán / myśmy piſáli / ſtánowiąc / áby nic tákowego nie záchowywáli / tylko áby śię wyſtrzegáli tego / co jeſt ofiárowáne báłwánom / y od krwi y od rzecży dłáwionych y od wƺetecżeńſtwá.
Tedy Páweł wziąwƺy z ſobą one męże / názájutrz ocżyƺcżony będąc z nimi / wƺedł do kośćiołá / opowiádájąc wypełnienie dni ocżyƺcżeniá / áż zá káżdego z nich oddáná byłá ofiárá.
A gdy śię miáło wypełnić śiedm dni / niektórzy Żydowie z Azyi / ujrzáwƺy go w kośćiele / wzbudzili wƺyſtek lud y wrzućili náń ręce /
Wołájąc : Mężowie Izráelſcy / rátujćie! Tenći to jeſt cżłowiek / który przećiwko ludowi y zákonowi y miejſcu temu wƺyſtkich wƺędy ucży / nádto y Greki wprowádził do kośćiołá / y ſplugáwił to miejſce święte.
Abowiem przedtem widzieli z nim w mieśćie Trofimá Efeſkiego / o którym mniemáli / żeby go Páweł wprowádził do kośćiołá.
Y wzruƺyło śię miáſto wƺyſtko / y zbiegł śię lud ; á pojmáwƺy Páwłá / wywlekli go precż z kośćiołá / á zátem záráz drzwi zámkniono.
A gdy śię ſtáráli / jákoby go zábili / dáno znáć hetmánowi wojſká / iż śię wzruƺyło wƺyſtko Jeruzálem.
Który zárázem wziąwƺy z ſobą żołnierze y ſetniki / przybieżáł do nich. A oni ujrzáwƺy hetmáná y żołnierze / przeſtáli Páwłá bić.
Tedy hetmán przybliżywƺy śię / pojmáł go y kázáł go dwomá łáńcuchámi związáć / y wywiádywáł śię / ktoby był y coby ucżynił?
A jedni ták / drudzy inácżej między ludem wołáli. A gdy śię nic pewnego dlá zgiełku dowiedzieć nie mógł / rozkázáł go wieść do obozu.
A gdy był u wſchodu / przydáło śię / że go práwie żołnierze nieśli dlá gwáłtu onego ludu.
Abowiem wielki lud ƺedł zá nim / wołájąc : Zgłádź go.
A gdy miáł być Páweł prowádzony do obozu / rzekł hetmánowi : A godzi mi śię co mówić do ćiebie? A on rzekł : Umieƺ po grecku?
Y nie tyżeś jeſt on Egipcżánin / któryś przed temi dniámi ucżynił rozruch / y wywiodłeś ná puƺcżę cżtery tyśiące mężów zbójców?
A Páweł rzekł : Jámći jeſt cżłowiek Żyd Tárſeńcżyk / mieƺcżánin nie z podłego miáſtá w Cylicyi : przetoż proƺę ćię / dopuść mi mówić do ludu.
A gdy on dopuśćił / Páweł ſtojąc ná wſchodzie / ſkinął ręką ná lud. A gdy było wielkie milcżenie / ucżynił rzecż do nich żydowſkim językiem / mówiąc :
