Nábuchodonozor król / wƺyſtkim ludziom / národom / y językom / którzy mieƺkáją po wƺyſtkiej ziemi : Pokój śię wám niech rozmnoży!
Znáki y dziwy / które ucżynił ze mną Bóg nájwyżƺy / zdáło mi śię zá rzecż przyſtojną opowiedzieć.
O jákoż ſą wielkie znáki jego! á dziwy jego jáko mocne! króleſtwo jego króleſtwo wiecżne / y włádzá jego od národu do národu.
Já Nábuchodonozor żyjąc w pokoju w domu moim / y kwitnąc ná páłácu moim /
Miáłem ſen / który mię przeſtráƺył / y myśli / którem miáł ná łożu mojem / á widzeniá / którem widziáł / zátrwożyły mię.
A przetoż wydány jeſt ode mnie dekret / áby przywiedziono przed mię wƺyſtkich mędrców Bábilońſkich / którzyby mi wykłád ſnu tego oznájmili.
Tedy przyƺli mędrcy y práktykárze Cháldejſcy / y wieƺcżkowie ; y powiedziáłem im ſen / á wƺákże wykłádu jego nie mogli mi oznájmić ;
Aż ná oſtátek przyƺedł przed mię Dánijel / którego imię Báltázár według imieniá bogá mego / á w którym jeſt duch bogów świętych / á ſen powiedziáłem przed nim /
Mówiąc : Báltázárze / przedniejƺy z mędrców! Já wiem / iż duch bogów świętych jeſt w tobie / á żádná tájemnicá nie jeſt ći trudná ; widzeniá ſnu mego / którym miáł / poſłucháj / á wykłád jego powiedz mi.
Te ſą widzeniá / którem widziáł ná łożu mojem : Widziáłem / á oto drzewo w pośrodku ziemi / którego wyſokość zbytniá byłá.
Wielkie było ono drzewo y mocne / á wyſokość jego dośięgáłá niebá / á okázáłe było áż do gránic wƺyſtkiej ziemi ;
Gáłęzie jego piękne / á owoc jego obfity / y pokárm dlá wƺyſtkich był ná nim ; pod ſobą dáwáło ćień zwierzowi polnemu / á ná gáłęziách jego mieƺkáło ptáſtwo niebieſkie / á z niego miáło pożywienie wƺelkie ćiáło.
Widziáłem nádto w widzeniách moich ná łożu mojem / á oto ſtróż y Święty z niebá zſtąpiwƺy /
Wołáł ze wƺyſtkiej mocy / y ták rzekł : Podrąbćie to drzewo / y obetnijćie gáłęzie jego / á otłucżćie liśćie jego / y rozrzuććie owoc jego ; niech śię rozbieży zwierz / który jeſt pod nim / y ptáſtwo z gáłęzi jego ;
Wƺákże pień korzeniá jego w ziemi zoſtáwćie / á niech będzie związány łáńcuchem żeláznym y miedziánym ná tráwie polnej / áby roſą niebieſką był ſkrápiány / á z zwierzętámi niech śię páśie w tráwie ziemſkiej ;
Serce jego od cżłowiecżego niech śię odmieni / á ſerce zwierzęce niech mu dáne będzie / á śiedm lát niech pominą nád nim.
Tá rzecż według wyroku ſtróżów / á to żądánie według mowy świętych ſtánie śię / áż do tego przyjdzie / że poznáją ludzie / iż Nájwyżƺy pánuje nád króleſtwem ludzkiem / á dáje je / komu chce / á nájpodlejƺego z ludzi ſtánowi nád nim.
Ten ſen widziáłem já król Nábuchodonozor ; á ty / Báltázárze! powiedz wykłád jego / gdyż wƺyſcy mędrcy króleſtwá mego nie mogli mi tego wykłádu oznájmić ; ále ty możeƺ / bo duch bogów świętych jeſt w tobie.
Tedy Dánijel / którego imię Báltázár / zdumiewáł śię przez jednę godzinę / á myśli jego trwożyły go. A odpowiádájąc król rzekł : Báltázárze! ſen y wykłád jego niech ćię nie trwoży. Odpowiedziáł Báltázár / y rzekł : Pánie mój! ten ſen niech przyjdzie ná tych / którzy ćię nienáwidzą / á wykłád jego ná nieprzyjáćiół twoich.
Drzewo / któreś widziáł roſłe y mocne / którego wyſokość dośięgáłá niebá / á które okázáłe było wƺyſtkiej ziemi /
Którego gáłąź piękná / á owoc jego obfity / á pokárm dlá wƺyſtkich ná nim / pod którem mieƺkáł zwierz polny / á ná gáłęziách jego przebywáło ptáſtwo niebieſkie /
Tyś jeſt tym / o królu! któryś śię rozwielmożył y zmocnił / á wielkość twojá uroſłá / y podnioſłá śię áż do niebá / á włádzá twojá áż do końcżyn ziemi.
A iż król widziáł ſtróżá y Świętego zſtępującego z niebá á mówiącego : Podrąbćie to drzewo / á zepſujćie je / wƺákże pień y z korzeniem jego w ziemi zoſtáwćie / y niech będzie związány żeláznym y miedziánym łáńcuchem ná tráwie polnej / áby roſą niebieſką był ſkrápiány / á z zwierzętámi polnemi niech śię páśie / áżby śię wypełniło śiedm lát nád nim ;
Tenći jeſt wykłád / o królu! y ten dekret Nájwyżƺego / który wyƺedł ná królá / páná mego ;
Bo ćię wyrzucą od ludzi / á z zwierzem polnym będzie mieƺkánie twoje / á tráwą jáko wół páść śię będzieƺ / á roſą niebieſką ſkrápiány będzieƺ / áż śię wypełni śiedm lát nád tobą / dokądbyś nie poznáł / że Nájwyżƺy pánuje nád króleſtwem ludzkiem / á że je dáje / komu chce.
A iż rozkázáno zoſtáwić pień y z korzeniem onego drzewá / znácży / że króleſtwo twoje tobie zoſtánie / gdy poznáƺ / że niebioſá pánują.
Przetoż o królu! rádá mojá niech ći śię podobá / á grzechy twoje przerwij ſpráwiedliwośćią / á niepráwośći twoje miłośierdziem nád utrápionym / owá ſnáć ſtánie śię przedłużenie pokoju twego.
Wƺyſtko to przyƺło ná królá Nábuchodonozorá ;
Bo po wyjśćiu dwunáſtu mieśięcy / przechodząc śię w Bábilonie ná páłácu królewſkim /
Mówił król y rzekł : Izáli nie to jeſt on Bábilon wielki / którym já w śile mocy mojej zbudowáł / áby był ſtolicą króleſtwá y ku ozdobie ſłáwy mojej?
A gdy jeƺcże tá mowá byłá w uſtách królewſkich / oto głoſ z niebá przyƺedł mówiąc : Tobie śię mówi / królu Nábuchodonozorze! że króleſtwo twoje odeƺło od ćiebie ;
Y od ludzi wyrzucą ćię / á z zwierzem polnym będzie mieƺkánie twoje ; tráwą jáko wół páść śię będzieƺ / áżby śię wypełniło śiedm lát nád tobą / dokądbyś nie poznáł / że Nájwyżƺy pánuje nád króleſtwem ludzkiem / á że je dáje / komu chce.
Tejże godziny wypełniło śię ono ſłowo nád Nábuchodonozorem ; bo go wyrzucono od ludzi / á tráwę jádáł jáko wół / á roſą niebieſką ćiáło jego ſkrápiáne było / áż ná nim włoſy uroſły jáko pierze orle / á páznogćie jego jáko pázury u ptáków.
A po ſkońcżeniu onych dni podnioſłem já Nábuchodonozor w niebo ocży moje / á rozum mój do mnie śię záś wróćił / y błogoſłáwiłem Nájwyżƺego / á Żyjącego ná wieki chwáliłem y wyſłáwiáłem ; bo włádzá jego włádzá wiecżná / á króleſtwo jego od národu do národu.
A wƺyſcy obywátele ziemi jáko zá nic pocżytáni ſą. Według woli ſwojej poſtępuje / z wojſkiem niebieſkiem y z obywátelámi ziemi / á niemáƺ / ktoby wſtręt ucżynił ręce jego y rzekł mu : Cóż to cżyniƺ?
Tegoż cżáſu rozum mój wróćił śię do mnie / á do ſłáwy króleſtwá mego ozdobá mojá / y doſtojność mojá wróćiłá śię do mnie ; nádto hetmáni moi y kśiążętá moi ƺukáli mię / á ná króleſtwie mojem zmocniłem śię / y wielmożność więkƺá mi jeſt przydáná.
A ták teráz já Nábuchodonozor chwálę / y wywyżƺám y wyſłáwiám królá niebieſkiego / którego wƺyſtkie ſpráwy ſą práwdą / á śćieƺki jego ſądem / á który chodzących w hárdośći poniżyć może.
