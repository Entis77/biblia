Nábuchodonozor król / wƺyſtkim ludźiom / národom / y językom / którzy mieƺkáją po wƺyſtkiej źiemi : Pokój śię wam niech rozmnoży!
Znáki y dźiwy / które ucżynił ze mną Bóg nájwyżƺy / zdáło mi śię zá rzecż przyſtojną opowiedźieć.
O jákoż ſą wielkie znáki jego! á dźiwy jego jáko mocne! króleſtwo jego króleſtwo wiecżne / y włádzá jego od národu do národu.
Ja Nábuchodonozor żyjąc w pokoju w domu moim / y kwitnąc ná páłácu moim /
Miáłem ſen / który mię przeſtráƺył / y myśli / którem miał ná łożu mojim / á widzeniá / którem widźiał / zátrwożyły mię.
A przetoż wydány jeſt ode mnie dekret / áby przywiedźiono przed mię wƺyſtkich mędrców Bábilońſkich / którzyby mi wykłád ſnu tego oznájmili.
Tedy przyƺli mędrcy y práktykárze Cháldejſcy / y wieƺcżkowie ; y powiedźiáłem im ſen / á wƺákże wykłádu jego nie mogli mi oznájmić ;
Aż ná oſtátek przyƺedł przed mię Dánijel / którego imię Báltázár według imieniá bogá mego / á w którym jeſt duch bogów świętych / á ſen powiedźiáłem przed nim /
Mówiąc : Báltázárze / przedniejƺy z mędrców! Ja wiem / iż duch bogów świętych jeſt w tobie / á żádná tájemnicá nie jeſt ći trudná ; widzeniá ſnu mego / którym miał / poſłucháj / á wykłád jego powiedz mi.
Te ſą widzeniá / którem widźiał ná łożu mojim : Widźiáłem / á oto drzewo w pośrzodku źiemi / którego wyſokość zbytniá byłá.
Wielkie było ono drzewo y mocne / á wyſokość jego dośięgáłá niebá / á okázáłe było áż do gránic wƺyſtkiej źiemi ;
Gáłęźie jego piękne / á owoc jego obfity / y pokárm dla wƺyſtkich był ná nim ; pod ſobą dáwáło ćień zwierzowi polnemu / á ná gáłęźiách jego mieƺkáło ptáſtwo niebieſkie / á z niego miáło pożywienie wƺelkie ćiáło.
Widźiáłem nádto w widzeniách mojich ná łożu mojim / á oto ſtróż y Święty z niebá zſtąpiwƺy /
Wołał ze wƺyſtkiej mocy / y ták rzekł : Podrąbćie to drzewo / y obetnijćie gáłęźie jego / á otłucżćie liśćie jego / y rozrzuććie owoc jego ; niech śię rozbieży zwierz / który jeſt pod nim / y ptáſtwo z gáłęźi jego ;
Wƺákże pień korzeniá jego w źiemi zoſtáwćie / á niech będźie związány łáńcuchem żeláznym y miedźiánym ná tráwie polney / áby roſą niebieſką był ſkrápiány / á z zwierzętámi niech śię páśie w tráwie źiemſkiej ;
Serce jego od cżłowiecżego niech śię odmieni / á ſerce zwierzęce niech mu dáne będźie / á śiedm lat niech pominą nád nim.
Tá rzecż według wyroku ſtróżów / á to żądánie według mowy świętych ſtánie śię / áż do tego przydźie / że poznáją ludźie / iż Nájwyżƺy pánuje nád króleſtwem ludzkiem / á dáje je / komu chce / á nájpodlejƺego z ludźi ſtánowi nád nim.
Ten ſen widźiáłem ja król Nábuchodonozor ; á ty / Báltázárze! powiedz wykłád jego / gdyż wƺyſcy mędrcy króleſtwá mego nie mogli mi tego wykłádu oznájmić ; ále ty możeƺ / bo duch bogów świętych jeſt w tobie.
Tedy Dánijel / którego imię Báltázár / zdumiewał śię przez jednę godźinę / á myśli jego trwożyły go. A odpowiádájąc król rzekł : Báltázárze! ſen y wykłád jego niech ćię nie trwoży. Odpowiedźiał Báltázár / y rzekł : Pánie mój! ten ſen niech przydźie ná tych / którzy ćię nienáwidzą / á wykłád jego ná nieprzyjáćiół twojich.
Drzewo / któreś widźiał roſłe y mocne / którego wyſokość dośięgáłá niebá / á które okázáłe było wƺyſtkiej źiemi /
Którego gáłąź piękná / á owoc jego obfity / á pokárm dla wƺyſtkich ná nim / pod którem mieƺkał zwierz polny / á ná gáłęźiách jego przebywáło ptáſtwo niebieſkie /
Tyś jeſt tym / o królu! któryś śię rozwielmożył y zmocnił / á wielkość twojá uroſłá / y podnioſłá śię áż do niebá / á włádzá twojá áż do końcżyn źiemi.
A iż król widźiał ſtróżá y Świętego zſtępującego z niebá á mówiącego : Podrąbćie to drzewo / á zepſujćie je / wƺákże pień y z korzeniem jego w źiemi zoſtáwćie / y niech będźie związány żeláznym y miedźiánym łáńcuchem ná tráwie polney / áby roſą niebieſką był ſkrápiány / á z zwierzętámi polnemi niech śię páśie / áżby śię wypełniło śiedm lat nád nim ;
Tenći jeſt wykłád / o królu! y ten dekret Nájwyżƺego / który wyƺedł ná królá / páná mego ;
Bo ćię wyrzucą od ludźi / á z zwierzem polnym będźie mieƺkánie twoje / á tráwą jáko wół páść śię będźieƺ / á roſą niebieſką ſkrápiány będźieƺ / áż śię wypełni śiedm lat nád tobą / dokądbyś nie poznał / że Nájwyżƺy pánuje nád króleſtwem ludzkiem / á że je dáje / komu chce.
A iż rozkázáno zoſtáwić pień y z korzeniem onego drzewá / znácży / że króleſtwo twoje tobie zoſtánie / gdy poznáƺ / że niebioſá pánują.
Przetoż o królu! rádá mojá niech ći śię podobá / á grzechy twoje przerwij ſpráwiedliwośćią / á niepráwośći twoje miłośierdźiem nád utrápionym / owá ſnadź ſtánie śię przedłużenie pokoju twego.
Wƺyſtko to przyƺło ná królá Nábuchodonozorá ;
Bo po wyjśćiu dwunáſtu mieśięcy / przechodząc śię w Bábilonie ná páłácu królewſkim /
Mówił król y rzekł : Izali nie to jeſt on Bábilon wielki / którym ja w śile mocy mojey zbudował / áby był ſtolicą króleſtwá y ku ozdobie ſłáwy mojey?
A gdy jeƺcże tá mowá byłá w uſtách królewſkich / oto głos z niebá przyƺedł mówiąc : Tobie śię mówi / królu Nábuchodonozorze! że króleſtwo twoje odeƺło od ćiebie ;
Y od ludźi wyrzucą ćię / á z zwierzem polnym będźie mieƺkánie twoje ; tráwą jáko wół páść śię będźieƺ / áżby śię wypełniło śiedm lat nád tobą / dokądbyś nie poznał / że Nájwyżƺy pánuje nád króleſtwem ludzkiem / á że je dáje / komu chce.
Tejże godźiny wypełniło śię ono ſłowo nád Nábuchodonozorem ; bo go wyrzucono od ludźi / á tráwę jádał jáko wół / á roſą niebieſką ćiáło jego ſkrápiáne było / áż ná nim włoſy uroſły jáko pierze orle / á páznogćie jego jáko pázury u ptáków.
A po ſkońcżeniu onych dni podnioſłem ja Nábuchodonozor w niebo ocży moje / á rozum mój do mnie śię záś wróćił / y błogoſłáwiłem Nájwyżƺego / á Żyjącego ná wieki chwáliłem y wyſłáwiáłem ; bo włádzá jego włádzá wiecżná / á króleſtwo jego od národu do národu.
A wƺyſcy obywátele źiemi jáko zá nic pocżytáni ſą. Według woli ſwojey poſtępuje / z wojſkiem niebieſkiem y z obywátelámi źiemi / á niemáƺ / ktoby wſtręt ucżynił ręce jego y rzekł mu : Cóż to cżyniƺ?
Tegoż cżáſu rozum mój wróćił śię do mnie / á do ſłáwy króleſtwá mego ozdobá mojá / y doſtojność mojá wróćiłá śię do mnie ; nádto hetmáni moi y kśiążętá moi ƺukáli mię / á ná króleſtwie mojim zmocniłem śię / y wielmożność więkƺá mi jeſt przydáná.
A ták teraz ja Nábuchodonozor chwálę / y wywyżƺám y wyſłáwiám królá niebieſkiego / którego wƺyſtkie ſpráwy ſą práwdą / á śćieƺki jego ſądem / á który chodzących w hárdośći poniżyć może.
