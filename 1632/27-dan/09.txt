Roku pierwƺego Dáryjuƺá / ſyná Aſweruſowego / z náśieniá Medów / który był poſtánowiony królem ná króleſtwem Cháldejſkiem ;
Roku pierwƺego królowániá jego / já Dánijel zrozumiáłem z kśiąg licżbę lát / o których było ſłowo Páńſkie do Jeremijáƺá proroká / że śię wypełnić miáło ſpuſtoƺenie Jeruzálemſkie w śiedmdzieśięt lát.
Y obróćiłem oblicże moje do Páná Bogá / ƺukájąc go modlitwą y prośbámi w pośćie y w worze y w popiele.
Modliłem śię tedy Pánu Bogu memu / á wzywájąc rzekłem : Proƺę Pánie! Boże wielki y ſtráƺny / ſtrzegący przymierzá y miłośierdziá tym / którzy ćię miłują / y ſtrzegą przykázáń twoich ;
Zgrzeƺyliśmy y przewrotnieśmy cżynili / y niezbożnieśmy śię ſpráwowáli / y ſprzećiwiliśmy śię / á odſtąpiliśmy od przykázáń twoich y od ſądów twoich ;
Y nie ſłucháliśmy ſług twoich / proroków / którzy máwiáli w imieniu twojim do królów náƺych / do kśiążąt náƺych / y do ojców náƺych / y do wƺyſtkiego ludu ziemi.
Tobie / Pánie! ſpráwiedliwość / á nám záwſtydzenie twárzy náleży / jáko śię to dzieje dniá tego mężom Judzkim y obywátelom Jeruzálemſkim y wƺyſtkiemu Izráelowi / bliſkim y dálekim we wƺyſtkich ziemiách / do którycheś ich wygnáł dlá przeſtępſtwá ich / którem wyſtąpili przećiwko tobie.
Pánie! námći náleży záwſtydzenie twárzy / królom náƺym / kśiążętom náƺym y ojcom náƺym / bośmy zgrzeƺyli przećiwko tobie :
Ale Pánu / Bogu náƺemu / miłośierdzie y litość ; poniewáżeśmy mu odporni byli /
A nie byliśmy poſłuƺni głoſowi Páná / Bogá náƺego / żebyśmy chodzili w uſtáwách jego / które on dáł przed oblicże náƺe przez proroków / ſług ſwoich ;
Owƺem / wƺyſcy Izráelcżycy przeſtąpili zákon twój y odchylili śię / żeby nie ſłucháli głoſu twego : przetoż śię wyláło ná náſ to złorzecżeńſtwo y przeklęſtwo / które jeſt nápiſáne w zákonie Mojżeƺá / ſługi Bożego ; bośmy zgrzeƺyli przećiwko niemu.
Skąd ſpełnił ſłowá ſwoje / które mówił przećiwko nám / y przećiwko ſędziom náƺym / którzy náſ ſądzili / á przywiódł ná náſ to wielkie złe / które śię nie ſtáło pod wƺyſtkiem niebem / jákie śię ſtáło w Jeruzálemie.
Ták jáko nápiſáno w zákonie Mojżeƺowym / wƺyſtko to złe przyƺło ná náſ ; á wżdyśmy nie prośili oblicżá Páná / Bogá náƺego / ábyśmy śię odwróćili od niepráwośći náƺych / á mieli wzgląd ná práwdę jego.
Przetoż nie omieƺkáł Pán z tem złem / ále je przywiódł ná náſ ; bo ſpráwiedliwy jeſt Pán / Bóg náƺ / we wƺyſtkich ſpráwách ſwoich / które cżyni / któregośmy głoſu nie ſłucháli.
Wƺákże teráz / o Pánie / Boże náƺ! któryś wywiódł lud ſwój z ziemi Egipſkiej ręką możną / y ucżyniłeś ſobie imię / jáko śię to dziś pokázuje / zgrzeƺyliśmy / niepobożnieśmy cżynili.
O Pánie! według wƺyſtkich ſpráwiedliwośći twoich niech śię proƺę odwróći popędliwość twojá y gniew twój od miáſtá twego Jeruzálemu / góry świętobliwośći twojej : bo dlá grzechów náƺych y dlá niepráwośći ojców náƺych Jeruzálem y lud twój nośi poháńbienie u wƺyſtkich / którzy ſą około náſ.
Teráz tedy wyſłucháj / o Boże náƺ! modlitwę ſługi twego y prośby jego / á oświeć oblicże twoje nád ſpuſtoƺoną świątnicą twoją / dlá Páná.
Nákłoń / Boże mój! uchá twego á uſłyƺ ; otwórz ocży twoje á obácż ſpuſtoƺeniá náƺe y miáſto / które jeſt názwáne od imieniá twego ; bo my przekłádámy modlitwy náƺe przed oblicżem twojim / nie dlá jákiej náƺej ſpráwiedliwośći / ále dlá obfitego miłośierdziá twego.
O Pánie! wyſłucháj / Pánie! odpuść / Pánie! obácż á ucżyń ; nie odwłácżáj ſám dlá śiebie / Boże mój! bo od imieniá twego názwáne jeſt to miáſto y lud twój.
A gdym já jeƺcże mówił / y modliłem śię / y wyznáwáłem grzech mój y grzech ludu mego Izráelſkiego / y przekłádáłem modlitwę moję przed twárzą Páná / Bogá mego / zá górę świętobliwośći Bogá mego ;
Práwie gdym já jeƺcże mówił y modliłem śię / oto mąż on Gábryjel / któregom widziáł w widzeniu ná pocżątku / prędko lecąc dotknął śię mnie cżáſu ofiáry wiecżornej /
A uſługując mi do zrozumieniá mówił ze mną y rzekł : Dánijelu! terázem wyƺedł / ábym ćię náucżył wyrozumieniá tájemnicy.
Ná pocżątku modlitw twoich wyƺło ſłowo / á jám przyƺedł / ábym ći je oznájmił / boś ty wielce przyjemny ; á ták miej wzgląd ná to ſłowo / á zrozumiej to widzenie.
Siedmdzieśiąt tego dni zámierzono ludowi twemu y miáſtu twemu świętemu ná znieśienie przeſtępſtwá / y ná zágłádzenie grzechów y ná ocżyƺcżenie niepráwośći / y ná przywiedzienie ſpráwiedliwośći wiecżnej / y ná zápiecżętowánie widzeniá y proroctwá / á ná pomázánie Świętego świętych.
Przetoż wiedz á zrozumiej / że od wyjśćiá ſłowá o przywróceniu y zbudowániu Jeruzálemu áż do Meſyjáƺá wodzá będzie tygodni śiedm / potem tygodni ƺeśćdzieśiąt y dwá / gdy znowu zbudowáná będzie ulicá y przekopánie / á te cżáſy będą bárdzo trudne.
A po onych ƺeśćdześięćiu y dwóch tygodniách zábity będzie Meſyjáƺ / wƺákże mu to nic nie záƺkodzi ; owƺem / to miáſto y tę świątnicę ſkázi lud wodzá przyƺłego / ták / że koniec jego będzie z powodzią / y áż do ſkońcżeniá wojny będzie uſtáwicżne puſtoƺenie.
Wƺákże zmocni przymierze wielom ich w tygodniu oſtátnim ; á w połowie onego tygodniá ucżyni koniec ofierze pálonej y ofierze śniednej / á przez wojſko obrzydliwe puſtoƺyćiel przyjdzie / y áż do ſkońcżeniá náznácżonego wyleje śię ſpuſtoƺenie ná tego / który má być ſpuſtoƺony.
