Roku pierwƺego Dáryjuƺá / ſyná Aſweruſowego / z náśieniá Medów / który był poſtánowiony królem ná króleſtwem Cháldejſkiem ;
Roku pierwƺego królowánia jego / ja Dánijel zrozumiáłem z kśiąg licżbę lat / o których było ſłowo Páńſkie do Jeremijáƺá proroká / że śię wypełnić miáło ſpuſtoƺenie Jeruzálemſkie w śiedmdźieśięt lat.
Y obróćiłem oblicże moje do Páná Bogá / ƺukájąc go modlitwą y prośbámi w pośćie y w worze y w popiele.
Modliłem śię tedy Pánu Bogu memu / á wzywájąc rzekłem : Proƺę Pánie! Boże wielki y ſtráƺny / ſtrzegący przymierza y miłośierdźiá tym / którzy ćię miłują / y ſtrzegą przykázáń twojich ;
Zgrzeƺyliśmy y przewrotnieśmy cżynili / y niezbożnieśmy śię ſpráwowáli / y ſprzećiwiliśmy śię / á odſtąpiliśmy od przykázáń twojich y od ſądów twojich ;
Y nie ſłucháliśmy ſług twojich / proroków / którzy máwiáli w imieniu twojim do królów náƺych / do kśiążąt náƺych / y do ojców náƺych / y do wƺyſtkiego ludu źiemi.
Tobie / Pánie! ſpráwiedliwość / á nam záwſtydzenie twárzy náleży / jáko śię to dźieje dniá tego mężom Judzkim y obywátelom Jeruzálemſkim y wƺyſtkiemu Izráelowi / bliſkim y dálekim we wƺyſtkich źiemiách / do którycheś ich wygnał dla przeſtępſtwá ich / którem wyſtąpili przećiwko tobie.
Pánie! námći náleży záwſtydzenie twárzy / królom náƺym / kśiążętom náƺym y ojcom náƺym / bośmy zgrzeƺyli przećiwko tobie :
Ale Pánu / Bogu náƺemu / miłośierdźie y litość ; poniewáżeśmy mu odporni byli /
A nie byliśmy poſłuƺni głoſowi Páná / Bogá náƺego / żebyſmy chodźili w uſtáwách jego / które on dał przed oblicże náƺe przez proroków / ſług ſwojich ;
Owƺem / wƺyſcy Izráelcżycy przeſtąpili zakon twój y odchylili śię / żeby nie ſłucháli głoſu twego : przetoż śię wyláło ná nas to złorzecżeńſtwo y przeklęſtwo / które jeſt nápiſáne w zakonie Mojżeƺá / ſługi Bożego ; bośmy zgrzeƺyli przećiwko niemu.
Skąd ſpełnił ſłowá ſwoje / które mówił przećiwko nam / y przećiwko ſędźiom náƺym / którzy nas ſądźili / á przywiódł ná nas to wielkie złe / które śię nie ſtáło pod wƺyſtkiem niebem / jákie śię ſtáło w Jeruzálemie.
Ták jáko nápiſano w zakonie Mojżeƺowym / wƺyſtko to złe przyƺło ná nas ; á wżdyśmy nie prośili oblicżá Páná / Bogá náƺego / ábyſmy śię odwróćili od niepráwośći náƺych / á mieli wzgląd ná práwdę jego.
Przetoż nie omieƺkał Pán z tym złem / ále je przywiódł ná nas ; bo ſpráwiedliwy jeſt Pán / Bóg náƺ / we wƺyſtkich ſpráwách ſwojich / które cżyni / któregośmy głoſu nie ſłucháli.
Wƺákże teraz / o Pánie / Boże náƺ! któryś wywiódł lud ſwój z źiemi Egipſkiej ręką możną / y ucżyniłeś ſobie imię / jáko śię to dźiś pokázuje / zgrzeƺyliśmy / niepobożnieśmy cżynili.
O Pánie! według wƺyſtkich ſpráwiedliwośći twojich niech śię proƺę odwróći popędliwość twojá y gniew twój od miáſtá twego Jeruzálemu / góry świętobliwośći twojey : bo dla grzechów náƺych y dla niepráwośći ojców náƺych Jeruzálem y lud twój nośi poháńbienie u wƺyſtkich / którzy ſą około nas.
Teraz tedy wyſłucháj / o Boże náƺ! modlitwę ſługi twego y prośby jego / á oświeć oblicże twoje nád ſpuſtoƺoną świątnicą twoją / dla Páná.
Nákłoń / Boże mój! uchá twego á uſłyƺ ; otwórz ocży twoje á obácż ſpuſtoƺeniá náƺe y miáſto / które jeſt názwáne od imieniá twego ; bo my przekłádámy modlitwy náƺe przed oblicżem twojim / nie dla jákiej náƺej ſpráwiedliwośći / ále dla obfitego miłośierdźiá twego.
O Pánie! wyſłucháj / Pánie! odpuść / Pánie! obácż á ucżyń ; nie odwłácżáj ſám dla śiebie / Boże mój! bo od imieniá twego názwáne jeſt to miáſto y lud twój.
A gdym ja jeƺcże mówił / y modliłem śię / y wyznáwáłem grzech mój y grzech ludu mego Izráelſkiego / y przekłádáłem modlitwę moję przed twárzą Páná / Bogá mego / zá górę świętobliwośći Bogá mego ;
Práwie gdym ja jeƺcże mówił y modliłem śię / oto mąż on Gábryjel / któregom widźiał w widzeniu ná pocżątku / prędko lecąc dotknął śię mnie cżáſu ofiáry wiecżorney /
A uſługując mi do zrozumieniá mówił ze mną y rzekł : Dánijelu! terazem wyƺedł / ábym ćię náucżył wyrozumieniá tájemnicy.
Ná pocżątku modlitw twojich wyƺło ſłowo / á jam przyƺedł / ábym ći je oznájmił / boś ty wielce przyjemny ; á ták miej wzgląd ná to ſłowo / á zrozumiej to widzenie.
Siedmdźieśiąt tego dni zámierzono ludowi twemu y miáſtu twemu świętemu ná znieśienie przeſtępſtwá / y ná zágłádzenie grzechów y ná ocżyƺcżenie niepráwośći / y ná przywiedźienie ſpráwiedliwośći wiecżney / y ná zápiecżętowánie widzeniá y proroctwá / á ná pomázánie Świętego świętych.
Przetoż wiedz á zrozumiej / że od wyjśćiá ſłowá o przywróceniu y zbudowániu Jeruzálemu áż do Meſyjáƺá wodzá będźie tygodni śiedm / potem tygodni ƺeśćdźieśiąt y dwá / gdy znowu zbudowáná będźie ulicá y przekopánie / á te cżáſy będą bárzo trudne.
A po onych ƺeśćdześięćiu y dwu tygodniách zábity będźie Meſyjáƺ / wƺákże mu to nic nie záƺkodźi ; owƺem / to miáſto y tę świątnicę ſkáźi lud wodzá przyƺłego / ták / że koniec jego będźie z powodźią / y áż do ſkońcżeniá wojny będźie uſtáwicżne puſtoƺenie.
Wƺákże zmocni przymierze wielom ich w tegodniu oſtátnim ; á w połowie onego tygodniá ucżyni koniec ofierze páloney y ofierze śniedney / á przez wojſko obrzydliwe puſtoƺyćiel przydźie / y áż do ſkońcżeniá náznácżonego wyleje śię ſpuſtoƺenie ná tego / który ma być ſpuſtoƺony.
