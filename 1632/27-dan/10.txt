Roku trzećiego Cyruſá / królá Perſkiego objáwione było ſłowo Dánijelowi / którego imię názwáno Báltázár ; á to ſłowo było práwdźiwe / y cżáſ zámierzony długi ; y zrozumiáł ono ſłowo / bo wźiął zrozumienie w widzeniu.
W one dni já Dánijel byłem ſmutny przez trzy tygodnie dni ;
Chlebám ſmácżnego nie jádł / á mięſo y wino nie wchodźiło w uſtá moje / ánim śię mázáł olejkiem / áż śię wypełniły dni trzech tygodni.
A dniá dwudźieſtego y cżwártego mieśiącá pierwƺego byłem nád brzegiem rzeki wielkiej / to jeſt Chydekel ;
A podnióſłƺy ocży moje ujrzáłem / á oto mąż niejáki ubrány w ƺátę lniáną / á biodrá jego przepáſáne były złotem ƺcżerem z Ufáſ ;
A ćiáło jego było jáko Tárſyſ / á oblicże jego ná wejrzeniu jáko błyſkáwicá / á ocży jego były jáko lámpy gorejące / á rámioná jego y nogi jego ná wejrzeniu jáko miedź wypolerowáná / á głoſ ſłów jego jáko głoſ mnóſtwá.
A widźiałem ja Dániel ſam to widzenie : Lecż mężowie którzy byli ze mną / nie widźieli tego widzenia ; ále ſtrách wielki przypadł ná nie / y poućiekáli / á pokryli śię.
A jám ſám zoſtáł / y widźiáłem to wielkie widzenie ; ále śiłá nie zoſtáłá we mnie / y kráſá mojá odmieniłá śię we mnie / y ſkáźiłá śię / y nie miáłem żádnej śiły.
Tedym ſłyƺáł głoſ ſłów jego ; á uſłyƺáwƺy głoſ ſłów jego uſnąłem twárdo ná twárzy mojej / ná twárzy mojej / mówię / ná źiemi.
Wtem oto ręká dotknęłá śię mnie / y podnioſłá mię ná koláná moje / y ná dłonie rąk moich.
Y rzekł do mnie : Dánijelu / mężu wielce przyjemny! miej wzgląd ná ſłowá moje / które já będę mówił do ćiebie / á ſtój ná miejſcu ſwem / bom teráz poſłány do ćiebie. A gdy przemówił do mnie to ſłowo / ſtáłem drżąc.
Y rzekł do mnie : Nie bój śię Dánijelu! bo od pierwƺego dniá / jákoś przyłożył ſerce twoje ku wyrozumieniu y trápiłeś śię Bogiem twoim / wyſłucháne ſą ſłowá twoje / á jám przyƺedł dlá ſłów twoich.
Lecż kśiążę króleſtwá Perſkiego ſprzećiwiáł mi śię przez dwádźieśćiá dni y jeden / áż oto Micháł / jeden z przedniejƺych kśiążąt / przyƺedł mi ná pomoc ; przetożem já tám zoſtáł przy królách Perſkich.
Alem przyƺedł / ábym ći oznájmił / co má przyjść ná lud twój w oſtátecżne dni ; bo jeƺcże widzenie będźie o tych dniách.
A gdy mówił do mnie temi ſłowy / ſpuśćiłem twárz moję ku źiemi / y zámilknąłem.
A oto jáko podobieńſtwo ſynów ludzkich dotknęło śię wárg moich ; á otworzywƺy uſtá ſwe mówiłem y rzekłem do ſtojącego przećiwko mnie : Pánie mój! dlá tego widzeniá obróćiły śię ná mię boleśći moje / y nie miáłem żádnej śiły.
A jákoż będźie mógł táki ſługá Páná mego rozmówić śię z tákim Pánem moim? Gdyż od tegoż cżáſu nie zoſtáłá we mnie śiłá / áni tchnienie zoſtáło we mnie.
Tedy śię mnie znowu dotknął ná wejrzeniu jáko cżłowiek / y pośilił mię /
Y rzekł : Nie bój śię / mężu wielce przyjemny / pokój tobie! pośil śię / pośil śię / mówię. A gdy mówił ze mną / wźiąłem śiłę y rzekłem : Niech mówi Pán mój ; ábowiemeś mię pośilił.
Y rzekł : Wieƺże / dlácżegom przyƺedł do ćiebie? Potem śię wrócę / ábym wálcżył z kśiążęćiem Perſkim / á gdy odejdę / oto kśiążę Grecki przyćiągnie.
Wƺákże oznájmięć to / co jeſt wyráżono w piśmie práwdy ; ále y jednego niemáƺ / któryby mężnie ſtáł przy mnie w tych rzecżách / oprócż Micháłá / kśiążęćiá wáƺego.
