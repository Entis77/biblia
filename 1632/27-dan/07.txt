Roku pierwƺego Bálſázárá / królá Bábilońſkiego / miał Dánijel ſen y widzeniá ſwoje ná łożu ſwem ; tedy ſpiſáł on ſen / y ſumę rzecży powiedźiáł.
A mówiąc Dánijel rzekł : Widźiáłem w widzeniu mojem w nocy / á oto cżtery wiátry niebieſkie potykáły śię ná morzu wielkiem ;
A cżtery beſtyje wielkie wyſtępowáły z morzá / różne jedná od drugiej.
Pierwƺá podobná lwowi / májąc ſkrzydłá orle ; y przypátrywáłem śię / áż wyrwáne były ſkrzydłá jey / któremi śię podnośiłá od źiemi / ták / że ná nogách jáko cżłowiek ſtánęłá / á ſerce cżłowiecże jey dáne jeſt.
Potem oto beſtyjá drugá podobná niedźwiedźiowi ; y ſtánęłá przy jedney ſtronie / á trzy żebrá były w páƺcżęce jey między zębámi jey / y ták mówiono do niej : Wſtáń / náżryj śię doſtátkiem mięſá.
Potemem widźiáł / á oto inná beſtyjá podobná lámpártowi / która miáłá cżtery ſkrzydłá ptáƺe ná grzbiećie ſwym / cżtery też głowy miáłá tá beſtyjá / y dáno jey włádzę wielką.
Potemem widźiáł w widzeniách nocnych / á oto beſtyjá cżwártá ſtráƺná y ſrogá y bárdzo mocná / májącá zęby żelázne wielkie / która pożeráłá y kruƺyłá / á oſtátek nogámi ſwemi deptáłá ; á tá byłá różná od wƺyſtkich beſtyj / które były przed nią / y miał á dźieśięć rogów.
Pilniem śię przypátrywáł rogom / á oto róg pośledni máły wyráſtáł między niemi / y trzy z tych rogów pierwƺych wyłámáne ſą przed nim ; á oto w onym rogu były ocży podobne ocżom cżłowiecżym / y uſtá mówiące rzecży wielkie.
Y przypátrywáłem śię / áż one ſtolice poſtáwione były / á Stárodáwny uśiádł / którego ƺátá byłá jáko śnieg biáłá / á włoſy głowy jego jáko wełná cżyſtá / ſtolicá jego jáko płomienie ogniſte / á kołá jey jáko ogień gorejący.
Rzeká ogniſtá płynąc wychodźiłá od oblicżá jego. Tyśiąc tyśięcy ſłużyło mu / á dźieśięć kroć tyśiąc tyśięcy ſtáło przed nim ; ſąd záśiádł / á kśięgi otworzone były.
Tedym śię przypátrywáł / ſkoro śię głoſ pocżął tych ſłów wielkich / które on róg mówił ; przypátrywáłem śię / áż byłá tá beſtyjá zábitá / y zginęło ćiáło jey / á podáne było ná ſpálenie ogniem.
Tákże y pozoſtáłym beſtyjom odjętá jeſt włádzá ich ; bo długość żywotá dáná im byłá áż do cżáſu / á to do zámierzonego cżáſu.
Widźiáłem też w widzeniu nocnem / á oto przychodźił w obłokách niebieſkich podobny ſynowi cżłowiecżemu / á przyƺedł áż do Stárodáwnego / y przywiedźiono go przed oblicżność jego.
Y dáł mu włádzę y cżeść y króleſtwo / áby mu wƺyſcy ludźie / národy y języki ſłużyli ; włádzá jego włádzá wiecżná / która nie będźie odjętá / á króleſtwo jego / które nie będźie ſkáżone.
Y zátrwożył śię we mnie Dánijelu duch mój w pośród ćiáłá mego / á widzeniá / którem widźiáł / przeſtráƺyły mię.
Tedym przyſtąpił do jednego z tych / którzy tám ſtáli / á pewnośćim śię dowiádywáł od niego o tem wƺyſtkiem / y powiedźiáł mi / y wykłád mów oznájmił mi.
Te beſtyje wielkie / których ſą cżtery / ſą cżtery królowie / którzy powſtáną z źiemi.
Ci ujmą króleſtwo świętych nájwyżƺych miejſc / którzy pośiąść máją króleſtwo áż ná wieki / y áż ná wieki wiecżne.
Tedym prágnął wźiąć ſpráwę o beſtyi cżwártey / która byłá różná od wƺyſtkich innych / bárdzo ſtráƺná / którey zęby były żelázne / á páznogćie jey miedźiáne ; która pożeráłá y kruƺyłá / á oſtátek nogámi ſwemi deptáłá.
Tákże o onych rogách dźieśięćiu / które były ná głowie jey / y o poślednim / który był wyróſł / przed którym wypádły trzy ; o tym rogu mówię / który miał ocży y uſtá mówiące wielkie rzecży / á ná wejrzeniu był więkƺy nád inne rogi.
Y przypátrywáłem śię / á oto róg ten wálcżył z świętymi / y przemágáł ich ;
Aż przyƺedł Stárodáwny á podány jeſt ſąd świętym nájwyżƺych miejſc / á cżáſ przyƺedł / áby to króleſtwo święći otrzymáli.
Y rzekł ták : Beſtyjá cżwártá / cżwárte króleſtwo znácży ná źiemi / które będźie różne od wƺyſtkich króleſtw / á pożre wƺyſtkę źiemię / á podepcże á pokruƺy ją ;
A dźieśięć rogów to znácży / że z króleſtwá onego dźieśięć królów powſtánie ; á po nich powſtánie pośledni / który będźie różny od pierwƺych / y trzech królów poniży ;
A ſłowá przećiw Nájwyżƺemu mówić będźie / y święte nájwyżƺych miejſc zetrze ; nádto będźie zámyśláł / áby odmienił cżáſy y práwá / gdyż wydáne będą w ręce jego áż do cżáſu y cżáſów / y pół cżáſu.
Potem záśiądźie ſąd / á tám włádzę jego odejmą / áby był zniƺcżony y wytrácony áż do końcá.
A króleſtwo y włádzá / y doſtojeńſtwo królewſkie pod wƺyſtkiem niebem dáne będźie ludowi świętych nájwyżƺych miejſc / którego króleſtwo będźie króleſtwo wiecżne / á wƺyſtkie zwierzchnośći jemu ſłużyć y onego ſłucháć będą.
Aż dotąd koniec tych ſłów. A mnie Dánijelá myśli moje wielce zátrwożyły / á jáſność mojá zmieniłá śię przy mnie ; wƺákżem to ſłowo w ſercu mojem záchowáł.
