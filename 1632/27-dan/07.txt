Roku pierwƺego Bálſázárá / królá Bábilońſkiego / miáł Dánijel ſen y widzeniá ſwoje ná łożu ſwem ; tedy ſpiſáł on ſen / y ſumę rzecży powiedziáł.
A mówiąc Dánijel rzekł : Widziáłem w widzeniu mojem w nocy / á oto cżtery wiátry niebieſkie potykáły śię ná morzu wielkiem ;
A cżtery beſtyje wielkie wyſtępowáły z morzá / różne jedná od drugiej.
Pierwƺá podobná lwowi / májąc ſkrzydłá orle ; y przypátrywáłem śię / áż wyrwáne były ſkrzydłá jej / któremi śię podnośiłá od ziemi / ták / że ná nogách jáko cżłowiek ſtánęłá / á ſerce cżłowiecże jej dáne jeſt.
Potem oto beſtyjá drugá podobná niedźwiedziowi ; y ſtánęłá przy jednej ſtronie / á trzy żebrá były w páƺcżęce jej między zębámi jej / y ták mówiono do niej : Wſtáń / náżryj śię doſtátkiem mięſá.
Potemem widziáł / á oto inná beſtyjá podobná lámpártowi / którá miáłá cżtery ſkrzydłá ptáƺe ná grzbiećie ſwym / cżtery też głowy miáłá tá beſtyjá / y dáno jej włádzę wielką.
Potemem widziáł w widzeniách nocnych / á oto beſtyjá cżwártá ſtráƺná y ſrogá y bárdzo mocná / májącá zęby żelázne wielkie / którá pożeráłá y kruƺyłá / á oſtátek nogámi ſwemi deptáłá ; á tá byłá różná od wƺyſtkich beſtyj / które były przed nią / y miáł á dzieśięć rogów.
Pilniem śię przypátrywáł rogom / á oto róg pośledni máły wyráſtáł między niemi / y trzy z tych rogów pierwƺych wyłámáne ſą przed nim ; á oto w onym rogu były ocży podobne ocżom cżłowiecżym / y uſtá mówiące rzecży wielkie.
Y przypátrywáłem śię / áż one ſtolice poſtáwione były / á Stárodáwny uśiádł / którego ƺátá byłá jáko śnieg biáłá / á włoſy głowy jego jáko wełná cżyſtá / ſtolicá jego jáko płomienie ogniſte / á kołá jej jáko ogień gorejący.
Rzeká ogniſtá płynąc wychodziłá od oblicżá jego. Tyśiąc tyśięcy ſłużyło mu / á dzieśięć kroć tyśiąc tyśięcy ſtáło przed nim ; ſąd záśiádł / á kśięgi otworzone były.
Tedym śię przypátrywáł / ſkoro śię głoſ pocżął tych ſłów wielkich / które on róg mówił ; przypátrywáłem śię / áż byłá tá beſtyjá zábitá / y zginęło ćiáło jej / á podáne było ná ſpálenie ogniem.
Tákże y pozoſtáłym beſtyjom odjętá jeſt włádzá ich ; bo długość żywotá dáná im byłá áż do cżáſu / á to do zámierzonego cżáſu.
Widziáłem też w widzeniu nocnem / á oto przychodził w obłokách niebieſkich podobny ſynowi cżłowiecżemu / á przyƺedł áż do Stárodáwnego / y przywiedziono go przed oblicżność jego.
Y dáł mu włádzę y cżeść y króleſtwo / áby mu wƺyſcy ludzie / národy y języki ſłużyli ; włádzá jego włádzá wiecżná / którá nie będzie odjętá / á króleſtwo jego / które nie będzie ſkáżone.
Y zátrwożył śię we mnie Dánijelu duch mój w pośród ćiáłá mego / á widzeniá / którem widziáł / przeſtráƺyły mię.
Tedym przyſtąpił do jednego z tych / którzy tám ſtáli / á pewnośćim śię dowiádywáł od niego o tem wƺyſtkiem / y powiedziáł mi / y wykłád mów oznájmił mi.
Te beſtyje wielkie / których ſą cżtery / ſą cżtery królowie / którzy powſtáną z ziemi.
Ci ujmą króleſtwo świętych nájwyżƺych miejſc / którzy pośiąść máją króleſtwo áż ná wieki / y áż ná wieki wiecżne.
Tedym prágnął wziąć ſpráwę o beſtyi cżwártej / którá byłá różná od wƺyſtkich innych / bárdzo ſtráƺná / której zęby były żelázne / á páznogćie jej miedziáne ; którá pożeráłá y kruƺyłá / á oſtátek nogámi ſwemi deptáłá.
Tákże o onych rogách dzieśięćiu / które były ná głowie jej / y o poślednim / który był wyróſł / przed którym wypádły trzy ; o tym rogu mówię / który miáł ocży y uſtá mówiące wielkie rzecży / á ná wejrzeniu był więkƺy nád inne rogi.
Y przypátrywáłem śię / á oto róg ten wálcżył z świętymi / y przemágáł ich ;
Aż przyƺedł Stárodáwny á podány jeſt ſąd świętym nájwyżƺych miejſc / á cżáſ przyƺedł / áby to króleſtwo święći otrzymáli.
Y rzekł ták : Beſtyjá cżwártá / cżwárte króleſtwo znácży ná ziemi / które będzie różne od wƺyſtkich króleſtw / á pożre wƺyſtkę ziemię / á podepcże á pokruƺy ją ;
A dzieśięć rogów to znácży / że z króleſtwá onego dzieśięć królów powſtánie ; á po nich powſtánie pośledni / który będzie różny od pierwƺych / y trzech królów poniży ;
A ſłowá przećiw Nájwyżƺemu mówić będzie / y święte nájwyżƺych miejſc zetrze ; nádto będzie zámyśláł / áby odmienił cżáſy y práwá / gdyż wydáne będą w ręce jego áż do cżáſu y cżáſów / y pół cżáſu.
Potem záśiądzie ſąd / á tám włádzę jego odejmą / áby był zniƺcżony y wytrácony áż do końcá.
A króleſtwo y włádzá / y doſtojeńſtwo królewſkie pod wƺyſtkiem niebem dáne będzie ludowi świętych nájwyżƺych miejſc / którego króleſtwo będzie króleſtwo wiecżne / á wƺyſtkie zwierzchnośći jemu ſłużyć y onego ſłucháć będą.
Aż dotąd koniec tych ſłów. A mnie Dánijelá myśli moje wielce zátrwożyły / á jáſność mojá zmieniłá śię przy mnie ; wƺákżem to ſłowo w ſercu mojem záchowáł.
