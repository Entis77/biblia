Já tedy roku pierwƺego zá Dáryjuƺá Medſkiego ſtánąłem / ábym go pośilił y zmocnił.
A teráz ći práwdę oznájmię : Oto jeƺcże trzej królowie królowáć będą w Perſkiej ziemi ; potem cżwárty zbogáći śię bogáctwy wielkimi náde wƺyſtkich / á gdy śię zmocni w bogáctwách ſwoich / pobudzi wƺyſtkich przećiw króleſtwu Greckiemu.
Y powſtánie król mocny / á będzie pánowáł mocą wielką / á będzie cżynił według woli ſwojej.
A gdy śię on zmocni / będzie ſkruƺone króleſtwo jego / y będzie rozdzielone ná cżtery ſtrony świátá / wƺákże nie między potomków jego / áni będzie páńſtwo jego tákie / jákie było ; bo wykorzenione będzie króleſtwo jego / á innym mimo onych doſtánie śię.
Tedy śię zmocni król z południá y jeden z kśiążąt jego ; ten mocniejƺy będzie nádeń / y pánowáć będzie / á páńſtwo jego będzie páńſtwo ƺerokie.
Lecż po wyjśćiu kilku lát złącżą śię ; bo córká królá od południá pójdzie zá królá północnego / áby ucżyniłá przymierze ; wƺákże nie otrzymá śiły rámieniá / áni śię oſtoi z rámieniem ſwojim / ále wydáná będzie oná / y ći / którzy ją przyprowádzą / y ſyn jej / y ten / co ją zmácniáł zá onych cżáſów.
Potem powſtánie z látorośli korzeniá jej ná miejſce jego / który przyćiągnie z wojſkiem ſwem / á uderzy ná miejſce obronne królá północnego / y przewiedzie nád nimi y zmocni śię.
Nádto y bogów ich z kśiążętámi ich / z nácżyniem ich drogiem / ſrebrnem y złotem w niewolę záwiedzie do Egiptu ; á ten będzie bezpiecżen przez wiele lát od królá północnego.
A ták wtárgnie w króleſtwo król od południá / y wróći śię do ziemi ſwojej.
Ale ſynowie jego wálcżyć będą / y zbiorą mnóſtwo wojſk wielkich ; á z nágłá náſtępując jáko powódź przechodzić będzie / potem wrácájąc śię / wojſkiem náćieráć będzie áż ná twierdze jego.
Skąd rozdráżniony będąc król z południá wyćiągnie / y będzie wálcżył z nim / to jeſt / z królem północnym ; á uƺykuje mnóſtwo wielkie / ále ono mnóſtwo będzie podáne w rękę jego.
A gdy znieśione będzie ono mnóſtwo / podnieśie śię ſerce jego ; á choć porázi wiele tyśięcy / przećie śię nie zmocni.
Bo śię wróći król północny / y uƺykuje więkƺe mnóſtwo niż pierwƺe ; lecż po wyjśćiu cżáſu kilku lát z nágłá przyjdzie z wielkiem wojſkiem y z wielkim doſtátkiem.
Onychże cżáſów wiele ich powſtánie przećiwko królowi z południá ; ále ſynowie przeſtępników z ludu twego będą znieśieni dlá utwierdzeniá tego widzeniá / y upádną.
Bo przyćiągnie król z północy / y uſypie wáły / y weźmie miáſto obronne / á rámioná południowe nie oprą śię / áni lud jego wybrány / y nie ſtánie im śiły / áby dáli odpór.
Y ucżyni on / który przyćiągnie przećiwko niemu / według woli ſwojej / y nie będzie nikogo / coby śię ſtáwił przećiwko niemu ; ſtáwi śię też w ziemi ozdobnej / którá zniƺcżeje przez rękę jego.
Potem obróći twárz ſwoję / áby przyƺedł z mocą wƺyſtkiego króleſtwá ſwego / y okázáł śię / jákoby zgody ƺukáł / y ucżyni coś ; bo mu dá córkę piękną / áby go zgubił przez nią ; ále oná w tem nie będzie ſtátecżná / y nie będzie z nim przeſtáwáłá.
Zátem obróći twárz ſwoję do wyſep / y wiele ich pobierze ; ále wódz wſtręt ucżyni háńbieniu jego / owƺem / ono háńbienie jego náń obróći.
Dlácżego obróći twárz ſwoję ku twierdzom ziemi ſwej ; lecż śię potknie y upádnie / y nie będzie więcej ználeziony.
Y powſtánie ná miejſce jego táki / który roześle poborców w ſłáwie królewſkiej ; ále ten po niewielu dniách ſtárty będzie / á to nie w gniewie áni przez wojnę.
Potem powſtánie wzgárdzony ná jego miejſce / ácż nie włożą náń ozdoby królewſkiej ; wƺákże przyƺedłƺy w pokoju / otrzymá króleſtwo pochlebſtwem.
A rámionámi jáko powodzią wiele ich záchwyceni będą przed oblicżem jego / y ſkruƺeni będą / tákże też y ſám wódz / który z nimi przymierze ucżynił.
Bo wƺedłƺy z nimi w przyjáźń / ucżyni zdrádę / á przyćiągnąwƺy zmocni śię w máłym pocżćie ludu.
Bezpiecżnie y do nájobfitƺych miejſc onej kráiny wpádnie / á ucżyni to / cżego nie cżynili ojcowie jego / áni ojcowie ojców jego ; łup y korzyść y májętnośći rozdzieli im / náwet y o miejſcách obronnych chytrze przemyśliwáć będzie / á to áż do cżáſu.
Potem wzbudzi moc ſwoję / y ſerce ſwoje przećiw królowi z południá z wojſkiem wielkiem / z którem król z południá wálecżnie śię potykáć będzie z wojſkiem wielkiem y bárdzo mocnem ; ále śię nie oprze / przeto / że wymyśli przećiwko niemu zdrádę.
Bo ći / którzy jedzą chleb jego / zniƺcżą go / gdy wojſko onego jáko powódź przypádnie / á pobitych wiele polęże.
Nátencżáſ obáj królowie w ſercu ſwem myślić będą / jákoby jeden drugiemu ƺkodzić mógł / á przy jednymże ſtole kłámſtwo mówić będą ; ále śię im nie nádá / gdyż jeƺcże koniec ná inƺy cżáſ odłożony jeſt.
Przetoż śię wróći do ziemi ſwojej z bogáctwy wielkiemi / á ſerce jego obróći śię przećiwko przymierzu świętemu ; co ucżyniwƺy wróći śię do ziemi ſwojej.
A cżáſu zámierzonego wróći śię y poćiągnie ná południe ; ále mu śię nie ták powiedzie / jáko zá pierwƺym y zá oſtátnim rázem.
Bo przyjdą przećiwko niemu okręty z Cytym / ſkąd on nád tem bolejąc znowu śię rozgniewá przećiwko przymierzu świętemu ; co ucżyniwƺy wróći śię / á będzie miáł porozumienie z onymi / którzy opuśćili przymierze święte ;
A wojſká wielkie przy nim ſtáć będą / które ſplugáwią świątnicę / y twierdze znioſą ; odejmą też uſtáwicżną ofiárę / á poſtáwią obrzydliwość ſpuſtoƺeniá.
Ták áby tych / którzy niezbożnie przećiwko przymierzu poſtępowáć będą / w obłudzie pochlebſtwem utwierdził / á żeby lud znájący Bogá ſwego imáli / co też ucżynią.
Zácżem ći / którzy náucżáją lud / którzy náucżáją wielu / pádáć będą od miecżá y od ogniá / od pojmániá y od łupu przez wiele dni.
A gdy pádáć będą / máłą pomoc mieć będą ; bo śię do nich wiele pochlebców przyłącży.
A z tych / którzy innych náucżáją / pádáć będą / áby doświádcżeni y ocżyƺcżeni / y wybieleni byli áż do cżáſu zámierzonego ; bo to jeƺcże potrwá áż do cżáſu zámierzonego.
Ták ucżyni król według woli ſwojej / y podnieśie śię y wielmożnym śię ucżyni nád káżdego bogá / y przećiwko Bogu nád bogámi dziwne rzecży mówić będzie / y poƺcżęśći mu śię / áż śię dokoná gniew / áżby śię to / co jeſt poſtánowiono / wykonáło.
Ani ná bogów ojców ſwych nie będzie dbáł / áni o miłość niewiáſt / áni o żádnego Bogá dbáć będzie / przeto / że śię náde wƺyſtko wynieśie.
A ná miejſcu Bogá nájmocniejƺego cżćić będzie bogá / którego nie ználi ojcowie jego ; cżćić będzie złotem y ſrebrem y kámieniem drogiem y rzecżámi koƺtownemi.
A ták dowiedzie tego / że twierdze Nájmocniejƺego będą bogá obcego ; á których mu śię będzie zdáło / tych rozmnoży ſłáwę / y ucżyni / áby pánowáli nád wielą / á rozdzieli im ziemię miáſto zápłáty.
A przy ſkońcżeniu tego cżáſu będzie śię z nim potykáł król z południá ; ále król północny jáko burzá náń przyjdzie z wozámi y z jezdnymi y z wielą okrętów / á wtárgnie w ziemię / y jáko powódź przejdzie.
Potem przyćiągnie do ziemi ozdobnej / y wiele kráin upádnie ; wƺákże ći ujdą rąk jego / Edomcżycy y Moábcżycy / y pierwoćiny ſynów Ammonowych.
A gdy rękę ſwą śćiągnie ná kráiny / áni ziemiá Egipſká tego ujść nie będzie mogłá.
Bo opánuje ſkárby złotá y ſrebrá / y wƺyſtkie rzecży drogie Egipſkie / á Libijcżycy y Murzynowie zá nim pójdą.
W tem wieśći od wſchodu ſłońcá y od północy przeſtráƺą go ; przetoż wyćiągnie z popędliwośćią wielką / áby wygubił y zámordowáł wielu.
Y rozbije námioty páłácu ſwego między morzámi ná górze ozdobnej świętobliwośći ; á gdy przyjdzie do końcá ſwego / nie będzie miáł nikogo ná pomocy.
