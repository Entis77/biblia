Ja tedy roku pierwƺego zá Dáryjuƺá Medſkiego ſtánąłem / ábym go pośilił y zmocnił.
A teraz ći práwdę oznájmię : Oto jeƺcże trzej królowie królowáć będą w Perſkiej źiemi ; potem cżwárty zbogáći śię bogáctwy wielkimi náde wƺyſtkich / á gdy śię zmocni w bogáctwách ſwojich / pobudźi wƺyſtkich przećiw króleſtwu Greckiemu.
Y powſtánie król mocny / á będźie pánował mocą wielką / á będźie cżynił według woli ſwojey.
A gdy śię on zmocni / będźie ſkruƺone króleſtwo jego / y będźie rozdźielone ná cżtery ſtrony świátá / wƺákże nie miedzy potomków jego / áni będźie páńſtwo jego tákie / jákie było ; bo wykorzenione będźie króleſtwo jego / á innym mimo onych doſtánie śię.
Tedy śię zmocni król z południá y jeden z kśiążąt jego ; ten mocniejƺy będźie nádeń / y pánowáć będźie / á páńſtwo jego będźie páńſtwo ƺerokie.
Lecż po wyśćiu kilku lat złącżą śię ; bo córká królá od południá pójdźie zá królá północnego / áby ucżyniłá przymierze ; wƺákże nie otrzymá śiły rámieniá / áni śię oſtoi z rámieniem ſwojim / ále wydáná będźie oná / y ći / którzy ją przyprowádzą / y ſyn jey / y ten / co ją zmácniał zá onych cżáſów.
Potem powſtánie z látorośli korzeniá jey ná miejſce jego / który przyćiągnie z wojſkiem ſwem / á uderzy ná miejſce obronne królá północnego / y przewiedźie nád nimi y zmocni śię.
Nádto y bogów ich z kśiążętámi ich / z nácżyniem ich drogiem / ſrebrnem y złotem w niewolę záwiedźie do Egiptu ; á ten będźie bezpiecżen przez wiele lat od królá północnego.
A ták wtárgnie w króleſtwo król od południá / y wróći śię do źiemi ſwojey.
Ale ſynowie jego wálcżyć będą / y zbiorą mnóſtwo wojſk wielkich ; á z nágłá náſtępując jáko powódź przechodźić będźie / potem wrácájąc śię / wojſkiem náćieráć będźie áż ná twierdze jego.
Skąd rozdráżniony będąc król z południá wyćiągnie / y będźie wálcżył z nim / to jeſt / z królem północnym ; á uƺykuje mnóſtwo wielkie / ále ono mnóſtwo będźie podáne w rękę jego.
A gdy znieśione będźie ono mnóſtwo / podnieśie śię ſerce jego ; á choć poráźi wiele tyśięcy / przećie śię nie zmocni.
Bo śię wróći król północny / y uƺykuje więkƺe mnóſtwo niż pierwƺe ; lecż po wyśćiu cżáſu kilku lat z nágłá przydźie z wielkiem wojſkiem y z wielkim doſtátkiem.
Onychże cżáſów wiele ich powſtánie przećiwko królowi z południá ; ále ſynowie przeſtępników z ludu twego będą znieśieni dla utwierdzeniá tego widzeniá / y upádną.
Bo przyćiągnie król z północy / y uſypie wáły / y weźmie miáſto obronne / á rámioná południowe nie oprą śię / áni lud jego wybrány / y nie ſtánie im śiły / áby dáli odpór.
Y ucżyni on / który przyćiągnie przećiwko niemu / według woli ſwojey / y nie będźie nikogo / coby śię ſtáwił przećiwko niemu ; ſtáwi śię też w źiemi ozdobney / która zniƺcżeje przez rękę jego.
Potem obróći twárz ſwoję / áby przyƺedł z mocą wƺyſtkiego króleſtwá ſwego / y okázał śię / jákoby zgody ƺukał / y ucżyni coś ; bo mu da córkę piękną / áby go zgubił przez nią ; ále oná w tym nie będźie ſtátecżná / y nie będźie z nim przeſtáwáłá.
Zátem obróći twárz ſwoję do wyſep / y wiele ich pobierze ; ále wódz wſtręt ucżyni háńbieniu jego / owƺem / ono háńbienie jego nań obróći.
Dlácżego obróći twárz ſwoję ku twierdzom źiemi ſwej ; lecż śię potknie y upádnie / y nie będźie więcey ználeźiony.
Y powſtánie ná miejſce jego táki / który roześle poborców w ſłáwie królewſkiej ; ále ten po niewielu dniách ſtárty będźie / á to nie w gniewie áni przez wojnę.
Potem powſtánie wzgárdzony ná jego miejſce / ácż nie włożą nań ozdoby królewſkiej ; wƺákże przyƺedƺy w pokoju / otrzymá króleſtwo pochlebſtwem.
A rámionámi jáko powodźią wiele ich záchwyceni będą przed oblicżem jego / y ſkruƺeni będą / tákże też y ſám wódz / który z nimi przymierze ucżynił.
Bo wƺedƺy z nimi w przyjáźń / ucżyni zdrádę / á przyćiągnąwƺy zmocni śię w máłym pocżćie ludu.
Bezpiecżnie y do nájobfitƺych miejſc oney kráiny wpádnie / á ucżyni to / cżego nie cżynili ojcowie jego / áni ojcowie ojców jego ; łup y korzyść y májętnośći rozdźieli im / náwet y o miejſcách obronnych chytrze przemyśliwáć będźie / á to áż do cżáſu.
Potem wzbudźi moc ſwoję / y ſerce ſwoje przećiw królowi z południá z wojſkiem wielkiem / z którem król z południá wálecżnie śię potykáć będźie z wojſkiem wielkiem y bárzo mocnem ; ále śię nie oprze / przeto / że wymyśli przećiwko niemu zdrádę.
Bo ći / którzy jedzą chleb jego / zniƺcżą go / gdy wojſko onego jáko powódź przypádnie / á pobitych wiele polęże.
Nátencżás obáj królowie w ſercu ſwem myślić będą / jákoby jeden drugiemu ƺkodźić mógł / á przy jednymże ſtole kłámſtwo mówić będą ; ále śię im nie nádá / gdyż jeƺcże koniec ná inƺy cżás odłożony jeſt.
Przetoż śię wróći do źiemi ſwojey z bogáctwy wielkiemi / á ſerce jego obróći śię przećiwko przymierzu świętemu ; co ucżyniwƺy wróći śię do źiemi ſwojey.
A cżáſu zámierzonego wróći śię y poćiągnie ná południe ; ále mu śię nie ták powiedźie / jáko zá pierwƺym y zá oſtátnim rázem.
Bo przyjdą przećiwko niemu okręty z Cytym / ſkąd on nád tym bolejąc znowu śię rozgniewá przećiwko przymierzu świętemu ; co ucżyniwƺy wróći śię / á będźie miał porozumienie z onymi / którzy opuśćili przymierze święte ;
A wojſká wielkie przy nim ſtáć będą / które ſplugáwią świątnicę / y twierdze znioſą ; odejmą też uſtáwicżną ofiárę / á poſtáwią obrzydliwość ſpuſtoƺeniá.
Ták áby tych / którzy niezbożnie przećiwko przymierzu poſtępowáć będą / w obłudźie pochlebſtwem utwierdźił / á żeby lud znájący Bogá ſwego imáli / co też ucżynią.
Zácżem ći / którzy náucżáją lud / którzy náucżáją wielu / pádáć będą od miecżá y od ogniá / od pojmánia y od łupu przez wiele dni.
A gdy pádáć będą / máłą pomoc mieć będą ; bo śię do nich wiele pochlebców przyłącży.
A z tych / którzy innych náucżáją / pádáć będą / áby doświádcżeni y ocżyƺcżeni / y wybieleni byli áż do cżáſu zámierzonego ; bo to jeƺcże potrwá áż do cżáſu zámierzonego.
Ták ucżyni król według woli ſwojey / y podnieśie śię y wielmożnym śię ucżyni nád káżdego bogá / y przećiwko Bogu nád bogámi dźiwne rzecży mówić będźie / y poƺcżęśći mu śię / áż śię dokoná gniew / áżby śię to / co jeſt poſtánowiono / wykonáło.
Ani ná bogów ojców ſwych nie będźie dbał / áni o miłość niewiáſt / áni o żádnego Bogá dbáć będźie / przeto / że śię náde wƺyſtko wynieśie.
A ná miejſcu Bogá nájmocniejƺego cżćić będźie bogá / którego nie ználi ojcowie jego ; cżćić będźie złotem y ſrebrem y kámieniem drogiem y rzecżámi koƺtownemi.
A ták dowiedźie tego / że twierdze Nájmocniejƺego będą bogá obcego ; á których mu śię będźie zdáło / tych rozmnoży ſłáwę / y ucżyni / áby pánowáli nád wielą / á rozdźieli im źiemię miáſto zápłáty.
A przy ſkońcżeniu tego cżáſu będźie śię z nim potykał król z południá ; ále król północny jáko burzá nań przydźie z wozámi y z jezdnymi y z wielą okrętów / á wtárgnie w źiemię / y jáko powódź przejdźie.
Potem przyćiągnie do źiemi ozdobney / y wiele kráin upádnie ; wƺákże ći ujdą rąk jego / Edomcżycy y Moábcżycy / y pierwoćiny ſynów Ammonowych.
A gdy rękę ſwą śćiągnie ná kráiny / áni źiemiá Egipſká tego ujść nie będźie mogłá.
Bo opánuje ſkárby złotá y ſrebrá / y wƺyſtkie rzecży drogie Egipſkie / á Libijcżycy y Murzynowie zá nim pójdą.
W tym wieśći od wſchodu ſłońcá y od północy przeſtráƺą go ; przetoż wyćiągnie z popędliwośćią wielką / áby wygubił y zámordował wielu.
Y rozbije námioty páłácu ſwego miedzy morzámi ná górze ozdobney świętobliwośći ; á gdy przydźie do końcá ſwego / nie będźie miał nikogo ná pomocy.
