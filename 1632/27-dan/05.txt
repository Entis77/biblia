Bálſázár król ucżynił ucżtę wielką ná tyśiąc kśiążąt ſwojich / y przed onym tyśiącem pił wino.
A gdy pił wino Bálſázár / rozkázał przynieść nacżynie złote y ſrebrne / które był zábrał Nábuchodonozor / oćiec jego / z kośćiołá Jeruzálemſkiego / áby pili z niego król y kśiążętá jego / żony jego / y záłożnice jego.
Tedy przynieśiono nacżynia złote / które byli zábráli z kośćiołá domu Bożego / który był w Jeruzálemie / y pili z nich król y kśiążętá jego / żony jego / y záłożnice jego ;
A pijąc wino chwálili bogi złote y ſrebrne / miedźiáne / żelázne / drewniáne / y kámienne.
Tejże godźiny wyƺły pálce ręki cżłowiecżey / które piſáły przećiwko świecżnikowi ná śćiánie páłácu królewſkiego / á król widźiał cżęść ręki / która piſáłá.
Tedy śię jáſność królewſká zmieniłá / á myśli jego zátrwożyły nim / y zwiąſki biódr jego rozwiązáły śię / á koláná jego jedno o drugie śię tłukły.
Y záwołał król ze wƺyſtkiej śiły / áby przywiedźiono práktykárzy / Cháldejcżyków y wieƺcżków. A mówiąc król rzekł do mędrców Bábilońſkich : Ktokolwiek to piſmo przecżytá / á wykłád jego mnie oznájmi / oblecżony będźie w ƺárłát / á łáńcuch złoty dádzą ná ƺyję jego / y trzećim w króleſtwie po mnie będźie.
Tedy weƺli wƺyſcy mędrcy królewſcy ; ále nie mogli áni piſmá przecżytáć / áni wykłádu jego królowi oznájmić.
Skąd król Bálſázár był bárzo zátrwożony / á jáſność jego zmieniłá śię ná nim / y kśiążętá jego potrwożyli śię.
Tedy królowá weƺłá do domu ucżty dla tego / co śię przydáło królowi y kśiążętom jego ; á przemówiwƺy królowá rzekłá : Królu / żyj ná wieki! Niech ćię nie trwożą myśli twoje / á jáſność twojá niech śię nie mieni.
Jeſt mąż w króleſtwie twojim / w którym jeſt duch bogów świętych / w którym śię ználázło zá dni ojcá twego oświecenie / y rozum / y mądrość / jáko mądrość bogów / którego król Nábuchodonozor / oćiec twój / przedniejƺym miedzy mędrcámi / y práktykárzámi Cháldejcżykámi / y wieƺcżkámi / poſtánowił / oćiec twój mówię / o królu!
Dla tego / iż duch obfity / y umiejętność / y zrozumienie / wykłádánie ſnów / y objáwienie zágádek / y rozwiązánie rzecży trudnych ználázły śię przy Dánijelu / któremu król dał imię Báltázár / teraz tedy niech przyzowią Dánijelá / á oznájmić ten wykłád.
Tedy przywiedźiony jeſt Dánijel do królá ; á król mówiąc rzekł Dánijelowi : Tyżeś jeſt on Dánijel / któryś jeſt z ſynów więźniów Judzkich / którego przywiódł król / oćiec mój / z źiemi Judzkiej?
Słyƺáłem zájiſte o tobie / iż duch bogów jeſt w tobie / á oświecenie y rozum y mądrość obfitá ználázłá śię w tobie.
A teraz przywiedźiono przed mię mędrców y práktykárzy / áby mi to piſmo przecżytáli / y wykłád jego oznájmili : wƺákże nie mogli wykłádu tey rzecży oznájmić.
A jam ſłyƺał o tobie / że możeƺ to / co jeſt niepojętego / wykłádáć / á co jeſt trudnego / rozwiązywáć ; przetoż teraz / możeƺli to piſmo przecżytáć á wykłád jego mnie oznájmić / w ƺárłát oblecżony będźieƺ / y łáńcuch złoty ná ƺyję twoję włożony będźie / á trzećim w króleſtwie po mnie będźieƺ.
Tedy odpowiedźiał Dánijel przed królem y rzekł : Upominki twoje niech tobie zoſtáną / á dáry twoje dáj innemu ; wƺákże piſmo przecżytám królowi / y wykłád mu oznájmię.
Ty / królu! ſłucháj. Bóg nájwyżƺy króleſtwo y wielmożność y ſłáwę / y zácność dał Nábuchodonozorowi / ojcu twemu ;
A dla wielmożnośći / którą mu był dał / wƺyſcy ludźie / narody y języki drżeli y báli śię przed oblicżem jego ; bo kogo chćiał / zábijał / á kogo chćiał / żywił / á kogo chćiał / wywyżƺał / á kogo chćiał / poniżał.
Ale gdy śię wynioſło ſerce jego / á duch jego zmocnił śię w pyƺe / złożony jeſt z ſtolicy króleſtwá ſwego / á ſłáwá odjętá byłá od niego ;
Y był wyrzucony od ſynów ludzkich / á ſerce jego zwierzęcemu podobne było / y z dźikiemi oſłámi było mieƺkánie jego ; tráwą śię páſł jáko wół / y roſą niebieſką ćiáło jego ſkrápiáne było / dokąd nie poznał / że Bóg nájwyżƺy ma włádzę nád króleſtwem ludzkiem / á tego / kogo chce / ſtánowi nád nim.
Ty / też / Bálſázárze / ſynu jego! nie upokorzyłeś ſercá ſwego / choćiáżeś to wƺyſtko wiedźiał.
Owƺem / przećiwko Pánu niebá podnioſłeś śię / y nacżynie domu jego przynieśiono przed ćię ; á ty y kśiążętá twoi / żony twoje / y záłożnice twoje / piliśćie wino z niego ; nádto bogi ſrebrne y złote / miedźiáne / żelázne / drewniáne / y kámienne / którzy nie widzą / áni ſłyƺą / y nic nie wiedzą / chwáliłeś / á Bogá / w którego ręku jeſt tchnienie twoje / y u którego ſą wƺyſtkie drogi twoje / nie ucżćiłeś.
Przetoż teraz od niego poſłáná jeſt tá cżęść ręki / y piſmo to wyráżone jeſt.
A toć jeſt piſmo / które wyráżone jeſt : Mene / Mene / Thekel / uphárśin.
A tenći jeſt wykłád tych ſłów : Mene / zlicżył Bóg króleſtwo twoje y do końcá je przywiódł.
Thekel / zwáżonyś ná wádze / á ználeźionyś lekki.
Peres / rozdźielone jeſt króleſtwo twoje / á dáne jeſt Medom y Perſom.
Tedy rozkázał Bálſázár ; y oblecżono Dánijelá w ƺárłát / á łáńcuch złoty włożono ná ƺyję jego / y obwołáno o nim / że ma być trzećim pánem w króleſtwie.
Tejże nocy zábity jeſt Bálſázár / król Cháldejſki.
A Dáryjuƺ / Medcżyk / ujął króleſtwo / májąc lat około ƺeśćdźieśiąt y dwá.
