Roku wtórego królowánia Nábuchodonozorá miał Nábuchodonozor ſen / y ſtrwożył śię duch jego / y przerwał mu śię ſen jego.
Tedy król rozkázał zwołáć mędrców / y práktykárzy / y cżárnokśiężników / y Cháldejcżyków / áby oznájmili królowi ſen jego ; którzy przyƺli y ſtánęli przed oblicżem królewſkiem.
Y rzekł król do nich : Miáłem ſen / y ſtrwożył śię duch mój / ták / że nie wiem / co mi śię śniło.
Tedy odpowiedźieli Cháldejcżycy królowi po ſyryjſku : Królu / żyj ná wieki! Powiedz ſen ſługom twoim / á oznájmiemyć wykłád jego.
Odpowiedźiał król y rzekł do Cháldejcżyków : Tá rzecż już mi z pámięći wypádłá ; jeſli mi nie oznájmićie ſnu y wykłádu jego / ná ƺtuki rozśiekáni będźiećie / á domy wáƺe w gnojowiſko obrócone będą ;
Ale jeſli mi ſen y wykłád jego oznájmićie / dáry y upominki y ucżćiwość wielką odnieśiećie odemnie ; przetoż ſen y wykłád jego oznájmijćie mi.
Odpowiedźieli powtóre / y rzekli : Król niech powie ſen ſługom ſwoim / á wykłád jego oznájmiemy.
Odpowiedźiał król / y rzekł : Zájiſte wiem / że umyślnie odwłácżáćie / bácżąc / że mi ten ſen z pámięći wyƺedł.
Jeſliże mi ſnu nie oznájmićie / pewny jeſt o was dekret / bośćie rzecż kłámliwą y przewrotną umyślili mówić przedemną / áżby cżás przeminął ; przetoż mi ſen powiedzćie / á dowiem śię / będźiećieli mogli wykłád jego oznájmić.
Odpowiedźieli Cháldejcżycy królowi / y rzekli : Niemáƺ cżłowieká ná źiemi / któryby tę rzecż królowi oznájmić mógł ; dotego żáden król / kśiążę álbo pán o táką śię rzecż nie pytał żádnego mędrcá / y práktykárzá / y Cháldejcżyká.
Bo rzecż / o którą śię król pytá / trudná jeſt / á niemáƺ nikogo / coby ją mógł królowi oznájmić / oprócż bogów / którzy nie mieƺkáją z ludźmi.
Z tey przycżyny záſrożył śię król / y rozgniewał śię bárzo / á rozkázał wytráćić wƺyſtkich mędrców Bábilońſkich.
A gdy wyƺedł dekret / áby mordowano mędrców / ƺukáno y Dánijelá y towárzyƺów jego / áby ich zámordowano.
Tedy Dánijel odpowiedźiał mądrze y roztropnie Aryjochowi / hetmánowi nád żołnierzámi królewſkimi / który wyƺedł / áby zábijał mędrców Bábilońſkich ;
A odpowiádájąc rzekł do Aryjochá / hetmáná królewſkiego : Przecż ten dekret ták prędko wyƺedł od królá? Y oznájmił tę rzecż Aryjoch Dánijelowi.
Skąd Dánijel wƺedł / y prośił królá / áby mu dał cżás ná oznájmienie wykłádu królowi.
Odƺedƺy tedy Dánijel do domu ſwego oznájmił tę rzecż Anánijáƺowi / Miſáelowi y Azáryjáƺowi / towárzyƺom ſwoim /
Aby o miłośierdźie prośili Bogá niebieſkiego dla tey tájemnicy / żeby nie zginęli Dánijel y towárzyƺe jego z pozoſtáłymi mędrcámi Bábilońſkimi.
Tedy objáwioná jeſt Dánijelowi w widzeniu nocnem tá tájemnicá / zá co Dánijel błogoſłáwił Bogu niebieſkiemu.
A mówiąc Dánijel rzekł : Niech będźie imię Boże błogoſłáwione od wieku áż ná wieki ; ábowiem mądrość y moc jego jeſt ;
On ſám odmieniá cżáſy y chwile ; zrzucá królów y ſtánowi królów ; dáje mądrość mądrym / á umiejętnym rozum ;
On odkrywá rzecży głębokie y ſkryte / zná / co jeſt w ćiemnośćiách / á świátłość z nim mieƺká.
Ciebie ja / o Boże ojców mojich! wyſłáwiám y chwálę / żeś mi dał mądrość y moc / owƺem / żeś mi teraz oznájmił to / o cośmy ćię prośili ; boś nam ſen królewſki oznájmił.
Dla tego Dánijel wƺedł do Aryjochá / którego był poſtánowił król / áby wytráćił mędrców Bábilońſkich ; á przyƺedƺy ták rzekł do niego : Nie tráć mędrców Bábilońſkich / wprowádź mię do królá / á ja ten wykłád królowi oznájmię.
Tedy Aryjoch z kwápieniem wprowádźił Dánijelá do królá y ták mu rzekł : Ználázłem mężá z więźniów ſynów Judzkich / który ten wykłád królowi oznájmi.
Odpowiedźiał król / y rzekł Dánijelowi / któremu imię było Báltázár : Izali mnie ty możeƺ oznájmić ſen / którym widźiał / y wykłád jego?
Odpowiedźiał Dánijel królowi / y rzekł : Tájemnicy / o którey śię król pytá / mędrcy / práktykárze / cżárnokśiężnicy y wieƺcżkowie królowi oznájmić nie mogą ;
A wƺákże jeſt Bóg ná niebie / który objáwiá tájemnice / á on okázał królowi Nábuchodonozorowi / co ma być potomnych dni. Sen twój y widzeniá / któreś widźiał ná łożu twojim / te ſą :
Tobie o królu! przychodźiło ná myśl ná łożu twojim / coby miáło być ná potem / á ten / który odkrywá tájemnice / oznájmił ći to / co ma być.
Mnie też nie przez mądrość / któráby przy mnie byłá nád wƺyſtkich ludźi / tájemnicá tá objáwioná jeſt / ále przez modlitwę / áby ten wykłád królowi oznájmiony był / á iżbyś śię myśli ſercá twego dowiedźiał.
Tyś / królu! widźiał / á oto obraz jeden wielki ( obraz to był wielki / á bláſk jego znácżny ) ſtał przećiwko tobie / który ná wejrzeniu był ſtráƺny.
Tego obrázu głowá byłá ze złotá ƺcżerego / pierśi jego y rámioná jego ze ſrebrá / brzuch jego y biodrá jego z miedźi ;
Golenie jego z żelázá / nogi jego cżęśćią z żelázá / á cżęśćią z gliny.
Pátrzáłeś ná to / áż odćięty był kámień / który nie bywał w ręku / á uderzył ten obraz w nogi jego żelázne y gliniáne / y ſkruƺył je.
Tedy śię ſkruƺyło ſpołem żelázo / gliná / miedź / ſrebro y złoto / á było to wƺyſtko jáko plewy ná bojewiſku w lećie / y roznióſł to wiátr / ták / że ich ná żádnem miejſcu nie ználeźiono ; á kámień on / który uderzył o obraz / ſtał śię górą wielką y nápełnił wƺyſtkę źiemię.
Tenći jeſt ſen. Wykłád też jego powiemy przed królem :
Tyś / królu! królem królów ; bo tobie Bóg niebieſki króleſtwo / moc / potęgę y ſłáwę dał ;
Y wƺyſtko / gdźie jedno mieƺkáją ſynowie ludzcy / zwierz polny y ptáſtwo niebieſkie / dał w rękę twoję y poſtánowił ćię pánem nád tym wƺyſtkim / á tyś jeſt tą głową złotą.
Ale po tobie powſtánie króleſtwo inƺe / podlejƺe niżeli twoje / á inne króleſtwo trzećie miedźiáne / które pánowáć będźie po wƺyſtkiej źiemi.
A króleſtwo cżwárte będźie mocne jáko żelázo ; bo jáko żelázo łámie y kruƺy wƺyſtko / jáko żelázo / mówię / kruƺy wƺyſtko / ták y ono połámie y pokruƺy wƺyſtko.
A iżeś widźiał nogi / y pálce cżęśćią z gliny gárncárſkiej á cżęśćią z żelázá / króleſtwo rozdźielone znácży / w którym będźie nieco mocy żelázney / ták jákoś widźiał żelázo zmięƺáne z ſkorupą gliniáną ;
Ale pálce nóg cżęśćią z żelázá á cżęśćią z gliny znácżą króleſtwo cżęśćią mocne á cżęśćią do ſkruƺeniá ſnádne.
A iżeś widźiał żelázo zmięƺáne z ſkorupą gliniáną / znácży / że śię ſpokrewnią z ſobą ludźie ; á wƺákże nie będźie śię trzymał jeden drugiego / ták jáko żelázo nie może śię zmięƺáć z gliną.
Ale zá dni tych królów wzbudźi Bóg niebieſki króleſtwo / które ná wieki zepſute nie będźie / á króleſtwo to ná inƺy naród nie ſpádnie / ále ono połámie / y koniec ucżyni tym wƺyſtkim króleſtwom / á ſámo ſtáć będźie ná wieki.
Ták jákoś widźiał / iż z góry odćięty był kámień / który nie bywał w ręku / á ſkruƺył żelázo / miedź / glinę / ſrebro y złoto / przez to Bóg wielki królowi oznájmił / co ma być ná potem ; y práwdźiwy jeſt ten ſen / y wierny wykłád jego.
Tedy król Nábuchodonozor pádł ná oblicże ſwoje / y ukłonił śię Dánijelowi / á rozkázał / áby mu ofiárę y kádzeniá ofiárowáli.
Tedy odpowiádájąc król Dánijelowi rzekł : Zápráwdę Bóg wáƺ jeſt Bogiem bogów / á Pánem królów / który odkrywá tájemnice / poniewáżeś mógł objáwić tę tájemnicę.
Zátem król zácnie wywyżƺył Dánijelá / y dárów wielkich y wiele dał mu / y ucżynił go pánem nád wƺyſtką kráiną Bábilońſką / y kśiążęćiem nád przełożonymi y nád wƺyſtkimi mędrcámi Bábilońſkimi.
Ale Dánijel prośił królá / áby przełożył nád ſpráwámi kráiny Bábilońſkiej Sádráchá / Meſáchá y Abednego ; á Dánijel bywał w bramie królewſkiej.
