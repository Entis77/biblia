Y podobáło śię Dáryjuƺowi / áby poſtánowił nád króleſtwem ſto y dwádźieśćiá ſtároſtów / którzyby byli we wƺyſtkim króleſtwie.
A nád nimi troje kśiążąt / z których był Dánijel przedniejƺym / którymby oni ſtároſtowie licżbę cżynili / áby król ƺkody nie miał.
A ſám Dánijel przewyżƺał onych kśiążąt y ſtárƺych / przeto / że duch známienitƺy był w nim / ſkąd go król myślał poſtánowić nád wƺyſtkim króleſtwem.
Tedy kśiążętá y ſtároſtowie ƺukáli / áby ználeſli przycżynę przećiwko Dánijelowi z ſtrony króleſtwá ; wƺákże żádney przycżyny áni wády ználeść nie mogli / ponieważ on był wiernym / áni żádná winá áni wádá nie znájdowáłá śię w nim.
Przetoż rzekli oni mężowie : Nie znájdźiemy przećiwko temu Dánijelowi żádney przycżyny / chybá żebyſmy co ználeſli przećiwko niemu w zakonie Bogá jego.
Tedy oni kśiążętá y ſtároſtowie zgromádźili śię do królá / y ták mu rzekli : Dáryjuƺu królu / żyj ná wieki!
Urádźili wƺyſcy kśiążętá króleſtwá / przełożeni y ſtároſtowie / urzędnicy y hetmáni / áby poſtánowiony był dekret królewſki / y ſtwierdzony wyrok / áby káżdy / któryby do trzydźieſtu dni o cokolwiek prośił którego bogá álbo cżłowieká oprócż ćiebie / królu! był wrzucony do dołu lwiego.
A ták teraz / o królu! potwierdź ten wyrok / á podáj go ná piśmie / żeby śię nie odmienił według práwá Medſkiego y Perſkiego / które śię nie odmieniá.
Skąd król Dáryjuƺ podał ná piśmie ten wyrok.
Cżego gdy śię Dánijel dowiedźiał / że był podány ná piśmie / wƺedł do domu ſwego / gdźie otworzone były okná w pokoju jego przećiw Jeruzálemowi / á trzy kroć przez dźień klękał ná koláná ſwoje / y modlił śię / á chwałę dáwał Bogu ſwemu / jáko to był zwykł przed tym cżynić.
Tedy oni mężowie zgromádźiwƺy śię / á ználázƺy Dánijelá modlącego śię y prośby wylewájącego do Bogá ſwego /
Przyſtąpili y mówili królowi o wyroku królewſkim : Izáliś wyroku nie wydał / áby káżdy cżłowiek / któryby do trzydźieſtu dni o cokolwiek prośił którego bogá álbo cżłowieká oprócż ćiebie / królu! był wrzucony do dołu lwiego? Odpowiedźiał król / y rzekł : Práwdźiwá to mowá według práwá Medſkiego y Perſkiego / które śię nie odmieniá.
Tedy odpowiádájąc rzekli do królá : Ten Dánijel / który jeſt z więźniów ſynów Judzkich / nie ma względu ná ćię / o królu! áni ná twój wyrok / któryś wydał ; bo trzy kroć przez dźień odpráwuje modlitwy ſwoje.
Te ſłowá gdy król uſłyƺał / bárzo śię záſmućił nád tym ; y ſkłonił król do Dánijelá ſerce ſwoje / áby go wyſwobodźił ; áż do záchodu ſłońcá ſtárał śię / áby go wyrwał.
Ale mężowie oni zgromádźili śię do królá / y rzekli królowi : Wiedz / królu! iż to jeſt práwo u Medów y u Perſów / áby żáden wyrok y dekret / któryby król poſtánowił / nie był odmieniony.
Tedy król rozkázał / áby przywiedźiono Dánijelá / y wrzucono go do dołu lwiego ; á król mówiąc rzekł do Dánijelá : Bóg twój / któremu ty uſtáwicżnie ſłużyƺ / ten ćię wybáwi.
Tedy przynieśiono kámień jeden / y położono go ná dźiurze onego dołu / y zápiecżętował go król ſygnetem ſwoim / y ſygnetámi kśiążąt ſwojich / áby nie był odmieniony dekret wydány przećiwko Dánijelowi.
Potem odƺedł król ná páłác ſwój / y przenocował nic nie jádƺy / y nic nie przypuśćił przed śię / coby go uweſelić mogło / ták / że y ſen jego odſtąpił od niego.
Tedy król wſtáwƺy bárzo ráno ná świtániu z kwápieniem poƺedł do dołu lwiego ;
A gdy śię przybliżył do dołu / záwołał ná Dánijelá głoſem żáłośnym / á mówiąc król rzekł do Dánijelá : Dánijelu / ſługo Bogá żywego! Bóg twój / któremu ty uſtáwicżnie ſłużyƺ / mógłże ćię wybáwić ode lwów?
Tedy Dánijel do królá rzekł : Królu / żyj ná wieki!
Bóg mój poſłał Aniołá ſwego / który zámknął páƺcżękę lwom / áby mi nie záƺkodźiły / dla tego / że śię przed nim ználázłá niewinność we mnie ; owƺem / áni przed tobą / królu! nicem złego nie ucżynił.
Tedy śię król wielce ućieƺył z tego / y rozkázał Dánijelá wyćiągnąć z dołu ; y wyćiągniono Dánijelá z dołu / á żádnego obráżeniá nie ználeźiono ná nim ; bo wierzył w Bogá ſwego.
Y rozkázał król / áby przywiedźiono onych mężów / którzy byli oſkárżyli Dánijelá / y wrzucono ich do dołu lwiego / onych ſámych / y ſynów ich / y żony ich ; á pierwej niż dopádli do dná onego dołu / pochwyćiły ich lwy / y wƺyſtkie kośći ich pokruƺyły.
Tedy król Dáryjuƺ nápiſał do wƺyſtkich ludźi / narodów / y języków / którzy mieƺkáli po wƺyſtkiej źiemi : Pokój śię wam niech rozmnoży!
Wydány jeſt odemnie ten wyrok / áby po wƺyſtkim páńſtwie króleſtwá mego wƺyſcy drżeli á báli śię oblicża Bogá Dánijelowego ; bo on jeſt Bóg żyjący y trwájący ná wieki / á króleſtwo jego áni włádzá jego nie będźie ſkáżoná áż do końcá ;
On wyrywá y wybáwiá / á cżyni znáki y cudá ná niebie y ná źiemi / który wyrwał Dánijelá z mocy lwów.
A Dánijelowi śię ƺcżęśliwie powodźiło w króleſtwie Dáryjuƺá / y w króleſtwie Cyruſá / Perſy.
