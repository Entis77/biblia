Y podobáło śię Dáryjuƺowi / áby poſtánowił nád króleſtwem ſto y dwádzieśćiá ſtároſtów / którzyby byli we wƺyſtkiem króleſtwie.
A nád nimi troje kśiążąt / z których był Dánijel przedniejƺym / którymby oni ſtároſtowie licżbę cżynili / áby król ƺkody nie miáł.
A ſám Dánijel przewyżƺáł onych kśiążąt y ſtárƺych / przeto / że duch známienitƺy był w nim / ſkąd go król myśláł poſtánowić nád wƺyſtkiem króleſtwem.
Tedy kśiążętá y ſtároſtowie ƺukáli / áby ználeźli przycżynę przećiwko Dánijelowi z ſtrony króleſtwá ; wƺákże żádnej przycżyny áni wády ználeść nie mogli / poniewáż on był wiernym / áni żádná winá áni wádá nie znájdowáłá śię w nim.
Przetoż rzekli oni mężowie : Nie znájdziemy przećiwko temu Dánijelowi żádnej przycżyny / chybá żebyśmy co ználeźli przećiwko niemu w zákonie Bogá jego.
Tedy oni kśiążętá y ſtároſtowie zgromádzili śię do królá / y ták mu rzekli : Dáryjuƺu królu / żyj ná wieki!
Urádzili wƺyſcy kśiążętá króleſtwá / przełożeni y ſtároſtowie / urzędnicy y hetmáni / áby poſtánowiony był dekret królewſki / y ſtwierdzony wyrok / áby káżdy / któryby do trzydzieſtu dni o cokolwiek prośił którego bogá álbo cżłowieká oprócż ćiebie / królu! był wrzucony do dołu lwiego.
A ták teráz / o królu! potwierdź ten wyrok / á podáj go ná piśmie / żeby śię nie odmienił według práwá Medſkiego y Perſkiego / które śię nie odmieniá.
Skąd król Dáryjuƺ podáł ná piśmie ten wyrok.
Cżego gdy śię Dánijel dowiedziáł / że był podány ná piśmie / wƺedł do domu ſwego / gdzie otworzone były okná w pokoju jego przećiw Jeruzálemowi / á trzy kroć przez dzień klękáł ná koláná ſwoje / y modlił śię / á chwáłę dáwáł Bogu ſwemu / jáko to był zwykł przedtem cżynić.
Tedy oni mężowie zgromádziwƺy śię / á ználázłƺy Dánijelá modlącego śię y prośby wylewájącego do Bogá ſwego /
Przyſtąpili y mówili królowi o wyroku królewſkim : Izáliś wyroku nie wydáł / áby káżdy cżłowiek / któryby do trzydzieſtu dni o cokolwiek prośił którego bogá álbo cżłowieká oprócż ćiebie / królu! był wrzucony do dołu lwiego? Odpowiedziáł król / y rzekł : Práwdziwá to mowá według práwá Medſkiego y Perſkiego / które śię nie odmieniá.
Tedy odpowiádájąc rzekli do królá : Ten Dánijel / który jeſt z więźniów ſynów Judzkich / nie má względu ná ćię / o królu! áni ná twój wyrok / któryś wydáł ; bo trzy kroć przez dzień odpráwuje modlitwy ſwoje.
Te ſłowá gdy król uſłyƺáł / bárdzo śię záſmućił nád tem ; y ſkłonił król do Dánijelá ſerce ſwoje / áby go wyſwobodził ; áż do záchodu ſłońcá ſtáráł śię / áby go wyrwáł.
Ale mężowie oni zgromádzili śię do królá / y rzekli królowi : Wiedz / królu! iż to jeſt práwo u Medów y u Perſów / áby żáden wyrok y dekret / któryby król poſtánowił / nie był odmieniony.
Tedy król rozkázáł / áby przywiedziono Dánijelá / y wrzucono go do dołu lwiego ; á król mówiąc rzekł do Dánijelá : Bóg twój / któremu ty uſtáwicżnie ſłużyƺ / ten ćię wybáwi.
Tedy przynieśiono kámień jeden / y położono go ná dziurze onego dołu / y zápiecżętowáł go król ſygnetem ſwoim / y ſygnetámi kśiążąt ſwoich / áby nie był odmieniony dekret wydány przećiwko Dánijelowi.
Potem odƺedł król ná páłác ſwój / y przenocowáł nic nie jádłƺy / y nic nie przypuśćił przed śię / coby go uweſelić mogło / ták / że y ſen jego odſtąpił od niego.
Tedy król wſtáwƺy bárdzo ráno ná świtániu z kwápieniem poƺedł do dołu lwiego ;
A gdy śię przybliżył do dołu / záwołáł ná Dánijelá głoſem żáłośnym / á mówiąc król rzekł do Dánijelá : Dánijelu / ſługo Bogá żywego! Bóg twój / któremu ty uſtáwicżnie ſłużyƺ / mógłże ćię wybáwić ode lwów?
Tedy Dánijel do królá rzekł : Królu / żyj ná wieki!
Bóg mój poſłáł Aniołá ſwego / który zámknął páƺcżękę lwom / áby mi nie záƺkodziły / dlátego / że śię przed nim ználázłá niewinność we mnie ; owƺem / áni przed tobą / królu! nicem złego nie ucżynił.
Tedy śię król wielce ućieƺył z tego / y rozkázáł Dánijelá wyćiągnąć z dołu ; y wyćiągniono Dánijelá z dołu / á żádnego obráżeniá nie ználeziono ná nim ; bo wierzył w Bogá ſwego.
Y rozkázáł król / áby przywiedziono onych mężów / którzy byli oſkárżyli Dánijelá / y wrzucono ich do dołu lwiego / onych ſámych / y ſynów ich / y żony ich ; á pierwej niż dopádli do dná onego dołu / pochwyćiły ich lwy / y wƺyſtkie kośći ich pokruƺyły.
Tedy król Dáryjuƺ nápiſáł do wƺyſtkich ludzi / národów / y języków / którzy mieƺkáli po wƺyſtkiej ziemi : Pokój śię wám niech rozmnoży!
Wydány jeſt odemnie ten wyrok / áby po wƺyſtkiem páńſtwie króleſtwá mego wƺyſcy drżeli á báli śię oblicżá Bogá Dánijelowego ; bo on jeſt Bóg żyjący y trwájący ná wieki / á króleſtwo jego áni włádzá jego nie będzie ſkáżoná áż do końcá ;
On wyrywá y wybáwiá / á cżyni znáki y cudá ná niebie y ná ziemi / który wyrwáł Dánijelá z mocy lwów.
A Dánijelowi śię ƺcżęśliwie powodziło w króleſtwie Dáryjuƺá / y w króleſtwie Cyruſá / Perſy.
