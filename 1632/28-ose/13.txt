Gdy máwiał Efráim / ſtrách bywał / bo był wywyżƺony w Izráelu ; ále gdy zgrzeƺył przy Báálu / tedy umárł.
A teraz jeƺcże przycżyniáją grzechu ; bo ſobie cżynią y leją z ſrebrá ſwego według przemyſłu ſwego ſtráƺne báłwány / co wƺyſtko tylko jeſt robotą rzemieślniká / o których jednák ſámi mówią : Ludźie / którzy chcą ofiárowáć / niech cáłują ćielce ;
( Przetoż śię ſtáną jáko obłok poránny / á jáko roſá poránná przechodzącá / y jáko plewy od wichru porwáne z bojewiſká / y jáko dym z kominá. )
Gdyżem Ja jeſt Pán / Bóg twój / od wyśćia z źiemi Egipſkiej ; á Bogá oprócż mnie nie poznáłeś / y niemáƺ zbáwićielá oprócż mnie.
Jam ćię poznał ná puƺcży w źiemi bárzo ſuchej.
Dobremi páſtwiſkámi ſwemi náſyceni ſą ; ále gdy śię náſyćili / podnioſło śię ſerce ich ; przetoż mię zápomnieli.
Dla tego będę im jáko lew ſrogi / jáko lámpárt przy drodze będę cżyhał.
Zábieżę im jáko niedźwiedź ośieroćiáły / á roztárgám záwárćie ſercá ich / y pożrę ich tám jáko lew / jáko zwierz dźiki ƺárpájąc ich.
Zginienie twoje z ćiebie / o Izráelu! ále ze mnie wſpomożenie twoje.
Gdźież jeſt król twój? gdźież jeſt? Niech ćię záchowá we wƺyſtkich miáſtách twojich! Y ſędźiowie twoi / o którycheś mówił : Dáj mi królá y kśiążąt.
Dáłem ći tedy królá w zápálcżywośći mojey / álem go odjął w zágniewániu mojim.
Związáná jeſt niepráwość Efráimowá / ſchowány jeſt grzech jego.
Boleśći rodzącey ogárną go ; on ſynem niemądrym / bo inácżey nie zoſtáwáłby ták długo w żywoćie mátki.
Z ręki grobu wybáwię ich / od śmierći wykupię ich. O śmierći! będę śmierćią twoją ; o grobie! będę ſkáżeniem twojim ; żáłość ſkrytá będźie od ocżów mojich.
Bo on miedzy bráćmi owoc przynieśie ; pierwej jednák przydźie wiátr ze wſchodu / wiátr Páńſki od puƺcży wyſtępujący / y wyſuƺy źródło jego / wyſuƺy y zdrój jego ; onći rozchwyći ſkárby wƺelkiego nacżynia pożądánego.
Sámáryjá będźie ſpuſtoƺoná / przeto / że śię ſprzećiwiłá Bogu ſwemu ; od miecżá upádną / málucżcy jey roztrąceni będą / á brzemienne jey rozćięte będą.
