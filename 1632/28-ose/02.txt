Mówćie bráćiom wáƺym / o ludu mój! y śioſtrom wáƺym / o ty / coś miłośierdźiá doſtąpiłá!
Rozpierájćie śię z mátką wáƺą / rozpierájćie śię ; bo oná nie jeſt żoną moją / á Jám też nie jeſt mężem jej / póki nie odejmie wƺetecżeńſtw ſwoich od oblicżá ſwego / á cudzołóſtw ſwych z pośród pierśi ſwoich ;
Bym jej ſnáć nágo nie zewlekł á nie poſtáwił jej / jáką byłá w dźień národzeniá ſwego / y nie ucżynił jej jáko puſtyniá / y nie zoſtáwił jej jáko źiemiá ſuchá / y nie umorzył jej prágnieniem :
Y nád ſynámi jej nie zmiłowáłbym śię / przeto / że ſą ſynámi z wƺetecżeńſtwá /
Bo wƺetecżeńſtwo płodźi mátká ich / ſproſność cżyni rodźićielká ich ; bo mówiłá : Pójdę zá miłośnikámi moimi / którzy mi dodáwáją chlebá mego / y wody mojej / wełny mojej / y lnu mego / oliwy mojej y nápojów moich.
Przetoż oto Já zágrodzę ćierniem drogę jej / á ugrodzę płot / áby śćieżek ſwoich nie ználázłá.
Tedy biegáć będźie zá miłośnikámi ſwymi / wƺákże ich nie dogoni ; y ƺukáć ich będźie / ále nie znájdźie. Tedy rzecże : Pójdę á wrócę śię do mężá ſwego pierwƺego ; bo mi lepiej było ná on cżáſ / niżeli teráz.
Bo oná nie wie tego / żem Já dáwáł jej zboże / y moƺcż / y oliwę ; náwet dáwáłem jej obfitość ſrebrá y złotá / które oni obrácáją ná Báálá.
Dlátego wrócę śię / y zábiorę zboże moje cżáſu jego / y moƺcż mój cżáſu jego / y odbiorę jej wełnę moję / y len mój dány ná okrywánie nágośći jej.
A teráz odkryję ſproſność jej przed ocżymá miłośników jej / á nikt jej nie wyrwie z ręki mojej.
Y ucżynię koniec wƺyſtkiemu weſelu jej / świętom jej / nowym mieśiącom jej / y ſábátom jej / y wƺyſtkim urocżyſtym świętom jej.
Popuſtoƺę też winne máćice jej y figowe drzewá jej / przeto / że mówi : Mojá to zápłátá / którą mi dáli miłośnicy moi ; y obrócę je w láſy / á pożrą je zwierzętá polne.
Y náwiedzę ná niej dni Báálowe / w które im kádźi / á ſtrojąc śię w náuƺnice ſwoje y w klejnoty ſwoje / chodźi zá miłośnikámi ſwymi / ále mnie zápominá / mówi Pán.
Przetoż oto Já nią łudźić będę / gdy ją wywiodę ná puƺcżę / á łáſkáwie z nią mówić będę ;
Y dám jej winnice jej od tegoż miejſcá / y dolinę Achor / miáſto drzwi nádźiei ; y będźie tám śpiewáłá / jáko zá dni młodośći ſwojej / to jeſt / jáko w dźień / którego wychodźiłá z źiemi Egipſkiej.
A dniá onego mówi Pán / będźieƺ mię zwáłá : Mężu mój! á nie będźieƺ mię więcej zwáłá : Báálu mój!
Bo odejmę imioná Báálów od uſt twoich / że áni wſpomináni będą więcej imieniem ſwojim.
Y ucżynię dlá ćiebie przymierze dniá onego z zwierzem polnym / y z ptáſtwem niebieſkiem y z płázem źiemſkim / á łuk y miecż pokruƺę / y wojnę z źiemi znioſę / y ſpráwię / że bezpiecżnie mieƺkáć będą.
Y poślubię ćię ſobie ná wieki : poślubię ćię / mówię / ſobie w ſpráwiedliwośći y w ſądźie y w miłośierdźiu y w litośćiách ;
Poślubię ćię też ſobie w wierze / y poznáƺ Páná.
Dniá onego wyſłuchám / mówi Pán / wyſłuchám / mówi / niebioſá / á one wuſłucháją źiemię ;
A źiemiá wyſłuchá zboże y moƺcż y oliwę / á te rzecży wyſłucháją Jezreelá.
Bo ją ſobie rozśieję ná źiemi / á zmiłuję śię nád tą / co byłá w niełáſce / y rzekę do tego / co nie był ludem moim : Tyś jeſt lud mój! á on rzecże : Tyś jeſt Bóg mój.
