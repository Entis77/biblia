Mówćie bráćiom wáƺym / o ludu mój! y śioſtrom wáƺym / o ty / coś miłośierdziá doſtąpiłá!
Rozpierájćie śię z mátką wáƺą / rozpierájćie śię ; bo oná nie jeſt żoną moją / á Jám też nie jeſt mężem jej / póki nie odejmie wƺetecżeńſtw ſwoich od oblicżá ſwego / á cudzołóſtw ſwych z pośród pierśi ſwoich ;
Bym jej ſnáć nágo nie zewlekł á nie poſtáwił jej / jáką byłá w dzień národzeniá ſwego / y nie ucżynił jej jáko puſtyniá / y nie zoſtáwił jej jáko ziemiá ſuchá / y nie umorzył jej prágnieniem :
Y nád ſynámi jej nie zmiłowáłbym śię / przeto / że ſą ſynámi z wƺetecżeńſtwá /
Bo wƺetecżeńſtwo płodzi mátká ich / ſproſność cżyni rodzićielká ich ; bo mówiłá : Pójdę zá miłośnikámi moimi / którzy mi dodáwáją chlebá mego / y wody mojej / wełny mojej / y lnu mego / oliwy mojej y nápojów moich.
Przetoż oto Já zágrodzę ćierniem drogę jej / á ugrodzę płot / áby śćieżek ſwoich nie ználázłá.
Tedy biegáć będzie zá miłośnikámi ſwymi / wƺákże ich nie dogoni ; y ƺukáć ich będzie / ále nie znájdzie. Tedy rzecże : Pójdę á wrócę śię do mężá ſwego pierwƺego ; bo mi lepiej było ná on cżáſ / niżeli teráz.
Bo oná nie wie tego / żem Já dáwáł jej zboże / y moƺcż / y oliwę ; náwet dáwáłem jej obfitość ſrebrá y złotá / które oni obrácáją ná Báálá.
Dlátego wrócę śię / y zábiorę zboże moje cżáſu jego / y moƺcż mój cżáſu jego / y odbiorę jej wełnę moję / y len mój dány ná okrywánie nágośći jej.
A teráz odkryję ſproſność jej przed ocżymá miłośników jej / á nikt jej nie wyrwie z ręki mojej.
Y ucżynię koniec wƺyſtkiemu weſelu jej / świętom jej / nowym mieśiącom jej / y ſábátom jej / y wƺyſtkim urocżyſtym świętom jej.
Popuſtoƺę też winne máćice jej y figowe drzewá jej / przeto / że mówi : Mojá to zápłátá / którą mi dáli miłośnicy moi ; y obrócę je w láſy / á pożrą je zwierzętá polne.
Y náwiedzę ná niej dni Báálowe / w które im kádzi / á ſtrojąc śię w náuƺnice ſwoje y w klejnoty ſwoje / chodzi zá miłośnikámi ſwymi / ále mnie zápominá / mówi Pán.
Przetoż oto Já nią łudzić będę / gdy ją wywiodę ná puƺcżę / á łáſkáwie z nią mówić będę ;
Y dám jej winnice jej od tegoż miejſcá / y dolinę Achor / miáſto drzwi nádziei ; y będzie tám śpiewáłá / jáko zá dni młodośći ſwojej / to jeſt / jáko w dzień / którego wychodziłá z ziemi Egipſkiej.
A dniá onego mówi Pán / będzieƺ mię zwáłá : Mężu mój! á nie będzieƺ mię więcej zwáłá : Báálu mój!
Bo odejmę imioná Báálów od uſt twoich / że áni wſpomináni będą więcej imieniem ſwojim.
Y ucżynię dlá ćiebie przymierze dniá onego z zwierzem polnym / y z ptáſtwem niebieſkiem y z płázem ziemſkim / á łuk y miecż pokruƺę / y wojnę z ziemi znioſę / y ſpráwię / że bezpiecżnie mieƺkáć będą.
Y poślubię ćię ſobie ná wieki : poślubię ćię / mówię / ſobie w ſpráwiedliwośći y w ſądzie y w miłośierdziu y w litośćiách ;
Poślubię ćię też ſobie w wierze / y poznáƺ Páná.
Dniá onego wyſłuchám / mówi Pán / wyſłuchám / mówi / niebioſá / á one wuſłucháją ziemię ;
A ziemiá wyſłuchá zboże y moƺcż y oliwę / á te rzecży wyſłucháją Jezreelá.
Bo ją ſobie rozśieję ná ziemi / á zmiłuję śię nád tą / co byłá w niełáſce / y rzekę do tego / co nie był ludem moim : Tyś jeſt lud mój! á on rzecże : Tyś jeſt Bóg mój.
