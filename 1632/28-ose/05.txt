Słuchájćie tego / o kápłáni! á miejćie wzgląd ná to / domie Izráelſki! y ty / domie królewſki! ſłuchájćie ; bo przećiwko wám ſąd jeſt / przeto / żeśćie śidłem w Máſfá / á śiećią rozćiągnioną ná wierzchu Tábor.
Owƺem / udáwájąc śię ná zábijánie / przypádáją do ziemi ; ále Já pokárzę káżdego z nich.
Já znám Efráimá / y Izráel nie jeſt ſkryty przedemną ; bo teráz nierząd płodziƺ / Efráimie! á Izráel ſplugáwił śię.
Nie máją śię do tego / áby śię náwróćili do Bogá ſwego / przeto / że duch wƺetecżeńſtwá jeſt w pośrodku nich / á Páná nie poználi.
Świádcżyć też będzie hárdość Izráelſká przećiwko niemu ; przetoż Izráel y Efráim upádną dlá niepráwośći ſwojej / upádnie też y Judá z nimi.
Z trzodámi ſwemi y z bydłem ſwojim pójdą ƺukáć Páná ; wƺákże go nie znájdą ; bo odſtąpił od nich.
Wyſtąpili przećiwko Pánu / bo ſynów cudzych nápłodzili ; á ták teráz pożre ich mieśiąc z májętnośćiámi ich.
Zátrąbćie w trąbę w Gábáá / y w trąbę w Rámá ; krzycżćie w Bet Awen : Nieprzyjáćiel zá tobą / o Benjáminie!
Efráim ſpuſtoƺony będzie w dzień kárániá / w pokoleniách Izráelſkich oznájmiłem / że to práwdá.
Kśiążętá Judzcy ſtáli śię jáko ći / którzy gránice przenoƺą ; wyleję ná nich popędliwość moję jáko wodę.
Ućiśniony jeſt Efráim / pokruƺony ſądem ; wƺákże przećie má wolę chodzić zá rozkázániem ludzkiem.
Przetoż y Já byłem Efráimowi jáko mól / á domowi Judzkiemu jáko ſpróchniáłość.
Skąd widząc Efráim mdłość ſwoję á Judá ránę ſwoję / ućiekł śię Efráim do Aſſurá / y poſłáł do królá Járebá ; ále on wáſ nie będzie mógł uzdrowić / áni wáſ ulecży od rány wáƺej.
Bom Já jeſt Efráimowi jáko lew ſrogi / á domowi Judzkiemu jáko lwię ; Já / Já porwę / y odejdę ; wezmę / á nikt mi nie wydrze ;
Pójdę / wrócę śię do miejſcá mego / áż śię winnymi dádzą á ƺukáć będą oblicżá mojego.
