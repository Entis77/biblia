Gdy lecżę Izráelá / tedy śię odkrywá niepráwość Efráimowá y złośći Sámáryjſkie / bo śię báwią kłámſtwem ; w miáſtách złodźiejſtwo / á ná dworze łupieſtwo przewodzą ;
A nie myślą w ſercu ſwojim / że ná wƺyſtkie złośći ich pámiętám ; á teraz ich ogárniáją ſpráwy ich / y ſą przed oblicżem mojim.
Królá uweſeláją złośćią ſwoją á kśiążąt kłámſtwy ſwemi.
Wƺyſcy zgołá cudzołożą / ſą jáko piec rozpálony od piekárzá / który przeſtáje cżuć / gdy zácżynił ćiáſto / áżby ukiśiáło.
W dźień królá náƺego w chorobę go wpráwiáją kśiążętá łágwiámi winá / ták że y on rękę ſwą z náśmiewcámi wyćiągá.
Bo przyłożyli ſerce ſwe do záſádzek / jáko piec rozpálony ; cáłą noc śpi piekárz ich / á z poránku gore jáko płomień ogniá.
Wƺyſcy zgołá rozpálili śię jáko piec / á pożeráją ſędźiów ſwojich ; wƺyſcy królowie ich upádáją / niemáƺ miedzy nimi / ktoby wołał do mnie.
Efráim ten śię z národámi zmięƺał ; Efráim będźie jáko podpłomyk nieprzewrácány.
Pożárli obcy śiłę jego / á on o tym nie wie ; ſędźiwość też nań wyſtępuje / wƺákże on tego nie wie.
A choć pychá Izráelſká świádcży w ocży przećiwko niemu / wƺákże śię nie náwrácáją do Páná / Bogá ſwego / áni go w tym wƺyſtkiem ƺukáją.
Y ſtał śię Efráim jáko gołębicá głupiá / nie májącá ſercá ; Egiptu przyzywáją á do Aſſurá śię ućiekáją.
Ale gdy odejdą / rozćiągnę ná nich śieć moję / á jáko ptáſtwo niebieſkie potárgnę ich ; ſkárzę ich / jáko im o tym powiádáno w zgromádzeniu ich.
Biádá im / że śię rozbiegli odemnie! ſpuſtoƺenie przydźie ná nich / bo wyſtąpili przećiwko mnie ; choćem Ja ich odkupił / ále oni przećiwko mnie mówili kłámſtwá ;
Y nie wołáją do mnie z ſercá ſwego / gdy wyją ná łożách ſwojich ; wpráwdźieć dla zbożá y moƺcżu zgromádzáją śię / ále mię potem odſtępują.
Choćem utwierdzał rámioná ich pokáráwƺy ich / ále oni złe myślą przećiwko mnie.
Náwrácáją śię / ále nie do Nájwyżƺego ; ſą jáko łuk zdrádliwy / polegną od miecżá / kśiążętá ich od popędliwośći języká ich / co im jeſt ku pośmiewiſku w źiemi Egipſkiej.
