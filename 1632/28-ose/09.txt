Nie weſel śię / Izráelu! nie ráduj śię jáko inne národy / że nierząd płodźiƺ / odwrácájąc śię od Bogá ſwego / á miłujeƺ zápłátę wƺetecżnicy ná wƺyſtkich bojewiſkách zbożá.
Bojewiſko áni práſá nie będźie ich żywiłá / á moƺcż omyli ich.
Nie będą mieƺkáć w źiemi Páńſkiej ; ále śię Efráim wróći do Egiptu / á w Aſſyryi niecżyſte rzecży jeść będą.
Nie będą Pánu winá ofiárowáli / áni mu będą przyjemne. Ofiáry ich będą im jáko chleb płácżących ; którzybykolwiek z niego jedli / zmázáliby śię / przeto / że chleb ich zá umárłych ich nie wnijdźie do domu Páńſkiego.
Cóż będźiećie cżynić w dźień urocżyſty y w dźień świętá Páńſkiego?
Bo oto záginą przez ſpuſtoƺenie / Egipt zgromádźi ich á Memfiſ pogrzebie ich ; ſrebro ich pożądáne pokrzywá odźiedźicży / á ćiernie poroſną w przybytkách ich.
Przyjdą dni náwiedzeniá / przyjdą dni zápłáty ; pozná to Izráel / że ten prorok ich jeſt głupi / ƺálony / mąż nikcżemny dlá mnóſtwá niepráwośći twojej y dlá wielkiej nienáwiśći twojej / o Izráelu!
Prorok / który ſtráż trzymá nád Efráimem poſpołu z Bogiem moim / ſtáł śię śidłem ptáƺniká ná wƺyſtkich drogách jego / nienáwiść jeſt w domu bogá jego.
Głęboko zábrnęli / popſuli śię jáko zá dni Gábáá ; wſpomnić Pán ná niepráwośći ich / y náwiedźi grzechy ich.
Ználázłem Izráelá jáko jágody winne ná puƺcży ; widźiáłem ojców wáƺych jáko pierwƺy owoc figowy ná pocżątku ich ; ále oni poƺli zá Báál Fegorem / á oddáli śię tej obrzydliwośći ; przetoż będą obrzydłymi / ták jáko śię im upodobáło.
Efráim uleći jáko pták / á ſłáwá ich záráz od porodzeniá / y od żywotá / y od pocżęćiá.
A choćby odchowáli ſynów ſwych / przećie ich pozbáwię wieku męſkiego ; owƺem / y onym biádá / gdy Já ich odſtąpię.
Efráim / jáko widzę / jeſt jáko Tyr wƺcżepiony ná miejſcu rozkoƺnem ; wƺákże Efráim wywiedźie do mordercy ſynów ſwoich.
Dáj im Pánie! Cóż im dáƺ? Dáj im żywot niepłodny á pierśi wyſchłe.
Wƺyſtká złość ich jeſt w Gálgál / przetoż y tám ich nienáwidzę ; dlá złośći ucżynków ich z domu mego wyrzucę ich / nie będę ich więcej miłowáł / wƺyſcy kśiążętá ich odporni ſą.
Efráim poráżony będźie / korzeń ich uſchnie / owocu nie przynioſą ; á choćby też ſpłodźili / tedy wybiję nájmilƺe dźiátki żywotá ich.
Odrzući ich Bóg mój / bo go ſłucháć nie chcą ; á między pogánámi tułácżámi będą.
