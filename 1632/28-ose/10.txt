Izráel jeſt máćicą próżną / owoc przynośi ſobie ; cżem więcey miewáł owocu ſwego / tem więcey nábudowáł ołtárzów ; á cżem obfitƺá jeſt źiemiá jego / tem więcey náſtáwiáł obrázów.
Rozdźielá śię ſerce ich / przetoż winnymi ſą ; on pokruƺy ołtárze ich / y obrázy ich popſuje.
Poniewáż mówią : Nie mámy królá / náwet nie boimy śię Páná / á król nám co ucżyni?
Mówią ſłowá / przyśięgájąc kłámliwie / gdy cżynią przymierze ; y wzrośnie ſąd jáko jád ná zágonách polnych.
Dla jáłowic Bet Aweńſkich zátrwożą śię obywátele Sámáryjſcy / gdy kwilić będą nád nim / lud jego / y popi jego / ( którzy śię więc rádowáli z niego ) przeto / że ſłáwá jego odchodźi od niego.
Dla jáłowic Bet Aweńſkich zátrwożą śię obywátele Sámáryjſcy / gdy kwilić będą nád nim lud jego / y Popi jego / ( którzy śię więc rádowáli z niego ) przeto / że ſławá jego odchodźi od niego.
Náwet y ſám do Aſſyryi záprowádzony będźie zá dár królowi Járeb ; záwſtydźi śię Efráim / y záſromá śię Izráel zá rádę ſwoję.
Wyćięty będźie król Sámáryjſki / jáko piáná ná wierzchu wody.
Y będą wyżyny Awenu wytrácone / grzech Izráelſki ; ćiernie y oſet wyrośnie ná ołtárzách ich ; á rzekną górom : Przykryjćie nas ; á págórkom : Upádnijćie ná nas.
Ode dni Gábáá grzeƺyłeś / Izráelu! tám śię oſtáli / nie záchwyćiłá ich w Gábáá bitwá przećiwko ſynom niepráwośći podnieśioná.
Przetoż ich pokárzę według upodobániá mego ; bo śię zbiorą ná nich národy / áby byli káráni dla dwojákich niepráwośći ſwoich.
Bo Efráim jeſt jáko jáłowicá wyucżoná / kochá śię w młóceniu / choćiáżem Ja náſtępowáł ná cudny kárk jey / ábym do jázdy używáł Efráimá / Judá áby oráł / á Jákób áby włócżył.
Y mówiłem : Siejćie ſobie ku ſpráwiedliwośći ; żnijćie ku miłośierdźiu / orzćie ſobie nowinę ; bo cżáſ jeſt ƺukáć Páná / áż przyjdźie / y ſpuśći ná was ſpráwiedliwość jáko deƺcż.
Aleśćie oráli niepobożność / żęliśćie niepráwość / jedliśćie owoc kłámſtwá ; ábowiem ufáƺ w drodze ſwej y w mnóſtwie mocárzów twoich.
Dlátegoż powſtánie rozruch między ludem twoim / á wƺyſtkie zámki twoje poburzone będą / ták jáko poburzył Sálmán Bet Arbel w dźień bitwy ; mátki z ſynámi roztrącone będą.
Oto ták wám ucżyni Betel dla wielkiej złośći wáƺej ; król Izráelſki ná świtániu do ƺcżętu zgłádzony będźie.
