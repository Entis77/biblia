Abowiem wy ſámi wiećie bráćia / że przyśćie náƺe do was nie było prózne.
Ale choćiaſmy y przed tym ćierpieli / y zelżeni byli ( jáko wiećie ) <i>w mieśćie</i> Filippiech ; <i>wƺákże</i> beſpiecżnieſmy ſobie pocżęli w Bogu náƺym / w opowiádániu u was Ewángeliey Bożey / z wielkim bojem.
Abowiem nápominánie náƺe nie <i>było</i> z oƺukánia / áni z niecżyſtośći / áni z zdrády :
Ale jáko nas Bóg <i>ſobie</i> upodobał ; żeby nam byłá zwierzona Ewángelia / ták mówimy ; nie jáko ludźiom śię podobájąc / ále Bogu / który <i>ſobie</i> upodobywa ſercá náƺe.
Abowiem nigdyſmy nie używáli mowy pochlebney / jáko wiećie ; ániſmy pod zaſłoną cżego łákomſtwu dogadzáli : Bóg jeſt świádkiem.
Aniſmy ƺukáli od ludźi chwały / áni od was / áni od inƺych : mogąc <i>wam</i> bydź ćiężkimi / jáko Apoſtołowie CHryſtuſowi.
Aleſmy byli ſkromnymi między wámi / jáko gdy mámká odchowywa dźiatki ſwoje.
Ták wam życżliwymi będąc / gotowiſmy byli użycżyć wam nietylko Ewángeliey Bożey / ále też y duƺ náƺych ; dla tego iżeśćie nam miłymi byli.
Abowiem pámiętaćie bráćia pracę náƺę y trudy ; gdyż w nocy y we dnie robiąc / ábyſmy żadnego z was nie obćiążyli / kazáliſmy u was Ewángelią Bożą.
Wyśćie świádkámi y Bóg / żeſmy świętobliwie y ſpráwiedliwie y bez nágány żyli <i>między</i> wámi / którzyśćie uwierzyli.
Jákoż wiećie / żeſmy káżdego z was / jáko Oćiec dźiatki ſwoje nápomináli y ćieƺyli :
Y oświadcżáli śię / ábyśćie chodźili godnie Bogu / który was powołał do ſwego króleſtwá y do chwały.
Przetoż y my dźiękujemy Bogu bez przeſtanku / iż przyjąwƺy ſłowo Boże / któreśćie ſłyƺeli od nas / przyjęliśćie nie jáko ſłowo ludzkie / ále ( jáko w prawdźie jeſt ) <i>jáko</i> ſłowo Boże ; które też ſkutecżne jeſt w was / którzy wierzyćie.
Abowiem wy bráćia ſtáliśćie śię náśládowcámi Zborów Bożych które ſą w Żydowſkiey źiemi w CHryſtuśie JEzuśie : bośćie y wy toż ćierpieli od źiemków ſwojich / jáko y oni od Żydów :
Którzy y PAná JEzuſá zábili / y ſwoje właſne Proroki / y nas wygnáli / á Bogu śię nie podobáją / y wƺyſtkim ludźiom ſą przećiwni.
Zábraniájąc nam / żebyſmy nie mówili Pogánom żeby byli zbáwieni : áby ták záwƺe dopełniáli grzechów ſwojich. Abowiem przyƺedł ná nie gniew áż do końcá.
Lecż my bráćia będąc pozbáwieni was ná chwilecżkę / oblicżem nie ſercem / tym pilniſmy śię ſtáráli oglądáć oblicże wáƺe z wielką żądośćią.
Dla tegoſmy chćieli przyść do was / ( zwłaƺcżá ja Páweł ) y raz y drugi ; ále nam przekáźił ƺátan.
Abowiem któraż <i>jeſt</i> nádźiejá náƺá álbo rádość / álbo koroná chluby? Izali nie wy przed oblicżnośćią PAná náƺego JEzuſá CHryſtuſá / w przyśćie jego?
Wy zájiſte jeſteśćie chwałą náƺą y rádośćią.
