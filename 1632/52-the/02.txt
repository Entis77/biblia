Abowiem wy ſámi wiećie / bráćia! że przyjśćie náƺe do was nie było próżne ;
Ale choćiáżeśmy y przedtem ćierpieli y zelżeni byli ( jáko wiećie ) w mieśćie Filipách / wƺákże bezpiecżnieśmy ſobie pocżęli w Bogu náƺym / w opowiádániu u was Ewángielii Bożej z wielkim bojem /
Abowiem nápominánie náƺe nie było z oƺukániá / áni z niecżyſtośći / áni z zdrády ;
Ale jáko nas Bóg ſobie upodobáł / żeby nám byłá zwierzoná Ewángielijá / ták mówimy / nie jáko ludźiom śię podobájąc / ále Bogu / który ſobie upodobywá ſercá náƺe.
Abowiem nigdyśmy nie używáli mowy pochlebney / jáko wiećie ; ániśmy pod záſłoną cżego łákomſtwu dogádzáli ; Bóg jeſt świádkiem ;
Aniśmy ƺukáli od ludźi chwáły / áni od was / áni od inƺych / mogąc wám być ćiężkimi / jáko Apoſtołowie CHryſtuſowi ;
Aleśmy byli ſkromnymi między wámi / jáko gdy mámká odchowuje dźiátki ſwoje.
Ták wám życżliwymi będąc / gotowiśmy byli użycżyć wám nie tylko Ewángielii Bożej / ále też y duƺ náƺych / dlátego / iżeśćie nám miłymi byli.
Abowiem pámiętájćie / bráćia! prácę náƺę y trudy / gdyż w nocy y we dnie robiąc / ábyśmy żádnego z was nie obćiążyli / kázáliśmy u was Ewángielię Bożą.
Wyśćie świádkámi y Bóg / żeśmy świętobliwie y ſpráwiedliwie / y bez nágány żyli między wámi / którzyśćie uwierzyli.
Jákoż wiećie / żeśmy káżdego z was / jáko ojćiec dźiátki ſwoje / nápomináli y ćieƺyli /
Y oświádcżáli śię / ábyśćie chodźili godnie Bogu / który was powołáł do ſwego króleſtwá y do chwáły.
Przetoż y my dźiękujemy Bogu bez przeſtánku / iż przyjąwƺy ſłowo Boże / któreśćie ſłyƺeli od nas / przyjęliśćie / nie jáko ſłowo ludzkie / ále ( jáko w práwdźie jeſt / ) jáko ſłowo Boże / które też ſkutecżne jeſt w was / którzy wierzyćie.
Abowiem wy / bráćia! ſtáliśćie śię náśládowcámi zborów Bożych / które ſą w Żydowſkiej źiemi / w CHryſtuśie JEzuśie ; bośćie y wy też ćierpieli od źiomków wáƺych / jáko y oni od Żydów.
Którzy y Páná JEzuſá zábili / y ſwoich włáſnych proroków / y nas wygnáli / á Bogu śię nie podobáją / y wƺyſtkim ludźiom ſą przećiwni /
Zábrániájąc nám / żebyśmy nie mówili pogánom / żeby byli zbáwieni / áby ták záwƺe dopełniáli grzechów ſwoich ; ábowiem przyƺedł ná nich gniew áż do końcá.
Lecż my / bráćia! będąc pozbáwieni was ná chwilecżkę oblicżem / nie ſercem / tem pilniejśmy śię ſtáráli oglądáć oblicże wáƺe z wielką żądośćią.
Dlátegośmy chćieli przyjść do was ( zwłáƺcżá ja Páweł / ) y ráz y drugi ; ále nám przekáźił ƺátán.
Abowiem któráż jeſt nádźiejá náƺá / álbo rádość / álbo koroná chluby? Izáli nie wy przed oblicżnośćią Páná náƺego JEzuſá CHryſtuſá w przyjśćie jego?
Wy záiſte jeſteśćie chwáłą náƺą y rádośćią.
