Abowiem wy ſámi wiećie / bráćiá! że przyjśćie náƺe do wáſ nie było próżne ;
Ale choćiáżeśmy y przedtem ćierpieli y zelżeni byli ( jáko wiećie ) w mieśćie Filipách / wƺákże bezpiecżnieśmy ſobie pocżęli w Bogu náƺym / w opowiádániu u wáſ Ewángielii Bożej z wielkim bojem /
Abowiem nápominánie náƺe nie było z oƺukániá / áni z niecżyſtośći / áni z zdrády ;
Ale jáko náſ Bóg ſobie upodobáł / żeby nám byłá zwierzoná Ewángielijá / ták mówimy / nie jáko ludziom śię podobájąc / ále Bogu / który ſobie upodobywá ſercá náƺe.
Abowiem nigdyśmy nie używáli mowy pochlebnej / jáko wiećie ; ániśmy pod záſłoną cżego łákomſtwu dogádzáli ; Bóg jeſt świádkiem ;
Aniśmy ƺukáli od ludzi chwáły / áni od wáſ / áni od inƺych / mogąc wám być ćiężkimi / jáko Apoſtołowie Chryſtuſowi ;
Aleśmy byli ſkromnymi między wámi / jáko gdy mámká odchowuje dziátki ſwoje.
Ták wám życżliwymi będąc / gotowiśmy byli użycżyć wám nie tylko Ewángielii Bożej / ále też y duƺ náƺych / dlátego / iżeśćie nám miłymi byli.
Abowiem pámiętájćie / bráćiá! prácę náƺę y trudy / gdyż w nocy y we dnie robiąc / ábyśmy żádnego z wáſ nie obćiążyli / kázáliśmy u wáſ Ewángielię Bożą.
Wyśćie świádkámi y Bóg / żeśmy świętobliwie y ſpráwiedliwie / y bez nágány żyli między wámi / którzyśćie uwierzyli.
Jákoż wiećie / żeśmy káżdego z wáſ / jáko ojćiec dziátki ſwoje / nápomináli y ćieƺyli /
Y oświádcżáli śię / ábyśćie chodzili godnie Bogu / który wáſ powołáł do ſwego króleſtwá y do chwáły.
Przetoż y my dziękujemy Bogu bez przeſtánku / iż przyjąwƺy ſłowo Boże / któreśćie ſłyƺeli od náſ / przyjęliśćie / nie jáko ſłowo ludzkie / ále ( jáko w práwdzie jeſt / ) jáko ſłowo Boże / które też ſkutecżne jeſt w wáſ / którzy wierzyćie.
Abowiem wy / bráćiá! ſtáliśćie śię náśládowcámi zborów Bożych / które ſą w Żydowſkiej ziemi / w Chryſtuśie Jezuśie ; bośćie y wy też ćierpieli od ziomków wáƺych / jáko y oni od Żydów.
Którzy y Páná Jezuſá zábili / y ſwoich włáſnych proroków / y náſ wygnáli / á Bogu śię nie podobáją / y wƺyſtkim ludziom ſą przećiwni /
Zábrániájąc nám / żebyśmy nie mówili pogánom / żeby byli zbáwieni / áby ták záwƺe dopełniáli grzechów ſwoich ; ábowiem przyƺedł ná nich gniew áż do końcá.
Lecż my / bráćiá! będąc pozbáwieni wáſ ná chwilecżkę oblicżem / nie ſercem / tem pilniejśmy śię ſtáráli oglądáć oblicże wáƺe z wielką żądośćią.
Dlátegośmy chćieli przyjść do wáſ ( zwłáƺcżá já Páweł / ) y ráz y drugi ; ále nám przekáził ƺátán.
Abowiem któráż jeſt nádziejá náƺá / álbo rádość / álbo koroná chluby? Izáli nie wy przed oblicżnośćią Páná náƺego Jezuſá Chryſtuſá w przyjśćie jego?
Wy záiſte jeſteśćie chwáłą náƺą y rádośćią.
