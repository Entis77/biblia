A prośimy wáſ / bráćiá! przez przyjśćie Páná náƺego JEzuſá CHryſtuſá y náƺe zgromádzenie do niego /
Abyśćie śię nie záráz dáli zráżáć z zmyſłu wáƺego / áni ſobą trwożyć / lub przez duchá lub przez mowę / lub przez liſt jákoby od náſ piſány / jákoby nádchodźił dźień CHryſtuſowy.
Niech wáſ nikt nie zwodźi żádnym ſpoſobem ; ábowiem nie przyjdźie on dźień / áżby pierwej przyƺło odſtąpienie / y byłby objáwiony cżłowiek on grzechu / on ſyn zátráceniá.
Który śię ſprzećiwiá y wynośi nád to wƺyſtko / co śię zowie Bogiem / álbo co má Boſką cżeść / ták iż on w kośćiele Bożym jáko Bóg uśiądźie / udáwájąc śię zá Bogá.
Izáli nie pámiętáćie / iż jeƺcże z wámi będąc / o temem wám powiádáł?
A teráz wiećie / co przeƺkádzá / áby był objáwiony cżáſu ſwego.
Abowiem śię już ſpráwuje tájemnicá niepráwośći / tylko że ten / który teráz przeƺkádzá / przeƺkádzáć będźie / áżby był z pośrodku odjęty.
A tedy objáwiony będźie on niezbożnik / którego Pán zábije duchem uſt ſwoich / y znieśie objáwieniem przyjśćiá ſwego.
Którego niezbożniká przyjśćie jeſt podług ſkutku ƺátáńſkiego / ze wƺelką mocą y znákámi y cudámi kłámliwemi /
Y ze wƺelkiem oƺukániem niepráwośći w tych / którzy giną / przeto / iż miłośći práwdy nie przyjęli / áby byli zbáwieni.
A przetoż pośle im Bóg ſkutek błędów / áby wierzyli kłámſtwu /
Aby byli oſądzeni wƺyſcy / którzy nie uwierzyli práwdźie / ále ſobie upodobáli nieſpráwiedliwość.
Lecż my powinniśmy / Bogu dźiękowáć záwƺe zá wáſ / bráćiá umiłowáni od Páná! iż wáſ Bóg od pocżątku wybráł ku zbáwieniu w poświęceniu Duchá y w wierze práwdy /
Do cżego wáſ powołáł przez Ewángieliję náƺę ku doſtąpieniu chwáły Páná náƺego JEzuſá CHryſtuſá.
Przetoż / bráćiá! ſtójćie / á trzymájćie śię náuki podánej / którejśćie śię náucżyli / lub przez mowę lub przez liſt náƺ.
A ſám Pán náƺ JEzuſ CHryſtuſ y Bóg á Ojćiec náƺ / który náſ umiłowáł / y dáł poćiechę wiecżną / y nádźieję dobrą przez łáſkę /
Niech ćieƺy ſercá wáƺe y utwierdzá wáſ w káżdej mowie y w ucżynku dobrym.
