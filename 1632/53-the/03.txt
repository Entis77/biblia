Ná oſtátek / bráćiá! módlćie śię zá náſ / áby śię ſłowo Páńſkie ƺerzyło y rozſłáwiáło / jáko y u wáſ ;
Y ábyśmy byli wyrwáni od niezbędnych y złych ludźi ; ábowiem nie wƺyſtkich jeſt wiárá.
Aleć wierny jeſt Pán / który wáſ utwierdźi y ſtrzedz będźie od złego.
A ufámy w Pánu o wáſ / iż to / co wám rozkázujemy / y cżynićie y cżynić będźiećie.
A Pán niech ſpráwuje ſercá wáƺe ku miłośći Bożej / y ku ćierpliwemu ocżekiwániu CHryſtuſá.
A rozkázujemy wám / bráćiá! w imieniu Páná náƺego JEzuſá CHryſtuſá / ábyśćie śię odłącżyli od káżdego brátá nieporządnie chodzącego / á nie według podánej náuki / którą wźiął od náſ.
Abowiem ſámi wiećie / jáko náſ potrzebá náśládowáć / poniewáżeśmy nie żyli między wámi nieporządnie ;
Aniśmy dármo chlebá jedli u kogo / ále z prácą y z kłopotem we dnie y w nocy robiąc / ábyśmy nikomu z wáſ ćiężkimi nie byli ;
Nie iżbyśmy tej mocy nie mieli / ále żebyśmy wám ſámi śiebie zá przykłád podáli / ábyśćie náſ náśládowáli.
Abowiem gdyśmy byli u wáſ / tośmy wám rozkázáli / że jeźli kto nie chce robić / niechájże też nie je.
Bo ſłyƺymy / iż niektórzy między wámi nieporządnie chodzą / nic nie robiąc / ále śię niepotrzebnemi rzecżámi báwiąc.
Przetoż tákowym rozkázujemy y nápominámy ich przez Páná náƺego JEzuſá CHryſtuſá / áby w pokoju robiąc / ſwój włáſny chleb jedli.
A wy / bráćiá! nie ſłábiejćie / dobrze cżyniąc.
A jeźli kto jeſt niepoſłuƺny ſłowu náƺemu przez liſt piſánemu / tego náznácżćie / á nie mięƺájćie śię z nim / áby śię záwſtydźił ;
Wƺákże nie miejćie go zá nieprzyjáćielá / ále nápominájćie jáko brátá.
A ſám Pán pokoju niech wám dá pokój záwƺe y wƺelkim ſpoſobem. Pán niech będźie z wámi wƺyſtkimi.
Pozdrowienie ręką moją Páwłową / co jeſt znákiem w káżdym liśćie ; ták piƺę.
Łáſká Páná náƺego JEzuſá CHryſtuſá niech będźie z wámi wƺyſtkimi. Amen.
