Przetoż / bráćiá moi mili y pożądáni! rádośći y korono mojá! ták ſtójćie w Pánu / nájmilśi moi!
Ewodyi proƺę / y Syntychy proƺę / áby jednegoż rozumieniá były w Pánu.
Proƺę też y ćię / towárzyƺu wierny! bądź tym ná pomoc / które w Ewángielii weſpół ze mną prácowáły / y z Klementem y z innymi pomocnikámi moimi / których imioná ſą w kśięgách żywotá.
Rádujćie śię záwƺe w Pánu ; znowu mówię / rádujćie śię.
Skromność wáƺá niech będźie wiádomá wƺyſtkim ludźiom ; Pán bliſko jeſt.
Nie troƺcżćie śię o żádną rzecż / ále we wƺyſtkiem przez modlitwę y prośbę z dźiękowániem żądośći wáƺe niech będą znájome u Bogá.
A pokój Boży / który przewyżƺá wƺelki rozum / będźie ſtrzegł ſerc wáƺych / y myśli wáƺych w CHryſtuśie JEzuśie.
A dálej mówiąc / bráćiá / cokolwiek jeſt práwdźiwego / cokolwiek pocżćiwego / cokolwiek ſpráwiedliwego / cokolwiek cżyſtego / cokolwiek przyjemnego / cokolwiek chwálebnego / jeźli którá cnotá / y jeźli którá chwáłá / o tem przemyślájćie.
Cżegośćie śię też náucżyli / y cośćie przyjęli y ſłyƺeli / y widźieli przy mnie / to cżyńćie / á Bóg pokoju będźie z wámi.
A urádowáłem śię wielce w Pánu / żeśćie śię już wżdy znowu záźielenili w ſwojim ſtárániu o mię / jákoż y ſtáráliśćie śię o to / lecż wám ná ſpoſobnym cżáśie ſchodźiło.
Nie żebym to mówił dlá niedoſtátku ; bomći śię já náucżył / ná tem przeſtáwáć / co mám.
Umiem y uniżáć śię / umiem y obfitowáć ; wƺędy y we wƺyſtkich rzecżách jeſtem wyćwicżony / y náſyconym być / y łáknąć / y obfitowáć / y niedoſtátek ćierpieć ;
Wƺyſtko mogę w CHryſtuśie / który mię pośilá.
Wƺákże dobrześćie ucżynili / żeśćie ſpołecżnie dogodźili ućiſkowi mojemu.
A wiećie y wy Filipenſowie / iż ná pocżątku Ewángielii / gdym wyƺedł z Mácedonii / żáden mi zbór nie udźielił ná ráchunek dáwániá y brániá / tylko wy ſámi ;
Poniewáż y do Teſáloniki ráz y drugi / cżego potrzebá było / poſłáliśćie mi /
Nie przeto / żebym dátku ƺukáł ; ále ƺukám pożytku / któryby obfitowáł ná ráchunku wáƺym.
Gdyżem odebráł wƺyſtko / y mám doſtátek / pełenem / wźiąwƺy od Epáfrodytá / co poſłáno od wáſ / wonność dobrego zápáchu / ofiárę przyjemną y Bogu śię podobájącą.
A Bóg mój nápełni wƺelką potrzebę wáƺę według bogáctwá ſwego / chwálebnie / w CHryſtuśie JEzuśie.
A Bogu y Ojcu náƺemu niech będźie chwáłá ná wieki wieków. Amen.
Pozdrówćie wƺyſtkich świętych w CHryſtuśie JEzuśie. Pozdráwiáją wáſ bráćiá / którzy ſą ze mną.
Pozdráwiáją wáſ wƺyſcy święći ; ále oſobliwie / którzy ſą z ceſárſkiego domu.
Łáſká Páná náƺego JEzuſá CHryſtuſá niech będźie z wámi wƺyſtkimi. Amen.
