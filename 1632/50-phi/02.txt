Jeźli tedy máćie jáką poćiechę w Chryſtuśie / jeźli jáką ućiechę miłośći / jeźli jáką ſpołecżność duchá / jeźli ſą jákie wnętrznośći y zlitowániá w wáſ /
Dopełnćież weſelá mojego / ábyśćie jednoż rozumieli / jednoſtájną miłość májąc / będąc jednomyślni y jednoż rozumiejący ;
Nic nie cżyniąc ſpornie / álbo przez próżną chwáłę / ále w pokorze jedni drugich májąc zá wyżƺych nád śię.
Nie upátrujćie káżdy tylko / co jeſt jego / ále káżdy też / co jeſt drugich.
Tego tedy bądźćie o ſobie rozumieniá / które było y w Chryſtuśie Jezuśie.
Który / będąc w kƺtáłćie Bożym / nie pocżytáł ſobie tego zá drápieſtwo równym być Bogu /
Ale wyniƺcżył ſámego śiebie / przyjąwƺy kƺtáłt niewolniká / ſtáwƺy śię podobny ludziom ;
Y poſtáwą ználeziony jáko cżłowiek / ſám śię poniżył / będąc poſłuƺnym áż do śmierći / á to śmierći krzyżowej.
Dlátego też Bóg náder go wywyżƺył / y dárowáł mu imię / które jeſt nád wƺyſtkie imię ;
Aby w imieniu Jezuſowem wƺelkie śię koláno ſkłániáło / tych / którzy ſą ná niebieśiech / y tych / którzy ſą ná ziemi / y tych / którzy ſą pod ziemią.
A wƺelki język áby wyznáwáł / że Jezuſ Chryſtuſ jeſt Pánem ku chwále Bogá Ojcá.
Przetoż / moi mili! jákośćie záwƺe poſłuƺni byli / nie tylko w przytomnośći mojej / ále teráz dáleko więcej w niebytnośći mojej / z bojáźnią y ze drżeniem zbáwienie ſwoje ſpráwujćie.
Abowiem Bóg jeſt / który ſpráwuje w wáſ chcenie y ſkutecżne wykonánie według upodobániá ſwego.
Wƺyſtko cżyńćie bez ƺemrániá y poſwárków /
Abyśćie byli bez nágány y ƺcżeremi dziátkámi Bożemi / nienágánionymi w pośrodku národu złego y przewrotnego / między którymi świećićie jáko świátłá ná świećie /
Záchowywując ſłowá żywotá ku chlubie mojej w dzień Chryſtuſowy / żem dármo nie bieżáł y dármo nie prácowáł.
Ale choćbym ofiárowány był dlá ofiáry y uſługi wiáry wáƺej / weſelę śię / y ſpółweſelę śię ze wƺyſtkimi wámi ;
Z tegoż tedy y wy weſelćie śię / y ſpółweſelćie śię ze mną.
A mám nádzieję w Pánu Jezuśie / iż Tymoteuƺá w rychle poślę do wáſ / ábym śię y já ućieƺył / dowiedziáwƺy śię / co śię z wámi dzieje.
Abowiem nie mám nikogo w umyśle jemu równego / któryby śię uprzejmie o rzecży wáƺe ſtáráć chćiáł ;
Bo wƺyſcy ſwoich rzecży ƺukáją / á nie tych / które ſą Chryſtuſá Jezuſá.
Ale wiećie / iż on jeſt doświádcżonym / á iż jáko ſyn z ojcem ze mną ſłużył w Ewángielii.
Mám tedy nádzieję / że tego do wáſ poślę / ſkoro obácżę / co śię ze mną dálej dziáć będzie ;
A mám ufność w Pánu / że y ſám w rychle do wáſ przyjdę.
Alem rozumiáł rzecżą potrzebną / Epáfrodytá / brátá y pomocniká y ſpółbojowniká mego / á wáƺego Apoſtołá y ſługę w potrzebie mojej / poſłáć do wáſ /
Poniewáż prágnął wáſ wƺyſtkich / y bárdzo śię fráſowáł / żeśćie ſłyƺeli / iż záchorowáł.
Bo wpráwdzie chorowáł máło nie ná śmierć ; ále śię Bóg zmiłowáł nád nim / á nie tylko nád nim / ále y nádemną / ábym nie miáł ſmutku ná ſmutek.
Przetoż tem ochotniej poſłáłem go / ábyśćie záśię ujrzáwƺy go / uweſelili śię / á já ábym miáł mniej ſmutku.
Przyjmijćież go tedy w Pánu ze wƺyſtkiem weſelem ; á tákich w pocżćiwośći miejćie ;
Boć dlá dziełá Chryſtuſowego bliſkim był śmierći / odwáżywƺy zdrowie ſwoje / áby dopełnił tego / cżego niedoſtáwáło w uſłudze wáƺej przećiwko mnie.
