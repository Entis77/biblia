Páweł y Tymoteuƺ / ſłudzy JEzuſá CHryſtuſá / wƺyſtkim świętym w CHryſtuśie JEzuśie / którzy ſą w mieśćie Filipiſ / z biſkupámi y z dyjákonámi.
Łáſká wám y pokój niech będźie od Bogá / Ojcá náƺego / y od Páná JEzuſá CHryſtuſá.
Dźiękuję Bogu memu / ile kroć ná wáſ wſpominám /
( Záwƺe w káżdej modlitwie mojej zá wƺyſtkich wáſ z rádośćią prośbę cżyniąc. )
Zá ſpołecżność wáƺę w Ewángielii / od pierwƺego dniá áż dotąd ;
Pewien tego będąc / iż ten / który pocżął w wáſ dobrą ſpráwę / dokoná áż do dniá JEzuſá CHryſtuſá.
Jákoż ſprwiedliwá jeſt / ábym já to rozumiáł o wáſ wƺyſtkich / dlátego / iż wáſ mám w ſercu mojem y w więźieniu mojem / y w obronie / y w utwierdzeniu Ewángielii / wáſ / mówię / wƺyſtkich / którzyśćie ze mną ucżeſnikámi łáſki.
Abowiem świádkiem mi jeſt Bóg / jáko wáſ wƺyſtkich prágnę we wnętrznośćiách JEzuſá CHryſtuſá.
Y o to śię modlę / áby miłość wáƺá im dálej tem więcej pomnáżáłá śię w znájomośći y we wƺelkim zmyśle /
Abyśćie mogli rozeznáć rzecży różne / żebyśćie byli ƺcżerymi y bez obráżeniá ná dźień CHryſtuſowy /
Będąc nápełnieni owocámi ſpráwiedliwośći / które przynośićie przez JEzuſá CHryſtuſá ku ſłáwie y chwále Bożej.
A chcę / bráćiá! ábyśćie wiedźieli / iż to / co śię ze mną dźieje / ná więkƺe pomnożenie Ewángielii wyƺło.
Ták iż związki moje dlá CHryſtuſá rozgłoƺone ſą po wƺyſtkim páłácu ceſárſkim / y u wƺyſtkich inƺych.
A wiele ich z bráći w Pánu ſercá nábywƺy z moich związek / śmielƺymi ſą / bez bojáźni mówić ſłowo.
Wƺákże niektórzy z zázdrośći y z ſporu / á niektórzy też z dobrej woli CHryſtuſá káżą.
A ći / którzy z ſporu CHryſtuſá opowiádáją nieƺcżerze / mniemáją / iż przydáwáją ućiſku związkom moim ;
A którzy z miłośći / wiedzą / żem jeſt wyſtáwiony ku obronie Ewángielii /
Ale cóż ná tem? Owƺem jákimkolwiek ſpoſobem / lub poſtáwnie / lub w práwdźie CHryſtuſ bywá opowiádány / y z tego śię ráduję / y jeƺcże śię rádowáć będę ;
Gdyż wiem / iż mi to wynijdźie ná zbáwienie przez modlitwę wáƺę y pomoc Duchá JEzuſá CHryſtuſá /
Według troſkliwego ocżekiwániá y nádźiei mojej / iż śię w nicżem nie záwſtydzę ; ále z wƺelákiem bezpiecżeńſtwem / jáko záwƺe / ták y teráz / uwielbionym będźie CHryſtuſ w ćiele mojem / lub przez żywot / lub przez śmierć.
Abowiem mnie żyćiem jeſt CHryſtuſ / á umrzeć zyſk.
A jeźliż żyć w ćiele jeſt mi to owocem prácy mojej / jednák nie wiem / cobym obráć miáł.
Abowiem jeſtem śćiśniony od tego obojgá / prágnąc być rozwiązány / á być z CHryſtuſem / bo to dáleko lepiej :
Ale zoſtáć w ćiele potrzebniej jeſt dlá wáſ.
A będąc tego pewien / wiem / iż zoſtánę / y z wámi wƺyſtkimi pomieƺkám ku wáƺemu pomnożeniu y weſelu wiáry /
Aby obfitowáłá chlubá wáƺá w CHryſtuśie JEzuśie ze mnie / gdy do wáſ záśię przybędę.
Tylko śię ták ſpráwujćie / jáko przyſtoi Ewángielii CHryſtuſowej / ábym / lub przyjdę y oglądám wáſ / lub nie przyjdę / ſłyƺáł o wáſ / iż ſtoićie w jednym duchu / jednomyślnie bojując w wierze Ewángielii.
Ani w cżem nie ſtráchájąc śię przećiwników / co onym jeſt pewnym znákiem zginieniá / á wám zbáwieniá / á to od Bogá ;
Gdyż wám to dáne dlá CHryſtuſá / ábyśćie nie tylko weń wierzyli / ále ábyśćie też dlá niego ćierpieli /
Tenże bój májąc / jákiśćie widźieli we mnie / y jáki teráz o mnie ſłyƺyćie.
