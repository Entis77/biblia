To mi jeƺcże ukázał pánujący Pán / oto był koƺ letniego owocu.
Tedy rzekł : Cóż widźiƺ Amośie? Y rzekłem : Koƺ letniego owocu. Znowu rzekł Pán do mnie : Przyƺedł koniec ludowi memu Izráelſkiemu / nie będę mu już więcey przeglądał.
Tedy śię obrócą w kwilenie pieśni kośćielne dniá onego / mówi pánujący Pán / mnóſtwá trupów ná káżde miejſce po ćichu nárzucáją.
Słuchájćież tego / którzy pożeráćie ubogiego / ábyśćie wygubili chudźiny z źiemi ;
Y máwiáćie : Kiedyż przeminie nów mieśiącá / ábyſmy ſprzedáwáli zboże? y ſábát / ábyſmy otworzyli ſpichlerze? ábyſmy umniejƺyli miáry efá / á podwyżƺyli wági / á ƺále zdrádliwie ſfáłƺowáli /
Kupując ubogich zá pieniądze / á chudźinę zá párę trzewików ; nádto ábyſmy odmiećiny zbóż ſprzedáwáli.
Przyśiągł Pán przez zácność Jákóbową / że nie zápomnę ná wieki wƺyſtkich ſpráw ich.
Izáliby śię y źiemiá nád tym nie poruƺyłá / y nie płákáłby káżdy / kto mieƺká ná niej? y owƺem / wzbierze wƺyſtká jáko rzeká / y porwáná y zátopioná będźie jáko rzeką Egipſką.
A dniá onego / mówi pánujący Pán / ſpráwię / że ſłońce zájdźie o południu / y przywiodę ćiemność ná źiemię w dźień jáſny ;
Y obrócę w płácż świętá wáƺe / á wƺyſtkie pieśni wáƺe w nárzekánie / y ſpráwię to / że będźie ná wƺyſtkich biodrách wór / y ná káżdej głowie obłyśienie ; y będźie w tey źiemi kwilenie / jáko nád jednorocżnym / á oſtátecżne rzecży jey jáko dźień gorzkośći.
Oto dni przychodzą / mówi pánujący Pán / że poślę głód ná źiemię / nie głód chlebá / áni prágnienie wody / ále ſłuchánia ſłów Páńſkich /
Ták / że śię tułáć będą od morzá áż do morzá / y od północy áż ná wſchód biegáć będą / ƺukájąc ſłowá Páńſkiego / wƺákże nie znájdą.
Dniá onego pomdleją pánienki piękne / náwet y młodźieńcy od onego prágnieniá ;
Którzy przyśięgáją przez obrzydliwość Sámáryi / y mówią : Jáko żyje Bóg twój / o Dán! y jáko żyje drogá Beerſebá ; y upádną / á nie powſtáną więcey.
