Słuchájćie ſłowá tego / które mówi Pán przećiwko wám / ſynowie Izráelſcy! przećiwko wƺyſtkiemu rodzájowi / którym wywiódł z źiemi Egipſkiej / mówiąc :
Tylkom was ſámych poznał ze wƺyſtkich rodzájów źiemi ; przetoż was náwiedzę dla wƺyſtkich niepráwośći wáƺych.
Izáli dwá ſpołem pójdą nie zgodźiwƺy śię?
Izáli zárycży lew w leśie / gdyby nie miał łupu? Izáli wydá lwię głoſ ſwój z jáſkini ſwojey / gdyby łápáć nie miáło?
Izáli pták wpádnie w śidło ná źiemi / gdyby śidłá nie było? Izáli będźie podnieśione śidło z źiemi / gdyby nic nie uwięzło?
Izáli śię ludźie nie lękáją / gdy trąbá w mieśćie zábrzmi? Izáli jeſt w mieśćie co złego / któregoby Pán nie ucżynił?
Záiſte nic nie cżyni pánujący Pán / chybá żeby objáwił tájemnicę ſwoję ſługom ſwoim / prorokom.
Lew rycży / któżby śię nie báł? Pánujący Pán mówi / któżby nie prorokowáł?
Obwołájćie w páłácách w Azoćie y w páłácách źiemi Egipſkiej / á mówćie : Zbierzćie śię ná góry Sámáryi / á obácżćie wielkie zámięƺániá w pośrodku jey / y ućiſk ćierpiących w niej ;
Y że nie umieją cżynić / co jeſt práwego / mówi Pán / zbierájąc ná páłácách ſwoich ſkárby z zdźierſtwá y z łupieſtwá.
Przetoż ták mówi pánujący Pán : Oto nieprzyjáćiel około tey źiemi / á ten odejmie od ćiebie śiłę twoję / y rozchwycone będą páłáce twoje.
Ták mówi Pán : Jáko wyrywá páſterz z páƺcżęki lwiej dwá gnáty álbo kęſ uchá / ták wyrwáni będą ſynowie Izráelſcy / którzy mieƺkáją w Sámáryi ná ſtronie łożá / y ná ſtronie pośćieli.
Słuchájćie á oświádcżájćie w domu Jákóbowym / mówi pánujący Pán / Bóg záſtępów.
Bo dniá onego / którego Izráelá náwiedzę dla przeſtępſtwá jego / náwiedzę też ołtárze w Betel / y odćięte będą rogi ołtárzá / ták / że ná źiemię upádną ;
Y uderzę dom źimy o dom látá / á zginą domy z kośći ſłoniowych / á domy zácne koniec wezmą / mówi Pán.
