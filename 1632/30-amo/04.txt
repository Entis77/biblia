Słuchájćie ſłowá tego / o krowy Báſáńſkie! któreśćie ná górách Sámáryi / które ućiſkáćie nędzników á niƺcżyćie ubogich / które mówićie pánom ich : Przynieśćie / ábyśmy piły.
Przyśiągł pánujący Pán przez świętobliwość ſwoję / iż oto dni idą ná wáſ / których nieprzyjáćiel weźmie wáſ ná háki / á potomki wáƺe ná wędy rybáckie / y wyjdziećie przerwámi / káżdá ták jáko ſtoi ;
Y będziećie rozrzucáć / cokolwiek było w páłácách wáƺych / mówi Pán.
Idźćież do Betel á bądzćie tułácżámi w Gálgál ; rozmnóżćie przeſtępſtwá / á przynośćie ná káżdy poránek ofiáry wáƺe / y trzećiego roku dzieśięćiny wáƺe ;
A páląc ofiárę chwáły z kwáƺonych rzecży / obwołájćie ofiáry dobrowolne / y rozgłośćie / poniewáż śię wám ták podobá / o ſynowie Izráelſcy! mówi pánujący Pán.
A choćiáżem Já wám dáł cżyſtość zębów we wƺyſtkich miáſtách wáƺych / to jeſt / niedoſtátek chlebá po wƺyſtkich miejſcách wáƺych / wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Jám też záhámowáł od wáſ deƺcż / gdy jeƺcże były trzy mieśięce do żniwá / á ſpuśćiłem deƺcż ná jedne miáſto / á ná drugiem miáſto nie ſpuśćił ; jedná dziedziná byłá deƺcżem odwilżoná / á drugá dziedziná / ná którą deƺcż nie pádáł / uſchłá.
Y chodziły dwá y trzy miáſtá do jednego miáſtá / áby piły wodę / á nie mogły śię nápić ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Uderzyłem wáſ ſuƺą y rdzą ; obfitość / którą przynośiły ogrody wáƺe / y winnice wáƺe / y figowe ſády wáƺe / y oliwnice wáƺe / gąśienice pożárły / á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Poſłáłem ná wáſ mór / ták jáko ná Egipt / pobiłem miecżem młodzieńców wáƺych / w pojmániem podáł konie wáƺe / y ſpráwiłem / że ſmród wojſk wáƺych wyſtępowáł w nozdrzá wáƺe ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Wywróćiłem wáſ / jáko Bóg wywróćił Sodomę y Gomorę / ták / żeśćie byli jáko głowniá wyrwáná z ogniá ; á wƺákżeśćie śię nie náwróćili do mnie / mówi Pán.
Przetoż ták ći ucżynię / o Izráelu! á iż ći ták ucżynić chcę / bądźże gotowym ná zábieżenie Bogu ſwemu / o Izráelu!
Abowiem oto on jeſt / który kƺtáłtuje góry / á tworzy wiátry / y który oznájmuje cżłowiekowi / jáká jeſt myśl jego ; on z ránnej zorzy ćiemność cżyni / á depcże wyſokośći ziemi ; Pán Bóg záſtępów jeſt imię jego.
