Widziáłem Páná ſtojącego ná ołtárzu / który rzekł : Uderz w gáłkę / áż zádrżą podwoje / á rozetnij je wƺyſtkie od wierzchu ich / á oſtátek miecżem pobiję ; żáden z nich nie ućiecże / y nie będzie z nich nikt / coby tego uƺedł.
Choćby śię zákopáli w ziemię / y ſtámtądby ich ręká mojá wzięłá ; choćby wſtąpili áż do niebá / y ſtámtądbym ich ſtárgnął.
A choćby śię ſkryli ná wierzchu Kármelu / wyƺpieguję y wezmę ich ſtámtąd ; á choćby śię ſkryli przed ocżymá mojemi ná dnie morſkiem / przykáżę wężowi / áby ich y ſtámtąd wykąſáł ;
A choćby poƺli w niewolę przed nieprzyjáćiółmi ſwymi / y tám przykáżę miecżowi / áby ich pomordowáł ; obrócę záiſte przećiwko nim oko ſwe ná złe / á nie ná dobre.
Bo pánujący Pán záſtępów / gdy śię dotknie ziemi / rozpływá śię / á płácżą wƺyſcy mieƺkájący ná niej / y wzbierá wƺyſtká jáko rzeká / á zátopioná bywá jáko rzeką Egipſką.
Który ná niebieśiech zbudowáł páłáce ſwoje / á záſtęp ſwój ná ziemi uƺykowáł ; który może záwołáć wody morſkie / á wyláć je ná oblicże ziemi ; Pán jeſt imię jego.
Izáli nie jeſteśćie podobni ſynom Murzyńſkim przedemną / o ſynowie Izráelſcy? mówi Pán ; izálim Izráelá nie wywiódł z ziemi Egipſkiej jáko Filiſtyńcżyków z Káftor / y Syryjcżyków z Kir?
Oto ocży pánującego Páná przećiwko temu króleſtwu grzeƺącemu / ábym je wygłádził z oblicżá ziemi ; wƺákże nie wygłádzę do ƺcżętu domu Jákóbowego / mówi Pán.
Bom oto Já rozkázáł / á rozmiecę między wƺyſtkie národy dom Izráelſki jáko miotáná bywá pƺenicá ná przetáku / ták / iż nieprzepádnie y kámyk ná ziemię.
Wƺyſcy grzeƺnicy z ludu mojego od miecżá pomrą / którzy mówią : Nie przybliży śię do náſ / áni náſ záchwyći to złe.
Dniá onego wyſtáwię upádły przybytek Dawidowy / á zágrodzę rozerwánie jego / y obáliny jego nápráwię / á pobuduję go / jáko zá dni dáwnych ;
Aby pośiedli oſtátki Edomcżyków y wƺyſtkie národy / nád którymi wzywáno imieniá mojego / mówi Pán / który to cżyni.
Oto dni idą / mówi Pán / że orácż żeńcę zájmie / á ten / co tłocży winne jágody / rozśiewájącego náśienie ; á góry moƺcżem kropić będą / á wƺyſtkie págórki śię rozpłyną.
Y náwrócę záś z więzieniá lud mój Izráelſki / y pobudują miáſtá ſpuſtoƺone / á mieƺkáć w nich będą ; ſądzić też będą winnice / y wino z nich pić będą ; ſádów też náƺcżepią / y owoc ich jeść będą.
A ták ich wƺcżepię w ziemi ich / że nie będą więcej wykorzenieni z ziemi ſwojej / którąm im dáł / mówi Pán / Bóg twój.
