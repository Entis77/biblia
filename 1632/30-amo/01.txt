Słowá Amoſá / który był między páſterzámi z Tekuá / które widźiáł o Izráelu zá dni Uzyjáƺá / królá Judzkiego / y zá dni Jeroboámá / ſyná Joázowego / królá Izráelſkiego / dwá látá przed trzęśieniem źiemi.
Y rzekł : Zárycży Pán z Syonu / z Jeruzálemu wydá głoſ ſwój ; y będą płákáły mieƺkániá páſterzy / á wyſchną polá nájwyborniejƺe.
Ták mówi Pán : Dlá trzech wyſtępków Dámáƺku / owƺem / dlá cżterech / nie przepuƺcżę mu / przeto / że młóćili wozámi żeláznemi Gáláádá ;
Ale poślę ogień ná dom Házáelá / który pożre páłáce Benádádowe.
Połámię też záworę w Dámáƺku / á wykorzenię obywátelá z doliny Awen / y tego / który trzymá ſceptr z domu Heden ; y pójdźie w niewolę lud Syryjſki do Kir / mówi Pán.
Ták mówi Pán : Dlá trzech wyſtępków Gázy / owƺem / dlá cżterech / nie przepuƺcżę im / przeto / że ich pojmáwƺy / w więźienie wiecżne podáwáli Edomcżykom ;
Ale poślę ogień ná mur Gázy / który pożre páłáce jej.
Wykorzenię też obywátelá z Azotu / y tego / który trzymá ſceptr z Aƺkálonu / y obrócę rękę moję przećiwko Akkáronowi / że zginie oſtátek Filiſtynów / mówi pánujący Pán.
Ták mówi Pán : Dlá trzech wyſtępków Tyru / owƺem / dlá cżterech / nie przepuƺcżę mu / przeto / że ich w więźienie wiecżne podáli Edomcżykom / á nie wſpomnieli ná przymierze bráterſkie ;
Ale poślę ogień ná mur Tyrſki / który pożre páłáce jego.
Ták mówi Pán : Dlá trzech wyſtępków Edomá / owƺem / dlá cżterech / nie przepuƺcżę mu / przeto / że zepſowáwƺy w ſobie wƺeláką litość ſwoję prześláduje miecżem brátá ſwego / á gniewem ſwym uſtáwicżnie páłá / owƺem / zápálcżywość jego rozśilá śię bez przeſtániá ;
Ale poślę ogień ná Temán / y pożre páłáce w Bocrá.
Ták mówi Pán : Dlá trzech wyſtępków ſynów Amonowych / owƺem / dlá cżterech / nie przepuƺcżę mu / przeto / iż rozćináli brzemienne w Gáláád / tylko áby rozƺerzáli gránice ſwoje ;
Ale rozniecę ogień ná murze Rábby / który pożre páłáce jego z krzykiem w dźień wojny / y z wichrem w dźień niepogody.
Y pójdźie król ich w niewolę / on y kśiążętá jego z nim / mówi Pán.
