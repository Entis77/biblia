Słuchájćie ſłowá tego / które Ja wydáję przećiwko wam / to jeſt nárzekánia / o domie Izráelſki!
Upádnie / á nie powſtánie więcey pánná Izráelſká ; opuƺcżoná będźie w źiemi ſwej / á nie będźie / ktoby ją podnióſł.
Bo ták mówi pánujący Pán : W mieśćie / z którego wychodźiło tyśiąc / zoſtánie ſto / á w tym / z którego wychodźiło ſto / zoſtánie dźieśięć domowi Izráelſkiemu.
Bo ták mówi Pán domowi Izráelſkiemu : Szukájćie mię / á żyć będźiećie ;
A nie ƺukájćie Betelá / áni chodźćie do Gálgál / y do Beerſeby nie udáwájćie śię ; bo Gálgál w niewolę záwiedźione będźie / á Betel śię wniwecż obróći.
Szukájćie Páná / á żyć będźiećie / by ſnadź domu Józefowego nie przeniknął jáko ogień / y nie pochłonął Betel / á nie byłby / ktoby ugáśił ;
Którzy obrácáćie ſąd w piołun / á ſpráwiedliwość ná źiemi opuƺcżáćie : Szukájćie / mówię.
Tego / który ucżynił Báby ná niebie y Oryjoná / który ćień śmierći w poránek odmieniá y dźień w ćiemnośći nocne ; który przywołuje wody morſkie / á wylewá je ná oblicże źiemi / Pán jeſt imię jego ;
Który pokrzepiá ſłábego przećiwko mocárzowi / ták że ten oſłábiáły do twierdzy uchodźi.
Máją w nienáwiśći tego / który ich w brámie kárze ; á tym / co mówi rzecży dobre / brzydzą śię.
Przetoż / iż ućiſkáćie ubogiego / á brzemię zbożá bierzećie od niego / domóweśćie wpráwdźie z ćioſánego kámieniá nábudowáli / ále nie będźiećie w nich mieƺkáć ; winnic rozkoƺnych náſádźiliśćie / ále winá z nich pić nie będźiećie.
Bo wiem o wielkich przeſtępſtwách wáƺych / y ſrogich grzechách wáƺych / że ćiemiężyćie ſpráwiedliwego / biorąc pocżty / á ubogich ſpráwy w brámie podwrácáćie.
Przetoż roztropny cżáſu onego milcżeć muśi ; bo cżás zły jeſt.
Szukájćie dobrego á nie złego / ábyśćie żyli ; á będźie ták Pán Bóg zaſtępów z wámi / jáko mówićie.
Miejćie w nienáwiśći złe / á miłujćie dobre / á ſąd poſtánówćie w brámie ; owá śię ſnadź Pán / Bóg zaſtępów / nád oſtátkiem Józefá zmiłuje.
Przetoż ták mówi pánujący Pán / Bóg zaſtępów : Po wƺyſtkich ulicách będźie nárzekánie / á po wƺyſtkich ſtronách zákrzykną : Biádá / biádá! y záwołáją orácżá do płácżu y do kwileniá z tymi / którzy nárzekáć umieją.
Owƺem / y po wƺyſtkich winnicách będźie nárzekánie / gdy przejdę przez pośrodek ćiebie / mówi Pán.
Biádá tym / którzy żądáją dniá Páńſkiego! cóż wam po tym dniu Páńſkim / poniewáż jeſt ćiemnośćią / á nie świátłośćią?
Jáko gdyby kto ućiekał przede lwem / á zábieżał mu niedźwiedź ; álbo gdyby wƺedł do domu / á podpárł śię ręką ſwą ná śćiánie / ukąśiłby go wąż.
Izali dźień Páńſki nie jeſt dźień ćiemnośći / á nie świátłośći / w którym niemáƺ jáſnośći / ále chmurá?
Mam w nienáwiśći y odrzućiłem urocżyſte świętá wáƺe / áni śię kochám w ofiárách zgromádzenia wáƺego.
Bo jeſli mi ofiárowáć będźiećie cáłopalenia / y śniedne ofiáry wáƺe / nie przyjmę ich / á ná ſpokojne ofiáry tłuſtych bydeł wáƺych nie wejrzę.
Odejmij odemnie wrzáſk pieśni ſwojich ; bo ich y dźwięku hárf wáƺych ſłucháć nie chcę.
Ale ſąd náwálnie popłynie / jáko wodá / á ſpráwiedliwość jáko ſtrumień gwáłtowny.
Izáliśćie mi ofiáry y dár ofiárowáli ná puƺcży przez cżterdźieśći lat / domie Izráelſki?
Owƺem / nośiliśćie námiot Molochá wáƺego y Kijuná / obrázy wáƺe / gwiázdę bogów wáƺych / którycheśćie ſobie nácżynili.
Przetoż was záprowádzę zá Dámáƺek / mówi Pán / Bóg zaſtępów imię jego.
