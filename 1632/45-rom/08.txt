Przetoż teraz żadne° potępienia niemáƺ tym / który będąc w CHryſtuśie JEzuśie nie według ćiáłá chodzą / ále według Duchá.
Abowiem Zakon Duchá żywotá / <i>który jeſt</i> w CHryſtuśie JEzuśie / uwolnił mię od zakonu grzechu y śmierći.
Bo co niemożnego <i>było</i> zakonowi / w cżym <i>on</i> był ſłáby dla ćiáłá ; Bóg poſławƺy Syná ſwego w podobieńſtwie grzeƺne° ćiáłá / y dla grzechu / potępił grzech w ćiele :
Aby oná ſpráwiedliwość Zakonna byłá wypełniona w nas / którzy nie według ćiáłá chodźimy / ále według Duchá.
Abowiem którzy ſą według ćiáłá / o tym myślą co jeſt ćieleſnego : ále którzy <i>ſą</i> według Duchá / <i>myślą</i> o tym co jeſt duchownego.
Gdyż zmyſł ćiáłá / <i>jeſt</i> śmierć : ále zmyſł duchá <i>jeſt</i> żywot y pokój.
Przeto / iż zmyſł ćiáłá jeſt nieprzyjaćielem Bogu : bo śię Zakonowi Bożemu nie poddawa / gdyż też y nie może.
Przetoż którzy ſą w ćiele / Bogu śię podobáć nie mogą.
Lecż wy nie jeſteśćie w ćiele / ále w duchu ; gdyż Duch Boży mieƺka w was. A jeſli kto Duchá CHryſtuſowego nie ma / ten nie jeſt jego.
Ale jeſli CHryſtus w was jeſt / tedy ćiáło jeſt martwe dla grzechu / á Duch jeſt żywy dla ſpráwiedliwośći.
A jeſli Duch tego / który JEzuſá wzbudźił zmartwych / mieƺka w was ; ten który wzbudźił CHryſtuſá z martwych / ożywi y śmiertelne ćiáłá wáƺe przez Duchá ſwego który w was mieƺka.
A ták bráćia / dłużnikámi jeſteſmy nie ćiáłu / ábyſmy według ćiáłá żyli.
Abowiem jeſlibyśćie według ćiáłá żyli / pomrzećie : ále jeſlibyśćie Duchem ſpráwy ćiáłá umartwili / żyć będźiećie.
Bo którzykolwiek Duchem Bożym prowádzeni bywáją / ći ſą ſynámy Bożymi.
Gdyżeśćie nie wźięli Duchá niewoli znowu ku bojáźni / áleśćie wźięli Duchá przyſpoſobienia ſynowſkiego ; przez którego wołamy Abbá / <i>to jeſt,</i> Ojcże.
Tenże Duch poświadcża Duchowi náƺemu / yżeſmy dźiećmi Bożymi.
A jeſliż dźiećmi / <i>tedy</i> y dźiedźicámi : dźiedźicámi w prawdźie Bożymi / á ſpółdźiedźicámi CHryſtuſowymi jeſli tylko z nim ćierpimy / ábyſmy też z nim byli uwielbieni.
Abowiem / Mam zá to / iż nie ſą równe utrapienia teráźniejƺego cżáſu / z oną przyƺłą chwałą która śię ma objáwić w nas.
Bo troſkliwe wyglądánie ſtworzenia / ocżekawa objáwienia ſynów Bożych?
Gdyż ſtworzenie márnośći jeſt poddáne / nie dobrowolnie ; ále dla tego / który je poddał :
Pod nádźieją ; że y ſámo ſtworzenie będźie uwolnione z niewoli ſkáżenia / ná wolność chwały dźiatek Bożych.
Bo wiemy / iż wƺyſtko ſtworzenie weſpół wzdycha / y weſpół boleje / áż do tąd.
A nie tylko <i>ono ſtworzenie,</i> ále y my którzy mamy pierwiaſtki Duchá / y my ſámi w ſobie wzdychamy ocżekawájąc przyſpoſobienia ſynowſkiego / <i>to jeſt</i> odkupienia ćiáłá náƺego.
Abowiem nádźiejąſmy zbáwieni. A nádźiejá widoma / nie jeſt nádźiejá : bo co kto widźi / przecż śię <i>tego</i> ſpodźiewa.
Ale cżego nie widźiemy / tego śię ſpodźiewamy / <i>y tego</i> przez ćierpliwość ocżekawamy.
Tákże też y Duch dopomaga mdłośćiam náƺym. Abowiem o cobyſmy śię modlić mieli / jáko potrzebá nie wiemy : ále tenże Duch przycżynia śię zá námi wzdychániem niewymownym.
A ten który śię ſerc bada / wie który jeſt zmyſł Duchá : ponieważ według Bogá przycżynia śię zá świętymi.
A wiemy iż tym którzy miłują Bogá / wƺyſtkie rzecży dopomagáją ku dobremu ; <i>to jeſt</i> tym którzy według poſtánowienia <i>Bożego</i> powołáni ſą.
Abowiem które on przejrzał / <i>to</i> też przenáznácżył / áby byli przypodobáni obrázowi Syná jego ; żeby on był pierworodnym miedzy wielą bráći.
A które przenáznácżył / te też powołał : á które powołał / te też uſpráwiedliwił : á które uſpráwiedliwił / te też uwielbił.
Cóż tedy rzecżemy ná to? Jeſli Bóg zá námi / któż przećiwko nam?
Który áni właſnemu Synowi nie przepuśćił / ále go zá nas wƺyſtkich wydał : y jákożby nam wƺyſtkiego z nim nie dárował?
Któż będźie ſkárżył ná wybráne Boże? Bóg <i>jeſt</i> który uſpráwiedliwia.
Któż <i>jeſt</i> coby <i>je</i> potępił? CHryſtus jeſt / który umárł ; owƺem y <i>zmartwych</i>wſtał : który też jeſt ná práwicy Bożey / który śię też przycżynia zá námi.
Któż nas odłącży od miłośći CHryſtuſowey? <i>cżyli</i> utrapienie? cżyli ućiſk? cżyli prześládowánie? cżyli głód? cżyli nágość? cżyli niebeſpiecżeńſtwo? cżyli miecż?
Jáko nápiſano ; Dla ćiebie cáły dźień zábijáni bywamy / pocżytániſmy jáko owce / ná rzeź <i>náznácżone</i>.
Ale w tym wƺyſtkim przewyćiężamy przez tego / który nas umiłował.
Abowiem pewienem tego / iż áni śmierć / áni żywot / áni Anjołowie / áni kśięſtwá / áni mocárſtwá / áni teráźniejƺe / áni przyƺłe <i>rzecży</i> :
Ani wyſokość / áni głębokość / áni żadne inƺe ſtworzenie / nie będźie nas mogło odłącżyć od miłośći Bożey / która jeſt w JEzuśie CHryſtuśie PAnu náƺym.
