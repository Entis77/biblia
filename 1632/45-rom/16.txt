A zálecám wám Febę / śioſtrę náƺę / którá jeſt ſłużebnicą zboru Kienchreeńſkiego ;
Abyśćie ją przyjęli w Pánu / jáko przyſtoi świętym / y ſtáli przy niej / w którejby wáſ kolwiek rzecży potrzebowáłá ; ábowiem y oná wielom goſpody użycżáłá / áż y mnie ſámemu.
Pozdrówćie Pryſcyllę y Akwilę / pomocniki moje w Chryſtuśie Jezuśie ;
( Którzy zá duƺę moję ſwojej włáſnej ƺyi nádſtáwiáli ; którym nie já ſám dziękuję / ále y wƺyſtkie zbory pogáńſkie. )
Tákże zbór / który jeſt w domu ich. Pozdrówćie Epenetá miłego mojego / który jeſt pierwiáſtkiem Achái do Chryſtuſá.
Pozdrówćie Máryję / którá wiele prácowáłá dlá náſ.
Pozdrówćie Androniká y Junijáƺá / krewnych moich y ſpółwięźni moich / którzy znácżnymi ſą między Apoſtołámi / którzy y przedemną byli w Chryſtuśie.
Pozdrówćie Amplijáſá / miłego mojego w Pánu.
Pozdrówćie Urbáná / pomocniká náƺego w Chryſtuśie / y Stáchyná mnie miłego.
Pozdrówćie Apelleſá doświádcżonego w Chryſtuśie. Pozdrówćie tych / którzy ſą z domu Aryſtobulowego.
Pozdrówćie Herodijoná / pokrewnego mojego. Pozdrówćie tych / którzy ſą z domu Nárcyſſowego / tych / którzy ſą w Pánu.
Pozdrówćie Tryfenę / y Tryfoſę / które prácują w Pánu. Pozdrówćie Perſydę miłą / którá wiele prácowáłá w Pánu.
Pozdrówćie Rufá / wybránego w Pánu y mátkę jego y moję.
Pozdrówćie Aſynkrytá / Flegontá / Hermáná / Pátrobę / Hermená / y bráći / którzy ſą z nimi.
Pozdrówćie Filologá / y Juliję / Nerego y śioſtrę jego / y Olimpá y wƺyſtkich świętych / którzy ſą z nimi.
Pozdrówćie jedni drugich z pocáłowániem świętym. Pozdráwiáją wáſ zbory Chryſtuſowe.
A proƺę wáſ / bráćiá! ábyśćie upátrywáli tych / którzy cżynią rozerwániá y zgorƺeniá przećiwko tej náuce / którejśćie śię wy náucżyli ; y chrońćie śię ich.
Abowiem tákowi Pánu náƺemu Jezuſowi Chryſtuſowi nie ſłużą / ále włáſnemu brzuchowi ſwemu / á przez łágodną mowę y pochlebſtwo ſercá proſtych zwodzą.
Bo poſłuƺeńſtwo wáƺe wƺyſtkich doƺło. A przetoż ráduję śię z wáſ ; ále chcę / ábyśćie byli mądrymi ná dobre / á proſtymi ná złe.
A Bóg pokoju zetrze ƺátáná pod nogi wáƺe w rychle. Łáſká Páná náƺego Jezuſá Chryſtuſá niech będzie z wámi. Amen.
Pozdráwiáją wáſ Tymoteuƺ / pomocnik mój / y Lucyjuƺ y Jáſon y Soſypáter / pokrewni moi.
Pozdráwiám wáſ w Pánu já Tercyjuƺ / którym ten liſt piſáł.
Pozdráwiá wáſ Gájuſ / goſpodárz mój y wƺyſtkiego zboru. Pozdráwiá wáſ Eráſtuſ ƺáfárz miejſki / y Kwártuſ brát.
Łáſká Páná náƺego Jezuſá Chryſtuſá niech będzie z wámi wƺyſtkimi. Amen.
A temu / który wáſ może utwierdzić według Ewángielii mojej y opowiádániá Jezuſá Chryſtuſá / według objáwieniá tájemnicy od cżáſów wiecżnych zámilcżánej /
Lecż teráz objáwionej y przez Piſmá prorockie według poſtánowieniá wiecżnego Bogá ku poſłuƺeńſtwu wiáry między wƺyſtkimi národy oznájmionej ;
Temu / ſámemu mądremu Bogu niech będzie chwáłá przez Jezuſá Chryſtuſá ná wieki. Amen.
