Cżemże tedy zácniejƺy Żyd? álbo co zá pożytek obrzezki?
Wielki z káżdej miáry. Abowiem to nájpierwƺá / iż im zwierzone były wyroki Boże.
Bo cóż ná tem / jeſli niektórzy nie uwierzyli? Azáż niedowiárſtwo ich niƺcży wiárę Bożą?
Nie dáj tego Boże! Y owƺem niech Bóg będźie práwdźiwy / á wƺelki cżłowiek kłámcá / jáko nápiſáno : A ábyś był uſpráwiedliwiony w mowách twoich / á żebyś zwyćiężył / gdybyś ſądźił.
Jeſli tedy nieſpráwiedliwość náƺá Bożą ſpráwiedliwość zálecá / cóż rzecżemy? Azáż nieſpráwiedliwy jeſt Bóg / który gniew przywodźi? ( Po ludzku mówię. )
Nie dáj tego Boże! Abowiem jákożby Bóg ſądźił świát?
Bo jeſli práwdá Bożá przez moje kłámſtwo obfitowáłá ku chwále jego / cżemuż jeƺcże y ja bywám ſądzony jáko grzeƺnik?
A nie rácżej ták mówimy : ( jáko nas ƺkálują / y jáko niektórzy udáwáją / żebyśmy mówili : ) Będźiemy cżynić złe rzecży / áby przyƺły dobre? Których potępienie jeſt ſpráwiedliwe.
Cóż tedy? Mámyż nád nie? Żádnym ſpoſobem ; gdyżeśmy przedtem dowiedli / iż Żydowie y Grekowie / wƺyſcy ſą pod grzechem /
Jáko nápiſáno : Niemáƺ ſpráwiedliwego áni jednego ;
Niemáƺ rozumnego / y niemáƺ / ktoby ƺukáł Bogá.
Wƺyſcy śię odchylili / weſpół śię ſtáli nieużytecżnymi / niemáƺ ktoby cżynił dobre / niemáƺ áż do jednego.
Grobem otworzonym jeſt gárdło ich / językámi ſwemi zdrádzáli / jád żmii pod wárgámi ich.
Których uſtá nápełnione ſą przeklinániá y gorzkośći ;
Nogi ich prędkie ſą ku wylewániu krwi ;
Skruƺenie z biedą ná drogách ich /
A drogi pokoju nie poználi ;
Niemáƺ bojáźni Bożej przed ocżymá ich.
A wiemy / iż cokolwiek zakon mówi / tym / którzy ſą pod zakonem / mówi / áby wƺelkie uſtá były zátulone / y áby wƺyſtek świát podlegáł kárániu Bożemu.
Przeto z ucżynków zakonu nie będźie uſpráwiedliwione żádne ćiáło przed oblicżnośćią jego / gdyż przez zakon jeſt poznánie grzechu.
Lecż teraz bez zakonu ſpráwiedliwość Bożá objáwioná jeſt / májącá świádectwo z zakonu y z proroków ;
Spráwiedliwość / mówię / Bożá przez wiárę JEzuſá CHryſtuſá ku wƺyſtkim y ná wƺyſtkie wierzące ; boć różnośći niemáƺ.
Abowiem wƺyſcy zgrzeƺyli / y nie doſtáje im chwáły Bożej.
A bywáją uſpráwiedliwieni dármo z łáſki jego przez odkupienie / które śię ſtáło w CHryſtuśie JEzuśie.
Którego Bóg wyſtáwił ubłágániem przez wiárę we krwi jego / ku okázániu ſpráwiedliwośći ſwojey przez odpuƺcżenie przedtem popełnionych grzechów w ćierpliwośći Bożej /
Ku okázániu ſpráwiedliwośći ſwojey w teráźniejƺym cżáśie / ná to / áby on był ſpráwiedliwym y uſpráwiedliwiájącym tego / który jeſt z wiáry JEzuſowej.
Gdźież tedy jeſt chlubá? Odrzuconá jeſt. Przez któryż zakon? Cżyli ucżynków? Nie / ále przez zakon wiáry.
Przetoż mámy zá to / że cżłowiek bywá uſpráwiedliwiony wiárą bez ucżynków zakonu.
Izáli Bóg jeſt tylko Bogiem Żydów? izáli też nie pogán? Záiſte y pogán.
Poniewáż jeden jeſt Bóg / który uſpráwiedliwi obrzezkę z wiáry y nieobrzezkę przez wiárę.
To tedy zakon niƺcżymy przez wiárę? Nie dáj tego Boże! y owƺem zakon ſtánowimy.
