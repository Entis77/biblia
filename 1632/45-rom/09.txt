Práwdę mówię w Chryſtuśie / á nie kłámię / w cżem mi poświádcżá ſumienie moje przez Duchá Świętego :
Że mám wielki ſmutek y nieuſtáwájący ból w ſercu mojem.
Abowiem żądáłbym ſám / ábym śię ſtáł odłącżonym od Chryſtuſá zá bráći moich / zá pokrewnych moich według ćiáłá.
Którzy ſą Izráelcżycy / których jeſt przyſpoſobienie ſynowſkie / y chwáłá / y przymierzá y zákonu dánie / y ſłużbá Bożá / y obietnice ;
Których ſą ojcowie / y z których poƺedł Chryſtuſ ile według ćiáłá / który jeſt nád wƺyſtkimi Bóg błogoſłáwiony ná wieki. Amen.
Lecż nie możná / żeby miáło upáść ſłowo Boże ; ábowiem nie wƺyſcy / którzy ſą z Izráelá / ſą Izráelem ;
Ani iż ſą náśieniem Abráhámowem / wƺyſcy ſą dziećmi ; ále rzecżono : W Izááku będzie tobie názwáne náśienie ;
To jeſt / nie dzieći ćiáłá ſą dziećmi Bożymi ; ále dzieći obietnicy bywáją w náśienie policżone.
Abowiem obietnicy ſłowo to jeſt : O tym włáśnie cżáśie przyjdę / á Sárá będzie miáłá ſyná ;
A nie tylko to / ále y Rebeká / gdy z jednego ojcá náƺego Izááká brzemienną zoſtáłá.
Gdy śię jeƺcże były dziátki nie národziły / áni co dobrego álbo złego ucżyniły / áby śię oſtáło poſtánowienie Boże według wybrániá / nie z ucżynków / ále z tego / który powołuje /
Rzecżono jej / że więkƺy będzie ſłużył mniejƺemu ;
Jáko nápiſáno : Jákóbám umiłowáł / álem Ezáwá miáł w nienáwiśći.
Cóż tedy rzecżemy? Jeſtże nieſpráwiedliwość u Bogá? Nie dáj tego Boże!
Abowiem do Mojżeƺá mówi : Zmiłuję śię / nád kim śię zmiłuję ; á zlituję śię / nád kim śię zlituję.
A przetoż nie záleży ná tym co chce / áni ná tym / co bieży / ále ná Bogu / który śię zmiłowywá.
Abowiem mówi Piſmo do Fáráoná : Ná tom ćię ſámo wzbudził / ábym okázáł moc moję ná tobie / á iżby opowiádáne było imię moje po wƺyſtkiej ziemi.
A ták nád kim chce / zmiłowywá śię / á kogo chce / zátwárdzá.
Ale mi rzecżeƺ : Przecżże śię jeƺcże uſkárżá? bo któż śię ſprzećiwił woli jego?
Y owƺem / o cżłowiecże! któżeś ty jeſt / który ſpór wiedzieƺ z Bogiem? Izáli lepiánká rzecże lepiárzowi : Przecżżeś mię ták ucżynił?
Izáli nie má mocy gárncárz nád gliną / żeby z tejże gliny ucżynił jedno nácżynie ku ucżćiwośći / á drugie ku zelżywośći?
A jeźliż Bóg chcąc okázáć gniew / y znájomą ucżynić możność ſwoję / znośił w wielkiej ćierpliwośći nácżyniá gniewu ná zginienie zgotowáne /
A iżby znájome ucżynił bogáctwo chwáły ſwojej nád nácżyniem miłośierdziá / które zgotowáł ku chwále ;
Których y powołáł / to jeſt náſ / nie tylko z Żydów / ále y z pogánów.
Jáko też u Ozeáƺá mówi : Názwię lud / który nie był moim / ludem moim / á onę / którá nie byłá umiłowáną / názwię umiłowáną.
Y ſtánie śię / że ná tem miejſcu / gdzie im máwiáno : Nie jeſteśćie wy ludem moim / tám názwáni będą ſynámi Bogá żywego.
A Izájáƺ wołá nád Izráelem / mówiąc : Choćby licżbá ſynów Izráelſkich byłá jáko piáſek morſki / oſtánki záchowáne będą.
Abowiem ſpráwę ſkońcży y ſkróći w ſpráwiedliwośći ; ſpráwę záiſte ſkróconą ucżyni Pán ná ziemi.
Y jáko przedtem powiedziáł Izájáƺ : By nám był Pán záſtępów nie zoſtáwił náśieniá / bylibyśmy śię ſtáli jáko Sodomá / y Gomorze bylibyśmy podobni.
Cóż tedy rzecżemy? To / iż pogánie / którzy nie ƺukáli ſpráwiedliwośći / doſtąpili ſpráwiedliwośći / á ſpráwiedliwośći / którá jeſt z wiáry.
A Izráel ƺukájąc zákonu ſpráwiedliwośći / nie doƺedł zákonu ſpráwiedliwośći.
Dlácżegoż? Iż nie z wiáry / ále jáko z ucżynków zákonu jej ƺukáli ; ábowiem śię obrázili o kámień obráżeniá /
Jáko nápiſáno : Oto kłádę w Syonie kámień obráżeniá / y opokę otrąceniá / á wƺelki / który weń wierzy / nie będzie poháńbiony.
