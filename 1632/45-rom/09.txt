Prawdę mówię w CHryſtuśie / <i>á</i> nie kłamam ; w cżym mi poświadcżá ſumnienie moje przez Duchá Świętego :
Że mam wielki ſmętek / y nieuſtawájący ból w ſercu mojim.
Abowiem żądam ja ſam ábym śię ſtał odłącżonym od CHryſtuſá zá bráćią moję / zá pokrewne moje według ćiáłá.
Którzy ſą Izráelcżycy / których <i>jeſt</i> przyſpoſobienie ſynowſkie / y chwałá / y przymierza / y zakonu dánie / y ſłużbá <i>Boża,</i> y obietnice.
Których <i>ſą</i> Ojcowie / y z których <i>poƺedł</i> CHryſtus ile według ćiáłá / który jeſt nád wƺyſtkimi Bóg błogoſłáwiony ná wieki. Amen.
Lecż nie można / żeby miáło upáść ſłowo Boże. Abowiem nie wƺyſcy którzy ſą z Izráelá / ſą Izráelem.
Ani iż ſą naśieniem Abráhámowym / wƺyſcy ſą dźiećmi : ále <i>rzecżono</i> ; W Izááku będźie tobie názwáne naśienie.
To jeſt nie dźieći ćiáłá / ſą dźiećmi Bożymi : ále dźieći obietnice / bywáją w naśienie policżone.
Abowiem obietnice ſłowo to <i>jeſt</i> : O tym właśnie cżáśie przydę / á Sará będźie miáłá ſyná.
A nie tylko <i>to,</i> ále y Rebeká gdy z jednego Ojcá náƺego Izááká brzemienną zoſtáłá :
Gdy śię jeƺcże były <i>dźiatki</i> nie národźiły / áni co dobrego álbo złego ucżyniły ; áby śię oſtáło poſtánowienie Boże według wybránia / nie z ucżynków / ále z tego który powoływa / Rzecżono jey.
Że więkƺy będźie ſłużył mniejƺemu ;
Jáko nápiſano : Jákóbám umiłował / álem Eſawá miał w nienawiśći.
Cóż tedy rzecżemy? Jeſtże nieſpráwiedliwość u Bogá? Nie daj tego Boże!
Abowiem do Mojzeƺá mówi : Zmiłuję śię / nád kim śię zmiłuję ; á zlituję śię / nád kim śię zlituję.
A przetoż nie <i>należy</i> ná tym co chce / áni ná tym co bieży ; ále ná Bogu który śię zmiłowywa.
Abowiem mówi piſmo do Fáráoná : Ná tom ćię ſámo wzbudźił / ábym okazał moc moję ná tobie ; á iżby opowiádáne było Imię moje po wƺyſtkiey źiemi.
A ták nád kim chce / zmiłowywa śię : á kogo chce zátwardza.
Ale mi rzecżeƺ? Przecżże śię jeƺcże uſkárża? bo któż śię zprzećiwił woli jego?
Y owƺem : O cżłowiecże! Któżes ty jeſt który zpór wiedźieƺ z Bogiem? Izali lepianká rzecże lepiarzowi? Przecżżeś mię ták ucżynił?
Izali nie ma mocy gárncarz <i>nád</i> gliną / żeby z tejże bryły ucżynił jedno nacżynie ku ucżćiwośći / á drugie ku zelżywośći?
A jeſliż Bóg chcąc okazáć gniew / y znájomą ucżynić możność ſwoję / znośił w wielkiey ćierpliwośći nacżynia gniewu ná zginienie zgotowáne :
A iżby znájome ucżynił bogáctwo chwały ſwojey nád nácżyniem miłośierdźia / które zgotował ku chwale :
Których y powołał ; <i>to jeſt,</i> nas : Nie tylko z Żydów / ále y z Pogánów :
Jáko też u Ozeaƺá mówi ; Názowię lud który nie był mojim / ludem mojim : á onę która nie byłá umiłowáną / <i>názwię</i> umiłowáną.
Y ſtánie śię / że ná tym miejſcu / gdźie im mawiano ; Nie <i>jeſteśćie</i> wy ludem mojim / tám názwáni będą ſynámi Bogá żywego.
A Izájaƺ woła nád Izráelem / <i>mówiąc</i> ; Choćby licżbá ſynów Izráelſkich byłá jáko piaſek morſki / oſtanki záchowáne będą.
Abowiem ſpráwę ſkońcży y ſkróći w ſpráwiedliwośći ; ſpráwę zájiſte ſkróconą ucżyni PAn ná źiemi.
Y jáko przed tym powiedźiał Izájaƺ : By nam był PAN Zaſtępów nie zoſtáwił naśienia / bylibyſmy śię ſtáli jáko Sodomá ; y Gomorze bylibyſmy podobni.
Cóż tedy rzecżemy? <i>To,</i> iż Pogánie / którzy nie ƺukáli ſpráwiedliwośći / doſtąpili ſpráwiedliwośći ; á ſpráwiedliwośći / która jeſt z wiáry.
A Izráel ƺukájąc Zakonu ſpráwiedliwośći / nie doƺedł zakonu ſpráwiedliwośći.
Dla cżegoż? Yż <i>nie</i> z wiáry / ále jáko z ucżynków zakonu <i>jey ƺukáli</i> : Abowiem śię obráźili o kámień obráżenia ;
Jáko nápiſano : Oto kłádę w Syonie kámień obráżenia / y opokę otrącenia : A wƺelki który weń wierzy / nie będźie poháńbiony.
