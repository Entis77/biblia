Mówię tedy : Izáli Bóg odrzućił lud ſwój? Nie dáj tego Boże! Abowiem y jám jeſt Izráelcżyk z náśieniá Abráhámowego / z pokoleniá Benjáminowego.
Nie odrzućiłći Bóg ludu ſwego / który przejrzáł. Azáż nie wiećie / co mówi Piſmo o Elijáƺu? jáko śię przycżyniá do Bogá przećiwko Izráelowi mówiąc :
Pánie! Proroki twoje pomordowáli / y ołtárze twoje zburzyli / á zoſtáłem já ſám / y ƺukáją duƺy mojej.
Ale cóż mu mówi Boſká odpowiedź? Zoſtáwiłem ſobie śiedm tyśięcy mężów / którzy nie ſkłonili koláná Báálowi.
Ták tedy y teráźniejƺego cżáſu oſtánki podług wybrániá z łáſki zoſtáły.
A poniewáż z łáſki / tedyć już nie z ucżynków / inácżej łáſká jużby nie byłá łáſką ; á jeźli z ucżynków / jużći nie jeſt łáſká ; inácżej ucżynek jużby nie był ucżynkiem.
Cóż tedy? Cżego Izráel ƺuká / tego nie doſtąpił ; ále wybráni doſtąpili / á inni zátwárdzeni ſą /
( Jáko nápiſáno : Dáł im Bóg duchá twárdego ſnu / ocży / áby nie widzieli / y uƺy / áby nie ſłyƺeli ) / áż do dziśiejƺego dniá.
A Dawid mówi : Niecháj im będzie ſtół ich śidłem y ułowieniem y otrąceniem y odpłátą.
Niech záćmione będą ocży ich / áby nie widzieli / á grzbietu ich záwƺe náchyláj.
Mówię tedy : Azáż śię potknęli / áby pádli? Nie dáj tego Boże! Ale przez ich upádek doƺło zbáwienie pogán / áby je do záwiśći przywiódł.
A poniewáż upádek ich jeſt bogáctwem świátá / á umniejƺenie ich bogáctwem pogán / jákoż dáleko więcej ich zupełność?
Abowiem mówię wám pogánom / ilem já jeſt Apoſtołem pogán / uſługiwánie moje zálecám /
Azáżbym jáko ku záwiśći pobudził ćiáło moje / y zbáwiłbym niektóre z nich.
Abowiem jeźlić odrzucenie ich jeſt pojednániem świátá / cóż będzie przyjęćie ich / tylko ożyćie od umárłych?
Poniewáż jeźli pierwiáſtki święte / tedyć y zácżynienie ; á jeźli korzeń święty / tedyć y gáłęzie.
A jeźli niektóre z gáłęzi odłámáne ſą / á ty / któryś był płonną oliwą / jeſteś wƺcżepiony zámiáſt nich / y ſtáłeś śię ucżeſnikiem korzeniá y tłuſtośći oliwnego drzewá :
Nie chlubże śię przećiw gáłęziom / bo jeźli śię chlubiƺ / wiedzże / iż nie ty korzeniá nośiƺ / ále korzeń ćiebie.
Ale rzecżeƺ : Odłámáne ſą gáłęzie / ábym já był wƺcżepiony.
Dobrze ; dlá niedowiárſtwá odłámáne ſą / ále ty wiárą ſtoiƺ ; nie bądźże hárdej myśli / ále śię bój.
Abowiem jeźli Bóg przyrodzonym gáłęziom nie przepuśćił / wiedz / że y tobie nie przepuśći.
Obácżże tedy dobrotliwość y ſrogość Bożą ; przećiwko tym wpráwdzie / którzy upádli / ſrogość / ále przećiwko tobie dobrotliwość / jeźlibyś trwáł w dobroći ; inácżej y ty będzieƺ wyćięty.
Aleć y oni jeźli nie będą trwáli w niedowiárſtwie / wƺcżepieni záś będą / gdyż mocny jeſt Bóg one záśię wƺcżepić.
Abowiem jeźliś ty jeſt wyćięty z oliwy / z przyrodzeniá płonnej / á przećiwko przyrodzeniu jeſteś wƺcżepiony w dobrą oliwę / jákoż dáleko więcej / którzy ſą według przyrodzeniá / wƺcżepieni będą w ſwoję włáſną oliwę!
Bo nie chcę / ábyśćie nie mieli wiedzieć / bráćiá! tej tájemnicy / ( żebyśćie nie byli ſámi u śiebie mądrymi / ) iż zátwárdzenie z cżęśći przyƺło ná Izráelá / pókiby nie weƺłá zupełność pogán.
A ták wƺyſtek Izráel będzie zbáwiony / jáko nápiſáno : Przyjdzie z Syonu wybáwićiel / y odwróći niepobożnośći od Jákóbá.
A toć będzie przymierze moje z nimi / gdy odejmę grzechy ich.
A ták według Ewángielii nieprzyjáćiółmi ſą dlá wáſ ; lecż według wybrániá ſą miłymi dlá ojców.
Abowiem dárów ſwoich y wezwániá Bóg nie żáłuje.
Bo jáko y wy niekiedy nie wierzyliśćie Bogu / ále teráz doſtąpiliśćie miłośierdziá dlá ich niedowiárſtwá.
Ták y oni teráz ſtáli śię niepoſłuƺnymi / áby dlá miłośierdziá wám okázánego y oni miłośierdziá doſtąpili.
Abowiem zámknął je Bóg wƺyſtkie w niedowiárſtwo / áby śię nád wƺyſtkimi zmiłowáł.
O głębokośći bogáctwá y mądrośći y znájomośći Bożej! Jáko ſą niewybádáne ſądy jego / y niedośćignione drogi jego!
Bo któż poznáł umyſł Páńſki? álbo kto był rájcą jego?
Albo kto mu co pierwej dáł / á będzie mu záśię oddáno?
Abowiem z niego y przezeń y w nim ſą wƺyſtkie rzecży ; jemu niech będzie chwáłá ná wieki. Amen.
