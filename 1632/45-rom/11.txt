Mówię tedy : Izáli Bóg odrzućił lud ſwój? Nie dáj tego Boże! Abowiem y jám jeſt Izráelcżyk z náśieniá Abráhámowego / z pokoleniá Benjáminowego.
Nie odrzućiłći Bóg ludu ſwego / który przejrzáł. Azáż nie wiećie / co mówi Piſmo o Elijáƺu? jáko śię przycżyniá do Bogá przećiwko Izráelowi mówiąc :
Pánie! Proroki twoje pomordowáli / y ołtárze twoje zburzyli / á zoſtáłem ja ſám / y ƺukáją duƺy mojey.
Ale cóż mu mówi Boſká odpowiedź? Zoſtáwiłem ſobie śiedm tyśięcy mężów / którzy nie ſkłonili koláná Báálowi.
Ták tedy y teráźniejƺego cżáſu oſtánki podług wybrániá z łáſki zoſtáły.
A poniewáż z łáſki / tedyć już nie z ucżynków / inácżej łáſká jużby nie byłá łáſką ; á jeſli z ucżynków / jużći nie jeſt łáſká ; inácżej ucżynek jużby nie był ucżynkiem.
Cóż tedy? Cżego Izráel ƺuká / tego nie doſtąpił ; ále wybráni doſtąpili / á inni zátwárdzeni ſą /
( Jáko nápiſáno : Dáł im Bóg duchá twárdego ſnu / ocży / áby nie widźieli / y uƺy / áby nie ſłyƺeli ) / áż do dźiśiejƺego dniá.
A Dawid mówi : Niecháj im będźie ſtół ich śidłem y ułowieniem y otrąceniem y odpłátą.
Niech záćmione będą ocży ich / áby nie widźieli / á grzbietu ich záwƺe náchyláj.
Mówię tedy : Azáż śię potknęli / áby pádli? Nie dáj tego Boże! Ale przez ich upádek doƺło zbáwienie pogán / áby je do záwiśći przywiódł.
A poniewáż upádek ich jeſt bogáctwem świátá / á umniejƺenie ich bogáctwem pogán / jákoż dáleko więcey ich zupełność?
Abowiem mówię wám pogánom / ilem ja jeſt Apoſtołem pogán / uſługiwánie moje zálecám /
Azáżbym jáko ku záwiśći pobudźił ćiáło moje / y zbáwiłbym niektóre z nich.
Abowiem jeſlić odrzucenie ich jeſt pojednániem świátá / cóż będźie przyjęćie ich / tylko ożyćie od umárłych?
Poniewáż jeſli pierwiáſtki święte / tedyć y zácżynienie ; á jeſli korzeń święty / tedyć y gáłęźie.
A jeſli niektóre z gáłęźi odłámáne ſą / á ty / któryś był płonną oliwą / jeſteś wƺcżepiony zámiáſt nich / y ſtáłeś śię ucżeſnikiem korzeniá y tłuſtośći oliwnego drzewá :
Nie chlubże śię przećiw gáłęźiom / bo jeſli śię chlubiƺ / wiedzże / iż nie ty korzeniá nośiƺ / ále korzeń ćiebie.
Ale rzecżeƺ : Odłámáne ſą gáłęźie / ábym ja był wƺcżepiony.
Dobrze ; dla niedowiárſtwá odłámáne ſą / ále ty wiárą ſtoiƺ ; nie bądźże hárdej myśli / ále śię bój.
Abowiem jeſli Bóg przyrodzonym gáłęźiom nie przepuśćił / wiedz / że y tobie nie przepuśći.
Obácżże tedy dobrotliwość y ſrogość Bożą ; przećiwko tym wpráwdźie / którzy upádli / ſrogość / ále przećiwko tobie dobrotliwość / jeſlibyś trwáł w dobroći ; inácżej y ty będźieƺ wyćięty.
Aleć y oni jeſli nie będą trwáli w niedowiárſtwie / wƺcżepieni záś będą / gdyż mocny jeſt Bóg one záśię wƺcżepić.
Abowiem jeſliś ty jeſt wyćięty z oliwy / z przyrodzeniá płonney / á przećiwko przyrodzeniu jeſteś wƺcżepiony w dobrą oliwę / jákoż dáleko więcey / którzy ſą według przyrodzeniá / wƺcżepieni będą w ſwoję włáſną oliwę!
Bo nie chcę / ábyśćie nie mieli wiedźieć / bráćia! tey tájemnicy / ( żebyśćie nie byli ſámi u śiebie mądrymi / ) iż zátwárdzenie z cżęśći przyƺło ná Izráelá / pókiby nie weƺłá zupełność pogán.
A ták wƺyſtek Izráel będźie zbáwiony / jáko nápiſáno : Przyjdźie z Syonu wybáwićiel / y odwróći niepobożnośći od Jákóbá.
A toć będźie przymierze moje z nimi / gdy odejmę grzechy ich.
A ták według Ewángielii nieprzyjáćiółmi ſą dla was ; lecż według wybrániá ſą miłymi dla ojców.
Abowiem dárów ſwoich y wezwániá Bóg nie żáłuje.
Bo jáko y wy niekiedy nie wierzyliśćie Bogu / ále teraz doſtąpiliśćie miłośierdźiá dla ich niedowiárſtwá.
Ták y oni teraz ſtáli śię niepoſłuƺnymi / áby dla miłośierdźiá wám okázánego y oni miłośierdźiá doſtąpili.
Abowiem zámknął je Bóg wƺyſtkie w niedowiárſtwo / áby śię nád wƺyſtkimi zmiłowáł.
O głębokośći bogáctwá y mądrośći y znájomośći Bożej! Jáko ſą niewybádáne ſądy jego / y niedośćignione drogi jego!
Bo któż poznał umyſł Páńſki? álbo kto był rájcą jego?
Albo kto mu co pierwej dáł / á będźie mu záśię oddáno?
Abowiem z niego y przezeń y w nim ſą wƺyſtkie rzecży ; jemu niech będźie chwałá ná wieki. Amen.
