Mówię tedy : Izali Bóg odrzućił lud ſwój? Nie daj tego Boże! Abowiem <i>y</i> jam jeſt Izráelcżyk z náśienia Abráhámowego / z pokolenia Benjáminowego.
Nie odrzućiłći Bóg ludu ſwego który przejrzał. Azaż nie wiećie co mówi piſmo o Eliaƺu / jáko śię przycżynia do Bogá przećiwko Izráelowi / mówiąc ;
PAnie / Proroki twoje pomordowáli / y ołtarze twoje zburzyli ; á zoſtałem ja ſam / y ƺukáją duƺe mojey.
Ale cóż mu mówi Boſka odpowiedź? Zoſtáwiłem ſobie śiedm tyśięcy mężów / którzy nie ſkłonili koláná Báálowi.
Ták tedy y teráźniejƺego cżáſu / oſtanki podług wybránia z łáſki zoſtáły.
A ponieważ z łáſki / tedyć już nie z ucżynków ; inácżey łáſká / jużby nie byłá łáſką : á jeſli z ucżynków / jużći nie jeſt łáſká ; inácżey ucżynek jużby nie był ucżynkiem.
Cóż tedy? Cżego Izráel ƺuka / tego nie doſtąpił ; ále wybráni doſtąpili / á inni zátwárdzeni ſą.
Jáko nápiſano : Dał im Bóg Duchá twárdego ſnu ; ocży áby nie widźieli / y uƺy áby nie ſłyƺeli / áż do dźiśiejƺego dniá.
A Dawid mówi ; Niechaj <i>im</i> będźie ſtół ich śidłem / y ułowieniem y otrąceniem / y odpłátą.
Niech záćmione będą ocży ich áby nie widźieli / á grzbietu ich záwżdy náchylaj.
Mówię tedy ; Azaż śię potknęli / áby pádli? Nie daj tego Boże! Ale przez ich upadek <i>doƺło</i> zbáwienie Pogánów / áby je do zawiśći przywiódł.
A ponieważ upadek ich jeſt bogáctwem świátá / á umniejƺenie ich bogáctwem Pogánów ; jákoż dáleko więcey ich zupełność.
Abowiem mówię wam Pogánom / ilem ja jeſt Apoſtołem Pogánów / uſługowánie moje zálecam :
Azażbym jáko ku zawiśći pobudźił ćiáło moje / y zbáwiłbym niektóre z nich.
Abowiem jeſlić odrzucenie ich jeſt pojednániem świátá / cóż <i>będźie</i> przyjęćie <i>ich,</i> tylko ożyćie od umárłych?
Ponieważ jeſli pierwiaſtki święte / <i>tedyć</i> y zácżynienie : á jeſli korzeń święty / <i>tedyć</i> y gáłęźie.
A jeſli niektóre z gáłęźi odłamáne ſą ; á ty któryś był płonną oliwą jeſteś wƺcżepiony miáſto nich / y ſtałeś śię ucżeſnikiem korzenia / y tłuſtośći oliwnego drzewá :
Nie chlubże śię przećiw gáłęźiam. Bo jeſli śię chlubiƺ / <i>wiedzże yż</i> nie ty korzenia nośiƺ / ále korzeń ćiebie.
Ale rzecżeƺ ; Odłamáne ſą gáłęźie / ábym ja był wƺcżepiony.
Dobrze : Dla niedowiárſtwá odłamáne ſą / ále ty wiárą ſtojiƺ ; Nie bądźże hárdey myśli / ále śię bój.
Abowiem jeſli Bóg przyrodzonym gáłęźiam nie przepuśćił / wiedz że y tobie nie przepuśći.
Obzcżże tedy dobrotliwość y ſrogość Bożą : przećiwko tym wprawdźie którzy upádli / ſrogość ; ále przećiwko tobie dobrotliwość / jeſlibyś trwał w dobroći : Inácżey y ty będźieƺ wyćięty.
Aleć y oni jeſli nie będą trwáli w niedowiárſtwie / wƺcżepieni <i>záś</i> będą : gdyż mocny jeſt Bóg one záśię wƺcżepić.
Abowiem jeſliś ty jeſt wyćięty z oliwy z przyrodzenia płonney / á przećiwko przyrodzeniu jeſteś wƺcżepiony w dobrą oliwę : jákoż dáleko więcey którzy ſą według przyrodzenia / wƺcżepieni będą w ſwoję właſną oliwę.
Bo nie chcę ábyśćie nie mieli wiedźieć bráćia tey tájemnice ( żebyśćie nie byli ſámi u śiebie mądrymi / ) iż zátwárdzenie z cżęśći przyƺło ná Izráelá / pókiby nie weƺłá zupełność Pogánów.
A ták wƺyſtek Izráel będźie zbáwiony / jáko nápiſano : Przydźie z Syonu Wybáwićiel / y odwróći niepobożnośći od Jákóbá.
A toć <i>będźie</i> przymierze moje z nimi ; gdy odejmę grzechy ich.
A ták według Ewángeliey / nieprzyjáćiółmi <i>ſą</i> dla was : lecż według wybránia ſą miłymi dla ojców.
Abowiem dárów <i>ſwojich</i> y wezwánia Bóg nie żáłuje.
Bo jáko y wy niekiedy byliście niepoſłuƺni Bogu / ále teraz doſtąpiliśćie miłośierdźia <i>dla</i> ich niepoſłuƺeństwá :
Ták y oni teraz ſtáli śię niepoſłuƺnymi / áby dla miłośierdźia wam <i>okázánego</i> y oni miłośierdźia doſtąpili.
Abowiem zámknął je Bóg wƺyſtkie w niepoſłuƺeńſtwo / áby śię nád wƺyſtkimi nimi zmiłował.
O głębokośći bogáctwá y mądrośći / y znájomośći Bożey! jáko <i>ſą</i> niewybadáne ſądy jego! y nie dośćignione drogi je°!
Bo któż poznał umyſł PAńſki? álbo kto był rajcą jego?
Albo kto mu <i>co</i> pierwey dał / á będźie mu záśię oddano?
Abowiem z niego / y przezeń / y w nim ſą wƺyſtkie rzecży : Jemu <i>niech będźie</i> chwałá ná wieki / Amen.
