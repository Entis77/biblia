Proƺę was tedy bráćia przez litośći Boże / ábyśćie ſtawiáli ćiáłá wáƺe ofiárą żywą / świętą / przyjemną Bogu / <i>to jeſt,</i> rozumną ſłużbę wáƺę.
A nie przypodobywajćie śię temu świátu ; ále śię przemieńćie przez odnowienie umyſłu wáƺe° : ná to / ábyśćie doświadcżyli która jeſt wola Boża dobra / przyjemna / y doſkonáła.
Abowiem powiádam / przez łáſkę która mi jeſt dána / káżdemu co jeſt miedzy wámi áby więcey <i>o ſobie</i> nie rozumiał / niżeli potrzebá rozumieć ; ále żeby <i>o ſobie</i> rozumiał ſkromnie / ták jáko komu Bóg udźielił miárę wiáry.
Abowiem jáko w jednym ćiele wiele cżłonków mamy / ále wƺyſtkie cżłonki nie jednoż dźieło máją :
Ták wiele nas jeſt jednem ćiáłem w CHryſtuśie / áleſmy z oſobná jedni drugich cżłonkámi.
Májąc tedy rózne dáry według łáſki która nam jeſt dána ; jeſli proroctwo / niech będźie według ƺnuru wiáry :
Jeſli poſługowánie / niech będźie w poſługowániu ; jeſli kto náucżá / niech <i>trwa</i> w náucżániu :
Jeſli kto nápomina / w nápominániu ; kto rozdawa / w ƺcżerośći ; kto przełożony jeſt / w pilnośći ; kto cżyni miłośierdźie / niech cżyni z ochotą.
Miłość <i>niech będźie</i> nie obłudna. Miejćie w obrzydliwośći złe / jimując śię dobrego.
Miłośćią bráterſką jedni ku drugim ſkłonni bądźćie : Ucżćiwośćią jedni drugich uprzedzájąc.
W pracy nie leniwi : Duchem pałájący : PAnu ſłużący.
W nádźiejey śię weſelący : W ućiſku ćierpliwi : W modlitwie uſtáwicżni.
Potrzebam świętych udźielájący : Gośćinnośći náśládujący.
Dobrorżecżćie tym / którzy was prześládują : Dobrorżecżćie / á nie przeklinajćie.
Weſelćie śię z weſelącymi ; á płácżćie z płácżącymi.
Bądźćie miedzy ſobą jednomyślni / wyſoko <i>o ſobie</i> nie rozumiejąc / ále śię do niſkich nákłaniájąc.
Nie bądźćie mądrymi ſámi u śiebie. Żadnemu złym zá złe nie oddawajćie : obmyślawájąc to co jeſt ucżćiwego przed wƺyſtkimi ludźmi.
Jeſli można / ile z was jeſt / ze wƺyſtkimi ludźmi pokój miejćie.
Nie mśćićie śię ſámi. Namilśi : ále dájćie miejſce gniewowi. Abowiem nápiſano ; Mnie pomſtá / á Ja oddam / mówi PAn.
Jeſli tedy łáknie nieprzyjaćiel twój / nákarm go ; jeſli prágnie / nápój go : bo to cżyniąc / węgle rozpalone zgárnieƺ ná głowę jego.
Nie dáj śię zwyćiężyć złemu / ále złe dobrym zwyćiężaj.
