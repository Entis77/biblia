Azáż nie wiećie / bráćiá! ( bo powiádomym zákonu mówię ) / iż zákon pánuje nád cżłowiekiem / póki żyje?
Abowiem niewiáſtá / którá jeſt zá mężem / póki żyw mąż / obowiązáná mu jeſt zákonem ; á jeźliby mąż umárł / uwolnioná jeſt od zákonu mężowego.
Przetoż tedy / póki mąż żyje / będzie zwáná cudzołożnicą / jeźliby żoną inƺego mężá zoſtáłá ; á jeźliby mąż jej umárł / wolná jeſt od zákonu onego / áby nie byłá cudzołożnicą / choćby śię inƺego mężá żoną ſtáłá.
A ták / bráćiá moi! y wyśćie umártwieni zákonowi przez ćiáło Chryſtuſowe / ábyśćie śię ſtáli inƺego / to jeſt / tego / który wzbudzony jeſt z mártwych / ábyśmy owoc przynośili Bogu.
Abowiem gdyśmy byli w ćiele / námiętnośći grzechów / które śię wzniecáły przez zákon / mocy dokázywáły w cżłonkách náƺych ku przynoƺeniu owocu śmierći.
Lecż teráz ſtáliśmy śię wolni od zákonu / umárłƺy temu / w którymeśmy byli zátrzymáni / ábyśmy Bogu ſłużyli w nowośći duchá / á nie w ſtárośći litery.
Cóż tedy rzecżemy? Iż zákon jeſt grzechem? Nie dáj tego Boże! Y owƺemem grzechu nie poznáł / tylko przez zákon ; bo y o pożądliwośćibym był nie wiedziáł / by był zákon nie rzekł : Nie będzieƺ pożądáł.
Lecż grzech wziąwƺy przycżynę przez przykázánie / ſpráwił we mnie wƺelką pożądliwość ; ábowiem bez zákonu grzech jeſt mártwy.
Y jám żył niekiedy bez zákonu ; lecż gdy przyƺło przykázánie / grzech ożył / á jám umárł.
Y ználázło śię / że to przykázánie / które miáło być ku żywotowi / jeſt mi ku śmierći.
Gdyż grzech / wziąwƺy przycżynę przez przykázánie / zwiódł mię / y przez nie zábił mię.
A ták zákon jeſt święty / y przykázánie święte y ſpráwiedliwe / y dobre.
To tedy dobre ſtáłoż mi śię śmierćią? Nie dáj tego Boże! Y owƺem grzech / áby śię pokázáł być grzechem / ſpráwił mi śmierć przez dobre / żeby śię ſtáł náder grzeƺącym on grzech przez ono przykázánie.
Bo wiemy / iż zákon jeſt duchowny / álem já ćieleſny / záprzedány pod grzech.
Abowiem tego / co cżynię / nie pochwálám ; bo nie / co chcę / to cżynię / ále cżego nienáwidzę / to cżynię.
A jeźli cżego nie chcę / to cżynię / przyzwálám zákonowi / że dobry jeſt.
Już tedy teráz nie já to cżynię / ále grzech we mnie mieƺkájący ;
Gdyż wiem / że nie mieƺká we mnie ( to jeſt / w ćiele mojem / ) dobre ; ábowiem chęć jeſt we mnie / ále wykonáć to / co jeſt dobrego / nie znájduję.
Bo nie cżynię dobrego / które chcę ; ále złe / którego nie chcę / to cżynię.
A jeźliż já to cżynię / cżego nie chcę / już já więcej nie cżynię tego / ále grzech / który we mnie mieƺká.
Znájduję tedy ten zákon w ſobie / gdy chcę dobre cżynić / że śię mnie złe trzymá.
Abowiem kochám śię w zákonie Bożym według wnętrznego cżłowieká.
Lecż widzę inƺy zákon w cżłonkách moich / odporny zákonowi umyſłu mego / y który mię zniewálá pod zákon grzechu / który jeſt w cżłonkách moich.
Nędznyż já cżłowiek! Któż mię wybáwi z tego ćiáłá śmierći?
Dziękuję Bogu przez Jezuſá Chryſtuſá / Páná náƺego. Przetoż tedy já ſám umyſłem ſłużę zákonowi Bożemu / lecż ćiáłem zákonowi grzechu.
