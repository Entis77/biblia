Azáż nie wiećie / bráćia! ( bo powiádomym zakonu mówię ) / iż zakon pánuje nád cżłowiekiem / póki żyje?
Abowiem niewiáſtá / która jeſt zá mężem / póki żyw mąż / obowiązáná mu jeſt zakonem ; á jeſliby mąż umárł / uwolnioná jeſt od zakonu mężowego.
Przetoż tedy / póki mąż żyje / będźie zwáná cudzołożnicą / jeſliby żoną inƺego mężá zoſtáłá ; á jeſliby mąż jey umárł / wolná jeſt od zakonu onego / áby nie byłá cudzołożnicą / choćby śię inƺego mężá żoną ſtáłá.
A ták / bráćia moi! y wyśćie umártwieni zakonowi przez ćiáło CHryſtuſowe / ábyśćie śię ſtáli inƺego / to jeſt / tego / który wzbudzony jeſt z martwych / ábyśmy owoc przynośili Bogu.
Abowiem gdyśmy byli w ćiele / námiętnośći grzechów / które śię wzniecáły przez zakon / mocy dokázywáły w cżłonkách náƺych ku przynoƺeniu owocu śmierći.
Lecż teraz ſtáliśmy śię wolni od zakonu / umárłƺy temu / w którymeśmy byli zátrzymáni / ábyśmy Bogu ſłużyli w nowośći duchá / á nie w ſtárośći litery.
Cóż tedy rzecżemy? Iż zakon jeſt grzechem? Nie dáj tego Boże! Y owƺemem grzechu nie poznał / tylko przez zakon ; bo y o pożądliwośćibym był nie wiedźiáł / by był zakon nie rzekł : Nie będźieƺ pożądáł.
Lecż grzech wźiąwƺy przycżynę przez przykázánie / ſpráwił we mnie wƺelką pożądliwość ; ábowiem bez zakonu grzech jeſt mártwy.
Y jám żył niekiedy bez zakonu ; lecż gdy przyƺło przykázánie / grzech ożył / á jám umárł.
Y ználázło śię / że to przykázánie / które miáło być ku żywotowi / jeſt mi ku śmierći.
Gdyż grzech / wźiąwƺy przycżynę przez przykázánie / zwiódł mię / y przez nie zábił mię.
A ták zakon jeſt święty / y przykázánie święte y ſpráwiedliwe / y dobre.
To tedy dobre ſtáłoż mi śię śmierćią? Nie dáj tego Boże! Y owƺem grzech / áby śię pokázáł być grzechem / ſpráwił mi śmierć przez dobre / żeby śię ſtáł náder grzeƺącym on grzech przez ono przykázánie.
Bo wiemy / iż zakon jeſt duchowny / álem ja ćieleſny / záprzedány pod grzech.
Abowiem tego / co cżynię / nie pochwálám ; bo nie / co chcę / to cżynię / ále cżego nienáwidzę / to cżynię.
A jeſli cżego nie chcę / to cżynię / przyzwálám zakonowi / że dobry jeſt.
Już tedy teraz nie ja to cżynię / ále grzech we mnie mieƺkájący ;
Gdyż wiem / że nie mieƺká we mnie ( to jeſt / w ćiele mojem / ) dobre ; ábowiem chęć jeſt we mnie / ále wykonáć to / co jeſt dobrego / nie znájduję.
Bo nie cżynię dobrego / które chcę ; ále złe / którego nie chcę / to cżynię.
A jeſliż ja to cżynię / cżego nie chcę / już ja więcey nie cżynię tego / ále grzech / który we mnie mieƺká.
Znájduję tedy ten zakon w ſobie / gdy chcę dobre cżynić / że śię mnie złe trzymá.
Abowiem kochám śię w zakonie Bożym według wnętrznego cżłowieká.
Lecż widzę inƺy zakon w cżłonkách moich / odporny zakonowi umyſłu mego / y który mię zniewálá pod zakon grzechu / który jeſt w cżłonkách moich.
Nędznyż ja cżłowiek! Któż mię wybáwi z tego ćiáłá śmierći?
Dźiękuję Bogu przez JEzuſá CHryſtuſá / Páná náƺego. Przetoż tedy ja ſám umyſłem ſłużę zakonowi Bożemu / lecż ćiáłem zakonowi grzechu.
