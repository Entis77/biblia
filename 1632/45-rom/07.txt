Azaż nie wiećie bráćia / ( bo powiádomym zakonu mówię ) iż Zakon pánuje nád cżłowiekiem póki żyje?
Abowiem niewiáſtá / która jeſt zá mężem / póki żyw mąż / obowiązána mu jeſt zakonem : á jeſliby mąż umárł / uwolniona jeſt od zakonu mężowego.
Przetoż tedy póki mąż żywie / będźie zwána cudzołożnicą / jeſliby żoną inƺego mężá zoſtáłá : á jeſliby mąż jey umárł / wolna jeſt od zakonu onego / áby nie byłá cudzołożnicą / choćby śię inƺego mężá żoną ſtáłá.
A ták bráćia moji / y wyśćie umartwieni zakonowi przez ćiáło CHryſtuſowe / ábyśćie śię ſtáli inƺego / <i>to jeſt</i> tego który wzbudzony jeſt zmartwych / ábyſmy owoc przynośili Bogu.
Abowiem gdyſmy byli w ćiele / namiętnośći grzechów które <i>śię wzniecáły</i> przez Zakon / mocy dokázowáły w cżłonkách náƺych ku przynoƺeniu owocu śmierći.
Lecż teraz ſtáliſmy śię wolni od zakonu / gdyż już umárł ten / przez któregoſmy byli zátrzymáni ; ábyſmy <i>Bogu</i> ſłużyli w nowośći duchá / á nie w ſtárośći litery.
Cóż tedy rzecżemy? Yż Zakon <i>jeſt</i> grzechem? Nie daj tego Boże! Y owƺemem grzechu nie poznał / tylko przez zakon : Bá y o pożądliwośćibym był nie wiedźiał / by był zakon nie rzekł ; Nie będźieƺ pożądał.
Lecż grzech wźiąwƺy przycżynę przez przykazánie / ſpráwił we mnie wƺeláką pożądliwość. Abowiem bez zakonu grzech jeſt martwy.
Y jam żył niekiedy bez zakonu. Lecż gdy przyƺło przykazánie / grzech ożył ; á jam umárł.
Y ználázło śię / że to przykazánie / które <i>miáło bydź</i> ku żywotowi / jeſt <i>mi</i> ku śmierći.
Gdyż grzech wźiąwƺy przycżynę przez ono przykazánie / zwiódł mię / y przez nie zábił <i>mię</i>.
A ták zakon <i>jeſt</i> święty / y ono przykazánie święte y ſpráwiedliwe / y dobre.
To tedy dobre / ſtáłoż mi śię śmierćią? Nie daj tego Boże! Y owƺem grzech / áby śię pokazał bydź grzechem / ſpráwił mi śmierć przez ono dobre : żeby śię ſtał náder grzeƺącym on grzech przez ono przykazánie.
Bo wiemy iż Zakon jeſt duchowny : álem ja ćieleſny / záprzedány pod grzech.
Abowiem <i>tego</i> co cżynię / nie pochwalam : bo nie co chcę / to cżynię ; ále cżego nienawidzę / to cżynię.
A jeſli cżego nie chcę / to cżynię ; przyzwalam Zakonowi że dobry <i>jeſt</i>.
Już tedy teraz nie ja to cżynię / ále grzech we mnie mieƺkájący.
Gdyż wiem / że nie mieƺka we mnie ( to jeſt / w ćiele mojim / ) dobre. Abowiem chęć jeſt we mnie ; ále wykonáć to co <i>jeſt</i> dobrego / nie znájduję.
Bo nie cżynię dobrego które chcę : ále złe którego nie chcę / to cżynię.
A jeſliż ja to cżynię / cżego nie chcę : <i>już</i> ja więcey nie cżynię tego / ále grzech który we mnie mieƺka.
Znájduję tedy ten Zakon <i>w ſobie</i> ; gdy chcę dobre cżynić / że śię mnie złe trzyma.
Abowiem kocham śię w Zakonie Bożym / według wnętrznego cżłowieká.
Lecż widzę inƺy zakon w cżłonkách mojich / odporny zakonowi umyſłu mego / y który mię zniewala pod Zakon grzechu / który jeſt w cżłonkách mojich.
Nędznyż ja cżłowiek! Któż mię wybáwi z tego ćiáłá śmierći?
Dźiękuję Bogu przez JEzuſá CHryſtuſá PAná náƺego. Przetoż tedy ja ſam umyſłem ſłużę zakonowi Bożemu / lecż ćiáłem zakonowi grzechu.
