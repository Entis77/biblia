Bráćia / przychylna wola ſercá mego / y modlitwá którą cżynię do Bogá zá Izráelem / jeſtći ku zbáwieniu.
Abowiem dawam im świádectwo / iż gorliwość ku Bogu máją ; ále nie według wiádomośći.
Bo nie znájąc ſpráwiedliwośći Bożey ; á chcąc właſną ſpráwiedliwość wyſtáwić / ſpráwiedliwośći Bożey nie byli poddáni.
Abowiem koniec zakonu jeſt CHryſtus / ku ſpráwiedliwośći káżdemu wierzącemu.
Gdyż Mojzeƺ piƺe o ſpráwiedliwośći która <i>jeſt</i> z Zakonu / iż ktobykolwiek te rzecży cżynił / przez nie żyć będźie.
Ale ſpráwiedliwość / która jeſt z wiáry / ták mówi : Nie mów w ſercu ſwym / Kto wſtąpi ná niebo? to jeſt / CHryſtuſá ná dół zprowádźić.
Albo kto zſtąpi do przepáśći? to jeſt / CHryſtuſá od umárłych wyprowádźić.
Ale cóż mówi? Bliſko ćiebie jeſt ſłowo w uśćiech twojich / y w ſercu twojim : Toć jeſt ſłowo wiáry / które opowiádamy :
Że jeſlibyś uſty wyznał PAná JEzuſá / y uwierzyłbyś w ſercu twojim że go Bóg zmartwych wzbudźił / zbáwiony będźieƺ.
Abowiem ſercem wierzono bywa ku ſpráwiedliwośći / ále śię uſty wyznánie dźieje ku zbáwieniu.
Bo piſmo mówi : Wƺelki / kto weń wierzy / nie będźie poháńbiony.
Gdyż niemáƺ różnośći między Żydem y Grekiem. Bo tenże PAN wƺyſtkich / bogáty jeſt ku wƺyſtkim / którzy go wzywáją.
Káżdy bowiem ktoby wzywał Imienia PAńſkiego / zbáwiony będźie.
Jákoż tedy wzywáć będą <i>tego</i> w którego nie uwierzyli? A jáko uwierzą <i>w tego</i> o którym nie ſłyƺeli? A jáko uſłyƺą bez káznodźieje?
Jákoż też będą kazáć / jeſliby nie byli poſłáni? ( Jáko nápiſáno : O jáko ślicżne <i>ſą</i> nogi tych / którzy opowiádáją pokój ; tych którzy opowiádáją dobre rzecży.
Aleć nie wƺyſcy poſłuƺni byli Ewángeliey. Abowiem Izájaƺ mówi ; PAnie / Któż uwierzył kazániu náƺemu.
Wiárá tedy jeſt z ſłuchánia / á ſłuchánie przez ſłowo Boże.
Ale mówię ; Izali nie ſłyƺeli? y owƺem ná wƺyſtkę źiemię wyƺedł głos ich / y ná końcżyny okręgu źiemie ſłowá ich.
Ale mówię : Izali tego nie poznał Izráel? Pierwƺy Mojzeƺ mówi ; Ja was do zazdrości pobudzę / przez <i>naród</i> który nie jeſt narodem ; przez naród nierozumny rozdraźnię was.
A Izájaƺ śmiáłośći używájąc mówi / Jeſtem ználeźiony <i>od</i> tych / którzy mię nie ƺukáli ; y jeſtem objáwiony tym którzy śię o mnie nie pytáli.
Ale przećiwko Izráelowi mówi ; Cáły dźień wyćiągałem ręce moje do ludu upornego y zprzećiwiájącego śię.
