Bráćiá! przychylná wolá ſercá mego y modlitwá / którą cżynię do Bogá zá Izráelem / jeſtći ku zbáwieniu.
Abowiem dáję im świádectwo / iż gorliwość ku Bogu máją / ále nie według wiádomośći.
Bo nie znájąc ſpráwiedliwośći Bożej / á chcąc włáſną ſpráwiedliwość wyſtáwić / ſpráwiedliwośći Bożej nie byli poddáni.
Abowiem koniec zákonu jeſt Chryſtuſ ku ſpráwiedliwośći káżdemu wierzącemu.
Gdyż Mojżeƺ piƺe o ſpráwiedliwośći / którá jeſt z zákonu / iż ktobykolwiek te rzecży cżynił / przez nie żyć będzie.
Ale ſpráwiedliwość / którá jeſt z wiáry / ták mówi : Nie mów w ſercu ſwem : Kto wſtąpi ná niebo? to jeſt Chryſtuſá ná dół ſprowádzić :
Albo kto zſtąpi do przepáśći? to jeſt Chryſtuſá od umárłych wyprowádzić.
Ale ( Mojżeƺ ) cóż mówi : Bliſko ćiebie jeſt ſłowo w uſtách twoich y w ſercu twojim. Toć jeſt ſłowo wiáry / które opowiádámy :
Że jeźlibyś uſty wyznáł Páná Jezuſá / y uwierzyłbyś w ſercu twojim / że go Bóg z mártwych wzbudził / zbáwiony będzieƺ.
Abowiem ſercem wierzono bywá ku ſpráwiedliwośći / ále śię uſty wyznánie dzieje ku zbáwieniu.
Bo Piſmo mówi : Wƺelki / kto weń wierzy / nie będzie poháńbiony ;
Gdyż niemáƺ różnośći między Żydem y Grekiem ; bo tenże Pán wƺyſtkich / bogáty jeſt ku wƺyſtkim / którzy go wzywáją.
Káżdy bowiem / ktoby wzywáł imieniá Páńſkiego / zbáwiony będzie.
Jákoż tedy wzywáć będą tego / w którego nie uwierzyli? á jáko uwierzą w tego / o którym nie ſłyƺeli? á jáko uſłyƺą bez káznodziei?
Jákoż też będą kázáć / jeźliby nie byli poſłáni? Jáko nápiſáno : O jáko ślicżne ſą nogi tych / którzy opowiádáją pokój / tych / którzy opowiádáją dobre rzecży.
Aleć nie wƺyſcy poſłuƺni byli Ewángielii ; ábowiem Izájáƺ mówi : Pánie! któż uwierzył kázániu náƺemu?
Wiárá tedy jeſt z ſłuchániá / á ſłuchánie przez ſłowo Boże.
Ale mówię : Izáli nie ſłyƺeli? y owƺem ná wƺyſtkę ziemię wyƺedł głoſ ich / y ná końcżyny okręgu ziemi ſłowá ich.
Ale mówię : Izáli tego nie poznáł Izráel? Pierwƺy Mojżeƺ mówi : Já wáſ do záwiśći pobudzę przez náród / który nie jeſt národem / przez náród nierozumny rozdráźnię wáſ.
A Izájáƺ śmiáłośći używájąc mówi : Jeſtem ználeziony od tych / którzy mnie nie ƺukáli / y jeſtem objáwiony tym / którzy śię o mnie nie pytáli.
Ale przećiwko Izráelowi mówi : Cáły dzień wyćiągáłem ręce moje do ludu upornego y ſprzećiwiájącego śię.
