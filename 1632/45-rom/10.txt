Bráćia! przychylná wolá ſercá mego y modlitwá / którą cżynię do Bogá zá Izráelem / jeſtći ku zbáwieniu.
Abowiem dáję im świádectwo / iż gorliwość ku Bogu máją / ále nie według wiádomośći.
Bo nie znájąc ſpráwiedliwośći Bożej / á chcąc włáſną ſpráwiedliwość wyſtáwić / ſpráwiedliwośći Bożej nie byli poddáni.
Abowiem koniec zakonu jeſt CHryſtuſ ku ſpráwiedliwośći káżdemu wierzącemu.
Gdyż Mojżeƺ piƺe o ſpráwiedliwośći / która jeſt z zakonu / iż ktobykolwiek te rzecży cżynił / przez nie żyć będźie.
Ale ſpráwiedliwość / która jeſt z wiáry / ták mówi : Nie mów w ſercu ſwem : Kto wſtąpi ná niebo? to jeſt CHryſtuſá ná dół ſprowádźić :
Albo kto zſtąpi do przepáśći? to jeſt CHryſtuſá od umárłych wyprowádźić.
Ale ( Mojżeƺ ) cóż mówi : Bliſko ćiebie jeſt ſłowo w uſtách twoich y w ſercu twojim. Toć jeſt ſłowo wiáry / które opowiádámy :
Że jeſlibyś uſty wyznáł Páná JEzuſá / y uwierzyłbyś w ſercu twojim / że go Bóg z martwych wzbudźił / zbáwiony będźieƺ.
Abowiem ſercem wierzono bywá ku ſpráwiedliwośći / ále śię uſty wyznánie dźieje ku zbáwieniu.
Bo Piſmo mówi : Wƺelki / kto weń wierzy / nie będźie poháńbiony ;
Gdyż niemáƺ różnośći między Żydem y Grekiem ; bo tenże Pán wƺyſtkich / bogáty jeſt ku wƺyſtkim / którzy go wzywáją.
Káżdy bowiem / ktoby wzywáł imieniá Páńſkiego / zbáwiony będźie.
Jákoż tedy wzywáć będą tego / w którego nie uwierzyli? á jáko uwierzą w tego / o którym nie ſłyƺeli? á jáko uſłyƺą bez káznodźiei?
Jákoż też będą kázáć / jeſliby nie byli poſłáni? Jáko nápiſáno : O jáko ślicżne ſą nogi tych / którzy opowiádáją pokój / tych / którzy opowiádáją dobre rzecży.
Aleć nie wƺyſcy poſłuƺni byli Ewángielii ; ábowiem Izájáƺ mówi : Pánie! któż uwierzył kázániu náƺemu?
Wiárá tedy jeſt z ſłuchániá / á ſłuchánie przez ſłowo Boże.
Ale mówię : Izáli nie ſłyƺeli? y owƺem ná wƺyſtkę źiemię wyƺedł głoſ ich / y ná końcżyny okręgu źiemi ſłowá ich.
Ale mówię : Izáli tego nie poznał Izráel? Pierwƺy Mojżeƺ mówi : Ja was do záwiśći pobudzę przez náród / który nie jeſt národem / przez náród nierozumny rozdráźnię was.
A Izájáƺ śmiáłośći używájąc mówi : Jeſtem ználeźiony od tych / którzy mnie nie ƺukáli / y jeſtem objáwiony tym / którzy śię o mnie nie pytáli.
Ale przećiwko Izráelowi mówi : Cáły dźień wyćiągáłem ręce moje do ludu upornego y ſprzećiwiájącego śię.
