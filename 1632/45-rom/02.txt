Przetoż jeſteś bez wymówki / o cżłowiecże! który oſądzáƺ ; bo w cżem drugiego oſądzáƺ / ſámego śiebie oſądzáƺ / poniewáż toż cżyniƺ / który drugiego oſądzáƺ.
Lecż wiemy / iż ſąd Boży jeſt według práwdy przećiwko tym / którzy tákowe rzecży cżynią.
Cży mniemáƺ / o cżłowiecże! który oſądzáƺ tych / co tákowe rzecży cżynią / á ſám je cżyniƺ / że ty ujdźieƺ ſądu Bożego?
Cży bogáctwy dobrotliwośći jego / y ćierpliwośći / y nieſkwápliwośći pogárdzáƺ / nie wiedząc / iż ćię dobrotliwość Bożá do pokuty prowádźi?
Ale podług zátwárdźiáłośći twojey y ſercá niepokutującego ſkárbiƺ ſobie ſámemu gniew ná dźień gniewu y objáwieniá ſpráwiedliwego ſądu Bożego.
Który oddá káżdemu podług ucżynków jego ;
Tym / którzy przez wytrwánie w ucżynku dobrym ƺukáją ſłáwy y cżći y nieſkáźitelnośći / oddá żywot wiecżny ;
A záś ſwárliwym y práwdźie niepoſłuƺnym / lecż poſłuƺnym nieſpráwiedliwośći / oddá zápálcżywość y gniew ;
Utrápienie y ućiſk duƺy káżdego cżłowieká / który złość popełniá / Żydá nájprzód / potem y Greká ;
A chwáłę y cżeść y pokój wƺelkiemu cżyniącemu dobre / Żydowi nájprzód / potem y Grekowi.
Abowiem niemáƺ względu ná oſoby u Bogá.
A którzykolwiek bez zakonu zgrzeƺyli / bez zakonu też poginą ; á którzykolwiek w zakonie zgrzeƺyli / przez zakon ſądzeni będą.
( Gdyż nie ſłuchácże zakonu ſpráwiedliwymi ſą u Bogá ; ále cżynićiele zakonu uſpráwiedliwieni będą. )
Bo poniewáż pogánie nie májący zakonu z przyrodzeniá cżynią / co jeſt w zakonie / ći / zakonu nie májąc / ſámi ſobie ſą zakonem ;
Którzy ukázują ſkutek zakonu / nápiſány ná ſercách ſwych / z poświádcżániem ſumieniá ich y myśli weſpół śiebie oſkárżájących álbo też wymáwiájących /
W dźień / gdy ſądźić będźie Bóg ſkryte rzecży ludzkie według Ewángielii mojey przez JEzuſá CHryſtuſá.
Oto śię ty názywáƺ Żydem / y polegáƺ ná zakonie / á chlubiƺ śię Bogiem.
Y znáƺ wolę jego / y rozeznájeƺ rzecży różne od niej / wyćwicżony będąc z zakonu ;
Y máƺ zá to / żeś jeſt wodzem ślepych / świátłośćią tych / którzy ſą w ćiemnośći ;
Miſtrzem bezrozumnych / náucżyćielem niemowlątek / májąc kƺtáłt znájomośći y práwdy w zakonie.
Który tedy ucżyƺ drugiego / śiebie ſámego nie ucżyƺ? Który opowiádáƺ / żeby nie krádźiono / krádnieƺ?
Który mówiƺ / żeby nie cudzołożono / cudzołożyƺ? który śię brzydźiƺ báłwány / święte rzecży krádnieƺ?
Który śię chlubiƺ zakonem / przez przeſtępſtwo zakonu Bogá lżyƺ?
Abowiem imię Boże dla was bluźnione bywá między pogány / jáko nápiſáno.
Boć obrzezánie jeſt pożytecżne / jeſlibyś pełnił zakon : ále jeſlibyś był przeſtępcą zakonu / twoje obrzezánie ſtáło śię nieobrzezką.
Jeſliby tedy nieobrzezká przeſtrzegáłá práw zakonnych / ázáż jego nieobrzezká nie będźie pocżytáná zá obrzezkę?
Y oſądźi nieobrzezká z przyrodzeniá zakon pełniącá / ćiebie / który przez literę y obrzezkę jeſteś przeſtępcą zakonu.
Abowiem nie ten jeſt Żydem / który jeſt Żydem ná jáwie / áni to jeſt obrzezká / która jeſt ná jáwie ná ćiele ;
Ale który jeſt w ſkrytośći Żydem / y obrzezká ſercá / która jeſt w duchu / nie w literze / którey chwałá nie jeſt z ludźi / ále z Bogá.
