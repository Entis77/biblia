Przetoż jeſteś bez wymówki o cżłowiecże / który oſądzaƺ. Bo w cżym drugiego oſądzaƺ / ſámego śiebie oſądzaƺ : ponieważ toż cżyniƺ / który drugiego oſądzaƺ.
Lecż wiemy iż ſąd Boży jeſt według prawdy przećiwko tym / którzy tákowe <i>rzecży</i> cżynią.
Cży mniemaƺ o cżłowiecże / który oſądzaƺ tych co tákowe <i>rzecży</i> cżynią / á ſam je cżyniƺ / że ty ujdźieƺ ſądu Bożego?
Cży bogáctwy dobrotliwośći jego / y ćierpliwośći / y nieſkwápliwośći pogardzaƺ? Nie wiedząc iż ćię dobrotliwość Boża do pokuty prowádźi?
Ale podług zátwárdźiáłośći twojey / y ſercá nie pokutującego / ſkárbiƺ ſobie ſámemu gniew ná dźień gniewu / y objáwienia ſpráwiedliwego ſądu Bożego.
Który odda káżdemu podług ucżynków jego.
Tym którzy przez wytrwánie w ucżynku dobrym / ƺukáją ſławy / y cżći / y nieſkáźitelnośći / <i>odda</i> żywot wiecżny.
A záś ſwarliwym / y prawdźie niepoſłuƺnym / lecż poſłuƺnym nieſpráwiedliwośći / <i>odda</i> zápálcżywość y gniew :
Utrapienie y ućiſk / duƺy káżdego cżłowieká który złość popełnia ; Żydá naprzód / potym y Greká.
A chwałę y cżęść y pokój / wƺelkiemu cżyniącemu dobre / Żydowi naprzód / potym Grekowi.
Abowiem niemáƺ względu ná oſoby u Bogá.
Ale którzykolwiek bez zakonu zgrzeƺyli / bez zakonu też poginą : á którzykolwiek w zakonie grzeƺyli / przez zakon ſądzeni będą.
( Gdyż nie ſłuchácże Zakonu ſpráwiedliwymi <i>ſą</i> u Bogá ; ále cżynićiele zakonu uſpráwiedliwieni będą. )
Bo ponieważ Pogánie nie májący zakonu / z przyrodzenia cżynią co jeſt w zakonie ; ći zakonu nie májąc / ſámi ſobie ſą zakonem.
Którzy ukázują ſkutek Zakonu nápiſány ná ſercách ſwych / z poświadcżániem ſumnienia ich / y myśli weſpół śiebie oſkarżájących / álbo też wymawiájących.
W dźień gdy ſądźić będźie Bóg ſkryte rzecży ludzkie / według Ewángeliey mojey / przez JEzuſá CHryſtuſá.
Oto śię ty názywaƺ Żydem / y polegaƺ ná Zakonie / á chlubiƺ śię Bogiem :
Y znaƺ wolą <i>jego,</i> y rozeznawaƺ <i>rzecży</i> rózne <i>od niey,</i> wyćwicżony będąc z Zakonu :
Y maƺ zá to żeś jeſt wodzem ślepych ; świátłośćią tych którzy ſą w ćiemnośći :
Miſtrzem bezrozumnych ; náucżyćielem niemowiątek ; májąc kƺtałt znájomośći y prawdy w Zakonie.
Który tedy ucżyƺ drugiego / śiebie ſámego nie ucżyƺ? Który opowiádaƺ żeby nie krádźiono / krádnieƺ?
Który mówiƺ / żeby nie cudzołożono / cudzołożyƺ? Który śię brzydźiƺ báłwany / święte rzecży krádnieƺ?
Który śię chlubiƺ Zakonem / przez przeſtępſtwo zakonu Bogá lżyƺ?
Abowiem Imię Boże dla was bluźnione bywa miedzy Pogány / jáko nápiſano.
Boć obrzezánie jeſt pożytecżne / jeſlibyś pełnił Zakon : ále jeſlibyś był przeſtępcą Zakonu / twoje obrzezánie ſtáło śię nieobrzezką.
Jeſliby tedy nieobrzezká przeſtrzegáłá praw zakonnych / ázaż jego nieobrzezká nie będźie pocżytána zá obrzezkę?
Y oſądźi nieobrzezká z przyrodzenia zakon pełniąca / ćiebie / który przez literę y obrzezkę jeſteś przeſtępcą Zakonu.
Abowiem nie ten jeſt Żydem / który jeſt <i>Żydem</i> ná jáwie ; áni to jeſt obrzezká / która jeſt ná jáwie ná ćiele :
Ale który jeſt w ſkrytośći Żydem ; y obrzezká ſercá <i>która jeſt</i> w duchu / nie w literze : którey chwałá nie <i>jeſt</i> z ludźi / ále z Bogá.
