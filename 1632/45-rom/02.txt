Przetoż jeſteś bez wymówki / o cżłowiecże! który oſądzáƺ ; bo w cżem drugiego oſądzáƺ / ſámego śiebie oſądzáƺ / poniewáż toż cżyniƺ / który drugiego oſądzáƺ.
Lecż wiemy / iż ſąd Boży jeſt według práwdy przećiwko tym / którzy tákowe rzecży cżynią.
Cży mniemáƺ / o cżłowiecże! który oſądzáƺ tych / co tákowe rzecży cżynią / á ſám je cżyniƺ / że ty ujdźieƺ ſądu Bożego?
Cży bogáctwy dobrotliwośći jego / y ćierpliwośći / y nieſkwápliwośći pogárdzáƺ / nie wiedząc / iż ćię dobrotliwość Bożá do pokuty prowádźi?
Ale podług zátwárdźiáłośći twojej y ſercá niepokutującego ſkárbiƺ ſobie ſámemu gniew ná dźień gniewu y objáwieniá ſpráwiedliwego ſądu Bożego.
Który oddá káżdemu podług ucżynków jego ;
Tym / którzy przez wytrwánie w ucżynku dobrym ƺukáją ſłáwy y cżći y nieſkáźitelnośći / oddá żywot wiecżny ;
A záś ſwárliwym y práwdźie niepoſłuƺnym / lecż poſłuƺnym nieſpráwiedliwośći / oddá zápálcżywość y gniew ;
Utrápienie y ućiſk duƺy káżdego cżłowieká / który złość popełniá / Żydá nájprzód / potem y Greká ;
A chwáłę y cżeść y pokój wƺelkiemu cżyniącemu dobre / Żydowi nájprzód / potem y Grekowi.
Abowiem niemáƺ względu ná oſoby u Bogá.
A którzykolwiek bez zákonu zgrzeƺyli / bez zákonu też poginą ; á którzykolwiek w zákonie zgrzeƺyli / przez zákon ſądzeni będą.
( Gdyż nie ſłuchácże zákonu ſpráwiedliwymi ſą u Bogá ; ále cżynićiele zákonu uſpráwiedliwieni będą. )
Bo poniewáż pogánie nie májący zákonu z przyrodzeniá cżynią / co jeſt w zákonie / ći / zákonu nie májąc / ſámi ſobie ſą zákonem ;
Którzy ukázują ſkutek zákonu / nápiſány ná ſercách ſwych / z poświádcżániem ſumieniá ich y myśli weſpół śiebie oſkárżájących álbo też wymáwiájących /
W dźień / gdy ſądźić będźie Bóg ſkryte rzecży ludzkie według Ewángielii mojej przez JEzuſá CHryſtuſá.
Oto śię ty názywáƺ Żydem / y polegáƺ ná zákonie / á chlubiƺ śię Bogiem.
Y znáƺ wolę jego / y rozeznájeƺ rzecży różne od niej / wyćwicżony będąc z zákonu ;
Y máƺ zá to / żeś jeſt wodzem ślepych / świátłośćią tych / którzy ſą w ćiemnośći ;
Miſtrzem bezrozumnych / náucżyćielem niemowlątek / májąc kƺtáłt znájomośći y práwdy w zákonie.
Który tedy ucżyƺ drugiego / śiebie ſámego nie ucżyƺ? Który opowiádáƺ / żeby nie krádźiono / krádnieƺ?
Który mówiƺ / żeby nie cudzołożono / cudzołożyƺ? który śię brzydźiƺ báłwány / święte rzecży krádnieƺ?
Który śię chlubiƺ zákonem / przez przeſtępſtwo zákonu Bogá lżyƺ?
Abowiem imię Boże dlá wáſ bluźnione bywá między pogány / jáko nápiſáno.
Boć obrzezánie jeſt pożytecżne / jeźlibyś pełnił zákon : ále jeźlibyś był przeſtępcą zákonu / twoje obrzezánie ſtáło śię nieobrzezką.
Jeźliby tedy nieobrzezká przeſtrzegáłá práw zákonnych / ázáż jego nieobrzezká nie będźie pocżytáná zá obrzezkę?
Y oſądźi nieobrzezká z przyrodzeniá zákon pełniącá / ćiebie / który przez literę y obrzezkę jeſteś przeſtępcą zákonu.
Abowiem nie ten jeſt Żydem / który jeſt Żydem ná jáwie / áni to jeſt obrzezká / którá jeſt ná jáwie ná ćiele ;
Ale który jeſt w ſkrytośći Żydem / y obrzezká ſercá / którá jeſt w duchu / nie w literze / której chwáłá nie jeſt z ludźi / ále z Bogá.
