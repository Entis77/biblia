Cóż tedy / rzecżemy / ználazł Abráhám Oćiec náƺ według ćiáłá?
Bo jeſli Abráhám z ucżynków jeſt uſpráwiedliwiony ; ma śię cżym chlubić / ále nie u Bogá.
Abowiem cóż piſmo mówi? Uwierzył Abráhám Bogu / y przycżytano mu to zá ſpráwiedliwość.
A robiącemu zapłátá nie bywa przycżytána podług łáſki / ále podług długu.
Nie robiącemu záś / lecż wierzącemu w tego który uſpráwiedliwia niepobożnego / przycżytána bywa wiárá jego zá ſpráwiedliwość.
Jáko y Dawid powiáda / Że błogoſłáwieńſtwo cżłowieká jeſt / któremu Bóg przycżytá ſpráwiedliwość bez ucżynków / <i>mówiąc :</i>
Błogoſłáwieni których odpuƺcżone <i>ſą</i> niepráwośći / á których zákryte ſą grzechy.
Błogoſłáwiony mąż / któremu PAn grzechu nie przycżyta.
To tedy błogoſłáwieńſtwo tylko ná obrzezkę <i>przychodźi,</i> cży też ná nieobrzezkę? Gdyż mówimy / Iż wiárá Abráhámowi jeſt przycżytána zá ſpráwiedliwość.
Jákoż mu tedy jeſt przycżytána? Gdy był w obrzezce ? Cżyli w nieobrzezce? Nie w obrzezce ; ále w nieobrzezce.
Y przyjął znák obrzezki zá piecżęć ſpráwiedliwośći oney wiáry / która byłá w nieobrzezce : Ná to / áby był Ojcem wƺyſtkich wierzących w nieobrzezce / áby y onym przycżytána byłá ſpráwiedliwość :
Y <i>áby był</i> Ojcem obrzezki / nie tylko tych którzy ſą z obrzezki / ále też y tych którzy chodzą ſtopámi wiáry Ojcá náƺego Abráhámá / która byłá w nieobrzezce.
Abowiem nie przez zakon <i>śię ſtáłá</i> obietnicá Abráhámowi álbo náśieniu jego / áby był dźiedźicem świátá ; ále przez ſpráwiedliwość wiáry.
Bo jeſli ći którzy <i>ſą</i> z zakonu / dźiedźicámi <i>ſą</i> ; <i>tedyć</i> zniƺcżáłá wiárá / y wniwecż śię obróćiłá obietnicá.
Gdyż zakon gniew ſpráwuje. Abowiem gdźie zakonu niemáƺ / <i>tám</i> áni przeſtępſtwá.
A przetoż z wiáry <i>jeſt dźiedźictwo,</i> áby było z łáſki / <i>y</i> żeby byłá wárowna obietnicá wƺyſtkiemu náśieniu : nie tylko temu które jeſt z zakonu / ále y temu <i>które jeſt</i> z wiáry Abráhámowey / który jeſt ojcem nas wƺyſtkich /
( Jáko nápiſano : Ojcem wiela narodów wyſtáwiłem ćię ) przed Bogiem / któremu uwierżył / który ożywia umárłe ; y który przywoływa tych <i>rzecży</i> których niemáƺ / jákoby były.
Który <i>Abráhám</i> przećiwko nádźiejey w nádźieję uwierzył / że śię ſtánie ojcem wiela narodów / według tego <i>co mu</i> powiedźiano : Ták będźie naśienie twoje.
A nie będąc ſłábym w wierze / nie pátrzał ná ćiáło ſwoje już obumárłe májąc około ſtá lat / áni ná obumárły żywot Sary.
O obietnicy tedy Bożey nie wątpił z niedowiárſtwá ále śię umocnił wiárą / y dał chwałę Bogu.
Będąc też tego pewien / że cokolwiek <i>on</i> obiecał / mocen jeſt y ucżynić.
A przetoż przycżytano mu to zá ſpráwiedliwość.
A nie nápiſano <i>tego</i> dla niego ſámego / iż mu <i>to</i> przycżytano :
Ale y dla nas / którym ma być przycżytano ; którzy wierzymy w tego / który wzbudźił JEzuſá PAná náƺego z martwych.
Który wydány jeſt dla grzechów náƺych / á wſtał zmartwych dla uſpráwiedliwienia náƺego.
