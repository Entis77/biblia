Cóż tedy / rzecżemy / ználázł Abráhám / ojćiec náƺ / według ćiáłá?
Bo jeźli Abráhám z ucżynków jeſt uſpráwiedliwiony / má śię cżem chlubić / ále nie u Bogá.
Abowiem cóż Piſmo mówi? Uwierzył Abráhám Bogu / y przycżytáno mu to zá ſpráwiedliwość.
A robiącemu zápłátá nie bywá przycżytáná podług łáſki / ále podług długu ;
Nierobiącemu záś / lecż wierzącemu w tego / który uſpráwiedliwiá niepobożnego / przycżytáná bywá wiárá jego zá ſpráwiedliwość.
Jáko y Dawid powiádá / że błogoſłáwieńſtwo cżłowieká jeſt / któremu Bóg przycżytá ſpráwiedliwość bez ucżynków / mówiąc :
Błogoſłáwieni / których odpuƺcżone ſą niepráwośći / á których zákryte ſą grzechy ;
Błogoſłáwiony mąż / któremu Pán grzechu nie przycżytá.
To tedy błogoſłáwieńſtwo tylko ná obrzezkę przychodźi / cży też ná nieobrzezkę? Gdyż mówimy / iż wiárá Abráhámowi jeſt przycżytáná zá ſpráwiedliwość.
Jákoż mu tedy jeſt przycżytáná? Gdy był w obrzezce / cżyli w nieobrzezce? Nie w obrzezce / ále w nieobrzezce.
Y przyjął znák obrzezki zá piecżęć ſpráwiedliwośći onej wiáry / którá byłá w nieobrzezce / ná to / áby był ojcem wƺyſtkich wierzących w nieobrzezce / áby y onym przycżytáná byłá ſpráwiedliwość ;
Y áby był ojcem obrzezki / nie tylko tych / którzy ſą z obrzezki / ále też y tych / którzy chodzą ſtopámi wiáry ojcá náƺego Abráhámá / którá byłá w nieobrzezce.
Abowiem nie przez zákon śię ſtáłá obietnicá Abráhámowi álbo náśieniu jego / áby był dźiedźicem świátá / ále przez ſpráwiedliwość wiáry.
Bo jeźli ći / którzy ſą z zákonu / dźiedźicámi ſą / tedyć zniƺcżáłá wiárá y wniwecż śię obróćiłá obietnicá.
Gdyż zákon gniew ſpráwuje ; ábowiem gdźie zákonu niemáƺ / tám áni przeſtępſtwá.
A przetoż z wiáry jeſt dźiedźictwo / áby było z łáſki / y żeby byłá wárowná obietnicá wƺyſtkiemu náśieniu / nie tylko temu / które jeſt z zákonu / ále y temu / które jeſt z wiáry Abráhámowej / który jeſt ojcem náſ wƺyſtkich ;
( Jáko nápiſáno : Ojcem wielu národów wyſtáwiłem ćię ) przed Bogiem / któremu uwierzył / który ożywiá umárłe / y który przywołuje te rzecży / których niemáƺ / jákoby były.
Który ( Abráhám ) przećiwko nádźiei w nádźieję uwierzył / że śię ſtánie ojcem wielu národów według tego / co mu powiedźiáno : Ták będźie náśienie twoje.
A nie będąc ſłábym w wierze / nie pátrzáł ná ćiáło ſwoje już obumárłe / májąc około ſtu lát / áni ná obumárły żywot Sáry.
O obietnicy tedy Bożej nie wątpił z niedowiárſtwá ; ále śię umocnił wiárą y dáł chwáłę Bogu /
Będąc też tego pewien / że cokolwiek on obiecáł / mocen jeſt y ucżynić.
A przetoż przycżytáno mu to zá ſpráwiedliwość.
A nie nápiſáno tego dlá niego ſámego / iż mu to przycżytáno /
Ale y dlá náſ / którym má być przycżytáno / którzy wierzymy w tego / który wzbudźił JEzuſá / Páná náƺego / z mártwych ;
Który wydány jeſt dlá grzechów náƺych / á wſtáł z mártwych dlá uſpráwiedliwieniá náƺego.
