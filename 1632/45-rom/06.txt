Cóż tedy rzecżemy? Zoſtániemyż w grzechu / áby łáſká obfitowáłá?
Nie dáj tego Boże! ábowiem którzyśmy umárli grzechowi / jákoż jeƺcże w nim żyć będźiemy?
Azáż nie wiećie / iż którzykolwiek ochrzcżeni jeſteśmy w CHryſtuſá JEzuſá / w śmierć jego ochrzcżeni jeſteśmy?
Pogrzebieniśmy tedy z nim przez chrzeſt w śmierć / áby jáko CHryſtuſ wzbudzony jeſt z mártwych przez chwáłę ojcowſką / ták żebyśmy y my w nowośći żywotá chodźili.
Bo jeźliżeśmy z nim wƺcżepieni w podobieńſtwo śmierći jego / tedy też y w podobieńſtwo zmártwychwſtániá wƺcżepieni z nim będźiemy.
To wiedząc / że ſtáry náƺ cżłowiek poſpołu z nim jeſt ukrzyżowány / áby ćiáło grzechu było zniƺcżone / żebyśmy już więcej nie ſłużyli grzechowi ;
Bo ktoć umárł / uſpráwiedliwiony jeſt od grzechu.
Jeźliśmy tedy z CHryſtuſem umárli / wierzymy / iż też z nim żyć będźiemy /
Wiedząc / że CHryſtuſ powſtáwƺy z mártwych / więcej nie umierá / y śmierć mu więcej nie pánuje.
Bo iż umárł / grzechowi ráz umárł / á iż żyje / żyje Bogu.
Ták też y wy rozumiejćie / żeśćie wy umárłymi grzechowi / áleśćie żywymi Bogu w CHryſtuśie JEzuśie / Pánu náƺym.
Niechże tedy nie króluje grzech w śmiertelnem ćiele wáƺem / żebyśćie mu poſłuƺni byli w pożądliwośćiách jego.
Ani ſtáwiájćie cżłonków wáƺych orężem nieſpráwiedliwośći grzechowi : ále ſtáwiájćie śiebie ſámych Bogu / jáko z umárłych żywi / y cżłonki wáƺe orężem ſpráwiedliwośći Bogu.
Abowiem grzech pánowáć nád wámi nie będźie ; bośćie nie pod zákonem / ále pod łáſką.
Cóż tedy? Będźiemyż grzeƺyli / żeśmy nie pod zákonem / ále pod łáſką? Nie dáj tego Boże!
Azáż nie wiećie / że komu śię ſtáwiáćie zá ſługi ku poſłuƺeńſtwu / tegośćie ſługámi / komuśćie poſłuƺni ; bądź grzechowi ku śmierći / bądź poſłuƺeńſtwu ku ſpráwiedliwośći?
Ale chwáłá Bogu / że bywƺy ſługámi grzechu / uſłucháliśćie z ſercá ſpoſobu onej náuki / którejśćie śię poddáli.
A będąc uwolnieni od grzechu / ſtáliśćie śię niewolnikámi ſpráwiedliwośći.
Po ludzku mówię dlá mdłośći ćiáłá wáƺego. Abowiem jákośćie ſtáwiáli cżłonki wáƺe ná ſłużbę niecżyſtośći y niepráwośći ku cżynieniu niepráwośći : ták teráz ſtáwiájćie cżłonki wáƺe ná ſłużbę ſpráwiedliwośći ku poświęceniu.
Bo pókiśćie byli ſługámi grzechu / byliśćie wolnymi od ſpráwiedliwośći :
Jákiżeśćie tedy náoncżáſ pożytek mieli onych rzecży / zá które śię teráz wſtydźićie? Bo koniec onych jeſt śmierć.
Lecż teráz / będąc uwolnieni od grzechu / á zniewoleni Bogu / máćie pożytek ſwój ku poświęceniu / á koniec żywot wiecżny.
Abowiem zápłátá zá grzech jeſt śmierć ; ále dár z łáſki Bożej jeſt żywot wiecżny w CHryſtuśie JEzuśie / Pánu náƺym.
