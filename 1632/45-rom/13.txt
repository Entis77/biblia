Każdá duƺá niech będźie Zwierzchnośćiam wyżƺym poddána. Boć niemaƺ zwierzchnośći / tylko od Bogá : á te które ſą zwierzchnośći / od Bogá ſą poſtánowione.
A ták kto śię zwierzchnośći zprzećiwia / Bożemu śię poſtánowieniu zprzećiwia : á którzy śię zprzećiwiáją / ſámi ſobie potępienie zjednáją.
Abowiem Przełożeni / nie ſą ná poſtrách dobrym ucżynkom / ále złym. A chceƺ śię nie bać Zwierzchnośći / cżyń co jeſt dobrego ; á będźieƺ miał pochwałę od niey.
Bożym bowiem jeſt ſługą / tobie ku dobremu. Ale jeſli ucżyniƺ co <i>jeſt</i> złego / bój śię : boć nie dármo miecż nośi ; gdyż jeſt ſługą Bożym / mƺcżącym śię w gniewie nád cżyniącym co jeſt złego.
Przetoż trzebá być poddánym ; nie tylko dla gniewu / ále y dla ſumnienia.
Abowiem dla tego też podátki dawaćie / gdyż ſą ſługámi Bożymi / którzy tego ſámego uſtáwicżnie pilnują.
Oddawajćież tedy káżdemu cośćie powinni : Komu podátek / <i>temu</i> podátek : Komu cło / <i>temu</i> cło : komu bojaźń / <i>temu</i> bojaźń : komu cżęść / <i>temu</i> cżęść.
Nikomu nic winni nie bądźćie / tylko ábyśćie śię ſpołecżnie miłowáli : bo kto miłuje bliźniego / zakon wypełnił.
Gdyż to <i>przykazánie</i> : Nie będźieƺ cudzołożył / Nie będźieƺ zábijał / Nie będźieƺ kradł / Nie będźieƺ fałƺywie świadcżył / Nie będźieƺ pożądał : Y jeſli które inƺe jeſt przykazánie / w tym ſłowie ſummownie śię zámyka / miánowićie : Będźieƺ miłował bliźniego twego / jáko śiebie ſámego.
Miłość bliźniemu złośći nie wyrządza. A ták wypełnieniem zakonu jeſt miłość.
A to <i>cżyńćie</i> wiedząc cżás / iż już <i>przyƺłá</i> godźiná / ábyſmy śię ze ſnu ocućili. Abowiem teraz bliżƺe nas jeſt zbáwienie / á niż kiedyſmy uwierzyli.
Noc przeminęłá / á dźień śię przybliżył. Odrzućmyż tedy ucżynki ćiemnośći / á oblecżmy śię we zbroję świátłośći.
Chodźmy ucżćiwie jáko we dnie : nie w bieśiádách / y w pijáńſtwách : nie we wƺetecżeńſtwách / y roſpuſtách : nie w poſwarkách áni w zazdrośći.
Ale oblecżćie śię w PAná JEzuſá CHryſtuſá : á nie cżyńćie ſtáránia o ćiele <i>ku wykonywániu</i> pożądliwośći.
