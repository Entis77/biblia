Páweł / ſługá Jezuſá Chryſtuſá / powołány Apoſtoł / odłącżony ku opowiádániu Ewángielii Bożej ;
Którą przedtem obiecáł przez proroki ſwoje w Piſmách świętych /
O Synu ſwoim / który śię národził z náśieniá Dawidowego według ćiáłá ;
A pokázáł śię Synem Bożym możnie / według Duchá poświęceniá / przez zmártwychwſtánie / to jeſt / o Jezuśie Chryſtuśie / Pánu náƺym.
Przez którego wzięliśmy łáſkę y urząd ápoſtolſki ku poſłuƺeńſtwu wiáry między wƺyſtkimi národy / dlá imieniá jego ;
Między którymi jeſteśćie y wy powołáni od Jezuſá Chryſtuſá.
Wƺyſtkim / którzyśćie w Rzymie / umiłowánym Bożym / powołánym świętym / łáſká niech będzie wám y pokój od Bogá / Ojcá náƺego / y od Páná Jezuſá Chryſtuſá.
Nájprzód tedy dziękuję Bogu mojemu przez Jezuſá Chryſtuſá zá wáſ wƺyſtkich / iż wiárá wáƺá ſłynie po wƺyſtkim świećie.
Świádkiem mi bowiem jeſt on Bóg / któremu ſłużę w duchu moim w Ewángielii Syná jego / iż bez przeſtánku wzmiánkę o wáſ cżynię /
Záwƺe w modlitwách moich proƺąc / iżby mi śię wżdy kiedykolwiek drogá zdárzyłá zá wolą Bożą przyjść do wáſ.
Abowiem prágnę wáſ widzieć / ábym wám mógł udzielić jákiego dáru duchownego ku utwierdzeniu wáƺemu ;
To jeſt / ábyśmy śię u wáſ zobopólnie ućieƺyli przez ſpołecżną wiárę / y wáƺę y moję.
A nie chcę / ábyśćie wy wiedzieć nie mieli / bráćiá! żem cżęſto zámyśláł przyjść do wáſ ; ( álem był dotąd záwśćiągniony / ) ábym miáł jáki pożytek y między wámi / jáko y między inƺymi pogány.
Y Grekom / y grubym národom / y mądrym y głupim jeſtem dłużnikiem /
Ták / iż ile ze mnie jeſt / gotowym jeſt y wám / którzyśćie w Rzymie / Ewángieliję opowiádáć.
Abowiem nie wſtydzę śię zá Ewángieliję Chryſtuſową / poniewáż jeſt mocą Bożą ku zbáwieniu káżdemu wierzącemu / Żydowi nájprzód / potem y Grecżynowi.
Bo ſpráwiedliwość Bożá w niej bywá objáwioná z wiáry w wiárę / jáko nápiſáno : Że ſpráwiedliwy z wiáry żyć będzie.
Bo gniew Boży objáwiá śię z niebá przećiwko wƺelkiej niepobożnośći y nieſpráwiedliwośći tych ludzi / którzy zátrzymują práwdę Bożą w nieſpráwiedliwośći.
Przeto iż co może być wiedziáno o Bogu / jeſt w nich jáwno / gdyż im Bóg objáwił.
Bo rzecży jego niewidziálne od ſtworzeniá świátá / przez rzecży ucżynione widziálne bywáją / to jeſt / oná wiecżná jego moc y Bóſtwo / ná to / áby oni byli bez wymówki.
Przeto iż poznáwƺy Bogá / nie chwálili jáko Bogá / áni mu dziękowáli / owƺem znikcżemnieli w myślách ſwoich / y záćmiło śię bezrozumne ſerce ich :
Mieniąc śię być mądrymi / zgłupieli ;
Y odmienili chwáłę nieſkázitelnego Bogá w podobieńſtwo obrázu ſkázitelnego cżłowieká y ptáków y cżworonogich zwierząt y płázów.
A przetoż podáł je Bóg pożądliwośćiom ſerc ich ku niecżyſtośći / áby lżyli ćiáłá ſwoje między ſobą /
Jáko te / którzy odmienili práwdę Bożą w kłámſtwo / y chwálili ſtworzenie / y ſłużyli mu rácżej niż Stworzyćielowi / który jeſt błogoſłáwiony ná wieki. Amen.
Dlátego podáł je Bóg w námiętnośći ſromotne / gdyż y niewiáſty ich odmieniły używánie przyrodzone w ono / które jeſt przećiwko przyrodzeniu.
Tákże y mężcżyzny opuśćiwƺy przyrodzone używánie niewiáſty / zápálili śię w ſwej pożądliwośći jedni ku drugim / mężcżyzná z mężcżyzną háńbę płodząc á nágrodę náleżącą błędowi ſwemu ná śię biorąc.
A jáko śię im nie upodobáło mieć w znájomośći Bogá / ták też Bóg je podáł w umyſł opácżny / áby cżynili / co nie przyſtoi ;
Nápełnieni będąc wƺelákiej niepráwośći / wƺetecżeńſtwá / przewrotnośći / łákomſtwá / złośći / pełni zázdrośći / morderſtwá / ſporu / zdrády / złych obycżájów ;
Záuƺnicy / obmówcy / Bogá nienáwidzący potwárcy / pyƺni / chlubni / wynálázcy złych rzecży / rodzicom niepoſłuƺni /
Bezrozumni / przymierzá nie trzymájący / bez przyrodzonej miłośći / nieprzejednáni y niemiłośierni ;
Którzy poznáwƺy práwo Boże / iż ći / co tákowe rzecży cżynią / godni ſą śmierći / nie tylko ſámi je cżynią / ále też przeſtáwáją z tymi / co je cżynią.
