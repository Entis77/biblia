Páweł ſługá JEzuſá CHryſtuſá / powołány Apoſtoł / odłącżony ku <i>opowiádániu</i> Ewángeliey Bożey ;
Którą zdawná obiecał przez Proroki ſwoje w piśmiech świętych :
O Synu ſwojim / który śię národźił z naśienia Dawidowego według ćiáłá :
A pokazał śię Synem Bożym możnie / według Duchá poświęcenia prżez zmartwychwſtánie ; <i>to jeſt,</i> o JEzuśie CHryſtuśie PAnie náƺym.
Przez którego wźięliſmy łáſkę y urząd Apoſtolſki / ku poſłuƺeńſtwu wiáry miedzy wƺyſtkimi narody / dla Imienia jego.
Miedzy którymi jeſteśćie y wy powołáni <i>od</i> JEzuſá CHryſtuſá :
Wƺyſtkim którzyśćie w Rzymie / umiłowánym Bożym powołánym świętym : Łáſká <i>niech będźie</i> wam y pokój / od Bogá Ojcá náƺego / y <i>od</i> PAná JEzuſá CHryſtuſá.
Náprzód tedy dźiękuję Bogu mojemu przez JEzuſá CHryſtuſá zá was wƺyſtkich / iż wiárá wáƺá ſłynie po wƺyſtkim świećie.
Świádkiem mi bowiem jeſt on Bóg któremu ſłużę w Duchu mojim w Ewángeliey Syná jego / yż bez przeſtanku wzmiankę o was cżynię.
Záwƺe w modlitwách mojich proƺąc / iżby mi śię wżdy kiedykolwiek drogá zdárzyłá zá wolą Bożą przyść do was.
Abowiem prágnę was widźieć / ábym wam mógł udźielić jákiego dáru Duchownego / ku utwirdzeniu wáƺemu :
To jeſt / ábyſmy śię u was zobopólnie ućieƺyli przez ſpołecżną wiárę / y wáƺę y moję.
A nie chcę / ábyśćie wy wiedźieć nie mieli bráćia / żem cżęſto zámyślał przyść do was ; ( álem był do tąd záwśćiągniony ) á bym miał jáki pożytek y miedzy wámi / jáko y miedzy inƺymi Pogány.
Y Grekom / y grubym <i>narodom,</i> y mądrym / y głupim / jeſtem dłużnikiem ;
Ták / iż ile ze mnie jeſt / gotowem jeſt y wam którzyśćie w Rzymie Ewángelią opowiádáć.
Abowiem nie wſtydam śię zá Ewángelią CHryſtuſowę : ponieważ jeſt Mocą Bożą ku zbáwieniu káżdemu wierzącemu / Żydowi naprzód <i>potym</i> y Grecżynowi.
Bo ſpráwiedliwość Boża przez nię bywa objáwiona z wiáry w wiárę / jáko nápiſano ; Że ſpráwiedliwy z wiáry żyć będźie.
Bo gniew Boży objawia śię z niebá / przećiwko wƺelkiey niepobożnośći y nieſpráwiedliwośći tych ludźi / którzy zátrzymawáją prawdę <i>Bożą</i> w nieſpráwiedliwośći.
Przeto iż co może być wiedźiano o Bogu / jeſt miedzy nimi jáwne ; gdyż im <i>to</i> Bóg objáwił.
Bo <i>rzecży</i> jego niewidźiálne od ſtworzenia świátá / przez rzecży ucżynione / widźiálne bywáją : to jeſt / oná wiecżná jego moc y bóſtwo ; ná to / áby oni byli bez wymówki.
Przeto iż poznawƺy Bogá / nie chwalili jáko Bogá / áni <i>mu</i> dźiękowáli : owƺem znikcżemnieli w myślách ſwojich / y záćmiło śię bezrozumne ſerce ich.
Mieniąc śię być mądrymi / zgłupieli.
Y odmienili chwałę nieſkáźitelnego Bogá / w podobieńſtwo obrázu ſkáźitelnego cżłowieká / y ptaków / y cżworonogich <i>zwierząt</i> y płázów.
A przetoż podał je Bóg pożądliwośćiam ſerc ich ku niecżyſtośći / áby lżyli ćiáłá ſwoje miedzy ſobą.
<i>Jáko te,</i> którzy odmienili prawdę Bożą w kłamſtwo / y chwalili ſtworzenie / y ſłużyli <i>mu</i> rácżey niż Stworzyćielowi / który jeſt błogoſłáwiony ná wieki / Amen.
Dla tego podał je Bóg w namiętnośći ſromotne : gdyż y niewiáſty ich odmieniły używánie przyrodzone / w ono <i>które jeſt</i> przećiwko przyrodzeniu.
Tákże y mężcżyzny opuśćiwƺy przyrodzone używánie niewiáſty / zápalili śię w ſwey pożądliwośći jedni ku drugim / mężcżyzná z mężcżyzną háńbę płodząc ; á nagrodę należącą błędowi ſwemu ná śię biorąc.
A jáko śię im nie upodobáło mieć w znájomośći Bogá / <i>ták też</i> Bóg je podał w umyſł opácżny / áby cżynili co nie przyſtoji.
Nápełnieni będąc wƺelákiey niepráwośći / wƺetecżeńſtwá / przewrotnośći / łákomſtwá / złośći / pełni zazdrośći / morderſtwá / ſporu / zdrády / złych obycżájów :
Zauƺnicy / obmówcy / Bogá nienawidzący / potwarcy / pyƺni / chlubni / wynalezcy złych rzecży / rodźicom niepoſłuƺni :
Bezrozumni / przymierza nietrzymájący / bez przyrodzoney miłośći / nie przejednáni / <i>y</i> niemiłośierni.
Którzy poznawƺy práwo Boże / iż ći co tákowe <i>rzecży</i> cżynią / godni ſą śmierći : nie tylko ſámi je cżynią / ále też przeſtawają z tymi co je cżynią.
