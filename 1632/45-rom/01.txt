Páweł / ſługá JEzuſá CHryſtuſá / powołány Apoſtoł / odłącżony ku opowiádániu Ewángielii Bożej ;
Którą przedtem obiecáł przez proroki ſwoje w Piſmách świętych /
O Synu ſwoim / który śię národźił z náśieniá Dawidowego według ćiáłá ;
A pokázáł śię Synem Bożym możnie / według Duchá poświęceniá / przez zmartwychwſtánie / to jeſt / o JEzuśie CHryſtuśie / Pánu náƺym.
Przez którego wźięliśmy łáſkę y urząd ápoſtolſki ku poſłuƺeńſtwu wiáry między wƺyſtkimi národy / dla imieniá jego ;
Między którymi jeſteśćie y wy powołáni od JEzuſá CHryſtuſá.
Wƺyſtkim / którzyśćie w Rzymie / umiłowánym Bożym / powołánym świętym / łáſká niech będźie wám y pokój od Bogá / Ojcá náƺego / y od Páná JEzuſá CHryſtuſá.
Nájprzód tedy dźiękuję Bogu mojemu przez JEzuſá CHryſtuſá zá was wƺyſtkich / iż wiárá wáƺá ſłynie po wƺyſtkim świećie.
Świádkiem mi bowiem jeſt on Bóg / któremu ſłużę w duchu moim w Ewángielii Syná jego / iż bez przeſtánku wzmiánkę o was cżynię /
Záwƺe w modlitwách moich proƺąc / iżby mi śię wżdy kiedykolwiek drogá zdárzyłá zá wolą Bożą przyjść do was.
Abowiem prágnę was widźieć / ábym wám mógł udźielić jákiego dáru duchownego ku utwierdzeniu wáƺemu ;
To jeſt / ábyśmy śię u was zobopólnie ućieƺyli przez ſpołecżną wiárę / y wáƺę y moję.
A nie chcę / ábyśćie wy wiedźieć nie mieli / bráćia! żem cżęſto zámyśláł przyjść do was ; ( álem był dotąd záwśćiągniony / ) ábym miał jáki pożytek y między wámi / jáko y między inƺymi pogány.
Y Grekom / y grubym národom / y mądrym y głupim jeſtem dłużnikiem /
Ták / iż ile ze mnie jeſt / gotowym jeſt y wám / którzyśćie w Rzymie / Ewángieliję opowiádáć.
Abowiem nie wſtydzę śię zá Ewángieliję CHryſtuſową / poniewáż jeſt mocą Bożą ku zbáwieniu káżdemu wierzącemu / Żydowi nájprzód / potem y Grecżynowi.
Bo ſpráwiedliwość Bożá w niej bywá objáwioná z wiáry w wiárę / jáko nápiſáno : Że ſpráwiedliwy z wiáry żyć będźie.
Bo gniew Boży objáwiá śię z niebá przećiwko wƺelkiej niepobożnośći y nieſpráwiedliwośći tych ludźi / którzy zátrzymują práwdę Bożą w nieſpráwiedliwośći.
Przeto iż co może być wiedźiáno o Bogu / jeſt w nich jáwno / gdyż im Bóg objáwił.
Bo rzecży jego niewidźiálne od ſtworzeniá świátá / przez rzecży ucżynione widźiálne bywáją / to jeſt / oná wiecżná jego moc y Bóſtwo / ná to / áby oni byli bez wymówki.
Przeto iż poznáwƺy Bogá / nie chwálili jáko Bogá / áni mu dźiękowáli / owƺem znikcżemnieli w myślách ſwoich / y záćmiło śię bezrozumne ſerce ich :
Mieniąc śię być mądrymi / zgłupieli ;
Y odmienili chwáłę nieſkáźitelnego Bogá w podobieńſtwo obrázu ſkáźitelnego cżłowieká y ptáków y cżworonogich zwierząt y płázów.
A przetoż podáł je Bóg pożądliwośćiom ſerc ich ku niecżyſtośći / áby lżyli ćiáłá ſwoje między ſobą /
Jáko te / którzy odmienili práwdę Bożą w kłámſtwo / y chwálili ſtworzenie / y ſłużyli mu rácżej niż Stworzyćielowi / który jeſt błogoſłáwiony ná wieki. Amen.
Dlátego podáł je Bóg w námiętnośći ſromotne / gdyż y niewiáſty ich odmieniły używánie przyrodzone w ono / które jeſt przećiwko przyrodzeniu.
Tákże y mężcżyzny opuśćiwƺy przyrodzone używánie niewiáſty / zápálili śię w ſwej pożądliwośći jedni ku drugim / mężcżyzná z mężcżyzną háńbę płodząc á nágrodę náleżącą błędowi ſwemu ná śię biorąc.
A jáko śię im nie upodobáło mieć w znájomośći Bogá / ták też Bóg je podáł w umyſł opácżny / áby cżynili / co nie przyſtoi ;
Nápełnieni będąc wƺelákiej niepráwośći / wƺetecżeńſtwá / przewrotnośći / łákomſtwá / złośći / pełni zázdrośći / morderſtwá / ſporu / zdrády / złych obycżájów ;
Záuƺnicy / obmówcy / Bogá nienáwidzący potwárcy / pyƺni / chlubni / wynálázcy złych rzecży / rodźicom niepoſłuƺni /
Bezrozumni / przymierzá nie trzymájący / bez przyrodzoney miłośći / nieprzejednáni y niemiłośierni ;
Którzy poznáwƺy práwo Boże / iż ći / co tákowe rzecży cżynią / godni ſą śmierći / nie tylko ſámi je cżynią / ále też przeſtáwáją z tymi / co je cżynią.
