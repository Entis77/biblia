Judáſ / ſługá JEzuſá CHryſtuſá á brát Jákóbá / od Bogá Ojcá poświęconym y w JEzuśie CHryſtuśie záchowánym y powołánym :
Miłośierdźie / y pokój / y miłość niech śię wám rozmnoży.
Nájmilśi! wƺelką pilność cżyniąc / ábym wám piſáł o ſpołecżnem zbáwieniu / muśiáłem wám piſáć / nápominájąc / iżbyśćie bojowáli o wiárę ráz świętym podáną.
Abowiem wkrádli śię niektórzy ludźie / dáwno już przedtem náznácżeni ná to potępienie / niepobożni / którzy łáſkę Bogá náƺego obrácáją w rozpuſtę / y ſámego śię Bogá / y pánującego Páná náƺego / JEzuſá CHryſtuſá zápieráją.
Przetoż chcę wám przypomnieć / którzy już ráz o tem wiećie / że choćiáż Pán lud z źiemi Egipſkiej wyſwobodźił / przećież potem tych / którzy nie wierzyli / potráćił.
Tákże Aniołów / którzy nie záchowáli pierwƺego ſtánu ſwego / ále opuśćili mieƺkánie ſwoje / ná ſąd dniá wielkiego związkámi wiecżnemi pod chmurą záchowáł.
Jáko Sodomá y Gomorrá / y okolicżne miáſtá / gdy tymże ſpoſobem jáko y one zwƺetecżniáły y udáły śię zá cudzem ćiáłem / wyſtáwione ſą ná przykłád / ogniá wiecżnego káránie ponoƺąc :
Tákże też y ći jáko ſnem zmożeni ćiáło plugáwią / ále zwierzchnośćią pogárdzáją y przełożeńſtwá bluźnią.
Lecż Micháł Archánioł / gdy śię z dyjábłem rozpierájąc wádźił o ćiáło Mojżeƺowe / nie śmiáł podnieść przećiwko niemu ſądu bluźniercżego / ále rzekł : Niech ćię Pán zgromi.
A ći / cżego nie znáją / to bluźnią ; á co z przyrodzeniá znáją / jáko bezrozumne bydłá w tem śię pſują /
Biádá im! bo śię drogą Káinową událi / á zá błędem Báálámowej zápłáty puśćili śię / y przećiwieńſtwem Korego poginęli.
Cić ſą ná świętych ucżtách wáƺych zmázáni / którzy z wámi godując bez wſtydu / ſámi śię páſą ; ſą obłoki bezwodne / które od wiátrów tám y ſám unoƺone bywáją ; drzewá zwiędłe nieużytecżne / dwákroć zmárłe y wykorzenione ;
Wáły wśćiekłe morſkie / wyrzucájące ſwoje ſprośnośći / gwiázdy błąkájące śię / którym chmurá ćiemnośći záchowáná jeſt ná wieki.
A prorokowáł też o niech śiódmy od Adámá Enoch / mówiąc : Oto Pán idźie z świętymi tyśiącámi ſwoimi /
Aby ucżynił ſąd wƺyſtkim / y káráł wƺyſtkich niezbożników między nimi ze wƺyſtkich niepobożnośći ich / które niezbożnie płodźili y ze wƺyſtkich przykrośći / które mówili przećiwko niemu niezbożni grzeƺnicy.
Ci ſą ƺemrácże utyſkujący ſobie / według pożądliwośći ſwoich chodzący / y których uſtá mówią bárdzo hárde ſłowá ; pochlebiájąc oſobom dlá ſwego pożytku.
Lecż wy / nájmilśi! pámiętájćie ná ſłowá przepowiedźiáne od Apoſtołów Páná náƺego JEzuſá CHryſtuſá ;
Iż wám powiádáli / że w oſtátecżny cżáſ będą náśmiewcy / chodzący według ſwoich niezbożnych pożądliwośći.
Cić ſą / którzy śię ſámi odłącżáją / bydlęcy / Duchá CHryſtuſowego niemájący.
Ale wy / nájmilśi! budując śię ná nájświętƺej wierze wáƺej / y modląc śię w Duchu Świętym /
Sámych śiebie w miłośći Bożej záchowájćie / ocżekując miłośierdźiá Páná náƺego / JEzuſá CHryſtuſá / ku żywotowi wiecżnemu.
A nád niektórymi zmiłujćie śię / rozſądkiem śię rządząc ;
A drugich przez poſtrách do zbáwieniá przywódźćie / z ogniá ich wyrywájąc / májąc w nienáwiśći y ſuknię / któráby byłá od ćiáłá pokáláná.
A temu / który wáſ może záchowáć od upádku / y ſtáwić przed oblicżnośćią chwáły ſwojej bez nágány z weſelem /
Sámemu mądremu Bogu / Zbáwićielowi náƺemu / niech będźie chwáłá y wielmożność / moc y zwierzchność / y teráz y po wƺyſtkie wieki. Amen.
