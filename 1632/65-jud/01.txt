Judas ſługá JEzuſá CHryſtuſá / á brát Jákubów ; od Bogá Ojcá poświęconym / y w JEzuśie CHryſtuśie záchowánym y powołánym :
Miłośierdźie y pokój / y miłość / niech śię wam rozmnoży.
Namilƺy / wƺelką pilność cżyniąc / ábym wam piſał o ſpołecżnym zbáwieniu / muśiałem wam piſáć : nápominájąc / iżbyśćie bojowáli o wiárę raz świętym podáną.
Abowiem wkrádli śię niektórzy ludźie / dawno już przed tym náznácżeni ná to potępienie / niepobożni / którzy łáſkę Bogá náƺego obracáją w roſpuſtę / y ſámego śię PAná Bogá / y PAná náƺego JEzuſá CHryſtuſá zápieráją.
Przetoż chcę wam to przypomnieć / którzy już raz o tym wiećie / że choćiaż PAN lud z źiemie Egipſkiey wyſwobodźił / przećie potym te którzy nie wierzyli / potráćił.
Tákże Anjoły / którzy nie záchowáli pierwƺego ſtanu ſwego / ále opuśćili mieƺkánie ſwoje / ná ſąd dniá wielkiego związkámi wiecżnymi pod chmurą záchował.
Jáko Sodomá y Gomorrá / y okolicżne miáſtá ; gdy tymże ſpoſobem jáko y one zwƺetecżniáły / y udáły śię zá cudzym ćiáłem ; wyſtáwione ſą ná przykład / ogniá wiecżnego karánie ponoƺąc.
Tákże też y ći jáko ſnem zmorzeni / ćiáłoć plugáwią / ále zwierzchnośćią pogárdzáją / y przełożeńſtwá bluźnią.
Lecż Michał Archánjoł gdy śię z dyabłem roſpierájąc wádźił o ćiáło Mojzeƺowe / nie śmiał podnieść przećiwko niemu ſądu bluźnierſkie / ále rzekł ; Niech ćię PAN zgromi.
A ći cżego nie znáją / to bluźnią : á co z przyrodzenia i znáją / jáko bezrozumne bydło w tym śię pſują.
Biádá im / bo śię drogą Kajinową událi : á zá błędem Báláámowey zapłáty puśćili śię / y przećiwieńſtwem Korego poginęli.
Cić ſą ná świętych ucżtách wáƺych zmázámi / którzy z wámi godując bez wſtydu / ſámi śię páſą : ſą obłoki bezwodne / które od wiátrów tám y ſám unoƺone bywáją ; drzewá zwiędłe / nieużytecżne / dwákroć zmárłe y wykorzenione :
Wáły wśćiekłe morſkie / wyrzucájące ſwoje ſproſnośći : gwiazdy błąkájące śię / którym chmurá ćiemnośći záchowána jeſt ná wieki.
A prorokował też o nich śiódmy od Adámá Enoch / mówiąc ; Oto PAN idźie z świętymi tyśiącámi ſwojimi :
Aby ucżynił ſąd wƺyſtkim / y karał wƺyſtkie niezbożniki miedzy nimi / ze wƺyſtkich niepobożnośći ich / które niezbożnie płodźili ; y ze wƺyſtkich przykrośći / które mówili przećiwko niemu niezbożni grzeƺnicy.
Ci ſą ƺemrácże utyſkujący ſobie : według pożądliwośći ſwojich chodzący / y których uſtá mówią bárzo hárde ſłowá ; pochlebując oſobam dla ſwego pożytku.
Lecż wy namilƺy / pámiętajćie ná ſłowá przepowiedźiáne od Apoſtołów PAná náƺego JEzuſá CHryſtuſá :
Iż wam powiádáli / że w oſtátecżny cżás będą naśmiewcy / chodzący według ſwojich niezbożnych pożądliwośći.
Cić ſą którzy śię ſámi odłącżáją / bydlęcy / Duchá CHryſtuſowego nie májący.
Ale wy namilƺy budując śię ná nájświętƺey wierze wáƺey / y modląc śię w Duchu Świętym :
Sámych śiebie w miłośći Bożey záchowájćie / ocżekawájąc miłośierdźia PAná náƺego JEzuſá CHryſtuſá / ku żywotowi wiecżnemu.
A nád niektórymi zmiłujćie śię / rozſądkiem śię rządząc.
A drugie przez poſtrách do zbáwienia przywódźćie / z ogniá je wyrywájąc ; májąc w nienáwiśći y ſuknią / któraby byłá od ćiáłá pokalána.
A temu który was może záchowáć od upadku / y ſtáwić przed oblicżnośćią chwały ſwojey / bez nágány z weſelem ;
Sámemu mądremu Bogu Zbáwićielowi náƺemu / niech będźie chwałá y wielmożność / moc y zwierzchność / y teraz / y po wƺyſtkie wieki / Amen.
