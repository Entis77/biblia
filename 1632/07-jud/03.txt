A teć ſą národy / które pozoſtáwił Pán / áby kuśił przez nie Izráelá / wƺyſtkie / którzy nie wiedźieli o żádnych wálkách Chánánejſkich ;
Aby wżdy wiedźieli potomkowie ſynów Izráelſkich / y poználi / co jeſt wálká / którzy jey zgołá przedtem nie ználi.
Pięćioro kśiążąt Filiſtyńſkich y wƺyſtkie Chánánejcżyki / y Sydońcżyki y Hewejcżyki / mieƺkájące ná górze Libánie / od góry Báál Hermon áż tám / gdźie wchodzą do Hemát.
Cić byli / przez które doświádcżał Izráelá / áby śię dowiedźiał / będąli poſłuƺni przykázániom Páńſkim / które rozkázał ojcom ich przez Mojżeƺá.
A ták ſynowie Izráelſcy mieƺkáli w pośród Chánánejcżyków / Hetejcżyków / y Amorejcżyków / y Ferezejcżyków y Hewejcżyków / y Jebuzejcżyków.
Y bráli ſobie córki ich zá żony / á córki ſwe dáwáli ſynom ich / y ſłużyli bogom ich.
Y cżynili Izráelſcy ſynowie złe przed ocżymá Páńſkimi : zápomniáwƺy Páná / Bogá ſwego / ſłużyli Báálom / y święconym gájom.
Tedy śię zápálił gniew Páńſki przećiw Izráelowi / y podał je w rękę Chuſánráſátáimá / królá Syrſkiego w Mezopotámii : A ſłużyli ſynowie Izráelſcy Chuſánráſátáimowi przez ośm lat.
Potem wołáli ſynowie Izráelſcy do Páná ; Y wzbudźił Pán wybáwićielá ſynom Izráelſkim / áby je wybáwił / Otonijelá / ſyná Kenezowego / brátá Kálebowego młodƺego.
Y był nád nim Duch Páńſki / á ſądźił Izráelá ; á gdy śię ruƺył ná wojnę / podał Pán w rękę jego Chuſánráſátáimá / królá Syryjſkiego / y zmocniłá śię ręká jego nád Chuſánráſátáimem.
A ták byłá w pokoju źiemiá przez cżterdźieśći lat / áż umárł Otonijel / ſyn Kenezów.
Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi. Y zmocnił Pán Egloná / królá Moábſkiego / przećiw Izráelowi / przeto iż cżynili złe przed ocżymá Páńſkimi.
Bo zebráwƺy do śiebie ſyny Ammonowe y Amálekowe ruƺył śię / y poráźił Izráelá / y opánował miáſto Pálm.
Służyli tedy ſynowie Izráelſcy Eglonowi / królowi Moábſkiemu / ośmnáśćie lat.
Potem wołáli ſynowie Izráelſcy do Páná. Y wzbudźił im Pán wybáwićielá / Aodá / ſyná Gery / ſyná Jemini / mężá ręką práwą niewłádájącego ; y poſłáli ſynowie Izráelſcy przezeń dár Eglonowi / królowi Moábſkiemu.
Y ucżynił ſobie Aod miecż z obu ſtron oſtry / ná łokieć wzdłuż / y przypáſał go pod ƺáty ſwe do práwego biodrá ſwojego.
Y przynióſł dár Eglonowi / królowi Moábſkiemu : á Eglon był cżłowiek bárzo otyły.
A gdy oddał dár / odpráwił lud / który był dár przynióſł ;
A ſám wróćiwƺy śię do gór kámiennych / które były w Gálgál / rzekł : Rzecż tájemną mam do ćiebie / o królu! któremu on odpowiedźiał : Milcż ; y wyƺli od niego wƺyſcy / którzy ſtáli przed nim.
Y wƺedƺy Aod do niego / ( á on ſám śiedźiał ná ſáli letniej / którą miał ſám dla śiebie / ) y rzekł Aod : Mam rozkázánie Boże do ćiebie. Y powſtał z ſtolicy ſwojey.
Tedy Aod wyćiągnąwƺy lewą rękę ſwą / dobył miecżá od práwego biodrá ſwego / y wráźił go w brzuch jego /
Ták iż wpádłá y rękojeść zá żelázem / y záwárło śię w ſádle żelázo ; bo był nie wyjął miecżá z brzuchá jego / áż śię y gnój rzućił.
Wyƺedł potem Aod przez przyśionek / á zámknął drzwi gmáchu zá ſobą / y záwárł zámkiem.
A gdy on wyƺedł / ſłudzy jego przyƺli / á widząc / iż drzwi gmáchu zámknione były / rzekli : Podobno ſobie król cżyni wcżás ná ſáli letniej.
A nácżekáwƺy śię / áż śię wſtydźili / widząc / że on nie otwierá drzwi ſáli / wźiąwƺy klucż otworzyli ; á oto / pán ich leżał ná źiemi umárły.
Lecż Aod uƺedł / póki śię oni báwili / á minąwƺy góry kámienne / ƺedł do Seirátu.
A gdy przyƺedł / zátrąbił w trąbę ná górze Efráim ; y zſtąpili z nim ſynowie Izráelſcy z góry / á on przed nimi.
Y rzekł do nich : Pójdźćie zá mną : ábowiem podał Pán nieprzyjáćioły wáƺe Moábity w ręce wáƺe. Tedy ƺli zá nim / á odjąwƺy bród Jordáńſki Moábitom / nie dopuƺcżáli nikomu przepráwy.
Y pobili Moábitów ná ten cżás około dźieśięćiu tyśięcy mężów / wƺyſtko bogátych / y wƺyſtko mężów dużych / á nikt nie uƺedł.
Y poniżony jeſt Moáb dniá onego pod ręką Izráelá / á byłá w pokoju źiemiá przez ośmdźieśiąt lat.
A po nim był Sámgár / ſyn Anátów / który poráźił z Filiſtynów ƺeść ſet mężów ſtykiem wołowym / á wybáwił y ten Izráelá.
