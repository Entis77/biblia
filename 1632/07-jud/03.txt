A teć ſą národy / które pozoſtáwił Pán / áby kuśił przez nie Izráelá / wƺyſtkie / którzy nie wiedzieli o żádnych wálkách Chánánejſkich ;
Aby wżdy wiedzieli potomkowie ſynów Izráelſkich / y poználi / co jeſt wálká / którzy jej zgołá przedtem nie ználi.
Pięćioro kśiążąt Filiſtyńſkich y wƺyſtkie Chánánejcżyki / y Sydońcżyki y Hewejcżyki / mieƺkájące ná górze Libánie / od góry Báál Hermon áż tám / gdzie wchodzą do Hemát.
Cić byli / przez które doświádcżáł Izráelá / áby śię dowiedziáł / będąli poſłuƺni przykázániom Páńſkim / które rozkázáł ojcom ich przez Mojżeƺá.
A ták ſynowie Izráelſcy mieƺkáli w pośród Chánánejcżyków / Hetejcżyków / y Amorejcżyków / y Ferezejcżyków y Hewejcżyków / y Jebuzejcżyków.
Y bráli ſobie córki ich zá żony / á córki ſwe dáwáli ſynom ich / y ſłużyli bogom ich.
Y cżynili Izráelſcy ſynowie złe przed ocżymá Páńſkimi : zápomniáwƺy Páná / Bogá ſwego / ſłużyli Báálom / y święconym gájom.
Tedy śię zápálił gniew Páńſki przećiw Izráelowi / y podáł je w rękę Chuſánráſátáimá / królá Syrſkiego w Mezopotámii : A ſłużyli ſynowie Izráelſcy Chuſánráſátáimowi przez ośm lát.
Potem wołáli ſynowie Izráelſcy do Páná ; Y wzbudził Pán wybáwićielá ſynom Izráelſkim / áby je wybáwił / Otonijelá / ſyná Kenezowego / brátá Kálebowego młodƺego.
Y był nád nim Duch Páńſki / á ſądził Izráelá ; á gdy śię ruƺył ná wojnę / podáł Pán w rękę jego Chuſánráſátáimá / królá Syryjſkiego / y zmocniłá śię ręká jego nád Chuſánráſátáimem.
A ták byłá w pokoju ziemiá przez cżterdzieśći lát / áż umárł Otonijel / ſyn Kenezów.
Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi. Y zmocnił Pán Egloná / królá Moábſkiego / przećiw Izráelowi / przeto iż cżynili złe przed ocżymá Páńſkimi.
Bo zebráwƺy do śiebie ſyny Ammonowe y Amálekowe ruƺył śię / y poráził Izráelá / y opánowáł miáſto Pálm.
Służyli tedy ſynowie Izráelſcy Eglonowi / królowi Moábſkiemu / ośmnáśćie lát.
Potem wołáli ſynowie Izráelſcy do Páná. Y wzbudził im Pán wybáwićielá / Aodá / ſyná Gery / ſyná Jemini / mężá ręką práwą niewłádájącego ; y poſłáli ſynowie Izráelſcy przezeń dár Eglonowi / królowi Moábſkiemu.
Y ucżynił ſobie Aod miecż z obu ſtron oſtry / ná łokieć wzdłuż / y przypáſáł go pod ƺáty ſwe do práwego biodrá ſwojego.
Y przynióſł dár Eglonowi / królowi Moábſkiemu : á Eglon był cżłowiek bárdzo otyły.
A gdy oddáł dár / odpráwił lud / który był dár przynióſł ;
A ſám wróćiwƺy śię do gór kámiennych / które były w Gálgál / rzekł : Rzecż tájemną mám do ćiebie / o królu! któremu on odpowiedziáł : Milcż ; y wyƺli od niego wƺyſcy / którzy ſtáli przed nim.
Y wƺedłƺy Aod do niego / ( á on ſám śiedziáł ná ſáli letniej / którą miáł ſám dlá śiebie / ) y rzekł Aod : Mám rozkázánie Boże do ćiebie. Y powſtáł z ſtolicy ſwojej.
Tedy Aod wyćiągnąwƺy lewą rękę ſwą / dobył miecżá od práwego biodrá ſwego / y wráził go w brzuch jego /
Ták iż wpádłá y rękojeść zá żelázem / y záwárło śię w ſádle żelázo ; bo był nie wyjął miecżá z brzuchá jego / áż śię y gnój rzućił.
Wyƺedł potem Aod przez przyśionek / á zámknął drzwi gmáchu zá ſobą / y záwárł zámkiem.
A gdy on wyƺedł / ſłudzy jego przyƺli / á widząc / iż drzwi gmáchu zámknione były / rzekli : Podobno ſobie król cżyni wcżáſ ná ſáli letniej.
A nácżekáwƺy śię / áż śię wſtydzili / widząc / że on nie otwierá drzwi ſáli / wziąwƺy klucż otworzyli ; á oto / pán ich leżáł ná ziemi umárły.
Lecż Aod uƺedł / póki śię oni báwili / á minąwƺy góry kámienne / ƺedł do Seirátu.
A gdy przyƺedł / zátrąbił w trąbę ná górze Efráim ; y zſtąpili z nim ſynowie Izráelſcy z góry / á on przed nimi.
Y rzekł do nich : Pójdźćie zá mną : ábowiem podáł Pán nieprzyjáćioły wáƺe Moábity w ręce wáƺe. Tedy ƺli zá nim / á odjąwƺy bród Jordáńſki Moábitom / nie dopuƺcżáli nikomu przepráwy.
Y pobili Moábitów ná ten cżáſ około dzieśięćiu tyśięcy mężów / wƺyſtko bogátych / y wƺyſtko mężów dużych / á nikt nie uƺedł.
Y poniżony jeſt Moáb dniá onego pod ręką Izráelá / á byłá w pokoju ziemiá przez ośmdzieśiąt lát.
A po nim był Sámgár / ſyn Anátów / który poráził z Filiſtynów ƺeść ſet mężów ſtykiem wołowym / á wybáwił y ten Izráelá.
