Potem ƺedł Sámſon do Gázy / á ujrzáwƺy tám niewiáſtę nierządną / wƺedł do niej.
Y powiedźiáno mieƺcżánom w Gáźie : Przyƺedł tu Sámſon ; którzy obſtąpiwƺy go / ſtrzegli nań cáłą noc w brámie miejſkiej / á ſpráwując śię ćicho przez onę cáłą noc / mówili : Gdy śię pocżnie rozedniwáć / zábijemy go.
Ale Sámſon ſpał áż do północy / á wſtáwƺy o północy / ujął wrotá brámy miejſkiej ze dwiemá podwojámi / y wyrwał je z záworą / y włożył ná rámioná ſwoje / á zánióſł je ná wierzch góry / która byłá przećiw Hebronowi.
Y ſtáło śię potem / że śię rozmiłował niewiáſty w dolinie Sorek / którey imię Dálilá.
Y przyƺli do niej kśiążętá Filiſtyńſkie / y mówili jey : Oƺukáj go / á wywiedz śię / w cżym jeſt moc jego wielká / á jákobyſmy go przemóc y związáwƺy utrápić mogli? á dáć káżdy z nas tyśiąc y ſto ſrebrników.
Tedy rzekłá Dálilá do Sámſoná : Powiedz mi proƺę / w cżym jeſt moc twojá wielká / á cżembyś związány y utrápiony być mógł?
Y odpowiedźiał jey Sámſon : Jeſliby mię związáno śiedmią wići ſurowych / które jeƺcże nie uſchły / tedy oſłábieję / y będę jáko inny cżłowiek.
Y przynioſły jey kśiążętá Filiſtyńſkie śiedm wići ſurowych / które jeƺcże nie były uſchły / y związáłá go niemi.
A oni śię byli nań záſádźili w komorze / y rzekłá mu : Filiſtynowie nád tobą / Sámſonie ; ále on zerwał wići / jákoby kto zerwał nić zgrzebną / ogniem nápáloną ; y nie poznáno / w cżym byłá moc jego.
Rzekłá potem Dálilá do Sámſoná : Otoś mię oƺukał / y ſkłámáłeś przedemną ; teraz powiedz mi proƺę / cżemby ćię związáć?
A on jey odpowiedźiał : Jeſliby mię związáno powrozámi nowemi / których jeƺcże nie używano / tedy oſłábieję / y będę jáko inny cżłowiek.
A ták wźięłá Dálilá powrozy nowe / y związáłá go niemi / y rzekłá do niego : Filiſtynowie nád tobą / Sámſonie ; ( á oni śię byli nań záſádźili w komorze / ) ále porwał je ná rámionách ſwych jáko nići.
Rzekłá zátem Dálilá do Sámſoná : Pókiż ze mnie ƺydźić będźieƺ / y kłámáć przedemną? powiedzże mi / cżembyś mógł być związány? Y powiedźiał jey : Gdybyś przywiłá śiedm kędźierzy głowy mojey do wáłu tkáckiego.
Oná tedy przybiwƺy gwoźdźiem do wáłu tkáckiego rzekłá do niego : Filiſtynowie nád tobą / Sámſonie ; ále on ocućiwƺy śię ze ſnu ſwego / wyrwał gwóźdź z oſnową y z wáłem.
Znowu rzekłá do niego : Jákoż mówiƺ / miłuję ćię? á ſerce twoje nie jeſt ze mną. Jużeś mię po trzy kroć oƺukał / y nie powiedźiáłeś mi / w cżym jeſt twojá moc wielká.
A gdy mu śię uprzykrzáłá ſłowy ſwemi ná káżdy dźień / y trápiłá go / áż zemdláłá duƺá jego ná śmierć /
Tedy jey otworzył cále ſerce ſwoje / y powiedźiał jey : Brzytwá nigdy nie poſtáłá ná głowie mojey / gdyżem jeſt Názárejcżykiem Bożym záráz z żywotá mátki mojey ; gdyby mię ogolono / odejdźie odemnie moc mojá / y oſłábieję / y będę jáko inny cżłowiek.
Widząc tedy Dálilá / że jey otworzył cále ſerce ſwoje / poſłáłá y wezwáłá kśiążąt Filiſtyńſkich / mówiąc : Pójdźćież jeƺcże raz / boć mi otworzył cále ſerce ſwoje ; y przyƺły do niej kśiążętá Filiſtyńſkie / nioſąc ſrebro w rękách ſwych.
Tedy go uśpiłá ná łonie ſwojim / á przyzwáwƺy niektórego cżłowieká / dáłá ogolić śiedm kędźierzy głowy jego ; potem go jęłá dráźnić / gdy odeƺłá moc jego od niego /
Y rzekłá : Filiſtynowie nád tobą / Sámſonie. A ocućiwƺy śię ze ſnu ſwego / rzekł : Wynijdę jáko y pierwej / á wybiję śię ; á nie wiedźiał / że Pán odſtąpił od niego.
Tedy pojmáwƺy go Filiſtynowie / wyłupili mu ocży / y wiedli go do Gázy / związáwƺy go dwomá miedźiánymi łáńcuchámi / y muśiał mleć w domu więźniów.
Potem pocżęły włoſy ná głowie jego odráſtáć po onem goleniu.
A kśiążętá Filiſtyńſkie zebráli śię ſpráwowáć ofiáry wielkie Dágonowi / bogu ſwemu / weſelili śię / y mówili : Podał bóg náƺ w ręce náƺe Sámſoná / nieprzyjáćielá náƺego.
Którego też ujrzáwƺy lud chwálili bogá ſwego / bo mówili : Podał bóg náƺ w ręce náƺe nieprzyjáćielá náƺego / á tego / który puſtoƺył źiemię náƺę / y który wiele z náƺych pozábijał.
Y ſtáło śię / gdy byli dobrej myśli / że rzekli : Záwołájćie Sámſoná / áby błáznował przed námi. A ták záwołáno Sámſoná z domu więźniów / áby błáznował przed nimi ; y poſtáwili go miedzy dwomá ſłupámi.
Zátem rzekł Sámſon do chłopcá / który go trzymał zá rękę jego : Przywiedź mię / ábym pomácał ſłupów / ná których dom ſtoi / y podpárł śię ná nich.
A dom pełen był mężów y niewiáſt ; támże były wƺyſtkie kśiążętá Filiſtyńſkie / á ná dáchu około trzech tyśięcy mężów y niewiáſt / którzy śię przypátrowáli / gdy błáznował Sámſon.
Wzywał tedy Sámſon Páná / y rzekł : Pánie Boże / wſpomnij ná mię / proƺę / á zmocnij mię proƺę tylko ten raz ; Boże / ábym śię raz pomśćił obu ocżu mojich nád Filiſtynámi.
A ująwƺy Sámſon obá ſłupy pośrednie / ná których dom ſtał / wſpárł śię o nie / o jeden práwą ręką ſwoją á o drugi lewą ręką ſwoją.
Zátem rzekł Sámſon : Niech umrze duƺá mojá z Filiſtynámi ; á gdy śię o nie mocno opárł / upádł dom ná kśiążętá / y ná wƺyſtek lud / który w nim był / y było umárłych / które on pobił umierájąc / więcey niż onych / które pobił zá żywotá ſwego.
A przyƺedƺy bráćia jego / y wƺyſtek dom ojcá jego / wźięli go / á wróćiwƺy śię pogrzebli go miedzy Sáráá / y miedzy Eſtáol / w grobie Mánue / ojcá jego. A on ſądźił Izráelá przez dwádźieśćiá lat.
