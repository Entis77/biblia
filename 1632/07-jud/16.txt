Potem ƺedł Sámſon do Gázy / á ujrzáwƺy tám niewiáſtę nierządną / wƺedł do niej.
Y powiedziáno mieƺcżánom w Gázie : Przyƺedł tu Sámſon ; którzy obſtąpiwƺy go / ſtrzegli náń cáłą noc w brámie miejſkiej / á ſpráwując śię ćicho przez onę cáłą noc / mówili : Gdy śię pocżnie rozedniwáć / zábijemy go.
Ale Sámſon ſpáł áż do północy / á wſtáwƺy o północy / ujął wrotá brámy miejſkiej ze dwiemá podwojámi / y wyrwáł je z záworą / y włożył ná rámioná ſwoje / á zánióſł je ná wierzch góry / którá byłá przećiw Hebronowi.
Y ſtáło śię potem / że śię rozmiłowáł niewiáſty w dolinie Sorek / której imię Dálilá.
Y przyƺli do niej kśiążętá Filiſtyńſkie / y mówili jej : Oƺukáj go / á wywiedz śię / w cżem jeſt moc jego wielká / á jákobyśmy go przemóc y związáwƺy utrápić mogli? á dáć káżdy z náſ tyśiąc y ſto ſrebrników.
Tedy rzekłá Dálilá do Sámſoná : Powiedz mi proƺę / w cżem jeſt moc twojá wielká / á cżembyś związány y utrápiony być mógł?
Y odpowiedziáł jej Sámſon : Jeźliby mię związáno śiedmią wići ſurowych / które jeƺcże nie uſchły / tedy oſłábieję / y będę jáko inny cżłowiek.
Y przynioſły jej kśiążętá Filiſtyńſkie śiedm wići ſurowych / które jeƺcże nie były uſchły / y związáłá go niemi.
A oni śię byli náń záſádzili w komorze / y rzekłá mu : Filiſtynowie nád tobą / Sámſonie ; ále on zerwáł wići / jákoby kto zerwáł nić zgrzebną / ogniem nápáloną ; y nie poznáno / w cżem byłá moc jego.
Rzekłá potem Dálilá do Sámſoná : Otoś mię oƺukáł / y ſkłámáłeś przedemną ; teráz powiedz mi proƺę / cżemby ćię związáć?
A on jej odpowiedziáł : Jeźliby mię związáno powrozámi nowemi / których jeƺcże nie używáno / tedy oſłábieję / y będę jáko inny cżłowiek.
A ták wzięłá Dálilá powrozy nowe / y związáłá go niemi / y rzekłá do niego : Filiſtynowie nád tobą / Sámſonie ; ( á oni śię byli náń záſádzili w komorze / ) ále porwáł je ná rámionách ſwych jáko nići.
Rzekłá zátem Dálilá do Sámſoná : Pókiż ze mnie ƺydzić będzieƺ / y kłámáć przedemną? powiedzże mi / cżembyś mógł być związány? Y powiedziáł jej : Gdybyś przywiłá śiedm kędzierzy głowy mojej do wáłu tkáckiego.
Oná tedy przybiwƺy gwoździem do wáłu tkáckiego rzekłá do niego : Filiſtynowie nád tobą / Sámſonie ; ále on ocućiwƺy śię ze ſnu ſwego / wyrwáł gwóźdź z oſnową y z wáłem.
Znowu rzekłá do niego : Jákoż mówiƺ / miłuję ćię? á ſerce twoje nie jeſt ze mną. Jużeś mię po trzy kroć oƺukáł / y nie powiedziáłeś mi / w cżem jeſt twojá moc wielká.
A gdy mu śię uprzykrzáłá ſłowy ſwemi ná káżdy dzień / y trápiłá go / áż zemdláłá duƺá jego ná śmierć /
Tedy jej otworzył cále ſerce ſwoje / y powiedziáł jej : Brzytwá nigdy nie poſtáłá ná głowie mojej / gdyżem jeſt Názárejcżykiem Bożym záráz z żywotá mátki mojej ; gdyby mię ogolono / odejdzie odemnie moc mojá / y oſłábieję / y będę jáko inny cżłowiek.
Widząc tedy Dálilá / że jej otworzył cále ſerce ſwoje / poſłáłá y wezwáłá kśiążąt Filiſtyńſkich / mówiąc : Pójdźćież jeƺcże ráz / boć mi otworzył cále ſerce ſwoje ; y przyƺły do niej kśiążętá Filiſtyńſkie / nioſąc ſrebro w rękách ſwych.
Tedy go uśpiłá ná łonie ſwojim / á przyzwáwƺy niektórego cżłowieká / dáłá ogolić śiedm kędzierzy głowy jego ; potem go jęłá dráźnić / gdy odeƺłá moc jego od niego /
Y rzekłá : Filiſtynowie nád tobą / Sámſonie. A ocućiwƺy śię ze ſnu ſwego / rzekł : Wynijdę jáko y pierwej / á wybiję śię ; á nie wiedziáł / że Pán odſtąpił od niego.
Tedy pojmáwƺy go Filiſtynowie / wyłupili mu ocży / y wiedli go do Gázy / związáwƺy go dwomá miedziánymi łáńcuchámi / y muśiáł mleć w domu więźniów.
Potem pocżęły włoſy ná głowie jego odráſtáć po onem goleniu.
A kśiążętá Filiſtyńſkie zebráli śię ſpráwowáć ofiáry wielkie Dágonowi / bogu ſwemu / weſelili śię / y mówili : Podáł bóg náƺ w ręce náƺe Sámſoná / nieprzyjáćielá náƺego.
Którego też ujrzáwƺy lud chwálili bogá ſwego / bo mówili : Podáł bóg náƺ w ręce náƺe nieprzyjáćielá náƺego / á tego / który puſtoƺył ziemię náƺę / y który wiele z náƺych pozábijáł.
Y ſtáło śię / gdy byli dobrej myśli / że rzekli : Záwołájćie Sámſoná / áby błáznowáł przed námi. A ták záwołáno Sámſoná z domu więźniów / áby błáznowáł przed nimi ; y poſtáwili go między dwomá ſłupámi.
Zátem rzekł Sámſon do chłopcá / który go trzymáł zá rękę jego : Przywiedź mię / ábym pomácáł ſłupów / ná których dom ſtoi / y podpárł śię ná nich.
A dom pełen był mężów y niewiáſt ; támże były wƺyſtkie kśiążętá Filiſtyńſkie / á ná dáchu około trzech tyśięcy mężów y niewiáſt / którzy śię przypátrowáli / gdy błáznowáł Sámſon.
Wzywáł tedy Sámſon Páná / y rzekł : Pánie Boże / wſpomnij ná mię / proƺę / á zmocnij mię proƺę tylko ten ráz ; Boże / ábym śię ráz pomśćił obu ocżu moich nád Filiſtynámi.
A ująwƺy Sámſon obá ſłupy pośrednie / ná których dom ſtáł / wſpárł śię o nie / o jeden práwą ręką ſwoją á o drugi lewą ręką ſwoją.
Zátem rzekł Sámſon : Niech umrze duƺá mojá z Filiſtynámi ; á gdy śię o nie mocno opárł / upádł dom ná kśiążętá / y ná wƺyſtek lud / który w nim był / y było umárłych / które on pobił umierájąc / więcej niż onych / które pobił zá żywotá ſwego.
A przyƺedłƺy bráćiá jego / y wƺyſtek dom ojcá jego / wzięli go / á wróćiwƺy śię pogrzebli go między Sáráá / y między Eſtáol / w grobie Mánue / ojcá jego. A on ſądził Izráelá przez dwádzieśćiá lát.
