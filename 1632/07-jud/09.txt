Tedy odƺedł Abimelech / ſyn Jerobáálów / do Sychem / do bráći mátki ſwojey / y mówił do nich / y do wƺyſtkiego národu domu ojcá mátki ſwej / á rzekł :
Mówćie proƺę / gdźieby ſłyƺeli wƺyſcy przełożeni Sychem : Co wám lepƺego / áby nád wámi pánowáło śiedmdźieśiąt mężów / wƺyſcy ſynowie Jerobáálowi / cżyli żeby pánowáł nád wámi mąż jeden? Wżdy pámiętájćie / żem ja kość wáƺá / y ćiáło wáƺe.
Tedy mówili bráćia mátki jego o nim / gdźie ſłyƺeli wƺyſcy przełożeni Sychem / wƺyſtkie te ſłowá / y nákłoniło śię ſerce ich zá Abimelechem / bo rzekli : Brát náƺ jeſt.
Y dáli mu śiedmdźieśiąt ſrebrników z domu Báálberyt / y náprzyjmowáł zá nie Abimelech ludźi lekkomyślnych / y tułácżów / którzy chodźili zá nim.
Y przyƺedł w dom ojcá ſwego do Efrá / y pobił bráćią ſwą / ſyny Jerobáálowe / śiedmdźieśiąt mężów ná jednym kámieniu ; tylko zoſtáł Jotám / ſyn Jerobáálów / nájmniejƺy / iż śię był ſkrył.
Y zebráli śię wƺyſcy mężowie Sychem / y wƺyſtek dom Mello / á ƺedłƺy obráli Abimelechá królem ná równinie / kędy ſtáł ſłup w Sychem.
Co gdy powiedźiáno Jotámowi / ƺedłƺy ſtánął ná wierzchu góry Gáryzym / á podnióſłƺy głoſ ſwój wołáł / y rzekł im : Poſłuchájćie mię mężowie Sychem / á was też Bóg uſłyƺy.
Zeƺły śię drzewá / áby pomázáły nád ſobą królá / y rzekły do oliwnego drzewá : Króluj nád námi.
Którym odpowiedźiáło oliwne drzewo : Izáli opuƺcżę tłuſtość moję / przez którą ucżcżony bywá Bóg y ludźie / á pójdę / ábym wyſtáwione było nád drzewy?
Rzekły potem drzewá do figowego drzewá : Pójdź ty / króluj nád námi.
Którym odpowiedźiáło figowe drzewo : Izáli opuƺcżę ſłodkość moję / y owoc mój wyborny / á pójdę / ábym wyſtáwione było nád drzewy?
Potem rzekły drzewá do máćicy winney : Pójdź ty / króluj nád námi.
Tedy im odpowiedźiáłá máćicá : Izáli opuƺcżę moƺcż mój / który uweſelá Bogá y ludźie / á pójdę / ábym wyſtáwioná byłá nád drzewy?
Y rzekły wƺyſtkie drzewá do oſtu : Pójdź ty / króluj nád námi.
Tedy odpowiedźiáł oſet drzewom : jeſli wy práwdźiwie chcećie pomázáć mię zá królá nád ſobą / pójdźćież / á odpocżywájćie pod ćieniem moim / á jeſliż nie / niech wynijdźie ogień z oſtu / á ſpáli Cedry Libáńſkie.
A ták teraz jeſliśćie práwdźiwie á ƺcżerze ucżynili / obráwƺy ſobie królem Abimelechá ; á jeſliśćie śię dobrze obeƺli z Jerobáálem y z domem jego / á jeſliśćie podług dobrodźiejſtw ręki jego ucżynili z nim ;
( Abowiem wálcżył ojćiec mój zá was / y podáł duƺę ſwą w niebezpiecżeńſtwo / áby was wyrwáł z ręki Mádyjáńcżyków ;
Aleśćie wy powſtáli przećiw domowi ojcá mego dźiś / y pobiliśćie ſyny jego / śiedmdźieśiąt mężów ná jednym kámieniu / y obráliśćie królem Abimelechá / ſyná ſłużebnicy jego / nád mężámi Sychem / iż brátem wáƺym jeſt. )
Jeſliśćie práwdźiwie á ƺcżerze obeƺli śię z Jerobáálem / y z domem jego dniá tego / weſelćież śię z Abimelechá / á on niech śię też weſeli z was.
Ale jeſliż nie / niechájże wynijdźie ogień z Abimelechá / á pożre męże Sychem / y dom Mello ; niechájże też wynijdźie ogień od mężów Sychem / y z domu Mello / á pożre Abimelechá.
Tedy ućiekł Jotám / á ućiekłƺy poƺedł do Beer / y mieƺkáł tám / bojąc śię Abimelechá / brátá ſwego.
A ták pánowáł Abimelech nád Izráelem przez trzy látá.
Y poſłáł Bóg duchá złego między Abimelechá y między męże Sychemſkie / á złámáli wiárę mężowie Sychem Abimelechowi ;
Aby śię zemƺcżono krzywdy śiedmdźieśięćiu ſynów Jerobáálowych / áby krew ich przyƺłá ná Abimelechá / brátá ich / który je pobił / y ná męże Sychemſkie / którzy zmocnili ręce jego / áby pobił bráćią ſwoję.
Y záſádźili śię náń mężowie Sychemſcy ná wierzchu gór / á rozbijáli káżdego / który jedno ƺedł támtą drogą. Y powiedźiáno to Abimelechowi.
Nádto przyƺedł Gáál / ſyn Obedów / y bráćia jego / y przyƺedł do Sychem / á poufáli mu mężowie Sychem.
A wyƺedłƺy ná polá zbieráli winá ſwoje / y tłocżyli / weſeląc śię ; á wƺedłƺy w dom bogów ſwoich / jedli y pili / á złorzecżyli Abimelechowi.
Tedy rzekł Gáál / ſyn Obedów : Cóż jeſt Abimelech? y co jeſt Sychem / żebyśmy mu ſłużyli? ázáż nie jeſt ſyn Jerobáálów / á Zebul urzędnikiem jego? rácżej ſłużćie mężom Hemorá / ojcá Sychemowego ; bo cżemuż byśmy onemu ſłużyć mieli?
O by kto podáł ten lud w ręce moje / áby ſprzątnął Abimelechá! Y rzekł Abimelechowi : Zbierz ſwoje wojſko / á wynijdź.
A uſłyƺáwƺy Zebul / przełożony miáſtá onego / ſłowá Gáálá / ſyná Obedowego / zápálił śię gniew jego.
Y wypráwił poſły do Abimelechá potájemnie / mówiąc : Oto Gáál / ſyn Obedów / y bráćia jego przyƺli do Sychem / á oto chcą wálcżyć z miáſtem przećiwko tobie.
A ták teraz wſtáń nocą / ty y lud / który jeſt z tobą / á ucżyń záſádzkę w polu.
A ráno / gdy ſłońce wznijdźie / wſtáwƺy uderzyƺ ná miáſto ; á gdy on y lud / który jeſt z nim / wynijdźie przećiw tobie / ucżyniƺ z nim / co będźie chćiáłá ręká twojá.
Tedy wſtáwƺy Abimelech / y wƺyſtek lud / który z nim był / w nocy / záſádźili śię przećiw Sychem ná cżterech miejſcách.
A wyƺedłƺy Gáál / ſyn Obedów / ſtánął w ſámej brámie miáſtá ; wſtáł też y Abimelech / y lud / który z nim był / z záſádzki.
A widząc Gáál lud / rzekł do Zebulá : Oto / lud idźie z wierzchu gór. Któremu odpowiedźiáł Zebul : Cień góry ty widźiƺ / jákoby ludźie.
Tedy po wtóre rzekł Gáál / mówiąc : Oto / lud zſtępuje z góry á jeden huf idźie drogą równiny Meonenim.
Tedy rzekł do niego Zebul : Gdźież teraz uſtá twoje / które mówiły : Co jeſt Abimelech / ábyśmy mu ſłużyć mieli? izáli to nie ten lud / któryś wzgárdźił? wynijdź teraz / á wálcż przećiw niemu.
A ták wyƺedł Gáál przed mężámi Sychem / á wálcżył przećiw Abimelechowi.
Y gonił go Abimelech / gdy przed nim ućiekáł / á poległo wiele ránnych áż do ſámej brámy.
Y zoſtáł Abimelech w Arumá ; á Zebul wygnáł Gáálá z bráćią jego / áby nie mieƺkáli w Sychem.
Ale názájutrz wyƺedł lud w pole / y powiedźiáno to Abimelechowi.
Wźiąwƺy tedy lud / rozdźielił go ná trzy hufce / y záſádźił śię w polu ; á widząc / á oto lud wychodźił z miáſtá / wypádł ná nie / y pobił je.
Bo Abimelech y hufy / które z nim były / uderzyły ná nie / y ſtánęli u ſámej brámy miáſtá / á inne dwá hufy uderzyły ná wƺyſtkie / którzy byli w polu / y pobili je.
A Abimelech dobywáł miáſtá przez on wƺyſtek dźień / y wźiął je ; á lud / który w nim był / pomordowáł / á zburzywƺy miáſto / pośiáł je ſolą.
A uſłyƺáwƺy wƺyſcy mężowie / którzy byli ná wieży Sychem / przyƺli do twierdzy domu bogá Beryt.
Y opowiedźiáno Abimelechowi / że śię tám zgromádźili wƺyſcy mężowie wieży Sychem.
Tedy wƺedł Abimelech ná górę Sálmon / on y wƺyſtek lud / który był z nim ; á nábráwƺy z ſobą śiekier / ućiął gáłąź z drzewá / á wźiąwƺy ją / włożył ná rámioná ſwoje / y rzekł do ludu / który z nim był : Co widźićie / żem ucżynił / prędko cżyńćie ták / jáko ja.
A ták ućiąwƺy z onego wƺyſtkiego ludu káżdy gáłąź ſwoję / ƺli zá Abimelechem / á kłádli je około twierdzy y ſpálili niemi twierdzą ogniem / y poginęło tám wƺyſtkich mężów wieży Sychemſkiej około tyśiącá mężów y niewiáſt.
Potem ƺedł Abimelech do Tebeſ / á położywƺy śię przećiwko Tebeſ / dobył go.
Ale wieżá byłá mocná w pośrodku miáſtá / ná którą ućiekli wƺyſcy mężowie / y niewiáſty / y wƺyſcy przedniejśi miáſtá / á zámknąwƺy ją zá ſobą / weƺli ná dách wieży.
Tedy przyƺedł Abimelech áż do ſámej wieży / y dobywáł jey / á ſtánąwƺy u ſámych drzwi wieży / chćiáł ją ſpálić ogniem.
Między tem zrzućiłá niewiáſtá niektóra ƺtukę kámieniá od żárn ná głowę Abimelechowę / y rozbiłá wierzch głowy jego.
Który nátychmiáſt záwołáwƺy pácholiká / co nośił broń jego / rzekł do niego : Dobądź miecżá twego / á zábij mię / by ſnáć nie rzecżono o mnie : Niewiáſtá go zábiłá : á ták przebił go pácholik jego / y umárł.
A widząc mężowie Izráelſcy / iż umárł Abimelech / rozeƺli śię káżdy do miejſcá ſwego.
Y oddáł Bóg ono złe Abimelechowi / które ucżynił nád ojcem ſwoim / zábiwƺy śiedmdźieśiąt bráći ſwych.
Y wƺyſtko złe mężów Sychem obróćił Bóg ná głowę ich ; á przyƺło ná nie przeklęſtwo Jotámá / ſyná Jerobáálowego.
