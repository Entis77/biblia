Nádto przyśięgli mężowie Izráelſcy w Máſfá / mówiąc : Żáden z nas nie da córki ſwej Benjámincżykom zá żonę.
A ták poƺedł lud do domu Bożego / y trwáli tám áż do wiecżorá przed Bogiem / á podnióſƺy głos ſwój / płákáli płácżem wielkim.
Y rzekli : O Pánie / Boże Izráelſki / cżemuż śię to ſtáło w Izráelu / że ubyło dźiśiáj z Izráelá jedno pokolenie?
Tedy názájutrz wſtáwƺy ráno lud zbudowáli tám ołtárz / á ſpráwowáli cáłopálone y ſpokojne ofiáry.
Zátem rzekli ſynowie Izráelſcy : Któż jeſt / co nie przyƺedł do zgromádzenia ze wƺyſtkich pokoleń Izráelſkich do Páná? ( Bo śię byli wielką przyśięgą záwiązáli przećiw temu / któryby nie przyƺedł do Páná do Máſfá / mówiąc : Śmierćią umrze.
Y żáłowáli ſynowie Izráelſcy Benjáminá / brátá ſwego / á mówili : Wygłádzone jeſt dźiś pokolenie jedno z Izráelá.
Cóż ucżynimy tym / co pozoſtáli / áby mieli żony / gdyżeśmy przyśięgli przez Páná / że im nie mamy dáć córek náƺych zá żony? )
Rzekli tedy : Jeſtże kto z pokoleń Izráelſkich / coby nie przyƺedł do Páná do Máſfá? á oto / nie przyƺedł był nikt do obozu z Jábes Gáláád do zgromádzenia.
Bo gdy licżono lud / tedy nikogo tám nie było z obywáteli Jábes Gáláád.
Y poſłáło tám zgromádzenie dwánaśćie tyśięcy mężów wálecżnych / rozkázując im y mówiąc : Idźćie / á pobijćie obywátele Jábes Gáláád oſtrzem miecżá / y niewiáſty y dźieći.
A ták ſobie poſtąpićie : Káżdego mężcżyznę / y káżdą niewiáſtę / która mężá uznáłá / zábijećie.
Náleſli tedy z obywáteli Jábes Gáláád cżtery ſtá dźiewecżek / pánien / które nie uznáły mężá obcując z nim / y przywiedli je do obozu do Sylo / które było w źiemi Chánánejſkiej.
Potem poſłáło wƺyſtko zgromádzenie / á mówiło do ſynów Benjáminowych / którzy byli ná ſkále Remmon / y przyzwáli ich w pokoju.
Przetoż wróćił śię Benjámin onego cżáſu / y dáli im żony / które byli żywo záchowáli z niewiáſt Jábes Gáláád / ále im śię ich jeƺcże niedoſtáwáło.
A lud żáłował Benjáminá / iż ucżynił Pán przerwę w pokoleniách Izráelſkich.
Tedy rzekli ſtárśi zgromádzenia tego : A z tymi drugimi cóż ucżynimy / áby mieli żony / gdyż niewiáſty wygłádzone ſą z Benjáminá?
Nádto rzekli : Dźiedźictwo Benjáminá pozoſtáłym náleży / áby nie záginęło pokolenie z Izráelá.
A my nie możemy im dáć żon z córek náƺych / ( gdyż byli przyśięgli ſynowie Izráelſcy / mówiąc : Przeklęty / kto da żonę Benjámincżykowi. )
Potem rzekli : Oto święto Páńſkie urocżyſte bywa ná káżdy rok w Sylo / które jeſt ku północy od Betel / á ná wſchód ſłońcá ku drodze / która idźie od Betel ku Sychem / á pod południá leży ku Lebnie.
A ták rozkázáli ſynom Benjáminowym / mówiąc : Idźćie / á záſádźćie śię w winnicách /
A pátrzájćie / gdy córki Sylo wynijdą gromádą do táńcá ; tám wyƺedƺy z winnic / porwij káżdy z was ſobie żonę z córek Sylo / á potem idźćie do źiemi Benjámin.
A gdy przyjdą ojcowie ich / álbo bráćia ich ſkárżyć śię przed námi / tedy im rzecżemy : Zmiłujćie śię nád nimi dla nas ; bośmy nie wźięli dla káżdego z nich żony ná wojnie / á wyśćie im ich też nie dáli ; przetoż nie jeſteśćie winni.
Tedy ucżynili ták ſynowie Benjámin / y nábráli żon według licżby ſwojey z onych co táńcowáły / które porwáwƺy odeƺli / y wróćili śię do dźiedźictwá ſwego / á pobudowáwƺy miáſtá mieƺkáli w nich.
A ták rozeƺli śię z támtąd ſynowie Izráelſcy onego cżáſu / káżdy do pokolenia ſwego y do domu ſwego / á ƺedł z támtąd káżdy do dźiedźictwá ſwego.
W one dni nie było królá w Izráelu ; káżdy / co mu śię dobrego zdáło / to cżynił.
