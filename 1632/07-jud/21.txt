Nádto przyśięgli mężowie Izráelſcy w Máſfá / mówiąc : Żáden z náſ nie dá córki ſwej Benjámincżykom zá żonę.
A ták poƺedł lud do domu Bożego / y trwáli tám áż do wiecżorá przed Bogiem / á podnióſłƺy głoſ ſwój / płákáli płácżem wielkim.
Y rzekli : O Pánie / Boże Izráelſki / cżemuż śię to ſtáło w Izráelu / że ubyło dźiśiáj z Izráelá jedno pokolenie?
Tedy názájutrz wſtáwƺy ráno lud zbudowáli tám ołtárz / á ſpráwowáli cáłopálone y ſpokojne ofiáry.
Zátem rzekli ſynowie Izráelſcy : Któż jeſt / co nie przyƺedł do zgromádzeniá ze wƺyſtkich pokoleń Izráelſkich do Páná? ( Bo śię byli wielką przyśięgą záwiązáli przećiw temu / któryby nie przyƺedł do Páná do Máſfá / mówiąc : Śmierćią umrze.
Y żáłowáli ſynowie Izráelſcy Benjáminá / brátá ſwego / á mówili : Wygłádzone jeſt dźiś pokolenie jedno z Izráelá.
Cóż ucżynimy tym / co pozoſtáli / áby mieli żony / gdyżeśmy przyśięgli przez Páná / że im nie mámy dáć córek náƺych zá żony? )
Rzekli tedy : Jeſtże kto z pokoleń Izráelſkich / coby nie przyƺedł do Páná do Máſfá? á oto / nie przyƺedł był nikt do obozu z Jábeſ Gáláád do zgromádzeniá.
Bo gdy licżono lud / tedy nikogo tám nie było z obywáteli Jábeſ Gáláád.
Y poſłáło tám zgromádzenie dwánáśćie tyśięcy mężów wálecżnych / rozkázując im y mówiąc : Idźćie / á pobijćie obywátele Jábeſ Gáláád oſtrzem miecżá / y niewiáſty y dźieći.
A ták ſobie poſtąpićie : Káżdego mężcżyznę / y káżdą niewiáſtę / którá mężá uznáłá / zábijećie.
Náleźli tedy z obywáteli Jábeſ Gáláád cżtery ſtá dźiewecżek / pánien / które nie uznáły mężá obcując z nim / y przywiedli je do obozu do Sylo / które było w źiemi Chánánejſkiej.
Potem poſłáło wƺyſtko zgromádzenie / á mówiło do ſynów Benjáminowych / którzy byli ná ſkále Remmon / y przyzwáli ich w pokoju.
Przetoż wróćił śię Benjámin onego cżáſu / y dáli im żony / które byli żywo záchowáli z niewiáſt Jábeſ Gáláád / ále im śię ich jeƺcże niedoſtáwáło.
A lud żáłowáł Benjáminá / iż ucżynił Pán przerwę w pokoleniách Izráelſkich.
Tedy rzekli ſtárśi zgromádzeniá tego : A z tymi drugimi cóż ucżynimy / áby mieli żony / gdyż niewiáſty wygłádzone ſą z Benjáminá?
Nádto rzekli : Dźiedźictwo Benjáminá pozoſtáłym náleży / áby nie záginęło pokolenie z Izráelá.
A my nie możemy im dáć żon z córek náƺych / ( gdyż byli przyśięgli ſynowie Izráelſcy / mówiąc : Przeklęty / kto dá żonę Benjámincżykowi. )
Potem rzekli : Oto święto Páńſkie urocżyſte bywá ná káżdy rok w Sylo / które jeſt ku północy od Betel / á ná wſchód ſłońcá ku drodze / którá idźie od Betel ku Sychem / á pod południá leży ku Lebnie.
A ták rozkázáli ſynom Benjáminowym / mówiąc : Idźćie / á záſádźćie śię w winnicách /
A pátrzájćie / gdy córki Sylo wynijdą gromádą do táńcá ; tám wyƺedłƺy z winnic / porwij káżdy z wáſ ſobie żonę z córek Sylo / á potem idźćie do źiemi Benjámin.
A gdy przyjdą ojcowie ich / álbo bráćiá ich ſkárżyć śię przed námi / tedy im rzecżemy : Zmiłujćie śię nád nimi dlá náſ ; bośmy nie wźięli dlá káżdego z nich żony ná wojnie / á wyśćie im ich też nie dáli ; przetoż nie jeſteśćie winni.
Tedy ucżynili ták ſynowie Benjámin / y nábráli żon według licżby ſwojej z onych co táńcowáły / które porwáwƺy odeƺli / y wróćili śię do dźiedźictwá ſwego / á pobudowáwƺy miáſtá mieƺkáli w nich.
A ták rozeƺli śię ſtámtąd ſynowie Izráelſcy onego cżáſu / káżdy do pokoleniá ſwego y do domu ſwego / á ƺedł ſtámtąd káżdy do dźiedźictwá ſwego.
W one dni nie było królá w Izráelu ; káżdy / co mu śię dobrego zdáło / to cżynił.
