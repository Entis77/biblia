Potem cżynili ſynowie Izráelſcy złe przed ocżymá Páńſkimi / y podał je Pán w ręce Mádyjánitów przez śiedm lat.
A zmocniłá śię ręká Mádyjánitów nád Izráelem / ták iż przed Mádyjánitámi kopáli ſobie ſynowie Izráelſcy lochy / które były w górách / y jáſkinie y twierdze.
A bywáło / gdy cżego náśiał Izráel / że przychodźił Mádyján y Amálek / y ludźie ze wſchodu ſłońcá / á nájeżdżáli go ;
Y położywƺy śię obozem przećiwko nim / pſowáli zbożá źiemi / áż gdźie chodzą do Gázy / nic nie zoſtáwując ná pożywienie Izráelcżykom / áni owiec / áni wołów / áni oſłów.
Abowiem oni y ſtádá ich przyćiągáli / y námioty ich / á przychodźili jáko ƺáráńcżá w mnóſtwie / y nie było im y wielbłądom ich licżby ; ták przychodząc do źiemi ſpuſtoƺyli ją.
Tedy znędzony był Izráel bárzo od Mádyjánitów / y wołáli ſynowie Izráelſcy do Páná.
A gdy wołáli ſynowie Izráelſcy do Páná z przycżyny Mádyjánitów /
Poſłał Pán mężá proroká do ſynów Izráelſkich / y mówił do nich : Ták mówi Pán / Bóg Izráelſki : Jam was wywiódł z źiemi Egipſkiej / á wywiodłem was z domu niewoli.
Y wyrwáłem was z ręki Egipcżánów / y z ręki wƺyſtkich / którzy was trápili / którem przed wámi wygnał / y dáłem wam źiemię ich ;
A powiedźiáłem wam : Jam Pán / Bóg wáƺ / nie bójćież śię bogów Amorejſkich / w których źiemi wy mieƺkáćie ; áleśćie nie uſłucháli głoſu mego.
Przyƺedł potem Anioł Páńſki y ſtánął pod dębem / który był w Efrá / w dźiedźictwie Joáſá / ojcá Eſrowego. A Giedeon / ſyn jego / młóćił zboże ná bojewiſku / áby z nim ućiekł przed Mádyjánitámi.
Tedy mu śię ukázał Anioł Páńſki / y rzekł do niego : Pán z tobą / mężu wálecżny.
Y odpowiedźiał mu Giedeon : Proƺę Pánie mój / jeſli Pán jeſt z námi / á cżemuż ná nas przyƺło to wƺyſtko? gdźież teraz ſą wƺyſtkie cudá jego / które nam opowiádáli ojcowie náśi / mówiąc : Izali z Egiptu nie wywiódł nas Pán? á teraz opuśćił nas Pán / y podał nas w ręce Mádyjánitów.
Tedy wejrzáwƺy nań Pán rzekł : Idźże z tą twoją mocą / á wybáwiƺ Izráelá z ręki Mádyjáńcżyków ; izálim ćię nie poſłał?
A on rzekł do niego : Proƺę Pánie mój / cżemże wybáwię Izráelá? oto náród mój podły jeſt w Mánáſe / á jam nájmniejƺy w domu ojcá mego.
Y rzekł do niego Pán : Poniewáż Ja będę z tobą / przetoż poráźiƺ Mádyjánity / jáko mężá jednego.
A on mu odpowiedźiał : Jeſliżem proƺę ználázł łáſkę przed ocżymá twemi / dáj mi znák / że ty mówiƺ ze mną.
Nie odchodź proƺę ſtąd / áż záś przydę do ćiebie / á przynioſęć ofiárę moję / y położę ją przed tobą. Y odpowiedźiał : Ja pocżekám / áż śię wróćiƺ.
Odƺedƺy tedy Giedeon zgotował koźlątko z ſtádá á z miáry mąki przáśne chleby / á mięſo włożył w koƺ / á polewkę mięſną wlał w gárnek / y przynióſł to do niego pod dąb / y ofiárował.
Y rzekł do niego Anioł Boży : Weźmij to mięſo y te chleby niekwáƺone / á połóż ná oney ſkále polewką poláwƺy ; y ucżynił ták.
Zátem śćiągnął Anioł Páńſki koniec láſki / którą miał w ręce ſwojey / y dotknął śię mięſá y przáśników / y wyƺedł ogień z ſkáły / á ſpálił mięſo y chleby przáśne ; á miedzy tym Anioł Páńſki odƺedł od ocżu jego.
A widząc Giedeon / iż to był Anioł Páńſki / rzekł : Ach / Pánie Boże / cżemużem widźiał Aniołá Páńſkiego twárzą w twárz?
Y rzekł mu Pán : Pokój z tobą ; nie bój śię / nie umrzeƺ.
Przetoż zbudował tám Giedeon ołtárz Pánu / y názwał go : Pán pokoju ; áż do dniá tego ten jeƺcże jeſt w Efráćie / ojcá Eſrowego.
Y ſtáło śię oney nocy / że mu rzekł Pán : Weźmij ćielcá doroſłego / który jeſt ojcá twego / tego ćielcá drugiego śiedmioletniego / á rozwál ołtárz Báálów / który jeſt ojcá twego / y gáj / który jeſt około niego / wyśiecż ;
A zbuduj ołtárz Pánu / Bogu twemu / ná wierzchu tey ſkáły ná równinie / á weźmij tego ćielcá drugiego / y ucżyń z niego cáłopálenie przy drwách z gáju / który wyśiecżeƺ.
Wźiąwƺy tedy Giedeon dźieśięć mężów z ſług ſwojich / ucżynił jáko mu rozkázał Pán ; á iż śię bał domu ojcá ſwego y mężów miáſtá / nie ucżynił tego we dnie / ále ucżynił w nocy.
A gdy wſtáli mężowie miáſtá ráno / ujrzeli rozwálony ołtárz Báálów / y gáj / który był podle niego / wyrąbány / y ćielcá onego drugiego ofiárowánego ná cáłopálenie ná ołtárzu zbudowánym.
Zátem rzekł jeden do drugiego : Któż to wżdy ucżynił? A gdy śię pytáli y dowiádowáli / powiedźiáno : Giedeon / ſyn Joáſów / ucżynił to.
Tedy rzekli mężowie miáſtá do Joáſá : Wywiedź ſyná twego / niech umrze / iż rozrzućił ołtárz Báálów / á iż wyćiął gáj / który był około niego.
Y odpowiedźiał Joás wƺyſtkim / którzy ſtáli około niego : A wyż śię to ſwárzyć máćie o Báálá? Izali wy go wybáwićie? Kto by śię oń záſtáwiał / niech umrze tegoż poránku ; jeſli bogiem jeſt / niech śię mśći tego / że rozwálono ołtárz jego.
Y názwano go onegoż dniá Jerubáál / mówiąc : Niech śię mśći nád nim Báál / iż rozwálił ołtárz jego.
Tedy wƺyſcy Mádyjánitowie / y Amálekitowie / y ludźie od wſchodu ſłońcá zebráli śię weſpół / á przepráwiwƺy śię przez Jordán / położyli śię obozem w dolinie Jezreel.
Ale Duch Páńſki przyoblókł Giedeoná / który zátrąbiwƺy w trąbę zwołał domu Abiezerowego do śiebie.
Y wypráwił poſły do wƺyſtkiego pokolenia Mánáſeſowego / y zebráli śię do niego ; poſły też poſłał do Aſerá / y do Zábuloná / y do Neftálimá / y zájecháli im.
Tedy rzekł Giedeon do Bogá : Jeſli wybáwiƺ przez rękę moję Izráelá / jákoś powiedźiał /
Oto / ja położę runo wełny ná bojewiſku ; jeſliż roſá tylko ná runo upádnie / á wƺyſtká źiemiá ſuchá będźie / tedy będę wiedźiał / iż wybáwiƺ przez rękę moję Izráelá / jákoś powiedźiał.
Y ſtáło śię ták ; bo wſtáwƺy názájutrz / śćiſnął runo / y wyżdżął roſy z runá pełną cżáƺę wody.
Nádto rzekł Giedeon do Bogá : Niech śię nie wzruƺá gniew twój przećiwko mnie / że przemówię jeƺcże raz. Niech doświádcżę proƺę jeƺcże raz ná tym runie ; niech będźie / proƺę / ſuche ſámo runo tylko / á ná wƺyſtkiej źiemi niech będźie roſá.
Y ucżynił ták Bóg oney nocy / że było ſámo runo ſuche / á ná wƺyſtkiej źiemi byłá roſá.
