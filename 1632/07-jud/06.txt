Potem cżynili ſynowie Izráelſcy złe przed ocżymá Páńſkimi / y podáł je Pán w ręce Mádyjánitów przez śiedm lát.
A zmocniłá śię ręká Mádyjánitów nád Izráelem / ták iż przed Mádyjánitámi kopáli ſobie ſynowie Izráelſcy lochy / które były w górách / y jáſkinie y twierdze.
A bywáło / gdy cżego náśiáł Izráel / że przychodźił Mádyján y Amálek / y ludźie ze wſchodu ſłońcá / á nájeżdżáli go ;
Y położywƺy śię obozem przećiwko nim / pſowáli zbożá źiemi / áż gdźie chodzą do Gázy / nic nie zoſtáwując ná pożywienie Izráelcżykom / áni owiec / áni wołów / áni oſłów.
Abowiem oni y ſtádá ich przyćiągáli / y námioty ich / á przychodźili jáko ƺáráńcżá w mnóſtwie / y nie było im y wielbłądom ich licżby ; ták przychodząc do źiemi ſpuſtoƺyli ją.
Tedy znędzony był Izráel bárdzo od Mádyjánitów / y wołáli ſynowie Izráelſcy do Páná.
A gdy wołáli ſynowie Izráelſcy do Páná z przycżyny Mádyjánitów /
Poſłáł Pán mężá proroká do ſynów Izráelſkich / y mówił do nich : Ták mówi Pán / Bóg Izráelſki : Jám wáſ wywiódł z źiemi Egipſkiej / á wywiodłem wáſ z domu niewoli.
Y wyrwáłem wáſ z ręki Egipcżánów / y z ręki wƺyſtkich / którzy wáſ trápili / którem przed wámi wygnáł / y dáłem wám źiemię ich ;
A powiedźiáłem wám : Jám Pán / Bóg wáƺ / nie bójćież śię bogów Amorejſkich / w których źiemi wy mieƺkáćie ; áleśćie nie uſłucháli głoſu mego.
Przyƺedł potem Anioł Páńſki y ſtánął pod dębem / który był w Efrá / w dźiedźictwie Joáſá / ojcá Eſrowego. A Giedeon / ſyn jego / młóćił zboże ná bojewiſku / áby z nim ućiekł przed Mádyjánitámi.
Tedy mu śię ukázáł Anioł Páńſki / y rzekł do niego : Pán z tobą / mężu wálecżny.
Y odpowiedźiáł mu Giedeon : Proƺę Pánie mój / jeźli Pán jeſt z námi / á cżemuż ná náſ przyƺło to wƺyſtko? gdźież teráz ſą wƺyſtkie cudá jego / które nám opowiádáli ojcowie náśi / mówiąc : Izáli z Egiptu nie wywiódł náſ Pán? á teráz opuśćił náſ Pán / y podáł náſ w ręce Mádyjánitów.
Tedy wejrzáwƺy náń Pán rzekł : Idźże z tą twoją mocą / á wybáwiƺ Izráelá z ręki Mádyjáńcżyków ; izálim ćię nie poſłáł?
A on rzekł do niego : Proƺę Pánie mój / cżemże wybáwię Izráelá? oto náród mój podły jeſt w Mánáſe / á jám nájmniejƺy w domu ojcá mego.
Y rzekł do niego Pán : Poniewáż Já będę z tobą / przetoż poráźiƺ Mádyjánity / jáko mężá jednego.
A on mu odpowiedźiáł : Jeźliżem proƺę ználázł łáſkę przed ocżymá twemi / dáj mi znák / że ty mówiƺ ze mną.
Nie odchodź proƺę ſtąd / áż záś przyjdę do ćiebie / á przynioſęć ofiárę moję / y położę ją przed tobą. Y odpowiedźiáł : Já pocżekám / áż śię wróćiƺ.
Odƺedłƺy tedy Giedeon zgotowáł koźlątko z ſtádá á z miáry mąki przáśne chleby / á mięſo włożył w koƺ / á polewkę mięſną wláł w gárnek / y przynióſł to do niego pod dąb / y ofiárowáł.
Y rzekł do niego Anioł Boży : Weźmij to mięſo y te chleby niekwáƺone / á połóż ná onej ſkále polewką poláwƺy ; y ucżynił ták.
Zátem śćiągnął Anioł Páńſki koniec láſki / którą miáł w ręce ſwojej / y dotknął śię mięſá y przáśników / y wyƺedł ogień z ſkáły / á ſpálił mięſo y chleby przáśne ; á między tem Anioł Páńſki odƺedł od ocżu jego.
A widząc Giedeon / iż to był Anioł Páńſki / rzekł : Ach / Pánie Boże / cżemużem widźiáł Aniołá Páńſkiego twárzą w twárz?
Y rzekł mu Pán : Pokój z tobą ; nie bój śię / nie umrzeƺ.
Przetoż zbudowáł tám Giedeon ołtárz Pánu / y názwáł go : Pán pokoju ; áż do dniá tego ten jeƺcże jeſt w Efráćie / ojcá Eſrowego.
Y ſtáło śię onej nocy / że mu rzekł Pán : Weźmij ćielcá doroſłego / który jeſt ojcá twego / tego ćielcá drugiego śiedmioletniego / á rozwál ołtárz Báálów / który jeſt ojcá twego / y gáj / który jeſt około niego / wyśiecż ;
A zbuduj ołtárz Pánu / Bogu twemu / ná wierzchu tej ſkáły ná równinie / á weźmij tego ćielcá drugiego / y ucżyń z niego cáłopálenie przy drwách z gáju / który wyśiecżeƺ.
Wźiąwƺy tedy Giedeon dźieśięć mężów z ſług ſwoich / ucżynił jáko mu rozkázáł Pán ; á iż śię báł domu ojcá ſwego y mężów miáſtá / nie ucżynił tego we dnie / ále ucżynił w nocy.
A gdy wſtáli mężowie miáſtá ráno / ujrzeli rozwálony ołtárz Báálów / y gáj / który był podle niego / wyrąbány / y ćielcá onego drugiego ofiárowánego ná cáłopálenie ná ołtárzu zbudowánym.
Zátem rzekł jeden do drugiego : Któż to wżdy ucżynił? A gdy śię pytáli y dowiádowáli / powiedźiáno : Giedeon / ſyn Joáſów / ucżynił to.
Tedy rzekli mężowie miáſtá do Joáſá : Wywiedź ſyná twego / niech umrze / iż rozrzućił ołtárz Báálów / á iż wyćiął gáj / który był około niego.
Y odpowiedźiáł Joáſ wƺyſtkim / którzy ſtáli około niego : A wyż śię to ſwárzyć máćie o Báálá? Izáli wy go wybáwićie? Kto by śię oń záſtáwiáł / niech umrze tegoż poránku ; jeźli bogiem jeſt / niech śię mśći tego / że rozwálono ołtárz jego.
Y názwáno go onegoż dniá Jerubáál / mówiąc : Niech śię mśći nád nim Báál / iż rozwálił ołtárz jego.
Tedy wƺyſcy Mádyjánitowie / y Amálekitowie / y ludźie od wſchodu ſłońcá zebráli śię weſpół / á przepráwiwƺy śię przez Jordán / położyli śię obozem w dolinie Jezreel.
Ale Duch Páńſki przyoblókł Giedeoná / który zátrąbiwƺy w trąbę zwołáł domu Abiezerowego do śiebie.
Y wypráwił poſły do wƺyſtkiego pokoleniá Mánáſeſowego / y zebráli śię do niego ; poſły też poſłáł do Aſerá / y do Zábuloná / y do Neftálimá / y zájecháli im.
Tedy rzekł Giedeon do Bogá : Jeźli wybáwiƺ przez rękę moję Izráelá / jákoś powiedźiáł /
Oto / já położę runo wełny ná bojewiſku ; jeźliż roſá tylko ná runo upádnie / á wƺyſtká źiemiá ſuchá będźie / tedy będę wiedźiáł / iż wybáwiƺ przez rękę moję Izráelá / jákoś powiedźiáł.
Y ſtáło śię ták ; bo wſtáwƺy názájutrz / śćiſnął runo / y wyżdżął roſy z runá pełną cżáƺę wody.
Nádto rzekł Giedeon do Bogá : Niech śię nie wzruƺá gniew twój przećiwko mnie / że przemówię jeƺcże ráz. Niech doświádcżę proƺę jeƺcże ráz ná tem runie ; niech będźie / proƺę / ſuche ſámo runo tylko / á ná wƺyſtkiej źiemi niech będźie roſá.
Y ucżynił ták Bóg onej nocy / że było ſámo runo ſuche / á ná wƺyſtkiej źiemi byłá roſá.
