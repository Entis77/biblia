Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi / y podáł je Pán w ręce Filiſtynów przez cżterdzieśći lát.
Tedy był mąż niektóry z Sáráá / z pokoleniá Dán / imieniem Mánue / á żoná jego byłá niepłodną / y nie rodziłá.
Y ukázáł śię Anioł Páńſki onej niewieśćie / á rzekł do niej : Otoś teráz niepłodną / ániś rodziłá ; ále pocżnieƺ y porodziƺ ſyná.
Przetoż śię teráz ſtrzeż / ábyś nie piłá winá / y nápoju mocnego / y ábyś nie jádłá nic niecżyſtego ;
Bo oto pocżnieƺ y porodziƺ ſyná / á brzytwá nie poſtoi ná głowie jego / bo Názárejcżykiem Bożym będzie to dziećię záráz z żywotá ; á on pocżnie wybáwiáć Izráelá z ręki Filiſtynów.
Tedy przyƺłá niewiáſtá / y powiedziáłá to mężowi ſwemu / mówiąc : Mąż Boży przyƺedł do mnie / którego oblicże było jáko oblicże Aniołá Bożego / bárdzo ſtráƺne / y nie pytáłám go / ſkąd był / áni mi imieniá ſwego oznájmił.
Tylko mi rzekł : Oto / pocżnieƺ y porodziƺ ſyná ; przetoż teráz nie pij winá / áni nápoju mocnego / áni jedz co niecżyſtego ; bo Názárejcżykiem Bożym będzie to dziećię záráz z żywotá áż do dniá śmierći ſwojej.
Tedy śię modlił Mánue Pánu / mówiąc : Proƺę Pánie mój / mąż Boży / któregoś poſłáł / niech przyjdzie proƺę znowu do náſ / á náucży náſ / co cżynić mámy z dziećięćiem / które śię národzi?
Y wyſłucháł Bóg głoſ Mánuego ; bo przyƺedł Anioł Boży znowu do niewiáſty onej / gdy śiedziáłá ná polu ; ále Mánue / mąż jej / nie był z nią.
Tedy kwápiąc śię oná niewiáſtá / bieżáłá / y opowiedziáłá mężowi ſwemu / y rzekłá mu : Oto mi śię ukázáł mąż on / który był przyƺedł przedtem do mnie.
A wſtáwƺy Mánue ƺedł zá żoną ſwoją ; á przyƺedłƺy do onego mężá / rzekł mu : Tyżeś jeſt ten mąż / któryś mówił z żoną moją? A on rzekł : Jám jeſt.
Y rzekł Mánue : Niech śię teráz ſpełni ſłowo twoje ; ále cóż będzie zá obycżáj dziećięćiá / y co zá ſpráwá jego?
Y odpowiedziáł Anioł Páńſki Mánuemu : Wƺyſtkiego / com powiedziáł żonie twojej / niech śię ſtrzeże.
Żádnej rzecży / którá pochodzi z winnej máćicy / niecháj nie je ; tákże winá áni nápoju mocnego / niech nie pije / áni żádnej rzecży niecżyſtej niech nie je / á com jej kolwiek przykázáł / tego niech przeſtrzegá.
Tedy rzekł Mánue do Aniołá Páńſkiego : Dáj śię proƺę zátrzymáć / á nágotujemy przed ćię koźlątko z ſtádá.
Ale Anioł Páńſki odpowiedziáł Mánuemu : Choćbyś mię zátrzymáł / nie będę jádł chlebá twego ; ále jeźli będzieƺ chćiáł ſpráwić cáłopálenie / ofiárujże je Pánu ; bo nie wiedziáł Mánue / żeby on był Anioł Páńſki.
Tedy rzekł Mánue do Aniołá Páńſkiego : Cóż zá imię twoje? ábyśmy / gdy śię ſpełni ſłowo twoje / ucżćili ćię.
Któremu odpowiedziáł Anioł Páńſki : Przecżże pytáƺ o imię moje / które jeſt dziwne?
Wziął tedy Mánue koźlę z ſtádá / y ofiárę śniedną / y ofiárowáł to ná opoce Pánu / y ucżynił cud / á Mánue y żoná jego pátrzyli ná to.
A gdy wſtępowáł płomień z ołtárzá ku niebu / tedy wſtąpił Anioł Páńſki w płomieniu ołtárzowym / á Mánue / y żoná jego widząc to / upádli ná twárze ſwe ná ziemię.
A potem nie ukázáł śię więcej Anioł Páńſki Mánuemu / áni żonie jego ; y poznáł Mánue / że to był Anioł Páńſki.
Y rzekł Mánue do żony ſwojej : Koniecżnie pomrzemy / bośmy Bogá widzieli.
Któremu odpowiedziáłá żoná jego : Gdyby náſ chćiáł Pán zábić / nie przyjąłby z rąk náƺych cáłopáleniá / y ofiáry śniednej / ániby nám był okázáł tego wƺyſtkiego / ániby nám ná ten cżáſ był objáwił tákowych rzecży.
Porodziłá tedy oná niewiáſtá ſyná / y názwáłá imię jego Sámſon ; y roſło dziećię / á błogoſłáwił mu Pán.
Y pocżął go Duch Páńſki umácniáć w obozie Dán między Sáráá y między Eſtáol.
