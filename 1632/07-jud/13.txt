Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi / y podał je Pán w ręce Filiſtynów przez cżterdźieśći lat.
Tedy był mąż niektóry z Sáráá / z pokolenia Dán / imieniem Mánue / á żoná jego byłá niepłodną / y nie rodźiłá.
Y ukázał śię Anioł Páńſki oney niewieśćie / á rzekł do niej : Otoś teraz niepłodną / ániś rodźiłá ; ále pocżnieƺ y porodźiƺ ſyná.
Przetoż śię teraz ſtrzeż / ábyś nie piłá winá / y nápoju mocnego / y ábyś nie jádłá nic niecżyſtego ;
Bo oto pocżnieƺ y porodźiƺ ſyná / á brzytwá nie poſtoi ná głowie jego / bo Názárejcżykiem Bożym będźie to dźiećię záráz z żywotá ; á on pocżnie wybáwiáć Izráelá z ręki Filiſtynów.
Tedy przyƺłá niewiáſtá / y powiedźiáłá to mężowi ſwemu / mówiąc : Mąż Boży przyƺedł do mnie / którego oblicże było jáko oblicże Aniołá Bożego / bárzo ſtráƺne / y nie pytáłám go / ſkąd był / áni mi imieniá ſwego oznájmił.
Tylko mi rzekł : Oto / pocżnieƺ y porodźiƺ ſyná ; przetoż teraz nie pij winá / áni nápoju mocnego / áni jedz co niecżyſtego ; bo Názárejcżykiem Bożym będźie to dźiećię záráz z żywotá áż do dniá śmierći ſwojey.
Tedy śię modlił Mánue Pánu / mówiąc : Proƺę Pánie mój / mąż Boży / któregoś poſłał / niech przydźie proƺę znowu do nas / á náucży nas / co cżynić mamy z dźiećięćiem / które śię národźi?
Y wyſłuchał Bóg głos Mánuego ; bo przyƺedł Anioł Boży znowu do niewiáſty oney / gdy śiedźiáłá ná polu ; ále Mánue / mąż jey / nie był z nią.
Tedy kwápiąc śię oná niewiáſtá / bieżáłá / y opowiedźiáłá mężowi ſwemu / y rzekłá mu : Oto mi śię ukázał mąż on / który był przyƺedł przedtem do mnie.
A wſtáwƺy Mánue ƺedł zá żoną ſwoją ; á przyƺedƺy do onego mężá / rzekł mu : Tyżeś jeſt ten mąż / któryś mówił z żoną moją? A on rzekł : Jam jeſt.
Y rzekł Mánue : Niech śię teraz ſpełni ſłowo twoje ; ále cóż będźie zá obycżaj dźiećięćiá / y co zá ſpráwá jego?
Y odpowiedźiał Anioł Páńſki Mánuemu : Wƺyſtkiego / com powiedźiał żonie twojey / niech śię ſtrzeże.
Żádney rzecży / która pochodźi z winney máćicy / niecháj nie je ; tákże winá áni nápoju mocnego / niech nie pije / áni żádney rzecży niecżyſtey niech nie je / á com jey kolwiek przykázał / tego niech przeſtrzegá.
Tedy rzekł Mánue do Aniołá Páńſkiego : Dáj śię proƺę zátrzymáć / á nágotujemy przed ćię koźlątko z ſtádá.
Ale Anioł Páńſki odpowiedźiał Mánuemu : Choćbyś mię zátrzymał / nie będę jádł chlebá twego ; ále jeſli będźieƺ chćiał ſpráwić cáłopálenie / ofiárujże je Pánu ; bo nie wiedźiał Mánue / żeby on był Anioł Páńſki.
Tedy rzekł Mánue do Aniołá Páńſkiego : Cóż zá imię twoje? ábyſmy / gdy śię ſpełni ſłowo twoje / ucżćili ćię.
Któremu odpowiedźiał Anioł Páńſki : Przecżże pytáƺ o imię moje / które jeſt dźiwne?
Wźiął tedy Mánue koźlę z ſtádá / y ofiárę śniedną / y ofiárował to ná opoce Pánu / y ucżynił cud / á Mánue y żoná jego pátrzyli ná to.
A gdy wſtępował płomień z ołtárzá ku niebu / tedy wſtąpił Anioł Páńſki w płomieniu ołtárzowym / á Mánue / y żoná jego widząc to / upádli ná twárze ſwe ná źiemię.
A potem nie ukázał śię więcey Anioł Páńſki Mánuemu / áni żonie jego ; y poznał Mánue / że to był Anioł Páńſki.
Y rzekł Mánue do żony ſwojey : Koniecżnie pomrzemy / bośmy Bogá widźieli.
Któremu odpowiedźiáłá żoná jego : Gdyby nas chćiał Pán zábić / nie przyjąłby z rąk náƺych cáłopalenia / y ofiáry śniedney / ániby nam był okázał tego wƺyſtkiego / ániby nam ná ten cżás był objáwił tákowych rzecży.
Porodźiłá tedy oná niewiáſtá ſyná / y názwáłá imię jego Sámſon ; y roſło dźiećię / á błogoſłáwił mu Pán.
Y pocżął go Duch Páńſki umácniáć w oboźie Dán miedzy Sáráá y miedzy Eſtáol.
