Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi / y podáł je Pán w ręce Filiſtynów przez cżterdźieśći lát.
Tedy był mąż niektóry z Sáráá / z pokoleniá Dán / imieniem Mánue / á żoná jego byłá niepłodną / y nie rodźiłá.
Y ukázáł śię Anioł Páńſki oney niewieśćie / á rzekł do niej : Otoś teraz niepłodną / ániś rodźiłá ; ále pocżnieƺ y porodźiƺ ſyná.
Przetoż śię teraz ſtrzeż / ábyś nie piłá winá / y nápoju mocnego / y ábyś nie jádłá nic niecżyſtego ;
Bo oto pocżnieƺ y porodźiƺ ſyná / á brzytwá nie poſtoi ná głowie jego / bo Názárejcżykiem Bożym będźie to dźiećię záráz z żywotá ; á on pocżnie wybáwiáć Izráelá z ręki Filiſtynów.
Tedy przyƺłá niewiáſtá / y powiedźiáłá to mężowi ſwemu / mówiąc : Mąż Boży przyƺedł do mnie / którego oblicże było jáko oblicże Aniołá Bożego / bárdzo ſtráƺne / y nie pytáłám go / ſkąd był / áni mi imieniá ſwego oznájmił.
Tylko mi rzekł : Oto / pocżnieƺ y porodźiƺ ſyná ; przetoż teraz nie pij winá / áni nápoju mocnego / áni jedz co niecżyſtego ; bo Názárejcżykiem Bożym będźie to dźiećię záráz z żywotá áż do dniá śmierći ſwojey.
Tedy śię modlił Mánue Pánu / mówiąc : Proƺę Pánie mój / mąż Boży / któregoś poſłáł / niech przyjdźie proƺę znowu do nas / á náucży nas / co cżynić mámy z dźiećięćiem / które śię národźi?
Y wyſłucháł Bóg głoſ Mánuego ; bo przyƺedł Anioł Boży znowu do niewiáſty oney / gdy śiedźiáłá ná polu ; ále Mánue / mąż jey / nie był z nią.
Tedy kwápiąc śię oná niewiáſtá / bieżáłá / y opowiedźiáłá mężowi ſwemu / y rzekłá mu : Oto mi śię ukázáł mąż on / który był przyƺedł przedtem do mnie.
A wſtáwƺy Mánue ƺedł zá żoną ſwoją ; á przyƺedłƺy do onego mężá / rzekł mu : Tyżeś jeſt ten mąż / któryś mówił z żoną moją? A on rzekł : Jám jeſt.
Y rzekł Mánue : Niech śię teraz ſpełni ſłowo twoje ; ále cóż będźie zá obycżáj dźiećięćiá / y co zá ſpráwá jego?
Y odpowiedźiáł Anioł Páńſki Mánuemu : Wƺyſtkiego / com powiedźiáł żonie twojey / niech śię ſtrzeże.
Żádney rzecży / która pochodźi z winney máćicy / niecháj nie je ; tákże winá áni nápoju mocnego / niech nie pije / áni żádney rzecży niecżyſtey niech nie je / á com jey kolwiek przykázáł / tego niech przeſtrzegá.
Tedy rzekł Mánue do Aniołá Páńſkiego : Dáj śię proƺę zátrzymáć / á nágotujemy przed ćię koźlątko z ſtádá.
Ale Anioł Páńſki odpowiedźiáł Mánuemu : Choćbyś mię zátrzymáł / nie będę jádł chlebá twego ; ále jeſli będźieƺ chćiáł ſpráwić cáłopálenie / ofiárujże je Pánu ; bo nie wiedźiáł Mánue / żeby on był Anioł Páńſki.
Tedy rzekł Mánue do Aniołá Páńſkiego : Cóż zá imię twoje? ábyśmy / gdy śię ſpełni ſłowo twoje / ucżćili ćię.
Któremu odpowiedźiáł Anioł Páńſki : Przecżże pytáƺ o imię moje / które jeſt dźiwne?
Wźiął tedy Mánue koźlę z ſtádá / y ofiárę śniedną / y ofiárowáł to ná opoce Pánu / y ucżynił cud / á Mánue y żoná jego pátrzyli ná to.
A gdy wſtępowáł płomień z ołtárzá ku niebu / tedy wſtąpił Anioł Páńſki w płomieniu ołtárzowym / á Mánue / y żoná jego widząc to / upádli ná twárze ſwe ná źiemię.
A potem nie ukázáł śię więcey Anioł Páńſki Mánuemu / áni żonie jego ; y poznał Mánue / że to był Anioł Páńſki.
Y rzekł Mánue do żony ſwojey : Koniecżnie pomrzemy / bośmy Bogá widźieli.
Któremu odpowiedźiáłá żoná jego : Gdyby nas chćiáł Pán zábić / nie przyjąłby z rąk náƺych cáłopáleniá / y ofiáry śniedney / ániby nám był okázáł tego wƺyſtkiego / ániby nám ná ten cżáſ był objáwił tákowych rzecży.
Porodźiłá tedy oná niewiáſtá ſyná / y názwáłá imię jego Sámſon ; y roſło dźiećię / á błogoſłáwił mu Pán.
Y pocżął go Duch Páńſki umácniáć w oboźie Dán między Sáráá y między Eſtáol.
