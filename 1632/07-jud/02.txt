Y przyƺedł Anioł Páńſki z Gálgál do Bochym / mówiąc : Wywiodłem wáſ z Egiptu / y wywiodłem wáſ do ziemi / o którąm przyśiągł ojcom wáƺym / y mówiłem : Nie wzruƺę przymierzá mego z wámi ná wieki ;
Jedno wy nie wchodźćie w przymierze z mieƺkájącymi w tej ziemi / owƺem ołtárze ich porozwálájćie ; áleśćie nie ſłucháli głoſu mojego. Przecżżeśćie to ucżynili?
Przetożem ták rzekł : Nie wypędzę ich od oblicżnośći wáƺej ; ále będą wám jáko ćiernie ná boki / á bogowie ich będą wám jáko śidło.
Y ſtáło śię / gdy mówił Anioł Páńſki te ſłowá do wƺyſtkich ſynów Izráelſkich / że lud podnióſł głoſ ſwój / y płákáli.
Y názwáli imię miejſcá onego Bochym / á támże ofiárowáli Pánu.
A Jozue rozpuśćił był lud / y rozeƺli śię ſynowie Izráelſcy káżdy do dziedzictwá ſwego / áby pośiedli ziemię.
Tedy ſłużył lud Pánu po wƺyſtkie dni Jozuego / y po wƺyſtkie dni ſtárƺych / którzy długo żyli po śmierći Jozuego / á którzy widzieli wƺyſtkie ſpráwy Páńſkie wielkie / które ucżynił Izráelowi.
Ale gdy umárł Jozue ſyn Nunów / ſługá Páńſki / będąc we ſtu y w dzieśięć lát ;
Y gdy go pogrzebli ná gránicy dziedzictwá jego w Támnáthereſ ná górze Efráim / od północy góry Gááſ ;
Tákże gdy wƺyſtek on rodzáj przyłącżony jeſt do ojców ſwoich / y powſtáł po nich inƺy náród / który nie znáł Páná / áni też ſpráw / który ucżynił Izráelowi ;
Tedy cżynili ſynowie Izráelſcy złe przed ocżymá Páńſkimi / á ſłużyli Báálom ;
Y opuśćili Páná / Bogá ojców ſwoich / który je wywiódł z ziemi Egipſkiej / y ƺli zá bogámi cudzymi / którzy byli z bogów onych národów okolicżnych / y kłániáli śię im / á ták rozdráźnili Páná.
Bo opuśćili Páná / á ſłużyli Báálowi y Aſtárotowi.
Y rozpálił śię gniew Páńſki przećiw Izráelowi / y podáł je w ręce łupieżcom / którzy je łupili ; á záprzedáł je w ręce nieprzyjáćiół ich okolicżnych / ták iż śię nie mogli dálej oſtáć przed nieprzyjáćioły ſwymi.
A gdzie śię kolwiek ruƺyli / ręká Páńſká byłá przećiwko nim ku złemu / jáko powiedziáł Pán / y jáko im przyśiągł Pán ; y byli śćiśnieni bárdzo.
Potem Pán wzbudził ſędzie / którzy je wyzwáláli z rąk łupieżców ich ;
Ale y ſędziów ſwych nie ſłucháli / owƺem śię zcudzołożyli z bogámi obcymi / kłániájąc śię im / y uſtępowáli prędko z drogi / którą chodzili ojcowie ich / á ſłuchájąc przykázáń Páńſkich / nie cżynili ták.
A gdy im wzbudzáł Pán ſędzie / bywáł Pán z káżdym ſędzią / y wybáwiáł je z rąk nieprzyjáćiół ich po wƺyſtkie dni onego ſędziego ; bo śię użálił Pán nárzekániá ich / do którego je przywodzili ći / którzy je ućiſkáli y trápili.
Wƺákże po śmierći ſędziego odwrácáli śię / y pſowáli śię bárdziej niż ojcowie ich / chodząc zá bogámi cudzymi / á ſłużąc im / y kłániájąc śię im / nic nie opuƺcżáli z ſpráw ſwoich y z drogi ſwojej upornej.
Przetoż wzruƺyłá śię popędliwość Páńſká przećiw Izráelowi / y rzekł : Dlá tego / że przeſtąpił ten náród przymierze moje / którem przykázáł ojcom ich / á nie byli poſłuƺni głoſowi mojemu /
Já też ná potem nie wypędzę żádnego od twárzy ich z tych národów / które pozoſtáwił Jozue / kiedy umárł.
Abym przez nie doświádcżáł Izráelá / będąli ſtrzedz drogi Páńſkiej / chodząc po niej / jáko jej ſtrzegli ojcowie ich / cżyli nie.
Y zoſtáwił Pán one národy / nie wygániájąc ich rychło / áni ich podáł w rękę Jozuego.
