Y przyƺedł Anjoł Páńſki z Gálgál do Bochym / mówiąc : Wywiodłem was z Egiptu / y wywiodłem was do źiemi / o którąm przyśiągł ojcom wáƺym / y mówiłem : Nie wzruƺę przymierza mego z wámi ná wieki ;
Jedno wy nie wchodźćie w przymierze z mieƺkájącymi w tey źiemi / owƺem ołtárze ich porozwálájćie ; áleśćie nie ſłucháli głoſu mojego. Przecżżeśćie to ucżynili?
Przetożem ták rzekł : Nie wypędzę ich od oblicżnośći wáƺej ; ále będą wam jáko ćiernie ná boki / á bogowie ich będą wam jáko śidło.
Y ſtáło śię / gdy mówił Anjoł Páńſki te ſłowá do wƺyſtkich ſynów Izráelſkich / że lud podnióſł głos ſwój / y płákáli.
Y názwáli imię miejſcá onego Bochym / á támże ofiárowáli Pánu.
A Jozue rozpuśćił był lud / y rozeƺli śię ſynowie Izráelſcy káżdy do dźiedźictwá ſwego / áby pośiedli źiemię.
Tedy ſłużył lud Pánu po wƺyſtkie dni Jozuego / y po wƺyſtkie dni ſtárƺych / którzy długo żyli po śmierći Jozuego / á którzy widźieli wƺyſtkie ſpráwy Páńſkie wielkie / które ucżynił Izráelowi.
Ale gdy umárł Jozue ſyn Nunów / ſługá Páńſki / będąc we ſtu y w dźieśięć lat ;
Y gdy go pogrzebli ná gránicy dźiedźictwá jego w Támnátheres ná górze Efráim / od północy góry Gáás ;
Tákże gdy wƺyſtek on rodzáj przyłącżony jeſt do ojców ſwojich / y powſtał po nich inƺy naród / który nie znał Páná / áni też ſpráw / który ucżynił Izráelowi ;
Tedy cżynili ſynowie Izráelſcy złe przed ocżymá Páńſkimi / á ſłużyli Báálom ;
Y opuśćili Páná / Bogá ojców ſwojich / który je wywiódł z źiemi Egipſkiej / y ƺli zá bogámi cudzymi / którzy byli z bogów onych narodów okolicżnych / y kłániáli śię im / á ták rozdráźnili Páná.
Bo opuśćili Páná / á ſłużyli Báálowi y Aſtárotowi.
Y rozpálił śię gniew Páńſki przećiw Izráelowi / y podał je w ręce łupieżcom / którzy je łupili ; á záprzedał je w ręce nieprzyjáćiół ich okolicżnych / ták iż śię nie mogli dálej oſtáć przed nieprzyjáćioły ſwymi.
A gdźie śię kolwiek ruƺyli / ręká Páńſka byłá przećiwko nim ku złemu / jáko powiedźiał Pán / y jáko im przyśiągł Pán ; y byli śćiśnieni bárzo.
Potem Pán wzbudźił ſędźie / którzy je wyzwáláli z rąk łupieżców ich ;
Ale y ſędźiów ſwych nie ſłucháli / owƺem śię zcudzołożyli z bogámi obcymi / kłániájąc śię im / y uſtępowáli prędko z drogi / którą chodźili ojcowie ich / á ſłuchájąc przykázáń Páńſkich / nie cżynili ták.
A gdy im wzbudzał Pán ſędźie / bywał Pán z káżdym ſędźią / y wybáwiał je z rąk nieprzyjáćiół ich po wƺyſtkie dni onego ſędźiego ; bo śię użálił Pán nárzekánia ich / do którego je przywodźili ći / którzy je ućiſkáli y trápili.
Wƺákże po śmierći ſędźiego odwrácáli śię / y pſowáli śię bárdźiej niż ojcowie ich / chodząc zá bogámi cudzymi / á ſłużąc im / y kłániájąc śię im / nic nie opuƺcżáli z ſpráw ſwojich y z drogi ſwojey uporney.
Przetoż wzruƺyłá śię popędliwość Páńſka przećiw Izráelowi / y rzekł : Dla tego / że przeſtąpił ten naród przymierze moje / którem przykázał ojcom ich / á nie byli poſłuƺni głoſowi mojemu /
Ja też ná potem nie wypędzę żádnego od twárzy ich z tych narodów / które pozoſtáwił Jozue / kiedy umárł.
Abym przez nie doświádcżał Izráelá / będąli ſtrzedz drogi Páńſkiej / chodząc po niej / jáko jey ſtrzegli ojcowie ich / cżyli nie.
Y zoſtáwił Pán one narody / nie wygániájąc ich rychło / áni ich podał w rękę Jozuego.
