Potem znowu ſynowie Izráelſcy cżynili złe przed ocżymá Páńſkimi po śmierći Aodowej.
Y podał je Pán w ręce Jábiná / królá Chánánejſkiego / który królował w Háſor / á hetmán wojſká jego był Syſárá / á ſám mieƺkał w Hároſet pogáńſkim.
Tedy wołáli ſynowie Izráelſcy do Páná ; ábowiem miał dźiewięć ſet wozów żeláznych / á ſrodze ućiſkał ſyny Izráelſkie przez dwádźieśćiá lat.
A Deborá / niewiáſtá prorokini / żoná Lápidotowá / ſądźiłá Izráelá ná on cżás.
Y mieƺkáłá pod pálmą Deborá miedzy Rámá y miedzy Betel ná górze Efráim / y chodźili do niej ſynowie Izráelſcy ná ſąd.
Która poſłáwƺy przyzwáłá Báráká / ſyná Abinoemowego / z Kádes Neftálim / mówiąc do niego : Izali nie rozkázał Pán / Bóg Izráelſki : idź / á zbierz lud ná górze Tábor / á weźmij z ſobą dźieśięć tyśięcy mężów z ſynów Neftálimowych / y z ſynów Zábulonowych?
Y przywiodę do ćiebie ku rzece Cyſon Syſárę / hetmáná wojſká Jábinowego / y wozy jego / y mnóſtwo jego / á podám go w ręce twoje.
Y rzekł do niej Bárák : Jeſli pójdźieƺ ze mną / pójdę / á jeſli nie pójdźieƺ ze mną / nie pójdę.
Która odpowiedźiáłá : Jáć w práwdźie pójdę z tobą / ále nie będźie z ſłáwą twoją tá drogá / którą ty pójdźieƺ ; ábowiem w rękę niewieśćią podá Pán Syſárę. A ták wſtáwƺy Deborá / ƺłá z Bárákiem do Kádes.
Zebrał tedy Bárák Zábuloná y Neftálimá do Kádes / á wywiódł z ſobą dźieśięć tyśięcy mężów / z którym też ƺłá y Deborá.
Ale Heber Cynejcżyk odłącżył śię od Cynejcżyków / od ſynów Hobáby / świekrá Mojżeƺowego / y rozbił námiot ſwój áż do Elon w Sánánim / które jeſt w Kádes.
Y powiedźiáno Syſárze / iż wyƺedł Bárák / ſyn Abinoemów ná górę Tábor.
Przetoż zebrał Syſárá wƺyſtkie wozy ſwoje / dźiewięć ſet wozów żeláznych / y wƺyſtek lud / który miał z ſobą od Hároſet pogáńſkiego áż do rzeki Cyſon.
Tedy rzekłá Deborá do Báráká : Wſtáń ; ábowiem tenći jeſt dźień / w który podał Pán Syſárę w ręce twoje ; izali Pán nie idźie przed tobą? A ták zƺedł Bárák z góry Tábor / á dźieśięć tyśięcy mężów zá nim.
Y poráźił Pán Syſárę / y wƺyſtkie wozy / y wƺyſtko wojſko jego oſtrzem miecżá przed Bárákiem / á ſkocżywƺy Syſárá z wozu / ućiekał pieƺo.
Ale Bárák gonił wozy y wojſko áż do Hároſet pogáńſkiego ; y poległo wƺyſtko wojſko Syſárowe od oſtrzá miecżá / ták iż z nich y jeden nie zoſtał.
A Syſárá ućiekł pieƺo do namiotu Jáeli / żony Heberá Cynejcżyká ; ábowiem był pokój miedzy Jábinem / królem Háſor / y miedzy domem Heberá Cynejcżyká.
A wyƺedƺy Jáel przećiwko Syſárze / rzekłá do niego : Skłoń śię pánie mój / ſkłoń śię do mnie / nie bój śię ; y ſkłonił śię do niej do namiotu / y przykryłá go kocem.
Tedy rzekł do niej : Dáj mi proƺę / nápić śię trochę wody / bom uprágnął ; á oná otworzywƺy łágiew mleká / dáłá mu śię nápić / y przykryłá go.
Y rzekł do niej : Stój we drzwiách u namiotu ; á jeſliby kto przyƺedł / y pytał ćię / mówiąc : Jeſtże tu kto? tedy rzecżeƺ : Niemáƺ.
Potem wźięłá Jáel / żoná Heberowá / gwóźdź od namiotu / wźięłá też y młot w rękę ſwą / á wƺedƺy do niego po ćichu / przebiłá gwoźdźiem ſkroń jego áż utknął w źiemi / ( bo był twárdo uſnął / będąc ſprácowánym / ) y umárł.
A oto / Bárák gonił Syſárę / y wyƺłá Jáel przećiwko niemu / y rzekłá mu : Pójdź / á ukáżęć mężá / którego ƺukáƺ. Y wyƺedł do niej / á oto Syſárá leżał umárły / á gwóźdź w ſkroni jego.
A ták poniżył Bóg dniá onego Jábiná / królá Chánánejſkiego / przed ſyny Izráelſkimi.
Y náćieráłá ręká ſynów Izráelſkich tym bárdźiej / á ćiężká byłá Jábinowi / królowi Chánánejſkiemu / áż zgłádźili tegoż Jábiná / królá Chánánejſkiego.
