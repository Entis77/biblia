Ale Jefte Gáláádcżyk był cżłowiekiem bárzo mężnym / á był ſynem niewiáſty nierządney / z którą ſpłodźił Gáláád tegoż Jeftego.
Ale y żoná Gáláádowá národźiłá mu ſynów ; á doroſƺy ſynowie tey żony / wygnáli Jeftego / mówiąc mu : Nie będźieƺ brał dźiedźictwá w domu ojcá náƺego / boś ty jeſt ſynem inƺej niewiáſty.
Ućiekł tedy Jefte przed bráćią ſwoją / á mieƺkał w źiemi Tob : y zebráli śię do niego ludźie ogołoceni / y poƺli z nim.
Y ſtáło śię potem / że wálcżyli ſynowie Ammonowi z Izráelem.
A gdy pocżęli wálcżyć Ammonitowie z Izráelem / tedy poƺli ſtárśi z Gáláád / áby wźięli Jeftego z źiemi Tob.
Y rzekli do niego : Pójdź / á bądź nam zá hetmáná / á będźiemy wálcżyli przećiwko ſynom Ammonowym.
Ale Jefte odpowiedźiał ſtárƺym Gáláád : Izáżeśćie wy mnie nie mieli w nienáwiśći / y wygnáliśćie mię z domu ojcá mego? przecżżeśćie przyƺli teraz do mnie / gdy ućiſk przyƺedł ná was?
Tedy rzekli ſtárśi z Gáláád do Jeftego : Dla tegośmy śię teraz wróćili do ćiebie / ábyś ƺedł z námi / á wálcżył przećiwko ſynom Ammonowym / á był nam zá hetmáná / wƺyſtkim mieƺkájącym w Gáláád.
Y odpowiedźiał Jefte ſtárƺym z Gáláád : Ponieważ wy mnie przywrácáćie / á chcećie / ábym wálcżył przećiwko ſynom Ammonowym / á jeſli mi je podá Pán / będęż wam zá hetmáná?
Y rzekli ſtárśi z Gáláád do Jeftego : Pán będźie świádkiem miedzy námi / jeſliż ták według ſłowá twego nie ucżynimy.
Tedy ƺedł Jefte z ſtárƺymi z Gáláád / y poſtánowił go lud hetmánem y kśiążęćiem nád ſobą ; y mówił Jefte wƺyſtkie te ſłowá przed Pánem w Máſfá.
Potem wypráwił Jefte poſły do królá ſynów Ammonowych / mówiąc : Co ja mam z tobą / żeś przyćiągnął ná mię / ábyś wálcżył przećiwko źiemi mojey?
Ná co odpowiedźiał król ſynów Ammonowych poſłom Jeftego : Że wźiął Izráel źiemię moję / gdy ƺedł z Egiptu / od Arnon áż do Jábok y áż do Jordánu : przetoż teraz wróć mi ją w pokoju.
Powtóre jeƺcże Jefte wypráwił poſły do królá ſynów Ammonowych.
Y rzekł mu : Ták mówi Jefte : Nie wźiął Izráel źiemi Moábſkiej / áni źiemi ſynów Ammonowych.
Ale gdy z Egiptu ƺedł Izráel przez puƺcżą áż ku morzu cżerwonemu / á przyƺedł do Kádes.
Skąd wypráwił Izráel poſły do królá Edomſkiego / mówiąc : Proƺę niech przejdę przez źiemię twoję / y nie pozwolił król Edomſki / tákże y do królá Moábſkiego poſłał / y nie pozwolił ; á ták zoſtał Izráel w Kádes.
A gdy ƺedł przez puƺcżą / obƺedł źiemię Edomſką / y źiemię Moábſką / á przyƺedł od wſchodu ſłońcá źiemi Moábſkiej / y położyli śię obozem zá Arnon / á nie wƺedł w gránice Moábſkie ; bo Arnon jeſt gránicą Moábſką.
Dlátegoż wypráwił Izráel poſły do Sehoná / królá Amorejſkiego / królá w Heſebon / y rzekł mu Izráel : Niech przejdę proƺę przez źiemię twoję áż do miejſcá mego.
Ale nie dowierzał Sehon Izráelowi / áby iść miał przez gránicę jego ; owƺem zebrał Sehon wƺyſtek lud ſwój / y położył śię obozem w Jáſá / y zwiódł bitwę z Izráelem.
Y dał Pán / Bóg Izráelſki / Sehoná / y wƺyſtek lud jego w ręce Izráelowe / y poráźił je / á pośiádł Izráel wƺyſtkę źiemię Amorejcżyká / który mieƺkał w oney źiemi.
A ták pośiedli wƺyſtkie gránice Amorejſkie od Arnon áż do Jábok / á od puƺcży áż do Jordánu.
Ponieważ tedy Pán / Bóg Izráelſki wypędźił Amorejcżyká przed ludem ſwym Izráelſkim / przecż ty chceƺ pánowáć nád nim?
Azaż / coć dał pośieść Kámos / bóg twój / tego nie pośiędźieƺ? ták / kogo Pán / Bóg náƺ / wygnał przed oblicżnośćią náƺą / tego też dźiedźictwo pośiádámy.
Do tego cżemżeś ty lepƺy nád Báláká ſyná Seferowego / królá Moábſkiego? záż śię on kiedy wádźił z Izráelem? záż kiedy wálcżył przećiwko niemu?
Oto przez trzy ſtá lat mieƺkał Izráel w Heſebon / y we wśiách jego / tákże w Aroer / y we wśiách jego / y we wƺyſtkich miáſtách / które ſą przy gránicy Arnon ; cżemużeśćie ich nie odjęli przez ten cżás?
A ták nie jam tobie winien / ále ty mnie źle cżyniƺ / że wálcżyƺ przećiwko mnie ; niech Pán / który jeſt Sędźią / rozſądźi dźiś miedzy ſyny Izráelſkimi y miedzy ſyny Ammonowymi.
Ale nie uſłuchał król ſynów Ammonowych ſłów Jeftego / które wſkázał do niego.
Y był nád Jeftem duch Páńſki / á przeƺedł przez Gáláád / y przez Mánáſe ; przeƺedł też przez Máſfá w Gáláád / á z Máſfy w Gáláád ćiągnął przećiw ſynom Ammonowym.
Támże ucżynił Jefte ślub Pánu / mówiąc : Jeſli pewnie podáƺ ſyny Ammonowe w ręce moje /
Tedy to / cobykolwiek wyƺło ze drzwi domu mego przećiwko mnie / gdy śię wrócę w pokoju od ſynów Ammonowych / to mówię będźie Pánu / álbo ofiárowáć je będę ná cáłopálenie.
A ták Jefte ćiągnął przećiwko ſynom Ammonowym / áby wálcżył z nimi / y podał je Pán w ręce jego.
Y poráźił je od Aroer áż idąc do Menit / dwádźieśćiá miaſt / y áż do równiny winnic poráżką bárzo wielką / á poniżeni ſą ſynowie Ammonowi przed ſyny Izráelſkimi.
A gdy śię wrácał Jefte do Máſfá do domu ſwego / oto / córká jego wyƺłá przećiw niemu / z bębny / y z muzyką ; á tá byłá jedynácżká / bo niemiał żádnego ſyná áni inney córki.
Y ſtáło śię / gdy ją ujrzał / rozdárł odźienie ſwoje / y rzekł : Ach / córko mojá / bárdzoś mię poniżyłá! y tyś jeſt z tych / którzy mię fráſują / gdyżem ślub ucżynił Pánu / á nie będę mógł odmienić.
Któremu oná odpowiedźiáłá : Ojcże mój / ucżyniłeś ślub Pánu / ucżyńże ze mną ták / jákoś wyrzekł uſty twojimi / gdyć tylko dał Pán pomſtę nád nieprzyjáćioły twemi / nád ſyny Ammonowymi.
Nádto rzekłá do ojcá ſwego : To mi tylko ucżyń : puść mię ná dwá mieśiące / że pójdę á wſtąpię ná góry y opłákiwáć będę pánieńſtwo moje / ja y towárzyƺki moje.
A on rzekł : Idź ; y puśćił ją ná dwá mieśiące. Poƺłá tedy oná y towárzyƺki jey / á opłákiwáłá pánieńſtwo ſwoje ná górách.
A gdy wyƺły dwá mieśiące / wróćiłá śię od ojcá ſwego / y wypełnił nád nią ślub ſwój / który był ucżynił ; á ták oná nie poznáłá mężá. Y weƺło to w zwycżáj w Izráelu /
Iż ná káżdy rok ſchodźiły śię córki Izráelſkie áby śię rozmáwiáły z córką Jeftego Gáláádcżyká / przez cżtery dni w rok.
