Y rzekli do niego mężowie z Efráim : Przecżżeś to nám ucżynił / iżeś náſ nie wezwáł / gdyś ƺedł wálcżyć przećiwko Mádyjánitom? y ſwárzyli śię z nim ſrodze.
A on rzekł : Cóżem já tákiego ucżynił / jáko wy? izáli nie lepƺe jeſt poślednie zbieránie winá Efráimowe / niż pierwƺe zbieránie Abieſerowe?
W rękę wáƺę podáł Bóg kśiążętá Mádyjáńſkie / Orebá y Zebá ; y cóżem mógł tákiego ucżynić / jáko wy? Tedy śię uśmierzył duch ich przećiw niemu / gdy mówił te ſłowá.
A gdy przyƺedł Giedeon do Jordánu / przepráwił śię przezeń ſám / y trzy ſtá mężów / którzy z nim byli ſprácowáni w pogoni.
Y rzekł do mieƺcżán w Sokot : Dájćie proƺę po bochenku chlebá ludowi / który idzie zá mną / bo ſą ſprácowáni / á já będę gonił Zebeę y Sálmáná / króle Mádyjáńſkie.
Ale mu rzekli przedniejśi z Sokot : Izáli już moc Zeby y Sálmáná jeſt w rękách twoich / żebyśmy dáć mieli wojſku twemu chlebá?
Którym rzekł Giedeon : Więc kiedy podá Pán Zebeę y Sálmáná w rękę moję / tedy będę młóćił ćiáłá wáƺe ćierniem z tej puƺcży y oſtem.
Szedł záśię ſtámtąd do Fánuel / y mówił tákże do nich ; ále mu odpowiedzieli mężowie z Fánuel / jáko odpowiedzieli mężowie w Sokot.
Tedy też powiedziáł mężom z Fánuel / mówiąc : Gdy śię wrócę w pokoju / rozwálę tę wieżę.
Ale Zebee y Sálmáná byli w Kárkor / y wojſká ich z nimi około piętnáſtu tyśięcy / wƺyſcy / którzy byli pozoſtáli ze wƺyſtkiego wojſká z ludzi od wſchodu ſłońcá ; á pobitych było ſto y dwádzieśćiá tyśięcy mężów wálecżnych.
Tedy ćiągnął Giedeon drogą tych / co mieƺkáli w námioćiech / od wſchodu ſłońcá Nobe y Jegbáá / y uderzył ná obóz / ( á obóz śię był ubezpiecżył / )
A ućiekli Zebee y Sálmáná / y gonił je / y pojmáł onych dwóch królów Mádyjáńſkich / Zebeę y Sálmáná / y wƺyſtko wojſko ich ſtrwożył.
Potem śię wróćił Giedeon / ſyn Joáſów / z bitwy / niż weƺło ſłońce ;
A pojmáwƺy młodzieńcá z mężów Sokot / wypytáł go / który ſpiſáł mu przedniejƺych w Sokot y ſtárƺych jego / śiedmdzieśiąt y śiedm mężów.
A przyƺedłƺy do mężów Sokot / rzekł : Otóż Zebeę y Sálmáná / którymiśćie mi urągáli / mówiąc : Izáli moc Zeby y Sálmáná jeſt w rękách twoich / ábyśmy mieli dáć mężom twoim ſprácowánym chlebá?
Przetoż wziąwƺy ſtárƺe miáſtá onego / y ćierniá z onej puſtyni y oſtu / dáł ná nich przykłád innym mężom Sokot.
Wieżę też Fánuel rozwálił / y pobił męże miáſtá.
Rzekł potem do Zeby y do Sálmáná : Co zácż byli mężowie oni / któreśćie pobili w Tábor? A oni odpowiedzieli : Tákowi byli jákoś ty ; káżdy z nich ná wejrzeniu był / jáko ſyn królewſki.
Y rzekł : Bráćiáć to moi / ſynowie mátki mojej byli ; żywie Pán / byśćie je byli żywo záchowáli / nie pobiłbym wáſ.
Y rzekł do Jetrá / pierworodnego ſwego : Wſtáń / á pobij je. Ale nie dobył młodzieńcżyk miecżá ſwego / przeto iż śię báł ; bo jeƺcże był pácholęćiem.
Tedy rzekli Zebee y Sálmáná : Wſtáń ty / á rzuć śię ná náſ ; bo jáki mąż / táká śiłá jego. A ták wſtáwƺy Giedeon / zábił Zebeę y Sálmáná / y pobráł klejnoty / które były ná ƺyjách wielbłądów ich.
Y rzekli Izráelcżycy do Giedeoná : Pánuj nád námi / y ty / y ſyn twój / y ſyn ſyná twego ; boś náſ wybáwił z ręki Mádyjáńcżyków.
Ná to odpowiedziáł im Giedeon : Nie będę já pánowáł nád wámi / áni będzie pánowáł ſyn mój nád wámi ; Pán pánowáć będzie nád wámi.
Nádto rzekł do nich Giedeon : Będę wáſ prośił o jednę rzecż / áby mi káżdy z wáſ dáł náuƺnicę z łupu ſwego ; ( bo náuƺnice złote mieli / będąc Iſmáelcżykámi. )
Y rzekli : Rádzić dámy ; y rozpoſtárłƺy ƺátę rzucáli ná nię káżdy náuƺnicę z łupów ſwoich.
Y byłá wágá náuƺnic onych złotych / które ſobie uprośił / tyśiąc y śiedm ſet ſyklów złotá / oprócż klejnotów y záwieƺeniá y ƺát ƺárłátowych / które były ná królách Mádyjáńſkich / y oprócż łáńcuchów / które były ná ƺyjách wielbłądów ich.
Y ſpráwił z tego Giedeon Efod / á położył go w mieśćie ſwem w Efrá ; y zcudzołożył śię tám wƺyſtek Izráel / chodząc zá nim / á było to Giedeonowi y domowi jego śidłem.
A ták byli poniżeni Mádyjáńcżycy przed ſynámi Izráelſkimi / y nie podnieśli więcej głowy ſwojej ; y byłá w pokoju ziemiá przez cżterdzieśći lát zá dni Giedeoná.
Wróćił śię tedy Jerobáál / ſyn Joáſów / y mieƺkáł w domu ſwoim.
A miáł Giedeon śiedmdzieśiąt ſynów / którzy poƺli z biódr jego ; ábowiem miáł wiele żon.
Miáł też záłożnicę / którá byłá z Sychem / á tá mu urodziłá ſyná / y dáłá mu imię Abimelech.
Umárł potem Giedeon / ſyn Joáſów / w ſtárośći dobrej / á pogrzebion jeſt w grobie Joáſá / ojcá ſwego w Efrá / które jeſt ojcá Eſrowego.
A gdy umárł Giedeon / odwróćili śię ſynowie Izráelſcy / y zcudzołożyli śię / idąc zá Báálem / y poſtáwili ſobie Báálberyt zá bogá.
Y nie pámiętáli ſynowie Izráelſcy ná Páná / Bogá ſwego / który je wyrwáł z rąk wƺyſtkich nieprzyjáćiół ich okolicżnych ;
Y nie ucżynili miłośierdziá z domem Jerobáálá Giedeoná według wƺyſtkich dobrodziejſtw / które on był ucżynił Izráelowi.
