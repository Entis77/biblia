Y rzekli do niego mężowie z Efráim : Przecżżeś to nam ucżynił / iżeś nas nie wezwał / gdyś ƺedł wálcżyć przećiwko Mádyjánitom? y ſwárzyli śię z nim ſrodze.
A on rzekł : Cóżem ja tákiego ucżynił / jáko wy? izali nie lepƺe jeſt poślednie zbieránie winá Efráimowe / niż pierwƺe zbieránie Abieſerowe?
W rękę wáƺę podał Bóg kśiążętá Mádyjáńſkie / Orebá y Zebá ; y cóżem mógł tákiego ucżynić / jáko wy? Tedy śię uśmierzył duch ich przećiw niemu / gdy mówił te ſłowá.
A gdy przyƺedł Giedeon do Jordánu / przepráwił śię przezeń ſám / y trzy ſtá mężów / którzy z nim byli ſprácowáni w pogoni.
Y rzekł do mieƺcżán w Sokot : Dájćie proƺę po bochenku chlebá ludowi / który idźie zá mną / bo ſą ſprácowáni / á ja będę gonił Zebeę y Sálmáná / króle Mádyjáńſkie.
Ale mu rzekli przedniejśi z Sokot : Izali już moc Zeby y Sálmáná jeſt w rękách twojich / żebyſmy dáć mieli wojſku twemu chlebá?
Którym rzekł Giedeon : Więc kiedy podá Pán Zebeę y Sálmáná w rękę moję / tedy będę młóćił ćiáłá wáƺe ćierniem z tey puƺcży y oſtem.
Szedł záśię ſtámtąd do Fánuel / y mówił tákże do nich ; ále mu odpowiedźieli mężowie z Fánuel / jáko odpowiedźieli mężowie w Sokot.
Tedy też powiedźiał mężom z Fánuel / mówiąc : Gdy śię wrócę w pokoju / rozwálę tę wieżę.
Ale Zebee y Sálmáná byli w Kárkor / y wojſká ich z nimi około piętnáſtu tyśięcy / wƺyſcy / którzy byli pozoſtáli ze wƺyſtkiego wojſká z ludźi od wſchodu ſłońcá ; á pobitych było ſto y dwádźieśćiá tyśięcy mężów wálecżnych.
Tedy ćiągnął Giedeon drogą tych / co mieƺkáli w námioćiech / od wſchodu ſłońcá Nobe y Jegbáá / y uderzył ná obóz / ( á obóz śię był ubezpiecżył / )
A ućiekli Zebee y Sálmáná / y gonił je / y pojmał onych dwu królów Mádyjáńſkich / Zebeę y Sálmáná / y wƺyſtko wojſko ich ſtrwożył.
Potem śię wróćił Giedeon / ſyn Joáſów / z bitwy / niż weƺło ſłońce ;
A pojmáwƺy młodźieńcá z mężów Sokot / wypytał go / który ſpiſał mu przedniejƺych w Sokot y ſtárƺych jego / śiedmdźieśiąt y śiedm mężów.
A przyƺedƺy do mężów Sokot / rzekł : Otóż Zebeę y Sálmáná / którymiśćie mi urągáli / mówiąc : Izali moc Zeby y Sálmáná jeſt w rękách twojich / ábyſmy mieli dáć mężom twoim ſprácowánym chlebá?
Przetoż wźiąwƺy ſtárƺe miáſtá onego / y ćierniá z oney puſtyni y oſtu / dał ná nich przykłád innym mężom Sokot.
Wieżę też Fánuel rozwálił / y pobił męże miáſtá.
Rzekł potem do Zeby y do Sálmáná : Co zácż byli mężowie oni / któreśćie pobili w Tábor? A oni odpowiedźieli : Tákowi byli jákoś ty ; káżdy z nich ná wejrzeniu był / jáko ſyn królewſki.
Y rzekł : Bráćiać to moi / ſynowie mátki mojey byli ; żywie Pán / byśćie je byli żywo záchowáli / nie pobiłbym was.
Y rzekł do Jetrá / pierworodnego ſwego : Wſtáń / á pobij je. Ale nie dobył młodźieńcżyk miecżá ſwego / przeto iż śię bał ; bo jeƺcże był pácholęćiem.
Tedy rzekli Zebee y Sálmáná : Wſtáń ty / á rzuć śię ná nas ; bo jáki mąż / táká śiłá jego. A ták wſtáwƺy Giedeon / zábił Zebeę y Sálmáná / y pobrał klejnoty / które były ná ƺyjách wielbłądów ich.
Y rzekli Izráelcżycy do Giedeoná : Pánuj nád námi / y ty / y ſyn twój / y ſyn ſyná twego ; boś nas wybáwił z ręki Mádyjáńcżyków.
Ná to odpowiedźiał im Giedeon : Nie będę ja pánował nád wámi / áni będźie pánował ſyn mój nád wámi ; Pán pánowáć będźie nád wámi.
Nádto rzekł do nich Giedeon : Będę was prośił o jednę rzecż / áby mi káżdy z was dał náuƺnicę z łupu ſwego ; ( bo náuƺnice złote mieli / będąc Iſmáelcżykámi. )
Y rzekli : Rádźić dámy ; y rozpoſtárƺy ƺátę rzucáli ná nię káżdy náuƺnicę z łupów ſwojich.
Y byłá wágá náuƺnic onych złotych / które ſobie uprośił / tyśiąc y śiedm ſet ſyklów złotá / oprócż klejnotów y záwieƺeniá y ƺát ƺárłátowych / które były ná królách Mádyjáńſkich / y oprócż łáńcuchów / które były ná ƺyjách wielbłądów ich.
Y ſpráwił z tego Giedeon Efod / á położył go w mieśćie ſwem w Efrá ; y zcudzołożył śię tám wƺyſtek Izráel / chodząc zá nim / á było to Giedeonowi y domowi jego śidłem.
A ták byli poniżeni Mádyjáńcżycy przed ſynámi Izráelſkimi / y nie podnieśli więcey głowy ſwojey ; y byłá w pokoju źiemiá przez cżterdźieśći lat zá dni Giedeoná.
Wróćił śię tedy Jerobáál / ſyn Joáſów / y mieƺkał w domu ſwoim.
A miał Giedeon śiedmdźieśiąt ſynów / którzy poƺli z biódr jego ; ábowiem miał wiele żon.
Miał też záłożnicę / która byłá z Sychem / á tá mu urodźiłá ſyná / y dáłá mu imię Abimelech.
Umárł potem Giedeon / ſyn Joáſów / w ſtárośći dobrej / á pogrzebion jeſt w grobie Joáſá / ojcá ſwego w Efrá / które jeſt ojcá Eſrowego.
A gdy umárł Giedeon / odwróćili śię ſynowie Izráelſcy / y zcudzołożyli śię / idąc zá Báálem / y poſtáwili ſobie Báálberyt zá bogá.
Y nie pámiętáli ſynowie Izráelſcy ná Páná / Bogá ſwego / który je wyrwał z rąk wƺyſtkich nieprzyjáćiół ich okolicżnych ;
Y nie ucżynili miłośierdźiá z domem Jerobáálá Giedeoná według wƺyſtkich dobrodźiejſtw / które on był ucżynił Izráelowi.
