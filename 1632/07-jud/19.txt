Y ſtáło śię w one dni / gdy królá nie było w Izráelu / że mąż niektóry Lewitá / mieƺkájący przy ſtronie góry Efráim / pojął ſobie żonę záłożnicę z Betlehem Judá.
A báwiłá śię nierządem przy nim záłożnicá jego ; potem odeƺłá od niego do domu ojcá ſwego / do Betlehem Judá ; y byłá tám u niego przez cżtery mieśiące.
Wſtáwƺy tedy mąż jej / ƺedł zá nią / áby ją ubłágáwƺy záśię ją przywiódł / májąc z ſobą ſługę ſwego / y párę oſłów. Tedy oná wwiodłá go w dom ojcá ſwego / którego gdy ujrzáł ojćiec onej dziewki / rádowáł śię z przyjśćiá jego.
Y przyjął go wdzięcżnie świekier jego / ojćiec dziewki onej / á mieƺkáł u niego / przez trzy dni / y jedli y pili y nocowáli tám.
A dniá cżwártego / gdy wſtáli bárdzo ráno / wſtáł y on / áby odƺedł. Ale rzekł ojćiec onej dziewki do zięćiá ſwego : Pośil ſerce twoje trochą chlebá / á potem pójdziećie.
Tedy śiedli y jedli oboje weſpół / y nápili śię. Zátem rzekł ojćiec onej dziewki do mężá jej : Zoſtáń proƺę / á przenocuj tu / y bądź dobrej myśli.
A gdy wſtáł on mąż / chcąc przećię iść w drogę / gwáłtem przymuśił go świekier jego / iż śię wróćiwƺy zoſtáł tám ná noc.
Wſtáł potem bárdzo ráno dniá piątego / chcąc iść ; ále mówił ojćiec onej dziewki : Pośil proƺę ſerce twoje ; y zábáwili śię / áż śię dzień náchylił / á jedli obá ſpołu.
Wſtáł tedy on mąż / áby ƺedł ſám y záłożnicá jego / y ſługá jego / któremu rzekł świekier jego / ojćiec onej dziewki : Oto śię już dzień náchylił ku wiecżorowi / przenocujćież tu proƺę ; oto ſchodzi dzień / przenocujże tu / á bądź dobrej myśli / á jutro ráno wypráwićie śię w drogę ſwą / y pójdzieƺ do przybytku twego.
Tedy on mąż nie chćiáł zoſtáć ná noc / ále wſtáł y odƺedł / á przyƺedł áż ku Jebuſ / ( które jeſt Jeruzálem ) májąc z ſobą dwóch oſłów z brzemiony / y záłożnicę ſwoję.
A gdy byli bliſko Jebuſ / á dzień śię już bárdzo náchylił / tedy rzekł ſługá do páná ſwego : Pójdź proƺę / á wſtąpmy do tego miáſtá Jebuzejcżyków / y przenocujmy w nim.
Któremu odpowiedziáł pán jego : Nie wſtępujmy do miáſtá cudzoziemców / które nie jeſt z ſynów Izráelſkich / ále idźmy áż do Gábáá.
Nádto rzekł do ſługi ſwego : Pójdź / ábyśmy przyƺli ná jedno z tych miejſc / y przenocowáli álbo w Gábáá álbo w Rámá.
A minąwƺy poƺli : y záƺło im ſłońce u Gábáá / które jeſt pokoleniá Benjáminowego.
Y událi śię tám / áby wƺedłƺy przenocowáli w Gábáá ; á gdy wƺedł / uśiádł ná ulicy w mieśćie / przeto że nie był / ktoby je przyjął w dom y przenocowáł.
A oto / mąż ſtáry ƺedł od roboty ſwojej z polá w wiecżór ; á ten mąż był z góry Efráim / będąc przychodniem w Gábáá / ále ludzie miejſcá onego byli ſynowie Jemini.
Ten podnióſłƺy ocży ſwe ujrzáł mężá onego podróżnego ná ulicy miáſtá / y rzekł do niego ſtárzec : Dokąd idzieƺ / y ſkądeś przyƺedł?
Któremu on odpowiedziáł : Idziemy z Betlehem Judá áż ku ſtronie góry Efráimowej / ſkądem jeſt ; bom chodził do Betlehem Judſkiego ; á teráz idę do domu Páńſkiego / ále niemáƺ nikogo / coby mię przyjął w dom ;
Choć y plewy y śiáno mám dlá oſłów náƺych / tákże chleb y wino mám dlá śiebie y dlá ſłużebnicy twej y dlá ſługi / który jeſt ze mną / ſługą twoim ; niemám niedoſtátku z żádnej rzecży.
Tedy mu rzekł on mąż ſtáry : Nie fráſuj śię ; cżegoćkolwiek nie doſtánie / to já opátrzę ; tylko ná ulicy nie zoſtáwáj przez noc.
Wwiódł go tedy do domu ſwego / y dáł oſłom obrok ; potem umywƺy nogi ſwoje / jedli y pili.
A gdy rozweſelili ſerce ſwoje / oto / mężowie miáſtá tego / mężowie niepobożni / obſtąpili dom / kołácąc we drzwi / y rzekli do goſpodárzá domu onego / do mężá ſtárego / mówiąc : Wywiedź mężá / który wƺedł w dom twój / ábyśmy go poználi.
A wyƺedłƺy do nich on mąż / goſpodárz domu / rzekł im : Nie ták bráćiá moi : nie cżyńćie proƺę tej złośći / gdyż wƺedł ten mąż do domu mego / nie cżyńćież tej ſproſnośći.
Oto córká mojá pánná / y záłożnicá jego / wywiodę je záráz / że je obelżyćie / á ucżynićie z niemi / co śię wám będzie dobrego zdáło ; tylko mężowi temu nie cżyńćie tej zelżywośći.
Ale nie chćieli oni mężowie ſłucháć głoſu jego ; przetoż wziąwƺy on mąż záłożnicę ſwoję / wywiódł ją do nich ná dwór ; y poználi ją / á cżynili jej gwáłt przez cáłą noc áż do záránku / á potem puśćili ją / gdy wſchodziłá zorzá.
A przyƺedłƺy oná niewiáſtá ná świtániu / upádłá u drzwi domu onegoż mężá / gdzie był pán jej / áż śię rozedniáło.
Potem wſtáwƺy pán jej ráno / otworzył drzwi u domu / y wyƺedł / chcąc iść w drogę ſwoję / á oto / oná niewiáſtá / záłożnicá jego / leżáłá u drzwi domu / á ręce jej były ná progu.
Y rzekł do niej : Wſtáń á pójdźmy ; ále nic nie odpowiedziáłá. Wziąwƺy ją tedy ná oſłá / wſtáł on mąż / y ƺedł do miejſcá ſwego.
Tám przyƺedłƺy w dom ſwój / porwáł miecż / á zdjąwƺy záłożnicę ſwoję rozrąbáł ją z kośćiámi jej ná dwánáśćie ƺtuk / y rozeſłáł ją po wƺyſtkich gránicách Izráelſkich.
A ktokolwiek to widziáł / mówił : Nigdy śię to nie ſtáło / áni co tákowego widziáno od onego dniá / jáko wyƺli ſynowie Izráelſcy z ziemi Egipſkiej / áż do tego dniá ; uwáżájćież to z pilnośćią / á rádźćie y mówćie o tem.
