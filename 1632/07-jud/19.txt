Y ſtáło śię w one dni / gdy królá nie było w Izráelu / że mąż niektóry Lewitá / mieƺkájący przy ſtronie góry Efráim / pojął ſobie żonę záłożnicę z Betlehem Judá.
A báwiłá śię nierządem przy nim záłożnicá jego ; potem odeƺłá od niego do domu ojcá ſwego / do Betlehem Judá ; y byłá tám u niego przez cżtery mieśiące.
Wſtáwƺy tedy mąż jey / ƺedł zá nią / áby ją ubłágáwƺy záśię ją przywiódł / májąc z ſobą ſługę ſwego / y párę oſłów. Tedy oná wwiodłá go w dom ojcá ſwego / którego gdy ujrzáł ojćiec oney dźiewki / rádowáł śię z przyjśćiá jego.
Y przyjął go wdźięcżnie świekier jego / ojćiec dźiewki oney / á mieƺkáł u niego / przez trzy dni / y jedli y pili y nocowáli tám.
A dniá cżwártego / gdy wſtáli bárdzo ráno / wſtáł y on / áby odƺedł. Ale rzekł ojćiec oney dźiewki do źięćiá ſwego : Pośil ſerce twoje trochą chlebá / á potem pójdźiećie.
Tedy śiedli y jedli oboje weſpół / y nápili śię. Zátem rzekł ojćiec oney dźiewki do mężá jey : Zoſtáń proƺę / á przenocuj tu / y bądź dobrej myśli.
A gdy wſtáł on mąż / chcąc przećię iść w drogę / gwáłtem przymuśił go świekier jego / iż śię wróćiwƺy zoſtáł tám ná noc.
Wſtáł potem bárdzo ráno dniá piątego / chcąc iść ; ále mówił ojćiec oney dźiewki : Pośil proƺę ſerce twoje ; y zábáwili śię / áż śię dźień náchylił / á jedli obá ſpołu.
Wſtáł tedy on mąż / áby ƺedł ſám y záłożnicá jego / y ſługá jego / któremu rzekł świekier jego / ojćiec oney dźiewki : Oto śię już dźień náchylił ku wiecżorowi / przenocujćież tu proƺę ; oto ſchodźi dźień / przenocujże tu / á bądź dobrej myśli / á jutro ráno wypráwićie śię w drogę ſwą / y pójdźieƺ do przybytku twego.
Tedy on mąż nie chćiáł zoſtáć ná noc / ále wſtáł y odƺedł / á przyƺedł áż ku Jebuſ / ( które jeſt Jeruzálem ) májąc z ſobą dwóch oſłów z brzemiony / y záłożnicę ſwoję.
A gdy byli bliſko Jebuſ / á dźień śię już bárdzo náchylił / tedy rzekł ſługá do páná ſwego : Pójdź proƺę / á wſtąpmy do tego miáſtá Jebuzejcżyków / y przenocujmy w nim.
Któremu odpowiedźiáł pán jego : Nie wſtępujmy do miáſtá cudzoźiemców / które nie jeſt z ſynów Izráelſkich / ále idźmy áż do Gábáá.
Nádto rzekł do ſługi ſwego : Pójdź / ábyśmy przyƺli ná jedno z tych miejſc / y przenocowáli álbo w Gábáá álbo w Rámá.
A minąwƺy poƺli : y záƺło im ſłońce u Gábáá / które jeſt pokoleniá Benjáminowego.
Y událi śię tám / áby wƺedłƺy przenocowáli w Gábáá ; á gdy wƺedł / uśiádł ná ulicy w mieśćie / przeto że nie był / ktoby je przyjął w dom y przenocowáł.
A oto / mąż ſtáry ƺedł od roboty ſwojey z polá w wiecżór ; á ten mąż był z góry Efráim / będąc przychodniem w Gábáá / ále ludźie miejſcá onego byli ſynowie Jemini.
Ten podnióſłƺy ocży ſwe ujrzáł mężá onego podróżnego ná ulicy miáſtá / y rzekł do niego ſtárzec : Dokąd idźieƺ / y ſkądeś przyƺedł?
Któremu on odpowiedźiáł : Idźiemy z Betlehem Judá áż ku ſtronie góry Efráimowej / ſkądem jeſt ; bom chodźił do Betlehem Judſkiego ; á teraz idę do domu Páńſkiego / ále niemáƺ nikogo / coby mię przyjął w dom ;
Choć y plewy y śiáno mam dla oſłów náƺych / tákże chleb y wino mam dla śiebie y dla ſłużebnicy twej y dla ſługi / który jeſt ze mną / ſługą twoim ; niemám niedoſtátku z żádney rzecży.
Tedy mu rzekł on mąż ſtáry : Nie fráſuj śię ; cżegoćkolwiek nie doſtánie / to ja opátrzę ; tylko ná ulicy nie zoſtáwáj przez noc.
Wwiódł go tedy do domu ſwego / y dáł oſłom obrok ; potem umywƺy nogi ſwoje / jedli y pili.
A gdy rozweſelili ſerce ſwoje / oto / mężowie miáſtá tego / mężowie niepobożni / obſtąpili dom / kołácąc we drzwi / y rzekli do goſpodárzá domu onego / do mężá ſtárego / mówiąc : Wywiedź mężá / który wƺedł w dom twój / ábyśmy go poználi.
A wyƺedłƺy do nich on mąż / goſpodárz domu / rzekł im : Nie ták bráćia moi : nie cżyńćie proƺę tey złośći / gdyż wƺedł ten mąż do domu mego / nie cżyńćież tey ſproſnośći.
Oto córká mojá pánná / y záłożnicá jego / wywiodę je záráz / że je obelżyćie / á ucżynićie z niemi / co śię wám będźie dobrego zdáło ; tylko mężowi temu nie cżyńćie tey zelżywośći.
Ale nie chćieli oni mężowie ſłucháć głoſu jego ; przetoż wźiąwƺy on mąż záłożnicę ſwoję / wywiódł ją do nich ná dwór ; y poználi ją / á cżynili jey gwáłt przez cáłą noc áż do záránku / á potem puśćili ją / gdy wſchodźiłá zorzá.
A przyƺedłƺy oná niewiáſtá ná świtániu / upádłá u drzwi domu onegoż mężá / gdźie był pán jey / áż śię rozedniáło.
Potem wſtáwƺy pán jey ráno / otworzył drzwi u domu / y wyƺedł / chcąc iść w drogę ſwoję / á oto / oná niewiáſtá / záłożnicá jego / leżáłá u drzwi domu / á ręce jey były ná progu.
Y rzekł do niej : Wſtáń á pójdźmy ; ále nic nie odpowiedźiáłá. Wźiąwƺy ją tedy ná oſłá / wſtáł on mąż / y ƺedł do miejſcá ſwego.
Tám przyƺedłƺy w dom ſwój / porwáł miecż / á zdjąwƺy záłożnicę ſwoję rozrąbáł ją z kośćiámi jey ná dwánáśćie ƺtuk / y rozeſłáł ją po wƺyſtkich gránicách Izráelſkich.
A ktokolwiek to widźiáł / mówił : Nigdy śię to nie ſtáło / áni co tákowego widźiáno od onego dniá / jáko wyƺli ſynowie Izráelſcy z źiemi Egipſkiej / áż do tego dniá ; uwáżájćież to z pilnośćią / á rádźćie y mówćie o tem.
