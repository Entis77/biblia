Y ſtáło śię w one dni / gdy królá nie było w Izráelu / że mąż niektóry Lewitá / mieƺkájący przy ſtronie góry Efráim / pojął ſobie żonę záłożnicę z Betlehem Judá.
A báwiłá śię nierządem przy nim záłożnicá jego ; potem odeƺłá od niego do domu ojcá ſwego / do Betlehem Judá ; y byłá tám u niego przez cżtery mieśiące.
Wſtáwƺy tedy mąż jey / ƺedł zá nią / áby ją ubłágáwƺy záśię ją przywiódł / májąc z ſobą ſługę ſwego / y párę oſłów. Tedy oná wwiodłá go w dom ojcá ſwego / którego gdy ujrzał oćiec oney dźiewki / rádował śię z przyjśćiá jego.
Y przyjął go wdźięcżnie świekier jego / oćiec dźiewki oney / á mieƺkał u niego / przez trzy dni / y jedli y pili y nocowáli tám.
A dniá cżwártego / gdy wſtáli bárzo ráno / wſtał y on / áby odƺedł. Ale rzekł oćiec oney dźiewki do źięćiá ſwego : Pośil ſerce twoje trochą chlebá / á potem pójdźiećie.
Tedy śiedli y jedli oboje weſpół / y nápili śię. Zátem rzekł oćiec oney dźiewki do mężá jey : Zoſtáń proƺę / á przenocuj tu / y bądź dobrej myśli.
A gdy wſtał on mąż / chcąc przećię iść w drogę / gwáłtem przymuśił go świekier jego / iż śię wróćiwƺy zoſtał tám ná noc.
Wſtał potem bárzo ráno dniá piątego / chcąc iść ; ále mówił oćiec oney dźiewki : Pośil proƺę ſerce twoje ; y zábáwili śię / áż śię dźień náchylił / á jedli obá ſpołu.
Wſtał tedy on mąż / áby ƺedł ſám y záłożnicá jego / y ſługá jego / któremu rzekł świekier jego / oćiec oney dźiewki : Oto śię już dźień náchylił ku wiecżorowi / przenocujćież tu proƺę ; oto ſchodźi dźień / przenocujże tu / á bądź dobrej myśli / á jutro ráno wypráwićie śię w drogę ſwą / y pójdźieƺ do przybytku twego.
Tedy on mąż nie chćiał zoſtáć ná noc / ále wſtał y odƺedł / á przyƺedł áż ku Jebus / ( które jeſt Jeruzálem ) májąc z ſobą dwu oſłów z brzemiony / y záłożnicę ſwoję.
A gdy byli bliſko Jebus / á dźień śię już bárzo náchylił / tedy rzekł ſługá do páná ſwego : Pójdź proƺę / á wſtąpmy do tego miáſtá Jebuzejcżyków / y przenocujmy w nim.
Któremu odpowiedźiał pán jego : Nie wſtępujmy do miáſtá cudzoźiemców / które nie jeſt z ſynów Izráelſkich / ále idźmy áż do Gábáá.
Nádto rzekł do ſługi ſwego : Pójdź / ábyſmy przyƺli ná jedno z tych miejſc / y przenocowáli álbo w Gábáá álbo w Rámá.
A minąwƺy poƺli : y záƺło im ſłońce u Gábáá / które jeſt pokolenia Benjáminowego.
Y událi śię tám / áby wƺedƺy przenocowáli w Gábáá ; á gdy wƺedł / uśiádł ná ulicy w mieśćie / przeto że nie był / ktoby je przyjął w dom y przenocował.
A oto / mąż ſtáry ƺedł od roboty ſwojey z polá w wiecżór ; á ten mąż był z góry Efráim / będąc przychodniem w Gábáá / ále ludźie miejſcá onego byli ſynowie Jemini.
Ten podnióſƺy ocży ſwe ujrzał mężá onego podróżnego ná ulicy miáſtá / y rzekł do niego ſtárzec : Dokąd idźieƺ / y ſkądeś przyƺedł?
Któremu on odpowiedźiał : Idźiemy z Betlehem Judá áż ku ſtronie góry Efráimowej / ſkądem jeſt ; bom chodźił do Betlehem Judſkiego ; á teraz idę do domu Páńſkiego / ále niemáƺ nikogo / coby mię przyjął w dom ;
Choć y plewy y śiáno mam dla oſłów náƺych / tákże chleb y wino mam dla śiebie y dla ſłużebnicy twej y dla ſługi / który jeſt ze mną / ſługą twoim ; niemám niedoſtátku z żádney rzecży.
Tedy mu rzekł on mąż ſtáry : Nie fráſuj śię ; cżegoćkolwiek nie doſtánie / to ja opátrzę ; tylko ná ulicy nie zoſtáwáj przez noc.
Wwiódł go tedy do domu ſwego / y dał oſłom obrok ; potem umywƺy nogi ſwoje / jedli y pili.
A gdy rozweſelili ſerce ſwoje / oto / mężowie miáſtá tego / mężowie niepobożni / obſtąpili dom / kołácąc we drzwi / y rzekli do goſpodárzá domu onego / do mężá ſtárego / mówiąc : Wywiedź mężá / który wƺedł w dom twój / ábyſmy go poználi.
A wyƺedƺy do nich on mąż / goſpodárz domu / rzekł im : Nie ták bráćia moi : nie cżyńćie proƺę tey złośći / gdyż wƺedł ten mąż do domu mego / nie cżyńćież tey ſproſnośći.
Oto córká mojá pánná / y záłożnicá jego / wywiodę je záráz / że je obelżyćie / á ucżynićie z niemi / co śię wam będźie dobrego zdáło ; tylko mężowi temu nie cżyńćie tey zelżywośći.
Ale nie chćieli oni mężowie ſłucháć głoſu jego ; przetoż wźiąwƺy on mąż záłożnicę ſwoję / wywiódł ją do nich ná dwór ; y poználi ją / á cżynili jey gwáłt przez cáłą noc áż do záránku / á potem puśćili ją / gdy wſchodźiłá zorzá.
A przyƺedƺy oná niewiáſtá ná świtániu / upádłá u drzwi domu onegoż mężá / gdźie był pán jey / áż śię rozedniáło.
Potem wſtáwƺy pán jey ráno / otworzył drzwi u domu / y wyƺedł / chcąc iść w drogę ſwoję / á oto / oná niewiáſtá / záłożnicá jego / leżáłá u drzwi domu / á ręce jey były ná progu.
Y rzekł do niej : Wſtáń á pójdźmy ; ále nic nie odpowiedźiáłá. Wźiąwƺy ją tedy ná oſłá / wſtał on mąż / y ƺedł do miejſcá ſwego.
Tám przyƺedƺy w dom ſwój / porwał miecż / á zdjąwƺy záłożnicę ſwoję rozrąbał ją z kośćiámi jey ná dwánáśćie ƺtuk / y rozeſłał ją po wƺyſtkich gránicách Izráelſkich.
A ktokolwiek to widźiał / mówił : Nigdy śię to nie ſtáło / áni co tákowego widźiáno od onego dniá / jáko wyƺli ſynowie Izráelſcy z źiemi Egipſkiej / áż do tego dniá ; uwáżájćież to z pilnośćią / á rádźćie y mówćie o tym.
