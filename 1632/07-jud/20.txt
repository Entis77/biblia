Wyƺli tedy wƺyſcy ſynowie Izráelſcy / á zgromádziło śię wƺyſtko poſpólſtwo jednomyślnie od Dán áż do Beerſebá / y do ziemi Gáláád do Páná do Máſfy.
Y ſtánęli przedniejśi wƺyſtkiego ludu y wƺyſtkie pokoleniá Izráelſkie w zgromádzeniu ludu Bożego / cżtery kroć ſto tyśięcy ludu pieƺego / godnego do boju.
( Y uſłyƺeli ſynowie Benjámin / iż śię zebráli ſynowie Izráelſcy w Máſfá. ) Rzekli tedy ſynowie Izráelſcy : Powiedzćie / jáko śię ſtáł ten zły ucżynek?
Y odpowiedziáł on mąż Lewitá / máłżonek niewiáſty zábitej / y rzekł : Do Gábáá / które jeſt w Benjámin / przyƺedłem / já y záłożnicá mojá / ábym tám przenocowáł.
Y powſtáli przećiwko mnie mężowie z Gábáá / á obſtąpili około mnie dom w nocy / umyśliwƺy mię zábić ; ále záłożnicę moję ták gwáłćili / áż umárłá.
Wziąłem tedy záłożnicę moję / y rozrąbáłem ją ná ƺtuki / y rozeſłáłem ją do wƺyſtkich kráin dziedzictwá Izráelſkiego ; ábowiem śię dopuśćili w Izráelu hániebnego y ſproſnego ucżynku.
Otośćie wy wƺyſcy ſynowie Izráelſcy ; uwáżćież to między ſobą / á rádźćie o tem.
Y powſtáł wƺyſtek lud jednoſtájnie / mówiąc : Nie pójdzie nikt do námiotu ſwego / áni odejdzie kto do domu ſwego.
Ale teráz to ucżynimy miáſtu Gábáá / rzućiwƺy loſ przećiwko niemu ;
Weźmiemy dzieśięć mężów ze ſtá w káżdem pokoleniu Izráelſkiem / á ſto z tyśiącá / á tyśiąc z dzieśięćiu tyśięcy / żeby dodáwáli żywnośći ludowi / który przyćiągnie do Gábáá Benjámin / y pomśći śię nád nim wƺyſtkiej ſproſnośći / której śię dopuśćili w Izráelu.
A ták zebráł śię wƺyſtek lud Izráelſki przećiwko miáſtu / zmówiwƺy śię jednoſtájnie.
Y poſłáły pokoleniá Izráelſkie poſły do wƺyſtkich domów ſynów Benjáminowych / mówiąc : Co to zá zły ucżynek / który śię ſtáł między wámi?
Przetoż teráz wydájćie męże niepobożne / którzy ſą w Gábáá / ábyśmy je pozábijáli / á uprzątnęli złe z Izráelá ; ále nie chćieli ſynowie Benjáminowi ſłucháć głoſu bráći ſwych / ſynów Izráelſkich.
Owƺem zgromádzili śię ſynowie Benjáminowi z miáſt ſwoich do Gábáá / áby wálcżyli przećiw ſynom Izráelſkim.
Y nálicżono ſynów Benjáminowych dniá onego z miáſt ich dwádzieśćiá y ƺeść tyśięcy mężów godnych do boju / oprócż obywáteli Gábáá / których nálicżono śiedm ſet mężów ná wybór.
Między tym wƺyſtkim ludem było śiedm ſet mężów ná wybór / którzy nie używáli ręki ſwej práwej / á káżdy z nich ćiſkájąc z procy kámieniem / y włoſá nie chybiáł.
Mężów záśię Izráelſkich nálicżono / oprócż ſynów Benjáminowych / cżtery kroć ſto tyśięcy mężów wálecżnych / y wƺyſtko godnych do boju.
Wſtáwƺy tedy ƺli do domu Bożego / y rádzili śię Bogá / á mówili ſynowie Izráelſcy : Któż zá náſ pójdzie wprzód ná wojnę przećiw ſynom Benjáminowym? Y odpowiedziáł Pán : Judá wprzód pójdzie.
A ták wſtáwƺy ſynowie Izráelſcy ráno / położyli śię obozem przećiw Gábáá.
A wyƺedłƺy mężowie Izráelſcy ku bitwie przećiw ſynom Benjáminowym / uƺykowáli śię mężowie Izráelſcy ku potykániu przećiw Gábáá.
Ale wyƺedłƺy ſynowie Benjáminowi z Gábáá / porázili z Izráelá dniá onego dwádzieśćiá y dwá tyśiące mężów ná głowę.
Potem pokrzepiwƺy śię mężowie ludu Izráelſkiego znowu śię uƺykowáli ku bitwie ná onemże miejſcu / gdzie śię byli uƺykowáli dniá pierwƺego.
Pierwej jednák poƺli ſynowie Izráelſcy / y płákáli przed Pánem áż do wiecżorá / y pytáli śię Páná / mówiąc : Izáli jeƺcże mámy iść wálcżyć przećiwko ſynom Benjáminá / brátá náƺego? Y rzekł Pán : Idźćie przećiwko nim.
Y ruƺyli śię ſynowie Izráelſcy przećiwko ſynom Benjáminowym drugiego dniá.
A wypádłƺy ſynowie Benjáminowi przećiwko nim z Gábáá drugiego dniá / porázili ſynów Izráelſkich znowu ośmnáśćie tyśięcy mężów ná głowę / wƺyſtko mężów wálecżnych.
Przetoż ƺli wƺyſcy ſynowie Izráelſcy / y wƺyſtek lud / á przyƺli do domu Bożego / y płácżąc trwáli tám przed Pánem / y pośćili dniá onego áż do wiecżorá / ofiárując cáłopáleniá / y ofiáry ſpokojne przed oblicżem Páńſkim.
Y pytáli ſynowie Izráelſcy Páná / ( bo tám byłá ſkrzyniá przymierzá Bożego ná on cżáſ ;
A Fineeſ / ſyn Eleázárá / ſyná Aáronowego / ſtáł przed nią ná ten cżáſ / ) mówiąc : Mámyli jeƺcże wynijść ná wojnę przećiwko ſynom Benjáminá / brátá náƺego / cżyli zániecháć? Y odpowiedziáł Pán : Idźćie ; bo jutro dám je w ręce wáƺe.
Tedy pocżynił Izráel záſádzki przećiw Gábáá zewƺąd w około.
A ruƺywƺy śię ſynowie Izráelſcy przećiwko ſynom Benjáminowym dniá trzećiego / uƺykowáli śię przećiw Gábáá / jáko pierwƺy y wtóry ráz.
Wyƺli też ſynowie Benjáminowi przećiwko ludowi / á odſádziwƺy śię od miáſtá / pocżęli bić lud y śiec / jáko pierwƺy y wtóry ráz po drogách / ( z których jedná ƺłá do Betel / á drugá do Gábáá / ) y po polu / á zábili około trzydzieſtu mężów z Izráelá.
Y rzekli ſynowie Benjáminowi : Poráżeni będą od náſ jáko y pierwej ; lecż ſynowie Izráelſcy mówili : Ućiekájmy / á uwiedźmy je od miáſtá / áż ná wielkie drogi.
Zátem wƺyſcy ſynowie Izráelſcy wſtáwƺy z miejſcá ſwego uƺykowáli śię w Bááltámár ; záſádzki też Izráelſkie wyƺły z miejſcá ſwego / z łąk Gábáá.
A ták przyƺło przed Gábáá dzieśięć tyśięcy mężów ná wybór ze wƺyſtkiego Izráelá / y byłá bitwá ſrogá / á oni nie widzieli / że ich nieƺcżęśćie potkáć miáło.
Y poráził Pán Benjáminá przed twárzą Izráelá / á zábili ſynowie Izráelſcy z Benjáminá dniá onego dwádzieśćiá y pięć tyśięcy y ſto mężów / wƺyſtko godnych do boju.
A widząc ſynowie Benjáminowi / że byli poráżeni / ( bo mężowie Izráelſcy uſtępowáli z plácu przed Benjáminem / ufájąc záſádzkom / które byli ucżynili przećiw Gábáá ;
A ći / co byli ná záſádzce / poſpieƺyli śię / y uderzyli ná Gábáá / á wpádłƺy pobili oſtrzem miecżá wƺyſtkie / którzy byli w mieśćie.
Abowiem znák poſtáwiony mieli mężowie Izráelſcy z onymi / co byli w záſádzce / miánowićie / że gdyby dym wielki wypuśćili z miáſtá /
Tedy śię mężowie Izráelſcy obróćili ku bitwie. Synowie záś Benjáminowi pocżęli bić y śiec / y zábili z mężów Izráelſkich około trzydzieſtu mężów ; bo rzekli : Záiſte poráżeni ſą przed námi jáko y w pierwƺej bitwie.
Ale gdy płomień y dym jáko ſłup pocżął wzgórę wſtępowáć z miáſtá / tedy obejrzáwƺy śię ſynowie Benjáminowi názád / ujrzeli / á oto / ogień z miáſtá / wſtępowáł áż ku niebu. )
A iż śię mężowie Izráelſcy obróćili / potrwożyli śię mężowie Benjáminowi / widząc / że nieƺcżęśćie náſtępowáło ná nie.
Y ućiekli przed mężámi Izráelſkimi drogą ku puƺcży ; á wojſko dogániáło ich / y ći / którzy wybieżeli z miáſt / bili je między ſobą.
Ogárnęli tedy Benjáminá / y gonili je bez przeſtánku / á wpárli je áż do Gábáá ná wſchód ſłońcá.
Poległo tedy z Benjáminá / ośmnáśćie tyśięcy mężów / wƺyſtko mężów dużych.
A z tych / którzy obróćiwƺy śię ućiekáli ná puƺcżą / ná ſkáłę Remmon / łápájąc je po drogách / zábili pięć tyśięcy mężów / á gonili je áż do Giedeon / gdzie zámordowáli z nich dwá tyśiące mężów.
A ták było wƺyſtkich / którzy polegli z Benjáminá dniá onego / dwádzieśćiá y pięć tyśięcy mężów wálecżnych / wƺyſtko mężów dużych.
Tylko śię obróćiło y ućiekło ná puƺcżę / ná ſkáłę Remmon / ƺeść ſet mężów / y zoſtáli ná ſkále Remmon przez cżtery mieśiące.
Potem mężowie Izráelſcy wróćiwƺy śię do ſynów Benjáminowych / wybili je oſtrzem miecżá / w mieśćie / pocżąwƺy od ludzi áż do bydlęćiá / y do wƺyſtkiego / co ználeźli ; przytem y wƺyſtkie miáſtá / które pozoſtáły / popálili ogniem.
