Y mówił im : Zápráwdę powiádám wám / iż ſą niektórzy z tych / co tu ſtoją / którzy nie ukuƺą śmierći / áżby ujrzeli / że króleſtwo Boże przyƺło w mocy.
A po ƺeśćiu dniách wziął z ſobą Jezuſ Piotrá / Jákóbá y Jáná / y wiódł je ná górę wyſoką ſáme oſobno / y przemienił śię przed nimi.
A ƺáty jego ſtáły śię lśniące / y bárdzo biáłe jáko śnieg / ják ich blechárz ná ziemi nie może wybielić.
Y ujrzeli Elijáƺá z Mojżeƺem / którzy rozmáwiáli z Jezuſem.
A odpowiádájąc Piotr / rzekł Jezuſowi : Miſtrzu! dobrze nám tu być ; przetoż ucżynimy tu trzy námioty / tobie jeden / Mojżeƺowi jeden / y Elijáƺowi jeden.
Abowiem nie wiedziáł / coby mówił ; bo przeſtráƺeni byli.
Y ſtáł śię obłok / który je záćienił / á przyƺedł głoſ z obłoku mówiący : Ten jeſt Syn mój miły / tegoż ſłuchájćie.
A wnet obejrzáwƺy śię / nikogo więcej nie widzieli / tylko Jezuſá ſámego z ſobą.
A gdy oni zſtępowáli z góry / przykázáł im / áby tego nikomu nie powiádáli / co widzieli / áż kiedyby Syn cżłowiecży zmártwychwſtáł.
A ták oni zátrzymáli tę rzecż u śiebie / pytájąc śię między ſobą / coby to było zmártwychwſtáć.
Y pytáli go / mówiąc : Cóż tedy náucżeni w Piśmie powiádáją / że Elijáƺ pierwej przyjść má?
A on odpowiádájąc / rzekł im : Elijáƺći przyƺedłƺy pierwej / nápráwi wƺyſtko / á jáko nápiſáno o Synu cżłowiecżym / że muśi wiele ućierpieć / á zá nic pocżytánym być.
Aleć wám powiádám / że y Elijáƺ przyƺedł / y ucżynili mu cokolwiek chćieli / jáko o nim nápiſáno.
A przyƺedłƺy do ucżniów / ujrzáł lud wielki około nich / y náucżone w Piśmie ſpór májące z nimi.
A wnetże lud wƺyſtek ujrzáwƺy go / polękáli śię / y zbieżáwƺy śię / przywitáli go.
Y pytáł náucżonych w Piśmie : O cóż ſpór máćie między ſobą?
A odpowiádájąc jeden z onego ludu / rzekł : Náucżyćielu! przywiodłem do ćiebie ſyná mego / który má duchá niemego.
Ten gdzie go kolwiek popádnie / rozdzierá go / á on śię ślini / y zgrzytá zębámi ſwemi y ſchnie ; y mówiłem ucżniom twoim / áby go wygnáli ; ále nie mogli.
Lecż on odpowiádájąc mu / rzekł : O rodzáju niewierny! dokądże z wámi będę? y dokądże wáſ ćierpieć będę? przywiedźćie go do mnie.
Y przywiedli go do niego ; á ſkoro go ujrzáł / záráz go duch rozdárł / á on upádłƺy ná ziemię / przewrácáł śię / śliniąc śię.
Zátem ſpytáł Jezuſ ojcá jego : Jákoż mu śię to dáwno przydáło? A on powiedziáł : Z dziećińſtwá.
Y cżęſto go miotáł w ogień y w wodę / żeby go ſtráćił / ále możeƺli co / rátuj náſ / użáliwƺy śię nád námi.
Ale mu Jezuſ rzekł : Jeźli możeƺ temu wierzyć? Wƺyſtko jeſt możno wierzącemu.
A zárázem záwołáwƺy ojćiec onego młodzieńcá / ze łzámi rzekł : Wierzę / Pánie! ty rátuj niedowiárſtwá mego.
A widząc Jezuſ / iż śię lud zbiegáł / zgromił onego duchá niecżyſtego / mówiąc mu : Duchu niemy y głuchy! já tobie rozkázuję / wynijdź z niego / á nie wchodź więcej weń.
Záwołáwƺy tedy bárdzo / rozdárłƺy go / wyƺedł ; y ſtáł śię on cżłowiek jáko umárły / ták / że ich wiele mówiło / iż umárł.
Ale Jezuſ ująwƺy go zá rękę / podnióſł go ; y wſtáł.
A gdy wƺedł w dom / pytáli go oſobno ucżniowie jego : Cżemużeśmy go wygnáć nie mogli?
A on im rzekł : Ten rodzáj dyjábłów inácżej wynijść nie może / tylko przez modlitwę y przez poſt.
A ſtámtąd wyƺedłƺy / ƺli z ſobą przez Gálileję ; ále nie chćiáł / áby kto o tem wiedziáł.
Abowiem ucżył ucżnie ſwoje / y mówił im : Syn cżłowiecży będzie wydány w ręce ludzkie / y zábiją go ; ále gdy będzie zábity / dniá trzećiego zmártwychwſtánie.
Lecż oni tej rzecży nie rozumieli ; wƺákże báli śię go ſpytáć.
Zátem przyƺedł do Kápernáum / á będąc w domu / pytáł ich : O cżemżeśćie w drodze między ſobą rozmáwiáli?
Lecż oni milcżeli ; ábowiem rozmáwiáli między ſobą w drodze / ktoby z nich był więkƺy.
A uśiádłƺy / záwołáł dwunáſtu y mówił im : Jeźli kto chce być pierwƺym / niech będzie ze wƺyſtkich oſtátnim / y ſługą wƺyſtkich.
A wziąwƺy dziećiątko / poſtáwił je w pośrodku nich / á wziąwƺy je ná ręce / rzekł im :
Ktoby jedno z tákich dziátecżek przyjął w imieniu mojem / mnie przyjmuje ; á ktoby mnie przyjął / nie mnie przyjmuje / ále onego / który mię poſłáł.
Tedy mu odpowiedziáł Ján / mówiąc : Náucżyćielu! widzieliśmy niektórego w imieniu twojim dyjábły wygániájącego / który nie chodzi zá námi / y zábrániáliśmy mu / przeto że nie chodzi zá námi.
Ale Jezuſ rzekł : Nie zábrániájćie mu ; ábowiem nikt nie jeſt / coby cżynił cudá w imieniu mojem / áby mógł ſnádnie mówić źle o mnie.
Bo kto nie jeſt przećiwko nám / zá námi jeſt.
Abowiem ktoby wáſ nápoił kubkiem wody w imieniu mojem / dlá tego / iżeśćie Chryſtuſowi / zápráwdę powiádám wám / nie ſtráći zápłáty ſwojej.
A ktoby zgorƺył jednego z tych máłych / którzy w mię wierzą / dálekoby mu lepiej było / áby był záwieƺony kámień młyńſki u ƺyi jego / y w morze był wrzucony.
A jeźliby ćię gorƺyłá ręká twojá / odetnij ją ; bo lepiej jeſt tobie ułomnym wnijść do żywotá / niżeli dwie ręce májąc / iść do piekłá w on ogień nieugáƺony /
Gdzie robák ich nie umierá / á ogień nie gáśnie.
A jeźliby ćię nogá twojá gorƺyłá / odetnij ją ; bo lepiej tobie chromym wnijść do żywotá / niżeli dwie nogi májąc / być wrzuconym do piekłá / w ogień nieugáƺony /
Gdzie robák ich nie umierá / á ogień nie gáśnie.
A jeźliby ćię oko twoje gorƺyło / wyłup je ; bo lepiej tobie jednookim wnijść do króleſtwá Bożego / niżeli dwá ocży májąc / wrzuconym być do ogniá piekielnego.
Gdzie robák ich nie umierá / á ogień nie gáśnie.
Abowiem káżdy cżłowiek ogniem oſolony będzie / y káżdá ofiárá ſolą oſoloná będzie.
Dobráć jeſt ſól ; ále jeźli śię ſól nieſłoną ſtánie / cżemże ją oſolićie? Miejćież ſól ſámi w ſobie / á miejćie pokój między ſobą.
