Y mówił im : Zápráwdę powiádám wám / iż ſą niektórzy z tych / co tu ſtoją / którzy nie ukuƺą śmierći / áżby ujrzeli / że króleſtwo Boże przyƺło w mocy.
A po ƺeśćiu dniách wźiął z ſobą JEzuſ Piotrá / Jákóbá y Jáná / y wiódł je ná górę wyſoką ſáme oſobno / y przemienił śię przed nimi.
A ƺáty jego ſtáły śię lśniące / y bárdzo biáłe jáko śnieg / ják ich blechárz ná źiemi nie może wybielić.
Y ujrzeli Elijáƺá z Mojżeƺem / którzy rozmáwiáli z JEzuſem.
A odpowiádájąc Piotr / rzekł JEzuſowi : Miſtrzu! dobrze nám tu być ; przetoż ucżynimy tu trzy námioty / tobie jeden / Mojżeƺowi jeden / y Elijáƺowi jeden.
Abowiem nie wiedźiáł / coby mówił ; bo przeſtráƺeni byli.
Y ſtáł śię obłok / który je záćienił / á przyƺedł głoſ z obłoku mówiący : Ten jeſt Syn mój miły / tegoż ſłuchájćie.
A wnet obejrzáwƺy śię / nikogo więcey nie widźieli / tylko JEzuſá ſámego z ſobą.
A gdy oni zſtępowáli z góry / przykázáł im / áby tego nikomu nie powiádáli / co widźieli / áż kiedyby Syn cżłowiecży zmartwychwſtáł.
A ták oni zátrzymáli tę rzecż u śiebie / pytájąc śię między ſobą / coby to było zmartwychwſtáć.
Y pytáli go / mówiąc : Cóż tedy náucżeni w Piśmie powiádáją / że Elijáƺ pierwej przyjść má?
A on odpowiádájąc / rzekł im : Elijáƺći przyƺedłƺy pierwej / nápráwi wƺyſtko / á jáko nápiſáno o Synu cżłowiecżym / że muśi wiele ućierpieć / á zá nic pocżytánym być.
Aleć wám powiádám / że y Elijáƺ przyƺedł / y ucżynili mu cokolwiek chćieli / jáko o nim nápiſáno.
A przyƺedłƺy do ucżniów / ujrzáł lud wielki około nich / y náucżone w Piśmie ſpór májące z nimi.
A wnetże lud wƺyſtek ujrzáwƺy go / polękáli śię / y zbieżáwƺy śię / przywitáli go.
Y pytáł náucżonych w Piśmie : O cóż ſpór máćie między ſobą?
A odpowiádájąc jeden z onego ludu / rzekł : Náucżyćielu! przywiodłem do ćiebie ſyná mego / który má duchá niemego.
Ten gdźie go kolwiek popádnie / rozdźierá go / á on śię ślini / y zgrzytá zębámi ſwemi y ſchnie ; y mówiłem ucżniom twoim / áby go wygnáli ; ále nie mogli.
Lecż on odpowiádájąc mu / rzekł : O rodzáju niewierny! dokądże z wámi będę? y dokądże was ćierpieć będę? przywiedźćie go do mnie.
Y przywiedli go do niego ; á ſkoro go ujrzáł / záráz go duch rozdárł / á on upádłƺy ná źiemię / przewrácáł śię / śliniąc śię.
Zátem ſpytáł JEzuſ ojcá jego : Jákoż mu śię to dáwno przydáło? A on powiedźiáł : Z dźiećińſtwá.
Y cżęſto go miotáł w ogień y w wodę / żeby go ſtráćił / ále możeƺli co / rátuj nas / użáliwƺy śię nád námi.
Ale mu JEzuſ rzekł : Jeſli możeƺ temu wierzyć? Wƺyſtko jeſt możno wierzącemu.
A zárázem záwołáwƺy ojćiec onego młodźieńcá / ze łzámi rzekł : Wierzę / Pánie! ty rátuj niedowiárſtwá mego.
A widząc JEzuſ / iż śię lud zbiegáł / zgromił onego duchá niecżyſtego / mówiąc mu : Duchu niemy y głuchy! ja tobie rozkázuję / wynijdź z niego / á nie wchodź więcey weń.
Záwołáwƺy tedy bárdzo / rozdárłƺy go / wyƺedł ; y ſtáł śię on cżłowiek jáko umárły / ták / że ich wiele mówiło / iż umárł.
Ale JEzuſ ująwƺy go zá rękę / podnióſł go ; y wſtáł.
A gdy wƺedł w dom / pytáli go oſobno ucżniowie jego : Cżemużeśmy go wygnáć nie mogli?
A on im rzekł : Ten rodzáj dyjábłów inácżej wynijść nie może / tylko przez modlitwę y przez poſt.
A ſtámtąd wyƺedłƺy / ƺli z ſobą przez Gálileję ; ále nie chćiáł / áby kto o tem wiedźiáł.
Abowiem ucżył ucżnie ſwoje / y mówił im : Syn cżłowiecży będźie wydány w ręce ludzkie / y zábiją go ; ále gdy będźie zábity / dniá trzećiego zmartwychwſtánie.
Lecż oni tey rzecży nie rozumieli ; wƺákże báli śię go ſpytáć.
Zátem przyƺedł do Kápernáum / á będąc w domu / pytáł ich : O cżemżeśćie w drodze między ſobą rozmáwiáli?
Lecż oni milcżeli ; ábowiem rozmáwiáli między ſobą w drodze / ktoby z nich był więkƺy.
A uśiádłƺy / záwołáł dwunáſtu y mówił im : Jeſli kto chce być pierwƺym / niech będźie ze wƺyſtkich oſtátnim / y ſługą wƺyſtkich.
A wźiąwƺy dźiećiątko / poſtáwił je w pośrodku nich / á wźiąwƺy je ná ręce / rzekł im :
Ktoby jedno z tákich dźiátecżek przyjął w imieniu mojem / mnie przyjmuje ; á ktoby mnie przyjął / nie mnie przyjmuje / ále onego / który mię poſłáł.
Tedy mu odpowiedźiáł Ján / mówiąc : Náucżyćielu! widźieliśmy niektórego w imieniu twojim dyjábły wygániájącego / który nie chodźi zá námi / y zábrániáliśmy mu / przeto że nie chodźi zá námi.
Ale JEzuſ rzekł : Nie zábrániájćie mu ; ábowiem nikt nie jeſt / coby cżynił cudá w imieniu mojem / áby mógł ſnádnie mówić źle o mnie.
Bo kto nie jeſt przećiwko nám / zá námi jeſt.
Abowiem ktoby was nápoił kubkiem wody w imieniu mojem / dla tego / iżeśćie CHryſtuſowi / zápráwdę powiádám wám / nie ſtráći zápłáty ſwojey.
A ktoby zgorƺył jednego z tych máłych / którzy w mię wierzą / dálekoby mu lepiej było / áby był záwieƺony kámień młyńſki u ƺyi jego / y w morze był wrzucony.
A jeſliby ćię gorƺyłá ręká twojá / odetnij ją ; bo lepiej jeſt tobie ułomnym wnijść do żywotá / niżeli dwie ręce májąc / iść do piekłá w on ogień nieugáƺony /
Gdźie robák ich nie umierá / á ogień nie gáśnie.
A jeſliby ćię nogá twojá gorƺyłá / odetnij ją ; bo lepiej tobie chromym wnijść do żywotá / niżeli dwie nogi májąc / być wrzuconym do piekłá / w ogień nieugáƺony /
Gdźie robák ich nie umierá / á ogień nie gáśnie.
A jeſliby ćię oko twoje gorƺyło / wyłup je ; bo lepiej tobie jednookim wnijść do króleſtwá Bożego / niżeli dwá ocży májąc / wrzuconym być do ogniá piekielnego.
Gdźie robák ich nie umierá / á ogień nie gáśnie.
Abowiem káżdy cżłowiek ogniem oſolony będźie / y káżdá ofiárá ſolą oſoloná będźie.
Dobráć jeſt ſól ; ále jeſli śię ſól nieſłoną ſtánie / cżemże ją oſolićie? Miejćież ſól ſámi w ſobie / á miejćie pokój między ſobą.
