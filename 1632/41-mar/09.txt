Y mówił im : Záprawdę powiádam wam / iż ſą niektórzy z tych co tu ſtoją / którzy nie ukuƺą śmierći / áżby ujrzeli / że Króleſtwo Boże przyƺło w mocy.
A po ƺeśći dniách / wźiął <i>z ſobą</i> JEzus Piotrá / Jákubá / y Janá / y wiódł je ná górę wyſoką ſáme oſobno / y przemienił śię przed nimi.
A ƺáty jego ſtáły śię łſniące <i>y</i> bárzo białe jáko śnieg / jákich blecharz ná źiemi nie może wybielić.
Y ujrzeli Eliaƺá z Mojzeƺem / którzy rozmawiáli z JEzuſem.
A odpowiádájąc Piotr / rzekł JEzuſowi : Miſtrzu / dobrze nam tu być ; przetoż ucżynimy <i>tu</i> trzy namioty / tobie jeden / Mojzeƺowi jeden / y Eliaƺowi jeden.
Abowiem nie wiedźiał coby mówił : bo przeſtráƺeni byli.
Y ſtał śię obłok który je záćienił / á przyƺedł głos z obłoku mówiący : Ten jeſt Syn mój miły / tegoż ſłuchajćie.
A wnet obejrzawƺy śię / nikogo więcey nie widźieli / tylko JEzuſá ſámego z ſobą.
A gdy oni zſtępowáli z góry / przykazał im áby tego nikomu nie powiádáli co widźieli / áż kiedyby Syn cżłowiecży zmartwychwſtał.
A ták <i>oni</i> zátrzymáli tę rzecż u śiebie / pytájąc śię miedzy ſobą coby to było zmartwychwſtáć?
Y pytáli go / mówiąc : Cóż tedy Náucżeni w piśmie powiádáją / że Eliaƺ pierwey przyść ma?
A on odpowiádájąc / rzekł im : Eliaƺći przyƺedƺy pierwey nápráwi wƺyſtko / á jáko nápiſano o Synu cżłowiecżym / <i>że muśi</i> wiele ućierpieć á zá nic pocżytánym bydź.
Aleć wam powiádam / że y Eliaƺ przyƺedł / y ucżynili mu cokolwiek chćieli / jáko o nim nápiſano.
A przyƺedƺy do ucżniów / ujrzał lud wielki około nich / y Náucżone w piśmie ſpór májące z nimi.
A wnetże lud wƺyſtek ujrzawƺy go / polękáli śię : y zbieżawƺy śię / przywitáli go.
Y pytał Náucżonych w piśmie : O cóż ſpór maćie miedzy ſobą?
A odpowiádájąc jeden z onego ludu / rzekł : Náucżyćielu / przywiodłem do ćiebie ſyná mego / który ma duchá niemego.
Ten gdźie go kolwiek popádnie / rozdźiera go / á on śię ślini / y zgrzyta zębámi ſwymi / y ſchnie : y mówiłė ucżniom twojim áby go wygnáli ; ále nie mogli.
Lecż on odpowiádájąc mu / rzekł : O rodzáju niewierny! Dokądże z wámi będę? y dokądże was ćierpieć będę? Przywiedźćie go do mnie.
Y przywiedli go do niego. A ſkoro go ujrzał / záraz go duch rozdárł : á on upadƺy ná źiemię / przewracał śię śliniąc śię.
Zátym zpytał JEzus Ojcá jego : Jákoż mu śię to dawno przydáło? A on powiedźiał : Z dźiećińſtwá.
Y cżęſto go miotał w ogień y w wodę / żeby go ſtráćił : Ale możeƺli co / rátuj nas użaliwƺy śię nád námi.
Ale mu JEzus rzekł : Jeſli możeƺ temu wierzyć / wƺyſtko jeſt możno wierzącemu.
A zárázem záwoławƺy oćiec onego młodźieńcá / ze łzámi rzekł : Wierzę PAnie ; ty rátuj niedowiárſtwá mego!
A widząc JEzus iż śię lud zbiegał / zgromił onego duchá niecżyſtego / mówiąc mu : Duchu niemy y głuchy / Ja tobie rozkázuję / wynidź z niego / á nie wchodź więcey weń.
Tedy go bárzo rozdárƺy wyƺedł : y ſtał śię on cżłowiek jáko umárły / ták że ich wiele mówiło iż umárł.
Ale JEzus ująwƺy go zá rękę / podnióſł go ; y wſtał.
A gdy wƺedł w dom / pytáli go oſobno ucżniowie jego : Cżemużeſmy go my wygnáć nie mogli?
A <i>on</i> im rzekł : Ten rodzaj <i>Dyabłów</i> inácżey wyniść nie może / tylko przez modlitwę y przez poſt.
A z támtąd wyƺedƺy / ƺli z ſobą przez Gálileą / ále nie chćiał áby kto o tem wiedźiał.
Abowiem ucżył ucżnie ſwoje / y mówił im : Syn cżłowiecży będźie wydány w ręce ludzkie / y zábiją go : ále gdy będźie zábity / dniá trzećiego <i>zmartwych</i>wſtánie.
Lecż oni tey rzecży nie rozumieli wƺákże bali śię go zpytáć.
Zátym przyƺedł do Kápernáum : á będąc w domu / pytał ich : O cżymżeśćie w drodze miedzy ſobą rozmawiáli.
Lecż oni milcżeli : Abowiem rozmawiáli miedzy ſobą w drodze / ktoby z nich był więkƺy.
A uśiadƺy / záwołał dwunaśći / y mówił im : Jeſli kto chce być pierwƺym / niech będźie ze wƺyſtkich oſtátnim / y ſługą wƺyſtkich.
A wźiąwƺy dźiećiątko / poſtáwił je w pośrzodku nich ; á wźiąwƺy je ná łokćie / rzekł im :
Ktoby jedno z tákich dźiatecżek przyjął w Imieniu mojim / mnie przyjmuje : á ktoby mnie przyjął / nie mnie przyjmuje / ále onego który mię poſłał.
Tedy mu odpowiedźiał Jan / mówiąc : Náucżyćielu / Widźieliſmy niektórego w Imieniu twojim dyabły wyganiájącego / który nie chodźi zá námi ; y zábraniáliſmy mu / przeto że nie chodźi zá námi.
Ale JEzus rzekł : Nie zábraniajćie mu : ábowiem nikt nie jeſt coby cżynił cudá w Imieniu mojim / áby mógł ſnádnie mówić źle o mnie.
Bo kto nie jeſt przećiwko nam / zá námi jeſt.
Abowiem ktoby was nápojił kubkiem wody w Imieniu mojim / <i>dla tego</i> iżeśćie CHryſtuſowi ; záprawdę powiádam wam / nie ſtráći zapłáty ſwojey.
A ktoby zgorƺył jednego z tych máłych którzy w mię wierzą / dálekoby mu lepiey było / áby był záwieƺony kámień młyńſki u ƺyjey jego / y w morze był wrzucony.
A jeſliby ćię gorƺyłá ręká twojá / odetni ją : bo lepiey jeſt tobie ułomnym wniść do żywotá / niżeli dwie ręce májąc / iść do piekłá w on ogień nieugáƺony.
Gdźie robak ich nie umiera / á ogień nie gáśnie.
A jeſliby ćię nogá twojá gorƺyłá / odetni ją : bo lepiey tobie chromym wniść do żywotá / niżeli dwie nodze májąc / być wrzuconym do piekłá w ogień nieugáƺony :
Gdźie robak ich nie umiera / á ogień nie gáśnie.
A jeſliby ćię oko twoje gorƺyło / wyłup je : bo lepiey tobie jednookim wniść do króleſtwá Bożego / niżeli dwie ocży májąc / wrzuconym być do ogniá piekielnego.
Gdźie robak ich nie umiera / á ogień nie gáśnie.
Abowiem káżdy <i>cżłowiek</i> ogniem oſolony będźie / y káżda ofiárá ſolą oſolona będźie.
Dobrać jeſt ſól : ále jeſli śię ſól nieſłoną ſtánie / cżymże ją oſolićie? Miejćież ſól ſámi wſobie / á miejćie pokój miedzy ſobą.
