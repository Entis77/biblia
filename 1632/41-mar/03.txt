Y wƺedł záśię do bóżnice / á był tám cżłowiek który miał rękę uſchłą.
Y podſtrzegáli go jeſliby go w Sábbát uzdrowił / áby go oſkárżyli.
Y rzekł onemu cżłowiekowi który miał rękę uſchłą ; Wyſtąp w pośrzodek.
Y rzekł do nich : Godźili śię w Sábbát dobrze cżynić / cżyli źle? cżłowieká záchowáć / cżyli zábić? A oni milcżeli.
Tedy pojrzawƺy po nich z gniewem / y záſmućiwƺy śię nád zátwárdzeniem ſercá ich / rzekł onemu cżłowiekowi : Wyćiągni rękę twoję ; y wyćiągnął : Y przywrócona jeſt ręká jego do zdrowia jáko y druga.
Tedy wyƺedƺy Fáryzeuƺowie / ucżynili wnet rádę z Herodyány przećiwko niemu / jákoby go ſtráćili.
Ale JEzus z ucżniámi ſwymi uſtąpił nád morze / á wielkie mnóſtwo ƺło zá nim z Gálilejey y z źiemie Judſkiey :
Y z Jeruzalem y z Idumejey / y z Zá-Jordania / y z tych którzy mieƺkáli około Tyru y Sydonu wielkie mnóſtwo / ſłyƺąc ják wielkie rzecży cżynił / przyƺli do niego.
Y rozkazał ucżniom ſwojim áby łódkę mieli záwƺe pogotowiu dla ludu / áby go nie ćiſnęli.
Abowiem ich wiele uzdrawiał / ták iż nań padáli / áby śię go dotykáli / którzykolwiek choroby mieli.
A duchowie niecżyśći gdy go ujrzeli / upadáli przed nim / y wołáli mówiąc : Tyś jeſt Syn Boży.
Ale je on ſrodze gromił / żeby go nie objawiáli.
Y wſtąpił ná górę / á wezwał do śiebie tych których ſam chćiał : y przyƺli do niego.
Y poſtánowił ich dwánaśćie áby z nim byli / á iżby je wyſłał kazáć <i>Ewángelią</i>.
Y żeby mieli moc uzdrawiáć choroby / y wyganiáć dyabły.
Symoná któremu imię dał Piotr :
Y Jákubá <i>Syná</i> Zebedeuƺowego / y Janá brátá Jákubowego : ( którym dał imię Boánerges / to jeſt / ſynowie gromu )
Y Andrzejá / y Filipá / y Bártłomiejá : y Mátteuƺá / y Tomaƺá / y Jákubá <i>ſyná</i> Alfeuƺowego / y Tádeuƺá / y Symoná Chánánejcżyká :
Y Judaƺá Iƺkáryotá / który go też wydał :
Y ƺli <i>z nim</i> do domu. Y zgromádźił śię z nowu lud / ták / iż nie mogli áni chlebá jeść.
A gdy o tym uſłyƺeli jego powinni / przyƺli áby go pojimáli : bo mówili / że odƺedł od rozumu.
A Náucżeni w piśmie którzy byli przyƺli z Jeruzalem / mówili ; Iż ma Beelzebubá / á iż przez kśiążę dyabelſkie wygania Dyabły.
Y wezwawƺy ich mówił do nich w podobieńſtwách : Jákoż może ƺátan ƺátáná wyganiáć?
A jeſliże króleſtwo ſámo w ſobie będźie rozdźielone / nie może śię oſtáć ono króleſtwo.
Y dom jeſliby ſam przećiwko ſobie był rozdźielony / nie będźie śię mógł oſtáć on dom.
<i>Tákći</i> jeſli ƺátan powſtał <i>ſam</i> przećiwko ſobie / y jeſt rozdźielony / nie może śię oſtáć ; ále koniec bierze.
Nikt nie może ſprzętu mocárzowego wƺedƺy do domu jego rozchwyćić / jeſliby pierwey mocárzá <i>onego</i> nie związał ; á potym dom jego ſplundruje.
Záprawdę powiádam wam / że wƺyſtkie grzechy ſynom ludzkim będą odpuƺcżone / y bluźnierſtwá którymibykolwiek bluźnili :
Ale kto bluźni przećiwko Duchowi Świętemu / nie ma odpuƺcżenia ná wieki ; ále winien jeſt ſądu wiecżnego.
Bo mówili : Ma duchá niecżyſtego.
Przyƺli tedy bráćia y mátká jego / á ſtojąc przed domem / poſłáli do niego / y kázáli go záwołáć.
A lud śiedźiał około niego ; y rzekli mu : Oto mátká twojá y bráćia twoji ƺukáją ćię przed domem.
Ale im on odpowiedźiał / mówiąc : Któż jeſt mátká mojá / y bráćia moji?
A pojrzawƺy w koło po tych którzy koło <i>niego</i> śiedźieli / rzekł ; Oto mátká mojá y bráćia moji.
Abowiem ktobykolwiek cżynił wolą Bożą / ten jeſt brát mój / y śioſtrá mojá / y mátká <i>mojá</i>.
