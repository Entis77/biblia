Y wƺedł záśię do bóżnicy / á był tám cżłowiek / który miał rękę uſchłą.
Y podſtrzegáli go / jeſliby go w ſábát uzdrowił / áby go oſkárżyli.
Y rzekł onemu cżłowiekowi / który miał rękę uſchłą : Wyſtąp w pośrodek.
Y rzekł do nich : Godźili śię w ſábát dobrze cżynić / cżyli źle cżynić? cżłowieká záchowáć / cżyli zábić? á oni milcżeli.
Tedy ſpojrzáwƺy po nich z gniewem / y záſmućiwƺy śię nád zátwárdzeniem ſercá ich / rzekł onemu cżłowiekowi : Wyćiągnij rękę twoję! y wyćiągnął ; y przywróconá jeſt ręká jego do zdrowiá jáko y drugá.
Tedy wyƺedłƺy Fáryzeuƺowie / ucżynili wnet rádę z Herodyjány przećiwko niemu / jákoby go ſtráćili.
Ale JEzuſ z ucżniámi ſwymi uſtąpił nád morze / á wielkie mnóſtwo ƺło zá nim z Gálilei y z źiemi Judzkiej /
Y z Jeruzálemu / y z Idumei / y z Zá-Jordániá / y z tych / którzy mieƺkáli około Tyru y Sydonu / wielkie mnóſtwo / ſłyƺąc / ják wielkie rzecży cżynił / przyƺli do niego.
Y rozkázáł ucżniom ſwoim / áby łódkę mieli záwƺe pogotowiu / dla ludu / áby go nie ćiſnęli.
Abowiem ich wiele uzdráwiáł / ták iż náń pádáli / áby śię go dotykáli / którzykolwiek choroby mieli.
A duchowie niecżyśći / gdy go ujrzeli / upádáli przed nim y wołáli / mówiąc : Tyś jeſt Syn Boży!
Ale ich on ſrodze gromił / żeby go nie objáwiáli.
Y wſtąpił ná górę / á wezwáł do śiebie tych / których ſám chćiáł / y przyƺli do niego.
Y poſtánowił ich dwánáśćie / áby z nim byli / á iżby je wyſłáł kázáć Ewángieliją :
Y żeby mieli moc uzdráwiáć choroby y wygániáć dyjábły :
Szymoná / któremu imię dáł Piotr ;
Y Jákóbá / ſyná Zebedeuƺowego / y Jáná / brátá Jákóbowego / ( którym dáł imię Boánergeſ / to jeſt : ſynowie gromu ; )
Y Andrzejá / y Filipá / y Bártłomiejá / y Máteuƺá / y Tomáƺá / y Jákóbá / ſyná Alfeuƺowego / y Tádeuƺá / y Szymoná Chánánejcżyká ;
Y Judáƺá Iƺkáryjotá / który go też wydáł.
Y przyƺli do domu. Y zgromádźił śię znowu lud / ták / iż nie mogli áni chlebá jeść.
A gdy o tem uſłyƺeli jego powinni / przyƺli / áby go pojmáli ; bo mówili / że odƺedł od rozumu.
A náucżeni w Piśmie / którzy byli przyƺli z Jeruzálemu / mówili : Iż má Beelzebubá / á iż przez kśiążęćiá dyjábelſkiego wygániá dyjábły.
Y wezwáwƺy ich / mówił do nich w podobieńſtwách : Jákoż może ƺátán ƺátáná wygániáć?
A jeſliże króleſtwo ſámo w ſobie będźie rozdźielone / nie może śię oſtáć ono króleſtwo.
Y dom / jeſliby ſám przećiwko ſobie był rozdźielony / nie będźie śię mógł oſtáć on dom.
Tákći / jeſli ƺátán powſtáł ſám przećiwko ſobie y jeſt rozdźielony / nie może śię oſtáć / ále koniec bierze.
Nikt nie może ſprzętu mocárzowego / wƺedłƺy do domu jego / rozchwyćić / jeſliby pierwej mocárzá onego nie związáł : á potem dom jego ſplundruje.
Zápráwdę powiádám wám / że wƺyſtkie grzechy ſynom ludzkim będą odpuƺcżone / y bluźnierſtwá / któremibykolwiek bluźnili ;
Ale kto bluźni przećiwko Duchowi Świętemu / niemá odpuƺcżeniá ná wieki / ále winien jeſt ſądu wiecżnego.
Bo mówili : Má duchá niecżyſtego.
Przyƺli tedy bráćia y mátká jego / á ſtojąc przed domem / poſłáli do niego / y kázáli go záwołáć.
A lud śiedźiáł około niego / y rzekli mu : Oto mátká twojá y bráćia twoi ƺukáją ćię przed domem.
Ale im on odpowiedźiáł / mówiąc : Któż jeſt mátká mojá / y bráćia moi?
A ſpojrzáwƺy w około po tych / którzy koło niego śiedźieli / rzekł : Oto mátká mojá / y bráćia moi!
Abowiem ktobykolwiek cżynił wolę Bożą / ten jeſt brát mój / y śioſtrá mojá / y mátká mojá.
