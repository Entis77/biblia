A wſtawƺy z támtąd / przyƺedł do gránic Judſkich przez krajinę zá Jordanem <i>leżącą</i> : y zƺedł śię záś do niego lud ; y ucżył je záś jáko miał zwycżaj.
Tedy przyſtąpiwƺy Fáryzeuƺowie / pytáli go : Godźili śię mężowi żonę opuśćić? <i>A to cżynili</i> kuƺąc go.
Ale on odpowiádájąc / rzekł im : Cóż wam przykazał Mojzeƺ?
A oni rzekli : Mojzeƺ pozwolił nápiſáć liſt rozwodny / y opuśćić <i>ją</i>.
A odpowiádájąc JEzus / rzekł im : Dla zátwárdzenia ſercá wáƺego nápiſał wam to przykazánie.
Aleć od pocżątku ſtworzenia / mężcżyznę y niewiáſtę ucżynił je Bóg.
Dla tego opuśći cżłowiek Ojcá ſwego y Mátkę / á przyłącży śię do żony ſwojey.
Y będą to dwoje jednym ćiáłem : á ták już nie ſą dwoje / ále jedno ćiáło.
Co tedy Bóg złącżył / cżłowiek niechaj nie rozłącża.
A w domu záś ucżniowie jego o toż go pytáli.
Y rzekł im : Ktobykolwiek opuśćił żonę ſwą / á pojąłby inną / cudzołoży przećiwko niey.
A jeſliby niewiáſtá opuśćiłá mężá ſwego / á ƺłáby zá drugiego / cudzołoży.
Tedy przynoƺono do niego dźiatki / áby śię ich dotykał : ále ucżniowie gromili tę / którzy je przynośili.
Co ujrzawƺy JEzus / rozgniewał śię / y rzekł im : Dopuśććie dźiatkom przychodźić do mnie / á nie zábraniajćie im : ábowiem tákich jeſt króleſtwo Boże.
Záprawdę powiádam wam : Ktobykolwiek nieprzyjął króleſtwá Bożego jáko dźiećiątko / nie wnidźie do niego.
Y biorąc je ná łokćie ſwoje / y kłádąc ná nie ręce / błogoſłáwił im.
A gdy on wychodźił w drogę / przybieżał jeden : y upadƺy przed nim ná koláná / pytał go : Náucżyćielu dobry / cóż cżynić mam ábym odźiedźicżył żywot wiecżny?
Ale mu JEzus rzekł : Przecż mię zowieƺ dobrym? Nikt nie jeſt dobry / tylko jeden / <i>to jeſt</i> Bóg.
Przykazánia umieƺ : Nie będźieƺ cudzołożył : Nie będźieƺ zábijał : Nie będźieƺ kradł : Nie będźieƺ mówił świádectwá fałƺywe° : Nie będźieƺ oƺukawał <i>nikogo</i> : Cżći Ojcá twego y Mátkę <i>twoję</i>.
A on odpowiádájąc / rzekł mu : Náucżyćielu / tegom wƺyſtkiego przeſtrzegał od młodośći mojey.
A JEzus pojrzawƺy nań / rozmiłował śię go / y rzekł mu : Jednegoć niedoſtawa : Idź / przedaj co maƺ / á rozdaj ubogim / á będźieƺ miał ſkarb w niebie : á przydź / náśláduj mię wźiąwƺy krzyż.
A on záfráſowawƺy śię dla tego ſłowá / odƺedł ſmutny : ábowiem miał wiele májętnośći.
A pojrzawƺy JEzus w koło / rzekł do ucżniów ſwojich : Jákoż trudno ći którzy máją bogáctwá wnidą do króleſtwá Bożego!
Tedy ucżniowie zdumieli śię nád tymi ſłowy jego. Lecż JEzus záś odpowiádájąc rzekł im : Dźiatki / jákoż jeſt trudno tym co ufáją w bogáctwách wniść do króleſtwá Bożego!
Snádniey jeſt wielbłądowi przejść przez ucho igielne / niż bogacżowi wniść do króleſtwá Bożego.
A oni śię tym więcey zdumiewáli / mówiąc miedzy ſobą : Y któż może być zbáwiony?
A JEzus pojrzawƺy ná nie / rzekł : U ludźić <i>to</i> niemożno / ále nie u Bogá : ábowiė u Bogá wƺyſtko jeſt możno.
Y pocżął Piotr mówić do niego : Otoſmy my opuśćili wƺyſtko / á poƺliſmy zá tobą.
A JEzus odpowiádájąc rzekł : Záprawdę powiádam wam : Nikt nie jeſt / ktoby opuśćił dom / álbo bráćią / álbo śioſtry / álbo ojcá / álbo mátkę / álbo żonę / álbo dźieći / álbo role / dla mnie y dla Ewángeliey :
Żeby nie miał wźiąć ſtokrotnie teraz w tym cżáśie domów / y bráćiey / y śióſtr / y mátek / y dźieći / y ról z prześládowániem ; á w przyƺłym wieku / żywotá wiecżnego.
Aleć wiele pierwƺych będą oſtátnimi / á oſtátnich pierwƺymi.
Y byli w drodze wſtępując do Jeruzalem : á JEzus ƺedł przed nimi / y zdumiewáli śię ; á idąc zá nim / bali śię. A on wźiąwƺy záśię z ſobą onych dwánaśćie / pocżął im powiádáć co nań przyść miáło <i>/</i>
<i>Mówiąc</i> : Oto wſtępujemy do Jeruzalem / á Syn cżłowiecży będźie wydány Przedniejƺym kápłanom y Náucżonym w piśmie / y oſądzą go ná śmierć / y wydádzą go Pogánom.
A <i>oni</i> śię z niego náśmiewáć będą / y ubicżują go / y będą nań plwáć / y zábiją go : ále dniá trzećiego <i>zmartwych</i>wſtánie.
Tedy przyſtąpili do niego / Jákub y Jan / ſynowie Zebedeuƺowi mówiąc : Náucżyćielu / chcemy byś nam ucżynił o co ćię prośić będźiemy.
A on im rzekł : Cóż chcećie ábym wam ucżynił?
A <i>oni</i> mu rzekli : Daj nam ábyſmy jeden ná práwicy twojey / á drugi ná lewicy twojey śiedźieli w chwale twojey.
Lecż im JEzus rzekł : Nie wiećie o co prośićie. Możećież pić kielich który ja piję? y chrztem którym śię Ja chrzcżę bydź ochrzcżeni?
A oni mu rzekli : Możemy. A JEzus im rzekł : Kielichći który Ja piję / pić będźiećie / y chrztem którym śię Ja chrzcżę / ochrzcżeni będźiećie :
Ale śiedźieć po práwicy mojey álbo po lewicy mojey / nie mojá rzecż jeſt dáć ; ále <i>będźie dano</i> tym / którym zgotowano.
A uſłyƺawƺy to oná dźieśięć / pocżęli śię gniewáć ná Jákubá y ná Janá.
Ale JEzus zwoławƺy ich / rzekł im : Wiećie iż ći którym śię zda że władzą máją nád narody / pánują nád nimi ; á którzy z nich wielcy ſą / moc przewodzą nád nimi.
Lecż nie ták będźie miedzy wámi : Ale ktobykolwiek chćiał być wielki miedzy wámi / będźie ſługą wáƺym.
A ktobykolwiek z was chćiał być pierwƺym / będźie ſługą wƺyſtkich.
Bo y Syn cżłowiecży nie przyƺedł áby mu ſłużono / ále áby ſłużył / y áby dał duƺę ſwą ná okup zá wiele ludźi.
Tedy przyƺli do Jerychá : á gdy on wychodźił z Jerychá / y ucżniowie jego / y lud wielki / ſyn Tymeuƺów Bártymeuƺ ślepy śiedźiał podle drogi żebrząc.
A uſłyƺawƺy yż jeſt JEzus on Názáreńſki / pocżął wołáć / mówiąc : JEzuśie Synu Dawidów zmiłuj śię náde mną!
Y gromiło go wiele ich áby milcżał. Ale on tym więcey wołał : Synu Dawidów / zmiłuj śię náde mną!
Tedy záſtánowiwƺy śię JEzus / kazał go záwołáć. Y záwołano ślepego / mówiąc mu : Ufaj / wſtań / woła ćię.
A on porzućiwƺy płaƺcż ſwój / wſtał / y przyƺedł do JEzuſá.
Y odpowiádájąc JEzus / rzekł mu : Cóż chceƺ ábymći ucżynił? A ślepy mu rzekł : Miſtrzu / ábym przejrzał.
A JEzus mu rzekł : Idź / wiárá twojá ćiebie uzdrowiłá. A zárázem przejrzał : y ƺedł drogą zá JEzuſem.
