A wſtáwƺy ſtámtąd / przyƺedł do gránic Judzkich przez kráinę zá Jordánem leżącą ; y zƺedł śię záś do niego lud / y ucżył je záś jáko miáł zwycżáj.
Tedy przyſtąpiwƺy Fáryzeuƺowie / pytáli go : Godzili śię mężowi żonę opuśćić? á to cżynili / kuƺąc go.
Ale on odpowiádájąc / rzekł im : Cóż wám przykázáł Mojżeƺ?
A oni rzekli : Mojżeƺ pozwolił / nápiſáć liſt rozwodny / y opuśćić ją.
A odpowiádájąc Jezuſ / rzekł im : Dlá zátwárdzeniá ſercá wáƺego nápiſáł wám to przykázánie.
Aleć od pocżątku ſtworzeniá mężcżyznę y niewiáſtę ucżynił je Bóg.
Dlátego opuśći cżłowiek ojcá ſwego y mátkę / á przyłącży śię do żony ſwojej /
Y będą dwoje jednem ćiáłem ; á ták już nie ſą dwoje / ále jedno ćiáło.
Co tedy Bóg złącżył / cżłowiek niecháj nie rozłącżá.
A w domu záś ucżniowie jego o toż go pytáli.
Y rzekł im : Ktobykolwiek opuśćił żonę ſwą / á pojąłby inną / cudzołoży przećiwko niej ;
A jeźliby niewiáſtá opuśćiłá mężá ſwego / á ƺłáby zá drugiego / cudzołoży.
Tedy przynoƺono do niego dziátki / áby śię ich dotykáł ; ále ucżniowie gromili tych / którzy je przynośili.
Co ujrzáwƺy Jezuſ / rozgniewáł śię y rzekł im : Dopuśććie dziátkom przychodzić do mnie / á nie zábrániájćie im ; ábowiem tákich jeſt króleſtwo Boże.
Zápráwdę powiádám wám : Ktobykolwiek nie przyjął króleſtwá Bożego jáko dziećiątko / nie wnijdzie do niego.
Y biorąc je ná ręce ſwoje / y kłádąc ná nie ręce / błogoſłáwił im.
A gdy on wychodził w drogę / przybieżáł jeden / y upádłƺy przed nim ná koláná / pytáł go : Náucżyćielu dobry! cóż cżynić mám / ábym odziedzicżył żywot wiecżny?
Ale mu Jezuſ rzekł : Przecż mię zowieƺ dobrym? Nikt nie jeſt dobry / tylko jeden / to jeſt Bóg.
Przykázániá umieƺ : nie będzieƺ cudzołożył / nie będzieƺ zábijáł / nie będzieƺ krádł / nie będzieƺ mówił świádectwá fáłƺywego / nie będzieƺ oƺukiwáł nikogo / cżćij ojcá twego y mátkę twoję.
A on odpowiádájąc / rzekł mu : Náucżyćielu! tegom wƺyſtkiego przeſtrzegáł od młodośći mojej.
A Jezuſ ſpojrzáwƺy náń / rozmiłowáł śię go / y rzekł mu : Jednegoć nie doſtáje ; idź / ſprzedáj co máƺ / á rozdáj ubogim / á będzieƺ miáł ſkárb w niebie / á przyjdź / náśláduj mię / wziąwƺy krzyż.
A on záfráſowáwƺy śię dlá tego ſłowá / odƺedł ſmutny ; ábowiem miáł wiele májętnośći.
A ſpojrzáwƺy Jezuſ w około / rzekł do ucżniów ſwoich : Jákoż trudno ći / którzy máją bogáctwá / wnijdą do króleſtwá Bożego!
Tedy ucżniowie zdumieli śię nád temi ſłowámi jego. Lecż Jezuſ záś odpowiádájąc / rzekł im : Dziátki! jákoż jeſt trudno tym / co ufáją w bogáctwách / wnijść do króleſtwá Bożego.
Snádniej jeſt wielbłądowi przejść przez ucho igielne / niż bogácżowi wnijść do króleſtwá Bożego.
A oni śię tem więcej zdumiewáli / mówiąc między ſobą : Y któż może być zbáwiony?
A Jezuſ ſpojrzáwƺy ná nie / rzekł : U ludzić to niemożno / ále nie u Bogá ; ábowiem u Bogá wƺyſtko jeſt możno.
Y pocżął Piotr mówić do niego : Otośmy my opuśćili wƺyſtko / á poƺliśmy zá tobą.
A Jezuſ odpowiádájąc / rzekł : Zápráwdę powiádám wám : Nikt nie jeſt / ktoby opuśćił dom / álbo bráći / álbo śioſtry / álbo ojcá / álbo mátkę / álbo żonę / álbo dzieći / álbo role dlá mnie y dlá Ewángielii /
Żeby nie miáł wziąć ſtokrotnie teráz w tym cżáśie domów / y bráći / y śióſtr / y mátek / y dzieći / y ról z prześládowániem / á w przyƺłym wieku żywotá wiecżnego.
Aleć wiele pierwƺych będą oſtátnimi / á oſtátnich pierwƺymi.
Y byli w drodze / wſtępując do Jeruzálemu ; á Jezuſ ƺedł przed nimi / y zdumiewáli śię / á idąc zá nim / báli śię. A on wziąwƺy záśię z ſobą onych dwánáśćie / pocżął im powiádáć / co náń przyjść miáło /
Mówiąc : Oto wſtępujemy do Jeruzálemu / á Syn cżłowiecży będzie wydány przedniejƺym kápłánom y náucżonym w Piśmie / y oſądzą go ná śmierć / y wydádzą go pogánom.
A oni śię z niego náśmiewáć będą / y ubicżują go / y będą náń plwáć / y zábiją go ; ále dniá trzećiego zmártwychwſtánie.
Tedy przyſtąpili do niego Jákób y Ján / ſynowie Zebedeuƺowi / mówiąc : Náucżyćielu! chcemy / ábyś nám ucżynił / o co ćię prośić będziemy.
A on im rzekł : Cóż chcećie / ábym wám ucżynił?
A oni mu rzekli : Dáj nám / ábyśmy jeden ná práwicy twojej á drugi ná lewicy twojej śiedzieli w chwále twojej.
Lecż im Jezuſ rzekł : Nie wiećie / o co prośićie. Możećież pić kielich / który já piję / y chrztem / którym śię já chrzcżę / być ochrzcżeni?
A oni mu rzekli : Możemy. A Jezuſ im rzekł : Kielichći / który já piję / pić będziećie y chrztem / którym śię já chrzcżę / ochrzcżeni będziećie.
Ale śiedzieć po práwicy mojej álbo po lewicy mojej / nie mojá rzecż jeſt dáć ; ále będzie dáno tym / którym zgotowáno.
A uſłyƺáwƺy to oni dzieśięćiu / pocżęli śię gniewáć ná Jákóbá y ná Jáná.
Ale Jezuſ zwołáwƺy ich / rzekł im : Wiećie / iż ći / którym śię zdá / że włádzę máją nád národy / pánują nád nimi / á którzy z nich wielcy ſą / moc przewodzą nád nimi.
Lecż nie ták będzie między wámi ; ále ktobykolwiek chćiáł być wielkim między wámi / będzie ſługą wáƺym ;
A ktobykolwiek z wáſ chćiáł być pierwƺym / będzie ſługą wƺyſtkich.
Bo y Syn cżłowiecży nie przyƺedł / áby mu ſłużono / ále áby ſłużył / y áby dáł duƺę ſwą ná okup zá wielu.
Tedy przyƺli do Jerychá ; á gdy on wychodził z Jerychá / y ucżniowie jego y lud wielki / ſyn Tymeuƺá / Bártymeuƺ ślepy / śiedziáł podle drogi żebrząc.
A uſłyƺáwƺy / iż jeſt Jezuſ on Názáreńſki / pocżął wołáć / mówiąc : Jezuśie / Synu Dawidá! zmiłuj śię nádemną.
Y gromiło go wiele ich / áby milcżáł ; ále on tem więcej wołáł : Synu Dawidá! zmiłuj śię nádemną.
Tedy záſtánowiwƺy śię Jezuſ / kázáł go záwołáć. Y záwołáno ślepego / mówiąc mu : Ufáj / wſtáń / wołá ćię.
A on porzućiwƺy płáƺcż ſwój / wſtáł / y przyƺedł do Jezuſá.
Y odpowiádájąc Jezuſ / rzekł mu : Cóż chceƺ / ábym ći ucżynił? A ślepy mu rzekł : Miſtrzu! ábym przejrzáł.
A Jezuſ mu rzekł : Idź / wiárá twojá ćiebie uzdrowiłá. A zárázem przejrzáł / y ƺedł drogą zá Jezuſem.
