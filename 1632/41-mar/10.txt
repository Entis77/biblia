A wſtáwƺy ſtámtąd / przyƺedł do gránic Judzkich przez kráinę zá Jordánem leżącą ; y zƺedł śię záś do niego lud / y ucżył je záś jáko miał zwycżáj.
Tedy przyſtąpiwƺy Fáryzeuƺowie / pytáli go : Godźili śię mężowi żonę opuśćić? á to cżynili / kuƺąc go.
Ale on odpowiádájąc / rzekł im : Cóż wám przykázáł Mojżeƺ?
A oni rzekli : Mojżeƺ pozwolił / nápiſáć liſt rozwodny / y opuśćić ją.
A odpowiádájąc JEzuſ / rzekł im : Dla zátwárdzeniá ſercá wáƺego nápiſáł wám to przykázánie.
Aleć od pocżątku ſtworzeniá mężcżyznę y niewiáſtę ucżynił je Bóg.
Dlátego opuśći cżłowiek ojcá ſwego y mátkę / á przyłącży śię do żony ſwojey /
Y będą dwoje jednem ćiáłem ; á ták już nie ſą dwoje / ále jedno ćiáło.
Co tedy Bóg złącżył / cżłowiek niecháj nie rozłącżá.
A w domu záś ucżniowie jego o toż go pytáli.
Y rzekł im : Ktobykolwiek opuśćił żonę ſwą / á pojąłby inną / cudzołoży przećiwko niej ;
A jeſliby niewiáſtá opuśćiłá mężá ſwego / á ƺłáby zá drugiego / cudzołoży.
Tedy przynoƺono do niego dźiátki / áby śię ich dotykáł ; ále ucżniowie gromili tych / którzy je przynośili.
Co ujrzáwƺy JEzuſ / rozgniewáł śię y rzekł im : Dopuśććie dźiátkom przychodźić do mnie / á nie zábrániájćie im ; ábowiem tákich jeſt króleſtwo Boże.
Zápráwdę powiádám wám : Ktobykolwiek nie przyjął króleſtwá Bożego jáko dźiećiątko / nie wnijdźie do niego.
Y biorąc je ná ręce ſwoje / y kłádąc ná nie ręce / błogoſłáwił im.
A gdy on wychodźił w drogę / przybieżáł jeden / y upádłƺy przed nim ná koláná / pytáł go : Náucżyćielu dobry! cóż cżynić mam / ábym odźiedźicżył żywot wiecżny?
Ale mu JEzuſ rzekł : Przecż mię zowieƺ dobrym? Nikt nie jeſt dobry / tylko jeden / to jeſt Bóg.
Przykázániá umieƺ : nie będźieƺ cudzołożył / nie będźieƺ zábijáł / nie będźieƺ krádł / nie będźieƺ mówił świádectwá fáłƺywego / nie będźieƺ oƺukiwáł nikogo / cżćij ojcá twego y mátkę twoję.
A on odpowiádájąc / rzekł mu : Náucżyćielu! tegom wƺyſtkiego przeſtrzegáł od młodośći mojey.
A JEzuſ ſpojrzáwƺy náń / rozmiłowáł śię go / y rzekł mu : Jednegoć nie doſtáje ; idź / ſprzedáj co máƺ / á rozdáj ubogim / á będźieƺ miał ſkárb w niebie / á przyjdź / náśláduj mię / wźiąwƺy krzyż.
A on záfráſowáwƺy śię dla tego ſłowá / odƺedł ſmutny ; ábowiem miał wiele májętnośći.
A ſpojrzáwƺy JEzuſ w około / rzekł do ucżniów ſwoich : Jákoż trudno ći / którzy máją bogáctwá / wnijdą do króleſtwá Bożego!
Tedy ucżniowie zdumieli śię nád temi ſłowámi jego. Lecż JEzuſ záś odpowiádájąc / rzekł im : Dźiátki! jákoż jeſt trudno tym / co ufáją w bogáctwách / wnijść do króleſtwá Bożego.
Snádniej jeſt wielbłądowi przejść przez ucho igielne / niż bogácżowi wnijść do króleſtwá Bożego.
A oni śię tem więcey zdumiewáli / mówiąc między ſobą : Y któż może być zbáwiony?
A JEzuſ ſpojrzáwƺy ná nie / rzekł : U ludźić to niemożno / ále nie u Bogá ; ábowiem u Bogá wƺyſtko jeſt możno.
Y pocżął Piotr mówić do niego : Otośmy my opuśćili wƺyſtko / á poƺliśmy zá tobą.
A JEzuſ odpowiádájąc / rzekł : Zápráwdę powiádám wám : Nikt nie jeſt / ktoby opuśćił dom / álbo bráći / álbo śioſtry / álbo ojcá / álbo mátkę / álbo żonę / álbo dźieći / álbo role dla mnie y dla Ewángielii /
Żeby nie miał wźiąć ſtokrotnie teraz w tym cżáśie domów / y bráći / y śióſtr / y mátek / y dźieći / y ról z prześládowániem / á w przyƺłym wieku żywotá wiecżnego.
Aleć wiele pierwƺych będą oſtátnimi / á oſtátnich pierwƺymi.
Y byli w drodze / wſtępując do Jeruzálemu ; á JEzuſ ƺedł przed nimi / y zdumiewáli śię / á idąc zá nim / báli śię. A on wźiąwƺy záśię z ſobą onych dwánáśćie / pocżął im powiádáć / co náń przyjść miáło /
Mówiąc : Oto wſtępujemy do Jeruzálemu / á Syn cżłowiecży będźie wydány przedniejƺym kápłánom y náucżonym w Piśmie / y oſądzą go ná śmierć / y wydádzą go pogánom.
A oni śię z niego náśmiewáć będą / y ubicżują go / y będą náń plwáć / y zábiją go ; ále dniá trzećiego zmartwychwſtánie.
Tedy przyſtąpili do niego Jákób y Ján / ſynowie Zebedeuƺowi / mówiąc : Náucżyćielu! chcemy / ábyś nám ucżynił / o co ćię prośić będźiemy.
A on im rzekł : Cóż chcećie / ábym wám ucżynił?
A oni mu rzekli : Dáj nám / ábyśmy jeden ná práwicy twojey á drugi ná lewicy twojey śiedźieli w chwále twojey.
Lecż im JEzuſ rzekł : Nie wiećie / o co prośićie. Możećież pić kielich / który ja piję / y chrztem / którym śię ja chrzcżę / być ochrzcżeni?
A oni mu rzekli : Możemy. A JEzuſ im rzekł : Kielichći / który ja piję / pić będźiećie y chrztem / którym śię ja chrzcżę / ochrzcżeni będźiećie.
Ale śiedźieć po práwicy mojey álbo po lewicy mojey / nie mojá rzecż jeſt dáć ; ále będźie dáno tym / którym zgotowáno.
A uſłyƺáwƺy to oni dźieśięćiu / pocżęli śię gniewáć ná Jákóbá y ná Jáná.
Ale JEzuſ zwołáwƺy ich / rzekł im : Wiećie / iż ći / którym śię zdá / że włádzę máją nád národy / pánują nád nimi / á którzy z nich wielcy ſą / moc przewodzą nád nimi.
Lecż nie ták będźie między wámi ; ále ktobykolwiek chćiáł być wielkim między wámi / będźie ſługą wáƺym ;
A ktobykolwiek z was chćiáł być pierwƺym / będźie ſługą wƺyſtkich.
Bo y Syn cżłowiecży nie przyƺedł / áby mu ſłużono / ále áby ſłużył / y áby dáł duƺę ſwą ná okup zá wielu.
Tedy przyƺli do Jerychá ; á gdy on wychodźił z Jerychá / y ucżniowie jego y lud wielki / ſyn Tymeuƺá / Bártymeuƺ ślepy / śiedźiáł podle drogi żebrząc.
A uſłyƺáwƺy / iż jeſt JEzuſ on Názáreńſki / pocżął wołáć / mówiąc : JEzuśie / Synu Dawidá! zmiłuj śię nádemną.
Y gromiło go wiele ich / áby milcżáł ; ále on tem więcey wołáł : Synu Dawidá! zmiłuj śię nádemną.
Tedy záſtánowiwƺy śię JEzuſ / kázáł go záwołáć. Y záwołáno ślepego / mówiąc mu : Ufáj / wſtáń / wołá ćię.
A on porzućiwƺy płáƺcż ſwój / wſtáł / y przyƺedł do JEzuſá.
Y odpowiádájąc JEzuſ / rzekł mu : Cóż chceƺ / ábym ći ucżynił? A ślepy mu rzekł : Miſtrzu! ábym przejrzáł.
A JEzuſ mu rzekł : Idź / wiárá twojá ćiebie uzdrowiłá. A zárázem przejrzáł / y ƺedł drogą zá JEzuſem.
