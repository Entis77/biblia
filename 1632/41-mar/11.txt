A gdy śię przybliżyli do Jeruzálemu y do Betfágie y do Betánii ku górze oliwnej / poſłáł dwóch z ucżniów ſwoich /
Y rzekł im : Idźćie do miáſtecżká / które jeſt przećiwko wám / á wƺedłƺy do niego / záráz znájdźiećie oślę uwiązáne / ná którem nikt z ludźi nie śiedźiáł ; odwiążćież je / á przywiedźćie.
A jeźliby wám kto rzekł : Cóż to cżynićie? Powiedzćie / iż go Pán potrzebuje ; á wnet je tu pośle.
Szli tedy y ználeźli oślę uwiązáne u drzwi ná dworze ná rozſtániu dróg / y odwiązáli je.
Tedy niektórzy z onych / co tám ſtáli / mówili : Cóż cżynićie / że odwiązujećie oślę?
A oni im rzekli / jáko im był rozkázáł JEzuſ. Y puśćili je.
Przywiedli tedy oślę do JEzuſá / y włożyli ná nie ƺáty ſwoje ; y wśiádł ná nie.
A wiele ich ſłáli ƺáty ſwoje ná drodze ; drudzy záśię obćináli gáłązki z drzew / y ſłáli ná drodze.
A którzy wprzód ƺli / y którzy pozád ƺli / wołáli / mówiąc : Hoſánná / błogoſłáwiony / który idźie w imieniu Páńſkim!
Błogoſłáwione króleſtwo ojcá náƺego Dawidá / które przyƺło w imieniu Páńſkim! Hoſánná ná wyſokośćiách!
Y wjecháł JEzuſ do Jeruzálemu y przyƺedł do kośćiołá / á obejrzáwƺy wƺyſtko / gdy już byłá wiecżorná godźiná / wyƺedł do Betánii z dwunáſtomá.
A drugiego dniá / gdy wychodźili z Betánii / łáknął.
Y ujrzáwƺy z dáleká figowe drzewo / májące liśćie / przyƺedł / jeźliby ſnáć co ná nim ználázł ; á gdy do niego przyƺedł / nic nie ználázł tylko liśćie ; bo nie był cżáſ figom.
A odpowiádájąc JEzuſ / rzekł mu : Niechájże więcej ná wieki nikt z ćiebie owocu nie je. A ſłyƺeli to ucżniowie jego.
Y przyƺli do Jeruzálemu ; á wƺedłƺy JEzuſ do kośćiołá / pocżął wygániáć ſprzedáwájące y kupujące w kośćiele / y poprzewrácáł ſtoły tych / co pieniędzmi hándlowáli / y ſtołki tych / co ſprzedáwáli gołębie ;
A nie dopuśćił / żeby kto miáł nieść nácżynie przez kośćiół.
Y náucżáł / mówiąc im : Azáż nie nápiſáno : Że dom mój / dom modlitwy będźie názwány od wƺyſtkich národów? á wyśćie go ucżynili jáſkinią zbójców.
A ſłyƺeli to náucżeni w Piśmie y przedniejśi kápłáni y ƺukáli / jákoby go ſtráćili ; ábowiem śię go báli / przeto / iż wƺyſtek lud zdumiewáł śię nád náuką jego.
A gdy przyƺedł wiecżór / wyƺedł z miáſtá.
A ráno idąc mimo figowe drzewo / ujrzeli / iż z korzeniá uſchło.
Tedy wſpomniáwƺy Piotr / rzekł mu : Miſtrzu! oto figowe drzewo / któreś przeklął / uſchło.
A JEzuſ odpowiádájąc / rzekł im : Miejćie wiárę w Bogá.
Bo zápráwdę powiádám wám / iż ktobykolwiek rzekł tej górze : Podnieś śię / á rzuć śię w morze / á nie wątpiłby w ſercu ſwojim / lecżby wierzył / że śię ſtánie / co mówi / ſtánie śię mu / cokolwiek rzecże.
Przetoż powiádám wám : O cokolwiekbyśćie / modląc śię / prośili / wierzćie / że weźmiećie / á ſtánie śię wám.
A gdy ſtoićie modląc śię / odpuśććież / jeźli co przećiw komu máćie / áby y Ojćiec wáƺ / który jeſt w niebieśiech / odpuśćił wám upádki wáƺe.
Bo jeźli wy nie odpuśćićie / y Ojćiec wáƺ / który jeſt w niebieśiech / nie odpuśći wám upádków wáƺych.
Y przyƺli znowu do Jeruzálemu. A gdy śię on przechodźił po kośćiele / przyſtąpili do niego przedniejśi kápłáni y náucżeni w Piśmie y ſtárśi ;
Y mówili do niego : Którąż to mocą cżyniƺ? á ktoć dáł tę moc / ábyś to cżynił?
Tedy JEzuſ odpowiádájąc / rzekł im : Spytám wáſ y já o jednę rzecż ; odpowiedzćież mi / á powiem / którą mocą to cżynię.
Chrzeſt Jáná z niebáż był / cżyli z ludźi? Odpowiedzćie mi.
Y rozbieráli to ſámi między ſobą / mówiąc : Jeźli powiemy / z niebá / rzecże : Przecżżeśćie mu tedy nie wierzyli?
A jeźli powiemy / z ludźi / bojemy śię ludu ; ábowiem wƺyſcy Jáná mieli zá práwdźiwego proroká.
Tedy odpowiádájąc rzekli JEzuſowi : Nie wiemy. JEzuſ też odpowiádájąc rzekł im : Y já wám nie powiem / którą mocą to cżynię.
