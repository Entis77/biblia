A gdy on wychodźił z kośćiołá / rzekł mu jeden z ucżniów jego : Náucżyćielu! pátrz / jákie to kámienie / y jákie budowániá?
A JEzuſ odpowiádájąc rzekł mu : Widźiƺ te wielkie budowániá? Nie będźie zoſtáwiony kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedźiáł ná górze oliwney przećiwko kośćiołowi / pytáli go oſobno Piotr / y Jákób / y Ján / y Andrzej :
Powiedz nám / kiedy śię to ſtánie / y co zá znák / kiedy śię to wƺyſtko pełnić będźie?
A JEzuſ odpowiádájąc im / pocżął mówić : Pátrzćie / áby was kto nie zwiódł.
Abowiem wiele ich przyjdźie pod imieniem mojem / mówiąc : Jám jeſt CHryſtuſ / á wiele ich zwiodą.
Gdy tedy uſłyƺyćie wojny / y wieśći o wojnách / nie trwożćież ſobą ; boć to muśi być ; ále jeƺcże nie tu koniec.
Abowiem powſtánie náród przećiwko národowi / y króleſtwo przećiwko króleſtwu / y będą miejſcámi trzęśieniá źiemi / y będą głody y zámięƺániá.
A toć pocżątki boleśći. Lecż wy ſtrzeżćie ſámych śiebie ; boć was podáwáć będą przed rády y do zgromádzeniá / będą was bić / á przed ſtároſtámi y królmi dla mnie ſtáwáć będźiećie ná świádectwo przećiwko nim.
Ale u wƺyſtkich národów muśi być przed tem kázáná Ewángielijá.
A gdy was powiodą wydáwájąc / nie troƺcżćież śię przed cżáſem / cobyśćie mówić mieli / áni o tem myślćie / ále co wám będźie dáno onejże godźiny / to mówćie ; ábowiem nie wy jeſteśćie / którzy mówićie / ále Duch Święty.
Y wydá brát brátá ná śmierć / á ojćiec ſyná ; y powſtáną dźieći przećiwko rodźicom / y będą je zábijáć.
A będźiećie w nienáwiśći u wƺyſtkich dla imieniá mego ; ále kto wytrwá áż do końcá / ten będźie zbáwion.
Gdy tedy ujrzyćie onę obrzydliwość ſpuſtoƺeniá / opowiedźiáną od Dánijelá proroká / ſtojącą / gdźie ſtáć nie miáłá ; ( kto cżytá / niecháj uwáżá / ) tedy ći / którzy będą w Judzkiej źiemi / niech ućiekáją ná góry ;
A kto będźie ná dáchu / niech nie zſtępuje do domu / áni wchodźi / áby co wźiął z domu ſwego ;
A kto będźie ná roli / niech śię nie wrácá názád / áby wźiął ƺátę ſwoję.
Lecż biádá brzemiennym y pierśiámi kármiącym w one dni!
Przetoż módlćie śię / áby ućiekánie wáƺe nie było w źimie.
Abowiem będą te dni tákiem ućiśnieniem / jákiego nie było od pocżątku ſtworzeniá / które ſtworzył Bóg / áż dotąd / áni będźie.
A jeſliby Pán nie ſkróćił dni onych / nie byłoby żádne ćiáło zbáwione ; lecż dla wybránych / które wybráł / ſkróćił dni onych.
A tedy jeſliby wám kto rzekł : Oto tu jeſt CHryſtuſ / álbo oto tám / nie wierzćie.
Boć powſtáną fáłƺywi CHryſtuſowie / y fáłƺywi prorocy / y będą cżynić známioná y cudá ku zwiedzeniu / by możná / y wybránych.
Wy tedy ſtrzeżćie śię ; otom wám wƺyſtko przepowiedźiáł.
Ale w one dni po ućiśnieniu onem / záćmi śię ſłońce / y kśiężyc nie wydá świátłośći ſwojey ;
Y gwiázdy niebieſkie będą pádáły / á mocy / które ſą ná niebie / poruƺą śię.
A tedy ujrzą Syná cżłowiecżego / przychodzącego w obłokách z mocą y z chwáłą wielką.
A tedy pośle Anioły ſwoje y zgromádźi wƺyſtkie wybráne ſwoje od cżterech wiátrów / od końcżyn źiemi áż do końcżyn niebá.
A od figowego drzewá náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmłádzá y puƺcżá liśćie / poznájećie / iż bliſko jeſt láto.
Tákże y wy / gdy ujrzyćie / iż śię to dźiáć będźie / poznáwájćie / że bliſko jeſt y we drzwiách.
Zápráwdę powiádám wám / żeć nie przeminie ten rodzáj / áżby śię to wƺyſtko ſtáło.
Niebo y źiemiá przeminą ; ále ſłowá moje nie przeminą.
Lecż o onym dniu y godźinie nikt nie wie / áni Aniołowie / którzy ſą w niebie / áni Syn / tylko Ojćiec.
Pátrzćież / cżujćie / á módlćie śię ; bo nie wiećie / kiedy ten cżáſ będźie.
Jáko cżłowiek / który precż odjeżdżájąc / zoſtáwił dom ſwój / y rozdáł urzędy ſługom ſwoim / y káżdemu robotę jego / y wrotnemu przykázáł / áby cżuł.
Cżujćież tedy ; ( bo nie wiećie / kiedy Pán domu onego przyjdźie / z wiecżoráli / cżyli o północy / cżyli gdy kury pieją / cżyli ráno. )
By ſnáć nieſpodźiánie przyƺedłƺy / nie ználázł was śpiącymi.
A co wám mówię / wƺyſtkimći mówię : Cżujćie.
