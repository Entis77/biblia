A gdy on wychodźił z Kośćiołá / rzekł mu jeden z ucżniów jego : Náucżyćielu / Pátrz jákie to kámienie / y jákie budowánia?
A JEzus odpowiádájąc rzekł mu : Widźiƺ te wielkie budowánia? Nie będźie zoſtáwiony kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedźiał ná górze oliwney przećiwko Kośćiołowi / pytáli go oſobno Piotr y Jákub y Jan / y Andrzyj ;
Powiedz nam kiedy śię to ſtánie / y co zá znák kiedy śię to wƺyſtko pełnić będźie?
A JEzus odpowiádájąc im / pocżął mówić : Pátrżćie áby was kto nie zwiódł.
Abowiem wiele ich przydźie pod Imieniem mojim / mówiąc : Jam jeſt <i>CHryſtus,</i> y wiele ich zwiodą.
Gdy tedy uſłyƺyćie wojny / y wieśći o wojnách / nie trwożćież ſobą ; boć to muśi być : áleć jeƺcże nie <i>tu</i> koniec.
Abowiem powſtánie naród przećiwko narodowi / y króleſtwo przećiwko króleſtwu : y będą miejſcámi trzęśienia źiemie / y będą głody y zámięƺánia.
<i>A</i> toć pocżątki boleśći. Lecż wy ſtrzeżćie ſámych śiebie ; boć was podawáć będą przed Rády / y do zgromádzenia : będą was bić / á przed ſtároſtámi y Królmi dla mnie ſtawáć będźiećie / ná świádectwo przećiwko nim.
Ale u wƺyſtkich narodów muśi bydź przed tym kazána Ewángelia.
A gdy was powiodą wydawájąc / nie troƺcżćież śię przed cżáſem / cobyśćie mówić mieli / áni o tym myślćie : Ale co wam będźie dano onejże godźiny / to mówćie : Abowiem nie wy jeſteśćie którzy mówićie / ále Duch Święty.
Y wyda brát brátá ná śmierć ; á Oćiec ſyná ; y powſtáną dźieći przećiwko rodźicom / y będą je zábijáć.
A będźiećie w nienawiśći u wƺyſtkich dla Imienia mego : Ale kto wytrwa áż do końcá / ten będźie zbáwion.
Gdy tedy ujrzyćie onę obrzydliwość ſpuſtoƺenia / opowiedźiáną od Dánielá Proroká / ſtojącą gdźie ſtać nie miáłá ; ( kto cżyta niechaj uważa ) tedy ći którzy będą w Judſkiey źiemi / niech ućiekáją ná góry.
A kto będźie ná dáchu / niech nie zſtępuje do domu / áni wchodźi áby co wźiął z domu ſwego.
A kto będźie ná roli / niech śię nie wraca ná zad / áby wźiął ƺátę ſwoję.
Lecż biádá brzemiennym y pierśiámi karmiącym w one dni.
Przetoż módlćie śię / áby ućiekánie wáƺe nie było źimie.
Abowiem będą te dni tákim ućiśnieniem / jákiego nie było od pocżątku ſtworzenia które ſtworzył Bóg áż do tąd / áni będźie.
A jeſliby PAn nie ſkróćił dni onych / nie byłoby żadne ćiáło zbáwione : Lecż dla wybránych / które wybrał / ſkróćił dni onych.
A tedy jeſliby wam kto rzekł : Oto tu <i>jeſt</i> CHryſtus / álbo oto tám ; nie wierzćie.
Boć powſtáną fałƺywi CHryſtuſowie / y fałƺywi Prorocy / y będą cżynić známioná y cudá ku zwiedzeniu / by można / y wybránych.
Wy tedy ſtrzeżćie śię : otom wā wƺyſtko przedpowiedźiał.
Ale w one dni po ućiśnieniu onym / záćmi śię ſłońce / y kśiężyc nie wyda świátłośći ſwojey :
Y gwiazdy niebieſkie będą pádáły / á mocy które ſą ná niebie poruƺą śię.
A tedy ujrzą Syná cżłowiecżego przychodzącego w obłokách / z mocą y z chwałą wielką.
A tedy pośle Anjoły ſwoje / y zgromádźi wƺyſtkie wybráne ſwoje od cżterech wiátrów / od końcżyn źiemie áż do końcżyn niebá.
A od figowego drzewá náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmładza y puƺcża liśćie / poznawaćie iż bliſko jeſt láto.
Tákże y wy gdy ujrzyćie iż śię to dźiać będźie / poznawajćie że bliſko jeſt / <i>y</i> we drżwiách.
Záprawdę powiádam wam / żeć nie przeminie ten rodzaj / áżby śię to wƺyſtko ſtáło.
Niebo y źiemiá przeminą / ále ſłowá moje nie przeminą.
Lecż o onym dniu y godźinie nikt nie wie / áni Anjołowie którzy ſą w niebie / áni Syn ; tylko Oćiec.
Pátrzćież / cżujćie á módlćie śię ; bo nie wiećie kiedy ten cżás będźie.
<i>Abowiem Syn cżłowiecży jest</i> jáko cżłowiek / który precż odjeżdżájąc / zoſtáwił dom ſwój / y rozdał urzędy ſługom ſwojim / y káżdemu robotę jego / y wrotnemu przykazał áby cżuł.
Cżujćież tedy : ( Bo niewiećie kiedy PAN domu onego przydźie / z wiecżoráli / cżyli o północy / cżyli gdy kurzy pieją / cżyli ráno. )
By ſnadź nieſpodźiánie przyƺedƺy / nie ználazł was ſpiącymi.
A co wam mówię / wƺyſtkimći mówię : Cżujćie.
