A gdy on wychodził z kośćiołá / rzekł mu jeden z ucżniów jego : Náucżyćielu! pátrz / jákie to kámienie / y jákie budowániá?
A Jezuſ odpowiádájąc rzekł mu : Widziƺ te wielkie budowániá? Nie będzie zoſtáwiony kámień ná kámieniu / któryby nie był rozwálony.
A gdy śiedziáł ná górze oliwnej przećiwko kośćiołowi / pytáli go oſobno Piotr / y Jákób / y Ján / y Andrzej :
Powiedz nám / kiedy śię to ſtánie / y co zá znák / kiedy śię to wƺyſtko pełnić będzie?
A Jezuſ odpowiádájąc im / pocżął mówić : Pátrzćie / áby wáſ kto nie zwiódł.
Abowiem wiele ich przyjdzie pod imieniem mojem / mówiąc : Jám jeſt Chryſtuſ / á wiele ich zwiodą.
Gdy tedy uſłyƺyćie wojny / y wieśći o wojnách / nie trwożćież ſobą ; boć to muśi być ; ále jeƺcże nie tu koniec.
Abowiem powſtánie náród przećiwko národowi / y króleſtwo przećiwko króleſtwu / y będą miejſcámi trzęśieniá ziemi / y będą głody y zámięƺániá.
A toć pocżątki boleśći. Lecż wy ſtrzeżćie ſámych śiebie ; boć wáſ podáwáć będą przed rády y do zgromádzeniá / będą wáſ bić / á przed ſtároſtámi y królmi dlá mnie ſtáwáć będziećie ná świádectwo przećiwko nim.
Ale u wƺyſtkich národów muśi być przed tem kázáná Ewángielijá.
A gdy wáſ powiodą wydáwájąc / nie troƺcżćież śię przed cżáſem / cobyśćie mówić mieli / áni o tem myślćie / ále co wám będzie dáno onejże godziny / to mówćie ; ábowiem nie wy jeſteśćie / którzy mówićie / ále Duch Święty.
Y wydá brát brátá ná śmierć / á ojćiec ſyná ; y powſtáną dzieći przećiwko rodzicom / y będą je zábijáć.
A będziećie w nienáwiśći u wƺyſtkich dlá imieniá mego ; ále kto wytrwá áż do końcá / ten będzie zbáwion.
Gdy tedy ujrzyćie onę obrzydliwość ſpuſtoƺeniá / opowiedziáną od Dánijelá proroká / ſtojącą / gdzie ſtáć nie miáłá ; ( kto cżytá / niecháj uwáżá / ) tedy ći / którzy będą w Judzkiej ziemi / niech ućiekáją ná góry ;
A kto będzie ná dáchu / niech nie zſtępuje do domu / áni wchodzi / áby co wziął z domu ſwego ;
A kto będzie ná roli / niech śię nie wrácá názád / áby wziął ƺátę ſwoję.
Lecż biádá brzemiennym y pierśiámi kármiącym w one dni!
Przetoż módlćie śię / áby ućiekánie wáƺe nie było w zimie.
Abowiem będą te dni tákiem ućiśnieniem / jákiego nie było od pocżątku ſtworzeniá / które ſtworzył Bóg / áż dotąd / áni będzie.
A jeźliby Pán nie ſkróćił dni onych / nie byłoby żádne ćiáło zbáwione ; lecż dlá wybránych / które wybráł / ſkróćił dni onych.
A tedy jeźliby wám kto rzekł : Oto tu jeſt Chryſtuſ / álbo oto tám / nie wierzćie.
Boć powſtáną fáłƺywi Chryſtuſowie / y fáłƺywi prorocy / y będą cżynić známioná y cudá ku zwiedzeniu / by możná / y wybránych.
Wy tedy ſtrzeżćie śię ; otom wám wƺyſtko przepowiedziáł.
Ale w one dni po ućiśnieniu onem / záćmi śię ſłońce / y kśiężyc nie wydá świátłośći ſwojej ;
Y gwiázdy niebieſkie będą pádáły / á mocy / które ſą ná niebie / poruƺą śię.
A tedy ujrzą Syná cżłowiecżego / przychodzącego w obłokách z mocą y z chwáłą wielką.
A tedy pośle Anioły ſwoje y zgromádzi wƺyſtkie wybráne ſwoje od cżterech wiátrów / od końcżyn ziemi áż do końcżyn niebá.
A od figowego drzewá náucżćie śię tego podobieńſtwá : Gdy śię już gáłąź jego odmłádzá y puƺcżá liśćie / poznájećie / iż bliſko jeſt láto.
Tákże y wy / gdy ujrzyćie / iż śię to dziáć będzie / poznáwájćie / że bliſko jeſt y we drzwiách.
Zápráwdę powiádám wám / żeć nie przeminie ten rodzáj / áżby śię to wƺyſtko ſtáło.
Niebo y ziemiá przeminą ; ále ſłowá moje nie przeminą.
Lecż o onym dniu y godzinie nikt nie wie / áni Aniołowie / którzy ſą w niebie / áni Syn / tylko Ojćiec.
Pátrzćież / cżujćie / á módlćie śię ; bo nie wiećie / kiedy ten cżáſ będzie.
Jáko cżłowiek / który precż odjeżdżájąc / zoſtáwił dom ſwój / y rozdáł urzędy ſługom ſwoim / y káżdemu robotę jego / y wrotnemu przykázáł / áby cżuł.
Cżujćież tedy ; ( bo nie wiećie / kiedy Pán domu onego przyjdzie / z wiecżoráli / cżyli o północy / cżyli gdy kury pieją / cżyli ráno. )
By ſnáć nieſpodziánie przyƺedłƺy / nie ználázł wáſ śpiącymi.
A co wám mówię / wƺyſtkimći mówię : Cżujćie.
