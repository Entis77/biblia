Y pocżął záśię ucżyć przy morzu ; y zgromádźił śię do niego lud wielki / ták iż wſtąpiwƺy w łódź / śiedźiáł ná morzu / á wƺyſtek lud był przy morzu ná źiemi.
Y náucżáł ich wiele rzecży w podobieńſtwách / á mówił do nich w náuce ſwojej : Słuchájćie!
Oto wyƺedł rozśiewcá / áby rozśiewáł.
Y ſtáło śię / gdy rozśiewáł / że jedno pádło podle drogi / á ptáki niebieſkie przylećiáły y pozobáły je.
Drugie záśię pádło ná miejſcá opocżyſte / gdźie nie miáło wiele źiemi ; y prędko weƺło / przeto iż nie miáło głębokośći źiemi ;
A gdy ſłońce weƺło / wygorzáło / á iż korzeniá nie miáło / uſchło.
A drugie pádło między ćiernie : y wzroſły ćiernie y záduśiły je / y nie wydáło pożytku.
Drugie záśię pádło ná źiemię dobrą / y wydáło pożytek bujno wſchodzący y roſnący : y przynioſło jedno trzydźieśiątny / á drugie ƺeśćdźieśiątny / á drugie ſetny.
Y mówił im : Kto má uƺy ku ſłuchániu / niecháj ſłuchá.
A gdy ſám tylko był / pytáli go ći / co przy nim byli ze dwunáſtomá / o to podobieńſtwo.
A on im odpowiedźiáł : Wám dáno wiedźieć tájemnicę króleſtwá Bożego ; ále tym / którzy ſą obcymi / wƺyſtko śię podáwá w podobieńſtwách ;
Aby pátrząc pátrzeli / ále nie widźieli / y ſłyƺąc ſłyƺeli / ále nie zrozumieli / by śię ſnáć nie náwróćili / á byłyby im grzechy odpuƺcżone.
Zátem rzekł do nich : Nierozumiećie tego podobieńſtwá? A jákoż zrozumiećie wƺyſtkie inne podobieńſtwá?
Rozśiewcá on rozśiewá ſłowo.
A którzy podle drogi / ći ſą / którym śię rozśiewá ſłowo ; ále gdy uſłyƺeli / záráz przychodźi ƺátán / á wybierá ſłowo wśiáne w ſercá ich.
Tákże y ći / którzy ná opocżyſtych miejſcách pośiáni ſą / ći ſą / którzy / gdy uſłyƺeli ſłowo / záráz je z rádośćią przyjmują ;
Wƺákże nie máją korzeniá w ſobie / ále ſą docżeſnymi ; potem / gdy przychodźi ućiſk álbo prześládowánie dlá ſłowá / wnet śię gorƺą ;
A którzy między ćierniem ſą pośiáni / ći ſą / którzy ſłucháją ſłowá ;
Ale piecżołowánie świátá tego y omámienie bugáctw / y pożądliwośći innych rzecży / wƺedłƺy záduƺáją ſłowo / y ſtáje śię bez pożytku.
A którzy ná dobrą źiemię przyjęli náśienie / ći ſą / co ſłucháją ſłowá / y przyjmują je / przynoƺą pożytek / jedno trzydźieśiątny / á drugie ƺeśćdźieśiątny / á drugie ſetny.
Nádto mówił im : Izáli przynoƺą świecę / áby wſtáwioná byłá pod korzec álbo pod łoże? izáli nie dlátego / áby ją ná świecżnik wſtáwiono?
Bo nic niemáƺ tájemnego / coby nie miáło być objáwiono ; áni śię ſtáło co ſkrytego / áby ná jáw nie wyƺło.
Jeźli kto má uƺy ku ſłuchániu / niecháj ſłuchá!
Y rzekł do nich : Pátrzćież / cżego ſłucháćie ; jáką miárą mierzyćie / táką wám będźie odmierzono / á będźie wám przydáno / którzy ſłucháćie.
Abowiem kto má / będźie mu dáno ; á kto niemá / y to / co má / będźie od niego odjęto.
Y mówił : Tákie jeſt króleſtwo Boże / jáko gdyby cżłowiek wrzućił náśienie w źiemię ;
A ſpáłby y wſtáwáłby we dnie y w nocy / á náśienieby weƺło y uroſło / gdy on nie wie.
Boć źiemiá ſámá z śiebie pożytek wydáwá / náprzód tráwę / potem kłoſ / á potem zupełne zboże w kłośie.
A ſkoro śię okáże urodzáj / wnet goſpodárz zápuƺcżá śierp ; bo żniwo przyƺło.
Nád to rzekł : Do cżego przypodobámy króleſtwo Boże / álbo którem je podobieńſtwem wyráźimy?
Jeſt jáko źiárno gorcżycżne ; które / gdy wśiáne bywá w źiemię / nájmniejƺe jeſt ze wƺyſtkich náśion / które ſą ná źiemi.
Ale gdy bywá wśiáne / wzráſtá / y bywá nájwiękƺe nád wƺyſtkie járzyny / y rozpuƺcżá gáłęźie wielkie / ták / iż pod ćieniem jego mogą ſobie cżynić gniázdá ptáki niebieſkie.
Y przez wiele tákich podobieńſtw mówił do nich ſłowo / ták jáko ſłucháć mogli.
A bez podobieńſtwá nie mówił do nich ; wƺákże ucżniom ſwym wƺyſtko z oſobná wykłádáł.
Y rzekł do nich w tenże dźień / gdy już był wiecżór : Przepráwmy śię ná drugą ſtronę.
A rozpuśćiwƺy lud / wźięli go z ſobą / ták jáko był w łodźi ; ále y inne łódki były przy nim.
Tedy powſtáłá wielká náwáłność wiátru / á wáły biły ná łódź / ták / że śię już nápełniáłá.
A on ná zádźie łodźi ſpáł ná wezgłówku ; y obudźili go y mówili mu : Náucżyćielu! nie dbáƺ / że giniemy?
A ták ocknąwƺy śię / zgromił wiátr / y rzekł morzu : Umilknij á uśmierz śię. Y przeſtáł wiátr / á ſtáło śię wielkie ućiƺenie.
Zátem rzekł im : Przecżżeśćie ták bojáźliwi? Jákoż nie máćie wiáry?
Y báli śię bojáźnią wielką / y mówili jedni do drugich : Któż wżdy ten jeſt / że mu y wiátr y morze ſą poſłuƺne?
