Y pocżął záśię ucżyć przy morzu ; y zgromádźił śię do niego lud wielki ; ták / yż wſtąpiwƺy w łódź / śiedźiał ná morzu / á wƺyſtek lud był przy morzu ná źiemi.
Y náucżał ich wiele rzecży w podobieńſtwách / á mówił do nich w náuce ſwojey ; Słuchajćie!
Oto wyƺedł rozśiewcá áby rozśiewał.
Y ſtáło śię gdy rozśiewał / że jedno pádło podle drogi / á ptacy niebieſcy przylećieli / y pozobáli je.
Drugie záśię pádło ná <i>miejſcá</i> opocżyſte / gdźie nie miáło wiele źiemie : y prętko weƺło / przeto iż nie miáło głębokośći źiemie.
A gdy ſłońce weƺło / wygorzáło : á iż korzenia nie miáło / uſchło.
A drugie pádło miedzy ćiernie : y wzroſło ćiernie / y záduśiło je / y nie wydáło pożytku.
Drugie záśię pádło ná źiemię dobrą : y wydáło pożytek bujno wſchodzący y roſtący : y przynioſło jedno trzydźieśiątny / á drugie ƺeśćdźieśiątny / á drugie ſetny.
Y mówił im : Kto ma uƺy ku ſłuchániu / niechaj ſłucha.
A gdy ſam tylko był / pytáli go ći co przy nim byli ze dwiemánaśćie o to podobieńſtwo.
A on im odpowiedźiał : Wam dano wiedźieć tájemnicę króleſtwá Bożego ; ále tym / którzy ſą obcymi / wƺyſtko śię podawa w podobieńſtwách.
Aby pátrząc pátrzáli / ále nie widźieli ; y ſłyƺąc ſłyƺeli / ále nie zrozumieli : by śię ſnadź nie náwróćili / á byłyby im grzechy odpuƺcżone.
Zátym rzekł do nich : Nierozumiećie tego podobieńſtwá? A jákoż zrozumiećie wƺyſtkie <i>inne</i> podobieńſtwá?
Rozśiewcá on rozśiewa ſłowo.
A którzy podle drogi / ći ſą którym śię rozśiewa ſłowo : ále gdy uſłyƺeli / záraz przychodźi ƺátan / á wybiera ſłowo wśiane w ſercá ich.
Tákże y ći / którzy ná opocżyſtych miejſcách pośiani ſą / ći ſą którzy gdy uſłyƺeli ſłowo / záraz je z rádośćią przyjmują.
Wƺákże nie máją korzenia w ſobie / ále ſą docżeſnymi : Potym gdy przychodźi ućiſk álbo prześládowánie dla ſłowá / wnet śię gorƺą.
A którzy miedzy ćierniem ſą pośiani / ći ſą którzy ſłucháją ſłowá.
Ale piecżołowánie świátá tego / y omamienie bogactw / y pożądliwośći innych rzecży / wƺedƺy záduƺáją ſłowo ; y ſtawa śię bez pożytku.
A którzy ná dobrą źiemię przyjęli naśienie / ći ſą co ſłucháją ſłowá / y przyjmują <i>je,</i> y przynoƺą pożytek ; jedno trzydźieśiątny / á drugie ƺeśćdźieśiątny / á drugie ſetny.
Nad to mówił im : Izali przynoƺą świecę / áby wſtáwiona byłá pod korzec álbo pod łoże? Izali <i>nie dla tego</i> áby ją ná świecżnik wſtáwiono?
Bo nic niemáƺ tájemnego coby nie miáło bydź objáwiono / áni śię ſtáło co ſkrytego / áby ná jáwią nie wyƺło.
Jeſli kto ma uƺy ku ſłuchániu / niechaj ſłucha.
Y rzekł do nich : Pátrżćież cżego ſłuchaćie : Jáką miárą mierzyćie / <i>táką</i> wam będźie odmierzono ; á będźie wam przydano / którzy ſłuchaćie.
Abowiem kto ma / będźie mu dano ; á kto nie ma / y to co ma / będźie od niego odjęto.
Y mówił : Tákie jeſt króleſtwo Boże / jáko gdyby cżłowiek wrzućił naśienie w źiemię ;
A ſpałby / y wſtawałby we dnie y w nocy / á naśienieby weƺło y uroſło / gdy on nie wie.
Boć źiemiá ſámá z śiebie pożytek wydawa / naprzód trawę / potym kłos / á potym zupełne zboże w kłośie.
A ſkoro śię okaże urodzaj / wnet <i>goſpodarz</i> zápuƺcża śierp ; bo żniwo przyƺło.
Nad to rzekł : Do cżego przypodobamy króleſtwo Boże / álbo którym je podobieńſtwem wyráźiemy?
Jeſt jáko źiárno gorcżycżne ; które gdy wśiane bywa w źiemię / namniejƺe jeſt ze wƺyſtkich naśion które ſą ná źiemi.
Ale gdy bywa wśiáne / wzraſta / y bywa nawiękƺe nád wƺyſtkie járzyny / y rozpuƺcża gáłęźie wielkie ; ták / iż pod ćieniem jego mogą ſobie cżynić gniazdá ptacy niebieſcy.
Y przez wiele tákich podobieńſtw mówił do nich ſłowo / ták jáko ſłucháć mogli.
A bez podobieńſtwá nie mówił do nich : Wƺákże ucżniom ſwym wƺyſtko z oſobná wykładał.
Y rzekł do nich w tenże dźień gdy już był wiecżór : Przeprawmy śię ná drugą ſtronę.
A rozpuśćiwƺy lud / wźięli go z ſobą / ták jáko był w łodźi ; ále y inne łódki były przy nim.
Tedy powſtáłá wielka náwáłność wiátru / á wáły biły ná łódź / ták że śię już nápełniáłá.
A on ná zádźie łodźi ſpał ná wezgłówku ; y obudźili go / y mówili mu : Náucżyćielu / nie dbaƺ że giniemy!
A ták ocknąwƺy śię zgromił wiátr / y rzekł morzu ; Umilkni / á uśmierz śię. Y przeſtał wiátr / á ſtáło śię wielkie ućiƺenie.
Zátym rzekł im : Przecżżeśćie ták bojáźliwi? Jákoż nie maćie wiáry?
Y bali śię bojáźnią wielką / y mówili jedni do drugich : Któż wżdy ten jeſt? że mu y wiátr y morze ſą poſłuƺne.
