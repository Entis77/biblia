A po dwóch dniách byłá wielkánoc / święto przáśników ; y ƺukáli przedniejśi kápłáni y náucżeni w Piśmie / jákoby go zdrádą pojmáwƺy / zábili.
Lecż mówili : Nie w święto / áby ſnáć nie był rozruch między ludem.
A gdy on był w Betánii / w domu Szymoná trędowátego / gdy śiedźiáł u ſtołu / przyƺłá niewiáſtá / májąc ſłoik álábáſtrowy máśći ƺpikánárdowej płynącey / bárdzo koƺtowney / á ſtłukłƺy ſłoik álábáſtrowy / wyláłá ją ná głowę jego.
Y gniewáli śię niektórzy ſámi w ſobie á mówili : Ná cóż śię ſtáłá utrátá tey máśći?
Abowiem śię to mogło ſprzedáć drożej niż zá trzyſtá groƺy / y rozdáć ubogim ; y ƺemráli przećiwko niej.
Ale JEzuſ rzekł : Zániechájćie jey / przecżże śię jey przykrzyćie? Dobryć ucżynek ucżyniłá przećiwko mnie.
Záwƺe bowiem ubogie máćie z ſobą / y kiedykolwiek chcećie / możećie im dobrze cżynić ; ále mnie nie záwƺe mieć będźiećie.
Oná / co mogłá / to ucżyniłá ; poprzedźiłá / áby ćiáło moje pomázáłá ku pogrzebowi.
Zápráwdę powiádám wám : Gdźiekolwiek kázáná będźie tá Ewángielijá po wƺyſtkim świećie / y to co oná ucżyniłá / powiádáno będźie ná pámiątkę jey.
Tedy Judáƺ Iƺkáryjot / jeden ze dwunáſtu / odƺedł do przedniejƺych kápłánów / áby im go wydáł.
Co oni uſłyƺáwƺy / urádowáli śię / y obiecáli mu dáć pieniądze. Y ƺukáł ſpoſobnego cżáſu / jákoby go wydáł.
Pierwƺego tedy dniá przáśników / gdy báránká wielkánocnego zábijáno / rzekli mu ucżniowie jego : Gdźie chceƺ / ábyśmy ƺedłƺy nágotowáli / żebyś jádł báránká?
Y poſłáł dwóch z ucżniów ſwych / y rzekł im : Idźćie do miáſtá / á ſpotká śię z wámi cżłowiek / dzbán wody nioſący ; idźćież zá nim.
A dokądkolwiek wnijdźie / rzecżćie goſpodárzowi : Náucżyćiel mówi : Gdźież jeſt goſpodá / kędybym jádł báránká z ucżniámi moimi?
A on wám ukáże ſálę wielką uſłáną y gotową / támże nám nágotujećie.
Y odeƺli ucżniowie jego / y przyƺli do miáſtá / y ználeſli ták / jáko im był powiedźiáł / y nágotowáli báránká.
A gdy był wiecżór / przyƺedł ze dwunáſtomá.
A gdy zá ſtołem śiedźieli y jedli / rzekł JEzuſ : Zápráwdę wám powiádám / iż jeden z was wydá mię / który je ze mną.
Tedy oni pocżęli śię ſmućić / y do niego mówić / káżdy z oſobná : Azáżem ja jeſt? A drugi : Azáż ja?
Lecż on odpowiádájąc rzekł im : Jeden ze dwunáſtu / który ze mną mácżá w miśie.
Synći cżłowiecży idźie / jáko o nim nápiſáno : ále biádá cżłowiekowi temu / przez którego Syn cżłowiecży będźie wydány! dobrzeby mu było / by śię był ten cżłowiek nie národźił.
A gdy oni jedli / wźiął JEzuſ chleb / á pobłogoſłáwiwƺy / łámáł y dáł im / mówiąc : Bierzćie / jedzćie / to jeſt ćiáło moje.
A wźiąwƺy kielich / y podźiękowáwƺy / dáł im ; y pili z niego wƺyſcy.
Y rzekł im : To jeſt krew mojá nowego teſtámentu / która śię zá wielu wylewá.
Zápráwdę powiádám wám : Iż nie będę więcey pił z rodzáju winney máćicy / áż do dniá onego / gdy go pić będę nowy w króleſtwie Bożem.
A záśpiewáwƺy pieśń / wyƺli ná górę oliwną.
Potem im rzekł JEzuſ : Wƺyſcy wy zgorƺyćie śię zemnie tey nocy ; bo nápiſáno : Uderzę páſterzá / y będą rozproƺone owce.
Lecż gdy zmartwychwſtánę / poprzedzę was do Gálileii.
Ale mu Piotr powiedźiáł : Choćby śię wƺyſcy zgorƺyli / ále ja nie.
Y rzekł mu JEzuſ : Zápráwdę powiádám tobie / iż dźiś tey nocy / pierwej niż dwákroć kur zápieje / trzykroć śię mnie záprzeƺ.
Ale on tem więcey mówił : Bym z tobą miał y umrzeć / nie záprę śię ćiebie. Toć też y wƺyſcy mówili :
Y przyƺli ná miejſce / które zwáno Gietſemáne ; tedy rzekł do ucżniów ſwoich : Siedźćie tu / áż śię pomodlę.
Y wźiąwƺy z ſobą Piotrá / y Jákóbá / y Jáná / pocżął śię lękáć / y bárdzo tęſknić ;
Y rzekł im : Bárdzo jeſt ſmutná duƺá mojá áż do śmierći ; zoſtáńćie tu á cżujćie.
A poſtąpiwƺy trochę / pádł ná źiemię y modlił śię / áby / jeſli możná / odeƺłá od niego tá godźiná ;
Y rzekł : Abbá Ojcże! wƺyſtko tobie jeſt możno / przenieś odemnie ten kielich ; wƺákże nie co ja chcę / ále co Ty.
Tedy przyƺedł / y ználázł je śpiące / y rzekł Piotrowi : Szymonie / śpiƺ? niemogłeś cżuć jedney godźiny?
Cżujćie á módlćie śię / ábyśćie nie weƺli w pokuƺenie ; duchći jeſt ochotny / ále ćiáło mdłe.
Y odƺedłƺy znowu / modlił śię / też ſłowá mówiąc.
A wróćiwƺy śię ználázł je záśię śpiące / ( bo ocży ich były obćiążone / ) á nie wiedźieli / co mu odpowiedźieć mieli.
Y przyƺedł po trzećie / á rzekł im : Śpijćież już y odpocżywájćie! Doſyćći! przyƺłáć tá godźiná / oto wydány bywá Syn cżłowiecży w ręce grzeƺników.
Wſtáńćie / pójdźmy! oto który mię wydáwá / bliſko jeſt.
A wnetże / gdy on jeƺcże mówił / przyƺedł Judáƺ / który był jeden ze dwunáſtu / á z nim wielká zgrájá z miecżámi y z kijámi od przedniejƺych kápłánów / y od náucżonych w Piśmie y od ſtárƺych.
A ten / który go wydáwáł / dáł im był znák / mówiąc : Któregokolwiek pocáłuję / tenći jeſt / imájćież go / á wiedźćie oſtrożnie.
A przyƺedłƺy / zárázem przyſtąpił do niego / y rzekł : Miſtrzu / Miſtrzu! y pocáłowáł go.
Tedy śię oni náń rękomá rzućili / y pojmáli go.
A jeden z tych / co tám ſtáli / dobywƺy miecżá / uderzył ſługę nájwyżƺego kápłáná / y ućiął mu ucho.
A JEzuſ odpowiádájąc rzekł im : Jáko ná zbójcę wyƺliśćie z miecżámi y z kijámi / ábyśćie mię pojmáli.
Ná káżdy dźień bywáłem u was w kośćiele / ucżąc / á nie pojmáliśćie mię : ále trzebá / áby śię wypełniły Piſmá.
A ták opuśćiwƺy go / wƺyſcy ućiekli.
A jeden jákiś młodźieniec ƺedł zá nim / przyodźiány prześćierádłem ná nágie ćiáło ; y uchwyćili go młodźieńcy.
Ale on opuśćiwƺy prześćierádło / nágo ućiekł od nich.
Tedy przywiedli JEzuſá do nájwyżƺego kápłáná : á zeƺli śię do niego wƺyſcy przedniejśi kápłáni / y ſtárśi / y náucżeni w Piśmie.
A Piotr ƺedł z nim z dáleká áż do dworu nájwyżƺego kápłáná / y śiedźiáł z ſługámi / grzejąc śię u ogniá.
Ale przedniejśi kápłáni / y wƺyſtká rádá ƺukáli przećiwko JEzuſowi świádectwá / áby go ná śmierć wydáli ; wƺákże nie ználeſli.
Abowiem ich wiele fáłƺywie świádcżyli przećiwko niemu ; ále świádectwá ich nie były zgodne.
Tedy niektórzy powſtáwƺy / fáłƺywie świádcżyli przećiwko niemu / mówiąc :
Myśmy to ſłyƺeli / że mówił : Ja rozwálę ten kośćiół ręką ucżyniony / á we trzech dniách inny nie ręką ucżyniony zbuduję.
Lecż y ták nie było zgodne świádectwo ich.
Tedy ſtánąwƺy w pośrodku nájwyżƺy kápłán / pytáł JEzuſá / mówiąc : Nie odpowiádáƺ nic? Cóż to jeſt / co ći przećiwko tobie świádcżą?
Ale on milcżáł á nic nie odpowiedźiáł. Znowu go pytáł nájwyżƺy kápłán / y rzekł mu : Tyżeś jeſt on CHryſtuſ / Syn onego Błogoſłáwionego?
A JEzuſ rzekł : Jám jeſt ; y ujrzyćie Syná cżłowiecżego / śiedzącego ná práwicy mocy Bożej / y przychodzącego z obłokámi niebieſkiemi.
Tedy nájwyżƺy kápłán rozdárłƺy ƺáty ſwoje / rzekł : Cóż jeƺcże potrzebujemy świádków?
Słyƺeliśćie bluźnierſtwo. Cóż śię wám zdá? A oni wƺyſcy oſądźili go winnym być śmierći.
Y pocżęli niektórzy náń plwáć / y zákrywáć oblicże jego / y bić weń pięśćiámi y mówić mu : Prorokuj! A ſłudzy policżkowáli go.
A gdy Piotr był we dworze ná dole / przyƺłá jedná z dźiewek nájwyżƺego kápłáná ;
A ujrzáwƺy Piotrá grzejącego śię / wejrzáłá náń / y rzekłá : Y tyś był z JEzuſem Názáreńſkim.
Ale śię on záprzáł / mówiąc : Nie znám go / á nie wiem / co ty mówiƺ. Y wyƺedł ná dwór do przyśionká / á kur zápiáł.
Tedy dźiewká ujrzáwƺy go záśię / pocżęłá mówić tym / którzy tám ſtáli : Ten jeſt jeden z nich.
A on záśię záprzáł śię. A znowu po máłej chwilce ći / co tám ſtáli / rzekli Piotrowi : Práwdźiwie z nich jeſteś ; boś y Gálilejcżyk / y mowá twojá podobná jeſt.
A on śię pocżął przeklináć y przyśięgáć / mówiąc : Nie znám cżłowieká tego / o którym mówićie.
Tedy powtóre kur zápiáł. Y wſpomniáł Piotr ná ſłowá / które mu był powiedźiáł JEzuſ : że pierwej niż kur dwákroć zápieje / trzykroć śię mnie záprzeƺ. A wyƺedłƺy / płákáł.
