A záráz ráno nárádźiwƺy śię przedniejśi kápłáni z ſtárƺymi y z náucżonymi w Piśmie y ze wƺyſtką rádą / związáli JEzuſá / y wiedli go / y podáli Piłátowi.
Y pytáł go Piłát : Tyżeś jeſt król Żydowſki? A on mu odpowiádájąc rzekł : Ty powiádáƺ.
Y ſkárżyli náń przedniejśi kápłáni o wiele rzecży ; ( ále on nic nie odpowiedźiáł. )
Tedy go záśię pytáł Piłát / mówiąc : Nic nie odpowiádáƺ? Oto jáko wiele rzecży świádcżą przećiwko tobie.
Ale JEzuſ przećię nic nie odpowiedźiáł / ták / iż śię Piłát dźiwowáł.
A ná święto zwykł im był wypuƺcżáć więźniá jednego / o któregoby prośili.
Y był jeden / którego zwáno Bárábbáƺ / w więźieniu z tymi / co rozruch cżynią / którzy byli w rozruchu mężobójſtwo popełnili.
Tedy lud wyſtąpiwƺy y głoſem záwołáwƺy / pocżął prośić / żeby ucżynił ták / jáko im záwƺe cżynił /
Ale Piłát im odpowiedźiáł / mówiąc : Chcećież / wypuƺcżę wám królá Żydowſkiego?
( Wiedźiáł bowiem / iż go z nienáwiśći wydáli przedniejśi kápłáni. )
Ale przedniejśi kápłáni podburzáli lud / iżby im rácżej Bárábbáƺá wypuśćił.
A odpowiádájąc Piłát / rzekł im záśię : Cóż tedy chcećie / ábym ucżynił temu / którego názywáćie królem Żydowſkim?
A oni znowu záwołáli : Ukrzyżuj go!
A Piłát rzekł do nich : Y cóż wżdy złego ucżynił? Ale oni tem bárdźiej wołáli : Ukrzyżuj go!
A ták Piłát / chcąc ludowi doſyć ucżynić / wypuśćił im Bárábbáƺá / á JEzuſá ubicżowáwƺy / podáł im / áby był ukrzyżowány.
Lecż żołnierze wprowádźili go do dworu / to jeſt do rátuƺá / y zwołáli wƺyſtkiej roty.
A oblekłƺy go w ƺárłát / y uplótłƺy koronę z ćierniá / włożyli náń ;
Y pocżęli go pozdráwiáć / mówiąc : Bądź pozdrowiony / królu Żydowſki!
Y bili głowę jego trzćiną y plwáli náń / á upádájąc ná koláná / kłániáli mu śię.
A gdy śię z niego náśmiáli / zewlekli go z ƺárłátu / y oblekli go w ƺáty jego włáſne / y wiedli go / áby go ukrzyżowáli.
Tedy przymuśili mimo idącego niektórego Szymoná Cyrenejcżyká / ( który ƺedł z polá / ) ojcá Alekſándrowego y Rufowego / áby nióſł krzyż jego.
Y przywiedli go ná miejſce Golgotá / co śię wykłádá : Miejſce trupich głów.
Y dáwáli mu pić wino z myrrą ; ále go on nie przyjął.
A gdy go ukrzyżowáli / rozdźielili ƺáty jego / miecąc o nie loſ / coby kto wźiąć miał.
A byłá trzećiá godźiná / gdy go ukrzyżowáli.
Był też nápiſ winy jego nápiſány : Król Żydowſki.
Ukrzyżowáli też z nim dwóch zbójców ; jednego po práwicy / á drugiego po lewicy jego.
Y wypełniło śię Piſmo / które mówi : Z złocżyńcámi jeſt policżony.
A ći / którzy mimo chodźili / bluźnili go / chwiejąc głowámi ſwemi á mówiąc : Ehej! który rozwáláƺ kośćiół á we trzech dniách budujeƺ go!
Rátuj ſámego śiebie / á zſtąp z krzyżá!
Tákże też y przedniejśi kápłáni náśmiewájąc śię / jedni do drugich z náucżonymi w Piśmie mówili : Innych rátowáł / á ſámego śiebie rátowáć nie może ;
Niechże teraz CHryſtuſ on król Izráelſki zſtąpi z krzyżá / ábyśmy ujrzeli y uwierzyli. Ci też / co z nim byli ukrzyżowáni / urągáli mu.
A gdy było o godźinie ƺóſtey / ſtáłá śię ćiemność po wƺyſtkiej źiemi / áż do godźiny dźiewiątey.
A o godźinie dźiewiątey záwołáł JEzuſ głoſem wielkim / mówiąc : Eloi! Eloi! Lámmá ſábáchtáni / co śię wykłádá : Boże mój! Boże mój! cżemuś mię opuśćił?
A niektórzy z tych / co tám ſtáli / uſłyƺáwƺy mówili : Oto Elijáƺá wołá.
Zátem bieżáwƺy jeden / nápełnił gąbkę octem / á włożywƺy ją ná trzćinę / dáwáł mu pić / mówiąc : Zániechájćie / pátrzmy / jeſli przyjdźie Elijáƺ / zdejmowáć go.
A JEzuſ záwołáwƺy głoſem wielkim / oddáł duchá.
Y rozerwáłá śię záſłoná kośćielná ná dwoje / od wierzchu áż do dołu.
Tedy widząc ſetnik / który ſtáł przećiwko niemu / iż ták wołájąc oddáł duchá / rzekł : Práwdźiwie cżłowiek ten był Synem Bożym.
Były też y niewiáſty z dáleká śię przypátrując / między któremi byłá Máryjá Mágdálená / y Máryjá / Jákóbá máłego y Jozeſá mátká / y Sálome ;
Które gdy jeƺcże były w Gálilei / chodźiły zá nim / á poſługowáły mu ; y wiele innych / które z nim były / wſtąpiły do Jeruzálemu.
A gdy już był wiecżór / ( poniewáż był dźień przygotowániá / ) który jeſt przed ſábátem /
Przyƺedłƺy Józef z Arymátyi / pocżeſny rádny pán / który też ſám ocżekiwáł króleſtwá Bożego / śmiele wƺedł do Piłátá / y prośił o ćiáło JEzuſowe.
A Piłát śię dźiwowáł / jeſliby już umárł ; y záwołáwƺy ſetniká / pytáł go / dáwnoli umárł?
A dowiedźiáwƺy śię od ſetniká / dárowáł ćiáło Józefowi.
A on kupiwƺy prześćierádło / y zdjąwƺy go / obwinął w prześćierádło / y położył go w grobie / który był wykowány z opoki / y przywálił kámień do drzwi grobowych.
Ale Máryjá Mágdálená / y Máryjá / mátká Jozeſowá / pátrzáły / kędy go położono.
