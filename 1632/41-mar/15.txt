A záraz ráno nárádźiwƺy śię Przedniejƺy Kápłani z Stárƺymi y z Náucżonymi w piśmie / y ze wƺyſtką Rádą / związáli JEzuſá / y wiedli go / y podáli Piłatowi.
Y pytał go Piłat ; Tyżeś jeſt Król Żydowſki? A on mu odpowiádájąc rzekł ; Ty powiádaƺ.
Y ſkárżyli nań Przedniejƺy Kápłani o wiele rzecży : ( ále on nic nieodpowiedźiał. )
Tedy go záśię pytał Piłat / mówiąc ; Nic nie odpowiádaƺ? Oto jáko wiele rzecży świadcżą przećiwko tobie.
Ale JEzus przećię nic nie odpowiedźiał / ták / iż śię Piłat dźiwował.
A ná Święto zwykł im był wypuƺcżáć więźniá jednego o któregoby prośili.
Y był <i>jeden,</i> którego zwano Bárábbaƺ / w więźieniu / z tymi co rozruch cżynią ; którzy byli w rozruchu mężobójſtwo popełnili.
Tedy lud głoſem záwoławƺy / pocżął prośić <i>żeby ucżynił</i> ták jáko im záwżdy cżyniwał.
Ale Piłat im odpowiedźiał / mówiąc ; Chcećież / wypuƺcżę wam Królá Żydowſkiego?
( Wiedźiał bowiem yż go z nienawiśći wydáli Przedniejƺy Kápłani. )
Ale Przedniejƺy Kápłani podburzáli lud / iżby im rácżey Bárábbaƺá wypuśćił.
A odpowiádájąc Piłat / rzekł im záśię ; Cóż tedy chcećie ábym ucżynił <i>temu,</i> którego názywaćie Królem Żydowſkim?
A oni znowu záwołáli ; Ukrzyżuj go!
A Piłat rzekł do nich ; Y cóż wżdy złego ucżynił? Ale oni tym bárźiey wołáli ; Ukrzyżuj go!
A ták Piłat chcąc ludowi doſyć ucżynić / wypuśćił im Bárábbaƺá ; á JEzuſá ubicżowawƺy / podał im / áby był ukrzyżowány.
Lecż żołnierze wprowádźili go do dworu / to jeſt do Ratuƺá ; y zwołáli wƺyſtkiey roty.
A oblekƺy go w ƺárłat / y uplótƺy koronę z ćiernia / włożyli nań.
Y pocżęli go pozdrawiáć / <i>mówiąc</i> : Bądź pozdrowiony Królu Żydowſki!
Y bili głowę jego trzćiną / y plwáli nań ; á upadájąc ná koláná / kłaniáli mu śię.
A gdy śię z niego náśmiali / zewlekli go z ƺárłatu / y oblekli go w ƺáty <i>jego</i> właſne : y wiedli go áby go ukrzyżowáli.
Tedy przymuśili mimo idącego niektórego Symoná Cyrenejcżyká / ( który ƺedł z polá ) Ojcá Alexándrowego y Rufowego / áby nióſł krzyż jego.
Y przywiedli go ná miejſce Golgotá / co śię wykłada ; Miejſce trupich głów.
Y dawáli mu pić wino z mirrą : ále go on nie przyjął.
A gdy go ukrzyżowáli / rozdźielili ƺáty jego / miecąc o nie los / coby kto wźiąć miał.
A byłá trzećia godźiná gdy go ukrzyżowáli.
Był też nápis winy jego nápiſány ; Król Żydowſki.
Ukrzyżowáli też z nim dwu zbójców : jednego po práwicy / á drugiego po lewicy jego.
Y wypełniło śię piſmo które mówi ; Z złocżyńcámi jeſt policżony.
A ći którzy mimo chodźili / bluźnili go / chwiejąc głowámi ſwymi / á mówiąc ; Ehey który rozwalaƺ Kośćiół / á we trzech dniách budujeƺ <i>go</i>!
Rátuj ſámego śiebie / á zſtąp z krzyżá!
Tákże też y Przedniejƺy Kápłani náśmiewájąc śię / jedni do drugich z Náucżonymi w piſmie mówili : Innych rátował / á ſámego śiebie rátowáć nie może.
Niechże teraz CHryſtus on Król Izráelſki zſtąpi z krzyżá / ábyſmy ujrzeli y uwierzyli. Ci też co z nim byli ukrzyżowáni / urągáli mu.
A gdy było o godźinie ƺóſtey / ſtáłá śię ćiemność po wƺyſtkiey źiemi / áż do godźiny dźiewiątey.
A o godźinie dźiewiątey záwołał JEzus głoſem wielkim / mówiąc ; Eloi! Eloi! lámmá ſábáchtáni : co śię wykłada ; Boże mój! Boże mój! cżemuś mię opuśćił?
A niektórzy z tych co tám ſtali uſłyƺawƺy / mówili : Oto Eliaƺá woła.
Zátym bieżawƺy jeden / nápełnił gębkę octem ; á włożywƺy ją ná trzćinę / dawał mu pić / mówiąc ; Zániechajćie / pátrzmy jeſli przydźie Eliaƺ zdejmowáć go.
A JEzus záwoławƺy głoſem wielkim / oddał duchá.
Y rozerwáłá śię zaſłoná kośćielna ná dwoję / od wierzchu áż do dołu.
Tedy widząc Setnik który ſtał przećiwko niemu / iż ták wołájąc oddał duchá / rzekł ; Prawdźiwie cżłowiek ten był Synem Bożym.
Były też y niewiáſty z dáleká / śię przypátrując / miedzy którymi byłá Márya Mágdálená / y Márya / Jákubá máłego y Jozeſá mátká / y Sálome :
Które gdy jeƺcże były w Gálilejey / chodźiły zá nim / á poſługowáły mu : y wiele innych które z nim były wſtąpiły do Jeruzalem.
A gdy już był wiecżór ( ponieważ był <i>dźień</i> przygotowánia ) który jeſt przed Sábbátem :
Przyƺedƺy Józef z Arymátyey / pocżeſny Rádny Pan / który też ſam ocżekawał króleſtwá Bożego / śmiele wƺedł do Piłatá / y prośił o ćiáło JEzuſowe.
A Piłat śię dźiwował jeſliby już umárł : y záwoławƺy Setniká / pytał go dawnoli umárł?
A dowiedźiawƺy śię od Setniká / dárował ćiáło Józefowi.
A on kupiwƺy prześćierádło / y zdjąwƺy go / obwinął w prześćierádło ; y położył go w grobie który był wykówány z opoki / y przywálił kámień do drzwi grobowych.
Ale Márya Mágdálená / y Márya <i>mátká</i> Joſeſowá / pátrzáły kędy go położono.
