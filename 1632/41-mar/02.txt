A záśię przyƺedł po kilku dniách do Kápernáum / y uſłyƺáno / że jeſt w domu.
A wnet zeƺło śię ich ták wiele / że śię zmieśćić nie mogli áni przede drzwiámi ; y opowiádáł im ſłowo Boże.
Tedy przyƺli do niego nioſący powietrzem ruƺonego / którego cżterej nieśli.
A gdy do niego przyſtąpić nie mogli dlá ćiżby / odárli dách / gdźie był JEzuſ / á przełámáwƺy go / ſpuśćili po powrozách ná dół łoże / ná którem leżáł powietrzem ruƺony.
A widząc JEzuſ wiárę ich / rzekł powietrzem ruƺonemu : Synu! odpuƺcżone ſą tobie grzechy twoje.
A byli tám niektórzy z náucżonych w Piśmie / śiedząc y myśląc w ſercách ſwoich :
Cżemuż ten tákie mówi bluźnierſtwá? któż może grzechy odpuƺcżáć / tylko ſám Bóg?
A záráz poznáwƺy JEzuſ duchem ſwym / iż ták w ſobie myśleli / rzekł im : Cżemuż ták myślićie w ſercách wáƺych?
Cóż łátwiejƺego jeſt / rzec powietrzem ruƺonemu : Odpuƺcżone ſą tobie grzechy / cżyli rzec : Wſtáń y weźmij łoże twoje á chodź?
Ale żebyśćie wiedźieli / iż Syn cżłowiecży má moc ná źiemi grzechy odpuƺcżáć / rzekł powietrzem ruƺonemu :
Tobie mówię : Wſtáń / á weźmij łoże twoje / á idź do domu twego.
A on zárázem wſtáł / y wźiąwƺy łoże ſwoje / wyƺedł przed wƺyſtkimi / ták iż śię wƺyſcy zdumiewáli y chwálili Bogá / mówiąc : Nigdyśmy nic tákiego nie widźieli.
Y wyƺedł záśię nád morze / á wƺyſtek lud przychodźił do niego / y náucżáł je.
A idąc mimo cłá / ujrzáł Lewiego / ſyná Alfeuƺowego / śiedzącego ná cle / y rzekł mu : Pójdź zá mną! á on wſtáwƺy ƺedł zá nim.
Y ſtáło śię / gdy JEzuſ śiedźiáł zá ſtołem w domu jego / że wiele celników y grzeƺników weſpół śiedźiáło z JEzuſem y z ucżniámi jego ; bo ich wiele było / y chodźili zá nim.
A náucżeni w Piśmie y Fáryzeuƺowie widząc go / że jádł z celnikámi y z grzeƺnikámi / mówili do ucżniów jego : Cóż jeſt / iż z celnikámi y z grzeƺnikámi je y pije?
A uſłyƺáwƺy to JEzuſ / rzekł im : Nie potrzebują zdrowi lekárzá / ále ći / co śię źle máją ; nie przyƺedłem / wzywáć ſpráwiedliwych / ále grzeƺnych do pokuty.
A ucżniowie Jánowi y Fáryzejſcy pośćili / á przyƺedłƺy mówili do niego : Cżemuż ucżniowie Jánowi y Fáryzejſcy poƺcżą / á twoi ucżniowie nie poƺcżą?
Y rzekł im JEzuſ : Izáli mogą ſynowie łożnicy máłżeńſkiej pośćić / póki z nimi jeſt oblubieniec? Póki z ſobą oblubieńcá máją / nie mogą pośćić.
Ale przyjdą dni / gdy od nich odjęty będźie oblubieniec / á tedy będą pośćić w one dni.
A żáden nie wpráwuje łáty ſukná nowego w ƺátę wiotchą / inácżej oná jego łátá nowá ujmuje nieco od wiotchej ƺáty / y ſtáwá śię gorƺe rozdárćie.
Y żáden nie leje winá młodego w ſtáre ſtátki ; bo inácżej wino młode rozſádzá ſtátki / y wyćieká wino / á ſtátki śię pſują ; ále wino nowe má być wlewáne w ſtátki nowe.
Y ſtáło śię / że ƺedł JEzuſ w ſábát przez zbożá / y pocżęli ucżniowie jego idąc rwáć kłoſy.
Ale Fáryzeuƺowie mówili do niego : Oto cżemu ći cżynią w ſábát / cżego śię nie godźi cżynić?
A on im rzekł : Nigdyśćie nie cżytáli / co ucżynił Dawid / gdy niedoſtátek ćierpiáł / á łáknął / ſám y ći / którzy z nim byli?
Jáko wƺedł do domu Bożego zá Abijátárá / kápłáná nájwyżƺego / y jádł chleby pokłádne / ( których śię nie godźiło jeść / tylko kápłánom ) á dáł y tym / którzy z nim byli.
Dotego rzekł im : Sábát dlá cżłowieká ucżyniony / á nie cżłowiek dlá ſábátu.
Dlátego Syn cżłowiecży jeſt Pánem y ſábátu.
