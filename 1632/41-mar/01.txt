Ten jeſt pocżątek Ewángielii Jezuſá Chryſtuſá / Syná Bożego.
Jáko nápiſáno w prorokách : Oto Já poſyłám Aniołá mego przed oblicżem twojim / który zgotuje drogę twoję przed tobą.
Głoſ wołájącego ná puƺcży : Gotujćie drogę Páńſką / proſte cżyńćie śćieżki jego.
Ján chrzćił ná puƺcży / y kázáł chrzeſt pokuty ná odpuƺcżenie grzechów.
Y wychodziłá do niego wƺyſtká kráiná Judzká / y Jeruzálemcżycy / á wƺyſcy byli od niego chrzcżeni w rzece Jordánie / wyznáwájąc grzechy ſwoje.
Ale Ján przyodziány był śierśćią wielbłądową / á páſ ſkórzány był około biódr jego / á jádáł ƺáráńcżę y miód leśny.
Y kázáł / mówiąc : Idzie zá mną możniejƺy niżeli já / któremum nie jeſt godzien / ſchyliwƺy śię / rozwiązáć rzemyká u obuwiá jego.
Jámći wáſ chrzćił wodą ; ále on wáſ będzie chrzćił Duchem Świętym.
Y ſtáło śię w one dni / że przyƺedł Jezuſ z Názáretu Gálilejſkiego / á ochrzcżony jeſt od Jáná w Jordánie.
A zárázem wyſtąpiwƺy z wody / ujrzáł rozſtępujące śię niebioſá / y Duchá jáko gołębicę náń zſtępującego.
Y ſtáł śię głoſ z niebá : Tyś jeſt on Syn mój miły / w którym mi śię upodobáło.
A záráz Duch wypędził go ná puƺcżą.
Y był tám ná puƺcży przez cżterdzieśći dni / będąc kuƺony od ƺátáná / á był z zwierzęty / á Aniołowie ſłużyli mu.
Lecż potem / gdy Ján był podány do więzieniá / przyƺedł Jezuſ do Gálilei / káżąc Ewángieliją króleſtwá Bożego /
A mówiąc : Wypełnił śię cżáſ / y przybliżyło śię króleſtwo Boże : Pokutujćie / á wierzćie Ewángielii.
A przechodząc śię nád morzem Gálilejſkiem / ujrzáł Szymoná y Andrzejá / brátá jego / zápuƺcżájących śieć w morze ; bo byli rybitwi.
Y rzekł im Jezuſ : Pójdźćie zá mną / á ucżynię wáſ rybitwámi ludzi.
A oni zárázem opuśćiwƺy śieći ſwoje / poƺli zá nim.
A ſtámtąd troƺecżkę odƺedłƺy / ujrzáł Jákóbá / ſyná Zebedeuƺowego / y Jáná / brátá jego / á oni w łodzi popráwiáli śieći.
Y záráz ich powołáł ; á oni zoſtáwiwƺy ojcá ſwego Zebedeuƺá w łodzi z cżeládzią / poƺli zá nim.
Potem weƺli do Kápernáum ; á záráz w ſábát wƺedłƺy Jezuſ do bóżnicy / náucżáł.
Y zdumiewáli śię nád náuką jego ; ábowiem on ich ucżył jáko májący moc / á nie jáko náucżeni w Piśmie.
A był w bóżnicy ich cżłowiek májący duchá niecżyſtego / który záwołáł /
Mówiąc : Ach! cóż my z tobą mámy / Jezuśie Názáreńſki? Przyƺedłeś / ábyś náſ wytráćił. Znám ćię / któś jeſt / żeś on święty Boży.
Y zgromił go Jezuſ / mówiąc : Umilknij á wynijdź z niego.
Tedy rozdárłƺy go duch niecżyſty y záwołáwƺy głoſem wielkim / wyƺedł z niego.
Y wylękli śię wƺyſcy / ták iż śię pytáli między ſobą / mówiąc : Cóż to jeſt? cóż to zá nowá náuká / iż mocą y duchom niecżyſtym rozkázuje / y ſą mu poſłuƺni?
Y rozeƺłá śię powieść o nim prędko po wƺyſtkiej kráinie / leżącej około Gálilei.
A wyƺedłƺy záráz z bóżnicy / przyƺli do domu Szymonowego y Andrzejowego z Jákóbem y z Jánem.
A świekrá Szymonowá leżáłá / májąc gorącżkę / o której mu wnet powiedzieli.
Tedy przyſtąpiwƺy podnióſł ją / ująwƺy ją zá rękę jej ; á záráz ją gorącżká opuśćiłá / y poſługowáłá im.
A gdy był wiecżór y ſłońce záchodziło / przynośili do niego wƺyſtkie / którzy śię źle mieli / y opętáne ;
A wƺyſtko miáſto zgromádziło śię do drzwi.
Y uzdrowił wiele tych / co śię źle mieli ná rozlicżne choroby / y wygnáł wiele dyjábłów / á nie dopuśćił mówić dyjábłom ; bo go ználi.
A bárdzo ráno przede dniem wſtáwƺy / wyƺedł y odƺedł ná puſte miejſce / á tám śię modlił.
Y poƺli zá nim Szymon y ći / którzy z nim byli ;
A ználázłƺy go / rzekli mu : Wƺyſcy ćię ƺukáją.
Tedy im on rzekł : Idźmy do przyległych miáſtecżek / ábym y tám kázáł ; bom ná to przyƺedł.
Y kázáł w bóżnicách ich po wƺyſtkiej Gálilei / y wygániáł dyjábły.
Tedy przyƺedł do niego trędowáty / proƺąc go y upádájąc przed nim ná koláná / y mówiąc mu : Jeźli chceƺ / możeƺ mię ocżyśćić.
A ták Jezuſ użáliwƺy śię / wyćiągnął rękę / á dotknął śię go y rzekł mu : Chcę / bądź ocżyƺcżony!
A gdy to Pán rzekł / zárázem odƺedł trąd od niego / y był ocżyƺcżony.
A ſrodze mu przygroziwƺy Jezuſ / záráz go odpráwił ;
Y rzekł mu : Pátrz / ábyś nikomu nie powiádáł / ále idź á ukáż śię kápłánowi / y ofiáruj zá ocżyƺcżenie twoje to / co rozkázáł Mojżeƺ / ná świádectwo przećiwko nim.
Ale on odƺedłƺy / pocżął wiele opowiádáć y rozſłáwiáć tę rzecż / ták iż już nie mógł Jezuſ jáwnie wnijść do miáſtá / ále był ná uſtroniu ná miejſcách puſtych. Y ſchodzili śię do niego zewƺąd.
