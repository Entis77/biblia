Tedy śię przepráwili zá morze do kráiny Gádáreńcżyków.
A gdy on wyƺedł z łodzi / záráz mu zábieżáł z grobów cżłowiek májący duchá niecżyſtego ;
Który miáł mieƺkánie w grobách / á nie mógł go nikt y łáńcuchámi związáć /
Przeto że on cżęſto będąc pętámi y łáńcuchámi związány / łáńcuchy porwáł / y pętá pokruƺył ; á nie mógł go nikt ukróćić.
A záwƺe we dnie y w nocy ná górách w grobách był / wołájąc y kámieniem śię tłukąc.
Ujrzáwƺy tedy Jezuſá z dáleká / bieżáł y pokłonił mu śię ;
A wołájąc głoſem wielkim / rzekł : Cóż mám z tobą Jezuśie / Synu Bogá nájwyżƺego? Poprzyśięgám ćię przez Bogá / ábyś mię nie trápił.
( Abowiem mu mówił : Wynijdź / duchu niecżyſty! z tego cżłowieká. )
Tedy go pytáł : Co máƺ zá imię? A on odpowiádájąc / rzekł : Imię moje jeſt wojſko : ábowiem náſ jeſt wiele.
Y prośił go bárdzo / áby ich nie wygániáł z onej kráiny.
A byłá tám przy górách wielká trzodá świni / którá śię páſłá.
Y prośili go oni wƺyſcy dyjábli / mówiąc : Puść náſ w te świnie / ábyśmy w nie weƺli.
Y pozwolił im záráz Jezuſ. A wyƺedłƺy oni duchowie niecżyśći / weƺli w one świnie ; y porwáłá śię oná trzodá z przykrá w morze ( á było ich około dwóch tyśięcy / ) y potonęły w morzu.
A oni / którzy świnie páśli / ućiekli / y oznájmili to w mieśćie y we wśiách ; y wyƺli / áby oglądáli to / co śię ſtáło.
Y przyƺli do Jezuſá / y ujrzeli onego / który był opętány / y śiedziáł oblecżony / będąc przy dobrem bácżeniu ; onego / mówię / w którym było wojſko dyjábłów ; y báli śię.
A ći / którzy to widzieli / opowiedzieli im / co śię dziáło z onym opętánym / y o świniách.
Tedy go pocżęli prośić / áby odƺedł z gránic ich.
A gdy wſtąpił w łódź / prośił go on / co był opętánym / áby był przy nim.
Lecż mu Jezuſ nie dopuśćił / ále mu rzekł : Idź do domu ſwego / do ſwoich / á oznájmij im / jákoć wielkie rzecży Pán ucżynił / á jáko śię nád tobą zmiłowáł.
Tedy odƺedł / y pocżął opowiádáć w dzieśięćiu miáſtách / jáko mu wielkie rzecży ucżynił Jezuſ ; y dziwowáli śię wƺyſcy.
A gdy śię záśię Jezuſ przepráwił w łodzi ná drugą ſtronę / zebráł śię do niego wielki lud ; á on był nád morzem.
A oto przyƺedł jeden z przełożonych bóżnicy / imieniem Jáiruſ / á ujrzáwƺy go / przypádł do nóg jego /
Y prośił go wielce / mówiąc : Poniewáż córecżká mojá koná / pójdźże / włóż ná nię ręce / áby byłá uzdrowioná / y będzie żywá. Y poƺedł z nim.
Y ƺedł zá nim lud wielki / y ćiſnęli go.
Tedy niektórá niewiáſtá / którá ćierpiáłá płynienie krwi ode dwunáſtu lát /
Y wiele ućierpiáłá od wielu lekárzy / y wynáłożyłá wƺyſtko / co miáłá ; á nic jej nie pomogło / owƺem śię jej tem więcej pogorƺáło :
Uſłyƺáwƺy o Jezuśie / przyƺłá z tyłu między ludem / y dotknęłá śię ƺáty jego ;
Bo mówiłá : Jeźli śię tylko dotknę ƺáty jego / będę uzdrowioná.
A zárázem wyſchło źródło krwi jej / y pocżułá ná ćiele / że uzdrowioná byłá od choroby ſwojej.
A wnet poznáwƺy Jezuſ ſám w ſobie / że z niego moc wyƺłá / obróćił śię do ludu y rzekł : Kto śię dotknął ƺát moich?
Y rzekli mu ucżniowie jego : Widziƺ / że ćię ten lud ćiśnie / á mówiƺ : Kto śię mnie dotknął?
Y ſpojrzáł w koło / áby ujrzáł tę / którá to ucżyniłá :
Ale niewiáſtá oná z bojáźnią y ze drżeniem / wiedząc / co śię przy niej ſtáło / przyſtąpiłá y upádłá przed nim / á powiedziáłá mu wƺyſtkę práwdę.
Zátem jej on rzekł : Córko! wiárá twojá ćiebie uzdrowiłá / idźże w pokoju / á bądź zdrowá od choroby twojej.
A gdy on jeƺcże mówił / przyƺli ſłudzy od przełożonego bóżnicy / mówiąc : Córká twojá umárłá / cżemuż jeƺcże trudziƺ náucżyćielá?
Ale Jezuſ ſkoro uſłyƺáł to / co oni mówili / rzekł do przełożonego bóżnicy : Nie bój śię / tylko wierz!
Y nie dopuśćił nikomu iść zá ſobą / tylko Piotrowi / y Jákóbowi / y Jánowi / brátu Jákóbowemu.
A przyƺedł do domu przełożonego bóżnicy / y ujrzáł tám zgiełk / y płácżące y bárdzo nárzekájące.
Wƺedłƺy tedy / rzekł im : Przecż zgiełk cżynićie y płácżećie? nie umárłáć dziewecżká / ále śpi.
Y náśmiewáli śię z niego. Ale on wygnáwƺy wƺyſtkie / wziął z ſobą ojcá y mátkę dziewecżki / y te / którzy przy nim byli / y wƺedł tám / gdzie dziewecżká leżáłá.
A ująwƺy zá rękę onę dziewecżkę / rzekł do niej : Tálitá kumi! co śię wykłádá : Dziewecżko ( tobie mówię ) wſtáń!
A záráz dziewecżká wſtáłá / y chodziłá ; ábowiem byłá w dwunáſtym roku. Y zdumieli śię zdumieniem wielkiem.
Tedy im przykázáł wielce / áby tego nikt nie wiedziáł / y rozkázáł / áby jej dáno jeść.
