Tedy śię przepráwili zá morze do krájiny Gádáreńcżyków.
A gdy on wyƺedł z łodźi / záraz mu zábieżał z grobów cżłowiek májący duchá niecżyſtego :
Który miał mieƺkánie w grobiech ; á nie mógł go nikt y łáncuchámi związáć.
Przeto że on cżęſto będąc pętámi y łáncuchámi związány / łáncuchy porwał / y pętá pokruƺył ; á nie mógł go nikt okróćić.
A záwżdy we dnie y w nocy ná górách w grobiech był / wołájąc y kámieniem śię tłukąc.
Ujrzawƺy tedy JEzuſá z dáleká / bieżał / y pokłonił mu śię ;
A wołájąc głoſem wielkim / rzekł : Cóż mam z tobą JEzuśie Synu Bogá Nawyżƺego? Poprzyśięgam ćię przez Bogá / ábyś mię nie trapił.
( Abowiem mu mówił : Wynidź duchu niecżyſty z tego cżłowieká )
Tedy go pytał : Co maƺ zá imię? A on odpowiádájąc rzekł : Imię moje jeſt / Wojſko : ábowiem nas jeſt wiele.
Y prośił go bárzo áby ich nie wyganiał z oney krájiny.
A byłá tám przy górách wielka trzodá świni która śię páſłá.
Y prośili go oni wƺyſcy dyabli / mówiąc : Puść nas w te świnie / ábyſmy w nie weƺli.
Y pozwolił im záraz JEzus. A wyƺedƺy oni duchowie niecżyśći / weƺli w one świnie : y porwáłá śię oná trzodá z przykrá w morze / ( á było ich około dwu tyśięcy ) y potonęły w morzu.
A oni którzy świnie páśli / ućiekli / y oznájmili <i>to</i> w mieśćie / y we wśiách : y wyƺli áby oglądáli to co śię ſtáło.
Y przyƺli do JEzuſá / y ujrzeli onego który był opętány / iż śiedźiał oblecżony / będąc przy dobrym bacżeniu : onego <i>mówię</i> w którym było wojſko <i>dyabłów</i> ; y bali śię.
A ći którzy to widźieli / opowiedźieli im co śię dźiało z onym opętánym / y o świniách.
Tedy go pocżęli prośić / áby odƺedł z gránic ich.
A gdy wſtąpił w łódź / prośił go on co był opętánym / áby był przy nim.
Lecż mu JEzus nie dopuśćił / ále mu rzekł : Idź do domu ſwego / do ſwojich / á oznájmi im jákoć wielkie rzecży PAn ucżynił / á jáko śię nád tobą zmiłował.
Tedy odƺedł / y pocżął opowiádáć w Dźieśiąći miáſtách / jáko mu wielkie rzecży ucżynił JEzus : y dźiwowáli śię wƺyſcy.
A gdy śię záśię JEzus przepráwił w łodźi ná drugą ſtronę / zebrał śię do niego wielki lud / á <i>on</i> był nád morzem.
A oto przyƺedł jeden z Przełożonych bóżnice imieniem Jájirus ; á ujrzawƺy go / przypadł do nóg jego.
Y prośił go wielce / mówiąc : Ponieważ Córecżká mojá kona / pódźże włóż ná nię ręce / áby byłá uzdrowiona ; y będźie żywá. Y poƺedł z nim.
Y ƺedł zá nim lud wielki / y ćiſnęli go.
( Tedy niektóra niewiáſtá która ćierpiáłá płynienie krwie ode dwunaſtu lat :
Y wiele ućierpiáłá od wiela lekárzom / y wynáłożyłá wƺyſtko co miáłá / á nic jey nie pomogło / owƺem śię jey tym więcey pogarƺáło :
Uſłyƺawƺy o JEzuśie / przyƺłá z tyłu miedzy ludem / y dotknęłá śię ƺáty jego.
Bo mówiłá : Jeſli śię tylko dotknę ƺáty jego / będę uzdrowiona.
A zárázem wyſchło źrzódło krwie jey / y pocżułá ná ćiele / że uzdrowiona byłá od choroby ſwojey.
A wnet poznawƺy JEzus ſam w ſobie / że z niego moc wyƺłá / obróćił śię do ludu / y rzekł : Kto śię dotknął ƺat mojich?
Y rzekli mu ucżniowie jego : Widźiƺ że ćię ten lud ćiśnie / á mówiƺ / Kto śię mnie dotknął?
Y pojrzał w koło / áby ujrzał tę która to ucżyniłá.
Ale niewiáſtá oná z bojáźnią y ze drzeniem / wiedząc co śię przy niey ſtáło / przyſtąpiłá y upádłá przed nim / á powiedźiáłá mu wƺyſtkę prawdę.
Zátym jey on rzekł : Córko / wiárá twojá ćiebie uzdrowiłá : idźże w pokoju / á bądź zdrowá od choroby twojey. )
A gdy on jeƺcże mówił / przyƺli ſłudzy od Przełożonego bóżnice mówiąc : Córká twojá umárłá / cżemuż jeƺcże trudźiƺ Náucżyćielá?
Ale JEzus ſkoro uſłyƺał to co oni mówili / rzekł do Przełożonego bóżnice : Nie bój śię / tylko wierż.
Y nie dopuśćił nikomu iść zá ſobą / tylko Piotrowi / y Jákubowi / y Janowi brátu Jákubowemu.
A przyƺedł do domu Przełożonego bóżnice / y ujrzał <i>tám</i> zgiełk y płácżące / y bárzo nárzekájące.
Wƺedƺy tedy rzekł im : Przecż zgiełk cżynićie y płácżećie? Nie umárłáć dźiewecżká / ále śpi.
Y náśmiewáli śię z niego. Ale on wygnawƺy wƺyſtkie / wźiął z ſobą ojcá y mátkę dźiewecżki / y te którzy przy nim byli / y wƺedł tám gdźie dźiewecżká leżáłá.
A ująwƺy zá rękę onę dźiewecżkę / rzekł do niey : Tálitá kumi! Co śię wykłada ; Dźiewecżko ( tobie mówię ) wſtań.
A záraz dźiewecżká wſtáłá / y chodźiłá : ábowiem byłá we dwunaści lećiech. Y zdumieli śię zdumieniem wielkim.
Tedy im przykazał wielce áby tego nikt nie wiedźiał : y rozkazał áby jey dano jeść.
