Tedy śię przepráwili zá morze do kráiny Gádáreńcżyków.
A gdy on wyƺedł z łodźi / záráz mu zábieżáł z grobów cżłowiek májący duchá niecżyſtego ;
Który miał mieƺkánie w grobách / á nie mógł go nikt y łáńcuchámi związáć /
Przeto że on cżęſto będąc pętámi y łáńcuchámi związány / łáńcuchy porwáł / y pętá pokruƺył ; á nie mógł go nikt ukróćić.
A záwƺe we dnie y w nocy ná górách w grobách był / wołájąc y kámieniem śię tłukąc.
Ujrzáwƺy tedy JEzuſá z dáleká / bieżáł y pokłonił mu śię ;
A wołájąc głoſem wielkim / rzekł : Cóż mam z tobą JEzuśie / Synu Bogá nájwyżƺego? Poprzyśięgám ćię przez Bogá / ábyś mię nie trápił.
( Abowiem mu mówił : Wynijdź / duchu niecżyſty! z tego cżłowieká. )
Tedy go pytáł : Co máƺ zá imię? A on odpowiádájąc / rzekł : Imię moje jeſt wojſko : ábowiem nas jeſt wiele.
Y prośił go bárdzo / áby ich nie wygániáł z oney kráiny.
A byłá tám przy górách wielká trzodá świni / która śię páſłá.
Y prośili go oni wƺyſcy dyjábli / mówiąc : Puść nas w te świnie / ábyśmy w nie weƺli.
Y pozwolił im záráz JEzuſ. A wyƺedłƺy oni duchowie niecżyśći / weƺli w one świnie ; y porwáłá śię oná trzodá z przykrá w morze ( á było ich około dwóch tyśięcy / ) y potonęły w morzu.
A oni / którzy świnie páśli / ućiekli / y oznájmili to w mieśćie y we wśiách ; y wyƺli / áby oglądáli to / co śię ſtáło.
Y przyƺli do JEzuſá / y ujrzeli onego / który był opętány / y śiedźiáł oblecżony / będąc przy dobrem bácżeniu ; onego / mówię / w którym było wojſko dyjábłów ; y báli śię.
A ći / którzy to widźieli / opowiedźieli im / co śię dźiáło z onym opętánym / y o świniách.
Tedy go pocżęli prośić / áby odƺedł z gránic ich.
A gdy wſtąpił w łódź / prośił go on / co był opętánym / áby był przy nim.
Lecż mu JEzuſ nie dopuśćił / ále mu rzekł : Idź do domu ſwego / do ſwoich / á oznájmij im / jákoć wielkie rzecży Pán ucżynił / á jáko śię nád tobą zmiłowáł.
Tedy odƺedł / y pocżął opowiádáć w dźieśięćiu miáſtách / jáko mu wielkie rzecży ucżynił JEzuſ ; y dźiwowáli śię wƺyſcy.
A gdy śię záśię JEzuſ przepráwił w łodźi ná drugą ſtronę / zebráł śię do niego wielki lud ; á on był nád morzem.
A oto przyƺedł jeden z przełożonych bóżnicy / imieniem Jáiruſ / á ujrzáwƺy go / przypádł do nóg jego /
Y prośił go wielce / mówiąc : Poniewáż córecżká mojá koná / pójdźże / włóż ná nię ręce / áby byłá uzdrowioná / y będźie żywá. Y poƺedł z nim.
Y ƺedł zá nim lud wielki / y ćiſnęli go.
Tedy niektóra niewiáſtá / która ćierpiáłá płynienie krwi ode dwunáſtu lát /
Y wiele ućierpiáłá od wielu lekárzy / y wynáłożyłá wƺyſtko / co miáłá ; á nic jey nie pomogło / owƺem śię jey tem więcey pogorƺáło :
Uſłyƺáwƺy o JEzuśie / przyƺłá z tyłu między ludem / y dotknęłá śię ƺáty jego ;
Bo mówiłá : Jeſli śię tylko dotknę ƺáty jego / będę uzdrowioná.
A zárázem wyſchło źródło krwi jey / y pocżułá ná ćiele / że uzdrowioná byłá od choroby ſwojey.
A wnet poznáwƺy JEzuſ ſám w ſobie / że z niego moc wyƺłá / obróćił śię do ludu y rzekł : Kto śię dotknął ƺát moich?
Y rzekli mu ucżniowie jego : Widźiƺ / że ćię ten lud ćiśnie / á mówiƺ : Kto śię mnie dotknął?
Y ſpojrzáł w koło / áby ujrzáł tę / która to ucżyniłá :
Ale niewiáſtá oná z bojáźnią y ze drżeniem / wiedząc / co śię przy niej ſtáło / przyſtąpiłá y upádłá przed nim / á powiedźiáłá mu wƺyſtkę práwdę.
Zátem jey on rzekł : Córko! wiárá twojá ćiebie uzdrowiłá / idźże w pokoju / á bądź zdrowá od choroby twojey.
A gdy on jeƺcże mówił / przyƺli ſłudzy od przełożonego bóżnicy / mówiąc : Córká twojá umárłá / cżemuż jeƺcże trudźiƺ náucżyćielá?
Ale JEzuſ ſkoro uſłyƺáł to / co oni mówili / rzekł do przełożonego bóżnicy : Nie bój śię / tylko wierz!
Y nie dopuśćił nikomu iść zá ſobą / tylko Piotrowi / y Jákóbowi / y Jánowi / brátu Jákóbowemu.
A przyƺedł do domu przełożonego bóżnicy / y ujrzáł tám zgiełk / y płácżące y bárdzo nárzekájące.
Wƺedłƺy tedy / rzekł im : Przecż zgiełk cżynićie y płácżećie? nie umárłáć dźiewecżká / ále śpi.
Y náśmiewáli śię z niego. Ale on wygnáwƺy wƺyſtkie / wźiął z ſobą ojcá y mátkę dźiewecżki / y te / którzy przy nim byli / y wƺedł tám / gdźie dźiewecżká leżáłá.
A ująwƺy zá rękę onę dźiewecżkę / rzekł do niej : Tálitá kumi! co śię wykłádá : Dźiewecżko ( tobie mówię ) wſtáń!
A záráz dźiewecżká wſtáłá / y chodźiłá ; ábowiem byłá w dwunáſtym roku. Y zdumieli śię zdumieniem wielkiem.
Tedy im przykázáł wielce / áby tego nikt nie wiedźiáł / y rozkázáł / áby jey dáno jeść.
