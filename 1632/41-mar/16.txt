A gdy minął ſábát / Máryjá Mágdálená / y Máryjá / mátká Jákóbowá / y Sálome / nákupiły wonnych rzecży / áby przyƺedłƺy námázáły go.
A bárdzo ráno pierwƺego dniá po ſábáćie przyƺły do grobu / gdy weƺło ſłońce.
Y mówili do śiebie : Któż nám odwáli kámień ode drzwi grobowych?
( A ſpojrzáwƺy ujrzáły / iż był kámień odwálony ; ) bo był bárdzo wielki.
Y wƺedłƺy w grób / ujrzáły młodzieńcá / śiedzącego ná práwicy / odziánego ƺátą biáłą ; y ulękły śię /
Ale im on rzekł : Nie lękájćie śię ; Jezuſá ƺukáćie onego Názáreńſkiego / który był ukrzyżowány ; wſtáł z mártwych / niemáƺ go tu ; oto miejſce / gdzie go było położono.
Ale idźćie / á powiedzćie ucżniom jego / y Piotrowi / że wáſ uprzedzá do Gálilei ; tám go oglądáćie / jáko wám powiedziáł.
A wyƺedłƺy prędko / ućiekły od grobu : ábowiem zdjęło je drżenie y zdumienie / á nikomu nic nie powiádáły ; bo śię báły.
A Jezuſ / gdy zmártwychwſtáł rániucżko pierwƺego dniá po ſábáćie / ukázáł śię náprzód Máryi Mágdálenie / z której był wygnáł śiedm dyjábłów.
A oná ƺedłƺy / opowiedziáłá tym / co z nim bywáli / którzy śię ſmućili y płákáli.
A oni uſłyƺáwƺy / iż żyje / á iż jeſt widziány od niej / nie wierzyli.
Potem śię też dwom z nich idącym ukázáł w innym kƺtáłćie / gdy ƺli przez pole.
A ći ƺedłƺy opowiedzieli drugim ; y tym nie uwierzyli.
Ná oſtátek śię też onym jedenáſtu weſpół śiedzącym ukázáł / y wyrzucáł im ná ocży niedowiárſtwo ich / y zátwárdzenie ſercá / iż tym / którzy go widzieli wzbudzonego / nie wierzyli.
Y rzekł im : Idąc ná wƺyſtek świát / káżćie Ewángieliją wƺyſtkiemu ſtworzeniu.
Kto uwierzy á ochrzći śię / zbáwion będzie ; ále kto nie uwierzy / będzie potępion.
A známioná tych / co uwierzą / te náśládowáć będą : W imieniu mojem dyjábły wygániáć będą / nowemi językámi mówić będą ;
Węże bráć będą / á choćby co śmiertelnego pili / nie záƺkodzi im ; ná niemocne ręce kłáść będą / á dobrze śię mieć będą.
A ták Pán przeſtáwƺy z nimi mówić / wzięty jeſt do niebá / y uśiádł ná práwicy Bożej.
A oni wyƺedłƺy kázáli wƺędy / á Pán im pomágáł / y ſłowá ich potwierdzáł przez cudá / które cżynili.
