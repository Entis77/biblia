A gdy minął Sábbát / Márya Mágdálená / y Márya <i>mátká</i> Jákubowá / y Sálome / nákupiły wonnych rzecży / áby przyƺedƺy námázáły go.
A bárzo ráno pierwƺego dniá po Sábbáćie / przyƺły do grobu gdy weƺło ſłońce.
Y mówili do śiebie ; Któż nam odwáli kámień ode drzwi grobowych?
( A pojrzawƺy / ujrzáły iż był kámień odwálony ) Bo był bárzo wielki.
Y wƺedƺy w grób / ujrzáły młodźieńcá śiedzącego ná práwicy / odźianego ƺátą białą : y ulękły śię.
Ale im on rzekł : Nie lękajćie śię ; JEzuſá ƺukaćie onego Názáreńſkiego / który był ukrzyżowány : wſtał <i>zmartwych,</i> niemáƺ <i>go</i> tu : oto miejſce gdźie go było położono.
Ale idźćie / á powiedzćie ucżniom jego y Piotrowi / że was uprzedza do Gálilejey : tám go oglądaćie / jáko wam powiedźiał.
A wyƺedƺy prętko / ućiekły od grobu : ábowiem zjęło je drżenie y zdumienie / á nikomu nic nie powiádáły ; bo śię bały.
A JEzus gdy zmartwychwſtał rániucżko pierwƺego <i>dniá</i> po Sábbáćie / ukazał śię naprzód Máryey Mágdálenie / z którey był wygnał śiedmi dyabłów.
<i>A</i> oná ƺedƺy / opowiedźiáłá tym co z nim bywáli / którzy śię ſmućili y płákáli.
A oni uſłyƺawƺy iż żyje / á iż jeſt widźiány od niey / nie wierzyli.
Potym śię też dwiemá z nich idącym ukazał w innym kƺtałćie / gdy ƺli przez pole.
A ći ƺedƺy opowiedźieli drugim : lecż y tym nie uwierzyli.
Náoſtátek śię <i>też</i> onym jedenaśći weſpół śiedzącym ukazał / y wyrzucał im ná ocży niedowiárſtwo ich / y zátwárdzenie ſercá : iż tym / którzy go widźieli wzbudzonego / nie wierzyli.
Y rzekł im : Idąc ná wƺyſtek świát / każćie Ewángelią / wƺyſtkiemu ſtworzeniu.
Kto uwierzy á ochrzći śię / zbáwion będźie : ále kto nie uwierzy / będźie potępion.
A známioná tych co uwierzą / te náśládowáć będą : w Imieniu mojim dyabły wyganiáć będą ; nowymi językámi mówić będą ;
Węże bráć będą ; á choćby co śmiertelnego pili / nie záƺkodźi im ; ná niemocne ręce kłáść będą á dobrze śię mieć będą.
A ták PAn przeſtawƺy z nimi mówić / wźięty jeſt wzgórę do niebá / y uśiadł ná práwicy Bożey.
A oni wyƺedƺy / kazáli wƺędy : á PAn im pomagał / y ſłów <i>ich</i> potwirdzał przez cudá / które cżynili.
