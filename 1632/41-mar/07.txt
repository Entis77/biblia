Tedy śię zgromádźili do niego Fáryzeuƺowie / y niektórzy z náucżonych w Piśmie / którzy byli przyƺli z Jeruzálemu ;
A ujrzáwƺy niektóre z ucżniów jego / że poſpolitemi rękomá / ( to jeſt / nieumytemi ) jedli chleb / gánili to.
Abowiem Fáryzeuƺowie y wƺyſcy Żydźi nie jedzą / jeſliby pilnie rąk nie umyli / trzymájąc uſtáwę ſtárƺych.
Y z rynku przyƺedłƺy / jeſliby śię nie umyli / nie jedzą ; y innych rzecży wiele jeſt / które przyjęli ku trzymániu / jáko umywánie kubków / konewek / y miednic / y ſtołów.
Potem go pytáli Fáryzeuƺowie y náucżeni w Piśmie : Przecż ucżniowie twoi nie chodzą według podániá ſtárƺych / ále nieumytemi rękomá chleb jedzą?
Tedy on odpowiádájąc / rzekł im : Dobrze Izájáƺ o was obłudnikách prorokowáł / jáko nápiſáno : Lud ten cżći mię wárgámi / ále ſerce ich dáleko jeſt odemnie.
Lecż próżno mię cżcżą náucżájąc náuk y uſtáw ludzkich.
Abowiem wy opuśćiwƺy przykázániá Boże / trzymáćie uſtáwy ludzkie / umywánie konewek y kubków / y wiele innych tákich tym podobnych rzecży cżynićie.
Mówił im też : Wy cżyśćie znośićie przykázánie Boże / ábyśćie uſtáwy wáƺe záchowáli.
Bo Mojżeƺ rzekł : Cżćij ojcá twego y mátkę twoję ; á kto złorzecży ojcu álbo mátce / niech śmierćią umrze.
Ale wy mówićie : Jeſliby cżłowiek rzekł ojcu álbo mátce : Korbán / ( co jeſt dár / ) którykolwiek będźie odemnie / tobie pożytecżny będźie / bez winy będźie ;
Y nie dopuśćićie mu nic więcey cżynić ojcu ſwemu álbo mátce ſwojey /
Wniwecż obrácájąc ſłowo Boże uſtáwą wáƺą / którąśćie uſtáwili ; y wiele innych rzecży tym podobnych cżynićie.
A zwołáwƺy wƺyſtkiego ludu / mówił im : Słuchájćie mię wƺyſcy / á zrozumiejćie!
Niemáƺ nic z rzecży zewnętrznych / które wchodzą w cżłowieká / coby go mogło pokáláć ; ále to / co pochodźi z niego / to jeſt / co pokálá cżłowieká.
Jeſli kto má uƺy ku ſłuchániu / niecháj ſłuchá!
A gdy wƺedł w dom od onego ludu / pytáli go ucżniowie jego o to podobieńſtwo.
Tedy im rzekł : Tákże y wy bezrozumni jeſteśćie? Azáż nie rozumiećie / iż wƺyſtko / co zewnątrz wchodźi w cżłowieká / nie może go pokáláć?
Abowiem nie wchodźi w ſerce jego / ále w brzuch / y do wychodu wychodźi / cżyƺcżąc wƺyſtkie pokármy.
Y powiedźiáł / że co pochodźi z cżłowieká / to pokálá cżłowieká.
Bo z wnętrznośći ſercá ludzkiego pochodzą złe myśli / cudzołóſtwá / wƺetecżeńſtwá / mężobójſtwá /
Krádźieże / łákomſtwá / złośći / zdrádá / niewſtyd / oko złe / bluźnierſtwo / pychá / głupſtwo.
Wƺyſtkie te złe rzecży pochodzą z wnętrznośći / y pokáláją cżłowieká.
A ſtámtąd wſtáwƺy / poƺedł ná gránice Tyru y Sydonu / á wƺedłƺy w dom / niechćiáł / áby kto wiedźiáł ; lecż śię utáić nie mógł.
Abowiem uſłyƺáwƺy o nim niewiáſtá / którey córecżká miáłá duchá niecżyſtego / przyƺłá y przypádłá do nóg jego /
( A tá niewiáſtá byłá Grecká / rodem z Syrofenicyi ) y prośiłá go / áby dyjábłá wygnáł z córki jey.
Ale jey JEzuſ rzekł : Niech śię pierwej dźieći náſycą ; boć nie jeſt dobrá / bráć chleb dźiećiom y miotáć ƺcżeniętom.
A oná odpowiedźiáłá y rzekłá mu : Ták jeſt / Pánie! Wƺákże y ƺcżeniętá jádáją pod ſtołem z odrobin dźiećinnych.
Y rzekł do niej : Dla tey mowy idź / wyƺedł dyjábeł z córki twojey.
A gdy oná odeƺłá do domu ſwego / ználázłá / iż dyjábeł wyƺedł á córká leżáłá ná łożu.
A wyƺedłƺy záś z gránic Tyrſkich y Sydońſkich / przyƺedł nád morze Gálilejſkie / pośrodkiem gránic dźieśięćiu miáſt.
Y przywiedli mu głuchego y z ćiężkośćią mówiącego / á prośili go / áby náń rękę włożył.
A wźiąwƺy go Pán od ludu oſobno / włożył pálce ſwoje w uƺy jego / á plunąwƺy dotknął śię języká jego ;
A wejrzáwƺy w niebo / weſtchnął y rzekł do niego : Efátá! to jeſt / otwórz śię.
Y wnet śię otworzyły uƺy jego / y rozwiązáłá śię związká języká jego / y wymáwiáł dobrze.
Tedy im zákázáł / áby tego nikomu nie powiádáli.
Ale cżem on im bárdźiej zákázywáł / tem oni to bárdźiej rozgłáƺáli / y náder śię bárdzo zdumiewáli / mówiąc : Dobrze wƺyſtko ucżynił ; bo cżyni / iż głuśi ſłyƺą y niemi mówią.
