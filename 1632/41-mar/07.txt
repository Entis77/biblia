Tedy śię zgromádźili do niego Fáryzeuƺowie / y niektórzy z Náucżonych w piśmie którzy byli przyƺli z Jeruzalem.
A ujrżawƺy niektóre z ucżniów jego / że poſpolitymi rękomá / ( to jeſt / nie umytymi ) jedli chleb / gánili to.
Abowiem Fáryzeuƺowie y wƺyſcy Żydźi nie jedzą / jeſliby pilnie ręku nie umyli ; trzymájąc uſtáwę ſtárƺych.
Y z rynku <i>przyƺedƺy,</i> jeſliby śię nie umyli / nie jedzą : Y innych rzecży wiele jeſt / które przyjęli ku trzymániu / <i>jáko</i> umywánie kubków / konewek / y miednic / y ſtołów.
Potym go pytáli Fáryzeuƺowie y Náucżeni w piśmie : Przecż ucżniowie twoji nie chodzą według podánia ſtárƺych / ále nieumytymi rękomá chleb jedzą?
Tedy on odpowiádájąc / rzekł im : Dobrze Izájaƺ o was obłudnikách prorokował / jáko nápiſano : Lud ten cżći mię wárgámi / ále ſerce ich dáleko jeſt ode mnie.
Lecż prózno mię cżcżą / náucżájąc náuk y uſtaw ludzkich.
Abowiem <i>wy</i> opuśćiwƺy przykazánia Boże / trzymaćie uſtáwy ludzkie / umywánie konewek y kubków ; y wiele innych tákich / tym podobnych rzecży / cżynićie.
Mówił im też : <i>Wy</i> cżyśćie znośićie przykazánie Boże / ábyśćie uſtáwy wáƺe záchowáli.
Bo Mojzeƺ rzekł : Cżći Ojcá twego y Mátkę twoję : A kto złorzecży ojcu álbo mátce / niech śmierćią umrze.
Ale wy mówićie : Jeſliby cżłowiek rzekł ojcu álbo mátce / Korban / ( co jeſt / Dar / ) którykolwiek będźie ode mnie / tobie pożytecżny będźie ; <i>bez winy będźie</i>.
Y nie dopuśćićie mu nic więcey cżynić ojcu ſwemu álbo mátce ſwojey :
Wniwecż obracájąc ſłowo Boże uſtáwą wáƺą którąśćie uſtáwili : y wiele innych rzecży tym podobnych cżynićie.
A zwoławƺy wƺyſtkiego ludu / mówił im : Słuchajćie mię wƺyſcy / á zrozumiejćie.
Niemáƺ nic z rzecży zewnętrznych które wchodzą w cżłowieká / coby go mogło pokaláć : ále to co pochodźi z niego / to jeſt co pokala cżłowieká.
Jeſli kto ma uƺy ku ſłuchániu / niechaj ſłucha.
A gdy wƺedł w dom od onego ludu / pytáli go ucżniowie jego <i>o to</i> podobieńſtwo.
Tedy im rzekł : Tákże y wy bezrozumni jeſteśćie? Azaż nie rozumiećie / iż wƺyſtko co zewnątrz wchodźi w cżłowieká / nie może go pokaláć?
Abowiem nie wchodźi w ſerce jego / ále w brzuch / y do wychodu wychodźi cżyśćiąc wƺyſtkie pokármy.
Y powiedźiał / że co pochodźi z cżłowieká / to pokala cżłowieká.
Bo z wnętrznośći ſercá ludzkiego pochodzą złe myśli / cudzołóſtwá / wƺetecżeńſtwá / mężobójſtwá :
Kradźieſtwá / łákomſtwá / złośći / zdrádá / niewſtyd / oko złe / bluźnierſtwo / pychá / głupſtwo.
Wƺyſtkie te złe rzecży / pochodzą z wnętrznośći / y pokaláją cżłowieká.
A z támtąd wſtawƺy / poƺedł ná gránice Tyru y Sydonu : á wƺedƺy w dom / niechćiał áby kto wiedźiał : lecż śię utájić nie mógł.
Abowiem uſłyƺawƺy o nim niewiáſtá / którey córecżká miáłá duchá niecżyſtego / przyƺłá y przypádłá do nóg jego /
( A tá niewiáſtá byłá Grecka / rodem z Syrofenicyey ) y prośiłá go / áby dyabłá wygnał z córki jey.
Ale jey JEzus rzekł : Niech śię pierwey dźieći náſycą : boć nie jeſt dobra bráć chleb dźiećiom / y miotáć ƺcżeniętom.
A oná odpowiedźiáłá / y rzekłá mu : Ták jeſt PAnie! Wƺákże y ƺcżeniętá jadáją pod ſtołem z odrobin dźiećinnych.
Y rzekł do niey : Dla tey mowy idź / wyƺedł dyabeł z córki twojey.
A gdy oná odeƺłá do domu ſwego / ználázłá yż dyabeł wyƺedł / á córká leżáłá ná łożu.
A wyƺedƺy záś z gránic Tyrſkich y Sydońſkich / przyƺedł nád morze Gálilejſkie / pośrzodkiem gránic Dźieśiąći miaſt.
Y przywiedli mu głuchego y z ćięƺkośćią mówiącego / á prośili go áby nań rękę włożył.
A wźiąwƺy go PAN od ludu oſobno / włożył pálce ſwoje w uƺy jego ; á plunąwƺy / dotknął śię języká jego.
A wejrzawƺy w niebo / weſtchnął y rzekł do niego : Efátá / to jeſt / otwórz śię.
Y wnet śię otworzyły uƺy jego / y rozwiązáłá śię związká języká jego / y wymawiał dobrze.
Tedy im zákazał áby <i>tego</i> nikomu nie powiádáli.
Ale cżym on im bárźiey zákázował / tym oni to bárźiey rozgłaƺáli. Y náder śię bárzo zdumiewáli mówiąc ; Dobrze wƺyſtko ucżynił : <i>bo</i> cżyni yż głuƺy ſłyƺą / y niemi mówią.
