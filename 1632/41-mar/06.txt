A wyƺedzy z támtąd / przyƺedł do Ojcżyzny ſwojey / y ƺli zá nim ucżniowie jego.
A gdy przyƺedł Sábbát / pocżął ucżyć w bóżnicy / á wiele ich ſłuchájąc / zdumiewáli śię y mówili : Z kądże temu to wƺyſtko? á co to zá mądrość która mu jeſt dána? że śię y tákie mocy dźieją przez ręce jego?
Izali ten nie jeſt ćieślá / ſyn Máryey / y brát Jákubá / y Jozeſá / y Judáſá / y Symoná? Azaż tu niemáƺ y śióſtr jego u nas? Y gorƺyli śię z niego.
Ale JEzus rzekł do nich : Nie jeſtći Prorok beze cżći / chybá w Ojcżyźnie ſwojey / á miedzy pokrewnymi / y w domu ſwojim.
Y nie mógł tám ucżynić żadnego cudu ; oprócż iż niektóre chore / wkładájąc ná nie ręce uzdrowił.
A dźiwował śię niedowiárſtwu ich : y obchodźił okolicżnie miáſtecżká náucżájąc.
Tedy zwoławƺy do śiebie onych dwunaści / pocżął je po dwu rozſyłáć / y dał im moc nád duchámi niecżyſtymi.
Y rozkazał im áby nic nie bráli ná drogę / jedno tylko laſkę ; áni tájſtry / áni chlebá / áni w trzos pieniędzy.
Ale żeby śię obuli w trzewiki / á nie obłocżyli dwu ſukien.
Zátym mówił do nich : Gdźiekolwiek wnidźiećie w dom / tám zoſtáńćie / pókibyśćie z támtąd nie wyƺli.
A którzyby was kolwiek nie przyjęli / áni was ſłucháli / wyƺedƺy z támtąd otrząśnićie proch z nóg wáƺych ná świádectwo im : Záprawdę powiádam wam ; Lżey będźie Sodomie y Gomorze w dźień ſądny / niż miáſtu onemu.
Tedy wyƺedƺy kazáli áby <i>ludźie</i> pokutowáli.
Y wyganiáli wiele dyabłów : y wiele chorych olejkiem mázáli / y uzdrawiáli je.
A uſłyƺał o tym król Herod / bo śię ( imię jego ſtáło rozſławione ) y rzekł : Jan Chrzćićiel zmartwychwſtał ; dla tego śię cudá dźieją przezeń.
<i>A</i> drudzy mówili ; Eliaƺ <i>to</i> jeſt : Drudzy záś mówili ; Prorok to jeſt / álbo jáko jeden z onych Proroków.
Co uſłyƺawƺy Herod / rzekł : Ten jeſt Jan / któregom ja zćiął / on zmartwychwſtał.
Abowiem tenże Herod poſławƺy pojimał Janá / y wſádźił go do więźienia dla Herodyády żony Filipá brátá ſwego / iż ją był pojął zá żonę.
Bo Jan mówił Herodowi : Niegodźić śię mieć żony brátá twego.
A Herodyás cżyháłá nań ; y chćiáłá go zábić / ále nie mogłá.
Abowiem Herod obawiał śię Janá / wiedząc iż był mężem ſpráwiedliwym y świętym : y oglądał śię nań / y ſłuchájąc go / wiele cżynił / y rad go ſłuchał.
A gdy przyƺedł dźień ſpoſobny / którego Herod obchodząc pámiątkę národzenia ſwego / wiecżerzą ſpráwił ná Kśiążętá ſwoje / y ná Hetmány / y ná przedniejƺe z Gálilejey.
A gdy weƺłá córká oney Herodyády y táńcowáłá / y podobáłá śię Herodowi y ſpół-śiedzącym / rzekł Król do dźiewecżki : Proś mię o co chceƺ / á damći.
Y przyśiągł jey : O cokolwiekbyś mię prośiłá damći / áż do połowice króleſtwá mego.
Oná tedy wyƺedƺy / rzekłá mátce ſwojey : O co mam prośić? A oná rzekłá ; O głowę Janá Chrzćićielá.
A ták <i>oná</i> záraz wƺedƺy prędko do Królá / prośiłá mówiąc : Chcę ábyś mi teraz dał ná miśie głowę Janá Chrzćićielá.
Y záſmućił śię Król bárzo ; wƺákże dla przyśięgi / y dla ſpół-śiedzących nie chćiał jey odmówić.
A zárázem poſławƺy Król kátá / rozkazał przynieść głowę jego.
A on poƺedƺy zćiął go w więźieniu / y przynióſł głowę jego ná miśie / á dał ją dźiewecżce ; á dźiewecżká dáłá ją mátce ſwojey.
Co gdy uſłyƺeli ucżniowie jego / przyƺli y wźięli ćiáło jego / y położyli je w grobie.
A Apoſtołowie zƺedƺy śię do JEzuſá / opowiedźieli mu wƺyſtko / y co cżynili / y cżego ucżyli.
Y rzekł im : Pódźćie wy ſámi oſobno ná miejſce puſte / á odpocżyńćie trochę : Bo ich wiele było / co przychodźili y odchodźili / ták iż nie mieli wolnego cżáſu żeby jedli.
Y odjácháli w łodźi ná miejſce puſte oſobno.
A widząc je lud że odjeżdżáli / poznáło go wiele ich / y zbieżeli śię tám pieƺo ze wƺyſtkich miaſt / y poprzedźili je / y zgromádźili śię do niego.
A wyƺedƺy JEzus ujrzał wielki lud / y użalił śię ich / iż byli jáko owce nie májące páſterzá / y pocżął ich náucżáć wiele rzecży.
A gdy już cżás mijał / przyſtąpiwƺy do niego ucżniowie jego / rzekli : <i>To</i> miejſce jeſt puſte / á już cżás mija.
Rozpuść je / áby poƺedƺy do okolicżnych wśi y miáſtecżek / nákupili ſobie chlebá ; bo nie máją coby jedli.
A on odpowiádájąc / rzekł im : Dajćie wy im jeść. Y rzekli mu. Szedƺy kupimy zá dwie śćie groƺy chlebá / á damy im jeść?
A on im rzekł : Wieleż chlebá maćie? Idźćie / á dowiedzćie śię. A oni dowiedźiawƺy śię / powiedźieli : Pięćioro / y dwie rybie.
Tedy im kazał wƺyſtkie gromádámi poſádźić / ná źieloney trawie.
Y uśiedli rząd podle rządu / tu po ſtu / tu záś po pięćidźieśiąt.
A wźiąwƺy one pięćioro chlebá / y one dwie rybie / wejrzawƺy w niebo / błogoſłáwił. Y połamał one chleby / y dawał ucżniom ſwojim / áby kłádli przed nie ; y one dwie rybie rozdźielił miedzy wƺyſtkie.
Y jedli wƺyſcy / y náſyceni byli.
Y zebráli ułomków dwánaśćie koƺów pełnych / y z onych ryb.
A było tych którzy jedli chleby / około piąći tyśięcy mężów.
Y wnet przymuśił ucżnie ſwoje / áby wſtąpili w łódź / y uprzedźili go ná drugą ſtronę ku Betſájdźie / áżby on rozpuśćił lud.
A odpráwiwƺy je / odƺedł ná górę / áby śię modlił.
A gdy był wiecżór / byłá łódź w pośrzód morzá / á on ſam <i>był</i> ná źiemi.
Y widźiał że śię zprácowáli wioſłámi robiąc ; ( bo wiátr mieli przećiwny ) á ták około cżwartey ſtráży nocney przyƺedł do nich chodząc po morzu / y chćiał je wyminąć.
Ale oni ujrzawƺy go chodzącego po morzu / mniemáli żeby byłá obłudá / y krzyknęli :
( Bo go wƺyſcy widźieli / y wylękli śię ) Ale záraz przemówił do nich / y rzekł im : Ufájćie / Jam jeſt ; nie bójćie śię.
Y wſtąpił do nich w łódź / y ućiƺył śię wiátr : á oni śię ſámi w ſobie náder zdumiewáli y dźiwowáli.
Bo nie zrozumieli ſtrony chlebów : gdyż ſerce ich było zdrętwiáło.
A przepráwiwƺy śię / przyƺli do źiemie Genezáret / y przybili śię <i>do brzegu</i>.
A gdy wyƺli z łodźi / záraz ći co go poználi /
Obieżawƺy wƺyſtkę onę okolicżną krájinę / pocżęli nośić ná łożách te / którzy śię źle mieli / gdźiekolwiek uſłyƺeli <i>o nim</i> że tám jeſt.
A gdźiekolwiek on wƺedł do miáſtecżek / álbo do miaſt / álbo do wśi / kłádli niemocne po ulicách : y prośili go / áby śię tylko dotykáli podołká ƺáty jego : á ile śię go ich dotknęli / byli uzdrowieni.
