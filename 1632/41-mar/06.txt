A wyƺedłƺy ſtámtąd przyƺedł do ojcżyzny ſwojej / y ƺli zá nim ucżniowie jego.
A gdy przyƺedł ſábát / pocżął ucżyć w bóżnicy / á wiele ich ſłuchájąc / zdumiewáli śię y mówili : Skądże temu to wƺyſtko? á co to zá mądrość / którá mu jeſt dáná / że śię y tákie mocy dzieją przez ręce jego?
Izáli ten nie jeſt ćieślá / ſyn Máryi / y brát Jákóbá / y Jozeſá / y Judáſá / y Szymoná? Azáż tu niemáƺ y śióſtr jego u náſ? Y gorƺyli śię z niego.
Ale Jezuſ rzekł do nich : Nie jeſtći prorok beze cżći / chybá w ojcżyźnie ſwojej / á między pokrewnymi / y w domu ſwoim.
Y nie mógł tám ucżynić żádnego cudu / oprócż iż niektóre chore / wkłádájąc ná nie ręce / uzdrowił.
A dziwowáł śię niedowiárſtwu ich / y obchodził okolicżne miáſtecżká / náucżájąc.
Tedy zwołáwƺy do śiebie onych dwunáſtu / pocżął je po dwóch rozſyłáć / y dáł im moc nád duchámi niecżyſtymi.
Y rozkázáł im / áby nic nie bráli ná drogę / jedno tylko láſkę : áni táiſtry / áni chlebá / áni w trzoſ pieniędzy ;
Ale żeby śię obuli w trzewiki / á nie obłocżyli dwóch ſukien.
Zátem mówił do nich : Gdziekolwiek wnijdziećie w dom / tám zoſtáńćie / pókibyśćie ſtámtąd nie wyƺli.
A którzyby wáſ kolwiek nie przyjęli / áni wáſ ſłucháli / wyƺedłƺy ſtámtąd / otrząśnijćie proch z nóg wáƺych ná świádectwo im ; zápráwdę powiádám wám : Lżej będzie Sodomie y Gomorze w dzień ſądny / niż miáſtu onemu.
Tedy wyƺedłƺy kázáli / áby ludzie pokutowáli.
Y wygániáli wiele dyjábłów / y wiele chorych olejkiem mázáli y uzdráwiáli je.
A uſłyƺáł o tem król Herod / ( bo śię imię jego ſtáło rozſłáwione / ) y rzekł : Ján Chrzćićiel zmártwychwſtáł / dlátego śię cudá dzieją przezeń.
A drudzy mówili : Elijáƺ to jeſt ; drudzy záś mówili : Prorok to jeſt / álbo jáko jeden z onych proroków.
Co uſłyƺáwƺy Herod / rzekł : Ten jeſt Ján / któregom já śćiął / on zmártwychwſtáł.
Abowiem tenże Herod poſłáwƺy pojmáł Jáná / y wſádził go do więzieniá dlá Herodyjády / żony Filipá / brátá ſwego / iż ją był pojął zá żonę.
Bo Ján mówił Herodowi : Nie godzi śię mieć żony brátá twego.
A Herodyjáſ cżyháłá náń / y chćiáłá go zábić / ále nie mogłá ;
Abowiem Herod obáwiáł śię Jáná / wiedząc / iż był mężem ſpráwiedliwym y świętym ; y oglądáł śię náń / y ſłuchájąc go / wiele cżynił y rád go ſłucháł.
A gdy przyƺedł dzień ſpoſobny / którego Herod / obchodząc pámiątkę národzeniá ſwego / wiecżerzą ſpráwił ná kśiążętá ſwoje y ná hetmány y ná przedniejƺe z Gálilei ;
A gdy weƺłá córká onej Herodyjády y táńcowáłá / y podobáłá śię Herodowi y ſpółśiedzącym / rzekł król do dziewecżki : Proś mię o co chceƺ / á dám ći.
Y przyśiągł jej : O cokolwiekbyś mię prośiłá / dám ći / áż do połowy króleſtwá mego.
Oná tedy wyƺedłƺy / rzekłá mátce ſwojej : O co mám prośić? A oná rzekłá : O głowę Jáná Chrzćićielá.
A ták oná záráz wƺedłƺy prędko do królá / prośiłá mówiąc : Chcę / ábyś mi teráz dáł ná miśie głowę Jáná Chrzćićielá.
Y záſmućił śię król bárdzo / wƺákże dlá przyśięgi y dlá ſpółśiedzących nie chćiáł jej odmówić.
A zárázem poſłáwƺy król kátá / rozkázáł przynieść głowę jego.
A on poƺedłƺy śćiął go w więzieniu / y przynióſł głowę jego ná miśie / á dáł ją dziewecżce / á dziewecżká dáłá ją mátce ſwojej.
Co gdy uſłyƺeli ucżniowie jego / przyƺli y wzięli ćiáło jego / y położyli je w grobie.
A Apoſtołowie zƺedłƺy śię do Jezuſá / opowiedzieli mu wƺyſtko / y co cżynili / y cżego ucżyli.
Y rzekł im : Pójdźćie wy ſámi oſobno ná miejſce puſte / á odpocżnijćie trochę ; bo ich wiele było / co przychodzili / y odchodzili / ták iż nie mieli wolnego cżáſu / żeby jedli.
Y odjecháli w łodzi ná miejſce puſte oſobno.
A widząc je lud / że odjeżdżáli / poznáło go wiele ich / y zbieżeli śię tám pieƺo ze wƺyſtkich miáſt / y poprzedzili je / y zgromádzili śię do niego.
A wyƺedłƺy Jezuſ ujrzáł wielki lud / y użálił śię ich / iż byli jáko owce nie májące páſterzá / y pocżął ich náucżáć wiele rzecży.
A gdy już cżáſ mijáł / przyſtąpiwƺy do niego ucżniowie jego / rzekli : To miejſce jeſt puſte / á już cżáſ mijá.
Rozpuść je / áby poƺedłƺy do okolicżnych wśi y miáſtecżek / nákupili ſobie chlebá ; bo nie máją / coby jedli.
A on odpowiádájąc / rzekł im : Dájćie wy im jeść. Y rzekli mu : Szedłƺy kupimy zá dwieśćie groƺy chlebá / á dámy im jeść?
A on im rzekł : Wieleż chlebá máćie? Idźćie / á dowiedzćie śię. A oni dowiedziáwƺy śię / powiedzieli : Pięćioro / y dwie ryby.
Tedy im kázáł wƺyſtkie gromádámi poſádzić ná zielonej tráwie.
Y uśiedli rząd podle rządu / tu po ſtu / tu záś po pięćdzieśiąt.
A wziąwƺy one pięć chlebów / y one dwie ryby / wejrzáwƺy w niebo / błogoſłáwił. Y połámáł one chleby y dáwáł ucżniom ſwoim / áby kłádli przed nie ; y one dwie ryby rozdzielił między wƺyſtkie.
Y jedli wƺyſcy / y náſyceni byli.
Y zebráli ułomków / dwánáśćie koƺów pełnych / y z onych ryb.
A było tych / którzy jedli chleby / około pięćiu tyśięcy mężów.
Y wnet przymuśił ucżnie ſwoje / áby wſtąpili w łódź / y uprzedzili go ná drugą ſtronę ku Betſáidzie / áżby on rozpuśćił lud.
A odpráwiwƺy je / odƺedł ná górę / áby śię modlił.
A gdy był wiecżór / byłá łódź w pośród morzá / á on ſám był ná ziemi.
Y widziáł / że śię ſprácowáli / wioſłámi robiąc ; ( bo wiátr mieli przećiwny / ) á ták około cżwártej ſtráży nocnej przyƺedł do nich / chodząc po morzu / y chćiáł je wyminąć.
Ale oni ujrzáwƺy go chodzącego po morzu / mniemáli / żeby byłá obłudá / y krzyknęli :
( Bo go wƺyſcy widzieli / y wylękli śię. ) Ale záráz przemówił do nich / y rzekł im : Ufájćie / jám jeſt ; nie bójćie śię!
Y wſtąpił do nich w łódź / y ućiƺył śię wiátr ; á oni śię ſámi w ſobie náder zdumiewáli y dziwowáli.
Bo nie zrozumieli z ſtrony chlebów / gdyż ſerce ich było zdrętwiáło.
A przepráwiwƺy śię przyƺli do ziemi Gienezáret y przybili śię do brzegu.
A gdy wyƺli z łodzi / záráz ći / co go poználi /
Obieżáwƺy wƺyſtkę onę okolicżną kráinę / pocżęli nośić ná łożách tych / którzy śię źle mieli / gdziekolwiek uſłyƺeli o nim / że tám jeſt.
A gdziekolwiek on wƺedł do miáſtecżek / álbo do miáſt / álbo do wśi / kłádli niemocne po ulicách / y prośili go / áby śię tylko dotykáli podołká ƺáty jego ; á ile śię go ich dotknęli / byli uzdrowieni.
