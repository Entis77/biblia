Tedy pocżął do nich mówić w podobieńſtwách : Cżłowiek jeden náſádźił winnicę / y ogrodźił ją płotem / y wykopáł práſę / y zbudowáł wieżę / y nájął ją winiárzom / y odjecháł precż.
Y poſłáł / gdy tego był cżáſ / ſługę do winiárzy / áby od winiárzy odebráł pożytki oney winnicy.
Lecż oni pojmáwƺy go / ubili / y odeſłáli próżnego.
Y záśię poſłáł do nich ſługę drugiego / którego też oni ukámionowáwƺy / głowę mu zránili / y odeſłáli obelżonego.
Y záśię poſłáł inƺego ſługę ; ále y tego zábili / y wiele innych / z których jedne ubili / á drugie pozábijáli.
A májąc jeƺcże jednegoż ſwego miłego ſyná / poſłáł ná oſtátek do nich y tego / mówiąc : Wżdyć śię będą wſtydźili ſyná mego.
Ale oni winiárze rzekli między ſobą : Tenći jeſt dźiedźic ; pójdźćie / zábijmy go / á będźie náƺe dźiedźictwo.
Y wźiąwƺy go zábili / á wyrzućili precż z winnicy.
Cóż tedy ucżyni pán oney winnicy? Przyjdźie á potráći one winiárze / y dá winnicę innym.
Izáliśćie nie cżytáli tego piſmá : Kámień / który odrzućili budujący / ten śię ſtáł głową węgielną?
Od Pánáć śię to ſtáło / y jeſt dźiwne w ocżách náƺych.
Stáráli śię tedy / jákoby go pojmáć / ále śię ludu báli ; bo poználi / iż przećiwko nim ono podobieńſtwo powiedźiáł. Y zániecháwƺy go / odeƺli.
Potem poſłáli do niego niektóre z Fáryzeuƺów y z Herodyjánów / áby go uśidlili w mowie.
A ták oni przyƺedłƺy rzekli mu : Náucżyćielu! wiemy / żeś jeſt práwdźiwy / á niedbáƺ ná nikogo ; ábowiem nie pátrzyƺ ná oſobę ludzką / ále w práwdźie drogi Bożej ucżyƺ ; godźiż śię dáć cżynƺ ceſárzowi / cżyli nie? Mámyż go dáć / cżyli nie dáć?
A on poznáwƺy obłudę ich / rzekł im : Cżemuż mię kuśićie? Przynieśćie mi groƺ / ábym go oglądáł.
Tedy mu oni przynieśli ; á on im rzekł : Cżyjże to jeſt obráz y nápiſ? A oni mu powiedźieli : Ceſárſki.
Y odpowiádájąc JEzuſ / rzekł im : Oddáwájćież tedy / co jeſt ceſárſkiego / ceſárzowi / á co jeſt Bożego / Bogu. Y dźiwowáli mu śię.
Y przyƺli do niego Sáduceuƺowie / którzy mówią / iż niemáƺ zmartwychwſtániá / y pytáli go mówiąc :
Náucżyćielu! Mojżeƺ nám nápiſáł / iż jeſliby cżyj brát umárł / y zoſtáwił żonę / á dźiátekby nie zoſtáwił / áby brát jego pojął żonę jego / á wzbudźił náśienie brátu ſwemu.
Było tedy śiedm bráći : á pierwƺy pojąwƺy żonę umárł / y nie zoſtáwił náśieniá ;
A drugi pojąwƺy ją / umárł / lecż y ten nie zoſtáwił náśieniá ; tákże y trzeći.
A ták ją pojęło onych śiedm bráći / á nie zoſtáwili náśieniá. Ná oſtátek po wƺyſtkich umárłá y oná niewiáſtá.
Przetoż przy zmartwychwſtániu gdy powſtáną / któregoż z nich będźie żoną? bo śiedm ich mieli ją zá żonę.
Ná to JEzuſ odpowiádájąc rzekł im : Záż nie dlátego błądźićie / iżeśćie nie powiádomi Piſmá áni mocy Bożej?
Abowiem gdy zmartwychwſtáną / áni śię żenią / áni zá mąż wydáwáją ; ále ſą jáko Aniołowie w niebieśiech.
A o umárłych / iż będą wzbudzeni / nie cżytáliśćież w kśięgách Mojżeƺowych / jáko Bóg do niego ze krzá mówił / y rzekł : Jám jeſt Bóg Abráhámá / Bóg Izááká / y Bóg Jákóbá?
Bóg nie jeſtći Bogiem umárłych / ále Bogiem żywych : przetoż wy bárdzo błądźićie.
A przyſtąpiwƺy jeden z náucżonych w Piśmie / ſłyƺąc / że z ſobą gádáli / á widząc / że im dobrze odpowiedźiáł / ſpytáł go : Które jeſt nájpierwƺe ze wƺyſtkich przykázánie?
A JEzuſ mu odpowiedźiáł : Nájpierwƺe ze wƺyſtkich przykázánie jeſt : Słucháj / Izráelu! Pán / Bóg náƺ / Pán jeden jeſt.
Przetoż będźieƺ miłowáł Páná / Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiej duƺy twojey / y ze wƺyſtkiej myśli twojey / y ze wƺyſtkiej śiły twojey ; toć jeſt pierwƺe przykázánie.
A wtóre temu podobne to jeſt : Będźieƺ miłowáł bliźniego twego / jáko ſámego śiebie. Więkƺego przykázániá innego nád to niemáƺ.
Tedy mu rzekł on náucżony w Piśmie : Náucżyćielu! zápráwdę dobrześ powiedźiáł / iż jeden jeſt Bóg / á niemáƺ inƺego oprócż niego.
Y miłowáć go ze wƺyſtkiego ſercá y ze wƺyſtkiej myśli y ze wƺyſtkiej duƺy y ze wƺyſtkiej śiły / á miłowáć bliźniego jáko ſámego śiebie / więcey jeſt nád wƺyſtkie cáłopáleniá y ofiáry.
A widząc JEzuſ / że on mądrze odpowiedźiáł / rzekł mu : Nie dálekoś jeſt od króleſtwá Bożego. Y nie śmiáł go nikt dálej pytáć.
Tedy JEzuſ odpowiádájąc rzekł / gdy ucżył w kośćiele : Jákoż mówią náucżeni w Piśmie / iż CHryſtuſ jeſt Syn Dawidá?
Bo ſám Dawid przez Duchá Świętego powiedźiáł : Rzekł Pán Pánu memu / śiądź po práwicy mojey / áż położę nieprzyjáćioły twoje podnóżkiem nóg twoich.
Poniewáż go ſám Dawid názywá Pánem / jákoż tedy jeſt ſynem jego? á wielki lud rád go ſłucháł.
Y mówił do nich w náuce ſwojey : Strzeżćie śię náucżonych w Piśmie / którzy chcą w długich ƺátách chodźić / á być pozdráwiáni ná rynkách ;
Y ná pierwƺych ſtołkách śiádáć w bóżnicách / y pierwƺe miejſcá mieć ná wiecżerzách ;
Którzy pożeráją domy wdów / á to pod pokrywką długich modlitw ; ćić odnioſą ćiężƺy ſąd.
A JEzuſ śiedząc przećiwko ſkárbnicy / przypátrywáł śię / jáko lud rzucáł pieniądze do ſkárbnicy / y jáko wiele bogácżów wiele rzucáło.
Y przyƺedłƺy jedná wdowá ubogá / wrzućiłá dwá drobne pieniążki / co cżyni kwártnik.
Tedy zwołáwƺy ucżniów ſwoich rzekł im : Zápráwdę wám powiádám / że tá ubogá wdowá więcey wrzućiłá / niżeli ći wƺyſcy / którzy rzucáli do ſkárbnicy.
Abowiem ći wƺyſcy z tego / co im zbywáło / rzucáli ; ále tá z ubóſtwá ſwego wƺyſtko / co miáłá / wrzućiłá / wƺyſtkę żywność ſwoję.
