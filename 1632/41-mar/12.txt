Tedy pocżął do nich mówić w podobieńſtwách : Cżłowiek <i>jeden</i> náſádźił winnicę / y ogrodźił ją płotem / y wykopał práſę / y zbudował wieżę / y nájął ją winiárzom / y odjáchał precż.
Y poſłał gdy tego był cżás ſługę do winiárzów / áby od winiárzów odebrał pożytki oney winnice.
Lecż oni pojimawƺy go / ubili ; y odeſłáli próznego.
Y záśię poſłał do nich ſługę drugiego / którego też <i>oni</i> ukámionowawƺy / głowę <i>mu</i> zránili / y odeſłáli obelżonego.
Y záśię poſłał inƺego <i>ſługę</i> : ále y tego zábili / y wiele innych ; z których jedne ubili / á drugie pozábijáli.
A májąc jeƺcże jednegoż ſwego miłego ſyná / poſłał ná oſtátek do nich y tego / mówiąc : Wżdyć śię będą wſtydźili ſyná mego.
Ale oni winiárze rzekli miedzy ſobą : Tenći jeſt dźiedźic ; Pódźćie zábijmy go / á będźie náƺe dźiedźictwo.
Y wźiąwƺy go zábili / á wyrzućili precż z winnice.
Cóż tedy ucżyni PAN oney winnice? Przydźie á potráći one winiárze / y da winnicę innym.
Izaliśćie nie cżytáli tego piſmá? Kámień który odrzućili budujący / ten śię ſtał głową węgielną.
Od PAnáć śię to ſtáło / y jeſt dźiwne w ocżách náƺych.
Stáráli śię tedy jákoby go pojimáć / ále śię ludu bali : bo poználi / yż przećiwko nim ono podobieńſtwo powiedźiał. Y zániechawƺy go / odeƺli.
Potym poſłáli do niego niektóre z Fáryzeuƺów y z Herodyánów ; áby go uśidlili w mowie.
A ták oni przyƺedƺy / rzekli mu : Náucżyćielu / wiemy żeś jeſt prawdźiwy / á nie dbaƺ ná nikogo : ábowiem nie pátrzaƺ ná oſobę ludzką / ále w prawdźie drogi Bożey ucżyƺ ; Godźiƺ śię dáć cżynƺ Ceſárzowi / cżyli nie? Mamyż go dáć / cżyli nie dáć?
A on poznawƺy obłudę ich / rzekł im : Cżemuż mię kuśićie? Przynieśćie mi groƺ / ábym <i>go</i> oglądał.
Tedy <i>mu</i> oni przynieśli ; á on im rzekł : Cżyjże to jeſt obraz y napis? A oni mu powiedźieli : Ceſárſki.
Y odpowiádájąc JEzus / rzekł im : Oddawajćież tedy co <i>jeſt</i> Ceſárſkiego / Ceſárzowi ; á co <i>jeſt</i> Bożego / Bogu. Y dźiwowáli mu śię.
Y przyƺli do niego Sáduceuƺowie / którzy mówią / yż niemáƺ zmartwychwſtánia ; y pytáli go mówiąc :
Náucżyćielu / Mojzeƺ nam nápiſał / iż jeſliby cżyj brát umárł / y zoſtáwił żonę / á dźiatekby nie zoſtáwił ; áby brát jego pojął żonę jego / á wzbudźił naśienie brátu ſwemu.
Było tedy śiedm bráći : á pierwƺy pojąwƺy żonę / umárł / y nie zoſtáwił naśienia.
A drugi pojąwƺy ją / umárł ; lecż y ten nie zoſtáwił naśienia. Tákże y trzeći.
A ták ją pojęło onych śiedm bráći ; á nie zoſtáwili naśienia. Náoſtátek po wƺyſtkich umárłá y oná niewiáſtá.
Przetoż / przy <i>zmartwych</i>wſtániu gdy powſtáną / któregoż z nich będźie żoną? Bo śiedm ich mieli ją zá żonę.
Ná to JEzus odpowiádájąc rzekł im : Zaż nie dla tego błądźićie / yżeśćie nie powiádomi piſmá / áni mocy Bożey?
Abowiem gdy zmartwychwſtáną áni śię żenią / áni zá mąż wydawáją : ále ſą jáko Anjołowie w niebieśiech.
A o umárłych iż będą wzbudzeni / niecżytáliśćież w kśięgách Mojzeƺowych / jáko Bóg do niego ze krzá mówił / y rzekł : Jam jeſt Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów.
Bóg / nie jeſtći Bogiem umárłych / ále Bogiem żywych : przetoż wy bárzo błądźićie.
A przyſtąpiwƺy jeden z Náucżonych w piśmie / ſłyƺąc że z ſobą gadáli / <i>á</i> widząc że im dobrze odpowiedźiał / ſpytał go : Które jeſt napierwƺe ze wƺyſtkich Przykazánie?
A JEzus mu odpowiedźiał : Napierwƺe ze wƺyſtkich przykazánie <i>jeſt</i> ; Słuchaj Izráelu / PAN Bóg náƺ / PAN jeden jeſt.
Przetoż będźieƺ miłował PAná Bogá twego / ze wƺyſtkiego ſercá twego / y ze wƺyſtkiey duƺe twojey / y ze wƺyſtkiey myśli twojey / y ze wƺyſtkiey śiły twojey : Toć <i>jeſt</i> pierwƺe przykazánie.
A wtóre <i>temu</i> podobne : to <i>jeſt,</i> Będźieƺ miłował bliźniego twego jáko ſámego śiebie. Więkƺego przykazánia innego nád to niemáƺ.
Tedy mu rzekł on Náucżony w piśmie : Náucżyćielu / záprawdę dobrześ powiedźiał / iż jeden jeſt Bóg / á niemáƺ inƺego oprócż niego.
Y miłowáć go ze wƺyſtkiego ſercá / y ze wƺyſtkiey myśli / y ze wƺyſtkiey duƺe / y ze wƺyſtkiey śiły : á miłować bliźniego jáko ſámego śiebie / więcey jeſt nád wƺyſtkie cáłopalenia y ofiáry.
A widząc JEzus / że on mądrze odpowiedźiał / rzekł mu : Nie dálekoś jeſt od króleſtwá Bożego. Y nie śmiał go nikt dáley pytáć.
Tedy JEzus odpowiádájąc rzekł / gdy ucżył w Kośćiele : Jákoż mówią Náucżeni w piśmie / iż CHryſtus jeſt Syn Dawidów?
Bo ſam Dawid przez Duchá Świętego powiedźiał : Rzekł PAN PANU memu / śiądź po práwicy mojey / áż położę nieprzyjaćioły twoje podnóżkiem nóg twojich.
<i>Ponieważ</i> go ſam Dawid názywa PAnem ; Jákoż tedy jeſt ſynem jego? A wielki lud rad go ſłuchał.
Y mówił do nich w náuce ſwojey : Strzeżćie śię Náucżonych w piśmie / którzy chcą w długich ƺátách chodźić / á bydź pozdrawiáni ná rynkách :
Y ná pierwƺych ſtołkách śiadáć w bóżnicách / y pierwƺe miejſcá mieć ná wiecżerzách :
Którzy pożeráją domy wdów / á <i>to</i> pod pokrywką długich modlitew. Ćić odnioſą ćiężƺy ſąd.
A JEzus śiedząc przećiwko ſkárbnicy / przypátrował śię jáko lud rzucał pieniądze do ſkárbnice ; y <i>jáko</i> wiele bogacżów / wiele rzucáło.
Y przyƺedƺy jedná wdowá uboga / wrzućiłá dwá drobne pieniążki / co cżyni kwartnik.
Tedy zwoławƺy ucżniów ſwojich / rzekł im : Záprawdę wam powiádam / że tá uboga wdowá więcey wrzućiłá niżeli ći wƺyſcy którzy rzucáli do ſkárbnice.
Abowiem ći wƺyſcy z tego co im zbywáło rzucáli : ále tá z ubóſtwá ſwego / wƺyſtko co miáłá wrzućiłá / wƺyſtkę żywność ſwoję.
