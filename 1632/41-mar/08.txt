A w oneż dni gdy náder wielki lud był / á nie mieli coby jedli / zwoławƺy JEzus ucżniów ſwojich / rzekł im :
Żal mi tego ludu ; bo już trzy dni trwáją przy mnie / á nie máją coby jedli.
A jeſli je rozpuƺcżę głodne do domów ich / pomdleją ná drodze : ábowiem niektórzy z nich z dáleká przyƺli.
Tedy mu odpowiedźieli ucżniowie jego : Z kądże te kto będźie mógł náſyćić chlebem tu ná puƺcży?
Y zpytał ich : Wieleż maćie chlebów? A oni rzekli : Siedm.
Y rozkazał ludowi żeby uśiadł ná źiemi : A wźiąwƺy onę śiedm chlebów / podźiękowawƺy łamał / y dawał ucżniom ſwojim áby przed lud kłádli : y kłádli przed lud.
Mieli też trochę rybek / które pobłogoſłáwiwƺy / kazał y one przed lud kłáść.
Jedli tedy / y náſyceni ſą : y zebráli co zbyło ułomków śiedm koƺów.
A było tych co jedli / około cżterech tyśięcy : y rozpuśćił je.
A wnet wſtąpiwƺy w łódź z ucżniámi ſwojimi / przyƺedł w ſtrony Dálmánutſkie.
Y wyƺli Fáryzeuƺowie / á pocżęli z nim ſpór wieść / ƺukájąc od niego známienia z niebá / á kuƺąc go.
Tedy weſtchnąwƺy ſerdecżnie w duchu ſwym / rzekł : Przecżże ten rodzaj známienia ƺuka? Záprawdę powiádam wam / że nie będźie dáne známię temu rodzájowi.
Y opuśćiwƺy je / wſtąpił záśię w łódź / y przepráwił śię ná drugą ſtronę.
A zápomnieli byli ucżniowie wźiąć z ſobą chlebá / y nie mieli z ſobą nic więcey / tylko jeden chleb w łodźi.
Tedy im przykazał / mówiąc : Bacżćież / á ſtrzeżćie śię kwáſu Fáryzeuƺów y kwáſu Herodowego.
Y rozmawiáli miedzy ſobą / y rzekli : <i>O tym ſnadź mówi</i> że nie mamy chlebá.
Co poznawƺy JEzus / rzekł im : Ocżymże rozmawiaćie / iż nie maćie chlebá? Jeƺcżeż nie bacżyćie y nie zrozumiewaćie? Jeƺcżeż maćie ſerce ſwoje zdrętwiáłe?
Ocży májąc nie widźićie? y uƺy májąc nie ſłyƺyćie / y nie pámiętáćie?
Gdym onych pięć chlebów łamał miedzy pięć tyśięcy ludźi / Wieleżeśćie pełnych koƺów ułomków zebráli? Rzekli mu : Dwánaśćie.
A gdym onych śiedm <i>chlebów</i> łamał miedzy cżtery tyśiące ludźi / wieleśćie koƺów pełnych ułomków zebráli? A oni rzekli : Siedm.
A on im rzekł : Jákoż tedy nie rozumiećie?
Potym przyƺedł do Betſájdy : y przywiedli do niego ślepego / proƺąc go áby śię go dotknął.
A ująwƺy onego ślepego zá rękę / wywiódł go precż zá miáſtecżko : y plunąwƺy ná ocży jego / włożył nań ręce / y pytał go jeſliby co widźiał?
A on pojrzawƺy wzgórę / rzekł : Widzę ludźie ; bo widzę że chodzą jáko drzewá.
Potym záśię włożył ręce ná ocży jego / y rozkazał mu wzgórę pojrżeć : y uzdrowiony jeſt <i>ná wzroku,</i> ták / że y z dáleká wƺyſtkie jáśnie widźiał.
Y odeſłał go do domu jego / mówiąc : Y do tego miáſtecżká nie wchodź / y nikomu z miáſtecżká nie powiádaj.
Tedy wyƺedł JEzus y ucżniowie jego do miáſtecżek <i>náleżących</i> Cezáryey Filipowey / á w drodze pytał ucżniów ſwojich / mówiąc im : Kimże mię powiádáją być ludźie?
A oni mu odpowiedźieli : <i>Jedni</i> Janem Chrzćićielem / á drudzy Eliaƺem / á drudzy jednym z Proroków.
Ale on im rzekł : A wy kim mię być powiádaćie? A odpowiádájąc Piotr / rzekł mu : Tyś jeſt on CHryſtus.
Y przygroźił im / áby o nim nikomu nie powiádáli.
Y pocżął je náucżáć / że Syn cżłowiecży muśi wiele ućierpieć / y odrzuconym być od ſtárƺych ludu / y od Przedniejƺych kápłanów / y Náucżonych w piśmie ; y być zábity / á po trzech dniách <i>zmartwych</i>wſtáć.
A to mówił jáwnie. Tedy go Piotr wźiąwƺy ná ſtronę / pocżął go ſtrofowáć.
Ale on obróćiwƺy śię / á wejrzawƺy ná ucżnie ſwoje / zgromił Piotrá / mówiąc : Idź ode mnie ƺátánie : ábowiem nie pojmujeƺ tego co jeſt Bożego / ále co jeſt ludzkiego.
A zwoławƺy ludu z ucżniámi ſwojimi / rzekł im : Ktokolwiek chce zá mną iść / niech ſáme° śiebie záprze / á weźmie krzyż ſwój / y náśláduje mię.
Abowiem ktoby chćiał záchowáć duƺę ſwą / ſtráći ją : á ktoby ſtráćił duƺę ſwą dla mnie y dla Ewángeliey / ten ją záchowa.
Bo / cóż pomoże cżłowiekowi choćby wƺyſtek świát pozyſkał / á ƺkodowałby ná duƺy ſwojey?
Albo co zá zamiánę da cżłowiek zá duƺę ſwoję?
Abowiem ktoby śię wſtydźił zá mię y zá ſłowá moje / miedzy tym rodzájem cudzołożnym y grzeƺnym / y Syn cżłowiecży wſtydźić śię zań będźie / gdy przydźie w chwale Ojcá ſwego z Anjoły świętymi.
