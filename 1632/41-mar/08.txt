A w oneż dni / gdy náder wielki lud był / á nie mieli / coby jedli / zwołáwƺy JEzuſ ucżniów ſwoich / rzekł im :
Żál mi tego ludu ; bo już trzy dni trwáją przy mnie / á nie máją / coby jedli ;
A jeſli je rozpuƺcżę głodne do domów ich / pomdleją ná drodze ; ábowiem niektórzy z nich z dáleká przyƺli.
Tedy mu odpowiedźieli ucżniowie jego : Skądże te kto będźie mógł náſyćić chlebem tu ná puƺcży?
Y ſpytáł ich : Wieleż máćie chlebów? A oni rzekli : Siedm.
Y rozkázáł ludowi / żeby uśiádł ná źiemi ; á wźiąwƺy one śiedm chlebów / podźiękowáwƺy łámáł / y dáwáł ucżniom ſwoim / áby przed lud kłádli ; y kłádli przed lud.
Mieli też trochę rybek / które pobłogoſłáwiwƺy / kázáł y one przed lud kłáść.
Jedli tedy y náſyceni ſą / y zebráli / co zbyło ułomków / śiedm koƺów.
A było tych / co jedli / około cżterech tyśięcy ; y rozpuśćił je.
A wnet wſtąpiwƺy w łódź z ucżniámi ſwoimi / przyƺedł w ſtrony Dálmánutſkie.
Y wyƺli Fáryzeuƺowie / á pocżęli z nim ſpór wieść / ƺukájąc od niego známieniá z niebá á kuƺąc go.
Tedy weſtchnąwƺy ſerdecżnie w duchu ſwym / rzekł : Przecżże ten rodzáj známieniá ƺuká? Zápráwdę powiádám wám / że nie będźie dáne známię temu rodzájowi.
Y opuśćiwƺy ich / wſtąpił záśię w łódź / y przepráwił śię ná drugą ſtronę.
A zápomnieli byli ucżniowie wźiąć z ſobą chlebá / y nie mieli z ſobą nic więcey / tylko jeden chleb w łodźi.
Tedy im przykázáł / mówiąc : Bácżćież / á ſtrzeżćie śię kwáſu Fáryzeuƺów y kwáſu Herodowego.
Y rozmáwiáli między ſobą y rzekli : O tem ſnáć mówi / że nie mámy chlebá.
Co poznáwƺy JEzuſ / rzekł im : O cżemże rozmáwiáćie / iż nie máćie chlebá? Jeƺcżeż nie bácżyćie y nie zrozumiewáćie? Jeƺcżeż máćie ſerce ſwoje zdrętwiáłe?
Ocży májąc nie widźićie / y uƺy májąc nie ſłyƺyćie / y nie pámiętáćie?
Gdym onych pięć chlebów łámáł między pięć tyśięcy ludźi / wieleżeśćie pełnych koƺów ułomków zebráli? Rzekli mu : Dwánáśćie ;
A gdym onych śiedm chlebów łámáł między cżtery tyśiące ludźi / wieleśćie koƺów pełnych ułomków zebráli? A oni rzekli : Siedm.
A on im rzekł : Jákoż tedy nie rozumiećie?
Potem przyƺedł do Betſáidy ; y przywiedli do niego ślepego / proƺąc go / áby śię go dotknął.
A ująwƺy onego ślepego zá rękę / wywiódł go precż zá miáſtecżko / y plunąwƺy ná ocży jego / włożył ná niego ręce / y pytáł go / jeſliby co widźiáł.
A on ſpojrzáwƺy w górę / rzekł : Widzę ludźi ; bo widzę / że chodzą jáko drzewá.
Potem záśię włożył ręce ná ocży jego / y rozkázáł mu w górę ſpojrzeć ; y uzdrowiony jeſt ná wzroku / ták / że y z dáleká wƺyſtkich jáſno widźiáł.
Y odeſłáł go do domu jego / mówiąc : Y do tego miáſtecżká nie wchodź / y nikomu z miáſtecżká nie powiádáj.
Tedy wyƺedł JEzuſ y ucżniowie jego do miáſtecżek náleżących do Cezáryi Filipowej / á w drodze pytáł ucżniów ſwoich / mówiąc im : Kimże mię powiádáją być ludźie?
A oni mu odpowiedźieli : Jedni Jánem Chrzćićielem / á drudzy Elijáƺem / á drudzy jednym z proroków.
Ale on im rzekł : A wy kim mię być powiádáćie? A odpowiádájąc Piotr / rzekł mu : Tyś jeſt on CHryſtuſ.
Y przygroźił im / áby o nim nikomu nie powiádáli.
Y pocżął je náucżáć / że Syn cżłowiecży muśi wiele ućierpieć / y odrzuconym być od ſtárƺych ludu / y od przedniejƺych kápłánów y náucżonych w Piśmie / y być zábity / á po trzech dniách zmartwychwſtáć.
A to mówił jáwnie. Tedy go Piotr wźiąwƺy ná ſtronę / pocżął go ſtrofowáć.
Ale on obróćiwƺy śię / á wejrzáwƺy ná ucżnie ſwoje / zgromił Piotrá / mówiąc : Idź ode mnie / ƺátánie ; ábowiem nie pojmujeƺ tego / co jeſt Bożego / ále co jeſt ludzkiego.
A zwołáwƺy ludu z ucżniámi ſwoimi / rzekł im : Ktokolwiek chce zá mną iść / niech ſámego śiebie záprze / á weźmie krzyż ſwój / y náśláduje mię.
Abowiem ktoby chćiáł záchowáć duƺę ſwą / ſtráći ją ; á ktoby ſtráćił duƺę ſwą dla mnie y dla Ewángielii / ten ją záchowá.
Bo cóż pomoże cżłowiekowi / choćby wƺyſtek świát pozyſkáł / á ƺkodowáłby ná duƺy ſwojey?
Albo co zá zámiánę dá cżłowiek zá duƺę ſwoję?
Abowiem ktoby śię wſtydźił zá mię y zá ſłowá moje między tym rodzájem cudzołożnym y grzeƺnym / y Syn cżłowiecży wſtydźić śię záń będźie / gdy przyjdźie w chwále Ojcá ſwego z Anioły świętymi.
