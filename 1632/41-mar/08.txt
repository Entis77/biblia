A w oneż dni / gdy náder wielki lud był / á nie mieli / coby jedli / zwołáwƺy Jezuſ ucżniów ſwoich / rzekł im :
Żál mi tego ludu ; bo już trzy dni trwáją przy mnie / á nie máją / coby jedli ;
A jeźli je rozpuƺcżę głodne do domów ich / pomdleją ná drodze ; ábowiem niektórzy z nich z dáleká przyƺli.
Tedy mu odpowiedzieli ucżniowie jego : Skądże te kto będzie mógł náſyćić chlebem tu ná puƺcży?
Y ſpytáł ich : Wieleż máćie chlebów? A oni rzekli : Siedm.
Y rozkázáł ludowi / żeby uśiádł ná ziemi ; á wziąwƺy one śiedm chlebów / podziękowáwƺy łámáł / y dáwáł ucżniom ſwoim / áby przed lud kłádli ; y kłádli przed lud.
Mieli też trochę rybek / które pobłogoſłáwiwƺy / kázáł y one przed lud kłáść.
Jedli tedy y náſyceni ſą / y zebráli / co zbyło ułomków / śiedm koƺów.
A było tych / co jedli / około cżterech tyśięcy ; y rozpuśćił je.
A wnet wſtąpiwƺy w łódź z ucżniámi ſwoimi / przyƺedł w ſtrony Dálmánutſkie.
Y wyƺli Fáryzeuƺowie / á pocżęli z nim ſpór wieść / ƺukájąc od niego známieniá z niebá á kuƺąc go.
Tedy weſtchnąwƺy ſerdecżnie w duchu ſwym / rzekł : Przecżże ten rodzáj známieniá ƺuká? Zápráwdę powiádám wám / że nie będzie dáne známię temu rodzájowi.
Y opuśćiwƺy ich / wſtąpił záśię w łódź / y przepráwił śię ná drugą ſtronę.
A zápomnieli byli ucżniowie wziąć z ſobą chlebá / y nie mieli z ſobą nic więcej / tylko jeden chleb w łodzi.
Tedy im przykázáł / mówiąc : Bácżćież / á ſtrzeżćie śię kwáſu Fáryzeuƺów y kwáſu Herodowego.
Y rozmáwiáli między ſobą y rzekli : O tem ſnáć mówi / że nie mámy chlebá.
Co poznáwƺy Jezuſ / rzekł im : O cżemże rozmáwiáćie / iż nie máćie chlebá? Jeƺcżeż nie bácżyćie y nie zrozumiewáćie? Jeƺcżeż máćie ſerce ſwoje zdrętwiáłe?
Ocży májąc nie widzićie / y uƺy májąc nie ſłyƺyćie / y nie pámiętáćie?
Gdym onych pięć chlebów łámáł między pięć tyśięcy ludzi / wieleżeśćie pełnych koƺów ułomków zebráli? Rzekli mu : Dwánáśćie ;
A gdym onych śiedm chlebów łámáł między cżtery tyśiące ludzi / wieleśćie koƺów pełnych ułomków zebráli? A oni rzekli : Siedm.
A on im rzekł : Jákoż tedy nie rozumiećie?
Potem przyƺedł do Betſáidy ; y przywiedli do niego ślepego / proƺąc go / áby śię go dotknął.
A ująwƺy onego ślepego zá rękę / wywiódł go precż zá miáſtecżko / y plunąwƺy ná ocży jego / włożył ná niego ręce / y pytáł go / jeźliby co widziáł.
A on ſpojrzáwƺy w górę / rzekł : Widzę ludzi ; bo widzę / że chodzą jáko drzewá.
Potem záśię włożył ręce ná ocży jego / y rozkázáł mu w górę ſpojrzeć ; y uzdrowiony jeſt ná wzroku / ták / że y z dáleká wƺyſtkich jáſno widziáł.
Y odeſłáł go do domu jego / mówiąc : Y do tego miáſtecżká nie wchodź / y nikomu z miáſtecżká nie powiádáj.
Tedy wyƺedł Jezuſ y ucżniowie jego do miáſtecżek náleżących do Cezáryi Filipowej / á w drodze pytáł ucżniów ſwoich / mówiąc im : Kimże mię powiádáją być ludzie?
A oni mu odpowiedzieli : Jedni Jánem Chrzćićielem / á drudzy Elijáƺem / á drudzy jednym z proroków.
Ale on im rzekł : A wy kim mię być powiádáćie? A odpowiádájąc Piotr / rzekł mu : Tyś jeſt on Chryſtuſ.
Y przygroził im / áby o nim nikomu nie powiádáli.
Y pocżął je náucżáć / że Syn cżłowiecży muśi wiele ućierpieć / y odrzuconym być od ſtárƺych ludu / y od przedniejƺych kápłánów y náucżonych w Piśmie / y być zábity / á po trzech dniách zmártwychwſtáć.
A to mówił jáwnie. Tedy go Piotr wziąwƺy ná ſtronę / pocżął go ſtrofowáć.
Ale on obróćiwƺy śię / á wejrzáwƺy ná ucżnie ſwoje / zgromił Piotrá / mówiąc : Idź ode mnie / ƺátánie ; ábowiem nie pojmujeƺ tego / co jeſt Bożego / ále co jeſt ludzkiego.
A zwołáwƺy ludu z ucżniámi ſwoimi / rzekł im : Ktokolwiek chce zá mną iść / niech ſámego śiebie záprze / á weźmie krzyż ſwój / y náśláduje mię.
Abowiem ktoby chćiáł záchowáć duƺę ſwą / ſtráći ją ; á ktoby ſtráćił duƺę ſwą dlá mnie y dlá Ewángielii / ten ją záchowá.
Bo cóż pomoże cżłowiekowi / choćby wƺyſtek świát pozyſkáł / á ƺkodowáłby ná duƺy ſwojej?
Albo co zá zámiánę dá cżłowiek zá duƺę ſwoję?
Abowiem ktoby śię wſtydził zá mię y zá ſłowá moje między tym rodzájem cudzołożnym y grzeƺnym / y Syn cżłowiecży wſtydzić śię záń będzie / gdy przyjdzie w chwále Ojcá ſwego z Anioły świętymi.
