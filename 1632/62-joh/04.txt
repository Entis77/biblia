Nájmilśi! nie káżdemu duchowi wierzćie ; ále doświádcżájćie duchów / jeźli z Bogá ſą. Bo wiele fáłƺywych proroków wyƺło ná świát.
Przez to poznáwájćie Duchá Bożego : Wƺelki duch / który wyznáje / iż Jezuſ Chryſtuſ w ćiele przyƺedł / z Bogá jeſt.
Ale wƺelki duch / który nie wyznáje / że Jezuſ Chryſtuſ w ćiele przyƺedł / nie jeſt z Bogá ; ále ten jeſt on duch ántychryſtuſowy / o którymeśćie ſłyƺeli / iż idzie / y teráz już jeſt ná świećie.
Wy z Bogá jeſteśćie / dziátecżki! y zwyćiężyliśćie ich ; iż więkƺy jeſt ten / który w wáſ jeſt / niż ten / który jeſt ná świećie.
Onić ſą z świátá ; przetoż o świećie mówią / á świát ich ſłuchá.
My z Bogá jeſteśmy. Kto zná Bogá / ſłuchá náſ ; kto nie jeſt z Bogá / nie ſłuchá náſ. Przez to poznájemy duchá práwdy y duchá błędu.
Nájmilśi! miłujmyż jedni drugich / gdyż miłość jeſt z Bogá ; y káżdy / co miłuje / z Bogá jeſt národzony y zná Bogá.
Kto nie miłuje / nie zná Bogá ; gdyż Bóg jeſt miłość.
Przez to objáwioná jeſt miłość Bożá ku nám / iż Syná ſwego jednorodzonego poſłáł Bóg ná świát / ábyśmy żyli przezeń.
W tem jeſt miłość / nie iżbyśmy my umiłowáli Bogá / ále iż on umiłowáł náſ / y poſłáł Syná ſwego / áby był ubłágániem zá grzechy náƺe.
Nájmilśi! poniewáż náſ ták Bóg umiłowáł / y myśmy powinni jedni drugich miłowáć.
Bogá żáden nigdy nie widziáł ; ále jeźli miłujemy jedni drugich / Bóg w náſ mieƺká / á miłość jego doſkonáłá jeſt w náſ.
Przez to poznájemy / iż w nim mieƺkámy / á on w náſ / iż z Duchá ſwego nám dáł.
A myśmy widzieli y świádcżymy / iż Ojćiec poſłáł Syná / áby był zbáwićielem świátá.
Ktobykolwiek wyznáł / iż Jezuſ jeſt Synem Bożym / Bóg w nim mieƺká / á on w Bogu.
Y myśmyć poználi y uwierzyli o miłośći / którą Bóg má ku nám. Bóg jeſt miłość ; á kto mieƺká w miłośći / w Bogu mieƺká / á Bóg w nim.
W tem doſkonáłá jeſt miłość Bożá z námi / ábyśmy ufánie mieli w dzień ſądny / iż jáki on jeſt / tácy y my jeſteśmy ná tym świećie.
Niemáƺći bojáźni w miłośći / ále miłość doſkonáłá precż wyrzucá bojáźń : bo bojáźń má udręcżenie / á kto śię boi / nie jeſt doſkonáły w miłośći.
My go miłujemy / iż on náſ pierwej umiłowáł.
Jeźliby kto rzekł : Miłuję Bogá / á brátáby ſwego nienáwidziáł / kłámcą jeſt ; ábowiem kto nie miłuje brátá ſwego / którego widziáł / Bogá / którego nie widziáł / jákoż może miłowáć?
A toć rozkázánie mámy od niego / áby ten / co miłuje Bogá / miłowáł y brátá ſwego.
