Wƺelki / co wierzy / iż Jezuſ jeſt Chryſtuſem / z Bogá śię národził ; á wƺelki / co miłuje tego / który urodził / miłuje y tego / który z niego jeſt národzony.
Przez to známy / iż miłujemy dziátki Boże / gdy Bogá miłujemy / y przykázániá jego chowámy.
Abowiem tá jeſt miłość Bożá / ábyśmy przykázániá jego chowáli ; á przykázániá jego nie ſą ćiężkie.
Bo wƺyſtko / co śię národziło z Bogá / zwyćiężá świát ; á to jeſt zwyćięſtwo / które zwyćiężyło świát / wiárá náƺá.
Któż jeſt / co zwyćiężá świát / tylko kto wierzy / iż Jezuſ jeſt Synem Bożym?
Tenći jeſt / który przyƺedł przez wodę y krew / Jezuſ Chryſtuſ / nie w wodzie tylko / ále w wodzie y we krwi ; á Duch jeſt / który świádcży / iż Duch jeſt práwdá.
Abowiem trzej ſą / którzy świádcżą ná niebie : Ojćiec / Słowo / y Duch Święty / á ći trzej jedno ſą.
A trzej ſą / którzy świádcżą ná ziemi : Duch / y wodá / y krew / á ći trzej ku jednemu ſą.
Poniewáż świádectwo ludzkie przyjmujemy / świádectwo Boże więkƺe jeſt ; ábowiem to jeſt świádectwo Boże / które świádcżył o Synu ſwoim.
Kto wierzy w Syná Bożego má świádectwo ſám w ſobie. Kto nie wierzy Bogu / kłámcą go ucżynił / iż nie uwierzył temu świádectwu / które Bóg świádcżył o Synu ſwoim.
A toć jeſt świádectwo / iż nám Bóg dáł żywot wiecżny ; á ten żywot jeſt w Synu jego.
Kto má Syná / má żywot ; kto nie má Syná Bożego / nie má żywotá.
Te rzecży nápiſáłem wám / którzy wierzyćie w imię Syná Bożego / żebyśćie wiedzieli / iż máćie żywot wiecżny / y ábyśćie wierzyli w imię Syná Bożego.
A toć jeſt ufánie / które mámy do niego / iż jeźlibyśmy o co prośili według woli jego / ſłyƺy náſ.
A jeźli wiemy / iż náſ ſłyƺy / o cokolwiekbyśmy prośili / tedy wiemy / iż mámy te rzecży / o któreśmy go prośili.
Jeźliby kto widziáł brátá ſwego grzeƺącego grzechem nie ná śmierć / niechże śię modli zá nim / á dá mu Bóg żywot / to jeſt / grzeƺącym nie ná śmierć. Jeſtći grzech ná śmierć ; nie zá tym / mówię / áby śię kto modlił.
Wƺelká nieſpráwiedliwość jeſt grzech ; ále jeſt grzech nie ná śmierć.
Wiemy / iż wƺelki / który śię z Bogá národził / nie grzeƺy ; ále który śię národził z Bogá / záchowuje ſámego śiebie / á on złośnik nie dotyká śię go.
Wiemy / iż z Bogá jeſteśmy ; ále świát wƺyſtek w złem położony jeſt.
A wiemy / iż Syn Boży przyƺedł / y dáł nám zmyſł / ábyśmy poználi onego práwdziwego Bogá / y jeſteśmy w onym práwdziwym / to jeſt / w Synu jego Jezuśie Chryſtuśie ; tenći jeſt práwdziwy Bóg y żywot wiecżny.
Dziátecżki! ſtrzeżćie śię báłwánów. Amen.
