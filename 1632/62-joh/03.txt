Pátrzćie / jáką miłość dáł nám Ojćiec / ábyśmy dźiátkámi Bożemi názwáni byli. Dlátegoć świát nie zná náſ / iż onego nie zná.
Nájmilśi! teráz dźiátkámi Bożemi jeſteśmy / ále śię jeƺcże nie objáwiło / cżem będźiemy ; lecż wiemy / iż gdy śię on objáwi / podobni mu będźiemy ; ábowiem ujrzymy go ták / jáko jeſt.
A ktokolwiek má tę nádźieję w nim / ocżyƺcżá śię / jáko y on cżyſty jeſt.
Káżdy / co cżyni grzech / ten y zákon przeſtępuje ; ábowiem grzech jeſt przeſtępſtwo zákonu.
A wiećie / iż śię on objáwił / áby grzechy náƺe zgłádźił / á grzechu w nim niemáƺ.
Wƺelki tedy / kto w nim mieƺká / nie grzeƺy ; ále ktokolwiek grzeƺy / nie widźiáł go / áni go poznáł.
Dźiátecżki! niecháj wáſ nikt nie zwodźi ; kto cżyni ſpráwiedliwość / ſpráwiedliwy jeſt / jáko y on ſpráwiedliwy jeſt ;
Kto cżyni grzech / z dyjábłá jeſt ; gdyż od pocżątku dyjábeł grzeƺy. Ná to śię objáwił Syn Boży / áby zepſowáł ucżynki dyjábelſkie.
Wƺelki / co śię národźił z Bogá / grzechu nie cżyni / iż náśienie jego w nim zoſtáje / y nie może grzeƺyć / iż z Bogá národzony jeſt.
Po tem poznáć dźiátki Boże y dźieći dyjábelſkie. Wƺelki / który nie cżyni ſpráwiedliwośći / nie jeſt z Bogá / y który nie miłuje brátá ſwego.
Abowiem to jeſt poſelſtwo / któreśćie ſłyƺeli od pocżątku / ábyśmy jedni drugich miłowáli.
Nie jáko Káin / który był z tego złośniká / y zábił brátá ſwego. A dlácżegoż go zábił? Iż ucżynki jego złe były / á brátá jego ſpráwiedliwe.
Nie dźiwujćie śię / bráćiá moi! jeźli wáſ świát nienáwidźi.
My wiemy / żeśmy przenieśieni z śmierći do żywotá / iż miłujemy bráći ; kto nie miłuje brátá / zoſtáje w śmierći.
Káżdy / co nienáwidźi brátá ſwego / mężobójcą jeſt ; á wiećie / iż żáden mężobójcá nie má żywotá wiecżnego w ſobie zoſtáwájącego.
Przez tośmy poználi miłość Bożą / iż on duƺę ſwoję zá náſ położył ; y myśmy powinni kłáść duƺę zá bráći.
A ktoby miáł májętność świátá tego / y widźiáłby brátá ſwego potrzebującego / á záwárłby wnętrznośći ſwoje przed nim / jákoż w nim zoſtáje miłość Bożá?
Dźiátecżki moje! nie miłujmy ſłowem áni językiem / ále ucżynkiem y práwdą.
A przez to poznájemy / iż z práwdy jeſteśmy y przed nim uſpokojemy ſercá náƺe.
Bo jeźliby náſ potępiáło ſerce náƺe / dáleko więkƺy jeſt Bóg / niż ſerce náƺe / y wie wƺyſtko.
Nájmilśi! jeźliby ſerce náƺe náſ nie potępiáło / ufánie mámy ku Bogu ;
Y o cokolwiekbyśmy prośili / bierzemy od niego ; bo przykázániá jego chowámy / y to / co śię podobá przed oblicżem jego / cżynimy.
A toć jeſt przykázánie jego / ábyśmy wierzyli imieniowi Syná jego JEzuſá CHryſtuſá / y miłowáli jedni drugich / jáko nám przykázáł.
Bo kto chowá przykázániá jego / w nim mieƺká / á on też w nim ; á przez to známy / iż mieƺká w náſ / to jeſt z Duchá / którego nám dáł.
