Dziátki moje! to wám piƺę / ábyśćie nie grzeƺyli ; y jeźliby kto zgrzeƺył / mámy orędowniká u Ojcá / Jezuſá Chryſtuſá ſpráwiedliwego ;
A on jeſt ubłágániem zá grzechy náƺe ; á nie tylko zá náƺe / ále też zá grzechy wƺyſtkiego świátá.
A przez to wiemy / żeśmy go poználi / jeźli przykázániá jego záchowujemy.
Kto mówi : Znám go / á przykázániá jego nie záchowuje / kłámcą jeſt / á práwdy w nim niemáƺ.
Lecż ktoby záchowáł ſłowá jego / práwdziwie śię w tym miłość Bożá wykonáłá ; przez to známy / iż w nim jeſteśmy.
Kto mówi / że w nim mieƺká / powinien / jáko on chodził / y ſám tákże chodzić.
Bráćiá! nie nowe przykázánie wám piƺę / ále przykázánie ſtáre / któreśćie mieli od pocżątku ; á to ſtáre przykázánie jeſt ono ſłowo / któreśćie ſłyƺeli od pocżątku.
Záśię przykázánie nowe piƺę wám / które jeſt práwdziwe w nim y w wáſ ; iż ćiemność przemijá / á práwdziwá oná świátłość już świeći.
Kto mówi / iż jeſt w świátłośći / á brátá ſwego nienáwidzi / w ćiemnośći jeſt áż dotąd.
Kto miłuje brátá ſwego / w świátłośći mieƺká y zgorƺeniá w nim nie máƺ.
Lecż kto nienáwidzi brátá ſwego / w ćiemnośći jeſt y w ćiemnośći chodzi / á nie wie / gdzie idzie / iż ćiemność záślepiłá ocży jego.
Piƺę wám / dziátki! iż wám ſą odpuƺcżone grzechy dlá imieniá jego.
Piƺę wám / ojcowie! żeśćie poználi tego / który jeſt od pocżątku. Piƺę wám / młodzieńcy! żeśćie zwyćiężyli onego złośniká.
Piƺę wám / dziátecżki! żeśćie poználi Ojcá. Piſáłem wám / ojcowie! żeśćie poználi onego / który jeſt od pocżątku. Piſáłem wám / młodzieńcy! żeśćie mocni / á ſłowo Boże mieƺká w wáſ / á żeśćie zwyćiężyli onego złośniká.
Nie miłujćie świátá / áni tych rzecży / które ſą ná świećie ; jeźli kto miłuje świát / nie máƺ w nim miłośći ojcowſkiej.
Abowiem wƺyſtko / co jeſt ná świećie / jáko pożądliwość ćiáłá y pożądliwość ocżu / y pychá żywotá / toć nie jeſt z Ojcá / ále jeſt z świátá.
Świátći przemijá y pożądliwość jego ; ále kto cżyni wolę Bożą / trwá ná wieki.
Dziátecżki! oſtátecżná godziná jeſt ; á jákośćie ſłyƺeli / że ántychryſt przyjść má / y teráz wiele ántychryſtów powſtáło ; ſtąd wiemy / iż jeſt oſtátecżná godziná.
Z náſ wyƺli / ále nie byli z náſ ; ábowiem gdyby byli z náſ / zoſtáliby byli z námi ; ále wyƺli z náſ / áby objáwieni byli / iż wƺyſcy nie byli z náſ.
Ale wy máćie pomázánie od onego Świętego / y wiećie wƺyſtko.
Nie piſáłem wám / przeto / żebyśćie práwdy nie ználi / ále że ją znáćie / á iż wƺelkie kłámſtwo nie jeſt z práwdy.
Któż jeſt kłámcą? Azáż nie ten / który zápierá / iż Jezuſ nie jeſt Chryſtuſem? Ten jeſt ántychryſt / który śię zápierá Ojcá y Syná.
Káżdy / co śię zápierá Syná / y Ojcá nie má ; á kto wyznáje Syná / má y Ojcá.
Wy tedy / cośćie ſłyƺeli od pocżątku / to niecháj w wáſ zoſtáje ; jeźliby w wáſ zoſtáwáło / cośćie ſłyƺeli od pocżątku / y wy w Synu y w Ojcu zoſtániećie.
A táć jeſt obietnicá / którą on nám obiecáł / to jeſt / żywot on wiecżny.
Tom wám nápiſáł o tych / którzy wáſ zwodzą.
Ale to pomázánie / któreśćie wy wzięli od niego / zoſtáje w wáſ / á nie potrzebujećie / áby wáſ kto ucżył : ále jáko to pomázánie ucży wáſ o wƺyſtkiem / á jeſt práwdziwe / y nie jeſt kłámſtwem / á jáko wáſ náucżyło / ták w nim zoſtániećie.
Y teráz / dziátecżki! zoſtáńćie w nim / ábyśmy / gdy śię ukáże / ufánie mieli / á nie byli záwſtydzeni od niego w przyjśćiu jego.
Poniewáż wiećie / że on ſpráwiedliwy jeſt / wiedzćież też / iż káżdy / który cżyni ſpráwiedliwość / z niego národzony jeſt.
