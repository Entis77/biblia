A oznájmujemy wám / bráćiá! o łáſce Bożej / którá jeſt dáná zborom Mácedońſkim ;
Iż w rozlicżnem doświádcżániu utrápieniá obfitá ich rádość / y bárdzo wielkie ubóſtwo ich obfitowáło w bogáctwo ƺcżerośći ich.
Bo dáję im świádectwo / że według możnośći / y nád możność ochotnymi śię pokázáli.
Z wielką prośbą náſ żądájąc / ábyśmy to dobrodźiejſtwo y ſpólne udźielenie / którem śię uſługuje świętym / przyjęli.
A nie tylko ták ſobie poſtąpili / jákośmy śię ſpodźiewáli ; ále nájprzód ſámych śiebie oddáli Pánu / potem y nám zá wolą Bożą.
Ták / żeśmy muśieli nápomnieć Tytuſá / áby jáko przedtem pocżął / ták áby też dokonáł u wáſ tegoż dobrodźiejſtwá.
Przetoż jáko we wƺyſtkiem obfitujećie w wierze y w mowie y w umiejętnośći / y we wƺelákiej pilnośći y w miłośći wáƺej przećiwko nám / ták y w tem dobrodźiejſtwie obfitujćie.
Nie mówię jáko rozkázując / ále przez pilność innych / jáko jeſt ƺcżerá miłość wáƺá / ná jáwią wyſtáwiájąc.
Abowiem znáćie łáſkę Páná náƺego JEzuſá CHryſtuſá / że dlá wáſ ſtáł śię ubogim / będąc bogátym / ábyśćie wy ubóſtwem jego ubogáceni byli.
A w tem podáję wám zdánie ſwoje ; ábowiem to wám jeſt pożytecżno / którzy nie tylko cżynić / ále y chćieć przedtemeśćie pocżęli roku przeƺłego.
A teráz to / cośćie cżynić pocżęli / wykonájćie / áby jáko byłá ochotná myśl ku chceniu / ták też áby było y dokońcżenie z tego / co máćie.
Abowiem jeźli przedtem byłá ochotná myśl / táż przyjemná jeſt według tego / co kto má / á nie według tego / cżego nie má.
Bo nie chcę / áby inśi mieli ulżenie á wy ućiśnienie / ále żeby zá równo nátencżáſ wáƺá obfitość ich niedoſtátkowi uſłużyłá ;
Aby też ich obfitość wáƺemu niedoſtátkowi uſłużyłá / żeby śię ſtáło porównánie /
Jáko nápiſáne : Kto wiele názbieráł / nie miáł názbyt ; á kto máło názbieráł / nie miáł máło.
Ale chwáłá Bogu / który dáł tákież ſtáránie o wáſ do ſercá Tytuſowego /
Iż ono nápominánie przyjął / á ſtáwƺy śię pilniejƺym / dobrowolnie poƺedł do wáſ.
A poſłáliśmy weſpół z nim brátá / który má chwáłę w Ewángielii po wƺyſtkich zborách ;
A nie tylko to / ále obrány jeſt przez głoſy od zborów zá towárzyƺá drogi náƺej z tem dobrodźiejſtwem / którem śię dźieje uſługá od náſ ku chwále ſámego Páná y ku oświádcżeniu ochotnego umyſłu wáƺego /
Uchodząc tego / áby nám kto nie przygánił dlá tej obfitośći / którą śię przez náſ uſługuje /
Pilnie śię ſtárájąc o ucżćiwe rzecży / nie tylko przed Pánem / ále też y przed ludźmi.
A poſłáliśmy z nimi brátá náƺego / któregośmy cżęſto doświádcżyli / w wielu rzecżách być pilnym / á teráz dáleko pilniejƺym dlá wielkiej dowiernośći / którą má przećiwko wám.
A jeźli idźie o Tytuſá / ten jeſt moim towárzyƺem y u wáſ pomocnikiem ; á jeźli też o bráći náƺych / poſłámi ſą zborów y chwáłą CHryſtuſową.
Przetoż oświádcżenie miłośći wáƺej y chluby náƺej z wáſ pokáżćie przećiwko nim przed oblicżnośćią zborów.
