Obyśćie chćieli ná chwilę znośić głupſtwo moje! ále y znáƺájćie mię.
Abowiem gorliwym jeſtem ku wám gorliwośćią Bożą ; bom wáſ przygotowáł / ábym wáſ ſtáwił cżyſtą pánną jednemu mężowi Chryſtuſowi.
Lecż boję śię / by ſnáć jáko wąż oƺukáł Ewę chytrośćią ſwoją / ták też ſkáżone myśli wáƺe nie odpádły od proſtoty onej / którá jeſt w Chryſtuśie.
Bo gdyby kto przyƺedł / coby inƺego Jezuſá opowiádáł / któregośmy nie opowiádáli ; álbo gdybyśćie innego duchá wzięli / któregośćie nie wzięli / álbo inƺą Ewángieliję / którejśćie nie przyjęli / dobrzebyśćie go znośili.
Boć mám zá to / żem nie był w nicżem podlejƺy / niżeli oni bárdzo wielcy Apoſtołowie.
Bo choćiáżem też y proſtákiem w mowie / wƺákże nie w umiejętnośći ; ále zgołá jáwnymiśmy śię ſtáli we wƺyſtkich rzecżách u wáſ.
Izálim śię grzechu dopuśćił / żem ſámego śiebie uniżył / ábyśćie wy byli wywyżƺeni / á żem wám dármo Ewángieliję Bożą opowiádáł?
Złupiłem inne zbory / biorąc od nich żołd / ábym wám ſłużył ; á będąc u wáſ y ćierpiąc niedoſtátek / nie obćiążyłem próżnując nikogo.
Abowiem niedoſtátek mój dopełnili bráćiá / którzy przyƺli z Mácedonii / y we wƺyſtkim ſtrzegłem śię / ábym wám ćiężkim nie był / y ná potem ſtrzedz śię będę.
Jeſtći práwdá Chryſtuſowá we mnie / iż tá chlubá nie będzie zátłumioná we mnie w kráinách Achájſkich.
Dlácżegoż? cży że wáſ nie miłuję? Bógći wie.
Ale co cżynię / cżynić jeƺcże będę dlátego / ábym odćiął przycżynę tym / którzy przycżyny ƺukáją / áby w tem / z cżego śię chlubią / byli ználezieni tácy / jáko y my.
Abowiem tákowi fáłƺywi Apoſtołowie ſą robotnicy zdrádliwi / którzy śię przemieniáją w Apoſtoły Chryſtuſowe.
A nie dziw : bo y ƺátán ſám przemieniá śię w Aniołá świátłośći.
Nie wielká tedy / jeźli też ſłudzy jego przemieniáją śię / jákoby byli ſługámi ſpráwiedliwośći / których koniec będzie podług ucżynków ich.
Znowu powiádám / żeby mię kto nie miáł zá głupiego ; jeźliż inácżej / więc jáko głupiego przyjmijćie mię / ábym śię já też nieco málucżko przechwáláł.
Co mówię / nie mówięć jáko od Páná / ále jáko w głupſtwie z ſtrony tej bezpiecżnej chluby.
Poniewáż śię ich wiele chlubi według ćiáłá / y já śię chlubić będę.
Bo rádzi znośićie głupich / będąc ſámi mądrymi.
Bo znośićie / choć wáſ kto zniewálá / choć kto pożerá / choć kto bierze / choć śię kto wynośi / choć wáſ kto policżkuje.
Mówięc według zelżywośći / jákobyśmy byli ſłábymi ; lecż w cżem kto śmiáłym jeſt / ( w głupſtwie mówię ) y jám śmiáły.
Żydowie ſą / jeſtem y já. Izráelcżycy ſą / jeſtem y já. Náśieniem Abráhámowem ſą / jeſtem y já.
Sługámi Chryſtuſowymi ſą / ( głupio mówię / ) więcej já ; w prácách obfićiej / w rázách nád miárę / w więzieniách obfićiej / w śmierćiách cżęſtokroć.
Od Żydów wziąłem pięćiokroć po cżterdzieśći plág bez jednej.
Trzykroćiem był bity rózgámi ; rázem był kámionowány ; trzykroć śię ze mną okręt rozbił / dzień y noc byłem w głębokośći morſkiej ;
W drogách cżęſtokroć / w niebezpiecżeńſtwách ná rzekách / w niebezpiecżeńſtwách od zbójców / w niebezpiecżeńſtwách od ſwego národu / w niebezpiecżeńſtwách od pogán / w niebezpiecżeńſtwách w mieśćie / w niebezpiecżeńſtwách ná puƺcży / w niebezpiecżeńſtwách ná morzu / w niebezpiecżeńſtwách między fáłƺywymi bráćmi ;
W prácy y w utrudzeniu / w niedoſypiániu cżęſto / w głodzie y w prágnieniu / w poſtách cżęſto / w zimnie y w nágośći ;
Oprócż tego / co ſkąd inąd przypádá / ono nálegánie ná mię ná káżdy dzień / y ono ſtáránie o wƺyſtkie zbory.
Któż choruje / á já nie choruję? Któż śię gorƺy / á já nie pálę?
Jeźli śię mám chlubić / z krewkośći moich chlubić śię będę.
Bóg y Ojćiec Páná náƺego Jezuſá Chryſtuſá / który jeſt błogoſłáwiony ná wieki / wie / iż nie kłámię.
W Dámáƺku hetmán ludu królá Arety / oſádził był ſtráżą miáſto Dámáƺek / chcąc mię pojmáć ; álem oknem po powrozie w koƺyku przez mur był ſpuƺcżony / y uƺedłem rąk jego.
