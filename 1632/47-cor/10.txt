Já też Páweł ſám wáſ proƺę przez ćichość y dobrotliwość Chryſtuſową / który gdym jeſt wám przytomny / jeſtem pokorny między wámi ; lecż gdym nie jeſt przytomny / jeſtem śmiáły przećiwko wám.
A proƺę / ábym będąc przytomny / nie muśiáł być śmiáły tą śmiáłośćią / o której myślę / ábym śmiáły był przećiwko niektórym / którzy náſ ƺácują / jákobyśmy według ćiáłá chodzili.
Abowiem w ćiele chodząc / nie według ćiáłá wálcżymy /
( Abowiem broń żołnierſtwá náƺego nie jeſt ćieleſná / ále z Bogá jeſt / mocná ku zburzeniu miejſc obronnych. )
Burząc rády y wƺeláką wyſokość / wynoƺącą śię przećiwko znájomośći Bożej / y podbijájąc wƺeláką myśl pod poſłuƺeńſtwo Chryſtuſowe ;
Y pogotowiu májąc pomſtę ná wƺelákie niepoſłuƺeńſtwo / gdy śię wypełni poſłuƺeńſtwo wáƺe.
Ná toż tylko / co przed ocżymá jeſt / pátrzyćie? Máli kto tę nádzieję o ſobie / iż jeſt Chryſtuſowy / niechże też to ſám z śiebie uwáżá / iż jáko on jeſt Chryſtuſowy / ták też y my Chryſtuſowymy jeſteśmy.
Abowiem choćbym śię já też co więcej chełpił z mocy náƺej / którą nám dáł Pán ku zbudowániu á nie ku zepſowániu wáƺemu / nie záwſtydzę śię ;
Abym śię nie zdáł / jákobym wáſ ſtráƺył przez liſty.
Abowiem mówią : Liſty wáżne ſą y potężne / ále ćiáłá obecność nie potężná jeſt / y mowá niepłátná.
To niecháj myśli táki / iż jákimiśmy w mowie przez liſty / nie będąc obecnymi / tákimiż będziemy y w ucżynku / będąc obecnymi.
Abowiem nie śmiemy ſámych śiebie w pocżet drugich kłáść / álbo porównywáć z niektórymi / którzy ſámi śiebie zálecáją ; ále ći nie zrozumiewáją / iż śię ſámi ſobą miárkują y ſámi śię do śiebie przyrównywáją.
Ale my nie będziemy śię chlubili nád miárę / ále według ƺnuru miáry / którą miárę wymierzył nám Bóg / ták żeśmy dośięgli áż do wáſ /
Bo śię nie rozćiągámy nád miárę / jákobyśmy nie dośięgli áż do wáſ ; bośmy áż y do wáſ przyƺli w Ewángielii Chryſtuſowej.
A nie chlubimy śię nád miárę z cudzych prác ; ále májąc nádzieję / iż gdy śię pomnoży wiárá wáƺá w wáſ / pomnożymy śię y my między wámi według ƺnuru náƺego z obfitośćią /
Ku opowiádániu Ewángielii w onych kráinách / które leżą zá wámi / nie chlubiąc śię z rzecży gotowych cudzego pomiáru.
Kto śię tedy chlubi / niech śię Pánu chlubi.
Abowiem nie ten / co śię ſám zálecá / doświádcżony jeſt / ále ten / którego Pán zálecá.
