Ja też Páweł ſám was proƺę przez ćichość y dobrotliwość CHryſtuſową / który gdym jeſt wám przytomny / jeſtem pokorny między wámi ; lecż gdym nie jeſt przytomny / jeſtem śmiáły przećiwko wám.
A proƺę / ábym będąc przytomny / nie muśiáł być śmiáły tą śmiáłośćią / o którey myślę / ábym śmiáły był przećiwko niektórym / którzy nas ƺácują / jákobyśmy według ćiáłá chodźili.
Abowiem w ćiele chodząc / nie według ćiáłá wálcżymy /
( Abowiem broń żołnierſtwá náƺego nie jeſt ćieleſná / ále z Bogá jeſt / mocná ku zburzeniu miejſc obronnych. )
Burząc rády y wƺeláką wyſokość / wynoƺącą śię przećiwko znájomośći Bożej / y podbijájąc wƺeláką myśl pod poſłuƺeńſtwo CHryſtuſowe ;
Y pogotowiu májąc pomſtę ná wƺelákie niepoſłuƺeńſtwo / gdy śię wypełni poſłuƺeńſtwo wáƺe.
Ná toż tylko / co przed ocżymá jeſt / pátrzyćie? Máli kto tę nádźieję o ſobie / iż jeſt CHryſtuſowy / niechże też to ſám z śiebie uwáżá / iż jáko on jeſt CHryſtuſowy / ták też y my CHryſtuſowymy jeſteśmy.
Abowiem choćbym śię ja też co więcey chełpił z mocy náƺej / którą nám dáł Pán ku zbudowániu á nie ku zepſowániu wáƺemu / nie záwſtydzę śię ;
Abym śię nie zdáł / jákobym was ſtráƺył przez liſty.
Abowiem mówią : Liſty wáżne ſą y potężne / ále ćiáłá obecność nie potężná jeſt / y mowá niepłátná.
To niecháj myśli táki / iż jákimiśmy w mowie przez liſty / nie będąc obecnymi / tákimiż będźiemy y w ucżynku / będąc obecnymi.
Abowiem nie śmiemy ſámych śiebie w pocżet drugich kłáść / álbo porównywáć z niektórymi / którzy ſámi śiebie zálecáją ; ále ći nie zrozumiewáją / iż śię ſámi ſobą miárkują y ſámi śię do śiebie przyrównywáją.
Ale my nie będźiemy śię chlubili nád miárę / ále według ƺnuru miáry / którą miárę wymierzył nám Bóg / ták żeśmy dośięgli áż do was /
Bo śię nie rozćiągámy nád miárę / jákobyśmy nie dośięgli áż do was ; bośmy áż y do was przyƺli w Ewángielii CHryſtuſowej.
A nie chlubimy śię nád miárę z cudzych prác ; ále májąc nádźieję / iż gdy śię pomnoży wiárá wáƺá w was / pomnożymy śię y my między wámi według ƺnuru náƺego z obfitośćią /
Ku opowiádániu Ewángielii w onych kráinách / które leżą zá wámi / nie chlubiąc śię z rzecży gotowych cudzego pomiáru.
Kto śię tedy chlubi / niech śię Pánu chlubi.
Abowiem nie ten / co śię ſám zálecá / doświádcżony jeſt / ále ten / którego Pán zálecá.
