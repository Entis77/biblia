Te tedy obietnice májąc / namilƺy / ocżyśćiajmy ſámych śiebie od wƺelákiey zmázy ćiáłá y duchá / wykonywájąc poświęcenie w bojáźni Bożey.
Przyjmićież nas. Nikogoſmy nie ukrzywdźili / nikogoſmy nie uƺkodźili / nikogoſmy przez łákomſtwo nie podeƺli.
Nie mówięć <i>tego</i> ábym was potępiáć miał : bom przed tym powiedźiał / iż wy w ſercách náƺych ták jeſteśćie / <i>żebyſmy rádźi</i> z wámi ſpołecżnie umieráli y ſpołecżnie żyli.
Mam wielkie beſpiecżeńſtwo do mówienia u was / mam wielką chlubę z was / nápełnionym jeſt poćiechy / náder obfituję weſelem w káżdym ućiſku náƺym.
Abowiem gdyſmy przyƺli do Mácedoniey / ćiáło náƺe żadnego odpocżynku nie miáło / ále we wƺyſtkim <i>byliſmy</i> ućiśnieni : zewnątrz walki / á wewnątrz poſtráchy.
Ale który ćieƺy uniżone / poćieƺył nas Bóg przez przyśćie Tytuſowe.
A nie tylko przez przyśćie jego / ále też przez poćiechę którą on ućieƺony jeſt z was / oznájmiwƺy nam żądość wáƺę / nárzekánie wáƺe / gorliwość wáƺę zá mną ; ták / żem śię też więcey uweſelił.
Bo choćiam was záſmućił przez liſt / nie żal mi <i>tego,</i> choćiaż mi żal było : bo widzę iż ten liſt / choćiaż ná chwilę záſmućił was był :
Jednák teraz weſelę śię nie dla tego / żeśćie záſmuceni byli / ále żeśćie záſmuceni byli ku pokućie : ábowiem byliśćie záſmuceni według Bogá / żebyśćie niwcżym nie ƺkodowáli przez nas.
Abowiem ſmętek <i>który jeſt</i> według Bogá / pokutę ſpráwuje ku zbáwieniu / którey nikt nie żáłuje : ále ſmętek <i>według</i> świátá / ſpráwuje śmierć.
Bo oto / to ſámo żeśćie według Bogá byli záſmuceni / jáko wielką w was pilność ſpráwiło / owƺem obronę / owƺem zápálcżywość / owƺem bojaźń / owƺem żądość / owƺem gorliwość / owƺem pomſtę ; <i>ták iż</i> we wƺyſtkim okazáliśćie śię być cżyſtymi w tey ſpráwie.
A ták choćiáżem piſał do was / nie <i>piſałem</i> dla tego który krzywdę ucżynił / áni dla owego któremu śię krzywdá ſtáłá ; ále dla tego / iżby wam wiádoma byłá oná pilność náƺá o was przed oblicżnośćią Bożą.
Dla tegoſmy śię ućieƺyli z poćiechy wáƺey : áleſmy śię więcey ućieƺyli z weſela Tytuſowego / iż ochłodzony jeſt duch jego od was wƺyſtkich.
<i>A</i> iż jeſlim śię w cżym przed nim z was chlubił / nie záwſtydźiłem śię : ále jákoſmy wam prawdźiwie wƺyſtko mówili / ták śię też chlubá náƺá przed Tytuſem prawdźiwa pokazáłá.
A wnętrznośći jego tym więcey <i>ſkłonione ſą</i> ku wam / gdy wſpomina poſłuƺeńſtwo wƺyſtkich was / <i>y</i> jákośćie go z bojáźnią y ze drżeniem przyjęli.
Ráduję śię tedy / iż wam we wƺyſtkim mogę podufáć.
