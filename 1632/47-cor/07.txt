Te tedy obietnice májąc / nájmilśi! ocżyƺcżájmy ſámych śiebie od wƺelákiej zmázy ćiáłá y duchá / wykonywájąc poświęcenie w bojáźni Bożej.
Przyjmijćież nas ; nikogośmy nie ukrzywdźili / nikogośmy nie uƺkodźili / nikogośmy przez łákomſtwo nie podeƺli.
Nie mówięć tego / ábym was potępiáć miał ; bom przedtem powiedźiáł / iż wy w ſercách náƺych ták jeſteśćie / żebyśmy rádźi z wámi ſpołecżnie umieráli y ſpołecżnie żyli.
Mam wielkie bezpiecżeńſtwo do mówieniá u was / mam wielką chlubę z was / nápełnionym jeſt poćiechą / náder obfituję weſelem w káżdym ućiſku náƺym.
Abowiem gdyśmy przyƺli do Mácedonii / ćiáło náƺe żádnego odpocżynku nie miáło / ále we wƺyſtkiem byliśmy ućiśnieni / zewnątrz wálki / á wewnątrz poſtráchy.
Ale Bóg / który ćieƺy uniżonych / poćieƺył nas przez przyjśćie Tytuſowe ;
A nie tylko przez przyjśćie jego / ále też przez poćiechę / którą on ućieƺony jeſt z was / oznájmiwƺy nám żądość wáƺę / nárzekánie wáƺe / gorliwość wáƺę zá mną / ták / żem śię też więcey uweſelił.
Bo choćiáżem was záſmućił przez liſt / nie żál mi tego / choćiáż mi żál było ; bo widzę / iż ten liſt / choćiáż ná chwilę / záſmućił was był.
Jednák teraz weſelę śię / nie dlátego / żeśćie záſmuceni byli / ále żeśćie záſmuceni byli w pokućie ; ábowiem byliśćie záſmuceni według Bogá / żebyśćie w nicżem nie ƺkodowáli przez nas.
Abowiem ſmutek / który jeſt według Bogá / pokutę ſpráwuje ku zbáwieniu / którey nikt nie żáłuje ; ále ſmutek według świátá ſpráwuje śmierć.
Bo oto to ſámo / żeśćie według Bogá byli záſmuceni / jáko wielką w was pilność ſpráwiło / owƺem obronę / owƺem zápálcżywość / owƺem bojáźń / owƺem żądość / owƺem gorliwość / owƺem pomſtę / ták iż we wƺyſtkiem okázáliśćie śię być cżyſtymi w tey ſpráwie.
A ták choćiáżem piſáł do was / nie piſáłem dla tego / który krzywdę ucżynił / áni dla owego / któremu śię krzywdá ſtáłá / ále dla tego / iżby wám wiádomá byłá oná pilność náƺá o was przed oblicżnośćią Bożą.
Dlátegośmy śię ućieƺyli z poćiechy wáƺej ; áleśmy śię więcey ućieƺyli z weſelá Tytuſowego / y ochłodzony jeſt duch jego od was wƺyſtkich.
A iż jeſlim śię w cżem przed nim z was chlubił / nie záwſtydźiłem śię ; ále jákośmy wám práwdźiwie wƺyſtko mówili / ták śię też chlubá náƺá przed Tytuſem práwdźiwá pokázáłá.
A wnętrznośći jego tem więcey ſkłonione ſą ku wám / gdy wſpominá poſłuƺeńſtwo wƺyſtkich was / y jákośćie go bojáźnią y ze drżeniem przyjęli.
Ráduję śię tedy / iż wám we wƺyſtkiem mogę záufáć.
