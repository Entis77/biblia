Te tedy obietnice májąc / nájmilśi! ocżyƺcżájmy ſámych śiebie od wƺelákiej zmázy ćiáłá y duchá / wykonywájąc poświęcenie w bojáźni Bożej.
Przyjmijćież náſ ; nikogośmy nie ukrzywdźili / nikogośmy nie uƺkodźili / nikogośmy przez łákomſtwo nie podeƺli.
Nie mówięć tego / ábym wáſ potępiáć miáł ; bom przedtem powiedźiáł / iż wy w ſercách náƺych ták jeſteśćie / żebyśmy rádźi z wámi ſpołecżnie umieráli y ſpołecżnie żyli.
Mám wielkie bezpiecżeńſtwo do mówieniá u wáſ / mám wielką chlubę z wáſ / nápełnionym jeſt poćiechą / náder obfituję weſelem w káżdym ućiſku náƺym.
Abowiem gdyśmy przyƺli do Mácedonii / ćiáło náƺe żádnego odpocżynku nie miáło / ále we wƺyſtkiem byliśmy ućiśnieni / zewnątrz wálki / á wewnątrz poſtráchy.
Ale Bóg / który ćieƺy uniżonych / poćieƺył náſ przez przyjśćie Tytuſowe ;
A nie tylko przez przyjśćie jego / ále też przez poćiechę / którą on ućieƺony jeſt z wáſ / oznájmiwƺy nám żądość wáƺę / nárzekánie wáƺe / gorliwość wáƺę zá mną / ták / żem śię też więcej uweſelił.
Bo choćiáżem wáſ záſmućił przez liſt / nie żál mi tego / choćiáż mi żál było ; bo widzę / iż ten liſt / choćiáż ná chwilę / záſmućił wáſ był.
Jednák teráz weſelę śię / nie dlátego / żeśćie záſmuceni byli / ále żeśćie záſmuceni byli w pokućie ; ábowiem byliśćie záſmuceni według Bogá / żebyśćie w nicżem nie ƺkodowáli przez náſ.
Abowiem ſmutek / który jeſt według Bogá / pokutę ſpráwuje ku zbáwieniu / której nikt nie żáłuje ; ále ſmutek według świátá ſpráwuje śmierć.
Bo oto to ſámo / żeśćie według Bogá byli záſmuceni / jáko wielką w wáſ pilność ſpráwiło / owƺem obronę / owƺem zápálcżywość / owƺem bojáźń / owƺem żądość / owƺem gorliwość / owƺem pomſtę / ták iż we wƺyſtkiem okázáliśćie śię być cżyſtymi w tej ſpráwie.
A ták choćiáżem piſáł do wáſ / nie piſáłem dlá tego / który krzywdę ucżynił / áni dlá owego / któremu śię krzywdá ſtáłá / ále dlá tego / iżby wám wiádomá byłá oná pilność náƺá o wáſ przed oblicżnośćią Bożą.
Dlátegośmy śię ućieƺyli z poćiechy wáƺej ; áleśmy śię więcej ućieƺyli z weſelá Tytuſowego / y ochłodzony jeſt duch jego od wáſ wƺyſtkich.
A iż jeźlim śię w cżem przed nim z wáſ chlubił / nie záwſtydźiłem śię ; ále jákośmy wám práwdźiwie wƺyſtko mówili / ták śię też chlubá náƺá przed Tytuſem práwdźiwá pokázáłá.
A wnętrznośći jego tem więcej ſkłonione ſą ku wám / gdy wſpominá poſłuƺeńſtwo wƺyſtkich wáſ / y jákośćie go bojáźnią y ze drżeniem przyjęli.
Ráduję śię tedy / iż wám we wƺyſtkiem mogę záufáć.
