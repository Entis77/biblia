Lecż o uſłudze która śię dźieje świętym / nie potrzebna mi jeſt wam piſać.
Bo znam ochotę umyſłu wáƺego / którą śię ja chlubię z was u Mácedońcżyków : Yż Achája gotowa byłá od przeƺłego roku ; á tá wáƺá gorliwość wiele ich pobudźiłá.
Poſłałem tedy <i>tą</i> bráćią / żeby chlubá náƺá którą <i>mamy</i> z was / nie byłá dáremna z tey miáry ; <i>ále</i> ábyśćie ( jákom powiedźiał ) gotowymi byli.
Abyſmy ſnadź / jeſliby ze mną przyƺli Mácedońcżycy / á ználeſli was niegotowymi / nie záwſtydźili śię my / ( że nie rzekę wy ) zá ták beſpiecżną chlubę.
Zdáło mi śię tedy zá <i>rzecż</i> potrzebną nápomnieć bráćią / áby do was w przód poƺli / y pierwey zgotowáli przed tym opowiedźiáną wáƺę ƺcżodrobliwość ; áby byłá gotowa ták jáko ƺcżodrobliwość / á nie jáko rzecż przymuƺona.
Ale to <i>mówię</i> : Kto ſkąpie śieje / ſkąpie też żąć będźie : á kto obfićie śieje / obfićie też żąć będźie.
Káżdy jáko umyślił w ſercu ſwym / <i>ták niech ucżyni,</i> nie z zámarƺcżeniem áni z przymuƺenia ; ábowiem ochotnego dawcę Bóg miłuje.
A mocen jeſt Bóg ucżynić / áby obfitowáłá ná was wƺelka łáſká / ábyśćie májąc we wƺyſtkim záwƺe wƺeláki doſtátek / obfitowáli ku wƺelákiemu ucżynkowi dobremu ;
Jáko nápiſano : Rozproƺył / dał ubogim / ſpráwiedliwość jego zoſtawa ná wieki.
A ten który dawa naśienia śiejącemu ; Niechże y wam da chleb ku jedzeniu / y rozmnoży náśienie wáƺe / y przyſporzy urodzájów ſpráwiedliwośći wáƺey :
<i>Abyśćie</i> z káżdey miáry byli ubogáceni ku wƺelkiey proſtośći / która ſpráwuje przez nas / áby dźięki Bogu cżynione były.
Abowiem uſługowánie tey ofiáry / nie tylko dopełnia niedoſtátki świętych / ále też opływa przez wielkie dźiękcżynienia ná Bogá / przez pochwałę tey poſługi :
Gdy Bogá chwalą zá wáƺe poddáńſtwo Ewángeliey CHryſtuſowey / zá ƺcżerość w udźielániu przećiwko ſobie / y przećiwko wƺyſtkim <i>innym</i> :
Y modlą śię zá wámi ; żądájąc was dla łáſki Bożey obfitującey w was.
Lecż Bogu <i>niech będźie</i> chwałá zá niewypowiedźiány dar jego.
