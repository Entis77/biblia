Lecż o uſłudze / która śię dźieje świętym / nie potrzebná mi jeſt wám piſáć.
Bo znám ochotę umyſłu wáƺego / którą śię ja chlubię z was u Mácedońcżyków / iż Achájá gotowá byłá od przeƺłego roku ; á tá wáƺá gorliwość wiele ich pobudźiłá.
Poſłáłem tedy tych bráći / żeby chlubá náƺá / którą mámy z was / nie byłá dáremná z tey miáry / ále ábyśćie ( jákom powiedźiáł / ) gotowymi byli ;
Abyśmy ſnáć / jeſliby ze mną przyƺli Mácedońcżycy / á ználeſli was niegotowymi / nie záwſtydźili śię my / ( że nie rzekę / wy / ) zá ták bezpiecżną chlubę.
Zdáło mi śię tedy zá rzecż potrzebną / nápomnieć bráći / áby do was wprzód poƺli / y pierwej zgotowáli przedtem opowiedźiáną wáƺę ƺcżodrobliwość / áby byłá gotowá ták jáko ƺcżodrobliwość / á nie jáko rzecż przymuƺoná.
Ale ták mówię : Kto ſkąpo śieje / ſkąpo też żąć będźie ; á kto obfićie śieje / obfićie też żąć będźie.
Káżdy jáko umyślił w ſercu ſwem / ták niech ucżyni / nie z zámárƺcżeniem áni z przymuƺeniá ; ábowiem ochotnego dáwcę Bóg miłuje.
A mocen jeſt Bóg ucżynić / áby obfitowáłá ná was wƺelká łáſká / ábyśćie májąc we wƺyſtkiem záwƺe wƺeláki doſtátek / obfitowáli ku wƺelákiemu ucżynkowi dobremu /
Jáko nápiſáne : Rozproƺył / dáł ubogim / ſpráwiedliwość jego zoſtáje ná wieki.
A ten / który dáje náśienie śiejącemu / niechże y wám dá chleb ku jedzeniu / y rozmnoży náśienie wáƺe / y przyſporzy urodzájów ſpráwiedliwośći wáƺej /
Abyśćie z káżdej miáry byli ubogáceni ku wƺelkiej proſtośći / która ſpráwuje przez nas / áby dźięki Bogu cżynione były.
Abowiem uſługiwánie tey ofiáry nie tylko dopełniá niedoſtátki świętych / ále też opływá przez wielkie dźiękcżynieniá ná Bogá przez pochwáłę tey poſługi ;
Gdy Bogá chwálą zá wáƺe poddáńſtwo Ewángielii CHryſtuſowej / zá ƺcżerość w udźielániu przećiwko ſobie y przećiwko wƺyſtkim innym ;
Y modlą śię zá wámi / żądájąc was dla łáſki Bożej obfitującey w was.
Lecż Bogu niech będźie chwałá zá niewypowiedźiány dár jego.
