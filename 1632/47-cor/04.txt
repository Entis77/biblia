Dlátego májąc to uſługiwánie / ták jákośmy miłośierdzie otrzymáli / nie ſłábiejemy.
Aleśmy śię odrzekli ſkrytej ſromoty / nie obchodząc śię chytrze / áni fáłƺując ſłowá Bożego ; ále objáwieniem práwdy zálecájąc ſámych śiebie u káżdego ſumieniá ludzkiego przed oblicżem Bożem.
Jeźli tedy zákrytá jeſt Ewángielijá náƺá / zákrytá jeſt przed tymi / którzy giną.
W których bóg świátá tego oślepił zmyſły / to jeſt w niewiernych / áby im nie świećiłá świátłość Ewángielii chwáły Chryſtuſowej / który jeſt wyobráżeniem Bożem.
Abowiem nie ſámych śiebie opowiádámy / ále Chryſtuſá Jezuſá / że jeſt Pánem / á ſámych śiebie ſługámi wáƺymi dlá Jezuſá.
Poniewáż Bóg / który rzekł / áby śię z ćiemnośći świátłość rozświećiłá / ten śię rozświećił w ſercách náƺych ku rozświeceniu ( w náſ ) znájomośći chwáły Bożej w oblicżu Jezuſá Chryſtuſá.
A mámy ten ſkárb w nácżyniu gliniánem / áby doſtojność tej mocy byłá z Bogá / á nie z náſ.
Gdy zewƺąd ućiśnieni bywámy / ále nie bywámy potłocżeni ; powątpiewámy / ále nie zwątpiámy.
Prześládowánie ćierpimy / ále nie bywámy opuƺcżeni ; bywámy porzuceni / ále nie giniemy.
Záwƺe umártwienie Páná Jezuſowe ná ćiele nośimy / áby y żywot Jezuſowy ná ćiele náƺem był objáwiony.
Záwƺe bowiem my / którzy żyjemy / bywámy wydáwáni ná śmierć dlá Jezuſá / áby też żywot Jezuſowy był objáwiony w śmiertelnem ćiele náƺem.
Dlátego śmierć mocy ſwojej w náſ dokázuje / ále w wáſ żywot.
Májąc tedy tegoż duchá wiáry / ták jáko nápiſáne : Uwierzyłem / przetom też mówił ; y my wierzymy / przeto też mówimy /
Wiedząc / iż ten / który wzbudził Páná Jezuſá / y náſ wzbudzi przez Jezuſá / y poſtáwi z wámi.
Boć to wƺyſtko dzieje śię dlá wáſ / áby łáſká oná obfitującá przez dziękowánie wielu ich rozmnożyłá śię ku chwále Bożej.
Dlátego nie ſłábiejmy / ále choć śię też náƺ zewnętrzny cżłowiek kázi / wƺákże on wewnętrzny odnáwiá śię ode dniá do dniá.
Abowiem ten króćiuchny y lekki ućiſk náƺ náder zácnej chwáły wiecżną wágę nám ſpráwuje ;
Gdy nie pátrzymy ná rzecży widziálne / ále ná niewidziálne ; ábowiem rzecży widziálne ſą docżeſne / ále niewidziálne ſą wiecżne.
