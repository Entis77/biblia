A poſtánowiłem to u śiebie / ábym znowu nie przyƺedł z záſmuceniem do was.
Bo jeſlibym ja was záſmućił / y któż jeſt / coby mię rozweſelił / tylko ten / który jeſt przez mię záſmucony?
A tomći wám nápiſáł / ábym przyƺedłƺy / nie miał ſmutku z tych / z którychbym śię miał weſelić ; pewien będąc o was wƺyſtkich / że rádość moję wƺyſcy zá ſwoję máćie.
Abowiem z wielkiego ućiſku y utrápieniá ſercá / y z wielą łez piſáłem wám / nie żebyśćie mieli być záſmuceni / ále żebyśćie miłość poználi / którą náder obfićie mam przećiwko wám.
Jeſli tedy kto záſmućił / nie mnieć záſmućił / ále poniekąd ( ábym go nie obćiążył / ) was wƺyſtkich.
Doſyćći má táki ná zgromieniu tem / które śię ſtáło od wielu /
Ták iż przećiwnym obycżájem / inácżejbyśćie wy mu odpuśćić mieli / y onego poćieƺyć / by ſnáć zbytni ſmutek tákiego nie pożárł.
Przetoż proƺę was / ábyśćie miłość przećiwko niemu utwierdźili /
Abowiem y dlátegom był nápiſáł / ábym doświádcżeniá wáƺego doznał / jeſliże we wƺyſtkiem poſłuƺni jeſteśćie.
A komu wy co odpuƺcżáćie / temu y ja ; gdyż y ja / jeſlim co odpuśćił temu / komum odpuśćił / ucżyniłem to dla was przed oblicżnośćią CHryſtuſową / áby was ƺátán nie podƺedł.
Abowiem zámyſły jego nie ſą nám tájne.
A gdym przyƺedł do Troády dla opowiádániá Ewángielii CHryſtuſowej / á drzwi mi były otworzone w Pánu.
Nie miáłem ulżeniá w duchu moim / dlátego / żem nie ználázł Tytuſá / brátá mego ; ále rozſtáwƺy śię z nimi / poƺedłem do Mácedonii.
Lecż chwałá Bogu / który nám záwƺe zwyćięſtwo dáje w CHryſtuśie / y wonność znájomośći ſwojey objáwiá przez nas ná káżdem miejſcu.
Bośmy dobrą wonnośćią CHryſtuſową Bogu w tych / którzy zbáwieni bywáją / y w tych / którzy giną ;
Tymći wonnośćią śmierći ná śmierć / ále onym wonnośćią żywotá ku żywotowi ; lecż do tego któż jeſt ſpoſobny?
Bo nie jeſteśmy jáko wiele ich / którzy fáłƺują ſłowo Boże ; ále jáko z ƺcżerośći / ále jáko z Bogá / przed oblicżnośćią Bożą o CHryſtuśie mówimy.
