A poſtánowiłem to u śiebie / ábym znowu nie przyƺedł z záſmuceniem do wáſ.
Bo jeźlibym já wáſ záſmućił / y któż jeſt / coby mię rozweſelił / tylko ten / który jeſt przez mię záſmucony?
A tomći wám nápiſáł / ábym przyƺedłƺy / nie miáł ſmutku z tych / z którychbym śię miáł weſelić ; pewien będąc o wáſ wƺyſtkich / że rádość moję wƺyſcy zá ſwoję máćie.
Abowiem z wielkiego ućiſku y utrápieniá ſercá / y z wielą łez piſáłem wám / nie żebyśćie mieli być záſmuceni / ále żebyśćie miłość poználi / którą náder obfićie mám przećiwko wám.
Jeźli tedy kto záſmućił / nie mnieć záſmućił / ále poniekąd ( ábym go nie obćiążył / ) wáſ wƺyſtkich.
Doſyćći má táki ná zgromieniu tem / które śię ſtáło od wielu /
Ták iż przećiwnym obycżájem / inácżejbyśćie wy mu odpuśćić mieli / y onego poćieƺyć / by ſnáć zbytni ſmutek tákiego nie pożárł.
Przetoż proƺę wáſ / ábyśćie miłość przećiwko niemu utwierdzili /
Abowiem y dlátegom był nápiſáł / ábym doświádcżeniá wáƺego doznáł / jeźliże we wƺyſtkiem poſłuƺni jeſteśćie.
A komu wy co odpuƺcżáćie / temu y já ; gdyż y já / jeźlim co odpuśćił temu / komum odpuśćił / ucżyniłem to dlá wáſ przed oblicżnośćią Chryſtuſową / áby wáſ ƺátán nie podƺedł.
Abowiem zámyſły jego nie ſą nám tájne.
A gdym przyƺedł do Troády dlá opowiádániá Ewángielii Chryſtuſowej / á drzwi mi były otworzone w Pánu.
Nie miáłem ulżeniá w duchu moim / dlátego / żem nie ználázł Tytuſá / brátá mego ; ále rozſtáwƺy śię z nimi / poƺedłem do Mácedonii.
Lecż chwáłá Bogu / który nám záwƺe zwyćięſtwo dáje w Chryſtuśie / y wonność znájomośći ſwojej objáwiá przez náſ ná káżdem miejſcu.
Bośmy dobrą wonnośćią Chryſtuſową Bogu w tych / którzy zbáwieni bywáją / y w tych / którzy giną ;
Tymći wonnośćią śmierći ná śmierć / ále onym wonnośćią żywotá ku żywotowi ; lecż do tego któż jeſt ſpoſobny?
Bo nie jeſteśmy jáko wiele ich / którzy fáłƺują ſłowo Boże ; ále jáko z ƺcżerośći / ále jáko z Bogá / przed oblicżnośćią Bożą o Chryſtuśie mówimy.
