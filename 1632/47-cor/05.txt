Wiemy bowiem / że jeźli tego náƺego źiemſkiego mieƺkániá námiot zburzony będźie / budowánie mámy od Bogá / dom nie rękomá urobiony / wiecżny w niebieśiech.
Abowiem w tym námioćie wzdychámy / domem náƺym / który jeſt z niebá / żądájąc być przyoblecżeni.
Jeźliż tylko przyoblecżonymi á nie nágimi ználeźieni będźiemy.
Bo którzyśmy w tym námioćie / wzdychámy / będąc obćiążeni / poniewáż nie żądámy być zewlecżeni / ále przyoblecżeni / áby pożártá byłá śmiertelność od żywotá.
A ten / który náſ ku temuż włáśnie przygotowáł / jeſtći Bóg / który nám też dáł zádátek Duchá.
Przetoż májąc záwƺe ufność / y wiedząc / że póki mieƺkámy w tem ćiele / pielgrzymujemy od Páná :
( Bo przez wiárę chodźimy / á nie przez widzenie. )
Lecż ufámy / y wolimy rácżej wynijść z ćiáłá / á iść ná mieƺkánie do Páná.
Przetoż też uśiłujemy / bądź w ćiele mieƺkámy / bądź z ćiáłá wychodźimy / onemu śię podobáć.
Abowiem muśimy śię wƺyſcy pokázáć przed ſądową ſtolicą CHryſtuſową / áby káżdy odnióſł / co cżynił w ćiele / według tego / co cżynił / lub dobre / lub złe.
Przetoż wiedząc o tym ſtráchu Páńſkim / ludźi do wiáry námáwiámy / á Bogu jáwnymi jeſteśmy ; y mám nádźieję / iż w ſumieniách wáƺych jáwnymi jeſteśmy.
Abowiem nie ſámych śiebie wám znowu zálecámy / ále wám dájemy przycżynę / ábyśćie śię chlubili námi / y żebyśćie mieli co mówić przećiwko tym / którzy śię chlubią z powierzchownych rzecży á nie z ſercá.
Bo choć od rozumu odchodźimy / Bogu odchodźimy / choć przy rozumie jeſteśmy / wám jeſteśmy.
Abowiem miłość CHryſtuſowá przyćiſká náſ / jáko tych / którzyśmy to oſądźili / iż poniewáż jeden zá wƺyſtkich umárł / tedy wƺyſcy byli umárłymi ;
A że zá wƺyſtkich umárł / áby ći / którzy żyją / już więcej ſobie nie żyli / ále temu / który zá nich umárł / y jeſt wzbudzony.
Dlátego my od tego cżáſu nikogo według ćiáłá nie známy / á choćiáśmy też ználi CHryſtuſá według ćiáłá / lecż już teráz więcej nie známy.
A ták jeźli kto jeſt w CHryſtuśie / nowem jeſt ſtworzeniem ; ſtáre rzecży przeminęły / oto śię wƺyſtkie nowemi ſtáły.
A to wƺyſtko z Bogá jeſt / który náſ z ſámym ſobą pojednáł przez JEzuſá CHryſtuſá / y dáł nám uſługiwánie tego pojednániá /
Poniewáż Bóg był w CHryſtuśie / świát z ſámym ſobą jednájąc / nie przycżytując im upádków ich / y położył w náſ to ſłowo pojednániá :
Przetoż ná miejſcu CHryſtuſowem poſelſtwo ſpráwujemy / jákoby wáſ Bóg upomináł przez náſ / prośimy ná miejſcu CHryſtuſowem / jednájćie śię z Bogiem ;
Abowiem on tego / który nie znáł grzechu / zá náſ grzechem ucżynił / ábyśmy śię my ſtáli ſpráwiedliwośćią Bożą w nim.
