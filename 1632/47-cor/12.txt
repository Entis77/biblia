Wprawdźieć mi śię chlubić nie jeſt pożytecżno : Jednák przydę do widzenia y objáwienia PAńſkiego.
Znam cżłowieká w CHryſtuśie przed cżternaſtą lat ( jeſli <i>śię to dźiáło</i> w ćiele / nie wiem ; jeſli prócż ćiáłá / nie wiem ; Bóg wie : ) który záchwycony był áż do trzećiego niebá.
A znam tákiego cżłowieká ( jeſli <i>śię to dźiáło</i> w ćiele / jeſli prócż ćiáłá / nie wiem ; Bóg wie : )
Iż był záchwycony do Ráju : y ſłyƺał niewypowiedźiáne ſłowá / których śię cżłowiekowi nie godźi mówić.
Z tákiego <i>cżłowieká</i> chlubić śię będę : ále z śiebie ſámego chlubić śię nie będę / tylko z krewkośći mojich.
Abowiem jeſlibym śię chćiał chlubić / nie będę głupim / bo prawdę powiem : Ale wſtrzymam śię : áby kto o mnie nie rozumiał nád to / cżym mię być widźi / álbo co ſłyƺy ode mnie.
A iżbym śię zacnośćią objáwienia náder nie wynośił / dány mi jeſt bodźiec ćiáłu / ánjoł ƺátan / áby mię policżkował / żebym śię nád miárę nie wynośił.
Dla tegom trzy kroć PAná prośił / áby to odſtąpiło ode mnie.
Ale mi rzekł : Doſyć maƺ ná łáſce mojey ; ábowiem moc mojá wykonywa śię w ſłábośći. Rácżey śię tedy więcey chlubić będę z krewkośći mojich / áby we mnie mieƺkáłá moc CHryſtuſowá.
Dla tego śię kocham w krewkośćiách / w potwarzách / w niedoſtátkách / w prześládowániách / <i>y</i> w ućiſkách / dla CHryſtuſá. Bo gdym jeſt ſłáby / tedym jeſt mocny.
Stałem śię głupim chlubiąc śię : wyśćie mię <i>do tego</i> przymuśili. Bom ja od was miał być chwalony ; ponieważem nie był podlejƺym niżeli oni bárzo wielcy Apoſtołowie / choćiażem nic nie jeſt.
Jednák znáki Apoſtołá pokazáne ſą u was / we wƺelkiey ćierpliwośći / w známioniách / y w cudách / y w mocách.
Bo cóż jeſt w cżymbyśćie podlejƺy byli nád inƺe Zbory / tylko żem ja ſam próżnując / nie obćiążył was? Odpuśććież mi tę krzywdę.
Oto trzeći raz jeſtem gotów przyść do was / á nie obćiążę was próznując. Abowiem nie ƺukam tego co jeſt wáƺego / ále was <i>ſámych</i> : boć nie dźiatki máją rodźicom ſkárbić / ále rodźicy dźiatkom.
Lecż ja bárzo rad nákład ucżynię / y ſámego śiebie wynáłożę zá duƺę wáƺę : ácżkolwiek bárzo was miłując / mniey bywam <i>od was</i> miłowány.
Ale niech ták będźie ; Jam was nie obćiążył / tylko chytrym będąc / zdrádam was pojimał.
Izalim was przez którego z tych / którem do was poſłał / oƺukał?
Uprośiłem Tytuſá y poſłałem z nim brátá <i>tego</i>. Izali was Tytus oƺukał? Izaliſmy nie jednym duchem poſtępowáli? Izali nie jednymi ſtopámi?
Znowuż mniemaćie / że śię przed wámi obmawiamy? Przed oblicżnośćią Bożą w CHryſtuśie mówimy : á to wƺyſtko / namilƺy / dla wáƺego zbudowánia.
Bo śię boję / bym ſnadź przyƺedƺy nie ználazł was tákimi / jákimibym nie chćiał : á ja też żebym nie był ználeźiony od was / jákiegobyśćie wy nie chćieli : by ſnadź <i>miedzy wámi nie było</i> ſwarów / zazdrośći / gniewów / zwad / obmowiſk / mrucżenia / nádymánia / <i>y</i> roſterków.
By mię záśię Bóg mój gdy przydę nie poniżył u was / y żáłowałbym wiela tych / którzy przed tym grzeƺyli / á nie pokutowáli z niecżyſtośći / y z wƺetecżeńſtwá / y z rozpuſty / którą popełnili.
