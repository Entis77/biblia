Potem rzekłá do niej Noemi / świekrá jej : Córko mojá / ázáżemći nie powinná ƺukáć odpocżnieniá / żebyś śię dobrze miáłá?
A teráz ázáż Booz nie jeſt powinowátym náƺym / z któregoś ty ſłużebnicámi byłá? Oto on będźie wiáł jęcżmień ná bojewiſku tej nocy.
Przetoż umywƺy śię / námáż śię olejkámi ; weźmij też ƺáty twoje ná śię / á idź ná bojewiſko / á nie dáj śię widźieć mężowi onemu / áżby śię nájádł y nápił.
A gdy on ſpáć pójdźie / upátrzże miejſce / ná którem śię ukłádźie / á przyƺedłƺy odkryjeƺ płáƺcż z nóg jego / á tám śię ukłádźieƺ / á on tobie oznájmi / co będźieƺ miáłá cżynić.
Y rzekłá do niej Rut : Cokolwiek mi káżeƺ / ucżynię.
A ták ƺłá ná one bojewiſko / y ucżyniłá to / co jej rozkázáłá świekrá jej.
A gdy śię nájádł Booz y nápił / y rozweſeliło śię ſerce jego / poƺedł á ukłádł śię przy ſtogu ; przyƺłá też y oná po ćichu / á odkrywƺy płáƺcż z nóg jego / ukłádłá śię.
A gdy było o północy / uląkł śię on mąż / á obróćiwƺy śię / ujrzáł / á oto / niewiáſtá leży u nóg jego.
Y rzekł : Któżeś ty? Y odpowiedźiáłá : Jám jeſt Rut / ſłużebnicá twojá ; rozćiągnijże płáƺcż twój ná ſłużebnicę twoję / boś mi pokrewny.
A on rzekł : Błogoſłáwionáś ty od Páná / córko mojá ; więkƺąś pobożność po ſobie pokázáłá teráz niż pierwej / żeś nie poƺłá zá młodźieńcámi ták ubogimi jáko y bogátymi ;
Przetoż teráz / córko mojá / nie bój śię ; bo wƺyſtko / cokolwiek rzecżeƺ / ucżynię / gdyż wie cáłe miáſto ludu mego / żeś ty niewiáſtá cnotliwá.
A teráz práwdáć to / żem já jeſt pokrewny twój / á wƺákże jeƺcże jeſt pokrewny bliżƺy nád mię.
Zoſtáńże tu tej nocy. A gdy będźie ráno / jeźli ćię będźie chćiáł pojąć práwem bliſkośći / dobrze / niech pojmie ; jeźli ćię nie będźie chćiáł pojąć / já ćię pojmę práwem bliſkośći ; żywie Pán! Śpijże tu áż do poránku.
A ták ſpáłá u nóg jego áż do poránku / á wſtáłá przedtem niż mógł rozeznáć jeden drugiego ; bo rzekł Booz : Niech nikt nie wie / że przyƺłá tá niewiáſtá ná bojewiſko.
Nádto rzekł : Dáj płáchtę / którą máƺ ná ſobie / á trzymáj ją ; á gdy ją trzymáłá / námierzył jej ƺeść miárek jęcżmieniá / y záłożył ná nię / y weƺłá do miáſtá.
A przyƺłá do świekry ſwej / którá jej ſpytáłá : Któżeś ty córko mojá? A Rut jej powiedźiáłá wƺyſtko / co jej ucżynił on mąż /
Y rzekłá : Oto ƺeść miárek tego jęcżmieniá dáł mi ; bo rzekł do mnie : Nie wróćiƺ śię próżno do świekry twojej.
Y rzekłá Noemi : Potrwájże / córko mojá / áż śię dowieƺ / jáko pádnie tá rzecż ; boć nie zániechá ten mąż / áż tę rzecż dźiś ſkońcży.
