Potem rzekłá do niej Noemi / świekrá jey : Córko mojá / ázáżemći nie powinná ƺukáć odpocżnieniá / żebyś śię dobrze miáłá?
A teraz ázaż Booz nie jeſt powinowátym náƺym / z któregoś ty ſłużebnicámi byłá? Oto on będźie wiał jęcżmień ná bojewiſku tey nocy.
Przetoż umywƺy śię / námáż śię olejkámi ; weźmij też ƺáty twoje ná śię / á idź ná bojewiſko / á nie dáj śię widźieć mężowi onemu / áżby śię nájádł y nápił.
A gdy on ſpáć pójdźie / upátrzże miejſce / ná którem śię ukłádźie / á przyƺedƺy odkryjeƺ płáƺcż z nóg jego / á tám śię ukłádźieƺ / á on tobie oznájmi / co będźieƺ miáłá cżynić.
Y rzekłá do niej Rut : Cokolwiek mi káżeƺ / ucżynię.
A ták ƺłá ná one bojewiſko / y ucżyniłá to / co jey rozkázáłá świekrá jey.
A gdy śię nájádł Booz y nápił / y rozweſeliło śię ſerce jego / poƺedł á ukłádł śię przy ſtogu ; przyƺłá też y oná po ćichu / á odkrywƺy płáƺcż z nóg jego / ukłádłá śię.
A gdy było o północy / uląkł śię on mąż / á obróćiwƺy śię / ujrzał / á oto / niewiáſtá leży u nóg jego.
Y rzekł : Któżeś ty? Y odpowiedźiáłá : Jam jeſt Rut / ſłużebnicá twojá ; rozćiągnijże płáƺcż twój ná ſłużebnicę twoję / boś mi pokrewny.
A on rzekł : Błogoſłáwionáś ty od Páná / córko mojá ; więkƺąś pobożność po ſobie pokázáłá teraz niż pierwej / żeś nie poƺłá zá młodźieńcámi ták ubogimi jáko y bogátymi ;
Przetoż teraz / córko mojá / nie bój śię ; bo wƺyſtko / cokolwiek rzecżeƺ / ucżynię / gdyż wie cáłe miáſto ludu mego / żeś ty niewiáſtá cnotliwá.
A teraz práwdáć to / żem ja jeſt pokrewny twój / á wƺákże jeƺcże jeſt pokrewny bliżƺy nád mię.
Zoſtáńże tu tey nocy. A gdy będźie ráno / jeſli ćię będźie chćiał pojąć práwem bliſkośći / dobrze / niech pojmie ; jeſli ćię nie będźie chćiał pojąć / ja ćię pojmę práwem bliſkośći ; żywie Pán! Śpijże tu áż do poránku.
A ták ſpáłá u nóg jego áż do poránku / á wſtáłá przed tym niż mógł rozeznáć jeden drugiego ; bo rzekł Booz : Niech nikt nie wie / że przyƺłá tá niewiáſtá ná bojewiſko.
Nádto rzekł : Dáj płáchtę / którą maƺ ná ſobie / á trzymáj ją ; á gdy ją trzymáłá / námierzył jey ƺeść miárek jęcżmieniá / y záłożył ná nię / y weƺłá do miáſtá.
A przyƺłá do świekry ſwej / która jey ſpytáłá : Któżeś ty córko mojá? A Rut jey powiedźiáłá wƺyſtko / co jey ucżynił on mąż /
Y rzekłá : Oto ƺeść miárek tego jęcżmieniá dał mi ; bo rzekł do mnie : Nie wróćiƺ śię prózno do świekry twojey.
Y rzekłá Noemi : Potrwájże / córko mojá / áż śię dowieƺ / jáko pádnie tá rzecż ; boć nie zániechá ten mąż / áż tę rzecż dźiś ſkońcży.
