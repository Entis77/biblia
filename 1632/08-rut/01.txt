Y ſtáło śię zá onych cżáſów / kiedy ſędźiowie ſądźili / był głód w źiemi ; y poƺedł niektóry mąż z Betlehem Judá ná mieƺkánie do źiemi Moábſkiej z żoną ſwoją y z dwomá ſynámi ſwymi.
A imię mężá onego było Elimelech / imię też żony jego Noemi / tákże imioná dwóch ſynów jego Máhálon y Chelijon / á ći byli Efrátejcżykámi z Betlehem Judá / którzy záƺedłƺy do kráiny Moábſkiej / mieƺkáli tám.
Potem umárł Elimelech / mąż Noemi / á oná pozoſtáłá z dwomá ſynámi ſwoimi.
Y pojęli ſobie żony Moábſkie ; imię jednej Orfá / á drugiej imię Rut ; y mieƺkáli tám około dźieśięćiu lát.
Umárli potem y oni obá / Máhálon y Chelijon ; y ták oná niewiáſtá ośieroćiáłá po obu ſynách ſwoich y po mężu ſwoim.
A wezbráwƺy śię z ſynowemi ſwemi wróćiłá śię z źiemi Moábſkiej ; bo ſłyƺáłá w kráinie Moábſkiej / że był náwiedźił Pán lud ſwój / y dáł im chleb.
Tedy wyƺłá z miejſcá / ná którem byłá z onemi dwiemá ſynowemi ſwemi / á udáły śię w drogę / áby śię wróćiły do źiemi Judá.
Zátem rzekłá Noemi do dwóch ſynowych ſwych : Idźćież / wróććie śię káżdá do domu mátki twojej ; niecháj ucżyni Pán z wámi miłośierdźie / jákośćie ucżyniły z umárłymi ſynámi moimi y ze mną.
Niech wám dá Pán ználeść odpocżnienie / káżdej w domu mężá ſwego ; y pocáłowáłá je / á one podnióſłƺy głoſ ſwój / płákáły /
Y mówiły do niej : Rácżej śię z tobą wróćimy do ludu twojego.
A Noemi rzekłá : Wróććież śię córki moje ; przecżżebyśćie ze mną iść miáły? Azáż já jeƺcże mogę mieć ſyny / którzyby byli mężámi wáƺymi?
Wróććież śię córki moje / á idźćie / bom śię już zſtárzáłá / á nie mogę iść zá mąż. Choćbym też rzekłá / jeƺcże mám nádźieję / álbo choćbym też dobrze tej nocy byłá zá mężem choćbym też náwet y porodźiłá ſyny ;
Izáli wy ich cżekáć będźiećie / áżby dorośli? záż śię dlá tego zátrzymáćie / ábyśćie nie ƺły zá mąż? Nie ták córki moje ; bo żáłość mojá więkƺá jeſt niźli wáƺá / gdyż śię obróćiłá przećiwko mnie ręká Páńſká.
Ale one podnióſłƺy głoſ ſwój / znowu płákáły. Y pocáłowáłá Orfá świekrę ſwoję ; á Rutá zoſtáłá przy niej.
Do której Noemi rzekłá : Oto śię wróćiłá powinná twojá do ludu ſwego / y do bogów ſwoich / wróćże śię y ty zá powinną ſwoją.
Ná co jej odpowiedźiáłá Rutá : Nie wiedź mię do tego / ábym ćię opuśćić y od ćiebie odejść miáłá ; owƺem gdźiekolwiek pójdźieƺ / z tobą pójdę / á gdźiekolwiek mieƺkáć będźieƺ / z tobą mieƺkáć będę ; lud twój lud mój / á Bóg twój Bóg mój.
Gdźie umrzeƺ / tám y já umrę / y tám pogrzebioná będę. To mi niech ucżyni Pán / y to niech przepuśći ná mię / że tylko śmierć rozłącży mię z tobą.
A ták oná widząc / że śię ná to upárłá / áby z nią ƺłá / przeſtáłá jej odrádzáć.
Y ƺły obie poſpołu / áż przyƺły do Betlehem. Y ſtáło śię / gdy przyƺły do Betlehem / wzruƺyło śię wƺyſtko miáſto dlá nich / mówiąc : Izáż nie tá jeſt Noemi?
Ale oná mówiłá do nich : Nie názywájćie mię Noemi / ále mię zowćie Márá ; ábowiem mię gorzkośćią wielką Wƺechmogący nápełnił.
Wyƺłám ſtąd obfitą á próżną mię przywróćił Pán. Przecżże mię tedy zowiećie Noemi / gdyż mię Pán utrápił / á Wƺechmogący złe ná mię dopuśćił?
A ták wróćiłá śię Noemi y Rut Moábitká / ſynowá jej / z nią ; wróćiłá śię z kráiny Moábſkiej / y przyƺły do Betlehem ná pocżątku żniwá jęcżmiennego.
