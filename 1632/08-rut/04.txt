Potem Booz ƺedł do brámy / y uśiádł tám ; á oto / pokrewny on ƺedł mimo / o którym powiedziáł był Booz ; y rzekł mu : Pójdź ſám á śiądź tu / ty á ty ; á on przyƺedłƺy śiádł.
Wziąwƺy tedy dzieśięć mężów ſtárƺych miáſtá onego / mówił do nich : Siądźćież też tu ; y uśiedli.
Zátem rzekł onemu powinowátemu : Dziáł roli / który był brátá náƺego Elimelechá / ſprzedáłá Noemi / którá śię wróćiłá z ziemi Moábſkiej.
Y zdáło mi śię to odnieść do uƺu twoich / mówiąc : Otrzymáj tę rolę przed tymi / którzy tu śiedzą / y przed ſtárƺymi ludu mego ; á chceƺli ją odkupić / odkup ; á jeźliż nie odkupiƺ / powiedz mi ; bo wiem / że nád ćię niemáƺ bliżƺego do wykupieniá / á jám po tobie. Tedy on rzekł : Já odkupię.
Nádto rzekł Booz : Dniá / którego otrzymáƺ rolą z rąk Noemi / tedy też y Rutę Moábitkę / żonę zmárłego / pojmieƺ / ábyś wzbudził imię zmárłego w dziedzictwie jego.
Odpowiedziáł powinowáty : Nie mogę odkupić / bym ſnáć nie ſtráćił dziedzictwá mego. Odkupże ty ſobie bliſkość moję / gdyż já nie mogę odkupić jej.
( A był to ſtárodáwny zwycżáj w Izráelu przy wykupnie / y przy zámiánie / áby wárowniejƺá byłá káżdá ſpráwá / tedy zzuwáł jeden z nich trzewik ſwój / y dáwáł go bliźniemu ſwojimu ; á toć było ná świádectwo uſtępowániá dóbr w Izráelu. )
Tedy rzekł on powinowáty do Boozá : Otrzymájże ty ; y zzuł trzewik ſwój.
Zátem rzekł Booz do onych ſtárƺych / y do wƺyſtkiego ludu : Świádkámi dziś jeſteśćie wy / żem to wƺyſtko otrzymáł / co było Elimelechowe / y to wƺyſtko / co było Chelijonowe / y Máhálonowe / z rąk Noemi.
Do tego Rutę Moábitkę / żonę Máchálonowę / wziąłem ſobie zá żonę / ábym wzbudził imię zmárłego w dziedzictwie jego / iżby nie zginęło imię onego zmárłego między bráćią jego / y z brámy miejſcá jego ; tego wyśćie dziśiáj świádkámi.
Y rzekł wƺyſtek lud / który był w brámie miáſtá / y ſtárśi : Jeſteśmy świádkámi ; niech ći dá Pán / áby niewiáſtá / którá wchodzi w dom twój / byłá jáko Ráchel y jáko Lijá / które obie zbudowáły dom Izráelſki. Pocżynájże ſobie mężnie w Efráćie / á zjednáj ſobie imię w Betlehemie.
Niechájże dom twój będzie jáko dom Fáreſá / ( którego porodziłá Támár Judzie / ) z náśieniá tego / któreć dá Pán z tej to biáłej głowy.
A ták pojął ſobie Booz Rutę / y byłá mu zá żonę ; á gdy wƺedł do niej / tedy jej dáł Pán / że pocżęłá / y porodziłá ſyná.
Y rzekły niewiáſty do Noemi : Błogoſłáwiony Pán / który ćię dziś nie chćiáł mieć bez powinowátego / áby zoſtáło imię jego w Izráelu.
Tenći ućieƺy duƺę twoję / y będzie ćię żywił w ſtárośći twojej ; ábowiem ſynowá twojá / którá ćię miłuje / porodziłá go / któráć dáleko jeſt lepƺá / niżeli śiedm ſynów.
A ták wziąwƺy Noemi dziećiątko / położyłá je ná łonie ſwojim / á byłá mu zá piáſtunkę.
Y dáły mu ſąśiády imię / mówiąc : Národził śię ſyn Noemi / y názwáły imię jego Obed ; tenći jeſt ojćiec Iſájego ojcá Dawidowego.
A teć ſą rodzáje Fáreſowe : Fáreſ ſpłodził Heſroná ;
A Heſron ſpłodził Rámá / á Rám ſpłodził Aminádábá ;
A Aminádáb ſpłodził Náháſoná / á Náháſon ſpłodził Sálmoná ;
A Sálmon ſpłodził Boozá / á Booz ſpłodził Obedá ;
A Obed ſpłodził Iſájego / á Iſáj ſpłodził Dawidá.
