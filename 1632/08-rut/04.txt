Potem Booz ƺedł do bramy / y uśiádł tám ; á oto / pokrewny on ƺedł mimo / o którym powiedźiał był Booz ; y rzekł mu : Pójdź ſám á śiądź tu / ty á ty ; á on przyƺedƺy śiádł.
Wźiąwƺy tedy dźieśięć mężów ſtárƺych miáſtá onego / mówił do nich : Siądźćież też tu ; y uśiedli.
Zátem rzekł onemu powinowátemu : Dźiał roli / który był brátá náƺego Elimelechá / ſprzedáłá Noemi / która śię wróćiłá z źiemi Moábſkiej.
Y zdáło mi śię to odnieść do uƺu twojich / mówiąc : Otrzymáj tę rolę przed tymi / którzy tu śiedzą / y przed ſtárƺymi ludu mego ; á chceƺli ją odkupić / odkup ; á jeſliż nie odkupiƺ / powiedz mi ; bo wiem / że nád ćię niemáƺ bliżƺego do wykupieniá / á jam po tobie. Tedy on rzekł : Ja odkupię.
Nádto rzekł Booz : Dniá / którego otrzymáƺ rolą z rąk Noemi / tedy też y Rutę Moábitkę / żonę zmárłego / pojmieƺ / ábyś wzbudźił imię zmárłego w dźiedźictwie jego.
Odpowiedźiał powinowáty : Nie mogę odkupić / bym ſnadź nie ſtráćił dźiedźictwá mego. Odkupże ty ſobie bliſkość moję / gdyż ja nie mogę odkupić jey.
( A był to ſtárodáwny zwycżáj w Izráelu przy wykupnie / y przy zámiánie / áby wárowniejƺá byłá káżda ſpráwá / tedy zzuwał jeden z nich trzewik ſwój / y dáwał go bliźniemu ſwojimu ; á toć było ná świádectwo uſtępowánia dóbr w Izráelu. )
Tedy rzekł on powinowáty do Boozá : Otrzymájże ty ; y zzuł trzewik ſwój.
Zátem rzekł Booz do onych ſtárƺych / y do wƺyſtkiego ludu : Świádkámi dźiś jeſteśćie wy / żem to wƺyſtko otrzymał / co było Elimelechowe / y to wƺyſtko / co było Chelijonowe / y Máhálonowe / z rąk Noemi.
Do tego Rutę Moábitkę / żonę Máchálonowę / wźiąłem ſobie zá żonę / ábym wzbudźił imię zmárłego w dźiedźictwie jego / iżby nie zginęło imię onego zmárłego miedzy bráćią jego / y z bramy miejſcá jego ; tego wyśćie dźiśiáj świádkámi.
Y rzekł wƺyſtek lud / który był w bramie miáſtá / y ſtárśi : Jeſteśmy świádkámi ; niech ći da Pán / áby niewiáſtá / która wchodźi w dom twój / byłá jáko Ráchel y jáko Lijá / które obie zbudowáły dom Izráelſki. Pocżynájże ſobie mężnie w Efráćie / á zjednáj ſobie imię w Betlehemie.
Niechájże dom twój będźie jáko dom Fáreſá / ( którego porodźiłá Támár Judźie / ) z náśieniá tego / któreć da Pán z tey to biáłej głowy.
A ták pojął ſobie Booz Rutę / y byłá mu zá żonę ; á gdy wƺedł do niej / tedy jey dał Pán / że pocżęłá / y porodźiłá ſyná.
Y rzekły niewiáſty do Noemi : Błogoſłáwiony Pán / który ćię dźiś nie chćiał mieć bez powinowátego / áby zoſtáło imię jego w Izráelu.
Tenći ućieƺy duƺę twoję / y będźie ćię żywił w ſtárośći twojey ; ábowiem ſynowá twojá / która ćię miłuje / porodźiłá go / któráć dáleko jeſt lepƺá / niżeli śiedm ſynów.
A ták wźiąwƺy Noemi dźiećiątko / położyłá je ná łonie ſwojim / á byłá mu zá piáſtunkę.
Y dáły mu ſąśiády imię / mówiąc : Národźił śię ſyn Noemi / y názwáły imię jego Obed ; tenći jeſt oćiec Iſájego ojcá Dawidowego.
A teć ſą rodzáje Fáreſowe : Fáres ſpłodźił Heſroná ;
A Heſron ſpłodźił Rámá / á Rám ſpłodźił Aminádábá ;
A Aminádáb ſpłodźił Náháſoná / á Náháſon ſpłodźił Sálmoná ;
A Sálmon ſpłodźił Boozá / á Booz ſpłodźił Obedá ;
A Obed ſpłodźił Iſájego / á Iſáj ſpłodźił Dawidá.
