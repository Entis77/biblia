A Noemi miáłá powinowátego po mężu ſwym / cżłowieká możnego z domu Elimelechowego / którego zwano Booz.
Y rzekłá Rutá Moábitká do Noemi : Pójdę proƺę ná pole / á niech zbierám kłoſy zá tym / przed którego ocżymá łáſkę znájdę ; á oná rzekłá : Idź / córko mojá.
Szłá tedy / á przyƺedƺy zbieráłá ná polu zá żeńcámi ; y tráfiło śię / że przyƺłá ná dźiał polá Boozowego / który był z domu Elimelechowego.
A wtem przyƺedł Booz z Betlehem / y rzekł do żeńców : Pán z wámi. A oni mu odpowiedźieli : Niechżeć Pán błogoſłáwi.
Rzekł tedy Booz do ſługi ſwego / który był przyſtáwem nád żeńcámi : Cżyjáż to dźiewecżká?
Y odpowiedźiał mu ſługá on / który był przyſtáwem nád żeńcámi / y rzekł : Tá dźiewecżká jeſt Moábitká / która przyƺłá z Noemi z źiemi Moábſkiej.
Y rzekłá mi : Niech proƺę zbierám y zgromádzám kłoſy miedzy ſnopámi zá żeńcámi ; á przyƺedƺy báwi śię tu od ſámego poránku áż dotąd / á bárzo máło w domu śiedźi.
Tedy rzekł Booz do Rut : Słucháj mię córko mojá ; nie chodź zbieráć kłoſów ná inƺe pole y nie odchodź ſtąd / ále śię tu trzymáj dźiewek mojich.
Pilnuj tego polá / ná którem żąć będą / á chodź zá nimi ; bom rozkázał ſługom moim / żeby śię ćiebie żáden nie tykał ; á jeſli uprágnieƺ / idź do nacżynia / á nápij śię z tego / co cżerpáją ſłudzy moi.
Tedy oná upádƺy ná oblicże ſwoje / ukłoniwƺy śię áż do źiemi rzekłá do niego : Skądżem ználázłá łáſkę w ocżách twojich / iż mię znáƺ / gdyżem jeſt cudzoźiemká?
Y odpowiedźiał Booz / á rzekł jey : Powiedźiáno mi zápewne wƺyſtko / coś ucżyniłá świekrze twojey po śmierći mężá twego / á jákoś opuśćiwƺy ojcá twego y mátkę twoję / y źiemię / w którejś śię urodźiłá / przyƺłá do ludu / któregoś nie znáłá przed tym.
Niechżeć oddá Pán ucżynek twój / y niech będźie zápłátá twojá doſkonáłá od Páná / Bogá Izráelſkiego / gdyżeś przyƺłá / ábyś nádźieję miáłá pod ſkrzydłámi jego.
A oná rzekłá : Ználázłám łáſkę w ocżách twojich / pánie mój / gdyżeś mię poćieƺył / á mówiłeś do ſercá ſłużebnicy twojey / choćiám ja nie jeſt / jáko jedná z ſłużebnic twojich.
Y rzekł jey Booz : Gdy będźie cżás jedzeniá / przychodź tu / á jedz chleb / omocżywƺy ƺtucżkę twoję w ocćie. Y uśiádłá przy żeńcách / y podał jey práżmá / które jádłá áż do ſytośći / y jeƺcże jey zbyło.
Potem wſtáłá / áby zbieráłá ; á Booz rozkázał ſługom ſwoim mówiąc : Niech y miedzy ſnopámi zbierá / á nie brońćie jey tego.
Owƺem umyślnie upuƺcżájćie jey z ſnopów / y zoſtáwiájćie / áby zbieráłá / á nie fukájćie ná nią.
A ták zbieráłá ná onem polu áż do wiecżorá ; y wymłóćiłá to / co zebráłá / y miáłá jákoby z efę jęcżmieniá.
A wźiąwƺy to / ƺłá do miáſtá / y oglądáłá świekrá jey to / co názbieráłá ; á wyjąwƺy dáłá jey y to / co jey zoſtáło / gdy śię nájádłá.
Y rzekłá do niej świekrá jey : Kędyżeś dźiś zbieráłá / á gdźieś robiłá? niechájże ten / który ná ćię miał bácżenie / błogoſłáwionym będźie. Y oznájmiłá świekrze ſwej / u kogo robiłá / mówiąc : Imię mężá / u któregom dźiś robiłá / Booz.
Potem rzekłá Noemi do ſynowy ſwojey : Niech będźie błogoſłáwionym od Páná / który nie záwśćiągnął miłośierdźiá ſwego od żywych y od umárłych. Nádto jeƺcże rzekłá Noemi : Ten mąż jeſt powinowátym náƺym / y z pokrewnych náƺych.
Rzekłá jey też Rut Moábitká : Nádto mi jeƺcże mówił on mąż : Trzymáj śię cżeládźi mojey / póki nie pożną wƺyſtkiego zbożá mego.
Tedy rzekłá Noemi do Ruty / ſynowy ſwej : Dobrze / córko mojá / iż będźieƺ chodźiłá z dźiewkámi jego / żebyć kto przećiwnym nie był ná polu innem.
Przetoż śię trzymáłá ſłużebnic Boozowych / y zbieráłá kłoſy / póki śię nie ſkońcżyło żniwo jęcżmienne / y żniwo pƺenicżne. Potem mieƺkáłá u świekry ſwojey.
