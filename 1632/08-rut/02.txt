A Noemi miáłá powinowátego po mężu ſwym / cżłowieká możnego z domu Elimelechowego / którego zwáno Booz.
Y rzekłá Rutá Moábitká do Noemi : Pójdę proƺę ná pole / á niech zbierám kłoſy zá tym / przed którego ocżymá łáſkę znájdę ; á oná rzekłá : Idź / córko mojá.
Szłá tedy / á przyƺedłƺy zbieráłá ná polu zá żeńcámi ; y tráfiło śię / że przyƺłá ná dziáł polá Boozowego / który był z domu Elimelechowego.
A wtem przyƺedł Booz z Betlehem / y rzekł do żeńców : Pán z wámi. A oni mu odpowiedzieli : Niechżeć Pán błogoſłáwi.
Rzekł tedy Booz do ſługi ſwego / który był przyſtáwem nád żeńcámi : Cżyjáż to dziewecżká?
Y odpowiedziáł mu ſługá on / który był przyſtáwem nád żeńcámi / y rzekł : Tá dziewecżká jeſt Moábitká / którá przyƺłá z Noemi z ziemi Moábſkiej.
Y rzekłá mi : Niech proƺę zbierám y zgromádzám kłoſy między ſnopámi zá żeńcámi ; á przyƺedłƺy báwi śię tu od ſámego poránku áż dotąd / á bárdzo máło w domu śiedzi.
Tedy rzekł Booz do Rut : Słucháj mię córko mojá ; nie chodź zbieráć kłoſów ná inƺe pole y nie odchodź ſtąd / ále śię tu trzymáj dziewek moich.
Pilnuj tego polá / ná którem żąć będą / á chodź zá nimi ; bom rozkázáł ſługom moim / żeby śię ćiebie żáden nie tykáł ; á jeźli uprágnieƺ / idź do nácżyniá / á nápij śię z tego / co cżerpáją ſłudzy moi.
Tedy oná upádłƺy ná oblicże ſwoje / ukłoniwƺy śię áż do ziemi rzekłá do niego : Skądżem ználázłá łáſkę w ocżách twoich / iż mię znáƺ / gdyżem jeſt cudzoziemká?
Y odpowiedziáł Booz / á rzekł jej : Powiedziáno mi zápewne wƺyſtko / coś ucżyniłá świekrze twojej po śmierći mężá twego / á jákoś opuśćiwƺy ojcá twego y mátkę twoję / y ziemię / w którejś śię urodziłá / przyƺłá do ludu / któregoś nie znáłá przedtem.
Niechżeć oddá Pán ucżynek twój / y niech będzie zápłátá twojá doſkonáłá od Páná / Bogá Izráelſkiego / gdyżeś przyƺłá / ábyś nádzieję miáłá pod ſkrzydłámi jego.
A oná rzekłá : Ználázłám łáſkę w ocżách twoich / pánie mój / gdyżeś mię poćieƺył / á mówiłeś do ſercá ſłużebnicy twojej / choćiám já nie jeſt / jáko jedná z ſłużebnic twoich.
Y rzekł jej Booz : Gdy będzie cżáſ jedzeniá / przychodź tu / á jedz chleb / omocżywƺy ƺtucżkę twoję w ocćie. Y uśiádłá przy żeńcách / y podáł jej práżmá / które jádłá áż do ſytośći / y jeƺcże jej zbyło.
Potem wſtáłá / áby zbieráłá ; á Booz rozkázáł ſługom ſwoim mówiąc : Niech y między ſnopámi zbierá / á nie brońćie jej tego.
Owƺem umyślnie upuƺcżájćie jej z ſnopów / y zoſtáwiájćie / áby zbieráłá / á nie fukájćie ná nią.
A ták zbieráłá ná onem polu áż do wiecżorá ; y wymłóćiłá to / co zebráłá / y miáłá jákoby z efę jęcżmieniá.
A wziąwƺy to / ƺłá do miáſtá / y oglądáłá świekrá jej to / co názbieráłá ; á wyjąwƺy dáłá jej y to / co jej zoſtáło / gdy śię nájádłá.
Y rzekłá do niej świekrá jej : Kędyżeś dziś zbieráłá / á gdzieś robiłá? niechájże ten / który ná ćię miáł bácżenie / błogoſłáwionym będzie. Y oznájmiłá świekrze ſwej / u kogo robiłá / mówiąc : Imię mężá / u któregom dziś robiłá / Booz.
Potem rzekłá Noemi do ſynowy ſwojej : Niech będzie błogoſłáwionym od Páná / który nie záwśćiągnął miłośierdziá ſwego od żywych y od umárłych. Nádto jeƺcże rzekłá Noemi : Ten mąż jeſt powinowátym náƺym / y z pokrewnych náƺych.
Rzekłá jej też Rut Moábitká : Nádto mi jeƺcże mówił on mąż : Trzymáj śię cżeládzi mojej / póki nie pożną wƺyſtkiego zbożá mego.
Tedy rzekłá Noemi do Ruty / ſynowy ſwej : Dobrze / córko mojá / iż będzieƺ chodziłá z dziewkámi jego / żebyć kto przećiwnym nie był ná polu innem.
Przetoż śię trzymáłá ſłużebnic Boozowych / y zbieráłá kłoſy / póki śię nie ſkońcżyło żniwo jęcżmienne / y żniwo pƺenicżne. Potem mieƺkáłá u świekry ſwojej.
