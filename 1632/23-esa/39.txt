Onego cżáſu poſłáł Merodách Báládán / ſyn Báládánowy / król Bábilońſki / liſt y dáry do Ezechyjáƺá ; bo záſłyƺáł / że záchorowáwƺy záś ozdrowiáł.
Y weſelił śię z tego Ezechyjáƺ / y ukázáł im ſkárbnicę klejnotów ſwoich / ſrebrá y złotá / y rzecży wonnych / y olejki nájwyborniejƺe / tákże y dom rynƺtunków ſwoich / y cokolwiek śię znájdowáło w ſkárbách jego : nie było nic / cżegoby im nie ukázáł Ezechyjáƺ w domu ſwym / y we wƺyſtkiem páńſtwie ſwojim.
Wtem przyƺedł Izájáƺ prorok do królá Ezechyjáƺá / y rzekł mu : Coć powiedźieli ći mężowie / y ſkąd przyƺli do ćiebie? Y odpowiedźiáł Ezechyjáƺ : Z źiemi dálekiej przyƺli do mnie / z Bábilonu.
Nádto rzekł : Cóż widźieli w domu twoim? Odpowiedźiáł Ezechyjáƺ : Wƺyſtko / co jeſt w domu moim / widźieli ; niemáƺ nic / cżegobym im nie ukázáł w ſkárbách moich.
Tedy rzekł Izájáƺ do Ezechyjáƺá : Słucháj ſłowá Páná záſtępów :
Oto przyjdą te dni / w które zábiorą wƺyſtko do Bábilonu / cokolwiek jeſt w domu twoim / y cokolwiek záchowáli ojcowie twoi áż do dniá tego ; nie zoſtánie nic / mówi Pán ;
Ale y ſynów twoich / którzy wyjdą z ćiebie / których ſpłodźiƺ / pobiorą / y będą komornikámi ná dworze królá Bábilińſkiego.
Tedy rzekł Ezechyjáƺ do Izájáƺá : Dobre jeſt ſłowo Páńſkie / któreś mówił ; ( y dołożył : Dobre / ) przeto / że pokój y práwdá będźie zá dni moich.
