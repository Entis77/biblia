Onego cżáſu poſłał Merodách Báládán / ſyn Báládánowy / król Bábilońſki / liſt y dáry do Ezechyjáƺá ; bo záſłyƺał / że záchorowáwƺy záś ozdrowiał.
Y weſelił śię z tego Ezechyjáƺ / y ukázał im ſkárbnicę klejnotów ſwojich / ſrebrá y złotá / y rzecży wonnych / y olejki nájwyborniejƺe / tákże y dom rynƺtunków ſwojich / y cokolwiek śię znájdowáło w ſkárbách jego : nie było nic / cżegoby im nie ukázał Ezechyjáƺ w domu ſwym / y we wƺyſtkim páńſtwie ſwojim.
Wtem przyƺedł Izájáƺ prorok do królá Ezechyjáƺá / y rzekł mu : Coć powiedźieli ći mężowie / y ſkąd przyƺli do ćiebie? Y odpowiedźiał Ezechyjáƺ : Z źiemi dálekiej przyƺli do mnie / z Bábilonu.
Nádto rzekł : Cóż widźieli w domu twoim? Odpowiedźiał Ezechyjáƺ : Wƺyſtko / co jeſt w domu moim / widźieli ; niemáƺ nic / cżegobym im nie ukázał w ſkárbách mojich.
Tedy rzekł Izájáƺ do Ezechyjáƺá : Słucháj ſłowá Páná zaſtępów :
Oto przyjdą te dni / w które zábiorą wƺyſtko do Bábilonu / cokolwiek jeſt w domu twoim / y cokolwiek záchowáli ojcowie twoi áż do dniá tego ; nie zoſtánie nic / mówi Pán ;
Ale y ſynów twojich / którzy wyjdą z ćiebie / których ſpłodźiƺ / pobiorą / y będą komornikámi ná dworze królá Bábilońſkiego.
Tedy rzekł Ezechyjáƺ do Izájáƺá : Dobre jeſt ſłowo Páńſkie / któreś mówił ; ( y dołożył : Dobre / ) przeto / że pokój y práwdá będźie zá dni mojich.
