Y ſtło śię cżternáſtego roku królowániá Ezechyjáƺá / że przyćiągnął Sennácheryb / król Aſſyryjſki / przećiwko wƺyſtkim miáſtom Judzkim obronnym / y pobráł je.
Y poſłáł król Aſſyryjſki Rábſáceſá z Láchyſ / do Jeruzálemu / do królá Ezechyjáƺá z wielkiem wojſkiem / który ſtánął u rur ſádzáwki wyżƺej przy drodze polá blechárzowego.
Tedy wyƺedł do niego Elijákim / ſyn Helkijáƺowy / przełożony nád domem / y Sobná piſárz / y Joách / ſyn Aſáfowy / kánclerz.
Y rzekł do nich Rábſáceſ : Proƺę / powiedzćie Ezechyjáƺowi : Ták mówi król wielki / król Aſſyryjſki : Cóż to zá ufność / którą ufáƺ?
Rzekłbym : ( Acż to rzecż dáremná ) Snáć rády y mocy do wojny doſyć máƺ ; ále w kimże ufáƺ / że mi śię ſprzećiwiáƺ?
Otoś ſpoległ ná láſce tej trzćiny náłámánej / ná Egipćie / którą jeźliby śię kto podpárł / wnijdzie w rękę jego / y przekole ją. Tákić jeſt Fáráo / król Egipſki / wƺyſtkim / którzy w nim ufáją.
A jeźli mi rzecżeƺ : W Pánu / Bogu náƺym / ufámy ; ázáż nie ten jeſt / którego znióſł Ezechyjáƺ wyżyny y ołtárze / y przykázáł Judzie y Jeruzálemowi mówiąc : Przed tym ołtárzem kłániáć śię będziećie?
Przetoż teráz proƺę / záręcż śię Pánu memu / królowi Aſſyryjſkiemu / á já tobie dám dwá tyśiące koni / będzieƺli je mógł oſádzić jezdnymi.
Y jákoż śię ty możeƺ oprzeć hetmánowi jednemu nájmniejƺemu z ſług páná mego / choć ufáƺ w Egipćie dlá wozów y jezdnych?
Nádto cży bez woli Páńſkiej przyćiągnąłem do tej ziemi / ábym ją ſpuſtoƺył? Pán rzekł do mnie : Ciągnij do tej ziemi / á ſpuſtoƺ ją.
Tedy rzekł Elijákim / y Sobná / y Joách do Rábſáceſá : Proƺę / mów do ſług twoich po ſyryjſku / wƺák rozumiemy / á nie mów do náſ po żydowſku przed tym ludem / który jeſt ná murze.
Y odpowiedziáł Rábſáceſ : Azáż mię do Páná twego álbo do ćiebie poſłáł Pán mój / ábym te ſłowá mówił? Poſłáł mię rácżej do mężów / którzy śiedzą ná murze / áby jedli łájná ſwoje / á mocż ſwój poſpołu z wámi pili.
A ták ſtánął Rábſáceſ y wołáł głoſem wielkim po żydowſku / mówiąc : Słuchájćie ſłów królá wielkiego / królá Aſſyryjſkiego.
Ták mówi król : Niech wáſ nie zwodzi Ezechyjáƺ : bo wáſ nie będzie mógł wybáwić.
A niech wám nie rozkázuje Ezechyjáƺ ufáć w Pánu / mówiąc : Zápewne náſ Pán wybáwi / á nie będzie to miáſto podáne w rękę królá Aſſyryjſkiego.
Nie ſłuchájćież Ezechyjáƺá ; ábowiem ták powiedziáł król Aſſyryjſki : Ucżyńćie zemną przymierze / á wynijdźćie do mnie / á jedz káżdy z wáſ z winnicy ſwojej / y káżdy z figowego drzewá ſwego / á pij káżdy z wáſ wodę z ſtudni ſwojej ;
Aż przyjdę á pobiorę wáſ do ziemi podobnej ziemi wáƺej / do ziemi zbożá y winá / do ziemi chlebá y winnic.
Niech wáſ nie zwodzi Ezechyjáƺ / mówiąc : Pán náſ wybáwi. Izáż mogli bogowie národów wybáwić káżdy ziemię ſwoję z ręki królá Aſſyryjſkiego?
Gdzież ſą bogowie Emát y Arfád? Gdzie ſą bogowie Sefárwáim? Azáż wybáwił Sámáryję z ręki mojej?
Którzyż ſą między wƺyſtkimi bogámi tych ziem / którzyby wydárli ziemię ſwoję z ręki mojej? A miáłby Pán wybáwić Jeruzálem z ręki mojej?
Ale oni milcżeli / y nie odpowiedzieli mu y ſłowá ; bo tákie było rozkázánie królewſkie / mówiąc : Nie odpowiádájćie mu.
Y przyƺedł Elijákim / ſyn Helkijáƺowy / przełożony domu / y Sobná piſárz / y Joách / ſyn Aſáfowy / kánclerz / do Ezechyjáƺá / rozdárłƺy ƺáty ſwe / y oznájmili mu ſłowá Rábſáceſowe.
