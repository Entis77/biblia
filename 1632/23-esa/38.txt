W one dni záchorował Ezechyjáƺ áż ná śmierć. Y przyƺedł do niego Izájáƺ prorok / ſyn Amoſowy / á rzekł do niego : Ták mówi Pán : Rozpráw dom ſwój ; ábowiem umrzeƺ / á nie zoſtánieƺ żyw.
Tedy obróćił Ezechyjáƺ twárz ſwoję do śćiány / á modlił śię Pánu.
Y rzekł : Proƺę / o Pánie! wſpomnij teraz / żem chodźił przed tobą w práwdźie y w ſercu uprzejmem / cżyniąc to / co dobrego jeſt w ocżách twojich. Y płákał Ezechyjáƺ płácżem wielkim.
Y ſtáło śię ſłowo Páńſkie do Izájáƺá / mówiąc :
Idź / á powiedz Ezechyjáƺowi : Ták mówi Pán / Bóg Dawidá / ojcá twego : Wyſłucháłem modlitwę twoję / widźiáłem łzy twoje ; oto Ja przycżynię do dni twojich piętnáśćie lat ;
Y z ręki królá Aſſyryjſkiego wyrwę ćiebie y to miáſto / á będę bronił miáſtá tego.
A to będźieƺ miał zá znák od Páná / że Pán ucżyni to / co mówił.
Oto Ja wrócę názád ćień po ſtopniách / po których ƺedł / ná zegárze ſłonecżnym Acházowym ná dźieśięć ſtopni po tychże ſtopniách / po których było zeƺło.
Piſánie Ezechyjáƺá / królá Judzkiego / gdy był záchorował y wyzdrowiał z niemocy ſwojey :
Jam rzekł w ukróceniu dni mojich : Wnijdę do brám grobu / pozbáwion będę oſtátká lat ſwojich ;
Rzekłem / że nie ujrzę Páná / Páná w źiemi żyjących ; nie oglądám więcey cżłowieká miedzy obywátelámi ná świećie.
Pobyt mój pomijá / á przenośi śię odemnie / jáko námiot páſterſki ; oderznąłem żywot ſwój / jáko tkácż ; od krośien oderznie mię ; dźiś / pierwej niż noc nádejdźie / dokonáƺ mię.
Rozmyśláłem ſobie z poránku / że jáko lew potrze wƺyſtkie kośći moje / dźiś / pierwej niż noc nádejdźie / dokonáƺ mię.
Jáko żóráw y jáſkółká ƺcżebiotáłem / ſtękáłem jáko gołębicá ; ocży moje ku górze podnieśione były / y rzekłem : Pánie! gwáłt ćierpię / przedłuż mi żywotá.
Ale cóż mam więcey rzec? Onći mi odpowiedźiał / y ſám ucżynił / że żyć będę mimo wƺyſtkie látá ſwe po gorzkośći duƺy mojey.
Pánie! kto po nich y w nich żyć będźie / wƺyſtkim znájomy będźie żywot dychánia mego / żeś mi zdrowie przywróćił / á záchowáłeś mię przy żywoćie.
Oto cżáſu pokoju przyƺłá ná mię byłá gorzkość nájgorzćiejƺá ; ále śię tobie podobáło wyrwáć duƺę moję z przepáśći ſkáżeniá / przeto / żeś zárzućił w tył ſwój wƺyſtkie grzechy moje.
Abowiem nie grób wyſłáwiá ćię / áni śmierć chwáli ćię / áni ći / którzy w dół wſtępują / ocżekują prawdy twojey.
Żywy / żywy / ten ćię wyſłáwiáć będźie / jáko ja dźiśiáj / á oćiec ſynom oznájmi práwdę twoję.
Pán mię wybáwił ; przetoż pieśń moję śpiewáć będźiemy po wƺyſtkie dni żywotá náƺego w domu Páńſkim.
Y rzekł był Izájáƺ : Niech wezmą bryłę ſuchych fig / y przyłożą ná wrzód / á będźie uzdrowiony.
Y rzekł był Ezechyjáƺ : Cóż jeſt zá znák / że wſtąpię do domu Páńſkiego?
