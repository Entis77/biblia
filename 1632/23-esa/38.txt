W one dni záchorowáł Ezechyjáƺ áż ná śmierć. Y przyƺedł do niego Izájáƺ prorok / ſyn Amoſowy / á rzekł do niego : Ták mówi Pán : Rozpráw dom ſwój ; ábowiem umrzeƺ / á nie zoſtánieƺ żyw.
Tedy obróćił Ezechyjáƺ twárz ſwoję do śćiány / á modlił śię Pánu.
Y rzekł : Proƺę / o Pánie! wſpomnij teráz / żem chodził przed tobą w práwdzie y w ſercu uprzejmem / cżyniąc to / co dobrego jeſt w ocżách twoich. Y płákáł Ezechyjáƺ płácżem wielkim.
Y ſtáło śię ſłowo Páńſkie do Izájáƺá / mówiąc :
Idź / á powiedz Ezechyjáƺowi : Ták mówi Pán / Bóg Dawidá / ojcá twego : Wyſłucháłem modlitwę twoję / widziáłem łzy twoje ; oto Já przycżynię do dni twoich piętnáśćie lát ;
Y z ręki królá Aſſyryjſkiego wyrwę ćiebie y to miáſto / á będę bronił miáſtá tego.
A to będzieƺ miáł zá znák od Páná / że Pán ucżyni to / co mówił.
Oto Já wrócę názád ćień po ſtopniách / po których ƺedł / ná zegárze ſłonecżnym Acházowym ná dzieśięć ſtopni po tychże ſtopniách / po których było zeƺło.
Piſánie Ezechyjáƺá / królá Judzkiego / gdy był záchorowáł y wyzdrowiáł z niemocy ſwojej :
Jám rzekł w ukróceniu dni moich : Wnijdę do brám grobu / pozbáwion będę oſtátká lát ſwoich ;
Rzekłem / że nie ujrzę Páná / Páná w ziemi żyjących ; nie oglądám więcej cżłowieká między obywátelámi ná świećie.
Pobyt mój pomijá / á przenośi śię odemnie / jáko námiot páſterſki ; oderznąłem żywot ſwój / jáko tkácż ; od krośien oderznie mię ; dziś / pierwej niż noc nádejdzie / dokonáƺ mię.
Rozmyśláłem ſobie z poránku / że jáko lew potrze wƺyſtkie kośći moje / dziś / pierwej niż noc nádejdzie / dokonáƺ mię.
Jáko żóráw y jáſkółká ƺcżebiotáłem / ſtękáłem jáko gołębicá ; ocży moje ku górze podnieśione były / y rzekłem : Pánie! gwáłt ćierpię / przedłuż mi żywotá.
Ale cóż mám więcej rzec? Onći mi odpowiedziáł / y ſám ucżynił / że żyć będę mimo wƺyſtkie látá ſwe po gorzkośći duƺy mojej.
Pánie! kto po nich y w nich żyć będzie / wƺyſtkim znájomy będzie żywot dychániá mego / żeś mi zdrowie przywróćił / á záchowáłeś mię przy żywoćie.
Oto cżáſu pokoju przyƺłá ná mię byłá gorzkość nájgorzćiejƺá ; ále śię tobie podobáło wyrwáć duƺę moję z przepáśći ſkáżeniá / przeto / żeś zárzućił w tył ſwój wƺyſtkie grzechy moje.
Abowiem nie grób wyſłáwiá ćię / áni śmierć chwáli ćię / áni ći / którzy w dół wſtępują / ocżekują práwdy twojej.
Żywy / żywy / ten ćię wyſłáwiáć będzie / jáko já dziśiáj / á ojćiec ſynom oznájmi práwdę twoję.
Pán mię wybáwił ; przetoż pieśń moję śpiewáć będziemy po wƺyſtkie dni żywotá náƺego w domu Páńſkim.
Y rzekł był Izájáƺ : Niech wezmą bryłę ſuchych fig / y przyłożą ná wrzód / á będzie uzdrowiony.
Y rzekł był Ezechyjáƺ : Cóż jeſt zá znák / że wſtąpię do domu Páńſkiego?
