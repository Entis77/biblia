Nuż wƺyſcy prágnący pójdźćie do wód / y wy / co niemáćie pieniędzy / pójdźćie / kupujćie á jedzćie ; pójdźćie / mówię / kupujćie bez pieniędzy y bez zápłáty / wino y mleko.
Przecż wynákłádáćie pieniądze nie zá chleb / á prácę ſwą ná to / co nie náſycá? Słuchájąc ſłuchájćie mię / á jedzćie to / co jeſt dobrego / y niech śię rozkochá w tłuſtośći duƺá wáƺá.
Nákłońćie uchá ſwego / á pójdźćie do mnie ; ſłuchájćie / á będźie żyłá duƺá wáƺá. Y poſtánowię z wámi przymierze wiecżne / miłośierdźie Dawidowe pewne wyleję ná was.
Oto dáłem go zá świádká národom / zá wodzá y zá náucżyćielá národom.
Oto náród / któregoś nie znał / powołáƺ / á národy / które ćię nie znáły / zbieżą śię do ćiebie dla Páná / Bogá twego / y Świętego Izráelſkiego ; bo ćię uwielbi.
Szukájćie Páná / póki może być ználeźiony ; wzywájćie go / póki bliſko jeſt.
Niech opuśći niepobożny drogę ſwoję / á cżłowiek niepráwy myśli ſwoje y niech śię náwróći do Páná / á zmiłuje śię ; y do Bogá náƺego / gdyż jeſt hojnym w odpuƺcżániu.
Boć záiſte myśli moje nie ſą jáko myśli wáƺe / áni drogi wáƺe jáko drogi moje / mówi Pán ;
Ale jáko wyżƺe ſą niebioſá niż źiemiá / ták przewyżƺáją drogi moje drogi wáƺe / á myśli moje myśli wáƺe.
Bo jáko zſtępuje deƺcż y śnieg z niebá / á tám śię więcey nie wrácá / ále nápáwá źiemię / á cżyni ją płodną / cżyni ją też urodzájną / ták że wydáje náśienie śiejącemu / á chleb jedzącemu :
Tákći będźie ſłowo moje / które wynijdźie z uſt mojich ; nie wróći śię do mnie prózno / ále ucżyni to / co mi śię podobá / y poƺcżęśći mu śię w tym / ná co je poślę.
Przetoż w weſelu wynijdźiećie / á w pokoju doprowádzeni będźiećie. Góry y págórki chwałę przed wámi głośno záśpiewáją / á wƺyſtkie drzewá polne rękámi kláſkáć będą.
Miáſto ćierniá wyrośnie jedliná / á miáſto pokrzywy wyrośnie mirt ; á to będźie Pánu ku ſłáwie / ná znák wiecżny / który nigdy nie będźie wygłádzony.
