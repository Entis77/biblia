Śpiewáj niepłodná! którá nie rodziƺ / śpiewáj głośno / á krzycż / którá w porodzeniu nie prácujeƺ ; bo więcej będzie ſynów opuƺcżonej / niż ſynów tej / którá má mężá / mówi Pán.
Rozprzeſtrzeń miejſce námiotu ſwego / á opon przybytków ſwych nie zábrániáj rozćiągnąć : wyćiągnij powrozy twoje / á kołki twoje utwierdź.
Bo śię ná práwo y ná lewo rozśiliƺ / á náśienie twoje národy odziedzicży / y miáſtá ſpuſtoƺone oſádzi.
Nie bój śię / bo poháńbioná nie będzieƺ ; á nie zápáláj śię / bo nie przyjdzieƺ ná poſromocenie ; owƺem ná zelżywość młodośći twojej zápomniƺ / á ná poháńbienie wdowſtwá twego więcej nie wſpomniƺ.
Abowiem máłżonkiem twoim jeſt ſtworzyćie twój / Pán záſtępów imię jego / á odkupićiel twój / Święty Izráelſki / Bogiem wƺyſtkiej ziemi zwány będzie.
Bo ćię jáko żony opuƺcżonej y ſtrápionej w duchu / Pán powołá / á jáko żony młodej / gdy odrzuconą będzieƺ / mówi Bóg twój.
Ná máłą chwilkę opuśćiłem ćię ; ále záś w litośćiách wielkich zgromádzę ćię.
W málucżkim gniewie ſkryłem málucżko twárz ſwoję przed tobą ; ále w miłośierdziu wiecżnem zlituję śię nád tobą / mówi Pán / odkupićiel twój.
Bo to jeſt u mnie / co przy potopie Noego ; jákom przyśiągł / że śię więcej nie będą rozlewáć wody Noego po ziemi : tákem przyśiągł / że śię nie rozgniewám ná ćię / áni ćię zgromię.
A choćby śię y góry poruƺyły / y págórki śię záchwiáły : jednák miłośierdzie moje od ćiebie nie odſtąpi / á przymierze pokoju mego nie wzruƺy śię / mówi twój miłośćiwy Pán.
O utrápioná / wichrem rozmiotáná / z poćiechy obráná! oto Já położę ná kárbunkułách kámienie twoje / á ná ƺáfirách záłożę ćię.
Y ucżynię z kryƺtáłu okná twoje / á brámy twoje z kámieniá rubinowego / y wƺyſtkie gránice twoje z kámieniá koƺtownego.
A wƺyſcy ſynowie twoi będą wyucżeni od Páná / y obfitość pokoju będą mieli ſynowie twoi.
Ná ſpráwiedliwośći ugruntowáná będzieƺ ; od ućiſku śię oddáliƺ / przetoż śię go báć nie będzieƺ ; y od ſtárćiá / bo śię nie przybliży do ćiebie.
Oto nie jeden mieƺkáć będzie z tobą / który nie jeſt mój ; ále ktoby mieƺkájąc z tobą / był przećiwnym tobie / upádnie.
Otom já ſtworzył kowálá poddymájącego węgle w ogniu / á wyjmującego nácżynie ku roboćie ſwojej : Jám też ſtworzył puſtoƺyćielá / áby wytrácáł.
Żádne nácżynie urobione przećiw tobie nie zdárzy śię / á káżdy język powſtáwájący przećiw tobie ná ſądzie potępiƺ. Toć jeſt dziedzictwo ſług Páńſkich / á ſpráwiedliwość ich odemnie / mówi Pán.
