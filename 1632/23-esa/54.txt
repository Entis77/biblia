Śpiewáj niepłodná! którá nie rodźiƺ / śpiewáj głośno / á krzycż / którá w porodzeniu nie prácujeƺ ; bo więcej będźie ſynów opuƺcżonej / niż ſynów tej / którá má mężá / mówi Pán.
Rozprzeſtrzeń miejſce námiotu ſwego / á opon przybytków ſwych nie zábrániáj rozćiągnąć : wyćiągnij powrozy twoje / á kołki twoje utwierdź.
Bo śię ná práwo y ná lewo rozśiliƺ / á náśienie twoje národy odźiedźicży / y miáſtá ſpuſtoƺone oſádźi.
Nie bój śię / bo poháńbioná nie będźieƺ ; á nie zápáláj śię / bo nie przyjdźieƺ ná poſromocenie ; owƺem ná zelżywość młodośći twojej zápomniƺ / á ná poháńbienie wdowſtwá twego więcej nie wſpomniƺ.
Abowiem máłżonkiem twoim jeſt ſtworzyćie twój / Pán záſtępów imię jego / á odkupićiel twój / Święty Izráelſki / Bogiem wƺyſtkiej źiemi zwány będźie.
Bo ćię jáko żony opuƺcżonej y ſtrápionej w duchu / Pán powołá / á jáko żony młodej / gdy odrzuconą będźieƺ / mówi Bóg twój.
Ná máłą chwilkę opuśćiłem ćię ; ále záś w litośćiách wielkich zgromádzę ćię.
W málucżkim gniewie ſkryłem málucżko twárz ſwoję przed tobą ; ále w miłośierdźiu wiecżnem zlituję śię nád tobą / mówi Pán / odkupićiel twój.
Bo to jeſt u mnie / co przy potopie Noego ; jákom przyśiągł / że śię więcej nie będą rozlewáć wody Noego po źiemi : tákem przyśiągł / że śię nie rozgniewám ná ćię / áni ćię zgromię.
A choćby śię y góry poruƺyły / y págórki śię záchwiáły : jednák miłośierdźie moje od ćiebie nie odſtąpi / á przymierze pokoju mego nie wzruƺy śię / mówi twój miłośćiwy Pán.
O utrápioná / wichrem rozmiotáná / z poćiechy obráná! oto Já położę ná kárbunkułách kámienie twoje / á ná ƺáfirách záłożę ćię.
Y ucżynię z kryƺtáłu okná twoje / á brámy twoje z kámieniá rubinowego / y wƺyſtkie gránice twoje z kámieniá koƺtownego.
A wƺyſcy ſynowie twoi będą wyucżeni od Páná / y obfitość pokoju będą mieli ſynowie twoi.
Ná ſpráwiedliwośći ugruntowáná będźieƺ ; od ućiſku śię oddáliƺ / przetoż śię go báć nie będźieƺ ; y od ſtárćiá / bo śię nie przybliży do ćiebie.
Oto nie jeden mieƺkáć będźie z tobą / który nie jeſt mój ; ále ktoby mieƺkájąc z tobą / był przećiwnym tobie / upádnie.
Otom já ſtworzył kowálá poddymájącego węgle w ogniu / á wyjmującego nácżynie ku roboćie ſwojej : Jám też ſtworzył puſtoƺyćielá / áby wytrácáł.
Żádne nácżynie urobione przećiw tobie nie zdárzy śię / á káżdy język powſtáwájący przećiw tobie ná ſądźie potępiƺ. Toć jeſt dźiedźictwo ſług Páńſkich / á ſpráwiedliwość ich odemnie / mówi Pán.
