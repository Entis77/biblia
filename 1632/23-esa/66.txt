Ták mówi Pán : Niebo jeſt ſtolicą moją / á źiemiá podnóżkiem nóg moich. Gdźież tedy będźie ten dom / który mi zbudujećie? álbo gdźie będźie miejſce odpocżnieniá mego?
Bo to wƺyſtko ręká mojá ucżyniłá / y nią ſtánęło to wƺyſtko / mówi Pán. Wƺákże Ja ná tego pátrzę / który jeſt utrápionego y ſkruƺonego duchá / á który drży ná ſłowo moje.
Inácżej ten / kto zábijá wołu ná ofiárę / jákoby zábił cżłowieká ; kto zábijá ná ofiárę bydlątko / jákoby pſá śćiął ; kto ofiáruje ofiárę ſuchą / jákoby krew świnią ofiárowáł ; kto kádźi kádźidłem / jákoby báłwánowi błogoſłáwił. A jáko oni ſobie obráli drogi ſwoje / y w obrzydliwośćiách ſwoich kocháłá śię duƺá ich ;
Ták y Ja obiorę zá wynálázki ich / á to / cżego śię boją / przywiodę ná nich / przeto / że gdym wołáł / żáden śię nie ozwáł / gdym mówił / nie ſłucháli / ále cżynili to / co złego jeſt przed ocżymá mojemi / á to / cżegom nie chćiáł / obieráli.
Słuchájćie ſłowá Páńſkiego / wy którzy drżyćie ná ſłowo jego. Bráćia wáśi nienáwidzący was / á wygániájący was dla imieniá mego / mówią : Niech śię okáże ſłáwá Páńſká. Okáżeć śię záiſte ku poćieƺe wáƺej ; ále oni poháńbieni będą.
Głoſ grzmotu z miáſtá ſłyƺány będźie / głoſ z kośćiołá / głoſ Páná oddáwájącego zápłátę nieprzyjáćiołom ſwoim.
Pierwej niż prácowáłá ku porodzeniu / porodźiłá / pierwej niż ją ogárnęłá boleść / porodźiłá mężcżyznę.
Któż ſłyƺáł co tákowego? Kto widźiáł co podobnego? Możeż to być / áby źiemiá národźiłá ludu zá jeden dźień? Izáli náród ſpłodzony bywá jednym rázem? Ale Syon ledwie pocżął prácowáć ku porodzeniu / álić porodźił ſynów ſwych.
Cóżbym Ja / który otwierám żywot / rodźić nie miał? mówi Pán. Cóżbym Ja / który to cżynię / że rodzą / záwártym był? mówi Bóg twój.
Weſelćie śię z Jeruzálemem á rádujćie śię w nim wƺyſcy / którzy go miłujećie. Weſelćie śię z nim wielce / wƺyſcy którzykolwiek płákáli nád nim.
Przeto / że ſſáć będźiećie / y ſyćić śię pierśiámi poćiech jego / ſſáć będźiećie / y rozkoƺámi opływáć w jáſnośći chwáły jego.
Bo ták mówi Pán : Oto Ja obrócę ná nich pokój jáko rzekę / á ſłáwę národów jáko ſtrumień zálewájący / y będźiećie ſſáć ; ná ręku noƺeni / y ná kolánách rozkoƺnie piáſtowáni będźiećie.
Jáko ten / którego ćieƺy mátká jego / ták Ja was ćieƺyć będę ; á ták w Jeruzálemie ućiechy miewáć będźiećie.
Ujrzyćie záiſte / á rádowáć śię będźie ſerce wáƺe / á kośći wáƺe jáko tráwá zákwitną. Y poznáná będźie ręká Páńſká przy ſługách jego ; ále śię gniewem zápáli przećiwko nieprzyjáćiołom ſwoim.
Bo oto Pán w ogniu przyjdźie / á pocżwórne jego jáko wicher / áby wyláł gniew ſwój w popędliwośći / á łájánie ſwoje w płomieniu ogniá.
Pán / mówię / przez ogień ſądźić będźie / y przez miecż ſwój wƺelkie ćiáło / á pobitych od Páná wiele będźie.
Y ći / którzy śię poświęcáją y ocżyƺcżáją w ogrodách / jeden zá drugim jáwnie ; którzy jedzą mięſo świnie / y inną obrzydłość / y myƺy / koniec tákże wezmą / mówi Pán.
Abowiem Ja znám ſpráwy ich / y myśli ich ; y przyjdźie ten cżáſ / że zgromádzę wƺyſtkie národy / y języki / y przyjdą á oglądáją chwáłę moję.
Y położę ná nich znák / á poślę z tych / którzy záchowáni będą / do národów przy morzu do Pul y Lud / którzy ćiągną łuk do Tubálá / y do Jáwánu / ná wyſpy dálekie / które nic o mnie nie ſłyƺáły / y nie widźiáły chwáły mojey ; y będą opowiádáły chwáłę moję między národámi.
Y przywiodę wƺyſtkich bráći wáƺych ze wƺyſtkich národów Pánu w dáry / ná koniách y ná wozách / y ná lektykách / y ná mułách / y ná záwodnikách / ná górę świętobliwośći mojey do Jeruzálemu / mówi Pán / ták jáko przynoƺą ſynowie Izráelſcy dár w nácżyniu cżyſtem do domu Páńſkiego.
Y z tych też nábiorę kápłánów y Lewitów / mówi Pán.
Bo jáko te niebioſá nowe / y tá źiemiá nowá / którą Ja ucżynię / ſtánie przedemną / mówi Pán / ták ſtánie náśienie wáƺe y imię wáƺe.
Y ſtánie śię / że od nowiu mieśiącá do nowiu mieśiącá / y od ſábátu do ſábátu przychodźić będźie wƺelkie ćiáło / áby śię kłániáło przed oblicżnośćią moją / mówi Pán.
Y wynijdą á oglądáją trupy ludźi tych / którzy wyſtąpili przećiwko mnie ; ábowiem robák ich nie zdechnie / á ogień ich nie zágáśnie / á będą obrzydliwośćią wƺelkiemu ćiáłu.
