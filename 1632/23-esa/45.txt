To mówi Pán pomázáńcowi ſwemu Cyruſowi / którego práwicę ujmę / á poráżę przed nim národy / y biodrá królów rozpáƺę / á pootwierám przed nim wrotá / y brámy nie będą zámknięte.
Ja przed tobą pójdę / á krzywe drogi wyproſtuję / wrotá miedźiáne ſkruƺę / á záwory żelázne porąbię ;
Y dám ći ſkárby ſkryte / y klejnoty ſchowáne / ábyś poznał / żem Ja Pán / Bóg Izráelſki / który ćię przyzywám imieniem twojim.
Dla ſługi mego Jákóbá / y dla Izráelá / wybránego mego / názwáłem ćię imieniem twojim / przezwiſkiem twojim / choćiáż mię nie znáƺ.
Jám Pán / á niemáƺ żádnego więcey / oprócż mnie niemáƺ żádnego Bogá ; przepáſáłem ćię / ácżkolwiek mię nie znáƺ :
Aby poználi od wſchodu ſłońcá / y od záchodu ſłońcá / iż niemáƺ żádnego oprócż mnie / Jám Pán / á niemáƺ żádnego więcey ;
Który cżynię świátłość / y ſtwárzám ćiemnośći ; ſpráwuję pokój / y ſtwárzám złe. Ja Pán cżynię to wƺyſtko.
Spuśćie niebioſá roſę z góry / á obłoki niech kropią ſpráwiedliwość ; niech śię otworzy źiemiá / á niech wyrośnie zbáwienie / á ſpráwiedliwość niech weſpół zákwitnie. Ja Pán ſpráwię to.
Biádá temu / który śię ſpierá z ſtworzyćielem ſwoim / będąc ſkorupą / jáko inne ſkorupy gliniáne. Izáli gliná rzecże gárncárzowi ſwemu : Cóż cżyniƺ? Robotá twojá zápráwdę nikcżemná jeſt.
Biádá temu / który mówi ojcu : Cóż płodźiƺ? á niewieśćie : Cóż porodźiƺ?
Ták mówi Pán / Święty Izráelſki / y Twórcá jego : O przyƺłe rzecży pytájćie mię / á ſynów moich / y ſpráwę rąk moich porucżájćie mi.
Jám ucżynił źiemię / y cżłowiekám ná niej ſtworzył. Jám jeſt / którego ręce rozćiągnęły niebioſá / á wƺyſtkiemu wojſku ich rozkázuję.
Jám go wzbudźił w ſpráwiedliwośći / y wƺyſtkie drogi jego wyproſtuję. Onći zbuduje miáſto moje / á więźniów moich wypuśći / nie zá okup / áni zá dár / mówi Pán záſtępów.
Ták mówi Pán : Prácá Egipſká / y kupiectwo Murzynów / y Sebejcżyków / mężowie wyſocy do ćiebie przyjdą / á twoi będą ; zá tobą chodźić będą / w pętách pójdą / tobie śię kłániáć / y tobie śię korzyć będą / mówiąc : Tylko w tobie jeſt Bóg / á niemáƺ żádnego więcey / oprócż tego Bogá.
Zápráwdę tyś jeſt Bóg ſkryty / Bóg Izráelſki / zbáwićiel.
Wƺyſcy śię oni záwſtydzą / y poháńbieni będą ; cżynićiele báłwánów ſpołu z háńbą odſtąpią.
Ale Izráel zbáwion będźie przez Páná zbáwieniem wiecżnem ; nie záwſtydźićie śię / áni będźiećie poháńbieni / áż ná wieki wiecżne.
Bo ták mówi Pán / który ſtworzył niebioſá ( ten Bóg / który ſtworzył źiemię / y ucżynił ją! który ją utwierdźił / nie ná próżno ſtworzył ją / ná mieƺkánie utworzył ją ) : Jám Pán / á niemáƺ żádnego więcey.
Nie mówiłem potájemnie ná miejſcu źiemſkiem ćiemnem ; nie ná próżno mówię náśieniu Jákóbowemu : Szukájćie mię. Ja Pán mówię ſpráwiedliwość / á zwiáſtuję práwość.
Zgromádźćie śię / á przyjdźćie ; przybliżćie śię weſpół / wy / którzyśćie pozoſtáli między pogánámi. Nic nie wiedzą / którzy śię z drewniánemi báłwánámi ſwemi noƺą ; bo śię modlą bogu / który nie może wybáwić.
Oznájmujćież á przywiedźćie innych / á niech poſpołu w rádę wnijdą / á ukáżą / kto to od dáwnego cżáſu przepowiedźiáł? kto od onego cżáſu oznájmił? Izáli nie Ja Pán? Boć niemáƺ żádnego innego Bogá oprócż mnie. Niemáƺ Bogá ſpráwiedliwego / y zbáwićielá innego oprócż mnie.
Obejrzyjćież śię ná mię / ábyśćie zbáwione były wƺyſtkie końcżyny źiemi ; bom Ja Bóg / á niemáƺ żádnego więcey.
Przyśiągłem ſám przez śię ; wyƺło z uſt moich ſłowo ſpráwiedliwe / które śię názád nie wróći : Że śię mnie kłániáć będźie wƺelkie koláno / y przyśięgáć káżdy język.
Mówiąc : Tylko w Pánu mam wƺelką ſpráwiedliwość y śiłę. Tákowi áż do niego przyjdą ; ále poháńbieni będą wƺyſcy / którzy śię gniewem zápáláją przećiwko niemu.
W Pánu uſpráwiedliwione będźie / y przechwáláć śię będźie wƺyſtko náśienie Izráelſkie.
