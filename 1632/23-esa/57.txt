Spráwiedliwy ginie / á nikt tego do ſercá nieprzypuƺcżá ; y mężowie pobożni ſchodzą / á nikt tego nie uwáżá / że przed przyjśćiem złego ſpráwiedliwy zebrány bywá ;
Że wchodzi do pokoju / á odpocżywá ná łożu ſwojim / ktokolwiek chodzi w uprzejmośći.
Ale wy ſámi przyſtąpćie / ſynowie cżárownicy / náśienie cudzołożniká y wƺetecżnicy!
Nád kimże śię ćieƺyćie? przećiwko komuż rozdzieráćie gębę / y wywieƺáćie język? Izáli nie jeſteśćie ſynowie nierządu / náśienie kłámliwe?
Którzy nierząd płodzićie w gájách pod káżdem drzewem zielonem zábijájąc ſynów ſwych przy potokách / pod wyſokiemi ſkáłámi.
Między głádkim kámieniem potokowym jeſt dziáł twój. Cić ſą / ći loſem twoim / ná które też wylewáƺ ofiárę mokrą / á ofiárujeƺ ofiárę śniedną / y w temże bym śię Já kocháł?
Ná górze wyſokiej y wynioſłej poſtáwiłeś łoże twoje / á tám wſtępujeƺ ku ſpráwowániu ofiár.
A zá drzwiámi y zá podwojim położyłáś pámiątkę twoję / gdyż odemnie odchodząc odkrywáƺ śię / á wſtąpiwƺy rozƺerzáƺ łoże ſwe / cżyniąc je przeſtworniejƺe / niżeli pogánie ; umiłowáłáś łoże ich / gdziekolwiek miejſce upátrzyƺ.
Chodziƺ y do królá / z olejkiem y z rozmáitemi wonnemi máśćiámi twemi ; poſyłáƺ bowiem poſłów ſwych dáleko / á poniżáƺ śię áż do grobu.
Mnóſtwem dróg ſwoich ſprácowáłáś śię / á nie mówiƺ : Dáremnáć to. Ználázłáś pomoc ręce ſwojej / dlátegoś nie zemdláłá.
Kogożeś śię obáwiáłá y lękáłá / iżeś kłámáłá? Ná mięś nie pomniáłá / ániś tego przypuśćiłá do ſercá ſwego : dlátegoż to / żem Já milcżáł / á to z dáwná / nie boiƺ śię mnie?
Já opowiem ſpráwiedliwość twoję y ſpráwy twoje / któreć nic nie pomogą.
Gdy záwołáƺ / niech ćię wybáwi zgrájá twojá ; ále wƺyſtkie one roznieśie wiátr / y pochwyći márność. Lecż ten / co we mnie ufá / odziedzicży ziemię / á pośiądzie górę świętą moję.
Bo rzeką : Wyrównájćie / wyrównájćie / zgotujćie drogę / uprzątnijćie záwády z drogi ludu mojego.
Bo ták mówi on nájdoſtojniejƺy y nájwyżƺy / który mieƺká w wiecżnośći / á święte jeſt imię jego : Já / który mieƺkám ná wyſokośći ná miejſcu świętym / mieƺkám y z tym / który jeſt ſkruƺonego y uniżonego duchá / ożywiájąc duchá pokornych / ożywiájąc ſerce ſkruƺonych.
Nie będę śię záiſte ná wieki wádził / áni śię wiecżnie gniewáł ; boćby duch przed oblicżem mojem zemdláł / y duƺe / którem Já ucżynił.
Dlá niepráwośći łákomſtwá jego rozgniewáłem śię / á uderzyłem go ; ukryłem śię / á rozgniewáłem śię / przeto / że odpornym będąc / poƺedł drogą ſercá ſwego.
Widzę drogi jego / wƺákże uzdrowię go ; doprowádzę go / y przywrócę mu poćiechy / y tym / którzy z nim płácżą.
Stworzę owoc wárg / pokój dálekiemu y bliſkiemu / mówi Pán ; á ták uzdrowię go.
Lecż niepobożni będą jáko morze wzburzone / gdy śię uſpokoić nie może / á którego wody wymiátáją káł y błoto.
Niemáƺ pokoju niepobożnym / mówi Bóg mój.
