Brzemię Dámáƺku. Oto Dámáƺek przeſtánie być miáſtem / á ſtánie śię kupą rumu.
Miáſtá Aroer opuƺcżone będą ; dlá trzód będą / które tám odpocżywáć będą / á nie będzie / ktoby je ſtráƺył.
Y uſtánie obroná od Efráimá / y króleſtwo od Dámáƺku / y od oſtátká Syryjcżyków / y jáko ſłáwá ſynów Izráelſkich zniƺcżeją / mówi Pán záſtępów.
Y ſtánie śię dniá onego / że umniejƺoná będzie ſłáwá Jákóbowá / á tłuſtość ćiáłá jego ſchudnie.
Abowiem Azur będzie jáko ten / który w żniwá zboże zbierá / á rámię jego żnie kłoſy ; y będzie jáko ten / co zbierá kłoſy w dolinie Refáim.
Wƺákże zoſtáną ná nim pominione groná / jáko ná otrzęśnionej oliwie dwie álbo trzy oliwiki zoſtáną ná wierzchu drzewá / á cżtery álbo pięć ná rodzájnych gáłęziách jego / mówi Pán / Bóg Izráelſki.
Dniá onego obejrzy śię cżłowiek ná ſtworzyćielá ſwego / á ocży jego ná Świętego Izráelſkiego poglądáć będą ;
A nie obejrzy śię ná ołtárze / ſpráwę rąk ſwoich / áni ná to / co ucżyniły pálce jego / pátrzyć będzie / áni ná gáje poświęcone / áni ná obrázy ſłonecżne.
Dniá onego miáſtá mocy jego będą opuƺcżone / jáko chrośćinká y róƺcżká / które opuƺcżone będą od ſynów Izráelſkich / y będzieƺ ziemią ſpuſtoƺoną.
Boś zápomniáłá ná Bogá zbáwieniá ſwego / y ná ſkáłę mocy twojej nie wſpomináłáś. Przetoż choć ƺcżepy rozkoƺne ƺcżepiƺ / y látorośli winne obce ſádziƺ ;
Cżáſu ƺcżepieniá twego ƺcżepy áby roſły / opátrujeƺ ; náwet tegoż poránku / co śiejeƺ / áby śię puśćiło / ſtáráƺ śię : wƺákże w dzień pożytku gromádno boleść y rozpácż żąć będzieƺ.
Biádá zgrái ludu wielkiego / którzy hucżą jáko ƺum morſki / y zgiełkowi národów / które ƺumią jáko ƺum wód gwáłtownych ;
Národom / które ƺumią jáko ƺum wód wielkich ; bo je Pán ſfuká / y ućiekną dáleko / y gonione będą od wiátru jáko plewy po wierzchách gór / á jáko wiecheć od wichru.
Bo cżáſu wiecżornego náſtąpi trwogá / á niż poránek przyjdzie / álić go niemáƺ. Tenći jeſt dziáł tych / którzy náſ puſtoƺą / y loſ tych / którzy náſ plundrują.
