Powſtáń / objáśnij śię! poniewáż przyƺłá świátłość twojá / á chwáłá Páńſká weƺłá nád tobą.
Bo oto ćiemnośći okryją ziemię / á záćmienie národy ; ále nád tobą wejdzie Pán / á chwáłá jego nád tobą widziáná będzie.
Y będą chodzić národy w świátłośći twojej / á królowie w jáſnośći / którá wejdzie nád tobą.
Podnieś w około ocży twe / á ſpojrzyj ; ći wƺyſcy / którzy śię zgromádzili / pójdą do ćiebie ; ſynowie twoi z dáleká przyjdą / á córki twoje przy boku twoim chowáne będą.
Tedy oglądáƺ to / á rozweſeliƺ śię ; tedy śię zdumieje y rozƺerzy ſerce twoje / gdy śię obróći ku tobie zgrájá morſká / á moc národów przyjdzie do ćiebie.
Obfitość wielbłądów okryje ćię / tákże dromedárze z Mádyján y z Efy. Wƺyſcy ći przyjdą z Sáby / złoto y kádzidło przynioſą / á chwáły Páńſkie opowiádáć będą.
Wƺyſtkie ſtádá z Kedár zgromádzą śię do ćiebie ; bárány z Nebájotu ſłużyć ći będą / á ofiárowáne będąc ná ołtárzu moim / przyjemne będą ; á ták dom májeſtátu mego ozdobię.
Y rzecżeƺ : Którzyż to ſą / co śię jáko obłoki zlátują / y jáko gołębie do okien ſwoich?
Ná mięć záiſte wyſpy ocżekują / y okręty morſkie zdáwná / áby przywiedli ſynów twoich z dáleká / tákże ſrebro ſwoje z ſobą / y złoto ſwoje imieniowi Páná / Bogá twego / y Świętego Izráelſkiego ; bo ćię uwielbi.
Y pobudują cudzoziemcy mury twoje / á królowie ich ſłużyć ći będą / gdyż w rozgniewániu mojem uderzę ćię / á w upodobániu mojem zlituję śię nád tobą.
Y będą otworzone brámy twoje uſtáwicżnie ; we dnie y w nocy nie będą zátkáne / áby przywiedziono do ćiebie moc pogán / y królowie ich áby byli przywiedzieni.
Náród ten y króleſtwo / którećby nie ſłużyło / zginie ; národy tákie / mówię / do ƺcżętu ſpuſtoƺone będą.
Słáwá Libánu do ćiebie przyjdzie / jedliná / ſoſná / tákże bukƺpán / dlá ozdoby miejſcá świątnicy mojej / ábym miejſce nóg moich uwielbił.
Przyjdą tákże do ćiebie w pokorze ſynowie tych / którzy ćię trápili / y będą śię kłániáć ſtopom nóg twoich / którzykolwiek pogárdzili tobą / y názwią ćię miáſtem Páńſkim / Syonem Świętego Izráelſkiego.
Miáſto tego / coś opuƺcżoná y w nienáwiśći byłá / ták / że nie było / ktoby przez ćię chodził / wyſtáwię ćię zá doſtojność wiecżną / y weſele od národu do národu.
Bo ſſáć będzieƺ mleko národów / y pierśiámi królów kármioná będzieƺ ; y poznáƺ / iżem Já Pán / zbáwiećiel twój y odkupićiel twój / mocárz Jákóbowy ;
Miáſto miedzi nánioſę złotá / á miáſto żelázá nánioſę ſrebrá / á miáſto drew miedzi / á miáſto kámieniá żelázá ; y poſtáwię nád tobą dozorców ſpokojnych / y urzędników ſpráwiedliwych.
Nie będzie więcej ſłycháć o drápieſtwie w ziemi twojej / o zburzeniu y ſpuſtoƺeniu ná gránicách twoich ; ále ogłáƺáć będzieƺ zbáwienie ná murách twoich / á chwáłę w brámách twoich.
Nie będzieƺ miáł więcej ſłońcá zá świátłość dzienną / á jáſność mieśiącá nie oświeći ćię / ále Pán będzie świátłośćią twoją wiecżną / á Bóg twój ſłáwą twoją.
Nie zájdzie więcej ſłońce twoje / á mieśiąc twój nie ſkryje śię ; bo Pán będzie wiecżną świátłośćią twoją ; á ták dokonáją śię dni ſmutku twego.
Lud tákże twój / którzybykolwiek byli ſpráwiedliwi / ná wieki odziedzicżą ziemię ; będą látoroślą ƺcżepieniá mego / dziełem rąk moich / ábym w nim był uwielbiony.
Nájmniejƺy rozmnoży śię ná tyśiące / á málucżki pocżet w náród niezlicżony. Já Pán cżáſu ſwego prędko to ucżynię.
