Oto nie jeſt ukróconá ręká Páńſká / áby zbáwić nie mogłá ; á nie jeſt obćiążone ucho jego / áby wyſłucháć nie mogło.
Ale niepráwośći wáƺe rozdźiáł ucżyniły między wámi y między Bogiem wáƺym / á grzechy wáƺe ſpráwiły / że ukrył twárz przed wámi / áby nie ſłyƺáł.
Bo ręce wáƺe krwią ſą zmázáne / á pálce wáƺe niepráwośćią ; wárgi wáƺe mówią kłámſtwo / á język wáƺ niepráwość świegoce.
Niemáƺ ktoby śię záſtáwiáł o ſpráwiedliwość / áni jeſt ktoby śię záſádzáł o práwdę. Ufáją w próżnośći / á mówią kłámſtwo ; pocżynáją ućiſk / á rodzą niepráwość.
Jájá bázyliƺkowe wylęgli / á płótná pájęcżego nátkáli. Ktoby jádł jájá ich / umrze / á jeźli je ſtłucże / wynijdźie jáƺcżórká.
Płótná ich nie godzą śię ná ƺátę / áni śię przyodźieją robotámi ſwemi. Ucżynki ich ſą ucżynki niepráwośći / á ſpráwá łupieſtwá jeſt w rękách ich.
Nogi ich bieżą do złego / y kwápią śię ná wylánie krwi niewinnej. Myśli ich ſą myśli niepráwośći ; ſpuſtoƺenie y ſtárćie jeſt ná drogách ich.
Drogi pokoju nieznáją / y niemáƺ ſpráwiedliwośći w drogách ich ; śćieƺki ſwe ſámi pokrzywili u śiebie ; káżdy / kto po nich chodźi / nie zná pokoju.
Dlátego oddálił śię ſąd od náſ / á nie dochodźi náſ ſpráwiedliwość ; cżekámy ná świátłość / á oto ćiemność ; ná jáſność / ále w ćmie chodźimy.
Mácámy śćiány jáko ślepi / á mácámy / jákobyśmy ocżów nie mieli. Potykámy śię w południe jáko w zmierzk ; w wielkich doſtátkách podobniśmy umárłym.
Mrucżymy wƺyſcy jáko niedźwiedź / jáko gołębicá uſtáwicżnie ſtękámy ; ocżekujemy ná ſąd / ále go niemáƺ ; ná wybáwienie / ále dálekie jeſt od náſ.
Bo śię rozmnożyły przeſtępſtwá náƺe przed tobą / á grzechy náƺe świádcżą przećiwko nám / poniewáż niepráwośći náƺe ſą przy náſ / y złośći náƺe uznájemy ;
Żeśmy wyſtąpili / y kłámáli przećiw Pánu / y odwróćiliśmy śię / ábyśmy nie ƺli zá Bogiem náƺym ; żeśmy mówili o potwárzy y o odſtąpieniu / żeśmy zmyśláli y wywieráli z ſercá ſwego ſłowá kłámliwe.
Ták / że śię ſąd opák obróćił / á ſpráwiedliwość z dáleká ſtoi ; bo ná ulicy práwdá ƺwánkowáłá / á práwość przejśćiá nie má.
Owƺem / práwdá zginęłá / á ten / co odſtępuje od złego / ná łup podány bywá. To widźi Pán / y nie podobá śię to w ocżách jego / że niemáƺ ſądu.
Gdy tedy widźiáł / że niemáƺ żádnego mężá / áż śię zdumiáł / że niemáƺ żádnego / coby śię záſtáwił / á przetoż wybáwienie ſpráwiło mu rámię jego / á ſpráwiedliwość jego ſámá go podpárłá.
Bo śię przyoblókł w ſpráwiedliwość jáko w páncerz / á hełm zbáwieniá ná głowie jego ; oblókł śię w odźienie pomſty jáko w ƺátę / á odźiáł śię zápálcżywośćią jáko płáƺcżem ;
Aby według ucżynków / áby według nich odpłáćił popędliwośćią przećiwnikom ſwoim / áby nágrodę nieprzyjáćiołom ſwoim / á wyſpom zápłátę oddáł.
Y będą śię báli / którzy ſą ná záchód / imieniá Páńſkiego / y którzy ná wſchód ſłońcá / ſłáwy jego. Gdy przypádnie nieprzyjáćiel jáko rzeká / tedy go duch Páńſki precż zápędźi.
Bo przyjdźie do Syonu odkupićiel / y do tych / którzy śię odwrácáją od wyſtępków w Jákóbie / mówi Pán.
A toć będźie przymierze moje z nimi / mówi Pán : Duch mój / który jeſt w tobie / y ſłowá moje / którem włożył w uſtá twoje / nie odſtąpią od uſt twoich / áni od uſt náśieniá twego / áni od uſt potomków náśieniá twego / mówi Pán / odtąd áż ná wieki.
