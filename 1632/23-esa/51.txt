Słuchájćie mię / którzy náśládujećie ſpráwiedliwośći / którzy ƺukáćie Páná. Spojrzyjćie ná ſkáłę / z którejśćie wyćięći / y ná głębokość dołu / ſkądeśćie wykopáni.
Spojrzyjćie ná Abráhámá / ojcá wáƺego / y ná Sárę / którá wáſ porodziłá / żem go jednego wezwáł / y pobłogoſłáwiłem mu / á rozmnożyłem go.
Gdyż poćieƺy Pán Syon / poćieƺy wƺyſtkie puſtynie jego / á ucżyni puƺcżę jego bárdzo rozkoƺną / á puſtynię jego jáko ogród Páńſki / rádość y weſele znájdzie śię w nim / dziękcżynienie / y głoſ śpiewániá.
Pilnujćie mię / ludu mój y rodzino mojá! nádſtáwćie mi uƺów ; bo zákon odemnie wyjdzie / á ſąd mój zá świátłość národom wyſtáwię.
Bliſko jeſt ſpráwiedliwość mojá / wynijdzie zbáwienie moje / á rámioná moje národy ſądzić będą. Ná mię wyſpy ocżekują / á po rámieniu mojem tęſknią.
Podnieśćie ku niebu ocży wáƺe / á ſpojrzyjćie ná ziemię ná dół. Niebioſá jáko dym zniƺcżeją / á ziemiá jáko odzienie zwiotƺeje / y obywátele jej / jáko y oná zginą ; ále zbáwienie moje ná wieki będzie / á ſpráwiedliwość mojá nie uſtánie.
Słuchájćie mię / którzy znáćie ſpráwiedliwość ludu / w którego ſercu jeſt zákon mój! Nie bójćie śię urągániá ludzkiego / á ſromoceniá ich nie lękájćie śię.
Abowiem ich mól jáko ƺátę pożre / á robák ich jáko wełnę pogryzie ; ále ſpráwiedliwość mojá ná wieki będzie / á zbáwienie moje od národu do národu.
Ocuć śię / ocuć śię / oblecż śię w śiłę / o rámię Páńſkie! Ocuć śię jáko zá dni dáwnych / y zá rodzájów przeƺłych! Izáliś nie ty jeſt / któreś zgłádziło Egipt / y zrániło ſmoká?
Izáliś nie ty jeſt / któreś wyſuƺyło morze / wody przepáśći wielkiej? któreś obróćiło głębokośći morſkie w drogę / áby przeƺli wybáwieni?
A ták ći / których odkupił Pán / niech śię náwrócą / y przyjdą do Syonu z śpiewániem / á weſele wiecżne niech będzie nád głową ich ; weſelá y rádośći niech doſtąpią / á niech ućiecże ſmutek y wzdychánie.
Já / Jám jeſt poćieƺyćiel wáƺ. Któżeś ty / że śię boiƺ cżłowieká śmiertelnego / y ſyná cżłowiecżego tráwie podobnego?
Że zápomináƺ ná Páná ſtworzyćielá ſwego / który rozćiągnął niebioſá / y záłożył ziemię? á że śię lękáƺ uſtáwicżnie káżdego dniá popędliwośći trápiącego / gdy śię gotuje / áby zátrácáł? Ale gdzież jeſt tá popędliwość trápiącego?
Poſpieƺy śię / áby więzień był uwolniony ; bo nie umrze w dole / áni będzie miáł jáki niedoſtátek chlebá ſwego.
Já záiſte jeſtem Pán / Bóg twój / który rozdzielám morze / ták / że ƺumią wáły jego ; Pán záſtępów jeſt imię moje.
Jám włożył ſłowá moje w uſtá twoje / á ćieniem ręki mojej zákryłem ćię / ábyś ƺcżepił niebioſá / á záłożył ziemię / y rzekł Syonowi : Tyś jeſt lud mój.
Ocuć śię / ocuć śię / powſtáń Jeruzálemie! któreś piło z ręki Páńſkiej kubek zápálcżywośći jego / drożdże z kubká trućizny śmiertelnej wypiłoś y wyſącżyłoś.
Nikt go nie prowádził ze wƺyſtkich ſynów / których nápłodziło / y nikt go nie ujął zá rękę jego ze wƺyſtkich ſynów / które wychowáło.
Dwie rzecży ſą / które ćię ſpotkáły ; ( któż śię ćiebie użálił? ) Spuſtoƺenie y ſkruƺenie / głód y miecż ; któż ćię poćieƺy?
Synowie twoi pomdláwƺy leżeli ná rogách wƺyſtkich ulic / jáko báwół w śieći / pełni będąc popędliwośći Páńſkiej / gromieniá Bogá twego.
A przetoż ſłucháj teráz tego / o utrápioná y pijáná / ále nie winem!
Ták mówi Pán twój / Pán y Bóg twój / który śię záſtáwiá zá lud ſwój : Oto biorę z ręki twojej kubek trućizny śmiertelnej / y drożdże kubká popędliwośći mojej ; nie będzieƺ więcej pić z niego ;
Ale podám go w rękę tych / którzy ćię trápią / którzy mówili duƺy twojej : Náchyl śię / niech przez ćię przejdziemy ; á tyś pokłádáło jáko ziemię grzbiet ſwój / y jáko ulicę przechodzącym.
