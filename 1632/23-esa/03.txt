Abowiem oto pánujący Pán záſtępów odejmie od Jeruzálemu y od Judy láſkę / y podporę / wƺeláką podporę chlebá / y wƺeláką podporę wody.
Mocárzá y mężá wálecżnego / y ſędźiego / y proroká / y mędrcá / y ſtárcá ;
Rotmiſtrzá nád pięćdźieśiąt / á mężá powáżnego / y rádcę / y mądrego rzemieślniká / y kráſomówcę.
Y dám im dźieći zá kśiążętá ; dźieći mówię pánowáć będą nád nimi.
Y będźie ućiſkáł między ludem jeden drugiego / y bliźni bliźniego ſwego : powſtánie dźiećię przećiwko ſtárcowi / á podły przećiwko zácnemu.
A gdy śię uchwyći káżdy brátá ſwego z domu ojcá ſwego / y rzecże : Máƺ odźienie / bądźże kśiążęćiem náƺym / á upádek ten zátrzymáj ręką ſwą :
Tedy on przyśięże dniá onego / mówiąc : Nie będę záwiązywáł tych rán : ábowiem w domu moim niemáƺ chlebá / áni odźieniá ; nie ſtánowćież mię kśiążęćiem nád ludem.
Bo Jeruzálem upádá / á Judá śię wáli / dlátego / że język ich / y ſpráwy ich ſą przećiwko Pánu / pobudzájąc do gniewu ocży májeſtátu jego.
Poſtáwá oblicżá ich świádcży przećiwko nim ; grzech ſwój / jáko Sodomcżycy / opowiádáją / á nie táją go. Biádá duƺy ich! ábowiem ſámi ná śię złe przywodzą.
Powiedzćie ſpráwiedliwemu / że mu dobrze będźie ; bo owocu ucżynków ſwoich pożywáć będźie.
Ale biádá niepobożnemu! źle mu będźie ; ábowiem odpłátá rąk jego dáná mu będźie.
Kśiążętá ludu mego ſą dźiećmi / á niewiáſty pánują nád nimi. O ludu mój! ći / którzy ćię wodzą / zwodzą ćię / á drogę śćieżek twoich ukrywáją.
Powſtáł Pán / áby ſądźił / ſtoi / áby ſądźił lud.
Pán przyjdźie ná ſąd przećiwko ſtárƺym ludu ſwego / y przećiwko kśiążętom ich / á rzecże : Wyśćie ſpuſtoƺyli winnicę moję / zdźierſtwo z ubogiego w domách wáƺych.
Przecżże trzećie lud mój / á oblicżá ubogich bijećie? mówi Pán / Pán záſtępów.
Y rzekł Pán : Iż śię wynoƺą córki Syońſkie / á chodzą ƺyje wyćiągnąwƺy / y mrugájąc ocżymá przechodzą śię / á drobno poſtępując nogámi ſwemi ƺeleſt cżynią :
Przetoż obłyśi Pán wierzch głowy córek Syońſkich / á Pán ſromotę ich obnáży.
Dniá onego odejmie Pán ochędóſtwo podwiązek / tákże cżepce y záwieƺeniá /
Piżmowe jábłká / y mánele / y zátycżki /
Bieretki / y zápony / y bindy / y przednicżki / y náuƺnice ;
Pierśćionki / y nácżelniki /
Odmienne ƺáty / y płáƺcżyki / y podwiki / y wácki /
Zwierćiádłá / y rántuƺki / y tkánki / y letniki.
Y będźie miáſto wonnych rzecży ſmród / á miáſto páſá rozpáſánie / á miáſto utrefionych włoſów łyśiná / á miáſto ƺerokiej ƺáty opáſánie worem / á miáſto pięknośći ogorzeliná.
Mężowie twoi od miecżá upádną / á mocárze twoi w bitwie.
Y záſmucą śię / á płákáć będą brámy jego / á ſpuſtoƺony ná źiemi śiedźieć będźie.
