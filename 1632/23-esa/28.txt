Biádá pyƺney koronie / pijánicom z Efráimá / y kwiátowi opádłemu z ozdoby ſłáwy ſwojey! Biádá tym / którzy rządzą doliną bárzo urodzájną / y znikcżemniáłym od winá!
Oto możny y śilny Páńſki będąc jáko náwáłność grádu / jáko wicher wywrácájący / jáko byſtrość wód gwáłtowney powodźi uderzy ją o źiemię ręką ſwą.
Nogámi podeptáná będźie pyƺná koroná / pijánicy Efráimſcy!
Tedy śię ſtánie / że kwiát opádájący z ozdoby y z ſłáwy ſwojey / tych / którzy rządzą doliną bárzo urodzájną / będźie jáko owoc ſkorożrzy / pierwej niż láto bywa ; który ſkoro kto obácży / nie puśći go z ręki / áż go zje.
Dniá onego będźie Pán zaſtępów koroną ozdoby / y koroną ſłáwy oſtátkowi ludu ſwego /
Y duchem ſądu śiedzącemu ná ſądźie / á mocą tym / którzy odpieráją bitwę áż do bramy.
Ale y ći od winá błądzą / y od mocnego nápoju potácżáją śię. Kśiążę y prorok błądzą od mocnego nápoju / utonęli w winie / potácżáją śię od mocnego nápoju / błądzą w widzeniu / potykáją śię w ſądźie.
Abowiem wƺyſtkie ſtoły ich pełne ſą zwrácánia y plugáſtwá / ták / áż miejſcá nie ſtáje.
Kogożby ucżyć miał umiejętnośći? á komu da zrozumieć co ſłyƺał? Izali odſtáwionym od mleká / á odſádzonym od pierśi?
Ponieważ podáwał im przykazánie zá przykázániem / przykazánie zá przykázániem / przepis zá przepiſem / przepis zá przepiſem / trochę tu / trochę owdźie :
A wƺákże jákoby nieznájomą mową / y językiem obcym mówił do ludu tego.
A gdy im rzekł : Toć jeſt odpocżnienie / ſpráwćie odpocżynek ſprácowánemu / toć jeſt odpocżnienie ; ále oni nie chćieli ſłucháć.
Y będźie im ſłowo Páńſkie : przykazánie zá przykázániem / przykazánie zá przykázániem / przepis zá przepiſem / przepis zá przepiſem ; trochę tu / trochę owdźie / ná to / áby ƺli / á pádƺy wznák ſtłukli śię / á uwikłáni będąc / pochwytáni byli.
Przetoż ſłuchájćie ſłowá Páńſkiego / mężowie náśmiewcy! pánujący nád tym ludem / który jeſt w Jeruzálemie.
Dla tego / że mówićie : Ucżyniliśmy przymierze z śmierćią / y z piekłem mamy porozumienie / bicż gwáłtowny nas nie dojdźie / gdy przechodźić będźie ; bośmy położyli kłámſtwo zá ućiecżkę ſwoję / á pod fáłƺem utáiliśmy śię ;
Dlátegoż ták powiedźiał pánujący Pán : Oto Ja zá grunt kłádę w Syonie kámień / kámień doświádcżony / węgielny / koƺtowny / gruntownie ugruntowány ; kto wierzy / nie pokwápi śię.
A wykonam ſąd według ƺnuru / á ſpráwiedliwość według wági ; y potłucże grád nádźieję omylną / á ućiecżkę wody zátopią.
A ták zgłádzone będźie przymierze wáƺe z śmierćią / á porozumienie wáƺe z piekłem nie oſtoi śię ; gdy bicż gwáłtowny przechodźić będźie / będźiećie od niego podeptáni.
Kiedy jedno pocżnie przechodźić / pochwyći was ; bo ná káżdy poránek przechodźić będźie we dnie y w nocy. A ſám poſtrách przywiedźie was ku zrozumieniu tego / cośćie ſłyƺeli ;
Zwłáƺcżá iż krótƺe będźie łoże / niżby śię kto mógł rozćiągnąć / y nákryćie wąſkie / choćby śię ſkurcżył.
Abowiem Pán powſtánie jáko ná górze Perazym / á rozgniewá śię jáko w dolinie Gábáon / áby wykonał ſpráwę ſwoję / niezwycżájną ſpráwę ſwoję / y áby dokońcżył ſpráwy ſwojey / niezwycżájney ſpráwy ſwojey.
A ták teraz nie náśmiewájćie śię / áby śię niezmocniły związki wáƺe / bom o pewnem zepſowániu wƺyſtkiej źiemi ſłyƺał od Páná / Páná zaſtępów /
Nádſtáwiájćie uƺu / á ſłuchájćie głoſu mego ; bądźćie pilni / á ſłuchájćie mowy mojey.
Izali káżdego dniá orácż orze / áby śiał? przegánia brozdy / á włócży rolę ſwoję?
Izali zrównáwƺy wierzch jey / nie rozśiewá wyki / y nie roztrząſá kminu / y nie śieje pƺenicy wyborney / y jęcżmieniá przedniego / y orkiƺu ná miejſcu ſpoſobnem?
Bo go ucży roztropnośći Bóg jego / y náucżá go.
Wyki nie młócą okowánem nácżyniem / áni tácżáją kołá wozowego po kminie ; ále kijem wybijáją wykę / á kmin láſką.
Pƺenicá młóconá bywa ; wƺákże y tey nie záwżdy młóćić będźie / áni jey potrze kołem wozá ſwego / áni jey zębámi jego pokruƺy.
Y toć od Páná zaſtępów wyƺło / który jeſt dźiwny w rádźie / á wielmożny w rzecży ſámej.
