Biádá pyƺnej koronie / pijánicom z Efráimá / y kwiátowi opádłemu z ozdoby ſłáwy ſwojej! Biádá tym / którzy rządzą doliną bárdzo urodzájną / y znikcżemniáłym od winá!
Oto możny y śilny Páńſki będąc jáko náwáłność grádu / jáko wicher wywrácájący / jáko byſtrość wód gwáłtownej powodźi uderzy ją o źiemię ręką ſwą.
Nogámi podeptáná będźie pyƺná koroná / pijánicy Efráimſcy!
Tedy śię ſtánie / że kwiát opádájący z ozdoby y z ſłáwy ſwojej / tych / którzy rządzą doliną bárdzo urodzájną / będźie jáko owoc ſkorożrzy / pierwej niż láto bywá ; który ſkoro kto obácży / nie puśći go z ręki / áż go zje.
Dniá onego będźie Pán záſtępów koroną ozdoby / y koroną ſłáwy oſtátkowi ludu ſwego /
Y duchem ſądu śiedzącemu ná ſądźie / á mocą tym / którzy odpieráją bitwę áż do brámy.
Ale y ći od winá błądzą / y od mocnego nápoju potácżáją śię. Kśiążę y prorok błądzą od mocnego nápoju / utonęli w winie / potácżáją śię od mocnego nápoju / błądzą w widzeniu / potykáją śię w ſądźie.
Abowiem wƺyſtkie ſtoły ich pełne ſą zwrácániá y plugáſtwá / ták / áż miejſcá nie ſtáje.
Kogożby ucżyć miáł umiejętnośći? á komu dá zrozumieć co ſłyƺáł? Izáli odſtáwionym od mleká / á odſádzonym od pierśi?
Poniewáż podáwáł im przykázánie zá przykázániem / przykázánie zá przykázániem / przepiſ zá przepiſem / przepiſ zá przepiſem / trochę tu / trochę owdźie :
A wƺákże jákoby nieznájomą mową / y językiem obcym mówił do ludu tego.
A gdy im rzekł : Toć jeſt odpocżnienie / ſpráwćie odpocżynek ſprácowánemu / toć jeſt odpocżnienie ; ále oni nie chćieli ſłucháć.
Y będźie im ſłowo Páńſkie : przykázánie zá przykázániem / przykázánie zá przykázániem / przepiſ zá przepiſem / przepiſ zá przepiſem ; trochę tu / trochę owdźie / ná to / áby ƺli / á pádłƺy wznák ſtłukli śię / á uwikłáni będąc / pochwytáni byli.
Przetoż ſłuchájćie ſłowá Páńſkiego / mężowie náśmiewcy! pánujący nád tym ludem / który jeſt w Jeruzálemie.
Dlátego / że mówićie : Ucżyniliśmy przymierze z śmierćią / y z piekłem mámy porozumienie / bicż gwáłtowny náſ nie dojdźie / gdy przechodźić będźie ; bośmy położyli kłámſtwo zá ućiecżkę ſwoję / á pod fáłƺem utáiliśmy śię ;
Dlátegoż ták powiedźiáł pánujący Pán : Oto Já zá grunt kłádę w Syonie kámień / kámień doświádcżony / węgielny / koƺtowny / gruntownie ugruntowány ; kto wierzy / nie pokwápi śię.
A wykonám ſąd według ƺnuru / á ſpráwiedliwość według wági ; y potłucże grád nádźieję omylną / á ućiecżkę wody zátopią.
A ták zgłádzone będźie przymierze wáƺe z śmierćią / á porozumienie wáƺe z piekłem nie oſtoi śię ; gdy bicż gwáłtowny przechodźić będźie / będźiećie od niego podeptáni.
Kiedy jedno pocżnie przechodźić / pochwyći wáſ ; bo ná káżdy poránek przechodźić będźie we dnie y w nocy. A ſám poſtrách przywiedźie wáſ ku zrozumieniu tego / cośćie ſłyƺeli ;
Zwłáƺcżá iż krótƺe będźie łoże / niżby śię kto mógł rozćiągnąć / y nákryćie wąſkie / choćby śię ſkurcżył.
Abowiem Pán powſtánie jáko ná górze Perázym / á rozgniewá śię jáko w dolinie Gábáon / áby wykonáł ſpráwę ſwoję / niezwycżájną ſpráwę ſwoję / y áby dokońcżył ſpráwy ſwojej / niezwycżájnej ſpráwy ſwojej.
A ták teráz nie náśmiewájćie śię / áby śię niezmocniły związki wáƺe / bom o pewnem zepſowániu wƺyſtkiej źiemi ſłyƺáł od Páná / Páná záſtępów /
Nádſtáwiájćie uƺów / á ſłuchájćie głoſu mego ; bądźćie pilni / á ſłuchájćie mowy mojej.
Izáli káżdego dniá orácż orze / áby śiáł? przegániá brozdy / á włócży rolę ſwoję?
Izáli zrównáwƺy wierzch jej / nie rozśiewá wyki / y nie roztrząſá kminu / y nie śieje pƺenicy wybornej / y jęcżmieniá przedniego / y orkiƺu ná miejſcu ſpoſobnem?
Bo go ucży roztropnośći Bóg jego / y náucżá go.
Wyki nie młócą okowánem nácżyniem / áni tácżáją kołá wozowego po kminie ; ále kijem wybijáją wykę / á kmin láſką.
Pƺenicá młóconá bywá ; wƺákże y tej nie záwżdy młóćić będźie / áni jej potrze kołem wozá ſwego / áni jej zębámi jego pokruƺy.
Y toć od Páná záſtępów wyƺło / który jeſt dźiwny w rádźie / á wielmożny w rzecży ſámej.
