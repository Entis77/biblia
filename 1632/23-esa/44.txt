A teráz ſłucháj Jákóbie ſługo mój! y ty / Izráelu! któregom wybráł.
Ták mówi Pán / który ćię ucżynił / y który ćię ukƺtáłtowáł záráz z żywotá mátki / y który ćię wſpomágá : Nie bój śię Jákóbie / ſługo mój! y uprzejmy / któregom wybráł.
Bo wyleję wody ná prágnącego / á potoki ná ſuchą ziemię ; wyleję Duchá mego ná náśienie twoje / y błogoſłáwieńſtwo moje ná potomki twoje.
Y rozkrzewią śię jáko między tráwą / y jáko wierzby przy ćiekących wodách.
Ten rzecże : Jám jeſt Páńſki / á ów śię ozowie do imieniá Jákóbowego / á inny śię zápiƺe ręką ſwą Pánu / y imieniem Izráelſkiem będzie śię názywáł.
Ták mówi Pán / król Izráelſki / y odkupićiel jego / Pán záſtępów : oprócż mnie niemáƺ Bogá.
Bo któż jáko Já ogłáƺá y opowiádá to / y ſporządzá mi to / záráz od onego cżáſu / jákom rozſądził lud ná świećie? á kto przyƺłe rzecży / y to / co má być / oznájmi im?
Nie bójćież śię / áni ſobą trwożćie. Izálim wám tego z dáwná nie oznájmił / y nie opowiedziáł? Tegośćie wy mnie ſámi świádkámi. Izáli jeſt Bóg oprócż mnie? Niemáƺ záiſte ſkáły ; Já o żádnej nie wiem.
Tworzyćiele báłwánów wƺyſcy nic nie ſą / y te nájmilƺe rzecży ich nic im nie pomogą ; cżego oni ſobie ſámi świádkámi będąc / nic nie widzą / áni rozumieją / żeby śię wſtydzić mogli.
Kto tworzy bogá / y báłwáná leje / do nicżego śię to nie przygodzi.
Oto wƺyſcy / y ucżeſtnicy ich będą poháńbieni ; owƺem / rzemieślnicy ich / ći nád innych ludzi / choćby śię wƺyſcy zebráli y ſtánęli / lękáć śię muƺą / y ſpołem poháńbieni będą.
Kowál kleƺcżámi robi przy węglu / á młotámi kƺtáłtuje báłwáná ; gdy go robi mocą rámieniá ſwego / áż od głodu w nim y śiły uſtáją / áni pije wody / áż y omdlewá.
Cieślá záś rozćięgá ƺnur / znácży ƺnurem fárbowánym / y oćioſuje toporem / y cyrklem rozmierzá go / y cżyni go ná podobieńſtwo mężá / y ná podobieńſtwo pięknego cżłowieká / áby mieƺkáł w domu.
Nárąbie ſobie cedrów / y bierze cypryſ y dąb / álbo to / co jeſt nájmocniejƺego między drzewem leśnem / álbo wſádzi jáwór / który zá deƺcżem odráſtá ;
Y używá tego cżłowiek do páleniá / álbo wziąwƺy z niego / ogrzewá śię przy nim / tákże rozniecá ogień / áby nápiekł chlebá / nádto z tegoż drzewá robi ſobie bogá / y kłániá mu śię ; cżyni z niego báłwáná / y klęká przed nim.
Cżęść jego páli ogniem / przy drugiej cżęśći jego mięſo je / piecże piecżeń y náſycony bywá ; tákże rozgrzewá śię / y mówi : Ehej! rozgrzáłem śię / widziáłem ogień.
A z oſtátku jego cżyni bogá / báłwáná ſwego ; klęká przed nim / kłániá śię / y modli mu śię / mówiąc : Wybáw mię / boś ty bóg mój.
Nie wiedzą / áni rozumieją / przeto / że Bóg záślepił ocży ich / áby nie widzieli / y ſercá ich / áby nie rozumieli.
Y nie uwáżáją tego w ſercu ſwojim / nie májąż to umiejętnośći áni bácżeniá / áby rzekli : Cżęść z niego ſpáliłem ogniem / á przy węglu jego nápiekłem chlebá / upiekłem mięſo / y nájádłem śię ; y mámże já z oſtátku jego obrzydliwość ucżynić / á przed klocem drewniánym klękáć?
Táki śię kármi popiołem / ſerce jego zwiedzione unośi go / áby nie mógł wybáwić duƺy ſwojej / áni rzec : Izáli to nie jeſt oƺukánie / co jeſt w práwicy mojej?
Pomnijże ná to / Jákóbie y Izráelu! przeto / żeś ty jeſt ſługą moim. Stworzyłem ćię / ſługáś ty mój ; o Izráelu! nie zápomnę ná ćię.
Głádzę niepráwośći twoje jáko obłok / á grzechy twoje jáko mgłę ; náwróć śię do mnie / bom ćię odkupił.
Śpiewájćie niebioſá / bo to Pán ucżynił ; wykrzykájćie niſkośći ziemi / zábrzmijćie chwáłę góry / láſ / y wƺyſtkie drzewá w nim ; ábowiem Pán odkupił Jákóbá / á w Izráelu ſłáwnym śię ucżynił.
Ták mówi Pán / odkupićiel twój / y który ćię utworzył wnet z żywotá mátki : Já Pán wƺyſtko cżynię / ſám rozćiągám niebioſá / rozpośćierám ziemię mocą ſwoją.
Wniwecż obrácám známioná práktykárzów / y wieƺcżków do ƺáleńſtwá przywodzę ; y mędrców ná wſtecż obrácám / á umiejętność ich głupią cżynię.
Potwierdzám ſłowá ſługi ſwego / á rádę poſłów ſwych wykonywám. Który mówię o Jeruzálemie : Mieƺkáć w nim będę ; á o miáſtách Judzkich : Pobudowáne będą ; bo ſpuſtoƺeniá ich pobuduję ;
Który mówię głębinie : Wyſchnij / Já potoki twe wyſuƺę ;
Który mówię o Cyruśie : On páſterz mój / bo wƺyſtkę wolę moję wykoná ; y rzecże Jeruzálemowi : Będzieƺ zbudowáne ; á kośćiołowi : Będzieƺ záłożony.
