Oto ſługá mój / ſpolegáć będę ná nim / wybrány mój / którego ſobie upodobáłá duƺá mojá. Dam mu Duchá ſwego / on ſąd národom wydá.
Nie będźie wołał / áni śię będźie wywyżƺał / áni będźie ſłyƺány ná ulicy głos jego.
Trzćiny náłámáney nie dołámie / á lnu kurzącego śię nie dogáśi ; ále ſąd wydá według prawdy.
Nie zámrocży śię / áni uſtánie / dokąd nie wykoná ſądu ná źiemi / á náuki jego wyſpy ocżekiwáć będą.
Ták mówi Bóg / Pán / który ſtworzył niebioſá y rozpoſtárł je ; który rozƺerzył źiemię / y co śię rodźi z niej ; który dáje tchnienie ludowi mieƺkájącemu ná niej / á duchá tym / co chodzą po niej.
Ja Pán wezwáłem ćię w ſpráwiedliwośći / y ująłem ćię zá rękę twą ; przetoż ſtrzedz ćię będę / y dam ćię zá przymierze ludowi / y zá świátłość národom.
Aby otwierał ocży ślepych / á wywodźił więźniów z ćiemnicy / y z domu więźieniá śiedzących w ćiemnośćiách.
Ja Pán / toć jeſt imię moje / á chwały mojey nie dam innemu / áni ſłáwy mojey báłwánom rytym.
Oto pierwƺe rzecży przyƺły / Ja też nowe opowiádám / pierwej / niż śię zácżną / dam wam o nich ſłyƺeć.
Śpiewájćie Pánu pieśń nową / chwałá jego jeſt od końcżyn źiemi / którzy śię płáwićie po morzu / y wƺyſtko / co w nim jeſt / wyſpy y obywátele ich.
Podnieśćie głos puſtynie / y miáſtá jey / y wśi / w których mieƺká Kedár ; wykrzykájćie obywátele ſkał / z wierzchu gór wołájćie.
Oddájćie cżeść Pánu / á chwałę jego ná wyſpách opowiádájćie.
Pán wynijdźie jáko mocárz / jáko mąż wálecżny wzruƺy śię gorliwośćią ; trąbić / owƺem krzycżeć będźie / á przećiw nieprzyjáćiołom ſwoim mężnie ſobie pocżnie /
Mówiąc : Milcżáłem dość długo / jákobym nie ſłyƺał / wſtrzymywáłem śię ; ále już jáko rodzącá krzycżeć będę / ſpuſtoƺę / y wƺyſtkich oráz połknę.
W puſtynie góry y págórki obrócę / y wƺyſtkie źiołá ich poſuƺę ; obrócę y rzeki w wyſpy / á jeźiorá wyſuƺę.
Y powiodę ślepych drogą / którey nie ználi / á śćieƺkámi / o których nie wiedźieli / poprowádzę ich ; obrócę przed nimi ćiemnośći w świátłość / á co nierównego / w równinę. Toć jeſt / co im ucżynię / á nie opuƺcżę ich.
Cofną śię názád / y záwſtydzą śię bárzo / którzy ufáją w báłwánách rytych / którzy mówią obrázom litym : Wyśćie bogowie náśi.
O głuśi! ſłuchájćie ; á wy ślepi! przejrzyjćie / ábyśćie widźieli.
Któż ślepy / jedno ſługá mój? á kto głuchy / jedno poſeł mój / którego poſyłám? Któż ták ślepy jáko doſkonáły / ślepy / mówię / jáko ſługá Páńſki?
Widźi wiele rzecży / á wƺákże nie zrozumiewá ; otworzone ma uƺy / wƺákże nie ſłyƺy.
Pán go ſobie upodobał dla ſpráwiedliwośći ſwojey ; uwielbił go zakonem / y ſłáwnym go ucżynił.
Ale ten lud jeſt złupiony y rozƺárpány / którego młodźieńców ile ich kolwiek jeſt / imáją / y do ćiemnic podáwáją ; podáni ſą ná łup / á niemáƺ ktoby ich wybáwił ; podáni ſą ná rozchwycenie / áni jeſt / ktoby rzekł : Wróć ich záś.
Któż to z was w uƺy przyjmuje? kto zrozumiewá / áby cżulƺym był nápotem?
Kto podał ná rozƺárpánie Jákóbá / á Izráelá łupieżcom? Izali nie Pán / przećiwko któremuśmy zgrzeƺyli? Bo nie chćieli drogámi jego chodźić / áni ſłucháć zakonu jego.
Dla tego nań Pán wylał popędliwość gniewu ſwego / y gwáłtowną wojnę / á zápálił go w około / á wƺákże nie poznał tego ; zápálił go / mówię / á wƺákże tego do ſercá nie przypuśćił.
