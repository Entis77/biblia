Oto ſługá mój / ſpolegáć będę ná nim / wybrány mój / którego ſobie upodobáłá duƺá mojá. Dám mu Duchá ſwego / on ſąd národom wydá.
Nie będźie wołáł / áni śię będźie wywyżƺáł / áni będźie ſłyƺány ná ulicy głoſ jego.
Trzćiny náłámánej nie dołámie / á lnu kurzącego śię nie dogáśi ; ále ſąd wydá według práwdy.
Nie zámrocży śię / áni uſtánie / dokąd nie wykoná ſądu ná źiemi / á náuki jego wyſpy ocżekiwáć będą.
Ták mówi Bóg / Pán / który ſtworzył niebioſá y rozpoſtárł je ; który rozƺerzył źiemię / y co śię rodźi z niej ; który dáje tchnienie ludowi mieƺkájącemu ná niej / á duchá tym / co chodzą po niej.
Já Pán wezwáłem ćię w ſpráwiedliwośći / y ująłem ćię zá rękę twą ; przetoż ſtrzedz ćię będę / y dám ćię zá przymierze ludowi / y zá świátłość národom.
Aby otwieráł ocży ślepych / á wywodźił więźniów z ćiemnicy / y z domu więźieniá śiedzących w ćiemnośćiách.
Já Pán / toć jeſt imię moje / á chwáły mojej nie dám innemu / áni ſłáwy mojej báłwánom rytym.
Oto pierwƺe rzecży przyƺły / Já też nowe opowiádám / pierwej / niż śię zácżną / dám wám o nich ſłyƺeć.
Śpiewájćie Pánu pieśń nową / chwáłá jego jeſt od końcżyn źiemi / którzy śię płáwićie po morzu / y wƺyſtko / co w nim jeſt / wyſpy y obywátele ich.
Podnieśćie głoſ puſtynie / y miáſtá jej / y wśi / w których mieƺká Kedár ; wykrzykájćie obywátele ſkáł / z wierzchu gór wołájćie.
Oddájćie cżeść Pánu / á chwáłę jego ná wyſpách opowiádájćie.
Pán wynijdźie jáko mocárz / jáko mąż wálecżny wzruƺy śię gorliwośćią ; trąbić / owƺem krzycżeć będźie / á przećiw nieprzyjáćiołom ſwoim mężnie ſobie pocżnie /
Mówiąc : Milcżáłem dość długo / jákobym nie ſłyƺáł / wſtrzymywáłem śię ; ále już jáko rodzącá krzycżeć będę / ſpuſtoƺę / y wƺyſtkich oráz połknę.
W puſtynie góry y págórki obrócę / y wƺyſtkie źiołá ich poſuƺę ; obrócę y rzeki w wyſpy / á jeźiorá wyſuƺę.
Y powiodę ślepych drogą / której nie ználi / á śćieƺkámi / o których nie wiedźieli / poprowádzę ich ; obrócę przed nimi ćiemnośći w świátłość / á co nierównego / w równinę. Toć jeſt / co im ucżynię / á nie opuƺcżę ich.
Cofną śię názád / y záwſtydzą śię bárdzo / którzy ufáją w báłwánách rytych / którzy mówią obrázom litym : Wyśćie bogowie náśi.
O głuśi! ſłuchájćie ; á wy ślepi! przejrzyjćie / ábyśćie widźieli.
Któż ślepy / jedno ſługá mój? á kto głuchy / jedno poſeł mój / którego poſyłám? Któż ták ślepy jáko doſkonáły / ślepy / mówię / jáko ſługá Páńſki?
Widźi wiele rzecży / á wƺákże nie zrozumiewá ; otworzone má uƺy / wƺákże nie ſłyƺy.
Pán go ſobie upodobáł dlá ſpráwiedliwośći ſwojej ; uwielbił go zákonem / y ſłáwnym go ucżynił.
Ale ten lud jeſt złupiony y rozƺárpány / którego młodźieńców ile ich kolwiek jeſt / imáją / y do ćiemnic podáwáją ; podáni ſą ná łup / á niemáƺ ktoby ich wybáwił ; podáni ſą ná rozchwycenie / áni jeſt / ktoby rzekł : Wróć ich záś.
Któż to z wáſ w uƺy przyjmuje? kto zrozumiewá / áby cżulƺym był nápotem?
Kto podáł ná rozƺárpánie Jákóbá / á Izráelá łupieżcom? Izáli nie Pán / przećiwko któremuśmy zgrzeƺyli? Bo nie chćieli drogámi jego chodźić / áni ſłucháć zákonu jego.
Dlátego náń Pán wyláł popędliwość gniewu ſwego / y gwáłtowną wojnę / á zápálił go w około / á wƺákże nie poznáł tego ; zápálił go / mówię / á wƺákże tego do ſercá nie przypuśćił.
