Biádá ſynom odpornym / mówi Pán / którzy cżynią rádę / ále nie ze mnie / y nákrywáją ją nákryćiem / ále nie z duchá mojego / áby przycżyniáli grzechu do grzechu.
Którzy chodzą á zſtępują do Egiptu / nie rádząc śię uſt moich / áby śię zmocnili mocą Fáráonową / y ukryli śię w ćieniu Egipſkim.
Bo moc Fáráonowá będźie wám ku záwſtydzeniu / á ućiecżká do ćieniu Egipſkiego ku poháńbieniu.
Przeto / że byli w Soán kśiążętá jego / á poſłowie jego do Cháneſ chodźili.
Wƺyſtkich do háńby przywiedźie przez lud / który im nie będźie ku dogodźie / áni ku pomocy / áni ku pożytkowi / ále tylko ku zelżywośći y ku háńbie.
Brzemię odnioſą ná bydlętách południowych do źiemi ućiſku y utrápieniá / ( ſkąd pochodźi lew y ƺcżnię lwie / żmijá y ſmok ogniſty látájący ; ) odnioſą mówię ná grzbietách bydlątek bogáctwá ſwoje / y ná gárbie wielbłądów ſkárby ſwoje / do ludu / który im nic nie pomoże ;
Bo Egipcżánie dáremno y próżno pomágáć będą. Dlátego to ogłáƺám / że ich moc jeſt / śiedźieć w pokoju.
Terazże idź / nápiƺ to ná táblicy przed ocżymá ich / á ná kśięgách to wyryſuj / áby to trwáło do dniá oſtátniego / y áż ná wieki wieków :
Że ten lud jeſt odporny / ſynowie kłámliwi / ſynowie / którzy nie chcą ſłucháć zakonu Páńſkiego.
Którzy mówią widzącym : Nie miewájćie widzeniá ; á prorokom : Nie prorokujćie nám / co práwego jeſt ; mówćie nám rzecży przyjemne / prorokujćie oƺukánie ;
Uſtąpćie z drogi / zejdźćie z śćieƺki ; niech będźie dáleki od oblicżá náƺego Święty Izráelſki.
Przetoż ták mówi Święty Izráelſki : Iż gárdźićie tem ſłowem / á ufáćie w potwárzy y w przewrotnośći / y ſpolegáćie ná niej :
Dlátego wám tá niepráwość będźie jáko mur przerwány upádájący / y jáko wydęćie ná murze wyſokim / którego bywá nágłe y prędkie obálenie ;
Y pokruƺy ją / jáko śię kruƺy ſtłucżone nácżynie gárncárſkie ; á ták mu nie ſfolguje / iż śię nie znájdźie po ſtłucżeniu jego y ſkorupá / którąby mógł nábráć ogniá z ogniſká / álbo nácżerpáć wody z káłuży.
Abowiem ták mówi pánujący Pán / Święty Izráelſki : Jeſli śię náwróćićie y uſpokoićie śię / záchowáni będźiećie ; w milcżeniu y w nádźiei będźie moc wáƺá. Ale nie chcećie ;
Owƺem mówićie : Nie ták / ále ná koniách ućiecżemy ; przetoż ućiekáć będźiećie. Ná prędkich koniách ujedźiemy ; ále prędśi będą ći / którzy was gonić będą.
Tyśiąc ich ućiecże przed okrzyknieniem jednego / á przed okrzyknieniem pięćiu wƺyſcy ućiecżećie / áż zoſtániećie jáko máƺt ná wierzchu góry / á jáko chorągiew ná págórku.
A dlátegoć Pán cżekáć będźie / áby śię zmiłowáł nád wámi / y dlátego śię wywyżƺy / áby śię zlitowáł nád wámi ; ábowiem Pán jeſt Bogiem ſądu ; błogoſłáwieni wƺyſcy / którzy náń ocżekują.
Bo lud ná Syonie y w Jeruzálemie będźie mieƺkáć ; płákáć więcey nie będźieƺ. Zápewne zlituje śię nád tobą ná głoſ wołánia twego ( Pán ) / á ſkoro uſłyƺy / ozwieć śię.
A choć wám Pán dá chleb utrápieniá / y wodę ućiſku / jednák nie odlecą więcey od ćiebie náucżyćiele twoi / ále ocży twoje pátrzáć będą ná náucżyćieli twoich ;
Y uƺy twoje uſłyƺą ſłowo z tyłu do ćiebie mówiącego : Táć jeſt drogá / chodźćie po niej / lubbyśćie śię w práwo álbo w lewo událi.
Tedy zárzućićie okryćie ſrebrnych ſwoich báłwánów rytych / y odźienie złotych ſwoich báłwánów odlewánych ; rozproƺyƺ je jáko plugáſtwo niewiáſty przyrodzoną niemoc ćierpiącey / á rzecżeƺ im : Precż ſtąd.
Dá Bóg y deƺcż ná śiewy twoje / któremibyś pośiáł źiemię / á chleb z urodzáju źiemi będźie ſyty y obfity ; dniá onego páść śię będą y bydłá twoje ná páſtwiſku ƺerokiem.
Woły tákże y oſły ſpráwujące źiemię / páſtwę cżyſtą jeść będą / która opáłką y łopátą wywiáná bywá.
Y będą ná káżdej górze wyſokiej / y ná káżdym págórku wynioſłym ſtrumienie y potoki wód w dźień poráżki wielkiej / gdy wieże upádną.
Świátłość też mieśiącá będźie jáko świátłość ſłonecżná ; á świátłość ſłonecżná będźie w śiedmiornáſób / jáko świátłość śiedmiu dni / dniá / którego záwiąże Pán złámánie ludu ſwego / á ránę zbićiá jego ulecży.
Oto imię Páńſkie przychodźi z dáleká / zápáliłá śię popędliwość jego / y ćiężká jeſt ku znoƺeniu ; wárgi jego pełne ſą gniewu / á język jego jáko ogień pożerájący.
A duch jego jeſt jáko rzeká wylewájącá / która áż do gárdłá śięgá / áby przewiewáł národy / áżby śię wniwecż obróćiły / á wędźidłem kiełznáł cżeluśći národów.
Tedy záśpiewáćie / jáko gdy śię w nocy obchodźi urocżyſte święto / á rozweſelićie śię w ſercu jáko ten / który idźie z piƺcżáłką / idąc ná górę Páńſką / do ſkáły Izráelſkiej ;
Gdy dá uſłyƺeć Pán wielmożność głoſu ſwego / y wyćiągnione rámię ſwoje okáże w popędliwośći gniewu ſwojego / y w płomieniu ogniá pożerájącego z rozproƺeniem / z gwáłtownym dżdżem / y z grádem kámiennym.
Bo od głoſu Páńſkiego ſtárty będźie Aſſyryjcżyk / który innych kijem bijáł.
Y ſtánie śię / że ná ſámym káżde uderzenie kijowe / którem go Pán uderzy / znácżne będźie / gdy z bębnámi y z lutniámi / y z bitwą weſołą wálcżyć będźie przećiwko niemu.
Abowiem dáwno już jeſt nágotowáne piekło / y dla ſámego królá nágotowáne jeſt ; które głębokie y ƺerokie ucżynił / podniáty jego ogniá y drew śiłá jeſt ; poddymánie Páńſkie jáko rzeká śiárcżáná zápálá je.
