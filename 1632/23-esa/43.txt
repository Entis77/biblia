Ale teraz ták mówi Pán / który ćię ſtworzył / o Jákóbie ; y który ćię ucżynił / o Izráelu! Nie bój śię / bom ćię odkupił / á wezwáłem ćię imieniem twojim ; mójeś ty.
Gdy pójdźieƺ przez wody / będę z tobą / á jeſli przez rzeki / nie záleją ćię ; pójdźieƺli przez ogień / nie ſpáliƺ śię / á płomień nie imie śię ćiebie.
Bom Ja Pán / Bóg twój / Święty Izráelſki / zbáwićiel twój. Dáłem zá ćię ná okup Egipt / źiemię Murzyńſką / y Sábę miáſto ćiebie.
Záráz jákoś drogim ucżyniony przed ocżymá memi / jeſteś uwielbionym / á Jam ćię umiłował ; przetoż dáłem ludźi zá ćię / y národy zá żywot twój.
Nie bój śię / bom Ja z tobą ; od wſchodu ſłońcá przyprowádzę záś náśienie twoje / y od záchodu zgromádzę ćię.
Rzekę północney ſtronie : Wróć : á południowi : Nie zábrániáj. Przywiedź záśię ſynów mojich z dáleká / á córki moje od końcżyn źiemi ;
Káżdego / który śię názywá imieniem mojim / y któregom ku chwále ſwojey ſtworzył / któregom ukƺtáłtował / y któregom ucżynił.
Wywiedź lud ślepy / który już ma ocży y głuchy / który już ma uƺy.
Wƺyſtkie národy niech śię ſpołu zejdą / y niech śię zgromádzą ludźie. Któż jeſt miedzy nimi / coby to opowiedźiał / á przeƺłe rzecży nam oznájmił? Niech ſtáwią świádków ſwojich / á będą uſpráwiedliwieni ; álbo niech ſłyƺą y rzekną : Práwdáć jeſt!
Wyśćie świádkowie moi / mówi Pán / y ſługá mój / któregom obrał / ábyśćie wiedźieli y wierzyli mi / y zrozumieli / żem Ja jeſt / á że przedemną nie był ſtworzony Bóg / áni po mnie będźie.
Ja / Jam jeſt Pán / á niemáƺ oprócż mnie zbáwićielá.
Ja oznájmuję y wyſwábádzám / y opowiádám / á niemáƺ nikogo miedzy wámi z obcych bogów ; y wyśćie mi tego świádkámi / mówi Pán / żem ja Bóg.
Pierwej niż dźień był / Jam jeſt / á niemáƺ / ktoby wyrwał z ręki mojey ; gdy co ucżynię / y któż to odwróći?
Ták mówi Pán / odkupićiel wáƺ / Święty Izráelſki : Dla was poślę do Bábilonu / y oderwę wƺyſtkie záwory / y Cháldejcżyków z okrętámi / w których śię oni chlubią.
Jam jeſt Pán / Święty wáƺ ; Stworzyćiel Izráelowy / Król wáƺ.
Ták mówi Pán / który ſpoſobił ná morzu drogę / y śćieƺkę ná byſtrych wodách.
Który wywodźi wozy y konie / wojſko y śiłę ; cżyni / że oráz upádáją / á nie powſtáwáją : gáſną jáko knot gáśnie.
Nie wſpominájćie pierwƺych rzecży / á ſtárodáwnych nie uwáżájćie.
Oto Ja cżynię rzecż nową / á záráz śię zjáwi ; izali tego nie poznáćie? Nádto ſpoſobię ná puƺcży drogę / á ná puſtyni rzeki.
Chwálić mię będźie zwierz polny / ſmoki / y ſowy / żem wywiódł ná puƺcży wody / á rzeki ná puſtyni / ábym dał nápój ludowi memu / wybránemu ludowi memu.
Lud ten / którym ſobie ſtworzył / chwałę moję opowiádáć będźie ;
A tyś mię nie wzywał / o Jákóbie! owƺemeś ſobie utęſknił ze mną / o Izráelu!
Nie przywiodłeś mi bydlątká ná cáłopálenie twoje / y ofiárámi twemi nie ucżćiłeś mię ; nie przymuƺáłem ćię / ábyś mi ſłużył ofiárámi śniednemi / ánim ćię obćiążał tym / ábyś mi kádźił ;
Nie kupiłeś mi zá pieniądze wonnych rzecży / ániś mię tłuſtośćią ofiár twojich opoił ; áleś mię obćiążył grzechámi twemi / á zádáłeś mi prácę niepráwośćiámi twojimi.
Ja / Ja ſám głádzę przeſtępſtwá twoje dla śiebie / á grzechów twojich nie wſpomnę.
Przywiedź mi ná pámięć / ſądźmy śię ſpołu ; powiedz ty / máƺli / cżembyś śię uſpráwiedliwił?
Oćiec twój pierwƺy zgrzeƺył / á náucżyćiele twoi wyſtąpili przećiwko mnie.
A ták zrzucę kśiążąt z miejſc świętych / y podám ná przeklęſtwo Jákóbá / á Izráelá ná poháńbienie.
