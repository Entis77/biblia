Ale teráz ták mówi Pán / który ćię ſtworzył / o Jákóbie ; y który ćię ucżynił / o Izráelu! Nie bój śię / bom ćię odkupił / á wezwáłem ćię imieniem twojim ; mójeś ty.
Gdy pójdzieƺ przez wody / będę z tobą / á jeźli przez rzeki / nie záleją ćię ; pójdzieƺli przez ogień / nie ſpáliƺ śię / á płomień nie imie śię ćiebie.
Bom Já Pán / Bóg twój / Święty Izráelſki / zbáwićiel twój. Dáłem zá ćię ná okup Egipt / ziemię Murzyńſką / y Sábę miáſto ćiebie.
Záráz jákoś drogim ucżyniony przed ocżymá memi / jeſteś uwielbionym / á Jám ćię umiłowáł ; przetoż dáłem ludzi zá ćię / y národy zá żywot twój.
Nie bój śię / bom Já z tobą ; od wſchodu ſłońcá przyprowádzę záś náśienie twoje / y od záchodu zgromádzę ćię.
Rzekę północnej ſtronie : Wróć : á południowi : Nie zábrániáj. Przywiedź záśię ſynów moich z dáleká / á córki moje od końcżyn ziemi ;
Káżdego / który śię názywá imieniem mojem / y któregom ku chwále ſwojej ſtworzył / któregom ukƺtáłtowáł / y któregom ucżynił.
Wywiedź lud ślepy / który już má ocży y głuchy / który już má uƺy.
Wƺyſtkie národy niech śię ſpołu zejdą / y niech śię zgromádzą ludzie. Któż jeſt między nimi / coby to opowiedziáł / á przeƺłe rzecży nám oznájmił? Niech ſtáwią świádków ſwoich / á będą uſpráwiedliwieni ; álbo niech ſłyƺą y rzekną : Práwdáć jeſt!
Wyśćie świádkowie moi / mówi Pán / y ſługá mój / któregom obráł / ábyśćie wiedzieli y wierzyli mi / y zrozumieli / żem Já jeſt / á że przedemną nie był ſtworzony Bóg / áni po mnie będzie.
Já / Jám jeſt Pán / á niemáƺ oprócż mnie zbáwićielá.
Já oznájmuję y wyſwábádzám / y opowiádám / á niemáƺ nikogo między wámi z obcych bogów ; y wyśćie mi tego świádkámi / mówi Pán / żem já Bóg.
Pierwej niż dzień był / Jám jeſt / á niemáƺ / ktoby wyrwáł z ręki mojej ; gdy co ucżynię / y któż to odwróći?
Ták mówi Pán / odkupićiel wáƺ / Święty Izráelſki : Dlá wáſ poślę do Bábilonu / y oderwę wƺyſtkie záwory / y Cháldejcżyków z okrętámi / w których śię oni chlubią.
Jám jeſt Pán / Święty wáƺ ; Stworzyćiel Izráelowy / Król wáƺ.
Ták mówi Pán / który ſpoſobił ná morzu drogę / y śćieƺkę ná byſtrych wodách.
Który wywodzi wozy y konie / wojſko y śiłę ; cżyni / że oráz upádáją / á nie powſtáwáją : gáſną jáko knot gáśnie.
Nie wſpominájćie pierwƺych rzecży / á ſtárodáwnych nie uwáżájćie.
Oto Já cżynię rzecż nową / á záráz śię zjáwi ; izáli tego nie poznáćie? Nádto ſpoſobię ná puƺcży drogę / á ná puſtyni rzeki.
Chwálić mię będzie zwierz polny / ſmoki / y ſowy / żem wywiódł ná puƺcży wody / á rzeki ná puſtyni / ábym dáł nápój ludowi memu / wybránemu ludowi memu.
Lud ten / którym ſobie ſtworzył / chwáłę moję opowiádáć będzie ;
A tyś mię nie wzywáł / o Jákóbie! owƺemeś ſobie utęſknił ze mną / o Izráelu!
Nie przywiodłeś mi bydlątká ná cáłopálenie twoje / y ofiárámi twemi nie ucżćiłeś mię ; nie przymuƺáłem ćię / ábyś mi ſłużył ofiárámi śniednemi / ánim ćię obćiążáł tem / ábyś mi kádził ;
Nie kupiłeś mi zá pieniądze wonnych rzecży / ániś mię tłuſtośćią ofiár twoich opoił ; áleś mię obćiążył grzechámi twemi / á zádáłeś mi prácę niepráwośćiámi twojimi.
Já / Já ſám głádzę przeſtępſtwá twoje dlá śiebie / á grzechów twoich nie wſpomnę.
Przywiedź mi ná pámięć / ſądźmy śię ſpołu ; powiedz ty / máƺli / cżembyś śię uſpráwiedliwił?
Ojćiec twój pierwƺy zgrzeƺył / á náucżyćiele twoi wyſtąpili przećiwko mnie.
A ták zrzucę kśiążąt z miejſc świętych / y podám ná przeklęſtwo Jákóbá / á Izráelá ná poháńbienie.
