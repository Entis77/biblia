Słuchájćie mię wyſpy / á narody dálekie pilnujćie! Pán záráz z żywotá wezwał mię / záráz z żywotá mátki mojey ucżynił wzmiánkę imienia mego :
Y ucżynił uſtá moje jáko miecż oſtry / w ćieniu ręki ſwej zákrył mię / á ucżyniwƺy mię ſtrzáłą wypolerowáną / do ſájdáku ſwego ſchował mię ;
Y rzekł mi : Sługáś ty mój / w Izráelu tobą śię chlubić będę.
A Jam rzekł : Nádármom prácował / próżnom y dáremnie zniƺcżył śiłę moję ; wƺákże ſąd mój jeſt u Páná / á prácá mojá u Bogá mego.
A teraz mówi Pán / który mię záráz z żywotá zá ſługę ſobie utworzył / ábym záś przywiódł do niego Jákóbá. ( Choćby Izráel nie był zebrány / ſłáwnym jednák będę przed ocżymá Páńſkimi ; ábowiem Bóg mój jeſt śiłą moją. )
Y rzekł : Máłoby mi to było / ábyś mi był ſługą ku podźwignieniu pokoleń Jákóbowych / y ku náwróceniu oſtátków z Izráelá ; przetoż dáłem ćię zá świátłość pogánom / ábyś był zbáwieniem mojim áż do końcżyn źiemi.
Ták mówi Pán / odkupićiel Izráelowy / Święty jego / do tego / którym káżdy gárdźi / á którym śię brzydzą narody / do ſługi pánujących : Królowie widząc ćię powſtáną / á kśiążętá kłániáć ći śię będą dla Páná / który jeſt wierny / dla Świętego Izráelſkiego / który ćię obrał.
Ták mówi Pán : Cżáſu przyjemnego wyſłuchám ćię / á w dźień zbáwieniá porátuję ćię ; nádto ſtrzedz ćię będę / y dam ćię zá przymierze ludowi / ábyś utwierdźił źiemię / á podał w ośiádłość dźiedźictwá ſpuſtoƺáłe ;
Abyś mówił więźniom : Wynijdźćie ; á tym / co ſą w ćiemnośćiách : Okáżćie śię. Podle dróg páść śię będą / á po wƺyſtkich miejſcách wyſokich będą páſtwiſká ich.
Nie będą łáknąć / áni prágnąć / y nie uderzy ná nich gorącość / áni ſłońce / bo ten / który ma litość nád nimi / poprowádźi ich / y podle źródeł wód powiedźie ich.
Nádto ſpoſobię ná wƺyſtkich górách mojich drogę / á gośćińce moje będą podwyżƺone.
Oto ći z dáleká przyjdą / á oto drudzy od północy y od morzá / á drudzy z źiemi Synim.
Śpiewájćie niebioſá / rozráduj śię źiemio / y głośno zábrzmijćie góry! ábowiem Pán poćieƺył lud ſwój / á nád ubogimi ſwoimi zmiłował śię.
Ale Syon rzekł : Opuśćił mię Pán / á Pán zápomniał ná mię.
Izali może zápomnieć niewiáſtá niemowlątká ſwego / áby śię nie zlitowáłá nád płodem żywotá ſwego? A choćby też y one zápomniáły / wƺákże Ja ćiebie nie zápomnę.
Oto ná dłoniách ſwojich wyryſowáłem ćię ; mury twoje záwżdy ſą przedemną.
Poſpieƺą śię do ćiebie ſynowie twoi / á ći / którzy ćię burzyli y káźili / odejdą od ćiebie.
Podnieś w około ocży ſwe / á obácż ; ći wƺyſcy zgromádźiwƺy śię przyjdą do ćiebie. Jákom żywy Ja / mówi Pán / że tymi wƺyſtkimi jáko ochędóſtwem przyodźiejeƺ śię / y obłożyƺ śię nimi jáko oblubienicá ;
Przeto / że puſtynie twoje / y ſpuſtoƺáłe miejſcá twoje / y źiemiá zburzeniá twego teraz będą ćiáſne dla obywáteli / gdyż oddaleni będą ći / którzy ćię pożeráli.
Ták / że rzeką w uƺy twoje ſynowie śieroctwá twego : Ciáſne mi jeſt to miejſce ; uſtąpże mi / ábym mieƺkáć mógł.
Y rzecżeƺ w ſercu ſwem : Któż mi tych nápłodźił? bom ja byłá ośieroćiáłá / y ſámotná / wygnánam byłá / y tułáłám śię ; któż wżdy tych odchował? Otom Ja tylko ſámá pozoſtáłá byłá / gdźież ći byli?
Ták mówi pánujący Pán : Oto wznioſę ná narody rękę moję / á do ludźi podnioſę choręgiew moję / áby przynieśli ſynów twojich ná ręku / y córki twoje áby ná rámionách przynoƺone były.
Y będą królowie piáſtunámi twoimi / á kśiężny ich mámkámi twemi ; twárzą ku źiemi kłániáć ći śię będą / y proch nóg twojich lizáć będą ; á dowieƺ śię / żem Ja Pán / á iż nie bywáją záwſtydzeni / którzy ná mię ocżekują.
Y rzecżeƺ : Izali korzyść od mocárzá odjętá będźie? Izali pojmány lud ſpráwiedliwego wybáwiony będźie?
Owƺem / ták mówi Pán : Y pojmány lud mocárzowi odjęty będźie / y korzyść okrutnikowi wydártá będźie ; ábowiem przećiwnikowi twemu Ja śię ſprzećiwię / á ſynów twojich Ja wyſwobodzę.
Y tych / którzy ćię puſtoƺą / włáſnem ich ćiáłem nákármię / á krwią ſwoją jáko moƺcżem upiją śię. Y pozná wƺelkie ćiáło / żem Ja Pán / zbáwićiel twój / y odkupićiel twój / mocny Jákóbowy.
