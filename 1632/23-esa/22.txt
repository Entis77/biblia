Brzemię doliny widzeniá. Cóż ći śię ſtáło / żeś wƺyſtká ná dáchy wſtąpiłá?
Miáſto pełne wrzáſku / y zgiełku / miáſto weſelące śię! Pobići twoi nie ſą pobići miecżem / áni zginęli w bitwie.
Wƺyſcy kśiążętá twoi náporząd śię rozpierzchnęli / od ſtrzelców powiązáni ſą weſpół / y ći / którzy z dáleká ućiekáją.
Dlátegom rzekł : Odſtąpćie odemnie / ábym gorzko płákał ; nie kwápćie śię / ćieƺyć mię w ſpuſtoƺeniu córki ludu mojego.
Abowiem to jeſt dźień ućiſku y podeptánia / y zámięƺánia od Páná / Páná zaſtępów / w dolinie widzeniá / dźień burzeniá murów / y wołánia ná góry.
Elám też wźiął ſájdák z wozámi ludu wojennego / á Kir okázał tárcżę ſwoję.
Y ſtáło śię / że wyborne doliny twoje nápełnione były wozámi / á jezdni śię potężnie záƺáńcowáli u brámy.
Y odkrytá byłá záſłoná Judowá ; á poglądáłeś dniá onego ná zbrojownię w domu láſu.
Y poglądáliśćie ná rozwáliny miáſtá Dawidowego / bo ich wiele było ; y zgromádźiły śię wody ſádzáwki dolney.
Tákże policżyliśćie domy w Jeruzálemie / á rozwáliliśćie domy ná opráwę murów.
Ucżyniliśćie też przekop miedzy dwomá murámi / dla wód ſtáwu ſtárego / á nie oglądáliśćie śię ná tego / co go ſpráwił / á tego / który go zdáwná zbudował / nie widźieliśćie.
Nádto / gdy wołał Pán / Pán zaſtępów / dniá onego do płácżu y do nárzekánia / y do obłyśieniá śię / y do przepáſánia śię worem ;
A oto rádość y weſele wáƺe / zábijáć woły / y bić owce / á jedząc mięſo / y pijąc wino / mówić : Jedzmy / pijmy / bo jutro pomrzemy.
Aleć to doƺło uƺów mojich / mówi Pán zaſtępów. Przetoż wam tá niepráwość nie będźie odpuƺcżoná ; áż pomrzećie / mówi Pán / Pán zaſtępów.
Ták mówi Pán / Pán zaſtępów : Idź / wnijdź do tego podſkárbiego / do Sobny / który jeſt przełożonym w domu / y rzecż :
Co ty tu máƺ? álbo kogo tu máƺ / żeś tu ſobie wykował grób? Wykowáłeś ſobie ná wyſokiem miejſcu grób ſwój / á wyſtáwiłeś ná ſkále przybytek ſwój?
Oto Pán / który ćię przykrył jáko zácnego mężá / á który ćię koƺtownie przyodźiał /
Prędko ćię zátocży jáko kulę do źiemi ƺerokiej y przeſtronney ; tám umrzeƺ / tám y wozy ſłáwy twojey zginą / o háńbo domu Páná ſwego!
A ták wypędzę ćię z ſtánowiſká twego / á z urzędu twego złożę ćię.
A dniá onego przyzwię ſługę ſwego Elijákimá / ſyná Helkijáƺowego ;
Y oblekę go w ƺátę twoję / y páſem twoim potwierdzę go / pánowánie też twoje dam w rękę jego ; y będźie zá ojcá obywátelom Jeruzálemſkim / y domowi Judzkiemu.
Y położę klucż domu Dawidowego ná rámieniu jego ; gdy otworzy / nikt nie záwrze / á gdy záwrze / nikt nie otworzy.
Y wbiję go jáko gwóźdź ná miejſcu pewnem / á będźie ſtolicą chwały domu ojcá ſwego.
A záwiśnie ná nim wƺyſtká ſłáwá domu ojcá jego / ſynowie y córki / y wƺyſtko nacżynie by nájmniejƺe / od nácżyniá / z którego piją / áż do káżdego nácżyniá winnego.
Dniá onego / mówi Pán zaſtępów / będźie wyjęty gwóźdź / który był wbity ná miejſcu pewnem / á będźie przyćięty y upádnie ; odćięte będźie y brzemię / które jeſt ná nim ; bo Pán mówił.
