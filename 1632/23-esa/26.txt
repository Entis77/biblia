Dniá onego śpiewáná będźie tá pieśń w źiemi Judzkiej : Mámy miáſto obronne / Bóg zbáwieniem opátrzył mury y báƺty jego.
Otwórzćie brámy / á niech wnijdźie náród ſpráwiedliwy / który ſtrzeże práwdy.
Cżłowieká ſpolegájącego ná tobie záchowywáƺ w pokoju / w pokoju mówię : bo w tobie ufá.
Miejćie nádźieję w Pánu áż ná wieki ; boć w Pánu / w Pánu jeſt ſkáłá wiecżná.
Ale poniżá mieƺkájącego ná wyſokośći ; miáſto wynioſłe poniżá / poniżá je áż do źiemi / y ſtrącá je áż do prochu ;
Depcże je nogá ; nogi ubogiego / ſtopá nędzników.
Śćieƺká ſpráwiedliwego jeſt proſtá ; proſtą drogą ſpráwiedliwego wyrównywáƺ.
Ná drodze ſądów twoich / Pánie! ocżekujemy ćię ; żądość duƺy náƺej jeſt do imieniá twego / y do wſpominániá ná ćię.
Duƺá mojá żądá ćię w nocy / owƺem / duchem ſwym / który jeſt we mnie / ráno ćię ƺukám ; ábowiem gdy śię ſądy twoje odpráwiáją ná źiemi / ſpráwiedliwośći śię ucżą obywátele okręgu źiemſkiego.
Gdy śię łáſká pokázuje niepobożnemu / nie ucży śię ſpráwiedliwośći ; w źiemi práwośći niepráwość cżyni / á nie dbá nic ná májeſtát Páńſki.
Pánie! choć wywyżƺoná jeſt ręká twojá / przećię tego nie widzą ; ujrząć / ále poháńbieni będą / zájrząc ludowi twemu ; nádto y ogień tych nieprzyjáćiół twoich pożre.
Pánie! zrządźiƺ nám pokój ; bo wƺyſtko / co śię dźiáło przy nas / cżyniłeś ku dobremu náƺemu.
Pánie Boże náƺ! pánowálić nád námi inni pánowie oprócż ćiebie ; ále myśmy tylko / w tobie ufájąc / wſpomináli ná imię twoje.
Pomárli / nie ożyją ; mártwymi będąc nie powſtáną / przeto / żeś ich náwiedźił y wykorzenił / y wygłádźił wƺyſtkę pámiątkę ich.
Rozmnożyłeś náród ; o Pánie! rozmnożyłeś náród ; uwielbionyś jeſt / ácżeś go był zápędźił ná wƺyſtkie gránice źiemi.
Pánie! w ućiſku ƺukáli ćię ; gdyś ich káráł / wylewáli modlitwy ſwe.
Jáko brzemienná / gdy śię przybliżá ku rodzeniu / boleje y wołá w boleśćiách ſwoich / tákeśmy byli przed oblicżem twojim / Pánie!
Pocżęliśmy / boleliśmy ; áleśmy tylko jákoby wiátr porodźili / á żádnegośmy wybáwieniá źiemi nie ſpráwili / y nie upádli mieƺkájący ná okręgu źiemſkim.
Ożyją umárli twoi / trupy moje wſtáną / gdy rzecżeƺ : Ocuććie śię / á śpiewájćie mieƺkájący w prochu! Abowiem roſá twojá będźie jáko roſá ná źiołách ; ále niezbożnych o źiemię uderzyƺ.
Idź / ludu mój! wnijdź do komór ſwoich / á zámknij drzwi twoje zá ſobą ; ſkryj śię ná málucżką chwilkę / dokąd nie przeminie rozgniewánie.
Abowiem oto Pán wychodźi z miejſcá ſwego / áby náwiedźił niepráwość mieƺkájących ná źiemi ; tedy źiemiá odkryje krew ſwoję / á nie zákryje dálej pobitych ſwoich.
