Y rzekł Pán do mnie : Weźmij ſobie kśięgi wielkie / á nápiƺ ná nich piſmem cżłowiecżem : Poſpieƺ śię do łupu / pokwáp śię do korzyśći.
Tedym wziął ſobie zá świádków wiernych Uryjáƺá kápłáná / y Zácháryjáƺá / ſyná Jeberechyjáƺowego.
Wtemem przyſtąpił do prorokini / którá pocżąwƺy porodziłá ſyná. Y rzekł Pán do mnie : Názów imię jego : Poſpieƺ śię do łupu / pokwáp śię do korzyśći.
Abowiem niżeli będzie umiáło to dziećię wołáć : Ojcże mój y mátko mojá / lud królá Aſſyryjſkiego pobierze bogáctwá Dámáƺku / y łupy Sámáryi.
Nádto rzekł jeƺcże Pán do mnie / mówiąc :
Poniewáż wzgárdził lud ten wody Syloe / które ćicho płyną / á wáſeli śię z Ráſyná / y ſyná Romelijáƺowego :
Przetoż oto Pán przywiedzie ná nich wody rzeki gwáłtownej y wielkiej / to jeſt królá Aſſyryjſkiego / y wƺyſtkę ſłáwę jego / ták / że wyſtąpi ze wƺyſtkich ſtrumieni ſwoich / á wyleje ze wƺyſtkich brzegów ſwoich.
Poćiecże y przez ziemię Judzką / wyleje á rozejdzie śię / áż do ƺyi wzbierze ; á rozƺerzone ſkrzydłá jego nápełnią ƺerokość ziemi twojej / o Immánuelu!
Zbierájćie śię národy / wƺákże potłumione będziećie. Przyjmujćie w uƺy wƺyſcy w dálekiej ziemi ; przepáƺćie śię / wƺákże potłumieni będziećie ; przepáƺćie śię / wƺákże potłumieni będziećie.
Wnijdźćie w rádę / á będzie rozerwáná ; námówćie śię / á nie oſtoi śię ; bo Bóg z námi.
Ták bowiem Pán rzekł do mnie / ująwƺy mię zá rękę / y dáł mi przeſtrogę / żebym nie chodził drogą ludu tego / mówiąc :
Nie mówćie : Sprzyśiężenie. Kiedykolwiek ten lud mówi : Sprzyśiężenie / nie ſtráchájćie śię jáko oni / áni śię lękájćie.
Páná záſtępów ſámego poświęcájćie ; á on niech będzie bojáźnią wáƺą / y on ſtráchem wáƺym.
A będzie wám poświęceniem ; ále kámieniem obráżeniá y opoką otrąceniá obydwom domom Izráelſkim / śidłem y śiećią obywátelom Jeruzálemſkim.
Y otrąći śię wielu ich o nie / upádną y ſkruƺeni będą / uśidlą śię á pojmáni będą.
Záwiąż to świádectwo / zápiecżętuj zákon między ucżniámi moimi.
Tedy będę ocżekiwáł Páná / który ſkrył oblicże ſwoje od domu Jákóbowego / y pocżekám go.
Oto já y dzieći / które mi dáł Pán / ſą ná znáki y ná cudá w Izráelu / od Páná záſtępów / który mieƺká ná górze Syon.
A ták jeźliby wám rzekli : Dowiádujćie śię od cżárowników y od wieƺcżków / którzy ƺepcą y márkocą / rzecżćie : Izáli śię nie má dowiádywáć lud u Bogá ſwego? ázáż umárłych miáſto żywych rádzić śię má?
Do zákonu rácżej y do świádectwá ; ále jeźli nie chcą / niechże mówią według ſłowá tego / w którym niemáƺ żádnej zorzy.
Dlácżego káżdy z nich utrápiony y zgłodniáły tułáćby śię muśiáł ; á będąc zgłodniáłym / ſám w ſobie gniewáć śię będzie / y złorzecżyć królowi ſwemu / y Bogu ſwemu / w górę poglądájąc.
A gdy ná ziemię ſpojrzy / oto ućiſk y ćiemność / záćmienie / biedá / y obácży / że jeſt wráżony do ćiemnośći.
