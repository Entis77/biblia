Y rzekł Pán do mnie : Weźmij ſobie kśięgi wielkie / á nápiƺ ná nich piſmem cżłowiecżem : Poſpieƺ śię do łupu / pokwáp śię do korzyśći.
Tedym wźiął ſobie zá świádków wiernych Uryjáƺá kápłáná / y Zácháryjáƺá / ſyná Jeberechyjáƺowego.
Wtemem przyſtąpił do prorokini / która pocżąwƺy porodźiłá ſyná. Y rzekł Pán do mnie : Názów imię jego : Poſpieƺ śię do łupu / pokwáp śię do korzyśći.
Abowiem niżeli będźie umiáło to dźiećię wołáć : Ojcże mój y mátko mojá / lud królá Aſſyryjſkiego pobierze bogáctwá Dámáƺku / y łupy Sámáryi.
Nádto rzekł jeƺcże Pán do mnie / mówiąc :
Poniewáż wzgárdźił lud ten wody Syloe / które ćicho płyną / á wáſeli śię z Ráſyná / y ſyná Romelijáƺowego :
Przetoż oto Pán przywiedźie ná nich wody rzeki gwáłtowney y wielkiej / to jeſt królá Aſſyryjſkiego / y wƺyſtkę ſłáwę jego / ták / że wyſtąpi ze wƺyſtkich ſtrumieni ſwoich / á wyleje ze wƺyſtkich brzegów ſwoich.
Poćiecże y przez źiemię Judzką / wyleje á rozejdźie śię / áż do ƺyi wzbierze ; á rozƺerzone ſkrzydłá jego nápełnią ƺerokość źiemi twojey / o Immánuelu!
Zbierájćie śię národy / wƺákże potłumione będźiećie. Przyjmujćie w uƺy wƺyſcy w dálekiej źiemi ; przepáƺćie śię / wƺákże potłumieni będźiećie ; przepáƺćie śię / wƺákże potłumieni będźiećie.
Wnijdźćie w rádę / á będźie rozerwáná ; námówćie śię / á nie oſtoi śię ; bo Bóg z námi.
Ták bowiem Pán rzekł do mnie / ująwƺy mię zá rękę / y dáł mi przeſtrogę / żebym nie chodźił drogą ludu tego / mówiąc :
Nie mówćie : Sprzyśiężenie. Kiedykolwiek ten lud mówi : Sprzyśiężenie / nie ſtráchájćie śię jáko oni / áni śię lękájćie.
Páná záſtępów ſámego poświęcájćie ; á on niech będźie bojáźnią wáƺą / y on ſtráchem wáƺym.
A będźie wám poświęceniem ; ále kámieniem obráżeniá y opoką otrąceniá obydwom domom Izráelſkim / śidłem y śiećią obywátelom Jeruzálemſkim.
Y otrąći śię wielu ich o nie / upádną y ſkruƺeni będą / uśidlą śię á pojmáni będą.
Záwiąż to świádectwo / zápiecżętuj zakon między ucżniámi moimi.
Tedy będę ocżekiwáł Páná / który ſkrył oblicże ſwoje od domu Jákóbowego / y pocżekám go.
Oto ja y dźieći / które mi dáł Pán / ſą ná znáki y ná cudá w Izráelu / od Páná záſtępów / który mieƺká ná górze Syon.
A ták jeſliby wám rzekli : Dowiádujćie śię od cżárowników y od wieƺcżków / którzy ƺepcą y márkocą / rzecżćie : Izáli śię nie má dowiádywáć lud u Bogá ſwego? ázáż umárłych miáſto żywych rádźić śię má?
Do zakonu rácżej y do świádectwá ; ále jeſli nie chcą / niechże mówią według ſłowá tego / w którym niemáƺ żádney zorzy.
Dlácżego káżdy z nich utrápiony y zgłodniáły tułáćby śię muśiáł ; á będąc zgłodniáłym / ſám w ſobie gniewáć śię będźie / y złorzecżyć królowi ſwemu / y Bogu ſwemu / w górę poglądájąc.
A gdy ná źiemię ſpojrzy / oto ućiſk y ćiemność / záćmienie / biedá / y obácży / że jeſt wráżony do ćiemnośći.
