Biádá tobie / który łupiƺ / choćiáżeś ſám nie złupiony / y który zdrádzáƺ / choćiáżeś ſám nie był zdrádzony! Gdy łupić przeſtánieƺ / będźieƺ też złupiony ; gdy zdrádzáć przeſtánieƺ / będą ćię też zdrádzáć.
Pánie! zmiłuj śię nád námi / ćiebie ocżekujemy. Bądź rámieniem ſwojich ná káżdy poránek / á zbáwieniem náƺem cżáſu utrapienia.
Przed głoſem ogromnym rozpierzchną śię národy ; przed wywyżƺeniem twojim rozproƺą śię pogánie.
Y będą zebráne łupy wáƺe / jáko zbieráją chrząƺcże ; á jáko przypádá ƺáráńcżá / ták oni przypádną ná nie.
Pán będźie wywyżƺony / bo mieƺká ná wyſokośći ; nápełni Syon ſądem y ſpráwiedliwośćią.
Mądrość y umiejętność będą utwierdzeniem cżáſów twojich / śiłą y obfitem zbáwieniem twem / á bojáźń Páńſká ſkárbem twoim.
Oto mocárze ich wołáją ná dworze / poſłowie pokoju gorzko płácżą.
Spuſtoƺáły drogi / przeſtáno śćieƺką chodźić ; złámał przymierze / zniewáżył miáſtá / á cżłowieká zá nic ſobie nie ma.
Płákáłá y zwątláłá źiemiá ; záwſtydzony jeſt Libán y uwiądł ; Sáron śię ſtał jáko puſtyniá / y otłucżono Báſán y Kármel.
Teraz powſtánę / mówi Pán / teraz śię wywyżƺę / teraz śię podnioſę.
Pocżąwƺy ſłomę / urodźićie mierzwę ; duch wáƺ was pożre jáko ogień.
Y będą národy / jáko wypálone wápno ; będą jáko ćiernie wyćięte / ogniem ſpálone.
Słuchájćie / którzyśćie dáleko / com ucżynił / á bliſcy poznájćie moc moję.
Zlękli śię ná Syonie grzeƺnicy / ſtrách zdjął obłudników mówiących : Któż z nas oſtáć śię może przed ogniem pożerájącym? Któż z nas oſtáć śię może przed płomieniem wiecżnym?
Ten / który chodźi w ſpráwiedliwośći / á mówi / co jeſt práwego ; który śię zyſkiem nieſpráwiedliwym brzydźi ; który otrząſá ręce ſwe / áby dárów nie brał ; który zátulá uƺy ſwe / áby nie ſłuchał o rozlániu krwi / y zámrużá ocży ſwoje / áby nie pátrzał ná złe :
Ten ná wyſokośćiách mieƺkáć będźie / zámki ná ſkáłách będą ućiecżką jego ; chleb jego dány mu będźie / wody jego nie uſtáną.
Królá w pięknośći jego oglądáją ocży twoje / ujrzą y źiemię dáleką.
Serce twoje będźie rozmyśláło o ſtráchu / mówiąc : Gdźie teraz jeſt piſárz? gdźież teraz jeſt poborcá? gdźież jeſt oblicżájący wieże?
Ludu okrutnego nie oglądáƺ / ludu głębokiej mowy / któregoś nie ſłyƺał / y języká obcego / któregobyś nie rozumiał.
Wejrzyj ná Syon / miáſto urocżyſtych ſwiąt náƺych / ocży twoje niecháj pátrzą ná Jeruzálem / ná mieƺkánie ſpokojne / ná námiot / który nie będźie przenieśion ; kołki jego ná wieki śię nie poruƺą / á żáden powróz jego nie zerwie śię /
Przeto / że nam ná tym miejſcu Pán wielmożny jeſt rzekámi ſtrumieni ƺerokich / po których nie pójdą z wioſłámi / áni okręt wielki po nich przechodźić będźie.
Bo Pán jeſt ſędźiá náƺ / Pán zakonodáwcá náƺ ; Pán król náƺ ; on nas zbáwi.
Oſłábiáły powrozy twoje / nie będą mogły w klubie zátrzymáć máƺtu twego / áni rozćiągną żáglów. Tedy rozdźielone będą łupy korzyśći wielkiej / że y chromi rozchwycą łupy.
A nie rzecże żáden z obywáteli : Záchorowáłem ; lud / który mieƺká w nim / uwolniony będźie od niepráwośći.
