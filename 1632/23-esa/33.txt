Biádá tobie / który łupiƺ / choćiáżeś ſám nie złupiony / y który zdrádzáƺ / choćiáżeś ſám nie był zdrádzony! Gdy łupić przeſtánieƺ / będzieƺ też złupiony ; gdy zdrádzáć przeſtánieƺ / będą ćię też zdrádzáć.
Pánie! zmiłuj śię nád námi / ćiebie ocżekujemy. Bądź rámieniem ſwoich ná káżdy poránek / á zbáwieniem náƺem cżáſu utrápieniá.
Przed głoſem ogromnym rozpierzchną śię národy ; przed wywyżƺeniem twojim rozproƺą śię pogánie.
Y będą zebráne łupy wáƺe / jáko zbieráją chrząƺcże ; á jáko przypádá ƺáráńcżá / ták oni przypádną ná nie.
Pán będzie wywyżƺony / bo mieƺká ná wyſokośći ; nápełni Syon ſądem y ſpráwiedliwośćią.
Mądrość y umiejętność będą utwierdzeniem cżáſów twoich / śiłą y obfitem zbáwieniem twem / á bojáźń Páńſká ſkárbem twoim.
Oto mocárze ich wołáją ná dworze / poſłowie pokoju gorzko płácżą.
Spuſtoƺáły drogi / przeſtáno śćieƺką chodzić ; złámáł przymierze / zniewáżył miáſtá / á cżłowieká zá nic ſobie nie má.
Płákáłá y zwątláłá ziemiá ; záwſtydzony jeſt Libán y uwiądł ; Sáron śię ſtáł jáko puſtyniá / y otłucżono Báſán y Kármel.
Teráz powſtánę / mówi Pán / teráz śię wywyżƺę / teráz śię podnioſę.
Pocżąwƺy ſłomę / urodzićie mierzwę ; duch wáƺ wáſ pożre jáko ogień.
Y będą národy / jáko wypálone wápno ; będą jáko ćiernie wyćięte / ogniem ſpálone.
Słuchájćie / którzyśćie dáleko / com ucżynił / á bliſcy poznájćie moc moję.
Zlękli śię ná Syonie grzeƺnicy / ſtrách zdjął obłudników mówiących : Któż z náſ oſtáć śię może przed ogniem pożerájącym? Któż z náſ oſtáć śię może przed płomieniem wiecżnym?
Ten / który chodzi w ſpráwiedliwośći / á mówi / co jeſt práwego ; który śię zyſkiem nieſpráwiedliwym brzydzi ; który otrząſá ręce ſwe / áby dárów nie bráł ; który zátulá uƺy ſwe / áby nie ſłucháł o rozlániu krwi / y zámrużá ocży ſwoje / áby nie pátrzáł ná złe :
Ten ná wyſokośćiách mieƺkáć będzie / zámki ná ſkáłách będą ućiecżką jego ; chleb jego dány mu będzie / wody jego nie uſtáną.
Królá w pięknośći jego oglądáją ocży twoje / ujrzą y ziemię dáleką.
Serce twoje będzie rozmyśláło o ſtráchu / mówiąc : Gdzie teráz jeſt piſárz? gdzież teráz jeſt poborcá? gdzież jeſt oblicżájący wieże?
Ludu okrutnego nie oglądáƺ / ludu głębokiej mowy / któregoś nie ſłyƺáł / y języká obcego / któregobyś nie rozumiáł.
Wejrzyj ná Syon / miáſto urocżyſtych ſwiąt náƺych / ocży twoje niecháj pátrzą ná Jeruzálem / ná mieƺkánie ſpokojne / ná námiot / który nie będzie przenieśion ; kołki jego ná wieki śię nie poruƺą / á żáden powróz jego nie zerwie śię /
Przeto / że nám ná tem miejſcu Pán wielmożny jeſt rzekámi ſtrumieni ƺerokich / po których nie pójdą z wioſłámi / áni okręt wielki po nich przechodzić będzie.
Bo Pán jeſt ſędziá náƺ / Pán zákonodáwcá náƺ ; Pán król náƺ ; on náſ zbáwi.
Oſłábiáły powrozy twoje / nie będą mogły w klubie zátrzymáć máƺtu twego / áni rozćiągną żáglów. Tedy rozdzielone będą łupy korzyśći wielkiej / że y chromi rozchwycą łupy.
A nie rzecże żáden z obywáteli : Záchorowáłem ; lud / który mieƺká w nim / uwolniony będzie od niepráwośći.
