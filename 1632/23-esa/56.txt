Ták mówi Pán : Strzeżćie ſądu / á cżyńćie ſpráwiedliwość ; bo bliſko tego / że zbáwienie moje przyjdzie / á ſpráwiedliwość mojá objáwioná będzie.
Błogoſłáwiony cżłowiek / który to cżyni / y ſyn cżłowiecży / który śię trzymá tego / przeſtrzegájąc ſábátu / áby go nie ſplugáwił / á ſtrzegąc ręki ſwej / áby nie ucżyniłá nic złego.
Niech tedy nie mówi cudzoziemiec / który przyſtáje do Páná / mówiąc : Záiſte Pán mię odłącżył od ludu ſwego ; niech też nie mówi trzebieniec : Otom já drzewo ſuche.
Abowiem ták mówi Pán o trzebieńcách / którzyby przeſtrzegáli ſábátów moich / á obráli to / co mi śię podobá / y trzymáli przymierze moje :
Żeć im dám w domu ſwym y między murámi mojemi miejſce / y imię lepƺe niżeli ſynów y córek ; dám im imię wiecżne / które nie będzie wygłádzone.
A cudzoziemców / którzyby przyſtáli do Páná / áby mu ſłużyli / á miłowáli imię Páńſkie / będąc u niego zá ſługi / wƺyſtkich przeſtrzegájących ſábáty / áby go nie ſplugáwili / y záchowujących przymierze moje ;
Tych przywiodę ná górę świętobliwośći mojej / á uweſelę ich w domu modlitwy mojej ; cáłopáleniá ich y ofiáry ich przyjemne będą ná ołtárzu moim ; bo dom mój domem modlitwy názwány będzie u wƺyſtkich národów.
Ták mówi pánujący Pán / który zgromádzá rozpędzonych z Izráelá : Jeƺcże zgromádzę do niego / y do zgromádzonych jego.
Wƺyſtkie zwierzętá polne przyjdźćie ná pożárćie / y wƺyſtkie zwierzętá leśne.
Stróżowie jego ślepi / wƺyſcy zgołá nic nie umieją / wƺyſcy ſą pſámi niememi / nie mogą ƺcżekáć ; oſpáłymi ſą / leżą / kocháją śię w drzemániu.
A ſą pſámi obżártemi / nie mogą śię nigdy náſyćić ; ſámi śię páſąc nie umieją náucżáć. Wƺyſcy śię zá drogą ſwoją událi / káżdy zá łákomſtwem ſwojim z ſtrony ſwej / mówiąc :
Pójdźćie / nábiorę winá / á upijemy śię mocnym nápojem / á będzie nám jáko dziś ták y jutro / y jeƺcże dáleko obfićiej.
