Ocuć śię / ocuć śię / oblecż śię w moc twoję / Syonie! oblecż śię w ƺátę ochędóſtwá twego / o Jeruzálemie / miáſto święte! Abowiem nie nátrze ná ćię nieobrzezány y niecżyſty.
Otrząśnij śię z prochu / powſtáń / śiądź / Jeruzálemie! dobądź śię z oków ƺyi ſwojey / o pojmáná córko Syońſká!
Ták záiſte Pán mówi : Dármośćie śię záprzedáli / przetoż bez pieniędzy odkupieni będźiećie.
Bo ták mówi pánujący Pán : Do Egiptu wſtąpił lud mój przedtem / áby tám pielgrzymował ; ále Aſſyryjcżyk bez przycżyny go trápi.
A teraz cóż mam cżynić? mówi Pán / poniewáż lud mój dármo jeſt pojmány / á ći / którzy pánują nád nim / do wzdychánia go przywodzą / mówi Pán ; nádto uſtáwicżnie káżdego dniá imię moje bluźnione bywá.
Przetoż pozná lud mój imię moje / przetoż pozná / mówię / dniá onego / żem Ja jeſt ten / który mówię ; otom Ja przytomny.
O jáko piękne ſą ná górách nogi tego / co poćieƺne rzecży zwiáſtuje / y opowiádá pokój ; tego / co zwiáſtuje dobre / y opowiádá zbáwienie / á mówi do Syonu : Bóg twój króluje!
Wynoƺą głos ſtróżowie twoi / głos wynoƺą / á ſpołem wykrzykáć będą ; bo okiem w oko ujrzą / że záśię Pán Syon przywiedźie.
Wykrzykájćie á śpiewájćie ſpołem / puſtynie Jeruzálemſkie! bo poćieƺył Pán lud ſwój / odkupił Jeruzálem.
Wyſmuknął Pán rámię świętobliwośći ſwojey przed ocżymá wƺyſtkich národów / áby oglądáły wƺyſtkie końcżyny źiemi zbáwienie Bogá náƺego.
Odſtąpćie / odſtąpćie / wynijdźćie z Bábilonu / niecżyſtego śię nie dotykájćie / wynijdźćie z pośrzodku jego ; ocżyśććie śię wy / którzy nośićie nacżynie Páńſkie.
Bo nie z trzáſkiem wynijdźiećie / áni ućiekájąc pójdźiećie ; pójdźie záiſte Pán przed wámi / á zgromádźi was Bóg Izráelſki.
Oto śię ƺcżęśliwie powiedźie ſłudze memu. Wywyżƺony y podnieśiony y bárzo uwielbiony będźie.
Jáko wiele ich zdumieją śię nád nim / że przemierzłá jeſt nád innych ludźi oſobá jego / á kƺtáłt jego nád ſynów ludzkich :
Ták záśię pokropi wiele národów / y królowie przed nim zátulą uſtá ſwe / przeto / że cżego im nie powiádáno / to oglądáją / á to / o cżym nie ſłyƺeli / wyrozumieją.
