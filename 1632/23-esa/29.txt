Biádá Aryjelowi! Aryjelowi miáſtu / w którym mieƺkał Dawid. Przydájćie rok do roku / niecháj rzeżą bárány.
Jednák ućiſnę Aryjelá / y będźie ſmutek y żáłość / bo mi będźie jáko Aryjel.
Położę śię zájiſte obozem w około przećiwko tobie / y śćiſnę ćię wáłámi / y wyſtáwię przećiwko tobie báƺty.
Tedy będąc zniżone / z źiemi mówić będźieƺ / y z prochu ƺeptáć będźie mowá twojá ; będźie mówił głos twój / jáko wieƺcżká z źiemi / á z prochu mowá twojá ƺeptáć będźie.
Bo mnóſtwo nieprzyjáćiół twojich będźie jáko proƺku drobnego / á zgrájá okrutników jáko plew látájących ; á to śię nágle w okámgnieniu ſtánie.
Od Páná zaſtępów náwiedźione będźie gromem y trzęśieniem źiemi / y głoſem wielkim / wichrem y burzą / y płomieniem ogniá pożerájącego.
Ale jáko ſen widzeniá nocnego / ták będźie zgrájá wƺyſtkich narodów wálcżących przećiwko Aryjelowi / y wƺyſtkich bojujących przećiwko niemu y twierdzom jego / y tych / którzy go ućiſkáją.
Będźie / mówię / jáko gdy śię śni głodnemu / jákoby jádł ; ále gdy śię ocući / álić cżcży żywot jego ; y jáko gdy śię śni prágnącemu / jákoby pił / á gdy śię ocući / álić zemdlony zoſtáje / á duƺá jego prágnie ; ták będźie zgrájá wƺyſtkich narodów wálcżących przećiwko górze Syońſkiej.
Jákoż tedy odwłácżáćie / choćbyśćie śię zdumiewáć mieli ; rozkoƺujećie / choćbyśćie mieli ná pomoc wołáć. Opili śię / ále nie winem ; potácżáją śię / ále nie od mocnego nápoju.
Bo was nápełnił Pán duchem ſnu twárdego / y záwárł ocży wáƺe ; proroków y kśiążąt wáƺych nájopátrzniejƺych ocży záſłonił.
Przetoż wam wƺelkie widzenie podobne jeſt ſłowom kśiąg zápiecżętowánych / które dánoliby temu / co zná piſmo / á rzecżono : Cżytáj to proƺę / tedy odpowie : Nie mogę / bo ſą zápiecżętowáne.
A dánoliby kśięgi temu / co nie zná piſmá / á rzecżono : Cżytáj to proƺę / tedy odpowie : Nie znam piſmá.
Bo mówi Pán : Przeto / że ten lud przybliżá śię do mnie uſty ſwemi / á ſerce jego dálekie jeſt odemnie / á bojáźni / którą śię mnie boją / z przykázáń ludzkich náucżyli śię :
Dla tego Ja też ſobie dźiwnie pocżnę z tym ludem / dźiwnie y cudownie / y zginie mądrość mądrych jego / á rozum roztropnych jego ſkryje śię.
Biádá tym / którzy głęboko przed Pánem ukrywáją rádę! których káżda ſpráwá dźieje śię w ćiemnośći / y mówią : Któż widźi? Kto wie o nas?
Przewrotne myśli wáƺe ſą jáko gliná gárncárſká. Izali rzecże robotá o tym / co ją urobił : Nie urobił mię? y ulepienie izali rzecże o tym / co je ulepił : Nie rozumiał?
Izali po málucżkim y króćiuchnym cżáśie nie obróći śię Libán w pole? á pole zá lás pocżytáne nie będźie?
Y uſłyƺą dniá onego głuśi ſłowá kśiąg / á z mroku y z ćiemnośći ocży ślepych pátrzáć będą.
Ale ćiśi náder śię rozweſelą w Pánu / á ubodzy ludźie rozweſelą śię w Świętym Izráelſkim.
Gdy uſtánie okrutnik / á zniƺcżeje náśmiewcá / wykorzenieni będą wƺyſcy / którzy pilnowáli niepráwośći ;
Którzy winują cżłowieká dla ſłowá / á ná tego / który ich ſtrofuje / w bramie śidłá ſtáwiáją / y bez przycżyny do upádku przywodzą ſpráwiedliwego.
Przetoż ták mówi o domu Jákóbowym Pán / który odkupił Abráhámá : Już dálej nie będźie záwſtydzony Jákób / áni więcey twárz jego zblednie.
Abowiem gdy ujrzy ſynów ſwojich / dźieło rąk mojich / w pośrzodku śiebie / poświęcájących imię moje ; tedy będą poświęcáć Świętego Jákóbowego / á Bogá Izráelſkiego báć śię będą.
Y ſtáną śię rozumnymi błądzący duchem / á ƺemrácże náucżą śię umiejętnośći.
