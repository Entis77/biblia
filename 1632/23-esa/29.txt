Biádá Aryjelowi! Aryjelowi miáſtu / w którym mieƺkáł Dawid. Przydájćie rok do roku / niecháj rzeżą bárány.
Jednák ućiſnę Aryjelá / y będzie ſmutek y żáłość / bo mi będzie jáko Aryjel.
Położę śię záiſte obozem w około przećiwko tobie / y śćiſnę ćię wáłámi / y wyſtáwię przećiwko tobie báƺty.
Tedy będąc zniżone / z ziemi mówić będzieƺ / y z prochu ƺeptáć będzie mowá twojá ; będzie mówił głoſ twój / jáko wieƺcżká z ziemi / á z prochu mowá twojá ƺeptáć będzie.
Bo mnóſtwo nieprzyjáćiół twoich będzie jáko proƺku drobnego / á zgrájá okrutników jáko plew látájących ; á to śię nágle w okámgnieniu ſtánie.
Od Páná záſtępów náwiedzione będzie gromem y trzęśieniem ziemi / y głoſem wielkim / wichrem y burzą / y płomieniem ogniá pożerájącego.
Ale jáko ſen widzeniá nocnego / ták będzie zgrájá wƺyſtkich národów wálcżących przećiwko Aryjelowi / y wƺyſtkich bojujących przećiwko niemu y twierdzom jego / y tych / którzy go ućiſkáją.
Będzie / mówię / jáko gdy śię śni głodnemu / jákoby jádł ; ále gdy śię ocući / álić cżcży żywot jego ; y jáko gdy śię śni prágnącemu / jákoby pił / á gdy śię ocući / álić zemdlony zoſtáje / á duƺá jego prágnie ; ták będzie zgrájá wƺyſtkich národów wálcżących przećiwko górze Syońſkiej.
Jákoż tedy odwłácżáćie / choćbyśćie śię zdumiewáć mieli ; rozkoƺujećie / choćbyśćie mieli ná pomoc wołáć. Opili śię / ále nie winem ; potácżáją śię / ále nie od mocnego nápoju.
Bo wáſ nápełnił Pán duchem ſnu twárdego / y záwárł ocży wáƺe ; proroków y kśiążąt wáƺych nájopátrzniejƺych ocży záſłonił.
Przetoż wám wƺelkie widzenie podobne jeſt ſłowom kśiąg zápiecżętowánych / które dánoliby temu / co zná piſmo / á rzecżono : Cżytáj to proƺę / tedy odpowie : Nie mogę / bo ſą zápiecżętowáne.
A dánoliby kśięgi temu / co nie zná piſmá / á rzecżono : Cżytáj to proƺę / tedy odpowie : Nie znám piſmá.
Bo mówi Pán : Przeto / że ten lud przybliżá śię do mnie uſty ſwemi / á ſerce jego dálekie jeſt odemnie / á bojáźni / którą śię mnie boją / z przykázáń ludzkich náucżyli śię :
Dlátego Já też ſobie dziwnie pocżnę z tym ludem / dziwnie y cudownie / y zginie mądrość mądrych jego / á rozum roztropnych jego ſkryje śię.
Biádá tym / którzy głęboko przed Pánem ukrywáją rádę! których káżdá ſpráwá dzieje śię w ćiemnośći / y mówią : Któż widzi? Kto wie o náſ?
Przewrotne myśli wáƺe ſą jáko gliná gárncárſká. Izáli rzecże robotá o tym / co ją urobił : Nie urobił mię? y ulepienie izáli rzecże o tym / co je ulepił : Nie rozumiáł?
Izáli po málucżkim y króćiuchnym cżáśie nie obróći śię Libán w pole? á pole zá láſ pocżytáne nie będzie?
Y uſłyƺą dniá onego głuśi ſłowá kśiąg / á z mroku y z ćiemnośći ocży ślepych pátrzáć będą.
Ale ćiśi náder śię rozweſelą w Pánu / á ubodzy ludzie rozweſelą śię w Świętym Izráelſkim.
Gdy uſtánie okrutnik / á zniƺcżeje náśmiewcá / wykorzenieni będą wƺyſcy / którzy pilnowáli niepráwośći ;
Którzy winują cżłowieká dlá ſłowá / á ná tego / który ich ſtrofuje / w brámie śidłá ſtáwiáją / y bez przycżyny do upádku przywodzą ſpráwiedliwego.
Przetoż ták mówi o domu Jákóbowym Pán / który odkupił Abráhámá : Już dálej nie będzie záwſtydzony Jákób / áni więcej twárz jego zblednie.
Abowiem gdy ujrzy ſynów ſwoich / dzieło rąk moich / w pośrodku śiebie / poświęcájących imię moje ; tedy będą poświęcáć Świętego Jákóbowego / á Bogá Izráelſkiego báć śię będą.
Y ſtáną śię rozumnymi błądzący duchem / á ƺemrácże náucżą śię umiejętnośći.
