Widzenie Izájáƺá / ſyná Amoſowego / które widziáł nád Judą y nád Jeruzálemem / zá dni Ozeáƺá / Joátámá / Acházá / y Ezechijáƺá / królów Judzkich.
Słuchájćie niebioſá / á ty ziemio przyjmij w uƺy ſwe! Abowiem Pán mówi : Synówem wychowáł y wywyżƺył ; ále oni odſtąpili odemnie.
Zná wół goſpodárzá ſwego / y ośieł żłób páná ſwego ; ále Izráel mię nie zná / lud mój nie zrozumiewá.
Biádá národowi grzeƺnemu / ludowi obćiążonemu niepráwośćią / náśieniu złośliwych ; ſynom ſkáżonym! Opuśćili Páná / do gniewu pobudzili świętego Izráelſkiego / odwróćili śię názád.
Przecżże tem więcej przycżyniáćie przeſtępſtwá / im więcej wáſ biją? Wƺyſtká głowá chorá / y wƺyſtko ſerce mdłe.
Od ſtopy nogi áż do wierzchu głowy niemáƺ ná nim nic cáłego ; ráná y śiność / y rány zágniłe nie ſą wyćiśnione / áni záwiązáne / áni olejkiem odmiękcżone.
Ziemiá wáƺá ſpuſtoƺoná / miáſtá wáƺe popálone ogniem. Ziemię wáƺę cudzoziemcy przed wámi pożeráją y puſtoƺą / jáko zwykli cudzoziemcy.
Y zoſtáłá córká Syońſká jáko chłodnik ná winnicy / jáko budká w ogrodzie ogórcżánym / y jáko miáſto zburzone.
By nám był Pán záſtępów nie zoſtáwił trochy oſtátków / bylibyśmy jáko Sodomá / ſtálibyśmy śię byli Gomorze podobnymi.
Słuchájćie ſłowá Páńſkiego / kśiążętá Sodemſcy! przyjmujćie w uƺy zákon Bogá náƺego / ludzie Gomorſcy!
Cóż mi po mnóſtwie ofiár wáƺych? mówi Pán. Jużem ſyty cáłopáleniá báránów / y łoju tłuſtego bydłá ; á krwi ćielców / y báránków / y kozłów nie prágnę.
Gdy przychodzićie / ábyśćie śię okázywáli przed twárzą moją / któż tego żądáł z rąk wáƺych / ábyśćie deptáli śieni moje?
Nie ofiárujćież więcej ofiáry dáremnej. Kádzenie jeſt mi obrzydłośćią ; nowiu mieśiącá y ſábátu / gdy zwoływáćie zgromádzeniá / nie mogę śćierpieć ( bo niepráwośćią jeſt ) áni dniá zápowiedziánego.
Nowych mieśięcy wáƺych / y urocżyſtych świąt wáƺych nienáwidzi duƺá mojá ; ſtáły mi śię ćiężárem ; uprácowáłem śię noƺąc je.
Przetoż gdy wyćiągniećie ręce wáƺe / ſkryję ocży moje przed wámi ; á gdy rozmnożyćie modlitwę / nie wyſłuchám ; bo ręce wáƺe krwi ſą pełne.
Omyjćie śię / cżyſtymi bądźćie / odejmijćie złość ucżynków wáƺych od ocżów moich ; przeſtáńćie źle cżynić.
Ucżćie śię dobrze cżynić ; ƺukájćie ſądu / podźwignijćie ućiśnionego / ſąd cżyńćie śieroćie / ujmujćie śię o krzywdę wdowy.
Przyjdźćież teráz / á rozpierájmy śię z ſobą / mówi Pán : Choćby były grzechy wáƺe jáko ƺárłát / jáko śnieg zbieleją ; choćby były cżerwone jáko kármázyn / jáko wełná biáłe będą.
Będziećieli powolni / á poſłucháćie mię / dóbr ziemi pożywáć będziećie.
Lecż jeźli nie będziećie poſłuƺnymi / ále odpornymi / od miecżá pożárći będziećie ; bo uſtá Páńſkie mówiły.
Jákoć śię ſtáło nierządnicą to miáſto wierne / pełne ſądu? Spráwiedliwość mieƺkáłá w nim ; lecż teráz mężobójcy.
Srebro twoje obróćiło śię w żużeł ; wino twoje pomięƺáło śię z wodą.
Kśiążętá twoi ſą uporni / y towárzyƺe złodziei ; káżdy z nich miłuje dáry / á jádą zá nágrodą ; śieroćie nie cżynią ſpráwiedliwośći / á ſpráwá wdowy nie przychodzi przed nich.
Przetoż mówi Pán / Pán záſtępów / możny Izráelſki : Oto ućieƺę śię nád nieprzyjáćiółmi moimi / á pomƺcżę śię nád przećiwnikámi ſwymi.
Y obrócę rękę moję ná ćię / á wypálę áż do cżyſtá zużelicę twoję / y odpędzę wƺyſtkę cenę twoję.
A przywrócę ſędziów twoich / jáko przedtem byli / y rádców twoich / jáko ná pocżątku. Potem ćię názywáć będą miáſtem ſpráwiedliwośći / miáſtem wiernem.
Syon w ſądzie okupione będzie / á ći / co śię do niego náwrócą / w ſpráwiedliwośći.
Ale przewrotnicy y grzeƺnicy weſpół ſtárći będą / á ći / co opuśćili Páná / zniƺcżeją.
Abowiem záwſtydzeni będziećie dlá gájów / którycheśćie pożądáli ; y poháńbieni dlá ogrodów / któreśćie ſobie obráli.
Gdy śię ſtániećie jáko dąb / z którego liśćie opádły / á jáko ogród / w którym wody niemáƺ.
Y będzie mocárz jáko zgrzebiá / á ten / który go ucżynił / jáko iſkrá ; y zápálą śię oboje poſpołu / á nie będzie / ktoby zágáśił.
