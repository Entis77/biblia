Oto Pán obnáży źiemię / y ſpuſtoƺy ją / y przemieni oblicże jey / á rozproƺy obywáteli jey.
Y będźie jáko lud poſpolity ták y kśiążę ; jáko ſługá / ták pán jego ; jáko dźiewká / ták páni jey ; jáko kupujący / ták ſprzedáwájący ; jáko pożycżájący / ták y ten / co u drugiego pożycżá ; jáko lichwiárz / ták ten / co lichwę dáje.
Wielce obnáżoná będźie źiemiá / y bárzo złupioná ; ábowiem Pán mówił to ſłowo.
Płákáć będźie y upádnie źiemiá / zwątleje y obáli śię okrąg źiemſki ; zemdleją wyſokie narody źiemſkie /
Przeto / że tá źiemiá ſplugáwioná jeſt pod obywátelámi ſwoimi ; ábowiem przeſtąpili práwá / odmienili uſtáwy / wzruƺyli przymierze wiecżne.
Dla tego przeklęſtwo pożre źiemię / á zniƺcżeją obywátele jey ; dla tego popáleni będą obywátele źiemi / á máło ludźi zoſtánie.
Smućić śię będźie moƺcż / uwiędnie winná máćicá / wzdycháć będą wƺyſcy weſołego ſercá.
Uſtánie weſele bębnów / uſtánie wykrzykánie weſelących śię / ućichnie weſele cytry.
Nie będą pić winá z śpiewániem ; gorzki będźie nápój mocny pijącym go.
Stárte będźie miáſto próżnośći ; káżdy dom záwárty będźie / áby do niego nie wchodzono.
Nárzekánie będźie ná ulicách dla winá ; záćmione będźie wƺelkie weſele / á przenieśie śię rádość źiemi.
Spuſtoƺenie w mieśćie zoſtánie / á bramy zburzone będą.
Abowiem ták będźie w pośród źiemi / w pośrzodku narodów / jáko gdy otrzęſą oliwy / y jáko bywa z gronámi / gdy śię dokoná zbieránie winá.
Ci podnioſą głos ſwój / wykrzykáć będą / w zácnośći Páńſkiej wykrzykáć będą / y przy morzu.
Przetoż w dolinách wyſłáwiájćie Páná / ná wyſpách morſkich imię Páná / Bogá Izráelſkiego.
Od końcżyn źiemi ſłyƺymy pioſnkę o ſłáwie ſpráwiedliwego. Alem ja rzekł : Wychudłem / wychudłem / biádá mnie! Przewrotni przewrotność broją / przewrotność / mówię / bez wƺelkiego wſtydu broją.
Strách / y dół / y śidło przydźie ná ćię / który mieƺkáƺ ná źiemi.
Y ſtánie śię / że kto ućiecże przed wieśćią ſtráchu / wpádnie w dół / á kto wyliźie z dołu / pojmány będźie śidłem ; bo upuſty z wyſokośći otworzone będą / á zátrząſną śię grunty źiemi.
Rozſtępując rozſtąpi śię źiemiá ; rozśiádájąc rozśiądźie śię źiemiá ; poruƺájąc poruƺy śię źiemiá.
Chwiejąc chwiáć śię będźie źiemiá jáko pijány á przenieśioná będźie jáko budká ; bo ją obćiąży niepráwość jey / y upádnie / á więcey nie powſtánie.
A dniá onego náwiedźi Pán wojſko wyſokie ná wyſokośći / tákże y królów źiemſkich ná źiemi.
Y będą zgromádzeni / jáko zgromádzeni bywáją więźniowie do ćiemnicy / á będą zámknieni w táráśie : po wielu / mówię / dniách / náwiedzeni będą.
Y záſromi śię mieśiąc / á záwſtydźi śię ſłońce / gdy królowáć będźie Pán zaſtępów / ná górze Syońſkiej / y w Jeruzálemie / y przed ſtárcámi ſwymi w wielkiej ſłáwie.
