Oto Pán obnáży ziemię / y ſpuſtoƺy ją / y przemieni oblicże jej / á rozproƺy obywáteli jej.
Y będzie jáko lud poſpolity ták y kśiążę ; jáko ſługá / ták pán jego ; jáko dziewká / ták páni jej ; jáko kupujący / ták ſprzedáwájący ; jáko pożycżájący / ták y ten / co u drugiego pożycżá ; jáko lichwiárz / ták ten / co lichwę dáje.
Wielce obnáżoná będzie ziemiá / y bárdzo złupioná ; ábowiem Pán mówił to ſłowo.
Płákáć będzie y upádnie ziemiá / zwątleje y obáli śię okrąg ziemſki ; zemdleją wyſokie národy ziemſkie /
Przeto / że tá ziemiá ſplugáwioná jeſt pod obywátelámi ſwoimi ; ábowiem przeſtąpili práwá / odmienili uſtáwy / wzruƺyli przymierze wiecżne.
Dlá tego przeklęſtwo pożre ziemię / á zniƺcżeją obywátele jej ; dlátego popáleni będą obywátele ziemi / á máło ludzi zoſtánie.
Smućić śię będzie moƺcż / uwiędnie winná máćicá / wzdycháć będą wƺyſcy weſołego ſercá.
Uſtánie weſele bębnów / uſtánie wykrzykánie weſelących śię / ućichnie weſele cytry.
Nie będą pić winá z śpiewániem ; gorzki będzie nápój mocny pijącym go.
Stárte będzie miáſto próżnośći ; káżdy dom záwárty będzie / áby do niego nie wchodzono.
Nárzekánie będzie ná ulicách dlá winá ; záćmione będzie wƺelkie weſele / á przenieśie śię rádość ziemi.
Spuſtoƺenie w mieśćie zoſtánie / á brámy zburzone będą.
Abowiem ták będzie w pośród ziemi / w pośrodku národów / jáko gdy otrzęſą oliwy / y jáko bywá z gronámi / gdy śię dokoná zbieránie winá.
Ci podnioſą głoſ ſwój / wykrzykáć będą / w zácnośći Páńſkiej wykrzykáć będą / y przy morzu.
Przetoż w dolinách wyſłáwiájćie Páná / ná wyſpách morſkich imię Páná / Bogá Izráelſkiego.
Od końcżyn ziemi ſłyƺymy pioſnkę o ſłáwie ſpráwiedliwego. Alem já rzekł : Wychudłem / wychudłem / biádá mnie! Przewrotni przewrotność broją / przewrotność / mówię / bez wƺelkiego wſtydu broją.
Strách / y dół / y śidło przyjdzie ná ćię / który mieƺkáƺ ná ziemi.
Y ſtánie śię / że kto ućiecże przed wieśćią ſtráchu / wpádnie w dół / á kto wylizie z dołu / pojmány będzie śidłem ; bo upuſty z wyſokośći otworzone będą á zátrząſną śię grunty ziemi.
Rozſtępując rozſtąpi śię ziemiá ; rozśiádájąc rozśiądzie śię ziemiá ; poruƺájąc poruƺy śię ziemiá.
Chwiejąc chwiáć śię będzie ziemiá jáko pijány á przenieśioná będzie jáko budká ; bo ją obćiąży niepráwość jej / y upádnie / á więcej nie powſtánie.
A dniá onego náwiedzi Pán wojſko wyſokie ná wyſokośći / tákże y królów ziemſkich ná ziemi.
Y będą zgromádzeni / jáko zgromádzeni bywáją więźniowie do ćiemnicy / á będą zámknieni w táráśie ; po wielu / mówię / dniách / náwiedzeni będą.
Y záſromi śię mieśiąc / á záwſtydzi śię ſłońce / gdy królowáć będzie Pán záſtępów / ná górze Syońſkiej / y w Jeruzálemie / y przed ſtárcámi ſwymi w wielkiej ſłáwie.
