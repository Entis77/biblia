Dla Syonu milcżeć nie będę / á dla Jeruzálemu nie uſpokoję śię / dokąd ſpráwiedliwość jego nie wynijdźie jáko jáſność / á zbáwienie jego jáko pochodniá gorzeć nie będźie.
Y oglądáją národy ſpráwiedliwość twoję / y wƺyſcy królowie ſłáwę twoję y názwą ćię imieniem nowem / które uſtá Páńſkie miánowáć będą.
Y będźieƺ koroną ozdobną w ręce Páńſkiej / y koroną króleſtwá w ręce Bogá twego.
Nie będą ćię więcey zwáć opuƺcżoną / y źiemiá twojá nie będźie więcey zwáná ſpuſtoƺoną ; ále ty názywáná będźieƺ rozkoƺą moją / á źiemiá twojá mężátką ; bo Pán będźie miał rozkoƺ w tobie / á źiemiá twojá będźie zámężná.
Abowiem jáko młodźieniec pánnę pojmuje / ták ćię ſobie pojmą ſynowie twoi ; á jáko śię oblubieniec weſeli z oblubienicy / ták śię weſelić będźie z ćiebie Bóg twój.
Ná murách twoich / o Jeruzálem! poſtáwię ſtróżów / którzy przez cáły dźień y cáłą noc nigdy nie umilkną ; którzy wſpomináćie Páná / nie milcżćie ;
A nie dáwájćie mu odpocżnieniá / dokąd nie utwierdźi / y dokąd nie ſpoſobi / áby Jeruzálem było ſłáwne ná źiemi.
Przyśiągł Pán przez práwicę ſwoję y przez rámię mocy ſwojey / mówiąc : Nie podám więcey pƺenicy twojey ná pokárm nieprzyjáćiołom twoim / y nie będą pić cudzoźiemcy winá twego / około któregoś prácowáł.
Ale ći / którzy je zgromádzą / pożywáć go / y chwálić Páná będą ; á którzy je zbieráją / będą je pić w śieniách świątnicy mojey.
Przechodźćie / przechodźćie przez brámy! gotujćie drogę ludowi ; wyrównájćie / wyrównájćie gośćińce ; wybierzćie kámienie / podnieśćie chorągiew do národów.
Oto Pán rozkáże obwołáć áż do końcżyn źiemi ; powiedzćie córce Syońſkiej : Oto zbáwićiel twój idźie / oto zápłátá jego z nim / á dźieło jego przed nim.
Y názwią ſynów twoich ludem świętym / odkupionymi Páńſkimi / á ćiebie názwią miáſtem zácnem y nie opuƺcżonem.
