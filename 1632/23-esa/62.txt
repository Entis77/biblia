Dlá Syonu milcżeć nie będę / á dlá Jeruzálemu nie uſpokoję śię / dokąd ſpráwiedliwość jego nie wynijdzie jáko jáſność / á zbáwienie jego jáko pochodniá gorzeć nie będzie.
Y oglądáją národy ſpráwiedliwość twoję / y wƺyſcy królowie ſłáwę twoję y názwą ćię imieniem nowem / które uſtá Páńſkie miánowáć będą.
Y będzieƺ koroną ozdobną w ręce Páńſkiej / y koroną króleſtwá w ręce Bogá twego.
Nie będą ćię więcej zwáć opuƺcżoną / y ziemiá twojá nie będzie więcej zwáná ſpuſtoƺoną ; ále ty názywáná będzieƺ rozkoƺą moją / á ziemiá twojá mężátką ; bo Pán będzie miáł rozkoƺ w tobie / á ziemiá twojá będzie zámężná.
Abowiem jáko młodzieniec pánnę pojmuje / ták ćię ſobie pojmą ſynowie twoi ; á jáko śię oblubieniec weſeli z oblubienicy / ták śię weſelić będzie z ćiebie Bóg twój.
Ná murách twoich / o Jeruzálem! poſtáwię ſtróżów / którzy przez cáły dzień y cáłą noc nigdy nie umilkną ; którzy wſpomináćie Páná / nie milcżćie ;
A nie dáwájćie mu odpocżnieniá / dokąd nie utwierdzi / y dokąd nie ſpoſobi / áby Jeruzálem było ſłáwne ná ziemi.
Przyśiągł Pán przez práwicę ſwoję y przez rámię mocy ſwojej / mówiąc : Nie podám więcej pƺenicy twojej ná pokárm nieprzyjáćiołom twoim / y nie będą pić cudzoziemcy winá twego / około któregoś prácowáł.
Ale ći / którzy je zgromádzą / pożywáć go / y chwálić Páná będą ; á którzy je zbieráją / będą je pić w śieniách świątnicy mojej.
Przechodźćie / przechodźćie przez brámy! gotujćie drogę ludowi ; wyrównájćie / wyrównájćie gośćińce ; wybierzćie kámienie / podnieśćie chorągiew do národów.
Oto Pán rozkáże obwołáć áż do końcżyn ziemi ; powiedzćie córce Syońſkiej : Oto zbáwićiel twój idzie / oto zápłátá jego z nim / á dzieło jego przed nim.
Y názwią ſynów twoich ludem świętym / odkupionymi Páńſkimi / á ćiebie názwią miáſtem zácnem y nie opuƺcżonem.
