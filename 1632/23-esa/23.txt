Brzemię Tyru. Kwilćie okręty morſkie! ábowiem zburzony jeſt / ták / iż niemáƺ áni domu / áni ktoby przychodził z ziemi Cytym.
To mi o nich objáwiono. Umilknijćież / obywátele wyſpy! którą kupcy Sydońſcy pływájąc przez morze nápełniáli.
A którego dochody ná wielkich wodách / náśienie Sychor / żniwo jego dochód z rzeki / á w którym był ſkłád národów.
Záwſtydź śię / Sydonie! bo rzekło morze / moc morſká / mówiąc : Nie prácuję w porodzeniu / y nie rodzę / y nie wychowuję młodzieńców / áni odchowuję pánien.
Jáko nád powieśćią o Egipćie / ták będą żáłośni nád powieśćią o Tyrze.
Przepráwćie śię przez morze / kwilćie obywátele wyſpy!
Toż to jeſt miáſto wáƺe weſelące śię? Jego ſtárożytność jeſtći ode dni dáwnych ; ále go záwiodą nogi jego ná dáleką wędrówkę.
Któż to poſtánowił o Tyrze / który koronuje inƺe? którego kupcy ſą kśiążętámi / á krámárze jego ſłáwnymi ná ziemi?
Pán záſtępów poſtánowił to / áby ohydził pychę wƺelkiej ſłáwy / á żeby do zniewági przywiódł wƺyſtkich zácnych ná ziemi.
Náwróć śię do ziemi ſwej / jáko rzeká / o córko morſká ; niemáƺći tám więcej páſá.
Rękę ſwoję wyćiągnął ná morze / zátrwożył króleſtwá. Pán rozkázáł o Chánáánie / áby zburzone były twierdze jego ;
Y rzekł : Już śię nie będzieƺ więcej weſeliłá / ty zgwáłconá pánno / córko Syońſká! Powſtáń / przepráw śię do Cytym ; lecż y tám nie będzieƺ miáłá odpocżynku.
Oto ziemiá Cháldejſká / ten lud nie był ludem. Aſſyryjcżyk záłożył ją dlá obywáteli puſtyń / którzy wyſłáwili zámki jej / pobudowáli páłáce jej ; ále on ją w gruz obróćił.
Kwilćie okręty morſkie! ábowiem zburzoná jeſt twierdzá wáƺá.
Y ſtánie śię dniá onego / że w zápámiętániu będzie Tyr przez śiedmdzieśiąt lát / przez wiek królá jednego. A po śiedmdzieśięćiu látách Tyr znowu będzie miáł pioſnkę / jáko pioſnkę nierządnicy.
Weźmij lutnię / obchodź miáſto / o nierządnico w zápomnienie podáná! gráj dobrze / długo śpiewáj / ábyś ná pámięć przyƺłá.
Y ſtánie śię po wyjśćiu śiedmdzieśięćiu lát / że Tyr Pán náwiedzi ; ále śię on záś wróći do nierządnicżego zyſku ſwego / y będzie nierząd płodził ze wƺyſtkiemi króleſtwámi ziemi / ná oblicżu ziemi.
Wƺákże kupiectwo jego / y zyſk jego będzie poświęcony Pánu. Do ſkárbu odłożony / y ſchowány nie będzie ; ále tym / którzy mieƺkáją przed Pánem / pożytecżne będzie kupiectwo jego / áby jedli do ſytośći / á mieli odzienie dobre.
