Brzemię Tyru. Kwilćie okręty morſkie! ábowiem zburzony jeſt / ták / iż niemáƺ áni domu / áni ktoby przychodźił z źiemi Cytym.
To mi o nich objáwiono. Umilknijćież / obywátele wyſpy! którą kupcy Sydońſcy pływájąc przez morze nápełniáli.
A którego dochody ná wielkich wodách / náśienie Sychor / żniwo jego dochód z rzeki / á w którym był ſkłád národów.
Záwſtydź śię / Sydonie! bo rzekło morze / moc morſká / mówiąc : Nie prácuję w porodzeniu / y nie rodzę / y nie wychowuję młodźieńców / áni odchowuję pánien.
Jáko nád powieśćią o Egipćie / ták będą żáłośni nád powieśćią o Tyrze.
Przepráwćie śię przez morze / kwilćie obywátele wyſpy!
Toż to jeſt miáſto wáƺe weſelące śię? Jego ſtárożytność jeſtći ode dni dáwnych ; ále go záwiodą nogi jego ná dáleką wędrówkę.
Któż to poſtánowił o Tyrze / który koronuje inƺe? którego kupcy ſą kśiążętámi / á krámárze jego ſłáwnymi ná źiemi?
Pán záſtępów poſtánowił to / áby ohydźił pychę wƺelkiej ſłáwy / á żeby do zniewági przywiódł wƺyſtkich zácnych ná źiemi.
Náwróć śię do źiemi ſwej / jáko rzeká / o córko morſká ; niemáƺći tám więcey páſá.
Rękę ſwoję wyćiągnął ná morze / zátrwożył króleſtwá. Pán rozkázáł o Chánáánie / áby zburzone były twierdze jego ;
Y rzekł : Już śię nie będźieƺ więcey weſeliłá / ty zgwáłconá pánno / córko Syońſká! Powſtáń / przepráw śię do Cytym ; lecż y tám nie będźieƺ miáłá odpocżynku.
Oto źiemiá Cháldejſká / ten lud nie był ludem. Aſſyryjcżyk záłożył ją dla obywáteli puſtyń / którzy wyſłáwili zámki jey / pobudowáli páłáce jey ; ále on ją w gruz obróćił.
Kwilćie okręty morſkie! ábowiem zburzoná jeſt twierdzá wáƺá.
Y ſtánie śię dniá onego / że w zápámiętániu będźie Tyr przez śiedmdźieśiąt lát / przez wiek królá jednego. A po śiedmdźieśięćiu látách Tyr znowu będźie miał pioſnkę / jáko pioſnkę nierządnicy.
Weźmij lutnię / obchodź miáſto / o nierządnico w zápomnienie podáná! gráj dobrze / długo śpiewáj / ábyś ná pámięć przyƺłá.
Y ſtánie śię po wyjśćiu śiedmdźieśięćiu lát / że Tyr Pán náwiedźi ; ále śię on záś wróći do nierządnicżego zyſku ſwego / y będźie nierząd płodźił ze wƺyſtkiemi króleſtwámi źiemi / ná oblicżu źiemi.
Wƺákże kupiectwo jego / y zyſk jego będźie poświęcony Pánu. Do ſkárbu odłożony / y ſchowány nie będźie ; ále tym / którzy mieƺkáją przed Pánem / pożytecżne będźie kupiectwo jego / áby jedli do ſytośći / á mieli odźienie dobre.
