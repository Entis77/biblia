Biádá tym / którzy zſtępują do Egiptu o pomoc / á ná koniách ſpolegáją / y ufáją w wozách / że ich wiele / y w jezdnych / iż ſą mocni bárdzo / á nie oglądáją śię ná Świętego Izráelſkiego / á Páná nie ƺukáją!
Aleć on też jeſt mądry / przetoż przywiedzie złe / á ſłów ſwoich nie odmieni ; lecż powſtánie przećiw domowi złośników y przećiwko rátunkowi tych / którzy broją niepráwość.
Abowiem Egipcżánie ſą ludzie á nie Bóg / á konie ich ćiáło / á nie duch. Przetoż ſkoro Pán wyćiągnie rękę ſwą / pádnie y pomocnik / pádnie y ten / któremu dáwáją pomoc ; á ták wƺyſcy ſpołem zginą.
Bo ták rzekł Pán do mnie : Jáko gdy rycży lew / y ƺcżenię lwie nád łupem ſwym / á choć zwoływáją przećiwko niemu gromádę páſterzy / przećie śię on wrzáſku ich nie lęká / áni śię korzy przed hukiem ich ; ták Pán záſtępów zſtąpi / áby wálcżył o górę Syońſką / y o págórek jej.
Jáko ptáki látáją około gniázdá ſwego / ták obroni Pán záſtępów Jeruzálem / y owƺem / broni y wybáwiá / á przechodząc z pomſtą záchowá.
Náwróććie śię do tego / od którego głęboko zábrnęli ſynowie Izráelſcy /
Abowiem dniá onego odrzući káżdy báłwány ſwe ſrebrne / y báłwány ſwe złote / które wám nácżyniły ręce wáƺe ná grzech.
Y upádnie Aſſyryjcżyk od miecżá nie męſkiego / á miecż nie cżłowiecży pożre go : y ućiecże przed miecżem / á młodzieńcy jego hołdownikámi będą.
A ták opokę ſwoję od ſtráchu minie / á kśiążętá jego ulękną śię przed chorągwią / mówi Pán / którego ogień jeſt ná Syonie / á piec w Jeruzálemie.
