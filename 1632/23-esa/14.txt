Abowiem zlituje śię Pán nád Jákóbem / á obierze záśię Izráelá / y dá im odpocżnąć w ziemi ich ; á przyłącży śię do nich cudzoziemiec / y przyſtáną do domu Jákóbowego.
Bo wezmą z ſobą národy / y przywiodą je do miejſcá ſwego ; y weźmie je ſobie dom Izráelſki w ziemi Páńſkiej w dziedzictwo zá ſługi y zá ſłużebnice ; y imáć będą tych / którzy ich imáli / á pánowáć będą nád tymi / którzy ich ćiemiężyli.
A dniá onego / któregoć Pán dá odpocżnienie od prácy twojej y od ſtráchu twego / y od niewoli ćiężkiej / w którąś był podbity /
Weźmieƺ tę przypowieść przećiw królowi Bábilońſkiemu / y rzecżeƺ : O jáko uſtáł poborcá / uſtáł podátek złotá!
Pán złámáł kij niezbożnych / y rózgę pánujących ;
Tego / który ludzi bijáł w zápálcżywośći bićiem uſtáwicżnem / pánowáł w gniewie nád národámi bez litośći dręcżonemi ;
Teráz odpocżywá y jeſt w pokoju wƺyſtká ziemiá / wƺyſcy głośno śpiewáją ;
Y jodły śię weſelą nád tobą / y cedry Libáńſkie / mówiąc : Od tego cżáſu / jákoś ty poległ / nie powſtáł / ktoby náſ podćináł.
Y piekło ze ſpodku wzruƺyło śię dlá ćiebie / áby tobie przychodzącemu záƺło ; wzbudziło dlá ćiebie umárłych / wƺyſtkich kśiążąt ziemi ; rozkázáło powſtáć z ſtolic ſwoich y wƺyſtkim królom národów.
Ci wƺyſcy odpowiádájąc mówią do ćiebie : Y tyś zemdlony jáko y my / á ſtáłeś śię nám podobnym.
Strąconá jeſt do piekłá / pychá twojá y dźwięk muzyki twojej ; podeſłánoć mole / á robáki ćię przykrywáją.
Jákoż to / żeś ſpádł z niebá / o jutrzenko! którá wſchodziƺ ráno? powálonyś áż ná ziemię / któryś wątlił národy!
Wƺákieś ty máwiáł w ſercu ſwem : Wſtąpię ná niebo / nád gwiázdy Boże wywyżƺę ſtolicę moję / á uśiądę ná górze zgromádzeniá / ná ſtronách północnych :
Wſtąpię ná wyſokość obłoków / będę równy Nájwyżƺemu.
Wƺákże ſtrącon jeſteś áż do piekłá / w głębokość dołu.
Którzy ćię ujrzą / zá tobą śię oglądáć / y przypátrywáć ći śię będą mówiąc : Onże to mąż / który trwożył ziemię? który trząſáł króleſtwámi?
Który obróćił okrąg świátá w puſtynię / á miáſtá jego poburzył / á więźniom ſwoim nie otwárzáł ćiemnicy?
Wƺyſcy królowie národów / cokolwiek ich było / pochowáni ſą w ſłáwie / káżdy w domu ſwoim.
Aleś ty odrzucony od grobu ſwego / jáko látorośl obrzydłá / jáko ƺátá zábitych / których poprzebijáno miecżem / którzy zſtępują do grobu kámieniſtego / jáko śćierw podeptány.
Nie będziƺ miáł ucżeſnictwá z nimi w pogrzebie ; boś ziemię twoję pokáził / y lud ſwój pomordowáł ; ábowiem náśienie złośników nie będzie wſpomináne ná wieki.
Gotujćie ſynów jego ná zámordowánie dlá niepráwośći ojców ich / áby nie powſtáli / y nie odziedzicżyli ziemi / nie nápełnili miáſtámi okręgu ziemſkiego.
Bo powſtánę przećiwko nim / mówi Pán záſtępów / á wykorzenię imię Bábilonu / y oſtátki jego / ták ſyná jáko y wnuká / mówi Pán ;
Y ucżynię je ośiádłośćią bąków / y káłużámi wód / y wymiotę go miotłą ſpuſtoƺeniá / mówi Pán záſtępów.
Przyśiągł Pán záſtępów / mówiąc : Záiſte / jákom umyślił / ták będzie / á jákom urádził / ták śię ſtánie ;
Iż potrę Aſſyryjcżyká w ziemi mojej / á ná górách moich podepcżę go ; á odejdzie od nich járzmo jego / y brzemię jego z rámieniá jego zdjęte będzie.
Táć jeſt rádá urádzoná przećiw onej wƺyſtkiej ziemi ; á táć jeſt ręká wyćiągnioná przećiwko tym wƺyſtkim národom.
A poniewáż Pán záſtępów poſtánowił / któż to wzruƺy? á rękę jego wyćiągnioną któż odwróći?
Roku / którego umárł król Acház / ſtáło śię to proroctwo ;
Nie ráduj śię / ty wƺyſtká ziemio Filiſtyńſká! iż złámáná jeſt rózgá tego / który ćię bił ; bo z korzeniá wężowego wynijdzie bázyliƺek / á płód jego będzie ſmok ogniſty látájący.
Y będą śię páść pierworodni nędznych / á ubodzy bezpiecżnie odpocżną ; ále korzeń twój głodem wygubię á oſtátki twoje wybiję.
Kwilże brámo! krzycż miáſto! jużeś śię rozpłynęłá wƺyſtká ty ziemio Filiſtyńſká ; bo od północy ogień przyjdzie / á nie będzie / coby ſtronił od pocżtów jego.
A cóż odpowiedzą poſłom národu? To / że Pán ugruntowáł Syon / á do niego śię ućiekáć będą ubodzy ludu jego.
