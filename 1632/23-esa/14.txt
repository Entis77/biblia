Abowiem zlituje śię Pán nád Jákóbem / á obierze záśię Izráelá / y da im odpocżnąć w źiemi ich ; á przyłącży śię do nich cudzoźiemiec / y przyſtáną do domu Jákóbowego.
Bo wezmą z ſobą národy / y przywiodą je do miejſcá ſwego ; y weźmie je ſobie dom Izráelſki w źiemi Páńſkiej w dźiedźictwo zá ſługi y zá ſłużebnice ; y imáć będą tych / którzy ich imáli / á pánowáć będą nád tymi / którzy ich ćiemiężyli.
A dniá onego / któregoć Pán da odpocżnienie od prácy twojey y od ſtráchu twego / y od niewoli ćiężkiej / w którąś był podbity /
Weźmieƺ tę przypowieść przećiw królowi Bábilońſkiemu / y rzecżeƺ : O jáko uſtał poborcá / uſtał podátek złotá!
Pán złámał kij niezbożnych / y rózgę pánujących ;
Tego / który ludźi bijał w zápálcżywośći bićiem uſtáwicżnem / pánował w gniewie nád národámi bez litośći dręcżonemi ;
Teraz odpocżywá y jeſt w pokoju wƺyſtká źiemiá / wƺyſcy głośno śpiewáją ;
Y jodły śię weſelą nád tobą / y cedry Libáńſkie / mówiąc : Od tego cżáſu / jákoś ty poległ / nie powſtał / ktoby nas podćinał.
Y piekło ze ſpodku wzruƺyło śię dla ćiebie / áby tobie przychodzącemu záƺło ; wzbudźiło dla ćiebie umárłych / wƺyſtkich kśiążąt źiemi ; rozkázáło powſtáć z ſtolic ſwojich y wƺyſtkim królom národów.
Ci wƺyſcy odpowiádájąc mówią do ćiebie : Y tyś zemdlony jáko y my / á ſtáłeś śię nam podobnym.
Strąconá jeſt do piekłá / pychá twojá y dźwięk muzyki twojey ; podeſłánoć mole / á robáki ćię przykrywáją.
Jákoż to / żeś ſpádł z niebá / o jutrzenko! która wſchodźiƺ ráno? powálonyś áż ná źiemię / któryś wątlił národy!
Wƺákieś ty máwiał w ſercu ſwem : Wſtąpię ná niebo / nád gwiázdy Boże wywyżƺę ſtolicę moję / á uśiądę ná górze zgromádzenia / ná ſtronách północnych :
Wſtąpię ná wyſokość obłoków / będę równy Nájwyżƺemu.
Wƺákże ſtrącon jeſteś áż do piekłá / w głębokość dołu.
Którzy ćię ujrzą / zá tobą śię oglądáć / y przypátrywáć ći śię będą mówiąc : Onże to mąż / który trwożył źiemię? który trząſał króleſtwámi?
Który obróćił okrąg świátá w puſtynię / á miáſtá jego poburzył / á więźniom ſwoim nie otwárzał ćiemnicy?
Wƺyſcy królowie národów / cokolwiek ich było / pochowáni ſą w ſłáwie / káżdy w domu ſwoim.
Aleś ty odrzucony od grobu ſwego / jáko látorośl obrzydłá / jáko ƺátá zábitych / których poprzebijáno miecżem / którzy zſtępują do grobu kámieniſtego / jáko śćierw podeptány.
Nie będźiƺ miał ucżeſnictwá z nimi w pogrzebie ; boś źiemię twoję pokáźił / y lud ſwój pomordował ; ábowiem náśienie złośników nie będźie wſpomináne ná wieki.
Gotujćie ſynów jego ná zámordowánie dla niepráwośći ojców ich / áby nie powſtáli / y nie odźiedźicżyli źiemi / nie nápełnili miáſtámi okręgu źiemſkiego.
Bo powſtánę przećiwko nim / mówi Pán zaſtępów / á wykorzenię imię Bábilonu / y oſtátki jego / ták ſyná jáko y wnuká / mówi Pán ;
Y ucżynię je ośiádłośćią bąków / y káłużámi wód / y wymiotę go miotłą ſpuſtoƺeniá / mówi Pán zaſtępów.
Przyśiągł Pán zaſtępów / mówiąc : Záiſte / jákom umyślił / ták będźie / á jákom urádźił / ták śię ſtánie ;
Iż potrę Aſſyryjcżyká w źiemi mojey / á ná górách mojich podepcżę go ; á odejdźie od nich járzmo jego / y brzemię jego z rámieniá jego zdjęte będźie.
Táć jeſt rádá urádzoná przećiw oney wƺyſtkiej źiemi ; á táć jeſt ręká wyćiągnioná przećiwko tym wƺyſtkim národom.
A poniewáż Pán zaſtępów poſtánowił / któż to wzruƺy? á rękę jego wyćiągnioną któż odwróći?
Roku / którego umárł król Acház / ſtáło śię to proroctwo ;
Nie ráduj śię / ty wƺyſtká źiemio Filiſtyńſká! iż złámáná jeſt rózgá tego / który ćię bił ; bo z korzeniá wężowego wynijdźie bázyliƺek / á płód jego będźie ſmok ogniſty látájący.
Y będą śię páść pierworodni nędznych / á ubodzy bezpiecżnie odpocżną ; ále korzeń twój głodem wygubię á oſtátki twoje wybiję.
Kwilże brámo! krzycż miáſto! jużeś śię rozpłynęłá wƺyſtká ty źiemio Filiſtyńſká ; bo od północy ogień przydźie / á nie będźie / coby ſtronił od pocżtów jego.
A cóż odpowiedzą poſłom národu? To / że Pán ugruntował Syon / á do niego śię ućiekáć będą ubodzy ludu jego.
