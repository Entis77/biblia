Ale jednák nie ták záćmioná będzie oná ziemiá / którá ućiśnioná będzie / jáko pierwƺego cżáſu / gdy Bóg dotknął ziemię Zábulon / y ziemię Neftálim ; áni jáko potem / gdy obćiążył ku drodze morſkiej przy Jordánie Gálileę ludną.
Bo lud on / który chodzi w ćiemnośći / ujrzy świátłość wielką / á tym / którzy mieƺkáją w ziemi ćieniá śmierći / świátłość świećić będzie.
Rozmnożyłeś ten náród / áleś nie ucżynił wielkiego weſelá ; wƺákże weſelić śię będą przed tobą / jáko śię weſelą cżáſu żniwá / jáko śię rádują / którzy łupy dzielą ;
Gdyż járzmo brzemieniá jego / á láſkę rámieniá jego / y pręt poborcy jego złámieƺ / jáko zá dni Mádyjáńcżyków /
Gdzie śię wƺyſtká bitwá bojujących z trzáſkiem ſtáłá / y ƺáty były we krwi zbrocżone / á co śię ſpálić mogło / ogniem ſpálono.
Abowiem dziećię národziło śię nám / á ſyn dány jeſt nám ; y będzie pánowánie ná rámieniu jego / á názwią imię jego : Dziwny / Rádny / Bóg mocny / Ojćiec wiecżnośći / Kśiążę pokoju ;
A ku rozmnożeniu tego páńſtwá y pokoju / któremu końcá nie będzie / uśiądzie ná ſtolicy Dawidowej / y ná króleſtwie jego / áż je poſtánowi y utwierdzi w ſądzie y w ſpráwiedliwośći / odtąd áż ná wieki. Ucżyni to záwiſná miłość Páná záſtępów.
Poſłáł Pán ſłowo do Jákóbá / á upádło w Izráelu.
Y dowie śię wƺyſtek lud Efráim / y mieƺkájący w Sámáryi / którzy w hárdośći y w wynioſłośći ſercá mówią :
Cegły upádły / ále my ćioſánym kámieniem budowáć będziemy / podrąbáno płonne figi / ále my to w cedry odmienimy.
Aleć Pán wywyżƺy nieprzyjáćiół Ráſynowych nádeń / á nieprzyjáćiół jego zbierze ;
Syryjcżyków z przodku / á Filiſtyńcżyków z tyłu / y pożrą Izráelá cáłą gębą. A wƺákże w tem wƺyſtkiem nie odwróći śię zápálcżywość jego / ále jeƺcże ręká jego będzie wyćiągnioná.
Przeto / że śię lud ten nie náwrácá do tego / który go bije / á Páná záſtępów nie ƺuká :
Dlátego Pán odetnie od Izráelá głowę y ogon / gáłąź y śitowie / dniá jednego.
( Stárzec y ucżćiwy cżłowiek / ten jeſt głową / á prorok / który ucży kłámſtwá / ten jeſt ogonem. )
Abowiem wodzowie ludu tego ſą zwodzićiele / á którzy śię im wodzić dádzą / zginęli.
Dlátego z młodzieńców jego Pán śię nie ućieƺy / á nád śierotámi jego / y nád wdowámi jego nie zmiłuje śię ; ábowiem wƺyſcy ſą obłudni y złośliwi / á káżde uſtá mówią ſproſność. A wƺákże w tem wƺyſtkiem nie odwróći śię zápálcżywość jego ; ále jeƺcże ręká jego będzie wyćiągnioná.
Abowiem gdy śię niepobożność jáko ogień roznieći / pożre głóg y ćiernie : potem zápáli gęſtwinę láſu / ſkąd śię rozwieją jáko dym ná powietrzu.
Abowiem dlá rozgniewániá Páná záſtępów záćmi śię ziemiá / á ten lud będzie jáko ſtráwá ogniá / y żáden brátu ſwemu nie przepuśći.
A porwieli co po práwej ſtronie / przećie łáknąć będzie ; á będzieli żárł po lewej / przećie śię nie náſyći ; káżdy z nich ćiáło rámieniá ſwego żreć będzie ;
Mánáſeſ Efráimá / á Efráim Mánáſeſá / á obáj ſpołu będą przećiwko Judzie. Wƺákże w tem wƺyſtkiem nie odwróći śię zápálcżywość jego ; ále jeƺcże ręká jego będzie wyćiągnioná.
