Ale jednák nie ták záćmioná będźie oná źiemiá / która ućiśnioná będźie / jáko pierwƺego cżáſu / gdy Bóg dotknął źiemię Zábulon / y źiemię Neftálim ; áni jáko potem / gdy obćiążył ku drodze morſkiej przy Jordánie Gálileę ludną.
Bo lud on / który chodźi w ćiemnośći / ujrzy świátłość wielką / á tym / którzy mieƺkáją w źiemi ćieniá śmierći / świátłość świećić będźie.
Rozmnożyłeś ten naród / áleś nie ucżynił wielkiego weſelá ; wƺákże weſelić śię będą przed tobą / jáko śię weſelą cżáſu żniwá / jáko śię rádują / którzy łupy dźielą ;
Gdyż járzmo brzemieniá jego / á láſkę rámieniá jego / y pręt poborcy jego złámieƺ / jáko zá dni Mádyjáńcżyków /
Gdźie śię wƺyſtká bitwá bojujących z trzáſkiem ſtáłá / y ƺáty były we krwi zbrocżone / á co śię ſpálić mogło / ogniem ſpálono.
Abowiem dźiećię národźiło śię nam / á ſyn dány jeſt nam ; y będźie pánowánie ná rámieniu jego / á názwią imię jego : Dźiwny / Rádny / Bóg mocny / Oćiec wiecżnośći / Kśiążę pokoju ;
A ku rozmnożeniu tego páńſtwá y pokoju / któremu końcá nie będźie / uśiądźie ná ſtolicy Dawidowej / y ná króleſtwie jego / áż je poſtánowi y utwierdźi w ſądźie y w ſpráwiedliwośći / odtąd áż ná wieki. Ucżyni to záwiſná miłość Páná zaſtępów.
Poſłał Pán ſłowo do Jákóbá / á upádło w Izráelu.
Y dowie śię wƺyſtek lud Efráim / y mieƺkájący w Sámáryi / którzy w hárdośći y w wynioſłośći ſercá mówią :
Cegły upádły / ále my ćioſánym kámieniem budowáć będźiemy / podrąbáno płonne figi / ále my to w cedry odmienimy.
Aleć Pán wywyżƺy nieprzyjáćiół Ráſynowych nádeń / á nieprzyjáćiół jego zbierze ;
Syryjcżyków z przodku / á Filiſtyńcżyków z tyłu / y pożrą Izráelá cáłą gębą. A wƺákże w tym wƺyſtkim nie odwróći śię zápálcżywość jego / ále jeƺcże ręká jego będźie wyćiągnioná.
Przeto / że śię lud ten nie náwrácá do tego / który go bije / á Páná zaſtępów nie ƺuká :
Dla tego Pán odetnie od Izráelá głowę y ogon / gáłąź y śitowie / dniá jednego.
( Stárzec y ucżćiwy cżłowiek / ten jeſt głową / á prorok / który ucży kłámſtwá / ten jeſt ogonem. )
Abowiem wodzowie ludu tego ſą zwodźićiele / á którzy śię im wodźić dádzą / zginęli.
Dla tego z młodźieńców jego Pán śię nie ućieƺy / á nád śierotámi jego / y nád wdowámi jego nie zmiłuje śię ; ábowiem wƺyſcy ſą obłudni y złośliwi / á káżde uſtá mówią ſproſność. A wƺákże w tym wƺyſtkim nie odwróći śię zápálcżywość jego ; ále jeƺcże ręká jego będźie wyćiągnioná.
Abowiem gdy śię niepobożność jáko ogień roznieći / pożre głóg y ćiernie : potem zápáli gęſtwinę láſu / ſkąd śię rozwieją jáko dym ná powietrzu.
Abowiem dla rozgniewánia Páná zaſtępów záćmi śię źiemiá / á ten lud będźie jáko ſtráwá ogniá / y żáden brátu ſwemu nie przepuśći.
A porwieli co po práwej ſtronie / przećie łáknąć będźie ; á będźieli żárł po lewej / przećie śię nie náſyći ; káżdy z nich ćiáło rámieniá ſwego żreć będźie ;
Mánáſes Efráimá / á Efráim Mánáſeſá / á obáj ſpołu będą przećiwko Judźie. Wƺákże w tym wƺyſtkim nie odwróći śię zápálcżywość jego ; ále jeƺcże ręká jego będźie wyćiągnioná.
