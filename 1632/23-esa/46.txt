Pochylił śię Bel / upádł Nebo ; báłwány ich włożone ſą ná beſtyje / y ná bydlętá ; tem záiſte / co wy nośićie / będą bárdzo obćiążone áż do uſtániá.
Pochyliły śię / y upádły ſpołem / y Bábilońcżycy nie będą mogli rátowáć brzemion ; owƺem / y duƺá ich w niewolę pójdźie.
Słuchájćie mię / domie Jákóbowy / y wƺyſtkie oſtátki domu Izráelſkiego! które noƺę záráz z żywotá / które piáſtuję záráz od národzeniá ;
Já ſám áż do ſtárośći / y owƺem áż do ſędźiwośći wáſ nośić będę. Jám wáſ ucżynił / Já też nośić będę ; Já mówię nośić wáſ będę / y wybáwię.
Komuż mię przypodobáćie / y przyrównáćie / álbo podobnym ucżynićie / żebym mu był podobny?
Ci / którzy márnie wydáwáją złoto z worká / á ſrebro ná ƺálách wáżą / nájmują zá zápłátę złotniká / áby ucżynił z niego bogá / przed którym pádáją y kłániáją śię.
Noƺą go ná rámieniu / dźwigáją go / y ſtáwiáją go ná miejſcu jego. Y ſtoi / á z miejſcá ſwego śię nie ruƺy ; jeźli kto záwołá do niego / nie ozywá śię / áni go z utrápieniá jego wybáwiá.
Pámiętájćież ná to / á wſtydźćie śię ; przypuśććie to do ſercá / o przeſtępnicy!
Wſpomnijćie ſobie ná rzecży pierwƺe / które śię dźiáły od wieku ; bom Já Bóg / á niemáƺ żádnego Bogá więcej / y niemáƺ mnie podobnego ;
Który opowiádám od pocżątku rzecży oſtátnie / y zdáwná to / co śię jeƺcże nie ſtáło ; rzekęli co / rádá mojá oſtoi śię / y wƺyſtkę wolę moję ucżynię.
Który záwołám od wſchodu ſłońcá ptáká / z źiemi dálekiej tego / któryby wykonáł rádę moję. Rzekłem / á dowiodę tego ; umyśliłem / á ucżynię to.
Słuchájćie mię / wy upornego ſercá / którzy jeſteśćie dálekimi od ſpráwiedliwośći.
Spráwię / że śię przybliży ſpráwiedliwość mojá / nie pójdźie w długą / á zbáwienie moje nie omieƺká ; bo położę w Syonie zbáwienie / á w Izráelu ſłáwę moję.
