Brzemię puſtego morzá. Jáko wicher ná południe bieży / ták przyjdźie z puƺcży / z źiemi ſtráƺney.
Widzenie ſrogie jeſt mi okázáne. Przewrotny przewrotność broi / á puſtoƺyćiel puſtoƺy. Przyćiągnijże / Elámie! Oblęż / Medźie! Bábilon ; wƺelkiemu wzdychániu jego koniec ucżynię.
Dlátego nápełnione ſą biodrá moje boleśćią / á ućiſk ogárnął mię / jáko ućiſk rodzącą. Skrzywiłem śię ſłyƺąc / á ſtrwożyłem śię widząc.
Ulękło śię ſerce moje / ſtrách mię ogárnął ; noc rozkoƺy moich obróćiłá mi śię w lękánie.
Przygotuj ſtół ; niech ſtráż ná ſtráży będźie ; jedz / pij ; wſtáńćie kśiążętá / ſmárujćie tárcże.
Abowiem ták mi rzekł Pán : Idź / poſtáw ſtróżá / któryby powiedźiáł / cokolwiek ujrzy.
Y ujrzáł wozy / y dwá rzędy jezdnych ; wozy / które oſły / y wozy / które wielbłądy ćiągnęły : y przypátrywáł śię im z wielką bárdzo pilnośćią.
Tedy záwołáł jáko lew : Pánie mój! jáć ſtoję ná ſtráży uſtáwicżnie we dnie ; náwet ná ſtráży mojey ſtáję ná káżdą noc.
( A oto wtem przyjecháli mężowie ná wozách / y jázdá dwomá rzędámi. ) Y záwołáł ſtráżnik / á rzekł : Upádł / upádł Bábilon / y wƺyſtkie ryte obrázy bogów jego pokruƺone o źiemię.
Bábilon jeſt gumno moje / y zboże bojewiſká mego. Com ſłyƺáł od Páná záſtępów / Bogá Izráelſkiego / tom wám opowiedźiáł.
Brzemię Dumy. Wołá ná mię ktoś z Seiru : Hej / ſtróżu! co śię ſtáło w nocy? Stróżu! co śię ſtáło w nocy?
Rzekł ſtróż : Przyƺedł poránek / tákże y noc. Chcećieli ƺukáć / ƺukájćie / náwróććie śię á przyjdźćie.
Brzemię ná Arábiję. Po láſách Arábii noclegi miewáć będźiećie / o podróżni Dedáńſcy!
Niech zábieżą prágnącemu / nioſąc wodę obywátele źiemi Temáńſkiej ; z chlebem jego niech wynijdą przećiw ućiekájącemu.
Bo przed miecżámi ućiekáć będą / przed miecżem dobytym / przed łukiem nápiętym / przed ćiężkośćią bitwy.
Gdyż ták rzekł Pán do mnie : Że po roku / jáki jeſt rok nájemnicży / uſtánie wƺyſtká ſłáwá Kedár.
A oſtátek pocżtu ſtrzelców mężnych ſynów Kedár będźie umniejƺony ; ábowiem to Pán Bóg Izráelſki mówił.
