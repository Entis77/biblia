Brzemię puſtego morzá. Jáko wicher ná południe bieży / ták przydźie z puƺcży / z źiemi ſtráƺney.
Widzenie ſrogie jeſt mi okázáne. Przewrotny przewrotność broi / á puſtoƺyćiel puſtoƺy. Przyćiągnijże / Elámie! Oblęż / Medźie! Bábilon ; wƺelkiemu wzdychániu jego koniec ucżynię.
Dla tego nápełnione ſą biodrá moje boleśćią / á ućiſk ogárnął mię / jáko ućiſk rodzącą. Skrzywiłem śię ſłyƺąc / á ſtrwożyłem śię widząc.
Ulękło śię ſerce moje / ſtrách mię ogárnął ; noc rozkoƺy mojich obróćiłá mi śię w lękánie.
Przygotuj ſtół ; niech ſtráż ná ſtráży będźie ; jedz / pij ; wſtáńćie kśiążętá / ſmárujćie tárcże.
Abowiem ták mi rzekł Pán : Idź / poſtáw ſtróżá / któryby powiedźiał / cokolwiek ujrzy.
Y ujrzał wozy / y dwá rzędy jezdnych ; wozy / które oſły / y wozy / które wielbłądy ćiągnęły : y przypátrywał śię im z wielką bárzo pilnośćią.
Tedy záwołał jáko lew : Pánie mój! jáć ſtoję ná ſtráży uſtáwicżnie we dnie ; náwet ná ſtráży mojey ſtáję ná káżdą noc.
( A oto wtem przyjecháli mężowie ná wozách / y jázdá dwomá rzędámi. ) Y záwołał ſtráżnik / á rzekł : Upádł / upádł Bábilon / y wƺyſtkie ryte obrázy bogów jego pokruƺone o źiemię.
Bábilon jeſt gumno moje / y zboże bojewiſká mego. Com ſłyƺał od Páná zaſtępów / Bogá Izráelſkiego / tom wam opowiedźiał.
Brzemię Dumy. Wołá ná mię ktoś z Seiru : Hej / ſtróżu! co śię ſtáło w nocy? Stróżu! co śię ſtáło w nocy?
Rzekł ſtróż : Przyƺedł poránek / tákże y noc. Chcećieli ƺukáć / ƺukájćie / náwróććie śię á przyjdźćie.
Brzemię ná Arábiję. Po láſách Arábii noclegi miewáć będźiećie / o podróżni Dedáńſcy!
Niech zábieżą prágnącemu / nioſąc wodę obywátele źiemi Temáńſkiej ; z chlebem jego niech wynijdą przećiw ućiekájącemu.
Bo przed miecżámi ućiekáć będą / przed miecżem dobytym / przed łukiem nápiętym / przed ćiężkośćią bitwy.
Gdyż ták rzekł Pán do mnie : Że po roku / jáki jeſt rok nájemnicży / uſtánie wƺyſtká ſłáwá Kedar.
A oſtátek pocżtu ſtrzelców mężnych ſynów Kedar będźie umniejƺony ; ábowiem to Pán Bóg Izráelſki mówił.
