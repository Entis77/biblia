Záśpiewám teraz miłemu memu pioſnkę miłego mego o winnicy jego. Winnicę ma miły mój ná págórku urodzájnym ;
Którą ogrodźił / y wybrał z niej kámienie / á náſádźił ją máćicámi wybornemi / y zbudował wieże w pośrzodku niej / tákże y práſę poſtáwił w niej / á cżekał / áby wydáłá groná ; ále oná zrodźiłá płonne wino.
A ták / obywátele Jeruzálemſcy y mężowie Judzcy! proƺę / rozſądźćie teraz miedzy mną y miedzy winnicą moją.
Cóż dálej cżynić było winnicy mojey / cżegobym jey nie ucżynił? Gdym rzekał / áby wydáłá groná / cżemuż zrodźiłá płonne wino?
A przetoż oznájmię wam / co ja ucżynię winnicy mojey : Rozbiorę płot jey / á będźie ſpuſtoƺoná ; rozwálę ogrodzenie jey / á będźie podeptáná.
Y ucżynię ją puſtą ; nie będźie obrzezywáná / áni okopywáná / ále porośnie oſtem y ćierniem ; obłokom też przykáżę / áby ná nią więcey dżdżu nie ſpuƺcżáły.
Winnicá zájiſte Páná zaſtępów jeſt dom Izráelſki / á mąż Judzki ƺcżepieniem jego rozkoƺnem. Ocżekiwał ſądu / á oto ućiśnienie ; ocżekiwał ſpráwiedliwośći / á oto krzyk.
Biádá wam / którzy przyłącżáćie dom do domu / á rolę do roli przycżyniáćie / ták / że miejſcá innym nie ſtáje / jákobyśćie tylko ſámi mieƺkáć mieli ná źiemi!
Pán zaſtępów rzekł w uƺy moje : Zájiſte wiele domów ſpuſtoƺeje / á wielkie y piękne domy będą bez obywátelá.
Do tego dźieśięć ſtáján winnicy przynioſą jednę báryłę winá / á jeden chomer náśieniá wydá efá.
Biádá tym / którzy ráno wſtáwájąc chodzą zá pijáńſtwem / á trwáją ná nim do wiecżorá / áż ich wino rozpáli!
A cytrá / y lutniá / bęben y piƺcżáłká / y wino bywa ná bieśiádách ich ; ále ná ſpráwy Páńſkie nie pátrzą / á ná ucżynki rąk jego nie oglądáją śię.
Przetoż w niewolę pójdźie lud mój / iż nie ma umiejętnośći ; á zácni jego będą głodnymi / y poſpólſtwo jego wyſchnie od prágnieniá.
Dla tego rozƺerzyło piekło gárdło ſwoje / á rozdárło nád miárę páƺcżękę ſwoję / y zſtąpią do niego ƺláchtá y poſpólſtwo jego / y zgiełk jego / y ći / którzy śię weſelą w nim.
A ták będźie náchylony cżłowiek / á zácny mąż poniżony będźie / y ocży wynioſłych zniżone będą.
Ale Pán zaſtępów wywyżƺony będźie w ſądźie / á Bóg święty ukáże śię świętym w ſpráwiedliwośći.
Y będą śię páść báránki według zwycżáju ſwego / á przychodniowie puſtyń bogácżów pożywáć będą.
Biádá tym / którzy ćiągną niepráwość powrozámi márnośći / á grzech jáko powrozem wozowym!
Którzy mówią : Niech śię poſpieƺy / á niecháj nie omieƺkuje ſpráwá jego / ábyſmy ją widźieli ; niech śię przybliży y przydźie rádá świętego Izráelſkiego / żebyſmy śię dowiedźieli.
Biádá tym / którzy názywáją złe dobrem á dobre złem ; którzy pokłádáją ćiemność zá świátłość / á świátłość zá ćiemność ; którzy pokłádáją gorzkość zá ſłodkość / á ſłodkość zá gorzkość!
Biádá tym / którzy śię ſobie zdádzą być mądrymi / á ſámi u śiebie roztropnymi!
Biádá tym / którzy ſą mocni ná pićie winá / á mężom dużym ku nálewániu nápoju mocnego!
Którzy uſpráwiedliwiáją niezbożnego zá podárki / á ſpráwiedliwość ſpráwiedliwych odejmują od nich!
Przetoż jáko płomień ogniowy pożerá páźdźieże / y jáko płomień plewy tráwi : ták korzeń ich będźie jáko zgnilizná / á kwiát ich jáko proch ku górze pójdźie ; ábowiem odrzućili zakon Páná zaſtępów / á wyrokiem świętego Izráelſkiego pogárdźili.
Dla tego śię zápáliłá popędliwość Páńſka przećiw ludowi ſwemu / á wyćiągnąwƺy nań rękę ſwą poráźił go / ták że śię zátrząſnęły góry / y były trupy ich jáko gnój po ulicách. W tym jednák wƺyſtkim nie odwróćiłá śię zápálcżywość jego / ále jeƺcże ręká jego jeſt wyćiągnioná.
Bo podnieśie chorągiew do narodu dálekiego / á záświśnie nań od końcżyn źiemi / á oto rychło y prędko przydźie.
Żádnego ſprácowánego y upádájącego nie będźie miedzy nimi ; nie będźie drzemiącego áni śpiącego / áni śię rozepnie pás ná biodrách jego / áni śię rozerwie rzemyk u trzewików jego.
Strzáły jego oſtre / y wƺyſtkie łuki jego náćiągnione ; kopytá koni jego jáko krzemień pocżytáne będą / á kołá jego jáko burzá.
Ryk jego jáko lwi ; będźie rycżał jáko ƺcżeniętá lwie ; będźie zgrzytał / y porwie łup / y ućiecże z nim / á nie będźie ktoby go wydárł.
Y záƺumi nád nim dniá onego jáko ƺum morſki. Tedy ſpojrzymy ná źiemię / á oto ćiemność y ućiſk ; bo y świátło záćmi śię przy wytráceniu jego.
