Dniá onego náwiedźi Pán miecżem ſwoim ſrogim / wielkim y mocnym / Lewiátáná / wężá długiego / y Lewiátáná / wężá ſkręconego / á zábije ſmoká / który jeſt w morzu.
Dniá onego śpiewájćie o winnicy wybornego winá.
Ja Pán / który jey ſtrzegę / co chwilká odwilżáć ją będę / á żeby jey kto nie pſuł / w nocy y we dnie ſtrzedz jey będę.
Zápálcżywośći żádney we mnie niemáƺ. Któż mi da oſet álbo ćiernie / ábym przećiwko niej wálcżył / y ſpálił ją do ƺcżętu?
Izali kto ujmie śiłę moję / áby ucżynił pokój zemną? áby pokój / mówię / ucżynił zemną?
Przydźie do tego / że śię Jákób rozkorzeni / zákwitnie y rozrodźi śię Izráel / y nápełni okrąg źiemſki owocem.
Bo izali go ták uderzy / jáko uderzył nieprzyjáćielá jego? álbo izali go zámordował / jáko inni ſą zámordowáni od niego?
Owƺem / miernie go kárał / y w ten cżás / gdy go wypychał y gdy go nieprzyjáćiel wiátrem ſwoim gwáłtownym w dźień wſchodniego wiátru / zábierał.
Przetoż tym ſpoſobem ocżyƺcżoná będźie niepráwość Jákóbowá ; á tenći jeſt wƺyſtek pożytek / że odejmie grzech jego / gdy rozrzući wƺyſtkie kámienie ołtárzá / jáko kámienie wápienne rozƺárpáne / á nie oſtoją śię gáje y obrázy ſłonecżne.
Gdy miáſto obronne ſpuſtoƺeje / á będźie mieƺkániem porzuconem y ſpuſtoƺonem jáko puſtyniá. Tám śię páść / y tám legáć będźie ćielec / y ogryźie látoroſłki jego.
Gdy poſchną gáłązki jego / pokruƺone będą / á niewiáſty przyƺedƺy zápálą je. Abowiem ten lud nie ma żádnego rozumu ; przetoż nie zmiłuje śię nád nim / który go ucżynił / á który go ſtworzył / nie zlituje śię nád nim.
Dniá onego / gdy śię Pán będźie mśćił od łożyſká rzeki áż do potoku Egipſkiego / wy ſynowie Izráelſcy po jednemu zebráni będźiećie.
Stánie śię też dniá onego / że zátrąbią w trąbę wielką / y przyjdą / którzy byli poginęli w źiemi Aſſyryjſkiej / y którzy byli zágnáni do źiemi Egipſkiej ; y będą śię Pánu kłániáli ná górze świętey w Jeruzálemie.
