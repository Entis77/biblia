Dniá onego náwiedzi Pán miecżem ſwoim ſrogim / wielkim y mocnym / Lewiátáná / wężá długiego / y Lewiátáná / wężá ſkręconego / á zábije ſmoká / który jeſt w morzu.
Dniá onego śpiewájćie o winnicy wybornego winá.
Já Pán / który jej ſtrzegę / co chwilká odwilżáć ją będę / á żeby jej kto nie pſuł / w nocy y we dnie ſtrzedz jej będę.
Zápálcżywośći żádnej we mnie niemáƺ. Któż mi dá oſet álbo ćiernie / ábym przećiwko niej wálcżył / y ſpálił ją do ƺcżętu?
Izáli kto ujmie śiłę moję / áby ucżynił pokój zemną? áby pokój / mówię / ucżynił zemną?
Przyjdzie do tego / że śię Jákób rozkorzeni / zákwitnie y rozrodzi śię Izráel / y nápełni okrąg ziemſki owocem.
Bo izáli go ták uderzy / jáko uderzył nieprzyjáćielá jego? álbo izáli go zámordowáł / jáko inni ſą zámordowáni od niego?
Owƺem / miernie go káráł / y w ten cżáſ / gdy go wypycháł y gdy go nieprzyjáćiel wiátrem ſwoim gwáłtownym w dzień wſchodniego wiátru / zábieráł.
Przetoż tym ſpoſobem ocżyƺcżoná będzie niepráwość Jákóbowá ; á tenći jeſt wƺyſtek pożytek / że odejmie grzech jego / gdy rozrzući wƺyſtkie kámienie ołtárzá / jáko kámienie wápienne rozƺárpáne / á nie oſtoją śię gáje y obrázy ſłonecżne.
Gdy miáſto obronne ſpuſtoƺeje / á będzie mieƺkániem porzuconem y ſpuſtoƺonem jáko puſtyniá. Tám śię páść / y tám legáć będzie ćielec / y ogryzie látoroſłki jego.
Gdy poſchną gáłązki jego / pokruƺone będą / á niewiáſty przyƺedłƺy zápálą je. Abowiem ten lud nie má żádnego rozumu ; przetoż nie zmiłuje śię nád nim / który go ucżynił / á który go ſtworzył / nie zlituje śię nád nim.
Dniá onego / gdy śię Pán będzie mśćił od łożyſká rzeki áż do potoku Egipſkiego / wy ſynowie Izráelſcy po jednemu zebráni będziećie.
Stánie śię też dniá onego / że zátrąbią w trąbę wielką / y przyjdą / którzy byli poginęli w ziemi Aſſyryjſkiej / y którzy byli zágnáni do ziemi Egipſkiej ; y będą śię Pánu kłániáli ná górze świętej w Jeruzálemie.
