Któż uwierzył kázániu náƺemu / á rámię Páńſkie komu objáwione jeſt?
Bo wyróſł jáko látoroſtká przed nim / á jáko korzeń z źiemi ſuchej / nie májąc kƺtáłtu áni pięknośći ; y widźieliśmy go ; ále nic nie było widźieć / cżemubyśmy go żądáć mieli.
Nájwzgárdzeńƺy był / y nájpodlejƺy z ludźi / mąż boleśći / á świádomy niemocy / y jáko zákrywájący twárz ſwoję ; nájwzgárdzeńƺy mówię / ſkądeśmy go zá nic nie mieli.
Záiſte on niemocy náƺe wźiął ná śię / á boleśći náƺe włáſne nośił ; á myśmy mniemáli / że jeſt zrániony / ubity od Bogá y utrápiony.
Lecż on zrániony jeſt dla wyſtępków náƺych / ſtárty jeſt dla niepráwośći náƺych ; káźń pokoju náƺego jeſt ná nim / á śinośćią jego jeſteśmy uzdrowieni.
Wƺyſcyśmy jáko owce zbłądźili / káżdy ná drogę ſwą obróćiliśmy śię / á Pán włożył náń niepráwość wƺyſtkich nas.
Ućiśniony jeſt y utrápiony / á nie otworzył uſt ſwoich ; jáko báránek ná zábićie wiedźiony był / y jáko owcá przed tymi / którzy ją ſtrzygą / oniemiáł / y nie otworzył uſt ſwoich.
Z więźieniá y z ſądu wyjęty jeſt ; przetoż rodzáj jego któż wypowie? Abowiem wyćięty jeſt z źiemi żyjących / á zrániony dla przeſtępſtwá ludu mojego ;
Który to lud podáł niezbożnym grób jego / á bogátemu śmierć jego / choć jednák niepráwośći nie ucżynił / áni zdrádá ználeźioná jeſt w uſtách jego.
Tákći śię Pánu upodobáło zetrzeć go / y niemocą utrápić / áby położywƺy ofiárą zá grzech duƺę ſwą / ujrzáł náśienie ſwoje / przedłużył dni ſwoich ; á to / co śię podobá Pánu / przez rękę jego áby śię ƺcżęśliwie wykonáło.
Z prácy duƺy ſwej ujrzy owoc / którym náſycon będźie. Znájomośćią ſwoją wielu uſpráwiedliwi ſpráwiedliwy ſługá mój ; bo niepráwośći ich on ſám ponieśie.
Przetoż mu dám dźiáł dla wielu / áby śię dźielił korzyśćią z mocárzámi / poniewáż wyláł ná śmierć duƺę ſwoję / á z przeſtępcámi policżon będąc / on ſám grzech wielu odnióſł / y zá przeſtępców śię modlił.
