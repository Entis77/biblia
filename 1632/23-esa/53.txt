Któż uwierzył kázániu náƺemu / á rámię Páńſkie komu objáwione jeſt?
Bo wyróſł jáko látoroſtká przed nim / á jáko korzeń z ziemi ſuchej / nie májąc kƺtáłtu áni pięknośći ; y widzieliśmy go ; ále nic nie było widzieć / cżemubyśmy go żądáć mieli.
Nájwzgárdzeńƺy był / y nájpodlejƺy z ludzi / mąż boleśći / á świádomy niemocy / y jáko zákrywájący twárz ſwoję ; nájwzgárdzeńƺy mówię / ſkądeśmy go zá nic nie mieli.
Záiſte on niemocy náƺe wziął ná śię / á boleśći náƺe włáſne nośił ; á myśmy mniemáli / że jeſt zrániony / ubity od Bogá y utrápiony.
Lecż on zrániony jeſt dlá wyſtępków náƺych / ſtárty jeſt dlá niepráwośći náƺych ; káźń pokoju náƺego jeſt ná nim / á śinośćią jego jeſteśmy uzdrowieni.
Wƺyſcyśmy jáko owce zbłądzili / káżdy ná drogę ſwą obróćiliśmy śię / á Pán włożył náń niepráwość wƺyſtkich náſ.
Ućiśniony jeſt y utrápiony / á nie otworzył uſt ſwoich ; jáko báránek ná zábićie wiedziony był / y jáko owcá przed tymi / którzy ją ſtrzygą / oniemiáł / y nie otworzył uſt ſwoich.
Z więzieniá y z ſądu wyjęty jeſt ; przetoż rodzáj jego któż wypowie? Abowiem wyćięty jeſt z ziemi żyjących / á zrániony dlá przeſtępſtwá ludu mojego ;
Który to lud podáł niezbożnym grób jego / á bogátemu śmierć jego / choć jednák niepráwośći nie ucżynił / áni zdrádá ználezioná jeſt w uſtách jego.
Tákći śię Pánu upodobáło zetrzeć go / y niemocą utrápić / áby położywƺy ofiárą zá grzech duƺę ſwą / ujrzáł náśienie ſwoje / przedłużył dni ſwoich ; á to / co śię podobá Pánu / przez rękę jego áby śię ƺcżęśliwie wykonáło.
Z prácy duƺy ſwej ujrzy owoc / którym náſycon będzie. Znájomośćią ſwoją wielu uſpráwiedliwi ſpráwiedliwy ſługá mój ; bo niepráwośći ich on ſám ponieśie.
Przetoż mu dám dziáł dlá wielu / áby śię dzielił korzyśćią z mocárzámi / poniewáż wyláł ná śmierć duƺę ſwoję / á z przeſtępcámi policżon będąc / on ſám grzech wielu odnióſł / y zá przeſtępców śię modlił.
