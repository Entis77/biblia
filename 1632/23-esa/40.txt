Cieƺćie / ćieƺćie lud mój! mówi Bóg wáƺ.
Mówćie do ſercá Jeruzálemu : ogłáƺájćie mu / że śię już dopełnił cżáſ poſtánowiony jego / że jeſt odpuƺcżoná niepráwość jego / y że wźiął z ręki Páńſkiej w dwójnáſób zá wƺyſtkie grzechy ſwoje.
Głoſ wołájącego ná puƺcży : Gotujćie drogę Páńſką / proſtą cżyńćie ná puſtyni śćieƺkę Bogá náƺego.
Káżdá doliná niech podnieśioná będźie / á káżdá górá y págórek niech poniżony będźie ; co jeſt krzywego / niech śię wyproſtuje / á miejſcá nierówne niech będą równiną.
Bo śię objáwi chwałá Páńſká / á ujrzy wƺelkie ćiáło ſpołem / iż uſtá Páńſkie mówiły.
Głoſ mówiącego : Wołáj. Y rzekł : Cóż mam wołáć? To : Wƺelkie ćiáło jeſt tráwá / á wƺyſtká zácność jego jáko kwiát polny.
Tráwá uſychá / kwiát opádá ; ſkoro wiátr Páńſki powionie náń ; zápráwdęć ludźie ſą tą tráwą.
Tráwá uſychá / kwiát opádá ; ále ſłowo Bogá náƺego trwá ná wieki.
Wſtąp ſobie ná górę wyſoką / Syonie! który opowiádáƺ rzecży ućieƺne. Podnieś mocno głoſ twój / Jeruzálemie! które opowiádáƺ rzecży poćieƺne ; podnieś / nie bój śię / rzecż miáſtom Judzkim : Oto Bóg wáƺ.
Oto pánujący Pán przyjdźie przećiwko mocnemu / á rámię jego pánowáć będźie nád nim ; oto zápłátá jego z nim / á dźieło jego przed nim.
Jáko páſterz trzodę ſwoję páść będźie ; do náręcżá ſwego zgromádźi báránki / y ná łonie ſwem piáſtowáć je będźie / á kotne zwolná poprowádźi.
Kto zmierzył wody gárśćią ſwoją / á niebioſá piędźią rozmierzył? á kto proch źiemi miárą zmierzył? kto zwáżył ná wádze góry / á págórki ná ƺálách?
Któż dośćignął duchá Páńſkiego / á kto rádcą jego był / żeby mu oznájmił?
Z kim wƺedł w rádę / żeby mu rozumu przydáł / á náucżył go śćieżek ſądu? Kto go náucżył umiejętnośći / á drogę wƺelákiej roztropnośći ukázáł mu?
Oto národy ſą jáko kroplá z wiádrá / á jáko proƺek ná ƺálách pocżytáne ſą ; wyſpy jáko nájmniejƺą rzecż porywá.
Y Libán nie wyſtárcżyłby ku wznieceniu ogniá / y zwierzętá jego nie wyſtárcżyłyby ná cáłopálenie.
Wƺyſtkie národy ſą jáko nic przed nim ; zá nic y zá márność pocżytáne ſą u niego.
Komuż tedy podobnym ucżynićie Bogá? A jákie podobieńſtwo przyrównáćie mu?
Rzemieślnik uleje báłwáná á złotnik złotem go powlecże / y łáńcuƺki ſrebrne do niego odleje.
A ten / który dla ubóſtwá nie má co ofiárowáć / obierá drzewo / któreby nie próchniáło / y rzemieślniká umiejętnego ſobie ƺuká / áby wygotowáł báłwáná rytego / któryby śię nie poruƺył.
Izáli nie wiećie? Izáli nie ſłyƺyćie? Izáli śię wám nie opowiádá od pocżątku? Izáli nie zrozumiewáćie od záłożeniá gruntów źiemi?
Ten / który śiedźi nád okręgiem źiemi / którey obywátele ſą jáko ƺáráńcżá ; ten / który rozpoſtárł niebioſá jáko ćienkie płótno / á rozćiągnął je / jáko námiot ku mieƺkániu :
Tenći kśiążąt w niwecż obrácá / ſędźiów źiemſkich jáko nic rozpráƺá.
Że nie bywáją ƺcżepieni áni wśiáni / áni śię też wkorzeni w źiemi pień ich ; y jáko jedno powienie ná nich / wnet uſycháją / á wicher jáko źdźbło unośi ich.
Komuż mię tedy przyrównáćie / ábym mu był podobny? mówi Święty.
Podnieśćie ku górze ocży wáƺe / á obácżćie! Kto to ſtworzył? kto wywiódł w pocżćie wojſko ich / á to wƺyſtko z imieniá przyzywá / według wielkośći śiły / y wielkiej mocy / ták / że áni jedno z nich nie zginie?
Przecżże tedy powiádáƺ / Jákóbie! przecżże ták mówiƺ Izráelu : Skrytá jeſt drogá mojá przed Pánem / á ſpráwá mojá przed Bogá mego nie przychodźi?
Izáli nie wieƺ? izáliś nie ſłyƺáł / że Bóg wiecżny Pán / który ſtworzył gránice źiemi / nie uſtánie / áni śię ſprácuje / y że nie może być dośćignioná mądrość jego?
Który dodáje ſprácowánemu śiły / á tego / który nie má żádney śiły / moc rozmnáżá.
Młódź uſtáje y omdlewá / á młodźieńcy w młodośći upádáją :
Ale którzy ocżekują Páná / nábywáją nowej śiły ; podnoƺą śię piórámi jáko orły / bieżą á nie ſprácują śię / chodzą á nie uſtáwáją.
