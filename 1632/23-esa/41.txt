Umilknijćie przedemną / wyſpy! á narody niech śię pośilą. Niech przyſtąpią á niech mówią : Przyſtąpmy ſpołem do ſądu.
Któż wzbudźił od wſchodu ſłońcá ſpráwiedliwego / y wezwał go / áby go náśládował? Któż mu podbił narody / áby nád królámi pánował / podáwƺy je jáko proch pod miecż jego / á jáko plewy rozproƺone pod łuk jego?
Ugániał śię z nimi / przeƺedł ſpokojnie śćieƺkę / po którey nogámi ſwemi nie chádzał.
Któż to ſpráwił y ucżynił? któż wzywał rodzáje od pocżątku? Ja Pán / pierwƺy y oſtátni / Ja ſám.
Widźiáły wyſpy / y ulękły śię ; końcżyny źiemi zdumiáły śię ; zgromádźiły śię / y zeƺły śię.
Jeden drugiemu pomágał / á brátu ſwemu mówił : Zmácniáj śię!
A ták zmácniał teƺárz złotniká / bláchę młotem głádzącego / kującego ná kowádle / mówiąc : Do lutowánia to dobre. Potem to ſtwierdźił gwoźdźiámi / áby śię nie ruƺyło.
Ale ty / Izráelu / ſługo mój! ty Jákóbie / któregom obrał / naśienie Abráhámá / przyjáćielá mego!
Ty / któregom pochwyćił od końcżyn źiemi / owƺem / pominąwƺy przedniejƺych ich / powołáłem ćię mówiąc do ćiebie : Sługáś ty mój / obráłem ćię / á nie odrzućiłem ćię.
Nie bój śię! bom Ja z tobą. Nie lękáj śię! bom Ja Bogiem twoim. Zmocnię ćię / á dam ći pomoc / y podeprę ćię práwicą ſpráwiedliwośći ſwojey.
Oto záwſtydzą śię / á będą poháńbieni wƺyſcy gniewem páłájący przećiwko tobie : ſtáną śię jáko nic / y zginą ći / którzy śię tobie ſprzećiwiáją.
Szukáłlibyś ich / nie znájdźieƺ ich ; ći / którzy śię ſprzećiwiáją tobie / będą jáko nic / á ći / którzy wálcżą z tobą / w niwecż obróceni będą.
Bom ja Pán / Bóg twój / trzymám ćię zá práwicę twoję / á mówięć : Nie bój śię! Ja ćię wſpomogę.
Nie bój śię / robácżku Jákóbie / gárſtko ludu Izráelſkiego! Jáć będę ná pomocy / mówi Pán á odkupićiel twój / Święty Izráelſki.
Otom ćię ucżynił jáko wóz z zębámi nowemi po obu ſtronách ; y pomłóćiƺ góry / á potrzeƺ je / á págórki jáko plewę położyƺ.
Przewiejeƺ je / wtem je wiátr porwie / á wicher rozproƺy je ; ále śię ty rozrádujeƺ w Pánu / w Świętym Izráelſkim będźieƺ śię chlubił.
Ubogich y nędznych / którzy ƺukáją wody / á niemáƺ jey / których język uſechł od prágnieniá / Ja Pán wyſłuchám ich ; Ja / Bóg Izráelſki / nie opuƺcżę ich.
Otworzę rzeki ná miejſcách wyſokich / á źródłá w pośród równin ; obrócę puſtynie w jeźiorá wód / á źiemię ſuchą w ſtrumienie wód.
Náſádzę ná puƺcży cedrów / wybornych cedrów / ſośien / y oliwnych drzew ; náſádzę puſtynię jedliną / wiązem / y bukƺpánem ;
Aby widźieli / y poználi / y uwáżáli / y zrozumieli / że to ręká Páńſka ucżyniłá / y że to Święty Izráelſki ſtworzył.
Przedłóżćie ſpráwę wáƺę / mówi Pán ; ukáżćie mocne dowody ſwoje / mówi król Jákóbowy.
Niech przyſtąpi / á niech nam oznájmi to / co śię ma ſtáć ; rzecży pierwƺe / które były / powiedzćie / ábyſmy uwáżyli w ſercu ſwem / á poználi cel ich ; álbo przynájmniej nam przyƺłe rzecży oznájmijćie.
Oznájmijćie / co ma przyść nápotem / á poznámy / żeśćie bogowie ; álbo ucżyńćie co dobrego lub złego / ábyſmy śię zdumiewáli / gdybyſmy to ſpołem widźieli.
Otośćie wy zgołá ná nic / á ſpráwá wáƺá tákże ná nic nie jeſt ; przetoż obrzydły jeſt ten / co was ſobie obierá.
Wzbudzę od północy lud / ten przyćiągnie ; y od wſchodu ſłońcá / ten wzywáć będźie imienia mego ; oborzy śię ná kśiążąt jáko ná błoto / á podepcże ich / jáko gárncárz glinę.
Kto oznájmi od pocżątku? tedy będźiemy wiedźieli ; álbo co było od dáwnych cżáſów? tedy rzecżemy : Tyś jeſt ſpráwiedliwy? Niemáƺ zgołá nikogo / coby oznájmił / áni jeſt / ktoby śię dał ſłyƺeć / álbo ktoby ſłyƺał mowy wáƺe.
Jam pierwƺy / który Syonowi opowiádám : Oto / oto ſą ; á Jeruzálemowi dam opowiádácży rzecży poćieƺnych.
Bo widzę / że niemáƺ nikogo / niemáƺ nikogo miedzy nimi / coby dał rádę ; ácż śię ich pytáją / wƺákże nie odpowiádáją y ſłowá.
Oto ći wƺyſcy ſą márnośćią / zá nic nie ſtoją ucżynki ich ; wiátrem y próżnośćią ſą odlewáne báłwány ich.
