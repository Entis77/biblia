Umilknijćie przedemną / wyſpy! á národy niech śię pośilą. Niech przyſtąpią á niech mówią : Przyſtąpmy ſpołem do ſądu.
Któż wzbudźił od wſchodu ſłońcá ſpráwiedliwego / y wezwáł go / áby go náśládowáł? Któż mu podbił národy / áby nád królámi pánowáł / podáwƺy je jáko proch pod miecż jego / á jáko plewy rozproƺone pod łuk jego?
Ugániáł śię z nimi / przeƺedł ſpokojnie śćieƺkę / po której nogámi ſwemi nie chádzáł.
Któż to ſpráwił y ucżynił? któż wzywáł rodzáje od pocżątku? Já Pán / pierwƺy y oſtátni / Já ſám.
Widźiáły wyſpy / y ulękły śię ; końcżyny źiemi zdumiáły śię ; zgromádźiły śię / y zeƺły śię.
Jeden drugiemu pomágáł / á brátu ſwemu mówił : Zmácniáj śię!
A ták zmácniáł teƺárz złotniká / bláchę młotem głádzącego / kującego ná kowádle / mówiąc : Do lutowániá to dobre. Potem to ſtwierdźił gwoźdźiámi / áby śię nie ruƺyło.
Ale ty / Izráelu / ſługo mój! ty Jákóbie / któregom obráł / náśienie Abráhámá / przyjáćielá mego!
Ty / któregom pochwyćił od końcżyn źiemi / owƺem / pominąwƺy przedniejƺych ich / powołáłem ćię mówiąc do ćiebie : Sługáś ty mój / obráłem ćię / á nie odrzućiłem ćię.
Nie bój śię! bom Já z tobą. Nie lękáj śię! bom Já Bogiem twoim. Zmocnię ćię / á dám ći pomoc / y podeprę ćię práwicą ſpráwiedliwośći ſwojej.
Oto záwſtydzą śię / á będą poháńbieni wƺyſcy gniewem páłájący przećiwko tobie : ſtáną śię jáko nic / y zginą ći / którzy śię tobie ſprzećiwiáją.
Szukáłlibyś ich / nie znájdźieƺ ich ; ći / którzy śię ſprzećiwiáją tobie / będą jáko nic / á ći / którzy wálcżą z tobą / w niwecż obróceni będą.
Bom já Pán / Bóg twój / trzymám ćię zá práwicę twoję / á mówięć : Nie bój śię! Já ćię wſpomogę.
Nie bój śię / robácżku Jákóbie / gárſtko ludu Izráelſkiego! Jáć będę ná pomocy / mówi Pán á odkupićiel twój / Święty Izráelſki.
Otom ćię ucżynił jáko wóz z zębámi nowemi po obu ſtronách ; y pomłóćiƺ góry / á potrzeƺ je / á págórki jáko plewę położyƺ.
Przewiejeƺ je / wtem je wiátr porwie / á wicher rozproƺy je ; ále śię ty rozrádujeƺ w Pánu / w Świętym Izráelſkim będźieƺ śię chlubił.
Ubogich y nędznych / którzy ƺukáją wody / á niemáƺ jej / których język uſechł od prágnieniá / Já Pán wyſłuchám ich ; Já / Bóg Izráelſki / nie opuƺcżę ich.
Otworzę rzeki ná miejſcách wyſokich / á źródłá w pośród równin ; obrócę puſtynie w jeźiorá wód / á źiemię ſuchą w ſtrumienie wód.
Náſádzę ná puƺcży cedrów / wybornych cedrów / ſośien / y oliwnych drzew ; náſádzę puſtynię jedliną / wiązem / y bukƺpánem ;
Aby widźieli / y poználi / y uwáżáli / y zrozumieli / że to ręká Páńſká ucżyniłá / y że to Święty Izráelſki ſtworzył.
Przedłóżćie ſpráwę wáƺę / mówi Pán ; ukáżćie mocne dowody ſwoje / mówi król Jákóbowy.
Niech przyſtąpi / á niech nám oznájmi to / co śię má ſtáć ; rzecży pierwƺe / które były / powiedzćie / ábyśmy uwáżyli w ſercu ſwem / á poználi cel ich ; álbo przynájmniej nám przyƺłe rzecży oznájmijćie.
Oznájmijćie / co má przyjść nápotem / á poznámy / żeśćie bogowie ; álbo ucżyńćie co dobrego lub złego / ábyśmy śię zdumiewáli / gdybyśmy to ſpołem widźieli.
Otośćie wy zgołá ná nic / á ſpráwá wáƺá tákże ná nic nie jeſt ; przetoż obrzydły jeſt ten / co wáſ ſobie obierá.
Wzbudzę od północy lud / ten przyćiągnie ; y od wſchodu ſłońcá / ten wzywáć będźie imieniá mego ; oborzy śię ná kśiążąt jáko ná błoto / á podepcże ich / jáko gárncárz glinę.
Kto oznájmi od pocżątku? tedy będźiemy wiedźieli ; álbo co było od dáwnych cżáſów? tedy rzecżemy : Tyś jeſt ſpráwiedliwy? Niemáƺ zgołá nikogo / coby oznájmił / áni jeſt / ktoby śię dáł ſłyƺeć / álbo ktoby ſłyƺáł mowy wáƺe.
Jám pierwƺy / który Syonowi opowiádám : Oto / oto ſą ; á Jeruzálemowi dám opowiádácży rzecży poćieƺnych.
Bo widzę / że niemáƺ nikogo / niemáƺ nikogo między nimi / coby dáł rádę ; ácż śię ich pytáją / wƺákże nie odpowiádáją y ſłowá.
Oto ći wƺyſcy ſą márnośćią / zá nic nie ſtoją ucżynki ich ; wiátrem y próżnośćią ſą odlewáne báłwány ich.
