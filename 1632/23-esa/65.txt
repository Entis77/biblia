Objáwiłem śię tym / którzy śię o mię nie pytáli ; ználeźionym jeſt od tych / którzy mię nie ƺukáli ; do národu / który śię nie názywáł imieniem mojem / rzekłem : Otom Ja! otom Ja!
Rozćiągnąłem ręce moje ná káżdy dźień do ludu upornego / który chodźi drogą nie dobrą zá myślámi ſwemi ;
Do ludu / który mię jáwnie wzruƺá do gniewu / uſtáwicżnie ofiárując w ogrodách / á kádząc ná cegłách ;
Którzy śiádáją przy grobách / á przy báłwánách ſwoich nocują ; którzy jedzą świnie mięſo / y polewkę obrzydłą z nácżyniá ſwego /
Mówiąc : Odſtąp precż / nie przyſtępuj do mnie ; bom jeſt świętobliwƺy niżeli ty. Cić ſą dymem w nozdrzách moich / y ogniem páłájącym przez cáły dźień.
Oto zápiſáno to przedemną : Nie zámilcżę / ále oddám y odpłácę ná łono ich.
Niepráwośći wáƺe / tákże y niepráwośći ojców wáƺych / mówi Pán / którzy kádźili po górách / á ná págórkách háńbili mię ; przetoż odmierzę ſpráwę ich pierwƺą ná łono ich.
Ták mówi Pán : Jáko gdyby kto ználázł wino w gronie / y rzekłby : Nie pſuj go / bo błogoſłáwieńſtwo jeſt w nim ; ták y Ja ucżynię dla ſług moich / że ich wƺyſtkich nie wygubię.
Bo wywiodę z Jákóbá náśienie / á z Judy dźierżáwcę gór moich ; y pośiędą ją wybráni moi / á ſłudzy moi tám mieƺkáć będą.
A Sáron będźie zá páſtwiſko owcom / á doliná Achor zá legowiſko wołów ludu mojego / którzy mię ƺukáli.
Ale was / którzyśćie opuśćili Páná / którzy zápomináćie ná górę świętobliwośći mojey / którzy gotujećie temu wojſku ſtół / á którzy oddáwáćie temu pocżtowi mokre ofiáry :
Was / mówię policżę pod miecż / ták / że wy wƺyſcy do zábićiá ſchyláć śię będźiećie / przeto / żem wołáł / á nie ozwáliśćie mi śię / mówiłem / á nie ſłyƺeliśćie / áleśćie cżynili / co złego jeſt przed ocżymá mojemi / á cżegom Ja nie chćiáł / obieráliśćie.
Przetoż ták mówi pánujący Pán : Oto ſłudzy moi jeść będą / á wy łáknąć będźiećie ; oto ſłudzy moi pić będą / á wy prágnąć będźiećie ; oto ſłudzy moi weſelić śię będą / á wy záwſtydzeni będźiećie.
Oto ſłudzy moi wykrzykáć będą od rádośći ſerdecżney / á wy będźiećie wołáć od boleśći ſercá / y od ſkruƺeniá duchá wyć będźiećie.
Y zoſtáwićie imię wáƺe ná przeklinánie wybránym moim / gdy was pomorduje pánujący Pán / á ſługi ſwe názwie innem imieniem.
Ten / który ſobie będźie błogoſłáwił ná źiemi / będźie ſobie błogoſłáwił w Bogu práwdźiwym ; á kto będźie przyśięgáł ná źiemi / będźie przyśięgáł przez Bogá práwdźiwego ; w zápomnienie záiſte przyjdą te ućiſki pierwƺe / á będą zákryte od ocżów moich.
Abowiem oto Ja tworzę niebioſá nowe / y źiemię nową / á nie będą wſpomináne rzecży pierwƺe / áni wſtąpią ná ſerce.
Owƺem weſelćie śię / á rádujćie śię ná wieki wieków z tego / co Ja ſtworzę ; bo oto Ja ſtworzę Jeruzálem ná rádość / á lud jego ná weſele.
Y rozráduję śię w Jeruzálemie / á weſelić śię będę w ludu moim ; á nie będźie ſłycháć w nim głoſu płácżu y głoſu nárzekániá.
Nie będźie tám więcey nikogo w wieku dźiećinnym / áni ſtárcá / któryby nie dopełnił dni ſwoich ; bo dźiećię we ſtu látách umrze ; ále grzeƺnik / choćby miał y ſto lát / przeklęty będźie.
Pobudują też domy / á będą w nich mieƺkáli ; náſádzą też winnic / á będą jeść owoce ich.
Nie będą budowáć ták / áby tám inƺy mieƺkáł ; nie będą ƺcżepić / áby inny jádł ; bo dni ludu mojego będą jáko dni drzewá / á dźiełá rąk ſwoich do zwietƺeniá używáć będą wybráni moi.
Nie będą robić próżno / áni płodźić będą ná poſtrách ; bo będą náśieniem błogoſłáwionych od Páná / oni y potomkowie ich z nimi.
Nádto ſtánie śię / że pierwej niż záwołáją / Ja śię ozwę ; jeƺcże mówić będą / á Ja wyſłuchám.
Wilk z báránkiem páść śię będą ſpołem ; lew jáko wół plewy jeść będźie / á wężowi proch będźie chlebem jego ; nie będą ƺkodźić áni zátrácáć ná wƺyſtkiej górze świętey mojey / mówi Pán.
