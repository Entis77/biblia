Y ſtáło śię zá dni Acházá / ſyná Joátámowego / ſyná Uzyjáƺá / królá Judzkiego / że przyćiągnął Ráſyn / król Syryjſki / y Fácejáƺ / ſyn Romelijáƺá / królá Izráelſkiego / pod Jeruzálem / áby wálcżył przećiw niemu : ále go nie mógł dobyć.
Y oznájmiono domowi Dawidowemu / mówiąc : Zmówiłá śię Syryjá z Efráimem. Tedy śię poruƺyło ſerce jego / y ſerce ludu jego / jáko śię poruƺáją drzewá leśne od wiátru.
Tedy rzekł Pán do Izájáƺá : Wyjdź teráz przećiw Acházowi / ty / y Seár Jáſub / ſyn twój / ná koniec rur ſádzáwki wyżƺej / ná drogę polá fárbierzowego ;
A powiedz mu : Pátrz / ábyś śię nie fráſowáł ; nie bój śię / á ſerce twoje niecháj śię nie lęká tych dwóch oſtátków głowien kurzących śię / to jeſt / zápálcżywośći gniewu Ráſyná z Syryjcżykámi / y ſyná Romelijáƺowego /
Przeto / że złą rádę urádzili przećiw tobie Syryjcżyk / Efráim / y ſyn Romelijáƺowy / mówiąc :
Ciągnijmy przećiwko ziemi Judzkiej / á utrápmy ją leżą / y oderwijmy ją do śiebie / á poſtánówmy królá w pośród niej / ſyná Tábeálowego.
Ták mówi Pán Pánujący : Nie ſtánie śię / y nie będzie to.
Abowiem głową Syryi jeſt Dámáƺek / á głową Dámáƺku Ráſyn ; á po ƺeśćdzieśięćiu y pięćiu látách będzie potárty Efráim / ták / iż więcej ludem nie będzie.
Między tem głową Efráimową będzie Sámáryjá / á głową Sámáryi ſyn Romelijáƺowy. Jeźli nie uwierzyćie / pewnie śię nie oſtoićie.
Nádto jeƺcże rzekł Pán do Acházá / mówiąc :
Żądáj ſobie znáku od Páná / Bogá twego / bądź ná dole niſko / bądź wyſoko w górze.
Tedy odpowiedziáł Acház : Nie będę żądáł / áni będę kuśił Páná.
A prorok rzekł : Słucháj teráz / domie Dawidowy! Máłoż śię wám zdá / uprzykrzáć śię ludziom / że śię uprzykrzáćie y Bogu mojemu?
Przetoż wám ſám Pán znák dá. Oto pánná pocżnie y porodzi ſyná / á názwie imię jego Immánuel.
Máſło y miód jeść będzie / áżby umiáł odrzucáć złe / á obieráć dobre.
Owƺem / pierwej niż będzie umiáło to dziećię odrzucáć złe y obieráć dobre / ziemiá / którą śię ty brzydziƺ / opuƺcżoná będzie od dwóch królów ſwoich.
Ale ná ćię Pán przywiedzie y ná lud twój / y ná dom ojcá twego / dni / jákich nie było ode dniá / którego odſtąpił Efráim od Judy / á to przez królá Aſſyryjſkiego.
Abowiem ſtánie śię dniá onego / że záświśnie Pán ná muchy / które ſą ná końcu rzek Egipſkich / y ná pƺcżoły / które ſą w ziemi Aſſyryjſkiej.
Y przyjdą á uśiądą wƺyſtkie w dolinách puſtych / y w rozpádlinách ſkálnych / y ná wƺyſtkich drzewách urodzájnych.
Dniá onego ogoli Pán brzytwą nájętą przez tych / którzy ſą zá rzeką / to jeſt ( przez królá Aſſyryjſkiego ) głowę / y włoſy ná nogách / tákże y brodę wƺcżąt ogoli.
Y ſtánie śię dniá onego / że ledwie cżłowiek żywo krówkę / álbo dwie owce záchowá.
A wƺákże dlá obfitośći mleká / którego nádoi / będzie jádł máſło ; máſło záiſte y miód będzie jádł / ktokolwiek pozoſtánie w ziemi.
Stánie śię też onegoż dniá / iż káżde miejſce / gdzie było tyśiąc winnych máćic zá tyśiąc ſrebrników / oſtem y ćierniem porośnie.
Tedy z ſtrzáłámi y z łukiem tám chodzić będą ; bo oſtem y ćierniem zárośnie wƺyſtká ziemiá.
Ná wƺyſtkie też góry / które motyką kopáne być mogą / nie przyjdzie ſtrách oſtu y ćierniá ; ále będą ná páſtwiſko wołom / y ná podeptánie owcom.
