Wołáj wƺyſtkiem gárdłem / nie záwśćiągáj ; wynoś głoſ ſwój jáko trąbá / á opowiedz ludowi mojemu przeſtępſtwá ich / á domowi Jákóbowemu grzechy ich ;
Choćiáż mię káżdego dniá ƺukáją / á znáć chcą drogi moje / jáko náród / który ſpráwiedliwość cżyni / á ſądu Bogá ſwego nie opuƺcżá ; pytáją mię o ſądách ſpráwiedliwośći á prágną śię przybliżyć do Bogá / mówiąc :
Przecżże pośćimy / gdyż ná to nie pátrzyƺ? trápimy duƺę náƺę / á nie widziƺ? Oto w dzień poſtu wáƺego przewodzićie wolę ſwoję / á wƺyſtkie práce ſwoje wyćiągáćie.
Oto pośćićie ná ſwáry / y ná zwády / y bijećie pięśćią niemiłośćiwie ; nie pośćićie / ják śię godzi tych dni / áby był ſłyƺány ná wyſokośći głoſ wáƺ.
Izáli to jeſt tákowy poſt / jákim obráł / á dzień / w któryby trápił cżłowiek duƺę ſwoję? żeby zwieśił jáko śitowie głowę ſwoję / á wór y popiół ſobie podśćieláł? Toż to názwieƺ poſtem / y dniem przyjemnym Pánu?
Ale to jeſt poſt / którym obráł : Rozwiąż związki niepobożnośći / rozwiąż brzemioná ćiężkie / y wolno puść ſkruƺonych / á ták wƺelákie járzmo rozerwij ;
Ułámuj łáknącemu chlebá twego / á ubogich wygnáńców wprowádź do domu twego ; ujrzyƺli nágiego / przyodziej go / á przed ćiáłem ſwojim nie ukrywáj śię.
Tedy wyniknie jáko zorzá ránná świátłość twojá / á zdrowie twoje prędko zákwitnie! y pójdzie przed tobą ſpráwiedliwość twojá / á chwáłá Páńſká zbierze ćię.
Tedy wzywáć będzieƺ / á Pán wyſłuchá ; záwołáƺ / á odpowieć : Owom Já. Jeźli odejmieƺ z pośrodku śiebie y járzmo / á przeſtánieƺ pálcá wyćiągáć / y mówić niepráwośći ;
Jeźli wylejeƺ łáknącemu duƺę ſwoję / á duƺę utrápioną náſyćiƺ : tedy wejdzie w ćiemnośći świátłość twojá / á zmierzk twój będzie jáko południe.
Bo ćię Pán uſtáwicżnie poprowádzi / y náſyći pod nájwiękƺą ſuƺą duƺę twoję / á kośći twoje utucży / y będzieƺ jáko ogród wilgotny / á jáko zdrój wód / którego wody nie uſtáwáją.
Y pobudują ſpłodzeni od ćiebie puſtynie ſtárodáwne / grunty od národu do národu wywiedzieƺ ; y názwą ćię nápráwcą obálin / y przepráwcą śćieżek ku mieƺkániu.
Jeźliże odwróćiƺ od ſábátu nogę ſwoję / ábyś nie przewodził woli ſwojej w dzień mój święty ; y jeżeli názwieƺ ſábát rozkoƺą / dniem świętym á Pánu ſłáwnym / y będzieƺli go miáł w ucżćiwośći / ták / ábyś weń nie cżynił dróg ſwoich / y nie przewodził woli ſwej / y nie mówił ſłowo próżnego :
Tedy będzieƺ rozkoƺowáł w Pánu ; y wprowádzą ćię ná wyſokie miejſcá ziemi / y ſpráwię to / ábyś pożywáł dziedzictwá Jákóbá / ojcá twego ; bo uſtá Páńſkie mówiły.
