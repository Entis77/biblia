Roku / którego umárł król Uzyjáƺ / widziáłem Páná / śiedzącego ná ſtolicy wyſokiej y wynioſłej / á podołek jego nápełniáł kośćiół.
Seráfinowie ſtáli nád nim / ƺeść ſkrzydeł miáł káżdy z nich ; dwomá zákrywáł twárz ſwoję / á dwomá przykrywáł nogi ſwoje / á dwomá látáł.
Y wołáł jeden do drugiego / mówiąc : Święty / święty / święty / Pán záſtępów ; pełná jeſt wƺyſtká ziemiá chwáły jego.
Y poruƺyły śię podwoje u drzwi od głoſu wołájącego / á dom pełny był dymu.
Y rzekłem : Biádá mnie! jużem zginął / przeto / żem cżłowiek ſplugáwionych wárg / á mieƺkám w pośrodku ludu / który má ſplugáwione wárgi ; á iż królá / Páná záſtępów / widziáły ocży moje.
Y przylećiáł do mnie jeden z Seráfinów / májąc w ręce ſwej węgiel rozpálony / który kleƺcżykámi wziął z ołtárzá ;
Y dotknął śię uſt moich / á rzekł : Oto śię dotknął ten węgiel wárg twoich / á odejdzie niepráwość twojá / á grzech twój zgłádzony będzie.
Potemem ſłyƺáł głoſ Páná mówiącego : Kogoż poślę? á kto nám pójdzie? Tedym rzekł : Otom já / poślij mię.
A on rzekł : Idź / á powiedz ludowi temu : Słuchájćie ſłuchájąc / á nie rozumijćie / á widząc pátrzájćie / á nie poznáwájćie.
Zátwárdź ſerce ludu tego / á uƺy jego obćiąż / y ocży jego záwrzyj / áby nie widziáł ocżymá ſwemi / á uƺymá ſwemi nie ſłyƺáł / y ſercem ſwem nie zrozumiáł / á nie náwróćił śię / y nie był uzdrowion.
A gdym rzekł : Dokądże Pánie? A on rzekł : Dokąd nie ſpuſtoƺeją miáſtá / ták áby nie było obywátelá ; y domy / áby nie było w nich cżłowieká / á ziemiá do ƺcżętu nie ſpuſtoƺeje ;
Dokąd Pán dáleko nie zápędzi wƺelkiego cżłowieká / á nie będzie doſkonáłe ſpuſtoƺenie w pośród ziemi ;
Dokąd jeƺcże ná nią dzieśiątá zgubá nie przyjdzie / á dopiero ſkáżoná będzie. A wƺákże jáko one dęby / które ſą przy brámie Zállechet podporą / ták náśienie święte jeſt podporą jej.
