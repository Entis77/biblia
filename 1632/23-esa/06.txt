Roku / którego umárł król Uzyjáƺ / widźiáłem Páná / śiedzącego ná ſtolicy wyſokiej y wynioſłej / á podołek jego nápełniał kośćiół.
Seráfinowie ſtáli nád nim / ƺeść ſkrzydeł miał káżdy z nich ; dwomá zákrywał twárz ſwoję / á dwomá przykrywał nogi ſwoje / á dwomá látał.
Y wołał jeden do drugiego / mówiąc : Święty / święty / święty / Pán zaſtępów ; pełna jeſt wƺyſtká źiemiá chwały jego.
Y poruƺyły śię podwoje u drzwi od głoſu wołájącego / á dom pełny był dymu.
Y rzekłem : Biádá mnie! jużem zginął / przeto / żem cżłowiek ſplugáwionych wárg / á mieƺkám w pośrzodku ludu / który ma ſplugáwione wárgi ; á iż królá / Páná zaſtępów / widźiáły ocży moje.
Y przylećiał do mnie jeden z Seráfinów / májąc w ręce ſwej węgiel rozpálony / który kleƺcżykámi wźiął z ołtárzá ;
Y dotknął śię uſt mojich / á rzekł : Oto śię dotknął ten węgiel wárg twojich / á odejdźie niepráwość twojá / á grzech twój zgłádzony będźie.
Potemem ſłyƺał głos Páná mówiącego : Kogoż poślę? á kto nam pójdźie? Tedym rzekł : Otom ja / poślij mię.
A on rzekł : Idź / á powiedz ludowi temu : Słuchájćie ſłuchájąc / á nie rozumijćie / á widząc pátrzájćie / á nie poznáwájćie.
Zátwárdź ſerce ludu tego / á uƺy jego obćiąż / y ocży jego záwrzyj / áby nie widźiał ocżymá ſwemi / á uƺymá ſwemi nie ſłyƺał / y ſercem ſwem nie zrozumiał / á nie náwróćił śię / y nie był uzdrowion.
A gdym rzekł : Dokądże Pánie? A on rzekł : Dokąd nie ſpuſtoƺeją miáſtá / ták áby nie było obywátelá ; y domy / áby nie było w nich cżłowieká / á źiemiá do ƺcżętu nie ſpuſtoƺeje ;
Dokąd Pán dáleko nie zápędźi wƺelkiego cżłowieká / á nie będźie doſkonáłe ſpuſtoƺenie w pośród źiemi ;
Dokąd jeƺcże ná nią dźieśiątá zgubá nie przydźie / á dopiero ſkáżoná będźie. A wƺákże jáko one dęby / które ſą przy brámie Zállechet podporą / ták náśienie święte jeſt podporą jey.
