Roku / którego umárł król Uzyjáƺ / widźiáłem Páná / śiedzącego ná ſtolicy wyſokiej y wynioſłej / á podołek jego nápełniáł kośćiół.
Seráfinowie ſtáli nád nim / ƺeść ſkrzydeł miał káżdy z nich ; dwomá zákrywáł twárz ſwoję / á dwomá przykrywáł nogi ſwoje / á dwomá látáł.
Y wołáł jeden do drugiego / mówiąc : Święty / święty / święty / Pán záſtępów ; pełná jeſt wƺyſtká źiemiá chwáły jego.
Y poruƺyły śię podwoje u drzwi od głoſu wołájącego / á dom pełny był dymu.
Y rzekłem : Biádá mnie! jużem zginął / przeto / żem cżłowiek ſplugáwionych wárg / á mieƺkám w pośrodku ludu / który má ſplugáwione wárgi ; á iż królá / Páná záſtępów / widźiáły ocży moje.
Y przylećiáł do mnie jeden z Seráfinów / májąc w ręce ſwej węgiel rozpálony / który kleƺcżykámi wźiął z ołtárzá ;
Y dotknął śię uſt moich / á rzekł : Oto śię dotknął ten węgiel wárg twoich / á odejdźie niepráwość twojá / á grzech twój zgłádzony będźie.
Potemem ſłyƺáł głoſ Páná mówiącego : Kogoż poślę? á kto nám pójdźie? Tedym rzekł : Otom ja / poślij mię.
A on rzekł : Idź / á powiedz ludowi temu : Słuchájćie ſłuchájąc / á nie rozumijćie / á widząc pátrzájćie / á nie poznáwájćie.
Zátwárdź ſerce ludu tego / á uƺy jego obćiąż / y ocży jego záwrzyj / áby nie widźiáł ocżymá ſwemi / á uƺymá ſwemi nie ſłyƺáł / y ſercem ſwem nie zrozumiáł / á nie náwróćił śię / y nie był uzdrowion.
A gdym rzekł : Dokądże Pánie? A on rzekł : Dokąd nie ſpuſtoƺeją miáſtá / ták áby nie było obywátelá ; y domy / áby nie było w nich cżłowieká / á źiemiá do ƺcżętu nie ſpuſtoƺeje ;
Dokąd Pán dáleko nie zápędźi wƺelkiego cżłowieká / á nie będźie doſkonáłe ſpuſtoƺenie w pośród źiemi ;
Dokąd jeƺcże ná nią dźieśiątá zgubá nie przyjdźie / á dopiero ſkáżoná będźie. A wƺákże jáko one dęby / które ſą przy brámie Zállechet podporą / ták náśienie święte jeſt podporą jey.
