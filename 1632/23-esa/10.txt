Biádá tym / którzy ſtánowią práwá nieſpráwiedliwe! y piſárzom / którzy ućiſk ná innych ſpiſują!
Aby odpycháli ubogiego od ſądu / á wydźieráli ſpráwiedliwość ubogich ludu mego ; áby wdowy były korzyśćią ich / á śierotki łupem ich.
Cóż ucżynićie w dźień náwiedzeniá / y ſpuſtoƺeniá / które z dáleká przyjdźie? do kogoż śię ućiecżećie o wſpomożenie? á gdźie zoſtáwićie ſłáwę wáƺę?
Aby śię nie miáłá między więźniámi uniżyć / y między pobitymi upáść. A wƺákże w tem wƺyſtkiem nie odwróći śię zápálcżywość jego ; ále jeƺcże ręką jego będźie wyćiągnioná.
Biádá Aſſurowi / rózdze gniewu mego! choćiáż kij rozgniewániá mego jeſt w ręku jego.
Ná náród obłudny poślę go / á o ludu zápálcżywośći mojej przykáżę mu / áby bráł łup y wydźieráł korzyśći á położył go ná podeptánie / jáko błoto ná ulicách.
Lecż on nie ták będźie mniemáł / y ſerce jego nie ták będźie myśláło / poniewáż w ſercu ſwem ułożył / áby wytráćił y wykorzenił niemáło národów.
Abowiem rzecże : Izáli kśiążętá moi nie ſą też y królmi?
Izáli Chálmo nie jeſt jáko Kárchemiſ? Izáli Arfát nie jeſt jáko Emát? Izáli Sámáryjá nie jeſt jáko Dámáƺek?
Jáko ręká mojá ználázłá króleſtwá báłwáńſkie / choćiáż báłwány ich więkƺe były / niż w Jeruzálemie y w Sámáryi.
Izáli Jeruzálemowi y báłwánom jego ták nie ucżynię / jákom ucżynił Sámáryi y báłwánom jej?
Y ſtánie śię / gdy Pán wykoná wƺyſtkę ſpráwę ſwoję ná górze Syońſkiej y w Jeruzálemie / że náwiedzę owoc wynioſłego ſercá królá Aſſyryjſkiego / y pychę wyſokich ocżów jego ;
Bo rzecże : W mocy ręki mojej ucżyniłem to / y w mądrośći mojej ; bom był mądry / y odjąłem gránice národów / á ſkárby ich zábráłem / y wytráćiłem obywáteli jáko mocárz.
Owƺem ręká mojá ználázłá májętność národów jáko gniázdo ; á jáko zbieráją jájká / które ſą opuƺcżone / tákiem já wƺyſtkę źiemię zebráł / á nie był ktoby ſkrzydłem ruƺył / álbo otworzył uſtá / y coby mrucżáł.
Izáli śię będźie przechwáláłá śiekierá przećiw temu / który nią rąbie? Izáli śię będźie wynośiłá piłá przećiw temu / który nią trze? jákożby śię wynośiłá rózgá przećiw temu / który ją podnióſł? jákożby śię przechwáláł kij / że nie jeſt drewnem?
Przetoż Pán / Pán záſtępów / pośle ná tłuſtych jego ſuchoty / á pod ſłáwą jego z prędká śię zápáli / jáko gwáłtowny ogień ;
Bo Świátłość Izráelowá będźie ogniem / á Święty jego płomieniem / który ſpáli y pożre ćiernie jego y oſet jego dniá jednego.
Tákże wſpániáłość láſu jego y urodzájnych pól jego / od duƺy áż do ćiáłá zniƺcży / y ſtánie śię jáko chorąży od ſtráchu ućiekájący.
A pozoſtáłych drzew láſu jego máłá licżbá będźie / ták / że je y dźiećię będźie popiſáć mogło.
Y ſtánie śię dniá onego / że oſtátki Izráelſkie / y ći / którzy zoſtáli z domu Jákóbowego / nie będą więcej ſpolegáć ná tym / co ich bije ; ále práwdźiwie ſpolegáć będą ná Pánu / Świętym Izráelſkim.
Oſtátek náwróći śię / oſtátek Jákóbowy do Bogá mocnego.
Bo choćby lud twój / o Izráelu! był jáko piáſek morſki / oſtátek tylko z niego náwróći śię. Wytrácenie náznácżone ſpráwi / że źiemiá będźie opływáłá ſpráwiedliwośćią.
Wytrácenie mówię náznácżone ucżyni Pán / Pán záſtępów / w pośrodku tej wƺyſtkiej źiemi.
Przetoż ták mówi Pán / Pán záſtępów : Nie bój śię Aſſyryjcżyká / ludu mój! który mieƺkáƺ w Syonie ; rózgą ubije ćię / á láſkę ſwą podnieśie ná ćię / jáko ná drodze Egipſkiej.
Abowiem po málucżkim cżáśie ſkońcży śię gniew mój przećiwko tobie / á ná wygłádzenie ich zápálcżywość mojá powſtánie.
Gdyż bicż náń wzbudźi Pán záſtępów / jáko poráżkę Mádyjáńcżyków ná ſkále Horeb ; á jáko podnióſł rózgę ſwoję ná morze ná drodze Egipſkiej / ták ją náń podnieśie.
A dniá onego zdjęte będźie brzemię jego z rámieniá twego / y járzmo jego z ƺyi twojej ; owƺem / ſkáżone będźie járzmo od przytomnośći pomázánego.
Przyćiągnie do Aját / przejdźie przez Migron / w Máchmáſ złoży oręż ſwój.
Przejdą bród / w Giebá jáko w goſpodźie nocowáć będą ; ulęknie śię Rámá / Gábáá Saulowe ućiecże.
Podnieś głoſ twój / córko Gállim! niech ſłyƺą w Láiſ / o ubogie Anátot!
Uſtąpi Mádmená ; obywátele Gábim zbiorą śię do ućiekániá.
Jeƺcże przez dźień záſtánowiwƺy śię w Nobie / pogroźi ręką ſwą górze córki Syońſkiej / y págórkowi Jeruzálemſkiemu.
Oto Pán / Pán záſtępów / okrzeƺe wƺyſtkę śiłę látorośli / á te / którzy ſą wyſokiego wzroſtu / podetnie ; y będą wyſocy poniżeni.
Gęſtwiny tákże láſów śiekierá wytnie / á Libán od wielmożnego upádnie.
