Biádá tym / którzy ſtánowią práwá nieſpráwiedliwe! y piſárzom / którzy ućiſk ná innych ſpiſują!
Aby odpycháli ubogiego od ſądu / á wydźieráli ſpráwiedliwość ubogich ludu mego ; áby wdowy były korzyśćią ich / á śierotki łupem ich.
Cóż ucżynićie w dźień náwiedzeniá / y ſpuſtoƺeniá / które z dáleká przydźie? do kogoż śię ućiecżećie o wſpomożenie? á gdźie zoſtáwićie ſłáwę wáƺę?
Aby śię nie miáłá miedzy więźniámi uniżyć / y miedzy pobitymi upáść. A wƺákże w tym wƺyſtkim nie odwróći śię zápálcżywość jego ; ále jeƺcże ręką jego będźie wyćiągnioná.
Biádá Aſſurowi / rózdze gniewu mego! choćiáż kij rozgniewánia mego jeſt w ręku jego.
Ná naród obłudny poślę go / á o ludu zápálcżywośći mojey przykáżę mu / áby brał łup y wydźierał korzyśći á położył go ná podeptánie / jáko błoto ná ulicách.
Lecż on nie ták będźie mniemał / y ſerce jego nie ták będźie myśláło / ponieważ w ſercu ſwem ułożył / áby wytráćił y wykorzenił niemáło narodów.
Abowiem rzecże : Izali kśiążętá moi nie ſą też y królmi?
Izali Chálmo nie jeſt jáko Kárchemis? Izali Arfát nie jeſt jáko Emát? Izali Sámáryjá nie jeſt jáko Dámáƺek?
Jáko ręká mojá ználázłá króleſtwá báłwáńſkie / choćiáż báłwány ich więkƺe były / niż w Jeruzálemie y w Sámáryi.
Izali Jeruzálemowi y báłwánom jego ták nie ucżynię / jákom ucżynił Sámáryi y báłwánom jey?
Y ſtánie śię / gdy Pán wykoná wƺyſtkę ſpráwę ſwoję ná górze Syońſkiej y w Jeruzálemie / że náwiedzę owoc wynioſłego ſercá królá Aſſyryjſkiego / y pychę wyſokich ocżów jego ;
Bo rzecże : W mocy ręki mojey ucżyniłem to / y w mądrośći mojey ; bom był mądry / y odjąłem gránice narodów / á ſkárby ich zábráłem / y wytráćiłem obywáteli jáko mocárz.
Owƺem ręká mojá ználázłá májętność narodów jáko gniázdo ; á jáko zbieráją jájká / które ſą opuƺcżone / tákiem ja wƺyſtkę źiemię zebrał / á nie był ktoby ſkrzydłem ruƺył / álbo otworzył uſtá / y coby mrucżał.
Izali śię będźie przechwáláłá śiekierá przećiw temu / który nią rąbie? Izali śię będźie wynośiłá piłá przećiw temu / który nią trze? jákożby śię wynośiłá rózgá przećiw temu / który ją podnióſł? jákożby śię przechwálał kij / że nie jeſt drewnem?
Przetoż Pán / Pán zaſtępów / pośle ná tłuſtych jego ſuchoty / á pod ſłáwą jego z prędká śię zápáli / jáko gwáłtowny ogień ;
Bo Świátłość Izráelowá będźie ogniem / á Święty jego płomieniem / który ſpáli y pożre ćiernie jego y oſet jego dniá jednego.
Tákże wſpániáłość láſu jego y urodzájnych pól jego / od duƺy áż do ćiáłá zniƺcży / y ſtánie śię jáko chorąży od ſtráchu ućiekájący.
A pozoſtáłych drzew láſu jego máłá licżbá będźie / ták / że je y dźiećię będźie popiſáć mogło.
Y ſtánie śię dniá onego / że oſtátki Izráelſkie / y ći / którzy zoſtáli z domu Jákóbowego / nie będą więcey ſpolegáć ná tym / co ich bije ; ále práwdźiwie ſpolegáć będą ná Pánu / Świętym Izráelſkim.
Oſtátek náwróći śię / oſtátek Jákóbowy do Bogá mocnego.
Bo choćby lud twój / o Izráelu! był jáko piáſek morſki / oſtátek tylko z niego náwróći śię. Wytrácenie náznácżone ſpráwi / że źiemiá będźie opływáłá ſpráwiedliwośćią.
Wytrácenie mówię náznácżone ucżyni Pán / Pán zaſtępów / w pośrzodku tey wƺyſtkiej źiemi.
Przetoż ták mówi Pán / Pán zaſtępów : Nie bój śię Aſſyryjcżyká / ludu mój! który mieƺkáƺ w Syonie ; rózgą ubije ćię / á láſkę ſwą podnieśie ná ćię / jáko ná drodze Egipſkiej.
Abowiem po málucżkim cżáśie ſkońcży śię gniew mój przećiwko tobie / á ná wygłádzenie ich zápálcżywość mojá powſtánie.
Gdyż bicż nań wzbudźi Pán zaſtępów / jáko poráżkę Mádyjáńcżyków ná ſkále Horeb ; á jáko podnióſł rózgę ſwoję ná morze ná drodze Egipſkiej / ták ją nań podnieśie.
A dniá onego zdjęte będźie brzemię jego z rámieniá twego / y járzmo jego z ƺyi twojey ; owƺem / ſkáżone będźie járzmo od przytomnośći pomázánego.
Przyćiągnie do Aját / przejdźie przez Migron / w Máchmás złoży oręż ſwój.
Przejdą bród / w Giebá jáko w goſpodźie nocowáć będą ; ulęknie śię Rámá / Gábáá Saulowe ućiecże.
Podnieś głos twój / córko Gállim! niech ſłyƺą w Láis / o ubogie Anátot!
Uſtąpi Mádmená ; obywátele Gábim zbiorą śię do ućiekánia.
Jeƺcże przez dźień záſtánowiwƺy śię w Nobie / pogroźi ręką ſwą górze córki Syońſkiej / y págórkowi Jeruzálemſkiemu.
Oto Pán / Pán zaſtępów / okrzeƺe wƺyſtkę śiłę látorośli / á te / którzy ſą wyſokiego wzroſtu / podetnie ; y będą wyſocy poniżeni.
Gęſtwiny tákże láſów śiekierá wytnie / á Libán od wielmożnego upádnie.
