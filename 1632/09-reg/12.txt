Y rzekł Sámuel do wƺyſtkiego Izráelá : Otom uſłucháł głoſu wáƺego we wƺyſtkiem / o cośćie ze mną mówili / y poſtánowiłem nád wámi królá.
A oto / teráz król chodzi przed wámi / á jám śię zſtárzáł y ośiwiáł ; oto / y ſynowie moi ſą z wámi / á jám też chodził przed wámi od młodoſći mojej áż do dniá tego.
Otom já tu. Świádcżćież przećiwko mnie przed Pánem / y przed pomázáńcem jego / jeźlim wziął któremu z wáſ wołu / álbo jeźlim wziął któremu z wáſ oſłá / y jeźlim kogo ućiſnął / álbo gwáłt komu ucżynił / y jeźlim z ręki cżyjej wziął dár / żebym miáł kryć ocży ſwoje dlá niego ; á nágrodzę wám.
Y odpowiedzieli : Nie ućiſnąłeś náſ / ániś nám gwáłtu ucżynił / ániś wziął z ręki cżyjej żádnej rzecży.
Nádto rzekł do nich : Świádkiem Pán przećiwko wám / y świádkiem pomázániec jego dniá tego / iżeśćie nic nie ználeźli w ręce mojej. A oni rzekli : Świádkiem.
Y rzekł Sámuel do ludu : Pán świádkiem / który ucżynił Mojżeƺá y Aároná / y który wywiódł ojce wáƺe z ziemi Egipſkiej.
Przetoż teráz ſtáńćie / ábym śię rozpieráł z wámi przed Pánem / o wƺyſtkie dobrodziejſtwá Páńſkie / które wám cżynił y ojcom wáƺym.
Gdy záƺedł Jákób do Egiptu / wołáli ojcowie wáśi do Páná / y poſłáł Pán Mojżeƺá y Aároná / którzy wywiedli ojce wáƺe z Egiptu / á poſádzili je ná tem miejſcu ;
A gdy zápomnieli Páná Bogá ſwego / podáł je w rękę Syſárze / hetmánowi wojſká Háſor / y w rękę Filiſtynów / tákże w rękę królá Moábſkiego / którzy wálcżyli przećiwko nim.
Ale gdy wołáli do Páná / y mówili : Zgrzeƺyliśmy / żeśmy opuśćili Páná / á ſłużyliśmy Báálom y Aſtárotowi / przetoż teráz wybáw náſ z rąk nieprzyjáćiół náƺych / á będziemyć ſłużyli :
Tedy poſłáł Pán Jerubáálá / y Bedoná / y Jeftego / y Sámuelá / á wyrwáł wáſ z ręki nieprzyjáćiół wáƺych okolicżnych / y mieƺkáliśćie bezpiecżnie.
Potem widząc / iż Náháſ / król ſynów Ammonowych / przyćiągnął przećiwko wám / rzekliśćie do mnie : Żádnym ſpoſobem ; ále król będzie królowáł nád námi : choć Pán Bóg wáƺ był królem wáƺym /
Teráz tedy oto król / któregośćie obráli / któregośćie żądáli ; oto / przełożył Pán królá nád wámi.
Jeźli śię będziećie báli Páná / á jemu ſłużyli / y ſłucháli głoſu jego á nie rozdráźnićie uſt Páńſkich / tedy y wy / y król / który króluje nád wámi / będziećie ƺcżęśliwie chodzić zá Pánem / Bogiem wáƺym.
Ale jeźliż nie będziećie ſłucháć głoſu Páńſkiego / á rozdráźnićie uſtá Páńſkie / będzie ręká Páńſká przećiwko wám / jáko y przećiwko ojcom wáƺym.
Jeƺcże teráz ſtójćie / á obácżćie tę rzecż wielką / którą Pán ucżyni przed ocżymá wáƺemi.
Izáli dziś nie pƺenicżne żniwá? Będę wzywáł Páná / á puśći gromy y dżdże / á dowiećie śię / y obácżyćie / jáká jeſt wielká złość wáƺá / którejśćie śię dopuśćili przed ocżymá Páńſkimi / żądájąc ſobie królá.
Przetoż wołáł Sámuel do Páná / y puśćił Pán gromy y deƺcż dniá onego / y báł śię wƺyſtek lud bárdzo Páná y Sámuelá.
Y rzekł wƺyſtek lud do Sámuelá : Módl śię zá ſługámi twymi Pánu Bogu twemu / żebyśmy nie pomárli : bośmy przydáli do wƺyſtkich grzechów náƺych tę złość / żeśmy ſobie prośili o królá.
Tedy rzekł Sámuel do ludu : Nie bójćie śię / ácżeśćie wy to wƺyſtko złe ucżynili ; wƺákże przeto nie odſtępujćie od Páná / ále ſłużćie Pánu ze wƺyſtkiego ſercá wáƺego ;
A nie udáwájćie śię zá próżnośćiámi / które wám nic nie pomogą / áni wáſ wybáwią / gdyż próżnośćiámi ſą.
Abowiemći nie opuśći Pán ludu ſwego / dlá imieniá ſwego wielkiego / gdyż śię upodobáło Pánu / ucżynić wáſ ſobie ludem.
A mnie nie dáj Boże / ábym miáł grzeƺyć przećiw Pánu / przeſtáwájąc modlić śię zá wámi ; owƺem wáſ będę náucżáł drogi dobrej y proſtej.
Jedno śię bójćie Páná / á ſłużćie mu w práwdzie ze wƺyſtkiego ſercá wáƺego / á to upátrujćie / jáko wielmożnie pocżynáł z wámi.
Ale jeźli przećię w złośći trwáć będziećie / tedy y wy / y król wáƺ poginiećie.
