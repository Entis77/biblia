Y rzekł Sámuel do wƺyſtkiego Izráelá : Otom uſłuchał głoſu wáƺego we wƺyſtkim / o cośćie ze mną mówili / y poſtánowiłem nád wámi królá.
A oto / teraz król chodźi przed wámi / á jam śię zſtárzał y ośiwiał ; oto / y ſynowie moi ſą z wámi / á jam też chodźił przed wámi od młodoſći mojey áż do dniá tego.
Otom ja tu. Świádcżćież przećiwko mnie przed Pánem / y przed pomázáńcem jego / jeſlim wźiął któremu z was wołu / álbo jeſlim wźiął któremu z was oſłá / y jeſlim kogo ućiſnął / álbo gwáłt komu ucżynił / y jeſlim z ręki cżyjey wźiął dar / żebym miał kryć ocży ſwoje dla niego ; á nágrodzę wam.
Y odpowiedźieli : Nie ućiſnąłeś nas / ániś nam gwáłtu ucżynił / ániś wźiął z ręki cżyjey żádney rzecży.
Nádto rzekł do nich : Świádkiem Pán przećiwko wam / y świádkiem pomázániec jego dniá tego / iżeśćie nic nie ználeſli w ręce mojey. A oni rzekli : Świádkiem.
Y rzekł Sámuel do ludu : Pán świádkiem / który ucżynił Mojżeƺá y Aároná / y który wywiódł ojce wáƺe z źiemi Egipſkiej.
Przetoż teraz ſtáńćie / ábym śię rozpierał z wámi przed Pánem / o wƺyſtkie dobrodźiejſtwá Páńſkie / które wam cżynił y ojcom wáƺym.
Gdy záƺedł Jákób do Egiptu / wołáli ojcowie wáśi do Páná / y poſłał Pán Mojżeƺá y Aároná / którzy wywiedli ojce wáƺe z Egiptu / á poſádźili je ná tym miejſcu ;
A gdy zápomnieli Páná Bogá ſwego / podał je w rękę Syſárze / hetmánowi wojſká Háſor / y w rękę Filiſtynów / tákże w rękę królá Moábſkiego / którzy wálcżyli przećiwko nim.
Ale gdy wołáli do Páná / y mówili : Zgrzeƺyliśmy / żeśmy opuśćili Páná / á ſłużyliśmy Báálom y Aſtárotowi / przetoż teraz wybáw nas z rąk nieprzyjáćiół náƺych / á będźiemyć ſłużyli :
Tedy poſłał Pán Jerubáálá / y Bedoná / y Jeftego / y Sámuelá / á wyrwał was z ręki nieprzyjáćiół wáƺych okolicżnych / y mieƺkáliśćie bezpiecżnie.
Potem widząc / iż Náhás / król ſynów Ammonowych / przyćiągnął przećiwko wam / rzekliśćie do mnie : Żádnym ſpoſobem ; ále król będźie królował nád námi : choć Pán Bóg wáƺ był królem wáƺym /
Teraz tedy oto król / któregośćie obráli / któregośćie żądáli ; oto / przełożył Pán królá nád wámi.
Jeſli śię będźiećie báli Páná / á jemu ſłużyli / y ſłucháli głoſu jego á nie rozdráźnićie uſt Páńſkich / tedy y wy / y król / który króluje nád wámi / będźiećie ƺcżęśliwie chodźić zá Pánem / Bogiem wáƺym.
Ale jeſliż nie będźiećie ſłucháć głoſu Páńſkiego / á rozdráźnićie uſtá Páńſkie / będźie ręká Páńſka przećiwko wam / jáko y przećiwko ojcom wáƺym.
Jeƺcże teraz ſtójćie / á obácżćie tę rzecż wielką / którą Pán ucżyni przed ocżymá wáƺemi.
Izali dźiś nie pƺenicżne żniwá? Będę wzywał Páná / á puśći gromy y dżdże / á dowiećie śię / y obácżyćie / jáká jeſt wielka złość wáƺá / którejśćie śię dopuśćili przed ocżymá Páńſkimi / żądájąc ſobie królá.
Przetoż wołał Sámuel do Páná / y puśćił Pán gromy y deƺcż dniá onego / y bał śię wƺyſtek lud bárzo Páná y Sámuelá.
Y rzekł wƺyſtek lud do Sámuelá : Módl śię zá ſługámi twymi Pánu Bogu twemu / żebyſmy nie pomárli : bośmy przydáli do wƺyſtkich grzechów náƺych tę złość / żeśmy ſobie prośili o królá.
Tedy rzekł Sámuel do ludu : Nie bójćie śię / ácżeśćie wy to wƺyſtko złe ucżynili ; wƺákże przeto nie odſtępujćie od Páná / ále ſłużćie Pánu ze wƺyſtkiego ſercá wáƺego ;
A nie udáwájćie śię zá próżnośćiámi / które wam nic nie pomogą / áni was wybáwią / gdyż próżnośćiámi ſą.
Abowiemći nie opuśći Pán ludu ſwego / dla imienia ſwego wielkiego / gdyż śię upodobáło Pánu / ucżynić was ſobie ludem.
A mnie nie dáj Boże / ábym miał grzeƺyć przećiw Pánu / przeſtáwájąc modlić śię zá wámi ; owƺem was będę náucżał drogi dobrej y proſtey.
Jedno śię bójćie Páná / á ſłużćie mu w práwdźie ze wƺyſtkiego ſercá wáƺego / á to upátrujćie / jáko wielmożnie pocżynał z wámi.
Ale jeſli przećię w złośći trwáć będźiećie / tedy y wy / y król wáƺ poginiećie.
