Potem wyƺedł Dawid ſtámtąd / y uƺedł do jáſkini Adullám. Co gdy uſłyƺeli bráćiá jego y wƺyſtek dom ojcá jego / przyƺli tám do niego.
Y zebráli śię do niego wƺyſcy / którzy byli utrápieni / y wƺyſcy / którzy byli dłużni / y wƺyſcy / którzykolwiek byli w gorzkośći ſercá / y był nád nimi kśiążęćiem / á było ich z nim około cżterech ſet mężów.
Y poƺedł Dawid ſtámtąd do Máſfá Moábſkiego / y rzekł do królá Moábſkiego : Niech śię przeprowádzi proƺę ojćiec mój / y mátká mojá / áby mieƺkáli z wámi / áż śię dowiem / co ucżyni Bóg ze mną.
A ták przywiódł je przed królá Moábſkiego ; y mieƺkáli z nim po wƺyſtkie dni / których był Dawid ná onym zámku.
Rzekł potem Gád prorok do Dawidá : Nie mieƺkáj więcej ná tym zámku ; idź / á wróć śię do ziemi Judſkiej. Tedy poƺedł Dawid / á przyƺedł do láſu Háret.
A uſłyƺáwƺy Saul / że śię pojáwił Dawid / y mężowie / którzy byli z nim / ( bo Saul mieƺkáł w Gábáá pod gájem w Rámáćie / májąc włócżnią ſwoję w rękách ſwych / á wƺyſcy ſłudzy jego ſtáli przed nim. )
Rzekł tedy Saul do ſług ſwych / którzy ſtáli przed nim : Słuchájćie proƺę ſynowie Jemini : Izáż wám wƺyſtkim dá ſyn Iſájego role y winnice / á wƺyſtkich wáſ pocżyni półkownikámi y rotmiſtrzámi /
Żeśćie śię ſprzyśięgli wy wƺyſcy przećiwko mnie / á niemáƺ ktoby mi objáwił? gdyż śię zbuntowáł y ſyn mój z ſynem Iſájego / á niemáƺ ktoby śię mnie użálił między wámi / á oznájmił mi / iż podburzył ſyn mój ſługę mego przećiwko mnie / áby cżyháł ná mię / jáko śię to dziś okázuje.
Tedy odpowiedziáł Doeg Edomcżyk / który też ſtáł z ſługámi Saulowymi / y rzekł : Widziáłem ſyná Iſájego / gdy przyƺedł do Noby / do Achimelechá / ſyná Achitobowego.
Który śię oń rádził Páná / y dáł mu żywnośći / dáł mu też y miecż Golijátá Filiſtyńcżyká.
A ták poſłáł król / áby przyzwáno Achimelechá / ſyná Achitobowego / kápłáná / y wƺyſtkiego domu ojcá jego kápłánów / którzy byli w Nobe. Y przyƺli oni wƺyſcy do królá.
Tedy rzekł Saul : Słucháj teráz ſynu Achitobá ; á on rzekł : Owom já / pánie mój.
Y rzekł do niego Saul : Cżemuśćie śię ſprzyśięgli przećiwko mnie / ty y ſyn Iſájego / gdyś mu dáł chleb y miecż / á rádziłeś śię oń Bogá / áby powſtáł przećiwko mnie / cżyhájąc ná mię / jáko śię to dziś okázuje?
Y odpowiedziáł Achimelech królowi / á rzekł : Y któż jeſt ták wierny między wƺyſtkimi ſługámi twoimi / jáko Dawid / który jeſt y zięćiem królewſkim y idzie zá rozkázániem twojim / á jeſt zácnym w domu twoim?
Azáż dziś pocżąłem śię oń rádzić Bogá? Uchowáj mię Boże! Niech nie wkłádá król ná ſługę twego nic tákiego / áni ná wƺyſtek dom ojcá mego ; bo nie wiedziáł ſługá twój o tem wƺyſtkiem nájmniejƺej rzecży.
Y rzekł król : Śmierćią umrzeƺ Achimelechu / ty y wƺyſtek dom ojcá twego.
Przytem rzekł król ſługom / którzy ſtáli przed nim : Obróććie śię / á pobijćie kápłány Páńſkie ; bo też ręká ich jeſt z Dawidem / gdyż wiedząc / że on ućiekáł / nie oznájmili mi. Ale ſłudzy królewſcy nie chćieli podnieść ręki ſwej / áni śię rzućić ná kápłány Páńſkie.
Przetoż rzekł król do Doegá : Obróć śię ty / á rzuć śię ná kápłány. A ták obróćiwƺy śię Doeg Edomcżyk / rzućił śię ná kápłány / y zábił onegoż dniá ośmdzieśiąt y pięć mężów / którzy nośili efod lniáne.
Nobe też miáſto kápłáńſkie wyśiekł oſtrzem miecżá / od mężá áż do niewiáſty / od máłego áż do ſſącego / y woły / y oſły / y owce wyśiekł oſtrzem miecżá.
Uƺedł tylko ſyn jeden Achimelechá / ſyná Achitobowego ; á imię jego Abijátár ; y ućiekł do Dawidá.
Tedy oznájmił Abijátár Dawidowi / że pobił Saul kápłány Páńſkie.
Y rzekł Dawid do Abijátárá : Wiedziáłem onegoż dniá / gdyż tám był Doeg Edomcżyk / że pewnie oznájmić miáł Saulowi : Jám jeſt przycżyną śmierći wƺyſtkich duƺ domu ojcá twego.
Zoſtáńże przy mnie / nie bój śię ; bo ktoby ƺukáł duƺy mojej / będzie ƺukáł duƺy twojej ; ále ty będzieƺ ochroniony przy mnie.
