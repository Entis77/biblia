Potem wyƺedł Dawid z támtąd / y uƺedł do jáſkini Adullám. Co gdy uſłyƺeli bráćia jego y wƺyſtek dom ojcá jego / przyƺli tám do niego.
Y zebráli śię do niego wƺyſcy / którzy byli utrápieni / y wƺyſcy / którzy byli dłużni / y wƺyſcy / którzykolwiek byli w gorzkośći ſercá / y był nád nimi kśiążęćiem / á było ich z nim około cżterech ſet mężów.
Y poƺedł Dawid z támtąd do Máſfá Moábſkiego / y rzekł do królá Moábſkiego : Niech śię przeprowádźi proƺę oćiec mój / y mátká mojá / áby mieƺkáli z wámi / áż śię dowiem / co ucżyni Bóg ze mną.
A ták przywiódł je przed królá Moábſkiego ; y mieƺkáli z nim po wƺyſtkie dni / których był Dawid ná onym zámku.
Rzekł potem Gád prorok do Dawidá : Nie mieƺkáj więcey ná tym zámku ; idź / á wróć śię do źiemi Judſkiej. Tedy poƺedł Dawid / á przyƺedł do láſu Háret.
A uſłyƺáwƺy Saul / że śię pojáwił Dawid / y mężowie / którzy byli z nim / ( bo Saul mieƺkał w Gábáá pod gájem w Rámáćie / májąc włócżnią ſwoję w rękách ſwych / á wƺyſcy ſłudzy jego ſtáli przed nim. )
Rzekł tedy Saul do ſług ſwych / którzy ſtáli przed nim : Słuchájćie proƺę ſynowie Jemini : Izáż wam wƺyſtkim da ſyn Iſájego role y winnice / á wƺyſtkich was pocżyni półkownikámi y rotmiſtrzámi /
Żeśćie śię ſprzyśięgli wy wƺyſcy przećiwko mnie / á niemáƺ ktoby mi objáwił? gdyż śię zbuntował y ſyn mój z ſynem Iſájego / á niemáƺ ktoby śię mnie użálił miedzy wámi / á oznájmił mi / iż podburzył ſyn mój ſługę mego przećiwko mnie / áby cżyhał ná mię / jáko śię to dźiś okázuje.
Tedy odpowiedźiał Doeg Edomcżyk / który też ſtał z ſługámi Saulowymi / y rzekł : Widźiałem ſyná Iſájego / gdy przyƺedł do Noby / do Achimelechá / ſyná Achitobowego.
Który śię oń rádźił Páná / y dał mu żywnośći / dał mu też y miecż Golijátá Filiſtyńcżyká.
A ták poſłał król / áby przyzwano Achimelechá / ſyná Achitobowego / kápłáná / y wƺyſtkiego domu ojcá jego kápłanów / którzy byli w Nobe. Y przyƺli oni wƺyſcy do królá.
Tedy rzekł Saul : Słucháj teraz ſynu Achitobá ; á on rzekł : Owom ja / pánie mój.
Y rzekł do niego Saul : Cżemuśćie śię ſprzyśięgli przećiwko mnie / ty y ſyn Iſájego / gdyś mu dał chleb y miecż / á rádźiłeś śię oń Bogá / áby powſtał przećiwko mnie / cżyhájąc ná mię / jáko śię to dźiś okázuje?
Y odpowiedźiał Achimelech królowi / á rzekł : Y któż jeſt ták wierny miedzy wƺyſtkimi ſługámi twoimi / jáko Dawid / który jeſt y źięćiem królewſkim y idźie zá rozkázániem twojim / á jeſt zácnym w domu twoim?
Azaż dźiś pocżąłem śię oń rádźić Bogá? Uchowáj mię Boże! Niech nie wkłádá król ná ſługę twego nic tákiego / áni ná wƺyſtek dom ojcá mego ; bo nie wiedźiał ſługá twój o tym wƺyſtkim nájmniejƺej rzecży.
Y rzekł król : Śmierćią umrzeƺ Achimelechu / ty y wƺyſtek dom ojcá twego.
Przytem rzekł król ſługom / którzy ſtáli przed nim : Obróććie śię / á pobijćie kápłány Páńſkie ; bo też ręká ich jeſt z Dawidem / gdyż wiedząc / że on ućiekał / nie oznájmili mi. Ale ſłudzy królewſcy nie chćieli podnieść ręki ſwej / áni śię rzućić ná kápłány Páńſkie.
Przetoż rzekł król do Doegá : Obróć śię ty / á rzuć śię ná kápłány. A ták obróćiwƺy śię Doeg Edomcżyk / rzućił śię ná kápłány / y zábił onegoż dniá ośmdźieśiąt y pięć mężów / którzy nośili efod lniáne.
Nobe też miáſto kápłáńſkie wyśiekł oſtrzem miecżá / od mężá áż do niewiáſty / od máłego áż do ſſącego / y woły / y oſły / y owce wyśiekł oſtrzem miecżá.
Uƺedł tylko ſyn jeden Achimelechá / ſyná Achitobowego ; á imię jego Abijátár ; y ućiekł do Dawidá.
Tedy oznájmił Abijátár Dawidowi / że pobił Saul kápłány Páńſkie.
Y rzekł Dawid do Abijátárá : Wiedźiáłem onegoż dniá / gdyż tám był Doeg Edomcżyk / że pewnie oznájmić miał Saulowi : Jam jeſt przycżyną śmierći wƺyſtkich duƺ domu ojcá twego.
Zoſtáńże przy mnie / nie bój śię ; bo ktoby ƺukał duƺy mojey / będźie ƺukał duƺy twojey ; ále ty będźieƺ ochroniony przy mnie.
