A gdy śię wróćił Dawid y mężowie jego do Sycelegu / był dzień trzeći / jáko Amálekitowie wtárgnęli byli ná południe / y do Sycelegu / á zburzyli Syceleg / y ſpálili go ogniem.
Y pobráli w niewolą niewiáſty / które były w nim ; od nájmniejƺego áż do wielkiego / nie zábili nikogo / ále tylko pojmáli / y odeƺli drogą ſwą.
A gdy przyƺedł Dawid y mężowie jego do miáſtá / oto / ſpálone było ogniem / á żony ich / y ſyny ich / y córki ich w niewolą zábráno.
Tedy podnióſł Dawid / y lud / który był z nim / głoſ ſwój / y płákáli / áż im śiły do płácżu nie ſtáło.
Obiedwie też żony Dawidowe były wzięte w niewolą : Achinoám Jezreelitká / y Abigáil / przedtem żoná Nábálowá z Kármelu.
Y był utrápiony Dawid bárdzo ; bo śię zmáwiáł lud ukámionowáć go / gdyż gorzkośći pełná byłá duƺá wƺyſtkiego ludu / káżdego dlá ſynów ſwych / y dlá córek ſwych ; wƺákże Dawid zmocnił śię w Pánu / Bogu ſwoim.
Tedy rzekł Dawid do Abijátárá kápłáná / ſyná Achimelechowego : Weźmij proƺę dlá mnie efod ; y wziął Abijátár efod dlá Dawidá.
A ták śię Dawid rádził Páná / mówiąc ; Mámli gonić to wojſko / y dogonięli go? A Pán mu rzekł : Goń ; bo zápewne ich dogoniƺ / y zápewne odbijeƺ plon.
Szedł tedy Dawid / ſám y one ƺeść ſet mężów / które miáł z ſobą / á przyƺli áż do potoku Beſor ; á niektórzy pozoſtáli.
Y gonił je Dawid / ſám y cżtery ſtá mężów ; bo pozoſtáło było dwieśćie mężów ſprácowánych / á nie przeƺli zá potok Beſor.
Y ználeźli mężá Egipcżániná ná polu / á przywiedli go do Dawidá / y dáli mu chlebá / y jádł ; dáli mu też wody / y pił ;
Dáli mu tákże y wiązánkę fig y dwie gronie rodzynków. Jádł tedy / y wróćił śię duch jego weń ; bo nie jádł chlebá / áni pił wody przez trzy dni y przez trzy nocy.
Y rzekł do niego Dawid : Cżyjeś ty? á ſkądeś? Który odpowiedziáł : Jeſtem rodem z Egiptu / ſługá mężá Amálekity / y zoſtáwił mię pán mój / żem śię rozniemógł dziś trzeći dzień.
Wtárgnęliśmy byli ná południe do Cerety y do Judá / y ná południe do Káleb / y ſpáliliśmy Syceleg ogniem.
Y rzekł mu Dawid : Mógłżebyś mię dowieść do tego wojſká? Który rzekł : Przyśięż mi przez Bogá / iż mię nie zábijeƺ / áni mię wydáƺ w rękę páná mego ; tedy ćię náwiodę ná to wojſko.
Náwiódł go tedy ; á oto / oni leżeli po wƺyſtkiej onej ziemi / jedząc y pijąc / y weſeląc śię ze wƺyſtkich korzyśći wielkich / które byli zábráli z ziemi Filiſtyńſkiej / y z ziemi Judzkiej.
Przetoż bił je Dawid od wiecżorá áż do wiecżorá dniá drugiego / ták iż z nich żáden nie uƺedł / oprócż cżterech ſet młodzieńców / którzy wśiádłƺy ná wielbłądy / ućiekli.
A ták odjął Dawid wƺyſtko / co byli pobráli Amálekitowie / y dwie żony ſwoje odjął też Dawid ;
Ták iż im nic nie zginęło od máłá áż do wielá / y áż do ſynów / y córek / y do korzyśći / y áż do wƺyſtkiego / cokolwiek im zábráli / wƺyſtko záśię przywiódł Dawid.
Przytem zábráł Dawid wƺyſtkie trzody y ſtádá / które gnáno przed bydłem jego / y mówiono : Toć jeſt korzyść Dawidowá.
Y przyƺedł Dawid do onych dwóch ſet mężów / którzy byli ſprácowáni / że nie mogli iść zá Dawidem / którym był kázáł zoſtáć u potoku Beſor / którzy wyƺli przećiw Dawidowi y przećiw ludowi / który z nim był ; á przyſtąpiwƺy Dawid do ludu / pozdrowił je ſpokojnie.
A odpowiádájąc wƺyſcy mężowie źli y niepobożni / którzy chodzili z Dawidem / rzekli : Poniewáż ći nie chodzili z námi / nie dámy im z łupów / któreśmy odjęli / tylko káżdemu żonę jego / y ſyny jego ; te wziąwƺy / niech odejdą.
Tedy rzekł Dawid : Nieucżynićie ták / bráćiá moi / z tem / co nám dáł Pán / który náſ ſtrzegł / á podáł wojſko / które było wyƺło przećiwko nám / w ręce náƺe.
Y któż wáſ w tem uſłuchá? Bo jáki dziáł tego / który wyƺedł ná wojnę / táki dziáł y tego / który zoſtáł przy tłomokách ; równo śię podzielą.
Y ſtáło śię od onego dniá y nápotem / że uchwálono to práwo y ten zwycżáj w Izráelu / áż do dniá tego.
A ták przyƺedł Dawid do Sycelegá / y poſłáł z onego łupu ſtárƺym w Judá / przyjáćiołom ſwym / mówiąc : Oto máćie błogoſłáwieńſtwo z korzyśći nieprzyjáćiół Páńſkich :
Tym / co byli w Betel / y co w Rámáćie ná południe / y co byli w Gieter /
Y co byli w Aroer / y co byli w Sefámot / y co byli w Eſtámo ;
Y co byli w Ráchá / y co byli w miáſtách Jerámeel / y co byli w miáſtách Ceni /
Y co byli w Hormá / y co byli w Choráſán / y co byli w Atách /
Y co byli w Hebronie / y co byli ná wƺyſtkich miejſcách / kędy przemieƺkiwáł Dawid / ſám y mężowie jego.
