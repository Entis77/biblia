Przyƺli tedy mężowie z Káryjátyjárym / y odwieźli ſkrzynię Páńſką / á wnieśli ją do domu Abinádábowego w Gábáá ; á Eleázárá ſyná jego poświęćili / áby ſtrzegł ſkrzyni Páńſkiej.
Y ſtáło śię / gdy od onego dniá / jáko zoſtáłá ſkrzyniá w Káryjátyjárym / wyƺedł niemáły cżáſ / to jeſt dwádzieśćiá lát / że płákáł wƺyſtek dom Izráelſki zá Pánem.
Y rzekł Sámuel do wƺyſtkiego domu Izráelſkiego mówiąc : Jeźliże ze wƺyſtkiego ſercá wáƺego náwrácáćie śię do Páná / wyrzuććież bogi cudze z pośrodku śiebie / y Aſtárotá / á zgotujćie ſerce wáƺe Pánu / y ſłużćie jemu ſámemu / tedyć wáſ wybáwi z ręki Filiſtynów.
Przetoż wyrzućili ſynowie Izráelſcy Báálá y Aſtárotá / á ſłużyli Pánu ſámemu.
Tedy rzekł Sámuel : Zgromádźćie wƺyſtkiego Izráelá do Máſfá / ábym śię modlił zá wámi Pánu.
A ták zgromádzili śię do Máſfá / á cżerpájąc wodę / wylewáli przed Pánem / y pośćili tám dniá onego / mówiąc : Zgrzeƺyliśmy Pánu. Y ſądził Sámuel ſyny Izráelſkie w Máſfá.
A gdy uſłyƺeli Filiſtynowie / że śię zgromádzili ſynowie Izráelſcy do Máſfá / ruƺyły śię kśiążętá Filiſyńſkie przećiw Izráelowi. Co gdy uſłyƺeli ſynowie Izráelſcy / zlękli śię przed Filiſtynámi.
Y rzekli ſynowie Izráelſcy do Sámuelá : Nie przeſtáwáj zá námi wołáć do Páná / Bogá náƺego / áby náſ wybáwił z ręki Filiſtynów.
Przetoż wziął Sámuel báránká ſſącego jednego / y ofiárowáł go cáłego ná cáłopálenie Pánu ; y wołáł Sámuel do Páná zá Izráelem / á wyſłucháł go Pán.
Y ſtáło śię / gdy Sámuel ſpráwowáł cáłopálenie / że Filiſtynowie przyćiągnęli bliſko / áby wálcżyli przećiw Izráelowi ; ále zágrzmiáł Pán grzmotem wielkim dniá onego nád Filiſtynámi / á potárł je / y poráżeni ſą przed oblicżem Izráelá.
A mężowie Izráelſcy wypádłƺy z Máſfá / gonili Filiſtyny / y bili je áż pod Betchár.
Tedy wziął Sámuel kámień jeden / y poſtáwił go między Máſfá á między Sen / y názwáł imię jego Ebenezer / mówiąc : Aż póty pomágáł nám Pán.
A ták poniżeni ſą Filiſtynowie / á potem więcej nie przychodzili ná gránicę Izráelſką ; ábowiem byłá ręká Páńſká przećiwko Filiſtynom po wƺyſtkie dni Sámuelowe.
Y przywrócone ſą miáſtá Izráelowi / które byli wzięli Filiſtynowie Izráelowi / od Akkáronu áż do Get / y gránice ich oſwobodził Izráel z ręki Filiſtynów ; y był pokój między Izráelem / y między Amorejcżykiem.
Y ſądził Sámuel Izráelá po wƺyſtkie dni żywotá ſwego.
A chodząc ná káżdy rok / obchodził Betel / y Gilgál / y Máſfá / ſądząc Izráelá po onych wƺyſtkich miejſcách.
Potem śię wrácáł do Rámáty ; bo tám był dom jego / y tám ſądził Izráelá ; támże też zbudowáł ołtárz Pánu.
