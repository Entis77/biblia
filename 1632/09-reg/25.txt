W tem umárł Sámuel. A zebráwƺy śię wƺyſcy Izráelcżycy / płákáli go / y pogrzebli go w domu jego w Rámáćie. Tedy wſtáwƺy Dawid / poƺedł ná puƺcżą Fárán.
A był niektóry mąż w Máon / który miáł májętność ná Kármelu ; á on mąż był możny bárdzo / májąc owiec trzy tyśiące / á tyśiąc kóz ; y tráfiło śię / że ſtrzygł owce ſwoje ná Kármelu.
A było imię mężá onego Nábál / á imię żony jego Abigáil / którá niewiáſtá byłá mądrá / y piękná ; ále mąż jej był nieużyty y złych poſtępków / á był národu Kálebowego.
A uſłyƺáwƺy Dawid ná puƺcży / iż Nábál ſtrzygł owce ſwoje /
Poſłáł dzieśięćiu ſług / y rzekł im : Idźćie do Kármelu / á przyƺedłƺy do Nábálá / pozdrówćie go imieniem mojem ſpokojnie /
A mówćie ták : Żyj / á niech będzie tobie pokój / y domowi twemu pokój / y wƺyſtkiemu / co máƺ / pokój!
A teráz ſłyƺáłem / że máƺ te / coć owce ſtrzygą ; á páſterze twoi bywáli z námi / niebyliśmy im przykrymi / y nic im nie zginęło po wƺyſtkie dni / których byli w Kármelu ;
Spytáj ſług twoich / á powiedząć. Przetoż niech znájdą ſłudzy łáſkę w ocżách twoich / gdyżeśmy w dobry dzień przyƺli ; dáj proƺę cokolwiek znájdzie ręká twojá / ſługom twoim / y ſynowi twemu Dawidowi.
A ták przyƺli ſłudzy Dawidowi / y mówili do Nábálá wƺyſtkie one ſłowá imieniem Dawidowem / y przeſtáli.
A odpowiádájąc Nábál ſługom Dawidowym / rzekł : Cóż jeſt Dawid? á co zácż ſyn Iſájego? dziś śię námnożyło ſług / którzy ućiekáją od pánów ſwoich.
Y wezmęż já chleb mój / y wodę moję / y mięſá bydłá mego / którem pobił dlá tych / którzy ſtrzygą owce moje / á dám mężom / których nie znám / ſkąd ſą?
A obróćiwƺy śię ſłudzy Dawidowi w drogę ſwoję / wróćili śię / y przyƺli á powiedzieli mu wƺyſtkie te ſłowá.
Tedy rzekł Dawid mężom ſwym : Przypáƺćie káżdy miecż ſwój. Y przypáſáli káżdy miecż ſwój ; przypáſáł też y Dawid miecż ſwój / y ƺło zá Dawidem około cżterech ſet mężów / á dwieśćie zoſtáło przy rzecżách.
Ale Abigáili / żonie Nábálowej / oznájmił to jeden cżeládnik z ſług Nábálowych / mówiąc : Oto przyſłáł Dawid poſły z puƺcży / áby błogoſłáwili Pánu náƺemu / lecż je on ſfukáł.
A mężowie ći dobrzy nám byli bárdzo / y nie przykrzyli śię nám ; nic nám nie zginęło po wƺyſtkie dni / pókiśmy z nimi chodzili / będąc ná polu ;
Miáſto muru byli nám / ták w nocy jáko we dnie / po wƺyſtkie dni / pókiśmy przy nich trzody páśli.
Przetoż teráz obácż / á rozmyśl śię / co máƺ cżynić ; boć już gotowe nieƺcżęśćie ná páná náƺego / y ná wƺyſtek dom jego ; lecż on jeſt cżłowiekiem niezbożnym / że z nim trudno mówić.
Poſpieƺyłá śię tedy Abigáil / y wzięłá dwieśćie chlebá / y dwie łágwi winá / y pięć owiec opráwnych / y pięć miár práżmá / y ſto wiązánek rodzynków / y dwieśćie funtów fig / á włożyłá to ná oſły ;
Y rzekłá ſługom ſwoim : Idźćie przedemną / á já pojádę zá wámi ; ále mężowi ſwemu Nábálowi nie oznájmiłá.
Y ſtáło śię / że jádąc ná ośle / y zjeżdżájąc głębią góry / oto też Dawid y mężowie jego zjeżdżáli przećiwko niej / y ſpotkáłá śię z nimi.
( A Dawid był rzekł : Zápráwdę dármom ſtrzegł wƺyſtkiego / co ten miáł ná puƺcży / że nic nie zginęło ze wƺyſtkiego / co má ; bo mi oddáł złem zá dobre.
To niech ucżyni Bóg nieprzyjáćiołom Dawidowym / y to niech przycżyni / jeźli co do zárániá zoſtáwię ze wƺyſtkiego co má / áż do nájmniejƺego ƺcżenięćiá. )
Tedy ujrzáwƺy Abigáil Dawidá / poſpieƺyłá śię / y zśiádłá z oſłá / y upádłá przed Dawidem ná oblicże ſwoje / y ukłoniłá śię áż do ziemi ;
A pádłƺy do nóg jego / mówiłá : Niech będzie ná mnie / pánie mój / tá niepráwość / á niech mówi proƺę ſłużebnicá twojá do uƺu twoich / y poſłucháj ſłów ſłużebnicy twojej.
Niech śię proƺę nie obráżá pán mój ná mężą tego bezbożnego Nábálá / gdyż według imieniá ſwego tákim jeſt. Nábál jeſt imię jego / y głupſtwo jeſt przy nim ; álemći já ſłużebnicá twojá nie widziáłá ſług páná mego / któreś był poſłáł.
Przetoż teráz pánie mój / jáko żywy Pán / jáko żywá y duƺá twojá / że ćię záwśćiągnął Pán / ábyś nie ƺedł ná rozlánie krwi / y żeby śię nie mśćiłá ręká twojá ; á teráz niech będą jáko Nábál nieprzyjáćiele twoi / y którzy ƺukáją złego pánu memu.
Teráz tedy to błogoſłáwieńſtwo / które przynioſłá ſłużebnicá twojá pánu ſwemu / niech będzie dáne ſługom / którzy chodzą zá pánem moim.
Przepuść proƺę wyſtępek ſłużebnicy twojej / gdyż zápewne ucżyni Pán pánu memu dom trwáły / poniewáż wálki Páńſkie pán mój odpráwuje / á złość nie jeſt ználezioná w tobie áż dotąd.
A choćby powſtáł cżłowiek / coby ćię prześládowáł / y ƺukáł duƺy twojej / tedy będzie duƺá páná mego záchowáná w wiązánce żywiących u Páná / Bogá twego ; lecż duƺę nieprzyjáćiół twoich Bóg jáko z procy wyrzući.
A gdy ucżyni Pán pánu memu wƺyſtko / co mówił dobrego o tobie / á zlećić / ábyś był wodzem nád Izráelem :
Tedy nie będzieć to ku záchwiániu / áni ku urázie ſercá páná mego / jáko gdyby rozláł krew niewinną / y gdyby śię ſám pomśćił pán mój. Gdy tedy dobrze ucżyni Pán pánu memu / wſpomniƺ ná ſłużebnicę twoję.
Y rzekł Dawid do Abigáili : Błogoſłáwiony Pán / Bóg Izráelſki / który ćię dziś poſłáł przećiwko mnie.
Błogoſłáwioná wymowá twojá ; y błogoſłáwionáś ty / któráś mię záwśćiągnęłá dziśiáj / żem nie ƺedł ná rozlánie krwi / á żem śię ſám nie mśćił krzywdy ſwojej.
A zápráwdę / jáko żywy Pán / Bóg Izráelſki / który mię záwśćiągnął / ábym ći nic złego nie ucżynił : bo gdybyś śię byłá nie poſpieƺyłá / á nie zájecháłá mi drogi / nie zoſtáłoby było Nábálowi áż do świtániá / y nájmniejƺego ƺcżenięćiá.
A ták przyjął Dawid z ręki jej / co mu byłá przynioſłá / y rzekł do niej : Idź w pokoju do domu twego : otom uſłucháł głoſu twego / y przyjąłem ćię łáſkáwie.
Tedy śię wróćiłá Abigáil do Nábálá ; á on miáł ucżtę w domu ſwoim / jáko ucżtę królewſką / á ſerce Nábálowe było weſołe w nim / á był pijány bárdzo ; y nie oznájmiłá mu nájmniejƺej rzecży áż do poránku.
Ale názájutrz / gdy wytrzeźwiáł Nábál z winá / oznájmiłá mu żoná jego te rzecży ; y zmártwiáło w nim ſerce jego / y ſtáł śię jáko kámień.
A gdy wyƺło jákoby dzieśięć dni / uderzył Pán Nábálá / y umárł.
A uſłyƺáwƺy Dawid / iż umárł Nábál / rzekł : Błogoſłáwiony Pán / który śię pomśćił poháńbieniá mego nád Nábálem / á ſługę ſwego zátrzymáł od złego / á złość Nábálowę obróćił Pán ná głowę jego. Tedy poſłáł Dawid wſkázując do Abigáili / że ją ſobie chce wziąć zá żonę.
Y przyƺli ſłudzy Dawidowi do Abigáili do Kármelu / y rzekli do niej / mówiąc : Dawid poſłáł náſ do ćiebie / áby ćię ſobie wziął zá żonę.
Którá wſtáwƺy / pokłoniłá śię oblicżem do ziemi / y rzekłá : Oto / ſłużebnicá twojá niech będzie ſługą / áby umywáłá nogi ſług páná mego.
Przetoż poſpieƺywƺy śię wſtáłá Abigáil / y wśiádłá ná oſłá z pięćiomá pánienkámi ſwemi / które zá nią chodziły ; y ták jecháłá zá poſłámi Dawidowymi / á byłá mu zá żonę.
Ale y Achinoámę wziął Dawid z Jezreel / y były mu te dwie zá żony.
Abowiem Saul dáł był Michol / córkę ſwoję / żonę Dawidowę / Fáltemu / ſynowi Láiſowemu / który był z Gállim.
