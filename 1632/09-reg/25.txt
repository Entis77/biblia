W tym umárł Sámuel. A zebráwƺy śię wƺyſcy Izráelcżycy / płákáli go / y pogrzebli go w domu jego w Rámáćie. Tedy wſtáwƺy Dawid / poƺedł ná puƺcżą Fárán.
A był niektóry mąż w Máon / który miał májętność ná Kármelu ; á on mąż był możny bárzo / májąc owiec trzy tyśiące / á tyśiąc kóz ; y tráfiło śię / że ſtrzygł owce ſwoje ná Kármelu.
A było imię mężá onego Nábál / á imię żony jego Abigáil / która niewiáſtá byłá mądrá / y piękná ; ále mąż jey był nieużyty y złych poſtępków / á był narodu Kálebowego.
A uſłyƺáwƺy Dawid ná puƺcży / iż Nábál ſtrzygł owce ſwoje /
Poſłał dźieśięćiu ſług / y rzekł im : Idźćie do Kármelu / á przyƺedƺy do Nábálá / pozdrówćie go imieniem mojim ſpokojnie /
A mówćie ták : Żyj / á niech będźie tobie pokój / y domowi twemu pokój / y wƺyſtkiemu / co maƺ / pokój!
A teraz ſłyƺáłem / że maƺ te / coć owce ſtrzygą ; á páſterze twoi bywáli z námi / niebyliśmy im przykrymi / y nic im nie zginęło po wƺyſtkie dni / których byli w Kármelu ;
Spytáj ſług twojich / á powiedząć. Przetoż niech znájdą ſłudzy łáſkę w ocżách twojich / gdyżeśmy w dobry dźień przyƺli ; dáj proƺę cokolwiek znájdźie ręká twojá / ſługom twoim / y ſynowi twemu Dawidowi.
A ták przyƺli ſłudzy Dawidowi / y mówili do Nábálá wƺyſtkie one ſłowá imieniem Dawidowem / y przeſtáli.
A odpowiádájąc Nábál ſługom Dawidowym / rzekł : Cóż jeſt Dawid? á co zácż ſyn Iſájego? dźiś śię námnożyło ſług / którzy ućiekáją od pánów ſwojich.
Y wezmęż ja chleb mój / y wodę moję / y mięſá bydłá mego / którem pobił dla tych / którzy ſtrzygą owce moje / á dam mężom / których nie znam / ſkąd ſą?
A obróćiwƺy śię ſłudzy Dawidowi w drogę ſwoję / wróćili śię / y przyƺli á powiedźieli mu wƺyſtkie te ſłowá.
Tedy rzekł Dawid mężom ſwym : Przypáƺćie káżdy miecż ſwój. Y przypáſáli káżdy miecż ſwój ; przypáſał też y Dawid miecż ſwój / y ƺło zá Dawidem około cżterech ſet mężów / á dwieśćie zoſtáło przy rzecżách.
Ale Abigáili / żonie Nábálowej / oznájmił to jeden cżeládnik z ſług Nábálowych / mówiąc : Oto przyſłał Dawid poſły z puƺcży / áby błogoſłáwili Pánu náƺemu / lecż je on ſfukał.
A mężowie ći dobrzy nam byli bárzo / y nie przykrzyli śię nam ; nic nam nie zginęło po wƺyſtkie dni / pókiśmy z nimi chodźili / będąc ná polu ;
Miáſto muru byli nam / ták w nocy jáko we dnie / po wƺyſtkie dni / pókiśmy przy nich trzody páśli.
Przetoż teraz obácż / á rozmyśl śię / co maƺ cżynić ; boć już gotowe nieƺcżęśćie ná páná náƺego / y ná wƺyſtek dom jego ; lecż on jeſt cżłowiekiem niezbożnym / że z nim trudno mówić.
Poſpieƺyłá śię tedy Abigáil / y wźięłá dwieśćie chlebá / y dwie łágwi winá / y pięć owiec opráwnych / y pięć miár práżmá / y ſto wiązánek rodzynków / y dwieśćie funtów fig / á włożyłá to ná oſły ;
Y rzekłá ſługom ſwoim : Idźćie przedemną / á ja pojádę zá wámi ; ále mężowi ſwemu Nábálowi nie oznájmiłá.
Y ſtáło śię / że jádąc ná ośle / y zjeżdżájąc głębią góry / oto też Dawid y mężowie jego zjeżdżáli przećiwko niej / y ſpotkáłá śię z nimi.
( A Dawid był rzekł : Zápráwdę dármom ſtrzegł wƺyſtkiego / co ten miał ná puƺcży / że nic nie zginęło ze wƺyſtkiego / co ma ; bo mi oddał złem zá dobre.
To niech ucżyni Bóg nieprzyjáćiołom Dawidowym / y to niech przycżyni / jeſli co do záránia zoſtáwię ze wƺyſtkiego co ma / áż do nájmniejƺego ƺcżenięćiá. )
Tedy ujrzáwƺy Abigáil Dawidá / poſpieƺyłá śię / y zśiádłá z oſłá / y upádłá przed Dawidem ná oblicże ſwoje / y ukłoniłá śię áż do źiemi ;
A pádƺy do nóg jego / mówiłá : Niech będźie ná mnie / pánie mój / tá niepráwość / á niech mówi proƺę ſłużebnicá twojá do uƺu twojich / y poſłucháj ſłów ſłużebnicy twojey.
Niech śię proƺę nie obráżá pán mój ná mężá tego bezbożnego Nábálá / gdyż według imienia ſwego tákim jeſt. Nábál jeſt imię jego / y głupſtwo jeſt przy nim ; álemći ja ſłużebnicá twojá nie widźiáłá ſług páná mego / któreś był poſłał.
Przetoż teraz pánie mój / jáko żywy Pán / jáko żywá y duƺá twojá / że ćię záwśćiągnął Pán / ábyś nie ƺedł ná rozlánie krwi / y żeby śię nie mśćiłá ręká twojá ; á teraz niech będą jáko Nábál nieprzyjáćiele twoi / y którzy ƺukáją złego pánu memu.
Teraz tedy to błogoſłáwieńſtwo / które przynioſłá ſłużebnicá twojá pánu ſwemu / niech będźie dáne ſługom / którzy chodzą zá pánem moim.
Przepuść proƺę wyſtępek ſłużebnicy twojey / gdyż zápewne ucżyni Pán pánu memu dom trwáły / ponieważ wálki Páńſkie pán mój odpráwuje / á złość nie jeſt ználeźioná w tobie áż dotąd.
A choćby powſtał cżłowiek / coby ćię prześládował / y ƺukał duƺy twojey / tedy będźie duƺá páná mego záchowáná w wiązánce żywiących u Páná / Bogá twego ; lecż duƺę nieprzyjáćiół twojich Bóg jáko z procy wyrzući.
A gdy ucżyni Pán pánu memu wƺyſtko / co mówił dobrego o tobie / á zlećić / ábyś był wodzem nád Izráelem :
Tedy nie będźieć to ku záchwiániu / áni ku uráźie ſercá páná mego / jáko gdyby rozlał krew niewinną / y gdyby śię ſám pomśćił pán mój. Gdy tedy dobrze ucżyni Pán pánu memu / wſpomniƺ ná ſłużebnicę twoję.
Y rzekł Dawid do Abigáili : Błogoſłáwiony Pán / Bóg Izráelſki / który ćię dźiś poſłał przećiwko mnie.
Błogoſłáwioná wymowá twojá ; y błogoſłáwionáś ty / któráś mię záwśćiągnęłá dźiśiáj / żem nie ƺedł ná rozlánie krwi / á żem śię ſám nie mśćił krzywdy ſwojey.
A zápráwdę / jáko żywy Pán / Bóg Izráelſki / który mię záwśćiągnął / ábym ći nic złego nie ucżynił : bo gdybyś śię byłá nie poſpieƺyłá / á nie zájecháłá mi drogi / nie zoſtáłoby było Nábálowi áż do świtánia / y nájmniejƺego ƺcżenięćiá.
A ták przyjął Dawid z ręki jey / co mu byłá przynioſłá / y rzekł do niej : Idź w pokoju do domu twego : otom uſłuchał głoſu twego / y przyjąłem ćię łáſkáwie.
Tedy śię wróćiłá Abigáil do Nábálá ; á on miał ucżtę w domu ſwoim / jáko ucżtę królewſką / á ſerce Nábálowe było weſołe w nim / á był pijány bárzo ; y nie oznájmiłá mu nájmniejƺej rzecży áż do poránku.
Ale názájutrz / gdy wytrzeźwiał Nábál z winá / oznájmiłá mu żoná jego te rzecży ; y zmártwiáło w nim ſerce jego / y ſtał śię jáko kámień.
A gdy wyƺło jákoby dźieśięć dni / uderzył Pán Nábálá / y umárł.
A uſłyƺáwƺy Dawid / iż umárł Nábál / rzekł : Błogoſłáwiony Pán / który śię pomśćił poháńbieniá mego nád Nábálem / á ſługę ſwego zátrzymał od złego / á złość Nábálowę obróćił Pán ná głowę jego. Tedy poſłał Dawid wſkázując do Abigáili / że ją ſobie chce wźiąć zá żonę.
Y przyƺli ſłudzy Dawidowi do Abigáili do Kármelu / y rzekli do niej / mówiąc : Dawid poſłał nas do ćiebie / áby ćię ſobie wźiął zá żonę.
Która wſtáwƺy / pokłoniłá śię oblicżem do źiemi / y rzekłá : Oto / ſłużebnicá twojá niech będźie ſługą / áby umywáłá nogi ſług páná mego.
Przetoż poſpieƺywƺy śię wſtáłá Abigáil / y wśiádłá ná oſłá z pięćiomá pánienkámi ſwemi / które zá nią chodźiły ; y ták jecháłá zá poſłámi Dawidowymi / á byłá mu zá żonę.
Ale y Achinoámę wźiął Dawid z Jezreel / y były mu te dwie zá żony.
Abowiem Saul dał był Michol / córkę ſwoję / żonę Dawidowę / Fáltemu / ſynowi Láiſowemu / który był z Gállim.
