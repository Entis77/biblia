Y ſtáło śię / gdy śię zſtárzáł Sámuel / że poſtánowił ſyny ſwe ſędziámi nád Izráelem.
A było imię ſyná jego pierworodnego Joel / á imię drugiego ſyná jego Abijá ; ći byli ſędziámi w Beerſebá.
Ale nie chodzili ſynowie jego drogámi jego ; lecż událi śię zá łákomſtwem / y bráli dáry / á wywrácáli ſąd.
Przetoż zebráli śię wƺyſcy ſtárśi Izráelſcy / y przyƺli do Sámuelá do Rámáty /
Y rzekli mu : Otoś śię ty zſtárzáł / á ſynowie twoi nie chodzą drogámi twojimi ; przetoż poſtánów nám królá / áby náſ ſądził / jáko je wƺyſtkie národy máją.
Ale śię nie podobáłá tá rzecż Sámuelowi / że mówili : Dáj nám królá / áby náſ ſądził ; przetoż modlił śię Sámuel Pánu.
Tedy rzekł Pán do Sámuelá : Uſłucháj głoſu ludu tego we wƺyſtkiem / coć powiedzą ; ábowiem nie tobą wzgárdzili / ále mną wzgárdzili / iżbym nie królowáł nád nimi.
A według wƺyſtkich ſpráw / które cżynili od onego dniá / któregom je wywiódł z Egiptu / áż do dniá tego / gdy mię opuśćili y ſłużyli bogom obcym / ták też cżynią y tobie.
Przetoż teráz uſłucháj głoſu ich / á wƺákże oświádcż śię jáko nájpilniej przed nimi / y oznájmij im práwo królá / który nád nimi má królowáć.
A ták Sámuel odnióſł wƺyſtkie ſłowá Páńſkie do ludu / który go prośił o królá /
Y mówił : Toć będzie práwo królá / który królowáć má nád wámi : Syny wáƺe bráć będzie á oſádzi nimi wozy ſwoje / y pocżyni je jezdnymi / á będą biegáć przed wozem jego ;
Pocżyni też ſobie z nich pułkowniki nád tyśiącámi / y rotmiſtrze nád pięćdzieśięćiomá ; pocżyni z nich orácże ról ſwoich / y żeńce żniwá ſwego / y te / którzyby robili rynƺtunki wojenne / y potrzeby do wozów jego.
Córki też wáƺe pobierze / by gotowáły rzecży wonne / y były kuchárkámi y piekárkámi.
Polá też wáƺe / y winnice wáƺe / y oliwnice wáƺe co nájlepƺe pobierze / á rozdá ſługom ſwoim.
Przytem z záśiewków wáƺych / y z winnic wáƺych będzie bráł dzieśięćiny / y rozdá je komornikom ſwoim / y ſługom ſwoim.
Tákże ſługi wáƺe / y dziewki wáƺe / y młodzieńce wáƺe co nájgrzecżniejƺe będzie bráł / y oſły wáƺe pobierze / y obróći do roboty ſwojej.
Z bydłá wáƺego dzieśięćinę będzie bráł / á wy będziećie niewolnikámi jego.
Y będziećie wołáć dniá onego dlá królá wáƺego / którego ſobie obierzećie / á nie wyſłuchá wáſ Pán dniá onego.
Ale nie chćiáł lud uſłucháć głoſu Sámuelowego ; owƺem mówili : Nic z tego ; ále król niech będzie nád námi /
Abyśmy byli y my / jáko y wƺyſtkie národy ; będzie náſ ſądził król náƺ / á wychodząc przed námi / będzie odpráwowáł wojny náƺe.
A wyſłuchwáƺy Sámuel wƺyſtkich ſłów ludu / odnióſł je do uƺu Páńſkich.
Y rzekł Pán do Sámuelá : Uſłucháj głoſu ich / á poſtánów nád nimi królá. Przetoż rzekł Sámuel do mężów Izráelſkich : Idźćie káżdy do miáſtá ſwego.
