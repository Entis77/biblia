Y ſtáło śię / gdy śię zſtárzał Sámuel / że poſtánowił ſyny ſwe ſędźiámi nád Izráelem.
A było imię ſyná jego pierworodnego Joel / á imię drugiego ſyná jego Abijá ; ći byli ſędźiámi w Beerſebá.
Ale nie chodźili ſynowie jego drogámi jego ; lecż událi śię zá łákomſtwem / y bráli dáry / á wywrácáli ſąd.
Przetoż zebráli śię wƺyſcy ſtárśi Izráelſcy / y przyƺli do Sámuelá do Rámáty /
Y rzekli mu : Otoś śię ty zſtárzał / á ſynowie twoi nie chodzą drogámi twojimi ; przetoż poſtánów nam królá / áby nas ſądźił / jáko je wƺyſtkie národy máją.
Ale śię nie podobáłá tá rzecż Sámuelowi / że mówili : Dáj nam królá / áby nas ſądźił ; przetoż modlił śię Sámuel Pánu.
Tedy rzekł Pán do Sámuelá : Uſłucháj głoſu ludu tego we wƺyſtkiem / coć powiedzą ; ábowiem nie tobą wzgárdźili / ále mną wzgárdźili / iżbym nie królował nád nimi.
A według wƺyſtkich ſpráw / które cżynili od onego dniá / któregom je wywiódł z Egiptu / áż do dniá tego / gdy mię opuśćili y ſłużyli bogom obcym / ták też cżynią y tobie.
Przetoż teraz uſłucháj głoſu ich / á wƺákże oświádcż śię jáko nájpilniej przed nimi / y oznájmij im práwo królá / który nád nimi ma królowáć.
A ták Sámuel odnióſł wƺyſtkie ſłowá Páńſkie do ludu / który go prośił o królá /
Y mówił : Toć będźie práwo królá / który królowáć ma nád wámi : Syny wáƺe bráć będźie á oſádźi nimi wozy ſwoje / y pocżyni je jezdnymi / á będą biegáć przed wozem jego ;
Pocżyni też ſobie z nich pułkowniki nád tyśiącámi / y rotmiſtrze nád pięćdźieśięćiomá ; pocżyni z nich orácże ról ſwojich / y żeńce żniwá ſwego / y te / którzyby robili rynƺtunki wojenne / y potrzeby do wozów jego.
Córki też wáƺe pobierze / by gotowáły rzecży wonne / y były kuchárkámi y piekárkámi.
Polá też wáƺe / y winnice wáƺe / y oliwnice wáƺe co nájlepƺe pobierze / á rozdá ſługom ſwoim.
Przytem z záśiewków wáƺych / y z winnic wáƺych będźie brał dźieśięćiny / y rozdá je komornikom ſwoim / y ſługom ſwoim.
Tákże ſługi wáƺe / y dźiewki wáƺe / y młodźieńce wáƺe co nájgrzecżniejƺe będźie brał / y oſły wáƺe pobierze / y obróći do roboty ſwojey.
Z bydłá wáƺego dźieśięćinę będźie brał / á wy będźiećie niewolnikámi jego.
Y będźiećie wołáć dniá onego dla królá wáƺego / którego ſobie obierzećie / á nie wyſłuchá was Pán dniá onego.
Ale nie chćiał lud uſłucháć głoſu Sámuelowego ; owƺem mówili : Nic z tego ; ále król niech będźie nád námi /
Abyſmy byli y my / jáko y wƺyſtkie národy ; będźie nas ſądźił król náƺ / á wychodząc przed námi / będźie odpráwował wojny náƺe.
A wyſłuchwáƺy Sámuel wƺyſtkich ſłów ludu / odnióſł je do uƺu Páńſkich.
Y rzekł Pán do Sámuelá : Uſłucháj głoſu ich / á poſtánów nád nimi królá. Przetoż rzekł Sámuel do mężów Izráelſkich : Idźćie káżdy do miáſtá ſwego.
