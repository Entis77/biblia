Y ſtáło śię według mowy Sámuelowej wƺyſtkiemu Izráelowi. Bo gdy wyćiągnął Izráel przećiw Filiſtynom ná wojnę / á położyli śię obozem u Ebenezer / Filiſtynowie záś położyli śię obozem w Afeku ;
Y gdy śię uƺykowáli Filiſtynowie przećiwko Izráelowi / á ſtocżyłá śię bitwá : tedy poráżony jeſt Izráel od Filiſtynów / á pobito ich w onej bitwie ná polu około cżterech tyśięcy mężów.
Y wróćił śię lud do obozu. Y rzekli ſtárśi Izráelſcy : Przecżże náſ dźiś poráźił Pán przed Filiſtynámi? weźmijmyż do śiebie z Sylo ſkrzynię przymierzá Páńſkiego / á niech przyjdźie między náſ / á wybáwi náſ z rąk nieprzyjáćiół náƺych.
Przetoż poſłáł lud do Sylo / y wźięli ſtámtąd ſkrzynię przymierzá Páná záſtępów / śiedzącego ná Cherubinách ; byli też tám dwáj ſynowie Heli z ſkrzynią przymierzá Páńſkiego / Ofni y Fineeſ.
A gdy przyƺłá ſkrzyniá przymierzá Páńſkiego do obozu / zákrzyknął wƺyſtek Izráel głoſem wielkim / ták iż źiemiá zábrzmiáłá.
A uſłyƺáwƺy Filiſtynowie głoſ onego krzyku / rzekli : Cóż to zá głoſ ták wielkiego wykrzykániá w oboźie Hebrejſkim? Y poználi / że ſkrzyniá Páńſká przyƺłá do obozu.
Przetoż zlękli śię Filiſtynowie / gdyż mówiono : Przyƺedł Bóg do obozu ich / y rzekli : Biádá nám! bo nie było nic tákowego przedtem.
Biádáż nám! któż náſ wybáwi z rąk tych bogów mocnych? ćić to bogowie / którzy poráźili Egipt wƺelką plágą ná puƺcży.
Zmácniájćież śię / á bądźćie mężámi / o Filiſtynowie! byśćie ſnáć nie ſłużyli Hebrejcżykom / jáko oni wám ſłużyli. Bądźćież tedy mężámi / á potykájćie śię.
Zwiedli tedy bitwę Filiſtynowie / y poráżony jeſt Izráel / á ućiekáł káżdy do námiotu ſwego ; y ſtáłá śię poráżká bárdzo wielká / ták iż poległo z Izráelá trzydźieśći tyśięcy piechoty.
Támże ſkrzyniá Bożá wźiętá jeſt / y dwáj ſynowie Heli polegli / Ofni y Fineeſ.
Y bieżáł niektóry z ſynów Benjáminowych z bitwy / á przyƺedł do Sylo tegoż dniá / májąc ƺáty rozdárte / á proch ná głowie ſwojej.
A gdy przyƺedł / oto / Heli śiedźiáł ná ſtołku przy drodze wyglądájąc / bo śię ſerce jego lękáło o ſkrzynię Bożą ; á przyƺedłƺy on mąż / opowiedzáł miáſtu / y krzycżáło wƺyſytko miáſto.
A uſłyƺáwƺy Heli głoſ krzyku onego / rzekł : Cóż to zá głoſ rozruchu tego? lecż on mąż ſpieƺąc śię / przyƺedł / áby to oznájmił Heliemu.
A Heli już miáł dźiewiędźieśiąt y ośm lát / á ocży jego już śię były záćmiły / że nie mógł dojrzeć.
Tedy rzekł on mąż do Heliego : Já idę z bitwy / jám záiſte z bitwy dźiś ućiekł. Do którego on rzekł : Cóż śię tám ſtáło / ſynu mój?
Y odpowiedzáł on poſeł / y rzekł : Ućiekł Izráel przed Filiſtynámi y ſtáłá śię wielká poráżká ludu ; támże y dwáj ſynowie twoi legli / Ofni y Fineeſ / y ſkrzyniá Bożá wźiętá jeſt.
A gdy wſpomniáł ſkrzynię Bożą / ſpádł Heli z ſtołká ná wznák u brámy / á złámáwƺy ſobie ƺyję umárł ; ábowiem był cżłowiek ſtáry y oćiężáły. A on ſądźił Izráelá przez cżterdźieśći lát.
Synowá też jego / żoná Fineeſowá / będąc brzemienną y bliſką porodzeniá / gdy uſłyƺáłá oną nowinę / iż wźiętá jeſt ſkrzyniá Bożá / y że umárł świekier jej / y mąż jej / tedy śię náchyliłá / y porodźiłá ; bo przypádły ná nię bole jej.
A gdy umieráłá / rzekły niewiáſty / które były przy niej : Nie bój śię / ábowiemeś ſyná porodźiłá ; ále oná nic nie odpowiedźiáłá / áni tego przypuśćiłá do ſercá ſwego.
Y názwáłá dźiećiątko Ichábod / mówiąc : Przeprowádźiłá śię ſłáwá od Izráelá / iż wźięto ſkrzynię Bożą / á iż umárł świekier jej / y mąż jej.
Prztoż rzekłá : Przeprowádźiłá śię ſłáwá od Izráelá ; bo wźięto ſkrzynię Bożą.
