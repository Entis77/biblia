Y przyƺli Zyfejcżycy do Saulá do Gábáá / á mówili : Izali śię nie kryje Dawid ná págórku Háchilá / przećiw Jeſymon?
Ruƺył śię tedy Saul / y ćiągnął ná puƺcżą Zyf / á z nim trzy tyśiące mężów przebránych z Izráelá / áby ƺukał Dawidá ná puƺcży Zyf.
Y położył śię Saul obozem ná págórku Háchilá / które jeſt przećiw Jeſymon podle drogi ; á Dawid mieƺkał ná puƺcży / y dowiedźiał śię / że przyćiągnął Saul zá nim ná puƺcżą.
Bo poſłáwƺy Dawid ƺpiegi dowiedźiał śię / że przyćiągnął Saul zápewne.
Przetoż wſtał Dawid / y przyƺedł áż ku miejſcu / gdźie śię położył obozem Saul ; y upátrzył Dawid miejſce / gdźie ſpał Saul / y Abner / ſyn Nerá / hetmán wojſká jego ; bo Saul ſpał w oboźie / á lud leżał około niego.
Y odpowiedźiał Dawid / á rzekł do Achimelechá Hetejcżyká / y do Abiſájego / ſyná Sárwii / brátá Joábowego / mówiąc : Któż pójdźie ze mną do Saulá do obozu? Odpowiedźiał Abiſáj : Ja z tobą pójdę.
A ták przyƺedł Dawid y Abiſáj do ludu w nocy / á oto / Saul leżąc ſpał w oboźie / á włócżniá jego byłá utknioná w źiemi u głowy jego ; Abner też z ludem leżeli około niego.
Tedy rzekł Abiſáj do Dawidá : Zámknął dźiś Bóg nieprzyjáćielá twego w ręce twojey / á teraz niech go przebiję proƺę włócżnią ku źiemi raz / á więcey nie powtórzę.
Ale rzekł Dawid do Abiſájego : Nie zábijáj go ; bo któż śćiągnąwƺy rękę ſwą ná pomázáńcá Páńſkiego / niewinnym będźie?
Nádto rzekł Dawid : Jáko żyje Pán / że jeſli go Pán niezábije / álbo dźień jego nie przydźie / áby umárł / álbo ná wojnę wyjecháwƺy / nie zginie /
Tedy uchowáj mię Pánie / ábym miał śćiągnąć rękę moję ná pomázáńcá Páńſkiego ; ále weźmij proƺę włócżnią / która jeſt u głów jego / y kubek od wody / á odejdźmy.
Tedy wźiął Dawid włócżnią / y kubek od wody / który był u głów Saulowych / y odeƺli / á nie był / ktoby widźiał / áni ktoby wiedźiał / áni ktoby śię ocućił / ále wƺyſcy ſpáli ; bo ſen twárdy od Páná przypádł był ná nie.
Y przyƺedł Dawid ná drugą ſtronę / y ſtánął ná wierzchu góry z dáleká / á był plác wielki miedzy nimi.
Y záwołał Dawid ná lud / y ná Abnerá / ſyná Nerowego / mówiąc : Nie ozwieƺże śię Abnerze? Y odpowiádájąc Abner / rzekł : Któżeś ty / co wołáƺ ná królá?
Y rzekł Dawid do Abnerá : Azaż ty nie mąż? A któż jákoś ty w Izráelu? przecżżeś tedy nie ſtrzegł królá / páná twego? bo przyƺedł jeden z ludu / chcąc zábić królá / páná twego.
Nie dobrá to / coś ucżynił. Jáko żywy Pán / żeśćie winni śmierći / którzyśćie nie ſtrzegli páná wáƺego / pomázáńcá Páńſkiego. A teraz / pátrz / kędy jeſt włócżniá królewſká / y kubek od wody / co był w głowách jego?
Poznał tedy Saul głos Dawidá / y rzekł : Twójże to głos / ſynu mój Dawidźie? Odpowiedźiał Dawid : Głos to mój / królu / pánie mój.
Nádto rzekł : Cżemuż pán mój prześláduje ſługę ſwego? bo cóżem ucżynił? á co jeſt złego w ręce mojey?
Przetoż teraz niech poſłuchá proƺę król / pán mój / ſłów ſługi ſwego ; jeſli ćię Pán pobudźił przećiwko mnie / niech powoniá ofiáry ; ále jeſli ſynowie Judzcy / przeklęći ſą przed Pánem / którzy mię dźiś wygnáli / ábym nie mieƺkał w dźiedźictwie Páńſkim / jákoby rzekli : Idź / ſłuż bogom cudzym.
A teraz niech nie będźie wyláná krew mojá ná źiemię przed oblicżem Páńſkim ; bo wyƺedł król Izráelſki ƺukáć pchły jedney / jákoby też kto gonił kuropátwę po górách.
Tedy rzekł Saul : Zgrzeƺyłem. Wróćże śię / ſynu mój Dawidźie / boć już nic złego nie ucżynię więcey / poniewáż drogá byłá duƺá mojá w ocżách twojich dniá tego ; otom głupio ucżynił / y zbłądźiłem náder bárzo.
A odpowiádájąc Dawid rzekł : Oto włócżniá królewſká ; niech ſám przydźie kto z ſług / á weźmie ją.
A Pán niech oddá káżdemu ſpráwiedliwość jego / y wiárę jego. Abowiem podał ćię był Pán dźiś w ręce moje ; álem nie chćiał śćiągnąć ręki mojey ná pomázáńcá Páńſkiego.
Przetoż jáko dźiś powáżoná byłá duƺá twojá w ocżách mojich / ták niech będźie powáżoná duƺá mojá w ocżách Páńſkich / á niech mię wyrwie Pán ze wƺego ućiſku.
Y rzekł Saul do Dawidá : Błogoſłáwionyś ty / ſynu mój Dawidźie ; ták cżyniąc dokáżeƺ / á ták śię zmácniájąc / mocnym będźieƺ. Odƺedł potem Dawid w drogę ſwą / á Saul śię wróćił ná miejſce ſwoje.
