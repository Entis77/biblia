Y przyƺli Zyfejcżycy do Saulá do Gábáá / á mówili : Izáli śię nie kryje Dawid ná págórku Háchilá / przećiw Jeſymon?
Ruƺył śię tedy Saul / y ćiągnął ná puƺcżą Zyf / á z nim trzy tyśiące mężów przebránych z Izráelá / áby ƺukáł Dawidá ná puƺcży Zyf.
Y położył śię Saul obozem ná págórku Háchilá / które jeſt przećiw Jeſymon podle drogi ; á Dawid mieƺkáł ná puƺcży / y dowiedźiáł śię / że przyćiągnął Saul zá nim ná puƺcżą.
Bo poſłáwƺy Dawid ƺpiegi dowiedźiáł śię / że przyćiągnął Saul zápewne.
Przetoż wſtáł Dawid / y przyƺedł áż ku miejſcu / gdźie śię położył obozem Saul ; y upátrzył Dawid miejſce / gdźie ſpáł Saul / y Abner / ſyn Nerá / hetmán wojſká jego ; bo Saul ſpáł w oboźie / á lud leżáł około niego.
Y odpowiedźiáł Dawid / á rzekł do Achimelechá Hetejcżyká / y do Abiſájego / ſyná Sárwii / brátá Joábowego / mówiąc : Któż pójdźie ze mną do Saulá do obozu? Odpowiedźiáł Abiſáj : Já z tobą pójdę.
A ták przyƺedł Dawid y Abiſáj do ludu w nocy / á oto / Saul leżąc ſpáł w oboźie / á włócżniá jego byłá utknioná w źiemi u głowy jego ; Abner też z ludem leżeli około niego.
Tedy rzekł Abiſáj do Dawidá : Zámknął dźiś Bóg nieprzyjáćielá twego w ręce twojej / á teráz niech go przebiję proƺę włócżnią ku źiemi ráz / á więcej nie powtórzę.
Ale rzekł Dawid do Abiſájego : Nie zábijáj go ; bo któż śćiągnąwƺy rękę ſwą ná pomázáńcá Páńſkiego / niewinnym będźie?
Nádto rzekł Dawid : Jáko żyje Pán / że jeźli go Pán niezábije / álbo dźień jego nie przyjdźie / áby umárł / álbo ná wojnę wyjecháwƺy / nie zginie /
Tedy uchowáj mię Pánie / ábym miáł śćiągnąć rękę moję ná pomázáńcá Páńſkiego ; ále weźmij proƺę włócżnią / którá jeſt u głów jego / y kubek od wody / á odejdźmy.
Tedy wźiął Dawid włócżnią / y kubek od wody / który był u głów Saulowych / y odeƺli / á nie był / ktoby widźiáł / áni ktoby wiedźiáł / áni ktoby śię ocućił / ále wƺyſcy ſpáli ; bo ſen twárdy od Páná przypádł był ná nie.
Y przyƺedł Dawid ná drugą ſtronę / y ſtánął ná wierzchu góry z dáleká / á był plác wielki między nimi.
Y záwołáł Dawid ná lud / y ná Abnerá / ſyná Nerowego / mówiąc : Nie ozwieƺże śię Abnerze? Y odpowiádájąc Abner / rzekł : Któżeś ty / co wołáƺ ná królá?
Y rzekł Dawid do Abnerá : Azáż ty nie mąż? A któż jákoś ty w Izráelu? przecżżeś tedy nie ſtrzegł królá / páná twego? bo przyƺedł jeden z ludu / chcąc zábić królá / páná twego.
Nie dobrá to / coś ucżynił. Jáko żywy Pán / żeśćie winni śmierći / którzyśćie nie ſtrzegli páná wáƺego / pomázáńcá Páńſkiego. A teráz / pátrz / kędy jeſt włócżniá królewſká / y kubek od wody / co był w głowách jego?
Poznáł tedy Saul głoſ Dawidá / y rzekł : Twójże to głoſ / ſynu mój Dawidźie? Odpowiedźiáł Dawid : Głoſ to mój / królu / pánie mój.
Nádto rzekł : Cżemuż pán mój prześláduje ſługę ſwego? bo cóżem ucżynił? á co jeſt złego w ręce mojej?
Przetoż teráz niech poſłuchá proƺę król / pán mój / ſłów ſługi ſwego ; jeźli ćię Pán pobudźił przećiwko mnie / niech powoniá ofiáry ; ále jeźli ſynowie Judzcy / przeklęći ſą przed Pánem / którzy mię dźiś wygnáli / ábym nie mieƺkáł w dźiedźictwie Páńſkim / jákoby rzekli : Idź / ſłuż bogom cudzym.
A teráz niech nie będźie wyláná krew mojá ná źiemię przed oblicżem Páńſkim ; bo wyƺedł król Izráelſki ƺukáć pchły jednej / jákoby też kto gonił kuropátwę po górách.
Tedy rzekł Saul : Zgrzeƺyłem. Wróćże śię / ſynu mój Dawidźie / boć już nic złego nie ucżynię więcej / poniewáż drogá byłá duƺá mojá w ocżách twoich dniá tego ; otom głupio ucżynił / y zbłądźiłem náder bárdzo.
A odpowiádájąc Dawid rzekł : Oto włócżniá królewſká ; niech ſám przyjdźie kto z ſług / á weźmie ją.
A Pán niech oddá káżdemu ſpráwiedliwość jego / y wiárę jego. Abowiem podáł ćię był Pán dźiś w ręce moje ; álem nie chćiáł śćiągnąć ręki mojej ná pomázáńcá Páńſkiego.
Przetoż jáko dźiś powáżoná byłá duƺá twojá w ocżách moich / ták niech będźie powáżoná duƺá mojá w ocżách Páńſkich / á niech mię wyrwie Pán ze wƺego ućiſku.
Y rzekł Saul do Dawidá : Błogoſłáwionyś ty / ſynu mój Dawidźie ; ták cżyniąc dokáżeƺ / á ták śię zmácniájąc / mocnym będźieƺ. Odƺedł potem Dawid w drogę ſwą / á Saul śię wróćił ná miejſce ſwoje.
