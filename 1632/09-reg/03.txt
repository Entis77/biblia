A pácholę Sámuel ſłużył Pánu przed Heli / á ſłowo Páńſkie było drogie w one dni / bo nie bywáło widzeniá jáwnego.
Y ſtáło śię dniá onego / gdy Heli leżał ná miejſcu ſwojim / ( á ocży jego już śię były pocżęły záćiemniáć / y nie mógł dojrzeć. )
A lámpá Boża jeƺcże nie byłá zágáƺoná / Sámuel też ſpał w kośćiele Páńſkim / gdźie byłá ſkrzyniá Boża /
Że záwołał Pán ná Sámuelá / á on śię ozwał : Owom ja.
Y przybieżał do Heliego y rzekł : Owom ja / gdyżeś mię wołał. A on rzekł : Nie wołáłem / wróć śię / śpij ; y poƺedł á ſpał.
Powtóre Pán jeƺcże záwołał Sámuelá ; y wſtał Sámuel / á poƺedł do Heliego / y rzekł : Owom ja / gdyżeś mię wołał ; któremu on rzekł : nie wołáłem / ſynu mój / wróć śię á śpij.
A Sámuel jeƺcże nie znał Páná / y jeƺcże mu nie było objáwione ſłowo Páńſkie.
Nádto jeƺcże Pán záwołał Sámuelá po trzećie ; á on wſtáwƺy ƺedł do Heliego y rzekł : Owom ja / gdyżeś mię wołał. Tedy zrozumiał Heli / że Pán wołał pácholęćiá.
Y rzekł Heli do Sámuelá : Idź / śpij / á jeſli ćię kto záwołá / rzecżeƺ : Mów Pánie / bo ſłyƺy ſługá twój. A ták Sámuel ƺedł y ſpał ná miejſcu ſwojim.
Potem przyƺedł Pán / y ſtánął á záwołał jáko y pierwƺy y drugi raz : Sámuelu / Sámuelu! Y rzekł Sámuel : Mów Pánie / bo ſługá twój ſłuchá.
Tedy rzekł Pán do Sámuelá : Oto / Ja ucżynię rzecż w Izráelu / którą ktokolwiek uſłyƺy / zábrzmi mu w obu uƺách jego.
Dniá onego wzbudzę przećiw Heliemu wƺyſtko / com mówił przećiwko domowi jego ; pocżnę y dokonam.
Y okáżę mu / iż Ja ſądzę dom jego áż ná wieki dla niepráwośći / o którey wiedźiał ; bo wiedząc / że ná śię przeklęſtwo przywodźili ſynowie jego / wƺákże nie bronił im tego.
A przetoż przyśiągłem domowi Heli / że nie będźie ocżyƺcżoná niepráwość domu Heliego żádną ofiárą / áni ofiárą śniedną / áż ná wieki.
Y ſpał Sámuel áż do poránku / y otworzył drzwi domu Páńſkiego. A Sámuel bał śię oznájmić widzeniá tego Heliemu.
Tedy záwołał Heli Sámuelá / y rzekł : Sámuelu / ſynu mój ; który odpowiedźiał : Otom ja.
Y rzekł : Cóż to zá ſłowá / któreć Pán powiedźiał? proƺę / nie táj przedemną ; to á toć Bóg niecháj ucżyni / jeſliże co zátáiƺ przedemną ze wƺyſtkich ſłów / które mówił do ćiebie.
Y oznájmił mu Sámuel wƺyſtkie ſłowá / á nie zátáił nic przed nim. A on rzekł : Pán jeſt ; co dobrego w ocżách jego / niech cżyni.
Y róſł Sámuel / á Pán był z nim / y nie dopuśćił upáść żádnemu ze wƺyſtkich ſłów jego ná źiemię.
Poznał tedy wƺyſtek Izráel od Dán áż do Beerſebá / iż Sámuel był wiernym prorokiem Pánu.
Bo y nápotem ukázywał śię Pán Sámuelowi w Sylo / ták jáko mu śię przed tym objáwił Pán w Sylo przez ſłowo ſwoje.
