A pácholę Sámuel ſłużył Pánu przed Heli / á ſłowo Páńſkie było drogie w one dni / bo nie bywáło widzeniá jáwnego.
Y ſtáło śię dniá onego / gdy Heli leżáł ná miejſcu ſwojim / ( á ocży jego już śię były pocżęły záćiemniáć / y nie mógł dojrzeć. )
A lámpá Bożá jeƺcże nie byłá zágáƺoná / Sámuel też ſpáł w kośćiele Páńſkim / gdzie byłá ſkrzyniá Bożá /
Że záwołáł Pán ná Sámuelá / á on śię ozwáł : Owom já.
Y przybieżáł do Heliego y rzekł : Owom já / gdyżeś mię wołáł. A on rzekł : Nie wołáłem / wróć śię / śpij ; y poƺedł á ſpáł.
Powtóre Pán jeƺcże záwołáł Sámuelá ; y wſtáł Sámuel / á poƺedł do Heliego / y rzekł : Owom já / gdyżeś mię wołáł ; któremu on rzekł : nie wołáłem / ſynu mój / wróć śię á śpij.
A Sámuel jeƺcże nie znáł Páná / y jeƺcże mu nie było objáwione ſłowo Páńſkie.
Nádto jeƺcże Pán záwołáł Sámuelá po trzećie ; á on wſtáwƺy ƺedł do Heliego y rzekł : Owom já / gdyżeś mię wołáł. Tedy zrozumiáł Heli / że Pán wołáł pácholęćiá.
Y rzekł Heli do Sámuelá : Idź / śpij / á jeźli ćię kto záwołá / rzecżeƺ : Mów Pánie / bo ſłyƺy ſługá twój. A ták Sámuel ƺedł y ſpáł ná miejſcu ſwojim.
Potem przyƺedł Pán / y ſtánął á záwołáł jáko y pierwƺy y drugi ráz : Sámuelu / Sámuelu! Y rzekł Sámuel : Mów Pánie / bo ſługá twój ſłuchá.
Tedy rzekł Pán do Sámuelá : Oto / Já ucżynię rzecż w Izráelu / którą ktokolwiek uſłyƺy / zábrzmi mu w obu uƺách jego.
Dniá onego wzbudzę przećiw Heliemu wƺyſtko / com mówił przećiwko domowi jego ; pocżnę y dokonám.
Y okáżę mu / iż Já ſądzę dom jego áż ná wieki dlá niepráwośći / o której wiedziáł ; bo wiedząc / że ná śię przeklęſtwo przywodzili ſynowie jego / wƺákże nie bronił im tego.
A przetoż przyśiągłem domowi Heli / że nie będzie ocżyƺcżoná niepráwość domu Heliego żádną ofiárą / áni ofiárą śniedną / áż ná wieki.
Y ſpáł Sámuel áż do poránku / y otworzył drzwi domu Páńſkiego. A Sámuel báł śię oznájmić widzeniá tego Heliemu.
Tedy záwołáł Heli Sámuelá / y rzekł : Sámuelu / ſynu mój ; który odpowiedziáł : Otom já.
Y rzekł : Cóż to zá ſłowá / któreć Pán powiedziáł? proƺę / nie táj przedemną ; to á toć Bóg niecháj ucżyni / jeźliże co zátáiƺ przedemną ze wƺyſtkich ſłów / które mówił do ćiebie.
Y oznájmił mu Sámuel wƺyſtkie ſłowá / á nie zátáił nic przed nim. A on rzekł : Pán jeſt ; co dobrego w ocżách jego / niech cżyni.
Y róſł Sámuel / á Pán był z nim / y nie dopuśćił upáść żádnemu ze wƺyſtkich ſłów jego ná ziemię.
Poznáł tedy wƺyſtek Izráel od Dán áż do Beerſebá / iż Sámuel był wiernym prorokiem Pánu.
Bo y nápotem ukázywáł śię Pán Sámuelowi w Sylo / ták jáko mu śię przedtem objáwił Pán w Sylo przez ſłowo ſwoje.
