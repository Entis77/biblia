Tedy śię modliłá Anná / y rzekłá : Rozweſeliło śię ſerce moje w Pánu / wywyżƺon jeſt róg mój w Pánu / rozƺerzyły śię uſtá moje przećiw nieprzyjáćiołom moim ; ábowiem śię rozrádowáłá w zbáwieniu twojim.
Niemáƺći świętego jáko Pán ; bo niemáƺ innego oprócż ćiebie / y niemáƺ ták mocnego / jáko Bóg náƺ.
Nie mówćież nápotem ſłów pyƺnych / á niech nie wychodzą ſłowá hárde z uſt wáƺych ; ábowiem Bóg jeſt umiejętnośći Pánem / á nádáwáją śię ſpráwy jego.
Łuk y mocárze pokruƺeni ſą / á mdli przepáſáni ſą mocą.
Którzy byli náſyceni / nájmują śię zá chleb / á głodni przeſtáli łáknąć ; ták iż niepłodná śiedmioro porodźiłá / á która rodźiłá wiele dźiátek / zemdláłá.
Pán zábijá y ożywiá / wwodźi do grobu y wywodźi.
Pán ubogiego cżyni y zbogácá / uniżá y wywyżƺá.
Wzbudzá z prochu ubogiego / á z gnoju podnośi żebráká / áby je poſádźił z kśiążęty / á dał im ſtolicę chwálebną ośiádáć ; ábowiem Páńſkie ſą grunty źiemi / á ná nich záłożył świát.
Nóg świętych ſwojich ochránia / á niepobożni w ćiemnośćiách zámilkną ; bo nie w śile ſwojey będźie śię mąż zmácniał.
Pán pokruƺy przećiwniki ſwoje / á zágrzmi ná nie z niebá ; Pán będźie ſądźił gránice źiemi / á da moc królowi ſwemu / y wywyżƺy róg pomázáńcá ſwego.
A ták odƺedł Elkáná do Rámáty do domu ſwego / á dźiećię ſłużyło Pánu przed Heli kápłánem.
Ale ſynowie Heli byli ſynowie bezbożni / á nie ználi Páná.
Abowiem obycżaj kápłánów ten był około ludu : ktokolwiek ſpráwował ofiáry / przychodźił ſługá kápłáńſki / gdy wárzono mięſo / májąc widełki o trzech zębách w ręce ſwojey.
Y wráżał je w ſtátek / álbo w koćieł / álbo w pánew / álbo w gárniec / á coklowiek wyjął widełkámi / to ſobie brał kápłán. Ták cżynili wƺyſtkim Izráelcżykom / którzy tám do Sylo przychodźili.
Tákże pierwej niż zápálono tłuſtość / tedy przychodźił ſługá kápłáńſki / á mówił do cżłowieká ofiárującego : Oddáj mięſo / ábym je upiekł kápłánowi ; ábowiem nie weźmie od ćiebie mięſá wárzonego / jedno ſurowe.
A jeſliż mu odpowiedźiał on cżłowiek : Niech śię pierwej ſpáli tłuſtość / potem ſobie weźmieƺ / cżego będźie żądáłá duƺá twojá / tedy on mówił : Nic z tego ; teraz dáj! á nie dáƺli / wezmę gwáłtem.
Y był to grzech onych ſług bárzo wielki przed Pánem ; bo śię odtrącáli ludźie od ofiár Páńſkich.
Ale Sámuel ſłużył przed Pánem / ubráne chłopiątko w efod lniány.
A mátká jego ucżyniwƺy mu ſukienkę máłą / przynáƺáłá mu co rok / gdy chádzáłá z mężem ſwym ſpráwowáć ofiárę urocżyſtą.
Y błogoſłáwił Heli Elkánie / y żonie jego / mówiąc : Niech ći da Pán potomſtwo z tey niewiáſty zá oddánego / którego wyprośiłá u Páná. Y poƺli ná miejſcá ſwoje.
Tedy náwiedźił Pán Annę / która pocżęłá y porodźiłá trzech ſynów / y dwie córek ; á pácholę Sámuel uroſł przed Pánem.
Ale Heli zſtárzał śię był bárzo / y ſłyƺał wƺyſtko / co cżynili ſynowie jego cáłemu Izráelowi / y jáko ſypiáli z niewiáſtámi / które śię ſchádzáły przede drzwi namiotu zgromádzenia.
Y rzekł do nich : Przecżże tákie rzecży cżynićie? Cóż ja ſłyƺę o wáƺych złych ſpráwách od wƺyſtkiego ludu?
Nie ták ſynowie moi ; bo nie dobrá ſłáwá / którą ja ſłyƺę / że przywodźićie ku przeſtępſtwu lud Páńſki.
Gdy kto zgrzeƺy przećiw cżłowiekowi / ſądźić go będźie ſędźiá / ále jeſli przećiw Pánu kto zgrzeƺy / któż śię zá nim ujmie? Lecż nie uſłucháli głoſu ojcá ſwego ; bo je chćiał Pán pobić.
Ale pácholę Sámuel poſtępował á róſł / y podobał śię ták Pánu jáko y ludźiom.
Potem przyƺedł mąż Boży do Heli / y rzekł mu : Ták mówi Pán : Azálim śię nie jáwnie objáwił domowi ojcá twego / gdy byli w Egipćie w domu Fáráonowym?
Y obráłem go ſobie ze wƺyſtkich pokoleń Izráelſkich zá kápłáná / áby ofiárował ná ołtárzu moim / á kádźił rzecżámi wonnemi / y nośił efod przedemną / y dáłem domowi ojcá twego wƺyſtkie ofiáry pálone od ſynów Izráelſkich.
Przecżżeśćie podeptáli ofiárę moję / y śniedną ofiárę moję / którąm rozkázał ſpráwowáć w przybytku? y więcejś ucżćił ſyny ſwoje nád mię / ábyśćie śię utucżyli z pierwoćin wƺyſtkich ofiár śniednych Izráelá / ludu mego?
Prztoż mówi Pán / Bóg Izráelſki : Rzekłem wpráwdźie : Dom twój y dom ojcá twego będźie ſłużył przedemną áż ná wieki ; ále teraz mówi Pán : Nie będźieć to / gdyż ja te / którzy mnie cżcżą / cżćić będę / á którzy mną gárdzą / będą wzgárdzeni.
Oto / dni przychodzą ; á odetnę rámię twe / y rámię domu ojcá twego / áby nie było ſtárcá w domu twoim ;
Y oglądáƺ wielki ućiſk przybytku Páńſkiego / miáſto ƺcżęśćiá / które Pán dáwał Izráelowi / y nie będźie ſtárcá w domu twoim po wƺyſtkie dni.
Wƺákże mężá nie wytrácę z ćiebie do końcá od ołtárzá mego / ábym utrápił ocży twe / á boleśćią śćiſnął duƺę twoję ; á wƺyſtko mnóſtwo domu twego pomrze / doroſƺy lat męſkich.
A toć będźie ná znák / co przydźie ná dwu ſynów twojich / Ofni y Fineeſá ; dniá jednego pomrą ći obá.
Y wzbudzę ſobie kápłáná wiernego / który według ſercá mego / y według myśli mojey cżynić będźie / y zbuduję mu dom trwáły / á będźie ſłużył przed pomázáńcem moim po wƺyſtkie dni.
Y ſtánie śię / ktokolwiek pozoſtánie z domu twego / przydźie / áby mu śię ukłonił zá pieniądz ſrebrny y zá ƺtukę chlebá / mówiąc : Przypuść mię proƺę do jedney cżąſtki kápłáńſkiej / ábym jádł ƺtucżkę chlebá.
