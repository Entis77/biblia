Tedy śię modliłá Anná / y rzekłá : Rozweſeliło śię ſerce moje w Pánu / wywyżƺon jeſt róg mój w Pánu / rozƺerzyły śię uſtá moje przećiw nieprzyjáćiołom moim ; ábowiem śię rozrádowáłá w zbáwieniu twojim.
Niemáƺći świętego jáko Pán ; bo niemáƺ innego oprócż ćiebie / y niemáƺ ták mocnego / jáko Bóg náƺ.
Nie mówćież nápotem ſłów pyƺnych / á niech nie wychodzą ſłowá hárde z uſt wáƺych ; ábowiem Bóg jeſt umiejętnośći Pánem / á nádáwáją śię ſpráwy jego.
Łuk y mocárze pokruƺeni ſą / á mdli przepáſáni ſą mocą.
Którzy byli náſyceni / nájmują śię zá chleb / á głodni przeſtáli łáknąć ; ták iż niepłodná śiedmioro porodziłá / á którá rodziłá wiele dziátek / zemdláłá.
Pán zábijá y ożywiá / wwodzi do grobu y wywodzi.
Pán ubogiego cżyni y zbogácá / uniżá y wywyżƺá.
Wzbudzá z prochu ubogiego / á z gnoju podnośi żebráká / áby je poſádził z kśiążęty / á dáł im ſtolicę chwálebną ośiádáć ; ábowiem Páńſkie ſą grunty ziemi / á ná nich záłożył świát.
Nóg świętych ſwoich ochrániá / á niepobożni w ćiemnośćiách zámilkną ; bo nie w śile ſwojej będzie śię mąż zmácniáł.
Pán pokruƺy przećiwniki ſwoje / á zágrzmi ná nie z niebá ; Pán będzie ſądził gránice ziemi / á dá moc królowi ſwemu / y wywyżƺy róg pomázáńcá ſwego.
A ták odƺedł Elkáná do Rámáty do domu ſwego / á dziećię ſłużyło Pánu przed Heli kápłánem.
Ale ſynowie Heli byli ſynowie bezbożni / á nie ználi Páná.
Abowiem obycżáj kápłánów ten był około ludu : ktokolwiek ſpráwowáł ofiáry / przychodził ſługá kápłáńſki / gdy wárzono mięſo / májąc widełki o trzech zębách w ręce ſwojej.
Y wráżáł je w ſtátek / álbo w koćieł / álbo w pánew / álbo w gárniec / á coklowiek wyjął widełkámi / to ſobie bráł kápłán. Ták cżynili wƺyſtkim Izráelcżykom / którzy tám do Sylo przychodzili.
Tákże pierwej niż zápálono tłuſtość / tedy przychodził ſługá kápłáńſki / á mówił do cżłowieká ofiárującego : Oddáj mięſo / ábym je upiekł kápłánowi ; ábowiem nie weźmie od ćiebie mięſá wárzonego / jedno ſurowe.
A jeźliż mu odpowiedziáł on cżłowiek : Niech śię pierwej ſpáli tłuſtość / potem ſobie weźmieƺ / cżego będzie żądáłá duƺá twojá / tedy on mówił : Nic z tego ; teráz dáj! á nie dáƺli / wezmę gwáłtem.
Y był to grzech onych ſług bárdzo wielki przed Pánem ; bo śię odtrącáli ludzie od ofiár Páńſkich.
Ale Sámuel ſłużył przed Pánem / ubráne chłopiątko w efod lniány.
A mátká jego ucżyniwƺy mu ſukienkę máłą / przynáƺáłá mu co rok / gdy chádzáłá z mężem ſwym ſpráwowáć ofiárę urocżyſtą.
Y błogoſłáwił Heli Elkánie / y żonie jego / mówiąc : Niech ći dá Pán potomſtwo z tej niewiáſty zá oddánego / którego wyprośiłá u Páná. Y poƺli ná miejſcá ſwoje.
Tedy náwiedził Pán Annę / którá pocżęłá y porodziłá trzech ſynów / y dwie córek ; á pácholę Sámuel uroſł przed Pánem.
Ale Heli zſtárzáł śię był bárdzo / y ſłyƺáł wƺyſtko / co cżynili ſynowie jego cáłemu Izráelowi / y jáko ſypiáli z niewiáſtámi / które śię ſchádzáły przede drzwi námiotu zgromádzeniá.
Y rzekł do nich : Przecżże tákie rzecży cżynićie? Cóż já ſłyƺę o wáƺych złych ſpráwách od wƺyſtkiego ludu?
Nie ták ſynowie moi ; bo nie dobrá ſłáwá / którą já ſłyƺę / że przywodzićie ku przeſtępſtwu lud Páńſki.
Gdy kto zgrzeƺy przećiw cżłowiekowi / ſądzić go będzie ſędziá / ále jeźli przećiw Pánu kto zgrzeƺy / któż śię zá nim ujmie? Lecż nie uſłucháli głoſu ojcá ſwego ; bo je chćiáł Pán pobić.
Ale pácholę Sámuel poſtępowáł á róſł / y podobáł śię ták Pánu jáko y ludziom.
Potem przyƺedł mąż Boży do Heli / y rzekł mu : Ták mówi Pán : Azálim śię nie jáwnie objáwił domowi ojcá twego / gdy byli w Egipćie w domu Fáráonowym?
Y obráłem go ſobie ze wƺyſtkich pokoleń Izráelſkich zá kápłáná / áby ofiárowáł ná ołtárzu moim / á kádził rzecżámi wonnemi / y nośił efod przedemną / y dáłem domowi ojcá twego wƺyſtkie ofiáry pálone od ſynów Izráelſkich.
Przecżżeśćie podeptáli ofiárę moję / y śniedną ofiárę moję / którąm rozkázáł ſpráwowáć w przybytku? y więcejś ucżćił ſyny ſwoje nád mię / ábyśćie śię utucżyli z pierwoćin wƺyſtkich ofiár śniednych Izráelá / ludu mego?
Prztoż mówi Pán / Bóg Izráelſki : Rzekłem wpráwdzie : Dom twój y dom ojcá twego będzie ſłużył przedemną áż ná wieki ; ále teráz mówi Pán : Nie będzieć to / gdyż já te / którzy mnie cżcżą / cżćić będę / á którzy mną gárdzą / będą wzgárdzeni.
Oto / dni przychodzą ; á odetnę rámię twe / y rámię domu ojcá twego / áby nie było ſtárcá w domu twoim ;
Y oglądáƺ wielki ućiſk przybytku Páńſkiego / miáſto ƺcżęśćiá / które Pán dáwáł Izráelowi / y nie będzie ſtárcá w domu twoim po wƺyſtkie dni.
Wƺákże mężá nie wytrácę z ćiebie do końcá od ołtárzá mego / ábym utrápił ocży twe / á boleśćią śćiſnął duƺę twoję ; á wƺyſtko mnóſtwo domu twego pomrze / doroſłƺy lát męſkich.
A toć będzie ná znák / co przyjdzie ná dwóch ſynów twoich / Ofni y Fineeſá ; dniá jednego pomrą ći obá.
Y wzbudzę ſobie kápłáná wiernego / który według ſercá mego / y według myśli mojej cżynić będzie / y zbuduję mu dom trwáły / á będzie ſłużył przed pomázáńcem moim po wƺyſtkie dni.
Y ſtánie śię / ktokolwiek pozoſtánie z domu twego / przyjdzie / áby mu śię ukłonił zá pieniądz ſrebrny y zá ƺtukę chlebá / mówiąc : Przypuść mię proƺę do jednej cżąſtki kápłáńſkiej / ábym jádł ƺtucżkę chlebá.
