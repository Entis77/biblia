Y rzekł Sámuel do Saulá : Poſłał mię Pán / ábym ćię pomázał zá królá nád ludem jego Izráelſkim ; przetoż teraz poſłucháj głoſu ſłów Páńſkich.
Ták mówi Pán zaſtępów : Wſpomniáłem ná to / co ucżynił Amálek Izráelowi / jáko śię nań záſádźił ná drodze / gdy wychodźił z Egiptu.
Przetoż idź / y pobij Amáleká / y wytráć jáko przeklęte wƺyſtko / co ma ; nie folguj mu / ále wybij od mężá áż do niewiáſty / od máłego áż do ſſącego / od wołu áż do owcy / od wielbłądá áż do oſłá.
A ták Saul zebráwƺy lud / policżył go w Teláim / dwá kroć ſto tyśięcy pieƺych / á dźieśięć tyśięcy mężów z Judá.
A gdy przyćiągnął Saul áż do miáſtá Amálek / áby zwiódł bitwę nád potokiem /
Rzekł Saul do Cynejcżyká : Idźćie / odſtąpćie / á wynijdźćie z pośrzodku Amálekitów / ábym was nie wytráćił z nimi ; boś ty ucżynił miłośierdźie ze wƺyſtkimi ſyny Izráelſkimi / gdy ƺli z Egiptu. A ták odſtąpił Cynejcżyk z pośrzodku Amálekitów.
Y poráźił Saul Amáleká / od Hewilá / którędy chodzą do Sur / które jeſt przećiw Egiptowi.
Y pojmał Agágá / królá Amálekitów / żywego / á wƺyſtek lud pobił oſtrzem miecżá.
Ale przepuśćił Saul y lud jego Agágowi / y co nájlepƺym owcom / y wołom / y bydłu tłuſtemu / y báránom / y wƺyſtkiemu / co było nájlepƺego / á nie chćieli tego wygubić ; tylko cokolwiek było nikcżemnego y podłego / to wygubili.
Przetoż ſtáło śię ſłowo Páńſkie do Sámuelá / mówiąc :
Żál mi / żem poſtánowił Saulá zá królá ; ábowiem odwróćił śię odemnie / á ſłowá mego nie wypełnił. Y rozgniewał śię bárzo Sámuel / y wołał do Páná przez cáłą noc.
Wſtáwƺy tedy Sámuel / ƺedł przećiwko Saulowi ráno ; bo dáno znáć Sámuelowi / mówiąc : Przyƺedł Saul do Kármelu / támże wyſtáwił ſobie pámiątkę zwyćięſtwá ; á obróćiwƺy śię poƺedł / y przyƺedł do Gálgál.
A gdy przyƺedł Sámuel do Saulá / rzekł mu Saul : Błogoſłáwionyś ty od Páná / wypełniłem ſłowo Páńſkie.
Ale Sámuel rzekł : A to co zá wrzáſk trzód w uƺách mojich / y co zá ryk wołów / który ja ſłyƺę?
Y odpowiedźiał Saul : Od Amálekitów przygnáno je ; ábowiem lud przepuśćił co nájlepƺym owcom / y wołom / áby je ofiárował Pánu / Bogu twemu / á oſtátekieśmy wytráćili jáko przeklęte.
Tedy rzekł Sámuel do Saulá : Dopuść / á powiem ći / co mówił Pán do mnie w nocy ; á on mu rzekł : Powiedz.
Y rzekł mu Sámuel : Izali / gdyś był máły w ocżách twojich / nie ſtáłeś śię głową pokoleń Izráelſkich / y nie pomázał ćię Pán zá królá nád Izráelem?
Y poſłał ćię Pán w tę drogę / y rzekł : Idź / á wytráć Amálekity jáko przklęte grzeƺniki / y wálcż przećiwko nim / áżbyś je do ƺcżętu wytráćił.
Przecżżeś tedy nie uſłuchał głoſu Páńſkiego / áleś śię udał zá korzyśćią / y ucżyniłeś złe przed ocżymá Páńſkimi?
Tedy odpowiedźiał Saul Sámuelowi : Y owƺem uſłucháłem głoſu Páńſkiego / á ƺedłem drogą / którą mię poſłał Pán / y przywiodłem Agágá / królá Amálekitſkiego / á Amálekity wytráćiłem jáko przeklęte.
Ale lud pobrał z korzyśći owce y woły co przedniejƺe z przeklęſtwá / áby je ofiárował Pánu / Bogu twemu / w Gálgál.
Y rzekł Sámuel : Izali śię ták kocha Pán w cáłopáleniách y w ofiárách / jáko gdy kto ſłuchá głoſu Páńſkiego? oto / poſłuƺeńſtwo lepƺe jeſt niżeli ofiárá / á ſłucháć lepiej jeſt / niż ofiárowáć tłuſtość báránów.
Bo przećiwić śię jeſt jáko grzech cżárowánia / á przeſtąpić przykazánie jeſt jáko báłwochwálſtwo y obrázy ; przetoż iżeś odrzućił ſłowo Páńſkie / tedy ćię też odrzućił Pán / ábyś nie był królem.
Tedy rzekł Saul do Sámuelá : Zgrzeƺyłem / żem przeſtąpił rozkázánie Páńſkie y ſłowá twoje / gdyżem śię bał ludu / y uſłucháłem głoſu ich.
A teraz znieś proƺę grzech mój / á wróć śię ze mną / ábym śię pokłonił Pánu.
Y rzekł Sámuel do Saulá : Nie wrócę śię z tobą ; gdyżeś odrzućił ſłowo Páńſkie / ćiebie też odrzućił Pán / ábyś nie był królem nád Izráelem.
A gdy śię odwróćił Sámuel / żeby odƺedł / uchwyćił ſkrzydło płáƺcżá jego / y oderwáło śię.
Tedy mu rzekł Sámuel : Oderwał Pán króleſtwo Izráelſkie dźiśiáj od ćiebie / y dał je bliźniemu twemu / lepƺemu niżeś ty.
A záiſteć Mocárz Izráelſki nie ſkłámie / áni będźie żáłował ; bo nie jeſt cżłowiekiem / áby miał żáłowáć.
A on rzekł : Zgrzeƺyłem ; wƺákże ucżćij mię proƺę przed ſtárƺymi ludu mego / y przed Izráelem / á wróć śię zemną / ábym śię pokłonił Pánu / Bogu twemu.
Wróćiwƺy śię tedy Sámuel ƺedł zá Saulem / y pokłonił śię Saul Pánu.
Y rzekł Sámuel : Przywiedźćie do mnie Agágá / królá Amálekitſkiego ; y ƺedł do niego Agág powáżnie / y rzekł Agág : Zájiſte uƺedłem gorzkośći śmierći.
Ale rzekł Sámuel : Jáko ośieroćił niewiáſty miecż twój / ták ośieroconá będźie nád inne niewiáſty mátká twojá. Y rozśiekał w kęſy Sámuel Agágá przed oblicżem Páńſkim w Gálgál.
Potem odƺedł Sámuel do Rámáty / á Saul ƺedł do domu ſwego / do Gábáá Saulowego.
A już potem więcey Sámuel nie widźiał Saulá / áż do dniá śmierći ſwojey : wƺákże żáłował Sámuel Saulá / á Pán też żáłował / że ucżynił królem Saulá nád Izráelem.
