Potem przyƺedł Dawid do Noby do Achimelechá kápłáná / á zlękłƺy śię Achimelech wyƺedł przećiwko Dawidowi / y rzekł mu : Przecżżeś ty ſám / á niemáƺ nikogo z tobą?
Y odpowiedzáł Dawid Achimelechowi kápłánowi : Rozkázáł mi król nieco / y rzekł do mnie : Niech nikt nie wie tego / po co ćię poſyłám / y com ći zlećił / przetożem ſługi zoſtáwił ná pewnem miejſcu.
A ták teráz máƺli co przy rękách twoich / áby z pięćioro chlebá / dáj do ręki mojej / álbo cokolwiek znájdźieƺ.
Y odpowiedźiáł kápłán Dawidowi / y rzekł : Niemám chlebá poſpolitego przy ręce mojej / tylko chleb poświęcony ; jeźli śię tylko wſtrzymáli ſłudzy od niewiáſt.
Tedy odpowiedźiáł Dawid kápłánowi / y rzekł mu : Záiſte niewiáſty oddálone były od náſ od wcżorájƺego y dźiś trzećiego dniá / gdym wyƺedł ; przetoż były nácżyniá ſług święte. Ale jeźli tá drogá zmázáná jeſt / wƺákże y tá dźiśiáj poświęconá będźie w nácżyniách.
A ták dáł mu kápłán chleby poświęcone ; ábowiem nie było tám chlebá / tylko chleby pokłádne / które były odjęte od oblicżnośći Páńſkiej / áby położono chleby ćiepłe onegoż dniá / którego one odjęte były.
A był tám mąż z ſług Saulowych onego dniá / zábáwiony przed Pánem / którego imię Doeg / Edomcżyk / nájmożniejƺy z páſterzy / które miáł Saul.
Y rzekł Dawid do Achimelechá : A niemáƺże tu przy ręce ſwej włócżni / álbo miecżá? bom áni miecżá mego / áni żádnej broni mojej nie wźiął w rękę moję / gdyż ſłowo królewſkie przynágláło.
Tedy rzekł kápłán : Miecż Golijátá Filiſtyńcżyká / któregoś zábił w dolinie Elá / oto jeſt uwiniony w ſukno zá efodem ; jeźli ten chceƺ ſobie wźiąć / weźmij ; bo tu inƺego niemáƺ oprócż tego. Y rzekł Dawid : Niemáƺ podobnego temu / dáj mi go.
A ták wſtáł Dawid / y ućiekł dniá onego przed Saulem / y przyƺedł do Achiſá / królá Getſkiego.
Tedy rzekli ſłudzy Achiſowi do niego : Izáli nie ten jeſt Dawid / król źiemi? Izáli nie temu śpiewáno w hufcách / mówiąc : Poráźił Saul ſwój tyśiąc / á Dawid ſwoich dźieśięć tyśięcy?
Y złożył Dawid ſłowá te do ſercá ſwego / á báł śię bárdzo Achiſá / królá Getſkiego.
Przetoż zmienił obycżáje ſwoje przed ocżymá ich / á cżynił śię ƺálonym w rękách ich / y kreślił ná drzwiách brámy / y puƺcżáł śliny ná brodę ſwoję.
Tedy rzekł Achiſ do ſług ſwoich : Otośćie widźieli cżłowieká ƺálonego / cżemużeśćie go przywiedli do mnie?
Nie doſtáwá mi ƺálonych / żeśćie przywiedli tego / áby ƺáláł przedemną? tenże má wnijść do domu mego?
