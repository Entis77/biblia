Potem przyƺedł Dawid do Noby do Achimelechá kápłáná / á zlękƺy śię Achimelech wyƺedł przećiwko Dawidowi / y rzekł mu : Przecżżeś ty ſám / á niemáƺ nikogo z tobą?
Y odpowiedzał Dawid Achimelechowi kápłánowi : Rozkázał mi król nieco / y rzekł do mnie : Niech nikt nie wie tego / po co ćię poſyłám / y com ći zlećił / przetożem ſługi zoſtáwił ná pewnem miejſcu.
A ták teraz maƺli co przy rękách twojich / áby z pięćioro chlebá / dáj do ręki mojey / álbo cokolwiek znájdźieƺ.
Y odpowiedźiał kápłán Dawidowi / y rzekł : Niemám chlebá poſpolitego przy ręce mojey / tylko chleb poświęcony ; jeſli śię tylko wſtrzymáli ſłudzy od niewiáſt.
Tedy odpowiedźiał Dawid kápłánowi / y rzekł mu : Zájiſte niewiáſty oddálone były od nas od wcżorájƺego y dźiś trzećiego dniá / gdym wyƺedł ; przetoż były nacżynia ſług święte. Ale jeſli tá drogá zmázáná jeſt / wƺákże y tá dźiśiáj poświęconá będźie w nácżyniách.
A ták dał mu kápłán chleby poświęcone ; ábowiem nie było tám chlebá / tylko chleby pokłádne / które były odjęte od oblicżnośći Páńſkiej / áby położono chleby ćiepłe onegoż dniá / którego one odjęte były.
A był tám mąż z ſług Saulowych onego dniá / zábáwiony przed Pánem / którego imię Doeg / Edomcżyk / nájmożniejƺy z páſterzy / które miał Saul.
Y rzekł Dawid do Achimelechá : A niemáƺże tu przy ręce ſwej włócżni / álbo miecżá? bom áni miecżá mego / áni żádney broni mojey nie wźiął w rękę moję / gdyż ſłowo królewſkie przynágláło.
Tedy rzekł kápłán : Miecż Golijátá Filiſtyńcżyká / któregoś zábił w dolinie Elá / oto jeſt uwiniony w ſukno zá efodem ; jeſli ten chceƺ ſobie wźiąć / weźmij ; bo tu inƺego niemáƺ oprócż tego. Y rzekł Dawid : Niemáƺ podobnego temu / dáj mi go.
A ták wſtał Dawid / y ućiekł dniá onego przed Saulem / y przyƺedł do Achiſá / królá Getſkiego.
Tedy rzekli ſłudzy Achiſowi do niego : Izali nie ten jeſt Dawid / król źiemi? Izali nie temu śpiewano w hufcách / mówiąc : Poráźił Saul ſwój tyśiąc / á Dawid ſwojich dźieśięć tyśięcy?
Y złożył Dawid ſłowá te do ſercá ſwego / á bał śię bárzo Achiſá / królá Getſkiego.
Przetoż zmienił obycżáje ſwoje przed ocżymá ich / á cżynił śię ƺálonym w rękách ich / y kreślił ná drzwiách bramy / y puƺcżał śliny ná brodę ſwoję.
Tedy rzekł Achis do ſług ſwojich : Otośćie widźieli cżłowieká ƺálonego / cżemużeśćie go przywiedli do mnie?
Nie doſtáwá mi ƺálonych / żeśćie przywiedli tego / áby ƺálał przedemną? tenże ma wnijść do domu mego?
