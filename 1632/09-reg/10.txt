Tedy Sámuel wźiął báńkę oliwy / y wyláł ná głowę jego / á pocáłowáwƺy go / rzekł : Izáli ćię nie pomázáł Pán nád dźiedźictwem ſwojim zá wodzá?
Gdy dźiś odejdźieƺ ode mnie / znájdźieƺ dwóch mężów u grobu Rácheli / ná gránicách Benjámin w Selſá / którzyć powiedzą : Nálázły śię oślice / którycheś chodźił ƺukáć / á oto zániecháwƺy ojćiec twój ſtárániá o oślicách / fráſuje śię o wáſ / mówiąc ; Cóż mám cżynić z ſtrony ſyná mego?
Potem odƺedłƺy ſtámtąd dálej przyjdźieƺ áż ná pole Tábor ; y ſpotkáją ćię tám trzej mężowie idący do Bogá / do domu Bożego / jeden nieśie troje koźląt / á drugi nieśie trzy bochny chlebá / á trzeći nieśie łágiew winá ;
Y pozdrowią ćię w pokoju / y dádząć dwá chleby / które weźmieƺ z rąk ich.
Potem przyjdźieƺ ná págórek Boży / kędy jeſt ſtráż Filiſtyńſká ; á gdy tám wnijdźieƺ do miáſtá / ſpotkáƺ śię z gromádą proroków zſtępujących z góry / á przed nimi będźie hárfá / y bęben / y piƺcżáłká / y lutniá / á oni będą prorokowáli.
Y zſtąpi ná ćię Duch Páńſki / y będźieƺ z nimi prorokowáł / á odmieniƺ śię w inƺego mężá.
A gdy przyjdą te znáki ná ćię / cżyń cokolwiek znájdźie ręká twojá : bo Bóg jeſt z tobą.
Potem pójdźieƺ przedemną do Gálgál / á oto / já przyjdę do ćiebie dlá ſpráwowániá ofiár cáłopálonych / y dlá ofiárowániá ofiár ſpokojnych ; przez śiedm dni będźieƺ cżekáł / áż przyjdę do ćiebie y ukáżęć / co będźieƺ miáł cżynić.
Y ſtáło śię / gdy śię obróćił / áby odƺedł od Sámuelá / odmienił Bóg ſerce jego w inƺe ; y ſpełniły śię wƺyſtkie one znáki dniá onego.
Y przyƺli tám ná págórek / á oto / gromádá proroków ſpotkáłá śię z nim / y odpocżął ná nim Duch Boży / y prorokowáł w pośrodku nich.
Stáło śię tedy / że wƺyſcy / którzy go przedtem ználi / ujrzeli / á oto / z prorokámi prorokowáł ; y mówili wƺyſcy jeden do drugiego : Cóż śię ſtáło ſynowi Cyſowemu? Izáli też Saul między prorokámi?
Y odpowiedźiáł mąż niektóry ſtámtąd / y rzekł : Y któż jeſt ojcem ich? przetoż weƺło to w przypowieść : Izáli y Saul między prorokámi?
Y przeſtáł prorokowáć / á przyƺedł ná górę.
Potem rzekł ſtryj Saulá do niego / y do ſługi jego : Gdźieżeśćie chodźili? Y odpowiedźiáł : Szukáć oślic ; á widząc / żeśmy ich nie mogli ználeść / poƺliśmy do Sámuelá.
Y rzekł ſtryj Saulá : Powiedz mi proƺę / co wám powiedźiáł Sámuel.
Y odpowiedźiáł Saul ſtryjowi ſwemu : Oznájmił nám zá pewne / iż ználeźiono oślice ; ále o ſpráwie króleſtwá / o którem mu Sámuel powiedźiáł / nie oznájmił mu.
Potem zwołáł Sámuel ludu do Páná do Máſfá /
Y rzekł do ſynów Izráelſkich : Ták mówi Pán / Bóg Izráelſki : Jám wywiódł Izráelá z Egiptu / y wybáwiłem wáſ z rąk Egipcżánów / y z rąk wƺyſtkich króleſtw / które wáſ trápiły ;
Aleśćie wy dźiś odrzućili Bogá wáƺego / który wáſ ſám wybáwiá od wƺyſtkiego złego wáƺego / y od ućiſków wáƺych / y rzekliśćie mu : Poſtánów królá nád námi. Przetoż teráz ſtáńćie przed Pánem według pokoleń wáƺych / y wedle tyśiąców wáƺych.
A gdy kázáł przyſtąpić Sámuel wƺyſtkim pokoleniom Izráelſkim / pádł loſ ná pokolenie Benjáminowe.
Potem kázáł przyſtąpić pokoleniu Benjáminowemu według domów jego / y pádł loſ ná dom Metry / á tráfił ná Saulá / ſyná Cyſowego ; y ƺukáno go / ále go nie ználeźiono.
Przetoż pytáli śię znowu Páná : Przyjdźieli jeƺcże ſám ten mąż? Y odpowiedźiáł Pán : Oto śię ſkrył między ſprzętem.
Tedy poƺedłƺy wźięli go ſtámtąd. Y ſtánął w pośród ludu / y był głową wyżƺy nád wƺyſtek lud.
Y rzekł Sámuel do wƺyſtkiego ludu : Widźićież / kogo to Pán obráł / że mu niemáƺ równego między wƺyſtkim ludem? przetoż zákrzyknął wƺyſtek lud / mówiąc : Niech żyje król!
Tedy powiedźiáł Sámuel ludowi práwo królewſkie / y ſpiſáł je ná kśięgách / które położył przed Pánem. Potem rozpuśćił Sámuel wƺyſtek lud / káżdego do domu ſwego.
Tákże y Saul ƺedł do domu ſwego do Gábáá / y ƺły zá nim wojſká / których Bóg ſercá dotknął.
Lecż ludźie niepobożni rzekli : Cóż / tenże náſ wybáwi? Y wzgárdźili nim / áni mu przynieśli dárów ; ále on cżynił / jákoby nie ſłyƺáł.
