Tedy Sámuel wźiął báńkę oliwy / y wylał ná głowę jego / á pocáłowáwƺy go / rzekł : Izali ćię nie pomázał Pán nád dźiedźictwem ſwojim zá wodzá?
Gdy dźiś odejdźieƺ ode mnie / znájdźieƺ dwu mężów u grobu Rácheli / ná gránicách Benjámin w Selſá / którzyć powiedzą : Nálázły śię oślice / którycheś chodźił ƺukáć / á oto zániecháwƺy oćiec twój ſtáránia o oślicách / fráſuje śię o was / mówiąc ; Cóż mam cżynić z ſtrony ſyná mego?
Potem odƺedƺy ſtámtąd dálej przyjdźieƺ áż ná pole Tábor ; y ſpotkáją ćię tám trzej mężowie idący do Bogá / do domu Bożego / jeden nieśie troje koźląt / á drugi nieśie trzy bochny chlebá / á trzeći nieśie łágiew winá ;
Y pozdrowią ćię w pokoju / y dádząć dwá chleby / które weźmieƺ z rąk ich.
Potem przyjdźieƺ ná págórek Boży / kędy jeſt ſtráż Filiſtyńſká ; á gdy tám wnijdźieƺ do miáſtá / ſpotkáƺ śię z gromádą proroków zſtępujących z góry / á przed nimi będźie hárfá / y bęben / y piƺcżáłká / y lutniá / á oni będą prorokowáli.
Y zſtąpi ná ćię Duch Páńſki / y będźieƺ z nimi prorokował / á odmieniƺ śię w inƺego mężá.
A gdy przyjdą te znáki ná ćię / cżyń cokolwiek znájdźie ręká twojá : bo Bóg jeſt z tobą.
Potem pójdźieƺ przedemną do Gálgál / á oto / ja przydę do ćiebie dla ſpráwowánia ofiár cáłopálonych / y dla ofiárowánia ofiár ſpokojnych ; przez śiedm dni będźieƺ cżekał / áż przydę do ćiebie y ukáżęć / co będźieƺ miał cżynić.
Y ſtáło śię / gdy śię obróćił / áby odƺedł od Sámuelá / odmienił Bóg ſerce jego w inƺe ; y ſpełniły śię wƺyſtkie one znáki dniá onego.
Y przyƺli tám ná págórek / á oto / gromádá proroków ſpotkáłá śię z nim / y odpocżął ná nim Duch Boży / y prorokował w pośrzodku nich.
Stáło śię tedy / że wƺyſcy / którzy go przedtem ználi / ujrzeli / á oto / z prorokámi prorokował ; y mówili wƺyſcy jeden do drugiego : Cóż śię ſtáło ſynowi Cyſowemu? Izali też Saul miedzy prorokámi?
Y odpowiedźiał mąż niektóry ſtámtąd / y rzekł : Y któż jeſt ojcem ich? przetoż weƺło to w przypowieść : Izali y Saul miedzy prorokámi?
Y przeſtał prorokowáć / á przyƺedł ná górę.
Potem rzekł ſtryj Saulá do niego / y do ſługi jego : Gdźieżeśćie chodźili? Y odpowiedźiał : Szukáć oślic ; á widząc / żeśmy ich nie mogli ználeść / poƺliśmy do Sámuelá.
Y rzekł ſtryj Saulá : Powiedz mi proƺę / co wam powiedźiał Sámuel.
Y odpowiedźiał Saul ſtryjowi ſwemu : Oznájmił nam zá pewne / iż ználeźiono oślice ; ále o ſpráwie króleſtwá / o którem mu Sámuel powiedźiał / nie oznájmił mu.
Potem zwołał Sámuel ludu do Páná do Máſfá /
Y rzekł do ſynów Izráelſkich : Ták mówi Pán / Bóg Izráelſki : Jam wywiódł Izráelá z Egiptu / y wybáwiłem was z rąk Egipcżánów / y z rąk wƺyſtkich króleſtw / które was trápiły ;
Aleśćie wy dźiś odrzućili Bogá wáƺego / który was ſám wybáwiá od wƺyſtkiego złego wáƺego / y od ućiſków wáƺych / y rzekliśćie mu : Poſtánów królá nád námi. Przetoż teraz ſtáńćie przed Pánem według pokoleń wáƺych / y wedle tyśiąców wáƺych.
A gdy kázał przyſtąpić Sámuel wƺyſtkim pokoleniom Izráelſkim / pádł los ná pokolenie Benjáminowe.
Potem kázał przyſtąpić pokoleniu Benjáminowemu według domów jego / y pádł los ná dom Metry / á tráfił ná Saulá / ſyná Cyſowego ; y ƺukáno go / ále go nie ználeźiono.
Przetoż pytáli śię znowu Páná : Przyjdźieli jeƺcże ſám ten mąż? Y odpowiedźiał Pán : Oto śię ſkrył miedzy ſprzętem.
Tedy poƺedƺy wźięli go ſtámtąd. Y ſtánął w pośród ludu / y był głową wyżƺy nád wƺyſtek lud.
Y rzekł Sámuel do wƺyſtkiego ludu : Widźićież / kogo to Pán obrał / że mu niemáƺ równego miedzy wƺyſtkim ludem? przetoż zákrzyknął wƺyſtek lud / mówiąc : Niech żyje król!
Tedy powiedźiał Sámuel ludowi práwo królewſkie / y ſpiſał je ná kśięgách / które położył przed Pánem. Potem rozpuśćił Sámuel wƺyſtek lud / káżdego do domu ſwego.
Tákże y Saul ƺedł do domu ſwego do Gábáá / y ƺły zá nim wojſká / których Bóg ſercá dotknął.
Lecż ludźie niepobożni rzekli : Cóż / tenże nas wybáwi? Y wzgárdźili nim / áni mu przynieśli dárów ; ále on cżynił / jákoby nie ſłyƺał.
