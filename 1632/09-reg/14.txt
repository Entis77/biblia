Y ſtáło śię dniá niektórego / że rzekł Jonátán / ſyn Saulá / do ſługi / który nośił broń jego : Pójdź / przejdziemy do ſtráży Filiſtyńſkiej / którá jeſt ná onej ſtronie ; á ojcu ſwemu o tem nie oznájmił.
Ale Saul zoſtáł był przy págórku pod jábłonią gránátową / którá byłá w Migron / y lud / który był z nim / około ƺeśćiu ſet mężów.
A Achijáſ / ſyn Achitobá / brátá Ichábodá / ſyná Fineeſowego / ſyná Heli / kápłáná Páńſkiego w Sylo / nośił efod : á lud nie wiedziáł / iż odƺedł Jonátán.
Ale między przechodámi / kędy ƺukáł Jonátán przejśćiá ku ſtráży Filiſtyńſkiej / byłá ſkáłá oſtrá po jednej ſtronie / tákże ſkáłá oſtrá po drugiej ſtronie ; jednej imię Boſeſ á drugiej Sene.
Skáłá jedná byłá ná północy przećiwko Máchmáſ / á drugá ná południe przećiwko Gábáá.
Y rzekł Jonátán do wyroſtká / który nośił broń jego : Pójdźmy / á przejdziemy do ſtráży tych nieobrzezáńców / ſnáć ucżyni Pán przez náſ wybáwienie ; boć nie trudno Pánu wybáwić w wielu álbo w troƺe.
Y rzekł mu ſługá / noƺący broń jego : Cżyń / co śię podobá ſercu twemu ; idź / gdzie chceƺ / oto já będę z tobą według woli twojej.
Tedy rzekł Jonátán : oto my idziemy do tych mężów / á ukáżemy śię im.
Jeźli nám ták rzeką : Cżekájćie / áż przyjdziemy do wáſ / ſtójmyż ná miejſcu ſwem / á nie chodźmy do nich ;
Ale jeźliż ták rzeką : Pójdźćie do náſ / pójdźmyż ; boć je dáł Pán w rękę náƺę / á to będziemy mieli zá znák.
Ukázáli śię tedy obáj ſtráży Filiſtyńſkiej. Y rzekli Filiſtynowie : Onoż Hebrejcżycy wychodzą z jáſkini / w której śię byli pokryli.
Y mówili mężowie / co ná ſtráży byli / do Jonátáná / y do wyroſtká / co zá nim broń nośił / y rzekli : Pójdźćie ku nám / á oznájmiemy wám coś. Y rzekł Jonátán do ſługi ſwego : Pójdź zá mną ; boć je Pán podáł w ręce Izráelcżykom.
Lázł tedy Jonátán ná rękách ſwych / y ná nogách ſwych / á wyroſtek jego zá nim ; y pádli przed Jonátánem / y przed wyroſtkiem jego / który też zábijáł / idąc zá nim.
A táć byłá poráżká pierwƺá / w której pobił Jonátán y wyroſtek jego / co broń zá nim nośił / około dwudzieſtu mężów / jákoby ná pół ſtájá roli.
Y przyƺedł ſtrách ná obóz ná polu / y ná wƺyſtek lud ; ſtráż też / y ći / którzy byli wyjecháli ná zdobycż / lękáli śię / áż śię ziemiá trzęſłá ; bo byłá w ſtráchu Bożym.
Y obácżyłá ſtráż Saulowá w Gábáá Benjáminowym / że śię ono mnóſtwo rozſypowáło / y pierzcháło / y że śię go urywáło.
Tedy rzekł Saul do ludu / który przy nim był : Wywiedzćie śię záráz / á obácżćie / kto odƺedł z náƺych ; á gdy śię wywiádowáli / oto / nie było Jonátáná / y wyroſtká / co zá nim broń nośił.
Y rzekł Saul do Achijáſá : Przyſtáw ſkrzynię Bożą ; ( bo byłá ſkrzyniá Bożá dniá onego z ſyny Izráelſkimi. )
Y ſtáło śię / gdy jeƺcże Saul mówił do kápłáná / że zámięƺánie / które było w obozie Filiſtyńſkim / wzmágáło śię y rozmnáżáło ; przetoż rzekł Saul do kápłáná : Záwśćiągnij ręki twojej.
A ták zebráwƺy śię Saul / y wƺyſtek lud / który był z nim / przyƺli / gdzie byłá bitwá / á oto / káżdego miecż był obrócony ná towárzyƺá jego / y byłá poráżká bárdzo wielká.
A Hebrejcżycy / którzy przedtem przeſtáwáli z Filiſtynámi / którzy z nimi ćiągnęli w obozie tám y ſám / ći śię też obróćiwƺy ſtánęli przy Izráelu / który był z Saulem y z Jonátánem.
Nádto wƺyſcy mężowie Izráelſcy / którzy śię byli pokryli ná górze Efráim / gdy uſłyƺeli / iż ućiekáją Filiſtynowie / ƺli zá nimi w pogoń w onej bitwie.
Y wybáwił Pán dniá onego Izráelá / á bitwá oná záƺłá áż do Betáwen.
A mężowie Izráelſcy ſtrudzeni byli onego dniá. Y poprzyśiągł Saul lud / mówiąc : Przeklęty mąż / któryby jádł chleb przed wiecżorem / áż śię pomƺcżę nád nieprzyjáćioły mymi. Y nie ſkoƺtowáł wƺyſtek lud chlebá.
Tedy wƺyſtek lud onej ziemi przyƺedł do láſu / gdzie było wiele miodu ná ziemi.
Wƺedłƺy tedy lud do láſu / ujrzáł płynący miód ; wƺákże nie donióſł żáden z miodem ręki ſwojej do uſt ſwoich / bo śię báł lud onej przyśięgi.
Ale Jonátán nie ſłyƺáł / gdy poprzyśięgáł lud ojćiec jego ; y śćiągnął koniec láſki / którą miáł w ręce ſwej / á omocżył go w pláſtrze miodu / y obróćił rękę ſwoję do uſt ſwoich / y oświećiły śię ocży jego.
A odpowiádájąc jeden z ludu / rzekł : Przyśięgą záwiązáł ojćiec twój lud / mówiąc : Przeklęty mąż / któryby jádł chleb dziśiáj ; ſtądże uſtáł lud.
Tedy rzekł Jonátán : Strwożył ojćiec mój lud ziemi. Pátrzćie proƺę / jáko ſą oświecone ocży moje / iżem ſkoƺtowáł trochę miodu tego ;
Jáko dáleko więcej / gdyby śię był nájádł dziś lud z łupu nieprzyjáćiół ſwoich / których nábył ; izáliby nie byłá więkƺá poráżká między Filiſtynámi?
Porázili tedy dniá onego Filiſtyny od Máchmáſ áż do Ajálon / y ſprácowáł śię lud bárdzo.
Tedy śię lud udáł ná łup / á nábráwƺy owiec / y wołów / y ćieląt / rzezáli je ná ziemi / á jádł lud ze krwią.
Y powiedziáno Saulowi mówiąc : Oto lud grzeƺy przećiw Pánu / jedząc ze krwią ; który rzekł : Zgrzeƺyliśćie ; przytocżćież ſám do mnie teráz kámień wielki.
Zátem rzekł Saul : Rozejdźćie śię między lud / á rzecżćie do nich : Przywiedźćie do mnie káżdy wołu ſwego / y káżdy owcę ſwą / bijćież je tu / á jedzćie / á nie zgrzeƺyćie przećiwko Pánu / jedząc ze krwią. Y przywiedli wƺyſtek lud / káżdy wołu ſwego w ręce ſwej w nocy / y bito je tám.
Y zbudowáł Saul ołtárz Pánu ; toć nájpierwƺy ołtárz / który zbudowáł Pánu.
Y rzekł Saul : Puśćmy śię zá Filiſtynámi nocą / á bijmy je áż do świtániá / á nie zoſtáwujmy z nich y jednego. Którzy mu odpowiedzieli : Cokolwiek dobrego jeſt w ocżách twoich / ucżyń ; ále kápłán rzekł : Przyſtąpmy ſám do Bogá.
Tedy śię rádził Saul Bogá : Mámli śię puśćić zá Filiſtynámi? podáƺli je w ręce Izráelá? Y nie odpowiedziáł mu dniá tego.
Przetoż rzekł Saul : Przyſtąpćie ſám wƺyſcy celniejśi z ludu / y wywiedzćie śię / á pátrzćie / przy kimby grzech dziś był.
Bo jáko żywy Pán / który wybáwiá Izráelá / choćby był y przy Jonátánie / ſynu moim / że śmierćią umrze. Y nie odpowiedziáł mu nikt ze wƺyſtkiego ludu.
Nádto rzekł do wƺyſtkiego Izráelá : Wy będziećie ná jednej ſtronie / á já y Jonátán / ſyn mój / będziemy ná drugiej ſtronie. Y odpowiedziáł lud Saulowi : Co dobrego jeſt w ocżách twoich / ucżyń.
Zátem rzekł Saul do Páná / Bogá Izráelſkiego : Pánie / pokáż ſpráwiedliwą ; y nálezion jeſt Jonátán y Saul / á lud wyƺedł z tego.
Potem rzekł Saul : Rzuććie loſ między mną y między Jonátánem / ſynem moim ; y náleziony jeſt Jonátán.
Zátem rzekł Saul do Jonátáná : Powiedz mi / coś ucżynił? Y powiedzáł mu Jonátán / y rzekł : Skoƺtowáłem tylko końcem láſki / którąm miáł w ręce mojej / trochę miodu / y dlá tegoż já mám umrzeć?
Y odpowiedzáł Saul : To á to mi niech Bóg ucżyni / że śmierćią umrzeƺ / Jonátánie.
Ale lud rzekł do Saulá : Izáli Jonátán umrze / który ucżynił to wybáwienie wielkie w Izráelu? Boże uchowáj! jáko żywy Pán / nie ſpádnie y włoſ z głowy jego ná ziemię ; ábowiem zá pomocą Bożą ucżynił to dziśiáj. A ták wybáwił lud Jonátáná / że nie umárł.
Tedy śię wróćił Saul od Filiſtynów / á Filiſtynowie odeƺli ná miejſce ſwoje.
A Saul otrzymáwƺy króleſtwo nád Izráelem / wálcżył przećiwko okolicżnym wƺyſtkim nieprzyjáćiołom ſwoim / przećiw Moábitom / y przećiw ſynom Ammonowym / y przećiw Edomcżykom / y przećiw królom Sobá / y przećiw Filiſtynom ; á gdzie śię kolwiek obróćił / mężnie śię ſpráwowáł.
Zebráwƺy też wojſko / poráził Amálekity / y wyrwáł Izráelá z ręki tego / który go puſtoƺył.
A miáł Saul ſyny / Jonátáná / y Jeſujego / y Melchiſuá ; á imioná dwóch córek jego : imię pierworodnej Merob / á młodƺej Michol ;
A imię żony Saulowej Achinoám / którá byłá córką Achimááſową ; á imię Hetmáná wojſká jego Abner / ſyn Nerá / ſtryjá Saulowego.
Bo Cyſ był ojćiec Saulá / á Ner ojćiec Abnerá / ſyn Abijelá.
Y byłá wojná wielká z Filiſtynámi po wƺyſtkie dni Saulowe. Przetoż / gdziekolwiek widziáł Saul jákiego śilnego y dzielnego mężá / przyjmowáł go do śiebie.
