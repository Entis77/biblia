Y ſtáło śię dniá niektórego / że rzekł Jonátán / ſyn Saulá / do ſługi / który nośił broń jego : Pójdź / przejdźiemy do ſtráży Filiſtyńſkiej / która jeſt ná oney ſtronie ; á ojcu ſwemu o tem nie oznájmił.
Ale Saul zoſtáł był przy págórku pod jábłonią gránátową / która byłá w Migron / y lud / który był z nim / około ƺeśćiu ſet mężów.
A Achijáſ / ſyn Achitobá / brátá Ichábodá / ſyná Fineeſowego / ſyná Heli / kápłáná Páńſkiego w Sylo / nośił efod : á lud nie wiedźiáł / iż odƺedł Jonátán.
Ale między przechodámi / kędy ƺukáł Jonátán przejśćiá ku ſtráży Filiſtyńſkiej / byłá ſkáłá oſtrá po jedney ſtronie / tákże ſkáłá oſtrá po drugiej ſtronie ; jedney imię Boſeſ á drugiej Sene.
Skáłá jedná byłá ná północy przećiwko Máchmáſ / á drugá ná południe przećiwko Gábáá.
Y rzekł Jonátán do wyroſtká / który nośił broń jego : Pójdźmy / á przejdźiemy do ſtráży tych nieobrzezáńców / ſnáć ucżyni Pán przez nas wybáwienie ; boć nie trudno Pánu wybáwić w wielu álbo w troƺe.
Y rzekł mu ſługá / noƺący broń jego : Cżyń / co śię podobá ſercu twemu ; idź / gdźie chceƺ / oto ja będę z tobą według woli twojey.
Tedy rzekł Jonátán : oto my idźiemy do tych mężów / á ukáżemy śię im.
Jeſli nám ták rzeką : Cżekájćie / áż przyjdźiemy do was / ſtójmyż ná miejſcu ſwem / á nie chodźmy do nich ;
Ale jeſliż ták rzeką : Pójdźćie do nas / pójdźmyż ; boć je dáł Pán w rękę náƺę / á to będźiemy mieli zá znák.
Ukázáli śię tedy obáj ſtráży Filiſtyńſkiej. Y rzekli Filiſtynowie : Onoż Hebrejcżycy wychodzą z jáſkini / w którey śię byli pokryli.
Y mówili mężowie / co ná ſtráży byli / do Jonátáná / y do wyroſtká / co zá nim broń nośił / y rzekli : Pójdźćie ku nám / á oznájmiemy wám coś. Y rzekł Jonátán do ſługi ſwego : Pójdź zá mną ; boć je Pán podáł w ręce Izráelcżykom.
Lázł tedy Jonátán ná rękách ſwych / y ná nogách ſwych / á wyroſtek jego zá nim ; y pádli przed Jonátánem / y przed wyroſtkiem jego / który też zábijáł / idąc zá nim.
A táć byłá poráżká pierwƺá / w którey pobił Jonátán y wyroſtek jego / co broń zá nim nośił / około dwudźieſtu mężów / jákoby ná pół ſtájá roli.
Y przyƺedł ſtrách ná obóz ná polu / y ná wƺyſtek lud ; ſtráż też / y ći / którzy byli wyjecháli ná zdobycż / lękáli śię / áż śię źiemiá trzęſłá ; bo byłá w ſtráchu Bożym.
Y obácżyłá ſtráż Saulowá w Gábáá Benjáminowym / że śię ono mnóſtwo rozſypowáło / y pierzcháło / y że śię go urywáło.
Tedy rzekł Saul do ludu / który przy nim był : Wywiedzćie śię záráz / á obácżćie / kto odƺedł z náƺych ; á gdy śię wywiádowáli / oto / nie było Jonátáná / y wyroſtká / co zá nim broń nośił.
Y rzekł Saul do Achijáſá : Przyſtáw ſkrzynię Bożą ; ( bo byłá ſkrzyniá Bożá dniá onego z ſyny Izráelſkimi. )
Y ſtáło śię / gdy jeƺcże Saul mówił do kápłáná / że zámięƺánie / które było w oboźie Filiſtyńſkim / wzmágáło śię y rozmnáżáło ; przetoż rzekł Saul do kápłáná : Záwśćiągnij ręki twojey.
A ták zebráwƺy śię Saul / y wƺyſtek lud / który był z nim / przyƺli / gdźie byłá bitwá / á oto / káżdego miecż był obrócony ná towárzyƺá jego / y byłá poráżká bárdzo wielká.
A Hebrejcżycy / którzy przedtem przeſtáwáli z Filiſtynámi / którzy z nimi ćiągnęli w oboźie tám y ſám / ći śię też obróćiwƺy ſtánęli przy Izráelu / który był z Saulem y z Jonátánem.
Nádto wƺyſcy mężowie Izráelſcy / którzy śię byli pokryli ná górze Efráim / gdy uſłyƺeli / iż ućiekáją Filiſtynowie / ƺli zá nimi w pogoń w oney bitwie.
Y wybáwił Pán dniá onego Izráelá / á bitwá oná záƺłá áż do Betáwen.
A mężowie Izráelſcy ſtrudzeni byli onego dniá. Y poprzyśiągł Saul lud / mówiąc : Przeklęty mąż / któryby jádł chleb przed wiecżorem / áż śię pomƺcżę nád nieprzyjáćioły mymi. Y nie ſkoƺtowáł wƺyſtek lud chlebá.
Tedy wƺyſtek lud oney źiemi przyƺedł do láſu / gdźie było wiele miodu ná źiemi.
Wƺedłƺy tedy lud do láſu / ujrzáł płynący miód ; wƺákże nie donióſł żáden z miodem ręki ſwojey do uſt ſwoich / bo śię báł lud oney przyśięgi.
Ale Jonátán nie ſłyƺáł / gdy poprzyśięgáł lud ojćiec jego ; y śćiągnął koniec láſki / którą miał w ręce ſwej / á omocżył go w pláſtrze miodu / y obróćił rękę ſwoję do uſt ſwoich / y oświećiły śię ocży jego.
A odpowiádájąc jeden z ludu / rzekł : Przyśięgą záwiązáł ojćiec twój lud / mówiąc : Przeklęty mąż / któryby jádł chleb dźiśiáj ; ſtądże uſtáł lud.
Tedy rzekł Jonátán : Strwożył ojćiec mój lud źiemi. Pátrzćie proƺę / jáko ſą oświecone ocży moje / iżem ſkoƺtowáł trochę miodu tego ;
Jáko dáleko więcey / gdyby śię był nájádł dźiś lud z łupu nieprzyjáćiół ſwoich / których nábył ; izáliby nie byłá więkƺá poráżká między Filiſtynámi?
Poráźili tedy dniá onego Filiſtyny od Máchmáſ áż do Ajálon / y ſprácowáł śię lud bárdzo.
Tedy śię lud udáł ná łup / á nábráwƺy owiec / y wołów / y ćieląt / rzezáli je ná źiemi / á jádł lud ze krwią.
Y powiedźiáno Saulowi mówiąc : Oto lud grzeƺy przećiw Pánu / jedząc ze krwią ; który rzekł : Zgrzeƺyliśćie ; przytocżćież ſám do mnie teraz kámień wielki.
Zátem rzekł Saul : Rozejdźćie śię między lud / á rzecżćie do nich : Przywiedźćie do mnie káżdy wołu ſwego / y káżdy owcę ſwą / bijćież je tu / á jedzćie / á nie zgrzeƺyćie przećiwko Pánu / jedząc ze krwią. Y przywiedli wƺyſtek lud / káżdy wołu ſwego w ręce ſwej w nocy / y bito je tám.
Y zbudowáł Saul ołtárz Pánu ; toć nájpierwƺy ołtárz / który zbudowáł Pánu.
Y rzekł Saul : Puśćmy śię zá Filiſtynámi nocą / á bijmy je áż do świtániá / á nie zoſtáwujmy z nich y jednego. Którzy mu odpowiedźieli : Cokolwiek dobrego jeſt w ocżách twoich / ucżyń ; ále kápłán rzekł : Przyſtąpmy ſám do Bogá.
Tedy śię rádźił Saul Bogá : Mámli śię puśćić zá Filiſtynámi? podáƺli je w ręce Izráelá? Y nie odpowiedźiáł mu dniá tego.
Przetoż rzekł Saul : Przyſtąpćie ſám wƺyſcy celniejśi z ludu / y wywiedzćie śię / á pátrzćie / przy kimby grzech dźiś był.
Bo jáko żywy Pán / który wybáwiá Izráelá / choćby był y przy Jonátánie / ſynu moim / że śmierćią umrze. Y nie odpowiedźiáł mu nikt ze wƺyſtkiego ludu.
Nádto rzekł do wƺyſtkiego Izráelá : Wy będźiećie ná jedney ſtronie / á ja y Jonátán / ſyn mój / będźiemy ná drugiej ſtronie. Y odpowiedźiáł lud Saulowi : Co dobrego jeſt w ocżách twoich / ucżyń.
Zátem rzekł Saul do Páná / Bogá Izráelſkiego : Pánie / pokáż ſpráwiedliwą ; y náleźion jeſt Jonátán y Saul / á lud wyƺedł z tego.
Potem rzekł Saul : Rzuććie loſ między mną y między Jonátánem / ſynem moim ; y náleźiony jeſt Jonátán.
Zátem rzekł Saul do Jonátáná : Powiedz mi / coś ucżynił? Y powiedzáł mu Jonátán / y rzekł : Skoƺtowáłem tylko końcem láſki / którąm miał w ręce mojey / trochę miodu / y dla tegoż ja mam umrzeć?
Y odpowiedzáł Saul : To á to mi niech Bóg ucżyni / że śmierćią umrzeƺ / Jonátánie.
Ale lud rzekł do Saulá : Izáli Jonátán umrze / który ucżynił to wybáwienie wielkie w Izráelu? Boże uchowáj! jáko żywy Pán / nie ſpádnie y włoſ z głowy jego ná źiemię ; ábowiem zá pomocą Bożą ucżynił to dźiśiáj. A ták wybáwił lud Jonátáná / że nie umárł.
Tedy śię wróćił Saul od Filiſtynów / á Filiſtynowie odeƺli ná miejſce ſwoje.
A Saul otrzymáwƺy króleſtwo nád Izráelem / wálcżył przećiwko okolicżnym wƺyſtkim nieprzyjáćiołom ſwoim / przećiw Moábitom / y przećiw ſynom Ammonowym / y przećiw Edomcżykom / y przećiw królom Sobá / y przećiw Filiſtynom ; á gdźie śię kolwiek obróćił / mężnie śię ſpráwowáł.
Zebráwƺy też wojſko / poráźił Amálekity / y wyrwáł Izráelá z ręki tego / który go puſtoƺył.
A miał Saul ſyny / Jonátáná / y Jeſujego / y Melchiſuá ; á imioná dwóch córek jego : imię pierworodney Merob / á młodƺej Michol ;
A imię żony Saulowej Achinoám / która byłá córką Achimááſową ; á imię Hetmáná wojſká jego Abner / ſyn Nerá / ſtryjá Saulowego.
Bo Cyſ był ojćiec Saulá / á Ner ojćiec Abnerá / ſyn Abijelá.
Y byłá wojná wielká z Filiſtynámi po wƺyſtkie dni Saulowe. Przetoż / gdźiekolwiek widźiáł Saul jákiego śilnego y dźielnego mężá / przyjmowáł go do śiebie.
