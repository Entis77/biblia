A ták wyćiągnął ſtámtąd Dawid / y mieƺkáł ná miejſcách obronnych Engáddy.
Y ſtáło śię / gdy śię wróćił Saul z pogoni zá Filiſtynámi / powiedziáno mu / mówiąc : Oto / Dawid jeſt ná puƺcży Engáddy.
Wziąwƺy tedy Saul trzy tyśiące mężów przebránych ze wƺyſtkiego Izráelá / poƺedł ƺukáć Dawidá y mężów jego / po wierzchu ſkáł kóz dzikich.
Y przyƺedł ku oborom owcżym / które były podle drogi / kędy byłá jáſkiniá ; do której wƺedł Saul ná potrzebę przyrodzoną / á Dawid y mężowie jego śiedzieli po ſtronách jáſkini.
Y rzekli mężowie Dawidowi do niego : Oto dzień / o którym ći powiedziáł Pán : Oto Já dáwám nieprzyjáćielá twego w ręce twoje / á ucżyniƺ mu / jáko śię będzie podobáło w ocżách twoich. Wſtáł tedy Dawid / y urznął po ćichu kráj płáƺcżá Saulowego.
Y ſtáło śię / że uderzyło Dawidá ſerce jego / przeto że urznął kráj płáƺcżá Saulowego.
Y rzekł do mężów ſwoich : Uchowáj mię tego Pánie / żebym to ucżynić miáł pánu memu / pomázáńcowi Páńſkimu / żebym miáł śćiągnąć náń rękę moję / poniewáż jeſt pomázáńcem Páńſkim.
Y przełomił Dawid męże ſwe ſłowy / á nie dopuśćił im powſtáć przećiw Saulowi ; zátem Saul wſtáwƺy z jáſkini / poƺedł w drogę.
Potem też Dawid wſtáł / y wyƺedł z jáſkini / á záwołáł zá Saulem / mówiąc : Królu / Pánie mój! Tedy śię obejrzáł Saul názád / á Dawid ſchyliwƺy śię twárzą ku ziemi / pokłonił śię.
Y rzekł Dawid do Saulá : Cżemuż ſłucháƺ powieśći ludzi mówiących : Oto Dawid ƺuká twego złego?
Oto / dniá tego widzą ocży twoje / że ćię był podáł Pán dziś w ręce moje w jáſkini / y mówiono mi / ábym ćię zábił ; álem ći ſfolgowáł / y rzekłem : Nie śćiągnę ręki mojej ná páná mego ; bo jeſt pomázáńcem Páńſkim.
Oto / ojcże mój / obácż á oglądáj kráj płáƺcżá twego w ręce mojej / że gdym urzynáł kráj płáƺcżá twego / nie zábiłem ćię. Poznáj á obácż / że niemáƺ w ręce mojej złośći y niepráwośći / ánim zgrzeƺył przećiwko tobie : á ty godziƺ ná duƺę moję / ábyś mi ją odjął.
Niech rozſądzi Pán między mną y między tobą / á niech śię zemśći Pán krzywdy mojej nád tobą ; lecż ręká mojá nie będzie ná tobie.
Jáko mówi przypowieść ſtárodáwná : Od niezbożnych wynijdzie niezbożność ; przetoż ręká mojá nie będzie ná tobie.
Zá kimże wżdy wyƺedł król Izráelſki? kogóż goniƺ? pſá zdechłego? pchłę jednę?
Niechże będzie Pán ſędzią / á niech rozſądzi między mną y między tobą / á niech obácży y rozejmie przą moję / á niech mię wyſwobodzi z ręki twojej.
A gdy przeſtáł Dawid mówić ſłów tych do Saulá / rzekł Saul : A twójże to głoſ / ſynu mój Dawidzie? Y podnióſłƺy Saul głoſ ſwój / płákáł.
Y rzekł do Dawidá : Spráwiedliwƺyś ty niźli já : bo tyś mnie oddáł dobrem / á jám tobie oddáł złem.
Tyś záiſte okázáł dziśiáj / żeś mi ucżynił dobre ; bo choć mię podáł Pán w rękę twoję / przećięś mię nie zábił.
Izáż ználázłƺy kto nieprzyjáćielá ſwego / wypuśći go ná drogę dobrą? niechájżeć Pán dobrem oddá zá to / coś mi dziś ucżynił.
A teráz oto wiem / że zápewne będzieƺ królowáł / á oſtoi śię w ręce twojej króleſtwo Izráelſkie.
Przetoż proƺę / przyśięż mi przez Páná / że nie wygubiƺ náśieniá mego po mnie / y nie wytráćiƺ imieniá mego z domu ojcá mego.
A ták przyśiągł Dawid Saulowi. Y odƺedł Saul do domu ſwego / á Dawid y mężowie jego poƺli ná miejſcá obronne.
