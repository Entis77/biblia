Y ſtáło śię / gdy przeſtáł mówić do Saulá / że duƺá Jonátánowá ſpoiłá śię z duƺą Dwidową / y umiłowáł go Jonátán / jáko duƺę ſwoję.
Y wziął go Saul dniá onego / áni mu dopuśćił / żeby śię wrácáł do domu ojcá ſwego.
A ták ucżynił Jonátán z Dawidem przymierze / bo go miłowáł ják duƺę ſwoję.
A zdjáwƺy z śiebie Jonátán płáƺcż / który miáł ná ſobie / dáł go Dawidowi / y ƺáty ſwe / áż do miecżá ſwego / y áż do łuku ſwego / y áż do páſá ſwego rycerſkiego.
Y wychádzáł Dawid do wƺyſtkiego / do cżego go kolwiek poſyłáł Saul / á roztropnie śię ſpráwowáł ; y przełożył go Saul nád rycerſtwem / y był wdzięcżnym w ocżách wƺyſtkiego ludu / tákże y w ocżách ſług Saulowych.
Y ſtáło śię / gdy śię wrácáli / á Dawid śię też wrácáł od poráżki Filiſtynów / że wyƺły niewiáſty ze wƺyſtkich miáſt Izráelſkich / śpiewájąc y grájąc przećiwko Saulowi królowi z bębnámi / z weſelem / y z gęślámi.
A śpiewájąc ná przemiány one niewiáſty / gráły y mówiły : Poráził Saul ſwój tyśiąc / ále Dawid ſwoich dzieśięć tyśięcy.
Y rozgniewáł śię Saul bárdzo / bo śię nie podobáły w ocżách jego te ſłowá ; y rzekł : Przywłáƺcżyli Dawidowi dzieśięć tyśięcy / á mnie przywłáƺcżyli tyśiąc : á cżegoż mu niedoſtáje / jedno króleſtwá?
Przetoż Saul krzywo pátrzył ná Dawidá od onegoż dniá y ná potem.
Y ſtáło śię drugiego dniá / że przypádł Duch Boży zły ná Saulá / y prorokowáł w pośrodku domu / á Dawid gráł ręką ſwoją dniá onego / jáko y przedtem / á Saul miáł włócżnią w ręce ſwej.
Y ćiſnął Saul włócżnią / mówiąc : Przebiję Dawidá áż ku śćiánie ; ále śię uchylił Dawid przed nim po dwá kroć.
Y báł śię Saul Dawidá / przeto że Pán był z nim / á od Saulá odſtąpił.
Y odpráwił go Saul od śiebie / á ucżynił go hetmánem nád tyśiącem / y wychádzáł á wchádzáł przed ludem.
Owá Dawid we wƺyſtkich drogách ſwych roztropnie śię ſpráwowáł ; bo Pán był z nim.
Co gdy widziáł Saul / iż ták bárdzo roztropnie ſobie pocżynáł / báł śię go.
Ale wƺyſtek Izráel y Judá miłowáł Dawidá ; bo on wychádzáł y wchádzáł przed nimi.
Y rzekł Saul do Dawidá : Oto / córkę moję ſtárƺą Merob dám ći zá żonę / jedno bądź mężem mocnym / y odpráwuj wojny Páńſkie ; bo ták Saul ſobie mówił : Niech nie będzie ręká mojá ná nim / ále niech będzie ná nim ręká Filiſtynów.
Tedy rzekł Dawid do Saulá : Któżem já? álbo co zá ſtán mój? álbo co zá dom ojcá mego w Izráelu / żebym był zięćiem królewſkim?
Y ſtáło śię / gdy przyƺedł cżáſ / którego miáłá być dáná Merob / córká Saulowá / Dawidowi / że oná dáná jeſt Adryjelowi Meholátyckiemu zá żonę.
Ale śię rozmiłowáłá Michol / córká Saulowá / Dawidá ; co gdy powiedziáno Saulowi / miło mu to było.
Y rzekł Saul : Dám mu ją / żeby mu byłá śidłem / á żeby byłá ná nim ręká Filiſtynów. Przetoż rzekł Saul do Dawidá : Po drugie będzieƺ zięćiem moim dziśiáj.
Tedy rozkázáł Saul ſługom ſwoim : Rzecżćie do Dawidá potájemnie / mówiąc : Oto / upodobáł ćię ſobie król / y wƺyſcy ſłudzy jego miłują ćię / á ták teráz bądź zięćiem królewſkim.
A gdy mówili ſłudzy Saulowi w uƺy Dawidowe te ſłowá / odpowiedziáł Dawid : Cży śię wám máłá rzecż widzi / być zięćiem królewſkim / gdyżem já jeſt mężem ubogim y podłym?
Tedy ſłudzy Saulowi oznájmili mu / mówiąc : Ták powiedziáł Dawid.
Y rzekł Saul : Ták powiedzćie Dawidowi : Nie dbáć król o wiáno / tylko chce mieć ſto nieobrzezek Filiſtyńſkich / áby śię ſtáłá pomſtá nád nieprzyjáćiołmi królewſkimi ; bo Saul myślił / jákoby Dawidá podáć w ręce Filiſtynom.
Tedy ſłudzy jego powiedzieli te ſłowá Dawidowi ; y ſpodobáło śię to Dawidowi / áby zoſtáł zięćiem królewſkim ; á jeƺcże śię nie były wypełniły dni one.
Wſtáł tedy Dawid / y poƺedł / on y mężowie jego / y zábił z Filiſtynów dwieśćie mężów / y przynióſł Dawid nieobrzezki ich / y oddáno je ſpełná królowi / áby był zięćiem królewſkim. A ták dáł mu Saul Michol / córkę ſwą zá żonę.
A widząc Saul / y bácżąc / że Pán był z Dawidem / á iż Michol / córká jego / miłowáłá go /
Tem więcej Saul obáwiáł śię Dawidá / y ſtáł śię Saul nieprzyjáćielem Dawidowi po wƺyſtkie dni.
Y wpádáły kśiążętá Filiſtyńſkie do ziemi. A kiedykolwiek wpádáły / roztropniej ſobie pocżynáł Dawid nád wƺyſtkie ſługi Saulowe ; przetoż ſłáwne było imię jego bárdzo.
