Tedy rzekł Pán do Sámuelá : Y pókiż ty będźieƺ żáłował Saulá / gdyżem go Ja odrzućił / áby nie królował nád Izráelem? Nápełnij róg twój oliwą / á pójdź / poślę ćię do Iſájego Betlehemcżyká ; bom tám ſobie upátrzył miedzy ſyny jego królá.
Y rzekł Sámuel : Jákoż mam iść? Bo uſłyƺy Saul / y zábije mię. Y odpowiedźiał Pán : Weźmij z ſobą jáłowicę z ſtádá / y rzecżeƺ : Przyƺedłem / ábym ofiárował Pánu.
Y wezwieƺ Iſájego ná ofiárę / á Ja tobie oznájmię / co máƺ cżynić ; y pomáżeƺ mi tego / o którym ći powiem.
Y ucżynił Sámuel / jáko mu powiedźiał Pán / á przyƺedł do Betlehem ; á ulękƺy śię ſtárśi miáſtá / zábieżeli mu / y mówili : Spokojneli jeſt przyśćie twoje?
Y rzekł : Spokojne ; przyƺedłem / ábym ofiárował Pánu. Poświęććież śię / á pójdźćie ze mną ná ofiárę. Y poświęćił Iſájego / y ſyny jego / á wezwał ich ná ofiárę.
A gdy przyƺli / ujrzał Elijábá / y rzekł : Záiſte ten jeſt przed Pánem pomázániec jego.
Lecż rzekł Pán do Sámuelá : Nie pátrz ná urodę jego áni ná wyſokość wzroſtá jego / gdyżem go odrzućił. Abowiem Ja nie pátrzę ná to / ná co pátrzy cżłowiek ; bo cżłowiek pátrzy ná to / co jeſt przed ocżymá / ále Pán pátrzy ná ſerce.
Záwołał tedy Iſáj Abinádábá / y kázał mu iść przed Sámuelá ; który rzekł : Y tego nie obrał Pán.
Potem kázał przyść Iſáj Sámmie / á on rzekł : Y tego nie obrał Pán.
Tedy kázał przyść Iſáj śiedmiu ſynom ſwoim przed Sámuelá. Y rzekł Sámuel do Iſájego : Nie obrał Pán y tych.
Potem rzekł Sámuel do Iſájego : Wƺyſcyż to już ſynowie? Odpowiedźiał : Jeƺcże zoſtał nájmłodƺy / który páśie owce. Tedy rzekł Sámuel do Iſájego : Poślijże / á przywiedź go ; boć nie uśiądźiemy / áż on tu przydźie.
A ták poſłał y przywiódł go ; á on był liſowáty / y wdźięcżnych ocżu / á piękny ná wejrzeniu. Tedy rzekł Pán : Wſtáń / á pomáż go / boć ten jeſt.
Wźiąwƺy tedy Sámuel róg z oliwą / pomázał go w pośród bráći jego. Y zoſtał Duch Páńſki nád Dawidem od onegoż dniá / y ná potem. Zátem Sámuel wſtał / y poƺedł do Rámáty.
A Duch Páńſki odſtąpił od Saulá / y trwożył go duch zły od Páná.
Y rzekli ſłudzy Saulowi do niego : Oto teraz Duch Boży zły trwoży ćię :
Niech rozkáże Pán náƺ / á ſłudzy twoi / którzy ſą przed tobą / poƺukáją mężá / coby umiał gráć ná hárfie / że gdy ćię nápádnie Duch Boży zły / zágrá ręką ſwą / á ulżyć śię.
Rzekł tedy Saul do ſług ſwojich : Upátrzćie mi proƺę mężá / coby dobrze gráć umiał / á przywiedźćie do mnie.
Y odpowiedźiał jeden z ſług / y rzekł : Otom widźiał ſyná Iſájego Betlehemcżyká / który umie gráć / á jeſt cżłowiek mężny / y rycerſki / y ſpráwny / y głádki / á Pán jeſt z nim.
Poſłał tedy Saul poſły do Iſájego / mówiąc : Poślij do mnie Dawidá / ſyná twego / który jeſt przy ſtádźie.
Tedy wźiąwƺy Iſáj oſłá / chleb / y fláƺkę winá / wźiąwƺy y koźiołká jednego z ſtádá / poſłał przez Dawidá / ſyná ſwego / Saulowi.
A gdy przyƺedł Dawid do Saulá / ſtánął przed nim ; á rozmiłował śię go bárzo / y był u niego zá wyroſtká broń noƺącego.
Y poſłał Saul do Iſájego / mówiąc : Niech ſtoi proƺę Dawid przedemną ; bo ználázł łáſkę w ocżách mojich.
Y bywáło / gdy przychodźił Duch Boży ná Saulá / że wźiąwƺy Dawid hárfę / grał ręką ſwą ; tedy Saul miał ulżenie / y lepiej śię miał / bo odchodźił od niego on duch zły.
