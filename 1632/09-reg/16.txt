Tedy rzekł Pán do Sámuelá : Y pókiż ty będźieƺ żáłowáł Saulá / gdyżem go Ja odrzućił / áby nie królowáł nád Izráelem? Nápełnij róg twój oliwą / á pójdź / poślę ćię do Iſájego Betlehemcżyká ; bom tám ſobie upátrzył między ſyny jego królá.
Y rzekł Sámuel : Jákoż mam iść? Bo uſłyƺy Saul / y zábije mię. Y odpowiedźiáł Pán : Weźmij z ſobą jáłowicę z ſtádá / y rzecżeƺ : Przyƺedłem / ábym ofiárowáł Pánu.
Y wezwieƺ Iſájego ná ofiárę / á Ja tobie oznájmię / co máƺ cżynić ; y pomáżeƺ mi tego / o którym ći powiem.
Y ucżynił Sámuel / jáko mu powiedźiáł Pán / á przyƺedł do Betlehem ; á ulękłƺy śię ſtárśi miáſtá / zábieżeli mu / y mówili : Spokojneli jeſt przyjśćie twoje?
Y rzekł : Spokojne ; przyƺedłem / ábym ofiárowáł Pánu. Poświęććież śię / á pójdźćie ze mną ná ofiárę. Y poświęćił Iſájego / y ſyny jego / á wezwáł ich ná ofiárę.
A gdy przyƺli / ujrzáł Elijábá / y rzekł : Záiſte ten jeſt przed Pánem pomázániec jego.
Lecż rzekł Pán do Sámuelá : Nie pátrz ná urodę jego áni ná wyſokość wzroſtá jego / gdyżem go odrzućił. Abowiem Ja nie pátrzę ná to / ná co pátrzy cżłowiek ; bo cżłowiek pátrzy ná to / co jeſt przed ocżymá / ále Pán pátrzy ná ſerce.
Záwołáł tedy Iſáj Abinádábá / y kázáł mu iść przed Sámuelá ; który rzekł : Y tego nie obráł Pán.
Potem kázáł przyjść Iſáj Sámmie / á on rzekł : Y tego nie obráł Pán.
Tedy kázáł przyjść Iſáj śiedmiu ſynom ſwoim przed Sámuelá. Y rzekł Sámuel do Iſájego : Nie obráł Pán y tych.
Potem rzekł Sámuel do Iſájego : Wƺyſcyż to już ſynowie? Odpowiedźiáł : Jeƺcże zoſtáł nájmłodƺy / który páśie owce. Tedy rzekł Sámuel do Iſájego : Poślijże / á przywiedź go ; boć nie uśiądźiemy / áż on tu przyjdźie.
A ták poſłáł y przywiódł go ; á on był liſowáty / y wdźięcżnych ocżu / á piękny ná wejrzeniu. Tedy rzekł Pán : Wſtáń / á pomáż go / boć ten jeſt.
Wźiąwƺy tedy Sámuel róg z oliwą / pomázáł go w pośród bráći jego. Y zoſtáł Duch Páńſki nád Dawidem od onegoż dniá / y ná potem. Zátem Sámuel wſtáł / y poƺedł do Rámáty.
A Duch Páńſki odſtąpił od Saulá / y trwożył go duch zły od Páná.
Y rzekli ſłudzy Saulowi do niego : Oto teraz Duch Boży zły trwoży ćię :
Niech rozkáże Pán náƺ / á ſłudzy twoi / którzy ſą przed tobą / poƺukáją mężá / coby umiáł gráć ná hárfie / że gdy ćię nápádnie Duch Boży zły / zágrá ręką ſwą / á ulżyć śię.
Rzekł tedy Saul do ſług ſwoich : Upátrzćie mi proƺę mężá / coby dobrze gráć umiáł / á przywiedźćie do mnie.
Y odpowiedźiáł jeden z ſług / y rzekł : Otom widźiáł ſyná Iſájego Betlehemcżyká / który umie gráć / á jeſt cżłowiek mężny / y rycerſki / y ſpráwny / y głádki / á Pán jeſt z nim.
Poſłáł tedy Saul poſły do Iſájego / mówiąc : Poślij do mnie Dawidá / ſyná twego / który jeſt przy ſtádźie.
Tedy wźiąwƺy Iſáj oſłá / chleb / y fláƺkę winá / wźiąwƺy y koźiołká jednego z ſtádá / poſłáł przez Dawidá / ſyná ſwego / Saulowi.
A gdy przyƺedł Dawid do Saulá / ſtánął przed nim ; á rozmiłowáł śię go bárdzo / y był u niego zá wyroſtká broń noƺącego.
Y poſłáł Saul do Iſájego / mówiąc : Niech ſtoi proƺę Dawid przedemną ; bo ználázł łáſkę w ocżách moich.
Y bywáło / gdy przychodźił Duch Boży ná Saulá / że wźiąwƺy Dawid hárfę / gráł ręką ſwą ; tedy Saul miał ulżenie / y lepiej śię miał / bo odchodźił od niego on duch zły.
