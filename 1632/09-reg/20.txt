Ale Dawid ućiekłƺy z Nájotu / który jeſt w Rámáćie / przyƺedł / y mówił przed Jonátánem : Cóżem ucżynił? co zá niepráwość mojá? y co zá grzech mój przećiw ojcu twemu / że ƺuká duƺy mojej?
Który mu odpowiedźiáł : Boże uchowáj! nie umrzeƺ ; oto nie cżyni ojćiec mój nic wielkiego álbo máłego / áż mi pierwej oznájmi ; ázáżby táić miáł ojćiec mój przedemną y tego? Nie ucżyni tego.
A nádto przyśiągł Dawid / rzekłƺy : Wie záiſte ojćiec twój / żem ználázł łáſkę w ocżách twoich / y myśli : Niech o tem niewie Jonátán / by śię ſnáć nie fráſowáł ; y owƺem jáko żywy Pán / żywá y duƺá twojá / że tylko krok jeden jeſt między mną / y między śmierćią.
Y odpowiedźiáł Jonátán Dawidowi : Co mi kolwiek rzecże duƺá twojá / ucżynięć.
Tedy rzekł Dawid do Jonátáná : Oto / nów mieśiącá jutro / á jám zwykł śiádáć z królem przy ſtole ; puść mię tedy / że śię ſkryję ná polu áż do wiecżorá trzećiego dniá.
A jeźliby śię pilnie pytáł o mnie ojćiec twój / rzecżeƺ : Prośił mię bárdzo Dawid / áby ƺedł do Betlehem / miáſtá ſwego ; bo tám ofiárę urocżyſtą má ſpráwowáć wƺyſtká rodźiná jego.
Jeźli ták rzecże : Dobrze / pokój będźie ſłudze twemu ; ále jeźli śię rozgniewá / wiedz / iż śię dopełniłá złość jego.
Przetoż ucżyń miłośierdźie nád ſługą twoim / gdyżeś w przymierze Páńſkie przywiódł z ſobą ſługę twego ; á jeźli we mnie jeſt niepráwość / ty mię zábij ; á do ojcá twego przecżbyś mię miáł wodźić?
Y rzekł Jonátán : Boże ćię tego uchowáj ; bo jeźli śię pewnie dowiem / że śię dopełniłá złość ojcá mego / áby przyƺłá przećiw tobie / izálibym ći tego nie oznájmił?
Y rzekł Dawid do Jonátáná : Któż mi oznájmi / jeźliżeć co odpowie ojćiec twój przykrego?
Odpowiedźiáł Jonátán Dawidowi : Pójdź / á wynijdźmy ná pole. Y wyƺli obáj ná pole.
Tedy rzekł Jonátán do Dawidá : Pán / Bóg Izráelſki / ( ſkoro śię wywiem o woli ojcá mego o tym cżáśie jutro / álbo dniá trzećiego / á będźie co dobrego o Dawidźie / á jeźli zárázem nie poślę do ćiebie / y nie oznájmięć / )
To niech ucżyni Pán / Bóg Izráelſki / mówię / Jonátánowi / y to niech przycżyni. A jeźliże będźie chćiáł ojćiec mój przywieść złe ná ćię / y toć objáwię / y puƺcżę ćię / ábyś ƺedł w pokoju / á niech będźie Pán z tobą / jáko był z ojcem moim.
Tákże y ty / będęli żyw / y ty mówię ucżyniƺ ze mną miłośierdźie Páńſkie ; á choćbym y umárł /
Przećię nie oddáliƺ miłośierdźiá twego od domu mego áż ná wieki / áni gdy wykorzeni Pán nieprzyjáćioły Dawidowe wƺyſtkie z źiemi.
Y ucżynił Jonátán przymierze z domem Dawidowym / mówiąc : Niech tego ƺuká Pán z ręki nieprzyjáćiół Dawidowych.
Nádto jeƺcże Jonátán przyśiągł Dawidowi przez miłość / którą go miłowáł ; bo jáko miłowáł duƺę ſwoję / ták go też miłowáł.
Y rzekł do niego Jonátán : Jutro nów mieśiącá / á będą śię pytáć o tobie / poniewáż próżne będźie miejſce twoje.
Przetoż przez trzy dni będźieƺ śię ukrywáł / y zſtąpiƺ prędko / á przyjdźieƺ ná miejſce / gdźieś śię był ukrył / gdy byłá ſpráwá o tobie / á będźieƺ śiedźiáł u kámieniá Ezel.
A já wyſtrzelę trzy ſtrzáły po bok jego / zmierzájąc ſobie do celu.
A potem poślę chłopcá / mówiąc mu : Idź / nájdźij ſtrzáły. A jeźli rzekę chłopcu : Owo ſtrzáły zá tobą ſám bliżej / przynieś je / tedy przyjdź ; bo máƺ pokój / y nie ſtánieć śię nic złego / jáko żywy Pán.
Ale jeźliż rzekę chłopcu : Oto ſtrzáły przed tobą tám dálej ; idź / bo ćię wypuśćił Pán.
A tego / o cżemeśmy mówili já y ty / tego Pán świádkiem będźie między mną á między tobą áż ná wieki.
A ták ſkrył śię Dawid w polu. A gdy przyƺedł nów mieśiącá / śiádł król do ſtołu / áby jádł.
A gdy uśiádł król ná ſtolicy ſwojej według zwycżáju / ná ſtolicy przy śćiánie / powſtáł Jonátán ; y śiádł Abner podle Saulá / á zoſtáło próżne miejſce Dawidowe.
Lecż nie rzekł Saul nic onego dniá / bo myśláł : Przydáło mu śię podobno coś / lub jeſt cżyſtym lub niecżyſtym.
A gdy było názájutrz dniá wtórego po nowiu mieśiącá / było záś próżne miejſce Dawidowe. Y rzekł Saul do Jonátáná / ſyná ſwego : Cżemuż nie przyƺedł ſyn Iſájego / áni wcżoráj / áni dźiś do ſtołu?
Odpowiedźiáł Jonátán Saulowi : Uśilnie mię prośił Dawid / áby ƺedł do Betlehem ;
Y mówił : Puść mię proƺę / bo ſpráwuje ofiárę rodźiná náƺá w mieśćie ; támże mię wezwáł brát mój. A ták teráz jeźlim ználázł łáſkę w ocżách twoich / pójdę proƺę / y oglądám bráćią moję ; dlá tegoć nie przyƺedł do ſtołu królewſkiego.
Y zápálił śię gniewem Saul ná Jonátáná / y rzekł mu : Synu złośliwy / á upornej mátki / ázáż nie wiem / iżeś ſobie obráł ſyná Iſájego ku zelżywośći twojej / y ku poháńbieniu y ſromoćie mátki twojej?
Bo po wƺyſtkie dni / których ſyn Iſájego będźie żył ná źiemi / nie będźieƺ umocniony / ty y króleſtwo twoje ; á ták teráz poślij / á przywiedź go do mnie / bo jeſt godźien śmierći.
Tedy odpowiedźiáł Jonátán Saulowi / ojcu ſwemu / y rzekł do niego : Przecż má umrzeć? cóż ucżynił?
Y ćiſnął Saul włócżnią ná niego / áby go przebił. Tedy poznáł Jonátán / że koniecżnie ojćiec jego umyślił zábić Dawidá.
Y wſtáł Jonátán od ſtołu z wielkim gniewem / y nie jádł dniá wtórego po nowiu mieśiącá chlebá ; bo śię záfráſowáł o Dawidá / á iż go zelżył ojćiec jego.
A ráno wyƺedł Jonátán ná pole według cżáſu poſtánowionego z Dawidem / y chłopiec máły z nim.
Y rzekł do chłopcá ſwego : Bież / ƺukáj prędko ſtrzáł / które já wyſtrzelę. Tedy chłopiec bieżáł ; á on wyſtrzelił ſtrzáły dálej przedeń.
A gdy przyƺedł chłopiec áż ná miejſce ſtrzáły / którą był wyſtrzelił Jonátán / záwołáł Jonátán zá chłopcem / y rzekł : Azáż ſtrzáłá nie jeſt zá tobą tám dálej?
Y wołáł Jonátán zá chłopcem : Spieƺ śię co nájrychlej / nie ſtój. Tedy zebráwƺy chłopiec Jonátáná ſtrzáły / przyƺedł do páná ſwego.
( Ale chłopiec nic nie wiedźiáł / tylko Jonátán y Dawid wiedźieli / co śię dźiáło. )
Y dáł Jonátán oręż ſwój chłopcu / który z nim był / y rzekł mu : Idź / odnieś do miáſtá.
A gdy odƺedł chłopiec / Dawid wſtáł od ſtrony południowej / y upádłƺy twárzą ſwoją ná źiemię / ukłonił śię po trzy kroć / y pocáłowáwƺy jeden drugiego / płákáli poſpołu ; ále Dawid obfićiej.
Y rzekł Jonátán do Dawidá : Idź w pokoju ; á to / cośmy ſobie obáj przyśięgli przez imię Páńſkie / mówiąc : Pán niech będźie między mną y między tobą / y między náśieniem mojem / y między náśieniem twojim świádkiem áż ná wieki / trzymáć będźiemy. 
A ták wſtáwƺy Dawid odƺedł / á Jonátán wƺedł do miáſtá.
