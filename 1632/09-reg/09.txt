Y był mąż z pokoleniá Benjámin / którego imię było Cyſ / ſyn Abijelów / ſyná Seror / ſyná Bechorát / ſyná Afijá / ſyná mężá Jemini / duży w śiłę.
Ten miáł ſyná imieniem Saulá / młodzieńcá urodziwego / á nie było nikogo z ſynów Izráelſkich urodziwƺego nádeń ; głową był wyżƺy nád wƺyſtek inny lud.
A zginęły były oślice Cyſowi / ojcu Saulowemu. Y rzekł Cyſ do Saulá / ſyná ſwego : Weźmij teráz z ſobą jednego z ſług / á wſtáwƺy idź / y ƺukáj oślic.
Tedy on ƺedł przez górę Efráim / y przeƺedł ziemię Sáliſá / lecż nie ználeźli. Przeƺli tákże ziemię Sálim / á nie ználeźli. Nádto przeƺli y ziemię Jemini / á nie ználeźli.
A przyƺedłƺy do ziemi Suf / rzekł Saul do ſługi ſwego / który był z nim : Pójdź / á wróćmy śię / by ſnáć zániechwáƺy ojćiec mój oślic / nie fráſowáł śię o náſ /
Który mu odpowiedziáł : Oto teráz jeſt mąż Boży w tem mieśćie / á mąż to zácny ; cokolwiek powie / wƺyſtko śię ſtáwá ; przetoż pójdźmy tám / ſnáć nám powie o drodze náƺej / którą iść mámy.
Tedy odpowiedziáł Saul ſłudze ſwemu : Więc pójdziemy ; ále cóż przynieśiemy onemu mężowi? Bo chlebá nie ſtáło w ſumkách náƺych / á podárku niemáƺ / którybyśmy przynieśli mężowi Bożemu ; cóż mámy?
Tedy ſługá znowu odpowiedziáł Saulowi / y rzekł : Otom ználázł u śiebie cżwártą cżęść ſyklá ſrebrnego / którą dámy mężowi Bożemu / áby nám oznájmił drogę náƺę.
Przedtem w Izráelu ták máwiáł káżdy / gdy śię ƺedł rádzić Bogá : Chodźćie / á pójdziemy áż do widzącego ; bo którego dziś zowią prorokiem / tego przedtem názywáno widzącym.
Tedy rzekł Saul do ſługi ſwego : Dobre jeſt ſłowo twoje ; chodź / pójdźmy ; y ƺli do miáſtá / w którym był mąż Boży.
A gdy wſtępowáli ná górę miáſtá / á potkáli dziewecżki / wychodzące cżerpáć wodę / rzekli im : A jeſtże tu widzący?
Które odpowiádájąc im / rzekły : Jeſt / oto przed tobą ; ſpieƺ śię tedy / dziś bowiem przyƺedł do miáſtá / gdyż dziś ofiáry ſpráwuje lud ná górze.
Skoro wnijdziećie do miáſtá / znájdziećie go / pierwej niż pójdzie ná górę / áby jádł ; ábowiem lud nie będzie jádł / áż on przyjdzie ; bo on będzie błogoſłáwił ofierze / potem będą jeść wezwáni. A przetoż idźćie / bo go o tej godzinie znájdziećie.
Weƺli tedy do miáſtá ; á gdy przyƺli w pośrodek miáſtá / oto / Sámuel wychodził przećiwko nim / áby ƺedł ná górę.
A Pán objáwił był Sámuelowi dzień przedtem / niźli Saul przyƺedł / mówiąc :
O tym cżáśie jutro poślę do ćiebie mężá z ziemi Benjámin / którego pomáżeƺ zá wodzá nád ludem moim Izráelſkim ; á on wybáwi lud mój z rąk Filiſtyńſkich. Bom wejrzáł ná lud mój / gdyż przyƺło wołánie jego do mnie.
A gdy Sámuel wejrzáł ná Saulá / rzekł mu Pán. Otóż mąż / o którymemći powiedziáł ; tenći będzie pánowáł nád ludem moim.
A ták przyſtąpił Saul do Sámuelá w pośrodku brámy / y rzekł : Proƺę powiedz mi / gdzie tu jeſt dom widzącego?
Y odpowiedziáł Sámuel / Saulowi / mówiąc : Jám jeſt widzący. Wſtąp przedemną ná górę / á będziećie dziś jedli ze mną ; potem ćię odpráwię ráno / á cokolwiek jeſt w ſercu twem / oznájmię tobie.
A o oślice / któreć zginęły dziś trzeći dzień / nie fráſuj śię / boć śię ználázły.Y cżyjeż wƺyſtko co nájlepƺego w Izráelu? izáli nie twoje / y nie wƺyſtkiego domu ojcá twego?
A odpowiádájąc Saul / rzekł : Izálim já nie ſyn Jemini z nájmniejƺego pokoleniá Izráelſkiego? á dom mój ázáż nie nájpodlejƺy między wƺyſtkiemi domy pokoleniá Benjáminowego? Przecżżeś tedy mówił do mnie tákowe ſłowá?
A ták wziąwƺy Sámuel Saulá y ſługę jego / wywiódł je ná ſálę / y dáł im miejſce przedniejƺe między wezwánymi / których było około trzydzieſtu mężów.
Y rzekł Sámuel kuchárzowi : Dáj ſám tę cżąſtkę / którąm ći dáł / y o którejm ći rzekł : Schowáj ją u śiebie.
A gdy przynióſł kuchárz łopátkę / y to / co było ná niej / położył Sámuel przed Saulá / y rzekł : Oto / co zoſtáło / weźmij przed śię / á jedz ; bo ná ten cżáſ ſchowáno to dlá ćiebie / gdym rzekł : Wezwáłem ludu. Y jádł Saul z Sámuelem dniá onego.
A gdy zſtąpili z góry do miáſtá / rozmáwiáł z Saulem ná dáchu.
Potem wſtáli bárdzo ráno. Y ſtáło śię / gdy śię pocżęło rozedniewáć / záwołáł Sámuel Saulá ná dách / mówiąc : Wſtáń / á odpráwię ćię ; wſtáwƺy tedy Saul / wyƺli obáj z domu / on y Sámuel.
A gdy ſchodzili ku końcowi miáſtá / rzekł Sámuel do Saulá : Rzecż ſłudze / áby ƺedł przed námi / y ƺedł ; á ty pozoſtáń trochę / żeć opowiem ſłowo Boże.
