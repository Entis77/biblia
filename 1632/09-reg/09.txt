Y był mąż z pokoleniá Benjámin / którego imię było Cyſ / ſyn Abijelów / ſyná Seror / ſyná Bechorát / ſyná Afijá / ſyná mężá Jemini / duży w śiłę.
Ten miał ſyná imieniem Saulá / młodźieńcá urodźiwego / á nie było nikogo z ſynów Izráelſkich urodźiwƺego nádeń ; głową był wyżƺy nád wƺyſtek inny lud.
A zginęły były oślice Cyſowi / ojcu Saulowemu. Y rzekł Cyſ do Saulá / ſyná ſwego : Weźmij teraz z ſobą jednego z ſług / á wſtáwƺy idź / y ƺukáj oślic.
Tedy on ƺedł przez górę Efráim / y przeƺedł źiemię Sáliſá / lecż nie ználeſli. Przeƺli tákże źiemię Sálim / á nie ználeſli. Nádto przeƺli y źiemię Jemini / á nie ználeſli.
A przyƺedłƺy do źiemi Suf / rzekł Saul do ſługi ſwego / który był z nim : Pójdź / á wróćmy śię / by ſnáć zániechwáƺy ojćiec mój oślic / nie fráſowáł śię o nas /
Który mu odpowiedźiáł : Oto teraz jeſt mąż Boży w tem mieśćie / á mąż to zácny ; cokolwiek powie / wƺyſtko śię ſtáwá ; przetoż pójdźmy tám / ſnáć nám powie o drodze náƺej / którą iść mámy.
Tedy odpowiedźiáł Saul ſłudze ſwemu : Więc pójdźiemy ; ále cóż przynieśiemy onemu mężowi? Bo chlebá nie ſtáło w ſumkách náƺych / á podárku niemáƺ / którybyśmy przynieśli mężowi Bożemu ; cóż mámy?
Tedy ſługá znowu odpowiedźiáł Saulowi / y rzekł : Otom ználázł u śiebie cżwártą cżęść ſyklá ſrebrnego / którą dámy mężowi Bożemu / áby nám oznájmił drogę náƺę.
Przedtem w Izráelu ták máwiáł káżdy / gdy śię ƺedł rádźić Bogá : Chodźćie / á pójdźiemy áż do widzącego ; bo którego dźiś zowią prorokiem / tego przedtem názywáno widzącym.
Tedy rzekł Saul do ſługi ſwego : Dobre jeſt ſłowo twoje ; chodź / pójdźmy ; y ƺli do miáſtá / w którym był mąż Boży.
A gdy wſtępowáli ná górę miáſtá / á potkáli dźiewecżki / wychodzące cżerpáć wodę / rzekli im : A jeſtże tu widzący?
Które odpowiádájąc im / rzekły : Jeſt / oto przed tobą ; ſpieƺ śię tedy / dźiś bowiem przyƺedł do miáſtá / gdyż dźiś ofiáry ſpráwuje lud ná górze.
Skoro wnijdźiećie do miáſtá / znájdźiećie go / pierwej niż pójdźie ná górę / áby jádł ; ábowiem lud nie będźie jádł / áż on przyjdźie ; bo on będźie błogoſłáwił ofierze / potem będą jeść wezwáni. A przetoż idźćie / bo go o tey godźinie znájdźiećie.
Weƺli tedy do miáſtá ; á gdy przyƺli w pośrodek miáſtá / oto / Sámuel wychodźił przećiwko nim / áby ƺedł ná górę.
A Pán objáwił był Sámuelowi dźień przedtem / niźli Saul przyƺedł / mówiąc :
O tym cżáśie jutro poślę do ćiebie mężá z źiemi Benjámin / którego pomáżeƺ zá wodzá nád ludem moim Izráelſkim ; á on wybáwi lud mój z rąk Filiſtyńſkich. Bom wejrzáł ná lud mój / gdyż przyƺło wołánie jego do mnie.
A gdy Sámuel wejrzáł ná Saulá / rzekł mu Pán. Otóż mąż / o którymemći powiedźiáł ; tenći będźie pánowáł nád ludem moim.
A ták przyſtąpił Saul do Sámuelá w pośrodku brámy / y rzekł : Proƺę powiedz mi / gdźie tu jeſt dom widzącego?
Y odpowiedźiáł Sámuel / Saulowi / mówiąc : Jám jeſt widzący. Wſtąp przedemną ná górę / á będźiećie dźiś jedli ze mną ; potem ćię odpráwię ráno / á cokolwiek jeſt w ſercu twem / oznájmię tobie.
A o oślice / któreć zginęły dźiś trzeći dźień / nie fráſuj śię / boć śię ználázły.Y cżyjeż wƺyſtko co nájlepƺego w Izráelu? izáli nie twoje / y nie wƺyſtkiego domu ojcá twego?
A odpowiádájąc Saul / rzekł : Izálim ja nie ſyn Jemini z nájmniejƺego pokoleniá Izráelſkiego? á dom mój ázáż nie nájpodlejƺy między wƺyſtkiemi domy pokoleniá Benjáminowego? Przecżżeś tedy mówił do mnie tákowe ſłowá?
A ták wźiąwƺy Sámuel Saulá y ſługę jego / wywiódł je ná ſálę / y dáł im miejſce przedniejƺe między wezwánymi / których było około trzydźieſtu mężów.
Y rzekł Sámuel kuchárzowi : Dáj ſám tę cżąſtkę / którąm ći dáł / y o którejm ći rzekł : Schowáj ją u śiebie.
A gdy przynióſł kuchárz łopátkę / y to / co było ná niej / położył Sámuel przed Saulá / y rzekł : Oto / co zoſtáło / weźmij przed śię / á jedz ; bo ná ten cżáſ ſchowáno to dla ćiebie / gdym rzekł : Wezwáłem ludu. Y jádł Saul z Sámuelem dniá onego.
A gdy zſtąpili z góry do miáſtá / rozmáwiáł z Saulem ná dáchu.
Potem wſtáli bárdzo ráno. Y ſtáło śię / gdy śię pocżęło rozedniewáć / záwołáł Sámuel Saulá ná dách / mówiąc : Wſtáń / á odpráwię ćię ; wſtáwƺy tedy Saul / wyƺli obáj z domu / on y Sámuel.
A gdy ſchodźili ku końcowi miáſtá / rzekł Sámuel do Saulá : Rzecż ſłudze / áby ƺedł przed námi / y ƺedł ; á ty pozoſtáń trochę / żeć opowiem ſłowo Boże.
