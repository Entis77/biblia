Y był mąż z pokolenia Benjámin / którego imię było Cys / ſyn Abijelów / ſyná Seror / ſyná Bechorát / ſyná Afijá / ſyná mężá Jemini / duży w śiłę.
Ten miał ſyná imieniem Saulá / młodźieńcá urodźiwego / á nie było nikogo z ſynów Izráelſkich urodźiwƺego nádeń ; głową był wyżƺy nád wƺyſtek inny lud.
A zginęły były oślice Cyſowi / ojcu Saulowemu. Y rzekł Cys do Saulá / ſyná ſwego : Weźmij teraz z ſobą jednego z ſług / á wſtáwƺy idź / y ƺukáj oślic.
Tedy on ƺedł przez górę Efráim / y przeƺedł źiemię Sáliſá / lecż nie ználeſli. Przeƺli tákże źiemię Sálim / á nie ználeſli. Nádto przeƺli y źiemię Jemini / á nie ználeſli.
A przyƺedƺy do źiemi Suf / rzekł Saul do ſługi ſwego / który był z nim : Pójdź / á wróćmy śię / by ſnadź zániecháwƺy oćiec mój oślic / nie fráſował śię o nas /
Który mu odpowiedźiał : Oto teraz jeſt mąż Boży w tym mieśćie / á mąż to zácny ; cokolwiek powie / wƺyſtko śię ſtáwá ; przetoż pójdźmy tám / ſnadź nam powie o drodze náƺej / którą iść mamy.
Tedy odpowiedźiał Saul ſłudze ſwemu : Więc pójdźiemy ; ále cóż przynieśiemy onemu mężowi? Bo chlebá nie ſtáło w ſumkách náƺych / á podárku niemáƺ / którybyſmy przynieśli mężowi Bożemu ; cóż mamy?
Tedy ſługá znowu odpowiedźiał Saulowi / y rzekł : Otom ználázł u śiebie cżwártą cżęść ſyklá ſrebrnego / którą dámy mężowi Bożemu / áby nam oznájmił drogę náƺę.
Przed tym w Izráelu ták máwiał káżdy / gdy śię ƺedł rádźić Bogá : Chodźćie / á pójdźiemy áż do widzącego ; bo którego dźiś zowią prorokiem / tego przed tym názywano widzącym.
Tedy rzekł Saul do ſługi ſwego : Dobre jeſt ſłowo twoje ; chodź / pójdźmy ; y ƺli do miáſtá / w którym był mąż Boży.
A gdy wſtępowáli ná górę miáſtá / á potkáli dźiewecżki / wychodzące cżerpáć wodę / rzekli im : A jeſtże tu widzący?
Które odpowiádájąc im / rzekły : Jeſt / oto przed tobą ; ſpieƺ śię tedy / dźiś bowiem przyƺedł do miáſtá / gdyż dźiś ofiáry ſpráwuje lud ná górze.
Skoro wnijdźiećie do miáſtá / znájdźiećie go / pierwej niż pójdźie ná górę / áby jádł ; ábowiem lud nie będźie jádł / áż on przydźie ; bo on będźie błogoſłáwił ofierze / potem będą jeść wezwáni. A przetoż idźćie / bo go o tey godźinie znájdźiećie.
Weƺli tedy do miáſtá ; á gdy przyƺli w pośrodek miáſtá / oto / Sámuel wychodźił przećiwko nim / áby ƺedł ná górę.
A Pán objáwił był Sámuelowi dźień przed tym / niżli Saul przyƺedł / mówiąc :
O tym cżáśie jutro poślę do ćiebie mężá z źiemi Benjámin / którego pomáżeƺ zá wodzá nád ludem moim Izráelſkim ; á on wybáwi lud mój z rąk Filiſtyńſkich. Bom wejrzał ná lud mój / gdyż przyƺło wołánie jego do mnie.
A gdy Sámuel wejrzał ná Saulá / rzekł mu Pán. Otóż mąż / o którymemći powiedźiał ; tenći będźie pánował nád ludem moim.
A ták przyſtąpił Saul do Sámuelá w pośrzodku bramy / y rzekł : Proƺę powiedz mi / gdźie tu jeſt dom widzącego?
Y odpowiedźiał Sámuel / Saulowi / mówiąc : Jam jeſt widzący. Wſtąp przedemną ná górę / á będźiećie dźiś jedli ze mną ; potem ćię odpráwię ráno / á cokolwiek jeſt w ſercu twem / oznájmię tobie.
A o oślice / któreć zginęły dźiś trzeći dźień / nie fráſuj śię / boć śię ználázły. Y cżyjeż wƺyſtko co nájlepƺego w Izráelu? izali nie twoje / y nie wƺyſtkiego domu ojcá twego?
A odpowiádájąc Saul / rzekł : Izálim ja nie ſyn Jemini z nájmniejƺego pokolenia Izráelſkiego? á dom mój ázaż nie nájpodlejƺy miedzy wƺyſtkiemi domy pokolenia Benjáminowego? Przecżżeś tedy mówił do mnie tákowe ſłowá?
A ták wźiąwƺy Sámuel Saulá y ſługę jego / wywiódł je ná ſálę / y dał im miejſce przedniejƺe miedzy wezwánymi / których było około trzydźieſtu mężów.
Y rzekł Sámuel kuchárzowi : Dáj ſám tę cżąſtkę / którąm ći dał / y o którejm ći rzekł : Schowáj ją u śiebie.
A gdy przynióſł kuchárz łopátkę / y to / co było ná niej / położył Sámuel przed Saulá / y rzekł : Oto / co zoſtáło / weźmij przed śię / á jedz ; bo ná ten cżás ſchowano to dla ćiebie / gdym rzekł : Wezwáłem ludu. Y jádł Saul z Sámuelem dniá onego.
A gdy zſtąpili z góry do miáſtá / rozmáwiał z Saulem ná dáchu.
Potem wſtáli bárzo ráno. Y ſtáło śię / gdy śię pocżęło rozedniewáć / záwołał Sámuel Saulá ná dách / mówiąc : Wſtáń / á odpráwię ćię ; wſtáwƺy tedy Saul / wyƺli obáj z domu / on y Sámuel.
A gdy ſchodźili ku końcowi miáſtá / rzekł Sámuel do Saulá : Rzecż ſłudze / áby ƺedł przed námi / y ƺedł ; á ty pozoſtáń trochę / żeć opowiem ſłowo Boże.
