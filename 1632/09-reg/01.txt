Był niektóry mąż z Rámátáim Sofim / z góry Efráim / któremu było imię Elkáná / ſyn Jerohámá / ſyná Elihu / ſyná Tuhu / ſyná Suf / Efrátejcżyká.
A ten miał dwie żony / imię jedney Anná / á imię drugiej Fenenná ; y miáłá Fenenná dźiátki / ále Anná nie miáłá dźiátek.
Y chádzał on mąż z miáſtá ſwego ná káżdy rok / áby chwałę dáwał y ofiárował Pánu zaſtępów w Sylo / gdźie byli dwáj ſynowie Heli / Ofni y Finees / kápłáni Páńſcy.
A gdy przyƺedł dźień / którego ſpráwował ofiáry Elkáná / dał Fenennie / żonie ſwej / y wƺyſtkim ſynom jey / y córkom jey / cżąſtki ;
Ale Annie dał jednę cżęść wyborną ; bo Annę miłował / choćiáż był Pán żywot jey zámknął.
Y dráźniłá ją bárzo przećiwnicá jey / áby ją tylko rozgniewáłá / dla tego / iż zámknął był Pán żywot jey.
To gdy cżynił Elkáná ná káżdy rok / á Anná też chodźiłá do domu Páńſkiego / ták ją dráźniłá przećiwnicá / że płákiwáłá y nie jádáłá.
Rzekł jey tedy Elkáná / mąż jey : Anno / cżemu płácżeƺ y cżemu nie jeƺ? á przecż śię ták trápi ſerce twe? izálim ja tobie nie jeſt lepƺy niż dźieśięć ſynów?
Wſtáłá tedy Anná / gdy śię nájedli y nápili w Sylo ; á Heli kápłán śiedźiał ná ſtołku u podwojá kośćiołá Páńſkiego.
A oná będąc w gorzkośći ſercá / modliłá śię Pánu / y wielce płákáłá.
Y ucżyniłá ślub / mówiąc : Pánie zaſtępów / jeſliż wejrzáwƺy wejrzyƺ ná utrápienie ſłużebnicy twojey / y wſpomniƺ ná mię / á nie zábácżyƺ ſłużebnicy twojey / y dáƺ ſłużebnicy twej potomſtwo męſkiej płći / tedy je dam Pánu po wƺyſtkie dni żywotá jego / á brzytwá nie poſtoi ná głowie jego.
Y ſtáło śię / gdy przedłużáłá modlitwy przed Pánem / że Heli przypátrował śię uſtom jey.
Ale Anná mówiłá w ſercu ſwem / tylko wárgi jey rucháły śię / ále głoſu jey ſłycháć nie było ; y miał ją Heli zá pijáną.
Przetoż rzekł do niej Heli : Długoż będźieƺ pijáną? wytrzeźwij śię z winá twego.
Ale odpowiedźiáłá Anná y rzekłá : Nie ták / pánie mój / niewiáſtá utrápionego duchá jeſtem / ánim winá áni nápoju mocnego nie piłá / álem wyláłá duƺę moję przed oblicżem Páńſkim.
Nie rozumiejże o ſłużebnicy twojey / jáko o niewieśćie niepobożney / gdyż z wielkiego myśleniá y fráſunku mego mówiłám áż dotąd.
Tedy odpowiedźiał Heli / y rzekł : Idźże w pokoju / á Bóg Izráelſki niech ći da prośbę twoję / którejś żądáłá od niego.
Y rzekłá : Niech znájdźie ſłużebnicá twojá łáſkę przed ocżymá twemi ; y odeƺłá niewiáſtá w drogę ſwą / y jádłá / á twárz jey nie byłá więcey ſmętná.
Y wſtáli bárzo ráno / á pokłoniwƺy śię przed Pánem / wróćili śię / y przyƺli do domu ſwego do Rámátá. Tedy poznał Elkáná Annę / żonę ſwą / á wſpomniał ná nię Pán.
Y ſtáło śię po wypełnieniu dni / jáko pocżęłá Anná / że porodźiłá ſyná / y názwáłá imię jego Sámuel ; bo rzekłá : U Pánam go uprośiłá.
Potem ƺedł on mąż Elkáná / y wƺyſtek dom jego / áby oddał Pánu ofiárę urocżyſtą / y ślub ſwój.
Ale Anná nie ƺłá ; bo mówiłá mężowi ſwemu : Nie pójdę / áż zoſtáwię dźiećiątko / potem odwiodę je / że śię ukáże przed Pánem / y zoſtánie tám záwƺe.
Y rzekł jey Elkáná / mąż jey : Ucżyń co jeſt dobrego w ocżách twojich / zoſtáń / áż go zoſtáwiƺ ; tylko niech utwierdźi Pán ſłowo ſwoje. Zoſtáłá tedy niewiáſtá / y kármiłá pierśiámi ſyná ſwego / áż go zoſtáwiłá.
A gdy go zoſtáwiłá / przywiodłá go z ſobą ze trzemá ćielcámi / y z jednem efá mąki / y z łágwią winá / y przywiodłá go do domu Páńſkiego w Sylo ; á dźiećię było máłe.
Y zábiwƺy ćielcá / przywiedli dźiećię do Heli.
A oná rzekłá : Słucháj / pánie mój! żywie duƺá twojá / pánie mój : Jam jeſt oná niewiáſtá / którám tu ſtáłá przy tobie / modląc śię Pánu.
Prośiłám o to dźiećiątko / y dał mi Pán prośbę moję / którejm żądáłá od niego.
Przetoż je też ja oddáwám Pánu ; ná wƺyſtkie dni / których będźie żyło / jeſt oddáne Pánu. Y pokłonili śię tám Pánu.
