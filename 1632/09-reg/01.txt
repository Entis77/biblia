Był niektóry mąż z Rámátáim Sofim / z góry Efráim / któremu było imię Elkáná / ſyn Jerohámá / ſyná Elihu / ſyná Tuhu / ſyná Suf / Efrátejcżyká.
A ten miáł dwie żony / imię jednej Anná / á imię drugiej Fenenná ; y miáłá Fenenná dźiátki / ále Anná nie miáłá dźiátek.
Y chádzáł on mąż z miáſtá ſwego ná káżdy rok / áby chwáłę dáwáł y ofiárowáł Pánu záſtępów w Sylo / gdźie byli dwáj ſynowie Heli / Ofni y Fineeſ / kápłáni Páńſcy.
A gdy przyƺedł dźień / którego ſpráwowáł ofiáry Elkáná / dáł Fenennie / żonie ſwej / y wƺyſtkim ſynom jej / y córkom jej / cżąſtki ;
Ale Annie dáł jednę cżęść wyborną ; bo Annę miłowáł / choćiáż był Pán żywot jej zámknął.
Y dráźniłá ją bárdzo przećiwnicá jej / áby ją tylko rozgniewáłá / dlá tego / iż zámknął był Pán żywot jej.
To gdy cżynił Elkáná ná káżdy rok / á Anná też chodźiłá do domu Páńſkiego / ták ją dráźniłá przećiwnicá / że płákiwáłá y nie jádáłá.
Rzekł jej tedy Elkáná / mąż jej : Anno / cżemu płácżeƺ y cżemu nie jeƺ? á przecż śię ták trápi ſerce twe? izálim já tobie nie jeſt lepƺy niż dźieśięć ſynów?
Wſtáłá tedy Anná / gdy śię nájedli y nápili w Sylo ; á Heli kápłán śiedźiáł ná ſtołku u podwojá kośćiołá Páńſkiego.
A oná będąc w gorzkośći ſercá / modliłá śię Pánu / y wielce płákáłá.
Y ucżyniłá ślub / mówiąc : Pánie záſtępów / jeźliż wejrzáwƺy wejrzyƺ ná utrápienie ſłużebnicy twojej / y wſpomniƺ ná mię / á nie zábácżyƺ ſłużebnicy twojej / y dáƺ ſłużebnicy twej potomſtwo męſkiej płći / tedy je dám Pánu po wƺyſtkie dni żywotá jego / á brzytwá nie poſtoi ná głowie jego.
Y ſtáło śię / gdy przedłużáłá modlitwy przed Pánem / że Heli przypátrowáł śię uſtom jej.
Ale Anná mówiłá w ſercu ſwem / tylko wárgi jej rucháły śię / ále głoſu jej ſłycháć nie było ; y miáł ją Heli zá pijáną.
Przetoż rzekł do niej Heli : Długoż będźieƺ pijáną? wytrzeźwij śię z winá twego.
Ale odpowiedźiáłá Anná y rzekłá : Nie ták / pánie mój / niewiáſtá utrápionego duchá jeſtem / ánim winá áni nápoju mocnego nie piłá / álem wyláłá duƺę moję przed oblicżem Páńſkim.
Nie rozumiejże o ſłużebnicy twojej / jáko o niewieśćie niepobożnej / gdyż z wielkiego myśleniá y fráſunku mego mówiłám áż dotąd.
Tedy odpowiedźiáł Heli / y rzekł : Idźże w pokoju / á Bóg Izráelſki niech ći dá prośbę twoję / którejś żądáłá od niego.
Y rzekłá : Niech znájdźie ſłużebnicá twojá łáſkę przed ocżymá twemi ; y odeƺłá niewiáſtá w drogę ſwą / y jádłá / á twárz jej nie byłá więcej ſmętná.
Y wſtáli bárdzo ráno / á pokłoniwƺy śię przed Pánem / wróćili śię / y przyƺli do domu ſwego do Rámátá. Tedy poznáł Elkáná Annę / żonę ſwą / á wſpomniáł ná nię Pán.
Y ſtáło śię po wypełnieniu dni / jáko pocżęłá Anná / że porodźiłá ſyná / y názwáłá imię jego Sámuel ; bo rzekłá : U Pánám go uprośiłá.
Potem ƺedł on mąż Elkáná / y wƺyſtek dom jego / áby oddáł Pánu ofiárę urocżyſtą / y ślub ſwój.
Ale Anná nie ƺłá ; bo mówiłá mężowi ſwemu : Nie pójdę / áż zoſtáwię dźiećiątko / potem odwiodę je / że śię ukáże przed Pánem / y zoſtánie tám záwƺe.
Y rzekł jej Elkáná / mąż jej : Ucżyń co jeſt dobrego w ocżách twoich / zoſtáń / áż go zoſtáwiƺ ; tylko niech utwierdźi Pán ſłowo ſwoje. Zoſtáłá tedy niewiáſtá / y kármiłá pierśiámi ſyná ſwego / áż go zoſtáwiłá.
A gdy go zoſtáwiłá / przywiodłá go z ſobą ze trzemá ćielcámi / y z jednem efá mąki / y z łágwią winá / y przywiodłá go do domu Páńſkiego w Sylo ; á dźiećię było máłe.
Y zábiwƺy ćielcá / przywiedli dźiećię do Heli.
A oná rzekłá : Słucháj / pánie mój! żywie duƺá twojá / pánie mój : Jám jeſt oná niewiáſtá / którám tu ſtáłá przy tobie / modląc śię Pánu.
Prośiłám o to dźiećiątko / y dáł mi Pán prośbę moję / którejm żądáłá od niego.
Przetoż je też já oddáwám Pánu ; ná wƺyſtkie dni / których będźie żyło / jeſt oddáne Pánu. Y pokłonili śię tám Pánu.
