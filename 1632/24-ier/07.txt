Słowo / które śię ſtáło do Jeremijáƺá od Páná / mówiąc :
Stáw śię w brámie domu Páńſkiego / á opowiádáj tám to ſłowo / y mów : Słuchájćie ſłowá Páńſkiego wƺyſtek Judo / którzy wchodzićie do brám ich / ábyśćie śię kłániáli Pánu.
Ták mówi Pán záſtępów / Bóg Izráelſki : Polepƺájćie dróg ſwoich / y przedśięwzięćiá ſwego / á ſpráwię to / ábyśćie mieƺkáli ná tem miejſcu.
Nie pokłádájćie nádziei ſwej w ſłowách kłámliwych / mówiąc : Kośćiół Páńſki / kośćiół Páńſki / kośćiół Páńſki jeſt!
Ale jeźliże polepƺájąc polepƺyćie dróg ſwoich / y przedśięwzięćiá ſwego ; jeźliże ſpráwiedliwy ſąd cżynić będziećie między mężem á między bliźnim jego ;
Przychodniá / śierotki y wdowy nie ućiśniećie / y krwi niewinnej nie rozlejećie ná tem miejſcu / á zá bogámi cudzymi nie pójdziećie ná ſwe złe :
Tedy ſpráwię / ábyśćie mieƺkáli ná tem miejſcu w ziemi / którąm dáł ojcom wáƺym / od wieku áż ná wieki.
Oto wy pokłádáćie nádzieję ſwoję w ſłowách kłámliwych / które nie pomogą.
Izáli krádnąc / zábijájąc / y cudzołożąc / y krzywoprzyśięgájąc / y kádząc Báálowi / á chodząc zá bogámi obcymi / których nie znáćie /
Przećież chodzić / á ſtáwáć będziećie przed oblicżem mojem w tym domu / który názwány jeſt od imieniá mego / y mówić : Wybáwieniśmy / ábyśmy cżynili te wƺyſtkie obrzydliwośći?
Azáż jáſkinią łotrowſką jeſt dom ten przed ocżymá wáƺemi / który názwány jeſt od imieniá mego? Oto widzęć Já to / mówi Pán.
Ale idźćie przynájmniej ná miejſce moje / które było w Sylo / gdziem był ſpráwił przybytek imieniowi memu z pocżątku / á obácżćie / com mu ucżynił dlá złośći ludu mego Izráelſkiego.
Przetoż teráz / poniewáż cżynićie te wƺyſtkie ſpráwy / mówi Pán / á gdy mówię do wáſ ráno wſtáwájąc á to uſtáwicżnie / á nie ſłucháćie / gdy wołám ná wáſ / á nie ozywáćie śię :
Przetoż ucżynię domowi temu / który názwány jeſt od imieniá mego / w którym wy ufáćie / y miejſcu temu / którem wám dáł y ojcom wáƺym / jákom ucżynił Sylo ;
Y odrzucę wáſ od oblicżá mego / jákom odrzućił bráći wáƺych / wƺyſtko náśienie Efráimowe.
Ty tedy nie módl śię zá tym ludem / áni podnoś zá nim głoſu modlitwy / y nie przycżyniáj śię do mnie ; bo ćię nie wyſłuchám.
Azáż ſám nie widziƺ / co oni broją w miáſtách Judzkich y po ulicách Jeruzálemſkich?
Synowie zbieráją drwá / á ojcowie rozniecáją ogień / á żony ich rozcżyniáją ćiáſto / áby cżyniły plácki królowej niebieſkiej / y ſpráwowáły mokre ofiáry bogom cudzym / áby mię do gniewu pobudzáli.
Izáli to przećiwko mnie jeſt / że mię do gniewu wzruƺáją? mówi Pán ; izáli to nie rácżej przećiwko nim / ku poháńbieniu twárzy ich?
Przetoż ták mówi pánujący Pán : Oto gniew mój y popędliwość mojá będzie wyláná ná to miejſce / ná ludzi y ná bydlętá / y ná drzewá polne / y ná owoce ziemi / y zápáli śię / á nie ugáśnie.
Ták mówi Pán záſtępów / Bóg Izráelſki : Cáłopálenie wáƺe przydájćie do ofiár wáƺych / á jedzćie mięſo.
Bom nie mówił z ojcámi wáƺymi / ánim im przykázáł onego dniá / któregom ich wywiódł z ziemi Egipſkiej / o cáłopáleniu y o ofiárách ;
Ale tom im przykázáł / mówiąc : Słuchájćie głoſu mojego / y będę Bogiem wáƺym / á wy będziećie ludem moim ; á chodźćie káżdą drogą / którąm wám przykázáł / áby wám dobrze było.
Lecż nie poſłucháli / áni nákłonili uchá ſwego / áby chodzili zá rádámi y zá uporem ſercá ſwego złego ; y obróćili śię grzbietem / á nie twárzą.
Ode dniá / którego wyƺli ojcowie wáśi z ziemi egipſkiej / áż do dniá tego / poſyłáłem do wáſ wƺyſtkich ſług moich proroków / co dzień ráno wſtáwájąc y poſyłájąc ;
A wƺákże nie ſłucháli mię / y nie nákłonili uchá ſwego / ále zátwárdziwƺy kárk ſwój / gorzej cżynili niżeli ojcowie ich.
Gdy im będzieƺ mówił te wƺyſtkie ſłowá / y ćiebie nie uſłucháją ; á gdy ná nich wołáć będzieƺ / nie ozwąć śię.
Przetoż mów do nich : Ten jeſt náród / który nie ſłuchá głoſu Páná / Bogá ſwego / áni przyjmuje náuki ; zginęłá práwdá / y odjętá jeſt od uſt ich.
Ogól włoſy ſwe y odrzuć / á nárzekáj głośno ná miejſcách wyſokich ; bo odrzućił Pán y opuśćił rodzáj / ná który śię bárdzo gniewá.
Záiſte ſynowie Judzcy cżynili złość przed ocżymá mojemi / mówi Pán ; náſtáwiáli obrzydliwośći ſwych w tym domu / który názwány jeſt od imieniá mojego / áby go ſplugáwili.
Nádto pobudowáli wyżyny Tofet / które jeſt w dolinie ſyná Hennomowego / áby pálili ſynów ſwych y córki ſwe ogniem / cżegom nie rozkázáł / áni wſtąpiło ná ſerce moje.
Dlátego oto dni idą / mówi Pán / gdy to więcej nie będzie zwáno Tofet / áni doliná ſyná Hennomowego / ále doliná morderſtwá ; y będą pogrzebywáć w Tofet ; bo indziej miejſcá nie będzie.
Y będą trupy ludu tego pokármem ptáſtwu niebieſkiemu / y zwierzowi ziemſkiemu / á nie będzie / ktoby odegnáł.
Y ucżynię / że uſtánie w miáſtách Judzkich / y w ulicách Jeruzálemſkich głoſ rádośći / y głoſ weſelá / głoſ oblubieńcá / y głoſ oblubienicy ; bo ziemiá będzie ſpuſtoƺoná.
