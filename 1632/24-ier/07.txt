Słowo / które śię ſtáło do Jeremijáƺá od Páná / mówiąc :
Stáw śię w brámie domu Páńſkiego / á opowiádáj tám to ſłowo / y mów : Słuchájćie ſłowá Páńſkiego wƺyſtek Judo / którzy wchodźićie do brám ich / ábyśćie śię kłániáli Pánu.
Ták mówi Pán zaſtępów / Bóg Izráelſki : Polepƺájćie dróg ſwojich / y przedśięwźięćiá ſwego / á ſpráwię to / ábyśćie mieƺkáli ná tym miejſcu.
Nie pokłádájćie nádźiei ſwej w ſłowách kłámliwych / mówiąc : Kośćiół Páńſki / kośćiół Páńſki / kośćiół Páńſki jeſt!
Ale jeſliże polepƺájąc polepƺyćie dróg ſwojich / y przedśięwźięćiá ſwego ; jeſliże ſpráwiedliwy ſąd cżynić będźiećie miedzy mężem á miedzy bliźnim jego ;
Przychodniá / śierotki y wdowy nie ućiśniećie / y krwi niewinney nie rozlejećie ná tym miejſcu / á zá bogámi cudzymi nie pójdźiećie ná ſwe złe :
Tedy ſpráwię / ábyśćie mieƺkáli ná tym miejſcu w źiemi / którąm dał ojcom wáƺym / od wieku áż ná wieki.
Oto wy pokłádáćie nádźieję ſwoję w ſłowách kłámliwych / które nie pomogą.
Izali krádnąc / zábijájąc / y cudzołożąc / y krzywoprzyśięgájąc / y kádząc Báálowi / á chodząc zá bogámi obcymi / których nie znáćie /
Przećież chodźić / á ſtáwáć będźiećie przed oblicżem mojim w tym domu / który názwány jeſt od imieniá mego / y mówić : Wybáwieniśmy / ábyſmy cżynili te wƺyſtkie obrzydliwośći?
Azaż jáſkinią łotrowſką jeſt dom ten przed ocżymá wáƺemi / który názwány jeſt od imieniá mego? Oto widzęć Ja to / mówi Pán.
Ale idźćie przynájmniej ná miejſce moje / które było w Sylo / gdźiem był ſpráwił przybytek imieniowi memu z pocżątku / á obácżćie / com mu ucżynił dla złośći ludu mego Izráelſkiego.
Przetoż teraz / poniewáż cżynićie te wƺyſtkie ſpráwy / mówi Pán / á gdy mówię do was ráno wſtáwájąc á to uſtáwicżnie / á nie ſłucháćie / gdy wołám ná was / á nie ozywáćie śię :
Przetoż ucżynię domowi temu / który názwány jeſt od imieniá mego / w którym wy ufáćie / y miejſcu temu / którem wam dał y ojcom wáƺym / jákom ucżynił Sylo ;
Y odrzucę was od oblicżá mego / jákom odrzućił bráći wáƺych / wƺyſtko náśienie Efráimowe.
Ty tedy nie módl śię zá tym ludem / áni podnoś zá nim głoſu modlitwy / y nie przycżyniáj śię do mnie ; bo ćię nie wyſłuchám.
Azaż ſám nie widźiƺ / co oni broją w miáſtách Judzkich y po ulicách Jeruzálemſkich?
Synowie zbieráją drwá / á ojcowie rozniecáją ogień / á żony ich rozcżyniáją ćiáſto / áby cżyniły plácki królowej niebieſkiej / y ſpráwowáły mokre ofiáry bogom cudzym / áby mię do gniewu pobudzáli.
Izali to przećiwko mnie jeſt / że mię do gniewu wzruƺáją? mówi Pán ; izali to nie rácżey przećiwko nim / ku poháńbieniu twárzy ich?
Przetoż ták mówi pánujący Pán : Oto gniew mój y popędliwość mojá będźie wyláná ná to miejſce / ná ludźi y ná bydlętá / y ná drzewá polne / y ná owoce źiemi / y zápáli śię / á nie ugáśnie.
Ták mówi Pán zaſtępów / Bóg Izráelſki : Cáłopálenie wáƺe przydájćie do ofiár wáƺych / á jedzćie mięſo.
Bom nie mówił z ojcámi wáƺymi / ánim im przykázał onego dniá / któregom ich wywiódł z źiemi Egipſkiej / o cáłopáleniu y o ofiárách ;
Ale tom im przykázał / mówiąc : Słuchájćie głoſu mojego / y będę Bogiem wáƺym / á wy będźiećie ludem moim ; á chodźćie káżdą drogą / którąm wam przykázał / áby wam dobrze było.
Lecż nie poſłucháli / áni nákłonili uchá ſwego / áby chodźili zá rádámi y zá uporem ſercá ſwego złego ; y obróćili śię grzbietem / á nie twárzą.
Ode dniá / którego wyƺli ojcowie wáśi z źiemi egipſkiej / áż do dniá tego / poſyłáłem do was wƺyſtkich ſług mojich proroków / co dźień ráno wſtáwájąc y poſyłájąc ;
A wƺákże nie ſłucháli mię / y nie nákłonili uchá ſwego / ále zátwárdźiwƺy kárk ſwój / gorzej cżynili niżeli ojcowie ich.
Gdy im będźieƺ mówił te wƺyſtkie ſłowá / y ćiebie nie uſłucháją ; á gdy ná nich wołáć będźieƺ / nie ozwąć śię.
Przetoż mów do nich : Ten jeſt náród / który nie ſłuchá głoſu Páná / Bogá ſwego / áni przyjmuje náuki ; zginęłá práwdá / y odjętá jeſt od uſt ich.
Ogól włoſy ſwe y odrzuć / á nárzekáj głośno ná miejſcách wyſokich ; bo odrzućił Pán y opuśćił rodzáj / ná który śię bárzo gniewá.
Záiſte ſynowie Judzcy cżynili złość przed ocżymá mojemi / mówi Pán ; náſtáwiáli obrzydliwośći ſwych w tym domu / który názwány jeſt od imieniá mojego / áby go ſplugáwili.
Nádto pobudowáli wyżyny Tofet / które jeſt w dolinie ſyná Hennomowego / áby pálili ſynów ſwych y córki ſwe ogniem / cżegom nie rozkázał / áni wſtąpiło ná ſerce moje.
Dla tego oto dni idą / mówi Pán / gdy to więcey nie będźie zwano Tofet / áni doliná ſyná Hennomowego / ále doliná morderſtwá ; y będą pogrzebywáć w Tofet ; bo indźiej miejſcá nie będźie.
Y będą trupy ludu tego pokármem ptáſtwu niebieſkiemu / y zwierzowi źiemſkiemu / á nie będźie / ktoby odegnał.
Y ucżynię / że uſtánie w miáſtách Judzkich / y w ulicách Jeruzálemſkich głos rádośći / y głos weſelá / głos oblubieńcá / y głos oblubienicy ; bo źiemiá będźie ſpuſtoƺoná.
