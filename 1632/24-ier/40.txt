Słowo / które śię ſtáło do Jeremijáƺá od Páná / gdy go wypuśćił Nábuzárdán / hetmán żołnierſki / z Rámy / wziąwƺy go / gdy był związány łáńcuchámi w pośród wƺyſtkich więźniów Jeruzálemſkich y Judzkich / których wiedziono do Bábilonu.
A ták wziął hetmán żołnierſki Jeremijáƺá / y rzekł do niego : Pán / Bóg twój / opowiedziáł był to złe przećiwko miejſcu temu ;
Przetoż je przywiódł / y ucżynił Pán / jáko mówił ; bośćie zgrzeƺyli Pánu / á nie ſłucháliśćie głoſu jego / y dlátego śię wám to ſtáło.
Teráz tedy / oto ćię rozwiązuję dziś z tych łáńcuchów / które ſą ná rękách twoich. Jeźlić śię zdá rzecżą dobrą / iść zemną do Bábilonu / pójdź / já o tobie będę záwiádywáł ; á jeźlić śię nie podobá iść zemną do Bábilonu / tedy zániecháj. Oto tá wƺyſtká ziemiá jeſt przed oblicżem twojim ; gdzie woliƺ / á gdzieć śię podobá iść / tám idź.
A poniewáż śię tu on więcej nie wróći / udáj śię do Godolijáƺá / ſyná Ahikámowego / ſyná Sáfánowego / którego przełożył król Bábilońſki nád miáſtámi Judzkiemi / á mieƺkáj z nim w pośród ludu / álbo gdzieć śię kolwiek podobá iść / idź. Y dáł mu hetmán żołnierſki ná drogę y upominek / y odpráwił go.
Przyƺedł tedy Jeremijáƺ do Godolijáƺá / ſyná Ahikámowego / do Máſfy / y mieƺkáł z nim w pośród ludu / który był pozoſtáł w ziemi.
A gdy uſłyƺeli wƺyſcy hetmáni wojſk / którzy byli w polách / oni y wƺyſtek lud ich / że król Bábilońſki poſtánowił Godolijáƺá / ſyná Ahikámowego / nád oną ziemią / á iż mu zlećił mężów / y niewiáſty / y dziátki / á to nájpodlejƺych onej ziemi / tych / którzy nie byli záprowádzeni do Bábilonu :
Tedy przyƺli do Godolijáƺá do Máſfy / to jeſt / Izmáel / ſyn Nátánijáƺowy / tákże Johánán y Jonátán / ſynowie Káreáƺowi / y Serájáƺ / ſyn Tánchumetowy / y ſynowie Efáj Netofátcżyká / y Jáſánijáƺ / ſyn Máchátowy / oni y lud ich.
Tedy im przyśiągł Godolijáƺ / ſyn Ahikámá / ſyná Sáfánowego / y ludowi ich / mówiąc : Nie bójćie śię ſłużyć Cháldejcżykom / zoſtáńćie w ziemi / y ſłużćie królowi Bábilońſkiemu / á dobrze wám będzie.
Bo oto y já mieƺkám w Máſfie / ábym ſłużył Cháldejcżykom / którzy przychodzą do náſ ; á wy zbierájćie wino y letni owoc y oliwę / á ſkłádájćie do nácżyniá wáƺego / y mieƺkájćie w miáſtách wáƺych / które trzymáćie.
Tákże y wƺyſcy Żydzi / którzy byli u Moábcżyków / y między ſynámi Ammonowymi / y między Edomcżykámi / y którzy byli we wƺyſtkich ziemiách / uſłyƺáwƺy / że król Bábilońſki zoſtáwił oſtátek ludu z Judy / á iż przełożył nád nimi Godolijáƺá / ſyná Ahikámá / ſyná Sáfánowego /
Wróćili śię wƺyſcy Żydzi ze wƺyſtkich miejſc / do których byli zágnáni / y przyƺli do ziemi Judzkiej do Godolijáƺá do Máſfy / y názbieráli winá / y letniego owocu bárdzo wiele.
Ale Johánán / ſyn Káreáƺowy / y wƺyſcy kśiążętá wojſk / którzy byli w polu / przyƺli do Godolijáƺá do Máſfy /
Y rzekli do niego : Wieƺże o tem / że Bááliſ / król ſynów Ammonowych / poſłáł Izmáelá / ſyná Nátánijáƺowego / áby ćię zábił? Ale im nie uwierzył Godolijáƺ / ſyn Ahikámá.
Nádto Johánán / ſyn Káreáƺowy / rzekł do Godolijáƺá potájemnie w Máſfie / mówiąc : Niech idę / proƺę / á zábiję Izmáelá / ſyná Nátánijáƺowego / wƺák śię o tem nikt nie dowie. Przecżby ćię miáł zábić / á rozproƺeniby mieli być wƺyſcy Żydowie / którzy śię zebráli do ćiebie / y zginąć oſtátek z Judy?
Ale Godolijáƺ / ſyn Ahikámowy / rzekł do Johánáná / ſyná Káreáƺowego : Nie cżyń tego ; bo ty niepráwdę mówiƺ o Izmáelu.
