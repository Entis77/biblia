Cżáſu onego / mówi Pán / wybiorą kośći królów Judzkich / y kośći kśiążąt ich / y kośći kápłanów / y kośći proroków / y kośći obywáteli Jeruzálemſkich z grobów ich ;
Y rozrzucą je przed ſłońce / y przed mieśiąc / y przed wƺyſtko wojſko niebieſkie / które miłują / y którym ſłużą / y zá którymi chodzą / y których ƺukáją / y którym śię kłániáją ; nie pozbieráją ich / áni pogrzebią / ále będą miáſto gnoju ná wierzchu źiemi.
Y obiorą rácżey śmierć niżeli żywot wƺyſtkie oſtátki / które zoſtáną z tego rodzáju złośliwego po wƺyſtkich miejſcách / gdźiebykolwiek zoſtáli / tám / kędy ich zápędzę / mówi Pán zaſtępów.
Przetoż rzecżeƺ do nich : Ták mówi Pán : Tákże upádli / áby nie mogli powſtáć? tákże śię odwróćił / áby śię záś nie mógł náwróćić?
Przecżże śię odwróćił ten lud Jeruzálemſki odwróceniem wiecżnem? chwytáją śię kłámſtwá / á nie chcą śię náwrácáć.
Pilnowáłem y ſłucháłem : nic nie mówiąc / co jeſt práwego ; niemáƺ ktoby żáłował złośći ſwej / mówiąc : Cóżem ucżynił? Káżdy śię obróćił zá biegiem ſwoim / jáko koń / który pędem bieży ku potkániu.
Y boćián ná powietrzu zná uſtáwione cżáſy ſwoje / y ſynogárlicá / y żóráw / y jáſkółká przeſtrzegáją cżáſu przyleceniá ſwego ; ále lud mój nie zná ſądu Páńſkiego.
Jákoż mówićie : Myśmy mądrzy / á zakon Páńſki jeſt przy nas? zápráwdę / oto dáremnie pióro piſárz cżyni ; dáremnie ſą w zakonie biegłymi.
Kogoż záwſtydźili ći mędrcy? Którzyż ſą przeſtráƺeni y pojmáni? Oto ſłowo Páńſkie odrzucáją ; cóż to tedy zá mądrość ich?
Dla tego dam żony ich innym / polá ich tym / którzy ich opánują ; bo od nájmniejƺego áż do nájwiękƺego / wƺyſcy zgołá událi śię zá łákomſtwem ; od proroká áż do kápłáná wƺyſcy przewodzą kłámſtwo.
Bo lecżą ſkruƺenie córki ludu mego tylko po wierzchu / mówiąc : Pokój / pokój! choć niemáƺ pokoju.
Izali śię záwſtydźili / przeto / że obrzydliwość cżynili? Zájiſte / áni śię zápáłáć áni wſtydźić umieli ; przetoż upádną miedzy pádájącymi / cżáſu náwiedzeniá ſwego upádną / mówi Pán.
Do ƺcżętu ich wykorzenię / mówi Pán ; nie będźie żádnego groná ná winney máćicy / áni żádnych fig ná drzewie figowem ; náwet y liść opádnie / á com im dał / odjęte będźie.
Przecż my tu śiedźimy? Zejdźćie śię / á wnidźmy do miaſt obronnych / á tám odpocżniemy ; ále Pán / Bóg náƺ / káże nam odpocżnąć / gdy nas nápoi wodą żółći / iżeśmy zgrzeƺyli przećiwko Pánu.
Cżekáj pokoju / álić nic dobrego ; cżáſu uzdrowieniá / álić oto ſtrách.
Od Dán ſłyƺeć chrápánie koni jego / od głoſu wykrzykánia mocárzy jego wƺyſtká źiemiá zádrżáłá / którzy ćiągną / áby pożárli źiemię / y wƺyſtko / co jeſt ná niej / miáſto y tych / którzy mieƺkáją w nim.
Bo oto Ja poślę ná was węże nájjádowitƺe / przećiwko którym niemáƺ záklinánia ; y pokąſáją was / mówi Pán.
Serce moje we mnie / któreby mię miáło pośiláć w ſmutku / mdłe jeſt.
Oto głos krzyku córki ludu mego z źiemi bárzo dálekiej mówiącey : Izali Páná niemáƺ ná Syonie? Izali królá jego niemáƺ ná nim? Przecżże mię wzruƺyli do gniewu báłwánámi ſwemi / próżnośćiámi cudzoźiemców? mówi Pán.
Pominęło żniwo / ſkońcżyło śię láto / á myśmy nie wybáwieni.
Dla ſkruƺeniá córki ludu mojego ſkruƺonym jeſt / żáłobę ponoƺę / zdumienie zdjęło mię.
Izali niemáƺ bálſámu w Gáláád? Izali tám niemáƺ lekárzá? Cżemuż tedy nie jeſt ulecżoná córká ludu mojego?
