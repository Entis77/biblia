Ná pocżątku królowániá Joákimá / ſyná Jozyjáƺá / królá Judzkiego / ſtáło śię to ſłowo do Jeremijáƺá od Páná / mówiąc :
Ták mówi Pán do mnie : Ucżyń ſobie okowy y járzmá / á włóż je ná ƺyję ſwoję ;
Potem je poślij do królá Edomſkiego / y do królá Moábſkiego / y do królá ſynów Ammonowych / y do królá Tyrſkiego / y do królá Sydońſkiego / przez rękę poſłów / którzy przyjdą do Jeruzálemu do Sedekijáƺá / królá Judzkiego ;
A rozkáż im / áby pánom ſwym powiedźieli : Ták mówi Pán záſtępów / Bóg Izráelſki / ták powiećie pánom wáƺym :
Jám ucżynił źiemię / tákże cżłowieká / y bydlętá / którekolwiek ſą ná oblicżu źiemi / mocą ſwoją wielką y rámieniem ſwojim wyćiągnionem ; przeto ja dáję temu / który śię podobá ocżom moim.
A teraz dáłem te wƺyſtkie źiemie w rękę Nábuchodonozorá / królá Bábilońſkiego / ſługi mego ; nádto y zwierzętá polne dáłem mu / áby mu ſłużyły.
Przetoż będą mu ſłużyły te wƺyſtkie národy / y ſynowi jego y ſynowi ſyná jego / dokądby nie przyƺedł cżáſ źiemi jego y jego ſámego / gdy go też záś ſobie w niewolę podbiją národy zácne / y królowie wielcy.
A ten náród y to króleſtwo / któreby mu nie ſłużyło / to jeſt Nábuchodonozorowi / królowi Bábilońſkiemu / y któryby nie poddáł ƺyi ſwojey pod járzmo królá Bábilońſkiego / miecżem / y głodem / y morem náwiedzę ten náród / mówi Pán / dokądbym ich do końcá nie wytráćił ręką jego.
Przetoż wy nie ſłuchájćie proroków ſwoich / y wieƺcżków ſwoich / áni ſnów ſwoich / áni wróżków ſwoich / y cżárowników ſwoich / którzy wám powiádáją / mówiąc : Nie będźiećie ſłużyli królowi Bábilońſkiemu ;
Bo wám oni kłámſtwo prorokują / ábym was oddálił od źiemi wáƺej / á wygnáł was / ábyśćie poginęli.
A náród / który poddáł ƺyję ſwą pod járzmo królá Bábilońſkiego / á któryby mu ſłużył / ten záiſte zoſtáwię w źiemi jego / ( mówi Pán / ) áby ją ſpráwowáł / y mieƺkáł w niej.
A do Sedekijáƺá / królá Judzkiego / rzekłem według tych wƺyſtkich ſłów / mówiąc : Poddájćie ƺyje wáƺe pod járzmo królá Bábilońſkiego / á ſłużćie mu y ludowi jego / á żyć będźiećie.
Przecżże máćie zginąć / ty y lud twój / od miecżá / od głodu y od powietrzá / jáko mówił Pán o národźie / któryby nie ſłużył królowi Bábilońſkiemu?
Nie ſłuchájćież tedy ſłów tych proroków / którzy mówiąc do was powiádáją : Nie będźiećie ſłużyli królowi Bábilońſkiemu ; ábowiem wám oni kłámſtwo prorokują.
Nie poſłáłem ich záiſte / mówi Pán ; wƺákże oni prorokują w imię moje kłámliwie / ábym was zápędźił / gdźiebyśćie zginęli / wy y ći prorocy / którzy wám prorokują.
Do kápłánów też / y do tego wƺyſtkiego ludu rzekłem / mówiąc : Ták mówi Pán : Nie ſłuchájćie ſłów proroków ſwoich / którzy wám prorokują / mówiąc : Oto nácżyniá domu Páńſkiego teraz w rychle przywrócone będą z Bábilonu ; bo wám oni kłámſtwo prorokują.
Nie ſłuchájćież ich ; ſłużćie królowi Bábilońſkiemu / á żyć będźiećie ; przecżżeby to miáſto miáło być puſtynią?
A jeſliż oni ſą prorokámi / y jeſli jeſt ſłowo Páńſkie w nich / proƺę / niech zábieżą Pánu záſtępów / áby te nácżyniá / które pozoſtáły w domu Páńſkim ; y w domu królá Judzkiego / y w Jeruzálemie / nie doſtáły śię do Bábilonu.
Bo ták mówi Pán záſtępów o tych ſłupách / y o tem morzu / y o tych podſtáwkách / y o oſtátku nácżyniá / które pozoſtáło w tem mieśćie ;
Którego nie zábráł Nábuchodonozor / król Bábilońſki / gdy w niewolę prowádźił Jechonijáƺá / ſyná Joákimowego / królá Judzkiego / z Jeruzálemu do Bábilonu / y wƺyſtkich przedniejƺych / z Judy y z Jeruzálemu :
Ták záiſte mówi Pán záſtępów / Bóg Izráelſki / o tych nácżyniách / które pozoſtáły w domu Páńſkim / y w domu królá Judzkiego / y w Jeruzálemie :
Do Bábilonu záwieźione będą / á będą tám áż do dniá / którego je náwiedzę / mówi Pán / y káżę je przywieźć y przywróćić ná to miejſce.
