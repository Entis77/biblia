Ná pocżątku królowániá Joákimá / ſyná Jozyjáƺá / królá Judzkiego / ſtáło śię to ſłowo do Jeremijáƺá od Páná / mówiąc :
Ták mówi Pán do mnie : Ucżyń ſobie okowy y járzmá / á włóż je ná ƺyję ſwoję ;
Potem je poślij do królá Edomſkiego / y do królá Moábſkiego / y do królá ſynów Ammonowych / y do królá Tyrſkiego / y do królá Sydońſkiego / przez rękę poſłów / którzy przyjdą do Jeruzálemu do Sedekijáƺá / królá Judzkiego ;
A rozkáż im / áby pánom ſwym powiedzieli : Ták mówi Pán záſtępów / Bóg Izráelſki / ták powiećie pánom wáƺym :
Jám ucżynił ziemię / tákże cżłowieká / y bydlętá / którekolwiek ſą ná oblicżu ziemi / mocą ſwoją wielką y rámieniem ſwojim wyćiągnionem ; przeto já dáję temu / który śię podobá ocżom moim.
A teráz dáłem te wƺyſtkie ziemie w rękę Nábuchodonozorá / królá Bábilońſkiego / ſługi mego ; nádto y zwierzętá polne dáłem mu / áby mu ſłużyły.
Przetoż będą mu ſłużyły te wƺyſtkie národy / y ſynowi jego y ſynowi ſyná jego / dokądby nie przyƺedł cżáſ ziemi jego y jego ſámego / gdy go też záś ſobie w niewolę podbiją národy zácne / y królowie wielcy.
A ten náród y to króleſtwo / któreby mu nie ſłużyło / to jeſt Nábuchodonozorowi / królowi Bábilońſkiemu / y któryby nie poddáł ƺyi ſwojej pod járzmo królá Bábilońſkiego / miecżem / y głodem / y morem náwiedzę ten náród / mówi Pán / dokądbym ich do końcá nie wytráćił ręką jego.
Przetoż wy nie ſłuchájćie proroków ſwoich / y wieƺcżków ſwoich / áni ſnów ſwoich / áni wróżków ſwoich / y cżárowników ſwoich / którzy wám powiádáją / mówiąc : Nie będziećie ſłużyli królowi Bábilońſkiemu ;
Bo wám oni kłámſtwo prorokują / ábym wáſ oddálił od ziemi wáƺej / á wygnáł wáſ / ábyśćie poginęli.
A náród / który poddáł ƺyję ſwą pod járzmo królá Bábilońſkiego / á któryby mu ſłużył / ten záiſte zoſtáwię w ziemi jego / ( mówi Pán / ) áby ją ſpráwowáł / y mieƺkáł w niej.
A do Sedekijáƺá / królá Judzkiego / rzekłem według tych wƺyſtkich ſłów / mówiąc : Poddájćie ƺyje wáƺe pod járzmo królá Bábilońſkiego / á ſłużćie mu y ludowi jego / á żyć będziećie.
Przecżże máćie zginąć / ty y lud twój / od miecżá / od głodu y od powietrzá / jáko mówił Pán o národzie / któryby nie ſłużył królowi Bábilońſkiemu?
Nie ſłuchájćież tedy ſłów tych proroków / którzy mówiąc do wáſ powiádáją : Nie będziećie ſłużyli królowi Bábilońſkiemu ; ábowiem wám oni kłámſtwo prorokują.
Nie poſłáłem ich záiſte / mówi Pán ; wƺákże oni prorokują w imię moje kłámliwie / ábym wáſ zápędził / gdziebyśćie zginęli / wy y ći prorocy / którzy wám prorokują.
Do kápłánów też / y do tego wƺyſtkiego ludu rzekłem / mówiąc : Ták mówi Pán : Nie ſłuchájćie ſłów proroków ſwoich / którzy wám prorokują / mówiąc : Oto nácżyniá domu Páńſkiego teráz w rychle przywrócone będą z Bábilonu ; bo wám oni kłámſtwo prorokują.
Nie ſłuchájćież ich ; ſłużćie królowi Bábilońſkiemu / á żyć będziećie ; przecżżeby to miáſto miáło być puſtynią?
A jeźliż oni ſą prorokámi / y jeźli jeſt ſłowo Páńſkie w nich / proƺę / niech zábieżą Pánu záſtępów / áby te nácżyniá / które pozoſtáły w domu Páńſkim ; y w domu królá Judzkiego / y w Jeruzálemie / nie doſtáły śię do Bábilonu.
Bo ták mówi Pán záſtępów o tych ſłupách / y o tem morzu / y o tych podſtáwkách / y o oſtátku nácżyniá / które pozoſtáło w tem mieśćie ;
Którego nie zábráł Nábuchodonozor / król Bábilońſki / gdy w niewolę prowádził Jechonijáƺá / ſyná Joákimowego / królá Judzkiego / z Jeruzálemu do Bábilonu / y wƺyſtkich przedniejƺych / z Judy y z Jeruzálemu :
Ták záiſte mówi Pán záſtępów / Bóg Izráelſki / o tych nácżyniách / które pozoſtáły w domu Páńſkim / y w domu królá Judzkiego / y w Jeruzálemie :
Do Bábilonu záwiezione będą / á będą tám áż do dniá / którego je náwiedzę / mówi Pán / y káżę je przywieźć y przywróćić ná to miejſce.
