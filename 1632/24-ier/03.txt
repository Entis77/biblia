Pán mówi : Opuśćiłliby mąż żonę ſwoję / á oná odƺedƺy od niego ƺłáby zá innego mężá / izali śię więcey do niej wróći? Izáliby nie byłá wielce ſplugáwioná oná źiemiá? Ale ty / choćiáżeś nierząd płodźiłá z wielą zálotników / wƺákże náwróć śię do mnie / mówi Pán.
Podnieś ocżu ſwych ná miejſcá wyſokie / á obácż / jeſliś gdźie nierządu nie płodźiłá. Ná drogách śiádáłáś kwoli nim / jáko Arábcżyk ná puƺcży / á ſplugáwiłáś źiemię wƺetecżeńſtwem twem / y złośćią twoją.
A choćiáż záwśćiągnione ſą dżdże jeśienne / á deƺcżu ná wioſnę nie bywáło / przećieżeś cżoło niewiáſty wƺetecżney májąc / nie chćiáłáś śię wſtydźić.
Azaż od tego cżáſu wołáć będźieƺ ná mnie : Ojcże mój! Tyś wodzem młodośći mojey?
Izali Bóg zátrzymá gniew ná wiecżność? Izali go záchowá ná wieki? Oto mówiƺ y cżyniƺ złe / ile możeƺ.
Tedy Pán rzekł do mnie zá dni Jozyjáƺá królá : Widźiáłżeś / co ucżyniłá odporná córká Izráelſká? jáko chodźiłá ná káżdą górę wyſoką / y pod káżde drzewo źielone / y tám nierząd płodźiłá.
A choćiáżem rzekł / gdy to wƺyſtko ucżyniłá : Náwróć śię do mnie! przećie śię nie náwróćiłá ; á ná to pátrzyłá przeſtępnicá śioſtrá jey / córká Judzká.
A ták zdáło mi śię dla tych wƺyſtkich przycżyn / poniewáż nierząd płodźiłá uporná córká Izráelſká / opuśćić ją / y dáć jey liſt rozwodny ; á przećież śię nie ulękłá przeſtępnicá śioſtrá jey / córká Judzká / ále ƺedƺy y ſámá nierząd płodźiłá.
Y ſtáło śię / że hániebnym nierządem ſwoim ſplugáwiłá źiemię ; bo cudzołożyłá z kámieniem y z drewnem.
A wƺákże w tym wƺyſtkiem nie náwróćiłá śię do mnie przeſtępnicá śioſtrá jey / córká Judzká / z wƺyſtkiego ſercá ſwego ; ále obłudnie / mówi Pán.
Przetoż rzekł Pán do mnie : Uſpráwiedliwiłá duƺę ſwą odporná córká Izráelſká więcey / niżeli przeſtępnicá Judzká.
Idźże / á wołáj temi ſłowy ku północy á mów : Náwróć śię / odporná córko Izráelſká! mówi Pán / á nie obórzy śię twárz mojá ſurowá ná was / bom Ja dobrotliwy / mówi Pán / á nie chowám gniewu ná wieki.
Tylko uznáj niepráwość twoję / żeś od Páná / Bogá ſwego / odſtąpiłá / á tám y ſám biegáłá drogámi ſwemi do obcych bogów pod káżde drzewo źielone / á głoſu mojego nie ſłucháliśćie / mówi Pán.
Náwróććież śię / ſynowie uporni! mówi Pán ; bom Ja jeſt máłżonkiem wáƺym / á przyjmę was jednego z miáſtá / á dwu z rodzáju / ábym was wprowádźił do Syonu ;
Gdźie wam dam páſterzy według ſercá mego / y będą was páść umiejętnie y rozumnie.
Y ſtánie śię / gdy śię rozmnożyćie á rozrodźićie śię w tey źiemi w oneż dni / mówi Pán / nie będą więcey mówić : "Skrzyniá przymierza Páńſkiego" / áni wſtąpi ná ſerce / áni wſpomną ná nią / áni jey náwiedzáć / áni jey więcey powáżáć będą.
Cżáſu onego názwáne będźie Jeruzálem ſtolicą Páńſką / á zgromádzą śię do niego wƺyſtkie národy / do imieniá Páńſkiego do Jeruzálemu / y nie będą więcey chodźić zá uporem ſercá ſwego złośliwego.
W one dni pójdą dom Judzki z domem Izráelſkim / y przyjdą poſpołu z źiemi północney do źiemi / którąm dał w dźiedźictwo ojcom wáƺym.
Choćiáżem Ja rzekł : Jákożbym ćię położył miedzy ſynámi / á dał ći źiemię pożądáną / dźiedźictwo zácne zaſtępów pogáńſkich? chybá żebyś mię wzywał / mówiąc : Ojcże mój! á od náśládowánia mnie nie odwróćił śię.
Poniewáż jáko żoná przeniewierzá śię mężowi ſwemu / tákeśćie mi śię przeniewierzyli / o domie Izráelſki! mówi Pán.
Głos ná wyſokich miejſcách niech będźie ſłyƺány / płácż modlitw ſynów Izráelſkich ; bo przewrotne ucżyniwƺy drogi ſwe zápámiętáli ná Páná / Bogá ſwego /
Mówiącego : Náwróććie śię / ſynowie odporni! á ulecżę odwróceniá wáƺe ; mówćie : Oto my idźiemy do ćiebie / boś ty jeſt Pán / Bóg náƺ.
Záiſte próżná jeſt nádźiejá w págórkách y w mnóſtwie gór ; záiſte w Pánu / Bogu náƺym / jeſt zbáwienie Izráelſkie.
Bo tá háńbá pożárłá prácę ojców náƺych od młodośći náƺej / trzody ich / y ſtádá ich / ſyny ich / y córki ich.
Leżymy w poháńbieniu ſwem / á przykrywá nas zelżywość náƺá ; ábowiemeśmy przećiwko Pánu / Bogu náƺemu / zgrzeƺyli / my y ojcowie náśi / od młodośći náƺej áż do dniá tego / á nie uſłucháliśmu głoſu Páná / Bogá náƺego.
