Słowo / które śię ſtáło do Jeremijáƺá od Páná mówiąc :
Słuchájćie ſłów przymierza tego / którebyśćie mówili do mężów Judzkich y do obywáteli Jeruzálemſkich ;
A rzecżeƺ do nich : Ták mówi Pán / Bóg Izráelſki : Przeklęty ten cżłowiek / któryby nie uſłuchał ſłów przymierza tego /
Którem przykázał ojcom wáƺym / dniá / któregom ich wywiódł z źiemi Egipſkiej / z piecá żeláznego / mówiąc : Słuchájćie głoſu mojego / á cżyńćie to wƺyſtko / co wam rozkázuję / y będźiećie ludem moim / á Ja będę Bogiem wáƺym ;
Abym ſpełnił przyśięgę / którąm przyśiągł ojcom wáƺym / że im dam źiemię opływájącą mlekiem y miodem : jáko śię to dźiś okázuje. Któremu odpowiedźiáwƺy rzekłem : Amen / Pánie!
Y rzekł Pán do mnie : Obwoływáj wƺyſtkie te ſłowá w miáſtách Judzkich y po ulicách Jeruzálemſkich / mówiąc : Słuchájćie ſłów przymierza tego / á cżyńćie je.
Bo oświádcżájąc oświádcżáłem śię przed ojcámi wáƺymi ode dniá / któregom ich wywiódł z źiemi Egipſkiej / áż do dniá tego ; ráno wſtáwájąc y oświádcżájąc śię / máwiáłem : Słuchájćie głoſu mego.
Ale nie uſłucháli / áni nákłonili uchá ſwego ; owƺem káżdy ƺedł zá uporem ſercá ſwego złego. Przetożem przywiódł ná nich wƺyſtkie ſłowá przymierza tego / którem rozkázał / áby cżynili ; ále oni nie cżynili.
Y rzekł Pán do mnie : Ználázło śię ſprzyśiężenie miedzy mężámi Judzkimi / y miedzy obywátelámi Jeruzálemſkimi ;
Obróćili śię do niepráwośći ojców ſwojich pierwƺych / którzy nie chćieli ſłucháć ſłów mojich ; tákże y ći chodzą zá bogámi cudzymi / ſłużąc im ; zgwáłćili dom Izráelſki y dom Judzki przymierze moje / którem był poſtánowił z ojcámi ich.
Dla tego ták mówi Pán : Oto Ja przywiodę ná nich złe / z którego nie będą mogli wyjść ; choćby wołáli do mnie / nie wyſłuchám ich.
Y pójdą miáſtá Judzkie y obywátele Jeruzálemſcy / á będą wołáli do bogów / którym kádzą ; ále ich żádnym ſpoſobem nie wybáwią cżáſu utrapienia ich.
Acżkolwiek ile jeſt miaſt twojich / tyle bogów twojich / o Judo! á ile ulic Jeruzálemſkich / tyleśćie náſtáwiáli ołtárzów obrzydliwośći / ołtárzów do kádzeniá Báálowi.
Przetoż śię ty nie módl zá tym ludem / áni podnoś zá nimi głoſu y modlitwy ; bo Ja ich nie wyſłuchám nátencżás / gdy do mnie záwołáją w utrápieniu ſwojim.
Cóż miłemu memu do domu mego? ponieważ bez wſtydu páchá złośći z wielomá / á ofiáry święte odeƺły od ćiebie ; y że śię w złośći ſwojey rádujeƺ.
Oliwą źieloną / piękną / dla owocu ślicżnego názwał był Pán imię twoje ; ále z ƺumem burzy wielkiej zápáli ją ogniem z góry / gdy połámie gáłęźie jey.
Bo Pán zaſtępów / który ćię był wƺcżepił / wyrzekł złe przećiwko tobie dla złośći domu Izráelſkiego y domu Judzkiego / które cżynili miedzy ſobą / áby mię dráźnili / kádząc Báálowi.
Pán zájiſte oznájmił mi / y dowiedźiáłem śię ; tedyś mi ukázał przedśięwźięćiá ich /
Gdym był jáko báránek y wół / którego wiodą ná rzeź ; bom nie wiedźiał / áby przećiwko mnie rády zmyśláli mówiąc : Popſujmy drzewo z owocem jego / á wykorzeńmy go z źiemi żyjących / áby imię jego nie było więcey wſpomniáne.
Ale / o Pánie zaſtępów! który ſpráwiedliwie ſądźiƺ / á doświádcżáƺ nerek y ſercá / niech widzę pomſtę twoję nád nimi ; bom ći objáwił ſpráwę moję.
Dla tego ták mówi Pán o mężách z Anátot / którzy ƺukáją duƺy twojey / á mówią : Nie prorokuj w imieniu Páńſkim / ábyś nie umárł od rąk náƺych ;
A przetoż ták mówi Pán zaſtępów : Oto ja náwiedzę ich ; młodźieńcy ich pomrą od miecżá / ſynowie ich y córki ich pomrą głodem.
Y nic nie zoſtánie z nich ; bo przywiodę złe ná mężów z Anátot roku náwiedzeniá ich.
