Słowo / które śię ſtáło do Jeremijáƺá od Páná mówiąc :
Słuchájćie ſłów przymierzá tego / którebyśćie mówili do mężów Judzkich y do obywáteli Jeruzálemſkich ;
A rzecżeƺ do nich : Ták mówi Pán / Bóg Izráelſki : Przeklęty ten cżłowiek / któryby nie uſłucháł ſłów przymierzá tego /
Którem przykázáł ojcom wáƺym / dniá / któregom ich wywiódł z źiemi Egipſkiej / z piecá żeláznego / mówiąc : Słuchájćie głoſu mojego / á cżyńćie to wƺyſtko / co wám rozkázuję / y będźiećie ludem moim / á Já będę Bogiem wáƺym ;
Abym ſpełnił przyśięgę / którąm przyśiągł ojcom wáƺym / że im dám źiemię opływájącą mlekiem y miodem : jáko śię to dźiś okázuje. Któremu odpowiedźiáwƺy rzekłem : Amen / Pánie!
Y rzekł Pán do mnie : Obwoływáj wƺyſtkie te ſłowá w miáſtách Judzkich y po ulicách Jeruzálemſkich / mówiąc : Słuchájćie ſłów przymierzá tego / á cżyńćie je.
Bo oświádcżájąc oświádcżáłem śię przed ojcámi wáƺymi ode dniá / któregom ich wywiódł z źiemi Egipſkiej / áż do dniá tego ; ráno wſtáwájąc y oświádcżájąc śię / máwiáłem : Słuchájćie głoſu mego.
Ale nie uſłucháli / áni nákłonili uchá ſwego ; owƺem káżdy ƺedł zá uporem ſercá ſwego złego. Przetożem przywiódł ná nich wƺyſtkie ſłowá przymierzá tego / którem rozkázáł / áby cżynili ; ále oni nie cżynili.
Y rzekł Pán do mnie : Ználázło śię ſprzyśiężenie miedzy mężámi Judzkimi / y między obywátelámi Jeruzálemſkimi ;
Obróćili śię do niepráwośći ojców ſwoich pierwƺych / którzy nie chćieli ſłucháć ſłów moich ; tákże y ći chodzą zá bogámi cudzymi / ſłużąc im ; zgwáłćili dom Izráelſki y dom Judzki przymierze moje / którem był poſtánowił z ojcámi ich.
Dlátego ták mówi Pán : Oto Já przywiodę ná nich złe / z którego nie będą mogli wyjść ; choćby wołáli do mnie / nie wyſłuchám ich.
Y pójdą miáſtá Judzkie y obywátele Jeruzálemſcy / á będą wołáli do bogów / którym kádzą ; ále ich żádnym ſpoſobem nie wybáwią cżáſu utrápieniá ich.
Acżkolwiek ile jeſt miáſt twoich / tyle bogów twoich / o Judo! á ile ulic Jeruzálemſkich / tyleśćie náſtáwiáli ołtárzów obrzydliwośći / ołtárzów do kádzeniá Báálowi.
Przetoż śię ty nie módl zá tym ludem / áni podnoś zá nimi głoſu y modlitwy ; bo Já ich nie wyſłuchám nátencżáſ / gdy do mnie záwołáją w utrápieniu ſwojim.
Cóż miłemu memu do domu mego? poniewáż bez wſtydu páchá złośći z wielomá / á ofiáry święte odeƺły od ćiebie ; y że śię w złośći ſwojej rádujeƺ.
Oliwą źieloną / piękną / dlá owocu ślicżnego názwáł był Pán imię twoje ; ále z ƺumem burzy wielkiej zápáli ją ogniem z góry / gdy połámie gáłęźie jej.
Bo Pán záſtępów / który ćię był wƺcżepił / wyrzekł złe przećiwko tobie dlá złośći domu Izráelſkiego y domu Judzkiego / które cżynili między ſobą / áby mię dráźnili / kádząc Báálowi.
Pán záiſte oznájmił mi / y dowiedźiáłem śię ; tedyś mi ukázáł przedśięwźięćiá ich /
Gdym był jáko báránek y wół / którego wiodą ná rzeź ; bom nie wiedźiáł / áby przećiwko mnie rády zmyśláli mówiąc : Popſujmy drzewo z owocem jego / á wykorzeńmy go z źiemi żyjących / áby imię jego nie było więcej wſpomniáne.
Ale / o Pánie záſtępów! który ſpráwiedliwie ſądźiƺ / á doświádcżáƺ nerek y ſercá / niech widzę pomſtę twoję nád nimi ; bom ći objáwił ſpráwę moję.
Dlátego ták mówi Pán o mężách z Anátot / którzy ƺukáją duƺy twojej / á mówią : Nie prorokuj w imieniu Páńſkim / ábyś nie umárł od rąk náƺych ;
A przetoż ták mówi Pán záſtępów : Oto já náwiedzę ich ; młodźieńcy ich pomrą od miecżá / ſynowie ich y córki ich pomrą głodem.
Y nic nie zoſtánie z nich ; bo przywiodę złe ná mężów z Anátot roku náwiedzeniá ich.
