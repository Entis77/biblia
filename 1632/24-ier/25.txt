Słowo / które śię ſtáło do Jeremijáƺá przećiwko wƺyſtkiemu ludowi Judzkiemu roku cżwártego Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / ( który jeſt rok pierwƺy Nábuchodonozorá / królá Bábilońſkiego ; )
Które mówił Jeremijáƺ prorok do wƺyſtkiego ludu Judzkiego / y do wƺyſtkich obywáteli Jeruzálemſkich mówiąc :
Od trzynáſtego roku Jozyjáƺá / ſyná Amonowego / królá Judzkiego / áż do dniá tego / już to przez dwádzieśćiá y trzy látá bywáło ſłowo Páńſkie do mnie / którem do wáſ máwiáł / ráno wſtáwájąc y opowiádájąc ; áleśćie nie ſłucháli.
Poſyłáł też Pán do wáſ wƺyſtkich ſług ſwoich proroków / ráno wſtáwájąc / y poſyłájąc / ( którycheśćie nie uſłucháli / ániśćie nákłonili uchá ſwego / ábyśćie ſłyƺeli ; )
Którzy mówili : Náwróććie śię teráz káżdy od złej drogi ſwojej / y od złośći ſpráw wáƺych ; á ták będziećie mieƺkáli w tej ziemi / którą wám dáł Pán / y ojcom wáƺym od wieku áż ná wieki.
A nie chodźćie zá bogámi cudzymi / ábyśćie im ſłużyli / y kłániáli śię im / áni mię gniewájćie ſpráwą rąk wáƺych / á Já wám źle nie ucżynię.
Aleśćie mię nie uſłucháli / mówi Pán / ábyśćie mię pobudzáli do gniewu ſpráwą rąk ſwioch ná ſwe złe.
Przetoż ták mówi Pán záſtępów : Dlátego / iżeśćie nie uſłucháli ſłów moich /
Oto Já poślę y pobiorę wƺyſtkie národy północne / mówi Pán / y do Nábuchodonozorá królá Bábilońſkiego / ſługi mego / y przywiodę je ná tę ziemię / y ná obywáteli jej / y ná te wƺyſtkie národy okolicżne / które do ƺcżętu wygłádzę / y położę je ná podziw / y ná poświſtánie / y ná ſpuſtoƺenie wiecżne.
Y ſpráwię to / áby im zginął głoſ weſelá / y głoſ rádośći / głoſ oblubieńcá / y głoſ oblubienicy / głoſ żárn / y świátłość pochodni.
Y będzie tá wƺyſtká ziemiá ſpuſtoƺeniem / y zdumieniem / á ſłużyć będą te národy królowi Bábilońſkiemu śiedmdzieśiąt lát.
Ale potem / gdy śię wypełni śiedmdzieśiąt lát / náwiedzę ná królu Bábilońſkim y ná tym národzie / mówi Pán / niepráwość ich / y ná ziemi Cháldejſkiej / ták / że ją obrócę w puſtynie wiecżne.
A przywiodę ná tę ziemię wƺyſtkie ſłowá moje / którem mówił o niej / miánowićie to wƺyſtko / co nápiſáno w tych kśięgách / cokolwiek prorokowáł Járemijáƺ o wƺyſtkich národách.
Gdyż w niewolę podbići będą od národów / tákże jáko y oni możnych / y od królów wielkich / tedy im oddám według ſpráw ich y według ucżynków rąk ich.
Bo ták rzekł Pán / Bóg Izráelſki / do mnie : Weźmij kubek winá tej popędliwośći z ręki mojej / á nápáwáj nim wƺyſtkie národy / do których Já ćiebie poślę ;
Aby pili y potácżli śię / owƺem / áby ƺáleli od oſtrzá miecżá / który Já poślę między nich.
Wziąłem tedy kubek z ręki Páńſkiej / y nápoiłem wƺyſtkie one národy / do których mię Pán poſłáł :
Jeruzálem / y miáſtá ziemi Judzkiej / y królów jej / y kśiążąt jej / ábym ich podáł ná ſpuſtoƺenie / ná zdumienie / ná poświſtánie / y ná przeklęſtwo / jáko śię to dziś okázuje.
Fáráoná też / królá Egipſkiego / y ſług jego / y kśiążąt jego / y wƺyſtek lud jego ;
Y to wƺyſtko poſpólſtwo / tákże wƺyſtkich króli ziemi Uz / y wƺyſtkich króli ziemi Filiſtyńſkiej / y Aƺkálon / y Gázę / y Akkáron / y oſtátek Azotu ;
Edomcżyków / y Moábcżyków / y ſynów Ammonowych ;
Y wƺyſtkich królów Tyrſkich / y wƺyſtkich królów Sydońſkich / y królów tej kráiny / którá jeſt przy morzu ;
Dedáná y Temę / y Buzę / y wƺyſtkich / którzy mieƺkáją w oſtátnich kątách :
Y wƺyſtkich królów Arábſkich / y wƺyſtkich królów tego poſpólſtwá / które mieƺká ná puƺcży ;
Tákże wƺyſtkich królów Zymry y wƺyſtkich królów Elám / y wƺyſtkich królów Medſkich ;
Owƺem / wƺyſtkich królów północnych / bliſkich y dálekich / jednego jáko drugiego ; wƺyſtkie też króleſtwá ziemi / którekolwiek ſą ná oblicżu ziemi ; á król Seſák będzie pił po nich.
Y rzecż do nich : Ták mówi Pán záſtępów / Bóg Izráelſki : Pijćie / á popijćie śię / owƺem zwrácájćie / y pádájćie ták / ábyśćie nie powſtáli dlá miecżá / który Já poślę między wáſ.
A jeźliby nie chćieli wziąć kubká z ręki twojej / áby pili / tedy rzecżeƺ do nich : Ták mówi Pán záſtępów : Koniecżnie pić muśićie.
Bo poniewáż ná to miáſto / które názwáne jeſt od imieniá mego / Já zácżynám przywodzić złe rzecż / á wybyśćie bez kárániá być mieli? Nie będziećie bez kárániá ; bom Já miecż przyzwáł ná wƺyſtkich obywáteli tej ziemi / mówi Pán záſtępów.
Przetoż ty prorokuj przećiwko nim te wƺyſkie ſłowá / á mów do nich : Pán z wyſoká zárycży / á z mieƺkániá świętobliwośći ſwojej wydá głoſ ſwój / rycżąc zárycży z mieƺkániá ſwego ; krzyk pobudzájących śię / jáko tłocżących práſę / rozlegáć śię będzie przećiwko wƺyſtkim obywátelom tej ziemi.
Y przejdzie huk áż do końcżyn ziemi ; bo śię Pán rozpierá z tymi národámi / w ſąd ſám wchodzi ze wƺelkiem ćiáłem / niezbożnych podá pod miecż / mówi Pán.
Ták mówi Pán záſtępów : Oto udręcżenie pójdzie z národu do národu / á wicher wielki powſtánie od końcżyn ziemi ;
Y będą pobići od Páná cżáſu onego od końcá ziemi áż do końcá ziemi ; nie będą ich płákáć / áni zbieráć / áni chowáć ; będą jáko gnój ná polu.
Nárzekájćie páſterze y wołájćie / á wálájćie śię w popiele / wy nájzácniejśi tej trzody! bo śię wypełniły dni wáƺe / zábićiá y rozproƺeniá wáƺego / y upádniećie jáko nácżynie drogie.
Y zginie ućiecżká páſterzom / á ujśćie nájzácniejƺym tej trzody.
Głoſ wołániá páſterzy / y nárzekánie nájzácniejƺych tej trzody ſłycháć będzie ; bo Pán ſpuſtoƺy páſtwiſká ich.
Y zágubione będą ſpokojne páſtwiſká dlá zápálcżywośći gniewu Páńſkiego /
Który opuśći jáko lew jáſkinię ſwoję ; bo ziemiá ich przyjdzie ná ſpuſtoƺenie dlá zápálcżywośći puſtoƺyćielá / y dlá popędliwośći gniewu jego.
