Słowá Jeremijáƺá / ſyná Helkijáƺowego / z kápłánów / którzy byli w Anátot / w ziemi Benjámin.
Do którego ſtáło śię ſłowo Páńſkie zá dni Jozyjáƺá / ſyná Amonowego / królá Judzkiego / trzynáſtego roku królowániá jego.
A ſtáło śię zá dni Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / áż do ſkońcżeniá jedenáſtego roku Sedekijáƺá / ſyná Jozyjáƺá / królá Judzkiego / áż do przeprowádzeniá Jeruzálemcżyków / mieśiącá piątego ;
Stáło śię / mówię / ſłowo Páńſkie do mnie / mówiąc :
Pierwej niżelim ćię utworzył w żywoćie / znáłem ćię / á pierwej niżeliś wyƺedł z żywotá / poświęćiłem ćię ; zá proroká národom dáłem ćię /
Y rzekłem : Ach / ách pánujący Pánie! oto nie umiem mówić / bom jeſt dziećięćiem.
Ale Pán rzekł do mnie : Nie mów : Jeſtem dziećięćiem ; owƺem / ná wƺyſtko / ná co ćię poślę / idź / á wƺyſtko / coć rozkáżę / mów.
Nie bój śię oblicżá ich ; bom Já jeſt z tobą / ábym ćię wybáwił / mówi Pán.
A wyćiągnąwƺy Pán rękę ſwoję / dotknął śię uſt moich / y rzekł mi Pán : Otom dáł ſłowá moje do uſt twoich.
Oto ćię dziś poſtánáwiám nád národámi y nád króleſtwámi / ábyś wykorzeniáł / y pſuł / y wytrácáł / y obáláł / ábyś budowáł y ƺcżepił.
Potem ſtáło śię ſłowo Páńſkie do mnie / mówiąc : Co widziƺ Jeremijáƺu? Y rzekłem : Widzę rózgę migdáłową.
Y rzekł Pán do mnie : Dobrze widziƺ ; ábowiem śię Já poſpieƺám z ſłowem ſwem / ábym je wykonáł.
Y ſtáło śię ſłowo Páńſkie do mnie powtóre / mówiąc : Co widziƺ? Y rzekłem : Widzę gárniec wrzący / á przedniá ſtroná jego ku ſtronie północnej.
Y rzekł Pán do mnie : Od północy przypádnie złe ná wƺyſtkich mieƺkájących ná tej ziemi /
Bo oto Já záwołám wƺyſtkich rodzájów z króleſtw północnych / mówi Pán / áby przyćiągnąwƺy káżdy z nich / poſtánowił ſtolicę ſwoję w wejśćiu brám Jeruzálemſkich / y przy wƺyſtkich murách jego / y przy wƺyſtkich miáſtách Judzkich.
A ták opowiem ſądy moje przećiwko nim dlá wƺelákiej złośći tych / którzy mię opuśćili / á kádzili bogom obcym / y kłániáli śię roboćie rąk ſwoich.
Przetoż ty przepáƺ biodrá ſwoje / á wſtáwƺy mów do nich wƺyſtko / co Já tobie rozkázuję : nie bój śię ich / bym ćię ſnáć nie ſtárł przed oblicżem ich ;
Bo oto Já poſtánáwiám ćię dziś miáſtem obronnem / y ſłupem żeláznym / y murem miedziánym przećiwko tej wƺyſtkiej ziemi / przećiwko królom Judzkim / przećiwko kśiążętom ich / przećiwko kápłánom ich / y przećiwko ludowi tej ziemi ;
Którzy wálcżyć będą przećiwko tobie / ále ćię nie przemogą ; bom Já z tobą / mówi Pán / ábym ćię wybáwił.
