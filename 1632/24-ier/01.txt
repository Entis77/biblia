Słowá Jeremijáƺá / ſyná Helkijáƺowego / z kápłanów / którzy byli w Anátot / w źiemi Benjámin.
Do którego ſtáło śię ſłowo Páńſkie zá dni Jozyjáƺá / ſyná Amonowego / królá Judzkiego / trzynáſtego roku królowánia jego.
A ſtáło śię zá dni Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / áż do ſkońcżeniá jedenáſtego roku Sedekijáƺá / ſyná Jozyjáƺá / królá Judzkiego / áż do przeprowádzeniá Jeruzálemcżyków / mieśiącá piątego ;
Stáło śię / mówię / ſłowo Páńſkie do mnie / mówiąc :
Pierwej niżelim ćię utworzył w żywoćie / znáłem ćię / á pierwej niżeliś wyƺedł z żywotá / poświęćiłem ćię ; zá proroká národom dáłem ćię /
Y rzekłem : Ach / ách pánujący Pánie! oto nie umiem mówić / bom jeſt dźiećięćiem.
Ale Pán rzekł do mnie : Nie mów : Jeſtem dźiećięćiem ; owƺem / ná wƺyſtko / ná co ćię poślę / idź / á wƺyſtko / coć rozkáżę / mów.
Nie bój śię oblicża ich ; bom Ja jeſt z tobą / ábym ćię wybáwił / mówi Pán.
A wyćiągnąwƺy Pán rękę ſwoję / dotknął śię uſt mojich / y rzekł mi Pán : Otom dał ſłowá moje do uſt twojich.
Oto ćię dźiś poſtánáwiám nád národámi y nád króleſtwámi / ábyś wykorzeniał / y pſuł / y wytrácał / y obálał / ábyś budował y ƺcżepił.
Potem ſtáło śię ſłowo Páńſkie do mnie / mówiąc : Co widźiƺ Jeremijáƺu? Y rzekłem : Widzę rózgę migdáłową.
Y rzekł Pán do mnie : Dobrze widźiƺ ; ábowiem śię Ja poſpieƺám z ſłowem ſwem / ábym je wykonał.
Y ſtáło śię ſłowo Páńſkie do mnie powtóre / mówiąc : Co widźiƺ? Y rzekłem : Widzę gárniec wrzący / á przedniá ſtroná jego ku ſtronie północney.
Y rzekł Pán do mnie : Od północy przypádnie złe ná wƺyſtkich mieƺkájących ná tey źiemi /
Bo oto Ja záwołám wƺyſtkich rodzájów z króleſtw północnych / mówi Pán / áby przyćiągnąwƺy káżdy z nich / poſtánowił ſtolicę ſwoję w wejśćiu bram Jeruzálemſkich / y przy wƺyſtkich murách jego / y przy wƺyſtkich miáſtách Judzkich.
A ták opowiem ſądy moje przećiwko nim dla wƺelákiej złośći tych / którzy mię opuśćili / á kádźili bogom obcym / y kłániáli śię roboćie rąk ſwojich.
Przetoż ty przepáƺ biodrá ſwoje / á wſtáwƺy mów do nich wƺyſtko / co Ja tobie rozkázuję : nie bój śię ich / bym ćię ſnadź nie ſtárł przed oblicżem ich ;
Bo oto Ja poſtánáwiám ćię dźiś miáſtem obronnem / y ſłupem żeláznym / y murem miedźiánym przećiwko tey wƺyſtkiej źiemi / przećiwko królom Judzkim / przećiwko kśiążętom ich / przećiwko kápłánom ich / y przećiwko ludowi tey źiemi ;
Którzy wálcżyć będą przećiwko tobie / ále ćię nie przemogą ; bom Ja z tobą / mówi Pán / ábym ćię wybáwił.
