Słowá Jeremijáƺá / ſyná Helkijáƺowego / z kápłánów / którzy byli w Anátot / w źiemi Benjámin.
Do którego ſtáło śię ſłowo Páńſkie zá dni Jozyjáƺá / ſyná Amonowego / królá Judzkiego / trzynáſtego roku królowániá jego.
A ſtáło śię zá dni Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / áż do ſkońcżeniá jedenáſtego roku Sedekijáƺá / ſyná Jozyjáƺá / królá Judzkiego / áż do przeprowádzeniá Jeruzálemcżyków / mieśiącá piątego ;
Stáło śię / mówię / ſłowo Páńſkie do mnie / mówiąc :
Pierwej niżelim ćię utworzył w żywoćie / znáłem ćię / á pierwej niżeliś wyƺedł z żywotá / poświęćiłem ćię ; zá proroká národom dáłem ćię /
Y rzekłem : Ach / ách pánujący Pánie! oto nie umiem mówić / bom jeſt dźiećięćiem.
Ale Pán rzekł do mnie : Nie mów : Jeſtem dźiećięćiem ; owƺem / ná wƺyſtko / ná co ćię poślę / idź / á wƺyſtko / coć rozkáżę / mów.
Nie bój śię oblicżá ich ; bom Já jeſt z tobą / ábym ćię wybáwił / mówi Pán.
A wyćiągnąwƺy Pán rękę ſwoję / dotknął śię uſt moich / y rzekł mi Pán : Otom dáł ſłowá moje do uſt twoich.
Oto ćię dźiś poſtánáwiám nád národámi y nád króleſtwámi / ábyś wykorzeniáł / y pſuł / y wytrácáł / y obáláł / ábyś budowáł y ƺcżepił.
Potem ſtáło śię ſłowo Páńſkie do mnie / mówiąc : Co widźiƺ Jeremijáƺu? Y rzekłem : Widzę rózgę migdáłową.
Y rzekł Pán do mnie : Dobrze widźiƺ ; ábowiem śię Já poſpieƺám z ſłowem ſwem / ábym je wykonáł.
Y ſtáło śię ſłowo Páńſkie do mnie powtóre / mówiąc : Co widźiƺ? Y rzekłem : Widzę gárniec wrzący / á przedniá ſtroná jego ku ſtronie północnej.
Y rzekł Pán do mnie : Od północy przypádnie złe ná wƺyſtkich mieƺkájących ná tej źiemi /
Bo oto Já záwołám wƺyſtkich rodzájów z króleſtw północnych / mówi Pán / áby przyćiągnąwƺy káżdy z nich / poſtánowił ſtolicę ſwoję w wejśćiu brám Jeruzálemſkich / y przy wƺyſtkich murách jego / y przy wƺyſtkich miáſtách Judzkich.
A ták opowiem ſądy moje przećiwko nim dlá wƺelákiej złośći tych / którzy mię opuśćili / á kádźili bogom obcym / y kłániáli śię roboćie rąk ſwoich.
Przetoż ty przepáƺ biodrá ſwoje / á wſtáwƺy mów do nich wƺyſtko / co Já tobie rozkázuję : nie bój śię ich / bym ćię ſnáć nie ſtárł przed oblicżem ich ;
Bo oto Já poſtánáwiám ćię dźiś miáſtem obronnem / y ſłupem żeláznym / y murem miedźiánym przećiwko tej wƺyſtkiej źiemi / przećiwko królom Judzkim / przećiwko kśiążętom ich / przećiwko kápłánom ich / y przećiwko ludowi tej źiemi ;
Którzy wálcżyć będą przećiwko tobie / ále ćię nie przemogą ; bom Já z tobą / mówi Pán / ábym ćię wybáwił.
