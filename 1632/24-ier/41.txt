Y ſtáło śię mieśiącá śiódmego / że przyƺedł Izmáel / ſyn Nátánijáƺá / ſyná Eliſámowego / z náśieniá królewſkiego / y hetmáni królewſcy / to jeſt / dźieśięć mężów z nim / do Godolijáƺá / ſyná Ahikámowego / do Máſfy / y jedli tám ſpołu chleb w Máſfie.
Potem wſtáwƺy Izmáel / ſyn Nátánijáƺowy / y dźieśięć mężów / którzy z nim byli / zábili Godolijáƺá / ſyná Ahikámá / ſyná Sáfánowego / miecżem ; zábili mówię tego / którego był przełożył król Bábilońſki nád oną źiemią.
Wƺyſtkich tákże Żydów / którzy byli z nim / z Godolijáƺem w Máſfie / y onych Cháldejcżyków / których tám ználázł / mężów wálecżnych / pobił Izmáel.
A dniá wtórego / gdy zábił Godolijáƺá / ( o cżym nikt nie zwiedźiał / )
Przyƺli niektórzy z Sychem / z Sylo / y z Sámáryi / mężów ośmdźieśiąt / ogoliwƺy brody / y rozdárƺy ƺáty / y podrápáwƺy śię / którzy ofiárę śniedną / y kádźidło w rękách ſwych mieli / áby je odnieśli do domu Páńſkiego.
Tedy Izmáel / ſyn Nátánijáƺowy / wyƺedł przećiwko nim z Máſfy / á idąc ƺedł y płákał ; á gdy śię ſpotkał z nimi / rzekł do nich : Pójdźćie do Godolijáƺá / ſyná Ahikámowego.
Ale gdy przyƺli w pośród miáſtá / pobił ich Izmáel / ſyn Nátánijáƺowy / y wrzućił ich w dół / ſám y mężowie / którzy z nim byli.
Lecż śię dźieśięć mężów ználázło miedzy nimi / którzy rzekli do Izmáelá : Nie zábijáj nas ; bo mamy ſkárby ſkryte w polu / pƺenicy y jęcżmieniá / y oliwy / y miodu. Y pohámował śię ; á nie zábił ich miedzy bráćmi ich.
A dół / do którego Izmáel wrzućił do Godolijáƺá wƺyſtkie trupy onych mężów / których pobił / ten jeſt / który ucżynił król Azá / bojąc śię Báázy / królá Izráelſkiego / który nápełnił Izmáel / ſyn Nátánijáƺowy / pobitymi.
Y pobrał w niewolę Izmáel wƺyſtkie oſtátki ludu / które były w Máſfie / córki królewſkie / y wƺyſtek lud / który był zoſtał w Máſfie / które był porucżył Nábuzárdán / hetmán żołnierſki / Godolijáƺowi / ſynowi Ahikámowemu / y wźiął je w pojmánie Izmáel / ſyn Nátánijáƺowy / y poƺedł uchodząc do ſynów Ammonowych.
Wtem uſłyƺał Johánán / ſyn Káreáƺowy / y wƺyſcy hetmáni onych wojſk / którzy byli z nim / o tym wƺyſtkiem złem / które ucżynił Izmáel / ſyn Nátánijáƺowy ;
Y wźięli wƺyſtek ſwój lud / y ćiągnęli / áby zwiedli bitwę z Izmáelem / ſynem Nátánijáƺowym / którego ználeſli u wód wielkich / które ſą w Gábáon.
A gdy ujrzał wƺyſtek lud / który był z Izmáelem / Johánáná / ſyná Káreáƺowego / y wƺyſtkich kśiążąt wojſk / którzy z nim byli / urádowáli śię ;
A obróćiwƺy śię wƺyſtek on lud / który był wźiął w niewolę Izmáel z Máſfy / wróćił śię záśię / á przyƺedł do Johánáná / ſyná Káreáƺowego.
Ale Izmáel / ſyn Nátánijáƺowy / uƺedł z ośmiomá mężámi przed Johánánem / y przyƺedł do ſynów Ammonowych.
Przetoż wźiął Johánán / ſyn Káreáƺowy / y wƺyſcy kśiążętá wojſk / którzy z nim byli / wƺyſtek oſtátek ludu / który záś przywiódł od Izmáelá / ſyná Nátánijáƺowego / z Máſfy / gdy zábił Godolijáƺá / ſyná Ahikámowego / mężów wálecżnych / y niewiáſty / y dźiátki / y komorników / których záś przywiódł z Gábáonu.
A odƺedƺy pomieƺkáli w goſpodźie Chimchámowej / która jeſt u Bethlehemu / áby idąc uƺli do Egiptu przed Cháldejcżykámi ;
Bo śię ich báli / przeto / że był zábił Izmáel / ſyn Nátánijáƺowy / Godolijáƺá / ſyná Ahikámowego / którego był przełożył król Bábilońſki nád oną źiemią.
