Y ſtáło śię mieśiącá śiódmego / że przyƺedł Izmáel / ſyn Nátánijáƺá / ſyná Eliſámowego / z náśieniá królewſkiego / y hetmáni królewſcy / to jeſt / dzieśięć mężów z nim / do Godolijáƺá / ſyná Ahikámowego / do Máſfy / y jedli tám ſpołu chleb w Máſfie.
Potem wſtáwƺy Izmáel / ſyn Nátánijáƺowy / y dzieśięć mężów / którzy z nim byli / zábili Godolijáƺá / ſyná Ahikámá / ſyná Sáfánowego / miecżem ; zábili mówię tego / którego był przełożył król Bábilońſki nád oną ziemią.
Wƺyſtkich tákże Żydów / którzy byli z nim / z Godolijáƺem w Máſfie / y onych Cháldejcżyków / których tám ználázł / mężów wálecżnych / pobił Izmáel.
A dniá wtórego / gdy zábił Godolijáƺá / ( o cżem nikt nie zwiedziáł / )
Przyƺli niektórzy z Sychem / z Sylo / y z Sámáryi / mężów ośmdzieśiąt / ogoliwƺy brody / y rozdárłƺy ƺáty / y podrápáwƺy śię / którzy ofiárę śniedną / y kádzidło w rękách ſwych mieli / áby je odnieśli do domu Páńſkiego.
Tedy Izmáel / ſyn Nátánijáƺowy / wyƺedł przećiwko nim z Máſfy / á idąc ƺedł y płákáł ; á gdy śię ſpotkáł z nimi / rzekł do nich : Pójdźćie do Godolijáƺá / ſyná Ahikámowego.
Ale gdy przyƺli w pośród miáſtá / pobił ich Izmáel / ſyn Nátánijáƺowy / y wrzućił ich w dół / ſám y mężowie / którzy z nim byli.
Lecż śię dzieśięć mężów ználázło między nimi / którzy rzekli do Izmáelá : Nie zábijáj náſ ; bo mámy ſkárby ſkryte w polu / pƺenicy y jęcżmieniá / y oliwy / y miodu. Y pohámowáł śię ; á nie zábił ich między bráćmi ich.
A dół / do którego Izmáel wrzućił do Godolijáƺá wƺyſtkie trupy onych mężów / których pobił / ten jeſt / który ucżynił król Azá / bojąc śię Báázy / królá Izráelſkiego / który nápełnił Izmáel / ſyn Nátánijáƺowy / pobitymi.
Y pobráł w niewolę Izmáel wƺyſtkie oſtátki ludu / które były w Máſfie / córki królewſkie / y wƺyſtek lud / który był zoſtáł w Máſfie / które był porucżył Nábuzárdán / hetmán żołnierſki / Godolijáƺowi / ſynowi Ahikámowemu / y wziął je w pojmánie Izmáel / ſyn Nátánijáƺowy / y poƺedł uchodząc do ſynów Ammonowych.
Wtem uſłyƺáł Johánán / ſyn Káreáƺowy / y wƺyſcy hetmáni onych wojſk / którzy byli z nim / o tem wƺyſtkiem złem / które ucżynił Izmáel / ſyn Nátánijáƺowy ;
Y wzięli wƺyſtek ſwój lud / y ćiągnęli / áby zwiedli bitwę z Izmáelem / ſynem Nátánijáƺowym / którego ználeźli u wód wielkich / które ſą w Gábáon.
A gdy ujrzáł wƺyſtek lud / który był z Izmáelem / Johánáná / ſyná Káreáƺowego / y wƺyſtkich kśiążąt wojſk / którzy z nim byli / urádowáli śię ;
A obróćiwƺy śię wƺyſtek on lud / który był wziął w niewolę Izmáel z Máſfy / wróćił śię záśię / á przyƺedł do Johánáná / ſyná Káreáƺowego.
Ale Izmáel / ſyn Nátánijáƺowy / uƺedł z ośmiomá mężámi przed Johánánem / y przyƺedł do ſynów Ammonowych.
Przetoż wziął Johánán / ſyn Káreáƺowy / y wƺyſcy kśiążętá wojſk / którzy z nim byli / wƺyſtek oſtátek ludu / który záś przywiódł od Izmáelá / ſyná Nátánijáƺowego / z Máſfy / gdy zábił Godolijáƺá / ſyná Ahikámowego / mężów wálecżnych / y niewiáſty / y dziátki / y komorników / których záś przywiódł z Gábáonu.
A odƺedłƺy pomieƺkáli w goſpodzie Chimchámowej / którá jeſt u Bethlehemu / áby idąc uƺli do Egiptu przed Cháldejcżykámi ;
Bo śię ich báli / przeto / że był zábił Izmáel / ſyn Nátánijáƺowy / Godolijáƺá / ſyná Ahikámowego / którego był przełożył król Bábilońſki nád oną ziemią.
