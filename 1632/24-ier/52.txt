Dwádźieśćiá y jeden lat miał Sedekijáƺ / gdy królowáć pocżął / á jedenaśćie lat królował w Jeruzálemie ; á imię mátki jego było Chámutál / córká Jeremijáƺowá z Lebny ;
Y cżynił złość przed ocżymá Páńſkimi według wƺyſtkiego / co cżynił Joákim.
Abowiem śię to ſtáło dla rozgniewánia Páńſkiego przećiwko Jeruzálemowi y Judźie / áż ich odrzućił od twárzy ſwej. Wtem záśię odſtąpił Sedekijáƺ od królá Bábilońſkiego.
Y ſtáło śię roku dźiewiątego króleſtwá jego / mieśiącá dźieśiątego / dniá dźieśiątego tegoż mieśiącá / że przyćiągnął Nábuchodonozor / król Bábilońſki / on y wƺyſtko wojſko jego przećiwko Jeruzálemowi / y położył śię obozem u niego / y porobił przećiwko niemu ƺáńce w około.
A ták było miáſto oblężone áż do jedenáſtego roku królá Sedekijáƺá.
Tedy mieśiącá cżwártego / dźiewiątego dniá tegoż mieśiącá / był wielki głód w mieśćie / y nie miał chlebá lud oney źiemi.
Y przełámáno mur miejſki / á wƺyſcy ludźie rycerſcy poućiekáli / y wyƺli z miáſtá w nocy drogą do bramy / która jeſt miedzy dwomá murámi podle ogrodu królewſkiego ; ( ále Cháldejcżycy leżeli około miáſtá / ) y poƺli drogą ku puſtyni.
Y goniło wojſko Cháldejſkie królá / á dośćignęli Sedekijáƺá ná polách u Jerychá / á wƺyſtko wojſko jego rozpierzchnęło śię od niego.
A ták pojmáwƺy królá przywiedli go do królá Bábilońſkiego do Rybláty w źiemi Emát / kędy o nim ucżynił ſąd.
Y pozábijał król Bábilońſki ſynów Sedekijáƺowych przed ocżymá jego / tákże też wƺyſtkich kśiążąt Judzkich pozábijał w Rybláćie.
A Sedekijáƺá oślepiwƺy y związáwƺy go łáńcuchámi miedźiánemi / záwiódł go król Bábilońſki do Bábilonu / y podał go do domu więźieniá áż do śmierći jego.
Potem mieśiącá piątego / dniá dźieśiątego tegoż mieśiącá / ten jeſt rok dźiewiętnáſty królowánia Nábuchodonozorá / królá Bábilońſkiego / przyćiągnął Nábuzárdán / hetmán żołnierſki / który ſtáwał przed królem Bábilońſkim / do Jeruzálemu.
Y ſpálił dom Páńſki / y dom królewſki / y wƺyſtkie domy Jeruzálemſkie ; owá wƺyſtko budowánie koƺtowne popálił ogniem.
Y wƺyſtkie mury Jeruzálemſkie w około rozwáliło wƺyſtko wojſko Cháldejſkie / które było z onym hetmánem żołnierſkim.
A z ubogich ludźi y z oſtátku poſpólſtwá / które było pozoſtáło w mieśćie / y zbiegów / którzy byli zbiegli do królá Bábilońſkiego / y inne poſpólſtwo przenióſł Nábuzárdán hetmán żołnierſki.
Tylko z ubogich oney źiemi zoſtáwił Nábuzárdán / hetmán żołnierſki / áby byli winiárzámi y orácżámi.
Nádto ſłupy miedźiáne / które były w domu Páńſkim / y podſtáwki / y morze miedźiáne / które było w domu Páńſkim / potłukli Cháldejcżycy / y przenieśli wƺyſtkę miedź ich do Bábilonu ;
Kotły też y łopáty / y nacżynia muzycżne / y miednice / y cżáƺe / y wƺyſtko nacżynie miedźiáne / którem uſługiwano / pobráli ;
Nádto wiádrá / y kádźielnice / y miednice / y gárnce / y świecżniki / y cżáƺki / y kufle / co było złotego w złoćie / á co było ſrebrnego w ſrebrze / pobrał hetmán żołnierſki ;
Słupy dwá / morze jedno / y wołów miedźiánych dwánaśćie / które były pod podſtáwkámi / które był ſpráwił król Sálomon w domu Páńſkim ; nie było wági miedźi onego wƺyſtkiego nacżynia.
A z tych ſłupów ośmnaśćie łokći wzwyż był ſłup jeden / á w mięƺ w około dwánaśćie łokći / á w miążƺość jego cżtery pálce / á wewnątrz był dęty ;
A gáłká ná nim miedźiáná / á wyſokość gáłki jedney byłá ná pięć łokći / śiátká też y jábłká gránátowe ná gáłce w około wƺyſtko miedźiáne ; táki też był y drugi ſłup z jábłkámi gránátowemi ;
A było jábłek gránátowych dźiewięćdźieśiąt y ƺeść po káżdej ſtronie ; wƺyſtkich jábłek gránátowych było po ſto ná śiátce w około.
Wźiął też hetmán żołnierſki Sárájego / kápłáná przedniego / y Sofonijáƺá / kápłáná wtórego po nim / y trzech ſtróżów progu.
Wźiął też z miáſtá dworzániná niektórego / który był przełożonym nád ludem rycerſkim / y śiedmiu mężów z tych / którzy ſtáwáli przed królem / którzy śię ználeſli w mieśćie / y piſárzá przedniego wojſkowego / który ſpiſywał wojſko z ludu źiemi / y ƺeśćdźieśiąt mężów z ludu źiemi / którzy śię ználeſli w mieśćie.
Wźiąwƺy ich tedy Nábuzárdán / hetmán żołnierſki / záwiódł ich do królá Bábilońſkiego do Rybláty ;
Y pobił ich król Bábilońſki / á pomordował ich w Rybláćie w źiemi Emát. A ták przenieśiony jeſt Judá z źiemi ſwojey.
Tenći jeſt lud / który záprowádźił Nábuchodonozor roku śiódmego : Żydów trzy tyśiące / y dwádźieśćiá y trzy.
Roku ośmnáſtego Nábuchodonozorá / záprowádźił z Jeruzálemu duƺ ośm ſet / trzydźieśći y dwie.
Roku dwudźieſtego y trzećiego Nábuchodonozorá ; záprowádźił Nábuzárdán / hetmán żołnierſki / z Żydów duƺ śiedm ſet / cżterdźieśći y pięć ; wƺyſtkich duƺ cżtery tyśiące y ƺeść ſet.
A trzydźieſtego y śiódmego roku / po pojmániu Joáchyná / królá Judzkiego / dwunáſtego mieśiącá / dwudźieſtego y piątego dniá tegoż mieśiącá / wywyżƺył Ewilmerodách / król Bábilońſki / tego roku / gdy pocżął królowáć / głowę Joáchyná / królá Judzkiego / uwolniwƺy go z domu więźieniá ;
Y rozmáwiał z nim łáſkáwie / y wyſtáwił ſtolicę jego nád ſtolice królów / którzy byli z nim w Bábilonie.
Odmienił też y odźienie / w którym był w więźieniu / y jádał chleb záwƺe przed oblicżem jego po wƺyſtkie dni żywotá ſwego.
Obrok też jemu náznácżony / obrok uſtáwicżny dáwano mu od królá Bábilońſkiego ná káżdy dźień áż do śmierći jego / po wƺyſtkie dni żywotá jego.
