Dwádzieśćiá y jeden lát miáł Sedekijáƺ / gdy królowáć pocżął / á jedenáśćie lát królowáł w Jeruzálemie ; á imię mátki jego było Chámutál / córká Jeremijáƺowá z Lebny ;
Y cżynił złość przed ocżymá Páńſkimi według wƺyſtkiego / co cżynił Joákim.
Abowiem śię to ſtáło dlá rozgniewániá Páńſkiego przećiwko Jeruzálemowi y Judzie / áż ich odrzućił od twárzy ſwej. Wtem záśię odſtąpił Sedekijáƺ od królá Bábilońſkiego.
Y ſtáło śię roku dziewiątego króleſtwá jego / mieśiącá dzieśiątego / dniá dzieśiątego tegoż mieśiącá / że przyćiągnął Nábuchodonozor / król Bábilońſki / on y wƺyſtko wojſko jego przećiwko Jeruzálemowi / y położył śię obozem u niego / y porobił przećiwko niemu ƺáńce w około.
A ták było miáſto oblężone áż do jedenáſtego roku królá Sedekijáƺá.
Tedy mieśiącá cżwártego / dziewiątego dniá tegoż mieśiącá / był wielki głód w mieśćie / y nie miáł chlebá lud onej ziemi.
Y przełámáno mur miejſki / á wƺyſcy ludzie rycerſcy poućiekáli / y wyƺli z miáſtá w nocy drogą do brámy / którá jeſt między dwomá murámi podle ogrodu królewſkiego ; ( ále Cháldejcżycy leżeli około miáſtá / ) y poƺli drogą ku puſtyni.
Y goniło wojſko Cháldejſkie królá / á dośćignęli Sedekijáƺá ná polách u Jerychá / á wƺyſtko wojſko jego rozpierzchnęło śię od niego.
A ták pojmáwƺy królá przywiedli go do królá Bábilońſkiego do Rybláty w ziemi Emát / kędy o nim ucżynił ſąd.
Y pozábijáł król Bábilońſki ſynów Sedekijáƺowych przed ocżymá jego / tákże też wƺyſtkich kśiążąt Judzkich pozábijáł w Rybláćie.
A Sedekijáƺá oślepiwƺy y związáwƺy go łáńcuchámi miedziánemi / záwiódł go król Bábilońſki do Bábilonu / y podáł go do domu więzieniá áż do śmierći jego.
Potem mieśiącá piątego / dniá dzieśiątego tegoż mieśiącá / ten jeſt rok dziewiętnáſty królowániá Nábuchodonozorá / królá Bábilońſkiego / przyćiągnął Nábuzárdán / hetmán żołnierſki / który ſtáwáł przed królem Bábilońſkim / do Jeruzálemu.
Y ſpálił dom Páńſki / y dom królewſki / y wƺyſtkie domy Jeruzálemſkie ; owá wƺyſtko budowánie koƺtowne popálił ogniem.
Y wƺyſtkie mury Jeruzálemſkie w około rozwáliło wƺyſtko wojſko Cháldejſkie / które było z onym hetmánem żołnierſkim.
A z ubogich ludzi y z oſtátku poſpólſtwá / które było pozoſtáło w mieśćie / y zbiegów / którzy byli zbiegli do królá Bábilońſkiego / y inne poſpólſtwo przenióſł Nábuzárdán hetmán żołnierſki.
Tylko z ubogich onej ziemi zoſtáwił Nábuzárdán / hetmán żołnierſki / áby byli winiárzámi y orácżámi.
Nádto ſłupy miedziáne / które były w domu Páńſkim / y podſtáwki / y morze miedziáne / które było w domu Páńſkim / potłukli Cháldejcżycy / y przenieśli wƺyſtkę miedź ich do Bábilonu ;
Kotły też y łopáty / y nácżyniá muzycżne / y miednice / y cżáƺe / y wƺyſtko nácżynie miedziáne / którem uſługiwáno / pobráli ;
Nádto wiádrá / y kádzielnice / y miednice / y gárnce / y świecżniki / y cżáƺki / y kufle / co było złotego w złoćie / á co było ſrebrnego w ſrebrze / pobráł hetmán żołnierſki ;
Słupy dwá / morze jedno / y wołów miedziánych dwánáśćie / które były pod podſtáwkámi / które był ſpráwił król Sálomon w domu Páńſkim ; nie było wági miedzi onego wƺyſtkiego nácżyniá.
A z tych ſłupów ośmnáśćie łokći wzwyż był ſłup jeden / á w mięƺ w około dwánáśćie łokći / á w miążƺość jego cżtery pálce / á wewnątrz był dęty ;
A gáłká ná nim miedziáná / á wyſokość gáłki jednej byłá ná pięć łokći / śiátká też y jábłká gránátowe ná gáłce w około wƺyſtko miedziáne ; táki też był y drugi ſłup z jábłkámi gránátowemi ;
A było jábłek gránátowych dziewięćdzieśiąt y ƺeść po káżdej ſtronie ; wƺyſtkich jábłek gránátowych było po ſto ná śiátce w około.
Wziął też hetmán żołnierſki Sárájego / kápłáná przedniego / y Sofonijáƺá / kápłáná wtórego po nim / y trzech ſtróżów progu.
Wziął też z miáſtá dworzániná niektórego / który był przełożonym nád ludem rycerſkim / y śiedmiu mężów z tych / którzy ſtáwáli przed królem / którzy śię ználeźli w mieśćie / y piſárzá przedniego wojſkowego / który ſpiſywáł wojſko z ludu ziemi / y ƺeśćdzieśiąt mężów z ludu ziemi / którzy śię ználeźli w mieśćie.
Wziąwƺy ich tedy Nábuzárdán / hetmán żołnierſki / záwiódł ich do królá Bábilońſkiego do Rybláty ;
Y pobił ich król Bábilońſki / á pomordowáł ich w Rybláćie w ziemi Emát. A ták przenieśiony jeſt Judá z ziemi ſwojej.
Tenći jeſt lud / który záprowádził Nábuchodonozor roku śiódmego : Żydów trzy tyśiące / y dwádzieśćiá y trzy.
Roku ośmnáſtego Nábuchodonozorá / záprowádził z Jeruzálemu duƺ ośm ſet / trzydzieśći y dwie.
Roku dwudzieſtego y trzećiego Nábuchodonozorá ; záprowádził Nábuzárdán / hetmán żołnierſki / z Żydów duƺ śiedm ſet / cżterdzieśći y pięć ; wƺyſtkich duƺ cżtery tyśiące y ƺeść ſet.
A trzydzieſtego y śiódmego roku / po pojmániu Joáchyná / królá Judzkiego / dwunáſtego mieśiącá / dwudzieſtego y piątego dniá tegoż mieśiącá / wywyżƺył Ewilmerodách / król Bábilońſki / tego roku / gdy pocżął królowáć / głowę Joáchyná / królá Judzkiego / uwolniwƺy go z domu więzieniá ;
Y rozmáwiáł z nim łáſkáwie / y wyſtáwił ſtolicę jego nád ſtolice królów / którzy byli z nim w Bábilonie.
Odmienił też y odzienie / w którym był w więzieniu / y jádáł chleb záwƺe przed oblicżem jego po wƺyſtkie dni żywotá ſwego.
Obrok też jemu náznácżony / obrok uſtáwicżny dáwáno mu od królá Bábilońſkiego ná káżdy dzień áż do śmierći jego / po wƺyſtkie dni żywotá jego.
