Słowo / które śię ſtáło do Jeremijáƺá od Páná / ( gdy Nábuchodonozor / król Bábilońſki / y wƺyſtko wojſko jego / y wƺyſtkie króleſtwá źiemi / które były pod włádzą ręki jego / y wƺyſtkie národy wálcżyły przećiwko Jeruzálemowi y przećiwko wƺyſtkim miáſtom jego / ) mówiąc :
Ták mówi Pán / Bóg Izráelſki : Idź / á mów Sedekijáƺowi / królowi Judzkiemu / y powiedz mu : Ták mówi Pán : Oto Já to miáſto podám w rękę królá Bábilońſkiego / áby je ogniem ſpálił ;
Y ty nie ujdźieƺ ręki jego / ále zápewne będźieƺ pojmány / y w ręce jego podány / á ocży twoje oglądáją ocży królá Bábilońſkiego / y uſtá jego z uſty twemi mówić będą / á do Bábilonu wnijdźieƺ.
A wƺákże ſłucháj ſłowá Páńſkiego / Sedekijáƺu / królu Judzki! Ták mówi Pán o tobie : Nie umrzeƺ od miecżá :
W pokoju umrzeƺ ; á jáko wonne rzecży pálono ojcom twoim / królom przeƺłym / którzy byli przed tobą / ták pálić będą y tobie / á płákáć ćię będą / mówiąc : Ach / pánie! Bom Já to ſłowo rzekł / mówi Pán.
Tedy mówił Jeremijáƺ prorok do Sedekijáƺá / królá Judzkiego / wƺyſtkie te ſłowá w Jeruzálemie.
Gdy wojſko królá Bábilońſkiego wálcżyło przećiwko Jeruzálemowi / y przećiwko wƺyſtkim miáſtom Judzkim pozoſtáłym / przećiwko Láchyſ / y przećiw Aſekowi ; ábowiem te były pozoſtáły z miáſt Judzkich miáſtá obronne.
Słowo / które śię ſtáło do Jeremijáƺá od Páná / gdy ucżynił król Sedekijáƺ przymierze ze wƺyſtkim ludem / co był w Jeruzálemie / wolność im ogłáƺájąc ;
To jeſt / áby káżdy wolno puśćił ſługę ſwego / y káżdy ſłużebnicę ſwoję / Żydá y Żydówkę / áby ſobie nikt nie zniewáláł Żydá / brátá ſwego.
A ták uſłucháły wƺyſcy kśiążętá / y wƺyſtek lud / którzy byli weƺli w przymierze / żeby káżdy wolno puśćił ſługę ſwego / y káżdy ſłużebnicę ſwoję / áby ich więcej nie zniewáláli : uſłucháli / mówię / y puśćili ich wolno.
Lecż potem rozmyśliwƺy śię / pobráli záś ſługi y ſłużebnice / które byli puśćili wolno / á zniewolili ich ſobie zá ſługi y zá ſłużebnice.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá od Páná / mówiąc :
Ták mówi Pán / Bóg Izráelſki : Jám poſtánowił przymierze z ojcámi wáƺymi w dźień / któregom ich wywiódł z źiemi Egipſkiej / z domu niewoli / mówiąc :
Gdy śię ſkońcżą śiedm lát / niech wolno puśći káżdy brátá ſwego Żydá / któryćby był ſprzedány / á ſłużyłćiby przez ƺeść lát ; wolno / mówię / puśći go od śiebie. Ale mię nie ſłucháli ojcowie wáśi / áni nákłonili uchá ſwego.
Wyśćieć śię záiſte dźiś náwróćili / y ucżyniliśćie to / co jeſt dobrego przed ocżymá mojemi / żeśćie ogłośili wolność káżdy bliźniemu ſwemu / y ucżyniliśćie przymierze przed twárzą moją w domu tym / który jeſt názwány od imieniá mego.
Aleśćie śię záś cofnęli / y ſplugáwiliśćie imię moje / żeśćie záś wźięli káżdy ſługę ſwego / y káżdy ſłużebnicę ſwoję / któreśćie byli wolno puśćili według żądnośći ich / y zniewoliliśćie ich / áby byli ſługámi y ſłużebnicámi wáƺymi.
Dlátegoż ták mówi Pán : Wyśćie mię nie uſłucháli / ábyśćie ogłośili wolność káżdy brátu ſwemu / y káżdy bliźniemu ſwemu ; otoż Já przećiwko wám ogłáƺám wolność / mówi Pán / miecżá / moru / y głodu / á podám wáſ ná potłukánie po wƺyſtkich króleſtwách źiemi.
Podám záiſte tych ludźi / którzy przeſtąpili przymierze moje / którzy nie dotrzymáli ſłów przymierzá tego / które ucżynili przed twárzą moją / gdy ćielcá ná dwoje rozćięli / y przeƺli między cżęśćiámi jego ;
To jeſt kśiążąt Judzkich / y kśiążąt Jeruzálemſkich / komorników y kápłánów / y wƺyſtek lud tej źiemi / którzy przeƺli między cżęśćiámi tego ćielcá ;
Podám ich / mówię / w rękę nieprzyjáćiół ich / y w rękę ƺukájących duƺy ich / y będą trupy ich żerem ptáſtwu niebieſkiemu y beſtyjom źiemſkim.
Sedekijáƺá też / królá Judzkiego / y kśiążąt jego podám w rękę nieprzyjáćiół ich / y w rękę ƺukájących duƺy ich / w rękę / mówię / wojſká królá Bábilońſkiego / które odſtąpiło od wáſ.
Oto Já rozkáżę / mówi Pán / y przywiodę ich záś ná to miáſto / áby wálcżyli przećiwko niemu / á wźiąwƺy je ſpálili je ogniem ; miáſtá też Judzkie obrócę w puſtynię / ták iż będą bez obywátelá.
