Ták mówi Pán : Zſtąp do domu królá Judzkiego / á mów tám to ſłowo /
Y rzecż : Słucháj ſłowá Páńſkiego / królu Judzki! który śiedziƺ ná ſtolicy Dawidowej / ty y ſłudzy twoi / y lud twój / którzy chodzićie brámámi temi.
Ták mówi Pán : Cżyńćie ſąd y ſpráwiedliwość / á wyzwálájćie ućiśnionego z ręki gwáłtowniká / á przychdniowi / śierotce y wdowie nie cżyńćie krzywdy / áni ich ućiſkájćie ; áni krwi niewinnej nie wylewájćie ná tem miejſcu.
Bo jeźli cżyniąc ucżynićie to ſłowo / tedy pewnie wnijdą brámámi domu tego królowie / śiedzący miáſto Dawidá ná ſtolicy jego / jeżdżący ná wozách y ná koniách / ſám król y ſłudzy jego / y lud jego.
Lecż jeżli nie poſłucháćie tych ſłów / ſám ná śię przyśięgám / mówi Pán / że ten dom puſtynią będzie.
Bo ták mówi Pán o domie królá Judzkiego : Tyś mi był jáko Gáláád y wierzch Libáńſki ; ále ćię pewnie obrócę w puſtynię / y miáſtá / w których nie mieƺkáją ;
Y zgotuję ná ćię burzyćieli / káżdego z orężem jego / którzy wyrąbią wyborne cedry twoje / y wrzucą je ná ogień.
A gdy pójdzie wiele národów mimo to miáſto / y rzecże jeden do drugiego : Dlácżegoż ták ucżynił Pán temu miáſtu wielkimu?
Tedy odpowiedzą : Przeto / iż opuśćili przymierze Páná / Bogá ſwego / á kłániáli śię bogom cudzym / y ſłużyli im.
Nie płácżćież umárłego / áni go żáłujćie / ále uſtáwicżnie płácżćie nád tym / który odchodzi ; bo śię więcej nie wróći / áby oglądáł ziemię / w której śię národził.
Bo ták mówi Pán o Sellumie / ſynu Jozyjáƺá / królá Judzkiego / który króluje miáſto Jozyjáƺá / ojcá ſwego : Gdy wyjdzie z miejſcá tego / nie wróći śię więcej /
Ale tám ná onem miejſcu / gdzie go przenioſą / umrze / á ták tej ziemi więcej nie oglądá.
Biádá temu / kto buduje dom ſwój z nieſpráwiedliwośćią / á páłáce ſwoje z krzywdą / który bliźniego ſwego dármo zniewálá / á zápłáty mu jego nie dáje!
Który mówi : Zbuduję ſobie dom wielki / y páłáce przeſtworne ; y wyćiná ſobie okná / á obijá drzewem cedrowem / y máluje cynobrem.
Izáli będzieƺ królowáł / że mieƺkáƺ między cedrámi? Ojćiec twój izáli nie jádáł y nie pijáł? kiedy cżynił ſąd y ſpráwiedliwość / tedy śię miáł dobrze ;
Gdy ſądził ſpráwę ubogiego / y nędznego / tedy śię miáł dobrze ; izáli to nie jeſt poznáć mię? mówi Pán.
Ale ocży twoje y ſerce twoje nie ƺuká jedno łákomſtwá ſwego / y ábyś krew niewinną wylewáł / á gwáłt y krzywdę cżynił.
Przetoż ták mówi Pán o Joákimie / ſynu Jozyjáƺá / królá Judzkiego : Nie bądą go płákáć áni mówić : Ach bráćie mój! álbo : Ach śioſtro! Nie będą go płákáć áni mówić : Ach pánie! álbo : Ach! gdzież doſtojność jego?
Pogrzebem oślim pogrzebiony bądzie ; wywlecżony y wyrzucony będzie zá brámy Jeruzálemſkie.
Wſtąp ná Libán / á wołáj / y ná górze Báſán wydáj głoſ twój ; wołáj y u brodów / gdyż ſtárći będą wƺyſcy miłośnicy twoi.
Máwiáłem z tobą w nájwiękƺem ƺcżęśćiu twojim ; áleś ty rzekłá : Nie poſłuchám. Táć jeſt drogá twojá od dziećińſtwá twego / nie uſłucháłeś záiſte głoſu mego.
Wƺyſtkich páſterzy twoich wiátr ſpáśie / á miłośnicy twoi w niewolę pójdą ; tedy śię záiſte zápáłáć y wſtydzić będzieƺ dlá wƺelákiej złośći twojej.
O ty / którá mieƺkáƺ ná Libánie / którá ſobie gniázdo cżyniƺ ná cedrách! jáko wdzięcżná będzieƺ / gdy ćię ogárną boleśći / á ućiſk jáko rodzącą.
Jáko żyję Já / mówi Pán / iż choćby był Chonijáƺ / ſyn Joákimá / królá Judzkiego / ſygnetem ná práwej ręce mojej / wƺákże ćię y ſtámtąd zerwę ;
Y podám ćię w rękę tych / którzy ƺukáją duƺy twojej / y w rękę tych / których śię ty twárzy lękáƺ / to jeſt / w rękę Nábuchodonozorá / królá Bábilońſkiego / y w rękę Cháldejcżyków ;
A wyrzucę ćię y mátkę twoję / którá ćię urodziłá / do ziemi cudzej / w którejśćie śię nie rodzili / y tám pomrzećie.
Ale do ziemi / do której tęſknić będziećie / ábyśćie śię tám wróćili / nie wróćićie śię.
Izáli ten mąż Chonijáƺ będzie báłwánem nikcżemnym / który podruzgotány bywá? Albo nácżyniem / w którym niemáƺ żednej wdzięcżnośći? Przecżżeby odrzuceni byli on y náśienie jego / á wyrzuceni do ziemi / której nie znáją?
O ziemio / ziemio / ziemio! ſłucháj ſłowá Páńſkiego.
Ták mówi Pán : Zápiƺćie to / że ten mąż będzie bez dzieći / á że mu śię nie poƺcżęśći zá dni jego ; owƺem / nie poƺcżęśći śię y mężowi / któryby z náśieniá jego śiedziáł ná ſtolicy Dawidowej / á pánowáł jeƺcże w Judzie.
