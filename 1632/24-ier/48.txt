Przećiwko Moábowi. Ták mówi Pán záſtępów / Bóg Izráelſki : Biádá miáſtu Nebo / bo ſpuſtoƺone będzie ; poháńbione y wzięte będzie Káryjátáim : záwſtydzone będzie miáſto ná miejſcu wyſokiem / y báć śię będzie.
Nie będzie śię więcej chlubił Moáb z Heſebonu ; myślą złe przećiwko niemu / mówiąc : Pójdźćie / á wytráćmy ich z národu. Y ty / Mádmenie! wykorzeniony będzieƺ / miecż pójdzie zá tobą.
Głoſ krzyku z Choronáim : O ſpuſtoƺenie y zburzenie wielkie!
Stárty będzie Moáb / ſłyƺány będzie krzyk málucżkich jego /
Przeto / że ná drodze Luchytſkiej będzie uſtáwicżny płácż / á którędy zſtępują do Choronáim / nieprzyjáćiele krzyk zburzeniá ſłyƺeć będą /
Mówiących : Ućiekájćie / wybáwćie duƺę ſwoję / á ſtáńćie śię jáko wrzoſ ná puƺcży.
Bo dlátego / że máƺ nádzieję w doſtátku twoim / y w ſkárbách twoich / będzieƺ też wzięte / y Chámoſ pójdzie w pojmánie / kápłáni jego / tákże y kśiążętá jego.
Bo burzyćiel przyjdzie ná káżde miáſto / á żádne miáſto nie ujdzie ; zginie y doliná / y równiny ſpuſtoƺone będą / jáko mówi Pán.
Dájćie ſkrzydłá Moábowi / niech prędko uleći ; bo miáſtá jego przyjdą w ſpuſtoƺenie / ták / że nie będzie w nich obywátelá.
Przeklęty / kto zdrádliwie cżyni ſpráwę Páńſką ; przeklęty tákże / kto hámuje miecż ſwój ode krwi.
Miáłći Moáb pokój od dziećińſtwá ſwego / y uſádził śię ná drożdżách ſwoich / áni był przelewány z nácżyniá w nácżynie / to jeſt / w pojmánie nie chodził / dlá cżego zoſtáł w nim ſmák jego / á woń jego nie zmieniłá śię.
Przetoż oto dni idą / mówi Pán / że poślę náń tych / którzy wtárgnieniá cżynią / á pojmáją go / y nácżyniá jego wypróżnią / á łágwie jego potłuką.
Y záwſtydzony będzie Moáb od Chámoſá / jáko záwſtydzony jeſt dom Izráelſki od Betel / nádziei ſwojej.
Jákoż mówićie : Mocniśmy / á mężowie duży do boju?
Zburzony będzie Moáb / y z miáſt ſwoich wynijdzie / á wyborni młodzieńcy jego pójdą ná zábićie / mówi król / Pán záſtępów imię jego.
Bliſko jeſt zginienie Moábowe / y przyjdzie ; á nieƺcżęśćie jego prędko śię poſpieƺy.
Użálćie śię go wƺyſcy / którzy mieƺkáćie około niego / y wƺyſcy / którzy znáćie imię jego / mówćie : Jákoż śię złámáłá láſká mocy / y kij ozdobny?
Zſtąp z ſłáwy / á śiądź w prágnieniu / obywátelko / córko Dybońſká! bo zburzyćiel Moábu przyćiągnie przećiwko tobie / rozrzući twierdze twoje.
Stáń ná drodze á przypátrz śię pilnie / obywátelko Aroer! pytáj tego / który ućieká / y tej / którá uchodzi / mówiąc : Co śię dzieje?
Záwſtydzony jeſt Moáb / bo jeſt potárty ; nárzekájćie á wołájćie / opowiádájćie w Arnon / że puſtoƺą Moábá.
Bo ſąd przyƺedł ná ziemię tej równiny / ná Holon / y ná Jáſſá / y ná Mefáát /
Y ná Dybon / y ná Nebo / y ná Bet Dyblátáim /
Y ná Kiryjátáim / y ná Betgámul / y Betmehon.
Y ná Káryjot / y ná Bocrę / y ná wƺyſtkie miáſtá ziemi Moábſkiej / dálekie y bliſkie.
Odćięty będzie róg Moábſki / y rámię jego będzie ſtárte / mówi Pán.
Opójćie go / poniewáż śię przećiw Pánu podnióſł ; niech śię wálá Moáb w blwoćinách ſwoich / niech będzie y on ná pośmiech.
Bo ázáż w pośmiewiſku nie był u ćiebie Izráel? Izáli go między złodziejámi záſtáno? że / ilekroć mówiƺ o nim / wyſkákujeƺ.
Opuƺcżájćie miáſtá á mieƺkájćie ná ſkále / obywátele Moábſcy! á bądźćie jáko gołębicá / którá śćiele gniázdo ſwoje ná kráju dziury.
Słyƺeliśmy o pyƺe Moábowej / że jeſt bárdzo pyƺny ; o wyſokomyślnośći jego / y o hárdośći jego / y o nádętośći jego / y o wynioſłośći ſercá jego.
Známći Já / mówi Pán / zágniewánie jego ; lecż nie má śiły ; kłámſtwá jego nie dowiodą tego.
Dlátego nád Moábcżykámi nárzekám / á náde wƺyſtkim Moábem wołám / á dlá obywáteli Kirchereſ wzdychá ſerce moje.
Bárdziej niż płákáno Jázercżyków / płácżę nád tobą / o winná máćico Sábámá! Látorośli twoje doſtáną śię zá morze / áż do morzá Jázer dośięgną ; ná letnie owoce twoje / y ná zbieránie winá twego burzyćiel przypádnie.
Y uſtánie weſele y rádość nád polem urodzájnem w ziemi Moábſkiej / á winu z práſy wſtręt ucżynię ; nie będą go tłocżyć z wykrzykániem / á wykrzykánie nie będzie wykrzykániem.
Bárdziej krzycżeć będą niż Heſebońcżycy ; áż do Eleále / áż do Jázy wydádzą głoſ ſwój / od Zoár áż do Choronáim / jáko jáłowicá trzećioletniá ; bo też y wody Nimrym zniƺcżeją.
Y ucżynię / mówi Pán / że uſtánie w Moábie ofiárujący ná wyżynách / y kádzący bogom ſwoim.
Przetoż ſerce moje nád Moábem jáko piƺcżáłki piƺcżeć będzie ; tákże nád obywátelámi w Kirchereſ ſerce moje jáko piƺcżáłki piƺcżeć będzie / dlátego / że y zboże zgromádzone wniwecż śię obróći.
Bo ná káżdej głowie będzie łyśiná / y káżdá brodá ogoloná będzie ; ná wƺyſtkich rękách będą ƺrámy / á ná biodrách wór.
Po wƺyſtkich dáchách Moábſkich y po ulicách jego / wƺędy nic nie będzie / tylko nárzekánie ; bom ſkruƺył Moábá jáko nácżynie nieużytecżne / mówi Pán.
Nárzekáć będą / mówiąc : Jákoć jeſt ſtárty! Jáko tył podáł Moáb z háńbą! y jeſt Moáb pośmiewiſkiem / y poſtráchem wƺyſtkim / którzy ſą około niego.
Bo ták mówi Pán : Oto nieprzyjáćiel jáko orzeł przyleći / á rozćiągnie ſkrzydłá ſwe ná Moábá.
Káryjot wzięte będzie / y zámki wzięte będą / á ſerce mocárzów Moábſkich dniá onego będzie jáko ſerce niewiáſty bolejącej.
Y wygłádzony będzie Moáb z ludu ; bo śię przećiwko Pánu podnośił.
Strách y dół / y śidło nád tobą / o obywátelu Moábſki! mówi Pán.
Kto ućiecże przed ſtráchem / wpádnie w dół ; á kto wynijdzie z dołu / śidłem ułápiony będzie ; bo przywiodę náń / to jeſt ná Moábá / rok náwiedzieniá jego / mówi Pán.
W ćieniu Heſebon ſtáwáli ći / którzy ućiekáli przed gwáłtem ; ále ogień wynijdzie z Heſebonu / y płomień z pośrodku Sehonu / y pożre kąt Moábſki / y wierzch głowy tych / którzy go burzą.
Biádá tobie / Moábie! záginieć lud Chámoſowy ; bo ſynowie twoi zábráni będą w niewolę / y córki twoje do więzieniá.
Wƺákże záśię przywrócę więźniów Moábſkich w oſtátecżne dni / mówi Pán. Aż dotąd ſąd o Moábie.
