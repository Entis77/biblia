Spráwiedliwym zoſtánieƺ / Pánie! jeźli śię z tobą rozpieráć będę ; á wƺákże o ſądách twoich z tobą mówić będę. Cżemuż śię drogá niezbożnych ƺcżęśći? Cżemuż ſpokojnie żyją wƺyſcy / którzy bárdzo wyſtąpili przećiwko tobie?
Wƺcżepiłeś ich / y rozkorzenili śię ; roſną y owoc wydáwáją ći / którycheś uſt bliſkim / ále dálekim od nerek ich.
Ale ty / Pánie / znáƺ mię / wypátrujeƺ mię / á doświádcżyłeś ſercá mego / że z tobą jeſt ; ále onych ćiągnieƺ jáko owce ná rzeź y gotujeƺ ich ná dzień zábićiá / y mówiƺ :
Dokądżeby ziemiá płákáć / á tráwá ná wƺyſtkich polách ſchnąć miáłá? Dlá złośći mieƺkájących w niej giną wƺyſtkie zwierzętá y ptáſtwo ; bo mówią : Nie widzić Pán ſkońcżeniá náƺego.
Poniewáż ćię z pieƺymi bieżącego do uſtániá przywodzą / jákożbyś miáł zdążyć przy koniách? á poniewáż w ziemi pokoju / w którejś ufáł / uſtáwáƺ / á cóż ſpráwiƺ przy tej nádętośći Jordánu?
Bo y bráćiá twoi y dom ojcá twego przeniewierzyli śię tobie / y ći tákże wołáją zá tobą pełnemi uſty ; ále nie wierz im / choćby mówili z tobą po przyjáćielſku.
Opuśćiłem dom ſwój / odrzućiłem dziedzictwo moje ; dáłem to / co miłowáłá duƺá mojá / w ręce nieprzyjáćiół jego.
Stáło mi śię dziedzictwo moje jáko lew w leśie ; wydáje przećiwko mnie głoſ ſwój / przetoż go nienáwidzę.
Izáli ptákiem drápieżnym jeſt mi dziedzictwo moje? Izáli ptáſtwo będzie w około przećiwko niemu? Idźćież / zbierzćie śię wƺyſtkie zwierzętá polne / zejdźći śię do żeru.
Wiele páſterzy popſuje winnicę moję / podepcżą dziáł mój ; dziáł mój bárdzo miły obrócą w puſtynię ſrogą.
Obrócą go w puſtynię ; płákáć będzie / ſpuſtoƺony będąc odemnie ; tá wƺyſtká ziemiá ſpuſtoƺeje / bo niemáƺ / ktoby to ſkłádáł do ſercá.
Ná wƺyſtkie miejſcá wyſokie w puſtyniách przyjdą burzyćiele / bo miecż Páńſki pożre wƺyſtko od końcá ziemi áż do końcá ziemi ; nie będzie miáło pokoju żádne ćiáło.
Náśieją pƺenicy / ále ćiernie żąć będą ; fráſowáć śię będą / ále nic nie ſpráwią / y wſtydzić śię będą zá urodzáje ſwoje dlá gniewu popędliwośći Páńſkiej.
Ták mówi Pán o wƺyſtkich złych ſąśiádách moich / którzy śię dotykáją dziedzictwá / którem dáł w dziedzictwo ludowi memu Izráelſkiemu : Oto Já wykorzenię ich z ziemi ich / kiedy dom Judzki wyplenię z pośrodku ich.
Wƺákże gdy ich wyplenię / náwrócę śię y zmiłuję śię nád nimi / á przywiodę záśię káżdego z nich do dziedzictwá jego / y káżdego z nich do ziemi jego.
Y ſtánie śię / jeźli śię ucżąc náucżą dróg ludu mojego / á przyśięgáć będą w imieniu mojem / mówiąc : Jáko żyje Pán / jáko oni náucżáli lud mój przyśięgáć przez Báálá / tedy pobudowáni będą w pośrodku ludu mego.
Ale jeźliby nie uſłucháli / tedy wykorzenię ten náród / wyplenię y wytrácę go / mówi Pán.
