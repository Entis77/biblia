Spráwiedliwym zoſtánieƺ / Pánie! jeſli śię z tobą rozpieráć będę ; á wƺákże o ſądách twojich z tobą mówić będę. Cżemuż śię drogá niezbożnych ƺcżęśći? Cżemuż ſpokojnie żyją wƺyſcy / którzy bárzo wyſtąpili przećiwko tobie?
Wƺcżepiłeś ich / y rozkorzenili śię ; roſną y owoc wydáwáją ći / którycheś uſt bliſkim / ále dálekim od nerek ich.
Ale ty / Pánie / znáƺ mię / wypátrujeƺ mię / á doświádcżyłeś ſercá mego / że z tobą jeſt ; ále onych ćiągnieƺ jáko owce ná rzeź y gotujeƺ ich ná dźień zábićiá / y mówiƺ :
Dokądżeby źiemiá płákáć / á tráwá ná wƺyſtkich polách ſchnąć miáłá? Dla złośći mieƺkájących w niej giną wƺyſtkie zwierzętá y ptáſtwo ; bo mówią : Nie widźić Pán ſkońcżeniá náƺego.
Ponieważ ćię z pieƺymi bieżącego do uſtánia przywodzą / jákożbyś miał zdążyć przy koniách? á ponieważ w źiemi pokoju / w którejś ufał / uſtáwáƺ / á cóż ſpráwiƺ przy tey nádętośći Jordánu?
Bo y bráćia twoi y dom ojcá twego przeniewierzyli śię tobie / y ći tákże wołáją zá tobą pełnemi uſty ; ále nie wierz im / choćby mówili z tobą po przyjáćielſku.
Opuśćiłem dom ſwój / odrzućiłem dźiedźictwo moje ; dáłem to / co miłowáłá duƺá mojá / w ręce nieprzyjáćiół jego.
Stáło mi śię dźiedźictwo moje jáko lew w leśie ; wydáje przećiwko mnie głos ſwój / przetoż go nienáwidzę.
Izali ptákiem drápieżnym jeſt mi dźiedźictwo moje? Izali ptáſtwo będźie w około przećiwko niemu? Idźćież / zbierzćie śię wƺyſtkie zwierzętá polne / zejdźćie śię do żeru.
Wiele páſterzy popſuje winnicę moję / podepcżą dźiał mój ; dźiał mój bárzo miły obrócą w puſtynię ſrogą.
Obrócą go w puſtynię ; płákáć będźie / ſpuſtoƺony będąc odemnie ; tá wƺyſtká źiemiá ſpuſtoƺeje / bo niemáƺ / ktoby to ſkłádał do ſercá.
Ná wƺyſtkie miejſcá wyſokie w puſtyniách przyjdą burzyćiele / bo miecż Páńſki pożre wƺyſtko od końcá źiemi áż do końcá źiemi ; nie będźie miáło pokoju żádne ćiáło.
Náśieją pƺenicy / ále ćiernie żąć będą ; fráſowáć śię będą / ále nic nie ſpráwią / y wſtydźić śię będą zá urodzáje ſwoje dla gniewu popędliwośći Páńſkiej.
Ták mówi Pán o wƺyſtkich złych ſąśiádách mojich / którzy śię dotykáją dźiedźictwá / którem dał w dźiedźictwo ludowi memu Izráelſkiemu : Oto Ja wykorzenię ich z źiemi ich / kiedy dom Judzki wyplenię z pośrzodku ich.
Wƺákże gdy ich wyplenię / náwrócę śię y zmiłuję śię nád nimi / á przywiodę záśię káżdego z nich do dźiedźictwá jego / y káżdego z nich do źiemi jego.
Y ſtánie śię / jeſli śię ucżąc náucżą dróg ludu mojego / á przyśięgáć będą w imieniu mojim / mówiąc : Jáko żyje Pán / jáko oni náucżáli lud mój przyśięgáć przez Báálá / tedy pobudowáni będą w pośrzodku ludu mego.
Ale jeſliby nie uſłucháli / tedy wykorzenię ten naród / wyplenię y wytrácę go / mówi Pán.
