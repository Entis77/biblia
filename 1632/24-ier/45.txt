Słowo / które mówił Jeremijáƺ prorok do Báruchá / ſyná Neryjáƺowego / gdy piſáł te ſłowá w kśięgi z uſt Jeremijáƺowych roku cżwártego zá Joákimá / ſyná Jozyjáƺá / królá Judzkiego / mówiąc :
Ták mówi Pán / Bóg Izráelſki / o tobie / Báruchu!
Rzekłeś : Biádá mnie teráz! bo Pán przycżyniá żáłośći do boleśći mojej ; uprácowáłem śię w wzdychániu mojem / á odpocżynku nie znájduję.
Ták rzecżeƺ do niego : Ták mówi Pán : Oto com zbudowáł / Já rozwálám / á com wƺcżepił / Já wyrywám / y tę wƺyſtkę źiemię.
A ty ſobie ƺukáƺ rzecży wielkich? Nie ƺukáj. Bo oto Já przywiodę złe ná wƺelkie ćiáło / mówi Pán : ále tobie dám duƺę twoję w korzyśći ná wƺelkich miejſcách / dokądkolwiek pójdźieƺ.
