Potem przyſtąpili wƺyſcy kśiążętá wojſk / y Johánán / ſyn Káryjáƺowy / y Jáſánijáƺ / ſyn Hoſájáƺowy / y wƺyſtek lud od máłego áż do wielkiego ;
Y rzekli do Jeremijáƺá proroká : Niech przyjdźie proƺę prośbá náƺá przed oblicże twoje / á módl śię zá námi Pánu / Bogu twemu / zá wƺyſtek ten oſtátek ; bo náſ máło zoſtáło z wielu / jáko to widźiƺ ocżymá twemi ;
A niech nám oznájmi Pán / Bóg twój / drogę / którąbyśmy chodźić / y cobyśmy cżynić mieli.
Y rzekł do nich Jeremijáƺ prorok : Słyƺę ; oto já modlić śię będę Pánu / Bogu wáƺemu / według ſłów wáƺych / á cokolwiek wám Pán odpowie / oznájmię wám / nie zátáję nic przed wámi.
Oni záś rzekli do Jeremiáƺá : Niech będźie Pán między námi świádkiem práwdźiwym y wiernym / jeźli nie ucżynimy według káżdego ſłowá / z którem ćię pośle Pán / Bóg twój / do náſ.
Bądź dobrze bądź źle / głoſu Páná / Bogá náƺego / dlá którego ćię poſyłámy do niego / uſłuchámy / áby śię nám dobrze dźiáło / gdy będźiemy ſłucháć głoſu Páná / Bogá náƺego.
A po wyjśćiu dźieśięćiu dni / gdy śię ſtáło ſłowo Páńſkie do Jeremijáƺá /
Záwołáł Johánáná / ſyná Kárejáƺowego / y wƺyſtkich kśiążąt wojſk / którzy z nim byli / y wƺyſtkiego ludu / od máłego áż do wielkiego /
Y rzekł do nich : Ták mówi Pán / Bóg Izráelſki / do któregośćie mię poſłáli / ábym przedłożył prośbę wáƺą przed oblicżem jego :
Jeźli śię náwróćićie / y zoſtániećie w tej źiemi / záiſte pobuduję wáſ / á nie zepſuję / y wƺcżepię wáſ / á nie wykorzenię ; bo mi żál tego złego / którem wám ucżynił. Nie bójćież śię oblicżá królá Bábilońſkiego / którego śię wy boićie ;
Nie bójćie śię go / mówi Pán ; bom jeſt z wámi / ábym wáſ wybáwił y wyrwáł wáſ z ręki jego ;
Nádto zjednám wám łáſkę / áby śię zmiłowáł nád wámi / y dáł śię wám wróćić do źiemi wáƺej.
Ale rzecżećieli : Nie zoſtániemy w tej źiemi / nie ſłuchájąc głoſu Páná / Bogá wáƺego /
A mówiąc : Żádną miárą ; ále do źiemi Egipſkiej pójdźiemy / gdźie nie ujrzymy wojny / áni głoſu trąby nie uſłyƺymy / á chlebá łáknąć nie będźiemy / y tám mieƺkáć będźiemy ;
Przetoż teráz ſłuchájćie ſłowá Páńſkiego / oſtátki Judzkie! Ták mówi Pán záſtępów / Bóg Izráelſki : Jeźli wy upornie przy tem zoſtániećie / ábyśćie ƺli do źiemi Egipſkiej / á pójdźiećieli / ábyśćie tám mieƺkáli :
Tedy wáſ pewnie miecż / którego śię boićie / tám w źiemi Egipſkiej dośćignie ; y głód / którego śię obáwiáćie / tám przyjdźie ná wáſ w Egipćie / y tám pomrzećie.
Tákći śię ſtánie tym wƺyſtkim mężom / którzy śię ná to koniecżnie událi iść do Egiptu / áby tám pielgrzymowáli / że pomrą od miecżá / od głodu y od moru / á żáden z nich nie zoſtánie / áni kto ujdźie przed tem złem / które já przywiodę ná nich.
Bo ták mówi Pán záſtępów / Bóg Izráelſki : Jáko śię wyláłá popędliwość mojá y gniew mój ná obywáteli Jeruzálemſkich / ták śię wyleje zápálcżywość mojá ná wáſ / gdy wnijdźiećie do Egiptu ; y będźiećie ná przeklinánie / y ná zdumienie / y ná złorzecżenie y ná háńbę / á nie oglądáćie więcej miejſcá tego.
Do wáſći mówi Pán / o oſtátki Judzkie! Nie wchodźćie do Egiptu ; wiedzćie wiedząc / ( bo śię dźiś oświádcżám przećiwko wám. )
Poniewáżeśćie zwiedli duƺe wáƺe / poſłáwƺy mię do Páná / Bogá wáƺego / mówiąc : Módl śię zá námi Pánu / Bogu náƺemu / á według wƺyſtkiego / cokolwiek powie Pán / Bóg náƺ / ták nám oznájmij / á my ucżynimy.
A gdy wám to dźiś oznájmuję / wƺákże nie ſłucháćie głoſu Páná / Bogá wáƺego / we wƺyſtkiem / o co mię do wáſ poſłáł.
Przetoż mówię : Wiedzćie wiedząc / że miecżem / głodem y morem pomrzećie ná tem miejſcu / do którego prágniećie wnijść / ábyśćie tám pielgrzymowáli.
