Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Idź / á wołáj w uƺy Jeruzálemſkie / mówiąc : Ták mówi Pán : Wſpomniáłem ná ćię dlá miłośierdziá młodośći twojej pokázánego / y dlá miłośći ślubin twoich ; gdyś chodziłá zá mną ná puƺcży / w ziemi / w której nie ośiewáją.
Kiedy Izráel był świątobliwośćią Pánu / y pierwoćinámi urodzájów jego ; wƺyſcy / którzy go pożeráli / winni byli / złe rzecży przyƺły ná nich / mówi Pán.
Słuchájćie ſłowá Páńſkiego / domie Jákóbowy / y wƺyſtkie rodzáje domu Izráelowego!
Ták mówi Pán : Jákąż niepráwość ználeźli ojcowie wáśi przy mnie / iż śię oddálili odemnie / á chodząc zá márnośćią márnymi śię ſtáli?
Ták iż áni rzekli : Gdzież jeſt Pán / który náſ wywiódł z ziemi Egipſkiej? który náſ wodził po puƺcży / po ziemi puſtej y ſtráƺnej / po ziemi ſuchej y ćieniu śmierći / po ziemi / po której nikt nie chodził / á gdzie żáden cżłowiek nie mieƺkáł?
Owƺem / gdym wáſ wprowádził do ziemi obfitej / ábyśćie pożywáli owoców jej / y dóbr jej / wƺedłƺy tám ſplugáwiliśćie ziemię moję / á dziedzictwo moje ucżyniliśćie obrzydliwośćią.
Kápłáni nie rzekli : Gdzież jeſt Pán? áni ći / którzy śię obieráją ucżeni w zákonie / poználi mię / y páſterze odſtąpili odemnie / y prorocy prorokowáli przez Báálá / y zá rzecżámi nie użytecżnemi chodzili.
Przecżże śię wżdy wádzę z wámi / mówi Pán / á z ſynámi ſynów wáƺych śpieráć śię muƺę?
Przejdźćie przynájmniej wyſpy Cytym / á obácżćie ; y do Kedár poślijćie á uwáżájćie pilnie / y przypátrzćie śię / jeźli śię ſtáło co tákowego ;
Jeźli odmienił który náród bogów ſwoich / choćiáż oni nie ſą bogámi ; ále lud mój odmienił ſłáwę ſwoję w rzecż niepożytecżną.
Zdumiejćie śię niebioſá nád tem / á ulęknijćie śię / á zátrwożćie śię bárdzo / mówi Pán ;
Bo dwojáką złość popełnił lud mój : mnie opuśćili / źródło wód żywych / á wykopáli ſobie cyſterny / cyſterny dziuráwe / które wody zátrzymáć nie mogą.
Izáli Izráel jeſt niewolnikiem álbo wychowáńcem w domu ſpłodzonym? Cżemuż jeſt podány ná łup?
Rycżą náń lwiętá / y wydáwáją głoſ ſwój / á obrácáją ziemię jego w puſtynię ; miáſtá jego ſpálone ſą / ták / że niemáƺ y jednego obywátelá.
Synowie też Nof y Táchpáneſ wierzch głowy twojej zetrą.
Záż tego ſám ſobie nie ſpráwujeƺ? opuƺcżájąc Páná / Bogá ſwego / wtencżáſ / kiedy ćię prowádzi drogą ſwą.
A teráz co zá ſpráwę máƺ ná drogách Egipſkich / iż pijeƺ wodę z Nilu? álbo co máƺ zá ſpráwę ná drogách Aſſyryjcżyków / iż pijeƺ wodę z rzeki ich?
Skárze ćię złość twojá á odwrócenie twoje ſfuká ćię. Wiedzże tedy y obácż / iż jeſt rzecż złá y gorzká / iżeś opuśćił Páná / Bogá twego / á niemáƺ bojáźni mojej w tobie / mówi Pán / Pán záſtępów.
Gdym dáwno połámáł járzmo twoje / y rozerwáłem związki twoje / mówiłáś : Nie będę ſłużyłá báłwánom ; á przećie ná káżdym págórku wyſokim / y pod káżdem drzewem gáłęziſtem tułáƺ śię / o nierządnico!
A Jám ćię był náſádził winną máćicą wyborną / którejby wƺyſtko náśienie było práwdziwe ; jákożeś mi śię tedy odmieniłá w płonne gáłęzie obcej máćicy?
Bo choćbyś śię umywáłá y ſáletrą / y mydłem śię jáko nájbárdziej tárłá / przećież znácżná zoſtánie niepráwość twojá przedemną / mówi pánujący Pán.
Jákoż mówiƺ : Nie jeſtem ſplugáwioná / zá Báálámi nie chodziłám? Spojrzyj ná drogę twoję w tej dolinie / obácż / coś cżyniłá / o wielbłądzico prędká / którá wikleƺ drogi ſwoje?
Oślicáś dziká / przywykłáś ná puƺcży / którá według żądzy duƺy ſwej wiátr łápie / gdy śię jej przycżyná dá ; któż ją odwróći? Wƺyſcy / którzy jej ƺukáją / nie ſtrudzą śię / y w mieśiącu jej znájdą ją.
Rzekęlić : Záwśćiągnij nogi twojej / áby boſá nie byłá / y gárdło twe od prágnieniá / tedy mówiƺ : Już to próżno / nie ucżynię ; bom śię rozmiłowáłá w cudzych / y zá nimi pójdę.
Jáko wſtyd złodziejá / kiedy go záſtáną / ták śię záwſtydzi dom Izráelſki / ſámi królowie ich / kśiążętá ich / y kápłáni ich / y prorocy ich /
Którzy mówią drewnu : Tyś jeſt ojćiec mój / á kámieniowi : Tyś mię ſpłodził. Bo śię do mnie obróćili tyłem / á nie twárzą ; ále cżáſu utrápieniá ſwego máwiáją : Wſtáń á wybáw náſ.
Y gdzież ſą bogowie twoi / którycheś ſobie nácżynił? Niech wſtáną / jeźli ćię mogą wybáwić cżáſu utrápieniá twego / poniewáż ile máƺ miáſt ſwoich / tyle máƺ bogów ſwoich / o Judo!
Cżemuż śię zemną ſpieráć chcećie? Wyśćie wƺyſcy odſtąpili odemnie / mówi Pán.
Próżnom bił ſynów wáƺych / kárániá nie przyjęli ; miecż wáƺ pożárł przećie proroków wáƺych / jáko lew trácący.
O národzie! wy rozſądźćie ſłowá Páńſkie. Izálim był puſtynią Izráelowi? Izáli ziemią ćiemną? Przecżże mówi lud mój : Pánujemy / nie pójdziemy więcej do ćiebie?
Izáli zápominá pánná ubioru ſwego / y oblubienicá klejnotów ſwoich? Ale lud mój zápomniáł mię przez dni niezlicżone.
Przecż dobrą być twierdziƺ drogę twoję / ƺukájąc tego / w cżem śię kocháƺ? Przecż y innych nierządnic ucżyƺ złośliwych dróg twoich?
Nádto y ná podołkách twoich znájduje śię krew duƺ ubogich y niewinnych ; nie z prácą ználázłem to / bo to widzieć ná wƺyſtkich podołkách twoich.
A przećież mówiƺ : Poniewáżem niewinną / pewnie odwróconá jeſt zápálcżywość jego odemnie. Oto Já w ſąd wnijdę z tobą / przeto / że mówiƺ : Nie zgrzeƺyłám.
Przecżże ták biegáƺ / odmieniájąc drogi ſwe? Ták będzieƺ poháńbioná od Egipcżánów / jákoś poháńbioná byłá od Aſſyryjcżyków.
Y ſtámtąd wyjdzieƺ / májąc ręce ſwe nád głową ſwą : bo Pán odrzucá ufnośći twoje / á nie poƺcżęśćić śię w nich.
