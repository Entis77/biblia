Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Idź / á wołáj w uƺy Jeruzálemſkie / mówiąc : Ták mówi Pán : Wſpomniáłem ná ćię dla miłośierdźiá młodośći twojey pokázánego / y dla miłośći ślubin twoich ; gdyś chodźiłá zá mną ná puƺcży / w źiemi / w którey nie ośiewáją.
Kiedy Izráel był świątobliwośćią Pánu / y pierwoćinámi urodzájów jego ; wƺyſcy / którzy go pożeráli / winni byli / złe rzecży przyƺły ná nich / mówi Pán.
Słuchájćie ſłowá Páńſkiego / domie Jákóbowy / y wƺyſtkie rodzáje domu Izráelowego!
Ták mówi Pán : Jákąż niepráwość ználeſli ojcowie wáśi przy mnie / iż śię oddálili odemnie / á chodząc zá márnośćią márnymi śię ſtáli?
Ták iż áni rzekli : Gdźież jeſt Pán / który nas wywiódł z źiemi Egipſkiej? który nas wodźił po puƺcży / po źiemi puſtey y ſtráƺney / po źiemi ſuchej y ćieniu śmierći / po źiemi / po którey nikt nie chodźił / á gdźie żáden cżłowiek nie mieƺkáł?
Owƺem / gdym was wprowádźił do źiemi obfitey / ábyśćie pożywáli owoców jey / y dóbr jey / wƺedłƺy tám ſplugáwiliśćie źiemię moję / á dźiedźictwo moje ucżyniliśćie obrzydliwośćią.
Kápłáni nie rzekli : Gdźież jeſt Pán? áni ći / którzy śię obieráją ucżeni w zakonie / poználi mię / y páſterze odſtąpili odemnie / y prorocy prorokowáli przez Báálá / y zá rzecżámi nie użytecżnemi chodźili.
Przecżże śię wżdy wádzę z wámi / mówi Pán / á z ſynámi ſynów wáƺych śpieráć śię muƺę?
Przejdźćie przynájmniej wyſpy Cytym / á obácżćie ; y do Kedár poślijćie á uwáżájćie pilnie / y przypátrzćie śię / jeſli śię ſtáło co tákowego ;
Jeſli odmienił który náród bogów ſwoich / choćiáż oni nie ſą bogámi ; ále lud mój odmienił ſłáwę ſwoję w rzecż niepożytecżną.
Zdumiejćie śię niebioſá nád tem / á ulęknijćie śię / á zátrwożćie śię bárdzo / mówi Pán ;
Bo dwojáką złość popełnił lud mój : mnie opuśćili / źródło wód żywych / á wykopáli ſobie cyſterny / cyſterny dźiuráwe / które wody zátrzymáć nie mogą.
Izáli Izráel jeſt niewolnikiem álbo wychowáńcem w domu ſpłodzonym? Cżemuż jeſt podány ná łup?
Rycżą náń lwiętá / y wydáwáją głoſ ſwój / á obrácáją źiemię jego w puſtynię ; miáſtá jego ſpálone ſą / ták / że niemáƺ y jednego obywátelá.
Synowie też Nof y Táchpáneſ wierzch głowy twojey zetrą.
Záż tego ſám ſobie nie ſpráwujeƺ? opuƺcżájąc Páná / Bogá ſwego / wtencżáſ / kiedy ćię prowádźi drogą ſwą.
A teraz co zá ſpráwę máƺ ná drogách Egipſkich / iż pijeƺ wodę z Nilu? álbo co máƺ zá ſpráwę ná drogách Aſſyryjcżyków / iż pijeƺ wodę z rzeki ich?
Skárze ćię złość twojá á odwrócenie twoje ſfuká ćię. Wiedzże tedy y obácż / iż jeſt rzecż złá y gorzká / iżeś opuśćił Páná / Bogá twego / á niemáƺ bojáźni mojey w tobie / mówi Pán / Pán záſtępów.
Gdym dáwno połámáł járzmo twoje / y rozerwáłem związki twoje / mówiłáś : Nie będę ſłużyłá báłwánom ; á przećie ná káżdym págórku wyſokim / y pod káżdem drzewem gáłęźiſtem tułáƺ śię / o nierządnico!
A Jám ćię był náſádźił winną máćicą wyborną / którejby wƺyſtko náśienie było práwdźiwe ; jákożeś mi śię tedy odmieniłá w płonne gáłęźie obcey máćicy?
Bo choćbyś śię umywáłá y ſáletrą / y mydłem śię jáko nájbárdźiej tárłá / przećież znácżná zoſtánie niepráwość twojá przedemną / mówi pánujący Pán.
Jákoż mówiƺ : Nie jeſtem ſplugáwioná / zá Báálámi nie chodźiłám? Spojrzyj ná drogę twoję w tey dolinie / obácż / coś cżyniłá / o wielbłądźico prędká / która wikleƺ drogi ſwoje?
Oślicáś dźiká / przywykłáś ná puƺcży / która według żądzy duƺy ſwej wiátr łápie / gdy śię jey przycżyná dá ; któż ją odwróći? Wƺyſcy / którzy jey ƺukáją / nie ſtrudzą śię / y w mieśiącu jey znájdą ją.
Rzekęlić : Záwśćiągnij nogi twojey / áby boſá nie byłá / y gárdło twe od prágnieniá / tedy mówiƺ : Już to próżno / nie ucżynię ; bom śię rozmiłowáłá w cudzych / y zá nimi pójdę.
Jáko wſtyd złodźiejá / kiedy go záſtáną / ták śię záwſtydźi dom Izráelſki / ſámi królowie ich / kśiążętá ich / y kápłáni ich / y prorocy ich /
Którzy mówią drewnu : Tyś jeſt ojćiec mój / á kámieniowi : Tyś mię ſpłodźił. Bo śię do mnie obróćili tyłem / á nie twárzą ; ále cżáſu utrápieniá ſwego máwiáją : Wſtáń á wybáw nas.
Y gdźież ſą bogowie twoi / którycheś ſobie nácżynił? Niech wſtáną / jeſli ćię mogą wybáwić cżáſu utrápieniá twego / poniewáż ile máƺ miáſt ſwoich / tyle máƺ bogów ſwoich / o Judo!
Cżemuż śię zemną ſpieráć chcećie? Wyśćie wƺyſcy odſtąpili odemnie / mówi Pán.
Próżnom bił ſynów wáƺych / kárániá nie przyjęli ; miecż wáƺ pożárł przećie proroków wáƺych / jáko lew trácący.
O národźie! wy rozſądźćie ſłowá Páńſkie. Izálim był puſtynią Izráelowi? Izáli źiemią ćiemną? Przecżże mówi lud mój : Pánujemy / nie pójdźiemy więcey do ćiebie?
Izáli zápominá pánná ubioru ſwego / y oblubienicá klejnotów ſwoich? Ale lud mój zápomniáł mię przez dni niezlicżone.
Przecż dobrą być twierdźiƺ drogę twoję / ƺukájąc tego / w cżem śię kocháƺ? Przecż y innych nierządnic ucżyƺ złośliwych dróg twoich?
Nádto y ná podołkách twoich znájduje śię krew duƺ ubogich y niewinnych ; nie z prácą ználázłem to / bo to widźieć ná wƺyſtkich podołkách twoich.
A przećież mówiƺ : Poniewáżem niewinną / pewnie odwróconá jeſt zápálcżywość jego odemnie. Oto Ja w ſąd wnijdę z tobą / przeto / że mówiƺ : Nie zgrzeƺyłám.
Przecżże ták biegáƺ / odmieniájąc drogi ſwe? Ták będźieƺ poháńbioná od Egipcżánów / jákoś poháńbioná byłá od Aſſyryjcżyków.
Y ſtámtąd wyjdźieƺ / májąc ręce ſwe nád głową ſwą : bo Pán odrzucá ufnośći twoje / á nie poƺcżęśćić śię w nich.
