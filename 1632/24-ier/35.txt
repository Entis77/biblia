Słowo / które śię ſtáło do Jeremijáƺá od Páná zá dni Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / mówiąc :
Idź do domu Rechábitów / á mów z nimi / y wprowádź ich do domu Páńſkiego / do jednej komory / á dáj im pić wino.
Wziąłem tedy z ſobą Jáſánijáƺá / ſyná Jeremijáƺowego / ſyná Chábázymijáƺowego / y bráći jego / y wƺyſtkich ſynów jego / y wƺyſtek dom Rechábitów /
Y wprowádziłem ich do domu Páńſkiego / do komory ſynów Chánáná / ſyná Jegdálijáƺowego / mężá Bożego / którá byłá podle komory kśiążęcej / którá byłá nád komorą Mááſejáƺá / ſyná Sállumowego / ſtrzegącego progu.
Potem poſtáwiłem przed ſynámi domu Rechábitów cżáƺę pełną winá y kubki / y mówiłem do nich : Pijćie wino.
Którzy odpowiedzieli : Nie pijámy winá ; bo Jonádáb / ſyn Rechábowy / ojćiec náƺ / zákázáł nám / mówiąc : Nie pijájćie winá / wy y ſynowie wáśi áż ná wieki ;
A domu nie budujćie / y náśieniá nie rozśiewájćie / y winnicy nie ſádźćie / áni miewájćie ; ále w námiotách mieƺkájćie po wƺyſtkie dni wáƺe / ábyśćie żyli przez wiele dni ná oblicżu ziemi / w której jeſteśćie przychodniámi.
Przetoż uſłucháliśmy głoſu Jonádábá / ſyná Rechábowego / ojcá náƺego / we wƺyſtkiem / co nám rozkázáł / żebyśmy nie pili winá po wƺyſtkie dni náƺe / my / żony náƺe / ſynowie náśi / y córki náƺe ;
Y żebyśmy nie budowáli domów ku mieƺkániu náƺemu / á winnicy / y roli / y żádnego śiewu nie mieli /
Ale ábyśmy mieƺkáli w námiotách ; y uſłucháliśmy / y ucżynimy według wƺyſtkiego / co nám rozkázáł Jonádáb / ojćiec náƺ.
A gdy przyćiągnął / Nábuchodonozor król Bábilońſki / do ziemi náƺej / rzekliśmy : Pójdźćie / á uſtąpmy do Jeruzálemu przed wojſkiem Cháldejſkiem / y przed wojſkiem Syryjſkiem ; á tákeśmy zoſtáli w Jeruzálemie.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Ták mówi Pán záſtępów / Bóg Izráelſki : Idź / á rzecż mężom Judzkim y obywátelom Jeruzálemſkim : Y nie przyjmiećież ćwicżeniá / ábyśćie poſłuƺni byli ſłowom moim? mówi Pán.
Wáżne ſą ſłowá Jonádábá / ſyná Rechábowego / które przykázáł ſynom ſwoim / żeby nie pili winá ; y nie piją go áż do dniá tego / bo poſłuƺni ſą przykázániu ojcá ſwego ; ále Já mówię do wáſ / ráno wſtáwájąc y mówiąc / á przećie jeſteśćie niepoſłuƺni.
Poſyłám też do wáſ wƺyſtkich ſług ſwych / proroków ráno wſtáwájąc y poſyłájąc / áby mówili : Náwróććie śię już káżdy od złej drogi ſwej / á polepƺájćie ſpráw ſwoich / á nie náśládujćie bogów cudzych / áni im ſłużćie ; á ták mieƺkájćie w tej ziemi / którąm dáł wám / y ojcom wáƺym ; áleśćie nie nákłonili uchá ſwego / ániśćie mię uſłucháli ;
Choć ſynowie Jonádábá / ſyná Rechábowego / doſyć ucżynili rozkázániu ojcá ſwego / które im przykázáł ; ále ten lud nie jeſt mi poſłuƺny.
Przetoż ták mówi Pán / Bóg záſtępów / Bóg Izráelſki : Oto Já przywiodę ná Judę y ná wƺyſtkich obywáteli Jeruzálemſkich wƺyſtko złe / którem wyrzekł przećiwko im / przeto / żem mówił do nich / á nie ſłucháli / y wołáłem ich / á nie ozwáli mi śię.
A domowi Rechábitów rzekł Jeremijáƺ : Ták mówi Pán záſtępów / Bóg Izráelſki : Dlátego / żeśćie poſłuƺni rozkázániu Jonádábá / ojcá wáƺego / y ſtrzeżećie wƺyſtkich przykázáń jego / á cżynićie według wƺyſtkiego / co wám rozkázáł :
Przetoż ták mówi Pán záſtępów / Bóg Izráelſki : Nie będzie wygłádzony mąż z rodu Jonádábá / ſyná Rechábowego / któryby ſtáł przed oblicżem mojem po wƺyſtkie dni.
