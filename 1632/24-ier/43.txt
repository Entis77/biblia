A gdy przeſtáł Jeremijáƺ mówić do wƺyſtkiego ludu wƺyſtkich ſłów Páná / Bogá ich / z któremi go był poſłáł Pán / Bóg ich / do nich / wƺyſtkich mówię tych ſłów /
Rzekł Azáryjáƺ / ſyn Hoſájáƺowy y Johánán / ſyn Kárejáƺowy / y wƺyſcy mężowie pyƺni / mówiąc do Jeremijáƺá : Kłámſtwo ty powiádáƺ ; nie poſłáł ćię Pán / Bóg náƺ / mówiąc : Nie chodźćie do Egiptu / ábyśćie tám mieƺkáli ;
Ale Báruch / ſyn Neryjáƺowy / podƺcżuwá ćię przećiwko nám / áby náſ wydáł w ręce Cháldejcżyków / żeby náſ pobili / álbo náſ zábráli do Bábilonu.
Y nie uſłucháł Johánán / ſyn Kárejáƺowy / y wƺyſcy kśiążętá wojſk / tákże y wƺyſtek lud głoſu Páńſkiego / żeby zoſtáli w ziemi Judzkiej.
Ale Johánán / ſyn Kárejáƺowy / y wƺyſcy kśiążętá wojſk wzięli wƺyſtek oſtátek z Judy / którzy śię byli wróćili ze wƺyſtkich národów / do których byli wygnáni / áby mieƺkáli w ziemi Judzkiej :
Mężów / y niewiáſty / y dzieći / y córki królewſkie / y káżdą duƺę / którą Nábuzárdán / hetmán żołnierſki / z Godolijáƺem / ſynem Ahikámowym / ſyná Sáfánowego / zoſtáwił / y z Jeremijáƺem prorokiem / y z Báruchem / ſynem Neryjáƺowym ;
Y weƺli do ziemi Egipſkiej / bo nie byli poſłuƺni głoſowi Páńſkimu / y przyƺli do Táchpáncheſ.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá w Táchpánchoſ / mówiąc :
Nábierz w ręce ſwe kámieni wielkich / á ſkryj je w glinę w cegielnicy / którá jeſt przed brámą domu Fáráonowego w Táchpáncheſ / przed ocżymá mężów Judzkich ;
A rzecż im : Ták mówi Pán záſtępów / Bóg Izráelſki : Oto Já poślę y przywiodę Nábuchodonozorá / królá Bábilońſkiego / ſługę mego / y poſtáwię ſtolicę jego ná tych kámieniách / którem ſkrył : y rozbije májeſtát ſwój ná nich.
Bo przyćiągnąwƺy wytráći ziemię Egipſką ; którzy ná śmierć oddáni / ná śmierć pójdą / á którzy do więzieniá / do więzieniá / á którzy pod miecż / pod miecż.
Y zápálę ogień w domách bogów Egipſkich / y popáli je / á one pobierze do więzieniá : y odzieje śię ziemią Egipſką / jáko śię odziewá páſterz ƺátą ſwoją / y wynijdzie ſtámtąd w pokoju /
Gdy pokruƺy ſłupy w Betſemeſ / które jeſt w ziemi Egipſkiej / y domy bogów Egipſkich popáli ogniem.
