A gdy przeſtáł Jeremijáƺ mówić do wƺyſtkiego ludu wƺyſtkich ſłów Páná / Bogá ich / z któremi go był poſłáł Pán / Bóg ich / do nich / wƺyſtkich mówię tych ſłów /
Rzekł Azáryjáƺ / ſyn Hoſájáƺowy y Johánán / ſyn Kárejáƺowy / y wƺyſcy mężowie pyƺni / mówiąc do Jeremijáƺá : Kłámſtwo ty powiádáƺ ; nie poſłáł ćię Pán / Bóg náƺ / mówiąc : Nie chodźćie do Egiptu / ábyśćie tám mieƺkáli ;
Ale Báruch / ſyn Neryjáƺowy / podƺcżuwá ćię przećiwko nám / áby nas wydáł w ręce Cháldejcżyków / żeby nas pobili / álbo nas zábráli do Bábilonu.
Y nie uſłucháł Johánán / ſyn Kárejáƺowy / y wƺyſcy kśiążętá wojſk / tákże y wƺyſtek lud głoſu Páńſkiego / żeby zoſtáli w źiemi Judzkiej.
Ale Johánán / ſyn Kárejáƺowy / y wƺyſcy kśiążętá wojſk wźięli wƺyſtek oſtátek z Judy / którzy śię byli wróćili ze wƺyſtkich národów / do których byli wygnáni / áby mieƺkáli w źiemi Judzkiej :
Mężów / y niewiáſty / y dźieći / y córki królewſkie / y káżdą duƺę / którą Nábuzárdán / hetmán żołnierſki / z Godolijáƺem / ſynem Ahikámowym / ſyná Sáfánowego / zoſtáwił / y z Jeremijáƺem prorokiem / y z Báruchem / ſynem Neryjáƺowym ;
Y weƺli do źiemi Egipſkiej / bo nie byli poſłuƺni głoſowi Páńſkimu / y przyƺli do Táchpáncheſ.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá w Táchpánchoſ / mówiąc :
Nábierz w ręce ſwe kámieni wielkich / á ſkryj je w glinę w cegielnicy / która jeſt przed brámą domu Fáráonowego w Táchpáncheſ / przed ocżymá mężów Judzkich ;
A rzecż im : Ták mówi Pán záſtępów / Bóg Izráelſki : Oto Ja poślę y przywiodę Nábuchodonozorá / królá Bábilońſkiego / ſługę mego / y poſtáwię ſtolicę jego ná tych kámieniách / którem ſkrył : y rozbije májeſtát ſwój ná nich.
Bo przyćiągnąwƺy wytráći źiemię Egipſką ; którzy ná śmierć oddáni / ná śmierć pójdą / á którzy do więźieniá / do więźieniá / á którzy pod miecż / pod miecż.
Y zápálę ogień w domách bogów Egipſkich / y popáli je / á one pobierze do więźieniá : y odźieje śię źiemią Egipſką / jáko śię odźiewá páſterz ƺátą ſwoją / y wynijdźie ſtámtąd w pokoju /
Gdy pokruƺy ſłupy w Betſemeſ / które jeſt w źiemi Egipſkiej / y domy bogów Egipſkich popáli ogniem.
