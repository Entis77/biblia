Tedy uſłyƺáwƺy Fáſſur / ſyn Immerowy / kápłán / który był poſtánowiony przedniejƺym w domu Páńſkim / Jeremijáƺá prorokującego o tem :
Ubił Fáſſur Jeremijáƺá proroká / y dáł go do więzieniá / które było nájwyżƺe w brámie Benjáminowej / á tá byłá przy domu Páńſkim.
A názájutrz / gdy wywiódł Fáſſur Jeremijáƺá z więzieniá / rzekł do niego Jeremijáƺ : Nie názwáł ćię Pán Fáſſurem / ále Mágor Miſſábib.
Bo ták mówi Pán : Oto Já puƺcżę ná ćię ſtrách / ná ćię y ná wƺyſtkich przyjáćiół twoich / którzy upádną od miecżá nieprzyjáćiół ſwych / ná co ocży twoje pátrzyć będą ; á wƺyſtkiego Judę podám w ręce królá Bábilońſkiego / który ich záprowádzi do Bábilonu / y pozábijá ich miecżem.
Dám też wƺyſtkę májętność miáſtá tego / y wƺſtkę prácę jego / y wƺyſtkie koƺtowne rzecży jego / y wƺyſtkie ſkárby królów Judzkich dám w ręce nieprzyjáćiół ich ; y rozchwycą je / y zábiorą je / y záprowádzą je do Bábilonu.
Ale ty / Fáſſurze! y wƺyſcy / którzy mieƺkáją w domu twym / pójdziećie w pojmánie / y do Bábilonu przyjdzieƺ / y tám umrzeƺ / y tám pogrzebiony będzieƺ ; ty y wƺyſcy miłujący ćię / którymeś kłámliwie prorokowáł.
Námówiłeś mię / Pánie! á dáłem śię námówić ; mocniejƺyś był niż já / y przemogłeś ; jeſtem ná pośmiech káżdy dzień / káżdy śię ze mnie náśmiewá.
Bo jákom pocżął mówić / wołám / dlá gwáłtu y ſpuſtoƺeniá krzycżę ; bo mi ſłowo Páńſkie jeſt ku poháńbieniu y ná pośmiech káżdy dzień.
Y rzekłem : Nie będę go wſpomináł / áni będę więcej mówił w imieniu jego ; ále ſłowo Boże jeſt w ſercu mojem / jáko ogień páłájący / zámkniony w kośćiách moich / którym uśiłowáł zátrzymáć / álem nie mógł /
Choćiáż ſłyƺę urągánie od wielu y od Mágor Miſſábibá / mówiących : Powiedzćie co náń / á oznájmiemy to królowi. Wƺyſcy przyjáćiele moi cżyháją ná upádek mój / mówiąc : Azá ſnáć zwiedziony będzie / y przemożemy go / á pomśćimy śię nád nim.
Aleć Pán jeſt ze mną / jáko mocárz ſtráƺny ; przetoż ći / którzy mię prześládują / upádną / á nie przemogą ; bárdzo będą poháńbieni / że ſobie niemądrze pocżęli / háńbá ich wiecżná nie będzie zápámiętáná.
Przetoż / o Pánie záſtępów! który doświádcżáƺ ſpráwiedliwego / który wypátrujeƺ nerki y ſerce / niech widzę pomſtę twoję nád nimi ; tobiem záiſte odkrył ſpráwę moję.
Śpiewájćież Pánu / chwálćie Páná / że wybáwił duƺę ubogiego z ręki złośników.
Przeklęty dzień / w którym śię urodził ; dzień / którego mię porodziłá mátká mojá / niech nie będzie błogoſłáwiony.
Przeklęty mąż / który oznájmił ojcu memu / mówiąc : Urodziłoć śię dziećię płći męſkiej / áby go bárdzo uweſelił.
Niechże będzie on mąż jáko miáſtá / które Pán podwróćił / á nie żáłowáł tego ; niech ſłyƺy krzyk z poránku / y nárzekánie cżáſu południá.
O że mię nie zábił záráz z żywotá! Oby mi byłá mátká mojá grobem moim / á żywot jej wiecżnie brzemiennym!
Przecżżem wyƺedł z żywotá / ábym doznáł prácy y ſmutku / á żeby dni moje w háńbie ſtráwione były?
