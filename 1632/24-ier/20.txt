Tedy uſłyƺáwƺy Fáſſur / ſyn Immerowy / kápłán / który był poſtánowiony przedniejƺym w domu Páńſkim / Jeremijáƺá prorokującego o tym :
Ubił Fáſſur Jeremijáƺá proroká / y dał go do więźieniá / które było nájwyżƺe w brámie Benjáminowej / á tá byłá przy domu Páńſkim.
A názájutrz / gdy wywiódł Fáſſur Jeremijáƺá z więźieniá / rzekł do niego Jeremijáƺ : Nie názwał ćię Pán Fáſſurem / ále Mágor Miſſábib.
Bo ták mówi Pán : Oto Ja puƺcżę ná ćię ſtrách / ná ćię y ná wƺyſtkich przyjáćiół twojich / którzy upádną od miecżá nieprzyjáćiół ſwych / ná co ocży twoje pátrzyć będą ; á wƺyſtkiego Judę podám w ręce królá Bábilońſkiego / który ich záprowádźi do Bábilonu / y pozábijá ich miecżem.
Dam też wƺyſtkę májętność miáſtá tego / y wƺſtkę prácę jego / y wƺyſtkie koƺtowne rzecży jego / y wƺyſtkie ſkárby królów Judzkich dam w ręce nieprzyjáćiół ich ; y rozchwycą je / y zábiorą je / y záprowádzą je do Bábilonu.
Ale ty / Fáſſurze! y wƺyſcy / którzy mieƺkáją w domu twym / pójdźiećie w pojmánie / y do Bábilonu przyjdźieƺ / y tám umrzeƺ / y tám pogrzebiony będźieƺ ; ty y wƺyſcy miłujący ćię / którymeś kłámliwie prorokował.
Námówiłeś mię / Pánie! á dáłem śię námówić ; mocniejƺyś był niż ja / y przemogłeś ; jeſtem ná pośmiech káżdy dźień / káżdy śię ze mnie náśmiewá.
Bo jákom pocżął mówić / wołám / dla gwáłtu y ſpuſtoƺeniá krzycżę ; bo mi ſłowo Páńſkie jeſt ku poháńbieniu y ná pośmiech káżdy dźień.
Y rzekłem : Nie będę go wſpominał / áni będę więcey mówił w imieniu jego ; ále ſłowo Boże jeſt w ſercu mojim / jáko ogień páłájący / zámkniony w kośćiách mojich / którym uśiłował zátrzymáć / álem nie mógł /
Choćiáż ſłyƺę urągánie od wielu y od Mágor Miſſábibá / mówiących : Powiedzćie co nań / á oznájmiemy to królowi. Wƺyſcy przyjáćiele moi cżyháją ná upádek mój / mówiąc : Azá ſnadź zwiedźiony będźie / y przemożemy go / á pomśćimy śię nád nim.
Aleć Pán jeſt ze mną / jáko mocárz ſtráƺny ; przetoż ći / którzy mię prześládują / upádną / á nie przemogą ; bárzo będą poháńbieni / że ſobie niemądrze pocżęli / háńbá ich wiecżná nie będźie zápámiętáná.
Przetoż / o Pánie zaſtępów! który doświádcżáƺ ſpráwiedliwego / który wypátrujeƺ nerki y ſerce / niech widzę pomſtę twoję nád nimi ; tobiem záiſte odkrył ſpráwę moję.
Śpiewájćież Pánu / chwálćie Páná / że wybáwił duƺę ubogiego z ręki złośników.
Przeklęty dźień / w którym śię urodźił ; dźień / którego mię porodźiłá mátká mojá / niech nie będźie błogoſłáwiony.
Przeklęty mąż / który oznájmił ojcu memu / mówiąc : Urodźiłoć śię dźiećię płći męſkiej / áby go bárzo uweſelił.
Niechże będźie on mąż jáko miáſtá / które Pán podwróćił / á nie żáłował tego ; niech ſłyƺy krzyk z poránku / y nárzekánie cżáſu południá.
O że mię nie zábił záráz z żywotá! Oby mi byłá mátká mojá grobem moim / á żywot jey wiecżnie brzemiennym!
Przecżżem wyƺedł z żywotá / ábym doznał prácy y ſmutku / á żeby dni moje w háńbie ſtráwione były?
