Słowo / które śię ſtáło do Jeremijáƺá od Páná / mówiąc :
Ták powiádá Pán / Bóg Izráelſki / mówiąc : Nápiƺ ſobie wƺyſtkie ſłowá / którem mówił do ćiebie / ná kśięgách.
Abowiem oto dni idą / mówi Pán / á przywrócę więźniów ludu ſwego Izráelſkiego y Judzkiego / mówi Pán / y przyprowádzę ich do źiemi / którąm był dał ojcom ich / y pośiędą ją.
A teć ſą ſłowá / które mówił Pán o Izráelu y o Judźie ;
Ták záiſte mówi Pán : Słyƺeliśmy głos ſtráchu y lękánia / y że niemáƺ pokoju.
Pytájćież śię teraz / á obácżćie / jeſli rodźi mężcżyzná ; przecżże tedy widzę / że káżdy mąż rękámi ſwemi trzymá śię zá biodrá ſwe jáko rodzącá / y że śię obróćiły wƺyſtkich oblicżá w bládość?
Biádá! bo wielki jeſt ten dźień / ták / że mu nie było podobnego ; ále jákiżkolwiek jeſt cżás utrapienia Jákóbowego / przećie z niego wybáwiony będźie.
Stánie śię bowiem dniá onego / mówi Pán zaſtępów / iż ſkruƺę járzmo jego z ƺyi twojey / á związki twoje potárgám / y nie będą go więcey cudzoźiemcy w niewolę podbijáć ;
Ale ſłużyć będą Pánu / Bogu ſwemu / y Dawidowi / królowi ſwemu / którego im wzbudzę.
Przetoż nie bój śię ty / ſługo mój Jákóbie! mówi Pán / áni śię ſtrácháj o Izráelu! bo oto Ja ćię wybáwię z dáleká / y náśienie twoje z źiemi pojmánia ich. Y wróći śię Jákób / áby odpocżywał y pokój miał / á nie będźie / ktoby go uſtráƺył ;
Bom Ja z tobą / mówi Pán / ábym ćię wybáwił. Poniewáż ucżynię koniec wƺyſtkim národom / miedzy które ćię rozproƺę / wƺákże tobie nie ucżynię końcá / ále ćię miernie káráć będę / á cále ćię bez káránia nie zániechám.
Ták záiſte mówi Pán : Ciężkie bárzo będźie ſkruƺenie twoje / náder boleſná ráná twojá.
Nie będźie / ktoby ſądźił ſpráwę twoję ku ulecżeniu ; lekárſtwá ku ulecżeniu mieć nie będźieƺ.
Wƺyſcy miłośnicy twoi zápomną ćię / áni ćię náwiedzą / gdy ćię zránię ráną nieprzyjáćielſką / y okrutnem kárániem / dla wielkośći niepráwośći twojey y niezlicżonych grzechów twojich.
Przecżże wołáƺ nád ſkruƺeniem ſwem y ćiężką boleśćią ſwoją? Dla wielkośći niepráwośći twojey / y dla niezlicżonych grzechów twojich ucżyniłem ći to.
A wƺákże wƺyſcy / którzy ćię pożeráją / pożárći będą ; á wƺyſcy / którzy ćię ćiemiężą / wƺyſcy / mówię / w niewolę pójdą ; á którzy ćię plundrują / ſplundrowáni będą ; á wƺyſtkich / którzy ćię łupią / podám ná łup.
Tedyć zdrowie przywrócę / y od rán twojich ulecżę ćię / mówi Pán / przeto / że wygnáną názwáli ćię ( mówiąc : ) Táć jeſt Syon / niemáƺ nikogo / ktoby ją náwiedźił.
Ták mówi Pán : Oto Ja przywrócę pojmánych z námiotów Jákóbowych / á nád przybytkámi jego zmiłuję śię ; y będźie záś zbudowáne miáſto ná págórku ſwoim / á páłác według porządku ſwego wyſtáwiony będźie.
Y wynijdźie od nich dźiękcżynienie / y głos weſelących śię ; bo ich rozmnożę / á nie ubędźie ich / y uwielbię ich / á nie będą poniżeni.
Y będą ſynowie jego / jáko y przedtem / á zgromádzenie jego przedemną utwierdzone będźie ; ále náwiedzę wƺyſtkich / którzy ich trápią.
Y powſtánie z niego nájzácniejƺy jego / á pánujący nád nim z pośrzodku jego wynijdźie / któremu śię rozkáżę przybliżyć / áby przyſtąpił przed mię ; bo któż jeſt ten / coby ręcżył zá śię / że przyſtąpi przed mię? mówi Pán.
Y będźiećie ludem moim / á Ja będę Bogiem wáƺym.
Oto wicher Páńſki z popędliwośćią wynijdźie / wicher trwájący nád głową niezbożników zoſtánie.
Nie odwróći śię zápálcżywość gniewu Páńſkiego / póki nie ucżyni tego / á póki nie wykoná umyſłu ſercá ſwego ; w oſtátnie dni zrozumiećie to.
