Słowo / które śię ſtáło do Jeremijáƺá od Páná / mówiąc :
Ták powiádá Pán / Bóg Izráelſki / mówiąc : Nápiƺ ſobie wƺyſtkie ſłowá / którem mówił do ćiebie / ná kśięgách.
Abowiem oto dni idą / mówi Pán / á przywrócę więźniów ludu ſwego Izráelſkiego y Judzkiego / mówi Pán / y przyprowádzę ich do ziemi / którąm był dáł ojcom ich / y pośiędą ją.
A teć ſą ſłowá / które mówił Pán o Izráelu y o Judzie ;
Ták záiſte mówi Pán : Słyƺeliśmy głoſ ſtráchu y lękániá / y że niemáƺ pokoju.
Pytájćież śię teráz / á obácżćie / jeźli rodzi mężcżyzná ; przecżże tedy widzę / że káżdy mąż rękámi ſwemi trzymá śię zá biodrá ſwe jáko rodzącá / y że śię obróćiły wƺyſtkich oblicżá w bládość?
Biádá! bo wielki jeſt ten dzień / ták / że mu nie było podobnego ; ále jákiżkolwiek jeſt cżáſ utrápieniá Jákóbowego / przećie z niego wybáwiony będzie.
Stánie śię bowiem dniá onego / mówi Pán záſtępów / iż ſkruƺę járzmo jego z ƺyi twojej / á związki twoje potárgám / y nie będą go więcej cudzoziemcy w niewolę podbijáć ;
Ale ſłużyć będą Pánu / Bogu ſwemu / y Dawidowi / królowi ſwemu / którego im wzbudzę.
Przetoż nie bój śię ty / ſługo mój Jákóbie! mówi Pán / áni śię ſtrácháj o Izráelu! bo oto Já ćię wybáwię z dáleká / y náśienie twoje z ziemi pojmániá ich. Y wróći śię Jákób / áby odpocżywáł y pokój miáł / á nie będzie / ktoby go uſtráƺył ;
Bom Já z tobą / mówi Pán / ábym ćię wybáwił. Poniewáż ucżynię koniec wƺyſtkim národom / między które ćię rozproƺę / wƺákże tobie nie ucżynię końcá / ále ćię miernie káráć będę / á cále ćię bez kárániá nie zániechám.
Ták záiſte mówi Pán : Ciężkie bárdzo będzie ſkruƺenie twoje / náder boleſná ráná twojá.
Nie będzie / ktoby ſądził ſpráwę twoję ku ulecżeniu ; lekárſtwá ku ulecżeniu mieć nie będzieƺ.
Wƺyſcy miłośnicy twoi zápomną ćię / áni ćię náwiedzą / gdy ćię zránię ráną nieprzyjáćielſką / y okrutnem kárániem / dlá wielkośći niepráwośći twojej y niezlicżonych grzechów twoich.
Przecżże wołáƺ nád ſkruƺeniem ſwem y ćiężką boleśćią ſwoją? Dlá wielkośći niepráwośći twojej / y dlá niezlicżonych grzechów twoich ucżyniłem ći to.
A wƺákże wƺyſcy / którzy ćię pożeráją / pożárći będą ; á wƺyſcy / którzy ćię ćiemiężą / wƺyſcy / mówię / w niewolę pójdą ; á którzy ćię plundrują / ſplundrowáni będą ; á wƺyſtkich / którzy ćię łupią / podám ná łup.
Tedyć zdrowie przywrócę / y od rán twoich ulecżę ćię / mówi Pán / przeto / że wygnáną názwáli ćię ( mówiąc : ) Táć jeſt Syon / niemáƺ nikogo / ktoby ją náwiedził.
Ták mówi Pán : Oto Já przywrócę pojmánych z námiotów Jákóbowych / á nád przybytkámi jego zmiłuję śię ; y będzie záś zbudowáne miáſto ná págórku ſwoim / á páłác według porządku ſwego wyſtáwiony będzie.
Y wynijdzie od nich dziękcżynienie / y głoſ weſelących śię ; bo ich rozmnożę / á nie ubędzie ich / y uwielbię ich / á nie będą poniżeni.
Y będą ſynowie jego / jáko y przedtem / á zgromádzenie jego przedemną utwierdzone będzie ; ále náwiedzę wƺyſtkich / którzy ich trápią.
Y powſtánie z niego nájzácniejƺy jego / á pánujący nád nim z pośrodku jego wynijdzie / któremu śię rozkáżę przybliżyć / áby przyſtąpił przed mię ; bo któż jeſt ten / coby ręcżył zá śię / że przyſtąpi przed mię? mówi Pán.
Y będziećie ludem moim / á Já będę Bogiem wáƺym.
Oto wicher Páńſki z popędliwośćią wynijdzie / wicher trwájący nád głową niezbożników zoſtánie.
Nie odwróći śię zápálcżywość gniewu Páńſkiego / póki nie ucżyni tego / á póki nie wykoná umyſłu ſercá ſwego ; w oſtátnie dni zrozumiećie to.
