Tedy rzekł Pán do mnie : Choćby ſtánął Mojżeƺ y Sámuel przed oblicżem mojem / nie miáłbym ſercá do ludu tego ; puść ich od oblicżá mego / á niech precż idą.
A jeſliby rzekli : Dokądże pójdźiemy? Tedy im rzecżeƺ : Ták mówi Pán : Kto oddány ná śmierć / ná śmierć pójdźie ; á kto pod miecż / pod miecż ; á kto ná głód / ná głód ; á kto w niewolę / w niewolę.
Bo ich tą cżworáką rzecżą náwiedzę / mówi Pán : Miecżem ná zámordowánie / y pſámi ná rozƺárpánie / y ptáſtwem niebieſkiem / y zwierzętámi źiemſkiemi ná pożárćie y ná wygubienie.
Y podám ich ná potłukánie śię po wƺyſtkich króleſtwách źiemi dla Mánáſeſá / ſyná Ezechijáƺá / królá Judzkiego / zá to / co ucżynił w Jeruzálemie.
Bo któżby śię zmiłowáł nád tobą? Jeruzálemie! álbo ktoby śię użálił nád tobą? álbo ktoby przyƺedł / áby śię pytáł / jákoć śię powodźi?
Tyś mię opuśćiło / mówi Pán / poƺłoś názád. Przetoż wyćiągnę rękę moję ná ćię / ábym ćię wytráćił ; uſtáłem od żálu.
Przetoż ich rozwieję wiejácżką po brámách tey źiemi / ośierocę y wygubię ich ; bo śię od dróg ſwoich nie náwrácáli.
Więcey śię námnoży wdów jego / niż piáſku morſkiego ; przywiodę ná nich / ná mátki / ná młodźieńców burzyćielá y w południe ; ſpráwię / że przypádną nágle ná to miáſto ; y będą przeſtráƺeni.
Zemdleje y tá / która rodźiłá po śiedmiorgu / wypuśći duƺę ſwoję / zájdźie jey ſłońce jeƺcże zá dniá / zápłonie y wſtydźić śię będźie ; á oſtátek ich dám pod miecż przed oblicżem nieprzyjáćiół ich / mówi Pán.
Biádá mnie / mátko mojá! żeś mię urodźiłá mężá ſwáru / y mężá ſporu po wƺyſtkiej źiemi ; nie dáwáłem im ná lichwę / áni mnie oni ná lichwę dáwáli / á wżdy mi káżdy złorzecży.
Y rzekł Pán : Izáli tobie / który pozoſtánieƺ / nie będźie dobrze? Izáli śię nie záſtáwię o ćię nieprzyjáćielowi cżáſu utrápieniá y cżáſu ućiſku?
Izáli żelázo proſte pokruƺy żelázo północne y ſtál?
Májętność twoję / o Judo! y ſkárby twoje dám w rozƺárpánie dármo po wƺyſtkich gránicách twoich / á to dla wƺyſtkich grzechów twoich ;
A ſpráwię to / że pójdźieƺ z nieprzyjáćiółmi twymi do źiemi / którejś nie znáł ; ábowiem ogień rozniecony w zápálcżywośći mojey ná was páłáć będźie.
Ty mię znáƺ / Pánie! wſpomnijże ná mię / á náwiedź mię / y pomśćij śię zá mię nád tymi / co doćieráją ná mię ; odwłácżájąc zápálcżywośći twojey przećiwko nim / nie porywáj mię ; wiedz / że podejmuję dla ćiebie poháńbienie.
Gdy śię ználázły mowy twoje / zjádłem ich / á było mi ſłowo twoje weſelem y rádośćią ſercá mego / poniewáż śię názywám od imieniá twego / Pánie / Boże záſtępów!
Nie śiádám w rádźie náśmiewców / áni śię z nimi ráduję ; ále dla ſurowośći ręki twojey ſámotny śiádám ; bo zápálcżywośćią nápełniłeś mię.
Przecżże má być żál mój wiecżny? á ráná mojá śmiertelná / która śię ulecżyć nie dá? Przecżże mi ták máƺ być jáko omylny / jáko wody niepewne?
Przetoż ták mówi Pán : Jeſli śię náwróćiƺ / tedy ćię náwrócę / ábyś ſtáł przed oblicżem mojem ; á jeſli odłącżyƺ rzecż koƺtowną od nikcżemney / będźieƺ jáko uſtá moje ; oni niech śię obrócą do ćiebie / ále śię ty nie obrácáj do nich.
Bom ćię poſtáwił przećiw ludowi temu jáko mur miedźiány y obronny ; y będą wálcżyć przećiwko tobie / ále ćię nie przemogą ; bom Ja z tobą / ábym ćię wybáwiáł y wyrywáł / mówi Pán.
Wyrwę ćię záiſte z rąk ludźi złych / y odkupię ćię z rąk okrutników.
