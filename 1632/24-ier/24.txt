Ukázał mi Pán / á oto dwá koƺe fig poſtáwione były przed kośćiołem Páńſkim / gdy był w niewolę zábrał Nábuchodonozor / król Bábilońſki / Jechonijáƺá / ſyná Joákimá / królá Judzkiego / y kśiążąt Judzkich / y ćieśli / y kowáli z Jeruzálemu / á záwiódł ich do Bábilonu.
Koƺ jeden miał figi bárzo dobre / jáko bywáją figi dojrzáłe ; á koƺ drugi miał figi bárzo złe / których jeść nie można / przeto / iż były złe.
Y rzekł Pán do mnie : Cóż widźiƺ? Jeremijáƺu! Y rzekłem : Figi. Figi dobre ſą bárzo dobre / á złe ſą bárzo złe / których jeść nie mogą / przeto / iż ſą złe.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Ták mówi Pán / Bóg Izráelſki : Jáko te figi ſą dobre / ták mi przyjemni będą w niewolę záprowádzeni z Judy / którychem wyſłał z miejſcá tego do źiemi Cháldejſkiej ku dobremu.
Y obrócę oko moje ná nich ku dobremu / y przywiodę ich do tey źiemi / gdźie ich pobuduję / á nie ſkáżę / y wƺcżepię ich / á nie wykorzenię.
Abowiem dam im ſerce / áby mię poználi / żem Ja Pán ; y będą mi ludem moim / á Ja będę Bogiem ich / gdy śię náwrócą do mnie cáłem ſercem ſwojim.
A jáko figi złe / których nie jádáją / przeto / że ſą złe / ták zárzucę ( toć záiſte mówi Pán / ) Sedekijáƺá / królá Judzkiego / y kśiążąt jego / y oſtátki z Jeruzálemu / które pozoſtáły w tey źiemi / y tych / którzy mieƺkáją w źiemi Egipſkiej.
Podám ich / mówię / ná utrápienie / y ná ućiſk po wƺyſtkich króleſtwách źiemi / ná poháńbienie y ná przypowieść / ná przyſłowie y ná przeklęſtwo po wƺyſtkich miejſcách / do których ich zápędzę ;
Y będę poſyłał ná nich miecż / głód y mór / áż do końcá wytráceni będą z źiemi / którąm był dał im / y ojcom ich.
