Ukázáł mi Pán / á oto dwá koƺe fig poſtáwione były przed kośćiołem Páńſkim / gdy był w niewolę zábráł Nábuchodonozor / król Bábilońſki / Jechonijáƺá / ſyná Joákimá / królá Judzkiego / y kśiążąt Judzkich / y ćieśli / y kowáli z Jeruzálemu / á záwiódł ich do Bábilonu.
Koƺ jeden miáł figi bárdzo dobre / jáko bywáją figi dojrzáłe ; á koƺ drugi miáł figi bárdzo złe / których jeść nie możná / przeto / iż były złe.
Y rzekł Pán do mnie : Cóż widziƺ? Jeremijáƺu! Y rzekłem : Figi. Figi dobre ſą bárdzo dobre / á złe ſą bárdzo złe / których jeść nie mogą / przeto / iż ſą złe.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Ták mówi Pán / Bóg Izráelſki : Jáko te figi ſą dobre / ták mi przyjemni będą w niewolę záprowádzeni z Judy / którychem wyſłáł z miejſcá tego do ziemi Cháldejſkiej ku dobremu.
Y obrócę oko moje ná nich ku dobremu / y przywiodę ich do tej ziemi / gdzie ich pobuduję / á nie ſkáżę / y wƺcżepię ich / á nie wykorzenię.
Abowiem dám im ſerce / áby mię poználi / żem Já Pán ; y będą mi ludem moim / á Já będę Bogiem ich / gdy śię náwrócą do mnie cáłem ſercem ſwojim.
A jáko figi złe / których nie jádáją / przeto / że ſą złe / ták zárzucę ( toć záiſte mówi Pán / ) Sedekijáƺá / królá Judzkiego / y kśiążąt jego / y oſtátki z Jeruzálemu / które pozoſtáły w tej ziemi / y tych / którzy mieƺkáją w ziemi Egipſkiej.
Podám ich / mówię / ná utrápienie / y ná ućiſk po wƺyſtkich króleſtwách ziemi / ná poháńbienie y ná przypowieść / ná przyſłowie y ná przeklęſtwo po wƺyſtkich miejſcách / do których ich zápędzę ;
Y będę poſyłáł ná nich miecż / głód y mór / áż do końcá wytráceni będą z ziemi / którąm był dáł im / y ojcom ich.
