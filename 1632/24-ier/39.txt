Roku dźiewiątego Sedekijáƺá / królá Judzkiego / mieśiącá dźieśiątego / przyćiągnął Nábuchodonozor / król Bábilońſki / ze wƺyſtkim wojſkiem ſwojim do Jeruzálemu / y oblegli je.
A jedenáſtego roku Sedekijáƺá / mieśiącá cżwártego / dźiewiątego dniá tegoż mieśiącá / dobyto miáſtá.
Y wpádli do niego wƺyſcy kśiążętá królá Bábilońſkiego / y uśiedli w bramie średniej : Nergálſchárezer / Sámgárnebu / Sárſechym / Rábſárys / Nergálſchárezer / Rábmág / y wƺyſcy inni kśiążętá królá Bábilońſkiego.
A gdy ich ujrzał Sedekijáƺ / król Judzki / y że wƺyſcy mężowie wálecżni ućiekli / y wyƺli w nocy z miáſtá drogą ogrodu królewſkiego / bramą miedzy dwomá murámi / uƺedł też y król drogą ku puƺcży.
A wojſko Cháldejcżyków goniło ich / y dośćignęli Sedekijáƺá ná równinách Jerychá / y wźięli go / y przywiedli go do Nábuchodonozorá / królá Bábilońſkiego / do Reble / do źiemi Emát / gdźie wydał przećiwko niemu dekret.
Bo pomordował król Bábilońſki ſynów Sedekijáƺowych w Rebli przed ocżymá jego / y wƺyſtkich nájprzedniejƺych z Judy pomordował król Bábilońſki.
Ale ocży Sedekijáƺowi wyłupił / á związáwƺy go łáńcuchámi miedźiánemi prowádźił go do Bábilonu.
Dom tákże królewſki y dom onego ludu ſpálili Cháldejcżycy ogniem / y mury Jeruzálemſkie rozwálili.
Ale oſtátek ludu / który był zoſtał w mieśćie / y zbiegi / którzy byli poućiekáli do niego / y inny lud pozoſtáły záwiódł Nábuzárdán / hetmán żołnierſki / do Bábilonu.
Tylko nájpodlejƺych z ludu / którzy nic nie mieli / záſtáwił Nábuzárdán / hetmán żołnierſki / w źiemi Judzkiej / którym rozdał winnice y role dniá onego.
A o Jeremijáƺu przykázał Nábuchodonozor / król Bábilońſki / Nábuzárdánowi / hetmánowi żołnierſkiemu / mówiąc :
Weźmij go / á pilnie go doglądáj / á nie cżyń mu nic złego / ále jákoć rzekę / ták z nim poſtąp.
Przetoż poſłáwƺy Nábuzárdán / hetmán żołnierſki / y Nebuſáſbán / Rábſárys y Nergálſchárezer / Rábmág / y wƺyſcy hetmáni królá Bábilońſkiego :
Poſłáwƺy / mówię / wźięli Jeremijáƺá z śieni ſtráży / y porucżyli go Godolijáƺowi / ſynowi Ahikámá / ſyná Sáfánowego / áby go dowiódł do domu. A ták mieƺkał w pośród ludu.
Y ſtáło śię do Jeremijáƺá ſłowo Páńſkie / gdy jeƺcże był zámknięty w śieni ſtráży / mówiąc :
Idź / á powiedz Ebedmelechowi Murzynowi / mówiąc : Ták mówi Pán zaſtępów / Bóg Izráelſki : Oto Ja przywiodę ſłowá moje ná to miáſto ku złemu á nie ku dobremu / y wypełnią śię przed oblicżem twojim dniá onego ;
Ale ćiebie wybáwię onegoż dniá / mówi Pán / y nie będźieƺ podány w rękę mężów / których śię ty oblicża boiƺ.
Abowiem ćię pewnie wyrwę / ábyś od miecżá nie upádł ; ále będźieƺ miał duƺę twoję zá korzyść / przeto / żeś położył nádźieję we mnie / mówi Pán.
