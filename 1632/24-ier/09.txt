Kto mi to dá / áby głowá mojá wodą byłá / á ocży moje źródłem łez / ábym we dnie y w nocy płákáł pomordowánych córki ludu mego!
Któż mi dá ná puƺcży goſpodę podróżnych / ábym opuśćił lud mój / y odƺedł od nich? bo wƺyſcy ſą cudzołożnicy / zgrájá przeſtępników ;
Y náćiągáją języká ſwego do kłámſtwá jáko łuku ſwego / zmocnili śię ná źiemi / ále nie ku práwdźie ; bo ze złego w złe poſtępują / á mnie nie znáją / mówi Pán.
Káżdy niech śię ſtrzeże bliźniego ſwego / á nie káżdemu brátu dowierzá ; bo káżdy brát jeſt ná tem jákoby podƺedł / á káżdy bliźni zdrádliwie poſtępuje.
Káżdy też bliźniego ſwego oƺukuje / á práwdy nie mówi ; náucżá języká ſwego mówić kłámſtwo / źle cżyniąc uſtáwáją.
Mieƺkánie twoje / o proroku! jeſt w pośrodku ludu zdrádliwego ; dlá zdrád nie chcą mię poznáć / mówi Pán.
A przetoż ták mówi Pán záſtępów : Oto Já płáwiąc ich próbowáłem ich ; jákoż śię tedy już mám obchodźić z córką ludu mego?
Strzáłą śmiertelną jeſt język ich / zdrádę mówi ; uſty ſwemi o pokoju z przyjáćielem ſwym mówi / ále w ſercu ſwem zákłádá náń śidłá ſwoje.
Izáli dlátego nienáwidzę ich? mówi Pán ; izáli nád národem tákowym nie pomśći śię duƺá mojá?
Dlá tych gór udám śię ná płácż y ná nárzekánie / y dlá páſtwiſk / które ſą ná puƺcży / ná kwilenie ; bo ſpálone będą / ták / że nie będźie / ktoby je przechodźił / áni tám głoſu bydlęćiá ſłycháć będźie ; ptáſtwo niebieſkie y bydlętá rozbieżą śię y odejdą.
Y obrócę Jeruzálem w gromády rumu / w mieƺkánie ſmoków ; á miáſtá Judzkie obrócę w puſtynię / ták / iż nie będźie obywátelá.
Któż jeſt ták mądry / coby to wyrozumiáł? á do kogo mówiły uſtá Páńſkie / coby to oznájmił / dlácżego zginąć má tá źiemiá / y wypáloná być má jáko puſtyniá / ták áby nie było / ktoby ją przeƺedł?
Bo Pán mówi : Iż opuśćili zákon mój / którym im przedłożył / á nie ſłucháli głoſu mojego / áni chodźili zá nim ;
Ale chodźili zá uporem ſercá ſwego y zá Báálem / cżego ich náucżyli ojcowie ich.
Dlátego ták mówi Pán záſtępów / Bóg Izráelſki : Oto Já nákármię ich / to jeſt lud ten / piołunem / á nápoję ich wodą żółći.
Abowiem rozproƺę ich między národy / których nie ználi oni y ojcowie ich / y poślę zá nimi miecż / áż ich do końcá wygłádzę.
Ták mówi Pán záſtępów : Uwáżćie to / á przyzówćie nárzekájących niewiáſt / niech przyjdą / á do tych / które ſą w tem wyćwicżone / poślijćie / áby przyƺły ;
Niech śię poſpieƺą / á niech ucżynią nád námi nárzekánie / áby ocży náƺe łzy wylewáły / á powieki náƺe opływáły wodą.
Głoſ záiſte nárzekániá ſłyƺeć z Syonu : O jákośmy ſpuſtoƺeni! bárdzośmy zelżeni ; bośmy ſtráćili źiemię / bo rozrzucone ſą przybytki náƺe.
Owƺem / ſłuchájćie niewiáſty! ſłowá Páńſkiego / á niech przyjmie ucho wáƺe wyrok uſt jego / ábyśćie ucżyły córek ſwoich lámentu / á káżdá z wáſ towárzyƺkę ſwoję nárzekániá ;
Bo wlázłá śmierć oknámi náƺemi / weƺłá ná páłáce náƺe / áby wytráćiłá dźieći z rynku / á młodźieńce z ulic.
( Mów y to : Ták mówi Pán : ) Y pádły trupy ludzkie jáko gnój po polu / á jáko ſnopy zá żeńcámi / á niemáƺ ktoby pochowáł.
Ták mówi Pán : Niech śię nie chlubi mądry z mądrośći ſwojej / y niech śię nie chlubi mocárz z mocy ſwojej / y niech śię nie chlubi bogáty z bogáctw ſwoich ;
Ale w tem niecháj śię chlubi / kto śię chlubi / że rozumie á zná mię / żem Já jeſt Pán / który cżynię miłośierdźie / ſąd y ſpráwiedliwość ná źiemi ; bo mi śię to podobá / mówi Pán.
Oto dni idą / mówi Pán / w których náwiedzę káżdego obrzezáńcá y nieobrzezáńcá :
Egipcżánów / y Judę / y Edomcżyków / y Amonitcżyków / y Moábcżyków / y wƺyſtkich / którzy w oſtátnim kąćie mieƺkáją ná puƺcży ; bo te wƺyſtkie národy nieobrzezáne ſą / á wƺyſtek dom Izráelſki jeſt nieobrzezány ſercem.
