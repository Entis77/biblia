Y ſtáło śię roku cżwártego Joákimá / ſyná Jozyjáƺowego ; królá Judzkiego : ſtáło śię / mówię / to ſłowo do Jeremijáƺá od Páná / mówiąc :
Weźmij ſobie kśięgi / á nápiƺ ná nich wƺyſtkie ſłowá / którem mówił do ćiebie przećiw Izráelowi / y przećiw Judźie / y przećiw wƺyſtkim národom / ode dniá / króregom máwiał z tobą / ode dni Jozyjáƺowych áż do dniá tego ;
Azá ſnadź / gdy uſłyƺy dom Judzki o tym wƺyſtkiem złem / które Ja im myślę ucżynić / náwróći śię káżdy od złej drogi ſwej / ábym był miłośćiw niepráwośćiom ich y grzechom ich.
Przetoż wezwał Jeremijáƺ Báruchá / ſyná Neryjáƺowego ; y nápiſał Báruch w kśięgi z uſt Jeremijáƺowych wƺyſtkie ſłowá Páńſkie / które mówił do niego.
Potem przykázał Jeremijáƺ Báruchowi / mówiąc : Ja będąc zátrzymány nie mogę wnijść do domu Páńſkiego ;
Przetoż ty idź / á cżytáj ná tych kśięgách / coś nápiſał z uſt mojich / ſłowá Páńſkie / przed uƺymá ludu w domu Páńſkim w dźień poſtu ; tákże też przed uƺymá wƺyſtkich z Judy którzyby przyƺli z miaſt ſwojich / cżytáj je.
Owá ſnadź przydźie modlitwá ich przed oblicże Páńſkie / á náwróći śię káżdy od złej drogi ſwojey ; bo wielká jeſt zápálcżywość y gniew / w którym mówił Pán przećiwko temu ludowi.
Tedy ucżynił Báruch / ſyn Neryjáƺowy / według wƺyſtkiego / co mu rozkázał Jeremijáƺ prorok / cżytájąc z kśiąg ſłowá Páńſkie w domu Páńſkim.
Y ſtáło śię roku piątego zá Joákimá / ſyná Jozyjáƺowego / królá Judzkiego / w mieśiącu dźiewiątym / że zápowiedźiáno poſt przed twárzą Páńſką wƺyſtkiemu ludowi w Jeruzálemie / y wƺyſtkiemu ludowi / który śię był zƺedł z miaſt Judzkich do Jeruzálemu.
Y cżytał Báruch z kśiąg ſłowá Jeremijáƺowe w domu Páńſkim / w pokoju Giemáryjáƺá / ſyná Sáfánowego / piſárzá / w śieni wyżƺej / w wejśćiu brámy nowej domu Páńſkiego przed uƺymá wƺyſtkiego ludu.
A gdy uſłyƺał Micheáƺ / ſyn Giemáryjáƺá / ſyná Sáfánowego / wƺyſtkie ſłowá Páńſkie z kśiąg /
Zſtąpił do domu królewſkiego / do komory piſárzowej / á oto tám wƺyſcy kśiążętá śiedźieli : Eliſámá piſárz / y Delájáƺ / ſyn Semájáƺowy / y Elnátán / ſyn Achborowy / y Giemáryjáƺ / ſyn Sáfánowy / y Sedekijáƺ / ſyn Chánánijáƺowy / y wƺyſcy kśiążętá.
Y powiedźiał im Micheáƺ wƺyſtkie ſłowá / które ſłyƺał / gdy cżytał Báruch z kśiąg przed uƺymá ludu.
Przetoż poſłáli wƺyſcy kśiążętá do Báruchá Jehudę / ſyná Nátánijáƺowego / ſyná Selemijáƺowego / ſyná Chuſowego / áby rzekł : Kśięgi / ná którycheś cżytał przed uƺymá ludu / weźmij w rękę twą / á pójdź. Wźiął tedy Báruch / ſyn Neryjáƺowy / kśięgi w rękę ſwą / y przyƺedł do nich.
Y rzekli do niego : Siądź proƺę / á cżytáj to przed uƺymá náƺemi. Y cżytał Báruch przed uƺymá ich.
A gdy uſłyƺeli wƺyſtkie one ſłowá / ulękƺy śię wejrzał jeden ná drugiego / y rzekli do Báruchá : Záiſte oznájmiemy królowi te wƺyſtkie ſłowá.
Y pytáli Báruchá / mówiąc : Powiedz nam teraz / jákoś piſał wƺyſtkie te ſłowá z uſt jego?
Y rzekł im Báruch : Z uſt ſwych mówił do mnie wƺyſtkie te ſłowá / á jam piſał ná kśięgách inkáuſtem.
Tedy rzekli kśiążętá do Báruchá : Idź / á ſkryj śię / ty y Jeremijáƺ / á niech nikt nie wie / gdźieśćie.
Potem weƺli do królá do śieni / á kśięgi dáli ſchowáć do komory Eliſámá / piſárzá / y oznájmili przed królem te wƺyſtkie ſłowá.
Poſłał tedy król Juhudę / áby wźiął one kśięgi ; który je wźiął z komory Eliſámá piſárzá. Y cżytał je Jehudá przed uƺymá królewſkiemi / y przed uƺymá wƺyſtkich kśiążąt / którzy ſtáli przed królem.
A król śiedźiał w domu / w którym w źimie bywał / mieśiącá dźiewiątego / á ná ogniſku przed nim pálił śię ogień.
A gdy przecżytał Jehudá trzy álbo cżtery kárty / porzezał je król nożykiem piſárſkim / y wrzućił je w ogień / który był ná ogniſku / áż zgorzáły wƺyſtkie kśięgi w ogniu / który był ná ogniſku ;
Ale śię nie ulękli / áni rozdárli ƺát ſwojich / król y wƺyſcy ſłudzy jego / którzy ſłyƺeli wƺyſtkie te ſłowá.
Owƺem / jeƺcże gdy Elnátán / y Delájáƺ / y Giemáryjáƺ przycżyniáli śię do królá / áby nie pálił onych kśiąg / tedy ich nie uſłuchał ;
Ale rozkázał król Jerámeelowi / ſynowi królewſkiemu / y Sárájáƺowi / ſynowi Aſryjelowemu / y Selemijáƺowi / ſynowi Abdeelowemu / áby pojmáli Báruchá piſárzá / y Jeremijáƺá proroká ; ále ich Pán ſkrył.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá / gdy król ſpálił one kśięgi y ſłowá / które był ſpiſał Báruch z uſt Jeremijáƺowych / mówiąc :
Weźmij ſobie záśię kśięgi inne / á nápiƺ ná nich wƺyſtkie ſłowá pierwƺe / które były w onych kśięgách pierwƺych / które ſpálił Joákim / król Judzki.
A o Joákimie / królu Judzkim / rzecżeƺ : Ták mówi Pán : Tyś ſpálił te kśięgi / mówiąc : Cżemuś piſał ná nich / rzekƺy : Zápewne przyćiągnie król Bábilońſki / y ſpuſtoƺy tę źiemię / y wygłádźi z niej cżłowieká y bydlę.
Przetoż ták mówi Pán o Joákimie / królu Judzkim : Nie będźie miał / ktoby śiedźiał ná ſtolicy Dawidowej / á trup jego wyrzucony będźie ná gorącość we dnie / á ná mróz w nocy.
Bo náwiedzę ná nim / y ná náśieniu jego / y ná ſługách jego niepráwość ich / y przywiodę ná nich y ná obywáteli Jeruzálemſkich / y ná mężów Judzkich to wƺyſtko złe / o któremem máwiał do nich ; ále nie ſłucháli.
Tedy Jeremijáƺ wźiął kśięgi inne / y dał je Báruchowi / ſynowi Neryjáƺowemu / piſárzowi / który ná nich ſpiſał z uſt Jeremijáƺowych wƺyſtkie ſłowá onych kśiąg / które był ſpálił ogniem Joákim / król Judzki ; á nádto przydáno do onych ſłów wiele rzecży tym podobnych.
