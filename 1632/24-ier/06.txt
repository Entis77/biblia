Zgromádźćie śię / ſynowie Benjáminowi! z pośrodku Jeruzálemu / á w Tekue trąbćie w trąbę / y nád Betcherem podnieśćie chorągiew! bo złe ukázáło śię z północy / y zburzenie wielkie.
Pięknej / rozkoƺnej pánnie przypodobáłem był córkę Syońſką ;
Ale do niej przyćiągną páſterze y trzody ich ; rozbiją przećiwko niej námioty w około / ſpáśie káżdy miejſce ſwoje / y rzeką :
Podnieśćie przećiwko niej wojnę / wſtáńćie / á wtárgniemy w południe ; biádá nám / że śię náchylił dźień / że śię rozćiągnęły ćienie wiecżorne!
Wſtáńćie / á wtárgniemy w nocy / y rozwálmy páłáce jej.
Bo ták mówi Pán záſtępów : Nárąbćie drzewá / á uſypćie przećiw Jeruzálemowi ƺáńce ; toć to miáſto jeſt / które má być náwiedźione ; jákożkolwiek wielkie / niemáƺ jedno ućiſk w pośrodku jego.
Jáko źródło wylewá wody ſwe / ták ono wylewá złość ſwoję ; ućiſk y ſpuſtoƺenie ſłycháć w nim przed oblicżem mojem uſtáwicżnie / boleść y bićie.
Ćwicż śię Jeruzálemie! by ſnáć nie odſtąpiłá duƺá mojá od ćiebie / bym ćię ſnáć nie obróćił w puſtynię źiemi do mieƺkániá nieſpoſobną /
Ták mówi Pán záſtępów : Oſtátek Izráelá áż do groná wyzbieráją / jáko winnicę / y rzeką : Sięgáj ręką twoją / jáko ten / co zbierá wino do koƺá.
Do kogoż mówić będę / y kim oświádcżę / áby ſłyƺeli? Oto nieobrzezáne ſą uƺy ich / ták / że ſłucháć nie mogą ; oto ſłowo Páńſkie máją zá háńbę y nie kocháją śię w nim.
Przetoż pełenem zápálcżywośći Páńſkiej / uprácowáłem śię / záwśćiągájąc ją w ſobie. Wyláná będźie ták ná málucżkiego ná ulicy / jáko y ná zebránie młodźieńców ; owƺem / y mąż z żoną / á ſtárzec ze zgrzybiáłym pojmány będźie.
Y przypádną domy ich ná inƺych / tákże polá y żony ich / gdyż wyćiągnę rękę moję ná obywáteli tej źiemi / mówi Pán.
Záiſte / od nájmniejƺego z nich áż do nájwiękƺego z nich / wƺyſcy śię událi zá łákomſtwem ; od proroká áż do kápłáná / wƺyſcy zgołá báwią śię kłámſtwem.
Y lecżą ſkruƺenie córki ludu mego tylko po wierzchu / mówiąc : Pokój / pokój! choć niemáƺ pokoju.
Izáli śię záwſtydźili / przeto że obrzydłość cżynili? Záiſte áni śię lud wſtydźił / áni ich prorocy do wſtydu przywieść mogli ; przetoż upádną między pádájącymi ; cżáſu / którego ich náwiedzę / upádną / mówi Pán.
Gdy ták Pán máwiáł : Záſtánówćie śię ná drogách / á ſpojrzyjćie y pytájćie śię o śćieƺkách ſtárych / któráby byłá drogá dobrá / á chodźćie nią / á znájdźiećie odpocżnienie duƺy wáƺej : tedy odpowiádáli : Nie będźiemy chodźili.
A gdym poſtánáwiáł nád wámi ſtróżów / mówiąc : Słuchájćie głoſu trąby! tedy máwiáli : Nie będźiemy ſłucháć.
Przetoż ſłuchájćie / o národy! á poznáj / o zgromádzenie! co śię dźieje między nimi.
Słucháj / o źiemio! Oto Já przywiodę złe ná ten lud / owoce myśli ich / przeto / że nie ſłucháją ſłów moich / áni zákonu mego / ále go odrzucáją.
Ná cóż mi kádźidło z Sáby przychodźi / á cynámon wonny wyborny z źiemi dálekiej? Cáłopáleniá wáƺe nie ſą mi przyjemne / y ofiáry wáƺe nie podobáją mi śię.
Przetoż ták mówi Pán : Oto já nákłádę ludowi temu záwád / o które śię otrącáć będą ojcowie / tákże y ſynowie / ſąśiád y bliźni jego / y poginą.
Ták mówi Pán : Oto lud przyćiągnie z źiemi północnej / á náród wielki powſtánie od końcżyn źiemi ;
Łuk y włócżnię pochwyći / okrutny będźie / á nie zlituje śię. Głoſ ich jáko morze záhucży / á ná koniách jeźdźić będą / náród uƺykowány jáko mąż do boju przećiwko tobie / o córko Syońſká!
Skoro uſłyƺymy wieść o nim / oſłábieją ręce náƺe / ućiſk náſ ogárnie / y boleść jáko rodzącą.
Nie wychodźćie ná pole / y w drogę nie chodźćie ; bo miecż nieprzyjáćielſki á ſtrách w około.
O córko ludu mojego! przepáƺ śię worem á wáláj śię w popiele ; ucżyń ſobie żál jáko po jedynáku / żál gorzki ; bo ná náſ nágle burzyćiel przypádnie.
Dáłem ćię zá báƺtę y zá wieżę w ludu moim / ábyś upátrywáł y doświádcżáł drogi ich.
Wƺyſcy ſą między krnąbrnymi nájkrnąbrniejśi / chodzą jáko obmowcá / ſą jáko miedź y żelázá ; wƺyſcy zgołá ſą ſkáżonymi.
Murzƺeją miechy / ołów od ogniá niƺcżeje / próżno uſtáwicżnie złotnik płáwi ; bo złe rzecży nie mogą być oddálone.
Srebrem fáłƺywem będą názwáni ; bo ich Pán odrzućił.
