Zgromádźćie śię / ſynowie Benjáminowi! z pośrzodku Jeruzálemu / á w Tekue trąbćie w trąbę / y nád Betcherem podnieśćie chorągiew! bo złe ukázáło śię z północy / y zburzenie wielkie.
Piękney / rozkoƺney pánnie przypodobáłem był córkę Syońſką ;
Ale do niej przyćiągną páſterze y trzody ich ; rozbiją przećiwko niej námioty w około / ſpáśie káżdy miejſce ſwoje / y rzeką :
Podnieśćie przećiwko niej wojnę / wſtáńćie / á wtárgniemy w południe ; biádá nam / że śię náchylił dźień / że śię rozćiągnęły ćienie wiecżorne!
Wſtáńćie / á wtárgniemy w nocy / y rozwálmy páłáce jey.
Bo ták mówi Pán zaſtępów : Nárąbćie drzewá / á uſypćie przećiw Jeruzálemowi ƺáńce ; toć to miáſto jeſt / które ma być náwiedźione ; jákożkolwiek wielkie / niemáƺ jedno ućiſk w pośrzodku jego.
Jáko źródło wylewá wody ſwe / ták ono wylewá złość ſwoję ; ućiſk y ſpuſtoƺenie ſłycháć w nim przed oblicżem mojim uſtáwicżnie / boleść y bićie.
Ćwicż śię Jeruzálemie! by ſnadź nie odſtąpiłá duƺá mojá od ćiebie / bym ćię ſnadź nie obróćił w puſtynię źiemi do mieƺkánia nieſpoſobną.
Ták mówi Pán zaſtępów : Oſtátek Izráelá áż do groná wyzbieráją / jáko winnicę / y rzeką : Sięgáj ręką twoją / jáko ten / co zbierá wino do koƺá.
Do kogoż mówić będę / y kim oświádcżę / áby ſłyƺeli? Oto nieobrzezáne ſą uƺy ich / ták / że ſłucháć nie mogą ; oto ſłowo Páńſkie máją zá háńbę y nie kocháją śię w nim.
Przetoż pełenem zápálcżywośći Páńſkiej / uprácowáłem śię / záwśćiągájąc ją w ſobie. Wyláná będźie ták ná málucżkiego ná ulicy / jáko y ná zebránie młodźieńców ; owƺem / y mąż z żoną / á ſtárzec ze zgrzybiáłym pojmány będźie.
Y przypádną domy ich ná inƺych / tákże polá y żony ich / gdyż wyćiągnę rękę moję ná obywáteli tey źiemi / mówi Pán.
Zájiſte / od nájmniejƺego z nich áż do nájwiękƺego z nich / wƺyſcy śię událi zá łákomſtwem ; od proroká áż do kápłáná / wƺyſcy zgołá báwią śię kłámſtwem.
Y lecżą ſkruƺenie córki ludu mego tylko po wierzchu / mówiąc : Pokój / pokój! choć niemáƺ pokoju.
Izali śię záwſtydźili / przeto że obrzydłość cżynili? Zájiſte áni śię lud wſtydźił / áni ich prorocy do wſtydu przywieść mogli ; przetoż upádną miedzy pádájącymi ; cżáſu / którego ich náwiedzę / upádną / mówi Pán.
Gdy ták Pán máwiał : Záſtánówćie śię ná drogách / á ſpojrzyjćie y pytájćie śię o śćieƺkách ſtárych / któráby byłá drogá dobrá / á chodźćie nią / á znájdźiećie odpocżnienie duƺy wáƺej : tedy odpowiádáli : Nie będźiemy chodźili.
A gdym poſtánáwiał nád wámi ſtróżów / mówiąc : Słuchájćie głoſu trąby! tedy máwiáli : Nie będźiemy ſłucháć.
Przetoż ſłuchájćie / o narody! á poznáj / o zgromádzenie! co śię dźieje miedzy nimi.
Słucháj / o źiemio! Oto Ja przywiodę złe ná ten lud / owoce myśli ich / przeto / że nie ſłucháją ſłów mojich / áni zakonu mego / ále go odrzucáją.
Ná cóż mi kádźidło z Sáby przychodźi / á cynámon wonny wyborny z źiemi dálekiej? Cáłopalenia wáƺe nie ſą mi przyjemne / y ofiáry wáƺe nie podobáją mi śię.
Przetoż ták mówi Pán : Oto ja nákłádę ludowi temu záwád / o które śię otrącáć będą ojcowie / tákże y ſynowie / ſąśiád y bliźni jego / y poginą.
Ták mówi Pán : Oto lud przyćiągnie z źiemi północney / á naród wielki powſtánie od końcżyn źiemi ;
Łuk y włócżnię pochwyći / okrutny będźie / á nie zlituje śię. Głos ich jáko morze záhucży / á ná koniách jeźdźić będą / naród uƺykowány jáko mąż do boju przećiwko tobie / o córko Syońſká!
Skoro uſłyƺymy wieść o nim / oſłábieją ręce náƺe / ućiſk nas ogárnie / y boleść jáko rodzącą.
Nie wychodźćie ná pole / y w drogę nie chodźćie ; bo miecż nieprzyjáćielſki á ſtrách w około.
O córko ludu mojego! przepáƺ śię worem á wáláj śię w popiele ; ucżyń ſobie żál jáko po jedynáku / żál gorzki ; bo ná nas nágle burzyćiel przypádnie.
Dáłem ćię zá báƺtę y zá wieżę w ludu moim / ábyś upátrywał y doświádcżał drogi ich.
Wƺyſcy ſą miedzy krnąbrnymi nájkrnąbrniejśi / chodzą jáko obmowcá / ſą jáko miedź y żelázo ; wƺyſcy zgołá ſą ſkáżonymi.
Murzƺeją miechy / ołów od ogniá niƺcżeje / prózno uſtáwicżnie złotnik płáwi ; bo złe rzecży nie mogą być oddálone.
Srebrem fáłƺywem będą názwáni ; bo ich Pán odrzućił.
