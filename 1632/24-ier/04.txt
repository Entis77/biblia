Jeźlibyś śię chćiáł náwróćić / Izráelu! mówi Pán / do mnie śię náwróć. Bo jeźli odejmieƺ obrzydłośći twoje od oblicżá mego / á nie będzieƺ śię tułáł /
Y przyśiężeƺli w práwdzie / w ſądzie y w ſpráwiedliwośći / mówiąc : Jáko żyje Pán : tedy błogoſłáwić ſobie w nim będą národy / y w nim śię przechwáláć.
Abowiem ták mówi Pán mężom Judzkim y Jeruzálemſkim : Poorzćie ſobie nowinę / á nie śiejćie ná ćierniu ;
Obrzeżćie śię Pánu / á odejmijćie nieobrzezki ſercá wáƺego / mężowie Judzcy / á obywátele Jeruzálemſcy! by ſnáć nie wyƺłá jáko ogień popędliwość mojá / á nie zápáliłá śię / á nie byłby / ktoby ugáśił dlá złośći przedśięwzięćiá wáƺego.
Opowiádájćie w Judzie / á w Jeruzálemie ogłáƺájćie / y mówćie : Zátrąbćie w trąbę w ziemi / zwołájćie / zgromádźćie lud / á mówćie : Zbierzćie śię / á wejdźmy do miáſt obronnych.
Podnieśćie chorągiew ná Syonie / bądźćie ſercá dobrego / nie záſtánáwiájćie śię ; bo Já złe przywiodę od północy / y poráżkę wielką.
Wychodzi lew z jáſkini ſwojej / á ten / który niƺcży národy / wyƺedłƺy z miejſcá ſwego / ćiągnie / áby obróćił ziemię twoję w puſtynię / á miáſtá twoje áby zburzone były / áby nie było żádnego obywátelá.
Przetoż przepáśćie śię worámi / nárzekájćie á kwilćie / bo nie jeſt odwrócony gniew zápálcżywośći Páńſkiej od náſ.
Y ſtánie śię dniá onego / mówi Pán / że zginie ſerce królewſkie / y ſerce kśiążąt / á zdumieją śię kápłáni / y prorocy dziwowáć śię będą.
Y rzekłem : Ach pánujący Pánie! zápráwdęś bárdzo ten lud y Jeruzálem omylił / mówiąc : Pokój mieć będziećie! á wżdy miecż przeniknął áż do duƺy.
Cżáſu onego rzeką temu ludowi y Jeruzálemowi : Wiátr gwáłtowny z miejſc wyſokich ná puƺcży idzie proſto ná lud mój / nie żeby przewiewáł / áni wycżyƺcżáł.
Wiátr gwáłtowniejƺy niż oni przyjdzie mi ; teráz Já też opowiem im ſądy.
Oto wyſtępuje jáko obłoki / á wozy jego jáko wicher / prędƺe ſą niż orły konie jego. Biádá nám! bośmy ſpuſtoƺeni.
Omyj od złośći ſerce twoje / Jeruzálem! ábyś wybáwione było. Dokądże trwáć będą w pośrodku ćiebie myśli niepráwośći twojej.
Bo głoſ opowiádájącego idzie od Dánu / á tego / który ogłáƺá niepráwość / z góry Efráim.
Przypominájćież tym národom : Oto ogłáƺájćie Jeruzálemcżykom / że ſtróżowie przychodzą z ziemi dálekiej / á wydáwáją przećiwko miáſtom Judzkim głoſ ſwój.
Jáko ſtróżowie pól położą śię przećiwko niemu w około ; bo mię do gniewu wzruƺyło / mówi Pán.
Drogá twojá y poſtępki twoje to ucżyniły tobie ; toć złość twojá przynioſłá / że to jeſt gorzkie / á że przeniká áż do ſercá twego.
O wnętrznośći moje / wnętrznośći moje! boleść ćierpię. O ośierdziá moje! trwoży śię we mnie ſerce moje / nie zámilcżę ; bo głoſ trąby ſłyƺyƺ / duƺo mojá! y okrzyk wojenny.
Poráżká zá poráżką ogłáƺá śię / ſpuſtoƺoná będzie záiſte wƺyſtká ziemiá ; nágle ſpuſtoƺone będą námioty moje / y opony moje w okámgnieniu.
Dokądże widzieć będę chorągiew / y ſłyƺeć głoſ trąby?
Bo głupi lud mój nie zná mię / ſynowie niemądrzy y nierozumni ſą ; mądrzy ſą do cżynieniá złego / ále dobrze cżynić nie umieją.
Spojrzęli ná ziemię / á oto jeſt niepozorná y próżná ; jeżeli ná niebo / niemáƺ ná nim świátłośći.
Spojrzęli ná góry / á oto śię trzęſą / y wƺyſtkie págórki chwieją śię.
Spojrzęli / á oto niemáƺ cżłowieká / y wƺelkie ptáſtwo niebieſkie odlećiáło.
Spojrzęli / á oto pole urodzájne jeſt puſtynią / á wƺyſtkie miáſtá jego zburzone ſą od oblicżá Páńſkiego / y od oblicżá gniewu zápálcżywośći jego.
Bo ták mówi Pán : Spuſtoƺoná będzie wƺyſtká ziemiá / wƺákże końcá jeƺcże nie ucżynię.
Nád tem ziemiá kwilić będzie / á niebioſá w górze záćmią śię / przeto / żem mówił / com umyślił / á nie żáłuję / áni śię odwrócę od tego.
Przed grzmotem jeźdźców y ſtrzelájących z łuku ućiecże wƺyſtko miáſto ; wejdą do gęſtych obłoków / y ná ſkáły wſtąpią. Wƺyſtkie miáſtá opuƺcżone będą / á nie będzie / ktoby w nich mieƺkáł.
A ty zburzoná będąc cóż ucżyniƺ? Choćbyś śię ubráłá w ƺárłát / choćbyś śię ozdobiłá ozdobą złotą / choćbyś też oblicże twe przypráwiłá bárwicżką / próżno śię ſtroiƺ ; wzgárdzą tobą zálotnicy twoi / á duƺy twojej ƺukáć będą.
Bom ſłyƺáł głoſ jáko rodzącej / ućiſki jáko pierworodzącej / głoſ córki Syońſkiej nárzekájącej / á záłámującej ręce ſwe / mówiąc : Biádá mnie teráz! bo uſtáłá duƺá mojá dlá morderców.
