Słowo / które śię ſtáło do Jeremijáƺá o wƺyſtkich Żydách / którzy mieƺkáli w źiemi Egipſkiej / którzy mieƺkáli w Migdolu / y w Táchpáncheſ / y w Nof / y w źiemi Pátroſ / mówiąc :
Ták mówi Pán záſtępów / Bóg Izráelſki : Wyśćie widźieli wƺyſtko ono złe / którem przywiódł ná Jeruzálem / y ná wƺyſtkie miáſtá Judzkie / że oto puſte ſą po dźiś dźień / y niemáƺ w nich obywátelá.
Dlá złośći ich / którą cżynili / áby mię do gniewu pobudzáli / chodząc kádźić y ſłużyć Bogom cudzym / których nie ználi ſámi / wy y ojcowie wáśi ;
Choćiáżem poſyłáł do wáſ wƺyſtkich ſług moich / proroków / ráno wſtáwájąc y poſyłájąc á mówiąc : Nie cżyńćie proƺę tej obrzydliwośći / której nienáwidzę.
Ale nie uſłucháli áni nákłonili uchá ſwego / áby śię odwróćili od złośći ſwojej / á nie kádźili bogom cudzym.
Przetoż wylány jeſt gniew mój / á zápálcżywość mojá zápáliłá śię w miáſtách Judzkich / y w ulicách Jeruzálemſkich / y obróćiły śię w puſtynię / y w zburzenie / jáko śię to dźiś pokázuje.
Teráz tedy ták mówi Pán / Bóg záſtępów / Bóg Izráelſki : Cżemu wy cżynićie tę złość wielką przećiwko duƺom wáƺym / áby z wáſ był wykorzeniony mąż y niewiáſtá / dźiećię y ſſący z pośrodku Judy / ták / żeby z wáſ nic nie zoſtáło /
Dráźniąc mię ſpráwámi rąk wáƺych / kádząc bogom cudzym w źiemi Egipſkiej / do którejśćie weƺli / ábyśćie tám pielgrzymowáli / y iżbyśćie byli wykorzenieni / á byli ná przeklęſtwo y ná háńbę u wƺyſtkich národów ná źiemi?
Azáśćie zápámiętáli ná złość ojców wáƺych / y ná złość królów Judzkich / y ná złość żon ich / y ná złośći wáƺe / y ná złośći żon wáƺych / których śię dopuśćiły w źiemi Judzkiej y po ulicách Jeruzálemſkich?
Nie upokorzyli śię áż do dniá tego / y nie báli śię / áni chodźili w zákonie moim y w uſtáwách moich / które podáję wám y ojcom wáƺym.
Przetoż ták mówi Pán záſtępów / Bóg Izráelſki : Oto Já obrócę oblicże moje przećiwko wám ná złe / ábym wykorzenił wƺyſtkiego Judę.
Wygubię záiſte oſtátki Judzkie / którzy upornie weƺli do źiemi Egipſkiej / áby tám pielgrzymowáli / ták / że zniƺcżeją wƺyſcy w źiemi Egipſkiej / polegną od miecżá / od głodu zniƺcżeją od nájmniejƺego áż do nájwiękƺego / od miecżá y od głodu pomrą ; nádto będą ná przeklinánie / y ná wzdumienie / y ná złorzecżenie / y ná háńbę.
Bo náwiedzę tych / którzy mieƺkáją w źiemi Egipſkiej / jákom náwiedźił Jeruzálem miecżem / głodem y morem.
Y nie będźie / ktoby uƺedł y zoſtáł z oſtátków Judzkich / którzy weƺli do źiemi Egipſkiej / áby tám pielgrzymowáli / áby śię záś wróćić mieli do źiemi Judzkiej / do której śię oni prágną wróćić / y mieƺkáć tám ; ále śię nie wrócą / tylko ći / którzy ujdą.
Tedy odpowiedźieli Jeremijáƺowi wƺyſcy mężowie / którzy wiedźieli / iż kádzáły żony ich bogom cudzym / one wƺyſtkie niewiáſty / których ſtáło mnóſtwo wielkie / y wƺyſtek lud / który mieƺkáł w źiemi Egipſkiej w Pátroſ / mówiąc :
W ſłowie / któreś mówił do náſ imieniem Páńſkim / nie uſłuchámy ćię ;
Ale doſyć ucżynimy káżdemu ſłowu / które wynijdźie z uſt náƺych / kádząc królowej niebieſkiej / y ſpráwując jej ofiáry mokre / jákośmy cżynili / my y ojcowie náśi / królowie náśi / y kśiążętá náśi w miáſtách Judzkich y po ulicách Jeruzálemſkich / á nájádáliśmy śię chlebá / y dobrze nám było / á nic złegośmy nie widźieli.
Ale odtąd jákośmy przeſtáli kádźić królowej niebieſkiej / y ſpráwowáć jej ofiáry mokre / ná wƺyſtkiem nám ſchodźi / á od miecżá y od głodu niƺcżejemy.
A gdy kádźimy królowej niebieſkiej / y ſpráwujemy jej ofiáry mokre / izáli jej bez mężów náƺych plácki cżynimy / kƺtáłtując ją / y ſpráwując jej ofiáry mokre?
Tedy rzekł Jeremijáƺ do wƺyſtkiego ludu / do mężów y do niewiáſt / y do wƺyſtkiego poſpólſtwá / którzy mu ták odpowiedźieli / mówiąc :
Izáli ná kádzenie / któremeśćie kádźili w miáſtách Judzkich y w ulicách Jeruzálemſkich / wy y ojcowie wáśi / królowie wáśi / y kśiążętá wáśi / y lud źiemi / nie wſpomniáł Pán / y nie wſtąpiło to ná ſerce jego?
Ták / że nie mógł Pán dálej znośić złośći ſpráw wáƺych / y obrzydliwośći / któreśćie broili ; dlátego śię ſtáłá źiemiá wáƺá ſpuſtoƺeniem y zdumieniem / y przeklęſtwem / ták / że niemáƺ w niej obywátelá / jáko śię to dźiś pokázuje /
Dlátego / żeśćie kádźiili báłwánom / y żeśćie grzeƺyli przećiw Pánu / á nie ſłucháliśćie głoſu Páńſkiego / á ták w zákonie jego / y w uſtáwách jego / áni w świádectwách jego nie chodźiliśćie / dlátego przyƺło ná wáſ to złe / jáko śię to dźiś pokázuje.
Nádto rzekł Jeremijáƺ do wƺyſtkiego ludu / y do wƺyſtkich niewiáſt : Słuchájćie ſłowá Páńſkiego wƺyſcy ludźie Judzcy / którzyśćie z źiemi Egipſkiej.
Ták rzekł Pán záſtępów / Bóg Izráelſki / mówiąc : Wy y żony wáƺe mówiliśćie uſty ſwemi / y wypełniliśćie rękámi ſwemi / mówiąc : Ucżynimy doſyć ślubom náƺym / któreśmy poślubili / ábyśmy kádźili królowej niebieſkiej / y ſpráwowáli jej ofiáry mokre / á ták wƺyſtką śiłą wypełniáćie śluby wáƺe / y ſámym ſkutkiem wykonywáćie śluby wáƺe.
Przetoż ſłuchájćie ſłowá Páńſkiego wƺyſcy ludźie Judzcy / którzy mieƺkáćie w źiemi Egipſkiej : Oto Já przyśięgám przez imię moje wielkie / mówi Pán / że nie będźie więcej imię moje wzywáne uſty żádnego mężá Judzkiego po wƺyſtkiej źiemi Egipſkiej / któryby rzekł : Jáko żyje pánujący Pán!
Oto Já będę cżuł nád nimi ná złe / á nie ná dobre ; y niƺcżeć będą wƺyſcy mężowie Judzcy / którzy ſą w źiemi Egipſkiej / miecżem y głodem / áż do ƺcżętu wyginą ;
A którzy ujdą miecżá / wrócą śię z źiemi Egipſkiej do źiemi Judzkiej / ludu máły pocżet ; y pozná wƺyſtek oſtátek Judzki / którzy weƺli do źiemi Egipſkiej / áby tám pielgrzymowáli / cżyje śię ſłowo oſtoi / mojeli / cżyli ich?
A to miejćie zá znák / mówi Pán / że Já wáſ náwiedzę ná tem miejſcu / ábyśćie wiedźieli / iż śię práwdźiwie ſpełnią ſłowá moje nád wámi ku złemu.
Ták mówi Pán : Oto Já podám Fáráoná Chofrá / królá Egipſkiego / w rękę nieprzyjáćiół jego / y w rękę ƺukájących duƺy jego / jákom podáł Sedekijáƺá / królá Judzkiego / w rękę Nábuchodonozorá / królá Bábilońſkiego / nieprzyjáćielá jego / który ƺukáł duƺy jego.
