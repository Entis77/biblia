Słowo / które śię ſtáło do Jeremijáƺá o wƺyſtkich Żydách / którzy mieƺkáli w ziemi Egipſkiej / którzy mieƺkáli w Migdolu / y w Táchpáncheſ / y w Nof / y w ziemi Pátroſ / mówiąc :
Ták mówi Pán záſtępów / Bóg Izráelſki : Wyśćie widzieli wƺyſtko ono złe / którem przywiódł ná Jeruzálem / y ná wƺyſtkie miáſtá Judzkie / że oto puſte ſą po dziś dzień / y niemáƺ w nich obywátelá.
Dlá złośći ich / którą cżynili / áby mię do gniewu pobudzáli / chodząc kádzić y ſłużyć Bogom cudzym / których nie ználi ſámi / wy y ojcowie wáśi ;
Choćiáżem poſyłáł do wáſ wƺyſtkich ſług moich / proroków / ráno wſtáwájąc y poſyłájąc á mówiąc : Nie cżyńćie proƺę tej obrzydliwośći / której nienáwidzę.
Ale nie uſłucháli áni nákłonili uchá ſwego / áby śię odwróćili od złośći ſwojej / á nie kádzili bogom cudzym.
Przetoż wylány jeſt gniew mój / á zápálcżywość mojá zápáliłá śię w miáſtách Judzkich / y w ulicách Jeruzálemſkich / y obróćiły śię w puſtynię / y w zburzenie / jáko śię to dziś pokázuje.
Teráz tedy ták mówi Pán / Bóg záſtępów / Bóg Izráelſki : Cżemu wy cżynićie tę złość wielką przećiwko duƺom wáƺym / áby z wáſ był wykorzeniony mąż y niewiáſtá / dziećię y ſſący z pośrodku Judy / ták / żeby z wáſ nic nie zoſtáło /
Dráźniąc mię ſpráwámi rąk wáƺych / kádząc bogom cudzym w ziemi Egipſkiej / do którejśćie weƺli / ábyśćie tám pielgrzymowáli / y iżbyśćie byli wykorzenieni / á byli ná przeklęſtwo y ná háńbę u wƺyſtkich národów ná ziemi?
Azáśćie zápámiętáli ná złość ojców wáƺych / y ná złość królów Judzkich / y ná złość żon ich / y ná złośći wáƺe / y ná złośći żon wáƺych / których śię dopuśćiły w ziemi Judzkiej y po ulicách Jeruzálemſkich?
Nie upokorzyli śię áż do dniá tego / y nie báli śię / áni chodzili w zákonie moim y w uſtáwách moich / które podáję wám y ojcom wáƺym.
Przetoż ták mówi Pán záſtępów / Bóg Izráelſki : Oto Já obrócę oblicże moje przećiwko wám ná złe / ábym wykorzenił wƺyſtkiego Judę.
Wygubię záiſte oſtátki Judzkie / którzy upornie weƺli do ziemi Egipſkiej / áby tám pielgrzymowáli / ták / że zniƺcżeją wƺyſcy w ziemi Egipſkiej / polegną od miecżá / od głodu zniƺcżeją od nájmniejƺego áż do nájwiękƺego / od miecżá y od głodu pomrą ; nádto będą ná przeklinánie / y ná wzdumienie / y ná złorzecżenie / y ná háńbę.
Bo náwiedzę tych / którzy mieƺkáją w ziemi Egipſkiej / jákom náwiedził Jeruzálem miecżem / głodem y morem.
Y nie będzie / ktoby uƺedł y zoſtáł z oſtátków Judzkich / którzy weƺli do ziemi Egipſkiej / áby tám pielgrzymowáli / áby śię záś wróćić mieli do ziemi Judzkiej / do której śię oni prágną wróćić / y mieƺkáć tám ; ále śię nie wrócą / tylko ći / którzy ujdą.
Tedy odpowiedzieli Jeremijáƺowi wƺyſcy mężowie / którzy wiedzieli / iż kádzáły żony ich bogom cudzym / one wƺyſtkie niewiáſty / których ſtáło mnóſtwo wielkie / y wƺyſtek lud / który mieƺkáł w ziemi Egipſkiej w Pátroſ / mówiąc :
W ſłowie / któreś mówił do náſ imieniem Páńſkim / nie uſłuchámy ćię ;
Ale doſyć ucżynimy káżdemu ſłowu / które wynijdzie z uſt náƺych / kádząc królowej niebieſkiej / y ſpráwując jej ofiáry mokre / jákośmy cżynili / my y ojcowie náśi / królowie náśi / y kśiążętá náśi w miáſtách Judzkich y po ulicách Jeruzálemſkich / á nájádáliśmy śię chlebá / y dobrze nám było / á nic złegośmy nie widzieli.
Ale odtąd jákośmy przeſtáli kádzić królowej niebieſkiej / y ſpráwowáć jej ofiáry mokre / ná wƺyſtkiem nám ſchodzi / á od miecżá y od głodu niƺcżejemy.
A gdy kádzimy królowej niebieſkiej / y ſpráwujemy jej ofiáry mokre / izáli jej bez mężów náƺych plácki cżynimy / kƺtáłtując ją / y ſpráwując jej ofiáry mokre?
Tedy rzekł Jeremijáƺ do wƺyſtkiego ludu / do mężów y do niewiáſt / y do wƺyſtkiego poſpólſtwá / którzy mu ták odpowiedzieli / mówiąc :
Izáli ná kádzenie / któremeśćie kádzili w miáſtách Judzkich y w ulicách Jeruzálemſkich / wy y ojcowie wáśi / królowie wáśi / y kśiążętá wáśi / y lud ziemi / nie wſpomniáł Pán / y nie wſtąpiło to ná ſerce jego?
Ták / że nie mógł Pán dálej znośić złośći ſpráw wáƺych / y obrzydliwośći / któreśćie broili ; dlátego śię ſtáłá ziemiá wáƺá ſpuſtoƺeniem y zdumieniem / y przeklęſtwem / ták / że niemáƺ w niej obywátelá / jáko śię to dziś pokázuje /
Dlátego / żeśćie kádziili báłwánom / y żeśćie grzeƺyli przećiw Pánu / á nie ſłucháliśćie głoſu Páńſkiego / á ták w zákonie jego / y w uſtáwách jego / áni w świádectwách jego nie chodziliśćie / dlátego przyƺło ná wáſ to złe / jáko śię to dziś pokázuje.
Nádto rzekł Jeremijáƺ do wƺyſtkiego ludu / y do wƺyſtkich niewiáſt : Słuchájćie ſłowá Páńſkiego wƺyſcy ludzie Judzcy / którzyśćie z ziemi Egipſkiej.
Ták rzekł Pán záſtępów / Bóg Izráelſki / mówiąc : Wy y żony wáƺe mówiliśćie uſty ſwemi / y wypełniliśćie rękámi ſwemi / mówiąc : Ucżynimy doſyć ślubom náƺym / któreśmy poślubili / ábyśmy kádzili królowej niebieſkiej / y ſpráwowáli jej ofiáry mokre / á ták wƺyſtką śiłą wypełniáćie śluby wáƺe / y ſámym ſkutkiem wykonywáćie śluby wáƺe.
Przetoż ſłuchájćie ſłowá Páńſkiego wƺyſcy ludzie Judzcy / którzy mieƺkáćie w ziemi Egipſkiej : Oto Já przyśięgám przez imię moje wielkie / mówi Pán / że nie będzie więcej imię moje wzywáne uſty żádnego mężá Judzkiego po wƺyſtkiej ziemi Egipſkiej / któryby rzekł : Jáko żyje pánujący Pán!
Oto Já będę cżuł nád nimi ná złe / á nie ná dobre ; y niƺcżeć będą wƺyſcy mężowie Judzcy / którzy ſą w ziemi Egipſkiej / miecżem y głodem / áż do ƺcżętu wyginą ;
A którzy ujdą miecżá / wrócą śię z ziemi Egipſkiej do ziemi Judzkiej / ludu máły pocżet ; y pozná wƺyſtek oſtátek Judzki / którzy weƺli do ziemi Egipſkiej / áby tám pielgrzymowáli / cżyje śię ſłowo oſtoi / mojeli / cżyli ich?
A to miejćie zá znák / mówi Pán / że Já wáſ náwiedzę ná tem miejſcu / ábyśćie wiedzieli / iż śię práwdziwie ſpełnią ſłowá moje nád wámi ku złemu.
Ták mówi Pán : Oto Já podám Fáráoná Chofrá / królá Egipſkiego / w rękę nieprzyjáćiół jego / y w rękę ƺukájących duƺy jego / jákom podáł Sedekijáƺá / królá Judzkiego / w rękę Nábuchodonozorá / królá Bábilońſkiego / nieprzyjáćielá jego / który ƺukáł duƺy jego.
