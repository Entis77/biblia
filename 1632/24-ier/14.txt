Słowo Páńſkie / które śię ſtáło do Jeremijáƺá o ſuƺy.
Ziemiá Judzká płákáć będźie / á brámy jey zemdleją / żáłobę nośić będą ná źiemi / á nárzekánie Jeruzálemſkie wſtąpi w górę ;
Y zácniejśi z nich rozſyłáć będą nájpodlejƺych ſwoich po wodę ; á przyƺedłƺy do cyſtern / y nie ználázłƺy wody / náwrócą śię z nácżyniem ſwojim próżnem / zápłonąwƺy y záwſtydźiwƺy śię ; przetoż nákryją głowę ſwoję.
Dla źiemi uprágnioney / przeto / że deƺcżu nie będźie ná źiemi / y orácże wſtydząc śię nákryją głowy ſwoje.
Owƺem y łáni / co ná polu porodźiłá / opuśći ; bo ná polu tráwy nie będźie.
A oſły dźikie / ſtájąc ná wyſokich miejſcách / chwytáć będą wiátr jáko ſmoki ; uſtáną ocży ich / bo nie będźie tráwy.
O Pánie! poniewáż niepráwośći náƺe świádcżą przećiwko nám / zmiłuj śię dla imieniá twego ; boć wielkie ſą odwróceniá náƺe / tobieśmy zgrzeƺyli.
O nádzejo Izráelowá / wybáwićielu jego cżáſu utrápieniá! cżemuż máƺ być jáko przychodźień w tey źiemi / á jáko podróżny wſtępujący ná nocleg?
Cżemuż śię pokázujeƺ jáko mąż ſtrudzony / álbo jáko mocárz / który nie może wybáwić? Wƺákeś ty jeſt w pośrodku nas / Pánie! á imię twoje wzywáne jeſt nád námi ; nie opuƺcżájże nas.
Ták mówi Pán o tym ludu : Iż ták miłują tułánie / á nóg ſwych nie powśćiągáją / przetoż śię Pánu niepodobáją / y teraz wſpominá niepráwośći ich / á náwiedzá grzechy ich.
Potem rzekł Pán do mnie : Nie módl śię zá tym ludem.
Gdy pośćić będą / Ja nie wyſłuchám wołánia ich ; á gdy ofiárowáć będą cáłopálenie / y ofiárę śniedną / Ja tego nie przyjmę ; ále miecżem / y głodem / y morem wytrácę ich.
Y rzekłem : Ach / pánujący Pánie! oto im ći prorocy mówią : Nie oglądáćie miecżá / á głód nie przyjdźie ná was / ále pokój pewny dám wám ná tem miejſcu.
Y rzekł Pán do mnie : Fáłƺ prorokują ći prorocy w imieniu mojem ; nie poſłáłem ich / ánim im rozkázáł / owƺem / ánim mówił do nich ; widzenie kłámliwe / y wieƺcżbę / y márność / y kłámſtwo ſercá ſwego oni wám prorokują.
Przetoż ták mówi Pán o prorokách / którzy prorokują w imieniu mojem / choćiáżem Ja ich nie poſłáł / y którzy mówią : Miecżá áni głodu nie będźie w tey źiemi ; ći ſámi prorocy miecżem y głodem zginą.
A lud ten / któremu oni prorokują / rozrzucony będźie po ulicách Jeruzálemſkich od głodu y od miecżá / á nie będźie / ktoby ich pogrzebáł / onych ſámych / żony ich / y ſynów ich / y córki ich ; ták wyleję ná nich złość ich.
Przetoż rzecżeƺ do nich to ſłowo : Ocży moje wylewáją łzy w nocy y we dnie bez przeſtánku ; bo ſkruƺeniem wielkiem ſkruƺoná będźie pánná / córká ludu mojego / y ráną bárdzo boleſną.
Wyjdęli ná pole / oto tám pomordowáni miecżem ; wyjdęli do miáſtá / oto y tám zmorzeni głodem ; bo jáko prorok ták y kápłán obchodząc kupcżą źiemią / á ludźie tego nie bácżą.
Izáli do końcá odrzucáƺ Judę? Izáli Syon obrzydźiłá ſobie duƺá twojá? Przecż nas bijeƺ / ták ábyśmy już nie byli uzdrowieni? Ocżekujemyli ná pokój / álić oto náſtępuje nic dobrego ; á jeſli ná cżáſ ulecżeniá / á oto zátrwożenie.
Uznájemy / Pánie! niezbożność ſwoję / y niepráwość ojców náƺych / iżeśmy zgrzeƺyli przećiw tobie.
Nie odrzucájże nas dla imieniá twego / nie podáwájże w lekkość ſtolicy chwáły twojey ; wſpomnijże / nie tárgáj przymierzá twego z námi.
Izáli ſą między márnośćiámi pogáńſkimi / coby ſpuƺcżáli deƺcż? álbo niebioſá mogąli ſáme przez śię dáwáć deƺcże? Izáliś nie ty ſám Pán / Bóg náƺ? Przetoż ocżekujemy ná ćię ; bo to wƺyſtko ty cżyniƺ.
