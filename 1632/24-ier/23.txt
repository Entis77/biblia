Biádá páſterzom gubiącym y rozpráƺájącym trzodę páſtwiſká mego! mówi Pán.
Przetoż ták mówi Pán / Bóg Izráelſki / do páſterzy / którzy páſą lud mój : Wy rozpráƺáćie owce moje / owƺem / rozgániáćie je / á nie náwiedzáćie ich ; oto Ja náwiedzę was dla złośći ſpráw wáƺych / mówi Pán.
A oſtátek owiec mojich Ja zgromádzę ze wƺyſtkich źiem / do którychem je rozegnał / y przywrócę je do obór ich / gdźie śię rozpłodzą y rozmnożą.
Nádto poſánowię nád niemi páſterzy / którzyby je páśli / áby śię więcey nie lękáły / áni ſtrácháły / y żeby ich nie ubywáło / mówi Pán.
Oto idą dni / mówi Pán / których wzbudzę Dawidowi látorośl ſpráwiedliwą / y będźie królował król / á poƺcżęśći mu śię ; ſąd zájiſte y ſpráwiedliwość będźie cżynił ná źiemi.
Zá dni jego Judá zbáwiony będźie / á Izráel bezpiecżnie mieƺkáć będźie ; á toć jeſt imię jego / którem go zwáć będą : Pán ſpráwiedliwość náƺá.
Przetoż oto przychodzą dni / mówi Pán / których nie rzeką więcey : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z źiemi Egipſkiej.
Ale : Jáko żyje Pán / który wywiódł / y który ſprowádźił naśienie domu Izráelſkiego z źiemi północney y ze wƺyſtkich źiem / do którychem ich był rozegnał ; gdyż będą mieƺkáć w źiemi ſwojej.
Dla proroków ſkruƺyło śię ſerce moje we mnie / poruƺyły śię wƺyſtkie kośći moje / ſtałem śię jáko cżłowiek pijány / á jáko mąż / po którym śię rozeƺło wino / dla Páná y dla ſłów świętobliwośći jego.
Bo tá źiemiá pełna jeſt cudzołożników / á dla krzywoprzyśięſtwá płácże tá źiemiá / páſtwiſká ná puƺcży poſchły ; zájiſte zły jeſt bieg ich / á moc ich niepráwá.
Bo y prorok y kápłán ſą obłudnikámi / á domu moim znájduje śię złość ich / mówi Pán.
Przetoż im będźie drogá ich jáko ślizgáwicá w ćiemnośći / ná którą wepchnieni będą y upádną / gdy przywiodę ná nich biedę cżáſu náwiedzeniá ich / mówi Pán.
Tákże przy prorokách Sámáryjſkich widźiałem głupſtwo / prorokowáli przez Báále / y zwodźili lud mój Izráelſki.
Ale przy prorokách Jeruzálemſkich widzę rzecż brzydką / że cudzołożąc y kłámliwie śię obchodząc utwierdzáją też ręce złośników / áby śię nie náwrácáli żáden od złośći ſwojey ; ſtáli śię wƺyſcy przedemną jáko Sodomá / á obywátele jego jáko Gomorá.
Dlátegoż ták mówi Pán zaſtępów o tych prorokách : Oto Ja nákármię ich piołunem / á nápoję ich wodą gorzką ; bo od proroków Jeruzálemſkich wyƺło ſplugáwienie ná tę wƺyſtkę źiemię.
Ták mówi Pán zaſtępów : Nie ſłuchájćie ſłów tych proroków / którzy wam prorokują / y zwodzą was / widzenie ſercá ſwego opowiádáją / á nie z uſt Páńſkich ;
Uſtáwicżnie mówią tym / którzy mną gárdzą : Pán mówił / pokój mieć będźiećie ; y káżdemu chodzącemu według uporu ſercá ſwego / mówią : Nie przydźie ná was nic złego.
Bo któż ſtánął w rádźie Páńſkiej / á widźiał y ſłyƺał ſłowo jego? kto pilnował ſłowá jego / y ſłuchał go?
Oto wicher Páńſki z zápálcżywośćią wyjdźie / á wicher trwáły nád głową niepobożnych zoſtánie ;
Nie odwróći śię gniew Páńſki / áż ucżyni y wykoná myśli ſercá ſwego ; w oſtátecżne dni to doſkonále zrozumiećie.
Nie poſyłáłem tych proroków / á wƺákże bieżeli ; nie mówiłem do nich / á wƺákże oni prorokowáli /
Bo gdyby byli ſtáli w rádźie mojey / tedyby byli ogłáƺáli ſłowá moje ludowi memu / á byliby ich odwrácáli od drogi ich złej / y od złośći ſpráw ich.
Izálim ja tylko Bogiem z bliſká? mówi Pán / á nie jeſtem Bogiem y z dáleká?
Izali śię kto ſkryje w ſkrytośći / ábym go Ja nie widźiał? mówi Pán. Izali Ja niebá y źiemi nie nápełniám? mówi Pán.
Słyƺęć Ja / co mówią prorocy / którzy prorokują / kłámſtwo w imieniu mojim / mówiąc : Śniło mi śię / śniło mi śię!
Długoż tego będźie? Izali w ſercu tych proroków / którzy prorokują / niemáƺ kłámſtwá? owƺem / ſą prorokámi zdrády ſercá ſwego ;
Którzy myślą / jákoby z pámięći wywieść ludowi mojemu imię moje ſnámi ſwemi / które powiádáją káżdy bliźniemu ſwemu / jáko zápomnieli ojcowie ich ná imię moje dla Báálá.
Prorok / który ma ſen / niech powiádá ſen : ále który ma ſłowo moje / niech mówi ſłowo moje w práwdźie. Cóż plewie do pƺenicy? mówi Pán.
Izali ſłowo moje nie jeſt jáko ogień? mówi Pán / y jáko młot kruƺący ſkáłę?
Przetoż oto Ja powſtáję przećiwko tym prorokom / mówi Pán / którzy krádną ſłowá moje / káżdy przed bliźnim ſwoim.
Oto Ja powſtáję przećiwko tym prorokom / mówi Pán / którzy używáją języká ſwego / áby mówili : Mówi Pán.
Oto Ja powſtáję / mówi Pán / przećiwko tym / którzy prorokują ſny kłámliwe / á opowiádájąc je zwodzą lud mój kłámſtwámi ſwemi y plotkámi ſwemi / choćiáżem Ja ich nie poſłał / ánim im rozkázał ; ſkąd zgołá nic nie pomágáją ludowi temu / mówi Pán.
Przetoż gdyby śię ćiebie pytał lud ten / álbo prorok / álbo kápłán / mówiąc : Cóż jeſt zá brzemię Páńſkie? Tedy rzecżeƺ do nich które brzemię. To : Opuƺcżę was / mówi Pán.
Bo proroká y kápłáná tego / y lud ten / któryby rzekł : Toć jeſt brzemię Páńſkie / pewnie náwiedzę mężá tego y dom jego.
Ale ták mówćie káżdy do bliźniego ſwego / y káżdy do brátá ſwego : Cóż odpowiedźiał Pán? álbo : Cóż mówił Pán?
A brzmieniá Páńſkiego nie wſpominájćie więcey ; bo brzmieniem będźie káżdemu ſłowo jego / gdyżeśćie wy wywróćili ſłowá Bogá żywego / Páná zaſtępów / Bogá náƺego.
Ták tedy rzecżeƺ do proroká : Cóż ći odpowiedźiał Pán? álbo : Co mówił Pán?
Ale ponieważ mówićie : Brzemię Páńſkie / tedy ták mówi Pán : Ponieważ mówićie to ſłowo : Brzemię Páńſkie / choćiáżem do was poſyłał / mówiąc : Nie mówćie : Brzemię Páńſkie ;
Przetoż oto y Ja was zápomnę do końcá / y odrzucę was y to miáſto / którem wam dał y ojcom wáƺym / od oblicża mego ;
Y podám was ná urągánie wiecżne / y ná háńbę wiecżną / która nigdy nie przydźie w zápámiętánie.
