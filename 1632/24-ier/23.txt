Biádá páſterzom gubiącym y rozpráƺájącym trzodę páſtwiſká mego! mówi Pán.
Przetoż ták mówi Pán / Bóg Izráelſki / do páſterzy / którzy páſą lud mój : Wy rozpráƺáćie owce moje / owƺem / rozgániáćie je / á nie náwiedzáćie ich ; oto Já náwiedzę wáſ dlá złośći ſpráw wáƺych / mówi Pán.
A oſtátek owiec moich Já zgromádzę ze wƺyſtkich ziem / do którychem je rozegnáł / y przywrócę je do obór ich / gdzie śię rozpłodzą y rozmnożą.
Nádto poſánowię nád niemi páſterzy / którzyby je páśli / áby śię więcej nie lękáły / áni ſtrácháły / y żeby ich nie ubywáło / mówi Pán.
Oto idą dni / mówi Pán / których wzbudzę Dawidowi látorośl ſpráwiedliwą / y będzie królowáł król / á poƺcżęśći mu śię ; ſąd záiſte y ſpráwiedliwość będzie cżynił ná ziemi.
Zá dni jego Judá zbáwiony będzie / á Izráel bezpiecżnie mieƺkáć będzie ; á toć jeſt imię jego / którem go zwáć będą : Pán ſpráwiedliwość náƺá.
Przetoż oto przychodzą dni / mówi Pán / których nie rzeką więcej : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z ziemi Egipſkiej.
Ale : Jáko żyje Pán / który wywiódł / y który ſprowádził náśienie domu Izráelſkiego z ziemi północnej y ze wƺyſtkich ziem / do którychem ich był rozegnáł ; gdyż będą mieƺkáć w ziemi ſwoiej.
Dlá proroków ſkruƺyło śię ſerce moje we mnie / poruƺyły śię wƺyſtkie kośći moje / ſtáłem śię jáko cżłowiek pijány / á jáko mąż / po którym śię rozeƺło wino / dlá Páná y dlá ſłów świętobliwośći jego.
Bo tá ziemiá pełná jeſt cudzołożników / á dlá krzywoprzyśięſtwá płácże tá ziemiá / páſtwiſká ná puƺcży poſchły ; záiſte zły jeſt bieg ich / á moc ich niepráwá.
Bo y prorok y kápłán ſą obłudnikámi / á domu moim znájduje śię złość ich / mówi Pán.
Przetoż im będzie drogá ich jáko ślizgáwicá w ćiemnośći / ná którą wepchnieni będą y upádną / gdy przywiodę ná nich biedę cżáſu náwiedzeniá ich / mówi Pán.
Tákże przy prorokách Sámáryjſkich widziáłem głupſtwo / prorokowáli przez Báále / y zwodzili lud mój Izráelſki.
Ale przy prorokách Jeruzálemſkich widzę rzecż brzydką / że cudzołożąc y kłámliwie śię obchodząc utwierdzáją też ręce złośników / áby śię nie náwrácáli żáden od złośći ſwojej ; ſtáli śię wƺyſcy przedemną jáko Sodomá / á obywátele jego jáko Gomorá.
Dlátegoż ták mówi Pán záſtępów o tych prorokách : Oto Já nákármię ich piołunem / á nápoję ich wodą gorzką ; bo od proroków Jeruzálemſkich wyƺło ſplugáwienie ná tę wƺyſtkę ziemię.
Ták mówi Pán záſtępów : Nie ſłuchájćie ſłów tych proroków / którzy wám prorokują / y zwodzą wáſ / widzenie ſercá ſwego opowiádáją / á nie z uſt Páńſkich ;
Uſtáwicżnie mówią tym / którzy mną gárdzą : Pán mówił / pokój mieć będziećie ; y káżdemu chodzącemu według uporu ſercá ſwego / mówią : Nie przyjdzie ná wáſ nic złego.
Bo któż ſtánął w rádzie Páńſkiej / á widziáł y ſłyƺáł ſłowo jego? kto pilnowáł ſłowá jego / y ſłucháł go?
Oto wicher Páńſki z zápálcżywośćią wyjdzie / á wicher trwáły nád głową niepobożnych zoſtánie ;
Nie odwróći śię gniew Páńſki / áż ucżyni y wykoná myśli ſercá ſwego ; w oſtátecżne dni to doſkonále zrozumiećie.
Nie poſyłáłem tych proroków / á wƺákże bieżeli ; nie mówiłem do nich / á wƺákże oni prorokowáli /
Bo gdyby byli ſtáli w rádzie mojej / tedyby byli ogłáƺáli ſłowá moje ludowi memu / á byliby ich odwrácáli od drogi ich złej / y od złośći ſpráw ich.
Izálim já tylko Bogiem z bliſká? mówi Pán / á nie jeſtem Bogiem y z dáleká?
Izáli śię kto ſkryje w ſkrytośći / ábym go Já nie widziáł? mówi Pán. Izáli Já niebá y ziemi nie nápełniám? mówi Pán.
Słyƺęć Já / co mówią prorocy / którzy prorokują / kłámſtwo w imieniu mojem / mówiąc : Śniło mi śię / śniło mi śię!
Długoż tego będzie? Izáli w ſercu tych proroków / którzy prorokują / niemáƺ kłámſtwá? owƺem / ſą prorokámi zdrády ſercá ſwego ;
Którzy myślą / jákoby z pámięći wywieść ludowi mojemu imię moje ſnámi ſwemi / które powiádáją káżdy bliźniemu ſwemu / jáko zápomnieli ojcowie ich ná imię moje dlá Báálá.
Prorok / który má ſen / niech powiádá ſen : ále który má ſłowo moje / niech mówi ſłowo moje w práwdzie. Cóż plewie do pƺenicy? mówi Pán.
Izáli ſłowo moje nie jeſt jáko ogień? mówi Pán / y jáko młot kruƺący ſkáłę?
Przetoż oto Já powſtáję przećiwko tym prorokom / mówi Pán / którzy krádną ſłowá moje / káżdy przed bliźnim ſwoim.
Oto Já powſtáję przećiwko tym prorokom / mówi Pán / którzy używáją języká ſwego / áby mówili : Mówi Pán.
Oto Já powſtáję / mówi Pán / przećiwko tym / którzy prorokują ſny kłámliwe / á opowiádájąc je zwodzą lud mój kłámſtwámi ſwemi y plotkámi ſwemi / choćiáżem Já ich nie poſłáł / ánim im rozkázáł ; ſkąd zgołá nic nie pomágáją ludowi temu / mówi Pán.
Przetoż gdyby śię ćiebie pytáł lud ten / álbo prorok / álbo kápłán / mówiąc : Cóż jeſt zá brzemię Páńſkie? Tedy rzecżeƺ do nich które brzemię. To : Opuƺcżę wáſ / mówi Pán.
Bo proroká y kápłáná tego / y lud ten / któryby rzekł : Toć jeſt brzemię Páńſkie / pewnie náwiedzę mężá tego y dom jego.
Ale ták mówćie káżdy do bliźniego ſwego / y káżdy do brátá ſwego : Cóż odpowiedziáł Pán? álbo : Cóż mówił Pán?
A brzmieniá Páńſkiego nie wſpominájćie więcej ; bo brzmieniem będzie káżdemu ſłowo jego / gdyżeśćie wy wywróćili ſłowá Bogá żywego / Páná záſtępów / Bogá náƺego.
Ták tedy rzecżeƺ do proroká : Cóż ći odpowiedziáł Pán? álbo : Co mówił Pán?
Ale poniewáż mówićie : Brzemię Páńſkie / tedy ták mówi Pán : Poniewáż mówićie to ſłowo : Brzemię Páńſkie / choćiáżem do wáſ poſyłáł / mówiąc : Nie mówćie : Brzemię Páńſkie ;
Przetoż oto y Já wáſ zápomnę do końcá / y odrzucę wáſ y to miáſto / którem wám dáł y ojcom wáƺym / od oblicżá mego ;
Y podám wáſ ná urągánie wiecżne / y ná háńbę wiecżną / którá nigdy nie przyjdzie w zápámiętánie.
