Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Nie pojmuj ſobie żony / áni miej ſynów áni córek ná tym miejſcu.
Abowiem ták mówi Pán o ſynách y o córkách ſpłodzonych ná tym miejſcu / y o mátkách ich / które ich zrodźiły / y o ojcách ich / którzy ich ſpłodźili w tey źiemi ;
Śmierćiámi ćiężkiemi pomrą ; nie będą ich płákáć / áni ich pochowáją / ále miáſto gnoju ná wierzchu źiemi będą ; á miecżem y głodem wytráceni będą ; y będą trupy ich pokármem ptáſtwu niebieſkiemu y zwierzowi źiemſkiemu.
Bo ták mówi Pán : Nie wchodź do domu żáłoby / áni chodź ná płácż / áni ich żáłuj ; bom odjął pokój mój od ludu tego / miłośierdźie y litość / mówi Pán.
Gdy pomrą wielcy y máli w tey źiemi / nie będą pogrzebieni / áni ich płákáć będą ; y nie będą śię rzezáć / áni ſobie łyśiny cżynić dla nich ;
Ani im dádzą jeść / áby ich w ſmutku ćieƺyli nád umárłym ; áni im dádzą pić z kubká poćieƺeniá po ojcu ich y po mátce ich ;
Tákże do domu ucżty nie wchodź / ábyś záśiádał z nimi / y jádł / y pił.
Abowiem ták mówi Pán zaſtępów / Bóg Izráelſki : Oto Ja ſpráwię / iż uſtánie ná tym miejſcu przed ocżymá wáƺemi / y zá dni wáƺych głos weſelá / y głos rádośći / głos oblubieńcá / y głos oblubienicy.
A gdy opowieƺ ludowi temu wƺyſtkie te ſłowá / á rzekliby do ćiebie : Przecż Pán wyrzekł przećiwko nam to wƺyſtko wielkie złe? Y cóż jeſt zá niepráwość náƺá / y co zá grzech náƺ / którymeśmy zgrzeƺyli przećiwko Pánu / Bogu náƺemu?
Tedy rzecżeƺ do nich : Przeto / iż mię opuśćili ojcowie wáśi / ( mówi Pán / ) á chodźili zá bogámi cudzymi / y ſłużyli im / y kłániáli śię im / lecż mnie opuśćili / y zakonu mego nie przeſtrzegáli.
A wy dálekośćie gorzej cżynili / niż ojcowie wáśi ; ábowiem oto káżdy chodźi zá uporem złego ſercá ſwego / nie ſłuchájąc mię ;
Dla tego wyrzucę was z tey źiemi do źiemi / którejśćie nie ználi / wy y ojcowie wáśi / á tám ſłużyć będźiećie bogom cudzym we dnie y w nocy / dokąd wam nie okáżę miłośierdźiá.
Przetoż oto dni idą / mówi Pán / że nie rzeką więcey : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z źiemi Egipſkiej.
Ale : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z źiemi północney / y ze wƺyſtkich źiem / do których ich był wygnał / gdy ich záśię przywiodę do źiemi ich / którąm dał ojcom ich.
Oto Ja poślę do wielu rybitwów / ( mówi Pán / ) áby ich łowili ; potem poślę do wielu łowców / áby ich łápáli ná wƺelkiej górze y ná wƺelkim págórku / y w dźiurách ſkálnych.
Ocży moje pátrzą ná wƺyſtkie drogi ich ; nie ſą utájone przed oblicżem mojim / áni jeſt zákryta niepráwość ich przed ocżymá mojemi.
Y oddam im pierwej w dwójnáſób zá niepráwośći ich / y zá grzechy ich / przeto / że źiemię moję ſplugáwili trupámi obrzydliwośći ſwojey / y ſproſnośćiámi ſwemi nápełnili dźiedźictwo moje.
Pánie / mocy mojá y śiło mojá / y ućiecżko mojá w dźień utrapienia! do ćiebie przyjdą narody od końcżyn źiemi / y rzekną : Zájiſte śię fáłƺu trzymáli ojcowie náśi / y márnośći / w których żádnego pożytku nie było.
Izali ſobie cżłowiek ucżynić może bogów? ponieważ ſámi nie ſą bogámi.
Dla tego oto Ja ſpráwię / áby poználi tym rázem ; ſpráwię / mówię / áby poználi rękę moję y moc moję / y dowiedźieli śię / że imię moje jeſt Pán.
