Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Nie pojmuj ſobie żony / áni miej ſynów áni córek ná tem miejſcu.
Abowiem ták mówi Pán o ſynách y o córkách ſpłodzonych ná tem miejſcu / y o mátkách ich / które ich zrodźiły / y o ojcách ich / którzy ich ſpłodźili w tej źiemi ;
Śmierćiámi ćiężkiemi pomrą ; nie będą ich płákáć / áni ich pochowáją / ále miáſto gnoju ná wierzchu źiemi będą ; á miecżem y głodem wytráceni będą ; y będą trupy ich pokármem ptáſtwu niebieſkiemu y zwierzowi źiemſkiemu.
Bo ták mówi Pán : Nie wchodź do domu żáłoby / áni chodź ná płácż / áni ich żáłuj ; bom odjął pokój mój od ludu tego / miłośierdźie y litość / mówi Pán.
Gdy pomrą wielcy y máli w tej źiemi / nie będą pogrzebieni / áni ich płákáć będą ; y nie będą śię rzezáć / áni ſobie łyśiny cżynić dlá nich ;
Ani im dádzą jeść / áby ich w ſmutku ćieƺyli nád umárłym ; áni im dádzą pić z kubká poćieƺeniá po ojcu ich y po mátce ich ;
Tákże do domu ucżty nie wchodź / ábyś záśiádáł z nimi / y jádł / y pił.
Abowiem ták mówi Pán záſtępów / Bóg Izráelſki : Oto Já ſpráwię / iż uſtánie ná tem miejſcu przed ocżymá wáƺemi / y zá dni wáƺych głoſ weſelá / y głoſ rádośći / głoſ oblubieńcá / y głoſ oblubienicy.
A gdy opowieƺ ludowi temu wƺyſtkie te ſłowá / á rzekliby do ćiebie : Przecż Pán wyrzekł przećiwko nám to wƺyſtko wielkie złe? Y cóż jeſt zá niepráwość náƺá / y co zá grzech náƺ / którymeśmy zgrzeƺyli przećiwko Pánu / Bogu náƺemu?
Tedy rzecżeƺ do nich : Przeto / iż mię opuśćili ojcowie wáśi / ( mówi Pán / ) á chodźili zá bogámi cudzymi / y ſłużyli im / y kłániáli śię im / lecż mnie opuśćili / y zákonu mego nie przeſtrzegáli.
A wy dálekośćie gorzej cżynili / niż ojcowie wáśi ; ábowiem oto káżdy chodźi zá uporem złego ſercá ſwego / nie ſłuchájąc mię ;
Dlátego wyrzucę wáſ z tej źiemi do źiemi / którejśćie nie ználi / wy y ojcowie wáśi / á tám ſłużyć będźiećie bogom cudzym we dnie y w nocy / dokąd wám nie okáżę miłośierdźiá.
Przetoż oto dni idą / mówi Pán / że nie rzeką więcej : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z źiemi Egipſkiej.
Ale : Jáko żyje Pán / który wywiódł ſynów Izráelſkich z źiemi północnej / y ze wƺyſtkich źiem / do których ich był wygnáł / gdy ich záśię przywiodę do źiemi ich / którąm dáł ojcom ich.
Oto Já poślę do wielu rybitwów / ( mówi Pán / ) áby ich łowili ; potem poślę do wielu łowców / áby ich łápáli ná wƺelkiej górze y ná wƺelkim págórku / y w dźiurách ſkálnych.
Ocży moje pátrzą ná wƺyſtkie drogi ich ; nie ſą utájone przed oblicżem mojem / áni jeſt zákrytá niepráwość ich przed ocżymá mojemi.
Y oddám im pierwej w dwójnáſób zá niepráwośći ich / y zá grzechy ich / przeto / że źiemię moję ſplugáwili trupámi obrzydliwośći ſwojej / y ſproſnośćiámi ſwemi nápełnili dźiedźictwo moje.
Pánie / mocy mojá y śiło mojá / y ućiecżko mojá w dźień utrápieniá! do ćiebie przyjdą národy od końcżyn źiemi / y rzekną : Záiſte śię fáłƺu trzymáli ojcowie náśi / y márnośći / w których żádnego pożytku nie było.
Izáli ſobie cżłowiek ucżynić może bogów? poniewáż ſámi nie ſą bogámi.
Dlátego oto Já ſpráwię / áby poználi tym rázem ; ſpráwię / mówię / áby poználi rękę moję y moc moję / y dowiedźieli śię / że imię moje jeſt Pán.
