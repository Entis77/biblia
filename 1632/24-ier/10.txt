Słuchájćie ſłowá tego / które Pán mówi do was / o domie Izráelſki!
Ták mówi Pán : Drogi pogáńſkiej nie ucżćie śię / á zánámion niebieſkich nie bójćie śię ; bo śię ich pogánie boją.
Uſtáwy záiſte tych národów ſą wierutná márność ; bo ućiąwƺy drzewo śiekierą w leśie / dźieło rąk rzemieślniká /
Srebrem y złotem ozdábiá je / gwoźdźiámi y młotámi utwierdzá je / áby śię nie rucháło.
Stoją proſto jáko pálmá / á nie mówią ; noƺone być muƺą / bo chodźić nie mogą. Nie bójćie śię ich ; bo źle cżynić nie mogą / y dobrze cżynić nie mogą.
Żáden z tych nie jeſt tobie podobny / Pánie! wielkiś ty / y wielkie jeſt imię twoje w mocy.
Któżby śię ćiebie nie báł? Królu národów! Tobie to záiſte náleży / poniewáż między wƺyſtkimi mędrcámi národów / y we wƺyſtkich króleſtwách ich nigdy nie był podobny tobie.
A wƺákże ſpołem zgłupieli y poƺáleli ; z drewná bráć náukę / jeſt wierutná márność.
Srebro ćiągnione z zámorzá przywożone bywá / á złoto z Ufáſ / dźieło rzemieślnicże / y ręki złotniká ; hijácynt y ƺárłát odźienie ich / wƺyſtko to jeſt dźieło umiejętnych.
Ale Pán jeſt Bóg práwy / jeſt Bóg żywy / y król wiecżny ; przed jego zápálcżywośćią źiemiá drży / á národy nie mogą znieść rozgniewániá jego.
Ták im tedy powiećie : Bogowie ći / którzy niebá y źiemi nie ſtworzyli / niech zginą z źiemi / á niech ich nie będźie pod niebem.
Ale on ucżynił źiemię mocą ſwą ; on utwierdźił okrąg świátá mądrośćią ſwoją / y roztropnośćią ſwoją rozćiągnął niebioſá.
Gdy on wydáje głoſ / ƺum wód bywá ná niebie / y to ſpoſábiá / áby wyſtępowáły páry z krájów źiemi ; błyſkáwice z deƺcżem przywodźi / á wywodźi wiátry z ſkárbów ſwoich.
Ták zgłupiáł káżdy cżłowiek / że tego nie zná / iż poháńbiony bywá káżdy rzemieślnik dla báłwáná ; bo fáłƺem jeſt to / co uláł / y niemáƺ duchá w nich.
Márnośćią ſą / á dźiełem błędów ; cżáſu náwiedzeniá ſwego poginą.
Nie jeſt tym podobien dźiáł Jákóbowy / bo on jeſt ſtworzyćiel wƺyſtkiego ; Izráel tákże jeſt prętem dźiedźictwá jego / Pán záſtępów jeſt imię jego.
Zbierz z źiemi towáry twoje / ty / która mieƺkáƺ ná miejſcu obronnem.
Bo ták mówi Pán : Oto Ja jáko z procy ugodzę obywáteli źiemi jednym rázem / y udręcżę / áby tego doználi y rzekli :
Biádá mnie nád zniƺcżeniem mojem : boleſná jeſt ráná mojá / choćiáżem był rzekł : Záiſte tę niemoc będę mógł znieść.
Námiot mój zburzony jeſt / y wƺyſtkie powrozy moje porwáne ſą ; ſynowie moi poƺli odemnie / y niemáƺ ich ; niemáƺ / ktoby więcey rozbijáł námiot mój / á rozćiągáł opony moje.
Bo páſterze zgłupieli / á Páná śię nie dokłádáli ; dlátego nie powodźi śię im ƺcżęśliwie / á wƺyſtká trzodá páſtwiſká ich rozproƺoná jeſt.
Oto wieść pewná przychodźi / á wzruƺenie wielkie z źiemi północney / áby obrócone były miáſtá Judzkie w puſtynie / y w mieƺkánie ſmoków.
Wiem / Pánie! że nie jeſt w mocy cżłowieká drogá jego / áni jeſt w mocy mężá tego / który chodźi / áby ſpráwowáł poſtępki ſwe.
Kárz mię / Pánie! ále łáſkáwie / nie w gniewie ſwym / byś mię ſnáć wniwecż nie obróćił.
Wylej popędliwość twoję ná te národy / które ćię nie znáją y ná rodzáje / które imieniá twego nie wzywáją ; bo jedzą Jákóbá / y pożeráją go / áby go wƺyſtkiego ſtráwili / y mieƺkánie jego w puſtki obróćili.
