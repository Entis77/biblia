Słuchájćie ſłowá tego / które Pán mówi do was / o domie Izráelſki!
Ták mówi Pán : Drogi pogáńſkiej nie ucżćie śię / á zánámion niebieſkich nie bójćie śię ; bo śię ich pogánie boją.
Uſtáwy zájiſte tych narodów ſą wierutná márność ; bo ućiąwƺy drzewo śiekierą w leśie / dźieło rąk rzemieślniká /
Srebrem y złotem ozdábiá je / gwoźdźiámi y młotámi utwierdzá je / áby śię nie rucháło.
Stoją proſto jáko pálmá / á nie mówią ; noƺone być muƺą / bo chodźić nie mogą. Nie bójćie śię ich ; bo źle cżynić nie mogą / y dobrze cżynić nie mogą.
Żáden z tych nie jeſt tobie podobny / Pánie! wielkiś ty / y wielkie jeſt imię twoje w mocy.
Któżby śię ćiebie nie bał? Królu narodów! Tobie to zájiſte náleży / ponieważ miedzy wƺyſtkimi mędrcámi narodów / y we wƺyſtkich króleſtwách ich nigdy nie był podobny tobie.
A wƺákże ſpołem zgłupieli y poƺáleli ; z drewná bráć náukę / jeſt wierutná márność.
Srebro ćiągnione z zámorzá przywożone bywa / á złoto z Ufás / dźieło rzemieślnicże / y ręki złotniká ; hijácynt y ƺárłát odźienie ich / wƺyſtko to jeſt dźieło umiejętnych.
Ale Pán jeſt Bóg práwy / jeſt Bóg żywy / y król wiecżny ; przed jego zápálcżywośćią źiemiá drży / á narody nie mogą znieść rozgniewánia jego.
Ták im tedy powiećie : Bogowie ći / którzy niebá y źiemi nie ſtworzyli / niech zginą z źiemi / á niech ich nie będźie pod niebem.
Ale on ucżynił źiemię mocą ſwą ; on utwierdźił okrąg świátá mądrośćią ſwoją / y roztropnośćią ſwoją rozćiągnął niebioſá.
Gdy on wydáje głos / ƺum wód bywa ná niebie / y to ſpoſábiá / áby wyſtępowáły páry z krájów źiemi ; błyſkáwice z deƺcżem przywodźi / á wywodźi wiátry z ſkárbów ſwojich.
Ták zgłupiał káżdy cżłowiek / że tego nie zná / iż poháńbiony bywa káżdy rzemieślnik dla báłwáná ; bo fáłƺem jeſt to / co ulał / y niemáƺ duchá w nich.
Márnośćią ſą / á dźiełem błędów ; cżáſu náwiedzeniá ſwego poginą.
Nie jeſt tym podobien dźiał Jákóbowy / bo on jeſt ſtworzyćiel wƺyſtkiego ; Izráel tákże jeſt prętem dźiedźictwá jego / Pán zaſtępów jeſt imię jego.
Zbierz z źiemi towáry twoje / ty / która mieƺkáƺ ná miejſcu obronnem.
Bo ták mówi Pán : Oto Ja jáko z procy ugodzę obywáteli źiemi jednym rázem / y udręcżę / áby tego doználi y rzekli :
Biádá mnie nád zniƺcżeniem mojim : boleſná jeſt ráná mojá / choćiáżem był rzekł : Zájiſte tę niemoc będę mógł znieść.
Námiot mój zburzony jeſt / y wƺyſtkie powrozy moje porwáne ſą ; ſynowie moi poƺli odemnie / y niemáƺ ich ; niemáƺ / ktoby więcey rozbijał námiot mój / á rozćiągał opony moje.
Bo páſterze zgłupieli / á Páná śię nie dokłádáli ; dla tego nie powodźi śię im ƺcżęśliwie / á wƺyſtká trzodá páſtwiſká ich rozproƺoná jeſt.
Oto wieść pewná przychodźi / á wzruƺenie wielkie z źiemi północney / áby obrócone były miáſtá Judzkie w puſtynie / y w mieƺkánie ſmoków.
Wiem / Pánie! że nie jeſt w mocy cżłowieká drogá jego / áni jeſt w mocy mężá tego / który chodźi / áby ſpráwował poſtępki ſwe.
Kárz mię / Pánie! ále łáſkáwie / nie w gniewie ſwym / byś mię ſnadź wniwecż nie obróćił.
Wylej popędliwość twoję ná te narody / które ćię nie znáją y ná rodzáje / które imienia twego nie wzywáją ; bo jedzą Jákóbá / y pożeráją go / áby go wƺyſtkiego ſtráwili / y mieƺkánie jego w puſtki obróćili.
