Y ſtáło śię roku onego / ná pocżątku królowánia Sedekijáƺá / królá Judzkiego / roku cżwártego / mieśiącá piątego : Hánánijáƺ / ſyn Aſurowy / prorok / który był z Gábáonu / rzekł do mnie w domu Páńſkim przed kápłánámi y przed wƺyſtkim ludem / mówiąc :
Ták mówi Pán zaſtępów / Bóg Izráelſki : Połámáłem járzmo królá Bábilońſkiego ;
Po dwu látách przywrócę záśię ná to miejſce wƺyſtkie nacżynia domu Páńſkiego / które pobrał Nábuchodonozor / król Bábilońſki / z tego miejſcá / á záwiózł je do Bábilonu.
Tákże Jechonijáƺá / ſyná Joákimowego / królá Judzkiego / y wƺyſtkich / którzy ſą w niewolę záprowádzeni z Judy / którzy śię doſtáli do Bábilonu / Ja záśię przywiodę ná to miejſce / mówi Pán ; bo ſkruƺę járzmo królá Bábilońſkiego.
Tedy rzekł Jeremijáƺ prorok do Hánánijáƺá proroká przed ocżymá kápłanów / y przed ocżymá wƺyſtkiego ludu / którzy ſtáli w domu Páńſkim ;
Rzekł / mówię / Jeremijáƺ prorok : Amen / niech ták ucżyni Pán ; niech utwierdźi Pán ſłowá twoje / któreś prorokował o przywróceniu z Bábilonu ná to miejſce nacżynia domu Páńſkiego / y wƺyſtkich / którzy ſą záprowádzeni w niewolę.
Wƺákże poſłucháj teraz ſłowá tego / które ja mówię w uƺy twoje / y w uƺy tego wƺyſtkiego ludu.
Prorocy / którzy byli przedemną y przed tobą z dáwná / ći prorokowáli przećiwko źiemiom zácnym / y przećiwko króleſtwom wielkim o wojnie / y o ućiśnieniu / y o morze.
Ten prorok / który prorokuje o pokoju / ten prorok / mówię / wtencżás poznány bywa / że go Pán práwdźiwie poſłał / gdy śię iśći ſłowo jego.
Tedy zdjął Hánánijáƺ prorok járzmo z ƺyi Jeremijáƺá proroká / y połámał je.
Y rzekł Hánánijáƺ przed ocżymá wƺyſtkiego ludu / mówiąc : Ták mówi Pán : Ták połámię járzmo Nábuchodonozorá / królá Bábilońſkiego / po dwu látách z ƺyi wƺyſtkich narodów. Y poƺedł Jeremijáƺ prorok w drogę ſwoję.
Ale ſtáło śię ſłowo Páńſkie do Jeremijáƺá / gdy połámał Hánánijáƺ prorok ono járzmo z ƺyi Jeremijáƺá proroká / mówiąc :
Idź / á rzecż do Hánánijáƺá / mówiąc : Ták mówi Pán : Połámáłeś járzmá drewniáne ; przetoż ucżyń ná to miejſce járzmo żelázne.
Bo ták mówi Pán zaſtępów / Bóg Izráelſki : Járzmo żelázne włożę ná ƺyję tych wƺyſtkich narodów / áby ſłużyły Nábuchodonozorowi / królowi Bábilońſkiemu / y będą mu ſłużyły ; tákże y zwierzętá polne podáłem mu.
Zátem rzekł Jeremijáƺ prorok do Hánánijáƺá proroká : Słucháj teraz Hánánijáƺu! Nie poſłał ćię Pán / á tyś kázał nádźieję mieć temu ludowi w kłámſtwie.
Przetoż ták mówi Pán : Oto Ja ćiebie uprzątnę z tey źiemi / tego roku umrzeƺ ; boś rádźił / áby odſtąpił lud od Páná.
Y umárł Hánánijáƺ prorok onegoż roku / mieśiącá śiódmego.
