Przećiwko ſynom Amonowym. Ták mówi Pán : Izali Izráel nie ma ſynów? Izali żádnego dźiedźicá nie ma? cżemuż król ich dźiedźicżnie opánował Gád / á lud jego cżemu mieƺká w miáſtách jego?
Przetoż oto dni idą / mówi Pán / ſpráwię / że uſłyƺą trąbienie wojenne przećiwko Rábbie ſynów Amonowych / y będźie obróconá w kupę rumu / á inne miáſtá jego ogniem ſpálone będą / y pośiędźie Izráel dźierżáwców ſwojich / mówi Pán.
Rozrzewnij śię / Heſebonie! bo Háj ſpuſtoƺone będźie ; krzycżćie / o córki Rábby! przepáƺćie śię worámi / nárzekájćie / á tułájćie śię około płotów ; bo król wáƺ do więźieniá pójdźie / tákże kápłáni jego / y kśiążętá jego ſpołem.
Cóż śię przechwáláƺ dolinámi? Spłynęłá doliná twojá / o córko uporná! która ufáƺ w ſkárbách twojich / mówiąc : Któż przyćiągnie przećiwko mnie?
Oto Ja przywiodę ná ćię ſtrách / mówi Pán / Pán zaſtępów od wƺyſtkich / którzy ſą około ćiebie / przez który rozegnáni będźiećie jeden od drugiego / á nie będźie / ktoby zebrał tułájących śię.
Wƺákże potem przywiodę záś więźniów ſynów Amonowych / mówi Pán.
Przećiwko Edomcżykom. Ták mówi Pán zaſtępów : Izali niemáƺ więcey mądrośći w Temán? Izali zginęłá rádá od roztropnych / á wniwecż śię obróćiłá mądrość ich?
Ućiekájćie / obróććie śię / zſtąpćie w głębokość ná mieƺkánie / obywátele Dedán! bo przywiodę zátrácenie ná Ezáwá cżáſu náwiedzeniá jego.
Gdyby ći / którzy zbieráją wino / przyƺli ná ćię / izáliby nie zoſtáwili jákich gron? Gdyby śię wkrádli złodźieje w nocy / ázáżby ƺkodźili więcey nád potrzebę ſwoję?
Lecż Ja obnáżę Ezáwá / odkryję ſkrytośći jego / ták / iż śię ukryć nie będźie mógł : zniƺcżone będźie naśienie jego / y bráćia jego / y ſąśiedźi jego / ták / że zgołá nie będźie / ktoby rzekł :
Zániecháj śierotek twojich / ja je żywić będę / á wdowy twoje we mnie ufáć będą.
Ták zájiſte mówi Pán : Oto ći / którym nie przyſądzono pić z kubká tego / przećie piją z niego / á tybyś miał zgołá tego ujść?
Nie ujdźieƺ / ále koniecżnie pić będźieƺ ; bo przez śię przyśięgám / mówi Pán / iż ná ſpuſtoƺenie / ná poháńbienie / ná zburzenie / y ná przeklęſtwo Bocrá przydźie / y wƺyſtkie miáſtá jego będą puſtemi ná wieki.
Słyƺáłem wieść od Páná / że do narodów poſłány jeſt poſeł mówiący : Zgromádźćie śię / á ćiągnijćie przećiwko niemu / wſtáńćież do bitwy.
Bo oto ſpráwię / ábyś był nájlichƺym miedzy národámi / y wzgárdzonym miedzy ludźmi.
Hárdość twojá zdrádźi ćię / y pychá ſercá twego / ty / który mieƺkáƺ w rozśiádlinách ſkálnych / który śię trzymáƺ wyſokich pogórków ; byś też wywyżƺył jáko orzeł gniázdo ſwoje / y z támtąd ćię ſtárgnę / mówi Pán.
Y będźie źiemiá Edomſká puſtynią / y ktokolwiek pójdźie przez nię / zdumieje śię / y świſtáć będźie nád wƺyſtkiemi plágámi jey.
Jáko podwróconá jeſt Sodomá y Gomorá z przyległośćiámi ſwojimi / mówi Pán / ták śię tám nikt nie oſádźi / áni w niej ſyn cżłowiecży mieƺkáć będźie.
Oto ácżkolwiek jáko lew wyſtępuje / y bárdźiej niż nádęćie Jordánu śię podnośi przećiwko przybytkowi Mocnego : wƺákże go nágle wypędzę z tey źiemi / á tego / który jeſt obrány / przełożę nád nią : bo któż mnie jeſt podobny? y kto mi da rok ; á kto jeſt tym páſterzem / któryby śię poſtáwił przećiwko mnie?
Przetoż ſłuchájćie rády Páńſkiej / którą urádźił przećiwko Edomcżykom ; y zámyſłów jego / które umyślił przećiwko obywátelom Temáńſkim ; zájiſte żeć ich wywleką nájmniejśi z tey trzody / zájiſte poburzą ich / y przybytki ich.
Od grzmotu upádku ich wzruƺy śię źiemiá ; głos y krzyk ich ſłyƺeć będą ná morzu Cżerwonem.
Oto jáko orzeł przypádnie y przyleći / á rozćiągnie ſkrzydłá ſwe nád Bocrą / y ſtánie śię ſerce mocárzów z Edom dniá onego / jáko ſerce niewiáſty bolejącey.
Przećiwko Dámáƺkowi. Záwſtydźi śię Emát y Arfád ; bo wieść złą uſłyƺą / y zátrwożą śię / ták / że śię y morze wzruƺy / á nie będźie śię mogło uſpokoić.
Oſłábieje Dámáƺek / udá śię do ućiekánia / á ſtrách go ogárnie ; ućiſki y boleśći ogárną go / jáko rodzącą.
Ale rzeką : Jákożby śię nie miáło oſtáć miáſto ſłáwne / miáſto weſelá mego?
Przetoż upádną młodźieńcy jego ná ulicách jego / á wƺyſcy mężowie wálecżni dniá onego wytráceni będą / mówi Pán zaſtępów.
Y rozniecę ogień w murze Dámáƺku / który ſtráwi páłáce Benádádowe.
Przećiwko Kiedar / y przećiwko króleſtwom Háſor / które wytráćić ma Nábuchodonozor / król Bábilońſki ; ták mówi Pán : Wſtáńćie / ćiągnijćie przećiwko Kiedar / á zburzćie narody wſchodnie.
Námioty ich y trzody ich zábiorą ; opony ich ze wƺyſtkim nácżyniem ich y wielbłądy ich wezmą z ſobą / y záwołáją ná nich : Strách zewƺąd.
Ućiekájćie / rozpierzchnijćie śię prędko / zſtąpćie w głębokośći ná mieƺkánie / obywátele Háſor! mówi Pán ; bo záwárł rádę przećiwko wam Nábuchodonozor / król Bábilońſki / y umyślił przećiwko wam zdrádę.
Wſtáńćie / ćiągnijćie przećiwko národowi ſpokojnemu / mieƺkájącemu bezpiecżnie / mówi Pán : nie ma áni wrót / áni záwór / ſámotni mieƺkáją.
Będą zájiſte wielbłądy ich podáne ná łup / á mnóſtwo dobytku ich ná korzyść ; y rozproƺę ná wƺyſtkie wiátry tych / którzy y w nájoſtátecżniejƺych kątách mieƺkáją / y ze wƺyſtkich ſtron złe ná nich przywiodę / mówi Pán.
Y ſtánie śię Háſor mieƺkániem ſmoków / puſtynią áż ná wieki ; nie oſádźi śię tám nikt / áni mieƺkáć będźie w nim ſyn cżłowiecży.
Słowo Páńſkie / które śię ſtáło do Jeremijáƺá proroká przećiwko Elámcżykom ná pocżątku królowánia Sedekijáƺá / królá Judzkiego / mówiąc :
Ták mówi Pán zaſtępów : Oto ja złámię łuk Elámcżyków / nájwiękƺą śiłę ich ;
A przywiodę przećiwko Elámcżykom cżtery wiátry ze cżterech ſtron świátá / y rozproƺę ich ná wƺyſtkie one wiátry / ták / iż nie będźie narodu / do któregoby śię nie doſtáli wygnáńcy z Elám ;
Y zátrwożę Elámcżyków przed oblicżem nieprzyjáćiół ich / y przed oblicżem tych / którzy ƺukáją duƺy ich ; przywiodę / mówię / ná nich złe / gniew popędliwośći mojey / mówi Pán / á poſyłáć zá nimi będę miecż / dokąd ich nie wyniƺcżę ;
Y poſtáwię ſtolicę moję miedzy Elámcżykámi / á wytrácę z támtąd królá y kśiążąt / mówi Pán.
Wƺákże ſtánie śię / że w oſtátecżne dni przywrócę záś więźniów Elám / mówi Pán.
