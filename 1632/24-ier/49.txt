Przećiwko ſynom Amonowym. Ták mówi Pán : Izáli Izráel nie má ſynów? Izáli żádnego dziedzicá nie má? cżemuż król ich dziedzicżnie opánowáł Gád / á lud jego cżemu mieƺká w miáſtách jego?
Przetoż oto dni idą / mówi Pán / ſpráwię / że uſłyƺą trąbienie wojenne przećiwko Rábbie ſynów Amonowych / y będzie obróconá w kupę rumu / á inne miáſtá jego ogniem ſpálone będą / y pośiędzie Izráel dzierżáwców ſwoich / mówi Pán.
Rozrzewnij śię / Heſebonie! bo Háj ſpuſtoƺone będzie ; krzycżćie / o córki Rábby! przepáƺćie śię worámi / nárzekájćie / á tułájćie śię około płotów ; bo król wáƺ do więzieniá pójdzie / tákże kápłáni jego / y kśiążętá jego ſpołem.
Cóż śię przechwáláƺ dolinámi? Spłynęłá doliná twojá / o córko uporná! którá ufáƺ w ſkárbách twoich / mówiąc : Któż przyćiągnie przećiwko mnie?
Oto Já przywiodę ná ćię ſtrách / mówi Pán / Pán záſtępów od wƺyſtkich / którzy ſą około ćiebie / przez który rozegnáni będziećie jeden od drugiego / á nie będzie / ktoby zebráł tułájących śię.
Wƺákże potem przywiodę záś więźniów ſynów Amonowych / mówi Pán.
Przećiwko Edomcżykom. Ták mówi Pán záſtępów : Izáli niemáƺ więcej mądrośći w Temán? Izáli zginęłá rádá od roztropnych / á wniwecż śię obróćiłá mądrość ich?
Ućiekájćie / obróććie śię / zſtąpćie w głębokość ná mieƺkánie / obywátele Dedán! bo przywiodę zátrácenie ná Ezáwá cżáſu náwiedzeniá jego.
Gdyby ći / którzy zbieráją wino / przyƺli ná ćię / izáliby nie zoſtáwili jákich gron? Gdyby śię wkrádli złodzieje w nocy / ázáżby ƺkodzili więcej nád potrzebę ſwoję?
Lecż Já obnáżę Ezáwá / odkryję ſkrytośći jego / ták / iż śię ukryć nie będzie mógł : zniƺcżone będzie náśienie jego / y bráćiá jego / y ſąśiedzi jego / ták / że zgołá nie będzie / ktoby rzekł :
Zániecháj śierotek twoich / já je żywić będę / á wdowy twoje we mnie ufáć będą.
Ták záiſte mówi Pán : Oto ći / którym nie przyſądzono pić z kubká tego / przećie piją z niego / á tybyś miáł zgołá tego ujść?
Nie ujdzieƺ / ále koniecżnie pić będzieƺ ; bo przez śię przyśięgám / mówi Pán / iż ná ſpuſtoƺenie / ná poháńbienie / ná zburzenie / y ná przeklęſtwo Bocrá przyjdzie / y wƺyſtkie miáſtá jego będą puſtemi ná wieki.
Słyƺáłem wieść od Páná / że do národów poſłány jeſt poſeł mówiący : Zgromádźćie śię / á ćiągnijćie przećiwko niemu / wſtáńćież do bitwy.
Bo oto ſpráwię / ábyś był nájlichƺym między národámi / y wzgárdzonym miedzy ludźmi.
Hárdość twojá zdrádzi ćię / y pychá ſercá twego / ty / który mieƺkáƺ w rozśiádlinách ſkálnych / który śię trzymáƺ wyſokich pogórków ; byś też wywyżƺył jáko orzeł gniázdo ſwoje / y ſtámtąd ćię ſtárgnę / mówi Pán.
Y będzie ziemiá Edomſká puſtynią / y ktokolwiek pójdzie przez nię / zdumieje śię / y świſtáć będzie nád wƺyſtkiemi plágámi jej.
Jáko podwróconá jeſt Sodomá y Gomorá z przyległośćiámi ſwojimi / mówi Pán / ták śię tám nikt nie oſádzi / áni w niej ſyn cżłowiecży mieƺkáć będzie.
Oto ácżkolwiek jáko lew wyſtępuje / y bárdziej niż nádęćie Jordánu śię podnośi przećiwko przybytkowi Mocnego : wƺákże go nágle wypędzę z tej ziemi / á tego / który jeſt obrány / przełożę nád nią : bo któż mnie jeſt podobny? y kto mi dá rok ; á kto jeſt tym páſterzem / któryby śię poſtáwił przećiwko mnie?
Przetoż ſłuchájćie rády Páńſkiej / którą urádził przećiwko Edomcżykom ; y zámyſłów jego / które umyślił przećiwko obywátelom Temáńſkim ; záiſte żeć ich wywleką nájmniejśi z tej trzody / záiſte poburzą ich / y przybytki ich.
Od grzmotu upádku ich wzruƺy śię ziemiá ; głoſ y krzyk ich ſłyƺeć będą ná morzu Cżerwonem.
Oto jáko orzeł przypádnie y przyleći / á rozćiągnie ſkrzydłá ſwe nád Bocrą / y ſtánie śię ſerce mocárzów z Edom dniá onego / jáko ſerce niewiáſty bolejącej.
Przećiwko Dámáƺkowi. Záwſtydzi śię Emát y Arfád ; bo wieść złą uſłyƺą / y zátrwożą śię / ták / że śię y morze wzruƺy / á nie będzie śię mogło uſpokoić.
Oſłábieje Dámáƺek / udá śię do ućiekániá / á ſtrách go ogárnie ; ućiſki y boleśći ogárną go / jáko rodzącą.
Ale rzeką : Jákożby śię nie miáło oſtáć miáſto ſłáwne / miáſto weſelá mego?
Przetoż upádną młodzieńcy jego ná ulicách jego / á wƺyſcy mężowie wálecżni dniá onego wytráceni będą / mówi Pán záſtępów.
Y rozniecę ogień w murze Dámáƺku / który ſtráwi páłáce Benádádowe.
Przećiwko Kiedár / y przećiwko króleſtwom Háſor / które wytráćić má Nábuchodonozor / król Bábilońſki ; ták mówi Pán : Wſtáńćie / ćiągnijćie przećiwko Kiedár / á zburzćie národy wſchodnie.
Námioty ich y trzody ich zábiorą ; opony ich ze wƺyſtkiem nácżyniem ich y wielbłądy ich wezmą z ſobą / y záwołáją ná nich : Strách zewƺąd.
Ućiekájćie / rozpierzchnijćie śię prędko / zſtąpćie w głębokośći ná mieƺkánie / obywátele Háſor! mówi Pán ; bo záwárł rádę przećiwko wám Nábuchodonozor / król Bábilońſki / y umyślił przećiwko wám zdrádę.
Wſtáńćie / ćiągnijćie przećiwko národowi ſpokojnemu / mieƺkájącemu bezpiecżnie / mówi Pán : nie má áni wrót / áni záwór / ſámotni mieƺkáją.
Będą záiſte wielbłądy ich podáne ná łup / á mnóſtwo dobytku ich ná korzyść ; y rozproƺę ná wƺyſtkie wiátry tych / którzy y w nájoſtátecżniejƺych kątách mieƺkáją / y ze wƺyſtkich ſtron złe ná nich przywiodę / mówi Pán.
Y ſtánie śię Háſor mieƺkániem ſmoków / puſtynią áż ná wieki ; nie oſádzi śię tám nikt / áni mieƺkáć będzie w nim ſyn cżłowiecży.
Słowo Páńſkie / które śię ſtáło do Jeremijáƺá proroká przećiwko Elámcżykom ná pocżątku królowániá Sedekijáƺá / królá Judzkiego / mówiąc :
Ták mówi Pán záſtępów : Oto já złámię łuk Elámcżyków / nájwiękƺą śiłę ich ;
A przywiodę przećiwko Elámcżykom cżtery wiátry ze cżterech ſtron świátá / y rozproƺę ich ná wƺyſtkie one wiátry / ták / iż nie będzie národu / do któregoby śię nie doſtáli wygnáńcy z Elám ;
Y zátrwożę Elámcżyków przed oblicżem nieprzyjáćiół ich / y przed oblicżem tych / którzy ƺukáją duƺy ich ; przywiodę / mówię / ná nich złe / gniew popędliwośći mojej / mówi Pán / á poſyłáć zá nimi będę miecż / dokąd ich nie wyniƺcżę ;
Y poſtáwię ſtolicę moję między Elámcżykámi / á wytrácę ſtámtąd królá y kśiążąt / mówi Pán.
Wƺákże ſtánie śię / że w oſtátecżne dni przywrócę záś więźniów Elám / mówi Pán.
