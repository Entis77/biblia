Słowo Páńſkie / które śię ſtáło do Jeremijáƺá proroká przećiwko tym národom.
Przećiwko Egiptowi. Przećiwko wojſku Fáráoná Nechá / królá Egipſkiego / ( które było nád rzeką Eufrátes u Kárchemis / które poráźił Nábuchodonozor / król Bábilońſki ) roku cżwártego Joákimá / ſyná Jozyjáƺowego / królá Judzkiego ;
Gotujćie tárcż y páwężę / á wychodźćie ná wojnę ;
Záprzęgájćie konie / á wśiádájćie jezdni / ſtáńćie w hełmách / wyćierájćie oƺcżepy / oblecżćie śię w páncerze.
Cżemuż tych widzę zátrwożonych / tył podáwájących / á mocárzy ich ſtártych y prędko ućiekájących / ták / że śię áni oglądáją? Strách jeſt zewƺąd / mówi Pán /
Aby nie ućiekł prędki / á nie uƺedł mocárz ; áby śię ná północy o brzeg rzeki Eufrátes otrąćili y upádli.
Któż to jeſt / który jáko rzeká wzbierá? Którego śię wody wzruƺáją jáko rzeki?
Egipt jáko rzeká wzbierá / á jego wody wzruƺáją śię jáko rzeki / y mówi : Poćiągnę / okryję źiemię / wygubię miáſto / y tych / co w nim mieƺkáją.
Poſkocżćie konie / á zágrzmijćie wozy / á niech śię ruƺą y mocárze / Murzynowie / y Putejcżycy noƺący tárcż / y Ludymcżycy / którzy noƺą y ćiągną łuk /
Bo ten dźień pánującego Páná zaſtępów będźie dźień pomſty / áby śię pomśćił nád nieprzyjáćiołmi ſwymi / których miecż pożre / á náſyći śię / y opije śię krwią ich ; bo ofiárá pánującego Páná zaſtępów będźie w źiemi północney u rzeki Eufrátes.
Wſtąp do Gáláád / á nábierz ſoku bálſámowego / pánno / córko Egipſká! Aleć prózno używáƺ wiele lekárſtw ; bo nie będźieƺ ulecżoná.
narodowie ſłyƺą o ſromoćie twojey / á nárzekánie twoje nápełniło źiemię ; bo mocárz ná mocárzá nátárł / ták / że ſpołem obá upádáją.
Słowo / które mówił Pán do Jeremijáƺá / proroká / o tym / że ma przyść Nábuchodonozor / król Bábilońſki / á poráźić źiemię Egipſką.
Oznájmijćie w Egipćie / á rozgłośćie w Migdolu ; opowiádájćie tákże w Nof / y w Táchpánches ; rzecżćie : Poſtój á nágotuj śię ; wƺákże miecż pożre to / co jeſt około ćiebie.
Przecż poráżony jeſt káżdy z mocárzów twojich? Nie może śię oſtáć / przeto / że Pán nátárł nań.
Wieleć będźie tych / którzy poƺwánkują á pádną jeden ná drugiego / y rzeką : Wſtáń / á wróćmy śię do ludu náƺego / y do źiemi urodzeniá náƺego przed oſtrzem miecżá puſtoƺącego.
Tám będą wołáć : Fáráo / król Egipſki / jeſt tylko próżny trzáſk / już mu pominął cżás poſtánowiony.
Jákom żywy Ja / mówi król / Pán zaſtępów imię jego ; że jáko Tábor miedzy górámi / y jáko Kármel przy morzu / ták to przydźie.
Spráw ſobie nacżynie przeprowádzeniá / obywátelko / córko Egipſká! bo Nof puſtynią będźie y ſpuſtoƺeje / y będźie bez obywátelá.
Egipt jeſt jáko piękná jáłowicá ; ále zábićie jey od północy idźie / idźie.
Więc y nájemnicy jego w pośrzodku niego ſą jáko ćiele utucżone / ále y oni tákże obróćiwƺy śię ućiekną ſpołem / nie oſtoją śię ; bo dźień poráżki ich przyƺedł ná nich / cżás náwiedzeniá ich.
Głos jego wynijdźie jáko wężowy ; bo z wojſkiem idą / á z śiekierámi przyjdą nań / jáko ći / co wyrąbują drzewo.
Wyrąbią lás jego / mówi Pán / choć policżony być nie może ; bo śię nád ƺáráńcżę rozmnożyli / y niemáƺ im licżby.
Záwſtydźi śię córká Egipſká ; podáná będźie w rękę ludu północnego.
Pán zaſtępów / Bóg Izráelſki / mówi : Oto Ja náwiedzę ludne miáſto No / tákże Fáráoná y Egipt / y bogów jego / y królów jego / Fáráoná mówię / y tych / którzy w nim ufáją :
Y podám ich w rękę tych / którzy ƺukáją duƺy ich / to jeſt w rękę Nábuchodonozorá / królá Bábilońſkiego / y w rękę ſług jego ; lecż potem mieƺkáć w nim będą / jáko zá dáwnych dni / mówi Pán.
Ale śię ty nie bój / ſługo mój Jákóbie! á nie lękáj śię / o Izráelu! Bo oto Ja ćiebie wybáwię z dáleká / y naśienie twoje z źiemi pojmánia ich ; y wróći śię Jákób / áby odpocżywał y áby miał pokój / á nie będźie / ktoby go przeſtráƺył ;
Ty / mówię / Jákóbie / ſługo mój! nie bój śię / mówi Pán ; bom Ja z tobą. Ucżynię zájiſte koniec wƺyſtkim národom / do których ćię wypędzę ; lecż tobie nie ucżynię końcá / ále ćię miernie káráć będę / á cále ćię bez káránia nie zoſtáwię.
