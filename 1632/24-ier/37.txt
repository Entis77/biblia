Potem królowáł król Sedekijáƺ / ſyn Jozyjáƺowy / miáſto Chonijáƺá / ſyná Joákimowego / którego Nábuchodonozor / król Bábilońſki / w źiemi Judzkiej królem poſtánowił.
Lecż nie był poſłuƺny on / y ſłudzy jego / y lud oney źiemi ſłowom Páńſkim / które mówił przez Jeremijáƺá proroká.
Ale jednák król Sedekijáƺ poſłáł Jechuchálá / ſyná Selemijáƺowego / y Sofonijáƺá / ſyná Máázejáƺowego / kápłáná / do Jeremijáƺá proroká / áby mówili : Módl śię proƺę zá námi Pánu / Bogu náƺemu.
Bo Jeremijáƺ jeƺcże wolno chodźił między ludem / y jeƺcże go było nie wſádzono do więźieniá.
A wojſko Fáráonowe wyćiągnęło było z Egiptu ; ( á uſłyƺáwƺy Cháldejcżycy / którzy byli oblegli Jeruzálem / wieść o tem / odćiągnęli od Jeruzálemu. )
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá proroká / mówiąc :
To mówi Pán / Bóg Izráelſki : Ták powiedzćie królowi Judzkiemu / który was poſłáł do mnie / ábyśćie śię mnie rádźili : Oto wojſko Fáráonowe / które wám przyćiągnęło ná pomoc / wróći śię do źiemi ſwojey do Egiptu.
Y wrócą śię záśię Cháldejcżycy / y będą wálcżyli przećiwko temu miáſtu / y wezmą je / y ſpálą je ogniem.
Ták mówi Pán : Nie zwodźćie duƺ wáƺych / mówiąc : Zápewne odćiągną od nas Cháldejcżycy ; boć nie odćiągną.
Owƺem / choćbyśćie poráźili wƺyſtko wojſko Cháldejcżyków / którzy wálcżą z wámi / á zoſtáliby z nich tylko zránieni / ći z námiotów ſwoich powſtáną / á to miáſto ogniem ſpálą.
Gdy tedy odćiągnęło wojſko Cháldejcżyków od Jeruzálemu przed wojſkiem Fáráonowem /
Wychodźił Jeremijáƺ z Jeruzálemu / áby ƺedł do źiemi Benjáminowej / áby ták uƺedł z pośrodku ludu.
A gdy już był w brámie Benjáminowej / był tám przełożony nád ſtráżą / imieniem Jeryjáƺ / ſyn Selemijáƺá / ſyná Chánánijáƺowego / który pojmáł Jeremijáƺá proroká / mówiąc : Do Cháldejcżyków ty ućiekáƺ.
A Jeremijáƺ odpowiedźiáł : Niepráwdá / nie ućiekám do Cháldejcżyków ; ále go nie chćiáł ſłucháć / owƺem / pojmáł Jeryjáƺ Jeremijáƺá / y przywiódł go do kśiążąt.
Tedy rozgniewáwƺy śię kśiążętá ná Jeremijáƺá / ubili go / y podáli go do więźieniá / do domu Jonátáná piſárzá ; bo z niego byli ucżynili dom więźieniá.
A gdy wƺedł Jeremijáƺ do onego dołu á do táráſu ich / y śiedźiáł tám Jeremijáƺ przez wiele dni :
Tedy poſłáwƺy król Sedekijáƺ / wźiął go / y pytáł go król w domu ſwoim potájemnie / mówiąc : Jeſtże jákie ſłowo od Páná? Y odpowiedźiáł Jeremijáƺ : Jeſt ; przytem rzekł : W rękę królá Bábilońſkiego podány będźieƺ.
Nádto rzekł Jeremijáƺ do królá Sedekijáƺá : Cóżem zgrzeƺył przećiwko tobie / y ſługom twoim / y ludowi twemu / żeśćie mię podáli do tego domu więźieniá?
Y gdźież ſą prorocy wáśi / którzy wám prorokują / mówiąc : Nie przyjdźie król Bábilońſki ná was / áni ná tę źiemię?
Słuchájże teraz / proƺę / królu / pánie mój! niech będźie / proƺę / wáżná prośbá mojá przed tobą : Nie odſyłájże mię do domu Jonátáná piſárzá / ábym tám nie umárł.
A ták rozkázáł król Sedekijáƺ / áby wſádzony był Jeremijáƺ do śieni ſtráży / á iżby mu dáwáno bochenek chlebá ná dźień z ulicy piekárſkiej / pókiby nie był ſtráwiony wƺyſtek chleb w mieśćie. A ták śiedźiáł Jeremijáƺ w śieni ſtráży.
