Potem królował król Sedekijáƺ / ſyn Jozyjáƺowy / miáſto Chonijáƺá / ſyná Joákimowego / którego Nábuchodonozor / król Bábilońſki / w źiemi Judzkiej królem poſtánowił.
Lecż nie był poſłuƺny on / y ſłudzy jego / y lud oney źiemi ſłowom Páńſkim / które mówił przez Jeremijáƺá proroká.
Ale jednák król Sedekijáƺ poſłał Jechuchálá / ſyná Selemijáƺowego / y Sofonijáƺá / ſyná Máázejáƺowego / kápłáná / do Jeremijáƺá proroká / áby mówili : Módl śię proƺę zá námi Pánu / Bogu náƺemu.
Bo Jeremijáƺ jeƺcże wolno chodźił miedzy ludem / y jeƺcże go było nie wſádzono do więźieniá.
A wojſko Fáráonowe wyćiągnęło było z Egiptu ; ( á uſłyƺáwƺy Cháldejcżycy / którzy byli oblegli Jeruzálem / wieść o tym / odćiągnęli od Jeruzálemu. )
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá proroká / mówiąc :
To mówi Pán / Bóg Izráelſki : Ták powiedzćie królowi Judzkiemu / który was poſłał do mnie / ábyśćie śię mnie rádźili : Oto wojſko Fáráonowe / które wam przyćiągnęło ná pomoc / wróći śię do źiemi ſwojey do Egiptu.
Y wrócą śię záśię Cháldejcżycy / y będą wálcżyli przećiwko temu miáſtu / y wezmą je / y ſpálą je ogniem.
Ták mówi Pán : Nie zwodźćie duƺ wáƺych / mówiąc : Zápewne odćiągną od nas Cháldejcżycy ; boć nie odćiągną.
Owƺem / choćbyśćie poráźili wƺyſtko wojſko Cháldejcżyków / którzy wálcżą z wámi / á zoſtáliby z nich tylko zránieni / ći z námiotów ſwojich powſtáną / á to miáſto ogniem ſpálą.
Gdy tedy odćiągnęło wojſko Cháldejcżyków od Jeruzálemu przed wojſkiem Fáráonowem /
Wychodźił Jeremijáƺ z Jeruzálemu / áby ƺedł do źiemi Benjáminowej / áby ták uƺedł z pośrzodku ludu.
A gdy już był w bramie Benjáminowej / był tám przełożony nád ſtráżą / imieniem Jeryjáƺ / ſyn Selemijáƺá / ſyná Chánánijáƺowego / który pojmał Jeremijáƺá proroká / mówiąc : Do Cháldejcżyków ty ućiekáƺ.
A Jeremijáƺ odpowiedźiał : Niepráwdá / nie ućiekám do Cháldejcżyków ; ále go nie chćiał ſłucháć / owƺem / pojmał Jeryjáƺ Jeremijáƺá / y przywiódł go do kśiążąt.
Tedy rozgniewáwƺy śię kśiążętá ná Jeremijáƺá / ubili go / y podáli go do więźieniá / do domu Jonátáná piſárzá ; bo z niego byli ucżynili dom więźieniá.
A gdy wƺedł Jeremijáƺ do onego dołu á do táráſu ich / y śiedźiał tám Jeremijáƺ przez wiele dni :
Tedy poſłáwƺy król Sedekijáƺ / wźiął go / y pytał go król w domu ſwoim potájemnie / mówiąc : Jeſtże jákie ſłowo od Páná? Y odpowiedźiał Jeremijáƺ : Jeſt ; przytem rzekł : W rękę królá Bábilońſkiego podány będźieƺ.
Nádto rzekł Jeremijáƺ do królá Sedekijáƺá : Cóżem zgrzeƺył przećiwko tobie / y ſługom twoim / y ludowi twemu / żeśćie mię podáli do tego domu więźieniá?
Y gdźież ſą prorocy wáśi / którzy wam prorokują / mówiąc : Nie przydźie król Bábilońſki ná was / áni ná tę źiemię?
Słuchájże teraz / proƺę / królu / pánie mój! niech będźie / proƺę / wáżná prośbá mojá przed tobą : Nie odſyłájże mię do domu Jonátáná piſárzá / ábym tám nie umárł.
A ták rozkázał król Sedekijáƺ / áby wſádzony był Jeremijáƺ do śieni ſtráży / á iżby mu dáwano bochenek chlebá ná dźień z ulicy piekárſkiej / pókiby nie był ſtráwiony wƺyſtek chleb w mieśćie. A ták śiedźiał Jeremijáƺ w śieni ſtráży.
