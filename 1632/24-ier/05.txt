Obchodźćie ulice Jeruzálemſkie / á upátrujćie teráz / y obácżćie / á ƺukájćie po ulicách jego / jeźli znájdziećie mężá / jeźli kto jeſt / coby cżynił ſąd y ƺukáł práwdy / á przepuƺcżę mu.
Ale choć mówią : Jáko żyje Pán / tedy przećię krzywo przyśięgáją.
O Pánie! izáli ocży twoje nie pátrzą ná práwdę? Bijeƺ ich / ále ich nie boli ; wniwecż ich obrácáƺ / ále nie chcą przyjąć kárániá ; zátwárdzili oblicżá ſwe nád opokę / nie chcą śię náwróćić.
Tedym Já rzekł : Podobno ći nędzni ſą / głupio ſobie pocżynáją ; bo nie ſą powiádomi drogi Páńſkiej / y ſądu Bogá ſwego.
Pójdę do celniejƺych / y będę mówił do nich ; bo oni ſą powiádomi drogi Páńſkiej / y ſądu Bogá ſwego ; ále y ći weſpół połámáli járzmo / potárgáli związki.
Przetoż ich pobije lew z láſu / wilk wiecżorny wygubi ich / lámpárt cżyháć będzie u miáſt ich. Ktokolwiek wyjdzie z nich / rozƺárpány będzie ; bo śię rozmnożyły przeſtępſtwá ich / y zmogły śię odwróceniá ich.
Jeſtże co / dlácżegobym ći miáł przepuśćić? Synowie twoi opuśćili mię / á przyśięgáją przez onych / którzy nie ſą bogámi. Jákom ich jedno nákármił / záráz cudzołożą / á do domu wƺetecżnicy hurmem śię wálą.
Ráno wſtáwájąc ſą jáko konie wytucżone / káżdy z nich rży do żony bliźniego ſwego.
Izáli dlátego náwiedzić ich nie mám? mówi Pán ; izáli śię nád tákim národem nie má mśćić duƺá mojá?
Wſtąpćie ná mury jego / á rozwálćie je / wƺákże ich do gruntu nie znośćie ; znieśćie filárzyki murów jego / gdyż nie ſą Páńſkie.
Wielce záiſte wyſtąpił przećiwko mnie dom Izráelſki y dom Judzki / mówi Pán.
Zádáli kłámſtwo Pánu / y rzekli : Nie ták / nie przyjdzieć ná náſ nic złego / á miecżá y głodu nie doznámy.
A ći prorocy pominą z wiátrem / á żádnego ſłowá Bożego niemáƺ u nich ; y owƺem ták śię im ſámym ſtánie.
Przetoż ták mówi Pán / Bóg záſtępów : Poniewáżeśćie to mówili / oto Já kłádę ſłowá moje w uſtá twoje zá ogień / á lud ten zá drwá / y pożre ich.
Oto Já przywiodę ná wáſ náród z dáleká / o domie Izráelſki! mówi Pán / náród mocny / náród ſtárodáwny / náród / którego języká umieć nie będzieƺ / áni zrozumieƺ / co mówi.
Którego ſájdák jáko grób otwárty / wƺyſcy ſą mężni.
Y zjedzą urodzáj twój / y chleb twój : pożrą ſynów twoich y córki twoje ; poje trzody twoje y woły twoje ; poje winną máćicę twoję / y figi twoje / á miáſtá twoje obronne / w których ty ufáƺ / miecżem znędzi.
A wƺákże y w one dni / mówi Pán / końcá z wámi nie ucżynię.
Abowiem gdy rzecżećie : Przecżże nám Pán / Bóg náƺ / to wƺyſtko cżyni? Tedy im odpowieƺ : Jákośćie mię opuśćili / á ſłużyli bogom cudzym w ziemi wáƺej / ták ſłużyć będziećie cudzoziemcom w ziemi nie wáƺej.
Oznájmijćie to domowi Jákóbowemu / á rozgłośćie w Judzie / mówiąc :
Słuchájćież teráz tego / ludu głupi! który niemáƺ ſercá / który ocży májąc / á nie widziƺ / który uƺy májąc / á nie ſłyƺyƺ.
Y nie będziećież śię mnie báli? mówi Pán ; á przed oblicżem mojem nie będziećież śię lękáli? którym położył morzu piáſek zá gránicę uſtáwą wiecżną / á nie przeſtąpi jej. Choć śię wzruƺą / wƺákże nie przemogą ; choć śię wzburzą wáły jego / wƺákże nie przeſkocżą go.
Ale ten lud má ſerce oćiążáłe y odporne ; odſtąpili odemnie y odeƺli ;
Ani rzekli w ſercu ſwem : Bójmy śię już Páná / Bogá náƺego / który dáje deƺcż y w jeśieni y ná wioſnę cżáſu ſwego / który tygodni pewnych y żniwá náƺego przeſtrzegá.
Ale niepráwośći wáƺe odwróćiły to / á grzechy wáƺe záhámowáły to dobro od wáſ.
Bo śię znájdują w ludu moim niezbożnicy / którzy cżyháją jáko łowcy / rozćiągáją śieći / záſtáwiáją śidłá / á łápią ludzi.
Jáko klátká pełná ptáków / ták domy ich pełne ſą zdrády ; przetoż śię wzmogli y zbogáćili.
Roztyli / lśnią śię / y innych w złośćiách przewyżƺáją ; ſpráwy nie ſądzą / áni ſpráwy śierotki ; wƺákże śię im ƺcżęśći / choćiáż ſpráwy ubogiego nie rozſądzili.
Izáli dlátego nie náwiedzę ich? mówi Pán ; izáli śię nád národem tákowym nie będzie mśćić duƺá mojá?
Rzecż dziwná y ſrogá dzieje śię w tej ziemi :
Prorocy kłámliwie prorokują / á kápłáni pánują przez ręce ich / á lud mój kochá śię w tem ; cżegożbyśćie ná oſtátek nie ucżynili?
