Obchodźćie ulice Jeruzálemſkie / á upátrujćie teraz / y obácżćie / á ƺukájćie po ulicách jego / jeſli znájdźiećie mężá / jeſli kto jeſt / coby cżynił ſąd y ƺukał prawdy / á przepuƺcżę mu.
Ale choć mówią : Jáko żyje Pán / tedy przećię krzywo przyśięgáją.
O Pánie! izali ocży twoje nie pátrzą ná práwdę? Bijeƺ ich / ále ich nie boli ; wniwecż ich obrácáƺ / ále nie chcą przyjąć káránia ; zátwárdźili oblicżá ſwe nád opokę / nie chcą śię náwróćić.
Tedym Ja rzekł : Podobno ći nędzni ſą / głupio ſobie pocżynáją ; bo nie ſą powiádomi drogi Páńſkiej / y ſądu Bogá ſwego.
Pójdę do celniejƺych / y będę mówił do nich ; bo oni ſą powiádomi drogi Páńſkiej / y ſądu Bogá ſwego ; ále y ći weſpół połámáli járzmo / potárgáli związki.
Przetoż ich pobije lew z láſu / wilk wiecżorny wygubi ich / lámpárt cżyháć będźie u miaſt ich. Ktokolwiek wyjdźie z nich / rozƺárpány będźie ; bo śię rozmnożyły przeſtępſtwá ich / y zmogły śię odwróceniá ich.
Jeſtże co / dlácżegobym ći miał przepuśćić? Synowie twoi opuśćili mię / á przyśięgáją przez onych / którzy nie ſą bogámi. Jákom ich jedno nákármił / záráz cudzołożą / á do domu wƺetecżnicy hurmem śię wálą.
Ráno wſtáwájąc ſą jáko konie wytucżone / káżdy z nich rży do żony bliźniego ſwego.
Izali dla tego náwiedźić ich nie mam? mówi Pán ; izali śię nád tákim národem nie ma mśćić duƺá mojá?
Wſtąpćie ná mury jego / á rozwálćie je / wƺákże ich do gruntu nie znośćie ; znieśćie filárzyki murów jego / gdyż nie ſą Páńſkie.
Wielce záiſte wyſtąpił przećiwko mnie dom Izráelſki y dom Judzki / mówi Pán.
Zádáli kłámſtwo Pánu / y rzekli : Nie ták / nie przyjdźieć ná nas nic złego / á miecżá y głodu nie doznámy.
A ći prorocy pominą z wiátrem / á żádnego ſłowá Bożego niemáƺ u nich ; y owƺem ták śię im ſámym ſtánie.
Przetoż ták mówi Pán / Bóg zaſtępów : Poniewáżeśćie to mówili / oto Ja kłádę ſłowá moje w uſtá twoje zá ogień / á lud ten zá drwá / y pożre ich.
Oto Ja przywiodę ná was náród z dáleká / o domie Izráelſki! mówi Pán / náród mocny / náród ſtárodáwny / náród / którego języká umieć nie będźieƺ / áni zrozumieƺ / co mówi.
Którego ſájdák jáko grób otwárty / wƺyſcy ſą mężni.
Y zjedzą urodzáj twój / y chleb twój : pożrą ſynów twojich y córki twoje ; poje trzody twoje y woły twoje ; poje winną máćicę twoję / y figi twoje / á miáſtá twoje obronne / w których ty ufáƺ / miecżem znędźi.
A wƺákże y w one dni / mówi Pán / końcá z wámi nie ucżynię.
Abowiem gdy rzecżećie : Przecżże nam Pán / Bóg náƺ / to wƺyſtko cżyni? Tedy im odpowieƺ : Jákośćie mię opuśćili / á ſłużyli bogom cudzym w źiemi wáƺej / ták ſłużyć będźiećie cudzoźiemcom w źiemi nie wáƺej.
Oznájmijćie to domowi Jákóbowemu / á rozgłośćie w Judźie / mówiąc :
Słuchájćież teraz tego / ludu głupi! który niemáƺ ſercá / który ocży májąc / á nie widźiƺ / który uƺy májąc / á nie ſłyƺyƺ.
Y nie będźiećież śię mnie báli? mówi Pán ; á przed oblicżem mojim nie będźiećież śię lękáli? którym położył morzu piáſek zá gránicę uſtáwą wiecżną / á nie przeſtąpi jey. Choć śię wzruƺą / wƺákże nie przemogą ; choć śię wzburzą wáły jego / wƺákże nie przeſkocżą go.
Ale ten lud ma ſerce oćiążáłe y odporne ; odſtąpili odemnie y odeƺli ;
Ani rzekli w ſercu ſwem : Bójmy śię już Páná / Bogá náƺego / który dáje deƺcż y w jeśieni y ná wioſnę cżáſu ſwego / który tygodni pewnych y żniwá náƺego przeſtrzegá.
Ale niepráwośći wáƺe odwróćiły to / á grzechy wáƺe záhámowáły to dobro od was.
Bo śię znájdują w ludu moim niezbożnicy / którzy cżyháją jáko łowcy / rozćiągáją śieći / záſtáwiáją śidłá / á łápią ludźi.
Jáko klátká pełna ptáków / ták domy ich pełne ſą zdrády ; przetoż śię wzmogli y zbogáćili.
Roztyli / lśnią śię / y innych w złośćiách przewyżƺáją ; ſpráwy nie ſądzą / áni ſpráwy śierotki ; wƺákże śię im ƺcżęśći / choćiáż ſpráwy ubogiego nie rozſądźili.
Izali dla tego nie náwiedzę ich? mówi Pán ; izali śię nád národem tákowym nie będźie mśćić duƺá mojá?
Rzecż dźiwná y ſrogá dźieje śię w tey źiemi :
Prorocy kłámliwie prorokują / á kápłáni pánują przez ręce ich / á lud mój kocha śię w tym ; cżegożbyśćie ná oſtátek nie ucżynili?
