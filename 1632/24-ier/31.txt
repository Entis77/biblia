Cżáſu onego / mówi Pán / będę Bogiem wƺyſtkim rodzájom Izráelſkim / á oni będą ludem moim.
Ták mówi Pán : Ználázł łáſkę ná puƺcży lud / który uƺedł miecżá / gdym chodźił przed nim / ábym odpocżnienie ucżynił Izráelowi.
Rzecżeƺli : Zdáwná mi śię Pán ukázywał. Y owƺem / miłośćią wiecżną umiłowáłem ćię / dlátegoć uſtáwicżnie miłośierdźie pokázuję ;
Jeƺcże ćię zbuduję / á zbudowáná będźieƺ / pánno Izráelſká! jeƺcże śię weſelić będźieƺ przy bębnách ſwojich / á wychodźić z hufem pląſájących ;
Jeƺcże będźieƺ ſádźiłá winnice ná górách Sámáryjſkich ; á ƺcżepiący ƺcżepić będą y jeść będą.
Bo náſtánie dźień / którego ſtróżowie wołáć będą ná górze Efráimowej : Wſtáńćie / á wſtąpmy ná Syon do Páná / Bogá ſwego.
Ták zájiſte mówi Pán : Śpiewájćie Jákóbowi o rzecżách weſołych / á wykrzykájćie jáwnie przed tymi národámi ; wydáwájćie głos / chwałę oddáwájćie á mówćie : Pánie! wybáw oſtátek ludu twego Izráelſkiego.
Oto Ja przywiodę ich z źiemi północney / á zbiorę ich ze wƺyſtkich ſtron źiemi ; z nimi poſpołu ślepego y chromego / brzemienną y rodzącą : tu śię gromádą wielką náwrócą.
Przywiodę ich záśię z płácżem y z modlitwámi idących / á powiodę ich podle potoków wód drogą proſtą / ná którejby śię nie potknęli ; bom śię ſtał Izráelowi ojcem / á Efráim jeſt pierworodnym moim.
Słuchájćie ſłowá Páńſkiego / o narody! á opowiádájćie je ná wyſpách dálekich / y mówćie : Ten / który rozproƺył Izráelá / zgromádźi go / á ſtrzedz go będźie / jáko páſterz trzody ſwojey.
Bo wykupił Pán Jákóbá ; przeto wybáwi go z ręki tego / który jeſt mocniejƺy nádeń.
Y przyjdą á śpiewáć będą ná wyſokośći Syonu / y zbieżą śię do dobrotliwośći Páńſkiej ze zbożem y z winem / y z oliwą y z jágniętámi / y z ćielętámi / á duƺá ich podobná będźie ogrodowi wilgotnemu / á nie będźie śię więcey ſmućiłá.
Tedy śię weſelić będźie pánná z pląſániem / tákże młodźieńcy y ſtárcy ſpołem ; ábowiem kwilenie ich obrócę w rádość / á poćieƺę ich / y rozweſelę ich po ſmutku ich ;
Y opoję duƺę kápłanów tłuſtośćią / á lud mój dobroćią moją náſyći śię / mówi Pán.
Ták mówi Pán : Głos w Rámá ſłyƺány jeſt / nárzekánie y płácż bárzo gorzki ; Ráchel płácżącá ſynów ſwojich nie dáłá śię poćieƺyć po ſynách ſwojich / przeto / że ich niemáƺ.
Ták mówi Pán : Záwśćiągnij głos ſwój od płácżu / á ocży ſwe od łez ; bo będźieƺ miáłá zápłátę zá prácę ſwoję / mówi Pán / że śię náwrócą z źiemi nieprzyjáćielſkiej.
Jeſt mówię nádźiejá / że śię potem náwrócą / mówi Pán / ſynowie twoi do kráiny ſwojey.
Wpráwdźie ſłyƺę Efráimá / że ſobie utyſkuje / mówiąc : Pokáráłeś mię / ábym był pokárány jáko ćielec nieokrócony. Náwróć mię / ábym był náwrócony ; tyś zájiſte Pánie! Bóg mój.
Bo po náwróceniu mojim pokutowáć będą ; á gdy ſámego śiebie poznam / uderzę śię w biodro ; wſtydzę śię / owƺem y zápáłám śię / że odnoƺę háńbę dźiećińſtwá ſwego.
Izali ſynem moim miłym nie jeſt Efráim? Izali nie jeſt dźiećięćiem rozkoƺnem? Bo od onego cżáſu / jákom mówił przećiwko niemu / przećież nań uſtáwicżnie wſpominam ; dla tego poruƺáją śię wnętrznośći moje nád nim / zájiſte zlituję śię nád nim / mówi Pán.
Wyſtáw ſobie pámiątki ; nákłádź ſobie gromád kámieni ; pámiętáj ná ten gośćiniec / y ná drogę / którąś chodźiłá ; náwróć śię / pánno Izráelſká! náwróć śię do tych miaſt ſwojich.
Dokądże śię tułáć będźieƺ / córko odporná? Bo ucżyni Pán rzecż nową ná źiemi : niewiáſtá ogárnie mężá.
Ták mówi Pán zaſtępów / Bóg Izráelſki : Jeƺcże mówić będę ſłowo to w źiemi Judzkiej / y w miáſtách jey / gdy przywiodę więźniów ich : Niech ćię błogoſłáwi Pán / o mieƺkánie ſpráwiedliwośći! o góro świętobliwośći!
Abowiem oſádzáć śię będą w źiemi Judzkiej we wƺyſtkich miáſtách jego ſpołem orácże / y ći / którzy chodzą zá ſtádem.
Nápoję zájiſte duƺę ſprácowáną / á wƺelką duƺę ſmutną náſycę.
Wtemem ocućił / y ſpojrzáłem / á ſen mój był mi wdźięcżny.
Oto dni idą / mówi Pán / w których pośieję dom Izráelſki y dom Judzki náśieniem cżłowiecżem y náśieniem bydlęcem ;
A jákom śię ſtárał / ábym ich wykorzenił / y burzył / y káźił / y gubił / y trápił : ták śię ſtáráć będę / ábym ich pobudował y rozſádźił / mówi Pán.
Zá onych dni nie będą więcey mówić : Ojcowie jedli groná ćierpkie / á ſynów zęby śćierpnęły ;
Owƺem / rácżey rzeką : káżdy dla niepráwośći ſwojey umrze ; káżdego cżłowieká / któryby jádł groná ćierpkie / śćierpną zęby jego.
Oto dni idą / mówi Pán / których ucżynię z domem Izráelſkim y z domem Judzkim przymierze nowe ;
Nie tákie przymierze / jákiem ucżynił z ojcámi ich w on dźień / któregom ich ujął zá rękę ich / ábym ich przywiódł z źiemi Egipſkiej ; ábowiem oni przymierze moje wzruƺyli / choćiáżem Ja był máłżonkiem ich / mówi Pán.
Ale to jeſt przymierze / które poſtánowię z domem Izráelſkim po tych dniách / mówi Pán : Dam zakon mój do wnętrznośći ich / á ná ſercu ich nápiƺę go / y będę Bogiem ich / á oni będą ludem moim.
Y nie będźie więcey ucżył żáden bliźniego ſwego / y żáden brátá ſwego / mówiąc : Poznájćie Páná ; bo mię oni wƺyſcy poznáją / od nájmniejƺego z nich áż do nájwiękƺego z nich / mówi Pán ; bo miłośćiw będę niepráwośćiom ich / á grzechów ich nie wſpomnę więcey.
Ták mówi Pán / który dáje ſłońce ná świátłość we dnie / poſtánowienie mieśiącá y gwiázd ná świátłość w nocy ; który rozdźielá morze / á hucżą náwáłnośći jego ; Pán zaſtępów imię jego.
Jeſli odſtąpią te uſtáwy od oblicża mego / mówi Pán / tedyć y naśienie Izráelſkie przeſtánie być národem przed oblicżem mojim po wƺyſtkie dni.
Ták mówi Pán : Jeſli mogą być rozmierzone niebioſá z góry / á dośćignione grunty źiemi ná dole / tedyć y Ja cále odrzucę naśienie Izráelſkie dla tego wƺyſtkiego / co ucżynili / mówi Pán.
Oto idą dni ( mówi Pán / ) których będźie zbudowáne to miáſto Pánu od wieży Chánáneel áż do bramy nárożney ;
A pójdźie jeƺcże ƺnur pomiáru ná przećiwko niej ku págórkowi Gáreb / á udá śię ku Goá.
Y wƺyſtká doliná trupów y popiołu / y to wƺyſtko pole áż do potoku Cedron / áż do węgłá bramy końſkiej wſchodniej / poświęcone będą Pánu ; nie będźie wykorzenione áni zepſute więcey ná wieki.
