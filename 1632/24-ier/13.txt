Ták rzekł Pán do mnie : Idź / á kup ſobie páſ lniány / á opáƺ nim biodrá ſwoje ; ále do wody nie kłádź go.
Kupiłem tedy páſ według rozkázániá Páńſkiego / y opáſáłem biodrá moje.
Potem ſtáło śię ſłowo Páńſkie do mnie powtóre / mówiąc :
Weźmij ten páſ / któryś kupił / który jeſt ná biodrách twoich / á wſtáwƺy idź do Eufráteſá / á ſkryj go tám w dziurę ſkálną.
Y ƺedłem á ſkryłem go u Eufráteſá / jáko mi był Pán rozkázáł.
A po wyjśćiu wielu dni rzekł Pán do mnie : Wſtáń / idź do Eufráteſá / á weźmij ſtámtąd on páſ / którym ći tám rozkázáł ſkryć.
Szedłem tedy do Eufráteſá / á wykopáwƺy wziąłem on páſ z miejſcá onego / gdziem go był ſkrył / á oto ſkáżony był on páſ / ták / iż śię nicżemu nie godził.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Ták mówi Pán : Ták ſkáżę pychę Judzką y wielką pychę Jeruzálemſką /
Ludu tego bárdzo złego / który śię zbrániá ſłucháć ſłów moich / który chodzi w uporze ſercá ſwego / y chodzi zá bogámi obcymi / ſłużąc im y kłániájąc śię im ; y będzie podobien temu páſowi / który śię nicżemu nie godzi.
Bo jáko páſ przylegá do biódr mężá / tákiem Já był przypoił do śiebie wƺyſtek dom Izráelſki / y wƺyſtek dom Judzki / mówi Pán / áby byli ludem moim / á to ku ſłáwie y ku chwále / y ku ozdobie ; ále nie byli poſłuƺni.
Przetoż rzecż im to ſłowo. Ták mówi Pán / Bóg Izráelſki : Wƺelkie nácżynie winne bywá nápełnione winem ; á gdy rzeką : Wiemyć to dobrze / że wƺelkie nácżynie winne bywá nápełnione winem /
Tedy im rzecżeƺ : Ták mówi Pán : Oto Já nápełnię wƺyſtkich obywáteli tej ziemi / y królów / którzy śiedzą miáſto Dawidá ná ſtolicy jego / y kápłánów y proroków / tákże y wƺyſtkich obywáteli Jeruzálemſkich pijáńſtwem ;
Y rozráżę jednego o drugiego / jáko ojców ták y ſynów / mówi Pán ; nie przepuƺcżę / nie ſfolguję / áni śię zmiłuję / ábym ich ſkázić nie miáł.
Słuchájćież / á pojmujćie uƺymá / nie podnośćie śię ; boć Pán mówi.
Dájćie Pánu / Bogu ſwemu / chwáłę / pierwej niżby ćiemnośći przywiódł / á pierwej niżby śię obráziły nogi wáƺe o góry ćiemne ; y cżekálibyśćie świátłośći / ále Bóg obróćiłby je w ćień śmierći y przemieniłby je w záćmienie.
A jeźliż tego ſłucháć nie będziećie / w ſkrytośćiách płákáć będzie duƺá mojá dlá pychy wáƺej / á płácżąc płákáć będzie / y wyleje oko moje łzy / bo pojmáná będzie trzodá Páńſká.
Mów królowi y królowej : Upokorzćie śię / uśiądźćie ná ziemi ; bo ſpádłá z głowy wáƺej koroná chwáły wáƺej.
Miáſtá ná południe záwárte będą / ták / że nie będzie / ktoby je otworzył ; przenieśiony będzie wƺyſtek Judá / przenieśiony będzie do ƺcżętu.
Podnieśćie ocży wáƺe / á obácżćie tych / którzy idą z północy. Gdzie jeſt tá trzodá / którejć śię zwierzono? gdzie jeſt ſtádo chwáły twojej?
Cóż rzecżeƺ / gdy ćię ( nieprzyjáćiel ) náwiedzi? Boś ich ty náucżyłá / áby byli nád tobą kśiążętámi przednimi ; izáli ćię boleśći nie ogárną / jáko niewiáſtę rodzącą?
Mówiƺli w ſercu ſwojim : Przecżżeby to przypáść miáło ná mię? Dlá mnóſtwá niepráwośći twojej odkryte będą podołki twoje / gwáłtem obnáżone będą pięty twoje.
Azáż może murzyn odmienić ſkórę ſwoję / álbo lámpárt pſtroćiny ſwoje? tákże y wy / ázáż będziećie mogli dobrze cżynić / náucżywƺy śię źle cżynić?
Przetoż rozproƺę ich jáko źdźbło / które śię rozlátuje od wiátru z puſtyni.
Tenći będzie loſ twój / y dziáł odmierzony tobie odemnie / mówi Pán / przeto / żeś mię zápomniáłá / á ufáłáś w kłámſtwie.
A ták y Já odkryję podołek twój áż ná twárz twoję / áby śię okázáłá ſromotá twojá.
Widziáłem cudzołóſtwá twoje y poryzánie twoje / ſproſność wƺetecżeńſtwá twego ná págórkách / y ná polu ; widziáłem / mówię / obrzydliwośći twoje. Biádá tobie / Jeruzálemie! y pókiż śię nie ocżyśćiƺ? kiedyż to wżdy będzie?
