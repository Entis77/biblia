Ták rzekł Pán do mnie : Idź / á kup ſobie pás lniány / á opáƺ nim biodrá ſwoje ; ále do wody nie kłádź go.
Kupiłem tedy pás według rozkázánia Páńſkiego / y opáſáłem biodrá moje.
Potem ſtáło śię ſłowo Páńſkie do mnie powtóre / mówiąc :
Weźmij ten pás / któryś kupił / który jeſt ná biodrách twojich / á wſtáwƺy idź do Eufráteſá / á ſkryj go tám w dźiurę ſkálną.
Y ƺedłem á ſkryłem go u Eufráteſá / jáko mi był Pán rozkázał.
A po wyjśćiu wielu dni rzekł Pán do mnie : Wſtáń / idź do Eufráteſá / á weźmij ſtámtąd on pás / którym ći tám rozkázał ſkryć.
Szedłem tedy do Eufráteſá / á wykopáwƺy wźiąłem on pás z miejſcá onego / gdźiem go był ſkrył / á oto ſkáżony był on pás / ták / iż śię nicżemu nie godźił.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Ták mówi Pán : Ták ſkáżę pychę Judzką y wielką pychę Jeruzálemſką /
Ludu tego bárzo złego / który śię zbránia ſłucháć ſłów mojich / który chodźi w uporze ſercá ſwego / y chodźi zá bogámi obcymi / ſłużąc im y kłániájąc śię im ; y będźie podobien temu páſowi / który śię nicżemu nie godźi.
Bo jáko pás przylegá do biódr mężá / tákiem Ja był przypoił do śiebie wƺyſtek dom Izráelſki / y wƺyſtek dom Judzki / mówi Pán / áby byli ludem moim / á to ku ſłáwie y ku chwále / y ku ozdobie ; ále nie byli poſłuƺni.
Przetoż rzecż im to ſłowo. Ták mówi Pán / Bóg Izráelſki : Wƺelkie nacżynie winne bywá nápełnione winem ; á gdy rzeką : Wiemyć to dobrze / że wƺelkie nacżynie winne bywá nápełnione winem /
Tedy im rzecżeƺ : Ták mówi Pán : Oto Ja nápełnię wƺyſtkich obywáteli tey źiemi / y królów / którzy śiedzą miáſto Dawidá ná ſtolicy jego / y kápłánów y proroków / tákże y wƺyſtkich obywáteli Jeruzálemſkich pijáńſtwem ;
Y rozráżę jednego o drugiego / jáko ojców ták y ſynów / mówi Pán ; nie przepuƺcżę / nie ſfolguję / áni śię zmiłuję / ábym ich ſkáźić nie miał.
Słuchájćież / á pojmujćie uƺymá / nie podnośćie śię ; boć Pán mówi.
Dájćie Pánu / Bogu ſwemu / chwałę / pierwej niżby ćiemnośći przywiódł / á pierwej niżby śię obráźiły nogi wáƺe o góry ćiemne ; y cżekálibyśćie świátłośći / ále Bóg obróćiłby je w ćień śmierći y przemieniłby je w záćmienie.
A jeſliż tego ſłucháć nie będźiećie / w ſkrytośćiách płákáć będźie duƺá mojá dla pychy wáƺej / á płácżąc płákáć będźie / y wyleje oko moje łzy / bo pojmáná będźie trzodá Páńſká.
Mów królowi y królowej : Upokorzćie śię / uśiądźćie ná źiemi ; bo ſpádłá z głowy wáƺej koroná chwały wáƺej.
Miáſtá ná południe záwárte będą / ták / że nie będźie / ktoby je otworzył ; przenieśiony będźie wƺyſtek Judá / przenieśiony będźie do ƺcżętu.
Podnieśćie ocży wáƺe / á obácżćie tych / którzy idą z północy. Gdźie jeſt tá trzodá / którejć śię zwierzono? gdźie jeſt ſtádo chwały twojey?
Cóż rzecżeƺ / gdy ćię ( nieprzyjáćiel ) náwiedźi? Boś ich ty náucżyłá / áby byli nád tobą kśiążętámi przednimi ; izali ćię boleśći nie ogárną / jáko niewiáſtę rodzącą?
Mówiƺli w ſercu ſwojim : Przecżżeby to przypáść miáło ná mię? Dla mnóſtwá niepráwośći twojey odkryte będą podołki twoje / gwáłtem obnáżone będą pięty twoje.
Azaż może murzyn odmienić ſkórę ſwoję / álbo lámpárt pſtroćiny ſwoje? tákże y wy / ázaż będźiećie mogli dobrze cżynić / náucżywƺy śię źle cżynić?
Przetoż rozproƺę ich jáko źdźbło / które śię rozlátuje od wiátru z puſtyni.
Tenći będźie los twój / y dźiał odmierzony tobie odemnie / mówi Pán / przeto / żeś mię zápomniáłá / á ufáłáś w kłámſtwie.
A ták y Ja odkryję podołek twój áż ná twárz twoję / áby śię okázáłá ſromotá twojá.
Widźiáłem cudzołóſtwá twoje y poryzánie twoje / ſproſność wƺetecżeńſtwá twego ná págórkách / y ná polu ; widźiáłem / mówię / obrzydliwośći twoje. Biádá tobie / Jeruzálemie! y pókiż śię nie ocżyśćiƺ? kiedyż to wżdy będźie?
