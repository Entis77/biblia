Słowo / które śię ſtáło do Jeremijáƺá od Páná mówiąc :
Wſtáń / á wſtąp do domu gárncárzowego / á tám ſpráwię / że uſłyƺyƺ ſłowá moje.
Y wſtąpiłem do domu gárncárzowego / á oto on robił robotę ná kręgu.
A gdy śię zepſuło nacżynie w ręce gárncárzowej / które on cżynił z gliny / tedy záś ucżynił z niej nacżynie inƺe / jáko śię mu nájlepiej zdáło ucżynić.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Izálibym ták nie mógł z wámi poſtąpić / jáko ten gárncárz / o domie Izráelſki? mówi Pán. Oto jáko gliná w ręce gárncárzowej / tákeśćie wy w ręce mojey / o domie Izráelſki!
Jeſlibym rzekł nágle przećiwko národowi / y przećiwko króleſtwu / że je wykorzenię / y zepſuję / y wygubię ;
Wƺákże jeſliby śię odwróćił on naród od złośći ſwojey / przećiw któremum mówił : y Jábym żáłował tego złego / którem mu umyślił ucżynić.
Záśię / jeſlibym rzekł nágle o národźie y o króleſtwie / że je pobuduję y wƺcżepię ;
Wƺákże jeſliby cżynił / co złego jeſt przed ocżymá memi / nie ſłuchájąc głoſu mego : y Jábym żáłował tego dobrodźiejſtwá / którem mu obiecał ucżynić.
A przetoż rzecż teraz do mężów Judzkich / y do obywáteli Jeruzálemſkich / mówiąc : Ták mówi Pán : Oto Ja gotuję ná was złą rzecż / y myślę coś przećiwko wam. Náwróććież śię już káżdy od złej drogi ſwojey / á popráwćie káżdy dróg wáƺych / y ſpráw wáƺych.
Ale oni rzekli : Nic z tego ; bo zá myślámi náƺemi pójdźiemy / á káżdy upór ſercá ſwojego złego cżynić będźiemy.
Przetoż ták mówi Pán : Pytájćie teraz miedzy pogánámi / któż ſłyƺał co tákowego? Sproſność wielką popełniłá pánná Izráelſká.
Izali kto opuśći polá moje dla ſkał y dla śniegu ná Libánie? Izali kto opuśći wody ćiekące dla wody bárzo źimney?
Ale lud mój zápomniáwƺy ná mię / márnośći kádzą y potykáją śię ná drogách ſwych / ná śćieƺkách ſtárodáwnych / chodząc śćieƺkámi drogi nieutorowáney ;
Ták / ábym podał źiemię ich ná ſpuſtoƺenie / ná świſtánie wiecżne / áby káżdy / ktoby ƺedł przez nią / zdumiał śię / y kiwał głową ſwoją.
Wiátrem wſchodnim rozproƺę ich przed nieprzyjáćielem ; tył á nie twárz ukáżę im w dźień zátráceniá ich.
Y rzekli : Pójdźćie á wymyślmy co przećiwko Jeremijáƺowi ; bo nie zginie zakon od kápłáná / áni rádá od mądrego / áni ſłowo od proroká ; pójdźćież / á ubijmy go językiem / á nie dbájmy ná żádne ſłowá jego.
Pilnuj mię Pánie! á ſłucháj głoſu tych / którzy śię ſpieráją ze mną.
Izali śię ma oddáwáć złem zá dobre / że ukopáli dół duƺy mojey? Wſpomnij / żem ſtáwał przed oblicżem twojim / ábym zá nimi mówił ku ich dobremu / y odwróćił zápálcżywość twoję od nich.
Dla tego dopuść głód ná ſynów ich / á ſpráw / że okrutnie będą pobići od miecżá / że będą żony ich ośieroćiáłe y owdowiáłe / á mężowie ich że będą hániebnie zámordowáni / á dźieći ich pobite miecżem ná wojnie.
Niech będźie ſłyƺány krzyk z domów ich / gdy ná nich nágle wojſko przywiedźieƺ ; bo ukopáli dół / áby mię ułápili á śidłá ukryli ná nogi moje.
Aleś ty / Pánie! powiádomy wƺyſtkiej rády ich przećiwko mnie ná śmierć ; nie bądź miłośćiw niepráwośćiom ich / á grzechu ich przed oblicżem twojim nie zágłádzáj ; ále niech śię potkną przed oblicżnośćią twoją / cżáſu zápálcżywośći twojey ſurowo śię obchodź z nimi.
