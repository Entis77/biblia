Słowo / które śię ſtáło do Jeremijáƺá od Páná mówiąc :
Wſtáń / á wſtąp do domu gárncárzowego / á tám ſpráwię / że uſłyƺyƺ ſłowá moje.
Y wſtąpiłem do domu gárncárzowego / á oto on robił robotę ná kręgu.
A gdy śię zepſuło nácżynie w ręce gárncárzowej / które on cżynił z gliny / tedy záś ucżynił z niej nácżynie inƺe / jáko śię mu nájlepiej zdáło ucżynić.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Izálibym ták nie mógł z wámi poſtąpić / jáko ten gárncárz / o domie Izráelſki? mówi Pán. Oto jáko gliná w ręce gárncárzowej / tákeśćie wy w ręce mojej / o domie Izráelſki!
Jeźlibym rzekł nágle przećiwko národowi / y przećiwko króleſtwu / że je wykorzenię / y zepſuję / y wygubię ;
Wƺákże jeźliby śię odwróćił on náród od złośći ſwojej / przećiw któremum mówił : y Jábym żáłowáł tego złego / którem mu umyślił ucżynić.
Záśię / jeźlibym rzekł nágle o národźie y o króleſtwie / że je pobuduję y wƺcżepię ;
Wƺákże jeźliby cżynił / co złego jeſt przed ocżymá memi / nie ſłuchájąc głoſu mego : y Jábym żáłowáł tego dobrodźiejſtwá / którem mu obiecáł ucżynić.
A przetoż rzecż teráz do mężów Judzkich / y do obywáteli Jeruzálemſkich / mówiąc : Ták mówi Pán : Oto Já gotuję ná wáſ złą rzecż / y myślę coś przećiwko wám. Náwróććież śię już káżdy od złej drogi ſwojej / á popráwćie káżdy dróg wáƺych / y ſpráw wáƺych.
Ale oni rzekli : Nic z tego ; bo zá myślámi náƺemi pójdźiemy / á káżdy upór ſercá ſwojego złego cżynić będźiemy.
Przetoż ták mówi Pán : Pytájćie teráz między pogánámi / któż ſłyƺáł co tákowego? Sproſność wielką popełniłá pánná Izráelſká.
Izáli kto opuśći polá moje dlá ſkáł y dlá śniegu ná Libánie? Izáli kto opuśći wody ćiekące dlá wody bárdzo źimnej?
Ale lud mój zápomniáwƺy ná mię / márnośći kádzą y potykáją śię ná drogách ſwych / ná śćieƺkách ſtárodáwnych / chodząc śćieƺkámi drogi nieutorowánej ;
Ták / ábym podáł źiemię ich ná ſpuſtoƺenie / ná świſtánie wiecżne / áby káżdy / ktoby ƺedł przez nią / zdumiáł śię / y kiwáł głową ſwoją.
Wiátrem wſchodnim rozproƺę ich przed nieprzyjáćielem ; tył á nie twárz ukáżę im w dźień zátráceniá ich.
Y rzekli : Pójdźćie á wymyślmy co przećiwko Jeremijáƺowi ; bo nie zginie zákon od kápłáná / áni rádá od mądrego / áni ſłowo od proroká ; pójdźćież / á ubijmy go językiem / á nie dbájmy ná żádne ſłowá jego.
Pilnuj mię Pánie! á ſłucháj głoſu tych / którzy śię ſpieráją ze mną.
Izáli śię má oddáwáć złem zá dobre / że ukopáli dół duƺy mojej? Wſpomnij / żem ſtáwáł przed oblicżem twojim / ábym zá nimi mówił ku ich dobremu / y odwróćił zápálcżywość twoję od nich.
Dlátego dopuść głód ná ſynów ich / á ſpráw / że okrutnie będą pobići od miecżá / że będą żony ich ośieroćiáłe y owdowiáłe / á mężowie ich że będą hániebnie zámordowáni / á dźieći ich pobite miecżem ná wojnie.
Niech będźie ſłyƺány krzyk z domów ich / gdy ná nich nágle wojſko przywiedźieƺ ; bo ukopáli dół / áby mię ułápili á śidłá ukryli ná nogi moje.
Aleś ty / Pánie! powiádomy wƺyſtkiej rády ich przećiwko mnie ná śmierć ; nie bądź miłośćiw niepráwośćiom ich / á grzechu ich przed oblicżem twojim nie zágłádzáj ; ále niech śię potkną przed oblicżnośćią twoją / cżáſu zápálcżywośli twojej ſurowo śię obchodź z nimi.
