Ták mówi Pán : Oto Ja wzbudzę przećiwko Bábilonowi / y przećiwko tym / którzy mieƺkáją w pośród powſtáwájących przećiwko mnie / wiátr záráźliwy ;
Y poślę ná Bábiilon przewiewácży / którzy go przewiewáć będą / y wypróżnią źiemię jego / gdyż będą przećiwko niemu zewƺąd w dźień ućiſku.
Do tego / który mocno ćiągnie łuk ſwój / á poſtępuje w páncerzu ſwoim / rzekę : Nie folgujćie młodźieńcom jego / wygłádźćie wƺyſtko wojſko jego.
Niech polegną pobići w źiemi Cháldejſkiej / á poprzebijáni po ulicách jego.
Bo nie jeſt opuƺcżony Izráel y Judá od Bogá ſwego / od Páná zaſtępów / choć źiemiá ich pełna jeſt przeſtępſtwá przećiwko Świętemu Izráelſkiemu.
Ućiekájćie z pośrzodku Bábilonu / á niech záchowá káżdy duƺę ſwoję / ábyśćie nie byli zátráceni w niepráwośći jego ; bo cżás będźie pomſty Páńſkiej / ſám mu zápłátę oddá.
Byłći Bábilon kubkiem złotym w ręce Páńſkiej / upájájącym wƺyſtkę źiemię ; wino jego piły narody / dla tego poƺáláły narody ;
Ale nágle upádnie Bábilon / y ſtárty będźie ; rozkwilćie śię nád nim / nábierzćie olejku bálſámowego dla boleśći jego / owá śię wylecży.
Lecżyliśmy Bábilon / ále nie jeſt ulecżony. Opuśćmyż go / á pójdźmy káżdy do źiemi ſwej ; bo ſąd jego áż do niebá śięgá / y wynióſł śię áż pod obłoki.
Wywiódł Pán ſpráwiedliwośći náƺe. Pójdźćie / á opowiádájmy ná Syonie dźieło Páná / Bogá náƺego.
Wyoſtrzćie ſtrzáły / ſporządźćie tárcże. Wzbudźił Pán duchá królów Medſkich ; bo przećiwko Bábilonowi zámyſł jego / áby go wytráćił / gdyż pomſtá Páńſka jeſt pomſtá kośćiołá jego.
Podnieśćie chorągiew ná murách Bábilońſkich / przycżyńćie ſtráży / poſtáwćie ſtróżów / nágotujćie záſádzki ; bo y umyślił Pán y wykoná / co wyrzekł przećiwko obywátelom Bábilońſkim.
O ty / który mieƺkáƺ nád wodámi wielkiemi! o bogáty w ſkárby! przyƺedł koniec twój / kres łákomſtwá twego.
Przyśiągł Pán zaſtępów ná duƺę ſwoję / że ćię nápełni ludźmi / jáko chrząƺcżámi / którzy ucżynią nád tobą okrzyk wojenny.
Onći to jeſt / który ucżynił źiemię mocą ſwoją / który utwierdźił okrąg świátá mądrośćią ſwoją / y roztropnośćią ſwoją rozpoſtárł niebioſá ;
Który gdy głos wypuƺcżá / wody ná niebie ƺumią / á który ſpráwuje / áby wyſtępowáły páry od końcżyn źiemi / y błyſkáwice ze dżdżem przywodźi / á wywodźi wiátr z ſkárbów ſwojich.
Ták zgłupiał káżdy cżłowiek / że tego nie zná / że poháńbiony bywa złotnik od obrázu rytego ; bo kłámſtwem jeſt ulánie jego / á niemáƺ w nich duchá.
Márnośćią ſą á dźieło błędów ; zginą cżáſu náwiedzeniá ſwego.
Nie tákowyć jeſt dźiał Jákóbowy ; bo on jeſt / który wƺyſtko ſtworzył / á Izráel jeſt prętem dźiedźictwá jego ; Pán zaſtępów imię jego.
Młotemeś ty moim kruƺącym / orężem wojennym / ábym pokruƺył przez ćię narody / y poburzył przez ćię króleſtwá ;
Abym pokruƺył przez ćię koniá y jezdnego / ábym pokruƺył przez ćię wóz y tego / co ná nim jeźdźi ;
Abym pokruƺył przez ćię mężá y niewiáſtę / ábym pokruƺył przez ćię ſtárcá y dźiećię / ábym pokruƺył przez ćię młodźieńcá y pánnę ;
Abym pokruƺył przez ćię páſterzá y trzodę jego / ábym pokruƺył przez ćię orácżá / y ſprzężáj jego / ábym pokruƺył przez ćię kśiążąt y hetmánów.
Ale już oddam Bábilonowi / y wƺyſtkim obywátelom Cháldejſkim zá wƺyſtkie złośći ich / które cżynili Syonowi przed ocżymá wáƺemi / mówi Pán.
Otom Ja jeſt przećiwko tobie / o góro káżącá! mówi Pán / która káźiƺ wƺyſtkę źiemię / y wyćiągnę rękę moję przećiwko tobie / á zwálę ćię z ſkał / y ucżynię ćię górą ſpáleniá ;
A nie wezmą z ćiebie kámieniá do węgłá / áni kámieniá do gruntów ; bo puſtynią wiecżną będźieƺ / mówi Pán.
Podnieśćie chorągiew w źiemi / trąbćie w trąbę miedzy národámi / zgotujćie przećiwko niemu narody / zwołájćie przećiwko niemu króleſtwá Arárád / Minny / y Aſchenás ; poſtánówćie przećiwko niemu hetmáná / przywiedźćie konie jáko chrząƺcże nájeżone ;
Zgotujćie przećiwko niemu narody / królów Medſkich / kśiążąt ich / y wƺyſtkich hetmánów ich / ze wƺyſtką źiemią włádzy ich ;
Tedy zádrży źiemiá / y rozboleje śię / gdy wykonáne będą przećiwko Bábilonowi myśli Páńſkie / áby obróćił źiemię Bábilońſką w puſtynię / áby zoſtáłá bez obywátelá.
Przeſtáną mocárze Bábilońſcy wálcżyć / uśiądą w zámkách / uſtánie męſtwo ich / będą jáko niewiáſty ; zápáli mieƺkánia ich / pokruƺone będą záwory ich.
Goniec ſpotká drugiego gońcá / á poſeł zábieży poſłowi / áby opowiedźieli królowi Bábilońſkiemu / iż wźięto miáſto jego z jedney ſtrony /
A iż brody ubieżono / y jeźiorá wypáláno ogniem / á mężowie wálecżni uſtráƺeni ſą.
Bo ták mówi Pán zaſtępów / Bóg Izráelſki : Córká Bábilońſká jeſt jáko bojewiſko / cżás deptánia jey przyƺedł ; jeƺcże málucżko / á przydźie cżás żniwá jey.
Pożárł mię / potárł mię Nábuchodonozor / król Bábilońſki / ucżynił mię nácżyniem próżnem / połknął mię jáko ſmok / nápełnił brzuch ſwój rozkoƺámi mojemi / y wygnał mię.
Gwáłt mnie y ćiáłu memu ucżyniony niech przydźie ná Bábilon / mówi obywátelká Syońſká / á krew mojá ná obywáteli Cháldejſkich / mówi Jeruzálem.
Przetoż ták mówi Pán : Oto śię Ja záſtáwię o krzywdę twoję / á pomƺcżę śię zá ćię ; bo wyſuƺę morze jego / wyſuƺę y źródłá jego.
Y będźie Bábilon obrócony w mogiły / w mieƺkánie ſmoków / w zdumienie / y w poświſtánie / y będźie bez obywátelá.
Poſpołu jáko lwy rycżeć będą / á ſkomleć jáko ƺcżeniętá lwie.
Gdy śię zápálą / ucżynię im ucżtę / y ták ich upoję / że krzycżeć y ſnem wiecżnym záſnąć muƺą / ták / áby nie ocućili / mówi Pán.
Powiodę ich jáko báránki ku zábićiu / jáko bárány y kozły.
Jákożby dobyty mógł być Seſách? Jákożby wźiętá być mogłá chwałá wƺyſtkiej źiemi? Jákożby mógł przyść ná ſpuſtoƺenie Bábilon miedzy národámi?
Wyſtąpi przećiwko Bábilonowi morze / mnóſtwem wáłów jego okryte będźie.
Miáſtá jego będą ſpuſtoƺeniem / źiemią ſuchą y puſtą / źiemią / w którey miáſtách nikt nie będźie mieƺkał / áni będźie chodźił przez nią ſyn cżłowiecży.
Náwiedzę też Belá w Bábilonie / y wydrę / co był połknął / z gęby jego ; y nie będą śię więcey do niego zbiegáć narody / y mury tákże Bábilońſkie upádną.
Wyjdźćie z pośrzodku jego / ludu mój! á wybáw káżdy duƺę ſwoję przed gniewem zápálcżywośći Páńſkiej.
A nie bądźćie miękkiego ſercá / áni śię lękájćie wieśći / którą będźie ſłycháć w tey źiemi ; gdy przydźie jednego roku nowiná / potem drugiego roku wieść y gwáłt w źiemi / á pán ná páná.
Przetoż oto dni przyjdą / w które Ja náwiedzę báłwány ryte Bábilońſkie / á wƺyſtká źiemiá jego poháńbioná będźie / y wƺyſcy pobići jego polegną w pośrzodku niego.
Y będą nád Bábilonem śpiewáć niebioſá y źiemiá / y wƺyſtko / co ná nich jeſt / gdy z północy przyjdą nań puſtoƺyćiele / mówi Pán.
Jáko Bábilon poráźił onych pobitych Izráelſkich / ták z Bábilonu polegną pobići po wƺyſtkiej źiemi.
O którzyśćie uƺli miecżá / idźćie / nie ſtójćie! wſpominájćie z dáleká ná Páná / á Jeruzálem niech wſtępuje ná ſerce wáƺe.
Rzecżćie : Wſtydźimy śię / że ſłyƺymy urągánie ; háńbá okryłá twárzy náƺe / bo cudzoźiemcy wchodzą do świątnic domu Páńſkiego.
Przetoż oto dni przychodzą / mówi Pán / że náwiedzę ryte báłwány jego / á po wƺyſtkiej źiemi jego zrániony ſtękáć będźie.
Choćiáżby Bábilon wſtąpił ná niebo / y obwárował ná wyſokośći moc ſwoję / przećie odemnie przyjdą puſtoƺyćiele jego / mówi Pán.
Głos wołánia z Bábilonu / á ſtárćie wielkie z źiemi Cháldejſkiej ;
Bo Pán Bábilon zburzy y wytráći z niego głos wielki / choćby hucżáły wáły ich jáko wody wielkie / y wydány był ƺum głoſu ich.
Gdy nań / to jeſt ná Bábilon / puſtoƺyćiel przyćiągnie pojmáni będą mocárze jego / pokruƺone będą łuki ich ; bo Bóg nágrody / Pán nágrodźi im ſowićie ;
Opoi kśiążąt jego y mędrców jego / wodzów jego / y urzędników jego / y mocárzy jego / áby záſnęli ſnem wiecżnym / á nie ocućili / mówi król / Pán zaſtępów imię jego.
Ták mówi Pán zaſtępów : Mur Bábilońſki ƺeroki do gruntu zburzony będźie / y bramy jego wyſokie ogniem ſpálone będą / á ludźie dármo prácowáć będą / á narody przy ogniu pomdleją.
Toć jeſt ſłowo / które rozkázał Jeremijáƺ prorok Sárájáƺowi / ſynowi Neryjáƺá / ſyná Mááſejáƺowego / gdy odƺedł od Sedekijáƺá / królá Judzkiego / do Bábilonu / roku cżwártego królowánia jego ; ( á Sárájáƺ był kśiążęćiem w Menuchá. )
Gdy zápiſał Jeremijáƺ wƺyſtko złe / które przyść miáło ná Bábilon / w kśięgi jedne / wƺyſtkie te ſłowá / które ſą nápiſáne przećiwko Bábilonowi.
Y rzekł Jeremijáƺ do Sárájáƺá : Gdy przyjdźieƺ do Bábilonu / y oglądáƺ go / tedy przecżytáƺ te wƺyſtkie ſłowá /
A rzecżeƺ : O Pánie! tyś mówił przećiwko miejſcu temu / że je wytráćiƺ / áby w nim nie mieƺkał nikt / áni z ludźi áni z bydląt / ále żeby było puſtkámi wiecżnemi.
A gdy do końcá przecżytáƺ te kśięgi / przywiążeƺ do nich kámień / y wrzućiƺ je w pośród Eufráteſá /
A rzecżeƺ : Ták zátopiony będźie Bábilon / á nie powſtánie więcey z tego złego / które Ja nań przywiodę / choć uſtáwáć będą. Aż dotąd ſłowá Jeremijáƺowe.
