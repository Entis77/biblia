Ták mówi Pán : Oto Já wzbudzę przećiwko Bábilonowi / y przećiwko tym / którzy mieƺkáją w pośród powſtáwájących przećiwko mnie / wiátr záráźliwy ;
Y poślę ná Bábiilon przewiewácży / którzy go przewiewáć będą / y wypróżnią ziemię jego / gdyż będą przećiwko niemu zewƺąd w dzień ućiſku.
Do tego / który mocno ćiągnie łuk ſwój / á poſtępuje w páncerzu ſwoim / rzekę : Nie folgujćie młodzieńcom jego / wygłádźćie wƺyſtko wojſko jego.
Niech polegną pobići w ziemi Cháldejſkiej / á poprzebijáni po ulicách jego.
Bo nie jeſt opuƺcżony Izráel y Judá od Bogá ſwego / od Páná záſtępów / choć ziemiá ich pełná jeſt przeſtępſtwá przećiwko Świętemu Izráelſkiemu.
Ućiekájćie z pośrodku Bábilonu / á niech záchowá káżdy duƺę ſwoję / ábyśćie nie byli zátráceni w niepráwośći jego ; bo cżáſ będzie pomſty Páńſkiej / ſám mu zápłátę oddá.
Byłći Bábilon kubkiem złotym w ręce Páńſkiej / upájájącym wƺyſtkę ziemię ; wino jego piły národy / dlátego poƺáláły národy ;
Ale nágle upádnie Bábilon / y ſtárty będzie ; rozkwilćie śię nád nim / nábierzćie olejku bálſámowego dlá boleśći jego / owá śię wylecży.
Lecżyliśmy Bábilon / ále nie jeſt ulecżony. Opuśćmyż go / á pójdźmy káżdy do ziemi ſwej ; bo ſąd jego áż do niebá śięgá / y wynióſł śię áż pod obłoki.
Wywiódł Pán ſpráwiedliwośći náƺe. Pójdźćie / á opowiádájmy ná Syonie dzieło Páná / Bogá náƺego.
Wyoſtrzćie ſtrzáły / ſporządźćie tárcże. Wzbudził Pán duchá królów Medſkich ; bo przećiwko Bábilonowi zámyſł jego / áby go wytráćił / gdyż pomſtá Páńſká jeſt pomſtá kośćiołá jego.
Podnieśćie chorągiew ná murách Bábilońſkich / przycżyńćie ſtráży / poſtáwćie ſtróżów / nágotujćie záſádzki ; bo y umyślił Pán y wykoná / co wyrzekł przećiwko obywátelom Bábilońſkim.
O ty / który mieƺkáƺ nád wodámi wielkiemi! o bogáty w ſkárby! przyƺedł koniec twój / kreſ łákomſtwá twego.
Przyśiągł Pán záſtępów ná duƺę ſwoję / że ćię nápełni ludźmi / jáko chrząƺcżámi / którzy ucżynią nád tobą okrzyk wojenny.
Onći to jeſt / który ucżynił ziemię mocą ſwoją / który utwierdził okrąg świátá mądrośćią ſwoją / y roztropnośćią ſwoją rozpoſtárł niebioſá ;
Który gdy głoſ wypuƺcżá / wody ná niebie ƺumią / á który ſpráwuje / áby wyſtępowáły páry od końcżyn ziemi / y błyſkáwice ze dżdżem przywodzi / á wywodzi wiátr z ſkárbów ſwoich.
Ták zgłupiáł káżdy cżłowiek / że tego nie zná / że poháńbiony bywá złotnik od obrázu rytego ; bo kłámſtwem jeſt ulánie jego / á niemáƺ w nich duchá.
Márnośćią ſą á dzieło błędów ; zginą cżáſu náwiedzeniá ſwego.
Nie tákowyć jeſt dziáł Jákóbowy ; bo on jeſt / który wƺyſtko ſtworzył / á Izráel jeſt prętem dziedzictwá jego ; Pán záſtępów imię jego.
Młotemeś ty moim kruƺącym / orężem wojennym / ábym pokruƺył przez ćię národy / y poburzył przez ćię króleſtwá ;
Abym pokruƺył przez ćię koniá y jezdnego / ábym pokruƺył przez ćię wóz y tego / co ná nim jeździ ;
Abym pokruƺył przez ćię mężá y niewiáſtę / ábym pokruƺył przez ćię ſtárcá y dziećię / ábym pokruƺył przez ćię młodzieńcá y pánnę ;
Abym pokruƺył przez ćię páſterzá y trzodę jego / ábym pokruƺył przez ćię orácżá / y ſprzężáj jego / ábym pokruƺył przez ćię kśiążąt y hetmánów.
Ale już oddám Bábilonowi / y wƺyſtkim obywátelom Cháldejſkim zá wƺyſtkie złośći ich / które cżynili Syonowi przed ocżymá wáƺemi / mówi Pán.
Otom Já jeſt przećiwko tobie / o góro káżącá! mówi Pán / którá káziƺ wƺyſtkę ziemię / y wyćiągnę rękę moję przećiwko tobie / á zwálę ćię z ſkáł / y ucżynię ćię górą ſpáleniá ;
A nie wezmą z ćiebie kámieniá do węgłá / áni kámieniá do gruntów ; bo puſtynią wiecżną będzieƺ / mówi Pán.
Podnieśćie chorągiew w ziemi / trąbćie w trąbę między národámi / zgotujćie przećiwko niemu národy / zwołájćie przećiwko niemu króleſtwá Arárád / Minny / y Aſchenáſ ; poſtánówćie przećiwko niemu hetmáná / przywiedźćie konie jáko chrząƺcże nájeżone ;
Zgotujćie przećiwko niemu národy / królów Medſkich / kśiążąt ich / y wƺyſtkich hetmánów ich / ze wƺyſtką ziemią włádzy ich ;
Tedy zádrży ziemiá / y rozboleje śię / gdy wykonáne będą przećiwko Bábilonowi myśli Páńſkie / áby obróćił ziemię Bábilońſką w puſtynię / áby zoſtáłá bez obywátelá.
Przeſtáną mocárze Bábilońſcy wálcżyć / uśiądą w zámkách / uſtánie męſtwo ich / będą jáko niewiáſty ; zápáli mieƺkániá ich / pokruƺone będą záwory ich.
Goniec ſpotká drugiego gońcá / á poſeł zábieży poſłowi / áby opowiedzieli królowi Bábilońſkiemu / iż wzięto miáſto jego z jednej ſtrony /
A iż brody ubieżono / y jeziorá wypáláno ogniem / á mężowie wálecżni uſtráƺeni ſą.
Bo ták mówi Pán záſtępów / Bóg Izráelſki : Córká Bábilońſká jeſt jáko bojewiſko / cżáſ deptániá jej przyƺedł ; jeƺcże málucżko / á przyjdzie cżáſ żniwá jej.
Pożárł mię / potárł mię Nábuchodonozor / król Bábilońſki / ucżynił mię nácżyniem próżnem / połknął mię jáko ſmok / nápełnił brzuch ſwój rozkoƺámi mojemi / y wygnáł mię.
Gwáłt mnie y ćiáłu memu ucżyniony niech przyjdzie ná Bábilon / mówi obywátelká Syońſká / á krew mojá ná obywáteli Cháldejſkich / mówi Jeruzálem.
Przetoż ták mówi Pán : Oto śię Já záſtáwię o krzywdę twoję / á pomƺcżę śię zá ćię ; bo wyſuƺę morze jego / wyſuƺę y źródłá jego.
Y będzie Bábilon obrócony w mogiły / w mieƺkánie ſmoków / w zdumienie / y w poświſtánie / y będzie bez obywátelá.
Poſpołu jáko lwy rycżeć będą / á ſkomleć jáko ƺcżeniętá lwie.
Gdy śię zápálą / ucżynię im ucżtę / y ták ich upoję / że krzycżeć y ſnem wiecżnym záſnąć muƺą / ták / áby nie ocućili / mówi Pán.
Powiodę ich jáko báránki ku zábićiu / jáko bárány y kozły.
Jákożby dobyty mógł być Seſách? Jákożby wziętá być mogłá chwáłá wƺyſtkiej ziemi? Jákożby mógł przyjść ná ſpuſtoƺenie Bábilon między národámi?
Wyſtąpi przećiwko Bábilonowi morze / mnóſtwem wáłów jego okryte będzie.
Miáſtá jego będą ſpuſtoƺeniem / ziemią ſuchą y puſtą / ziemią / w której miáſtách nikt nie będzie mieƺkáł / áni będzie chodził przez nią ſyn cżłowiecży.
Náwiedzę też Belá w Bábilonie / y wydrę / co był połknął / z gęby jego ; y nie będą śię więcej do niego zbiegáć národy / y mury tákże Bábilońſkie upádną.
Wyjdźćie z pośrodku jego / ludu mój! á wybáw káżdy duƺę ſwoję przed gniewem zápálcżywośći Páńſkiej.
A nie bądźćie miękkiego ſercá / áni śię lękájćie wieśći / którą będzie ſłycháć w tej ziemi ; gdy przyjdzie jednego roku nowiná / potem drugiego roku wieść y gwáłt w ziemi / á pán ná páná.
Przetoż oto dni przyjdą / w które Já náwiedzę báłwány ryte Bábilońſkie / á wƺyſtká ziemiá jego poháńbioná będzie / y wƺyſcy pobići jego polegną w pośrodku niego.
Y będą nád Bábilonem śpiewáć niebioſá y ziemiá / y wƺyſtko / co ná nich jeſt / gdy z północy przyjdą náń puſtoƺyćiele / mówi Pán.
Jáko Bábilon poráził onych pobitych Izráelſkich / ták z Bábilonu polegną pobići po wƺyſtkiej ziemi.
O którzyśćie uƺli miecżá / idźćie / nie ſtójćie! wſpominájćie z dáleká ná Páná / á Jeruzálem niech wſtępuje ná ſerce wáƺe.
Rzecżćie : Wſtydzimy śię / że ſłyƺymy urągánie ; háńbá okryłá twárzy náƺe / bo cudzoziemcy wchodzą do świątnic domu Páńſkiego.
Przetoż oto dni przychodzą / mówi Pán / że náwiedzę ryte báłwány jego / á po wƺyſtkiej ziemi jego zrániony ſtękáć będzie.
Choćiáżby Bábilon wſtąpił ná niebo / y obwárowáł ná wyſokośći moc ſwoję / przećie odemnie przyjdą puſtoƺyćiele jego / mówi Pán.
Głoſ wołániá z Bábilonu / á ſtárćie wielkie z ziemi Cháldejſkiej ;
Bo Pán Bábilon zburzy y wytráći z niego głoſ wielki / choćby hucżáły wáły ich jáko wody wielkie / y wydány był ƺum głoſu ich.
Gdy náń / to jeſt ná Bábilon / puſtoƺyćiel przyćiągnie pojmáni będą mocárze jego / pokruƺone będą łuki ich ; bo Bóg nágrody / Pán nágrodzi im ſowićie ;
Opoi kśiążąt jego y mędrców jego / wodzów jego / y urzędników jego / y mocárzy jego / áby záſnęli ſnem wiecżnym / á nie ocućili / mówi król / Pán záſtępów imię jego.
Ták mówi Pán záſtępów : Mur Bábilońſki ƺeroki do gruntu zburzony będzie / y brámy jego wyſokie ogniem ſpálone będą / á ludzie dármo prácowáć będą / á národy przy ogniu pomdleją.
Toć jeſt ſłowo / które rozkázáł Jeremijáƺ prorok Sárájáƺowi / ſynowi Neryjáƺá / ſyná Mááſejáƺowego / gdy odƺedł od Sedekijáƺá / królá Judzkiego / do Bábilonu / roku cżwártego królowániá jego ; ( á Sárájáƺ był kśiążęćiem w Menuchá. )
Gdy zápiſáł Jeremijáƺ wƺyſtko złe / które przyjść miáło ná Bábilon / w kśięgi jedne / wƺyſtkie te ſłowá / które ſą nápiſáne przećiwko Bábilonowi.
Y rzekł Jeremijáƺ do Sárájáƺá : Gdy przyjdzieƺ do Bábilonu / y oglądáƺ go / tedy przecżytáƺ te wƺyſtkie ſłowá /
A rzecżeƺ : O Pánie! tyś mówił przećiwko miejſcu temu / że je wytráćiƺ / áby w nim nie mieƺkáł nikt / áni z ludzi áni z bydląt / ále żeby było puſtkámi wiecżnemi.
A gdy do końcá przecżytáƺ te kśięgi / przywiążeƺ do nich kámień / y wrzućiƺ je w pośród Eufráteſá /
A rzecżeƺ : Ták zátopiony będzie Bábilon / á nie powſtánie więcej z tego złego / które Já náń przywiodę / choć uſtáwáć będą. Aż dotąd ſłowá Jeremijáƺowe.
