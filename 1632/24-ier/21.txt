Słowo / które śię ſtáło do Jeremijáƺá od Páná / gdy do niego król Sedekijáƺ poſłáł Fáſſurá / ſyná Málchyjáƺowego / y Sofonijáƺá / ſyná Mááſejáƺowego / kápłáná / áby rzekli :
Porádź śię / proƺę / o náſ Páná ; bo Nábuchodonozor / król Bábilońſki / wálcży przećiwko nám : owá ſnáć ucżyni Pán z námi według wƺyſtkich dźiwnych ſpráw ſwoich / żeby odćiągnął od náſ.
Y rzekł Jeremijáƺ do nich : Ták powiedzćie Sedekijáƺowi :
Ták mówi Pán / Bóg Izráelſki : Oto Já odwrócę nácżyniá wojenne / które ſą w rękách wáƺych / y któremi wy wálcżyćie przećiw królowi Bábilońſkiemu / y Cháldejcżykom / którzy wáſ oblegli około muru / y zgromádzę ich w pośrodek miáſtá tego.
A Já ſám wálcżyć będę przećiwko wám ręką wyćiągnioną y rámieniem możnem / á to w gniewie / y w popędliwośći / y w zápálcżywośći wielkiej ;
Y uderzę obywáteli tego miáſtá / ták / że y ludźie y bydlętá morem wielkim pomrą.
A potem / ták mówi Pán / podám Sedekijáƺá / królá Judzkiego / y ſługi jego / y lud / to jeſt tych / którzy pozoſtáną w tem mieśćie po morze / y po miecżu / y po głodźie / w rękę Nábuchodonozorá / królá Bábilońſkiego / y w rękę nieprzyjáćiół ich / á ták w rękę ƺukájących duƺy ich / który ich pobije oſtrzem miecżá : nie przepuśći im / áni im ſfolguje / áni śię zmiłuje.
Przetoż rzecż do ludu tego : Ták mówi Pán : Oto já kłádę przed wámi drogę żywotá y drogę śmierći.
Ktokolwiek zoſtánie w tem mieśćie / zginie od miecżá / álbo od głodu / álbo od moru : ále kto wyjdźie y podá śię Cháldejcżykom / którzy wáſ oblegli / pewnie żyw zoſtánie / y będźie mu duƺá jego w korzyśći.
Bom obróćił oblicże moje przećiwko temu miáſtu ku złemu / á nie ku dobremu / mówi Pán. W rękę królá Bábilońſkiego podáne będźie / y ſpáli je ogniem.
Ale domowi królá Judzkiego rzecż : Słuchájćie ſłowá Páńſkiego.
O domie Dawidowy! Ták mówi Pán : Odpráwujćie ſąd káżdego poránku / á wyrywájćie ućiśnionego z ręki gwáłtowniká / by ſnáć nie wyƺłá jáko ogień zápálcżywość mojá / á nie gorzáłá / ták / żeby nie był / ktoby ugáśił dlá złośći ſpráw wáƺych.
Otom Já przećiwko tobie / którá mieƺkáƺ w tej dolinie / jáko ſkáłá w tej równinie / mówi Pán ; którzy mówićie : Któż przyćiągnie ná náſ? á kto wnijdźie do przybytków náƺych?
Bo wáſ náwiedzę według owocu ſpráw wáƺych / mówi Pán ; á rozniecę ogień w leśie twoim / który pożre wƺyſtko około niego.
