Grzech Judzki nápiſány jeſt piórem żeláznem / á oſtrym dyjámentem wyryty jeſt ná táblicy ſercá ich / y ná rogách ołtárzów wáƺych ;
Gdy wſpomináją ſynowie ich ná ołtárze ich / y ná gáje ich pod drzewem zielonem / ná págórkách wyſokich.
O góro / y pole moje! májętność twoję y wƺyſtkie ſkárby twoje podám ná rozƺárpánie dlá grzechu wyżyn twoich / które máƺ po wƺyſtkich gránicách twoich.
A ty muśiƺ zániecháć zá przewinieniem twojim dziedzictwá twego / którem ći dáł. Y dám ćię w niewolę nieprzyjáćiołom twoim / y ziemi / której nie znáƺ ; bośćie ogień rozniećili w popędliwośći mojej / który áż ná wieki gorzeć będzie.
Ták mówi Pán : Przeklęty mąż / który ufá w cżłowieku / y który pokłádá ćiáło rámieniem ſwojim / á od Páná odſtępuje ſerce jego.
Abowiem ſtánie śię jáko wrzoſ ná puƺcży / który nie cżuje / gdy co dobrego przychodzi / ále bywá ná ſuchych miejſcách ná puƺcży w ziemi ſłonej / y w której nikt nie mieƺká.
Błogoſłáwiony mąż / który ufá w Pánu / á Pán jeſt nádzieją jego.
Bo będzie jáko drzewo wƺcżepione nád wodámi / á nád ſtrumieniem zápuƺcżájące korzenie ſwoje / które nie cżuje / gdy przychodzi gorącość / ále liść jego zoſtáje zielony / á roku ſuchego nie fráſuje śię / y nie przeſtáje przynośić owocu.
Nájzdrádliwƺe jeſt ſerce nádewƺyſtko y nájprzewrotniejƺe ; któż je pozná?
Já Pán / który śię bádám ſercá / y doświádcżám nerek / ták ábym káżdemu oddáł według drogi jego / y według owocu ſpráw jego.
Jáko kuropátwá zgromádzá jájká / ále ich nie wylęgá : ták / kto zbierá bogáctwá / á nieſpráwiedliwie / w połowie dni ſwoich opuśći je / á ná oſtátek będzie głupim ;
Ale miejſce świątnicy náƺej / to jeſt ſtolicá chwáły Nájwyżƺego / wiecżnie trwá.
O nádziejo Izráelſká / Pánie! wƺyſcy / którzy ćię opuƺcżáją / niech będą záwſtydzeni ; którzy odſtępują odemnie / niech ná ziemi zápiſáni będą ; ábowiem opuśćili źródło wód żywych / Páná.
Uzdrów mię / Pánie! á będę uzdrowiony : zbáw mię / á będę zbáwiony ; ábowiemeś ty chwáłá mojá.
Oto oni do mnie mówią : Gdzież jeſt to ſłowo Páńſkie? Niechże już przyjdzie ;
Choćiáżem Já tego nie zábiegáł / ábym był páſterzem twoim / ánim dniá boleśći prágnął / ty wieƺ / cokolwiek wyƺło z uſt moich / przed oblicżem twojim jeſt.
Nie bądźże mi ná poſtrách ; tyś nádziejá mojá w dzień utrápieniá.
Niech będą pocháńbieni / którzy mię prześládują / á já niech nie będę záwſtydzony ; niech śię oni lękáją / á já niech śię nie lękám ; przywiedź ná nich dzień utrápieniá / á dwojákiem ſkruƺeniem ſkruƺ ich.
Ták Pán rzekł do mnie : Idź / á ſtáń w brámie ſynów ludu tego / którą wchodzą królowie Judzcy / y którą wychodzą / y we wƺyſtkich brámách Jeruzálemſkich /
Y rzecż do nich : Słuchájćie ſłowá Páńſkiego / królowie Judzcy / y wƺyſtek Judo / y wƺyſcy obywátele Jeruzálemſcy / którzy chádzáćie temi brámámi!
Ták mówi Pán : Strzeżćie pilnie duƺ wáƺych / á nie nośćie brzemion żádnych w dzień ſábátu / áni ich wnośćie brámámi Jeruzálemſkiemi ;
Ani wynáƺájćie brzemion z domów wáƺych w dzień ſábátu / áni żádnej roboty odpráwujćie / ále święććie dzień ſábátu / jákom rozkázáł ojcom wáƺym.
Wƺákże nie uſłucháli / áni nákłonili uchá ſwego / owƺem / zátwárdzili kárk ſwój / nie ſłuchájąc áni przyjmując náuki.
A jeźli mię pilnie ſłucháć będziećie / mówi Pán / ták / żebyśćie nie wnośili brzemion brámámi miáſtá tego w dzień ſábátu / ále święćili dzień ſábátu / nie odpráwując weń żádnej roboty ;
Tedy wchodzić będą brámámi miáſtá tego królowie y kśiążętá śiedzący ná ſtolicy Dawidowej / jeżdżąc ná wozách y ná koniách / oni y kśiążętá ich / mężowie Judzcy y obywátele Jeruzálemſcy / y ſtáć będzie to miáſto áż ná wieki.
Y zbieżą śię z miáſt Judzkich y z okolicżnych miejſc Jeruzálemſkich / y z ziemi Benjáminowej / y z równin / y z tej góry / y od południá / przynoƺąc cáłopálenie / y ofiárę / y dár / y kádzidło / tákże y dziękcżynienie nioſąc do domu Páńſkiego.
Ale jeźli mię nie uſłucháćie / ábyśćie święćili dzień ſábátu / á nie nośili brzemion / wchodząc brámámi Jeruzálemſkiemi w dzień ſábátu / tedy rozniecę ogień w brámách jego / który pożre páłáce Jeruzálemſkie / á nie będzie ugáƺony.
