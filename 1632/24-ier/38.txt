Y uſłyƺáł Sefátyjáƺ / ſyn Mátánowy / y Godolijáƺ / ſyn Fáſſurowy / y Juchál / ſyn Selemijáƺowy / y Fáſſur / ſyn Melchijáƺowy / ſłowá / które Jeremijáƺ mówił do wƺyſtkiego ludu / mówiąc :
Ták mówi Pán : Ktoby zoſtáł w tem mieśćie / zginie od miecżá / od głodu y od moru ; ále ktoby wyƺedł do Cháldejcżyków / żyć będzie / á będzie mu duƺá jego zá korzyść / y żyw zoſtánie.
Ták mówi Pán : Pewnie podáne będzie to miáſto w ręce wojſká królá Bábilońſkiego / y weźmie je.
Przetoż rzekli oni kśiążętá do królá : Niech umrze ten mąż / poniewáż on oſłábiá ręce mężów wálecżnych / pozoſtáłych w tem mieśćie / y ręce wƺyſtkiego ludu / mówiąc do nich tákie ſłowá ; ábowiem ten mąż nie ƺuká pokoju ludowi temu / ále złego.
Tedy rzekł król Sedekijáƺ : Oto jeſt w ręce wáƺej ; bo król nic zgołá nie może przećiwko wám.
A ták wzięli Jeremijáƺá / który był w śieni ſtráży / y wrzućili go do dołu Málchyjáƺá / ſyná królewſkiego / y ſpuśćili Jeremijáƺá po powrozách ; á w tym dole nie było nic wody / tylko błoto / ták tonął Jeremijáƺ w onem błoćie.
Ale gdy uſłyƺáł Ebedmelech Murzyn / dworzánin / który był w domu królewſkim / że Jeremijáƺá podáno do dołu / ( á król śiedziáł w brámie Benjáminowej. )
Wnet wyƺedł Ebedmelech z domu królewſkiego / y rzekł do królá / mówiąc :
Królu / pánie mój! źle ucżynili ći mężowie wƺyſtko / co ucżynili Jeremijáƺowi prorokowi / że go wrzućili do tego dołu ; boćby był umárł ná pierwƺem miejſcu od głodu / poniewáż już niemáƺ żádnego chlebá w mieśćie.
Przetoż rozkázáł król Ebedmelechowi Murzynowi / mówiąc : Weźmij z ſobą ſtąd trzydzieśći mężów / á wyćiągnij Jeremijáƺá proroká z tego dołu / niżby umárł.
Tedy wziął Ebedmelech onych mężów z ſobą / y wƺedł do domu królewſkiego pod ſkárbnicę / y nábráł ſtámtąd ſtárych ƺmáćiſk podártych / ƺmáćiſk / mówię / zbótwiáłych / które ſpuśćił do Jeremijáƺá do onego dołu po powrozách.
Y rzekł Ebedmelech Murzyn do Jeremijáƺá : Nuże podłóż te ſtáre podárte y zbótwiáłe ƺmáćiſká pod páchy rąk twoich pod powrozy ; y ucżynił ták Jeremijáƺ.
Wyćiągnęli tedy Jeremijáƺá powrozámi / y dobyli go z onego dołu ; y śiedziáł Jeremijáƺ w śieni ſtráży.
Tedy poſłáł król Sedekijáƺ y wziął Jeremijáƺá proroká do śiebie do trzećich drzwi / które były przy domu Páńſkim. Y rzekł król Jeremijáƺowi : Spytám ćię o jednę rzecż / nie táj nic przedemną.
Y rzekł Jeremijáƺ do Sedekijáƺá : Jeźlić co powiem / pewnie mię zábijeƺ? A jeźlić co porádzę / nie uſłucháƺ mię.
Tedy przyśiągł król Sedekijáƺ Jeremijáƺowi potájemnie / mówiąc : Jáko żyje Pán / który nám tę duƺę ſtworzył / że ćię nie zábiję / áni ćię wydám w rękę mężów tych / którzy ƺukáją duƺy twojej.
Y rzekł Jeremijáƺ do Sedekijáƺá : Ták mówi Pán / Bóg záſtępów / Bóg Izráelſki : Jeźli dobrowolnie wynijdzieƺ do kśiążąt królá Bábilońſkiego / tedy żyć będzie duƺá twojá / á to miáſto nie będzie ſpálone ogniem ; á ták żyw zoſtánieƺ ty y dom twój ;
Ale jeźli nie wynijdzieƺ do kśiążąt królá Bábilońſkiego / pewnie będzie podáne to miáſto w rękę Cháldejcżyków / y ſpálą je ogniem / á y ty nie ujdzieƺ ręki ich.
Tedy rzekł król Sedekijáƺ do Jeremijáƺá : Bárdzo śię boję Żydów / którzy poućiekáli do Cháldejcżyków / bym ſnáć nie był wydány w rękę ich / á ƺydziliby ze mnie.
Y rzekł Jeremijáƺ : Nie wydádzą. Słucháj proƺę głoſu Páńſkiego / któryć já opowiádám / á będzieć dobrze / y żyć będzie duƺá twojá ;
A jeźli śię będzieƺ zbrániáł wynijść / tedy to jeſt ſłowo / które mi Pán pokázáł :
Że oto wƺyſtkie niewiáſty / które zoſtáły w domu królá Judzkiego / będą wywiedzione do kśiążąt królá Bábilońſkiego / á te ſáme rzeką : Námówili ćię / y otrzymáli to ná tobie przyjáćiele twoi / że ulgnęły w błoćie nogi twoje / y cofnęły śię názád.
Wƺyſtkie tákże żony twoje y ſynów twoich wywiodą do Cháldejcżyków / y ty ſám nie ujdzieƺ ręki ich / owƺem ręką królá Bábilońſkiego będzieƺ pojmány / y to miáſto ſpálą ogniem.
Tedy rzekł Sedekijáƺ do Jeremijáƺá : Niecháj nikt nie wie o tem / ábyś nie umárł ;
A jeźliby kśiążętá uſłyƺáwƺy / żem mówił z tobą / przyƺli do ćiebie / y rzeklić : Powiedz nám / proƺę / coś mówił z królem / nie táj przed námi / á nie zábijemy ćię ; co z tobą król mówił?
Tedy im rzecżeƺ : Przełożyłem prośbę moję przed królem / áby mię záś nie kázáł odwiść do domu Jonátánowego / żebym tám nie umárł.
A zeƺli śię wƺyſcy kśiążętá do Jeremijáƺá / y pytáli go ; który im powiedziáł według tego wƺſytkiego / jáko mu był król rozkázáł. A ták milcżąc odeƺli od niego / gdyż śię to było nie ogłośiło.
A Jeremijáƺ śiedziáł w śieni ſtráży áż do onego dniá / którego wzięto Jeruzálem / gdzie był / gdy dobywáno Jeruzálemu.
