A teć ſą ſłowá liſtu / który poſłáł Jeremijáƺ prorok z Jeruzálemu do oſtátków ſtárƺych / którzy byli w pojmániu / y do kápłánów / y do proroków / y do wƺyſtkiego ludu / których był przenióſł Nábuchodonozor z Jeruzálemu do Bábilonu /
Gdy wyƺedł Jechonijáƺ król y królowá / y komornicy / kśiążętá Judzcy / y Jeruzálemſcy / tákże ćieśle y kowále z Jeruzálemu ;
Przez Elháſá / ſyná Sáfánowego / y Giemáryjáƺá / ſyná Helkijáƺowego / ( których był poſłáł Sedekijáƺ / król Judzki / do Nábuchodonozorá / królá Bábilońſkiego / do Bábilonu ) mówiąc :
Ták mówi Pán záſtępów / Bóg Izráelſki / do wƺyſtkich pojmánych / którychem przenióſł z Jeruzálemu do Bábilonu :
Budujćie domy / y oſádzájćie śię ; ƺcżepćie też ſády / á jedzćie owoc ich ;
Pojmujćie żony / á płodźćie ſynów y córki / y dáwájćie ſynom wáƺym żony / á córki wáƺe wydáwájćie zá mąż / áby rodziły ſynów y córki ; rozmnáżájćie śię tám / y niech wáſ nie ubywá.
Szukájćie też pokoju miáſtu temu / do któregom wáſ przenióſł / á módlćie śię zá nie Pánu ; bo w pokoju jego będziećie mieli pokój.
Ták záiſte mówi Pán záſtępów / Bóg Izráelſki : Niech wáſ nie zwodzą prorocy wáśi / którzy ſą między wámi / y wieƺcżkowie wáśi / á nie ſpráwujćie śię ſnámi wáƺemi / które śię wám śnią.
Bo wám oni kłámliwie prorokują w imieniu mojem ; nie poſłáłem ich / mówi Pán.
Ták záiſte mówi Pán : Jáko śię jedno wypełnią śiedmdzieśiąt lát Bábilonowi / náwiedzę wáſ / á potwierdzę wám ſłowá ſwego wybornego o przywróceniu wáſ ná to miejſce.
Bo Já nájlepiej wiem myśli / które myślę o wáſ / mówi Pán : myśli o pokoju / á nie o utrápieniu / ábym ucżynił koniec pożądány biedom wáƺym.
Gdy mię wzywáć będziećie á pójdziećie / y modlić mi śię będziećie / tedy wáſ wyſłuchám ;
A ƺukájąc mię / znájdziećie ; gdy mię ƺukáć będziećie ze wƺyſtkiego ſercá ſwego /
Dám śię wám záiſte ználeść / mówi Pán : á przywrócę więźniów wáƺych / y zgromádzę wáſ ze wƺyſtkich národów / y ze wƺyſtkich miejſc / gdziemkolwiek wáſ zágnáł / mówi Pán ; y przyprowádzę wáſ ná to miejſce / ſkądem wáſ wyprowádził.
Gdy rzecżećie : Wzbudzáł nám Pán proroków prorokujących o záprowádzeniu do Bábilonu.
Bo ták mówi Pán o królu / który śiedzi ná ſtolicy Dawidowej / y o wƺyſtkim ludu / który mieƺká w tem mieśćie / bráćiách wáƺych / którzy nie wyƺli z wámi w tę niewolę ;
Ták mówi Pán záſtępów : Oto Já poślę ná nich miecż / głód / y mór / á ucżynię ich jáko złe figi / których nie jádáją / przeto / że ſą złe.
Abowiem prześládowáć ich będę miecżem / głodem y morem / y dám ich ná potłukánie po wƺyſtkich króleſtwách ziemi / ná przeklęſtwo / y ná zdumienie / owƺem / ná poświſtánie / y ná urągánie między wƺyſtkimi národámi / tám / gdzie ich zápędzę /
Przeto / iż nie ſłucháją ſłów moich / mówi Pán / gdy poſyłám do nich ſług ſwoich / proroków / ráno wſtáwájąc y poſyłájąc ; á nie ſłucháliśćie / mówi Pán.
Przetoż ſłuchájćie ſłowá Páńſkiego wy wƺyſcy pojmáni / którychem wyſłáł z Jeruzálemu do Bábilonu.
Ták mówi Pán záſtępów / Bóg Izráelſki / o Achábie / ſynu Kolájáƺowym / y o Sedekijáƺu / ſynu Máázejáƺowym / którzy wám prorokują w imieniu mojem kłámſtwo : Oto Já podám ich w rękę Nábuchodonozorá / królá Bábilońſkiego / áby ich pobił przed ocżymá wáƺemi.
Y będzie wzięte z nich przeklinánie między wƺyſtkich pojmánych z Judy / którzy ſą w Bábilonie / áby mówili : Niech ćię ucżyni Pán jáko Sedekijáƺá / y jáko Achábá / których upiekł król Bábilońſki ná ogniu /
Przeto / że popełniáli złość w Izráelu / cudzołożąc z żonámi bliźnich ſwoich / á kłámliwie mówiąc ſłowo w imieniu mojem / cżegom im nie przykázáł ; á Já o tem wiem / y jeſtem tego świádkiem / mówi Pán.
A do Semejáƺá Nechálámity rzecż / mówiąc :
Ták powiádá Pán záſtępów / Bóg Izráelſki / mówiąc : Przeto żeś ty poſłáł imieniem ſwojim liſty do wƺyſtkiego ludu / który jeſt w Jeruzálemie / y do Sofonijáƺá / ſyná Máázejáƺowego / kápłáná / y do wƺyſtkich kápłánów / mówiąc :
Pán ćię dáł zá kápłáná miáſto Jojády kápłáná / ábyśćie mieli dozór w domu Páńſkim ná káżdego mężá w rozum záƺłego á prorokującego / ábyś dáł tákowego do więzieniá y do kłody.
Przecżżeś tedy teráz nie zgromił Jeremijáƺá Anátotcżyká / który wám prorokuje?
Bo poſłáł do náſ do Bábilonu / mówiąc : Jeƺcżeć długo cżekáć ; budujćie domy / á oſádzájćie śię ƺcżepćie ſády / á jedzćie owoce ich.
Bo Sofonijáƺ kápłán cżytáł ten liſt przed Jeremijáƺem prorokiem.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá mówiąc : Poślij do wƺyſtkich zábránych w niewolę / mówiąc : Ták mówi Pán o Semejáƺu Nechálámitcżyku : Przeto / że wám prorokowáł Semejáƺ / choćiáżem go Já nie poſłáł / á przywodzi wáſ do tego / ábyśćie mieli nádzieję w kłámſtwie ;
Przetoż ták mówi Pán : Oto Já náwiedzę Semejáƺá Nechálámitcżyká y náśienie jego ; nie będzie miáł nikogo / ktoby mieƺkáł w pośrodku ludu tego / áni oglądá tego dobrá / które Já ucżynię ludowi ſwemu / mówi Pán ; bo rádził / áby odſtąpił lud od Páná.
