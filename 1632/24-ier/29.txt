A teć ſą ſłowá liſtu / który poſłał Jeremijáƺ prorok z Jeruzálemu do oſtátków ſtárƺych / którzy byli w pojmániu / y do kápłanów / y do proroków / y do wƺyſtkiego ludu / których był przenióſł Nábuchodonozor z Jeruzálemu do Bábilonu /
Gdy wyƺedł Jechonijáƺ król y królowá / y komornicy / kśiążętá Judzcy / y Jeruzálemſcy / tákże ćieśle y kowále z Jeruzálemu ;
Przez Elháſá / ſyná Sáfánowego / y Giemáryjáƺá / ſyná Helkijáƺowego / ( których był poſłał Sedekijáƺ / król Judzki / do Nábuchodonozorá / królá Bábilońſkiego / do Bábilonu ) mówiąc :
Ták mówi Pán zaſtępów / Bóg Izráelſki / do wƺyſtkich pojmánych / którychem przenióſł z Jeruzálemu do Bábilonu :
Budujćie domy / y oſádzájćie śię ; ƺcżepćie też ſády / á jedzćie owoc ich ;
Pojmujćie żony / á płodźćie ſynów y córki / y dáwájćie ſynom wáƺym żony / á córki wáƺe wydáwájćie zá mąż / áby rodźiły ſynów y córki ; rozmnáżájćie śię tám / y niech was nie ubywá.
Szukájćie też pokoju miáſtu temu / do któregom was przenióſł / á módlćie śię zá nie Pánu ; bo w pokoju jego będźiećie mieli pokój.
Ták zájiſte mówi Pán zaſtępów / Bóg Izráelſki : Niech was nie zwodzą prorocy wáśi / którzy ſą miedzy wámi / y wieƺcżkowie wáśi / á nie ſpráwujćie śię ſnámi wáƺemi / które śię wam śnią.
Bo wam oni kłámliwie prorokują w imieniu mojim ; nie poſłáłem ich / mówi Pán.
Ták zájiſte mówi Pán : Jáko śię jedno wypełnią śiedmdźieśiąt lat Bábilonowi / náwiedzę was / á potwierdzę wam ſłowá ſwego wybornego o przywróceniu was ná to miejſce.
Bo Ja nájlepiej wiem myśli / które myślę o was / mówi Pán : myśli o pokoju / á nie o utrápieniu / ábym ucżynił koniec pożądány biedom wáƺym.
Gdy mię wzywáć będźiećie á pójdźiećie / y modlić mi śię będźiećie / tedy was wyſłuchám ;
A ƺukájąc mię / znájdźiećie ; gdy mię ƺukáć będźiećie ze wƺyſtkiego ſercá ſwego /
Dam śię wam zájiſte ználeść / mówi Pán : á przywrócę więźniów wáƺych / y zgromádzę was ze wƺyſtkich narodów / y ze wƺyſtkich miejſc / gdźiemkolwiek was zágnał / mówi Pán ; y przyprowádzę was ná to miejſce / ſkądem was wyprowádźił.
Gdy rzecżećie : Wzbudzał nam Pán proroków prorokujących o záprowádzeniu do Bábilonu.
Bo ták mówi Pán o królu / który śiedźi ná ſtolicy Dawidowej / y o wƺyſtkim ludu / który mieƺká w tym mieśćie / bráćiach wáƺych / którzy nie wyƺli z wámi w tę niewolę ;
Ták mówi Pán zaſtępów : Oto Ja poślę ná nich miecż / głód / y mór / á ucżynię ich jáko złe figi / których nie jádáją / przeto / że ſą złe.
Abowiem prześládowáć ich będę miecżem / głodem y morem / y dam ich ná potłukánie po wƺyſtkich króleſtwách źiemi / ná przeklęſtwo / y ná zdumienie / owƺem / ná poświſtánie / y ná urągánie miedzy wƺyſtkimi národámi / tám / gdźie ich zápędzę /
Przeto / iż nie ſłucháją ſłów mojich / mówi Pán / gdy poſyłám do nich ſług ſwojich / proroków / ráno wſtáwájąc y poſyłájąc ; á nie ſłucháliśćie / mówi Pán.
Przetoż ſłuchájćie ſłowá Páńſkiego wy wƺyſcy pojmáni / którychem wyſłał z Jeruzálemu do Bábilonu.
Ták mówi Pán zaſtępów / Bóg Izráelſki / o Achábie / ſynu Kolájáƺowym / y o Sedekijáƺu / ſynu Máázejáƺowym / którzy wam prorokują w imieniu mojim kłámſtwo : Oto Ja podám ich w rękę Nábuchodonozorá / królá Bábilońſkiego / áby ich pobił przed ocżymá wáƺemi.
Y będźie wźięte z nich przeklinánie miedzy wƺyſtkich pojmánych z Judy / którzy ſą w Bábilonie / áby mówili : Niech ćię ucżyni Pán jáko Sedekijáƺá / y jáko Achábá / których upiekł król Bábilońſki ná ogniu /
Przeto / że popełniáli złość w Izráelu / cudzołożąc z żonámi bliźnich ſwojich / á kłámliwie mówiąc ſłowo w imieniu mojim / cżegom im nie przykázał ; á Ja o tym wiem / y jeſtem tego świádkiem / mówi Pán.
A do Semejáƺá Nechálámity rzecż / mówiąc :
Ták powiádá Pán zaſtępów / Bóg Izráelſki / mówiąc : Przeto żeś ty poſłał imieniem ſwojim liſty do wƺyſtkiego ludu / który jeſt w Jeruzálemie / y do Sofonijáƺá / ſyná Máázejáƺowego / kápłáná / y do wƺyſtkich kápłanów / mówiąc :
Pán ćię dał zá kápłáná miáſto Jojády kápłáná / ábyśćie mieli dozór w domu Páńſkim ná káżdego mężá w rozum záƺłego á prorokującego / ábyś dał tákowego do więźieniá y do kłody.
Przecżżeś tedy teraz nie zgromił Jeremijáƺá Anátotcżyká / który wam prorokuje?
Bo poſłał do nas do Bábilonu / mówiąc : Jeƺcżeć długo cżekáć ; budujćie domy / á oſádzájćie śię ƺcżepćie ſády / á jedzćie owoce ich.
Bo Sofonijáƺ kápłán cżytał ten liſt przed Jeremijáƺem prorokiem.
Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá mówiąc : Poślij do wƺyſtkich zábránych w niewolę / mówiąc : Ták mówi Pán o Semejáƺu Nechálámitcżyku : Przeto / że wam prorokował Semejáƺ / choćiáżem go Ja nie poſłał / á przywodźi was do tego / ábyśćie mieli nádźieję w kłámſtwie ;
Przetoż ták mówi Pán : Oto Ja náwiedzę Semejáƺá Nechálámitcżyká y naśienie jego ; nie będźie miał nikogo / ktoby mieƺkał w pośrzodku ludu tego / áni oglądá tego dobrá / które Ja ucżynię ludowi ſwemu / mówi Pán ; bo rádźił / áby odſtąpił lud od Páná.
