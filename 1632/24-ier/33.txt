Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá powtóre / gdy jeƺcże był zámknięty w śieni ćiemnicy / mówiąc :
Ták mówi Pán / który to ucżyni : Pán / który to utworzy / potwierdzi to / Pán jeſt imię jego.
Wołáj do mnie / á ozwęć śię y oznájmięć rzecży wielkie y ſkryte / o których nie wieƺ.
Abowiem ták mówi Pán / Bóg Izráelſki / o domách miáſtá tego / y o domách królów Judzkich / które pokáżone być máją táránámi wojennemi y miecżem :
Poćiągną / żeby wálcżyli z Cháldejcżykámi / á żeby nápełnili te domy trupámi ludzi / które pobiję w zápálcżywośći mojej y w gniewie moim / zákrywájąc twárz moję od tego miáſtá dlá wƺelákich złośći ich.
Wƺákże Já przywiodę ich do zdrowiá / y ulecżę á uzdrowię ich / y objáwię im obfitość pokoju / á pokoju pewnego.
Bo przywrócę pojmánych z Judy / y pojmánych z Izráelá / á pobuduję ich jáko przedtem ;
Y ocżyƺcżę ich od wƺelkiej niepráwośći ich / którą zgrzeƺyli przećiwko mnie / y przepuƺcżę wƺyſtkim złośćiom ich / któremi zgrzeƺyli przećiwko mnie / y któremi wyſtąpili przećiwko mnie.
A będzie mi to ſłáwą / weſelem / chwáłą / y ozdobą przed wƺyſtkiemi národámi ziemi / które uſłyƺą o wƺyſtkiem dobrem / które Já im cżynię / y ulękną śię / á zátrwożą śię nád wƺyſtkiem dobrem y nád wƺyſtkim pokojem / który Já im ſpoſobię.
Ták mówi Pán : Jeƺcże ſłyƺány będzie ná tem miejſcu / ( o którem wy powiádáćie : Jeſt ſpuſtoƺone / ták / że niemáƺ áni cżłowieká / áni bydlęćiá / w miáſtách Judzkich y ná ulicách Jeruzálemſkich ſpuſtoƺonych / ták / że niemáƺ áni cżłowieká / áni obywátelá / áni bydlęćiá. )
Słyƺány / mówię / będzie głoſ rádośći / y głoſ weſelá / głoſ oblubieńcá / y głoſ oblubienicy / głoſ mówiących : Wyſłáwiájćie Páná záſtępów ; ábowiem dobry jeſt Pán / ábowiem ná wieki miłośierdzie jego ; y głoſ przynoƺących ofiárę chwáły do domu Páńſkiego / gdyż przywrócę pojmánych z tej ziemi / jáko ná pocżątku / mówi Pán.
Ták mówi Pán záſtępów : Jeƺcże będzie ná tem miejſcu puſtem / ná którem niemáƺ áni cżłowieká / áni bydlęćiá / y we wƺyſtkich miáſtách jego mieƺkánie páſterzy / gdzieby chowáli trzody ;
W miáſtách ná górách / w miáſtách ná równinách / y w miáſtách ná południe / y w ziemi Benjáminowej / y około Jeruzálemu / y w miáſtách Judzkich jeƺcże przychodzić będą trzody pod ręką licżącego / mówi Pán.
Oto dni idą / mówi Pán / w których utwierdzę to ſłowo dobre / którem był wyrzekł o domu Izráelſkim y o domu Judzkim.
W one dni w onym cżáśie ucżynię to / iż wyrośnie Dawidowi látorośl ſpráwiedliwá / którá cżynić będzie ſąd y ſpráwiedliwość ná ziemi.
Onych dni będzie zbáwiony Judá / á Jeruzálem bezpiecżnie mieƺkáć będzie. A toć jeſt imię / którem ją názowią : Pán ſpráwiedliwość náƺá.
Bo ták mówi Pán : Nie będzie wykorzeniony mąż z rodu Dawidowego / áby nie miáł śiedzieć ná ſtolicy domu Izráelſkiego.
Z kápłánów też y z Lewitów nie będzie wykorzeniony mąż od oblicżá mego / áby nie miáł ofiárowáć cáłopáleniá / y zápáláć śniednej ofiáry / y ſpráwowáć ofiár po wƺyſtkie dni.
Potem ſtáło śię ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Ták mówi Pán : Jeźli będziećie mogli złámáć przymierze moje ze dniem / y przymierze moje z nocą / áby nie bywáło dniá áni nocy cżáſu ſwego /
Tedy też przymierze moje złámáne będzie z Dawidem / ſługą moim / áby nie miáł ſyná / któryby królowáł ná ſtolicy jego / y z Lewitámi kápłánámi / áby nie byli ſługámi moimi.
A jáko nie może być policżone wojſko niebieſkie / áni zmierzony piáſek morſki / ták rozmnożę náśienie Dawidá / ſługi mojego / y Lewitów / którzy mi ſłużą.
Znowu śię ſtáło ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Azá nie widziƺ / co ten lud powiádá / mówiąc : Że dwá domy / które był Pán obráł / te już odrzućił / á że ludem moim pogárdzáją / jákoby nie był więcej národem przed oblicżem ich?
Ták mówi Pán : Nie będzieli przymierze moje ze dniem y z nocą ſtáłe / á jeźlim pocżątku niebioſ y ziemi nie poſtánowił :
Tedyć y náśienie Jákóbowe y Dawidá ſługi mego odrzucę / ábym nie bráł z náśieniá jego tych / którzyby pánowáć mieli nád náśieniem Abráhámowem / Izáákowem / y Jákóbowem / gdyż przywrócę więźniów ich / á zlituję śię nád nimi.
