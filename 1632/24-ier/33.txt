Y ſtáło śię ſłowo Páńſkie do Jeremijáƺá powtóre / gdy jeƺcże był zámknięty w śieni ćiemnicy / mówiąc :
Ták mówi Pán / który to ucżyni : Pán / który to utworzy / potwierdźi to / Pán jeſt imię jego.
Wołáj do mnie / á ozwęć śię y oznájmięć rzecży wielkie y ſkryte / o których nie wieƺ.
Abowiem ták mówi Pán / Bóg Izráelſki / o domách miáſtá tego / y o domách królów Judzkich / które pokáżone być máją táránámi wojennemi y miecżem :
Poćiągną / żeby wálcżyli z Cháldejcżykámi / á żeby nápełnili te domy trupámi ludźi / które pobiję w zápálcżywośći mojey y w gniewie moim / zákrywájąc twárz moję od tego miáſtá dla wƺelákich złośći ich.
Wƺákże Ja przywiodę ich do zdrowiá / y ulecżę á uzdrowię ich / y objáwię im obfitość pokoju / á pokoju pewnego.
Bo przywrócę pojmánych z Judy / y pojmánych z Izráelá / á pobuduję ich jáko przedtem ;
Y ocżyƺcżę ich od wƺelkiej niepráwośći ich / którą zgrzeƺyli przećiwko mnie / y przepuƺcżę wƺyſtkim złośćiom ich / któremi zgrzeƺyli przećiwko mnie / y któremi wyſtąpili przećiwko mnie.
A będźie mi to ſłáwą / weſelem / chwáłą / y ozdobą przed wƺyſtkiemi národámi źiemi / które uſłyƺą o wƺyſtkiem dobrem / które Ja im cżynię / y ulękną śię / á zátrwożą śię nád wƺyſtkiem dobrem y nád wƺyſtkim pokojem / który Ja im ſpoſobię.
Ták mówi Pán : Jeƺcże ſłyƺány będźie ná tem miejſcu / ( o którem wy powiádáćie : Jeſt ſpuſtoƺone / ták / że niemáƺ áni cżłowieká / áni bydlęćiá / w miáſtách Judzkich y ná ulicách Jeruzálemſkich ſpuſtoƺonych / ták / że niemáƺ áni cżłowieká / áni obywátelá / áni bydlęćiá. )
Słyƺány / mówię / będźie głoſ rádośći / y głoſ weſelá / głoſ oblubieńcá / y głoſ oblubienicy / głoſ mówiących : Wyſłáwiájćie Páná záſtępów ; ábowiem dobry jeſt Pán / ábowiem ná wieki miłośierdźie jego ; y głoſ przynoƺących ofiárę chwáły do domu Páńſkiego / gdyż przywrócę pojmánych z tey źiemi / jáko ná pocżątku / mówi Pán.
Ták mówi Pán záſtępów : Jeƺcże będźie ná tem miejſcu puſtem / ná którem niemáƺ áni cżłowieká / áni bydlęćiá / y we wƺyſtkich miáſtách jego mieƺkánie páſterzy / gdźieby chowáli trzody ;
W miáſtách ná górách / w miáſtách ná równinách / y w miáſtách ná południe / y w źiemi Benjáminowej / y około Jeruzálemu / y w miáſtách Judzkich jeƺcże przychodźić będą trzody pod ręką licżącego / mówi Pán.
Oto dni idą / mówi Pán / w których utwierdzę to ſłowo dobre / którem był wyrzekł o domu Izráelſkim y o domu Judzkim.
W one dni w onym cżáśie ucżynię to / iż wyrośnie Dawidowi látorośl ſpráwiedliwá / która cżynić będźie ſąd y ſpráwiedliwość ná źiemi.
Onych dni będźie zbáwiony Judá / á Jeruzálem bezpiecżnie mieƺkáć będźie. A toć jeſt imię / którem ją názowią : Pán ſpráwiedliwość náƺá.
Bo ták mówi Pán : Nie będźie wykorzeniony mąż z rodu Dawidowego / áby nie miał śiedźieć ná ſtolicy domu Izráelſkiego.
Z kápłánów też y z Lewitów nie będźie wykorzeniony mąż od oblicżá mego / áby nie miał ofiárowáć cáłopáleniá / y zápáláć śniedney ofiáry / y ſpráwowáć ofiár po wƺyſtkie dni.
Potem ſtáło śię ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Ták mówi Pán : Jeſli będźiećie mogli złámáć przymierze moje ze dniem / y przymierze moje z nocą / áby nie bywáło dniá áni nocy cżáſu ſwego /
Tedy też przymierze moje złámáne będźie z Dawidem / ſługą moim / áby nie miał ſyná / któryby królowáł ná ſtolicy jego / y z Lewitámi kápłánámi / áby nie byli ſługámi moimi.
A jáko nie może być policżone wojſko niebieſkie / áni zmierzony piáſek morſki / ták rozmnożę náśienie Dawidá / ſługi mojego / y Lewitów / którzy mi ſłużą.
Znowu śię ſtáło ſłowo Páńſkie do Jeremijáƺá / mówiąc :
Azá nie widźiƺ / co ten lud powiádá / mówiąc : Że dwá domy / które był Pán obráł / te już odrzućił / á że ludem moim pogárdzáją / jákoby nie był więcey národem przed oblicżem ich?
Ták mówi Pán : Nie będźieli przymierze moje ze dniem y z nocą ſtáłe / á jeſlim pocżątku niebioſ y źiemi nie poſtánowił :
Tedyć y náśienie Jákóbowe y Dawidá ſługi mego odrzucę / ábym nie bráł z náśieniá jego tych / którzyby pánowáć mieli nád náśieniem Abráhámowem / Izáákowem / y Jákóbowem / gdyż przywrócę więźniów ich / á zlituję śię nád nimi.
