Y powiedźiał Mojzeƺ ſynom Izráelſkim to wƺyſtko / co rozkázał Pán Mojżeƺowi.
Potem mówił Mojzeƺ do kśiążąt w pokoleniách miedzy ſynámi Izráelſkimi / y rzekł : Toć jeſt / co rozkázał Pán.
Jeſliby mąż poślubił ślub Pánu / álbo też przyśięgę ucżynił / obowiązkiem obowiązáwƺy duƺę ſwoję / nie złámie ſłowá ſwego : według wƺyſtkiego / coby wyƺło z uſt jego / ucżyni.
Ale jeſliby niewiáſtá poślubiłá ślub Pánu / y obowiązáłáby śię obowiązkiem w domu ojcá ſwego w młodośći ſwojey ;
A ſłyƺáłby oćiec jey on ślub jey / y obowiązek jey / którym obowiązáłá duƺę ſwoję / á milcżáłby ná to oćiec jey / tedy płátne będą wƺyſtkie śluby jey / y káżdy obowiązek / którymby obowiązáłá duƺę ſwą / płátny będźie.
Ale jeſliby był onegoż dniá przećiw temu oćiec jey / któregoby ſłyƺał wƺyſtkie śluby jey / y obowiązki jey / któremi obowiązáłá duƺę ſwoję / nie będą płátne ; Pán odpuśći jey / bo był przećiw temu oćiec jey.
Ale gdyby májącá mężá ślub jáki ucżyniłá / álbo wyrzekłá co uſty ſwemi / cżemby obowiązáłá duƺę ſwoję ;
A ſłyƺąc to mąż jey / milcżáłby ná to onegoż dniá / którego ſłyƺał / płátne będą śluby jey / y obowiązki jey / któremi obowiązáłá duƺę ſwoję / płátne będą.
Ale jeſliby onego dniá / którego ſłyƺał mąż jey / ſprzećiwił śię temu / y wzruƺyłby ślub jey / który ná ſobie miáłá / y co wymówiłá uſty ſwemi / cżym obowiązáłá duƺę ſwoję / tákże Pán odpuśći jey.
Ale ślub káżdy wdowy / y odrzuconey / którymby obowiązáłá duƺę ſwoję / płátny będźie.
Lecż jeſliby / póki byłá w domu mężá ſwego / ślub ucżyniłá / y obowiązáłá obowiązkiem duƺę ſwoję z przyśięgą ;
A ſłyƺąc mąż jey milcżáłby ná to / y nie ſprzećiwiłby śię temu / tedy płátne będą wƺyſtkie śluby jey y káżdy obowiązek / którym obowiązáłá duƺę ſwoję / płátny będźie.
Ale jeſli cále ſprzećiwił śię temu mąż jey dniá / którego to ſłyƺał / wƺelki ślub / który wyƺedł z uſt jey / y obowiązek duƺy jey / nie będźie płátny ; mąż jey wzruƺył to / á Pán odpuśći jey.
Wƺelkiego ślubu y wƺelkiej przyśięgi obowiązku ná utrápienie duƺy / mąż jey potwierdźi go / y mąż jey wzruƺy go.
A jeſliby cále milcżał mąż jey ode dniá do dniá / tedy tym ſtwierdźi wƺyſtkie śluby jey / y wƺyſtkie obowiązki jey / które ma ná ſobie ; ſtwierdźi je / przeto że milcżał ná to w dźień / którego ſłyƺał ;
A jeſliby to koniecżnie wzruƺyć chćiał / nie záráz gdy ſłyƺał / ále potem / ponieśie niepráwość jey.
Teć ſą uſtáwy / które przykázał Pán Mojżeƺowi / miedzy mężem á żoną jego / miedzy ojcem á córką jego w młodośći jey / póki jeſt w domu ojcá ſwego.
