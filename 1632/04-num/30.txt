Y powiedźiáł Mojżeƺ ſynom Izráelſkim to wƺyſtko / co rozkázáł Pán Mojżeƺowi.
Potem mówił Mojżeƺ do kśiążąt w pokoleniách między ſynámi Izráelſkimi / y rzekł : Toć jeſt / co rozkázáł Pán.
Jeźliby mąż poślubił ślub Pánu / álbo też przyśięgę ucżynił / obowiązkiem obowiązáwƺy duƺę ſwoję / nie złámie ſłowá ſwego : według wƺyſtkiego / coby wyƺło z uſt jego / ucżyni.
Ale jeźliby niewiáſtá poślubiłá ślub Pánu / y obowiązáłáby śię obowiązkiem w domu ojcá ſwego w młodośći ſwojej ;
A ſłyƺáłby ojćiec jej on ślub jej / y obowiązek jej / którym obowiązáłá duƺę ſwoję / á milcżáłby ná to ojćiec jej / tedy płátne będą wƺyſtkie śluby jej / y káżdy obowiązek / którymby obowiązáłá duƺę ſwą / płátny będźie.
Ale jeźliby był onegoż dniá przećiw temu ojćiec jej / któregoby ſłyƺáł wƺyſtkie śluby jej / y obowiązki jej / któremi obowiązáłá duƺę ſwoję / nie będą płátne ; Pán odpuśći jej / bo był przećiw temu ojćiec jej.
Ale gdyby májącá mężá ślub jáki ucżyniłá / álbo wyrzekłá co uſty ſwemi / cżemby obowiązáłá duƺę ſwoję ;
A ſłyƺąc to mąż jej / milcżáłby ná to onegoż dniá / którego ſłyƺáł / płátne będą śluby jej / y obowiązki jej / któremi obowiązáłá duƺę ſwoję / płátne będą.
Ale jeźliby onego dniá / którego ſłyƺáł mąż jej / ſprzećiwił śię temu / y wzruƺyłby ślub jej / który ná ſobie miáłá / y co wymówiłá uſty ſwemi / cżem obowiązáłá duƺę ſwoję / tákże Pán odpuśći jej.
Ale ślub káżdy wdowy / y odrzuconej / którymby obowiązáłá duƺę ſwoję / płátny będźie.
Lecż jeźliby / póki byłá w domu mężá ſwego / ślub ucżyniłá / y obowiązáłá obowiązkiem duƺę ſwoję z przyśięgą ;
A ſłyƺąc mąż jej milcżáłby ná to / y nie ſprzećiwiłby śię temu / tedy płátne będą wƺyſtkie śluby jej y káżdy obowiązek / którym obowiązáłá duƺę ſwoję / płátny będźie.
Ale jeźli cále ſprzećiwił śię temu mąż jej dniá / którego to ſłyƺáł / wƺelki ślub / który wyƺedł z uſt jej / y obowiązek duƺy jej / nie będźie płátny ; mąż jej wzruƺył to / á Pán odpuśći jej.
Wƺelkiego ślubu y wƺelkiej przyśięgi obowiązku ná utrápienie duƺy / mąż jej potwierdźi go / y mąż jej wzruƺy go.
A jeźliby cále milcżáł mąż jej ode dniá do dniá / tedy tem ſtwierdźi wƺyſtkie śluby jej / y wƺyſtkie obowiązki jej / które má ná ſobie ; ſtwierdźi je / przeto że milcżáł ná to w dźień / którego ſłyƺáł ;
A jeźliby to koniecżnie wzruƺyć chćiáł / nie záráz gdy ſłyƺáł / ále potem / ponieśie niepráwość jej.
Teć ſą uſtáwy / które przykázáł Pán Mojżeƺowi / między mężem á żoną jego / między ojcem á córką jego w młodośći jej / póki jeſt w domu ojcá ſwego.
