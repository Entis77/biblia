Y powiedźiáł Mojżeƺ ſynom Izráelſkim to wƺyſtko / co rozkázáł Pán Mojżeƺowi.
Potem mówił Mojżeƺ do kśiążąt w pokoleniách między ſynámi Izráelſkimi / y rzekł : Toć jeſt / co rozkázáł Pán.
Jeſliby mąż poślubił ślub Pánu / álbo też przyśięgę ucżynił / obowiązkiem obowiązáwƺy duƺę ſwoję / nie złámie ſłowá ſwego : według wƺyſtkiego / coby wyƺło z uſt jego / ucżyni.
Ale jeſliby niewiáſtá poślubiłá ślub Pánu / y obowiązáłáby śię obowiązkiem w domu ojcá ſwego w młodośći ſwojey ;
A ſłyƺáłby ojćiec jey on ślub jey / y obowiązek jey / którym obowiązáłá duƺę ſwoję / á milcżáłby ná to ojćiec jey / tedy płátne będą wƺyſtkie śluby jey / y káżdy obowiązek / którymby obowiązáłá duƺę ſwą / płátny będźie.
Ale jeſliby był onegoż dniá przećiw temu ojćiec jey / któregoby ſłyƺáł wƺyſtkie śluby jey / y obowiązki jey / któremi obowiązáłá duƺę ſwoję / nie będą płátne ; Pán odpuśći jey / bo był przećiw temu ojćiec jey.
Ale gdyby májącá mężá ślub jáki ucżyniłá / álbo wyrzekłá co uſty ſwemi / cżemby obowiązáłá duƺę ſwoję ;
A ſłyƺąc to mąż jey / milcżáłby ná to onegoż dniá / którego ſłyƺáł / płátne będą śluby jey / y obowiązki jey / któremi obowiązáłá duƺę ſwoję / płátne będą.
Ale jeſliby onego dniá / którego ſłyƺáł mąż jey / ſprzećiwił śię temu / y wzruƺyłby ślub jey / który ná ſobie miáłá / y co wymówiłá uſty ſwemi / cżem obowiązáłá duƺę ſwoję / tákże Pán odpuśći jey.
Ale ślub káżdy wdowy / y odrzuconey / którymby obowiązáłá duƺę ſwoję / płátny będźie.
Lecż jeſliby / póki byłá w domu mężá ſwego / ślub ucżyniłá / y obowiązáłá obowiązkiem duƺę ſwoję z przyśięgą ;
A ſłyƺąc mąż jey milcżáłby ná to / y nie ſprzećiwiłby śię temu / tedy płátne będą wƺyſtkie śluby jey y káżdy obowiązek / którym obowiązáłá duƺę ſwoję / płátny będźie.
Ale jeſli cále ſprzećiwił śię temu mąż jey dniá / którego to ſłyƺáł / wƺelki ślub / który wyƺedł z uſt jey / y obowiązek duƺy jey / nie będźie płátny ; mąż jey wzruƺył to / á Pán odpuśći jey.
Wƺelkiego ślubu y wƺelkiej przyśięgi obowiązku ná utrápienie duƺy / mąż jey potwierdźi go / y mąż jey wzruƺy go.
A jeſliby cále milcżáł mąż jey ode dniá do dniá / tedy tem ſtwierdźi wƺyſtkie śluby jey / y wƺyſtkie obowiązki jey / które má ná ſobie ; ſtwierdźi je / przeto że milcżáł ná to w dźień / którego ſłyƺáł ;
A jeſliby to koniecżnie wzruƺyć chćiáł / nie záráz gdy ſłyƺáł / ále potem / ponieśie niepráwość jey.
Teć ſą uſtáwy / które przykázáł Pán Mojżeƺowi / między mężem á żoną jego / między ojcem á córką jego w młodośći jey / póki jeſt w domu ojcá ſwego.
