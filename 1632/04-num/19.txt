Y rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Táć jeſt uſtáwá zakonu / którą rozkázáł Pán / mówiąc : Powiedz ſynom Izráelſkim / áby przywiedli do ćiebie jáłowicę płową / zdrową / ná którejby nie było zmázy / y ná którejby nie poſtáło járzmo ;
Tę oddáćie Eleázárowi kápłánowi / który ją wywiedźie zá obóz / y zábitá będźie przed nim.
A wźiąwƺy Eleázár kápłán ze krwi jey ná pálec ſwój / kropić będźie przećiw námiotowi zgromádzeniá oną krwią śiedm kroć.
Potem káże tę jáłowicę ſpálić przed ocżymá ſwemi ; ſkórę jey / y mięſo jey / y krew jey / z gnojem jey ſpáli.
Y weźmie kápłán drzewo cedrowe / y hizop / y kármázyn dwá kroć fárbowány / á wrzući do ogniá / gdźie śię jáłowicá páli ;
Y upierze ƺáty ſwe kápłán / á omyje ćiáło ſwoje wodą ; á potem wnijdźie do obozu / y będźie niecżyſtym kápłán áż do wiecżorá.
Ten też / który ją pálić będźie / upierze ƺáty ſwe w wodźie / y omyje ćiáło ſwe wodą / á będźie niecżyſtym áż do wiecżorá.
Y zbierze cżłowiek cżyſty popiół oney jáłowicy / y wyſypie go precż zá obóz / ná miejſcu cżyſtem / á będźie dla zgromádzeniá ſynów Izráelſkich chowány do wody ocżyƺcżeniá / gdyż jeſt ofiárá zá grzech.
Y upierze ten / co będźie zbieráł popiół oney jáłowicy / ƺáty ſwe / y będźie niecżyſtym áż do wiecżorá. A będźie to ſynom Izráelſkim / y przychodniowi mieƺkájącemu między nimi / uſtáwą wiecżną.
Ktoby śię dotknął jákiegokolwiek trupá cżłowiecżego / niecżyſtym będźie przez śiedm dni ;
Táki ocżyƺcżáć śię będźie tą wodą dniá trzećiego y dniá śiódmego / á cżyſtym będźie ; á jeſliby śię nie ocżyśćił dniá trzećiego y dniá śiódmego / niecżyſtym będźie.
Ktoby śię kolwiek dotknął mártwego ćiáłá cżłowieká / który umárł / á nie ocżyśćił śię / przybytek Páńſki ſplugáwił ; przetoż tákowy wytrácony będźie z Izráelá / bo wodą ocżyƺcżeniá nie był pokropiony ; niecżyſtym będźie / niecżyſtość jego zoſtánie ná nim.
Tá záś jeſt uſtáwá : gdyby cżłowiek umárł w námioćie / ktobykolwiek wƺedł do tego námiotu / y cokolwiek było w onym námioćie / niecżyſte będźie przez śiedm dni.
Tákże wƺelkie nácżynie odkryte / któreby nie miáło nákryćiá ná ſobie / niecżyſte będźie.
Tákże ktoby śię kolwiek dotknął ná polu / bądź miecżem zábitego / bądź umárłego / bądź kośći cżłowiecżej / álbo grobu / niecżyſtym będźie przez śiedm dni.
Wezmą tedy dla onego niecżyſtego popiołu jáłowicy ſpáloney zá grzech / y náleją náń wody żywej do nácżyniá.
Weźmie też hizopu / y omocży go w oney wodźie cżłowiek cżyſty / y pokropi námiot / y wƺyſtko nácżynie / y wƺyſtkich ludźi / którzyby tám byli / tákże onego / który śię dotknął kośći / álbo zábitego / álbo zmárłego / álbo grobu ;
Pokropi tedy cżyſty niecżyſtego dniá trzećiego y dniá śiódmego ; á gdy go ocżyśći dniá śiódmego / tedy upierze ƺáty ſwe / y omyje śię wodą / á będźie cżyſtym w wiecżór.
A mąż / któryby niecżyſtym będąc nie ocżyśćił śię / wykorzenioná będźie tá duƺá z pośrodku zgromádzeniá / bo świątnicę Páńſką ſplugáwił ; wodą ocżyƺcżeniá nie będąc pokropionym / niecżyſtym jeſt.
Y będźie im to zá uſtáwę wiecżną ; á kto będźie pokrápiáł wodą ocżyƺcżeniá / upierze ƺáty ſwoje ; á ktoby śię dotknął tey wody ocżyƺcżeniá / niecżyſtym będźie áż do wiecżorá.
Cżegokolwiek śię dotknie niecżyſty / niecżyſte będźie ; cżłowiek tákże / któryby śię tego dotknął / niecżyſty będźie áż do wiecżorá.
