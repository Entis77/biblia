Y rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Táć jeſt uſtáwá zákonu / którą rozkázáł Pán / mówiąc : Powiedz ſynom Izráelſkim / áby przywiedli do ćiebie jáłowicę płową / zdrową / ná którejby nie było zmázy / y ná którejby nie poſtáło járzmo ;
Tę oddáćie Eleázárowi kápłánowi / który ją wywiedzie zá obóz / y zábitá będzie przed nim.
A wziąwƺy Eleázár kápłán ze krwi jej ná pálec ſwój / kropić będzie przećiw námiotowi zgromádzeniá oną krwią śiedm kroć.
Potem káże tę jáłowicę ſpálić przed ocżymá ſwemi ; ſkórę jej / y mięſo jej / y krew jej / z gnojem jej ſpáli.
Y weźmie kápłán drzewo cedrowe / y hizop / y kármázyn dwá kroć fárbowány / á wrzući do ogniá / gdzie śię jáłowicá páli ;
Y upierze ƺáty ſwe kápłán / á omyje ćiáło ſwoje wodą ; á potem wnijdzie do obozu / y będzie niecżyſtym kápłán áż do wiecżorá.
Ten też / który ją pálić będzie / upierze ƺáty ſwe w wodzie / y omyje ćiáło ſwe wodą / á będzie niecżyſtym áż do wiecżorá.
Y zbierze cżłowiek cżyſty popiół onej jáłowicy / y wyſypie go precż zá obóz / ná miejſcu cżyſtem / á będzie dlá zgromádzeniá ſynów Izráelſkich chowány do wody ocżyƺcżeniá / gdyż jeſt ofiárá zá grzech.
Y upierze ten / co będzie zbieráł popiół onej jáłowicy / ƺáty ſwe / y będzie niecżyſtym áż do wiecżorá. A będzie to ſynom Izráelſkim / y przychodniowi mieƺkájącemu między nimi / uſtáwą wiecżną.
Ktoby śię dotknął jákiegokolwiek trupá cżłowiecżego / niecżyſtym będzie przez śiedm dni ;
Táki ocżyƺcżáć śię będzie tą wodą dniá trzećiego y dniá śiódmego / á cżyſtym będzie ; á jeźliby śię nie ocżyśćił dniá trzećiego y dniá śiódmego / niecżyſtym będzie.
Ktoby śię kolwiek dotknął mártwego ćiáłá cżłowieká / który umárł / á nie ocżyśćił śię / przybytek Páńſki ſplugáwił ; przetoż tákowy wytrácony będzie z Izráelá / bo wodą ocżyƺcżeniá nie był pokropiony ; niecżyſtym będzie / niecżyſtość jego zoſtánie ná nim.
Tá záś jeſt uſtáwá : gdyby cżłowiek umárł w námioćie / ktobykolwiek wƺedł do tego námiotu / y cokolwiek było w onym námioćie / niecżyſte będzie przez śiedm dni.
Tákże wƺelkie nácżynie odkryte / któreby nie miáło nákryćiá ná ſobie / niecżyſte będzie.
Tákże ktoby śię kolwiek dotknął ná polu / bądź miecżem zábitego / bądź umárłego / bądź kośći cżłowiecżej / álbo grobu / niecżyſtym będzie przez śiedm dni.
Wezmą tedy dlá onego niecżyſtego popiołu jáłowicy ſpálonej zá grzech / y náleją náń wody żywej do nácżyniá.
Weźmie też hizopu / y omocży go w onej wodzie cżłowiek cżyſty / y pokropi námiot / y wƺyſtko nácżynie / y wƺyſtkich ludzi / którzyby tám byli / tákże onego / który śię dotknął kośći / álbo zábitego / álbo zmárłego / álbo grobu ;
Pokropi tedy cżyſty niecżyſtego dniá trzećiego y dniá śiódmego ; á gdy go ocżyśći dniá śiódmego / tedy upierze ƺáty ſwe / y omyje śię wodą / á będzie cżyſtym w wiecżór.
A mąż / któryby niecżyſtym będąc nie ocżyśćił śię / wykorzenioná będzie tá duƺá z pośrodku zgromádzeniá / bo świątnicę Páńſką ſplugáwił ; wodą ocżyƺcżeniá nie będąc pokropionym / niecżyſtym jeſt.
Y będzie im to zá uſtáwę wiecżną ; á kto będzie pokrápiáł wodą ocżyƺcżeniá / upierze ƺáty ſwoje ; á ktoby śię dotknął tej wody ocżyƺcżeniá / niecżyſtym będzie áż do wiecżorá.
Cżegokolwiek śię dotknie niecżyſty / niecżyſte będzie ; cżłowiek tákże / któryby śię tego dotknął / niecżyſty będzie áż do wiecżorá.
