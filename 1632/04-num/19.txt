Y rzekł PAN do Mojzeƺá y do Aároná / mówiąc :
Táć jeſt uſtáwá zakonu którą rozkazał PAN / mówiąc : Powiedz Synom Izráelſkim / áby przywiedli do ćiebie jáłowicę płową / zdrową / ná któreyby nie było zmázy / y ná któreyby nie poſtáło járzmo :
Tę oddaćie Eleázárowi Kápłanowi / który ją wywiedźie zá Obóz / y zábita będźie przed nim :
A wźiąwƺy Eleázár Kápłan ze krwie jey ná pálec ſwój / kropić będźie przećiw Namiotowi zgromádzenia oną krwią / śiedm kroć.
Potym káże tę jáłowicę ſpalić przed oczymá ſwymi : ſkórę jey / y mięſo jey / y krew jey / z gnojem jey ſpali :
Y weźmie Kápłan drzewo Cedrowe / y Hyſop / y kármázyn dwá kroć fárbowány / á wrzući do Ogniá / gdźie śię jáłowicá pali :
Y upierze ƺáty ſwe Kápłan / á omyje ćiáło ſwoje wodą / á potym wnidźie do Obozu / y będźie nieczyſtym Kápłan / áż do wieczorá.
Ten też który ją palić będźie / upierze ƺáty ſwe w wodźie / y omyje ćiáło ſwe wodą / á będźie nieczyſtym áż do wieczorá.
Y zbierze człowiek czyſty / popiół oney jáłowice / y wyſypie go precz zá Obóz / ná miejſcu czyſtym / á będźie dla zgromádzenia Synów Izráelſkich chowány / do wody oczyśćiánia / <i>gdyż</i> jeſt ofiárá zá grzech.
Y upierze / ten co będźie zbierał popiół oney jáłowice ƺáty ſwe / y będźie nieczyſtym / áż do wieczorá : A będźie to Synom Izráelſkim / y przychodniowi mieƺkájącemu miedzy nimi / uſtáwą wieczną.
Ktoby śię dotknął jákiegokolwiek trupá człowieczego / nieczyſtym będźie przez śiedm dni :
Táki oczyśćiáć śię będźie tą <i>wodą,</i> dniá trzećiego / y dniá śiódmego / <i>á</i> czyſtym będźie : á jeſliby śię nie oczyśćił dniá trzećiego / y dniá śiódmego / nieczyſtym będźie.
Ktoby śię kolwiek dotknął martwego ćiáłá / człowieká który umárł / á nie oczyśćił śię / Przybytek PAńſki ſplugáwił : przetoż tákowy wytrácony będźie z Izráelá / bo wodą oczyśćiánia nie był pokropiony : nieczyſtym będźie / nieczyſtość jego zoſtánie ná nim.
Tá <i>záś jeſt</i> uſtáwá / gdyby człowiek umárł w Namiećie / ktobykolwiek wƺedł do tego Namiotu / y cokolwiek <i>było</i> w onym Namiećie / nieczyſte będźie przez śiedm dni.
Tákże wƺelkie naczynie odkryte / któreby nie miáło nákryćia ná ſobie / nieczyſte będźie.
Tákże ktoby śię kolwiek dotknął ná polu / <i>bądź</i> mieczem zábitego / bądź umárłego / bądź kośći człowieczey / álbo grobu / nieczyſtym będźie przez śiedm dni.
Wezmą tedy dla <i>onego</i> nieczyſtego / popiołu <i>jáłowice</i> ſpaloney zá grzech / y náleją nań wody żywey do naczynia :
Weźmie też Hyſopu / y omoczy <i>go</i> w oney wodźie / człowiek czyſty / y pokropi Namiot / y wƺyſtko naczynie / y wƺyſtkie ludźie którzyby tám byli / tákże onego który śię dotknął kośći / álbo zábitego / álbo zmárłego / álbo grobu :
Pokropi tedy czyſty nieczyſtego / dniá trzećiego / y dniá śiódmego : á gdy go oczyśći dniá śiódmego / tedy upierze ƺáty ſwe / y omyje śię wodą / á będźie czyſtym wieczór.
A mąż któryby nieczyſtym będąc / nie oczyśćił śię / wykorzeniona będźie tá duƺá z pośrzodku zgromádzenia / bo świątnicę PAńſką ſplugáwił : wodą oczyścienia nie będąc pokropionym / nieczyſtym jeſt.
Y będźie im to zá uſtáwę wieczną : á kto będźie pokrapiał wodą oczyśćienia / upierze ƺáty ſwoje : á ktoby śię dotknął tey wody oczyśćienia / nieczyſtym będźie áż do wieczorá.
Czegokolwiek śię dotknie nieczyſty / nieczyſte będźie : człowiek tákże któryby śię <i>tego</i> dotknął / nieczyſty będźie áż do wieczorá.
