Y przyƺło wƺyſtko mnóſtwo ſynów Izráelſkich ná puƺcżą Syn / mieśiącá pierwƺego ; y mieƺkał lud w Kádes ; gdźie umárłá Máryjá / y támże jeſt pogrzebioná.
A gdy lud niemiał wody / zebráli śię przećiw Mojżeƺowi / y przećiw Aáronowi.
Y ſwárzył śię lud z Mojżeƺem / y rzekli mówiąc : Obyſmy byli pomárli / gdy pomárli bráćia náśi przed Pánem.
Y przecżżeśćie záwiedli to zgromádzenie Páńſkie ná tę puƺcżą / ábyſmy tu pomárli / my y dobytki náƺe?
A pocóżeśćie nas wywiedli z Egiptu / ábyśćie nas wprowádźili ná to złe miejſce / ná miejſce / ná którem śię nie rodźi áni zboże / áni figi / áni groná winne / áni jábłká gránátowe ; náwet wody nie maƺ dla nápoju?
Tedy odƺedł Mojzeƺ y Aáron od ludu do drzwi namiotu zgromádzenia / y upádli ná oblicża ſwoje ; y ukázáłá śię chwałá Páńſka nád nimi.
Y rzekł Pán do Mojżeƺá / mówiąc :
Weźmij láſkę / á zgromádźiwƺy wƺyſtek lud / ty y Aáron / brát twój / mówćie do tey ſkáły przed ocżymá ich / á wydá wodę ſwą ; y wywiedźieƺ im wodę z ſkáły / á dáƺ nápój temu mnóſtwu / y bydłu ich.
Tedy wźiął Mojzeƺ láſkę przed oblicżem Páńſkim / jáko mu rozkázał.
Y zgromádźił Mojzeƺ z Aáronem wƺyſtek lud przed ſkáłę / y mówił do nich : Słuchájćież teraz ludźie odporni / izali z tey ſkáły możemy wam wywieść wodę?
Zátem podnióſł Mojzeƺ rękę ſwoję / y uderzył w ſkáłę láſką ſwą dwá kroć / á wyƺły wody obfite y piło ono mnóſtwo / y bydło ich.
Y rzekł Pán do Mojżeƺá y do Aároná : Dla tego / żeśćie mi nie uwierzyli / ábyśćie mię poświęćili przed ocżymá ſynów Izráelſkich / przetoż nie wprowádźićie zgromádzenia tego do źiemi / którąm im dał.
Teć ſą wody poſwárku / o które śię ſwárzyli ſynowie Izráelſcy z Pánem / y poświęcony jeſt w nich.
Potem poſłał Mojzeƺ poſły z Kádes do królá Edomſkiego / mówiąc : Ták ći kázał powiedźieć brát twój Izráel : Ty wieƺ o wƺyſtkich trudnośćiách / które przyƺły ná nas ;
Jáko zſtąpili byli ojcowie náśi do Egiptu / y mieƺkáliśmy w Egipćie przez wiele lat ; y jáko nas trápili Egipcżánie / y ojce náƺe ;
Y wołáliśmy do Páná / á wyſłuchał głos náƺ / y poſłáwƺy Aniołá / wywiódł nas z Egiptu ; á otośmy już w Kádes / mieśćie przy gránicy twojey.
Niech / proƺę / przejdźiemy przez źiemię twoję ; nie pójdźiemy przez polá / áni przez winnice / áni będźiemy pić wód z twojich ſtudźień ; gośćińcem pójdźiemy / nie uchylimy śię ná práwo áni ná lewo / áż przejdźiemy gránice twoje.
Ná to odpowiedźiał mu Edom : Nie pójdźieƺ przez moję źiemię / bym ſnadź z miecżem nie wyƺedł przećiw tobie.
Y rzekli mu ſynowie Izráelſcy : Bitym gośćińcem pójdźiemy / á jeſlibyſmy wody twoje pili / my y bydłá náƺe / zápłáćimyć je ; nic innego nie żądámy / tylko ábyſmy pieƺo przeƺli.
Y powiedźiał : Nie przejdźieƺ. Y ruƺył śię Edom przećiwko nim / z wojſkiem wielkiem / y możną ręką.
A gdy nie chćiał Edom pozwolić Izráelowi przejśćiá przez gránice ſwoje / tedy Izráel ucżynił odwrót od niego.
A ruƺywƺy śię z Kádes ; przyƺli ſynowie Izráelſcy y wƺyſtko zgromádzenie do góry Hor.
Y rzekł Pán do Mojżeƺá y do Aároná ná górze Hor / nád gránicą źiemi Edomſkiej / mówiąc :
Będźie Aáron przyłącżon do ludu ſwego ; ábowiem nie wnijdźie do źiemi / którąm dał ſynom Izráelſkim / przeto żeśćie odpornymi byli ſłowu mojemu przy wodách poſwárku.
Weźmijże Aároná y Eleázárá ſyná jego / á káż im wſtąpić ná górę Hor ;
Y zewlecż Aároná z ƺát jego / á oblecż w nie Eleázárá / ſyná jego ; bo Aáron przyłącżon będźie do ludu ſwego / y tám umrze.
Y ucżynił Mojzeƺ / jáko rozkázał Pán ; y wſtąpili ná górę Hor przed ocżymá wƺyſtkiego zgromádzenia.
Y zewlókł Mojzeƺ Aároná z ƺát jego / á oblekł w nie Eleázárá ſyná jego ; y umárł tám Aáron ná wierzchu góry / á Mojzeƺ z Eleázárem zſtąpili z góry.
Widząc tedy wƺyſtko zgromádzenie / iż Aáron umárł / płákáli Aároná przez trzydźieśći dni / wƺyſtek dom Izráelſki.
