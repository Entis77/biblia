Y przyƺło wƺyſtko mnóſtwo ſynów Izráelſkich ná puƺcżą Syn / mieśiącá pierwƺego ; y mieƺkáł lud w Kádeſ ; gdzie umárłá Máryjá / y támże jeſt pogrzebioná.
A gdy lud niemiáł wody / zebráli śię przećiw Mojżeƺowi / y przećiw Aáronowi.
Y ſwárzył śię lud z Mojżeƺem / y rzekli mówiąc : Obyśmy byli pomárli / gdy pomárli bráćiá náśi przed Pánem.
Y przecżżeśćie záwiedli to zgromádzenie Páńſkie ná tę puƺcżą / ábyśmy tu pomárli / my y dobytki náƺe?
A pocóżeśćie náſ wywiedli z Egiptu / ábyśćie náſ wprowádzili ná to złe miejſce / ná miejſce / ná którem śię nie rodzi áni zboże / áni figi / áni groná winne / áni jábłká gránátowe ; náwet wody nie máƺ dlá nápoju?
Tedy odƺedł Mojżeƺ y Aáron od ludu do drzwi námiotu zgromádzeniá / y upádli ná oblicżá ſwoje ; y ukázáłá śię chwáłá Páńſká nád nimi.
Y rzekł Pán do Mojżeƺá / mówiąc :
Weźmij láſkę / á zgromádziwƺy wƺyſtek lud / ty y Aáron / brát twój / mówćie do tej ſkáły przed ocżymá ich / á wydá wodę ſwą ; y wywiedzieƺ im wodę z ſkáły / á dáƺ nápój temu mnóſtwu / y bydłu ich.
Tedy wziął Mojżeƺ láſkę przed oblicżem Páńſkim / jáko mu rozkázáł.
Y zgromádził Mojżeƺ z Aáronem wƺyſtek lud przed ſkáłę / y mówił do nich : Słuchájćież teráz ludzie odporni / izáli z tej ſkáły możemy wám wywieść wodę?
Zátem podnióſł Mojżeƺ rękę ſwoję / y uderzył w ſkáłę láſką ſwą dwá kroć / á wyƺły wody obfite y piło ono mnóſtwo / y bydło ich.
Y rzekł Pán do Mojżeƺá y do Aároná : Dlá tego / żeśćie mi nie uwierzyli / ábyśćie mię poświęćili przed ocżymá ſynów Izráelſkich / przetoż nie wprowádzićie zgromádzeniá tego do ziemi / którąm im dáł.
Teć ſą wody poſwárku / o które śię ſwárzyli ſynowie Izráelſcy z Pánem / y poświęcony jeſt w nich.
Potem poſłáł Mojżeƺ poſły z Kádeſ do królá Edomſkiego / mówiąc : Ták ći kázáł powiedzieć brát twój Izráel : Ty wieƺ o wƺyſtkich trudnośćiách / które przyƺły ná náſ ;
Jáko zſtąpili byli ojcowie náśi do Egiptu / y mieƺkáliśmy w Egipćie przez wiele lát ; y jáko náſ trápili Egipcżánie / y ojce náƺe ;
Y wołáliśmy do Páná / á wyſłucháł głoſ náƺ / y poſłáwƺy Aniołá / wywiódł náſ z Egiptu ; á otośmy już w Kádeſ / mieśćie przy gránicy twojej.
Niech / proƺę / przejdziemy przez ziemię twoję ; nie pójdziemy przez polá / áni przez winnice / áni będziemy pić wód z twoich ſtudzień ; gośćińcem pójdziemy / nie uchylimy śię ná práwo áni ná lewo / áż przejdziemy gránice twoje.
Ná to odpowiedziáł mu Edom : Nie pójdzieƺ przez moję ziemię / bym ſnáć z miecżem nie wyƺedł przećiw tobie.
Y rzekli mu ſynowie Izráelſcy : Bitym gośćińcem pójdziemy / á jeźlibyśmy wody twoje pili / my y bydłá náƺe / zápłáćimyć je ; nic innego nie żądámy / tylko ábyśmy pieƺo przeƺli.
Y powiedziáł : Nie przejdzieƺ. Y ruƺył śię Edom przećiwko nim / z wojſkiem wielkiem / y możną ręką.
A gdy nie chćiáł Edom pozwolić Izráelowi przejśćiá przez gránice ſwoje / tedy Izráel ucżynił odwrót od niego.
A ruƺywƺy śię z Kádeſ ; przyƺli ſynowie Izráelſcy y wƺyſtko zgromádzenie do góry Hor.
Y rzekł Pán do Mojżeƺá y do Aároná ná górze Hor / nád gránicą ziemi Edomſkiej / mówiąc :
Będzie Aáron przyłącżon do ludu ſwego ; ábowiem nie wnijdzie do ziemi / którąm dáł ſynom Izráelſkim / przeto żeśćie odpornymi byli ſłowu mojemu przy wodách poſwárku.
Weźmijże Aároná y Eleázárá ſyná jego / á káż im wſtąpić ná górę Hor ;
Y zewlecż Aároná z ƺát jego / á oblecż w nie Eleázárá / ſyná jego ; bo Aáron przyłącżon będzie do ludu ſwego / y tám umrze.
Y ucżynił Mojżeƺ / jáko rozkázáł Pán ; y wſtąpili ná górę Hor przed ocżymá wƺyſtkiego zgromádzeniá.
Y zewlókł Mojżeƺ Aároná z ƺát jego / á oblekł w nie Eleázárá ſyná jego ; y umárł tám Aáron ná wierzchu góry / á Mojżeƺ z Eleázárem zſtąpili z góry.
Widząc tedy wƺyſtko zgromádzenie / iż Aáron umárł / płákáli Aároná przez trzydzieśći dni / wƺyſtek dom Izráelſki.
