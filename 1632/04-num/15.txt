Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á mów do nich : Gdy przyjdziećie do ziemi mieƺkániá wáƺego / którą Já wám dám /
A będziećie chćieli cżynić ofiárę ogniſtą Pánu ná cáłopálenie / álbo ofiárę / bądź poślubioną bądź dobrowolną / álbo też ná urocżyſte świętá wáƺe / cżyniąc wdzięcżną wonność Pánu z wołów álbo z owiec :
Tedy / ktobykolwiek ofiárowáł ofiárę ſwoję Pánu / niechże ofiáruje ofiárę śniedną / pƺennej mąki dzieśiątą cżęść / zágniećionej z oliwą / której będzie cżwártá cżęść hynu.
Przytem winá ná ofiárę mokrą cżwártą cżęść hynu ofiárowáć będzieƺ przy cáłopáleniu / álbo przy ofierze innej do káżdego báránká.
Przy báránie też ofiárowáć będzieƺ ofiárę śniedną / mąki pƺennej dwie dzieśiąte cżęśći / zácżynionej z oliwą z trzećią cżęśćią hynu.
Winá tákże ná ofiárę mokrą trzećią cżęść hynu ofiárowáć będzieƺ ná wdzięcżną wonność Pánu.
Jeźli záś ćielcá ofiárowáć będzieƺ ná ofiárę cáłopáleniá / álbo ná ofiárę wypełnieniá ślubu / álbo ná ofiárę ſpokojną Pánu /
Tedy będzieƺ ofiárowáł ſpołem z ćielcem ofiárę śniedną / pƺennej mąki trzy dzieśiąte cżęśći / zágniećionej z oliwą z połową hynu.
Winá tákże będzieƺ ofiárowáł ná ofiárę mokrą połowę hynu / ná ofiárę ogniſtą ku wdzięcżnej wonnośći Pánu.
Tákże ucżyniƺ przy káżdym wole / y przy káżdym báránie y báránku / bądź z owiec bądź z kóz.
Według licżby / którą ofiárowáć będziećie / ták ucżynićie przy káżdym z nich według licżby ich.
Káżdy w domu zrodzony ták też będzie cżynił / gdy będzie oddáwáł ofiárę ogniſtą ná wdzięcżną wonność Pánu.
A gdyby / gośćiem będąc między wámi przychodzień / álbo mieƺkájący między wámi w národziech wáƺych / ofiárowáł ogniſtą ofiárę ku wdzięcżnej wonnośći Pánu / jáko wy cżynićie / ták y on ucżyni.
O ludu mój! Uſtáwá jedná niecháj będzie ták wám / jáko y przychodniowi / mieƺkájącemu z wámi ; uſtáwá to jeſt wiecżná w národziech wáƺych ; jáko wy / ták przychodzień będzie przed Pánem.
Práwo jedno / y jeden ſąd będzie wám y przychodniowi mieƺkájącemu z wámi.
Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á rzecż do nich : Gdy wnijdziećie do ziemi / do której Já wáſ wprowádzę ;
A jeść będziećie chleb onej ziemi / ofiárowáć będziećie ofiárę podnoƺeniá Pánu.
Z pierwƺych ćiáſt wáƺych plácek ofiárowáć będziećie ná ofiárę podnoƺeniá ; jáko ofiárę z bojewiſká / ták go ofiárowáć będziećie.
Z pierwƺych ćiáſt wáƺych oddáwáć będziećie Pánu ofiárę podnoƺeniá w národziech wáƺych.
A gdybyśćie pobłądzili / y nie ucżynilibyśćie wƺyſtkich przykázáń tych / które rozkázáł Pán przez Mojżeƺá ;
Wƺyſtkiego / co wám Pán rozkázáł przez Mojżeƺá / od onego dniá / którego wydáł Pán przykázánie / y potem w národziech wáƺych :
Tedy jeźliby z niewiádomośći zgromádzeniá ſtáło śię to pobłądzenie / wƺyſtko zgromádzenie ofiárowáć będzie ćielcá młodego jednego ná cáłopálenie / ná wdzięcżną wonność Pánu / tákże śniedną ofiárę jego / y mokrą ofiárę jego według zwycżáju / y kozłá jednego z ſtádá zá grzech.
Ták ocżyśći kápłán wƺyſtko zgromádzenie ſynów Izráelſkich / y będzie im odpuƺcżono / gdyż śię z niewiádomośći ſtáło. A oni ofiárowáć będą ofiárę ſwoję ná ofiárę ogniſtą Pánu / y ná ofiárę zá grzech ſwój przed Pánem zá obłądzenie ſwoje.
Y będzie odpuƺcżono wƺyſtkiemu zgromádzeniu ſynów Izráelſkich / y przychodniowi / który mieƺká między nimi / poniewáż wƺyſtkiego ludu pobłądzenie jeſt.
A jeźliby kto ſám tylko zgrzeƺył z niewiádomośći / tedy przynieśie Pánu kozę rocżną ná ofiárę zá grzech ;
Y ocżyśći kápłán cżłowieká obłądzonego / któryby zgrzeƺył z niewiádomośći ; przed Pánem ocżyśći go / y będzie mu odpuƺcżono.
Ták zrodzonemu między ſynámi Izráelſkimi / jáko przychodniowi / który mieƺká między nimi / zákon jeden będzie wám / gdyby kto zgrzeƺył z niewiádomośći.
Ale cżłowiek / któryby z hárdośći ſwáwolnie zgrzeƺył / ták urodzony w domu / jáko y przychodzień / tákowy Páná zelżył ; przetoż wykorzeniony będzie on cżłowiek z pośrodku ludu ſwego.
Abowiem ſłowem Páńſkim pogárdził / y przykázánie jego zgwáłćił ; koniecżnie wytrácony będzie tákowy cżłowiek ; niepráwość jego ná nim zoſtánie.
Y ſtáło śię / gdy byli ſynowie Izráelſcy ná puƺcży / że ználeźli cżłowieká zbierájącego drwá w dzień ſábátu.
Y przywiedli go / którzy go ználeźli zbierájącego drwá / przed Mojżeƺá / y przed Aároná / y przed wƺyſtko zgromádzenie.
Y dáli go do więzieniá ; bo jeƺcże im nie było oznájmiono / coby miáno cżynić z tákowym.
Tedy rzekł Pán do Mojżeƺá : Śmierćią niech umrze cżłowiek ten ; bez litośći niecháj go ukámionuje wƺyſtko zgromádzenie zá obozem.
Y wywiedli go wƺyſtko zgromádzenie zá obóz / y ćiſkáli náń kámieniem / áż umárł / jáko rozkázáł Pán Mojżeƺowi.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á powiedz im / áby ſobie pocżynili brámy ná krájách ƺát ſwoich w národziech ſwych / á niech przypráwią do brám ƺnurek hijácyntowy.
Y będziećie mieli te brámy / żebyśćie poglądájąc ná nie / wſpomináli ſobie ná wƺyſtkie przykázániá Páńſkie / ábyśćie je cżynili / y ábyśćie śię nie unośili zá ſercem wáƺem / y zá ocżymá wáƺemi / zá któremi wy idąc cudzołożylibyśćie.
Ale żebyśćie pámiętáli y cżynili wƺyſtkie przykázániá moje / á byli świętymi Bogu wáƺemu.
Já Pán / Bóg wáƺ / którym wáſ wywiódł z ziemi Egipſkiej / ábym wám był zá Bogá ; Jám Pán / Bóg wáƺ.
