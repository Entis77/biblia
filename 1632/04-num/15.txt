Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á mów do nich : Gdy przyjdźiećie do źiemi mieƺkániá wáƺego / którą Já wám dám /
A będźiećie chćieli cżynić ofiárę ogniſtą Pánu ná cáłopálenie / álbo ofiárę / bądź poślubioną bądź dobrowolną / álbo też ná urocżyſte świętá wáƺe / cżyniąc wdźięcżną wonność Pánu z wołów álbo z owiec :
Tedy / ktobykolwiek ofiárowáł ofiárę ſwoję Pánu / niechże ofiáruje ofiárę śniedną / pƺennej mąki dźieśiątą cżęść / zágniećionej z oliwą / której będźie cżwártá cżęść hynu.
Przytem winá ná ofiárę mokrą cżwártą cżęść hynu ofiárowáć będźieƺ przy cáłopáleniu / álbo przy ofierze innej do káżdego báránká.
Przy báránie też ofiárowáć będźieƺ ofiárę śniedną / mąki pƺennej dwie dźieśiąte cżęśći / zácżynionej z oliwą z trzećią cżęśćią hynu.
Winá tákże ná ofiárę mokrą trzećią cżęść hynu ofiárowáć będźieƺ ná wdźięcżną wonność Pánu.
Jeźli záś ćielcá ofiárowáć będźieƺ ná ofiárę cáłopáleniá / álbo ná ofiárę wypełnieniá ślubu / álbo ná ofiárę ſpokojną Pánu /
Tedy będźieƺ ofiárowáł ſpołem z ćielcem ofiárę śniedną / pƺennej mąki trzy dźieśiąte cżęśći / zágniećionej z oliwą z połową hynu.
Winá tákże będźieƺ ofiárowáł ná ofiárę mokrą połowę hynu / ná ofiárę ogniſtą ku wdźięcżnej wonnośći Pánu.
Tákże ucżyniƺ przy káżdym wole / y przy káżdym báránie y báránku / bądź z owiec bądź z kóz.
Według licżby / którą ofiárowáć będźiećie / ták ucżynićie przy káżdym z nich według licżby ich.
Káżdy w domu zrodzony ták też będźie cżynił / gdy będźie oddáwáł ofiárę ogniſtą ná wdźięcżną wonność Pánu.
A gdyby / gośćiem będąc między wámi przychodźień / álbo mieƺkájący między wámi w národźiech wáƺych / ofiárowáł ogniſtą ofiárę ku wdźięcżnej wonnośći Pánu / jáko wy cżynićie / ták y on ucżyni.
O ludu mój! Uſtáwá jedná niecháj będźie ták wám / jáko y przychodniowi / mieƺkájącemu z wámi ; uſtáwá to jeſt wiecżná w národźiech wáƺych ; jáko wy / ták przychodźień będźie przed Pánem.
Práwo jedno / y jeden ſąd będźie wám y przychodniowi mieƺkájącemu z wámi.
Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á rzecż do nich : Gdy wnijdźiećie do źiemi / do której Já wáſ wprowádzę ;
A jeść będźiećie chleb onej źiemi / ofiárowáć będźiećie ofiárę podnoƺeniá Pánu.
Z pierwƺych ćiáſt wáƺych plácek ofiárowáć będźiećie ná ofiárę podnoƺeniá ; jáko ofiárę z bojewiſká / ták go ofiárowáć będźiećie.
Z pierwƺych ćiáſt wáƺych oddáwáć będźiećie Pánu ofiárę podnoƺeniá w národźiech wáƺych.
A gdybyśćie pobłądźili / y nie ucżynilibyśćie wƺyſtkich przykázáń tych / które rozkázáł Pán przez Mojżeƺá ;
Wƺyſtkiego / co wám Pán rozkázáł przez Mojżeƺá / od onego dniá / którego wydáł Pán przykázánie / y potem w národźiech wáƺych :
Tedy jeźliby z niewiádomośći zgromádzeniá ſtáło śię to pobłądzenie / wƺyſtko zgromádzenie ofiárowáć będźie ćielcá młodego jednego ná cáłopálenie / ná wdźięcżną wonność Pánu / tákże śniedną ofiárę jego / y mokrą ofiárę jego według zwycżáju / y kozłá jednego z ſtádá zá grzech.
Ták ocżyśći kápłán wƺyſtko zgromádzenie ſynów Izráelſkich / y będźie im odpuƺcżono / gdyż śię z niewiádomośći ſtáło. A oni ofiárowáć będą ofiárę ſwoję ná ofiárę ogniſtą Pánu / y ná ofiárę zá grzech ſwój przed Pánem zá obłądzenie ſwoje.
Y będźie odpuƺcżono wƺyſtkiemu zgromádzeniu ſynów Izráelſkich / y przychodniowi / który mieƺká między nimi / poniewáż wƺyſtkiego ludu pobłądzenie jeſt.
A jeźliby kto ſám tylko zgrzeƺył z niewiádomośći / tedy przynieśie Pánu kozę rocżną ná ofiárę zá grzech ;
Y ocżyśći kápłán cżłowieká obłądzonego / któryby zgrzeƺył z niewiádomośći ; przed Pánem ocżyśći go / y będźie mu odpuƺcżono.
Ták zrodzonemu między ſynámi Izráelſkimi / jáko przychodniowi / który mieƺká między nimi / zákon jeden będźie wám / gdyby kto zgrzeƺył z niewiádomośći.
Ale cżłowiek / któryby z hárdośći ſwáwolnie zgrzeƺył / ták urodzony w domu / jáko y przychodźień / tákowy Páná zelżył ; przetoż wykorzeniony będźie on cżłowiek z pośrodku ludu ſwego.
Abowiem ſłowem Páńſkim pogárdźił / y przykázánie jego zgwáłćił ; koniecżnie wytrácony będźie tákowy cżłowiek ; niepráwość jego ná nim zoſtánie.
Y ſtáło śię / gdy byli ſynowie Izráelſcy ná puƺcży / że ználeźli cżłowieká zbierájącego drwá w dźień ſábátu.
Y przywiedli go / którzy go ználeźli zbierájącego drwá / przed Mojżeƺá / y przed Aároná / y przed wƺyſtko zgromádzenie.
Y dáli go do więźieniá ; bo jeƺcże im nie było oznájmiono / coby miáno cżynić z tákowym.
Tedy rzekł Pán do Mojżeƺá : Śmierćią niech umrze cżłowiek ten ; bez litośći niecháj go ukámionuje wƺyſtko zgromádzenie zá obozem.
Y wywiedli go wƺyſtko zgromádzenie zá obóz / y ćiſkáli náń kámieniem / áż umárł / jáko rozkázáł Pán Mojżeƺowi.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á powiedz im / áby ſobie pocżynili brámy ná krájách ƺát ſwoich w národźiech ſwych / á niech przypráwią do brám ƺnurek hijácyntowy.
Y będźiećie mieli te brámy / żebyśćie poglądájąc ná nie / wſpomináli ſobie ná wƺyſtkie przykázániá Páńſkie / ábyśćie je cżynili / y ábyśćie śię nie unośili zá ſercem wáƺem / y zá ocżymá wáƺemi / zá któremi wy idąc cudzołożylibyśćie.
Ale żebyśćie pámiętáli y cżynili wƺyſtkie przykázániá moje / á byli świętymi Bogu wáƺemu.
Já Pán / Bóg wáƺ / którym wáſ wywiódł z źiemi Egipſkiej / ábym wám był zá Bogá ; Jám Pán / Bóg wáƺ.
