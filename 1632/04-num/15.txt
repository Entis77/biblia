Y rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim / á mów do nich : Gdy przydźiećie do źiemie mieƺkánia wáƺego / którą ja wam dam :
A będźiećie chćieli czynić Ofiárę ogniſtą PAnu ná cáłopalenie / álbo Ofiárę / bądź poślubioną bądź dobrowolną / álbo też ná uroczyſte świętá wáƺe / czyniąc wdźięczną wonność PAnu / z wołów álbo z owiec :
Tedy ktobykolwiek ofiárował Ofiárę ſwoję PAnu / niechże ofiáruje Ofiárę śniedną / pƺenney mąki dźieśiątą część / nágniećioney z Oliwą / <i>którey</i> będźie czwartá część Hinu :
Przytym winá ná Ofiárę mokrą / czwartą część Hinu / ofiárowáć będźieƺ przy cáłopaleniu / álbo przy ofierze <i>inney</i> do káżdego báránká.
Przy báránie też ofiárowáć będźieƺ Ofiárę śniedną / mąki pƺenney dwie dźieśiąte częśći / zádziáłáney z Oliwą z trzećią częśćią Hinu :
Winá tákże ná Ofiárę mokrą / trzećią część Hinu / ofiárowáć będźieƺ ná wdźięczną wonność PAnu.
Jeſli záś ćielcá ofiárowáć będźieƺ ná Ofiárę cáłopalenia / álbo ná Ofiárę wypełnienia ślubu / álbo ná Ofiárę ſpokojną PAnu :
Tedy będźieƺ ofiárował ſpołu z ćielcem / Ofiárę śniedną / pƺenney mąki trzy dźieśiąte częśći zágniećioney z Oliwą / z połowicą Hinu :
Winá tákże będźieƺ ofiárował / ná Ofiárę mokrą / połowicę Hinu / ná Ofiárę ogniſtą / <i>ku</i> wdźięczney wonnośći PAnu.
Tákże uczyniƺ przy káżdym wole / y przy káżdym báránie / <i>y</i> báránku / bądź z Owiec / bądź z kóz.
Według liczby którą ofiárowáć będźiećie / ták uczynićie przy káżdym z nich / według liczby ich.
Káżdy domá zrodzony / ták też będźie czynił / gdy będźie oddawał Ofiárę ogniſtą / ná wdźięczną wonność PAnu.
A gdyby gośćiem będąc miedzy wámi przychodźień / álbo mieƺkájący miedzy wámi / w Narodźiech wáƺych / ofiárował ogniſtą Ofiárę / ku wdźięczney wonnośći PAnu : jáko <i>wy</i> czynićie / ták y <i>on</i> uczyni.
O ludu mój! Uſtáwá jedná niechaj będźie / ták wam jáko y przychodniowi mieƺkájącemu <i>z wámi</i> uſtáwá to jeſt wieczna w Narodźiech wáƺych : jáko wy / ták przychodźień będźie przed PAnem.
Práwo jedno / y jeden ſąd będźie wam / y przychodniowi mieƺkájącemu z wámi.
Y rzekł PAN do Mojzeƺá mówiąc :
Powiedz Synom Izráelſkim / á rzecz do nich : Gdy wnidźiećie do źiemie / do którey ja was wprowádzę :
A jeść będźiećie chleb oney źiemie / ofiárowáć będźiećie Ofiárę podnoƺenia PAnu.
Z pierwƺych ćiaſt wáƺych plácek ofiárowáć będźiećie ná Ofiárę podnoƺenia : jáko ofiárę z bojewiſká / ták go ofiárowáć będźiećie.
Z pierwƺych ćiaſt wáƺych / oddawáć będźiećie PAnu Ofiárę podnoƺenia / w Narodźiech wáƺych.
A gdybyśćie pobłądźili / y nie uczynilibyśćie wƺyſtkich przykazań tych / które rozkazał PAN przez Mojzeƺá /
Wƺyſtkiego / co wam PAN rozkzzał przez Mojzeƺá / od onego dniá którego wydał PAN Przykazánie / y potym w Narodźiech wáƺych :
Tedy jeſliby z niewiádomośći zgromádzenia ſtáło śię to pobłądzenie / wƺyſtko zgromádzenie ofiárowáć będźie ćielcá młodego jednego ná cáłopalenie / ná wdźięczną wonność PANU : tákże śniedną Ofiárę jego / y mokrą Ofiárę jego / według zwyczáju / y kozłá jednego z ſtádá / zá grzech.
Ták oczyśći Kápłan wƺyſtko zgromádzenie Synów Izráelſkich / á będźie im odpuƺczono / gdyż śię z niewiádomośći ſtáło : A oni ofiárowáć będą ofiárę ſwoję ná Ofiárę ogniſtą PAnu / y ná Ofiárę zá grzech ſwój przed PAnem / zá obłądzenie ſwoje.
Y będźie odpuƺczono wƺyſtkiemu zgromádzeniu Synów Izráelſkich / y przychodniowi / który mieƺka miedzy nimi : ponieważ wƺyſtkiego ludu pobłądzenie jeſt.
A jeſliby kto ſam tylko zgrzeƺył z niewiádomośći / tedy przynieśie ná ofiárę kozę roczną <i>PAnu</i> zá grzech.
Y oczyśći Kápłan człowieká obłądzonego któryby zgrzeƺył z niewiádomośći : przed PAnem oczyśći go / y będźie mu odpuƺczono.
Ták zrodzonemu miedzy Synámi Izráelſkimi / jáko przychodniowi / który mieƺka miedzy nimi / zakon jeden będźie wam / gdyby kto zgrzeƺył z niewiádomośći.
Ale człowiek któryby z hárdośći ſwowolnie zgrzeƺył / ták urodzony w domá / jáko y przychodźień / tákowy PAná zelżył / przetoż wykorzeniony będźie on człowiek / z pośrzodku ludu ſwego :
Abowiem ſłowem PAńſkim pogárdźił / y przykazánie jego zgwałćił : koniecznie wytrácony będźie tákowy człowiek : niepráwość jego ná nim <i>zoſtánie</i>.
Y ſtáło śię / gdy byli Synowie Izráelſcy ná puƺczy / że ználeźli człowieká / zbierájącego drwá w dźień Sábbátu :
Y przywiedli go / którzy go náleźli zbierájącego drwá / przed Mojzeƺá y przed Aároná / y przed wƺyſtko zgromádzenie :
Y dáli go do więźienia : Bo jeƺcze im nie było oznájmiono coby miano czynić z tákowym.
Tedy rzekł PAN do Mojzeƺá : śmierćią niech umrze człowiek ten / bez litośći niechaj go ukámionuje wƺyſtko zgromádzenie zá Obozem.
Y wywiedli go wƺyſtko zgromádzenie zá Obóz / y ćiſkáli nań kámieniem / áż umárł / jáko rozkazał PAN Mojzeƺowi.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / á powiedz im / áby ſobie poczynili brámy ná krájách ƺat ſwojich / w Narodźiech ſwych / á niech przypráwią do brámów ƺnurek hyácyntowy.
Y będźiećie mieli te brámy / żebyśćie poglądájąc ná nie wſpomináli ſobie ná wƺyſtkie przykazánia PAńſkie / ábyśćie je czynili / y ábyśćie śię nie unośili zá ſercem wáƺym / y zá oczymá wáƺymi / zá którymi wy idąc cudzołożylibyśćie /
<i>Ale</i> żebyśćie pámiętáli y czynili wƺyſtkie przykazánia moje / á byli świętymi Bogu wáƺemu.
<i>JA PAN</i> Bóg wáƺ / którym was wywiódł z źiemie Egipſkiey / ábym wam był zá Bogá : <i>JAM PAN /</i> Bóg wáƺ.
