Y rzekł Pán do Mojżeƺá ná polách Moábſkich / nád Jordánem przećiw Jerychowi / mówiąc :
Rozkáż ſynom Izráelſkim / áby dáli Lewitom z dźiedźictwá ośiádłośći ſwojey miáſtá do mieƺkánia / y przedmieśćiá około miaſt ich oddáćie Lewitom.
Y będą mieli miáſtá ſobie do mieƺkánia / á przedmieśćiá ich będą im dla bydłá ich / y dla májętnośći ich / y dla wƺyſtkiego dobytku ich.
A przedmieśćiá miaſt / które dáćie Lewitom / od muru miejſkiego pójdą ná tyśiąc łokći wƺędy w około.
Przetoż wymierzyćie zá káżdym miáſtem dwá tyśiące łokći ná wſchód ſłońcá / ná południe też dwá tyśiące łokći / tákże ná záchód dwá tyśiące łokći / y ná północy dwá tyśiące łokći / á miáſto w pośrzodku będźie ; tákoweć będą przedmieśćiá miaſt ich.
A miedzy temi miáſty / które dáćie Lewitom / ƺeść miaſt będą dla ućiecżki / które dáćie / áby tám ućiekał mężobójcá ; á nád te dáćie im cżterdźieśći miaſt y dwá.
Ták iż wƺyſtkich miaſt / które Lewitom dáćie / będźie cżterdźieśći y ośm miaſt y z przedmieśćiámi ich.
A miaſt / które dáćie z dźierżáw ſynów Izráelſkich / od tych / którzy więcey máją / więcey dáćie / á od tych / którzy mniej máją / dáćie mniej ; káżdy według miáry dźiedźictwá ſwego / które pośiędźie / udźieli z miaſt ſwojich Lewitom.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / y rzecż im : Gdy przejdźiećie przez Jordán do źiemi Chánánejſkiej.
Poſtánowćież ſobie miáſtá ; miáſtá dla ućiecżki mieć będźiećie / áby tám ućiekał mężobójcá / któryby zábił kogo z nieobácżeniá.
A będą wam te miáſtá dla ućiecżki przed powinowátym zábitego / áby nie dał gárdłá ten co zábił / pókiby nie ſtánął przed zgromádzeniem ná ſąd.
A miaſt / które odłącżyćie / ƺeść miaſt dla ućiecżki mieć będźiećie.
Trzy miáſtá dáćie z tey ſtrony Jordánu / á trzy miáſtá dáćie w źiemi Chánánejſkiej ; te miáſtá dla ućiecżki będą.
Synom Izráelſkim / y przychodniowi / y mieƺkájącemu miedzy nimi / będą te ƺeść miaſt do ućiecżki / áby tám ućiekł káżdy / ktoby zábił cżłowieká z nieobácżeniá.
Wƺákże / jeſliby go żelázną bronią uderzył / ták żeby umárł / mężobójcą jeſt ; śmierćią umrze on mężobójcá.
Albo jeſliby májąc kámień w ręku / którymby mógł zábić / uderzył go / ták żeby umárł / mężobójcą jeſt ; śmierćią umrze on mężobójcá.
Tákże jeſliby májąc w ręku drewno / któremby mógł zábić / uderzył go / y umárłby / mężobójcą jeſt ; śmierćią umrze on mężobójcá.
Powinowáty zábitego zábije tego mężobójcę ; gdźiekolwiek śię z nim ſpotká / on ſám zábije go.
A jeſliby kogo z nienáwiśći popchnął / álbo nań cżym ćiſnął z záſádzki / á umárłby ;
Albo jeſliby go z wáśni uderzył ręką ſwoją / á umárłby / śmierćią umrze ten / który uderzył / mężobójcą jeſt ; powinny zábitego zábije mężobójcę / gdźiekolwiek go tráfi.
Ale jeſliby z przygody bez wáśni kogo popchnął / álboby nań ćiſnął cżemkolwiek nie umyślnie ;
Albo jeſliby jákim kámieniem / od któregoby mógł umrzeć / rzućił nań z nieobácżeniá / á umárłby / nie będąc mu nieprzyjáćielem / áni ƺukájąc jego złego :
Tedy rozſądek ucżyni zgromádzenie miedzy tym / który zábił / á miedzy powinnym zábitego według tego práwá.
Y wybáwi zgromádzenie mężobójcę tego z rąk powinnego onego zábitego / y káże mu śię wróćić zgromádzenie do miáſtá ućiecżki jego / gdźie był ućiekł ; y támże będźie mieƺkał áż do śmierći kápłáná nájwyżƺego / który jeſt pomázány olejkiem świętym.
A jeſliby wyƺedł mężobójcá zá gránice miáſtá ućiecżki ſwojey / do którego ućiekł ;
Y tráfiłby go powinny zábitego zá gránicą miáſtá ućiecżki jego / choćiáżby zábił powinny zábitego mężobójcę onego / nie będźie winien krwi.
Abowiem w mieśćie ućiecżki ſwojey mieƺkáć ma áż do śmierći kápłáná nájwyżƺego / á po śmierći kápłáná nájwyżƺego wróći śię on mężobójcá do źiemi ośiádłośći ſwojey.
A będźiećie to mieli zá uſtáwę práwną w národźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Ktobykolwiek chćiał zábić cżłowieká / zá świádectwem świádków zábije mężobójcę ; ále świádek jeden nie będźie mógł świádcżyć ná ſkázánie kogo ná śmierć.
Nie weźmiećie też okupu zá żywot mężobójcy / który záſłużył śmierć ; niech śmierćią umrze.
Nie weźmiećie też zápłáty od onego / który ućiekł do miáſtá ućiecżki ſwojey / áby śię náwróćił ná mieƺkánie do źiemi ſwojey / pierwej niżby kápłán umárł :
Byśćie nie ſplugáwili źiemię / w którey będźiećie ; bo krew tákowá ſplugáwiłáby źiemię ; á źiemiá nie może być ocżyƺcżoná od krwi / która jeſt wyláná ná niej / jedno krwią tego / który ją przelał.
Przetoż nie plugáwćie źiemi / w którey mieƺkáćie / w którey też Ja mieƺkám ; bom Ja Pán / który mieƺkám miedzy ſynámi Izráelſkimi.
