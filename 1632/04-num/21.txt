A gdy uſłyƺáł Chánánejcżyk / król Hárád / który mieƺkáł ná południe / że Izráelcżycy ćiągnęli oną drogą / którą byli ƺpiegowie przeƺli / tedy zwiódł bitwę z Izráelem / y pojmáł ich wiele.
Tám ucżynił Izráel ślub Pánu / mówiąc : Jeźliże podáƺ lud ten w ręce moje / do gruntu wywrócę miáſtá ich.
Y wyſłucháł Pán głoſ Izráelá / á podáł mu Chánánejcżyki : y wytráćił je z gruntu / y miáſtá ich / á názwáł imię onego miejſcá Chormá.
Potem ruƺyli śię od góry Hor drogą ku morzu cżerwonemu / áby obeƺli ziemię Edomſką ; y utrudził śię lud bárdzo w onej drodze.
Przetoż mówił lud przećiw Bogu / y przećiw Mojżeƺowi : Przecżżeśćie náſ wywiedli z Egiptu / ábyśmy pomárli ná tej puƺcży? bo niemáƺ chlebá / áni wody / á duƺá náƺá obrzydziłá ſobie ten chleb nikcżemny.
Przetoż przepuśćił Pán ná lud węże ogniſte / którzy kąſáli lud ; y pomárło wiele ludu z Izráelá.
Y przyƺedłƺy lud do Mojżeƺá / rzekli : Zgrzeƺyliśmy / żeśmy mówili przećiw Pánu / y przećiw tobie. Módl śię Pánu / áby oddálił od náſ te węże ; y modlił śię Mojżeƺ zá ludem.
Y rzekł Pán do Mojżeƺá : Ucżyń ſobie wężá miedziánego / á wyſtáw go ná drzewcu ; y ſtánie śię / ktokolwiek ukąƺony będąc wejrzy náń / że żyw zoſtánie.
Spráwił tedy Mojżeƺ wężá miedziánego / y wyſtáwił go ná drzewcu ; y było to / gdy kogo wąż ukąśił / á ſpojrzáł ná wężá miedziánego / że żyw zoſtáł.
Zátem ruƺyli śię ſynowie Izráelſcy / á ſtánęli obozem w Obot.
A z Obot ruƺywƺy śię położyli śię obozem ná págórkách gór Hábárym ná puƺcży / którá jeſt przećiw Moábcżykom od wſchodu ſłońcá.
A odƺedłƺy ſtámtąd położyli śię obozem nád potokiem Zered.
Stámtąd odćiągnąwƺy położyli śię obozem u brodu Arnon / który jeſt ná puƺcży / á wychodzi z gránicy Amorejſkiej : ábowiem Arnon jeſt gránicá Moábſká między Moábcżykiem y Amorejcżykiem.
Przetoż mówi śię w kśięgách wojen Páńſkich : Przećiwko Wáhebowi w wichrze wálcżył / y przy potokách Arnon.
Bo śćiekánie tych potoków / które śię náchyliło ( tocży ) ku położeniu Hár / to śię śćiągá ku gránicy Moábſkiej.
Stámtąd potem przyƺli do Beer ; á táć jeſt oná ſtudniá / o której mówił Pán do Mojżeƺá : Zgromádź lud / á dám im wody.
Tedy śpiewáł Izráel tę pioſnkę : Wyſtąp ſtudnio ; śpiewájćież o niej ;
Studniá / którą wykopáli kśiążętá / wykopáli ją hetmáni ludu z uſtáwcą zákonu / láſkámi ſwojimi. A z tej puƺcży ruƺyli śię do Mátáná ;
A z Mátáná do Náhálijelu / á z Náhálijelu do Bámotu ;
A z Bámotu ku Hágáj / które jeſt w polách Moábſkich / ná wierzchu págórká / który leży ku puƺcży.
Y poſłáł Izráel poſły do Sehoná / królá Amorejſkiego / mówiąc :
Niech przejdziemy przez ziemię twoję ; nie pójdziemy áni przez polá áni przez winnice ; nie będziemy pić wód z ſtudzien twoich ; gośćińcem pójdziemy / áż przejdziemy gránice twoje.
Ale nie pozwolił Sehon Izráelowi iść przez gránice ſwoje ; y zebráwƺy Sehon wƺyſtek lud ſwój / wyćiągnął przećiw Izráelowi ná puƺcżą / á gdy przyƺedł do Jáházy / zwiódł bitwę z Izráelem.
Y poráził go Izráel oſtrzem miecżá / y odziedzicżył ziemię jego od Arnonu áż do Jáboku / y áż do ziemi ſynów Ammonowych ; ábowiem opátrzone były gránice Ammonitów.
Tedy pobráł Izráel wƺyſtkie miáſtá one / y mieƺkáł we wƺyſtkich miáſtách Amorejſkich / w Heſebon / y we wƺyſtkich wśiách jego.
Bo Heſebon było miáſto Sehoná / królá Amorejſkiego / który / gdy pierwej wálcżył z królem Moábſkim / wziął mu był wƺyſtkę ziemię jego z rąk jego áż po Arnon.
Dlá tegoż mówią w przypowieśći : Pójdźćie do Heſebon / á niech zbudują y nápráwią miáſto Sehonowe.
Abowiem wyƺedł ogień z Heſebon / á płomień z miáſtá Sehonowego / y popálił Ar Moábſkie / y obywátele wyſokich miejſc Arnon.
Biádá tobie Moáb / zginąłeś o ludu Chámoſ! podáł ſyny ſwoje ná ućiekánie / y córki ſwoje do więzieniá królowi Amorejſkiemu Sehonowi.
A záginęło pánowánie ich od Heſeboná áż do Dyboná ; á poburzyliśmy je áż do Nofe / które idzie áż do Medády.
Y mieƺkáł Izráel w ziemi Amorejſkiej.
Tedy poſłáł Mojżeƺ ná ƺpiegi do Jázer / którego wśi pobráli / wypędziwƺy Amorejcżyki / którzy tám byli.
Potem obróćiwƺy śię ƺli ku Báſán ; gdzie wyćiągnął Og / król Báſáńſki / przećiwko nim / ſám y wƺyſtek lud jego / áby z nimi ſtocżył bitwę w Edrej.
Tedy rzekł Pán do Mojżeƺá : Nie bój śię go ; bo w ręce twoje podáłem go / y wƺyſtek lud jego / y ziemię jego / y ucżyniƺ mu / jákoś ucżynił Sehonowi / królowi Amorejſkiemu / który mieƺkáł w Heſebon.
Y porázili go / y ſyny jego / ze wƺyſtkim ludem jego / ták iż nikogo z niego nie zoſtáwili / y pośiedli dziedzicżnie ziemię jego.
