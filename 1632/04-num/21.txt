A gdy uſłyƺał Chánánejcżyk / król Hárád / który mieƺkał ná południe / że Izráelcżycy ćiągnęli oną drogą / którą byli ƺpiegowie przeƺli / tedy zwiódł bitwę z Izráelem / y pojmał ich wiele.
Tám ucżynił Izráel ślub Pánu / mówiąc : Jeſliże podáƺ lud ten w ręce moje / do gruntu wywrócę miáſtá ich.
Y wyſłuchał Pán głos Izráelá / á podał mu Chánánejcżyki : y wytráćił je z gruntu / y miáſtá ich / á názwał imię onego miejſcá Chormá.
Potem ruƺyli śię od góry Hor drogą ku morzu cżerwonemu / áby obeƺli źiemię Edomſką ; y utrudźił śię lud bárzo w oney drodze.
Przetoż mówił lud przećiw Bogu / y przećiw Mojżeƺowi : Przecżżeśćie nas wywiedli z Egiptu / ábyſmy pomárli ná tey puƺcży? bo niemáƺ chlebá / áni wody / á duƺá náƺá obrzydźiłá ſobie ten chleb nikcżemny.
Przetoż przepuśćił Pán ná lud węże ogniſte / którzy kąſáli lud ; y pomárło wiele ludu z Izráelá.
Y przyƺedƺy lud do Mojżeƺá / rzekli : Zgrzeƺyliśmy / żeśmy mówili przećiw Pánu / y przećiw tobie. Módl śię Pánu / áby oddálił od nas te węże ; y modlił śię Mojzeƺ zá ludem.
Y rzekł Pán do Mojżeƺá : Ucżyń ſobie wężá miedźiánego / á wyſtáw go ná drzewcu ; y ſtánie śię / ktokolwiek ukąƺony będąc wejrzy nań / że żyw zoſtánie.
Spráwił tedy Mojzeƺ wężá miedźiánego / y wyſtáwił go ná drzewcu ; y było to / gdy kogo wąż ukąśił / á ſpojrzał ná wężá miedźiánego / że żyw zoſtał.
Zátem ruƺyli śię ſynowie Izráelſcy / á ſtánęli obozem w Obot.
A z Obot ruƺywƺy śię położyli śię obozem ná págórkách gór Hábárym ná puƺcży / która jeſt przećiw Moábcżykom od wſchodu ſłońcá.
A odƺedƺy ſtámtąd położyli śię obozem nád potokiem Zered.
Stámtąd odćiągnąwƺy położyli śię obozem u brodu Arnon / który jeſt ná puƺcży / á wychodźi z gránicy Amorejſkiej : ábowiem Arnon jeſt gránicá Moábſká miedzy Moábcżykiem y Amorejcżykiem.
Przetoż mówi śię w kśięgách wojen Páńſkich : Przećiwko Wáhebowi w wichrze wálcżył / y przy potokách Arnon.
Bo śćiekánie tych potoków / które śię náchyliło ( tocży ) ku położeniu Hár / to śię śćiągá ku gránicy Moábſkiej.
Stámtąd potem przyƺli do Beer ; á táć jeſt oná ſtudniá / o którey mówił Pán do Mojżeƺá : Zgromádź lud / á dam im wody.
Tedy śpiewał Izráel tę pioſnkę : Wyſtąp ſtudnio ; śpiewájćież o niej ;
Studniá / którą wykopáli kśiążętá / wykopáli ją hetmáni ludu z uſtáwcą zakonu / láſkámi ſwojimi. A z tey puƺcży ruƺyli śię do Mátáná ;
A z Mátáná do Náhálijelu / á z Náhálijelu do Bámotu ;
A z Bámotu ku Hágáj / które jeſt w polách Moábſkich / ná wierzchu págórká / który leży ku puƺcży.
Y poſłał Izráel poſły do Sehoná / królá Amorejſkiego / mówiąc :
Niech przejdźiemy przez źiemię twoję ; nie pójdźiemy áni przez polá áni przez winnice ; nie będźiemy pić wód z ſtudźien twojich ; gośćińcem pójdźiemy / áż przejdźiemy gránice twoje.
Ale nie pozwolił Sehon Izráelowi iść przez gránice ſwoje ; y zebráwƺy Sehon wƺyſtek lud ſwój / wyćiągnął przećiw Izráelowi ná puƺcżą / á gdy przyƺedł do Jáházy / zwiódł bitwę z Izráelem.
Y poráźił go Izráel oſtrzem miecżá / y odźiedźicżył źiemię jego od Arnonu áż do Jáboku / y áż do źiemi ſynów Ammonowych ; ábowiem opátrzone były gránice Ammonitów.
Tedy pobrał Izráel wƺyſtkie miáſtá one / y mieƺkał we wƺyſtkich miáſtách Amorejſkich / w Heſebon / y we wƺyſtkich wśiách jego.
Bo Heſebon było miáſto Sehoná / królá Amorejſkiego / który / gdy pierwej wálcżył z królem Moábſkim / wźiął mu był wƺyſtkę źiemię jego z rąk jego áż po Arnon.
Dla tegoż mówią w przypowieśći : Pójdźćie do Heſebon / á niech zbudują y nápráwią miáſto Sehonowe.
Abowiem wyƺedł ogień z Heſebon / á płomień z miáſtá Sehonowego / y popálił Ar Moábſkie / y obywátele wyſokich miejſc Arnon.
Biádá tobie Moáb / zginąłeś o ludu Chámos! podał ſyny ſwoje ná ućiekánie / y córki ſwoje do więźieniá królowi Amorejſkiemu Sehonowi.
A záginęło pánowánie ich od Heſeboná áż do Dyboná ; á poburzyliśmy je áż do Nofe / które idźie áż do Medády.
Y mieƺkał Izráel w źiemi Amorejſkiej.
Tedy poſłał Mojzeƺ ná ƺpiegi do Jázer / którego wśi pobráli / wypędźiwƺy Amorejcżyki / którzy tám byli.
Potem obróćiwƺy śię ƺli ku Báſán ; gdźie wyćiągnął Og / król Báſáńſki / przećiwko nim / ſám y wƺyſtek lud jego / áby z nimi ſtocżył bitwę w Edrej.
Tedy rzekł Pán do Mojżeƺá : Nie bój śię go ; bo w ręce twoje podáłem go / y wƺyſtek lud jego / y źiemię jego / y ucżyniƺ mu / jákoś ucżynił Sehonowi / królowi Amorejſkiemu / który mieƺkał w Heſebon.
Y poráźili go / y ſyny jego / ze wƺyſtkim ludem jego / ták iż nikogo z niego nie zoſtáwili / y pośiedli dźiedźicżnie źiemię jego.
