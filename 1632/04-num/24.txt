A gdy bácżył Báláám / że śię podobáło Pánu / áby błogoſłáwił Izráelowi / już nie chodźił / jáko przedtem / raz y drugi dla wieƺcżby ; ále obróćił ku puƺcży twárz ſwoję.
A podnióſƺy Báláám ocży ſwe / obácżył Izráelá mieƺkájącego według pokoleń ſwojich / y był nád nim Duch Boży.
Y zácżął przypowieść ſwoję / á mówił :
Rzekł Báláám / ſyn Beorów / rzekł mąż / którego ocży ſą otworzone / rzekł ſłyƺący wymowy Boże / á który widzenie Wƺechmocnego widźiał / który / kiedy pádnie / otworzone ma ocży :
Jáko piękne ſą námioty twoje / o Jákubie! przybytki twoje / o Izráelu!
Jáko potoki rozćiągnęły śię / jáko ogrody przy rzece / jáko drzewá wonne / które Pán náſádźił / jáko cedry nád wodámi.
Popłynie wodá z wiádrá jego / á náśienie jego będźie nád wodámi obfitemi / á będźie wywyżƺon nád Agágá król jego / á wynieśie śię króleſtwo jego.
Bóg wywiódł go z Egiptu / mocą jednorożcową był mu ; pożre národy przećiwne ſobie / á kośći ich pokruƺy / y ſtrzáłámi ſwemi przeráźi je.
Położył śię / leży jáko lwię / y jáko lew śilny ; któż go obudźi? ktobyć błogoſłáwił / błogoſłáwiony / á ktoby ćię przeklinał / przeklęty będźie.
Tedy śię zápálił gniew Báláká ná Báláámá / á kláſnąwƺy rękámi ſwemi / rzekł Bálák do Báláámá : Dla złorzecżeniá nieprzyjáćiołom moim przyzwáłem ćię / á otoś im błogoſłáwił już po trzy kroć.
Przetoż teraz uchodź ná miejſce ſwoje ; rzekłem ći był : Zácnie ćię ucżcżę ; ále oto pozbáwił ćię Pán tey cżći.
Y rzekł Báláám do Báláká : Izáżem y poſłom twoim / któreś ſłał do mnie / nie powiedźiał mówiąc :
Choćby mi dał Bálák pełen dom ſwój ſrebrá y złotá / nie będę mógł przeſtąpić ſłowá Páńſkiego / ábym cżynił co dobrego álbo złego ſám z śiebie ; co mi opowie Pán / to będę mówił.
A teraz oto ja odchodzę do ludu mego / jednák oznájmięć / co ucżyni lud ten ludowi twemu ná potem.
Y zácżął przypowieść ſwoję y rzekł : Mówił Báláám / ſyn Beorów / mówił mąż / którego ocży ſą otworzone ;
Mówił ten / który ſłyƺał wyroki Boże / á który ma umiejętność Nájwyżƺego ; który widźiał widzenie Wƺechmocnego ; który / kiedy pádnie / otworzone ma ocży :
Ujrzę go / ále nie teraz ; oglądám go / ále nie z bliſká ; wynijdźie gwiázdá z Jákubá y powſtánie láſká z Izráelá / y pobije kśiążętá Moábſkie / y wytráći wƺyſtkie ſyny Setowe.
Y przydźie Edom w opánowánie / á Seir będźie pod włádzą nieprzyjáćiół ſwojich / á Izráel będźie ſobie pocżynał mężnie.
Y będźie pánował / który wynijdźie z Jákubá / á wytráći oſtátki z miaſt.
A gdy ſpojrzał ná Amáleká / zácżął przypowieść ſwoję / y rzekł : Pocżątek národów jeſt Amálek / ále oſtátek jego do ƺcżętu záginie.
Potem wejrzáwƺy ná Kenejcżyká zácżął przypowieść ſwoję y rzekł : Mocneć jeſt mieƺkánie twoje / á záłożyłeś ná ſkále gniázdo twoje ;
Wƺákże ſpuſtoƺony będźie Kenejcżyk / áż ćię Aſſur záprowádźi do więźieniá.
Znowu zácżął przypowieść ſwoję / y rzekł : Ach! któż będźie żyw / gdy to ucżyni Bóg?
Bo okręty przypłyną od brzegów Chyttymſkich / y utrápią Aſſyryjány / utrápią y Hebrejcżyki ; ále też ſáme do ƺcżętu záginą.
Wſtał tedy Báláám y odƺedł / á wróćił śię ná miejſce ſwoje ; tákże y Bálák poƺedł w drogę ſwą.
