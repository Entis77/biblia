A gdy bácżył Báláám / że śię podobáło Pánu / áby błogoſłáwił Izráelowi / już nie chodził / jáko przedtem / ráz y drugi dlá wieƺcżby ; ále obróćił ku puƺcży twárz ſwoję.
A podnióſłƺy Báláám ocży ſwe / obácżył Izráelá mieƺkájącego według pokoleń ſwoich / y był nád nim Duch Boży.
Y zácżął przypowieść ſwoję / á mówił :
Rzekł Báláám / ſyn Beorów / rzekł mąż / którego ocży ſą otworzone / rzekł ſłyƺący wymowy Boże / á który widzenie Wƺechmocnego widziáł / który / kiedy pádnie / otworzone má ocży :
Jáko piękne ſą námioty twoje / o Jákubie! przybytki twoje / o Izráelu!
Jáko potoki rozćiągnęły śię / jáko ogrody przy rzece / jáko drzewá wonne / które Pán náſádził / jáko cedry nád wodámi.
Popłynie wodá z wiádrá jego / á náśienie jego będzie nád wodámi obfitemi / á będzie wywyżƺon nád Agágá król jego / á wynieśie śię króleſtwo jego.
Bóg wywiódł go z Egiptu / mocą jednorożcową był mu ; pożre národy przećiwne ſobie / á kośći ich pokruƺy / y ſtrzáłámi ſwemi przerázi je.
Położył śię / leży jáko lwię / y jáko lew śilny ; któż go obudzi? ktobyć błogoſłáwił / błogoſłáwiony / á ktoby ćię przeklináł / przeklęty będzie.
Tedy śię zápálił gniew Báláká ná Báláámá / á kláſnąwƺy rękámi ſwemi / rzekł Bálák do Báláámá : Dlá złorzecżeniá nieprzyjáćiołom moim przyzwáłem ćię / á otoś im błogoſłáwił już po trzy kroć.
Przetoż teráz uchodź ná miejſce ſwoje ; rzekłem ći był : Zácnie ćię ucżcżę ; ále oto pozbáwił ćię Pán tej cżći.
Y rzekł Báláám do Báláká : Izáżem y poſłom twoim / któreś ſłáł do mnie / nie powiedziáł mówiąc :
Choćby mi dáł Bálák pełen dom ſwój ſrebrá y złotá / nie będę mógł przeſtąpić ſłowá Páńſkiego / ábym cżynił co dobrego álbo złego ſám z śiebie ; co mi opowie Pán / to będę mówił.
A teráz oto já odchodzę do ludu mego / jednák oznájmięć / co ucżyni lud ten ludowi twemu ná potem.
Y zácżął przypowieść ſwoję y rzekł : Mówił Báláám / ſyn Beorów / mówił mąż / którego ocży ſą otworzone ;
Mówił ten / który ſłyƺáł wyroki Boże / á który má umiejętność Nájwyżƺego ; który widziáł widzenie Wƺechmocnego ; który / kiedy pádnie / otworzone má ocży :
Ujrzę go / ále nie teráz ; oglądám go / ále nie z bliſká ; wynijdzie gwiázdá z Jákubá y powſtánie láſká z Izráelá / y pobije kśiążętá Moábſkie / y wytráći wƺyſtkie ſyny Setowe.
Y przyjdzie Edom w opánowánie / á Seir będzie pod włádzą nieprzyjáćiół ſwoich / á Izráel będzie ſobie pocżynáł mężnie.
Y będzie pánowáł / który wynijdzie z Jákubá / á wytráći oſtátki z miáſt.
A gdy ſpojrzáł ná Amáleká / zácżął przypowieść ſwoję / y rzekł : Pocżątek národów jeſt Amálek / ále oſtátek jego do ƺcżętu záginie.
Potem wejrzáwƺy ná Kenejcżyká zácżął przypowieść ſwoję y rzekł : Mocneć jeſt mieƺkánie twoje / á záłożyłeś ná ſkále gniázdo twoje ;
Wƺákże ſpuſtoƺony będzie Kenejcżyk / áż ćię Aſſur záprowádzi do więzieniá.
Znowu zácżął przypowieść ſwoję / y rzekł : Ach! któż będzie żyw / gdy to ucżyni Bóg?
Bo okręty przypłyną od brzegów Chyttymſkich / y utrápią Aſſyryjány / utrápią y Hebrejcżyki ; ále też ſáme do ƺcżętu záginą.
Wſtáł tedy Báláám y odƺedł / á wróćił śię ná miejſce ſwoje ; tákże y Bálák poƺedł w drogę ſwą.
