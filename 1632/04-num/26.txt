Y ſtáło śię po onej pládze / że mówił Pán do Mojżeƺá y do Eleázárá / ſyná Aároná kápłáná / mówiąc :
Policżćie pocżet wƺyſtkiego zgromádzeniá ſynów Izráelſkich / od tych / którzy máją dwádźieśćiá lát y wyżej / według domów ojców ich / káżdego któryby mógł wynijść ná wojnę z Izráelá.
Tedy rzekł Mojżeƺ / y Eleázár kápłán do nich ná polách Moábſkich / nád Jordánem przećiw Jerychu / mówiąc :
Licżćie lud / od tych / którzy máją dwádźieśćiá lát y wyżej / jáko był rozkázáł Pán Mojżeƺowi / y ſynom Izráelſkim / gdy wyƺli z źiemi Egipſkiej.
Ruben pierworodny Izráelá : ſynowie Rubenowi Henoch / od którego poƺedł dom Henochytów ; Fállu / od którego dom Fálluitów ;
Y Heſron / od którego dom Heſronitów ; Chármi / od którego dom Chármitów.
Teć ſą domy Rubenitów ; á było ich policżonych cżterdźieśći y trzy tyśiące / śiedm ſet y trzydźieśći.
A ſyn Fállów Elijáb.
Synowie záśię Elijábowi byli : Námuel / y Dátán / y Abiron. A ći / Dátán y Abiron / zácniejśi byli między zgromádzeniem / którzy śię ſwárzyli z Mojżeƺem y z Aáronem w ſpiknieniu Korego / gdy śię byli zbuntowáli przećiwko Pánu.
Y otworzyłá źiemiá uſtá ſwoje / á pożárłá onych / y Korego / gdy zginęłá oná rotá / á pożárł ich ogień dwieśćie y pięćdźieśiąt mężów / którzy śię ſtáli ná przykłád innym ;
Ale ſynowie Korego nie pomárli.
Synowie Symeonowi wedle domów ſwych / ći ſą : Námuel / od którego poƺedł dom Námuelitów ; Jámin / od którego dom Jáminitów ; Jáchin / od którego dom Jáchinitów ; Záre / od którego dom Záreitów ;
Saul / od którego dom Saulitów.
Teć były domy Symeonitów / których było dwádźieśćiá y dwá tyśiące y dwieśćie.
Synowie Gádowi według domów ſwych : Sefon / od którego poƺedł dom Sefonitów ; Aggi / od którego dom Aggitów ; Suni / od którego dom Sunitów.
Ozni / od którego dom Oznitów ; Hery / od którego dom Herytów ;
Arod / od którego dom Arodytów ; Aryjel / od którego dom Aryjelitów.
Teć ſą domy ſynów Gádowych / według pocżtów ich cżterdźieśći tyśięcy y pięć ſet.
Synowie Judowi : Her / y Onán ; ále pomárli Her y Onán w źiemi Chánánejſkiej.
Y byli ſynowie Judowi wedle domów ſwych : Selá / od którego poƺedł dom Seláitów ; Fáreſ / od którego dom Fáreſytów ; Záre / od którego dom Zárejcżyków ;
Byli też ſynowie Fáreſowi : Heſron / od którego dom Heſronitów ; Hámuel / od którego dom Hámuelitów.
Teć ſą domy Judy / według pocżtów ich śiedmdźieśiąt tyśięcy / y ƺeść / y pięć ſet.
Synowie Iſáſchárowi według domów ſwych : Tolá / od którego dom Toláitów ; Fuá / od którego dom Fuáitów ;
Jáſub / od którego dom Jáſubitów ; Semrám / od którego dom Semrámitów.
Teć ſą domy Iſáſchárowe / wedle pocżtów ich ƺeśćdźieśiąt tyśięcy y cżtery / y trzy ſtá.
Synowie Zábulonowi według domów ſwych : Záred / od którego dom Záredcżyków ; Elon / od którego dom Elonitów ; Jáleel / od którego dom Jáleelitów.
A teć ſą domy Zábulonitów / według pocżtów ich ƺeśćdźieśiąt tyśięcy y pięć ſet.
Synowie Józefowi według domów ſwych : Mánáſeſ y Efráim.
Synowie Mánáſeſowi : Máchir / od którego dom Máchirytów ; á Máchir ſpłodźił Gáláádá / od Gáláádá dom Gáláádytów.
Ci ſą ſynowie Gáláádowi : Jezer / od którego dom Jezerytów ; Chelek / od którego dom Chelekitów ;
Y Aſryjel / od którego dom Aſryjelitów ; y Sechem / od którego dom Sechemitów ;
Y Semidá / od którego dom Semidáitów ; y Chefer / od którego dom Cheferytów.
A Sálfáád / ſyn Cheferów / nie miáł ſynów / tylko córki / á imioná córek Sálfáádowych : Máchlá / y Noá / Heglá / Melchá / y Terſá.
Teć ſą domy Mánáſeſowe / á pocżet ich pięćdźieśiąt y dwá tyśiące y śiedm ſet.
Synowie záś Efráimowi według domów ſwych : Sutálá / od którego dom Sutálitów ; Becher / od którego dom Becherytów ; Techen / od którego dom Techenitów.
A ći ſą ſynowie Sutálego : Herán / od którego dom Heránitów.
Teć ſą domy ſynów Efráimowych / według pocżtów ich trzydźieśći tyśięcy y dwá / y pięć ſet. Ci ſą ſynowie Józefowi według domów ſwych.
A ſynowie Benjáminowi według domów ſwych / ći ſą : Belá / od którego dom Belitów ; Aſbel / od którego dom Aſbelitów ; Achirám / od którego dom Achirámitów ;
Sufám / od którego dom Sufámitów ; Hufám / od którego dom Hufámitów.
Byli też ſynowie Beli : Heredá y Noemáná ; z Heredá dom Heredytów / á z Noemáná dom Noemánitów.
Cić ſą ſynowie Benjáminowi / według domów ich / á pocżet ich cżterdźieśći y pięć tyśięcy y ƺeść ſet.
Synowie záś Dánowi według domów ſwych : Suchám / od którego dom Suchámitów. Teć były domy Dánowe według fámilii ich.
Wƺyſtkie domy Suchámitów według pocżtów ich ƺeśćdźieśiąt y cżtery tyśiące y cżtery ſtá.
Synowie Aſerowi według domów ſwych byli : Jemná / od którego dom Jemnitów ; Iſwi / od którego dom Iſwitów ; Beryjá / od którego dom Berytów.
Synowie Beryjego : Heber / od którego dom Heberytów ; Melchyjel / od którego dom Melchyjelitów.
A imię córki Aſerowej było Sárá.
Te ſą domy ſynów Aſerowych / według pocżtów ich pięćdźieśiąt y trzy tyśiące y cżtery ſtá.
Synowie Neftálimowi według domów ſwych : Jáchſel / od którego dom Jáchſelitów ; Guni / od którego dom Gunitów ; Jeſſer / od którego dom Jeſſerytów.
Selem / od którego dom Selemitów.
Teć ſą domy Neftálimowe / według fámilii ich / á pocżet ich cżterdźieśći y pięć tyśięcy y cżtery ſtá.
Táć jeſt licżbá ſynów Izráelſkich / ƺeść kroć ſto tyśięcy y tyśiąc / śiedm ſet y trzydźieśći.
Zátem Pán rzekł do Mojżeƺá / mówiąc :
Między te podźielićie tę źiemię w dźiedźictwo według licżby imion.
Więkƺej licżbie więcej dźiedźictwá dáƺ / á mniejƺej mniejƺe dźiedźictwo dáƺ ; káżdemu według pocżtu policżonych jego będźie dáne dźiedźictwo jego.
Wƺákże loſem niech będźie rozdźieloná źiemiá ; według imion pokoleń ojców ſwych dźiedźictwo bráć będą.
Loſem rozdźielone będźie dźiedźictwo jej / bądź ich wiele bądź máło będźie.
Ci záśię ſą policżeni z Lewitów według domów ſwych : Gerſon / od którego dom Gerſonitów ; Káát / od którego dom Káátytów ; Meráry / od którego dom Merárytów.
Teć ſą domy Lewi : dom Libnitów / dom Hebronitów / dom Moholitów / dom Muſytów / dom Korytów ; á Káát ſpłodźił Amrámá.
A imię żony Amrámowej było Jochábod / córká Lewiego / którá mu śię urodźiłá w Egipćie ; tá Amrámowi urodźiłá Aároná / y Mojżeƺá / y Máryją / śioſtrę ich.
Aáronowi też urodźili śię Nádáb y Abiju / Eleázár y Itámár.
Ale pomárli Nádáb y Abiju / gdy ofiárowáli ogień obcy przed Pánem.
A byłá licżbá ich dwádźieśćiá y trzy tyśiące / wƺyſtkich mężcżyzn urodzonych od mieśiącá y wyżej ; jednák nie byli policżeni między ſyny Izráelſkie / bo im nie dáno dźiedźictwá między ſyny Izráelſkimi.
Ci policżeni byli od Mojżeƺá y Eleázárá kápłáná / którzy policżyli ſyny Izráelſkie ná polách Moábſkich / nád Jordánem przećiw Jerychu.
A między tymi nie był żáden z onych policżonych od Mojżeƺá y Aároná kápłáná / gdy licżyli ſyny Izráelſkie ná puƺcży Synáj ;
Bo rzekł był Pán o nich : Śmierćią pomrą ná puƺcży ; á nie zoſtáł żáden z nich / oprócż Kálebá / ſyná Jefunowego / y Jozuego / ſyná Nunowego.
