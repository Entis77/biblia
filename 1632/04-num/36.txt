Tedy przyſtąpili mężowie przedniejśi z ojców pokoleniá ſynów Gáláádá / ſyná Máchyrowego / ſyná Mánáſeſowego / z domów ſynów Józefowych / y mówili przed Mojżeƺem / y przed kśiążęty przedniejƺymi ojców ſynów Izráelſkich / y rzekli :
Tobie / pánu memu / rozkázáł Pán / ábyś podzielił ziemię w dziedzictwo loſem ſynom Izráelſkim ; nádto pánu memu rozkázáno od Páná / ábyś dáł dziedzictwo Sálfáádá / brátá náƺego / córkom jego.
Które jeźliby kto z inƺego pokoleniá ſynów Izráelſkich wziął zá żony / odjęte będzie ich dziedzictwo od dziedzictwá ojców náƺych / á przyłącży śię do dziedzictwá onego pokoleniá / do któregoby je wzięto zá żony / á ták z loſu dziedzictwá náƺego ubędzie.
A gdy przyjdzie miłośćiwe láto ſynom Izráelſkim / tedy przyłącżone będzie dziedzictwo ich do dziedzictwá onego pokoleniá / do któregoby poƺły zá mąż ; á ták od dziedzictwá pokoleniá ojców náƺych odjęte będzie dziedzictwo ich.
Tedy powiedziáł Mojżeƺ ſynom Izráelſkim według ſłowá Páńſkiego / mówiąc : Dobrze mówi pokolenie ſynów Józefowych.
Toć to jeſt / co rozkázáł Pán o córkách Sálfáádowych / mówiąc : Jáko śię im upodobá / niech idą zá mąż ; tylko w domu pokoleniá ojców ſwoich niech idą zá mąż.
Aby nie było przenoƺone dziedzictwo ſynów Izráelſkich z pokoleniá ná pokolenie ; bo káżdy z ſynów Izráelſkich zoſtáwáć má przy dziedzictwie pokoleniá ojców ſwych.
Y káżdá córká / któráby miáłá dziedzictwo z pokoleń ſynów Izráelſkich / zá kogokolwiek z domu pokoleniá ojcá ſwego pójdzie / żeby otrzymáli dziedzicżnie ſynowie Izráelſcy / káżdy dziedzictwo ojców ſwych.
Bo nie má być przenoƺone dziedzictwo / z pokoleniá ná pokolenie inƺe ; ále káżdy przy dziedzictwie ſwojim zoſtáć má z pokoleniá ſynów Izráelſkich.
Jáko tedy rozkázáł Pán Mojżeƺowi / ták ucżyniły córki Sálfáádowe.
Bo Máhálá / Terſá / y Heglá / y Melchá / y Noá / córki Sálfáádowe / ƺły zá mąż / zá ſyny ſtryjów ſwoich.
W domy ſynów Mánáſeſá / ſyná Józefowego poƺły zá mąż ; y ták zoſtáło dziedzictwo ich przy pokoleniu domu ojcá ich.
Teć ſą przykázániá y práwá / które rozkázáł Pán przez Mojżeƺá ſynom Izráelſkim ná polách Moábſkich / nád Jordánem przećiw Jerychowi.
