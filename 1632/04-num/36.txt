Tedy przyſtąpili mężowie przedniejśi z ojców pokoleniá ſynów Gáláádá / ſyná Máchyrowego / ſyná Mánáſeſowego / z domów ſynów Józefowych / y mówili przed Mojżeƺem / y przed kśiążęty przedniejƺymi ojców ſynów Izráelſkich / y rzekli :
Tobie / pánu memu / rozkázáł Pán / ábyś podźielił źiemię w dźiedźictwo loſem ſynom Izráelſkim ; nádto pánu memu rozkázáno od Páná / ábyś dáł dźiedźictwo Sálfáádá / brátá náƺego / córkom jego.
Które jeſliby kto z inƺego pokoleniá ſynów Izráelſkich wźiął zá żony / odjęte będźie ich dźiedźictwo od dźiedźictwá ojców náƺych / á przyłącży śię do dźiedźictwá onego pokoleniá / do któregoby je wźięto zá żony / á ták z loſu dźiedźictwá náƺego ubędźie.
A gdy przyjdźie miłośćiwe láto ſynom Izráelſkim / tedy przyłącżone będźie dźiedźictwo ich do dźiedźictwá onego pokoleniá / do któregoby poƺły zá mąż ; á ták od dźiedźictwá pokoleniá ojców náƺych odjęte będźie dźiedźictwo ich.
Tedy powiedźiáł Mojżeƺ ſynom Izráelſkim według ſłowá Páńſkiego / mówiąc : Dobrze mówi pokolenie ſynów Józefowych.
Toć to jeſt / co rozkázáł Pán o córkách Sálfáádowych / mówiąc : Jáko śię im upodobá / niech idą zá mąż ; tylko w domu pokoleniá ojców ſwoich niech idą zá mąż.
Aby nie było przenoƺone dźiedźictwo ſynów Izráelſkich z pokoleniá ná pokolenie ; bo káżdy z ſynów Izráelſkich zoſtáwáć má przy dźiedźictwie pokoleniá ojców ſwych.
Y káżdá córká / któráby miáłá dźiedźictwo z pokoleń ſynów Izráelſkich / zá kogokolwiek z domu pokoleniá ojcá ſwego pójdźie / żeby otrzymáli dźiedźicżnie ſynowie Izráelſcy / káżdy dźiedźictwo ojców ſwych.
Bo nie má być przenoƺone dźiedźictwo / z pokoleniá ná pokolenie inƺe ; ále káżdy przy dźiedźictwie ſwojim zoſtáć má z pokoleniá ſynów Izráelſkich.
Jáko tedy rozkázáł Pán Mojżeƺowi / ták ucżyniły córki Sálfáádowe.
Bo Máhálá / Terſá / y Heglá / y Melchá / y Noá / córki Sálfáádowe / ƺły zá mąż / zá ſyny ſtryjów ſwoich.
W domy ſynów Mánáſeſá / ſyná Józefowego poƺły zá mąż ; y ták zoſtáło dźiedźictwo ich przy pokoleniu domu ojcá ich.
Teć ſą przykázániá y práwá / które rozkázáł Pán przez Mojżeƺá ſynom Izráelſkim ná polách Moábſkich / nád Jordánem przećiw Jerychowi.
