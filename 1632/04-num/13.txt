Potem ruƺył śię lud z Háſerotu / á położyli śię obozem ná puƺcży Fárán.
Y rzekł Pán do Mojżeƺá / mówiąc :
Poślij męże / którzyby przeƺpiegowáli źiemię Chánánejſką / którą Já dáwám ſynom Izráelſkim ; po jednym mężu z káżdego pokoleniá poślećie / którzyby byli przedniejśi między nimi.
Wyſłáł je tedy Mojżeƺ z puƺcży Fárán według rozkázániá Páńſkiego ; á oni mężowie wƺyſcy byli co przedniejśi z ſynów Izráelſkich.
A teć ſą imioná ich : Z pokoleniá Ruben Sámuá / ſyn Záchurów.
Z pokolenie Symeonowego Sáfát / ſyn Hurów.
Z pokoleniá Judy Káleb / ſyn Jefunów.
Z pokoleniá Iſáſchár Igál / ſyn Józefów.
Z pokoleniá Efráimowego Ozeáƺ / ſyn Nunów.
Z pokoleniá Benjáminowego Fálty / ſyn Ráfuów.
Z pokoleniá Zábulon Gedyjel / ſyn Sodego.
Z pokoleniá Józefowego / to jeſt z potomſtwá Mánáſeſowego / Gády / ſyn Suſego.
Z pokoleniá Dán Ammijel / ſyn Gemmálego.
Z pokoleniá Aſer Setur / ſyn Micháelów.
Z pokoleniá Neftáli Nábi / ſyn Wáfſego.
Z pokoleniá Gád Guel / ſyn Máchego.
Teć ſą imioná mężów / które poſłáł Mojżeƺ ná przeƺpiegowánie źiemi : y názwáł Mojżeƺ Ozeáƺá / ſyná Nunowego / Jozue.
A poſyłájąc je Mojżeƺ ná przeƺpiegowánie źiemi Chánánejſkiej / mówił do nich : Idźćie w tę ſtronę ku południowi / á wnijdźćie ná górę :
Y oglądájćie źiemię / jáká jeſt / y lud / który mieƺká w niej / jeźli jeſt mocny / cżyli mdły? jeźli ich máło / cżyli wiele?
Jáká też jeſt źiemiá / w której mieƺkáją / jeźli dobrá / cżyli złá? y co ſą zá miáſtá / w których mieƺkáją / jeźli w námiećiech / cżyli w obronnych miejſcách?
Tákże co zá źiemiá / jeźli urodzájná / cżyli niepłodná? jeźli w niej ſą drzewá / cżyli nie? á bądźćie mężnego ſercá y przynieśćie nám owocu támtej źiemi ; á było to w ten cżáſ / gdy śię winá doſtáwáły.
Y ƺli / á przeƺpiegowáli źiemię od puƺcży Syn áż do Rochob / którędy chodzą do Emát.
A idąc ku południowi przyƺli áż do Hebron / gdźie byli Ahimán / Seſái y Tálmái / ſynowie Enákowi ; á Hebron śiedm lát było zbudowáne przed Soán Egipſkiem.
Przyƺli potem áż do rzeki Eſchol / y urznęli tám gáłąź z gronem jednem jágód winnych / y nieśli ją ná drążku / dwá tákże gránátowe jábłká y figi.
Y názwáno miejſce ono Náchál Eſchol / od groná winnego / które tám byli urznęli ſynowie Izráelſcy.
Zátem wróćili śię od onego ƺpiegowániá źiemi po wyjśćiu cżterdźieſtu dni.
A wróćiwƺy śię / przyƺli do Mojżeƺá y do Aároná / y do wƺyſtkiego zgromádzeniá ſynów Izráelſkich ná puƺcżą Fárán / którá jeſt w Kádeſ ; y dáli im y wƺyſtkiemu poſpólſtwu ſpráwę / ukázáwƺy im owoc onej źiemi.
A powiádáli im / mówiąc : Przyƺliśmy do źiemi / do którejś náſ był poſłáł / którá w práwdźie opływá mlekiem y miodem / á oto / ten jeſt owoc jej ;
Tylko że mocny jeſt lud / który mieƺká w onej źiemi ; miáſtá tákże obronne ſą / y bárdzo wielkie ; do tego y ſyny Enákowe támeśmy widźieli.
Amálek mieƺká w źiemi ná południe / á Hetejcżyk / y Jebuzejcżyk / y Amorejcżyk mieƺká ná górách ; Chánánejcżyk záś mieƺká nád morzem / y nád brzegiem Jordánu.
Y hámowáł Káleb lud ƺemrzący przećiw Mojżeƺowi / y mówił : Pójdźmy á pośiądźmy źiemię / bo ją pewnie ſobie podbijemy.
Ale mężowie oni / którzy z nim chodźili / rzekli : Nie będźiem mogli ćiągnąć przećiw támtemu ludowi ; bo mocniejƺy jeſt nád náſ.
Y zgánili onę źiemię / którą byli przeƺpiegowáli / ſynom Izráelſkim / mówiąc : Ziemiá tá / którąśmy przeƺli / y przeƺpiegowáli ją / jeſt źiemiá pożerájącá obywátele ſwoje / á lud wƺyſtek / któryśmy w niej widźieli / lud jeſt wyſokiego wzroſtu.
Támeśmy też widźieli olbrzymy / ſyny Enákowe z rodu olbrzymów ; y zdáliśmy śię ſobie przy nich jáko ƺáráńcżá / tákimiż zdáliśmy śię y onym.
