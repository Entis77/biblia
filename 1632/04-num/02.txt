Zatym rzekł PAN do Mojzeƺá y do Aároná / mówiąc :
Káżdy z Synów Izráelſkich / kłáść śię będą Obozem pod chorągwią ſwoją według znáków domów Ojców ſwych / ná przećiwko około Namiotu zgromádzenia / kłáść śię będą.
A ći śię Obozem położą ná wſchód ſłońcá : Chorągiew wojſká Judowego według hufców ſwych : á Hetmánem nád Syny Judowymi / Náháſon Syn Aminádábów :
A w Wojſku jego policzonych / śiedmdźieśiąt y cztery tyśiące / y ƺeść ſet.
Podle niego położy śię Obozem / pokolenie Iſáſchárowe : á Hetmánem nád Syny Iſáſchárowymi / Nátánáel Syn Suhárów :
A w Wojſku jego policzonych / pięćdźieśiąt y cztery tyśiące / y cztery ſtá.
<i>Podle nich</i> pokolenie Zábulonowe : á Hetmánem nád Syny Zábulonowemi / Elijáb Syn Helonów :
A w Wojſku jego policzonych / pięćdźieśiąt y śiedm tyśięcy / y cztery ſtá.
Wƺyſtkich policzonych w Oboźie Judowym / ſto tyśięcy / ośmdźieśiąt tyśięcy / y ƺeść tyśięcy / y cztery ſtá : według hufców ich : ći naprzód poćiągną.
Chorągiew Obozu Rubenowego / <i>położy śię</i> ná południe według hufców ſwych : á Hetmánem nád Syny Rubenowymi / Elizur Syn Sedeurów :
A w Wojſku jego policzonych / czterdźieśći y ƺeść tyśięcy / y pięć ſet.
Podle niego położy śię Obozem pokolenie Symeonowe : á Hetmánem nád Syny Symeonowymi / Sálámiel Syn Suri Sáddáj :
A w Wojſku jego policzonych / pięćdźieśiąt y dźiewięć tyśięcy / y trzy ſtá.
Potym pokolenie Gádowe : á Hetmánem nád Syny Gádowymi / Eliáſáf Syn Rehuelów :
A w Wojſku jego policzonych czterdźieśći y pięć tyśięcy / y ƺeść ſet / y pięćdźieśiąt.
Wƺyſtkich policzonych w Oboźie Rubenowym / ſto tyśięcy / pięćdźieśiąt tyśięcy y jeden / cztery ſtá y pięćdźieśiąt / według hufców ich / á ći w rzędźie wtórym poćiągną.
Potym pójdźie Namiot zgromádzenia z Wojſkiem Lewitów / w pośrzodku Wojſká / jákim porządkiem ſtawáć będą Obozem / tákim poćiągną / káżdy w ƺyku ſwym / pod Chorągwią ſwoją.
Chorągiew Obozu Efráimowego / według Hufców ſwych / ku Zachodu : á Hetmánem nád Syny Efráimowymi / Eliſám Syn Hammiádów :
A w Wojſku jego policzonych / czterdźieśći tyśięcy / y pięć ſet.
A podle niego pokolenie Mánáſſeſſowe : á Hetmánem nád Syny Mánáſſeſſowymi / Gámáliel Syn Pedáſurów :
A w Wojſku jego policzonych / trzydźieśći y dwá tyśiące / y dwieśćie.
Podle <i>nich</i> Pokolenie Benjáminowe : á Hetmánem nád Syny Benjáminowymi / Abydán Syn Gedeonów :
A w Wojſku jego policzonych / trzydźieśći y pięć tyśięcy / y cztery ſtá.
Wƺyſtkich policzonych w Oboźie Efráimowym / ſto tyśięcy / y ośm tyśięcy / y ſto / według hufców ſwojich / á <i>ći</i> w trzećim rzędźie poćiągną.
Chorągiew Obozu Dánowego <i>położy śię</i> ku Pułnocy według hufców ſwych : á Hetmánem nád Syny Dánowemi / Achiezer Syn Ammiſádájów :
A w Wojſku jego policzonych / ƺeśćdźieśiąt y dwá tyśiące / y śiedm ſet.
A podle niego położy śię Obozem pokolenie Aſerowe : á Hetmánem nád Syny Aſerowymi / Págiel Syn Ochránów :
A w Wojſku jego policzonych / czterdźieśći tyśięcy y jeden / y pięć ſet.
Potym pokolenie Neftálimowe : á Hetmánem nád Syny Neftálimowymi / Ahirá Syn Enánów :
A w Wojſku jego policzonych / pięćdźieśiąt / y trzy tyśiące / y cztery ſtá.
<i>A ták</i> wƺyſtkich policzonych Obozu Dánowego / ſto tyśięcy / pięćdźieśiąt y śiedm tyśięcy / y ƺeść ſet. A ći ná oſtátku poćiągną pod chorągwią ſwoją.
Cić ſą policzeni Synów Izráelſkich według domów Ojców ich / wƺyſtkich policzonych w Oboźie / według hufców ich / ƺeść kroć ſto tyśięcy / y trzy tyśiące / y pięć ſet / y pięćdźieśiąt.
Ale Lewitów nie liczono miedzy Syny Izráelſkie / jáko był PAN rozkazał Mojzeƺowi.
Y uczynili Synowie Izráelſcy według wƺyſtkiego / jáko rozkazał PAN Mojzeƺowi / ták śię ſtánowili Obozem przy Chorągwiách ſwych : y ćiągnęli káżdy według fámilii ſwych / <i>y</i> według domów Ojców ſwych.
