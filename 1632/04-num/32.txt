Y mieli ſynowie Rubenowi / y ſynowie Gádowi bydłá bárdzo wiele ; á obácżywƺy źiemię Jázer y źiemię Gáláád / że miejſce ono było ſpoſobne dlá bydłá /
Przyƺli ćiż ſynowie Gádowi / y ſynowie Rubenowi / y mówili do Mojżeƺá y do Eleázárá kápłáná / y do kśiążąt zgromádzeniá / y rzekli :
Ziemiá Atáret y Dybon / y Jázer / y Nemrá / y Heſebon / y Eleále / y Sebán / y Nebo / y Beon :
Ziemiá / którą zwojowáł Pán przed zgromádzeniem Izráelſkiem / jeſt źiemiá ſpoſobná dlá bydłá / á my ſłudzy twoi mámy bydłá wiele. Przetoż rzekli :
Jeźliśmy ználeźli łáſkę przed ocżymá twemi / niechże będźie dáná tá źiemiá ſługom twym ná ośiádłość / á niech nie chodźimy zá Jordán.
Tedy odpowiedźiáł Mojżeƺ ſynom Gádowym / y ſynom Rubenowym : Więc bráćiá wáśi pójdą ná wojnę / á wy tu śiedźieć będźiećie?
Cżemuż pſujećie ſerce ſynom Izráelſkim / żeby nie ƺli do źiemi / którą im dáł Pán?
Tákći ucżynili ojcowie wáśi / gdym je był poſłáł z Kádeſ Bárne ku przeƺpiegowániu tej źiemi ;
Bo gdy przyƺli áż do doliny Eſchol / obejrzáwƺy onę źiemię popſowáli ſerce ſynom Izráelſkim / áby nie ƺli do źiemi / którą im dáł Pán ;
Skąd zápáliwƺy śię gniewem Pán / dniá onego przyśiągł / mówiąc :
Záiſte nie oglądáją ludźie ći / którzy wyƺli z Egiptu / od dwudźieſtu lát y wyżej / tej źiemi / o którąm przyśiągł Abráhámowi / Izáákowi / y Jákóbowi / przeto iż mię cále nie náśládowáli ;
Oprócż Kálebá / ſyná Jefunowego / Kenezejcżyká / y Jozuego / ſyná Nunowego / poniewáż ći cále náśládowáli Páná.
Y zápálił śię gniewem Pán ná Izráelá / y ſpráwił / że śię tułáli po puƺcży przez cżterdźieśći lát / áż poginął wƺyſtek on náród / który cżynił źle przed ocżymá Páńſkimi.
A oto / wy powſtáliśćie miáſto ojców wáƺych / plemię ludźi grzeƺnych / ábyśćie jeƺcże przycżynili gniewu zápálcżywośći Páńſkiej przećiwko Izráelowi.
Bo jeźli śię odwróćićie od náśládowániá jego / tedy on też zániechá go jeƺcże ná tej puƺcży ; á ták wy zgubićie ten wƺyſtek lud.
Tedy przyſtąpiwƺy do niego rzekli : Obory bydłu y dobytkowi náƺemu / y miáſtá dźiátkom náƺym tu pobudujemy ;
Ale ſámi zbrojno ochotnie pójdźiemy przed ſyny Izráelſkimi / áż je záprowádźimy ná miejſce ich / á dźiátki náƺe będą mieƺkáły w mieśćiech obronnych dlá obywáteli tej źiemi.
Nie wróćimy śię do domów náƺych / áż pośiądą ſynowie Izráelſcy káżdy dźiedźictwo ſwoje ;
Ani weźmiemy dźiedźictwá z nimi zá Jordánem y dálej / poniewáż przychodźi dźiedźictwo náƺe ná náſ z tej ſtrony Jordánu ná wſchód ſłońcá.
Y rzekł im Mojżeƺ : Jeźliż ucżynićie / cośćie rzekli / á pójdźiećie zbrojno przed oblicżem Páńſkim ná wojnę ;
Y pójdźie káżdy z wáſ zbrojno zá Jordán przed oblicżnośćią Páńſką / áżby wypędźił nieprzyjáćioły ſwoje od oblicżá ſwego ;
Y áż będźie poddáná źiemiá oná Pánu / á potem śię wróćićie / y będźiećie bez winy przed Pánem y przed Izráelem : tedy wám będźie tá źiemiá zá ośiádłość przed oblicżem Páńſkim.
Ale jeźli tego nie ucżynićie / oto zgrzeƺyćie Pánu / á wiedzćie / że grzech wáƺ znájdźie wáſ.
Budujćież tedy miáſtá dźiátkom wáƺym / y obory bydłu wáƺemu / á co wyƺło z uſt wáƺych / ucżyńćie.
Tedy rzekli ſynowie Gádowi / y ſynowie Rubenowi do Mojżeƺá / mówiąc : Słudzy twoi ucżynią / jáko pán náƺ rozkázuje.
Dźiátki náƺe / y żony náƺe / ſtádá náƺe / y wƺyſtkie bydłá náƺe / zoſtáną tu w mieśćiech Gáláádzkich ;
Ale ſłudzy twoi pójdą wƺyſcy zbrojno przed Pánem ná wojnę / jáko pán náƺ mówi.
Y przykázáł o nich Mojżeƺ Eleázárowi kápłánowi / y Jozuemu / ſynowi Nunowemu / y kśiążętom ojców pokoleń ſynów Izráelſkich /
Y rzekł im : Jeźli przejdą ſynowie Gádowi / y ſynowe Rubenowi z wámi zá Jordán / wƺyſcy zbrojno ná wojnę przed Pánem / á będźie pohołdowáná źiemiá przed wámi / tedy im dáćie źiemię Gáláád w dźiedźictwo ;
Ale jeźli nie zbrojno z wámi przejdą / tedy niech máją dźiedźictwo między wámi w źiemi Chánánejſkiej.
Y odpowiedźieli ſynowie Gádowi / y ſynowie Rubenowi / mówiąc : Co wyrzekł Pán do ſług ſwoich / to ucżynimy ;
Pójdźiemy zbrojno przed Pánem do źiemi Chánánejſkiej / á zoſtánie przy náſ ośiádłość dźiedźictwá náƺego z tej ſtrony Jordánu.
Dáł tedy Mojżeƺ ſynom Gádowym / y ſynom Rubenowym / y połowie pokoleniá Mánáſeſá / ſyná Józefowego / króleſtwo Sehoná / królá Amorejſkiego / y króleſtwo Ogá / królá Báſáńſkiego / źiemię z miáſty jej / z gránicámi / y miáſtá źiemi onej w około.
Y pobudowáli ſynowie Gádowi Dybon / y Atárot / y Aroer :
Y Atrot / y Sofán / y Jázer / y Jegbá /
Y Betnimerá / y Betárán / miáſtá obronne / y obory dlá bydłá.
Synowie też Rubenowi pobudowáli Heſebon / y Eleále / y Káryjátáim.
Y Nebo / y Báálmeon / odmieniwƺy im imioná / tákże Sábámá ; y dáli imioná inƺe onym miáſtom / które pobudowáli.
Wtárgnęli też ſynowie Máchyrá / ſyná Mánáſeſowego / do Gáláád / á wźiąwƺy je / wygnáli Amorejcżyká / który tám mieƺkáł.
Y dáł Mojżeƺ Gáláád Máchyrowi / ſynowi Mánáſeſowemu / y mieƺkáł w nim.
Potem Jáir / ſyn Mánáſeſów / wtárgnął / y pobráł wśi ich / które przezwáł Cháwot Jáir.
Tákże Nobe wtárgnął / y wźiął Kánát z jego wśiámi / y názwáł je Nobe od imieniá ſwego.
