Y mieli ſynowie Rubenowi / y ſynowie Gádowi bydłá bárzo wiele ; á obácżywƺy źiemię Jázer y źiemię Gáláád / że miejſce ono było ſpoſobne dla bydłá /
Przyƺli ćiż ſynowie Gádowi / y ſynowie Rubenowi / y mówili do Mojżeƺá y do Eleázárá kápłáná / y do kśiążąt zgromádzenia / y rzekli :
Ziemiá Atáret y Dybon / y Jázer / y Nemrá / y Heſebon / y Eleále / y Sebán / y Nebo / y Beon :
Ziemiá / którą zwojował Pán przed zgromádzeniem Izráelſkiem / jeſt źiemiá ſpoſobná dla bydłá / á my ſłudzy twoi mamy bydłá wiele. Przetoż rzekli :
Jeſliśmy ználeſli łáſkę przed ocżymá twemi / niechże będźie dáná tá źiemiá ſługom twym ná ośiádłość / á niech nie chodźimy zá Jordán.
Tedy odpowiedźiał Mojzeƺ ſynom Gádowym / y ſynom Rubenowym : Więc bráćia wáśi pójdą ná wojnę / á wy tu śiedźieć będźiećie?
Cżemuż pſujećie ſerce ſynom Izráelſkim / żeby nie ƺli do źiemi / którą im dał Pán?
Tákći ucżynili ojcowie wáśi / gdym je był poſłał z Kádes Bárne ku przeƺpiegowániu tey źiemi ;
Bo gdy przyƺli áż do doliny Eſchol / obejrzáwƺy onę źiemię popſowáli ſerce ſynom Izráelſkim / áby nie ƺli do źiemi / którą im dał Pán ;
Skąd zápáliwƺy śię gniewem Pán / dniá onego przyśiągł / mówiąc :
Zájiſte nie oglądáją ludźie ći / którzy wyƺli z Egiptu / od dwudźieſtu lat y wyżey / tey źiemi / o którąm przyśiągł Abráhámowi / Izáákowi / y Jákóbowi / przeto iż mię cále nie náśládowáli ;
Oprócż Kálebá / ſyná Jefunowego / Kenezejcżyká / y Jozuego / ſyná Nunowego / ponieważ ći cále náśládowáli Páná.
Y zápálił śię gniewem Pán ná Izráelá / y ſpráwił / że śię tułáli po puƺcży przez cżterdźieśći lat / áż poginął wƺyſtek on naród / który cżynił źle przed ocżymá Páńſkimi.
A oto / wy powſtáliśćie miáſto ojców wáƺych / plemię ludźi grzeƺnych / ábyśćie jeƺcże przycżynili gniewu zápálcżywośći Páńſkiej przećiwko Izráelowi.
Bo jeſli śię odwróćićie od náśládowánia jego / tedy on też zániechá go jeƺcże ná tey puƺcży ; á ták wy zgubićie ten wƺyſtek lud.
Tedy przyſtąpiwƺy do niego rzekli : Obory bydłu y dobytkowi náƺemu / y miáſtá dźiátkom náƺym tu pobudujemy ;
Ale ſámi zbrojno ochotnie pójdźiemy przed ſyny Izráelſkimi / áż je záprowádźimy ná miejſce ich / á dźiátki náƺe będą mieƺkáły w mieśćiech obronnych dla obywáteli tey źiemi.
Nie wróćimy śię do domów náƺych / áż pośiądą ſynowie Izráelſcy káżdy dźiedźictwo ſwoje ;
Ani weźmiemy dźiedźictwá z nimi zá Jordánem y dálej / ponieważ przychodźi dźiedźictwo náƺe ná nas z tey ſtrony Jordánu ná wſchód ſłońcá.
Y rzekł im Mojzeƺ : Jeſliż ucżynićie / cośćie rzekli / á pójdźiećie zbrojno przed oblicżem Páńſkim ná wojnę ;
Y pójdźie káżdy z was zbrojno zá Jordán przed oblicżnośćią Páńſką / áżby wypędźił nieprzyjáćioły ſwoje od oblicża ſwego ;
Y áż będźie poddáná źiemiá oná Pánu / á potem śię wróćićie / y będźiećie bez winy przed Pánem y przed Izráelem : tedy wam będźie tá źiemiá zá ośiádłość przed oblicżem Páńſkim.
Ale jeſli tego nie ucżynićie / oto zgrzeƺyćie Pánu / á wiedzćie / że grzech wáƺ znájdźie was.
Budujćież tedy miáſtá dźiátkom wáƺym / y obory bydłu wáƺemu / á co wyƺło z uſt wáƺych / ucżyńćie.
Tedy rzekli ſynowie Gádowi / y ſynowie Rubenowi do Mojżeƺá / mówiąc : Słudzy twoi ucżynią / jáko pán náƺ rozkázuje.
Dźiátki náƺe / y żony náƺe / ſtádá náƺe / y wƺyſtkie bydłá náƺe / zoſtáną tu w mieśćiech Gáláádzkich ;
Ale ſłudzy twoi pójdą wƺyſcy zbrojno przed Pánem ná wojnę / jáko pán náƺ mówi.
Y przykázał o nich Mojzeƺ Eleázárowi kápłánowi / y Jozuemu / ſynowi Nunowemu / y kśiążętom ojców pokoleń ſynów Izráelſkich /
Y rzekł im : Jeſli przejdą ſynowie Gádowi / y Synowie Rubenowi z wámi zá Jordán / wƺyſcy zbrojno ná wojnę przed Pánem / á będźie pohołdowáná źiemiá przed wámi / tedy im dáćie źiemię Gáláád w dźiedźictwo ;
Ale jeſli nie zbrojno z wámi przejdą / tedy niech máją dźiedźictwo miedzy wámi w źiemi Chánánejſkiej.
Y odpowiedźieli ſynowie Gádowi / y ſynowie Rubenowi / mówiąc : Co wyrzekł Pán do ſług ſwojich / to ucżynimy ;
Pójdźiemy zbrojno przed Pánem do źiemi Chánánejſkiej / á zoſtánie przy nas ośiádłość dźiedźictwá náƺego z tey ſtrony Jordánu.
Dał tedy Mojzeƺ ſynom Gádowym / y ſynom Rubenowym / y połowie pokolenia Mánáſeſá / ſyná Józefowego / króleſtwo Sehoná / królá Amorejſkiego / y króleſtwo Ogá / królá Báſáńſkiego / źiemię z miáſty jey / z gránicámi / y miáſtá źiemi oney w około.
Y pobudowáli ſynowie Gádowi Dybon / y Atárot / y Aroer :
Y Atrot / y Sofán / y Jázer / y Jegbá /
Y Betnimerá / y Betárán / miáſtá obronne / y obory dla bydłá.
Synowie też Rubenowi pobudowáli Heſebon / y Eleále / y Káryjátáim.
Y Nebo / y Báálmeon / odmieniwƺy im imioná / tákże Sábámá ; y dáli imioná inƺe onym miáſtom / które pobudowáli.
Wtárgnęli też ſynowie Máchyrá / ſyná Mánáſeſowego / do Gáláád / á wźiąwƺy je / wygnáli Amorejcżyká / który tám mieƺkał.
Y dał Mojzeƺ Gáláád Máchyrowi / ſynowi Mánáſeſowemu / y mieƺkał w nim.
Potem Jáir / ſyn Mánáſeſów / wtárgnął / y pobrał wśi ich / które przezwał Cháwot Jáir.
Tákże Nobe wtárgnął / y wźiął Kánát z jego wśiámi / y názwał je Nobe od imienia ſwego.
