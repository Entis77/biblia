Y mieli ſynowie Rubenowi / y ſynowie Gádowi bydłá bárdzo wiele ; á obácżywƺy ziemię Jázer y ziemię Gáláád / że miejſce ono było ſpoſobne dlá bydłá /
Przyƺli ćiż ſynowie Gádowi / y ſynowie Rubenowi / y mówili do Mojżeƺá y do Eleázárá kápłáná / y do kśiążąt zgromádzeniá / y rzekli :
Ziemiá Atáret y Dybon / y Jázer / y Nemrá / y Heſebon / y Eleále / y Sebán / y Nebo / y Beon :
Ziemiá / którą zwojowáł Pán przed zgromádzeniem Izráelſkiem / jeſt ziemiá ſpoſobná dlá bydłá / á my ſłudzy twoi mámy bydłá wiele. Przetoż rzekli :
Jeźliśmy ználeźli łáſkę przed ocżymá twemi / niechże będzie dáná tá ziemiá ſługom twym ná ośiádłość / á niech nie chodzimy zá Jordán.
Tedy odpowiedziáł Mojżeƺ ſynom Gádowym / y ſynom Rubenowym : Więc bráćiá wáśi pójdą ná wojnę / á wy tu śiedzieć będziećie?
Cżemuż pſujećie ſerce ſynom Izráelſkim / żeby nie ƺli do ziemi / którą im dáł Pán?
Tákći ucżynili ojcowie wáśi / gdym je był poſłáł z Kádeſ Bárne ku przeƺpiegowániu tej ziemi ;
Bo gdy przyƺli áż do doliny Eſchol / obejrzáwƺy onę ziemię popſowáli ſerce ſynom Izráelſkim / áby nie ƺli do ziemi / którą im dáł Pán ;
Skąd zápáliwƺy śię gniewem Pán / dniá onego przyśiągł / mówiąc :
Záiſte nie oglądáją ludzie ći / którzy wyƺli z Egiptu / od dwudzieſtu lát y wyżej / tej ziemi / o którąm przyśiągł Abráhámowi / Izáákowi / y Jákóbowi / przeto iż mię cále nie náśládowáli ;
Oprócż Kálebá / ſyná Jefunowego / Kenezejcżyká / y Jozuego / ſyná Nunowego / poniewáż ći cále náśládowáli Páná.
Y zápálił śię gniewem Pán ná Izráelá / y ſpráwił / że śię tułáli po puƺcży przez cżterdzieśći lát / áż poginął wƺyſtek on náród / który cżynił źle przed ocżymá Páńſkimi.
A oto / wy powſtáliśćie miáſto ojców wáƺych / plemię ludzi grzeƺnych / ábyśćie jeƺcże przycżynili gniewu zápálcżywośći Páńſkiej przećiwko Izráelowi.
Bo jeźli śię odwróćićie od náśládowániá jego / tedy on też zániechá go jeƺcże ná tej puƺcży ; á ták wy zgubićie ten wƺyſtek lud.
Tedy przyſtąpiwƺy do niego rzekli : Obory bydłu y dobytkowi náƺemu / y miáſtá dziátkom náƺym tu pobudujemy ;
Ale ſámi zbrojno ochotnie pójdziemy przed ſyny Izráelſkimi / áż je záprowádzimy ná miejſce ich / á dziátki náƺe będą mieƺkáły w mieśćiech obronnych dlá obywáteli tej ziemi.
Nie wróćimy śię do domów náƺych / áż pośiądą ſynowie Izráelſcy káżdy dziedzictwo ſwoje ;
Ani weźmiemy dziedzictwá z nimi zá Jordánem y dálej / poniewáż przychodzi dziedzictwo náƺe ná náſ z tej ſtrony Jordánu ná wſchód ſłońcá.
Y rzekł im Mojżeƺ : Jeźliż ucżynićie / cośćie rzekli / á pójdziećie zbrojno przed oblicżem Páńſkim ná wojnę ;
Y pójdzie káżdy z wáſ zbrojno zá Jordán przed oblicżnośćią Páńſką / áżby wypędził nieprzyjáćioły ſwoje od oblicżá ſwego ;
Y áż będzie poddáná ziemiá oná Pánu / á potem śię wróćićie / y będziećie bez winy przed Pánem y przed Izráelem : tedy wám będzie tá ziemiá zá ośiádłość przed oblicżem Páńſkim.
Ale jeźli tego nie ucżynićie / oto zgrzeƺyćie Pánu / á wiedzćie / że grzech wáƺ znájdzie wáſ.
Budujćież tedy miáſtá dziátkom wáƺym / y obory bydłu wáƺemu / á co wyƺło z uſt wáƺych / ucżyńćie.
Tedy rzekli ſynowie Gádowi / y ſynowie Rubenowi do Mojżeƺá / mówiąc : Słudzy twoi ucżynią / jáko pán náƺ rozkázuje.
Dziátki náƺe / y żony náƺe / ſtádá náƺe / y wƺyſtkie bydłá náƺe / zoſtáną tu w mieśćiech Gáláádzkich ;
Ale ſłudzy twoi pójdą wƺyſcy zbrojno przed Pánem ná wojnę / jáko pán náƺ mówi.
Y przykázáł o nich Mojżeƺ Eleázárowi kápłánowi / y Jozuemu / ſynowi Nunowemu / y kśiążętom ojców pokoleń ſynów Izráelſkich /
Y rzekł im : Jeźli przejdą ſynowie Gádowi / y ſynowe Rubenowi z wámi zá Jordán / wƺyſcy zbrojno ná wojnę przed Pánem / á będzie pohołdowáná ziemiá przed wámi / tedy im dáćie ziemię Gáláád w dziedzictwo ;
Ale jeźli nie zbrojno z wámi przejdą / tedy niech máją dziedzictwo między wámi w ziemi Chánánejſkiej.
Y odpowiedzieli ſynowie Gádowi / y ſynowie Rubenowi / mówiąc : Co wyrzekł Pán do ſług ſwoich / to ucżynimy ;
Pójdziemy zbrojno przed Pánem do ziemi Chánánejſkiej / á zoſtánie przy náſ ośiádłość dziedzictwá náƺego z tej ſtrony Jordánu.
Dáł tedy Mojżeƺ ſynom Gádowym / y ſynom Rubenowym / y połowie pokoleniá Mánáſeſá / ſyná Józefowego / króleſtwo Sehoná / królá Amorejſkiego / y króleſtwo Ogá / królá Báſáńſkiego / ziemię z miáſty jej / z gránicámi / y miáſtá ziemi onej w około.
Y pobudowáli ſynowie Gádowi Dybon / y Atárot / y Aroer :
Y Atrot / y Sofán / y Jázer / y Jegbá /
Y Betnimerá / y Betárán / miáſtá obronne / y obory dlá bydłá.
Synowie też Rubenowi pobudowáli Heſebon / y Eleále / y Káryjátáim.
Y Nebo / y Báálmeon / odmieniwƺy im imioná / tákże Sábámá ; y dáli imioná inƺe onym miáſtom / które pobudowáli.
Wtárgnęli też ſynowie Máchyrá / ſyná Mánáſeſowego / do Gáláád / á wziąwƺy je / wygnáli Amorejcżyká / który tám mieƺkáł.
Y dáł Mojżeƺ Gáláád Máchyrowi / ſynowi Mánáſeſowemu / y mieƺkáł w nim.
Potem Jáir / ſyn Mánáſeſów / wtárgnął / y pobráł wśi ich / które przezwáł Cháwot Jáir.
Tákże Nobe wtárgnął / y wziął Kánát z jego wśiámi / y názwáł je Nobe od imieniá ſwego.
