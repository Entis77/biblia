Potem rzekł Pán do Aároná : Ty y ſynowie twoi / y dom ojcá twego z tobą / ponieśiećie niepráwość świątnicy. Y ty y ſynowie twoi z tobą ponieśiećie niepráwość kápłáńſtwá wáƺego.
Bráćią tákże twoję / pokolenie Lewiego / ród ojcá twego / przypuśćiƺ do śiebie / áby byli przy tobie / y poſługowáli tobie ; á ty y ſynowie twoi z tobą ſłużyć będźiećie przed námiotem świádectwá.
Oni będą przeſtrzegáli rozkázániá twego / y pilnowáli wƺyſtkiego námiotu ; wƺákże do nácżyniá śwątnicy / y do ołtárzá / przyſtępowáć nie będą / áby nie pomárli / y oni y wy.
Y przyłącżą śię do ćiebie / pilnie ſtrzegąc námiotu zgromádzeniá w káżdej uſłudze námiotu ; á nikt obcy niecháj śię nie mięƺá między was.
Wy tedy pilnie ſtrzeżćie świątnicy / y uſługi ołtárzowej / by śię nápotem nie wzruƺył gniew przećiwko ſynom Izráelſkim.
Bom oto ja obráł bráćią wáƺę Lewity / z pośród ſynów Izráelſkich / wám zá dár / oddáne Pánu / áby odpráwowáli uſługę w námioćie zgromádzeniá.
Ale ty y ſynowie twoi z tobą / przeſtrzegáć będźiećie kápłáńſtwá wáƺego przy káżdej uſłudze ołtárzowej / y zá záſłoną ſłużyć będźiećie ; urząd kápłáńſtwá wáƺego dáłem wám dármo ; gdyby kto obcy przyſtąpił / umrze.
Nádto mówił Pán do Aároná : Otom ja dáł tobie pod ſtráż ofiáry moje podnoƺone ; wƺyſtkie rzecży poświęcone od ſynów Izráelſkich tobiem je dáł dla pomázániá / y ſynom twoim práwem wiecżnem.
To będźie twoje z rzecży poświęconych / które nie bywáją pálone. Káżdá ofiárá ich / bądź ofiárá śniedná ich / bądź ofiárá zá grzech ich / álbo ofiárá zá wyſtępek ich / cokolwiek mi oddáwáć będą / to będźie rzecżą poświęconą tobie y ſynom twoim /
Ná miejſcu nájświętƺem jádáć to będźieƺ ; wƺelki mężcżyzná będźie jádł z tego ; rzecżą poświęconą to będźie tobie.
To też twoje będźie / ofiárá podnoƺeniá dárów ich ze wƺyſtkiemi ofiárámi tám y ſám obrácániá ſynów Izráelſkich ; tobiem je dáł / y ſynom twoim / y córkom twoim z tobą / práwem wiecżnem ; káżdy cżyſty w domu twoim będźie je jádł.
Káżdą co nájprzedniejƺą oliwę / y káżde co nájlepƺe wino / y zboże / pierwiáſtki ich / które ofiárują Pánu / tobiem je dáł.
Pierwoćiny ze wƺyſtkich rzecży / które będą w źiemi ich / które przynioſą Pánu / twoje będą ; káżdy cżyſty w domu twoim jádáć je będźie.
Wƺyſtko / cokolwiek jeſt poślubione w Izráelu / twoje będźie.
Cokolwiek otwierá żywot wƺelkiego ćiáłá / á bywá ofiárowáne Pánu / ták z ludźi jáko z bydłá / twoje będźie ; ále pierworodne z ludźi okupiƺ / tákże pierworodne niecżyſtego bydłá okupiƺ.
A okup jego / gdy mu mieśiąc minie / dáƺ według ƺácunku twego pięć ſyklów ſrebrá według ſyklá świątnicy ; dwádźieśćiá pieniędzy wáży ſykiel.
Ale pierworodnego od krowy / álbo pierworodnego od owcy / álbo pierworodnego od kozy / nie dáƺ ná okup ; bo poświęcone ſą ; krew ich wylejeƺ ná ołtárz / á tłuſtość ich zápáliƺ ná ofiárę ogniſtą dla wdźięcżney wonnośći Pánu.
Ale mięſo ich twoje będźie ; jáko moſtek podnoƺeniá / y jáko łopátká práwá / twoje będą.
Wƺyſtkie ofiáry podnoƺeniá z rzecży poświęconych / które przynośić będą ſynowie Izráelſcy Pánu / dáłem tobie / y ſynom twym / y córkom twoim z tobą / práwem wiecżnem ; uſtáwá to nienáruƺoná / y wiecżná przed Pánem / tobie y náśieniu twemu z tobą.
Potem mówił Pán do Aároná : W źiemi ich dźiedźictwá mieć nie będźieƺ / y dźiáłu nie będźieƺ miał między nimi ; Jám dźiáł twój / y dźiedźictwo twoje w pośród ſynów Izráelſkich.
Synom záś Lewiego otom dáł wƺyſtkę dźieśięćinę w Izráelu dźiedźicżnie zá ſłużbę ich / którą wykonywáją / ſłużąc około námiotu zgromádzeniá.
A niecháj nieprzyſtępują więcey ſynowie Izráelſcy do námiotu zgromádzeniá / áby nie ponieśli grzechu y nie pomárli ;
Ale ſámi Lewitowie odpráwowáć będą uſługę około námiotu zgromádzeniá / y ſámi ponioſą niepráwość ſwoję. Uſtáwá to wiecżná będźie w národźiech wáƺych / áby w pośród ſynów Izráelſkich dźiedźictwá nie mieli.
Abowiem dźieśięćiny ſynów Izráelſkich / które przynośić będą ná ofiárę podnoƺeniá Pánu / dáłem Lewitom zá dźiedźictwo ; przetoż rzekłem o nich : W pośród ſynów Izráelſkich nie będą mieli dźiedźictwá.
Potem Pán rzekł do Mojżeƺá / mówiąc :
Mów też do Lewitów / á powiedz im : Gdy weźmiećie od ſynów Izráelſkich dźieśięćiny / którem Ja wám dáł od nich zá dźiedźictwo wáƺe / tedy ofiárowáć będźiećie ofiárę podnoƺeniá Pánu dźieśiątą cżęść dźieśięćin.
A pocżytáná wám będźie tá ofiárá wáƺá jáko zboże z bojewiſká / y jáko obfitość winá z práſy.
Ták wy też ofiárowáć będźiećie ofiárę podnoƺeniá Pánu ze wƺyſtkich dźieśięćin wáƺych / które weźmiećie od ſynów Izráelſkich / á oddáćie z nich ofiárę podnoƺeniá Pánu / Aáronowi kápłánowi :
Ze wƺyſtkich dochodów wáƺych ofiárowáć będźiećie wƺelką ofiárę podnoƺeniá Pánu ; ze wƺyſtkiego / co nájlepƺego jeſt / ofiárujećie cżąſtkę poświęconą.
Powieƺ im też : Gdy oddáwáć będźiećie z tego / co nájlepƺego jeſt / tedy pocżytáno będźie Lewitom jáko urodzáje z bojewiſká ; y jáko urodzáj z práſy.
Y będźiećie to jeść ná káżdem miejſcu / wy y cżeládź wáƺá ; ábowiem to jeſt zápłátá wáƺá zá ſłużbę wáƺę przy námioćie zgromádzeniá ;
Y nie ponieśiećie zá to grzechu / gdy ofiárowáć będźiećie co nájlepƺego jeſt z tego / y nie ſplugáwićie rzecży poświęconych od ſynów Izráelſkich / y nie pomrzećie.
