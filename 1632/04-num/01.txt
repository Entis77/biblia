Y mówił PAN do Mojzeƺá ná puƺczy Synáj w Namiećie zgromádzenia / pierwƺego <i>dniá</i> mieśiącá wtórego / roku wtórego po wyśćiu ich z źiemie Egipſkiey / tymi ſłowy :
Obliczćie ſummę wƺyſtkiego zgromádzenia Synów Izráelſkich / według Narodów ich / y według domów Ojców ich / według imion ich / káżdego mężczyznę według głów ich.
Ode dwudźieſtu lat / y wyżey / wƺyſtkich wychodzących ná wojnę z Izráelá : policzyćie je / według hufców ich / ty y Aáron.
Y będźie z wámi z káżdego pokolenia jeden mąż / któryby przednieyƺy był w domu Ojców ſwojich.
A teć ſą imioná mężów którzy z wámi będą : Z <i>pokolenia</i> Rubenowego / Eliſur / ſyn Sedeurów.
Z <i>pokolenia</i> Symeonowego / Semuliel / Syn Suriſáddájów.
Z <i>pokolenia</i> Judowego / Náſon / Syn Aminádábów.
Z <i>pokolenia</i> Iſáſchárowego / Nátánáel / Syn Suhárów.
Z <i>pokolenia</i> Zábulonowego / Eliáb / Syn Chelonów.
Z Synów Józefowych / z <i>pokolenia</i> Efráimowego / Eliſámáh / Syn Ammiudów : Z <i>pokolenia</i> Mánáſſeſſowego / Gámáliel / Syn Pedáſurów.
Z <i>pokolenia</i> Beniáminowego / Abidán / Syn Gedeonów.
Z <i>pokolenia</i> Dánowego / Achiezer / Syn Hammiſádájów.
Z <i>pokolenia</i> Aſerowego / Págiel / Syn Ochránów.
Z <i>pokolenia</i> Gádowego / Eliázáf / Syn Duelów.
Z <i>pokolenia</i> Neftálimowego / Achyrá / Syn Henánów.
Ci zwoływáni będą <i>nazacniejƺy</i> z ludu / Kśiążętá / w pokoleniách Ojców ſwych / wodzámi wojſk Izráelſkich będą.
Przyzwáli tedy do śiebie Mojzeƺ y Aáron mężów tych / którzy z imienia miánowáni ſą.
Y zebráli wƺyſtko zgromádzenie / dniá pierwƺego / mieśiącá wtórego / y przyznawáli śię do rodzájów ſwych / według fámilii ſwych / według domów Ojców ſwych / <i>y</i> według liczby imion / od dwudźieſtu lat y wyżey / według oſób ſwych :
Jáko rozkazał PAN Mojzeƺowi / ták je policzył ná puƺczy Synáj.
Y było Synów Rubená pierworodnego Izráelowego / rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / według oſób ich / wƺyſtkiego pogłowia męſkiego od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z Pokolenia Rubenowego czterdźieśći y ƺeść tyśięcy / y pięć ſet.
Z Synów Symeonowych rodzájów ich / według fámilii ich / według domów Ojców ich / náliczonych jego / według liczby imion / według oſób ich / wƺyſtkiego pogłowia męſkiego / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Symeonowego / pięćdźieśiąt y dźiewięć tyśięcy / y trzy ſtá.
Z Synów Gádowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Gádowego / czterdźieśći y pięć tyśięcy / y ƺeść ſet y pięćdźieśiąt.
Z Synów Judowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Judowego / śiedmdźieśiąt y cztery tyśiące / y ƺeść ſet.
Z pokolenia Iſáſchárowego rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Iſáſchárowego / pięćdźieśiąt y cztery tyśiące / y cztery ſtá.
Z Synów Zábulonowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Zábulonowego / pięćdźieśiąt y śiedm tyśięcy / y cztery ſtá.
Z Synów Józefowych / <i>á naprzód</i> z Synów Efráimowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Efráimowego / czterdźieśći tyśięcy / y pięć ſet.
Z Synów <i>záś</i> Mánáſſeſſowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Mánáſſeſſowego / trzydźieśći y dwá tyśiące / y dwie śćie.
Z Synów Benjáminowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Benjáminowego / trzydźieśći y pięć tyśięcy / y cztery ſtá.
Z Synów Dánowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Dánowego / ƺeśćdźieśiąt y dwá tyśiące / y śiedm ſet.
Z Synów Aſerowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Aſerowego / czterdźieśći tyśięcy y jeden / y pięć ſet.
Z Synów Neftálimowych rodzájów ich / według fámilii ich / według domów Ojców ich / według liczby imion / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę /
Náliczono ich z pokolenia Neftálimowego / pięćdźieśiąt y trzy tyśiące / y cztery ſtá.
Cić ſą policzeni które policzył Mojzeƺ y Aáron / y kśiążętá Izráelſkie / dwánaśćie mężów : którzy byli <i>wybráno</i> po jednemu z domów Ojców ſwych.
Y było wƺyſtkich policzonych Synów Izráelſkich / według domów Ojców ich / od dwudźieſtu lat y wyżey / wƺyſtkich wychodzących ná wojnę z Izráelá :
Było wƺyſtkich policzonych ƺeść kroć ſto tyśięcy / y trzy tyśiące / y pięć ſet y pięćdźieśiąt.
Ale Lewitowie / według pokolenia Ojców ſwych / nie byli policzeni miedzy nimi.
Bo rozkazał był PAN Mojzeƺowi / mówiąc :
Tylko pokolenia Lewi nie będźieƺ liczył / á ſummy ich nie położyƺ miedzy Syny Izráelſkie :
Ale poſtánowiƺ Lewity nád Przybytkiem świádectwá / y nád wƺyſtkim naczyniem jego / y nád wƺyſtkim co należy do niego : oni nośić będą przybytek / y wƺyſtkie naczynia jego / oni też ſłużyć będą w nim / á około przybytku Obozem śię kłáść będą.
A gdy śię będźie ruƺał przybytek zkładáć go będą Lewitowie : tákże gdy ſtánowić śię będźie Przybytek / ſtáwiáć go będą Lewitowie : á ktoby obcy do niego przyſtąpił / umrze.
Y będą ſtawáć Obozem Synowie Izráelſcy káżdy według pułków ſwojich / y káżdy pod Chorągwią ſwoją w wojſku ſwym.
Ale Lewitowie kłáść śię będą Obozem około Przybytku świádectwá / áby nie przyƺedł gniew <i>mój</i> ná zgromádzenie Synów Izráelſkich / y będą Lewitowie trzymáć ſtraż <i>u</i> Przybytku świádectwá.
Uczynili tedy Synowie Izráelſcy : według wƺyſtkiego co był rozkazał PAN Mojzeƺowi / ták uczynili.
