Y ſtáło śię w dźień / którego dokońcżył Mojżeƺ / á wyſtáwił przybytek / á pomázáł go / y poświęćił go ze wƺyſtkim ſprzętem jego / y ołtárz ze wƺyſtkiem nácżyniem jego / pomázáł je / y poświęćił je /
Że ofiárowáły kśiążętá Izráelſkie / przedniejƺe z domów ojców ſwych / ( co byli hetmány z káżdego pokoleniá / y przełożonymi nád policżonymi. )
A przynieśli ofiáry ſwe przed Páná : ƺeść wozów przykrytych / y dwánáśćie wołów / jeden wóz od dwojgá kśiążąt / á od káżdego wół jeden / y poſtáwili to przed przybytkiem.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Weźmij od nich / áby to było ná potrzebę przy ſłużbie w námioćie zgromádzeniá / y oddáj to Lewitom / káżdemu według potrzeby urzędu jego.
Wźiął tedy Mojżeƺ one wozy y woły y oddáł je Lewitom.
Dwá wozy / y cżtery woły dáł ſynom Gerſonowym według potrzeby urzędów ich.
Cżtery záś wozy y ośm wołów dáł ſynom Merárego według potrzeby urzędów ich / pod włádzę Itámárá / ſyná Aároná kápłáná.
Ale ſynom Káátowym nic nie dáł : bo uſługá świątnicy byłá przy nich / ná rámieniu ją nośić muśieli.
Ofiárowáły tedy kśiążętá ku poświęceniu ołtárzá onegoż dniá / gdy był pomázány ; y ofiárowáły kśiążętá dáry ſwe przed ołtárzem.
Y rzekł Pán do Mojżeƺá : Jeden kśiążę jednego dniá / drugi kśiążę drugiego dniá oddáwáć będźie dáry ſwoje ku poświęceniu ołtárzá.
Y ofiárowáł pierwƺego dniá dár ſwój Nááſon / ſyn Aminádábów z pokoleniá Judy.
A dár jego był : miſá ſrebrná jedná / ſto y trzydźieśći ſyklów wági jey / cżáƺá ſrebrná jedná / śiedmdźieśiąt ſyklów wági jey według ſyklá świątnicy / obie pełne pƺenney mąki / zágniećioney z oliwą / ná ofiárę śniedną ;
Kádźielnicá jedná z dźieśięćiu ſyklów złotá / pełná kádźidłá dla kádzeniá ;

Toć było poświęcenie ołtárzá / onegoż dniá / gdy pomázán jeſt od kśiążąt Izráelſkich : Miſ ſrebrnych dwánáśćie / cżáƺ ſrebrnych dwánáśćie / kádźielnic złotych dwánáśćie ;
Sto y trzydźieśći ſyklów jedná miſá ſrebrná wáżyłá / śiedmdźieśiąt ſyklów cżáƺá jedná ; wƺyſtkiego ſrebrá w onem nácżyniu było dwá tyśiące y cżtery ſtá ſyklów według ſyklá świątnicy ;
Kádźielnic złotych dwánáśćie / pełnych kádźidłá ; dźieśięć ſyklów wáżyłá káżdá według ſyklá świątnicy ; wƺyſtkiego złotá w onych kádźielnicách było ſto y dwádźieśćiá ſyklów.
A wƺyſtkiego bydłá ku ofierze páloney dwánáśćie ćielców / báránów dwánáśćie / báránków rocżnych dwánáśćie / z ofiárą ich śniedną / y kozłów z kóz zá grzech dwánáśćie.
Wƺyſtkiego záśię bydłá ná ofiárę ſpokojną było wołów dwádźieśćiá y cżtery / báránów ƺeśćdźieśiąt / kozłów ƺeśćdźieśiąt ; báránków rocżnych ƺeśćdźieśiąt. Toć było poświęcenie ołtárzá po pomázániu jego.
A gdy Mojżeƺ wchodźił do námiotu zgromádzeniá / by śię rozmáwiáł z Bogiem / tedy ſłyƺáł głoſ mówiącego do śiebie z ubłágálni / która byłá nád ſkrzynią świádectwá / między dwiemá Cheruby / á ſtámtąd máwiáł do niego.
