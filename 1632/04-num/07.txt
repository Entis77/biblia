Y ſtáło śię w dźień / którego dokońcżył Mojzeƺ / á wyſtáwił przybytek / á pomázał go / y poświęćił go ze wƺyſtkim ſprzętem jego / y ołtárz ze wƺyſtkim nácżyniem jego / pomázał je / y poświęćił je /
Że ofiárowáły kśiążętá Izráelſkie / przedniejƺe z domów ojców ſwych / ( co byli hetmány z káżdego pokolenia / y przełożonymi nád policżonymi. )
A przynieśli ofiáry ſwe przed Páná : ƺeść wozów przykrytych / y dwánaśćie wołów / jeden wóz od dwojgá kśiążąt / á od káżdego wół jeden / y poſtáwili to przed przybytkiem.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Weźmij od nich / áby to było ná potrzebę przy ſłużbie w námioćie zgromádzenia / y oddáj to Lewitom / káżdemu według potrzeby urzędu jego.
Wźiął tedy Mojzeƺ one wozy y woły y oddał je Lewitom.
Dwá wozy / y cżtery woły dał ſynom Gerſonowym według potrzeby urzędów ich.
Cżtery záś wozy y ośm wołów dał ſynom Merárego według potrzeby urzędów ich / pod włádzę Itámárá / ſyná Aároná kápłáná.
Ale ſynom Káátowym nic nie dał : bo uſługá świątnicy byłá przy nich / ná rámieniu ją nośić muśieli.
Ofiárowáły tedy kśiążętá ku poświęceniu ołtárzá onegoż dniá / gdy był pomázány ; y ofiárowáły kśiążętá dáry ſwe przed ołtárzem.
Y rzekł Pán do Mojżeƺá : Jeden kśiążę jednego dniá / drugi kśiążę drugiego dniá oddáwáć będźie dáry ſwoje ku poświęceniu ołtárzá.
Y ofiárował pierwƺego dniá dar ſwój Nááſon / ſyn Aminádábów z pokolenia Judy.

Sto y trzydźieśći ſyklów jedná miſá ſrebrná wáżyłá / śiedmdźieśiąt ſyklów cżáƺá jedná ; wƺyſtkiego ſrebrá w onem nácżyniu było dwá tyśiące y cżtery ſtá ſyklów według ſyklá świątnicy ;

A wƺyſtkiego bydłá ku ofierze páloney dwánaśćie ćielców / báránów dwánaśćie / báránków rocżnych dwánaśćie / z ofiárą ich śniedną / y kozłów z kóz zá grzech dwánaśćie.
Wƺyſtkiego záśię bydłá ná ofiárę ſpokojną było wołów dwádźieśćiá y cżtery / báránów ƺeśćdźieśiąt / kozłów ƺeśćdźieśiąt ; báránków rocżnych ƺeśćdźieśiąt. Toć było poświęcenie ołtárzá po pomázániu jego.
