Mieśiącá záś śiódmego w pierwƺy dzień jego / zgromádzenie święte mieć będziećie ; żádnej roboty ſłużebnicżej nie będziećie cżynić ; dzień jeſt weſołego trąbieniá wáƺego.
A będziećie ofiárowáli cáłopálenie ku wdzięcżnej wonnośći Pánu / ćielcá młodego jednego / báráná jednego / báránków rocżnych śiedm zupełnych ;
A ná ofiárę śniedną ich z mąki pƺennej nágniećionej z oliwą trzy dzieśiąte cżęśći efy do ćielcá / á dwie dzieśiąte cżęśći do báráná.
A dzieśiątą cżęść jednę do káżdego báránká z onych śiedmiu báránków ;
Tákże kozłá jednego z kóz ku ofierze zá grzech ná ocżyƺcżenie wáſ.
Oprócż cáłopáleniá nowego mieśiącá / y ofiáry śniednej jego / y oprócż cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y ofiár ich mokrych według obrzędów ich ku wdzięcżnej wonnośći ; ofiárá to ogniſtá Pánu.
Potem dzieśiątego dniá tegoż mieśiącá śiódmego / zgromádzenie święte mieć będziećie / á będziećie trápić duƺe wáƺe ; żádnej roboty nie będziećie robić.
A będziećie ofiárowáli cáłopálenie Pánu ku wdzięcżnej wonnośći : ćielcá młodego jednego / báráná jednego / báránków rocżnych śiedm ; zupełni niech wám będą ;
A ná ofiárę śniedną ich z pƺennej mąki nágniećionej z oliwą : trzy dzieśiąte cżęśći do káżdego ćielcá / dwie záś dzieśiąte cżęśći do káżdego báráná ;
A dzieśiątą cżęść jednę do káżdego báránká z onych śiedmiu báránków ;
Kozłá z kóz jednego ná ofiárę zá grzech / oprócż ofiáry zá grzech ná ocżyƺcżenie / y oprócż cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y mokrych ofiár ich.
W piętnáſty záś dzień tegoż śiódmego mieśiącá zgromádzenie święte mieć będziećie ; żádnej roboty ſłużebnicżej nie będziećie cżynić weń ; ále obchodzić będziećie święto urocżyſte Pánu przez śiedm dni.
Y ofiárowáć będziećie cáłopálenie ná ofiárę ogniſtą ku wdzięcżnej wonnośći Pánu / ćielców młodych trzynáśćie / báránów dwá / báránków rocżnych cżternáśćie ; y zupełni będą.
A ná ofiárę ich śniedną z pƺennej mąki nágniećionej z oliwą trzy dzieśiąte cżęśći efy do káżdego ćielcá z onych trzynáśćie ćielców / dwie dzieśiąte cżęśći do káżdego báráná z onych dwóch báránów ;
A jedná dzieśiątá cżęść do káżdego báránká z onych cżternáśćie báránków.
Tákże kozłá jednego z kóz ná ofiárę zá grzech / oprócż cáłopáleniá uſtáwicżnego / ofiáry śniednej jego / y ofiáry mokrej jego.
Wtórego záś dniá ofiárowáć będziećie ćielców młodych dwánáśćie / báránów dwá / báránków rocżnych cżternáśćie zupełnych ;
Y ofiárę śniedną ich / y ofiáry ich mokre do káżdego ćielcá / do káżdego báráná / y do káżdego báránká według licżby ich / y według zwycżáju ich.
Nádto kozłá jednego z kóz ná ofiárę zá grzech / oprócż cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y ofiár ich mokrych.
Dniá záś trzećiego ofiárowáć będziećie jedenáśćie ćielców / báránów dwá / báránków rocżnych cżternáśćie zupełnych.
Y ofiárę śniedną ich / y ofiáry mokre ich do káżdego ćielcá / do káżdego báráná / y do káżdego báránká według licżby ich / y według zwycżáju ich ;
Do tego / kozłá jednego ná ofiárę zá grzech / okrom cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y mokrej ofiáry jego.
A dniá cżwártego ofiárowáć będziećie ćielców dzieśięć / báránów dwá / báránków rocżnych cżternáśćie zupełnych ;
Ofiárę śniedną ich / y ofiáry mokre ich do káżdego ćielcá / do káżdego báráná / y do káżdego báránká według licżby ich / y według zwycżáju ich ;
Kozłá też jednego z kóz ná ofiárę zá grzech / oprócż cáłopáleniá uſtáwicżnego / ofiáry śniednej jego / y mokrej ofiáry jego.
A dniá piątego ofiárowáć będziećie ćielców dziewięć / báránów dwá / báránków rocżnych cżternáśćie zupełnych ;
Y ofiárę śniedną ich / y ofiáry mokre ich do káżdego ćielcá / do káżdego báráná / y do káżdego báránká według licżby ich / y według zwycżáju ich ;
Tákże kozłá jednego ná ofiárę zá grzech / oprócż cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y ofiáry mokrej jego.
A dniá ƺóſtego ofiárowáć będziećie ćielców ośm / báránów dwá / báránków rocżnych cżternáśćie zupełnych ;
Y ofiárę śniedną ich / y ofiáry mokre ich do káżdego ćielcá / y do káżdego báráná / y do káżdego báránká według licżby ich / według zwycżáju ich.
Nádto kozłá ná ofiárę zá grzech jednego / okrom cáłopáleniá uſtáwicżnego / ofiáry śniednej jego / y ofiár mokrych jego ;
Tákże dniá śiódmego ofiárowáć będziećie ćielców śiedm / báránów dwá / báránków rocżnych cżternáśćie zupełnych.
Y ofiárę śniedną ich / y ofiáry mokre ich do káżdego ćielcá / do káżdego báráná / do káżdego báránká według licżby ich / y według zwycżáju ich ;
Przytem kozłá ná ofiárę zá grzech jednego / oprócż cáłopáleniá uſtáwicżnego / y ofiáry śniednej jego / y ofiáry mokrej jego.
A dniá óſmego zácne święto mieć będziećie ; żádnej roboty ſłużebnicżej nie będziećie cżynić.
A ofiárowáć będziećie cáłopálenie / y ofiárę ogniſtą ku wdzięcżnej wonnośći Pánu / ćielcá jednego / báráná jednego / báránków rocżnych śiedm zupełnych ;
Ofiárę śniedną ich / y ofiáry mokre ich do ćielcá / do báráná / do káżdego báránká według licżby ich / y według zwycżáju ich.
Nádto kozłá ná ofiárę zá grzech jednego / okrom cáłopáleniá uſtáwicżnego / ofiáry śniednej jego / y ofiáry mokrej jego.
To ofiárowáć będziećie Pánu w świętá urocżyſte wáƺe / oprócż ślubów wáƺych y dobrowolnych ofiár wáƺych w cáłopáleniách wáƺych / y w śniednych ofiárách wáƺych / y w mokrych ofiárách wáƺych / y w ſpokojnych ofiárách wáƺych.
