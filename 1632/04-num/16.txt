Tedy śię zbuntowáł Kore / ſyn Izáárá / ſyná Káátowego / ſyná Lewiego / tákże Dátán y Abiron / ſynowie Elijábowi / y Hon / ſyn Fáletów z ſynów Rubenowych.
Y powſtáli przećiw Mojżeƺowi / á z nimi mężów z ſynów Izráelſkich dwieśćie y pięćdźieśiąt / kśiążętá między ludem / których do rády przyzywáno / ludźie zácni.
Ci zebráwƺy śię przećiw Mojżeƺowi / y przećiw Aáronowi / rzekli im : Wiele to ná wáſ / poniewáż wƺyſtek ten lud / wƺyſcy ſą święći / á w pośrodku nich jeſt Pán ; przecżże śię wy wynośićie nád zgromádzeniem Páńſkim?
Co gdy uſłyƺáł Mojżeƺ / upádł ná oblicże ſwoje /
Y rzekł do Korego y do wƺyſtkiej roty jego / mówiąc : Ráno pokáże Pán / kto jeſt jego / y kto jeſt święty / y kto má przyſtępowáć przedeń ; bo kogo ſobie wybráł / temu rozkáże przyſtąpić do śiebie.
To tedy ucżynićie : Weźmiećie ſobie kádźielnice / ty Kore / y wƺyſtká rotá twojá.
Y nákłádłƺy w nie ogniá / włóżćie náń kádźidłá przed Pánem jutro ; y ſtánie śię / że kogokolwiek obierze Pán / ten będźie świętym : wiele to ná wáſ ſynowie Lewiego.
Nád to rzekł Mojżeƺ do Korego : Słuchájćie proƺę ſynowie Lewiego ;
Izáli wám to máło / że wáſ oddźielił Bóg Izráelſki od zgromádzeniá Izráelſkiego / ábyśćie przyſtępowáli do niego ku odpráwowániu uſługi w przybytku Páńſkim / á iżbyśćie ſtáli przed zgromádzeniem / y ſłużyli mu?
Y przyjął ćiebie / y wƺyſtkę bráćią twoję / ſyny Lewiego z tobą / że jeƺcże ƺukáćie kápłáńſtwá?
Dlá tegoż / ty y wƺyſtká rotá twojá / zbuntowáliśćie śię przećiw Pánu ; bo Aáron cóż jeſt / żeśćie ƺemráli przećiw niemu?
Tedy poſłáł Mojżeƺ / áby záwołáno Dátáná / y Abironá / ſynów Elijábowych / którzy odpowiedźieli : Nie pójdźiemy.
Izáli máło ná tem / żeś náſ wywiódł z źiemi opływájącej mlekiem y miodem / ábyś náſ pomorzył ná tej puƺcży / że jeƺcże chceƺ mieć nád námi zwierzchność y nám rozkázowáć?
Ku temu do źiemi opływájącej mlekiem y miodem nie wprowádźiłeś náſ / ániś nám dáł w dźiedźictwo pól y winnic : izáli ocży tym mężom wyłupić chceƺ? Nie pójdźiemy.
Tedy śię rozgniewáł Mojżeƺ bárdzo / y rzekł do Páná : Nie pátrz ná ofiárę ich ; ni jednego oſłá nie wźiąłem od nich / ánim co złego komu z nich ucżynił.
Potem rzekł Mojżeƺ do Korego : Ty / y wƺyſtká rotá twojá / ſtáwćie śię przed Páná jutro / ty / y oni / y Aáron :
A wźiąwƺy káżdy kádźielnicę ſwoję / włóżćie w nię kádźidłá / y ſtáwćie śię przed Páná / káżdy z kádźielnicą ſwoją / dwieśćie y pięćdźieśiąt kádźielnic / y ty / y Aáron / káżdy z kádźielnicą ſwoją.
Wźiął tedy káżdy kádźielnicę ſwoję / á włożywƺy w nią ogniá nákłádli w nią kádźidłá ; y ſtánęli u drzwi námiotu zgromádzeniá Mojżeƺ y Aáron.
Ale Kore już był zebráł przećiwko nim wƺyſtkę rotę do drzwi námiotu zgromádzeniá ; tedy chwáłá Páńſká ukázáłá śię wƺyſtkiemu ludowi.
Y rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Odłącżćie śię z pośrodku zebrániá tego / ábym je nágle zátráćił.
Lecż oni upádli ná oblicżá ſwe y rzekli : O Boże / Boże Duchów / y wƺelkiego ćiáłá! cżłowiek jeden zgrzeƺył / á ná wƺyſtekże lud gniewáć śię będźieƺ?
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Rzecż do zgromádzeniá / á mów : Odſtąpćie od námiotu Korego / Dátáná y Abironá.
A wſtáwƺy Mojżeƺ / ƺedł do Dátáná y Abironá ; y ƺli zá nim ſtárśi Izráelſcy.
Y rzekł do zgromádzeniá / mówiąc : Odſtąpćie / proƺę / od námiotów mężów tych niepobożnych / áni śię dotykájćie wƺyſtkiego / co ich jeſt / byśćie ſnáć nie zginęli we wƺyſtkich grzechách ich.
Y odſtąpili od námiotu Korego / Dátáná y Abironá zewƺąd. Ale Dátán y Abiron wyƺedłƺy ſtáli u drzwi námiotów ſwoich / y żony ich / y ſynowie ich / y málucżcy ich.
Tedy rzekł Mojżeƺ : Po tem poznáćie / że mię Pán poſłáł / ábym cżynił te wƺyſtkie ſpráwy / á że nic z domyſłu ſercá ſwego nie cżynię ;
Jeźliże ták / jáko inni ludźie umieráją / pomrą ći / á zwykłem innych ludźi kárániem / káráni będą / nie poſłáł mię Pán ;
Ale jeźliż co nowego ucżyni Pán / że otworzy źiemiá uſtá ſwe / y pożre je y wƺyſtkiego co máją / y zſtąpią żywo do piekłá / tedy poznáćie / że rozdráźnili ći mężowie Páná.
Y ſtáło śię / gdy przeſtáł mówić wƺyſtkich tych ſłów / że śię rozſtąpiłá źiemiá pod nimi ;
A otworzywƺy źiemiá páƺcżękę ſwoję / pożárłá je / y domy ich / ze wƺyſtkimi ludźmi / którzy byli przy Korem / y wƺyſtkie májętnośći ich.
Y zſtąpili oni ze wƺyſtkiem co mieli / żywo do piekłá / y okryłá je źiemiá / y poginęli z pośrodku zgromádzeniá.
Wƺyſcy záś Izráelitowie / którzy byli około nich / ućiekáli ná krzyk ich / bo mówili : By ſnáć nie pożárłá y náſ źiemiá.
Wyƺedł tákże ogień od Páná / y ſpálił onych dwieśćie y pięćdźieśiąt mężów / którzy ofiárowáli kádzenie.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Rzecż do Eleázárá / ſyná Aáronowego / kápłáná / áby pozbieráł kádźielnice z onego pogorzeliſká / á ogień y tám y ſám niech rozrzući ; bo ſą poświęcone.
A z kádźielnic tych / którzy zgrzeƺyli przećiwko duƺom ſwym / rozbiwƺy je ná bláchy / niech obije ołtárz / bo iż w nich ofiárowáli przed Pánem / poświęcone ſą / przetoż będą ná znák ſynom Izráelſkim.
Tedy pozbieráł Eleázár kápłán one miedźiáne kádźielnice / w których ofiárowáli oni popáleni ; y rozbito je ná bláchy / ná obićie ołtárzá.
Ná pámiątkę ſynom Izráelſkim / áby nie przyſtępowáł żáden obcy / któryby nie był z náśieniá Aáronowego / do odpráwowániá kádzeniá przed Pánem / áby mu śię nie ſtáło jáko Koremu / y jáko roćie jego / jáko mu był powiedźiáł Pán przez Mojżeƺá.
Y ƺemráło wƺyſtko zgromádzenie ſynów Izráelſkich názájutrz przećiwko Mojżeƺowi / y przećiwko Aáronowi / mówiąc : Wyśćie przycżyną śmierći ludu Páńſkiego /
Y ſtáło śię / gdy śię zbieráł lud przećiw Mojżeƺowi / y przećiw Aáronowi / że ſpojrzeli ku námiotowi zgromádzeniá / á oto / okrył go obłok / y okázáłá śię chwáłá Páńſká.
Y przyƺedł Mojżeƺ z Aáronem przed námiot zgromádzeniá.
Y rzekł Pán do Mojżeƺá / mówiąc :
Wynijdźćie z pośrodku zgromádzeniá tego / á wytrácę je w okámgnieniu ; y upádli ná oblicżá ſwoje.
Potem rzekł Mojżeƺ do Aároná : Weźmij kádźielnicę / á włóż w nią ogniá z ołtárzá / włóż też kádźidłá / á nátychmiáſt idź do zgromádzeniá / y ocżyść je ; boć już wyƺłá popędliwość od twárzy Páńſkiej / á już śię zácżęło káránie.
Wźiął tedy Aáron kádźielnicę / jáko mu rozkázáł Mojżeƺ / y przybieżáł w pośrodek zgromádzeniá / á oto już śię byłá zácżęłá plágá w ludźie ; y ucżyniwƺy kádzenie ocżyśćił lud.
Y ſtánął Aáron między umárłymi y między żywymi / á záhámowáná jeſt plágá.
A było umárłych od onej plági cżternáśćie tyśięcy y śiedm ſet / oprócż onych / którzy pomárli z przycżyny Korego.
Zátem śię wróćił Aáron do Mojżeƺá ku drzwiom námiotu zgromádzeniá / gdy plágá záhámowáná byłá.
