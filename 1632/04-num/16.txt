Tedy śię zbuntował Kore / ſyn Izáárá / ſyná Káátowego / ſyná Lewiego / tákże Dátán y Abiron / ſynowie Elijábowi / y Hon / ſyn Fáletów z ſynów Rubenowych.
Y powſtáli przećiw Mojżeƺowi / á z nimi mężów z ſynów Izráelſkich dwieśćie y pięćdźieśiąt / kśiążętá miedzy ludem / których do rády przyzywano / ludźie zácni.
Ci zebráwƺy śię przećiw Mojżeƺowi / y przećiw Aáronowi / rzekli im : Wiele to ná was / ponieważ wƺyſtek ten lud / wƺyſcy ſą święći / á w pośrzodku nich jeſt Pán ; przecżże śię wy wynośićie nád zgromádzeniem Páńſkim?
Co gdy uſłyƺał Mojzeƺ / upádł ná oblicże ſwoje /
Y rzekł do Korego y do wƺyſtkiej roty jego / mówiąc : Ráno pokáże Pán / kto jeſt jego / y kto jeſt święty / y kto ma przyſtępowáć przedeń ; bo kogo ſobie wybrał / temu rozkáże przyſtąpić do śiebie.
To tedy ucżynićie : Weźmiećie ſobie kádźielnice / ty Kore / y wƺyſtká rotá twojá.
Y nákłádƺy w nie ogniá / włóżćie nań kádźidłá przed Pánem jutro ; y ſtánie śię / że kogokolwiek obierze Pán / ten będźie świętym : wiele to ná was ſynowie Lewiego.
Nád to rzekł Mojzeƺ do Korego : Słuchájćie proƺę ſynowie Lewiego ;
Izali wam to máło / że was oddźielił Bóg Izráelſki od zgromádzenia Izráelſkiego / ábyśćie przyſtępowáli do niego ku odpráwowániu uſługi w przybytku Páńſkim / á iżbyśćie ſtáli przed zgromádzeniem / y ſłużyli mu?
Y przyjął ćiebie / y wƺyſtkę bráćią twoję / ſyny Lewiego z tobą / że jeƺcże ƺukáćie kápłáńſtwá?
Dla tegoż / ty y wƺyſtká rotá twojá / zbuntowáliśćie śię przećiw Pánu ; bo Aáron cóż jeſt / żeśćie ƺemráli przećiw niemu?
Tedy poſłał Mojzeƺ / áby záwołáno Dátáná / y Abironá / ſynów Elijábowych / którzy odpowiedźieli : Nie pójdźiemy.
Izali máło ná tym / żeś nas wywiódł z źiemi opływájącey mlekiem y miodem / ábyś nas pomorzył ná tey puƺcży / że jeƺcże chceƺ mieć nád námi zwierzchność y nam rozkázowáć?
Ku temu do źiemi opływájącey mlekiem y miodem nie wprowádźiłeś nas / ániś nam dał w dźiedźictwo pól y winnic : izali ocży tym mężom wyłupić chceƺ? Nie pójdźiemy.
Tedy śię rozgniewał Mojzeƺ bárzo / y rzekł do Páná : Nie pátrz ná ofiárę ich ; ni jednego oſłá nie wźiąłem od nich / ánim co złego komu z nich ucżynił.
Potem rzekł Mojzeƺ do Korego : Ty / y wƺyſtká rotá twojá / ſtáwćie śię przed Páná jutro / ty / y oni / y Aáron :
A wźiąwƺy káżdy kádźielnicę ſwoję / włóżćie w nię kádźidłá / y ſtáwćie śię przed Páná / káżdy z kádźielnicą ſwoją / dwieśćie y pięćdźieśiąt kádźielnic / y ty / y Aáron / káżdy z kádźielnicą ſwoją.
Wźiął tedy káżdy kádźielnicę ſwoję / á włożywƺy w nią ogniá nákłádli w nią kádźidłá ; y ſtánęli u drzwi namiotu zgromádzenia Mojzeƺ y Aáron.
Ale Kore już był zebrał przećiwko nim wƺyſtkę rotę do drzwi namiotu zgromádzenia ; tedy chwałá Páńſka ukázáłá śię wƺyſtkiemu ludowi.
Y rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Odłącżćie śię z pośrzodku zebránia tego / ábym je nágle zátráćił.
Lecż oni upádli ná oblicża ſwe y rzekli : O Boże / Boże Duchów / y wƺelkiego ćiáłá! cżłowiek jeden zgrzeƺył / á ná wƺyſtekże lud gniewáć śię będźieƺ?
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Rzecż do zgromádzenia / á mów : Odſtąpćie od namiotu Korego / Dátáná y Abironá.
A wſtáwƺy Mojzeƺ / ƺedł do Dátáná y Abironá ; y ƺli zá nim ſtárśi Izráelſcy.
Y rzekł do zgromádzenia / mówiąc : Odſtąpćie / proƺę / od námiotów mężów tych niepobożnych / áni śię dotykájćie wƺyſtkiego / co ich jeſt / byśćie ſnadź nie zginęli we wƺyſtkich grzechách ich.
Y odſtąpili od namiotu Korego / Dátáná y Abironá zewƺąd. Ale Dátán y Abiron wyƺedƺy ſtáli u drzwi námiotów ſwojich / y żony ich / y ſynowie ich / y málucżcy ich.
Tedy rzekł Mojzeƺ : Po tym poznáćie / że mię Pán poſłał / ábym cżynił te wƺyſtkie ſpráwy / á że nic z domyſłu ſercá ſwego nie cżynię ;
Jeſliże ták / jáko inni ludźie umieráją / pomrą ći / á zwykłem innych ludźi kárániem / káráni będą / nie poſłał mię Pán ;
Ale jeſliż co nowego ucżyni Pán / że otworzy źiemiá uſtá ſwe / y pożre je y wƺyſtkiego co máją / y zſtąpią żywo do piekłá / tedy poznáćie / że rozdráźnili ći mężowie Páná.
Y ſtáło śię / gdy przeſtał mówić wƺyſtkich tych ſłów / że śię rozſtąpiłá źiemiá pod nimi ;
A otworzywƺy źiemiá páƺcżękę ſwoję / pożárłá je / y domy ich / ze wƺyſtkimi ludźmi / którzy byli przy Korem / y wƺyſtkie májętnośći ich.
Y zſtąpili oni ze wƺyſtkim co mieli / żywo do piekłá / y okryłá je źiemiá / y poginęli z pośrzodku zgromádzenia.
Wƺyſcy záś Izráelitowie / którzy byli około nich / ućiekáli ná krzyk ich / bo mówili : By ſnadź nie pożárłá y nas źiemiá.
Wyƺedł tákże ogień od Páná / y ſpálił onych dwieśćie y pięćdźieśiąt mężów / którzy ofiárowáli kádzenie.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Rzecż do Eleázárá / ſyná Aáronowego / kápłáná / áby pozbierał kádźielnice z onego pogorzeliſká / á ogień y tám y ſám niech rozrzući ; bo ſą poświęcone.
A z kádźielnic tych / którzy zgrzeƺyli przećiwko duƺom ſwym / rozbiwƺy je ná bláchy / niech obije ołtárz / bo iż w nich ofiárowáli przed Pánem / poświęcone ſą / przetoż będą ná znák ſynom Izráelſkim.
Tedy pozbierał Eleázár kápłán one miedźiáne kádźielnice / w których ofiárowáli oni popáleni ; y rozbito je ná bláchy / ná obićie ołtárzá.
Ná pámiątkę ſynom Izráelſkim / áby nie przyſtępował żáden obcy / któryby nie był z náśieniá Aáronowego / do odpráwowánia kádzeniá przed Pánem / áby mu śię nie ſtáło jáko Koremu / y jáko roćie jego / jáko mu był powiedźiał Pán przez Mojżeƺá.
Y ƺemráło wƺyſtko zgromádzenie ſynów Izráelſkich názájutrz przećiwko Mojżeƺowi / y przećiwko Aáronowi / mówiąc : Wyśćie przycżyną śmierći ludu Páńſkiego /
Y ſtáło śię / gdy śię zbierał lud przećiw Mojżeƺowi / y przećiw Aáronowi / że ſpojrzeli ku námiotowi zgromádzenia / á oto / okrył go obłok / y okázáłá śię chwałá Páńſka.
Y przyƺedł Mojzeƺ z Aáronem przed námiot zgromádzenia.
Y rzekł Pán do Mojżeƺá / mówiąc :
Wynijdźćie z pośrzodku zgromádzenia tego / á wytrácę je w okámgnieniu ; y upádli ná oblicża ſwoje.
Potem rzekł Mojzeƺ do Aároná : Weźmij kádźielnicę / á włóż w nią ogniá z ołtárzá / włóż też kádźidłá / á nátychmiáſt idź do zgromádzenia / y ocżyść je ; boć już wyƺłá popędliwość od twárzy Páńſkiej / á już śię zácżęło káránie.
Wźiął tedy Aáron kádźielnicę / jáko mu rozkázał Mojzeƺ / y przybieżał w pośrodek zgromádzenia / á oto już śię byłá zácżęłá plágá w ludźie ; y ucżyniwƺy kádzenie ocżyśćił lud.
Y ſtánął Aáron miedzy umárłymi y miedzy żywymi / á záhámowáná jeſt plágá.
A było umárłych od oney plági cżternaśćie tyśięcy y śiedm ſet / oprócż onych / którzy pomárli z przycżyny Korego.
Zátem śię wróćił Aáron do Mojżeƺá ku drzwiom namiotu zgromádzenia / gdy plágá záhámowáná byłá.
