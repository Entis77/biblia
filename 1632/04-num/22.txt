Z támtąd ruƺyli śię ſynowie Izráelſcy / y położyli śię obozem ná polách Moábſkich / z tey ſtrony Jordánu przećiw Jerychowi.
A widząc Bálák / ſyn Seforów / wƺyſtko / co ucżynił Izráel Amorejcżykowi /
Uląkł śię Moáb tego ludu wielce / bo go było wiele ; y zátrwożył ſobą Moáb dla ſynów Izráelſkich.
Przetoż rzekł Moáb do ſtárƺych Mádyjáńſkich : Teraz pożre to mnóſtwo wƺyſtko / co jeſt około nas / jáko wół pożerá tráwę polną. A Bálák / ſyn Seforów / był królem Moábſkim ná on cżás.
Y poſłał poſły do Báláámá / ſyná Beorowego / do Pethoru miáſtá / które jeſt nád rzeką źiemi ſynów ludu jego / áby go wezwano / mówiąc : Oto lud wyƺedł z Egiptu / oto okrył wierzch źiemi / y oſádzá śię przećiwko mnie.
Przetoż teraz przyjdź proƺę / á przeklináj mnie kwoli lud ten / bo możniejƺy jeſt nád mię ; owá ſnadź go będę mógł poráźić / y wygnáć go z źiemi ; bo ja wiem / że komu błogoſłáwiƺ / błogoſłáwiony będźie ; á kogo przeklináƺ / przeklęty będźie.
Poƺli tedy ſtárśi Moábſcy / y ſtárśi Mádyjáńſcy / májąc zápłátę zá wieƺcżbę w rękách ſwych.
A przyƺedƺy do Báláámá / powiedźieli mu ſłowá Bálákowe. Y rzekł do nich : Zoſtáńćie tu przez noc / á dam wam odpowiedź / jáko mi oznájmi Pán.
Y zoſtáły kśiążętá Moábſkie z Báláámem. Tedy przyƺedł Bóg do Báláámá / y rzekł : Cóż to zá mężowie u ćiebie?
Y odpowiedźiał Báláám Bogu : Bálák / ſyn Seforów / król Moábſki / poſłał do mnie / mówiąc :
Oto lud / który wyƺedł z Egiptu / y okrył wierzch źiemi ; terazże pójdź / przeklináj mi go ; ſnadź będę mógł wálcżyć z nim / y wypędzę go.
Tedy rzekł Bóg do Báláámá : Nie chodź z nimi / áni przeklináj ludu tego ; bo jeſt błogoſłáwiony.
A wſtáwƺy ráno Báláám rzekł do kśiążąt Bálákowych : Wróććie śię do źiemi wáƺej ; bo mi nie pozwálá Pán puśćić śię w drogę z wámi.
Wſtáwƺy tedy kśiążętá Moábſkie / wróćili śię do Báláká / y powiedźieli : Nie chćiał Báláám iść z námi.
Tedy powtóre poſłał Bálák więcey kśiążąt / y zácniejƺych nád pierwƺe.
Którzy przyƺedƺy do Báláámá / mówili mu : Ták mówi Bálák / ſyn Seforów : Nie oćiągáj śię proƺę przyść do mnie ;
Abowiem ći wielką ucżćiwość wyrządzę / y wƺyſtko / cobyś mi rzekł / ucżynię ; tylko przyjdź proƺę á przeklináj mi ten lud.
Tedy odpowiedźiał Báláám / y rzekł do ſług Bálákowych : Choćby mi dał Bálák pełen dom ſwój ſrebrá y złotá / nie mógłbym przeſtąpić ſłowá Páná / Bogá mego / y ucżynić przećiwko niemu / bądź máło bądź wiele ;
Ale proƺę zoſtáńćie tu y wy przez noc / á dowiem śię / co jeƺcże Pán będźie mówił ze mną.
Tedy przyƺedł Bóg do Báláámá w nocy / y rzekł do niego : Jeſliż / áby ćię wezwáli / przyƺli mężowie ći / wſtáńże / idź z nimi ; á wƺákże / coć rozkáżę / to ucżyniƺ.
Tedy wſtáwƺy Báláám ráno / ośiodłał oślicę ſwoję / y jechał z kśiążęty Moábſkimi.
Y rozpálił śię gniew Boży / że on jechał ; y ſtánął Anjoł Páńſki ná drodze / áby mu záſtąpił ; á on jechał ná oślicy ſwojey / y dwoje pácholąt jego z nim.
A gdy ujrzáłá oślicá Aniołá Páńſkiego / ſtojącego ná drodze / á miecż jego dobyty w ręce jego / tedy uſtąpiłá oślicá z drogi á ƺłá ná rolą / lecż bił Báláám oślicę / áby ją náwiódł ná drogę.
Tedy ſtánął Anjoł Páńſki ná śćieƺce u winnicy miedzy dwomá płoty.
A widząc oślicá Aniołá Páńſkiego / przyćiſkáłá śię do płotá / ták iż przypárłá nogę Báláámową do śćiány ; á on znowu ją bił.
Potem Anjoł Páńſki ƺedł dálej / y ſtánął ná miejſcu ćiáſnem / gdźie nie było drogi do uſtąpieniá ná práwo áni ná lewo ;
A widząc oślicá Aniołá Páńſkiego / pádłá pod Báláámem ; y rozgniewał śię Báláám wielce / á bił oślicę kijem.
Zátem otworzył Pán uſtá oney oślicy / y rzekłá do Báláámá : Cóżem ći ucżyniłá / żeś mię bił już po trzy kroć?
Y rzekł Báláám do oślicy : Iż ze mnie ƺydźiƺ ; bym był miał miecż w ręku ſwych / byłbym ćię teraz zábił.
Tedy oślicá rzekłá do Báláámá : Azáżem ja nie oślicá twojá / ná którejś jeżdżał / jákoś mię doſtał / áż do tego cżáſu? Azáżem zwykłá ták tobie cżynić? A on rzekł : Nigdy.
Wtem otworzył Pán ocży Báláámowe / że obácżył Aniołá Páńſkiego / ſtojącego ná drodze / y miecż jego dobyty w ręce jego ; tedy ſkłoniwƺy śię / pokłonił śię twárzą ſwoją.
Y rzekł do niego Anjoł Páńſki : Przecżżeś bił oślicę ſwoję już po trzy kroć? Otom Ja wyƺedł / ábym śię tobie ſprzećiwił ; bo przewrotná jeſt drogá twojá przedemną ;
A widząc mię oślicá uſtąpiłá przedemną po trzy kroć / á gdyby byłá nie uſtąpiłá przedemną / jużbym ćię był teraz zábił á onębym był żywą zoſtáwił.
Zátem rzekł Báláám do Aniołá Páńſkiego : Zgrzeƺyłem / ábowiem nie wiedźiáłem / żeś ty ſtánął przećiwko mnie ná drodze ; przetoż teraz / jeſlić śię to nie podobá / wrócę śię.
Lecż Anjoł Páńſki rzekł do Báláámá : Jedź z ludźmi tymi ; wƺákże tylko / co Ja tobie powiem / mówić będźieƺ. Y jechał Báláám z kśiążęty Bálákowymi.
A gdy uſłyƺał Bálák / iż przyjeżdżá Báláám / wyjechał przećiwko niemu / do niektórego miáſtá Moábſkiego / które jeſt ná gránicy Arnonu / które jeſt przy końcu gránicy.
Tedy rzekł Bálák do Báláámá : Azáżem z pilnośćią nie poſyłał do ćiebie wzywájąc ćię? cżemużeś nie przyjechał do mnie? ázaż ćię zácnie ucżćić nie mogę?
Y rzekł Báláám do Báláká : Otom przyjechał do ćiebie ; izali teraz / choćbym chćiał / będę mógł co mówić? ſłowo / które włoży Bóg w uſtá moje / mówić będę.
Tedy jechał Báláám z Bálákiem / á przyjecháli do miáſtá Huſot.
A ták Bálák dał nábić wołów y owiec / y poſłał do Báláámá / y do kśiążąt / którzy z nim byli.
Y ſtáło śię názájutrz / że wźiął Bálák Báláámá / y wprowádźił go ná wyżyny Báálowe / ſkąd widźiał y nájdálƺą cżęść ludu.
