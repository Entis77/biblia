Stámtąd ruƺyli śię ſynowie Izráelſcy / y położyli śię obozem ná polách Moábſkich / z tej ſtrony Jordánu przećiw Jerychowi.
A widząc Bálák / ſyn Seforów / wƺyſtko / co ucżynił Izráel Amorejcżykowi /
Uląkł śię Moáb tego ludu wielce / bo go było wiele ; y zátrwożył ſobą Moáb dlá ſynów Izráelſkich.
Przetoż rzekł Moáb do ſtárƺych Mádyjáńſkich : Teráz pożre to mnóſtwo wƺyſtko / co jeſt około náſ / jáko wół pożerá tráwę polną. A Bálák / ſyn Seforów / był królem Moábſkim ná on cżáſ.
Y poſłáł poſły do Báláámá / ſyná Beorowego / do Pethoru miáſtá / które jeſt nád rzeką źiemi ſynów ludu jego / áby go wezwáno / mówiąc : Oto lud wyƺedł z Egiptu / oto okrył wierzch źiemi / y oſádzá śię przećiwko mnie.
Przetoż teráz przyjdź proƺę / á przeklináj mnie kwoli lud ten / bo możniejƺy jeſt nád mię ; owá ſnáć go będę mógł poráźić / y wygnáć go z źiemi ; bo já wiem / że komu błogoſłáwiƺ / błogoſłáwiony będźie ; á kogo przeklináƺ / przeklęty będźie.
Poƺli tedy ſtárśi Moábſcy / y ſtárśi Mádyjáńſcy / májąc zápłátę zá wieƺcżbę w rękách ſwych.
A przyƺedłƺy do Báláámá / powiedźieli mu ſłowá Bálákowe. Y rzekł do nich : Zoſtáńćie tu przez noc / á dám wám odpowiedź / jáko mi oznájmi Pán.
Y zoſtáły kśiążętá Moábſkie z Báláámem. Tedy przyƺedł Bóg do Báláámá / y rzekł : Cóż to zá mężowie u ćiebie?
Y odpowiedźiáł Báláám Bogu : Bálák / ſyn Seforów / król Moábſki / poſłáł do mnie / mówiąc :
Oto lud / który wyƺedł z Egiptu / y okrył wierzch źiemi ; terázże pójdź / przeklináj mi go ; ſnáć będę mógł wálcżyć z nim / y wypędzę go.
Tedy rzekł Bóg do Báláámá : Nie chodź z nimi / áni przeklináj ludu tego ; bo jeſt błogoſłáwiony.
A wſtáwƺy ráno Báláám rzekł do kśiążąt Bálákowych : Wróććie śię do źiemi wáƺej ; bo mi nie pozwálá Pán puśćić śię w drogę z wámi.
Wſtáwƺy tedy kśiążętá Moábſkie / wróćili śię do Báláká / y powiedźieli : Nie chćiáł Báláám iść z námi.
Tedy powtóre poſłáł Bálák więcej kśiążąt / y zácniejƺych nád pierwƺe.
Którzy przyƺedłƺy do Báláámá / mówili mu : Ták mówi Bálák / ſyn Seforów : Nie oćiągáj śię proƺę przyjść do mnie ;
Abowiem ći wielką ucżćiwość wyrządzę / y wƺyſtko / cobyś mi rzekł / ucżynię ; tylko przyjdź proƺę á przeklináj mi ten lud.
Tedy odpowiedźiáł Báláám / y rzekł do ſług Bálákowych : Choćby mi dáł Bálák pełen dom ſwój ſrebrá y złotá / nie mógłbym przeſtąpić ſłowá Páná / Bogá mego / y ucżynić przećiwko niemu / bądź máło bądź wiele ;
Ale proƺę zoſtáńćie tu y wy przez noc / á dowiem śię / co jeƺcże Pán będźie mówił ze mną.
Tedy przyƺedł Bóg do Báláámá w nocy / y rzekł do niego : Jeźliż / áby ćię wezwáli / przyƺli mężowie ći / wſtáńże / idź z nimi ; á wƺákże / coć rozkáżę / to ucżyniƺ.
Tedy wſtáwƺy Báláám ráno / ośiodłáł oślicę ſwoję / y jecháł z kśiążęty Moábſkimi.
Y rozpálił śię gniew Boży / że on jecháł ; y ſtánął Anioł Páńſki ná drodze / áby mu záſtąpił ; á on jecháł ná oślicy ſwojej / y dwoje pácholąt jego z nim.
A gdy ujrzáłá oślicá Aniołá Páńſkiego / ſtojącego ná drodze / á miecż jego dobyty w ręce jego / tedy uſtąpiłá oślicá z drogi á ƺłá ná rolą / lecż bił Báláám oślicę / áby ją náwiódł ná drogę.
Tedy ſtánął Anioł Páńſki ná śćieƺce u winnicy między dwomá płoty.
A widząc oślicá Aniołá Páńſkiego / przyćiſkáłá śię do płotá / ták iż przypárłá nogę Báláámową do śćiány ; á on znowu ją bił.
Potem Anioł Páńſki ƺedł dálej / y ſtánął ná miejſcu ćiáſnem / gdźie nie było drogi do uſtąpieniá ná práwo áni ná lewo ;
A widząc oślicá Aniołá Páńſkiego / pádłá pod Báláámem ; y rozgniewáł śię Báláám wielce / á bił oślicę kijem.
Zátem otworzył Pán uſtá onej oślicy / y rzekłá do Báláámá : Cóżem ći ucżyniłá / żeś mię bił już po trzy kroć?
Y rzekł Báláám do oślicy : Iż ze mnie ƺydźiƺ ; bym był miáł miecż w ręku ſwych / byłbym ćię teráz zábił.
Tedy oślicá rzekłá do Báláámá : Azáżem já nie oślicá twojá / ná którejś jeżdżáł / jákoś mię doſtáł / áż do tego cżáſu? Azáżem zwykłá ták tobie cżynić? A on rzekł : Nigdy.
Wtem otworzył Pán ocży Báláámowe / że obácżył Aniołá Páńſkiego / ſtojącego ná drodze / y miecż jego dobyty w ręce jego ; tedy ſkłoniwƺy śię / pokłonił śię twárzą ſwoją.
Y rzekł do niego Anioł Páńſki : Przecżżeś bił oślicę ſwoję już po trzy kroć? Otom Já wyƺedł / ábym śię tobie ſprzećiwił ; bo przewrotná jeſt drogá twojá przedemną ;
A widząc mię oślicá uſtąpiłá przedemną po trzy kroć / á gdyby byłá nie uſtąpiłá przedemną / jużbym ćię był teráz zábił á onębym był żywą zoſtáwił.
Zátem rzekł Báláám do Aniołá Páńſkiego : Zgrzeƺyłem / ábowiem nie wiedźiáłem / żeś ty ſtánął przećiwko mnie ná drodze ; przetoż teráz / jeźlić śię to nie podobá / wrócę śię.
Lecż Anioł Páńſki rzekł do Báláámá : Jedź z ludźmi tymi ; wƺákże tylko / co Já tobie powiem / mówić będźieƺ. Y jecháł Báláám z kśiążęty Bálákowymi.
A gdy uſłyƺáł Bálák / iż przyjeżdżá Báláám / wyjecháł przećiwko niemu / do niektórego miáſtá Moábſkiego / które jeſt ná gránicy Arnonu / które jeſt przy końcu gránicy.
Tedy rzekł Bálák do Báláámá : Azáżem z pilnośćią nie poſyłáł do ćiebie wzywájąc ćię? cżemużeś nie przyjecháł do mnie? ázáż ćię zácnie ucżćić nie mogę?
Y rzekł Báláám do Báláká : Otom przyjecháł do ćiebie ; izáli teráz / choćbym chćiáł / będę mógł co mówić? ſłowo / które włoży Bóg w uſtá moje / mówić będę.
Tedy jecháł Báláám z Bálákiem / á przyjecháli do miáſtá Huſot.
A ták Bálák dáł nábić wołów y owiec / y poſłáł do Báláámá / y do kśiążąt / którzy z nim byli.
Y ſtáło śię názájutrz / że wźiął Bálák Báláámá / y wprowádźił go ná wyżyny Báálowe / ſkąd widźiáł y nájdálƺą cżęść ludu.
