Y rzekł Pán do Mojżeƺá / mówiąc :
Rozkáż ſynom Izráelſkim / áby wyrzućili z obozu káżdego trędowátego / y káżdego / który ćierpi płynienie náśieniá / y káżdego / który śię ſplugáwił nád umárłym ;
Ták mężcżyznę jáko y niewiáſtę wyrzućićie ; precż zá obóz wyrzućićie je / áby nie ſplugáwili obozu tych / miedzy którymi Ja mieƺkám.
Y ucżynili ták ſynowie Izráelſcy / á wygnáli je precż zá obóz ; jáko rozkázał Pán Mojżeƺowi / ták ucżynili ſynowie Izráelſcy.
Nád to rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim : Mąż álbo niewiáſtá / gdyby popełnili jákikolwiek grzech ludzki / dopuśćiwƺy śię wyſtępku przećiwko Pánu / á byłáby winná oná duƺá :
Tedy wyznáją grzech ſwój / którego śię dopuśćili / y wrócą to / w cżemby winni byli cále ; á przydáwƺy jeƺcże nád to piątą cżęść / oddádzą onemu / przećiw któremu zgrzeƺyli.
A jeſliby nie było tego / komuby ƺkodę trzebá nágrodźić / oná ƺkodá oddáná będźie Pánu / y zoſtánie kápłánowi oprócż báráná ocżyƺcżeniá / przez którego ma być ocżyƺcżony.
Káżda też ofiárá podnoƺeniá ze wƺech rzecży poświęconych od ſynów Izráelſkich / którą przynioſą do kápłáná / jemu śię doſtánie.
Owá rzecży poświęcone od kogożkolwiek / jego będą ; y ktoby co oddał kápłánowi / jemu zoſtánie.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á powiedz im : Káżdy mąż / któregoby żoná wyſtąpiłá / y dopuśćiłáby śię grzechu przećiwko niemu ;
A złącżyłby śię inƺy z nią złącżeniem náśieniá / á byłoby to ſkryte przed ocżymá mężá jey / y táiłáby śię / będąc ſplugáwioną / á świádkáby nie było przećiwko niej / ániby jey záſtáno ;
Jednák przypádłby nań duch zápálcżywośći / y miáłby w podejrzeniu żonę ſwą / któráby ſplugáwioná byłá ; álbo żeby przypádł nań duch zápálcżywośći / y miáłby w podejrzeniu żonę ſwą / któráby ſplugáwioná nie byłá :
Tedy przywiedźie on mąż żonę ſwoję do kápłáná / y przynieśie z nią ofiárę jey / dźieśiątą cżęść efy mąki jęcżmienney / nie lejąc ná nią oliwy / áni kłádąc ná nią kádźidłá ; ábowiem jeſt ofiárá podejrzeniá / ofiárá śniedná / pámiętná / przywodzącá ná pámięć grzech.
A ták będźie ją ofiárował kápłán / y ſtáwi ją przed oblicżnośćią Páńſką.
Y weźmie kápłán wody świętey w nacżynie gliniáne / y prochu / który będźie ná tle przybytku / weźmie kápłán / á wſypie do wody.
Potem poſtáwi kápłán niewiáſtę przed Pánem / y odkryje głowę niewiáſty / á da w ręce jey ofiárę śniedną pámiętną ; ofiárá to śniedná podejrzeniá ; á kápłán będźie miał w ręce wodę gorzką przeklęſtwá.
Y poprzyśięże ją kápłán / y rzecże do niewiáſty : Jeſli nie ſpał kto inƺy z tobą / á jeſliś śię nie unioſłá w grzech niecżyſty przy mężu ſwym / bądź nienáruƺoná od tey wody gorzkiej przeklęſtwá ;
Ale jeſliżeś uſtąpiłá od mężá twego / y jeſteś ſplugáwioná / á kto inny ſpał z tobą oprócż mężá twego :
Tedy poprzyśięże kápłán niewiáſtę onę przyśięgą przeklęſtwá / y rzecże do niej : Niecháj ćię podá Pán ná złorzecżenie y ná przeklinánie miedzy ludem twoim / przepuśćiwƺy / áby łono twoje wypádło / y żywot twój opuchł :
Niechże przenikną te wody przeklęte wnętrznośći twoje / áby opuchł żywot twój / y wypádło łono twoje ; y odpowie niewiáſtá : Amen. Amen.
Tedy nápiƺe te przeklęſtwá kápłán ná kśięgách / á omyje je oną wodą gorzką ;
Y da śię nápić niewieśćie wody gorzkiej przeklęſtwá / y przenikną ją wody przeklęſtwá / y obrócą śię w gorzkość.
Potem weźmie kápłán z rąk niewiáſty oney ofiárę śniedną podejrzeniá / á będźie ją podnośił przed Pánem / ofiárując ją ná ołtárzu ;
Weźmie też kápłán gárść pámiętnego z ofiáry śniedney / y ſpáli to ná ołtárzu / potem da wypić wodę niewieśćie.
A gdy śię jey da nápić oney wody / ſtánie śię / jeſliby ſplugáwioná byłá / y wyſtąpiłá grzechem przećiwko mężowi ſwemu / że ją przenikną wody przeklęſtwá / y obrócą śię w gorzkość / y opuchnie żywot jey / y wypádnie łono jey / y ſtánie śię niewiáſtá oná przeklęſtwem miedzy ludem ſwoim.
A jeſliby nie byłá ſplugáwioná niewiáſtá / áleby cżyſtá byłá / niewinná zoſtánie / y dźiátki rodźić będźie.
Táć jeſt uſtáwá podejrzeniá / gdyby uſtąpiłá żoná od mężá ſwego / y byłáby ſplugáwioná ;
Albo żeby ná mężá przypádł duch zápálcżywy / á miáłby w podejrzeniu żonę ſwoję / y poſtáwiłby ją przed Pánem / á ucżyniłby z nią kápłán wƺyſtko według tey uſtáwy ;
Tedy nie będźie mąż on winien grzechu ; ále niewiáſtá oná ponieśie niepráwość ſwoję.
