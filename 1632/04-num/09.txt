Y rzekł Pán do Mojżeƺá ná puƺcży Synáj / roku wtórego po wyjśćiu ich z ziemi Egipſkiej / mieśiącá pierwƺego / mówiąc :
Niech obchodzą ſynowie Izráelſcy święto przejśćiá cżáſu náznácżonego.
Cżternáſtego dniá mieśiącá tego / między dwomá wiecżorámi / obchodzić je będziećie cżáſu náznácżonego ; według wƺyſtkich obrzędów jego / y według wƺyſtkich ceremonii jego / obchodzić je będziećie.
Mówił tedy Mojżeƺ do ſynów Izráelſkich / áby obchodzili święto przejśćiá.
Y obchodzili święto przejśćiá / pierwƺego mieśiącá / cżternáſtego dniá / między dwomá wiecżorámi / ná puƺcży Synáj ; według wƺyſtkiego / jáko rozkázáł Pán Mojżeƺowi / ták ucżynili ſynowie Izráelſcy.
Y byli niektórzy ludzie / którzy śię byli ſplugáwili nád umárłym cżłowiekiem / y nie mogli obchodzić świętá przejśćiá dniá onego ; tedy przyſtąpili do Mojżeƺá y do Aároná w tenże dzień ;
Y rzekli oni ludzie do niego : Zmázáliśmy śię nád umárłym : y nie będzież nám wolno oddáć ofiáry Pánu cżáſu náznácżonego weſpół z ſynmi Izráelſkimi?
Którym odpowiedziáł Mojżeƺ : Poſtójćie / áż uſłyƺę / co rozkáże Pán o wáſ.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim y rzecż : Jeźliby śię kto zmázáł nád umárłym / álboby ná drodze dálekiej był / ták z wáſ / jáko y z potomſtwá wáƺego / przećię będzie odpráwowáł święto przejśćiá Pánu.
Mieśiącá wtórego / cżternáſtego dniá / między dwomá wiecżorámi / odpráwowáć je będą ; z przáśnemi chleby / y z gorzkiemi zioły jeść je będą :
Nie zoſtáwią nic z niego do jutrá / y kośći nie złámią w nim ; według wƺyſtkiego poſtánowieniá świętá przejśćiá odpráwowáć je będą.
Ale cżłowiek / któryby był cżyſty / á nie byłby w drodze / y nie obchodziłby świętá przejśćiá / tedy duƺá oná wykorzenioná będzie z ludu ſwego / bo ofiáry Páńſkiej nie odpráwowáł cżáſu náznácżonego ; grzech ſwój ponieśie on cżłowiek.
A jeźliby przychodzień mieƺkájący między wámi obchodził święto przejśćiá Pánu / według uſtáwy świętá przejśćiá y według obrzędów jego obchodzić je będzie ; uſtáwá jedná będzie / wám y przychodniowi / y zrodzonemu w ziemi.
Dniá tedy onego / którego wyſtáwiony był przybytek / obłok okrył przybytek nád námiotem świádectwá / á wiecżór bywáło nád przybytkiem jáko widzenie ogniá áż do poránku.
Ták bywáło uſtáwicżnie ; we dnie okrywáł go obłok / á jáko widzenie ogniá w nocy.
A gdy śię podnáƺáł obłok od námiotu / tedy śię ruƺáli ſynowie Izráelſcy ; á gdziekolwiek ſtáwáł obłok / támże ſtánowili obóz ſynowie Izráelſcy.
Ná rozkázánie Páńſkie ćiągnęli ſynowie Izráelſcy / y ná rozkázánie Páńſkie ſtánowili obóz ; po wƺyſtkie dni / których zoſtáwáł obłok nád przybytkiem / y oni leżeli obozem.
A gdy trwáł obłok on nád przybytkiem przez wiele dni / tedy odpráwowáli ſynowie Izráelſcy ſtráż Pánu / á nie ruƺáli śię.
Ale gdy nie długo trwáł obłok nád przybytkiem / ná rozkázánie Páńſkie ſtánowili obóz / y ná rozkázánie Páńſkie ćiągnęli.
A gdy bywáł obłok od wiecżorá áż do poránku / á podnośił śię záś poránku / tedy ćiągnęli ; ták we dnie jáko y w nocy / gdy śię podnióſł obłok / ćiągnęli.
A jeźli przez dwá dni / álbo przez mieśiąc / álbo też przez rok trwáł obłok nád przybytkiem / zoſtáwájąc nád nim / obozem leżeli ſynowie Izráelſcy / y nie ruƺáli śię ; ále gdy śię on podnośił / y oni śię ruƺáli.
Ná rozkázánie Páńſkie ſtánowili obóz / y ná rozkázánie Páńſkie ćiągnęli / ſtráż Páńſką trzymájąc / jáko im Pán rozkázáł przez Mojżeƺá.
