Y rzekł Pán do Mojżeƺá ná puƺcży Synáj / roku wtórego po wyjśćiu ich z źiemi Egipſkiej / mieśiącá pierwƺego / mówiąc :
Niech obchodzą ſynowie Izráelſcy święto przejśćiá cżáſu náznácżonego.
Cżternáſtego dniá mieśiącá tego / między dwomá wiecżorámi / obchodźić je będźiećie cżáſu náznácżonego ; według wƺyſtkich obrzędów jego / y według wƺyſtkich ceremonii jego / obchodźić je będźiećie.
Mówił tedy Mojżeƺ do ſynów Izráelſkich / áby obchodźili święto przejśćiá.
Y obchodźili święto przejśćiá / pierwƺego mieśiącá / cżternáſtego dniá / między dwomá wiecżorámi / ná puƺcży Synáj ; według wƺyſtkiego / jáko rozkázáł Pán Mojżeƺowi / ták ucżynili ſynowie Izráelſcy.
Y byli niektórzy ludźie / którzy śię byli ſplugáwili nád umárłym cżłowiekiem / y nie mogli obchodźić świętá przejśćiá dniá onego ; tedy przyſtąpili do Mojżeƺá y do Aároná w tenże dźień ;
Y rzekli oni ludźie do niego : Zmázáliśmy śię nád umárłym : y nie będźież nám wolno oddáć ofiáry Pánu cżáſu náznácżonego weſpół z ſynmi Izráelſkimi?
Którym odpowiedźiáł Mojżeƺ : Poſtójćie / áż uſłyƺę / co rozkáże Pán o was.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim y rzecż : Jeſliby śię kto zmázáł nád umárłym / álboby ná drodze dálekiej był / ták z was / jáko y z potomſtwá wáƺego / przećię będźie odpráwowáł święto przejśćiá Pánu.
Mieśiącá wtórego / cżternáſtego dniá / między dwomá wiecżorámi / odpráwowáć je będą ; z przáśnemi chleby / y z gorzkiemi źioły jeść je będą :
Nie zoſtáwią nic z niego do jutrá / y kośći nie złámią w nim ; według wƺyſtkiego poſtánowieniá świętá przejśćiá odpráwowáć je będą.
Ale cżłowiek / któryby był cżyſty / á nie byłby w drodze / y nie obchodźiłby świętá przejśćiá / tedy duƺá oná wykorzenioná będźie z ludu ſwego / bo ofiáry Páńſkiej nie odpráwowáł cżáſu náznácżonego ; grzech ſwój ponieśie on cżłowiek.
A jeſliby przychodźień mieƺkájący między wámi obchodźił święto przejśćiá Pánu / według uſtáwy świętá przejśćiá y według obrzędów jego obchodźić je będźie ; uſtáwá jedná będźie / wám y przychodniowi / y zrodzonemu w źiemi.
Dniá tedy onego / którego wyſtáwiony był przybytek / obłok okrył przybytek nád námiotem świádectwá / á wiecżór bywáło nád przybytkiem jáko widzenie ogniá áż do poránku.
Ták bywáło uſtáwicżnie ; we dnie okrywáł go obłok / á jáko widzenie ogniá w nocy.
A gdy śię podnáƺáł obłok od námiotu / tedy śię ruƺáli ſynowie Izráelſcy ; á gdźiekolwiek ſtáwáł obłok / támże ſtánowili obóz ſynowie Izráelſcy.
Ná rozkázánie Páńſkie ćiągnęli ſynowie Izráelſcy / y ná rozkázánie Páńſkie ſtánowili obóz ; po wƺyſtkie dni / których zoſtáwáł obłok nád przybytkiem / y oni leżeli obozem.
A gdy trwáł obłok on nád przybytkiem przez wiele dni / tedy odpráwowáli ſynowie Izráelſcy ſtráż Pánu / á nie ruƺáli śię.
Ale gdy nie długo trwáł obłok nád przybytkiem / ná rozkázánie Páńſkie ſtánowili obóz / y ná rozkázánie Páńſkie ćiągnęli.
A gdy bywáł obłok od wiecżorá áż do poránku / á podnośił śię záś poránku / tedy ćiągnęli ; ták we dnie jáko y w nocy / gdy śię podnióſł obłok / ćiągnęli.
A jeſli przez dwá dni / álbo przez mieśiąc / álbo też przez rok trwáł obłok nád przybytkiem / zoſtáwájąc nád nim / obozem leżeli ſynowie Izráelſcy / y nie ruƺáli śię ; ále gdy śię on podnośił / y oni śię ruƺáli.
Ná rozkázánie Páńſkie ſtánowili obóz / y ná rozkázánie Páńſkie ćiągnęli / ſtráż Páńſką trzymájąc / jáko im Pán rozkázáł przez Mojżeƺá.
