A teć ſą rodzáje Aároná y Mojżeƺá w dzień / którego mówił Pán z Mojżeƺem ná górze Synáj.
A te ſą imioná ſynów Aáronowych : Pierworodny Nádáb / potem Abiu / Eleázár / y Itámár.
Te ſą imioná ſynów Aáronowych / kápłánów pomázánych / których ręce poświęcone były ku ſpráwowániu urzędu kápłáńſkiego.
Ale pomárli Nádáb y Abiu przed Pánem / gdy ofiárowáli ogień obcy przed Pánem ná puƺcży Synáj / á zeƺli bez potomſtwá ; dlá tego Eleázár y Itámár odpráwowáli urząd kápłáńſki przed oblicżem Aároná / ojcá ſwego.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Káż przyſtąpić pokoleniu Lewiego / á poſtáw je przed Aáronem kápłánem / áby mu ſłużyli /
A trzymáli ſtráż jego / y ſtráż wƺyſtkiego zgromádzeniá / przed námiotem zgromádzeniá / wykonywájąc uſługę przybytku.
Tákże áby ſtrzegli wƺyſtkiego nácżyniá námiotu zgromádzeniá / y trzymáli ſtráż ſynów Izráelſkich / á odpráwowáli uſługę przybytku.
Oddáƺ tedy Lewity Aáronowi / y ſynom jego ; bo włáśnie oddáni ſą jemu z ſynów Izráelſkich.
Aároná záś y ſyny jego przełożyƺ / áby przeſtrzegáli kápłáńſtwá ſwego : bo przyſtąpiłliby kto obcy / umrze.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
A oto / Já wziąłem Lewity z pośrodku ſynów Izráelſkich miáſto wƺelkiego pierworodnego / otwierájącego żywot / między ſynámi Izráelſkimi / y będą moi Lewitowie.
Abowiem mnie przynáleży wƺelkie pierworodne ; ode dniá / któregom pobił wƺelkie pierworodne w ziemi Egipſkiej / poświęćiłem ſobie káżde pierworodne w Izráelu ; od cżłowieká áż do bydlęćiá moi będą ; Jám Pán.
Rzekł też Pán do Mojżeƺá ná puƺcży Synáj / mówiąc :
Policż ſyny Lewiego / według domów ojców ich / według fámilii ich ; káżdego mężcżyznę urodzonego od mieśiącá y wyżej / policżyƺ je.
Y policżył je Mojżeƺ według mowy Páńſkiej / jáko mu było rozkázáno.
Y były ſynów Lewiego te imioná : Gerſon / y Káát / y Meráry.
Te záś imioná ſynów Gerſonowych według domów ich : Lobni y Semei.
A ſynowie Káátowi według domów ſwych : Amrám / y Izáár / Hebron / y Huſyjel.
Synowie záś Merárego według domów ſwych : Náheli y Muzy ; te ſą fámilije Lewiego według domów ojców ich.
Z Gerſoná fámilijá Lobnická / y fámilijá Semeická ; teć ſą fámilije Gerſonowe.
Policżonych ich według licżby káżdego mężcżyzny urodzonego od mieśiącá y wyżej / było policżonych śiedm tyśięcy y pięć ſet.
Te fámilije Gerſonowe zá przybytkiem kłáść śię będą ku záchodowi.
A kśiążęćiem domu ojcá Gerſonitów : Elijázáf / ſyn Láelów.
A pod ſtráżą ſynów Gerſonowych będzie przy námioćie przybytek zgromádzeniá / przybytek y námiot / przykryćie jego / y záſłoná u drzwi námiotu zgromádzeniá ;
Y opony śieni / y záſłoná we drzwiách u śieni / którá jeſt przed przybytkiem y przy ołtárzu w około / y ƺnury jego / do wƺelkiej potrzeby jego.
Z Káátá záś poƺłá fámilijá Amrámitów / y fámilijá Izááritów / y fámilijá Huſyjelitów. Teć były domy Káátytów.
W licżbie wƺyſtkich mężcżyzn urodzonych od mieśiącá y wyżej / ośm tyśięcy y ƺeść ſet / trzymájących ſtráż przy świątnicy.
Te fámilije ſynów Káátowych kłáść śię będą obozem po bok przybytku ku południowi ;
A kśiążęćiem domu ojcá fámilii Káátytów Eliſáfán / ſyn Huſyjelów.
A będzie pod ſtráżą ich ſkrzyniá / y ſtół / y świecżnik / y ołtárze / y nácżyniá świątnicy / któremi uſługowáć będą / y záſłoná / y ze wƺyſtkiemi potrzebámi jej.
A kśiążęćiem nád kśiążęty Lewitów będzie Eleázár / ſyn Aároná kápłáná / poſtánowiony nád tymi / którzy trzymáją ſtráż przy świątnicy.
Od Merárego záś poƺłá fámilijá Máhelitów / y fámilijá Muzytów ; á teć ſą domy Merárytów.
A policżonych ich / według licżby káżdego mężcżyzny urodzonego od mieśiącá y wyżej / ƺeść tyśięcy y dwieśćie.
Kśiążęćiem záś domu ojcá fámilii Merárego Suryjel / ſyn Abiháilów ; á ći kłáść śię będą po bok przybytku / ku północy.
A náleżeć będą do ſtráży ſynów Merárego deſki przybytku / y drągi jego / ſłupy jego / y podſtáwki jego / y wƺyſtkie nácżyniá jego / y wƺyſtkie potrzeby jego ;
Tákże ſłupy śienne w około / y podſtáwki ich / y kotły ich / y ƺnury ich.
A kłáść śię będą obozem przed przybytkiem / po przedniej ſtronie námiotu zgromádzeniá / ná wſchód Mojżeƺ / y Aáron / y ſynowie jego / trzymájący ſtráż przy świątnicy / ſtráż zá ſyny Izráelſkie ; á obcy gdyby przyſtąpił / umrze.
A ták wƺyſtkich policżonych Lewitów od Mojżeƺá y Aároná / ná rozkázánie Páńſkie / według domów ich / wƺyſtkich mężcżyzn urodzonych od mieśiącá y wyżej / było dwádzieśćiá tyśięcy y dwá.
Tedy rzekł Pán do Mojżeƺá : Policż wƺyſtkie pierworodne mężcżyzny między ſyny Izráelſkimi od mieśiącá y wyżej / á ucżyń ſummę imion ich.
A weźmieƺ mi Lewity ( Já Pán ) miáſto wƺyſtkich pierworodnych w ſyniech Izráelſkich / tákże bydłá Lewitów zá wƺyſtkie pierworodne bydłá ſynów Izráelſkich.
Policżył tedy Mojżeƺ / jáko mu Pán rozkázáł / wƺyſtkie pierworodne w ſyniech Izráelſkich.
A było wƺyſtkich pierworodnych mężcżyzn według licżby imion / urodzonych od mieśiącá y wyżej / policżonych ich dwádzieśćiá y dwá tyśiące / dwieśćie / śiedmdzieśiąt y trzy.
Y rzekł Pán do Mojżeƺá / mówiąc :
Weźmij Lewity miáſto wƺyſtkich pierworodnych z ſynów Izráelſkich / bydłá tákże Lewitów miáſto bydłá ich / y będą moimi Lewitowie ; Jám Pán.
A zá okup onych dwu ſet / śiedmiudzieśiąt y trzech / którzy zbywáją nád licżbę Lewitów / z pierworodnych ſynów Izráelſkich /
Weźmieƺ po pięć ſyklów ná káżdą głowę ; według ſyklá świątnicy bráć będzieƺ ; dwádzieśćiá pieniędzy wáży ſykiel.
Y oddáƺ te pieniądze Aáronowi y ſynom jego zá okup onych / którzy zbywáją nád licżbę ich.
Wziął tedy Mojżeƺ pieniądze okupu od tych / którzy zbywáli nád te / które okupili ſobą Lewitowie.
Od pierworodnych ſynów Izráelſkich wziął pieniędzy onych tyśiąc / trzy ſtá / ƺeśćdzieśiąt y pięć ſyklów według ſyklá świątnicy ;
Y oddáł te pieniądze okupu Mojżeƺ Aáronowi y ſynom jego według ſłowá Páńſkiego / jáko Pán rozkázáł Mojżeƺowi.
