A teć ſą rodzáje Aároná y Mojzeƺá / w dźień którego mówił PAN z Mojzeƺem ná górze Synáj.
A te <i>ſą</i> imioná Synów Aáronowych : pierworodny Nádáb / potym Abiu / Eleázár / y Itámár.
Te <i>ſą</i> imioná Synów Aáronowych / Kápłanów pomázánych / których ręce poświęcone były ku ſpráwowániu Urzędu kápłáńſkiego.
Ale pomárli Nádáb y Abiu przed PAnem / gdy ofiárowáli Ogień obcy przed PAnem / ná puƺczy Synáj á zeƺli bez potomſtwá : dla tego Eleázár y Itámár / odpráwował Urząd kápłáńſki przed obliczem Aároná Ojcá ſwego.
Tedy rzekł PAN do Mojzeƺá / mówiąc :
Każ przyſtąpić pokoleniu Lewiego / á poſtaw je przed Aáronem kápłanem / áby mu ſłużyli /
A trzymáli ſtraż jego / y ſtraż wƺyſtkiego zgromádzenia / przed Namiotem zgromádzenia / wykonywájąc uſługę Przybytku :
Tákże áby ſtrzegli wƺyſtkiego naczynia Namiotu zgromádzenia / y trzymáli ſtraż Synów Izráelſkich / á odpráwowáli uſługę Przybytku.
Oddaƺ tedy Lewity Aáronowi / y Synom jego / <i>bo</i> właśnie oddáni ſą jemu z Synów Izráelſkich.
Aároná záś y Syny jego przełożyƺ / áby przeſtrzegáli kápłáńſtwá ſwego : bo / przyſtąpiłliby kto obcy / umrze.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
A oto ja / wźiąłem Lewity z pośrzodku Synów Izráelſkich miáſto wƺelkiego pierworodnego / otwierájącego żywot / miedzy Synámi Izráelſkimi / y będą moji Lewitowie :
Abowiem mnie przynależy wƺelkie pierworodne / ode dniá któregom pobił wƺelkie pierworodne w źiemi Egipſkiey / poświęćiłem ſobie káżde pierworodne w Izráelu / od człowieká áż do bydlęćiá / moji będą : <i>JA PAN.</i>
Rzekł też PAN do Mojzeƺá ná puƺczy Synáj / mówiąc :
Policz Syny Lewiego / według domów Ojców ich / według fámilii ich : Káżdego mężczyznę / urodzonego od mieśiącá y wyżey / policzyƺ je.
Y policzył je Mojzeƺ według mowy PAńſkiey / jáko mu było rozkazano.
Y były Synów Lewiego te imioná : Gerſon / y Káát / y Merári.
Te záś imioná Synów Gerſonowych według domów ich : Lebni y Semej.
A Synowie Káátowi według domów ſwych : Amrā / y Iſáár / Hebron / y Huziel.
Synowie záś Merárego / według domów ſwych : Máheli y Muſy : te ſą fámilie Lewiego / według domów Ojców ich.
Z Gerſoná fámilia Lebnicka / y fámilia Semeicka : teć ſą fámilie Gerſonowe :
Policzonych ich / według liczby káżdego mężczyzny / <i>urodzonego</i> od mieśiącá y wyżey / było policzonych śiedm tyśięcy / y pięć ſet.
<i>Te</i> fámilie Gerſonowe / zá Przybytkiem kłáść śię będą / ku Zachodowi.
A Kśiążęćiem domu Ojcá Gerſonitów : Eliáſáf Syn Láelów.
A pod ſtrażą Synów Gerſonowych <i>będźie</i> przy Namiećie Przybytek zgromádzenia : Przybytek y Namiot / przykryćie jego / y zaſłoná u drzwi Namiotu zgromádzenia :
Y opony śieni / y zaſłoná we drzwiách / u śieni która jeſt przed Przybytkiem y przy Ołtarzu w około / y ƺnury jego / do wƺelkiey potrzeby jego.
Z Káátá záś <i>poƺłá</i> fámilia Amrámitów / y fámilia Iſáárytów / y fámilia Hebronitów / y fámilia Huzielitów : Teć były domy Káátytów.
W liczbie wƺyſtkich mężczyzn urodzonych od mieśiącá y wyżey / ośm tyśięcy / y ƺeść ſet / trzymájących ſtraż przy świątnicy.
<i>Te</i> fámilie Synów Káátowych kłáść śię będą Obozem : Po bok Przybytku ku Południowi :
A Kśiążęćiem domu Ojcá fámiliej Káátytów / Eliſáfán Syn Huzielów.
A będźie pod ſtrażą ich ſkrzyniá / y ſtół / y świecznik / y Ołtarze / y naczynia świątnice / którymi uſługowáć będą / y zaſłoná / y ze wƺyſtkiemi potrzebámi jey.
A Kśiążęćiem nád Kśiążęty Lewitów / <i>będźie</i> Eleázár / Syn Aároná kápłáná / poſtánowiony nád tymi którzy trzymáją ſtraż przy świątnicy.
Od Merárego záś <i>poƺłá</i> fámilia Máhálitów / y fámilia Muſytów : á teć ſą domy Merárytów.
A policzonych ich / według liczby káżdego mężczyzny / urodzonego od mieśiącá y wyżey / ƺeść tyśięcy / y dwie śćie.
Kśiążęćiem záś domu Ojcá fámilii Merárego / Suriel Syn Abiháilów / <i>á ći</i> kłáść śię będą po bok Przybytku / ku Pułnocy.
A należeć będą do ſtraży Synów Merárego / deſki Przybytku / y drągi jego / ſłupy jego / y podſtáwki jego / y wƺyſtkie naczynia jego / y wƺyſtkie potrzeby jego.
Tákże ſłupy śienne w około / y podſtáwki ich / y kotły ich / y ƺnury ich.
A kłáść śię będą Obozem / przed Przybytkiem po przedniey ſtronie Namiotu zgromádzenia / ná Wſchód / Mojzeƺ y Aáron y Synowie jego / trzymájący ſtraż przy świątnicy / ſtraż zá Syny Izráelſkie : á obcy gdyby przyſtąpił / umrze.
<i>A ták</i> wƺyſtkich policzonych Lewitów od Mojzeƺá y Aároná / ná rozkazánie PAńſkie / według domów ich / wƺyſtkich mężczyzn urodzonych / od mieśiącá y wyżey / <i>było</i> dwádźieśćiá tyśięcy y dwá.
Tedy rzekł PAN do Mojzeƺá : Policz wƺyſtkie pierworodne mężczyzny / miedzy Syny Izráelſkimi / od mieśiącá y wyżey / á uczyń ſummę imion ich.
A weźmieƺ mi Lewity / <i>JA PAN,</i> miáſto wƺyſtkich pierworodnych w Syniech Izráelſkich : Tákże bydłá Lewitów / zá wƺyſtkie pierworodne bydłá / Synów Izráelſkich.
Policzył tedy Mojzeƺ jáko mu PAN rozkazał / wƺyſtkie pierworodne / w Syniech Izráelſkich.
A było wƺyſtkich pierworodnych mężczyzn / według liczby imion / urodzonych od mieśiącá y wyżey / policzonych ich / dwádźieśćiá y dwá tyśiące / dwie śćie / śiedm dźieśiąt y trzy.
Y rzekł PAN do Mojzeƺá / mówiąc :
Weźmi Lewity miáſto wƺyſtkich pierworodnych z Synów Izráelſkich / bydłá tákże Lewitów miáſto bydłá ich : y będą mojimi Lewitowie / <i>JA PAN.</i>
A zá okup onych dwu ſet / śiedmidźieśiąt / y trzech / którzy zbywáją nád <i>liczbę</i> Lewitów z pierworodnych Synów Izráelſkich /
Weźmieƺ po piąći ſyklów ná káżdą głowę / według Syklá świątnice bráć będźieƺ : dwádźieśćiá pieniędzy <i>waży</i> Sykiel.
Y oddaƺ te pieniądze Aáronowi / y Synom jego / zá okup onych którzy zbywáją nád <i>liczbę</i> ich.
Wźiął tedy Mojzeƺ pieniądze okupu / od tych którzy zbywáli / nád te które okupili <i>ſobą</i> Lewitowie.
Od pierworodnych Synów Izráelſkich / wźiął pieniędzy <i>onych</i> tyśiąc trzy ſtá ƺeśćdźieśiąt / y pięć <i>Syklów</i> / według Syklá świątnice.
Y oddał te pieniądze okupu Mojzeƺ Aáronowi / y Synom jego / według ſłowá PAńſkiego. Jáko PAN rozkazał Mojzeƺowi.
