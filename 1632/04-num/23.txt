Y rzekł Báláám do Báláká : Zbuduj mi tu śiedm ołtárzów / á nágotuj mi tu śiedm ćielców / y śiedm báránów ;
Ucżynił tedy Bálák / jáko mówił Báláám / y ofiárowáł Bálák z Báláámem ćielcá / y báráná ná káżdym ołtárzu.
Potem rzekł Báláám do Báláká : Stáń przy cáłopáleniu twojim / á já odejdę ; owá śię ſnáć ſpotká Pán ze mną / á cokolwiek mi objáwi / opowiem ći ; y odƺedł ſám.
Y ſpotkáł śię Pán z Báláámem ; y rzekł mu Báláám : Poſtáwiłem śiedm ołtárzów / y ofiárowáłem ćielcá y báráná ná káżdym ołtárzu.
Tedy Pán włożył ſłowá w uſtá Báláámowe / y rzekł : Wróć śię do Báláká / á mów ták.
Y wróćił śię do niego / á oto ſtáł u ofiáry ſwojej pálonej / on y wƺyſtkie kśiążętá Moábſkie.
A ták zácżął przypowieść ſwoję / y rzekł : Z Arám przywiódł mię Bálák / król Moábſki / z gór wſchodnich / mówiąc : Przyjdź / przeklináj mi Jákóbá / á przyjdź / złorzecż Izráelowi.
Jákoż já przeklináć mám / kogo Bóg nie przekliná? álbo jáko złorzecżyć mám / komu Pán nie złorzecży?
Bo z wierzchu ſkáł oglądám go / á z págórków będę náń pátrzáł ; oto / lud ten ſám mieƺkáć będźie / á między národy mieƺáć śię nie będźie.
Któż policży proch Jákóbów / y licżbę cżwártej cżęśći Izráelá? Niech umrze duƺá mojá śmierćią ſpráwiedliwych / y niech będźie dokonánie moje / jáko ich.
Tedy rzekł Bálák do Báláámá : Cóżeś mi ucżynił? Ná przeklinánie nieprzyjáćiół moich przyzwáłem ćię / á oto / błogoſłáwiąc błogoſłáwiłeś im.
A on odpowiedźiáł y rzekł : Azáż nie mám tego pilnowáć y mówić / co Pán włożył w uſtá moje?
Y rzekł do niego Bálák : Pójdź proƺę ze mną ná miejſce inne / żebyś go ſtámtąd widźiáł ; ( ále tylko cżęść jego ujrzyƺ / á wƺyſtkiego widźieć nie będźieƺ ; ) przeklinájże mi go ſtámtąd.
Y záwiódł go ná pole Sofim / ná wierzch góry Fázgá / y zbudowáł śiedm ołtárzów / y ofiárowáł ćielcá y báráná ná káżdym ołtárzu.
Rzekł tedy Báláám do Báláká : Zoſtáń tu przy cáłopáleniu twojim / á já zábieżę tám Pánu.
Y záƺedł Pán Báláámowi / który włożył ſłowá w uſtá jego / y rzekł :
Wróć śię do Báláká / á ták mów. Przyƺedł tedy do niego / á oto on ſtáł przy cáłopáleniu ſwojim / y kśiążętá Moábſkie z nim ; y rzekł mu Bálák : Cóż ći powiedźiáł Pán?
Y zácżął rzecż ſwą temi ſłowy : Wſtáń Báláku / á ſłucháj : przyjmij w uƺy ſwe ſłowá moje / ſynu Seforów.
Nie jeſtći Bóg jáko cżłowiek / áby kłámáł / áni jáko ſyn cżłowiecży / żeby żáłowáł ; ázáż on rzecże / á nie ucżyni? wymówi / á nie wypełni?
Otom wźiął rozkázánie / ábym błogoſłáwił ; on błogoſłáwieńſtwo dáł á já go nie odwrócę.
Nie bácży niepráwośći w Jákóbie ; áni widźi przeſtępſtwá w Izráelu ; Pán / Bóg jego / jeſt z nim / á trąbienie zwyćięſtwá królewſkiego przy nim.
Bóg wywiódł je z Egiptu / mocą jednorożcową był mu.
Abowiem niemáƺ wieƺcżby przećiw Jákóbowi / áni wróżki przećiw Izráelowi ; od tego cżáſu mówiono będźie o Jákóbie y o Izráelu / co z nimi ucżynił Bóg.
Oto lud ten jáko lew śilny powſtánie / jáko lwię młode podnieśie śię ; nie ukłádźie śię / áż pożre łupy / y krew pobitych wypije.
Tedy rzekł Bálák do Báláámá : Ani ich przeklináj więcej / áni im też błogoſłáw więcej.
Y odpowiedźiáł Báláám / á rzekł do Báláká : Azáżem ći nie powiádáł / mówiąc / że cokolwiek mówić będźie Pán / to ucżynię?
Y rzekł Bálák do Báláámá : Pójdź / proƺę / záwiodę ćię ná inƺe miejſce / jeźli ſnáć podobá śię Bogu / żebyś je ſtámtąd przeklináł.
Tedy wiódł Bálák Báláámá ná wierzch góry Fegor / którá pátrzy ku puƺcży.
Y rzekł Báláám do Báláká : Zbuduj mi tu śiedm ołtárzów / á nágotuj mi tu śiedm ćielców y śiedm báránów.
Y ucżynił Bálák / jáko mu rozkázáł Báláám / y ofiárowáł ćielcá y báráná ná káżdym ołtárzu.
