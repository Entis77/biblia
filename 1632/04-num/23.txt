Y rzekł Báláám do Báláká : Zbuduj mi tu śiedm ołtárzów / á nágotuj mi tu śiedm ćielców / y śiedm báránów ;
Ucżynił tedy Bálák / jáko mówił Báláám / y ofiárował Bálák z Báláámem ćielcá / y báráná ná káżdym ołtárzu.
Potem rzekł Báláám do Báláká : Stáń przy cáłopáleniu twojim / á ja odejdę ; owá śię ſnadź ſpotká Pán ze mną / á cokolwiek mi objáwi / opowiem ći ; y odƺedł ſám.
Y ſpotkał śię Pán z Báláámem ; y rzekł mu Báláám : Poſtáwiłem śiedm ołtárzów / y ofiárowáłem ćielcá y báráná ná káżdym ołtárzu.
Tedy Pán włożył ſłowá w uſtá Báláámowe / y rzekł : Wróć śię do Báláká / á mów ták.
Y wróćił śię do niego / á oto ſtał u ofiáry ſwojey páloney / on y wƺyſtkie kśiążętá Moábſkie.
A ták zácżął przypowieść ſwoję / y rzekł : Z Arám przywiódł mię Bálák / król Moábſki / z gór wſchodnich / mówiąc : Przyjdź / przeklináj mi Jákóbá / á przyjdź / złorzecż Izráelowi.
Jákoż ja przeklináć mam / kogo Bóg nie przekliná? álbo jáko złorzecżyć mam / komu Pán nie złorzecży?
Bo z wierzchu ſkał oglądám go / á z págórków będę nań pátrzał ; oto / lud ten ſám mieƺkáć będźie / á miedzy národy mieƺáć śię nie będźie.
Któż policży proch Jákóbów / y licżbę cżwártey cżęśći Izráelá? Niech umrze duƺá mojá śmierćią ſpráwiedliwych / y niech będźie dokonánie moje / jáko ich.
Tedy rzekł Bálák do Báláámá : Cóżeś mi ucżynił? Ná przeklinánie nieprzyjáćiół mojich przyzwáłem ćię / á oto / błogoſłáwiąc błogoſłáwiłeś im.
A on odpowiedźiał y rzekł : Azaż nie mam tego pilnowáć y mówić / co Pán włożył w uſtá moje?
Y rzekł do niego Bálák : Pójdź proƺę ze mną ná miejſce inne / żebyś go ſtámtąd widźiał ; ( ále tylko cżęść jego ujrzyƺ / á wƺyſtkiego widźieć nie będźieƺ ; ) przeklinájże mi go ſtámtąd.
Y záwiódł go ná pole Sofim / ná wierzch góry Fázgá / y zbudował śiedm ołtárzów / y ofiárował ćielcá y báráná ná káżdym ołtárzu.
Rzekł tedy Báláám do Báláká : Zoſtáń tu przy cáłopáleniu twojim / á ja zábieżę tám Pánu.
Y záƺedł Pán Báláámowi / który włożył ſłowá w uſtá jego / y rzekł :
Wróć śię do Báláká / á ták mów. Przyƺedł tedy do niego / á oto on ſtał przy cáłopáleniu ſwojim / y kśiążętá Moábſkie z nim ; y rzekł mu Bálák : Cóż ći powiedźiał Pán?
Y zácżął rzecż ſwą temi ſłowy : Wſtáń Báláku / á ſłucháj : przyjmij w uƺy ſwe ſłowá moje / ſynu Seforów.
Nie jeſtći Bóg jáko cżłowiek / áby kłámał / áni jáko ſyn cżłowiecży / żeby żáłował ; ázaż on rzecże / á nie ucżyni? wymówi / á nie wypełni?
Otom wźiął rozkázánie / ábym błogoſłáwił ; on błogoſłáwieńſtwo dał á ja go nie odwrócę.
Nie bácży niepráwośći w Jákóbie ; áni widźi przeſtępſtwá w Izráelu ; Pán / Bóg jego / jeſt z nim / á trąbienie zwyćięſtwá królewſkiego przy nim.
Bóg wywiódł je z Egiptu / mocą jednorożcową był mu.
Abowiem niemáƺ wieƺcżby przećiw Jákóbowi / áni wróżki przećiw Izráelowi ; od tego cżáſu mówiono będźie o Jákóbie y o Izráelu / co z nimi ucżynił Bóg.
Oto lud ten jáko lew śilny powſtánie / jáko lwię młode podnieśie śię ; nie ukłádźie śię / áż pożre łupy / y krew pobitych wypije.
Tedy rzekł Bálák do Báláámá : Ani ich przeklináj więcey / áni im też błogoſłáw więcey.
Y odpowiedźiał Báláám / á rzekł do Báláká : Azáżem ći nie powiádał / mówiąc / że cokolwiek mówić będźie Pán / to ucżynię?
Y rzekł Bálák do Báláámá : Pójdź / proƺę / záwiodę ćię ná inƺe miejſce / jeſli ſnadź podobá śię Bogu / żebyś je ſtámtąd przeklinał.
Tedy wiódł Bálák Báláámá ná wierzch góry Fegor / która pátrzy ku puƺcży.
Y rzekł Báláám do Báláká : Zbuduj mi tu śiedm ołtárzów / á nágotuj mi tu śiedm ćielców y śiedm báránów.
Y ucżynił Bálák / jáko mu rozkázał Báláám / y ofiárował ćielcá y báráná ná káżdym ołtárzu.
