Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á mów do nich : Mąż álbo niewiáſtá / gdy śię odłącży / cżyniąc ślub Názárejſtwá / áby byli odłącżeni Pánu /
Od winá y mocnego nápoju wſtrzymywáć śię będźie ; octu z winá / y octu z mocnego nápoju pić nie będźie / y wƺyſtkiego / co śię z jágód wytłácżá / nie będźie pił ; tákże jágód winnych / świeżych áni ſuchych / jeść nie będźie.
Po wƺyſtkie dni Názárejſtwá ſwego ze wƺyſtkiego / co wyráſtá z máćicy winnej / od źiárnká áż do łupiny / jeść nie będźie.
Po wƺyſtkie dni ślubu Názárejſtwá ſwego brzytwá nie poſtoi ná głowie jego / áż wynijdźie cżáſ / do którego śię poświęćił Pánu ; będźie świętym / á zápuśći włoſ ná głowie ſwojej.
Po wƺyſtkie dni / których śię odłącży Pánu / do umárłego nie wnijdźie.
Nád ojcem ſwym / y nád mátką ſwą / nád brátem ſwym / y nád śioſtrą ſwą / nie ſplugáwi śię / gdyby zmárli ; ábowiem poświęcenie Bogá ſwego má ná głowie ſwojej.
Po wƺyſtkie dni Názárejſtwá ſwego świętym będźie Pánu.
Y gdyby kto umárł przy nim z prędká á nágle / y ſplugáwiłby głowę poświęceniá jego / ogoli głowę ſwoję w dźień ocżyƺcżeniá ſwego ; dniá śiódmego ogoli ją.
A dniá óſmego przynieśie dwie ſynogárlice / álbo dwoje gołąbiąt do kápłáná ku drzwiom námiotu zgromádzeniá ;
Y będźie kápłán ofiárowáł jedno zá grzech / á drugie ná ofiárę cáłopáleniá / y ocżyśći go od tego / cżem zgrzeƺył nád umárłym / á poświęći głowę jego dniá onego.
Potem odłącży Pánu dni Názárejſtwá ſwego / ofiárując báránká rocżnego zá wyſtępek ; á dni one pierwƺe dáremne będą / gdyż ſplugáwione było Názárejſtwo jego.
A toć jeſt práwo Názárejcżyká : Gdy śię wypełnią dni Názárejſtwá jego / przyjdźie do drzwi námiotu zgromádzeniá /
Y ofiárowáć będźie ofiárę ſwą Pánu / báránká rocżnego / zupełnego jednego ná ofiárę cáłopáleniá / y owcę jednę rocżną y zdrową ná ofiárę zá grzech / y báráná jednego zupełnego ná ofiárę ſpokojną ;
Przytem koƺ chlebów przáśnych / z mąki pƺennej / plácki nágniátáne z oliwą / y kreple przáśne oliwą námázáne / z ofiárą ich śniedną / y z ofiárą ich mokrą.
Y będźie ofiárowáł kápłán przed Pánem / y ucżyni ofiárę zá grzech jego / y cáłopálenie jego.
Báráná tákże ofiárowáć będźie ná ſpokojną ofiárę Pánu z koƺem chlebów przáśnych ; tákże ofiárowáć będźie kápłán ofiárę jego śniedną y ofiárę jego mokrą.
Y ogoli Názárejcżyk przede drzwiámi námiotu zgromádzeniá głowę Názárejſtwá ſwego / á wźiąwƺy włoſy z głowy Názárejſtwá ſwego / włoży je ná ogień / który jeſt pod ofiárą ſpokojną.
Przytem weźmie kápłán łopátkę wárzoną báránią / y jeden plácek przáśny z koƺá / y jeden krepel niekwáƺony / á dá w ręce Názárejcżykowe po ogoleniu Názárejſtwá jego ;
Y będźie to tám y ſám obrácáł kápłán ná ofiárę obrácániá przed Pánem ; á rzecż tá poświęconá doſtánie śię kápłánowi / ták pierśi obrácániá / jáko y łopátká podnoƺeniá ; á potem będźie mógł Názárejcżyk pić wino.
Toć jeſt práwo Názárejcżyká / któryby ślub ucżynił / y tá ofiárá jego Pánu zá Názárejſtwo jego / okrom tego / coby więcej ucżynić mógł ; według ślubu ſwego / który ucżynił / ták ucżyni według práwá Názárejſtwá ſwego.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do Aároná y do ſynów jego / á rzecż : Ták błogoſłáwić będźiećie ſynom Izráelſkim / mówiąc do nich :
Niech ći błogoſłáwi Pán / á niecháj ćię ſtrzeże ;
Niech rozjáśni Pán oblicże ſwoje nád tobą / á niech ći miłośćiw będźie ;
Niech obróći Pán twárz ſwoję ku tobie / á niecháj ći dá pokój.
Y będą wzywáć imieniá mego nád ſynámi Izráelſkimi / á Já im błogoſłáwić będę.
