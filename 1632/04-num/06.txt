Y rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim / á mów do nich : Mąż álbo Niewiáſtá gdy śię odłączy czyniąc ślub Názárejſtwá / áby byli odłączeni PAnu /
Od winá y mocnego napoju / wſtrzymywáć śię będźie : Octu z winá / y Octu z mocnego napoju pić nie będźie / y wƺyſtkiego / co śię z jágód wytłacza nie będźie pił / tákże jágód winnych / świeżych áni ſuchych / jeść nie będźie.
Po wƺyſtkie dni Názárejſtwá ſwego / ze wƺyſtkiego co wyraſta z máćice winney / od źiarnká áż do łupiny jeść nie będźie.
Po wƺyſtkie dni ślubu Názárejſtwá ſwego / brzytwá nie poſtoji ná głowie jego áż wynidźie czás / do którego śię poświęćił PAnu będźie świętym / á zápuśći włos ná głowie ſwojey.
Po wƺyſtkie dni których śię odłączy PAnu / do umárłego nie wnidźie.
Nád Ojcem ſwym / y nád Mátką ſwą / nád brátem ſwym / y nád śioſtrą ſwą / nie ſplugáwi śię gdyby zmárli : ábowiem poświęcenie Bogá ſwego ma ná głowie ſwojey.
Po wƺyſtkie dni Názárejſtwá ſwego / świętym będźie PAnu.
Y gdyby kto umárł przy nim z prędká <i>á</i> nagle / y ſplugáwiłby głowę poświęcenia jego / ogoli głowę ſwoję / w dźień oczyśćienia ſwego : dniá śiódmego ogoli ją.
A dniá óſmego przynieśie dwie śynogárlice / álbo dwoje gołąbiąt do Kápłaná / ku drzwiom Namiotu zgromádzenia :
Y będźie ofiárował / jedno zá grzech / á drugie ná Ofiárę cáłopalenia / y oczyśći go od tego czym zgrzeƺył nád umárłym / á poświęći głowę jego / dniá onego.
Potym odłączy PAnu dni Názárejſtwá ſwego / ofiárując báránká rocznego / zá wyſtępek / á dni one pierwƺe dáremne będą : gdyż ſplugáwione było Názárejſtwo jego.
A toć <i>jeſt</i> Práwo Názárejczyká / gdy śię wypełnią dni Názárejſtwá jego. Przyjdźie do drzwi Namiotu zgromádzenia /
Y ofiárowáć będźie Ofiárę ſwą PAnu / báránká rocznego / zupełnego / jednego / ná Ofiárę cáłopalenia / y Owcę jednę roczną / zdrową / ná Ofiárę zá grzech / y báráná jednego / zupełnego / ná Ofiárę ſpokojną.
Przytym koƺ chlebów przáſnych / z mąki pƺenney / plácki nágniatáne z Oliwą / y kreple przáſne / Oliwą námázáne / z Ofiárą ich śniedną / y z Ofiárą ich mokrą.
Y będźie ofiárował Kápłan przed PAnem / y uczyni Ofiárę zá grzech jego / y cáłopalenie jego.
Báráná tákże ofiárowáć będźie ná ſpokojną Ofiárę PAnu / z koƺem Chlebów przásnych : tákże ofiárowáć będźie Kápłan Ofiárę jego śniedną y Ofiárę jego mokrą.
Y ogoli Názárejczyk przede drzwiámi Namiotu zgromádzenia / głowę Názárejſtwá ſwego / á wźiąwƺy włoſy z głowy Názárejſtwá ſwego / włoży <i>je</i> ná ogień / który jeſt pod Ofiárą ſpokojną :
Przytym weźmie Kápłan łopátkę wárzoną báránią / y jeden plácek przásny z koƺá / y jeden krepel nie kwáƺony / á da w ręce Názárejczykowe / po ogoleniu Názárejſtwá jego :
Y będźie to tám y ſám obracał Kápłan / ná Ofiárę obracánia przed PAnem / á rzecz tá poświęcona / doſtánie śię Kápłanowi / ták pierśi obracánia / jáko y łopátká podnoƺenia : á potym będźie mógł Názárejczyk pić wino.
Toć <i>jeſt</i> práwo Názárejczyká / któryby ślub uczynił / y tá Ofiárá jego PAnu / zá Názárejſtwo jego / okrom tego / coby więcey uczynić mógł : według ślubu ſwego / który uczynił / ták uczyni według práwá Názárejſtwá ſwego.
Potym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Aároná y do ſynów jego / á rzecz : Ták błogoſłáwić będźiećie Synom Izráelſkim / mówiąc do nich :
Niech ći błogoſłáwi PAN / á niechaj ćię ſtrzeże :
Niech rozjáśni PAN oblicze ſwoje nád tobą / á niech ći miłośćiw będźie.
Niech obróći PAN twarz ſwoję ku tobie / á niechaj ći da pokój.
Y będą wzywáć imienia mego nád Synámi Izráelſkimi / á ja im błogoſłáwić będę.
