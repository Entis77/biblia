Potem gdy mieƺkáł Izráel w Syttim / pocżął lud cudzołożyć z córkámi Moábſkiemi /
Które wzywáły ludu ku ofiárom bogów ſwoich ; á jedząc lud kłániáł śię bogom ich.
Y przyłącżył śię Izráel do ſłużby Báál Fegorá ; ſkąd śię rozgniewáł Pán bárdzo ná Izráelá.
Rzekł tedy Pán do Mojżeƺá : Zbierz wƺyſtkie kśiążętá z ludu / á káż im / te przeſtępce powieƺáć Pánu przed ſłońcem / áby śię odwróćił gniew popędliwośći Páńſkiej od Izráelá.
Przetoż rzekł Mojżeƺ do ſędźiów Izráelſkich : Zábijćie káżdy z wáſ męże ſwe / którzy śię ſpoſpolitowáli z Báál Fegorem.
A oto / niektóry z ſynów Izráelſkich przyƺedł y przywiódł do bráći ſwej Mádyjánitkę przed ocżymá Mojżeƺowemi / y przed ocżymá wƺyſtkiego zgromádzeniá ſynów Izráelſkich ; á oni płákáli przed drzwiámi námiotu zgromádzeniá.
Co gdy ujrzáł Fineeſ / ſyn Eleázárá / ſyná Aároná kápłáná / wſtáwƺy z pośrodku zgromádzeniá / wźiął oƺcżep w ręce ſwoje.
A wƺedłƺy zá onym mężem Izráelſkim do námiotu / przebił oboje / mężá Izráelſkiego / y niewiáſtę przez żywot jej / y odwróconá byłá plágá od ſynów Izráelſkich.
A było tych / co pomárli oną plágą / dwádźieśćiá y cżtery tyśiące.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Fineeſ / ſyn Eleázárá / ſyná Aároná kápłáná / odwróćił gniew mój od ſynów Izráelſkich / będąc wzruƺony zápálcżywą miłośćią ku mnie w pośrodku ich / ták iżem nie wytráćił ſynów Izráelſkich w zápálcżywośći mojej.
Przetoż powiedz mu : Oto / Já ſtánowię z nim przymierze moje / przymierze pokoju ;
Y przyjdźie náń / y ná náśienie jego po nim / przymierze kápłáńſtwá wiecżnego / że śię wzruƺył zápálcżywośćią zá Bogá ſwego / y ocżyśćił ſyny Izráelſkie.
A imię onego mężá Izráelſkiego zábitego / który zábity był z Mádyjánitką / było Zámry / ſyn Sálów / kśiążę domu ojcá ſwego / z pokoleniá Symeonowego.
Imię też niewiáſty zábitej Mádyjánitki było Kozbá / córká Sury / kśiążęćiá w národźie ſwym / w domu ojcżyſtym między Mádyjánity.
Y rzekł Pán do Mojżeƺá / mówiąc :
Stáw śię nieprzyjáćielem Mádyjánitom / á pobijćie je /
Poniewáż y oni ſtáwili śię wám nieprzyjáćiołmi zdrádámi ſwemi / á podeƺli wáſ przez Báál Fegorá / y przez Kozbę / córkę kśiążęćiá Mádyjáńſkiego / śioſtrę ſwą / którá zábitá jeſt w dźień káźni dlá báłwáná Fegor.
