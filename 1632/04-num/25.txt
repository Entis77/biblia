Potem gdy mieƺkał Izráel w Syttim / pocżął lud cudzołożyć z córkámi Moábſkiemi /
Które wzywáły ludu ku ofiárom bogów ſwojich ; á jedząc lud kłániał śię bogom ich.
Y przyłącżył śię Izráel do ſłużby Báál Fegorá ; ſkąd śię rozgniewał Pán bárzo ná Izráelá.
Rzekł tedy Pán do Mojżeƺá : Zbierz wƺyſtkie kśiążętá z ludu / á káż im / te przeſtępce powieƺáć Pánu przed ſłońcem / áby śię odwróćił gniew popędliwośći Páńſkiej od Izráelá.
Przetoż rzekł Mojzeƺ do ſędźiów Izráelſkich : Zábijćie káżdy z was męże ſwe / którzy śię ſpoſpolitowáli z Báál Fegorem.
A oto / niektóry z ſynów Izráelſkich przyƺedł y przywiódł do bráći ſwej Mádyjánitkę przed ocżymá Mojżeƺowemi / y przed ocżymá wƺyſtkiego zgromádzenia ſynów Izráelſkich ; á oni płákáli przed drzwiámi namiotu zgromádzenia.
Co gdy ujrzał Finees / ſyn Eleázárá / ſyná Aároná kápłáná / wſtáwƺy z pośrzodku zgromádzenia / wźiął oƺcżep w ręce ſwoje.
A wƺedƺy zá onym mężem Izráelſkim do namiotu / przebił oboje / mężá Izráelſkiego / y niewiáſtę przez żywot jey / y odwróconá byłá plágá od ſynów Izráelſkich.
A było tych / co pomárli oną plágą / dwádźieśćiá y cżtery tyśiące.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Finees / ſyn Eleázárá / ſyná Aároná kápłáná / odwróćił gniew mój od ſynów Izráelſkich / będąc wzruƺony zápálcżywą miłośćią ku mnie w pośrzodku ich / ták iżem nie wytráćił ſynów Izráelſkich w zápálcżywośći mojey.
Przetoż powiedz mu : Oto / Ja ſtánowię z nim przymierze moje / przymierze pokoju ;
Y przydźie nań / y ná náśienie jego po nim / przymierze kápłáńſtwá wiecżnego / że śię wzruƺył zápálcżywośćią zá Bogá ſwego / y ocżyśćił ſyny Izráelſkie.
A imię onego mężá Izráelſkiego zábitego / który zábity był z Mádyjánitką / było Zámry / ſyn Sálów / kśiążę domu ojcá ſwego / z pokolenia Symeonowego.
Imię też niewiáſty zábitey Mádyjánitki było Kozbá / córká Sury / kśiążęćiá w národźie ſwym / w domu ojcżyſtym miedzy Mádyjánity.
Y rzekł Pán do Mojżeƺá / mówiąc :
Stáw śię nieprzyjáćielem Mádyjánitom / á pobijćie je /
Poniewáż y oni ſtáwili śię wam nieprzyjáćiołmi zdrádámi ſwemi / á podeƺli was przez Báál Fegorá / y przez Kozbę / córkę kśiążęćiá Mádyjáńſkiego / śioſtrę ſwą / która zábitá jeſt w dźień káźni dla báłwáná Fegor.
