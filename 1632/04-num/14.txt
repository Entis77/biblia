Tedy wzruƺywƺy śię wƺyſtko mnóſtwo krzycżeli y płákał lud przez onę noc.
Y ƺemráli przećiwko Mojżeƺowi / y przećiwko Aáronowi wƺyſcy ſynowie Izráelſcy ; y mówiło do nich wƺyſtko mnóſtwo : Obyſmy byli pomárli w źiemi Egipſkiej / álbo ná tey puƺcży!
Obyſmy byli pomárli! Cżemuż wżdy Pán prowádźi nas do tey źiemi / ábyſmy pádli od miecżá? żony náƺe / y dźiátki náƺe áby były ná łup? Izali nam nie lepiej wróćić śię do Egiptu?
Y mówili miedzy ſobą : Poſtánówmy ſobie wodzá / á wróćmy śię do Egiptu.
Tedy upádł Mojzeƺ y Aáron ná oblicże ſwoje przed wƺyſtkim zgromádzeniem ſynów Izráelſkich.
A Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / którzy ƺpiegowáli źiemię / rozdárli ƺáty ſwoje ;
Y rzekli do wƺyſtkiego zgromádzenia ſynów Izráelſkich / mówiąc : Ziemiá / którąśmy przeƺli / y przeƺpiegowáli ją / źiemiá jeſt bárzo dobrá.
Będźieli nam Pán miłośćiw / tedy nas wprowádźi do tey źiemi / á da ją nam / źiemię tę / która opływá mlekiem y miodem.
Jedno Pánu nie bądźćie odpornymi / áni śię wy bójćie ludu oney źiemi / bo jáko chleb pojeść je możemy ; odſtąpiłá obroná ich od nich / ále Pán jeſt z námi ; nie bójćież śię ich.
Y mówiło wƺyſtko zgromádzenie / áby je ukámionowano ; ále chwałá Páńſka okázáłá śię nád námiotem zgromádzenia wƺyſtkim ſynom Izráelſkim.
Y rzekł Pán do Mojżeƺá : Dokądże mię dráźnić będźie ten lud? Y dokądże mi wierzyć nie będą dla tych wƺyſtkich znáków / którem cżynił miedzy nimi?
Poráżę je morem / y rozproƺę je ; á ćiebie ucżynię w naród wielki y możniejƺy / niż ten jeſt.
Ale rzekł Mojzeƺ do Páná : Oto / uſłyƺą Egipcżánie / z których pośrzodku wywiodłeś mocą ſwoją ten lud ;
Y mówić będą z obywátelmi źiemi tey / bo ſłyƺeli / żeś ty Pánie był w pośrzodku ludu tego ; żeś okiem w oko widźiány był / o Pánie / á obłok twój ſtáwał nád nimi / á iż w ſłupie obłokowym chodźiłeś przed nimi we dnie / á w ſłupie ogniſtym w nocy.
Gdybyś tedy pobił lud ten wƺyſtek áż do jednego / rzekliby pogánie / którzy o twej ſłáwie ſłycháli / mówiąc :
Iż nie mógł wprowádźić Pán ludu tego do źiemi / o którą im przyśiągł : przeto je pobił ná puƺcży.
A ták teraz niech proƺę uwielbioná będźie moc Páńſka / jákoś rzekł / mówiąc :
Pán nie rychły ku gniewowi á wielki w miłośierdźiu / znoƺąc niepráwość y przeſtępſtwo / który winnego nie cżyni niewinnym / kárząc niepráwość ojców w ſynách do trzećiego y do cżwártego pokolenia ;
Odpuść proƺę niepráwość ludu tego według wielkośći miłośierdźiá twego / ták jákoś odpuƺcżał ludowi temu z Egiptu áż dotąd.
Tedy rzekł Pán : Odpuśćiłem według ſłowá twego.
A wƺákże / jáko Ja żyję / y nápełnioná jeſt chwáłą Páńſką wƺyſtká źiemiá :
Ták wƺyſcy / którzy widźieli chwałę moję / y znáki moje / którem cżynił w Egipćie / y ná puƺcży / á kuśili mię już po dźieśięć kroć / áni byli poſłuƺni głoſowi memu /
Nie oglądáją źiemi tey / o którąm przyśiągł ojcom ich / á żáden z tych / którzy mię dráźnili / nie oglądáją jey.
Ale ſługę mego Kálebá / gdyż był w nim duch inákƺy / y trwał ſtátecżnie przy mnie / wprowádzę do źiemi / do którey chodźił / á naśienie jego odźiedźicży ją.
Ale ponieważ Amálekitá y Chánánejcżyk mieƺkáją w dolinie / przetoż jutro obróććie śię / á idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
Nád to rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Y dokądże znośić mam ten zły lud / który ƺemrze przećiwko mnie? długoż ƺemránia ſynów Izráelſkich / którzy ƺemrzą przećiwko mnie / ſłucháć będę?
Mów do nich : Żyję Ja / mówi Pán / że jákośćie mówili w uƺy moje / ták ucżynię wam.
Ná tey puƺcży polegą ćiáłá wáƺe / y wƺyſcy policżeni wáśi według wƺyſtkiej licżby wáƺej od dwudźieſtego roku y wyżey / którzyśćie ƺemráli przećiwko mnie.
A wy nie wnijdźiećie do źiemi tey / o którąm podnióſł rękę moję / ábym ją wam dał ná mieƺkánie / okrom Kálebá / ſyná Jefunowego / y Jozuego / ſyná Nunowego ;
A dźiátki wáƺe / o którycheśćie mówili / że będą ná łup / te wprowádzę / y oglądáją tę źiemię / którąśćie wy wzgárdźili.
Ale trupy wáƺe / wáƺe trupy mówię / polegą ná tey puƺcży ;
A ſynowie wáśi będą śię tułáli po tey puƺcży przez cżterdźieśći lat / y ponioſą káránie zá cudzołóſtwá wáƺe / áż wyginą trupy wáƺe ná puƺcży.
Według licżby dni / w którycheśćie ƺpiegowáli źiemię / to jeſt cżterdźieśći dni / dźień káżdy zá rok licżąc / ponieśiećie niepráwośći wáƺe cżterdźieśći lat / y poznáćie pomſtę ſwego odſtąpieniá ode mnie.
Ja Pán mówiłem / że to ucżynię temu wƺyſtkiemu zgromádzeniu złemu / które śię ſpiknęło przećiwko mnie ; ná tey puƺcży poginą / y tu pomrą.
Oni tedy mężowie / których ſłał Mojzeƺ ná przeƺpiegowánie źiemi / którzy wróćiwƺy śię pobudźili do ƺemránia przećiwko niemu wƺyſtek lud / puƺcżájąc złą ſłáwę o źiemi oney ;
Pomárli mężowie oni / którzy puƺcżáli ſłáwę złą o źiemi / ſrogą plágą przed Pánem.
Ale Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / zoſtáli żywi z mężów onych / którzy chodźili ku przeƺpiegowániu źiemi.
Y opowiedźiał Mojzeƺ te ſłowá wƺyſtkim ſynom Izráelſkim / y płákał lud bárzo.
Tedy ráno wſtáwƺy wſtąpili ná wierzch góry / mówiąc : Oto my pójdźiemy ná to miejſce / o którem nam Pán powiedźiał ; bośmy zgrzeƺyli.
Ale im powiedźiał Mojzeƺ : Przecżże wy przeſtępujećie ſłowo Páńſkie? to śię wam nie nádá.
Nie chodźćie ; bo niemáƺ Páná miedzy wámi / ábyśćie nie byli pobići od nieprzyjáćiół wáƺych.
Bo Amálekitá y Chánánejcżyk tuż przed wámi ſą / y polężećie od miecżá ; bo dla tego / żeśćie śię odwróćili od Páná / nie będźie Pán z wámi.
A oni przećię kuśili śię wnijść ná wierzch góry ; lecż ſkrzyniá przymierza Páńſkiego y Mojzeƺ nie odchodźili od obozu.
Tedy zſtąpił Amálekitá y Chánánejcżyk / mieƺkájący ná oney górze / á poráźili je / y gonili je áż do Hormy.
