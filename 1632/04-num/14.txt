Tedy wzruƺywƺy śię wƺyſtko mnóſtwo krzycżeli y płákáł lud przez onę noc.
Y ƺemráli przećiwko Mojżeƺowi / y przećiwko Aáronowi wƺyſcy ſynowie Izráelſcy ; y mówiło do nich wƺyſtko mnóſtwo : Obyśmy byli pomárli w źiemi Egipſkiej / álbo ná tey puƺcży!
Obyśmy byli pomárli! Cżemuż wżdy Pán prowádźi nas do tey źiemi / ábyśmy pádli od miecżá? żony náƺe / y dźiátki náƺe áby były ná łup? Izáli nám nie lepiej wróćić śię do Egiptu?
Y mówili między ſobą : Poſtánówmy ſobie wodzá / á wróćmy śię do Egiptu.
Tedy upádł Mojżeƺ y Aáron ná oblicże ſwoje przed wƺyſtkiem zgromádzeniem ſynów Izráelſkich.
A Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / którzy ƺpiegowáli źiemię / rozdárli ƺáty ſwoje ;
Y rzekli do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / mówiąc : Ziemiá / którąśmy przeƺli / y przeƺpiegowáli ją / źiemiá jeſt bárdzo dobrá.
Będźieli nám Pán miłośćiw / tedy nas wprowádźi do tey źiemi / á dá ją nám / źiemię tę / która opływá mlekiem y miodem.
Jedno Pánu nie bądźćie odpornymi / áni śię wy bójćie ludu oney źiemi / bo jáko chleb pojeść je możemy ; odſtąpiłá obroná ich od nich / ále Pán jeſt z námi ; nie bójćież śię ich.
Y mówiło wƺyſtko zgromádzenie / áby je ukámionowáno ; ále chwałá Páńſká okázáłá śię nád námiotem zgromádzeniá wƺyſtkim ſynom Izráelſkim.
Y rzekł Pán do Mojżeƺá : Dokądże mię dráźnić będźie ten lud? Y dokądże mi wierzyć nie będą dla tych wƺyſtkich znáków / którem cżynił między nimi?
Poráżę je morem / y rozproƺę je ; á ćiebie ucżynię w náród wielki y możniejƺy / niż ten jeſt.
Ale rzekł Mojżeƺ do Páná : Oto / uſłyƺą Egipcżánie / z których pośrodku wywiodłeś mocą ſwoją ten lud ;
Y mówić będą z obywátelmi źiemi tey / bo ſłyƺeli / żeś ty Pánie był w pośrodku ludu tego ; żeś okiem w oko widźiány był / o Pánie / á obłok twój ſtáwáł nád nimi / á iż w ſłupie obłokowym chodźiłeś przed nimi we dnie / á w ſłupie ogniſtym w nocy.
Gdybyś tedy pobił lud ten wƺyſtek áż do jednego / rzekliby pogánie / którzy o twej ſłáwie ſłycháli / mówiąc :
Iż nie mógł wprowádźić Pán ludu tego do źiemi / o którą im przyśiągł : przeto je pobił ná puƺcży.
A ták teraz niech proƺę uwielbioná będźie moc Páńſká / jákoś rzekł / mówiąc :
Pán nie rychły ku gniewowi á wielki w miłośierdźiu / znoƺąc niepráwość y przeſtępſtwo / który winnego nie cżyni niewinnym / kárząc niepráwość ojców w ſynách do trzećiego y do cżwártego pokoleniá ;
Odpuść proƺę niepráwość ludu tego według wielkośći miłośierdźiá twego / ták jákoś odpuƺcżáł ludowi temu z Egiptu áż dotąd.
Tedy rzekł Pán : Odpuśćiłem według ſłowá twego.
A wƺákże / jáko Ja żyję / y nápełnioná jeſt chwáłą Páńſką wƺyſtká źiemiá :
Ták wƺyſcy / którzy widźieli chwáłę moję / y znáki moje / którem cżynił w Egipćie / y ná puƺcży / á kuśili mię już po dźieśięć kroć / áni byli poſłuƺni głoſowi memu /
Nie oglądáją źiemi tey / o którąm przyśiągł ojcom ich / á żáden z tych / którzy mię dráźnili / nie oglądáją jey.
Ale ſługę mego Kálebá / gdyż był w nim duch inákƺy / y trwáł ſtátecżnie przy mnie / wprowádzę do źiemi / do którey chodźił / á náśienie jego odźiedźicży ją.
Ale poniewáż Amálekitá y Chánánejcżyk mieƺkáją w dolinie / przetoż jutro obróććie śię / á idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
Nád to rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Y dokądże znośić mam ten zły lud / który ƺemrze przećiwko mnie? długoż ƺemrániá ſynów Izráelſkich / którzy ƺemrzą przećiwko mnie / ſłucháć będę?
Mów do nich : Żyję Ja / mówi Pán / że jákośćie mówili w uƺy moje / ták ucżynię wám.
Ná tey puƺcży polegą ćiáłá wáƺe / y wƺyſcy policżeni wáśi według wƺyſtkiej licżby wáƺej od dwudźieſtego roku y wyżej / którzyśćie ƺemráli przećiwko mnie.
A wy nie wnijdźiećie do źiemi tey / o którąm podnióſł rękę moję / ábym ją wám dáł ná mieƺkánie / okrom Kálebá / ſyná Jefunowego / y Jozuego / ſyná Nunowego ;
A dźiátki wáƺe / o którycheśćie mówili / że będą ná łup / te wprowádzę / y oglądáją tę źiemię / którąśćie wy wzgárdźili.
Ale trupy wáƺe / wáƺe trupy mówię / polegą ná tey puƺcży ;
A ſynowie wáśi będą śię tułáli po tey puƺcży przez cżterdźieśći lát / y ponioſą káránie zá cudzołóſtwá wáƺe / áż wyginą trupy wáƺe ná puƺcży.
Według licżby dni / w którycheśćie ƺpiegowáli źiemię / to jeſt cżterdźieśći dni / dźień káżdy zá rok licżąc / ponieśiećie niepráwośći wáƺe cżterdźieśći lát / y poznáćie pomſtę ſwego odſtąpieniá ode mnie.
Ja Pán mówiłem / że to ucżynię temu wƺyſtkiemu zgromádzeniu złemu / które śię ſpiknęło przećiwko mnie ; ná tey puƺcży poginą / y tu pomrą.
Oni tedy mężowie / których ſłáł Mojżeƺ ná przeƺpiegowánie źiemi / którzy wróćiwƺy śię pobudźili do ƺemrániá przećiwko niemu wƺyſtek lud / puƺcżájąc złą ſłáwę o źiemi oney ;
Pomárli mężowie oni / którzy puƺcżáli ſłáwę złą o źiemi / ſrogą plágą przed Pánem.
Ale Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / zoſtáli żywi z mężów onych / którzy chodźili ku przeƺpiegowániu źiemi.
Y opowiedźiáł Mojżeƺ te ſłowá wƺyſtkim ſynom Izráelſkim / y płákáł lud bárdzo.
Tedy ráno wſtáwƺy wſtąpili ná wierzch góry / mówiąc : Oto my pójdźiemy ná to miejſce / o którem nám Pán powiedźiáł ; bośmy zgrzeƺyli.
Ale im powiedźiáł Mojżeƺ : Przecżże wy przeſtępujećie ſłowo Páńſkie? to śię wám nie nádá.
Nie chodźćie ; bo niemáƺ Páná między wámi / ábyśćie nie byli pobići od nieprzyjáćiół wáƺych.
Bo Amálekitá y Chánánejcżyk tuż przed wámi ſą / y polężećie od miecżá ; bo dla tego / żeśćie śię odwróćili od Páná / nie będźie Pán z wámi.
A oni przećię kuśili śię wnijść ná wierzch góry ; lecż ſkrzyniá przymierzá Páńſkiego y Mojżeƺ nie odchodźili od obozu.
Tedy zſtąpił Amálekitá y Chánánejcżyk / mieƺkájący ná oney górze / á poráźili je / y gonili je áż do Hormy.
