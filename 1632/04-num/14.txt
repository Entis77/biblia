Tedy wzruƺywƺy śię wƺyſtko mnóſtwo krzycżeli y płákáł lud przez onę noc.
Y ƺemráli przećiwko Mojżeƺowi / y przećiwko Aáronowi wƺyſcy ſynowie Izráelſcy ; y mówiło do nich wƺyſtko mnóſtwo : Obyśmy byli pomárli w ziemi Egipſkiej / álbo ná tej puƺcży!
Obyśmy byli pomárli! Cżemuż wżdy Pán prowádzi náſ do tej ziemi / ábyśmy pádli od miecżá? żony náƺe / y dziátki náƺe áby były ná łup? Izáli nám nie lepiej wróćić śię do Egiptu?
Y mówili między ſobą : Poſtánówmy ſobie wodzá / á wróćmy śię do Egiptu.
Tedy upádł Mojżeƺ y Aáron ná oblicże ſwoje przed wƺyſtkiem zgromádzeniem ſynów Izráelſkich.
A Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / którzy ƺpiegowáli ziemię / rozdárli ƺáty ſwoje ;
Y rzekli do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / mówiąc : Ziemiá / którąśmy przeƺli / y przeƺpiegowáli ją / ziemiá jeſt bárdzo dobrá.
Będzieli nám Pán miłośćiw / tedy náſ wprowádzi do tej ziemi / á dá ją nám / ziemię tę / którá opływá mlekiem y miodem.
Jedno Pánu nie bądźćie odpornymi / áni śię wy bójćie ludu onej ziemi / bo jáko chleb pojeść je możemy ; odſtąpiłá obroná ich od nich / ále Pán jeſt z námi ; nie bójćież śię ich.
Y mówiło wƺyſtko zgromádzenie / áby je ukámionowáno ; ále chwáłá Páńſká okázáłá śię nád námiotem zgromádzeniá wƺyſtkim ſynom Izráelſkim.
Y rzekł Pán do Mojżeƺá : Dokądże mię dráźnić będzie ten lud? Y dokądże mi wierzyć nie będą dlá tych wƺyſtkich znáków / którem cżynił między nimi?
Poráżę je morem / y rozproƺę je ; á ćiebie ucżynię w náród wielki y możniejƺy / niż ten jeſt.
Ale rzekł Mojżeƺ do Páná : Oto / uſłyƺą Egipcżánie / z których pośrodku wywiodłeś mocą ſwoją ten lud ;
Y mówić będą z obywátelmi ziemi tej / bo ſłyƺeli / żeś ty Pánie był w pośrodku ludu tego ; żeś okiem w oko widziány był / o Pánie / á obłok twój ſtáwáł nád nimi / á iż w ſłupie obłokowym chodziłeś przed nimi we dnie / á w ſłupie ogniſtym w nocy.
Gdybyś tedy pobił lud ten wƺyſtek áż do jednego / rzekliby pogánie / którzy o twej ſłáwie ſłycháli / mówiąc :
Iż nie mógł wprowádzić Pán ludu tego do ziemi / o którą im przyśiągł : przeto je pobił ná puƺcży.
A ták teráz niech proƺę uwielbioná będzie moc Páńſká / jákoś rzekł / mówiąc :
Pán nie rychły ku gniewowi á wielki w miłośierdziu / znoƺąc niepráwość y przeſtępſtwo / który winnego nie cżyni niewinnym / kárząc niepráwość ojców w ſynách do trzećiego y do cżwártego pokoleniá ;
Odpuść proƺę niepráwość ludu tego według wielkośći miłośierdziá twego / ták jákoś odpuƺcżáł ludowi temu z Egiptu áż dotąd.
Tedy rzekł Pán : Odpuśćiłem według ſłowá twego.
A wƺákże / jáko Já żyję / y nápełnioná jeſt chwáłą Páńſką wƺyſtká ziemiá :
Ták wƺyſcy / którzy widzieli chwáłę moję / y znáki moje / którem cżynił w Egipćie / y ná puƺcży / á kuśili mię już po dzieśięć kroć / áni byli poſłuƺni głoſowi memu /
Nie oglądáją ziemi tej / o którąm przyśiągł ojcom ich / á żáden z tych / którzy mię dráźnili / nie oglądáją jej.
Ale ſługę mego Kálebá / gdyż był w nim duch inákƺy / y trwáł ſtátecżnie przy mnie / wprowádzę do ziemi / do której chodził / á náśienie jego odziedzicży ją.
Ale poniewáż Amálekitá y Chánánejcżyk mieƺkáją w dolinie / przetoż jutro obróććie śię / á idźćie ná puƺcżą / drogą ku morzu cżerwonemu.
Nád to rzekł Pán do Mojżeƺá y do Aároná / mówiąc :
Y dokądże znośić mám ten zły lud / który ƺemrze przećiwko mnie? długoż ƺemrániá ſynów Izráelſkich / którzy ƺemrzą przećiwko mnie / ſłucháć będę?
Mów do nich : Żyję Já / mówi Pán / że jákośćie mówili w uƺy moje / ták ucżynię wám.
Ná tej puƺcży polegą ćiáłá wáƺe / y wƺyſcy policżeni wáśi według wƺyſtkiej licżby wáƺej od dwudzieſtego roku y wyżej / którzyśćie ƺemráli przećiwko mnie.
A wy nie wnijdziećie do ziemi tej / o którąm podnióſł rękę moję / ábym ją wám dáł ná mieƺkánie / okrom Kálebá / ſyná Jefunowego / y Jozuego / ſyná Nunowego ;
A dziátki wáƺe / o którycheśćie mówili / że będą ná łup / te wprowádzę / y oglądáją tę ziemię / którąśćie wy wzgárdzili.
Ale trupy wáƺe / wáƺe trupy mówię / polegą ná tej puƺcży ;
A ſynowie wáśi będą śię tułáli po tej puƺcży przez cżterdzieśći lát / y ponioſą káránie zá cudzołóſtwá wáƺe / áż wyginą trupy wáƺe ná puƺcży.
Według licżby dni / w którycheśćie ƺpiegowáli ziemię / to jeſt cżterdzieśći dni / dzień káżdy zá rok licżąc / ponieśiećie niepráwośći wáƺe cżterdzieśći lát / y poznáćie pomſtę ſwego odſtąpieniá ode mnie.
Já Pán mówiłem / że to ucżynię temu wƺyſtkiemu zgromádzeniu złemu / które śię ſpiknęło przećiwko mnie ; ná tej puƺcży poginą / y tu pomrą.
Oni tedy mężowie / których ſłáł Mojżeƺ ná przeƺpiegowánie ziemi / którzy wróćiwƺy śię pobudzili do ƺemrániá przećiwko niemu wƺyſtek lud / puƺcżájąc złą ſłáwę o ziemi onej ;
Pomárli mężowie oni / którzy puƺcżáli ſłáwę złą o ziemi / ſrogą plágą przed Pánem.
Ale Jozue / ſyn Nunów / y Káleb / ſyn Jefunów / zoſtáli żywi z mężów onych / którzy chodzili ku przeƺpiegowániu ziemi.
Y opowiedziáł Mojżeƺ te ſłowá wƺyſtkim ſynom Izráelſkim / y płákáł lud bárdzo.
Tedy ráno wſtáwƺy wſtąpili ná wierzch góry / mówiąc : Oto my pójdziemy ná to miejſce / o którem nám Pán powiedziáł ; bośmy zgrzeƺyli.
Ale im powiedziáł Mojżeƺ : Przecżże wy przeſtępujećie ſłowo Páńſkie? to śię wám nie nádá.
Nie chodźćie ; bo niemáƺ Páná między wámi / ábyśćie nie byli pobići od nieprzyjáćiół wáƺych.
Bo Amálekitá y Chánánejcżyk tuż przed wámi ſą / y polężećie od miecżá ; bo dlá tego / żeśćie śię odwróćili od Páná / nie będzie Pán z wámi.
A oni przećię kuśili śię wnijść ná wierzch góry ; lecż ſkrzyniá przymierzá Páńſkiego y Mojżeƺ nie odchodzili od obozu.
Tedy zſtąpił Amálekitá y Chánánejcżyk / mieƺkájący ná onej górze / á porázili je / y gonili je áż do Hormy.
