Y ſtáło śię / że śię lud uſkárżáł nieſłuƺnie / co śię nie podobáło Pánu. Przetoż uſłyƺáwƺy to Pán / bárdzo śię rozgniewáł / y zápálił śię przećiwko nim ogień Páńſki / y popálił oſtátnią cżęść obozu.
Tedy wołáł lud ná Mojżeƺá ; y modlił śię Mojżeƺ Pánu / y zgáſł ogień.
Y názwáł imię miejſcá onego Táberá : bo śię zápálił przećiwko nim ogień Páńſki.
A lud poſpolity / który był między nimi / chćiwośćią wielką zjęty odwrácáł śię ; y płákáli też ſynowie Izráelſcy / mówiąc : Któż náſ nákármi mięſem?
Wſpominámy ſobie ná ryby / któreśmy jádáli w Egipćie dármo / ná ogórki / y ná melony / y ná łucżek / y ná cebulę / y ná cżoſnek.
A teráz duƺá náƺá wywiędłá nic inƺego nie májąc / oprócż tej mánny / przed ocżymá ſwemi.
A mánná byłá jáko náśienie koryjándrowe / á bárwá tej jáko bárwá Bdelijowá.
Y wychodził lud / á zbieráli ją / y mełli w żárnách / álbo tłukli w moździerzách á wárzyli w kotłách y cżynili z niej podpłomyki ; á był ſmák jej jáko ſmák świeżej oliwy.
Gdy bowiem pádáłá roſá ná obóz w nocy / pádáłá też mánná náń.
Tedy uſłyƺáł Mojżeƺ / że lud płákáł po domách ſwych / káżdy u drzwi námiotu ſwego ; dlá cżego zápáliłá śię popędliwość Páńſká wielce / y nie podobáło śię też to Mojżeƺowi.
Y rzekł Mojżeƺ do Páná : Cżemużeś ták źle ucżynił ſłudze twemu? cżemużem nie ználázł łáſki w ocżách twoich / żeś włożył ćiężár tego wƺyſtkiego ludu ná mię?
Izálim já pocżął ten wƺyſtek lud? izálim go já zrodził / iż mi mówiƺ : Nieś je ná łonie twojim / jáko piáſtun nośi niemowlątko / do ziemi / o którąś przyśiągł ojcom ich?
Gdzież mám mięſo / ábym dáł wƺyſtkiemu temu ludowi? bo płácżą ná mię / mówiąc : Dáj nám mięſá / ábyśmy jedli.
Nie mogę já ſám znieść wƺyſtkiego ludu tego ; bo to nád możność moję.
A jeźli śię ták ze mną obchodzić chceƺ / proƺę rácżej mię zábij / jeźlim ználázł łáſkę w ocżách twoich / ábym nie pátrzył ná ſwoje złe.
Y rzekł Pán do Mojżeƺá : Zbierz mi śiedmdzieśiąt mężów z ſtárƺych Izráelſkich / które znáƺ / że ſą ſtárƺymi ludu / y kśiążętá jego / á przywiedź je przed námiot zgromádzeniá / y ſtáć tám będę z tobą ;
A Já zſtąpię / y będę tám mówił z tobą / y wziąwƺy z Duchá / który jeſt w tobie / udzielę im ; y ponioſą z tobą brzemię ludu / á nie ponieśieƺ go ty ſám.
A do ludu rzecżeƺ : Poświęććie śię ná jutro / á będziećie jeść mięſo ; bośćie płákáli w uƺách Páńſkich / mówiąc : Któż náſ nákármi mięſem? bo nám lepiej było w Egipćie ; y dá wám Pán mięſá / y będziećie jedli.
Nie przez jeden dzień jeść będziećie / áni przez dwá dni / áni przez pięć / áni przez dzieśięć dni / áni przez dwádzieśćiá dni :
Ale przez cáły mieśiąc / áż polezie przez nozdrzá wáƺe / á zbrzydzi śię wám / przeto żeśćie pogárdzili Pánem / który jeſt między wámi / á płákáliśćie przed nim mówiąc : Ná cóżeśmy wyƺli z Egiptu?
Y rzekł Mojżeƺ : Sześć kroć ſto tyśięcy pieƺych jeſt ludu / między którym já mieƺkám / á tyś powiedziáł : Dám im mięſá / że będą jeść cáły mieśiąc.
Izáli im owiec y wołów nábiją / áby śię im doſtáło? Izáli wƺyſtkie ryby morſkie zbiorą im / áby doſtátek mieli?
Y rzekł Pán do Mojżeƺá : Azáż ręká Páńſká jeſt ſkurcżoná? teráz ujrzyƺ / jeśli śię wypełni ſłowo moje / cżyli nie.
Tedy wyƺedł Mojżeƺ / y opowiedziáł ludowi ſłowá Páńſkie ; á zebráwƺy śiedmdzieśiąt mężów ſtárƺych z ludu / poſtáwił je około námiotu.
Y zſtąpił Pán w obłoku / y mówił do niego / á wziąwƺy z Duchá / który był w nim / podzielił go między śiedmdzieśiąt mężów ſtárƺych ; y ſtáło śię / gdy odpocżął nád nimi on Duch / tedy prorokowáli / á potem nigdy.
Ale zoſtáli byli dwá mężowie w obozie / imię jednego Eldád / á imię drugiego Medád ; ná których też odpocżął on Duch / bo oni byli nápiſáni / choć byli nie przyƺli do námiotu ; á ták ći prorokowáli w obozie.
Tedy przybieżáło pácholę / y oznájmiło Mojżeƺowi / mówiąc : Eldád y Medád prorokują w obozie.
Ale odpowiedziáwƺy Jozue / ſyn Nunów / ſługá Mojżeƺów / jeden z młodzieńców jego / rzekł : Pánie mój Mojżeƺu / zábroń im.
Któremu odpowiedziáł Mojżeƺ : Cżemuż im ty zázdrośćiƺ dlá mnie? Boże dáj / áby wƺyſtek lud Páńſki prorokowáł / á iżby dáł Pán Duchá ſwego ná nie!
Wróćił śię tedy Mojżeƺ do obozu / on y ſtárśi Izráelſcy.
Zátem wyƺedł wiátr od Páná / y porwáwƺy przepiórki od morzá / ſpuśćił je ná obóz / z jednej ſtrony jáko ná jeden dzień chodu / z drugiej ſtrony tákże jáko ná jeden dzień chodu / około obozu / á jákoby ná dwá łokćie były nád ziemią.
Tedy wſtáwƺy lud przez cáły on dzień y przez cáłą noc / y názájutrz przez wƺyſtek dzień zbieráli one przepiórki / á kto nájmniej názbieráł / miáł dzieśięć homerów : y náwieƺáli ich wƺędy ſobie około obozu.
Mięſo jeƺcże było między zębámi ich nie zeżwáne / gdy gniew Páńſki zápálił śię ná lud / y pobił Pán lud on plágą bárdzo wielką.
Y názwáne jeſt imię miejſcá onego Kibrot Hátááwá ; ábowiem tám pogrzebli lud / który pożądáł mięſá.
A z Kibrot Hátááwy ruƺył śię lud do Háſerotu / y mieƺkáli w Háſeroćie.
