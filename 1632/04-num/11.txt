Y ſtáło śię / że śię lud uſkárżał nieſłuƺnie / co śię nie podobáło Pánu. Przetoż uſłyƺáwƺy to Pán / bárzo śię rozgniewał / y zápálił śię przećiwko nim ogień Páńſki / y popálił oſtátnią cżęść obozu.
Tedy wołał lud ná Mojżeƺá ; y modlił śię Mojzeƺ Pánu / y zgáſł ogień.
Y názwał imię miejſcá onego Táberá : bo śię zápálił przećiwko nim ogień Páńſki.
A lud poſpolity / który był miedzy nimi / chćiwośćią wielką zjęty odwrácał śię ; y płákáli też ſynowie Izráelſcy / mówiąc : Któż nas nákármi mięſem?
Wſpominámy ſobie ná ryby / któreśmy jádáli w Egipćie dármo / ná ogórki / y ná melony / y ná łucżek / y ná cebulę / y ná cżoſnek.
A teraz duƺá náƺá wywiędłá nic inƺego nie májąc / oprócż tey mánny / przed ocżymá ſwemi.
A mánná byłá jáko náśienie koryjándrowe / á bárwá tey jáko bárwá Bdelijowá.
Y wychodźił lud / á zbieráli ją / y mełli w żárnách / álbo tłukli w moźdźierzách á wárzyli w kotłách y cżynili z niej podpłomyki ; á był ſmák jey jáko ſmák świeżey oliwy.
Gdy bowiem pádáłá roſá ná obóz w nocy / pádáłá też mánná nań.
Tedy uſłyƺał Mojzeƺ / że lud płákał po domách ſwych / káżdy u drzwi namiotu ſwego ; dla cżego zápáliłá śię popędliwość Páńſká wielce / y nie podobáło śię też to Mojżeƺowi.
Y rzekł Mojzeƺ do Páná : Cżemużeś ták źle ucżynił ſłudze twemu? cżemużem nie ználázł łáſki w ocżách twojich / żeś włożył ćiężár tego wƺyſtkiego ludu ná mię?
Izálim ja pocżął ten wƺyſtek lud? izálim go ja zrodźił / iż mi mówiƺ : Nieś je ná łonie twojim / jáko piáſtun nośi niemowlątko / do źiemi / o którąś przyśiągł ojcom ich?
Gdźież mam mięſo / ábym dał wƺyſtkiemu temu ludowi? bo płácżą ná mię / mówiąc : Dáj nam mięſá / ábyſmy jedli.
Nie mogę ja ſám znieść wƺyſtkiego ludu tego ; bo to nád możność moję.
A jeſli śię ták ze mną obchodźić chceƺ / proƺę rácżey mię zábij / jeſlim ználázł łáſkę w ocżách twojich / ábym nie pátrzył ná ſwoje złe.
Y rzekł Pán do Mojżeƺá : Zbierz mi śiedmdźieśiąt mężów z ſtárƺych Izráelſkich / które znáƺ / że ſą ſtárƺymi ludu / y kśiążętá jego / á przywiedź je przed námiot zgromádzenia / y ſtáć tám będę z tobą ;
A Ja zſtąpię / y będę tám mówił z tobą / y wźiąwƺy z Duchá / który jeſt w tobie / udźielę im ; y ponioſą z tobą brzemię ludu / á nie ponieśieƺ go ty ſám.
A do ludu rzecżeƺ : Poświęććie śię ná jutro / á będźiećie jeść mięſo ; bośćie płákáli w uƺách Páńſkich / mówiąc : Któż nas nákármi mięſem? bo nam lepiej było w Egipćie ; y da wam Pán mięſá / y będźiećie jedli.
Nie przez jeden dźień jeść będźiećie / áni przez dwá dni / áni przez pięć / áni przez dźieśięć dni / áni przez dwádźieśćiá dni :
Ale przez cáły mieśiąc / áż poleźie przez nozdrzá wáƺe / á zbrzydźi śię wam / przeto żeśćie pogárdźili Pánem / który jeſt miedzy wámi / á płákáliśćie przed nim mówiąc : Ná cóżeśmy wyƺli z Egiptu?
Y rzekł Mojzeƺ : Sześć kroć ſto tyśięcy pieƺych jeſt ludu / miedzy którym ja mieƺkám / á tyś powiedźiał : Dam im mięſá / że będą jeść cáły mieśiąc.
Izali im owiec y wołów nábiją / áby śię im doſtáło? Izali wƺyſtkie ryby morſkie zbiorą im / áby doſtátek mieli?
Y rzekł Pán do Mojżeƺá : Azaż ręká Páńſká jeſt ſkurcżoná? teraz ujrzyƺ / jeśli śię wypełni ſłowo moje / cżyli nie.
Tedy wyƺedł Mojzeƺ / y opowiedźiał ludowi ſłowá Páńſkie ; á zebráwƺy śiedmdźieśiąt mężów ſtárƺych z ludu / poſtáwił je około namiotu.
Y zſtąpił Pán w obłoku / y mówił do niego / á wźiąwƺy z Duchá / który był w nim / podźielił go miedzy śiedmdźieśiąt mężów ſtárƺych ; y ſtáło śię / gdy odpocżął nád nimi on Duch / tedy prorokowáli / á potem nigdy.
Ale zoſtáli byli dwá mężowie w oboźie / imię jednego Eldád / á imię drugiego Medád ; ná których też odpocżął on Duch / bo oni byli nápiſáni / choć byli nie przyƺli do namiotu ; á ták ći prorokowáli w oboźie.
Tedy przybieżáło pácholę / y oznájmiło Mojżeƺowi / mówiąc : Eldád y Medád prorokują w oboźie.
Ale odpowiedźiáwƺy Jozue / ſyn Nunów / ſługá Mojżeƺów / jeden z młodźieńców jego / rzekł : Pánie mój Mojżeƺu / zábroń im.
Któremu odpowiedźiał Mojzeƺ : Cżemuż im ty zázdrośćiƺ dla mnie? Boże dáj / áby wƺyſtek lud Páńſki prorokował / á iżby dał Pán Duchá ſwego ná nie!
Wróćił śię tedy Mojzeƺ do obozu / on y ſtárśi Izráelſcy.
Zátem wyƺedł wiátr od Páná / y porwáwƺy przepiórki od morzá / ſpuśćił je ná obóz / z jedney ſtrony jáko ná jeden dźień chodu / z drugiej ſtrony tákże jáko ná jeden dźień chodu / około obozu / á jákoby ná dwá łokćie były nád źiemią.
Tedy wſtáwƺy lud przez cáły on dźień y przez cáłą noc / y názájutrz przez wƺyſtek dźień zbieráli one przepiórki / á kto nájmniej názbierał / miał dźieśięć homerów : y náwieƺáli ich wƺędy ſobie około obozu.
Mięſo jeƺcże było miedzy zębámi ich nie zeżwáne / gdy gniew Páńſki zápálił śię ná lud / y pobił Pán lud on plágą bárzo wielką.
Y názwáne jeſt imię miejſcá onego Kibrot Hátááwá ; ábowiem tám pogrzebli lud / który pożądał mięſá.
A z Kibrot Hátááwy ruƺył śię lud do Háſerotu / y mieƺkáli w Háſeroćie.
