Y ſtáło śię / że śię lud uſkárżáł nieſłuƺnie / co śię nie podobáło Pánu. Przetoż uſłyƺáwƺy to Pán / bárdzo śię rozgniewáł / y zápálił śię przećiwko nim ogień Páńſki / y popálił oſtátnią cżęść obozu.
Tedy wołáł lud ná Mojżeƺá ; y modlił śię Mojżeƺ Pánu / y zgáſł ogień.
Y názwáł imię miejſcá onego Táberá : bo śię zápálił przećiwko nim ogień Páńſki.
A lud poſpolity / który był między nimi / chćiwośćią wielką zjęty odwrácáł śię ; y płákáli też ſynowie Izráelſcy / mówiąc : Któż nas nákármi mięſem?
Wſpominámy ſobie ná ryby / któreśmy jádáli w Egipćie dármo / ná ogórki / y ná melony / y ná łucżek / y ná cebulę / y ná cżoſnek.
A teraz duƺá náƺá wywiędłá nic inƺego nie májąc / oprócż tey mánny / przed ocżymá ſwemi.
A mánná byłá jáko náśienie koryjándrowe / á bárwá tey jáko bárwá Bdelijowá.
Y wychodźił lud / á zbieráli ją / y mełli w żárnách / álbo tłukli w moźdźierzách á wárzyli w kotłách y cżynili z niej podpłomyki ; á był ſmák jey jáko ſmák świeżej oliwy.
Gdy bowiem pádáłá roſá ná obóz w nocy / pádáłá też mánná náń.
Tedy uſłyƺáł Mojżeƺ / że lud płákáł po domách ſwych / káżdy u drzwi námiotu ſwego ; dla cżego zápáliłá śię popędliwość Páńſká wielce / y nie podobáło śię też to Mojżeƺowi.
Y rzekł Mojżeƺ do Páná : Cżemużeś ták źle ucżynił ſłudze twemu? cżemużem nie ználázł łáſki w ocżách twoich / żeś włożył ćiężár tego wƺyſtkiego ludu ná mię?
Izálim ja pocżął ten wƺyſtek lud? izálim go ja zrodźił / iż mi mówiƺ : Nieś je ná łonie twojim / jáko piáſtun nośi niemowlątko / do źiemi / o którąś przyśiągł ojcom ich?
Gdźież mam mięſo / ábym dáł wƺyſtkiemu temu ludowi? bo płácżą ná mię / mówiąc : Dáj nám mięſá / ábyśmy jedli.
Nie mogę ja ſám znieść wƺyſtkiego ludu tego ; bo to nád możność moję.
A jeſli śię ták ze mną obchodźić chceƺ / proƺę rácżej mię zábij / jeſlim ználázł łáſkę w ocżách twoich / ábym nie pátrzył ná ſwoje złe.
Y rzekł Pán do Mojżeƺá : Zbierz mi śiedmdźieśiąt mężów z ſtárƺych Izráelſkich / które znáƺ / że ſą ſtárƺymi ludu / y kśiążętá jego / á przywiedź je przed námiot zgromádzeniá / y ſtáć tám będę z tobą ;
A Ja zſtąpię / y będę tám mówił z tobą / y wźiąwƺy z Duchá / który jeſt w tobie / udźielę im ; y ponioſą z tobą brzemię ludu / á nie ponieśieƺ go ty ſám.
A do ludu rzecżeƺ : Poświęććie śię ná jutro / á będźiećie jeść mięſo ; bośćie płákáli w uƺách Páńſkich / mówiąc : Któż nas nákármi mięſem? bo nám lepiej było w Egipćie ; y dá wám Pán mięſá / y będźiećie jedli.
Nie przez jeden dźień jeść będźiećie / áni przez dwá dni / áni przez pięć / áni przez dźieśięć dni / áni przez dwádźieśćiá dni :
Ale przez cáły mieśiąc / áż poleźie przez nozdrzá wáƺe / á zbrzydźi śię wám / przeto żeśćie pogárdźili Pánem / który jeſt między wámi / á płákáliśćie przed nim mówiąc : Ná cóżeśmy wyƺli z Egiptu?
Y rzekł Mojżeƺ : Sześć kroć ſto tyśięcy pieƺych jeſt ludu / między którym ja mieƺkám / á tyś powiedźiáł : Dám im mięſá / że będą jeść cáły mieśiąc.
Izáli im owiec y wołów nábiją / áby śię im doſtáło? Izáli wƺyſtkie ryby morſkie zbiorą im / áby doſtátek mieli?
Y rzekł Pán do Mojżeƺá : Azáż ręká Páńſká jeſt ſkurcżoná? teraz ujrzyƺ / jeśli śię wypełni ſłowo moje / cżyli nie.
Tedy wyƺedł Mojżeƺ / y opowiedźiáł ludowi ſłowá Páńſkie ; á zebráwƺy śiedmdźieśiąt mężów ſtárƺych z ludu / poſtáwił je około námiotu.
Y zſtąpił Pán w obłoku / y mówił do niego / á wźiąwƺy z Duchá / który był w nim / podźielił go między śiedmdźieśiąt mężów ſtárƺych ; y ſtáło śię / gdy odpocżął nád nimi on Duch / tedy prorokowáli / á potem nigdy.
Ale zoſtáli byli dwá mężowie w oboźie / imię jednego Eldád / á imię drugiego Medád ; ná których też odpocżął on Duch / bo oni byli nápiſáni / choć byli nie przyƺli do námiotu ; á ták ći prorokowáli w oboźie.
Tedy przybieżáło pácholę / y oznájmiło Mojżeƺowi / mówiąc : Eldád y Medád prorokują w oboźie.
Ale odpowiedźiáwƺy Jozue / ſyn Nunów / ſługá Mojżeƺów / jeden z młodźieńców jego / rzekł : Pánie mój Mojżeƺu / zábroń im.
Któremu odpowiedźiáł Mojżeƺ : Cżemuż im ty zázdrośćiƺ dla mnie? Boże dáj / áby wƺyſtek lud Páńſki prorokowáł / á iżby dáł Pán Duchá ſwego ná nie!
Wróćił śię tedy Mojżeƺ do obozu / on y ſtárśi Izráelſcy.
Zátem wyƺedł wiátr od Páná / y porwáwƺy przepiórki od morzá / ſpuśćił je ná obóz / z jedney ſtrony jáko ná jeden dźień chodu / z drugiej ſtrony tákże jáko ná jeden dźień chodu / około obozu / á jákoby ná dwá łokćie były nád źiemią.
Tedy wſtáwƺy lud przez cáły on dźień y przez cáłą noc / y názájutrz przez wƺyſtek dźień zbieráli one przepiórki / á kto nájmniej názbieráł / miał dźieśięć homerów : y náwieƺáli ich wƺędy ſobie około obozu.
Mięſo jeƺcże było między zębámi ich nie zeżwáne / gdy gniew Páńſki zápálił śię ná lud / y pobił Pán lud on plágą bárdzo wielką.
Y názwáne jeſt imię miejſcá onego Kibrot Hátááwá ; ábowiem tám pogrzebli lud / który pożądáł mięſá.
A z Kibrot Hátááwy ruƺył śię lud do Háſerotu / y mieƺkáli w Háſeroćie.
