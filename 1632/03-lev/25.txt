Rzekł nád to Pán do Mojżeƺá ná górze Synáj / mówiąc :
Powiedz ſynom Izráelſkim / á mów do nich : Gdy wnijdziećie do ziemi / którą Já wám dáję / tedy święćić będzie ziemiá ſábát Pánu.
Przez ƺeść lát ośiewáć będzieƺ pole twoje / y przez ƺeść lát winnice twoje obrzynáć będzieƺ / zbierájąc urodzáje z niej ;
Ale roku śiódmego ſábát odpocżynieniá mieć będzie ziemiá / ſábát Páńſki ; polá twego nie będzieƺ ośiewáł / áni winnicy twojej obrzynáł.
Co śię ſámo przez śię zrodzi zbożá twego / nie będzieƺ żął / y jágód zániechánej winnicy twojej nie będzieƺ zbieráł ; rok odpocżynieniá będzie miáłá ziemiá.
Y będzie / co śię urodzi w onem odpocżynieniu ziemi / tobie ná pokárm / y ſłudze twemu / y ſłużebnicy twej / y nájemnikowi twemu / y przychodniowi twemu / który mieƺká z tobą.
Tákże bydłu twemu / y zwierzowi / który jeſt w ziemi twojej / będzie wƺyſtek urodzáj jej ná pokárm.
Nálicżyƺ też ſobie śiedm tygodni lát / to jeſt śiedm kroć śiedm lát ; y ucżyniąć dni śiedmiu tygodni lát cżterdzieśći y dziewięć lát.
Tedy káżeƺ zátrąbić w trąbę hucżną mieśiącá śiódmego / dniá dzieśiątego tegoż mieśiącá ; w dzień ocżyƺcżeniá káżećie zátrąbić w trąbę po wƺyſtkiej ziemi wáƺej.
Y święćić będziećie rok pięćdzieśiąty / á obwołáćie wolność w ziemi wƺyſtkim obywátelom jej. Láto miłośćiwe mieć będziećie ; y wróći śię káżdy do ośiádłośći ſwojej / y káżdy do rodziny ſwojej wróći śię.
To miłośćiwe láto pięćdzieśiątego roku miewáć będziećie ; nie będziećie śiáć / y nie będziećie żąć tego / co śię ſámo przez śię zrodzi / áni zbieráć będziećie gron z winnic zániechánych.
Bo miłośćiwy rok jeſt / święty wám będzie ; co śię ná polu przedtem zrodziło / to jeść będziećie.
W ten miłośćiwy rok wróći śię káżdy do ośiádłośći ſwojej.
Jeźli co ſprzedáƺ bliźniemu twemu / álbo co kupiƺ od bliźniego twego / niech nie oƺukiwá jeden drugiego.
Według licżby lát po miłośćiwym roku kupiƺ od bliźniego twego ; y według licżby lát dochody ſprzedá tobie.
Jeźli więcej będzie lát / tem drożej oƺácujeƺ kupno ono ; á jeźli mniej będzie lát / tedy też tániej oƺácujeƺ kupno ono / poniewáż tylko licżbá dochodów ſprzedáwá śię tobie.
A ták nie oƺukiwájćie żáden bliźniego ſwego / ále śię bój káżdy Bogá ſwego ; bom Já Pán / Bóg wáƺ.
Przeſtrzegájćie uſtáw moich / y ſądy moje záchowywájćie / y cżyńćie je / ábyśćie mieƺkáć mogli w ziemi onej bezpiecżnie.
Tedy wydá ziemiá owoc ſwój / á będziećie jeść áż do ſytośći / y będziećie mieƺkáć bezpiecżnie w niej.
A jeźlibyśćie rzekli : Cóż będziemy jeść roku śiódmego / jeźli nie będziem śiáć áni zbieráć urodzájów náƺych?
Tedy rozkáżę błogoſłáwieńſtwu memu przyjść ná wáſ roku ƺóſtego / y przynieśie urodzáj ná trzy látá.
Y będziećie śiáć roku óſmego / á będziećie jeść urodzáj ſtáry áż do roku dziewiątego ; póki nie nádejdą pożytki jego / ſtáre jeść będziećie.
Ziemiá tedy nie będzie ſprzedáwáná ná wiecżność ; bo mojá jeſt ziemiá / á wyśćie gośćmi y przychodniámi u mnie.
A po wƺyſtkiej ziemi ośiádłośći wáƺej pozwolićie wykupywáć ziemię.
Gdyby zubożáł brát twój / á ſprzedáłby nieco z májętnośći ſwojej / y przyƺedłby májący práwo odkupieniá / powinny jego niech wykupi / co ſprzedáł brát jego.
A jeźliby kto nie miáł tego / coby odkupić mógł / á ſám by przemógł / y ználázł doſtátek ná to wykupno :
Tedy obráchowáwƺy látá od ſprzedániá ſwego / wróći co zbywá temu / któremu ſprzedáł : á wróći śię do májętnośći ſwojej.
A jeźliby nie miáł doſtátku / áby wróćił / tedy zoſtánie májętność ſprzedáná w ręku tego / który ją kupił / áż do roku miłośćiwego / y uſtąpi mu jej w rok miłośćiwy / á on wróći śię do májętnośći ſwojej.
Jeźliby też kto ſprzedáł dom mieƺkániá w mieśćie murowánem / będzie miáł wolność wykupić go / póki nie wynijdzie rok ſprzedániá jego / cáły rok będzie miáł práwo do wykupieniá jego.
A jeźli go nie wykupi / póki nie wynijdzie rok cáły / tedy zoſtánie on dom w mieśćie murowánem temu / który go kupił / dziedzicżnie / y potomkom jego.
Y nie uſtąpi w miłośćiwe láto. Ale domy we wśiách / które nie ſą murem obtocżone / te práwem jáko pole ziemi ƺácowáne będą ; będą mogły być odkupowáne / y w miłośćiwe láto z rąk obcych wynijdą.
Ale miáſtá Lewitów / y domy w dziedzicżnych mieśćiech ich káżdego cżáſu wykupowáne być mogą przez Lewity.
Lecż temu co kupuje od Lewitów / wynijdzie kupno domu / y miejſkiej ośiádłośći jego / w rok miłośćiwy / gdyż domy miáſt Lewickich ſą dziedzicżne ich / w pośrodku ſynów Izráelſkich.
Ale pole ná przedmieśćiu ich nie będzie ſprzedáwáne ; bo dziedzictwem ich jeſt wiecżnem.
Gdyby też zubożáł brát twój / á oſłábiáłáby ręká jego przy tobie / podeprzeƺ go ; á jáko y przychodzień niech śię żywi przy tobie.
Nie bierz od niego lichwy / áni płátu / ále śię bój Bogá ſwego / áby śię żywił brát twój przy tobie.
Pieniędzy twoich nie dáwáj mu ná lichwę / áni mu z zyſku pożycżáj żywnośći twojej.
Jám Pán / Bóg wáƺ / którym wáſ wywiódł z ziemi Egipſkiej / ábym wám dáł ziemię Chánánejſką / á był wám zá Bogá.
Jeźliby też zubożáł brát twój przy tobie / ták żeby śię tobie záprzedáł / nie będzieƺ go dręcżył ſłużbą niewolnicżą ;
Jáko nájemnik / jáko przychodzień będzie u ćiebie / áż do roku miłośćiwego ſłużyć ći będzie.
Potem wynijdzie od ćiebie on / y dzieći jego z nim / á wróći śię do rodziny ſwojej / y do dziedzictwá przodków ſwych wróći śię.
Słudzy bowiem moi ſą / którem Já wywiódł z ziemi Egipſkiej / niechże nie będą ſprzedáwáni jáko niewolnicy.
Nie będzieƺ pánowáł nád nimi ſurowie / ále śię będzieƺ báł Páná Bogá twego.
Niewolnik też twój / y niewolnicá twojá / które mieć będzieƺ / będą z národów tych / które ſą około wáſ / z nich kupowáć będziećie niewolniká y niewolnicę.
Tákże też ſyny przychodniów mieƺkájących między wámi kupowáć będziećie / y z potomſtwá tych / którzy ſą z wámi / które ſpłodzili w ziemi wáƺej / á ći będą wám zá dziedzictwo.
Práwem dziedzicżnem trzymáć je będziećie / y ſynowie wáśi po wáſ / ábyśćie je dziedzicżnie odzierżeli / ná wieki ſłużby ich używáć będziećie ; lecż nád bráćią ſwą / ſyny Izráelſkimi / żáden nád brátem ſwoim nie będzie pánowáł ſurowie.
Jeźliby śię też gość álbo przychodzień zbogáćił / który mieƺká z tobą / á zubożáłby brát twój przy nim / ták żeby śię záprzedáł gośćiowi / álbo przychodniowi / który jeſt z tobą / álbo potomſtwu z domu cudzoziemców /
Gdyby śię záprzedáł / może być wykupiony ; ktokolwiek z bráći jego odkupi go ;
Albo ſtryj jego / álbo ſyn ſtryjá jego odkupi go / álbo z bliſkich pokrewnych jego z rodziny jego / odkupi go / álbo jeźliby przemógł / wykupi śię ſám.
Y poráchuje śię z onym / co go kupił / od roku / którego mu śię ſprzedáł / áż do miłośćiwego látá / áby pieniądze / zá które śię ſprzedáł / odłożone były według licżby lát / jáko z nájemnikiem / z nim ſobie poſtąpi.
Jeźliby jeƺcże nie máło lát zoſtáwáło / wedle nich wróći okup ſwój z pieniędzy / zá które kupiony jeſt.
A jeźliby nie wiele lát zoſtáwáło do miłośćiwego látá / tedy poráchuje śię z nim / á według onych lát wróći okup ſwój.
Jáko nájemnik dorocżny niech będzie u niego ; nie będzie nád nim ſurowie pánowáł przed ocżymá twemi.
A jeźliby śię tym obycżájem nie wykupił / tedy wynijdzie w miłośćiwe láto / on y dzieći jego z nim ;
Abowiem ſynowie Izráelſcy ſą ſługámi moimi ; ſługámi moimi ſą / którem wywiódł z ziemi Egipſkiej / Já Pán / Bóg wáƺ.
