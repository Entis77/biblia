Rzekł nád to Pán do Mojżeƺá ná górze Synáj / mówiąc :
Powiedz ſynom Izráelſkim / á mów do nich : Gdy wnijdźiećie do źiemi / którą Ja wam dáję / tedy święćić będźie źiemiá ſábát Pánu.
Przez ƺeść lat ośiewáć będźieƺ pole twoje / y przez ƺeść lat winnice twoje obrzynáć będźieƺ / zbierájąc urodzáje z niej ;
Ale roku śiódmego ſábát odpocżynieniá mieć będźie źiemiá / ſábát Páńſki ; polá twego nie będźieƺ ośiewał / áni winnicy twojey obrzynał.
Co śię ſámo przez śię zrodźi zbożá twego / nie będźieƺ żął / y jágód zániecháney winnicy twojey nie będźieƺ zbierał ; rok odpocżynieniá będźie miáłá źiemiá.
Y będźie / co śię urodźi w onem odpocżynieniu źiemi / tobie ná pokárm / y ſłudze twemu / y ſłużebnicy twej / y nájemnikowi twemu / y przychodniowi twemu / który mieƺká z tobą.
Tákże bydłu twemu / y zwierzowi / który jeſt w źiemi twojey / będźie wƺyſtek urodzáj jey ná pokárm.
Nálicżyƺ też ſobie śiedm tygodni lat / to jeſt śiedm kroć śiedm lat ; y ucżyniąć dni śiedmiu tygodni lat cżterdźieśći y dźiewięć lat.
Tedy káżeƺ zátrąbić w trąbę hucżną mieśiącá śiódmego / dniá dźieśiątego tegoż mieśiącá ; w dźień ocżyƺcżeniá káżećie zátrąbić w trąbę po wƺyſtkiej źiemi wáƺej.
Y święćić będźiećie rok pięćdźieśiąty / á obwołáćie wolność w źiemi wƺyſtkim obywátelom jey. Láto miłośćiwe mieć będźiećie ; y wróći śię káżdy do ośiádłośći ſwojey / y káżdy do rodźiny ſwojey wróći śię.
To miłośćiwe láto pięćdźieśiątego roku miewáć będźiećie ; nie będźiećie śiáć / y nie będźiećie żąć tego / co śię ſámo przez śię zrodźi / áni zbieráć będźiećie gron z winnic zániechánych.
Bo miłośćiwy rok jeſt / święty wam będźie ; co śię ná polu przedtem zrodźiło / to jeść będźiećie.
W ten miłośćiwy rok wróći śię káżdy do ośiádłośći ſwojey.
Jeſli co ſprzedáƺ bliźniemu twemu / álbo co kupiƺ od bliźniego twego / niech nie oƺukiwá jeden drugiego.
Według licżby lat po miłośćiwym roku kupiƺ od bliźniego twego ; y według licżby lat dochody ſprzedá tobie.
Jeſli więcey będźie lat / tym drożey oƺácujeƺ kupno ono ; á jeſli mniej będźie lat / tedy też tániej oƺácujeƺ kupno ono / poniewáż tylko licżbá dochodów ſprzedáwá śię tobie.
A ták nie oƺukiwájćie żáden bliźniego ſwego / ále śię bój káżdy Bogá ſwego ; bom Ja Pán / Bóg wáƺ.
Przeſtrzegájćie uſtáw mojich / y ſądy moje záchowywájćie / y cżyńćie je / ábyśćie mieƺkáć mogli w źiemi oney bezpiecżnie.
Tedy wydá źiemiá owoc ſwój / á będźiećie jeść áż do ſytośći / y będźiećie mieƺkáć bezpiecżnie w niej.
A jeſlibyśćie rzekli : Cóż będźiemy jeść roku śiódmego / jeſli nie będźiem śiáć áni zbieráć urodzájów náƺych?
Tedy rozkáżę błogoſłáwieńſtwu memu przyść ná was roku ƺóſtego / y przynieśie urodzáj ná trzy látá.
Y będźiećie śiáć roku óſmego / á będźiećie jeść urodzáj ſtáry áż do roku dźiewiątego ; póki nie nádejdą pożytki jego / ſtáre jeść będźiećie.
Ziemiá tedy nie będźie ſprzedáwáná ná wiecżność ; bo mojá jeſt źiemiá / á wyśćie gośćmi y przychodniámi u mnie.
A po wƺyſtkiej źiemi ośiádłośći wáƺej pozwolićie wykupywáć źiemię.
Gdyby zubożał brát twój / á ſprzedáłby nieco z májętnośći ſwojey / y przyƺedłby májący práwo odkupieniá / powinny jego niech wykupi / co ſprzedał brát jego.
A jeſliby kto nie miał tego / coby odkupić mógł / á ſám by przemógł / y ználázł doſtátek ná to wykupno :
Tedy obráchowáwƺy látá od ſprzedánia ſwego / wróći co zbywá temu / któremu ſprzedał : á wróći śię do májętnośći ſwojey.
A jeſliby nie miał doſtátku / áby wróćił / tedy zoſtánie májętność ſprzedáná w ręku tego / który ją kupił / áż do roku miłośćiwego / y uſtąpi mu jey w rok miłośćiwy / á on wróći śię do májętnośći ſwojey.
Jeſliby też kto ſprzedał dom mieƺkánia w mieśćie murowánem / będźie miał wolność wykupić go / póki nie wynijdźie rok ſprzedánia jego / cáły rok będźie miał práwo do wykupieniá jego.
A jeſli go nie wykupi / póki nie wynijdźie rok cáły / tedy zoſtánie on dom w mieśćie murowánem temu / który go kupił / dźiedźicżnie / y potomkom jego.
Y nie uſtąpi w miłośćiwe láto. Ale domy we wśiách / które nie ſą murem obtocżone / te práwem jáko pole źiemi ƺácowáne będą ; będą mogły być odkupowáne / y w miłośćiwe láto z rąk obcych wynijdą.
Ale miáſtá Lewitów / y domy w dźiedźicżnych mieśćiech ich káżdego cżáſu wykupowáne być mogą przez Lewity.
Lecż temu co kupuje od Lewitów / wynijdźie kupno domu / y miejſkiej ośiádłośći jego / w rok miłośćiwy / gdyż domy miaſt Lewickich ſą dźiedźicżne ich / w pośrzodku ſynów Izráelſkich.
Ale pole ná przedmieśćiu ich nie będźie ſprzedáwáne ; bo dźiedźictwem ich jeſt wiecżnem.
Gdyby też zubożał brát twój / á oſłábiáłáby ręká jego przy tobie / podeprzeƺ go ; á jáko y przychodźień niech śię żywi przy tobie.
Nie bierz od niego lichwy / áni płátu / ále śię bój Bogá ſwego / áby śię żywił brát twój przy tobie.
Pieniędzy twojich nie dáwáj mu ná lichwę / áni mu z zyſku pożycżáj żywnośći twojey.
Jam Pán / Bóg wáƺ / którym was wywiódł z źiemi Egipſkiej / ábym wam dał źiemię Chánánejſką / á był wam zá Bogá.
Jeſliby też zubożał brát twój przy tobie / ták żeby śię tobie záprzedał / nie będźieƺ go dręcżył ſłużbą niewolnicżą ;
Jáko nájemnik / jáko przychodźień będźie u ćiebie / áż do roku miłośćiwego ſłużyć ći będźie.
Potem wynijdźie od ćiebie on / y dźieći jego z nim / á wróći śię do rodźiny ſwojey / y do dźiedźictwá przodków ſwych wróći śię.
Słudzy bowiem moi ſą / którem Ja wywiódł z źiemi Egipſkiej / niechże nie będą ſprzedáwáni jáko niewolnicy.
Nie będźieƺ pánował nád nimi ſurowie / ále śię będźieƺ bał Páná Bogá twego.
Niewolnik też twój / y niewolnicá twojá / które mieć będźieƺ / będą z národów tych / które ſą około was / z nich kupowáć będźiećie niewolniká y niewolnicę.
Tákże też ſyny przychodniów mieƺkájących miedzy wámi kupowáć będźiećie / y z potomſtwá tych / którzy ſą z wámi / które ſpłodźili w źiemi wáƺej / á ći będą wam zá dźiedźictwo.
Práwem dźiedźicżnem trzymáć je będźiećie / y ſynowie wáśi po was / ábyśćie je dźiedźicżnie odźierżeli / ná wieki ſłużby ich używáć będźiećie ; lecż nád bráćią ſwą / ſyny Izráelſkimi / żáden nád brátem ſwoim nie będźie pánował ſurowie.
Jeſliby śię też gość álbo przychodźień zbogáćił / który mieƺká z tobą / á zubożáłby brát twój przy nim / ták żeby śię záprzedał gośćiowi / álbo przychodniowi / który jeſt z tobą / álbo potomſtwu z domu cudzoźiemców /
Gdyby śię záprzedał / może być wykupiony ; ktokolwiek z bráći jego odkupi go ;
Albo ſtryj jego / álbo ſyn ſtryjá jego odkupi go / álbo z bliſkich pokrewnych jego z rodźiny jego / odkupi go / álbo jeſliby przemógł / wykupi śię ſám.
Y poráchuje śię z onym / co go kupił / od roku / którego mu śię ſprzedał / áż do miłośćiwego látá / áby pieniądze / zá które śię ſprzedał / odłożone były według licżby lat / jáko z nájemnikiem / z nim ſobie poſtąpi.
Jeſliby jeƺcże nie máło lat zoſtáwáło / wedle nich wróći okup ſwój z pieniędzy / zá które kupiony jeſt.
A jeſliby nie wiele lat zoſtáwáło do miłośćiwego látá / tedy poráchuje śię z nim / á według onych lat wróći okup ſwój.
Jáko nájemnik dorocżny niech będźie u niego ; nie będźie nád nim ſurowie pánował przed ocżymá twemi.
A jeſliby śię tym obycżájem nie wykupił / tedy wynijdźie w miłośćiwe láto / on y dźieći jego z nim ;
Abowiem ſynowie Izráelſcy ſą ſługámi moimi ; ſługámi moimi ſą / którem wywiódł z źiemi Egipſkiej / Ja Pán / Bóg wáƺ.
