Rzekł potym PAN do Mojzeƺá : y do Aároná / mówiąc :
Powiedzćie Synom Izráelſkim / á mówćie do nich : Mąż któryby ćierpiał płynienie naśienia / z ćiáłá ſwego / nieczyſty jeſt.
A táć będźie nieczyſtość płynienia jego : Jeſli wypuśći ćiáło jego płynienie ſwe / álbo żeby śię to płynienie záſtánowiło w ćiele jego / nieczyſtość jego jeſt.
Káżda pośćiel / ná któreyby leżał płynienie ćierpiący / nieczyſta będźie / y wƺyſtko / ná cżymby uśiadł / nieczyſte będźie.
Ktoby śię dotknął pośćieli jego / upierze ƺáty ſwoje / y umyje śię wodą / á będźie nieczyſtym áż do wieczorá.
Ktoby też śiadł ná tym / ná cżym ten śiedźiał / co z niego <i>naśienie</i> płynie / upierze ƺáty ſwe / y umyje śię wodą / á będźie nieczyſtym / áż do wieczorá.
Jesliby śię też kto dotknął ćiáłá mężá ćierpiącego płynienie / upierze ƺáty ſwe / y umyje śię wodą á będźie nieczyſtym áż do wieczorá.
A jeſliby plunął płynienie ćierpiący ná czyſtego / <i>oplwány</i> upierze ƺáty ſwe y umyje śię wodą / á nieczyſty będźie / áż do wieczorá.
Káżde też śiodło ná którymby śiedźiał płynienie ćierpiący / nieczyſte będźie.
Ktoby śię też jákeykolwiek rzeczy dotknął / która byłá po nim / nieczyſty będźie áż do wieczorá : á ktoby co z tego nośił / upierze ƺáty ſwe / y umyje śię wodą / á będźie nieczyſtym áż do wieczorá.
Tákże káżdy któregoby śię dotknął ćierpiący płynienie / nie umywƺy przed tym rąk ſwojich w wodźie / upierze ƺáty ſwoje / y umyje śię wodą / y będźie nieczyſtym áż do wieczorá.
Naczynie też gliniáne / któregoby śię dotknął co płynienie ćierpi / ſtłuczone będźie : á káżde naczynie drzewiane / wodą umyte będźie.
A gdyby oczyśćiony był / ten który cierpi płynienie / od płynienia ſwego / náliczy ſobie śiedm dni / podług ſwego oczyśćienia / y upierze ƺáty ſwe / y omyje ćiáło ſwoje wodą żywą / y będźie czyſtym.
Potym dniá óſmego / weźmie ſobie dwie Synogárlice / álbo dwoje gołąbiąt / á przyƺedƺy przed PANA do drzwi Namiotu zgromádzenia / odda je kápłanowi.
Tedy ofiárowáć będźie kápłan jedno z nich zá grzech / á drugie ná ofiárę cáłopalenia : Ták oczyśći go kápłan przed obliczem PAńſkim od płynienia jego.
A mąż z któregoby wyƺło naśienie złączenia / omyje wodą wƺyſtko ćiáło ſwe / y będźie nieczyſtym áż do wieczorá.
Káżda też ƺátá / y káżda ſkórá / ná któreyby było naśienie złączenia / wyprána będźie wodą / á będźie nieczyſtą áż do wieczorá.
Niewiáſtá tákże z którąby obcował mąż ćierpiący płynienie naśienia / <i>oboje</i> umyją śię wodą / á nieczyſtymi będą áż do wieczorá.
Tákże Niewiáſtá / któraby ćierpiáłá chorobę ſwoję / á płynęłáby krew z ćiáłá jey : przez śiedm dni będźie w odłącżeniu ſwym / káżdy coby śię jey dotknął / nieczyſty będźie áż do wieczorá.
Ná czymbykolwiek leżáłá w odłączeniu ſwym / nieczyſte będźie : tákże ná czymby śiedźiáłá / nieczyſte będźie.
Ktoby śię też dotknął pośćieli jey / upierze ƺáty ſwe / y umyje śię wodą / á będźie nieczyſty áż do wieczorá.
Tákże ktoby śię dotknął tego ná czymby śiedźiáłá / upierze ƺáty ſwe / y umyje śię wodą / á będźie nieczyſty áż do wieczorá.
Jeſliby też co było ná łożu jey / álbo ná czymby oná śiedźiáłá / á dotknąłby śię kto tego / nieczyſtym będźie áż do wieczorá.
A jesliby mąż ſpał z nią / á zoſtáłáby nieczyſtość jey ná nim / nieczyſty będźie przez śiedm dni / y káżde łoże / ná którymby leżał / nieczyſte będźie.
Też jeſliby niewiáſtá płynienie krwie ćierpiáłá przez wiele dni / mimo czás mieśiąców jey / álbo żeby krwie płynienie ćierpiáłá w zwyczájney chorobie / <i>tedy</i> po wƺyſtkie dni płynienia nieczyſtośći ſwojey / jáko y czáſu choroby ſwojey nieczyſtą będźie.
Káżda pośćiel jey ná któreyby leżáłá po wƺyſtkie dni płynienia ſwego / jáko pośćiel w przyrodzoney chorobie będźie / y káżda rzecż ná któreyby śiedźiáłá / nieczyſta będźie / według nieczyſtośći <i>przyrodzoney</i> choroby jey.
Ktoby śię kolwiek dotknął tych rzeczy / nieczyſtym będźie / y upierze ƺáty ſwe / y umyje śię wodą / á będźie nieczyſtym áż do wieczorá.
A gdy będźie oczyśćiona od płynienia ſwego / náliczy ſobie śiedm dni / á potym oczyśćiáć śię będźie :
A dniá óſmego / weźmie ſobie dwie śynogárlice álbo dwoje gołąbiąt / y przynieśie je kápłanowi do drzwi Namiotu zgromádzenia.
Z których ofiárowáć będźie kápłan / jedno ná ofiárę zá grzech / á drugie ná ofiárę cáłopalenia : ták ją oczyśći kápłan przed PAnem / od płynienia nieczyſtośći jey.
Ták odłączáć będźiećie Syny Izráelſkie od nieczyſtośći ich / áby nie pomárli w nieczyſtośći ſwey / gdyby zplugáwili przybytek mój / który jeſt w pośrzodku ich.
Táć jeſt uſtáwá około tego który płynienie ćierpi / y z którego wychodźi naśienie złączenia / dla czego ſplugáwiony bywa.
Tákże y około <i>Niewiáſty</i> chorującey w odłączeniu ſwym / y káżdego ćierpiącego płynienie ſwe ták mężcżyzny / jáko y niewiáſty / y mężá który leżał z nieczyſtą.
