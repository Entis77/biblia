Rzekł potem Pán do Mojżeƺá y do Aároná / mówiąc :
Powiedzćie ſynom Izráelſkim / á mówćie do nich : Mąż / któryby ćierpiał płynienie náśieniá z ćiáłá ſwego / niecżyſty jeſt.
A táć będźie niecżyſtość płynieniá jego : Jeſli wypuśći ćiáło jego płynienie ſwe / álbo żeby śię to płynienie záſtánowiło w ćiele jego / niecżyſtość jego jeſt.
Káżda pośćiel / ná którejby leżał płynienie ćierpiący / niecżyſtá będźie / y wƺyſtko / ná cżemby uśiádł / niecżyſte będźie.
Ktoby śię dotknął pośćieli jego / upierze ƺáty ſwoje / y umyje śię wodą / á będźie niecżyſtym áż do wiecżorá.
Ktoby też śiádł ná tym / ná cżym ten śiedźiał / co z niego náśienie płynie / upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſtym áż do wiecżorá.
Jeſliby śię też kto dotknął ćiáłá mężá ćierpiącego płynienie / upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſtym áż do wiecżorá.
A jeſliby plunął płynienie ćierpiący ná cżyſtego / oplwány upierze ƺáty ſwe / y umyje śię wodą / á niecżyſty będźie áż do wiecżorá.
Káżde też śiodło / ná któremby śiedźiał płynienie ćierpiący / niecżyſte będźie.
Ktoby śię też jákiejkolwiek rzecży dotknął / która byłá po nim / niecżyſty będźie áż do wiecżorá ; á ktoby co z tego nośił / upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſtym áż do wiecżorá.
Tákże káżdy / któregoby śię dotknął ćierpiący płynienie / nie umywƺy przedtem rąk ſwojich w wodźie / upierze ƺáty ſwoje / y umyje śię wodą / y będźie niecżyſtym áż do wiecżorá.
Nacżynie też gliniáne / któregoby śię dotknął / co płynienie ćierpi / ſtłucżone będźie / á káżde nacżynie drzewiáne wodą umyte będźie.
A gdyby ocżyƺcżony był ten / który ćierpi płynienie / od płynieniá ſwego / nálicży ſobie śiedm dni podług ſwego ocżyƺcżeniá / y upierze ƺáty ſwe / y omyje ćiáło ſwoje wodą żywą / y będźie cżyſtym.
Potem dniá óſmego weźmie ſobie dwie ſynogárlice / álbo dwoje gołąbiąt / á przyƺedƺy przed Páná do drzwi namiotu zgromádzenia / oddá je kápłánowi.
Tedy ofiárowáć będźie kápłán jedno z nich zá grzech / á drugie ná ofiárę cáłopalenia. Ták ocżyśći go kápłán przed oblicżem Páńſkim od płynieniá jego.
A mąż / z któregoby wyƺło náśienie złącżeniá / omyje wodą wƺyſtko ćiáło ſwe / y będźie niecżyſtym áż do wiecżorá.
Káżda też ƺátá / y káżda ſkórá / ná którejby było náśienie złącżeniá / wypráná będźie wodą / á będźie niecżyſtą áż do wiecżorá.
Niewiáſtá tákże / z którąby obcował mąż ćierpiący płynienie náśieniá / oboje umyją śię wodą / á niecżyſtymi będą áż do wiecżorá.
Tákże niewiáſtá / któráby ćierpiáłá chorobę ſwoję / á płynęłáby krew z ćiáłá jey / przez śiedm dni będźie w odłącżeniu ſwem ; káżdy / coby śię jey dotknął / niecżyſty będźie áż do wiecżorá.
Ná cżembykolwiek leżáłá w odłącżeniu ſwem / niecżyſte będźie ; tákże ná cżemby śiedźiáłá / niecżyſte będźie.
Ktoby śię też dotknął pośćieli jey / upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſty áż do wiecżorá.
Tákże ktoby śię dotknął tego / ná cżemby śiedźiáłá / upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſty áż do wiecżorá.
Jeſliby też co było ná łożu jey / álbo ná cżemby oná śiedźiáłá / á dotknąłby śię kto tego / niecżyſtym będźie áż do wiecżorá.
A jeſliby mąż ſpał z nią / á zoſtáłáby niecżyſtość jey ná nim / niecżyſty będźie przez śiedm dni / y káżde łoże / ná któremby leżał / niecżyſte będźie.
Jeſliby też niewiáſtá płynienie krwi ćierpiáłá przez wiele dni / mimo cżáſu mieśiąców jey / álbo żeby krwi płynienie ćierpiáłá w zwycżájney chorobie / tedy po wƺyſtkie dni płynieniá niecżyſtośći ſwojey / jáko y cżáſu choroby ſwojey / niecżyſtą będźie.
Káżda pośćiel jey / ná którejby leżáłá po wƺyſtkie dni płynieniá ſwego / jáko pośćiel w przyrodzoney chorobie będźie / y káżda rzecż / ná którejby śiedźiáłá / niecżyſtá będźie według niecżyſtośći przyrodzoney choroby jey.
Ktoby śię kolwiek dotknął tych rzecży / niecżyſtym będźie / y upierze ƺáty ſwe / y umyje śię wodą / á będźie niecżyſtym áż do wiecżorá.
A gdy będźie ocżyƺcżoná od płynieniá ſwego / nálicży ſobie śiedm dni / á potem ocżyƺcżáć śię będźie.
A dniá óſmego weźmie ſobie dwie ſynogárlice / álbo dwoje gołąbiąt / y przynieśie je kápłánowi do drzwi namiotu zgromádzenia ;
Z których ofiárowáć będźie kápłán / jedno ná ofiárę zá grzech / á drugie ná ofiárę cáłopalenia : ták ją ocżyśći kápłán przed Pánem od płynieniá niecżyſtośći jey.
Ták odłącżáć będźiećie ſyny Izráelſkie od niecżyſtośći ich / áby nie pomárli w niecżyſtośći ſwej / gdyby ſplugáwili przybytek mój / który jeſt w pośrzodku ich.
Táć jeſt uſtáwá około tego / który płynienie ćierpi / y z którego wychodźi náśienie złącżeniá / dla cżego ſplugáwiony bywá.
Tákże y około niewiáſty / chorującey w odłącżeniu ſwem / y káżdego ćierpiącego płynienie ſwe / ták mężcżyzny / jáko y niewiáſty / y mężá / który leżał z niecżyſtą.
