Rzekł záśię Pán do Mojżeƺá y do Aároná / mówiąc :
Cżłowiek / któryby miáł ná ſkórze ćiáłá ſwego ſádzel / álbo liƺáj / álbo biáłe blizny / ták iżby śię ná ſkórze ćiáłá jego okázáłá plágá trądu / przywiedźiony będźie do Aároná kápłáná / álbo do którego z ſynów jego kápłánów.
Tedy oglądá kápłán on ſádzel ná ſkórze ćiáłá jego ; jeźliby włoſ ná onym ſádzelu pobieláł / á on ſádzel ná ſpojrzeniu byłby głębƺy / niż inƺá ſkórá ćiáłá jego / plágá trądu jeſt ; przetoż oglądáwƺy go kápłán / oſądźi go być niecżyſtym.
A jeźliby biáłá tylko blizná byłá ná ſkórze ćiáłá jego / á nie byłáby głębƺá ná ſpojrzeniu / niż inná ſkórá / y włoſy w niej nie pobieláłyby / tedy zámknie kápłán májącego táką zárázę przez śiedm dni.
Potem obejrzy go kápłán dniá śiódmego ; á jeźliby oná blizná ták zoſtáłá w ocżách jego / á nie ƺerzyłá śię oná blizná po ſkórze / tedy go zámknie kápłán przez śiedm dni powtóre.
Y obejrzy go kápłán dniá śiódmego powtóre ; á jeźliby tá zárázá pocżerniáłá á nie ƺerzyłáby śię tá zárázá po ſkórze / tedy go zá cżyſtego oſądźi kápłán / bo świerzb jeſt ; á on upierze ƺáty ſwe / á będźie cżyſtym.
Ale jeźliby śię bárdźiej rozƺerzáł po ſkórze świerzb on po oglądániu kápłánowem y po ocżyƺcżeniu jego / pokáże śię znowu kápłánowi.
Tedy obejrzy go kápłán ; á jeźliby śię tá zárázá rozƺerzáłá po ſkórze jego / oſądźi go zá niecżyſtego kápłán ; bo trąd jeſt.
Zárázá trądu gdy będźie ná cżłowieku / przywiedźion będźie do kápłáná.
Którego obejrzy kápłán ; á będźieli ſádzel biáły ná ſkórze / żeby śię ucżyniły włoſy biáłe / choćby y zdrowe ćiáło było ná tym ſádzelu /
Trąd záſtárzáły jeſt ná ſkórze ćiáłá jego ; y oſądźi go zá niecżyſtego kápłán / á nie będźie go záwieráł / gdyż niecżyſtym jeſt.
A jeźliby śię trąd rozƺerzáł ná ſkórze / y okryłby trąd wƺyſtkę ſkórę záráżonego od głowy jego áż do nóg jego / wƺędy gdźie ocżymá kápłán dojrzeć może ;
Y obejrzy kápłán ; á jeźli okrył trąd wƺyſtko ćiáło jego / zá cżyſtą oſądźi zárázę jego ; bo iż wƺyſtká pobieláłá / dlá tego cżyſty jeſt.
Ale którego dniá ukázáłoby śię ná tákowym mięſo dźiwie / niecżyſtym będźie.
A ták oglądá kápłán mięſo dźiwie / á oſądźi go być zá niecżyſtego / bo ono mięſo dźiwie niecżyſte jeſt ; trąd to jeſt.
Ale jeźliby záś zginęło mięſo dźiwie / y obróćiłoby śię w biáłość / tedy przyjdźie do kápłáná.
A widząc go kápłán / iż śię obróćiłá zárázá w biáłość / zá cżyſtego oſądźi kápłán záráżonego ; bo cżyſtym jeſt.
Jeźliby záś był ná ſkórze ćiáłá jego wrzód / á zágoiłby śię ;
A ná miejſcu wrzodu onego ucżyniłby śię ſádzel biáły / álbo blizná biáłá zácżerwieniáłá / tedy okázáná będźie kápłánowi.
A widząc kápłán / że ná wejrzeniu głębƺá jeſt / niż inná ſkórá / y włoſyby jej pobieláły / zá niecżyſtego oſądźi go kápłán ; zárázá trądu jeſt z wrzodu wyroſłá.
Ale jeźliby ją obácżył kápłán / że w niej włoſ nie bieleje / y nie jeſt głębƺá nád inƺą ſkórę / ále tylko nácżerniáłá / tedy zámknie kápłán tákowego przez śiedm dni.
A jeźliby śię ƺerzyłá po ſkórze / zá niecżyſtego oſądźi go kápłán ; zárázá to trądu.
Wƺákże jeźliby blizná oná biáłá ná ſwem miejſcu zoſtáwáłá ; y nie ƺerzyłáby śię / zápálenie wrzodu jeſt ; przetoż zá cżyſtego oſądźi go kápłán.
Tákże ćiáło / ná którego ſkórze byłáby ſpárzeliná od ogniá / á po zgojeniu onej ſpárzeliny byłáby blizná biáłá / zácżerwieniáłá / álbo biáłá tylko /
Oglądá to kápłán ; á jeźliby włoſ w bliźnie pobieláł y lśnił śię / á ná ſpojrzeniu byłáby głębƺá oná blizná niż ſkórá / trąd jeſt z ſpárzeliny wyroſły ; przetoż zá niecżyſtego oſądźi go kápłán / bo zárázá trądu jeſt.
A jeźli kápłán obácży / że ná onej bliźnie biáłej włoſ nie pobieláł / á iż nie jeſt głębƺá nád inƺą ſkórę / ále iż nieco nácżerniáłá / zámknie kápłán tákowego przez śiedm dni.
Y obejrzy go kápłán dniá śiódmego ; jeźliby śię bárdźiej ƺerzyłá po ſkórze / oſądźi go zá niecżyſtego kápłán : zárázá to trądu.
A jeźliż tá blizná biáłá zoſtáwá ná ſwem miejſcu / á nie ƺerzy śię po ſkórze / ále śię zácżerniwá / przyƺcżel z ſpárzeniá jeſt ; y oſądźi go zá cżyſtego kápłán / bo blizná ſpárzeliny jeſt.
Gdyby mąż / álbo niewiáſtá mieli jáką plámę ná głowie álbo ná brodźie :
Tedy obejrzy kápłán onę plámę ; á będźieli ná ſpojrzeniu głębƺá niż inƺá ſkórá / y byłby ná niej włoſ pożółkły y ſubtelny / tedy tákowego zá niecżyſtego oſądźi kápłán / zmázá jeſt ; trąd ná głowie álbo ná brodźie jeſt.
Ale jeźliby obácżył kápłán zárázę onej plámy / á oto / ná wejrzeniu jeſt głębƺá nád inƺą ſkórę / á nie byłby ná niej włoſ cżárny / zámknie kápłán zárázę plámy májącego przez śiedm dni.
Potem obejrzy kápłán tę zárázę dniá śiódmego ; á jeźli śię nie ƺerzy zmázá / y niemáƺ ná niej pożółkłego włoſá / y ná ſpojrzeniu tá zmázá nie byłáby głębƺá nád inƺą ſkórę :
Tedy ogolony będźie ten cżłowiek / ále zmázy onej golić nie będźie ; y zámknie kápłán májącego zmázę przez śiedm dni powtóre.
Y oglądá kápłán onę zmázę dniá śiódmego ; á jeźli śię nie rozƺerzyłá zmázá po ſkórze / á ná ſpojrzeniu nie jeſt głębƺá nád inƺą ſkórę / oſądźi go zá cżyſtego kápłán ; á on upráwƺy odźienie ſwoje / cżyſtym będźie.
A jeźliby śię pocżęłá ƺerzyć oná zmázá ná ſkórze po ocżyƺcżeniu jego :
Tedy obejrzy go kápłán ; á jeźliż śię ƺerzy oná zmázá po ſkórze / nie będźie więcej upátrowáł kápłán włoſu żółtego ; niecżyſtym jeſt.
Wƺákże jeźli przed ocżymá jego ták zoſtáwá oná zmázá / y włoſ cżárny wyróſłby ná niej / zgoiłá śię oná zmázá / cżyſty jeſt y zá cżyſtego oſądźi go kápłán.
A gdyby też ná ſkórze ćiáłá mężcżyzny álbo niewiáſty były blizny biáłe /
Tedy obejrzy je kápłán ; á jeźliby śię blizny one biáłe ná ſkórze ćiáłá jego zácżerniwáły / blizná biáłá jeſt / wyroſłá ná ſkórze ; cżyſtym jeſt.
Mąż tákże / któremuby opádły włoſy z głowy jego / łyſy jeſt / y cżyſty jeſt.
A jeźliżby przećiwko jednej ſtronie twárzy opádły włoſy głowy jego / przełyśiáły jeſt / cżyſty jeſt.
Wƺákże jeźliby ná łyśinie álbo ná tem przełyśieniu / okázáłá śię blizná biáłá á zcżerwieniáłáby / trąd wyróſł z łyśiny jego álbo z przełyśieniá jego.
Y obejrzy go kápłán ; á jeźliżby ſádzel zárázy jego był biáły / álbo zcżerwieniáły ná łyśinie jego / álbo ná obłyśieniu jego / ná kƺtáłt trądu ná ſkórze ćiáłá :
Tákowy cżłowiek trędowáty jeſt / niecżyſty jeſt / y oſądźi go bezpiecżnie kápłán zá niecżyſtego ; bo ná głowie jego jeſt trąd jego.
Trędowáty záś / któryby miáł ná ſobie tę zárázę / ƺáty jego będą rozdárte / y głowá jego będźie odkrytá / y uſtá ſobie zákryje ; á wołáć będźie : Niecżyſty / niecżyſty jeſtem.
Po wƺyſtkie dni / póki jeſt zárázá ná nim / niecżyſtym będźie / bo niecżyſtym jeſt / ſám będźie mieƺkáł ; precż zá obozem będźie mieƺkánie jego.
Jeźliby też ná ƺáćie byłá zárázá trądu / ná ƺáćie ſukniánej álbo ná ƺáćie lniánej /
Albo ná oſnowie / álbo ná wątku ze lnu álbo z wełny / álbo ná ſkórze / álbo ná jákiejkolwiek rzecży ſkórzánej ;
A byłáby tá zárázá źieloná / álbo cżerwoná ná ƺáćie / álbo ná ſkórze álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem / zárázá trądu jeſt ; będźie ukázáná kápłánowi.
A oglądáwƺy kápłán zárázę onę / zámknie onę rzecż záráżoną przez śiedm dni.
Potem obejrzy zárázę onę dniá śiódmego : jeźliby śię ƺerzyłá zmázá oná ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná ſkórze / y ná káżdej rzecży z ſkóry urobionej / trąd jeſt jádowity / zárázá niecżyſtá jeſt.
Tedy ſpáli tę ƺátę / álbo oſnowę / álbo wątek z wełny / álbo ze lnu / álbo jákiekolwiek nácżynie ſkórzáne / ná któremby byłá zárázá : ábowiem jeſt trąd jádowity / ogniem ſpálono będźie.
Ale gdyby obácżył kápłán / iż oná zmázá nie ƺerzy śię ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem :
Rozkáże kápłán / áby upráno to / ná cżem jeſt zárázá / y zámknie to przez śiedm dni powtóre.
Y obejrzy kápłán po uprániu onę zárázę ; á jeźli nie odmieniłá oná zárázá bárwy ſwojej / choćby śię oná zárázá nie rozƺerzyłá / rzecż niecżyſtá jeſt / ogniem ją ſpáliƺ ; záráźliwá rzecż jeſt / bądź ná zwierzchniej bądź ná ſpodniej ſtronie jej.
Wƺákże / jeźliby kápłán obácżył / iż przycżerniejƺá będźie zárázá po wyprániu ſwem / odedrze ją od ƺáty / álbo od ſkóry / álbo od oſnowy / álbo od wątku.
A jeźliby śię jeƺcże ukázáłá ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákiem nácżyniu ſkórzánem / trąd jeſt ƺerzący śię : ogniem to ſpáliƺ / ná cżemby byłá tákowá zárázá.
Szátę záś / álbo oſnowę / álbo wątek / álbo káżde nácżynie ſkórzáne / którebyś upráł / á odeƺłáby od niego zárázá / upierzeƺ je powtóre / á cżyſte będźie.
Táć jeſt uſtáwá o záráźie trądu / ná ƺáćie ſukniánej / álbo lniánej álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem / jáko to má być rozeznáno / iż jeſt cżyſte álbo niecżyſte.
