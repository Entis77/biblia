Rzekł záśię Pán do Mojżeƺá y do Aároná / mówiąc :
Cżłowiek / któryby miał ná ſkórze ćiáłá ſwego ſádzel / álbo liƺáj / álbo biáłe blizny / ták iżby śię ná ſkórze ćiáłá jego okázáłá plágá trądu / przywiedźiony będźie do Aároná kápłáná / álbo do którego z ſynów jego kápłanów.
Tedy oglądá kápłán on ſádzel ná ſkórze ćiáłá jego ; jeſliby włos ná onym ſádzelu pobielał / á on ſádzel ná ſpojrzeniu byłby głębƺy / niż inƺá ſkórá ćiáłá jego / plágá trądu jeſt ; przetoż oglądáwƺy go kápłán / oſądźi go być niecżyſtym.
A jeſliby biáłá tylko blizná byłá ná ſkórze ćiáłá jego / á nie byłáby głębƺá ná ſpojrzeniu / niż inná ſkórá / y włoſy w niej nie pobieláłyby / tedy zámknie kápłán májącego táką zárázę przez śiedm dni.
Potem obejrzy go kápłán dniá śiódmego ; á jeſliby oná blizná ták zoſtáłá w ocżách jego / á nie ƺerzyłá śię oná blizná po ſkórze / tedy go zámknie kápłán przez śiedm dni powtóre.
Y obejrzy go kápłán dniá śiódmego powtóre ; á jeſliby tá zárázá pocżerniáłá á nie ƺerzyłáby śię tá zárázá po ſkórze / tedy go zá cżyſtego oſądźi kápłán / bo świerzb jeſt ; á on upierze ƺáty ſwe / á będźie cżyſtym.
Ale jeſliby śię bárdźiej rozƺerzał po ſkórze świerzb on po oglądániu kápłánowem y po ocżyƺcżeniu jego / pokáże śię znowu kápłánowi.
Tedy obejrzy go kápłán ; á jeſliby śię tá zárázá rozƺerzáłá po ſkórze jego / oſądźi go zá niecżyſtego kápłán ; bo trąd jeſt.
Zárázá trądu gdy będźie ná cżłowieku / przywiedźion będźie do kápłáná.
Którego obejrzy kápłán ; á będźieli ſádzel biáły ná ſkórze / żeby śię ucżyniły włoſy biáłe / choćby y zdrowe ćiáło było ná tym ſádzelu /
Trąd záſtárzáły jeſt ná ſkórze ćiáłá jego ; y oſądźi go zá niecżyſtego kápłán / á nie będźie go záwierał / gdyż niecżyſtym jeſt.
A jeſliby śię trąd rozƺerzał ná ſkórze / y okryłby trąd wƺyſtkę ſkórę záráżonego od głowy jego áż do nóg jego / wƺędy gdźie ocżymá kápłán dojrzeć może ;
Y obejrzy kápłán ; á jeſli okrył trąd wƺyſtko ćiáło jego / zá cżyſtą oſądźi zárázę jego ; bo iż wƺyſtká pobieláłá / dla tego cżyſty jeſt.
Ale którego dniá ukázáłoby śię ná tákowym mięſo dźiwie / niecżyſtym będźie.
A ták oglądá kápłán mięſo dźiwie / á oſądźi go być zá niecżyſtego / bo ono mięſo dźiwie niecżyſte jeſt ; trąd to jeſt.
Ale jeſliby záś zginęło mięſo dźiwie / y obróćiłoby śię w biáłość / tedy przydźie do kápłáná.
A widząc go kápłán / iż śię obróćiłá zárázá w biáłość / zá cżyſtego oſądźi kápłán záráżonego ; bo cżyſtym jeſt.
Jeſliby záś był ná ſkórze ćiáłá jego wrzód / á zágoiłby śię ;
A ná miejſcu wrzodu onego ucżyniłby śię ſádzel biáły / álbo blizná biáłá zácżerwieniáłá / tedy okázáná będźie kápłánowi.
A widząc kápłán / że ná wejrzeniu głębƺá jeſt / niż inná ſkórá / y włoſyby jey pobieláły / zá niecżyſtego oſądźi go kápłán ; zárázá trądu jeſt z wrzodu wyroſłá.
Ale jeſliby ją obácżył kápłán / że w niej włos nie bieleje / y nie jeſt głębƺá nád inƺą ſkórę / ále tylko nácżerniáłá / tedy zámknie kápłán tákowego przez śiedm dni.
A jeſliby śię ƺerzyłá po ſkórze / zá niecżyſtego oſądźi go kápłán ; zárázá to trądu.
Wƺákże jeſliby blizná oná biáłá ná ſwem miejſcu zoſtáwáłá ; y nie ƺerzyłáby śię / zápálenie wrzodu jeſt ; przetoż zá cżyſtego oſądźi go kápłán.
Tákże ćiáło / ná którego ſkórze byłáby ſpárzeliná od ogniá / á po zgojeniu oney ſpárzeliny byłáby blizná biáłá / zácżerwieniáłá / álbo biáłá tylko /
Oglądá to kápłán ; á jeſliby włos w bliźnie pobielał y lśnił śię / á ná ſpojrzeniu byłáby głębƺá oná blizná niż ſkórá / trąd jeſt z ſpárzeliny wyroſły ; przetoż zá niecżyſtego oſądźi go kápłán / bo zárázá trądu jeſt.
A jeſli kápłán obácży / że ná oney bliźnie biáłej włos nie pobielał / á iż nie jeſt głębƺá nád inƺą ſkórę / ále iż nieco nácżerniáłá / zámknie kápłán tákowego przez śiedm dni.
Y obejrzy go kápłán dniá śiódmego ; jeſliby śię bárdźiej ƺerzyłá po ſkórze / oſądźi go zá niecżyſtego kápłán : zárázá to trądu.
A jeſliż tá blizná biáłá zoſtáwá ná ſwem miejſcu / á nie ƺerzy śię po ſkórze / ále śię zácżerniwá / przyƺcżel z ſpárzeniá jeſt ; y oſądźi go zá cżyſtego kápłán / bo blizná ſpárzeliny jeſt.
Gdyby mąż / álbo niewiáſtá mieli jáką plámę ná głowie álbo ná brodźie :
Tedy obejrzy kápłán onę plámę ; á będźieli ná ſpojrzeniu głębƺá niż inƺá ſkórá / y byłby ná niej włos pożółkły y ſubtelny / tedy tákowego zá niecżyſtego oſądźi kápłán / zmázá jeſt ; trąd ná głowie álbo ná brodźie jeſt.
Ale jeſliby obácżył kápłán zárázę oney plámy / á oto / ná wejrzeniu jeſt głębƺá nád inƺą ſkórę / á nie byłby ná niej włos cżárny / zámknie kápłán zárázę plámy májącego przez śiedm dni.
Potem obejrzy kápłán tę zárázę dniá śiódmego ; á jeſli śię nie ƺerzy zmázá / y niemáƺ ná niej pożółkłego włoſá / y ná ſpojrzeniu tá zmázá nie byłáby głębƺá nád inƺą ſkórę :
Tedy ogolony będźie ten cżłowiek / ále zmázy oney golić nie będźie ; y zámknie kápłán májącego zmázę przez śiedm dni powtóre.
Y oglądá kápłán onę zmázę dniá śiódmego ; á jeſli śię nie rozƺerzyłá zmázá po ſkórze / á ná ſpojrzeniu nie jeſt głębƺá nád inƺą ſkórę / oſądźi go zá cżyſtego kápłán ; á on upráwƺy odźienie ſwoje / cżyſtym będźie.
A jeſliby śię pocżęłá ƺerzyć oná zmázá ná ſkórze po ocżyƺcżeniu jego :
Tedy obejrzy go kápłán ; á jeſliż śię ƺerzy oná zmázá po ſkórze / nie będźie więcey upátrował kápłán włoſu żółtego ; niecżyſtym jeſt.
Wƺákże jeſli przed ocżymá jego ták zoſtáwá oná zmázá / y włos cżárny wyróſłby ná niej / zgoiłá śię oná zmázá / cżyſty jeſt y zá cżyſtego oſądźi go kápłán.
A gdyby też ná ſkórze ćiáłá mężcżyzny álbo niewiáſty były blizny biáłe /
Tedy obejrzy je kápłán ; á jeſliby śię blizny one biáłe ná ſkórze ćiáłá jego zácżerniwáły / blizná biáłá jeſt / wyroſłá ná ſkórze ; cżyſtym jeſt.
Mąż tákże / któremuby opádły włoſy z głowy jego / łyſy jeſt / y cżyſty jeſt.
A jeſliżby przećiwko jedney ſtronie twárzy opádły włoſy głowy jego / przełyśiáły jeſt / cżyſty jeſt.
Wƺákże jeſliby ná łyśinie álbo ná tym przełyśieniu / okázáłá śię blizná biáłá á zcżerwieniáłáby / trąd wyróſł z łyśiny jego álbo z przełyśieniá jego.
Y obejrzy go kápłán ; á jeſliżby ſádzel zárázy jego był biáły / álbo zcżerwieniáły ná łyśinie jego / álbo ná obłyśieniu jego / ná kƺtáłt trądu ná ſkórze ćiáłá :
Tákowy cżłowiek trędowáty jeſt / niecżyſty jeſt / y oſądźi go bezpiecżnie kápłán zá niecżyſtego ; bo ná głowie jego jeſt trąd jego.
Trędowáty záś / któryby miał ná ſobie tę zárázę / ƺáty jego będą rozdárte / y głowá jego będźie odkrytá / y uſtá ſobie zákryje ; á wołáć będźie : Niecżyſty / niecżyſty jeſtem.
Po wƺyſtkie dni / póki jeſt zárázá ná nim / niecżyſtym będźie / bo niecżyſtym jeſt / ſám będźie mieƺkał ; precż zá obozem będźie mieƺkánie jego.
Jeſliby też ná ƺáćie byłá zárázá trądu / ná ƺáćie ſukniáney álbo ná ƺáćie lniáney /
Albo ná oſnowie / álbo ná wątku ze lnu álbo z wełny / álbo ná ſkórze / álbo ná jákiejkolwiek rzecży ſkórzáney ;
A byłáby tá zárázá źieloná / álbo cżerwoná ná ƺáćie / álbo ná ſkórze álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem / zárázá trądu jeſt ; będźie ukázáná kápłánowi.
A oglądáwƺy kápłán zárázę onę / zámknie onę rzecż záráżoną przez śiedm dni.
Potem obejrzy zárázę onę dniá śiódmego : jeſliby śię ƺerzyłá zmázá oná ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná ſkórze / y ná káżdej rzecży z ſkóry urobioney / trąd jeſt jádowity / zárázá niecżyſtá jeſt.
Tedy ſpáli tę ƺátę / álbo oſnowę / álbo wątek z wełny / álbo ze lnu / álbo jákiekolwiek nacżynie ſkórzáne / ná któremby byłá zárázá : ábowiem jeſt trąd jádowity / ogniem ſpálono będźie.
Ale gdyby obácżył kápłán / iż oná zmázá nie ƺerzy śię ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem :
Rozkáże kápłán / áby upráno to / ná cżym jeſt zárázá / y zámknie to przez śiedm dni powtóre.
Y obejrzy kápłán po uprániu onę zárázę ; á jeſli nie odmieniłá oná zárázá bárwy ſwojey / choćby śię oná zárázá nie rozƺerzyłá / rzecż niecżyſtá jeſt / ogniem ją ſpáliƺ ; záráźliwá rzecż jeſt / bądź ná zwierzchniej bądź ná ſpodniej ſtronie jey.
Wƺákże / jeſliby kápłán obácżył / iż przycżerniejƺá będźie zárázá po wyprániu ſwem / odedrze ją od ƺáty / álbo od ſkóry / álbo od oſnowy / álbo od wątku.
A jeſliby śię jeƺcże ukázáłá ná ƺáćie / álbo ná oſnowie / álbo ná wątku / álbo ná jákiem nácżyniu ſkórzánem / trąd jeſt ƺerzący śię : ogniem to ſpáliƺ / ná cżemby byłá tákowá zárázá.
Szátę záś / álbo oſnowę / álbo wątek / álbo káżde nacżynie ſkórzáne / którebyś uprał / á odeƺłáby od niego zárázá / upierzeƺ je powtóre / á cżyſte będźie.
Táć jeſt uſtáwá o záráźie trądu / ná ƺáćie ſukniáney / álbo lniáney álbo ná oſnowie / álbo ná wątku / álbo ná jákiemkolwiek nácżyniu ſkórzánem / jáko to ma być rozeznáno / iż jeſt cżyſte álbo niecżyſte.
