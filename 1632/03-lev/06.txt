Potem rzekł Pán do Mojżeƺá / mówiąc :
Gdyby cżłowiek zgrzeƺył / á wyſtępkiem wyſtąpił przećiwko Pánu / á záprzáłby rzecży ſobie zwierzoney / y do ſchowániá dáney / álboby co wydárł / álboby gwáłtem wźiął od bliźniego ſwego ;
Tákże jeſliby rzecż zgubioną ználázł / á záprzáłby jey / álboby też przyśiągł fáłƺywie o którąkolwiek rzecż z tych / które cżyni cżłowiek / grzeƺąc przez nię ;
Gdyby tedy zgrzeƺył / á winien był / wróći rzecż / którą wydárł / álbo którą gwáłtem wźiął / álbo też rzecż ſobie powierzoną álbo rzecż zgubioną / którą ználázł ;
Albo też o cobykolwiek fáłƺywie przyśiągł : tedy wróći to cále / y piątą cżęść do tego przydá temu / cżyje było to ; wróći w dźień ofiáry zá grzech ſwój.
A ofiárę zá wyſtępek ſwój przywiedźie Pánu / báráná zupełnego z drobnego bydłá według oƺácowániá twego ná ofiárę zá grzech do kápłáná.
Y ocżyśći go kápłán przed Pánem / á będźie mu odpuƺcżoná káżdá z tych rzecży / którą ucżynił / y był jey winien.
Y rzekł Pán do Mojżeƺá / mówiąc :
Rozkáż Aáronowi y ſynom jego / y rzecż : Táć będźie uſtáwá ofiáry cáłopáleniá ; ofiárá cáłopáleniá jeſt od páleniá ná ołtárzu / przez cáłą noc áż do poránku ; bo ogień ná ołtárzu uſtáwicżnie gorzeć będźie.
Y oblecże śię kápłán w odźienie ſwoje lniáne / y ubiory lniáne oblecże ná ćiáło ſwoje / y zbierze popiół / gdy ſpáli ogień ofiárę cáłopáleniá ná ołtárzu / á wyſypie go podle ołtárzá.
Potem zewlecże ƺáty ſwe / y oblecże śię w ƺáty inne / á wynieśie popiół on zá obóz ná miejſce cżyſte.
A ogień ná ołtárzu uſtáwicżnie gorzeć będźie / nie będźie gáƺony ; á będźie zápáláł ná nim kápłán drwá ná káżdy poránek / y włoży náń ofiárę cáłopáleniá / á pálić będźie ná nim tłuſtość ofiár ſpokojnych.
Ogień uſtáwicżnie będźie gorzáł ná ołtárzu / nie będźie gáƺony.
A táć też jeſt uſtáwá ofiáry śniedney / którą ofiárowáć będą ſynowie Aáronowi przed oblicżem Páńſkim u ołtárzá.
Weźmie gárść ſwoję pƺenney mąki z tey ofiáry śniedney / y z oliwy jey ze wƺyſtkiem kádźidłem / które będźie ná ofierze śniedney / y to ſpáli ná ołtárzu ku wdźięcżney wonnośći ná pámiątkę jey Pánu.
A co zoſtánie z niej / jeść będą Aáron y ſynowie jego ; bez kwáſu jedzone będźie ná miejſcu świętym ; w śieni námiotu zgromádzeniá jeść to będą.
Nie będą tego wáżyć z kwáſem / bo zá dźiáł ich dáłem im to / z ofiár moich ogniſtych ; rzecż nájświętƺá to jeſt jáko y ofiárá zá grzech / y jáko ofiárá zá wyſtępek.
Káżdy mężcżyzná z ſynów Aáronowych jeść to będźie ; uſtáwá to wiecżná w národźiech wáƺych o pálonych ofiárách Páńſkich ; wƺyſtko / co śię ich dotknie / poświęcone będźie.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Táć jeſt ofiárá Aároná / y ſynów jego / którą ofiárowáć będą Pánu w dźień pomázániá ſwego : dźieśiątą cżęść efy mąki pƺenney zá ofiárę śniedną uſtáwicżną / połowę jey ráno / á połowę jey w wiecżór.
W páńwi z oliwą będźie gotowáná ; ſmáżoną przynieśieƺ ją / uſmáżoną ofiárę śniedną w ƺtukách ofiárowáć będźieƺ ku wdźięcżney wonnośći Pánu.
A kápłán pomázány z ſynów jego po nim ofiárowáć ją będźie ; uſtáwá to wiecżná Pánu / wƺyſtká ſpáloná będźie.
Y káżdá śniedná ofiárá kápłáńſká / wƺyſtká ſpáloná będźie ; nie będą jey jeść.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do Aároná / y do ſynów jego / á rzecż : Tá będźie uſtáwá ofiáry zá grzech : Ná miejſcu / gdźie biją ofiáry cáłopáleniá / będźie zábitá ofiárá zá grzech przed Pánem ; rzecż nájświętƺá jeſt.
Kápłán / któryby ją ofiárowáł zá grzech / jeść ją będźie ; ná miejſcu świętym jedzoná będźie ; w śieni námiotu zgromádzeniá.
Wƺyſtko / co śię dotknie mięſá jey / będźie poświęcone ; á jeſliby krwią jey ƺátá pokropioná byłá / co śię pokropiło / omyjeƺ ná miejſcu świętym.
Nácżynie też gliniáne / w któremby ją wárzono / ſtłucżone będźie ; á jeſliby w nácżyniu miedźiánem wárzoná byłá / wytrą je / y wymyją wodą.
Wƺelki mężcżyzná z kápłánów jeść ją będźie ; nájświętƺá to rzecż jeſt.
A żádná ofiárá zá grzech / którey krew wnáƺáná bywá do námiotu zgromádzeniá dla ocżyƺcżeniá w świątnicy / nie będźie jedzoná / ále ogniem ſpáloná będźie.
