Potem rzekł Pán do Mojżeƺá / mówiąc :
Gdyby cżłowiek zgrzeƺył / á wyſtępkiem wyſtąpił przećiwko Pánu / á záprzáłby rzecży ſobie zwierzonej / y do ſchowániá dánej / álboby co wydárł / álboby gwáłtem wziął od bliźniego ſwego ;
Tákże jeźliby rzecż zgubioną ználázł / á záprzáłby jej / álboby też przyśiągł fáłƺywie o którąkolwiek rzecż z tych / które cżyni cżłowiek / grzeƺąc przez nię ;
Gdyby tedy zgrzeƺył / á winien był / wróći rzecż / którą wydárł / álbo którą gwáłtem wziął / álbo też rzecż ſobie powierzoną álbo rzecż zgubioną / którą ználázł ;
Albo też o cobykolwiek fáłƺywie przyśiągł : tedy wróći to cále / y piątą cżęść do tego przydá temu / cżyje było to ; wróći w dzień ofiáry zá grzech ſwój.
A ofiárę zá wyſtępek ſwój przywiedzie Pánu / báráná zupełnego z drobnego bydłá według oƺácowániá twego ná ofiárę zá grzech do kápłáná.
Y ocżyśći go kápłán przed Pánem / á będzie mu odpuƺcżoná káżdá z tych rzecży / którą ucżynił / y był jej winien.
Y rzekł Pán do Mojżeƺá / mówiąc :
Rozkáż Aáronowi y ſynom jego / y rzecż : Táć będzie uſtáwá ofiáry cáłopáleniá ; ofiárá cáłopáleniá jeſt od páleniá ná ołtárzu / przez cáłą noc áż do poránku ; bo ogień ná ołtárzu uſtáwicżnie gorzeć będzie.
Y oblecże śię kápłán w odzienie ſwoje lniáne / y ubiory lniáne oblecże ná ćiáło ſwoje / y zbierze popiół / gdy ſpáli ogień ofiárę cáłopáleniá ná ołtárzu / á wyſypie go podle ołtárzá.
Potem zewlecże ƺáty ſwe / y oblecże śię w ƺáty inne / á wynieśie popiół on zá obóz ná miejſce cżyſte.
A ogień ná ołtárzu uſtáwicżnie gorzeć będzie / nie będzie gáƺony ; á będzie zápáláł ná nim kápłán drwá ná káżdy poránek / y włoży náń ofiárę cáłopáleniá / á pálić będzie ná nim tłuſtość ofiár ſpokojnych.
Ogień uſtáwicżnie będzie gorzáł ná ołtárzu / nie będzie gáƺony.
A táć też jeſt uſtáwá ofiáry śniednej / którą ofiárowáć będą ſynowie Aáronowi przed oblicżem Páńſkim u ołtárzá.
Weźmie gárść ſwoję pƺennej mąki z tej ofiáry śniednej / y z oliwy jej ze wƺyſtkiem kádzidłem / które będzie ná ofierze śniednej / y to ſpáli ná ołtárzu ku wdzięcżnej wonnośći ná pámiątkę jej Pánu.
A co zoſtánie z niej / jeść będą Aáron y ſynowie jego ; bez kwáſu jedzone będzie ná miejſcu świętym ; w śieni námiotu zgromádzeniá jeść to będą.
Nie będą tego wáżyć z kwáſem / bo zá dziáł ich dáłem im to / z ofiár moich ogniſtych ; rzecż nájświętƺá to jeſt jáko y ofiárá zá grzech / y jáko ofiárá zá wyſtępek.
Káżdy mężcżyzná z ſynów Aáronowych jeść to będzie ; uſtáwá to wiecżná w národziech wáƺych o pálonych ofiárách Páńſkich ; wƺyſtko / co śię ich dotknie / poświęcone będzie.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Táć jeſt ofiárá Aároná / y ſynów jego / którą ofiárowáć będą Pánu w dzień pomázániá ſwego : dzieśiątą cżęść efy mąki pƺennej zá ofiárę śniedną uſtáwicżną / połowę jej ráno / á połowę jej w wiecżór.
W páńwi z oliwą będzie gotowáná ; ſmáżoną przynieśieƺ ją / uſmáżoną ofiárę śniedną w ƺtukách ofiárowáć będzieƺ ku wdzięcżnej wonnośći Pánu.
A kápłán pomázány z ſynów jego po nim ofiárowáć ją będzie ; uſtáwá to wiecżná Pánu / wƺyſtká ſpáloná będzie.
Y káżdá śniedná ofiárá kápłáńſká / wƺyſtká ſpáloná będzie ; nie będą jej jeść.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Mów do Aároná / y do ſynów jego / á rzecż : Tá będzie uſtáwá ofiáry zá grzech : Ná miejſcu / gdzie biją ofiáry cáłopáleniá / będzie zábitá ofiárá zá grzech przed Pánem ; rzecż nájświętƺá jeſt.
Kápłán / któryby ją ofiárowáł zá grzech / jeść ją będzie ; ná miejſcu świętym jedzoná będzie ; w śieni námiotu zgromádzeniá.
Wƺyſtko / co śię dotknie mięſá jej / będzie poświęcone ; á jeźliby krwią jej ƺátá pokropioná byłá / co śię pokropiło / omyjeƺ ná miejſcu świętym.
Nácżynie też gliniáne / w któremby ją wárzono / ſtłucżone będzie ; á jeźliby w nácżyniu miedziánem wárzoná byłá / wytrą je / y wymyją wodą.
Wƺelki mężcżyzná z kápłánów jeść ją będzie ; nájświętƺá to rzecż jeſt.
A żádná ofiárá zá grzech / której krew wnáƺáná bywá do námiotu zgromádzeniá dlá ocżyƺcżeniá w świątnicy / nie będzie jedzoná / ále ogniem ſpáloná będzie.
