Zátem rzekł Pán do Mojżeƺá / mówiąc :
Powiedz Aáronowi y ſynom jego / áby śię wſtrzymywáli od rzecży / które ſą poświęcone od ſynów Izráelſkich / á nie plugáwili świętego imieniá mojego w tem / co mi oni poświęcáją ; Jám Pán.
A ták rzecż do nich : W národźiech wáƺych / ktobykolwiek przyſtąpił ze wƺyſtkiego potomſtwá wáƺego do poświęconych rzecży / któreby poświęćili ſynowie Izráelſcy Pánu / gdy niecżyſtość jego ná nim jeſt / wytrácony będźie ten od oblicżnośći mojej ; Jám Pán.
Ktobykolwiek z náśieniá Aáronowego był trędowátym álbo płynienie náśieniá ćierpiącym / rzecży poświęconych jeść nie będźie / pókiby śię nie ocżyśćił ; tákże ktoby śię dotknął jákiej niecżyſtośći ćiáłá zmárłego / álbo tego / z któregoby płynęło náśienie złącżeniá.
Tákże ktoby śię dotknął cżego / co śię cżołgá po źiemi / przez coby śię niecżyſtym ſtáł / álbo cżłowieká / przez któregoby śię ſplugáwił według wƺelákiej niecżyſtośći jego ;
Ten / ktoby śię cżego z tych rzecży dotknął / niecżyſtym będźie áż do wiecżorá / y nie będźie jádł rzecży poświęconych / áżby umył ćiáło ſwoje wodą.
Y áż po záchodźie ſłońcá cżyſtym będźie ; á potem będźie jeść z rzecży poświęconych / bo to jeſt pokárm jego.
Śćierwu też y rozƺárpánego od zwierzá jeść nie będźie / áby śię tem nie ſplugáwił ; Jám Pán.
A ták przeſtrzegáć będą rozkázániá mego / áby nie podlegli grzechowi / y nie pomárli w nim / gdyby śię ſplugáwili ; Jám Pán / który je poświęcám.
Żáden obcy nie będźie jádł z rzecży poświęconych ; komornik kápłáńſki / áni nájemnik nie będźie jádł rzecży poświęconych.
A jeźliby kápłán cżłowieká kupił zá pieniądze ſwoje / ten jeść będźie z rzecży tych ; tákże zrodzony w domu jego / ći będą jádáć z pokármów jego.
Lecż córká kápłáńſká / któráby ƺłá zá mężá obcego / tá z ofiár podnoƺeniá rzecży świętych jeść nie będźie.
Gdyby záś córká kápłáńſká wdową zoſtáłá / álbo odrzuconą byłá od mężá / y dźiátek nie miáłá / á wróćiłáby śię w dom ojcá ſwego / ták jáko w dźiećińſtwie ſwem chleb ojcá ſwego jeść będźie ; ále żáden obcy jeść z niego nie będźie.
A jeźliby kto jádł z niewiádomośći rzecży poświęcone / náddá piątą cżęść do tego / y oddá kápłánowi rzecż poświęconą.
Aby nie plugáwili rzecży poświęconych / które ſynowie Izráelſcy ofiárują Pánu /
Y nie przywodźili ná śię kárániá zá wyſtępek / gdyby jedli poświęcone rzecży ich ; bom Já Pán / który je poświęcám.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Powiedz Aáronowi y ſynom jego / y wƺyſtkim ſynom Izráelſkim / á mów do nich : Ktobykolwiek z domu Izráelſkiego / álbo z przychodniów w Izráelu ofiárowáł ofiárę ſwoję według wƺyſtkich ślubów ſwoich / y według wƺyſtkich dárów dobrowolnych ſwoich / któreby ofiárowáli Pánu ná ofiárę cáłopáleniá ;
Z dobrej woli ſwej ofiárowáć będźie zupełnego ſámcá z bydłá rogátego / z owiec / y z kóz.
Coby miáło ná ſobie wádę / ofiárowáć nie będźiećie ; bo nie będźie przyjemne od wáſ.
Jeźliby kto ofiárowáł ofiárę ſpokojną Pánu / pełniąc ślub / álbo dobrowolny dár oddájąc z rogátego bydłá / álbo z drobnego bydłá / bez wády będźie / áby przyjemne było ; żádnej wády nie będźie ná nim.
Ślepego / álbo ułomnego / álbo ná cżem ochromionego / álbo guzowátego / álbo kroſtáwego / álbo párƺywego / nie ofiárujćie Pánu / áni ná ofiárę ogniſtą dáwájćie ich ná ołtárz Pánu.
Wołu też álbo owcę zbytnich álbo niezupełnych cżłonków zá dobrowolny dár ofiárowáć je możeƺ : ále ślub z nich przyjemny nie będźie.
Zgniećionego y ſtłucżonego y przerwánego / y rzezánego nie będźiećie ofiárowáć Pánu ; w źiemi wáƺej nieucżynićie tego.
Ani z ręki cudzoźiemcá nie będźiećie ofiárowáć chlebá Bogu wáƺemu z tych wƺyſtkich rzecży / bo ułomek jeſt w nich ; wádę máją / nie będą przyjemne od wáſ.
Nád to rzekł Pán do Mojżeƺá / mówiąc :
Wół / álbo owcá / álbo kozá / gdy śię urodźi / niech będźie śiedm dni przy mátce ſwojej / á dniá óſmego / y potem będźie przyjemne ku pálonej ofierze Pánu.
Krowy też / áni owcy z płodem ich / nie zábijećie dniá jednego.
A gdybyśćie ofiárowáli ofiárę dźiękcżynieniá Pánu / z dobrej woli ſwej ofiárowáć będźiećie.
Onegoż dniá jedzoná będźie ; nie zoſtáwićie z niej nic áż do jutrá ; Jám Pán.
Przetoż ſtrzeżćie przykázáń moich / á cżyńćie je ; Jám Pán.
Y nie plugáwćie imieniá mego świętego / ábym był poświęcony w pośrodku ſynów Izráelſkich. Já Pán / który wáſ poświęcám ;
Którym wáſ wywiódł z źiemi Egipſkiej / ábym wám był zá Bogá ; Já Pán.
