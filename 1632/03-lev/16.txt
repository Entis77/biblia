Potem mówił Pán do Mojżeƺá po śmierći dwu ſynów Aáronowych / którzy ofiárując przed Pánem / pomárli ;
Y rzekł Pán do Mojżeƺá : Mów do Aároná / brátá twego / niech nie wchodzi káżdego cżáſu do świątnicy wewnątrz zá záſłonę przed ubłágálnią / którá jeſt ná ſkrzyni / áby nie umárł / bo w obłoku okázowáć śię będę nád ubłágálnią.
Ale ták wchodzić będzie Aáron do świątnicy z ćielcem ná ofiárę zá grzech / á z báránem ná ofiárę cáłopáleniá.
W ƺátę lniáną poświęconą oblecże śię / á ubiory lniáne będą ná ćiele jego ; y páſem lniánym opáƺe śię / y cżápkę lniáną włoży ná głowę ; ƺáty święte ſą ; y umyje wodą ćiáło ſwe / á oblecże śię w nie.
A od zgromádzeniá ſynów Izráelſkich weźmie dwu kozłów ná ofiárę zá grzech / y jednego báránká ná cáłopálenie.
Y będzie ofiárowáł Aáron ćielcá ſwego ná ofiárę zá grzech / y ucżyni ocżyƺcżenie zá śię / y zá dom ſwój.
Weźmie też dwu kozłów / á poſtáwi je przed Pánem u drzwi námiotu zgromádzeniá.
Y rzući Aáron ná obá kozły loſy ; loſ jeden Pánu / á loſ drugi Azázelowi.
Y będzie ofiárowáł Aáron onego kozłá / ná którego pádł loſ Pánu / y ofiárowáć go będzie zá grzech.
Ale kozłá / ná którego pádł loſ Azázelá / poſtáwi żywego przed Pánem / áby ocżyƺcżenie ucżynił przezeń / á wypuśćił go do Azázelá ná puƺcżą.
Y będzie ofiárowáł Aáron ćielcá / ná ofiárę zá grzech twój / á ocżyƺcżenie ucżyni zá śię / y zá dom ſwój / y zábije ćielcá ná ofiárę zá grzech ſwój.
Tedy weźmie pełną kádzielnicę węglá rozpálonego z ołtárzá przed oblicżnośćią Páńſką / y pełne gárśći ſwe kádzeniá wonnego utłucżonego y wnieśie zá záſłonę.
A włoży ono kádzenie ná ogień przed Pánem / áby okrył dym kádzeniá ubłágálnią / którá jeſt nád świádectwem / á nie umrze.
Potem wziąwƺy ze krwi ćielcá onego / kropić będzie pálcem ſwym ná ubłágálni ku wſchodowi ſłońcá ; tákże przed ubłágálnią kropić będzie śiedm kroć tą krwią pálcem ſwym.
Zábije też kozłá ná ofiárę zá grzech ludu / á wnieśie wewnątrz krew jego zá záſłonę ; y ucżyni ze krwią jego / jáko ucżynił ze krwią ćielcá / y kropić będzie nią nád ubłágálnią y przed ubłágálnią.
Ták ocżyśći świątnicę od niecżyſtót ſynów Izráelſkich / y od przeſtępſtw ich / y od wƺyſtkich grzechów ich ; toż też ucżyni námiotowi zgromádzeniá / który jeſt między nimi / w pośrodku niecżyſtót ich.
A żáden cżłowiek niech nie będzie w námioćie zgromádzeniá / gdy on wchodzić będzie ku ocżyƺcżániu do świątnicy / áż wynijdzie y wykoná ocżyƺcżenie ſám zá śię y zá dom ſwój / y zá wƺyſtko zgromádzenie Izráelſkie.
Y wynijdzie do ołtárzá / który jeſt przed Pánem / á ocżyśći go ; y wziąwƺy krwi ćielcowej y krwi kozłowej / pomáże rogi ołtárzá w około ;
A pokropi go z wierzchu tąż krwią pálcem ſwym śiedm kroć / á ocżyśći go / y poświęći go od niecżyſtót ſynów Izráelſkich.
Potem gdy odpráwi ocżyƺcżenie świątnicy y námiotu zgromádzeniá y ołtárzá / ofiárowáć będzie kozłá żywego.
A włożywƺy Aáron obie ręce ſwe ná głowę kozłá żywego / wyznáwáć będzie nád nim wƺyſtkie niepráwośći ſynów Izráelſkich / y wƺyſtkie przeſtępſtwá ich ze wƺyſtkiemi grzechámi ich / á włoży je ná głowę kozłá onego / y wypuśći go przez cżłowieká ná to obránego ná puƺcżą.
A ták ponieśie on kozieł ná ſobie wƺyſtkie niepráwośći ich do ziemi puſtej ; y wypuśći kozłá onego ná puƺcżą.
Potem wróćiwƺy śię Aáron do námiotu zgromádzeniá / złoży z śiebie ƺáty lniáne / w które śię był oblekł / wchodząc do świątnicy / y zoſtáwi je tám.
Omyje też ćiáło ſwoje wodą / ná miejſcu świętym / y oblecże śię w ƺáty ſwe ; á wyƺedłƺy ſpráwowáć będzie ofiárę cáłopáleniá ſwego / y ofiárę cáłopáleniá ludu / y ucżyni ocżyƺcżenie zá śię / y zá lud.
A tłuſtość ofiáry zá grzech ſpáli ná ołtárzu.
A ten / który záwiódł kozłá do Azázelá / upierze ƺáty ſwe ; á omywƺy ćiáło ſwoje wodą / potem wnijdzie do obozu.
Cielcá záś ofiárowánego zá grzech / y kozłá zá grzech / których krew wnieśioná byłá ku ſpráwowániu ocżyƺcżeniá do świątnicy / wynioſą precż zá obóz / y ſpálą ogniem ſkóry ich / y mięſo ich / y gnój ich.
A ten / co je pálić będzie / upierze ƺáty ſwoje / á omywƺy ćiáło ſwoje wodą / potem wnijdzie do obozu.
To też będzie wám zá uſtáwę wiecżną : Mieśiącá śiódmego / dzieśiątego dniá tegoż mieśiącá / trápić będziećie duƺe wáƺe / y żádnej roboty nie będziećie robić / ták w domu zrodzony / jáko przychodzień / który gośćiem jeſt między wámi ;
Bo w ten dzień ocżyśći wáſ kápłán / ábyśćie ocżyƺcżeni byli ; od wƺyſtkich grzechów wáƺych przed Pánem ocżyƺcżeni będziećie.
Sábátem odpocżynieniá będzie wám to / w który trápić będziećie duƺe wáƺe uſtáwą wiecżną.
A ocżyƺcżáć będzie kápłán / który jeſt pomázány / á którego poświęcone ſą ręce ku ſpráwowániu urzędu miáſto ojcá jego / á oblecże śię w ƺáty lniáne / w ƺáty święte ;
Y ocżyśći świątnicę świętobliwośći / y námiot zgromádzeniá ; y ołtárz ocżyśći / y kápłány / y wƺyſtek lud zgromádzony ocżyśći.
Y będzie to wám zá uſtáwę wiecżną ku ocżyƺcżániu ſynów Izráelſkich od wƺyſtkich grzechów ich ráz w rok.
Y ucżynił Mojżeƺ / jáko mu był rozkázáł Pán.
