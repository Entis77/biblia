Y ſtáło śię dniá óſmego / wezwał Mojzeƺ Aároná y ſynów jego / y ſtárƺych Izráelſkich.
Y rzekł do Aároná : Weźmij ſobie ćielcá młodego ná ofiárę zá grzech / y báráná ná ofiárę cáłopalenia / oboje zupełne / y ofiáruj je przed oblicżem Páńſkim.
Do ſynów záś Izráelſkich rzecżeƺ / mówiąc : Weźmijćie kozłá z kóz ná ofiárę zá grzech / y ćielcá / y báráná / rocżniáki zupełne / zdrowe / ná ofiárę cáłopalenia ;
Tákże wołu / y báráná ná ofiáry ſpokojne ku ofiárowániu przed Pánem / y ofiárę śniedną nágniećioną z oliwą ; ábowiem śię wam dźiś Pán ukáże.
Y przynieśli / co rozkázał Mojzeƺ / przed námiot zgromádzenia ; á przyſtąpiwƺy wƺyſtek lud / ſtánął przed Pánem.
Zátem rzekł Mojzeƺ : Táć jeſt rzecż / którą wam Pán rozkázał ; cżyńćież ją / á ukáże śię wam chwałá Páńſka.
Rzekł záś Mojzeƺ do Aároná : Przyſtąp do ołtárzá / á ucżyń ofiárę zá grzech ſwój / y ofiárę páloną twoję / á wykonáj ocżyƺcżenie zá śię y zá lud ; ucżyń też ofiárę od ludu / y ucżyń ocżyƺcżenie zá lud / jáko rozkázał Pán.
Tedy przyſtąpił Aáron do ołtárzá / y zábił ćielcá ná ofiárę zá grzech ſwój.
Y podáli mu ſynowie Aáronowi krew / który omocżywƺy pálec ſwój we krwi pomázał rogi ołtárzá / á oſtátek krwi wylał u ſpodku ołtárzá ;
Ale tłuſtość z nerkámi / y odźiecżkę z wątrobą z ofiáry zá grzech ſpálił ná ołtárzu / jáko był rozkázał Pán Mojżeƺowi ;
Mięſo záś y ſkórę ſpálił ogniem precż zá obozem.
Zábił też ofiárę cáłopalenia ; y podáli mu ſynowie Aáronowi krew / którą pokropił wierzch ołtárzá w około.
Przynieśli mu też ofiárę cáłopalenia / y ƺtuki jey y głowę jey ; á ſpálił ją ná ołtárzu ;
Omył też wnętrznośći / y nogi / y ſpálił je z ofiárą cáłopalenia ná ołtárzu.
Potem ſpráwował ofiárę wƺyſtkiego ludu / y wźiął kozłá ná ofiárę zá grzech ludu / którego zábił / y ofiárował go / jáko y pierwƺego.
Ofiárował też ofiárę cáłopalenia / y ucżynił jey według zwycżáju.
Ofiárował też ofiárę śniedną / á wźiąwƺy z niej pełną gárść ſwoję / ſpálił ná ołtárzu oprócż ofiáry cáłopalenia poránney.
Zábił też wołu / y báráná ná ofiárę ſpokojną / która byłá zá lud ; y podáli mu ſynowie Aáronowi krew / którą pokropił ołtárz z wierzchu około.
Podáli mu tákże tłuſtość z wołu ; y z báráná ogon / y tłuſtość okrywájącą wnętrznośći y nerki / y odźiecżkę z wątroby.
Włożyli też tłuſtośći ná moſtek / y ſpálili też tłuſtość ná ołtárzu ;
Ale moſtek y łopátkę práwą obrácał Aáron tám y ſám ná ofiárę obrácánia przed oblicżem Páńſkim / jáko był Pán rozkázał Mojżeƺowi.
Tedy podnióſƺy Aáron ręce ſwe do ludu błogoſłáwił im / á zſtąpił od ofiárowánia ofiáry zá grzech / y ofiáry cáłopalenia / y ofiáry ſpokojney.
Y wƺedł Mojzeƺ y Aáron do namiotu zgromádzenia / á wƺedƺy błogoſłáwili ludowi ; y okázáłá śię chwałá Páńſka wƺyſtkiemu ludowi ;
Bo zſtąpiwƺy ogień od oblicżnośći Páńſkiej ſpálił ná ołtárzu ofiárę cáłopalenia y tłuſtośći ; co gdy widźiał wƺyſtek lud / wykrzykáli á pádáli ná twárzy ſwoje.
