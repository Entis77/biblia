Y ſtáło śię dniá óſmego / wezwáł Mojżeƺ Aároná y ſynów jego / y ſtárƺych Izráelſkich.
Y rzekł do Aároná : Weźmij ſobie ćielcá młodego ná ofiárę zá grzech / y báráná ná ofiárę cáłopáleniá / oboje zupełne / y ofiáruj je przed oblicżem Páńſkim.
Do ſynów záś Izráelſkich rzecżeƺ / mówiąc : Weźmijćie kozłá z kóz ná ofiárę zá grzech / y ćielcá / y báráná / rocżniáki zupełne / zdrowe / ná ofiárę cáłopáleniá ;
Tákże wołu / y báráná ná ofiáry ſpokojne ku ofiárowániu przed Pánem / y ofiárę śniedną nágniećioną z oliwą ; ábowiem śię wám dźiś Pán ukáże.
Y przynieśli / co rozkázáł Mojżeƺ / przed námiot zgromádzeniá ; á przyſtąpiwƺy wƺyſtek lud / ſtánął przed Pánem.
Zátem rzekł Mojżeƺ : Táć jeſt rzecż / którą wám Pán rozkázáł ; cżyńćież ją / á ukáże śię wám chwáłá Páńſká.
Rzekł záś Mojżeƺ do Aároná : Przyſtąp do ołtárzá / á ucżyń ofiárę zá grzech ſwój / y ofiárę páloną twoję / á wykonáj ocżyƺcżenie zá śię y zá lud ; ucżyń też ofiárę od ludu / y ucżyń ocżyƺcżenie zá lud / jáko rozkázáł Pán.
Tedy przyſtąpił Aáron do ołtárzá / y zábił ćielcá ná ofiárę zá grzech ſwój.
Y podáli mu ſynowie Aáronowi krew / który omocżywƺy pálec ſwój we krwi pomázáł rogi ołtárzá / á oſtátek krwi wyláł u ſpodku ołtárzá ;
Ale tłuſtość z nerkámi / y odźiecżkę z wątrobą z ofiáry zá grzech ſpálił ná ołtárzu / jáko był rozkázáł Pán Mojżeƺowi ;
Mięſo záś y ſkórę ſpálił ogniem precż zá obozem.
Zábił też ofiárę cáłopáleniá ; y podáli mu ſynowie Aáronowi krew / którą pokropił wierzch ołtárzá w około.
Przynieśli mu też ofiárę cáłopáleniá / y ƺtuki jej y głowę jej ; á ſpálił ją ná ołtárzu ;
Omył też wnętrznośći / y nogi / y ſpálił je z ofiárą cáłopáleniá ná ołtárzu.
Potem ſpráwowáł ofiárę wƺyſtkiego ludu / y wźiął kozłá ná ofiárę zá grzech ludu / którego zábił / y ofiárowáł go / jáko y pierwƺego.
Ofiárowáł też ofiárę cáłopáleniá / y ucżynił jej według zwycżáju.
Ofiárowáł też ofiárę śniedną / á wźiąwƺy z niej pełną gárść ſwoję / ſpálił ná ołtárzu oprócż ofiáry cáłopáleniá poránnej.
Zábił też wołu / y báráná ná ofiárę ſpokojną / którá byłá zá lud ; y podáli mu ſynowie Aáronowi krew / którą pokropił ołtárz z wierzchu około.
Podáli mu tákże tłuſtość z wołu ; y z báráná ogon / y tłuſtość okrywájącą wnętrznośći y nerki / y odźiecżkę z wątroby.
Włożyli też tłuſtośći ná moſtek / y ſpálili też tłuſtość ná ołtárzu ;
Ale moſtek y łopátkę práwą obrácáł Aáron tám y ſám ná ofiárę obrácániá przed oblicżem Páńſkim / jáko był Pán rozkázáł Mojżeƺowi.
Tedy podnióſłƺy Aáron ręce ſwe do ludu błogoſłáwił im / á zſtąpił od ofiárowániá ofiáry zá grzech / y ofiáry cáłopáleniá / y ofiáry ſpokojnej.
Y wƺedł Mojżeƺ y Aáron do námiotu zgromádzeniá / á wƺedłƺy błogoſłáwili ludowi ; y okázáłá śię chwáłá Páńſká wƺyſtkiemu ludowi ;
Bo zſtąpiwƺy ogień od oblicżnośći Páńſkiej ſpálił ná ołtárzu ofiárę cáłopáleniá y tłuſtośći ; co gdy widźiáł wƺyſtek lud / wykrzykáli á pádáli ná twárzy ſwoje.
