Potym rzekł PAN do Mojzeƺá / mówiąc :
Mów do wƺyſtkiego zgromádzenia Synów Izráelſkich / á powiedz im : Świętymi bądźćie / bom ja <i>jeſt</i> święty / <i>PAN</i> / Bóg wáƺ.
Káżdy mátki ſwojey y Ojcá ſwego bójćie śię / á Sábbátów moich przeſtrzegajćie / <i>JAM PAN</i> / Bóg wáƺ.
Nie udawajćie śię zá báłwány / á Bogów litych nie czyńćie ſobie : <i>JAM PAN</i> Bóg wáƺ.
A gdy ofiárowáć będźiećie Ofiárę ſpokojną PANU / tedy z dobrey woli ſwey ofiárowáć ją będźiećie.
W dźień / którego ofiárowáć będźiećie / jedzćie <i>ją</i> y názájutrz / ále coby zoſtáło áż do trzećiego dniá / ogniem ſpalono będźie.
A jeſlibyśćie to jedli dniá trzećiego / obrzydłe będźie / <i>y</i> nie przyjemne.
Ktobykolwiek to jadł / karánie zá niepráwość ſwoję ponieśie / bo świątość PAńſką ſplugáwił / przetoż wytrácona będźie duƺá oná z ludu ſwego.
Gdy będźiećie żąć zbożá źiemie wáƺey / nie będźieƺ do końcá polá twego wyżynał / áni pozoſtáłych <i>kłoſów</i> żniwá twego zbieráć będźieƺ :
Tákże winnice twojey gron / do ƺczętu obieráć nie będźieƺ / áni jágód <i>opadájących</i> z winnice twey nie pozbieraƺ : ubogiemu y przychodniowi zoſtáwiƺ je : <i>JA PAN</i> Bóg wáƺ.
Nie krádnićie / áni zápierajćie / y nie oƺukiwajćie żaden bliźniego ſwego :
Nie przyśięgajćie fałƺywie przez Imię moje / y nie lży Imienia Bogá twego : <i>JA PAN.</i>
Nie ućiſkaj gwałtem bliźniego twego / áni go odźieraj : nie zoſtánie zapłátá najemniká u ćiebie do jutrá.
Nie złorzecz głuchemu / á przed ślepym nie kłádź záwády / ále śię bój Bogá twego : <i>JA PAN.</i>
Nie cżyń niepráwośći w ſądźie. Nie oględuj śię ná Oſobę ubogiego áni ƺánuj Oſoby bogátego : ſpráwiedliwie ſądź bliźniego twego.
Nie będźieƺ chodźił jáko obmówcá / miedzy ludem twojim : Nie będźieƺ ſtał o krew bliźniego twego : <i>JA PAN.</i>
Nie będźieƺ nienawidźiał brátá twego w ſercu twojim / jáwnie ſtrofowáć będźieƺ bliźniego twego / á nie zćierpiƺ przy nim grzechu.
Nie mśći śię y nie chowaj gniewu przećiw Synom ludu twego / ále miłuj bliźniego twego / jáko śiebie ſámego : <i>JA PAN.</i>
Uſtaw mojich przeſtrzegajćie : Bydlęćiá twego nie ſpuƺczaj z <i>bydlęty</i> rodzáju inƺego. Polá twego nie ośiewaj z mieƺánym naśieniem. Tákże ƺáty z różnych <i>rzecży</i> utkáney / jáko z wełny y ze lnu / nie obłócż ná śię.
Jeſliby mąż ſpał z niewiáſtą y obcował z nią / á oná będąc niewolnicą / byłáby mężowi poślubiona / á nie byłáby okupiona / áni wolnośćią dárowána / <i>oboje</i> będą karáni : ále nie ná gárdle ponieważ nie byłá wolno puƺczoną.
Y przywiedźie Ofiárę zá wyſtępek ſwój PAnu / do drzwi Namiotu zgromádzenia / báráná zá wyſtępek.
Tedy oczyśći go Kápłan przez onego báráná zá wyſtępek przed PAnem od grzechu jego / którym zgrzeƺył : á będźie mu odpuƺczony grzech jego / który popełnił.
Gdy też wnidźiećie do źiemie / á náƺczepićie wƺelákiego drzewá / rodzącego Owoc / tedy oberzniećie nieobrzeſkę jego : owoce jego / przez trzy látá / miejćie je zá nieobrzezánie <i>y</i> jeść go nie będźiećie.
Ale Roku cżwartego wƺyſtek owoc ich poświęcony będźie ná ofiárę chwały PANU.
( A piątego roku jeść będźiećie owoc jego / ) áby śię wam rozmnożył urodzaj jego : <i>JAM PAN /</i> Bóg wáƺ.
Nie jedzćie nic ze krwią : Nie bawćie śię wieƺczbámi áni czárámi.
Nie ſtrzyżćie w koło <i>włoſów</i> głowy wáƺey / áni brody ſwojey oƺpecajćie.
Dla umárłego nie rzeżćie ćiáłá wáƺego / áni <i>żadnego</i> piątná ná ſobie nie cżyńćie : <i>JAM PAN.</i>
Nie podaƺ ná ſplugáwienie córki twey / dopuƺczájąc jey wƺeteczeńſtwá / áby śię źiemiá nie ſplugáwiłá / y nie byłá nápełniona źiemiá ſproſnośćią.
Sábbáty moje záchowywajćie / á świątnicę moję w ucżćiwośći miejćie / <i>JA PAN.</i>
Nie udawajćie śię do cżárowników / áni do wieƺcżków / áni od nich rády ƺukajćie / ábyśćie śię od nich nie ſplugáwili : <i>JA PAN /</i> Bóg wáƺ.
Przed cżłowiekiem ſędźiwym powſtań / á czći oſobę ſtárego / y bój śię Bogá ſwego : <i>JA PAN.</i>
Będźieli mieƺkał z tobą przychodźień w źiemi wáƺey / nie cżyńćie mu krzywdy.
Jáko jeden z wáƺych domá zrodzonych / będźie u was przychodźień / który jeſt u was gośćiem y miłowáć go będźieƺ jáko ſam śiebie : bośćie <i>y wy</i> przychodniámi byli w źiemi Egipſkiey. <i>JA PAN,</i> Bóg wáƺ.
Nie czyńćie niepráwośći w ſądźie / w rozmierzániu / w wadze / y w mierze.
Szale ſpráwiedliwe / gwichty ſpráwiedliwe / korzec ſpráwiedliwy / y kwartę ſpráwiedliwą mieć będźiećie. <i>JA</i> jest <i>PAN,</i> Bóg wáƺ / którym was wywiódł z źiemie Egipſkiey.
Przetoż ſtrzeżćie wƺyſtkich uſtaw moich / y wƺyſtkich ſądów moich / á czyńćie je : <i>JA PAN.</i>
