Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / á powiedz im : Świętymi bądźćie / bom Já jeſt święty / Pán / Bóg wáƺ.
Káżdy mátki ſwojej y ojcá ſwego bójćie śię / á ſábátów moich przeſtrzegájćie ; Jám Pán / Bóg wáƺ.
Nie udáwájćie śię zá báłwány / á bogów litych nie cżyńćie ſobie ; Jám Pán / Bóg wáƺ.
A gdy ofiárowáć będziećie ofiárę ſpokojną Pánu / tedy z dobrej woli ſwej ofiárowáć ją będziećie.
W dzień / którego ofiárowáć będziećie / jedzćie ją y názájutrz ; ále coby zoſtáło áż do trzećiego dniá / ogniem ſpálone będzie.
A jeźlibyśćie to jedli dniá trzećiego / obrzydłe będzie / y nie przyjemne.
Ktobykolwiek to jádł / káránie zá niepráwość ſwoję ponieśie / bo świętość Páńſką ſplugáwił ; przetoż wytráconá będzie duƺá oná z ludu ſwego.
Gdy będziećie żąć zbożá ziemi wáƺej / nie będzieƺ do końcá polá twego wyrzynáł / áni pozoſtáłych kłoſów żniwá twego zbieráć będzieƺ.
Tákże winnicy twojej gron do ƺcżętu obieráć nie będzieƺ / áni jágód opádájących z winnicy twej nie pozbieráƺ ; ubogiemu y przychodniowi zoſtáwiƺ je ; Jám Pán / Bóg wáƺ.
Nie krádnijćie / áni zápierájćie / y nie oƺukiwájćie żáden bliźniego ſwego.
Nie przyśięgájćie fáłƺywie przez imię moje / y nie lżyj imieniá Bogá twego ; Jám Pán.
Nie ućiſkáj gwáłtem bliźniego twego / áni go odzieráj ; nie zoſtánie zápłátá nájemniká u ćiebie do jutrá.
Nie złorzecż głuchemu / á przed ślepym nie kłádź záwády / ále śię bój Bogá twego ; Jám Pán.
Nie cżyń niepráwośći w ſądzie. Nie oglądáj śię ná oſobę ubogiego / áni ƺánuj oſoby bogátego ; ſpráwiedliwie ſądź bliźniego twego.
Nie będzieƺ chodził jáko obmówcá między ludem twoim ; nie będzieƺ ſtáł o krew bliźniego twego ; Jám Pán.
Nie będzieƺ nienáwidziáł brátá twego w ſercu twojim ; jáwnie ſtrofowáć będzieƺ bliźniego twego / á nie śćierpiƺ przy nim grzechu.
Nie mśćij śię / y nie chowáj gniewu przećiw ſynom ludu twego ; ále miłuj bliźniego twego / jáko śiebie ſámego ; Jám Pán.
Uſtáw moich przeſtrzegájćie ; bydlęćiá twego nie ſpuƺcżáj z bydlęty rodzáju inƺego ; polá twego nie ośiewáj z mieƺánem náśieniem ; tákże ƺáty z różnych rzecży utkánej / jáko z wełny y ze lnu / nie obłócż ná śię.
Jeźliby mąż ſpáł z niewiáſtą / y obcowáł z nią / á oná będąc niewolnicą / byłáby mężowi poślubioná / á nie byłáby okupioná / áni wolnośćią dárowáná / oboje będą káráni ; ále nie ná gárdle / poniewáż nie byłá wolno puƺcżoną.
Y przywiedzie ofiárę zá wyſtępek ſwój Pánu do drzwi námiotu zgromádzeniá / báráná zá wyſtępek.
Tedy ocżyśći go kápłán przez onego báráná zá wyſtępek przed Pánem od grzechu jego / którym zgrzeƺył ; á będzie mu odpuƺcżony grzech jego / który popełnił.
Gdy też wnijdziećie do ziemi / á náƺcżepićie wƺelákiego drzewá rodzącego owoc / tedy oberzniećie nieobrzezkę jego / owoce jego ; przez trzy látá miejćie je zá nieobrzezánie / y jeść ich nie będziećie.
Ale roku cżwártego wƺyſtek owoc ich poświęcony będzie ná ofiárę chwáły Pánu.
A piątego roku jeść będziećie owoc jego / áby śię wám rozmnożył urodzáj jego ; Jám Pán / Bóg wáƺ.
Nie jedzćie nic ze krwią. Nie báwćie śię wieƺcżbámi / áni cżárámi.
Nie ſtrzyżćie w koło włoſów głowy wáƺej / áni brody ſwojej oƺpecájćie.
Dlá umárłego nie rzeżćie ćiáłá wáƺego / áni żádnego piątná ná ſobie nie cżyńćie ; Jám Pán.
Nie podáƺ ná ſplugáwienie córki twej / dopuƺcżájąc jej wƺetecżeńſtwá / áby śię ziemiá nie ſplugáwiłá / y nie byłá nápełnioná ziemiá ſproſnośćią.
Sábáty moje záchowywájćie / á świątnicę moję w ucżćiwośći miejćie ; Jám Pán.
Nie udáwájćie śię do cżárowników / áni do wieƺcżków / áni od nich rády ƺukájćie / ábyśćie śię od nich nie ſplugáwili ; Jám Pán / Bóg wáƺ.
Przed cżłowiekiem ſędziwym powſtáń / á cżćij oſobę ſtárego / y bój śię Bogá ſwego ; Jám Pán.
Będzieli mieƺkáł z tobą przychodzień w ziemi wáƺej / nie cżyńćie mu krzywdy ;
Jáko jeden z wáƺych w domu zrodzonych będzie u wáſ przychodzień / który jeſt u wáſ gośćiem / y miłowáć go będzieƺ jáko ſám śiebie ; bośćie y wy przychodniámi byli w ziemi Egipſkiej ; Jám Pán / Bóg wáƺ.
Nie cżyńćie niepráwośći w ſądzie ; w rozmierzániu / w wádze / y w mierze.
Szále ſpráwiedliwe / gwichty ſpráwiedliwe / korzec ſpráwiedliwy / y kwártę ſpráwiedliwą mieć będziećie ; Jám jeſt Pán / Bóg wáƺ / którym wáſ wywiódł z ziemi Egipſkiej.
Przetoż ſtrzeżćie wƺyſtkich uſtáw moich / y wƺyſtkich ſądów moich / á cżyńćie je ; Jám Pán.
