Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / á powiedz im : Świętymi bądźćie / bom Ja jeſt święty / Pán / Bóg wáƺ.
Káżdy mátki ſwojey y ojcá ſwego bójćie śię / á ſábátów moich przeſtrzegájćie ; Jám Pán / Bóg wáƺ.
Nie udáwájćie śię zá báłwány / á bogów litych nie cżyńćie ſobie ; Jám Pán / Bóg wáƺ.
A gdy ofiárowáć będźiećie ofiárę ſpokojną Pánu / tedy z dobrej woli ſwej ofiárowáć ją będźiećie.
W dźień / którego ofiárowáć będźiećie / jedzćie ją y názájutrz ; ále coby zoſtáło áż do trzećiego dniá / ogniem ſpálone będźie.
A jeſlibyśćie to jedli dniá trzećiego / obrzydłe będźie / y nie przyjemne.
Ktobykolwiek to jádł / káránie zá niepráwość ſwoję ponieśie / bo świętość Páńſką ſplugáwił ; przetoż wytráconá będźie duƺá oná z ludu ſwego.
Gdy będźiećie żąć zbożá źiemi wáƺej / nie będźieƺ do końcá polá twego wyrzynáł / áni pozoſtáłych kłoſów żniwá twego zbieráć będźieƺ.
Tákże winnicy twojey gron do ƺcżętu obieráć nie będźieƺ / áni jágód opádájących z winnicy twej nie pozbieráƺ ; ubogiemu y przychodniowi zoſtáwiƺ je ; Jám Pán / Bóg wáƺ.
Nie krádnijćie / áni zápierájćie / y nie oƺukiwájćie żáden bliźniego ſwego.
Nie przyśięgájćie fáłƺywie przez imię moje / y nie lżyj imieniá Bogá twego ; Jám Pán.
Nie ućiſkáj gwáłtem bliźniego twego / áni go odźieráj ; nie zoſtánie zápłátá nájemniká u ćiebie do jutrá.
Nie złorzecż głuchemu / á przed ślepym nie kłádź záwády / ále śię bój Bogá twego ; Jám Pán.
Nie cżyń niepráwośći w ſądźie. Nie oglądáj śię ná oſobę ubogiego / áni ƺánuj oſoby bogátego ; ſpráwiedliwie ſądź bliźniego twego.
Nie będźieƺ chodźił jáko obmówcá między ludem twoim ; nie będźieƺ ſtáł o krew bliźniego twego ; Jám Pán.
Nie będźieƺ nienáwidźiáł brátá twego w ſercu twojim ; jáwnie ſtrofowáć będźieƺ bliźniego twego / á nie śćierpiƺ przy nim grzechu.
Nie mśćij śię / y nie chowáj gniewu przećiw ſynom ludu twego ; ále miłuj bliźniego twego / jáko śiebie ſámego ; Jám Pán.
Uſtáw moich przeſtrzegájćie ; bydlęćiá twego nie ſpuƺcżáj z bydlęty rodzáju inƺego ; polá twego nie ośiewáj z mieƺánem náśieniem ; tákże ƺáty z różnych rzecży utkáney / jáko z wełny y ze lnu / nie obłócż ná śię.
Jeſliby mąż ſpáł z niewiáſtą / y obcowáł z nią / á oná będąc niewolnicą / byłáby mężowi poślubioná / á nie byłáby okupioná / áni wolnośćią dárowáná / oboje będą káráni ; ále nie ná gárdle / poniewáż nie byłá wolno puƺcżoną.
Y przywiedźie ofiárę zá wyſtępek ſwój Pánu do drzwi námiotu zgromádzeniá / báráná zá wyſtępek.
Tedy ocżyśći go kápłán przez onego báráná zá wyſtępek przed Pánem od grzechu jego / którym zgrzeƺył ; á będźie mu odpuƺcżony grzech jego / który popełnił.
Gdy też wnijdźiećie do źiemi / á náƺcżepićie wƺelákiego drzewá rodzącego owoc / tedy oberzniećie nieobrzezkę jego / owoce jego ; przez trzy látá miejćie je zá nieobrzezánie / y jeść ich nie będźiećie.
Ale roku cżwártego wƺyſtek owoc ich poświęcony będźie ná ofiárę chwáły Pánu.
A piątego roku jeść będźiećie owoc jego / áby śię wám rozmnożył urodzáj jego ; Jám Pán / Bóg wáƺ.
Nie jedzćie nic ze krwią. Nie báwćie śię wieƺcżbámi / áni cżárámi.
Nie ſtrzyżćie w koło włoſów głowy wáƺej / áni brody ſwojey oƺpecájćie.
Dla umárłego nie rzeżćie ćiáłá wáƺego / áni żádnego piątná ná ſobie nie cżyńćie ; Jám Pán.
Nie podáƺ ná ſplugáwienie córki twej / dopuƺcżájąc jey wƺetecżeńſtwá / áby śię źiemiá nie ſplugáwiłá / y nie byłá nápełnioná źiemiá ſproſnośćią.
Sábáty moje záchowywájćie / á świątnicę moję w ucżćiwośći miejćie ; Jám Pán.
Nie udáwájćie śię do cżárowników / áni do wieƺcżków / áni od nich rády ƺukájćie / ábyśćie śię od nich nie ſplugáwili ; Jám Pán / Bóg wáƺ.
Przed cżłowiekiem ſędźiwym powſtáń / á cżćij oſobę ſtárego / y bój śię Bogá ſwego ; Jám Pán.
Będźieli mieƺkáł z tobą przychodźień w źiemi wáƺej / nie cżyńćie mu krzywdy ;
Jáko jeden z wáƺych w domu zrodzonych będźie u was przychodźień / który jeſt u was gośćiem / y miłowáć go będźieƺ jáko ſám śiebie ; bośćie y wy przychodniámi byli w źiemi Egipſkiej ; Jám Pán / Bóg wáƺ.
Nie cżyńćie niepráwośći w ſądźie ; w rozmierzániu / w wádze / y w mierze.
Szále ſpráwiedliwe / gwichty ſpráwiedliwe / korzec ſpráwiedliwy / y kwártę ſpráwiedliwą mieć będźiećie ; Jám jeſt Pán / Bóg wáƺ / którym was wywiódł z źiemi Egipſkiej.
Przetoż ſtrzeżćie wƺyſtkich uſtáw moich / y wƺyſtkich ſądów moich / á cżyńćie je ; Jám Pán.
