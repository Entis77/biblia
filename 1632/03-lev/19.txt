Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do wƺyſtkiego zgromádzenia ſynów Izráelſkich / á powiedz im : Świętymi bądźćie / bom Ja jeſt święty / Pán / Bóg wáƺ.
Káżdy mátki ſwojey y ojcá ſwego bójćie śię / á ſábátów mojich przeſtrzegájćie ; Jam Pán / Bóg wáƺ.
Nie udáwájćie śię zá báłwány / á bogów litych nie cżyńćie ſobie ; Jam Pán / Bóg wáƺ.
A gdy ofiárowáć będźiećie ofiárę ſpokojną Pánu / tedy z dobrej woli ſwej ofiárowáć ją będźiećie.
W dźień / którego ofiárowáć będźiećie / jedzćie ją y názájutrz ; ále coby zoſtáło áż do trzećiego dniá / ogniem ſpálone będźie.
A jeſlibyśćie to jedli dniá trzećiego / obrzydłe będźie / y nie przyjemne.
Ktobykolwiek to jádł / káránie zá niepráwość ſwoję ponieśie / bo świętość Páńſką ſplugáwił ; przetoż wytráconá będźie duƺá oná z ludu ſwego.
Gdy będźiećie żąć zbożá źiemi wáƺej / nie będźieƺ do końcá polá twego wyrzynał / áni pozoſtáłych kłoſów żniwá twego zbieráć będźieƺ.
Tákże winnicy twojey gron do ƺcżętu obieráć nie będźieƺ / áni jágód opádájących z winnicy twej nie pozbieráƺ ; ubogiemu y przychodniowi zoſtáwiƺ je ; Jam Pán / Bóg wáƺ.
Nie krádnijćie / áni zápierájćie / y nie oƺukiwájćie żáden bliźniego ſwego.
Nie przyśięgájćie fáłƺywie przez imię moje / y nie lżyj imieniá Bogá twego ; Jam Pán.
Nie ućiſkáj gwáłtem bliźniego twego / áni go odźieráj ; nie zoſtánie zápłátá nájemniká u ćiebie do jutrá.
Nie złorzecż głuchemu / á przed ślepym nie kłádź záwády / ále śię bój Bogá twego ; Jam Pán.
Nie cżyń niepráwośći w ſądźie. Nie oglądáj śię ná oſobę ubogiego / áni ƺánuj oſoby bogátego ; ſpráwiedliwie ſądź bliźniego twego.
Nie będźieƺ chodźił jáko obmówcá miedzy ludem twoim ; nie będźieƺ ſtał o krew bliźniego twego ; Jam Pán.
Nie będźieƺ nienáwidźiał brátá twego w ſercu twojim ; jáwnie ſtrofowáć będźieƺ bliźniego twego / á nie śćierpiƺ przy nim grzechu.
Nie mśćij śię / y nie chowáj gniewu przećiw ſynom ludu twego ; ále miłuj bliźniego twego / jáko śiebie ſámego ; Jam Pán.
Uſtáw mojich przeſtrzegájćie ; bydlęćiá twego nie ſpuƺcżáj z bydlęty rodzáju inƺego ; polá twego nie ośiewáj z mieƺánem náśieniem ; tákże ƺáty z różnych rzecży utkáney / jáko z wełny y ze lnu / nie obłócż ná śię.
Jeſliby mąż ſpał z niewiáſtą / y obcował z nią / á oná będąc niewolnicą / byłáby mężowi poślubioná / á nie byłáby okupioná / áni wolnośćią dárowáná / oboje będą káráni ; ále nie ná gárdle / poniewáż nie byłá wolno puƺcżoną.
Y przywiedźie ofiárę zá wyſtępek ſwój Pánu do drzwi namiotu zgromádzenia / báráná zá wyſtępek.
Tedy ocżyśći go kápłán przez onego báráná zá wyſtępek przed Pánem od grzechu jego / którym zgrzeƺył ; á będźie mu odpuƺcżony grzech jego / który popełnił.
Gdy też wnijdźiećie do źiemi / á náƺcżepićie wƺelákiego drzewá rodzącego owoc / tedy oberzniećie nieobrzezkę jego / owoce jego ; przez trzy látá miejćie je zá nieobrzezánie / y jeść ich nie będźiećie.
Ale roku cżwártego wƺyſtek owoc ich poświęcony będźie ná ofiárę chwały Pánu.
A piątego roku jeść będźiećie owoc jego / áby śię wam rozmnożył urodzáj jego ; Jam Pán / Bóg wáƺ.
Nie jedzćie nic ze krwią. Nie báwćie śię wieƺcżbámi / áni cżárámi.
Nie ſtrzyżćie w koło włoſów głowy wáƺej / áni brody ſwojey oƺpecájćie.
Dla umárłego nie rzeżćie ćiáłá wáƺego / áni żádnego piątná ná ſobie nie cżyńćie ; Jam Pán.
Nie podáƺ ná ſplugáwienie córki twej / dopuƺcżájąc jey wƺetecżeńſtwá / áby śię źiemiá nie ſplugáwiłá / y nie byłá nápełnioná źiemiá ſproſnośćią.
Sábáty moje záchowywájćie / á świątnicę moję w ucżćiwośći miejćie ; Jam Pán.
Nie udáwájćie śię do cżárowników / áni do wieƺcżków / áni od nich rády ƺukájćie / ábyśćie śię od nich nie ſplugáwili ; Jam Pán / Bóg wáƺ.
Przed cżłowiekiem ſędźiwym powſtáń / á cżćij oſobę ſtárego / y bój śię Bogá ſwego ; Jam Pán.
Będźieli mieƺkał z tobą przychodźień w źiemi wáƺej / nie cżyńćie mu krzywdy ;
Jáko jeden z wáƺych w domu zrodzonych będźie u was przychodźień / który jeſt u was gośćiem / y miłowáć go będźieƺ jáko ſám śiebie ; bośćie y wy przychodniámi byli w źiemi Egipſkiej ; Jam Pán / Bóg wáƺ.
Nie cżyńćie niepráwośći w ſądźie ; w rozmierzániu / w wádze / y w mierze.
Szále ſpráwiedliwe / gwichty ſpráwiedliwe / korzec ſpráwiedliwy / y kwártę ſpráwiedliwą mieć będźiećie ; Jam jeſt Pán / Bóg wáƺ / którym was wywiódł z źiemi Egipſkiej.
Przetoż ſtrzeżćie wƺyſtkich uſtáw mojich / y wƺyſtkich ſądów mojich / á cżyńćie je ; Jam Pán.
