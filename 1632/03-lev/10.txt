Tedy ſynowie Aáronowi / Nádáb y Abiju / wźiąwƺy káżdy kádźielnicę ſwoję / włożyli w nią ogniá / y włożywƺy nań kádźidłá ofiárowáli przed oblicżem Páńſkim ogień obcy / cżego im był nie rozkázał.
Przetoż wyƺedƺy ogień od twárzy Páńſkiej / poráźił je ; y pomárli przed Pánem.
Zátem rzekł Mojzeƺ do Aároná : Toć to jeſt / co opowiedźiał Pán / mówiąc : W tych / którzy przyſtępują do mnie / poświęcony będę / y przed oblicżnośćią wƺyſtkiego ludu uwielbiony będę ; y zámilkł Aáron.
Tedy wezwał Mojzeƺ Miſáelá y Eliſáfáná / ſynów Huſyjelá / ſtryjá Aáronowego / y rzekł do nich : Przyſtąpćie / á wynieśćie bráćią wáƺę z świątnicy precż zá obóz.
Przyƺli tedy / á wynieśli je y z ƺátámi ich precż zá obóz / jáko był rozkázał Mojzeƺ.
Rzekł potem Mojzeƺ do Aároná y do Eleázárá y do Itámárá / ſynów jego : Głów wáƺych nie obnáżájćie / áni ƺát ſwych rozdźierájćie / byśćie nie pomárli / á Bóg nie rozgniewał śię ná wƺyſtko zgromádzenie. Ale bráćia wáśi / wƺyſtek dom ſynów Izráelſkich / niech płácżą tego ſpáleniá / które ucżynił Pán.
Ze drzwi też namiotu zgromádzenia nie wychodźćie / byśćie ſnadź nie pomárli ; ábowiem olejek pomázánia Páńſkiego jeſt ná was. Y ucżynili według rozkázánia Mojżeƺowego.
Zátem rzekł Pán do Aároná / mówiąc :
Winá y nápoju mocnego nie będźieƺ pił / ty y ſynowie twoi z tobą / gdy będźiećie mieli wchodźić do namiotu zgromádzenia / ábyśćie nie pomárli ; uſtáwá to wiecżná będźie w národźiech wáƺych ;
Abyśćie rozeznáwáć mogli miedzy rzecżą świętą / y miedzy rzecżą poſpolitą / y miedzy rzecżą niecżyſtą / y miedzy rzecżą cżyſtą ;
Ażebyśćie náucżáli ſynów Izráelſkich wƺyſtkich uſtáw / które im rozkázał Pán przez Mojżeƺá.
Mówił potem Mojzeƺ do Aároná y do Eleázárá y Itámárá / ſynów jego / którzy byli pozoſtáli : Weźmijćie ofiárę śniedną / która zoſtáłá od ogniſtych ofiár Páńſkich / á jedzćie ją z przáśnikámi przy ołtárzu ; bo rzecż nájświętƺá jeſt.
Przetoż jeść ją będźiećie ná miejſcu świętym / bo to práwo twoje / y práwo ſynów twojich / z ogniſtych ofiár Páńſkich ; bo mi ták rozkázáno.
Tákże moſtek obrácánia / y łopátkę podnoƺeniá będźiećie jedli ná miejſcu cżyſtem / ty y ſynowie twoi / y córki twoje z tobą ; ábowiem to práwem tobie / y práwem ſynom twoim dáno z ofiár ſpokojnych ſynów Izráelſkich.
Łopátkę podnoƺeniá / y moſtek obrácánia z ofiárámi ogniſtemi / y tłuſtośći przynioſą / áby je tám y ſám obrácáno przed oblicżem Páńſkim ; á to będźie tobie y ſynom twoim z tobą práwem wiecżnem / jáko rozkázał Pán.
Potem Mojzeƺ ƺukał pilnie kozłá ofiárowánego zá grzech / á oto / już ſpálony był ; y dla tego rozgniewáwƺy śię ná Eleázárá y Itámárá / ſyny Aáronowe / którzy byli pozoſtáli / mówił :
Przecżżeśćie nie jedli ofiáry zá grzech ná miejſcu świętym? ábowiem to jeſt rzecż nájświętƺá / ponieważ ją wam dáno / ábyśćie nośili niepráwość wƺyſtkiego ludu ná ocżyƺcżenie ich przed oblicżem Páńſkim.
A oto / nie jeſt wnieśioná krew jego wewnątrz do świątnicy ; mieliśćie go jeść w świątnicy / jákom rozkázał.
Tedy Aáron odpowiedźiał Mojżeƺowi : Oto / dźiś ofiárowáli ofiárę ſwoję zá grzech / y ofiárę cáłopalenia ſwego przed oblicżem Páńſkim / á oto mię ſpotkáło ; gdybym był jádł dźiś ofiárę zá grzech / izáliby śię to było podobáło Pánu?
To gdy uſłyƺał Mojzeƺ / przeſtał ná tym.
