Tedy ſynowie Aáronowi / Nádáb y Abiju / wziąwƺy káżdy kádzielnicę ſwoję / włożyli w nią ogniá / y włożywƺy náń kádzidłá ofiárowáli przed oblicżem Páńſkim ogień obcy / cżego im był nie rozkázáł.
Przetoż wyƺedłƺy ogień od twárzy Páńſkiej / poráził je ; y pomárli przed Pánem.
Zátem rzekł Mojżeƺ do Aároná : Toć to jeſt / co opowiedziáł Pán / mówiąc : W tych / którzy przyſtępują do mnie / poświęcony będę / y przed oblicżnośćią wƺyſtkiego ludu uwielbiony będę ; y zámilkł Aáron.
Tedy wezwáł Mojżeƺ Miſáelá y Eliſáfáná / ſynów Huſyjelá / ſtryjá Aáronowego / y rzekł do nich : Przyſtąpćie / á wynieśćie bráćią wáƺę z świątnicy precż zá obóz.
Przyƺli tedy / á wynieśli je y z ƺátámi ich precż zá obóz / jáko był rozkázáł Mojżeƺ.
Rzekł potem Mojżeƺ do Aároná y do Eleázárá y do Itámárá / ſynów jego : Głów wáƺych nie obnáżájćie / áni ƺát ſwych rozdzierájćie / byśćie nie pomárli / á Bóg nie rozgniewáł śię ná wƺyſtko zgromádzenie. Ale bráćiá wáśi / wƺyſtek dom ſynów Izráelſkich / niech płácżą tego ſpáleniá / które ucżynił Pán.
Ze drzwi też námiotu zgromádzeniá nie wychodźćie / byśćie ſnáć nie pomárli ; ábowiem olejek pomázániá Páńſkiego jeſt ná wáſ. Y ucżynili według rozkázániá Mojżeƺowego.
Zátem rzekł Pán do Aároná / mówiąc :
Winá y nápoju mocnego nie będzieƺ pił / ty y ſynowie twoi z tobą / gdy będziećie mieli wchodzić do námiotu zgromádzeniá / ábyśćie nie pomárli ; uſtáwá to wiecżná będzie w národziech wáƺych ;
Abyśćie rozeznáwáć mogli między rzecżą świętą / y między rzecżą poſpolitą / y między rzecżą niecżyſtą / y między rzecżą cżyſtą ;
Ażebyśćie náucżáli ſynów Izráelſkich wƺyſtkich uſtáw / które im rozkázáł Pán przez Mojżeƺá.
Mówił potem Mojżeƺ do Aároná y do Eleázárá y Itámárá / ſynów jego / którzy byli pozoſtáli : Weźmijćie ofiárę śniedną / którá zoſtáłá od ogniſtych ofiár Páńſkich / á jedzćie ją z przáśnikámi przy ołtárzu ; bo rzecż nájświętƺá jeſt.
Przetoż jeść ją będziećie ná miejſcu świętym / bo to práwo twoje / y práwo ſynów twoich / z ogniſtych ofiár Páńſkich ; bo mi ták rozkázáno.
Tákże moſtek obrácániá / y łopátkę podnoƺeniá będziećie jedli ná miejſcu cżyſtem / ty y ſynowie twoi / y córki twoje z tobą ; ábowiem to práwem tobie / y práwem ſynom twoim dáno z ofiár ſpokojnych ſynów Izráelſkich.
Łopátkę podnoƺeniá / y moſtek obrácániá z ofiárámi ogniſtemi / y tłuſtośći przynioſą / áby je tám y ſám obrácáno przed oblicżem Páńſkim ; á to będzie tobie y ſynom twoim z tobą práwem wiecżnem / jáko rozkázáł Pán.
Potem Mojżeƺ ƺukáł pilnie kozłá ofiárowánego zá grzech / á oto / już ſpálony był ; y dlá tego rozgniewáwƺy śię ná Eleázárá y Itámárá / ſyny Aáronowe / którzy byli pozoſtáli / mówił :
Przecżżeśćie nie jedli ofiáry zá grzech ná miejſcu świętym? ábowiem to jeſt rzecż nájświętƺá / poniewáż ją wám dáno / ábyśćie nośili niepráwość wƺyſtkiego ludu ná ocżyƺcżenie ich przed oblicżem Páńſkim.
A oto / nie jeſt wnieśioná krew jego wewnątrz do świątnicy ; mieliśćie go jeść w świątnicy / jákom rozkázáł.
Tedy Aáron odpowiedziáł Mojżeƺowi : Oto / dziś ofiárowáli ofiárę ſwoję zá grzech / y ofiárę cáłopáleniá ſwego przed oblicżem Páńſkim / á oto mię ſpotkáło ; gdybym był jádł dziś ofiárę zá grzech / izáliby śię to było podobáło Pánu?
To gdy uſłyƺáł Mojżeƺ / przeſtáł ná tem.
