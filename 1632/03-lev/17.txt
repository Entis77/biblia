Tedy rzekł PAN do Mojzeƺá mówiąc :
Mów do Aároná y do Synów jego / y do wƺyſtkich Synów Izráelſkich / á powiedz im : Táć jeſt rzecz którą przykazał PAN / mówiąc :
Ktobykolwiek z domu Izráelſkiego zábił wołu / álbo Owcę / álbo kozę w oboźie : álbo ktoby zábił zá obozem :
A do drzwi Namiotu zgromádzenia / nie przywiódłby tego / áby Ofiárował ofiárę PAnu / przed Przybytkiem PAńſkim / krwie winien będźie on mąż / krew przelał / przetoż wytrácony będźie on mąż / z pośrzodku ludu ſwego.
Synowie tedy Izráelſcy przywiodą ofiáry ſwoje / które zábijáli ná polu / przywiodą je PAnu do drzwi Namiotu zgromádzenia / do Kápłaná / á ták niechaj ſpráwują ofiáry ſpokojne PAnu.
Y wyleje Kápłan krew ná Ołtarz PAńſki / u drzwi Namiotu zgromádzenia / á ſpali tłuſtość ku wdźięcżney wonnośći PAnu.
Y nie będą ofiárowáć więcey ofiar ſwych Diabłom / z któremi cudzołożyli / tá uſtáwá wiecżná będźie im w narodźiech ich.
Nád to im jeƺcze powiedz / Jesliby kto z domu Izráelſkiego / álbo z przychodniów miedzy wámi mieƺkájących chćiał ofiárowáć ofiárę cáłopalenia <i>álbo inƺą</i> Ofiárę :
A do drzwi Namiotu zgromádzenia nie przywiódłby jey / áby ją ofiárował PANU : wytrácony będźie cżłowiek ten / z ludu ſwego.
A ktobykolwiek z domu Izráelſkiego / álbo z przychodniów którzyby gośćiámi byli miedzy nimi / jadł krew jáką / poſtáwię <i>rozgniewaną</i> twarz ſwą prżećiwko cżłowiekowi krew jedzącemu / y wygłádzę go z pośrzodku ludu jego :
Abowiem duƺá <i>wƺelkiego</i> ćiáłá / we krwi jego jeſt / á ja dałem ją wam ná Ołtarz / ku oczyśćieniu duƺ wáƺych / bo krew jeſt która duƺę oczyśćia.
Dla tegoż powiedźiałem Synom Izráelſkim / żaden miedzy wámi nie będźie jadał krwie / áni przychodźień który gośćiem jeſt miedzy wámi / nie będźie jadał krwie.
Y ktobykolwiek z Synów Izráelſkich / álbo z przychodniów / którzy ſą gośćmi miedzy wámi goniąc ułowił jákie zwierzę / álbo ptaká / co śię godźi jeść / tedy krew z niego wypuśći / y záſypie ją piaſkiem.
Bo duƺá káżdego ćiáłá <i>jeſt</i> krew jego / która <i>jeſt</i> miáſto duƺe jego : przetożem powiedźiał Synom Izráelſkim / krwie wƺelkiego ćiáłá jeść nie będźiećie / bo duƺá wƺelkiego ćiáłá / <i>jeſt</i> krew jego : ktoby ją kolwiek jadł / wytrácony będźie.
Jeſliby też kto jadł co zdechłego / álbo <i>od zwierza</i> rozƺárpánego / ták w domá zrodzony / jáko przychodźień / tedy upierze ƺáty ſwoje / y omyje śię wodą / á nieczyſtym będźie áż do wiecżorá : potym czyſty będźie.
Ale jeſliby nie uprał <i>ƺát ſwoich,</i> á ćiáłá ſwego nie omył / ponieśie nie práwość ſwoję.
