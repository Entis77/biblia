Y wezwał Pán Mojżeƺá / y rzekł do niego z namiotu zgromádzenia / mówiąc :
Mów do ſynów Izráelſkich / á rzecż im : Gdyby kto z was ofiárował ofiárę Pánu / z bydłá / z wołów / y z drobnego bydłá / ofiárowáć będźiećie ofiárę wáƺę.
Jeſli cáłopáloná ofiárá jego będźie z rogátego bydłá / ſámcá zupełnego ofiárowáć będźie ; u drzwi namiotu zgromádzenia ofiárowáć go będźie dobrowolnie przed oblicżem Páńſkim.
Y położy rękę ſwą ná głowę ofiáry cáłopalenia / á będźie przyjemną záń ná ocżyƺcżenie jego.
Zábije tedy ćielcá tego kápłán przed oblicżnośćią Páńſką ; á ſynowie Aáronowi / kápłáni / ofiárowáć będą krew / á pokropią tą krwią ołtárz z wierzchu w około / który jeſt przede drzwiámi namiotu zgromádzenia.
A wźiąwƺy ſkórę z ofiáry cáłopalenia rozrąbi ją ná ƺtuki.
Potem náłożą ſynowie Aároná kápłáná / ogień ná ołtárzu / á ułożą drwá ná ogień.
Potem porządnie włożą ſynowie Aáronowi / kápłáni / one ƺtuki / głowę / y tłuſtość / ná drwá / które ſą ná ogniu / który jeſt ná ołtárzu.
A wnętrznośći jego / y nogi jego / opłócże wodą / y zápáli kápłán to wƺyſtko ná ołtárzu ; cáłopálenie jeſt ofiáry ogniſtey ku wdźięcżney wonnośći Pánu.
A jeſliżby z drobnego bydłá kto chćiał ofiárowáć z owiec álbo z kóz / ná ofiárę cáłopalenia / ſámcá zupełnego ofiárowáć będźie ;
Y zábije go po bok ołtárzá ku północy przed oblicżnośćią Páńſką ; á pokropią ſynowie Aáronowi / kápłáni / krwią jego po wierzchu ołtárzá w około.
Y rozrąbie go ná ƺtuki / y głowę jego / y tłuſtość jego ; á włoży je kápłán porządnie ná drwá / które ſą ná ogniu / który jeſt ná ołtárzu.
A wnętrznośći y nogi opłócże wodą ; y będźie ofiárował kápłán to wƺyſtko / y zápáli to ná ołtárzu. Cáłopálenie jeſt ofiáry ogniſtey ku wdźięcżney wonnośći Pánu.
A jeſliby z ptáſtwá cáłopalenia ofiárę chćiał kto ofiárowáć Pánu / tedy niech przynieśie z ſynogárlic / álbo z gołąbiąt ofiárę ſwoję.
A będźie ją ofiárował kápłán ná ołtárzu / y páznogćiem nádrze głowę jego / y zápáli ná ołtárzu / wyćiſnąwƺy krew jego ná ſtronie ołtárzá.
Odejmie też gárdźiel jego z pierzem jego / á porzući je bliſko ołtárzá ku wſchodniej ſtronie / ná miejſce / gdźie popiół bywa ;
Y rozedrze mu ſkrzydłá jego / á wƺákże ich nie oderwie ; y ſpáli to kápłán ná ołtárzu / ná drwách / które ſą ná ogniu ; cáłopálenie jeſt ofiáry ogniſtey ku wdźięcżney wonnośći Pánu.
