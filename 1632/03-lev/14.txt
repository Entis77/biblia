Potem rzekł Pán do Mojżeƺá / mówiąc :
Táć jeſt uſtáwá około trędowátego w dzień ocżyƺcżeniá jego : przywiedziony będzie do kápłáná.
A wynijdzie kápłán precż zá obóz : á obácżyli kápłán / że oto ulecżoná jeſt zárázá trądu / trądem záráżonego /
Tedy rozkáże kápłán temu / który śię ocżyƺcżá / áby wziął dwá wróble żywe y zdrowe / y drzewo cedrowe / y jedwábiu kármázynowego / y hizopu.
Y rozkáże kápłán zábić jednego wróblá nád nácżyniem gliniánem / nád wodą żywą.
Wróblá tedy żywego weźmie / y drzewo cedrowe / y jedwáb kármázynowy y hizop / á omocży to wƺyſtko z wróblem żywym we krwi wróblá zábitego nád wodą żywą.
Y pokropi tego / który śię ocżyƺcżá od trądu / śiedm kroć / y ogłośi go być cżyſtym / á puśći wróblá żywego w pole.
A ten / który śię ocżyƺcżá / upierze ƺáty ſwoje / y ogoli wƺyſtkie włoſy ſwoje / á umyje śię wodą / y cżyſtym będzie. Potem wnijdzie do obozu / á będzie mieƺkáł przed námiotem ſwoim przez śiedm dni.
Potem dniá śiódmego ogoli wƺyſtkie włoſy ſwe / głowę ſwą / y brodę ſwą / y brwi nád ocżymá ſwemi / y wƺyſtkie inne włoſy ſwe ogoli ; przytem upierze ƺáty ſwe / y ćiáło ſwe omyje wodą / á ták ocżyƺcżon będzie.
A dniá óſmego weźmie dwu báránków zupełnych / y owcę jednę rocżną / zupełną / y trzydzieśiąte cżęśći efy mąki pƺennej ná ofiárę śniedną / zmieƺáną z oliwą / y miárkę oliwy /
Tedy kápłán / który ocżyƺcżá cżłowieká / który má być ocżyƺcżony / poſtáwi z temi rzecżámi przed Pánem / u drzwi námiotu zgromádzeniá.
Potem weźmie kápłán báráná jednego / y będzie go ofiárowáł ná ofiárę zá wyſtępek / z oną miárką oliwy / y będzie to tám y ſám obrácáł ná ofiárę obrácániá przed oblicżem Páńſkim.
Zábije też báránká onego ná miejſcu / gdzie biją ofiáry zá grzech y ofiárę cáłopáleniá / ná miejſcu świętym ; bo jáko ofiárá zá grzech ták ofiárá zá wyſtępek náleży kápłánowi ; rzecż nájświętƺá jeſt.
Y weźmie kápłán krwi z ofiáry zá wyſtępek / y pomáże kápłán koniec uchá práwego onemu / który śię ocżyƺcżá ; tákże pálec wielki u práwej ręki jego y pálec wielki u práwej nogi jego.
Weźmie też kápłán z onej miárki oliwy / á náleje ná dłoń ſwoję lewą ;
A omocży pálec ſwój práwy w oliwie / którá jeſt ná lewej dłoni jego / y pokropi oliwą z pálcá ſwego śiedm kroć przed oblicżem Páńſkim.
A z oſtátku oliwy / którá jeſt ná dłoni jego / pomáże kápłán koniec uchá práwego onemu / który śię ocżyƺcżá / y wielki pálec práwej ręki jego / tákże wielki pálec práwej nogi jego z onejże krwi / którá jeſt ofiárą zá wyſtępek.
A coby zoſtáło oliwy / którá jeſt ná dłoni kápłánowej / pomáże tem głowę onego / który śię ocżyƺcżá ; y ták go ocżyśći kápłán przed oblicżem Páńſkim.
Ucżyni tákże kápłán ofiárę zá grzech / y ocżyśći tego / który śię ocżyƺcżá / od niecżyſtośći jego / á potem zábije ofiárę cáłopáleniá.
Y ofiárowáć będzie kápłán ofiárę cáłopáleniá / y ofiárę śniedną ná ołtárzu ; ták ocżyśći go kápłán / y cżyſtym będzie.
A jeźliby kto był ták ubogi / iżby tego nie przemógł / tedy weźmie báránká jednego ná ofiárę zá wyſtępek ná podnoƺenie dlá ocżyƺcżeniá ſwego / y jednę dzieśiątą cżęść efy mąki pƺennej zágniećionej z oliwą ná ofiárę śniedną y miárkę oliwy.
Nád to dwie ſynogárlice / álbo dwoje gołąbiąt / cżego doſtáć może / z których jedno będzie ná ofiárę zá grzech / á drugie ná ofiárę cáłopáleniá ;
Y przynieśie je w óſmy dzień ocżyƺcżeniá ſwego do kápłáná / do drzwi námiotu zgromádzeniá / przed oblicżność Páńſką.
Weźmie tedy kápłán báránká ofiáry zá wyſtępek / y miárkę oliwy ; y będzie to obrácáł tám y ſám kápłán ná ofiárę obrácániá przed Pánem.
A zábije báránká ná ofiárę zá wyſtępek : á wziąwƺy kápłán ze krwi ofiáry zá wyſtępek / pomáże koniec uchá práwego temu / który śię ocżyƺcżá ; y pálec wielki práwej ręki jego / y pálec wielki práwej nogi jego.
Oliwy tákże náleje kápłán ná lewą dłoń ſwoję.
Y kropić będzie kápłán pálcem ſwoim práwym z oliwy / którá jeſt ná lewej ręce jego / śiedm kroć przed oblicżem Páńſkim.
Pomáże też kápłán oną oliwą / którá jeſt ná dłoni jego / koniec uchá práwego temu / który śię ocżyƺcżá ; tákże wielki pálec práwej ręki jego / y wielki pálec práwej nogi jego ná miejſcu krwi z ofiáry zá wyſtępek ;
A oſtátkiem oliwy / którá jeſt ná dłoni kápłáná / pomáże głowę onego / który śię ocżyƺcżá / áby go ocżyśćił przed Pánem.
Tákże ucżyni z jedną ſynogárlicą / álbo z jednem gołębięćiem / cżegokolwiek z tych doſtáć może.
Cżego doſtáć mógł / jedno z tych będzie ofiárą zá grzech / á drugie ná ofiárę cáłopáleniá z ofiárą śniedną ; á ták ocżyśći kápłán tego / który śię ocżyƺcżá przed oblicżem Páńſkim.
A táć jeſt uſtáwá o tym / ná którymby byłá zárázá trądu / który wƺyſtkiego mieć nie może ku ocżyƺcżeniu ſwemu.
Rzekł potem Pán do Mojżeƺá y do Aároná / mówiąc :
Gdy wnijdziećie do ziemi Chánánejſkiej / którą Já wám dáwám w ośiádłość / á dopuśćiłbym zárázę trądu / ná który dom ośiádłośći wáƺej :
Tedy on / którego dom jeſt / przyjdzie y opowie to kápłánowi / mówiąc : Jákoby zárázá trądu zdá mi śię być w domu moim /
Rozkáże tedy kápłán wyprzątnąć dom / pierwej niż ſám wnijdzie / áby oglądáł zárázę onę / iżby śię nic nie ſplugáwiło / coby było w domu / á potem kápłán wnijdzie / áby oglądáł on dom.
A oglądájąc onę zárázę / ujrzyli zárázę ná śćiánie domu / jákoby dołki cżárne / przyzieleńƺym / álbo przycżerwieńƺym / á ná ſpojrzeniu byłoby głębƺe niż śćiáná.
Tedy wynijdzie kápłán z domu onego przede drzwi / y zámknie on dom przez śiedm dni.
Wróći śię potem kápłán dniá śiódmego y obejrzy ; á jeźli śię rozƺerzyłá zárázá ná śćiánách domu onego /
Rozkáże kápłán wyłámáć ono kámienie / ná któremby byłá zárázá / y wyrzućić je precż zá miáſto ná miejſce niecżyſte ;
A dom rozkáże wewnątrz oſkrobáć wƺędy w około ; y wyrzucą on proch który oſkrobáli / precż zá miáſto ná miejſce niecżyſte ;
Y wezmą kámienie inƺe / y wpráwią ná miejſce innych kámieni ; y wápná też inƺego wezmą á potynkują dom.
A jeźliby śię odnowiłá oná zárázá / y rozƺerzyłá śię po domu po wyrzuceniu kámieniá / y po wyſkrobániu domu y po tynkowániu jego :
Tedy wnijdzie kápłán ; á ujrzyli / że śię rozƺerzyłá oná zárázá po domu / trąd jeſt jádowity w domu onym / niecżyſty jeſt.
Zátem rozwálą on dom / kámienie jego / y drzewo jego y wƺyſtko wápno domu onego / á wynioſą precż zá miáſto ná miejſce niecżyſte.
A ten / ktoby wƺedł do domu onego / po wƺyſtkie dni / póki był záwárty / niecżyſtym będzie áż do wiecżorá.
A ktoby ſpáł w onym domu / upierze ƺáty ſwoje ; tákże ktoby jádł w tymże domu / upierze ƺáty ſwoje.
Lecż jeźliby wyƺedłƺy kápłán obácżył / iż śię nie ƺerzy zárázá po domu po tynkowániu jego / tedy oſądzi kápłán / że dom on jeſt cżyſty ; bo ulecżoná jeſt zárázá oná.
A weźmie ná ocżyƺcżenie onego domu dwu wróblów / y drzewo cedrowe / y jedwábiu / kármázynu / y hizopu ;
Y zábije wróblá jednego nád nácżyniem gliniánem / nád wodą żywą ;
A wziąwƺy drzewo cedrowe y hizop / y jedwáb kármázynowy / y wróblá żywego / omocży to wƺyſtko we krwi wróblá zábitego y w wodzie żywej / á pokropi ten dom śiedm kroć.
Y ták ocżyśći on dom krwią wróblá onego / y wodą żywą y wróblem żywym y drzewem cedrowem / y hizopem / y jedwábiem cżerwonym.
Potem puśći wróblá żywego precż zá miáſto w pole ; ták ocżyśći on dom / y cżyſtym będzie.
Táć jeſt uſtáwá o káżdej zárázie trądu / y plámy cżárnej ;
Y o trądzie ná ƺáćie y ná domu ;
Y o ſádzelu / y o świerzbie / y o biáłej plámie /
Żeby poznáć / gdy kto jeſt niecżyſtym / y gdy kto cżyſtym. Táć jeſt uſtáwá około trądu.
