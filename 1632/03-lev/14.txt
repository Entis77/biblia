Potem rzekł Pán do Mojżeƺá / mówiąc :
Táć jeſt uſtáwá około trędowátego w dźień ocżyƺcżeniá jego : przywiedźiony będźie do kápłáná.
A wynijdźie kápłán precż zá obóz : á obácżyli kápłán / że oto ulecżoná jeſt zárázá trądu / trądem záráżonego /
Tedy rozkáże kápłán temu / który śię ocżyƺcżá / áby wźiął dwá wróble żywe y zdrowe / y drzewo cedrowe / y jedwábiu kármázynowego / y hizopu.
Y rozkáże kápłán zábić jednego wróblá nád nácżyniem gliniánem / nád wodą żywą.
Wróblá tedy żywego weźmie / y drzewo cedrowe / y jedwáb kármázynowy y hizop / á omocży to wƺyſtko z wróblem żywym we krwi wróblá zábitego nád wodą żywą.
Y pokropi tego / który śię ocżyƺcżá od trądu / śiedm kroć / y ogłośi go być cżyſtym / á puśći wróblá żywego w pole.
A ten / który śię ocżyƺcżá / upierze ƺáty ſwoje / y ogoli wƺyſtkie włoſy ſwoje / á umyje śię wodą / y cżyſtym będźie. Potem wnijdźie do obozu / á będźie mieƺkał przed námiotem ſwoim przez śiedm dni.
Potem dniá śiódmego ogoli wƺyſtkie włoſy ſwe / głowę ſwą / y brodę ſwą / y brwi nád ocżymá ſwemi / y wƺyſtkie inne włoſy ſwe ogoli ; przytem upierze ƺáty ſwe / y ćiáło ſwe omyje wodą / á ták ocżyƺcżon będźie.
A dniá óſmego weźmie dwu báránków zupełnych / y owcę jednę rocżną / zupełną / y trzydźieśiąte cżęśći efy mąki pƺenney ná ofiárę śniedną / zmieƺáną z oliwą / y miárkę oliwy /
Tedy kápłán / który ocżyƺcżá cżłowieká / który ma być ocżyƺcżony / poſtáwi z temi rzecżámi przed Pánem / u drzwi namiotu zgromádzenia.
Potem weźmie kápłán báráná jednego / y będźie go ofiárował ná ofiárę zá wyſtępek / z oną miárką oliwy / y będźie to tám y ſám obrácał ná ofiárę obrácánia przed oblicżem Páńſkim.
Zábije też báránká onego ná miejſcu / gdźie biją ofiáry zá grzech y ofiárę cáłopalenia / ná miejſcu świętym ; bo jáko ofiárá zá grzech ták ofiárá zá wyſtępek náleży kápłánowi ; rzecż nájświętƺá jeſt.
Y weźmie kápłán krwi z ofiáry zá wyſtępek / y pomáże kápłán koniec uchá práwego onemu / który śię ocżyƺcżá ; tákże pálec wielki u práwej ręki jego y pálec wielki u práwej nogi jego.
Weźmie też kápłán z oney miárki oliwy / á náleje ná dłoń ſwoję lewą ;
A omocży pálec ſwój práwy w oliwie / która jeſt ná lewej dłoni jego / y pokropi oliwą z pálcá ſwego śiedm kroć przed oblicżem Páńſkim.
A z oſtátku oliwy / która jeſt ná dłoni jego / pomáże kápłán koniec uchá práwego onemu / który śię ocżyƺcżá / y wielki pálec práwej ręki jego / tákże wielki pálec práwej nogi jego z onejże krwi / która jeſt ofiárą zá wyſtępek.
A coby zoſtáło oliwy / która jeſt ná dłoni kápłánowej / pomáże tym głowę onego / który śię ocżyƺcżá ; y ták go ocżyśći kápłán przed oblicżem Páńſkim.
Ucżyni tákże kápłán ofiárę zá grzech / y ocżyśći tego / który śię ocżyƺcżá / od niecżyſtośći jego / á potem zábije ofiárę cáłopalenia.
Y ofiárowáć będźie kápłán ofiárę cáłopalenia / y ofiárę śniedną ná ołtárzu ; ták ocżyśći go kápłán / y cżyſtym będźie.
A jeſliby kto był ták ubogi / iżby tego nie przemógł / tedy weźmie báránká jednego ná ofiárę zá wyſtępek ná podnoƺenie dla ocżyƺcżeniá ſwego / y jednę dźieśiątą cżęść efy mąki pƺenney zágniećioney z oliwą ná ofiárę śniedną y miárkę oliwy.
Nád to dwie ſynogárlice / álbo dwoje gołąbiąt / cżego doſtáć może / z których jedno będźie ná ofiárę zá grzech / á drugie ná ofiárę cáłopalenia ;
Y przynieśie je w óſmy dźień ocżyƺcżeniá ſwego do kápłáná / do drzwi namiotu zgromádzenia / przed oblicżność Páńſką.
Weźmie tedy kápłán báránká ofiáry zá wyſtępek / y miárkę oliwy ; y będźie to obrácał tám y ſám kápłán ná ofiárę obrácánia przed Pánem.
A zábije báránká ná ofiárę zá wyſtępek : á wźiąwƺy kápłán ze krwi ofiáry zá wyſtępek / pomáże koniec uchá práwego temu / który śię ocżyƺcżá ; y pálec wielki práwej ręki jego / y pálec wielki práwej nogi jego.
Oliwy tákże náleje kápłán ná lewą dłoń ſwoję.
Y kropić będźie kápłán pálcem ſwoim práwym z oliwy / która jeſt ná lewej ręce jego / śiedm kroć przed oblicżem Páńſkim.
Pomáże też kápłán oną oliwą / która jeſt ná dłoni jego / koniec uchá práwego temu / który śię ocżyƺcżá ; tákże wielki pálec práwej ręki jego / y wielki pálec práwej nogi jego ná miejſcu krwi z ofiáry zá wyſtępek ;
A oſtátkiem oliwy / która jeſt ná dłoni kápłáná / pomáże głowę onego / który śię ocżyƺcżá / áby go ocżyśćił przed Pánem.
Tákże ucżyni z jedną ſynogárlicą / álbo z jednem gołębięćiem / cżegokolwiek z tych doſtáć może.
Cżego doſtáć mógł / jedno z tych będźie ofiárą zá grzech / á drugie ná ofiárę cáłopalenia z ofiárą śniedną ; á ták ocżyśći kápłán tego / który śię ocżyƺcżá przed oblicżem Páńſkim.
A táć jeſt uſtáwá o tym / ná którymby byłá zárázá trądu / który wƺyſtkiego mieć nie może ku ocżyƺcżeniu ſwemu.
Rzekł potem Pán do Mojżeƺá y do Aároná / mówiąc :
Gdy wnijdźiećie do źiemi Chánánejſkiej / którą Ja wam dáwám w ośiádłość / á dopuśćiłbym zárázę trądu / ná który dom ośiádłośći wáƺej :
Tedy on / którego dom jeſt / przydźie y opowie to kápłánowi / mówiąc : Jákoby zárázá trądu zda mi śię być w domu moim /
Rozkáże tedy kápłán wyprzątnąć dom / pierwej niż ſám wnijdźie / áby oglądał zárázę onę / iżby śię nic nie ſplugáwiło / coby było w domu / á potem kápłán wnijdźie / áby oglądał on dom.
A oglądájąc onę zárázę / ujrzyli zárázę ná śćiánie domu / jákoby dołki cżárne / przyźieleńƺym / álbo przycżerwieńƺym / á ná ſpojrzeniu byłoby głębƺe niż śćiáná.
Tedy wynijdźie kápłán z domu onego przede drzwi / y zámknie on dom przez śiedm dni.
Wróći śię potem kápłán dniá śiódmego y obejrzy ; á jeſli śię rozƺerzyłá zárázá ná śćiánách domu onego /
Rozkáże kápłán wyłámáć ono kámienie / ná któremby byłá zárázá / y wyrzućić je precż zá miáſto ná miejſce niecżyſte ;
A dom rozkáże wewnątrz oſkrobáć wƺędy w około ; y wyrzucą on proch który oſkrobáli / precż zá miáſto ná miejſce niecżyſte ;
Y wezmą kámienie inƺe / y wpráwią ná miejſce innych kámieni ; y wápná też inƺego wezmą á potynkują dom.
A jeſliby śię odnowiłá oná zárázá / y rozƺerzyłá śię po domu po wyrzuceniu kámieniá / y po wyſkrobániu domu y po tynkowániu jego :
Tedy wnijdźie kápłán ; á ujrzyli / że śię rozƺerzyłá oná zárázá po domu / trąd jeſt jádowity w domu onym / niecżyſty jeſt.
Zátem rozwálą on dom / kámienie jego / y drzewo jego y wƺyſtko wápno domu onego / á wynioſą precż zá miáſto ná miejſce niecżyſte.
A ten / ktoby wƺedł do domu onego / po wƺyſtkie dni / póki był záwárty / niecżyſtym będźie áż do wiecżorá.
A ktoby ſpał w onym domu / upierze ƺáty ſwoje ; tákże ktoby jádł w tymże domu / upierze ƺáty ſwoje.
Lecż jeſliby wyƺedƺy kápłán obácżył / iż śię nie ƺerzy zárázá po domu po tynkowániu jego / tedy oſądźi kápłán / że dom on jeſt cżyſty ; bo ulecżoná jeſt zárázá oná.
A weźmie ná ocżyƺcżenie onego domu dwu wróblów / y drzewo cedrowe / y jedwábiu / kármázynu / y hizopu ;
Y zábije wróblá jednego nád nácżyniem gliniánem / nád wodą żywą ;
A wźiąwƺy drzewo cedrowe y hizop / y jedwáb kármázynowy / y wróblá żywego / omocży to wƺyſtko we krwi wróblá zábitego y w wodźie żywej / á pokropi ten dom śiedm kroć.
Y ták ocżyśći on dom krwią wróblá onego / y wodą żywą y wróblem żywym y drzewem cedrowem / y hizopem / y jedwábiem cżerwonym.
Potem puśći wróblá żywego precż zá miáſto w pole ; ták ocżyśći on dom / y cżyſtym będźie.
Táć jeſt uſtáwá o káżdej záráźie trądu / y plámy cżárney ;
Y o trądźie ná ƺáćie y ná domu ;
Y o ſádzelu / y o świerzbie / y o biáłej plámie /
Żeby poznáć / gdy kto jeſt niecżyſtym / y gdy kto cżyſtym. Táć jeſt uſtáwá około trądu.
