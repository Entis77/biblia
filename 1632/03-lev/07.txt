Táć jeſt uſtáwá ofiáry zá wyſtępek <i>która jeſt</i> Naświętƺa.
Ná miejſcu gdźie biją Ofiáry cáłopalenia / zábiją Ofiárę zá wyſtępek / á krwią jey pokropią Ołtarz z wierzchu w około.
A wƺyſtkę tłuſtość jey ofiárowáć będźie z niey : Ogon / y tłuſtość okrywájącą wnętrznośći :
Obiedwie też nerki z tłuſtośćią / która jeſt ná nich / y ná polędźwicách y odźiedzkę / która <i>jeſt</i> ná wątrobie y ná nerkách / odejmie.
Y ſpali to Kápłán ná Ołtarzu / ná ofiárę ogniſtą PANU : Ofiárá to jeſt zá wyſtępek.
Wƺelki mężcżyzná z Kápłanów / będźie ją jadł / ná miejſcu świętym jedźiona będźie / rzecz to Naświętƺa.
Jáko ofiárá zá grzech / ták ofiárá zá wyſtępek / jednáką uſtáwę máją : Kápłánowi / któryby go oczyśćiał / <i>náleżeć</i> będźie.
Kápłánowi / któryby czyję Ofiárę cáłopaloną ofiárował / ſkórá teyże ofiáry / którą ofiárował / <i>należeć</i> będźie.
Tákże káżda Ofiárá śniedna / w piecu upiecżona / y wƺyſtko co ná panwi álbo w kotle gotowano / będźie Kápłánowi / który to ofiáruje / <i>należeć</i> będźie.
Przytym wƺelka Ofiárá śniedna / zágniećiona z Oliwą / álbo práżona wƺyſtkim Synom Aáronowym <i>należeć</i> będźie : ták jednemu jáko drugiemu.
Táć też jeſt Uſtáwá Ofiáry ſpokojney / którą będą ofiárowáli PANU.
Jeſliby <i>kto</i> ofiárował ná ofiárę dźiękczynienia / tedy ofiárowáć będźie / ná ofiárę dźiękczynienia / plácki przáſne / zágniatáne z Oliwą / y kreple przáſne / pomázáne Oliwą / y mąkę pƺenną / ſmáżoną z tymi pláckámi w Oliwie zágniećionymi.
Przy tych pláckách / będźie też chleb kwáƺony ofiárował ná ofiárę ſwoję / z ofiárą dźiękczynienia ſpokojnych ofiar ſwojich.
Y będźie ofiárował z niego / jeden <i>chleb</i> z káżdey ofiáry / ná podnoƺenie PANU : Kápłánowi / który kropi krwią <i>Ofiar</i> ſpokojnych / <i>należeć</i> to będźie.
Mięſo záś ofiáry dźiękczynienia / <i>która jeſt</i> ſpokojna / w dźień ofiárowánia jey / jedźione będźie : nie zoſtáwią z niego nic do jutrá.
A jeſliby kto ślubną álbo dobrowolną przynióſł Ofiárę ſwoję : w dźień ofiárowánia ofiárę jego / jedźiona będźie / á názájutrz / coby zoſtáło z niey / zjedzą.
Ale / jeſliby co zoſtáło mięsá / z tey ofiáry / do trzećiego dniá / ogniem ſpalone będźie.
A jeſliby kto przećię jadł mięſo tey ofiáry ſpokojney / dniá trzećiego / nie będźie przyjemny ten / który ją ofiárował : nie będźie mu płátna / <i>owƺem</i> obrzydliwośćią będźie / á ktoby jadł z niey / niepráwość ſwoję ponieśie.
Mięſo też / któreby śię dotknęło cżego niecżyſtego / nie będźie jedźione / ále ogniem ſpalone będźie : Mięſo záś <i>inne</i> káżdy czyſty jeść je będźie.
A ktobykolwiek jadł mięſo z ofiáry ſpokojney ofiárowáney PANU / á byłby nieczyſty / wytrácony będźie cżłowiek ten z ludu ſwego.
Jeſliby śię też kto dotknął cżego niecżyſtego / <i>bądź</i> nieczyſtośći cżłowieczey / bądź nieczyſtośći bydlęcey / bądź jákieykolwiek obrzydliwośći nieczyſtey / á jadłby mięſo z ofiáry ſpokojney ofiárowáney PAnu : tedy wytrácony będźie cżłowiek ten / z ludu ſwego.
Rzekł jeƺcze PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim / y rzecż : żadney tłuſtośći z Wołu / áni z Owiec / áni z kozy / nie będźieƺ jadł.
Acżkolwiek tłuſtość <i>bydlęćiá</i> zdechłego / álbo tłuſtość rozƺárpánego / może być do wƺelákiey potrzeby / ále jeść jey żadnym ſpoſobem nie będźiećie :
Abowiem ktobykolwiek jadł tłuſtość z bydlęćiá / które ofiárowáć będźie <i>cżłowiek</i> ná ofiárę ogniſtą PANU / niechay wytrácony będźie cżłowiek ten który jadł / z ludu ſwego.
Tákże żadney krwie jeść nie będźiećie / we wƺyſtkich mieƺkániách wáƺych / ták z ptaków jáko y z bydląt.
Wƺelki cżłowiek / któryby jadł jákąkolwiek krew wytrácony będźie cżłowiek on / z ludu ſwego.
Rzekł jeƺcze PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / á rzecz <i>im</i> : Ktoby ofiárował Ofiárę ſpokojną ſwoję PAnu / przynieśie Ofiárę ſwoję PAnu / z ofiáry ſpokojney ſwojey.
Ręká jego przynieśie / ofiárę ogniſtą PAnu / tłuſtość z moſtkiem przynieśie / á moſtek niech będźie tám y ſám obracány ná Ofiárę przed PAnem.
Potym ſpali Kápłán tłuſtość ná Ołtarzu : ále moſtek zoſtánie Aáronowi y Synom jego.
A łopátkę práwą oddáćie / ná podnoƺenie Kápłánowi / z Ofiar ſpokojnych wáƺych.
Kto też z Synów Aáronowych ofiárowáć będźie krew / Ofiar ſpokojnych / y tłuſtość / temu śię doſtánie łopátká práwa / dźiałem :
Abowiem moſtek ſám y tám obracánia y łopátkę podnoƺenia / wźiąłem od Synów Izráelſkich / z ofiar ich ſpokojnych / y dałem je Aáronowi Kápłánowi / y Synom jego / práwem wiecżnym od Synów Izráelſkich.
Toć jeſt <i>dźiał</i> pomázánego Aároná / y pomázánych Synów jego z Ofiar ogniſtych PAńſkich / od dniá / któregom im przyſtąpić rozkazał / ku ſpráwowániu urzędu Kápłáńſkiego / PANU.
Y rozkazał PAN áby im to dawano było / od dniá którego je pomázał / od Synów Izráelſkich / práwem wiecznym / w narodźiech ich.
Táć jeſt uſtáwá Ofiáry Cáłopalenia / ofiáry śniedney / y ofiáry zá grzech / y zá wyſtępek / y poświącánia / y ofiáry ſpokojney :
Którą rozkazał PAN Mojzeƺowi / ná górze Synáj / dniá którego przykázował Synom Izráelſkim / áby ofiárowáli Ofiáry ſwe PANU ná Puƺczy Synáj.
