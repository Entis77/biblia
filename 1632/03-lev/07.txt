Táć jeſt uſtáwá ofiáry zá wyſtępek / która jeſt nájświętƺá.
Ná miejſcu / gdźie biją ofiáry cáłopáleniá / zábiją ofiárę zá wyſtępek / á krwią jey pokropią ołtárz z wierzchu w około.
A wƺyſtkę tłuſtość jey ofiárowáć będźie z niej / ogon y tłuſtość okrywájącą wnętrznośći ;
Obiedwie też nerki z tłuſtośćią / która jeſt ná nich / y ná polędwicách / y odźiecżkę / która jeſt ná wątrobie y ná nerkách / odejmie.
Y ſpáli to kápłán ná ołtárzu ná ofiárę ogniſtą Pánu ; ofiárá to jeſt zá wyſtępek.
Wƺelki mężcżyzná z kápłánów będźie ją jádł / ná miejſcu świętym jedzoná będźie ; rzecż to nájświętƺá.
Jáko ofiárá zá grzech / ták ofiárá zá wyſtępek jednáką uſtáwę máją ; kápłánowi / któryby go ocżyƺcżáł / náleżeć będźie.
Kápłánowi / któryby cżyję ofiárę cáłopáloną ofiárowáł / ſkórá tejże ofiáry / którą ofiárowáł / náleżeć będźie.
Tákże káżdá ofiárá śniedná w piecu upiecżoná / y wƺyſtko / co ná páńwi álbo w kotle gotowáne będźie / kápłánowi / który to ofiáruje / náleżeć będźie.
Przytem wƺeláká ofiárá śniedná / zágniećioná z oliwą álbo práżoná / wƺyſtkim ſynom Aáronowym náleżeć będźie / ták jednemu / jáko drugiemu.
Táć też jeſt uſtáwá ofiáry ſpokojney / którą będą ofiárowáli Pánu.
Jeſliby kto ofiárowáł ná ofiárę dźiękcżynieniá / tedy ofiárowáć będźie ná ofiárę dźiękcżynieniá plácki przáśne / zágniátáne z oliwą / y kreple przáśne / pomázáne oliwą / y mąkę pƺenną / ſmáżoną z temi pláckámi w oliwie zágniećionemi.
Przy tych pláckách będźie też chleb kwáƺony ofiárowáł ná ofiárę ſwoję z ofiárą dźiękcżynieniá ſpokojnych ofiár ſwoich.
Y będźie ofiárowáł z niego jeden chleb z káżdej ofiáry ná podnoƺenie Pánu. Kápłánowi / który kropi krwią ofiár ſpokojnych / náleżeć to będźie.
Mięſo záś ofiáry dźiękcżynieniá / która jeſt ſpokojná / w dźień ofiárowániá ofiáry jego jedzone będźie ; nie zoſtáwią z niego nic do jutrá.
A jeſliby kto ślubną álbo dobrowolną przynióſł ofiárę ſwoję / w dźień ofiárowániá ofiárę jego jedzone będźie ; á názájutrz / coby zoſtáło z niej / zjedzą.
Ale jeſliby co zoſtáło mięſá z tey ofiáry do trzećiego dniá / ogniem ſpálone będźie.
A jeſliby kto przećię jádł mięſo tey ofiáry ſpokojney dniá trzećiego / nie będźie przyjemny ten / który ją ofiárowáł ; nie będźie mu płátná / owƺem obrzydliwośćią będźie ; á ktoby jádł z niej / niepráwość ſwoję ponieśie.
Mięſo też / któreby śię dotknęło cżego niecżyſtego / nie będźie jedzone / ále ogniem ſpálone będźie ; mięſo záś inne / káżdy cżyſty jeść je będźie.
A ktobykolwiek jádł mięſo z ofiáry ſpokojney / ofiárowáney Pánu / á byłby niecżyſty / wytrácony będźie cżłowiek ten z ludu ſwego.
Jeſliby śię też kto dotknął cżego niecżyſtego / bądź niecżyſtośći cżłowiecżej / bądź niecżyſtośći bydlęcey / bądź jákiejkolwiek obrzydliwośći niecżyſtey / á jádłby mięſo z ofiáry ſpokojney / ofiárowáney Pánu / tedy wytrácony będźie cżłowiek ten z ludu ſwego.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / y rzecż : Żádney tłuſtośći z wołu / áni z owiec / áni z kozy / nie będźieƺ jádł /
Acżkolwiek tłuſtość bydlęćiá zdechłego / álbo tłuſtość rozƺárpánego może być do wƺelákiej potrzeby ; ále jeść jey żádnym ſpoſobem nie będźiećie.
Abowiem ktobykolwiek jádł tłuſtość z bydlęćiá / które ofiárowáć będźie cżłowiek ná ofiárę ogniſtą Pánu / niecháj wytrácony będźie cżłowiek ten / który jádł / z ludu ſwego.
Tákże żádney krwi jeść nie będźiećie we wƺyſtkich mieƺkániách wáƺych / ták z ptáków / jáko y z bydląt.
Wƺelki cżłowiek / któryby jádł jákąkolwiek krew / wytrácony będźie cżłowiek on z ludu ſwego.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á rzecż im : Ktoby ofiárowáł ofiárę ſpokojną ſwoję Pánu / przynieśie ofiárę ſwoję Pánu z ofiáry ſpokojney ſwojey.
Ręká jego przynieśie ofiárę ogniſtą Pánu ; tłuſtość z moſtkiem przynieśie / á moſtek niech będźie tám y ſám obrácány ná ofiárę przed Pánem.
Potem ſpáli kápłán tłuſtość ná ołtárzu ; ále moſtek zoſtánie Aáronowi y ſynom jego.
A łopátkę práwą oddáćie ná podnoƺenie kápłánowi z ofiár ſpokojnych wáƺych.
Kto też z ſynów Aáronowych ofiárowáć będźie krew ofiár ſpokojnych y tłuſtość / temu śię doſtánie łopátká práwá dźiáłem.
Abowiem moſtek ſám y tám obrácániá / y łopátkę podnoƺeniá / wźiąłem od ſynów Izráelſkich z ofiár ich ſpokojnych / y dáłem je Aáronowi kápłánowi / y ſynom jego práwem wiecżnem od ſynów Izráelſkich.
Toć jeſt dźiáł pomázánego Aároná / y pomázánych ſynów jego z ofiár ogniſtych Páńſkich / od dniá / któregom im przyſtąpić rozkázáł ku ſpráwowániu urzędu kápłáńſkiego Pánu.
Y rozkázáł Pán / áby im to dáwáno było od dniá / którego je pomázáł / od ſynów Izráelſkich práwem wiecżnem w národźiech ich.
Táć jeſt uſtáwá ofiáry cáłopáleniá / ofiáry śniedney / y ofiáry zá grzech / y zá wyſtępek / y poświęceniá / y ofiáry ſpokojney.
Którą rozkázáł Pán Mojżeƺowi ná górze Synáj / dniá / którego przykázywáł ſynom Izráelſkim / áby ofiárowáli ofiáry ſwe Pánu ná puƺcży Synáj.
