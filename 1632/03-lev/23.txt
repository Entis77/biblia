Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á mów im : Świętá urocżyſte Páńſkie / które názywáć będziećie zgromádzeniá święte / te ſą świętá urocżyſte moje.
Przez ƺeść dni robić będziećie ; ále w dzień śiódmy ſábát odpocżynieniá / zgromádzenie święte / żádnej roboty cżynić nie będziećie ; ſábát Páńſki jeſt we wƺyſtkich mieƺkániách wáƺych.
A teć ſą urocżyſte świętá Páńſkie / zgromádzeniá święte / które obchodzić będziećie pewnego ich cżáſu.
Mieśiącá pierwƺego / dniá cżternáſtego tegoż mieśiącá / między dwomá wiecżorámi święto przejśćiá Páńſkiego.
Potem dniá piętnáſtego tegoż mieśiącá / święto przáśników będzie Pánu ; przez śiedm dni chleby przáśne jeść będziećie.
A dniá pierwƺego zgromádzenie święte mieć będziećie ; żádnej roboty ſłużebnicżej cżynić nie będziećie.
Ale będziećie ofiárowáli ofiárę ogniſtą Pánu przez śiedm dni. Dniá tákże śiódmego zgromádzenie święte będzie ; żádnej roboty ſłużebnicżej cżynić nie będziećie.
Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / y rzecż im : Gdy wnijdziećie do ziemi / którą Já wám dáwám / á będziećie żąć zboże wáƺe / tedy przynieśiećie ſnop pierwiáſtek żniwá wáƺego do kápłáná.
Y będzie tám y ſám obrácáł on ſnop przed oblicżem Páńſkim / áby był przyjemny zá wáſ ; názájutrz po ſábáćie podnośić go będzie kápłán.
Zábijećie też dniá / którego obrácáć będziećie on ſnop / báránká zupełnego / rocżnego ná ofiárę cáłopáleniá Pánu ;
Przytem ofiárę jego śniedną ze dwu dzieśiątych cżęśći efy mąki pƺennej / zádziáłánej z oliwą ná páloną ofiárę Pánu dlá wdzięcżnej wonnośći ; tákże ofiárę jego mokrą / winá cżwártą cżęść hynu.
A chlebá y práżmá / y nowego zbożá jeść nie będziećie áż do dniá / którego przynieśiećie ofiárę Bogu wáƺemu ; uſtáwá to wiecżná będzie w národziech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Nálicżyćie tákże ſobie od dniá pierwƺego po ſábáćie / od dniá / któregośćie ofiárowáli ſnop podnoƺeniá / śiedm tygodni zupełnych niech będzie.
Aż do pierwƺego dniá po śiódmym tygodniu nálicżyćie pięćdzieśiąt dni ; tedy ofiárowáć będziećie ofiárę śniedną nową Pánu.
Z domów wáƺych przynieśiećie chleby ná obrácánie tám y ſám ; dwá chleby / ze dwu dzieśiątych cżęśći pƺennej mąki z kwáſem upiecżone będą ; pierwiáſtki to Pánu.
A ofiárowáć z tym chlebem będziećie śiedm báránków rocżnych zupełnych / y ćielcá jednego / y dwu báránów ; ná ofiárę cáłopáleniá będą Pánu z ofiárą śniedną ich y z mokremi ofiárámi ich ; ofiárá to ogniſtá ná wdzięcżną wonność Pánu.
Zábijećie też kozłá jednego zá grzech / y dwá báránki rocżne ná ofiárę ſpokojną.
Y będzie je obrácáł tám y ſám kápłán z chlebem pierwiáſtek ná ofiárę ſám y tám obrácániá przed oblicżem Páńſkim / y ze dwiemá báránkámi ; y będą święte rzecży Pánu dlá kápłáná.
Y ogłośićie w ten dzień święto ; zgromádzenie święte mieć będziećie ; żádnej roboty ſłużebnicżej cżynić nie będziećie ; uſtáwá to będzie wiecżná we wƺyſtkich mieƺkániách wáƺych / w národziech wáƺych.
A gdy żąć będziećie zboże ziemi wáƺej / nie będzieƺ do końcá polá twego dożynáł / y kłoſów pozoſtáłych żniwá twego zbieráć nie będzieƺ : ubogiemu / y przychodniowi zoſtáwiƺ je ; Jám Pán Bóg wáƺ.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / mówiąc : Mieśiącá śiódmego / pierwƺego dniá tegoż mieśiącá / będziećie mieli ſábát / pámiątkę trąbieniá / zgromádzenie święte.
Żádnej roboty ſłużebnicżej nie będziećie cżynili / lecż ofiárowáć będziećie ofiárę ogniſtą Pánu.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Lecż dzieśiątego dniá tegoż mieśiącá śiódmego / dzień ocżyƺcżániá jeſt ; zgromádzenie święte mieć będziećie / á będziećie trápić duƺe wáƺe / ofiárując ogniſtą ofiárę Pánu.
Żádnej roboty nie będziećie cżynili w ten dzień ; bo dzień ocżyƺcżániá jeſt ná ocżyƺcżenie wáſ przed oblicżem Páná / Bogá wáƺego.
A wƺelká duƺá / któráby śię nie trápiłá tego dniá / wytráconá będzie z ludu ſwego.
Tákże / ktobykolwiek cżynił robotę jáką w tenże dzień / wytrácę cżłowieká tego z pośrodku ludu jego.
Żádnej roboty nie cżyńćie ; uſtáwá to będzie wiecżná w národziech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Sábát odpocżynieniá mieć będziećie / gdy trápić będziećie duƺe ſwe ; dziewiątego dniá tegoż mieśiącá / wiecżór / od wiecżorá áż do wiecżorá / obchodzić będziećie ſábát wáƺ.
Rzekł záś Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / y rzecż : Piętnáſtego dniá tegoż śiódmego mieśiącá będzie święto kucżek przez śiedm dni Pánu.
Dniá pierwƺego zgromádzenie święte będzie ; żádnej roboty ſłużebnicżej cżynić nie będziećie.
Przez śiedm dni ofiárowáć będziećie ofiárę ogniſtą Pánu ; dniá óſmego zgromádzenie święte mieć będziećie / á będziećie ofiárowáli ofiárę ogniſtą Pánu ; święto jeſt / żádnej roboty ſłużebnicżej nie będziećie cżynili.
Teć ſą świętá urocżyſte Páńſkie / które obchodzić będziećie / zgromádzeniá święte / ábyśćie ofiárowáli ofiárę ogniſtą Pánu / cáłopálenie / y ofiárę śniedną / y ofiárę ſpokojną y ofiáry mokre / káżdą w dzień ſwój.
Oprócż ſábátów Páńſkich / y oprócż dárów wáƺych / y oprócż wƺyſtkich ślubów wáƺych / y oprócż wƺyſtkich dobrowolnych podárków wáƺych / które oddáwáć będziećie Pánu.
Wƺákże piętnáſtego dniá mieśiącá śiódmego / gdy zbierzećie urodzáj ziemi / będziećie obchodzili święto Pánu przez śiedm dni ; dniá pierwƺego odpocżynienie / tákże dniá óſmego odpocżynienie będzie.
Tedy weźmiećie ſobie pierwƺego dniá owocu z drzewá co nájpiękniejƺego / y gáłązek pálmowych / y gáłązek drzewá gęſtego / y wierzbiny od potoku / y weſelić śię będziećie przed Pánem Bogiem wáƺym przez śiedm dni.
A obchodzić będziećie to święto Pánu przez śiedm dni ná káżdy rok. Uſtáwá to wiecżná w národziech wáƺych ; káżdego mieśiącá śiódmego obchodzić je będziećie.
W kucżkách mieƺkáć będziećie przez śiedm dni ; káżdy zrodzony w Izráelu mieƺkáć będzie w kucżkách /
Aby wiedzieli potomkowie wáśi / iżem w námiotách kázáł mieƺkáć ſynom Izráelſkim / gdym je wywiódł z ziemi Egipſkiej ; Já Pán / Bóg wáƺ.
Y opowiedziáł Mojżeƺ świętá urocżyſte Páńſkie ſynom Izráelſkim.
