Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / á mów im : Świętá urocżyſte Páńſkie / które názywáć będźiećie zgromádzenia święte / te ſą świętá urocżyſte moje.
Przez ƺeść dni robić będźiećie ; ále w dźień śiódmy ſábát odpocżynieniá / zgromádzenie święte / żádney roboty cżynić nie będźiećie ; ſábát Páńſki jeſt we wƺyſtkich mieƺkániách wáƺych.
A teć ſą urocżyſte świętá Páńſkie / zgromádzenia święte / które obchodźić będźiećie pewnego ich cżáſu.
Mieśiącá pierwƺego / dniá cżternáſtego tegoż mieśiącá / miedzy dwomá wiecżorámi święto przejśćiá Páńſkiego.
Potem dniá piętnáſtego tegoż mieśiącá / święto przáśników będźie Pánu ; przez śiedm dni chleby przáśne jeść będźiećie.
A dniá pierwƺego zgromádzenie święte mieć będźiećie ; żádney roboty ſłużebnicżey cżynić nie będźiećie.
Ale będźiećie ofiárowáli ofiárę ogniſtą Pánu przez śiedm dni. Dniá tákże śiódmego zgromádzenie święte będźie ; żádney roboty ſłużebnicżey cżynić nie będźiećie.
Y rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / y rzecż im : Gdy wnijdźiećie do źiemi / którą Ja wam dáwám / á będźiećie żąć zboże wáƺe / tedy przynieśiećie ſnop pierwiáſtek żniwá wáƺego do kápłáná.
Y będźie tám y ſám obrácał on ſnop przed oblicżem Páńſkim / áby był przyjemny zá was ; názájutrz po ſábáćie podnośić go będźie kápłán.
Zábijećie też dniá / którego obrácáć będźiećie on ſnop / báránká zupełnego / rocżnego ná ofiárę cáłopalenia Pánu ;
Przytem ofiárę jego śniedną ze dwu dźieśiątych cżęśći efy mąki pƺenney / zádźiáłáney z oliwą ná páloną ofiárę Pánu dla wdźięcżney wonnośći ; tákże ofiárę jego mokrą / winá cżwártą cżęść hynu.
A chlebá y práżmá / y nowego zbożá jeść nie będźiećie áż do dniá / którego przynieśiećie ofiárę Bogu wáƺemu ; uſtáwá to wiecżná będźie w národźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Nálicżyćie tákże ſobie od dniá pierwƺego po ſábáćie / od dniá / któregośćie ofiárowáli ſnop podnoƺeniá / śiedm tygodni zupełnych niech będźie.
Aż do pierwƺego dniá po śiódmym tegodniu nálicżyćie pięćdźieśiąt dni ; tedy ofiárowáć będźiećie ofiárę śniedną nową Pánu.
Z domów wáƺych przynieśiećie chleby ná obrácánie tám y ſám ; dwá chleby / ze dwu dźieśiątych cżęśći pƺenney mąki z kwáſem upiecżone będą ; pierwiáſtki to Pánu.
A ofiárowáć z tym chlebem będźiećie śiedm báránków rocżnych zupełnych / y ćielcá jednego / y dwu báránów ; ná ofiárę cáłopalenia będą Pánu z ofiárą śniedną ich y z mokremi ofiárámi ich ; ofiárá to ogniſtá ná wdźięcżną wonność Pánu.
Zábijećie też kozłá jednego zá grzech / y dwá báránki rocżne ná ofiárę ſpokojną.
Y będźie je obrácał tám y ſám kápłán z chlebem pierwiáſtek ná ofiárę ſám y tám obrácánia przed oblicżem Páńſkim / y ze dwiemá báránkámi ; y będą święte rzecży Pánu dla kápłáná.
Y ogłośićie w ten dźień święto ; zgromádzenie święte mieć będźiećie ; żádney roboty ſłużebnicżey cżynić nie będźiećie ; uſtáwá to będźie wiecżná we wƺyſtkich mieƺkániách wáƺych / w národźiech wáƺych.
A gdy żąć będźiećie zboże źiemi wáƺej / nie będźieƺ do końcá polá twego dożynał / y kłoſów pozoſtáłych żniwá twego zbieráć nie będźieƺ : ubogiemu / y przychodniowi zoſtáwiƺ je ; Jam Pán Bóg wáƺ.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / mówiąc : Mieśiącá śiódmego / pierwƺego dniá tegoż mieśiącá / będźiećie mieli ſábát / pámiątkę trąbieniá / zgromádzenie święte.
Żádney roboty ſłużebnicżey nie będźiećie cżynili / lecż ofiárowáć będźiećie ofiárę ogniſtą Pánu.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Lecż dźieśiątego dniá tegoż mieśiącá śiódmego / dźień ocżyƺcżánia jeſt ; zgromádzenie święte mieć będźiećie / á będźiećie trápić duƺe wáƺe / ofiárując ogniſtą ofiárę Pánu.
Żádney roboty nie będźiećie cżynili w ten dźień ; bo dźień ocżyƺcżánia jeſt ná ocżyƺcżenie was przed oblicżem Páná / Bogá wáƺego.
A wƺelká duƺá / któráby śię nie trápiłá tego dniá / wytráconá będźie z ludu ſwego.
Tákże / ktobykolwiek cżynił robotę jáką w tenże dźień / wytrácę cżłowieká tego z pośrzodku ludu jego.
Żádney roboty nie cżyńćie ; uſtáwá to będźie wiecżná w národźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Sábát odpocżynieniá mieć będźiećie / gdy trápić będźiećie duƺe ſwe ; dźiewiątego dniá tegoż mieśiącá / wiecżór / od wiecżorá áż do wiecżorá / obchodźić będźiećie ſábát wáƺ.
Rzekł záś Pán do Mojżeƺá / mówiąc :
Powiedz ſynom Izráelſkim / y rzecż : Piętnáſtego dniá tegoż śiódmego mieśiącá będźie święto kucżek przez śiedm dni Pánu.
Dniá pierwƺego zgromádzenie święte będźie ; żádney roboty ſłużebnicżey cżynić nie będźiećie.
Przez śiedm dni ofiárowáć będźiećie ofiárę ogniſtą Pánu ; dniá óſmego zgromádzenie święte mieć będźiećie / á będźiećie ofiárowáli ofiárę ogniſtą Pánu ; święto jeſt / żádney roboty ſłużebnicżey nie będźiećie cżynili.
Teć ſą świętá urocżyſte Páńſkie / które obchodźić będźiećie / zgromádzenia święte / ábyśćie ofiárowáli ofiárę ogniſtą Pánu / cáłopálenie / y ofiárę śniedną / y ofiárę ſpokojną y ofiáry mokre / káżdą w dźień ſwój.
Oprócż ſábátów Páńſkich / y oprócż dárów wáƺych / y oprócż wƺyſtkich ślubów wáƺych / y oprócż wƺyſtkich dobrowolnych podárków wáƺych / które oddáwáć będźiećie Pánu.
Wƺákże piętnáſtego dniá mieśiącá śiódmego / gdy zbierzećie urodzáj źiemi / będźiećie obchodźili święto Pánu przez śiedm dni ; dniá pierwƺego odpocżynienie / tákże dniá óſmego odpocżynienie będźie.
Tedy weźmiećie ſobie pierwƺego dniá owocu z drzewá co nájpiękniejƺego / y gáłązek pálmowych / y gáłązek drzewá gęſtego / y wierzbiny od potoku / y weſelić śię będźiećie przed Pánem Bogiem wáƺym przez śiedm dni.
A obchodźić będźiećie to święto Pánu przez śiedm dni ná káżdy rok. Uſtáwá to wiecżná w národźiech wáƺych ; káżdego mieśiącá śiódmego obchodźić je będźiećie.
W kucżkách mieƺkáć będźiećie przez śiedm dni ; káżdy zrodzony w Izráelu mieƺkáć będźie w kucżkách /
Aby wiedźieli potomkowie wáśi / iżem w námiotách kázał mieƺkáć ſynom Izráelſkim / gdym je wywiódł z źiemi Egipſkiej ; Ja Pán / Bóg wáƺ.
Y opowiedźiał Mojzeƺ świętá urocżyſte Páńſkie ſynom Izráelſkim.
