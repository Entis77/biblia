Rzekł jeƺcze PAN do Mojzeƺá mówiąc :
Powiedz Synom Izráelſkim / á mów im : Świętá uroczyſte PAńſkie / które názywáć będźiećie / zgromádzenia święte / te ſą Świętá uroczyſte moje.
Przez ƺeść dni robić będźiećie / ále w dźień śiódmy / Sábbát odpoczynienia / zgromádzenie święte / żadney roboty czynić nie będźiećie / Sábbát PAńſki jeſt / we wƺyſtkich mieƺkániách wáƺych.
A teć ſą uroczyſte świętá PAńſkie / zgromádzenia święte / które obchodźić będźiećie pewnego ich czáſu.
Mieśiącá pierwƺego / dniá czternaſtego <i>tegoż mieśiącá</i> miedzy dwiemá wieczorámi święto Prześćia PAńſkiego.
Potym dniá piętnaſtego tegoż mieśiącá Święto Przáſników <i>będźie</i> PAnu : przez śiedm dni chleby przáſne jeść będźiećie /
A dniá pierwƺego / zgromádzenie święte mieć będźiećie : żadney roboty ſłużebniczey czynić nie będźiećie.
Ale będźiećie ofiárowáli ofiárę ogniſtą PAnu / przez śiedm dni : dniá <i>tákże</i> śiódmego zgromádzenie święte będźie / żadney roboty ſłużebniczey czynić nie będźiećie.
Y rzekł PAN do Mojzeƺá mówiąc :
Powiedz Synom Izráelſkim / y rzecz im : Gdy wnidźiećie do źiemie którą ja wam dawam / á będźiećie żąć zboże wáƺe / tedy przynieśiećie ſnop pierwiaſtek żniwá wáƺego do Kápłaná.
Y będźie tám y ſám obracał on ſnop przed oblicżem PAńſkim / áby był przyjemny zá was : názájutrz po Sábbáćie podnośić go będźie Kápłan.
Zábijećie też dniá którego obracáć będźiećie on ſnop / báránká zupełnego rocznego / ná ofiárę cáłopalenia PAnu.
Przytym Ofiárę jego śniedną / ze dwu dźieśiątych cżęśći Efy mąki pƺenney / zádźiáłáney z oliwą ná paloną ofiárę PAnu / dla wdźięczney wonnośći : tákże Ofiárę jego mokrą / winá czwartą część Hinu.
A chlebá y práżmá / y nowego zbożá jeść nie będźiećie / áż do dniá którego przynieśiećie Ofiárę Bogu wáƺemu / uſtáwá to wieczna będźie w narodźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Náliczyćie tákże ſobie od dniá pierwƺego po Sábbáćie / od dniá któregośćie ofiárowáli / ſnop podnoƺenia / śiedm tegodniów zupełnych niech będźie.
Aż do pierwƺego dniá po śiódmym tegodniu / náliczyćie pięćdźieśiąt dni : tedy ofiárowáć będźiećie ofiárę śniedną nową PANU.
Z domów wáƺych przynieśiećie chleby ná obracánie tám y ſám : dwá <i>chleby</i> ze dwu dźieśiątych częśći pƺenney mąki z kwáſem upieczone będą : pierwiaſtki <i>to</i> PAnu.
A ofiárowáć z tym chlebem będźiećie śiedm báránków rocznych zupełnych / y ćielcá jednego y dwu báránów ná ofiárę cáłopalenia będą PAnu / z ofiárą śniedną ich / y z mokrymi ofiárámi ich / ofiárá <i>to</i> ogniſta / ná wdźięczną wonność PAnu.
Zábijećie też kozłá jednego zá grzech / y dwá báránki roczne ná ofiárę ſpokojną.
Y będźie je obracał tám y ſám Kápłan / z chlebem pierwiaſtek / ná ofiárę ſám y tám obracánia przed obliczem PAńſkim / y ze dwiemá báránkámi / y będą święte rzeczy PAnu / dla Kápłaná.
Y ogłośićie w ten dźień święto : zgromádzenie święte mieć będźiećie : żadney roboty ſłużebniczey czynić nie będźiećie : Uſtáwá <i>to będźie</i> wieczna / we wƺyſtkich mieƺkániách wáƺych w narodźiech wáƺych.
A gdy żąć będźiećie zboże źiemie wáƺey / nie będźieƺ do końcá polá twego dożynał / y kłoſów pozoſtáłych żniwá twego zbieráć nie będźieƺ / ubogiemu y przychodniowi zoſtáwiƺ je : <i>JA PAN /</i> Bóg wáƺ.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Synom Izráelſkim / mówiąc : Mieśiącá śiódmego / pierwƺego <i>dniá</i> tegoż mieśiącá / będźiećie mieli Sábbát / pámiątkę trąbienia / zgromádzenie święte.
Żadney roboty ſłużebniczey nie będźiećie czynili / lecz ofiárowáć będźiećie ofiárę ogniſtą PANU.
Rzekł jeƺcże PAN do Mojzeƺá / mówiąc :
Lecż dźieśiątego <i>dniá</i> tegoż mieśiącá śiódmego / dźień oczyśćiánia jeſt : zgromádzenie święte mieć będźiećie / á będźiećie trapić duƺe wáƺe / ofiárując ogniſtą ofiárę PAnu.
Żadney roboty nie będźiećie czynili w ten dźień / bo dźień oczyśćiánia jeſt : ná oczyśćienie was / przed obliczem PANA Bogá wáƺego.
A wƺelka duƺá któraby śię nie trapiłá tego dniá / wytrácona będźie z ludu ſwego.
Tákże ktobykolwiek czynił robotę jáką w tenże dźień / wytrácę cżłowieká tego z pośrzodku ludu jego.
Żadney roboty nie czyńćie : Uſtáwá <i>to</i> będźie wieczna w narodźiech wáƺych / we wƺyſtkich mieƺkániách wáƺych.
Sábbát odpoczynienia mieć będźiećie / gdy trapić będźiećie duƺe ſwe / dźiewiątego dniá / tegoż mieśiącá wieczór : od wieczorá áż do wieczorá obchodźić będźiećie Sábbát wáƺ.
Rzekł záś PAN do Mojzeƺá mówiąc :
Powiedz Synom Izráelſkim y rzecz : Piętnaſtego dniá / tegoż śiódmego mieśiącá / będźie święto Kuczek przez śiedm dni / PAnu.
Dniá pierwƺego zgromádzenie Święte będźie / żadney roboty ſłużebniczey czynić nie będźiećie :
Przez śiedm dni ofiárowáć będźiećie Ofiárę Ogniſtą PAnu : dniá óſmego / zgromádzenie święte mieć będźiećie : á będźiećie Ofiárowáli Ofiárę ogniſtą PAnu : święto jeſt / żadney roboty ſłużebniczey nie będźiećie cżynili.
Teć ſą świętá uroczyſte PAńſkie / które obchodźić będźiećie zgromádzenia święte : ábyśćie ofiárowáli ofiárę ogniſtą PAnu / cáłopalenie / y Ofiárę śniedną / y Ofiárę <i>ſpokojną,</i> y ofiáry mokre / káżdą w dźień ſwój.
Oprócz Sábbátów PAńſkich / y oprócz dárów wáƺych / y oprócz wƺyſtkich ślubów wáƺych / y oprócz wƺyſtkich dobrowolnych podárków wáƺych / które oddawáć będźiećie PANU.
Wƺákże piętnaſtego dniá / mieśiącá śiódmego / gdy zbierzećie urodzáj źiemie / będźiećie obchodźili święto PAnu przez śiedm dni : Dniá pierwƺego odpoczynienie / tákże dniá óſmego odpoczynienie będźie.
Tedy weźmiećie ſobie pierwƺego dniá Owocu z drzewá co napiękniejƺego / y gáłązek pálmowych / y gáłązek drzewá gęſtego / y wierzbiny od potoku / y weſelić śię będźiećie przed PAnem Bogiem wáƺym / przez śiedm dni.
A obchodźić będźiećie to Święto PAnu przez śiedm dni / ná <i>káżdy</i> rok / uſtáwá <i>to</i> wieczná w Narodźiech wáƺych : káżdego mieśiącá śiódmego obchodźić je będźiećie :
W kuczkách mieƺkáć będźiećie przez śiedm dni / káżdy zrodzony w Izráelu mieƺkáć będźie w kuczkách :
Aby wiedźieli potomkowie wáƺy / yżem w Namiotách kazał mieƺkáć Synom Izráelſkim / gdym je wywiódł z źiemie Egipſkiey / <i>JA PAN</i> Bóg wáƺ.
Y opowiedźiał Mojzeƺ świętá uroczyſte PAńſkie / synom Izráelſkim.
