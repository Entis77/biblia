Potym rzekł PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / á powiedz im : gdyby człowiek ślubem poślubił duƺę PANU / według ƺácunku twego <i>da okup</i>.
A będźie <i>ták</i> ƺácunek twój : Zá mężczyznę od dwudźieſtu lat / áż do ƺeśćidźieśiąt lat będźie ƺácunek twój / pięćdźieśiąt Syklów śrebrá / według wagi świątnice.
A jeſli jeſt biała głowá / ƺácunek twój będźie trzydźieśći Syklów.
A jeſli od piątego roku áż do dwudźieſtego roku / tedy będźie ƺácunek twój : zá mężczyznę dwadźieśćiá Syklów / á zá białą głowę / dźieśięć Syklów.
A jeſli zá dźiećię od <i>jednego</i> mieśiącá áż do piąćiu lat / tedy będźie ƺácunek twój : Zá mężczyznę / pięć Syklów śrebrá / á zá dźieweczkę ƺácunek twój / trzy Sykle śrebrá.
A jeſli od ƺeśćidźieśiąt lat y wyżey : będźieli mężczyzná / tedy będźie ƺácunek twój piętnaśćie Syklów / á zá białą głowę dźieśięć Syklów.
Lecz jeſliby był <i>ták</i> ubogi żeby nie mógł oddáć ƺácunku twego / tedy go ſtáwią przed Kápłaná / y oƺácuje go Kápłan : według przemożenia tego który ślubował oƺácuje go Kápłan.
Jesliby też bydlę z tych / które śię ofiárują ná ofiárę PAnu <i>poślubił</i> : káżde które odda PAnu / będźie święte :
Nie odmieni go / áni da innego zá nie / lepƺego zá gorƺe / álbo gorƺego zá lepƺe : jeſliby też jákokolwiek odmienił / bydlę zá bydlę / tedy y ono y to które zá nie dano / będźie święte.
A jeſliby które nieczyſte bydlę <i>poślubił</i> z tych co nie bywáją ofiárowáne PAnu : tedy ſtáwi to bydlę przed Kápłaná.
Y oƺácuje Kápłan bądź dobre / bądź złe : á jáko je oƺácuje Kápłan / ták będźie.
A jeſliby je kto odkupić chćiał / przyda piątą część nád ƺácunek twój.
Jesliby też kto poświęćił Dom ſwój / żeby był święty PAnu / tedy go oƺácuje Kápłan : bądź dobry / bądź zły : jáko go oƺácuje Kápłan / ták zoſtánie.
A gdyby ten który poświęćił / chćiał odkupić dom ſwój / przyda piątą część pieniędzy / nád ƺácunek twój / y będźie jego.
Jesli też kto część roli z dźiedźictwá ſwego poświęćił PAnu / tedy będźie ƺácunek twój według zaśiewku jey : gdźie śię wyśieje chomer jęczmieniá / zá pięćdźieśiąt Syklów śrebrá / <i>ƺácowáno</i> będźie.
Jeſli do miłośćiwego látá poświęćił rolą ſwoję / według ƺácunku twego zoſtánie.
Ale jeſliżby po miłośćiwym lećie poświęćił rolą ſwoję / tedy Kápłan obráchuje mu pieniądze / według lat zoſtawájących do miłośćiwego látá / y umnieyƺy mu śię z ƺácunku twego.
A chćiałliby odkupić rolą / ten który ją poświęćił / przyda piątą część pieniędzy do ƺácunku twego : y zoſtánie przy niey.
Ale gdźieby nie odkupił roli oney / á przedánaby byłá rola komu inƺemu / nie może być odkupiona.
Y będźie oná rola gdy wynidźie miłośćiwe láto / święta PAnu / jáko rola poświęcona / á przydźie w ośiádłość Kápłanowi.
A jeſliby kto rolą kupioną / która nie byłá z ról dźiedźictwá jego / poślubił PAnu.
Tedy poráchuje mu Kápłan ſummę ƺácunku twego / áż do roku miłośćiwego / y da ƺácunek ten dniá onego zá rzecz poświęconą PAnu.
A w miłośćiwe láto wróći śię rola od tego / od kogo ją kupiono / do <i>tego</i> który dźiedźicżnie trzymał rolą onę.
A káżdy ƺácunek twój / będźie wedle Syklá świątnice / <i>á</i> dwádźieśćiá pieniędzy Sykiel waży.
Wƺákże pierworodnego / á które <i>prawem pierworodztwá</i> bywa ofiárowáne PAnu z bydłá / nikt go nie poświęći / bądź wół / bądź owcá : <i>ponieważ</i> PAńſkie ſą.
A jeſliby z bydląt nieczyſtych <i>było,</i> odkupi je według ƺácunku twego / y przyda piątą część nád to. A jeſliby go nie odkupiono : niechże przedáne będźie według ƺácunku twego.
Káżda jednák rzecz poślubiona / którąby kto poślubił PAnu ze wƺyſtkiego co ma z ludźi / y z bydłá / y z ról ośiádłośći ſwojey / nie będźie przedawána / áni odkupowána : bo wƺelka rzecz poślubiona / naświętƺa jeſt PAnu.
Wƺelkie <i>bydlę</i> poślubione / które śię pod ślubem oddawa / od cżłowieká nie będźie odkupione : <i>ále</i> śmierćią umrze.
Wƺyſtkie tákże dźieśięćiny źiemie / z naśienia źiemie / z Owocu drzewá / PAńſkie ſą : bo poświęcone ſą PAnu.
Ale ktoby chćiał odkupić <i>co</i> z dźieśięćin ſwojich / piątą część <i>ceny</i> przyda do nich :
Tákże wƺyſtkie dźieśięćiny / z rogátego bydłá y z drobnego bydłá / wƺyſtkiego co przechodźi pod laſką <i>páſterſką,</i> káżde dźieśiąte / będźie poświęcone PAnu.
Nie będźie przebierał miedzy dobrym álbo złym / áni go odmieniáć będźie : á jeſliby je jákokolwiek odmienił / będźie to y ono odmienione poświęcone / nie ma być odkupione.
Teć ſą przykazánia / które rozkazał PAN Mojzeƺowi / do Synów Izráelſkich / ná górze Synáj.
