Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á powiedz im : Gdyby cżłowiek ślubem poślubił duƺę Pánu / według ƺácunku twego da okup.
A będźie ták ƺácunek twój : Zá mężcżyznę od dwudźieſtu lat áż do ƺeśćdźieśiąt lat / będźie ƺácunek twój pięćdźieśiąt ſyklów ſrebrá według wági świątnicy.
A jeſli jeſt biáłá głowá ƺácunek twój będźie trzydźieśći ſyklów.
A jeſli od piątego roku áż do dwudźieſtego roku / tedy będźie ƺácunek twój zá mężcżyznę dwádźieśćiá ſyklów / á zá biáłą głowę dźieśięć ſyklów.
A jeſli zá dźiećię od jednego mieśiącá áż do pięćiu lat / tedy będźie ƺácunek twój zá mężcżyznę pięć ſyklów ſrebrá / á zá dźiewecżkę ƺácunek twój trzy ſykle ſrebrá.
A jeſli od ƺeśćdźieśiąt lat y wyżey będźieli mężcżyzná / tedy będźie ƺácunek twój piętnáśćie ſyklów / á zá biáłą głowę dźieśięć ſyklów.
Lecż jeſliby był ták ubogi / żeby nie mógł oddáć ƺácunku twego / tedy go ſtáwią przed kápłáná / y oƺácuje go kápłán / według przemożeniá tego / który ślubował / oƺácuje go kápłán.
Jeſliby też bydlę z tych / które śię ofiárują ná ofiárę Pánu / poślubił / káżde / które oddá Pánu / będźie święte.
Nie odmieni go / áni da innego zá nie / lepƺego zá gorƺe / álbo gorƺego zá lepƺe ; jeſliby też jákokolwiek odmienił bydlę zá bydlę / tedy y ono / y to / które zá nie dáno / będźie święte.
A jeſliby które niecżyſte bydlę poślubił z tych / co nie bywáją ofiárowáne Pánu / tedy ſtáwi to bydlę przed kápłáná.
Y oƺácuje kápłán bądź dobre / bądź złe / á jáko je oƺácuje kápłán / ták będźie.
A jeſliby je kto odkupić chćiał / przydá piątą cżęść nád ƺácunek twój.
Jeſliby też kto poświęćił dom ſwój / żeby był święty Pánu / tedy go oƺácuje kápłán / bądź dobry / bądź zły ; jáko go oƺácuje kápłán / ták zoſtánie.
A gdyby ten / który poświęćił / chćiał odkupić dom ſwój / przydá piątą cżęść pieniędzy ná ƺácunek twój / y będźie jego.
Jeſli też kto cżęść roli z dźiedźictwá ſwego poświęćił Pánu / tedy będźie ƺácunek twój według záśiewku jey ; gdźie śię wyśieje chomer jęcżmieniá / zá pięćdźieśiąt ſyklów ſrebrá ƺácowáne będźie.
Jeſli do miłośćiwego látá poświęćił rolą ſwoję / według ƺácunku twego zoſtánie.
Ale jeſliżby po miłośćiwem lećie poświęćił rolą twoję / tedy kápłán obráchuje mu pieniądze według lat zoſtáwájących do miłośćiwego látá / y umniejƺy mu śię z ƺácunku twego.
A chćiáłliby odkupić rolą / ten / który ją poświęćił / przydá piątą cżęść pieniędzy do ƺácunku twego / y zoſtánie przy niej.
Ale gdźieby nie odkupił roli oney / á ſprzedánáby byłá rolá komu inƺemu / nie może być odkupioná.
Y będźie oná rolá / gdy wynijdźie miłośćiwe láto świętá Pánu / jáko rolá poświęconá ; á przydźie w ośiádłość kápłánowi.
A jeſliby kto rolą kupioną / która nie byłá z ról dźiedźictwá jego / poślubił Pánu /
Tedy poráchuje mu kápłán ſumę ƺácunku twego áż do roku miłośćiwego / y da ƺácunek ten dniá onego zá rzecż poświęconą Pánu.
A w miłośćiwe láto wróći śię rolá od tego / od kogo ją kupiono / do tego / który dźiedźicżnie trzymał rolą onę.
A káżdy ƺácunek twój będźie wedle ſyklá świątnicy / á dwádźieśćiá pieniędzy ſykiel wáży.
Wƺákże pierworodnego / á które práwem pierworodztwá bywá ofiárowáne Pánu z bydłá / nikt go nie poświęći / bądź wół bądź owcá / poniewáż Páńſkie ſą.
A jeſliby z bydląt niecżyſtych było / odkupi je według ƺácunku twego / y przydá piątą cżęść nád to ; á jeſliby go nie odkupiono / niechże ſprzedáne będźie według ƺácunku twego.
Káżda jednák rzecż poślubioná / którąby kto poślubił Pánu ze wƺyſtkiego / co ma z ludźi / y z bydłá / y z ról ośiádłośći ſwojey / nie będźie ſprzedáwáná / áni odkupowáná ; bo wƺelká rzecż poślubioná nájświętƺá jeſt Pánu.
Wƺelkie bydlę poślubione / które śię pod ślubem oddáwá / od cżłowieká nie będźie odkupione / ále śmierćią umrze.
Wƺyſtkie tákże dźieśięćiny źiemi z náśieniá źiemi / z owocu drzewá / Páńſkie ſą ; bo poświęcone ſą Pánu.
Ale ktoby chćiał odkupić co z dźieśięćin ſwojich / piątą cżęść ceny przydá do nich.
Tákże wƺyſtkie dźieśięćiny z rogátego bydłá / y z drobnego bydłá / wƺyſtkiego / co przechodźi pod láſką páſterſką / káżde dźieśiąte będźie poświęcone Pánu.
Nie będźie przebierał miedzy dobrem álbo złem / áni go odmieniáć będźie ; á jeſliby je jákokolwiek odmienił / będźie to y ono odmienione poświęcone / nie ma być odkupione.
Teć ſą przykázánia / które rozkázał Pán Mojżeƺowi do ſynów Izráelſkich ná górze Synáj.
