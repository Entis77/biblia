Potem rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á powiedz im : Gdyby cżłowiek ślubem poślubił duƺę Pánu / według ƺácunku twego dá okup.
A będźie ták ƺácunek twój : Zá mężcżyznę od dwudźieſtu lát áż do ƺeśćdźieśiąt lát / będźie ƺácunek twój pięćdźieśiąt ſyklów ſrebrá według wági świątnicy.
A jeźli jeſt biáłá głowá ƺácunek twój będźie trzydźieśći ſyklów.
A jeźli od piątego roku áż do dwudźieſtego roku / tedy będźie ƺácunek twój zá mężcżyznę dwádźieśćiá ſyklów / á zá biáłą głowę dźieśięć ſyklów.
A jeźli zá dźiećię od jednego mieśiącá áż do pięćiu lát / tedy będźie ƺácunek twój zá mężcżyznę pięć ſyklów ſrebrá / á zá dźiewecżkę ƺácunek twój trzy ſykle ſrebrá.
A jeźli od ƺeśćdźieśiąt lát y wyżej będźieli mężcżyzná / tedy będźie ƺácunek twój piętnáśćie ſyklów / á zá biáłą głowę dźieśięć ſyklów.
Lecż jeźliby był ták ubogi / żeby nie mógł oddáć ƺácunku twego / tedy go ſtáwią przed kápłáná / y oƺácuje go kápłán / według przemożeniá tego / który ślubowáł / oƺácuje go kápłán.
Jeźliby też bydlę z tych / które śię ofiárują ná ofiárę Pánu / poślubił / káżde / które oddá Pánu / będźie święte.
Nie odmieni go / áni dá innego zá nie / lepƺego zá gorƺe / álbo gorƺego zá lepƺe ; jeźliby też jákokolwiek odmienił bydlę zá bydlę / tedy y ono / y to / które zá nie dáno / będźie święte.
A jeźliby które niecżyſte bydlę poślubił z tych / co nie bywáją ofiárowáne Pánu / tedy ſtáwi to bydlę przed kápłáná.
Y oƺácuje kápłán bądź dobre / bądź złe / á jáko je oƺácuje kápłán / ták będźie.
A jeźliby je kto odkupić chćiáł / przydá piątą cżęść nád ƺácunek twój.
Jeźliby też kto poświęćił dom ſwój / żeby był święty Pánu / tedy go oƺácuje kápłán / bądź dobry / bądź zły ; jáko go oƺácuje kápłán / ták zoſtánie.
A gdyby ten / który poświęćił / chćiáł odkupić dom ſwój / przydá piątą cżęść pieniędzy ná ƺácunek twój / y będźie jego.
Jeźli też kto cżęść roli z dźiedźictwá ſwego poświęćił Pánu / tedy będźie ƺácunek twój według záśiewku jej ; gdźie śię wyśieje chomer jęcżmieniá / zá pięćdźieśiąt ſyklów ſrebrá ƺácowáne będźie.
Jeźli do miłośćiwego látá poświęćił rolą ſwoję / według ƺácunku twego zoſtánie.
Ale jeźliżby po miłośćiwem lećie poświęćił rolą twoję / tedy kápłán obráchuje mu pieniądze według lát zoſtáwájących do miłośćiwego látá / y umniejƺy mu śię z ƺácunku twego.
A chćiáłliby odkupić rolą / ten / który ją poświęćił / przydá piątą cżęść pieniędzy do ƺácunku twego / y zoſtánie przy niej.
Ale gdźieby nie odkupił roli onej / á ſprzedánáby byłá rolá komu inƺemu / nie może być odkupioná.
Y będźie oná rolá / gdy wynijdźie miłośćiwe láto świętá Pánu / jáko rolá poświęconá ; á przyjdźie w ośiádłość kápłánowi.
A jeźliby kto rolą kupioną / którá nie byłá z ról dźiedźictwá jego / poślubił Pánu /
Tedy poráchuje mu kápłán ſumę ƺácunku twego áż do roku miłośćiwego / y dá ƺácunek ten dniá onego zá rzecż poświęconą Pánu.
A w miłośćiwe láto wróći śię rolá od tego / od kogo ją kupiono / do tego / który dźiedźicżnie trzymáł rolą onę.
A káżdy ƺácunek twój będźie wedle ſyklá świątnicy / á dwádźieśćiá pieniędzy ſykiel wáży.
Wƺákże pierworodnego / á które práwem pierworodztwá bywá ofiárowáne Pánu z bydłá / nikt go nie poświęći / bądź wół bądź owcá / poniewáż Páńſkie ſą.
A jeźliby z bydląt niecżyſtych było / odkupi je według ƺácunku twego / y przydá piątą cżęść nád to ; á jeźliby go nie odkupiono / niechże ſprzedáne będźie według ƺácunku twego.
Káżdá jednák rzecż poślubioná / którąby kto poślubił Pánu ze wƺyſtkiego / co má z ludźi / y z bydłá / y z ról ośiádłośći ſwojej / nie będźie ſprzedáwáná / áni odkupowáná ; bo wƺelká rzecż poślubioná nájświętƺá jeſt Pánu.
Wƺelkie bydlę poślubione / które śię pod ślubem oddáwá / od cżłowieká nie będźie odkupione / ále śmierćią umrze.
Wƺyſtkie tákże dźieśięćiny źiemi z náśieniá źiemi / z owocu drzewá / Páńſkie ſą ; bo poświęcone ſą Pánu.
Ale ktoby chćiáł odkupić co z dźieśięćin ſwoich / piątą cżęść ceny przydá do nich.
Tákże wƺyſtkie dźieśięćiny z rogátego bydłá / y z drobnego bydłá / wƺyſtkiego / co przechodźi pod láſką páſterſką / káżde dźieśiąte będźie poświęcone Pánu.
Nie będźie przebieráł między dobrem álbo złem / áni go odmieniáć będźie ; á jeźliby je jákokolwiek odmienił / będźie to y ono odmienione poświęcone / nie má być odkupione.
Teć ſą przykázániá / które rozkázáł Pán Mojżeƺowi do ſynów Izráelſkich ná górze Synáj.
