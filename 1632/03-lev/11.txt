Potem mówił Pán do Mojżeƺá y do Aároná / y rzekł do nich : Powiedzćie ſynom Izráelſkim / mówiąc :
Te ſą zwierzętá / które jeść będziećie ze wƺyſtkich zwierząt / które ſą ná ziemi.
Wƺelkie bydlę / które má rozdzielone ſtopy / y rozdwojone kopytá / á przeżuwá / to jeść będziećie.
Ale z tych jeść nie będziećie / które tylko przeżuwáją / y z tych / które tylko kopytá dwoją : Wielbłąd / który choć przeżuwá / ále kopytá rozdzielonego nie má / niecżyſtym wám będzie.
Tákże królik / który choć przeżuwá / ále kopytá rozdzielonego nie má / niecżyſtym wám będzie.
Zájąc też / choć przeżuwá / ále kopytá rozdzielonego nie má / niecżyſtym wám będzie.
Świniá tákże / choć má rozdzielone ſtopy y rozdwojone kopyto / ále iż nie przeżuwá / niecżyſtą wám będzie.
Mięſá ich nie będziećie jeść / áni śćierwu ich dotykáć śię będziećie / niecżyſte wám będą.
To jeść będziećie ze wƺyſtkich rzecży żyjących w wodách / wƺyſtko / co má ſkrzele y łuſkę / w wodách / w morzu / y w rzekách / to jeść będziećie.
Wƺyſtko záś / co nie má ſkrzeli y łuſki w morzu y w rzekách / cokolwiek śię ruchá w wodách / y káżdá rzecż żywiącá / którá jeſt w wodách / obrzydliwośćią wám będzie.
Obrzydliwośćią będą wám ; mięſá ich jeść nie będziećie / á śćierwem ich brzydzić śię będziećie.
Owá cokolwiek nie má ſkrzeli y łuſki w wodách / obrzydliwośćią wám będzie.
Tem śię też brzydzić będzie z ptáſtwá / y jeść ich nie będziećie / bo ſą obrzydliwośćią ; jáko orłá / y gryfá / y morſkiego orłá /
Y ſępá / y káni / według rodzáju ich ;
Káżdego kruká według rodzáju jego ;
Tákże ſtruśiá / y ſowy y wodnej káni y jáſtrzębiá / według rodzáju ich ;
Y puchácżá / y norká / y lelká /
Y łábędziá / y bąká / y boćiáná /
Y cżápli / y ſojki / według rodzáju ich / y dudká / y nietoperzá.
Wƺyſtko / co śię cżołgá po ziemi ſkrzydłá májąc / á ná cżterech nogách chodzi / obrzydliwośćią wám będzie.
Wƺákże jeść będziećie wƺyſtko / co śię cżołgá po ziemi ſkrzydłá májące / co ná cżterech nogách chodzi / co má w nogách śćięgnecżká przedłużƺe ku ſkákániu ná nich po ziemi.
Te z nich jeść będziećie : Száráńcżą według rodzáju jej / y koniki według rodzáju ich / y ſkocżki według rodzáju ich / y chrząƺcże według rodzáju ich.
Wƺyſtko záś / co śię cżołgá po ziemi ſkrzydláſte / cżtery nogi májące / obrzydliwośćią wám będzie ;
Bo śię niemi pokáláćie. Ktoby śię dotknął zdechliny ich / nie będzie cżyſtym áż do wiecżorá ;
A ktobykolwiek nośił śćierw ich / upierze ƺáty ſwoje / y będzie niecżyſtym áż do wiecżorá.
Wƺelkie bydlę / które má rozdzieloną ſtopę / á kopytá rozdwojonego nie má / áni też przeżuwá / niecżyſte wám będzie ; ktoby śię go dotknął / niecżyſtym będzie.
A cokolwiek chodzi ná łápách ſwych ze wƺyſtkich zwierząt / które chodzą ná cżterech nogách / niecżyſte wám będzie ; ktoby śię dotknął śćierwu ich / niecżyſtym będzie áż do wiecżorá.
A ktoby nośił śćierw ich / upierze odzienie ſwe / á niecżyſtym będzie áż do wiecżorá : bo niecżyſte wám ſą.
Tákże y te zá niecżyſte mieć będziećie między płázámi / które śię włócżą po ziemi / łáśicá / y myƺ / y żábá według rodzájów ſwoich ;
Y jeż / y jáƺcżórká / y tchórz / y ślimák / y kret.
Te niecżyſte wám będą między wƺyſtkiemi płázámi ; ktoby śię dotknął zdechliny ich / niecżyſtym będzie áż do wiecżorá.
A káżdá rzecż / ná którąby co zdechłego z tych rzecży upádło / niecżyſtá będzie / ták drzewiáne nácżynie / jáko ƺátá / ták ſkórá / jáko wór ; owá káżde nácżynie / w którym co ſpráwują / do wody włożone będzie / y niecżyſte zoſtánie áż do wiecżorá / potem cżyſte będzie.
Wƺelkie záś nácżynie gliniáne / w któreby co z tych rzecży wpádło / ze wƺyſtkiem / coby w nim było / niecżyſte śię ſtánie / á ſámo ſtłucżone będzie.
Káżdá też potráwá / którą jádáją / gdyby wody niecżyſtej do niej wláno / niecżyſtą będzie ; y wƺelki nápój / który pijáją z káżdego tákiego nácżyniá / niecżyſtym będzie.
Owá wƺyſtko / ná coby upádło co z onych zdechlin / niecżyſte będzie ; piec y ogniſko rozwálone będą / bo niecżyſte ſą / y zá niecżyſte wám będą.
Ale ſtudniá y cyſterná / y káżde zgromádzenie wód cżyſte będą ; coby śię jednák dotknęło śćierwu tych rzecży / niecżyſte będzie.
A jeźliby upádło nieco z śćierwu ich ná jákie náśienie / które śiáne bywá / cżyſte zoſtánie.
Ale jeźliby ná náśienie w wodzie mocżone upádło co z śćierwu ich / niecżyſte wám będzie.
Jeźliby zdechło bydlę / które jádáćie : ktoby śię dotknął śćierwu jego / niecżyſtym będzie áż do wiecżorá.
A ktoby jádł śćierw jego / upierze ƺáty ſwoje / y niecżyſtym będzie áż do wiecżorá ; ten / coby precż wynośił on śćierw / upierze ƺáty ſwoje / y niecżyſtym będzie áż do wiecżorá.
Wƺelki tákże płáz / co śię cżołgá po ziemi / obrzydliwośćią jeſt ; nie będziećie go jeść.
Cokolwiek śię cżołgá po brzuchu / y cokolwiek ná cżterech álbo więcej nogách śię włócży między wƺyſtkim płázem / który śię cżołgá po ziemi / nie będziećie ich jeść / bo obrzydliwośćią ſą.
Nie plugáwćie duƺ wáƺych wƺelkim płázem / który śię cżołgá po ziemi / y nie máżćie śię niemi / byśćie nie byli ſplugáwieni przez nie ;
Abowiem Jám jeſt Pán Bóg wáƺ : przetoż poświęcájćie śię / á bądźćie świętymi / bom Já święty jeſt ; á nie plugáwćie duƺ wáƺych żádnym płázem / który śię cżołgá po ziemi.
Bom Já jeſt Pán / którym wáſ wywiódł z ziemi Egipſkiej / ábym wám był zá Bogá ; przetoż bądźćie świętymi / bom Já święty jeſt.
Táć jeſt uſtáwá około bydłá / y ptáſtwá / y wƺelkiej duƺy żywej / którá śię ruchá w wodách / y wƺelkiej duƺy żywej / którá śię cżołgá po ziemi.
Ku rozeznániu między niecżyſtem y między cżyſtem / á między zwierzęty / które śię jeść godzi / y między zwierzęty / których śię jeść nie godzi.
