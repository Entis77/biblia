Potem mówił Pán do Mojżeƺá y do Aároná / y rzekł do nich : Powiedzćie ſynom Izráelſkim / mówiąc :
Te ſą zwierzętá / które jeść będźiećie ze wƺyſtkich zwierząt / które ſą ná źiemi.
Wƺelkie bydlę / które ma rozdźielone ſtopy / y rozdwojone kopytá / á przeżuwá / to jeść będźiećie.
Ale z tych jeść nie będźiećie / które tylko przeżuwáją / y z tych / które tylko kopytá dwoją : Wielbłąd / który choć przeżuwá / ále kopytá rozdźielonego nie ma / niecżyſtym wam będźie.
Tákże królik / który choć przeżuwá / ále kopytá rozdźielonego nie ma / niecżyſtym wam będźie.
Zájąc też / choć przeżuwá / ále kopytá rozdźielonego nie ma / niecżyſtym wam będźie.
Świniá tákże / choć ma rozdźielone ſtopy y rozdwojone kopyto / ále iż nie przeżuwá / niecżyſtą wam będźie.
Mięſá ich nie będźiećie jeść / áni śćierwu ich dotykáć śię będźiećie / niecżyſte wam będą.
To jeść będźiećie ze wƺyſtkich rzecży żyjących w wodách / wƺyſtko / co ma ſkrzele y łuſkę / w wodách / w morzu / y w rzekách / to jeść będźiećie.
Wƺyſtko záś / co nie ma ſkrzeli y łuſki w morzu y w rzekách / cokolwiek śię ruchá w wodách / y káżda rzecż żywiącá / która jeſt w wodách / obrzydliwośćią wam będźie.
Obrzydliwośćią będą wam ; mięſá ich jeść nie będźiećie / á śćierwem ich brzydźić śię będźiećie.
Owá cokolwiek nie ma ſkrzeli y łuſki w wodách / obrzydliwośćią wam będźie.
Tym śię też brzydźić będźie z ptáſtwá / y jeść ich nie będźiećie / bo ſą obrzydliwośćią ; jáko orłá / y gryfá / y morſkiego orłá /
Y ſępá / y káni / według rodzáju ich ;
Káżdego kruká według rodzáju jego ;
Tákże ſtruśiá / y ſowy y wodney káni y jáſtrzębiá / według rodzáju ich ;
Y puchácżá / y norká / y lelká /
Y łábędźiá / y bąká / y boćiáná /
Y cżápli / y ſojki / według rodzáju ich / y dudká / y nietoperzá.
Wƺyſtko / co śię cżołgá po źiemi ſkrzydłá májąc / á ná cżterech nogách chodźi / obrzydliwośćią wam będźie.
Wƺákże jeść będźiećie wƺyſtko / co śię cżołgá po źiemi ſkrzydłá májące / co ná cżterech nogách chodźi / co ma w nogách śćięgnecżká przedłużƺe ku ſkákániu ná nich po źiemi.
Te z nich jeść będźiećie : Száráńcżą według rodzáju jey / y koniki według rodzáju ich / y ſkocżki według rodzáju ich / y chrząƺcże według rodzáju ich.
Wƺyſtko záś / co śię cżołgá po źiemi ſkrzydláſte / cżtery nogi májące / obrzydliwośćią wam będźie ;
Bo śię niemi pokáláćie. Ktoby śię dotknął zdechliny ich / nie będźie cżyſtym áż do wiecżorá ;
A ktobykolwiek nośił śćierw ich / upierze ƺáty ſwoje / y będźie niecżyſtym áż do wiecżorá.
Wƺelkie bydlę / które ma rozdźieloną ſtopę / á kopytá rozdwojonego nie ma / áni też przeżuwá / niecżyſte wam będźie ; ktoby śię go dotknął / niecżyſtym będźie.
A cokolwiek chodźi ná łápách ſwych ze wƺyſtkich zwierząt / które chodzą ná cżterech nogách / niecżyſte wam będźie ; ktoby śię dotknął śćierwu ich / niecżyſtym będźie áż do wiecżorá.
A ktoby nośił śćierw ich / upierze odźienie ſwe / á niecżyſtym będźie áż do wiecżorá : bo niecżyſte wam ſą.
Tákże y te zá niecżyſte mieć będźiećie miedzy płázámi / które śię włócżą po źiemi / łáśicá / y myƺ / y żábá według rodzájów ſwojich ;
Y jeż / y jáƺcżórká / y tchórz / y ślimák / y kret.
Te niecżyſte wam będą miedzy wƺyſtkiemi płázámi ; ktoby śię dotknął zdechliny ich / niecżyſtym będźie áż do wiecżorá.
A káżda rzecż / ná którąby co zdechłego z tych rzecży upádło / niecżyſtá będźie / ták drzewiáne nacżynie / jáko ƺátá / ták ſkórá / jáko wór ; owá káżde nacżynie / w którym co ſpráwują / do wody włożone będźie / y niecżyſte zoſtánie áż do wiecżorá / potem cżyſte będźie.
Wƺelkie záś nacżynie gliniáne / w któreby co z tych rzecży wpádło / ze wƺyſtkiem / coby w nim było / niecżyſte śię ſtánie / á ſámo ſtłucżone będźie.
Káżda też potráwá / którą jádáją / gdyby wody niecżyſtey do niej wláno / niecżyſtą będźie ; y wƺelki nápój / który pijáją z káżdego tákiego nácżyniá / niecżyſtym będźie.
Owá wƺyſtko / ná coby upádło co z onych zdechlin / niecżyſte będźie ; piec y ogniſko rozwálone będą / bo niecżyſte ſą / y zá niecżyſte wam będą.
Ale ſtudniá y cyſterná / y káżde zgromádzenie wód cżyſte będą ; coby śię jednák dotknęło śćierwu tych rzecży / niecżyſte będźie.
A jeſliby upádło nieco z śćierwu ich ná jákie náśienie / które śiáne bywá / cżyſte zoſtánie.
Ale jeſliby ná náśienie w wodźie mocżone upádło co z śćierwu ich / niecżyſte wam będźie.
Jeſliby zdechło bydlę / które jádáćie : ktoby śię dotknął śćierwu jego / niecżyſtym będźie áż do wiecżorá.
A ktoby jádł śćierw jego / upierze ƺáty ſwoje / y niecżyſtym będźie áż do wiecżorá ; ten / coby precż wynośił on śćierw / upierze ƺáty ſwoje / y niecżyſtym będźie áż do wiecżorá.
Wƺelki tákże płáz / co śię cżołgá po źiemi / obrzydliwośćią jeſt ; nie będźiećie go jeść.
Cokolwiek śię cżołgá po brzuchu / y cokolwiek ná cżterech álbo więcey nogách śię włócży miedzy wƺyſtkim płázem / który śię cżołgá po źiemi / nie będźiećie ich jeść / bo obrzydliwośćią ſą.
Nie plugáwćie duƺ wáƺych wƺelkim płázem / który śię cżołgá po źiemi / y nie máżćie śię niemi / byśćie nie byli ſplugáwieni przez nie ;
Abowiem Jam jeſt Pán Bóg wáƺ : przetoż poświęcájćie śię / á bądźćie świętymi / bom Ja święty jeſt ; á nie plugáwćie duƺ wáƺych żádnym płázem / który śię cżołgá po źiemi.
Bom Ja jeſt Pán / którym was wywiódł z źiemi Egipſkiej / ábym wam był zá Bogá ; przetoż bądźćie świętymi / bom Ja święty jeſt.
Táć jeſt uſtáwá około bydłá / y ptáſtwá / y wƺelkiej duƺy żywej / która śię ruchá w wodách / y wƺelkiej duƺy żywej / która śię cżołgá po źiemi.
Ku rozeznániu miedzy niecżyſtem y miedzy cżyſtem / á miedzy zwierzęty / które śię jeść godźi / y miedzy zwierzęty / których śię jeść nie godźi.
