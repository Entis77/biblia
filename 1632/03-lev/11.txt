Potem mówił Pán do Mojżeƺá y do Aároná / y rzekł do nich : Powiedzćie ſynom Izráelſkim / mówiąc :
Te ſą zwierzętá / które jeść będźiećie ze wƺyſtkich zwierząt / które ſą ná źiemi.
Wƺelkie bydlę / które má rozdźielone ſtopy / y rozdwojone kopytá / á przeżuwá / to jeść będźiećie.
Ale z tych jeść nie będźiećie / które tylko przeżuwáją / y z tych / które tylko kopytá dwoją : Wielbłąd / który choć przeżuwá / ále kopytá rozdźielonego nie má / niecżyſtym wám będźie.
Tákże królik / który choć przeżuwá / ále kopytá rozdźielonego nie má / niecżyſtym wám będźie.
Zájąc też / choć przeżuwá / ále kopytá rozdźielonego nie má / niecżyſtym wám będźie.
Świniá tákże / choć má rozdźielone ſtopy y rozdwojone kopyto / ále iż nie przeżuwá / niecżyſtą wám będźie.
Mięſá ich nie będźiećie jeść / áni śćierwu ich dotykáć śię będźiećie / niecżyſte wám będą.
To jeść będźiećie ze wƺyſtkich rzecży żyjących w wodách / wƺyſtko / co má ſkrzele y łuſkę / w wodách / w morzu / y w rzekách / to jeść będźiećie.
Wƺyſtko záś / co nie má ſkrzeli y łuſki w morzu y w rzekách / cokolwiek śię ruchá w wodách / y káżdá rzecż żywiącá / którá jeſt w wodách / obrzydliwośćią wám będźie.
Obrzydliwośćią będą wám ; mięſá ich jeść nie będźiećie / á śćierwem ich brzydźić śię będźiećie.
Owá cokolwiek nie má ſkrzeli y łuſki w wodách / obrzydliwośćią wám będźie.
Tem śię też brzydźić będźie z ptáſtwá / y jeść ich nie będźiećie / bo ſą obrzydliwośćią ; jáko orłá / y gryfá / y morſkiego orłá /
Y ſępá / y káni / według rodzáju ich ;
Káżdego kruká według rodzáju jego ;
Tákże ſtruśiá / y ſowy y wodnej káni y jáſtrzębiá / według rodzáju ich ;
Y puchácżá / y norká / y lelká /
Y łábędźiá / y bąká / y boćiáná /
Y cżápli / y ſojki / według rodzáju ich / y dudká / y nietoperzá.
Wƺyſtko / co śię cżołgá po źiemi ſkrzydłá májąc / á ná cżterech nogách chodźi / obrzydliwośćią wám będźie.
Wƺákże jeść będźiećie wƺyſtko / co śię cżołgá po źiemi ſkrzydłá májące / co ná cżterech nogách chodźi / co má w nogách śćięgnecżká przedłużƺe ku ſkákániu ná nich po źiemi.
Te z nich jeść będźiećie : Száráńcżą według rodzáju jej / y koniki według rodzáju ich / y ſkocżki według rodzáju ich / y chrząƺcże według rodzáju ich.
Wƺyſtko záś / co śię cżołgá po źiemi ſkrzydláſte / cżtery nogi májące / obrzydliwośćią wám będźie ;
Bo śię niemi pokáláćie. Ktoby śię dotknął zdechliny ich / nie będźie cżyſtym áż do wiecżorá ;
A ktobykolwiek nośił śćierw ich / upierze ƺáty ſwoje / y będźie niecżyſtym áż do wiecżorá.
Wƺelkie bydlę / które má rozdźieloną ſtopę / á kopytá rozdwojonego nie má / áni też przeżuwá / niecżyſte wám będźie ; ktoby śię go dotknął / niecżyſtym będźie.
A cokolwiek chodźi ná łápách ſwych ze wƺyſtkich zwierząt / które chodzą ná cżterech nogách / niecżyſte wám będźie ; ktoby śię dotknął śćierwu ich / niecżyſtym będźie áż do wiecżorá.
A ktoby nośił śćierw ich / upierze odźienie ſwe / á niecżyſtym będźie áż do wiecżorá : bo niecżyſte wám ſą.
Tákże y te zá niecżyſte mieć będźiećie między płázámi / które śię włócżą po źiemi / łáśicá / y myƺ / y żábá według rodzájów ſwoich ;
Y jeż / y jáƺcżórká / y tchórz / y ślimák / y kret.
Te niecżyſte wám będą między wƺyſtkiemi płázámi ; ktoby śię dotknął zdechliny ich / niecżyſtym będźie áż do wiecżorá.
A káżdá rzecż / ná którąby co zdechłego z tych rzecży upádło / niecżyſtá będźie / ták drzewiáne nácżynie / jáko ƺátá / ták ſkórá / jáko wór ; owá káżde nácżynie / w którym co ſpráwują / do wody włożone będźie / y niecżyſte zoſtánie áż do wiecżorá / potem cżyſte będźie.
Wƺelkie záś nácżynie gliniáne / w któreby co z tych rzecży wpádło / ze wƺyſtkiem / coby w nim było / niecżyſte śię ſtánie / á ſámo ſtłucżone będźie.
Káżdá też potráwá / którą jádáją / gdyby wody niecżyſtej do niej wláno / niecżyſtą będźie ; y wƺelki nápój / który pijáją z káżdego tákiego nácżyniá / niecżyſtym będźie.
Owá wƺyſtko / ná coby upádło co z onych zdechlin / niecżyſte będźie ; piec y ogniſko rozwálone będą / bo niecżyſte ſą / y zá niecżyſte wám będą.
Ale ſtudniá y cyſterná / y káżde zgromádzenie wód cżyſte będą ; coby śię jednák dotknęło śćierwu tych rzecży / niecżyſte będźie.
A jeźliby upádło nieco z śćierwu ich ná jákie náśienie / które śiáne bywá / cżyſte zoſtánie.
Ale jeźliby ná náśienie w wodźie mocżone upádło co z śćierwu ich / niecżyſte wám będźie.
Jeźliby zdechło bydlę / które jádáćie : ktoby śię dotknął śćierwu jego / niecżyſtym będźie áż do wiecżorá.
A ktoby jádł śćierw jego / upierze ƺáty ſwoje / y niecżyſtym będźie áż do wiecżorá ; ten / coby precż wynośił on śćierw / upierze ƺáty ſwoje / y niecżyſtym będźie áż do wiecżorá.
Wƺelki tákże płáz / co śię cżołgá po źiemi / obrzydliwośćią jeſt ; nie będźiećie go jeść.
Cokolwiek śię cżołgá po brzuchu / y cokolwiek ná cżterech álbo więcej nogách śię włócży między wƺyſtkim płázem / który śię cżołgá po źiemi / nie będźiećie ich jeść / bo obrzydliwośćią ſą.
Nie plugáwćie duƺ wáƺych wƺelkim płázem / który śię cżołgá po źiemi / y nie máżćie śię niemi / byśćie nie byli ſplugáwieni przez nie ;
Abowiem Jám jeſt Pán Bóg wáƺ : przetoż poświęcájćie śię / á bądźćie świętymi / bom Já święty jeſt ; á nie plugáwćie duƺ wáƺych żádnym płázem / który śię cżołgá po źiemi.
Bom Já jeſt Pán / którym wáſ wywiódł z źiemi Egipſkiej / ábym wám był zá Bogá ; przetoż bądźćie świętymi / bom Já święty jeſt.
Táć jeſt uſtáwá około bydłá / y ptáſtwá / y wƺelkiej duƺy żywej / którá śię ruchá w wodách / y wƺelkiej duƺy żywej / którá śię cżołgá po źiemi.
Ku rozeznániu między niecżyſtem y między cżyſtem / á między zwierzęty / które śię jeść godźi / y między zwierzęty / których śię jeść nie godźi.
