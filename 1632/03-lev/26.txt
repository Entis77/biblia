Nie cżyńćie ſobie báłwánów / áni obrázu rytego ; áni ſłupów ſtáwiájćie ſobie / áni kámieniá w obráz wyrytego ſtáwiájćie w ziemi wáƺej / ábyśćie mu śię kłániáli ; bom Já Pán / Bóg wáƺ.
Sábáty moje záchowywájćie / á świątnicę moję w ucżćiwośći miejćie ; Jám Pán.
Jeźli w uſtáwách moich chodzić będziećie / y przykázániá moje chowáć y cżynić będziećie :
Spuƺcżę wám deƺcż cżáſu ſwego / y wydá ziemiá urodzáj ſwój / y drzewá polne wydádzą owoc ſwój ;
Y trwáć będzie młoćbá do zbierániá winá / á zbieránie winá trwáć będzie do śiewu ; będziećie jeść chleb ſwój do ſytośći / y mieƺkáć będziećie bezpiecżnie w ziemi ſwej.
Bo dám pokój w ziemi / y będziećie ſpáli / á nie będzie / ktoby wáſ przeſtráƺył ; wyplenię też złego zwierzá z ziemi / á miecż nie przejdzie ziemi wáƺej.
Owƺem będziećie gonić nieprzyjáćioły wáƺe / y upádną przed wámi od miecżá.
Pięć wáƺych będą gonić ſto / á ſto wáƺych dzieśięć tyśięcy gonić będą / y polegną nieprzyjáćiele wáśi przed wámi od miecżá.
Bo obrócę śię do wáſ / á rozkrzewię wáſ / y rozmnożę wáſ / y utwierdzę przymierze moje z wámi.
Y będziećie jedli z dáwná záchowáłe zboże / y ſtáre / gdy nowe náſtáną / wyprzątniećie.
Y wyſtáwię przybytek mój między wámi / á nie uprzykrzy wáſ ſobie duƺá mojá.
Y będę chodził między wámi / á będę wám zá Bogá / á wy mnie będziećie zá lud.
Jám Pán / Bóg wáƺ / którym wáſ wywiódł z ziemi Egipſkiej / ábyśćie im nie ſłużyli ; y połámáłem łáńcuchy járzmá wáƺego / ábyśćie chodzili proſto.
A jeźlibyśćie mię nie ſłucháli / y nie cżynili wƺyſtkich tych przykázáń ;
Y jeźli uſtáwy moje wzgárdzićie / á ſądámi moimi będzie śię brzydziłá duƺá wáƺá / żebyśćie nie cżynili wƺyſtkich przykázáń moich / y wzruƺylibyśćie przymierze moje :
Já też wám to ucżynię : náwiedzę wáſ ſtráchem / ſuchotámi y gorącżką / które wám ocży popſują á boleśćią nápełnią duƺe wáƺe / á śiáć będziećie próżno náśienie wáƺe / bo je zjedzą nieprzyjáćiele wáśi ;
Y poſtáwię twárz moję przećiwko wám / y poráżeni będziećie od nieprzyjáćiół wáƺych / y pánowáć będą nád wámi / którzy wáſ máją w nienáwiśći ; y będziećie ućiekáli / choć wáſ nikt gonić nie będzie.
A jeźliż áni ták nie uſłucháćie mię / przydám śiedm kroć więcej kárániá dlá grzechów wáƺych ;
Y zetrę pychę mocy wáƺej / y ucżynię niebo nád wámi jáko żelázo / á ziemię wáƺę jáko miedź ;
Y wniwecż śię obróći prácá wáƺá ; bo nie wydá ziemiá wáƺá użytku ſwego / y drzewá ziemi nie wydádzą owocu ſwego.
A jeźli chodzić będziećie / mnie śię ſprzećiwiájąc / á nie zechcećie mię ſłucháć / przydám káźni wáƺych śiedmioráko dlá grzechów wáƺych.
Bo puƺcżę ná wáſ zwierz polny / y ośieroći wáſ / y wyniƺcży bydło wáƺe / y upleni wáſ / y ſpuſtoƺeją drogi wáƺe.
A jeźliż tem śię nie nákárzećie / ále chodzić będziećie / mnie śię ſprzećiwiájąc :
Já też pójdę wám śię ſprzećiwiájąc / y bić wáſ będę śiedmioráko dlá grzechów wáƺych ;
Y przywiodę ná wáſ miecż / który śię ſowićie zemśći z gwáłceniá przymierzá : á gdy śię zbieżyćie do miáſt wáƺych / tedy puƺcżę powietrze morowe między wáſ / á będziećie podáni w ręce nieprzyjáćielſkie.
A gdy złámię podporę chlebá wáƺego / będą piekły dzieśięć niewiáſt chleb wáƺ w piecu jednym / y będą wám oddáwáć chleb wáƺ pod wágą / y będziećie jeść / á nie nájećie śię.
A jeźli y przeto nie uſłucháćie mię / ále chodzić będziećie / mnie śię ſprzećiwiájąc :
Já też pójdę w gniewie przećiwko wám ; y Já też káráć wáſ będę śiedmioráko więcej dlá grzechów wáƺych.
Y będziećie jeść ćiáło ſynów wáƺych / y ćiáło córek wáƺych jeść będziećie.
Y wygubię po górách káplice wáƺe / á porozwálám ſłonecżne báłwány wáƺe ; y ſkłádę trupy wáƺe ná kloce obrzydłych báłwánów wáƺych / á będzie śię wámi brzydziłá duƺá mojá.
Y podám miáſtá wáƺe ná ſpuſtoƺenie / á poburzę świątnice wáƺe / y nie przyjmę więcej wdzięcżnej wonnośći wáƺej.
Y ſpuſtoƺę ziemię / że śię nád nią zdumieją nieprzyjáćiele wáśi / mieƺkájąc w niej.
A wáſ ſámych rozproƺę między národy / y dobędę zá wámi miecżá ; á będzie ziemiá wáƺá puſtá / y miáſtá wáƺe zburzone.
Tedy rádá będzie ziemiá odpocżynieniu ſwemu po wƺyſtkie dni ſpuſtoƺeniá ſwego ; á wy będziećie w ziemi nieprzyjáćiół wáƺych / tedy odpocżnie ziemiá / y rádá będzie odpocżynieniu ſwemu.
Przez wƺyſtkie dni ſpuſtoƺeniá ſwego odpocżywáć będzie ; bo nie miáłá odpocżynieniá w ſábáty wáƺe / gdyśćie wy mieƺkáli w niej.
A którzy z wáſ pozoſtáną / tedy przywiodę ſtrách ná ſercá ich / w ziemiách nieprzyjáćiół ich / że je gonić będzie chrzęſt liśćiá pádájącego ; y będą ućiekáli jáko przed miecżem / y pádáć będą / choćiáż ich nikt gonić nie będzie.
Y pádnie jeden ná drugiego jáko od miecżá / choć ich nikt gonić nie będzie ; áni śię oſtoićie przed nieprzyjáćioły wáƺymi.
Y poginiećie między národy / y pożre wáſ ziemiá nieprzyjáćiół wáƺych.
A którzy z wáſ zoſtáną / wywiędną dlá niepráwośći ſwojej w ziemi nieprzyjáćiół ſwoich ; tákże dlá niepráwośći ojców ſwych z nimi wywiędną.
Ale jeźli wyznáją niepráwość ſwoję / y niepráwość ojców ſwych według przeſtępſtwá ſwego / którem wyſtąpili przećiwko mnie / y według którego chodzili / ſprzećiwiájąc mi śię ;
Żem też y Já chodził ſprzećiwiájąc śię im / á iżem je wprowádził do ziemi nieprzyjáćiół ich ; jeźli / mówię / ná ten cżáſ poniży śię ſerce ich nieobrzezáne / y ćierpliwie znośić będą káźń zá niepráwośći ſwoje :
Tedy já też wſpomnę ná przymierze moje z Jákóbem / y ná przymierze moje z Izáákiem / y ná przymierze moje z Abráhámem wſpomnę / y ná tę ziemię wſpomnę.
A ziemiá będąc od nich uwolnioná / rádá będzie odpocżynieniu ſwemu / gdy puſtá będzie dlá nich ; á oni będą ćierpliwie nośić káránie zá niepráwość ſwą / przeto że ſądy moje wzgárdzili / y uſtáwámi mojemi brzydziłá śię duƺá ich.
Wƺákże dlá tego y ná ten cżáſ / gdy będą w ziemi nieprzyjáćiół ſwoich / nie odrzucę ich / áni ich ták ſobie obrzydzę / żebym je wyniƺcżyć miáł / y wzruƺyć przymierze moje z nimi ;
Bom Já Pán / Bóg ich. Ale wſpomnę ná nie dlá przymierzá ucżynionego z przodkámi ich ; którem wywiódł z ziemi Egipſkiej / przed ocżymá pogánów / ábym im był zá Bogá / Já Pán.
Teć ſą uſtáwy ; y ſądy y práwá / które poſtánowił Pán między ſobą / y między ſyny Izráelſkimi ná górze Synáj przez Mojżeƺá.
