Nie cżyńćie ſobie báłwánów / áni obrázu rytego ; áni ſłupów ſtáwiájćie ſobie / áni kámieniá w obraz wyrytego ſtáwiájćie w źiemi wáƺej / ábyśćie mu śię kłániáli ; bom Ja Pán / Bóg wáƺ.
Sábáty moje záchowywájćie / á świątnicę moję w ucżćiwośći miejćie ; Jam Pán.
Jeſli w uſtáwách mojich chodźić będźiećie / y przykázánia moje chowáć y cżynić będźiećie :
Spuƺcżę wam deƺcż cżáſu ſwego / y wydá źiemiá urodzáj ſwój / y drzewá polne wydádzą owoc ſwój ;
Y trwáć będźie młoćbá do zbieránia winá / á zbieránie winá trwáć będźie do śiewu ; będźiećie jeść chleb ſwój do ſytośći / y mieƺkáć będźiećie bezpiecżnie w źiemi ſwej.
Bo dam pokój w źiemi / y będźiećie ſpáli / á nie będźie / ktoby was przeſtráƺył ; wyplenię też złego zwierzá z źiemi / á miecż nie przejdźie źiemi wáƺej.
Owƺem będźiećie gonić nieprzyjáćioły wáƺe / y upádną przed wámi od miecżá.
Pięć wáƺych będą gonić ſto / á ſto wáƺych dźieśięć tyśięcy gonić będą / y polegną nieprzyjáćiele wáśi przed wámi od miecżá.
Bo obrócę śię do was / á rozkrzewię was / y rozmnożę was / y utwierdzę przymierze moje z wámi.
Y będźiećie jedli z dáwná záchowáłe zboże / y ſtáre / gdy nowe náſtáną / wyprzątniećie.
Y wyſtáwię przybytek mój miedzy wámi / á nie uprzykrzy was ſobie duƺá mojá.
Y będę chodźił miedzy wámi / á będę wam zá Bogá / á wy mnie będźiećie zá lud.
Jam Pán / Bóg wáƺ / którym was wywiódł z źiemi Egipſkiej / ábyśćie im nie ſłużyli ; y połámáłem łáńcuchy járzmá wáƺego / ábyśćie chodźili proſto.
A jeſlibyśćie mię nie ſłucháli / y nie cżynili wƺyſtkich tych przykázáń ;
Y jeſli uſtáwy moje wzgárdźićie / á ſądámi moimi będźie śię brzydźiłá duƺá wáƺá / żebyśćie nie cżynili wƺyſtkich przykázáń mojich / y wzruƺylibyśćie przymierze moje :
Ja też wam to ucżynię : náwiedzę was ſtráchem / ſuchotámi y gorącżką / które wam ocży popſują á boleśćią nápełnią duƺe wáƺe / á śiáć będźiećie prózno naśienie wáƺe / bo je zjedzą nieprzyjáćiele wáśi ;
Y poſtáwię twárz moję przećiwko wam / y poráżeni będźiećie od nieprzyjáćiół wáƺych / y pánowáć będą nád wámi / którzy was máją w nienáwiśći ; y będźiećie ućiekáli / choć was nikt gonić nie będźie.
A jeſliż áni ták nie uſłucháćie mię / przydám śiedm kroć więcey káránia dla grzechów wáƺych ;
Y zetrę pychę mocy wáƺej / y ucżynię niebo nád wámi jáko żelázo / á źiemię wáƺę jáko miedź ;
Y wniwecż śię obróći prácá wáƺá ; bo nie wydá źiemiá wáƺá użytku ſwego / y drzewá źiemi nie wydádzą owocu ſwego.
A jeſli chodźić będźiećie / mnie śię ſprzećiwiájąc / á nie zechcećie mię ſłucháć / przydám káźni wáƺych śiedmioráko dla grzechów wáƺych.
Bo puƺcżę ná was zwierz polny / y ośieroći was / y wyniƺcży bydło wáƺe / y upleni was / y ſpuſtoƺeją drogi wáƺe.
A jeſliż tym śię nie nákárzećie / ále chodźić będźiećie / mnie śię ſprzećiwiájąc :
Ja też pójdę wam śię ſprzećiwiájąc / y bić was będę śiedmioráko dla grzechów wáƺych ;
Y przywiodę ná was miecż / który śię ſowićie zemśći z gwáłceniá przymierza : á gdy śię zbieżyćie do miaſt wáƺych / tedy puƺcżę powietrze morowe miedzy was / á będźiećie podáni w ręce nieprzyjáćielſkie.
A gdy złámię podporę chlebá wáƺego / będą piekły dźieśięć niewiáſt chleb wáƺ w piecu jednym / y będą wam oddáwáć chleb wáƺ pod wágą / y będźiećie jeść / á nie nájećie śię.
A jeſli y przeto nie uſłucháćie mię / ále chodźić będźiećie / mnie śię ſprzećiwiájąc :
Ja też pójdę w gniewie przećiwko wam ; y Ja też káráć was będę śiedmioráko więcey dla grzechów wáƺych.
Y będźiećie jeść ćiáło ſynów wáƺych / y ćiáło córek wáƺych jeść będźiećie.
Y wygubię po górách káplice wáƺe / á porozwálám ſłonecżne báłwány wáƺe ; y ſkłádę trupy wáƺe ná kloce obrzydłych báłwánów wáƺych / á będźie śię wámi brzydźiłá duƺá mojá.
Y podám miáſtá wáƺe ná ſpuſtoƺenie / á poburzę świątnice wáƺe / y nie przyjmę więcey wdźięcżney wonnośći wáƺej.
Y ſpuſtoƺę źiemię / że śię nád nią zdumieją nieprzyjáćiele wáśi / mieƺkájąc w niej.
A was ſámych rozproƺę miedzy narody / y dobędę zá wámi miecżá ; á będźie źiemiá wáƺá puſtá / y miáſtá wáƺe zburzone.
Tedy rádá będźie źiemiá odpocżynieniu ſwemu po wƺyſtkie dni ſpuſtoƺeniá ſwego ; á wy będźiećie w źiemi nieprzyjáćiół wáƺych / tedy odpocżnie źiemiá / y rádá będźie odpocżynieniu ſwemu.
Przez wƺyſtkie dni ſpuſtoƺeniá ſwego odpocżywáć będźie ; bo nie miáłá odpocżynieniá w ſábáty wáƺe / gdyśćie wy mieƺkáli w niej.
A którzy z was pozoſtáną / tedy przywiodę ſtrách ná ſercá ich / w źiemiách nieprzyjáćiół ich / że je gonić będźie chrzęſt liśćiá pádájącego ; y będą ućiekáli jáko przed miecżem / y pádáć będą / choćiáż ich nikt gonić nie będźie.
Y pádnie jeden ná drugiego jáko od miecżá / choć ich nikt gonić nie będźie ; áni śię oſtoićie przed nieprzyjáćioły wáƺymi.
Y poginiećie miedzy narody / y pożre was źiemiá nieprzyjáćiół wáƺych.
A którzy z was zoſtáną / wywiędną dla niepráwośći ſwojey w źiemi nieprzyjáćiół ſwojich ; tákże dla niepráwośći ojców ſwych z nimi wywiędną.
Ale jeſli wyznáją niepráwość ſwoję / y niepráwość ojców ſwych według przeſtępſtwá ſwego / którem wyſtąpili przećiwko mnie / y według którego chodźili / ſprzećiwiájąc mi śię ;
Żem też y Ja chodźił ſprzećiwiájąc śię im / á iżem je wprowádźił do źiemi nieprzyjáćiół ich ; jeſli / mówię / ná ten cżás poniży śię ſerce ich nieobrzezáne / y ćierpliwie znośić będą káźń zá niepráwośći ſwoje :
Tedy ja też wſpomnę ná przymierze moje z Jákóbem / y ná przymierze moje z Izáákiem / y ná przymierze moje z Abráhámem wſpomnę / y ná tę źiemię wſpomnę.
A źiemiá będąc od nich uwolnioná / rádá będźie odpocżynieniu ſwemu / gdy puſtá będźie dla nich ; á oni będą ćierpliwie nośić káránie zá niepráwość ſwą / przeto że ſądy moje wzgárdźili / y uſtáwámi mojemi brzydźiłá śię duƺá ich.
Wƺákże dla tego y ná ten cżás / gdy będą w źiemi nieprzyjáćiół ſwojich / nie odrzucę ich / áni ich ták ſobie obrzydzę / żebym je wyniƺcżyć miał / y wzruƺyć przymierze moje z nimi ;
Bom Ja Pán / Bóg ich. Ale wſpomnę ná nie dla przymierza ucżynionego z przodkámi ich ; którem wywiódł z źiemi Egipſkiej / przed ocżymá pogánów / ábym im był zá Bogá / Ja Pán.
Teć ſą uſtáwy ; y ſądy y práwá / które poſtánowił Pán miedzy ſobą / y miedzy ſyny Izráelſkimi ná górze Synáj przez Mojżeƺá.
