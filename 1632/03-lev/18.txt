Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / y rzecż im : Jam jeſt Pán / Bóg wáƺ.
Według obycżájów źiemi Egipſkiej / w którejśćie mieƺkáli / nie cżyńćie / áni według obycżájów źiemi Chánánejſkiej / do którey Ja was prowádzę / nie cżyńćie / á w uſtáwách ich nie chodźćie.
Sądy moje cżyńćie / á uſtáw mojich ſtrzeżćie / ábyśćie chodźili w nich ; Jam Pán / Bóg wáƺ.
Przeſtrzegájćież tedy uſtáw mojich y ſądów mojich : które záchowywájąc cżłowiek / będźie w nich żył ; Jam Pán.
Żáden cżłowiek do bliſkiej pokrewney ſwojey nieprzyſtępuj / áby odkrył ſromotę jey ; Jam Pán.
Sromoty ojcá twego / tákże ſromoty mátki twojey nie odkryjeƺ ; mátką twoją jeſt / nie odkryjeƺ ſromoty jey.
Sromoty żony ojcá twego nie odkryjeƺ ; ſromotá ojcá twego jeſt.
Sromoty śioſtry twej / córki ojcá twego / tákże córki mátki twej / ták rodzoney / jáko y przyrodney / nie odkryjeƺ ſromoty ich.
Sromoty córki ſyná twego / tákże ſromoty córki córki twojey / nie odkryjeƺ ; bo to ſromotá twojá.
Sromoty córki żony ojcá twego / która śię národźiłá z ojcá twego / śioſtrá twojá jeſt / nie odkryjeƺ ſromoty jey.
Sromoty śioſtry ojcá twego nie odkryjeƺ ; bo jeſt pokrewná ojcá twego.
Sromoty śioſtry mátki twojey nie odkryjeƺ ; bo pokrewná mátki twojey jeſt.
Sromoty brátá ojcá twego nie odkryjeƺ / do żony jego nie wnijdźieƺ ; żoná ſtryjá twego jeſt.
Sromoty ſynowej twojey nie odkryjeƺ ; żoná jeſt ſyná twego / nie odkryjeƺ ſromoty jey ;
Sromoty żony brátá twego nie odkryjeƺ ; ſromotá brátá twego jeſt.
Sromoty żony / y córki jey / nie odkryjeƺ ; córki ſyná jey / y córki córki jey nie pojmieƺ / ábyś odkrył ſromotę jey ; bo pokrewne ſą / y ſproſná to rzecż jeſt.
Sioſtry żony twej nie pojmuj / ábyś jey nie trápił / odkrywájąc ſromotę jey / póki oná żywá.
Do niewiáſty / gdy jeſt w odłącżeniu niecżyſtośći / nieprzyſtępuj / ábyś odkrył ſromotę jey.
Z żoną bliźniego twego obcowáć nie będźieƺ / bobyś śię ſplugáwił z nią /
Z náśieniá twego nie dopuƺcżáj ofiárowáć Molochowi / ábyś nie ſplugáwił imienia Bogá twego ; Jam Pán.
Z mężcżyzną nie będźieƺ obcował / jáko z niewiáſtą ; obrzydliwośćią to jeſt.
Tákże z bydlęćiem żádnem obcowáć nie będźieƺ / ábyś śię z nim miał ſplugáwiáć. Niewiáſtá też niech nie podlegá bydlęćiu dla obcowánia z nim ; ſprośná rzecż jeſt.
Nie plugáwćież śię temi wƺyſtkiemi rzecżámi ; bo tym wƺyſtkim ſplugáwili śię pogánie / które Ja wyrzucám przed oblicżem wáƺem.
Bo ſplugáwiłá śię źiemiá ; przetoż náwiedzę niepráwość jey ná niej / y wyrzući źiemiá obywátele ſwoje.
A ták wy przeſtrzegájćie uſtáw mojich y ſądów mojich / á niecżyńćie żádnych obrzydliwośći tych / w domu zrodzony / y przychodźień / który jeſt gośćiem w pośrzodku was.
Abowiem wƺyſtkie te obrzydliwośći cżynili ludźie tey źiemi / którzy byli przed wámi / cżym ſplugáwioná jeſt źiemiá.
Aby was niewyrzućiłá źiemiá / gdybyśćie ją ſplugáwili / jáko wyrzućiłá naród / który był przed wámi.
Abowiem ktobykolwiek co ucżynił z tych wƺyſtkich obrzydliwośći / zájiſte wytrácone będą duƺe to cżyniące z pośrzodku ludu ſwego.
Strzeżćież tedy uſtáw mojich / nie cżyniąc uſtáw obrzydliwych / które cżyniono przed wámi / áni śię plugáwćie niemi ; Jam Pán / Bóg wáƺ.
