Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / y rzecż im : Jám jeſt Pán / Bóg wáƺ.
Według obycżájów źiemi Egipſkiej / w którejśćie mieƺkáli / nie cżyńćie / áni według obycżájów źiemi Chánánejſkiej / do której Já wáſ prowádzę / nie cżyńćie / á w uſtáwách ich nie chodźćie.
Sądy moje cżyńćie / á uſtáw moich ſtrzeżćie / ábyśćie chodźili w nich ; Jám Pán / Bóg wáƺ.
Przeſtrzegájćież tedy uſtáw moich y ſądów moich : które záchowywájąc cżłowiek / będźie w nich żył ; Jám Pán.
Żáden cżłowiek do bliſkiej pokrewnej ſwojej nieprzyſtępuj / áby odkrył ſromotę jej ; Jám Pán.
Sromoty ojcá twego / tákże ſromoty mátki twojej nie odkryjeƺ ; mátką twoją jeſt / nie odkryjeƺ ſromoty jej.
Sromoty żony ojcá twego nie odkryjeƺ ; ſromotá ojcá twego jeſt.
Sromoty śioſtry twej / córki ojcá twego / tákże córki mátki twej / ták rodzonej / jáko y przyrodnej / nie odkryjeƺ ſromoty ich.
Sromoty córki ſyná twego / tákże ſromoty córki córki twojej / nie odkryjeƺ ; bo to ſromotá twojá.
Sromoty córki żony ojcá twego / którá śię národźiłá z ojcá twego / śioſtrá twojá jeſt / nie odkryjeƺ ſromoty jej.
Sromoty śioſtry ojcá twego nie odkryjeƺ ; bo jeſt pokrewná ojcá twego.
Sromoty śioſtry mátki twojej nie odkryjeƺ ; bo pokrewná mátki twojej jeſt.
Sromoty brátá ojcá twego nie odkryjeƺ / do żony jego nie wnijdźieƺ ; żoná ſtryjá twego jeſt.
Sromoty ſynowej twojej nie odkryjeƺ ; żoná jeſt ſyná twego / nie odkryjeƺ ſromoty jej ;
Sromoty żony brátá twego nie odkryjeƺ ; ſromotá brátá twego jeſt.
Sromoty żony / y córki jej / nie odkryjeƺ ; córki ſyná jej / y córki córki jej nie pojmieƺ / ábyś odkrył ſromotę jej ; bo pokrewne ſą / y ſproſná to rzecż jeſt.
Sioſtry żony twej nie pojmuj / ábyś jej nie trápił / odkrywájąc ſromotę jej / póki oná żywá.
Do niewiáſty / gdy jeſt w odłącżeniu niecżyſtośći / nieprzyſtępuj / ábyś odkrył ſromotę jej.
Z żoną bliźniego twego obcowáć nie będźieƺ / bobyś śię ſplugáwił z nią /
Z náśieniá twego nie dopuƺcżáj ofiárowáć Molochowi / ábyś nie ſplugáwił imieniá Bogá twego ; Jám Pán.
Z mężcżyzną nie będźieƺ obcowáł / jáko z niewiáſtą ; obrzydliwośćią to jeſt.
Tákże z bydlęćiem żádnem obcowáć nie będźieƺ / ábyś śię z nim miáł ſplugáwiáć. Niewiáſtá też niech nie podlegá bydlęćiu dlá obcowániá z nim ; ſprośná rzecż jeſt.
Nie plugáwćież śię temi wƺyſtkiemi rzecżámi ; bo tem wƺyſtkiem ſplugáwili śię pogánie / które Já wyrzucám przed oblicżem wáƺem.
Bo ſplugáwiłá śię źiemiá ; przetoż náwiedzę niepráwość jej ná niej / y wyrzući źiemiá obywátele ſwoje.
A ták wy przeſtrzegájćie uſtáw moich y ſądów moich / á niecżyńćie żádnych obrzydliwośći tych / w domu zrodzony / y przychodźień / który jeſt gośćiem w pośrodku wáſ.
Abowiem wƺyſtkie te obrzydliwośći cżynili ludźie tej źiemi / którzy byli przed wámi / cżem ſplugáwioná jeſt źiemiá.
Aby wáſ niewyrzućiłá źiemiá / gdybyśćie ją ſplugáwili / jáko wyrzućiłá náród / który był przed wámi.
Abowiem ktobykolwiek co ucżynił z tych wƺyſtkich obrzydliwośći / záiſte wytrácone będą duƺe to cżyniące z pośrodku ludu ſwego.
Strzeżćież tedy uſtáw moich / nie cżyniąc uſtáw obrzydliwych / które cżyniono przed wámi / áni śię plugáwćie niemi ; Jám Pán / Bóg wáƺ.
