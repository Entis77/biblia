Rzekł jeƺcze PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / y rzecz im : Jam jeſt PAN Bóg wáƺ.
Według obycżájów źiemie Egipſkiey w któryśćie mieƺkáli / nie czyńćie / áni według obyczájów źiemie Chánánejſkiey / do którey ja was prowádzę / nie czyńćie / á w uſtáwách ich nie chodźćie.
Sądy moje czyńćie / á uſtaw moich ſtrzeżćie / ábyśćie chodźili w nich : <i>JAM PAN</i> Bóg wáƺ.
Przeſtrzegajćież tedy uſtaw moich / y ſądów moich : które záchowywájąc cżłowiek / będźie w nich żył : <i>JAM PAM.</i>
Żaden cżłowiek do bliſkey pokrewney ſwojey nieprzyſtępuj / áby odkrył ſromotę jey : <i>JAM PAN.</i>
Sromoty Ojcá twego / tákże ſromoty mátki twojey nie odkryjeƺ : mátką twoją jeſt / nie odkryjeƺ ſromoty jey.
Sromoty żony Ojcá twego nie odkryjeƺ / Sromotá Ojcá twego jeſt.
Sromoty śioſtry twey / córki Ojcá twego / tákże córki mátki twey / ták rodzoney jáko y przyrodney / nie odkryjeƺ ſromoty ich.
Sromoty córki ſyná twego / tákże <i>ſromoty</i> córki córki twojey nie odkryjeƺ : bo to ſromotá twojá.
Sromoty córki żony Ojcá twego która śię národźiłá z Ojcá twego / śioſtrá twojá jeſt / nie odkryjeƺ ſromoty jey.
Sromoty śioſtry Ojcá twego nie odkryjeƺ / bo jeſt pokrewna Ojcá twego.
Sromoty śioſtry mátki twojey nie odkryjeƺ / bo pokrewna mátki twojey jeſt.
Sromoty brátá Ojcá twego nie odkryjeƺ / do żony jego nie wnidźieƺ / żoná ſtryjá twego jeſt.
Sromoty ſynowey twojey nie odkryjeƺ / żoną jeſt ſyná twego / nie odkryjeƺ ſromoty jey.
Sromoty żony brátá twego nie odkryjeƺ : ſromotá brátá twego jeſt.
Sromoty Żony / y córki jey nie odkryjeƺ : córki ſyná jey / y córki Córki jey nie pojmieƺ / ábyś odkrył ſromotę jey / bo pokrewne ſą / y ſproſna <i>to</i> rzecz jeſt.
Sioſtry żony twey nie pojmuj / ábyś jey nie trapił / odkrywájąc ſromotę jey / póki oná żywá.
Do niewiáſty / gdy jeſt w odłączeniu nieczyſtośći / nieprzyſtępuj / ábyś odkrył ſromotę jey.
Z żoną bliźniego twego obcowáć nie będźieƺ / bo byś śię ſplugáwił z nią.
Z naśienia twego nie dopuƺczaj ofiárowáć Molochowi / ábyś nie ſplugáwił imienia Bogá twego : <i>JAM PAN.</i>
Z mężcżyzną nie będźieƺ obcował jáko z niewiáſtą : obrzydliwośćią <i>to</i> jeſt.
Tákże z bydlęćiem żadnym obcowáć nie będźieƺ / ábyś śię z nim miał ſplugáwiáć : Niewiáſtá też niech nie podlega bydlęćiu / dla obcowánia z nim : ſproſna rzecz jeſt.
Nie plugawćież śię tymi wƺyſtkimi rzecżámi / bo tym wƺyſtkim ſplugáwili śię Pogánie / które ja wyrzucam przed obliczē wáƺym.
Bo ſplugáwiłá śię źiemiá / przetoż náwiedzę niepráwość jey ná niey / y wyrzući źiemiá obywátele ſwoje.
A ták wy przeſtrzegajćie uſtaw mojich y ſądów mojich á niecżyńćie żadnych obrzydliwośći tych : w domá zrodzony / y przychodźień / który jeſt gośćiem w pośrzodku was.
( Abowiem wƺyſtkie te obrzydliwośći czynili ludźie tey źiemie / którzy <i>byli</i> przed wámi / czym ſplugáwiona jeſt źiemiá. )
Aby was nie wyrzućiłá źiemiá / gdybyśćie ją ſplugáwili / jáko wyrzućiłá Naród który był przed wámi.
Abowiem ktobykolwiek <i>co</i> uczynił z tych wƺyſtkich obrzydliwośći / záiſte / wytrácone będą duƺe to czyniące z pośrzodku ludu ſwego.
Strzeżćieƺ tedy uſtaw mojich / nie czyniąc uſtaw obrzydliwych / które czyniono przed wámi / áni śię plugawćie nimi / <i>JAM PAN,</i> Bóg wáƺ.
