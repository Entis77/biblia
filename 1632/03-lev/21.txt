Rzekł też PAN do Mojzeƺá : Mów do Kápłanów ſynów Aáronowych / á powiedz im : Niech śię nád umárłym nie plugáwi <i>żaden Kápłan</i> w ludu ſwym.
Tylko przy pokrewnym ſwojim / powinowátym ſwojim / przy Mátce ſwey / y przy Ojcu ſwym / y przy ſynu ſwym / y przy córce ſwey / y przy bráćie ſwym /
Tákże przy śieſtrze ſwey Pánnie / ſobie nabliżƺey / która nie miáłá mężá : przy tych ſplugáwić śię może.
Nie ſplugáwi śię <i>przy</i> Przełożonym ludu ſwego / ták żeby śię zmázał.
Nie będą ſobie czynić łyśiny ná głowie ſwey / y brody ſwey nie máją golić / áni ná ćiele ſwym czynić będą rzezánia :
Świętymi będą Bogu ſwemu / y nie ſplugáwią imienia Bogá ſwego : ábowiem Ofiáry ogniſte PAńſkie / chleb Bogá ſwego / ofiárują : przetoż będą świętymi.
Niewiáſty wƺeteczney / y <i>w panienſtwie</i> náruƺoney pojmowáć nie będą : tákże Niewiáſty odrzuconey od mężá jey pojmowáć nie będą / bo święty jeſt káżdy <i>z nich</i> Bogu ſwemu.
A ták będźieƺ go miał zá świętego / bo chleb Bogá twego ofiáruje / przetoż świętym będźie tobie / bom <i>JA</i> święty <i>PAN /</i> który poświącam was.
Jesliby śię córká Kápłańſka nierządu dopuśćiłá / Ojcá ſwego zelżyłá : ogniem ſpalona będźie.
Nawyżƺy też Kápłan miedzy bráćią ſwą ná którego głowę wylany jeſt Olejek pomázánia / <i>y</i> który poświęćił ręce ſwe / áby obłoczył ƺáty <i>święte,</i> głowy ſwey nie obnáży / y ƺat ſwojich nie rozedrze.
Y do żadnego z umárłych nie przyſtąpi / á <i>náwet,</i> przy Ojcu ſwym / y przy mátce ſwey / plugáwić śię nie będźie.
Z świątnice też nie wynidźie / áby nie ſplugáwił świątnice Bogá ſwego / gdyż koroná Olejku pomázánia Bogá jego <i>jeſt</i> ná nim : <i>JA PAN.</i>
Tenże Pánnę w pánieńſtwie jey pojmie.
Wdowy / y odrzuconey / y ſplugáwioney nierządnice / żadney z tych nie pojmie : ále Pánnę z ludu ſwego weźmie ſobie zá żonę.
A nie będźie plugáwił naśienia ſwego w ludu ſwym : bom <i>JA PAN</i> który go poświącam.
Przytym rzekł PAN do Mojzeƺá / mówiąc :
Powiedz Aáronowi / y rzecz : ktobykolwiek z potomſtwá twego w narodźiech ſwych miał ná ſobie wádę niechaj nie przyſtępuje / áby ofiárował chleb Bogá ſwego :
Bo żaden mąż któryby miał ná ſobie wádę przyſtępowáć nie ma : mąż ślepy / álbo chromy / álbo nie zupełnych / álbo zbytnich cżłonków :
Tákże mąż któryby miał złąmáną nogę / álbo złąmáną rękę :
Tákże gárbáty / y płynących ocżu / álbo który ma bielmo ná oku ſwym / álbo kroſtáwy álbo párƺywy / álbo wypukły.
Wƺelki mąż któryby miał jáką wádę / z potomſtwá Aároná Kápłaná nie przyſtąpi áby ofiárował / Ofiáry ogniſte PAnu : wádá ná nim jeſt / nie przyſtąpi áby ofiárował chleb Bogá ſwego.
Chlebá <i>jednák</i> Bogá ſwego / z rzeczy naświętƺych y poświęconych pożywáć będźie.
Wƺákże zá záſłonę nie wnidźie / y do Ołtarzá nie przyſtąpi / bo wádá ná nim jeſt / áby nie ſplugáwił świątnice mojey / bom <i>JA PAN /</i> który je poświącam.
To mówił Mojzeƺ do Aároná / y do ſynów jego / y do wƺyſtkich ſynów Izráelſkich.
