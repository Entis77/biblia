Rzekł też Pán do Mojżeƺá : Mów do kápłánów / ſynów Aáronowych / á powiedz im : Niech śię nád umárłym nie plugáwi żáden kápłán w ludu ſwym ;
Tylko przy pokrewnym ſwoim / powinowátym ſwoim / przy mátce ſwej / y przy ojcu ſwym / y przy ſynu ſwym / y przy córce ſwej / y przy bráćie ſwym ;
Tákże przy śioſtrze ſwej / pánnie ſobie nájbliżƺej / która nie miáłá mężá ; przy tych ſplugáwić śię może.
Nie ſplugáwi śię przy przełożonym ludu ſwego / ták żeby śię zmázał.
Nie będą ſobie cżynić łyśiny ná głowie ſwej / y brody ſwej nie máją golić / áni ná ćiele ſwem cżynić będą rzezánia.
Świętymi będą Bogu ſwemu / y nie ſplugáwią imieniá Bogá ſwego ; ábowiem ofiáry ogniſte Páńſkie / chleb Bogá ſwego / ofiárują ; przetoż będą świętymi.
Niewiáſty wƺetecżney / y w pánieńſtwie náruƺoney / pojmowáć nie będą ; tákże niewiáſty odrzuconey od mężá jey / pojmowáć nie będą ; bo święty jeſt káżdy z nich Bogu ſwemu.
A ták będźieƺ go miał zá świętego / bo chleb Bogá twego ofiáruje ; przetoż świętym będźie tobie / bom Ja święty Pán / który poświęcám was.
Jeſliby śię córká kápłáńſká nierządu dopuśćiłá / ojcá ſwego zelżyłá / ogniem ſpáloná będźie.
Nájwyżƺy też kápłán miedzy bráćią ſwą / ná którego głowę wylány jeſt olejek pomázánia / y który poświęćił ręce ſwe / áby obłocżył ƺáty święte / głowy ſwej nie obnáży / y ƺát ſwojich nie rozedrze ;
Y do żádnego z umárłych nie przyſtąpi / á náwet y przy ojcu ſwym / y przy mátce ſwej plugáwić śię nie będźie.
Z świątnicy też nie wynijdźie / áby nie ſplugáwił świątnicy Bogá ſwego / gdyż koroná olejku pomázánia Bogá jego jeſt ná nim : Jam Pán.
Tenże pánnę w pánieńſtwie jey pojmie.
Wdowy / y odrzuconey y ſplugáwioney nierządnicy / żádney z tych nie pojmie ; ále pánnę z ludu ſwego weźmie ſobie zá żonę.
A nie będźie plugáwił náśieniá ſwego w ludu ſwym ; bom Ja Pán / który go poświęcám.
Przytem rzekł Pán do Mojżeƺá / mówiąc :
Powiedz Aáronowi / y rzecż : Ktobykolwiek z potomſtwá ſwego w národźiech ſwych / miał ná ſobie wádę / niecháj nie przyſtępuje / áby ofiárował chleb Bogá ſwego ;
Bo żáden mąż / któryby miał ná ſobie wádę / przyſtępowáć nie ma ; mąż ślepy / álbo chromy / álbo niezupełnych álbo zbytnich cżłonków ;
Tákże mąż / któryby miał złámáną nogę / álbo złámáną rękę ;
Tákże gárbáty / y płynących ocżu / álbo który ma bielmo ná oku ſwem / álbo kroſtáwy / álbo párƺywy / álbo wypukły :
Wƺelki mąż / któryby miał jáką wádę / z potomſtwá Aároná kápłáná / nie przyſtąpi / áby ofiárował ofiáry ogniſte Pánu ; wádá ná nim jeſt / nie przyſtąpi / áby ofiárował chleb Bogá ſwego.
Chlebá jednák Bogá ſwego z rzecży nájświętƺych y poświęconych pożywáć będźie.
Wƺákże zá záſłonę nie wnijdźie / y do ołtárzá nie przyſtąpi / bo wádá ná nim jeſt / áby nie ſplugáwił świątnicy mojey ; bom Ja Pán / który ją poświęcám.
To mówił Mojzeƺ do Aároná / y do ſynów jego / y do wƺyſtkich ſynów Izráelſkich.
