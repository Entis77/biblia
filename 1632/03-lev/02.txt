Gdyby też kto ofiárowáć chćiał dar ofiáry śniedney PAnu / pƺenna mąká będźie ofiárá jego : y poleje ją Oliwą / y nákłádźie ná nię kádźidłá.
Y przynieśie ją do ſynów Aáronowych / kápłanów / á weźmie z tąd pełną garść ſwoję tey pƺenney mąki / y tey oliwy / ze wƺyſtkim kádźidłem / y zápali to kápłán / ná pámiątkę jey / ná Ołtarzu : Ofiárá ogniſta jeſt ku wdźięcżney wonnośći PAnu.
Ale co zoſtánie od oney Ofiáry śniedney / Aáronowi y ſynom jego będźie : Naświętƺa rzecż jeſt z ogniſtych Ofiar PAnu.
A jeslibyś też ofiárował dar Ofiáry śniedney / w piecu pieczoney / <i>niechże będźie</i> z pƺenney mąki / plácek przásny / zágniećiony w Oliwie / y kreple przásne / pomázáne Oliwą.
Jesliże záś Ofiárę śniedną ſmáżoną w panwi ofiárowáć będźieƺ / niechże będźie z mąki pƺenney / zágniećioney w Oliwie oprócz zákwáƺenia.
Połamieƺ ją ná kęsy / y polejeƺ ją Oliwą : Ofiárá to śniedna <i>jeſt</i>.
A jesli Ofiárę śniedną w kotle zgotowáną ofiárowáć będźieƺ / z mąki pƺenney z Oliwą będźie.
Y przynieśieƺ Ofiárę śniedną z tych rzeczy ſpráwioną PANU / y oddaƺ ją Kápłánowi / á odnieśie ją ná Ołtarz.
Y weźmie Kápłán z oney Ofiáry śniedney / pámiątkę jey / y zápali ná Ołtarzu : Ofiárá to ogniſta / ku wdźięczney wonnośći PAnu.
A co pozoſtánie od oney Ofiáry śniedney / Aáronowi / y ſynom jego <i>będźie</i> : naświętƺa rzecz jeſt / z ogniſtych ofiar PANU.
Wƺelka Ofiárá śniedna którą ofiárowáć będźiećie PANU / bez kwáſu będźie : bo żadnego kwáſu / y żadnego miodu / nie będźiećie zápaláć ná Ofiárę ogniſtą PANU :
Tylko w Ofiárách pierwiaſtek / ofiárowáć to będźiećie PAnu / ále ná Ołtarz nie będźiećie ich kłáść / ku wdźięczney wonnośći.
Káżdy dar Ofiáry twojey śniedney / ſolą poſoliƺ : á nie odejmieƺ ſoli przymierza Bogá twojego od Ofiáry twojey śniedney : przy káżdey Ofierze twojey / ofiárowáć będźieƺ ſól.
A jesli ofiárowáć będźieƺ Ofiárę śniedną z pierwƺych urodzájów PAnu / świeże kłosy upráżyƺ ogniem / <i>á</i> zboże wykruƺone z kłosów świeżych / ofiárowáć będźieƺ ná Ofiárę śniedną / pierwƺych urodzájów twojich.
Y nálejeƺ ná nię Oliwy / á nákłádźieƺ ná nię kádźidłá / <i>bo</i> Ofiárá śniedna jeſt.
Tedy zápali Kápłán / pámiątkę jey / ze zbożá wykruƺonego jey / y z Oliwy jey / ze wƺyſtkiem kádźidłem jey / <i>bo</i> ofiárá ogniſta <i>jeſt</i> PAnu.
