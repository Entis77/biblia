Jesliby też cżłowiek zgrzeƺył / żeby ſłyƺał głos bluźnierſtwá / á byłby świádkiem <i>tego co</i> álbo widźiał álbo ſłyƺał / á nie oznájmiłby / ponieśie karánie zá niepráwość ſwoję.
Albo jeſliby śię kto dotknął rzecży niecżyſtey / bądź śćierwu zwierzá nieczyſtego / bądź śćierwu bydlęćiá nieczyſtego / bądź śćierwu gádźiny niecżyſtey / á byłoby to zákryto przed nim / przećię nieczyſty będźie / y winien jeſt.
Albo / jeſliby śię <i>kto</i> dotknął nieczyſtośći cżłowieczey / jákażbykolwiek byłá nieczyſtość jego / przez którą śię nieczyſtym ſtawa / á byłoby to ſkryto przed nim / y dowiedźiałby śię / winien jeſt :
Albo / jeſliby kto przyśiągł / wymówiwƺy uſty / że źle álbo dobrze uczynił / O wƺyſtko / co wymawia cżłowiek z przyśięgą / á byłoby to ſkryto przed nim / y dowiedźiałby śię <i>potym</i> że winien jeſt / w jedney rzeczy z tych.
Będąc tedy winien / w jedney rzeczy z tych / wyzna grzech ſwój :
Y przywiedźie Ofiárę zá winę ſwoję PANU / zá grzech ſwój którym zgrzeƺył / ſámicę z drobnego bydłá / Owcę álbo kozę zá grzech : á oczyśći go kápłán od grzechu jego.
A jeſliby nie przemógł ofiárowáć bydlątká / tedy przynieśie ofiárę zá wyſtępek ſwój / którym zgrzeƺył parę Synogárlic / álbo párę gołąbiąt PAnu : jedno ná Ofiárę zá grzech / á drugie / ná Ofiárę Cáłopalenia.
Y przynieśie je do kápłáná / á <i>on</i> naprzód Ofiárowáć będźie / <i>to</i> co ma być ná ofiárę zá grzech / y páznogćiem nádrze głowę jego ku ƺyjey / ále jey nie oderwie :
Y pokropi krwią z Ofiáry zá grzech ſtronę Ołtarzá / á oſtátek oney krwie wyćiśnie u ſpodku Ołtarzá : Ofiárá to zá grzech <i>jeſt</i>.
Z drugiego záśię / uczyni ofiárę cáłopalenia / według zwyczáju : A ták oczyśći go kápłán od grzechu jego / którym zgrzeƺył / á będźie mu odpuƺcżony.
A jeſliby nie przemógł Ofiárowáć pary ſynogárlic álbo dwojgá gołąbiąt / tedy przynieśie Ofiárę ſwoję zá to co zgrzeƺył / dźieśiątą część Efy mąki pƺenney / ná Ofiárę zá grzech : nie náleje ná nię Oliwy / áni włoży ná nię kádźidłá / bo jeſt Ofiárá zá grzech :
A <i>gdy</i> ją przynieśie do kápłáná / tedy nábrawƺy kápłán z niey pełną garść ſwoję ná Pámiątkę jego ſpali ją ná Ołtarzu / mimo Ofiárę ogniſtą PAnu : Ofiárá to zá grzech <i>jeſt</i>.
Y ocżyśći go kápłán od grzechu jego / którym zgrzeƺył / w któreykolwiek z tych rzecży / á będźie mu odpuƺczony / á <i>oſtátek</i> będźie kápłánowi jáko przy Ofierze śniedney.
Nád to rzekł PAN do Mojzeƺá / mówiąc :
Gdyby cżłowiek przeſtąpił przeſtępſtwem / á zgrzeƺyłby z niewiádomośći / <i>ujmując</i> rzecży poświęconych PAnu / tedy przywiedźie Ofiárę zá wyſtępek ſwój PANU / báráná zupełnego / z drobnego bydłá / według oƺácowánia twego / zá ſykle śrebrá według ſyklá świątnice <i>ná Ofiárę</i> zá wyſtępek.
A to / coby wźiął z poświęconych rzeczy / wróći / y piątą część nád to przyda / y odda Kápłánowi : á Kápłán go oczyśći przez báráná ofiáry zá grzech / á będźie mu odpuƺczono.
Jeſliby też cżłowiek zgrzeƺył / uczyniwƺy przećiw któremukolwiek z przykazań PAńſkich / coby nie miáło być / z niewiádomośći / á byłby winien / ponieśie karánie zá niepráwość ſwoję :
Tedy przywiedźie báráná zupełnego z drobnego bydłá / według ƺácunku twojego / ná Ofiárę zá wyſtępek / do kápłáná : y oczyśći go kápłán od niewiádomośći jego / którey śię dopuśćił nie wiedząc / á będźie mu odpuƺczono.
Ofiárá to zá wyſtępek jeſt / którym wyſtąpił przećiwko PANU.
