Jeſliby też cżłowiek zgrzeƺył / żeby ſłyƺał głos bluźnierſtwá / á byłby świádkiem tego / co álbo widźiał / álbo ſłyƺał / á nie oznájmiłby / ponieśie káránie zá niepráwość ſwoję.
Albo jeſliby śię kto dotknął rzecży niecżyſtey / bądź śćierwu zwierzá niecżyſtego / bądź śćierwu bydlęćiá niecżyſtego / bądź śćierwu gádźiny niecżyſtey / á byłoby to zákryto przed nim / przećię niecżyſty będźie / y winien jeſt.
Albo jeſliby śię kto dotknął niecżyſtośći cżłowiecżey / jákáżbykolwiek byłá niecżyſtość jego / przez którą śię niecżyſtym ſtáwá / á byłoby to ſkryto przed nim / y dowiedźiáłby śię / winien jeſt.
Albo jeſliby kto przyśiągł wymówiwƺy uſty / że źle álbo dobrze ucżynił / o wƺyſtko / co wymáwiá cżłowiek z przyśięgą / á byłoby to ſkryto przed nim y dowiedźiáłby śię potem / że winien jeſt w jedney rzecży z tych :
Będąc tedy winien w jedney rzecży z tych / wyzná grzech ſwój ;
Y przywiedźie ofiárę zá winę ſwoję Pánu zá grzech ſwój / którym zgrzeƺył / ſámicę z drobnego bydłá / owcę / álbo kozę zá grzech / á ocżyśći go kápłán od grzechu jego.
A jeſliby nie przemógł ofiárowáć bydlątká / tedy przynieśie ofiárę zá wyſtępek ſwój / którym zgrzeƺył / párę ſynogárlic / álbo párę gołąbiąt Pánu / jedno ná ofiárę zá grzech / á drugie ná ofiárę cáłopalenia ;
Y przynieśie je do kápłáná / á on náprzód ofiárowáć będźie to / co ma być ná ofiárę zá grzech / y páznogćiem nádrze głowę jego ku ƺyi / ále jey nie oderwie.
Y pokropi krwią z ofiáry zá grzech ſtronę ołtárzá / á oſtátek oney krwi wyćiśnie u ſpodku ołtárzá. Ofiárá to zá grzech jeſt.
Z drugiego záśię ucżyni ofiárę cáłopalenia według zwycżáju. A ták ocżyśći go kápłán od grzechu jego / którym zgrzeƺył / á będźie mu odpuƺcżony.
A jeſliby nie przemógł ofiárowáć párę ſynogárlic / álbo párę gołąbiąt / tedy przynieśie ofiárę ſwoję zá to / co zgrzeƺył / dźieśiątą cżęść efy mąki pƺenney ná ofiárę zá grzech / nie náleje ná nię oliwy / áni włoży ná nię kádźidłá ; bo jeſt ofiárá zá grzech.
A gdy ją przynieśie do kápłáná / tedy nábráwƺy kápłán z niej pełną gárść ſwoję ná pámiątkę jego / ſpáli ją ná ołtárzu mimo ofiárę ogniſtą Pánu ; ofiárá to zá grzech jeſt.
Y ocżyśći go kápłán od grzechu jego / którym zgrzeƺył w którejkolwiek z tych rzecży / á będźie mu odpuƺcżony ; á oſtátek będźie kápłánowi / jáko przy ofierze śniedney.
Nád to rzekł Pán do Mojżeƺá / mówiąc :
Gdyby cżłowiek przeſtąpił przeſtępſtwem / á zgrzeƺyłby z niewiádomośći / ujmując rzecży poświęconych Pánu / tedy przywiedźie ofiárę zá wyſtępek ſwój Pánu / báráná zupełnego z drobnego bydłá według oƺácowánia twego / zá dwá ſykle ſrebrá według ſyklá świątnicy / ná ofiárę zá wyſtępek.
A to / coby wźiął z poświęconych rzecży / wróći / y piątą cżęść nádto przydá y oddá kápłánowi ; á kápłán go ocżyśći przez báráná ofiáry zá grzech / á będźie mu odpuƺcżono.
Jeſliby też cżłowiek zgrzeƺył / ucżyniwƺy przećiw któremukolwiek z przykázáń Páńſkich / coby nie miáło być / z niewiádomośći / á byłby winien / ponieśie káránie zá niepráwość ſwoję.
Tedy przywiedźie báráná zupełnego z drobnego bydłá według ƺácunku twojego / ná ofiárę zá wyſtępek do kápłáná ; y ocżyśći go kápłán od niewiádomośći jego / którey śię dopuśćił nie wiedząc / á będźie mu odpuƺcżono.
Ofiárá to zá wyſtępek jeſt / którym wyſtąpił przećiwko Pánu.
