Jeźliby też cżłowiek zgrzeƺył / żeby ſłyƺáł głoſ bluźnierſtwá / á byłby świádkiem tego / co álbo widziáł / álbo ſłyƺáł / á nie oznájmiłby / ponieśie káránie zá niepráwość ſwoję.
Albo jeźliby śię kto dotknął rzecży niecżyſtej / bądź śćierwu zwierzá niecżyſtego / bądź śćierwu bydlęćiá niecżyſtego / bądź śćierwu gádziny niecżyſtej / á byłoby to zákryto przed nim / przećię niecżyſty będzie / y winien jeſt.
Albo jeźliby śię kto dotknął niecżyſtośći cżłowiecżej / jákáżbykolwiek byłá niecżyſtość jego / przez którą śię niecżyſtym ſtáwá / á byłoby to ſkryto przed nim / y dowiedziáłby śię / winien jeſt.
Albo jeźliby kto przyśiągł wymówiwƺy uſty / że źle álbo dobrze ucżynił / o wƺyſtko / co wymáwiá cżłowiek z przyśięgą / á byłoby to ſkryto przed nim y dowiedziáłby śię potem / że winien jeſt w jednej rzecży z tych :
Będąc tedy winien w jednej rzecży z tych / wyzná grzech ſwój ;
Y przywiedzie ofiárę zá winę ſwoję Pánu zá grzech ſwój / którym zgrzeƺył / ſámicę z drobnego bydłá / owcę / álbo kozę zá grzech / á ocżyśći go kápłán od grzechu jego.
A jeźliby nie przemógł ofiárowáć bydlątká / tedy przynieśie ofiárę zá wyſtępek ſwój / którym zgrzeƺył / párę ſynogárlic / álbo párę gołąbiąt Pánu / jedno ná ofiárę zá grzech / á drugie ná ofiárę cáłopáleniá ;
Y przynieśie je do kápłáná / á on náprzód ofiárowáć będzie to / co má być ná ofiárę zá grzech / y páznogćiem nádrze głowę jego ku ƺyi / ále jej nie oderwie.
Y pokropi krwią z ofiáry zá grzech ſtronę ołtárzá / á oſtátek onej krwi wyćiśnie u ſpodku ołtárzá. Ofiárá to zá grzech jeſt.
Z drugiego záśię ucżyni ofiárę cáłopáleniá według zwycżáju. A ták ocżyśći go kápłán od grzechu jego / którym zgrzeƺył / á będzie mu odpuƺcżony.
A jeźliby nie przemógł ofiárowáć párę ſynogárlic / álbo párę gołąbiąt / tedy przynieśie ofiárę ſwoję zá to / co zgrzeƺył / dzieśiątą cżęść efy mąki pƺennej ná ofiárę zá grzech / nie náleje ná nię oliwy / áni włoży ná nię kádzidłá ; bo jeſt ofiárá zá grzech.
A gdy ją przynieśie do kápłáná / tedy nábráwƺy kápłán z niej pełną gárść ſwoję ná pámiątkę jego / ſpáli ją ná ołtárzu mimo ofiárę ogniſtą Pánu ; ofiárá to zá grzech jeſt.
Y ocżyśći go kápłán od grzechu jego / którym zgrzeƺył w którejkolwiek z tych rzecży / á będzie mu odpuƺcżony ; á oſtátek będzie kápłánowi / jáko przy ofierze śniednej.
Nád to rzekł Pán do Mojżeƺá / mówiąc :
Gdyby cżłowiek przeſtąpił przeſtępſtwem / á zgrzeƺyłby z niewiádomośći / ujmując rzecży poświęconych Pánu / tedy przywiedzie ofiárę zá wyſtępek ſwój Pánu / báráná zupełnego z drobnego bydłá według oƺácowániá twego / zá dwá ſykle ſrebrá według ſyklá świątnicy / ná ofiárę zá wyſtępek.
A to / coby wziął z poświęconych rzecży / wróći / y piątą cżęść nádto przydá y oddá kápłánowi ; á kápłán go ocżyśći przez báráná ofiáry zá grzech / á będzie mu odpuƺcżono.
Jeźliby też cżłowiek zgrzeƺył / ucżyniwƺy przećiw któremukolwiek z przykázáń Páńſkich / coby nie miáło być / z niewiádomośći / á byłby winien / ponieśie káránie zá niepráwość ſwoję.
Tedy przywiedzie báráná zupełnego z drobnego bydłá według ƺácunku twojego / ná ofiárę zá wyſtępek do kápłáná ; y ocżyśći go kápłán od niewiádomośći jego / której śię dopuśćił nie wiedząc / á będzie mu odpuƺcżono.
Ofiárá to zá wyſtępek jeſt / którym wyſtąpił przećiwko Pánu.
