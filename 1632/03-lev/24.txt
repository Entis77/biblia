Y rzekł Pán do Mojżeƺá / mówiąc :
Rozkáż ſynom Izráelſkim / ábyć przynieśli oliwy z drzewá oliwnego cżyſtey / wytłocżoney ku świeceniu / áby lámpy gorzáły uſtáwicżnie.
Przed záſłoną świádectwá w námioćie zgromádzenia ſporządźi je Aáron / áby śię páliły od wiecżorá áż do poránku przed oblicżem Páńſkim uſtáwicżnie ; uſtáwá to wiecżná w národźiech wáƺych.
Ná świecżniku cżyſtym ſtáwiáć będźie lámpy przed oblicżem Páńſkim záwżdy.
Weźmieƺ też mąki pƺenney / á upiecżeƺ z niej dwánáśćie plácków ; ze dwu dźieśiątych cżęśći efy będźie plácek jeden.
Potem położyƺ je dwiemá rzędámi / ƺeść w rzędźie jednym ná ſtole cżyſtym przed oblicżem Páńſkim.
Włożyƺ też ná káżdy rząd kádźidłá cżyſtego / áby było miáſto chlebá ſpálone / ná pámiątkę ku ofierze ogniſtey Pánu.
Ná káżdy dźień ſábátu kłáść je będźie kápłán porządnie przed Pánem záwżdy / biorąc je od ſynów Izráelſkich przymierzem wiecżnem.
Y będą náleżáły Aáronowi y ſynom jego / którzy je jeść będą ná miejſcu świętym / ábowiem rzecżą im to nájświętƺą jeſt z ogniſtych ofiár Páńſkich / uſtáwą wiecżną.
Tedy wyƺedł ſyn niewiáſty Izráelſkiej / którego miáłá z mężem Egipſkim / miedzy ſyny Izráelſkimi ; y poſwárzyli śię w oboźie ſyn oney niewiáſty Izráelſkiej z mężem Izráelſkim.
Y złorzecżył ſyn niewiáſty Izráelſkiej á imię Boże bluźnił / dla cżego przywiedźiony był do Mojżeƺá. A imię mátki jego było Sálomit / córká Dybrego z pokolenia Dán.
Y podáli go do więźieniá / áżby im oznájmiono / co z nim rozkáże Pán cżynić.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Wywiedź tego bluźniercę precż zá obóz / á niech włożą wƺyſcy / którzy to ſłyƺeli / ręce ſwe ná głowę jego / y niech go ukámionuje wƺyſtko zgromádzenie.
A ſynom Izráelſkim opowiedz / mówiąc : Ktobykolwiek złorzecżył Bogu ſwemu / odnieśie káránie zá grzech ſwój.
Tákże ktoby zbluźnił imię Páńſkie / śmierćią umrze / kámionując ukámionuje go wƺyſtko zgromádzenie ; ták przychodźień jáko w domu zrodzony / gdyby zbluźnił imię Páńſkie / umrze.
Tákże jeſliby kto zábił jákiegokolwiek cżłowieká / śmierćią umrze.
A jeſliby kto zábił bydlę / wróći inne bydlę zá bydlę.
Ktoby też oƺkárádźił bliźniego ſwego / według tego / jáko ucżynił / niech mu śię ſtánie.
Złámánie zá złámánie / oko zá oko / ząb zá ząb ; według tego / jáko oƺkárádźił cżłowieká / ták mu śię też niech ſtánie.
Ktoby zábił bydlę / wróći inƺe ; ále ktoby zábił cżłowieká / umrze.
Práwo jednákie mieć będźiećie ; ták przychodźień / jáko y w domu zrodzony będźie u was ; bom ja Pán / Bóg wáƺ.
To gdy opowiedźiał Mojzeƺ ſynom Izráelſkim / wywiedli onego bluźniercę zá obóz / y ukámionowáli go.
Y ucżynili ſynowie Izráelſcy według tego / jáko przykázał Pán Mojżeƺowi.
