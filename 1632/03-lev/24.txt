Y rzekł Pán do Mojżeƺá / mówiąc :
Rozkáż ſynom Izráelſkim / ábyć przynieśli oliwy z drzewá oliwnego cżyſtej / wytłocżonej ku świeceniu / áby lámpy gorzáły uſtáwicżnie.
Przed záſłoną świádectwá w námioćie zgromádzeniá ſporządzi je Aáron / áby śię páliły od wiecżorá áż do poránku przed oblicżem Páńſkim uſtáwicżnie ; uſtáwá to wiecżná w národziech wáƺych.
Ná świecżniku cżyſtym ſtáwiáć będzie lámpy przed oblicżem Páńſkim záwżdy.
Weźmieƺ też mąki pƺennej / á upiecżeƺ z niej dwánáśćie plácków ; ze dwu dzieśiątych cżęśći efy będzie plácek jeden.
Potem położyƺ je dwiemá rzędámi / ƺeść w rzędzie jednym ná ſtole cżyſtym przed oblicżem Páńſkim.
Włożyƺ też ná káżdy rząd kádzidłá cżyſtego / áby było miáſto chlebá ſpálone / ná pámiątkę ku ofierze ogniſtej Pánu.
Ná káżdy dzień ſábátu kłáść je będzie kápłán porządnie przed Pánem záwżdy / biorąc je od ſynów Izráelſkich przymierzem wiecżnem.
Y będą náleżáły Aáronowi y ſynom jego / którzy je jeść będą ná miejſcu świętym / ábowiem rzecżą im to nájświętƺą jeſt z ogniſtych ofiár Páńſkich / uſtáwą wiecżną.
Tedy wyƺedł ſyn niewiáſty Izráelſkiej / którego miáłá z mężem Egipſkim / między ſyny Izráelſkimi ; y poſwárzyli śię w obozie ſyn onej niewiáſty Izráelſkiej z mężem Izráelſkim.
Y złorzecżył ſyn niewiáſty Izráelſkiej á imię Boże bluźnił / dlá cżego przywiedziony był do Mojżeƺá. A imię mátki jego było Sálomit / córká Dybrego z pokoleniá Dán.
Y podáli go do więzieniá / áżby im oznájmiono / co z nim rozkáże Pán cżynić.
Tedy rzekł Pán do Mojżeƺá / mówiąc :
Wywiedź tego bluźniercę precż zá obóz / á niech włożą wƺyſcy / którzy to ſłyƺeli / ręce ſwe ná głowę jego / y niech go ukámionuje wƺyſtko zgromádzenie.
A ſynom Izráelſkim opowiedz / mówiąc : Ktobykolwiek złorzecżył Bogu ſwemu / odnieśie káránie zá grzech ſwój.
Tákże ktoby zbluźnił imię Páńſkie / śmierćią umrze / kámionując ukámionuje go wƺyſtko zgromádzenie ; ták przychodzień jáko w domu zrodzony / gdyby zbluźnił imię Páńſkie / umrze.
Tákże jeźliby kto zábił jákiegokolwiek cżłowieká / śmierćią umrze.
A jeźliby kto zábił bydlę / wróći inne bydlę zá bydlę.
Ktoby też oƺkárádził bliźniego ſwego / według tego / jáko ucżynił / niech mu śię ſtánie.
Złámánie zá złámánie / oko zá oko / ząb zá ząb ; według tego / jáko oƺkárádził cżłowieká / ták mu śię też niech ſtánie.
Ktoby zábił bydlę / wróći inƺe ; ále ktoby zábił cżłowieká / umrze.
Práwo jednákie mieć będziećie ; ták przychodzień / jáko y w domu zrodzony będzie u wáſ ; bom já Pán / Bóg wáƺ.
To gdy opowiedziáł Mojżeƺ ſynom Izráelſkim / wywiedli onego bluźniercę zá obóz / y ukámionowáli go.
Y ucżynili ſynowie Izráelſcy według tego / jáko przykázáł Pán Mojżeƺowi.
