Y rzekł PAN do Mojzeƺá mówiąc :
Rozkaż Synom Izráelſkim / ábyć przynieśli Oliwy z drzewá Oliwnego / czyſtey / wytłocżoney / ku świeceniu / áby lámpy gorzáły uſtáwicznie.
Przed zaſłoną Świádectwá / w Namiećie zgromádzenia ſporządźi je Aáron / <i>áby śię paliły</i> od wieczorá áż do poránku / przed obliczem PAńſkim uſtáwicznie : uſtáwá <i>to</i> wieczna w Narodźiech wáƺych.
Ná świecżniku cżyſtym ſtawiáć będźie lámpy przed obliczem PAńſkim záwżdy.
Weźmieƺ też mąki pƺenney / á upieczeƺ z niey dwánaśćie plácków / ze dwu dźieśiątych częśći <i>Efa</i> będźie plácek jeden.
Potym położyƺ je dwiemá rzędámi / ƺeść w rzędźie <i>jednym</i> ná ſtole czyſtym / przed obliczem PAńſkim.
Włożyƺ też ná <i>káżdy</i> rząd kádźidłá czyſtego / áby było miáſto chlebá ſpalone ná pámiątkę ku ofierze ogniſtey PAnu.
Ná káżdy dźień Sábbátu kłáść je będźie <i>Kápłan</i> porządnie przed PAnem záwżdy : biorąc je od Synów Izráelſkich / przymierzem wiecznym.
Y będą <i>należáły</i> Aáronowi y Synom jego / którzy je jeść będą ná miejſcu świętym : ábowiem rzecżą im to naświętƺą jeſt / z ogniſtych ofiár PAńſkich / uſtáwą wiecżną.
Tedy wyƺedł Syn niewiáſty Izráelſkiey / którego miáłá z mężem Egipſkim / miedzy Syny Izráelſkimi / y poſwárzyli śię w Oboźie / Syn <i>oney</i> Niewiáſty Izráelſkiey z mężem Izráelſkim.
Y złorzeczył Syn niewiáſty Izráelſkiey / á imię <i>Boże</i> bluźnił : dla czego przywiedźiony był do Mojzeƺá ( A imię mátki jego było Sálomit córká Dibrego z pokolenia Dán. )
Y podáli go do więźienia / áżby im oznájmiono co <i>z nim</i> rozkaże PAN czynić.
Tedy rzekł PAN do Mojzeƺá / mówiąc :
Wywiedź tego bluźniercę precz zá Obóz / á niech włożą wƺyſcy którzy <i>to</i> ſłyƺeli ręce ſwe ná głowę jego / y niech go ukámionuje wƺyſtko zgromádzenie.
A Synom Izráelſkim opowiedz / mówiąc / ktobykolwiek złorzeczył Bogu ſwemu / odnieśie <i>karánie</i> zá grzech ſwój.
Tákże ktoby zbluźnił imię PAńſkie / śmierćią umrze / kámionując ukámionuje go wƺyſtko zgromádzenie : ták przychodźień / jáko domá zrodzony gdyby zbluźnił imię PAńſkie / umrze.
Tákże jeſliby kto zábił jákiegokolwiek cżłowieká / śmierćią umrze.
A jeſliby kto zábił bydlę / wróći inne bydlę zá bydlę.
Ktoby też oƺkárádźił bliźniego ſwego / według tego / jáko uczynił / niech mu śię ſtánie :
Złamánie / zá złamánie / oko zá oko / ząb zá ząb / według tego / jáko oƺkárádźił cżłowieká / ták mu śię też niech ſtánie.
Ktoby zábił bydlę / wróći inƺe : ále ktoby zábił cżłowieká / umrze.
Práwo jednákie mieć będźiećie / ták przychodźień / jáko y domá zrodzony będźie <i>u was,</i> bom <i>JA PAN</i> / Bóg wáƺ.
To <i>gdy</i> opowiedźiał Mojzeƺ Synom Izráelſkim / wywiedli onego bluźniercę zá obóz / y ukámionowáli go :
Y uczynili Synowie Izráelſcy według tego / jáko przykazał PAN Mojzeƺowi.
