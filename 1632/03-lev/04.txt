Y rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á rzecż : Gdyby kto zgrzeƺył z niewiádomośći / przećiw któremu ze wƺyſtkich przykázáń Páńſkich / cżyniąc / cżegoby cżynić nie miáł / á przeſtąpiłby jedno z nich ;
Jeźliby kápłán pomázány zgrzeƺył jáko jeden z ludu poſpolitego / tedy niech ofiáruje zá grzech ſwój / którego śię dopuśćił / ćielcá młodego zupełnego Pánu ná ofiárę zá grzech.
Y przywiedzie ćielcá onego do drzwi námiotu zgromádzeniá przed oblicżność Páńſką / á włoży rękę ſwoję ná głowę onego ćielcá / y zábije go przed oblicżem Páńſkim.
Tedy weźmie kápłán pomázány ze krwi onego ćielcá / y wnieśie ją do námiotu zgromádzeniá.
Potem omocży kápłán pálec ſwój we krwi / á kropić będzie oną krwią śiedm kroć przed oblicżem Páńſkim przed záſłoną świątnicy.
Y pomáże kápłán krwią oną rogi ołtárzá kádzeniá wonnego / przed oblicżem Páńſkim / który jeſt w námioćie zgromádzeniá / á oſtátek krwi onego ćielcá wyleje u ſpodku ołtárzá cáłopáleniá / który jeſt u drzwi námiotu zgromádzeniá.
Wƺyſtkę záś tłuſtość ćielcá tego zá grzech ofiárowánego wyjmie z niego tłuſtość okrywájącą wnętrznośći y wƺyſtkę tłuſtość / którá jeſt ná wnętrznośćiách.
Obie też nerki z tłuſtośćią / którá jeſt ná nich / y ná polędwicách / y odziecżkę / którá jeſt ná wątrobie / y ná nerkách / odejmie.
Jáko odejmują z wołu ofiáry ſpokojnej / y zápáli to kápłán ná ołtárzu cáłopálonych ofiár.
Skórę záś ćielcá tego / y wƺyſtko mięſo jego z głową jego y z nogámi jego y z wnętrznośćiámi jego y z gnojem jego ;
Owá / cáłego ćielcá wynieśie precż zá obóz ná miejſce cżyſte / tám gdzie śię wyſypuje popiół / y ſpáli go ná drwách ogniem ; gdzie wyſypują popiół / tám ſpálony będzie.
Jeźliby też wƺyſtko zgromádzenie Izráelſkie z nieobácżeniá zgrzeƺyło / á byłáby rzecż zákrytá od ocżu zgromádzeniá tego / y ucżyniliby przećiw któremu ze wƺyſtkich przykázáń Páńſkich / coby być nie miáło / á byliby winni /
Y poználiby grzech / którym zgrzeƺyli / ofiárowáć będzie ono zgromádzenie ćielcá młodego ná ofiárę zá grzech / á przywiodą go przed námiot zgromádzeniá ;
Y położą ſtárśi zgromádzeniá ręce ſwe ná głowę ćielcá onego przed oblicżem Páńſkim y zábiją tegoż ćielcá przed oblicżem Páńſkim.
Tedy wnieśie kápłán pomázány / ze krwi ćielcá onego do námiotu zgromádzeniá /
Y omocży kápłán pálec ſwój w onej krwi / á będzie nią kropił śiedm kroć przed oblicżem Páńſkim / przed záſłoną.
A oną krwią pomáże rogi ołtárzá / który jeſt przed oblicżem Páńſkim / w námioćie zgromádzeniá ; á oſtátek krwi wyleje u ſpodku ołtárzá cáłopáleniá / który jeſt u drzwi námiotu zgromádzeniá.
Wƺyſtkę też tłuſtość jego wyjmie z niego / y zápáli ná ołtárzu.
Y ucżyni z tem ćielcem / jáko ucżynił z ćielcem / zá grzech ofiárowánym / ták ucżyni z nim ; á ták ocżyśći je kápłán / y będzie im odpuƺcżono.
Potem wynieśie ćielcá onego precż zá obóz / y ſpáli go / jáko ſpálił ćielcá pierwƺego. Táć jeſt ofiárá zá grzech zgromádzeniá.
Jeźliby kśiążę zgrzeƺył / y ucżynił przećiw któremu ze wƺyſtkich przykázáń Páná Bogá ſwego / coby być nie miáło / á to z nieobácżeniá / á byłby winien :
Y byłby jáwny grzech jego / którym zgrzeƺył / przywiedzie ná ofiárę ſwoję kozłá z kóz / ſámcá zupełnego ;
Y położy rękę ſwoję ná głowę tegoż kozłá / y zábije go ná miejſcu / gdzie biją ofiáry ná cáłopálenie / przed oblicżem Páńſkim. Ofiárá to jeſt zá grzech.
Y weźmie kápłán ze krwi ofiáry zá grzech ná pálec ſwój / á pomáże rogi ołtárzá cáłopálonych ofiár / á oſtátek krwi jego wyleje u ſpodku ołtárzá cáłopáleniá.
Wƺyſtkę záś tłuſtość jego zápáli ná ołtárzu / jáko y tłuſtość ofiáry ſpokojnej ; á ták ocżyśći go kápłán od grzechu jego / y będzie mu odpuƺcżony.
A jeźliby kto zgrzeƺył z ludu poſpolitego z nieobácżeniá / á ucżyniłby przećiw któremu z przykázáń Páńſkich / coby być nie miáło / y byłby winien ;
A byłby znájomy grzech jego / którym zgrzeƺył / przywiedzie ofiárę ſwoję / kozę z kóz / zupełną ſámicę / zá grzech ſwój / którego śię dopuśćił.
A położywƺy rękę ſwą ná głowę tej ofiáry zá grzech / zábije tę ofiárę zá grzech ná miejſcu ofiár cáłopálonych.
Potem wziąwƺy kápłán ze krwi onej ná pálec ſwój / pomáże rogi ołtárzá cáłopálonych ofiár / á oſtátek krwi jej wyleje u ſpodku onegoż ołtárzá.
Wƺyſtkę tákże tłuſtość jej odejmie / jáko śię odejmuje tłuſtość od ofiáry ſpokojnej / y ſpáli to kápłán ná ołtárzu ku wdzięcżnej wonnośći Pánu ; á ták ocżyśći go kápłán / y będzie mu odpuƺcżono.
A jeźliby owcę przywiódł ná ofiárę ſwoję zá grzech / ſámicę zupełną niech przywiedzie.
Y włoży rękę ſwą ná głowę onej ofiáry zá grzech / y zábije ją ná ofiárę zá grzech ná miejſcu / gdzie zábijáją ofiáry cáłopáleniá.
Potem weźmie kápłán onej krwi z ofiáry zá grzech ná pálec ſwój / y pomáże rogi u ołtárzá pálonych ofiár / á oſtátek krwi jej wyleje u ſpodku onego ołtárzá.
Y wƺyſtkę tłuſtość jej odejmie / jáko odejmują tłuſtość báránká z ofiáry ſpokojnej / y ſpáli ją kápłán ná ołtárzu ná cáłopáloną ofiárę Pánu. A ták ocżyśći go kápłán od grzechu jego / którym zgrzeƺył / y będzie mu odpuƺcżony.
