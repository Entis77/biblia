Y rzekł Pán do Mojżeƺá / mówiąc :
Mów do ſynów Izráelſkich / á rzecż : Gdyby kto zgrzeƺył z niewiádomośći / przećiw któremu ze wƺyſtkich przykázáń Páńſkich / cżyniąc / cżegoby cżynić nie miał / á przeſtąpiłby jedno z nich ;
Jeſliby kápłán pomázány zgrzeƺył jáko jeden z ludu poſpolitego / tedy niech ofiáruje zá grzech ſwój / którego śię dopuśćił / ćielcá młodego zupełnego Pánu ná ofiárę zá grzech.
Y przywiedźie ćielcá onego do drzwi namiotu zgromádzenia przed oblicżność Páńſką / á włoży rękę ſwoję ná głowę onego ćielcá / y zábije go przed oblicżem Páńſkim.
Tedy weźmie kápłán pomázány ze krwi onego ćielcá / y wnieśie ją do namiotu zgromádzenia.
Potem omocży kápłán pálec ſwój we krwi / á kropić będźie oną krwią śiedm kroć przed oblicżem Páńſkim przed záſłoną świątnicy.
Y pomáże kápłán krwią oną rogi ołtárzá kádzeniá wonnego / przed oblicżem Páńſkim / który jeſt w námioćie zgromádzenia / á oſtátek krwi onego ćielcá wyleje u ſpodku ołtárzá cáłopalenia / który jeſt u drzwi namiotu zgromádzenia.
Wƺyſtkę záś tłuſtość ćielcá tego zá grzech ofiárowánego wyjmie z niego tłuſtość okrywájącą wnętrznośći y wƺyſtkę tłuſtość / która jeſt ná wnętrznośćiách.
Obie też nerki z tłuſtośćią / która jeſt ná nich / y ná polędwicách / y odźiecżkę / która jeſt ná wątrobie / y ná nerkách / odejmie.
Jáko odejmują z wołu ofiáry ſpokojney / y zápáli to kápłán ná ołtárzu cáłopálonych ofiár.
Skórę záś ćielcá tego / y wƺyſtko mięſo jego z głową jego y z nogámi jego y z wnętrznośćiámi jego y z gnojem jego ;
Owá / cáłego ćielcá wynieśie precż zá obóz ná miejſce cżyſte / tám gdźie śię wyſypuje popiół / y ſpáli go ná drwách ogniem ; gdźie wyſypują popiół / tám ſpálony będźie.
Jeſliby też wƺyſtko zgromádzenie Izráelſkie z nieobácżeniá zgrzeƺyło / á byłáby rzecż zákryta od ocżu zgromádzenia tego / y ucżyniliby przećiw któremu ze wƺyſtkich przykázáń Páńſkich / coby być nie miáło / á byliby winni /
Y poználiby grzech / którym zgrzeƺyli / ofiárowáć będźie ono zgromádzenie ćielcá młodego ná ofiárę zá grzech / á przywiodą go przed námiot zgromádzenia ;
Y położą ſtárśi zgromádzenia ręce ſwe ná głowę ćielcá onego przed oblicżem Páńſkim y zábiją tegoż ćielcá przed oblicżem Páńſkim.
Tedy wnieśie kápłán pomázány / ze krwi ćielcá onego do namiotu zgromádzenia /
Y omocży kápłán pálec ſwój w oney krwi / á będźie nią kropił śiedm kroć przed oblicżem Páńſkim / przed záſłoną.
A oną krwią pomáże rogi ołtárzá / który jeſt przed oblicżem Páńſkim / w námioćie zgromádzenia ; á oſtátek krwi wyleje u ſpodku ołtárzá cáłopalenia / który jeſt u drzwi namiotu zgromádzenia.
Wƺyſtkę też tłuſtość jego wyjmie z niego / y zápáli ná ołtárzu.
Y ucżyni z tym ćielcem / jáko ucżynił z ćielcem / zá grzech ofiárowánym / ták ucżyni z nim ; á ták ocżyśći je kápłán / y będźie im odpuƺcżono.
Potem wynieśie ćielcá onego precż zá obóz / y ſpáli go / jáko ſpálił ćielcá pierwƺego. Táć jeſt ofiárá zá grzech zgromádzenia.
Jeſliby kśiążę zgrzeƺył / y ucżynił przećiw któremu ze wƺyſtkich przykázáń Páná Bogá ſwego / coby być nie miáło / á to z nieobácżeniá / á byłby winien :
Y byłby jáwny grzech jego / którym zgrzeƺył / przywiedźie ná ofiárę ſwoję kozłá z kóz / ſámcá zupełnego ;
Y położy rękę ſwoję ná głowę tegoż kozłá / y zábije go ná miejſcu / gdźie biją ofiáry ná cáłopálenie / przed oblicżem Páńſkim. Ofiárá to jeſt zá grzech.
Y weźmie kápłán ze krwi ofiáry zá grzech ná pálec ſwój / á pomáże rogi ołtárzá cáłopálonych ofiár / á oſtátek krwi jego wyleje u ſpodku ołtárzá cáłopalenia.
Wƺyſtkę záś tłuſtość jego zápáli ná ołtárzu / jáko y tłuſtość ofiáry ſpokojney ; á ták ocżyśći go kápłán od grzechu jego / y będźie mu odpuƺcżony.
A jeſliby kto zgrzeƺył z ludu poſpolitego z nieobácżeniá / á ucżyniłby przećiw któremu z przykázáń Páńſkich / coby być nie miáło / y byłby winien ;
A byłby znájomy grzech jego / którym zgrzeƺył / przywiedźie ofiárę ſwoję / kozę z kóz / zupełną ſámicę / zá grzech ſwój / którego śię dopuśćił.
A położywƺy rękę ſwą ná głowę tey ofiáry zá grzech / zábije tę ofiárę zá grzech ná miejſcu ofiár cáłopálonych.
Potem wźiąwƺy kápłán ze krwi oney ná pálec ſwój / pomáże rogi ołtárzá cáłopálonych ofiár / á oſtátek krwi jey wyleje u ſpodku onegoż ołtárzá.
Wƺyſtkę tákże tłuſtość jey odejmie / jáko śię odejmuje tłuſtość od ofiáry ſpokojney / y ſpáli to kápłán ná ołtárzu ku wdźięcżney wonnośći Pánu ; á ták ocżyśći go kápłán / y będźie mu odpuƺcżono.
A jeſliby owcę przywiódł ná ofiárę ſwoję zá grzech / ſámicę zupełną niech przywiedźie.
Y włoży rękę ſwą ná głowę oney ofiáry zá grzech / y zábije ją ná ofiárę zá grzech ná miejſcu / gdźie zábijáją ofiáry cáłopalenia.
Potem weźmie kápłán oney krwi z ofiáry zá grzech ná pálec ſwój / y pomáże rogi u ołtárzá pálonych ofiár / á oſtátek krwi jey wyleje u ſpodku onego ołtárzá.
Y wƺyſtkę tłuſtość jey odejmie / jáko odejmują tłuſtość báránká z ofiáry ſpokojney / y ſpáli ją kápłán ná ołtárzu ná cáłopáloną ofiárę Pánu. A ták ocżyśći go kápłán od grzechu jego / którym zgrzeƺył / y będźie mu odpuƺcżony.
