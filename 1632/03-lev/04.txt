Y rzekł PAN do Mojzeƺá / mówiąc :
Mów do Synów Izráelſkich / á rzecż. Gdyby kto zgrzeƺył z niewiádomośći / przećiw <i>któremu</i> ze wƺyſtkich przykazań PAńſkich / <i>czyniąc</i> czegoby czynić nie miał / á przeſtąpiłby jedno z nich :
Jesliby kápłán pomázány zgrzeƺył / jáko jeden z ludu poſpolitego / tedy niech ofiáruje zá grzech ſwój / którego śię dopuśćił / ćielcá młodego / zupełnego / PANU ná ofiárę zá grzech.
Y przywiedźie ćielcá onego do drzwi Namiotu zgromádzenia / przed oblicżność PAńſką : á włoży rękę ſwoję ná głowę onego ćielcá / y zábije go / przed oblicżem PAŃſkim.
Tedy weźmie kápłán pomázány / ze krwie onego ćielcá / y wnieśie ją do Namiotu zgromádzenia.
Potym omocży kápłán pálec ſwój we krwi / á kropić będźie oną krwią śiedm kroć / przed oblicżem PAŃſkim / przed zaſłoną świątnice.
Y pomáże kápłán krwią oną / rogi Ołtarzá kádzenia wonnego / przed oblicżem PAŃſkim / który jeſt w Namiećie zgromádzenia : á oſtátek krwie / onego ćielcá / wyleje u ſpodku Ołtarzá cáłopalenia / który jeſt u drzwi Namiotu zgromádzenia.
Wƺyſtkę záś tłuſtość / ćielcá tego zá grzech <i>ofiárowánego,</i> wyjmie z niego : tłuſtość okrywájącą wnętrznośći y wƺyſtkę tłuſtość / która jeſt ná wnętrznośćiách.
Obie też nerki / z tłuſtośćią która <i>jeſt</i> ná nich / y ná polędźwicách / y odźieczkę <i>która jeſt</i> ná wątrobie / y ná nerkách / odejmie /
Jáko odejmują z wołu ofiáry ſpokojney : y zápali to kápłán ná Ołtarzu cáłopalonych ofiar.
Skórę záś ćielcá tego / y wƺyſtko mięſo jego / z głową jego / y z nogámi jego / y z wnętrznośćiámi jego / y z gnojem jego /
Owa / cáłego ćielcá wynieśie precz zá Obóz ná miejſce czyſte / tám gdźie śię wyſypuje popiół : y ſpali go ná drwách ogniem : gdźie wyſypują popiół / tám ſpalony będźie.
Jesliby też wƺyſtko zgromádzenie Izráelſkie z nieobaczenia zgrzeƺyło / á byłáby rzecz zákryta od ocżu zgromádzenia tego / y uczyniliby przećiw któremu ze wƺyſtkich przykazań PAńſkich / coby być nie miáło / á byliby winni :
Y poználiby grzech którym zgrzeƺyli : ofiárowáć będźie / ono zgromádzenie ćielcá młodego / ná ofiárę zá grzech / á przywiodą go przed Namiot zgromádzenia.
Y położą ſtárƺy zgromádzenia ręce ſwe ná głowę ćielcá onego / przed oblicżem PAŃſkim / y zábiję tegoż ćielcá przed oblicżem PAŃſkim.
Tedy wnieśie kápłán pomázány / ze krwie ćielcá onego / do Namiotu zgromádzenia :
Y omoczy kápłán pálec ſwój w oney krwi / á będźie nią kropił śiedm kroć przed oblicżem PAŃſkim / przed zaſłoną :
A oną krwią pomáże rogi Ołtarzá / który jeſt przed oblicżem PAńſkim / w Namiećie zgromádzenia : á oſtátek krwie wyleje / u ſpodku Ołtarzá cáłopalenia / który jeſt u drzwi Namiotu zgromádzenia :
Wƺyſtkę też tłuſtość jego wyjmie z niego / y zápali ná Ołtarzu :
Y uczyni z tym ćielcem / jáko uczynił z ćielcem zá grzech <i>ofiárowánym</i> ták uczyni z nim : á ták ocżyśći je kápłán / y będźie im odpuƺczono.
Potym wynieśie ćielcá onego precz zá obóz / y ſpali go / jáko ſpalił ćielcá pierwƺego : táć jeſt ofiárá zá grzech zgromádzenia.
Jesliby kśiążę zgrzeƺył / y ucżynił przećiw któremu ze wƺyſtkich przykazań PANA Bogá ſwego / coby być nie miáło / á to z nieobacżenia / á byłby winien /
Y byłby jáwny grzech jego / którym zgrzeƺył : przywiedźie ná ofiárę ſwoję kozłá z kóz / ſámcá zupełnego.
Y położy rękę ſwoję ná głowę tegoż kozłá / y zábije go / ná miejſcu gdźie biją ofiáry ná cáłopalenie / przed oblicżem PAŃſkim : Ofiárá to <i>jeſt</i> zá grzech.
Y weźmie kápłán ze krwie ofiáry zá grzech ná pálec ſwój : á pomáże rogi Ołtarzá cáłopalonych ofiár / á <i>oſtátek</i> krwie jego wyleje u ſpodku Ołtarzá cáłopalenia.
Wƺyſtkę záś tłuſtość jego zápali ná Ołtarzu / jáko y tłuſtość Ofiáry ſpokojney. A ták ocżyśći go kápłán od grzechu jego / y będźie mu odpuƺczony.
A jesliby kto zgrzeƺył z ludu poſpolitego / z nieobacżenia / á uczyniłby przećiw któremu z przykazań PAńſkich coby być nie miáło / y byłby winien /
A byłby znájomy grzech jego / którym zgrzeƺył : przywiedźie ofiárę ſwoję kozę z kóz / zupełną ſámicę / zá grzech ſwój / którego śię dopuśćił :
A położywƺy rękę ſwą ná głowę tey Ofiáry zá grzech / zábije tę ofiárę zá grzech ná miejſcu ofiár cáłopalonych.
Potym wźiąwƺy kápłán ze krwie oney ná pálec ſwój / pomáże rogi Ołtarzá cáłopalonych ofiár / á oſtátek krwie jey wyleje u ſpodku onegoƺ Ołtarzá.
Wƺyſtkę tákże tłuſtość jey odejmie / jáko śię odejmuje tłuſtość od Ofiáry ſpokojney : y ſpali to kápłán ná Ołtarzu / ku wdźięczney wonnośći PAnu / á ták oczyśći go kápłán / y będźie mu odpuƺcżono.
A jesliby owcę przywiódł ná ofiárę ſwoję zá grzech / ſámicę zupełną niech przywiedźie :
Y włoży rękę ſwą ná głowę oney ofiáry zá grzech / y zábije ją ná Ofiárę zá grzech / ná miejſcu gdźie zábijáją Ofiáry cáłopalenia :
Potym weźmie kápłán oney krwie z Ofiáry zá grzech / ná pálec ſwój / y pomáże rogi u Ołtarzá palonych Ofiar / á oſtátek krwie jey wyleje u ſpodku onego Ołtarzá :
Y wƺyſtkę tłuſtość jey odejmie / jáko odejmują tłuſtość báránká z ofiáry ſpokojney / y ſpali ją kápłán ná Ołtarzu ná cáłopaloną Ofiárę PAnu : á ták oczyśći go kápłán od grzechu jego / którym zgrzeƺył / y będźie mu odpuƺcżony.
