To gdy uſłyƺáł Jábin / król Háſor / poſłáł do Johábá / królá Mádon / y do królá Symron / y do królá Achſáf /
Y do królów / którzy byli ná północy / ná górách y ná polách / ná południe Cynerot / y w równinách / y w kráinách Dor / ku záchodowi ;
Do Chánánejcżyká ná wſchód y ná záchód ſłońcá / y do Amorejcżyká / y do Hetejcżyká / y do Ferezejcżyká / y do Jebuzejcżyká po górách / y do Hewejcżyká pod górą Hermon / w ziemi Máſfá.
Y wyćiągnęli ſámi / y wƺyſtkie wojſká ich z nimi / lud wielki / jáko piáſek / który jeſt ná brzegu morſkim / y koni y wozów bárdzo wiele.
A zgromádziwƺy śię wƺyſcy oni królowie przyƺli / y położyli śię poſpołu obozem u wód Merom / áby zwiedli bitwę z Izráelem.
Y rzekł Pán do Jozuego : Nie bój śię ich / ábowiem jutro o tym cżáśie Já podám te wƺyſtkie pobite przed Izráelem ; koniom ich żyły poderznieƺ / á wozy ich ogniem ſpáliƺ.
Wyćiągnął tedy Jozue / y wƺyſtek lud wálecżny z nim / przećiwko nim ku wodom Merom z nágłá / y uderzyli ná nie.
Y podáł je Pán w rękę Izráelowi / y porázili je / á gonili je áż do Sydonu wielkiego / y áż do wód gorących / y áż do polá Máſfá ná wſchód ſłońcá / y pobili je / ták iż jednego z nich nie zoſtáwili żywego.
Y ucżynił im Jozue / jáko mu był rozkázáł Pán ; koniom ich żyły poderznął / á wozy ich popálił ogniem.
Potem wróćiwƺy śię Jozue tego cżáſu wziął Háſor / á królá jego zábił miecżem ; á Háſor było przedtem głową wƺyſtkich tych króleſtw.
Zábili też káżdą duƺę / którá byłá w nim / oſtrzem miecżá mordując / ták iż nie zoſtáło nic żywego ; á Háſor ſpálił ogniem.
Tákże ucżynił wƺyſtkim miáſtom królów onych / y wƺyſtkie króle ich pojmáł Jozue / y pobił je oſtrzem miecżá / mordując je / jáko był rozkázáł Mojżeƺ / ſługá Páńſki.
Tylko tych wƺyſtkich miáſt / które były obronne / nie pálił Izráel / oprócż ſámego Háſorá / które ſpálił Jozue.
Wƺyſtkie też łupy miáſt onych / y bydłá / pobráli ſobie ſynowie Izráelſcy / tylko wƺyſtkie ludzie zábijáli oſtrzem miecżá / áż je wytráćili / nie zoſtáwując nikogo żywego.
Jáko był rozkázáł Pán Mojżeƺowi / ſłudze ſwemu / ták rozkázáł Mojżeƺ Jozuemu ; ták też ucżynił Jozue / nie opuśćił nicżego ze wƺyſtkiego / co był rozkázáł Pán Mojżeƺowi.
A ták wziął Jozue wƺyſtkę onę ziemię górną / y wƺyſtkę ná południe leżącą / y wƺyſtkę ziemię Goſen / y równiny / y polá / y górę Izráel z równiną jej ;
Od góry Hálák / którá idzie ku Seir / áż do Báálgád / w równinie Libáńſkiej pod górą Hermon ; y wƺyſtkie króle ich pojmáł / y poráził je / y pozábijáł je.
Przez wiele dni prowádził Jozue z onymi wƺyſtkimi królmi wojnę.
A nie było miáſtá / któreby pokój ucżyniło z ſyny Izráelſkimi / oprócż Hewejcżyków / którzy mieƺkáli w Gábáon ; wƺyſtkie inƺe wzięli przez wojnę.
Abowiem od Páná śię to ſtáło / że zátwárdził ſercá ich / áby ƺli ku bitwie przećiw Izráelowi / żeby je wyniƺcżył / nie májąc nád nimi miłośierdziá / ále żeby je wytráćił / jáko był rozkázáł Pán Mojżeƺowi.
Y ćiągnął Jozue onegoż cżáſu / y wybił ſyny Enákowe z gór / z Hebronu / z Dábiru / z Anáb / y ze wƺyſtkich gór Judzkich / y ze wƺyſtkich gór Izráelſkich / poſpołu z miáſty ich wykorzenił je Jozue.
Nie zoſtáł nikt z Enákitów / w ziemi ſynów Izráelſkich ; tylko w Gázie / w Gád / y w Azdod zoſtáli.
Wziął tedy Jozue wƺyſtkę onę ziemię / ták jáko mówił Pán do Mojżeƺá : y podáł ją Jozue w dziedzictwo Izráelowi według dziáłów ich / y według pokoleniá ich / á uſpokoiłá śię ziemiá od wojen.
