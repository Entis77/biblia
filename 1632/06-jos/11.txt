To gdy uſłyƺał Jábin / król Háſor / poſłał do Johábá / królá Mádon / y do królá Symron / y do królá Achſáf /
Y do królów / którzy byli ná północy / ná górách y ná polách / ná południe Cynerot / y w równinách / y w kráinách Dor / ku záchodowi ;
Do Chánánejcżyká ná wſchód y ná záchód ſłońcá / y do Amorejcżyká / y do Hetejcżyká / y do Ferezejcżyká / y do Jebuzejcżyká po górách / y do Hewejcżyká pod górą Hermon / w źiemi Máſfá.
Y wyćiągnęli ſámi / y wƺyſtkie wojſká ich z nimi / lud wielki / jáko piáſek / który jeſt ná brzegu morſkim / y koni y wozów bárzo wiele.
A zgromádźiwƺy śię wƺyſcy oni królowie przyƺli / y położyli śię poſpołu obozem u wód Merom / áby zwiedli bitwę z Izráelem.
Y rzekł Pán do Jozuego : Nie bój śię ich / ábowiem jutro o tym cżáśie Ja podám te wƺyſtkie pobite przed Izráelem ; koniom ich żyły poderznieƺ / á wozy ich ogniem ſpáliƺ.
Wyćiągnął tedy Jozue / y wƺyſtek lud wálecżny z nim / przećiwko nim ku wodom Merom z nágłá / y uderzyli ná nie.
Y podał je Pán w rękę Izráelowi / y poráźili je / á gonili je áż do Sydonu wielkiego / y áż do wód gorących / y áż do polá Máſfá ná wſchód ſłońcá / y pobili je / ták iż jednego z nich nie zoſtáwili żywego.
Y ucżynił im Jozue / jáko mu był rozkázał Pán ; koniom ich żyły poderznął / á wozy ich popálił ogniem.
Potem wróćiwƺy śię Jozue tego cżáſu wźiął Háſor / á królá jego zábił miecżem ; á Háſor było przedtem głową wƺyſtkich tych króleſtw.
Zábili też káżdą duƺę / która byłá w nim / oſtrzem miecżá mordując / ták iż nie zoſtáło nic żywego ; á Háſor ſpálił ogniem.
Tákże ucżynił wƺyſtkim miáſtom królów onych / y wƺyſtkie króle ich pojmał Jozue / y pobił je oſtrzem miecżá / mordując je / jáko był rozkázał Mojzeƺ / ſługá Páńſki.
Tylko tych wƺyſtkich miaſt / które były obronne / nie pálił Izráel / oprócż ſámego Háſorá / które ſpálił Jozue.
Wƺyſtkie też łupy miaſt onych / y bydłá / pobráli ſobie ſynowie Izráelſcy / tylko wƺyſtkie ludźie zábijáli oſtrzem miecżá / áż je wytráćili / nie zoſtáwując nikogo żywego.
Jáko był rozkázał Pán Mojżeƺowi / ſłudze ſwemu / ták rozkázał Mojzeƺ Jozuemu ; ták też ucżynił Jozue / nie opuśćił nicżego ze wƺyſtkiego / co był rozkázał Pán Mojżeƺowi.
A ták wźiął Jozue wƺyſtkę onę źiemię górną / y wƺyſtkę ná południe leżącą / y wƺyſtkę źiemię Goſen / y równiny / y polá / y górę Izráel z równiną jey ;
Od góry Hálák / która idźie ku Seir / áż do Báálgád / w równinie Libáńſkiej pod górą Hermon ; y wƺyſtkie króle ich pojmał / y poráźił je / y pozábijał je.
Przez wiele dni prowádźił Jozue z onymi wƺyſtkimi królmi wojnę.
A nie było miáſtá / któreby pokój ucżyniło z ſyny Izráelſkimi / oprócż Hewejcżyków / którzy mieƺkáli w Gábáon ; wƺyſtkie inƺe wźięli przez wojnę.
Abowiem od Páná śię to ſtáło / że zátwárdźił ſercá ich / áby ƺli ku bitwie przećiw Izráelowi / żeby je wyniƺcżył / nie májąc nád nimi miłośierdźiá / ále żeby je wytráćił / jáko był rozkázał Pán Mojżeƺowi.
Y ćiągnął Jozue onegoż cżáſu / y wybił ſyny Enákowe z gór / z Hebronu / z Dábiru / z Anáb / y ze wƺyſtkich gór Judzkich / y ze wƺyſtkich gór Izráelſkich / poſpołu z miáſty ich wykorzenił je Jozue.
Nie zoſtał nikt z Enákitów / w źiemi ſynów Izráelſkich ; tylko w Gáźie / w Gád / y w Azdod zoſtáli.
Wźiął tedy Jozue wƺyſtkę onę źiemię / ták jáko mówił Pán do Mojżeƺá : y podał ją Jozue w dźiedźictwo Izráelowi według dźiáłów ich / y według pokolenia ich / á uſpokoiłá śię źiemiá od wojen.
