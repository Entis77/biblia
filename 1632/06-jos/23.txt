Y ſtáło śię po niemáłym cżáśie / gdy odpocżynek dał Pán Izráelowi od wƺyſtkich nieprzyjáćiół ich okolicżnie / á Jozue śię zſtárzał / y był zeƺłym w lećiech /
Że przyzwał Jozue wƺyſtkiego Izráelá / ſtárƺych jego / y przedniejƺych jego / y ſędźiów jego / y przełożonych jego / y rzekł do nich. Jam śię zſtárzał á zƺedłem w lećiech.
A wyśćie widźieli wƺyſtko / co ucżynił Pán / Bóg wáƺ / wƺyſtkim tym národom przed oblicżem wáƺem : bo Pán / Bóg wáƺ ſám wálcżył zá was.
Obácżćież / rozdźieliłem wam loſem te narody pozoſtáłe w dźiedźictwo miedzy pokolenia wáƺe od Jordánu / y wƺyſtkie narody / którem wytráćił / áż do morzá wielkiego ná záchód ſłońcá.
A Pán / Bóg wáƺ / ſám je wypędźi od twárzy wáƺej / y wyżenie je od oblicżnośći wáƺej / y pośiędźiećie dźiedźicżnie źiemię ich / jáko wam to powiedźiał Pán / Bóg wáƺ.
Zmácniájćież śię bárzo / ábyśćie ſtrzegli á cżynili wƺyſtko / co nápiſano w kśięgách Zakonu Mojżeƺowego / nie odſtępując od niego ná práwo áni ná lewo.
Ani śię też mięƺájćie z temi národámi / które zoſtáły z wámi ; áni imienia bogów ich nie wſpominájćie / áni przyśięgájćie przez nie / áni im ſłużćie / áni śię im kłániájćie ;
Ale śię Páná / Bogá wáƺego / trzymájćie / jákośćie cżynili áż do dniá tego.
Bo jáko wypędźił Pán od oblicża wáƺego narody wielkie y możne / y nie opárł śię wam nikt áż do dniá tego :
Ták mąż jeden z was będźie ugániał tyśiąc ; ábowiem Pán / Bóg wáƺ / on wálcży zá wámi / jáko wam obiecał.
Przetoż przeſtrzegájćie z pilnośćią / ábyśćie miłowáli Páná / Bogá wáƺego.
Bo jeſli śię cále odwróćićie / á przyſtániećie do tych pozoſtáłych narodów / do tych / które zoſtáwáją miedzy wámi / y ſpowinowáćićie śię z nimi / á będźiećie śię mięƺáć z nimi / one też z wámi :
Wiedzćież wiedząc / żeć nie będźie więcey Pán / Bóg wáƺ / wygániał tych narodów od twárzy wáƺej ; ále będą wam śidłem / y záwádą / y bicżem ná boki wáƺe / y ćierniem ná ocży wáƺe / póki nie wyginiećie z tey przewyborney źiemi / którą wam dał Pán / Bóg wáƺ.
A oto / ja idę dźiś w drogę wƺyſtkiej źiemi ; poznájćież tedy ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej / żeć nie chybiło żádne ſłowo ze wƺyſtkich ſłów nájlepƺych / które mówił Pán / Bóg wáƺ / o was ; wƺyſtkie śię nád wámi wypełniły / á nie chybiło z nich żádne ſłowo.
Przetoż jáko śię wypełniło nád wámi káżde ſłowo dobre / które mówił Pán / Bóg wáƺ / do was / ták przywiedźie Pán ná was káżde ſłowo złe / áż was wytráći z źiemi tey przewyborney / którą wam dał Pán / Bóg wáƺ.
Jeſli przeſtąpićie przymierze Páná / Bogá wáƺego / które wam rozkázał / á ƺedƺy ſłużyć będźiećie bogom obcym / y kłániáć śię im będźiećie / tedy śię rozpáli popędliwość Páńſka przećiwko wam / y zginiećie prędko z tey przewyborney źiemi / którą wam dał.
