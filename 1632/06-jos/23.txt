Y ſtáło śię po niemáłym cżáśie / gdy odpocżynek dáł Pán Izráelowi od wƺyſtkich nieprzyjáćiół ich okolicżnie / á Jozue śię zſtárzáł / y był zeƺłym w lećiech /
Że przyzwáł Jozue wƺyſtkiego Izráelá / ſtárƺych jego / y przedniejƺych jego / y ſędziów jego / y przełożonych jego / y rzekł do nich. Jám śię zſtárzáł á zƺedłem w lećiech.
A wyśćie widzieli wƺyſtko / co ucżynił Pán / Bóg wáƺ / wƺyſtkim tym národom przed oblicżem wáƺem : bo Pán / Bóg wáƺ ſám wálcżył zá wáſ.
Obácżćież / rozdzieliłem wám loſem te národy pozoſtáłe w dziedzictwo między pokoleniá wáƺe od Jordánu / y wƺyſtkie národy / którem wytráćił / áż do morzá wielkiego ná záchód ſłońcá.
A Pán / Bóg wáƺ / ſám je wypędzi od twárzy wáƺej / y wyżenie je od oblicżnośći wáƺej / y pośiędziećie dziedzicżnie ziemię ich / jáko wám to powiedziáł Pán / Bóg wáƺ.
Zmácniájćież śię bárdzo / ábyśćie ſtrzegli á cżynili wƺyſtko / co nápiſáno w kśięgách Zákonu Mojżeƺowego / nie odſtępując od niego ná práwo áni ná lewo.
Ani śię też mięƺájćie z temi národámi / które zoſtáły z wámi ; áni imieniá bogów ich nie wſpominájćie / áni przyśięgájćie przez nie / áni im ſłużćie / áni śię im kłániájćie ;
Ale śię Páná / Bogá wáƺego / trzymájćie / jákośćie cżynili áż do dniá tego.
Bo jáko wypędził Pán od oblicżá wáƺego národy wielkie y możne / y nie opárł śię wám nikt áż do dniá tego :
Ták mąż jeden z wáſ będzie ugániáł tyśiąc ; ábowiem Pán / Bóg wáƺ / on wálcży zá wámi / jáko wám obiecáł.
Przetoż przeſtrzegájćie z pilnośćią / ábyśćie miłowáli Páná / Bogá wáƺego.
Bo jeźli śię cále odwróćićie / á przyſtániećie do tych pozoſtáłych národów / do tych / które zoſtáwáją między wámi / y ſpowinowáćićie śię z nimi / á będziećie śię mięƺáć z nimi / one też z wámi :
Wiedźćież wiedząc / żeć nie będzie więcej Pán / Bóg wáƺ / wygániáł tych národów od twárzy wáƺej ; ále będą wám śidłem / y záwádą / y bicżem ná boki wáƺe / y ćierniem ná ocży wáƺe / póki nie wyginiećie z tej przewybornej ziemi / którą wám dáł Pán / Bóg wáƺ.
A oto / já idę dziś w drogę wƺyſtkiej ziemi ; poznájćież tedy ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej / żeć nie chybiło żádne ſłowo ze wƺyſtkich ſłów nájlepƺych / które mówił Pán / Bóg wáƺ / o wáſ ; wƺyſtkie śię nád wámi wypełniły / á nie chybiło z nich żádne ſłowo.
Przetoż jáko śię wypełniło nád wámi káżde ſłowo dobre / które mówił Pán / Bóg wáƺ / do wáſ / ták przywiedzie Pán ná wáſ káżde ſłowo złe / áż wáſ wytráći z ziemi tej przewybornej / którą wám dáł Pán / Bóg wáƺ.
Jeźli przeſtąpićie przymierze Páná / Bogá wáƺego / które wám rozkázáł / á ƺedłƺy ſłużyć będziećie bogom obcym / y kłániáć śię im będziećie / tedy śię rozpáli popędliwość Páńſká przećiwko wám / y zginiećie prędko z tej przewybornej ziemi / którą wám dáł.
