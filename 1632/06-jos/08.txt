Potem rzekł Pán do Jozuego : Nie bój śię / áni śię lękáj ; weźmij z ſobą wƺyſtek lud wojenny / á wſtáwƺy ćiągnij do Háj / otom dał w ręce twoje królá Háj / y lud jego / y źiemię jego.
A ucżyniƺ Hájowi y królowi jego / jákoś ucżynił Jerychu y królowi jego / wƺákże łupy jego / y bydłá jego rozbierzećie miedzy śię ; ucżyńże záſádzkę ná miáſto z tyłu jego.
A ták wſtał Jozue y wƺyſtek lud wálecżny / áby ćiągnęli ku Háj ; y przebrał Jozue trzydźieśći tyśięcy mężów bárzo mocnych / y poſłał je nocą.
Y rozkázał im / mówiąc : Pátrzájćie wy / ábyśćie ucżynili záſádzkę zá miáſtem ; nie oddálájćie śię od miáſtá dáleko bárzo / á bądźćie wƺyſcy pogotowiu.
A ja / y wƺyſtek lud / który ze mną jeſt / przyćiągniemy pod miáſto ; á gdy oni wynijdą przećiwko nam / jáko pierwej ućiecżemy przed nimi.
A oni pójdą zá námi / áż je uwiedźiemy od miáſtá ; bo rzeką : Ućiekáją przed námi / jáko y pierwej / gdyż ućiekáć będźiemy przed nimi.
Tedy wy wſtániećie z záſádzki / y wyprzećie oſtátek ludu z miáſtá / y da je Pán / Bóg wáƺ / w rękę wáƺę.
A wźiąwƺy miáſto / zápálićie je ogniem / według ſłowá Páńſkiego ucżynićie ; pátrzájćież / rozkazałem wam.
Poſłał je tedy Jozue / y ƺli ná záſádzkę ; á zoſtáli miedzy Betel / y miedzy Háj ná záchód Hájowi ; á Jozue przez onę noc zoſtał w pośrzodku ludu.
Potem wſtáwƺy Jozue bárzo ráno / oblicżył lud / á ƺedł ſám y ſtárśi z Izráelá przed ludem przećiw Háj.
Wƺyſtek też lud wojenny / który z nim był / ruƺyli śię / y przyćiągnąwƺy przyƺli pod miáſto / y położyli śię obozem ná ſtronie północney ku Háj ; á byłá doliná miedzy nim / y miedzy Háj.
Nádto wźiął około pięćiu tyśięcy mężów / które poſtáwił ná záſádzce miedzy Betel / y miedzy Háj / od ſtrony záchodniej miáſtá.
Y przybliżył śię lud / to jeſt / wƺyſtko wojſko / które było od północy miáſtá / y którzy byli ná záſádzce jego od záchodu miáſtá ; y przyćiągnął Jozue oney nocy w pośrodek doliny.
Y ſtáło śię / gdy je ujrzał król Háj / poſpieƺyli śię y wſtáli ráno / y wyƺli ludźie z miáſtá przećiw Izráelowi ku bitwie / ſám król / y wƺyſtek lud jego / ná cżás náznácżony przed równinę / nie wiedząc / że záſádzká byłá ucżynioná nań zá miáſtem.
Tedy Jozue y wƺyſtek Izráel / jákoby od nich poráżeni / ućiekáli drogą ku puƺcży.
Y zwołány jeſt wƺyſtek lud / który był w mieśćie / áby je gonili / y gonili Jozuego ; y ták uwiedźieni byli od miáſtá.
Y nie zoſtał nikt w Háj y w Betel / któryby nie wyƺedł zá Izráelem ; y zoſtáwili miáſto otworzone / á gonili Izráelá.
Tedy rzekł Pán do Jozuego : Podnieś chorągiew / którą maƺ w ręce ſwej / przećiwko Háj ; bo je w ręce twoje dam. Y podnióſł Jozue chorągiew / którą miał w ręce ſwej / przećiwko miáſtu.
A oni / co byli ná záſádzce / wſtáwƺy prędko z miejſcá ſwego ; bieżáli / gdy on podnióſł rękę ſwą / á ubieżáwƺy miáſto / wźięli je / y záráz je zápálili ogniem.
A obejrzáwƺy śię mężowie miáſtá Háj ujrzeli / á oto / wſtępował dym miáſtá ku niebu / y nie mieli miejſcá do ućiekánia / áni tám / áni ſám ; bo lud / który ućiekał ku puƺcży / obróćił śię ná one / co je gonili.
Tedy Jozue y wƺyſtek lud Izráelſki / widząc / iż oni / co byli ná záſádzce / wźięli miáſto / á iż wychodźił dym z miáſtá / obróćili śię y pobili męże miáſtá Háj.
Oni też drudzy wyƺli z miáſtá przećiwko nim / y obtocżyli je Izráelcżycy / jedni ſtąd / á drudzy zowąd / y poráźili je ták / iż z nich żáden nie zoſtał / áni uƺedł.
Támże królá Háj pojmáli żywo / y przywiedli go przed Jozuego.
Gdy tedy Izráelcżycy pobili wƺyſtkie obywátele Háj ná polu przy puƺcży / tedy zá nimi ƺli w pogoń / á polegli oni wƺyſcy od miecżá / áż wygłádzeni ſą ; obróćili śię wƺyſcy Izráelcżycy do Háj / y wyśiekli je oſtrzem miecżá.
Y było wƺyſtkich / którzy polegli dniá onego / od mężá áż do niewiáſty dwánaśćie tyśięcy / wƺyſtkich obywáteli Háj.
A Jozue nie ſpuśćił ręki ſwej / którą był podnióſł z chorągwią / áż pobił wƺyſtkie obywátele Háj.
Tylko bydło / y łupy miáſtá onego rozebráli miedzy śię Izráelcżycy według ſłowá Páńſkiego / które rozkázał Jozuemu.
Tedy zápálił Jozue Háj / y ucżynił je mogiłą wiecżną / y puſtynią áż do dniá tego.
A królá Háj obwieśił ná drzewie áż do wiecżorá ; á gdy ſłońce záƺło / rozkázał Jozue / áby zdjęto trupá jego z drzewá / á porzucono go w ſámem wejśćiu bramy miejſkiej / y námiotáli nań kupę wielką kámieni / która jeſt áż do dniá tego.
Tedy Jozue zbudował ołtárz Pánu / Bogu Izráelſkiemu / ná górze Hebál.
Jáko był rozkázał Mojzeƺ / ſługá Páńſki / ſynom Izráelſkim / á jáko nápiſano w kśięgách zakonu Mojżeƺowego / ołtárz z cáłego kámieniá / ná którym żádne żelázo nie poſtáło ; y ſpráwowáli ná nim cáłopalenia Pánu / ofiárowáli też ſpokojne ofiáry.
Támże nápiſał ná kámieniách powtórzenie zakonu Mojżeƺowego / który nápiſał przed oblicżnośćią ſynów Izráelſkich.
A wƺyſtek Izráel / y ſtárśi jego / y przełożeni / y ſędźiowie jego / ſtáli po obu ſtronách ſkrzyni przed kápłánámi Lewitámi / którzy nośili ſkrzynię przymierza Páńſkiego / ták przychodźień / jáko w domu zrodzony / połowá ich przećiw górze Gáryzym / á połowá ich przećiw górze Hebál / jáko był przed tym rozkázał Mojzeƺ / ſługá Páńſki / áby błogoſłáwiono ludowi Izráelſkiemu.
A potem cżytał wƺyſtkie ſłowá zakonu / błogoſłáwieńſtwo / y przeklęſtwo / według wƺyſtkiego / co nápiſano w kśięgách zakonu.
Nie było y ſłowá ze wƺyſtkiego / co rozkázał Mojzeƺ / cżegoby nie cżytał Jozue przed wƺyſtkim zgromádzeniem Izráelſkiem / przed niewiáſtámi / y przed dźiátkámi / y przed przychodniámi / którzy mieƺkáli miedzy nimi.
