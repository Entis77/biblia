A gdy uſłyƺał Adoniſedek / król Jerozolimſki / iż wźiął Jozue Háj / y zburzył je / ( bo jáko ucżynił Jerychowi y królowi jego / ták ucżynił Hájowi y królowi jego / ) á iż ucżynili pokój obywátele Gábáon z Izráelem / y mieƺkáją w pośrzodku ich ;
Tedy śię uląkł bárzo / przeto że miáſto wielkie było Gábáon / jáko jedno z miaſt królewſkich / á że było więkƺe niż Háj / á wƺyſcy mężowie jego wálecżni.
Przetoż poſłał Adoniſedek / król Jerozolimſki / do Hohámá / królá Hebron / y do Fárán / królá Jerymotá / y do Jáfijá / królá Láchys / y do Dábir / królá Eglon / mówiąc :
Przyjedźćie do mnie / á dájćie mi pomoc / ábyſmy pobili Gábáonity / którzy ucżynili pokój z Jozuem / y z ſyny Izráelſkimi.
Zebráło śię tedy / á wyćiągnęło pięć królów Amorejſkich / król Jerozolimſki / król Hebron / król Jerymot / król Láchys / król Eglon / ſámi / y wƺyſtkie wojſká ich / y położyli śię obozem u Gábáon / y dobywáli go.
Tedy poſłáli obywátele Gábáon do Jozuego / y do obozu w Gálgál / mówiąc : Nie záwśćiągáj ręki ſwej od ſług twojich ; przyćiągnij do nich rychło / á wybáw nas y pomóż nam ; boć śię zebráli przećiwko nam wƺyſcy królowie Amorejſcy / którzy mieƺkáją po górách.
Ruƺył śię tedy Jozue z Gálgál / ſám y wƺyſtek lud wojenny z nim / y wƺyſcy mężowie wálecżni.
( Bo był rzekł Pán do Jozuego : Nie bój śię ich ; ábowiem w ręce twoje podáłem je / á nie oſtoi śię żáden z nich przed tobą. )
Y przypádł ná nie Jozue nágle ; bo cáłą noc ćiągnął z Gálgál.
Y potrwożył je Pán przed oblicżem Izráelá / który je poráźił poráżką wielką w Gábáon / y gonił je drogą / którą chodzą ku Betoron / á bił je áż do Aſeká y áż do Mácedá.
Y ſtáło śię / gdy ućiekáli przed Izráelem / bieżąc z góry do Betoron / że Pán ſpuśćił ná nie kámienie wielkie z niebá áż do Aſeká / y umieráli ; więcey ich pomárło od kámieniá grádowego / niż ich pobili ſynowie Izráelſcy miecżem.
Tedy mówił Jozue do Páná / dniá / którego podał Pán Amorejcżyká w ręce ſynom Izráelſkim / y rzekł przed ocżymá Izráelá : Słońce w Gábáon záſtánów śię / á mieśiącu w dolinie Ajálon!
Y záſtánowiło śię ſłońce / á mieśiąc ſtánął / áż śię lud pomśćił nád nieprzyjáćioły ſwymi. Izali to nie jeſt nápiſano w kśięgách ſpráwiedliwego? Tedy ſtánęło ſłońce w pośród niebá / á nie poſpieƺyło śię záchodźić / jákoby przez cáły dźień.
Y nie był tákowy dźień przed tym / áni potem / w któryby uſłucháć miał Pán głoſu cżłowiecżego / bo Pán wálcżył zá Izráelem.
Potem śię wróćił Jozue / y wƺyſtek Izráel z nim / do obozu do Gálgál.
A ućiekło było onych pięć królów / y ſkryli śię w jáſkinią przy Mácedá.
Y dáno znáć Jozuemu / mówiąc : Ználeźiono pięć królów / którzy śię pokryli w jáſkini w Mácedá.
Y rzekł Jozue : Przywálćie kámienie wielkie do dźiury jáſkini / á poſtáwćie u niej męże / áby ich ſtrzegli.
A wy nie ſtójćie / gońćie nieprzyjáćioły wáƺe / á bijćie oſtátek ich / áni im dájćie uchodźić do miaſt ich ; boć je podał Pán / Bóg wáƺ / w rękę wáƺę.
A gdy przeſtał Jozue z ſyny Izráelſkimi bić ich poráżką bárzo wielką / áż je do ƺcżętu wytráćili / á którzy żywo zoſtáli z nich / uƺli do miaſt obronnych ;
Tedy wróćił śię wƺyſtek lud zdrowo do obozu / do Jozuego w Mácedá / á nie ruƺył przećiwko ſynom Izráelſkim nikt językiem ſwoim.
Potem rzekł Jozue : Otwórzćie tę dźiurę jáſkini / á wywiedźćie do mnie tych pięćiu królów z jáſkini.
Y ucżynili ták / y wywiedli do niego pięćiu królów onych z jáſkini / królá Jerozolimſkiego / królá Hebron / królá Jerymot / królá Láchys / królá Eglon.
A gdy wywiedli one króle do Jozuego / tedy przyzwał Jozue wƺyſtkich mężów Izráelſkich / y rzekł do rotmiſtrzów / mężów wálecżnych / którzy z nim chodźili : Przyſtąpćie ſám / á náſtąpćie nogámi wáƺemi ná ƺyje tych królów ; którzy przyſtąpiwƺy náſtąpili nogámi ſwemi ná ƺyje ich.
Zátem rzekł do nich Jozue : Nie bójćie śię / áni śię lękájćie ; zmácniájćie śię / y mężnie ſobie pocżynájćie ; boć ták ucżyni Pán wƺyſtkim nieprzyjáćiołom wáƺym / przećiw którym wálcżyćie.
Potem pobił je Jozue / y pomordował je / y záwieśił je ná pięćiu drzewách / á wiśieli ná drzewách áż do wiecżorá.
A gdy záƺło ſłońce / rozkázał Jozue / że je złożono z drzewá / y wrzucono je do jáſkini / w którey śię byli ſkryli / á záwálono kámieńmi wielkiemi dźiurę u jáſkini / które tám ſą jeƺcże y do dniá tego.
Tegoż dniá wźiął Jozue Mácedá / y wyśiekł je oſtrzem miecżá / y królá ich zámordował weſpół z nimi / y wƺelką duƺę / która byłá w nim ; nie zoſtáwił żádnego żywo / y ucżynił królowi Mácedá / jáko ucżynił królowi Jerychá.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Mácedá do Lebny / y dobywał Lebny.
A podał Pán y ono w ręce Izráelá / y królá jego / y wyśiekł je oſtrzem miecżá / y wƺelką duƺę / która byłá w nim ; nie zoſtáwił w nim żádnego żywo / y ucżynił królowi jego / jáko ucżynił królowi Jerychá.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Lebny do Láchys / á położywƺy śię przy nim obozem / dobywał go.
Y podał Pán Láchys w ręce Izráelá / y wźiął je dniá drugiego / y wyśiekli je oſtrzem miecżá / y wƺelką duƺę / która byłá w nim / ták włáśnie jáko ucżynił Lebnie.
Tedy przyƺedł Horám / król Gázer / ná rátunek Láchyſowi / ále go poráźił Jozue / y lud jego / ták iż nie zoſtáwił mu żádnego żywo.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Láchys do Eglon / y położyli śię obozem przećiwko niemu / y dobywáli go ;
Które wźiąwƺy onegoż dniá / wyśiekli je oſtrzem miecżá / y wƺelką duƺę / która byłá w nim / onegoż dniá zábił / ták włáśnie jáko ucżynił Láchys.
Potem śię ruƺył Jozue / y wƺyſtek Izráel z nim / z Eglonu do Hebronu / y dobywał go ;
Y wźięli je / á wyśiekli je oſtrzem miecżá / y królá jego / y wƺyſtkie miáſtá jego / y wƺelką duƺę / która byłá w nim ; nie zoſtáwił żádnego żywo / ták włáśnie jáko ucżynił Eglonowi / y wytráćił je / y wƺelką duƺę / która w nim byłá.
Z támtąd obróćił śię Jozue / y wƺyſtek Izráel z nim / do Dábir / y dobywał go.
Y wźiął je / y królá jego / y wƺyſtkie miáſtá jego / y wyśiekli je oſtrzem miecżá / y pomordował wƺyſtkie duƺe / które w nim były ; nie zoſtáwił żádnego żywo ; jáko ucżynił Hebronowi ták ucżynił Dábirowi y królowi jego / y jáko ucżynił Lebnie y królowi jego.
A ták pobił Jozue wƺyſtkę źiemię górną / y południową / y polną / y podgórną / y wƺyſtkie króle ich ; nie zoſtáwił żádnego żywo / ále wƺyſtkie duƺe wytráćił / jáko mu był przykázał Pán / Bóg Izráelſki.
Y poráźił je Jozue od Kádes Bárny áż do Gázy / y wƺyſtkę źiemię Goſen / y áż do Gábáon.
A wƺyſtkie te króle / y źiemię ich / wźiął Jozue jednym rázem ; ábowiem Pán / Bóg Izráelſki / wálcżył zá Izráelem.
Zátem śię wróćił Jozue y wƺyſtek Izráel z nim do obozu do Gálgál.
