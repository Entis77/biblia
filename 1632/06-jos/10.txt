A gdy uſłyƺáł Adoniſedek / król Jerozolimſki / iż wźiął Jozue Háj / y zburzył je / ( bo jáko ucżynił Jerychowi y królowi jego / ták ucżynił Hájowi y królowi jego / ) á iż ucżynili pokój obywátele Gábáon z Izráelem / y mieƺkáją w pośrodku ich ;
Tedy śię uląkł bárdzo / przeto że miáſto wielkie było Gábáon / jáko jedno z miáſt królewſkich / á że było więkƺe niż Háj / á wƺyſcy mężowie jego wálecżni.
Przetoż poſłáł Adoniſedek / król Jerozolimſki / do Hohámá / królá Hebron / y do Fárán / królá Jerymotá / y do Jáfijá / królá Láchyſ / y do Dábir / królá Eglon / mówiąc :
Przyjedźćie do mnie / á dájćie mi pomoc / ábyśmy pobili Gábáonity / którzy ucżynili pokój z Jozuem / y z ſyny Izráelſkimi.
Zebráło śię tedy / á wyćiągnęło pięć królów Amorejſkich / król Jerozolimſki / król Hebron / król Jerymot / król Láchyſ / król Eglon / ſámi / y wƺyſtkie wojſká ich / y położyli śię obozem u Gábáon / y dobywáli go.
Tedy poſłáli obywátele Gábáon do Jozuego / y do obozu w Gálgál / mówiąc : Nie záwśćiągáj ręki ſwej od ſług twoich ; przyćiągnij do nich rychło / á wybáw náſ y pomóż nám ; boć śię zebráli przećiwko nám wƺyſcy królowie Amorejſcy / którzy mieƺkáją po górách.
Ruƺył śię tedy Jozue z Gálgál / ſám y wƺyſtek lud wojenny z nim / y wƺyſcy mężowie wálecżni.
( Bo był rzekł Pán do Jozuego : Nie bój śię ich ; ábowiem w ręce twoje podáłem je / á nie oſtoi śię żáden z nich przed tobą. )
Y przypádł ná nie Jozue nágle ; bo cáłą noc ćiągnął z Gálgál.
Y potrwożył je Pán przed oblicżem Izráelá / który je poráźił poráżką wielką w Gábáon / y gonił je drogą / którą chodzą ku Betoron / á bił je áż do Aſeká y áż do Mácedá.
Y ſtáło śię / gdy ućiekáli przed Izráelem / bieżąc z góry do Betoron / że Pán ſpuśćił ná nie kámienie wielkie z niebá áż do Aſeká / y umieráli ; więcej ich pomárło od kámieniá grádowego / niż ich pobili ſynowie Izráelſcy miecżem.
Tedy mówił Jozue do Páná / dniá / którego podáł Pán Amorejcżyká w ręce ſynom Izráelſkim / y rzekł przed ocżymá Izráelá : Słońce w Gábáon záſtánów śię / á mieśiącu w dolinie Ajálon!
Y záſtánowiło śię ſłońce / á mieśiąc ſtánął / áż śię lud pomśćił nád nieprzyjáćioły ſwymi. Izáli to nie jeſt nápiſáno w kśięgách ſpráwiedliwego? Tedy ſtánęło ſłońce w pośród niebá / á nie poſpieƺyło śię záchodźić / jákoby przez cáły dźień.
Y nie był tákowy dźień przedtem / áni potem / w któryby uſłucháć miáł Pán głoſu cżłowiecżego / bo Pán wálcżył zá Izráelem.
Potem śię wróćił Jozue / y wƺyſtek Izráel z nim / do obozu do Gálgál.
A ućiekło było onych pięć królów / y ſkryli śię w jáſkinią przy Mácedá.
Y dáno znáć Jozuemu / mówiąc : Ználeźiono pięć królów / którzy śię pokryli w jáſkini w Mácedá.
Y rzekł Jozue : Przywálćie kámienie wielkie do dźiury jáſkini / á poſtáwćie u niej męże / áby ich ſtrzegli.
A wy nie ſtójćie / gońćie nieprzyjáćioły wáƺe / á bijćie oſtátek ich / áni im dájćie uchodźić do miáſt ich ; boć je podáł Pán / Bóg wáƺ / w rękę wáƺę.
A gdy przeſtáł Jozue z ſyny Izráelſkimi bić ich poráżką bárdzo wielką / áż je do ƺcżętu wytráćili / á którzy żywo zoſtáli z nich / uƺli do miáſt obronnych ;
Tedy wróćił śię wƺyſtek lud zdrowo do obozu / do Jozuego w Mácedá / á nie ruƺył przećiwko ſynom Izráelſkim nikt językiem ſwoim.
Potem rzekł Jozue : Otwórzćie tę dźiurę jáſkini / á wywiedźćie do mnie tych pięćiu królów z jáſkini.
Y ucżynili ták / y wywiedli do niego pięćiu królów onych z jáſkini / królá Jerozolimſkiego / królá Hebron / królá Jerymot / królá Láchyſ / królá Eglon.
A gdy wywiedli one króle do Jozuego / tedy przyzwáł Jozue wƺyſtkich mężów Izráelſkich / y rzekł do rotmiſtrzów / mężów wálecżnych / którzy z nim chodźili : Przyſtąpćie ſám / á náſtąpćie nogámi wáƺemi ná ƺyje tych królów ; którzy przyſtąpiwƺy náſtąpili nogámi ſwemi ná ƺyje ich.
Zátem rzekł do nich Jozue : Nie bójćie śię / áni śię lękájćie ; zmácniájćie śię / y mężnie ſobie pocżynájćie ; boć ták ucżyni Pán wƺyſtkim nieprzyjáćiołom wáƺym / przećiw którym wálcżyćie.
Potem pobił je Jozue / y pomordowáł je / y záwieśił je ná pięćiu drzewách / á wiśieli ná drzewách áż do wiecżorá.
A gdy záƺło ſłońce / rozkázáł Jozue / że je złożono z drzewá / y wrzucono je do jáſkini / w której śię byli ſkryli / á záwálono kámieńmi wielkiemi dźiurę u jáſkini / które tám ſą jeƺcże y do dniá tego.
Tegoż dniá wźiął Jozue Mácedá / y wyśiekł je oſtrzem miecżá / y królá ich zámordowáł weſpół z nimi / y wƺelką duƺę / którá byłá w nim ; nie zoſtáwił żádnego żywo / y ucżynił królowi Mácedá / jáko ucżynił królowi Jerychá.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Mácedá do Lebny / y dobywáł Lebny.
A podáł Pán y ono w ręce Izráelá / y królá jego / y wyśiekł je oſtrzem miecżá / y wƺelką duƺę / którá byłá w nim ; nie zoſtáwił w nim żádnego żywo / y ucżynił królowi jego / jáko ucżynił królowi Jerychá.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Lebny do Láchyſ / á położywƺy śię przy nim obozem / dobywáł go.
Y podáł Pán Láchyſ w ręce Izráelá / y wźiął je dniá drugiego / y wyśiekli je oſtrzem miecżá / y wƺelką duƺę / którá byłá w nim / ták włáśnie jáko ucżynił Lebnie.
Tedy przyƺedł Horám / król Gázer / ná rátunek Láchyſowi / ále go poráźił Jozue / y lud jego / ták iż nie zoſtáwił mu żádnego żywo.
Potem ćiągnął Jozue / y wƺyſtek Izráel z nim / z Láchyſ do Eglon / y położyli śię obozem przećiwko niemu / y dobywáli go ;
Które wźiąwƺy onegoż dniá / wyśiekli je oſtrzem miecżá / y wƺelką duƺę / którá byłá w nim / onegoż dniá zábił / ták włáśnie jáko ucżynił Láchyſ.
Potem śię ruƺył Jozue / y wƺyſtek Izráel z nim / z Eglonu do Hebronu / y dobywáł go ;
Y wźięli je / á wyśiekli je oſtrzem miecżá / y królá jego / y wƺyſtkie miáſtá jego / y wƺelką duƺę / którá byłá w nim ; nie zoſtáwił żádnego żywo / ták włáśnie jáko ucżynił Eglonowi / y wytráćił je / y wƺelką duƺę / którá w nim byłá.
Stámtąd obróćił śię Jozue / y wƺyſtek Izráel z nim / do Dábir / y dobywáł go.
Y wźiął je / y królá jego / y wƺyſtkie miáſtá jego / y wyśiekli je oſtrzem miecżá / y pomordowáł wƺyſtkie duƺe / które w nim były ; nie zoſtáwił żádnego żywo ; jáko ucżynił Hebronowi ták ucżynił Dábirowi y królowi jego / y jáko ucżynił Lebnie y królowi jego.
A ták pobił Jozue wƺyſtkę źiemię górną / y południową / y polną / y podgórną / y wƺyſtkie króle ich ; nie zoſtáwił żádnego żywo / ále wƺyſtkie duƺe wytráćił / jáko mu był przykázáł Pán / Bóg Izráelſki.
Y poráźił je Jozue od Kádeſ Bárny áż do Gázy / y wƺyſtkę źiemię Goſen / y áż do Gábáon.
A wƺyſtkie te króle / y źiemie ich / wźiął Jozue jednym rázem ; ábowiem Pán / Bóg Izráelſki / wálcżył zá Izráelem.
Zátem śię wróćił Jozue y wƺyſtek Izráel z nim do obozu do Gálgál.
