Tedy Jozue wſtáł bárdzo ráno / y ruƺyli śię z Syttim / á przyƺli áż do Jordánu / on y wƺyſcy ſynowie Izráelſcy / y támże przenocowáli / niźli śię przepráwili.
A po trzećim dniu przeƺli przełożeni przez pośrodek obozu.
Y rozkázáli ludowi / mówiąc : Gdy ujrzyćie ſkrzynię przymierzá Páná / Bogá wáƺego / y kápłány Lewity / nioſące ją / wy też ruƺyćie śię z miejſcá ſwego / á pójdziećie zá nią ;
Wƺákże plác między wámi y między nią będzie ná dwá tyśiące łokći miáry zwycżájnej ; nie przyſtępujćie bliſko do niej / ábyśćie wiedzieli drogę / którą iść máćie ; ábowiem nie chodziliśćie tą drogą przedtem.
Tedy rzekł Jozue do ludu : Poświęććie śię ; ábowiem jutro ucżyni Pán między wámi dziwne rzecży.
Przytem rzekł Jozue do kápłánów / mówiąc : Weźmijćie ſkrzynię przymierzá / á idźćie przed ludem ; y wzięli ſkrzynię przymierzá / y ƺli przed ludem.
Y rzekł Pán do Jozuego : Dziś ćię pocżnę wywyżƺáć przed ocżymá wƺyſtkiego Izráelá / áby poználi / iż jákom był z Mojżeƺem / ták będę y z tobą.
Rozkáżże ty kápłánom / nioſącym ſkrzynię przymierzá / y rzecż im : Gdy wnijdziećie w brzeg wód Jordáńſkich / w Jordánie ſtániećie.
Rzekł też Jozue do ſynów Izráelſkich : Przyſtąpćie ſám / á ſłuchájćie ſłów Páná / Bogá wáƺego.
Y rzekł Jozue : W tem poznáćie / że Bóg żyjący jeſt w pośrodku wáſ / á iż koniecżnie wypędzi przed twárzą wáƺą Chánánejcżyká / y Hetejcżyká / y Hewejcżyká / y Ferezejcżyká / y Gergiezejcżyká / y Amorejcżyká / y Jebuzejcżyká.
Oto / ſkrzyniá przymierzá Pánującego nád wƺyſtką ziemią pójdzie przed wámi przez Jordán.
Przetoż teráz obierzćie ſobie dwánáśćie mężów z pokoleń Izráelſkich / po jednym mężu z káżdego pokoleniá ;
A gdy śię záſtánowią ſtopy nóg kápłánów / nioſących ſkrzynię Páná / Pánującego nád wƺyſtką ziemią / w wodzie Jordáńſkiej / tedy śię wody jordáńſkie rozſtąpią / ták iż wodá płynącá z góry ſtánie w jednej kupie.
Y ſtáło śię / gdy śię ruƺył lud z námiotów ſwych / áby śię przepráwili przez Jordán / á kápłáni / nioſący ſkrzynię przymierzá / ƺli przed ludem ;
A gdy przyƺli nioſący ſkrzynię áż do Jordánu / á nogi kápłánów / którzy nieśli ſkrzynię / omocżyły śię w brzegu wód / ( bo Jordán wzbierá y wylewá ná wƺyſtkie brzegi ſwoje / ná káżdy cżáſ żniwá. )
Tedy śię záſtánowiły wody płynące z gór / á ſtánęły w jednej kupie bárdzo dáleko od Adámá / miáſtá / które jeſt ku ſtronie Sártán ; á które płynęły ná dół do morzá puſtego / morzá ſłonego / zginęły y uſtáły ; á ták lud przepráwiáł śię przećiwko Jerychu.
A kápłáni / którzy nieśli ſkrzynię przymierzá Páńſkiego / ſtáli ná ſuƺy w pośród Jordánu porządnie / á wƺyſcy Izráelcżycy ƺli po ſuƺy / áż śię lud wƺyſtek przepráwił przez Jordán.
