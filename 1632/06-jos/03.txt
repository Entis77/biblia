Tedy Jozue wſtał bárzo ráno / y ruƺyli śię z Syttim / á przyƺli áż do Jordánu / on y wƺyſcy ſynowie Izráelſcy / y támże przenocowáli / niźli śię przepráwili.
A po trzećim dniu przeƺli przełożeni przez pośrodek obozu.
Y rozkázáli ludowi / mówiąc : Gdy ujrzyćie ſkrzynię przymierza Páná / Bogá wáƺego / y kápłány Lewity / nioſące ją / wy też ruƺyćie śię z miejſcá ſwego / á pójdźiećie zá nią ;
Wƺákże plác miedzy wámi y miedzy nią będźie ná dwá tyśiące łokći miáry zwycżájney ; nie przyſtępujćie bliſko do niej / ábyśćie wiedźieli drogę / którą iść máćie ; ábowiem nie chodźiliśćie tą drogą przedtem.
Tedy rzekł Jozue do ludu : Poświęććie śię ; ábowiem jutro ucżyni Pán miedzy wámi dźiwne rzecży.
Przytem rzekł Jozue do kápłánów / mówiąc : Weźmijćie ſkrzynię przymierza / á idźćie przed ludem ; y wźięli ſkrzynię przymierza / y ƺli przed ludem.
Y rzekł Pán do Jozuego : Dźiś ćię pocżnę wywyżƺáć przed ocżymá wƺyſtkiego Izráelá / áby poználi / iż jákom był z Mojżeƺem / ták będę y z tobą.
Rozkáżże ty kápłánom / nioſącym ſkrzynię przymierza / y rzecż im : Gdy wnijdźiećie w brzeg wód Jordáńſkich / w Jordánie ſtániećie.
Rzekł też Jozue do ſynów Izráelſkich : Przyſtąpćie ſám / á ſłuchájćie ſłów Páná / Bogá wáƺego.
Y rzekł Jozue : W tym poznáćie / że Bóg żyjący jeſt w pośrzodku was / á iż koniecżnie wypędźi przed twárzą wáƺą Chánánejcżyká / y Hetejcżyká / y Hewejcżyká / y Ferezejcżyká / y Gergiezejcżyká / y Amorejcżyká / y Jebuzejcżyká.
Oto / ſkrzyniá przymierza Pánującego nád wƺyſtką źiemią pójdźie przed wámi przez Jordán.
Przetoż teraz obierzćie ſobie dwánáśćie mężów z pokoleń Izráelſkich / po jednym mężu z káżdego pokolenia ;
A gdy śię záſtánowią ſtopy nóg kápłánów / nioſących ſkrzynię Páná / Pánującego nád wƺyſtką źiemią / w wodźie Jordáńſkiej / tedy śię wody jordáńſkie rozſtąpią / ták iż wodá płynącá z góry ſtánie w jedney kupie.
Y ſtáło śię / gdy śię ruƺył lud z námiotów ſwych / áby śię przepráwili przez Jordán / á kápłáni / nioſący ſkrzynię przymierza / ƺli przed ludem ;
A gdy przyƺli nioſący ſkrzynię áż do Jordánu / á nogi kápłánów / którzy nieśli ſkrzynię / omocżyły śię w brzegu wód / ( bo Jordán wzbierá y wylewá ná wƺyſtkie brzegi ſwoje / ná káżdy cżás żniwá. )
Tedy śię záſtánowiły wody płynące z gór / á ſtánęły w jedney kupie bárzo dáleko od Adámá / miáſtá / które jeſt ku ſtronie Sártán ; á które płynęły ná dół do morzá puſtego / morzá ſłonego / zginęły y uſtáły ; á ták lud przepráwiał śię przećiwko Jerychu.
A kápłáni / którzy nieśli ſkrzynię przymierza Páńſkiego / ſtáli ná ſuƺy w pośród Jordánu porządnie / á wƺyſcy Izráelcżycy ƺli po ſuƺy / áż śię lud wƺyſtek przepráwił przez Jordán.
