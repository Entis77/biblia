Y zſtárzáł śię Jozue / á był zeƺły w lećiech. Y rzekł Pán do niego : Tyś śię zſtárzáł / á zƺedłeś w lećiech / á źiemi zoſtáwá bárdzo wiele ku pośiádániu.
Táć jeſt źiemiá / która pozoſtáwá : Wƺyſtkie gránice Filiſtynów / y wƺyſcy Geſſurytowie /
Od Nilu / który oblewá Egipt / áż do gránicy Akáronu ná północy / przynáleży Chánánejcżykowi / pięćioro kśięſtw Filiſtyńſkich ; Aſáckie / y Aſdodźiejſkie / Aſkálońſkie / Getejſkie / y Akáronickie / y Háwejcżycy.
Od południá wƺyſtká źiemiá Chánánejſká / y Márá / które jeſt Sydońcżyków áż do Afeká / y áż do gránicy Amorejcżyká ;
Y źiemiá Giblitów ze wƺyſtkim Libánem ná wſchód ſłońcá / od Báálgád pod górę Hermon / áż gdźie chodzą do Emát.
Wƺyſtkie mieƺkájące ná górách od Libánu áż do wód gorących / wƺyſtkie Sydońcżyki Ja wypędzę przed ſyny Izráelſkimi ; tylko ją podźiel Izráelitom w dźiedźictwo / jákom ći rozkázáł.
Przetoż teraz rozdźiel tę źiemię w dźiedźictwo / dźiewięćiorgu pokoleniu / y połowie pokoleniá Mánáſeſowego.
Gdyż z drugą połową Rubenitowie y Gádytowie wźięli dźiedźictwo ſwoje / które im dáł Mojżeƺ zá Jordánem ná wſchód ſłońcá / jáko im dáł Mojżeƺ / ſługá Páńſki ;
Od Aroer / które jeſt nád brzegiem potoku Arnon / y miáſto / które jeſt w pośrodku potoku / y wƺyſtkę równinę Medebá áż do Dybon ;
Y wƺyſtkie miáſtá Sehoná / królá Amorejſkiego / który królowáł w Heſebon / áż do gránicy ſynów Ammonowych ;
Tákże Gáláád / y gránice Geſſurytów / y Máchátytów / y wƺyſtkę górę Hermon / y wƺyſtko Báſán áż do Sálechá ;
Wƺyſtko króleſtwo Ogá w Báſán / który królowáł w Aſtárot / y w Edrej ; ten był pozoſtáł z Refáimitów / á pobił je Mojżeƺ y wygłádźił je.
Ale nie wygnáli ſynowie Izráelſcy Geſſurytów y Máchátytów ; przetoż mieƺkáł Geſſur y Máchát w pośród Izráelcżyków áż do dniá tego.
Tylko pokoleniu Lewi nie dáł dźiedźictwá ; ofiáry ogniſte Páná Bogá Izráelſkiego ſą dźiedźictwem jego / jáko mu powiedźiáł Bóg.
A ták oddáł Mojżeƺ pokoleniu ſynów Rubenowych dźiedźictwo według fámilii ich.
Y byłá gránicá ich od Aroer / które jeſt nád brzegiem potoku Arnon / y miáſto / które jeſt w pośród potoku / y wƺyſtká równiná ku Medebá.
Heſebon / y wƺyſtkie miáſtá jego / które były w równinie ; Dybon y Bámot Báál / y Bet Báál Meon ;
Y Jáſſá / y Cedymot / y Mefáát ;
Y Káryjátáim / y Sebámá / y Sárátáſár ná górze w dolinie ;
Y Betfegor / y Aſdod / Fázgá / y Betyjeſymot.
Wƺyſtkie też miáſtá w równinie / y wƺyſtko króleſtwo Sehoná / królá Amorejſkiego / który królowáł w Heſebon / którego zábił Mojżeƺ / y kśiążętá Mádyjáńſkie Ewi / y Recem / y Sur / y Hur / y Rebá ; kśiążętá Sehonowe / obywátele źiemi.
Y Báláámá / ſyná Boerowego / wieƺcżká / zábili ſynowie Izráelſcy miecżem z innymi pobitymi.
Byłá tedy gránicá ſynów Rubenowych Jordán z gránicámi ſwemi. Toć jeſt dźiedźictwo ſynów Rubenowych według domów ich / miáſt y wśi ich.
Dáł też Mojżeƺ pokoleniu Gád / ſynom Gádowym / według domów ich dźiedźictwo.
A były ich gránice Jázer y wƺyſtkie miáſtá Gáláád / y połowá źiemi ſynów Ammonowych áż do Aroer / które jeſt przećiw Rábbá ;
Y od Heſebon áż do Rámát Máſſá y Betonim / á od Máhánáim áż do gránicy Dábir.
W dolinie też Berám / y Betnimrá / y Sochot / y Sáfon / oſtátek króleſtwá Sehoná / królá Heſebońſkiego / Jordán y pogránicże jego áż do końcá morzá Cynneret zá Jordánem ná wſchód ſłońcá.
Toć jeſt dźiedźictwo ſynów Gád według domów ich / miáſt y wśi ich.
Nádto dáł Mojżeƺ ośiádłość połowie pokoleniá Mánáſeſowego / y byłá tá połowá pokoleniá ſynów Mánáſeſowych według domów ich.
Byłá gránicá ich od Máchánáim wƺyſtko Báſán y wƺyſtko króleſtwo Ogá / królá Báſáńſkiego / y wƺyſtkie wśi Jáirowe / które ſą w Báſán / ƺeśćdźieśiąt miáſt.
Y połowę Gáláád / y Aſtárot / y Erdej / miáſtá króleſtwá Ogá w Báſán / dáł ſynom Máchyrá / ſyná Mánáſeſowego / połowie ſynów Máchyrowych według domów ich.
Teć ſą ośiádłośći / które podźielił Mojżeƺ w polách Moábſkich zá Jordánem przećiw Jerychu ná wſchód ſłońcá.
Ale pokoleniu Lewi nie dáł Mojżeƺ dźiedźictwá ; bo Pán / Bóg Izráelſki / ſám jeſt dźiedźictwem ich / jáko im powiedźiáł.
