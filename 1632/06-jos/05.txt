Y ſtáło śię / gdy uſłyƺeli wƺyſcy królowie Amorejſcy / którzy mieƺkáli zá Jordánem ku záchodowi / y wƺyſcy królowie Chánánejſcy / którzy mieƺkáli nád morzem / że wyſuƺył Pán wody Jordáńſkie przed ſyny Izráelſkimi / áż śię przepráwili / upádło ſerce ich / ták iż nie zoſtáł więcej w nich duch przed oblicżnośćią ſynów Izráelſkich.
Onegoż cżáſu rzekł Pán do Jozuego : Ucżyń ſobie noże oſtre / á znowu obrzeż ſyny Izráelſkie po wtóre.
Y ucżynił ſobie Jozue noże oſtre y obrzezáł ſyny Izráelſkie ná págórku nieobrzezek.
A táć byłá przycżyná / dlá cżego je obrzezáł Jozue : Wƺyſtek lud / który wyƺedł z Egiptu / płći męſkiej / wƺyſcy mężowie wojenni / pomárli byli ná puƺcży / w drodze / gdy wyƺli z Egiptu.
Bo obrzezán był wƺyſtek on lud / co wyƺedł ; ále wƺyſtek lud / który śię zrodził ná puƺcży / w drodze po wyjśćiu z Egiptu / nie był obrzezány.
( Abowiem przez cżterdzieśći lát chodzili ſynowie Izráelſcy po puƺcży / áż poginął wƺyſtek on náród mężów wojennych / którzy byli wyƺli z Egiptu / którzy nie ſłucháli głoſu Páńſkiego / którym przyśiągł Pán / iż im nie miáł okázáć ziemi / o którą przyśiągł Pán ojcom ich / iż nám ją dáć miáł / ziemię opływájącą mlekiem y miodem. )
Ale ſyny ich / które wyſtáwił ná miejſcá ich / te obrzezáł Jozue / bo byli w nieobrzezce ; bo ich nie obrzezáno w drodze.
A gdy już wƺyſtek lud był obrzezány / mieƺkáł ná miejſcu ſwem w obozie / áż śię wygoili.
Potem rzekł Pán do Jozuego : Dziśiájm zdjął poháńbienie Egipſkie z wáſ ; y názwáno imię miejſcá onego Gálgál / áż do dniá tego.
Tedy położyli śię obozem ſynowie Izráelſcy w Gálgál / á obchodzili święto przejśćiá cżternáſtego dniá mieśiącá w wiecżór ná polách Jerychá.
Y jedli z urodzájów onej ziemi názájutrz po święćie przejśćiá chleby przáśne / y kłoſy práżone onegoż dniá.
Y przeſtáłá mánná názájutrz / gdy pocżęli jeść zbożá onej ziemi ; y nie mieli więcej ſynowie Izráelſcy mánny / ále jedli z urodzájów ziemi Chánánejſkiej onegoż roku.
Y ſtáło śię / gdy Jozue był u Jerychá / że podnióſł ocżu ſwych á ujrzáł / á oto mąż ſtáł przećiwko niemu / májąc miecż ſwój dobyty w ręce ſwej ; y przyſtąpiwƺy do niego Jozue / rzekł mu : Z náƺychżeś ty / cży z nieprzyjáćiół náƺych?
A on rzekł : Nie ; álem Já hetmán wojſká Páńſkiego / terázem przyƺedł. Tedy upádłƺy Jozue oblicżem ſwem ná ziemię / pokłonił śię / y rzekł mu : Cóż Pán mój mówi do ſługi ſwego?
Y rzekł hetmán wojſká Páńſkiego do Jozuego : Zzuj obuwie twoje z nóg twoich / bo miejſce / ná którem ſtoiƺ / święte jeſt ; y ucżynił ták Jozue.
