Y ſtáło śię / gdy uſłyƺeli wƺyſcy królowie Amorejſcy / którzy mieƺkáli zá Jordánem ku záchodowi / y wƺyſcy królowie Chánánejſcy / którzy mieƺkáli nád morzem / że wyſuƺył Pán wody Jordáńſkie przed ſyny Izráelſkimi / áż śię przepráwili / upádło ſerce ich / ták iż nie zoſtáł więcey w nich duch przed oblicżnośćią ſynów Izráelſkich.
Onegoż cżáſu rzekł Pán do Jozuego : Ucżyń ſobie noże oſtre / á znowu obrzeż ſyny Izráelſkie po wtóre.
Y ucżynił ſobie Jozue noże oſtre y obrzezáł ſyny Izráelſkie ná págórku nieobrzezek.
A táć byłá przycżyná / dla cżego je obrzezáł Jozue : Wƺyſtek lud / który wyƺedł z Egiptu / płći męſkiej / wƺyſcy mężowie wojenni / pomárli byli ná puƺcży / w drodze / gdy wyƺli z Egiptu.
Bo obrzezán był wƺyſtek on lud / co wyƺedł ; ále wƺyſtek lud / który śię zrodźił ná puƺcży / w drodze po wyjśćiu z Egiptu / nie był obrzezány.
( Abowiem przez cżterdźieśći lát chodźili ſynowie Izráelſcy po puƺcży / áż poginął wƺyſtek on náród mężów wojennych / którzy byli wyƺli z Egiptu / którzy nie ſłucháli głoſu Páńſkiego / którym przyśiągł Pán / iż im nie miał okázáć źiemi / o którą przyśiągł Pán ojcom ich / iż nám ją dáć miał / źiemię opływájącą mlekiem y miodem. )
Ale ſyny ich / które wyſtáwił ná miejſcá ich / te obrzezáł Jozue / bo byli w nieobrzezce ; bo ich nie obrzezáno w drodze.
A gdy już wƺyſtek lud był obrzezány / mieƺkáł ná miejſcu ſwem w oboźie / áż śię wygoili.
Potem rzekł Pán do Jozuego : Dźiśiájm zdjął poháńbienie Egipſkie z was ; y názwáno imię miejſcá onego Gálgál / áż do dniá tego.
Tedy położyli śię obozem ſynowie Izráelſcy w Gálgál / á obchodźili święto przejśćiá cżternáſtego dniá mieśiącá w wiecżór ná polách Jerychá.
Y jedli z urodzájów oney źiemi názájutrz po święćie przejśćiá chleby przáśne / y kłoſy práżone onegoż dniá.
Y przeſtáłá mánná názájutrz / gdy pocżęli jeść zbożá oney źiemi ; y nie mieli więcey ſynowie Izráelſcy mánny / ále jedli z urodzájów źiemi Chánánejſkiej onegoż roku.
Y ſtáło śię / gdy Jozue był u Jerychá / że podnióſł ocżu ſwych á ujrzáł / á oto mąż ſtáł przećiwko niemu / májąc miecż ſwój dobyty w ręce ſwej ; y przyſtąpiwƺy do niego Jozue / rzekł mu : Z náƺychżeś ty / cży z nieprzyjáćiół náƺych?
A on rzekł : Nie ; álem Ja hetmán wojſká Páńſkiego / terazem przyƺedł. Tedy upádłƺy Jozue oblicżem ſwem ná źiemię / pokłonił śię / y rzekł mu : Cóż Pán mój mówi do ſługi ſwego?
Y rzekł hetmán wojſká Páńſkiego do Jozuego : Zzuj obuwie twoje z nóg twoich / bo miejſce / ná którem ſtoiƺ / święte jeſt ; y ucżynił ták Jozue.
