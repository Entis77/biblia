A gdy uſłyƺeli wƺyſcy królowie / którzy byli zá Jordánem ná górách / y ná równinách / y nád wƺyſtkiem brzegiem morzá wielkiego przećiw Libánowi / Hetejcżyk / y Amorejcżyk / y Chánánejcżyk / Ferezejcżyk / Hewejcżyk / y Jebuzejcżyk ;
Zebráli śię poſpołu / áby wálcżyli przećiw Jozuemu / y przećiw Izráelowi / jednomyślnie.
Ale obywátele Gábáon / uſłyƺáwƺy / co ucżynił Jozue Jerychowi y Hájowi /
Poſtąpili ſobie y oni chytrze / á poƺedłƺy zmyślili śię być poſłámi / y wźięli wory ſtáre ná oſły ſwe / y łágwie winne ſtáre / y potárte / y łátáne ;
Y obuwie ſtáre y łátáne ná nogi ſwoje / y ƺáty ſtáre ná śię / á wƺyſtek chleb / co go z ſobą nábráli w drogę / ſuchy był y ſpleśniáły.
Tedy przyƺli do Jozuego / do obozu w Gálgál / y rzekli do niego / y do mężów Izráelſkich : Z źiemiśmy dálekiej przyƺli ; przetoż teraz ucżyńćie z námi przymierze.
Ale odpowiedźieli mężowie Izráelſcy Hewejcżykowi : Podobno ty mieƺkáƺ między námi / á jákoż z tobą możemy ucżynić przymierze?
A oni rzekli do Jozuego : Słudzy twoi jeſteśmy. Y rzekł do nich Jozue : Cośćie wy zácż / á ſkądeśćie przyƺli?
Y odpowiedźieli mu : Z źiemi dálekiej bárdzo przyƺli ſłudzy twoi w imieniu Páná / Bogá twego ; bośmy ſłyƺeli ſłáwę jego / y wƺyſtko / co ucżynił w Egipćie ;
Y wƺyſtko / co ucżynił dwom królom Amorejſkim / którzy byli zá Jordánem / Sehonowi królowi Heſebon / y Ogowi królowi Báſán / którzy mieƺkáli w Aſtárot.
Y rozkázáli nám ſtárśi náśi / y wƺyſcy obywátele źiemi náƺej / mówiąc : Nábierzćie ſobie żywnośći ná drogę / á idźćie przećiwko nim / y mówćie im : Słudzy wáśi jeſteśmy / przetoż teraz ucżyńćie z námi przymierze.
Ten chleb náƺ ćiepłyśmy ná drogę wźięli z domów náƺych tego dniá / gdyśmy wyƺli / ábyśmy ƺli do was ; á teraz oto wyſechł / y popleśniáł.
Y te łágwie winne / któreśmy byli nápełnili / były nowe / á oto śię popękáły ; tákże te ƺáty náƺe / y obuwie náƺe zwiotƺáły dla bárdzo dálekiej drogi.
A ták wźięli oni mężowie Izráelſcy z oney żywnośći ich / á uſt śię Páńſkich nie pytáli.
Tedy z nimi ucżynił Jozue pokój / y poſtánowił z nimi przymierze / áby ich záchowáł przy żywoćie ; tákże przyśięgły im kśiążętá zgromádzeniá.
Ale po trzech dniách po ucżynieniu z nimi przymierzá / uſłyƺeli / że bliſko ich byli / á iż w pośrodku ich mieƺkáli.
A ruƺywƺy śię ſynowie Izráelſcy przyćiągnęli do miáſt ich dniá trzećiego / á miáſtá ich te były : Gábáon / y Káfirá / y Beerot / y Káryjátyjárym.
Y nie wytráćili ich ſynowie Izráelſcy ; ábowiem przyśięgły im były kśiążętá zgromádzeniá przez Páná / Bogá Izráelſkiego / ſkąd ƺemráło wƺyſtko zgromádzenie przećiw kśiążętom.
Y rzekły wƺyſtkie kśiążętá do cáłego zgromádzeniá : Myśmy im przyśięgli przez Páná / Bogá Izráelſkiego ; przetoż teraz nie możemy śię ich tknąć.
To im ucżynimy / á záchowámy je żywo / iżby nie przyƺło ná nas rozgniewánie dla przyśięgi / którąśmy im przyśięgli.
Nádto rzekły do nich kśiążętá : Niech żyją / á niech rąbią drwá / y niech noƺą wodę wƺyſtkiemu zgromádzeniu ; y przeſtáli ná tem / jáko im powiedźiáły kśiążętá.
Potem wezwáł ich Jozue / y rzekł do nich / mówiąc : Przecżżeśćie nas oƺukáli / powiádájąc : Dálekimiśmy od was bárdzo / á wy w pośrodku nas mieƺkáćie?
A ták teraz przeklęći jeſteśćie / y nie uſtáną z was ſłudzy / y rąbiący drwá / y noƺący wodę do domu Bogá mego.
Którzy odpowiedźieli Jozuemu / y rzekli : Zápewne oznájmiono było ſługom twoim / jáko był rozkázáł Pán / Bóg twój / Mojżeƺowi ſłudze ſwemu / áby wám dáł wƺyſtkę źiemię / á iżby wygłádźił wƺyſtkie mieƺkájące w tey źiemi przed twárzą wáƺą ; przetoż báliśmy śię bárdzo o żywot náƺ przed wámi / y ucżyniliśmy tę rzecż.
A teraz otośmy w rękách twoich ; coć śię dobrego y ſłuƺnego widźi ucżynić z námi / ucżyń.
Y ucżynił im ták / á wybáwił je z rąk ſynów Izráelſkich / że ich nie pobili.
Y poſtánowił je Jozue dniá onego / áby rąbáli drwá / y nośili wodę zgromádzeniu / y do ołtárzá Páńſkiego áż do tego dniá / ná miejſcu / któreby obráł.
