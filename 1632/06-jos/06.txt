A Jerycho było zámknione / y opátrzone przed ſynámi Izráelſkimi / y nikt z niego nie wychodźił / áni do niego wchodźił.
Tedy rzekł Pán do Jozuego : Otom dał w ręce twoje Jerycho / y królá jego / y możne wojſká jego.
A ták obchodźić będźiećie miáſto / wƺyſcy mężowie wálecżni / około miáſtá chodząc raz ná dźień ; ták ucżynićie po ƺeść dni.
Przytem śiedm kápłánów ponioſą śiedm trąb z rogów báránich / przed ſkrzynią ; á dniá śiódmego obejdźiećie miáſto śiedm kroć / á kápłáni trąbić będą w trąby.
A gdy przewłocżnie trąbić będą w trąby z rogów báránich / ſkoro uſłyƺyćie głos trąby / wƺyſtek lud ucżyni okrzyk bárzo wielki / y upádnie mur miáſtá ná miejſcu ſwem / y wnijdźie lud do miáſtá / káżdy przećiw miejſcu / gdźie ſtał.
Tedy wezwáwƺy Jozue / ſyn Nunów / kápłánów / rzekł do nich : weźmijćie ſkrzynię przymierza / á śiedm kápłánów niech nioſą śiedm trąb z báránich rogów przed ſkrzynią Páńſką.
Potem rzekł do ludu : Idźćie á obejdźćie miáſto / á zbrojni niech idą przed ſkrzynią Páńſką.
A gdy to Jozue ludowi powiedźiał / śiedm kápłánów wźiąwƺy śiedm trąb z rogów báránich / ƺli przed ſkrzynią Páńſką / y trąbili w trąby / á ſkrzyniá przymierza Páńſkiego ƺłá zá nimi.
A zbrojni ƺli przed kápłány trąbiącymi w trąby ; oſtátek też ludu poſpolitego ƺedł zá ſkrzynią / gdy idąc trąbiono w trąby.
A ludowi przykázał Jozue / mówiąc : Nie będźiećie wołáć / áni będźie ſłyƺán głos wáƺ / áni wynijdźie z uſt wáƺych ſłowo / áż do dniá / którego wam rzekę : Wołájćie ; y ucżynićie okrzyk.
Tedy obeƺłá ſkrzyniá Páńſká miáſto w około raz ; y wróćili śię do obozu / y zoſtáli w oboźie przez noc.
Wſtał záśię Jozue ráno / á kápłáni wźięli ſkrzynię Páńſką.
A śiedm kápłánów wźiąwƺy śiedm trąb z rogów báránich / przed ſkrzynią Páńſką ƺli / idąc y trąbiąc w trąby ; á zbrojni ƺli przed nimi / oſtátek też ludu poſpolitego ƺedł zá ſkrzynią Páńſką / gdy idąc trąbiono w trąby.
A ták obeƺli miáſto drugi raz dniá wtórego / y wróćili śię do obozu ; y ták cżynili po ƺeść dni.
Ale dniá śiódmego wſtáli ráno ná świtániu / y obeƺli miáſto tymże ſpoſobem śiedm kroć ; tylko dniá tego obeƺli miáſto śiedm kroć.
Y ſtáło śię / gdy śiódmy raz obchodźili / á kápłáni trąbili w trąby / rzekł Jozue do ludu : Krzycżćież teraz ; ábowiem Pán podał wam miáſto.
Y niech będźie to miáſto przeklęſtwem Pánu / ono / y wƺyſtko co w nim jeſt ; tylko Rácháb wƺetecżnicá żywo zoſtánie / oná y wƺyſcy / którzy z nią ſą w domu / gdyż utáiłá poſłów / któreśmy byli poſłáli.
A wƺákże śię wy ſtrzeżćie od rzecży przeklętych / ábyśćie śię nie ſtáli przeklęſtwem / biorąc co z rzecży przeklętych / ábyśćie nie wpráwili obozu Izráelſkiego w przeklęſtwo / y nie zámięƺáli go.
Ale wƺyſtko ſrebro y złoto y nácżyniá miedźiáne y żelázne / święte będą Pánu ; do ſkárbu Páńſkiego złożone będą.
Tedy krzycżał lud / gdy zátrąbiono w trąby / ábowiem gdy uſłyƺał lud głos trąb / krzycżał y lud wielkim głoſem / y upádł mur ná miejſcu ſwem / y wƺedł lud do miáſtá / káżdy przećiw miejſcu / gdźie ſtał / y wźięli miáſto ;
Y wytráćili wƺyſtko / co było w mieśćie / męże y niewiáſty / dźieći y ſtárce ; woły też y owce / y oſły oſtrzem miecżá pobili.
Ale dwom mężom / którzy ƺpiegowáli onę źiemię / rzekł Jozue : Wnijdźćie do domu niewiáſty wƺetecżney / á wywiedźćie ſtámtąd niewiáſtę / y wƺyſtko / co jey jeſt / jákośćie jey przyśięgli.
Tedy wƺedƺy młodźieńcy oni / co byli wyƺpiegowáli źiemię / Ráchábę / y ojcá jey / mátkę jey y bráćią jey / y wƺyſtko co było jey / y wƺyſtkę rodźinę jey wywiedli / y zoſtáwili je zá obozem Izráelſkim.
Ale miáſto ſpálili ogniem / y wƺyſtko / co w nim było ; tylko ſrebro y złoto / y nacżynie miedźiáne / y żelázne / złożyli do ſkárbu domu Páńſkiego.
Ráchábę tákże wƺetecżnicę / y dom ojcá jey / y wƺyſtko / co było jey / Jozue żywo zoſtáwił / y mieƺkáłá w pośrzodku Izráelá áż do teráźniejƺego dniá / dla tego / iż utáiłá poſłów / które był poſłał Jozue ku przeƺpiegowániu Jerychá.
Y wydał klątwę Jozue onego cżáſu / mówiąc : Przeklęty mąż przed Pánem / któryby powſtał á budował to miáſto Jerycho ; ná pierworodnym ſwoim záłoży je / á ná nájmniejƺym poſtáwi brámy jego.
Y był Pán z Jozuem / á rozchodźiłá śię ſłáwá jego po wƺyſtkiej źiemi.
