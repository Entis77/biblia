A Jerycho było zámknione / y opátrzone przed ſynámi Izráelſkimi / y nikt z niego nie wychodził / áni do niego wchodził.
Tedy rzekł Pán do Jozuego : Otom dáł w ręce twoje Jerycho / y królá jego / y możne wojſká jego.
A ták obchodzić będziećie miáſto / wƺyſcy mężowie wálecżni / około miáſtá chodząc ráz ná dzień ; ták ucżynićie po ƺeść dni.
Przytem śiedm kápłánów ponioſą śiedm trąb z rogów báránich / przed ſkrzynią ; á dniá śiódmego obejdziećie miáſto śiedm kroć / á kápłáni trąbić będą w trąby.
A gdy przewłocżnie trąbić będą w trąby z rogów báránich / ſkoro uſłyƺyćie głoſ trąby / wƺyſtek lud ucżyni okrzyk bárdzo wielki / y upádnie mur miáſtá ná miejſcu ſwem / y wnijdzie lud do miáſtá / káżdy przećiw miejſcu / gdzie ſtáł.
Tedy wezwáwƺy Jozue / ſyn Nunów / kápłánów / rzekł do nich : weźmijćie ſkrzynię przymierzá / á śiedm kápłánów niech nioſą śiedm trąb z báránich rogów przed ſkrzynią Páńſką.
Potem rzekł do ludu : Idźćie á obejdźćie miáſto / á zbrojni niech idą przed ſkrzynią Páńſką.
A gdy to Jozue ludowi powiedziáł / śiedm kápłánów wziąwƺy śiedm trąb z rogów báránich / ƺli przed ſkrzynią Páńſką / y trąbili w trąby / á ſkrzyniá przymierzá Páńſkiego ƺłá zá nimi.
A zbrojni ƺli przed kápłány trąbiącymi w trąby ; oſtátek też ludu poſpolitego ƺedł zá ſkrzynią / gdy idąc trąbiono w trąby.
A ludowi przykázáł Jozue / mówiąc : Nie będziećie wołáć / áni będzie ſłyƺán głoſ wáƺ / áni wynijdzie z uſt wáƺych ſłowo / áż do dniá / którego wám rzekę : Wołájćie ; y ucżynićie okrzyk.
Tedy obeƺłá ſkrzyniá Páńſká miáſto w około ráz ; y wróćili śię do obozu / y zoſtáli w obozie przez noc.
Wſtáł záśię Jozue ráno / á kápłáni wzięli ſkrzynię Páńſką.
A śiedm kápłánów wziąwƺy śiedm trąb z rogów báránich / przed ſkrzynią Páńſką ƺli / idąc y trąbiąc w trąby ; á zbrojni ƺli przed nimi / oſtátek też ludu poſpolitego ƺedł zá ſkrzynią Páńſką / gdy idąc trąbiono w trąby.
A ták obeƺli miáſto drugi ráz dniá wtórego / y wróćili śię do obozu ; y ták cżynili po ƺeść dni.
Ale dniá śiódmego wſtáli ráno ná świtániu / y obeƺli miáſto tymże ſpoſobem śiedm kroć ; tylko dniá tego obeƺli miáſto śiedm kroć.
Y ſtáło śię / gdy śiódmy ráz obchodzili / á kápłáni trąbili w trąby / rzekł Jozue do ludu : Krzycżćież teráz ; ábowiem Pán podáł wám miáſto.
Y niech będzie to miáſto przeklęſtwem Pánu / ono / y wƺyſtko co w nim jeſt ; tylko Rácháb wƺetecżnicá żywo zoſtánie / oná y wƺyſcy / którzy z nią ſą w domu / gdyż utáiłá poſłów / któreśmy byli poſłáli.
A wƺákże śię wy ſtrzeżćie od rzecży przeklętych / ábyśćie śię nie ſtáli przeklęſtwem / biorąc co z rzecży przeklętych / ábyśćie nie wpráwili obozu Izráelſkiego w przeklęſtwo / y nie zámięƺáli go.
Ale wƺyſtko ſrebro y złoto y nácżyniá miedziáne y żelázne / święte będą Pánu ; do ſkárbu Páńſkiego złożone będą.
Tedy krzycżáł lud / gdy zátrąbiono w trąby / ábowiem gdy uſłyƺáł lud głoſ trąb / krzycżáł y lud wielkim głoſem / y upádł mur ná miejſcu ſwem / y wƺedł lud do miáſtá / káżdy przećiw miejſcu / gdzie ſtáł / y wzięli miáſto ;
Y wytráćili wƺyſtko / co było w mieśćie / męże y niewiáſty / dzieći y ſtárce ; woły też y owce / y oſły oſtrzem miecżá pobili.
Ale dwom mężom / którzy ƺpiegowáli onę ziemię / rzekł Jozue : Wnijdźćie do domu niewiáſty wƺetecżnej / á wywiedźćie ſtámtąd niewiáſtę / y wƺyſtko / co jej jeſt / jákośćie jej przyśięgli.
Tedy wƺedłƺy młodzieńcy oni / co byli wyƺpiegowáli ziemię / Ráchábę / y ojcá jej / mátkę jej y bráćią jej / y wƺyſtko co było jej / y wƺyſtkę rodzinę jej wywiedli / y zoſtáwili je zá obozem Izráelſkim.
Ale miáſto ſpálili ogniem / y wƺyſtko / co w nim było ; tylko ſrebro y złoto / y nácżynie miedziáne / y żelázne / złożyli do ſkárbu domu Páńſkiego.
Ráchábę tákże wƺetecżnicę / y dom ojcá jej / y wƺyſtko / co było jej / Jozue żywo zoſtáwił / y mieƺkáłá w pośrodku Izráelá áż do teráźniejƺego dniá / dlá tego / iż utáiłá poſłów / które był poſłáł Jozue ku przeƺpiegowániu Jerychá.
Y wydáł klątwę Jozue onego cżáſu / mówiąc : Przeklęty mąż przed Pánem / któryby powſtáł á budowáł to miáſto Jerycho ; ná pierworodnym ſwoim záłoży je / á ná nájmniejƺym poſtáwi brámy jego.
Y był Pán z Jozuem / á rozchodziłá śię ſłáwá jego po wƺyſtkiej ziemi.
