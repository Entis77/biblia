Y ſtáło śię po śmierći Mojżeƺá / ſługi Páńſkiego / że mówił Pán do Jozuego ſyná Nunowego / ſługi Mojżeƺowego / y rzekł :
Mojżeƺ / ſługá mój / umárł ; przetoż teraz wſtáń / przepráw śię przez ten Jordán / ty / y wƺyſtek lud ten / do źiemi / którą Ja im / ſynom Izráelſkim / dáwám.
Káżde miejſce / po którem deptáć będźie ſtopá nogi wáƺej / dáłem wám / jákom obiecáł Mojżeƺowi.
Od puƺcży y od Libánu tego / y áż do rzeki wielkiej / rzeki Eufráteſ / y wƺyſtká źiemiá Hetejcżyków / y áż do morzá wielkiego ná záchód ſłońcá / będźie gránicá wáƺá.
Nie oſtoi śię nikt przed tobą po wƺyſtkie dni żywotá twego ; jákom był z Mojżeƺem / ták będę z tobą / nie odſtąpię ćię / áni ćię opuƺcżę.
Zmácniájże śię y mężnie ſobie pocżynáj ; ábowiem ty podáƺ w dźiedźictwo ludowi temu źiemię / o którąm przyśiągł ojcom ich / że im ją dám.
Tylko śię zmácniáj / y bárdzo mężnie ſobie pocżynáj / ábyś ſtrzegł / y cżynił wƺyſtko według zakonu / któryć rozkázáł Mojżeƺ / ſługá mój ; nie uchyláj śię od niego áni ná práwo áni ná lewo / żebyś śię roztropnie ſpráwowáł we wƺyſtkiem / do cżego śię udáƺ.
Niech nie odſtępują kśięgi zakonu tego od uſt twoich ; ále rozmyśláj w nich we dnie y w nocy / ábyś ſtrzegł y cżynił wƺyſtko / co nápiſáno w nim ; ábowiem ná ten cżáſ poƺcżęśćią śię drogi twoje / y ná ten cżáſ roztropnym będźieƺ.
Azáżemći nie rozkázáł : Zmocnij śię / y mężnie ſobie pocżynáj / nie lękáj śię / á nie trwóż ſobą / ábowiem z tobą jeſt Pán / Bóg twój / we wƺyſtkiem / do cżegokolwiek śię obróćiƺ?
A przetoż rozkázáł Jozue przełożonym nád ludem / mówiąc :
Przejdźćie przez pośrodek obozu / á rozkáżćie ludowi / mówiąc : Gotujćie ſobie żywność ; ábowiem po trzech dniách przejdźiećie przez ten Jordán / ábyśćie weƺli / á pośiedli źiemię / którą Pán / Bóg wáƺ / dáwá wám w ośiádłość.
Rubenitom też / y Gádytom / y połowie pokoleniá Mánáſeſowego rzekł Jozue mówiąc :
Pámiętájćie ná ſłowo / które wám rozkázáł Mojżeƺ / ſługá Páńſki / mówiąc : Pán / Bóg wáƺ / ſpráwił wám odpocżynienie / y dáł wám tę źiemię ;
Żony wáƺe / dźiátki wáƺe / y bydłá wáƺe niech zoſtáną w źiemi / którą wám dáł Mojżeƺ z tey ſtrony Jordánu ; ále wy pójdźiećie zbrojni przed bráćią wáƺą / wƺyſcy duży w śile / á będźiećie ich pośiłkowáć /
Aż odpocżynienie ſpráwi Pán bráći wáƺym jáko y wám / á oni pośiędą źiemię / którą im dáwá Pán / Bóg wáƺ ; potem śię wróćićie do źiemi ośiádłośći wáƺej / á będźiećie ją trzymáć / którą wám dáł Mojżeƺ / ſługá Páńſki / z tey ſtrony Jordánu ná wſchód ſłońcá.
Y odpowiedźieli Jozuemu / mówiąc : Wƺyſtko / coś nám rozkázáł / ucżynimy / á gdźiekolwiek nas pośleƺ / pójdźiemy.
Jákośmy byli poſłuƺni Mojżeƺowi / ták poſłuƺni będźiemy y tobie ; tylko niech będźie Pán / Bóg twój / z tobą / jáko był z Mojżeƺem.
Ktobykolwiek przećiwił śię uſtom twoim / á nie byłby poſłuƺnym ſłowom twoim we wƺyſtkiem / co mu rozkáżeƺ / niecháj umrze ; tylko śię zmácniáj / á mężnie ſobie pocżynáj.
