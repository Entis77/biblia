Y ſtáło śię po śmierći Mojżeƺá / ſługi Páńſkiego / że mówił Pán do Jozuego ſyná Nunowego / ſługi Mojżeƺowego / y rzekł :
Mojżeƺ / ſługá mój / umárł ; przetoż teráz wſtáń / przepráw śię przez ten Jordán / ty / y wƺyſtek lud ten / do ziemi / którą Já im / ſynom Izráelſkim / dáwám.
Káżde miejſce / po którem deptáć będzie ſtopá nogi wáƺej / dáłem wám / jákom obiecáł Mojżeƺowi.
Od puƺcży y od Libánu tego / y áż do rzeki wielkiej / rzeki Eufráteſ / y wƺyſtká ziemiá Hetejcżyków / y áż do morzá wielkiego ná záchód ſłońcá / będzie gránicá wáƺá.
Nie oſtoi śię nikt przed tobą po wƺyſtkie dni żywotá twego ; jákom był z Mojżeƺem / ták będę z tobą / nie odſtąpię ćię / áni ćię opuƺcżę.
Zmácniájże śię y mężnie ſobie pocżynáj ; ábowiem ty podáƺ w dziedzictwo ludowi temu ziemię / o którąm przyśiągł ojcom ich / że im ją dám.
Tylko śię zmácniáj / y bárdzo mężnie ſobie pocżynáj / ábyś ſtrzegł / y cżynił wƺyſtko według zákonu / któryć rozkázáł Mojżeƺ / ſługá mój ; nie uchyláj śię od niego áni ná práwo áni ná lewo / żebyś śię roztropnie ſpráwowáł we wƺyſtkiem / do cżego śię udáƺ.
Niech nie odſtępują kśięgi zákonu tego od uſt twoich ; ále rozmyśláj w nich we dnie y w nocy / ábyś ſtrzegł y cżynił wƺyſtko / co nápiſáno w nim ; ábowiem ná ten cżáſ poƺcżęśćią śię drogi twoje / y ná ten cżáſ roztropnym będzieƺ.
Azáżemći nie rozkázáł : Zmocnij śię / y mężnie ſobie pocżynáj / nie lękáj śię / á nie trwóż ſobą / ábowiem z tobą jeſt Pán / Bóg twój / we wƺyſtkiem / do cżegokolwiek śię obróćiƺ?
A przetoż rozkázáł Jozue przełożonym nád ludem / mówiąc :
Przejdźćie przez pośrodek obozu / á rozkáżćie ludowi / mówiąc : Gotujćie ſobie żywność ; ábowiem po trzech dniách przejdziećie przez ten Jordán / ábyśćie weƺli / á pośiedli ziemię / którą Pán / Bóg wáƺ / dáwá wám w ośiádłość.
Rubenitom też / y Gádytom / y połowie pokoleniá Mánáſeſowego rzekł Jozue mówiąc :
Pámiętájćie ná ſłowo / które wám rozkázáł Mojżeƺ / ſługá Páńſki / mówiąc : Pán / Bóg wáƺ / ſpráwił wám odpocżynienie / y dáł wám tę ziemię ;
Żony wáƺe / dziátki wáƺe / y bydłá wáƺe niech zoſtáną w ziemi / którą wám dáł Mojżeƺ z tej ſtrony Jordánu ; ále wy pójdziećie zbrojni przed bráćią wáƺą / wƺyſcy duży w śile / á będziećie ich pośiłkowáć /
Aż odpocżynienie ſpráwi Pán bráći wáƺym jáko y wám / á oni pośiędą ziemię / którą im dáwá Pán / Bóg wáƺ ; potem śię wróćićie do ziemi ośiádłośći wáƺej / á będziećie ją trzymáć / którą wám dáł Mojżeƺ / ſługá Páńſki / z tej ſtrony Jordánu ná wſchód ſłońcá.
Y odpowiedzieli Jozuemu / mówiąc : Wƺyſtko / coś nám rozkázáł / ucżynimy / á gdziekolwiek náſ pośleƺ / pójdziemy.
Jákośmy byli poſłuƺni Mojżeƺowi / ták poſłuƺni będziemy y tobie ; tylko niech będzie Pán / Bóg twój / z tobą / jáko był z Mojżeƺem.
Ktobykolwiek przećiwił śię uſtom twoim / á nie byłby poſłuƺnym ſłowom twoim we wƺyſtkiem / co mu rozkáżeƺ / niecháj umrze ; tylko śię zmácniáj / á mężnie ſobie pocżynáj.
