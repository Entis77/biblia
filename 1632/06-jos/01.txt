Y ſtáło śię po śmierći Mojżeƺá / ſługi Páńſkiego / że mówił Pán do Jozuego ſyná Nunowego / ſługi Mojżeƺowego / y rzekł :
Mojzeƺ / ſługá mój / umárł ; przetoż teraz wſtáń / przepráw śię przez ten Jordán / ty / y wƺyſtek lud ten / do źiemi / którą Ja im / ſynom Izráelſkim / dáwám.
Káżde miejſce / po którem deptáć będźie ſtopá nogi wáƺej / dáłem wam / jákom obiecał Mojżeƺowi.
Od puƺcży y od Libánu tego / y áż do rzeki wielkiej / rzeki Eufrátes / y wƺyſtká źiemiá Hetejcżyków / y áż do morzá wielkiego ná záchód ſłońcá / będźie gránicá wáƺá.
Nie oſtoi śię nikt przed tobą po wƺyſtkie dni żywotá twego ; jákom był z Mojżeƺem / ták będę z tobą / nie odſtąpię ćię / áni ćię opuƺcżę.
Zmácniájże śię y mężnie ſobie pocżynáj ; ábowiem ty podáƺ w dźiedźictwo ludowi temu źiemię / o którąm przyśiągł ojcom ich / że im ją dam.
Tylko śię zmácniáj / y bárzo mężnie ſobie pocżynáj / ábyś ſtrzegł / y cżynił wƺyſtko według zakonu / któryć rozkázał Mojzeƺ / ſługá mój ; nie uchyláj śię od niego áni ná práwo áni ná lewo / żebyś śię roztropnie ſpráwował we wƺyſtkiem / do cżego śię udáƺ.
Niech nie odſtępują kśięgi zakonu tego od uſt twojich ; ále rozmyśláj w nich we dnie y w nocy / ábyś ſtrzegł y cżynił wƺyſtko / co nápiſano w nim ; ábowiem ná ten cżás poƺcżęśćią śię drogi twoje / y ná ten cżás roztropnym będźieƺ.
Azáżemći nie rozkázał : Zmocnij śię / y mężnie ſobie pocżynáj / nie lękáj śię / á nie trwóż ſobą / ábowiem z tobą jeſt Pán / Bóg twój / we wƺyſtkiem / do cżegokolwiek śię obróćiƺ?
A przetoż rozkázał Jozue przełożonym nád ludem / mówiąc :
Przejdźćie przez pośrodek obozu / á rozkáżćie ludowi / mówiąc : Gotujćie ſobie żywność ; ábowiem po trzech dniách przejdźiećie przez ten Jordán / ábyśćie weƺli / á pośiedli źiemię / którą Pán / Bóg wáƺ / dáwá wam w ośiádłość.
Rubenitom też / y Gádytom / y połowie pokolenia Mánáſeſowego rzekł Jozue mówiąc :
Pámiętájćie ná ſłowo / które wam rozkázał Mojzeƺ / ſługá Páńſki / mówiąc : Pán / Bóg wáƺ / ſpráwił wam odpocżynienie / y dał wam tę źiemię ;
Żony wáƺe / dźiátki wáƺe / y bydłá wáƺe niech zoſtáną w źiemi / którą wam dał Mojzeƺ z tey ſtrony Jordánu ; ále wy pójdźiećie zbrojni przed bráćią wáƺą / wƺyſcy duży w śile / á będźiećie ich pośiłkowáć /
Aż odpocżynienie ſpráwi Pán bráći wáƺym jáko y wam / á oni pośiędą źiemię / którą im dáwá Pán / Bóg wáƺ ; potem śię wróćićie do źiemi ośiádłośći wáƺej / á będźiećie ją trzymáć / którą wam dał Mojzeƺ / ſługá Páńſki / z tey ſtrony Jordánu ná wſchód ſłońcá.
Y odpowiedźieli Jozuemu / mówiąc : Wƺyſtko / coś nam rozkázał / ucżynimy / á gdźiekolwiek nas pośleƺ / pójdźiemy.
Jákośmy byli poſłuƺni Mojżeƺowi / ták poſłuƺni będźiemy y tobie ; tylko niech będźie Pán / Bóg twój / z tobą / jáko był z Mojżeƺem.
Ktobykolwiek przećiwił śię uſtom twoim / á nie byłby poſłuƺnym ſłowom twoim we wƺyſtkiem / co mu rozkáżeƺ / niecháj umrze ; tylko śię zmácniáj / á mężnie ſobie pocżynáj.
