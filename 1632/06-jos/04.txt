A gdy śię wƺyſtek lud przepráwił zá Jordán / ( bo rzekł był Pán do Jozuego / mówiąc :
Obierzćie ſobie z ludu dwánaśćie mężów / po jednym mężu z káżdego pokolenia.
Y rozkáżćie im / mówiąc : Weźmijćie ſobie ſtąd z pośrzodku Jordánu / z tego miejſcá / gdźie ſtáły nogi kápłanów porządnie / dwánaśćie kámieni / które z ſobą wynióſƺy poſtáwićie ná ſtánowiſku / gdźie będźiećie leżeć przez tę noc. )
Tedy wezwał Jozue dwánaśćie mężów / które był wybrał z ſynów Izráelſkich / po jednym mężu z káżdego pokolenia.
Y rzekł do nich Jozue : Idźćie przed ſkrzynią Páná / Bogá wáƺego / w pośrodek Jordánu / á weźmij káżdy kámień jeden ná rámię ſwoje według licżby pokolenia ſynów Izráelſkich /
Aby to było ná znák miedzy wámi / gdyby potem pytáli ſynowie wáśi mówiąc : Co wam znácży ten kámień?
Tedy im powiećie / iż śię rozſtąpiły wody w Jordánie przed ſkrzynią przymierza Páńſkiego ; ábowiem gdy ƺłá przez Jordán / rozſtąpiły śię wody Jordáńſkie ; y będźie ten kámień ná pámiątkę ſynom Izráelſkim áż ná wieki.
Y ucżynili ták ſynowie Izráelſcy / jáko rozkázał Jozue / y wźięli dwánaśćie kámieni z pośród Jordánu / jáko mówił Pán do Jozuego / według licżby pokolenia ſynów Izráelſkich / á zánieśli je z ſobą áż do ſtánowiſká / y tám je złożyli.
Jozue też wyſtáwił dwánaśćie kámieni w pośród Jordánu / ná miejſcu / kędy ſtáły nogi kápłanów / nioſących ſkrzynię przymierza / które tám zoſtáły áż po dźiś dźień.
A ták kápłáni nioſący ſkrzynię ſtáli w pośród Jordánu / áż śię wypełniło to wƺyſtko / co był rozkázał Pán Jozuemu mówić do ludu według wƺyſtkiego / co był przykázał Mojzeƺ Jozuemu. Spieƺył śię tedy lud y przeƺedł Jordán.
Y ſtáło śię gdy wƺyſtek lud przeƺedł / że też przeƺłá y ſkrzyniá Páńſka y kápłáni przed oblicżnośćią ludu.
Przeƺli też ſynowie Rubenowi / y ſynowie Gádowi / z połową pokolenia Mánáſeſowego / zbrojno przed ſyny Izráelſkimi / jáko im był powiedźiał Mojzeƺ.
Około cżterdźieſtu tyśięcy ludu zbrojnego przeƺło przed Pánem do boju ná polu Jerychá.
Dniá onego wywyżƺył Pán Jozuego przed ocżymá wƺyſtkiego Izráelá / y báli śię go / jáko śię báli Mojżeƺá po wƺyſtkie dni żywotá jego.
Potem rzekł Pán do Jozuego / mówiąc :
Rozkáż kápłánom / nioſącym ſkrzynię świádectwá / áby wyſtąpili z Jordánu.
Y rozkázał Jozue kápłánom / mówiąc : Wyſtąpćie z Jordánu.
Y ſtáło śię / gdy wyſtąpili kápłáni / nioſący ſkrzynię przymierza Páńſkiego / z pośrzodku Jordánu / á ſtánęły ſtopy nóg kápłanów ná ſuƺy / wróćiły śię wody Jordáńſkie ná miejſce ſwoje / á płynęły / jáko przed tym / we wƺyſtkich brzegách ſwojich.
A lud / wyƺedƺy z Jordánu dźieśiątego dniá mieśiącá pierwƺego / położyli śię obozem w Gálgál ku ſtronie wſchodniej Jerychá.
A dwánaśćie onych kámieni / które wynieśli z Jordánu / poſtáwił Jozue w Gálgál.
Y rzekł do ſynów Izráelſkich / mówiąc : Gdy ſpytáją ſynowie wáśi potem ojców ſwojich / mówiąc : Co znácży ten kámień?
Tedy oznájmijćie ſynom wáƺym / mówiąc : Po ſuƺy przeƺedł Izráel ten Jordán ;
Abowiem oſuƺył Pán Bóg wody Jordáńſkie przed wámi / áżeśćie przeƺli / jáko ucżynił Pán / Bóg wáƺ / morzu cżerwonemu / które wyſuƺył przed námi / áżeśmy przeƺli ;
Aby poználi wƺyſcy národowie źiemi rękę Páńſką / że można jeſt / żebyśćie śię báli Páná / Bogá wáƺego / po wƺyſtkie dni.
