A gdy śię wƺyſtek lud przepráwił zá Jordán / ( bo rzekł był Pán do Jozuego / mówiąc :
Obierzćie ſobie z ludu dwánáśćie mężów / po jednym mężu z káżdego pokoleniá.
Y rozkáżćie im / mówiąc : Weźmijćie ſobie ſtąd z pośrodku Jordánu / z tego miejſcá / gdzie ſtáły nogi kápłánów porządnie / dwánáśćie kámieni / które z ſobą wynióſłƺy poſtáwićie ná ſtánowiſku / gdzie będziećie leżeć przez tę noc. )
Tedy wezwáł Jozue dwánáśćie mężów / które był wybráł z ſynów Izráelſkich / po jednym mężu z káżdego pokoleniá.
Y rzekł do nich Jozue : Idźćie przed ſkrzynią Páná / Bogá wáƺego / w pośrodek Jordánu / á weźmij káżdy kámień jeden ná rámię ſwoje według licżby pokoleniá ſynów Izráelſkich /
Aby to było ná znák między wámi / gdyby potem pytáli ſynowie wáśi mówiąc : Co wám znácży ten kámień?
Tedy im powiećie / iż śię rozſtąpiły wody w Jordánie przed ſkrzynią przymierzá Páńſkiego ; ábowiem gdy ƺłá przez Jordán / rozſtąpiły śię wody Jordáńſkie ; y będzie ten kámień ná pámiątkę ſynom Izráelſkim áż ná wieki.
Y ucżynili ták ſynowie Izráelſcy / jáko rozkázáł Jozue / y wzięli dwánáśćie kámieni z pośród Jordánu / jáko mówił Pán do Jozuego / według licżby pokoleniá ſynów Izráelſkich / á zánieśli je z ſobą áż do ſtánowiſká / y tám je złożyli.
Jozue też wyſtáwił dwánáśćie kámieni w pośród Jordánu / ná miejſcu / kędy ſtáły nogi kápłánów / nioſących ſkrzynię przymierzá / które tám zoſtáły áż po dziś dzień.
A ták kápłáni nioſący ſkrzynię ſtáli w pośród Jordánu / áż śię wypełniło to wƺyſtko / co był rozkázáł Pán Jozuemu mówić do ludu według wƺyſtkiego / co był przykázáł Mojżeƺ Jozuemu. Spieƺył śię tedy lud y przeƺedł Jordán.
Y ſtáło śię gdy wƺyſtek lud przeƺedł / że też przeƺłá y ſkrzyniá Páńſká y kápłáni przed oblicżnośćią ludu.
Przeƺli też ſynowie Rubenowi / y ſynowie Gádowi / z połową pokoleniá Mánáſeſowego / zbrojno przed ſyny Izráelſkimi / jáko im był powiedziáł Mojżeƺ.
Około cżterdzieſtu tyśięcy ludu zbrojnego przeƺło przed Pánem do boju ná polu Jerychá.
Dniá onego wywyżƺył Pán Jozuego przed ocżymá wƺyſtkiego Izráelá / y báli śię go / jáko śię báli Mojżeƺá po wƺyſtkie dni żywotá jego.
Potem rzekł Pán do Jozuego / mówiąc :
Rozkáż kápłánom / nioſącym ſkrzynię świádectwá / áby wyſtąpili z Jordánu.
Y rozkázáł Jozue kápłánom / mówiąc : Wyſtąpćie z Jordánu.
Y ſtáło śię / gdy wyſtąpili kápłáni / nioſący ſkrzynię przymierzá Páńſkiego / z pośrodku Jordánu / á ſtánęły ſtopy nóg kápłánów ná ſuƺy / wróćiły śię wody Jordáńſkie ná miejſce ſwoje / á płynęły / jáko przedtem / we wƺyſtkich brzegách ſwoich.
A lud / wyƺedłƺy z Jordánu dzieśiątego dniá mieśiącá pierwƺego / położyli śię obozem w Gálgál ku ſtronie wſchodniej Jerychá.
A dwánáśćie onych kámieni / które wynieśli z Jordánu / poſtáwił Jozue w Gálgál.
Y rzekł do ſynów Izráelſkich / mówiąc : Gdy ſpytáją ſynowie wáśi potem ojców ſwoich / mówiąc : Co znácży ten kámień?
Tedy oznájmijćie ſynom wáƺym / mówiąc : Po ſuƺy przeƺedł Izráel ten Jordán ;
Abowiem oſuƺył Pán Bóg wody Jordáńſkie przed wámi / áżeśćie przeƺli / jáko ucżynił Pán / Bóg wáƺ / morzu cżerwonemu / które wyſuƺył przed námi / áżeśmy przeƺli ;
Aby poználi wƺyſcy národowie ziemi rękę Páńſką / że możná jeſt / żebyśćie śię báli Páná / Bogá wáƺego / po wƺyſtkie dni.
