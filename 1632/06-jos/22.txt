Tedy przyzwáł Jozue Rubenitów y Gádytów / y połowę pokoleniá Mánáſeſowego /
Y rzekł do nich : Wyśćie ſtrzegli wƺyſtkiego / co wám rozkázáł Mojżeƺ / ſługá Páńſki / y byliśćie poſłuƺni głoſowi memu we wƺyſtkiem / com wám rozkázáł.
Nie opuśćiliśćie bráći wáƺej przez długi cżáſ áż do dniá tego ; áleśćie ſtrzegli pilnie rozkázániá Páná / Bogá wáƺego.
A teráz / poniewáż odpocżynek dáł Pán / Bóg wáƺ / bráćiom wáƺym / jáko im był obiecáł / przetoż teráz wróććie śię / á idźćie do przybytków wáƺych y do ziemi ośiádłośći wáƺej / którą wám dáł Mojżeƺ ſługá Páńſki / przed Jordánem.
Tylko ſtrzeżćie pilnie / ábyśćie záchowáli przykázánie / y Zákon / który wám rozkázáł Mojżeƺ / ſługá Páńſki : ábyśćie miłowáli Páná Bogá wáƺego / á chodzili wƺyſtkiemi drogámi jego / chowájąc rozkázániá jego / dzierżąc śię go y ſłużąc mu ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej.
Y błogoſłáwił im Jozue / á rozpuśćił je ; y odeƺli do przybytków ſwoich.
Ale połowie pokoleniá Mánáſeſowego dáł był Mojżeƺ ośiádłość w Báſán / á drugiej połowie jego dáł Jozue dziáł z bráćią ich z tej ſtrony Jordánu ná záchód ſłońcá ; á gdy je rozpuƺcżáł Jozue do przybytku ich / błogoſłáwił im.
Y rzekł do nich / mówiąc : Z wielkiemi bogáctwy wrácáćie śię do przybytków wáƺych / y z májętnośćią bárdzo wielką / z ſrebrem y z złotem / y z miedzią / y z żelázem / y ƺát bárdzo wielą ; dzielćież śię łupem nieprzyjáćiół wáƺych z bráćią ſwoją.
Tedy wrácájąc śię odeƺli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokoleniá Mánáſeſowego od ſynów Izráelſkich z Sylo / które jeſt w ziemi Chánánejſkiej / áby ƺli do ziemi Gáláád / do ziemi ośiádłośći ſwojej / którą dziedzicżnie otrzymáli według ſłowá Páńſkiego przez Mojżeƺá.
Y przyƺli do gránic Jordánu / które były w ziemi Chánánejſkiej / y zbudowáli tám ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokoleniá Mánáſeſowego ołtárz nád Jordánem / ołtárz wielki ná podziw.
Y uſłyƺeli ſynowie Izráelſcy / iż powiádáno : Oto zbudowáli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokoleniá Mánáſeſowego ołtárz przećiw ziemi Chánánejſkiej ná gránicách nád Jordánem / kędy przeƺli ſynowie Izráelſcy.
To gdy uſłyƺeli ſynowie Izráelſcy / zeƺło śię wƺyſtko zgromádzenie ich do Sylo / áby śię ruƺyli przećiwko nim ná wojnę.
Y poſłáli ſynowie Izráelſcy do ſynów Rubenowych / y do ſynów Gádowych / y do połowy pokoleniá Mánáſeſowego / do ziemi Gáláád / Fineeſá / ſyná Eleázárá kápłáná.
A z nim dzieśięć kśiążąt / po jednym kśiążęćiu z káżdego domu ojcowſkiego ze wƺyſtkich pokoleń Izráelſkich / á káżdy kśiążę z tych był przedniejƺym w domu ojców ſwoich / w tyśiącách Izráelſkich.
Tedy ći przyƺli do ſynów Rubenowych / y do ſynów Gádowych / y do połowy pokoleniá Mánáſeſowego / do ziemi Gáláád / y rzekli do nich / mówiąc :
Ták mówi wƺyſtko zgromádzenie Páńſkie : Cóż to jeſt zá przeſtępſtwo / któremeśćie wyſtąpili przećiwko Bogu Izráelſkiemu / żeśćie śię dziś odwróćili od Páná / budując ſobie ołtárz / ábyśćie dziś byli przećiwnymi Pánu?
Azáż nám máło ná złośći Fegorowej / od której nie jeſteśmy ocżyƺcżeni y po dziś dzień / ſkąd byłá pomſtá w zgromádzeniu Páńſkim /
Żeśćie śię dziś odwróćili / żebyśćie nie ƺli zá Pánem? Zácżem ſtánie śię / poniewáżeśćie wy dziś odpornymi Pánu / że śię on jutro ná wƺyſtko zgromádzenie Izráelſkie rozgniewá.
A jeźliż jeſt niecżyſtá ziemiá ośiádłośći wáƺej / przeprowádźćie śię do ziemi dziedzictwá Páńſkiego / w której przebywá przybytek Páńſki / y weźmiećie ośiádłośći w pośrodku náſ ; tylko Pánu nie bądźćie odpornymi / á nie odpádájćie od náſ / budując ſobie ołtárz oprócż ołtárzá Páná / Bogá náƺego.
Azáż przez Acháná / ſyná Zárego / gdy śię dopuśćił przeſtępſtwá w rzecży przeklętej / ná wƺyſtko zgromádzenie Izráelſkie nie przypádł gniew? á nie on ſám jeden umárł dlá niepráwośći ſwojej.
Tedy odpowiedzieli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokoleniá Mánáſeſowego / á mówili do kśiążąt tyśiąców Izráelſkich :
Bóg nád Bogi / Pán / Bóg nád Bogi / Pán / on to wie / y Izráel ſám pozná / jeźli śię to ſtáło z uporu / álbo jeźli z przeſtępſtwá przećiw Pánu / niechże náſ nie żywi dniá tego.
Jeźliżeśmy ſobie zbudowáli ołtárz / ábyśmy śię odwróćili od Páná / á jeźliż ku ofiárowániu ná nim cáłopáleniá / y ofiár śniednych / álbo ku ſpráwowániu ná nim ofiár ſpokojnych / Pán niech to rozezná ;
Jeźliżeśmy nie rácżej obáwiájąc śię tej rzecży / ucżynili to / mówiąc : Nápotem rzeką ſynowie wáśi ſynom náƺym / mówiąc : Cóż wám do Páná / Bogá Izráelſkiego?
Oto gránicę położył Pán między námi y między wámi / ſynowie Rubenowi y ſynowie Gádowi / Jordán ; nie máćie wy dziáłu w Pánu / y odwrócą ſynowie wáśi ſyny náƺe od bojáźni Páńſkiej.
Przetośmy rzekli : Ucżyńmy ták / á zbudujmy ſobie ołtárz / nie dlá cáłopáleniá / áni innych ofiár :
Ale iżby był świádkiem między námi y między wámi / y między potomſtwy náƺemi po náſ / y ábyśmy ſłużyli Pánu przed oblicżem jego w cáłopáleniách náƺych / y w śniednych ofiárách náƺych / y w ſpokojnych ofiárách náƺych / á iżby nie rzekli ſynowie wáśi nápotem ſynom náƺym : Nie máćie cżąſtki w Pánu.
Nádtośmy rzekli : Gdyby nápotem rzekli nám / álbo potomſtwu náƺemu / tedy rzecżemy : Pátrzájćie ná podobieńſtwo ołtárzá Páńſkiego / który ucżynili ojcowie náśi / nie dlá cáłopáleniá / áni innych ofiár / ále żeby on był świádkiem między námi y między wámi.
Boże náſ uchowáj / żebyśmy mieli przećiwnymi być Pánu / á odſtąpić dziś od Páná / zbudowáwƺy ołtárz dlá cáłopálonych / dlá śniednych y dlá innych ofiár / oprócż ołtárzá Páná / Bogá náƺego / który jeſt przed przybytkiem jego.
A uſłyƺáwƺy Fineeſ kápłán / y kśiążętá zgromádzeniá / y przełożeni nád tyśiącmi Izráelſkimi / którzy z nim byli / ſłowá / które mówili ſynowie Rubenowi / y ſynowie Gádowi / y ſynowie Mánáſeſowi / podobáło śię im to.
Y rzekł Fineeſ / ſyn Eleázárá kápłáná / do ſynów Rubenowych y do ſynów Gádowych y do ſynów Mánáſeſowych : Dziśiájśmy poználi / iż w pośrodku náſ jeſt Pán / iżeśćie śię nie dopuśćili przećiw Pánu przeſtępſtwá tego / y wyſwobodziliśćie ſyny Izráelſkie z ręki Páńſkiej.
A ták wróćili śię Fineeſ / ſyn Eleázárá kápłáná / z onymi kśiążęty od ſynów Rubenowych y do ſynów Gádowych z ziemi Gáláád do ziemi Chánánejſkiej do ſynów Izráelſkich / y odnieśli im tę rzecż.
Y podobáło śię to ſynom Izráelſkim ; á błogoſłáwili Bogá ſynowie Izráelſcy / y nie mówili więcej / żeby mieli iść przećiwko nim ná wojnę / y wytráćić ziemię / w której ſynowie Rubenowi y ſynowie Gádowi mieƺkáli.
Przezwáli tedy ſynowie Rubenowi y ſynowie Gádowi ołtárz on Ed / mówiąc : Świádkiem będzie między námi / że Pán jeſt Bogiem.
