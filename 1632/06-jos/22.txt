Tedy przyzwał Jozue Rubenitów y Gádytów / y połowę pokolenia Mánáſeſowego /
Y rzekł do nich : Wyśćie ſtrzegli wƺyſtkiego / co wam rozkázał Mojzeƺ / ſługá Páńſki / y byliśćie poſłuƺni głoſowi memu we wƺyſtkim / com wam rozkázał.
Nie opuśćiliśćie bráći wáƺej przez długi cżás áż do dniá tego ; áleśćie ſtrzegli pilnie rozkázánia Páná / Bogá wáƺego.
A teraz / ponieważ odpocżynek dał Pán / Bóg wáƺ / bráćiom wáƺym / jáko im był obiecał / przetoż teraz wróććie śię / á idźćie do przybytków wáƺych y do źiemi ośiádłośći wáƺej / którą wam dał Mojzeƺ ſługá Páńſki / przed Jordánem.
Tylko ſtrzeżćie pilnie / ábyśćie záchowáli przykazánie / y Zakon / który wam rozkázał Mojzeƺ / ſługá Páńſki : ábyśćie miłowáli Páná Bogá wáƺego / á chodźili wƺyſtkiemi drogámi jego / chowájąc rozkázánia jego / dźierżąc śię go y ſłużąc mu ze wƺyſtkiego ſercá wáƺego / y ze wƺyſtkiej duƺy wáƺej.
Y błogoſłáwił im Jozue / á rozpuśćił je ; y odeƺli do przybytków ſwojich.
Ale połowie pokolenia Mánáſeſowego dał był Mojzeƺ ośiádłość w Báſán / á drugiej połowie jego dał Jozue dźiał z bráćią ich z tey ſtrony Jordánu ná záchód ſłońcá ; á gdy je rozpuƺcżał Jozue do przybytku ich / błogoſłáwił im.
Y rzekł do nich / mówiąc : Z wielkiemi bogáctwy wrácáćie śię do przybytków wáƺych / y z májętnośćią bárzo wielką / z ſrebrem y z złotem / y z miedźią / y z żelázem / y ƺát bárzo wielą ; dźielćież śię łupem nieprzyjáćiół wáƺych z bráćią ſwoją.
Tedy wrácájąc śię odeƺli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokolenia Mánáſeſowego od ſynów Izráelſkich z Sylo / które jeſt w źiemi Chánánejſkiej / áby ƺli do źiemi Gáláád / do źiemi ośiádłośći ſwojey / którą dźiedźicżnie otrzymáli według ſłowá Páńſkiego przez Mojżeƺá.
Y przyƺli do gránic Jordánu / które były w źiemi Chánánejſkiej / y zbudowáli tám ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokolenia Mánáſeſowego ołtárz nád Jordánem / ołtárz wielki ná podźiw.
Y uſłyƺeli ſynowie Izráelſcy / iż powiádáno : Oto zbudowáli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokolenia Mánáſeſowego ołtárz przećiw źiemi Chánánejſkiej ná gránicách nád Jordánem / kędy przeƺli ſynowie Izráelſcy.
To gdy uſłyƺeli ſynowie Izráelſcy / zeƺło śię wƺyſtko zgromádzenie ich do Sylo / áby śię ruƺyli przećiwko nim ná wojnę.
Y poſłáli ſynowie Izráelſcy do ſynów Rubenowych / y do ſynów Gádowych / y do połowy pokolenia Mánáſeſowego / do źiemi Gáláád / Fineeſá / ſyná Eleázárá kápłáná.
A z nim dźieśięć kśiążąt / po jednym kśiążęćiu z káżdego domu ojcowſkiego ze wƺyſtkich pokoleń Izráelſkich / á káżdy kśiążę z tych był przedniejƺym w domu ojców ſwojich / w tyśiącách Izráelſkich.
Tedy ći przyƺli do ſynów Rubenowych / y do ſynów Gádowych / y do połowy pokolenia Mánáſeſowego / do źiemi Gáláád / y rzekli do nich / mówiąc :
Ták mówi wƺyſtko zgromádzenie Páńſkie : Cóż to jeſt zá przeſtępſtwo / któremeśćie wyſtąpili przećiwko Bogu Izráelſkiemu / żeśćie śię dźiś odwróćili od Páná / budując ſobie ołtárz / ábyśćie dźiś byli przećiwnymi Pánu?
Azaż nam máło ná złośći Fegorowej / od którey nie jeſteśmy ocżyƺcżeni y po dźiś dźień / ſkąd byłá pomſtá w zgromádzeniu Páńſkim /
Żeśćie śię dźiś odwróćili / żebyśćie nie ƺli zá Pánem? Zácżem ſtánie śię / poniewáżeśćie wy dźiś odpornymi Pánu / że śię on jutro ná wƺyſtko zgromádzenie Izráelſkie rozgniewá.
A jeſliż jeſt niecżyſtá źiemiá ośiádłośći wáƺej / przeprowádźćie śię do źiemi dźiedźictwá Páńſkiego / w którey przebywá przybytek Páńſki / y weźmiećie ośiádłośći w pośrzodku nas ; tylko Pánu nie bądźćie odpornymi / á nie odpádájćie od nas / budując ſobie ołtárz oprócż ołtárzá Páná / Bogá náƺego.
Azaż przez Acháná / ſyná Zárego / gdy śię dopuśćił przeſtępſtwá w rzecży przeklętey / ná wƺyſtko zgromádzenie Izráelſkie nie przypádł gniew? á nie on ſám jeden umárł dla niepráwośći ſwojey.
Tedy odpowiedźieli ſynowie Rubenowi / y ſynowie Gádowi / y połowá pokolenia Mánáſeſowego / á mówili do kśiążąt tyśiąców Izráelſkich :
Bóg nád Bogi / Pán / Bóg nád Bogi / Pán / on to wie / y Izráel ſám pozná / jeſli śię to ſtáło z uporu / álbo jeſli z przeſtępſtwá przećiw Pánu / niechże nas nie żywi dniá tego.
Jeſliżeśmy ſobie zbudowáli ołtárz / ábyſmy śię odwróćili od Páná / á jeſliż ku ofiárowániu ná nim cáłopalenia / y ofiár śniednych / álbo ku ſpráwowániu ná nim ofiár ſpokojnych / Pán niech to rozezná ;
Jeſliżeśmy nie rácżey obáwiájąc śię tey rzecży / ucżynili to / mówiąc : Nápotem rzeką ſynowie wáśi ſynom náƺym / mówiąc : Cóż wam do Páná / Bogá Izráelſkiego?
Oto gránicę położył Pán miedzy námi y miedzy wámi / ſynowie Rubenowi y ſynowie Gádowi / Jordán ; nie máćie wy dźiáłu w Pánu / y odwrócą ſynowie wáśi ſyny náƺe od bojáźni Páńſkiej.
Przetośmy rzekli : Ucżyńmy ták / á zbudujmy ſobie ołtárz / nie dla cáłopalenia / áni innych ofiár :
Ale iżby był świádkiem miedzy námi y miedzy wámi / y miedzy potomſtwy náƺemi po nas / y ábyſmy ſłużyli Pánu przed oblicżem jego w cáłopáleniách náƺych / y w śniednych ofiárách náƺych / y w ſpokojnych ofiárách náƺych / á iżby nie rzekli ſynowie wáśi nápotem ſynom náƺym : Nie máćie cżąſtki w Pánu.
Nádtośmy rzekli : Gdyby nápotem rzekli nam / álbo potomſtwu náƺemu / tedy rzecżemy : Pátrzájćie ná podobieńſtwo ołtárzá Páńſkiego / który ucżynili ojcowie náśi / nie dla cáłopalenia / áni innych ofiár / ále żeby on był świádkiem miedzy námi y miedzy wámi.
Boże nas uchowáj / żebyſmy mieli przećiwnymi być Pánu / á odſtąpić dźiś od Páná / zbudowáwƺy ołtárz dla cáłopálonych / dla śniednych y dla innych ofiár / oprócż ołtárzá Páná / Bogá náƺego / który jeſt przed przybytkiem jego.
A uſłyƺáwƺy Finees kápłán / y kśiążętá zgromádzenia / y przełożeni nád tyśiącmi Izráelſkimi / którzy z nim byli / ſłowá / które mówili ſynowie Rubenowi / y ſynowie Gádowi / y ſynowie Mánáſeſowi / podobáło śię im to.
Y rzekł Finees / ſyn Eleázárá kápłáná / do ſynów Rubenowych y do ſynów Gádowych y do ſynów Mánáſeſowych : Dźiśiájśmy poználi / iż w pośrzodku nas jeſt Pán / iżeśćie śię nie dopuśćili przećiw Pánu przeſtępſtwá tego / y wyſwobodźiliśćie ſyny Izráelſkie z ręki Páńſkiej.
A ták wróćili śię Finees / ſyn Eleázárá kápłáná / z onymi kśiążęty od ſynów Rubenowych y do ſynów Gádowych z źiemi Gáláád do źiemi Chánánejſkiej do ſynów Izráelſkich / y odnieśli im tę rzecż.
Y podobáło śię to ſynom Izráelſkim ; á błogoſłáwili Bogá ſynowie Izráelſcy / y nie mówili więcey / żeby mieli iść przećiwko nim ná wojnę / y wytráćić źiemię / w którey ſynowie Rubenowi y ſynowie Gádowi mieƺkáli.
Przezwáli tedy ſynowie Rubenowi y ſynowie Gádowi ołtárz on Ed / mówiąc : Świádkiem będźie miedzy námi / że Pán jeſt Bogiem.
