A ták poſłał Jozue / ſyn Nunów / z Syttim dwu ƺpiegów potájemnie / mówiąc : Idźćie / wypátrujćie źiemię / y Jerycho. Szli tedy y weƺli do niektórey niewiáſty wƺetecżney / którey imię Rácháb / y odpocżęli tám.
Y powiedźiáno to królowi Jerychá / mówiąc : Oto / mężowie jácyś przyƺli tu tey nocy z ſynów Izráelſkich / áby przeƺpiegowáli tę źiemię.
Tedy poſłał król Jerychá do Rácháby / mówiąc : Wywiedź męże / którzy przyƺli do ćiebie / á weƺli do domu twego ; bo ná przeƺpiegowánie wƺyſtkiej źiemi przyƺli.
Ale wźiąwƺy oná niewiáſtá tych dwu mężów / ſkryłá je / y rzekłá : Práwdáć jeſt / przyƺli do mnie mężowie ; álem nie wiedźiáłá / ſkąd byli.
A gdy bramę zámykáno w zmierzch / oni mężowie wyƺli ; y nie wiem / dokąd poƺli ; gońćież ich co nájrychlej / bo ich dośćigniećie.
A oná wwiodłá je byłá ná dách / y tám je przykryłá lnem nietártym / który byłá rozſtáwiłá ná dáchu.
Mężowie tedy wyſłáni gonili je drogą ku Jordánu áż do brodu ; á bramę zámkniono / ſkoro wyƺli ći / którzy ƺli zá nimi w pogoń.
A ták pierwej niż poſnęli / oná wſtąpiłá do nich ná dách ;
Y rzekłá do onych mężów : Wiem / że wam dał Pán źiemię tę ; bo ſtrách wáƺ przypádł ná nas / y oſłábiáli wƺyſcy obywátele tey źiemi przed wámi.
Bośmy ſłyƺeli / jáko wyſuƺył Pán wody morzá cżerwonego przed wámi / gdyśćie wychodźili z Egiptu / y cośćie ucżynili dwom królom Amorejſkim / którzy byli z oney ſtrony Jordánu / Sehonowi / y Ogowi / któreśćie pobili.
Co gdyśmy uſłyƺeli / upádło ſerce náƺe y nie oſtał śię więcey duch w nikim przed wámi ; ábowiem Pán / Bóg wáƺ / jeſt Bogiem ná niebie wzgórę / y ná źiemi niſko.
Przetoż teraz przyśiężćie mi proƺę przez Páná / iż jákom ja ucżyniłá z wámi miłośierdźie / tákże ucżyńćie y wy z domem ojcá mego miłośierdźie / á dájćie mi znák pewny /
Iż záchowáćie żywo ojcá mego y mátkę moję / y bráćią moję / y śioſtry moje / y wƺyſtko / co ich jeſt / á wybáwićie duƺe náƺe od śmierći.
Y odpowiedźieli jey oni mężowie : Duƺá náƺá będźie zá was ná śmierć / jeſli nie wydáćie tey ſpráwy náƺej / y będźie to / gdy nam podá Pán tę źiemię / że ucżynimy z tobą miłośierdźie y práwdę.
Y ſpuśćiłá je ná powroźie z okná ; bo dom jey był przy murze y oná ná murze mieƺkáłá.
Y rzekłá im : Ná górę idźćie / by śię ſnadź nie ſpotkáli z wámi / którzy was gonią : y tám śię kryjćie przez trzy dni / áż śię wrócą / którzy was gonią / á potem pójdźiećie drogą wáƺą.
Y rzekli jey mężowie oni : Będźiemy wolni od przyśięgi tey / którąś nas poprzyśięgłá /
Jeſli / gdy wnijdźiemy do źiemi / tego cżerwonego ƺnuru nie uwiążeƺ u okná / po którymeś nas ſpuśćiłá / á ojcá twego / y mátki twojey / y bráći twojey / y wƺyſtkiego domu ojcá twego nie zbierzeƺli do śiebie w dom ;
Abowiem ktobykolwiek wyƺedł ze drzwi domu twego / krew jego będźie ná głowę jego / á my będźiemy bez winy ; ále káżdego / ktokolwiek będźie z tobą w domu / krew jego obróći śię ná głowę náƺę / jeſli śię go kto ręką dotknie.
Lecż jeſli wydáƺ tę ſpráwę náƺę / tedy będźiemy wolni od przyśięgi twojey / którąś nas poprzyśięgłá.
Y odpowiedźiáłá ; Jákośćie powiedźieli / niechże ták będźie. Tedy je wypuśćiłá / y poƺli ; y uwiązáłá ƺnur cżerwony w onem oknie.
A odƺedƺy przyƺli ná górę / y zoſtáli tám przez trzy dni / áż śię wróćili / którzy je gonili ; bo ich ƺukáli ći / którzy je gonili / po wƺyſtkich drogách / ále nie ználeſli.
Y wróćili śię oni dwáj mężowie / á zſtąpiwƺy z góry / przepráwili śię / y przyƺli do Jozuego / ſyná Nunowego / y powiedźieli mu wƺyſtko / co śię z nimi dźiáło ;
Y mówili do Jozuego : Dał Pán w ręce náƺe tę wƺyſtkę źiemię ; bo śię ſtrwożyli wƺyſcy obywátele źiemi przed twárzą náƺą.
