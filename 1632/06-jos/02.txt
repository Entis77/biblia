A ták poſłáł Jozue / ſyn Nunów / z Syttim dwóch ƺpiegów potájemnie / mówiąc : Idźćie / wypátrujćie ziemię / y Jerycho. Szli tedy y weƺli do niektórej niewiáſty wƺetecżnej / której imię Rácháb / y odpocżęli tám.
Y powiedziáno to królowi Jerychá / mówiąc : Oto / mężowie jácyś przyƺli tu tej nocy z ſynów Izráelſkich / áby przeƺpiegowáli tę ziemię.
Tedy poſłáł król Jerychá do Rácháby / mówiąc : Wywiedź męże / którzy przyƺli do ćiebie / á weƺli do domu twego ; bo ná przeƺpiegowánie wƺyſtkiej ziemi przyƺli.
Ale wziąwƺy oná niewiáſtá tych dwóch mężów / ſkryłá je / y rzekłá : Práwdáć jeſt / przyƺli do mnie mężowie ; álem nie wiedziáłá / ſkąd byli.
A gdy brámę zámykáno w zmierzch / oni mężowie wyƺli ; y nie wiem / dokąd poƺli ; gońćież ich co nájrychlej / bo ich dośćigniećie.
A oná wwiodłá je byłá ná dách / y tám je przykryłá lnem nietártym / który byłá rozſtáwiłá ná dáchu.
Mężowie tedy wyſłáni gonili je drogą ku Jordánu áż do brodu ; á brámę zámkniono / ſkoro wyƺli ći / którzy ƺli zá nimi w pogoń.
A ták pierwej niż poſnęli / oná wſtąpiłá do nich ná dách ;
Y rzekłá do onych mężów : Wiem / że wám dáł Pán ziemię tę ; bo ſtrách wáƺ przypádł ná náſ / y oſłábiáli wƺyſcy obywátele tej ziemi przed wámi.
Bośmy ſłyƺeli / jáko wyſuƺył Pán wody morzá cżerwonego przed wámi / gdyśćie wychodzili z Egiptu / y cośćie ucżynili dwom królom Amorejſkim / którzy byli z onej ſtrony Jordánu / Sehonowi / y Ogowi / któreśćie pobili.
Co gdyśmy uſłyƺeli / upádło ſerce náƺe y nie oſtáł śię więcej duch w nikim przed wámi ; ábowiem Pán / Bóg wáƺ / jeſt Bogiem ná niebie wzgórę / y ná ziemi niſko.
Przetoż teráz przyśiężćie mi proƺę przez Páná / iż jákom já ucżyniłá z wámi miłośierdzie / tákże ucżyńćie y wy z domem ojcá mego miłośierdzie / á dájćie mi znák pewny /
Iż záchowáćie żywo ojcá mego y mátkę moję / y bráćią moję / y śioſtry moje / y wƺyſtko / co ich jeſt / á wybáwićie duƺe náƺe od śmierći.
Y odpowiedzieli jej oni mężowie : Duƺá náƺá będzie zá wáſ ná śmierć / jeźli nie wydáćie tej ſpráwy náƺej / y będzie to / gdy nám podá Pán tę ziemię / że ucżynimy z tobą miłośierdzie y práwdę.
Y ſpuśćiłá je ná powrozie z okná ; bo dom jej był przy murze y oná ná murze mieƺkáłá.
Y rzekłá im : Ná górę idźćie / by śię ſnáć nie ſpotkáli z wámi / którzy wáſ gonią : y tám śię kryjćie przez trzy dni / áż śię wrócą / którzy wáſ gonią / á potem pójdziećie drogą wáƺą.
Y rzekli jej mężowie oni : Będziemy wolni od przyśięgi tej / którąś náſ poprzyśięgłá /
Jeźli / gdy wnijdziemy do ziemi / tego cżerwonego ƺnuru nie uwiążeƺ u okná / po którymeś náſ ſpuśćiłá / á ojcá twego / y mátki twojej / y bráći twojej / y wƺyſtkiego domu ojcá twego nie zbierzeƺli do śiebie w dom ;
Abowiem ktobykolwiek wyƺedł ze drzwi domu twego / krew jego będzie ná głowę jego / á my będziemy bez winy ; ále káżdego / ktokolwiek będzie z tobą w domu / krew jego obróći śię ná głowę náƺę / jeźli śię go kto ręką dotknie.
Lecż jeźli wydáƺ tę ſpráwę náƺę / tedy będziemy wolni od przyśięgi twojej / którąś náſ poprzyśięgłá.
Y odpowiedziáłá ; Jákośćie powiedzieli / niechże ták będzie. Tedy je wypuśćiłá / y poƺli ; y uwiązáłá ƺnur cżerwony w onem oknie.
A odƺedłƺy przyƺli ná górę / y zoſtáli tám przez trzy dni / áż śię wróćili / którzy je gonili ; bo ich ƺukáli ći / którzy je gonili / po wƺyſtkich drogách / ále nie ználeźli.
Y wróćili śię oni dwáj mężowie / á zſtąpiwƺy z góry / przepráwili śię / y przyƺli do Jozuego / ſyná Nunowego / y powiedzieli mu wƺyſtko / co śię z nimi dziáło ;
Y mówili do Jozuego : Dáł Pán w ręce náƺe tę wƺyſtkę ziemię ; bo śię ſtrwożyli wƺyſcy obywátele ziemi przed twárzą náƺą.
