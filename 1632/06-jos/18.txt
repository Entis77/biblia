Tedy śię zebráło wƺyſtko zgromádzenie ſynów Izráelſkich do Sylo / y poſtáwili tám námiot zgromádzeniá / gdy źiemiá byłá od nich opánowáná.
A zoſtáło było z ſynów Izráelſkich / którym było nie oddźielono dźiedźictwá ich / śiedmioro pokoleniá.
Tedy rzekł Jozue do ſynów Izráelſkich : Dokądże zániedbywáćie wnijść / ábyśćie pośiedli źiemię / którą wám dáł Pán / Bóg ojców wáƺych?
Obierzćie między ſobą po trzech mężách z káżdego pokoleniá / które poślę / áby wſtáwƺy obeƺli źiemię / á rozpiſáli ją według dźiedźictwá ich / potem śię wrócą do mnie.
Y rozdźielą ją ná śiedm cżęśći : Judá ſtánie ná gránicách ſwoich od południá / á dom Józefów ſtánie ná gránicách ſwoich od północy.
Wy tedy rozpiƺećie źiemię ná śiedm cżęśći á przynieśiećie tu do mnie : tedy wám rzucę loſ tu przed Pánem / Bogiem náƺym.
Abowiem Lewitowie nie máją dźiáłu między wámi / gdyż kápłáńſtwo Páńſkie jeſt dźiedźictwo ich ; ále Gád / y Ruben / y połowá pokoleniá Mánáſeſowego wźięli dźiedźictwá ſwe zá Jordánem ná wſchód ſłońcá / które im oddáł Mojżeƺ / ſługá Páńſki.
Przetoż wſtáwƺy mężowie oni odeƺli ; á Jozue rozkázáł tym / którzy ƺli / áby rozpiſáli źiemię / mówiąc : Idźćie á obejdźćie źiemię / y popiƺćie ją / á potem wróććie śię do mnie / á tu wám rzucę loſ przed Pánem w Sylo.
Odeƺli tedy mężowie oni y obchodźili źiemię / y opiſywáli ją według miáſt ná śiedm cżęśći w kśięgi ; potem śię wróćili do Jozuego / do obozu w Sylo.
Rzućił im loſ Jozue w Sylo przed Pánem / á podźielił tám Jozue źiemię ſynom Izráelſkim według dźiáłów ich.
Tedy pádł loſ pokoleniu ſynów Benjáminowych według domów ich / á przyƺłá gránicá loſu ich między ſyny Judowe / y między ſyny Józefowe.
Y byłá gránicá ich ku ſtronie północnej od Jordánu / á ƺłá táż gránicá po bok Jerychá od północy / ćiągnąc śię ná górę ku záchodowi / á końcżyłá śię przy puƺcży Betáwen.
A ſtámtąd idźie tá gránicá do Luz / od ſtrony południowej Luzy / którá jeſt Betel / á puƺcżá śię tá gránicá do Attárot Adár podle góry / którá jeſt od południá Betoron dolnego.
Y bieży tá gránicá kołem po bok morzá ná południe od góry / którá jeſt przećiw Betoron / ná południe / y końcży śię w Káryját Báál / które jeſt Káryját Járym / miáſto ſynów Judowych ; á toć jeſt ſtroná záchodniá.
Stroná záśię ná południe od końcá Káryját Járym ; á wychodźi tá gránicá ku morzu / y bieży ku źródłu wód Neftoá.
Y ćiągnie śię tá gránicá do końcá góry / którá jeſt przećiwko dolinie ſynów Ennon / á jeſt w dolinie Refáim ná północy / y idźie przez dolinę Refáim ná północy / y idźie przez dolinę Ennon po ſtronie Jebuzejcżyká ná południe / ſtámtąd bieży do źródłá Rogiel.
A idźie kołem od północy / á dochodźi do Enſemeſ / á wychodźi do Gelilot / które jeſt przećiwko górze / wſtępując do Adommim / bieżąc ſtámtąd do kámieniá Bohená / ſyná Rubenowego.
Stámtąd idźie ku ſtronie / którá jeſt przećiwko równinom ná północy / y ćiągnie śię ku Arábá.
Stámtąd bieży tá gránicá ku ſtronie Betoglá ná północy / á końcży śię u ſkáły morzá ſłonego ná północy / ku końcowi Jordánu ná południe ; táć jeſt gránicá południowá.
Jordán záś końcży ją ku ſtronie ná wſchód ſłońcá ; á toć jeſt dźiedźictwo ſynów Benjáminowych według gránic ich w okrąg / wedle domów ich.
Były tedy te miáſtá pokoleniá ſynów Benjáminowych według domów ich : Jerycho y Betágál / y doliná Káſyſ.
Y Betárábá / y Sámráim / y Betel ;
Y Awim / y Afárá / y Oferá ;
Y Káfár Hámmonáj / y Ofni / y Gábá / miáſt dwánáśćie / y wśi ich ;
Gábon / y Rámá / y Berot ;
Y Miſfe / y Káfárá / y Moſá ;
Y Rekiem / y Jerefel / y Tárelá ;
Y Selá / Elef / y Jebuz ( które jeſt Jeruzálem ) / Gibeát / Kiryját / miáſt cżternáśćie / y wśi ich. Toć jeſt dźiedźictwo ſynów Benjáminowych według domów ich.
