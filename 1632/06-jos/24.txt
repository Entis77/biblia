Tedy zebráł Jozue wƺyſtkie pokoleniá Izráelſkie do Sychem / y zwołáł ſtárƺych z Izráelá / y przedniejƺych z nich / y ſędziów ich / y przełożonych ich / y ſtánęli przed oblicżem Bożem.
Y rzekł Jozue do wƺyſtkiego ludu : Ták mówi Pán / Bóg Izráelſki : Zá rzeką mieƺkáli ojcowie wáśi od dáwnych cżáſów / Táre / ojćiec Abráhámów / y ojćiec Náchorów / y ſłużyli bogom obcym.
Y wziąłem ojcá wáƺego Abráhámá z miejſcá / które jeſt zá rzeką / y prowádziłem go przez wƺyſtkę ziemię Chánánejſką / y rozmnożyłem náśienie jego / dáwƺy mu Izááká.
Dáłem też Izáákowi Jákóbá y Ezáwá / á podáłem Ezáwowi górę Seir / áby ją pośiádł ; ále Jákób y ſynowie jego záƺli do Egiptu.
Y poſłáłem Mojżeƺá y Aároná / á trápiłem Egipt ; á gdym to ucżynił w pośród niego / potemem wáſ wywiódł.
Y wywiodłem ojce wáƺe z Egiptu / á przyƺliśćie áż do morzá / y gonili Egipcżánie ojce wáƺe z wozmi y z jezdnymi áż do morzá cżerwonego.
Tedy wołáli do Páná / á on położył ćiemnośći między wámi y między Egipcżány / y przywiódł ná nie morze / á okryło je ; y widziáły ocży wáƺe / com ucżynił w Egipćie / y mieƺkáliśćie ná puƺcży przez długi cżáſ.
Potemem przywiódł wáſ do ziemi Amorejcżyká / mieƺkájącego zá Jordánem / y wálcżyli przećiwko wám ; álem je podáł w rękę wáƺę / y pośiedliśćie ziemię ich / á wygłádziłem je przed wámi.
Powſtáł też Bálák / ſyn Seforów / król Moábſki / áby wálcżył przećiw Izráelowi ; á poſłáwƺy przyzwáł Báláámá / ſyná Beorowego / áby wáſ przeklináł.
Y nie chćiáłem ſłucháć Báláámá ; przetoż błogoſłáwiąc błogoſłáwił wám / á ták wybáwiłem wáſ z rąk jego.
Przepráwiliśćie śię potem przez Jordán / y przyƺliśćie do Jerychá / y wálcżyli przećiwko wám mężowie z Jerychá / Amorejcżyk / y Ferezejcżyk / y Chánánejcżyk / y Hetejcżyk / y Gergiezejcżyk / y Hewejcżyk / y Jebuzejcżyk ; álem je podáł w ręce wáƺe.
Y poſłáłem przed wámi śierƺenie / którzy je wypędzili przed oblicżem wáƺem / dwu królów Amorejſkich / nie miecżem twoim áni łukiem twoim.
Y dáłem wám ziemię / w którejśćie nie robili / y miáſtá / którycheśćie nie budowáli / w których mieƺkáćie / á winnic y oliwnic / którycheśćie nie ſádzili / pożywáćie.
Przetoż teráz bójćie śię Páná / á ſłużćie mu w doſkonáłośći y w práwdzie / á znieśćie bogi / którym ſłużyli ojcowie wáśi zá rzeką / y w Egipćie / á ſłużćie Pánu.
A jeźli śię wám zdá źle ſłużyć Pánu / obierzćież ſobie dziś / komubyśćie ſłużyli / choćiáż bogi / którym ſłużyli ojcowie wáśi / co byli zá rzeką / choćiáż bogi Amorejſkie / w których wy ziemi mieƺkáćie ; áleć já y dom mój będziemy ſłużyli Pánu.
Y odpowiedziáł lud mówiąc : Nie dáj Boże / ábyśmy mieli odſtąpić Páná / á ſłużyć bogom cudzym :
Abowiem Pán / Bóg náƺ / on jeſt / który náſ wywiódł / y ojce náƺe z ziemi Egipſkiej / z domu niewoli / á który ucżynił przed ocżymá náƺemi te znáki wielkie / y ſtrzegł náſ we wƺyſtkiej drodze / którąśmy ƺli / y między wƺyſtkimi národy / przez któreśmy przeƺli ;
Y wypędził Pán wƺyſtkie národy / y Amorejcżyká mieƺkájącego w ziemi przed twárzą náƺą. A ták my będziemy ſłużyli Pánu ; bo on jeſt Bóg náƺ.
Tedy rzekł Jozue do ludu : Nie możećie wy ſłużyć Pánu ; bo Bóg święty jeſt / Bóg zápálcżywy jeſt / nie przepuśći złośćiom wáƺym / áni grzechom wáƺym.
Jeźliż opuśćićie Páná / á będziećie ſłużyli bogom cudzym / obróći śię / y utrápi wáſ / y zniƺcży wáſ / choć wám przedtem dobrze cżynił.
Y odpowiedziáł lud Jozuemu : Nie ták ; ále Pánu ſłużyć będziemy.
Tedy rzekł Jozue do ludu : Świádkámi będziećie ſámi przećiwko ſobie / iżeśćie ſobie obráli Páná / ábyśćie mu ſłużyli ; á oni rzekli : Świádkámi jeſteśmy.
Y rzekł : Terázże znieśćie bogi cudze / którzy ſą w pośrodku wáſ / á nákłońćie ſercá wáƺe ku Pánu / Bogu Izráelſkiemu.
Y odpowiedziáł lud Jozuemu : Pánu / Bogu náƺemu / ſłużyć będziemy / y głoſowi jego poſłuƺni być chcemy.
A ták ucżynił Jozue przymierze z ludem dniá onego / y przełożył im rozkázánie y ſąd w Sychem.
Y nápiſáł Jozue ſłowá te w kśięgi Zákonu Bożego ; wziął też kámień wielki / y poſtáwił go tám pod dębem / który był u świątnicy Páńſkiej.
Tedy rzekł Jozue do wƺyſtkiego ludu : Oto kámień ten będzie nám świádectwem ; ábowiem on ſłyƺáł wƺyſtkie ſłowá Páńſkie / które mówił z námi y będzie przećiwko wám ná świádectwo / byśćie ſnáć nie ſkłámáli przećiwko Bogu wáƺemu.
Zátem rozpuśćił Jozue lud / káżdego do dziedzictwá ſwego.
Y ſtáło śię potem / że umárł Jozue / ſyn Nunów / ſługá Páńſki / we ſtu y w dzieśięćiu lát.
Y pogrzebli go ná gránicy dziedzictwá jego w Támnát Sáre / które jeſt ná górze Efráim / ku północy góry Gááſ.
Y ſłużył Izráel Pánu po wƺyſtkie dni Jozuego / y po wƺyſtkie dni ſtárƺych / którzy długo żyli po Jozuem / á którzy wiedzieli o wƺyſtkich ſpráwách Páńſkich / które cżynił Izráelowi.
Kośći też Józefowe / które byli przenieśli ſynowie Izráelſcy z Egiptu / pogrzebli w Sychem / ná cżęśći polá / które był kupił Jákób od ſynów Hemorá / ojcá Sychemowego / zá ſto jágniąt ; y były u ſynów Józefowych w dziedzictwie ich.
Eleázár tákże / ſyn Aáronów / umárł ; y pogrzebli go ná págórku Fineeſá / ſyná jego / który mu był dány ná górze Efráim.
