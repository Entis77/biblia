Tedy zebráł Jozue wƺyſtkie pokoleniá Izráelſkie do Sychem / y zwołáł ſtárƺych z Izráelá / y przedniejƺych z nich / y ſędźiów ich / y przełożonych ich / y ſtánęli przed oblicżem Bożem.
Y rzekł Jozue do wƺyſtkiego ludu : Ták mówi Pán / Bóg Izráelſki : Zá rzeką mieƺkáli ojcowie wáśi od dáwnych cżáſów / Táre / ojćiec Abráhámów / y ojćiec Náchorów / y ſłużyli bogom obcym.
Y wźiąłem ojcá wáƺego Abráhámá z miejſcá / które jeſt zá rzeką / y prowádźiłem go przez wƺyſtkę źiemię Chánánejſką / y rozmnożyłem náśienie jego / dáwƺy mu Izááká.
Dáłem też Izáákowi Jákóbá y Ezáwá / á podáłem Ezáwowi górę Seir / áby ją pośiádł ; ále Jákób y ſynowie jego záƺli do Egiptu.
Y poſłáłem Mojżeƺá y Aároná / á trápiłem Egipt ; á gdym to ucżynił w pośród niego / potemem was wywiódł.
Y wywiodłem ojce wáƺe z Egiptu / á przyƺliśćie áż do morzá / y gonili Egipcżánie ojce wáƺe z wozmi y z jezdnymi áż do morzá cżerwonego.
Tedy wołáli do Páná / á on położył ćiemnośći między wámi y między Egipcżány / y przywiódł ná nie morze / á okryło je ; y widźiáły ocży wáƺe / com ucżynił w Egipćie / y mieƺkáliśćie ná puƺcży przez długi cżáſ.
Potemem przywiódł was do źiemi Amorejcżyká / mieƺkájącego zá Jordánem / y wálcżyli przećiwko wám ; álem je podáł w rękę wáƺę / y pośiedliśćie źiemię ich / á wygłádźiłem je przed wámi.
Powſtáł też Bálák / ſyn Seforów / król Moábſki / áby wálcżył przećiw Izráelowi ; á poſłáwƺy przyzwáł Báláámá / ſyná Beorowego / áby was przeklináł.
Y nie chćiáłem ſłucháć Báláámá ; przetoż błogoſłáwiąc błogoſłáwił wám / á ták wybáwiłem was z rąk jego.
Przepráwiliśćie śię potem przez Jordán / y przyƺliśćie do Jerychá / y wálcżyli przećiwko wám mężowie z Jerychá / Amorejcżyk / y Ferezejcżyk / y Chánánejcżyk / y Hetejcżyk / y Gergiezejcżyk / y Hewejcżyk / y Jebuzejcżyk ; álem je podáł w ręce wáƺe.
Y poſłáłem przed wámi śierƺenie / którzy je wypędźili przed oblicżem wáƺem / dwu królów Amorejſkich / nie miecżem twoim áni łukiem twoim.
Y dáłem wám źiemię / w którejśćie nie robili / y miáſtá / którycheśćie nie budowáli / w których mieƺkáćie / á winnic y oliwnic / którycheśćie nie ſádźili / pożywáćie.
Przetoż teraz bójćie śię Páná / á ſłużćie mu w doſkonáłośći y w práwdźie / á znieśćie bogi / którym ſłużyli ojcowie wáśi zá rzeką / y w Egipćie / á ſłużćie Pánu.
A jeſli śię wám zdá źle ſłużyć Pánu / obierzćież ſobie dźiś / komubyśćie ſłużyli / choćiáż bogi / którym ſłużyli ojcowie wáśi / co byli zá rzeką / choćiáż bogi Amorejſkie / w których wy źiemi mieƺkáćie ; áleć ja y dom mój będźiemy ſłużyli Pánu.
Y odpowiedźiáł lud mówiąc : Nie dáj Boże / ábyśmy mieli odſtąpić Páná / á ſłużyć bogom cudzym :
Abowiem Pán / Bóg náƺ / on jeſt / który nas wywiódł / y ojce náƺe z źiemi Egipſkiej / z domu niewoli / á który ucżynił przed ocżymá náƺemi te znáki wielkie / y ſtrzegł nas we wƺyſtkiej drodze / którąśmy ƺli / y między wƺyſtkimi národy / przez któreśmy przeƺli ;
Y wypędźił Pán wƺyſtkie národy / y Amorejcżyká mieƺkájącego w źiemi przed twárzą náƺą. A ták my będźiemy ſłużyli Pánu ; bo on jeſt Bóg náƺ.
Tedy rzekł Jozue do ludu : Nie możećie wy ſłużyć Pánu ; bo Bóg święty jeſt / Bóg zápálcżywy jeſt / nie przepuśći złośćiom wáƺym / áni grzechom wáƺym.
Jeſliż opuśćićie Páná / á będźiećie ſłużyli bogom cudzym / obróći śię / y utrápi was / y zniƺcży was / choć wám przedtem dobrze cżynił.
Y odpowiedźiáł lud Jozuemu : Nie ták ; ále Pánu ſłużyć będźiemy.
Tedy rzekł Jozue do ludu : Świádkámi będźiećie ſámi przećiwko ſobie / iżeśćie ſobie obráli Páná / ábyśćie mu ſłużyli ; á oni rzekli : Świádkámi jeſteśmy.
Y rzekł : Terazże znieśćie bogi cudze / którzy ſą w pośrodku was / á nákłońćie ſercá wáƺe ku Pánu / Bogu Izráelſkiemu.
Y odpowiedźiáł lud Jozuemu : Pánu / Bogu náƺemu / ſłużyć będźiemy / y głoſowi jego poſłuƺni być chcemy.
A ták ucżynił Jozue przymierze z ludem dniá onego / y przełożył im rozkázánie y ſąd w Sychem.
Y nápiſáł Jozue ſłowá te w kśięgi Zakonu Bożego ; wźiął też kámień wielki / y poſtáwił go tám pod dębem / który był u świątnicy Páńſkiej.
Tedy rzekł Jozue do wƺyſtkiego ludu : Oto kámień ten będźie nám świádectwem ; ábowiem on ſłyƺáł wƺyſtkie ſłowá Páńſkie / które mówił z námi y będźie przećiwko wám ná świádectwo / byśćie ſnáć nie ſkłámáli przećiwko Bogu wáƺemu.
Zátem rozpuśćił Jozue lud / káżdego do dźiedźictwá ſwego.
Y ſtáło śię potem / że umárł Jozue / ſyn Nunów / ſługá Páńſki / we ſtu y w dźieśięćiu lát.
Y pogrzebli go ná gránicy dźiedźictwá jego w Támnát Sáre / które jeſt ná górze Efráim / ku północy góry Gááſ.
Y ſłużył Izráel Pánu po wƺyſtkie dni Jozuego / y po wƺyſtkie dni ſtárƺych / którzy długo żyli po Jozuem / á którzy wiedźieli o wƺyſtkich ſpráwách Páńſkich / które cżynił Izráelowi.
Kośći też Józefowe / które byli przenieśli ſynowie Izráelſcy z Egiptu / pogrzebli w Sychem / ná cżęśći polá / które był kupił Jákób od ſynów Hemorá / ojcá Sychemowego / zá ſto jágniąt ; y były u ſynów Józefowych w dźiedźictwie ich.
Eleázár tákże / ſyn Aáronów / umárł ; y pogrzebli go ná págórku Fineeſá / ſyná jego / który mu był dány ná górze Efráim.
