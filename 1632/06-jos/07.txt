Ale zgrzeƺyli ſynowie Izráelſcy przeſtępſtwem przy rzecżách przeklętych ; ábowiem Achán / ſyn Chármiego / ſyná Zábdy / ſyná Záre / z pokoleniá Judá / wziął nieco z rzecży przeklętych ; zácżem zápálił śię gniew Páńſki przećiw ſynom Izráelſkim.
Bo gdy poſłáł Jozue kilká mężów z Jerychá do Háj / które było bliſko Betáwen ná wſchód ſłońcá od Betel / y rzekł do nich / mówiąc : Idźćie / á wyƺpiegujćie ziemię ; tedy ƺedłƺy oni mężowie / wyƺpiegowáli Háj.
A wróćiwƺy śię do Jozuego / rzekli mu : Niech nie ćiągnie wƺyſtek lud ; około dwóch tyśięcy mężów / álbo około trzech tyśięcy mężów niech idą / á zburzą Háj ; nie trudź tám wƺyſtkiego ludu / bo ich tám trochá.
Poƺło tedy około trzech tyśięcy mężów z ludu / y ućiekli przed mężámi z Háj.
A porázili z nich mężowie z Háj około trzydzieſtu y ƺeśćiu mężów / goniąc je od brámy áż do Sábárym / á porázili je / gdy ućiekáli z góry / y dlá tego rozpłynęło śię ſerce ludu / y było jáko wodá.
Tedy rozdárłƺy Jozue odzienie ſwoje / upádł twárzą ſwoją ná ziemię przed ſkrzynią Páńſką / á leżáł áż do wiecżorá / on y ſtárśi Izráelſcy / poſypáwƺy prochem głowy ſwoje.
Zátem rzekł Jozue : Ach! Pánie Pánujący / przecżżeś przeprowádził lud ten zá Jordán / ábyś náſ podáł w rękę Amorejcżyká ná wytrácenie? O byśmy byli rácżej mieƺkáli zá Jordánem!
O Pánie / cóż rzekę / poniewáż podáwá Izráel tył nieprzyjáćiołom ſwoim?
Bo uſłyƺáwƺy Chánánejcżycy / y wƺyſcy obywátele tej ziemi / obtocżą náſ zewƺąd / á wytrácą imię náƺe z ziemi. Y cóż to ucżyniƺ imieniowi twemu wielkiemu?
Tedy rzekł Pán do Jozuego : Wſtáń ; przecżżeś upádł ná oblicże twoje?
Zgrzeƺył Izráel / y przeſtąpili przymierze moje / którem im przykázáł ; ábowiem wzięli z rzecży przeklętych / á ukrádli je / y ſkłámáli / y ſchowáli je między nácżynie ſwoje.
A dlá tegoć ſynowie Izráelſcy nie będą śię mogli oſtáć przed nieprzyjáćioły ſwymi / tył będą podáwáli nieprzyjáćiołom ſwym / bo śię zmázáli rzecżą przeklętą ; nie będę więcej z wámi / jeźli nie wykorzenićie przeklęſtwá tego z pośrodku wáſ.
Wſtáń / poświęć lud y rzecż : Poświęććie śię ná jutro ; bo ták mówi Pán / Bóg Izráelſki : Przeklęſtwo jeſt w pośrodku ćiebie / Izráelu ; nie oſtoiƺ śię przed nieprzyjáćioły twymi / áż odejmiećie przeklęſtwo z pośrodku śiebie.
A ták przyſtąpćie ráno według pokoleń wáƺych ; á pokolenie / które okáże Pán / przyſtąpi według fámilii ; á fámilijá / którą okáże Pán / przyſtąpi według domów ; á dom / który okáże Pán / przyſtąpi według oſób.
A kto będzie ználeziony w przeklęſtwie / będzie ſpálony ogniem / on / y wƺyſtko / co jego jeſt / dlá tego że przeſtąpił przymierze Páńſkie / á dopuśćił śię niegodnej rzecży w Izráelu.
Przetoż wſtáwƺy Jozue ráno / rozkázáł przyſtępowáć Izráelowi według pokoleń ich ; y ználázło śię pokolenie Judá.
Y kázáł przyſtąpić fámilii Judá / y ználázłá śię fámilijá Záre / y kázáł przyſtąpić fámilii Zárego według oſób / y ználázł śię dom Zábdy.
Y kázáł przyſtąpić domowi jego według oſób / y ználázł śię Achán / ſyn Chármiego / ſyná Zábdy / ſyná Záre / z pokoleniá Judá.
Y rzekł Jozue do Acháná : Synu mój / dáj proƺę chwáłę Pánu / Bogu Izráelſkiemu / y wyznáj przed nim / á oznájmij mi proƺę / coś ucżynił / nie táj przedemną.
Tedy odpowiedziáł Achán Jozuemu / mówiąc : Zápráwdę / jám zgrzeƺył Pánu / Bogu Izráelſkiemu / ták á ták ucżyniłem.
Widziáłem między łupy płáƺcż bábilońſki jeden / piękny / y dwieśćie ſyklów ſrebrá / y pręt złoty jeden / pięćdzieśiąt ſyklów wáżący / y pożądáłem tego / y wziąłem to / á oto / te rzecży ſą zákopáne w ziemi / w pośród námiotu mego / á ſrebro pod niemi.
Tedy poſłáł Jozue poſły / którzy bieżeli do námiotu / á oto te rzecży były ſkryte w námioćie jego / á ſrebro pod niemi.
A wziąwƺy je z námiotu przynieśli je do Jozuego / y do wƺyſtkich ſynów Izráelſkich / á położyli je przed oblicżem Páńſkim.
A ták wziąwƺy Jozue / y wƺyſtek Izráel z nim / Acháná / ſyná Zárego / y ſrebro / y płáƺcż / y pręt złoty / y ſyny jego / y córki jego / y woły jego / y oſły jego / y owce jego / y námiot jego / y wƺyſtko co miáł / wywiedli je ná dolinę Achor.
Y rzekł Jozue : Przecżżeś náſ potrwożył? niechże ćię też Pán zátrwoży dniá tego. Y ukámionowáł go wƺyſtek Izráel / y ſpálili je ogniem / ukámionowáwƺy je kámieńmi ;
Potem wyſtáwili ná nim wielką kupę kámieni / którá trwá áż do dniá tego. Y odwróćił śię Pán od gniewu zápálcżywośći ſwojej ; przetoż názwáne jeſt imię miejſcá onego / doliná Achor / áż do dniá dziśiejƺego.
