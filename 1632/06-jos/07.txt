Ale zgrzeƺyli ſynowie Izráelſcy przeſtępſtwem przy rzecżách przeklętych ; ábowiem Achán / ſyn Chármiego / ſyná Zábdy / ſyná Záre / z pokolenia Judá / wźiął nieco z rzecży przeklętych ; zácżem zápálił śię gniew Páńſki przećiw ſynom Izráelſkim.
Bo gdy poſłał Jozue kilká mężów z Jerychá do Háj / które było bliſko Betáwen ná wſchód ſłońcá od Betel / y rzekł do nich / mówiąc : Idźćie / á wyƺpiegujćie źiemię ; tedy ƺedƺy oni mężowie / wyƺpiegowáli Háj.
A wróćiwƺy śię do Jozuego / rzekli mu : Niech nie ćiągnie wƺyſtek lud ; około dwu tyśięcy mężów / álbo około trzech tyśięcy mężów niech idą / á zburzą Háj ; nie trudź tám wƺyſtkiego ludu / bo ich tám trochá.
Poƺło tedy około trzech tyśięcy mężów z ludu / y ućiekli przed mężámi z Háj.
A poráźili z nich mężowie z Háj około trzydźieſtu y ƺeśćiu mężów / goniąc je od bramy áż do Sábárym / á poráźili je / gdy ućiekáli z góry / y dla tego rozpłynęło śię ſerce ludu / y było jáko wodá.
Tedy rozdárƺy Jozue odźienie ſwoje / upádł twárzą ſwoją ná źiemię przed ſkrzynią Páńſką / á leżał áż do wiecżorá / on y ſtárśi Izráelſcy / poſypáwƺy prochem głowy ſwoje.
Zátem rzekł Jozue : Ach! Pánie Pánujący / przecżżeś przeprowádźił lud ten zá Jordán / ábyś nas podał w rękę Amorejcżyká ná wytrácenie? O byſmy byli rácżey mieƺkáli zá Jordánem!
O Pánie / cóż rzekę / ponieważ podáwá Izráel tył nieprzyjáćiołom ſwoim?
Bo uſłyƺáwƺy Chánánejcżycy / y wƺyſcy obywátele tey źiemi / obtocżą nas zewƺąd / á wytrácą imię náƺe z źiemi. Y cóż to ucżyniƺ imieniowi twemu wielkiemu?
Tedy rzekł Pán do Jozuego : Wſtáń ; przecżżeś upádł ná oblicże twoje?
Zgrzeƺył Izráel / y przeſtąpili przymierze moje / którem im przykázał ; ábowiem wźięli z rzecży przeklętych / á ukrádli je / y ſkłámáli / y ſchowáli je miedzy nacżynie ſwoje.
A dla tegoć ſynowie Izráelſcy nie będą śię mogli oſtáć przed nieprzyjáćioły ſwymi / tył będą podáwáli nieprzyjáćiołom ſwym / bo śię zmázáli rzecżą przeklętą ; nie będę więcey z wámi / jeſli nie wykorzenićie przeklęſtwá tego z pośrzodku was.
Wſtáń / poświęć lud y rzecż : Poświęććie śię ná jutro ; bo ták mówi Pán / Bóg Izráelſki : Przeklęſtwo jeſt w pośrzodku ćiebie / Izráelu ; nie oſtoiƺ śię przed nieprzyjáćioły twymi / áż odejmiećie przeklęſtwo z pośrzodku śiebie.
A ták przyſtąpćie ráno według pokoleń wáƺych ; á pokolenie / które okaże Pán / przyſtąpi według fámilii ; á fámilijá / którą okaże Pán / przyſtąpi według domów ; á dom / który okaże Pán / przyſtąpi według oſób.
A kto będźie ználeźiony w przeklęſtwie / będźie ſpálony ogniem / on / y wƺyſtko / co jego jeſt / dla tego że przeſtąpił przymierze Páńſkie / á dopuśćił śię niegodney rzecży w Izráelu.
Przetoż wſtáwƺy Jozue ráno / rozkázał przyſtępowáć Izráelowi według pokoleń ich ; y ználázło śię pokolenie Judá.
Y kázał przyſtąpić fámilii Judá / y ználázłá śię fámilijá Záre / y kázał przyſtąpić fámilii Zárego według oſób / y ználázł śię dom Zábdy.
Y kázał przyſtąpić domowi jego według oſób / y ználázł śię Achán / ſyn Chármiego / ſyná Zábdy / ſyná Záre / z pokolenia Judá.
Y rzekł Jozue do Acháná : Synu mój / dáj proƺę chwałę Pánu / Bogu Izráelſkiemu / y wyznáj przed nim / á oznájmij mi proƺę / coś ucżynił / nie táj przedemną.
Tedy odpowiedźiał Achán Jozuemu / mówiąc : Zápráwdę / jam zgrzeƺył Pánu / Bogu Izráelſkiemu / ták á ták ucżyniłem.
Widźiałem miedzy łupy płáƺcż bábilońſki jeden / piękny / y dwieśćie ſyklów ſrebrá / y pręt złoty jeden / pięćdźieśiąt ſyklów wáżący / y pożądáłem tego / y wźiąłem to / á oto / te rzecży ſą zákopáne w źiemi / w pośród namiotu mego / á ſrebro pod niemi.
Tedy poſłał Jozue poſły / którzy bieżeli do namiotu / á oto te rzecży były ſkryte w námioćie jego / á ſrebro pod niemi.
A wźiąwƺy je z namiotu przynieśli je do Jozuego / y do wƺyſtkich ſynów Izráelſkich / á położyli je przed oblicżem Páńſkim.
A ták wźiąwƺy Jozue / y wƺyſtek Izráel z nim / Acháná / ſyná Zárego / y ſrebro / y płáƺcż / y pręt złoty / y ſyny jego / y córki jego / y woły jego / y oſły jego / y owce jego / y námiot jego / y wƺyſtko co miał / wywiedli je ná dolinę Achor.
Y rzekł Jozue : Przecżżeś nas potrwożył? niechże ćię też Pán zátrwoży dniá tego. Y ukámionował go wƺyſtek Izráel / y ſpálili je ogniem / ukámionowáwƺy je kámieńmi ;
Potem wyſtáwili ná nim wielką kupę kámieni / która trwá áż do dniá tego. Y odwróćił śię Pán od gniewu zápálcżywośći ſwojey ; przetoż názwáne jeſt imię miejſcá onego / doliná Achor / áż do dniá dźiśiejƺego.
