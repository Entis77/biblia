Jáko muchy zdechłe záſmrádzáją y pſują olejek áptekárſki : ták cżłowieká z mądrośći y z ſłáwy zácnágo trochę głupſtwá oƺpecá.
Serce mądrego jeſt po práwej ſtronie jego ; ále ſerce głupiego po lewej ſtronie jego.
Y ná ten cżáſ / gdy głupi drogą idzie / ſerce jego niedoſtátek ćierpi ; bo pokázuje wƺyſtkim / że głupim jeſt.
Jeźliby duch pánującego powſtáł przećiwko tobie / nie opuƺcżáj miejſcá twego ; ábowiem pokorá wſtręt cżyni grzechom wielkim.
Jeſt złe / którem widziáł pod ſłońcem / to jeſt / błąd / który pochodzi od zwierzchnośći :
Że głupi wywyżƺáni bywáją w godnośći wielkiej / á bogáći w mądrość niſko śiádáją ;
Widziáłem ſługi ná koniách / á kśiążąt chodzących piechotą jáko ſługi.
Kto kopie dół / ſám weń wpádá ; á kto rozrzucá płot / wąż go ukąśi.
Kto przenośi kámienie / urázi śię niemi ; á kto łupie drwá / niebezpiecżen jeſt od nich.
Jeźliże śię ſtępi żelázo / á nie náoſtrzyłby oſtrzá jego / tedy mocy przyłożyć muśi ; ále to dáleko lepiej mądrość ſpráwić może.
Jeźli ukąśi wąż przed záklęćiem / nic nie pomogą ſłowá záklinácżá.
Słowá uſt mądrego ſą wdzięcżne ; ále wárgi głupiego pożeráją go.
Pocżątek ſłów uſt jego głupſtwo / á koniec powieśći jego wielkie błázeńſtwo.
Bo głupi wiele mówi / choć nie wie ten cżłowiek / co má być. Abowiem któż mu oznájmi / co po nim náſtánie?
Głupi prácują áż do uſtániá / á przećie nie mogą dojść do miáſtá.
Biádá tobie / ziemio! której król jeſt dziećięćiem / y której kśiążętá ráno bieśiádują.
Błogoſłáwionáś ty / ziemio! której król jeſt ſynem zácnych / á której kśiążętá cżáſu ſłuƺnego jádáją dlá pośileniá / á nie dlá opilſtwá.
Dlá leniſtwá śię dách pochylá / á dlá oſłábiáłych rąk przećieká dom.
Dlá uweſeleniá gotują ucżty / y wino rozweſelá żywot ; ále pieniądze do wƺyſtkiego dopomágáją.
Ani w myśli twojej królowi nie złorzecż / áni w ſkrytym pokoju twoim nie przeklináj bogátego ; ábowiem y pták niebieſki donióſłby ten głoſ ; á to / co má ſkrzydłá / objáwiłoby powieść twoję.
