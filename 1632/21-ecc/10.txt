Jáko muchy zdechłe záſmrádzáją y pſują olejek áptekárſki : ták cżłowieká z mądrośći y z ſłáwy zácnágo trochę głupſtwá oƺpecá.
Serce mądrego jeſt po práwej ſtronie jego ; ále ſerce głupiego po lewej ſtronie jego.
Y ná ten cżás / gdy głupi drogą idźie / ſerce jego niedoſtátek ćierpi ; bo pokázuje wƺyſtkim / że głupim jeſt.
Jeſliby duch pánującego powſtał przećiwko tobie / nie opuƺcżáj miejſcá twego ; ábowiem pokorá wſtręt cżyni grzechom wielkim.
Jeſt złe / którem widźiał pod ſłońcem / to jeſt / błąd / który pochodźi od zwierzchnośći :
Że głupi wywyżƺáni bywáją w godnośći wielkiej / á bogáći w mądrość niſko śiádáją ;
Widźiałem ſługi ná koniách / á kśiążąt chodzących piechotą jáko ſługi.
Kto kopie dół / ſám weń wpádá ; á kto rozrzucá płot / wąż go ukąśi.
Kto przenośi kámienie / uráźi śię niemi ; á kto łupie drwá / niebezpiecżen jeſt od nich.
Jeſliże śię ſtępi żelázo / á nie náoſtrzyłby oſtrzá jego / tedy mocy przyłożyć muśi ; ále to dáleko lepiej mądrość ſpráwić może.
Jeſli ukąśi wąż przed záklęćiem / nic nie pomogą ſłowá záklinácżá.
Słowá uſt mądrego ſą wdźięcżne ; ále wárgi głupiego pożeráją go.
Pocżątek ſłów uſt jego głupſtwo / á koniec powieśći jego wielkie błázeńſtwo.
Bo głupi wiele mówi / choć nie wie ten cżłowiek / co ma być. Abowiem któż mu oznájmi / co po nim náſtánie?
Głupi prácują áż do uſtánia / á przećie nie mogą dojść do miáſtá.
Biádá tobie / źiemio! którey król jeſt dźiećięćiem / y którey kśiążętá ráno bieśiádują.
Błogoſłáwionáś ty / źiemio! którey król jeſt ſynem zácnych / á którey kśiążętá cżáſu ſłuƺnego jádáją dla pośileniá / á nie dla opilſtwá.
Dla leniſtwá śię dách pochylá / á dla oſłábiáłych rąk przećieká dom.
Dla uweſeleniá gotują ucżty / y wino rozweſelá żywot ; ále pieniądze do wƺyſtkiego dopomágáją.
Ani w myśli twojey królowi nie złorzecż / áni w ſkrytym pokoju twoim nie przeklináj bogátego ; ábowiem y pták niebieſki donióſłby ten głos ; á to / co ma ſkrzydłá / objáwiłoby powieść twoję.
