Nie bywáj porywcży do mówieniá / áni ſerce twoje prędkie ná wymówienie ſłowá przed oblicżem Bożem / ábowiem Bóg jeſt ná niebie / á ty ná źiemi ; przeto niech ſłów twojich máło będźie.
Bo jáko ſen przychodźi z wielkiej prácy / ták głos głupiego z wielu ſłów.
Gdy co Bogu poślubiƺ / nie omieƺkiwáj tego oddáć / boć mu śię głupi nie podobáją ; cokolwiek poślubiƺ / oddáj.
Lepiej jeſt nie ślubowáć / niżeli poślubiwƺy co / nie oddáć.
Nie dopuƺcżáj uſtom twoim / áby do grzechu przywodźiły ćiáło twoje / áni mów przed ániołem / że to jeſt błąd. Przecżże maƺ Bogá gniewáć mową ſwą / któryby wniwecż obróćił ſpráwę rąk twojich?
Bo gdźie jeſt wiele ſnów / tám y márnośći y ſłów wiele ; ále śię ty Bogá bój.
Jeſli ućiſk ubogiego / y zátrzymánie ſądu y ſpráwiedliwośći ujrzyƺ w którey krájinie / nie dźiwuj śię temu ; bo wyżƺy wyſokiego upátruje / á jeƺcże wyżśi ſą nád nimi.
Zábáwá koło źiemi ma pierwƺe miejſce u wƺyſtkich ; y król roli ſłuży.
Kto miłuje pieniądze / nie náſyći śię pieniędzy / á kto miłuje bogáctwá / nie będźie miał pożytku. Y toć jeſt márność.
Gdźie wiele májętnośći / wiele bywa tych / co ją jedzą. Cóż tedy zá pożytek pánu z tego? jedno że ná nie pátrzy ocżymá ſwemi.
Słodki jeſt ſen prácowitemu / choćiáż máło / choćiáż wiele jádł ; ále náſycenie bogátego ſpáć mu nie dopuśći.
Jeſt ćiężká biedá / którąm widźiał pod ſłońcem ; bogáctwá záchowáne ná złe páná ſwego.
Bo tákowe bogáctwo złą ſpráwą giną / á ſyn / którego ſpłodźi / nie będźie miał nic w rękách ſwych.
Jáko nági wyƺedł z żywotá mátki ſwojey / ták śię wrácá / jáko był przyƺedł / á nie odnośi nic z prácy ſwojey / coby miał wźiąć w rękę ſwoję.
A ták y toć jeſt ćiężká biedá / że jáko przyƺedł / ták odejdźie. Cóż tedy zá pożytek / że ná wiátr prácował?
Dotego / że po wƺyſtkie dni ſwoje w ćiemnośći jádał z wielkim kłopotem / z boleśćią y z gniewem.
Toć jeſt / com ja obácżył / że dobrá y oſobliwá rzecż jeſt / jeść y pić / y używáć dobrego ze wƺyſtkiej prácy ſwej / którą cżłowiek podejmuje pod ſłońcem po wƺyſtkie dni żywotá ſwego / które mu dał Bóg ; ábowiem to jeſt dźiał jego.
A któremukolwiek cżłowiekowi dał Bóg májętność y bogáctwo / y dał mu w moc / áby ich używał / y odbierał dźiał ſwój / á weſelił śię z prácy ſwojey : to jeſt dar Boży.
Bo nie będźie wiele pámiętał ná dni żywotá ſwego ; przeto / że mu Bóg życży weſelá ſercá jego.
