Nie bywáj porywcży do mówieniá / áni ſerce twoje prędkie ná wymówienie ſłowá przed oblicżem Bożem / ábowiem Bóg jeſt ná niebie / á ty ná ziemi ; przeto niech ſłów twoich máło będzie.
Bo jáko ſen przychodzi z wielkiej prácy / ták głoſ głupiego z wielu ſłów.
Gdy co Bogu poślubiƺ / nie omieƺkiwáj tego oddáć / boć mu śię głupi nie podobáją ; cokolwiek poślubiƺ / oddáj.
Lepiej jeſt nie ślubowáć / niżeli poślubiwƺy co / nie oddáć.
Nie dopuƺcżáj uſtom twoim / áby do grzechu przywodziły ćiáło twoje / áni mów przed ániołem / że to jeſt błąd. Przecżże máƺ Bogá gniewáć mową ſwą / któryby wniwecż obróćił ſpráwę rąk twoich?
Bo gdzie jeſt wiele ſnów / tám y márnośći y ſłów wiele ; ále śię ty Bogá bój.
Jeźli ućiſk ubogiego / y zátrzymánie ſądu y ſpráwiedliwośći ujrzyƺ w której kráinie / nie dziwuj śię temu ; bo wyżƺy wyſokiego upátruje / á jeƺcże wyżśi ſą nád nimi.
Zábáwá koło ziemi má pierwƺe miejſce u wƺyſtkich ; y król roli ſłuży.
Kto miłuje pieniądze / nie náſyći śię pieniędzy / á kto miłuje bogáctwá / nie będzie miáł pożytku. Y toć jeſt márność.
Gdzie wiele májętnośći / wiele bywá tych / co ją jedzą. Cóż tedy zá pożytek pánu z tego? jedno że ná nie pátrzy ocżymá ſwemi.
Słodki jeſt ſen prácowitemu / choćiáż máło / choćiáż wiele jádł ; ále náſycenie bogátego ſpáć mu nie dopuśći.
Jeſt ćiężká biedá / którąm widziáł pod ſłońcem ; bogáctwá záchowáne ná złe páná ſwego.
Bo tákowe bogáctwo złą ſpráwą giną / á ſyn / którego ſpłodzi / nie będzie miáł nic w rękách ſwych.
Jáko nági wyƺedł z żywotá mátki ſwojej / ták śię wrácá / jáko był przyƺedł / á nie odnośi nic z prácy ſwojej / coby miáł wziąć w rękę ſwoję.
A ták y toć jeſt ćiężká biedá / że jáko przyƺedł / ták odejdzie. Cóż tedy zá pożytek / że ná wiátr prácowáł?
Dotego / że po wƺyſtkie dni ſwoje w ćiemnośći jádáł z wielkim kłopotem / z boleśćią y z gniewem.
Toć jeſt / com já obácżył / że dobrá y oſobliwá rzecż jeſt / jeść y pić / y używáć dobrego ze wƺyſtkiej prácy ſwej / którą cżłowiek podejmuje pod ſłońcem po wƺyſtkie dni żywotá ſwego / które mu dáł Bóg ; ábowiem to jeſt dziáł jego.
A któremukolwiek cżłowiekowi dáł Bóg májętność y bogáctwo / y dáł mu w moc / áby ich używáł / y odbieráł dziáł ſwój / á weſelił śię z prácy ſwojej : to jeſt dár Boży.
Bo nie będzie wiele pámiętáł ná dni żywotá ſwego ; przeto / że mu Bóg życży weſelá ſercá jego.
