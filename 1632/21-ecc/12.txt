Pámiętáj tedy ná ſtworzyćielá ſwego we dni młodośći twojej / pierwej niżeli náſtáną złe dni / y nádejdą látá / o których rzecżeƺ : Nie podobáją mi śię.
Pierwej niż śię záćmi ſłońce / y świátło / y mieśiąc y gwiázdy / á náwrócą śię obłoki po dżdżu.
W dzień / którego śię poruƺą ſtróże domowi / y záchwieją śię mężowie duży y uſtáną melący / przeto / iż ich máło będzie / y záćmią śię wyglądájący oknámi ;
Y záwrą śię drzwi z dworu z ſłábym głoſem mełćiá ; y powſtánie ná głoſ ptáƺy / y uſtáną wƺyſtkie córki śpiewájące.
Náwet y wyſokiego miejſcá báć śię będą / y będą śię lękáć ná drodze / gdy zákwitnie migdáłowe drzewo / tákże y ƺáráńcżá będzie mu ćiężká / y żądzá go ominie ; bo cżłowiek idzie do domu wiecżnego / á płácżący po ulicách chodzić będą /
Pierwej niż śię przerwie ƺnur ſrebrny / y niż śię ſtłucże cżáƺá złotá / á rozſypie śię wiádro nád zdrojem / á ſkruƺy śię koło nád ſtudnią ;
Y wróći śię proch do ziemi / jáko przedtem był / á duch wróći śię do Bogá / który go dáł.
Márność nád márnośćiámi / mówi káznodziejá / á wƺyſtko márność.
A cżem więcej káznodziejá był mędrƺym / tem więcej náucżáł umiejętnośći ludu / á rozwáżáł / y wywiádywáł śię / y ſkłádáł wiele przypowieśći.
Stáráł śię káznodziejá / jákoby ználázł powieśći wdzięcżne / y nápiſáł / co jeſt dobrego / y ſłowá práwdziwe.
Słowá mądrych podobne ośćieniom / y podobne gwoździom wbitym ; ſłowá tych / którzy je złożyli / podáne ſą od páſterzá jednego.
A ták / ſynu mój! z tych ſłów śię doſtátecżnie upomnieć możeƺ ; álbowim ſkłádániu wielu kśiąg końcá niemáƺ / á wiele cżytáć / jeſt ſprácowánie ćiáłá.
Sumá wƺyſtkiego / coś ſłyƺáł : Bogá śię bój / á przykázáń jego przeſtrzegáj / bo ná tem cżłowiekowi wƺyſtko záleży ;
Poniewáż káżdy ucżynek / y káżdą rzecż tájną / lub dobrą / lub złą / Bóg ná ſąd przywiedzie.
