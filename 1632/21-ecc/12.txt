Pámiętáj tedy ná ſtworzyćielá ſwego we dni młodośći twojey / pierwej niżeli náſtáną złe dni / y nádejdą látá / o których rzecżeƺ : Nie podobáją mi śię.
Pierwej niż śię záćmi ſłońce / y świátło / y mieśiąc y gwiázdy / á náwrócą śię obłoki po dżdżu.
W dźień / którego śię poruƺą ſtróże domowi / y záchwieją śię mężowie duży y uſtáną melący / przeto / iż ich máło będźie / y záćmią śię wyglądájący oknámi ;
Y záwrą śię drzwi z dworu z ſłábym głoſem mełćiá ; y powſtánie ná głos ptáƺy / y uſtáną wƺyſtkie córki śpiewájące.
Náwet y wyſokiego miejſcá báć śię będą / y będą śię lękáć ná drodze / gdy zákwitnie migdáłowe drzewo / tákże y ƺáráńcżá będźie mu ćiężká / y żądzá go ominie ; bo cżłowiek idźie do domu wiecżnego / á płácżący po ulicách chodźić będą /
Pierwej niż śię przerwie ƺnur ſrebrny / y niż śię ſtłucże cżáƺá złotá / á rozſypie śię wiádro nád zdrojem / á ſkruƺy śię koło nád ſtudnią ;
Y wróći śię proch do źiemi / jáko przedtem był / á duch wróći śię do Bogá / który go dał.
Márność nád márnośćiámi / mówi káznodźiejá / á wƺyſtko márność.
A cżym więcey káznodźiejá był mędrƺym / tym więcey náucżał umiejętnośći ludu / á rozwáżał / y wywiádywał śię / y ſkłádał wiele przypowieśći.
Stárał śię káznodźiejá / jákoby ználázł powieśći wdźięcżne / y nápiſał / co jeſt dobrego / y ſłowá práwdźiwe.
Słowá mądrych podobne ośćieniom / y podobne gwoźdźiom wbitym ; ſłowá tych / którzy je złożyli / podáne ſą od páſterzá jednego.
A ták / ſynu mój! z tych ſłów śię doſtátecżnie upomnieć możeƺ ; álbowim ſkłádániu wielu kśiąg końcá niemáƺ / á wiele cżytáć / jeſt ſprácowánie ćiáłá.
Sumá wƺyſtkiego / coś ſłyƺał : Bogá śię bój / á przykázáń jego przeſtrzegáj / bo ná tym cżłowiekowi wƺyſtko záleży ;
Poniewáż káżdy ucżynek / y káżdą rzecż tájną / lub dobrą / lub złą / Bóg ná ſąd przywiedźie.
