Káżda rzecż ma ſwój cżás / y káżde przedśięwźięćie ma ſwój cżás pod niebem.
Jeſt cżás rodzeniá y cżás umieránia ; cżás ſádzeniá / y cżás wyćinánia tego / co ſádzono ;
Cżás zábijánia / y cżás lecżeniá ; cżás rozwálánia / y cżás budowánia ;
Cżás płácżu / y cżás śmiechu ; cżás ſmutku / y cżás ſkákánia ;
Cżás rozrzucánia kámieni / y cżás zbieránia kámieni ; cżás obłápiánia / y cżás oddáleniá śię od obłápiánia ;
Cżás ƺukánia / y cżás ſtráceniá ; cżás chowánia / y cżás odrzuceniá ;
Cżás rozdźieránia / y cżás zƺywánia ; cżás milcżeniá / y cżás mówieniá ;
Cżás miłowánia / y cżás nienáwidzeniá ; cżás wojny / y cżás pokoju.
Cóż tedy ma ten / co prácuje / z tego / około cżego prácuje?
Widźiałem prácę / którą dał Bóg ſynom ludzkim / áby śię nią báwili.
Wƺyſtko dobrze cżyni cżáſu ſwego ; owƺem y żądość świátá dał do ſercá ich / choć cżłowiek dźiełá tego / które Bóg ſpráwuje / áni pocżątku / áni końcá nie dochodźi.
Stąd wiem / że nic lepƺego nie máją / jedno áby śię weſelili / á cżynili dobrze zá żywotá ſwego.
Acż y to / gdy káżdy cżłowiek je y pije / y używá dobrze wƺyſtkiej prácy ſwojey / jeſt dar Boży.
Wiem y to / że cokolwiek Bóg cżyni / trwá ná wieki ; y że śię do tego nic nie może przydáć / áni z tego co ująć ; á cżyni to Bóg / áby śię báli oblicża jego.
To / co było / teraz jeſt / á co będźie / już było ; ábowiem Bóg odnáwiá to / co przeminęło.
Nádtom jeƺcże widźiał pod ſłońcem ná miejſcu ſądu niepobożność / á ná miejſcu ſpráwiedliwośći nieſpráwiedliwość.
Y rzekłem w ſercu ſwem : Spráwiedliwego y niezbożnego Bóg ſądźić będźie ; bo cżás káżdemu przedśięwźięćiu y káżdej ſpráwy tám będźie.
Nádto rzekłem w ſercu ſwem o ſpráwie ſynów ludzkich / że im Bóg okázał / áby wiedźieli / że ſą podobni bydłu.
Bo przypádek ſynów ludzkich / y przypádek bydłá / jeſt przypádek jednáki. Jáko umierá ono / ták umierá y ten / y duchá jednákiego wƺyſcy máją / á nie ma cżłowiek nic więcey nád bydlę ; bo wƺyſtko jeſt márność.
Wƺyſtko to idźie ná jedno miejſce ; á wƺyſtko jeſt z prochu / y wƺyſtko śię záś w proch obrácá.
A któż wie / że duch ſynów ludzkich wſtępuje w górę? á duch bydlęcy / że zſtępuje pod źiemię?
Przetoż obácżyłem / że cżłowiekowi niemáƺ nic lepƺego / jedno weſelić śię z prácy ſwej / gdyż to jeſt dźiał jego ; ábowiem któż go do tego przywiedźie / áby poznał to / co ma być po nim?
