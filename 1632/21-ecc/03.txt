Káżdá rzecż má ſwój cżáſ / y káżde przedśięwzięćie má ſwój cżáſ pod niebem.
Jeſt cżáſ rodzeniá y cżáſ umierániá ; cżáſ ſádzeniá / y cżáſ wyćinániá tego / co ſádzono ;
Cżáſ zábijániá / y cżáſ lecżeniá ; cżáſ rozwálániá / y cżáſ budowániá ;
Cżáſ płácżu / y cżáſ śmiechu ; cżáſ ſmutku / y cżáſ ſkákániá ;
Cżáſ rozrzucániá kámieni / y cżáſ zbierániá kámieni ; cżáſ obłápiániá / y cżáſ oddáleniá śię od obłápiániá ;
Cżáſ ƺukániá / y cżáſ ſtráceniá ; cżáſ chowániá / y cżáſ odrzuceniá ;
Cżáſ rozdzierániá / y cżáſ zƺywániá ; cżáſ milcżeniá / y cżáſ mówieniá ;
Cżáſ miłowániá / y cżáſ nienáwidzeniá ; cżáſ wojny / y cżáſ pokoju.
Cóż tedy má ten / co prácuje / z tego / około cżego prácuje?
Widziáłem prácę / którą dáł Bóg ſynom ludzkim / áby śię nią báwili.
Wƺyſtko dobrze cżyni cżáſu ſwego ; owƺem y żądość świátá dáł do ſercá ich / choć cżłowiek dziełá tego / które Bóg ſpráwuje / áni pocżątku / áni końcá nie dochodzi.
Stąd wiem / że nic lepƺego nie máją / jedno áby śię weſelili / á cżynili dobrze zá żywotá ſwego.
Acż y to / gdy káżdy cżłowiek je y pije / y używá dobrze wƺyſtkiej prácy ſwojej / jeſt dár Boży.
Wiem y to / że cokolwiek Bóg cżyni / trwá ná wieki ; y że śię do tego nic nie może przydáć / áni z tego co ująć ; á cżyni to Bóg / áby śię báli oblicżá jego.
To / co było / teráz jeſt / á co będzie / już było ; ábowiem Bóg odnáwiá to / co przeminęło.
Nádtom jeƺcże widziáł pod ſłońcem ná miejſcu ſądu niepobożność / á ná miejſcu ſpráwiedliwośći nieſpráwiedliwość.
Y rzekłem w ſercu ſwem : Spráwiedliwego y niezbożnego Bóg ſądzić będzie ; bo cżáſ káżdemu przedśięwzięćiu y káżdej ſpráwy tám będzie.
Nádto rzekłem w ſercu ſwem o ſpráwie ſynów ludzkich / że im Bóg okázáł / áby wiedzieli / że ſą podobni bydłu.
Bo przypádek ſynów ludzkich / y przypádek bydłá / jeſt przypádek jednáki. Jáko umierá ono / ták umierá y ten / y duchá jednákiego wƺyſcy máją / á nie má cżłowiek nic więcej nád bydlę ; bo wƺyſtko jeſt márność.
Wƺyſtko to idzie ná jedno miejſce ; á wƺyſtko jeſt z prochu / y wƺyſtko śię záś w proch obrácá.
A któż wie / że duch ſynów ludzkich wſtępuje w górę? á duch bydlęcy / że zſtępuje pod ziemię?
Przetoż obácżyłem / że cżłowiekowi niemáƺ nic lepƺego / jedno weſelić śię z prácy ſwej / gdyż to jeſt dziáł jego ; ábowiem któż go do tego przywiedzie / áby poznáł to / co má być po nim?
