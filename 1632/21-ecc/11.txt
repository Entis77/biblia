Puƺcżáj chleb twój po wodźie ; bo po wielu dniách znájdźieƺ go.
Dáj cżąſtkę śiedmiom álbo ośmiom ; bo nie wieƺ / co złego będźie ná źiemi.
Gdy śię nápełniáją obłoki / deƺcż ná źiemię wypuƺcáją ; á gdy upádá drzewo ná południe / álbo ná północy / ná któremkolwiek miejſcu upádnie to drzewo / tám zoſtánie.
Kto upátruje wiátr / nigdy nie będźie śiał ; á kto śię przypátruje obłokom / nie będźie żął.
Jáko ty nie wieƺ / która jeſt drogá wiátru / y jáko śię zráſtáją kośći w żywoćie brzemienney : ták nie wieƺ ſpráwy Bożey / który wƺyſtko cżyni.
Poránu śiej naśienie twoje / á w wiecżór nie dáwáj odpocżynku ręce twojey / gdyż ty nie wieƺ / co jeſt lepƺego / toli / cży owo / cżyli też oboje jednáko dobre.
Zápráwdę wdźięcżná jeſt świátłość / y miłá rzecż ocżom widźieć ſłońce.
A wƺákże / choćby przez wiele lat żyw był cżłowiek / á przez te wƺyſtkie weſeliłby śię / tedy przywiódƺy ſobie ná pámięć dni ćiemnośći / jáko ich wiele będźie / cokolwiek przeƺło / uzná być márnośćią.
Przetoż weſel śię / młodźieńcże! w młodośći twojey / á niech używá dobrej myśli ſerce twoje zá dni młodośći twojey / á chodź drogámi ſercá twego / y według zdánia ocżu twojich ; ále wiedz / że ćię dla tego wƺyſtkiego Bóg ná ſąd przywiedźie.
A ták oddál gniew od ſercá twego / y odrzuć złość od ćiáłá twego / gdyż dźiećińſtwo y młodość ſą márnośćią.
