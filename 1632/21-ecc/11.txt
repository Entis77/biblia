Puƺcżáj chleb twój po wodźie ; bo po wielu dniách znájdźieƺ go.
Dáj cżąſtkę śiedmiom álbo ośmiom ; bo nie wieƺ / co złego będźie ná źiemi.
Gdy śię nápełniáją obłoki / deƺcż ná źiemię wypuƺcáją ; á gdy upádá drzewo ná południe / álbo ná północy / ná któremkolwiek miejſcu upádnie to drzewo / tám zoſtánie.
Kto upátruje wiátr / nigdy nie będźie śiáł ; á kto śię przypátruje obłokom / nie będźie_żął.
Jáko ty nie wieƺ / którá jeſt drogá wiátru / y jáko śię zráſtáją kośći w żywoćie brzemiennej : ták nie wieƺ ſpráwy Bożej / który wƺyſtko cżyni.
Poránu śiej náśienie twoje / á w wiecżór nie dáwáj odpocżynku ręce twojej / gdyż ty nie wieƺ / co jeſt lepƺego / toli / cży owo / cżyli też oboje jednáko dobre.
Zápráwdę wdźięcżná jeſt świátłość / y miłá rzecż ocżom widźieć ſłońce.
A wƺákże / choćby przez wiele lát żyw był cżłowiek / á przez te wƺyſtkie weſeliłby śię / tedy przywiódłƺy ſobie ná pámięć dni ćiemnośći / jáko ich wiele będźie / cokolwiek przeƺło / uzná być márnośćią.
Przetoż weſel śię / młodźieńcże! w młodośći twojej / á niech używá dobrej myśli ſerce twoje zá dni młodośći twojej / á chodź drogámi ſercá twego / y według zdániá ocżu twoich ; ále wiedz / że ćię dlá tego wƺyſtkiego Bóg ná ſąd przywiedźie.
A ták oddál gniew od ſercá twego / y odrzuć złość od ćiáłá twego / gdyż dźiećińſtwo y młodość ſą márnośćią.
