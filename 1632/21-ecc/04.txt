Potemem śię obróćił y ujrzáłem wƺyſtkie ućiſki / które śię dzieją pod ſłońcem / á oto widziáłem łzy ućiśnionych / którzy nie máją poćieƺyćielá / áni mocy / áby uƺli rąk tych / którzy ich ćiemiężą ; á nie máją / mówię / poćieƺyćielá.
Dlátegom já umárłych / którzy już zeƺli / więcej chwálił / niżeli żywych / którzy jeƺcże áż dotąd żyją.
Owƺem ƺcżęśliwy jeſt nád tych obydwóch ten / który jeƺcże nie był / który nie widziáł nic złego / które śię dzieje pod ſłońcem.
Bom widziáł / że wƺeláká prácá y káżde dzieło dobre jeſt ku zázdrośći jednych drugim. Y toć jeſt márność y utrápienie duchá.
Głupi ſkłádá ręce ſwe / á je ćiáło ſwoje.
Lepƺá jeſt pełná gárść z pokojem / niżeli obie gárśći pełne z prácą y z udręcżeniem duchá.
Znowu obróćiwƺy śię ujrzáłem drugą márność pod ſłońcem.
Jeſt kto ſámotny / niemájąc żádnego / áni ſyná / áni brátá / á wżdy niemáƺ końcá wƺelákiej prácy jego / áni ocży jego mogą śię náſyćić bogáctwem. Nie myśli : Komuż já prácuję / ták że y żywotowi ſwemu ujmuję dobrego. Y toć jeſt márność / y ćiężkie udręcżenie.
Lepiej jeſt we dwóch być / niż jednemu ; máją záiſte dobry pożytek z prácy ſwojej.
Bo jeźli jeden upádnie / drugi podźwignie towárzyƺá ſwego. A ták biádá ſámotnemu / gdyby upádł! bo nie má drugiego / coby go podźwignął.
Tákże będąli dwáj ſpołu leżeć / zágrzeją śię ; ále jeden jákoż śię zágrzeje?
Owƺem jeźliby kto jednego przemágáł / dwáj mu śię záſtáwią ; á ƺnur troiſty nie łácno śię zerwie.
Lepƺy jeſt chłopiec ubogi á mądry / niżeli król ſtáry á głupi / który już nie umie przyjmowáć nápominániá.
Bo ów z więzieniá wychodzi / áby królowáł / á ten y w króleſtwie ſwojim zubożeć może.
Widziáłem wƺyſtkich żyjących / którzy chodzą pod ſłońcem / że przeſtáwáli z chłopięćiem / potomkiem onego / który miáł náſtąpić ná króleſtwo po nim.
Nie było końcá nieſtátkowi wƺyſtkiego ludu / którykolwiek był przed nimi ; nie będąć śię potomkowie ćieƺyć z niego. A ták y to jeſt márność / y utrápienie duchá.
Strzeż nogi twojej / gdy idzieƺ do domu Bożego / á bądź ſkłonniejƺym ku ſłuchániu / niżeli ku dáwániu ofiár ludzi głupich ; boć oni nie wiedzą / że źle cżynią.
