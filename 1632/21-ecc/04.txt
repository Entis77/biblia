Potemem śię obróćił y ujrzáłem wƺyſtkie ućiſki / które śię dźieją pod ſłońcem / á oto widźiáłem łzy ućiśnionych / którzy nie máją poćieƺyćielá / áni mocy / áby uƺli rąk tych / którzy ich ćiemiężą ; á nie máją / mówię / poćieƺyćielá.
Dlátegom ja umárłych / którzy już zeƺli / więcey chwálił / niżeli żywych / którzy jeƺcże áż dotąd żyją.
Owƺem ƺcżęśliwy jeſt nád tych obydwóch ten / który jeƺcże nie był / który nie widźiał nic złego / które śię dźieje pod ſłońcem.
Bom widźiał / że wƺeláká prácá y káżde dźieło dobre jeſt ku zázdrośći jednych drugim. Y toć jeſt márność y utrápienie duchá.
Głupi ſkłádá ręce ſwe / á je ćiáło ſwoje.
Lepƺá jeſt pełna gárść z pokojem / niżeli obie gárśći pełne z prácą y z udręcżeniem duchá.
Znowu obróćiwƺy śię ujrzáłem drugą márność pod ſłońcem.
Jeſt kto ſámotny / niemájąc żádnego / áni ſyná / áni brátá / á wżdy niemáƺ końcá wƺelákiej prácy jego / áni ocży jego mogą śię náſyćić bogáctwem. Nie myśli : Komuż ja prácuję / ták że y żywotowi ſwemu ujmuję dobrego. Y toć jeſt márność / y ćiężkie udręcżenie.
Lepiej jeſt we dwu być / niż jednemu ; máją záiſte dobry pożytek z prácy ſwojey.
Bo jeſli jeden upádnie / drugi podźwignie towárzyƺá ſwego. A ták biádá ſámotnemu / gdyby upádł! bo nie ma drugiego / coby go podźwignął.
Tákże będąli dwáj ſpołu leżeć / zágrzeją śię ; ále jeden jákoż śię zágrzeje?
Owƺem jeſliby kto jednego przemágał / dwáj mu śię záſtáwią ; á ƺnur troiſty nie łácno śię zerwie.
Lepƺy jeſt chłopiec ubogi á mądry / niżeli król ſtáry á głupi / który już nie umie przyjmowáć nápominánia.
Bo ów z więźieniá wychodźi / áby królował / á ten y w króleſtwie ſwojim zubożeć może.
Widźiáłem wƺyſtkich żyjących / którzy chodzą pod ſłońcem / że przeſtáwáli z chłopięćiem / potomkiem onego / który miał náſtąpić ná króleſtwo po nim.
Nie było końcá nieſtátkowi wƺyſtkiego ludu / którykolwiek był przed nimi ; nie będąć śię potomkowie ćieƺyć z niego. A ták y to jeſt márność / y utrápienie duchá.
Strzeż nogi twojey / gdy idźieƺ do domu Bożego / á bądź ſkłonniejƺym ku ſłuchániu / niżeli ku dáwániu ofiár ludźi głupich ; boć oni nie wiedzą / że źle cżynią.
