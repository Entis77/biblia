Zápráwdęm to wƺyſtko uwáżáł w ſercu ſwem / ábym to wƺyſtko objáśnił / że ſpráwiedliwi y mądrzy z ſpráwámi ſwemi ſą w rękách Bożych / á iż áni miłośći / áni nienáwiśći nie zná cżłowiek ze wƺyſtkich rzecży / które ſą przed oblicżem jego.
Wƺyſtko śię dźieje jednákowo wƺyſtkim ; jednoż przychodźi ná ſpráwiedliwego y niezbożnego / ná dobrego y ná cżyſtego y niecżyſtego / ná ofiárującego y ná tego / który nie ofiáruje ; ná dobrego / y ná grzeƺnego / ná przyśięgájącego / y ná tego / co śię przyśięgi boi.
A toć jeſt nájgorƺá między wƺyſtkiem / co śię dźieje pod ſłońcem / iż jednoż przychodźi ná wƺyſtkich ; á owƺem / że ſerce ſynów ludzkich pełne jeſt złego / á iż głupſtwo trzymá śię ſercá ich zá żywotá ich / á potem idą do umárłych.
Abowiem ktokolwiek śię towárzyƺy ze wƺyſtkimi żywymi / má nádźieję / ( Gdyż y pieſ żywy lepƺy jeſt / niż lew zdechły ; )
Boć ći / co żyją / wiedzą / że umrzeć máją ; ále umárli o nicżem nie wiedzą / y nie máją więcey żádney zápłáty / gdyż w zápámiętánie przyƺłá pámiątká ich.
Owƺem y miłość ich / y zázdrość ich y nienáwiść ich już zginęłá / á nie máją więcey dźiáłu ná wieki we wƺyſtkiem / co śię dźieje pod ſłońcem.
Idźże tedy / jedz z rádośćią chleb twój / á pij z dobrą myślą wino twoje ; ábowiem już wdźięcżne ſą Bogu ſpráwy twoje.
Ná káżdy cżáſ niech będą ƺáty twoje biáłe / á olejku ná głowie twojey niech śię nie przebierá.
Záżywáj żywotá z żoną / którąś umiłowáł / po wƺyſtkie dni żywotá márnośći twojey / któreć dáł Bóg pod ſłońcem po wƺyſtkie dni márnośći twojey ; boć ten jeſt dźiáł twój w żywoćie twoim y w prácy twojey / którą podejmujeƺ pod ſłońcem.
Wƺyſtko / co przedśięweźmie ręká twojá do cżynieniá / cżyń według możnośći twojey / ábowiem niemáƺ żádney prácy / áni myśli / áni umiejętnośći / áni mądrośći w grobie / do którego ty idźieƺ.
Potem obróćiwƺy śię ujrzáłem pod ſłońcem / że bieg nie jeſt w mocy prędkich / áni wojná w mocy mężnych / áni żywność w mocy mądrych / áni bogáctwo w mocy roztropnych / áni łáſká w mocy pomyślnych ; ále cżáſ y tráfunek wƺyſtko przynośi.
Bo cżłowiek nie wie cżáſu ſwego ; ále jáko ryby / które bywáją łowione śiećią ƺkodliwą / y jáko ptáki łápáne bywáją śidłem : ták ułowieni bywáją ſynowie ludzcy we zły cżáſ / gdy ná nie nágle przypádá.
Nádto widźiáłem y tę mądrość pod ſłońcem / która jeſt wielká u mnie :
Miáſto máłe / á w nim ludźi máło / przećiw któremu przyćiągnął król możny / y obległ je / y uſypáł przećiwko niemu wáły wielkie ;
Y ználázł śię w nim mąż ubogi mądry / który wybáwił miáſto ono mądrośćią ſwoją : choć nikt nie wſpomniáł ná onego mężá ubogiego.
Przetożem ja rzekł : Lepƺá jeſt mądrość / niżeli moc / ácżkolwiek mądrość onego ubogiego byłá wzgárdzoná / y ſłów jego nie ſłucháli.
Słów ludźi mądrych ſpokojnie ſłucháć náleży / rácżej niż krzyku pánującego między głupimi.
Lepƺá jeſt mądrość niż oręże wojenne ; ále jeden grzeƺnik pſuje wiele dobrego.
