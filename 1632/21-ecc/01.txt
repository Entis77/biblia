Słowá káznodźiei / ſyná Dawidowego / królá w Jeruzálemie.
Márność nád márnośćiámi / powiedźiał káznodźiejá ; márność nád márnośćiámi / y wƺyſtko márność.
Cóż zá pożytek ma cżłowiek ze wƺyſtkiej prácy ſwej / którą prowádźi pod ſłońcem?
Jeden rodzáj przemijá / á drugi rodzáj náſtáje ; lecż źiemiá ná wieki ſtoi.
Słońce wſchodźi / y ſłońce záchodźi / á ſpieƺy śię do miejſcá ſwego / kędy wſchodźi ;
Idźie ná południe / á obrácá śię ná północy ; wiátr uſtáwicżnie krążąc idźie / á po okręgách ſwojich wrácá śię wiátr.
Wƺyſtkie rzeki idą do morzá / wƺákże morze nie wylewá ; do miejſcá / z którego rzeki płyną / wrácáją śię / áby záś ſtámtąd wychodźiły.
Wƺyſtkie rzecży ſą pełne zábáw / á cżłowiek nie może ich wymówić ; oko nie náſyći śię widzeniem / á ucho nie nápełni śię ſłyƺeniem.
Co było / jeſt to / co być ma ; á co śię teraz dźieje / jeſt to / co śię dźiáć będźie / á niemáƺ nic nowego pod ſłońcem.
Jeſtże jáká rzecż / o którejby kto rzec mógł : Wej! to coś nowego? Y toć już było zá onych wieków / które były przed námi.
Niemáƺ pámiątki pierwƺych rzecży ; tákże też y potomnych / które będą / nie będźie pámiątki u tych / którzy potem náſtáną.
Ja káznodźiejá byłem królem Izráelſkim w Jeruzálemie ;
Y przyłożyłem do tego ſerce ſwe / ábym ƺukał / y doƺedł mądrośćią ſwoją wƺyſtkiego / co śię dźieje pod niebem. ( Tę zábáwę trudną dał Bóg ſynom ludzkim / áby śię nią trápili. )
Widźiáłem wƺyſtkie ſpráwy / które śię dźieją pod ſłońcem / á oto wƺyſtko jeſt márnośćią y utrapieniem duchá.
Co jeſt krzywego / nie może być wyproſtowáne / á niedoſtátki nie mogą być policżone.
Przetoż tákiem myślał w ſercu ſwem / mówiąc : Otom ja uwielbił y rozƺerzył mądrość nád wƺyſtkich / którzy byli przedemną w Jeruzálemie / á ſerce moje doſtąpiło wielkiej mądrośći y umiejętnośći.
Y przyłożyłem do tego ſerce moje / ábym poznał mądrość y umiejętność / ƺáleńſtwo y głupſtwo ; álem doznał / iż to jeſt utrapieniem duchá.
Bo gdźie wiele mądrośći / tám jeſt wiele gniewu ; á kto przycżyniá umiejętnośći / przycżyniá boleśći.
