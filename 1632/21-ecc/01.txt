Słowá káznodziei / ſyná Dawidowego / królá w Jeruzálemie.
Márność nád márnośćiámi / powiedziáł káznodziejá ; márność nád márnośćiámi / y wƺyſtko márność.
Cóż zá pożytek má cżłowiek ze wƺyſtkiej prácy ſwej / którą prowádzi pod ſłońcem?
Jeden rodzáj przemijá / á drugi rodzáj náſtáje ; lecż ziemiá ná wieki ſtoi.
Słońce wſchodzi / y ſłońce záchodzi / á ſpieƺy śię do miejſcá ſwego / kędy wſchodzi ;
Idzie ná południe / á obrácá śię ná północy ; wiátr uſtáwicżnie krążąc idzie / á po okręgách ſwoich wrácá śię wiátr.
Wƺyſtkie rzeki idą do morzá / wƺákże morze nie wylewá ; do miejſcá / z którego rzeki płyną / wrácáją śię / áby záś ſtámtąd wychodziły.
Wƺyſtkie rzecży ſą pełne zábáw / á cżłowiek nie może ich wymówić ; oko nie náſyći śię widzeniem / á ucho nie nápełni śię ſłyƺeniem.
Co było / jeſt to / co być má ; á co śię teráz dzieje / jeſt to / co śię dziáć będzie / á niemáƺ nic nowego pod ſłońcem.
Jeſtże jáká rzecż / o którejby kto rzec mógł : Wej! to coś nowego? Y toć już było zá onych wieków / które były przed námi.
Niemáƺ pámiątki pierwƺych rzecży ; tákże też y potomnych / które będą / nie będzie pámiątki u tych / którzy potem náſtáną.
Já káznodziejá byłem królem Izráelſkim w Jeruzálemie ;
Y przyłożyłem do tego ſerce ſwe / ábym ƺukáł / y doƺedł mądrośćią ſwoją wƺyſtkiego / co śię dzieje pod niebem. ( Tę zábáwę trudną dáł Bóg ſynom ludzkim / áby śię nią trápili. )
Widziáłem wƺyſtkie ſpráwy / które śię dzieją pod ſłońcem / á oto wƺyſtko jeſt márnośćią y utrápieniem duchá.
Co jeſt krzywego / nie może być wyproſtowáne / á niedoſtátki nie mogą być policżone.
Przetoż tákiem myśláł w ſercu ſwem / mówiąc : Otom já uwielbił y rozƺerzył mądrość nád wƺyſtkich / którzy byli przedemną w Jeruzálemie / á ſerce moje doſtąpiło wielkiej mądrośći y umiejętnośći.
Y przyłożyłem do tego ſerce moje / ábym poznáł mądrość y umiejętność / ƺáleńſtwo y głupſtwo ; álem doznáł / iż to jeſt utrápieniem duchá.
Bo gdzie wiele mądrośći / tám jeſt wiele gniewu ; á kto przycżyniá umiejętnośći / przycżyniá boleśći.
