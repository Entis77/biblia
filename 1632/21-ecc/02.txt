Rzekłem já do ſercá mego : Nuże teráz doświádcżę ćię w weſelu / używájże dobrych rzecży ; ále y toć márność.
Śmiechowi rzekłem : Szálejeƺ / á weſelu : Cóż to cżyniƺ?
Przemyśláłem w ſercu ſwem / ábym pozwolił winá ćiáłu memu / ( ſerce jednák ſwoje ſpráwując mądrośćią ) y ábym śię trzymáł głupſtwá dotąd / áżbym obácżył / coby lepƺego było ſynom ludzkim cżynić pod niebem / przez wƺyſtkie dni żywotá ich.
Wielkiem ſpráwy wykonáł ; pobudowáłem ſobie domy / náſádziłem ſobie winnic ;
Nácżyniłem ſobie ogrodów / y ſádów / y náƺcżepiłem w nich drzew wƺelákiego owocu ;
Pobudowáłem ſobie ſtáwy ku odwilżániu przez nie láſu / w którym rośnie drzewo ;
Nábyłem ſobie ſług y dziewek / y miáłem cżeládź w domu moim ; do tego y ſtádá wołów / y wielkie trzody owiec miáłem nád wƺyſtkich / którzy byli przedemną w Jeruzálemie.
Zgromádziłem też ſobie ſrebro y złoto / y klejnoty od królów y kráin. Sporządziłem też ſobie śpiewáków y śpiewácżki / y inne rozkoƺe ſynów ludzkich / y muzycżne nácżyniá rozlicżne.
A ták ſtáłem śię wielkim y możniejƺym nád wƺyſtkich / którzy byli przede mną w Jeruzálemie ; nádto mądrość mojá zoſtáwáłá przy mnie.
A wƺyſtkiego / cżego pożądáły ocży moje / nie zábrániáłem im / ánim odmáwiáł ſercu memu żádnego weſelá ; ále ſerce moje weſeliło śię ze wƺyſtkiej prácy mojej. A toć był dziáł mój ze wƺyſtkiej prácy mojej.
Lecż gdym śię obejrzáł ná wƺyſtkie ſpráwy ſwoje / które cżyniły ręce moje / y ná práce / którem podejmowáł prácując : oto wƺyſtko márność / y utrápienie duchá / y niemáƺ nic pożytecżnego pod ſłońcem.
Przetoż obróćiłem śię do tego / ábym śię przypátrywáł mądrośći / y ƺáleńſtwu / y głupſtwu ; ( bo cóżby cżłowiek cżynił ten / który náſtánie po królu? to / co już inni cżynili. )
Y obácżyłem / iż jeſt pożytecżniejƺá mądrość niżeli głupſtwo / ták jáko jeſt pożytecżniejƺá świátłość / niżeli ćiemność.
Mądry má ocży w głowie ſwej / ále głupi w ćiemnośćiách chodzi ; á wƺákżem poznáł / że jednákie przygody ná wƺyſtkich przychdzą.
Dlátegom rzekł w ſercu mojem : Máli mi śię ták dziáć / jáko śię głupiemu dzieje / przecżżem go já tedy mądrośćią przeƺedł? Przetożem rzekł w ſercu mojem : Y toć jeſt márność.
Abowiem nie ná wieki będzie pámiątki mądrego y głupiego / dlátego / iż to / co teráz jeſt / we dni przyƺłe wƺyſtkiego zápomną ; á jáko umierá mądry / ták y głupi.
Przetoż mi żywot omierzł ; bo mi śię nie podobá żádná rzecż / którá śię dzieje pod ſłońcem ; ábowiem wƺyſtkie ſą márnośćią / y utrápieniem duchá.
Náwet omierzłá mi y wƺyſtká prácá mojá / którąm podejmowáł pod ſłońcem / przeto / że ją zoſtáwić muƺę cżłowiekowi / który náſtánie po mnie.
A kto wie / będzieli mądrym / cżyli głupim? á wƺákże będzie pánowáł nád wƺyſtką prácą moją / którąm prowádził / y w którejm był mądry pod ſłońcem. Aleć y to márność.
Y przypádłem ná to / ábym zwątpił w ſercu mojem o wƺyſtkiej prácy / którąm śię mądrze báwił pod ſłońcem.
Nie jeden záiſte cżłowiek prácuje mądrze / y umiejętnie / y ſpráwiedliwie ; á wƺákże to innemu / który nie robił ná to / zá dziáł zoſtáwi. Y toć márność y wielká biedá.
Bo cóż má cżłowiek ze wƺyſtkiej prácy ſwej / y z uśiłowániá ſercá ſwego / które podejmuje pod ſłońcem?
Poniewáż wƺyſtkie dni jego ſą boleſne / á zábáwá jego jeſt fráſunek / ták iż y w nocy nie odpocżywá ſerce jego. Y toć jeſt márność.
Izáli nie lepƺá cżłowiekowi / áby jádł y pił / y dobrze ucżynił duƺy ſwojej z prácy ſwojej? álemći widziáł / że y to z ręki Bożej pochodzi.
Abowiem któżby ſłuƺniej miáł jeść / y pożywáć tego nád mię?
Bo cżłowiekowi / który mu śię podobá / dáje mądrość / umiejętność / y weſele ; ále grzeƺnikowi dáje fráſunek / áby zbieráł y zgromádzáł / coby zoſtáwił temu / który śię podobá Bogu. Y toć jeſt márność / á utrápienie duchá.
