Mądrość cżłowieká oświecá oblicże jego / á hárdość twárzy jego odmieniá.
Jáć rádzę / ábyś wyroku królewſkiego przeſtrzegáł á wƺákże według przyśięgi Bożej.
Nie ſkwápiáj śię odejść od oblicżá jego / áni trwáj w uporze ; ábowiem cobykolwiek chćiáł / ucżyniłćiby.
Bo gdźie ſłowo królewſkie / tám y moc jego : á któż mu rzecże : Co cżyniƺ?
Kto ſtrzeże przykázániá / nie uzná nic złego ; y cżáſ y przycżyny zná ſerce mądrego.
Abowiem wƺelki zámyſł má cżáſ y przycżyny ; áleć wielká biedá trzymá śię cżłowieká /
Że nie wie / co má być ; bo kiedy śię co ſtánie / któż mu oznájmi?
Niemáƺ cżłowieká / coby miáł moc nád żywotem / żeby záhámowáł duƺę / áni má mocy náde dniem śmierći ; áni má / cżemby śię bronił w tym boju / áni wyſwobodźi niezbożnego niepobożność.
Tom wƺyſtko widźiáł / gdym przyłożył ſerce ſwoje do tego wƺyſtkiego / co śię pod ſłońcem dźieje ; widźiáłem ten cżáſ / którego pánuje cżłowiek nád cżłowiekiem ná jego złe.
Tedym widźiáł niezbożnych pogrzebionych / że śię záś náwróćili ; ále którzy z miejſcá świętego odeƺli / przyƺli w zápámiętánie w onem mieśćie / w którym dobrze cżynili. Y toć jeſt márność.
Bo iż nie záráz wychodźi dekret ná złe ſpráwy / przetoż ná tem jeſt wƺyſtko ſerce ſynów ludzkich / áby cżynili złe rzecży.
A choćiáż grzeƺnik ſto kroć źle cżyni / y odwłácżá mu śię / wƺákże já wiem / że dobrze będźie bojącym śię Bogá / którzy śię boją oblicżá jego.
Ale niezbożnemu nie dobrze będźie / áni śię przedłużą dni jego / owƺem pomijá jáko ćień / przeto / iż śię nie boi oblicżá Bożego.
Jeſt też márność / którá śię dźieje ná źiemi / że bywáją ſpráwiedliwi / którym śię ták powodźi / jákoby cżynili ucżynki niepobożnych ; záśię bywáją niepobożni / którym śię ták powodźi / jákoby cżynili ucżynki ſpráwiedliwych. Przetożem rzekł : Y toć jeſt márność.
A ták chwáliłem weſele / przeto / iż niemáƺ nic lepƺego cżłowiekowi pod ſłońcem / jedno jeść / y pić / y weſelić śię / á iż mu jedno to zoſtáje z prácy jego po wƺyſtkie dni żywotá jego / które mu Bóg dáł pod ſłońcem.
A choćiáżem udáł ſerce ſwe ná to / ábym doƺedł mądrośći / y zrozumiáł kłopoty / które śię dźieją ná źiemi / dlá których cżłowiek áni we dnie áni w nocy nie śpi :
A wƺákże widźiáłem przy káżdym ucżynku Bożym / że nie może cżłowiek dośćignąć ſpráwy / którá śię dźieje pod ſłońcem. Stáráć śię cżłowiek chcąc tego dojść / ále nie dochodźi ; owƺem choćby rzekł mądry / że śię chce dowiedźieć / nie będźie mógł ználeść.
