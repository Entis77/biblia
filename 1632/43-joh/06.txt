Potym odƺedł JEzus zá morze Gálilejſkie / <i>które jeſt</i> Tyberyádſkie.
Y ƺedł zá nim lud wielki / iż widźieli cudá jego które cżynił nád chorymi.
Y wƺedł JEzus ná górę / y śiedźiał tám z ucżniámi ſwojimi.
A byłá bliſko Wielkanoc / Święto Żydowſkie.
Tedy podniózƺy JEzus ocżu / y ujrzawƺy iż wielki lud idźie do niego / rzekł do Filipá : Zkąd kupiemy chlebá áby ći jedli?
( Ale <i>to</i> mówił kuƺąc go ; bo on wiedźiał co miał cżynić. )
Odpowiedźiał mu Filip ; Zá dwie śćie groƺy chlebá nie doſyć im będźie / choćby káżdy z nich máło co wźiął.
Rzekł mu jeden z ucżniów jego / Andrzyj brát Symoná Piotrá ;
Jeſt tu jedno pácholę / co ma pięćioro chlebá jęcżmiennego / y dwie rybce ; ále cóż to jeſt ná ták wielu?
Tedy rzekł JEzus : Każćie ludowi uśieść ; á było trawy doſyć ná onymże miejſcu. Y uśiádło mężów w licżbie około piąći tyśięcy.
Wźiął tedy JEzus one chleby ; á podźiękowawƺy / rozdał ucżniom / á ucżniowie śiedzącym ; tákże y z onych rybek / ile jedno chćieli.
A gdy byli náſyceni / rzekł ucżniom ſwojim ; Zbierzćie te ułomki które zbywáją / żeby nic nie zginęło.
Y zebráli / y nápełnili dwánaśćie koƺów ułomków z onego pięćiorgá chlebá jęcżmiennego / które zbywáły tym co jedli.
A oni ludźie / ujrzawƺy cud który ucżynił JEzus / mówili ; Tenći jeſt záprawdę on Prorok / który miał przyść ná świát.
Tedy JEzus poznawƺy / iż mieli przyść y porwáć go / áby go ucżynili Królem / uƺedł záśię ſam tylko ná górę.
A gdy był wiecżór / zſtąpili ucżniowie jego do morzá.
A wſtąpiwƺy w łódź / jácháli zá morze do Kápernáum / á już było ćiemno ; á JEzus nie przyƺedł był do nich.
A morze / gdy powſtał wielki wiátr / burzyć śię pocżynáło.
Gdy tedy odpłynęli / jákoby ná dwudźieſtu y piąći / álbo trzydźieſtu ſtaj / ujrzeli JEzuſá chodzące° po morzu / przybliżájącego śię ku łodźi ; y ulękli śię.
A on im rzekł ; Jamći jeſt / nie bójćie śię.
Y wźięli go ochotnie do łodźi : á zárázem łódź przypłynęłá do źiemie / do którey jácháli.
Názájutrz lud który był zá morzem / widząc że tám nie było drugiey łodźi tylko oná jedná / w którą byli wſtąpili ucżniowie jego / á iż JEzus nie wƺedł był w łódź <i>z</i> ucżniámi ſwojimi / ále ſámi ucżniowie jego ujácháli :
( Przyƺły też były drugie łodźie z Tyberyády / bliſko do onego miejſcá gdźie jedli chleb / gdy był PAn dźięki ucżynił. )
To gdy obacżył lud iż tám nie było JEzuſá / áni ucżniów jego ; wſtąpili y oni w łodźie / y przepráwili śię do Kápernáum / ƺukájąc JEzuſá.
A ználazƺy go zá morzem / rzekli mu ; Miſtrzu / kiedyś tu przybył?
Odpowiedźiał im JEzus / y rzekł ; Záprawdę záprawdę powiádam wam ; Szukaćie mię nie przeto / iżeśćie widźieli cudá / ále iżeśćie jedli chleb / y byliśćie náſyceni.
Spráwujćież nie pokarm który ginie / ále pokarm który trwa ku żywotowi wiecżnemu / który wam da Syn cżłowiecży. Abowiem tego zápiecżętował Bóg Oćiec.
Rzekli tedy do niego : Cóż będźiemy cżynili / ábyſmy ſpráwowáli ſpráwy Boże?
Odpowiedźiał JEzus / y rzekł im ; Toć jeſt ſpráwá Boża / ábyśćie wierzyli w <i>tego,</i> którego on poſłał.
Rzekli mu tedy ; Cóż wżdy ty zá znák cżyniƺ / ábyſmy widźieli y wierzyli tobie? Cóż cżyniƺ?
Ojcowie náƺy jedli mánnę ná puƺcży / jáko jeſt nápiſano ; Chleb z niebá dał im ku jedzeniu.
Rzekł im tedy JEzus : Záprawdę záprawdę powiádam wam / Nie Mojzeƺci wam dał chleb z niebá / ále Oćiec mój dáje wam chleb on prawdźiwy z niebá.
Abowiem chleb Boży ten jeſt / który zſtępuje z niebá / y żywot dawa świátu.
Tedy mu rzekli : PAnie daj nam záwżdy tego chlebá.
Y rzekł im JEzus : Jamći jeſt on chleb żywotá / kto do mnie przychodźi / łáknąć nie będźie : á kto wierzy w mię / nigdy prágnąć nie będźie.
Alem wam powiedźiał : Owƺem widźieliśćie mię / á nie wierzyćie.
Wƺyſtko co mi dawa Oćiec / do mnie przydźie : á tego co do mnie przydźie / nie wyrzucę precż.
Bom zſtąpił z niebá / nie iżbym cżynił wolą moję / ále wolą onego który mię poſłał.
A táć jeſt wola onego który mię poſłał / Ojcá : ábym z tego wƺyſtkiego co mi dał / nic nie ztráćił : ále ábym to wzbudźił <i>w on</i> oſtátecżny dźień.
A táć jeſt wola onego który mię poſłał / áby káżdy kto widźi Syná / á wierzy weń / miał żywot wiecżny : á Ja go wzbudzę <i>w on</i> oſtátecżny dźień.
Y ƺemráli Żydowie o nim / iż rzekł : Jam jeſt on chleb / którym z niebá zſtąpił.
Y mówili : Izaż ten nie jeſt JEzus / ſyn Józefów / którego my Ojcá y mátkę znamy? Jákoż tedy ten powiáda : Żem z niebá zſtąpił?
Tedy odpowiedźiał JEzus / y rzekł im : Nie ƺemrzyćie miedzy ſobą.
Żaden do mnie przyść nie może / jeſli go Oćiec <i>mój</i> który mię poſłał / nie poćiągnie : á Ja go wzbudzę w oſtátecżny dźień.
Nápiſano w Prorocech : Y będą wƺyſcy wyucżeni od Bogá. Przetoż káżdy kto ſłyƺał od Ojcá / á náucżył śię / przychodźi do mnie.
Nie iżby kto widźiał Ojcá / oprócż tego / który jeſt od Bogá : ten widźiał Ojcá.
Záprawdę záprawdę powiádam wam : Kto w mię wierzy ma żywot wiecżny.
Jam jeſt on chleb żywotá.
Ojcowie wáƺy jedli mánnę ná puƺcży / á pomárli.
Ten jeſt on chleb / który z niebá zſtępuje : jeſliby go kto jadł / nie umrze.
Jamći jeſt chleb on żywy / którym z niebá zſtąpił : jeſliby kto jadł z tego chlebá / żyć będźie ná wieki. A chleb / który Ja dam / jeſt ćiáło moje : które Ja dam zá żywot świátá.
Wádźili śię tedy Żydowie miedzy ſobą / mówiąc : Jákoż ten może nam dáć ćiáło ſwoje ku jedzeniu?
Y rzekł im JEzus : Záprawdę záprawdę powiádam wam / Jeſli nie będźiećie jedli ćiáłá Syná cżłowiecżego / y pili krwie jego / nie maćie żywotá w ſobie.
Kto je ćiáło moje / á pije krew moję / ma żywot wiecżny : á Ja go wzbudzę w on oſtátecżny dźień.
Abowiem ćiáło moje / prawdźiwie jeſt pokarm : á krew mojá / prawdźiwie jeſt napój.
Kto je ćiáło moje / y pije krew moję / we mnie mieƺka á Ja w nim.
Jáko mię poſłał żywiący Oćiec / y Ja żywię przez Ojcá : <i>ták</i> kto mnie pożywa / y on żyć będźie przez mię.
Tenći jeſt chleb on / który z niebá zſtąpił. Nie jáko Ojcowie wáƺy jedli mánnę á pomárli : Kto je ten chleb / żyć będźie ná wieki.
To mówił w bóżnicy ucżąc w Kápernáum.
Wiele ich tedy z ucżniów jego ſłyƺąc to / mówili : Twárdać to jeſt mowá / któż jey ſłucháć może?
Ale wiedząc JEzus ſam w ſobie / iż o tym ƺemráli ucżniowie jego / rzekł im : Toż was obraża?
Cóż gdybyśćie ujrzeli Syná cżłowiecżego wſtępującego / gdźie był pierwey?
Duchći jeſt / który ożywia / ćiáło nic nie pomaga : Słowá które Ja wam mówię / Duch ſą / y żywot ſą.
Ale ſą niektórzy z was / co nie wierzą. Abowiem wiedźiał od pocżątku JEzus / którzy byli co nie wierzyli ; y kto jeſt co go miał wydáć.
Y mówił : Dla tegomći wam powiedźiał / Iż żaden nie może przyść do mnie / jeſliby mu nie było dano od Ojcá mojego.
Od tego cżáſu wiele ucżniów jego odeƺło ná zad / á więcey z nim nie chodźili.
Tedy rzekł JEzus do <i>onych</i> dwunaśći ; Izali y wy chcećie odejść?
Y odpowiedźiał mu Symon Piotr ; PAnie / do kogóż pójdźiemy? <i>Ty</i> maƺ ſłowá żywotá wiecżnego.
A myſmy uwierzyli y poználi / żeś ty jeſt CHryſtus / on Syn Bogá żywego.
Odpowiedźiał im JEzus ; Izalim Ja nie dwunaśći was obrał / á jeden z was jeſt Dyabeł.
A to mówił o Judaƺu / <i>ſynu</i> Symoná / Iƺkáryoćie : bo go ten wydáć miał będąc jednym z onych dwunaśći.
