Potym odƺedł JEzuſ zá morze Gálilejſkie / które <i>jeſt</i> Tyberyádzkie ;
Y ƺedł zá nim lud wielki / iż widźieli cudá jego które cżynił nád chorymi.
Y wƺedł JEzuſ ná gorę / y śiedźiáł tám z ucżniámi ſwoimi ;
A byłá bliſko Wielkanoc / Święto Żydowſkie.
Tedy podniozƺy JEzuſ ocżu y ujrzáwƺy / iż wielki lud idźie do niego / rzekł do Filipá : Zkąd kupiemy chlebá áby ći jedli?
( Ale <i>to</i> mowił kuƺąc go ; bo on wiedźiáł co miáł cżynić. )
Odpowiedźiáł mu Filip ; Zá dwie śćie groƺy chlebá nie doſyć im będźie / choćby káżdy z nich máło co wźiął.
Rzekł mu jeden z ucżniów jego / Andrzej brát Szymoná Piotrá ;
Jeſt tu jedno pácholę / co má pięćioro chlebá jęcżmiennego / y dwie rybce ; ále coż to jeſt ná ták wielu?
Tedy rzekł JEzuſ : Káżćie ludowi uśieść ; á było tráwy doſyć ná onymże miejſcu. Y uśiádło mężów w licżbie około pięći tyśięcy.
Wźiął tedy JEzuſ one chleby ; á podźiękowáwƺy rozdáł ucżniom / á ucżniowie śiedzącym ; tákże y z onych rybek / ile jedno chćieli.
A gdy byli náſyceni / rzekł ucżniom ſwojim ; Zbierzćie te ułomki które zbywáją / żeby nic nie zginęło.
Y zebráli / y nápełnili dwánaśćie koƺów ułomków z onego pięćiorgá chlebá jęcżmiennego / które zbywáły tym co jedli.
A oni ludźie / ujrzáwƺy cud który ucżynił JEzuſ / mowili ; Tenći jeſt záprawdę on Prorok / który miáł przyść ná świát.
Tedy JEzuſ poznawƺy / iż mieli przyść y porwáć go / áby go ucżynili Królem / uƺedł záśię ſám tylko ná gorę.
A gdy był wiecżor / zſtąpili ucżniowie jego do morzá.
A wſtąpiwƺy w łodź / jácháli zá morze do Kápernáum / á już było ćiemno ; á JEzuſ nieprzyƺedł był do nich.
A morze / gdy powſtáł wielki wiátr / burzyć śię pocżynáło.
Gdy tedy odpłynęli / jákoby ná dwudźieſtu y pięći álbo trzydźieſtu ſtáaj / ujrzeli JEzuſá chodzące° po morzu / przybliżájącego śię ku łodźi ; y ulękli śię.
A on im rzekł : Jamći jeſt / nie bojćie śię.
Y wźięli go ochotnie do łodźi : á zárázem łodź przypłynęłá do źiemie / do którey jácháli.
Názájutrz lud który był zá morzem / widząc że tám nie było drugiey łodźi / tylko oná jedná / w którą byli wſtąpili ucżniowie jego / á iż JEzuſ nie wƺedł był w łodź z ucżniámi ſwojimi / ále ſámi ucżniowie jego ujácháli :
( Przyƺły też były drugie łodźie z Tyberyády / bliſko do onego miejſcá / gdźie jedli chleb / gdy był PAn dźięki ucżynił. )
To gdy obacżył lud iż tám nie było JEzuſá / áni ucżniów jego ; wſtąpili y oni w łodźie y przepráwili śię do Kápernáum / ƺukájąc JEzuſá.
A ználazƺy go zá morzem / rzekli mu ; Miſtrzu / kiedyś tu przybył?
Odpowiedźiał im JEzuſ / y rzekł ; Záprawdę záprawdę powiádam wam ; Szukaćie mię nie przeto / iżeśćie widźieli cudá ále iżeśćie jedli chleb / y byliśćie náſyceni.
Spráwujćież nie pokarm / który ginie / ále pokarm który trwa ku żywotowi wiecżnemu który wám da Syn cżłowiecży. Albowiem tego zápiecżętował Bog Oćiec.
Rzekli tedy do niego : Coż będźiemy cżynili / ábyśmy ſpráwowáli ſpráwy Boże?
Odpowiedźiał JEzuſ / y rzekł im ; Toć jeſt ſpráwá Boża / ábyśćie wierzyli w <i>tego /</i> którego on poſłał.
Rzekli mu tedy ; Coż wżdy ty zá znák cżyniƺ / ábyśmy widźieli y wierzyli tobie? Coż cżyniƺ?
Ojcowie náƺy jedli mánnę ná puƺcży / jáko jeſt nápiſáno ; Chleb z niebá dáł im ku jedzeniu.
Rzekł im tedy JEzuſ : Záprawdę / záprawdę powiádam wam : Nie Mojżeƺci wam dał chleb z niebá / ále Oćiec mój dáje wam chleb on prawdźiwy z niebá.
Abowiem chleb Boży ten jeſt / który zſtępuje z niebá / y żywot dawa świátu.
Tedy mu rzekli : Pánie dáj nám záwżdy tego chlebá.
Y rzekł im JEzuſ : Jamći jeſt on chleb żywotá / kto do mnie przychodźi / łáknąć nie będźie : á kto wierzy w mię / nigdy prágnąć nie będźie.
Alem wám powiedźiáł : Owƺem / widźieliśćie mię / á nie wierzyćie.
Wƺyſtko co mi dwa Oćiec / do mnie przydźie : á tego co do mnie przydźie / nie wyrzucę precż.
Bom zſtąpił z niebá / nie iżbym cżynił wolę moję / ále wolę onego który mię poſłał.
A táć jeſt wola onego który mię poſłáł / Ojcá : ábym z tego wƺyſtkiego co mi dał / nic nie ſtráćił : ále ábym to wzbudźił <i>w on</i> oſtátecżny dźień.
A táć jeſt wola onego który mię poſłał / áby káżdy kto widźi Syná / á wierzy weń / miał żywot wiecżny : á Ja go wzbudzę <i>w on</i> oſtátecżny dźień.
Y ƺemráli Żydowie o nim / iż rzekł : Jam jeſt on chleb / którym z niebá zſtąpił.
Y mowili : Izáż ten nie jeſt JEzuſ / ſyn Jozefów / którego my Ojcá y mátkę znamy? Jákoż tedy ten powiáda : Żem z niebá zſtąpił?
Tedy odpowiedźiáł JEzuſ / y rzekł im : Nie ƺemrzyćie między ſobą.
Żaden do mnie przyść nie może / jeśli go Oćiec <i>moj</i> który mię poſłáł / nie poćiągnie : á Ja go wzbudzę w oſtátecżny dźień.
Nápiſáno w Prorokách : y będą wƺyſcy wyucżeni od Bogá. Przetoż káżdy kto ſłyƺał od Ojcá / á náucżył śię / przychodźi do mnie.
Nie iżby kto widźiał Ojcá / oprocż tego / który jeſt od Bogá : ten widźiał Ojcá.
Záprawdę záprawdę powiádam wám : Kto w mię wierzy / má żywot wiecżny.
Jam jeſt on chleb żywotá.
Ojcowie wáƺy jedli mánnę ná puƺcży / á pomárli.
Ten jeſt on chleb / który z niebá zſtępuje : jeśliby go kto jadł / nie umrze.
Jamći jeſt chleb on żywy / którym z niebá zſtąpił : jeśliby kto jadł z tego chlebá / żyć będźie ná wieki. A chleb / który Ja dam / jeſt ćiáło moje / które Ja dam zá żywot świátá.
Wádźili śię tedy Żydowie między ſobą / mowiąc : Jákoż ten może nam dáć ćiáło ſwoje ku jedzeniu?
Y rzekł im JEzuſ : Záprawdę záprawdę powiádam wam / Jeśli nie będźiećie jedli ćiáłá Syná cżłowiecżego / y pili krwi jego / nie máćie żywotá w ſobie.
Kto je ćiáło moje / á pije krew moję / ma żywot wiecżny : á Ja go wzbudzę w on oſtátecżny dźień.
Abowiem ćiáło moje / práwdźiwie jeſt pokárm : á krew mojá / prawdźiwie jeſt napój.
Kto je ćiáło moje / y pije krew moję / we mnie mieƺka á Ja w nim.
Jáko mię poſłał żywiący Oćiec / y Ja żywię przez Ojcá : <i>ták</i> kto mnie pożywa / y on żyć będźie przez mię.
Tenći jeſt chleb on / który z niebá zſtąpił. Nie jáko Ojcowie wáƺy jedli mánnę á pomárli : Kto je ten chleb / żyć będźie ná wieki.
To mowił w bożnicy ucżąc w Kápernáum.
Wile ich tedy z ucżniów jego ſłyƺąc to / mowili : Twárdać to jeſt mowá / ktoż jey ſłucháć może?
Ale wiedząc JEzuſ ſam w ſobie / iż o tym ƺemráli ucżniowie jego / rzekł im : Toż wáſ obraża?
Coż gdybyśćie ujrzeli Syná cżłowiecżego wſtępującego / gdźie był pierwey?
Duchći jeſt / który ożywia / ćiáło nic nie pomaga : Słowá które Ja wam mowię / Duch ſą / y żywot ſą.
Ale ſą niektórzy z was / co nie wierzą. Abowiem wiedźiał od pocżątku JEzuſ / którzy byli co nie wierzyli ; y kto jeſt co go miał wydáć.
Y mowił : Dlategomći wam powiedźiał / Iż żaden nie może przyść do mnie / jeźliby mu nie było dano od Ojcá mojego.
Od tego cżáſu wiele ucżniów jego odeƺło názad / á więcey z nim nie chodźili.
Tedy rzekł JEzuſ do <i>onych</i> dwunaśći ; Izali y wy chcećie odejść?
Y odpowiedźiał mu Szymon Piotr ; PAnie / do kogoż pojdźiemy? Ty máƺ ſłowá żywotá wiecżnego ;
A myśmy uwierzyli y poználi / żeś ty jeſt CHryſtuſ / on Syn Bogá żywego.
Odpowiedźiał im JEzuſ ; Izalim Ja nie dwunaści was obrał / á jeden z was jeſt Dyabeł.
A to mowił o Judaƺu / <i>ſynu</i> Szymoná / Iƺkáryoćie : bo go ten wydáć miał będąc jednym z onych dwunaści.
