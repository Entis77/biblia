A był niektóry cżłowiek z Fáryzeuƺów / imieniem Nikodem / kśiążę Żydowſkie.
Ten przyƺedł do JEzuſá w nocy / y rzekł mu : Miſtrzu / wiemy żeś przyƺedł od Bogá náucżyćielem : bo nikt tych cudów cżynić nie może / które ty cżyniƺ / jeſliby Bóg z nim nie był.
Odpowiedźiał JEzus / y rzekł mu : Záprawdę záprawdę powiádamći ; Jeſli śię kto nie národźi znowu / nie może widźieć króleſtwá Bożego.
Rzekł do niego Nikodem ; Jákoż śię może cżłowiek národźić będąc ſtáry? izali powtóre może wniść w żywot mátki ſwojey / y národźić śię?
Odpowiedźiał JEzus : Záprawdę záprawdę powiádamći ; Jeſliby śię kto nie národźił z wody y z Duchá / nie może wniść do króleſtwá Bożego.
Co śię národźiło z ćiáłá / ćiáło jeſt ; á co śię národźiło z Duchá / Duch jeſt.
Nie dźiwuj śię / żemći powiedźiał ; Muśićie śię znowu národźić.
Wiátr gdźie chce wieje / y głos jego ſłyƺyƺ / ále nie wieƺ z kąd przychodźi / y dokąd idźie : tákżeć jeſt káżdy / który śię národźił z Duchá.
Odpowiedźiał Nikodem / y rzekł mu ; Jákoż to być może?
Odpowiedźiał JEzus y rzekł mu ; Tyś jeſt náucżyćielem w Izráelu / á tego nie wieƺ?
Záprawdę záprawdę powiádamći / iż co wiemy / mówimy / á coſmy widźieli świadcżymy : ále świádectwá náƺego nie przyjmujećie.
Jeſliż <i>gdym</i> wam źiemſkie rzecży powiádał / á nie wierzyćie ; jákoż / będęli wam powiádał niebieſkie / uwierzyćie?
A nikt nie wſtąpił do niebá / tylko ten który zſtąpił z niebá / Syn cżłowiecży / który jeſt w niebie.
A jáko Mojzeƺ wężá ná puƺcży wywyżƺył / ták muśi być wywyżƺony Syn cżłowiecży.
Aby káżdy kto weń wierzy / nie zginął / ále miał żywot wiecżny.
Abowiem Ták Bóg umiłował świát / że Syná ſwego jednorodzonego dał : áby káżdy kto weń wierzy / nie zginął / ále miał żywot wiecżny.
Boć nie poſłał BÓG Syná ſwego ná świát / áby ſądźił świát ; ále áby świát był zbáwiony przezeń.
Kto wierzy weń / nie będźie oſądzony : ále kto nie wierzy / już jeſt oſądzony / iż nie uwierzył w Imię jednorodzonego Syná Bożego.
A tenći jeſt ſąd / że świátłość przyƺłá ná świát / lecż ludźie bárźiey umiłowáli ćiemność niż świátłość ; bo były złe ucżynki ich.
Káżdy bowiem kto źle cżyni / nienawidźi świátłośći / y nie idźie ná świátłość ; áby nie były zgánione ucżynki jego.
Lecż kto cżyni prawdę / przychodźi do świátłośći / áby były jáwne ucżynki jego / iż w Bogu ſą ucżynione.
Potym przyƺedł JEzus / y ucżniowie jego do Judſkiey źiemie / y tám przemieƺkawał z nimi y chrzćił.
Chrzćił też y Jan w Enon bliſko Sálim ; bo tám było wiele wód : á <i>ludźie</i> przychodźili / y chrzćili śię.
Bo jeƺcże Jan nie był podány do więźienia.
Wƺcżęłá śię tedy gadká miedzy ucżniámi Janowymi / y miedzy Żydámi o ocżyśćiániu.
Y przyƺli do Janá / y rzekli mu ; Miſtrzu / ten który był z tobą zá Jordanem / któremuś ty dał świádectwo / ten oto chrzći / á wƺyſcy idą do niego.
Odpowiedźiał Jan y rzekł : Nie może nic wźiąć cżłowiek / jeſliby mu nie było dano z niebá.
Wy ſámi jeſteśćie mi świádkámi / żem powiedźiał : Nie jeſtem ja CHryſtus / ále żem poſłány przed nim.
Kto ma oblubienicę / ten jeſt oblubieniec : á przyjaćiel oblubieńców / który ſtoji á ſłucha go / weſeli śię weſelem dla głoſu oblubieńcowego. Przetoż to weſele moje wypełnione jeſt.
On muśi rość / á mnie <i>muśi</i> ubywáć.
Kto z góry przyƺedł / náde wƺyſtkie jeſt : Kto z źiemie jeſt / źiemſki jeſt / y źiemſkie rzecży mówi : ten który z niebá przyƺedł / náde wƺyſtkie jeſt.
A co widźiał y ſłyƺał / to świadcży : ále świádectwá jego żaden nie przyjmuje.
Kto przyjmuje świádectwo jego / ten zápieczętował / że Bóg jeſt prawdźiwy.
Abowiem ten którego Bóg poſłał / ſłowo Boże mówi : boć <i>mu</i> nie pod miárą dawa Bóg Duchá.
Oćiec miłuje Syná / y wƺyſtko dał w ręce jego.
Kto wierzy w Syná / ma żywot wiecżny : ále kto nie wierzy Synowi / nie ogląda żywotá / lecż gniew Boży zoſtawa nád nim.
