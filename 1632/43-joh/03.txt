A był niektory cżłowiek z Fáryzeuƺow / imieniem Nikodem / kśiążę Żydowſkie.
Ten przyƺedł do JEzusá w nocy / y rzekł mu : Miſtrzu / wiemy / żeś przyƺedł od Bogá náućzyćielé : bo nikt tych cudow czynić nie może / ktore ty cżyniƺ / jeſliby Bog z nim nie był.
Odpowiedźiał JEzus / y rzekł mu : Záprawdę / záprawdę powiádamći ; Jeſli śię kto nie národźi znowu / nie może widźieć kroleſtwá Bożego.
Rzekł do niego Nikodem ; Jákoż śię może cżłowiek národźić będąc ſtáry? izáli powtore może wniść w żywot mátki ſwojey / y národźić śię?
Odpowiedźiał JEzus : Záprawdę / záprawdę powiádamći ; Jeſliby śię kto nie národźił z wody y z Duchá / nie może wniść do kroleſtwá Bożego.
Co śię národźiło z ćiáłá / ćiáło jeſt / á co śię národźiło z Duchá / duch jeſt.
Nie dźiwuj śię / żemći powiedźiał ; Muśićie śię znowu národźić.
Wiátr / gdźie chce wieje / y głos jego ſłyƺyƺ / ále nie wieƺ z kąd przychodźi / y dokąd idźie : tákżeć jeſt káżdy / ktory śię národźił z Duchá.
Odpowiedźiał Nikodem / y rzekł mu ; Jákoż to być może?
Odpowiedźiał JEzus / y rzekł mu ; Tyś jeſt náucżyćielem w Izráelu / á tego nie wieƺ?
Záprawdę záprawdę powiádamći / iż co wiemy / mowimy / á ćoſmy widźieli świadcżymy : ále świádectwá náƺego nie przyjmujećie.
Jeſliż <i>gdym</i> wam źiemſkie rzecży powiádał / á nie wierzyćie / jákoż / będęli wám powiádał niebieſkie / uwierzyćie?
A nikt nie wſtąpił do niebá / tylko ten ktory zſtąpił z niebá / Syn cżłowiecży / ktory jeſt w niebie.
A jáko Mojżeƺ wężá ná puƺcży wywyżƺył / ták muśi być wywyżƺony Syn cżłowiecży.
Aby káżdy / kto weń wierzy / nie zginął / ále miał żywot wiecżny.
Abowiem Ták Bog umiłowáł świát / że Syná ſwego jednorodzonego dał : áby káżdy kto weń wierzy / nie zginął / ále miał żywot wiecżny.
Boć nie poſłał BOG Syná ſwego ná świát / áby ſądźił świát ; ále áby świát był zbáwiony przezeń.
Kto wierzy weń / nie będźie oſądżony ; ále kto nie wierzy / już jeſt oſądżony / iż nie uwierzył w imię jednorodżonego ſyná Bożego.
A tenći jeſt ſąd / że świátłość przyƺłá ná świát / lećz ludźie bárdźiey umiłowáli ćiemność niż świátłość ; bo były złe ucżynki ich.
Káżdy bowiem kto źle cżyni / nienawidźi świátłośći / y nie idźie ná świátłość ; áby nie były zgánione ucżynki jego.
Lećz kto cżyni práwdę / przychodźi do świátłośći / áby były jáwne ucżynki jego / iż w Bogu ſą ucżynione.
Potym przyƺedł JEzus / y ucżniowie jego do Judſkiey źiemie / y tám przemieƺkawał z nimi i chrzćił.
Chrzćił też y Jan w Enon bliſko Sálim ; bo tám było wiele wod : á <i>ludźie</i> przychodźili y chrzćili śię.
Bo jeƺcże Jan nie był podány do więźienia.
Wƺcżęłá śię tedy gadká miedzy ućzniámi Janowymi / y miedzy Żydámi o ocźyśćiániu.
Y przyƺli do Janá / y rzekli mu ; Miſtrzu / ten ktory był z tobą zá Jordanem / ktoremuś ty dáł świádectwo / ten oto chrzći / á wƺyſćy idą do niego.
Odpowiedźiał Ján y rzekł : Nie może nic wźiąć cżłowiek / jeſliby mu nie było dáne z niebá.
Wy ſámi jeſteśćie mi świádkámi / żem powiedźiał : Nie jeſtem já Chryſtus / ále żem poſłány przed nim.
Kto má oblubienicę / ten jeſt oblubiéniec : przyjaćiel oblubieńcom / ktory ſtoji á ſłucha go / weſeli śię weſelem dla głoſu oblubieńcowego. Przetoż to weſele moje wypełnione jeſt.
On musi rość / mnie <i>muśi</i> ubywáć.
Kto z gory przyƺedł / nadé wƺyſtkie jeſt : Kto z źiemie jeſt / źiemſki jeſt / y źiemſkie rzećzy mowi : ten ktory z niebá przyƺedł / náde wƺyſtkie jeſt.
A co widźiał y ſłyƺáł / to świadcży : ále świádectwá jego żaden nieprzyjmuje.
Kto przyjmuje świádectwo jego / ten zápieczętował / że Bog jeſt prawdźiwy.
Abowiem ten ktorego Bog poſłał / ſłowo Boże mowi : boć <i>mu</i> nie pod miárą dawa y Bog Duchá.
Oćiec miłuje Syná / y wƺyſtko dał w ręce jego.
Kto wierzy w Syna / ma żywot wiecżny : ále kto nie wierzy Synowi / nie ogląda żywotá / lecż gniew Boży zoſtawa nád nim.
