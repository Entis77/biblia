A przed Świętem Wielkonocnym / wiedząc JEzus iż przyƺłá godźiná jego / áby przeƺedł z tego świátá do Ojcá : umiłowawƺy ſwoje którzy byli ná świećie / áż do końcá umiłował je.
A gdy byłá wiecżerza / ( á Dyabeł już był wrzućił w ſerce Judaƺá / <i>ſyná</i> Symonowego / Iƺkáryoty / áby go wydał. )
Widząc JEzus iż wƺyſtko Oćiec podał do rąk jego / á iż od Bogá wyƺedł / y do Bogá idźie :
Wſtał od wiecżerzy / y złożył ƺáty : á wźiąwƺy prześćierádło / przepáſał śię.
Potym nálał wody do miednice / y pocżął nogi umywáć ucżniom / y ućieráć prześćierádłem / którym był przepáſány.
Tedy przyƺedł do Symoná Piotrá. A on mu rzekł ; PAnie / y tyż mnie maƺ nogi umywáć?
Odpowiedźiał JEzus / y rzekł mu ; Co Ja cżynię / ty nie wieƺ teraz / ále śię potym dowieƺ.
Rzekł mu Piotr ; Nie będźieƺ ty nog mojich umywał ná wieki. Odpowiedźiał mu JEzus ; Jeśli ćię nie umyję / nie będźieƺ miał cżąſtki ze mną.
Tedy mu rzekł Symon Piotr ; PAnie / nie tylko nogi moje / ále y ręce / y głowę.
Rzekł mu JEzus : Ktoć jeſt umyty / nie potrzebuje jedno áby nogi umył : bo cżyſty jeſt wƺyſtek : y wy jeſteśćie cżyſtymi / ále nie wƺyſcy.
Abowiem wiedźiał który go wydáć miał / dla tegoż rzekł : Nie wƺyſcyśćie cżyſtymi.
Gdy tedy umył nogi ich / y wźiął ƺáty ſwoje / uśiadƺy záśię zá ſtoł / rzekł im : Wiećież com wam ucżynił.
Wy mię názywaćie Náucżyćielem y PAnem / á dobrze mowićie : bomći jeſt <i>nim :</i>
Ponieważem Ja tedy umył nogi wáƺe / PAn y Náucżyćiel / y wyśćie powinni jedni drugim nogi umywáć.
Abowiem dałem wam przykład / ábyśćie jáko Ja wám ucżynił / y wy cżynili.
Záprawdę záprawdę powiádam wam / Nie jeſt ſługá więkƺy nád Páná ſwego : áni poſeł jeſt więkƺy nád onego / który go poſłał.
Jeślić to wiećie / błogoſłáwieni jeſteśćie / jeźli to ucżynićie.
Nie o wƺyſtkichći was mowię / Jać wiem którem obrał : ále żeby śię wypełniło piſmo ; Który je ze mną chleb / podnioſł przećiwko mnie piętę ſwoję.
Terazći wam powiádam / przed tym niż śię to ſtánie / ábyśćie gdy śię to ſtánie / uwierzyli / żem Ja jeſt.
Záprawdę záprawdę powiádam wam / Kto przyjmuje tego któregobym poſłał / mnie przyjmuje : á kto mnie przyjmuje / onego przyjmuje który mię poſłał.
To rzekłƺy JEzus / záſmućił śię w Duchu / y oświadcżył / á rzekł : Záprawdę záprawdę powiádam wam / Że jeden z was wyda mię.
Tedy ucżniowie ſpoglądáli po ſobie wątpiąc o kimby to mowił.
A był jeden z ucżniów jego / który śię był położył ná łonie JEzuſowym / ten którego miłował JEzus.
Przetoż ná tego ſkinął Symon Piotr / áby śię wypytał / któryby to był / o którym mowił.
A on położywƺy śię ná pierśiách JEzuſowych / rzekł mu : PAnie / któryż to jeſt?
Odpowiedźiał JEzus : Ten jeſt / któremu Ja omocżywƺy ƺtucżkę chlebá / podam. A omocżywƺy ƺtucżkę chlebá / dáł Judaƺowi / <i>ſynowi</i> Szymoná / Iƺkáryotowi.
A záraz po oney ƺtucżce chlebá / wſtąpił weń Szátan. Tedy mu rzekł JEzus : Co cżyniƺ / cżyń rychło.
A tego żaden nie zrozumiał z ſpoł-śiedzących / ná co mu to rzekł.
Abowiem niektórzy mniemáli / gdyż Judaƺ miał mieƺek / iż mu rzekł JEzus : Nákup cżego nam potrzebá ná Święto : álbo iżby co dał ubogim.
Tedy on wźiąwƺy onę ƺtucżkę chlebá / zárázem wyƺedł. A noc byłá.
A gdy wyƺedł / rzekł JEzus : Teraz jeſt uwielbiony Syn cżłowiecży / á Bog uwielbiony jeſt w nim.
A ponieważ Bog uwielbiony jeſt w nim / tedy go też Bog uwielbi ſam w ſobie / y wnetże uwielbi go.
Synacżkowie / jeƺcże málucżko jeſtem z wámi. Będźiećie mię ƺukáć / ále já jákom rzekł Żydom ; Gdźie Ja idę / wy przyść nie możećie : <i>Ták</i> y wam teraz powiádam.
Przykazánie nowe dawam wam / ábyśćie śię ſpołecżnie miłowáli : jákom y já wáſ umiłowáł / ábyśćie śię y wy ſpołecżnie miłowáli.
Stądći poznáją wƺyſcy / żeśćie ucżniámi mojimi / jeźli miłość mieć będźiećie jedni przećiwko drugim.
Rzekł mu Szymon Piotr : PAnie / do kądże idźieƺ? Odpowiedźiał mu JEzus : Dokąd Ja idę / <i>ty</i> teraz zá mną iść nie możeƺ / ále potym pojdźieƺ zá mną.
<i>Tedy</i> mu rzekł Piotr : PAnie / przecż teraz zá tobą iść nie mogę? Duƺę moję zá ćię położę.
Odpowiedźiał mu JEzus : Duƺę twoję zá mię położyƺ? Záprawdę záprawdę powiádamći : Nie zápieje kur / áż śię mnie po trzy kroć záprzyƺ.
