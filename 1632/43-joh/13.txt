A przed Świętem Wielkonocnym / wiedząc JEzus iż przyƺłá godźiná jego / áby przeƺedł z tego świátá do Ojcá : umiłowawƺy ſwoje którzy byli ná świećie / áż do końcá umiłował je.
A gdy byłá wiecżerza / ( á Dyabeł już był wrzućił w ſerce Judaƺá / <i>ſyná</i> Symonowego / Iƺkáryoty / áby go wydał. )
Wiedząc JEzus iż wƺyſtko Oćiec podał do rąk jego / á iż od Bogá wyƺedł / y do Bogá idźie :
Wſtał od wiecżerzy / y złożył ƺáty : á wźiąwƺy prześćierádło / przepáſał śię.
Potym nálał wody do miednice / y pocżął nogi umywáć ucżniom / y ućieráć prześćierádłem / którym był przepáſány.
Tedy przyƺedł do Symoná Piotrá. A on mu rzekł ; PAnie / y tyż mnie maƺ nogi umywáć?
Odpowiedźiał JEzus y rzekł mu ; Co Ja cżynię / ty nie wieƺ teraz / ále śię potym dowieƺ.
Rzekł mu Piotr ; Nie będźieƺ ty nóg mojich umywał ná wieki. Odpowiedźiał mu JEzus ; Jeſli ćię nie umyję / nie będźieƺ miał cżąſtki ze mną.
Tedy mu rzekł Symon Piotr ; PAnie / nie tylko nogi moje / ále y ręce / y głowę.
Rzekł mu JEzus : Ktoć jeſt umyty / nie potrzebuje jedno áby nogi umył : bo cżyſty jeſt wƺyſtek : y wy jeſteśćie cżyſtymi / ále nie wƺyſcy.
Abowiem wiedźiał który go wydáć miał / dla tegoż rzekł : Nie wƺyſcyśćie cżyſtymi.
Gdy tedy umył nogi ich / y wźiął ƺáty ſwoje / uśiadƺy záśię zá ſtół / rzekł im : Wiećież com wam ucżynił.
Wy mię názywaćie Náucżyćielem y PAnem / á dobrze mówićie : bomći jeſt <i>nim</i> :
Ponieważem Ja tedy umył nogi wáƺe / PAN y Náucżyćiel / y wyśćie powinni jedni drugim nogi umywáć.
Abowiem dałem wam przykład / ábyśćie jáko Ja wam ucżynił / y wy cżynili.
Záprawdę záprawdę powiádam wam / Nie jeſt ſługá więkƺy nád Páná ſwego : áni poſeł jeſt więkƺy nád onego / który go poſłał.
Jeſlić to wiećie / błogoſłáwieni jeſteśćie / jeſli to ucżynićie.
Nie o wƺyſtkichći was mówię / Jać wiem którem obrał : ále żeby śię wypełniło piſmo ; Który je ze mną chleb / podnióſł przećiwko mnie piętę ſwoję.
Terazći wam powiádam / przed tym niż śię to ſtánie / ábyśćie gdy śię to ſtánie / uwierzyli żem Ja jeſt.
Záprawdę záprawdę powiádam wam / Kto przyjmuje tego któregobym poſłał / mnie przyjmuje : á kto mnie przyjmuje / onego przyjmuje który mię poſłał.
To rzekƺy JEzus / záſmućił śię w Duchu / y oświadcżył / á rzekł : Záprawdę záprawdę powiádam wam / Że jeden z was wyda mię.
Tedy ucżniowie poglądáli po ſobie wątpiąc o kimby to mówił.
A był jeden z ucżniów jego / który śię był położył ná łonie JEzuſowym / ten którego miłował JEzus.
Przetoż ná tego ſkinął Symon Piotr / áby śię wypytał / któryby to był / o którym mówił.
A on położywƺy śię ná pierśiách JEzuſowych / rzekł mu : PAnie / któryż to jeſt?
Odpowiedźiał JEzus : Ten jeſt / któremu Ja omocżywƺy ƺtucżkę chlebá / podam. A omocżywƺy ƺtucżkę chlebá / dał Judaƺowi / <i>ſynowi</i> Symoná / Iƺkáryotowi.
A záraz po oney ƺtucżce chlebá / wſtąpił weń Szátan. Tedy mu rzekł JEzus : Co cżyniƺ / cżyń rychło.
A tego żaden nie zrozumiał z ſpół-śiedzących / ná co mu to rzekł.
Abowiem niektórzy mniemáli / gdyż Judaƺ miał mięƺek / iż mu rzekł JEzus : Nákup cżego nam potrzebá ná Święto : álbo iżby co dał ubogim.
Tedy on wźiąwƺy onę ƺtucżkę chlebá / zárázem wyƺedł. A noc byłá.
A gdy wyƺedł / rzekł JEzus : Teraz jeſt uwielbiony Syn cżłowiecży / á Bóg uwielbiony jeſt w nim.
A ponieważ Bóg uwielbiony jeſt w nim / tedy go też Bóg uwielbi ſam w ſobie / y wnetże uwielbi go.
Synacżkowie / jeƺcże málucżko jeſtem z wámi. Będźiećie mię ƺukáć / ále jákom rzekł Żydom ; Gdźie Ja idę / wy przyść nie możećie : <i>Ták</i> y wam teraz powiádam.
Przykazánie nowe dawam wam / ábyśćie śię ſpołecżnie miłowáli : Jákom y Ja was umiłował / ábyśćie śię y wy ſpołecżnie miłowáli.
Stądći poznáją wƺyſcy / żeśćie ucżniámi mojimi / jeſli miłość mieć będźiećie jedni przećiwko drugim.
Rzekł mu Symon Piotr : PAnie / do kądże idźieƺ? Odpowiedźiał mu JEzus : Dokąd Ja idę / <i>ty</i> teraz zá mną iść nie możeƺ / ále potym pójdźieƺ zá mną.
<i>Tedy</i> mu rzekł Piotr : PAnie / przecż teraz zá tobą iść nie mogę? Duƺę moję zá ćię położę.
Odpowiedźiał mu JEzus : Duƺę twoję zá mię położyƺ? Záprawdę záprawdę powiádamći : Nie zápieje kur / áż śię mnie po trzy kroć záprzyƺ.
