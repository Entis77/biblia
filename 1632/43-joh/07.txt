A potym chodźił JEzus po Gálilejey : bo śię nie chćiał báwić w źiemi Judſkiey ; przeto / że Żydowie ƺukáli / áby go zábili.
Y było bliſko święto Żydowſkie Kucżek.
Tedy rzekli do niego bráćia jego ; Odejdź z tąd / á idź do Judſkiey źiemie : żeby ucżniowie twoji widźieli ſpráwy twoje / które cżyniƺ.
Abowiem żaden nic w ſkrytośći nie cżyni / kto chce bydź widźiány. Przetoż ty jeſli tákie rzecży cżyniƺ / objaw śię świátu.
Bo y bráćia jego nie wierzyli weń.
Y rzekł im JEzus : Cżás mój jeƺcże nie przyƺedł : ále cżás wáƺ záwżdy jeſt pogotowiu.
Nie możeć was świát nienawidźieć / ále mnie nienawidźi : bo Ja świadcżę o nim / iż ſpráwy jego złe ſą.
Idźćież wy ná to święto / Jać jeƺcże nie pójdę ná to święto : bo mój cżás jeƺcże śię nie wypełnił.
A to im powiedźiawƺy / zoſtał w Gálilejey.
A gdy poƺli bráćia jego / tedy y on ƺedł ná święto : nie jáwnie / ále jákoby potájemnie.
A Żydowie ƺukáli go w święto / y mówili : Gdźież on jeſt?
Y było o nim wielkie ƺemránie miedzy ludem. Bo jedni mówili ; Że jeſt dobry : á drudzy mówili ; Nie / ále zwodźi lud.
Wƺákże o nim żaden jáwnie nie mówił / dla bojáźni Żydowſkiey.
A gdy już było w pół świętá / wſtąpił JEzus do Kośćiołá / y ucżył.
Y dźiwowáli śię Żydowie / mówiąc : Jákoż ten umie piſmo / gdyż śię nie ucżył?
Odpowiedźiał im JEzus / y rzekł : Náuká mojá / nie jeſtći mojá / ále tego który mię poſłał.
Jeſliby kto chćiał cżynić wolą jego / ten będźie umiał rozeznáć / jeſli tá náuká jeſt z Bogá / cżyli Ja ſam od śiebie mówię.
Ktoć z ſámego śiebie mówi / chwały właſney ƺuka : ále kto ƺuka chwały tego który go poſłał / ten jeſt prawdźiwy / á niemáƺ w nim nieſpráwiedliwośći.
Izali wam Mojzeƺ nie dał Zakonu / á żaden z was nie przeſtrzega Zakonu? Przecżże ƺukaćie ábyśćie mię zábili?
Odpowiedźiał lud y rzekł : Dyabelſtwo maƺ ; Któż ćię ƺuka zábić?
Odpowiedźiał JEzus / y rzekł im : Jedenem ucżynek ucżynił / á wƺyſcy śię temu dźiwujećie!
Wƺák Mojzeƺ wydał wam obrzezkę / ( nie iżby byłá / z Mojzeƺá / ále z Ojców ) á w Sábbát obrzezujećie cżłowieká.
Ponieważ cżłowiek przyjmuje obrzezkę w Sábbát áby nie był zgwałcony Zakon Mojzeƺów / <i>przecż</i> śię ná mię gniewaćie / żem cáłego cżłowieká uzdrowił w Sábbát?
Nie ſądźćie według widzenia : ále ſpráwiedliwy ſąd ſądźćie.
Mówili tedy niektórzy z Jeruzalemcżyków : Izali to nie jeſt ten / którego ƺukáją zábić?
A oto jáwnie mówi / á nic mu nie mówią. Izali prawdźiwie poznáły Kśiążętá / iż ten jeſt prawdźiwie CHryſtus?
Ale o tym wiemy z kąd jeſt. Ale gdy CHryſtus przydźie / nikt nie będźie wiedźiał z kądby był.
Wołał tedy JEzus w Kośćiele ucżąc á mówiąc : Y mnie znaćie / y z kądem jeſt wiećie : á nie przyƺedłem ſam od śiebie / ále jeſt prawdźiwy / który mię poſłał / którego wy nie znaćie.
Lecż go Ja znam : bom od niego jeſt / á on mię poſłał.
Y ƺukáli jákoby go pojimáć : ále żaden nie śćiągnął nań ręki : bo jeƺcże nie przyƺłá byłá godźiná jego.
A wiele ich z ludu uwierzyli weń / y mówili : CHryſtus gdy przydźie / izaż więcey cudów ucżyni nád te / które ten ucżynił?
A ſłyƺeli Fáryzeuƺowie / yż to lud o nim ƺemrał : y poſłáli Fáryzeuƺowie y Przedniejƺy Kápłani ſługi / áby go pojimáli.
Rzekł im tedy JEzus : Jeƺcże ná máły cżás jeſtem z wámi / potym odejdę do tego który mię poſłał.
Szukáć mię będźiećie / ále nie znajdźiećie : á gdźie Ja będę / wy przyść nie możećie.
Mówili tedy Żydowie miedzy ſobą : Do kądże ten pójdźie / że my go nie znajdźiemy? cżyli do roſproƺonych pogánów pójdźie / y będźie ucżył pogány?
Cóż to zá mowá którą wyrzekł : Szukáć mię będźiećie / ále nie znajdźiećie : y gdźie Ja będę / wy przyść nie możećie?
A w on oſtátecżny dźień wielki Świętá onego / ſtánął JEzus / y wołał mówiąc : Jeſli kto prágnie / niech do mnie przydźie / á pije!
Kto wierzy w mię / jáko mówi piſmo / rzeki wody żywey popłyną z żywotá jego.
( A to mówił o Duchu którego wźiąć mieli wierzący weń : ábowiem jeƺcże nie był <i>dány</i> Duch Święty / przeto że jeƺcże JEzus nie był uwielbiony. )
Wiele ich tedy z onego ludu ſłyƺąc te ſłowá / mówili : Tenći jeſt prawdźiwie on Prorok.
A drudzy mówili : Ten jeſt CHryſtus. Ale niektórzy mówili : Azaż z Gálilejey przydźie CHryſtus?
Azaż nie mówi Piſmo / Iż z naśienia Dawidowego / y z Betlehem miáſtecżká / gdźie był Dawid / przydźie CHryſtus?
A ták ſtáło śię rozerwánie dla niego miedzy ludem.
Y chćieli go niektórzy z nich pojimáć / ále żaden nie śćiągnął nań rąk <i>ſwojich.</i>
Przyƺli tedy ſłudzy do Przedniejƺych Kápłanów / y do Fáryzeuƺów : Którzy im rzekli : Przecżżeśćie go nie przywiedli?
Odpowiedźieli oni ſłudzy : Nigdy ták nie mówił cżłowiek / jáko ten cżłowiek.
Y odpowiedźieli im Fáryzeuƺowie : Albośćie y wy zwiedźieni?
Izali kto uwierzył weń z Kśiążąt / álbo z Fáryzeuƺów?
Tylko ten gmin / który nie zna zakonu : przeklęći ſą.
Y rzekł do nich Nikodem / który był przyƺedł w nocy do niego / będąc jeden z nich :
Izali zakon náƺ ſądźi cżłowieká / jeſliby pierwey nie ſłyƺał od niego / y nie poznałby co cżyni?
A oni mu odpowiedźieli / y rzekli : Izaliś y ty Gálilejcżyk? Badajże śię á obacż / żeć Prorok z Gálilejey nie powſtał.
Y poƺedł káżdy do domu ſwego.
