A potym chodźił JEzuſ po Gálilejey : bo śię nie chćiał báwić w źiemi Judſkiey ; przeto / że Żydowie ƺukáli / áby go zábili.
Y było bliſko święto Żydowſkie Kucżek.
Tedy rzekli do niego bráćia jego ; Odejdź z tąd / á idź do Judſkiey źiemie : żeby ucżniowie twoji widźieli ſpráwy twoje / które cżyniƺ.
Abowiem żaden nic w ſkrytośći nie cżyni / kto chce bydź widźiány. Przetoż ty jeśli tákie rzecży cżyniƺ / objáw śię świátu.
Bo y bráćia jego nie wierzyli weń.
Y rzekł im JEzuſ : Cżáſ mój jeƺcże nie przyƺedł : ále cżáſ wáƺ záwżdy jeſt w pogotowiu.
Nie możeć wás świát nienawidźieć / ále mnie nienawidźi : bo Ja świadcżę o nim / iż ſpráwy jego złe ſą.
Idźćież wy ná to święto / Jáć jeƺcże nie pojdę ná to święto : bo mój cżás jeƺcże śię nie wypełnił.
A to im powiedźiawƺy / zoſtał w Gálilejey.
A gdy poƺli bráćia jego / tedy y on ƺedł ná święto : nie jáwnie / ále jákoby potájemnie.
A Żydowie ƺukáli go w święto y mowili : Gdźież on jeſt?
Y było o nim wielkie ƺemránie między ludem. Bo jedni mowili ; Że jeſt dobry : á drudzy mowili ; Nie / ále zwodźi lud.
Wƺákże o nim żaden jáwnie nie mowił / dla bojáźni Żydowſkiey.
A gdy już było w poł świętá / wſtąpił JEzuſ do Kośćiołá y ucżył.
Y dźiwowáli śię Żydowie / mowiąc : Jákoż ten umie piſmo / gdyż śię nie ucżył?
Odpowiedźiał im JEzuſ / y rzekł : Náuká mojá / nie jeſtći mojá / ále tego który mię poſłał.
Jeśliby kto chćiał cżynić wolę jego / ten będźie umiał rozeznáć / jeśli tá náuká jeſt z Bogá / cżyli Ja ſam od śiebie mowię.
Ktoć z ſámego śiebie mowi / chwały włáſney ƺuka ; ále kto ƺuka chwały tego który go poſłał / ten jeſt prawdźiwy / á niemáƺ w nim nieſpráwiedliwośći.
Izali wam Mojżeƺ nie dał Zakonu / á żaden z was nie przeſtrzega Zakonu? Przecżże ƺukáćie / ábyśćie mię zábili?
Odpowiedźiał lud y rzekł : Dyabelſtwo maƺ ; Ktoż ćię ƺuka zábić?
Odpowiedźiał JEzuſ / y rzekł im : Jedenem ucżynek ucżynił / á wƺyſcy śię temu dźiwujećie!
Wƺák Mojżeƺ wydał wam obrzezkę / ( nie iżby byłá z Mojżeƺá / ále z Ojców ) á w Sábbát obrzezujećie cżłowieká.
Ponieważ cżłowiek przyjmuje obrzezkę w Sábbát áby nie był zgwałcony Zakon Mojżeƺowy / <i>przecż</i> śię ná mię gniewaćie / żem cáłego cżłowieká uzdrowił w Sábbát?
Nie ſądźćie według widzenia : ále ſpráwiedliwy ſąd ſądźćie.
Mowili tedy niektórzy z Jeruzalemcżyków : Izali to nie jeſt ten / którego ƺukáją zábić?
A oto jáwnie mowi / á nic mu nie mowią. Izali prawdźiwie poznáły Kśiążętá / iż ten jeſt prawdźiwie CHryſtus?
Ale o tym wiemy z kąd jeſt. Ale gdy CHryſtus przydźie / nikt nie będźie wiedźiał z kądby był.
Wołał tedy JEzuſ w Kośćiele ucżąc á mowiąc : Y mnie znaćie / y z kądem jeſt wiećie : á nie przyƺedłem ſam od śiebie / ále jeſt prawdźiwy / który mię poſłał / którego wy nie znaćie.
Lecż go Ja znám : bom od niego jeſt / á on mię poſłał.
Y ƺukáli / jákoby go pojimáć : ále żáden nie śćiągnął nań ręki : bo jeƺcże nie przyƺłá byłá godźiná jego.
A wiele ich z ludu uwierzyli weń / y mowili : CHryſtuſ / gdy przydźie / izaż więcey cudów ucżyni nád te / które ten ucżynił?
A ſłyƺeli Fáryzeuƺowie / yż to lud o nim ƺemrał : y poſłáli Fáryzeuƺowie y Przedniejƺy Kápłani ſługi / áby go pojimáli.
Rzekł im tedy JEzuſ : Jeƺcże ná máły cżáſ jeſtem z wámi / potym odejdę do tego który mię poſłał.
Szukáć mię będźiećie / ále nie znajdźiećie : á gdźie Ja będę / wy przyść nie możećie.
Mowili tedy Żydowie między ſobą : Do kądże ten pojdźie / że my go nie znajdźiemy? cżyli do rozproƺonych Pogánów pojdźie / y będźie ucżył Pogány?
Coż to zá mowá którą wyrzekł : Szukáć mię będźiećie / ále nie znajdźiećie : y gdźie Ja będę / wy przyść nie możećie?
A w on oſtátecżny dźień wielki Świętá onego / ſtánął JEzuſ / y wołał mowiąc : Jeśli kto prágnie / niech do mnie przydźie / á pije!
Kto wierzy w mię / jáko mowi piſmo / rzeki wody żywey popłyną z żywotá jego.
( A to mowił o Duchu którego wźiąć mieli wierzący weń : ábowiem jeƺcże nie był <i>dány</i> Duch Święty / przeto że jeƺcże JEzuſ nie był uwielbiony. )
Wiele ich tedy z onego ludu ſłyƺąc te ſłowá / mowili : Tenći jeſt prawdźiwie on Prorok.
A drudzy mowili : Ten jeſt CHryſtus. Ale niektórzy mowili : Azaż z Gálilejey przydźie CHryſtus?
Azaż nie mowi Piſmo / Iż z naśienia Dawidowego / y z Betlehem miáſtecżká / gdźie był Dawid / przydźie CHryſtus?
A ták ſtáło śię rozerwánie dla niego miedzy ludem.
Y chćieli go niektórzy z nich pojimáć / ále żaden nie śćiągnął náń rąk <i>ſwojich.</i>
Przyƺli tedy ſłudzy do Przedniejƺych Kápłanów / y do Fáryzeuƺów : Którzy im rzekli : Przecżżeśćie go nie przywiedli?
Odpowiedźieli oni ſłudzy : Nigdy ták nie mowił cżłowiek / jáko ten cżłowiek.
Y odpowiedźieli im Fáryzeuƺowie : Albośćie y wy zwiedźieni?
Izali kto uwierzył weń z Kśiążąt / álbo z Fáryzeuƺow?
Tylko ten gmin / który nie zna zakonu : przeklęći ſą.
Y rzekł do nich Nikodem / który był przyƺedł w nocy do niego / będąc jeden z nich :
Izali zakon naƺ ſądźi cżłowieká / jeśliby pierwey nie ſłyƺał od niego / y nie poznałby / co cżyni?
A oni mu odpowiedźieli / y rzekli : Izaliś y ty Gálilejcżyk? Badajże śię / á obacż / żeć Prorok z Gálilejey nie powſtał.
Y poƺedł káżdy do domu ſwego.
