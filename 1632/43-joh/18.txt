To powiedźiawƺy JEzus / wyƺedł z ucżniámi ſwojimi przez potok Cedron / gdźie był ogrod : do ktorego on wƺedł y ucżniowie jego.
A wiedźiał y Judaƺ / ktory go wydawał / ono miejſce : bo śię tám cżęſto zchadzał JEzus z ucżniámi ſwojimi.
Przetoż Judaƺ wźiąwƺy rotę y ſługi od Przedniejƺych Kápłanow y Fáryzeuƺow / przyƺedł tám z láternámi y z pochodniámi / y z broniámi.
Tedy JEzus wiedząc wƺyſtko co nań przyść miáło / wyƺedłƺy rzekł im : Kogo ƺukaćie?
Odpowiedźieli mu : JEzuſá Názáreńſkiego. Rzekł im JEzus ; Jam jeſt. A ſtał z nimi y Judaƺ / ktory go wydawał.
A ſkoro im rzekł ; Jam jeſt : poſtąpili názad y pádli ná źiemię :
Tedy ich záśię ſpytał ; Kogo ƺukaćie? A oni rzekli ; JEzuſá Názáreńſkiego.
Odpowiedźiał JEzus : Powiedźiałem wam / żem Ja jeſt. Jeśli tedy mnie ƺukaćie / dopuśććież tym odejść.
Aby śię wypełniły ſłowá / ktore był powiedźiał ; Nie ſtráćiłem żadnego z tych / ktoreś mi dał.
Tedy Symon Piotr májąc miecż / dobył go : y uderzył ſługę Kápłaná Náwyżƺego / y ućiął mu ucho jego práwe. A temu ſłudze imię było Málchus.
Y rzekł JEzus Piotrowi : Włoż miecż twoj w pochwy. Izali nie mam pić kielichá tego / ktory mi dał Oćiec?
Rotá tedy y Rotmiſtrz / y ſłudzy Żydowſcy pojimáli JEzuſá y związáli go.
A wiedli go naprzod do Annaƺá : Bo był Świekier Káifaƺow / ktory był Nawyżƺym Kápłanem roku onego.
A Káifaƺ ten był / ktory Żydom rádźił ; że pożytecżno jeſt / áby jeden cżłowiek umárł zá lud.
Y ƺedł zá JEzuſem Symon Piotr / y drugi ucżeń. A ten ucżeń był znájomy Nawyżƺemu Kápłanowi / y wƺedł z JEzuſem do dworu Nawyżƺego Kápłaná.
Ale Piotr ſtał u drzwi ná dworze. Wyƺedł tedy on drugi ucżeń / ktory był znájomy Nawyżƺemu Kápłanowi ; y mowił z odźwierną / y wprowádźił tám Piotrá.
Tedy rzekłá Piotrowi dźiewká odźwierna ; Izaliś y ty nie jeſt z ucżniow tego cżłowieká? On odpowiedźiał : Nie jeſtem.
Stali tedy ſłudzy y cżeladź / ucżyniwƺy ogień / bo źimno było ; y grzali śię : Był też z nimi Piotr ſtojąc y grzejąc śię.
A ták Nawyżƺy Kápłan pytał JEzuſá o jego ucżnie / y o náukę jego.
Odpowiedźiał mu JEzus : Jam jáwnie mowił świátu : Jam záwƺe ucżył w bożnicy y w Kośćiele / gdźie śię zewƺąd Żydowie zchadzáją : á potájemnie nicem nie mowił.
Coż mię pytaƺ? Pytaj tych ktorzy ſłucháli / com im mowił : ćić oto wiedzą com Ja mowił.
A gdy on to mowił / jeden z ſług / ktory tám ſtał / uderzył JEzuſa laską / mowiąc ; Y tákże odpowiádaƺ Nawyżƺemu Kápłanowi?
Odpowiedźiał mu JEzus ; Jeślim źle rzekł / daj świádectwo o złym : á jeśli dobrze / przecżże mię bijeƺ?
<i>Y</i> odeſłał go Annaƺ związánego do Káifaƺá Nawyżƺego kápłáná.
A Symon Piotr ſtał y grzał śię. Y rzekli do niego ; Azażeś y ty nie jeſt z ucżniow jego? A on śię záprzał / mowiąc : Nie jeſtem.
Rzekł mu niektory z ſług Kápłaná Nawyżƺego / powinowáty onego / ktoremu był Piotr ućiął ucho : Izażem Ja ćiebie nie widźiał w ogrodźie z nim?
Záprzał śię záśię Piotr : á zárázem kur zápiał.
Prowádźili tedy JEzuſá od Káifaƺá ná ratuƺ : á było ráno. Y nie weƺli ſámi ná ratuƺ / áby śię nie zmázáli ; ále iżby pożywáli Báránká Wielkonocnego.
Tedy wyƺedł do nich Piłat / y rzekł : Jákąż ſkárgę przynośićie przećiwko cżłowiekowi temu?
Odpowiedźieli mu / y rzekli : Być ten nie był złocżyńcą / tedybyśmyć go nie podáli.
Y rzekł Piłat ; Weźmićież go wy / á według Zakonu wáƺego oſądźćie go. Rzekli mu Żydowie ; Nam śię nie godźi zábijáć nikogo.
Aby śię wypełniły ſłowá JEzuſowe ktore rzekł oznájmując / jáką miał śmierćią umrzeć.
Tedy záśię wƺedł Piłat ná ratuƺ / y záwołał JEzuſá / y rzekł mu ; Tyżeś jeſt Krol Żydowſki?
Odpowiedźiał mu JEzus ; A ſamże to od śiebie mowiƺ / cżylić inƺy powiedźieli o mnie?
Odpowiedźiał Piłat ; Azażem ja Żyd? Narod twoj y Przedniejƺy Kápłani podáli mi ćię : Cożeś wżdy ucżynił?
Odpowiedźiał JEzus ; Kroleſtwo moje nie jeſt z tego świátá : Gdyby kroleſtwo moje z tego świátá było / wżdyćby mię ſłudzy moji bronili / ábym nie był wydány Żydom : Lecż teraz kroleſtwo moje nie jeſt z tąd.
Tedy mu rzekł Piłat ; Toś ty przećię jeſt Krol ; Odpowiedźiał mu JEzus ; Ty powiádaƺ żem Ja jeſt Krolem. Jam śię ná to národźił / y ná tom przyƺedł ná świát / ábym świádectwo wydał prawdźie. Wƺelki ktory jeſt z prawdy / ſłucha głoſu mego.
Rzekł mu Piłat ; Coż jeſt prawdá? A to rzekƺy / wyƺedł záśię do Żydow / y rzekł im ; Ja w nim żadney winy nie znájduję.
A też u was jeſt ten zwycżaj / ábym wam jednego wypuśćił ná Wielkánoc : chcećież tedy ábym wam wypuśćił tego Krolá Żydowſkiego?
Tedy záśię wƺyſcy záwołáli / mowiąc : Nie tego / ále Bárábbaƺá! A ten Bárábbaƺ był zbojcá.
