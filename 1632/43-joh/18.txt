To powiedźiawƺy JEzus / wyƺedł z ucżniámi ſwojimi przez potok Cedron / gdźie był ogród : do którego on wƺedł y ucżniowie jego.
A wiedźiał y Judaƺ / który go wydawał / ono miejſce : bo śię tám cżęſto zchadzał JEzus z ucżniámi ſwojimi.
Przetoż Judaƺ wźiąwƺy rotę y ſługi od Przedniejƺych Kápłanów y Fáryzeuƺów / przyƺedł tám z láternámi y z pochodniámi / y z broniámi.
Tedy JEzus wiedząc wƺyſtko co nań przyść miáło / wyƺedƺy rzekł im : Kogo ƺukaćie?
Odpowiedźieli mu : JEzuſá Názáreńſkiego. Rzekł im JEzus ; Jam jeſt. A ſtał z nimi y Judaƺ / który go wydawał.
A ſkoro im rzekł ; Jam jeſt : poſtąpili ná zad / y pádli ná źiemię :
Tedy ich záśię ſpytał ; Kogo ƺukaćie? A oni rzekli ; JEzuſá Názáreńſkiego.
Odpowiedźiał JEzus : Powiedźiałem wam / żem Ja jeſt. Jeſli tedy mnie ƺukaćie / dopuśććież tym odejść.
Aby śię wypełniły ſłowá / które był powiedźiał ; Nie ſtráćiłem żadnego z tych / któreś mi dał.
Tedy Symon Piotr májąc miecż / dobył go : y uderzył ſługę Kápłaná Nawyżƺego / y ućiął mu ucho jego práwe. A temu ſłudze imię było Málchuƺ.
Y rzekł JEzus Piotrowi : Włóż miecż twój w pochwy. Izali nie mam pić kielichá tego / który mi dał Oćiec?
Rotá tedy y Rotmiſtrz / y ſłudzy Żydowſcy pojimáli JEzuſá / y związáli go.
A wiedli go naprzód do Annaƺá : Bo był Świekier Káifaƺów / który był Nawyżƺym Kápłanem roku onego.
A Káifaƺ ten był / który Żydom rádźił ; że pożytecżno jeſt / áby jeden cżłowiek umárł zá lud.
Y ƺedł zá JEzuſem Symon Piotr / y drugi ucżeń. A ten ucżeń był znájomy Nawyżƺemu Kápłanowi / y wƺedł z JEzuſem do dworu Nawyżƺego Kápłaná.
Ale Piotr ſtał u drzwi ná dworze. Wyƺedł tedy on drugi ucżeń / który był znájomy Nawyżƺemu Kápłanowi ; y mówił z odźwierną / y wprowádźił tám Piotrá.
Tedy rzekłá Piotrowi dźiewká odźwierna ; Izaliś y ty nie jeſt z ucżniów tego cżłowieká? On odpowiedźiał : Nie jeſtem.
Stali tedy ſłudzy y cżeladź / ucżyniwƺy ogień / bo źimno było ; y grzali śię : Był też z nimi Piotr ſtojąc y grzejąc śię.
A ták Nawyżƺy Kápłan pytał JEzuſá o jego ucżnie / y o náukę jego.
Odpowiedźiał mu JEzus : Jam jáwnie mówił świátu : Jam záwƺe ucżył w bóżnicy y w Kośćiele / gdźie śię zewƺąd Żydowie zchadzáją : á potájemnie nicem nie mówił.
Cóż mię pytaƺ? Pytaj tych którzy ſłucháli / com im mówił : ćić oto wiedzą com Ja mówił.
A gdy on to mówił / jeden z ſług który tám ſtał / uderzył JEzuſá laſką / mówiąc ; Y tákże odpowiádaƺ Nawyżƺemu Kápłanowi?
Odpowiedźiał mu JEzus ; Jeſlim źle rzekł / daj świádectwo o złym : á jeſli dobrze / przecżże mię bijeƺ?
<i>Y</i> odeſłał go Annaƺ związánego do Káifaƺá Nawyżƺego kápłaná.
A Symon Piotr ſtał y grzał śię. Y rzekli do niego ; Azażeś y ty nie jeſt z ucżniów jego? A on śię záprzał / mówiąc : Nie jeſtem.
Rzekł mu niektóry z ſług Kápłaná Nawyżƺego / powinowáty onego / któremu był Piotr ućiął ucho : Izażem Ja ćiebie nie widźiał w ogrodźie z nim?
Záprzał śię záśię Piotr : á zárázem kur zápiał.
Prowádźili tedy JEzuſá od Káifaƺá ná ratuƺ : á było ráno. Y nie weƺli ſámi ná ratuƺ / áby śię nie zmázáli ; ále iżby pożywáli Báránká Wielkonocnego.
Tedy wyƺedł do nich Piłat / y rzekł ; Jákąż ſkárgę przynośićie przećiwko cżłowiekowi temu?
Odpowiedźieli mu / y rzekli : Być ten nie był złocżyńcą / tedybyſmyć go nie podáli.
Y rzekł Piłat ; Weźmićież go wy / á według Zakonu wáƺego oſądźćie go. Rzekli mu Żydowie ; Nam śię nie godźi zábijáć nikogo.
Aby śię wypełniły ſłowá JEzuſowe które rzekł / oznájmując jáką miał śmierćią umrzeć.
Tedy záśię wƺedł Piłat ná ratuƺ / y záwołał JEzuſá / y rzekł mu ; Tyżeś jeſt Król Żydowſki?
Odpowiedźiał mu JEzus ; A ſamże to od śiebie mówiƺ / cżylić inƺy powiedźieli o mnie?
Odpowiedźiał Piłat ; Azażem ja Żyd? Naród twój y Przedniejƺy Kápłani podáli mi ćię : Cóżeś wżdy ucżynił?
Odpowiedźiał JEzus ; Króleſtwo moje nie jeſt z tego świátá : Gdyby króleſtwo moje z tego świátá było / wżdyćby mię ſłudzy moji bronili / ábym nie był wydány Żydom : Lecż teraz króleſtwo moje nie jeſt z tąd.
Tedy mu rzekł Piłat ; Toś ty przećię jeſt Król ; Odpowiedźiał mu JEzus ; Ty powiádaƺ żem Ja jeſt Królem. Jam śię ná to národźił / y ná tom przyƺedł ná świát / ábym świádectwo wydał prawdźie. Wƺelki który jeſt z prawdy / ſłucha głoſu mego.
Rzekł mu Piłat ; Cóż jeſt prawdá? A to rzekƺy / wyƺedł záśię do Żydów / y rzekł im ; Ja w nim żadney winy nie znájduję.
A też u was jeſt ten zwycżaj / ábym wam jednego wypuśćił ná Wielkanoc : Chcećież tedy ábym wam wypuśćił tego Królá Żydowſkiego?
Tedy záśię wƺyſcy záwołáli / mówiąc : Nie tego / ále Bárábbaƺá! A ten Bárábbaƺ był zbójcá.
