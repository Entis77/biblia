A JEzus poƺedł ná gorę oliwną.
Potym záśię rániucżko przyƺedł do Kośćiołá : á lud wƺyſtek zƺedł śię do niego ; y śiadƺy ucżył je.
Y przywiedli do niego Náucżeni w piśmie y Fáryzeuƺowie niewiáſtę ná cudzołoſtwie záſtáną / á poſtáwiwƺy ją w pośrodku /
Rzekli mu ; Náucżyćielu / tę niewiáſtę záſtáno ná ſámym ucżynku cudzołoſtwá.
A w Zakonie nam Mojżeƺ przykazał tákie kámionowáć : á ty co mowiƺ?
A to mowili kuƺąc go / áby go mogli oſkárżyć. A JEzuſ zchyliwƺy śię ná doł / piſał pálcem ná źiemi.
A gdy śię go nie przeſtawáli pytáć / podniozƺu śię rzekł do nich ; Kto z wás jeſt bez grzechu / niech ná nię pierwƺy kámieniem rzući.
A záśię zchyliwƺy śię ná doł / piſał ná źiemi.
Co gdy oni uſłyƺeli / będąc od ſumnienia przekonáni / jeden zá drugim wychodźili / pocżąwƺy od ſtárƺych áż do oſtátecżnych ; iż tylko ſam JEzus zoſtáł / á oná niewiáſtá w pośrzodku ſtojąca.
A podniozƺy śię JEzus / y żadnego nie widząc tylko onę niewiáſtę / rzekł jey ; Niewiáſto gdźież ſą oni co ná ćię ſkárżyli? Żaden ćię nie potępił?
A oná rzekłá ; Żaden PAnie. A JEzus jey rzekł ; Ani Ja ćiebie potępiam. Idźże / á już więcey nie grzeƺ.
Zaśię im rzekł JEzus / mowiąc ; Jam jeſt świátłość świátá. Kto mnie náśláduje / nie będźie chodźił w ćiemnośći / ále będźie miał świátłość żywotá.
Rzekli mu tedy Fáryzeuƺowie ; Ty ſam o ſobie świadcżyƺ / á świádectwo twoje nie jeſt prawdźiwe.
Odpowiedźiał JEzus / y rzekł im : Choćiaż Ja świadcżę ſám o ſobie / <i>jednák</i> prawdźiwe jeſt świádectwo moje : bo wiem z kądem przyƺedł  / y dokąd idę. Lecż wy nie wiećie z kądem przyƺedł / y dokąd idę.
Wy według ćiáłá ſądźićie ; ále Ja nikogo nie ſądzę.
A choćbym y Ja ſądźił / ſąd mój jeſt prawdźiwy ; bom nie jeſt ſam / ále Ja / y który mię poſłał / Oćiec.
A w Zakonie wáƺym nápiſano jeſt ; Iż dwojgá ludźi świádectwo prawdźiwe jeſt.
Jam jeſt który ſam o ſobie świadcżę ; świadcży o mnie y ten który mię poſłał / Oćiec.
Tedy mu rzekli : Gdźież jeſt ten twój Oćiec? Odpowiedźiał JEzus : Ani mnie znaćie / áni Ojcá mego : Byśćie mnie ználi / y Ojcá byśćie mego ználi.
Te ſłowá mowił JEzus w ſkárbnicy ucżąc w Kośćiele : á żaden go nie pojimał ; bo jeƺcże byłá nie przyƺłá godźiná jego.
Rzekł im tedy záśię JEzus : Ja idę / y będźiećie mię ƺukáć / á w grzechu wáƺym pomrzećie. Gdźie Ja idę / wy przyść nie możećie.
Mowili tedy Żydowie : Alboć śię ſam zábije że mowi ; Gdźie Ja idę / wy przyść nie możećie.
Y rzekł do nich ; Wyśćie z niſkośći / á Jam z wyſokośći : wyśćie z tego świátá / á Jam záś nie jeſt z tego świátá.
Przetomći wam powiedźiał : Iż pomrzećie w grzechách wáƺych. Bo jeſli nie uwierzyćie / żem Ja jeſt / pomrzećie w grzechách wáƺych.
Tedy mu rzekli : Ktożeś ty jeſt? Y rzekł im JEzus : To co wam y z pocżątku powiádam.
Wieleć mam o was mowić y ſądźić ; ále ten który mię poſłał jeſt prawdźiwy : á Ja com od niego ſłyƺał / to mowię ná świećie.
Ale nie zrozumieli / że im o Ojcu mowił.
Przetoż im rzekł JEzus : Gdy wywyżƺyćie Syná cżłowiecżego / tedy poznaćie żem Ja jeſt / á ſam od śiebie nic nie cżynię : ále jáko mię náucżył Oćiec mój / ták mowię.
A ten który mię poſłał / ze mną jeſt ; nie zoſtáwił mię ſámego Oćiec : bo co mu śię podoba / to Ja záwżdy cżynię.
To gdy on mowił / wiele ich weń uwierzyło.
Tedy mowił JEzus do tych Żydów / co mu uwierzyli : Jeśli wy zoſtániećie w ſłowie mojim / prawdźiwie ucżniami mojimi będźiećie.
Y poznaćie prawdę / á prawdá was wyſwobodźi.
Y odpowiedźieli mu : Myśmy naśienie Abráhámowe / á nigdyśmy nie ſłużyli nikomu : jákoż ty mowiƺ / Iż wolnymi będźiećie.
Odpowiedźiał im JEzus : Záprawdę / záprawdę / powiádam wam / Iż wƺelki kto cżyni grzech / ſługą jeſt grzechu.
A ſługáć nie mieƺka w domu ná wieki / <i>ále</i> ſyn mieƺka ná wieki.
A przetoż jeśli was Syn wyſwobodźi / prawdźiwie wolnymi będźiećie.
Wiem / żeśćie naśienie Abráhámowe : lecż ƺukaćie ábyśćie mię zábili / iż mowá mojá nie ma u was miejſcá.
Ja com widźiáł u Ojcá mego / powiádam : á wy też <i>to</i> cośćie widźieli u Ojcá wáƺego / cżynićie.
Odpowiedźieli mu y rzekli : Oćiec náƺ jeſt Abráhám. Rzekł im JEzus : Byśćie byli ſynámi Abráhámowymi / cżynilibyśćie ucżynki Abráhámowe.
Ale teraz ƺukaćie abyśćie mię zábili / cżłowieká tego / którym wam prawdę mowił / którąm ſłyƺał od Bogá. Tego Abráhám nie cżynił.
Wy cżynićie ucżynki Ojcá wáƺego. Rzekli mu tedy : My z nierządu nie jeſteśmy zpłodzeni / jednegoż Ojcá mamy / Bogá.
Tedy im rzekł JEzus : Być był Bog Ojcem wáƺym / tedybyśćie mię miłowáli : gdyżem Ja od Bogá wyƺedł / y przyƺedłem : ánim ſam od śiebie przyƺedł / ále mię on poſłał.
Przecżże tey powieśći mojey nie pojmujećie? Przeto iż nie możećie ſłucháć mowy mojey.
Wyśćie z Ojcá Dyábłá / y pożądliwośći Ojcá wáƺego cżynić chcećie : onći był mężobojcą od pocżątku / y w prawdźie nie ſtał / bo w nim prawdy niemáƺ : gdy mowi kłamſtwo / z ſwego właſnego mowi / iż jeſt kłamcą y ojcem kłamſtwá.
A Ja że prawdę mowię / nie wierzyćie mi.
Ktoż mię z was obwini z grzechu? Jeśliż prawdę mowię / przecżże wy mi nie wierzyćie?
Ktoć z Bogá jeſt / ſłów Bożych ſłucha : dlatego wy nie ſłuchaćie / że z Bogá nie jeſteśćie.
Odpowiedźieli tedy Żydowie / y rzekli mu : Izali my nie dobrze mowimy / żeś ty jeſt Sámárytán / y Dyabelſtwo maƺ?
Odpowiedźiał JEzus : Jać Dyabelſtwá nie mam / ále cżcżę Ojcá mego : á wyśćie mię nie ucżćili.
Jać nie ƺukam chwały mojey : jeſt ten który ƺuka y ſądźi.
Záprawdę záprawdę powiádam wam : Jeśli kto ſłowá me záchowywáć będźie / śmierći nie ogląda ná wieki.
Tedy mu rzekli Żydowie : Terażeśmy poználi że Dyabelſtwo maƺ. Abráhám umárł / y Prorocy : A ty powiádaƺ : Jeśli kto ſłowá moje záchowywáć będźie / śmierći nie ſkoƺtuje ná wieki.
Izaś ty jeſt więkƺy nád Ojcá náƺego Abráhámá? który umárł / y Prorocy pomárli : Kimże śię ty wżdy cżyniƺ?
Odpowiedźiał JEzus : Jeśli śię Ja ſam chwalę / chwałá mojá nic nie jeſt. Jeſtći Oćiec mój / który mię chwali : o którym wy powiádaćie / że jeſt Bogiem wáƺym.
Lecż nie go znaćie / á Ja go znam ; y jeślibym rzekł / że go nie znam / byłbym podobnym wam kłamcą ; ále go znam / y ſłowá jego záchowywam.
Abráhám Oćiec wáƺ z rádośćią żądał / áby oglądał dźień mój / y oglądał y rádował śię.
Tedy rzekli Żydowie do niego ; Piąćidźieśiąt lat jeƺcże nie maƺ / á Abráhámáś widźiał?
Rzekł im JEzus : Záprawdę záprawdę powiádam wam ; Pierwey niż Abráhám był / jam jeſt.
Porwáli tedy kámienie / áby nań ćiſkáli. Lecż JEzus zchronił śię / y wyƺedł z Kośćiołá przechodząc przez pośrodek ich ; y ták uƺedł.
