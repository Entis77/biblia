Było potym Święto Żydowſkie / y wſtąpił JEzus do Jeruzalem.
A byłá w Jeruzalem przy owcżey <i>bramie</i> ſadzawká / którą zowią po Żydowſku Beteſdá / májąca pięć ganków.
W tych leżáło mnóſtwo wielkie niedołężnych / ślepych / chromych / wyſchłych / którzy cżekáli poruƺenia wody.
Abowiem Anjoł cżáſu pewnego zſtępował w ſadzawkę / y poruƺał wodę. A ták kto pierwƺy wſtąpił po wrzuƺeniu wody / zſtawał śię zdrowym / jákąbykolwiek chorobą zjęty był.
A był tám niektóry cżłowiek trzydźieśći y ośm lat chorobą złożony.
Tego gdy JEzus ujrzał leżącego / á poznawƺy że już przez długi cżás chorował / rzekł mu : Chceƺ być zdrów?
Odpowiedźiał mu on chory ; PAnie / nie mam cżłowieká któryby mię / gdy bywa poruƺona wodá / wrzućił do ſadzawki ; ále gdy ja idę / inƺy przede mną wſtępuje.
Rzekł mu JEzus : Wſtań / weźmi łoże twoje / á chodź.
A zárázem ſtał śię zdrowym on cżłowiek : y wźiął łoże ſwoje / y chodźił. A był Sábbát onego dniá.
Tedy rzekli Żydowie onemu uzdrowionemu : Sábbát jeſt ; nie godźić śię łożá nośić.
Odpowiedźiał im : Ten który mię uzdrowił / tenże mi rzekł ; Weźmi łoże twoje / á chodź.
Y pytáli go : Któryż jeſt ten cżłowiek coć powiedźiał ; Weźmi łoże twoje / á chodź?
A on uzdrowiony nie wiedźiał ktoby był ; ábowiem był JEzus uſtąpił / ponieważ wiele ludu było ná onym miejſcu.
Potym go JEzus ználazł w Kośćiele / y rzekł mu : Otoś się ſtał zdrowym ; Nie grzeƺ więcey / áby co gorƺego ná ćię nie przyƺło.
A odƺedƺy on cżłowiek / powiedźiał Żydom / iż to był JEzus / który go uzdrowił.
A przetoż Żydowie prześládowáli JEzuſá / y ƺukáli jákoby go zábili / że to ucżynił w Sábbát.
A JEzus im odpowiedźiał : Oćiec mój áż do tąd prácuje / y Ja prácuję.
Dla tego tedy tym więcey ƺukáli Żydowie jákoby go zábili : nie tylko iż gwałćił Sábbát / ále że y Ojcá ſwego powiádał być Bogiem / cżyniąc śię równym Bogu.
Odpowiedźiał tedy JEzus / y rzekł im : Záprawdę záprawdę powiádam wam / Nie może Syn ſam od śiebie nic cżynić / jedno co widźi że Oćiec cżyni : Abowiem cokolwiek on cżyni / to tákże y Syn cżyni.
Boć Oćiec miłuje Syná / y ukázuje mu wƺyſtko co ſam cżyni / y więkƺe mu nád te ſpráwy pokaże / ábyśćie śię wy dźiwowáli.
Abowiem jáko Oćiec wzbudza umárłe y ożywia / ták y Syn które chce ożywia.
Bo Oćiec nikogo nie ſądźi / lecż wƺyſtek ſąd dał Synowi.
Aby wƺyſcy cżćili Syná ták jáko cżcżą Ojcá. Kto nie cżći Syná / nie cżći <i>y</i> Ojcá który go poſłał.
Záprawdę záprawdę powiádam wam / Kto ſłowá mego ſłucha / y wierzy onemu który mię poſłał / ma żywot wiecżny : y nie przydźie ná ſąd / ále przeƺedł z śmierći do żywotá.
Záprawdę záprawdę powiádam wam / Że idźie godźiná / y teraz jeſt / gdy umárli uſłyƺą głos Syná Bożego : á którzy uſłyƺą / żyć będą.
Abowiem jáko Oćiec ma żywot ſam w ſobie / ták dał y Synowi / áby miał żywot w ſámym ſobie.
Y dał mu moc y ſąd cżynić / bo jeſt Synem cżłowiecżym.
Nie dźiwujćież śię temu / boć przydźie godźiná / w którą wƺyſcy co ſą w grobiech uſłyƺą głos jego.
Y pójdą ći którzy dobrze cżynili / ná powſtánie żywotá : ále ći którzy źle cżynili / ná powſtánie ſądu.
Nie mogęć Ja ſam od śiebie nic cżynić. Jáko ſłyƺę / ták ſądzę : á ſąd mój jeſt ſpráwiedliwy : bo nie ƺukam woli mojey / ále woli tego który mię poſłał / Ojcá.
Jeſliżeć Ja ſam o ſobie świadcżę / świádectwo moje nie jeſt prawdźiwe.
Inƺy jeſt co o mnie świadcży / y wiem że prawdźiwe jeſt świádectwo / które wydawa o mnie.
Wyśćie ſłáli do Janá / á on dał świádectwo prawdźie.
Ale Ja nie od cżłowieká świádectwo biorę / ále to mówię / ábyśćie wy byli zbáwieni.
Onći był świecą gorájącą y świecącą : A wyśćie śię chćieli do cżáſu porádowáć w świátłośći jego.
Ale Ja mam świádectwo więkƺe niż Janowe. Abowiem ſpráwy które mi dał Oćiec / ábym je wykonał : te ſáme ſpráwy które Ja cżynię / świadcżą o mnie / iż mię Oćiec poſłał.
A Oćiec który mię poſłał / onże świadcżył o mnie ; któregośćie wy głoſu nigdy nie ſłyƺeli / ániśćie oſoby jego widźieli.
Y ſłowá jego nie maćie w ſobie mieƺkájącego ; ábowiem którego on poſłał / temu nie wierzyćie.
Badajćież śię piſm ; boć śię wam zda / że w nich żywot wiecżny maćie / á one ſą które świádectwo wydawáją o mnie.
A wżdy do mnie przyść nie chcećie / ábyśćie żywot mieli.
Chwały od ludźi nie przyjmuję.
Alem was poznał / że miłośći Bożey nie maćie w ſobie.
Jam przyƺedł w Imieniu Ojcá mego / á nie przyjmujećie mię : jeſliżby przyƺedł inny w imieniu ſwojim / onego przyjmiećie.
Jákoż wy możećie wierzyć / chwałę jedni od drugich przyjmując / ponieważ chwały która <i>jeſt</i> od ſámego Bogá nie ƺukaćie?
Nie mniemajćie / ábym Ja was miał oſkarżáć przed Ojcem : jeſtći który ſkárzy ná was / Mojzeƺ ; w którym wy nádźieję maćie.
Bo gdybyśćie wierzyli Mojzeƺowi / wierzylibyśćie y mnie ; gdyż on o mnie piſał.
Ale ponieważ Piſmom jego nie wierzyćie y jákoż ſłowom mojim uwierzyćie?
