Było potym Święto Żydowſkie / y wſtąpił JEzus do Jeruzalem.
A byłá w Jeruzalem przy owcżey <i>bramie</i> ſadzawká / ktorą zowią po Żydowſku Beteſdá / májąca pięć gankow.
W tych leżáło mnoſtwo wielkie niedołężnych / ślepych / chromych / wyſchłych / ktorzy cżekáli poruƺenia wody.
Abowiem Anioł cżáſu pewnego zſtępował w ſadzawkę y poruƺał wodę. A ták kto pierwƺy wſtąpił po wrzuƺeniu wody / zſtawał śię zdrowym / jákąbykolwiek chorobą zjęty był.
A był tám niektory cżłowiek trzydźieśći y ośm lat chorobą złożony.
Tego gdy JEzus ujrzał leżącego / á poznawƺy że już przez długi cżás chorował / rzekł mu : Chceƺ być zdrow?
Odpowiedźiał mu on chory ; Pánie / nie mam cżłowieká ktoryby mię / gdy bywa poruƺona wodá / wrzućił do ſadzawki ; ále gdy ja idę / inƺy przede mną wſtępuje.
Rzekł mu JEzus : Wſtáń / weźmi łoże twoje / á chodź.
A zárázem ſtáł śię zdrowym on cżłowiek : y wźiął łoże ſwoje / y chodźił. A był Sábbát onego dniá.
Tedy rzekli Żydowie onemu uzdrowionemu : Sábbát jeſt ; nie godźić śię łożá nośić.
Odpowiedźiáł im : Ten ktory mię uzdrowił / tenże mi rzekł ; Weźmi łoże twoje / á chodź.
Y pytáli go : Ktoryż jeſt ten cżłowiek / coć powiedźiał ; Weźmi łoże twoje / á chodź?
A on uzdrowiony nie wiedźiał / ktoby był ; ábowiem był JEzus uſtąpił / ponieważ wiele ludu było ná onym miejſcu.
Potym go JEzus ználazł w Kośćiele / y rzekł mu : Otoś się ſtáł zdrowym ; Nie grzeƺ więcey / áby co gorƺego ná ćię nie przyƺło.
A odƺedƺy on cżłowiek / powiedźiał Żydom / iż to był JEzus / ktory go uzdrowił.
A przetoż Żydowie prześládowáli JEzuſá / y ƺukáli jákoby go zábili / że to ucżynił w Sábbát.
A JEzus im odpowiedźiał : Oćiec moj áż do tąd prácuje / y Já prácuję.
Dla tego tedy tym więcey ƺukáli Żydowie jákoby go zábili : nie tylko iż gwałćił Sábbát / ále że y Ojcá ſwego powiádał być Bogiem / cżyniąc śię rownym Bogu.
Odpowiedźiał tedy JEzus / y rzekł im : Záprawdę záprawdę powiádam wam / Nie może Syn ſam od śiebie nic cżynić / jedno co widźi że Oćiec cżyni : Abowiem cokolwiek on cżyni / to tákże y Syn cżyni.
Boć Oćiec miłuje Syná / y ukázuje mu wƺyſtko co ſam cżyni / y więkƺe mu nád te ſpráwy pokaże / ábyśćie śię wy dźiwowáli.
Abowiem jáko Oćiec wzbudza umárłe y ożywia / ták y Syn ktore chce ożywia.
Bo Oćiec nikogo nie ſądźi / lecż wƺyſtek ſąd dał Synowi.
Ab wƺyſcy cżćili Syná ták jáko cżcżą Ojcá. Kto nie cżći Syná / nie cżći y Ojcá / ktory go poſłał.
Záprawdę / záprawdę powiádam wam : Kto ſłowá mego ſłucha / y wierzy onemu / ktory mię poſłał / ma żywot wiecżny : y nie przydźie ná ſąd / ále przeƺedł z śmierći do żywotá.
Záprawdę / záprawdę powiádam wam / Że idźie godźiná / y teraz ieſt / gdy umárli uſłyƺą głos Syná Bożego : á ktorzy uſłyƺą / żyć będą.
Abowiem jáko Oćiec ma żywot ſám w ſobie ták dáł y Synowi / áby miał żywot w ſámym ſobie.
Y dał mu moc y ſąd cżynić / bo jeſt Synem cżłowiecżym.
Nie dźiwujćież śię temu / boć przydźie godźiná / w ktorą wƺyſcy co ſą w grobiech / uſłyƺą głos jego.
Y pojdą ći ktorzy dobrze cżynili / ná powſtánie żywotá : ále ći ktorzy źle cżynili / ná powſtánie ſądu.
Nie mogęć já ſam od śiebie nic cżynić Jáko ſłyƺę / ták ſądzę : á ſąd moj jeſt ſpráwiedliwy : bo nie ƺukam woli mojey / ále woli tego ktory mię poſłał / Ojcá.
Jeſliżeć Ja ſam o ſobie świadcżę / świádectwo moje nie jeſt prawdźiwe.
Inƺy jeſt co o mnie świadcży / y wiem że prawdźiwe jeſt świádectwo / ktore wydawa o mnie.
Wyśćie ſłáli do Janá / á on dał świádectwo prawdźie.
Ale Ja nie od cżłowieká świádectwo biorę / ále to mowię / ábyśćie wy byli zbáwieni.
Onći był świecą gorájącą y świecącą : A wyśćie śię chćieli do cżáſu porádowáć w świátłośći jego.
Ale Ja mam świádectwo więkƺe niż Janowe. Abowiem ſpráwy ktore mi dał Oćiec / ábym je wykonał : te ſáme ſpráwy ktore Ja cżynię / świadcżą o mnie / iż mię Oćiec poſłał.
A Oćiec ktory mię poſłał / onże świadcżył o mnie ; ktoregośćie wy głoſu nigdy nie ſłyƺeli / ániśćie oſoby jego widźieli.
Y ſłowá jego nie maćie w ſobie mieƺkájącego ; ábowiem / ktorego on poſłał / temu nie wierzyćie.
Badajćież śię piſm ; boć śię wam zda / że w nich żywot wiecżny maćie / á one ſą ktore świádectwo wydawáją o mnie.
A wżdy do mnie przyść nie chcećie / ábyśćie żywot mieli.
Chwały od ludźi nie przyjmuję.
Alem was poznáł / że miłośći Bożey nie maćie w ſobie.
Jam przyƺedł w Imieniu Ojcá mego / á nie przyjmujećie mię : jeſliżby przyƺedł inny w imieniu ſwojim / onego przyjmiećie.
Jákoż wy możećie wierzyć / chwałę jedni od drugich przyjmując / ponieważ chwały / ktorá <i>jeſt</i> od ſámego Bogá / nie ƺukáćie?
Nie mniemajćie / ábym Ja was miał oſkarzáć przed Ojcem : jeſtći / ktory ſkárzy ná was / Mojzeƺ ; w ktorym wy nádźieję maćie.
Bo gdybyśćie wierzyli Mojzeƺowi / wierzylibyśćie y mnie ; gdyż on o mnie piſał.
Ale ponieważ Piſmom jego nie wierzyćie y jákoż ſłowom mojim uwierzyćie?
