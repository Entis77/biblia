Tomći wam powiedźiał / ábyśćie śię nie gorƺyli.
Wyłącżáć was będą z bożnic : owƺem przydźie godźiná / że wƺelki / który was zábije / będźie mniemał / że Bogu poſługę cżyni.
A toć wam ucżynią / iż nie poználi Ojcá áni mnie.
Alemći wam to powiedźiał / ábyśćie / gdy przydźie tá godźiná / wſpomnieli ná to / żem Ja wam opowiedźiał / á tegom wam z pocżątku niepowiádał / bom był z wámi.
Lecż teraz idę do onego / który mię poſłał : á żaden z was nie pyta mię ; Dokąd idźieƺ?
Ale żem wam to powiedźiał / ſmętek nápełnił ſerce wáƺe.
Lecż Ja wam prawdę mowię : Wamći to pożytecżno / ábym Ja odƺedł. Bo jeśli nie odejdę / Poćieƺyćiel on nie przydźie do was. Ale jeśli odejdę / poślę go do was.
A on przyƺedƺy / będźie karał świát z grzechu / y z ſpráwiedliwośći / y z ſądu.
Z grzechu mowię / iż nie uwierzyli w mnie ;
Z ſpráwiedliwośći záśię / iż do Ojcá mego idę ; á już mię więcey nie ujrzyćie.
A z ſądu / iż Kśiążę tego świátá już jeſt oſądzone.
Mamći wam jeƺcże wiele mowić / ále teraz znieść nie możećie.
Lecż gdy przydźie on Duch prawdy / wprowádźi was we wƺelką prawdę ; bo nie ſam od śiebie mowić będźie / ále cokolwiek uſłyƺy / mowić będźie / y przyƺłe rzecży wam opowie.
On mię uwielbi / bo z mego weźmie / á opowie wam.
Wƺyſtko co ma Oćiec / moje jeſt : dla tegom rzekł ; Że z mego weźmie / á wam opowie.
Malucżko / á nie ujrzyćie mię : y záśię málucżko / á ujrzyćie mię ; bo Ja idę do Ojcá.
Mowili tedy <i>niektórzy</i> z ucżniów jego między ſobą : Coż to jeſt co nam mowi : Málucżko / á nie ujrzyćie mię : y záśię málucżko / á ujrzyćie mię / á iż Ja idę do Ojcá?
Przetoż mowili : Coż to jeſt co mowi : Málucżko? Nie wiemy co mowi?
Tedy JEzus poznał / że go pytáć chćieli / y rzekł im : O tym śię pytaćie między ſobą / żem rzekł : Málucżko / á nie ujrzyćie mię : y záśię málucżko / á ujrzyćie mię.
Záprawdę záprawdę powiádam wam : Że wy będźiećie płákáć y nárzekáć / á świát śię będźie weſelił : wy ſmutni będźiećie / ále ſmętek wáƺ obroći śię wam w weſele.
Niewiáſtá gdy rodźi / ſmętek ma / bo przyƺłá godźiná jey : lecż gdy porodźi dźiećiątko / już nie pámięta ućiśnieniá dla rádośći / iż śię cżłowiek ná świát národźił.
Y wy teraz ſmutek maćie : ále záśię ujrzę was / á będźie śię rádowáło ſerce wáƺe / á radośći wáƺey nikt nie odejmie od was.
A dniá onego nie będźiećie mnie o nic pytáć. Záprawdę záprawdę powiádam wam : O cokolwiekbyśćie prośili Ojcá w Imieniu mojim / da wam.
Do tąd o niceśćie nie prośili w Imieniu mojim. Prośćież á weźmiećie / áby rádość wáƺá byłá doſkonáła.
Tomći wam przez przypowieśći mowił : ále idźie godźiná / gdy już dáliey nie przez przypowieśći mowić wam będę : lecż jáwnie o Ojcu mojim oznájmię wam.
W on dźień w Imieniu mojim prośić będźiećie : á nie mowię wam ; Iż Ja będę Ojcá prośił zá wámi :
Abowiem ſam Oćiec miłuje was / żeśćie wy mnie umiłowáli : y uwierzyliśćie żem Ja od Bogá wyƺedł.
Wyƺedłem od Ojcá / á przyƺedłem ná świát : y záśię opuƺcżam świát / á idę do Ojcá.
Rzekli mu ucżniowie jego : Oto teraz jáwnie mowiƺ / á żadney przypowieśći nie powiádaƺ.
Teraz wiemy że wƺyſtko wieƺ / á nie potrzebujeƺ / áby ćię kto pytał : przez to wierzymy / żeś od Bogá wyƺedł.
Odpowiedźiał im JEzus ; Teraz wierzyćie :
Oto przydźie godźiná / owƺem już przyƺłá / że śię rozproƺyćie káżdy do ſwego / á mnie ſámego zoſtáwićie : lecż nie jeſtem ſam / bo Oćiec jeſt ze mną.
Tomći wam powiedźiał / ábyśćie we mnie pokój mieli. Ná świećie ućiſk mieć będźiećie : ále ufajćie / Jam zwyćiężył świát.
