Potym śię záś ukazał JEzus ucżniom u morzá Tyberyádzkiego : á ukazał śię ták.
Byli poſpołu Symon Piotr / y Tomaƺ ktorego zowią Dydymus / y Nátánáel / ktory był z Kány Gálilejſkiey y ſynowie Zebedeuƺowi / y drudzy dwá z ucżniow jego.
Rzekł im Symon Piotr ; Pojdę ryby łowić. Mowią mu ; Pojdźiemy y my z tobą. Y ƺli / y wnet wſtąpili w łodź : á oney nocy nic nie pojmáli.
A gdy już było ráno / ſtánął JEzus ná brzegu : wƺákże nie wiedźieli ucżniowie / żeby był JEzus.
Rzekł im tedy JEzus : Dźieći / á maćież co jeść? Odpowiedźieli mu : Nie mamy.
A on im rzekł ; Zápuśććie śieć po práwey ſtronie łodźi / á znajdźiećie. Y zápuśćili : á już dáliey nie mogli jey ćiągnąć przed mnoſtwem ryb.
Y rzekł on ucżeń / ktorego miłował JEzus Piotrowi ; PAn jeſt. Symon tedy Piotr uſłyƺawƺy iż PAn jeſt / przepáſał śię koƺulą / ( ábowiem był nági ) y rzućił śię w morze.
A drudzy záśię ucżniowie przyƺli w łodźi ; ( bo nie dáleko było od brzegu / ále jákoby ná dwu ſet łokiet ) ćiągnąc śieć z rybámi.
A gdy wſtąpili ná brzeg / ujrzeli węgle náłożone / y rybę ná nim leżącą / y chleb.
Rzekł im JEzus : Przynieśćie z tych ryb ktoreśćie teraz pojmáli.
Wſtąpił tedy Symon Piotr / y wyćiągnął śieć ná źiemię / pełną wielkich ryb ; <i>Ktorych było</i> ſto pięćdźieśiąt y trzy : á choć ich ták wiele było / nie zdárłá śię śieć.
Rzekł im JEzus : Podźćie / obiedwajćie. A żaden z ucżniow nie śmiał go pytáć ; Ty ktoś jeſt / wiedząc że jeſt PAn.
Tedy przyƺedł JEzus / y wźiął on chleb / y dał im : tákże y rybę.
A toć już trzeći raz ukazał śię JEzus ucżniom ſwojim po zmartwychwſtániu.
A gdy obiad odpráwili / rzekł JEzus Symonowi Piotrowi ; Symonie Jonaƺow / miłujeƺ mię więcey niżeli ći? Rzekł mu ; Ták jeſt PAnie / ty wieƺ że ćię miłuję. Rzekł mu ; Páś-że báránki moje.
Rzekł mu záśię powtore ; Symonie Jonaƺow / miłujeƺ mię? Rzekł mu ; Ták <i>jeſt</i> PAnie / ty wieƺ że ćię miłuję. Rzekł mu ; Páś-że Owce moje.
Rzekł mu potrzećie ; Symonie Jonaƺow / miłujeƺ mię? Y záſmućił śię Piotr / że mu potrzećie rzekł ; Miłujeƺ mię? Y odpowiedźiał mu : PAnie ty wƺyſtko wieƺ / ty znaƺ że ćię miłuję. Rzekł mu JEzus : Páś-że owce moje.
Záprawdę záprawdę powiádam tobie : Gdyś był młodƺym / opáſowałeś śię y chodźiłeś kędyś chćiał : lecż gdy śię zſtárzejeƺ / wyćiągnieƺ ręce twoje ; á inny ćię opáƺe / y poprowádźi gdźiebyś nie chćiał.
A to powiedźiał dawájąc znáć / jáką śmierćią miał uwielbić Bogá. A to powiedźiawƺy / rzekłmu : Podź zá mną.
A Piotr obroćiwƺy śię / ujrzał onego ucżniá ktorego miłował JEzus / pozad idącego ; ktory śię też był położył przy wiecżerzy ná pierśiách jego / y rzekł był : PAnie / ktoryż jeſt ten / co ćię wyda?
Tego ujrzawƺy Piotr / rzekł JEzuſowi : PAnie / á ten co?
Rzekł mu JEzus ; Jeślibym chćiał żeby on zoſtał áż przydę / co tobie do tego? Ty podź zá mną.
Y wyƺłá tá powieść między bráćią / żeby on ucżeń umrzeć nie miał. Lecż mu nie rzekł JEzus / iż nie miał umrzeć : Ale : Jeśli chcę / áby zoſtał áż przydę / coż tobie do tego?
Tenći jeſt on ucżeń / ktory świadcży o tym / y to nápiſał : á wiemy że prawdźiwe jeſt świádectwo jego.
Jeſt też jeƺcże y innych wiele <i>rzecży /</i> ktore cżynił JEzus : ktore gdyby miáły być wƺyſtkie zoſobná ſpiſáne / tuƺę iż y ſam świát nie mogłby ogárnąć Kśiąg / ktoreby nápiſáne były. Amen.
