Potym śię záś ukazał JEzus ucżniom u morzá Tyberyádſkiego : á ukazał śię ták.
Byli poſpołu Symon Piotr / y Tomaƺ którego zowią Dydymus / y Nátánáel / który był z Kány Gálilejſkiey y ſynowie Zebedeuƺowi / y drudzy dwá z ucżniów jego.
Rzekł im Symon Piotr ; Pójdę ryby łowić. Mówią mu ; Pójdźiemy y my z tobą. Y ƺli / y wnet wſtąpili w łódź : á oney nocy nic nie pojimáli.
A gdy już było ráno / ſtánął JEzus ná brzegu : wƺákże nie wiedźieli ucżniowie / żeby był JEzus.
Rzekł im tedy JEzus : Dźieći / á maćież co jeść? Odpowiedźieli mu : Nie mamy.
A on im rzekł ; Zápuśććie śieć po práwey ſtronie łodźi / á znajdźiećie. Y zápuśćili : á już dáliey nie mogli jey ćiągnąć przed mnóſtwem ryb.
Y rzekł on ucżeń / którego miłował JEzus / Piotrowi ; PAn jeſt. Symon tedy Piotr uſłyƺawƺy iż PAn jeſt / przepáſał śię koƺulą / ( ábowiem był nági ) y rzućił śię w morze.
A drudzy záśię ucżniowie przyƺli w łodźi ; ( bo nie dáleko było od brzegu / ále jákoby ná dwu ſet łokiet ) ćiągnąc śieć z rybámi.
A gdy wyſtąpili ná brzeg / ujrzeli węgle náłożone / y rybę ná nim leżącą / y chleb.
Rzekł im JEzus : Przynieśćie z tych ryb któreśćie teraz pojimáli.
Wſtąpił tedy Symon Piotr / y wyćiągnął śieć ná źiemię / pełną wielkich ryb ; <i>Których było</i> ſto pięćdźieśiąt y trzy : á choć ich ták wiele było / nie zdárłá śię śieć.
Rzekł im JEzus : Pódźćie / obiedwajćie. A żaden z ucżniów nie śmiał go pytáć ; Ty ktoś jeſt / wiedząc że jeſt PAn.
Tedy przyƺedł JEzus / y wźiął on chleb / y dał im : tákże y rybę.
A toć już trzeći raz ukazał śię JEzus ucżniom ſwojim po zmartwychwſtániu.
A gdy obiad odpráwili / rzekł JEzus Symonowi Piotrowi ; Symonie Jonaƺów / miłujeƺ mię więcey niżeli ći? Rzekł mu ; Ták jeſt PAnie / ty wieƺ że ćię miłuję. Rzekł mu ; Páśże báránki moje.
Rzekł mu záśię powtóre ; Symonie Jonaƺów / miłujeƺ mię? Rzekł mu ; Ták <i>jeſt</i> PAnie / ty wieƺ że ćię miłuję. Rzekł mu ; Páśże Owce moje.
Rzekł mu potrzećie ; Symonie Jonaƺów / miłujeƺ mię? Y záſmućił śię Piotr / że mu potrzećie rzekł ; Miłujeƺ mię? Y odpowiedźiał mu : PAnie ty wƺyſtko wieƺ / ty znaƺ że ćię miłuję. Rzekł mu JEzus : Páśże owce moje.
Záprawdę záprawdę powiádam tobie : Gdyś był młodƺym / opáſowałeś śię / y chodźiłeś kędyś chćiał : lecż gdy śię zſtárzejeƺ / wyćiągnieƺ ręce twoje ; á inny ćię opáƺe / y poprowádźi gdźiebyś nie chćiał.
A to powiedźiał dawájąc znáć / jáką śmierćią miał uwielbić Bogá. A to powiedźiawƺy / rzekł mu : Pódź zá mną.
A Piotr obróćiwƺy śię / ujrzał onego ucżniá którego miłował JEzus / pozad idącego ; który śię też był położył przy wiecżerzy ná pierśiách jego / y rzekł był : PAnie / któryż jeſt ten co ćię wyda?
Tego ujrzawƺy Piotr / rzekł JEzuſowi : PAnie / á ten co?
Rzekł mu JEzus ; Jeſlibym chćiał żeby on zoſtał áż przydę / co tobie do tego? Ty pódź zá mną.
Y wyƺłá tá powieść miedzy bráćią / żeby on ucżeń umrzeć nie miał. Lecż mu nie rzekł JEzus / iż nie miał umrzeć : Ale : Jeſli chcę / áby zoſtał áż przydę / cóż tobie do tego?
Tenći jeſt on ucżeń / który świadcży o tym / y to nápiſał : á wiemy że prawdźiwe jeſt świádectwo jego.
Jeſt też jeƺcże y innych wiele <i>rzecży</i>, które cżynił JEzus : które gdyby miáły być wƺyſtkie zoſobná ſpiſáne / tuƺę iż y ſam świát nie mógłby ogárnąć Kśiąg / któreby nápiſáne były. Amen.
