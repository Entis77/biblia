To powiedźiawƺy JEzus / podnioſł ocży ſwoje w niebo y rzekł : Ojcże / przyƺłá godźiná / uwielbi Syná twego / áby też y Syn twój uwielbił ćiebie.
Jákoś mu dał moc nád wƺelkim ćiáłem / áby tym wƺyſtkim któreś mu dał / dał żywot wiecżny.
A toć jeſt żywot wiecżny / áby ćiebie poználi ſámego prawdźiwego Bogá / y któregoś poſłał / JEzuſá CHryſtuſá.
Jam ćię uwielbił ná źiemi / y dokońcżyłem ſpráwy / którąś mi dał / ábym ją cżynił.
A teraz uwielbi mię ty Ojcże u śiebie ſámego tą chwałą / którąm miał u ćiebie / pierwey niżeli świát był.
Objáwiłem Imię twoje ludźiom / któreś mi dał z świátá : Twojić byli / y dałeś mi je / y záchowáli ſłowá twoje.
A teraz poználi / iż wƺyſtko coś mi dał / od ćiebie jeſt.
Abowiem ſłowá któreś mi dał / dałem im : á oni <i>je</i> przyjęli / y poználi prawdźiwie / iżem od ćiebie wyƺedł ; á uwierzyli / żeś ty mnie poſłał.
Jać zá nimi proƺę : Nie zá świátem proƺę ; ále zá tymi / któreś mi dał : bo twoji ſą.
Y wƺyſtko moje / jeſt twoje / á twoje / moje : y uwielbionym jeſt w nich.
A nie jeſtem więcey ná świećie / ále oni ſą ná świećie ; á Ja do ćiebie idę. Ojcże święty / záchowaj je w Imieniu twojim / któreś mi dał : áby byli jedno / jáko y my.
Gdym z nimi był ná świećie / Jam je záchowywał w Imieniu twojim. Któreś mi dał / ſtrzegłem <i>ich ;</i> y żaden z nich nie zginął / tylko on ſyn zátráceniá : żeby śię piſmo wypełniło.
Ale teraz do ćiebie idę ; y mowię to ná świećie / áby mieli rádość moję doſkonáłą w ſobie.
Jam im dał ſłowo twoje / á świát je miał w nienawiśći : bo nie ſą z świátá / jáko y Ja nie jeſtem z świátá.
Nie proƺę ábyś je wźiął z świátá ; ále ábyś je záchował ode złego.
Nie ſąć z świátá / jáko y Ja nie jeſtem z świátá.
Poświęć-że je w prawdźie twojey. Słowo twoje jeſt prawdą.
Jákoś ty mnie poſłał ná świát / <i>tak</i> y Ja poſłałem je ná świát.
A Ja poświęcam ſámego śiebie zá nie / áby y oni poświęceni byli w prawdźie.
Nie tylko zá tymi proƺę / lecż y zá onymi / którzy przez ſłowo ich uwierzą w mię ;
Aby wƺyſcy byli jedno / jáko ty Ojcże we mnie / á Ja w tobie ; áby y oni w nas jedno byli : áby świát uwierzył / żeś ty mnie poſłał.
A Ja tę chwáłę / którąś mi dał / dałem im : áby byli jedno / jáko my jedno jeſteśmy ;
Ja w nich / á ty we mnie / áby byli doſkonáłymi w jedno : á iżby poznał świát / żeś ty mię poſłał ; á żeś je umiłował / jákoś y mię umiłował.
Ojcże któreś mi dał / chcę / áby gdźiem Ja jeſt / y oni byli ze mną : áby oglądáli chwałę moję / którąś mi dał ; ábowiemęś mię umiłował przed záłożeniem świátá.
Ojcże ſpráwiedliwy / y świát ćię nie poznał : álem Ja ćiebie poznał / á y ći poználi / żeś ty mnie poſłał.
Y ucżyniłem im znájome Imię twoje / y znájome ucżynię : áby miłość / którąś mnie umiłował / w nich byłá / á Ja w nich.
