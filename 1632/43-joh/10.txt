Záprawdę záprawdę powiádam wam ; Kto nie wchodźi drzwiámi do owcżárnie / ále wchodźi inędy / ten jeſt złodźiey y zbojcá ;
Lecż kto wchodźi drzwiámi / páſterzem jeſt owiec.
Temu odźwierny otwiera / y owce ſłucháją głoſu jego ; á on ſwojich właſnych owiec z imienia woła / y wywodźi je.
A gdy wypuśći owce ſwoje / idźie przed nimi / á owce idą zá nim ; bo znáją głoſ jego.
Ale zá cudzym nie idą / lecż ućiekáją od niego ; bo nie znáją głoſu obcych.
Tę im przypowieść JEzus powiedźiał : lecż oni nie zrozumieli tego co im mowił.
Rzekł im tedy záśię JEzus : Záprawdę záprawdę powiádam wam ; Iżem Ja jeſt drzwiámi owiec.
Wƺyſcy ile ich przede mną przyƺło / złodźieje ſą y zbojcy ; ále ich nie ſłucháły owce.
Jamći jeſt drzwiámi : jeśli kto przez mię wnidźie / zbáwiony będźie ; á wnidźie y wynidźie / á páſtwiſko znajdźie.
Złodźiey nieprzychodźi jedno żeby kradł / á zábijał y tráćił : Jam przyƺedł áby żywot miáły / y obfićie miáły.
Jam jeſt on dobry Páſterz. Dobry Páſterz duƺę ſwoję kłádźie zá owce.
Lecż najemnik / y ten który nie jeſt Páſterzem / którego nie ſą owce właſne / widząc wilká przychodzącego / opuƺcża owce y ućieka : á wilk porywa y roſpraƺa owce.
A najemnik ućieka / iż jeſt najemnik / y nie ma piecży o owcách.
Jam jeſt on páſterz dobry / y znám moje / á moje mię też znáją.
Jáko mię zna Oćiec / y Ja znam Ojcá / y duƺę moję kłádę zá owce.
A mam y drugie owce / które nie ſą z tey owcżárni / y teć muƺę przywieść : y głoſu mego ſłucháć będą / á będźie jedná owcżárniá / y jeden páſterz.
Dla tego mię miłuje Oćiec / iż Ja kłádę duƺę moję / ábym ją záśię wźiął.
Żaden jey nie bierze odemnie / ále Ja kłádę ją ſam od śiebie : mam moc położyć ją / y mam moc záśię wźiąć ją. Toć rozkazánie wźiąłem od Ojcá mego.
Tedy śię ſtáło znowu rozerwánie między Żydámi dla tych ſłów.
Y mowiło ich wiele z nich : Dyabelſtwo ma / y ƺáleje ; cżemuż go ſłuchaćie?
Drudzy mowili ; Te ſłowá nie ſą Dyabelſtwo májącego. Izali Dyabeł może ślepych ocży otwarzáć?
A było w Jeruzalem poświącánie <i>Kośćiołá ;</i> á źimá byłá.
Y przechadzał śię JEzus w Kośćiele / w przyśionku Sálomonowym.
Tedy go obſtąpili Żydowie / y rzekli mu : Dokądże duƺe náƺe <i>ná rzecży</i> trzymaƺ? Jeśliżeś ty jeſt Chryſtus / powiedz nam jáwnie?
Odpowiedźiał im JEzus : Powiedźiałem wam / á nie wierzyćie : Spráwy które Ja cżynię w Imieniu Ojcá mego / te o mnie świadcżą.
Ale wy nie wierzyćie : bo nie jeſteśćie z owiec mojich / jákom wam powiedźiał.
Owce moje głoſu mego ſłucháją / á Ja je znam / y idą zá mną.
A Ja żywot wiecżny dawam im : y nie zginą ná wieki / áni ich żaden wydrze z ręki mojey.
Oćiec mój / który mi je dał / więkƺy jeſt nád wƺyſtkie : á żaden nie może <i>ich</i> wydrzeć z ręki Ojcá mego.
Ja y Oćiec jedno jeſteśmy.
Porwáli tedy znowu kámienie Żydowie / áby go ukámionowáli.
Odpowiedźiał im JEzus ; Wiele dobrych ucżynków ukazałem wam od Ojcá mego / dlaktóregoż z tych ucżynków kámionujećie mię?
Odpowiedźieli mu Żydowie mowiąc ; Dla dobrego ucżynku nie kámionujemy ćię / ále dla bluźnierſtwá : to jeſt / że ty będąc cżłowiekiem / cżyniƺ śię ſam Bogiem.
Odpowiedźiał im JEzus : Izali nie jeſt nápiſano w Zakonie wáƺym? Jam rzekł ; Bogowie jeſteśćie.
Jeśliżeć one názwał Bogámi do których śię ſtáło ſłowo Boże / á niemoże być piſmo ſkáżone :
<i>A mnie /</i> którego Oćiec poświęćił y poſłał ná świát / wy mowićie ; Bluźniƺ / żem rzekł : Jeſtem Synem Bożym?
Jeśliż nie cżynię ſpraw Ojcá mego / nie wierzćież mi.
A jeśliż cżynię / choćiabyśćie mnie nie wierzyli / wierzćież ucżynkom : ábyśćie poználi y wierzyli / żeć Oćiec jeſt we mnie / á Ja w nim.
Tedy záśię ƺukáli / <i>jákoby</i> go pojimáć : ále uƺedł z rąk ich.
Y odƺedł záśię zá Jordan ná ono miejſce / gdźie przed tym Jan chrzćił ; y támże mieƺkał.
A wiele ich do niego przychodźiło / y mowili : Janći wprawdźie żadnego cudu nie ucżynił ; wƺákże wƺyſtko cokolwiek Jan o tym powiedźiał / prawdźiwe było.
Y wiele ich tám uwierzyło weń.
