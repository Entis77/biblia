A gdy poznał PAN / iż uſłyƺeli Fáryzeuƺowie / że JEzus więcey ucżniów cżynił y chrzćił niżeli Jan :
( Choćiaż ſam JEzus nie chrzćił / ále ucżniowie jego )
Opuśćił Judſką źiemię / y odƺedł záśię do Gálilejey.
A muśiał iść przez Sámáryą.
Y przyƺedł do miáſtá Sámáryjey / które zowią Sychár ; bliſko folwárku / który był dał Jákób Józefowi ſynowi ſwemu.
Y byłá tám ſtudnia Jákóbowá. Przetoż będąc JEzus ná drodze zprácowány / śiedźiał ták ná ſtudni ; á było około ƺóſtey godźiny.
<i>Y</i> przyƺłá niewiáſtá z Sámáryjey cżerpáć wodę. Którey rzekł JEzus ; Daj mi pić.
( Bo ucżniowie jego odeƺli byli do miáſtá / áby nákupili żywnośći )
Rzekłá mu tedy oná niewiáſtá Sámárytáńſka ; Jákoż ty będąc Żydem / żądaƺ ode mnie napoju / od niewiáſty Sámárytanki? ( gdyż Żydowie nie obcują z Sámárytany )
Odpowiedźiał JEzus y rzekł jey ; Gdybyś wiedźiáłá ten dar Boży / y kto jeſt ten coć mówi ; Daj mi pić : tybyś go prośiłá / á dałćiby wodę żywą.
Y rzekłá mu niewiáſtá ; PAnie / nie maƺ y cżym nácżerpáć / á ſtudnia jeſt głęboka ; z kądże tedy maƺ tę wodę żywą?
Yzażeś ty jeſt więkƺy niżeli Oćiec náƺ Jákób / który nam dał tę ſtudnią / y ſam z niey pił / y ſynowie jego / y dobytek jego?
Odpowiedźiał JEzus / y rzekł jey ; Káżdy kto pije tę wodę / záśię będźie prágnął :
Lecż ktoby pił onę wodę / którą Ja mu dam / nie będźie prágnął ná wieki : ále tá wodá którą Ja mu dam / ſtánie śię w nim ſtudnią wody wyſkákującey ku żywotowi wiecżnemu.
Rzekłá do niego niewiáſtá ; PAnie / daj mi tey wody ábym nie prágnęłá / áni tu cżerpáć chodźiłá.
Rzekł jey JEzus ; Idź záwołaj mężá ſwego / á przydź tu.
Odpowiedźiáłá niewiáſtá / y rzekłá : Nie mam mężá. Rzekł jey JEzus : Dobrześ rzekłá ; Nie mam mężá.
Abowiemeś piąći mężów miáłá : á teraz <i>ten</i> którego maƺ / nie jeſt mężem twojim. Toś prawdę powiedźiáłá.
Rzekłá mu niewiáſtá ; PAnie / widzę żeś ty jeſt Prorok.
Ojcowie náƺy ná tey górze chwalili <i>Bogá</i> ; á wy powiádaćie / że w Jeruzalem jeſt miejſce / kędy przyzwojita chwalić.
Rzekł jey JEzus : Niewiáſto / wierz mi iż idźie godźiná / gdy áni ná tey górze áni w Jeruzalem / nie będźiećie chwalili Ojcá.
Wy chwalićie co nie wiećie ; á my chwalimy co wiemy : ábowiem zbáwienie jeſt z Żydów.
Aleć idźie godźiná / y teraz jeſt / gdy prawdźiwi chwalcy będą chwalić Ojcá w Duchu y w prawdźie.
Bo y Oćiec tákowych ƺuka / którzyby go chwalili. Bóg <i>jeſt</i> Duch : á ći którzy go chwalą / powinni go chwalić w Duchu y w prawdźie.
Rzekłá mu niewiáſtá ; Wiem żeć przydźie Meſſyaƺ / którego zowią CHryſtuſem : ten gdy przydźie / oznájmi nam wƺyſtko.
Rzekł jey JEzus ; Jam jeſt ten / który z tobą mówię.
A w tym przyƺli ucżniowie jego / y dźiwowáli śię / iż z niewiáſtą mówił. Wƺákże żaden nie rzekł : Oco śię pytaƺ / álbo co z nią rozmawiaƺ?
Y zoſtáwiłá oná niewiáſtá wiádro ſwoje / á ƺłá do miáſtá / y rzekłá onym ludźiom :
Pódźćie / oglądajćie cżłowieká / który mi powiedźiał wƺyſtko comkolwiek cżyniłá : nie tenli jeſt CHryſtus?
A przetoż wyƺli z miáſtá / y przyƺli do niego.
A tym cżáſem prośili go ucżniowie / mówiąc : Miſtrzu jedz.
A on im rzekł : Mamći Ja pokarm ku jedzeniu / o którym wy nie wiećie.
Mówili tedy ucżniowie miedzy ſobą : Aboć mu kto przynióſł jeść?
Rzekł im JEzus : Mójći jeſt pokarm / ábym cżynił wolą <i>tego,</i> który mię poſłał / á dokonał ſpráwy jego.
Izaż wy nie mówićie : że jeƺcże ſą cżtery mieśiące / á żniwo przydźie? Otoż powiádam wam / Podnieśćie ocży wáƺe / á przypátrzćie śię krájinam / żeć już białe ſą ku żniwu.
A kto żnie / bierze zapłátę / y zbiera owoc do żywotá wiecżnego : áby y ten który śieje rádował śię weſpół / y ten który żnie.
Abowiem w tym prawdźiwe jeſt ono przyſłowie : że inƺy jeſt który śieje / á inƺy który żnie.
Jam was poſłał żąć to / około cżegośćie wy nie prácowáli : inƺyć prácowáli / á wyśćie weƺli w pracę ich.
Tedy z miáſtá onego wiele Sámárytanów uwierzyło weń / dla powieśći oney niewiáſty / która świadcżyłá : Że mi wƺyſtko powiedźiał comkolwiek cżyniłá.
Gdy tedy przyƺli do niego Sámárytani / prośili go áby u nich zoſtał : y zoſtał tám przez dwá dni.
Y dáleko więcey ich uwierzyło / dla ſłowá jego.
A <i>oney</i> niewieśćie mówili : Iż już nie dla twojey powieśći wierzymy : ábowiemeſmy ſámi ſłyƺeli / y wiemy / że ten jeſt prawdźiwie zbáwićiel świátá / CHryſtus.
A po dwu dniách / wyƺedł z támtąd / y ƺedł do Gálilejey.
Abowiem ſam JEzus świádectwo wydał / yż Prorok w Ojcżyznie ſwojey nie jeſt we cżći.
A gdy przyƺedł do Gálilejey / przyjęli go Gálilejcżycy / widząc wƺyſtko co cżynił w Jeruzalem w Święto : bo y oni byli przyƺli ná Święto.
Tedy záśię przyƺedł JEzus do Kány Gálilejſkiey / gdźie był ucżynił z wody wino. A był niektóry <i>dworzánin</i> królewſki w Kápernáum / którego ſyn chorował.
Ten uſłyƺawƺy iż JEzus przyƺedł z Judzkiey źiemie do Gálilejey / ƺedł do niego / y prośił go áby zſtąpił á uzdrowił ſyná jego ; bo pocżynał umieráć.
Y rzekł do niego JEzus ; Jeſli nie ujrzyćie známion y cudów / nie uwierzyćie.
Rzekł mu on królewſki <i>dworzánin</i> ; PAnie / zſtąp pierwey niż umrze dźiećię moje.
Rzekł mu JEzus ; Idź / ſyn twój żywie. Y uwierzył on cżłowiek mowie którą mu powiedźiał JEzus / y poƺedł.
A gdy już ƺedł / zábieżeli mu ſłudzy jego / y oznájmili / mówiąc ; Dźiećię twoje żywie.
Tedy ich pytał o godźinę / w którąby śię lepiey miáło? Y rzekli mu ; Że wcżorá o śiódmey godźinie opuśćiłá go gorącżká.
Poznał tedy Oćiec / iż to oná godźiná <i>byłá,</i> którey mu był rzekł JEzus ; <i>Iż</i> ſyn twój żywie. Y uwierzył ſam / y wƺyſtek dom jego.
Tenći záśię wtóry cud ucżynił JEzus / przyƺedƺy z Judſkiey źiemie do Gálilejey.
