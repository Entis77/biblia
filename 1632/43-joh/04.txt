A gdy poznał PAN / iż uſłyƺeli Fáryzeuƺowie / że JEzus więcey ucżniów cżynił y chrzćił niżeli Jan :
( Choćiáż ſam JEzus nie chrzćił / ále ucżniowie jego )
Opuśćił Judzką źiemię / y odƺedł záśię do Gálilejey.
A muśiał iść przez Sámáryę.
Y przyƺedł do miáſtá Sámáryjey / które zowią Sychár ; bliſko folwárku / który był dał Jákob Jozefowi / ſynowi ſwemu.
Y byłá tám ſtudnia Jákobowá. Przetoż będąc JEzus ná drodze sprácowány / śiedźiał ták ná ſtudni ; á było około ƺoſtey godźiny.
<i>Y</i> przyƺłá niewiáſtá z Sámáryjey cżerpáć wodę. Którey rzekł JEzus ; Daj mi pić.
( Bo ucżniowie jego odeƺli byli do miáſtá / áby nákupili żywnośći )
Rzekłá mu tedy oná niewiáſtá Sámárytáńſká ; Jákoż ty będąc Żydem / żądaƺ ode mnie napoju / od niewiáſty Sámárytanki? ( gdyż Żydowie nie obcują z Sámárytany )
Odpowiedźiał JEzus y rzekł jey ; Gdybyś wiedźiáłá ten dár Boży / y kto jeſt ten coć mowi ; Daj mi pić : tybyś go prośiłá / á dáłćiby wodę żywą.
Y rzekłá mu niewiáſtá ; PAnie / nie maƺ y cżym nácżerpáć / á ſtudnia jeſt głęboka / z kądże tedy máƺ tę wodę żywą?
Yzáżeś ty jeſt więkƺy niżeli Oćiec náƺ Jákob / który nam dał tę ſtudnią / y ſam z niey pił / y ſynowie jego / y dobytek jego?
Odpowiedźiał JEzus / y rzekł jey ; Káżdy / kto pije tę wodę / záśię będźie prágnął :
Lecż ktoby pił onę wodę / którą Ja mu dám / nie będźie prágnął ná wieki : ále tá wodá / którą Ja mu dam / ſtánie śię w nim ſtudnią wody wyſkákującey ku żywotowi wiecżnemu.
Rzekłá do niego niewiáſtá ; PAnie ; daj mi tey wody ábym nie prágnęłá / áni tu cżerpáć chodźiłá.
Rzekł jey JEzus ; Idź / záwołaj mężá ſwego / á przyjdź tu.
Odpowiedźiáłá niewiáſtá y rzekłá : Nie mam mężá. Rzekł jey JEzus : Dobrześ rzekłá ; Nie mam mężá.
Abowiemeś pięći mężów miáłá : á teraz <i>ten /</i> którego maƺ / nie jeſt mężem twojim. Toś prawdę powiedźiáłá.
Rzekłá mu niewiáſtá ; PAnie / widzę / żeś ty jeſt Prorok.
Ojcowie náƺy ná tey gorze chwalili <i>Bogá ;</i> á wy powiádaćie / że w Jeruzalem jeſt miejſce / kędy przyzwojita chwalić.
Rzekł jey JEzus : Niewiáſto / wierz mi / iż idźie godźiná / gdy áni ná tey gorze áni w Jeruzalem / nie będźiećie chwalili Ojcá.
Wy chwalićie co nie wiećie ; á my chwalimy co wiemy : ábowiem zbáwienie jeſt z Żydów.
Aleć idźie godźiná / y teraz jeſt / gdy prawdźiwi chwalcy będą chwalić Ojcá w Duchu y w prawdźie.
Bo y Oćiec tákowych ƺuká / którzyby go chwálili. Bog <i>jeſt</i> duch : á ći /którzy go chwalą / powinni go chwalić w Duchu y w prawdźie.
Rzekłá mu niewiáſtá ; Wiem / żeć przydźie Meſſyaƺ / którego zowią CHryſtuſem : ten gdy przydźie / oznájmi nam wƺyſtko.
Rzekł jey JEzus ; Jam jeſt ten / który z tobą mowię.
A w tym przyƺli ucżniowie jego / y dźiwowáli śię / iż z niewiáſtą mowił. Wƺákże żaden nie rzekł : Oco śię pytaƺ / álbo co z nią rozmawiaƺ?
Y zoſtáwiłá oná niewiáſtá wiádro ſwoje / á ƺłá do miáſtá / y rzekłá onym ludźiom :
Podźćie / oglądajćie cżłowieká / który mi powiedźiał wƺyſtko comkolwiek cżyniłá : nie tenli jeſt CHryſtus?
A przetoż wyƺli z miáſtá / y przyƺli do niego.
A tym cżáſem prośili go ucżniowie / mowiąc : Miſtrzu jedz.
A on im rzekł : Mamći Ja pokarm ku jedzeniu / o którym wy nie wiećie.
Mowili tedy ucżniowie między ſobą : Aboć mu kto przynioſł jeść?
Rzekł im JEzus : Mojći jeſt pokárm / ábym cżynił wolę <i>tego /</i> który mię poſłał / á dokonał ſpráwy jego.
Izáż wy nie mowićie : że jeƺcże ſą cżtery mieśiące / á żniwo przydźie? Otoż powiádam wam / Podnieśćie ocży wáƺe / á przypátrzćie śię krájinam / żeć już białe ſą ku żniwu.
A kto żnie / bierze zapłátę / y zbiera owoc do żywotá wiecżnego : áby y ten który śieje rádował śię weſpoł / y ten który żnie.
Abowiem w tym prawdźiwe jeſt ono przyſłowie : że inƺy jeſt który śieje / á inƺy który żnie.
Jam was poſłał żąć to / około cżegośćie wy nie prácowáli : inƺyć prácowáli / á wyśćie weƺli w pracę ich.
Tedy z miáſtá onego wiele Sámárytanów uwierzyło weń / dla powieśći oney niewiáſty / która świadcżyłá : Że mi wƺyſtko powiedźiał / comkolwiek cżyniłá.
Gdy tedy przyƺli do niego Sámárytani / prośili go áby u nich zoſtał : y zoſtał tám przez dwá dni.
Y dáleko więcey ich uwierzyło  / dla ſłowá jego.
A <i>oney</i> niewieśćie mowili : Iż już nie dla twojey powieśći wierzymy : ábowiemeśmy ſámi ſłyƺeli / y wiemy / że ten jeſt prawdźiwie zbáwićiel świátá / CHryſtus.
A po dwu dniách / wyƺedł z támtąd / y ƺedł do Gálileiey.
Abowiem ſam JEzus świádectwo wydał / yż Prorok w Ojcżyznie ſwojey nie jeſt we cżći.
A gdy przyƺedł do Gálilejey / przyjęli go Gálilejcżycy / widząc wƺyſtko co cżynił w Jeruzalem w Święto : bo y oni byli przyƺli ná Święto.
Tedy záśię przyƺedł JEzus do Kány Gálilejſkiey / gdźie był ucżynił z wody wino. A był niektóry <i>dworzánin</i> królewſki w Kápernáum / którego ſyn chorowáł.
Ten uſłyƺáwƺy iż JEzuss przyƺedł z Judzkiey źiemie do Gálilejey / ƺedł do niego / y prośił go áby zſtąpił á uzdrowił ſyná jego ; bo pocżynał umieráć.
Y rzekł do niego JEzus ; Jeſli nie ujrzyćie známion y cudów / nie uwierzyćie.
Rzekł mu on królewſki <i>dworzánin ;</i> PAnie / zſtąp pierwey niż umrze dźiećię moje.
Rzekł mu JEzus ; Idź / ſyn twój żywie. Y uwierzył on cżłowiek mowie którą mu powiedźiał JEzus / y poƺedł.
A gdy już ƺedł / zábieżeli mu ſłudzy jego / y oznájmili / mowiąc ; Dźiećię twoje żywie.
Tedy ich pytáł o godźinę / w którąby śię lepiey miáło? Y rzekli mu ; Że wcżorá o śiodmey godźinie opuśćiłá go gorącżká.
Poznał tedy Oćiec / iż to oná godźiná <i>byłá /</i> którey mu był rzekł JEzus ; Iż ſyn twój żywie. Y uwierzył ſám / y wƺyſtek dom jego.
Tenći záśię wtory cud ucżynił JEzus / przyƺedłƺy z Judzkiey źiemie do Gálilejey.
