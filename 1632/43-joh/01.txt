Ná pocżątku było Słowo / á ono Słowo było u Bogá / á Bogiem było ono Słowo.
To było ná pocżątku u Bogá.
Wƺyſtkie rzecży przez nie śię ſtáły : á bez niego nic śię nie ſtáło co śię ſtáło.
W nim był żywot / á żywot był oną świátłośćią ludzką.
A tá świátłość w ćiemnośćiách świeći / ále jey ćiemnośći nie ogárnęły.
Był cżłowiek poſłány od Bogá / któremu imię było Jan.
Ten pƺyƺedł ná świádectwo / áby świadcżył o tey świátłośći / áby przezeń wƺyſcy uwierzyli.
Nie byłći on tą świátłośćią ; ále <i>przyƺedł</i> áby świadcżył o tey świátłośći :
<i>Tenći</i> był tą prawdźiwą świátłośćią / która oświeca káżdego cżłowieká przychodzącego ná świát.
Ná świećie był / á świát przezeń ucżyniony jeſt ; ále go świát nie poznał.
Do ſwey właſnośći przyƺedł / ále go właſni jego nieprzyjęli.
Lecż którzy go kolwiek przyjęli / dał im tę moc / áby śię ſtáli ſynámi Bożymi ; <i>to jest,</i> tym / którzy wierzą w Imię jego.
Którzy nie ze krwie / áni z woli ćiáłá / áni z woli mężá / ále z Bogá národzeni ſą.
A to Słowo ćiáłem śię ſtáło / y mieƺkáło między námi ; ( y widźieliſmy chwałę jego / chwałę jáko jednorodzonego od Ojcá ) pełne łáſki y prawdy.
Jan świadcżył o nim / y wołał / mówiąc ; Tenći jeſt / o którymem powiádał ; Który po mnie przyƺedƺy / uprzedźił mię : bo pierwey był niż ja.
A z pełnośći jego myſmy wƺyſcy wźięli / y łáſkę zá łáſkę.
Abowiem zakon przez Mojżeƺá jeſt dány / <i>á</i> łáſká y prawdá przez JEzuſá CHryſtuſá ſtáłá śię.
Bogá nikt nigdy nie widźiał : on jednorodzony ſyn / który jeſt w łonie Ojcowſkim / ten <i>nam</i> opowiedźiał.
A toć jeſt świádectwo Janowe / gdy poſłáli Żydźi z Jeruzalem Kápłany y Lewity / áby go pytáli : Ty ktoś jeſt?
Y wyznał / á nie záprzał ; á wyznał / żem ja nie jeſt CHryſtus.
Y pytáli go ; Cóżeś tedy? Eliażeś ty? A <i>on</i> rzekł ; Nie jeſtem. <i>A oni</i> ; Prorokiemeś ty? y odpowiedźiał ; Nie <i>jeſtem</i>.
Rzekli mu tedy ; Któżeś jeſt / żebyſmy odpowiedź dáli tym / którzy nas poſłáli? Cóż <i>wżdy</i> powiádaƺ o ſobie?
Rzekł : Jam <i>jeſt</i> głos wołájącego ná puƺcży ; Proſtujćie drogę PAńſką / jáko powiedźiał Izájaƺ Prorok.
A ći którzy byli poſłány / byli z Fáryzeuƺów.
Y pytáli go / y rzekli mu ; Czemuż tedy chrzćiƺ / jeſliżeś ty nie jeſt CHryſtus / áni Eliaƺ / áni on Prorok.
Odpowiedźiał im Jan / mówiąc ; Jać chrzcżę wodą : ále w pośrzodku was ſtoji / którego wy nie znaćie.
Tenći jeſt który po mnie przyƺedƺy / uprzedźił mię ; któremum ja nie jeſt godźien / żebym rozwiązał rzemyk obuwia jego.
To śię ſtáło w Betábárze zá Jordanem / gdźie Jan chrzćił.
A názájutrz ujrzał Jan JEzuſá idącego do śiebie / y rzekł : Oto Báránek Boży / który głádźi grzech świátá.
Tenći jeſt / o którymem powiádał ; Że idźie zá mną mąż który mię uprzedźił : bo pierwey był niż ja.
A jam go nie znał ; ále áby był objáwiony Izráelowi / dla tegom ja przyƺedł chrzcżąc wodą.
Y świadcżył Jan mówiąc : Widźiałem Duchá zſtępującego / jáko gołębicę z niebá / y zoſtał ná nim.
A jam go nie znał : ále który mię poſłał chrzćić wodą / ten mi rzekł ; Ná kogobyś ujrzał Duchá zſtępującego / y zoſtawájącego ná nim / tenći jeſt / który chrzći Duchem Świętym.
A jam widźiał y świadcżył / że ten jeſt Syn Boży.
A názájutrz záśię ſtał Jan / y dwá z ucżniów jego.
A ujrzawƺy JEzuſá chodzącego rzekł ;
Oto Báránek Boży.
Y ſłyƺeli go oni dwá ucżniowie mówiącego / y ƺli zá JEzuſem.
A obróćiwƺy śię JEzus / y ujrzawƺy je zá ſobą idące / rzekł do nich ; Cżego ƺukaćie? A oni mu rzekli ; Rábbi / ( co śię wykłada / Miſtrzu ) gdźie mieƺkaƺ?
Rzekł im ; Pódźćie / á oglądajćie. <i>Y</i> ƺli y widźieli gdźie mieƺkał / á zoſtáli przy nim onego dniá ; bo było około dźieśiątey godźiny.
<i>A</i> był Andrzyj brát Symoná Piotrá / jeden z onych dwu / którzy <i>to</i> ſłyƺeli od Janá / y ƺli byli zá nim.
Ten napierwey ználazł Symoná brátá <i>ſwego</i> właſnego / y rzekł mu ; Ználeźliſmy Meſyaƺá / Co śię wykłada / CHryſtus.
Y przywiódł go do JEzuſá. A wejrzawƺy nań JEzus / rzekł ; Tyś jeſt Symon / ſyn Jonaƺów ; ty będźieƺ názwány Cefás / co śię wykłada / Piotr.
Názájutrz chćiał JEzus wyniść do Gálilejey / y ználazł Filipá / y rzekł mu ; Pódź zá mną.
A Filip był z Betſájdy / z miáſtá Andrzejowego y Piotrowego.
Filip ználazł Nátánáelá / y rzekł mu : Ználeźliſmy onego / o którym piſał Mojzeƺ w zakonie / y Prorocy / JEzuſá Syná Józefowego z Názáretu.
Y rzekł mu Nátánáel ; Możeż z Názáretu być co dobrego? Rzekł mu Filip : Pódź / á oglądaj.
Ujrzawƺy tedy JEzus Nátánáelá idącego do śiebie / rzekł o nim : Oto prawdźiwie Izráelcżyk / w którym niemáƺ zdrády.
Rzekł mu Nátánáel : Z kądże mię znaƺ? Odpowiedźiał JEzus / y rzekł mu : Pierwey niż ćię Filip záwołał / gdyś był pod figowym drzewem / widźiałem ćię.
Odpowiedźiał Nátánáel y rzekł mu : Miſtrzu / tyś jeſt on Syn Boży : tyś jeſt on Król Izráelſki.
Odpowiedźiał JEzus / y rzekł mu : Iżemći powiedźiáł : Widźiałem ćię pod figowym drzewem / wierzyƺ : więkƺe rzecży nád te ujrzyƺ.
Y Rzekł mu : Záprawdę záprawdę powiádam wam : Od tego cżáſu ujrzyćie niebo otworzone / y Anjoły Boże wſtępujące y zſtępujące ná Syná cżłowiecżego.
