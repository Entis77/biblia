Tedy Piłat wźiął JEzuſá / y ubicżował <i>go.</i>
A żołnierze uplotƺy koronę z ćiernia / włożyli ná głowę jego / y płaƺcżem ƺárłatowym przyodźiali go.
A mowili ; Bądź pozdrowiony Królu Żydowſki! y bili go laſkami.
Y záśię wyƺedł Piłat ná dwor / y rzekł im ; Oto go wam wywiodę ná dwor / ábyśćie wiedźieli / iż w nim żadney winy nie znájduję.
Tedy JEzus wyƺedł ná dwor nioſąc onę ćierniową koronę / y on płaƺcż ƺárłatowy / y rzekł im <i>Piłat ;</i> Oto cżłowiek!
A gdy go ujrzeli Przedniejśi Kápłani y ſłudzy <i>ich /</i> záwołáli mowiąc : Ukrzyżuj / ukrzyżuj <i>go!</i> Rzekł im Piłat : Weźmićież go wy / á ukrzyżujćie : boć ja w nim żadney winy nie znájduję.
Odpowiedźieli mu Żydowie : Myć Zakon mamy / y według Zakonu náƺego ma umrzeć : bo śię cżynił Synem Bożym.
A gdy Piłat uſłyƺał te ſłowá / bárdźiey śię ulękł.
Y wƺedł záśię do Ratuƺá / y rzekł do JEzuſá : Zkądżeś ty jeſt? Lecż mu JEzus nie dał odpowiedźi.
Tedy mu rzekł Piłat : Nie mowiƺ ze mną? Nie wieƺ / iż mam moc ukrzyżowáć ćię / y mam moc wypuśćić ćię?
Odpowiedźiał JEzus : Nie miałbyś żadney mocy nádemną / być nie byłá dána z gory : Przetoż kto mię tobie wydał / więkƺy grzech ma.
Odtąd Piłat ſtárał śię o to / <i>jákoby</i> go wypuśćił. Lecż Żydowie wołáli / mowiąc : Jeśli go wypuśćiƺ / nie jeſteś przyjaćielem Ceſárſkim. Káżdy <i>bowiem</i> co śię Królem cżyni / zprzećiwia śię Ceſárzowi.
A przetoż Piłat uſłyƺawƺy te ſłowá / wywiodł JEzuſá ná dwor / y śiadł ná ſtolicy / ná miejſcu które zowią Litoſtrotoſ / á po Żydowſku Gábbátá.
A było to w dźień Przygotowánia przed Wielkanocą około ƺoſtey godźiny / y rzekł <i>Piłat</i> Żydom : Oto Król wáƺ!
A oni záwołáli : Stráć / ſtráć! Ukrzyżuj go. Rzekł im Piłat : Króláż wáƺego ukrzyżuję? Odpowiedźieli Przedniejƺy Kápłani : Nie mamy Królá / tylko Ceſárzá.
Tedy im go wydał / żeby był ukrzyżowány. Y wźięli JEzuſá / y wywiedli.
A on nioſąc krzyż ſwój / wyƺedł ná ono miejſce / które zwano / Trupich głów / á po Żydowſku zowią <i>je /</i> Golgotá :
Gdźie go ukrzyżowáli. A z nim drugich dwu / z obu ſtron ; á w pośrzodku JEzuſá.
Nápiſał też Piłat y napis / y poſtáwił nád krzyżem. A było nápiſano : IEzus Názárenſki / Król Żydowſki.
A ten napis cżytáło wiele Żydów : bo bliſko miáſtá było ono miejſce / gdźie był ukrzyżowány JEzus. A było nápiſano po Żydowſku / po Grecku / y po Łáćinie.
Tedy rzekli Piłatowi Przedniejƺy Kápłani Żydowſcy : Nie piƺ Król Żydowſki : ále iż on powiádał ; Jeſtem Królem Żydowſkim.
Odpowiedźiał Piłat : Com nápiſał / <i>tom</i> nápiſał.
A gdy żołnierze JEzuſá ukrzyżowáli / wźięli ƺáty jego / ( y ucżynili cżtery cżęśći / káżdemu żołnierzowi cżęść ) y ſuknią. A byłá tá ſukniá nieƺyta : <i>ale</i> od wierzchu wƺyſtká dźiána.
Tedy rzekli jedni do drugich : Nie krajmy jey / ále o nię rzućmy loſy / cżyjá ma być. Aby śię piſmo wypełniło / które mowi : Podźielili między śię ƺáty moje / á o odźienie moje los miotáli. To tedy ucżynili żołnierze.
A ſtały podle krzyżá JEzuſowego / mátká jego / y śioſtrá mátki jego / Márya <i>żoná</i> Kleofáƺowa / y Márya Mágdálená.
Tedy JEzus ujrzawƺy mátkę / y ucżniá którego miłował / tuż ſtojącego / rzekł mátce ſwojey : Niewiáſto / oto ſyn twój.
Potym rzekł ucżniowi : Oto mátká twojá. A od oney godźiny wźiął ją <i>on</i> ucżeń do śiebie.
Potym wiedząc JEzus iż śię już wƺyſtko wykonáło / áby śię wypełniło piſmo / rzekł ; Prágnę.
A było <i>tám</i> nacżynie poſtáwione octu pełne. Tedy oni nápełniwƺy gębkę octem / á obłożywƺy hiſopem / podáli do uſt jego.
A gdy JEzus ſkoƺtował octu / rzekł ; Wykonáło śię. A náchyliwƺy głowę / oddał Duchá.
Tedy Żydowie / áby ćiáłá ná krzyżu ná Sábbát nie zoſtáły / ponieważ był dźień Przygotowánia ; ( ábowiem był wielki on dźień Sábbátu ) prośili Piłatá / áby im goleni połamano / y zjęto <i>je.</i>
Przyƺli tedy żołnierze : á pierwƺemu wprawdźie złamáli goleni / y drugiemu który z nim był ukrzyżowány :
Ale do JEzuſá przyƺedƺy / gdy ujrzeli że już umárł / nie łamáli goleni jego.
Lecż jeden z żołnierzów włocżnią otworzył bok jego ; á zárázem wyƺłá krew y wodá.
A ten co to widźiał / świadcżył <i>o tym /</i> y prawdźiwe jeſt świádectwo jego : <i>á</i> on wie / iż prawdę powiáda / ábyśćie wy wierzyli.
Abowiem śię to ſtáło / áby śię wypełniło piſmo ; Kość jego nie będźie złamána.
Y záśię drugie piſmo mowi ; Ujrzą / kogo przebodli.
A potym prośił Piłatá Jozef z Arymátyey / ( który był ucżniem JEzuſowym / ále tájemnym / dla bojáźni Żydowſkiey ) áby zdjął ćiáło JEzuſowe. Y pozwolił Piłat. Szedł tedy y zdjął ćiáło JEzuſowe.
Przyƺedł też y Nikodem ( który był przed tym przyƺedł w nocy do JEzuſá ) nioſąc zmieƺáney Myrry y Aloeſ / około ſtá funtów.
Wźięli tedy ćiáło JEzuſowe / y uwinęli je w prześćierádłá z onymi rzecżámi wonnymi / jáko jeſt zwycżaj Żydom <i>umárłe</i> chowáć.
A był ná onym miejſcu / gdźie był ukrzyżowány / ogrod : á w ogrodźie grob nowy / w którym jeƺcże nikt nie był położony.
Przetoż tám dla dniá przygotowániá Żydowſkiego / iż on grob był bliſko / położyli JEzuſá.
