Tedy JEzus ƺoſtego dniá przed Wielkanocą przyƺedł do Betániey / kędy był Łázarz ten / który był umárł / którego wzbudźił od umárłych.
Támże mu ſpráwili wiecżerzę / á Martá poſługowáłá : á Łázarz był jednym z onych / którzy z nim ſpołem u ſtołu śiedźieli.
A Márya wźiąwƺy funt máśći Szpikánárdowey bárzo drogiey / námázáłá nogi JEzuſowe / y utárłá włoſámi ſwojimi nogi jego : y nápełniony był on dom wonnośćią <i>oney</i> máśći.
Tedy rzekł jeden z Ucżniów jego / Judaƺ / <i>ſyn</i> Symoná Iƺkáryot / który go miał wydáć :
Przecżże tey máśći nie przedano zá trzy ſtá groƺy / á nie dano ubogim?
A to mowił / nie iżby miał piecżą o ubogich / ále iż był złodźiejem / y mieƺek miáł : á cokolwiek włożono / nośił.
Tedy rzekł JEzus : Zániechaj jey ; ná dźień pogrzebu mego to chowáłá.
Abowiem ubogie záwżdy z ſobą maćie / ále mnie nie záwżdy mieć będźiećie.
Dowiedźiał śię tedy lud wielki / iż tám był : y przyƺli nie tylko dla JEzuſá / ále też áby Łázárzá widźieli / którego był wzbudźił od umárłych.
Y rádźili śię Przedniejƺy kápłani żeby y Łázárzá zábili.
Bo wiele z Żydów dla niego odſtępowáli / y wierzyli w JEzuſá.
Názájutrz wielki lud który był przyƺedł ná Święto / uſłyƺawƺy iż JEzus idźie do Jeruzalem :
Nábráli gáłązek pálmowych / y wyƺli ná przećiwko niemu / y wołáli : Hoſánná / błogoſłáwiony który idźie w Imieniu Páńſkim Krol Izráelſki.
A doſtawƺy JEzus oślęćiá / wśiadł ná nie / jáko nápiſano jeſt :
Niebój śię córko Syońſka : Oto Krol twój idźie / śiedzący ná oślęćiu.
A tego z przodku nie zrozumieli ucżniowie jego : Ale gdy był JEzus uwielbiony / tedy wſpomnieli / iż to było o nim nápiſano / á że mu to ucżynili.
Świadcżył tedy lud który z nim był / iż Łázárzá záwołał z grobu / y wzbudźił go od umárłych.
Dla tego też wyƺedł przećiwko niemu lud / że ſłyƺał / iż on ten cud ucżynił.
Tedy mowili Fáryzeuƺowie między ſobą ; A widźićie że nic nie ſpráwićie : oto świát zá nim poƺedł :
A byli niektórzy Grekowie z tych którzy przychodźili <i>do Jeruzalem /</i> żeby śię modlili w Święto.
Ci tedy przyƺli do Filipá / który był z Betſájdy Gálilejſkiey / y prośili go / mowiąc : PAnie / chcemy JEzuſá widźieć.
Szedł Filip / y powiedźiał Andrzejowi : A Andrzyj záśię y Filip powiedźieli JEzuſowi.
A JEzus odpowiedźiał im / mowiąc : Przyƺłá godźiná / áby był uwielbiony Syn cżłowiecży.
Záprawdę záprawdę powiádam wam : Jeśliby źiárno pƺenicżne wpadƺy do źiemie / nie obumárło / ono ſámo zoſtawa : lecż jeśliby obumárło / wielki pożytek przynośi.
Kto miłuje duƺę ſwoję / utráći ją : á kto nienawidźi duƺe ſwojey ná tym świećie / ku wiecżnemu żywotowi ſtrzeże jey.
Jeśli mnie kto ſłuży / niechże mię náśláduje : á gdźiem Ja jeſt / tám y ſługá mój będźie. A jeśli mnie kto ſłużyć będźie / ucżći go Oćiec mój.
Terazći duƺá mojá zátrwożona jeſt ; y coż rzekę? Ojcże záchowaj mię od tey godźiny. Alemći dla tego przyƺedł / ná tę godźinę.
Ojcże / uwielbi Imię twoje. Przyƺedł tedy głos z niebá ; Y uwielbiłem / y jeƺcże uwielbię.
A lud ten który ſtał y ſłyƺał / mowił : Zágrzmiáło. A drudzy mowili : Anioł do niego mowił.
Odpowiedźiał JEzus / y rzekł : Nie dla mnie śię ten głoſ ſtał / ále dla was.
Teraz jeſt ſąd świátá tego : teraz Kśiążę świátá tego precż wyrzucone będźie.
A Ja jeśli będę podwyżƺony od źiemie / poćiągnę wƺyſtkich do śiebie.
( A mowił to oznájmując jáką śmierćią miał umrzeć. )
Odpowiedźiał mu on lud : Myśmy ſłyƺeli z Zakonu / iż Chryſtus trwa ná wieki : A jákoż ty mowiƺ / że muśi bydź podwyżƺony Syn cżłowiecży : Y któryż to jeſt Syn cżłowiecży?
Tedy im rzekł JEzus : Jeƺcże do máłego cżáſu jeſt z wámi świátłość. Chodźćież tedy poki świátłość maćie / żeby was ćiemnośći nie ogárnęły : bo kto w ćiemnośći chodźi / nie wie kędy idźie.
Poki świátłość maćie / wierzćie w świátłość / ábyśćie byli ſynámi świátłośći. To powiedźiał JEzus : A odƺedƺy / zchronił śię od nich.
A choć ták wiele cudów ucżynił przed nimi / <i>przećie</i> nie uwierzyli weń :
Aby śię wypełniło ſłowo Izájaƺá Proroká / które powiedźiał : PAnie / y ktoż uwierzył kazániu náƺemu / á rámię PAńſkie komuż jeſt objáwione?
Dla tego uwierzyć nie mogli / iż jeƺcże powiedźiał Izájaƺ :
Záślepił ocży ich / y zátwárdźił ſerce ich : áby ocżymá nie widźieli y ſercem nie zrozumieli / y nie náwroćili śię / ábym je uzdrowił.
To powiedźiał Izájaƺ / gdy widźiał chwałę jego / y mowił o nim.
Wƺákże jednák y z Kśiążąt wiele ich weń uwierzyło : ále dla Fáryzeuƺów nie wyznawáli / áby z bożnice nie byli wyłącżeni.
Bo umiłowáli chwałę ludzką / więcey niż chwałę Bożą.
Y wołał JEzus á mowił : Kto wierzy w mię / nie w mię wierzy / ále w onego który mię poſłał.
Y kto mię widźi / widźi onego który mię poſłał.
Ja świátłość przyƺedłem ná świát / áby żaden kto wierzy w mię / w ćiemnośćiách nie zoſtał.
A jeśliby kto ſłuchał ſłów mojich / á nie uwierzyłby / Jać go nie ſądzę : Bom nie przyƺedł / żebym ſądźił świát / ále żebym zbáwił świát.
Kto mną gárdźi á nieprzyjmuje ſłów mojich / ma ktoby go ſądźił. Słowá którem Ja mowił / one go oſądzą w oſtátecżny dźień.
Bom Ja z śiebie ſámego nie mowił / ále ten który mię poſłał / Oćiec / on mi rozkazánie dał / cobym mowić y cobym powiádáć miał.
Y wiem że rozkazánie jego jeſt żywot wiecżny. Przetoż to co Ja wam mowię / jáko mi powiedźiał Oćiec / ták mowię.
