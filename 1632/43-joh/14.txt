Niechaj śię nie trwoży ſerce wáƺe. Wierzyćie w Bogá / y w mię wierzćie.
W domu Ojcá mego wiele jeſt mieƺkánia : A jeſli nie / wżdybymći wáam powiedźiał.
Idę ábym wam zgotowáł miejſce ; á gdy odejdę y zgotuję wam miejſce / przydę záśię y wezmę was do śiebie : żebyśćie / gdźiem Ja jeſt / y wy byli.
A dokąd Ja idę / wiećie / y drogę wiećie.
Rzekł mu Tomaƺ : PAnie / nie wiemy dokąd idźieƺ : á jákoż możemy drogę wiedźieć?
Rzekł mu JEzus : Jamći jeſt tá drogá / y prawdá / y żywot : Żaden nie przychodźi do Ojcá / tylko przez mię.
Gdybyśćie mię ználi / y Ojcábyśćie też mego ználi ; y już go teraz znaćie / y widźieliśćie go.
Rzekł mu Filip ; PAnie ukaż nam Ojcá / á doſyć nam ná tym.
Rzekł mu JEzus ; Przez ták długi cżás jeſtem z wámi / á nie poznałeś mię : Filipie / kto mnie widźi / widźi y Ojcá <i>mego.</i> Jákoż ty mowiƺ ; Ukaż nam Ojcá?
Nie wierzyƺ / iżem Ja w Ojcu / á Oćiec we mnie? Słowá które Ja do was mowię / nie od ſámego śiebie mowię : lecż Oćiec który we mnie mieƺka / on cżyni ſpráwy.
Wierzćie mi żem Ja w Ojcu / á Oćiec we mnie : wżdy przynamniey dla ſámych ſpraw wierzćie mi.
Záprawdę záprawdę powiádam wam ; Kto wierzy w mię / ſpráwy które Ja cżynię / y on cżynić będźie / y więkƺe nád te cżynić będźie ; Bo Ja odchodzę do Ojcá mego.
A o cokolwiek prośić będźiećie w Imieniu mojim / to ucżynię : áby był uwielbiony Oćiec w Synu.
Jeſli o co będźiećie prośić w Imieniu mojim / Ja ucżynię.
Jeſli mię miłujećie przykazánia moje záchowajćie.
A Ja prośić będę Ojcá / á innego Poćieƺyćielá da wam / áby z wámi mieƺkał ná wieki :
Onego Duchá prawdy / którego świát przyjąć nie może ; bo go nie widźi / áni go zna : lecż wy go znaćie / gdyż u was mieƺka / y w was będźie.
Nie zoſtáwię was śierotámi : Przydę do was.
Jeƺcże málucżko / á świát mię już więcey nie ogląda ; lecż wy mnie oglądaćie : bo Ja żywię / y wy żyć będźiećie.
W on dźień wy poznaćie / żem Ja jeſt w Ojcu mojim / á wy we mnie / y Ja w was.
Kto ma przykazánia moje / y záchowywa je / ten jeſt który mię miłuje. A kto mnie miłuje / będźie go też miłował Oćiec mój ; y Ja go miłowáć będę / y objáwię mu śiebie ſámego.
Powiedźiał mu Judás / nie on Iƺkáryot ; PAnie / coż jeſt / że śię nam objáwić maƺ / á nie świátu?
Odpowiedźiał JEzus / y rzekł mu : Jeſli mię kto miłuje / ſłowá moje záchowywáć będźie : Y Oćiec mój umiłuje go / y do niego przydźiemy / á mieƺkánie u niego ucżynimy.
Kto mię nie miłuje / ſłów mojich nie záchowywa : á ſłowo / które ſłyƺyćie / nie jeſt moje / ále onego / który mię poſłał / Ojcá.
Tomći wam powiedźiał u was mieƺkájąc.
Lecż Poćieƺyćiel on / Duch Święty / którego pośle Oćiec w Imieniu mojim / onći was náucży wƺyſtkiego ; y przypomni wam wƺyſtko / comkolwiek wam powiedźiał.
Pokój zoſtáwuję wam / pokój on mój dawam wam : Nie jáko dawa świát / Ja wam dawam : Niechże śię nie trwoży ſerce wáƺe / áni śię lęka.
Słyƺeliśćie / żem Ja wam powiedźiał : Odchodzę / y záś przydę do was. Gdybyśćie mię miłowáli / wżdybyśćie śię rádowáli / żem rzekł : Idę do Ojcá ; bo Oćiec mój więkƺy jeſt niż Ja.
Y terazem wam powiedźiał / przed tym niż śię ſtánie : żebyśćie / gdy śię <i>to</i> ſtánie / uwierzyli.
Już dáliey z wámi wiele mowić nie będę ; Abowiem idźie Kśiążę świátá tego / á we mnie nic nie ma.
Ale iżby poznał świát / że miłuję Ojcá : á jáko mi rozkazał Oćiec / ták cżynię. Wſtańćieƺ / podźmy z tąd.
