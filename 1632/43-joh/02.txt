A dniá trzećiego było weſele w Kánie Gálilejſkiey : y byłá tám mátká JEzuſowá.
Wezwány też był y JEzus y ucżniowie jego / ná ono weſele.
A gdy nie ſtáło winá / rzekłá mátká JEzuſowá do niego : Winá nie máją.
Rzekł jey JEzus ; Co Ja mam z tobą niewiáſto? jeƺcżeć nie przyƺłá godźiná mojá.
Rzekłá mátká jego ſługom ; Cokolwiek wam rzecże / ucżyńćie.
Y było tám ƺeść ſtągwi kámiennych / poſtáwionych według ocżyśćienia Żydowſkiego / biorących w śię káżda dwie álbo trzy wiádrá.
Rzekł im JEzus ; Nápełnićie te ſtągwie wodą. Y nápełnili je áż do wierzchu.
Tedy im rzekł ; Cżerpajćież teraz / á donieśćie Przełożonemu weſela. Y donieśli.
A gdy ſkoƺtował Przełożony weſela oney wody / która śię ſtáłá winem / ( á nie wiedźiał z kądby było ; lecż ſłudzy wiedźieli którzy wodę cżerpáli ) záwołał on Przełożony weſela / oblubieńcá :
Y rzekł mu ; Káżdy cżłowiek pierwey dawa wino dobre ; á gdy ſobie podpiją / tedy podlejƺe : <i>á</i> tyś dobre wino záchował áż do tego cżáſu.
Tenći pocżątek cudów ucżynił JEzus w Kánie Gálilejſkiey / á objáwił chwałę ſwoję ; y uwierzyli weń ucżniowie jego.
Potym zſtąpił do Kápernáum / on / y mátká jego / y brácia jego / y ucżniowie jego / y zmieƺkáli tám nie wiele dni.
Abowiem byłá bliſko Wielkanoc Żydowſka / y wſtąpił JEzus do Jeruzalem.
Y ználazł w Kościele śiedzące te co przedawáli woły / y owce / y gołębie / y te co pieniędzmi hándlowáli.
A ucżyniwƺy bicż z powrozków / wƺyſtkie wygnał z Kośćiołá / y owce / y woły : á tych co pieniędzmi hándlowáli / pieniądze rozſypał / y ſtoły poprzewracał.
A tym co gołębie przedawáli / rzekł ; Wynieśćie to z tąd / á nie cżyńćie domu Ojcá mego / domem kupieckim.
Y wſpomnieli ſobie ucżniowie jego / iż nápiſano ; Gorliwość domu twego zżárłá mię.
Tedy odpowiedźieli Żydowie / y rzekli mu : Cóż nam zá znák pokażeƺ / iż to cżyniƺ?
Odpowiedźiał JEzus / y rzekł im : Rozwalćie ten Kościół ; á we trzech dniách wyſtáwię go.
Rzekli tedy Żydowie ; Cżterdźieśći y ƺeść lat budowano ten Kościół / á ty go we trzech dniách wyſtáwiƺ?
Ale on mówił o kośćiele ćiáłá ſwego.
Przetoż gdy zmartwychwſtał wſpomnieli ucżniowie jego / iż im to był powiedźiał : y uwierzyli piſmu / y ſłowu / które wyrzekł JEzus.
A gdy był w Jeruzalem ná Wielkanoc w Święto / wiele ich uwierzyło w Imię jego / widząc cudá jego / które cżynił.
Ale JEzus nie zwierzał im ſámego śiebie / przeto iż on znał wƺyſtkie :
A iż nie potrzebował / áby <i>mu</i> kto świádectwo wydawał o cżłowieku : ábowiem on wiedźiał co było w cżłowieku.
