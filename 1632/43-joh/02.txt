A dniá trzećiego było weſele w Kánie Gálilejſkiey : y byłá tám mátká Jezusowá.
Wezwány też był y JEzus y ucżniowie jego / ná ono weſele.
A gdy nie ſtáło winá / rzekłá mátká JEzusowá do niego : Winá nie máją.
Rzekł jey JEzus ; Co Já mám z tobą / niewiáſto? jeƺcżeć nie przyƺłá godźiná mojá.
Rzekłá mátká jego ſługom ; Cokolwiek wám rzecże / ucżyńćie.
Y było tám ƺeść ſtągwi kámiennych / poſtáwionych według ocżyśćieniá Żydowſkiego / biorących w śię káżdá dwie álbo trzy wiádrá.
Rzekł im JEzus ; Nápełnićie te ſtągwie wodą. Y nápełnili je áż do wierzchu.
Tedy im rzekł ; Cżerpájćież teráz / á donieśćie Przełożonemu weſelá. Y donieśli.
A gdy ſkoƺtowáł Przełożony weſelá oney wody / ktora śię ſtáłá winem / ( á nie wiedźiáł z kądby było ; lecż ſłudzy wiedźieli ktorzy wodę cżerpáli ) / záwołáł on Przełożony weſelá / oblubieńcá :
Y rzekł mu ; Káżdy cżłowiek pierwey dawa wino dobre ; á gdy ſobie podpiją / tedy podlejƺe : á tyś dobre wino záchował áż do tego cżáſu.
Tenći pocżątek cudow ucżynił JEzus w Kánie Gálilejſkiey / á objáwił chwáłę ſwoję ; y uwierzyli weń ucżniowie jego.
Potym zſtąpił do Kápernáum / on / y mátká jego / y bráciá jego / y ucżniowie jego / y zmieƺkáli tám nie wiele dni ;
Abowiem byłá bliſko Wielkanoc Żydowſká / y wſtąpił JEzus do Jeruzalem.
Y ználazł w Kościele śiedzące te co przedawáli woły / y owce / y gołębie / y te co pieniędzmi hándlowáli.
A ucżyniwƺy bicż z powrozkow / wƺyſtkie wygnáł z Kośćiołá / y owce / y woły : á tych co pieniędzmi hándlowáli / pieniądze rozſypáł / y ſtoły poprzewracał ;
A tym co gołębie ſprzedawáli/ rzekł ; Wynieśćie to z tąd / á nie cżyńćie domu Ojcá mego / domem kupieckim.
Y wſpomnieli ſobie ucżniowie jego / iż nápiſáno ; Gorliwość domu twego zżárłá mię.
Tedy odpowiedźieli Żydowie / y rzekli mu : Coż nám zá znák pokáżeƺ / iż to cżyniƺ?
Odpowiedźiáł JEzus / y rzekł im : Rozwalćie ten Kościoł ; á we trzech dniách wyſtáwię go.
Rzekli tedy Żydowie ; Cżterdźieśći i ƺeść lat budowáno ten Kościoł ; á ty go we trzech dniách wyſtáwiƺ?
Ale on mowił o kośćiele ćiáłá ſwego.
Przetoż gdy zmartwychwſtał wſpomnieli ucżniowie jego / iż im to był powiedźiáł : y uwierzyli piſmu / y ſłowu / ktore wyrzekł Jezus.
A gdy był w Jeruzalem ná Wielkanoc w Święto / wiele ich uwierzyło w Imię jego / widząc cudá jego / ktore cżynił.
Ale JEzus nie zwierzáł im ſámego śiebie / przeto iż on znáł wƺyſtkie :
A iż nie potrzebował / áby <i>mu</i> kto świádectwo wydawał o cżłowieku ; ábowiem on wiedźiáł co było w cżłowieku.
