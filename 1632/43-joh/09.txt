A mimo idąc / ujrzał cżłowieká ślepego od národzenia.
Y pytáli go ucżniowie jego / mówiąc : Miſtrzu / któż zgrzeƺył / ten / cżyli rodźicy jego? iż śię ślepym národźił.
Odpowiedźiał JEzus ; Ani ten zgrzeƺył / áni rodźicy jego : ále żeby śię okazáły ſpráwy Boże ná nim.
Jać muƺę ſpráwowáć ſpráwy <i>onego</i> który mię poſłał / pokąd dźień jeſt : przychodźi noc / gdy żaden nie będźie mógł <i>nic</i> ſpráwowáć.
Pókim jeſt ná świećie / jeſtem świátłośćią świátá.
To rzekƺy / plunął ná źiemię : á ucżynił błoto z śliny / y pomázał onym błotem ocży ślepego.
Y rzekł mu : Idź / umyj śię w ſadzawce Syloe / co śię wykłada / Poſłány. Poƺedł tedy y umył śię / y przyƺedł widząc.
A ták ſąśiedźi / y którzy go przed tym widáli ślepego / mówili ; Izali nie ten jeſt / który śiadał / y żebrał?
Drudzy mówili : Iż ten jeſt. A drudzy / Iż jeſt jemu podobny. <i>Lecż</i> on mówił / żem ja jeſt.
Tedy mu rzekli : Jákoż ſą otworzone ocży twoje?
A on odpowiedźiáł y rzekł : Cżłowiek którego zowią JEzuſem / ucżynił błoto y pomázał ocży moje / á rzekł mi : Idź do ſadzawki Syloe / á umyj śię. A ták odƺedƺy y umywƺy śię / przejrzałem.
Tedy mu rzekli : Gdźież on jeſt? Rzekł ; Nie wiem.
Tedy przywiedli onego który przed tym był ślepy do Fáryzeuƺów.
A był Sábbát gdy JEzus ucżynił błoto / y otworzył ocży jego.
Tedy go znowu pytáli y Fáryzeuƺowie / jáko przejrzał? A on im rzekł : Włożył mi błotá ná ocży ; y umyłem śię / y widzę.
Tedy niektóry z Fáryzeuƺów rzekł : Cżłowiek ten nie jeſt z Bogá ; bo nie ſtrzeże Sábbátu. Drudzy záśię mówili ; Jákoż może cżłowiek grzeƺny tákowe cudá cżynić? Y było rozerwánie miedzy nimi.
Rzekli tedy ślepemu powtóre ; Ty co mówiƺ o nim / ponieważ otworzył ocży twoje? A on rzekł ; Prorok jeſt.
A nie wierzyli Żydowie o nim żeby był ślepym / á że przejrzał : áż záwołáli rodźiców onego / który przejrzał.
Y pytáli ich mówiąc ; Tenże jeſt ſyn wáƺ / o którym wy powiádaćie / iż śię ślepo národźił : jákoż wżdy teraz widźi?
Odpowiedźieli im rodźicy jego / y rzekli ; Wiemy żeć to jeſt ſyn náƺ / y że śię ślepo národźił.
Lecż jáko teraz widźi / nie wiemy ; álbo kto otworzył ocży jego / my nie wiemy : Mać látá / pytajćież go ; on ſam o ſobie powie.
Ták mówili rodźicy jego / że śię bali Żydów : ábowiem już byli Żydowie poſtánowili / áby / ktoby go kolwiek CHryſtuſem wyznał / był z bóżnice wyłącżony.
Przetoż rzekli rodźicy jego : Mać látá / pytajćież go.
Tedy záwołáli powtóre cżłowieká onego który był ślepy / y rzekli mu : Daj chwałę Bogu : myć wiemy / iż ten cżłowiek jeſt grzeƺny.
A on odpowiedźiał / y rzekł : Jeſli grzeƺny jeſt / nie wiem : to tylko wiem / iż będąc ślepym teraz widzę.
Y rzekli mu znowu : Cóżći ucżynił? Jákoż otworzył ocży twoje?
Odpowiedźiał im : Jużemći wam powiedźiał / á nie ſłyƺeliśćie? przecżże jeƺcże ſłyƺeć chcećie? Izali y wy chcećie być ucżniámi je°?
Tedy mu złorzecżyli / y rzekli : Ty bądź ucżniem jego : áleſmy my ucżniámi Mojzeƺowymi.
My wiemy że Bóg do Mojzeƺá mówił : lecż ten z kądby był nie wiemy.
Odpowiedźiał <i>on</i> cżłowiek / y rzekł im : Toć záprawdę rzecż dźiwna : że wy nie wiećie z kąd jeſt / á otworzył ocży moje.
A wiemy iż Bóg grzeƺników nie wyſłuchawa : ále jeſliby kto chwalcą Bożym był / y wolą jego cżynił / tego wyſłuchawa.
Od wieku nie ſłychano / áby kto otworzył ocży ślepo národzonego.
Być ten nie był od Bogá / nie mógłćiby nic ucżynić.
Odpowiedźieli y rzekli mu : Tyś śię wƺyſtek w grzechách národźił / á ty nas ucżyƺ? Y wygnáli go precż.
A uſłyƺawƺy JEzus iż go precż wygnáli / y ználazƺy go / rzekł mu : Wierzyƺże ty w Syná Bożego?
A on odpowiedźiał y rzekł : A któż jeſt PAnie / ábym weń wierzył?
Y rzekł mu JEzus : Y widźiałeś go / y który mówi z tobą / onći jeſt.
A on rzekł : Wierzę PAnie : y pokłonił mu śię.
Y rzekł mu JEzus : Ná ſądemći Ja przyƺedł ná ten świát / áby ći którzy nie widzą / widźieli : á ći którzy widzą / áby ślepymi byli.
Y uſłyƺeli to niektórzy z Fáryzeuƺów / którzy byli z nim / y rzekli mu : Izali y my ślepymi jeſteſmy?
Rzekł im JEzus : Byśćie byli ślepymi / nie mielibyśćie grzechu. Lecż teraz mówićie : Iż widźiemy : Przetoż grzech wáƺ zoſtawa.
