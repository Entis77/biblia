A mimo idąc / ujrzał cżłowieká ślepego od národzenia.
Y pytáli go ucżniowie jego / mowiąc : Miſtrzu / ktoż zgrzeƺył / ten / cżyli rodźicy jego? iż śię ślepym národźił.
Odpowiedźiał JEzus ; Ani ten zgrzeƺył / áni rodźicy jego : ále żeby śię okazáły ſpráwy Boże ná nim.
Jać muƺę ſpráwowáć ſpráwy <i>onego</i> ktory mię poſłał / pokąd dźień jeſt : przychodźi noc / gdy żaden nie będźie mogł <i>nic</i> ſpráwowáć.
Pokim jeſt ná świećie / jeſtem świátłośćią świátá.
To rzekƺy plunął ná źiemię : á ucżynił błoto z śliny / y pomázał onym błotem ocży ślepego.
Y rzekł mu : Idź / umyj śię w ſadzawce Syloe / co śię wykłada  / Poſłány. Poƺedł tedy y umył śię / y przyƺedł widząc.
A ták ſąśiedźi / y ktorzy go przedtym widáli ślepego / mowili ; Izali nie ten jeſt / ktory śiadał / y żebrał?
Drudzy mowili : Iż ten jeſt. A drudzy / Iż jeſt jemu podobny. <i>Lecż</i> on mowił / żem ja jeſt.
Tedy mu rzekli : Jákoż ſą otworzone ocży twoje?
A on odpowiedźiáł y rzekł : Cżłowiek ktorego zowią JEzuſem / ucżynił błoto y pomázał ocży moje / á rzekł mi : Idź do ſadzawki Syloe / á umyj śię. A ták odƺedƺy y umywƺy śię / przejrzałem.
Tedy mu rzekli : Gdźież on jeſt? Rzekł ; Nie wiem.
Tedy przywiedli onego ktory przed tym był ślepy do Fáryzeuƺow.
A był Sábbát gdy JEzus ucżynił błoto / y otworzył ocży jego.
Tedy go znowu pytáli y Fáryzeuƺowie / jáko przejrzał? A on im rzekł : Włożył mi błotá ná ocży ; y umyłem śię / y widzę.
Tedy niektory z Fáryzeuƺow rzekł : Cżłowiek ten nie jeſt z Bogá ; bo nie ſtrzeże Sábbátu. Drudzy záśię mowili ; Jákoż może cżłowiek grzeƺny tákowe cudá cżynić? Y było rozerwánie między nimi.
Rzekli tedy ślepemu powtore ; Ty co mowiƺ o nim / ponieważ otworzył ocży twoje? A on rzekł ; Prorok jeſt.
A nie wierzyli Żydowie o nim żeby był ślepym / á że przejrzał : áż záwołáli rodźicow onego / ktory przejrzał.
Y pytáli ich mowiąc ; Tenże jeſt ſyn wáƺ / o ktorym wy powiádaćie / iż śię ślepo národźił : jákoż wżdy teraz widźi?
Odpowiedźieli im rodźicy jego / y rzekli ; Wiemy żeć to jeſt ſyn náƺ / y że śię ślepo národźił.
Lecż jáko teraz widźi / nie wiemy ; álbo kto otworzył ocży jego / my nie wiemy : Mać látá / pytájćież go ; on ſam o ſobie powie.
Ták mowili rodźicy jego / że śię bali Żydow : ábowiem już byli Żydowie poſtánowili / áby / ktoby go kolwiek Chryſtuſem wyznáł / był z bożnice wyłącżony.
Przetoż rzekli rodźicy jego : Mać látá / pytajćież go.
Tedy záwołáli powtore cżłowieká onego ktory był ślepy / y rzekli mu : Daj chwałę Bogu : myć wiemy / iż ten cżłowiek jeſt grzeƺny.
A on odpowiedźiał / y rzekł : Jeśli grzeƺny jeſt / nie wiem : to tylko wiem / iż będąc ślepym teraz widzę.
Y rzekli mu znowu : Cożći ucżynił? Jákoż otworzył ocży twoje?
Odpowiedźiał im : Jużemći wam powiedźiał / á nie ſłyƺeliśćie? przecżże jeƺcże ſłyƺeć chcećie? Izali y wy chcećie być ucżniámi je°?
Tedy mu złorzecżyli / y rzekli : Ty bądź ucżniem jego : áleśmy my ucżniámi Mojżeƺowymi.
My wiemy / że Bog do Mojżeƺá mowił : lecż ten / z kądby był nie wiemy.
Odpowiedźiał <i>on</i> cżłowiek / y rzekł im : Toć záprawdę rzecż dźiwna : że wy nie wiećie z kąd jeſt / á otworzył ocży moje.
A wiemy iż Bog grzeƺnikow nie wyſłuchawa : ále jeśliby kto chwalcą Bożym był / y wolą jego cżynił / tego wyſłuchawa.
Od wieku nie ſłyƺano / áby kto otworzył ocży ślepo národzonego.
Być ten nie był od Bogá / nie mogłćiby nic ucżynić.
Odpowiedźieli y rzekli mu : Tyś śię wƺyſtek w grzechách národźił / á ty nas ucżyƺ? Y wygnáli go precż.
A uſłyƺawƺy JEzus iż go precż wygnáli y ználazƺy go / rzekł mu : Wierzyƺże ty w Syná Bożego?
A on odpowiedźiał y rzekł : A ktoż jeſt PAnie / ábym weń wierzył?
Y rzekł mu JEzus : Y widźiałeś go / y ktory mowi z tobą / onći jeſt.
A on rzekł : Wierzę PAnie : y pokłonił mu śię.
Y rzekł mu JEzus : Ná ſądemći Ja przyƺedł ná ten świát / áby ći ktorzy nie widzą / widźieli : á ći ktorzy widzą / áby ślepymi byli.
Y uſłyƺeli to niektorzy z Fáryzeuƺow / ktorzy byli z nim / y rzekli mu ; Izali y my ślepymi jeſteśmy?
Rzekł im JEzus : Byśćie byli ślepymi / nie mielibyśćie grzechu. Lecż teraz mowićie : Iż widźiemy : Przetoż grzech wáƺ zoſtawa.
