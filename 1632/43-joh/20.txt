A pierwƺego dniá po Sábbáćie / Márya Mágdálená przyƺłá ráno do grobu / gdy jeƺcże było ćiemno / y ujrzáłá kámień odwálony od grobu.
Y bieżáłá á przyƺłá do Symoná Piotrá / y do <i>onego</i> drugiego ucżniá którego miłował JEzus / y rzekłá im : Wźięli PAná z grobu ; á nie wiemy gdźie go położyli.
Tedy wyƺedł Piotr / y on drugi ucżeń / á ƺli do grobu.
Y bieżeli obá ſpołem ; ále on drugi ucżeń wyśćignął Piotrá / y pierwey przyƺedł do grobu.
A náchyliwƺy śię / ujrzał leżące prześćierádłá : wƺákże tám nie wƺedł.
Przyƺedł też y Symon Piotr idąc zá nim / y wƺedł w grob / y ujrzał prześćierádłá leżące :
Y chuſtkę która byłá ná głowie jego ; nie z prześćierádły położoną / ále z oſobná ná jednym miejſcu zwinioną.
Potym wƺedł y on drugi ucżeń / który był pierwey przyƺedł do grobu ; y ujrzał / á uwierzył.
Abowiem jeƺcże nie rozumieli piſmá / iż miał zmartwychwſtáć.
Y odeƺli záś oni ucżniowie do domu.
Ale Máryá ſtałá u grobu ná dworze płácżąc. A gdy płákáłá / náchyliłá śię w grob.
Y ujrzáłá dwu Anjołów w bieli śiedzących ; jednego u głowy / á drugiego u nog / tám gdźie było położone ćiáło JEzuſowe.
Którzy jey rzekli ; Niewiáſto / cżemu płácżeƺ? Rzekłá im ; Iż wźięli PAná mego / á nie wiem gdźie go położyli.
A to rzekłƺy / obroćiłá śię ná zad / y ujrzáłá JEzuſá ſtojącego : lecż nie wiedźiáłá iż JEzus był.
Rzekł jey JEzus : Niewiáſto / cżemu płácżeƺ? kogo ƺukaƺ? A oná mniemájąc żeby był ogrodnik / rzekłá mu : PAnie / jeśliś go ty wźiął / powiedz mi gdźieś go położył / á ja go wezmę.
Rzekł jey JEzus : Márya. Która obroćiwƺy śię / rzekłá mu ; Rábboni : co śię wykłada : Náucżyćielu.
Rzekł jey JEzus ; Nie dotykaj śię mnie / bom jeƺcże nie wſtąpił do Ojcá mego ; ále idź do bráći mojey / á powiedz im : Wſtępuję do Ojcá mego y Ojcá wáƺego ; y <i>do</i> Bogá mego y Bogá wáƺego.
<i>Tedy</i> przyƺłá Márya Mágdálená / oznájmując ucżniom że widźiáłá PAná / á że jey to powiedźiał.
A gdy był wiecżor dniá onego pierwƺego po Sábbáćie / á drzwi były zámknione / gdźie byli ucżniowie zgromádzeni dla bojáźni Żydowſkiey / przyƺedł JEzus / y ſtánął w pośrodku <i>nich /</i> y rzekł im ; Pokój wam.
A to rzekƺy / pokazał im ręce y bok ſwój. Y urádowáli śię ucżniowie ujrzawƺy PAná.
Rzekł im záśię JEzus : Pokój wam. Jáko mię poſłał Oćiec / ták y Ja was poſyłam.
A to rzekƺy / tchnął <i>na nie /</i> y rzekł im ; Weźmićie Duchá Świętego.
Którymkolwiek grzechy odpuśćićie / będą im odpuƺcżone / á którymkolwiek zátrzymáćie / będą zátrzymáne.
A Tomaƺ jeden ze dwunaśći / którego zowią Dydymus / nie był z nimi / gdy był przyƺedł JEzus.
Y rzekli mu drudzy ucżniowie : Widźieliśmy PAná. Ale im on rzekł ; Jeśli nie ujrzę w ręku jego znáków goźdźi / á nie włożę pálcá mego w znáki goźdźi / y nie włożę ręki mojey w bok jego / nie uwierzę.
A po ośmi dni / byli záśię ucżniowie jego w domu / y Tomaƺ z nimi : Y przyƺedł JEzus gdy były drzwi zámknione / á ſtánął w pośrodku <i>nich /</i> y rzekł ; Pokój wam.
Potym rzekł Tomaƺowi : Włoż ſám pálec twój / á oglądaj ręce moję ; y zćiągni rękę twoję / y włoż ją w bok mój : á nie bądź niewiernym / ále wiernym.
Tedy odpowiedźiał Tomaƺ y rzekł mu ; PAn mój / y Bog mój.
Rzekł mu JEzus ; Żeś mię ujrzał / <i>Tomaƺu /</i> uwierzyłeś : Błogoſłáwieni którzy nie widźieli / á uwierzyli.
Wieleć y innych cudów ucżynił JEzus przed ocżymá ucżniów ſwojich / które nie ſą nápiſáne w tych kśięgách.
Ale te ſą nápiſáne / ábyśćie wierzyli / że JEzus jeſt Chryſtus Syn Boży : á żebyśćie wierząc / żywot mieli w Imieniu jego.
