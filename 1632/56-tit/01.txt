Páweł ſługá Boży / y Apoſtoł JEzuſá CHryſtuſá / według wiáry wybránych Bożych / y znájomośći prawdy ; która <i>jeſt</i> według pobożnośći.
Ku nádźiejey żywotá wiecżnego / który obiecał przed cżáſy wiekujiſtymi ten / który nie kłama / Bóg ; á objáwił cżáſów ſwojich /
To ſłowo ſwoje / przez kazánie które mi jeſt zwierzone / według rozrządzenia Zbáwićielá náƺego Bogá.
Tytuſowi właſnemu ſynowi według ſpólney wiáry / Łáſká / miłośierdźie y pokój / <i>niech będźie</i> od Bogá Ojcá / y PAná JEzuſá CHryſtuſá Zbáwićielá náƺego.
Dla tegom ćię zoſtáwił w Krećie / ábyś to co tám <i>jeƺcże</i> zoſtawa / w rząd dobry wpráwił / y poſtánowił po miáſtách ſtárƺe / jákomći ja był rozkazał.
Jeſli kto jeſt bez nágány / mąż jedney żony / dźieći wierne májący / któreby nie mogły bydź obwinione w zbytku / álbo niepoddáne rządowi.
Abowiem Biſkup ma być bez nágány / jáko ƺáfarz Boży / nie ſam ſobie śię podobájący / nie gniewliwy / nie pijánicá winá / nie bitny / nie ƺukájący zyſku ſproſnego :
Ale gośćinny / dobre miłujący / roſtropny / ſpráwiedliwy / świętobliwy / powśćiągliwy :
Trzymájący śię oney wierney mowy / która jeſt według náuki ; iżby też mógł nápomináć náuką zdrową / y tych którzy śię zprzećiwiáją przekonywáć.
Abowiem jeſt wiele rządowi niepoddánych / próznomównych / y zwodźićielów myśli ; á nawięcey tych którzy ſą z obrzezki :
Którym trzebá uſtá zátkáć ; którzy cáłe domy podwracáją / ucżąc rzecży nieſłuƺnych / dla zyſku ſproſnego.
Powiedźiał niektóry z nich właſny ich Prorok : Kreteńcżycy záwżdy ſą kłamcámi / złemi beſtyámi / brzuchámi leniwymi.
To świádectwo jeſt prawdźiwe : Dla którey przycżyny oſtrze je karż / áby zdrowi byli w wierze :
Nie pilnując Żydowſkich baśni / y przykazań ludźi tych / którzy śię odwracáją od prawdy.
Wƺyſtkoć cżyſte cżyſtym : lecż pokalánym y niewiernym niemáƺ nic cżyſtego / ále pokalány jeſt y umyſł y ſumnienie ich.
Udawáją że Bogá znáją : ále ucżynkámi <i>ſwymi tego</i> śię zápieráją / będąc obrzydłymi y niepoſłuƺnymi / á do wƺelkiego dobrego ucżynku nie ſpoſobnymi.
