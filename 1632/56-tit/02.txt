A ty mów / co náleży ná zdrową náukę.
Stárcy / áby byli trzeźwi / powáżni / roztropni / zdrowi w wierze / w miłośći / w ćierpliwośći.
Tákże y ſtáre niewiáſty niech chodzą w ubiorze przyſtojnym / jáko przyſtoi świętym ; niech nie będą potwárliwe / nie kochájące śię w wielu winá / pocżćiwych rzecży náucżájące ;
Aby młodych páń rozumu ucżyły / jákoby mężów ſwoich / y dźiátki miłowáć miáły /
A były roztropne / cżyſte / domu pilne / dobrotliwe / mężom ſwym poddáne / áby ſłowo Boże nie było bluźnione.
Młodźieńców tákże nápomináj / áby byli trzeźwi ;
We wƺyſtkiem ſámego śiebie wyſtáwiájąc zá wzór dobrych ucżynków / májąc w náuce cáłość / powágę /
Słowo zdrowe / nienágánione / áby ten / ktoby śię ſprzećiwił / záwſtydźić śię muśiáł / nie májąc nic / coby o wáſ miáł złego mówić.
Sług náucżáj / áby byli poddáni pánom ſwoim / we wƺyſtkiem śię im podobájąc / nie odmáwiájąc /
W nicżem nie oƺukując / ále we wƺyſtkiem wierność uprzejmą pokázując / áby náukę zbáwićielá náƺego Bogá we wƺyſtkiem zdobili.
Abowiem okázáłá śię łáſká Bożá / zbáwienná wƺyſtkim ludźiom /
Ćwicżącá náſ / ábyśmy odrzekłƺy śię niepobożnośći y świeckich pożądliwośći / trzeźwie / y ſpráwiedliwie y pobożnie żyli ná tym świećie /
Ocżekując onej błogoſłáwionej nádźiei / y objáwieniá chwáły wielkiego Bogá y zbáwićielá náƺego JEzuſá CHryſtuſá ;
Który dáł ſámego śiebie zá náſ / áby náſ wykupił od wƺelkiej niepráwośći / y ocżyśćił ſobie ſámemu lud włáſny / gorliwie náśládujący dobrych ucżynków.
To mów y nápomináj / y ſtrofuj ze wƺelką powágą ; żáden tobą niecháj nie gárdźi.
