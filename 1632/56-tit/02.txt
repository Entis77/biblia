A ty mów co należy ná zdrową náukę.
Stárcy áby byli trzeźwi / poważni / roſtropni / zdrowi w wierze / w miłośći / w ćierpliwośći.
Tákże <i>y</i> ſtáre niewiáſty niech chodzą w ubierze przyſtojnym / jáko przyſtoji świętym : niech nie będą potwarliwe / nie kochájące śię w wielu winá / pocżćiwych rzecży náucżájące :
Aby młodych <i>pań</i> rozumu ucżyły / <i>jákoby</i> męże ſwoje y dźiatki miłowáć miáły :
A <i>były</i> roſtropne / cżyſte / domu pilne / dobrotliwe / mężom ſwym poddáne / áby ſłowo Boże nie było bluźnione.
Młodźieńce tákże nápominaj áby byli trzeźwi.
We wƺyſtkim ſámego śiebie wyſtawiájąc zá wzór dobrych ucżynków / <i>májąc</i> w náuce cáłość / powagę /
Słowo zdrowe / nienágánione : áby ten ktoby śię zprzećiwił / záwſtydźić śię muśiał / nie májąc nic coby o was miał złego mówić.
Sług <i>náucżaj</i> áby byli poddáni pánom ſwojim / we wƺyſtkim śię <i>im</i> podobájąc / nie odmawiájąc.
<i>Ni wcżym</i> nieoƺukawájąc / ále we wƺyſtkim wierność uprzyjmą pokázując ; áby náukę Zbáwićielá náƺego Bogá we wƺyſtkim zdobili.
Abowiem okazáłá śię łáſká Bożá zbáwienna wƺyſtkim ludźiom.
Ćwicżąca nas / ábyſmy odrzekƺy śię niepobożnośći / y świeckich pożądliwośći / trzeźwie / y ſpráwiedliwie / y pobożnie żyli ná tym świećie.
Ocżekawájąc oney błogoſłáwioney nádźieje / y objáwienia chwały wielkiego Bogá y Zbáwićielá náƺego JEzuſá CHryſtuſá :
Który dał ſámego śiebie zá nas / áby nas wykupił od wƺelkiey niepráwośći / y ocżyśćił ſobie ſámemu lud właſny / gorliwie náśládujący dobrych ucżynków.
To mów y nápominaj y ſtrofuj / ze wƺelką powagą. Żaden tobą niechaj nie gárdźi.
