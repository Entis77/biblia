Y ſtáło śię roku jedenáſtego / pierwƺego dniá mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! przeto / iż Tyr mówił o Jeruzálemie wykrzykájąc : Hej / hej! zniƺcżone jeſt miáſto brám bárdzo ludnych / obrácá śię do mnie / teraz nápełniony będę / gdyż to jeſt ſpuſtoƺone ;
Dlátego ták mówi pánujący Pán : Otom Ja powſtáł przećiwko tobie / o Tyrze! á przywiodę ná ćię wiele národów / jákobym przywiódł morze z náwáłnośćiámi jego ;
Y zburzą mury Tyrſkie / y rozwálą wieże jego ; y wymiotę z niego proch jego / y ucżynię go wierzchołkiem ſkáły głádkiej /
Ták / że będą wyſuƺáć śieći w pośród morzá ; bom Ja rzekł / mówi pánujący Pán / przeto będźie ná rozchwycenie národom.
A córki jego / które będą ná polu / miecżem pobite będą / á dowiedzą śię / żem Ja Pán.
Bo ták mówi pánujący Pán : Oto Ja przywiodę przećiwko Tyrowi Nábuchodonozorá / królá Bábilońſkiego / od północy / królá nád królmi / z końmi y z wozámi / y z jezdnymi y z zgráją / y z ludem wielkim.
Córki twoje ná polu miecżem pomorduje / y przećiwko tobie porobi báƺty / y uſypie wáł przećiwko tobie / y poſtáwi przećiwko tobie tárcż ;
Y tárány záſádźi przećiwko murom twoim / á wieże twoje potłucże młotámi ſwemi.
Od mnóſtwá koni jego okryje ćię proch ich ; od grzmotu jezdnych y kár y wozów poruƺą śię mury twoje / gdy wchodźić będźie w brámy twoje / jáko wchodzą do miáſtá zburzonego.
Kopytámi koni ſwoich zdepcże wƺyſtkie ulice twoje / lud twój miecżem pobije / á mocne ſłupy twoje upádną ná źiemię ;
Y robiorą májętność twoję / á rozchwycą towáry twoje / y rozwálą mury twoje / y domy twoje rozkoƺne pobórzą / á kámienie twoje / y drzewo twoje / y proch twój do wody wrzucą.
Y ucżynię / że uſtánie głoſ pieśni twoich / á dźwięk hárf twoich nie będźie więcey ſłyƺány.
Y ucżynię ćię wierzchołkiem głádkiej ſkáły ; ſtánieƺ śię miejſcem ku wyſuƺániu śieći / nie będźieƺ więcey zbudowány ; bom Ja Pán powiedźiáł / mówi pánujący Pán.
Ták mówi pánujący Pán do Tyru : Izáli śię od trzáſku upádku twego / gdy ránni wołáć będą / gdy okrutne morderſtwo będźie w pośrodku ćiebie / wyſpy śię nie poruƺą?
Y powſtáną z ſtolic ſwoich wƺyſcy kśiążętá pomorſcy / y złożą z śiebie płáƺcże ſwoje / á ƺáty ſwe háftowáne zewleką / ſtráchem będą przyodźiáni / ná źiemi uśiądą / á wzdrygájąc śię co chwilá zdumiewáć śię będą nád tobą /
Y podnioſą nád tobą láment y rzekną do ćiebie : Jákoś zginęło / o miáſto! w którym mieƺkáno dla przyległośći morzá / miáſto ſłáwne / które było mocne ná morzu / ono y z obywátelámi ſwymi / którzy byli ſtráƺni wƺyſtkim obywátelom jego.
Tedy śię zátrwożą wyſpy w dźień upádku twego ; zátrwożą śię mówię wyſpy morſkie nád zginieniem twojim.
Bo ták mówi pánujący Pán : Gdy ćię ucżynię miáſtem ſpuſtoƺonem / jáko miáſtá / w których nie mieƺkáją / gdy ná ćię przepáść przywiodę / ták / że ćię wody wielkie przykryją ;
Gdy ucżynię / że zſtąpiƺ z tymi / którzy zſtępują do dołu / do ludu dáwnego / á położę ćię w nájniżƺych ſtronách źiemi / ná puſtyniách dáwnych / z tymi / co zſtępują do dołu / áby nie mieƺkáno w tobie / tedy dokáżę ſłáwy w źiemi żyjących.
Bo ucżynię to / że będźieƺ ná wielki poſtrách / gdy ćię nie ſtánie ; á choćby ćię ƺukáno / nie znájdą ćię ná wieki / mówi pánujący Pán.
