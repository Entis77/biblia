Stáło śię záś ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! zádáj zágádkę / á mów w podobieńſtwie o domu Izráelſkim /
Y rzecż : Ták mówi pánujący Pán : Orzeł wielki z wielkiemi ſkrzydłámi y z długiemi pióry / pełen pierzá pſtrego / przylećiáł ná Libán / y wźiął wierzch cedru.
Wierzch młodoćiánych látorośli jego ułámáł / y zánióſł go do źiemi kupieckiej / á w mieśćie kupieckiem położył go.
Wźiął też náśieniá onej źiemi / á wſádźił je ná polu urodzájnem / á wſádźił je bárdzo oſtrożnie przy wodách wielkich ;
Które weƺłoby było / y byłoby winną máćicą bujną / choć niſkiego wzroſtu ; y byłyby látorośli jej ku niemu / á korzenie jej byłyby mu poddáne. A ták byłoby było máćicą winną / któráby byłá wydáłá látorośli / y wypuśćiłá gáłązki.
Ale był orzeł jeden wielki z wielkiemi ſkrzydłámi y z gęſtem pierzem / á oto oná winná máćicá przypoiłá korzenie ſwoje ku niemu / y gáłązki ſwe rozćiągnęłá do niego / áby ją odwilżáł z brózd ſádu ſwego.
Choć ná polu dobrem przy wodách wielkich wſádzoná byłá / áby wypuśćiłá látorośli / y przynioſłá owoce / y byłá máćicą winną wſpániáłą.
Rzecżże tedy : Ták mówi pánujący Pán : Izáli śię jej poƺcżęśći? Izáli orzeł korzeniá jej nie wyrwie / y owocu jej nie oberwie / y nie poſuƺy? Izáli wƺyſtkich látorośli wyroſłych z niej nie uſuƺy? Izáli z wielką mocą á z obfitym ludem jej nie wygłádźi z korzeniá jej?
Oto jákożkolwiek wſádzoná jeſt / izáli śię jej poƺcżęśći? Izáli do ƺcżętu nie uſchnie / ſkoro śię jej dotknie wiátr wſchodni? Izáli przy brózdách / przy którch śię przyjęłá / nie uſchnie?
Zátem ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Mów teráz do domu odpornego : Azáż nie wiećie / co to jeſt? Zátem mów : Oto przyćiągnął król Bábilońſki do Jeruzálemu / y zábráł królá jego / y kśiążąt jego / y záwiódł ich z ſobą do Bábilonu.
Wźiął też z náśieniá królewſkiego / á ucżyniwƺy z nim przymierze / záwiązáł go przyśięgą / y mocnych z onej źiemi zábráł /
Aby było króleſtwo zniżone / przeto / żeby śię nie wynioſło / żeby ták ſtrzegąc przymierzá jego / ſtáło.
Ale mu śię ſprzećiwił / poſłáwƺy poſłów ſwych do Egiptu / áby mu dáł koni y wiele ludu. Izáli śię to poƺcżęśći? Izáli pomſty ujdźie ten / co ták ucżynił? Ten / który wzruƺá przymierze / izáli pomſty ujdźie?
Jáko żyję Já / mówi pánujący Pán / że ná miejſcu królá tego / który go królem ucżynił / którego przyśięgą wzgárdźił / á którego przymierze wzruƺył / u niego w Bábilonie umrze.
Ani mu Fáráo z wielkim wojſkiem y z ludem gęſtym co pomoże ná wojnie / gdy uſypie wáł / y porobi ƺáńce / áby wykorzenił mnóſtwo ludu /
Poniewáż pogárdźił przyśięgą / zrzućiwƺy przymierze ; bo oto dáł ná to rękę ſwą / á przećie to wƺyſtko ucżyni ; nie ujdźie pomſty.
Dlátego ták mówi pánujący Pán : Jáko żyję Já / że przyśięgę ſwoję / którą wzgárdźił / y przymierze moje / które wzruƺył / pewnie obrócę ná głowę jego.
Bo rozćiągnę náń śieć moję / á będźie niewodem moim pojmány / y záwiodę go do Bábilonu / á tám śię z nim rozſądzę o wyſtępek jego / którym wyſtąpił przećiwko mnie.
A wƺyſcy też / którzy ućiekli od niego ze wƺyſtkiemi hufámi jego / od miecżá polegną / á pozoſtáli ná wƺyſtkie ſtrony rozproƺeni będą / á dowiećie śię / że Já Pán mówiłem to.
Ták mówi pánujący Pán : Wƺákże wezmę z wierzchu tego wyſokiego cedru / y wſádzę ; z wierzchu młodoćiánych látorośli jego młodą látoroſtkę ułámię / á wƺcżepię ją ná górze wyſokiej y wynioſłej ;
Ná wyſokiej górze Izráelſkiej wƺcżepię ją / y wypuśći gáłązki / y wydá owoc / y będźie cedrem zácnym ; y będźie pod nim mieƺkáć wƺelkie ptáſtwo / y wƺyſtko / co má ſkrzydłá / pod ćieniem gáłęźiá jego mieƺkáć będźie.
A ták poznáją wƺyſtkie drzewá polne / żem Já Pán poniżył drzewo wyſokie / á wywyżƺyłem drzewo niſkie / uſuƺyłem drzewo źielone / á ucżyniłem / że zákwitło drzewo ſuche. Já Pán rzekłem to / y ucżynię.
