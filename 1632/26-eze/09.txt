Potem záwołał / gdźiem ja ſłyƺał / głoſem wielkim / mówiąc : Hetmáni náſtąpćie ná to miáſto / májąc káżdy broń ſwoję ku zábijániu w ręce ſwej.
A oto ƺeść mężów przyƺło drogą ku brami wyżƺej / która pátrzy ná północy / y miał káżdy broń ſwoję ku wytráceniu w ręce ſwej ; ále mąż jeden był w pośród nich odźiány ƺátą lniáną / á káłámárz piſárſki przy biodrách jego ; y przyƺedƺy ſtánęli u ołtárzá miedzánego.
A chwałá Bogá Izráelſkiego zſtąpiłá byłá z Cherubiná / ná którym byłá / do progu domu / y záwołáłá ná mężá onego odźiánego ƺátą lniáną / przy którego biodrách był káłámárz piſárſki.
Y rzekł Pán do niego : Przejdź przez pośrodek miáſtá / przez pośrodek Jeruzálemu / á ucżyń znák ná cżołách mężów / którzy wzdycháją y nárzekáją nád wƺyſtkiemi obrzydliwośćiámi / które śię dźieją w pośród niego.
A onym rzekł / gdźiem ja ſłyƺał : Idźćie po mieśćie zá nim ; á zábijájćie ; niech nie folguje oko wáƺe / áni śię zmiłujćie.
Stárcá / młodźieńcá / y pánnę / y málucżkich / y niewiáſty wybijćie do ƺcżętu ; ále do żádnego mężá / ná którymby był znák / nie przyſtępujćie / od świątnicy mojey pocżniećie. A ták pocżęli od onych mężów ſtárƺych / którzy byli przed domem Páńſkim ;
( Bo im był rzekł : Splugáwćie ten dom / á nápełnijćie śieni pobitymi ; idźćież. ) A wyƺedƺy zábijáli w miśćie.
A gdy ich pozábijáli / á jam pozoſtał / tedym pádł ná oblicże moje / y záwołáłem á rzekłem : Ach pánujący Pánie! izali ty wytráćiƺ wƺyſtek oſtátek Izráelſki / wylewájąc popędliwość ſwoję ná Jeruzálem?
Y rzekł do mnie : Niepráwość domu Izráelſkiego y Judzkiego náder jeſt bárzo wielka / y nápełnioná jeſt źiemiá krwią / á miáſto pełne jeſt przewrotnośći ; bo mówili : Pán tę źiemię opuśćił / á Pán nie widźi nas.
Przetoż y Ja toż ucżynię / nie ſfolguje oko moje / áni śię zmiłuję / drogę ich ná głowę ich obrócę.
A oto mąż on odźiány ƺátą lniáną / przy którego biodrách był káłámárz / oznájmił to mówiąc : Ucżyniłem ták / jákoś mi rozkázał.
