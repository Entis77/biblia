Potem záwołáł / gdziem já ſłyƺáł / głoſem wielkim / mówiąc : Hetmáni náſtąpćie ná to miáſto / májąc káżdy broń ſwoję ku zábijániu w ręce ſwej.
A oto ƺeść mężów przyƺło drogą ku brámi wyżƺej / którá pátrzy ná północy / y miáł káżdy broń ſwoję ku wytráceniu w ręce ſwej ; ále mąż jeden był w pośród nich odziány ƺátą lniáną / á káłámárz piſárſki przy biodrách jego ; y przyƺedłƺy ſtánęli u ołtárzá miedzánego.
A chwáłá Bogá Izráelſkiego zſtąpiłá byłá z Cherubiná / ná którym byłá / do progu domu / y záwołáłá ná mężá onego odziánego ƺátą lniáną / przy którego biodrách był káłámárz piſárſki.
Y rzekł Pán do niego : Przejdź przez pośrodek miáſtá / przez pośrodek Jeruzálemu / á ucżyń znák ná cżołách mężów / którzy wzdycháją y nárzekáją nád wƺyſtkiemi obrzydliwośćiámi / które śię dzieją w pośród niego.
A onym rzekł / gdziem já ſłyƺáł : Idźćie po mieśćie zá nim ; á zábijájćie ; niech nie folguje oko wáƺe / áni śię zmiłujćie.
Stárcá / młodzieńcá / y pánnę / y málucżkich / y niewiáſty wybijćie do ƺcżętu ; ále do żádnego mężá / ná którymby był znák / nie przyſtępujćie / od świątnicy mojej pocżniećie. A ták pocżęli od onych mężów ſtárƺych / którzy byli przed domem Páńſkim ;
( Bo im był rzekł : Splugáwćie ten dom / á nápełnijćie śieni pobitymi ; idźćież. ) A wyƺedłƺy zábijáli w miśćie.
A gdy ich pozábijáli / á jám pozoſtáł / tedym pádł ná oblicże moje / y záwołáłem á rzekłem : Ach pánujący Pánie! izáli ty wytráćiƺ wƺyſtek oſtátek Izráelſki / wylewájąc popędliwość ſwoję ná Jeruzálem?
Y rzekł do mnie : Niepráwość domu Izráelſkiego y Judzkiego náder jeſt bárdzo wielká / y nápełnioná jeſt ziemiá krwią / á miáſto pełne jeſt przewrotnośći ; bo mówili : Pán tę ziemię opuśćił / á Pán nie widzi náſ.
Przetoż y Já toż ucżynię / nie ſfolguje oko moje / áni śię zmiłuję / drogę ich ná głowę ich obrócę.
A oto mąż on odziány ƺátą lniáną / przy którego biodrách był káłámárz / oznájmił to mówiąc : Ucżyniłem ták / jákoś mi rozkázáł.
