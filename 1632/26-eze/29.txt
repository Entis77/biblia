Roku dzieſątego / dzieſątego mieśiącá / dwunáſtego dniá tegoż mieśiącá ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiwko Fáráonowi / królowi Egipſkiemu / á prorokuj przećiw niemu y przećiwko wƺyſtkiemu Egiptowi ;
Mów / á rzecż : Ták mówi pánujący Pán : Otom Já przećiwko tobie o Fáráonie / królu Egipſki / wielorybie wielki! który leżyƺ w pośrodku rzek twoich / y mówiƺ : Mojáć jeſt rzeká / y jám ją ſobie ucżynił ;
Przetoż włożę wędę w cżeluśći twoje / y ucżynię / że powięzną ryby rzek twoich ná łuſkách twych / y wywlekę ćię z pośrodku rzek twoich y wƺyſtkie ryby rzek twoich / które ná łuſkách twoich powięzną ;
Y zoſtáwię ćię ná puƺcży / ćiebie y wƺyſtkie ryby rzek twoich : polężeƺ ná polu / y nie będzieƺ zebrány áni zgromádzony ; dám ćię beſtyjom ziemſkim y ptáſtwu niebieſkiemu ku pożárćiu ;
Y dowiedzą śię wƺyſcy mieƺkájący w Egipćie / żem Já Pán / przeto żeśćie láſką trzćiniáną domowi Izráelſkiemu.
Gdy śię ćiebie ręką chwytáją / łámieƺ śię / y rozćináƺ im wƺyſtko rámię ; á gdy śię podpieráją tobą / kruƺyƺ śię / choć im nádſtáwiáƺ wƺyſtkich biódr.
Dlátegoż ták mówi pánujący Pán : Oto Já przywiodę ná ćię miecż / y wygłádzę z ćiebie cżłowieká y bydlę ;
A ziemiá Egipſká będzie puſtynią y ſpuſtoƺeniem / y dowiedzą śię / żem Já Pán / dlátego / żeś mówił : Rzeká mojá / y jám ją ucżynił ;
Przetoż oto Já będę przećiwko tobie y przećiwko rzece twojej / y podám ziemię Egipſką w ſpuſtoƺenie / y we ſrogie poburzenie / od wieży Sewene áż do gránic Murzyńſkich.
Nie przejdzie przez nią nogá cżłowiecżá / y nogá bydlęcá nie przejdzie przez nią / áni w niej będą mieƺkáć przez cżterdzieśći lát.
A ták ucżynię ziemię Egipſką puſtnią nád inne ziemie ſpuſtoƺone / á miáſtá jej nád inne miáſtá ſpuſtoƺone / będą ſpuſtoƺone przez cżterdzieśći lát : gdyż rozproƺę Egipcżán między národy / y rozwieję ich po ziemiách.
A wƺákże ták mówi pánujący Pán : Gdy śię ſkońcży cżterdzieśći lát / zgromádzę Egipcżán z národów / do których rozproƺeni będą.
Y przywrócę záśię więźniów Egipſkich / y przywiodę ich do ziemi Pátroſ / do ziemi mieƺkániá ich / y będą tám króleſtwem podłem.
Między innemi króleſtwámi będzie nájpodlejƺem / á nie wynieśie śię więcej nád inne národy / y umniejƺę ich / áby nie pánowáli nád národámi.
Y nie będzie więcej domowi Izráelſkiemu ufnośćią / któráby mi ná pámięć przywodziłá niepráwość / gdyby śię oglądáli ná nie ; y dowiedzą śię / żem Já pánujący Pán.
Potem ſtáło śię dwudzieſtego y śiódmego roku / pierwƺego mieśiącá / pierwƺego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! Nábuchodonozor / król Bábilońſki / przyniewolił gwáłtem wojſko ſwe do ſłużby ćiężkiej przećiwko Tyrowi ; káżdá głowá obłyśiáłá / y káżde rámię obnáżone / á przećie nie má zápłáty on / áni wojſko jego z Tyru zá onę ſłużbę / którą podejmowáł / wálcżąc przećiwko jemu.
Przetoż ták mówi pánujący Pán : Oto Já dáję Nábuchodonozorowi / królowi Bábilońſkiemu / ziemię Egipſką / áby zábráł doſtátki jej / y rozƺárpáł łupy jej / y rozchwyćił korzyśći jej / áby miáło zápłátę wojſko jego.
Zá prácę ich / którą dlá mnie podjęli / dám im ziemię Egipſką / przeto / że mnie prácowáli / mówi pánujący Pán.
Dniá onego ucżynię / że wyrośnie róg domu Izráelſkiego / tobie też uſtá twoje otworzę w pośrodku ich ; y dowiedzą śię / żem Já Pán.
