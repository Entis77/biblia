Y podnióſł mię duch / á przywiódł mię do bramy domu Páńſkiego wſchodniej / która pátrzy ná wſchód ſłońcá ; á oto w wejśćiu oney bramy było dwádźieśćiá y pięć mężów / miedzy którymi widźiałem Jázánijáƺá / ſyná Aſſurowego / y Pelátyjáƺá / ſyná Bánájáƺowego / kśiążąt ludu ;
Tedy mi rzekł : Synu cżłowiecży! onić to ſą mężowie / którzy zámyśláją o niepráwośći / á rádzą złą rádę w tym mieśćie /
Mówiąc : Nie budujmy domów bliſko ; boby ták miáſto było kotłem / á my mięſem.
Dla tego prorokuj przećiwko nim / prorokuj / ſynu cżłowiecży!
Tedy przypádł ná mię duch Páńſki / y rzekł do mnie : Mów : Ták mówi Pán : Tákeśćie mówili / domie Izráelſki! bo co wam kolwiek przychodźi ná myśl / to Ja znam ;
Wielkieśćie mnóſtwo pobili w tym mieśćie / á nápełniliśćie ulice jego pobitymi.
Dla tego ták mówi pánujący Pán : Którzy ſą pobići od was / którycheśćie ſkłádáli w pośrzodku jego / oni ſą mięſem / á miáſto kotłem ; ále was wywiodą z pośrzodku jego.
Báliśćie śię miecżá ; ále miecż przywiodę ná was / mówi pánujący Pán.
A wywiodę was z pośrzodku jego / á podám was w ręce obcych / y wykonam nád wámi ſądy.
Od miecżá polężećie / ná gránicách Izráelſkich oſądzę was / y dowiećie śię / żem Ja Pán.
Miáſto nie będźie wam kotłem / áni wy będźiećie w pośród jego mięſem ; ná gránicách Izráelſkich oſądzę was /
Y dowiećie śię / żem Ja Pán ; poniewáżeśćie w uſtáwách mojich nie chodźili / á ſądów mojich nie cżynili / áleśćie według ſądów tych narodów / którzy około was ſą / cżynili.
A gdym prorokował / tedy Pelátyjáƺ / ſyn Bánájáƺowy / umárł : dla tego upádłem ná twárz moję / á wołájąc głoſem wielkim / rzekłem : Ach / pánujący Pánie! do gruntu wygłádźiƺ oſtátki Izráelſkie.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! bráćia twoi / bráćia twoi / powinowáći twoi / y wƺyſtek dom Izráelſki / wƺyſtek mówię dom / ćić ſą / którym mówili obywátele Jeruzálemſcy : Oddálćie śię od Páná ; námći dáná jeſt tá źiemiá w ośiádłość.
Przetoż mów : Ták mówi pánujący Pán : Choćiáżem ich dáleko zágnał miedzy narody / y choćiáżem ich rozproƺył po źiemiách / wƺákże będę im świątnicą y przez ten máły cżás w źiemiách / do których przyjdą.
Przetoż mów : Ták mówi pánujący Pán : Zgromádzę was z narodów / á zbiorę was z źiem / do którycheśćie rozproƺeni / y dam wam źiemię Izráelſką.
Y wnijdą tám / á znioſą wƺyſtkie ſplugáwieniá jey / y wƺyſtkie jey obrzydliwośći z niej.
Bo im dam ſerce jedno / y duchá nowego dam do wnętrznośći wáƺych / y odejmę ſece kámienne z ćiáłá ich / á dam im ſerce mięśiſte /
Aby w uſtáwách mojich chodźili / á ſądów mojich ſtrzegli / y cżynili je ; y będą ludem moim / á Ja będę Bogiem ich.
Ale którychby ſerce chodźiło zá żądzámi plugáſtw ſwojich / y obrzydliwośći ſwojich / tych drogę obrócę ná głowy ich / mówi pánujący Pán.
Tedy podnieśli Cherubinowie ſkrzydłá ſwoje / y kołá z nimi / á chwałá Bogá Izráelſkiego byłá nád nimi z wierzchu.
Y odeƺłá chwałá Páńſka z pośrzodku miáſtá / á ſtánęłá ná górze / która jeſt ná wſchód miáſtá.
A duch podnióſł mię / y záś mię przywiódł do źiemi Cháldejſkiej do pojmánych / w widzeniu w Duchu Bożym. Y odeƺło odemnie widzenie / którem widźiał.
Y mówiłem do pojmánych te wƺyſtkie rzecży Páńſkie / które mi ukázał.
