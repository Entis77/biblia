Y podnióſł mię duch / á przywiódł mię do brámy domu Páńſkiego wſchodniej / która pátrzy ná wſchód ſłońcá ; á oto w wejśćiu oney brámy było dwádźieśćiá y pięć mężów / między którymi widźiáłem Jázánijáƺá / ſyná Aſſurowego / y Pelátyjáƺá / ſyná Bánájáƺowego / kśiążąt ludu ;
Tedy mi rzekł : Synu cżłowiecży! onić to ſą mężowie / którzy zámyśláją o niepráwośći / á rádzą złą rádę w tem mieśćie /
Mówiąc : Nie budujmy domów bliſko ; boby ták miáſto było kotłem / á my mięſem.
Dlátego prorokuj przećiwko nim / prorokuj / ſynu cżłowiecży!
Tedy przypádł ná mię duch Páńſki / y rzekł do mnie : Mów : Ták mówi Pán : Tákeśćie mówili / domie Izráelſki! bo co wám kolwiek przychodźi ná myśl / to Ja znám ;
Wielkieśćie mnóſtwo pobili w tem mieśćie / á nápełniliśćie ulice jego pobitymi.
Dlátego ták mówi pánujący Pán : Którzy ſą pobići od was / którycheśćie ſkłádáli w pośrodku jego / oni ſą mięſem / á miáſto kotłem ; ále was wywiodą z pośrodku jego.
Báliśćie śię miecżá ; ále miecż przywiodę ná was / mówi pánujący Pán.
A wywiodę was z pośrodku jego / á podám was w ręce obcych / y wykonám nád wámi ſądy.
Od miecżá polężećie / ná gránicách Izráelſkich oſądzę was / y dowiećie śię / żem Ja Pán.
Miáſto nie będźie wám kotłem / áni wy będźiećie w pośród jego mięſem ; ná gránicách Izráelſkich oſądzę was /
Y dowiećie śię / żem Ja Pán ; poniewáżeśćie w uſtáwách moich nie chodźili / á ſądów moich nie cżynili / áleśćie według ſądów tych národów / którzy około was ſą / cżynili.
A gdym prorokowáł / tedy Pelátyjáƺ / ſyn Bánájáƺowy / umárł : dlátego upádłem ná twárz moję / á wołájąc głoſem wielkim / rzekłem : Ach / pánujący Pánie! do gruntu wygłádźiƺ oſtátki Izráelſkie.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! bráćia twoi / bráćia twoi / powinowáći twoi / y wƺyſtek dom Izráelſki / wƺyſtek mówię dom / ćić ſą / którym mówili obywátele Jeruzálemſcy : Oddálćie śię od Páná ; námći dáná jeſt tá źiemiá w ośiádłość.
Przetoż mów : Ták mówi pánujący Pán : Choćiáżem ich dáleko zágnáł między národy / y choćiáżem ich rozproƺył po źiemiách / wƺákże będę im świątnicą y przez ten máły cżáſ w źiemiách / do których przyjdą.
Przetoż mów : Ták mówi pánujący Pán : Zgromádzę was z národów / á zbiorę was z źiem / do którycheśćie rozproƺeni / y dám wám źiemię Izráelſką.
Y wnijdą tám / á znioſą wƺyſtkie ſplugáwieniá jey / y wƺyſtkie jey obrzydliwośći z niej.
Bo im dám ſerce jedno / y duchá nowego dám do wnętrznośći wáƺych / y odejmę ſece kámienne z ćiáłá ich / á dám im ſerce mięśiſte /
Aby w uſtáwách moich chodźili / á ſądów moich ſtrzegli / y cżynili je ; y będą ludem moim / á Ja będę Bogiem ich.
Ale którychby ſerce chodźiło zá żądzámi plugáſtw ſwoich / y obrzydliwośći ſwoich / tych drogę obrócę ná głowy ich / mówi pánujący Pán.
Tedy podnieśli Cherubinowie ſkrzydłá ſwoje / y kołá z nimi / á chwałá Bogá Izráelſkiego byłá nád nimi z wierzchu.
Y odeƺłá chwałá Páńſká z pośrodku miáſtá / á ſtánęłá ná górze / która jeſt ná wſchód miáſtá.
A duch podnióſł mię / y záś mię przywiódł do źiemi Cháldejſkiej do pojmánych / w widzeniu w Duchu Bożym. Y odeƺło odemnie widzenie / którem widźiáł.
Y mówiłem do pojmánych te wƺyſtkie rzecży Páńſkie / które mi ukázáł.
