Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! oznájmij Jeruzálemowi obrzydliwośći jego / y rzecż :
Ták mówi pánujący Pán do córki Jeruzálemſkiej : Obcowánie twoje y ród twój jeſt z ziemi Chánánejſkiej ; ojćiec twój jeſt Amorejcżyk / á mátká twojá Hetejká.
A národzenie twoje tákie : W dzień / któregoś śię urodziłá / nie urzniono pępká twego / y wodą ćię nie obmyto dlá ochędożeniá / áni ćię ſolą poſolono / áni w pieluchy uwiniono.
Nie zlitowáło śię nád tobą oko / ábyć ucżyniło jedno z tych / ulitowáwƺy śię ćiebie ; ále ćię porzucono ná polu / przeto / żeś byłá obmierzłá w dzień / któregoś śię urodziłá.
A idąc mimo ćię / y widząc ćię ku podeptániu podáną we krwi twojej / rzekłem ći : Żyj we krwi twojej ; rzekłem ći / mówię : Żyj we krwi twojej.
Rozmnożyłem ćię ná tyśiące / jáko urodzáj polny / y rozmnożonáś / y ſtáłáś śię wielką / á przyƺłáś do bárdzo wielkiej ozdoby ; pierśi twoje odęły śię / á włoſy twoje uroſły / choćiáżeś byłá nágá y odkrytá.
Prztoż idąc mimo ćię / á widząc ćię / że oto cżáſ twój / cżáſ miłośći / rozćiągnąłem podołek mój ná ćię / y nákryłem nágość twoję / y obowiązáłem ći śię przyśięgą / á wƺedłem w przymierze z tobą / mówi pánujący Pán / y ſtáłáś śię moją.
Y omyłem ćię wodą / á ſpłókáwƺy krew twoję z ćiebie / pomázáłem ćię olejkiem ;
Nádto przyodziáłem ćię ƺátą háftowáną / y obułem ćię w koƺtowne trzewiki / y opáſáłem ćię biśiorem / á przyodziáłem ćię ƺátą jedwábną ;
Y przybráłem ćię w ochędoſtwo / á dáłem mánele ná ręce twoje / y łáńcuch złoty ná ƺyję twoję ;
Dáłem też nácżelnik ná cżoło twoje / á náuƺnice ná uƺy twoje / y koronę ozdobną ná głowę twoję ;
A ták byłáś ozdobioná złotem y ſrebrem / á odzienie twoje było biśior / y ƺátá jedwábná / y háftowáná ; jádáłáś bułkę y miód / y oliwę / á byłáś náder piękną / y ƺcżęśliwieć śię powodziło w króleſtwie /
Ták / że śię rozeƺłá powieść o tobie między národámi dlá pięknośći twojej ; boś doſkonáłą byłá dlá ſłáwy mojej / którąm był włożył ná ćię / mówi pánujący Pán.
Aleś ufáłá w pięknośći twojej / y płodziłáś nierząd / będąc ták ſłáwną ; boś płodziłá nierząd z káżdym mimo ćię idącym / káżdy ſnádnie użył pięknośći twojej.
A nábrwáƺy ƺát ſwoich nácżyniłáś ſobie wyżyn rozmáitych fárb / á płodziłáś nierząd przy nich / któremu podobny nigdy nie przyjdzie / áni będzie.
Nádto nábráwƺy klejnotów ozdoby ſwojej ze złotá mego y z ſrebrá mego / którem ći był dáł / nácżyniłáś ſobie obrázów pogłowiá męſkiego / y płodziłáś z nimi nierząd.
Wzięłáś też ƺáty ſwe háftowáne / á przyodziáłáś je / olejek mój y kádzidło moje kłádłáś przed nie ;
Náwet y chleb mój / którem ći dáł / bułkę y oliwę / y miód / którymem ćię kármił / kłádłáś przed nie ná wonność przyjemną / y było ták / mówi pánujący Pán.
Bráłáś też ſynów ſwoich / y córki ſwoje / którycheś mi národziłá / á oneś im ofiárowáłá ku pożárćiu ; izáli to jeƺcże máłe wƺetecżeńſtwá twoje?
Do tego y ſynóweś moich zábijáłá / á dáwáłáś ich / áby ich przenoƺono przez ogień im kwoli.
Nádto przy wƺyſtkich obrzydliwośćiách ſwoich / y wƺetecżeńſtwách ſwoich nie pámiętáłáś ná dni młodośći twojej / gdyś byłá nágą y odkrytą / podáną ná podeptánie we krwi twojej.
Nádto mimo wƺyſtkich onych złośći twoich ( Biádá / biádá tobie! mówi pánujący Pán. )
Zbudowáłáś ſobie dom nierządny / y wyſtáwiłáś ſobie wyżynę w káżdej ulicy.
Po wƺyſtkich rozſtániách dróg pobudowáłáś wyżyny ſwoje / á ucżyniłáś obmierzłą piękność ſwoję / rozkłádájąc nogi ſwoje káżdemu mimo idącemu / y rozmnożyłáś wƺetecżeńſtwá ſwoje.
Boś nierząd płodziłá z ſynámi Egipſkimi / ſąśiádámi twymi wielkich ćiáł / á rozmnożyłáś wƺetecżeńſtwá twe / ábyś mię do gniewu pobudzáłá.
Przetoż otom wyćiągnął rękę moję ná ćię / á uminiejƺyłem obroku twego / y podáłem ćię ná wolę nienáwidzących ćię córek Filiſtyńſkich / które śię wſtydzą zá złe drogi twoje.
Nádto płodziłáś nierząd z ſynámi Aſſyryjſkimi / przeto / żeś śię nie mogłá náſyćić / á nierząd płodząc z nimi / y tákeś śię nie náſyćiłá.
A ták rozmnożyłáś wƺetecżeńſtwo ſwe w ziemi Chánánejſkiej y Cháldejſkiej / á y ták nie náſyćiłáś śię.
O jáko jeſt zmámione ſerce twoje! mówi pánujący Pán / poniewáż śię dopuƺcżáƺ tych poſtępków niewiáſty nierządnej y wƺetecżnej.
Budując ſobie nierządne domy ná rozſtániu káżdego gośćińcá / á wyżynę ſobie ſtáwiájąc w káżdej ulicy / owƺem pogárdzájąc zápłátą nie jeſteś áni jáko wƺetecżnicá /
Ani jáko niewiáſtá cudzołożącá / którá mimo mężá ſwego obcych przypuƺcżá.
Wƺyſtkim wƺetecżnicom dáwáją zápłátę ; áleś ty dáwáłá zápłátę ſwoję wƺyſtkim zálotnikom twoim / y dáwáłáś im upominki / áby chodzili do ćiebie zewƺąd ná wƺetecżeńſtwá z tobą.
A ták znájduje śię przy tobie przećiwná rzecż od innych niewiáſt we wƺetecżeńſtwách twoich / poniewáż ćię dlá wƺetecżeńſtwá nie ƺukáją ; ále ty dájeƺ zápłátę / á nie tobie zápłátę dáwáją / co śię opák dzieje.
Przetoż / o wƺetecżnico! ſłucháj ſłowá Páńſkiego.
Ták mówi pánujący Pán : Dlátego / iż śię wyláłá ſprośność twojá / y odkryłá śię nágość twojá we wƺetecżeńſtwách twoich z zálotnikámi twoimi y ze wƺyſtkimi plugáwemi báłwánámi obrzydliwośći twoich / y dlá rozlániá krwi ſynów twoich / którycheś im dáłá :
Przetoż oto Já zgromádzę wƺyſtkich zálotników twoich / z którymiś obcowáłá / y wƺyſtkich / którycheś miłowáłá / ze wƺyſtkimi / coś ich nienáwidziáłá / y zgromádzę ich przećiwko tobie zewƺąd / y odkryję nágość twoję przed nimi / áby widzieli wƺyſtkę nágość twoję ;
Y oſądzę ćię ſądem cudzołożnic y krew rozlewájących / y podám ćię ná śmierć gniewu y zápálcżywośći.
Bo ćię podám w ręce ich / y zburzą dom twój nierządny / á rozrzucą wyżyny twoje / y zewloką ćię z ƺáty twojej / y pobiorą klejnoty ozdoby twojej / y zoſtáwią ćię nágą y odkrytą ;
Y przywiodą ná ćię gromádę / y ukámionują ćię kámieniem / y przebiją ćię miecżámi ſwemi ;
Y popálą domy twoje ogniem / á ucżynią nád tobą ſąd przed ocżymá wielu niewiáſt. A ták ucżynię wſtręt wƺetecżeńſtwu twemu / y nie będzieƺ więcej dáwáłá zápłáty.
A ták odpocżnie ſobie gniew mój ná tobie / y odſtąpi zápálcżywość mojá od ćiebie / y uſpokoję śię / á nie będę śię więcej gniewáł.
Dlátego / żeś nie pámiętáłá ná dni młodośći twojej / áleś mię tem wƺyſtkiem dráźniłá / przetoż oto y Já obróćiłem drogę twoję ná głowę twoję / mówi pánujący Pán / ták / że nie będzieƺ nierządu płodziłá / áni jákich obrzydliwośći ſwoich.
Oto ktokolwiek przez przypowieśći mówi / ná ćię przypowieśći obróći / mówiąc : Jáká mátká / táká córká jej.
Córką mátki twojej jeſteś / którá ſobie zbrzydziłá mężá ſwego y dziátki ſwoje ; á śioſtrą obu śióſtr ſwoich jeſteś / które ſobie zbrzydziły mężów ſwoich y dziátki ſwoje. Mátká wáƺá jeſt Hetejká / á ojćiec wáƺ Amorejcżyk.
Ale śioſtrá twojá ſtárƺá / którá śiedzi po lewicy twojej / jeſt Sámáryjá y córki jej ; á śioſtrá twojá młodƺá niż ty / którá śiedzi po práwicy twojej / jeſt Sodomá y córki jej.
Acżkolwiekeś po drogách ich nie chodziłá / ániś według obrzydliwośći ich cżyniłá / obrzydziwƺy to ſobie jáko rzecż máłą / przećięś śię bárdziej niż one popſowáłá ná wƺyſtkich drogách twoich.
Jáko żyję Já / mówi pánujący Pán / że Sodomá śioſtrá twojá y córki jej nie cżyniły / jákoś ty cżyniłá y córki twoje.
Oto tá byłá niepráwość Sodomy / śioſtry twojej / pychá / ſytość chlebá / y obfitość pokoju ; co oná májąc y córki jej / ręki jednák ubogiego y nędznego nie pośiláłá.
Owƺem / wynióſłƺy śię / cżyniły obrzydliwość przed oblicżem mojem ; przetożem je znióſł / jáko mi śię zdáło.
Sámáryjá tákże áni połowy grzechów twoich nie grzeƺyłá ; boś rozmnożyłá obrzydliwośći twoje nád nią ; á ták uſpráwiedliwiłáś śioſtry twoje wƺyſtkiemi obrzydliwośćiámi twojimi / któreś cżyniłá.
Ponośże y ty háńbę ſwoję / którąś przyſądziłá śioſtrom ſwoim / dlá grzechów ſwych / któremiś obrzydliwośći cżyniłá bárdziej niż one / ſpráwiedliwƺemić były niżeli ty ; y ty / mówię / záwſtydź śię / á noś ná ſobie háńbę ſwoję / gdyżeś uſpráwiedliwiłá śioſtry twoje.
Przywrócęli záż więźniów ich / to jeſt / więźniów Sodomy y córek jej / y więźniów Sámáryji y córek jej : tedyć też przywiodę pojmánych więźniów twoich w pośrodku ich ;
Abyś ták nośiłá háńbę twoję / á wſtydziłá śię zá wƺyſtko / coś cżyniłá / á ták ábyś je ućieƺyłá.
Jeźliżeć śioſtry twoje / Sodomá y córki jej / wrócą śię do pierwƺego ſtánu ſwego / tákże Sámáryjá y córki jej wrócą śię do pierwƺego ſtánu ſwego ; tedy śię też y ty z córkámi ſwemi náwróćiƺ do pierwƺego ſtánu ſwego.
Poniewáż Sodomá śioſtrá twojá nie byłá powieśćią w uſtách twoich w dzień pychy twojej.
Pierwej niż byłá objáwioná złość twojá / jáko zá cżáſu oblężeniá od córek Syryjſkich / y wƺyſtkich / którzy ſą około nich / córek Filiſtyńſkich / które ćię niƺcżyły ze wƺyſtkich ſtron.
Niecnotę twoję y obrzydliwość twoje ponośiƺ / mówi Pán.
Bo ták mówi pánujący Pán : Ták ucżynię z tobą / jákoś ucżyniłá / gdyś wzgárdziłá przyśięgą / y złámáłá przymierze.
Wƺákże wſpomnę ná przymierze moje z tobą / ucżynione zá dni młodośći twojej / y ſtwierdzę z tobą przymierze wiecżne.
Y wſpomniƺ ná drogi twoje / y záwſtydziƺ śię / gdy przyjmieƺ śioſtry twoje ſtárƺe nád ćię / y młodƺe niż ty / y dám ći je zá córki / ále nie według przymierzá twego.
A ták utwierdzę przymierze moje z tobą / á dowieƺ śię / żem Já Pán.
Abyś wſpomniáłá / y záwſtydziłá śię / y nie mogłá więcej otworzyć uſt dlá wſtydu ſwego / gdy ćię ocżyƺcżę od wƺyſtkiego / coś cżyniłá / mówi pánujący Pán.
