Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! oznájmij Jeruzálemowi obrzydliwośći jego / y rzecż :
Ták mówi pánujący Pán do córki Jeruzálemſkiej : Obcowánie twoje y ród twój jeſt z źiemi Chánánejſkiej ; oćiec twój jeſt Amorejcżyk / á mátká twojá Hetejká.
A národzenie twoje tákie : W dźień / któregoś śię urodźiłá / nie urzniono pępká twego / y wodą ćię nie obmyto dla ochędożeniá / áni ćię ſolą poſolono / áni w pieluchy uwiniono.
Nie zlitowáło śię nád tobą oko / ábyć ucżyniło jedno z tych / ulitowáwƺy śię ćiebie ; ále ćię porzucono ná polu / przeto / żeś byłá obmierzłá w dźień / któregoś śię urodźiłá.
A idąc mimo ćię / y widząc ćię ku podeptániu podáną we krwi twojey / rzekłem ći : Żyj we krwi twojey ; rzekłem ći / mówię : Żyj we krwi twojey.
Rozmnożyłem ćię ná tyśiące / jáko urodzáj polny / y rozmnożonáś / y ſtáłáś śię wielką / á przyƺłáś do bárzo wielkiej ozdoby ; pierśi twoje odęły śię / á włoſy twoje uroſły / choćiáżeś byłá nágá y odkrytá.
Prztoż idąc mimo ćię / á widząc ćię / że oto cżás twój / cżás miłośći / rozćiągnąłem podołek mój ná ćię / y nákryłem nágość twoję / y obowiązáłem ći śię przyśięgą / á wƺedłem w przymierze z tobą / mówi pánujący Pán / y ſtáłáś śię moją.
Y omyłem ćię wodą / á ſpłókáwƺy krew twoję z ćiebie / pomázáłem ćię olejkiem ;
Nádto przyodźiáłem ćię ƺátą háftowáną / y obułem ćię w koƺtowne trzewiki / y opáſáłem ćię biśiorem / á przyodźiáłem ćię ƺátą jedwábną ;
Y przybráłem ćię w ochędoſtwo / á dáłem mánele ná ręce twoje / y łáńcuch złoty ná ƺyję twoję ;
Dáłem też nácżelnik ná cżoło twoje / á náuƺnice ná uƺy twoje / y koronę ozdobną ná głowę twoję ;
A ták byłáś ozdobioná złotem y ſrebrem / á odźienie twoje było biśior / y ƺátá jedwábná / y háftowáná ; jádáłáś bułkę y miód / y oliwę / á byłáś náder piękną / y ƺcżęśliwieć śię powodźiło w króleſtwie /
Ták / że śię rozeƺłá powieść o tobie miedzy národámi dla pięknośći twojey ; boś doſkonáłą byłá dla ſłáwy mojey / którąm był włożył ná ćię / mówi pánujący Pán.
Aleś ufáłá w pięknośći twojey / y płodźiłáś nierząd / będąc ták ſłáwną ; boś płodźiłá nierząd z káżdym mimo ćię idącym / káżdy ſnádnie użył pięknośći twojey.
A nábrwáƺy ƺát ſwojich nácżyniłáś ſobie wyżyn rozmáitych fárb / á płodźiłáś nierząd przy nich / któremu podobny nigdy nie przydźie / áni będźie.
Nádto nábráwƺy klejnotów ozdoby ſwojey ze złotá mego y z ſrebrá mego / którem ći był dał / nácżyniłáś ſobie obrázów pogłowiá męſkiego / y płodźiłáś z nimi nierząd.
Wźięłáś też ƺáty ſwe háftowáne / á przyodźiáłáś je / olejek mój y kádźidło moje kłádłáś przed nie ;
Náwet y chleb mój / którem ći dał / bułkę y oliwę / y miód / którymem ćię kármił / kłádłáś przed nie ná wonność przyjemną / y było ták / mówi pánujący Pán.
Bráłáś też ſynów ſwojich / y córki ſwoje / którycheś mi národźiłá / á oneś im ofiárowáłá ku pożárćiu ; izali to jeƺcże máłe wƺetecżeńſtwá twoje?
Do tego y ſynóweś mojich zábijáłá / á dáwáłáś ich / áby ich przenoƺono przez ogień im kwoli.
Nádto przy wƺyſtkich obrzydliwośćiách ſwojich / y wƺetecżeńſtwách ſwojich nie pámiętáłáś ná dni młodośći twojey / gdyś byłá nágą y odkrytą / podáną ná podeptánie we krwi twojey.
Nádto mimo wƺyſtkich onych złośći twojich ( Biádá / biádá tobie! mówi pánujący Pán. )
Zbudowáłáś ſobie dom nierządny / y wyſtáwiłáś ſobie wyżynę w káżdej ulicy.
Po wƺyſtkich rozſtániách dróg pobudowáłáś wyżyny ſwoje / á ucżyniłáś obmierzłą piękność ſwoję / rozkłádájąc nogi ſwoje káżdemu mimo idącemu / y rozmnożyłáś wƺetecżeńſtwá ſwoje.
Boś nierząd płodźiłá z ſynámi Egipſkimi / ſąśiádámi twymi wielkich ćiał / á rozmnożyłáś wƺetecżeńſtwá twe / ábyś mię do gniewu pobudzáłá.
Przetoż otom wyćiągnął rękę moję ná ćię / á umniejƺyłem obroku twego / y podáłem ćię ná wolę nienáwidzących ćię córek Filiſtyńſkich / które śię wſtydzą zá złe drogi twoje.
Nádto płodźiłáś nierząd z ſynámi Aſſyryjſkimi / przeto / żeś śię nie mogłá náſyćić / á nierząd płodząc z nimi / y tákeś śię nie náſyćiłá.
A ták rozmnożyłáś wƺetecżeńſtwo ſwe w źiemi Chánánejſkiej y Cháldejſkiej / á y ták nie náſyćiłáś śię.
O jáko jeſt zmámione ſerce twoje! mówi pánujący Pán / ponieważ śię dopuƺcżáƺ tych poſtępków niewiáſty nierządney y wƺetecżney.
Budując ſobie nierządne domy ná rozſtániu káżdego gośćińcá / á wyżynę ſobie ſtáwiájąc w káżdej ulicy / owƺem pogárdzájąc zápłátą nie jeſteś áni jáko wƺetecżnicá /
Ani jáko niewiáſtá cudzołożącá / która mimo mężá ſwego obcych przypuƺcżá.
Wƺyſtkim wƺetecżnicom dáwáją zápłátę ; áleś ty dáwáłá zápłátę ſwoję wƺyſtkim zálotnikom twoim / y dáwáłáś im upominki / áby chodźili do ćiebie zewƺąd ná wƺetecżeńſtwá z tobą.
A ták znájduje śię przy tobie przećiwná rzecż od innych niewiáſt we wƺetecżeńſtwách twojich / ponieważ ćię dla wƺetecżeńſtwá nie ƺukáją ; ále ty dájeƺ zápłátę / á nie tobie zápłátę dáwáją / co śię opák dźieje.
Przetoż / o wƺetecżnico! ſłucháj ſłowá Páńſkiego.
Ták mówi pánujący Pán : Dla tego / iż śię wyláłá ſprośność twojá / y odkryłá śię nágość twojá we wƺetecżeńſtwách twojich z zálotnikámi twoimi y ze wƺyſtkimi plugáwemi báłwánámi obrzydliwośći twojich / y dla rozlánia krwi ſynów twojich / którycheś im dáłá :
Przetoż oto Ja zgromádzę wƺyſtkich zálotników twojich / z którymiś obcowáłá / y wƺyſtkich / którycheś miłowáłá / ze wƺyſtkimi / coś ich nienáwidźiáłá / y zgromádzę ich przećiwko tobie zewƺąd / y odkryję nágość twoję przed nimi / áby widźieli wƺyſtkę nágość twoję ;
Y oſądzę ćię ſądem cudzołożnic y krew rozlewájących / y podám ćię ná śmierć gniewu y zápálcżywośći.
Bo ćię podám w ręce ich / y zburzą dom twój nierządny / á rozrzucą wyżyny twoje / y zewloką ćię z ƺáty twojey / y pobiorą klejnoty ozdoby twojey / y zoſtáwią ćię nágą y odkrytą ;
Y przywiodą ná ćię gromádę / y ukámionują ćię kámieniem / y przebiją ćię miecżámi ſwemi ;
Y popálą domy twoje ogniem / á ucżynią nád tobą ſąd przed ocżymá wielu niewiáſt. A ták ucżynię wſtręt wƺetecżeńſtwu twemu / y nie będźieƺ więcey dáwáłá zápłáty.
A ták odpocżnie ſobie gniew mój ná tobie / y odſtąpi zápálcżywość mojá od ćiebie / y uſpokoję śię / á nie będę śię więcey gniewał.
Dla tego / żeś nie pámiętáłá ná dni młodośći twojey / áleś mię tym wƺyſtkim dráźniłá / przetoż oto y Ja obróćiłem drogę twoję ná głowę twoję / mówi pánujący Pán / ták / że nie będźieƺ nierządu płodźiłá / áni jákich obrzydliwośći ſwojich.
Oto ktokolwiek przez przypowieśći mówi / ná ćię przypowieśći obróći / mówiąc : Jáká mátká / táká córká jey.
Córką mátki twojey jeſteś / która ſobie zbrzydźiłá mężá ſwego y dźiátki ſwoje ; á śioſtrą obu śióſtr ſwojich jeſteś / które ſobie zbrzydźiły mężów ſwojich y dźiátki ſwoje. Mátká wáƺá jeſt Hetejká / á oćiec wáƺ Amorejcżyk.
Ale śioſtrá twojá ſtárƺá / która śiedźi po lewicy twojey / jeſt Sámáryjá y córki jey ; á śioſtrá twojá młodƺá niż ty / która śiedźi po práwicy twojey / jeſt Sodomá y córki jey.
Acżkolwiekeś po drogách ich nie chodźiłá / ániś według obrzydliwośći ich cżyniłá / obrzydźiwƺy to ſobie jáko rzecż máłą / przećięś śię bárdźiej niż one popſowáłá ná wƺyſtkich drogách twojich.
Jáko żyję Ja / mówi pánujący Pán / że Sodomá śioſtrá twojá y córki jey nie cżyniły / jákoś ty cżyniłá y córki twoje.
Oto tá byłá niepráwość Sodomy / śioſtry twojey / pychá / ſytość chlebá / y obfitość pokoju ; co oná májąc y córki jey / ręki jednák ubogiego y nędznego nie pośiláłá.
Owƺem / wynióſƺy śię / cżyniły obrzydliwość przed oblicżem mojim ; przetożem je znióſł / jáko mi śię zdáło.
Sámáryjá tákże áni połowy grzechów twojich nie grzeƺyłá ; boś rozmnożyłá obrzydliwośći twoje nád nią ; á ták uſpráwiedliwiłáś śioſtry twoje wƺyſtkiemi obrzydliwośćiámi twojimi / któreś cżyniłá.
Ponośże y ty háńbę ſwoję / którąś przyſądźiłá śioſtrom ſwoim / dla grzechów ſwych / któremiś obrzydliwośći cżyniłá bárdźiej niż one / ſpráwiedliwƺemić były niżeli ty ; y ty / mówię / záwſtydź śię / á noś ná ſobie háńbę ſwoję / gdyżeś uſpráwiedliwiłá śioſtry twoje.
Przywrócęli záż więźniów ich / to jeſt / więźniów Sodomy y córek jey / y więźniów Sámáryji y córek jey : tedyć też przywiodę pojmánych więźniów twojich w pośrzodku ich ;
Abyś ták nośiłá háńbę twoję / á wſtydźiłá śię zá wƺyſtko / coś cżyniłá / á ták ábyś je ućieƺyłá.
Jeſliżeć śioſtry twoje / Sodomá y córki jey / wrócą śię do pierwƺego ſtánu ſwego / tákże Sámáryjá y córki jey wrócą śię do pierwƺego ſtánu ſwego ; tedy śię też y ty z córkámi ſwemi náwróćiƺ do pierwƺego ſtánu ſwego.
Ponieważ Sodomá śioſtrá twojá nie byłá powieśćią w uſtách twojich w dźień pychy twojey.
Pierwej niż byłá objáwioná złość twojá / jáko zá cżáſu oblężeniá od córek Syryjſkich / y wƺyſtkich / którzy ſą około nich / córek Filiſtyńſkich / które ćię niƺcżyły ze wƺyſtkich ſtron.
Niecnotę twoję y obrzydliwość twoje ponośiƺ / mówi Pán.
Bo ták mówi pánujący Pán : Ták ucżynię z tobą / jákoś ucżyniłá / gdyś wzgárdźiłá przyśięgą / y złámáłá przymierze.
Wƺákże wſpomnę ná przymierze moje z tobą / ucżynione zá dni młodośći twojey / y ſtwierdzę z tobą przymierze wiecżne.
Y wſpomniƺ ná drogi twoje / y záwſtydźiƺ śię / gdy przyjmieƺ śioſtry twoje ſtárƺe nád ćię / y młodƺe niż ty / y dam ći je zá córki / ále nie według przymierza twego.
A ták utwierdzę przymierze moje z tobą / á dowieƺ śię / żem Ja Pán.
Abyś wſpomniáłá / y záwſtydźiłá śię / y nie mogłá więcey otworzyć uſt dla wſtydu ſwego / gdy ćię ocżyƺcżę od wƺyſtkiego / coś cżyniłá / mówi pánujący Pán.
