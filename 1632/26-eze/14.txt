Potem przyƺedłƺy do mnie mężowie z ſtárƺych Izráelſkich / uśiedli przedemną.
Y ſtáło śie ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! ći mężowie złożyli plugáwe báłwány ſwoje do ſercá ſwego / á niepráwość / którá im jeſt ku obráżeniu / położyli przed twárzą ſwoją ; mniemáƺże / iż mię uprzejmie pytáją o rádę?
Dlátegoż powiedz im / y mów do nich : Ták mówi pánujący Pán : Ktokolwiek z domu Izráelſkiego położył plugáwe báłwány ſwoje w ſercu ſwojim / á niepráwość / którá mu jeſt ku obráżeniu / położył przed twárzą ſwoją / á przyƺedłby do proroká / Já Pán odpowiem temu / który przyjdzie / o mnóſtwie plugáwych báłwánów jego.
Abym ułápáł dom Izráelſki w ſercu ich / że śię odwróćili odemnie od plugáwych báłwánów ſwoich wƺyſcy zgołá.
Przetoż rzecż do domu Izráelſkiego : Ták mówi pánujący Pán : Náwróććie śię / á cofnijćie śię od plugáwych báłwánów wáƺych / y od wƺyſtkich obrzydliwośći wáƺych odwróććie twárz ſwoję.
Bo ktobykolwiek z domu Izráelſkiego y z przychodniów / którzy mieƺkáją w Izráelu / odwróćił śię od náśládowániá mnie / á położyłby plugáwe báłwány ſwoje w ſercu ſwem / y niepráwość / którá mu jeſt ku obráżeniu / położyłby przed twárz ſwoją / y przyƺedłby do proroká / áby mię przezeń pytáł / Já Pán odpowiem mu ſám przez śię /
Y poſtáwię oblicże moje przećiw temu mężowi / y dám go ná znák / y ná przyſłowie / á wytrácę go z pośrodku ludu mojego ; y dowiećie śię / żem Já Pán.
A gdyby śię prorok dáł zwieść / áby mówił ſłowo / Já Pán zwiodłem proroká onego / y wyćiągnę náń rękę ſwoję / y wygłádzę go z pośrodku ludu mego Izráelſkiego.
Y ták ponioſą niepráwość ſwoję ; jáká jeſt káźń ná tego / któryby śię pytáł / táká też káźń ná proroká będzie /
Aby więcej nie błądził dom Izráelſki odemnie / á nie mázáli śię więcej żádnemi przeſtępſtwámi ſwojimi / áby byli ludem moim / á Já ábym był Bogiem ich / mówi pánujący Pán.
Znowu ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! gdyby ziemiá zgrzeƺyłá przećiwko mnie / dopuƺcżájąc śię przeſtępſtwá / tedy jeźlibym wyćiągnął rękę moję ná nią / á złámáłbym jej łáſkę chlebá / y poſłáłbym ná nią głód / y wytráćiłbym z niej ludzi y zwierzętá ;
Choćby byli w pośrodku niej ći trzej mężowie / Noe / Dánijel y Ijob / oni w ſpráwiedliwośći ſwojej wybáwiliby duƺe ſwe / mówi pánujący Pán.
Tákże jeźlibym zły zwierz przepuśćił ná ziemię / á ośieroćiłby ją / y byłáby ſpuſtoƺoná / żeby jej nikt przechodzić nie mógł dlá zwierzá /
Jáko żyję Já / mówi pánujący Pán / że choćby ći trzej mężowie byli w pośrodku jej / żádną miárąby nie wybáwili ſynów áni córek : oniby tylko ſámi wybáwieni byli / lecżby ziemiá ſpuſtoƺoná byłá.
Albo jeźlibym przywiódł miecż ná tę ziemię / á rzekłbym miecżowi : Przejdź przez tę ziemię / ábym wytráćił z niej ludzi y zwierzętá /
Jáko żyję Já / mówi pánujący Pán / że choćby ći trzej mężowie byli w pośrodku jej / żádną miárąby nie wybáwili ſynów áni córek / áleby oni tylko ſámi byli wybáwieni.
Albo poſłáłlibym mór ná tę ziemię / y wyláłbym popędliwość ſwoję ná nią ku wytráceniu / áby z niej ludzie y zwierzętá byli wytráceni.
Że choćby Noe / Dánijel y Ijob byli w pośrodku jej / jáko żyję Já / mówi pánujący Pán / żádną miárąby áni ſyná áni córki nie wybáwili / oniby tylko w ſpráwiedliwośći ſwojej wybáwili duƺe ſwe.
Owƺem / ták mówi pánujący Pán : Choćbym cżtery káźni moje ćiężkie / miecż / y głód / y zły zwierz / y mór poſłáł ná Jeruzálem / ábym wytráćił z niego ludzi y zwierzętá /
A oto jeźliby zoſtáli w nim / którzyby tego uƺli / á wywiedzeni byli ſynowie álbo córki / oto y oni muƺą iść do wáſ ; y oglądáćie drogę ich / y ſpráwy ich / á ućieƺyćie śię nád tem złem / które przywiodę ná Jeruzálem / nád wƺyſtkiem / co przywiodę ná nie.
A ták ućieƺą wáſ / gdy ujrzyćie drogę ich y ſpráwy ich ; y zrozumiećie / żem to wƺyſtko nie dármo ucżynił / com ucżynił w nim / mówi pánujący Pán.
