Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Cóż wam po tym / iż używáćie tey przypowieśći o źiemi Izráelſkiej mówiąc : Ojcowie jedli jágodę ćierpką / á ſynów zęby drętwieją.
Jáko żyję Ja / mówi pánujący Pán / że wy nie będźiećie więcey mogli używáć tey przypowieśći w Izráelu.
Oto duƺe wƺyſtkie moje ſą / jáko duƺá ojcowſká ták y duƺá ſynowſká moje ſą ; duƺá / która grzeƺy / tá umrze.
Bo byłliby mąż ſpráwiedliwy / á cżyniłby ſąd y ſpráwiedliwość /
Któryby ná górách nie jádał / á ocżówby ſwych nie podnośił do plugáwych báłwánów domu Izráelſkiego / á żonyby bliźniego ſwego nie zmázał / y do niewiáſty dla niecżyſtoty oddáloney nie przyſtąpił ;
Któryby nikogo nie ućiſkał / záſtáwby dłużnikowi ſwemu wrácał / cudzegoby gwáłtem nie brał / chlebáby ſwego łáknącemu udźielał / á nágiegoby ƺátą przyodźiewał ;
Któryby ná lichwę nie dáwał / y płátuby nie brał / od niepráwośćiby odwrácał rękę ſwoję / á ſądby ſpráwiedliwy cżynił miedzy mężem á mężem ;
Któryby w uſtáwách mojich chodźił / á ſądówby mojich przeſtrzegał / cżyniąc / co jeſt práwego : ten ſpráwiedliwy pewnie żyć będźie / mówi pánujący Pán.
A gdyby ſpłodźił ſyná łotrá / krew wylewájącego / któryby cżemkolwiek z tych rzecży brátu ƺkodźił /
A tegoby wƺyſtkiego nie cżynił / owƺemby y ná górách jádał / y żonęby bliźniego ſwego zmázał /
Ubogiegoby y nędznego ućiſkał / co cudzego jeſt / gwáłtemby zábierał / záſtáwuby nie wrácał / á do plugáwych báłwánów podnośiłby ocży ſwoje / y obrzydliwośćiby cżynił /
Ná lichwęby dáwał / y płát brał / izali żyć będźie? Nie będźie żył ; poniewáż te wƺyſtkie obrzydliwośći cżynił : śmierćią umrze / krew jego ná nim będźie.
A oto jeżeliby ſpłodźił ſyná / któryby widźiał wƺyſtkie grzechy ojcá ſwego / które cżynił / á widząc nie cżyniłby ták ;
Ná góráchby nie jádał / á ocżówby ſwych nie podnośił do plugáwych báłwánów domu Izráelſkiego / żonyby bliźniego ſwego nie zmázał /
Y nikogoby nie ućiſkał / záſtáwuby nie zátrzymywał / á cudzegoby gwáłtem nie brał / chlebáby ſwego łáknącemu udźielał / á nágiegoby ƺátą przyodźiał /
Od niepráwegoby odwróćił rękę ſwoję / lichwyby y płátu nie brał / ſądyby moje cżynił / w uſtáwáchby mojich chodźił : ten nie umrze dla niepráwośći ojcá ſwego / ále pewnie żyć będźie.
Lecż oćiec jego / przeto / że cżynił krzywdę / co jeſt cudzego / brátu gwáłtem brał / á to / co jeſt dobrego / nie cżynił w pośrzodku ludu ſwego : przetoż oto umrze dla niepráwośći ſwojey.
Ale mówićie : Cżemuż? Izali nie ponieśie ſyn niepráwośći ojcowſkiej? Gdy ſyn ſąd y ſpráwiedliwość cżyni / wƺyſtkich uſtáw mojich ſtrzeże y cżyni je / pewnie żyć będźie.
Duƺá / która grzeƺy / tá umrze ; ále ſyn nie ponieśie niepráwośći ojcowſkiej / áni oćiec ponieśie niepráwośći ſynowſkiej ; ſpráwiedliwość ſpráwiedliwego przy nim zoſtánie / á niepobożność niepobożnego nań przypádnie.
A jeſliby śię niepobożny odwróćił od wƺyſtkich grzechów ſwojich / które cżynił / á ſtrzegłby wƺyſtkich uſtáw mojich / y cżyniłby ſąd y ſpráwiedliwość / pewnie żyć będźie / nie umrze ;
Żádne przeſtępſtwá jego / których śię dopuśćił / nie będą mu przypomináne ; w ſpráwiedliwośći ſwej / którąby cżynił / żyć będźie.
Azaż Ja śię kochám w śmierći niepobożnego? mówi pánujący Pán. Izali nie rácżey / gdy śię odwróći od dróg ſwojich / áby żył?
Ale jeſliby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojey / á cżyniłby niepráwość / cżyniąc według wƺyſtkich obrzydliwośći / które cżyni niezbożny / izali táki żyć będźie? Wƺyſtkie ſpráwiedliwośći jego / które cżynił / nie będą wſpomináne ; dla przeſtępſtwá ſwego / które popełniał / y dla grzechu ſwego / którego śię dopuśćił / dla tych rzecży umrze.
A iż mówićie : Nie proſtá jeſt drogá Páńſká ; ſłuchájćież teraz / o domie Izráelſki! izali drogá mojá nie jeſt proſtá? Izali drogi wáƺe nie ſą krzywe?
Gdyby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojey / á cżyniąc niepráwość w temby umárł / dla niepráwośći ſwojey / którą cżynił / umrze.
Ale gdyby śię odwróćił niezbożny od niezbożnośći ſwojey / którey śię dopuśćił / á cżyniłby ſąd y ſpráwiedliwość / ten duƺę ſwoję záchowá.
Bo obácżywƺy śię odwróćił śię od wƺyſtkich wyſtępków ſwojich / których śię dopuƺcżał / pewnie żyć będźie / nie umrze.
A przećie mówi dom Izráelſki : Nie proſtá jeſt drogá Páńſká ; izali drogi moje nie ſą proſte / o domie Izráelſki? Izali nie rácżey drogi wáƺe ſą krzywe?
A przetoż káżdego z was według dróg jego ſądźić będę / o domie Izráelſki! mówi pánujący Pán. Náwróććież śię / á odwróććie śię od wƺyſtkich wyſtępków wáƺych / áby wam niepráwość nie byłá ná obráżenie.
Odrzuććie od śiebie wƺyſtkie przeſtępſtwá wáƺe / którycheśćie śię dopuƺcżáli / á ucżyńćie ſobie ſerce nowe y duchá nowego. Y przecżże máćie umrzeć / o domie Izráelſki?
Abowiem śię Ja nie kochám w śmierći umierájącego / mówi pánujący Pán ; náwróććież śię tedy / á żyć będźiećie.
