Potem mię przywiódł ku drzwiom domu / á oto wody wychodźiły z pod progu domu ná wſchód ſłońcá : bo przedniá ſtroná domu byłá ná wſchód ſłońcá / á wody one ſchodźiły ſpodkiem po práwej ſtronie domu po ſtronie południowej ołtárzá.
Z támtąd mię wywiódł drogą bramy północney / y obwiódł mię drogą zewnętrzną do bramy zewnętrzney / drogą / która pátrzy ná wſchód ſłońcá ; á oto wody wynikáły po práwej ſtronie.
A gdy wychodźił on mąż ná wſchód ſłońcá / w którego ręku byłá miárá / y wymierzył tyśiąc łokći / y przewiódł mię przez wodę / przez wodę áż do koſtek :
Potem wymierzył drugi tyśiąc / á przewiódł mię przez wodę / przez wodę áż do kolán ; y záś wymierzył trzeći tyśiąc / á przewiódł mię przez wodę áż do biódr.
A gdy záś wymierzył tyśiąc / był potok / któregom nie mógł przebrnąć ; bo były wezbráły wody / wody / które trzebá było przepłynąć / potok / któregom nie mógł przebrnąć.
Tedy rzekł do mnie : Widźiáłżeś / ſynu cżłowiecży? Y wywiódł mię / y obróćił mię ná brzeg onego potoku.
A gdym śię obróćił / oto ná brzegu onego potoku było drzewo bárzo wielkie po obu ſtronách ;
Y rzekł do mnie : Te wody wychodzą od Gálilei pierwƺej / á ſchodzą po równinie / y wchodzą w morze ; á gdy do morzá wpádną / uzdrowione bywáją wody.
Y ſtánie śię / że káżda duƺá żyjącá / która śię płázá / gdźiekolwiek przyjdą potoki / żyć będźie / y będźie ryb bárzo wiele / przeto / że gdy przyjdą tám one wody / ocżerſtwieją / y żyć będą wƺędy / kędykolwiek przydźie ten potok.
Stánie śię y to / że ſtáną podle niego rybitwi od Engáddy áż do zdroju Egláim / tám będą rozćiągáć śieći : ryb będźie bárzo wiele rozmáitego rodzáju / jáko ryb morzá wielkiego.
Błotá jego y káłuże jego nie będą uzdrowione / ále ſoli oddáne będą.
A nád potokiem wyrośnie ná brzegu jego po obu ſtronách wƺelkie drzewo owoce przynoƺące / którego liść nie opádá / áni owoc jego uſtáje / w mieśiącách ſwojich przynośi pierwoćiny ; bo wody jego z świątnicy wychodzą / przetoż owoc jego jeſt ná pokárm / á liśćie jego ná lekárſtwo.
Ták mówi pánujący Pán : Táć jeſt gránicá / w którey ſobie dźiedźicżnie przywłáƺcżyćie źiemię według dwánaśćie pokoleń Izráelſkich ; Józefowi śię doſtáną dwá ƺnury.
Dźiedźicżnie / mówię / pośiądźiećie ją / równie jeden jáko drugi / o którą podnioſłem rękę moję / że ją dam ojcom wáƺym ; y przypádnie wam tá źiemiá w dźiedźictwo.
Táć jeſt tedy gránicá tey źiemi ku ſtronie północney / od morzá wielkiego / drogą do Hetlonu / kędy wchodzą do Sedád.
Emát / Berotá / Sybráim / które ſą miedzy gránicą Dámáƺku y miedzy gránicą Emát / wśi pośrednie / które ſą przy gránicy Háwrán.
A ták będźie gránicá od morzá Chátzár Enon / będźie gránicá Dámáƺek / á północná ſtroná ná północy / y gránicá Emát ; á toć jeſt ſtroná północná.
A ſtroná wſchodniá miedzy Háwrán y miedzy Dámáƺkiem y miedzy Gáláád y miedzy źiemią Izráelſką przy Jordánie ; od tey gránicy przy morzu wſchodniem mierzyć będźiećie ; á toć jeſt ſtroná wſchodniá.
A ſtroná południowá ná południe od Támár áż do wód poſwárków w Kádes / od potoku áż do morzá wielkiego ; á toć jeſt ſtroná południowá ná południe.
Stroná záś záchodniá morze wielkie od gránicy áż przećiwko kędy śię wchodźi do Emát / táć jeſt ſtroná záchodniá.
A ták rozmierzyćie ſobie tę źiemię / według pokoleń Izráelſkich.
A gdy ją rozmierzyćie / będźie wam w dźiedźictwo / y przychodniom / którzyby mieƺkáli miedzy wámi / którzyby ſpłodźili ſynów miedzy wámi / bo wam będą jáko tu zrodzeni miedzy ſynámi Izráelſkimi ; z wámi dźiedźicżyć będą miedzy pokoleniámi Izráelſkiemi.
A w któremkolwiek pokoleniu przychodźień przychodniem będźie / tám mu dáćie dźiedźictwo jego / mówi pánujący Pán.
