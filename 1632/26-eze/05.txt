Potem ty / ſynu cżłowiecży! weźmij ſobie nóż oſtry / to jeſt brzytwę bárwierſką / weźmij ją ſobie / á ogól nią głowę ſwoję y brodę ſwoję ; potem weźmij ſobie wágę / y rozdziel je.
Jednę trzećią cżęść ogniem ſpál w pośród miáſtá / gdy śię wypełnią dni oblężeniá ; potem wziąwƺy drugą trzećią cżęść / pośiekáj miecżem około niego / á oſtátnią trzećią cżęść roztrząśnij ná wiátr ; bo miecżá dobędę zá nimi.
A wƺákże weźmij z nich jáką trochę / y záwiąż w kráje ƺáty twojej.
A y z tych jeƺcże wziąwƺy wrzuć je w pośród ogniá / y ſpál je w ogniu / ſkąd wynijdzie ogień ná wƺyſtek dom Izráelſki.
Ták mówi pánujący Pán : Toć jeſt Jeruzálem / którem poſtáwił w pośród pogán / á zewƺąd otocżył kráinámi ;
Ale odmieniło ſądy moje w niezbożność / bárdziej niż pogánie / á uſtáwy moje bárdziej niż inne kráiny / które ſą około niego ; bo ſądámi mojemi pogárdzili / y w uſtáwách moich nie chodzili.
Przetoż ták mówi pánujący Pán : Dlátego / żeśćie pogáńſtwo grzechámi przewyżƺyli / które jeſt około wáſ / á w uſtáwách moich nie chodziliśćie / y ſądów moich nie záchowáliśćie / náwet áni ták jáko pogánie / którzy ſą około wáſ / ſąduśćie nie cżynili :
Przetoż ták mówi pánujący Pán : Otom Já przećiwko tobie / otom Já / y wykonám w pośrodku ćiebie ſądy przed ocżymá pogán ;
Bo ucżynię przy tobie to / cżegom pierwej nie ucżynił / y cżego nápotem nie ucżynię dlá wƺyſtkich obrzydliwośći twoich /
Ták / że ojcowie jeść będą ſynów w pośrodku ćiebie / á ſynowie jeść będą ojców ſwoich ; y wykonám ſądy przećiwko tobie / á rozproƺę wƺyſtkie oſtátki twoje ná wƺyſtkie ſtrony.
Przetoż jáko żyję Já / mówi pánujący Pán : Poniewáżeś ty świątnicę moję ſplugáwiło wƺelákiemi niecżyſtośćiámi twojimi / y wƺelákiemi obrzydliwośćiámi twemi / y Já ćię też podám w lekkość / á nie ſfolgujeć oko moje / áni śię zlituję.
Trzećiá cżęść z ćiebie morem pomrze y głodem wyginie w pośrodku ćiebie / á drugá trzećiá cżęść od miecżá pádnie około ćiebie / á trzećią oſtátnią cżęść ná wƺyſtkie ſtrony rozproƺę / y miecżá dobędę zá nimi.
A ták śię do końcá wyleje zápálcżywość mojá / y nátrę popędliwośćią ſwoją ná nich / y ućieƺę śię / y doznáją / żem Já Pán / którym to wyrzekł w zápálcżywośći mojej / gdy wypełnię gniew ſwój nád nimi.
A podám ćię w ſpuſtoƺenie y w poháńbienie národom / które ſą około ćiebie / przed ocżymá káżdego mijájącego.
A ták będzieƺ ná háńbę / ná pośmiech / ná ſrogi przykłád y ná zdumienie národom / które ſą około ćiebie / gdy wykonám przećiwko tobie ſądy w popędliwośći y w gniewie y w ſrogiem zágniewániu. Já Pán mówiłem.
Gdy wypuƺcżę ſrogie ſtrzáły głodu ná zgubę wáƺę / które wypuƺcżę / ábym wáſ wygubił / á głód zgromádzę przećiwko wám / y złámię wám láſkę chlebá.
Poślę záiſte ná wáſ głód / y zwierzętá okrutne / które ćię ośierocą ; y mór y krew przyjdzie ná ćię / gdy ná ćię miecż przywiodę. Já Pán mówiłem.
