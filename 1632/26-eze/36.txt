A ty ſynu cżłowiecży! prorokuj o górách Izráelſkich / á mów : Góry Izráelſkie! ſłuchájćie ſłowá Páńſkiego ;
Ták mówi pánujący Pán : Przeto / iż nieprzyjáćiel rzekł o wáſ : Hej / hej! y wyſokośći wiecżne doſtáły śię nám w dźiedźictwo ;
Przetoż prorokuj á mów : Ták mówi pánujący Pán : Dlátego / dlátego / mówię / iż wáſ zburzyli á pochłonęli zewƺąd / y ſtáliśćie śię dźiedźictwem pozoſtáłym národom / y przyƺliśćie ná język y ná obmowiſko ludzkie ;
Przetoż / góry Izráelſkie! ſłuchájćie ſłowá pánującego Páná. Ták mówi pánujący Pán górom y págórkom / ſtrumieniom y dolinom / puſtyniom / obálinom / y miáſtom opuƺcżonym / które ſą ná ſplundrowánie / y ná pośmiewiſko oſtátkowi národów okolicżnych.
Dlátego / ták mówi pánujący Pán / zápráwdę w ogniu zápálcżywośći mojej mówić będę przećiw oſtátkom tych národów / y przećiwko wƺyſtkiej źiemi Edomſkiej / którzy ſobie przywłáƺcżyli źiemię moję zá dźiedźictwo z weſelem cáłego ſercá / y z ochotnem puſtoƺeniem / áby śiedliſko wygnánych jego było ná rozƺárpánie.
Przetoż prorokuj o źiemi Izrelſkiej / á mów górom / y págórkom / ſtrumieniom y dolinom : Ták mówi pánujący Pán : Oto Já w zápálcżywośći mojej / y w popędliwośći mojej mówię / dlátego / iż háńbę od národów ponośićie.
Przetoż ták mówi pánujący Pán : Jám podnióſł rękę moję / iż te národy / które ſą zewƺąd około wáſ / ſámi háńbę ſwoję ponioſą.
A wy / góry Izráelſkie! wypuśćie gáłązki ſwe / y owoc ſwój przynieśćie ludowi memu Izráelſkiemu / gdy śię przybliżą á przyjdą.
Bo oto Já idę do wáſ / y náwrócę śię do wáſ / á upráwiáne y pośiewáne będźiećie ;
Y rozmnożę ná wáſ ludźi / wƺyſtek zgołá dom Izárelſki ; y będą mieƺkáć w miáſtách / á miejſcá zburzone pobudowáne będą.
Rozmnożę / mówię / ná wáſ ludźi y dobytek / á rozmnożą śię y uroſną ; y ſpráwię / że mieƺkáć będźiećie jáko zá dáwnych cżáſów wáƺych / y lepiej wám cżynić będę niż przedtem / y dowiećie śię / żem Já Pán.
Bo przyprowádzę ná wáſ ludźi / lud mój Izráelſki / y pośiądą ćię / y będźieƺ im dźiedźictwem / á więcej ich nie ośieroćiƺ.
Ták mówi pánujący Pán : Dlátego / że o wáſ powiádáją : Tyś jeſt tá źiemiá / którá pożeráƺ ludźi / y ośierácáƺ národy twoje ;
Przetoż nie będźieƺ więcej ludźi pożeráłá / áni národów twoich więcej ośierácáłá / mówi pánujący Pán.
Y nie dopuƺcżę w tobie więcej ſłyƺeć háńby národów / áni zelżywośći ludzkiej nie ponieśieƺ więcej / y národów twoich nie przywiedźieƺ więcej do upádku / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! dom Izráelſki mieƺkájąc w źiemi ſwej ſpługáwiáł ją drogámi ſwemi y ſpráwámi ſwemi / ták / że drogá ich przed oblicżem mojem byłá jáko niecżyſtość niewiáſty odłącżonej.
Przetoż wyláłem gniew mój ná nich dlá krwi / którą wyláli ná źiemię / y dlá plugáwych báłwánów ich / któremi ją ſplugáwili.
Y rozproƺyłem ich między národy / á rozwiáni ſą po źiemiách ; według dróg ich y według ſpráw ich ſądźiłem ich.
A gdy weƺli do národów / do których przyƺli / pomázáli tám imię świątobliwośći mojej / gdy o nich mówiono : Lud to Páńſki / á z źiemi jego wyƺli.
Alem im ſfolgowáł dlá imieniá świętobliwośći mojej / które ſplugáwił dom Izráelſki między národámi / do których przyƺli.
Przetoż mów do domu Izráelſkiego : Ták mówi pánujący Pán : Nie dlá wáſ Já to cżynię / o domie Izráelſki! ále dlá imieniá świętobliwośći mojej / któreśćie ſplugáwili między národámi / do którycheśćie przyƺli ;
Abym poświęćił wielkie imię moje / które było ſplugáwione między národámi / któreśćie wy zmázáli w pośrodku ich ; y dowiedzą śię národy / żem Já Pán / mówi pánujący Pán / gdy poświęcony będę w wáſ przed ocżymá ich ;
Bo wáſ zbiorę z národów / y zgromádzę wáſ ze wƺyſtkich źiem / y przywiodę wáſ do źiemi wáƺej ;
Y pokropię wáſ wodą cżyſtą / á ocżyƺcżeni będźiećie od wƺyſtkich niecżyſtót wáƺych / y od wƺyſtkich plugáwych báłwánów wáƺych ocżyƺcżę wáſ ;
Y dám wám ſerce nowe / á duchá nowego dám do wnętrznośći wáƺych / y odejmę ſerce kámienne z ćiáłá wáƺego / á dám wám ſerce mięśiſte.
Duchá mego / mówię / dám do wnętrznośći wáƺej / á ucżynię / że w uſtáwách moich chodźić / y ſądów moich przeſtrzegáć / y cżynić je będźiećie.
A będźiećie mieƺkáć w źiemi / którąm dáł ojcom wáƺym / y będźiećie ludem moim / á Já będę Bogiem wáƺym.
Bo wáſ wyzwolę od wƺyſtkich niecżyſtót wáƺych / y przywołám zbożá / y rozmnożę je / á nie dopuƺcżę ná wáſ głodu.
Rozmnożę też owoc drzew / y urodzáje polne / żebyśćie więcej nie nośili háńby głodu między národámi.
Y wſpomnićie ná złe drogi wáƺe / y ná ſpráwy wáƺe / które nie były dobre / y omierzniećie ſámi ſobie w ocżách ſwoich dlá niepráwośći wáƺych y dlá obrzydliwośći wáƺych.
Nie dlá wáſći Já to cżynię / mówi pánujący Pán / niech wám to jáwno będźie ; ſromájćie śię / á wſtydźćie śię zá drogi wáƺe / domie Izráelſki!
Ták mówi pánujący Pán : Którego wáſ dniá ocżyƺcżę od wƺyſtkich niepráwośći wáƺych / oſádzę miáſtá / á miejſcá zburzone będą pobudowáne.
A ták źiemiá ſpuſtoƺáłá ſpráwowáná będźie / którá przedtem byłá ſpuſtoƺoná przed ocżymá wƺyſtkich przechodzących.
Y rzeką : Ziemiá tá ſpuſtoƺáłá ſtáłá śię jáko ogród Eden ; tákże miáſtá puſte y opuƺcżone y rozwálone / obronne ſą y oſádzone.
Y dowiedzą śię národy / którekolwiek zoſtáną około wáſ / żem Já Pán pobudowáł rozwáliny / á náſádźił miejſcá ſpuſtoƺone. Já Pán mówiłem / y ucżynię.
Ták mówi pánujący Pán : Jeƺcże tego będźie u mnie ƺukáł dom Izráelſki / ábym to im ucżynił / ábym ich rozmnożył w ludźi jáko trzodę.
Jáko trzodę ná ofiáry / jáko trzodę Jeruzálemſką ná świętá jego urocżyſte / ták ſpuſtoƺone miáſtá będą nápełnione trzodámi ludźi ; y dowiedzą śię / żem Já Pán.
