Y ſtáło śię ſłowo Páńſkie do mnie roku dźiewiątego / mieśiąc dźieśiątego / dźieśiątego dniá tegoż mieśiącá / mówiąc :
Synu cżłowiecży! nápiƺ ſobię imię dniá tego / tegoż włáśnie dniá ; bo obległ król Bábilońſki Jeruzálem práwie tegoż dniá.
A rzecż przez przypowieść do tego domu odpornego / podobieńſtwo mówiąc do nich : Ták mówi pánujący Pán : Przyſtáw ten gárniec / przyſtáw / á nálij weń wody ;
A zebráwƺy ƺtuki náleżące do niego / káżdą ƺtukę dobrą / udźiec y łopátkę / nájlepƺemi kośćiámi nápełń go.
Weźmijże y co nájwyborniejƺe bydlę / á náłóż ogień z kośći pod nim ; ſpráwże áby to wrzáło y kipiáło / żeby y kośći jego rozewrzáły w nim.
Przetoż ták mówi pánujący Pán : Biádá temu miáſtu krwáwemu / gárncowi / w którym zoſtáje przywárá jego / z którego / mówię / przywárá jego nie wychodźi : po ƺtukách / po ƺtukách wyćiągáj z niego / nie pádnieć náń loſ.
Bo iż krew jego jeſt w pośrodku jego / ná wierzchu ſkáły wyſtáwiło ją / nie wyláło jey ná źiemię / áby byłá zákrytá prochem :
Tedy y Ja rozniecę zápálcżywość ná wykonánie pomſty / wyſtáwię krew jego ná wierzchu ſkáły / áby nie byłá zákrytá.
Przetoż ták mówi pánujący Pán : Biádá temu miáſtu krwáwemu ; bo y Ja nániecę wielki ogień /
Przykłádájąc drew / rozniecájąc ogień / wniwecż obrácájąc mięſo / y zápráwiájąc korzeniem / ták / że y kośći ſpálone będą ;
A poſtáwię ten gárniec ná węgle jego próżny / áby śię zágrzáłá y rozpáliłá miedź jego / á żeby śię rozpłynęły w pośród jego plugáſtwá jego / á iżby znieśioná byłá przywárá jego.
Utrudźiło mię kłámſtwámi ſwemi / przetoż nie wynijdźie z niego wielkość ƺumowin jego / do ogniá muƺą ƺumowiny jego.
W niecżyſtośći twojey jeſt ſproſność ; dlátego / żem ćię ocżyƺcżáł ; á przećięś nie ocżyƺcżoná / y od niecżyſtośći twojey nie będźieƺ więcey ocżyƺcżáná / áż uſpokoję ná tobie rozgniewánie moje.
Ja Pán mówiłem ; przyjdźie to / y ucżynię to / nie cofnę śię / áni ſfolguję / áni mi żál będźie ; według dróg twoich / y według ſpráw twoich ſądźić ćię będę / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! oto Ja od ćiebie odejmę żądność ocżów twoich z prędká ; wƺákże nie kwil / áni płácż / á niech nie wychodzą łzy twoje.
Zániecháj kwileniá / nie cżyń żáłoby / jáko bywá nád umárłym ; cżápkę twoję włóż ná śię / á obuwie twoje wzuj ná nogi twoje / á nie záſłániáj wárg / á chlebá nicżyjego nie jedz.
Co gdym z poránku ludowi powiedźiáł / tedy umárłá żoná mojá w wiecżór ; y ucżyniłem ráno / jáko mi rozkázáno.
Y mówił lud do mnie : Y nie oznájmiƺże nám / co nám te rzecży znácżą / które ty cżyniƺ?
Tedym rzekł do nich : Słowo Páńſkie ſtáło śię do mnie / mówiąc :
Powiedz domowi Izráelſkiemu : Ták mówi pánujący Pán : Oto Ja ſplugáwię świątnicę moję / wynioſłość mocy wáƺej / żądność ocżów wáƺych / y to / w cżem śię kochá duƺá wáƺá / á ſynowie wáśi y córki wáƺe / któreśćie zoſtáwili / od miecżá polegną.
Y ucżynićie / jákom ucżynił ; nie záſłonićie wárgi / á chlebá nicżyjego jeść nie będźiećie ;
A cżápki ſwe ná głowách ſwych / y bóty ſwoje ná nogách ſwoich májąc / nie będźiećie kwilić áni płákáć ; ále będźiećie ſchnąć dla niepráwoſći wáƺych / á wzdycháć jeden z drugim.
Bo wám jeſt Ezechyjel dźiwem / według wƺyſtkiego / co on cżyni / cżynić będźiećie ; á gdy to przyjdźie / dowiećie śię / żem ja pánujący Pán.
A ty / ſynu cżłowiecży! ázáż w ten dźień / którego Ja odejmę od nich moc ich / weſele ozdoby ich / żądność ocżów ich / y to / po cżem tęſkni duƺá ich / ſynów ich y córki ich /
Izáli dniá onego nie przyjdźie do ćiebie ten / co ućiecże / oznájmując ći to?
Dniá onego otworzá śię uſtá twoje przy tym / który ujdźie ; á będźieƺ mówił / y nie będźieƺ więcey niemym ; y będźieƺ im dźiwem / á poznáją / żem Ja Pán.
