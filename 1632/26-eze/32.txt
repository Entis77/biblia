A dwunáſtego roku / mieśiącá dwunáſtego / pierwƺego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! podnieś láment nád Fáráonem / królem Egipſkim / á powiedz mu : Podobnyś ty lwowi młodemu między národámi / tyś jeſt jáko wieloryb w morzu / gdyż bujájąc po rzekách twoich mąćiƺ wodę nogámi twemi / y mięƺáƺ rzeki jego.
Ták mówi pánujący Pán : Rozćiągnę też ná ćię śieć moję przez zebránie wielu národów / którzy ćię wyćiągną niewodem moim ;
Y zoſtáwię ćię ná ziemi / ná polu porzucę ćię / y ſpráwię / że mieƺkáć będzie ná tobie wƺelkie ptáſtwo niebieſkie / y nákármię tobą zwierzá wƺyſtkiej ziemi ;
Y rozrzucę mięſo twoje po górách / y nápełnię doliny wyſokośćią twoją /
Y nápoję ziemię twoję / w której pływáƺ / krwią twoją áż do gór / ták / że y rzeki będą nápełnione tobą.
A gdy ćię zgáƺę / zákryję niebioſá / y ćiemne ucżynię gwiázdy ich / ſłońce obłokiem záſłonię / á kśiężyc nie dá świátłá ſwego.
Wƺyſtkie świátłá jáſne ná niebioſách záćmię dlá ćiebie / y przywiodę ćiemność ná ziemię twoję / mówi pánujący Pán.
Nádto záſmucę ſerce wielu národów / gdy zá ſpráwą moją przyjdzie wieść o ſtárćiu twojim między národy / do ziem / którycheś nie znáł.
Ucżynię / mówię / że śię zdumieje nád tobą wiele národów / á królowie ich zátrwożą śię bárdzo dlá ćiebie / gdy ƺermowáć będę miecżem ſwoim przed twárzą ich ; będą śię záiſte lękáć co chwilá káżdy o duƺę ſwoję w dzień upádku twego.
Bo ták mówi pánujący Pán : Miecż królá Bábilońſkiego przyjdzie ná ćię.
Miecżámi mocárzów poráżę zgráję twoję ; nájokrutniejśi ze wƺyſtkich národów / ći ſkáżą pychę Egipſką / y będzie zgłádzone wƺyſtko mnóſtwo jego.
Zágłádzę y wƺyſtko bydło jego / które jeſt przy wodách wielkich / ták / że ich więcej nie zmąći nogá cżłowiecżá / áni kopytá bydlęce mąćić ich będą.
Tedy ucżynię / że śię wody ich uſtoją / á rzeki ich jáko oliwá pójdą / mówi pánujący Pán.
Gdyż obrócę ziemię Egipſką w ſpuſtoƺenie / á ziemiá będzie wypróżnioná z pełnośći ſwojej / gdy pobiję wƺyſtkich mieƺkájących w niej ; y dowiedzą śię / żem Já Pán.
Toć jeſt láment / którym nád nią lámentowáć będą ; córki národów nárzekáć będą nád nią / nád Egiptem y nád wƺyſtkiem mnóſtwem jego nárzekáć będą / mówi pánujący Pán.
Potem dwunáſtego roku / piętnáſtego dniá tegoż mieśiącá / ſtáło śię ſłowo Páńſkie do mnie / mówiąc ;
Synu cżłowiecży! nárzekáj nád mnóſtwem Egipſkiem / á zepchnij je / y córki tych národów ſłáwnych áż do nájniżƺych miejſc ziemi / do tych / co zſtępują do dołu /
Y mów : Nád kogożeś wdzięcżniejƺy? Zſtąp / á połóż śię z nieobrzezáńcámi.
W pośród pobitych miecżem upádną ; pod miecż podány jeſt / wywlecżćież go ze wƺyſtką zgráją jego.
Mówić do niego będą nájmocniejśi z mocárzów z pośrodku grobu z pomocnikámi jego / którzy tám zſtąpiwƺy / leżą z nieobrzezáńcámi pobitymi miecżem.
Tám jeſt Aſſur / y wƺyſtká zgrájá jego / w około niego ſą groby jego ; wƺyſcy ći pobići upádli od miecżá.
Którego groby położone ſą przy ſtronách dołu / áby byłá zgrájá jego w około grobu jego ; ći wƺyſcy pobići polegli od miecżá / którzy puƺcżáli ſtrách w ziemi żyjácych.
Tám Elám y wƺyſtká zgrájá jego około grobu jego / ći wƺyſcy pobići upádli od miecżá / którzy zſtąpili w nieobrzezce do niſkośći ziemi / którzy puƺcżáli ſtrách ſwój w ziemi żyjących ; jużći odnoƺą háńbę ſwoją z tymi / którzy zſtępują do dołu.
W pośrodku pobitych poſtáwili mu łoże / y wƺyſtkiej zgrái jego / w około niego ſą groby jego ; wƺyſcy ći nieobrzezáńcy pobići ſą miecżem / przychodził ſtrách ich ná ziemię żyjących / jużći odnoƺą háńbę ſwoję z tymi / którzy zſtąpili do grobu / á w pośrodku pobitych położeni ſą.
Tám Meſech / Tubál y wƺyſtká zgrájá jego / y w około niego groby jego / ći wƺyſcy nieobrzezáńcy pobići miecżem / choć puƺcżáli ſtrách ſwój w ziemi żyjących.
Acżkolwiekći jeƺcże nie polegli z mocárzámi / którzy upádli z nieobrzezáńców / co zſtąpili do grobu z wojennym orężem ſwoim / y położyli miecże ſwe pod głowy ſwe : á wƺákże przyjdzie niepráwość ich ná kośći ich / choćiáż ſtrách tych mocárzów był w ziemi żyjących.
Y ty w pośrodku nieobrzezáńców ſtárty będzieƺ / y będzieƺ leżáł między pobitymi miecżem.
Tám Edom / y królowie jego / y wƺyſcy kśiążętá jego / którzy położeni ſą / z mocą ſwoją y z pobitymi miecżem ; y ći z nieobrzezáńcámi leżeć będą / y z tymi / którzy zſtępują do dołu.
Tám wƺyſcy zgołá kśiążętá północnej ſtrony / y wƺyſcy Sydońcżycy / którzy zſtępują do pobitych z ſtráchem ſwoim / zá moc ſwoję wſtydzić śię będą / y leżeć będą ći nieobrzezáńcy z pobitymi miecżem / á odnioſą háńbę ſwoję z tymi / którzy zſtępują do dołu.
Tych ujrzáwƺy Fáráo ućieƺy śię nád wƺyſtką zgráją ſwoją / którá jeſt miecżem pobitá / Fáráo y wƺyſtko wojſko jego / mówi pánujący Pán.
Bo puƺcżę ſtrách mój w ziemi żyjących / y położony będzie między nieobrzezáńcámi z pobitymi miecżem Fáráo y wƺyſtká zgrájá jego / mówi pánujący Pán.
