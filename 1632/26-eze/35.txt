Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiwko górze Seir / á prorokuj przećiwko niej /
Y mów do niej : Ták mówi pánujący Pán : Otom Já przećiwko tobie / góro Seir! á wyćiągnę rękę moję ná ćię / y podám ćię ná wielkie ſpuſtoƺenie.
Miáſtá twoje w ſpuſtoƺenie obrócę / że będzieƺ ſpuſtoƺoná / y dowieƺ śię / żem Já Pán.
Przeto / iż wiecżną nieprzyjáźń wiedzieƺ / y rozpráƺáƺ ſynów Izráelſkich oſtrzem miecżá cżáſu utrápieniá ich / cżáſu wykonywániá káźni ich ;
Dlátego / jáko żyję Já / mówi pánujący Pán / że ćię podám ná zábićie / á krew ćię śćigáć będzie ; poniewáż krwi rozlewániá w nienáwiśći nie máƺ / przetoż ćię krew śćigáć będzie.
Y obrócę górę Seir w wielkie ſpuſtoƺenie / á wygubię z niej przechodzącego / y wrácájącego śię ;
Y nápełnię góry jej pobitymi jej ; ná págórkách twoich / y ná dolinách twoich / y przy wƺyſtkich ſtrumieniách twoich pobići miecżem polegną ná nich ;
Ná puſtynie wiecżne podám ćię / y w miáſtách twoich mieƺkáć nie będą ; á dowiećie śię / żem Já Pán.
Przetoż / iż mówiƺ : Te dwá národy / y te dwie ziemie moje będą á pośiądziemy tę / w której Pán mieƺkáł ;
Dlátegoż / jáko żyję Já / mówi pánujący Pán / że ucżynię według gniewu twego y według záwiśći twojej / którąś cżyniłá w nienáwiśći ſwej przećiwko nim ; y będę poznány od nich / gdy ćię ſądzić będę ;
Y dowieƺ śię / żem Já Pán ſłyƺáł wƺyſtkie urągániá twoje / któreś wyrzekłá przećiw górom Izráelſkim / mówiąc : Spuſtoƺone ſą / á nám podáne ſą ku pożárćiu.
Bośćie śię przećiwko mnie wynośili uſty wáƺemi / á rozmnożyliśćie ſłowá ſwoje przećiwko mnie / com Já ſłyƺáł.
Ták mówi pánujący Pán : Jáko śię tá wƺyſtká ziemiá weſeli / ták ćię obrócę w puſtnię.
Jáko śię ty weſeliƺ nád dziedzictwem domu Izráelſkiego / przeto / że jeſt ſpuſtoƺone / ták ucżynię y tobie ; będzieƺ ſpuſtoƺoná / o góro Seir! á zgołá wƺyſtká ziemiá Edomſká ; y dowiedzą śię / żem Já Pán.
