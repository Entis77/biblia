Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiwko górze Seir / á prorokuj przećiwko niej /
Y mów do niej : Ták mówi pánujący Pán : Otom Ja przećiwko tobie / góro Seir! á wyćiągnę rękę moję ná ćię / y podám ćię ná wielkie ſpuſtoƺenie.
Miáſtá twoje w ſpuſtoƺenie obrócę / że będźieƺ ſpuſtoƺoná / y dowieƺ śię / żem Ja Pán.
Przeto / iż wiecżną nieprzyjáźń wiedźieƺ / y rozpráƺáƺ ſynów Izráelſkich oſtrzem miecżá cżáſu utrápieniá ich / cżáſu wykonywániá káźni ich ;
Dlátego / jáko żyję Ja / mówi pánujący Pán / że ćię podám ná zábićie / á krew ćię śćigáć będźie ; poniewáż krwi rozlewániá w nienáwiśći nie máƺ / przetoż ćię krew śćigáć będźie.
Y obrócę górę Seir w wielkie ſpuſtoƺenie / á wygubię z niej przechodzącego / y wrácájącego śię ;
Y nápełnię góry jey pobitymi jey ; ná págórkách twoich / y ná dolinách twoich / y przy wƺyſtkich ſtrumieniách twoich pobići miecżem polegną ná nich ;
Ná puſtynie wiecżne podám ćię / y w miáſtách twoich mieƺkáć nie będą ; á dowiećie śię / żem Ja Pán.
Przetoż / iż mówiƺ : Te dwá národy / y te dwie źiemie moje będą á pośiądźiemy tę / w którey Pán mieƺkáł ;
Dlátegoż / jáko żyję Ja / mówi pánujący Pán / że ucżynię według gniewu twego y według záwiśći twojey / którąś cżyniłá w nienáwiśći ſwej przećiwko nim ; y będę poznány od nich / gdy ćię ſądźić będę ;
Y dowieƺ śię / żem Ja Pán ſłyƺáł wƺyſtkie urągániá twoje / któreś wyrzekłá przećiw górom Izráelſkim / mówiąc : Spuſtoƺone ſą / á nám podáne ſą ku pożárćiu.
Bośćie śię przećiwko mnie wynośili uſty wáƺemi / á rozmnożyliśćie ſłowá ſwoje przećiwko mnie / com Ja ſłyƺáł.
Ták mówi pánujący Pán : Jáko śię tá wƺyſtká źiemiá weſeli / ták ćię obrócę w puſtnię.
Jáko śię ty weſeliƺ nád dźiedźictwem domu Izráelſkiego / przeto / że jeſt ſpuſtoƺone / ták ucżynię y tobie ; będźieƺ ſpuſtoƺoná / o góro Seir! á zgołá wƺyſtká źiemiá Edomſká ; y dowiedzą śię / żem Ja Pán.
