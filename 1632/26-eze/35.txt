Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiwko górze Seir / á prorokuj przećiwko niej /
Y mów do niej : Ták mówi pánujący Pán : Otom Ja przećiwko tobie / góro Seir! á wyćiągnę rękę moję ná ćię / y podám ćię ná wielkie ſpuſtoƺenie.
Miáſtá twoje w ſpuſtoƺenie obrócę / że będźieƺ ſpuſtoƺoná / y dowieƺ śię / żem Ja Pán.
Przeto / iż wiecżną nieprzyjáźń wiedźieƺ / y rozpráƺáƺ ſynów Izráelſkich oſtrzem miecżá cżáſu utrapienia ich / cżáſu wykonywánia káźni ich ;
Dla tego / jáko żyję Ja / mówi pánujący Pán / że ćię podám ná zábićie / á krew ćię śćigáć będźie ; ponieważ krwi rozlewánia w nienáwiśći nie maƺ / przetoż ćię krew śćigáć będźie.
Y obrócę górę Seir w wielkie ſpuſtoƺenie / á wygubię z niej przechodzącego / y wrácájącego śię ;
Y nápełnię góry jey pobitymi jey ; ná págórkách twojich / y ná dolinách twojich / y przy wƺyſtkich ſtrumieniách twojich pobići miecżem polegną ná nich ;
Ná puſtynie wiecżne podám ćię / y w miáſtách twojich mieƺkáć nie będą ; á dowiećie śię / żem Ja Pán.
Przetoż / iż mówiƺ : Te dwá narody / y te dwie źiemie moje będą á pośiądźiemy tę / w którey Pán mieƺkał ;
Dlátegoż / jáko żyję Ja / mówi pánujący Pán / że ucżynię według gniewu twego y według záwiśći twojey / którąś cżyniłá w nienáwiśći ſwej przećiwko nim ; y będę poznány od nich / gdy ćię ſądźić będę ;
Y dowieƺ śię / żem Ja Pán ſłyƺał wƺyſtkie urągánia twoje / któreś wyrzekłá przećiw górom Izráelſkim / mówiąc : Spuſtoƺone ſą / á nam podáne ſą ku pożárćiu.
Bośćie śię przećiwko mnie wynośili uſty wáƺemi / á rozmnożyliśćie ſłowá ſwoje przećiwko mnie / com Ja ſłyƺał.
Ták mówi pánujący Pán : Jáko śię tá wƺyſtká źiemiá weſeli / ták ćię obrócę w puſtynię.
Jáko śię ty weſeliƺ nád dźiedźictwem domu Izráelſkiego / przeto / że jeſt ſpuſtoƺone / ták ucżynię y tobie ; będźieƺ ſpuſtoƺoná / o góro Seir! á zgołá wƺyſtká źiemiá Edomſká ; y dowiedzą śię / żem Ja Pán.
