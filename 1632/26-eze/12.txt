Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! ty w pośrodku domu odpornego mieƺkáƺ / którzy máją ocży / áby widzieli / á nie widzą / uƺy máją / áby ſłyƺeli / á nie ſłyƺą ; przeto / że domem odpornym ſą.
Ty tedy / ſynu cżłowiecży! ſpráw ſobie ſprzęt przeprowádzeniá / á przeprowádzáj śię we dnie przed ocżymá ich / á przeprowádzáj śię z miejſcá ſwego ná miejſce inne przed ocżymá ich / ázá wżdy obácżą / ácż domem odpornym ſą.
Y wynieśieƺ ſprzęt ſwój / jáko ſprzęt przeprowádzeniá / we dnie przed ocżymá ich ; á ſám wynijdź w wiecżór przed ocżymá ich / jáko wychodzą / którzy śię przeprowádzáją.
Przed ocżymá ich przekop ſobie mur / á wynieś przezeń ſprzęt twój.
Przed ocżymá ich ná rámionách nieś / z zmierzkiem wynieś / twárz ſwoję zákryj / á nie pátrz ná ziemię ; bom ćię dáł zá dziw domowi Izráelſkiemu.
Y ucżyniłem ták / jáko mi rozkázáno ; ſprzęt mój wynioſłem jáko ſprzęt prowádzącego śię we dnie / á w wiecżór przekopáłem ſobie mur ręką ; z zmierzkiem wynioſłem go ná rámieniu nioſąc przed ocżymá ich.
Znowu ſtáło śię ſłowo Páńſkie do mnie ráno / mówiąc :
Synu cżłowiecży! Izáli nie rzekł do ćiebie dom Izráelſki / dom ten odporny : Cóż to cżyniƺ?
Rzecżże im / ták mówi pánujący Pán : Ná kśięćiá / który jeſt w Jeruzálemie / śćiągá śię to brzemię / y ná wƺyſtek dom Izráelſki / którzy ſą w pośród jego.
Rzecżże do nich : Jám jeſt dziwem wáƺym ; jákom ucżynił / ták śię im ſtánie ; przeprowádzą śię / y w niewolę pójdą.
A kśiążę / który jeſt w pośrodku nich / ná rámieniu ponieśie ſprzęt ſwój z zmierzkiem / y wynijdzie ; mur przekopią / áby go wywiedli przezeń ; twárz ſwoję zákryje / ták / że nie będzie widziáł ziemi okiem ſwojim.
Bo rozćiągnę náń śieć ſwoję / y pojmány będzie niewodem moim / y przywiodę go do Bábilonu / do ziemi Cháldejſkiej / á tej nie oglądá / y tám umrze.
Tákże też wƺyſtkich / którzy ſą około niego / pomoc jego / y wƺyſtkie hufy jego rozproƺę ná wƺyſtkie ſtrony / y miecżá dobędę zá nimi ;
Y poznáją / żem Já Pán / gdy ich rozproƺę między národy / y rozwieję ich po ziemiách.
Jednák zoſtáwię z nich trochę mężów po miecżu / po głodzie y po morze / áby opowiádáli wƺyſtkie obrzydliwośći ſwe między národámi / do których wnijdą / y poznáją / żem Já Pán.
Znowu ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! chleb ſwój z ſtráchem jedz y wodę twoję ze drżeniem y z ſmutkiem pij /
A rzecż do ludu tej ziemi : Ták mówi pánujący Pán o tych / którzy mieƺkáją w Jeruzálemie / o ziemi Izráelſkiej : Chleb ſwój z ſmutkiem jeść / á wodę ſwą z trwogą pić będą / áby byłá ziemiá jego złupioná z doſtátków ſwoich dlá bezpráwiá wƺyſtkich mieƺkájących w niej ;
Tákże miáſtá / w których mieƺkáją / ſpuſtoƺone będą / y ziemiá ſpuſtoƺeje ; y dowiećie śię / żem Já Pán.
Stáło śię záś ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! cóż to zá przypowieść u wáſ o ziemi Izráelſkiej / iż mówićie : Przedłużąć śię dni / á z tego widzeniá nic nie będzie?
Przetoż mów do nich : Ták mówi pánujący Pán : Spráwię Já to / iż uſtánie tá przypowieść / á nie będą używáć tej przypowieśći więcej w Izráelu ; powiedz im : Y owƺem / przybliżyły śię te dni / y ſpełnienie wƺelkiego widzeniá.
Bo nie będzie więcej żádnego márnego widzeniá / áni wieƺcżby pochlebcy w pośrodku domu Izráelſkiego.
Przeto / że Já Pán mówić będę / á którekolwiek ſłowo wyrzekę / ſtánie śię ; nie pójdzie w długą / ále zá dni wáƺych / domie odporny! wyrzekę ſłowo / y wypełnię je / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! oto dom Izráelſki mówią : To widzenie / które ten widzi / odwlecże śię ná wiele dni / á o dálekich cżáſách ten prorokuje ;
Przeto im rzecż : Ták mówi pánujący Pán : Nie pójdzieć w długą żádne ſłowo moje ; ále ſłowo / które wyrzekę / ſtánie śię / mówi pánujący Pán.
