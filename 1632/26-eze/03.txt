Y rzekł do mnie : Synu cżłowiecży! co przed tobą jeſt / zjedz ; zjedz te kśięgi / á idź y mów do domu Izráelſkiego.
Otworzyłem tedy uſtá ſwe / y dáł mi zjeść one kśięgi /
A mówił do mnie : Synu cżłowiecży! nákárm brzuch twój / á wnętrznośći twoje nápełnij temi kśięgámi / któreć dáję. Y zjádłem je / y były w uſtách moich ſłodkie jáko miód.
Zátem rzekł do mnie : Synu cżłowiecży! idź á wnijdź do domu Izráelſkiego / y mów ſłowy mojemi do nich.
Bo ćię nie do ludu nieznájomej mowy / álbo trudnego języká poſyłám / ále do domu Izráelſkiego ;
Nie do wielu národów nieznájomej mowy / y trudnego języká / którychbyś ſłów nie zrozumiáł / którzy jednák / gdybym ćię do nich poſłáł / uſłucháliby ćię.
Lecż dom Izráelſki nie będźie ćię chćiáł uſłucháć / poniewáż mnie ſámego uſłucháć nie chcą ; bo wƺyſtek dom Izráelſki jeſt twárdego cżołá y zátwárdzonego ſercá.
Ale otom ucżynił twárz twoję twárdą przećiwko twárzy ich / á cżoło twe twárde przećiwko cżołu ich.
Ucżyniłem cżoło twe jáko dyjáment / y twárdƺe nád ſkáłę ; nie bójże śię ich / áni śię lękáj twárzy ich / przeto / że ſą domem odpornym.
Y rzekł do mnie : Synu cżłowiecży! wƺyſtkie ſłowá moje / które mówić będę do ćiebie / przyjmij do ſercá twego / á ſłucháj uƺymá twemi.
Idź á wnijdź do pojmánych / do ſynów ludu twego / y mów do nich / á powiedz im : Ták mówi pánujący Pán / niech oni ſłucháją / álbo nie.
Tedy mię duch podnióſł / y ſłyƺáłem zá ſobą głoſ grzmotu wielkiego : Błogoſłáwioná niech będźie chwáłá Páńſká z miejſcá ſwego ;
Y ƺum ſkrzydeł onych zwierząt / które śię náſpół dotykáły / y głoſ kół ná przećiwko nim / y głoſ grzmotu wielkiego.
A duch podnióſł mię y wźiął mię. Y odƺedłem z gorzkośćią w rozgniewániu duchá mego ; ále ręká Páńſká nádemną mocná byłá.
Y przyƺedłem do pojmánych do Telábib / którzy mieƺkáli przy rzece Chebár / y śiádłem gdźie oni mieƺkáli / á śiedźiáłem tám śiedm dni w pośrodku ich / zdumiáwƺy śię.
A gdy minęło śiedm dni / było ſłowo Páńſkie do mnie mówiące :
Synu cżłowiecży! Dáłem ćię ſtróżem domowi Izráelſkiemu / ábyś ſłyƺąc ſłowo z uſt moich nápomniáł ich odemnie.
Gdybym Já rzekł niepobożnemu : Śmierćią umrzeƺ / á nie nápomniáłbyś go / y nie mówiłbyś / ábyś go odwiódł od niezbożnej drogi jego / ták / żebyś go przy żywoćie záchowáł / tedy onći niezbożny w niepráwośći ſwojej umrze ; ále krwi jego z ręki twojej ƺukáć będę.
Lecż jeźlibyś ty nápomniáł niezbożnego / á nie odwróćiłby śię od niezbożnośći ſwej / y od drogi ſwej niezbożnej / tedy onći w niepráwośći ſwojej umrze ; ále ty duƺę ſwoję wybáwiƺ.
Tákże jeźliby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojej / á cżyniłby niepráwość / á Jábym położył záwádę przed nim / y tákby umárł / á tybyś go nie nápomniáł : w grzechu ſwym umrze / á nie przyjdą ná pámięć ſpráwiedliwośći jego / które cżynił / lecż krwi jego z ręki twojej ƺukáć będę.
Ale jeźlibyś ty nápomniáł ſpráwiedliwego / áby nie zgrzeƺył ten ſpráwiedliwy / y nie grzeƺyłby / záiſte żyć będźie / bo nápomnienie przyjął / á ty duƺę ſwoję wybáwiƺ.
Y byłá tám nádemną ręká Páńſká / y rzekł do mnie : Wſtáwƺy wyjdź w pole / á tám śię z tobą rozmówię.
A ták wſtáwƺy ƺedłem w pole / á oto chwáłá Páńſká ſtáłá tám / jáko chwáłá / którąm widźiáł u rzeki Chebár / y upádłem ná oblicże moje.
Tedy duch wſtąpił w mię / á poſtáwiwƺy mię ná nogi moje mówił do mnie / y rzekł mi : Wnijdź / zámknij śię w domu ſwym.
Bo oto ná ćię / ſynu cżłowiecży! włożą powrozy / y zwiążą ćię niemi / á nie będźieƺ mógł wynijść między nich.
A Já ucżynię / że język twój przylgnie do podniebieniá twego / y będźieƺ niemy / á nie będźieƺ im mężem ſtrofującym / przeto / że ſą domem odpornym.
Ale gdy z tobą mówić będę / otworzę uſtá twoje ; tedy rzecżeƺ do nich : Ták mówi pánujący Pán : Kto chce ſłucháć / niech ſłuchá / á kto nie chce / niech nie ſłuchá / gdyż domem odpornym ſą.
