Y rzekł do mnie : Synu cżłowiecży! co przed tobą jeſt / zjedz ; zjedz te kśięgi / á idź y mów do domu Izráelſkiego.
Otworzyłem tedy uſtá ſwe / y dał mi zjeść one kśięgi /
A mówił do mnie : Synu cżłowiecży! nákárm brzuch twój / á wnętrznośći twoje nápełnij temi kśięgámi / któreć dáję. Y zjádłem je / y były w uſtách mojich ſłodkie jáko miód.
Zátem rzekł do mnie : Synu cżłowiecży! idź á wnijdź do domu Izráelſkiego / y mów ſłowy mojemi do nich.
Bo ćię nie do ludu nieznájomej mowy / álbo trudnego języká poſyłám / ále do domu Izráelſkiego ;
Nie do wielu narodów nieznájomej mowy / y trudnego języká / którychbyś ſłów nie zrozumiał / którzy jednák / gdybym ćię do nich poſłał / uſłucháliby ćię.
Lecż dom Izráelſki nie będźie ćię chćiał uſłucháć / ponieważ mnie ſámego uſłucháć nie chcą ; bo wƺyſtek dom Izráelſki jeſt twárdego cżołá y zátwárdzonego ſercá.
Ale otom ucżynił twárz twoję twárdą przećiwko twárzy ich / á cżoło twe twárde przećiwko cżołu ich.
Ucżyniłem cżoło twe jáko dyjáment / y twárdƺe nád ſkáłę ; nie bójże śię ich / áni śię lękáj twárzy ich / przeto / że ſą domem odpornym.
Y rzekł do mnie : Synu cżłowiecży! wƺyſtkie ſłowá moje / które mówić będę do ćiebie / przyjmij do ſercá twego / á ſłucháj uƺymá twemi.
Idź á wnijdź do pojmánych / do ſynów ludu twego / y mów do nich / á powiedz im : Ták mówi pánujący Pán / niech oni ſłucháją / álbo nie.
Tedy mię duch podnióſł / y ſłyƺáłem zá ſobą głos grzmotu wielkiego : Błogoſłáwioná niech będźie chwałá Páńſka z miejſcá ſwego ;
Y ƺum ſkrzydeł onych zwierząt / które śię náſpół dotykáły / y głos kół ná przećiwko nim / y głos grzmotu wielkiego.
A duch podnióſł mię y wźiął mię. Y odƺedłem z gorzkośćią w rozgniewániu duchá mego ; ále ręká Páńſka nádemną mocná byłá.
Y przyƺedłem do pojmánych do Telábib / którzy mieƺkáli przy rzece Chebár / y śiádłem gdźie oni mieƺkáli / á śiedźiáłem tám śiedm dni w pośrzodku ich / zdumiáwƺy śię.
A gdy minęło śiedm dni / było ſłowo Páńſkie do mnie mówiące :
Synu cżłowiecży! Dáłem ćię ſtróżem domowi Izráelſkiemu / ábyś ſłyƺąc ſłowo z uſt mojich nápomniał ich odemnie.
Gdybym Ja rzekł niepobożnemu : Śmierćią umrzeƺ / á nie nápomniáłbyś go / y nie mówiłbyś / ábyś go odwiódł od niezbożney drogi jego / ták / żebyś go przy żywoćie záchował / tedy onći niezbożny w niepráwośći ſwojey umrze ; ále krwi jego z ręki twojey ƺukáć będę.
Lecż jeſlibyś ty nápomniał niezbożnego / á nie odwróćiłby śię od niezbożnośći ſwej / y od drogi ſwej niezbożney / tedy onći w niepráwośći ſwojey umrze ; ále ty duƺę ſwoję wybáwiƺ.
Tákże jeſliby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojey / á cżyniłby niepráwość / á Jábym położył záwádę przed nim / y tákby umárł / á tybyś go nie nápomniał : w grzechu ſwym umrze / á nie przyjdą ná pámięć ſpráwiedliwośći jego / które cżynił / lecż krwi jego z ręki twojey ƺukáć będę.
Ale jeſlibyś ty nápomniał ſpráwiedliwego / áby nie zgrzeƺył ten ſpráwiedliwy / y nie grzeƺyłby / zájiſte żyć będźie / bo nápomnienie przyjął / á ty duƺę ſwoję wybáwiƺ.
Y byłá tám nádemną ręká Páńſka / y rzekł do mnie : Wſtáwƺy wyjdź w pole / á tám śię z tobą rozmówię.
A ták wſtáwƺy ƺedłem w pole / á oto chwałá Páńſka ſtáłá tám / jáko chwałá / którąm widźiał u rzeki Chebár / y upádłem ná oblicże moje.
Tedy duch wſtąpił w mię / á poſtáwiwƺy mię ná nogi moje mówił do mnie / y rzekł mi : Wnijdź / zámknij śię w domu ſwym.
Bo oto ná ćię / ſynu cżłowiecży! włożą powrozy / y zwiążą ćię niemi / á nie będźieƺ mógł wynijść miedzy nich.
A Ja ucżynię / że język twój przylgnie do podniebieniá twego / y będźieƺ niemy / á nie będźieƺ im mężem ſtrofującym / przeto / że ſą domem odpornym.
Ale gdy z tobą mówić będę / otworzę uſtá twoje ; tedy rzecżeƺ do nich : Ták mówi pánujący Pán : Kto chce ſłucháć / niech ſłuchá / á kto nie chce / niech nie ſłuchá / gdyż domem odpornym ſą.
