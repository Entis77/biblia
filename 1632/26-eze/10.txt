Y widźiáłem ; á oto ná rozpoſtárćiu / które było nád głową Cherubinów / jákoby kámień ƺáfirowy / ná wejrzeniu jáko podobieńſtwo ſtolicy / ukázáło śię nád nimi.
Tedy rzekł do onego mężá odźiánego ƺátą lniáną / mówiąc : Wnijdź między kołá pod Cherubinów / á nápełń ręce ſwe węglem ogniſtym z pośród Cherubinów / y rozrzuć po mieśćie. Y wƺedł przed ocżymá memi.
( A Cherubinowie ſtáli po práwej ſtronie domu / gdy wchodźił on mąż / á obłok nápełnił śień wnętrzną.
Bo gdy śię byłá podnioſłá chwałá Páńſká z Cherubinów / ku progowi domu / tedy nápełniony był dom obłokiem / á śień nápełnioná byłá jáſnośćią chwáły Páńſkiej ;
A ƺum ſkrzydeł Cherubinów ſłyƺány był áż do śieni zewnętrzney / jáko głoſ Bogá wƺechmocnego / gdy mówi ).
Gdy tedy rozkázáł onemu mężowi odźiánemu w ƺátę lniáną / mówiąc : Weźmij ogniá z pośrodku kół / z pośrodku Cherubinów : wƺedł y ſtánął podle kół.
Tedy wyćiągnął Cherubin jeden rękę ſwą w pośród Cherubinów do onego ogniá / który był w pośrodku Cherubinów / á wźiąwƺy podáł go w rękę onego odźiánego ƺátą lniáną / który wźiąwƺy go wyƺedł.
Bo śię ukázáło ná onych Cherubinách podobieńſtwo ręki cżłowiecżej pod ſkrzydłámi ich.
Potemem wejrzáł / á oto cżtery kołá podle Cherubinów / koło jedno podle jednego Cherubiná / á ták káżde koło podle káżdego Cherubiná / á podobieńſtwo kół jáko bárwá kánieniá Tárſyſ ;
A ná wejrzeniu miáły jednákie podobieńſtwá one kołá / jákoby było koło w pośrodku kołá.
Gdy chodźiły / ná cżtery ſtrony ſwoje chodźiły ; nie uchyláły śię / gdy ƺły / ále do onego miejſcá / do którego śię wódź obrácáł / zá nim ƺły ; nie uchyláły śię / gdy ƺły.
A wƺytko ćiáło ich / y grzbiety ich / y ręce ich / y ſkrzydłá ich / tákże y kołá pełne były ocżów około onych ſámych cżterech / y kół ich.
A kołá one názwáł okręgiem / gdźiem ja ſłyƺáł.
A káżde zwierzę miáło cżtery twárze ; twárz pierwƺá byłá twárz Cherubinowá / drugá twárz byłá twárz cżłowiecżá / trzećiá byłá twárz lwiá / á cżwártá byłá twárz orlá.
Y podnieśli śię Cherubinowie. Toć ſą one zwierzętá / którem widźiáł nád rzeką Chebár.
A gdy chodźili Cherubinowie / chodźiły y kołá podle nich ; á gdy ponośili Cherubinowie ſkrzydłá ſwoje / áby śię wzbili od źiemi / nie odwrácáły śię też kołá od nich.
Gdy oni ſtáli / ſtáły / á gdy śię ponośili / podnośiły śię też z nimi ; bo duch zwierząt był w nich.
Y odeƺłá chwałá Páńſká od progu domu / y ſtánęłá nád Cherubinámi.
Gdy podnieśli Cherubinowie ſkrzydłá ſwoje / á wzbili śię od źiemi przed ocżymá mojemi odchodząc / á kołá przećiwko nim / y ſtánęli w wejśćiu brámy domu Páńſkiego wſchodniej / tedy chwałá Bogá Izráelſkiego z wierzchu nád nimi byłá.
Toć ſą one zwierzętá / którem widźiáł pod Bogiem Izráelſkim nád rzeką Chebár ; y poznáłem / iż to byli Cherubinowie.
Po cżtery twárze miał káżdy z nich / y po cżtery ſkrzydłá káżdy z nich / á podobieńſtwo rąk ludzkich pod ſkrzydłámi ich.
A podobieńſtwo twárzy ich było jáko twárzy / którem widźiáł u rzeki Chebár ; tákże y oblicże ich tákież było / y oni ſámi ; káżdy z nich proſto ku ſwej ſtronie chodźił.
