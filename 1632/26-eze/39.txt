A ty / ſynu cżłowiecży! prorokuj przećiwko Gogowi / á mów : Ták mówi pánujący Pán : Otom Já przećiwko tobie / Gogu / kśięćiu główny w Meſechu y w Tubálu!
Y záwrócę ćię / á ƺeśćioráką plágą śćiſnę ćię / y wywiodę ćię z ſtron północych / á przywiodę ćię ná góry Izráelſkie ;
A wytrącę łuk twój z ręki twojej lewej / y ſtrzáły twoje z práwej ręki twojej wybiję.
Ná górách Izráelſkich upádnieƺ / ty y wƺyſtkie hufy twoje / y národy / które z tobą będą ; ptáſtwu y wƺelkiej rzecży ſkrzydláſtej / y zwierzowi polnenu podám ćię ná pożárćie.
Ná oblicżu polá upádnieƺ ; bom Já to wyrzekł / mówi pánujący Pán.
Y puƺcżę ogień ná Mágogá / y ná tych / co bezpiecżnie ná wyſpách mieƺkáją ; á dowiedzą śię / żem Já Pán.
A imię świętobliwośći mojej oznájmię w pośrodku ludu mego Izráelſkiego / y nie dopuƺcżę więcej zmázáć imieniá świętobliwośći mojej ; y dowiedzą śię národy / żem Já Pán / święty w Izráelu.
Oto przyjdzie y ſtánie śię to / mówi pánujący Pán / tegoż dniá / o którymem mówił.
Tedy wynijdą obywátele miáſt Izráelſkich / á zápáliwƺy ſpálą oręże y tárcże / y drzewcá / łuki y ſtrzáły / kije ręcżne y włócżnie / y będą z nich niećić ogień przez śiedm lát ;
A nie będą drew nośić z polá / áni ich rąbáć w láſách / ále z orężá ogień niećić będą á złupią tych / którzy ich łupili / y ſplundrują tych / którzy ich plundrowáli / mówi pánujący Pán.
Y ſtánie śię dniá onego / że dám Gogowi miejſce ná grób / tám w Izráelu / dolinę / którędy chodzą ná wſchód ſłońcá ku morzu / którá zátká uſtá mimo idących ; y pogrzebią tám Gogá y wƺyſtkie zgráje jego / á będą ją zwáć doliną mnóſtwá Gogowego ;
Bo ich grześć będzie dom Izrelſki przez śiedm mieśięcy / áby ocżyśćili ziemię.
A ták pogrzebie ich wƺyſtek lud onej ziemi / y obróći śię im to w ſłáwę / w dzień / którego Já będę uwielbiony / mówi pánujący Pán.
Y obiorą mężów uſtáwicżnych / którzyby śię przechodzili po onej ziemi / przechodzili / mówię / á chowáli tych / którzyby zoſtáli ná ziemi / áby ją wycżyśćili / á po wyjśćiu śiedmiu mieśięcy ƺukáć pocżną.
A ći przechodząc chodzić będą po ziemi / á ujrzáwƺy kośći cżłowiecże poſtáwią przy nich znák / áby je pochowáli ći / co chowáją umárłych / w dolinie mnóſtwá Gogowego.
Owƺem / to mnóſtwo jego będzie ku ſłáwie miáſtu / gdy ocżyƺcżą onę ziemię.
A ty / ſynu cżłowiecży! ták mówi pánujący Pán : Rzecż ptáſtwu y wƺelkiej rzecży ſkrzydláſtej / y káżdej beſtyi polnej : Zbierzćie śię / á przyjdźćie / zgromádźćie śię zewƺąd ná ofiárę moję / którą Já wám ſpráwuję / ofiárę wielką ná górách Izráelſkiech / żebyśćie jedli mięſo / y pili krew.
Mięſo mocárzy jeść będziećie / á krew kśiążąt ziemſkich pić będziećie / báránów / báránków / y kozłów / y ćielców / którzy wƺyſcy potyli w Báſán.
Nájećie śię tłuſtośći do ſytośći á nápijećie śię krwi do upićiá z tej ofiáry mojej / którą wám nágotuję.
Y náſyćićie śię z ſtołu mego końmi / y jeźdźcámi / mocárzámi / y káżdym mężem wálecżnym / mówi pánujący Pán.
A ták objáwię chwáłę moję między národámi / y oglądáją wƺyſtkie národy ſąd mój / którym ucżynił / y rękę moję / którąm ná nie wyćiągnął ;
A dowie śię dom Izráelſki / żem Já Pán / Bóg ich / od onego dniá y ná potem.
Poznáją też y národy / iż dlá niepráwośći ſwojej záwiedzieni ſą do więzieniá dom Izráelſki / dlátego / iż wyſtąpili przećiwko mnie. Dlátegom też był zákrył oblicże ſwoje przed nimi / á podáłem ich w ręce nieprzyjáćiół ich / áby wƺyſcy od miecżá polegli.
Według niecżyſtośći ich / y według przewrotnośći ich obƺedłem śię z nimi / y zákryłem oblicże moje przed nimi.
Przetoż ták mówi pánujący Pán : Jużći przywrócę więźniów Jákóbowych ; á zmiłuję śię nád wƺyſtkim domem Izráelſkim ; y gorliwym będę dlá imieniá świętobliwośći mojej /
Gdy odnioſą poháńbienie ſwoje y wƺyſtko przeſtępſtwo ſwoje / którem wyſtąpili przećiwko mnie / gdy bezpiecżnie mieƺkáli w ziemi ſwojej / á nie był / ktoby ich trwożył ;
A gdy ich przywrócę z národów / y zgromádzę ich z ziem nieprzyjáćiół ich / y będę poświęcony w nich przed ocżymá wielu národów.
Tedy śię dowiedzą / żem Já Pán / Bóg ich / gdy záwiódłƺy ich do národów záśię ich zgromádzę do ziemi ich / á nie zoſtáwię tám więcej żádnego z nich.
Y nie zákryję więcej oblicżá mego przed nimi / gdyż wyleję duchá mojego ná dom Izráelſki / mówi pánujący Pán.
