A ty / ſynu cżłowiecży! prorokuj przećiwko Gogowi / á mów : Ták mówi pánujący Pán : Otom Ja przećiwko tobie / Gogu / kśięćiu główny w Meſechu y w Tubálu!
Y záwrócę ćię / á ƺeśćioráką plágą śćiſnę ćię / y wywiodę ćię z ſtron północych / á przywiodę ćię ná góry Izráelſkie ;
A wytrącę łuk twój z ręki twojey lewej / y ſtrzáły twoje z práwej ręki twojey wybiję.
Ná górách Izráelſkich upádnieƺ / ty y wƺyſtkie hufy twoje / y narody / które z tobą będą ; ptáſtwu y wƺelkiej rzecży ſkrzydláſtey / y zwierzowi polnenu podám ćię ná pożárćie.
Ná oblicżu polá upádnieƺ ; bom Ja to wyrzekł / mówi pánujący Pán.
Y puƺcżę ogień ná Mágogá / y ná tych / co bezpiecżnie ná wyſpách mieƺkáją ; á dowiedzą śię / żem Ja Pán.
A imię świętobliwośći mojey oznájmię w pośrzodku ludu mego Izráelſkiego / y nie dopuƺcżę więcey zmázáć imienia świętobliwośći mojey ; y dowiedzą śię narody / żem Ja Pán / święty w Izráelu.
Oto przydźie y ſtánie śię to / mówi pánujący Pán / tegoż dniá / o którymem mówił.
Tedy wynijdą obywátele miaſt Izráelſkich / á zápáliwƺy ſpálą oręże y tárcże / y drzewcá / łuki y ſtrzáły / kije ręcżne y włócżnie / y będą z nich niećić ogień przez śiedm lat ;
A nie będą drew nośić z polá / áni ich rąbáć w láſách / ále z orężá ogień niećić będą á złupią tych / którzy ich łupili / y ſplundrują tych / którzy ich plundrowáli / mówi pánujący Pán.
Y ſtánie śię dniá onego / że dam Gogowi miejſce ná grób / tám w Izráelu / dolinę / którędy chodzą ná wſchód ſłońcá ku morzu / która zátká uſtá mimo idących ; y pogrzebią tám Gogá y wƺyſtkie zgráje jego / á będą ją zwáć doliną mnóſtwá Gogowego ;
Bo ich grześć będźie dom Izrelſki przez śiedm mieśięcy / áby ocżyśćili źiemię.
A ták pogrzebie ich wƺyſtek lud oney źiemi / y obróći śię im to w ſłáwę / w dźień / którego Ja będę uwielbiony / mówi pánujący Pán.
Y obiorą mężów uſtáwicżnych / którzyby śię przechodźili po oney źiemi / przechodźili / mówię / á chowáli tych / którzyby zoſtáli ná źiemi / áby ją wycżyśćili / á po wyśćiu śiedmiu mieśięcy ƺukáć pocżną.
A ći przechodząc chodźić będą po źiemi / á ujrzáwƺy kośći cżłowiecże poſtáwią przy nich znák / áby je pochowáli ći / co chowáją umárłych / w dolinie mnóſtwá Gogowego.
Owƺem / to mnóſtwo jego będźie ku ſłáwie miáſtu / gdy ocżyƺcżą onę źiemię.
A ty / ſynu cżłowiecży! ták mówi pánujący Pán : Rzecż ptáſtwu y wƺelkiej rzecży ſkrzydláſtey / y káżdej beſtyi polney : Zbierzćie śię / á przyjdźćie / zgromádźćie śię zewƺąd ná ofiárę moję / którą Ja wam ſpráwuję / ofiárę wielką ná górách Izráelſkiech / żebyśćie jedli mięſo / y pili krew.
Mięſo mocárzy jeść będźiećie / á krew kśiążąt źiemſkich pić będźiećie / báránów / báránków / y kozłów / y ćielców / którzy wƺyſcy potyli w Báſán.
Nájećie śię tłuſtośći do ſytośći á nápijećie śię krwi do upićiá z tey ofiáry mojey / którą wam nágotuję.
Y náſyćićie śię z ſtołu mego końmi / y jeźdźcámi / mocárzámi / y káżdym mężem wálecżnym / mówi pánujący Pán.
A ták objáwię chwałę moję miedzy národámi / y oglądáją wƺyſtkie narody ſąd mój / którym ucżynił / y rękę moję / którąm ná nie wyćiągnął ;
A dowie śię dom Izráelſki / żem Ja Pán / Bóg ich / od onego dniá y ná potem.
Poznáją też y narody / iż dla niepráwośći ſwojey záwiedźieni ſą do więźieniá dom Izráelſki / dla tego / iż wyſtąpili przećiwko mnie. Dlátegom też był zákrył oblicże ſwoje przed nimi / á podáłem ich w ręce nieprzyjáćiół ich / áby wƺyſcy od miecżá polegli.
Według niecżyſtośći ich / y według przewrotnośći ich obƺedłem śię z nimi / y zákryłem oblicże moje przed nimi.
Przetoż ták mówi pánujący Pán : Jużći przywrócę więźniów Jákóbowych ; á zmiłuję śię nád wƺyſtkim domem Izráelſkim ; y gorliwym będę dla imienia świętobliwośći mojey /
Gdy odnioſą poháńbienie ſwoje y wƺyſtko przeſtępſtwo ſwoje / którem wyſtąpili przećiwko mnie / gdy bezpiecżnie mieƺkáli w źiemi ſwojey / á nie był / ktoby ich trwożył ;
A gdy ich przywrócę z narodów / y zgromádzę ich z źiem nieprzyjáćiół ich / y będę poświęcony w nich przed ocżymá wielu narodów.
Tedy śię dowiedzą / żem Ja Pán / Bóg ich / gdy záwiódƺy ich do narodów záśię ich zgromádzę do źiemi ich / á nie zoſtáwię tám więcey żádnego z nich.
Y nie zákryję więcey oblicża mego przed nimi / gdyż wyleję duchá mojego ná dom Izráelſki / mówi pánujący Pán.
