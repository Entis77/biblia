Y ſtáło śię roku śiódmego / mieśiącá piątego / dzieśiątego dniá tegoż mieśiącá / przyƺli niektórzy z ſtárƺych Izráelſkich / áby śię rádzili Páná ; y uśiedli przedemną.
Tedy śię ſtáło ſłowo Páńſkie do mnie / mówąc :
Synu cżłowiecży! mów do ſtárƺych Izráelśich / á powiedz im : Ták mówi pánujący Pán : Izáli wy przychodzićie / ábyśćie śię mnie rádzili? Jáko żyję Já / że śię wy mnie nie rádzićie / mówi pánujący Pán ;
Izáli śię zá nich záſtáwiáć będzieƺ? Izáli śię zá nich záſtáwiáć będzieƺ / ſynu cżłowiecży? Oznájmij im rácżej obrzydliwośći ojców ich /
A rzecż do nich : Ták mówi pánujácy Pán : Tego dniá / któregom wybráł Izráelá / podnioſłem rękę moję náśieniu domu Jákóbowego / y dáłem śię im poznáć w ziemi Egipſkiej ; podnioſłem rękę moję dlá nich / mówiąc : Jám Pán / Bóg wáƺ.
Onego dniá podnioſłem rękę moję dlá nich / że ich wywiodę z ziemi Egipſkiej do ziemi / którąm im upátrzył / opływájącej mlekiem y miodem / którá jeſt ozdobą wƺyſtkich ziem.
Y rzekłem im : Káżdy z wáſ niech porzući obrzydliwośći ocżów ſwoich / á nie kálájćie śię plugáwemi báłwánámi egipſkiemi ; bom Já Pán / Bóg wáƺ.
Ale mi byli odpornymi / y nie chćieli mię ſłucháć ; żáden z nich obrzydliwośći ocżów ſwoich nie odrzućił / y plugáwych báłwánów egipſkich nie opóśćił ; prztożem rzekł : Wyleję gniew mój ná nich / á wypęłnie popędliwość moję nád nimi w pośrodku ziemi egipſkiej.
A wƺákżem ucżynił dlá imieniá mego / áby nie było zelżone przed ocżymá tych národów / między którymi oni byli / przed których ocżymá dáłem śię im poznáć / że ich chcę wywieść z ziemi egipſkiej.
A ták wywiodłem ich z ziemi egipſkiej / y przyprowádziłem ich ná puƺcżę ;
Y dáłem im uſtáwy moje / y ſądy moje podáłem im do wiádomośći ; które jeźliby cżłowiek záchowywáł / żyć w nich będzie.
Nádto y ſábáty moje dáłem im / áby były znákiem między mną y między nimi / áby wiedzieli / iżem Já jeſt Pán / który ich poświęcám.
Ale mi śię odpornym ſtáł dom Izráelſki ná puƺcży ; w uſtáwách moich nie chodzili / y ſądy moje odrzućili / które jeźliby cżłowiek cżynił / żyć w nich będzie. Sábáty też moje bárdzo zgwáłćili ; prztożem mówił / iż wleję popędliwość moję ná nich ná puƺcży / ábym ich wygubił.
Lecżem ucżynił dlá imieniá mego / áby nie było zelżone przed ocżymá tych národów / przed którychem ich ocżymá wywiódł.
A wƺákżem podnióſł rękę moję dlá nich ná tej puƺcży / że ich nie wprowádzę do ziemi / którąm im był dáł / opływájącej mlekiem y miodem / y którá jeſt ozdobą wƺyſtkich ziem ;
Poniewáż ſądy moje odrzućili / á w uſtáwách moich nie chodzili / y ſábáty moje pogwáłćili / á że ſerce ich zá plugáwemi báłwánámi chodzi.
Ale im przećie przepuśćiło oko moje / ták / żem ich nie wytráćił / y nie wygłádził do ƺcżętu ná puƺcży.
Y mówiłem do ſynów ich ná tej puƺcży : W uſtáwách ojców wáƺych nie chodźćie / y ſądów ich nie przeſtrzegájćie / áni śię plugáwemi báłwánámi ich kálájćie.
Jám Pán / Bóg wáƺ ; w uſtáwách moich chodźćie / á ſądów moich ſtrzeżćie / y cżyńćie ich ;
Sábáty też moje święććie / y będą znákiem między mną y między wámi / ábyśćie wiedzieli / żem Já Pán / Bóg wáƺ.
Lecż mi byli odpornymi ći ſynowie ; w uſtáwách moich nie chodzili / y ſądów moich nie przeſtrzegáli / áby je cżynili / ( które jeźliby cżynił cżłowiek pewnieby żył w nich ) y ſábáty moje pogwáłćili. Y rzekłem : Wyleję popędliwość moję ná nich / ábym wykonáł gniew ſwój ná nich ná tej puƺcży.
Alem odwróćił rękę moję / á ucżyniłem to dlá imieniá mego / áby nie było zelżone przed ocżymá tych národów / przed którychem ich ocżymá wywiódł.
Alem já podnióſł rękę moję dlá nich ná puƺcży / żem ich miáł rozproƺyć między pogán / y rozwiáć ich po ziemiách /
Przeto / że ſądów moich nie cżynili / y uſtáwy moje odrzućili / y ſábáty moje pogwáłćili / á zá plugáwemi báłwánámi ojców ſwoich ocży ſwe obróćili.
Dlátegoż y Jám im dáł uſtáwy nie dobre y ſądy / w których żyć nie będą ;
Y ſplugáwiłem ich z dárámi ich / gdy przewodzili przez ogień wƺelkie otwárzájące żywot / ábym ich ſpuſtoƺył / á żeby śię dowiedzieli / żem já Pán.
Prztoż mów do domu Izráelſkigo / ſynu cżłowiecży! á powiedz im : Ták mówi pánujący Pán : Jeƺcże y w tem lżyli mię ojcowie wáśi / dopuƺcżájąc śię przećiwko mnie przeſtępſtwá /
Że gdym ich wwiódł do ziemi / o którąm był podnióſł rękę moję / żem ją im dáć miáł / gdzie ujrzeli jáki págórek wyſoki / álbo jákie drzewo gáłęziſte / záráz tám ofiárowáli ofiáry ſwoje / y dáwáli tám dráźniące dáry ſwoje / támże kłádli y wdzięcżną wonność ſwoję / támże ſpráwowáli mokre ofiáry ſwoje.
A choćiáżem mówił do nich : Cóż to zá wyżyná / do której wy chádzáćie? Przećie ją zowią wyżyną áż do dniá tego.
Przetoż rzecż domowi Izráelſkiemu : Ták mówi pánujący Pán : Izáli śię wy drogámi ojców wáƺych káláć máćie / á z obrzydliwośćiámi ich nierząd płodzić?
Y máćież śię káláć przy wƺyſtkich plugáwych báłwánách wáƺych / przynoƺąc dáry wáƺe y przewodząc ſynów wáƺych przez ogień áż do dniá tego / á przećie odemnie rády ƺukáć / o domie Izráelſki? Jáko żyję Já / mówi pánujący Pán / że śię wy mnie więcej rádzić nie będziećie.
A to / co wſtępuje ná myśl wáƺę / nigdy śię nie ſtánie. Wy mówićie : Będziemy jáko inne národy / jáko pokoleniá innych ziem / drewnu y kámieniowi ſłużące ;
Jáko żyję Já / mówi pánujący Pán / że ręką możną y rámieniem wyćiągnionem / á w popędliwośći wylánej będę królowáł nád wámi ;
Y wywiodę wáſ z národów / á zgromádzę wáſ z ziem / do którycheśćie rozproƺeni / ręką możną / y rámieniem wyćiągnionem / y w popędliwośći wylánej ;
A prowádząc wáſ po puƺcży tych národów / tám śię z wámi twárzą w twárz ſądzić będę.
Jákom śię ſądził z ojcámi wáƺymi ná puƺcży ziemi Egipſkiej / ták śię z wámi ſądzić będę / mówi pánujący Pán ;
Y popędzę wáſ pod rózgą / ábym wáſ przywiódł do związku przymierzá.
Ale z wáſ wybiorę odpornych y wyſtępujących przećiwko mnie ; z ziemi pielgrzymſtwá ich wywiodę ich / wƺákże do ziemi Izráelſkiej nie wnijdą ; y poznáćie / żem Já Pán.
Wy tedy / o domie Izráelſki! ták mówi pánujący Pán : Idźćież / ſłużćie káżdy plugáwym báłwánom ſwoim y ná potem / poniewáż mnie nie ſłucháćie ; á imieniá mego świętego nie kálájćie więcej dárámi wáƺemi / y plugáwemi báłwánámi wáƺemi.
Bo ná świętej górze mojej / ná wyſokiej górze Izráeſkiej / mówi pánujący Pán / tám mi ſłużyć będzie wƺyſtek dom Izráeſki / ile ich będzie w onej ziemi ; tám ich łáſkáwie przyjmę / y tám śię pytáć będę o ofiárách wáƺych podnoƺonych / y o pierwiáſtkách dárów wáƺych ze wƺyſtkiemi świętymi rzecżámi wáƺemi.
Z wdzięcżną wonnośćią łáſkáwie wáſ przyjmę / gdy wáſ wywiodę z národów / á zgromádzę wáſ z onych ziem / do którycheśćie rozproƺeni byli / á ták poświęcony będę w wáſ przed ocżymá onych národów.
A poznáćie / żem Já Pán / gdy wáſ wprowádzę do ziemi Izráelſkiej / do ziemi onej / o którąm podnióſł rękę moję / że ją dám ojcom wáƺym ;
A wſpomnićie tám ná drogi wáƺe / y ná wƺyſtkie ſpráwy wáƺe / któremiśćie śię ſplugáwili / á obmierzniećie ſámi ſobie przed oblicżem wáƺem dlá wƺyſtkich złośći wáƺych / któreśćie cżynili.
Tám śię dowiećie / żem Já Pán / gdy wám to ucżynię dlá imieniá mego / nie według dróg wáƺych złych / áni według ſpráw wáƺych ſkáżonych / o domie Izráelſki! mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz twoję ku ſtronie południowej / á krop jáko roſą ná południe / y prorokuj przećiwko láſowi polá południowego /
A rzecż láſowi południowemu : Słucháj ſłowá Páńſkiego : Ták mówi pánujący Pán : Oto Já rozniecę w tobie ogień / który pożre w tobie wƺelkie drzewo zielone / y wƺelkie drzewo ſuche ; nie będzie ugáƺony płomień páłájący / y zgoreją w nim wƺyſtkie twárze od południá áż do północy.
Y ujrzy wƺelkie ćiáło / żem go Já Pán zápálił ; nie będzie ugáƺony.
Y rzekłem : Ach pánujący Pánie! Oni mówią o mnie : Ten tylko w przypowieśćiách mówi.
