Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! obróć twárz ſwoję ku Jeruzálemowi / á krop jáko roſą przećiwko miejſcom świętym / y prorokuj przećiwko ziemi Izráelſkiej /
A powiedz ziemi Izráelſkiej : Ták mówi pánujący Pán : Otom Já przećiwko tobie / y dobędę miecżá mego z pochew jego / y wytnę z ćiebie ſpráwiedliwego y niezbożnego.
Abym tedy wyćiął z ćiebie ſpráwiedliwego / y niezbożnego / przeto wynijdzie miecż mój z pochew ſwoich ná wƺelkie ćiáło od południá áż ku północy /
Y dowie śię wƺelkie ćiáło / żem Já Pán dobył miecżá mego z pochew jego ; nie náwróći śię więcej.
A ty / ſynu cżłowiecży! wzdycháj / jákobyś miáł biodro złámáne / á w gorzkośći wzdycháj przed ocżymá ich.
A gdyć rzeką : Dlácżego wzdycháƺ? Tedy odpowieƺ : Dlá wieśći / którá idzie / ná którą śię rozpłynie wƺelkie ſerce / y oſłábieją wƺelkie ręce / y śćiśniony będzie wƺelki duch / y wƺelkie koláno rozpłynie śię jáko wodá ; oto idzie / y ſtánie śię / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! prorokuj / á rzecż : Ták mówi Pán : Powiedz / miecż / miecż náoſtrzony jeſt y wypolerowány ;
Wyoſtrzony jeſt ná pomordowánie ku zábićiu náznácżonych / wypolerowány jeſt / áby śię lśnił. Izáli śię weſelić mámy? Gdyż rózgá ſyná mego káżde drzewo lekce wáży.
Dáłći go ná wypolerowánie / áby mógł być ujęty ręką / jeſtći wyoſtrzony ten miecż / jeſt y wypolerowány / áby dány był do ręki zábijájącego.
Wołáj á kwil / ſynu cżłowiecży! przeto / że ten miecż będzie przećiwko ludowi memu / ten też przećiwko wƺyſtkim kśiążętom Izráelſkim ; ſtráchy miecżá przyjdą ná lud mój ; przetoż śię w biodrę uderz.
Gdym ich káráł / cóż to pomogło? Y nie mámże rózgi wƺyſtko lekce powáżájącej ná nich wyćiągnąć? mówi pánujący Pán.
Ty tedy / ſynu cżłowiecży! prorokuj / á bij ręką w rękę ; bo powtóre y potrzećie miecż przyjdzie / miecż mordujących / ten miecż mordujących bez litośći / przenikájący áż do pokojów ich.
We wƺyſtkich brámách ich dáłem ſtrách miecżá / áby śię rozpłynęło ſerce / y upádków śię námnożyło. Ach! wypolerowány jeſt / áby śię błyƺcżáł / á wyoſtrzony / áby zábijáł.
Zbierz śię miecżu / udáj śię ná práwo y ná lewo / gdziekolwiek jeſt chęć twárzy twojej.
Boć y Já uderzę ręką moją w rękę moję / y uſpokoję rozgniewánie moje. Já Pán mówiłem to.
Wtem śię ſtáło ſłowo Páńſkie do mnie / mówiąc :
A ty / ſynu cżłowiecży! połóż przed ſobą dwie drogi / którędyby iść miáł miecż królá Bábilońſkiego ; z jednej ziemi niech wychodzą obiedwie / á ná rozdrożu obierz tę ku miáſtu / tę obierz.
Pokáż drogę / którąby miecż iść miáł / cżyli ku Rábbáćie ſynów Ammonowych / cżyli ku ziemi Judzkiej ná Jeruzálemſkie twierdze.
Abowiem ſtánie król Bábilońſki ná rozdrożu / ná pocżątku dwóch dróg / pytájąc śię wieƺcżby ; będzie polerowáł ſtrzáły / będzie śię rádził báłwánów / będzie pátrzył ná wątrobę.
Po práwej ręce jego wieƺcżbá ukáże Jeruzálem / áby ƺykowáł hetmánów / którzyby pobudzáli do mordowániá / á podnośili głoſ z okrzykiem / áby záſádzili tárány przećiwko brámom / áby uſypáli wáł / á urobili ƺáńce.
Y będą to mieli zá próżną wieƺcżbę przed ocżymá ſwemi ći / co śię obowiązáli przyśięgámi ; á to przywiedzie ná pámięć niepráwość ich / áby pojmáni byli.
Przetoż ták mówi pánujący Pán : Dlátego / że ná pámięć przywodzićie niepráwość ſwoję / á odkrywá śię przeſtępſtwo wáƺe / ták / że jáwne ſą grzechy wáƺe we wƺyſtkich ſpráwách wáƺych / przeto / mówię / że ná pámięć przychodzićie / tą ręką pojmáni będziećie.
A ty / niecżyſty bezbożniku / kśięćiu Izráelſki! którego dzień przychodzi / gdy niepráwość ſkońcżoná będzie :
Ták mówi pánujący Pán : Zdejm tę cżápkę / á zrzuć tę koronę / którá już nigdy tákowá nie będzie ; tego / który w poniżenie przyƺedł / wywyżƺę / á wywyżƺonego poniżę.
W niwecż / w niwecż / w niwecż ją obrócę / cżego pierwej nie bywáło / áż przyjdzie ten / co do niej má práwo / którem mu dáł.
Ale ty / ſynu cżłowiecży! prorokuj / á mów : Ták mówi pánujący Pán o ſynách Ammonowych y o háńbie ich ; rzecż mówię : Miecż / miecż dobyty jeſt / ku zábijániu wypoleroáwny jeſt / áby wytrácáł wƺyſtko / y áby śię błyſkáł.
A choćiáż ći opowiádáją márność / y wróżąć kłámſtwo / áby ćię przyłożyli do ƺyi niezbożników pobitych / których dzień przychodzi / gdy niepráwość / ſkońcżoná będzie /
Schowáj jednák miecż do pochew jego ; ná miejſcu / ná któremeś ſpłodzoná / w ziemi mieƺkániá twego / ſądzić ćię będę.
Y wyleję ná ćię rozgniewánie moje ; ogniem popędliwośći mojej ná ćię dmucháć będę / y podám ćię w ręce ludzi zuchwáłych y przemyślnych ná wytrácenie.
Ogniowi potráwą będzieƺ / krew twojá będzie w pośrodku ziemi / nie będzieƺ więcej wſpomináná ; bo Já Pán mówiłem to.
