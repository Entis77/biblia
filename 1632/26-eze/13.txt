Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! prorokuj przećiw prorokom Izráelſkim / którzy prorokują / á rzecż prorokującym z ſercá ſwego : Słuchájćie ſłowá Páńſkiego.
Ták mówi pánujący Pán : Biádá prorokom głupim / którzy idą zá duchem ſwoim / choć nic nie widzieli!
Izráelu! Prorocy twoi ſą jáko liƺki ná puƺcży.
Nie wſtępujćie ná przerwáne miejſcá / áni grodzćie płotu około domu Izráelſkiego / żeby śię mógł oſtáć w bitwie w dzień Páńſki.
Widzą márność y wieƺcżbę kłámliwą ; powiádáją : Pán mówi / choć ich Pán nie poſłáł ; y ćieƺą lud / áby tylko utwierdzili ſłowo ſwe.
Izáli widzeniá márnego nie widzićie / á wieƺcżby kłámliwej nie opowiádáćie? Y mówićie : Pán mówił / choćiáżem Já nie mówił.
Przetoż ták mówi pánujący Pán : Poniewáż mówićie márność / á widzićie kłámſtwo / przetoż oto Já jeſtem przećiwko wám / mówi pánujący Pán.
Bo ręká mojá będzie przećiwko prorokom / którzy widzą márność / á opowiádáją kłámſtwo ; w zgromádzeniu ludu mego nie będą / á w pocżet domu Izráelſkiego nie będą wpiſáni / y do ziemi Izráelſkiej nie wnijdą ; á dowiećie śię / żem Já pánujący Pán.
Przeto / przeto mówię / że w błąd wprowádzili lud mój / mówiąc : Pokój / choć nie było pokoju ; jeden záiſte zbudowáł śćiánę gliniáną / drudzy ją tynkowáli wápnem niecżynionem.
Mówże do tych / którzy ją tynkują wápnem niecżynionem : Upádnie to / przyjdzie deƺcż gwáłtowny / á wy / kámienie grádowe! ſpádniećie / y wiátr wichrowáty rozwáli ją.
A oto gdy upádnie oná śćiáná / izáli wám nie rzeką : Gdzież jeſt ono tynkowánie / któremeśćie tynkowáli?
Prztoż ták mówi pánujący Pán! Rozwálę ją / mówię / wiátrem wichrowátym w zápálcżywośći mojej / y deƺcż gwáłtowny w popędliwośći mojej przyjdzie / á kámienie grádowe w rozgniewániu mojem ná zniƺcżenie jej.
Bo obálę tę śćiánę / którąśćie potynkowáli wápnem niecżynionem / á zrównám ją z ziemią / ták / że odkryty będzie grunt jej / y upádnie / y ſkáżeni będziećie w pośrodku jej / y dowiećie śię / żem Já Pán.
A gdy wykonám popędliwość moję nád tą śćiáną / y nád tymi / którzy ją tynkowáli wápnem niecżynionem / rzekę do wáſ : Niemáƺ już onej śćiány / niemáƺ y tych / którzy ją tynkowáli /
To jeſt / proroków Izráelſkich / którzy prorokują o Jeruzálemie / y ogłáƺáją mu widzenie pokoju / choć niemáƺ pokoju / mówi pánujący Pán.
Ale ty / ſynu cżłowiecży! obróć twárz twoję przećiwko córkom ludu ſwego / które prorokują z ſercá ſwego / á prorokuj przećiwko nim /
Y rzecż : Ták mówi pánujący Pán : Biádá tym / które ƺyją wezgłówká pod wƺelkie łokćie rąk ludu mojego / á cżynią duchny ná głowy wƺelkiego wzroſtu / áby łowiły duƺe! izáli łowić máćie duƺe ludu mego / ábyśćie śię żywić mogły?
Bo mię podáwáćie w lekkość u ludu mego dlá gárśći jęcżmieniá / y dlá kęſá chlebá / zábijájąc duƺe / które nie umrą / á ożywiájąc duƺe / które żywe nie będą / kłámiąc ludowi memu / którzy ſłucháją kłámſtwá.
Dlátego / ták mówi pánujący Pán : Oto Já będę przećiwko wezgłówkom wáƺym / któremi wy tám duƺe łowićie / ábyśćie je zwiodły ; bo je ſtárgnę z rámion wáƺych / á wypuƺcżę duƺe / które wy łowićie / ábyśćie je zwiodły ;
Y rozerwę duchny wáƺe / á wybáwię lud mój z ręki wáƺej / ábyśćie ich więcej nie mogły łowić ręką ſwoją : á dowiećie śię / żem Já Pán.
Przeto / że záſmucáćie ſerce ſpráwiedliwego kłámſtwy / choćiáżem go Já nie záſmućił / á zmácniáćie ręce niezbożnego / áby śię nie odwróćił od złej drogi ſwojej / ożywiájąc go ;
Przetoż nie będziećie więcej widywáć márnośći / áni wieƺcżby więcej prorokowáć będziećie ; bo wyrwę lud mój z ręki wáƺej / á dowiećie śię / żem Já Pán.
