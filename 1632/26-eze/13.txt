Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! prorokuj przećiw prorokom Izráelſkim / którzy prorokują / á rzecż prorokującym z ſercá ſwego : Słuchájćie ſłowá Páńſkiego.
Ták mówi pánujący Pán : Biádá prorokom głupim / którzy idą zá duchem ſwoim / choć nic nie widźieli!
Izráelu! Prorocy twoi ſą jáko liƺki ná puƺcży.
Nie wſtępujećie ná przerwáne miejſcá / áni grodzićie płotu około domu Izráelſkiego / żeby śię mógł oſtáć w bitwie w dźień Páńſki.
Widzą márność y wieƺcżbę kłámliwą ; powiádáją : Pán mówi / choć ich Pán nie poſłał ; y ćieƺą lud / áby tylko utwierdźili ſłowo ſwe.
Izali widzeniá márnego nie widźićie / á wieƺcżby kłámliwej nie opowiádáćie? Y mówićie : Pán mówił / choćiáżem Ja nie mówił.
Przetoż ták mówi pánujący Pán : Ponieważ mówićie márność / á widźićie kłámſtwo / przetoż oto Ja jeſtem przećiwko wam / mówi pánujący Pán.
Bo ręká mojá będźie przećiwko prorokom / którzy widzą márność / á opowiádáją kłámſtwo ; w zgromádzeniu ludu mego nie będą / á w pocżet domu Izráelſkiego nie będą wpiſáni / y do źiemi Izráelſkiej nie wnijdą ; á dowiećie śię / żem Ja pánujący Pán.
Przeto / przeto mówię / że w błąd wprowádźili lud mój / mówiąc : Pokój / choć nie było pokoju ; jeden zájiſte zbudował śćiánę gliniáną / drudzy ją tynkowáli wápnem niecżynionem.
Mówże do tych / którzy ją tynkują wápnem niecżynionem : Upádnie to / przydźie deƺcż gwáłtowny / á wy / kámienie grádowe! ſpádniećie / y wiátr wichrowáty rozwáli ją.
A oto gdy upádnie oná śćiáná / izali wam nie rzeką : Gdźież jeſt ono tynkowánie / któremeśćie tynkowáli?
Przetoż ták mówi pánujący Pán! Rozwálę ją / mówię / wiátrem wichrowátym w zápálcżywośći mojey / y deƺcż gwáłtowny w popędliwośći mojey przydźie / á kámienie grádowe w rozgniewániu mojim ná zniƺcżenie jey.
Bo obálę tę śćiánę / którąśćie potynkowáli wápnem niecżynionem / á zrównam ją z źiemią / ták / że odkryty będźie grunt jey / y upádnie / y ſkáżeni będźiećie w pośrzodku jey / y dowiećie śię / żem Ja Pán.
A gdy wykonam popędliwość moję nád tą śćiáną / y nád tymi / którzy ją tynkowáli wápnem niecżynionem / rzekę do was : Niemáƺ już oney śćiány / niemáƺ y tych / którzy ją tynkowáli /
To jeſt / proroków Izráelſkich / którzy prorokują o Jeruzálemie / y ogłáƺáją mu widzenie pokoju / choć niemáƺ pokoju / mówi pánujący Pán.
Ale ty / ſynu cżłowiecży! obróć twárz twoję przećiwko córkom ludu ſwego / które prorokują z ſercá ſwego / á prorokuj przećiwko nim /
Y rzecż : Ták mówi pánujący Pán : Biádá tym / które ƺyją wezgłówká pod wƺelkie łokćie rąk ludu mojego / á cżynią duchny ná głowy wƺelkiego wzroſtu / áby łowiły duƺe! izali łowić máćie duƺe ludu mego / ábyśćie śię żywić mogły?
Bo mię podáwáćie w lekkość u ludu mego dla gárśći jęcżmieniá / y dla kęſá chlebá / zábijájąc duƺe / które nie umrą / á ożywiájąc duƺe / które żywe nie będą / kłámiąc ludowi memu / którzy ſłucháją kłámſtwá.
Dla tego / ták mówi pánujący Pán : Oto Ja będę przećiwko wezgłówkom wáƺym / któremi wy tám duƺe łowićie / ábyśćie je zwiodły ; bo je ſtárgnę z rámion wáƺych / á wypuƺcżę duƺe / które wy łowićie / ábyśćie je zwiodły ;
Y rozerwę duchny wáƺe / á wybáwię lud mój z ręki wáƺej / ábyśćie ich więcey nie mogły łowić ręką ſwoją : á dowiećie śię / żem Ja Pán.
Przeto / że záſmucáćie ſerce ſpráwiedliwego kłámſtwy / choćiáżem go Ja nie záſmućił / á zmácniáćie ręce niezbożnego / áby śię nie odwróćił od złej drogi ſwojey / ożywiájąc go ;
Przetoż nie będźiećie więcey widywáć márnośći / áni wieƺcżby więcey prorokowáć będźiećie ; bo wyrwę lud mój z ręki wáƺej / á dowiećie śię / żem Ja Pán.
