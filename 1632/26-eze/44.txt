Tedy mię záś przywiódł drogą ku brámie świątnicy zewnętrznej / którá pátrzy ná wſchód ſłońcá / á tá byłá zámknioná.
Y rzekł do mnie Pán : Tá brámá zámknioná będźie / nie będźie otworzoná / á żáden nie wnijdźie przez nią ; bo Pán / Bóg Izráelſki / przeƺedł przez nią / przetoż będźie zámknioná.
Kśiążęcá jeſt ; kśiążę ſám będźie śiádáł w niej / áby jádł chleb przed oblicżem Páńſkim ; drogą przyśionku tej brámy wchodźić / á drogą jej wychodźić będźie.
Y przywiódł mię drogą brámy północnej ku przedniej ſtronie domu ; y ujrzáłem / á oto nápełniłá chwáłá Páńſká dom Páńſki / y upádłem ná oblicże ſwoje.
A Pán rzekł do mnie : Synu cżłowiecży! uwáżáj to / á obácż ocżymá twemi / y uƺymá twemi ſłucháj wƺyſtkiego / co Já mówię tobie o wƺyſtkich uſtáwách domu Páńſkiego / y o wƺyſtkich práwách jego ; uwáżáj / mówię / ábyś obácżył wejśćie w dom / y wƺyſtkie wyjśćiá z świątnicy.
A rzecż odpornemu domowi Izráelſkiemu : Ták mówi pánujący Pán : Doſyć miejćie ná wƺyſtkich obrzydliwośćiách wáƺych / o domie Izráelſki!
Żeśćie tu przywodźili cudzoźiemców nieobrzezánych ná ſercu / y nieobrzezánych ná ćiele / żeby bywáli w świątnicy mojej á ſplugáwili dom mój ; wyśćie też ofiárowáli chleb mój / tłuſtość / y krew / gdy oni łámáli przymierze moje mimo wƺyſtkie obrzydliwośći wáƺe ;
A nie trzymáliśćie ſtráży nád świętymi rzecżámi mojemi / áleśćie poſtáwili ſtróżów ná ſtráży mojej w świątnicy mojej miáſto śiebie.
Ták mówi pánujący Pán : Żáden cudzoźiemiec nieobrzezány ná ſercu y nieobrzezány ná ćiele nie wnijdźie do świątnicy mojej ze wƺyſtkich cudzoźiemców / którzy ſą między ſynámi Izráelſkimi.
Tákże y Lewitowie / którzy śię oddáláli odemnie / gdy błądźił Izráel / którzy śię obłądźili odemnie zá plugáwemi báłwánámi ſwemi / ći ponioſą niepráwość ſwoję.
Bo będą w świątnicy mojej zá ſług w powinnośćiách przy brámách domu / y zá ſług przy domu ; onić będą bić ofiáry ná cáłopálenie / y ofiáry zá lud / y oni ſtáć będą przed oblicżem ich / áby im ſłużyli.
Dlátego / że im ſłużyli przed plugáwemi báłwánámi ich / á byli domowi Izráelſkiemi przycżyną upádku w nieprwość / przetożem podnióſł rękę moję dlá nich / mówi pánujący Pán / że ponioſą niepráwość ſwoję.
A nie przyſtąpią do mnie / áby mi ſpráwowáli urząd kápłáńſki / áni przyſtąpią do jákich świętych rzecży moich / álbo do nájświętƺych / ále ponioſą poháńbienie ſwoje y obrzydliwośći ſwoje / które cżynili.
Przetoż ich poſtánowię zá ſtróżów obrzędów domu ná káżdą poſługę jego / y ná wƺyſtko / co w nim będźie ſpráwowáne.
Lecż kápłáni z Lewitów / ſynowie Sádokowi / którzy trzymáli ſtráż nád świątnicą moją / gdy śię obłądźili ſynowie Izráelſcy odemnie / ći przſtąpią do mnie / áby mi ſłużyli / y ſtáną przed twárzą moją / áby mi ofiárowáli tłuſtość y krew / mówi pánujący Pán.
Ci wchodźić będą do świątnicy mojej / ći też przyſtąpią do ſtołu mego / áby mi ſłużyli y ſtráż moję trzymáli.
A gdy będą mieli wchodźić do brámy śieni wewnętrznej / tedy śię obleką w ƺáty lniáne / á nie wezmą ná śię nic wełniánego / gdy ſłużyć będą w brámách śieni wnętrznej / y wewnątrz.
Cżápki lniáne będą mieli ná głowie ſwojej / y ubiory płóćienne będą ná biodrách ich / á nie będą śię przepáſywáć nicżem / coby pot cżyniło.
A gdy wychodźić będą do śieni zewnętrznej / do śieni mówię zewnętrznej do ludu / zewleką ƺáty ſwe / w których ſłużyli / á położą je w komorách świątnicy / y obloką śię w inne ƺáty / á nie będą poświęcáli ludu ƺátámi ſwemi.
Głowy też ſwojej nie będą golić / áni włoſów zápuƺcżáć / ále je równo przyſtrzygáć będą ná głowách ſwoich.
Winá też nie będźie pił żáden kápłán / gdy będźie miáł wchodźić do śieni wnętrznej.
Wdowy też y porzuconej nie będą ſobie pojmowáć zá żony ; ále pánny z náśieniá domu Izráelſkiego / álbo wdowę / któráby pozoſtáłá wdową po kápłánie ; pojmowáć będą.
A ludu mego ucżyć będą różnośći między rzecżą świętą y nieświętą / tákże między niecżyſtem y cżyſtem niech ich ucżą różnośći /
A gdy będźie jáki ſpór / oni śię niech ſtáwią do rozſądzániá / á według ſądów moich rozſądzą go ; práw też moich y uſtáw moich we wƺyſtkie urocżyſte świętá moje ſtrzedz będą / á ſábáty moje święćić będą.
Do umárłego też cżłowieká nie wnijdźie kápłán / áby śię nie zmázáł ; chybá do ojcá y do mátki ; y do ſyná / y do córki / y do brátá / y do śioſtry / któráby jeƺcże nie byłá zá mężem ; przy tych może śię zmázáć.
A po ocżyƺcżeniu jego / ( śiedm dni nálicżą mu. )
W ten dźień / którego wnijdźie do świątnicy / do śieni wnętrznej / áby ſłużył w świątnicy / ucżyni ofiárę zá grzech ſwój / mówi pánujący Pán.
A miáſto dźiedźictwá ich Jám jeſt dźiedźictwem ich ; przetoż ośiádłośći nie dáwájćie im w Izráelu / Jám jeſt ośiádłośćią ich.
Ofiárę śniedną y ofiárę zá grzech y zá wyſtępek / to oni jeść będą ; tákże wƺelká rzecż ofiárowáná Bogu w Izráelu ich będźie.
Tákże też nájprzedniejƺe rzecży wƺyſtkich pierwoćin ze wƺyſtkiego / y káżdá ofiárá podnoƺeniá wƺyſtkich rzecży ze wƺyſtkich ofiár wáƺych kápłáńſká będźie ; pierwiáſtki też ćiáſt wáƺych dáćie kápłánowi / áby włożył błogoſłáwieńſtwo ná dom twój.
Żádnego śćierwu / y rozƺárpánego od zwierzá áni z ptáſtwá áni z bydląt kápłáni jeść nie będą.
