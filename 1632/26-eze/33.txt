Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! mów do ſynów ludu twego / á rzecż do nich : Gdy przywiodę miecż ná którą źiemię / jeſliże lud oney źiemi weźmie mężá jednego z gránic ſwoich / á poſtánowi go ſobie zá ſtróżá /
A on widząc miecż przychodzący ná onę źiemię / zátrąbiłby w trąbę y przeſtrzegłby lud /
A któryby ſłyƺáł głoſ trąby / y nie dbáłby ná przeſtrogę / á wtem przyƺedłƺy miecż / zgłádźiłby go : krew jego będźie ná głowie jego ;
Bo głoſ trąby ſłyƺáł / wƺákże nie dbáł ná przeſtrogę / dlátego krew jego ná nim będźie ; być był przyjął przeſtrogę / záchowáłby był duƺę ſwoję.
Ale jeſliby ſtróż ujrzáł miecż przychodzący / á nie zátrąbiłby w trąbę / á luduby nie przeſtrzegł / y przyƺedłby miecż / y znióſłby którego z nich / tákowy będźie w niepráwośći ſwojey záchwycony ; ále krwi jego z ręki onego ſtróżá ƺukáć będę.
Ciebieć / ſynu cżłowiecży! ćiebiem poſtánowił ſtróżem domu Izráelſkiego / ábyś ſłyƺąc ſłowo z uſt moich / przeſtrzegł ich odemnie.
Gdybym Ja tedy rzekł niezbożnemu : Niezbożniku! śmierćią / umrzeƺ / á tybyś mu tego nie powiedźiáł / przeſtrzegájąc niezbożniká od drogi jego ; tenći niezbożnik dla niepráwośći ſwojey umrze ; ále krwi jego z ręki twojey ƺukáć będę.
Ale jeſlibyś ty przeſtrzegł niezbożnego od drogi jego / áby śię od niej odwróćił / wƺákże nie odwróćiłby śię od drogi ſwojey / onći dla niepráwośći ſwojey umrze ; ále ty duƺę ſwoję wybáwiƺ.
A ták ty / ſynu cżłowiecży! mów do domu Izráelſkiego : Ták powiádáćie mówiąc : Przeto / że wyſtępki náƺe y grzechy náƺe ſą ná nas / ták / że w nich ſchniemy / y jákożbyśmy żyć mogli?
Rzecżże tedy do nich : Jáko żyję Ja / mówi pánujący Pán : Nie chcę śmierći niepobożnego / ále áby śię odwróćił niepobożny od drogi ſojey / á żył. Odwróććież śię / odwróććież śię od złych dróg ſwoich / przecżże máćie umrzeć / o domie Izráelſki!
Ty tedy / ſynu cżłowiecży! mów do ſynów ludu ſwego : Spráwiedliwość ſpráwiedliwego nie wybáwi go w dźień przeſtępſtwá jego / á niezbożny nie upádnie w ſwojey niezbożnośći w dźień / któregoby śię odwróćił od niezbożnośći ſwojey ; tákże ſpráwiedliwy nie będźie mógł żyć dla ſpráwiedliwośći ſwojey w dźień / któregoby zgrzeƺył.
Jeſlibym záś rzekł ſpráwiedliwemu : Pewnie żyć będźieƺ / á onby ufájąc ſpráwiedliwośći ſwojey cżynił niepráwość / żádná ſpráwiedliwość jego nie przyjdźie ná pámięć ; ále dla tey niepráwośći ſwojey / którą cżynił / umrze.
Záśię / rzekęli niepobożnemu : Śmierćią umrzeƺ / á onby śię odwróćił od grzechu ſwego / y cżyniłby ſąd y ſpráwiedliwość /
Wróćiłby niezbożny záſtáw / á co wydárł / oddáłliby / y chodźiłby w uſtáwách żywotá / nie cżyniąc niepráwośći / pewnie żyć będźie / á nie umrze.
Żádne grzechy jego / któremi grzeƺył / nie będą mu wſpomináne ; ſąd y ſpráwiedliwość cżynił / pewnie żyć będźie.
A wżdy mówią ſynowie ludu twego : Nie práwá jeſt drogá Páńſká / choć onych ſámych drogá nie jeſt práwá.
Gdyby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojey / á cżyniłby niepráwość / umrze dla niej ;
Ale gdyby śię odwróćił niezbożny od niezbożnośći ſwojey / á cżyniłby ſąd y ſpráwiedliwość / dlátego żyć będźie.
A przećie mówićie : Nie práwá jeſt drogá Páńſká ; káżdego z was według drogi jego ſądźić będę / o domie Izráelſki!
Y ſtáło śię dwunáſtego roku / dźieśiątego mieśiącá / piątego dniá tegoż mieśiącá od záprowádzeniá náƺego / że przyƺedł do mnie jeden / który uƺedł z Jeruzálemu / mówiąc : Dobyto miáſtá.
A ręká Páńſká byłá przy mnie w wiecżór przedtem / niż przyƺedł ten / który ućiekł / y otworzyłá uſtá moje / áż do mnie ráno przyƺedł ; otworzyłá mówię uſtá moje / ábym niemym dálej nie był.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! Obywátele tych ſpuſtoƺonych miejſc w źiemi Izráelſkiej powiádáją / mówiąc : Abráhám ſám jeden był / á wżdy pośiádł tę źiemię ; ále nas jeſt wiele / námći dáná jeſt tá źiemiá w ośiádłość.
Dlátegoż mów do nich : Ták mówi pánujący Pán : Ze krwią jádáćie / y ocży ſwe podnośićie do plugáwych báłwánów ſwoich / y krew wylewáćie / á chćielibyśćie tę źiemię pośiąść?
Stoićie ná miecżu wáƺym / cżynićie obrzydliwość / á káżdy żonę bliźniego ſwego plugáwi ; izáli źiemię pośiądźiećie?
Ták rzecż do nich : Ták mówi pánujący Pán : Jáko żyję Ja / że ći / którzy ſą ná miejſcách ſpuſtoƺonych / od miecżá polegną ; á kto jeſt ná polu / tego podám beſtyjom ná pożárćie ; á którzy ſą ná zámkách álbo w jáſkiniách / morem pomrą ;
Y podám źiemię ná wielkie ſpuſtoƺenie / y uſtánie pychá mocy jey ; y ſpuſtoƺeją góry Izráelſkie / á nie będźie / ktoby po nich chodźił.
Y dowiedzą śię / żem Ja Pán / gdy podám źiemię ich ná wielkie ſpuſtoƺenie dla wƺyſtkich obrzydliwośći ich / które cżynili.
Ale ty / ſynu cżłowiecży! ſłucháj. Synowie ludu twojego cżęſto mówią o tobie około śćián y we drzwiách domów / y mówi jeden do drugiego / y káżdy do bliźniego ſwego / mówiąc : Pójdźćie / á poſłuchájćie / co zá ſłowo od Páná wyƺło.
Y ſchodzą śię do ćiebie / ták / jáko śię ſchodźi lud / y śiádáją przed oblicżem twojim / jáko lud mój / y ſłucháją ſłów twoich / ále ich nie cżynią ; á choć je ſobie uſty ſwemi ſmákują / wƺákże zá ƺkárádnym zyſkiem ſwoim ſerce ich chodźi.
A oto tyś im jeſt jáko pieśń wdźięcżná pięknego głoſu / y dobrze umiejętnego śpiewáká ; ſłuchájąc w práwdźie ſłów twoich / ále ich nie cżynią.
Lecż gdy to przyjdźie / ( jákoż oto przychodźi ) dopiero śię dowiedzą / że prorok był w pośrodku nich.
