Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! mów do ſynów ludu twego / á rzecż do nich : Gdy przywiodę miecż ná którą ziemię / jeźliże lud onej ziemi weźmie mężá jednego z gránic ſwoich / á poſtánowi go ſobie zá ſtróżá /
A on widząc miecż przychodzący ná onę ziemię / zátrąbiłby w trąbę y przeſtrzegłby lud /
A któryby ſłyƺáł głoſ trąby / y nie dbáłby ná przeſtrogę / á wtem przyƺedłƺy miecż / zgłádziłby go : krew jego będzie ná głowie jego ;
Bo głoſ trąby ſłyƺáł / wƺákże nie dbáł ná przeſtrogę / dlátego krew jego ná nim będzie ; być był przyjął przeſtrogę / záchowáłby był duƺę ſwoję.
Ale jeźliby ſtróż ujrzáł miecż przychodzący / á nie zátrąbiłby w trąbę / á luduby nie przeſtrzegł / y przyƺedłby miecż / y znióſłby którego z nich / tákowy będzie w niepráwośći ſwojej záchwycony ; ále krwi jego z ręki onego ſtróżá ƺukáć będę.
Ciebieć / ſynu cżłowiecży! ćiebiem poſtánowił ſtróżem domu Izráelſkiego / ábyś ſłyƺąc ſłowo z uſt moich / przeſtrzegł ich odemnie.
Gdybym Já tedy rzekł niezbożnemu : Niezbożniku! śmierćią / umrzeƺ / á tybyś mu tego nie powiedziáł / przeſtrzegájąc niezbożniká od drogi jego ; tenći niezbożnik dlá niepráwośći ſwojej umrze ; ále krwi jego z ręki twojej ƺukáć będę.
Ale jeźlibyś ty przeſtrzegł niezbożnego od drogi jego / áby śię od niej odwróćił / wƺákże nie odwróćiłby śię od drogi ſwojej / onći dlá niepráwośći ſwojej umrze ; ále ty duƺę ſwoję wybáwiƺ.
A ták ty / ſynu cżłowiecży! mów do domu Izráelſkiego : Ták powiádáćie mówiąc : Przeto / że wyſtępki náƺe y grzechy náƺe ſą ná náſ / ták / że w nich ſchniemy / y jákożbyśmy żyć mogli?
Rzecżże tedy do nich : Jáko żyję Já / mówi pánujący Pán : Nie chcę śmierći niepobożnego / ále áby śię odwróćił niepobożny od drogi ſojej / á żył. Odwróććież śię / odwróććież śię od złych dróg ſwoich / przecżże máćie umrzeć / o domie Izráelſki!
Ty tedy / ſynu cżłowiecży! mów do ſynów ludu ſwego : Spráwiedliwość ſpráwiedliwego nie wybáwi go w dzień przeſtępſtwá jego / á niezbożny nie upádnie w ſwojej niezbożnośći w dzień / któregoby śię odwróćił od niezbożnośći ſwojej ; tákże ſpráwiedliwy nie będzie mógł żyć dlá ſpráwiedliwośći ſwojej w dzień / któregoby zgrzeƺył.
Jeźlibym záś rzekł ſpráwiedliwemu : Pewnie żyć będzieƺ / á onby ufájąc ſpráwiedliwośći ſwojej cżynił niepráwość / żádná ſpráwiedliwość jego nie przyjdzie ná pámięć ; ále dlá tej niepráwośći ſwojej / którą cżynił / umrze.
Záśię / rzekęli niepobożnemu : Śmierćią umrzeƺ / á onby śię odwróćił od grzechu ſwego / y cżyniłby ſąd y ſpráwiedliwość /
Wróćiłby niezbożny záſtáw / á co wydárł / oddáłliby / y chodziłby w uſtáwách żywotá / nie cżyniąc niepráwośći / pewnie żyć będzie / á nie umrze.
Żádne grzechy jego / któremi grzeƺył / nie będą mu wſpomináne ; ſąd y ſpráwiedliwość cżynił / pewnie żyć będzie.
A wżdy mówią ſynowie ludu twego : Nie práwá jeſt drogá Páńſká / choć onych ſámych drogá nie jeſt práwá.
Gdyby śię odwróćił ſpráwiedliwy od ſpráwiedliwośći ſwojej / á cżyniłby niepráwość / umrze dlá niej ;
Ale gdyby śię odwróćił niezbożny od niezbożnośći ſwojej / á cżyniłby ſąd y ſpráwiedliwość / dlátego żyć będzie.
A przećie mówićie : Nie práwá jeſt drogá Páńſká ; káżdego z wáſ według drogi jego ſądzić będę / o domie Izráelſki!
Y ſtáło śię dwunáſtego roku / dzieśiątego mieśiącá / piątego dniá tegoż mieśiącá od záprowádzeniá náƺego / że przyƺedł do mnie jeden / który uƺedł z Jeruzálemu / mówiąc : Dobyto miáſtá.
A ręká Páńſká byłá przy mnie w wiecżór przedtem / niż przyƺedł ten / który ućiekł / y otworzyłá uſtá moje / áż do mnie ráno przyƺedł ; otworzyłá mówię uſtá moje / ábym niemym dálej nie był.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Synu cżłowiecży! Obywátele tych ſpuſtoƺonych miejſc w ziemi Izráelſkiej powiádáją / mówiąc : Abráhám ſám jeden był / á wżdy pośiádł tę ziemię ; ále náſ jeſt wiele / námći dáná jeſt tá ziemiá w ośiádłość.
Dlátegoż mów do nich : Ták mówi pánujący Pán : Ze krwią jádáćie / y ocży ſwe podnośićie do plugáwych báłwánów ſwoich / y krew wylewáćie / á chćielibyśćie tę ziemię pośiąść?
Stoićie ná miecżu wáƺym / cżynićie obrzydliwość / á káżdy żonę bliźniego ſwego plugáwi ; izáli ziemię pośiądziećie?
Ták rzecż do nich : Ták mówi pánujący Pán : Jáko żyję Já / że ći / którzy ſą ná miejſcách ſpuſtoƺonych / od miecżá polegną ; á kto jeſt ná polu / tego podám beſtyjom ná pożárćie ; á którzy ſą ná zámkách álbo w jáſkiniách / morem pomrą ;
Y podám ziemię ná wielkie ſpuſtoƺenie / y uſtánie pychá mocy jej ; y ſpuſtoƺeją góry Izráelſkie / á nie będzie / ktoby po nich chodził.
Y dowiedzą śię / żem Já Pán / gdy podám ziemię ich ná wielkie ſpuſtoƺenie dlá wƺyſtkich obrzydliwośći ich / które cżynili.
Ale ty / ſynu cżłowiecży! ſłucháj. Synowie ludu twojego cżęſto mówią o tobie około śćián y we drzwiách domów / y mówi jeden do drugiego / y káżdy do bliźniego ſwego / mówiąc : Pójdźćie / á poſłuchájćie / co zá ſłowo od Páná wyƺło.
Y ſchodzą śię do ćiebie / ták / jáko śię ſchodzi lud / y śiádáją przed oblicżem twojim / jáko lud mój / y ſłucháją ſłów twoich / ále ich nie cżynią ; á choć je ſobie uſty ſwemi ſmákują / wƺákże zá ƺkárádnym zyſkiem ſwoim ſerce ich chodzi.
A oto tyś im jeſt jáko pieśń wdzięcżná pięknego głoſu / y dobrze umiejętnego śpiewáká ; ſłuchájąc w práwdzie ſłów twoich / ále ich nie cżynią.
Lecż gdy to przyjdzie / ( jákoż oto przychodzi ) dopiero śię dowiedzą / że prorok był w pośrodku nich.
