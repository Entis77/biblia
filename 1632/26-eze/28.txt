Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! mów kśiążęćiu Tyrſkiemu : Ták mówi pánujący Pán : Dla tego / iż śię podnioſło ſerce twoje / á mówiƺ : Jam jeſt Bóg / śiedzę w pośród morzá ná ſtolicy Boſkiej / gdyżeś ty cżłowiek / á nie Bóg / choć ſerce twoje ſtáwiáƺ jáko ſerce Boże ;
Otoś mędrƺym nád Dánijelá / żádná tájemnicá nie jeſt zákryta przed tobą /
Mądrośćią twoją y roztropnośćią twoją názbieráłeś ſobie bogáctw / y nábyłeś złotá y ſrebrá do ſkárbów twojich ;
Wielkośćią mądrośći twojey w kupiectwie twojim rozmnożyłeś bogáctwá twoje ; á ták podnioſło śię ſerce twoje dla bogáctw twojich.
Przetoż ták mówi pánujący Pán : Ponieważ ſtáwiáƺ ſerce twoje jáko ſerce Boże /
Dla tego oto Ja przywiodę ná ćię cudzoźiemców nájſrożƺych z narodów / którzy dobywƺy miecżów ſwojich ná piękność mądrośći twojey ſplugáwią jáſność twoję ;
W dół ćię wepchną / y umrzeƺ ſrogą śmierćią w pośród morzá.
Izali mówiąc rzecżeƺ przed tym / który ćię zábijáć będźie : Jeſtem Bóg? gdyżeś cżłowiek á nie Bóg w ręku mordercy twego.
Śmierćią nieobrzezáńców umrzeƺ od ręki cudzoźiemców ; bom Ja mówił / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! podnieś láment nád królem Tyrſkim / á mów do niego : Ták mówi pánujący Pán : Ty / co piecżętujeƺ ſumy / pełen mądrośći y doſkonáłej pięknośći ;
Byłeś w Eden / ogrodźie Bożym ; wƺelki kámień drogi był nákryćiem twojim / ſárdyjuƺ / topázyjuƺ / y jáſpis / chryſolit / onyks / y beryl / ƺáfir / kábunkuł / y ƺmárágd / y złoto ; w ten dźień / któregoś ty ſtworzony / zgotowáne ſą u ćiebie nárzędy bębnów twojich y piƺcżáłek twojich.
Tyś był Cherubinem pomázánym / nákrywájącym ; Jam ćię wyſtáwił / byłeś ná górze Bożey świętey / w pośród kámieniá ogniſtego przechádzáłeś śię /
Byłeś doſkonáły ná drogách twojich ode dniá tego / któregoś jeſt ſtworzony / áż śię ználázłá niepráwość w tobie.
Dla wielkośći kupiectwá twego pełno w pośród ćiebie bezpráwiá / y zgrzeƺyłeś ; dla tego wytrácę ćię z góry Bożey / o Cherubinie nákrywájący! á z pośrzodku kámieniá ogniſtego wygubię ćię.
Podnioſło śię ſerce twoje dla pięknośći twojey / ná złeś używał mądrośći ſwojey dla jáſnośći twojey ; przetoż ćię uderzę o źiemię / á przed oblicżem królów położę ćię / ábyć śię dźiwowáli.
Dla mnóſtwá niepráwośći twojich / y dla nieſpráwiedliwośći kupiectwá twego ſplugáwiłeś świątnicę twoję ; przetoż wywiodę ogień z pośrzodku ćiebie / który ćię pożre / á obrócę ćię w popiół ná źiemi przed ocżymá wƺyſtkich / co ná ćię pátrzą.
Wƺyſcy / co ćię ználi miedzy národámi / zdumieją śię nád tobą : będźieƺ ná wielki poſtrách / á nie będźie ćię áż ná wieki.
Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiw Sydonowi / á prorokuj przećiw niemu /
Y mów : Ták mówi pánujący Pán : Otom Ja przećiwko tobie / o Sydonie! á będę uwielbiony w pośród ćiebie ; y dowiedzą śię / żem Ja Pán / gdy nád nim ſądy wykonam / y w nim poświęcony będę.
Y poślę nań mór / y krew ná ulice jego / y upádną zránieni w pośrzodku niego od miecżá / który ná nich przydźie ze wƺyſtkich ſtron ; á dowiedzą śię / żem Ja Pán.
A ták nie będźie miał więcey dom Izráelſki ćierniá kolącego / y głogu boleść zádáwájącego z wƺyſtkich okolicżnych ſwych / którzy ich puſtoƺą ; y dowiedzą śię / żem Ja pánujący Pán.
Ták mówi pánujący Pán : Gdy zgromádzę dom Izráelſki z narodów / miedzy którymi ſą rozproƺeni / y poświęcony będę w nich przed ocżymá pogán / y będą mieƺkáli w źiemi ſwojey / którąm był dał ſłudze memu Jákóbowi :
Tedy w niej będą bezpiecżnie mieƺkáli / á pobudują domy / y náƺcżepią winnic ; mieƺkáć mówię będą bezpiecżnie / gdy wykonam ſądy nád wƺyſtkimi okolicżnymi ich / którzy ich puſtoƺyli : y dowiedzą śię / żem Ja Pán / Bóg ich.
