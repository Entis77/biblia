Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! mów kśiążęćiu Tyrſkiemu : Ták mówi pánujący Pán : Dlátego / iż śię podnioſło ſerce twoje / á mówiƺ : Jám jeſt Bóg / śiedzę w pośród morzá ná ſtolicy Boſkiej / gdyżeś ty cżłowiek / á nie Bóg / choć ſerce twoje ſtáwiáƺ jáko ſerce Boże ;
Otoś mędrƺym nád Dánijelá / żádná tájemnicá nie jeſt zákrytá przed tobą /
Mądrośćią twoją y roztropnośćią twoją názbieráłeś ſobie bogáctw / y nábyłeś złotá y ſrebrá do ſkárbów toich ;
Wielkośćią mądrośći twojej w kupiectwie twojim rozmnożyłeś bogáctwá twoje ; á ták podnioſło śię ſerce twoje dlá bogáctw twoich.
Przetoż ták mówi pánujący Pán : Poniewáż ſtáwiáƺ ſerce twoje jáko ſerce Boże /
Dlátego oto Já przywiodę ná ćię cudzoźiemców nájſrożƺych z národów / którzy dobywƺy miecżów ſwoich ná piękność mądrośći twojej ſplugáwią jáſność twoję ;
W dół ćię wepchną / y umrzeƺ ſrogą śmierćią w pośród morzá.
Izáli mówiąc rzecżeƺ przed tym / który ćię zábijáć będźie : Jeſtem Bóg? gdyżeś cżłowiek á nie Bóg w ręku mordercy twego.
Śmierćią nieobrzezáńców umrzeƺ od ręki cudzoźiemców ; bom Já mówił / mówi pánujący Pán.
Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! podnieś láment nád królem Tyrſkim / á mów do niego : Ták mówi pánujący Pán : Ty / co piecżętujeƺ ſumy / pełen mądrośći y doſkonáłej pięknośći ;
Byłeś w Eden / ogrodźie Bożym ; wƺelki kámień drogi był nákryćiem twojim / ſárdyjuƺ / topázyjuƺ / y jáſpiſ / chryſolit / onykſ / y beryl / ƺáfir / kábunkuł / y ƺmárágd / y złoto ; w ten dźień / któregoś ty ſtworzony / zgotowáne ſą u ćiebie nárzędy bębnów twoich y piƺcżáłek twoich.
Tyś był Cherubinem pomázánym / nákrywájącym ; Jám ćię wyſtáwił / byłeś ná górze Bożej świętej / w pośród kámieniá ogniſtego przechádzáłeś śię /
Byłeś doſkonáły ná drogách twoich ode dniá tego / któregoś jeſt ſtworzony / áż śię ználázłá niepráwość w tobie.
Dlá wielkośći kupiectwá twego pełno w pośród ćiebie bezpráwiá / y zgrzeƺyłeś ; dlátego wytrácę ćię z góry Bożej / o Cherubinie nákrywájący! á z pośrodku kámieniá ogniſtego wygubię ćię.
Podnioſło śię ſerce twoje dlá pięknośći twojej / ná złeś używáł mądrośći ſwojej dlá jáſnośći twojej ; przetoż ćię uderzę o źiemię / á przed oblicżem królów położę ćię / ábyć śię dźiwowáli.
Dlá mnóſtwá niepráwośći twoich / y dlá nieſpráwiedliwośći kupiectwá twego ſplugwiłeś świątnicę twoję ; przetoż wywiodę ogień z pośrodku ćiebie / który ćię pożre / á obrócę ćię w popiół ná źiemi przed ocżymá wƺyſtkich / co ná ćię pátrzą.
Wƺyſcy / co ćię ználi między národámi / zdumieją śię nád tobą : będźieƺ ná wielki poſtrách / á nie będźie ćię áż ná wieki.
Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! obróć twárz ſwoję przećiw Sydonowi / á prorokuj przećiw niemu /
Y mów : Ták mówi pánujący Pán : Otom Já przećiwko tobie / o Sydonie! á będę uwielbiony w pośród ćiebie ; y dowiedzą śię / żem Já Pán / gdy nád nim ſądy wykonám / y w nim poświęcony będę.
Y poślę náń mór / y krew ná ulice jego / y upádną zránieni w pośrodku niego od miecżá / który ná nich przyjdźie ze wƺyſtkich ſtron ; á dowiedzą śię / żem Já Pán.
A ták nie będźie miáł więcej dom Izráelſki ćierniá kolącego / y głogu boleść zádáwájącego z wƺyſtkich okolicżnych ſwych / którzy ich puſtoƺą ; y dowiedzą śię / żem Já pánujący Pán.
Ták mówi pánujący Pán : Gdy zgromádzę dom Izráelſki z národów / między którymi ſą rozproƺeni / y poświęcony będę w nich przed ocżymá pogán / y będą mieƺkáli w źiemi ſwojej / którąm był dáł ſłudze memu Jákóbowi :
Tedy w niej będą bezpiecżnie mieƺkáli / á pobudują domy / y náƺcżepią winnic ; mieƺkáć mówię będą bezpiecżnie / gdy wykonám ſądy nád wƺyſtkimi okolicżnymi ich / którzy ich puſtoƺyli : y dowiedzą śię / żem Já Pán / Bóg ich.
