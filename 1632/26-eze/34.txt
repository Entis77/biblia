Y ſtáło śię ſłowo Páńſkie do mnie mówiąc :
Synu cżłowiecży! prorokuj przećiwko páſterzom Izráelſkim / prorokuj / á mówi do tych páſterzy : Ták mówi pánujący Pán : Biádá páſterzom Izráelſkim / którzy ſámi śiebie páſą!
Izáli páſterze trzody páść nie máją? Tłuſtość jádáćie / á wełną śię przyodziewáćie / to / co jeſt tłuſtego zábijáćie / á trzody nie páśiećie ;
Słábych nie pośiláćie / á chorego nie lecżyćie / y złámánego nie záwiązujećie / á zápłoƺonego nie przywodzićie / áni zgubionego nie ƺukáćie / owƺem / nád nimi ſurowie y ſrodze pánujećie.
Ták / że rozproƺone będąc ſą bez páterzá y ſtáły śię ná pożárćie wƺelkiemu zwierzowi polnemu / poniewáż śię rozpierzchnęły.
Błąkáją śię owce moje po wƺyſtkich górách / y po káżdym págórku wyſokim ; owƺem / po wƺyſtkiej ziemi rozproƺyły śię owce moje / á nie był / ktoby ich ƺukáł / y ktoby śię zá niemi pytáł.
Dlátego / wy páſterze! ſłuchájćie ſłowá Páńſkiego /
Jáko żyję Já / mówi pánujący Pán : Przeto / iż trzodá mojá jeſt ná łup dáná / á owce moje ſą ná pożárćie wƺelkiemu zwierzowi polnemu / będąc bez páſterzá á iż nie ƺukáją páſterze moi owiec moich / ále tylko páſterze ſámych śiebie páſą / á owiec moich nie páſą ;
Przetoż o páſterze! ſłuchájćie ſłowo Páńſkiego.
Ták mówi pánujący Pán : Otom Já przećiwko páſterzom / y ƺukáć będę owiec moich z rąk ich / á ucżynię / że oni przeſtáną páść owiec moich / áby nie páśli więcej páſterze ſámych śiebie ; wydrę záiſte owce moje z gęby ich / y nie będą im więcej pokármem.
Bo ták mówi pánujący Pán : Oto Já / Já ƺukáć będę owiec moich / y pytáć śię zá niemi.
Jáko śię pytá páſterz o trzodę ſwoję / kiedy bywá w pośrodku owiec ſwoich rozproƺonych : ták śię Já będę pytáł zá owcámi mojemi / y wyrwę je ze wƺyſtkich miejſc / kędy były rozproƺone w dzień obłoku y chmury ;
Y wywiodę je z národów / á zgromádzę je z ziem / y przywiodę je do ziemi ich / á páść je będę ná górách Izráelſkich nád ſtrumieniámi / y po wƺyſtkich mieƺkániách tej ziemi.
Ná páſtwiſkách dobrych páść je będę / á ná górách wyſokich Izráelſkich będą páſtwiſká ich ; tám odpocżywáć będą w tráwách bujnych / á w páƺách tłuſtych páść śię będą ná górách Izráelſkich.
Já ſám páść będę owce moje / y Já im pocżynię legowiſká / mówi pánujący Pán.
Zgubionej ƺukáć będę / á zápłoƺoną przywiodę / y połámáną záwiążę / á ſłábą pośilę ; ále tłuſtą y mocną wytrácę ; bo je będę páſł w ſądzie.
A wy / trzodo mojá! ták mówi pánujący Pán : Oto Já ucżynię ſąd między owcą á owcą / między bárány á kozły.
Azáż wám ná tem máło / páść śię ná dobrej páƺy / że jeƺcże oſtátek páſtwiſk wáƺych depcżećie nogámi ſwojimi? á cżyſtą wodę pić / że oſtátek nogámi ſwemi mąćićie?
Ták / że śię owce moje tem / co było podeptáne nogámi wáƺemi / páść / á męćiny nóg wáƺych pić muƺą.
Przetoż ták mówi pánujący Pán do nich : Oto Já / Já ſąd ucżynię między bydlęćiem tłuſtem y między bydlęćiem chudem /
Dlátego / że wy bokámi y plecámi trącáćie / á rogámi wáƺemi bodziećie wƺyſtkie ſłábe / ták żeśćie je precż rozegnáli.
Przetoż wyzwolę owce moje / że już dálej łupem nie będą / y ucżynię ſąd między owcą y owcą ;
Y wzbudzę nád niemi páſterzá jednego / który je páść będzie / ſługę mego Dawidá / on je páść będzie / y on będzie páſterzem ich.
A Já Pán będę im Bogiem / á ſługá mój Dawid kśiążęćiem w pośrodku nich. Já Pán mówiłem to.
Y ucżynię z nimi przymierze pokoju / á wygubię zły zwierz z ziemi ; y będą ná puƺcży bezpiecżnie mieƺkáć / á w láſách ſypiáć będą ;
Nádto dám im / y okolicy págórká mego / błogáſłáwieńſtwo / y ſpuƺcżáć będę deƺcż cżáſu ſwego ; deƺcże to błogoſłwieńſtwá będą ;
Y wypuśći drzewo polne owoc ſwój / á ziemiá wydá urodzáj ſwój / y będą ná ziemi ſwojej bezpiecżni / á dowiedzą śię / żem Já Pán / gdy połámię záwory járzmá ich / á wyrwię je z ręki tych / którzy je zniewáláją.
Y nie będą więcej łupem národom / á zwierz ziemiſki pożeráć ich nie będzie ; ále mieƺkáć będą bezpiecżnie / á nie będzie / ktoby je ſtráƺył.
Y wzbudzę im látorośl ſłáwną / że nie będą więcej głodem niƺcżeni ná ziemi / áni ponioſą poháńbieniá od pogán.
Y dowiedzą śię / żem Já Pán / Bóg ich / z nimi / á oni lud mój / dom Izráelſki / mówi pánujący Pán.
Ale wy owce moje / owce páſtwiſká mego / wyśćie lud mój / á Jám Bóg wáƺ / mówi pánujący Pán.
