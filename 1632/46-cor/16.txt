A około ſkłádániá ná świętych / jákom poſtánowił we zborách Gáláckich / ták też y wy cżyńćie.
Káżdego pierwƺego dniá w tygodniu káżdy z wáſ niech odkłádá u śiebie / zbierájąc według tego / jáko mu śię powodzi / áby nie dopiero / gdy przyjdę / ſkłádániá cżynione były.
A gdy przyjdę / którychkolwiek uchwálićie przez liſty / tych poślę / áby odnieśli dobrodziejſtwo wáƺe do Jeruzálemu.
A jeźliby ſłuƺná rzecż byłá / żebym y já ƺedł / weſpół ze mną pójdą.
A przyjdę do wáſ / gdy przejdę Mácedoniję ; ( bo pójdę przez Mácedoniję. )
A podobno zámieƺkám u wáſ álbo y przezimuję / ábyśćie wy mię odprowádzili / kędykolwiek pójdę.
Abowiem nie chcę wáſ teráz widzieć / mijájąc ; ále ſpodziewám śię / iż pomieƺkám z wámi cżáſ niejáki / będzieli Pán chćiáł.
A zoſtánę w Efezie áż do Świątek.
Abowiem drzwi mi ſą otworzone wielkie y mocne / y mám wiele przećiwników.
Jeźliby tedy przyƺedł Tymoteuƺ / pátrzćie / áby bez bojáźni był między wámi / bo dzieło Páńſkie ſpráwuje / jáko y já.
Przetoż niecháj go nikt lekce nie wáży ; ále odprowádźćie go w pokoju / áby przyƺedł do mnie ; bo go cżekám z bráćmi.
A o bráćie Apollośie wiedzćie / żem go bárdzo prośił / áby ƺedł do wáſ z bráćmi ; ále zgołá nie miáł woli / áby teráz ƺedł ; przyjdzie jednák / gdy cżáſ po temu mieć będzie.
Cżujćież / ſtójćie w wierze / mężnie ſobie pocżynájćie / zmácniájćie śię.
Wƺyſtkie rzecży wáƺe niech śię dzieją w miłośći.
A proƺę wáſ / bráćiá! wiećie / iż dom Stefánowy jeſt pierwiáſtkiem Achái / á iż ſámi śiebie oddáli ná poſługiwánie świętym.
Abyśćie y wy poddáni byli tákowym / y káżdemu pomágájącemu y prácującemu.
A ćieƺę śię z przyjśćiá Stefáná y Fortunátá / y Acháiká ; bo ći niedoſtátek wáƺ nápełnili.
Ochłodzili bowiem duchá mego y wáƺego ; znájćież tedy tákowych.
Pozdráwiáją wáſ zbory / które ſą w Azyi. Pozdráwiáją wáſ wielce w Pánu Akwiláſ y Pryſcyllá / ze zborem / który jeſt w domu ich.
Pozdráwiáją wáſ bráćiá wƺyſcy. Pozdrówćie jedni drugich w świętym pocáłowániu.
Pozdrowienie ręką moją Páwłową.
Jeźliże kto nie miłuje Páná Jezuſá Chryſtuſá / niech będzie przeklęſtwem / które zowią Márán átá.
Łáſká Páná Jezuſá Chryſtuſá niech będzie z wámi.
Miłość mojá niech będzie z wámi wƺyſtkimi w Chryſtuśie Jezuśie. Amen.
