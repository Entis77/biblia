A około ſkłádániá ná świętych / jákom poſtánowił we zborách Gáláckich / ták też y wy cżyńćie.
Káżdego pierwƺego dniá w tygodniu káżdy z was niech odkłádá u śiebie / zbierájąc według tego / jáko mu śię powodźi / áby nie dopiero / gdy przyjdę / ſkłádániá cżynione były.
A gdy przyjdę / którychkolwiek uchwálićie przez liſty / tych poślę / áby odnieśli dobrodźiejſtwo wáƺe do Jeruzálemu.
A jeſliby ſłuƺná rzecż byłá / żebym y ja ƺedł / weſpół ze mną pójdą.
A przyjdę do was / gdy przejdę Mácedoniję ; ( bo pójdę przez Mácedoniję. )
A podobno zámieƺkám u was álbo y przeźimuję / ábyśćie wy mię odprowádźili / kędykolwiek pójdę.
Abowiem nie chcę was teraz widźieć / mijájąc ; ále ſpodźiewám śię / iż pomieƺkám z wámi cżáſ niejáki / będźieli Pán chćiáł.
A zoſtánę w Efeźie áż do Świątek.
Abowiem drzwi mi ſą otworzone wielkie y mocne / y mam wiele przećiwników.
Jeſliby tedy przyƺedł Tymoteuƺ / pátrzćie / áby bez bojáźni był między wámi / bo dźieło Páńſkie ſpráwuje / jáko y ja.
Przetoż niecháj go nikt lekce nie wáży ; ále odprowádźćie go w pokoju / áby przyƺedł do mnie ; bo go cżekám z bráćmi.
A o bráćie Apollośie wiedzćie / żem go bárdzo prośił / áby ƺedł do was z bráćmi ; ále zgołá nie miał woli / áby teraz ƺedł ; przyjdźie jednák / gdy cżáſ po temu mieć będźie.
Cżujćież / ſtójćie w wierze / mężnie ſobie pocżynájćie / zmácniájćie śię.
Wƺyſtkie rzecży wáƺe niech śię dźieją w miłośći.
A proƺę was / bráćia! wiećie / iż dom Stefánowy jeſt pierwiáſtkiem Achái / á iż ſámi śiebie oddáli ná poſługiwánie świętym.
Abyśćie y wy poddáni byli tákowym / y káżdemu pomágájącemu y prácującemu.
A ćieƺę śię z przyjśćiá Stefáná y Fortunátá / y Acháiká ; bo ći niedoſtátek wáƺ nápełnili.
Ochłodźili bowiem duchá mego y wáƺego ; znájćież tedy tákowych.
Pozdráwiáją was zbory / które ſą w Azyi. Pozdráwiáją was wielce w Pánu Akwiláſ y Pryſcyllá / ze zborem / który jeſt w domu ich.
Pozdráwiáją was bráćia wƺyſcy. Pozdrówćie jedni drugich w świętym pocáłowániu.
Pozdrowienie ręką moją Páwłową.
Jeſliże kto nie miłuje Páná JEzuſá CHryſtuſá / niech będźie przeklęſtwem / które zowią Márán átá.
Łáſká Páná JEzuſá CHryſtuſá niech będźie z wámi.
Miłość mojá niech będźie z wámi wƺyſtkimi w CHryſtuśie JEzuśie. Amen.
