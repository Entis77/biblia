Páweł powołány Apoſtoł JEzuſá CHryſtuſá przez wolą Bożą / y Soſtenes brát.
Zborowi Bożemu który jeſt w Korynćie / poświęconym w CHryſtuśie JEzuśie / powołánym świętym / ze wƺyſtkimi którzy wzywáją Imienia PAná náƺego JEzuſá CHryſtuſá ná wƺelkim miejſcu / y ich y náƺego <i>PAná</i>.
Łáſká wam y pokój <i>niech będzie</i> od Bogá Ojcá náƺego / y od PAná JEzuſá CHryſtuſá.
Dźiękuję Bogu mojemu záwƺe zá was / dla łáſki Bożey / która wam jeſt dána w CHryſtuśie JEzuśie.
Iżeśćie we wƺyſtkim ubogáceni w nim we wƺelkiey mowie y we wƺelkiey znájomośći.
Jáko świádectwo CHryſtuſowe utwirdzone jeſt w was.
Ták iż wam ná żadnym dárze nie zchodźi / którzy ocżekawaćie objáwienia PAná náƺego JEzuſá CHryſtuſá.
Który was też utwirdźi áż do końcá / ábyśćie byli bez nágány w dźień PAná náƺego JEzuſá CHryſtuſá.
Wiernyć <i>jeſt</i> Bóg przez którego jeſteśćie powołáni ku ſpołecżnośći Syná jego JEzuſá CHryſtuſá / PAná náƺego.
A proƺę was bráćia przez Imię PAná náƺego JEzuſá CHryſtuſá / ábyśćie toż mówili wƺyſcy / á iżby nie były miedzy wámi rozerwánia : ále ábyśćie byli zpojeni jednákim umyſłem / y jednákim zdániem.
Abowiem oznájmiono mi o was / bráćia moji / od <i>domowników</i> Chloji iż poſwarki ſą miedzy wámi.
A to powiádam / iż káżdy z was mówi ; Jamći jeſt Páwłów / á jam Apolloſów / á jam Kefáſów / álem ja CHryſtuſów.
Rozdźielonyż jeſt CHryſtus? Azaż Páweł zá was ukrzyżowány? Albośćie w imię Páwłowe ochrzcżeni?
Dźiękuję Bogu / żem żadnego z was nie chrzćił / oprócż Kryſpá y Gájuſá :
Aby kto nie rzekł / żem chrzćił w imię moje.
Ochrzćiłem też y dom Stefánów. Nád to nie wiem / jeſlim kogo drugiego ochrzćił.
Boć mię nie poſłał CHryſtus chrzćić / ále Ewángelią kazáć : <i>Wƺákże</i> nie w mądrośći mowy / áby nie był wyniƺcżony krzyż CHryſtuſów.
Abowiem mowá o krzyżu tym którzy giną / jeſt głupſtwem ; ále nam / którzy bywamy zbáwieni / jeſt mocą Bożą.
Bo nápiſano ; Wniwecż obrócę mądrość mądrych / á rozum rozumnych odrzucę.
Gdźież <i>jeſt</i> mądry? Gdźież <i>jeſt</i> ucżony w piśmie? Gdźież bádácż wieku tego? Izali w głupſtwo nie obróćił Bóg mądrośći świátá tego?
Abowiem ponieważ w mądrośći Bożey świát nie poznał BOgá przez mądrość / upodobáło śię Bogu przez głupie kazánie zbáwić wierzące.
Gdyż y Żydowie śię cudów domagáją / á Grekowie mądrośći ƺukáją :
Ale my każemy CHryſtuſá ukrzyżowánego ; Żydom w prawdźie zgorƺenie / á Grekom głupſtwo :
Lecż ſámym powołánym y Żydom y Grekom <i>każemy</i> CHryſtuſá / który jeſt mocą Bożą / y mądrośćią Bożą.
Abowiem głupſtwo Boże jeſt mędrƺe / niż ludźie ; á mdłość Boża jeſt mocniejƺa / niż ludźie.
Widźićie zájiſte powołánie wáƺe / bráćia / iż nie wiele mądrych według ćiáłá / nie wiele możnych / nie wiele zacnego rodu :
Ale co głupiego jeſt u świátá tego / <i>to</i> wybrał Bóg / áby záwſtydźił mądre : á co mdłego u świátá / wybrał Bóg / áby záwſtydźił mocne.
A podłego rodu u świátá y wzgárdzone wybrał Bóg ; owƺem te rzecży których niemáƺ / áby te które ſą zniƺcżył :
Aby śię nie chlubiło żadne ćiáło przed oblicżem jego.
Lecż z niego wy jeſteśćie w CHryſtuśie JEzuśie ; Który śię nam ſtał mądrośćią od Bogá / y ſpráwiedliwośćią / y poświęceniem / y odkupieniem :
Aby / jáko nápiſano ; Kto śię chlubi / w PANU śię chlubił.
