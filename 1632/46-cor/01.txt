Páweł / powołány Apoſtoł Jezuſá Chryſtuſá przez wolę Bożą / y Soſteneſ brát.
Zborowi Bożemu / który jeſt w Korynćie / poświęconym w Chryſtuśie Jezuśie / powołánym świętym / ze wƺyſtkimi / którzy wzywáją imieniá Páná náƺego Jezuſá Chryſtuſá ná wƺelkiem miejſcu / y ich y náƺem.
Łáſká wám y pokój niech będzie od Bogá / Ojcá náƺego / y od Páná Jezuſá Chryſtuſá.
Dziękuję Bogu mojemu záwƺe zá wáſ dlá łáſki Bożej / którá wám jeſt dáná w Chryſtuśie Jezuśie /
Iżeśćie we wƺyſtkiem ubogáceni w nim we wƺelkiej mowie y we wƺelkiej znájomośći ;
Jáko świádectwo Chryſtuſowe utwierdzone jeſt w wáſ /
Ták iż wám ná żádnym dárze nie ſchodzi / którzy ocżekujećie objáwieniá Páná náƺego Jezuſá Chryſtuſá.
Który wáſ też utwierdzi áż do końcá / ábyśćie byli bez nágány w dzień Páná náƺego Jezuſá Chryſtuſá.
Wiernyć jeſt Bóg / przez którego jeſteśćie powołáni ku ſpołecżnośći Syná jego Jezuſá Chryſtuſá / Páná náƺego.
A proƺę wáſ / bráćiá! przez imię Páná náƺego Jezuſá Chryſtuſá / ábyśćie toż mówili wƺyſcy / á iżby nie były między wámi rozerwániá / ále ábyśćie byli ſpojeni jednákim umyſłem y jednákiem zdániem.
Abowiem oznájmiono mi o wáſ / bráćiá moi! od domowników Chloi / iż poſwárki ſą między wámi.
A to powiádám / iż káżdy z wáſ mówi : Jámći jeſt Páwłowy / á jám Apolloſowy / á jám Kiefáſowy / álem já Chryſtuſowy.
Rozdzielonyż jeſt Chryſtuſ? Azáż Páweł zá wáſ ukrzyżowány? Albośćie w imię Páwłowe ochrzcżeni?
Dziękuję Bogu / żem żádnego z wáſ nie chrzćił / oprócż Kryſpá y Gájuſá ;
Aby kto nie rzekł / żem chrzćił w imię moje.
Ochrzćiłem też y dom Stefánowy ; nádto nie wiem / jeźlim kogo drugiego ochrzćił.
Boć mnie nie poſłáł Chryſtuſ chrzćić / ále Ewángieliję kázáć / wƺákże nie w mądrośći mowy / áby nie był wyniƺcżony krzyż Chryſtuſowy.
Abowiem mowá o krzyżu tym / którzy giną / jeſt głupſtwem ; ále nám / którzy bywámy zbáwieni / jeſt mocą Bożą.
Bo nápiſáno : Wniwecż obrócę mądrość mądrych / á rozum rozumnych odrzucę.
Gdzież jeſt mądry? gdzież jeſt ucżony w Piśmie? gdzież bádácż wieku tego? Izáli w głupſtwo nie obróćił Bóg mądrośći świátá tego?
Abowiem poniewáż w mądrośći Bożej świát nie poznáł Bogá przez mądrość / upodobáło śię Bogu / przez głupie kázánie zbáwić wierzących /
Gdyż y Żydowie śię cudów domágáją / á Grekowie mądrośći ƺukáją.
Ale my káżemy Chryſtuſá ukrzyżowánego / Żydom wpráwdzie zgorƺenie / á Grekom głupſtwo ;
Lecż ſámym powołánym y Żydom y Grekom káżemy Chryſtuſá / który jeſt mocą Bożą y mądrośćią Bożą.
Abowiem głupſtwo Boże jeſt mędrƺe / niż ludzie ; á mdłość Bożá jeſt mocniejƺá / niż ludzie.
Widzićie záiſte powołánie wáƺe / bráćiá! iż nie wiele mądrych według ćiáłá / nie wiele możnych / nie wiele zácnego rodu ;
Ale co głupiego jeſt u świátá tego / to wybráł Bóg / áby záwſtydził mądrych ; á co mdłego u świátá / wybráł Bóg / áby záwſtydził mocnych.
A podłego rodu u świátá y wzgárdzone wybráł Bóg / owƺem te rzecży / których niemáƺ / áby te / które ſą / zniƺcżył.
Aby śię nie chlubiło żádne ćiáło przed oblicżem jego.
Lecż z niego wy jeſteśćie w Chryſtuśie Jezuśie / który śię nám ſtáł mądrośćią od Bogá y ſpráwiedliwośćią y poświęceniem y odkupieniem /
Aby / jáko nápiſáno : Kto śię chlubi / w Pánu śię chlubił.
