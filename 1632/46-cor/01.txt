Páweł / powołány Apoſtoł JEzuſá CHryſtuſá przez wolę Bożą / y Soſteneſ brát.
Zborowi Bożemu / który jeſt w Korynćie / poświęconym w CHryſtuśie JEzuśie / powołánym świętym / ze wƺyſtkimi / którzy wzywáją imieniá Páná náƺego JEzuſá CHryſtuſá ná wƺelkiem miejſcu / y ich y náƺem.
Łáſká wám y pokój niech będźie od Bogá / Ojcá náƺego / y od Páná JEzuſá CHryſtuſá.
Dźiękuję Bogu mojemu záwƺe zá was dla łáſki Bożej / która wám jeſt dáná w CHryſtuśie JEzuśie /
Iżeśćie we wƺyſtkiem ubogáceni w nim we wƺelkiej mowie y we wƺelkiej znájomośći ;
Jáko świádectwo CHryſtuſowe utwierdzone jeſt w was /
Ták iż wám ná żádnym dárze nie ſchodźi / którzy ocżekujećie objáwieniá Páná náƺego JEzuſá CHryſtuſá.
Który was też utwierdźi áż do końcá / ábyśćie byli bez nágány w dźień Páná náƺego JEzuſá CHryſtuſá.
Wiernyć jeſt Bóg / przez którego jeſteśćie powołáni ku ſpołecżnośći Syná jego JEzuſá CHryſtuſá / Páná náƺego.
A proƺę was / bráćia! przez imię Páná náƺego JEzuſá CHryſtuſá / ábyśćie toż mówili wƺyſcy / á iżby nie były między wámi rozerwániá / ále ábyśćie byli ſpojeni jednákim umyſłem y jednákiem zdániem.
Abowiem oznájmiono mi o was / bráćia moi! od domowników Chloi / iż poſwárki ſą między wámi.
A to powiádám / iż káżdy z was mówi : Jámći jeſt Páwłowy / á jám Apolloſowy / á jám Kiefáſowy / álem ja CHryſtuſowy.
Rozdźielonyż jeſt CHryſtuſ? Azáż Páweł zá was ukrzyżowány? Albośćie w imię Páwłowe ochrzcżeni?
Dźiękuję Bogu / żem żádnego z was nie chrzćił / oprócż Kryſpá y Gájuſá ;
Aby kto nie rzekł / żem chrzćił w imię moje.
Ochrzćiłem też y dom Stefánowy ; nádto nie wiem / jeſlim kogo drugiego ochrzćił.
Boć mnie nie poſłáł CHryſtuſ chrzćić / ále Ewángieliję kázáć / wƺákże nie w mądrośći mowy / áby nie był wyniƺcżony krzyż CHryſtuſowy.
Abowiem mowá o krzyżu tym / którzy giną / jeſt głupſtwem ; ále nám / którzy bywámy zbáwieni / jeſt mocą Bożą.
Bo nápiſáno : Wniwecż obrócę mądrość mądrych / á rozum rozumnych odrzucę.
Gdźież jeſt mądry? gdźież jeſt ucżony w Piśmie? gdźież bádácż wieku tego? Izáli w głupſtwo nie obróćił Bóg mądrośći świátá tego?
Abowiem poniewáż w mądrośći Bożej świát nie poznał Bogá przez mądrość / upodobáło śię Bogu / przez głupie kázánie zbáwić wierzących /
Gdyż y Żydowie śię cudów domágáją / á Grekowie mądrośći ƺukáją.
Ale my káżemy CHryſtuſá ukrzyżowánego / Żydom wpráwdźie zgorƺenie / á Grekom głupſtwo ;
Lecż ſámym powołánym y Żydom y Grekom káżemy CHryſtuſá / który jeſt mocą Bożą y mądrośćią Bożą.
Abowiem głupſtwo Boże jeſt mędrƺe / niż ludźie ; á mdłość Bożá jeſt mocniejƺá / niż ludźie.
Widźićie záiſte powołánie wáƺe / bráćia! iż nie wiele mądrych według ćiáłá / nie wiele możnych / nie wiele zácnego rodu ;
Ale co głupiego jeſt u świátá tego / to wybráł Bóg / áby záwſtydźił mądrych ; á co mdłego u świátá / wybráł Bóg / áby záwſtydźił mocnych.
A podłego rodu u świátá y wzgárdzone wybráł Bóg / owƺem te rzecży / których niemáƺ / áby te / które ſą / zniƺcżył.
Aby śię nie chlubiło żádne ćiáło przed oblicżem jego.
Lecż z niego wy jeſteśćie w CHryſtuśie JEzuśie / który śię nám ſtáł mądrośćią od Bogá y ſpráwiedliwośćią y poświęceniem y odkupieniem /
Aby / jáko nápiſáno : Kto śię chlubi / w Pánu śię chlubił.
