Ták niecháj o náſ cżłowiek rozumie / jáko o ſługách Chryſtuſowych y o ƺáfárzách tájemnic Bożych.
A tegoć więc ƺukáją przy ƺáfárzách / áby káżdy ználeziony był wiernym.
Aleć u mnie to jeſt nájmniejƺá / żebym był od wáſ ſądzony / álbo od ſądu ludzkiego ; lecż y ſám śiebie nie ſądzę.
Abowiem choć nic ná śię nie wiem / wƺákże nie przeto jeſtem uſpráwiedliwiony ; ále ten / który mię ſądzi / Pán jeſt.
A ták nie ſądźćie przed cżáſem / áżby Pán przyƺedł / który też oświeći / co ſkrytego jeſt w ćiemnośći y objáwi rády ſerc ; á tedy káżdy będzie miáł chwáłę od Bogá.
A te rzecży / bráćiá! w podobieńſtwie obróćiłem ná śię y ná Apolloſá dlá wáſ / ábyśćie śię náucżyli z náſ / nád to / co nápiſáne / nie rozumieć / iżbyśćie śię jeden dlá drugiego nie nádymáli przećiwko drugiemu.
Abowiem któż ćię różnym cżyni? Y cóż máƺ / cżegobyś nie wziął? A jeźliżeś wziął / przecżże śię chlubiƺ / jákobyś nie wziął?
Jużeśćie náſyceni / jużeśćie ubogáceni / bez náſ królujećie ; á bodájeśćie królowáli / ábyśmy y my z wámi poſpołu królowáli!
Bo mám zá to / iż Bóg náſ oſtátnich Apoſtołów wyſtáwił jákoby ná śmierć ſkázánych ; ábowiem ſtáliśmy śię dziwowiſkiem świátu y Aniołom y ludziom.
Myśmy głupi dlá Chryſtuſá / áleśćie wy roztropni w Chryſtuśie ; myśmy ſłábi / áleśćie wy mocni ; wyśćie zácni / áleśmy my bezecni.
Jeƺcże áż do tej godziny y łákniemy y prágniemy / y nádzy jeſteśmy / y bywámy policżkowáni / y tułámy śię /
Y prácujemy / robiąc włáſnemi rękámi ; gdy náſ háńbią / dobrorzecżemy / gdy náſ prześládują / znośimy ;
Gdy nám złorzecżą / modlimy śię zá nich ; ſtáliśmy śię jáko śmieći tego świátá / y jáko omiećiny u wƺyſtkich / áż dotąd.
To piƺę / nie przeto / ábym wáſ záwſtydził ; ále jáko dziátki moje miłe nápominám.
Bo choćbyśćie mieli dzieśięć tyśięcy pedágogów w Chryſtuśie / wƺákże niewiele ojców máćie ; bom já wáſ w Jezuśie Chryſtuśie przez Ewángieliję ſpłodził.
Proƺę wáſ tedy / bądźćie náśládowcámi moimi.
Dlátegom poſłáł do wáſ Tymoteuƺá / który jeſt ſyn mój miły y wierny w Pánu ; ten wám przypomni drogi moje w Chryſtuśie / jáko wƺędy w káżdym zborze náucżám.
Ale ták śię niektórzy nádęli / jákobym nie miáł przyjść do wáſ.
Lecż przyjdę rychło do wáſ / jeźli Pán będzie chćiáł / y poznám nie mowę tych nádętych / ále moc.
Abowiem nie w mowie záleży króleſtwo Boże / ále w mocy.
Cóż chcećie? z rózgąli mám przyjść do wáſ / cżyli z miłośćią y z duchem ćichośći?
