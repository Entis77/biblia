Náśládujćie miłośći / ſtárájćie śię uśilnie o dáry duchowne ; lecż nájwięcej / ábyśćie prorokowáli.
Abowiem kto mówi językiem obcym / nie ludziom mówi / ále Bogu : bo żáden nie ſłuchá / lecż on duchem mówi tájemnice.
Ale kto prorokuje / mówi ludziom zbudowánie y nápominánie y poćiechę.
Kto językiem obcym mówi / ſámego śiebie buduje ; ále kto prorokuje / ten zbór buduje.
A chćiáłbym ábyśćie wy wƺyſcy językámi mówili / ále rácżej ábyśćie prorokowáli ; ábowiem więkƺy jeſt ten / co prorokuje / niż ten / co językámi obcymi mówi / chybá żeby tłumácżył / áby zbór bráł zbudowánie.
Teráz tedy / bráćiá! gdybym przyƺedł do wáſ / językámi obcymi mówiąc / cóż wám pomogę / jeźlibym wám nie mówił álbo przez objáwienie / álbo przez umiejętność / álbo przez proroctwo / álbo przez náukę?
Wƺák y rzecży nieżywe / które dźwięk wydáwáją jáko piƺcżáłká álbo cytrá / jeźliby różnego dźwięku nie wydáwáły / jákoż poznáne będzie / co ná piƺcżáłce / álbo co ná cytrze gráją?
Abowiem jeźliby trąbá niepewny głoſ dáłá / któż śię do boju gotowáć będzie?
Tákże y wy / jeźlibyśćie językiem nie wydáli mowy dobrze zrozumiáłej / jákoż będzie zrozumiáne / co śię mówi? ábowiem będziećie tylko ná wiátr mówić.
Ták wiele / jáko ſłyƺymy / jeſt różnych głoſów ná świećie / á nic nie jeſt bez głoſu.
Jeźlibym tedy nie znáł mocy głoſu / będę temu / który mówi / cudzoziemcem ; á ten / co mówi / będzie mi tákże cudzoziemcem.
Tákże y wy / poniewáż śię uśilnie ſtáráćie o dáry duchowne / ƺukájćież tego / ábyśćie obfitowáli ku zbudowániu zboru.
Dlátego kto mówi obcym językiem / niech śię modli / áby mógł tłumácżyć.
Bo jeźlibym śię modlił obcym językiem / modlić śię mój duch ; ále rozum mój jeſt bez pożytku.
Cóż tedy jeſt? Będę śię modlił duchem / będę śię też modlił y wyrozumieniem ; będę śpiewáł duchem / będę też śpiewáł y wyrozumieniem.
Bo jeźlibyś błogoſłáwił duchem / jákoż ten / który jeſt z pocżtu proſtáków / ná twoje dziękowánie rzecże Amen / poniewáż nie wie / co mówiƺ?
Bo choć ty wpráwdzie dobrze dziękujeƺ / ále śię drugi nie buduje.
Dziękuję Bogu mojemu / iż więcej / niż wy wƺyſcy / językámi mówię.
A wƺákże we zborze wolę pięć ſłów zrozumitelnie przemówić / ábym y drugich náucżył / niżeli dzieśięć tyśięcy ſłów językiem obcym.
Bráćiá! nie bądźćie dziećmi wyrozumieniem / ále bądźćie dziećmi złośćią / á wyrozumieniem doroſłymi bądźćie.
W zákonie nápiſáno : Iż obcemi językámi y obcemi wárgámi mówić będę ludowi temu ; á przećię mnie y ták nie uſłucháją / mówi Pán.
Przetoż języki ſą zá cud / nie tym / którzy wierzą / ále niewiernym ; á proroctwo nie niewiernym / ále wierzącym.
Jeźliby śię tedy wƺyſtek zbór ná jedno miejſce zeƺedł / á wƺyſcyby językámi obcemi mówili / á weƺliby tám proſtácy álbo niewierni / izáli nie rzeką / że ƺálejećie?
Ale jeźliby wƺyſcy prorokowáli / á wƺedłby który niewierny álbo proſták / od wƺyſtkich przekonány y od wƺyſtkich ſądzony bywá.
A ták ſkrytośći ſercá jego bywáją objáwione / á on upádłƺy ná oblicże / pokłonili śię Bogu / wyznáwájąc / że Bóg jeſt práwdziwie w wáſ.
Cóż tedy jeſt / bráćiá? Gdy śię ſchodzićie / káżdy z wáſ má pſálm / má náukę / má język / má objáwienie / má tłumácżenie ; wƺyſtko to niech śię dzieje ku zbudowániu.
Jeźli kto językiem mówi / niech to będzie po dwóch álbo nájwięcej po trzech / y to ná przemiány / á jeden niech tłumácży.
A jeźliby tłumácżá nie było niechże we zborze milcży ten / który obcym językiem mówi / á niech mówi ſobie y Bogu.
Ale prorocy niech mówią dwáj álbo trzej / á drudzy niech rozſądzáją.
Jeźliby też inƺemu śiedzącemu co było objáwione / on pierwƺy niecháj milcży.
Bo możećie wƺyſcy jeden po drugim prorokowáć / áby śię wƺyſcy ucżyli y wƺyſcy poćieƺeni byli.
Y duchy proroków ſą poddáne prorokom.
Abowiem Bóg nie jeſt powodem nieporządku / ále pokoju / jáko y we wƺyſtkich zborách świętych.
Niewiáſty wáƺe niech milcżą we zborách ; ábowiem nie pozwolono im / áby mówiły / ále áby poddánemi były / jáko y zákon mówi.
A jeźli śię cżego náucżyć chcą / niechże w domu mężów ſwoich pytáją / poniewáż ſromotá niewiáſtom / we zborze mówić.
Izáli od wáſ ſłowo Boże wyƺło? Izáli tylko do wáſ ſámych przyƺło?
Izáli kto zdá śię być prorokiem álbo duchownym / niech uzná / iż te rzecży / które wám piƺę / ſą Páńſkim rozkázániem.
A jeźli kto nie wie / niechájże nie wie.
A ták / bráćiá! ſtárájćie śię uśilnie o to / ábyśćie prorokowáli / á językámi obcemi mówić nie zábrániájćie.
Wƺyſtko śię niecháj dzieje przyſtojnie y porządnie.
