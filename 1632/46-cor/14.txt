Náśládujćie miłośći / ſtárajćie śię uśilnie o dáry duchowne : lecż nawięcey ábyśćie prorokowáli.
Abowiem kto mówi językiem <i>obcym,</i> nie ludźiom mówi / ále Bogu : Bo żaden nie ſłucha / lecż <i>on</i> duchem mówi tájemnice.
Ale kto prorokuje / mówi ludźiom zbudowánie / y nápominánie / y poćiechę.
Kto językiem <i>obcym</i> mówi / ſámego śiebie buduje : ále kto prorokuje / <i>ten</i> Zbór buduje.
A chćiałbym ábyśćie wy owƺyſcy językámi mówili / ále rácżey ábyśćie prorokowáli. Abowiem więkƺy <i>jeſt</i> ten co prorokuje / niż ten co językámi <i>obcymi</i> mówi ; chybá żeby tłumácżył / áby Zbór brał zbudowánie.
Teraz tedy bráćia / gdybym przyƺedł do was językámi <i>obcymi</i> mówiąc / cóż wam pomogę / jeſlibym wam nie mówił álbo przez objáwienie / álbo przez umiejętność / álbo przez proroctwo / álbo przez náukę.
Wƺák <i>y rzecży</i> nie żywe / które dźwięk wydawáją jáko piƺcżałká álbo Cytrá ; jeſliby róznego dźwięku nie wydawáły / jákoż poznano będźie co ná piƺcżałce álbo co ná cytrze gráją?
Abowiem jeſliby trąbá niepewny głos dáłá / któż śię do boju gotowáć będźie?
Tákże y wy / jeſlibyśćie językiem nie wydáli mowy dobrze zrozumitelney / jákoż będźie rozumiano co śię mówi? Abowiem będźiećie <i>tylko</i> ná wiátr mówić.
Ták wiele / jáko ſłychawamy / jeſt róznych głoſów ná świećie ; á nic nie jeſt bez głoſu.
Jeſlibym tedy nie znał mocy głoſu / będę temu który mówi / cudzoźiemcem : á ten co mówi / będźie mi <i>tákże</i> cudzoźiemcem.
Tákże y wy ponieważ śię uśilnie ſtáraćie o dáry duchowne / ƺukajćież tego ábyśćie obfitowáli ku zbudowániu Zboru.
Dla tego kto mówi <i>obcym</i> językiem / niech śię modli áby mógł tłumácżyć.
Bo jeſlibym śię modlił <i>obcym</i> językiem / modlić śię mój duch ; ále rozum mój jeſt bez pożytku.
Cóż tedy jeſtl? Będę śię modlił duchem / będę śię też modlił y wyrozumieniem. Będę śpiewał duchem / będę też śpiewał y wyrozumieniem.
Bo jeſlibyś błogoſłáwił duchem / jákoż ten który jeſt z pocżtu proſtaków ná twoje dźiękowánie rzecże Amen ? ponieważ nie wie co mówiƺ.
Bo choć ty wprawdźie dobrze dźiękujeƺ / ále śię drugi nie buduje.
Dźiękuję Bogu mojemu / iż więcey niż wy wƺyſcy językámi mówię.
A wƺákże we Zborze wolę pięć ſłów zrozumitelnie przemówić / ábym y drugich náucżył / niżeli dźieśięć tyśięcy ſłów językiem <i>obcym</i>.
Bráćia / nie bądźćie dźiećmi wyrozumieniem ; ále bądźćie dźiećmi złośćią / á wyrozumieniem / doroſłymi bądźćie.
W zakonie nápiſano : Iż obcymi językámi / y obcymi wárgámi mówić będę ludowi temu ; á przećię mię y ták nie uſłucháją / mówi PAn.
Przetoż języki ſą zá cud / nie tym którzy wierzą / ále niewiernym : á proroctwo nie niewiernym / ále wierzącym.
Jeſliby śię tedy wƺyſtek Zbór ná jedno miejſce zƺedł / á wƺyſcyby językámi obcymi mówili / á weƺliby <i>tám</i> proſtacy / álbo niewierni / izali nie rzeką że ƺálejećie?
Ale jeſliby wƺyſcy prorokowáli / á wƺedłby <i>który</i> niewierny álbo proſtak / od wƺyſtkich przekonány y od wƺyſtkich ſądzony bywa ;
A ták ſkrytośći ſercá jego bywáją objáwione : á on upadƺy ná oblicże / pokłoni śię Bogu / wyznawájąc że Bóg jeſt prawdźiwie między wámi.
Cóż tedy jeſt bráćia? Gdy śię zchodźićie / káżdy z was ma Pſálm / ma náukę / ma język / ma objáwienie / ma tłumácżenie ; wƺyſtko <i>to</i> niech śię dźieje ku zbudowániu.
Jeſli kto językiem mówi / <i>niech to będźie</i> po dwu / álbo nawięcey po trzech ; y to ná przemiány : á jeden niech tłumácży.
A jeſliby tłumácżá nie było / niechże we Zborze milcży <i>ten który obcym jężykiem mówi,</i> á niech mówi ſobie y Bogu.
Ale prorocy niech mówią dwá álbo trzej / á drudzy niech rozſądzáją.
Jeſliby też inƺemu śiedzącemu <i>co</i> było objáwiono / on pierwƺy niechaj milcży.
Bo możećie wƺyſcy jeden po drugim prorokowáć / áby śię wƺyſcy ucżyli / y wƺyſcy poćieƺeni byli.
Y duchowieć proroków / ſą poddáni Prorokom.
Abowiem Bóg nie jeſt <i>powodem</i> nierządu / ále pokoju ; jáko y we wƺyſtkich Zborách świętych.
Niewiáſty wáƺe niech milcżą we Zborách : ábowiem nie pozwolono im áby mówiły / ále áby poddánymi były ; jáko y zakon mówi.
A jeſli śię cżego náucżyć chcą / niechże domá mężów ſwojich pytáją : ponieważ ſromotá niewiáſtam we Zborze mówić.
Izali od was ſłowo Boże wyƺło? Izali tylko do was ſámych przyƺło?
Jeſli kto zda śię być Prorokiem álbo duchownym / niech uzna iż te rzecży które wam piƺę / ſą PAńſkim rozkazániem.
A jeſli kto nie wie / niechajże nie wie.
A ták bráćia / ſtárajćie śię uśilnie <i>oto</i> ábyśćie prorokowáli : á językámi <i>obcymi</i> mówić nie zábraniajćie.
Wƺyſtko śię niechaj dźieje przyſtojnie y porządnie.
