Choćbym mówił językámi ludzkimi y Anjelſkimi / á miłośćibym nie miał / ſtałem śię <i>jáko</i> miedź brząkájąca álbo cymbał brzmiący.
Y choćbym miał proroctwo / y wiedźiałbym wƺyſtkie tájemnice / y wƺelką umiejętność : y choćbym miał wƺyſtkę wiárę / ták żebym góry przenośił / á miłośćibym nie miał ; nicem nie jeſt.
Y choćbym wynáłożył ná żywność <i>ubogich</i> wƺyſtkę májętność moję / y choćbym wydał ćiáło moje ábym był ſpalony / á miłośćibym nie miał ; nic mi <i>to</i> nie pomoże.
Miłość jeſt długo ćierpliwa / dobrotliwa jeſt : miłość nie zajrzy / miłość nie jeſt roſpuſtna / nie nádyma śię :
Nie cżyni nic nieprzyſtojnego / nie ƺuka ſwojich rzecży / nie jeſt porywcża <i>do gniewu,</i> nie myśli złego :
Nie ráduje śię z nieſpráwiedliwośći ; ále śię ráduje z prawdy :
Wƺyſtko okrywa / wƺyſtkiemu wierzy / wƺyſtkiego śię ſpodźiewa / wƺyſtko ćierpi.
Miłość nigdy nie uſtawa : Bo choć <i>ſą</i> proroctwá / te zniƺcżeją ; choć języki / te uſtáną : choć umiejętność / wniwecż śię obróći.
Abowiem po cżęśći znamy / y po cżęśći prorokujemy.
Ale gdy przydźie to co jeſt doſkonáłego / tedy to co jeſt pocżęśći / zniƺcżeje.
Pókim był dźiećięćiem / mówiłem jáko dźiećię / rozumiałem jáko dźiećię / rozmyślałem jáko dźiećię : lecż gdym śię ſtał mężem / zániechałem rzecży dźiećińſkich.
Abowiem teraz widźiemy przez zwierćiádło / <i>y niby</i> w gadce : ále ná on cżás twarzą w twarz. Teraz poznawam po cżęśći ; ále ná on cżás poznam / jákom y poznány jeſt.
A teraz zoſtawa wiárá / nádźiejá / miłość / te trzy rzecży : lecż z nich nawiękƺa jeſt miłość.
