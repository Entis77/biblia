Choćbym mówił językámi ludzkimi y ánielſkimi / á miłośćibym nie miáł / ſtáłem śię jáko miedź brząkájącá / álbo cymbáł brzmiący.
Y choćbym miáł proroctwo y wiedziáłbym wƺyſtkie tájemnice / y wƺelką umiejętność / y choćbym miáł wƺyſtkę wiárę / ták / żebym góry przenośił / á miłośćibym nie miáł / nicem nie jeſt.
Y choćbym wynáłożył ná żywność ubogich wƺyſtkę májętność moję / y choćbym wydáł ćiáło moje / ábym był ſpálony / á miłośćibym nie miáł / nic mi to nie pomoże.
Miłość jeſt długo ćierpliwá / dobrotliwá jeſt ; miłość nie zájrzy / miłość nie jeſt rozpuſtná / nie nádymá śię ;
Nie cżyni nic nieprzyſtojnego / nie ƺuká ſwoich rzecży / nie jeſt porywcżá do gniewu / nie myśli złego ;
Nie ráduje śię z nieſpráwiedliwośći / ále śię ráduje z práwdy ;
Wƺyſtko okrywá / wƺyſtkiemu wierzy / wƺyſtkiego śię ſpodziewá / wƺyſtko ćierpi.
Miłość nigdy nie uſtáje ; bo choć ſą proroctwá / te zniƺcżeją ; choć języki / te uſtáną ; choć umiejętność / wniwecż śię obróći.
Abowiem po cżęśći známy y po cżęśći prorokujemy.
Ale gdy przyjdzie to / co jeſt doſkonáłego / tedy to / co jeſt po cżęśći / zniƺcżeje.
Pókim był dziećięćiem / mówiłem jáko dziećię / rozumiáłem jáko dziećię / rozmyśláłem jáko dziećię ; lecż gdym śię ſtáł mężem / zániecháłem rzecży dziećinnych.
Abowiem teráz widzimy przez zwierćiádło / y niby w zágádce ; ále ná on cżáſ twárzą w twárz ; teráz poznáję po cżęśći / ále ná on cżáſ poznám / jákom y poznány jeſt.
A teráz zoſtáje wiárá / nádziejá / miłość / te trzy rzecży ; lecż z nich nájwiękƺá jeſt miłość.
