A o duchownych dárách / bráćia! nie chcę / ábyśćie wiedźieć nie mieli.
Wiećie / iż gdyśćie pogánámi byli / do báłwánów niemych / jáko was wodzono / dáliśćie śię prowádźić.
Przetoż oznájmuję wám / iż nikt przez Duchá Bożego mówiąc / nie rzecże JEzuſá być przeklęſtwem ; y nikt nie może názwáć JEzuſá Pánem / tylko przez Duchá Świętego.
A różneć ſą dáry / ále tenże Duch.
Y różne ſą poſługi / ále tenże Pán.
Y różne ſą ſpráwy / ále tenże Bóg / który ſpráwuje wƺyſtko we wƺyſtkich.
A káżdemu bywá dáne objáwienie Duchá ku pożytkowi.
Abowiem jednemu przez Duchá bywá dáná mowá mądrośći / á drugiemu mowá umiejętnośći przez tegoż Duchá ;
A drugiemu wiárá w tymże Duchu / á drugiemu dár uzdráwiániá w tymże Duchu / á drugiemu cżynienie cudów / á drugiemu proroctwo / á drugiemu rozeznánie duchów.
A drugiemu rozmáite języki / á drugiemu wykłádánie języków.
A to wƺyſtko ſpráwuje jeden á tenże Duch / udźielájąc z oſobná káżdemu / jáko chce.
Abowiem jáko ćiáło jedno jeſt / á cżłonków má wiele / ále wƺyſtkie cżłonki ćiáłá jednego / choć ich wiele jeſt / ſą jednem ćiáłem : ták y CHryſtuſ.
Abowiem przez jednego Duchá my wƺyſcy w jedno ćiáło jeſteśmy ochrzcżeni / bądź Żydowie / bądź Grekowie / bądź niewolnicy / bądź wolni / á wƺyſcy nápojeni jeſteśmy w jednego Duchá.
Abowiem ćiáło nie jeſt jednym cżłonkiem / ále wielomá.
Jeſliby rzekłá nogá : Poniewáżem nie jeſt ręką / nie jeſtem z ćiáłá ; izáli dlátego nie jeſt z ćiáłá?
A jeſliby rzekło ucho : Poniewáżem nie jeſt okiem / nie jeſtem z ćiáłá ; izáli dlátego nie jeſt z ćiáłá?
Jeſliż wƺyſtko ćiáło jeſt okiem / gdźież ſłuch? á jeſliż wƺyſtko ſłuchem / gdźież powonienie?
Ale teraz Bóg ułożył cżłonki / káżdy z nich z oſobná w ćiele / jáko chćiáł.
A jeſliby wƺyſtkie były jednym cżłonkiem / gdźieżby było ćiáło?
Ale teraz / ácż jeſt wiele cżłonków / lecż jedno jeſt ćiáło.
Nie może tedy rzec oko ręce : Nie potrzebuję ćiebie / álbo záś głowá nogom : Nie potrzebuję was.
Y owƺem dáleko więcey cżłonki / które śię zdádzą być nájmdlejƺe w ćiele / potrzebne ſą.
A które mámy zá nájniepocżćiwƺe w ćiele / tym więkƺą pocżćiwość wyrządzámy / á niepocżćiwe cżłonki náƺe obfitƺą pocżćiwość máją.
Bo pocżćiwe cżłonki náƺe tego nie potrzebują ; lecż Bóg ták umiárkowáł ćiáło / dáwƺy cżłonkowi / któremu cżći nie doſtáje / obfitƺą pocżćiwość /
Aby nie było rozerwániá w ćiele / ále iżby jedne cżłonki o drugich jednákież ſtáránie miáły.
A przetoż jeſliże jeden cżłonek ćierpi / ćierpią z nim wƺyſtkie cżłonki ; á jeſli bywá ucżcżony jeden cżłonek / rádują śię z nim wƺyſtkie cżłonki.
Lecż wy jeſteśćie ćiáłem CHryſtuſowem / y cżłonkámi / káżdy z oſobná.
A niektórych Bóg poſtánowił we zborze / nájprzód Apoſtołów / potem proroków / po trzećie náucżyćieli / potem cudotwórców / potem dáry uzdráwiániá / pomocników / rządców / rozmáitość języków.
Izáli wƺyſcy ſą Apoſtołámi? Izáli wƺyſcy prorokámi? Izáli wƺyſcy náucżyćielámi? Izáli wƺyſcy cudotwórcámi?
Izáli wƺyſcy máją dáry uzdráwiániá? Izáli wƺyſcy językámi mówią? Izáli wƺyſcy tłumácżą?
Stárájćie śię uśilnie o lepƺe dáry ; á ja wám jeƺcże zácniejƺą drogę ukáżę.
