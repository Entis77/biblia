A o duchownych <i>dárzech,</i> bráćia / nie chcę ábyśćie wiedźieć nie mieli.
Wiećie iż gdyśćie pogány byli / do báłwanów niemych / jáko was wodzono / <i>dáliśćie śię</i> prowádźić.
Przetoż oznájmuję wam / iż nikt przez Duchá Bożego mówiąc / nie rzecże JEzuſá być przeklęctwem : y nikt nie może názwáć JEzuſá PAnem / tylko przez Duchá Świętego.
A różneć ſą dáry / ále tenże Duch.
Y różne ſą poſługi / ále tenże PAn.
Y różne ſą ſpráwy / ále tenże Bóg który ſpráwuje wƺyſtko we wƺyſtkich.
A káżdemu bywa dáne objáwienie duchá ku pożytkowi.
Abowiem jednemu przez Duchá bywa dána mowá mądrośći ; á drugiemu mowá umiejętnośći / przez tegoż Duchá :
A drugiemu wiárá w tymże Duchu : á drugiemu dar uzdrawiánia / w tymże Duchu : á drugiemu cżynienie cudów : á drugiemu proroctwo : á drugiemu rozeznánie duchów :
A drugiemu rozmájite języki : á drugiemu wykładánie języków.
A to wƺyſtko ſpráwuje jeden á tenże Duch / udźielájąc z oſobná káżdemu jáko chce.
Abowiem jáko ćiáło jedno jeſt / á cżłonków ma wiele ; ále wƺyſtkie cżłonki ćiáłá jednego choć ich wiele jeſt ſą jednym ćiáłem : ták y CHryſtus.
Abowiem przez jednego Duchá my wƺyſcy w jedno ćiáło jeſteſmy ochrzcżeni / bądź Żydowie / bądź Grekowie / bądź niewolnicy / bądź wolni : á wƺyſcy nápojeni jeſteſmy jednym Duchem.
Abowiem ćiáło nie jeſt jednym cżłonkiem / ále wielą.
Jeſliby rzekłá nogá : Ponieważem nie jeſt ręką / nie jeſtem z ćiáłá : Izali dla tego nie jeſt z ćiáłá?
A jeſliby rzekło ucho ; Ponieważem nie jeſt okiem / nie jeſtem z ćiáłá : Izali dla tego nie jeſt z ćiáłá?
Jeſliż wƺyſtko ćiáło jeſt okiem / gdźież ſłuch? <i>A</i> jeſliż wƺyſtko ſłuchem / gdźież powonienie?
Ale teraz Bóg ułożył cżłonki / káżdy z nich z oſobná w ćiele jáko chćiał.
A jeſliby wƺyſtkie były jednym cżłonkiem / gdźieżby było ćiáło.
Ale teraz ácż jeſt wiele cżłonków / lecż jedno jeſt ćiáło.
Niemoże tedy rzec oko ręce ; Nie potrzebuję ćię : Albo záś głowá nogam ; Nie potrzebuję was.
Y owƺem dáleko więcey cżłonki / które śię zdádzą być namdlejƺe w ćiele / potrzebne ſą.
A które mamy zá na niepocżćiwƺe w ćiele / tym więkƺą pocżćiwość wyrządzamy ; á niepocżćiwe <i>cżłonki</i> náƺe / obfitƺą pocżćiwość máją.
Bo pocżćiwe <i>cżłonki</i> náƺe <i>tego</i> nie potrzebują : lecż Bóg ták umiárkował ćiáło / dawƺy <i>cżłonkowi,</i> któremu <i>cżći</i> nie doſtawa / obfitƺą pocżćiwość :
Aby nie było rozerwánia w ćiele ; ále iżby jedne cżłonki o drugich jednákież ſtáránie miáły.
A przetoż jeſliże jeden cżłonek ćierpi / ćierpią z nim wƺyſtkie cżłonki : á jeſli bywa ucżcżony jeden cżłonek / rádują śię z nim wƺyſtkie cżłonki.
Lecż wy jeſteśćie ćiáłem CHryſtuſowym y cżłonkámi káżdy z oſobná.
A niektóre Bóg poſtánowił we Zborze : naprzód Apoſtoły / potym Proroki / po trzećie Náucżyćiele ; potym cudotwórce / potym dáry uzdrawiánia / pomocniki / rządce / rozmájitość języków.
Izali wƺyſcy <i>ſą</i> Apoſtołámi? Izali wƺyſcy Prorokámi? Izali wƺyſcy Náucżyćielámi? Izali wƺyſcy cudotwórcámi?
Izali wƺyſcy máją dáry uzdrawiánia? Izali wƺyſcy językámi mówią? Izali wƺyſcy tłumácżą.
Stárajćie śię uśilnie o lepƺe dáry. A <i>ja</i> wam jeƺcże zacniejƺą drogę ukażę.
