Y ja / bráćia! nie mogłem wám mówić jáko duchownym / ále jáko ćieleſnym / y jáko niemowlątkom w CHryſtuśie.
Nápáwáłem was mlekiem / á nie kármiłem was pokármem ; bośćie jeƺcże nie mogli znieść / owƺem y teraz jeƺcże nie możećie /
Gdyż jeƺcże / ćieleſnymi jeſteśćie. Bo poniewáż między wámi jeſt zázdrość y ſwáry / y rozterki / ázáżeśćie nie ćieleśni y według cżłowieká nie chodźićie?
Abowiem gdy kto mówi : Jám jeſt Páwłowy / á drugi : Jám Apolloſowy / ázáż ćieleſnymi nie jeſteśćie?
Bo któż jeſt Páweł? Kto Apolloſ? jedno ſłudzy / przez którycheśćie uwierzyli / á to jáko káżdemu Pán dáł.
Jám ƺcżepił / Apolloſ polewáł / ále Bóg wzroſt dáł.
A ták / áni ten / co ƺcżepi / jeſt cżem / áni ten / co polewá / ále Bóg / który wzroſt dáje.
Lecż ten / który ƺcżepi / y ten / który polewá / jedno ſą / á káżdy ſwoję zápłátę weźmie według prácy ſwojey.
Abowiem jeſteśmy pomocnikámi Bożymi ; wy Bożą rolą / Bożym budynkiem jeſteśćie.
Według łáſki Bożej / która mi jeſt dáná / jáko mądry budownik záłożyłem grunt / á drugi ná nim buduje : wƺákże káżdy niecháj bácży / jáko ná nim buduje.
Abowiem gruntu innego nikt nie może záłożyć / oprócż tego / który jeſt záłożony / który jeſt JEzuſ CHryſtuſ.
A jeſli kto ná tym grunćie buduje złoto / ſrebro / kámienie drogie / drwá / śiáno / ſłomę /
Káżdego robotá jáwná będźie ; bo to dźień pokáże ; gdyż przez ogień objáwioná będźie / á káżdego roboty / jáká jeſt / ogień doświádcży.
Jeſli cżyjá robotá zoſtánie / którą ná nim budowáł / zápłátę weźmie.
Jeſli cżyjá robotá zgore / ten ƺkodę podejmie ; lecż on ſám będźie záchowány / wƺákże ták jáko przez ogień.
Azáż nie wiećie / iż kośćiołem Bożym jeſteśćie / á Duch Boży mieƺká w was?
A jeſli kto gwáłći kośćiół Boży / tego Bóg ſkáźi / ábowiem kośćiół Boży święty jeſt / którym wy jeſteśćie.
Niechájże nikt ſámego śiebie nie zwodźi ; jeſli śię kto ſobie zdá być mądrym między wámi ná tym świećie / niech śię ſtánie głupim / áby śię ſtáł mądrym.
Abowiem mądrość tego świátá głupſtwem jeſt u Bogá ; bo nápiſáno : Który chwytá mądrych w chytrośći ich ;
Y záśię : Pán zná myśli mądrych / iż ſą márnośćią.
A ták niech śię nikt nie chlubi ludźmi ; ábowiem wƺyſtkie rzecży ſą wáƺe.
Bądź Páweł / bądź Apolloſ / bądź Kiefáſ / bądź świát / bądź żywot / bądź śmierć / bądź przytomne / bądź przyƺłe rzecży / wƺyſtkie ſą wáƺe ;
Aleśćie wy CHryſtuſowi / á CHryſtuſ Boży.
