A oznájmuję wám / bráćiá! Ewángieliję / którąm wám opowiedźiáł / którąśćie też przyjęli / y w której ſtoićie.
Przez którą też zbáwieniá doſtępujećie / jeźli pámiętáćie / jákim ſpoſobem opowiedźiáłem wám / chybá jeźliśćie próżno uwierzyli.
Abowiem náprzód podáłem wám / com też wźiął / iż CHryſtuſ umárł zá grzechy náƺe według Piſm ;
A iż był pogrzebiony / á iż zmártwychwſtáł dniá trzećiego według Piſm.
A iż widźiány jeſt od Kiefáſá / potem od onych dwunáſtu.
Potem widźiány jeſt więcej niż od pięćiu ſet bráći náráz / z których wiele ich zoſtáje áż dotąd / á niektórzy też záſnęli.
Potem jeſt widźiány od Jákóbá / potem od wƺyſtkich Apoſtołów.
A ná oſtátek po wƺyſtkich ukázáł śię y mnie / jáko poronionemu płodowi.
Bom já jeſt nájmniejƺy z Apoſtołów / którym nie jeſt godźien / ábym był zwány Apoſtołem / przeto / żem prześládowáł zbór Boży.
Lecż łáſką Bożą jeſtem tem / cżemem jeſt / á łáſká jego przećiwko mnie dáremná nie byłá ; álem obfićiej / niż oni wƺyſcy / prácowáł / wƺákże nie já / ále łáſká Bożá / którá jeſt zemną.
Przetoż y já y oni ták káżemy / y tákeśćie uwierzyli.
A poniewáż śię o CHryſtuśie káże / iż z mártwych wzbudzony jeſt / jákoż mówią niektórzy między wámi / iż zmártwychwſtániá nie máƺ?
Bo jeźlić zmártwychwſtániá nie máƺ / tedyć y CHryſtuſ nie jeſt wzbudzony.
A jeźlić CHryſtuſ nie jeſt wzbudzony / tedyć dáremne kázánie náƺe / dáremná też wiárá wáƺá.
Y bylibyśmy też ználeźieni fáłƺywymi świádkámi Bożymi / iżeśmy świádcżyli o Bogu / że CHryſtuſá wzbudźił / którego nie wzbudźił / jeźliże umárli nie bywáją wzbudzeni.
Abowiem jeźliż umárli nie bywáją wzbudzeni / y CHryſtuſ nie jeſt wzbudzony.
A jeźli CHryſtuſ nie jeſt wzbudzony / dáremná jeſt wiárá wáƺá / y jeƺcżeśćie w grzechách wáƺych ;
Zátem y ći poginęli / którzy záſnęli w CHryſtuśie.
Bo jeźli tylko w tym żywoćie w CHryſtuśie nádźieję mámy / nád wƺyſtkich ludźi jeſteśmy nájnędzniejƺymi.
Lecż teráz CHryſtuſ z mártwych wzbudzony jeſt / y ſtáł śię pierwiáſtkiem tych / którzy záſnęli.
Bo poniewáż przez cżłowieká śmierć / przez cżłowieká też powſtánie umárłych.
Abowiem jáko w Adámie wƺyſcy umieráją / ták y w CHryſtuśie wƺyſcy ożywieni będą /
Ale káżdy w ſwoim rzędźie / CHryſtuſ jáko pierwiáſtek / á potem ći / co ſą CHryſtuſowi / w przyjśćie jego.
A potem będźie koniec / gdy oddá króleſtwo Bogu y Ojcu / gdy zniƺcży wƺelkie przełożeńſtwo / y wƺelką zwierzchność y moc.
Bo on muśi królowáć / pókiby nie położył wƺyſtkich nieprzyjáćiół pod nogi jego.
A oſtátni nieprzyjáćiel / który będźie zniƺcżony / jeſt śmierć.
Bo wƺyſtkie rzecży poddáł pod nogi jego. A gdy mówi / że mu wƺyſtkie rzecży poddáne ſą / jáwná jeſt / iż oprócż tego / który mu poddáł wƺyſtkie rzecży.
A gdy mu wƺyſtkie rzecży poddáne będą / tedyć też y ſám Syn będźie poddány temu / który mu poddáł wƺyſtkie rzecży / áby był Bóg wƺyſtkim we wƺyſtkim.
Bo inácżej cóż ucżynią ći / którzy śię chrzcżą nád umárłymi / jeźliż zgołá umárli nie bywáją wzbudzeni? przecżże śię chrzcżą nád umárłymi?
Przecż y my niebezpiecżeńſtwá podejmujemy káżdej godźiny?
Ná káżdy dźień umierám przez chwáłę náƺę / którą mám w CHryſtuśie JEzuśie / Pánu náƺym.
Jeźliżem śię obycżájem ludzkim z beſtyjámi w Efeźie potykáł / cóż mám zá pożytek / jeźli umárli nie bywáją wzbudzeni? Jedzmy y pijmy ; boć jutro pomrzemy.
Nie błądźćież ; złe rozmowy pſują dobre obycżáje.
Ocuććież śię ku ſpráwiedliwośći / á nie grzeƺćie ; ábowiem niektórzy nie máją znájomośći Bożej ; ku záwſtydzeniu wáƺemu mówię.
Ale rzecże kto : Jákoż wzbudzeni bywáją umárli / y w jákiem ćiele wychodzą?
O głupi! To / co ty śiejeƺ / nie bywáć ożywione / jeźliby nie umárło.
Y co śiejeƺ / nie śiejeƺ ćiáłá / które má potem wyrość / ále gołe źiárno / jáko śię tráfi / álbo pƺenicżne / álbo jákiekolwiek inne.
Ale Bóg dáje mu ćiáło jáko chce / á káżdemu náśieniu jego włáſne ćiáło.
Nie káżde ćiáło jeſt jednákiem ćiáłem ; ále inƺeć jeſt ćiáło ludzkie / á inƺe ćiáło bydlęce / inƺe rybie / á inƺe ptáƺe.
Y ſą ćiáłá niebieſkie y ćiáłá źiemſkie ; lecż inƺá jeſt chwáłá ćiáł niebieſkich / á inƺá źiemſkich ;
Inƺá chwáłá ſłońcá / á inƺá chwáłá kśiężycá / y inƺá chwáłá gwiázd ; ábowiem gwiázdá od gwiázdy różná jeſt w jáſnośći.
Tákći będźie y powſtánie umárłych. Bywá wśiáne ćiáło w ſkáźitelnośći / á będźie wzbudzone w nieſkáźitelnośći.
Bywá wśiáne w nieſłáwie / á będźie wzbudzone w ſłáwie ; bywá wśiáne w ſłábośći / á będźie wzbudzone w mocy ; bywá wśiáne ćiáło ćieleſne / á będźie wzbudzone ćiáło duchowne.
Jeſt ćiáło ćieleſne / jeſt też ćiáło duchowne.
Tákći też nápiſáne : Stáł śię pierwƺy cżłowiek Adám w duƺę żywą / ále pośledni Adám w duchá ożywiájącego.
Wƺákże nie jeſt pierwƺe duchowne / ále ćieleſne / potem duchowne.
Pierwƺy cżłowiek z źiemi źiemſki ; wtóry cżłowiek ſám Pán z niebá.
Jáki jeſt ten źiemſki / tácy też y źiemſcy ; á jáki jeſt niebieſki / tácy też będą niebieſcy.
A jákośmy nośili wyobráżenie źiemſkiego / ták będźiemy nośili wyobráżenie niebieſkiego.
To jednák powiádám / bráćiá! iż ćiáło y krew króleſtwá Bożego odźiedźicżyć nie mogą ; áni ſkáźitelność nie odźiedźicży nieſkáźitelnośći.
Oto tájemnicę wám powiádám ; nie wƺyſcyć záśniemy / ále wƺyſcy przemienieni będźiemy / bárdzo prędko w okámgnieniu / ná trąbę oſtátecżną.
Abowiem zátrąbi / á umárli wzbudzeni będą nieſkáźitelni / á my będźiemy przemienieni.
Boć muśi to / co jeſt ſkáźitelnego / przyoblec nieſkáźitelność / y co jeſt śmiertelnego / przyoblec nieśmiertelność.
A gdy to / co jeſt ſkáźitelnego / przyoblecże nieſkáźitelność / y to / co jeſt śmiertelnego / przyoblecże nieśmiertelność / tedy śię wypełni ono ſłowo / które nápiſáne : Połknioná jeſt śmierć w zwyćięſtwie.
Gdźież jeſt / o śmierći! bodźiec twój? Gdźież jeſt / piekło! zwyćięſtwo twoje?
Lecż bodźiec śmierći jeſt grzech / á moc grzechu jeſt zákon.
Ale niech będą Bogu dźięká / który nám dáł zwyćięſtwo przez Páná náƺego JEzuſá CHryſtuſá.
A ták / bráćiá moi mili! bądźćie mocni / nieporuƺeni / obfitujący w ucżynku Páńſkim záwƺe / wiedząc / iż prácá wáƺá nie jeſt dáremná w Pánu.
