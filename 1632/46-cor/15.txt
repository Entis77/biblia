A oznájmuję wám / bráćiá! Ewángieliję / którąm wám opowiedziáł / którąśćie też przyjęli / y w której ſtoićie.
Przez którą też zbáwieniá doſtępujećie / jeźli pámiętáćie / jákim ſpoſobem opowiedziáłem wám / chybá jeźliśćie próżno uwierzyli.
Abowiem náprzód podáłem wám / com też wziął / iż Chryſtuſ umárł zá grzechy náƺe według Piſm ;
A iż był pogrzebiony / á iż zmártwychwſtáł dniá trzećiego według Piſm.
A iż widziány jeſt od Kiefáſá / potem od onych dwunáſtu.
Potem widziány jeſt więcej niż od pięćiu ſet bráći náráz / z których wiele ich zoſtáje áż dotąd / á niektórzy też záſnęli.
Potem jeſt widziány od Jákóbá / potem od wƺyſtkich Apoſtołów.
A ná oſtátek po wƺyſtkich ukázáł śię y mnie / jáko poronionemu płodowi.
Bom já jeſt nájmniejƺy z Apoſtołów / którym nie jeſt godzien / ábym był zwány Apoſtołem / przeto / żem prześládowáł zbór Boży.
Lecż łáſką Bożą jeſtem tem / cżemem jeſt / á łáſká jego przećiwko mnie dáremná nie byłá ; álem obfićiej / niż oni wƺyſcy / prácowáł / wƺákże nie já / ále łáſká Bożá / którá jeſt zemną.
Przetoż y já y oni ták káżemy / y tákeśćie uwierzyli.
A poniewáż śię o Chryſtuśie káże / iż z mártwych wzbudzony jeſt / jákoż mówią niektórzy między wámi / iż zmártwychwſtániá nie máƺ?
Bo jeźlić zmártwychwſtániá nie máƺ / tedyć y Chryſtuſ nie jeſt wzbudzony.
A jeźlić Chryſtuſ nie jeſt wzbudzony / tedyć dáremne kázánie náƺe / dáremná też wiárá wáƺá.
Y bylibyśmy też ználezieni fáłƺywymi świádkámi Bożymi / iżeśmy świádcżyli o Bogu / że Chryſtuſá wzbudził / którego nie wzbudził / jeźliże umárli nie bywáją wzbudzeni.
Abowiem jeźliż umárli nie bywáją wzbudzeni / y Chryſtuſ nie jeſt wzbudzony.
A jeźli Chryſtuſ nie jeſt wzbudzony / dáremná jeſt wiárá wáƺá / y jeƺcżeśćie w grzechách wáƺych ;
Zátem y ći poginęli / którzy záſnęli w Chryſtuśie.
Bo jeźli tylko w tym żywoćie w Chryſtuśie nádzieję mámy / nád wƺyſtkich ludzi jeſteśmy nájnędzniejƺymi.
Lecż teráz Chryſtuſ z mártwych wzbudzony jeſt / y ſtáł śię pierwiáſtkiem tych / którzy záſnęli.
Bo poniewáż przez cżłowieká śmierć / przez cżłowieká też powſtánie umárłych.
Abowiem jáko w Adámie wƺyſcy umieráją / ták y w Chryſtuśie wƺyſcy ożywieni będą /
Ale káżdy w ſwoim rzędzie / Chryſtuſ jáko pierwiáſtek / á potem ći / co ſą Chryſtuſowi / w przyjśćie jego.
A potem będzie koniec / gdy oddá króleſtwo Bogu y Ojcu / gdy zniƺcży wƺelkie przełożeńſtwo / y wƺelką zwierzchność y moc.
Bo on muśi królowáć / pókiby nie położył wƺyſtkich nieprzyjáćiół pod nogi jego.
A oſtátni nieprzyjáćiel / który będzie zniƺcżony / jeſt śmierć.
Bo wƺyſtkie rzecży poddáł pod nogi jego. A gdy mówi / że mu wƺyſtkie rzecży poddáne ſą / jáwná jeſt / iż oprócż tego / który mu poddáł wƺyſtkie rzecży.
A gdy mu wƺyſtkie rzecży poddáne będą / tedyć też y ſám Syn będzie poddány temu / który mu poddáł wƺyſtkie rzecży / áby był Bóg wƺyſtkim we wƺyſtkim.
Bo inácżej cóż ucżynią ći / którzy śię chrzcżą nád umárłymi / jeźliż zgołá umárli nie bywáją wzbudzeni? przecżże śię chrzcżą nád umárłymi?
Przecż y my niebezpiecżeńſtwá podejmujemy káżdej godziny?
Ná káżdy dzień umierám przez chwáłę náƺę / którą mám w Chryſtuśie Jezuśie / Pánu náƺym.
Jeźliżem śię obycżájem ludzkim z beſtyjámi w Efezie potykáł / cóż mám zá pożytek / jeźli umárli nie bywáją wzbudzeni? Jedzmy y pijmy ; boć jutro pomrzemy.
Nie błądźćież ; złe rozmowy pſują dobre obycżáje.
Ocuććież śię ku ſpráwiedliwośći / á nie grzeƺćie ; ábowiem niektórzy nie máją znájomośći Bożej ; ku záwſtydzeniu wáƺemu mówię.
Ale rzecże kto : Jákoż wzbudzeni bywáją umárli / y w jákiem ćiele wychodzą?
O głupi! To / co ty śiejeƺ / nie bywáć ożywione / jeźliby nie umárło.
Y co śiejeƺ / nie śiejeƺ ćiáłá / które má potem wyrość / ále gołe ziárno / jáko śię tráfi / álbo pƺenicżne / álbo jákiekolwiek inne.
Ale Bóg dáje mu ćiáło jáko chce / á káżdemu náśieniu jego włáſne ćiáło.
Nie káżde ćiáło jeſt jednákiem ćiáłem ; ále inƺeć jeſt ćiáło ludzkie / á inƺe ćiáło bydlęce / inƺe rybie / á inƺe ptáƺe.
Y ſą ćiáłá niebieſkie y ćiáłá ziemſkie ; lecż inƺá jeſt chwáłá ćiáł niebieſkich / á inƺá ziemſkich ;
Inƺá chwáłá ſłońcá / á inƺá chwáłá kśiężycá / y inƺá chwáłá gwiázd ; ábowiem gwiázdá od gwiázdy różná jeſt w jáſnośći.
Tákći będzie y powſtánie umárłych. Bywá wśiáne ćiáło w ſkázitelnośći / á będzie wzbudzone w nieſkázitelnośći.
Bywá wśiáne w nieſłáwie / á będzie wzbudzone w ſłáwie ; bywá wśiáne w ſłábośći / á będzie wzbudzone w mocy ; bywá wśiáne ćiáło ćieleſne / á będzie wzbudzone ćiáło duchowne.
Jeſt ćiáło ćieleſne / jeſt też ćiáło duchowne.
Tákći też nápiſáne : Stáł śię pierwƺy cżłowiek Adám w duƺę żywą / ále pośledni Adám w duchá ożywiájącego.
Wƺákże nie jeſt pierwƺe duchowne / ále ćieleſne / potem duchowne.
Pierwƺy cżłowiek z ziemi ziemſki ; wtóry cżłowiek ſám Pán z niebá.
Jáki jeſt ten ziemſki / tácy też y ziemſcy ; á jáki jeſt niebieſki / tácy też będą niebieſcy.
A jákośmy nośili wyobráżenie ziemſkiego / ták będziemy nośili wyobráżenie niebieſkiego.
To jednák powiádám / bráćiá! iż ćiáło y krew króleſtwá Bożego odziedzicżyć nie mogą ; áni ſkázitelność nie odziedzicży nieſkázitelnośći.
Oto tájemnicę wám powiádám ; nie wƺyſcyć záśniemy / ále wƺyſcy przemienieni będziemy / bárdzo prędko w okámgnieniu / ná trąbę oſtátecżną.
Abowiem zátrąbi / á umárli wzbudzeni będą nieſkázitelni / á my będziemy przemienieni.
Boć muśi to / co jeſt ſkázitelnego / przyoblec nieſkázitelność / y co jeſt śmiertelnego / przyoblec nieśmiertelność.
A gdy to / co jeſt ſkázitelnego / przyoblecże nieſkázitelność / y to / co jeſt śmiertelnego / przyoblecże nieśmiertelność / tedy śię wypełni ono ſłowo / które nápiſáne : Połknioná jeſt śmierć w zwyćięſtwie.
Gdzież jeſt / o śmierći! bodziec twój? Gdzież jeſt / piekło! zwyćięſtwo twoje?
Lecż bodziec śmierći jeſt grzech / á moc grzechu jeſt zákon.
Ale niech będą Bogu dzięká / który nám dáł zwyćięſtwo przez Páná náƺego Jezuſá Chryſtuſá.
A ták / bráćiá moi mili! bądźćie mocni / nieporuƺeni / obfitujący w ucżynku Páńſkim záwƺe / wiedząc / iż prácá wáƺá nie jeſt dáremná w Pánu.
