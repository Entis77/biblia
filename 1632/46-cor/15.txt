A oznámuję wam / bráćia / Ewángelią którąm wam opowiedźiał / którąśćie też przyjęli / y w którey ſtojićie.
Przez którą też zbáwienia doſtępujećie ; jeſli pámiętaćie / jákim ſpoſobem opowiedźiałem wam / chybá jeſliśćie prózno uwierzyli.
Abowiem naprzód podałem wam / com też wźiął : Iż CHryſtus umárł zá grzechy náƺe według piſm.
A iż był pogrzebiony / á iż zmartwychwſtał dniá trzećiego / według piſm.
A iż widźiány jeſt od Cefáſá / potym od onych dwunaśći.
Potym widźiány jeſt więcey niż od piąći ſet bráći oraz : z których wiele ich zoſtawa áż do tąd ; á niektórzy też záſnęli.
Potym jeſt widźiány od Jákubá / potym od wƺyſtkich Apoſtołów.
A ná oſtátek po wƺyſtkich ukazał śię y mnie / jáko poronionemu płodowi /
Bom ja jeſt namniejƺy z Apoſtołów / którym nie jeſt godźien / ábym był zwány Apoſtołem ; przeto / żem prześládował Zbór Boży.
Lecż łáſką Bożą jeſtem tym / cżymem jeſt : á łáſká jego przećiwko mnie dáremna nie byłá / álem obfićiey niż oni wƺyſcy prácował : wƺákże nie ja / ále łáſká Boża która <i>jeſt</i> ze mną.
Przetoż y ja / y oni / ták każemy / y tákeśćie uwierzyli.
A ponieważ śię o CHryſtuśie każe / iż z martwych wzbudzony jeſt ; jákoż mówią niektórzy miedzy wámi / iż zmartwychwſtánia niemáƺ?
Bo jeſlić zmartwychwſtánia niemáƺ / tedyć y CHryſtus nie jeſt wzbudzony.
A jeſlić CHryſtus nie jeſt wzbudzony / tedyć dáremne kazánie náƺe / dáremna też y wiárá wáƺá.
Y bylibyſmy też ználeźieni fałƺywymi świádkámi Bożymi / iżeſmy świadcżyli o Bogu / że CHryſtuſá wzbudźił : którego nie wzbudźił / jeſliże umárli nie bywáją wzbudzeni.
Abowiem jeſliż umárli nie bywáją wzbudzeni / y CHryſtus nie jeſt wzbudzony.
A jeſli CHryſtus nie jeſt wzbudzony / dáremna <i>jeſt</i> wiárá wáƺá / <i>y</i> jeƺcżeśćie w grzechách wáƺych.
Zátym y ći poginęli / którzy záſnęli w CHryſtuśie.
Bo jeſli tylko w tym żywoćie w CHryſtuśie nádźieję mamy / nád wƺyſtkie ludźie jeſteſmy nanędzniejƺymi.
Lecż teraz CHryſtus z martwych wzbudzony jeſt / <i>y</i> ſtał śię pierwiaſtkámi tych którzy záſnęli.
Bo ponieważ przez cżłowieká śmierć / przez cżłowieká też powſtánie umárłych.
Abowiem jáko w Adámie wƺyſcy umieráją / ták y w CHryſtuśie wƺyſcy ożywieni będą.
Ale káżdy w ſwojim rzędźie. CHryſtus jáko pierwiaſtki / á potym ći co ſą CHryſtuſowi / w przyśćie jego.
<i>A</i> potym będźie koniec / gdy odda króleſtwo Bogu y Ojcu ; gdy zniƺcży wƺelkie przełożeńſtwo / y wƺelką zwierzchność / y moc.
Bo on muśi królowáć / pókiby nie położył wƺyſtkich nieprzyjaćiół pod nogi jego.
A oſtátni nieprzyjaćiel który będźie zniƺcżony / <i>jeſt</i> śmierć.
Bo wƺyſtkie rzecży poddał pod nogi jego. A gdy mówi / że <i>mu</i> wƺyſtkie rzecży poddáne ſą / jáwna <i>jeſt,</i> iż oprócż tego / który mu poddał wƺyſtkie rzecży.
A gdy mu wƺyſtkie rzecży poddáne będą / tedyć też y ſam Syn będźie poddány temu który mu poddał wƺyſtkie rzecży : áby był Bóg wƺyſtkim we wƺyſtkich.
Bo inácżey cóż ucżynią ći którzy śię chrzcżą zá umárłe? Jeſliż zgołá umárli nie bywáją wzbudzeni? Przecżże śię chrzcżą zá umárłe?
Przecż y my niebeſpiecżeńſtwá podejmujemy káżdey godźiny.
Ná káżdy dźień umieram / przez chwałę náƺę którą mam w JEzuśie CHryſtuśie PAnu náƺym.
Jeſliżem śię obycżájem ludzkim z beſtyámi w Efeźie potykał / cóż mam zá pożytek jeſli umárli nie bywáją wzbudzeni? Jedzmy y pijmy / boć jutro pomrzemy.
Nie błądźćież ; Złe rozmowy pſują dobre obycżáje.
Ocuććież śię ku ſpráwiedliwośći / á nie grzeƺćie : ábowiem niektórzy nie máją znájomośći Bożey ; ku záwſtydzeniu wáƺemu mówię.
Ale rzecże kto : Jákoż wzbudzeni bywáją umárli / y w jákim ćiele wychodzą?
O głupi! To co ty śiejeƺ / nie bywać ożywiono / jeſliby nie umárło.
Y co śiejeƺ / nie śiejeƺ ćiáłá które ma potym wyrość / ále gołe źiárno / jáko śię tráfi álbo pƺenicżne / álbo jákiekolwiek inne.
Ale Bóg dáje mu ćiáło jáko chce : á káżdemu naśieniu jego właſne ćiáło.
Nie káżde ćiáło jeſt jednákim ćiáłem : ále inƺeć jeſt ćiáło ludzkie / á inƺe ćiáło bydlęce / inƺe rybie / á inƺe ptáƺe.
Y <i>ſą</i> ćiáłá niebieſkie / y ćiáłá źiemſkie / lecż inƺa jeſt chwałá <i>ćiał</i> niebieſkich / á inƺa źiemſkich.
Inƺa chwałá ſłońcá / á inƺa chwałá kśiężycá / y inƺa chwałá gwiazd. Abowiem gwiazdá od gwiazdy rózna jeſt w jáſnośći.
Tákći będźie y powſtánie umárłych. Bywa wśiane <i>ćiáło</i> w ſkáźitelnośći ; á będźie wzbudzone w nieſkáźitelnośći.
Bywa wśiane w nieſławie ; á będźie wzbudzone w ſławie. Bywa wśiane w ſłábośći ; będźie wzbudzone w mocy. Bywa wśiane ćiáło ćieleſne ; będźie wzbudzone ćiáło duchowne.
Jeſt ćiáło ćieleſne / jeſt też ćiáło duchowne.
Tákći też nápiſano : Stał śię pierwƺy cżłowiek Adam w duƺę żywą ; <i>ále</i> pośledni Adam w duchá ożywiájącego.
Wƺákże nie jeſt pierwƺe duchowne / ále ćieleſne ; potym duchowne.
Pierwƺy cżłowiek z źiemie / źiemſki ; wtóry cżłowiek <i>ſám</i> PAN z niebá.
Jáki jeſt ten źiemſki / tácy też y źiemſcy ; á jáki jeſt niebieſki / tácy też <i>będą</i> niebieſcy.
A jákoſmy nośili wyobráżenie źiemſkiego ; ták będźiemy nośili wyobráżenie niebieſkiego.
To jednák powiádam bráćia / iż ćiáło y krew króleſtwá Bożego odźiedźicżyć nie mogą ; áni ſkáźitelność nie odźiedźicży nieſkáźitelnośći.
Oto tájemnicę wam powiádam : Nie wƺyſcyć záſniemy / ále wƺyſcy przemienieni będźiemy : Bárzo prętko w oká mgnieniu / ná trąbę oſtátecżną.
Abowiem zátrąbi ; á umárli wzbudzeni będą nieſkáźitelni / á my będźiemy przemienieni.
Boć muśi to co jeſt ſkáźitelnego przyoblec nieſkáźitelność : y co jeſt śmiertelnego / przyoblec nieśmiertelność.
A gdy to co jeſt ſkáźitelnego przyoblecże nieſkáźitelność / y to co jeſt śmiertelnego przyoblecże nieśmiertelność / tedy śię wypełni ono ſłowo które nápiſano ; Połkniona jeſt śmierć w zwyćięſtwie.
Gdźież <i>jeſt</i> o śmierći bodźiec twój? Gdźież jeſt piekło zwyćięſtwo twoje?
Lecż bodźiec śmierći <i>jeſt</i> grzech / á moc grzechu jeſt Zakon.
Ale <i>niech będźie</i> Bogu dźięká / który nam dał zwyćięſtwo przez PAná náƺego JEzuſá CHryſtuſá.
A ták bráćia moji mili / bądźćie mocni / nieporuƺeni / obfitujący w ucżynku PAńſkim záwƺe ; wiedząc iż pracá wáƺá nie jeſt dáremna w PAnu.
