Lecż o tym cośćie mi piſáli : Dobrzećby cżłowiekowi nie tykáć śię niewiáſty.
Ale dla <i>uwárowánia śię</i> wƺetecżeńſtwá / niech káżdy ma ſwoję właſną żonę / á káżda niech ma ſwego właſnego mężá.
Mąż niech żenie powinną chęć oddáje ; tákże też y żoná mężowi.
Żoná właſnego ćiáłá ſwego w mocy nie ma / ále mąż : tákże też y mąż właſnego ćiáłá ſwego w mocy nie ma / ále żoná.
Nie oƺukawajćie jeden drugiego ; chybáby z ſpólnego zezwolenia do cżáſu / ábyśćie śię uwolnili do poſtu y do modlitwy : á záśię weſpół śię zchodźćie / áby was ƺátan nie kuśił dla wáƺey niepowśćiągliwośći.
Ale to mówię jáko pozwalájąc / á nie jáko rozkázując.
Abowiem chćiałbym / áby wƺyſcy ludźie ták byli / jáko y ja : áleć káżdy ma ſwój właſny dar od Bogá ; jeden ták / á drugi owák.
A mówię nieżonátym y wdowam : dobrze im jeſt / jeſliby ták zoſtáli / jáko y ja.
Ale jeſli śię wſtrzymáć nie mogą / niechże w ſtan małżeńſki wſtąpią. Boć lepiey w ſtan małżeńſki wſtąpić / niż upalenie ćierpieć.
Tym záśię którzy ſą w ſtanie małżeńſkim rozkázuję nie ja / ále PAn / <i>mówiąc</i> ; Żeby śię żoná od mężá nie odłącżáłá.
Ale jeſliby śię też odłącżyłá / niechajże zoſtawa bez mężá / álbo śię niech z mężem pojedna. A mąż żony niechaj nie opuƺcża.
A inƺym záśię ja mówię / <i>á</i> nie PAn : Jeſli który brát ma żonę niewierną / á tá z nim przyzwala mieƺkáć / niechże jey nie opuƺcża.
A <i>jeśli</i> która żoná mężá niewiernego ma / á on przyzwala z nią mieƺkáć / niechże go nie opuƺcża.
Abowiem poświęcony jeſt mąż niewierny przez żonę ; y żoná niewierná poświęcona jeſt przez mężá : bo inácżey dźiatki wáƺe byłyby niecżyſtymi ; lecż teraz świętymi ſą.
A jeſli ten co jeſt niewierny chce śię odłącżyć / niechże śię odłącży. Abowiem nie jeſt niewolnikiem brát álbo śioſtrá w tákowych <i>rzecżách</i> : áleć ku pokojowi nas Bóg powołał.
Abowiem co <i>ty</i> wieƺ żono / jeſli pozyſkaƺ mężá? Albo co <i>ty</i> wieƺ mężu / pozyſkaƺli żonę?
Jednák jáko káżdemu udźielił Bóg / jáko káżdego powołał PAn / ták niech poſtępuje. A tákći we wƺyſtkich Zborách ſtánowię.
Obrzezánym kto powołány jeſt ? niechaj nie wprowadza <i>ná śię nieobrzezki</i>. A w nieobrzezce jeſt kto powołány? Niech śię nie obrzezuje.
Obrzezká nic nie jeſt / tákże nieobrzezká nic nie jeſt : ále záchowywánie przykazań Bożych.
Káżdy w tym powołániu / w którym powołány jeſt / niech zoſtawa.
Jeſteś powołány niewolnikiem / nie dbajże <i>ná to</i> : ále jeſli też możeƺ być wolnym / rácżey <i>wolnośći</i> używaj.
Abowiem kto w PAnu powołány jeſt niewolnikiem / wolny jeſt w PAnu : tákże też kto jeſt powołány wolnym / niewolnikiem jeſt CHryſtuſowym.
Drogośćie kupieni ; nie bądźćie niewolnikámi ludzkimi.
Káżdy tedy jákim jeſt powołány / bráćia / tákim niechaj zoſtawa przed Bogiem.
A o Pánnách rozkazánia PAńſkiego nie mam : wƺákże rádę dawam jáko <i>ten</i> , któremu PAN z miłośierdźia <i>ſwego</i> dał / áby był wiernym.
Mniemam tedy / że to jeſt rzecż dobra dla náſtępującey potrzeby : że jeſt rzecż dobra / cżłowiekowi ták być.
Przywiązałeś śię do żony : nie ƺukajże rozwiązánia : rozwiązányś od żony ? nie ƺukajże żony.
A jeſlibyś śię ożenił / nie zgrzeƺyłeś : jeſliby też Pánná ƺłá zá mąż / nie zgrzeƺyłá. Wƺákże utrapienie w ćiele tákowi mieć będą : lecż ja was ƺánuję.
A toć mówię bráćia / ponieważ cżás potomny ukrócony jeſt : áby y ći którzy żony máją byli / jákoby ich nie mieli.
A którzy płácżą / jákoby nie płákáli : á którzy śię rádują / jákoby śię nie rádowáli : á którzy kupują / jákoby nie trzymáli.
A którzy używáją tego świátá / jákoby źle nie używáli : bo przemija kƺtałt tego świátá.
A chcę ábyśćie wy byli bez kłopotu. Bo kto nie ma żony / ſtára śię o rzecży PAńſkie / jákoby śię podobał PAnu.
Ale kto śię ożenił / ſtára śię o <i>rzecży</i> tego świátá / jákoby śię podobał żenie.
Jeſt różność miedzy mężátką y pánną : Która nie ƺłá zá mąż / ſtára śię o <i>rzecży</i> PAńſkie / áby byłá święta y ćiáłem y duchem : ále która ƺłá zá mąż / ſtára śię o rzecży tego świátá / jákoby śię podobáłá mężowi.
A toć mówię ku dobremu wáƺemu : nie ábym śidło ná was wrzućił ; ále ábyśćie ſłuƺnie y przyſtojnie <i>ſtáli przy</i> PAnu bez rozerwánia.
A jeſli kto mniema / że nie przyſtojnie ſobie pocżyna z pánną ſwoją / gdyby z lat ſwojich wyƺłá / y do tegoby <i>jey</i> przyƺło : co chce niechaj cżyni / nie grzeƺy : niechże idźie zá mąż.
Ale kto ſtátecżnie poſtánowił w ſercu <i>ſwym</i> potrzeby tego nie májąc / lecż ma w mocy właſną <i>ſwoję</i> wolą / y to uſądźił w ſercu ſwym / áby záchował pánnę ſwoję / dobrze cżyni.
A ták ten kto dawa zá mąż ; dobrze cżyni : ále który nie dawa zá mąż / lepiey cżyni.
Żoná związána jeſt Zakonem / póki żywie mąż jey : á jeſliby umárł mąż jey / wolna jeſt / áby ƺłá zá kogo chce ; tylko w PAnu.
Ale ƺcżeśliwƺa jeſt / jeſliby ták zoſtáłá według rády mojey. A mniemam że y ja mam Duchá Bożego.
