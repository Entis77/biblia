Lecż o tem / cośćie mi piſáli : Dobrzećby cżłowiekowi / nie tykáć śię niewiáſty ;
Ale dlá uwárowániá śię wƺetecżeńſtwá niech káżdy má ſwoję włáſną żonę / á káżdá niech má ſwego włáſnego mężá.
Mąż niech żonie powinną chęć oddáje / tákże też y żoná mężowi.
Żoná włáſnego ćiáłá ſwego w mocy nie má / ále mąż ; tákże też y mąż włáſnego ćiáłá ſwego w mocy nie má / ále żoná.
Nie oƺukiwájćie jeden drugiego ; chybáby z ſpólnego zezwoleniá do cżáſu / ábyśćie śię uwolnili do poſtu y do modlitwy ; á záśię weſpół śię ſchodźćie / áby wáſ ƺátán nie kuśił dlá wáƺej niepowśćiągliwośći.
Ale to mówię jáko pozwálájąc / á nie jáko rozkázując.
Abowiem chćiáłbym / áby wƺyſcy ludźie ták byli jáko y já ; áleć káżdy má ſwój włáſny dár od Bogá / jeden ták á drugi owák.
A mówię nieżonátym y wdowom : Dobrze im jeſt / jeźliby ták zoſtáli / jáko y já.
Ale jeźli śię wſtrzymáć nie mogą / niechże w ſtán máłżeńſki wſtąpią ; boć lepiej w ſtán máłżeńſki wſtąpić / niż upálenie ćierpieć.
Tym záśię / którzy ſą w ſtánie máłżeńſkim / rozkázuję nie já / ále Pán / mówiąc : Żeby śię żoná od mężá nie odłącżáłá.
Ale jeźliby śię też odłącżyłá / niechájże zoſtáje bez mężá / álbo śię niech z mężem pojedná / á mąż żony niecháj nie opuƺcżá.
A inƺym záśię já mówię / á nie Pán : Jeźli który brát má żonę niewierną / á tá z nim przyzwálá mieƺkáć / niechże jej nie opuƺcżá.
A jeźli którá żoná mężá niewiernego má / á on przyzwálá z nią mieƺkáć / niechże go nie opuƺcżá.
Abowiem poświęcony jeſt mąż niewierny przez żonę / y żoná niewierná poświęconá jeſt przez mężá ; bo inácżej dźiátki wáƺe byłyby niecżyſtemi / lecż teráz świętymi ſą.
A jeźli ten / co jeſt niewierny / chce śię odłącżyć / niechże śię odłącży ; ábowiem nie jeſt niewolnikiem brát álbo śioſtrá w tákowych rzecżách ; áleć ku pokojowi náſ Bóg powołáł.
Abowiem co ty wieƺ / żono! jeźli pozyſkáƺ mężá? Albo co ty wieƺ / mężu! pozyſkáƺli żonę?
Jednák jáko káżdemu udźielił Bóg / jáko káżdego powołáł Pán / ták niech poſtępuje ; á tákći we wƺyſtkich zborách ſtánowię.
Obrzezánym kto powołány jeſt / niecháj nie wprowádzá ná śię nieobrzezki ; á w nieobrzezce kto jeſt powołány / niech śię nie obrzezuje.
Obrzezká nic nie jeſt / tákże nieobrzezká nic nie jeſt ; ále záchowywánie przykázáń Bożych.
Káżdy w tem powołániu / w którym powołány jeſt / niech zoſtáje.
Jeſteś powołány niewolnikiem / nie dbájże ná to ; ále jeźli też możeƺ być wolnym / rácżej wolnośći używáj.
Abowiem kto w Pánu powołány jeſt niewolnikiem / wolny jeſt w Pánu ; tákże też / kto jeſt powołány wolnym / niewolnikiem jeſt CHryſtuſowym.
Drogośćie kupieni ; nie bądźćie niewolnikámi ludzkimi /
Káżdy tedy / jákim jeſt powołány / bráćiá! tákim niecháj zoſtáje przed Bogiem.
A o pánnách rozkázániá Páńſkiego nie mám ; wƺákże rádę dáję / jáko ten / któremu Pán z miłośierdźiá ſwego dáł / áby był wiernym.
Mniemám tedy / że to jeſt rzecż dobrá dlá náſtępującej potrzeby / że jeſt rzecż dobrá cżłowiekowi ták być.
Przywiązáłeś śię do żony / nie ƺukájże rozwiązániá ; rozwiązányś od żony / nie ƺukájże żony.
A jeźlibyś śię ożenił / nie zgrzeƺyłeś ; jeźliby też pánná ƺłá zá mąż / nie zgrzeƺyłá ; wƺákże utrápienie w ćiele tákowi mieć będą ; lecż já wáſ ƺánuję.
A toć mówię / bráćiá! poniewáż cżáſ potomny ukrócony jeſt / áby y ći / którzy żony máją / byli / jákoby ich nie mieli ;
A którzy płácżą / jákoby nie płákáli ; á którzy śię rádują / jákoby śię nie rádowáli ; á którzy kupują / jákoby nie trzymáli ;
A którzy używáją tego świátá / jákoby źle nie używáli ; bo przemijá kƺtáłt tego świátá.
A chcę / ábyśćie wy byli bez kłopotu / bo kto nie má żony / ſtárá śię o rzecży Páńſkie / jákoby śię podobáł Pánu ;
Ale kto śię ożenił / ſtárá śię o rzecży tego świátá / jákoby śię podobáł żonie.
Jeſt różność między mężátką y pánną ; którá nie ƺłá zá mąż / ſtárá śię o rzecży Páńſkie / áby byłá świętá y ćiáłem y duchem ; ále którá ƺłá zá mąż / ſtárá śię o rzecży tego świátá / jákoby śię podobáłá mężowi.
A toć mówię ku dobremu wáƺemu ; nie ábym śidło ná wáſ wrzućił / ále ábyśćie ſłuƺnie y przyſtojnie ſtáli przy Pánu bez rozerwániá.
A jeźli kto mniemá / że nie przyſtojnie ſobie pocżyná z pánną ſwoją / gdyby z lát ſwoich wyƺłá / y do tegoby jej przyƺło / co chce / niecháj cżyni / nie grzeƺy ; niechże idźie zá mąż.
Ale kto ſtátecżnie poſtánowił w ſercu ſwem / potrzeby tego nie májąc / lecż má w mocy włáſną ſwoję wolę / y to uſądźił w ſercu ſwem / áby záchowáł pánnę ſwoję / dobrze cżyni.
A ták ten / kto dáje zá mąż / dobrze cżyni ; ále który nie dáje zá mąż / lepiej cżyni.
Żoná związáná jeſt zákonem / póki żyje mąż jej ; á jeźliby umárł mąż jej / wolná jeſt / áby ƺłá / zá kogo chce / tylko w Pánu.
Ale ƺcżęśliwƺá jeſt / jeźliby ták zoſtáłá według rády mojej ; á mniemám / że y já mám Duchá Bożego.
