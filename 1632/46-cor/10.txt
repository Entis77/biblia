A nie chcę / ábyśćie nie mieli wiedźieć bráćia! iż ojcowie náśi wƺyſcy pod obłokiem byli / y wƺyſcy przez morze przeƺli ;
Y wƺyſcy w Mojżeƺá ochrzcżeni ſą w obłoku y w morzu ;
Y wƺyſcy tenże pokárm duchowny jedli ;
Y wƺyſcy tenże nápój duchowny pili ; ábowiem pili z opoki duchowney / która zá nimi ƺłá ; á tą opoką był CHryſtuſ.
Lecż więkƺej cżęśći z nich nie upodobáł ſobie Bóg ; ábowiem polegli ná puƺcży.
A te rzecży ſtáły śię nám zá wzór ná to / ábyśmy złych rzecży nie pożądáli / jáko y oni pożądáli
Nie bądźćież tedy báłwochwálcámi / jáko niektórzy z nich / ták jáko nápiſáno : Siádł lud / áby jádł y pił / y wſtáli gráć.
Ani śię dopuƺcżájmy wƺetecżeńſtwá / jáko śię niektórzy z nich wƺetecżeńſtwá dopuƺcżáli / y pádło ich jednego dniá dwádźieśćiá y trzy tyśiące.
Ani kuśmy CHryſtuſá / jáko niektórzy z nich kuśili / y od wężów poginęli.
Ani ƺemrzyjćie / jáko niektórzy z nich ƺemráli / y poginęli od tego / który wytrácá.
A te wƺyſtkie rzecży przydáły śię im zá wzór / á nápiſáne ſą dla nápomnieniá náƺego / ná których koniec świátá przyƺedł.
A ták kto mniemá / że ſtoi / niechże pátrzy / áby nie upádł.
Pokuƺenie śię was nie jęło / tylko ludzkie ; ále wiernyć jeſt Bóg / który nie dopuśći / ábyśćie byli kuƺeni nád możność wáƺę / ále ucżyni z pokuƺeniem y wyjśćie / ábyśćie znośić mogli.
Przetoż / nájmilśi moi! ućiekájćie przed báłwochwálſtwem.
Jáko mądrym mówię ; rozſądźćie wy / co mówię.
Kielich błogoſłáwieniá / który błogoſłáwimy / izáli nie jeſt ſpołecżnośćią krwi CHryſtuſowej? Chleb / który łámiemy / izáli nie jeſt ſpołecżnośćią ćiáłá CHryſtuſowego?
Abowiem jednym chlebem / jednym ćiáłem wiele nas jeſt ; bo wƺyſcy chlebá jednego jeſteśmy ucżeſnikámi.
Spojrzyjćie ná Izráelá / który jeſt według ćiáłá ; izáż ći / którzy jedzą ofiáry / nie ſą ucżeſnikámi ołtárzá?
Cóż tedy mówię? Żeby báłwán miał być cżem / á iżby to / co bywá báłwánom ofiárowáne / miáło cżem być?
Owƺem powiádám / iż to / co pogánie ofiárują / dyjábłom ofiárują á nie Bogu ; á nie chćiáłbym / ábyśćie byli ucżeſnikámi dyjábłów.
Nie możećie pić kielichá Páńſkiego y kielichá dyjábelſkiego ; nie możećie być ucżeſnikámi ſtołu Páńſkiego y ſtołu dyjábelſkiego.
Y mámyż dráżnić Páná? Izáliśmy mocniejśi niżeli on?
Wƺyſtko mi wolno / ále nie wƺyſtko pożytecżne ; wƺyſtko mi wolno / ále nie wƺyſtko buduje.
Nikt niecháj nie ƺuká tego / co jeſt jego / ále káżdy / co jeſt bliźniego.
Cokolwiek w játkách ſprzedáwáją / jedzćie / nic nie pytájąc dla ſumieniá.
Abowiem Páńſká jeſt źiemiá y nápełnienie jey.
A jeſliby was kto záprośił z niewiernych / á chcećie iść / wƺyſtko / co przed was położą / jedzćie / nic nie pytájąc dla ſumieniá.
Ale jeſliby wám kto rzekł : To jeſt báłwánom ofiárowáne / nie jedzćie dla onego / który to oznájmił / y dla ſumieniá ; ábowiem Páńſká jeſt źiemiá y nápełnienie jey.
A powiádám dla ſumieniá / nie twego / ále onego drugiego ; bo przecżże wolność mojá má być oſądzoná od cudzego ſumieniá?
A poniewáż ja z dźiękowániem pożywám / cżemuż o to mam być bluźniony / zá co ja dźiękuję?
Przetoż lub jećie lub pijećie / lub cokolwiek cżynićie / wƺyſtko ku chwále Bożej cżyńćie.
Nie bądźćie obráżeniem y Żydom y Grekom y zborowi Bożemu ;
Jáko y ja we wƺyſtkiem śię wƺyſtkim podobám / nie ƺukájąc w tem ſwego pożytku / ále wielu ich / áby byli zbáwieni.
