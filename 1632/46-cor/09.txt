Izálim nie jeſt Apoſtołem? Izálim nie jeſt wolnym? Izálim JEzuſá CHryſtuſá Páná náƺego nie widźiáł? Izáli wy nie jeſteśćie prácą moją w Pánu?
Choćbym innym nie był Apoſtołem / álem wám jeſt ; ábowiem piecżęćią Apoſtolſtwá mego wy jeſteśćie w Pánu.
Táć jeſt obroná mojá przećiwko tym / którzy mię ſądzą.
Izáli nie mámy wolnośći jeść y pić?
Izáli nie mámy wolnośći / wodźić z ſobą śioſtry żony / jáko y drudzy Apoſtołowie y bráćia Páńſcy y Kiefáſ?
Izáli ja tylko y Bárnábáƺ nie mámy wolnośći / ábyśmy nie prácowáli?
Któż kiedy ſłuży żołnierkę ſwoim koƺtem? Któż ſádźi winnicę / á owocu jey nie pożywá? Albo któż trzodę páśie / á mleká trzody nie pożywá?
Izáli to obycżájem ludzkim mówię? Izáli y zakon tegoż nie mówi?
Abowiem w zakonie Mojżeƺowym nápiſáno : Nie záwiążeƺ gęby wołowi młócącemu ; izáli śię Bóg o woły ſtárá?
Cżyli zgołá dla nas to mówi? Abowiem dla nas to nápiſáno ; gdyż w nádźiei má oráć ten / co orze / á kto młóći w nádźiei / nádźiei ſwojey má być ucżeſnikiem.
Poniewáżeśmy my wám duchowne dobrá śiáli / wielkáż to / gdybyśmy wáƺe ćieleſne żęli?
Jeſliż inśi tey wolnośći nád wámi używáją / cżemuż nie rácżej my? A wƺákżeśmy tey wolnośći nie używáli / ále wƺyſtko znáƺámy / ábyśmy jákiego wſtrętu Ewángielii CHryſtuſowej nie ucżynili.
Azáż nie wiećie / iż ći / którzy około rzecży świętych prácują / z świętych rzecży jádáją? á którzy ołtárzá pilnują / ſpólną cżąſtkę z ołtárzem máją?
Ták też Pán poſtánowił tym / którzy Ewángieliję opowiádáją / áby z Ewángielii żyli.
Alem ja nic z tych rzecży nie używáł. Y nie piſáłem tego / áby śię ták przy mnie dźiáło ; bo mnie dáleko lepiej umrzeć / niżby kto przechwálánie moje miał próżnem ucżynić.
Bo jeſli Ewángieliję opowiádám / nie mam śię cżem chlubić / gdyż tá powinność ná mnie leży ; á biádá mnie / jeſlibym Ewángielii nie opowiádáł.
Abowiem jeſli to dobrowolnie cżynię / mam zápłátę ; jeſli poniewolnie / ƺáfárſtwá mi powierzono.
Jákąż tedy mam zápłátę? Abym Ewángieliję opowiádájąc / bez nákłádu wyſtáwił Ewángieliję CHryſtuſową / ná to / żebym źle nie używáł wolnośći mojey przy Ewángielii.
Abowiem będąc wolnym od wƺyſtkich / ſámegom śiebie ucżynił niewolnikiem wƺyſtkim / ábym ich więcey pozyſkáł.
Y ſtáłem śię Żydom jáko Żyd / ábym Żydów pozyſkáł ; á tym / którzy ſą pod zakonem / jákobym był pod zakonem / ábym tych / którzy ſą pod zakonem / pozyſkáł ;
Tym / którzy ſą bez zakonu / jákobym bez zakonu / ( nie będąc bez zakonu Bogu / ále będąc pod zakonem CHryſtuſowi / ) ábym pozyſkáł tych / którzy ſą bez zakonu.
Stáłem śię mdłym jáko mdły / ábym mdłych pozyſkáł. Wƺyſtkim ſtáłem śię wƺyſtko / ábym przećię niektórych zbáwił.
A to cżynię dla Ewángielii / ábym śię jey ſtáł ucżeſnikiem.
Azáż nie wiećie / iż ći / którzy w záwód bieżą / wƺyſcyć więc bieżą / lecż jeden zákłád bierze? Tákże bieżćie / ábyśćie otrzymáli.
A káżdy / który śię potyká / we wƺyſtkiem śię powśćiągá / onić wpráwdźie / áby wźięli koronę ſkáźitelną / ále my nieſkáźitelną.
Ja tedy ták bieżę / nie jáko ná niepewne ; ták ƺermuję / nie jáko wiátr bijąc.
Ale kárzę ćiáło moje y w niewolę podbijám / ábym ſnáć inƺym káżąc / ſám nie był odrzucony.
