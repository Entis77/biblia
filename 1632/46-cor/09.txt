Izalim nie jeſt Apoſtołem? Izalim nie jeſt wolnym? Izalim JEzuſá CHryſtuſá PAná náƺego nie widźiał? Izali wy nie jeſteśćie pracą moją w PAnu?
Choćbym innym nie był Apoſtołem / álem wam jeſt ; ábowiem piecżęćią Apoſtolſtwá mego wy jeſteśćie w PAnu.
Táć jeſt obroná mojá przećiwko tym którzy mię ſądzą.
Izali nie mamy wolnośći jeść y pić?
Izali nie mamy wolnośći wodźić <i>z ſobą</i> śioſtry żony / jáko y drudzy Apoſtołowie y bráćia PAńſcy / y Cefás?
Izali ja <i>tylko</i> y Bárnábaƺ nie mamy wolnośći / ábyſmy nie prácowáli?
Któż kiedy ſłuży żołnierſką ſwojim koƺtem? Któż ſádźi winnicę / á owocu jey nie pożywa? Albo któż trzodę páśie / á mleká trzody nie pożywa?
Izali to obycżájem ludzkim mówię? Izali y zakon tegoż nie mówi?
Abowiem w Zakonie Mojżeƺowym nápiſano ; Nie záwiążeƺ gęby wołowi młócącemu : Izali śię Bóg o woły ſtára?
Cżyli zgołá dla nas <i>to</i> mówi? Abowiem dla nas <i>to</i> nápiſano ; Gdyż w nádźiejey ma oráć ten co orze : á kto młóći w nádźiejey / nádźieje ſwojey má być ucżeſnikiem.
Ponieważeſmy my wam duchowne <i>dobrá</i> śiali / wielkaƺ <i>to</i> gdybyſmy wáƺe ćieleſne żęli?
Jeſliż inƺy tey wolnośći nád wámi używáją / cżemuż nie rácżey my ? A wƺákżeſmy tey wolnośći nie używáli / ále wƺyſtko znaƺamy / ábyſmy jákiego wſtrętu Ewángeliey CHryſtuſowey nie ucżynili.
Azaż nie wiećie / iż ći którzy około <i>rzecży</i> świętych prácują / z świętych rzecży jadáją / á którzy ołtarzá pilnują / ſpólną cżąſtkę z ołtarzem máją.
Ták też PAn poſtánowił tym którzy Ewángelią opowiádáją / áby z Ewángeliey żyli.
Alem ja nic z tych <i>rzecży</i> nie używał. Y nie piſałem tego / áby śię ták przy mnie dźiało : bo mnie dáleko lepiey umrzeć / niżby kto przechwalánie moje miał próznym ucżynić.
Bo jeſli Ewángelią opowiádam / nie mam śię <i>cżym</i> chlubić ; gdyż tá powinność ná mnie leży ? A biádá mnie jeſlibym Ewángeliey nie opowiádał.
Abowiem jeſli to dobrowolnie cżynię / mam zapłátę : jeſli poniewolnie / ƺáfárſtwá mi powierzono.
Jákąż tedy mam zapłátę ? Abym Ewángelią opowiádájąc / bez nákłádu wyſtáwił Ewángelią CHryſtuſowę / ná to żebym źle nie używał wolnośći mojey przy Ewángeliey.
Abowiem będąc wolnym od wƺyſtkich / ſámegom śiebie ucżynił niewolnikiem wƺyſtkim / ábym <i>ich</i> więcey pozyſkał.
Y ſtałem śię Żydom jáko Żyd / ábym Żydy pozyſkał : á tym którzy ſą pod zakonem / jákobym <i>był</i> pod zakonem / ábym te którzy ſą pod zakonem pozyſkał :
Tym którzy ſą bez zakonu / jákobym był bez zakonu / ( nie będąc bez zakonu Bogu / ále będąc pod zakonem CHryſtuſowi ) ábym pozyſkał te którzy ſą bez zakonu :
Stałem śię mdłym jáko mdły / ábym mdłe pozyſkał. Wƺyſtkim ſtałem śię wƺyſtko / ábym przećię niektóre zbáwił.
A to cżynię dla Ewángeliey / ábym śię jey ſtał ucżeſnikiem.
Azaż nie wiećie / iż ći którzy w zawód bieżą wƺyſcyć więc bieżą / lecż jeden zákład bierze ? Tákże bieżćie / ábyśćie otrzymáli.
A káżdy który śię potyka / we wƺyſtkim śię powśćiąga : onić wprawdźie áby wźięli koronę ſkáźitelną / ále my nieſkáźitelną.
Ja tedy ták bieżę / nie jáko ná niepewną ; ták ƺermuję / nie jáko wiátr bijąc.
Ale karzę ćiáło moje / y w niewolą podbijam ; Abym ſnadź inƺym każąc / ſam nie był odrzucony.
