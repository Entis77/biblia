A o rzecżách które báłwanom ofiárowáne bywáją wiemy / iż wƺyſcy umiejętność mamy. Umiejętność nádyma / ále miłość buduje.
A jeſli kto mniema żeby co umiał / jeƺcże nic nie umie / ták jákoby miał umieć.
Lecż jeſli kto miłuje Bogá / ten jeſt wyucżony od niego.
A przetoż o pokármiech które bywáją báłwanom ofiárowáne / wiemy iż báłwan ná świećie nic nie jeſt ; á iż niemáƺ żadnego inƺego Bogá / tylko jeden :
Bo choć ſą / którzy bogámi názywáni bywáją y ná niebie y ná źiemi : ( jákoż jeſt wiele bogów y wiele pánów. )
Ale my mamy jednego Bogá Ojcá / z którego wƺyſtko / á my w nim ; y jednego PAná JEzuſá CHryſtuſá / przez którego wƺyſtko / á my przezeń.
Ale nie we wƺyſtkich <i>jeſt</i> tá umiejętność. Abowiem niektórzy ſumnienie májąc dla báłwaná áż do tąd / jedzą jáko <i>rzecż</i> báłwanom ofiárowáną ; á ſumnienie ich będąc mdłe / pokaláne bywa.
Aleć nas pokarm nie záleca Bogu : Bo choćbyſmy jedli / nic nam nie przybywa ; á choćbyſmy y nie jedli / nic nam nie ubywa.
Jednák bacżćie / áby ſnadź tá wolność wáƺá nie byłá mdłym ku zgorƺeniu.
Abowiem jeſliby kto ujrzał ćiebie który maƺ umiejętność / w báłwochwálni śiedzącego ; ázaż ſumnienie onego który jeſt mdły / nie będźie pobudzone ku jedzeniu <i>rzecży</i> báłwanom ofiárowánych?
Y zginie dla oney twojey umiejętnośći brát mdły / zá którego CHryſtus umárł.
A grzeƺąc ták przećiwko bráći y mdłe ich ſumnienie obrażájąc / grzeƺyćie przećiwko CHryſtuſowi.
Przeto / jeſli pokarm gorƺy brátá mego / nie będę jadł mięſá ná wieki / ábym brátá mego nie zgorƺył.
