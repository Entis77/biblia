A o rzecżách / które báłwánom ofiárowáne bywáją / wiemy / iż wƺyſcy umiejętność mámy. Umiejętność nádymá / ále miłość buduje.
A jeźli kto mniemá / żeby co umiáł / jeƺcże nic nie umie / ták jákoby miáł umieć ;
Lecż jeźli kto miłuje Bogá / ten jeſt wyucżony od niego.
A przetoż o pokármách / które bywáją báłwánom ofiárowáne / wiemy / iż báłwán ná świećie nic nie jeſt / á iż niemáƺ żádnego inƺego Bogá / tylko jeden.
Bo choć ſą / którzy bogámi názywáni bywáją y ná niebie y ná źiemi : ( jákoż jeſt wiele bogów y wiele pánów. )
Ale my mámy jednego Bogá Ojcá / z którego wƺyſtko / á my w nim ; y jednego Páná JEzuſá CHryſtuſá / przez którego wƺyſtko á my przezeń.
Ale nie we wƺyſtkich jeſt tá umiejętność ; ábowiem niektórzy ſumienie májąc dlá báłwáná áż dotąd / jedzą jáko rzecż báłwánom ofiárowáną / á ſumienie ich będąc mdłe / pokáláne bywá.
Aleć náſ pokárm nie zálecá Bogu ; bo choćbyśmy jedli / nic nám nie przybywá ; á choćbyśmy y nie jedli / nic nám nie ubywá.
Jednák bácżćie / áby ſnáć tá wolność wáƺá nie byłá mdłym ku zgorƺeniu.
Abowiem jeźliby kto ujrzáł ćię / który máƺ umiejętność / w báłwochwálni śiedzącego / ázáż ſumienie onego / który jeſt mdły / nie będźie pobudzone ku jedzeniu rzecży báłwánom ofiárowánych?
Y zginie dlá onej twojej umiejętnośći brát mdły / zá którego CHryſtuſ umárł.
A grzeƺąc ták przećiwko bráćiom / y mdłe ich ſumienie obráżájąc / grzeƺyćie przećiwko CHryſtuſowi.
Przeto / jeźli pokárm gorƺy brátá mego / nie będę jádł mięſá ná wieki / ábym brátá mego nie zgorƺył.
