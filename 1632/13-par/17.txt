Y ſtáło śię / gdy mieƺkáł Dawid w domu ſwym / że rzekł do Nátáná proroká : Oto já mieƺkám w domu cedrowym / á ſkrzyniá przymierzá Páńſkiego pod kortynámi.
Y rzekł Nátán do Dawidá : Cokolwiek jeſt w ſercu twem / ucżyń / gdyż Bóg jeſt z tobą.
Potem onej nocy ſtáło śię ſłowo Boże do Nátáná / mówiąc :
Idź / á mów do Dawidá / ſługi mego : Ták mówi Pán : Nie ty mi będzieƺ budowáł domu do mieƺkániá ;
Poniewáżem nie mieƺkáł w domu ode dniá / któregom wywiódł ſynów Izráelſkich / áż do dniá tego : álem śię przechádzáł z námiotu do námiotu / y z przybytku do przybytku.
Wƺędzie gdziemkolwiek chodził ze wƺyſtkim Izráelem / izálim y ſłowo rzekł któremu z ſędziów Izráelſkich / którymem rozkázáł / áby páśli lud mój / mówiąc : Przecżżeśćie mi nie zbudowáli domu cedrowego?
Przetoż teráz ták powieƺ ſłudze memu Dawidowi : Ták mówi Pán záſtępów : Jám ćiebie wziął z owcżárni / gdyś chodził zá trzodą / ábyś był wodzem nád ludem moim Izráelſkim ;
Y byłem z tobą wƺędzie / gdzieśkolwiek chodził / á wygłádziłem wƺyſtkich nieprzyjáćiół twoich przed twárzą twoją / y ucżyniłem ći imię / jáko imię wielkich ludzi / którzy ſą ná ziemi.
A poſtánowiłem miejſce ludowi memu Izráelſkiemu / y wƺcżepiłem go ; y będzie mieƺkáł ná miejſcu ſwem / á nie będzie więcej poruƺony / áni go więcej ſynowie niepráwośći trápić będą / jáko przedtem ;
Záráz ode dni / którychem poſtánowił ſędziów nád ludem moim Izráelſkim / y poniżyłem wƺyſtkich nieprzyjáćiół twoich / y oznájmiłem ći / żeć Pán dom zbuduje.
A gdy śię wypełnią dni twoje / ábyś ƺedł zá ojcámi twoimi / wzbudzę náśienie twoje po tobie / które będzie z ſynów twoich / y umocnię króleſtwo jego.
Ten mi zbuduje dom / y utwierdzę ſtolicę jego áż ná wieki.
Já mu będę zá ojcá á on mi będzie zá ſyná / á miłośierdziá mego nie odejmę od niego / jákom je odjął od tego / który był przed tobą ;
Owƺem poſtánowię go w domu moim / y w króleſtwie mojem áż ná wieki / á ſtolicá jego będzie trwáłá áż ná wieki.
Według tych wƺyſtkich ſłów y według wƺyſtkiego widzeniá tego / ták mówił Nátán do Dawidá.
Zátem wƺedłƺy król Dawid / śiádł przed oblicżem Páńſkim / y rzekł : Cóżem já jeſt / Pánie Boże! y co jeſt dom mój / żeś mię przywiódł áż dotąd?
Lecż y to máło było przed ocżymá twemi / o Boże! áleś też obietnicę ucżynił o domie ſługi twego ná cżáſ dáleki / y wejrzáłeś ná mię według obycżáju ludzkiego / wywyżƺájąc mię / o Pánie Boże!
Cóż jeƺcże więcej má mówić Dawid przed tobą o uwielbieniu ſługi twego? ábowiem ty znáƺ ſługę twego.
Pánie! dlá ſługi twego y według ſercá twego ucżyniłeś te wƺyſtkie wielkie rzecży / ábyś znájome ucżynił te wƺyſtkie wielmożne ſpráwy.
Pánie! nie máƺ podobnego tobie / y nie máƺ Bogá oprócż ćiebie / wedle wƺyſtkiego / cośmy ſłyƺeli w uƺy náƺe.
Y któż jeſt jáko lud twój / jáko Izráel / náród jedyny ná ziemi / dlá któregoby Bóg ƺedł / áby go ſobie odkupił zá lud / á ucżynił ſobie imię / cżyniąc wielkie rzecży y ſtráƺne / wygániájąc pogány przed twárzą ludu twego / któryś wykupił z Egiptu?
Y ucżyniłeś lud twój Izráelſki ſobie zá lud áż ná wieki / á ty Pánie! ſtáłeś śię im zá Bogá.
Przetoż teráz / o Pánie! ſłowo / któreś powiedziáł o ſłudze twoim / y o domu jego / niech będzie ſtwierdzone áż ná wieki / á ucżyń / jákoś powiedziáł.
Niechájże śię ták ſtánie / y niecháj będzie uwielbione imię twoje áż ná wieki / áby mówiono : Pán záſtępów / Bóg Izráelſki / Bogiem jeſt nád Izráelem ; á dom Dawidá / ſługi twego / niecháj umocniony będzie przed twárzą twoją.
Abowiemeś ty / Boże mój! objáwił ſłudze twemu / iż mu zbudujeƺ dom ; przetoż ználázł ſługá twój u śiebie / áby śię modlił przed tobą.
A ták o Pánie! tyś ſám Bóg / á mówiłeś to dobre o ſłudze ſwym.
Teráz tedy pocżąłeś błogoſłáwić domowi ſługi twego / áby trwáł ná wieki przed tobą ; boś ty Pánie! jemu błogoſłáwił / á będzie ubłogoſłáwiony ná wieki.
