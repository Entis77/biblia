Y rzekł Dawid : Toć jeſt miejſce domu Páná Bogá / y to ołtárz ná cáłopálenie Izráelowi.
Przetoż rozkázáł Dawid / áby zgromádzono cudzoźiemców / którzy byli w źiemi Izráelſkiej ; y poſtánowił z nich kámienników / áby ćioſáli kámienie cżworográniáſte ná budowánie domu Bożego.
Żelázá táż bárdzo wiele ná gwoźdźie / y ná drzwi w brámách / y ná ſpájánie nágotowáł Dawid / y miedźi wágę niezlicżoną.
Drzewá táż cedrowego bez licżby / ábowiem náwieźli Dawidowi Sydońcżycy y Tyryjcżycy drzewá cedrowego bárdzo wiele.
Bo rzekł był Dawid : Sálomon / ſyn mój / jeſt młodźieńcżykiem máłym / á dom má być zbudowány Pánu wielki y známienity / któregoby imię y ſłáwá po wƺyſtkiej źiemi byłá ; przetoż teráz nágotuję mu potrzeb. Y nágotowáł Dawid przed śmierćią ſwą bárdzo wiele potrzeb.
Tedy záwołáł Sálomoná / ſyná ſwego / á przykázáł mu / áby zbudowáł dom Pánu / Bogu Izráelſkiemu.
Y rzekł Dawid do Sálomoná : Synu mój! Umyśliłem był w ſercu mojem / zbudowáć dom imieniowi Páná Bogá mego.
Ale śię ſtáło do mnie ſłowo Páńſkie / mówiąc : Wieleś krwi rozláł / y wielkieś wojny prowádźił ; nie będźieƺ budowáł domu imieniowi memu / przeto żeś wiele krwi rozláł ná źiemię przedemną /
Oto ſyn / któryć śię urodźi / będźie mężem ſpokojnym ; bo mu dám odpocżnienie od wƺyſtkich nieprzyjáćiół jego zewƺąd. Przetoż Sálomon będźie imię jego ; ábowiem pokój y odpocżnienie dám Izráelowi zá dni jego.
On zbuduje dom imieniowi memu ; on mi będźie zá ſyná / á já mu będę zá ojcá / y utwierdzę ſtolicę króleſtwá jego nád Izráelem áż ná wieki.
Przetoż Pán będźie z tobą / ſynu mój! Y będźieć śię ƺcżęśćiło / y zbudujeƺ dom Páná / Bogá twego / jáko mówił o tobie.
Wƺákże niech ći dá Pán roztropność / y zmyſł / á niech ćię poſtánowi nád Izráelem / ábyś ſtrzegł zákonu Páná Bogá twego.
Tedy ƺcżęśliwym będźieƺ / jeźli ſtrzedz y cżynić będźieƺ przykázániá y ſądy / które rozkázáł Pán przez Mojżeƺá Izráelowi. Zmácniájże śię / á bądź mężnym / nie bój śię áni śię lękáj.
A otom já w utrápieniu mojem nágotowáł ná dom Páńſki złotá ſto tyśięcy tálentów / y ſrebrá tyśiąc tyśięcy tálentów do tego miedźi y żelázá bez wági / bo tego wiele jeſt : drzewá tákże / y kámieniá nágotowáłem / á ty do tego przycżyniƺ.
Máƺ też u śiebie wiele rzemieślników / kámienników y murárzy / y ćieśli / y wƺelkich biegłych w káżdem rzemieśle.
Złotá / ſrebrá / y miedźi / y żelázá niemáƺ licżby ; wſtáńże á cżyń / á Pán będźie z tobą.
Y przykázáł Dawid wƺyſtkim kśiążętom Izráelſkim / áby pomágáli Sálomonowi ſynowi jego ;
Mówiąc : Izáli Pán Bóg wáƺ nie jeſt z wámi / który wám dáł odpocżnienie zewƺąd? Bo dáł w rękę moję obywáteli tej źiemi / y poddáná jeſt tá źiemiá Pánu y ludowi jego.
Teráz tedy oddájćie ſerce ſwe y duƺę ſwoję / ábyśćie ƺukáli Páná / Bogá wáƺego ; y wſtáńćie / á budujćie świątnicę Pánu Bogu / żebyśćie tám wnieśli ſkrzynię przymierzá Páńſkiego / y nácżyniá święte Boże / do domu / który będźie zbudowány imieniowi Páńſkimu.
