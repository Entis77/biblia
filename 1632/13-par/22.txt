Y rzekł Dawid : Toć jeſt miejſce domu Páná Bogá / y to ołtárz ná cáłopálenie Izráelowi.
Przetoż rozkázał Dawid / áby zgromádzono cudzoźiemców / którzy byli w źiemi Izráelſkiej ; y poſtánowił z nich kámienników / áby ćioſáli kámienie cżworográniáſte ná budowánie domu Bożego.
Żelázá táż bárzo wiele ná gwoźdźie / y ná drzwi w bramách / y ná ſpájánie nágotował Dawid / y miedźi wágę niezlicżoną.
Drzewá táż cedrowego bez licżby / ábowiem náwieſli Dawidowi Sydońcżycy y Tyryjcżycy drzewá cedrowego bárzo wiele.
Bo rzekł był Dawid : Sálomon / ſyn mój / jeſt młodźieńcżykiem máłym / á dom ma być zbudowány Pánu wielki y známienity / któregoby imię y ſłáwá po wƺyſtkiej źiemi byłá ; przetoż teraz nágotuję mu potrzeb. Y nágotował Dawid przed śmierćią ſwą bárzo wiele potrzeb.
Tedy záwołał Sálomoná / ſyná ſwego / á przykázał mu / áby zbudował dom Pánu / Bogu Izráelſkiemu.
Y rzekł Dawid do Sálomoná : Synu mój! Umyśliłem był w ſercu mojim / zbudowáć dom imieniowi Páná Bogá mego.
Ale śię ſtáło do mnie ſłowo Páńſkie / mówiąc : Wieleś krwi rozlał / y wielkieś wojny prowádźił ; nie będźieƺ budował domu imieniowi memu / przeto żeś wiele krwi rozlał ná źiemię przedemną /
Oto ſyn / któryć śię urodźi / będźie mężem ſpokojnym ; bo mu dam odpocżnienie od wƺyſtkich nieprzyjáćiół jego zewƺąd. Przetoż Sálomon będźie imię jego ; ábowiem pokój y odpocżnienie dam Izráelowi zá dni jego.
On zbuduje dom imieniowi memu ; on mi będźie zá ſyná / á ja mu będę zá ojcá / y utwierdzę ſtolicę króleſtwá jego nád Izráelem áż ná wieki.
Przetoż Pán będźie z tobą / ſynu mój! Y będźieć śię ƺcżęśćiło / y zbudujeƺ dom Páná / Bogá twego / jáko mówił o tobie.
Wƺákże niech ći da Pán roztropność / y zmyſł / á niech ćię poſtánowi nád Izráelem / ábyś ſtrzegł zakonu Páná Bogá twego.
Tedy ƺcżęśliwym będźieƺ / jeſli ſtrzedz y cżynić będźieƺ przykázánia y ſądy / które rozkázał Pán przez Mojżeƺá Izráelowi. Zmácniájże śię / á bądź mężnym / nie bój śię áni śię lękáj.
A otom ja w utrápieniu mojim nágotował ná dom Páńſki złotá ſto tyśięcy tálentów / y ſrebrá tyśiąc tyśięcy tálentów do tego miedźi y żelázá bez wági / bo tego wiele jeſt : drzewá tákże / y kámieniá nágotowáłem / á ty do tego przycżyniƺ.
Maƺ też u śiebie wiele rzemieślników / kámienników y murárzy / y ćieśli / y wƺelkich biegłych w káżdym rzemieśle.
Złotá / ſrebrá / y miedźi / y żelázá niemáƺ licżby ; wſtáńże á cżyń / á Pán będźie z tobą.
Y przykázał Dawid wƺyſtkim kśiążętom Izráelſkim / áby pomágáli Sálomonowi ſynowi jego ;
Mówiąc : Izali Pán Bóg wáƺ nie jeſt z wámi / który wam dał odpocżnienie zewƺąd? Bo dał w rękę moję obywáteli tey źiemi / y poddáná jeſt tá źiemiá Pánu y ludowi jego.
Teraz tedy oddájćie ſerce ſwe y duƺę ſwoję / ábyśćie ƺukáli Páná / Bogá wáƺego ; y wſtáńćie / á budujćie świątnicę Pánu Bogu / żebyśćie tám wnieśli ſkrzynię przymierza Páńſkiego / y nacżynia święte Boże / do domu / który będźie zbudowány imieniowi Páńſkimu.
