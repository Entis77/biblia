Y ſtáło śię potem / że poráźił Dawid Filiſtynów y poniżył ich / á wźiął Get y wśi jego z rąk Filiſtynów.
Poráźił też Moábcżyki / y byli Moábcżycy ſługámi Dawidowymi / przynoƺąc mu hołd.
Poráźił też Dawid Hádárezerá / królá Soby w Emát / gdy był wyjecháł / áby rozprzeſtrzeniáł páńſtwo ſwoje nád rzeką Eufráteſ.
Zábráł mu tedy Dawid tyśiąc wozów / y śiedm tyśięcy jezdnych / y dwádźieśćiá tyśięcy mężów pieƺych / y poderznął Dawid żyły wƺyſtkich woźników / záchowáwƺy z nich koni do ſtá wozów.
Przyćiągnęli też Syryjcżycy z Dámáƺku ná pomoc Hádárezerowi / królowi Soby ; lecż poráźił Dawid z Syryjcżyków dwádźieśćiá y dwá tyśiące mężów.
Tedy Dawid oſádźił żołnierzem Syryję Dámáſką / á byli Syryjcżycy ſługámi Dawidowymi / oddáwájąc mu hołd ; y záchowywáł Pán Dwidá / gdźie śię kolwiek obróćił.
Pobráł też Dawid tárcże złote / które mieli ſłudzy Hádárezerowi / y wnióſł je do Jeruzálemu.
Przytem z Tybchát y z Chun / miáſt Hádárezerowych / nábráł Dawid miedźi bárdzo wiele / z którey Sálomon ſpráwił morze miedźiáne / y ſłupy / y nácżyniá miedźiáne.
A gdy uſłyƺáł Tohy / król Emát / że poráźił Dawid wƺyſtko wojſko Hádárezerá / królá Soby.
Poſłáł Adorámá / ſyná ſwego / do królá Dawidá / áby go pozdrowił w pokoju / y áby mu powinƺowáł / przeto / że zwálcżył Hádárezerá / y poráźił go ; ( ábowiem wálcżył Tohy z Hádárezerem ) który przynióſł z ſobą wƺelákie nácżynie złote / y ſrebrne / y miedźiáne.
Które też poświęćił król Dawid Pánu z ſrebrem y ze złotem / które był pobráł od wƺyſtkich národów / od Edomcżyków / y od Moábcżyków / o od ſynów Ammonowych / y od Filiſtynów / y od Amálekitów.
A Abiſáj / ſyn Sárwii / poráźił Edomcżyków w dolinie ſolney ośmnáśćie tyśięcy.
Y oſádźił Edomſką źiemię żołnierzem : á byli wƺyſcy Edomcżycy ſługámi Dawidowymi : y záchowywáł Pán Dawidá wƺędźie / gdźie śię obróćił.
A ták królowáł Dawid nád wƺyſtkim Izráelem / cżyniąc ſąd y ſpráwiedliwość wƺyſtkiemu ludowi ſwemu.
A był Joáb / ſyn Sárwii / nád wojſkiem / á Jozáfát / ſyn Ahiludowy / kánclerzem.
A Sádok / ſyn Achitobowy / y Abimelech / ſyn Abijátárá / byli kápłánámi / á Suſá był piſárzem.
Benájáƺ też / ſyn Jojády / był przełożonym nád Cheretcżykámi y Feletcżykámi ; á ſynowie Dawidowi byli pierwƺymi przy boku królewſkim.
