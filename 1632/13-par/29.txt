Potem mówił król Dawid do wƺyſtkiego zgromádzeniá : Sálomoná / ſyná mego jedynego / obráł Bóg młodźieńcżyká máłego. Ale to wielká ſpráwá ; bo nie cżłowiekowi páłác ten / ále Pánu Bogu będźie.
Ja według nájwyżƺego przemożeniá mego nágotowáłem ná dom Bogá mego złotá / ná nácżynie złote / y ſrebrá ná ſrebrne / y miedźi ná miedźiáne / żelázá ná żelázne / y drzewá ná drewniáne / kámieniá onychynowego ná oſádzánie / y kámieniá kárbunkułowego / y rozlicżnych fárb / á wƺelákiego kámieniá drogiego / y kámieniá mármurowego doſtátek wielki.
Nádto z chęći mojey ku domowi Bogá mego oſobne złoto y ſrebro / które mam / oddáję ná dom Bogá mego / oprócż tego wƺyſtkiego / com zgotowáł ná dom świątnicy ;
To jeſt trzy tyśiące tálentów złotá / złotá z Ofir / y śiedm tyśięcy tálentów ſrebrá nájcżyſtƺego ná okryćie śćián gmáchów ;
Złotá ná nácżynie złote / á ſrebrá ná ſrebrne / y ná wƺyſtkie roboty rąk rzemieślnicżych ; y jeſliby jeƺcże kto chćiáł co dobrowolnie dźiś ofiárowáć Pánu?
Tedy dobrowolnie ofiárowáli przedniejśi z domów y przedniejśi z pokoleń Izráelſkich / y półkownicy / y rotmiſtrze / y przełożeni nád robotą królewſką.
Y złożyli ná uſługę domu Bożego złotá tálentów pięć tyśięcy / y złotych dźieśięć tyśięcy / á ſrebrá tálentów dźieśięć tyśięcy / y miedźi ośmnáśćie tyśięcy tálentów / á żelázá ſto tyśięcy tálentów.
Ci też co mieli drogie kámienie / dáwáli je do ſkárbu domu Páńſkiego / do rąk Jehijelá Gierſońcżyká.
Y weſelił śię lud / że ták ochotnie ofiárowáli. Abowiem ſercem doſkonáłem chętnie ofiárowáli Pánu ; tákże y król Dawid weſelił śię weſelem wielkiem.
Przetoż błogoſłáwił Dawid Pánu przed oblicżem wƺyſtkiego zgromádzeniá / y rzekł : Błogoſłáwionyś ty Pánie / Boże Izráelá / ojcá náƺego / od wieku áż ná wieki.
Twojá jeſt / Pánie! wielmożność / y moc / y ſłáwá / y zwyćięſtwo / y cżeść / y wƺyſtko ná niebie y ná źiemi ; twoje jeſt / Pánie! króleſtwo / á tyś jeſt wywyżƺony nád wƺelką zwierzchność.
Y bogáctwá / y ſłáwá od ćiebie ſą / á ty pánujeƺ nád wƺyſtkimi / á w rękách twych jeſt moc y śiłá / y w ręce twojey jeſt wywyżƺyć y utwierdźić wƺyſtko.
Teraz tedy / Boże náƺ! wyznájemy ćię / á chwálimy imię ſłáwy twojey.
Abowiem cóżem ja / y co jeſt lud mój / żebyśmyto śiły mieli / tobie to dobrowolnie ofiárowáć? gdyż od ćiebie jeſt wƺyſtko / á z rąk twoich wźiąwƺy dáliśmy tobie.
Bośmy my pielgrzymámi y przychodniámi przed tobą / jáko y wƺyſcy ojcowie náśi ; dni náƺe ná źiemi jáko ćień / á nie máƺ cżego ocżekiwáć.
O Pánie / Boże náƺ! ten wƺyſtek doſtátek / któryśmy zgotowáli tobie ná budowánie domu imieniowi twemu świętemu / z ręki twojey jeſt / y twoje jeſt wƺyſtko.
Wiemći ja / Boże mój! iż ty doświádcżáƺ ſerc / á kocháƺ śię w ƺcżerośći ; przetoż ja w ƺcżerośći ſercá mego / ochotniem ofiárowáł to wƺyſtko / náwet y lud twój / który śię tu ználázł / widźiáłem z weſelem y z ochotą ofiárujący tobie.
Pánie / Boże Abráhámá / Izááká / y Izráelá / ojców náƺych! záchowájże ná wieki tę chęć / y umyſł ſercá ludu twego / á przygotuj ſobie ſerce ich.
Sálomonowi też / ſynowi memu dáj ſerce doſkonáłe / áby ſtrzegł przykázáń twoich / świádectw twoich / y uſtáw twoich / y cżynił wƺyſtko / y áby zbudowáł dom / dla któregom potrzeby zgotowáł.
Potem mówił Dawid do wƺyſtkiego zgromádzeniá : Błogoſłáwćież teraz Pánu / Bogu wáƺemu. Y błogoſłáwiło wƺyſtko zgromádzenie Pánu / Bogu ojców ſwoich / á náchyliwƺy śię pokłonili śię Pánu y królowi.
Zátem ofiárowáli Pánu ofiáry. Ofiárowáli też cáłopáleniá Pánu názájutrz po onym dniu / wołów tyśiąc / báránów tyśiąc / báránków tyśiąc z mokremi ofiárámi ich / y inƺych ofiár wielkie mnóſtwo / zá wƺyſtkiego Izráelá.
Y jedli á pili przed Pánem dniá onego z weſelem wielkiem / á poſtánowili powtóre królem Sálomoná / ſyná Dawidowego / y pomázáli go Pánu zá kśięćiá / á Sádoká zá kápłáná.
A ták uśiádł Sálomon ná ſtolicy Páńſkiej zá królá miáſto Dawidá / ojcá ſwego / y ƺcżęśćiło mu śię / á był mu poſłuƺny wƺyſtek Izráel.
Y wƺyſcy kśiążętá / y możni / tákże y wƺyſcy ſynowie królá Dawidá / dáli ręce ná poddáńſtwo Sálomonowi królowi.
Y uwielbił Pán Sálomoná bárdzo zácnie przed ocżymá wƺyſtkiego Izráelá / á dáł mu ſłáwę królewſką / jákiej żáden król przed nim nie miał w Izráelu.
A ták Dawid / ſyn Iſájego / królowáł nád wƺyſtkim Izráelem.
A dni / których królowáł nád Izráelem / było cżterdźieśći lát ; w Hebronie królowáł śiedm lát ; á w Jeruzálemie królowáł trzydźieśći y trzy látá.
Y umárł w ſtárośći dobrej / pełen dni / bogáctw y ſłáwy : á królowáł Sálomon / ſyn jego / miáſto niego.
A ſpráwy królá Dawidá pierwƺe y oſtátnie / oto ſą zápiſáne w kśięgách Sámuelá widzącego / y w kśiędze Nátáná proroká / y w kśiędze Gádá widzącego ;
Ze wƺyſtkiem królowániem jego / y możnośćią jego / y z cżáſámi / które zá niego y zá Izráelá / y zá wƺyſtkich króleſtw źiemſkich przeƺły.
