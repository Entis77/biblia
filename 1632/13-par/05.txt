A ſynowie Rubená pierworodnego Izráelowego / ( ten bowiem był pierworodnym ; ále gdy zgwáłćił łoże ojcá ſwego / dáne jeſt pierworodztwo jego ſynom Józefá / ſyná Izráelowego / ták jednák / że go nie pocżytáno zá pierworodnego :
Bo Judás był namężniejƺy miedzy bráćią ſwą / á kśiążęćiem miedzy nimi : ále pierworodztwo <i>należáło</i> Józefowi )
Synowie mówię Rubená / pierworodnego Izráelowego / byli : Henoch y Fállu / Heſron y Chármi.
Synowie Joelowi : Sámájáƺ ſyn jego / Gog ſyn jego / Semej ſyn jego ;
Michás ſyn jego / Reájáƺ ſyn jego / Báál ſyn jego ;
Berá ſyn jego / którego wźiął w niewolę Teglát Fáláſár / król Aſyryjſki ; ten był kśiążęćiem Rubenitów.
A bráćia jego według domów ſwych / gdy byli policżeni według ich narodów / mieli kśiążętá Jehielá y Zácháryjáƺá.
A Belá / ſyn Azázowy / ſyná Semmy / ſyná Joelowego ; ten mieƺkał w Aroer áż ku Nebo y Báálmeon.
Tákże y ná wſchód ſłońcá mieƺkał / áż kędy wchodzą ná puƺcżę od rzeki Eufrátes ; ábowiem ſtádá ich rozmnożyły śię w źiemi Gáláádſkiej.
Ci zá dni Saulowych wálcżyli z Agáreńcżykámi / którzy poráżeni ſą od ręki ich ; á ták mieƺkáli w námiotách ich po wƺyſtkiej krájinie wſchodniej źiemi Gáláádſkiej.
A ſynowie Gádowi ná przećiwko nich mieƺkáli w źiemi Bázán / áż do Selchy.
Joel był przedniejƺym ich / á Sáfám wtóry / á Jánáj y Sáfát zoſtáli w Bázán.
A bráći ich według domów ojców ſwych : Micháel y Meſullám / y Sebá / y Joráj / y Jáchán / y Zyjá / y Heber / śiedm.
( Cić ſą ſynowie Abiháilá / ſyná Hurowego / ſyná Jároáchowego / ſyná Gáláádowego / ſyná Micháelowego / ſyná Jeſyſowego / ſyná Jáchdowego / ſyná Buzowego ; )
Achy / ſyn Abdyjelá / ſyná Gunowego / kśiążę w domu ojców ich.
Y mieƺkáli w Gáláád / w Bázán / y w miáſtecżkách jego / y po wƺyſtkich przedmieśćiách Sáron áż do ich gránic.
Wƺyſcy ći policżeni byli zá dni Jotámá / królá Judzkiego / y zá dni Jeroboámá / królá Izráelſkiego.
Synów Rubenowych / y Gádowych / y połowy pokolenia Mánáſeſowego / ludźi wálecżnych / mężów noƺących tárcżę y miecż / y ćiągnących łuk / y ćwicżonych ku bojowi / cżterdźieśći y cżtery tyśiące / śiedm ſet y ƺeśćdźieśiąt / wychodzących do bitwy.
Ci wiedli wojnę z Agáreńcżykámi / z Jeturejcżykámi / y z Náfejcżykámi y Nodábcżykámi /
A mieli pomoc przećiwko nim. Y podáni ſą w rękę ich Agáreńcżycy ze wƺyſtkim / co mieli / przeto iż do Bogá wołáli w bitwie / á on ich wyſłuchał / iż ufáli w nim.
Y zábráli dobytki ich / wielbłądów ich pięćdźieśiąt tyśięcy / á owiec dwieśćie y pięćdźieśiąt tyśięcy / oſłów dwá tyśiące / á ludźi ſto tyśięcy.
Abowiem ránnych wiele poległo / iż od Bogá byłá oná poráżká. Y mieƺkáli ná miejſcu ich / áż ich zábráno w niewolę.
Ale ſynowie połowy pokolenia Mánáſeſowego mieƺkáli w oney źiemi od Bázán áż do Báál-Hermon y Sánir / które jeſt górá Hermon ; bo y oni rozmnożeni byli.
A ćić ſą kśiążętá domów ojców ich : Efer y Jeſy / y Elijel / y Azryjel / y Jeremijáƺ / y Hodáwijáƺ / y Jáchdyjel / mężowie bárzo mocni / mężowie ſłáwni / kśiążętá domu ojców ſwojich.
Ale gdy zgrzeƺyli przećiw Bogu ojców ſwych y cudzołożyli náśládując bogów narodów oney źiemi / które wykorzenił Bóg przed twárzą ich :
Wzbudźił Bóg Izráelſki duchá Fulá królá Aſſyryjſkiego / y duchá Teglát Fáláſerá / królá Aſſyryjſkiego / y przenióſł ich : Rubenity / y Gádity / y połowę pokolenia Mánáſſeſſowego / á záwiódł ich do Helá / y do Háboru / y do Ará / y do rzeki Gozán / áż do dniá tego.
