A gdy ſobie pobudował Dawid domy w mieśćie ſwojim / y nágotował miejſce ſkrzyni Bożey / y rozbił jey námiot /
Tedy rzekł Dawid : Nie godźi śię nośić ſkrzyni Bożey jedno Lewitom. Tych bowiem obrał Pán / áby nośili ſkrzynię Bożą / y ſłużyli mu áż ná wieki.
Przetoż zebrał Dawid wƺyſtkiego Izráelá do Jeruzálemu / áby przenióſł ſkrzynię Páńſką ná miejſce jey / które jey był zgotował.
Y zgromádźił Dawid ſynów Aáronowych y Lewitów.
Z ſynów Káátowych : Uryjelá przedniejƺego / y bráći jego ſto y dwádźieśćiá /
Z ſynów Merárego : Aſájáƺá przedniejƺego / y bráći jego dwieśćie y dwádźieśćiá.
Z ſynów Gierſonowych : Joelá przedniejƺego / y bráći jego ſto y trzydźieśći.
Z ſynów Eliſáfánowych : Semejáƺá przedniejƺego / y bráći jego dwieśćie.
Z ſynów Hebronowych : Elijelá przedniejƺego / y bráći jego ośmdźieśiąt.
Z ſynów Háſyjelowych : Aminádábá przedniejƺego / y bráći jego ſto y dwánáśćie.
Tedy wezwał Dawid Sádoká y Abijátárá / kápłánów / tákże Lewitów Uryjelá / Aſájáƺá / y Joelá / Semejáƺá / y Elijelá / y Aminádábá ;
Y rzekł do nich : Wyśćie przedniejśi z domów ojcowſkich miedzy Lewitámi ; poświęććież śię y z bráćmi ſwoimi / ábyśćie przynieśli ſkrzynię Páná / Bogá Izráelſkiego / ná miejſce / którem jey nágotował.
Abowiem iżeśćie tego pierwej nie ucżynili / ucżynił rozerwánie Pán / Bóg náƺ / miedzy námi ; bośmy go nie ƺukáli według przyſtojnośći.
Poświęćili śię tedy kápłáni y Lewitowie / áby przynieśli ſkrzynię Páná / Bogá Izráelſkiego.
Y nieśli ſynowie Lewitów ſkrzynię Bożą / jáko był rozkázał Mojzeƺ według ſłowá Páńſkiego / ná rámionách ſwych / ná drążkách które przy niej były.
Y rzekł Dawid przedniejƺym z Lewitów / áby poſtánowili z bráći ſwojich śpiewáków z inſtrumentámi muzycżnemi / z lutniámi / z cytrámi / y z cymbáłámi / áby ſłyƺány był wynieśiony głos z weſelem.
Y poſtánowili Lewitowie Hemáná / ſyná Joelowego / á z bráći jego Aſáfá / ſyná Báráchyjáƺowego / á z ſynów Merárego / bráći ich / Etáná ſyná Chuſájowego ;
A z nimi bráći ich w rzędźie wtórym : Zácháryjáƺá / Bená / y Jázyjelá / y Semirámotá / y Jechyjelá / y Unni / Elijábá / y Benájáƺá / y Mááſejáƺá / y Mátytyjáƺá / y Elifelego / y Miknejáƺá / y Obededomá / y Jehielá / odźwiernych.
A śpiewácy Hemán / Aſáf / y Etán ná cymbáłách miedźiánych głośno gráli.
A Zácháryjáƺ / y Jázyjel / y Semirámot / y Jechyjel / y Unni / y Elijáb / y Mááſejáƺ / y Benájáƺ gráli ná lutniách przy śpiewániu wyſokiem.
A Mátytyjáƺ / y Elifele / y Miknejáƺ / y Obededom / y Jechyjel / y Azázyjáƺ gráli ná cytrách przy śpiewániu niſkiem.
A Kienánijáƺ / przedniejƺy z Lewitów / którzy nieśli ſkrzynię / rozrządzał / jákoby nieść miáno ; bo był roztropny.
Ale Bácháryjáƺ y Elkáná byli odźwiernymi u ſkrzyni.
A Sebánijáƺ y Jozáfát / y Nátánáel / y Amáſáj / y Zácháryjáƺ / y Benájáƺ / y Eliezer / kápłáni / trąbili w trąby przed ſkrzynią Bożą ; ále Obededom y Jechyjáƺ byli odźwiernymi u ſkrzyni.
A ták Dawid y ſtárśi Izráelſcy / y hetmáni nád tyśiącámi ƺli / áby przeprowádźili ſkrzynię przymierza Páńſkiego z domu Obededomowego z weſelem.
Y ſtáło śię / gdy Bóg wſpomógł Lewitów / nioſących ſkrzynię przymierza Páńſkiego / że ofiárowáli śiedm wołów / y śiedm báránów.
A Dawid był oblecżony w ƺátę biśiorową / tákże y wƺyſcy Lewitowie / którzy nieśli ſkrzynię / y śpiewácy / y Kienánijáƺ / rządcá tych / co nieśli miedzy śpiewákámi ; á Dawid miał ná ſobie efod lniáne.
A ták wƺyſtek Izráel prowádźił ſkrzynię przymierza Páńſkiego z weſelem / y z hukiem kornetu / y trąby / y cymbáłów / grájąc ná lutniách y ná cytrách.
Y ſtáło śię / gdy ſkrzyniá przymierza Páńſkiego wchodźiłá do miáſtá Dawidowego / że Michol / córká Saulowá / wyglądájąc oknem ujrzáłá królá Dawidá ſkácżącego / y grájącego / y wzgárdźiłá go w ſercu ſwojim.
