Rozdźiáły záś odźwiernych były z Korejcżyków : Meſelemijáƺ / ſyn Korego / z ſynów Aſáfowych.
A z Meſelemijáƺowych ſynów : Zácháryjáƺ pierworodny / Jádyjáel wtóry / Zábádyjáƺ trzeći / Játnijel cżwárty.
Elám piąty / Johánán ƺóſty / á Elienáj śiódmy.
A z Obededomowych ſynów : Semájáƺ pierworodny / Jozábád wtóry / Joách trzeći / y Sáchár cżwárty / á Nátánáel piąty /
Ammijel ƺóſty / Iſáſchár śiódmy / Pechulletáj óſmy ; bo mu Bóg błogoſłáwił.
Semájáƺowi też / ſynowi jego / zrodźili śię ſynowie / którzy pánowáli w domu ojcá ſwego ; bo byli mężowie bárdzo mocni.
Synowie Semájáƺowi : Otni / y Ráfáel y Obed / Elzábed / bráćiá jego / mężowie mocni / Elihu y Semáchyjáƺ.
Wƺyſcy ći z ſynów Obededomowych / ſámi y ſynowie ich / y bráćiá ich / káżdy z nich bárdzo mocny y ſpoſobny ku poſługiwániu / ƺeśdźieśiąt y dwá wƺyſtkich z Obededomá.
A z Meſelemijáƺowych ſynów y bráći / mężów mocnych / ośmnáśćie.
A z Hoſy / który był z ſynów Merárego / ſynowie byli : Semry przedniejƺy ; nie iżby był pierworodny / ále iż go ojćiec jego ucżynił przedniejƺym ;
Helkijáƺ wtóry / Tebálijáƺ trzeći / Zácháryjáƺ cżwárty ; wƺyſtkich ſynów y bráći Hoſy trzynáśćie.
Ci ſą rozdźieleni ná odźwiernych / áby byli wrotnymi z mężów przedniejƺych / trzymájąc ſtráż ná przemiány z bráćmi ſwymi przy ſłużbie w domu Páńſkim.
Abowiem miotáli loſy / ták máły jáko wielki według domów ojców ſwych / o káżdą brámę.
Y pádł loſ ná wſchód ſłońcá Selemijáƺowi ; Zácháryjáƺowi tákże ſynowi jego / rádcy mądremu / rzućili loſy / y pádł loſ jego ná północy ;
A Obededomowi ná południe ; ále ſynom jego ná dom ſkárbów.
Suppimowi y Hoźie ná záchód z brámą Zállechet / przy śćieƺce uſypánej / idącej ku górze ; á ták byłá ſtráż ná przećiwko ſtráży.
Ná wſchód ſłońcá było Lewitów ƺeść / ná północy ná dźień cżterech / ná południe ná dźień cżterech / á przy domu ſkárbów dwóch á dwóch.
Przy ſtronie zewnętrznej ná záchód było cżterech ná drodze ſypánej / á dwóch przy ſtronie zewnętrznej.
Teć ſą rozdźiáły odźwiernych z ſynów Korego / y z ſynów Merárego.
A z drugich Lewitów Achyjáƺ był nád ſkárbámi domu Bożego / to jeſt nád ſkárbámi rzecży poświęconych.
Synowie Láádánowi / którzy byli z ſynów Gierſonickich : z Láádáná Gierſońcżyká przedniejśi w domách ojcowſkich / Jehyjel.
A ſynowie Jehyjelowi byli Zetám / y Joel / brát jego ; ći byli nád ſkárbámi domu Páńſkiego.
Z Amrámcżyków / y z Izáárcżyków / z Hebrońcżyków / y Huſyjelcżyków.
Był Sebuel / ſyn Gierſoná / ſyná Mojżeƺowego / przełożony nád ſkárbámi.
Ale bráćiá jego z Eleázárá byli Rechábijáƺ ſyn jego / y Jeſájáƺ ſyn jego / y Jorám ſyn jego / y Zychry ſyn jego / y Sálomit ſyn jego.
Ten Sálomit y bráćiá jego byli nád wƺyſtkiemi ſkárbámi rzecży poświęconych / które był poświęćił Dawid król / y przedniejśi z domów ojcowſkich / y półkownicy / y rotmiſtrze / y hetmáni wojſká.
Bo z wojen y z łupów poświęcáli ná popráwę domu Páńſkiego ;
Y wƺyſtko / co był poświęćił Sámuel widzący / y Saul / ſyn Cyſowy / y Abner / ſyn Nerá / y Joáb / ſyn Sárwii / y ktokolwiek co poświęcáł / oddáwáł do rąk Sálomitowych y bráći jego.
Z Izáárytów : Kienánijáƺ y ſynowie jego nád robotą zewnętrzną w Izráelu / byli zá urzędników y zá ſędźiów.
Z Hebrońcżyków : Háſábijáƺ y bráći jego / mężów dużych / było tyśiąc y śiedm ſet przełożonych nád Izráelem zá Jordánem ná záchód ſłońcá / w káżdej roboćie Páńſkiej y w poſłudze królewſkiej.
A z Hebrońcżyków był Jeryjáƺ przedniejƺy nád Hebrońcżykámi według národów ich y domów ojcowſkich. Bo roku cżterdźieſtego króleſtwá Dawidowego ƺukáno y ználeźiono między nimi mężów bárdzo mocnych w Jázer Gáláádſkiem.
A bráći jego / mężów godnych / było dwá tyśiące y śiedm ſet przedniejƺych w domách ojcowſkich. Y poſtánowił ich Dawid król nád Rubeńcżykámi / y nád Gádcżykámi / y nád połową pokoleniá Mánáſeſowego / nád wƺyſtkiemi ſpráwámi Bożemi y ſpráwámi królewſkiemi.
