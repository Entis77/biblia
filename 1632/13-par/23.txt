A ták Dawid będąc ſtáry y pełen dni / poſtánowił królem Sálomoná / ſyná ſwego / nád Izráelem.
Y zgromádźił wƺyſtkich kśiążąt Izráelſkich / y kápłanów / y Lewitów ;
A policżono Lewitów od trzydźieſtu lat y wyżey ; y był pocżet ich według głów ich oſób trzydźieśći y ośm tyśięcy.
Z których poſtánowiono ná poſługę domu Páńſkiego dwádźieśćiá y cżtery tyśiące / á przełożonych y ſędźiów ƺeść tyśięcy.
Nádto cżtery tyśiące odźwiernych / y cżtery tyśiące chwálących Páná ná inſtrumentách / których náſpráwiał Dawid ku chwáleniu Bogá.
Y rozdźielił ich Dawid ná pewne hufy według ſynów Lewiego / to jeſt / Gierſoná / Káátá / y Merárego.
Z Gierſoná byli Láádán / y Semej.
Synowie Láádánowi : przedniejƺy Jechijel / y Zetám / y Joel / ći trzej.
Synowie Semejowi : Sálomit / y Házyjel / y Hárán / ći trzej. Cić byli przedniejśi domów ojcowſkich z Láádáná.
A ſynowie Semejowi : Jáchát / Zyná / y Jehus / y Báryjáƺ ; ćić ſynowie Semejowi cżterej.
A Jáchát był pierwƺym / á Zyzá wtóry ; ále Jehus y Báryjáƺ nie mieli wiele ſynów ; przetoż byli w domu ojcowſkim policżeni zá jednę fámiliję.
Synowie Káátowi : Amrám / Izáár / Hebron / y Huſyjel / cżterej.
Synowie Amrámowi : Aáron y Mojzeƺ. Lecż Aáron był odłącżony / áby ſłużył w świątnicy nájświętƺej / ſám y ſynowie jego áż ná wieki / y áby kádźili przed Pánem / á ſłużyli mu / y błogoſłáwili w imieniu jego áż ná wieki.
Ale ſynowie Mojżeƺá / mężá Bożego / policżeni ſą w pokoleniu Lewiego.
Synowie Mojżeƺowi : Gierſon y Eliezer.
Synowie Gierſonowi : Sebujel pierwƺy.
A ſynowie Eliezerowi byli Rechábijáƺ pierwƺy. Y nie miał Eliezer ſynów innych ; ále ſynowie Rechábijáƺowi rozmnożyli śię bárzo.
Synowie Izáárowi : Sálomit pierwƺy.
Synowie Hebronowi : Jeryjáƺ pierwƺy / Amáryjáƺ wtóry / Jeházyjel trzeći / á Jekmáán cżwárty.
Synowie Huſyjelowi : Michá pierwƺy / á Jeſyjáƺ wtóry.
Synowie Merárego : Máheli y Muſy ; á ſynowie Máhelego : Eleázár y Cys.
Y umárł Eleázár / á nie miał ſynów / tylko córki / które pojmowáli ſynowie Cyſowi / bráćia ich.
Synowie Muſy : Máheli / y Eder / y Jerymot / trzej.
Cić ſą ſynowie Lewiego według domów ojców ſwych / przedniejśi domów ojcowſkich / którzy policżeni byli według pocżtu imion y oſób ſwych z oſobná / którzy odpráwowáli práce uſługiwánia w domu Páńſkim od dwudźieſtu lat y wyżey.
Abowiem rzekł Dawid : Dał odpocżnienie Pán / Bóg Izráelſki / ludowi ſwemu / y będźie mieƺkał w Jeruzálemie áż ná wieki.
Do tego y Lewitowie nie będą więcey nośić przybytku y wƺyſtkiego nacżynia jego ku poſługiwániu jego ;
Ale według poſtánowieniá Dawidowego oſtátniego / byli policżeni ſynowie Lewiego od dwudźieſtu lat y wyżey ;
Aby zoſtáwáli pod ręką ſynów Aáronowych ku uſłudze domu Páńſkiego w przyśionkách y w gmáchách / y ku ocżyƺcżániu wƺelkich rzecży poświęconych / y ku prácy około uſługi domu Bożego ;
Y około chlebá pokłádnego / y około mąki ná ofiárę / y około plácków niekwáƺonych / y około pánewek / y około rzecży ſmáżonych / y około wƺelkiej miáry y odmierzánia ;
A iżby ſtáli ná káżdy poránek ku wyſłáwiániu / y ku chwáleniu Páná / tákże y w wiecżór ;
Nádto / przy káżdym ofiárowániu cáłopalenia Pánu w ſábáty / ná nowiu mieśiącá / y w urocżyſte świętá / według licżby y porządku ich uſtáwicżnie przed Pánem ;
A ták áby pilnowáli ſtráży namiotu zgromádzenia / y ſtráży świątnicy / y ſtráży ſynów Aáronowych / bráći ſwych / w uſłudze domu Páńſkiego.
