Cić ſą ſynowie Izráelowi : Ruben / Symeon / Lewi / y Judá / Iſáſchár y Zábulon /
Dán / Józef / y Benjámin / Neftáli / Gád y Aſer.
Synowie Judy : Her / y Onán / y Selá. Ci trzej urodzili mu śię z córki Sui Chánánejſkiej. Ale Her / pierworodny Judy / był złym przed ocżymá Páńſkimi ; przetoż go zábił.
Támár záśię / niewiáſtká jego / urodziłá mu Fáreſá y Zárę. Wƺyſtkich ſynów Judowych pięć.
Synowie Fáreſowi : Heſron y Hámuel.
Synowie záś Záry : Zámry / y Etán / y Hemán / y Chálkol / y Dárdá ; wƺyſtkich tych było pięć.
A ſynowie Zámrego : Chármi / wnuk Achárowy / który zámięƺánie ucżynił w Izráelu / zgrzeƺywƺy krádzieżą rzecży przeklętych.
A ſynowie Etánowi : Azáryjáƺ.
A ſynowie Eſronowi / którzy mu śię urodzili : Jerámeel / y Rám / y Chálubáj.
Ale Rám ſpłodził Aminádábá / á Aminádáb ſpłodził Nááſoná / kśiążęćiá ſynów Judzkich.
A Nááſon ſpłodził Sálmoná / á Sálmon ſpłodził Boozá.
A Booz ſpłodził Obedá / á Obed ſpłodził Iſájego.
A Iſáj ſpłodził pierworodnego ſwego Elijábá / y Abinádábá wtórego / y Sámmá trzećiego.
Nátánáelá cżwártego / Ráddájá piątego.
Ozemá ƺóſtego / Dawidá śiódmego.
A śioſtry ich : Sárwijá / y Abigáil ; á ſynowie Sárwii : Abiſáj / y Joáb / y Aſáel / trzej.
A Abigáil urodziłá Amázę / á ojćiec Amázy był Jeter Iſmáelcżyk.
A Káleb / ſyn Heſronowy / ſpłodził z Azubą / máłżonką ſwoją / y z Jeryjotą ſynów. A ći byli ſynowie jego : Jeſer / y Sobáb / y Ardon.
A gdy umárłá Azubá / pojął ſobie Káleb Eſrátę / którá mu urodziłá Hurá.
A Hur ſpłodził Ury / á Ury ſpłodził Beſáleelá.
Potem wƺedł Heſron do córki Máchyrá / ojcá Gáláádowego / á pojął ją / będąc w ƺeśćdzieśiąt lát ; którá mu urodziłá Segubá.
A Segub ſpłodził Jáirá / który miáł dwádzieśćiá trzy miáſt w ziemi Gáláádſkiej.
Bo wziął Gieſſurytom / y Aſſyryjcżykom wśi Jáirowe / y Kánát z miáſtecżkámi jego / ƺeśćdzieśiąt miáſt. To wƺyſtko pobráli ſynowie Máchyrá / ojcá Gáláádowego.
A gdy umárł Heſron w Káleb Efrátá / tedy żoná Heſronowá Abijá porodziłá mu Aſſurá / ojcá Tekui.
Byli też ſynowie Jerámeelowi / pierworodnego Heſronowego : Pierworodny Rám / po nim Buná / y Oren / y Oſem z Abii.
Miáł tákże drugą żonę Jerámeel / imieniem Atárá ; tá jeſt mátká Onámowá.
Ale ſynowie Rámowi / pierworodnego Jerámeelowego / byli : Mááſ / y Jámin / y Achár.
Byli też ſynowie Onámowi : Semáj / y Jádá / á ſynowie Semejego : Nádád y Abiſur ;
A imię żony Abiſurowej Abiháil / którá mu urodziłá Achobbáná y Molidá.
Synowie Nádábowi : Sáled y Affáim ; lecż Sáled umárł bez potomſtwá.
A ſynowie Affáimowi Jeſy ; á ſynowie Jeſy Seſán / á córká Seſáná Achiáláj.
A ſynowie Jády / brátá Semejego / Jeter y Jonátán ; ále Jeter umárł bez potomſtwá.
A ſynowie Jonátánowi : Fálet y Zyzá. Cić byli ſynowie Jerámeelowi.
Lecż nie miáł Seſán ſynów / jedno córki ; miáł też Seſán ſługę Egipcżániná / imieniem Jeráchá.
Y dáł Seſán córkę Jeráchowi / ſłudze ſwemu / zá żonę / którá mu urodziłá Etájá.
Etáj ſpłodził Nátáná / á Nátán ſpłodził Zábádá.
A Zábád ſpłodził Efijálá / á Efijál ſpłodził Obedá.
A Obed ſpłodził Jehu / á Jehu ſpłodził Azáryjáƺá.
A Azáryjáƺ ſpłodził Heleſá / á Heleſ ſpłodził Eláſá.
A Eláſ ſpłodził Syſmájá / á Syſmáj ſpłodził Sellumá.
A Sellum ſpłodził Ikámijáƺá / á Ikámijáƺ ſpłodził Eliſámá.
Y ſynowie Kálebá / brátá Jerámeelowego : Meſá pierworodny jego / który był ojcem Zyfejcżyków y ſynów Máreſy / ojcá Hebronowego.
A ſynowie Hebronowi : Kore y Táfuá / y Rechem / y Semmá.
A Semmá ſpłodził Ráhámá / ſyná Jerkáámowego / á Rechem ſpłodził Sámmájego.
A Sámmáj był ſynem Máonowym / á Máon był ojcem Betſurcżyków.
Efá też / záłożnicá Kálebowá / urodziłá Háráná / y Moze / y Giezezá ; á Hárán ſpłodził Giezezá.
A ſynowie Jáchdájowi : Regiem / y Jotám / y Gieſán / y Fálet / y Efá / y Sááf /
Záłożnicá záś drugá Kálebowá Mááchá urodziłá Sáberá / y Tyrcháná.
A żoná Sááfowá urodziłá ojcá Mádmeńcżyków / y Sewá / ojcá Máchbeńcżyków / y ojcá Gábáońcżyków ; á córká Kálebowá byłá Achſá.
Cić byli ſynowie Kálebá / ſyná Hurowego / pierworodnego Efráty : Sobál / ojćiec Káryjátyjárymcżyków.
Sálmá / ojćiec Betlehemcżyków / Cháref / ojćiec Betgádercżyków.
Miáł też ſynów Sobál / ojćiec Káryjátyjárymcżyków / który doglądáł połowy Menuchoty.
A domy Káryjátyjárymſkie były Jetrejcżycy / y Futejcżycy / y Semátejcżycy / y Máſerejcżycy / z których też poƺli Sáráitowie / y Eſtáolitowie.
A ſynowie Sálmy : Betlehemcżycy / y Netofátcżycy / ozdoby domu Joábowego / y połowá Mánácháty / ojcá Sorygo.
A domy piſárzów mieƺkájących w Jebeſ : Tyryjátejcżycy / Symátejcżycy / Suchátejcżycy. Cić ſą Cynejcżycy / którzy poƺli z Hemátá / ojcá domu Rechábowego.
