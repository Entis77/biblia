A ták wƺyſcy Izráelcżycy oblicżeni ſą ; á oto zápiſáni ſą w kśięgách królów Izráelſkich y Judzkich / á przenieśieni ſą do Bábilonu dlá przeſtępſtwá ſwego.
Lecż którzy pierwśi mieƺkáli w ośiádłośćiách ſwych y w miáſtách ſwoich / Izráelcżycy / kápłáni / Lewitowie / y Netynejcżycy.
W Jeruzálemie mieƺkáli z ſynów Judowych / y z ſynów Benjáminowych / y z ſynów Efráimowych / y Mánáſeſowych ;
Uttáj / ſyn Ammiudá / ſyná Amry / ſyná Imry / ſyná Bonny / z ſynów Fáreſá / ſyná Judowego.
A z Syloná : Aſájáƺ pierworodny / y ſynowie jego.
A z ſynów Zeráchowych : Jehuel y bráći ich ƺeść ſet y dźiewięćdźieśiąt.
A z ſynów Benjáminowych : Sálu / ſyn Meſullámá / ſyná Hodowiego / ſyná Aſenuowego.
A Ibnijáƺ / ſyn Jerochámowy / y Elá / ſyn Uzego / ſyná Michry / y Meſullám / ſyn Sefátyjáƺá / ſyná Rehuelowego / ſyná Ibnijáƺowego.
Tákże bráći ich według národów ich było dźiewięć ſet y pięćdźieśiąt y ƺeść : ći wƺyſcy mężowie byli kśiążętámi rodzájów według domów ojców ſwoich.
Z kápłánów záśię : Jedájáƺ / y Jechojáryk / y Jáchyn /
Y Azáryjáƺ / ſyn Helkijáƺá / ſyná Meſullámowego / ſyná Sádokowego / ſyná Merájátowego / ſyná Achytobowego / był kśiążęćiám domu Bożego.
Y Adájáƺ / ſyn Jerohámá / ſyná Fáſſurowego / ſyná Melchyjáƺowego / y Mááſáj / ſyn Adyjelá / ſyná Jáchſerowego / ſyná Meſullámowego / ſyná Meſullemitowego / ſyná Immerowego.
A bráći ich kśiążęt według domów ojców ich było tyśiąc y śiedm ſet y ƺeśćdźieśiąt / mążów dużych ku ſpráwowániu poſługi w domu Bożym.
A z Lewitów : Semejáƺ / ſyn Háſſubá / ſyná Aſrykámowego / ſyná Háſábijáƺowego / z ſynów Merárego ;
Y Bákkábár / Chereſ / y Gálál / y Mátánijáƺ / ſyn Michy / ſyná Zychrego / ſyná Aſáfowego ;
Obádyjáƺ też / ſyn Semáhájáƺá / ſyná Gálálowego / ſyná Jedytunowego / y Báráchyjáƺ / ſyn Aſy / ſyná Elkánowego / który mieƺkáł we wśiách Netofátyckich.
A odźwierni : Sállum / y Akkub / y Tálmon / y Ahymán / y bráćiá ich ; Sállum kśiążę między nimi.
Który áż dotąd w brámie królewſkiej ſtáwáł ná wſchód ſłońcá ; ći byli odźwiernymi według pocżtów ſynów Lewiego.
Ale Sállum / ſyn Korego / ſyná Abijázáfowego / ſyná Korego / y bráćiá jego z domu ojcá jego / Korytcżycy byli ku odpráwowániu poſługi / ſtróżámi progów námiotu ; á ojcowie ich byli nád obozem Páńſkim ſtróżámi wejśćiá.
A Fineeſ / ſyn Eleázárowy / był kśiążęćiem nád nimi / á Pán był z nim.
Zácháryjáƺ záśię / ſyn Meſellemijáƺowy / był odźwiernym drzwi u námiotu zgromádzeniá.
Cić wƺyſcy ſą obráni zá odźwiernych do drzwi / dwieśćie oſób y dwánáśćie : ći we wśiách ſwych policżeni ſą / których poſtánowił Dawid y Sámuel widzący / dlá wiernośći ich /
Aby oni y ſynowie ich byli we drzwiách domu Páńſkiego / w domu námiotu ná ſtráży.
Y byli odźwierni po cżterech ſtronách / ná wſchód / ná záchód / ná północy / y ná południe.
Bráćiá záśię ich byli we wśiách ſwych / przychodząc káżdego śiódmego dniá / od cżáſu áż do cżáſu odmieniájąc śię z nimi.
Abowiem pod ſpráwą tych cżterech przedniejƺych odźwiernych byli Lewitowie / á byli przełożeni nád gmáchámi y nád ſkárbámi domu Bożego ;
A około domu Bożego nocowáli / gdyż im náleżáłá ſtráż jego ; á oni go ná káżdy poránek otwieráli.
A z nich niektórzy byli nád nácżyniem ku poſługiwániu ; ábowiem pod licżbą wnośili je / y pod licżbą wynośili je.
Niektórzy záśię z nich byli poſtánowieni nád innem nácżyniem / y nád wƺyſtkiem nácżyniem świątnicy / y nád mąką pƺenną y winem / y oliwą / y kádźidłem / y nád rzecżámi wonnemi.
A niektórzy z ſynów kápłáńſkich ſpráwowáli máśći z rzecży wonnych.
Mátátyjáƺ też z Lewitów / pierworodny Sállumá Korytcżyká / był przełożony nád rzecżámi / które w pánwiách ſmáżono.
A z ſynów Káátowych z bráći ich / byli niektórzy przełożonymi nád chlebámi pokłádnemi / áby je gotowáli ná káżdy ſábát.
A z tych byli śpiewácy / przedniejśi z domów ojcowſkich / między Lewitámi mieƺkájący w gmáchách / od inƺych prác wolni ; bo we dnie y w nocy powinnośći ſwej pilnowáć muśieli.
Ci przedniejśi z domów ojcowſkich między Lewitámi / według národów ſwych przedniejśi ; ći mieƺkáli w Jeruzálemie.
A w Gábáonie mieƺkáli ojćiec Gábáońcżyków Jehyjel / á imię żony jego Mááchá ;
A ſyn jego pierworodny Abdon / po nim Sur / y Cyſ / y Báál / y Neer / y Nádáb /
Y Giedeor / y Achyjo / y Zácháryjáƺ / y Michlot ;
( A Michlot ſpłodźił Symmámá ) á ći tákże przećiw bráći ſwych mieƺkáli w Jeruzálemie z bráćmi ſwymi.
A Neer ſpłodźił Cyſá / á Cyſ ſpłodźił Saulá / á Saul ſpłodźił Jonátáná / y Melchiſuego / y Abinádábá / y Eſbáálá.
A ſyn Jonátáná Merybbáál ; á Merybbáál ſpłodźił Micháſá.
Synowie záś Micháſowi : Fiton / y Melech / y Táreá / y Acház.
A Acház ſpłodźił Járę ; á Járá ſpłodźił Alemetá / y Aſmáwetá / y Zymrego ; á Zymry ſpłodźił Moſę.
A Moſá ſpłodźił Binę ; á Refájáƺ ſyn jego / Eláſá ſyn jego / Aſel ſyn jego.
A Aſel miáł ƺeść ſynów ; á teć imioná ich : Aſrykám / y Bochru / y Iſmáel / y Seáryjáƺ / y Obádyjáƺ / y Hánán. Cić ſą ſynowie Aſelowi.
