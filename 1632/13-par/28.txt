Tedy zgromádźił Dawid wƺyſtkich kśiążąt Izráelſkich / y przedniejƺych z káżdego pokolenia / y przełożonych nád hufcámi / którzy ſłużyli królowi / y półkowników / y rotmiſtrzów / y przełożonych nád wƺyſtką májętnośćią y ośiádłośćią królewſką ; ſynów też ſwojich z komornikámi / y z innymi możnymi / y ze wƺyſtkim ludem rycerſkim do Jeruzálemu.
A powſtáwƺy król Dawid ná nogi ſwoje / rzekł : Słuchájćie mię / bráćia moi / y ludu mój! Jam był umyślił w ſercu ſwem / budowáć dom / gdźieby odpocżywáłá ſkrzyniá przymierza Páńſkiego / y ná podnóżek nóg Bogá náƺego / y zgotowáłem był potrzeby ku budowániu ;
Ale Bóg rzekł do mnie : Nie będźieƺ budował domu imieniowi memu / przeto żeś mąż wálecżny / y rozlewáłeś krew.
Ale obrał mię Pán / Bóg Izráelſki / ze wƺyſtkiego domu ojcá mego / ábym był królem nád Izráelem ná wieki ; bo z Judy obrał kśięćiá / á z narodu Judzkiego dom ojcá mego ; y z ſynów ojcá mego upodobał mię ſobie zá królá nád wƺyſtkim Izráelem.
A ze wƺyſtkich ſynów mojich ( bo mi wiele ſynów Pán dał ) obrał Sálomoná / ſyná mego / áby śiedźiał ná ſtolicy króleſtwá Páńſkiego nád Izráelem.
Y mówił do mnie : Sálomon / ſyn twój / ten zbuduje dom mój / y przyśionki moje ; ábowiemem go ſobie obrał zá ſyná / á Ja mu będę zá ojcá.
Y umocnię króleſtwo jego áż ná wieki / będźieli ſtátecżnym w pełnieniu przykázáń mojich y ſądów mojich / jáko y dźiś.
Teraz tedy mówię wam przed oblicżem wƺyſtkiego Izráelá / zgromádzenia tego Páńſkiego / gdźie ſłyƺy Bóg náƺ : Strzeżćie á ƺukájćie wƺyſtkich rozkázáń Páná / Bogá wáƺego / ábyśćie ośiedli źiemię dobrą / y zoſtáwili ją w dźiedźictwo ſynom ſwoim po ſobie áż ná wieki.
A ty Sálomonie / ſynu mój! znáj Bogá / ojcá twojego / y ſłuż mu ſercem doſkonáłem / y umyſłem dobrowolnym ; bo wƺyſtkie ſercá przeglądá Pán / y wƺyſtkie zámyſły myśli zná. Jeſli go ƺukáć będźieƺ / znájdźieƺ go / á jeſli go opuśćiƺ / odrzući ćię ná wieki.
Obácżże teraz / iż ćię Pán obrał / ábyś zbudował dom świątnicy ; zmácniájże śię á wykonáj to.
Tedy oddał Dawid Sálomonowi / ſynowi ſwemu / wizerunek przyśionká / y gmáchów jego / y komor jego / y ſál jego / y wnętrznych pokojów jego / y domu ubłágálni.
Przytem wizerunek wƺyſtkiego / co był umyślił o śieni domu Páńſkiego / y o wƺyſtkich gmáchách dla ſkárbów domu Bożego / y dla ſkárbów rzecży świętych ;
Y dla pocżtów kápłáńſkich / y Lewitów / y dla wƺyſtkiej prácy w uſłudze domu Páńſkiego / y dla wƺyſtkiego nacżynia ſłużby domu Páńſkiego.
Tákże złotá pewną wágę ná wƺyſtkie nacżynia złote / od wƺyſtkiej uſługi ; ſrebrá tákże ná wƺyſtkie nacżynia ſrebrne pewną wágę / ná wƺyſtkie nacżynia ku wƺelákiej uſłudze ;
Miánowićie pewną wágę ná świecżniki złote y ná lámpy ich złote według wági káżdego świecżniká y lámp jego / y ná świecżniki ſrebrne według wági świecżniká káżdego y lámp jego / według potrzeby káżdego świecżniká.
Tákże pewną wágę złotá ná ſtoły chlebów pokłádnych / ná káżdy ſtół / przytem ſrebrá ná ſtoły ſrebrne.
A ná widełki / y ná koćiołki / y ná kádźielnice ƺcżerego złotá / y ná cżáƺe złote / pewną wágę ná káżdą cżáƺę / y ná cżáƺe ſrebrne / pewną wágę ná káżdą cżáƺę.
Tákże ná ołtárz do kádzeniá dał złotá ƺcżerego pewną wágę / y złotá ku wyſtáwieniu wozá Cherubinów / którzyby rozćiągnionemi ſkrzydłámi okrywáli ſkrzynię przymierza Páńſkiego.
To wƺyſtko / rzekł Dawid / opiſáne z ręki Páńſkiej mię doƺło / ábym zrozumiał wƺyſtko / jáko co urobić miáno.
A ták rzekł Dawid do Sálomoná / ſyná ſwego : Zmácniáj śię / á bądź mężnym / cżyń to ; nie bój śię / áni śię lękáj ; bo Pán Bóg / Bóg mój / będźie z tobą / nie opuśći ćię / áni ćię odſtąpi / áż dokońcżyƺ wƺyſtkiej roboty ſłużby domu Páńſkiego.
A oto pocżty kápłanów y Lewitów do káżdej poſługi w domu Bożym będą z tobą w káżdej prácy ; káżdy ochotny y roztropny przy wƺelkiej poſłudze / tákże kśiążętá / y wƺyſtek lud ſtáną ná káżde rozkázánie twoje.
