Tedy zgromádźił Dawid wƺyſtkich kśiążąt Izráelſkich / y przedniejƺych z káżdego pokoleniá / y przełożonych nád hufcámi / którzy ſłużyli królowi / y półkowników / y rotmiſtrzów / y przełożonych nád wƺyſtką májętnośćią y ośiádłośćią królewſką ; ſynów też ſwoich z komornikámi / y z innymi możnymi / y ze wƺyſtkim ludem rycerſkim do Jeruzálemu.
A powſtáwƺy król Dawid ná nogi ſwoje / rzekł : Słuchájćie mię / bráćia moi / y ludu mój! Jám był umyślił w ſercu ſwem / budowáć dom / gdźieby odpocżywáłá ſkrzyniá przymierzá Páńſkiego / y ná podnóżek nóg Bogá náƺego / y zgotowáłem był potrzeby ku budowániu ;
Ale Bóg rzekł do mnie : Nie będźieƺ budowáł domu imieniowi memu / przeto żeś mąż wálecżny / y rozlewáłeś krew.
Ale obráł mię Pán / Bóg Izráelſki / ze wƺyſtkiego domu ojcá mego / ábym był królem nád Izráelem ná wieki ; bo z Judy obráł kśięćiá / á z národu Judzkiego dom ojcá mego ; y z ſynów ojcá mego upodobáł mię ſobie zá królá nád wƺyſtkim Izráelem.
A ze wƺyſtkich ſynów moich ( bo mi wiele ſynów Pán dáł ) obráł Sálomoná / ſyná mego / áby śiedźiáł ná ſtolicy króleſtwá Páńſkiego nád Izráelem.
Y mówił do mnie : Sálomon / ſyn twój / ten zbuduje dom mój / y przyśionki moje ; ábowiemem go ſobie obráł zá ſyná / á Ja mu będę zá ojcá.
Y umocnię króleſtwo jego áż ná wieki / będźieli ſtátecżnym w pełnieniu przykázáń moich y ſądów moich / jáko y dźiś.
Teraz tedy mówię wám przed oblicżem wƺyſtkiego Izráelá / zgromádzeniá tego Páńſkiego / gdźie ſłyƺy Bóg náƺ : Strzeżćie á ƺukájćie wƺyſtkich rozkázáń Páná / Bogá wáƺego / ábyśćie ośiedli źiemię dobrą / y zoſtáwili ją w dźiedźictwo ſynom ſwoim po ſobie áż ná wieki.
A ty Sálomonie / ſynu mój! znáj Bogá / ojcá twojego / y ſłuż mu ſercem doſkonáłem / y umyſłem dobrowolnym ; bo wƺyſtkie ſercá przeglądá Pán / y wƺyſtkie zámyſły myśli zná. Jeſli go ƺukáć będźieƺ / znájdźieƺ go / á jeſli go opuśćiƺ / odrzući ćię ná wieki.
Obácżże teraz / iż ćię Pán obráł / ábyś zbudowáł dom świątnicy ; zmácniájże śię á wykonáj to.
Tedy oddáł Dawid Sálomonowi / ſynowi ſwemu / wizerunek przyśionká / y gmáchów jego / y komor jego / y ſál jego / y wnętrznych pokojów jego / y domu ubłágálni.
Przytem wizerunek wƺyſtkiego / co był umyślił o śieni domu Páńſkiego / y o wƺyſtkich gmáchách dla ſkárbów domu Bożego / y dla ſkárbów rzecży świętych ;
Y dla pocżtów kápłáńſkich / y Lewitów / y dla wƺyſtkiej prácy w uſłudze domu Páńſkiego / y dla wƺyſtkiego nácżyniá ſłużby domu Páńſkiego.
Tákże złotá pewną wágę ná wƺyſtkie nácżyniá złote / od wƺyſtkiej uſługi ; ſrebrá tákże ná wƺyſtkie nácżyniá ſrebrne pewną wágę / ná wƺyſtkie nácżyniá ku wƺelákiej uſłudze ;
Miánowićie pewną wágę ná świecżniki złote y ná lámpy ich złote według wági káżdego świecżniká y lámp jego / y ná świecżniki ſrebrne według wági świecżniká káżdego y lámp jego / według potrzeby káżdego świecżniká.
Tákże pewną wágę złotá ná ſtoły chlebów pokłádnych / ná káżdy ſtół / przytem ſrebrá ná ſtoły ſrebrne.
A ná widełki / y ná koćiołki / y ná kádźielnice ƺcżerego złotá / y ná cżáƺe złote / pewną wágę ná káżdą cżáƺę / y ná cżáƺe ſrebrne / pewną wágę ná káżdą cżáƺę.
Tákże ná ołtárz do kádzeniá dáł złotá ƺcżerego pewną wágę / y złotá ku wyſtáwieniu wozá Cherubinów / którzyby rozćiągnionemi ſkrzydłámi okrywáli ſkrzynię przymierzá Páńſkiego.
To wƺyſtko / rzekł Dawid / opiſáne z ręki Páńſkiej mię doƺło / ábym zrozumiáł wƺyſtko / jáko co urobić miáno.
A ták rzekł Dawid do Sálomoná / ſyná ſwego : Zmácniáj śię / á bądź mężnym / cżyń to ; nie bój śię / áni śię lękáj ; bo Pán Bóg / Bóg mój / będźie z tobą / nie opuśći ćię / áni ćię odſtąpi / áż dokońcżyƺ wƺyſtkiej roboty ſłużby domu Páńſkiego.
A oto pocżty kápłánów y Lewitów do káżdej poſługi w domu Bożym będą z tobą w káżdej prácy ; káżdy ochotny y roztropny przy wƺelkiej poſłudze / tákże kśiążętá / y wƺyſtek lud ſtáną ná káżde rozkázánie twoje.
