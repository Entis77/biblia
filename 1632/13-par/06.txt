Synowie Lewiego : Gierſon / Káát / y Meráry.
A ſynowie Káátowi : Amrám / Izáár / y Hebron / y Huſyjel.
A ſynowie Amrámowi : Aáron / y Mojzeƺ / y córká Máryjá ; á ſynowie Aáronowi : Nádáb / y Abiju / Eleázár / y Itámár.
Eleázár ſpłodźił Fineeſá ; Finees ſpłodźił Abiſuá.
Abiſue ſpłodźił Bokki / á Bokki ſpłodźił Uzy.
A Uzy ſpłodźił Zeráchyjáƺá / á Zeráchyjáƺ ſpłodźił Merájotá.
Merájot ſpłodźił Amáryjáƺá / á Amáryjáƺ ſpłodźił Achytobá.
A Achytob ſpłodźił Sádoká / á Sádok ſpłodźił Achymááſá.
Achymáás ſpłodźił Azáryjáƺá / á Azáryjáƺ ſpłodźił Johánáná.
A Johánán ſpłodźił Azáryjáƺá ; tenći jeſt / który kápłáńſki urząd ſpráwował w domu / który zbudował Sálomon w Jeruzálemie.
Spłodźił też Azáryjáƺ Amáryjáƺá / á Amáryjáƺ ſpłodźił Achytobá.
A Achytob ſpłodźił Sádoká / á Sádok ſpłodźił Sállumá.
A Sállum ſpłodźił Helkijáƺá / á Helkijáƺ ſpłodźił Azáryjáƺá.
A Azáryjáƺ ſpłodźił Sárájáƺá / á Sárájáƺ ſpłodźił Jozedeká.
Ale Jozedek poƺedł w niewolę / gdy Pán przenióſł Judę y Jeruzálem przez Nábuchodonozorá.
Synowie Lewi : Gierſon / Káát / y Meráry.
A teć ſą imioná ſynów Gierſonowych : Lobni y Semei.
A ſynowie Káátowi : Amrám y Izáár / y Hebron / y Huſyjel.
Synowie Merárego : Mácheli / y Muzy. A teć ſą domy Lewitów według ojców ich.
Gierſonowi : Lobni ſyn jego / Jáchát ſyn jego / Zámmá ſyn jego ;
Joách ſyn jego / Iddo ſyn jego / Zárá ſyn jego / Jetráj ſyn jego.
Synowie Káátowi : Aminádáb ſyn jego / Kore ſyn jego / Aſer ſyn jego.
Elkáná ſyn jego / y Abiázáf ſyn jego / y Aſſyr ſyn jego ;
Táchát ſyn jego / Uryjel ſyn jego / Ozyjáƺ ſyn jego / y Saul ſyn jego.
A ſynowie Elkánowi : Amáſáj y Achymot.
Elkáná. Synowie Elkánowi : Sofáj ſyn jego / y Náhát ſyn jego ;
Elijáb ſyn jego / Jerobám ſyn jego / Elkáná ſyn jego.
A ſynowie Sámuelowi : Pierworodny Wáſſni y Abijás.
Synowie Merárego : Máháli ; Lobni ſyn jego / Symej ſyn jego / Uzá ſyn jego ;
Symhá ſyn jego / Hággijáƺ ſyn jego / Aſájáƺ ſyn jego.
Ci ſą / których poſtánowił Dawid do śpiewánia w domu Páńſkim / gdy tám poſtáwiono ſkrzynię.
Y ſłużyli przed przybytkiem namiotu zgromádzenia / śpiewájąc / áż zbudował Sálomon dom Páńſki w Jeruzálemie / y ſtáli według porządku ſwego ná ſłużbie ſwojey.
A ćić ſą / którzy ſtáli / y ſynowie ich z ſynów Káátowych : Hemán śpiewák / ſyn Joelá / ſyná Sámuelowego /
Syná Elkánowego / ſyná Jerohámowego / ſyná Elijelowego / ſyná Tohu /
Syná Sufowego / ſyná Elkánowego / ſyná Máchátowego / ſyná Amáſájowego /
Syná Elkánowego / ſyná Joelowego / ſyná Azáryjáƺowego / ſyná Sofonijáƺowego /
Syná Táchátowego / ſyná Aſſyrowego / ſyná Abijáſowego /
Syná Korego / ſyná Iſárowego / ſyná Káátowego / ſyná Lewiego / ſyná Izráelowego.
A brát jego Aſáf / który ſtáwał po práwicy jego. Aſáf / ſyn Báráchyjáƺowy / ſyná Sámáowego /
Syná Micháelowego / ſyná Báſejáƺowego / ſyná Málchyjáƺowego /
Syná Etny / ſyná Zeráchowego / ſyná Adájowego /
Syná Etánowgo / ſyná Symmowego /
Syná Semejowego / ſyná Jáchátowego / ſyná Gierſonowego / ſyná Lewiego.
A ſynowie Merárego y bráćia ich ſtáwáli po lewej ſtronie : Etán / ſyn Kuzego / ſyná Abdego / ſyná Málluchowego /
Syná Háſábijáƺowego / ſyná Amázyjáƺowego / ſyná Helkijáƺowego.
Syná Amſego / ſyná Bánego / ſyná Semmerowego /
Syná Moholi / ſyná Muſego / ſyná Merárego / ſyná Lewiego.
A bráćia ich Lewitowie poſtáwieni ſą ku wƺelákiej poſłudze przybytku domu Bożego.
Ale Aáron y ſynowie jego pálili ná ołtárzu cáłopalenia / y ná ołtárzu kádzeniá przy káżdej poſłudze świątnicy świętych / y ku ocżyƺcżániu Izráelá podług wƺyſtkiego / jáko był przykázał Mojzeƺ / ſługá Boży.
A ći ſą ſynowie Aáronowi : Eleázár ſyn jego / Finees ſyn jego /
Abiſuá ſyn jego / Bokki ſyn jego / Uzy ſyn jego / Zeráchyjáƺ ſyn jego /
Merájot ſyn jego / Amáryjáƺ ſyn jego / Achytob ſyn jego /
Sádok ſyn jego / Achymáás ſyn jego.
A te ſą mieƺkánia ich / według páłáców ich w gránicy ich / to jeſt / ſynów Aáronowych według rodzáju Káátytów : bo to ich był los.
Przetoż dáli im Hebron w źiemi Judzkiej / y przedmieśćiá jego około niego ;
Ale pole miejſkie y wśi ich dáli Kálebowi / ſynowi Jefunowemu.
Synom záś Aáronowym dáli z miaſt Judzkich miáſtá ućiecżki Hebron / y Lobne y przedmieśćiá jego / y Jeter / y Eſtemoá / y z przedmieśćiámi jego ;
Y Holon y przedmieśćiá jego / y Dábir y przedmieśćiá jego ;
Y Aſán y przedmieśćiá jego / y Betſemes y przedmieśćiá jego.
A z pokolenia Benjáminowego : Gábáe y przedmieśćiá jego / y Almát y przedmieśćiá jego / y Anátot y przedmieśćiá jego. Wƺyſtkich miaſt ich trzynáśćie miaſt według domów ich.
A ſynom Káátowym / pozoſtáłym z rodzáju tegoż pokolenia / doſtáło śię w połowie pokolenia Mánáſeſowego loſem miaſt dźieśięć.
A ſynom Gierſonowym według domów ich doſtáło śię w pokoleniu Iſáſchárowem / y w pokoleniu Aſerowem / y w pokoleniu Neftálimowem / y w pokoleniu Mánáſeſowem w Bázán miaſt trzynáśćie.
Synom Merárego według domów ich doſtáło śię w pokoleniu Rubenowem / y w pokoleniu Gádowem / y w pokoleniu Zábulonowem loſem miaſt dwánáśćie.
Dáli też ſynowie Izráelſcy Lewitom miáſtá y przedmieśćiá ich ;
A dáli je loſem w pokoleniu ſynów Judowych / y w pokoleniu ſynów Symeonowych / y w pokoleniu ſynów Benjáminowych / miáſtá te / które názwáli imiony ſwemi.
A tym / którzy byli z rodu ſynów Káátowych / ( á były miáſtá y gránice ich w pokoleniu Efráim. )
Tym dáli z miaſt ućiecżki Sychem y przedmieśćiá jego ná górze Efráim / y Gázer y przedmieśćiá jego.
Y Jekmáán y przedmieśćiá jego / y Betoron y przedmieśćiá jego ;
Y Ajálon y przedmieśćiá jego / y Gátrymon y przedmieśćiá jego.
A w połowie pokolenia Mánáſeſowego : Aner y przedmieśćiá jego / Bálám y przedmieśćiá jego. To dáli rodzájowi pozoſtáłemu ſynów Káátowych.
Synom też Gierſonowym z rodu połowy pokolenia Mánáſeſowego dáli Golán w Bázán y przedmieśćiá jego / y Aſtárot y przedmieśćiá jego ;
A w pokoleniu Iſáſchárowem Kádes y przedmieśćiá jego / Dáberet y przedmieśćiá jego.
Y Rámot y przedmieśćiá jego / y Anam y przedmieśćiá jego.
A w pokoleniu Aſerowem Máſál y przedmieśćiá jego / y Abdon y przedmieśćiá jego /
Y Hukok y przedmieśćiá jego / y Rohob y przedmieśćiá jego.
A w pokoleniu Neftálimowem : Kádes w Gálilei / y przedmieśćiá jego / y Hámmon y przedmieśćiá jego / y Kiryjátáim y przedmieśćiá jego.
Synom Merárego / pozoſtáłym z pokolenia Zábulon / dáne ſą Remmon y przedmieśćiá jego / Tábor y przedmieśćiá jego.
A zá Jordánem u Jerychá ná wſchód ſłońcá od Jordánu / dáne ſą w pokoleniu Rubenowem : Beſor ná puƺcży / y przedmieśćiá jego / y Jáháſá y przedmieśćiá jego.
Y Kiedemot y przedmieśćiá jego / y Mefáát y przedmieśćiá jego.
A w pokoleniu Gádowem Rámot w Gáláád y przedmieśćiá jego ; y Máhánáim y przedmieśćiá jego.
Heſebon y przedmieśćiá jego / y Jázer y przedmieśćiá jego.
