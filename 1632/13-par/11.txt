Zebrał śię tedy wƺyſtek Izráel do Dawidá do Hebronu / mówiąc : Otośmy kość twojá y ćiáło twoje!
Jáko y przed tym / gdy jeƺcże był Saul królem / też wywodźił y wwodźił Izráelá. Ták Pán / Bóg twój / rzekł tobie : Ty będźieƺ páſł lud mój Izráelſki / á ty będźieƺ wodzem nád ludem moim Izráelſkim.
A ták przyƺli wƺyſcy ſtárśi Izráelſcy do królá do Hebronu / y ucżynił Dawid z nimi przymierze w Hebronie przed Pánem / y pomázáli Dawidá zá królá nád Izráelem według ſłowá Páńſkiego / które powiedźiał przez Sámuelá.
Jechał tedy Dawid ze wƺyſtkim Izráelem do Jeruzálemu / które jeſt Jebus / gdźie byli Jebuzejcżycy obywátelámi źiemi.
Y rzekli obywátele Jebuzejſcy do Dawidá : Nie wnijdźieƺ ſám. Ale Dawid wźiął zámek Syońſki / który jeſt miáſtem Dawidowem.
Bo był rzekł Dawid : Ktobykolwiek poráźił Jebuzejcżyká nájpierwej / ten będźie kśiążęćiem y hetmánem. Przetoż wſtąpił nájpierw Joáb / ſyn Sárwii / y zoſtał hetmánem.
Y mieƺkał Dawid ná onym zámku ; dla tego názwano go miáſtem Dawidowem.
Y zbudował miáſto w około / od Mello áż w okrąg ; á Joáb pobudował oſtátek miáſtá.
A ták Dawid im dálej / tym więcey rozmnáżał śię / y roſł ; ábowiem Pán zaſtępów był z nim.
A ćić ſą nájprzedniejśi rycerze / których miał Dawid / którzy śię mężnie ſtáráli z nim o króleſtwo jego ze wƺyſtkim Izráelem / áby go królem ucżynili według ſłowá Páńſkiego nád Izráelem.
A tenći jeſt pocżet rycerzy / których miał Dawid : Jáſobám / ſyn Cháchmonowy / przedniejƺy miedzy trzydźieſtomá ; ten podnióſƺy oƺcżep ſwój ná trzyſtu / jednym rázem ich zábił.
A po nim Eleázár / ſyn Dodonowy / Achochytcżyk ; ten był jednym miedzy trzemá mocárzámi.
Ten był z Dawidem w Páſdámim / gdźie śię zebráli byli Filiſtynowie ku bitwie ; á byłá oná cżęść polá pełna jęcżmieniá / á lud był ućiekł przed Filiſtynámi.
Y ſtánęli w pośród onego polá / y obronili go / á poráźili Filiſtynów : y wybáwił Pán lud wybáwieniem wielkiem.
Ci tákże trzej ze trzydźieſtu przedniejƺych wſtąpili ná ſkáłę do Dawidá do jáſkini Odollám / gdyż wojſko Filiſtyńſkie leżáło obozem w dolinie Ráfáim ;
Abowiem Dawid nátencżás mieƺkał ná zámku / á ſtánowiſko Filiſtyńſkie było nátencżás w Betlehem.
Prágnął tedy Dawid y rzekł : Oby mi śię kto dał nápić wody z ſtudni Betlehemſkiej / która jeſt u bramy!
Przetoż przebiwƺy śię ći trzej przez wojſko Filiſtyńſkie / nácżerpáli wody z ſtudni Betlehemſkiej / która jeſt u bramy / á wźiąwƺy przynieśli do Dawidá. Lecż jey nie chćiał Dawid pić / ále ją wylał ná ofiárę Pánu.
Y rzekł : Nie dáj mi tego / Boże mój / ábym to ucżynić miał! Izali krew tych mężów pić będą / którzy odwáżyli żywot ſwój? ábowiem z odwágą żywotá ſwego przynieśli ją ; y nie chćiał jey pić. Toć ucżynili trzej oni mocárze.
A Abiſáj / brát Joábowy / był przedniejƺy z onych trzech ; tenże podnióſł włócżnię ſwą ná trzyſtá ludu / które pobił / y otrzymał ſłáwę miedzy onymi trzemá.
Z tych trzech nád innych dwu był ſłáwniejƺy / á był ich kśiążęćiem ; jednák onych trzech pierwƺych nie doƺedł.
Bánájás też / ſyn Jojády / ſyn mężá dużego / wielkich ſpráw / z Kábſeelá / ten zábił dwu mocárzów Moábſkich ; ten też zƺedƺy zábił lwá w pośród jámy / gdy był śnieg.
Ten też zábił mężá Egipcżániná / mężá / którego wzroſt był ná pięć łokći. A choćiáż Egipcżánin miał w ręku oƺcżep jáko náwój tkácki / wƺákże przyƺedł do niego z kijem / y wydárł oƺcżep z ręki Egipcżániná / y zábił go oƺcżepem jego.
To ucżynił Bánájás / ſyn Jojády / który tákże ſłáwnym zoſtał miedzy onymi trzemá mocárzámi.
A choć był miedzy onymi trzydźieſtomá ſłáwnym / wƺákże nie doƺedł onych trzech. Y poſtánowił go Dawid nád drábántámi ſwymi.
