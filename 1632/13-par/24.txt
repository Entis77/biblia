A ſynowie Aáronowi tym ſpoſobem rozdźieleni byli : Synowie Aáronowi byli Nádáb y Abiju / Eleázár y Itámár ;
Ale iż Nádáb y Abiju umárli przed oblicżem ojcá ſwego / á ſynów nie mieli : przetoż odpráwowáli urząd kápłáńſki Eleázár y Itámár.
Y podźielił ich Dawid / to jeſt Sádoká z ſynów Eleázárowych / y Achimelechá z ſynów Itámárowych / według urzędu ich w uſługách ich.
Y ználázło śię ſynów Eleázárowych więcej przedniejƺych mężów / niż ſynów Itámárowych / gdy ich podźielił. Z ſynów Eleázárowych było przedniejƺych według domów ojcowſkich ƺeſnáśćie ; ále ſynów Itámárowych według domów ojcowſkich ośm.
A rozdźieleni ſą loſem jedni od drugich ; bo byli przełożonymi nád świątnicą / y przedniejƺymi przed Bogiem / ták z ſynów Eleázárowych / jáko y z ſynów Itámárowych.
A popiſáł ich Semejáƺ / ſyn Nátánáelowy / piſárz / z pokoleniá Lewiego / przed królem y kśiążętámi / y przed Sádokiem kápłánem / y Achimelechem / ſynem Abijátárowym / y przedniejƺymi z domów ojcowſkich / kápłánów y Lewitów ; á náznácżono jeden dom ojcowſki Eleázárowi / á drugi náznácżono Itámárowi.
Y pádł loſ pierwƺy ná Jehojárybá / ná Jedájáƺá wtóry ;
Ná Hárymá trzeći / ná Seorymá cżwárty ;
Ná Málchyjáƺá piąty / ná Mijámáná ƺóſty ;
Ná Akkoſá śiódmy / ná Abijáƺá óſmy ;
Ná Jeſuego dźiewiąty / ná Sechenijáƺá dźieśiąty ;
Ná Eliáſybá jedenáſty / ná Jákimá dwunáſty ;
Ná Huppá trzynáſty / ná Jeſebábá cżternáſty ;
Ná Bilgę piętnáſty / ná Immerá ƺeſnáſty ;
Ná Chezyrá śiedmnáſty / ná Háppiſeſá ośmnáſty ;
Ná Petáchyjáƺá dźiewiętnáſty / ná Ezechyjelá dwudźieſty ;
Ná Jáchyná dwudźieſty y pierwƺy / ná Gámuelá dwudźieſty y wtóry ;
Ná Delájáƺá dwudźieſty y trzeći / ná Máázyjáƺá dwudźieſty y cżwárty.
Cić ſą ſporządzeni w poſługiwániu ſwojim / áby wchodźili do domu Páńſkiego w przemiánách ſwych / jáko zwykli pod rządem Aároná / ojcá ich / jáko mu był rozkázáł Pán / Bóg Izráelſki.
A z ſynów Lewiego / którzy byli pozoſtáli z ſynów Amrámowych / Subájel ; z ſynów Subájelowych Jechdejáƺ.
Z Rechábijáƺá / z ſynów Rechábijáƺowych był przedniejƺy Jeſyjáƺ.
Z Iſááry Sálomit / z ſynów Sálomitowych Jáchát.
A ſynowie Jeryjáƺowi : Amáryjáƺ wtóry / Jeházyjel trzeći / Jekmáán cżwárty.
Synowie Huſyjelowi Michá ; z ſynów Michy Sámir.
Brát Micháſowy Jeſyjáƺ ; z ſynów Jeſyjáƺowych Zácháryjáƺ.
Synowie Merárego : Máheli y Muſy : ſynowie Jáháſyjáƺowi Beno.
Synowie Merárego z Jáháſyjáƺá : Beno / y Soám / y Záchur / y Hybry.
Z Máhálego Eleázár / który nie miáł ſynów.
Z Cyſá / ſynowie Cyſowi Jeráhmeel.
A ſynowie Muſy : Máheli y Eder / y Jerymot. Cić byli ſynowie Lewitów według domów ojców ich.
Y ći też miotáli loſy náprzećiwko bráći ſwoich / ſynom Aáronowym / przed Dawidem królem / y Sádokiem / y Achimelechem / y przedniejƺymi domów ojcowſkich / z kápłánów y Lewitów / z domów ojcowſkich / káżdy przedniejƺy przećiwko brátu ſwemu młodƺemu.
