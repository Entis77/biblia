A gdy przynieśli ſkrzynię Bożą / y poſtáwili ją w pośród namiotu / który był Dawid rozbił / tedy ſpráwowáli cáłopalenia y ofiáry ſpokojne przed Bogiem.
A gdy dokońcżył Dawid ofiárowáć cáłopalenia / y ofiár ſpokojnych / błogoſłáwił ludowi w imię Páńſkie.
Y rozdźielił wƺyſtkim mężom Izráelſkim / od mężá áż do niewiáſty / káżdemu po bochenku chlebá / y po ƺtuce mięſá / y po łágiewce winá.
A poſtánowił przed ſkrzynią Páńſką z Lewitów ſługi / áby wſpomináli / y wyznáwáli / y chwálili Páná / Bogá Izráelſkiego.
Aſáf był przedniejƺy / á wtóry po nim Zácháryjáƺ / Jehyjel / y Semirámot / y Jechyjel y Mátytyjáƺ / y Elijáb / y Benájáƺ / y Obededom / y Jechyjel ; ći ná inſtrumentách / ná lutniách / ná hárfách / ále Aſáf ná cymbáłách / gráli.
Benájáƺ záś y Jácházyjel kápłáni z trąbámi uſtáwicżne byli przed ſkrzynią przymierza Bożego.
Dopiero dniá onego nájpierwej poſtánowił Dawid / áby tym pſálmem chwálony był Pán przez Aſáfá y bráći jego :
Wyſłáwiájćie Páná / wzywájćie imienia jego / á opowiádájćie miedzy národámi ſpráwy jego.
Śpiewájćie mu / grájćie mu / rozmáwiájćie o wƺyſtkich cudách jego.
Chlubćie śię w imieniu świętym jego / á niech śię rozráduje ſerce ƺukájących Páná.
Szukájćie Páná / y mocy jego ; ƺukájćie oblicża jego záwżdy.
Wſpominájćie dźiwne ſpráwy jego / które cżynił / y cudá jego / y ſądy uſt jego.
O naśienie Izráelſkie / ſłudzy jego! O ſynowie Jákóbowi / wybráni jego!
On jeſt Pán / Bóg náƺ ; po wƺyſtkiej źiemi ſądy jego.
Pámiętájćie áż ná wieki ná przymierze jego / ná ſłowo / które przykázał do tyśiącżnego pokolenia ;
Które poſtánowił z Abráhámem / y ná przyśięgę jego z Izáákiem ;
Y poſtánowił to Jákóbowi zá práwo / á Izráelowi zá przymierze wiecżne /
Mówiąc : Tobie dam źiemię Chánánejſką zá ƺnur dźiedźictwá wáƺego.
Choć was byłá máłá licżbá / á przez krótki cżás byliśćie przychodniámi w niej ;
Y przechodźili od narodu do narodu / y od króleſtwá do innego ludu :
Nie dopuśćił nikomu / áby ich ućiſkáć miał / y kárał dla nich królów /
Mówiąc : Nie tykájćie pomázáńców mojich / á prorokom moim nie cżyńćie nic złego.
Śpiewájćie Pánu wƺyſtká źiemio ; opowiádájćie ode dniá do dniá zbáwienie jego.
Opowiádájćie miedzy národámi chwałę jego / y miedzy wƺyſtkimi ludźmi dźiwne ſpráwy jego ;
Bo wielki jeſt Pán / y chwálebny bárzo / y ſtráƺniejƺy nád wƺyſtkich bogów ;
Gdyż wƺyſcy bogowie pogáńſcy ſą báłwánámi ; ále Pán niebioſá ucżynił.
Słáwá y zácność przed nim / moc y weſele ná miejſcu jego.
Przynieśćie Pánu pokolenia narodów / przynieśćie Pánu chwałę y moc.
Przynieśćie Pánu chwałę imienia jego / przynieśćie dáry / á przychodźćie przed oblicżność jego ; kłániájćie śię Pánu w ozdobie świętobliwośći.
Bójćie śię oblicża jego wƺyſtká źiemio / á będźie utwierdzony okrąg źiemi / áby śię nie poruƺył.
Niech śię rozrádują niebioſá / á niech śię rozweſeli źiemiá / á niech mówią w národách : Pán króluje!
Niech záƺumi morze / y ze wƺyſtkim / co w nim jeſt ; niech śię rozráduje pole / y wƺyſtko / co ná nim jeſt.
Tedy śię rozweſelą drzewá leśne przed Pánem ; ábowiem przyƺedł ſądźić źiemię.
Wyſłáwiájćie Páná ; ábowiem dobry / bo ná wieki trwá miłośierdźie jego.
A mówćie : Záchowáj nas / Boże zbáwieniá náƺego! y zgromádź nas / á wyrwij nas od pogán / ábyſmy wielbili imię święte twoje / y chlubili śię w chwále twojey.
Błogoſłáwiony Pán / Bóg Izráelſki / od wieków áż ná wieki. Y rzekł wƺyſtek lud Amen / y chwálił Páná.
Y zoſtáwił tám Dawid przed ſkrzynią przymierza Páńſkiego Aſáfá y bráći jego / áby ſłużyli przed ſkrzynią uſtáwicżnie według potrzeby dniá káżdego.
Lecż Obededomá y bráći ich ƺeśdźieśiąt y ośm / Obededomá mówię / ſyná Jedytunowego / y Hoſę / ucżynił odźwiernymi.
A Sádoká kápłáná / y bráći jego kápłanów poſtáwił przed przybytkiem Páńſkim ná wyżynie / która byłá w Gábáon /
Aby ofiárowáli cáłopalenia Pánu ná ołtárzu cáłopalenia uſtáwicżnie ráno y w wiecżór / á to według wƺyſtkiego / co nápiſano w zakonie Páńſkim / który przykázał Izráelowi.
A z nimi Hemáná y Jedytuná / y innych ná to obránych / którzy byli z imienia miánowáni / áby chwálili Páná / przeto iż ná wieki trwá miłośierdźie jego.
A miedzy nimi Hemán y Jedytun / trąbili y gráli ná trąbách / ná cymbáłách / y ná innych inſtrumentách muzycżnych Bogu ; ále ſynów Jedytunowych poſtáwił u wrót.
A ták rozƺedł śię wƺyſtek lud / káżdy do domu ſwego. Dawid śię też wróćił / áby błogoſłáwił domowi ſwemu.
