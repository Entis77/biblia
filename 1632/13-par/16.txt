A gdy przynieśli ſkrzynię Bożą / y poſtáwili ją w pośród námiotu / który był Dawid rozbił / tedy ſpráwowáli cáłopáleniá y ofiáry ſpokojne przed Bogiem.
A gdy dokońcżył Dawid ofiárowáć cáłopáleniá / y ofiár ſpokojnych / błogoſłáwił ludowi w imię Páńſkie.
Y rozdźielił wƺyſtkim mężom Izráelſkim / od mężá áż do niewiáſty / káżdemu po bochenku chlebá / y po ƺtuce mięſá / y po łágiewce winá.
A poſtánowił przed ſkrzynią Páńſką z Lewitów ſługi / áby wſpomináli / y wyznáwáli / y chwálili Páná / Bogá Izráelſkiego.
Aſáf był przedniejƺy / á wtóry po nim Zácháryjáƺ / Jehyjel / y Semirámot / y Jechyjel y Mátytyjáƺ / y Elijáb / y Benájáƺ / y Obededom / y Jechyjel ; ći ná inſtrumentách / ná lutniách / ná hárfách / ále Aſáf ná cymbáłách / gráli.
Benájáƺ záś y Jácházyjel kápłáni z trąbámi uſtáwicżne byli przed ſkrzynią przymierzá Bożego.
Dopiero dniá onego nájpierwej poſtánowił Dawid / áby tym pſálmem chwálony był Pán przez Aſáfá y bráći jego :
Wyſłáwiájćie Páná / wzywájćie imieniá jego / á opowiádájćie między národámi ſpráwy jego.
Śpiewájćie mu / grájćie mu / rozmáwiájćie o wƺyſtkich cudách jego.
Chlubćie śię w imieniu świętym jego / á niech śię rozráduje ſerce ƺukájących Páná.
Szukájćie Páná / y mocy jego ; ƺukájćie oblicżá jego záwżdy.
Wſpominájćie dźiwne ſpráwy jego / które cżynił / y cudá jego / y ſądy uſt jego.
O náśienie Izráelſkie / ſłudzy jego! O ſynowie Jákóbowi / wybráni jego!
On jeſt Pán / Bóg náƺ ; po wƺyſtkiej źiemi ſądy jego.
Pámiętájćie áż ná wieki ná przymierze jego / ná ſłowo / które przykázáł do tyśiącżnego pokoleniá ;
Które poſtánowił z Abráhámem / y ná przyśięgę jego z Izáákiem ;
Y poſtánowił to Jákóbowi zá práwo / á Izráelowi zá przymierze wiecżne /
Mówiąc : Tobie dám źiemię Chánánejſką zá ƺnur dźiedźictwá wáƺego.
Choć wáſ byłá máłá licżbá / á przez krótki cżáſ byliśćie przychodniámi w niej ;
Y przechodźili od národu do národu / y od króleſtwá do innego ludu :
Nie dopuśćił nikomu / áby ich ućiſkáć miáł / y káráł dlá nich królów /
Mówiąc : Nie tykájćie pomázáńców moich / á prorokom moim nie cżyńćie nic złego.
Śpiewájćie Pánu wƺyſtká źiemio ; opowiádájćie ode dniá do dniá zbáwienie jego.
Opowiádájćie między národámi chwáłę jego / y między wƺyſtkimi ludźmi dźiwne ſpráwy jego ;
Bo wielki jeſt Pán / y chwálebny bárdzo / y ſtráƺniejƺy nád wƺyſtkich bogów ;
Gdyż wƺyſcy bogowie pogáńſcy ſą báłwánámi ; ále Pán niebioſá ucżynił.
Słáwá y zácność przed nim / moc y weſele ná miejſcu jego.
Przynieśćie Pánu pokoleniá národów / przynieśćie Pánu chwáłę y moc.
Przynieśćie Pánu chwáłę imieniá jego / przynieśćie dáry / á przychodźćie przed oblicżność jego ; kłániájćie śię Pánu w ozdobie świętobliwośći.
Bójćie śię oblicżá jego wƺyſtká źiemio / á będźie utwierdzony okrąg źiemi / áby śię nie poruƺył.
Niech śię rozrádują niebioſá / á niech śię rozweſeli źiemiá / á niech mówią w národách : Pán króluje!
Niech záƺumi morze / y ze wƺyſtkiem / co w nim jeſt ; niech śię rozráduje pole / y wƺyſtko / co ná nim jeſt.
Tedy śię rozweſelą drzewá leśne przed Pánem ; ábowiem przyƺedł ſądźić źiemię.
Wyſłáwiájćie Páná ; ábowiem dobry / bo ná wieki trwá miłośierdźie jego.
A mówćie : Záchowáj náſ / Boże zbáwieniá náƺego! y zgromádź náſ / á wyrwij náſ od pogán / ábyśmy wielbili imię święte twoje / y chlubili śię w chwále twojej.
Błogoſłáwiony Pán / Bóg Izráelſki / od wieków áż ná wieki. Y rzekł wƺyſtek lud Amen / y chwálił Páná.
Y zoſtáwił tám Dawid przed ſkrzynią przymierzá Páńſkiego Aſáfá y bráći jego / áby ſłużyli przed ſkrzynią uſtáwicżnie według potrzeby dniá káżdego.
Lecż Obededomá y bráći ich ƺeśdźieśiąt y ośm / Obededomá mówię / ſyná Jedytunowego / y Hoſę / ucżynił odźwiernymi.
A Sádoká kápłáná / y bráći jego kápłánów poſtáwił przed przybytkiem Páńſkim ná wyżynie / którá byłá w Gábáon /
Aby ofiárowáli cáłopáleniá Pánu ná ołtárzu cáłopáleniá uſtáwicżnie ráno y w wiecżór / á to według wƺyſtkiego / co nápiſáno w zákonie Páńſkim / który przykázáł Izráelowi.
A z nimi Hemáná y Jedytuná / y innych ná to obránych / którzy byli z imieniá miánowáni / áby chwálili Páná / przeto iż ná wieki trwá miłośierdźie jego.
A między nimi Hemán y Jedytun / trąbili y gráli ná trąbách / ná cymbáłách / y ná innych inſtrumentách muzycżnych Bogu ; ále ſynów Jedytunowych poſtáwił u wrót.
A ták rozƺedł śię wƺyſtek lud / káżdy do domu ſwego. Dawid śię też wróćił / áby błogoſłáwił domowi ſwemu.
