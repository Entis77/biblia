Ale ƺátán powſtał przećiw Izráelowi á pobudźił Dawidá / áby policżył Izráelá.
Przetoż rzekł Dawid do Joábá y do przełożonych nád ludem : Idźćie / oblicżćie Izráelá od Beerſebá áż do Dán / á odnieśćie do mnie / żebym wiedźiał pocżet ich.
Ale rzekł Joáb : Niech przymnoży Pán ludu ſwego / jáko teraz jeſt / tyle ſto kroć ; izali królu / pánie mój! nie ſą wƺyſcy oni ſługámi páná mego? Przecżże śię tego dowiáduje pán mój? Przecżżeby to miáło być ná upádek Izráelowi?
Wƺákże ſłowo królewſkie przemogło Joábá ; przetoż wyƺedł Joáb / á obƺedƺy wƺyſtkiego Izráelá / wróćił śię potem do Jeruzálemu.
Y oddał Joáb pocżet poráchowánego ludu Dawidowi. A było wƺyſtkiego Izráelá tyśiąc tyśięcy y ſto tyśięcy / mężów godnych ku bojowi ; á z Judy było cżtery kroć ſto tyśięcy / y śiedmdźieśięt tyśięcy mężów wálecżnych.
Lecż Lewitów y Benjámitów nie policżył miedzy nich / gdyż przykre było rozkázánie królewſkie Joábowi.
Owƺem nie podobáłá śię Bogu tá rzecż ; przetoż pokárał Izráelá.
Y rzekł Dawid do Bogá : Zgrzeƺyłem bárzo / żem to ucżynił ; ále teraz oddál / proƺę / niepráwość ſługi twego ; bom bárzo głupio ucżynił.
Zátem rzekł Pán do Gádá / proroká Dawidowego / mówiąc :
Idź / powiedz Dawidowi / á rzecż : Ták mówi Pán : Trzyć rzecży podáję ; obierz ſobie jednę z nich / ábym ći cżynił.
Tedy przyƺedł Gád do Dawidá / y rzekł mu : Ták mówi Pán : Obierz ſobie :
Albo przez trzy látá głód / álbo żebyś przez trzy mieśiące ginął od nieprzyjáćiół twych / á miecż nieprzyjáćiół twojich żeby ćię śćigał / álbo żeby przez trzy dni miecż Páńſki y mor był w źiemi / á Anioł Páńſki żeby niƺcżył wƺyſtkie gránice Izráelſkie. Przetoż teraz uwáż / co mam odpowiedźieć temu / który mię poſłał.
Y rzekł Dawid do Gádá : Bárdzom śćiśniony ; niech wpádnę / proƺę / w ręce Páńſkie / gdyż bárzo wielkie ſą zlitowánia jego / á w ręce ludzkie niecháj nie wpádám.
Tedy przepuśćił Pán powietrze morowe ná Izráelá. Y poległo z Izráelá śiedmdźieśiąt tyśięcy mężów.
Poſłał też Bóg Aniołá do Jeruzálemu / áby ich tráćił. A gdy ich tráćił / ujrzał Pán / y użálił śię nád tym złem / y rzekł Aniołowi trácącemu : Doſyć już / záwśćiągnij rękę twą. A Anioł Páńſki ſtał podle bojewiſká Ornáná Jebuzejcżyká.
Wtem podnióſƺy Dawid ocży ſwe ujrzał Aniołá Páńſkiego / który ſtał miedzy źiemią y miedzy niebem / á w ręce jego miecż jego dobyty / wyćiągniony przećiw Jeruzálemowi. Y upádł Dawid y ſtárśi / oblekƺy śię w wory / ná twárze ſwoje.
Zátem rzekł Dawid do Bogá : Izálim nie ja rozkázał licżyć ludu? Jámći jeſt ſám / którym zgrzeƺył / y bárzo źle ucżynił ; ále te owce cóż ucżyniły? Pánie / Boże mój! niech śię obróći / proƺę / ręká twojá ná mię y ná dom ojcá mego ; ále przećiwko ludowi twemu niech śię nie ſroży tá plágá.
Zátem Anioł Páńſki rzekł do Gádá / áby mówił Dawidowi / żeby ƺedł y zbudował ołtárz Pánu ná bojewiſku Ornáná Jebuzejcżyká.
A ták ƺedł Dawid według ſłowá Gádowego / które mówił imieniem Páńſkim.
Tedy obejrzáwƺy śię Ornán ujrzał onego Aniołá ; á cżterej ſynowie jego / którzy byli z nim / ſkryli śię ; á Ornán młóćił pƺenicę.
Wtem przyƺedł Dawid do Ornáná ; á ſpojrzáwƺy Ornán obácżył Dawidá / y wyƺedƺy z bojewiſká / pokłonił śię Dawidowi twárzą do źiemi.
Y rzekł Dawid do Ornáná : Dáj mi plác tego bojewiſká / ábym zbudował ná nim ołtárz Pánu ; zá ſłuƺne pieniądze ſpuść mi je / á będźie odwróconá tá plágá od ludu.
Y rzekł Ornán do Dawidá : Weźmij je ſobie / á niech ucżyni król / pán mój / co mu śię dobrego widźi ; otoć przydáję y woły ná cáłopalenia / y wóz ná drwá / y pƺenicę ná ofiárę śniedną : toć wƺyſtko dáję.
Y rzekł król Dawid do Ornáná : Nie ták / ále rácżey kupię zá ſłuƺne pieniądze ; bo nie wezmę co twego jeſt / áni będę ofiárował Pánu cáłopalenia dárowánego.
A ták Dawid dał Ornánowi zá on plác ƺeść ſyklów złotá dobrej wági.
Y zbudował tám Dawid ołtárz Pánu / á ofiárował cáłopalenia y ofiáry ſpokojne / y wzywał Páná / który go wyſłuchał / ſpuśćiwƺy ogień z niebá ná ołtárz cáłopalenia.
Y rzekł Pán do Aniołá / áby obróćił miecż ſwój w pochwy ſwoje.
Onego cżáſu widząc Dawid / iż go wyſłuchał Pán ná bojewiſku Ornáná Jebuzejcżyká / ofiárował tám ofiáry.
Abowiem przybytek Páńſki / który ucżynił Mojzeƺ ná puƺcży / y ołtárz cáłopalenia / náoncżás był ná wyżynie w Gábáonie.
A nie mógł Dawid iść do niego / áby śię rádźił Bogá ; bo przeſtráƺony był miecżem Aniołá Páńſkiego.
