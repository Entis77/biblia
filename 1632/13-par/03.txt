Cić też ſą ſynowie Dawidowi / którzy mu śię urodźili w Hebronie : Pierworodny Ammon z Achynoámy Jezreelitki ; wtóry Dánijel z Abigáili Kármelitki ;
Trzeći Abſálom / ſyn Mááchy / córki Tolmájá / królá Gieſſur ; cżwárty Adonijáƺ / ſyn Hággity ;
Piąty Sefátyjáƺ z Abitáili ; ƺóſty Jetráám z Egli / żony jego.
Tych ƺeść urodźiło mu śię w Hebronie / kędy królowáł przez śiedm lát / y przez ƺeść mieśięcy ; á trzydźieśći y trzy látá królowáł w Jeruzálemie.
A ći urodźili mu śię w Jeruzálemie : Sámná / y Sobáb / y Nátán / y Sálomon / cżtery ſynowie z Betſui / córki Ammielowej ;
Y Ibchár / y Eliſámá / y Elifet ;
Y Noge / y Nefeg / y Jáfijá ;
Y Elizámá / y Elijádá / y Elifelet / dźiewięć ſynów.
A ćić wƺyſcy ſą ſynowie Dawidowi / oprócż ſynów z záłożnic ; á Támár byłá śioſtrá ich.
Syn Sálomonowy Roboám ; Abiám ſyn jego / Azá ſyn jego / Jozáfát ſyn jego.
Jorám ſyn jego / Ochozyjáƺ ſyn jego / Joáz ſyn jego ;
Amázyjáƺ ſyn jego / Azáryjáƺ ſyn jego / Joátám ſyn jego ;
Acház ſyn jego / Ezechyjáƺ ſyn jego / Mánáſeſ ſyn jego ;
Amon ſyn jego / Jozyjáƺ ſyn jego ;
A ſynowie Jozyjáƺowi : Pierworodny Johánán / wtóry Joákim / trzeći Sedekijáƺ / cżwárty Sellum.
A ſynowie Joákimowi : Jechonijáƺ ſyn jego / Sedekijáƺ ſyn jego.
A ſynowie Jechonijáƺá więźniá : Sálátyjel ſyn jego.
A Sálátyjelowi : Málchirám / y Fádájáƺ / y Seneſer / Jekiemijá / Hoſámá / y Nádábijá.
A ſynowie Fádájáƺowi : Zorobábel / y Semej ; á ſyn Zorobábelowy Meſollám / y Hánánijáƺ / y Selomit / śioſtrá ich.
A Meſollámowi : Háſubá / y Ohol / y Báráchyjáƺ / y Házádyjáƺ Joſábcheſed / pięć ſynów.
A ſyn Hánánijáƺowy : Fáltyjáƺ / y Jeſájáƺ ; ſynowie Ráfájáƺowi / ſynowie Arnánáƺowi / ſynowie Obádyjáƺowi / ſynowie Sechenijáƺowi.
A ſynowie Sechenijáƺowi : Semejáƺ ; á ſynowie Semejáƺowi : Cháttuſ / y Igál / y Báryjá / y Nááryjáƺ / y Sáfát ; ƺeść ſynów.
A ſynowie Nááryjáƺowi : Elijenáj / y Ezechyjáƺ / y Eſrykám / trzej ſynowie.
A ſynowie Elijenájego : Hodáwijáƺ y Elijáſub / y Felejáƺ / y Akkub / y Jochánán / y Dálájáƺ / y Anáni / śiedm ſynów.
