Noe / Sem / Chám / y Jáfet.
Synowie Jáfetowi : Gomer / y Mágog / y Mádáj / y Jáwán / y Tubál / y Meſech / y Tyrás /
A ſynowie Gomerowi : Aſchenáz / y Ryfát / y Togormá.
Synowie też Jáwánowi : Eliſá / y Tárſys / Cytym / y Dodánim.
Synowie Chámowi : Chus y Miſráim / Put y Chánáán.
A ſynowie Chuſowi : Sebá / y Hewilá / y Sábtá / y Regmá / y Sábtáchá ; á ſynowie Regmy : Sebá y Dedán.
Spłodźił też Chus Nemrodá ; ten pocżął być możnym ná źiemi.
Miſráim też ſpłodźił Ludymá / y Hánánimá / y Láubimá / y Neftuhymá /
Y Pátruſymá / y Cháſłuchymá / ( z których poƺli Filiſtynowie ) y Káftorymá.
Potem Chánáán ſpłodźił Sydoná / pierworodnego ſwego / y Hetejcżyká.
Y Jebuzejcżyká / y Amorejcżyká / y Giergiezejcżyká /
Y Hewejcżyká / y Archájcżyká / y Symejcżyká /
Y Arádejcżyká / y Sámárejcżyká / y Chámátejcżyká.
Synowie Semowi : Elám / y Aſſur / y Arfáchſád / y Lud / y Arám / y Chus / y Hul / y Gieter / y Meſech.
A Arfáchſád ſpłodźił Selechá / á Selech ſpłodźił Heberá.
A Heberowi urodźili śię dwáj ſynowie / z których jednemu imię było Peleg / przeto / że zá jego cżáſów rozdźieloná jeſt źiemiá ; á imię brátá jego Jektán.
A Jektán ſpłodźił Elmodádá / y Sálefá / y Háſſármotá / y Járechá /
Y Adorámá / y Uzálá / y Deklá /
Y Hebálá / y Abimáelá / y Sebájá /
Y Ofirá / y Hewilá / y Jobábá. Ci wƺyſcy byli ſynowie Jektánowi.

Abrám ; ten jeſt Abráhám.
Synowie Abráhámowi : Izáák y Iſmáel.
A teć ſą rodzáje ich : Pierworodny Iſmáelowy Nebájot / y Kiedar / y Abdeel / y Mábſám.
Máſmá / y Dumá / Máſſá / Hádád / y Temá /
Jetur / Náfis y Kiedmá. Cić ſą ſynowie Iſmáelowi.
A ſynowie Cetury / záłożnicy Abráhámowej / których porodźiłá : Zámrám y Jokſán / y Mádán / y Midyján / y Jeſbok / y Suách. A ſynowie Jokſánowi ; Sábá y Dedán.
Synowie też Mádyjánowi : Hefá / y Hefer / y Henoch / y Abidá / y Eldáá. Cić wƺyſcy ſą ſynowie Cetury.
Y ſpłodźił Abráhám Izááká. A ſynowie Izáákowi byli : Ezáw y Izráel.
A ſynowie Ezáwowi : Elifás / Rehuel / y Jehus / y Jelom / y Kore.
Synowie Elifáſowi : Temán y Omár / Sefo y Gáátán / Kenáz y ſyn Támny / to jeſt / Amálek.
Synowie Rehuelowi : Náhát / Zárá / Sámmá / y Mezá.
A ſynowie Seirowi : Lotán / y Sobál / y Sebeon / y Háná / y Dyſſon / y Eſer / y Dyſán.
A ſynowie Lotánowi : Chory / y Hemán ; á śioſtrá Lotánowá byłá Támná.
Synowie Sobálowi : Hálmán / y Mánáát / y Hewál / Sefo / y Onam ; á ſynowie Sebeonowi : Ajá y Aná.
Synowie Aná : Dyſon ; á ſynowie Dyſoná : Hámdán / y Eſebán / y Jetrán / y Chárán.
Synowie Eſerowi : Báláán / y Záwán / Akán. Synowie Dyſánowi : Hus y Arán.
Cić ſą królowie / którzy królowáli w źiemi Edomſkiej / przed tym niż królował król nád ſynámi Izráelſkimi : Belá / ſyn Beorowy / á imię miáſtá jego Dynhábá.
A gdy umárł Belá / królował miáſto niego Jobáb / ſyn Zeráchowy z Boſry.
A gdy umárł Jobáb / królował miáſto niego Chuſám z źiemi Temáńſkiej.
A gdy umárł Chuſám / królował miáſto niego Hádád / ſyn Bádádowy / który poráźił Mádyjáńcżyków ná polu Moábſkiem ; á imię miáſtá jego Háwid.
A gdy umárł Hádád / królował miáſto niego Sámlá z Máſreki.
A gdy umárł Sámlá / królował miáſto niego Saul z Rechobot nád rzeką.
A gdy umárł Saul / królował miáſto niego Bálánán / ſyn Achoborowy.
A gdy umárł Bálánán / królował miáſto niego Hádar ; á imię miáſtá jego Pehu / imię też żony jego Mehetábel / córká Mátredy / córki Mezáábowej.
A gdy umárł Hádar / byli kśiążętámi w Edom : kśiążę Támná / kśiążę Hálwá / kśiążę Jetet /
Kśiążę Oolibámá / kśiążę Elá / kśiążę Pinon /
Kśiążę Kienáz / kśiążę Temán / kśiążę Mábſár /
Kśiążę Mágdyjel / kśiążę Hyrám. Toć byli kśiążętá Edomſcy.
