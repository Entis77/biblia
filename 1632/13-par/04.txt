Synowie Judowi : Fáres / Heſron / y Chármi / y Hur / y Sobál.
A Rejáƺ / ſyn Sobálowy / ſpłodźił Jáhátá / á Jáhát ſpłodźił Achumá / y Láádá. Teć ſą rodzáje Zárátego.
Ci też ſą z ojcá Etámá : Jezreel / y Jeſemá / y Idbás ; á imię śioſtry ich Selelfuni.
A Fánuel oćiec Giedory / y Ezer oćiec Hoſy. Cić ſą ſynowie Hurá / pierworodnego Efráty / ojcá Betlehemcżyków.
A Aſſur / oćiec Tekui / miał dwie żony : Chelę y Náárę.
Y urodźiłá mu Náárá Achuſámá / y Heferá / y Temáná / y Acháſtárá. Cić ſą ſynowie Nááry.
Synowie záśię Cheli : Seret / Jeſochár / y Etnán.
A Kos ſpłodźił Anubá / y Háſobobá / y dom Achárchelá / ſyná Hárumy.
A Jábes był zácniejƺy nád bráći ſwych / któremu mátká jego dáłá imię Jábes / mówiąc : Bom go w boleśći urodźiłá.
Y wzywał Jábes Bogá Izráelſkiego / mówiąc : Jeſli błogoſłáwiąc błogoſłáwić mi będźieƺ / á rozƺerzyƺ gránice moje / á będźie ręká twojá ze mną / á wybáwiƺ mię od złego / ábym utrápiony nie był. Y wypełnił to Bóg / o co go prośił.
A Chelub / brát Suchá / ſpłodźił Mechyrá ; ten jeſt ojcem Eſtoná.
A Eſton ſpłodźił Betráfá / y Páſeáchá / y Techynná / ojcá miáſtá Náhás. Cić ſą mężowie Rechy.
A ſynowie Kienezowi : Otonijel / y Serájá ; á ſynowie Otonijelowi : Hátát y Meánátáj.
A Meánátáj ſpłodźił Ofrá / á Serájáƺ ſpłodźił Joábá / ojcá w dolinie mieƺkájących rzemieślników ; bo rzemieślnicy byli.
Synowie záśię Kálebá / ſyná Jefuni : Hyru / Elá / y Náháin ; á ſyn Elá jeſt Kienáz.
A ſynowie Jeháleleelá : Zyf y Zyfá / Tyryjá / y Azárel.
A ſynowie Ezry : Jeter / y Mered / y Efer / y Jálon ; á żoná Merodowá urodźiłá Miryjámá / y Sámájá / y Iſbáchá / ojcá Eſtemoá.
Żoná też jego Judájá urodźiłá Jeredá / ojcá Giedor / y Heberá / ojcá Socho / y Jekutyjelá / ojcá Zánoách. Cić ſą ſynowie Betii / córki Fáráonowej / którą był pojął Mered.
A ſynowie żony Hodyjáƺowej / śioſtry Náhámá / ojcá Ceili : Gármi y Eſtemoá Máhátcżyk.
A ſynowie Symonowi : Amnon / y Rynná / Benchánán y Tylon ; á ſynowie Iſy : Zochet y Benſochet.
Synowie Seli / ſyná Judowego : Her / oćiec Lechá y Láádá / oćiec Máráá ; y rodzáje domów tych / którzy robili około biśioru w domu Aſbeát /
Y Jokim y mężowie Chozeby / y Joás y Sáráf / którzy pánowáli w Moáb / y Jáſubi Lechem ; ále te rzecży ſą dáwne.
Cić ſą gárncárze / którzy mieƺkáli w ſádách y miedzy płotámi / którzy tám przy królu dla robót jego mieƺkáli.
Synowie Symeonowi : Námuel / y Jámin / y Járyb / Zerá / Saul.
Sállum ſyn jego / Mábſám ſyn jego / Miſmá ſyn jego.
A ſynowie Miſmy : Hámuel ſyn jego / Záchur ſyn jego / Semej ſyn jego.
Ten Semej miał ſynów ƺeſnáśćie / y córek ƺeść ; ále bráćia jego nie mieli wiele ſynów / ták / że wƺyſtkiej rodźiny ich nie było ták wiele / jáko ſynów Judowych.
Y mieƺkáli w Beerſebá y w Moládá / y w Háſerſuál /
Y w Belá / y w Aſem / y w Etolád /
Y w Betul / y w Hormá / y w Sycelegu /
Y w Bet Márchábot / y w Háſerſuſá / y w Betbirze / y w Sááráim. Te miáſtá ich były / póki królował Dawid.
A wśi ich były : Etám / y Háin / Remnon / y Tochen y Aſán przy tych pięćiu miáſtách ;
Y wƺyſtkie wśi ich / które były około tych miaſt áż do Báál ; teć były mieƺkániem ich według národu ich.
A Moſobáb / y Jámlech / y Joſá / ſyn Amázyjáƺowy ;
Y Joel / y Jehu / ſyn Joſábijáƺá / ſyná Serájáƺowego / ſyná Aſyjelowego ;
Y Elichenáj / y Jákóbá / y Jeſochájá / y Aſájá / y Adyjel / y Jeſymijel / y Bánájás ;
Y Syſá / ſyn Syfy / ſyná Allonowego / ſyná Jodájáƺowego / ſyná Symry / y ſyná Semájáƺowego.
Ci miánowáni poſtánowieni ſą zá kśiążętá w národách ſwych / á domy ojców ich rozmnożyły śię bárzo.
Przetoż ruƺyli śię / áby ƺli do Gádor / áż ná wſchód ſłońcá oney doliny / áby ƺukáli páƺy bydłu ſwemu.
Y ználeſli obfite y wyborne páƺe / á źiemię przeſtronną y ſpokojną y rodzájną ; bo tám náród Chámów mieƺkał przedtem.
Przetoż przyƺedƺy ći z imieniá opiſáni zá dni Ezechyjáƺá / królá Judzkiego / poburzyli námioty ich y przybytki ich / które tám były ználeźione ; á wymordowáli ich / y nie máƺ ich áż do dniá tego / y ośiedli miejſce ich ; bo tám mieli páƺy dla bydeł ſwojich.
A z onych ſynów Symeonowych niektórzy ƺli ná górę Seir / pięć ſet mężów / á Fáltyjáƺ / y Necháryjáƺ / y Ráfájáƺ / y Huſel / ſynowie Iſy / byli wodzámi ich.
Y wymordowáli oſtátki / które były uƺły z Amálekitów / á mieƺkáli tám áż po dźiś dźień.
