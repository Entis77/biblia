O jákoż pośniedziáło złoto! zmieniło śię wyborne złoto / rozmiotáno kámienie świątnicy / po rogách wƺyſtkich ulic.
Szláchetni ſynowie Syońſcy / którzy byli przyrównáni do złotá ƺcżerego / jákoż ſą pocżytáni zá nácżynie gliniáne / zá dzieło rąk gárncárſkich!
Y ſmoki więc podáwájąc pierśi / kármią młode ſwoje ; ále córká ludu mojego dlá okrutniká podobná jeſt ſowie ná puƺcży.
Przylgnął język ſſącego do podniebieniá jego dlá uprágnieniá / dzieći proƺą o chleb ; ále niemáƺ / ktoby im go ułámáł.
Ci / którzy jádáli potráwy rozkoƺne / giną ná ulicách / á którzy byli wychowáni w ƺárłáćie / przytuláją śię do gnoju.
Więkƺe jeſt káránie córki ludu mojego / niżeli pomſtá Sodomy / którá jeſt podwróconá w jednem okámgnieniu / y nie zoſtáły ná niej ręce.
Cżyſtśi byli Názárejcżycy jego nád śnieg / jáśniejśi nád mleko / rumieńƺe ćiáłá ich / niżeli drogie kámienie / jákoby z ƺáfiru wyćioſáni byli ;
Ale teráz wejrzenie ich cżerniejƺe jeſt niż cżárność / nie mogą poznáni być ná ulicách ; przyſchłá ſkórá ich do kośći ich / wyſchłá jeſt jáko drzewo.
Lepiej śię tym ſtáło / którzy ſą pobići miecżem / niżeli tym / co umieráją głodem / gdyż oni zginęli przebitymi będąc / ále ći dlá niedoſtátku urodzájów polnych.
Ręce niewiáſt miłośiernych wárzyły ſynów ſwych / áby im byli zá pokárm w potárćiu córki ludu mego.
Wypełnił Pán popędliwość ſwoję / y wyláł gniew zápálcżywośći ſwojej / y zápálił ogień ná Syonie / który pożárł grunty jego.
Nigyby byli nie wierzyli królowie ziemſcy / y wƺyſcy obywátele świátá / żeby był miáł wnijść przećiwnik / y nieprzyjáćiel w brámy Jeruzálemſkie.
Ale śię to ſtáło dlá grzechów proroków jego / y niepráwośći kápłánów jego / którzy wylewáli w pośrodku jego krew ſpráwiedliwych.
Tułáli śię jáko ślepi po ulicách / máżąć śię krwią / której nie mogli / tylko śię dotykáć ƺátámi ſwemi.
Przetoż wołáli ná nich : Uſtępujćie / niecżyśći! uſtępujćie / uſtępujćie / nie dotykájćie śię! Práwieć uſtąpili / y tułáją śię ; dlátego mówią między národámi : Nie będą już więcej mieli włáſnego mieƺkániá.
Oblicże Páńſkie rozproƺyło ich / á nie wejrzy ná nich więcej ; nieprzyjáćiele kápłánów nie ƺánują / á nád ſtárcámi miłośierdziá nie używáją.
A wżdzy jeƺcże áż do uſtániá ocżów ſwych wyglądámy próżnego rátunku ſwego ; oglądájąc śię ná náród / który wybáwić nie może.
Szlákują ſtopy náƺe / ták / że áni po ulicách náƺych chodzić nie możemy ; przybliżył śię koniec náƺ / wypełniły śię dni náƺe / záiſte przyƺło dokońcżenie náƺe.
Prędśi ſą ći / którzy náſ gonią / niż orły niebieſkie ; po górách náſ gonią / ná puſtyniách cżyháją ná náſ.
Tchnienie nozdrzy náƺych / to jeſt pomázániec Páńſki / pojmány jeſt w jámách ich / o którymeśmy mówili : W ćieniu jego żyć będziemy między národámi.
Ráduj śię y weſel śię córko Edomſká! którá mieƺkáƺ w ziemi Huſ ; przyjdzie też do ćiebie kubek / upijeƺ śię / y obnáżyƺ śię.
Wzięło koniec káránie twoje / o córko Syońſká! nie zániechá ćię Bóg dłużej w pojmániu twojim ; ále twoję niepráwość náwiedzi / o córko Edomſká! á odkryje grzechy twoje.
