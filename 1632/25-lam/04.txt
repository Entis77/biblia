O jákoż pośniedźiáło złoto! zmieniło śię wyborne złoto / rozmiotáno kámienie świątnicy / po rogách wƺyſtkich ulic.
Szláchetni ſynowie Syońſcy / którzy byli przyrównáni do złotá ƺcżerego / jákoż ſą pocżytáni zá nacżynie gliniáne / zá dźieło rąk gárncárſkich!
Y ſmoki więc podáwájąc pierśi / kármią młode ſwoje ; ále córká ludu mojego dla okrutniká podobná jeſt ſowie ná puƺcży.
Przylgnął język ſſącego do podniebieniá jego dla uprágnieniá / dźieći proƺą o chleb ; ále niemáƺ / ktoby im go ułámał.
Ci / którzy jádáli potráwy rozkoƺne / giną ná ulicách / á którzy byli wychowáni w ƺárłáćie / przytuláją śię do gnoju.
Więkƺe jeſt káránie córki ludu mojego / niżeli pomſtá Sodomy / która jeſt podwróconá w jednem okámgnieniu / y nie zoſtáły ná niej ręce.
Cżyſtśi byli Názárejcżycy jego nád śnieg / jáśniejśi nád mleko / rumieńƺe ćiáłá ich / niżeli drogie kámienie / jákoby z ƺáfiru wyćioſáni byli ;
Ale teraz wejrzenie ich cżerniejƺe jeſt niż cżárność / nie mogą poznáni być ná ulicách ; przyſchłá ſkórá ich do kośći ich / wyſchłá jeſt jáko drzewo.
Lepiej śię tym ſtáło / którzy ſą pobići miecżem / niżeli tym / co umieráją głodem / gdyż oni zginęli przebitymi będąc / ále ći dla niedoſtátku urodzájów polnych.
Ręce niewiáſt miłośiernych wárzyły ſynów ſwych / áby im byli zá pokárm w potárćiu córki ludu mego.
Wypełnił Pán popędliwość ſwoję / y wylał gniew zápálcżywośći ſwojey / y zápálił ogień ná Syonie / który pożárł grunty jego.
Nigyby byli nie wierzyli królowie źiemſcy / y wƺyſcy obywátele świátá / żeby był miał wnijść przećiwnik / y nieprzyjáćiel w brámy Jeruzálemſkie.
Ale śię to ſtáło dla grzechów proroków jego / y niepráwośći kápłánów jego / którzy wylewáli w pośrzodku jego krew ſpráwiedliwych.
Tułáli śię jáko ślepi po ulicách / máżąć śię krwią / którey nie mogli / tylko śię dotykáć ƺátámi ſwemi.
Przetoż wołáli ná nich : Uſtępujćie / niecżyśći! uſtępujćie / uſtępujćie / nie dotykájćie śię! Práwieć uſtąpili / y tułáją śię ; dla tego mówią miedzy národámi : Nie będą już więcey mieli włáſnego mieƺkánia.
Oblicże Páńſkie rozproƺyło ich / á nie wejrzy ná nich więcey ; nieprzyjáćiele kápłánów nie ƺánują / á nád ſtárcámi miłośierdźiá nie używáją.
A wżdzy jeƺcże áż do uſtánia ocżów ſwych wyglądámy próżnego rátunku ſwego ; oglądájąc śię ná náród / który wybáwić nie może.
Szlákują ſtopy náƺe / ták / że áni po ulicách náƺych chodźić nie możemy ; przybliżył śię koniec náƺ / wypełniły śię dni náƺe / záiſte przyƺło dokońcżenie náƺe.
Prędśi ſą ći / którzy nas gonią / niż orły niebieſkie ; po górách nas gonią / ná puſtyniách cżyháją ná nas.
Tchnienie nozdrzy náƺych / to jeſt pomázániec Páńſki / pojmány jeſt w jámách ich / o którymeśmy mówili : W ćieniu jego żyć będźiemy miedzy národámi.
Ráduj śię y weſel śię córko Edomſká! która mieƺkáƺ w źiemi Hus ; przydźie też do ćiebie kubek / upijeƺ śię / y obnáżyƺ śię.
Wźięło koniec káránie twoje / o córko Syońſká! nie zániechá ćię Bóg dłużey w pojmániu twojim ; ále twoję niepráwość náwiedźi / o córko Edomſká! á odkryje grzechy twoje.
