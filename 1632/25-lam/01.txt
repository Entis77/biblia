W biblii śiedmdźieśięćiu tłumácżów / tá kśięgá świętá ták śię zácżyná. Y ſtáło śię / gdy Izráel pojmány był / á Jeruzálem ſpuſtoƺone / że Jeremijáƺ śiedźiał płácżąc / y nárzekał nárzekániem tákiem nád Jeruzálemem / á rzekł : Ach miáſto ták ludne jákoż śiedźi ſámotne! ſtáło śię jáko wdowá ; zácne miedzy národámi / przednie miedzy kráinámi ſtáło śię hołdowne.
Uſtáwicżnie w nocy płácże / á łzy jego ná jágodách jego ; niemáƺ / ktoby je ćieƺył ze wƺyſtkich miłośników jego ; wƺyſcy przyjáćiele jego przeniewierzyli mu śię / ſtáli mu śię nieprzyjáćiołmi.
Przenióſł śię Judá dla utrapienia y dla wielkiej niewoli ; wƺákże mieƺkájąc miedzy národámi nie znájduje odpocżnieniá ; wƺyſcy / którzy je gonią połápáli je w ćieśni.
Drogi Syońſkie płácżą / że nikt nie przychodźi ná święto urocżyſte. Wƺyſtkie bramy jego ſpuſtoƺáły / kápłáni jego wzdycháją / pánny jego ſmutne ſą / á ſámo pełne jeſt gorzkośći.
Nieprzyjáćiele jego ſą głową / przećiwnikom jego ƺcżęśliwie śię powodźi ; bo go Pán utrápił dla mnóſtwá przeſtępſtwá jego ; málucżcy jego poƺli w niewolę przed oblicżem trápiącego.
A ták odjętá jeſt od córki Syońſkiej wƺyſtká ozdobá jey ; kśiążętá jey ſtáły śię jáko jelenie nie znájdujący páƺy / y uchodzą bez śiły przed tym / który je goni.
Wſpominá córká Jeruzálemſká we dni utrapienia ſwego y kwileniá ſwego ná wƺyſtkie ućiechy ſwoje / które miewáłá ode dni dáwnych / gdy pádá lud jey od ręki nieprzyjáćielſkiej / nie májąc / ktoby jey rátował ; widząc ją nieprzyjáćiele náśmiewáli śię z ſábátów jey.
Ciężko zgrzeƺyłá córká Jeruzálemſká / przetoż jáko niecżyſtá odłącżoná jeſt. Wƺyſcy / którzy ją w ucżćiwośći mieli / lekce ją ſobie wáżą / przeto / że widzą nágość jey / á oná wzdychá / y tyłem śię obrácá.
Niecżyſtotá jey ná podołkách jey / á nie pomniáłá ná koniec ſwój ; przetoż znácżnie jeſt zniżoná / nie májąc / ktoby ją poćieƺył. Wejrzyj / Pánie! ná utrápienie moje ; boć śię wynióſł nieprzyjáćiel.
Rękę ſwoję wyćiągnął nieprzyjáćiel ná wƺyſtkie kochánia jey ; bo muśi pátrzyć ná pogán wchodzących do świątnicy jey / o cżemeś był przykázał / áby nie wchodźili do zgromádzenia twego.
Wƺyſtek lud jey wzdychájąc chlebá ƺuká / dáje koƺtowne rzecży ſwoje zá pokárm ku pośileniu duƺy. Wejrzyj / Pánie! á obácż ; bom zniewáżoná.
Nicże was to nie obchodźi? o wƺyſcy / którzy mimo idźiećie drogą! Obácżćie / á oglądájćie / jeſli jeſt boleść / jáko mojá boleść / która mi jeſt zádáná / jáko mię záſmućił Pán w dźień gniewu zápálcżywośći ſwojey.
Z wyſokośći poſłał ogień w kośći moje / który je opánował ; rozćiągnął śieć nogom moim / obróćił mię ná wſtecż / podał mię ná ſpuſtoƺenie / przez cáły dźień żáłośną.
Związáne jeſt járzmo niepráwośći mojich ręką jego / ſplotły śię / wſtąpiły ná ƺyję moję ; toć poráźiło śiłę moję ; podał mię Pán w ręce nieprzyjáćiół / nie mogę powſtáć.
Pán podeptał wƺyſtkich mocárzy mojich w pośród mnie / zwołał przećiwko mnie gromády / áby ſtárł młodźieńców mojich / Pán tłocżył jáko w práśie pánnę / córkę Judzką.
Przetoż ja płácżę ; z ocżów mojich / z ocżów mojich / mówię / wody ćieką / że jeſt dáleko odemnie poćieƺyćiel / któryby ochłodźił duƺę moję / ſynowie moi wytráceni ſą / przeto / iż wźiął górę nieprzyjáćiel.
Rozćiągá córká Sydońſká ręce ſwoje / nie ma / ktoby ją ćieƺył ; wzbudźił Pán ná Jákóbá zewƺąd w około nieprzyjáćiół jego ; córká Jeruzálemſká jeſt miedzy nimi / niby dla niecżyſtośći oddáloná.
Spráwiedliwy jeſt Pán ; bom uſtom jego odporná byłá. Słuchájćie / proƺę / wƺyſcy ludźie / á obácżćie boleść moję ; pánny moje / y młodźieńcy moi poƺli w niewolę.
Wołáłám ná przyjáćiół mojich / oni mię zdrádźili ; kápłáni moi y ſtárcy moi w mieśćie zginęli / ƺukájąc ſobie pokármu / áby pośilili duƺę ſwoję.
Wejrzyż / Pánie / bomći utrápioná / wnętrznośći moje ſtrwożone ſą / wywróćiło śię ſerce moje we mnie / przeto / żem byłá bárzo odporná ; ná dworze miecż ośierácá / á w domu nic niemáƺ jedno śmierć.
Słyƺąć / że ja wzdychám / ále niemáƺ / ktoby mię poćieƺył ; wƺyſcy nieprzyjáćiele moi ſłyƺąc o nieƺcżęśćiu mojim weſelą śię / żeś ty to ucżynił / á przywiodłeś dźień przed tym ogłoƺony ; áleć będą mnie podobni.
Niech przydźie wƺyſtká złość ich przed oblicżność twoję / á ucżyń im / jákoś mnie ucżynił dla wƺyſtkich przeſtępſtw mojich ; bo wielkie ſą wzdychánia moje / á ſerce moje żáłośne.
