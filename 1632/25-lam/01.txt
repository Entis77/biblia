W biblii śiedmdźieśięćiu tłumácżów / tá kśięgá świętá ták śię zácżyná. Y ſtáło śię / gdy Izráel pojmány był / á Jeruzálem ſpuſtoƺone / że Jeremijáƺ śiedźiáł płácżąc / y nárzekáł nárzekániem tákiem nád Jeruzálemem / á rzekł : Ach miáſto ták ludne jákoż śiedźi ſámotne! ſtáło śię jáko wdowá ; zácne między národámi / przednie między kráinámi ſtáło śię hołdowne.
Uſtáwicżnie w nocy płácże / á łzy jego ná jágodách jego ; niemáƺ / ktoby je ćieƺył ze wƺyſtkich miłośników jego ; wƺyſcy przyjáćiele jego przeniewierzyli mu śię / ſtáli mu śię nieprzyjáćiołmi.
Przenióſł śię Judá dla utrápieniá y dla wielkiej niewoli ; wƺákże mieƺkájąc między národámi nie znájduje odpocżnieniá ; wƺyſcy / którzy je gonią połápáli je w ćieśni.
Drogi Syońſkie płácżą / że nikt nie przychodźi ná święto urocżyſte. Wƺyſtkie brámy jego ſpuſtoƺáły / kápłáni jego wzdycháją / pánny jego ſmutne ſą / á ſámo pełne jeſt gorzkośći.
Nieprzyjáćiele jego ſą głową / przećiwnikom jego ƺcżęśliwie śię powodźi ; bo go Pán utrápił dla mnóſtwá przeſtępſtwá jego ; málucżcy jego poƺli w niewolę przed oblicżem trápiącego.
A ták odjętá jeſt od córki Syońſkiej wƺyſtká ozdobá jey ; kśiążętá jey ſtáły śię jáko jelenie nie znájdujący páƺy / y uchodzą bez śiły przed tym / który je goni.
Wſpominá córká Jeruzálemſká we dni utrápieniá ſwego y kwileniá ſwego ná wƺyſtkie ućiechy ſwoje / które miewáłá ode dni dáwnych / gdy pádá lud jey od ręki nieprzyjáćielſkiej / nie májąc / ktoby jey rátowáł ; widząc ją nieprzyjáćiele náśmiewáli śię z ſábátów jey.
Ciężko zgrzeƺyłá córká Jeruzálemſká / przetoż jáko niecżyſtá odłącżoná jeſt. Wƺyſcy / którzy ją w ucżćiwośći mieli / lekce ją ſobie wáżą / przeto / że widzą nágość jey / á oná wzdychá / y tyłem śię obrácá.
Niecżyſtotá jey ná podołkách jey / á nie pomniáłá ná koniec ſwój ; przetoż znácżnie jeſt zniżoná / nie májąc / ktoby ją poćieƺył. Wejrzyj / Pánie! ná utrápienie moje ; boć śię wynióſł nieprzyjáćiel.
Rękę ſwoję wyćiągnął nieprzyjáćiel ná wƺyſtkie kochániá jey ; bo muśi pátrzyć ná pogán wchodzących do świątnicy jey / o cżemeś był przykázáł / áby nie wchodźili do zgromádzeniá twego.
Wƺyſtek lud jey wzdychájąc chlebá ƺuká / dáje koƺtowne rzecży ſwoje zá pokárm ku pośileniu duƺy. Wejrzyj / Pánie! á obácż ; bom zniewáżoná.
Nicże was to nie obchodźi? o wƺyſcy / którzy mimo idźiećie drogą! Obácżćie / á oglądájćie / jeſli jeſt boleść / jáko mojá boleść / która mi jeſt zádáná / jáko mię záſmućił Pán w dźień gniewu zápálcżywośći ſwojey.
Z wyſokośći poſłáł ogień w kośći moje / który je opánowáł ; rozćiągnął śieć nogom moim / obróćił mię ná wſtecż / podáł mię ná ſpuſtoƺenie / przez cáły dźień żáłośną.
Związáne jeſt járzmo niepráwośći moich ręką jego / ſplotły śię / wſtąpiły ná ƺyję moję ; toć poráźiło śiłę moję ; podáł mię Pán w ręce nieprzyjáćiół / nie mogę powſtáć.
Pán podeptáł wƺyſtkich mocárzy moich w pośród mnie / zwołáł przećiwko mnie gromády / áby ſtárł młodźieńców moich / Pán tłocżył jáko w práśie pánnę / córkę Judzką.
Przetoż ja płácżę ; z ocżów moich / z ocżów moich / mówię / wody ćieką / że jeſt dáleko odemnie poćieƺyćiel / któryby ochłodźił duƺę moję / ſynowie moi wytráceni ſą / przeto / iż wźiął górę nieprzyjáćiel.
Rozćiągá córká Sydońſká ręce ſwoje / nie má / ktoby ją ćieƺył ; wzbudźił Pán ná Jákóbá zewƺąd w około nieprzyjáćiół jego ; córká Jeruzálemſká jeſt między nimi / niby dla niecżyſtośći oddáloná.
Spráwiedliwy jeſt Pán ; bom uſtom jego odporná byłá. Słuchájćie / proƺę / wƺyſcy ludźie / á obácżćie boleść moję ; pánny moje / y młodźieńcy moi poƺli w niewolę.
Wołáłám ná przyjáćiół moich / oni mię zdrádźili ; kápłáni moi y ſtárcy moi w mieśćie zginęli / ƺukájąc ſobie pokármu / áby pośilili duƺę ſwoję.
Wejrzyż / Pánie / bomći utrápioná / wnętrznośći moje ſtrwożone ſą / wywróćiło śię ſerce moje we mnie / przeto / żem byłá bárdzo odporná ; ná dworze miecż ośierácá / á w domu nic niemáƺ jedno śmierć.
Słyƺąć / że ja wzdychám / ále niemáƺ / ktoby mię poćieƺył ; wƺyſcy nieprzyjáćiele moi ſłyƺąc o nieƺcżęśćiu mojem weſelą śię / żeś ty to ucżynił / á przywiodłeś dźień przedtem ogłoƺony ; áleć będą mnie podobni.
Niech przyjdźie wƺyſtká złość ich przed oblicżność twoję / á ucżyń im / jákoś mnie ucżynił dla wƺyſtkich przeſtępſtw moich ; bo wielkie ſą wzdychániá moje / á ſerce moje żáłośne.
