Jákoż záćmił Pán w zápálcżywośći ſwojey córkę Syjońſką! zrzućił z niebá ná źiemię ſłáwę Izráelſką / á nie wſpomniáł ná podnóżek nóg ſwoich w dźień zápálcżywośći ſwojey.
Połknął Pán bez wƺelkiej litośći wƺyſtkie przybytki Jákóbowe / zburzył w popędliwośći ſwojey twierdze córki Judzkiej / uderzył je o źiemię / w háńbę oddáł króleſtwo y kśiążąt jey.
Odćiął w gniewie zápálcżywośći wƺyſtek róg Izráelſki / odwróćił názád práwicę ſwoję od nieprzyjáćielá / á rozpáliwƺy śię przećiwko Jákóbowi / jáko ogień páłájący pożerá do ƺcżętu w około.
Náćiągnął łuk ſwój / jáko nieprzyjáćiel / poſtáwił práwicę ſwoję jáko przećiwnik / y pozábijáł wƺyſtkich nájpozorniejƺych z ludu / á w námioćie córki Syońſkiej wyláł jáko ogień popędliwość ſwoję.
Pán śię ſtáł jáko nieprzyjáćiel / połknął Izráelá / połknął wƺyſtkie páłáce jego / popſuł twierdze jego / y rozmnożył w ludu Judzkim płácż y nárzekánie.
Oderwáł mocą płot ſwój jáko od ogrodu / zepſuł námiot ſwój ; Pán przywiódł w zápámiętánie w Syonie urocżyſte świętá y ſábáty / á odrzućił w gniewie popędliwośći ſwojey królá y kápłáná.
Pán odrzućił ołtárz ſwój / zbrzydźił ſobie świątnicę ſwoję / podáł do rąk nieprzyjáćielſkich mury y páłáce Syońſkie ; krzycżeli w domu Páńſkim jáko w dźień świętá urocżyſtego.
Umyślił Pán rozwálić mur córki Syońſkiej / rozćiągnął ƺnur / á nie odwróćił ręki ſwojey od ſkáżeniá ; rozkwilił báƺty / y mur / ták że weſpół omdlewáją.
Zápádły w źiemię brámy jey / połámáł y pokruƺył záwory jey ; król jey y kśiążętá jey ſą między pogánámi ; niemáƺ áni zakonu / tákże áni prorocy jey nie miewáją widzeniá od Páná.
Stárcy córki Syońſkiej uśiádłƺy ná źiemi umilknęli / poſypáli prochem głowę ſwoję / á przepáſują śię worámi ; pánny Jeruzálemſkie zwieƺáją ku źiemi głowy ſwe.
Ocży moje od łez uſtáły ; ſtrwożyły śię wnętrznośći moje / wyláłá śię ná źiemię wątrobá mojá dla ſtárćiá córki ludu mojego / gdy y niemowlątká / y dźiátki ſſące ná ulicách miáſtá omdlewáją ;
Mátkom ſwoim mówią : Gdźież jeſt zboże y wino? Gdy mdleją jáko zránieni po ulicách miáſtá / y wypuƺcżáją duƺę ſwoję ná łonie mátek ſwych.
Kogoć zá świádká ſtáwię? Kogo tobie przyrównám / o córko Jeruzálemſká? Kogoć przypodobám / ábym ćię ućieƺył / pánno / córko Syońſká? bo ſkruƺenie twoje wielkie jáko morze / któż ćię ulecży?
Prorocy twoi opowiádálić kłámſtwo y márność / á nie odkrywáli niepráwośći twojey / áby odwróćili pojmánie twoje ; áleć przepowiádáli ćiężáry / kłámſtwá y wygnánie.
Kláſkáją nád tobą rękomá wƺyſcy / którzy idą drogą / świſtáją / á chwieją głową ſwoją nád córką Jeruzálemſką / mówiąc : A onoż to miáſto / o którem powiádáno / że jeſt doſkonáłej pięknośći / y weſelem wƺyſtkiej źiemi?
Otworzyli ná ćię uſtá ſwe wƺyſcy nieprzyjáćiele twoi / świſtáją y zgrzytáją zębámi / mówiąc : Pożryjmy je ; tenći jeſt záiſte on dźień / któregośmy cżekáli / ználeſliśmy y oglądáliśmy go.
Ucżynił Pán / co był umyślił / wypełnił ſłowo ſwoje / które był przykázáł ode dni dáwnych ; zburzył bez litośći / á rozweſelił nád tobą nieprzyjáćielá / wywyżƺył róg przećiwników twoich.
Wołáło ſerce ich do Páná. O murze córki Syońſkiej! wylewáj łzy we dnie y w nocy jáko ſtrumień / nie dáwáj ſobie odpocżnieniá / á niech śię nie uſpokájá źrenicá oká twego.
Wſtáń / wołáj w nocy ná pocżątku ſtráży / wylewáj ſerce twoje przed oblicżem Páńſkim jáko wodę ; podnoś do niego ręce ſwoje zá duƺę dźiátek ſwych / które omdlewáją od głodu ná rogu wƺyſtkich ulic / á rzecż : Wejrzyj Pánie! á obácż / komuś ták kiedy ucżynił?
Izáli máją niewiáſty jeść płód ſwój / niemowlątká ućieƺne? Izáli zámordowány być má w świątnicy Páńſkiej kápłán y prorok?
Leży ná źiemi po ulicách dźiećię / y ſtárzec ; pánny moje / y młodźieńcy moi polegli od miecżá ; pobiłeś ich w dźień zápálcżywośći twojey / pomordowáłeś ich / á nie ſfolgowáłeś.
Zwołáłeś ſtráchów moich zewƺąd / jáko ná dźień urocżyſtego świętá / á nie był w dźień zápálcżywośći Páńſkiej / ktoby uƺedł á żyw zoſtáł ; którychem ná ręku piáſtowáłá y wychowywáłá / tych nieprzyjáćiel mój wyniƺcżył.
