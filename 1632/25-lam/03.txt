Jam jeſt ten mąż / którym widźiał utrápienie od rózgi rozgniewánia Bożego.
Záprowádźił mię / y záwiódł do ćiemnośći / á nie do świátłośći ;
Tylko śię ná mię obórzył / á obróćił rękę ſwoję przez cáły dźień.
Do ſtárośći przywiódł ćiáło moje y ſkórę moję / á połámał kośći moje.
Obudował mię á ogárnął żółćią y prácą ;
W ćiemnych miejſcách poſádźił mię / jáko tych / którzy dáwno pomárli.
Ogrodźił mię / ábym nie wyƺedł / obćiążył okowy moje ;
A choć wołám y krzycżę / zátulá uƺy ná modlitwę moję.
Ogrodźił drogę moję ćioſánym kámieniem / śćieƺki moje wywróćił.
Jeſt jáko niedźwiedźiem cżyhájącym ná mię / jáko lwem w ſkrytośćiách.
Drogi moje odwróćił / owƺem / rozƺárpał mię / y ucżynił mię ſpuſtoƺoną.
Náćiągnął łuk ſwój / á poſtáwił mię jáko cel ſtrzáłom ſwym.
Przeſtrzelił nerki moje ſtrzáłámi z ſájdáká ſwego.
Jeſtem pośmiewiſkiem ze wƺyſtkim ludem moim / pieśnią ich przez cáły dźień.
Náſycá mię gorzkośćiámi : upájá mię piołunem.
Nádto pokruƺył o kámycżki zęby moje / y pogrążył mię w popiele.
Tákeś oddálił / o Boże! od pokoju duƺę moję / áż ná wcżáſy zápominam.
Y mówię : Zginęłá śiłá mojá / y nádźiejá mojá / którąm miał w Pánu.
Wƺákże wſpominájąc ná utrápienie moje / y ná płácż mój / ná piołun / y ná żółć /
Wſpominájąc uſtáwicżnie / uniżá śię we mnie duƺá mojá.
Przywodząc to ſobie do ſercá ſwego / mam nádźieję.
Wielkie jeſt miłośierdźie Páńſkie / żeśmy do ƺcżętu nie zginęli ; nie uſtáwáją zájiſte litośći jego.
Ale śię ná káżdy poránek odnáwiáją ; wielka jeſt práwdá twojá.
Pán jeſt dźiáłem moim / mówi duƺá mojá / dla tego mam w nim nádźieję.
Dobry jeſt Pán tym / którzy nań ocżekują / duƺy tákowej / która go ƺuká.
Dobrze jeſt / ćierpliwie ocżekiwáć ná zbáwienie Páńſkie.
Dobrze jeſt mężowi nośić járzmo od dźiećińſtwá ſwego ;
Który będąc opuƺcżony / ćierpliwym jeſt w tym / co nań włożono ;
Kłádźie w prochu uſtá ſwe / áżby śię okázáłá nádźiejá ;
Nádſtáwiá bijącemu policżká / á náſycony bywa obelżeniem.
Bo Pán ná wieki nie odrzucá ;
Owƺem / jeſli záſmucá / záśię śię zmiłuje według mnóſtwá miłośierdźiá ſwego ;
Zájiſte nie z ſercá trápi y záſmucá ſynów ludzkich.
Aby kto ſtárł nogámi ſwemi wƺyſtkich więźniów w źiemi ;
Aby kto nieſpráwiedliwie ſądźił mężá przed oblicżem Nájwyżƺego ;
Aby kto wywróćił cżłowieká w ſpráwie jego / Pán śię w tym nie kocha.
Któż jeſt / coby rzekł : Stáło śię / á Pán nie przykázał?
Izali z uſt Nájwyżƺego nie pochodźi złe y dobre?
Przecżżeby tedy ſobie utyſkiwáć miał cżłowiek żyjący / á mąż nád káźnią zá grzechy ſwoje.
Dowiádujmy śię rácżey / á bádájmy śię dróg náƺych / y náwróćmy śię do Páná ;
Podnieśmy ſercá y ręce náƺe w niebo do Bogá.
Myśmy wyſtąpili y ſtáliśmy śię odpornymi ; przetoż ty nie odpuƺcżáƺ.
Okryłeś śię zápálcżywośćią / y goniƺ nas / mordujeƺ / á nie ƺánujeƺ.
Okryłeś śię obłokiem / áby ćię nie dochodźiłá modlitwá.
Zá śmieći y zá pomiotło położyłeś nas w pośrzodku tych narodów.
Otworzyli ná nas uſtá ſwoje wƺyſcy nieprzyjáćiele náśi.
Strách y dół przyƺedł ná nas / ſpuſtoƺenie y ſkruƺenie.
Strumienie wód płyną z ocżów mojich / dla ſkruƺeniá córki ludu mojego.
Ocży moje płyną bez przeſtánku / przeto / że niemáƺ żádney ulgi /
Ażby wejrzał y obácżył Pán z niebá.
Ocży moje trápią duƺę moję dla wƺyſtkich córek miáſtá mojego.
Łowili mię uſtáwicżnie jáko ptáká nieprzyjáćiele moi bez przycżyny.
Wrzućili do dołu żywot mój / á przywálili mię kámieniem.
Wezbráły wody nád głową moją / y rzekłem : Jużći po mnie!
Wzywám imienia twego / o Pánie! z dołu bárzo głębokiego.
Głos mój wyſłuchiwáłeś ; nie zátulájże uchá twego przed wzdychániem mojim / y przed wołániem mojim.
Przybliżájąc śię do mnie w dźień / któregom ćię wzywał / máwiáłeś : Nie bój śię.
Záſtáwiáłeś śię / Pánie! o ſpráwę duƺy mojey / á wybáwiáłeś żywot mój.
Widźiƺ / o Pánie! bezpráwie / które mi śię dźieje / oſądźże ſpráwę moję.
Widźiƺ wƺyſtkę pomſtę ich / y wƺyſtkie zámyſły ich przećiwko mnie.
Słyƺyƺ urągánie ich / o Pánie! y wƺyſtkie zámyſły ich przećiwko mnie.
Słyƺyƺ wárgi powſtáwájących przećiwko mnie / y przemyśliwánie ich przećiwko mnie przez cáły dźień.
Obácż śiádánie ich / y wſtáwánie ich ; jam záwżdy jeſt pieśnią ich.
Oddájże im nágrodę / Pánie! według ſpráwy rąk ich ;
Dájże im zátwárdźiáłe ſerce / y przeklęſtwo ſwe ná nich ;
Goń ich w zápálcżywośći / á zgłádź ich / áby nie byli pod niebem twojim / o Pánie!
