Przetoż ty ſynu mój / zmacniaj śię w łáſce / która jeſt w CHryſtuśie JEzuśie.
A coś ſłyƺał ode mnie przed wielą świadków / tegoż śię powierz wiernym ludźiom / którzyby ſpoſobni byli y inƺych náucżáć.
Przetoż ty ćierp złe / jáko dobry żołnierz JEzuſá CHryſtuſá.
Żaden który żołnierſką ſłuży / nie wikle śię ſpráwámi tego żywotá / áby śię temu / od którego zá żołnierzá przyjęty jeſt / podobał.
A choćby śię też kto potykał / nie bywa koronowány / jeſliby śię przyſtojnie nie potykał.
Oracż który prácuje / ma naprzód pożytki odbieráć.
Rozumiej co mówię : A PAN niechći da we wƺyſtkim wyrozumienie.
Pámiętaj iż JEzus CHryſtus wzbudzony jeſt zmartwych / <i>który jeſt</i> z naśienia Dawidowego / według Ewángeliey mojey.
W którey ćierpię złe / jákoby złocżyńcá / áż do związek : áleć ſłowo Boże nie jeſt związáne.
Przetoż wƺyſtko znoƺę dla wybránych : áby y oni zbáwienia doſtąpili / które jeſt w CHryſtuśie JEzuśie z chwałą wiecżną.
Wierna <i>jeſt tá</i> mowá : Abowiem jeſliſmy z nim umárli / z nim też żyć będźiemy.
Jeſli ćierpiemy / z nim też królowáć będźiemy : jeſli śię go zápieramy / y on śię nas záprzy.
Jeſliſmy niewiernymi / on wiernym zoſtawa / <i>y</i> záprzeć ſámego śiebie nie może.
Te rzecży przypominaj ; oświadcżájąc przed oblicżem PAńſkim / áby śię nie wdawáli w ſpory około ſłów / co ku nicżemu nie jeſt pożytecżne / tylko ku podwróceniu tych którzy ſłucháją.
Stáraj śię ábyś śię doświadcżonym ſtáwił Bogu / robotnikiem któryby śię nie záwſtydźił / y któryby dobrze rozbierał ſłowo prawdy.
A świeckim próznomównośćiam cżyń wſtręt : ábowiem poſtępują ku więkƺey nie pobożnośći.
A mowá ich ƺerzy śię jáko káncer / z których jeſt Hymeneuƺ y Filetus.
Którzy względem prawdy celu uchybili / gdy powiádáją / iż śię już ſtáło zmartwychwſtánie / y podwracáją wiárę niektórych.
A wƺákże mocny ſtoji Grunt Boży / májąc tę piecżęć : Zna PAN którzy ſą jego. Y / Niech odſtąpi od nieſpráwiedliwośći wƺelki który miánuje Imię CHryſtuſowe.
A w wielkim domu nie tylko ſą nácżynia złote y śrebrne / ále też drewniáne / y gliniáne : á niektóre ku ucżćiwośći / drugie záśię ku zelżywośći.
Jeſliby tedy kto ſámego śiebie ocżyśćił od tych <i>rzecży,</i> będźie nácżyniem ku ucżćiwośći / poświęconem y użytecżnym PAnu / do wƺelákey dobrey ſpráwy zgotowánym.
Chroń śię też pożądliwośći młodźieńſkich / á náśláduj ſpráwiedliwośći wiáry / miłośći / pokoju / z tymi którzy wzywáją CHryſtuſá z cżyſtego ſercá.
Chroń śię też gadek głupich y nieumiejętnych : wiedząc iż rodzą zwády.
Ale ſługá PAńſki nie ma bydź zwádliwy : lecż ma być ukłádny przećiwko wƺyſtkim / ſpoſobny ku náucżániu / złych ćierpliwie znaƺájący :
Któryby w ćichośći náucżał te którzy śię zprzećiwiáją : owaby im kiedy Bóg dał Pokutę ku uznániu prawdy.
Aby obacżywƺy śię / <i>wywikłáli śię</i> z śidłá dyabelſkiego / od którego pojimáni ſą ku <i>cżynieniu</i> woli jego.
