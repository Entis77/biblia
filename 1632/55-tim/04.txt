Já tedy oświádcżám śię przed Bogiem y Pánem JEzuſem CHryſtuſem / który má ſądźić żywych y umárłych w ſłáwnem przyjśćiu ſwojim y króleſtwie ſwojim :
Káż ſłowo Boże / nálegáj w cżáſ álbo nie wcżáſ / ſtrofuj / grom y nápomináj ze wƺelką ćierpliwośćią y náuką.
Abowiem przyjdźie cżáſ / gdy zdrowej náuki nie śćierpią / ále według ſwoich pożądliwośći zgromádzą ſobie ſámi náucżyćieli / májąc świerzbiące uƺy /
A odwrócą uƺy od práwdy / á ku báśniom je obrócą.
Ale ty bądź cżułym we wƺyſtkiem / ćierp złe / wykonywáj ucżynek káznodźiei / uſługiwániá twego zupełnie dowódź.
Abowiem já już bywám ofiárowány / á cżáſ rozwiązániá mego nádchodźi.
Dobrym bój bojowáł / biegem wykonáł / wiáręm záchowáł ;
Zátem odłożoná mi jeſt koroná ſpráwiedliwośći / którą mi oddá w on dźień Pán / ſędźiá ſpráwiedliwy / á nie tylko mnie / ále y wƺyſtkim / którzy umiłowáli ſłáwne przyjśćie jego.
Stáráj śię / ábyś do mnie przyƺedł rychło.
Abowiem Demáſ mię opuśćił / umiłowáwƺy ten świát / y poƺedł do Teſáloniki / Kreſcenſ do Gálácyi / Tytuſ do Dálmácyi ;
Sám tylko Łukáƺ ze mną jeſt. Márká wźiąwƺy / przywiedź ze ſobą ; bo mi jeſt bárdzo pożytecżny ku poſłudze.
A Tychykám poſłáł do Efezu.
Opońcżę / którąm zoſtáwił w Troádźie u Kárpuſá / gdy przyjdźieƺ / przynieś z ſobą / y kśięgi / zwłáƺcżá membrány.
Alekſánder kotlárz wiele mi złego wyrządźił ; niech mu Pán oddá według ucżynków jego.
Którego y ty śię ſtrzeż ; ábowiem śię bárdzo ſprzećiwił ſłowom náƺym.
W pierwƺej obronie mojej żáden przy mnie nie ſtáł / ále mię wƺyſcy opuśćili ; niech im to nie będźie przycżytáne.
Ale Pán przy mnie ſtáł y umocnił mię / áby przez mię zupełnie utwierdzone było kázánie / á iżby je ſłyƺeli wƺyſcy pogánie / y byłem wyrwány z páƺcżęki lwiej.
A wyrwie mię Pán z káżdego ucżynku złego / y záchowá do króleſtwá ſwego niebieſkiego ; któremu chwáłá ná wieki wieków. Amen.
Pozdrów Pryƺkę / y Akwilę / y dom Onezyforowy.
Eráſtuſ zoſtáł w Korynćie / á Trofimám zoſtáwił w Milećie chorego.
Stáráj śię / ábyś przyƺedł przed źimą. Pozdráwiá ćię Eubuluſ / y Pudenſ / y Linuſ / y Kláudyjá / y bráćiá wƺyſcy.
Pán JEzuſ CHryſtuſ niech będźie z duchem twoim. Łáſká Bożá niech będźie z wámi. Amen.
