A to wiedz / iż w oſtátecżne dni náſtáną cżáſy trudne.
Abowiem będą ludźie ſámi śiebie miłujący / łákomi / chlubni / pyƺni / bluźniercy / rodźicom niepoſłuƺni / niewdźięcżni / niepobożni /
Bez przyrodzonej miłośći / przymierzá nietrzymájący / potwárcy / niepowśćiągliwi / nieſkromni / dobrych niemiłujący /
Zdrájcy / ſkwápliwi / nádęći / rozkoƺy rácżej miłujący niż miłujący Bogá ;
Którzy máją kƺtáłt pobożnośći / ále śię ſkutku jej zápárli ; y tych śię chroń.
Abowiem z tych ſą ći / którzy śię wrywáją w domy / y pojmáne wiodą niewiáſtki grzechámi obćiążone / które uwodzą rozmáite pożądliwośći ;
Które śię záwƺe ucżą / á nigdy do znájomośći práwdy przyjść nie mogą.
A jáko Jánneſ y Jámbreſ ſprzećiwiáli śię Mojżeƺowi / ták y ći ſprzećiwiáją śię práwdźie / ludźie rozumu ſkáżonego / odrzuceni z ſtrony wiáry.
Ale nie poſtąpią dálej ; ábowiem głupſtwo ich jáwne będźie wƺyſtkim / jáko y onych było.
Aleś ty doƺedł náuki mojej / ſpoſobu żywotá mego / przedśięwźięćiá wiáry / nieſkwápliwośći / miłośći y ćierpliwośći /
Prześládowániá / ućierpieniá / które mię ſpotkáły w Antyjochii / w Ikonii y w Liſtrzech / jákiem prześládowániá podejmowáł ; á ze wƺyſtkich wyrwáł mię Pán.
Aleć y wƺyſcy / którzy chcą pobożnie żyć w CHryſtuśie JEzuśie / prześládowáni będą.
Lecż źli ludźie y zwodźićiele poſtąpią w gorƺe / jáko zwodzący ták y zwiedzeni.
Ale ty trwáj w tem / cżegoś śię náucżył / y cżegoć powierzono / wiedząc / od kogoś śię tego náucżył.
A iż z dźiećińſtwá umieƺ Piſmá święte / które ćię mogą ucżynić mądrym ku zbáwieniu przez wiárę / którá jeſt w CHryſtuśie JEzuśie.
Wƺyſtko Piſmo od Bogá jeſt nátchnione / y pożytecżne ku náuce / ku ſtrofowániu / ku nápráwie / ku ćwicżeniu / które jeſt w ſpráwiedliwośći ;
Aby cżłowiek Boży był doſkonáły / ku wƺelkiej ſpráwie dobrej doſtátecżnie wyćwicżony.
