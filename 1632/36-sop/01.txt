Słowo Páńſkie / które śię ſtáło do Sofonijáƺá / ſyná Chuſego / ſyná Godolijáƺowego / ſyná Amáryjáƺowego / ſyná Ezechyjáƺowego / zá dni Jozyjáƺá / ſyná Amonowego / królá Judzkiego.
Wƺyſtko zápewne znioſę z oblicżá tej ziemi / mówi Pán.
Znioſę ludzi y bydło / znioſę ptáſtwo niebieſkie y ryby morſkie / y zgorƺenie z niepobożnymi ; wykorzenię cżłowieká z oblicżá tej ziemi / mówi Pán.
Bo wyćiągnę rękę moję ná Judę / y ná wƺyſtkich obywáteli Jeruzálemſkich ; wykorzenię z miejſcá tego oſtátki Báálowe / y popy jego z kápłánámi ;
Y tych / którzy śię kłániáją ná dáchách wojſku niebieſkiemu / y tych / którzy kłániájąc śię przyśięgáją przez Páná / tákże y tych / którzy przyśięgáją przez Molochá ſwego ;
Y tych / którzy śię odwrácáją od náśládowániá Páná / y którzy nie ƺukáją Páná / áni śię pytáją o nim.
Umilknij przed oblicżem pánującego Páná / gdyż bliſki jeſt dzień Páńſki ; bo Pán zgotowáł ofiárę / y poświęćił wezwánych ſwoich.
A w dzień ofiáry Páńſkiej náwiedzę kśiążąt y ſynów królewſkich / y wƺyſtkich / którzy śię obłócżą w ƺáty cudzoziemſkie.
Náwiedzę też dniá onego káżdego / który próg przeſkákuje / którzy nápełniáją domy pánów ſwych łupieſtwem y bezpráwiem.
A dniá onego / mówi Pán / będzie głoſ wołániá od brámy rybnej / y nárzekánie od drugiej ſtrony ( miáſtá ) / y ſkruƺenie wielkie od págórków.
Nárzekájćie wy / którzy mieƺkáćie wewnątrz : bo wygłádzony będzie wƺyſtek lud kupiecki / wygłádzeni będą wƺyſcy / którzy ſrebro noƺą.
Y ſtánie śię w on cżáſ / że Jeruzálem ƺpiegowáć będę z pochodniámi / y náwiedzę mężów / którzy polgnęli w drożdżách ſwoich / mówiąc w ſercu ſwojim : Pán nie ucżyni dobrze / áni źle ucżyni.
Bo májętność ich przyjdzie ná rozchwycenie / y domy ich ná ſpuſtoƺenie ; pobudują domy / lecż w nich mieƺkáć nie będą ; y będą ſádzić winnice / ále z nich winá pić nie będą.
Bliſki jeſt wielki dzień Páńſki / bliſki jeſt y ſpieƺny bárdzo głoſ dniá Páńſkiego ; tám y mocárz gorzko wołáć będzie.
Dzień gniewu będzie ten dzień / dzień utrápieniá y ućiſku / dzień zámięƺániá y ſpuſtoƺeniá / dzień ćiemnośći y mroku / dzień obłoku y chmury ;
Dzień trąby y trąbieniá przećiwko miáſtom obronnym y przećiwko báƺtom wyſokim /
W który ludzi utrápieniem śćiſnę / że jáko ślepi chodzić będą / poniewáż przećiwko Pánu zgrzeƺyli ; y wyláná będzie krew ich jáko proch / á ćiáłá ich jáko gnój.
Ani ſrebro ich / áni złoto ich nie będzie ich mogło wyrwáć w dzień gniewu Páńſkiego ; bo ogniem zápálcżywośći jego będzie tá wƺyſtká ziemiá pożártá / przeto / że zápewne koniec prędki ucżyni wƺyſtkim obywátelom ziemi.
