Biádá temu miáſtu zmázánemu y ſplugáwionemu / gwáłty cżyniącemu!
Nie ſłuchá głoſu / áni przyjmuje kárnośći / w Pánu nie ufá / do Bogá ſwego nie przybliżá śię.
Kśiążętá jego w pośrodku jego ſą lwy rycżące / ſędziowie jego ſą wilki wiecżorne / które nie gryzą kośći áż do poránku.
Prorocy jego ſkwápliwi / mężowie przewrotni ; kápłáni jego ſplugáwili rzecży święte / zákon gwáłcą.
Pán ſpráwiedliwy jeſt w pośród niego / nie cżyni niepráwośći / káżdego dniá ſąd ſwój wydáje ná świátłość bez przeſtániá ; wƺákże złośnik wſtydzić śię nie umie.
Wykorzeniłem národy / ſpuſtoƺone ſą zámki ich / obróćiłem w puſtynię ulice ich / ták / że niemáƺ / ktoby przez nie chodził ; ſpuſtoƺone ſą miáſtá ich / ták / że niemáƺ áni cżłowieká áni obywátelá.
Rzekłem : Wżdyć śię mnie báć będzieƺ / przyjmieƺ kárność / áby nie był wytrácony przybytek twój / przez to / cżemem ćię náwiedzić chćiáł ; ále ráno wſtáwƺy pſują wƺyſtkie przedśięwzięćiá ſwoje.
Przetoż ocżekiwájćie ná mię / mówi Pán / do dniá / którego powſtánę do łupu ; bo ſąd mój jeſt / ábym zebráł národy / y zgromádził króleſtwá / ábym ná nie wyláł rozgniewánie moje y wƺyſtkę popędliwość gniewu mego ; ogniem záiſte gorliwośći mojej będzie pożártá tá wƺyſtká ziemiá.
Bo ná ten cżáſ przywrócę národom wárgi cżyſte / któremiby wzywáli wƺyſcy imieniá Páńſkiego / á ſłużyli mu jednomyślnie.
Ci / którzy ſą zá rzekámi ziemi Murzyńſkiej / pokłon mi oddáją z córką rozproƺonych moich / przynioſą mi dáry.
Dniá onego nie záwſtydziƺ śię zá żádne ucżynki twoje / któremiś wyſtąpiło przećiwko mnie ; bo ná ten cżáſ odejmę z pośrodku ćiebie tych / którzy śię weſelą z ſłáwy twojej / á nie będzieƺ śię więcej wywyżƺáło ná górze świętobliwośći mojej.
Y zoſtáwię w pośrodku ćiebie lud ubogi á nędzny / y ufáć będą w imieniu Páńſkim.
Oſtátki Izráelá nie będą cżyniły niepráwośći / áni będą mówiły kłámſtwá / áni śię znájdzie w uſtách ich język zdrádliwy ; ále śię páść będą y odpocżywáć / á nie będzie / ktoby je przeſtráƺył.
Záśpiewáj / córko Syońſká! wykrzykájćie / Izráelcżycy! weſel śię á ráduj śię ze wƺyſtkiego ſercá / córko Jeruzálemſká!
Że odjął Pán ſądy twoje / uprzątnął nieprzyjáćielá twego ; król Izráelſki jeſt Pánem w pośrodku ćiebie / nie oglądáƺ więcej złego.
Dniá onego rzeką do Jeruzálemu : Nie bój śię! á do Syonu : Niech nie mdleją ręce twoje!
Pán / Bóg twój / w pośrodku ćiebie mocny záchowá ćię / rozweſeli śię wielce nád tobą / przeſtánie ná miłośći ſwej przećiwko tobie / y rozweſeli śię nád tobą z śpiewániem / mówiąc :
Tęſkniących po Jeruzálemie záśię zgromádzę / bo z ćiebie ſą y ſmutni dlá brzemieniá zelżywośći włożonej ná ćię.
Oto Já koniec ucżynię wƺyſtkim / którzy ćię trápić będą ná on cżáſ / á záchowám chromą / y wygnáną zgromádzę ; owƺem ſpoſobię im chwáłę y imię po wƺyſtkiej ziemi / w której zelżywość ponośili.
Wtencżáſ przywiodę wáſ / wtencżáſ / mówię / zgromádzę wáſ ; ábowiem dám wám imię y chwáłę między wƺyſtkimi národámi ziemi / gdy záś przywiodę więźnie wáƺe przed ocżymá wáƺemi / mówi Pán.
