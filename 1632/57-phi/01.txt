Páweł / więźień CHryſtuſá JEzuſá y Tymoteuƺ brát / Filemonowi miłemu á pomocnikowi náƺemu /
Y Apfii miłej / y Archipowi / ſpołecżnemu náƺemu bojownikowi / y zborowi / który jeſt w domu twoim.
Łáſká wám y pokój niech będźie od Bogá / Ojcá náƺego / y Páná JEzuſá CHryſtuſá.
Dźiękuję Bogu mojemu záwƺe wzmiánkę cżyniąc o tobie w modlitwách moich /
Słyƺąc o miłośći twojey y o wierze / którą máƺ przećiwko Pánu JEzuſowi / y przećiwko wƺyſtkim świętym ;
Aby ſpołecżność wiáry twojey byłá ſkutecżná ku poznániu wƺyſtkiego dobrego / które w was jeſt przez CHryſtuſá JEzuſá.
Abowiem rádość wielką mámy y poćiechę z miłośći twojey / bráćie! iż wnętrznośći świętych ſą ochłodzone przez ćię.
Przetoż choć mam wielką bezpiecżność w CHryſtuśie / ábym ći rozkázáł / co przynáleży ;
Jednák dla miłośći rácżej proƺę / tákowym będąc / to jeſt Páweł ſtáry / á teraz y więźień JEzuſá CHryſtuſá.
Proƺę ćię tedy zá ſynem moim Onezymem / któregom urodźił w więźieniu mojem ;
Który tobie niekiedy był niepożytecżny / ále teraz tobie y mnie bárdzo pożytecżny ; któregom odeſłáł.
Przetoż go ty jáko wnętrznośći moje przyjmij.
Któregom ja chćiáł przy ſobie zátrzymáć / áby mi poſługiwáł zámiáſt ćiebie w więźieniu dla Ewángielii.
Ale bez woli twojey nie chćiáłem nic ucżynić / áby dobry twój ucżynek nie był jáko z przymuƺeniá / ále z dobrej woli.
Abowiem ſnáć dla tego odłącżył śię był ná chwilę od ćiebie / ábyś go záś miał wiecżnie /
Już nie jáko ſługę / ále więcey niż ſługę / to jeſt brátá miłego / zwłáƺcżá mnie / á jáko dáleko więcey tobie y według ćiáłá y w Pánu.
Przetoż máƺli mię zá towárzyƺá / przyjmij go jáko mię.
A jeſlić w cżem niepráw / álboć co winien / to mnie przycżytáj.
Jám Páweł nápiſáł ręką moją / ja nágrodzę / żeć nie rzekę / iżeś mi y ſámego śiebie winien.
Ták / bráćie! niech ćię w tem użyję w Pánu / ochłodź wnętrznośći moje w Pánu.
Pewien będąc poſłuƺeńſtwá twego / piſáłem ći / wiedząc / że y więcey / niż mówię / ucżyniƺ.
Záráz mi też y goſpodę zgotuj ; ábowiem ſpodźiewám śię / iż wám przez modlitwy wáƺe dárowány będę.
Pozdráwiáją ćię Epáfráſ / ſpółwięźień mój w CHryſtuśie JEzuśie /
Márek / Aryſtárchuſ / Demáſ / Łukáƺ / pomocnicy moi.
Łáſká Páná náƺego JEzuſá CHryſtuſá niech będźie z duchem wáƺym. Amen.
