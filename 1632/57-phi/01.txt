Páweł / więzień Chryſtuſá Jezuſá y Tymoteuƺ brát / Filemonowi miłemu á pomocnikowi náƺemu /
Y Apfii miłej / y Archipowi / ſpołecżnemu náƺemu bojownikowi / y zborowi / który jeſt w domu twoim.
Łáſká wám y pokój niech będzie od Bogá / Ojcá náƺego / y Páná Jezuſá Chryſtuſá.
Dziękuję Bogu mojemu záwƺe wzmiánkę cżyniąc o tobie w modlitwách moich /
Słyƺąc o miłośći twojej y o wierze / którą máƺ przećiwko Pánu Jezuſowi / y przećiwko wƺyſtkim świętym ;
Aby ſpołecżność wiáry twojej byłá ſkutecżná ku poznániu wƺyſtkiego dobrego / które w wáſ jeſt przez Chryſtuſá Jezuſá.
Abowiem rádość wielką mámy y poćiechę z miłośći twojej / bráćie! iż wnętrznośći świętych ſą ochłodzone przez ćię.
Przetoż choć mám wielką bezpiecżność w Chryſtuśie / ábym ći rozkázáł / co przynáleży ;
Jednák dlá miłośći rácżej proƺę / tákowym będąc / to jeſt Páweł ſtáry / á teráz y więzień Jezuſá Chryſtuſá.
Proƺę ćię tedy zá ſynem moim Onezymem / któregom urodził w więzieniu mojem ;
Który tobie niekiedy był niepożytecżny / ále teráz tobie y mnie bárdzo pożytecżny ; któregom odeſłáł.
Przetoż go ty jáko wnętrznośći moje przyjmij.
Któregom já chćiáł przy ſobie zátrzymáć / áby mi poſługiwáł zámiáſt ćiebie w więzieniu dlá Ewángielii.
Ale bez woli twojej nie chćiáłem nic ucżynić / áby dobry twój ucżynek nie był jáko z przymuƺeniá / ále z dobrej woli.
Abowiem ſnáć dlá tego odłącżył śię był ná chwilę od ćiebie / ábyś go záś miáł wiecżnie /
Już nie jáko ſługę / ále więcej niż ſługę / to jeſt brátá miłego / zwłáƺcżá mnie / á jáko dáleko więcej tobie y według ćiáłá y w Pánu.
Przetoż máƺli mię zá towárzyƺá / przyjmij go jáko mię.
A jeźlić w cżem niepráw / álboć co winien / to mnie przycżytáj.
Jám Páweł nápiſáł ręką moją / já nágrodzę / żeć nie rzekę / iżeś mi y ſámego śiebie winien.
Ták / bráćie! niech ćię w tem użyję w Pánu / ochłodź wnętrznośći moje w Pánu.
Pewien będąc poſłuƺeńſtwá twego / piſáłem ći / wiedząc / że y więcej / niż mówię / ucżyniƺ.
Záráz mi też y goſpodę zgotuj ; ábowiem ſpodziewám śię / iż wám przez modlitwy wáƺe dárowány będę.
Pozdráwiáją ćię Epáfráſ / ſpółwięzień mój w Chryſtuśie Jezuśie /
Márek / Aryſtárchuſ / Demáſ / Łukáƺ / pomocnicy moi.
Łáſká Páná náƺego Jezuſá Chryſtuſá niech będzie z duchem wáƺym. Amen.
