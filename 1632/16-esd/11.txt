Mieƺkáli tedy przedniejśi z ludu w Jeruzálemie / á inny lud miotáli loſy / áby wźięli dźieśiątego cżłowieká ná mieƺkánie w Jeruzálemie / mieśćie świętym / á dźiewięć cżęśći w innych miáſtách.
Y błogoſłáwił lud wƺyſtkim mężom / którzy śię dobrowolnie ofiárowáli / áby mieƺkáli w Jeruzálemie.
A ćić ſą przedniejśi oney kráiny / którzy mieƺkáli w Jeruzálemie. ( A w inƺych miáſtách Judzkich mieƺkał káżdy w ośiádłośći ſwojey / y w miáſtách ſwych / Izráelcżycy / kápłáni / y Lewitowie / y Netynejcżycy / y ſynowie ſług Sálomonowych. )
A ták w Jeruzálemie mieƺkáli niektórzy z ſynów Judowych y z ſynów Benjáminowych. Z ſynów Judowych : Atájáƺ / ſyn Uzyjáƺá / ſyná Zácháryjáƺowego / ſyná Amáryjáƺowego / ſyná Sefátyjáƺowego / ſyná Máháleelowego z ſynów Fáreſowych :
Tákże Mááſejáƺ / ſyn Báruchá / ſyná Cholhozowego / ſyná Háſájáƺowego / ſyná Hádájáƺowego / ſyná Jojárybowego / ſyná Zácháryjáƺowego / ſyná Sylońcżykowego.
Wƺyſtkich ſynów Fáreſowych / mieƺkájących w Jeruzálemie / cżtery ſtá ƺeśćdźieśiąt y ośm mężów dużych.
A ſynowie Benjáminowi ći ſą : Sállu / ſyn Meſullámá / ſyná Joedowego / ſyná Fádájáƺowego / ſyná Kolájáƺowego / ſyná Mááſájáƺowego / ſyná Ityjelowego / ſyná Izájáƺowego ;
A po nim Gábáj / Sálláj / wƺyſtkich dźiewięć ſet dwádźieśćiá y ośm.
Y Joel / ſyn Zychry / był przełożonym nád nimi / á Judá / ſyn Senuá / nád miáſtem wtóry.
Z kápłánów mieƺkáli Jedájáƺ / ſyn Jojárybá / y Jáchyn ;
Serájáƺ / ſyn Helkijáƺá / ſyná Meſullámowego / ſyná Sádokowego / ſyná Merájotowego / ſyná Achytobowego / przełożony w domu Bożym.
A bráći ich / którzy odpráwiáli roboty domowe / ośm ſet dwádźieśćiá y dwá ; y Adájáƺ / ſyn Jerohámá / ſyná Pelálijáƺowego / ſyná Amſego / ſyná Zácháryjáƺowego / ſyná Páſſurowego / ſyná Málchyjáƺowego.
A bráći jego przedniejƺych z domów ojcowſkich dwieśćie cżterdźieśći y dwá ; y Amáſeſáj / ſyn Aſáreli / ſyná Achzájowego / ſyná Meſullemitowego / ſyná Immerowego.
A bráći ich / mężów dużych / ſtá dwádźieśćiá y ośm / y przełożony nád nimi Zábdyjel / ſyn Giedolimá.
A z Lewitów Semejáƺ / ſyn Háſſuby / ſyná Aſrykámowego / ſyná Háſábijáƺowego / ſyná Bunni.
A Sábbátáj y Jozábád byli nád robotą / która byłá z dworu przy domu Bożym / á ćić byli z przedniejƺych Lewitów.
A Mátánijáƺ / ſyn Micháſá / ſyná Zábádyjáƺowego / ſyná Aſáfowego / był przedniejƺy w zácżynániu chwały przy modlitwie ; á Bákbukijáƺ wtóry z bráći jego / y Abdá / ſyn Sámmuj / ſyná Gálilowego / ſyná Jedytunowego.
Wƺyſtkich Lewitów było w mieśćie świętym dwieśćie ośmdźieśiąt y cżterech.
A z odźwiernych : Akkub / Tálmon / y bráći ich / y ſtróżów w brámách / ſto śiedmdźieśiąt y dwá.
A drudzy z Izráelá / z kápłánów / z Lewitów / mieƺkáli we wƺyſtkich miáſtách Judzkich / káżdy w dźiedźictwie ſwojim.
Ale Netynejcżycy mieƺkáli w Ofelu ; á Sychá y Gipſá byli nád Netynejcżykámi.
A przełożony nád Lewitámi w Jeruzálemie był Uzy / ſyn Báni / ſyná Cháſábájáƺowego / ſyná Mátánijáƺowego / ſyná Micháſowego. Ci byli z ſynów Aſáfowych śpiewácy przy ſłużbie domu Bożego.
Abowiem rozkázánie królewſkie było o nich / y pewne opátrzenie dla śpiewáków ná káżdy dźień.
A Petáchyjáƺ / ſyn Meſezábelowy / z ſynów Záchárá / ſyná Judowego / był ná miejſcu królewſkiem w káżdej ſpráwie do ludu.
A we wśiách y polách ich z ſynów Judowych mieƺkáli w Káryját Arbie y we wśiách jego / y w Dybon y we wśiách jego / y w Jekábſeel y we wśiách jego ;
Y w Jeſue / y w Moládá / y w Betfelet ;
Y w Háſerſuál / y w Beerſábe y we wśiách jego ;
Y w Sycelegu / y w Mechoná y we wśiách jego ;
Y w Enrymmon / y w Sáráá / y w Jerymut ;
W Zánoe / w Adullám y we wśiách ich ; w Láchys y ná polách jego ; w Aſeku y we wśiách jego. A ták mieƺkáli od Beerſeby áż do Giehennom.
A ſynowie Benjáminowi z Gábáá mieƺkáli w Máchmás / y w Háju / y w Betel y we wśiách jego ;
W Anátot / w Nobie / w Anánijá ;
W Cháſor / w Rámá / w Gietáim ;
W Hádyd / w Seboim / w Nebállát ;
W Lod / y w Ono / y w dolinie rzemieślników.
A z Lewitów mieƺkáli niektórzy w dźiáłách Judzkich y w Benjámickich.
