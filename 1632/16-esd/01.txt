Słowá Nehemijáƺá / ſyná Háchálijáƺowego. Y ſtáło śię mieśiącá Chyſlew / roku dwudźieſtego ( Aſweruſá królá ) gdym był ná zámku w Suſán /
Że przyƺedł Chánáni / jeden z bráći moich / á z nim niektórzy mężowie z Judy / którychem śię pytáł o Żydów / którzy pozoſtáli y wyƺli z więźieniá / y o Jeruzálem.
Y odpowiedźieli mi : Te oſtátki / które pozoſtáły z więźieniá tám w onej kráinie / ſą w wielkiem utrápieniu / y w zelżywośći : nádto mur Jeruzálemſki rozwálony jeſt / y brámy jego ſpálone ſą ogniem.
A gdym uſłyƺáł te ſłowá / śiádłƺy płákáłem y nárzekáłem przez kilká dni / poƺcżąc y modląc śię przed oblicżem Bogá niebieſkiego.
Y rzekłem : Proƺę Pánie / Boże niebieſki / mocny / wielki y ſtráƺny! który ſtrzeżeƺ umowy / y miłośierdźiá z tymi / którzy ćię miłują / y ſtrzegą przykázániá twego.
Niech będźie proƺę ucho twoje nákłonione / á ocży twoje otworzone / ábyś uſłyƺáł modlitwę ſługi twego / którą śię já dźiś modlę przed tobą we dnie y w nocy zá ſynámi Izráelſkimi / ſługámi twymi / y wyznáję grzechy ſynów Izráelſkich / któremiśmy zgrzeƺyli przećiw tobie ; y já y dom ojcá mego zgrzeƺyliśmy.
Srodześmy wyſtąpili przećiwko tobie / y nie przeſtrzegáliśmy przykázáń / y uſtáw / y ſądów / któreś przykázáł Mojżeƺowi / ſłudze twemu.
Wſpomnij proƺę ná ſłowo / któreś przykázáł Mojżeƺowi / ſłudze twemu / mówiąc : Jeźli wy wyſtąpićie / tedy Já wáſ rozproƺę między národy ;
Ale jeźli śię náwróćićie do mnie / á ſtrzedz będźiećie przykázáń moich / y cżynić je / choćbyśćie byli wygnáni ná kráj świátá / tedy y ſtámtąd wáſ zgromádzę / y przyprowádzę wáſ ná miejſce / którem obráł / áby tám przebywáło imię moje.
Wƺák oni ſą ſłudzy twoi / y lud twój / któryś odkupił mocą twoją wielką / y ręką twą śilną.
Proƺę o Pánie! niech teráz będźie ucho twoje nákłonione ku modlitwie ſługi twego y ku modlitwie ſług twoich / którzy máją wolę báć śię imieniá twego ; á zdárz dźiś / proƺę / ſłudze twemu / y ſpráw mu miłośierdźie przed tym mężem. A jám był podcżáƺym królewſkim.
