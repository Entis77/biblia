A gdy uſłyƺáł Sánbállát / y Tobijáƺ / y Gieſem Arábcżyk / y inni nieprzyjáćiele náśi / żem zbudowáł mur / á że w nim nie zoſtáwáło żádnej rozwáliny / choćiáżem jeƺcże wtencżáſ nie był przypráwił wrót do brám :
Tedy poſłáł Sánbállát / y Gieſem do mnie mówiąc : Przyjdź / á zejdźmy śię ſpołem we wśiách / które ſą ná polu Ono. Ale oni myślili ucżynić mi co złego.
Przetoż poſłáłem do nich poſłów / wſkázując : Zácżąłem robotę wielką / przetoż nie mogę zjecháć ; bo przecżżeby miáłá uſtáć tá robotá / gdybym jej zániecháwƺy jecháł do wáſ?
Tedy poſłáli do mnie w tejże ſpráwie po cżtery kroć. A jám im odpowiedziáł temiż ſłowy.
Potem Sánbállát poſłáł do mnie w tejże ſpráwie piąty ráz ſługę ſwego y liſt otwárty / w ręce jego /
W którym to było nápiſáne : Jeſt poſłuch między národámi / jáko Gáſmuſ powiádá / że ty y Żydowie myślićie śię z mocy wybić / á że ty dlátego budujeƺ mur / ábyś był nád nimi królem ich / jáko śię to pokázuje.
Do tego / żeś y proroków poſtánowił / áby powiádáli o tobie w Jeruzálemie / mówiąc : On jeſt królem w Judzie. Teráz tedy dojdzie to królá ; przetoż przyjdź / á nárádzimy śię ſpołecżnie.
Alem poſłáł do niego / mówiąc : Nie jeſt to / co powiádáƺ ; ále ſám ſobie to wymyśláƺ.
Abowiem oni wƺyſcy ſtráƺyli náſ / mówiąc : Oſłábieją ręce ich przy roboćie / y nie dokonáją ; prztoż teráz / o Boże! zmocnij ręce moje.
A gdym wƺedł w dom Semejáƺá / ſyná Delájáƺowego / ſyná Mehetábelowego / który był w záwárćiu / rzekł mi : Zejdźmy śię do domu Bożego / w pośród kośćiołá / y zámknijmy drzwi kośćielne ; bo przyjdą / chcąc ćię zábić / á w nocy przyjdą / áby ćię zábili.
Któremum rzekł : Tákowyżby mąż / jákim já jeſt / miáł ućiekáć? Któż tákowy / jákom já / coby wƺedłƺy do kośćiołá / żyw zoſtáł? Nie wnijdę.
Y poznáłem / że go Bóg nie poſłáł ále proroctwo mówił przećiwko mnie / bo go Tobijáƺ y Sánbállát byli przenájęli.
Przeto bowiem przenájęty był / ábym śię uląkł / y ták ucżynił / y zgrzeƺył / áżeby mi to u nich było ná złe imię / cżemby mi urągáli.
Pomnijże / o Boże mój! ná Tobijáƺá y Sánbállátá / według tákowych ucżynków ich : tákże ná Noádyję prorokinię / y ná innych proroków / którzy mię ſtráƺyli.
A ták dokonány jeſt on mur dwudzieſtego y piątego dniá mieśiącá Elul / pięćdzieśiątego y drugiego dniá.
A gdy to uſłyƺeli wƺyſcy nieprzyjáćiele náśi / y widzieli to wƺyſcy národowie / którzy byli około náſ / upádło im bárdzo ſerce ;
Bo poználi / że śię tá ſpráwá od Bogá náƺego ſtáłá. W oneż dni wiele przedniejƺych z Judy liſty ſwe cżęſto poſyłáli do Tobijáƺá / tákże od Tobijáƺá przychodziły do nich.
Bo wiele ich było w Judzie / co śię z nim ſprzyśięgli / gdyż on był zięćiem Sechánijáƺá / ſyná Aráchowego ; á Jochánán / ſyn jego / pojął był córkę Meſullámá / ſyná Báráchyjáƺowego.
Nádto y dobrocżynność jego opowiádáli przedemną / y ſłowá moje odnośili mu ; á liſty poſyłáł Tobijáƺ / áby mię ſtráƺył.
