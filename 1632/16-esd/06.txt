A gdy uſłyƺał Sánbállát / y Tobijáƺ / y Gieſem Arábcżyk / y inni nieprzyjáćiele náśi / żem zbudował mur / á że w nim nie zoſtáwáło żádney rozwáliny / choćiáżem jeƺcże wtencżás nie był przypráwił wrót do bram :
Tedy poſłał Sánbállát / y Gieſem do mnie mówiąc : Przyjdź / á zejdźmy śię ſpołem we wśiách / które ſą ná polu Ono. Ale oni myślili ucżynić mi co złego.
Przetoż poſłáłem do nich poſłów / wſkázując : Zácżąłem robotę wielką / przetoż nie mogę zjecháć ; bo przecżżeby miáłá uſtáć tá robotá / gdybym jey zániecháwƺy jechał do was?
Tedy poſłáli do mnie w tejże ſpráwie po cżtery kroć. A jam im odpowiedźiał temiż ſłowy.
Potem Sánbállát poſłał do mnie w tejże ſpráwie piąty raz ſługę ſwego y liſt otwárty / w ręce jego /
W którym to było nápiſáne : Jeſt poſłuch miedzy národámi / jáko Gáſmus powiádá / że ty y Żydowie myślićie śię z mocy wybić / á że ty dla tego budujeƺ mur / ábyś był nád nimi królem ich / jáko śię to pokázuje.
Do tego / żeś y proroków poſtánowił / áby powiádáli o tobie w Jeruzálemie / mówiąc : On jeſt królem w Judźie. Teraz tedy dojdźie to królá ; przetoż przyjdź / á nárádźimy śię ſpołecżnie.
Alem poſłał do niego / mówiąc : Nie jeſt to / co powiádáƺ ; ále ſám ſobie to wymyśláƺ.
Abowiem oni wƺyſcy ſtráƺyli nas / mówiąc : Oſłábieją ręce ich przy roboćie / y nie dokonáją ; przetoż teraz / o Boże! zmocnij ręce moje.
A gdym wƺedł w dom Semejáƺá / ſyná Delájáƺowego / ſyná Mehetábelowego / który był w záwárćiu / rzekł mi : Zejdźmy śię do domu Bożego / w pośród kośćiołá / y zámknijmy drzwi kośćielne ; bo przyjdą / chcąc ćię zábić / á w nocy przyjdą / áby ćię zábili.
Któremum rzekł : Tákowyżby mąż / jákim ja jeſt / miał ućiekáć? Któż tákowy / jákom ja / coby wƺedƺy do kośćiołá / żyw zoſtał? Nie wnijdę.
Y poznáłem / że go Bóg nie poſłał ále proroctwo mówił przećiwko mnie / bo go Tobijáƺ y Sánbállát byli przenájęli.
Przeto bowiem przenájęty był / ábym śię uląkł / y ták ucżynił / y zgrzeƺył / áżeby mi to u nich było ná złe imię / cżemby mi urągáli.
Pomnijże / o Boże mój! ná Tobijáƺá y Sánbállátá / według tákowych ucżynków ich : tákże ná Noádyję prorokinię / y ná innych proroków / którzy mię ſtráƺyli.
A ták dokonány jeſt on mur dwudźieſtego y piątego dniá mieśiącá Elul / pięćdźieśiątego y drugiego dniá.
A gdy to uſłyƺeli wƺyſcy nieprzyjáćiele náśi / y widźieli to wƺyſcy národowie / którzy byli około nas / upádło im bárzo ſerce ;
Bo poználi / że śię tá ſpráwá od Bogá náƺego ſtáłá. W oneż dni wiele przedniejƺych z Judy liſty ſwe cżęſto poſyłáli do Tobijáƺá / tákże od Tobijáƺá przychodźiły do nich.
Bo wiele ich było w Judźie / co śię z nim ſprzyśięgli / gdyż on był źięćiem Sechánijáƺá / ſyná Aráchowego ; á Jochánán / ſyn jego / pojął był córkę Meſullámá / ſyná Báráchyjáƺowego.
Nádto y dobrocżynność jego opowiádáli przedemną / y ſłowá moje odnośili mu ; á liſty poſyłał Tobijáƺ / áby mię ſtráƺył.
