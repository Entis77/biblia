A gdy był dobudowány mur / y wyſtáwiłem wrotá / y poſtánowieni byli odźwierni / y śpiewácy / y Lewitowie :
Rozkazałem Chánánijemu / brátu memu / y Chánánijáƺowi / ſtárośćie zámku Jeruzálemſkiego : ( bo ten był mąż wierny / y bojący śię Bogá nád wielu innych ; )
Y rzekłem do nich : Niech nie otwieráją bram Jeruzálemſkich / áż ogrzeje ſłońce ; á gdy ći / co tu ſtáwáją / zámkną bramę / opátrzćież ją. A ták poſtánowiłem ſtráż z obywáteli Jeruzálemſkich / káżdego ná ſtráży jego / y káżdego ná przećiwko domowi jego.
A miáſto było ƺerokie y wielkie ; ále ludu máło w murách jego / á jeƺcże nie były domy pobudowáne.
Przetoż podał Bóg mój do ſercá mego / ábym zebrał przedniejƺych / y przełożonych / y lud / áby byli oblicżeni podług rodzáju. Y ználázłem kśięgi rodu tych / którzy śię tu nájpierwej przyprowádźili / y ználázłem w nich to opiſánie.
Cić ſą ludźie oney kráiny / którzy wyƺli z pojmánia y z niewoli / w którą ich był záprowádźił Nábuchodonozor / król Bábilońſki ; á wróćili śię do Jeruzálemu y do Judy / káżdy do miáſtá ſwego.
Którzy przyƺli z Zorobábelem / z Jeſuą / Nehemijáƺem / Azáryjáƺem / Ráhámijáƺem / Nechámánem / Márdocheuƺem / Bilſánem / Miſperetem / Bigwájem / Nechumem / Bááną.
A pocżet mężów ludu Izráelſkiego táki jeſt : Synów Fároſowych dwá tyśiące ſto y śiedmdźieśiąt y dwá ;
Synów Sefátyjáƺowych trzy ſtá śiedmdźieśiąt y dwá ;
Synów Aráchowych ƺeść ſet pięćdźieśiąt y dwá.
Synów Páchátmoábowych / ſynów Jeſui / y Joábowych dwá tyśiące ośm ſet y ośmnaśćie ;
Synów Elámowych tyśiąc dwieśćie pięćdźieśiąt y cżtery ;
Synów Záttuowych ośm ſet cżterdźieśći y pięć ;
Synów Záchájowych śiedm ſet y ƺeśćdźieśiąt ;
Synów Binnujowych ƺeść ſet cżterdźieśći y ośm ;
Synów Bebájowych ƺeść ſet dwádźieśćiá y ośm ;
Synów Azgádowych dwá tyśiące trzy ſtá dwádźieśćiá y dwá ;
Synów Adonikámowych ƺeść ſet ƺeśćdźieśiąt y śiedm ;
Synów Bigwájowych dwá tyśiące ƺeśćdźieśiąt y śiedm ;
Synów Adynowych ƺeść ſet pięćdźieśiąt y pięć ;
Synów Aterowych / co poƺli z Ezechyjáƺá / dźiewięćdźieśiąt y ośm ;
Synów Háſumowych trzy ſtá dwádźieśćiá y ośm ;
Synów Beſájowych trzy ſtá dwádźieśćiá y cżtery ;
Synów Cháryfowych ſto y dwánaśćie ;
Synów z Gábáonu dźiewięćdźieśiąt y pięć.
Mężów z Betlehemu y Netofátu ſto ośmdźieśiąt y ośm ;
Mężów z Anátotu ſto dwádźieśćiá y ośm ;
Mężów z Bet Azmáwetu cżterdźieśći y dwá ;
Mężów z Káryjátyjárymu / z Káfiry y z Beerotu śiedm ſet cżterdźieśći y trzy ;
Mężów z Rámy y z Gábáá ƺeść ſet dwádźieśćiá y jeden ;
Mężów z Máchmás ſto y dwádźieśćiá y dwá ;
Mężów z Betelá y z Háj ſto dwádźieśćiá y trzy ;
Mężów z Nebo drugiego pięćdźieśiąt y dwá.
Synów Elámá drugiego tyśiąc dwieśćie pięćdźieśiąt y cżtery ;
Synów Hárymowych trzy ſtá y dwádźieśćiá ;
Synów Jerechowych trzy ſtá cżterdźieśći y pięć ;
Synów Lodowych / Hádydowych / y Onowych śiedm ſet y dwádźieśćiá y jeden.
Synów Senáá trzy tyśiące dźiewięć ſet y trzydźieśći.
Kápłanów : Synów Jedájáƺowych / z domu Jeſui / dźiewięć ſet śiedmdźieśiąt y trzy ;
Synów Immerowych tyśiąc pięćdźieśiąt y dwá ;
Synów Páſſurowych tyśiąc dwieśćie cżterdźieśći y śiedm ;
Synów Hárymowych tyśiąc y śiedmnaśćie ;
Lewitów : Synów Jeſuego / y Kádmielowych / ſynów Hodowijáƺowych śiedmdźieśiąt y cżtery.
Śpiewáków : Synów Aſáfowych ſto cżterdźieśći y ośm.
Odźwiernych : Synów Sállumowych / ſynów Aterowych / ſynów Tálmonowych / ſynów Akkubowych / ſynów Hátytowych / ſynów Sobájowych ſto trzydźieśći y ośm.
Z Netynejcżyków : Synów Sychá / ſynów Cháſufá / ſynów Tábbáotá /
Synów Kieroſá / ſynów Syjáá / ſynów Fádoná /
Synów Lebáná / ſynów Hágábá / ſynów Sálmájá /
Synów Hánáná / ſynów Giddelá / ſynów Gáchárá /
Synów Ráájáƺá / ſynów Rezyná / ſynów Nekodá /
Synów Gázámá / ſynów Uzy / ſynów Fáſeáchá.
Synów Beſájá / ſynów Mechynimá / ſynów Nefuſeſymá /
Synów Bákbuká / ſynów Chákufá / ſynów Chárchurá /
Synów Báſlutá / ſynów Mechydá / ſynów Chárſá /
Synów Bárkoſá / ſynów Syſerá / ſynów Támáchá /
Synów Nezyjáchá / ſynów Chátyfá /
Synów ſług Sálomonowych / ſynów Sotájá / ſynów Soferátá / ſynów Perudá /
Synów Jáhálá / ſynów Dárkoná / ſynów Giddelá /
Synów Sefátyjáƺá / ſynów Chátylá / ſynów Pocheretá z Háſebáim / ſynów Amoná :
Wƺyſtkich Netynejcżyków y ſynów ſług Sálomonowych trzy ſtá dźiewięćdźieśiąt y dwá.
A ćić ſą / którzy wyƺli z Telmeláchu y z Telchárſá : Cherub / Addán / y Immer : ále nie mogli okázáć domu ojców ſwojich y náśieniá ſwego / jeſli z Izráelá byli.
Synów Delájáƺowych / ſynów Tobijáƺowych / ſynów Nekodowych ƺeść ſet cżterdźieśći y dwá.
A z kápłanów ſynowie Hobájowi / ſynowie Kozowi / ſynowie Bárſylájego ; który był pojął z córek Bárſylájá Gáláádcżyká żonę / y názwány był od imienia ich.
Ci ƺukáli opiſánia ſwego / wywodząc ród ſwój / ále nie ználeſli ; przetoż zrzuceni ſą z kápłáńſtwá.
Y zákázał im Tyrſátá / áby nie jedli z rzecży nájświętƺych / áżby powſtał kápłán z Urym y z Tummim.
Wƺyſtkiego zgromádzenia w jednym pocżćie było cżterdźieśći tyśięcy dwá tyśiące trzy ſtá y ƺeśćdźieśiąt ;
Oprócż ſług ich y ſłużebnic ich / których było śiedm tyśięcy trzy ſtá trzydźieśći y śiedm ; á miedzy nimi było śpiewáków y śpiewácżek dwieśćie y cżterdźieśći y pięć.
Koni ich śiedm ſet trzydźieśći y ƺeść ; mułów ich dwieśćie cżterdźieśći y pięć.
Wielbłądów cżtery ſtá trzydźieśći y pięć ; oſłów ƺeść tyśięcy śiedm ſet y dwádźieśćiá.
A niektórzy przedniejśi z domów ojcowſkich dáwáli ná robotę. Tyrſátá dał do ſkárbu złotá tyśiąc łótów / cżáƺ pięćdźieśiąt / ƺát kápłáńſkich pięć ſet y trzydźieśći.
Niektórzy też z przedniejƺych domów ojcowſkich dáli do ſkárbu ná robotę złotá dwádźieśćiá tyśięcy łótów / á ſrebrá grzywien dwá tyśiące y dwieśćie.
A co dał inƺy lud / było złotá dwádźieśćiá tyśięcy łótów / á ſrebrá dwá tyśiące grzywien / á ƺát kápłáńſkich ƺeśćdźieśiąt y śiedm.
A ták ośiedli kápłáni y Lewitowie / y odźwierni / y śpiewácy / y lud poſpolity / y Netynejcżycy / y wƺyſtek Izráel miáſtá ſwoje. A gdy náſtał mieśiąc śiódmy / byli ſynowie Izráelſcy w miáſtách ſwojich.
