Y ſtáło śię mieśiącá Niſán roku dwudźieſtego Artákſerkſeſá królá / gdy było wino przed nim / że wźiąwƺy wino / podáłem je królowi / á nie bywáłem przedtem ták ſmutny przed nim.
Y rzekł mi król : Cżemuż twárz twojá ták ſmutná / gdyż nie chorujeƺ? Nic to innego / jedno ſmutek ſercá. Y zlękłem śię náder bárzo.
Y rzekłem do królá : Niech król ná wieki żyje. Jákoż nie ma być ſmutná twárz mojá / gdyż miáſto / dom grobów ojców mojich / zburzono / á brámy jego ogniem popálono?
Znowu rzekł do mnie król : Cżegoż ty żądáƺ? A jam śię modlił Bogu niebieſkiemu.
Y rzekłem do królá : Zdáli śię to zá rzecż dobrą królowi / y jeſli ma łáſkę ſługá twój przed oblicżem twojim / proƺę / ábyś mię poſłał do źiemi Judzkiej / do miáſtá grobów ojców mojich / ábym je pobudował.
Nádto rzekł mi król ( á królowá śiedźiáłá podle niego ) : Długoż będźieƺ ná tey drodze / y kiedy śię wróćiƺ? Y podobáło śię to królowi / y poſłał mię / gdym mu zámierzył pewny cżás.
Zátemem rzekł do królá : Zdáli śię to zá rzecż dobrą królowi / niech mi dádzą liſty do ſtároſtów zá rzeką / áby mię przeprowádźili / áżbym przyƺedł do źiemi Judzkiej ;
Y liſt do Aſáfá / dozorcy láſów królewſkich / áby mi dał drzewá ná przykryćie brám páłácu przy domu Bożym / y ná mur miejſki / y ná dom / do którego wnijdę. Y dał mi król liſty według ręki Bogá mego łáſkáwej nádemną.
A gdym przyƺedł do ſtároſtów zá rzeką / oddáłem im liſty królewſkie. Poſłał też był ze mną król rotmiſtrzów y jezdnych :
Co gdy uſłyƺał Sánbállát Horonitcżyk / y Tobijáƺ / ſługá Ammonitcżyk / bárzo ich to mierźiáło / że przyƺedł cżłowiek / który śię ſtárał o dobro ſynów Izráelſkich.
Zátem przyƺedƺy do Jeruzálemu / mieƺkáłem tám przez trzy dni.
A wſtáwƺy w nocy / ja y mężów trochá ze mną / nie oznájmiłem nikomu / co Bóg mój podał do ſercá mego / ábym ucżynił w Jeruzálemie ; bydlęćiá też nie miáłem z ſobą / oprócż bydlęćiá / ná któremem jechał.
Y wyjecháłem brámą nád doliną w nocy / ku źródłu ſmocżemu / y ku brámie gnojowej / y oglądáłem mury Jeruzálemſkie / które były rozwálone / y brámy jego / które były popálone ogniem.
Potem jecháłem ku brámie źródłá / y ku ſádzáwce królewſkiej / gdźie nie było miejſcá bydlęćiu / ná któremem jechał / áby przejść mogło.
Przetoż jecháłem nád potokiem w nocy / á oglądáłem mury ; ſkąd wrácájąc śię / wyjecháłem brámą nád doliną / y tákiem powróćił.
Ale kśiążętá nie wiedźieli / gdźiem jeźdźił / y com cżynił ; bom Żydom / áni kápłánom / áni kśiążętom / áni urzędnikom / áni żádnemu rzemieślnikowi tego áż dotąd nie oznájmił.
Przetożem rzekł do nich : Wy widźićie / w jákiemeśmy ućiśnieniu / á jáko Jeruzálem ſpuſtoƺone / y brámy jego popálone ſą ogniem. Pójdźćież / á budujmy mury Jeruzálemſkie / ábyſmy nie byli więcey ná háńbę.
A gdym im oznájmił / że ręká Bogá mego byłá łáſkáwá nádemną / tákże y ſłowá królewſkie / które do mnie mówił / rzekli : Wſtáńmyż á budujmy. Y zmocnili ręce ſwe ku dobremu.
Co ſłyƺąc Sánbállát Horonitcżyk / y Tobijáƺ / ſługá Ammonitcżyk y Gieſem Arábcżyk / ƺydźili z nas / y lekce nas ſobie powáżyli mówiąc : Cóż to zá rzecż / którą cżynićie? álbo śię przećiw królowi buntujećie?
Y odpowiedźiáłem im / á rzekłem do nich : Bóg niebieſki / ten nam poƺcżęśći / á my ſłudzy jego / wſtáńmy á budujmy ; ále wy nie máćie dźiáłu / áni práwá / áni pámiątki w Jeruzálemie.
