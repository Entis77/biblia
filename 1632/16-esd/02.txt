Y ſtáło śię mieśiącá Niſán roku dwudzieſtego Artákſerkſeſá królá / gdy było wino przed nim / że wziąwƺy wino / podáłem je królowi / á nie bywáłem przedtem ták ſmutny przed nim.
Y rzekł mi król : Cżemuż twárz twojá ták ſmutná / gdyż nie chorujeƺ? Nic to innego / jedno ſmutek ſercá. Y zlękłem śię náder bárdzo.
Y rzekłem do królá : Niech król ná wieki żyje. Jákoż nie má być ſmutná twárz mojá / gdyż miáſto / dom grobów ojców moich / zburzono / á brámy jego ogniem popálono?
Znowu rzekł do mnie król : Cżegoż ty żądáƺ? A jám śię modlił Bogu niebieſkiemu.
Y rzekłem do królá : Zdáli śię to zá rzecż dobrą królowi / y jeźli má łáſkę ſługá twój przed oblicżem twojim / proƺę / ábyś mię poſłáł do ziemi Judzkiej / do miáſtá grobów ojców moich / ábym je pobudowáł.
Nádto rzekł mi król ( á królowá śiedziáłá podle niego ) : Długoż będzieƺ ná tej drodze / y kiedy śię wróćiƺ? Y podobáło śię to królowi / y poſłáł mię / gdym mu zámierzył pewny cżáſ.
Zátemem rzekł do królá : Zdáli śię to zá rzecż dobrą królowi / niech mi dádzą liſty do ſtároſtów zá rzeką / áby mię przeprowádzili / áżbym przyƺedł do ziemi Judzkiej ;
Y liſt do Aſáfá / dozorcy láſów królewſkich / áby mi dáł drzewá ná przykryćie brám páłácu przy domu Bożym / y ná mur miejſki / y ná dom / do którego wnijdę. Y dáł mi król liſty według ręki Bogá mego łáſkáwej nádemną.
A gdym przyƺedł do ſtároſtów zá rzeką / oddáłem im liſty królewſkie. Poſłáł też był ze mną król rotmiſtrzów y jezdnych :
Co gdy uſłyƺáł Sánbállát Horonitcżyk / y Tobijáƺ / ſługá Ammonitcżyk / bárdzo ich to mierziáło / że przyƺedł cżłowiek / który śię ſtáráł o dobro ſynów Izráelſkich.
Zátem przyƺedłƺy do Jeruzálemu / mieƺkáłem tám przez trzy dni.
A wſtáwƺy w nocy / já y mężów trochá ze mną / nie oznájmiłem nikomu / co Bóg mój podáł do ſercá mego / ábym ucżynił w Jeruzálemie ; bydlęćiá też nie miáłem z ſobą / oprócż bydlęćiá / ná któremem jecháł.
Y wyjecháłem brámą nád doliną w nocy / ku źródłu ſmocżemu / y ku brámie gnojowej / y oglądáłem mury Jeruzálemſkie / które były rozwálone / y brámy jego / które były popálone ogniem.
Potem jecháłem ku brámie źródłá / y ku ſádzáwce królewſkiej / gdzie nie było miejſcá bydlęćiu / ná któremem jecháł / áby przejść mogło.
Przetoż jecháłem nád potokiem w nocy / á oglądáłem mury ; ſkąd wrácájąc śię / wyjecháłem brámą nád doliną / y tákiem powróćił.
Ale kśiążętá nie wiedzieli / gdziem jeździł / y com cżynił ; bom Żydom / áni kápłánom / áni kśiążętom / áni urzędnikom / áni żádnemu rzemieślnikowi tego áż dotąd nie oznájmił.
Przetożem rzekł do nich : Wy widzićie / w jákiemeśmy ućiśnieniu / á jáko Jeruzálem ſpuſtoƺone / y brámy jego popálone ſą ogniem. Pójdźćież / á budujmy mury Jeruzálemſkie / ábyśmy nie byli więcej ná háńbę.
A gdym im oznájmił / że ręká Bogá mego byłá łáſkáwá nádemną / tákże y ſłowá królewſkie / które do mnie mówił / rzekli : Wſtáńmyż á budujmy. Y zmocnili ręce ſwe ku dobremu.
Co ſłyƺąc Sánbállát Horonitcżyk / y Tobijáƺ / ſługá Ammonitcżyk y Gieſem Arábcżyk / ƺydzili z náſ / y lekce náſ ſobie powáżyli mówiąc : Cóż to zá rzecż / którą cżynićie? álbo śię przećiw królowi buntujećie?
Y odpowiedziáłem im / á rzekłem do nich : Bóg niebieſki / ten nám poƺcżęśći / á my ſłudzy jego / wſtáńmy á budujmy ; ále wy nie máćie dziáłu / áni práwá / áni pámiątki w Jeruzálemie.
