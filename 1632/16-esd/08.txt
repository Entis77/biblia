Zebráł śię tedy wƺyſtek lud jednoſtájnie ná ulicę / którá jeſt przed brámą wodną / y rzekli do Ezdráƺá / náucżonego w Piśmie / áby przynióſł kśięgi zákonu Mojżeƺowego / który był przykázáł Pán Izráelowi.
Tedy przynióſł Ezdráƺ kápłán zákon przed ono zgromádzenie mężów y niewiáſt / y wƺyſtkich / którzyby rozumnie ſłucháć mogli ; á dziáło śię to dniá pierwƺego / mieśiącá śiódmego.
Y cżytáł w nim ná onej ulicy / którá jeſt przed brámą wodną / od poránku áż do południá przed mężámi y niewiáſtámi / y którzy zrozumieć mogli / á uƺy wƺyſtkiego ludu obrócone były do kśiąg zákonu.
Y ſtánął Ezdráƺ náucżony w Piśmie ná kázálnicy / którą byli zgotowáli ná to / á podle niego ſtáł Mátytyjáƺ / y Semá / y Anánijáƺ / y Uryjáƺ / y Helkijáƺ / y Mááſyjáƺ / po práwej ręce jego / á po lewej ręce jego Fedájáƺ / y Miſáel / y Málchyjáƺ / y Cháſum / y Cháſbádáná / Zácháryjáƺ y Meſullám.
Otworzył tedy Ezdráƺ kśięgi przed ocżymá wƺyſtkiego ludu / bo ſtáł wyżej niż wƺyſtek lud ; á gdy je otworzył / wƺyſtek lud powſtáł.
Y błogoſłáwił Ezdráƺ Pánu / Bogu wielkiemu / á wƺyſtek lud odpowiádáł : Amen! Amen! podnoƺąc ręce ſwoje ; á náchyliwƺy głowy / kłániáli śię Pánu twrzą ku ziemi.
Tákże y Jeſuá / y Báni / y Serebijáƺ / Jámin / Chákub / Sábbetáj / Hodyjáƺ / Mááſyjáƺ / Kielitá / Azáryjáƺ / Jozábád / Chánán / Felájáƺ / y Lewitowie náucżáli ludu zákonu / á lud ſtáł ná miejſcu ſwem.
Bo cżytáli w kśięgách zákonu Bożego wyráźnie / á wykłádájąc zmyſł objáśniáli to / co cżytáli.
Zátem Nehemijáƺ ( ten jeſt Tyrſátá ) y Ezdráƺ kápłán / náucżony w Piśmie / y Lewitowie / którzy ucżyli lud / rzekli do wƺyſtkiego ludu : Ten dzień poświęcony jeſt Pánu / Bogu wáƺemu / nie ſmuććież śię / áni płácżćie. ( Bo płákáł wƺyſtek lud / ſłyƺąc ſłowá zákonu. )
Y rzekł im : Idźćież / jedzćie rzecży tłuſte á pijćie nápój ſłodki / á poſyłájćie cżąſtki tym / którzy ſobie nic nie nágotowáli ; ábowiem święty jeſt dzień Pánu náƺemu. Przetoż śię nie fráſujćie ; ábowiem weſele Páńſkie jeſt śiłą wáƺą.
A gdy Lewitowie ucżynili milcżenie między wƺyſtkim ludem / mówiąc : Milcżćież / bo dzień święty jeſt / á nie ſmuććie śię :
Tedy odƺedł wƺyſtek lud / áby jedli y pili / y áby innym cżąſtki poſyłáli. Y weſelili śię bárdzo / przeto / że zrozumieli ſłowá / których ich náucżáno.
Potem zebráli śię dniá drugiego przedniejśi domów ojcowſkich ze wƺyſtkiego ludu / kápłáni y Lewitowie / do Ezdráƺá náucżonego w Piśmie / áby wyrozumieli ſłowá zákonu.
Y ználeźli nápiſáne w zákonie / że rozkázáł Pán przez Mojżeƺá / áby mieƺkáli ſynowie Izráelſcy w kucżkách w święto urocżyſte mieśiącá śiódmego ;
A iżby to opowiedziáno y obwołáno we wƺyſtkich miáſtách ich / y w Jeruzálemie / mówiąc : Wynijdźćie ná górę / á nánośćie gáłęziá oliwnego / y gáłęziá ſoſnowego / y gáłęziá myrtowego / y gáłęziá pálmowego / y gáłęziá drzewá gęſtego / ábyśćie pocżynili kucżki / jáko jeſt nápiſáne.
Przetoż wyƺedł lud / á nánośili y pocżynili ſobie kucżki / káżdy ná dáchu ſwym / y w śieniách ſwych / y w śieniách domu Bożego / y ná ulicy brámy wodnej / y ná ulicy brámy Efráimowej.
A ták nácżyniło kucżek wƺyſtko zgromádzenie / które śię wróćiło z niewoli / y mieƺkáli w kucżkách / ( choć tego nie cżynili ſynowie Izráelſcy ode dni Jozuego / ſyná Nunowego / áż do dniá onego ; ) y było weſele bárdzo wielkie.
A Ezdráƺ cżytáł w kśięgách zákonu Bożego ná káżdy dzień ; od pierwƺego dniá áż do dniá oſtátniego ; y obchodzili święto urocżyſte przez śiedm dni / á dniá óſmego było zgromádzenie według zwycżáju.
