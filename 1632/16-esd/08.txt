Zebrał śię tedy wƺyſtek lud jednoſtájnie ná ulicę / która jeſt przed bramą wodną / y rzekli do Ezdráƺá / náucżonego w Piśmie / áby przynióſł kśięgi zakonu Mojżeƺowego / który był przykázał Pán Izráelowi.
Tedy przynióſł Ezdráƺ kápłán zakon przed ono zgromádzenie mężów y niewiáſt / y wƺyſtkich / którzyby rozumnie ſłucháć mogli ; á dźiáło śię to dniá pierwƺego / mieśiącá śiódmego.
Y cżytał w nim ná oney ulicy / która jeſt przed bramą wodną / od poránku áż do południá przed mężámi y niewiáſtámi / y którzy zrozumieć mogli / á uƺy wƺyſtkiego ludu obrócone były do kśiąg zakonu.
Y ſtánął Ezdráƺ náucżony w Piśmie ná kázálnicy / którą byli zgotowáli ná to / á podle niego ſtał Mátytyjáƺ / y Semá / y Anánijáƺ / y Uryjáƺ / y Helkijáƺ / y Mááſyjáƺ / po práwej ręce jego / á po lewej ręce jego Fedájáƺ / y Miſáel / y Málchyjáƺ / y Cháſum / y Cháſbádáná / Zácháryjáƺ y Meſullám.
Otworzył tedy Ezdráƺ kśięgi przed ocżymá wƺyſtkiego ludu / bo ſtał wyżey niż wƺyſtek lud ; á gdy je otworzył / wƺyſtek lud powſtał.
Y błogoſłáwił Ezdráƺ Pánu / Bogu wielkiemu / á wƺyſtek lud odpowiádał : Amen! Amen! podnoƺąc ręce ſwoje ; á náchyliwƺy głowy / kłániáli śię Pánu twárzą ku źiemi.
Tákże y Jeſuá / y Báni / y Serebijáƺ / Jámin / Chákub / Sábbetáj / Hodyjáƺ / Mááſyjáƺ / Kielitá / Azáryjáƺ / Jozábád / Chánán / Felájáƺ / y Lewitowie náucżáli ludu zakonu / á lud ſtał ná miejſcu ſwem.
Bo cżytáli w kśięgách zakonu Bożego wyráźnie / á wykłádájąc zmyſł objáśniáli to / co cżytáli.
Zátem Nehemijáƺ ( ten jeſt Tyrſátá ) y Ezdráƺ kápłán / náucżony w Piśmie / y Lewitowie / którzy ucżyli lud / rzekli do wƺyſtkiego ludu : Ten dźień poświęcony jeſt Pánu / Bogu wáƺemu / nie ſmuććież śię / áni płácżćie. ( Bo płákał wƺyſtek lud / ſłyƺąc ſłowá zakonu. )
Y rzekł im : Idźćież / jedzćie rzecży tłuſte á pijćie nápój ſłodki / á poſyłájćie cżąſtki tym / którzy ſobie nic nie nágotowáli ; ábowiem święty jeſt dźień Pánu náƺemu. Przetoż śię nie fráſujćie ; ábowiem weſele Páńſkie jeſt śiłą wáƺą.
A gdy Lewitowie ucżynili milcżenie miedzy wƺyſtkim ludem / mówiąc : Milcżćież / bo dźień święty jeſt / á nie ſmuććie śię :
Tedy odƺedł wƺyſtek lud / áby jedli y pili / y áby innym cżąſtki poſyłáli. Y weſelili śię bárzo / przeto / że zrozumieli ſłowá / których ich náucżáno.
Potem zebráli śię dniá drugiego przedniejśi domów ojcowſkich ze wƺyſtkiego ludu / kápłáni y Lewitowie / do Ezdráƺá náucżonego w Piśmie / áby wyrozumieli ſłowá zakonu.
Y ználeſli nápiſáne w zakonie / że rozkázał Pán przez Mojżeƺá / áby mieƺkáli ſynowie Izráelſcy w kucżkách w święto urocżyſte mieśiącá śiódmego ;
A iżby to opowiedźiáno y obwołáno we wƺyſtkich miáſtách ich / y w Jeruzálemie / mówiąc : Wynijdźćie ná górę / á nánośćie gáłęźiá oliwnego / y gáłęźiá ſoſnowego / y gáłęźiá myrtowego / y gáłęźiá pálmowego / y gáłęźiá drzewá gęſtego / ábyśćie pocżynili kucżki / jáko jeſt nápiſáne.
Przetoż wyƺedł lud / á nánośili y pocżynili ſobie kucżki / káżdy ná dáchu ſwym / y w śieniách ſwych / y w śieniách domu Bożego / y ná ulicy bramy wodney / y ná ulicy bramy Efráimowej.
A ták nácżyniło kucżek wƺyſtko zgromádzenie / które śię wróćiło z niewoli / y mieƺkáli w kucżkách / ( choć tego nie cżynili ſynowie Izráelſcy ode dni Jozuego / ſyná Nunowego / áż do dniá onego ; ) y było weſele bárzo wielkie.
A Ezdráƺ cżytał w kśięgách zakonu Bożego ná káżdy dźień ; od pierwƺego dniá áż do dniá oſtátniego ; y obchodźili święto urocżyſte przez śiedm dni / á dniá óſmego było zgromádzenie według zwycżáju.
