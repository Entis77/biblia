Potem dniá dwudzieſtego y cżwártego tegoż mieśiącá zgromádzili śię ſynowie Izráelſcy / y pośćili w worzech / y poſypáli śię prochem.
A odłącżyło śię náśienie Izráelſkie od wƺyſtkich cudzoziemców / á ſtánąwƺy wyznáwáli grzechy ſwe y niepráwośći ojców ſwych.
Y powſtáli ná miejſcách ſwych / á cżytáli kśięgi zákonu Páná / Bogá ſwego / cżtery kroć przez dzień / y cżtery kroć wyznáwáli á kłániáli śię Pánu Bogu ſwemu.
Zátem ſtánęli ná ſtopniách Lewitów Jeſuá y Báni / Kádmiel / Sebánijáƺ / Bunni / Serebijáƺ / Báni / Chenáni / á wołáli głoſem wielkim do Páná Bogá ſwego.
Y mówili Lewitowie Jeſuá / y Kádmiel / Báni / Háſábnejáƺ / Serebijáƺ / Odyjáƺ / Sebánijáƺ / Petáchyjáƺ : Wſtáńćie / błogoſłáwćie Pánu / Bogu wáƺemu / od wieku áż ná wieki / á niech błogoſłáwią imieniowi twojimu chwálebnemu y wywyżƺonemu nád wƺelkie błogoſłáwieńſtwo y chwáłę.
Ty / Pánie! ſám / ty ſám / tyś ucżynił niebo / niebá niebioſ / y wƺyſtko wojſko ich / ziemię / y wƺyſtko co jeſt ná niej / morzá / y wƺyſtko / co w nich jeſt / á ty ożywiáƺ to wƺyſtko ; á wojſká niebieſkie tobie śię kłániáją.
Tyś jeſt / Pánie Boże! któryś wybráł Abrámá / á wywiodłeś go z Ur Cháldejſkiego / y dáłeś mu imię Abráhám.
Y ználázłeś ſerce jego wierne przed oblicżem twojim / y ucżyniłeś z nim przymierze / że dáƺ ziemię Chánánejcżyká / Hetejcżyká / Amorejcżyká / y Ferezejcżyká / y Jebuzejcżyká / y Giergiezejcżyká / że ją dáƺ náśieniu jego / y ziśćiłeś ſłowá twoje ; boś ty ſpráwiedliwy.
Wejżáłeś záiſte ná utrápienie ojców náƺych w Egipćie / á wołánie ich wyſłucháłeś nád morzem Cżerwonem.
A pokázywáłeś znáki y cudá ná Fáráonie / y ná wƺyſtkich ſługách jego / y ná wƺyſtkim ludu ziemi jego ; boś poznáł / że ſobie hárdzie poſtępowáli przećiwko nim / y ucżyniłeś ſobie imię / jáko śię to dziś pokázuje.
Y rozdzieliłeś morze przed nimi / á przeƺli przez pośrodek morzá po ſuƺy ; á tych / którzy ich gonili / wrzućiłeś w głębokośći / jáko kámień w wody gwáłtowne.
A w ſłupie obłokowym prowádziłeś ich we dnie / á w ſłupie ogniſtym w nocy / ábyś im oświecáł drogę / którą iść mieli.
Potemeś ná górę Synáj zſtąpił / y mówiłeś do nich z niebá / á dáłeś im ſądy práwe / y zákony práwdziwe / uſtáwy y rozkázániá dobre.
Y ſábát twój święty oznájmiłeś im / á przykázániá y uſtáwy / y zákon wydáłeś im przez Mojżeƺá / ſługę twego.
Dáłeś im też chleb w głodzie ich z niebá / y wodęś im z ſkáły wywiódł w prágnieniu ich / á rozkázáłeś im / áby ƺli / y pośiedli ziemię / o którąś podnióſł rękę ſwą / że im ją dáƺ.
Ale oni y ojcowie náśi hárdzie ſobie pocżynáli / y zátwárdzili kárk ſwój / y nie ſłucháli rozkázániá twego.
Owƺem nie chćieli ſłucháć / áni wſpomnieli ná cudá twoje / któreś cżynił / przy nich ; ále zátwárdziwƺy kárk ſwój poſtánowili ſobie wodzá / chcąc śię wróćić w niewolę ſwoję w uporze ſwoim. Lecż ty / o Boże miłośćiwy / łáſkáwy / y miłośierny / nieſkwápliwy / y wielkiego miłośierdziá! nie opuśćiłeś ich.
Náwet gdy ſobie ucżynili ćielcá ulánego / á mówili : Ten jeſt Bóg twój / który ćię wywiódł z ziemi Egipſkiej / y dopuśćili śię bluźnierſtw wielkich /
Ty jednák dlá litośći twoich wielkich nie opuśćiłeś ich ná puƺcży ; ſłup obłokowy nie odſtąpił od nich we dnie / prowádząc ich w drodze / áni ſłup ogniſty w nocy / oświecájąc ich / y drogę / którą iść mieli.
Nádto duchá twojego dobrego dáłeś im / áby ich ucżył / y mánny twojej nie odjąłeś od uſt ich / y wodę dáłeś im w prágnieniu ich.
A ták przez cżterdzieśći lát żywiłeś ich ná puƺcży ; ná nicżem im nie ſchodziło / ƺáty ich nie zwiotƺáły / y nogi ich nie nápuchły.
Y podáłeś im króleſtwá y národy / któreś rozegnáł po kątách / ták / że pośiedli ziemię Sehonową / y ziemię królá Heſebońſkiego / y ziemię Ogá / królá Báſáńſkiego.
A ſynów ich rozmnożyłeś jáko gwiázdy niebieſkie / y wwiodłeś ich do ziemi / o którejś máwiáł ojcom ich / że wnijdą y pośiędą ją.
Bo przyƺedłƺy ſynowie ich pośiedli tę ziemię / gdyś poniżył przed nimi obywáteli onej ziemi / Chánánejcżyków / y podáłeś ich w ręce ich / y królów ich / y národy onej ziemi / áby śię z nimi obchodzili według woli ſwojej.
Pobráli tedy miáſtá obronne / y ziemię tłuſtą / y pośiedli domy pełne wƺelkich dóbr / ſtudnie wykopáne / winnice / oliwnice / y drzew rodzájnych bárdzo wiele ; á jedli / y byli náſyceni / y otyli / y opływáli w rozkoƺy z dobroći twojej wielkiej.
Ale gdy ćię rozdráźnili / y ſtálić śię odpornymi / zárzućiwƺy zákon twój w tył ſwój / á proroków twoich pobili / którzy śię oświádcżáli przed nimi / áby ich náwróćili do ćiebie / y dopuƺcżáli śię bluźnierſtwá wielkiego :
Podáłeś ich w ręce nieprzyjáćiołom ich / którzy ich trápili. A gdy cżáſu utrápieniá ſwego wołáli do ćiebie / tyś ich z niebá wyſłucháł / á według litośći twoich wielkich dáwáłeś im wybáwićieli / którzy ich wybáwiáli z rąk nieprzyjáćiół ich.
Wtem / gdy trochę odpocżęli / znowu cżynili złość przed twárzą twoją ; przetoż opuśćiłeś ich w ręce nieprzyjáćiół ich / áby pánowáli nád nimi. Lecż gdy śię znowu náwróćili / á wołáli do ćiebie / tyś ich z niebá wyſłucháł / y wybáwiłeś ich według litośći twoich przez wiele cżáſów.
Y oświádcżáłeś śię przed nimi / ábyś ich náwróćił do zákonu ſwego ; ále oni ſobie hárdzie pocżynáli / á nieſłucháli przykázáń twoich / owƺem przećiw ſądom twoim grzeƺyli / które gdyby cżłowiek cżynił / żyłby przez nie ; ále uchylájąc rámion ſwych / kárk ſwój zátwárdzili / y nie ſłucháli.
Wƺákżeś ty im folgowáł przez wiele lát / oświádcżájąc śię przed nimi Duchem twym przez proroków twoich ; á gdy nie ſłucháli / podáłeś ich w ręce národów onych ziem.
Ale dlá litośći twoich wielkich nie dáłeś ich ná wytrácenie / ániś ich opuśćił ; boś ty Bóg łáſkáwy y miłośierny.
Teráz tedy o Boże náƺ / Boże wielki / możny / y ſtráƺny! który ſtrzeżeƺ przymierzá y miłośierdziá / niech nie będzie máłe przed tobą káżde utrápienie / które przyƺło ná náſ / ná królów náƺych / ná kśiążąt náƺych / y ná kápłánów náƺych / y ná proroków náƺych / y ná ojców náƺych / y ná wƺyſtek lud twój / ode dni królów Aſſyryjſkich áż do dniá tego ;
Acżeś ty jeſt ſpráwiedliwy we wƺyſtkiem tem / co przyƺło ná náſ ; boś ſpráwiedliwie ucżynił / á myśmy niezbożnie cżynili.
Y królowie náśi / kśiążętá náśi / kápłáni náśi / y ojcowie náśi nie pełnili zákonu twego / y nie przeſtrzegáli przykázáń twoich / y świádectw twych / któremiś śię oświádcżáł przed nimi.
Bo oni w króleſtwie ſwem y w dobroći twojej wielkiej / którąś im pokázáł / y w ziemi przeſtronnej y tłuſtej / którąś im był dáł / nie ſłużyli tobie / áni śię odwróćili od ſpráw złych ſwoich.
Oto myśmy dziś niewolnikámi / y ziemiá / którąś dáł ojcom náƺym / áby jedli owoc jej / y dobrá jej / otośmy niewolnikámi w niej.
Już urodzáje ſwoje obfite wydáje królom / któreś poſtánowił nád námi dlá grzechów náƺych ; pánują nád ćiáły náƺemi / y nád bydłem náƺem według woli ſwej / ták / żeśmy w wielkiem ućiśnieniu.
Wƺákże w tem wƺyſtkiem cżynimy mocne przymierze / y zápiſujemy je / które piecżętują kśiążętá náśi / Lewitowie náśi / y kápłáni náśi.
