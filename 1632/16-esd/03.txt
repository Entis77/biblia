Potem powſtał Elijáſyb / kápłán nájwyżƺy / y bráćia jego kápłáni / y budowáli bramę owcżą. Y zbudowáli ją / y przypráwili wrotá do niej ; áż do wieży Meá zbudowáli ją / y áż do wieży Chánáneel.
A podle niego budowáli mężowie z Jerychá / á podle nich budował Záchur / ſyn Immrego.
A bramę rybną budowáli ſynowie Senáá / którzy ją też przykryli y przypráwili wrotá do niej / y zámki jey / y záwory jey.
A podle nich popráwiał Meremot / ſyn Uryjáƺá / ſyná Koſowego ; á podle nich popráwiał Meſullám / ſyn Báráchyjáƺowy / ſyná Meſezábelowego ; á podle nich popráwiał Sádok / ſyn Báány.
Podle nich záśię popráwiáli Tekuitcżycy ; ále ći / co byli zácniejśi z nich / nie podłożyli ƺyi ſwej pod robotę páná ſwego.
A bramę ſtárą popráwiáli Jojádá / ſyn Fáſeáchowy / y Meſullám / ſyn Beſodyjáƺowy ; ći ją przykryli / y przypráwili wrotá do niej / y zámki jey / y záwory jey.
A podle nich popráwiał Melátyjáƺ Gábáonitcżyk / Jádon Meronitcżyk / mężowie z Gábáon y z Máſfá / áż do ſtolicy kśiążęcey / z tey ſtrony rzeki.
Podle nich popráwiał Huſyjel / ſyn Chárchájáƺowy / z złotnikámi ; á podle niego popráwiał Chánánijáƺ / ſyn áptekárſki ; á Jeruzálemu zániecháli áż do muru ƺerokiego.
A podle nich popráwiał Ráfájáƺ / ſyn Churowy / przełożony nád połową powiátu Jeruzálemſkiego.
A podle nich popráwiał Jedájáƺ / ſyn Hárumáfowy / y przećiw ſwemu domowi ; á podle niego popráwiał Háttus / ſyn Háſbonijáƺowy.
Cżęśći záś drugiej popráwiał Málchyjáƺ / ſyn Hárymowy / y Háſub / ſyn Páchátmoábowy / tákże y wieżę Tánnurym.
A podle niego popráwiał Sállum / ſyn Hállocheſowy / przełożony nád połową powiátu Jeruzálemſkiego / ſám y córki jego.
bramy nád doliną popráwiał Chánun / y obywátele Zánoe ; ćić ją budowáli / y przypráwili wrotá do niej / zámki jey / y záwory jey ; y ná tyśiąc łokći muru áż do bramy gnojowej.
bramy záś gnojowej popráwiał Melchyjáƺ / ſyn Recháby / przełożony nád powiátem Betcherem ; tenći ją zbudował / y przypráwił wrotá do niej / zámki jey / y záwory jey /
Dotego bramy źródłá popráwiał Sállon / ſyn Cholhozowy / przełożony nád powiátem Máſfá ; á ten ją zbudował / y przykrył ją / y przypráwił wrotá do niej / y zámki jey / y záwory jey / y mur nád ſtáwem Selách ku ogrodowi królewſkiemu áż do ſchodu / po którym ſchodzą z miáſtá Dawidowego.
Zátem popráwiał Nehemijáƺ / ſyn Háſbuká / przełożony nád połową powiátu Betſur / áż przećiwko grobom Dawidowym y áż do ſtáwu urobionego / y áż do domu mocárzów.
Zá nim popráwiáli Lewitowie Rehum / ſyn Báni ; podle niego popráwiał Háſábijáƺ / przełożony nád połową powiátu Ceile z powiátem ſwoim.
Zá nim záśię popráwiáli bráćia ich / Báwáj / ſyn Chenádádowy / przełożony nád połową powiátu Ceile.
A podle niego popráwiał Eſer / ſyn Jeſui / przełożony nád Máſfą / cżęśći drugiej przećiw miejſcu / kędy chodzą do zbrojowni názwáney Mikzoá.
Po nim wzruƺony gorliwośćią popráwiał Báruch / ſyn Zábbájowy / cżęśći drugiej od Mikzoá áż do drzwi domu Elijáſybá / nájwyżƺego kápłáná.
Zá nim popráwiał Meremot / ſyn Uryjáƺá / ſyn Koſowego / cżęśći drugiej / ode drzwi domu Elijáſybowego áż do końcá domu jego.
A zá nim popráwiáli kápłáni / którzy mieƺkáli w równinie.
Zá nimi popráwiał Benjámin y Háſub / przećiw domom ſwoim ; zá nimi popráwiał Azáryjáƺ / ſyn Mááſejáƺá / ſyná Anánijáƺowego / podle domu ſwego.
Zá nim popráwiał Bennui / ſyn Chenádádowy / cżęśći drugiej od domu Azáryjáƺowego áż do Mikzoá / y áż do rogu.
Pálál / ſyn Uzájego przećiw Mikzoá / y wieży wyſokiej / wywiedźioney z domu królewſkiego / która byłá w śieni więźieniá ; po nim popráwiał Fádájáƺ / ſyn Fároſowy.
A Netynejcżycy / co mieƺkáli w Ofel / popráwiáli áż ná przećiwko bramie wodney ná wſchód ſłońcá / y wieży wyſokiej.
Zá nimi popráwiáli Tekuitcżykowie drugą cżęść przećiw wieży wielkiej y wyſokiej áż do muru Ofel.
Od bramy końſkiej popráwiáli kápłáni / káżdy przećiw domowi ſwemu.
Zá nimi popráwiał Sádok / ſyn Immerowy / przećiw domowi ſwemu / á zá nim popráwiał Semejáƺ / ſyn Sechenijáƺowy / ſtróż bármy wſchodniej.
Zá nim popráwiał Chánánijáƺ / ſyn Selemijáƺowy / y Chánun / ſyn Sáláfowy ƺóſty / cżęśći drugiej ; zá nim popráwiał Meſullám / ſyn Berechyjáƺowy / przećiw gmáchowi ſwemu.
Zá nim popráwiał Málchijáƺ / ſyn złotnicży / áż do domu Netynejcżyków / y kupców / przećiw bramie ſądowej / y áż do ſáli nárożney.
A miedzy ſálą nárożną áż do bramy owcżey popráwiáli złotnicy y kupcy.
