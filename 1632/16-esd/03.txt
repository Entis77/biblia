Potem powſtáł Elijáſyb / kápłán nájwyżƺy / y bráćiá jego kápłáni / y budowáli brámę owcżą. Y zbudowáli ją / y przypráwili wrotá do niej ; áż do wieży Meá zbudowáli ją / y áż do wieży Chánáneel.
A podle niego budowáli mężowie z Jerychá / á podle nich budowáł Záchur / ſyn Immrego.
A brámę rybną budowáli ſynowie Senáá / którzy ją też przykryli y przypráwili wrotá do niej / y zámki jej / y záwory jej.
A podle nich popráwiáł Meremot / ſyn Uryjáƺá / ſyná Koſowego ; á podle nich popráwiáł Meſullám / ſyn Báráchyjáƺowy / ſyná Meſezábelowego ; á podle nich popráwiáł Sádok / ſyn Báány.
Podle nich záśię popráwiáli Tekuitcżycy ; ále ći / co byli zácniejśi z nich / nie podłożyli ƺyi ſwej pod robotę páná ſwego.
A brámę ſtárą popráwiáli Jojádá / ſyn Fáſeáchowy / y Meſullám / ſyn Beſodyjáƺowy ; ći ją przykryli / y przypráwili wrotá do niej / y zámki jej / y záwory jej.
A podle nich popráwiáł Melátyjáƺ Gábáonitcżyk / Jádon Meronitcżyk / mężowie z Gábáon y z Máſfá / áż do ſtolicy kśiążęcej / z tej ſtrony rzeki.
Podle nich popráwiáł Huſyjel / ſyn Chárchájáƺowy / z złotnikámi ; á podle niego popráwiáł Chánánijáƺ / ſyn áptekárſki ; á Jeruzálemu zániecháli áż do muru ƺerokiego.
A podle nich popráwiáł Ráfájáƺ / ſyn Churowy / przełożony nád połową powiátu Jeruzálemſkiego.
A podle nich popráwiáł Jedájáƺ / ſyn Hárumáfowy / y przećiw ſwemu domowi ; á podle niego popráwiáł Háttuſ / ſyn Háſbonijáƺowy.
Cżęśći záś drugiej popráwiáł Málchyjáƺ / ſyn Hárymowy / y Háſub / ſyn Páchátmoábowy / tákże y wieżę Tánnurym.
A podle niego popráwiáł Sállum / ſyn Hállocheſowy / przełożony nád połową powiátu Jeruzálemſkiego / ſám y córki jego.
Brámy nád doliną popráwiáł Chánun / y obywátele Zánoe ; ćić ją budowáli / y przypráwili wrotá do niej / zámki jej / y záwory jej ; y ná tyśiąc łokći muru áż do brámy gnojowej.
Brámy záś gnojowej popráwiáł Melchyjáƺ / ſyn Recháby / przełożony nád powiátem Betcherem ; tenći ją zbudowáł / y przypráwił wrotá do niej / zámki jej / y záwory jej /
Dotego brámy źródłá popráwiáł Sállon / ſyn Cholhozowy / przełożony nád powiátem Máſfá ; á ten ją zbudowáł / y przykrył ją / y przypráwił wrotá do niej / y zámki jej / y záwory jej / y mur nád ſtáwem Selách ku ogrodowi królewſkiemu áż do ſchodu / po którym ſchodzą z miáſtá Dawidowego.
Zátem popráwiáł Nehemijáƺ / ſyn Háſbuká / przełożony nád połową powiátu Betſur / áż przećiwko grobom Dawidowym y áż do ſtáwu urobionego / y áż do domu mocárzów.
Zá nim popráwiáli Lewitowie Rehum / ſyn Báni ; podle niego popráwiáł Háſábijáƺ / przełożony nád połową powiátu Ceile z powiátem ſwoim.
Zá nim záśię popráwiáli bráćiá ich / Báwáj / ſyn Chenádádowy / przełożony nád połową powiátu Ceile.
A podle niego popráwiáł Eſer / ſyn Jeſui / przełożony nád Máſfą / cżęśći drugiej przećiw miejſcu / kędy chodzą do zbrojowni názwánej Mikzoá.
Po nim wzruƺony gorliwośćią popráwiáł Báruch / ſyn Zábbájowy / cżęśći drugiej od Mikzoá áż do drzwi domu Elijáſybá / nájwyżƺego kápłáná.
Zá nim popráwiáł Meremot / ſyn Uryjáƺá / ſyn Koſowego / cżęśći drugiej / ode drzwi domu Elijáſybowego áż do końcá domu jego.
A zá nim popráwiáli kápłáni / którzy mieƺkáli w równinie.
Zá nimi popráwiáł Benjámin y Háſub / przećiw domom ſwoim ; zá nimi popráwiáł Azáryjáƺ / ſyn Mááſejáƺá / ſyná Anánijáƺowego / podle domu ſwego.
Zá nim popráwiáł Bennui / ſyn Chenádádowy / cżęśći drugiej od domu Azáryjáƺowego áż do Mikzoá / y áż do rogu.
Pálál / ſyn Uzájego przećiw Mikzoá / y wieży wyſokiej / wywiedzionej z domu królewſkiego / którá byłá w śieni więzieniá ; po nim popráwiáł Fádájáƺ / ſyn Fároſowy.
A Netynejcżycy / co mieƺkáli w Ofel / popráwiáli áż ná przećiwko brámie wodnej ná wſchód ſłońcá / y wieży wyſokiej.
Zá nimi popráwiáli Tekuitcżykowie drugą cżęść przećiw wieży wielkiej y wyſokiej áż do muru Ofel.
Od brámy końſkiej popráwiáli kápłáni / káżdy przećiw domowi ſwemu.
Zá nimi popráwiáł Sádok / ſyn Immerowy / przećiw domowi ſwemu / á zá nim popráwiáł Semejáƺ / ſyn Sechenijáƺowy / ſtróż bármy wſchodniej.
Zá nim popráwiáł Chánánijáƺ / ſyn Selemijáƺowy / y Chánun / ſyn Sáláfowy ƺóſty / cżęśći drugiej ; zá nim popráwiáł Meſullám / ſyn Berechyjáƺowy / przećiw gmáchowi ſwemu.
Zá nim popráwiáł Málchijáƺ / ſyn złotnicży / áż do domu Netynejcżyków / y kupców / przećiw brámie ſądowej / y áż do ſáli nárożnej.
A między ſálą nárożną áż do brámy owcżej popráwiáli złotnicy y kupcy.
