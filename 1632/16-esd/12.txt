A ćić ſą kápłáni y Lewitowie / którzy przyƺli z Zorobábelem / ſynem Seáltyjelowym / y z Jeſuą : Serejáƺ / Jeremijáƺ / Ezdráƺ /
Amáryjáƺ / Málluch / Háttuſ /
Sechánijáƺ / Rehum / Meremot /
Iddo / Ginnetoj / Abijáƺ ;
Mijámin / Máádyjáƺ / Bilgál /
Semejáƺ / y Jojáryb / Jedájáƺ /
Sállu / Amok / Helkijáƺ / Jedájáƺ. Cić byli przedniejśi z kápłánów y z bráći ſwych / zá dni Jeſuego.
A Lewitowie : Jeſuá / Binnui / Kádmiel / Serebijáƺ / Judá ; Mátánijáƺ nád pieśniámi / ſám y bráćiá jego.
A Bákbukijáƺ y Hunni / bráćiá ich / byli przećiwko nim w porządku ſwoim.
A Jeſuá ſpłodźił Joákimá / á Joákim ſpłodźił Elijáſybá / á Elijáſyb ſpłodźił Jojádę :
A Jojádá ſpłodźił Jonátáná / á Jonátán ſpłodźił Jádduę.
A zá dni Joákimá byli kápłáni przedniejśi z domów ojcowſkich : z Serájáƺowego Merájáƺ / z Jeremijáƺowego Chánánijáƺ ;
Z Ezdráƺowego Meſullám / z Amáryjáƺowego Jochánán ;
Z Melchowego Jonátán / z Sechánijáƺowego Józef ;
Z Hárymowego Adná / z Merájotowego Helkáj ;
Z Iddowego Zácháryjáƺ / z Ginnetowego Meſullám ;
Z Abijáƺowego Zychry / z Miniáminowego y z Máádyjáƺowego Piltáj ;
Z Bilgowego Sámmuá / z Semejáƺowego Jonátán ;
A z Jojárybowego Máttenáj / z Jedájáƺowego Uzy ;
Z Sállájowego Káláj / z Amokowego Heber ;
Z Helkijáƺowego Háſábijáƺ / z Jedájáƺowego Nátánáel.
A Lewitowie zá dni Elijáſybá / Jojády / y Jochánáná / y Jádduego popiſáni ſą / którzy byli przedniejƺymi z domów ojcowſkich ; tákże y kápłáni áż do króleſtwá Dáryjuƺá / królá Perſkiego.
Synowie mówię Lewiego / przedniejśi z domów ojcowſkich / zápiſáni ſą w kśięgách kroniki áż do dni Jochánáná / ſyná Elijáſybowego.
Przedniejśi mówię z Lewitów byli Háſábijáƺ / Serebijáƺ / y Jeſuá / ſyn Kádmielowy / y bráćiá ich przećiwko nim / ku chwáleniu y wyſłáwiániu Bogá według rozkázániá Dawidá / mężá Bożego / ſtráż przećiwko ſtráży.
Mátánijáƺ / y Bákbukijáƺ / Obádyjáƺ / Meſullám / Tálmon / Akkub / byli ſtróżámi odźwiernymi przy domu ſkárbu u brám.
Cić byli zá dni Joákimá / ſyná Jeſuego / ſyná Jozedekowego / y zá dni Nehemijáƺá wodzá / y Ezdráƺá kápłáná / náucżonego w Piśmie.
A przy poświęcániu muru Jeruzálemſkiego ƺukáno Lewitów ze wƺyſtkich miejſc ich / áby ich przywiedźiono do Jeruzálemu / żeby wykonáli poświęcániá y weſelá / á to z wyſłáwiániem y z śpiewániem / z cymbáłámi / z lutniámi / y z cytrámi.
Przetoż zgromádzeni ſą ſynowie śpiewáków / y z równin około Jeruzálemu / y ze wśi Netofátyckich.
Tákże z domu Gilgál / y z pól Giebá / y z Azmáwet ; bo ſobie śpiewácy budowáli wśi około Jeruzálemu.
A ocżyśćiwƺy śię kápłáni y Lewitowie / ocżyśćili też lud / y brámy / y mur.
Zátemem rozkázáł wſtąpić kśiążętom Judzkim ná mur / y poſtáwiłem dwá hufy wielkie chwálących / z których jedni ƺli ná práwo od wyżƺej ſtrony muru ku brámie gnojowej.
A zá nimi ƺedł Hozájáƺ / y połowá kśiążąt Judzkich ;
Tákże Azáryjáƺ / Ezdráƺ / y Meſullám /
Judá / y Benjámin / y Semejáƺ / y Jeremijáƺ.
Potem niektórzy z ſynów kápłáńſkich z trąbámi / miánowićie Zácháryjáƺ / ſyn Jonátáná / ſyná Semejáƺowego / ſyná Mátánijáƺowego / ſyná Michájáƺowego / ſyná Záchurowego / ſyná Aſáfowego ;
A bráćiá jego Semejáƺ / y Aſárel / Miláláj / Giláláj / Mááj / Netáneel / y Judá / Chánáni / z inſtrumentámi muzycżnemi Dawidá / mężá Bożego / á Ezdráƺ / náucżony w Piśmie / przed nimi.
Potem ku brámie źródłá / którá przećiwko nim byłá / wſtępowáli po ſchodách miáſtá Dawidowego / którędy chodzą ná mur / á od muru przy domu Dawidowym áż do brámy wodnej ná wſchód ſłońcá.
A drugi huf chwálących ƺedł przećiwko nim / á já zá nim / á połowá ludu po murze od wieży Tánnurym áż do muru ƺerokiego ;
A od brámy Efráim ku brámie ſtárej / y ku brámie rybnej / y wieży Chánáneel / y wieży Meá / áż do brámy owcżej. Y ſtánął u brámy ſtráży.
Potem ſtánęły one dwá hufy chwálących w domu Bożym / y já y połowá przełożonych ze mną.
Tákże kápłáni : Elijákim / Mááſejáƺ / Minijámin / Michájáƺ / Elijenáj / Zácháryjáƺ / Chánánijáƺ / z trąbámi ;
Y Máázyjáƺ / y Semejáƺ / y Eleázár / y Uzy / y Jochánán / y Málchyjáƺ / y Elám / y Ezer ; á śpiewácy głośno śpiewáli / y Izráchyjáƺ / przełożony ich.
Spráwowáli tákże onegoż dniá ofiáry wielkie / y weſelili śię ; ábowiem Bóg rozweſelił ich był weſelem wielkiem / ták / iż śię y niewiáſty y dźiátki weſeliły ; y było ſłyƺeć weſele Jeruzálemſkie dáleko.
Obráni też ſą dniá onego mężowie nád komorámi ſkárbów / y ofiár / pierwoćin / y dźieśięćin / áby zgromádzáli do nich z pól miejſkich dźiáły / zákonem wárowáne kápłánom y Lewitom ; bo śię weſelił Judá z kápłánów y z Lewitów tám ſtojących ;
Którzy ſtrzegli ſtráży Bogá ſwego / y ſtráży ocżyƺcżániá / y śpiewáków / y odźwiernych / według rozkázániá Dawidá y Sálomoná / ſyná jego.
Bo zá dni Dawidá y Aſáfá byli poſtánowieni z ſtárodáwná przełożeni nád śpiewákámi dlá śpiewániá / wychwálániá y dźiękcżynieniá Bogu.
Przetoż wƺyſtek Izráel zá dni Zorobábelá / y zá dni Nehemijáƺá / dáwáli dźiáły dlá śpiewáków y odźwiernych / káżdodźienny wymiár / á oddáwáli to / co poświęćili / Lewitom ; Lewitowie záś oddáwáli ſynom Aáronowym.
