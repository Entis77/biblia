Y przyƺedł jeden z śiedmiu Aniołów / którzy mieli śiedm cżáƺ / y rzekł do mnie / mówiąc mi : Chodź / okáżęć oſądzenie onej wielkiej wƺetecżnicy / którá śiedźi nád wodámi wielkiemi /
Z którą wƺetecżeńſtwo płodźili królowie źiemi / y upili śię winem wƺetecżeńſtwá jej obywátele źiemi.
Y odnióſł mię ná puƺcżę w duchu. Y widźiáłem niewiáſtę śiedzącą ná ƺárłátnocżerwonej beſtyi / pełnej imion bluźnierſtwá / którá miáłá śiedm głów y dźieśięć rogów.
A oná niewiáſtá przyoblecżoná byłá w purpurę y w ƺárłát / y uzłoconá złotem y drogim kámieniem / y perłámi / májąc kubek złoty w ręce ſwej / pełen obrzydliwośći y niecżyſtośći wƺetecżeńſtwá ſwego.
A ná cżole jej było imię nápiſáne : Tájemnicá / Bábilon wielki / mátká wƺetecżeńſtw y obrzydliwośći źiemi.
Y widźiáłem niewiáſtę onę pijáną krwią świętych y krwią męcżenników JEzuſowych ; á widząc ją / dźiwowáłem śię wielkiem podźiwieniem.
Y rzekł mi Anioł : Cżemuż śię dźiwujeƺ? Já tobie powiem tájemnicę tej niewiáſty / y beſtyi / którá ją nośi / którá má śiedm głów y dźieśięć rogów.
Beſtyjá / którąś widźiáł / byłá / á nie jeſt / á má wyſtąpić z przepáśći / á iść ná zginienie ; y zádźiwią śię mieƺkájący ná źiemi / ( których imioná nie ſą nápiſáne w kśięgách żywotá od záłożeniá świátá / ) widząc beſtyję / którá byłá / á nie jeſt / á przećię jeſt.
Tuć jeſt rozum májący mądrość : Te śiedm głów ſą śiedm gór / ná których tá niewiáſtá śiedźi.
A królów jeſt śiedm ; pięć ich upádło / á jeden jeſt / inƺy jeƺcże nie przyƺedł / á gdy przyjdźie / ná máły cżáſ muśi trwáć.
A beſtyjá / którá byłá á nie jeſt / toć jeſt ten óſmy / á jeſt z onych śiedmiu / á idźie ná zginienie.
A dźieśięć rogów / któreś widźiáł / jeſt dźieśięć królów / którzy króleſtwá jeƺcże nie wźięli ; ále wezmą moc / jáko królowie / ná jednę godźinę z beſtyją.
Ci jednę rádę máją y moc / y zwierzchność ſwoję beſtyi podádzą.
Ci z Báránkiem wálcżyć będą / y Báránek ich zwyćięży / bo jeſt Pánem pánów / y królem królów / y którzy ſą z nim / powołáni y wybráni y wierni.
Y rzekł mi : Wody / któreś widźiáł / gdźie wƺetecżnicá śiedźi / ſą ludźie / y záſtępy / y národy / y języki.
A dźieśięć rogów / któreś widźiáł ná beſtyi / ćić w nienáwiśći mieć będą wƺetecżnicę / y ucżynią ją ſpuſtoƺoną y nágą / y ćiáło jej będą jeść / á ſámą ogniem ſpálą.
Abowiem Bóg podáł do ſerc ich / áby cżynili wolę jego / á cżynili jednomyślnie / y dáli króleſtwo ſwoje beſtyi / áżby śię wypełniły ſłowá Boże.
A niewiáſtá / którąś widźiáł / jeſt miáſto ono wielkie / które má króleſtwo nád królámi źiemi.
