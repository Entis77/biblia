A Anjołowi Zboru który jeſt w Sárdźiech nápiƺ ; To mówi ten który ma śiedm Duchów Bożych / y śiedm gwiazd : Znam ucżynki twoje / y maƺ imię że żywieƺ ; áleś jeſt umárły.
Bądz cżujny / á utwierdzaj inne którzy umrzeć mieli. Abowiem nie ználazłem ucżynków twojich zupełnych przed Bogiem.
Pámiętaj tedy / jákoś wźiął y ſłyƺał / á chowaj y pokutuj. Jeſli tedy cżuć nie będźieƺ / przydę ná ćię jáko złodźiey ; á nie zrozumieƺ którey godźiny przydę ná ćię.
Ale maƺ niektóre oſoby w Sárdźiech / które nie pokaláły ƺat ſwojich ; przetoż chodźić będą ze mną w ƺátách białych / iż godni ſą.
Kto zwyćięży / ten będźie oblecżony w ƺáty białe : y nie wymáżę imienia jego z kśiąg żywotá / ále wyznam imię jego przed oblicżem Ojcá mojego / y przed Anjoły jego.
Kto ma uƺy niecháj ſłucha / co Duch mówi Zborom!
A Anjołowi Zboru Filádelfſkiego nápiƺ : To mówi on Święty y Práwdźiwy / który ma klucż Dawidow ; który otwiera á nikt nie záwiera / y záwiera á nikt nie otwiera.
Znam ucżynki twoje : Otom wyſtáwił przed tobą drzwi otworzone / á żaden nie może ich zámknąć : Bo ácż maƺ nie wielką moc / przećięś záchował ſłowo moje / y nie záprzałeś śię Imienia mego.
Otoć dam niektóre z bóżnice ƺátáńſkiey / którzy śię powiádáją być żydámi / á nie ſą / ále kłamáją. Oto ſpráwię że przydą / y będą śię kłaniáli przed nogámi twymi / y poznáją żem Ja ćiebie umiłował.
Żeś záchował ſłowo ćierpliwośći mojey ; Ja też ćiebie záchowam od godźiny pokuƺenia / które ma przyść ná wƺyſtek świát / áby doświadcżyło mieƺkájących ná źiemi.
Oto idę rychło : trzymaj co maƺ / áby nikt nie wźiął korony twojey.
Kto zwyćięży / ucżynię go filárem w Kośćiele Bogá mojego / á więcey z niego już nie wynidźie : y nápiƺę ná nim Imię Bogá mego / y imię miáſtá Bogá mego nowego Jeruzalemu / które zſtępuje z niebá od Bogá mego / y Imię moje nowe.
Kto ma uƺy niechaj ſłucha / co Duch mówi Zborom!
A Anjołowi Zboru Láodyceńſkiego nápiƺ ; To mówi Amen / Świádek on wierny y prawdźiwy / pocżątek ſtworzenia Bożego.
Znam ucżynki twoje / żeś nie jeſt áni źimny / áni gorący : bodajżeś był źimny álbo gorący.
A ták ponieważeś letny / á áni źimny áni gorący / wyrzucę ćię z uſt mojich.
Abowiem mówiƺ ; Jeſt bogáty y zbogáćiałem / á nicżego nie potrzebuję : á nie wieƺ żeś ty biedny y mizerny / y ubogi / y ślepy / y nági.
Rádzęć ábyś kupił u mnie złotá w ogniu doświadcżonego / ábyś był bogáty : y ƺáty białe / ábyś był oblecżony / á żeby śię nie okázowáłá ſromotá nágośći twojey : á ocży twoje námáż máśćią wzrok náprawiájącą / ábyś widźiał.
Ja którekolwiek miłuję / ſtrofuję y karzę. Bądź tedy gorliwym / á pokutuj.
Oto ſtoję u drzwi / y kołácę : jeſliby kto uſłyƺał głos mój y otworzył drzwi / wnidę do niego y będę z nim wiecżerzał / á on ze mną.
Kto zwyćięży / dam mu śiedźieć z ſobą ná ſtolicy mojey ; jákom y Ja zwyćiężył / y uśiadłem z Ojcem mojim ná ſtolicy jego.
Kto ma uƺy niechaj ſłucha / co Duch mówi Zborom!
