A Aniołowi zboru / który jeſt w Sárdziech / nápiƺ : To mówi ten / który má śiedm duchów Bożych y śiedm gwiázd : Znám ucżynki twoje / y máƺ imię / że żyjeƺ ; áleś jeſt umárły.
Bądź cżujny / á utwierdzáj innych / którzy umrzeć máją ; ábowiem nie ználázłem ucżynków twoich zupełnych przed Bogiem.
Pámiętáj tedy / jákoś wziął y ſłyƺáł / á chowáj y pokutuj. Jeźli tedy cżuć nie będzieƺ / przyjdę ná ćię jáko złodziej / á nie zrozumieƺ / której godziny przyjdę ná ćię.
Ale máƺ niektóre oſoby w Sárdziech / które nie pokáláły ƺát ſwoich ; przetoż chodzić będą ze mną w ƺátách biáłych / iż godni ſą.
Kto zwyćięży / ten będzie oblecżony w ƺáty biáłe / y nie wymáżę imieniá jego z kśiąg żywotá / ále wyznám imię jego przed oblicżem Ojcá mojego y przed Aniołámi jego.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom.
A Aniołowi zboru Filádelfſkiego nápiƺ : To mówi on Święty y Práwdziwy / który má klucż Dawidowy / który otwierá á nikt nie záwierá / y záwierá á nikt nie otwierá.
Znám ucżynki twoje ; otom wyſtáwił przed tobą drzwi otworzone / á żáden nie może ich zámknąć ; bo ácż máƺ niewielką moc / przećięś záchowáł ſłowo moje / y nie záprzáłeś śię imieniá mego.
Otoć dám niektórych z bóżnicy ƺátáńſkiej / którzy śię powiádáją być Żydámi / á nie ſą / ále kłámią. Oto ſpráwię / że przyjdą y będą śię kłániáli przed nogámi twemi / y poznáją / żem já ćię umiłowáł.
Żeś záchowáł ſłowo ćierpliwośći mojej / já też ćię záchowám od godziny pokuƺeniá / którá przyjdzie ná wƺyſtek świát / áby doświádcżyłá mieƺkájących ná ziemi ;
Oto idę rychło ; trzymáj / co máƺ / áby nikt nie wziął korony twojej.
Kto zwyćięży / ucżynię go filárem w kośćiele Bogá mojego / á więcej z niego już nie wynijdzie ; y nápiƺę ná nim imię Bogá mego / y imię miáſtá Bogá mego / nowego Jeruzálemu / które zſtępuje z niebá od Bogá mego / y imię moje nowe.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom.
A Aniołowi zboru Láodyceńſkiego nápiƺ : To mówi Amen / świádek on wierny y práwdziwy / pocżątek ſtworzeniá Bożego :
Znám ucżynki twoje / żeś nie jeſt áni zimny áni gorący / bodájżeś był zimny / álbo gorący!
A ták / poniewáżeś letni / á áni zimny áni gorący / wyrzucę ćię z uſt moich.
Abowiem mówiƺ : Jeſtem bogáty y zbogáćiłem śię / á nicżego nie potrzebuję ; á nie wieƺ / żeś ty biedny y mizerny / y ubogi / y ślepy / y nági.
Rádzęć / ábyś kupił u mnie złotá w ogniu doświádcżonego / ábyś był bogáty / y ƺáty biáłe / ábyś był oblecżony / á żeby śię nie okázywáłá ſromotá nágośći twojej ; á ocży twoje námáż máśćią wzrok nápráwiájącą / ábyś widziáł.
Já którychkolwiek miłuję / ſtrofuję y kárzę. Bądź tedy gorliwym / á pokutuj.
Oto ſtoję u drzwi y kołácżę ; jeźliby kto uſłyƺáł głoſ mój / y otworzył drzwi / wnijdę do niego / y będę z nim wiecżerzáł / á on ze mną.
Kto zwyćięży / dám mu śiedzieć z ſobą ná ſtolicy mojej / jákom y já zwyćiężył / y uśiádłem z Ojcem moim ná ſtolicy jego.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom.
