Potymem widźiał po prawey ręce śiedzącego ná ſtolicy kśięgi nápiſáne / wewnątrz y zewnątrz / zápiecżętowáne śiedmią piecżęći.
Y widźiałem Anjołá mocnego wołájącego głoſem wielkim ; Kto jeſt godźien otworzyć te kśięgi / y odpiecżętowáć piecżęći ich?
A nikt nie mógł áni w niebie / áni ná źiemi / áni pod źiemią / otworzyć onych Kśiąg / áni wejrzeć w nie.
Y płákałem bárzo yż nikt nie był ználeźiony godny / áby otworzył y cżytáł kśięgi / y wejrżáł w nie.
Tedy mi jeden z onych ſtárców rzekł ; Nie płácż. Oto zwyćiężył Lew / który jeſt z pokolenia Judowego / korzeń Dawidow ; áby otworżył Kśięgi / y odpiecżętowáł śiedm piecżęći ich.
Y pojrzáłem ; á oto miedzy ſtolicą / y cżteremá onymi zwierzęty / y miedzy onymi ſtárcámi / Báránek ſtał jáko zábity / májąc śiedm rogew / y śiedm ocży : które ſą śiedm Duchów Bożych / poſłánych ná wƺyſtkę źiemię.
Ten przyƺedł ; y wźiął one kśięgi z práwey ręki śiedzącego ná ſtolicy.
A gdy wźiął one kśięgi / záráz ono cżworo zwierząt / y oni dwádźieśćiá y cżterej ſtárcy / upádli przed Báránkiem / májąc káżdy z nich cytry y cżáƺe złote / pełne wonnych rzecży : które ſą modlitwy świętych.
Y śpiewáli nową Pieśń / mówiąc ; Godźieneś jeſt wźiąć kśięgi / y otworzyć piecżęći ich ; żeś był zábity / y odkupiłeś nas Bogu przez krew ſwoję / ze wƺelkiego pokoleniá / y języká / y ludu / y národu :
Y ucżyniłeś nas Bogu náƺemu Królmi y Kápłany / y królowáć będźiemy ná źiemi.
Y widźiałem y ſłyƺałem głoſ wiela Anjołów / około oney ſtolice / y onych zwierząt / y onych ſtárców : á byłá licżbá ich / tyśiąckroć ſto tyśięcy / y dźieśięć kroc ſto tyśięcy /
Mówiących głoſem wielkim ; Godźien jeſt ten Báránek zábity wźiąć moc / y bogáctwo / y mądrość / y śiłę y cżeść / y chwałę / y błogoſłáwieńſtwo.
A wƺelkie ſtworzenie / które jeſt ná niebie / y ná źiemi / y pod źiemią / y w morzu / y wƺyſtko co w nich jeſt / ſłyƺałem mówiące ; Siedzącemu ná ſtolicy / y Báránkowi / błogoſłáwieńſtwo / y cżęść / y chwáłá / y śiłá ná wieki wieków.
A cżworo onych zwierząt rzekło / Amen. A oni dwádźieśćiá y cżterej ſtárcy upádli / y kłaniáli śię żywiącemu ná wieki wieków.
