A potymem widźiał drugiego Anjołá zſtępującego z niebá / májącego moc wielką : y oświećiłá śię źiemiá od chwały jego.
Y záwołał potężnie głoſem wielkim mówiąc ; Upadł / upadł Bábilon on wielki / y ſtał śię przybytkiem / cżártów / y mieƺkániem wƺelkiego duchá niecżyſtego ; y mieƺkániem wƺelkiego ptáſtwá niecżyſtego y przemierzłego.
Iż z win zápálcżywośći wƺetecżeńſtwá jego piły wƺyſtkie narody : á królowie źiemie wƺetecżeńſtwo z nim płodźili ; y kupcy źiemſcy z zbytecżney rozkoƺy jego zbogáćieli.
Y ſłyƺałem inƺy głos z niebá mówiący ; Wynidźćie z niego ludu mój / ábyśćie nie byli ucżeſnikámi grzechów jego / á iżbyśćie nie wźięli z plag jego.
Abowiem dośięgły grzechy jego áż do niebá / y wſpomniał Bóg ná niepráwośći jego.
Oddajćież mu jáko y on dawał wam ; á w dwój naſób oddajćie mu według ucżynków jego : w kubku który wam nálewał / nálejćie mu w dwój naſób.
Jáko śię wiele chlubił y rozkoƺował / ták mu wiele dajćie mąk y ſmętku : bo mówi w ſercu ſwojim ; Siedzę jako królowa / á nie jeſtem wdową / y ſmętku nie ujrzę.
Przetoż w jeden dźień przydą plagi jego / śmierć / y ſmętek / y głód / y ogniem będźie ſpalony : bo mocny jeſt PAn Bóg który go oſądźi.
Y będą go płákáć / y nárzekáć nád nim będą królowie źiemie / którzy z nim wƺetecżeńſtwo płodźili / y roſkoƺowáli / gdy ujrzą dym zápalenia jego /
Z dáleká ſtojąc dla bojáźni męki jego y mówiąc ; Biádá / biádá / miáſto ono wielkie Bábilon / miáſto ono mocne ; iż w jednę godźinę przyƺedł ſąd twój.
Do tego y kupcy źiemſcy płákáć będą / y nárzekáć nád nim : przeto iż towáru ich żaden więcey kupowáć nie będźie :
Towáru złotá y śrebrá / y kámienia drogiego / y pereł / y lnu ćienkiego / y purpury / y jedwábiu / y ƺárłatu / y wƺelkiego drzewá Tyinowego / y wƺelkiego nacżynia ſłoniowego / y wƺelkiego nacżynia z drzewá nakoƺtowniejƺego / y z miedźi / y z żelázá / y z mármoru :
Y cynámonu / y kádzenia / y máśći / y kádźidłá / y winá / y oliwy / y mąki cżyſtey / y pƺenice / y bydłá / y owiec / y koni / y wozów / y niewolników / y duƺ ludzkich.
Y owoce pożądliwośći duƺe twojey odeƺły od ćiebie / y wƺyſtkie rzecży tłuſte y świetne odeƺły od ćiebie : á tych rzecży już więcey nie znajdźieƺ.
A ći kupcy którzy tym kupcżyli y tym śię zbogáćili / z dáleká ſtać będą dla bojáźni męki jego / płácżąc y nárzekájąc ;
A mówiąc : Biádá / biádá ; miáſto ono wielkie / które było oblecżone w biśiór / y w purpurę / y w ƺárłat / y uzłocone złotem / y kámieniem drogim / y perłámi : gdyż w jedney godźinie zginęło ták wielkie bogáctwo.
Y wƺelki ſternik / y wƺyſtko mnóſtwo ludu które jeſt ná okręćie / y żeglarze / y którzykolwiek ná morzu pożytku ƺukáją / z dáleká ſtáną.
Y záwołáją widząc dym zápalenia jego / mówiąc ; Któreż miáſto było podobne temu miáſtu wielkiemu?
A miecąc proch ná głowy ſwoje / wołáć będą płácżąc / y ſmęcąc śię y mówiąc : Biádá / biádá ; miáſto ono wielkie / w którym zbogátnieli wƺyſcy / którzy mieli okręty ná morzu z doſtátków jego ; iż jedney godźiny ſpuſtoƺáło.
Rozráduj śię nád nim niebo / y święći Apoſtołowie / y Prorocy : bo śię pomśćił krzywdy wáƺey Bóg nád nim.
Y podnióſł jeden Anjoł mocny kámień jákoby młyńſki wielki / y wrzućił go w morze mówiąc : Tákim pędem wrzucony będźie Bábilon / miáſto ono wielkie ; y już więcey nie będźie ználeźiony.
Y głos cytryſtów / y śpiewaków / y piƺcżków / y trębácżów więcey w tobie ſłyƺány nie będźie ; y żáden rzemieśnik wƺelkiego rzemioſłá nie znajdźie śię więcey w tobie ; y grzmot młyná nie będźie więcey ſłyƺány w tobie :
Y świátłość świece nie będźie śię więcey świećiłá w tobie : y głos oblubieńcá y oblubienice nie będźie więcey ſłyƺány w tobie : iż kupcy twoji byli wielcy pánowie źiemſcy / iż cżárámi twojimi byli zwiedźieni wƺyſcy narodowie.
Y w nim ználázłá śię krew Proroków y świętych / y wƺyſtkich którzy ſą pobići ná źiemi.
