Y widźiáłem / gdy otworzył Báránek jednę z onych piecżęći / y ſłyƺáłem jedno ze cżterech zwierząt mówiące / jáko głoſ gromu : Chodź á pátrzáj!
Y widźiáłem / á oto koń biáły / á ten / który ná nim śiedźiáł / miáł łuk / y dáno mu koronę / y wyƺedł jáko zwyćiężcá / áżeby zwyćiężáł.
A gdy otworzył wtórą piecżęć / ſłyƺáłem wtóre zwierzę mówiące : Chodź á pátrzáj!
Y wyƺedł drugi koń rydzy ; á temu / który ná nim śiedźiáł / dáno / áby odjął pokój z źiemi / á iżby jedni drugich zábijáli / y dáno mu miecż wielki.
A gdy otworzył trzećią piecżęć / ſłyƺáłem trzećie zwierzę mówiące : Chodź á pátrzáj! Y widźiáłem / á oto koń wrony / á ten / co ná nim śiedźiáł / miáł ƺálę w ręce ſwojej.
Y ſłyƺáłem głoſ z pośrodku onych cżworgá zwierząt mówiący : Miárká pƺenicy zá groƺ / á trzy miárki jęcżmieniá zá groƺ ; á nie ƺkodź oliwie y winu.
A gdy otworzył cżwártą piecżęć / ſłyƺáłem głoſ cżwártego zwierzęćiá mówiący : Chodź á pátrzáj!
Y widźiáłem / á oto koń płowy / á tego / który śiedźiáł ná nim / imię było śmierć / á piekło ƺło zá nim ; y dáná im jeſt moc nád cżwártą cżęśćią źiemi / áby zábijáli miecżem / y głodem / y morem / y przez zwierzętá źiemſkie.
A gdy otworzył piątą piecżęć / widźiáłem pod ołtárzem duƺe pobitych dlá ſłowá Bożego y dlá świádectwá / które wydáwáli ;
Y wołáli głoſem wielkim / mówiąc : Dokądże / Pánie święty y práwdźiwy! nie ſądźiƺ y nie mśćiƺ śię krwi náƺej nád tymi / którzy mieƺkáją ná źiemi?
Y dáne ſą káżdemu z nich ƺáty biáłe / y powiedźiáno im / áby odpocżywáli jeƺcże ná máły cżáſ / áżby śię dopełnił pocżet ſpółſług ich y bráći ich / którzy máją być pobići / jáko y oni.
Y widźiáłem / gdy otworzył ƺóſtą piecżęć / á oto ſtáło śię wielkie trzęśienie źiemi / á ſłońce zcżerniáło jáko wór włośiány / y kśiężyc wƺyſtek ſtáł śię jáko krew ;
A gwiázdy niebieſkie pádáły ná źiemię / ták jáko drzewo figowe zrzucá z śiebie figi ſwoje niedoſtáłe / gdy od wiátru wielkiego bywá záchwiáne.
A niebo uſtąpiło jáko kśięgi zwinione / á wƺelká górá y wyſpy z miejſcá śię ſwego poruƺyły ;
A królowie źiemi / y kśiążętá / y bogácże / y hetmáni / y mocárze / y káżdy niewolnik / y káżdy wolny pokryli śię w jáſkinie y w ſkáły gór /
Y rzekli górom y ſkáłom : Upádnijćie ná náſ / y zákryjćie náſ przed oblicżem tego / który śiedźi ná ſtolicy / y przed gniewem tego Báránká ;
Abowiem przyƺedł dźień on wielki gniewu jego / y któż śię oſtáć może?
