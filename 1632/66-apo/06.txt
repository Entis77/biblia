Y widziáłem / gdy otworzył Báránek jednę z onych piecżęći / y ſłyƺáłem jedno ze cżterech zwierząt mówiące / jáko głoſ gromu : Chodź á pátrzáj!
Y widziáłem / á oto koń biáły / á ten / który ná nim śiedziáł / miáł łuk / y dáno mu koronę / y wyƺedł jáko zwyćiężcá / áżeby zwyćiężáł.
A gdy otworzył wtórą piecżęć / ſłyƺáłem wtóre zwierzę mówiące : Chodź á pátrzáj!
Y wyƺedł drugi koń rydzy ; á temu / który ná nim śiedziáł / dáno / áby odjął pokój z ziemi / á iżby jedni drugich zábijáli / y dáno mu miecż wielki.
A gdy otworzył trzećią piecżęć / ſłyƺáłem trzećie zwierzę mówiące : Chodź á pátrzáj! Y widziáłem / á oto koń wrony / á ten / co ná nim śiedziáł / miáł ƺálę w ręce ſwojej.
Y ſłyƺáłem głoſ z pośrodku onych cżworgá zwierząt mówiący : Miárká pƺenicy zá groƺ / á trzy miárki jęcżmieniá zá groƺ ; á nie ƺkodź oliwie y winu.
A gdy otworzył cżwártą piecżęć / ſłyƺáłem głoſ cżwártego zwierzęćiá mówiący : Chodź á pátrzáj!
Y widziáłem / á oto koń płowy / á tego / który śiedziáł ná nim / imię było śmierć / á piekło ƺło zá nim ; y dáná im jeſt moc nád cżwártą cżęśćią ziemi / áby zábijáli miecżem / y głodem / y morem / y przez zwierzętá ziemſkie.
A gdy otworzył piątą piecżęć / widziáłem pod ołtárzem duƺe pobitych dlá ſłowá Bożego y dlá świádectwá / które wydáwáli ;
Y wołáli głoſem wielkim / mówiąc : Dokądże / Pánie święty y práwdziwy! nie ſądziƺ y nie mśćiƺ śię krwi náƺej nád tymi / którzy mieƺkáją ná ziemi?
Y dáne ſą káżdemu z nich ƺáty biáłe / y powiedziáno im / áby odpocżywáli jeƺcże ná máły cżáſ / áżby śię dopełnił pocżet ſpółſług ich y bráći ich / którzy máją być pobići / jáko y oni.
Y widziáłem / gdy otworzył ƺóſtą piecżęć / á oto ſtáło śię wielkie trzęśienie ziemi / á ſłońce zcżerniáło jáko wór włośiány / y kśiężyc wƺyſtek ſtáł śię jáko krew ;
A gwiázdy niebieſkie pádáły ná ziemię / ták jáko drzewo figowe zrzucá z śiebie figi ſwoje niedoſtáłe / gdy od wiátru wielkiego bywá záchwiáne.
A niebo uſtąpiło jáko kśięgi zwinione / á wƺelká górá y wyſpy z miejſcá śię ſwego poruƺyły ;
A królowie ziemi / y kśiążętá / y bogácże / y hetmáni / y mocárze / y káżdy niewolnik / y káżdy wolny pokryli śię w jáſkinie y w ſkáły gór /
Y rzekli górom y ſkáłom : Upádnijćie ná náſ / y zákryjćie náſ przed oblicżem tego / który śiedzi ná ſtolicy / y przed gniewem tego Báránká ;
Abowiem przyƺedł dzień on wielki gniewu jego / y któż śię oſtáć może?
