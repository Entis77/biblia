Potymem widźiał cżterech Anjołów ſtojących ná cżterech węgłách źiemie / trzymájących cżtery wiátry źiemie ; áby nie wiał wiátr ná źiemię / áni ná morze / áni ná żadne drzewo.
Y widźiałem inƺego Anjołá wyſtępującego od Wſchodu ſłońcá / májącego piecżęć Bogá Żywego / który záwołał głoſem wielkim ná onych cżterech Anjołów / którym dano áby ƺkodźili źiemi y morzu /
Mówiąc ; Nie ƺkodźćie źiemi / áni morzu / áni drzewom ; áż popiecżętujemy ſługi Bogá náƺego ná cżołách ich.
Y uſłyƺáłem licżbę popiecżętowánych : których ſto cżterdźieśći y cżtery tyśiące popiecżętowano / ze wƺyſtkich pokoleń ſynów Izráelſkich.
Z pokolenia Judowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Rubenowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Gádowego / dwánaśćie tyśięcy popiecżętowánych.
Z pokolenia Aſerowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Neftálimowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Mánáſſeſſowego / dwánaśćie tyśięcy popiecżętowánych.
Z pokolenia Symeonowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Lewiego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Iſáſchárowego / dwánaśćie tyśięcy popiecżętowánych.
Z pokolenia Zábulonowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Józefowego / dwánaśćie tyśięcy popiecżętowánych. Z pokolenia Benjáminowego / dwánaśćie tyśięcy popiecżętowánych.
Potymem widźiał ; á oto lud wielki / którego nie mógł nikt zlicżyć z káżdego narodu y pokolenia / y ludźi / y języków : którzy ſtali przed ſtolicą / y przed oblicżnośćią Báránká / oblecżeni w ƺáty białe / á pálmy były w ręku ich.
Y wołáli głoſem wielkim / mówiąc ; Zbáwienie należy Bogu náƺemu / y śiedzącemu ná ſtolicy / y Báránkowi.
A wƺyſcy Anjołowie ſtali około ſtolice / y ſtárców / y cżworgá zwierząt ; y upádli przed ſtolicą ná oblicże ſwoje / y kłaniáli śię Bogu /
Mówiąc ; Amen / Błogoſłáwieńſtwo / y chwałá / y mądrość / y dźiękowánie / y cżęść / y moc / y śiłá Bogu náƺemu / ná wieki wieków / Amen.
Y odpowiedźiał jeden z onych ſtárców / y rzekł mi : Ci którzy ſą oblecżeni w ƺáty białe / co zácż ſą? Y z kąd przyƺli?
Y rzekłem mu ; Pánie ty wieƺ. Y rzekł mi ; Cić ſą / którzy przyƺli z ućiſku wielkiego ; y omyli ƺáty ſwoje / y wybielili je we krwi Báránkowey.
Dla tego ſą przed ſtolicą Bożą / y ſłużą mu we dnie y w nocy w Kośćiele jego : á ten który śiedźi ná ſtolicy / jáko námiotem záſłoni je.
Nie będą więcey łáknąć / y nie będą więcej prágnąć / y nie uderzy ná nie ſłońce / áni żadne gorąco.
Abowiem Báránek który jeſt w pośrodku ſtolice / będźie je páſł / y poprowádźi je do żywych źrżódeł wód : y otrze Bóg wƺelką łzę z ocżu ich.
