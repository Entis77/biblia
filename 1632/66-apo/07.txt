Potemem widziáł cżterech Aniołów / ſtojących ná cżterech węgłách ziemi / trzymájących cżtery wiátry ziemi / áby nie wiáł wiátr ná ziemię / áni ná morze / áni ná żádne drzewo.
Y widziáłem inƺego Aniołá wyſtępującego od wſchodu ſłońcá / májącego piecżęć Bogá żywego / y záwołáł głoſem wielkim ná onych cżterech Aniołów / którym dáno / áby ƺkodzili ziemi y morzu ;
Mówiąc : Nie ƺkodźćie ziemi áni morzu / áni drzewom / áż popiecżętujemy ſługi Bogá náƺego ná cżołách ich.
Y uſłyƺáłem licżbę popiecżętowánych : ſto cżterdzieśći y cżtery tyśiące jeſt popiecżętowánych ze wƺyſtkich pokoleń ſynów Izráelſkich :
Z pokoleniá Judowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Rubenowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Gádowego dwánáśćie tyśięcy popiecżętowánych ;
Z pokoleniá Aſerowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Neftálimowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Mánáſeſowego dwánáśćie tyśięcy popiecżętowánych ;
Z pokoleniá Symeonowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Lewiego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Iſáſchárowego dwánáśćie tyśięcy popiecżętowánych ;
Z pokoleniá Zábulonowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Józefowego dwánáśćie tyśięcy popiecżętowánych ; z pokoleniá Benjáminowego dwánáśćie tyśięcy popiecżętowánych.
Potemem widziáł / á oto lud wielki / którego nie mógł nikt zlicżyć / z káżdego národu y pokoleniá / y ludzi / y języków / którzy ſtáli przed ſtolicą y przed oblicżnośćią Báránká / oblecżeni w ƺáty biáłe / á pálmy były w rękách ich.
Y wołáli głoſem wielkim / mówiąc : Zbáwienie náleży Bogu náƺemu / śiedzącemu ná ſtolicy / y Báránkowi.
A wƺyſcy Aniołowie ſtáli około ſtolicy y ſtárców y cżworgá zwierząt / y upádli przed ſtolicą ná oblicże ſwoje / y kłániáli śię Bogu /
Mówiąc : Amen! Błogoſłáwieńſtwo / y chwáłá / y mądrość / y dziękowánie / y cżeść / y moc / y śiłá Bogu náƺemu ná wieki wieków. Amen.
Y odpowiedziáł jeden z onych ſtárców y rzekł mi : Ci / którzy ſą oblecżeni w ƺáty biáłe / co zácż ſą? y ſkąd przyƺli?
Y rzekłem mu : Pánie! ty wieƺ. Y rzekł mi : Cić ſą / którzy przyƺli z ućiſku wielkiego / y omyli ƺáty ſwoje / y wybielili je we krwi Báránkowej.
Dlátego ſą przed ſtolicą Bożą / y ſłużą mu we dnie y w nocy w kośćiele jego / á ten / który śiedzi ná ſtolicy / jáko námiotem záſłoni ich.
Nie będą więcej łáknąć / y nie będą więcej prágnąć / y nie uderzy ná nich ſłońce / áni żádne gorąco.
Abowiem Báránek / który jeſt w pośrodku ſtolicy / będzie ich páſł / y poprowádzi ich do żywych źródeł wód / y otrze Bóg wƺelką łzę z ocżów ich.
