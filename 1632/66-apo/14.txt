Y widźiáłem / á oto Báránek ſtáł ná górze Syońſkiej / á z nim ſto cżterdźieśći y cżtery tyśiące / májących imię Ojcá jego nápiſáne ná cżołách ſwoich.
Y ſłyƺáłem głoſ z niebá / jáko głoſ wielu wód / y jáko głoſ gromu wielkiego ; y ſłyƺáłem głoſ cytryſtów grájących ná cytrách ſwoich.
A śpiewáli / jákoby nową pieśń / przed ſtolicą / y przed onem cżworgiem zwierząt / y przed ſtárcámi / á żáden śię nie mógł onej pieśni náucżyć / oprócż onych ſtu cżterdźieſtu y cżterech tyśięcy / którzy ſą z źiemi kupieni.
Cić ſą / którzy śię z niewiáſtámi nie pokáláli ; bo pánnámi ſą. Ci ſą / którzy náśládują Báránká / gdźiekolwiek idźie. Ci kupieni ſą z ludźi / áby byli pierwiáſtkámi Bogu y Báránkowi.
A w uſtách ich nie ználázłá śię zdrádá ; ábowiem ſą bez zmázy przed ſtolicą Bożą.
Y widźiáłem drugiego Aniołá / lecącego przez pośrodek niebá / májącego Ewángieliję wiecżną / áby ją zwiáſtowáł mieƺkájącym ná źiemi / y wƺelkiemu národowi y pokoleniu y językowi y ludowi /
Mówiącego głoſem wielkim : Bójćie śię Bogá / y chwáłę mu dájćie / gdyż przyƺłá godźiná ſądu jego / á kłániájćie śię temu / który ucżynił niebo y źiemię / y morze / y źródłá wód.
A zá nim ƺedł drugi Anioł / mówiąc : Upádł Bábilon / ono miáſto wielkie! bo winem gniewu wƺetecżeńſtwá ſwego nápoił wƺyſtkie národy.
A trzeći Anioł ƺedł zá nimi / mówiąc głoſem wielkim : Jeźli śię kto pokłoni beſtyi y obrázowi jej / y jeźli weźmie piątno ná cżoło ſwoje álbo ná rękę ſwoję /
Y ten pić będźie z winá gniewu Bożego / z winá ƺcżerego y nálánego w kielich gniewu jego / y będźie męcżony w ogniu y śiárce przed oblicżnośćią Aniołów świętych / y przed oblicżnośćią Báránká.
A dym męki ich wyſtępuje ná wieki wieków / y nie máją odpocżynku we dnie y w nocy / którzy śię kłániáją beſtyi y obrázowi jej / y jeźli kto bierze piątno imieniá jej.
Tuć jeſt ćierpliwość świętych / tuć ſą ći / którzy chowáją przykázániá Boże y wiárę JEzuſową.
Y uſłyƺáłem głoſ z niebá / mówiący do mnie : Nápiƺ : Błogoſłáwieni ſą odtąd umárli / którzy w Pánu umieráją. Zápráwdę mówi Duch im / áby odpocżywáli od prác ſwoich / á ucżynki ich idą zá nimi.
Y widźiáłem / á oto obłok biáły ; á ná onym obłoku śiedźiáł podobny Synowi cżłowiecżemu / który miáł ná głowie ſwojej koronę złotą á w ręce ſwojej śierp oſtry.
A drugi Anioł wyƺedł z kośćiołá / wołájąc głoſem wielkim ná tego / który śiedźiáł ná obłoku : Zápuść śierp twój á żnij / gdyż tobie przyƺłá godźiná / ábyś żął / poniewáż śię doſtáło żniwo źiemi.
Y zápuśćił ten / który śiedźiáł ná obłoku / śierp ſwój ná źiemię / y pożętá jeſt źiemiá.
A drugi Anioł wyƺedł z kośćiołá onego / który jeſt w niebie / májąc y ten śierp oſtry.
Potem wyƺedł drugi Anioł z ołtárzá / który miáł moc nád ogniem / y záwołáł głoſem wielkim ná tego / który miáł śierp oſtry / mówiąc : Zápuść ten śierp twój oſtry / á zbieráj groná winnicy źiemi ; bo dojrzáłe ſą jágody jej.
Zápuśćił tedy Anioł śierp ſwój oſtry ná źiemię / y zebráł groná winnicy źiemi / y wrzućił je w práſę wielką gniewu Bożego.
Y tłocżoná jeſt práſá przed miáſtem / y wyƺłá krew z práſy áż do wędźideł końſkich przez tyśiąc y ƺeść ſet ſtáján.
