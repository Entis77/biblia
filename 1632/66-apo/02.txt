Aniołowi zboru Efeſkiego nápiƺ : To mówi ten / który trzymá śiedm gwiázd w práwej ręce ſwojej / który śię przechádzá w pośród onych śiedmiu świecżników złotych :
Znám ucżynki twoje / y prácę twoję / y ćierpliwość twoję / á iż nie możeƺ ćierpieć złych / y doświádcżyłeś tych / którzy śię mienią być Apoſtołámi / á nie ſą / y ználázłeś ich / że ſą kłámcámi ;
Y znáƺáłeś / y máƺ ćierpliwość / y prácowáłeś dlá imieniá mego / á nie uſtáłeś.
Ale mám nieco przećiwko tobie / żeś miłość twoję pierwƺą opuśćił.
Pámiętájże tedy / ſkądeś wypádł / á pokutuj / y cżyń ucżynki pierwƺe ; á jeźli nie chceƺ / przyjdę przećiwko tobie rychło / á poruƺę świecżnik twój z miejſcá ſwego / jeźlibyś nie pokutowáł.
Ale wżdy to máƺ / iż nienáwidziƺ ucżynków Nikoláitów / których y já nienáwidzę.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom : Temu / co zwyćięży / dám jeść z drzewá żywotá / które jeſt w pośrodku ráju Bożego.
A Aniołowi zboru Smyrneńſkiego nápiƺ : To mówi pierwƺy y oſtátni / który był umárł y ożył :
Znám ucżynki twoje / y ućiſk / y ubóſtwo / ( áleś bogáty / ) y bluźnierſtwo tych / którzy śię powiádáją być Żydámi / á nie ſą / ále ſą bóżnicą ƺátáńſką.
Nic śię nie bój tego / co máƺ ćierpieć. Oto wrzući dyjábeł niektórych z wáſ do więzieniá / ábyśćie byli doświádcżeni ; y będziećie mieli ućiſk przez dzieśięć dni. Bądź wierny áż do śmierći / á dám ći koronę żywotá.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom : Kto zwyćięży / nie będzie obráżony od wtórej śmierći.
A Aniołowi zboru Pergámeńſkiego nápiƺ : To mówi ten / który má miecż on z obydwóch ſtron oſtry ;
Znám ucżynki twoje / y gdzie mieƺkáƺ / to jeſt / gdzie jeſt ſtolicá ƺátáńſká / á iż trzymáƺ imię moje / y nie záprzáłeś śię wiáry mojej y w one dni / w które Antypáſ / świádek mój wierny / zábity jeſt u wáſ / gdzie ƺátán mieƺká.
Ale mám nieco przećiwko tobie / iż tám máƺ tych / którzy trzymáją náukę Báláámową / który ucżył Báláká / áby wrzućił zgorƺenie przed ſyny Izráelſkie / żeby jedli rzecży báłwánom ofiárowáne / y wƺetecżeńſtwo płodzili.
Tákże máƺ y tych / którzy trzymáją náukę Nikoláitów / co já mám w nienáwiśći.
Pokutujże : á jeźli nie będzieƺ / przyjdę przećiwko tobie w rychle / y będę wálcżył z nimi miecżem uſt moich.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom : Temu / co zwyćięży / dám jeść z onej mánny ſkrytej / y dám mu kámyk biáły / á ná onym kámyku imię nowe nápiſáne / którego nikt nie zná / tylko ten / który je przyjmuje.
A Aniołowi zboru Tyjátyrſkiego nápiƺ : To mówi ſyn Boży / który má ocży ſwoje jáko płomień ogniá / á nogi jego podobne ſą mośiądzowi :
Znám ucżynki twoje / y miłość y poſługi / y wiárę / y ćierpliwość twoję / y ucżynki twoje / á że oſtátnich rzecży więcej jeſt / niż pierwƺych.
Ale mám nieco przećiwko tobie / iż niewieśćie Jezábeli / którá śię mieni być prorokinią / dopuƺcżáƺ ucżyć y zwodzić ſługi moje / żeby wƺetecżeńſtwo płodzili / y rzecży báłwánom ofiárowáne jedli.
Y dáłem jej cżáſ / áby pokutowáłá z wƺetecżeńſtwá ſwego ; ále nie pokutowáłá.
Oto já porzucę ją ná łoże / y tych / którzy z nią cudzołożą / w ućiſk wielki / jeźliby nie pokutowáli z ucżynków ſwoich :
A dzieći jej pobiję ná śmierć ; y poznáją wƺyſtkie zbory / żem já jeſt ten / który śię bádám nerek y ſerc ; y dám káżdemu z wáſ według ucżynków wáƺych.
A wám mówię / y drugim / którzyśćie w Tyjátyrzech / którzykolwiek nie máją tej náuki / y którzy nie poználi głębokośći ƺátáńſkich / jáko mówią : Nie włożę ná wáſ innego brzemieniá.
Wƺákże to / co máćie / trzymájćie / áż przyjdę.
A kto zwyćięży y záchowá áż do końcá ucżynki moje / dám mu zwierzchność nád pogánámi.
Y będzie ich rządził láſką żelázną ; jáko ſtátki gárncárſkie ſkruƺeni będą / jákom y já wziął od Ojcá mego.
Y dám mu gwiázdę poránną.
Kto má uƺy / niecháj ſłuchá / co Duch mówi zborom.
