Stárƺy Gájowi miłemu / którego já miłuję w práwdzie.
Nájmilƺy! nájprzód żądám / ábyć śię dobrze powodziło y ábyś był zdrów / ták jáko śię dobrze powodzi duƺy twojej.
Abowiem wielcem śię urádowáł / gdy przyƺli bráćiá / y dáli świádectwo o twojej práwdzie / jáko ty w práwdzie chodziƺ.
Więkƺej nád tę rádośći nie mám / jáko gdy ſłyƺę / iż dziátki moje chodzą w ƺcżerośći.
Nájmilƺy! wiernie cżyniƺ / cokolwiek cżyniƺ przećiwko bráćiom y przećiw gośćiom /
Którzy świádectwo wydáli o miłośći twojej przed zborem ; y dobrze ucżyniƺ / jeźli ich odprowádziƺ / jáko przyſtoi przed Bogiem.
Abowiem dlá imieniá jego wyƺli / nic nie wziąwƺy od pogán.
My tedy tákowych powinniśmy przyjmowáć / ábyśmy byli pomocnikámi práwdzie.
Piſáłem do zboru wáƺego ; ále Dyjotrefeſ / który chce być przedniejƺy między nimi / nie przyjmuje náſ.
Przeto jeźli przyjdę / przypomnę ucżynki jego / które cżyni / ſłowámi złemi obmáwiájąc náſ / á nie májąc doſyć ná tem / y ſám bráći nie przyjmuje / y tym / coby przyjąć chćieli / zábrániá / y ze zboru ich wyłącżá.
Nájmilƺy! nie náśláduj złego / ále dobrego. Kto dobrze cżyni / z Bogá jeſt ; ále kto źle cżyni / nie widziáł Bogá.
Demetryjuƺowi świádectwo jeſt dáne od wƺyſtkich / y od ſámej práwdy ; lecż y my świádectwo o nim dájemy / á wiećie / iż świádectwo náƺe práwdziwe jeſt.
Wielem miáł piſáć ; lecż nie chcę piſáć inkáuſtem y piórem ;
Bo mám nádzieję / że ćię w rychle ujrzę / á tedy uſtnie mówić będziemy.
Pokój tobie. Pozdráwiáją ćię przyjáćiele. Pozdrów y ty przyjáćiół z imieniá.
