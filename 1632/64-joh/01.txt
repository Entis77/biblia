Stárƺy Gájowi miłemu / którego já miłuję w práwdźie.
Nájmilƺy! nájprzód żądám / ábyć śię dobrze powodźiło y ábyś był zdrów / ták jáko śię dobrze powodźi duƺy twojej.
Abowiem wielcem śię urádowáł / gdy przyƺli bráćiá / y dáli świádectwo o twojej práwdźie / jáko ty w práwdźie chodźiƺ.
Więkƺej nád tę rádośći nie mám / jáko gdy ſłyƺę / iż dźiátki moje chodzą w ƺcżerośći.
Nájmilƺy! wiernie cżyniƺ / cokolwiek cżyniƺ przećiwko bráćiom y przećiw gośćiom /
Którzy świádectwo wydáli o miłośći twojej przed zborem ; y dobrze ucżyniƺ / jeźli ich odprowádźiƺ / jáko przyſtoi przed Bogiem.
Abowiem dlá imieniá jego wyƺli / nic nie wźiąwƺy od pogán.
My tedy tákowych powinniśmy przyjmowáć / ábyśmy byli pomocnikámi práwdźie.
Piſáłem do zboru wáƺego ; ále Dyjotrefeſ / który chce być przedniejƺy między nimi / nie przyjmuje náſ.
Przeto jeźli przyjdę / przypomnę ucżynki jego / które cżyni / ſłowámi złemi obmáwiájąc náſ / á nie májąc doſyć ná tem / y ſám bráći nie przyjmuje / y tym / coby przyjąć chćieli / zábrániá / y ze zboru ich wyłącżá.
Nájmilƺy! nie náśláduj złego / ále dobrego. Kto dobrze cżyni / z Bogá jeſt ; ále kto źle cżyni / nie widźiáł Bogá.
Demetryjuƺowi świádectwo jeſt dáne od wƺyſtkich / y od ſámej práwdy ; lecż y my świádectwo o nim dájemy / á wiećie / iż świádectwo náƺe práwdźiwe jeſt.
Wielem miáł piſáć ; lecż nie chcę piſáć inkáuſtem y piórem ;
Bo mám nádźieję / że ćię w rychle ujrzę / á tedy uſtnie mówić będźiemy.
Pokój tobie. Pozdráwiáją ćię przyjáćiele. Pozdrów y ty przyjáćiół z imieniá.
