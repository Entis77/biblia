Ale śię ſtánie w oſtátecżne dni / że utwierdzoná będźie górá domu Páńſkiego ná wierzchu gór / y wywyżƺoná nád págórki / á národy śię do niej zbiegáć będą.
Y pójdźie wiele národów / mówiąc : Pójdźćie / á wſtąpmy ná górę Páńſką / to jeſt do domu Bogá Jákóbowego / á będźie nas náucżał dróg ſwojich / y będźiemy chodźili śćieƺkámi jego ; bo zakon z Syonu wyjdźie / á ſłowo Páńſkie z Jeruzálemu.
Onći ſądźić będźie miedzy wielomá národámi / á káráć będźie národy mocne ná długie cżáſy ; y przekują miecże ſwe ná lemieƺe / á oƺcżepy ſwe ná koſy : nie podnieśie miecżá náród przećiwko národowi / áni śię więcey ćwicżyć będą do boju ;
Ale káżdy będźie śiedźiał pod winną máćicą ſwoją / y pod figowem drzewem ſwojim / á nie będźie nikt / coby ich przeſtráƺył ; bo to uſtá Páná zaſtępów mówiły.
Bo wƺyſtkie národy chodźić będą / káżdy w imieniu bogá ſwego ; ále my chodźić będźiemy w imieniu Páná / Bogá náƺego / ná wieki wieków.
Dniá onego / mówi Pán / zgromádzę chromą y wygnáną zbiorę / y onę / którejm był źle ucżynił ;
A dam tey chromej potomſtwo / á precż wygnáney náród możny ; y będźie Pán królował nád nimi ná górze Syon odtąd áż ná wieki.
A ty / wieżo trzody / báƺto córki Syońſkiej! wiedz / że áż do ćiebie przydźie ; przydźie mówię / pierwƺe páńſtwo / y króleſtwo córki Jeruzálemſkiej.
Cżemuż teraz ták bárzo krzycżyƺ? Izali królá niemáƺ w tobie? Izali rádcá twój zginął / że ćię zjął ból / jáko rodzącą?
Bolej á ſtękáj / córko Syońſká? jáko rodzącá ; bo już wyjdźieƺ z miáſtá / y będźieƺ mieƺkáłá w polu / á przyjdźieƺ áż do Bábilonu ; tám będźieƺ wybáwioná / tám ćię Pán odkupi z rąk nieprzyjáćiół twojich.
A choć śię teraz zebráło przećiwko tobie wiele národów / które mówią : Niech będźie ſplugáwiony Syon / á niech ná to pátrzy oko náƺe ;
Wƺákże one nie znáją myśli Páńſkich / áni rozumieją rády jego / iż je zgromádzá jáko ſnopy ná bojewiſko.
Wſtáńże / á młóć / córko Syońſká! bo róg twój ucżynię żelázny / á kopytá twoje ucżynię miedźiáne / y pokruƺyƺ wiele národów ; y poświęcę Pánu korzyśći ich / á májętnośći ich Pánu wƺyſtkiej źiemi.
