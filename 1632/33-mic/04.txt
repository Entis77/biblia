Ale śię ſtánie w oſtátecżne dni / że utwierdzoná będzie górá domu Páńſkiego ná wierzchu gór / y wywyżƺoná nád págórki / á národy śię do niej zbiegáć będą.
Y pójdzie wiele národów / mówiąc : Pójdźćie / á wſtąpmy ná górę Páńſką / to jeſt do domu Bogá Jákóbowego / á będzie náſ náucżáł dróg ſwoich / y będziemy chodzili śćieƺkámi jego ; bo zákon z Syonu wyjdzie / á ſłowo Páńſkie z Jeruzálemu.
Onći ſądzić będzie między wielomá národámi / á káráć będzie národy mocne ná długie cżáſy ; y przekują miecże ſwe ná lemieƺe / á oƺcżepy ſwe ná koſy : nie podnieśie miecżá náród przećiwko národowi / áni śię więcej ćwicżyć będą do boju ;
Ale káżdy będzie śiedziáł pod winną máćicą ſwoją / y pod figowem drzewem ſwojim / á nie będzie nikt / coby ich przeſtráƺył ; bo to uſtá Páná záſtępów mówiły.
Bo wƺyſtkie národy chodzić będą / káżdy w imieniu bogá ſwego ; ále my chodzić będziemy w imieniu Páná / Bogá náƺego / ná wieki wieków.
Dniá onego / mówi Pán / zgromádzę chromą y wygnáną zbiorę / y onę / którejm był źle ucżynił ;
A dám tej chromej potomſtwo / á precż wygnánej náród możny ; y będzie Pán królowáł nád nimi ná górze Syon odtąd áż ná wieki.
A ty / wieżo trzody / báƺto córki Syońſkiej! wiedz / że áż do ćiebie przyjdzie ; przyjdzie mówię / pierwƺe páńſtwo / y króleſtwo córki Jeruzálemſkiej.
Cżemuż teráz ták bárdzo krzycżyƺ? Izáli królá niemáƺ w tobie? Izáli rádcá twój zginął / że ćię zjął ból / jáko rodzącą?
Bolej á ſtękáj / córko Syońſká? jáko rodzącá ; bo już wyjdzieƺ z miáſtá / y będzieƺ mieƺkáłá w polu / á przyjdzieƺ áż do Bábilonu ; tám będzieƺ wybáwioná / tám ćię Pán odkupi z rąk nieprzyjáćiół twoich.
A choć śię teráz zebráło przećiwko tobie wiele národów / które mówią : Niech będzie ſplugáwiony Syon / á niech ná to pátrzy oko náƺe ;
Wƺákże one nie znáją myśli Páńſkich / áni rozumieją rády jego / iż je zgromádzá jáko ſnopy ná bojewiſko.
Wſtáńże / á młóć / córko Syońſká! bo róg twój ucżynię żelázny / á kopytá twoje ucżynię miedziáne / y pokruƺyƺ wiele národów ; y poświęcę Pánu korzyśći ich / á májętnośći ich Pánu wƺyſtkiej ziemi.
