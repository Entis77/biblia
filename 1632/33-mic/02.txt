Biádá tym / którzy wymyśláją niepráwość / y knują złe ná łożách ſwoich / á ná świtániu ráno do ſkutku je przywodzą / gdyż to jeſt w mocy rąk ich.
Pożądáją pól / y wydzieráją ; tákże y domów / y odejmują ; á ták przewodzą gwáłt nád mężem y domem jego / nád káżdym mężem y nád dziedzictwem jego.
Dlátego ták mówi Pán : Oto Já złe myślę przećiwko rodzájowi temu / z którego nie wyjmiećie ƺyj wáƺych / áni będziećie chodzić pyƺnie ; bo cżáſ zły będzie.
W on dzień urośnie o wáſ przypowieść / y nárzekáć będą nád wámi płácżem żáłośnym / mówiąc : Spuſtoƺeniśmy do ƺcżętu / Pán odmienił dziáł ludu mojego ; o jákoć mi go odjął! jákoć wziąwƺy pole náƺe rozdzielił!
Dlátego nie będzieƺ miáł / ktoby rzućił ƺnurem ná loſ w zgromádzeniu Páńſkim.
Mówią : Nie prorokujćie / niech nám inni prorokują ; bo nie prorokują ták jáko ći ; żáden z nich nie przeſtáje mów zelżywych.
O ty ludu / który ſłynieƺ domem Jákóbowym! izáli ukrócony má być duch Páńſki? Izáli tákowe ſą ſpráwy jego? Azáż ſłowá moje nie ſą dobre temu / który chodzi w uprzejmośći?
Wcżoráj był ludem moim / á dziś jáko nieprzyjáćiel powſtáje ; májąc odzienie / zdzieráćie płáƺcż z tych / którzy chodzą bezpiecżnie / którzy śię wrácáją z wojny ;
Żony ludu mego wygániáćie z domu rozkoƺy ich / od dziátek ich uſtáwicżnie odejmujećie ſłáwę moję.
Wſtáńćie / á odejdźćie / bo tu nie máƺ odpocżnieniá ; pogubi wáſ dlá niecżyſtośći / á to pogubieniem ſrogiem.
Gdy śię kto zá proroká udáje / á kłámſtwo opowiádájąc mówi : Będęć prorokowáł o winie álbo o nápoju mocnym : tákowyć bywá miłym prorokiem ludu tego.
Zgromádzájąc cále ćię zgromádzę / Jákóbie! zgromádzájąc zgromádzę oſtátki Izráelá / á ſpędzę je w gromádę jáko owce Bocrá / jáko trzodę w pośród obory jego / y wyjdzie huk od ludu.
Zſtąpi ten / który przełámywáć będzie przed nimi ; przełámie / á przejdą brámą / y wnijdą przez nią ; náwet y król ich pójdzie przed nimi / á Pán ná cżele ich.
