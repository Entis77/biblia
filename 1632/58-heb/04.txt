Bójmyż śię tedy / áby ſnáć zániedbáwƺy obietnicy o wejśćiu do odpocżnieniá jego / nie zdáł śię kto z wáſ być upośledzony.
Abowiem y nám zwiáſtowáná jeſt Ewángielijá jáko y onym ; ále im nie pomogło ſłowo / które ſłyƺeli / przeto iż nie było złącżone z wiárą tych / którzy ſłyƺeli.
Abowiem wnijdźiemy do odpocżnieniá / którzyśmy uwierzyli / jáko powiedźiáł : Przetożem przyśiągł w gniewie moim / że nie wnijdą do odpocżnieniá mojego ; choć dokonáne ſą dźiełá Boże od záłożeniá świátá.
Abowiem ták powiedźiáł ná jednem miejſcu o śiódmym dniu : Y odpocżął Bóg dniá śiódmego od wƺyſtkich ſpráw ſwoich.
A tu záśię : Że nie wnijdą do odpocżnieniá mego.
Poniewáż tedy to zoſtáje / że niektórzy wchodzą do niego / á ći / którym pierwej zwiáſtowáno / nie weƺli dlá niedowiárſtwá /
Záśię náznácżá dźień niektóry : Dźiś / mówiąc przez Dáwidá po ták długim cżáśie ( jáko powiedźiáno ) Dźiś / jeźlibyśćie głoſ jego uſłyƺeli / nie zátwárdzájćież ſerc wáƺych.
Abowiem jeźliby im Jozue odpocżnienie ſpráwił / nie mówiłby był potem o inƺym dniu.
A ták zoſtáje jeƺcże odpocżnienie ludowi Bożemu.
Abowiem ktobykolwiek wƺedł do odpocżnieniá jego / y on tákże odpocżął od ſpráw ſwoich / jáko y Bóg od ſwoich.
Stárájmyż śię tedy / ábyśmy weƺli do onego odpocżnieniá / żeby kto nie wpádł w tenże przykłád niedowiárſtwá.
Boć żywe jeſt ſłowo Boże / y ſkutecżne / y przeráźliwƺe nád wƺelki miecż po obu ſtronách oſtry / y przenikájące áż do rozdźieleniá y duƺy y duchá / y ſtáwów / y śpików / y rozeznáwájące myśli y zdániá ſerdecżne.
A niemáƺ żádnego ſtworzeniá / któreby nie było jáwne przed oblicżem jego ; owƺem wƺyſtkie rzecży obnáżone ſą y odkryte ocżom tego / o którym mówimy.
Przetoż májąc nájwyżƺego kápłáná wielkiego / który przeniknął niebioſá / JEzuſá / Syná Bożego / trzymájmyż śię tego wyznániá.
Abowiem nie mámy nájwyżƺego kápłáná / któryby nie mógł z námi ćierpieć krewkośći náƺych / lecż ſkuƺonego we wƺyſtkiem ná podobieńſtwo náſ / oprócż grzechu.
Przyſtąpmyż tedy z ufnośćią do tronu łáſki / ábyśmy doſtąpili miłośierdźiá / y łáſkę ználeźli ku pomocy cżáſu przygodnego.
