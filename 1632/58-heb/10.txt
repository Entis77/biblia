Abowiem zakon májąc ćień przyƺłych dóbr / <i>á</i> nie ſam obraz rzecży / jednákimiż ofiárámi / które ná káżdy rok uſtáwicżnie ofiárują / nigdy nie może tych / którzy do nich przyſtępują / doſkonáłymi ucżynić.
Bo inácżey / przeſtanoby ich było ofiárowáć : Przeto / żeby już nie mieli żadnego ſumnienia o grzechy ći którzy ofiárują / będąc raz ocżyśćieni.
Ale przy tych <i>ofiárách</i> dźieje śię przypominánie grzechów ná káżdy rok.
Abowiem nie można rzecż áby krew wołów y kozłów miáłá głádźić grzechy.
Przetoż wchodząc ná świát / mówi : Ofiáry y objáty nie chćiałeś ; áleś mi ćiáło ſpoſobił.
Cáłopalenia y ofiáry zá grzech / nie upodobáłyć śię.
Tedym rzekł : Oto idę : ( Ná pocżątku kśiąg nápiſano o mnie ) ábym cżynił o Boże wolą twoję.
Powiedźiawƺy wyżey ; Żeś ofiáry / y objáty / y cáłopalenia / y <i>ofiáry</i> zá grzech nie chćiał / áni ſobie upodobał / ( które według Zakonu bywáją ofiárowáne. )
Tedy rzekł : Oto idę / ábym cżynił o Boże wolą twoję ; Znośi pierwƺe / áby wtóre poſtánowił.
Przez którą wolą jeſteſmy poświęceni / przez ofiárę ćiáłá JEzuſá CHryſtuſá raz <i>ucżynioną</i>.
A wƺelkić kápłan ſtoji ná káżdy dźień ſłużbę Bożą odpráwując / á jednákież ofiáry cżęſtokroć ofiárując / które nigdy grzechów zgłádźić nie mogą.
Lecż ten jednę ofiárę ofiárowawƺy zá grzechy / ná wieki śiedźi ná práwicy Bożey :
Ná koniec ocżekawájąc / áżby położeni byli nieprzyjaćiele jego podnóżkiem nóg jego.
Abowiem jedną ofiárą doſkonáłymi ucżynił ná wieki te / którzy bywáją poświęceni.
A świadcży nam <i>to</i> y ſam Duch Święty. Abowiem powiedźiawƺy pierwey ;
Toć jeſt przymierze które poſtánowię z nimi po onych dniách / mówi PAn : Dam práwá moje do ſercá ich / á ná myślách ich nápiƺę je :
A grzechów ich y niepráwośći ich / nie wſpomnię więcey.
A gdźieć jeſt odpuƺcżenie ich / jużći więcey ofiáry nie <i>potrzebá</i> zá grzech.
Májąc tedy / bráćia / wolność wniść do Świątnice przez krew JEzuſowę ;
( Drogą nową y żywą / którą nam poświęćił przez zaſłonę ; to jeſt / przez ćiáło ſwoje. )
Y Kápłaná wielkiego nád domem Bożym :
Przyſtąpmyż / z prawdźiwym ſercem / w zupełnośći wiáry / májąc ocżyśćione ſercá od ſumnienia złego :
Y omyte ćiáło wodą cżyſtą / trzymajmy wyznánie nádźieje niechwiejące śię : ( Boć wierny jeſt ten który obiecał. )
Y przypátrujmy śię jedni drugim ku pobudzániu śię do miłośći / y do dobrych ucżynków.
Nie opuƺcżájąc ſpołecżnego zgromádzenia náƺego / jáko niektórzy obycżaj máją : Ale nápominájąc <i>jedni drugich,</i> á to tym więcey / cżym <i>więcey</i> widźićie że śię on dźień przybliża.
Abowiem jeſlibyſmy dobrowolnie grzeƺyli po wźięćiu znájomośći prawdy / nie zoſtawáłáby już ofiárá zá grzechy :
Ale ſtráƺliwe niejákie ocżekawánie ſądu / y zápalenie ogniá który pożrzeć ma przećiwniki.
Ktoby odrzućił Zakon Mojzeƺów / bez miłośierdźia zá świádectwem dwu álbo trzech umiera.
Co śię wam zda? Jákoż ſrożƺego karánia godźien jeſt ten / ktoby Syná Bożego podeptał ; y krew przymierza / przez którą był poświęcony / zá poſpolitą miał / y Duchá łáſki zelżył?
Abowiem znamy tego który powiedźiał : Mnie pomſtá / Ja oddam / mówi PAn. Y záśię : PAn ſądźić będźie lud ſwój.
Stráƺnać rzecż jeſt wpáść w ręce Bogá żywego.
Wſpomnićie ná dni pierwƺe / w których będąc oświeceni / znośiliśćie wielki bój utrapienia :
Lubo to gdyśćie byli y urągániem y utrapieniem ná podźiw wyſtáwieni ; lub też gdyśćie śię ſtáli ucżeſnikámi tych z którymi śię ták obchodzono.
Abowiemeśćie y z więźienia mego ze mną utrapieni byli / y rozchwycenie májętnośći wáƺych przyjęliśćie z rádośćią ; wiedząc że maćie w ſobie lepƺą májętność w niebie / y trwájącą.
Przetoż nie odrzucajćie dufnośći wáƺey / która ma wielką zapłátę.
Abowiem ćierpliwośći wam potrzebá / ábyśćie wolą Bożą cżyniąc / odnieśli obietnicę.
Boć jeƺcże bárzo bárzo málucżko / á <i>oto</i> ten który ma przyść / przydźie / á nie omięƺka.
A ſpráwiedliwy / z wiáry żyć będźie : á jeſliby śię <i>kto</i> zchraniał / nie kocha śię w nim duƺá mojá.
Lecż my nie jeſteſmy z tych którzy śię zchraniáją ku zginieniu : ále z tych którzy wierzą ku pozyſkániu duƺe.
