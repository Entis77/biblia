Abowiem zákon májąc ćień przyƺłych dóbr / á nie ſám obráz rzecży / jednákiemiż ofiárámi / które ná káżdy rok uſtáwicżnie ofiárują / nigdy nie może tych / którzy do nich przyſtępują / doſkonáłymi ucżynić.
Bo inácżej przeſtánoby ich było ofiárowáć / przeto / żeby już nie mieli żádnego ſumieniá o grzechy ći / którzy ofiárują / będąc ráz ocżyƺcżeni.
Ale przy tych ofiárách dźieje śię przypomnienie grzechów ná káżdy rok.
Abowiem nie możná rzec / áby krew wołów y kozłów miáłá głádźić grzechy.
Przetoż wchodząc ná świát / mówi : Ofiáry y obiáty nie chćiáłeś / áleś mi ćiáło ſpoſobił.
Cáłopáleniá y ofiáry zá grzech nie upodobáłyć śię.
Tedym rzekł : Oto idę ( ná pocżątku kśięgi nápiſáno o mnie / ) ábym cżynił / o Boże! wolę twoję ;
Powiedźiáwƺy wyżej : Żeś ofiáry y obiáty y cáłopáleniá y ofiáry zá grzech nie chćiáł / áni ſobie upodobáł / ( które według zákonu bywáją ofiárowáne. )
Tedy rzekł : Oto idę / ábym cżynił / o Boże! wolę twoję ; znośi pierwƺe / áby wtóre poſtánowił.
Przez którą wolę jeſteśmy poświęceni przez ofiárę ćiáłá JEzuſá CHryſtuſá ráz ucżynioną.
A wƺelkić kápłán ſtoi ná káżdy dźień / ſłużbę Bożą odpráwując / á jednákież ofiáry cżęſtokroć ofiárując / które nigdy grzechów zgłádźić nie mogą.
Lecż ten jednę ofiárę ofiárowáwƺy zá grzechy / ná wieki śiedźi ná práwicy Bożej /
Ná koniec ocżekując / áżby położeni byli nieprzyjáćiele jego podnóżkiem nóg jego.
Abowiem jedną ofiárą doſkonáłymi ucżynił ná wieki tych / którzy bywáją poświęceni.
A świádcży nám to y ſám Duch Święty ; ábowiem powiedźiáwƺy pierwej :
Toć jeſt przymierze / które poſtánowię z nimi po onych dniách / mówi Pán : Dám práwá moje do ſercá ich / á ná myślách ich nápiƺę je.
A grzechów ich y niepráwośći ich nie wſpomnę więcej :
A gdźieć jeſt odpuƺcżenie ich / jużći więcej ofiáry nie potrzebá zá grzech.
Májąc tedy / bráćiá! wolność / wnijść do świątnicy przez krew JEzuſową /
( Drogą nową y żywą / którą nám poświęćił przez záſłonę / to jeſt / przez ćiáło ſwoje. )
Y kápłáná wielkiego nád domem Bożym ;
Przyſtąpmyż z práwdźiwem ſercem w zupełnośći wiáry / májąc ocżyƺcżone ſercá od ſumieniá złego /
Y omyte ćiáło wodą cżyſtą / trzymájmy wyznánie nádźiei niechwiejące śię ; ( boć wierny jeſt ten / który obiecáł ; )
Y przypátrujmy śię jedni drugim ku pobudzániu śię do miłośći y do dobrych ucżynków /
Nie opuƺcżájąc ſpołecżnego zgromádzeniá náƺego / jáko niektórzy obycżáj máją / ále nápominájąc jedni drugich / á to tem więcej / cżem więcej widźićie / iż śię on dźień przybliżá.
Abowiem jeźlibyśmy dobrowolnie grzeƺyli po wźięćiu znájomośći práwdy / nie zoſtáwáłáby już ofiárá zá grzechy ;
Ale ſtráƺliwe niejákie ocżekiwánie ſądu / y zápálenie ogniá / który pożreć má przećiwników.
Ktoby odrzućił zákon Mojżeƺowy / bez miłośierdźiá zá świádectwem dwóch álbo trzech umierá.
Co śię wám zdá? Jákoż ſrożƺego kárániá godźien jeſt ten / kto by Syná Bożego podeptáł / y krew przymierzá / przez którą był poświęcony / zá poſpolitą miáł / y Duchá łáſki zelżył?
Abowiem známy tego / który powiedźiáł : Mnie pomſtá / Já oddám / mówi Pán : y záśię : Pán ſądźić będźie lud ſwój.
Stráƺnáć rzecż jeſt / wpáść w ręce Bogá żywego.
Wſpomnijćie ná dni pierwƺe / w których będąc oświeceni / znośiliśćie wielki bój utrápieniá /
Lubo to / gdyśćie byli y urągániem y utrápieniem ná podźiw wyſtáwieni / lub też gdyśćie śię ſtáli ucżeſnikámi tych / z którymi śię ták obchodzono.
Abowiemeśćie y z więźieniá mego zemną utrápieni byli / y rozchwycenie májętnośći wáƺych przyjęliśćie z rádośćią / wiedząc / że máćie w ſobie lepƺą májętność w niebie / y trwájącą.
Przetoż nie odrzucájćie ufnośći wáƺej / którá má wielką zápłátę.
Abowiem ćierpliwośći wám potrzebá / ábyśćie wolę Bożą cżyniąc / odnieśli obietnicę.
Boć jeƺcże bárdzo / bárdzo málucżko / á oto ten / który má przyjść / przyjdźie / á nie omieƺká.
A ſpráwiedliwy z wiáry żyć będźie ; á jeźliby śię kto ſchrániáł / nie kochá śię w nim duƺá mojá.
Lecż my nie jeſteśmy z tych / którzy śię ſchrániáją ku zginieniu / ále z tych / którzy wierzą ku pozyſkániu duƺy.
