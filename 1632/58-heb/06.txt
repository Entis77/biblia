Przetoż zániecháwƺy pocżątkowych náuk o Chryſtuśie / miejmy śię ku doſkonáłośći / nie znowu zákłádájąc grunty pokuty od ucżynków mártwych / y wiáry w Bogá.
Náuki o chrzćie / y o wkłádániu rąk / y o powſtániu umárłych / y o ſądzie wiecżnym ;
A to ucżynimy / jeźli tylko Bóg dopuśći.
Abowiem niemożebne jeſt / áby ći / którzy ſą ráz oświeceni / y ſkoƺtowáli dáru niebieſkiego / y ucżeſnikámi śię ſtáli Duchá Świętego /
Skoƺtowáli też dobrego ſłowá Bożego / y mocy przyƺłego wieku /
Gdyby odpádli / áby śię záś odnowili ku pokućie / jáko ći / którzy ſobie znowu krzyżują Syná Bożego / y jáwnie go ſromocą.
Abowiem ziemiá / którá cżęſto ná śię przychodzący deƺcż pije / y rodzi ziele przygodne tym / którzy ją ſpráwują / bierze błogoſłáwieńſtwo od Bogá ;
Lecż którá przynośi ćiernie y oſty / odrzuconá jeſt y bliſká przeklęſtwá / którá ná koniec bywá ſpáloná.
A wƺákże / nájmilśi! pewniśmy o wáſ coś lepƺego y zbáwieniá bliżƺego / choćiáż ták mówimy.
Abowiem nie jeſt Bóg nieſpráwiedliwy / áby zápámiętáł prácy wáƺej y prácowitej miłośći / którąśćie okázáli ku imieniu jego / gdyśćie ſłużyli świętym y jeƺcże ſłużyćie.
A żądámy / áby káżdy z wáſ toż ſtáránie pokázywáł ku nábyćiu zupełnej nádziei áż do końcá.
Abyśćie nie byli gnuśnymi / ále náśládowcámi tych / którzy przez wiárę y ćierpliwość odziedzicżyli obietnicę.
Abowiem Bóg obietnicę cżyniąc Abráhámowi / gdy nie miáł przez kogo więkƺego przyśiądz / przyśiągł przez śiebie ſámego /
Mówiąc : Záiſte błogoſłáwiąc błogoſłáwić ći będę / y rozmnáżájąc rozmnożę ćię.
A ták długo cżekájąc / doſtąpił obietnicy.
Ludzieć wpráwdzie przez więkƺego przyśięgáją / á przyśięgá / którá śię dzieje ku potwierdzeniu / jeſt między nimi końcem wƺyſtkich ſporów.
Dlátego też Bóg chcąc doſtátecżnie okázáć dziedzicom obietnicy nieodmienność rády ſwojej / ucżynił ná to przyśięgę /
Abyśmy przez dwie rzecży nieodmienne / ( w których niemożna áby Bóg kłamał ) wárowną poćiechę mieli / my / którzyśmy śię ućiekli ku otrzymániu wyſtáwionej nádziei /
Którą mámy jáko kotwicę duƺy / y bezpiecżną / y pewną / y wchodzącą áż wewnątrz zá záſłonę /
Gdzie przewodnik dlá náſ wƺedł Jezuſ / ſtáwƺy śię według porządku Melchiſedekowego nájwyżƺym kápłánem ná wieki.
