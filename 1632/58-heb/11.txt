A wiárá jeſt gruntem tych <i>rzecży</i> których śię ſpodźiewamy <i>y</i> dowodem rzecży niewidźiálnych.
Abowiem przez nię świádectwá doƺli przodkowie.
Wiárą rozumiemy iż świát jeſt ſpráwiony ſłowem Bożym : ták yż rzecży które widźiemy / nie ſtáły śię z rzecży widźiálnych <i>/ ále z nicżego</i>.
Wiárą lepƺą ofiárę ofiárował Abel Bogu / niżeli Kajin : przez którą świádectwo otrzymał / że jeſt ſpráwiedliwy ; jákoż ſam Bóg świádectwo dał o dárzech jego / á przez te umárƺy jeƺcże mówi.
Wiárą Enoch jeſt przenieśiony / áby nieoglądał śmierći / y nie jeſt ználeźiony przeto / że go Bóg przenióſł. Abowiem pierwey niż jeſt przenieśiony / miał świádectwo / że śię podobał Bogu.
A bez wiáry nie można podobáć śię <i>Bogu</i>. Abowiem ten co przyſtępuje do Bogá / wierzyć muśi że jeſt <i>BÓG,</i> á że nadgrodę dawa tym / którzy go ƺukáją.
Wiárą nápomniony będąc od Bogá Noe o tym cżego jeƺcże nie było widźieć / bojąc się / przygotował Korab ku záchowániu domu ſwego / przez który potępił świát : y ſtał śię dźiedźicem ſpráwiedliwośći tey która jeſt z wiáry.
Wiárą powołány będąc Abráhám uſłuchał <i>Bogá,</i> áby poƺedł ná ono miejſce które miał wźiąć zá dźiedźictwo / y wyƺedł nie wiedząc dokąd idźie.
Wiárą mieƺkał w źiemi obiecáney jáko w cudzey / mieƺkájąc w namiećiech z Izáákiem y z Jákóbem / ſpólnymi dźiedźicámi teyże obietnice.
Abowiem ocżekawał miáſtá májącego grunty ; którego ſprawcą y budownicżym jeſt Bóg.
Wiárą tákże Sará wźięłá moc ku przyjęćiu naśienia / y mimo cżás wieku porodźiłá : gdyż miáłá zá wiernego tego który obiecał.
A przetoż z jednego / y to obumárłego / rozpłodźiło śię <i>potomſtwo</i> jáko mnóſtwo gwiazd niebieſkich / y jáko piaſek niezlicżony / który jeſt ná brzegu morſkim.
Według wiáry umárli ći wƺyſcy nie wźiąwƺy obietnic / ále z dáleká je upátrując / y ćieƺyli śię nimi / y witáli je / y wyznawáli / że ſą gośćmi y przychodniámi ná źiemi.
Bo ći którzy ták mówią / jáwnie okázują / iż ojcżyzny ƺukáją.
A wprawdźie gdyby byli ná onę pámiętáli z którey byli wyƺli / mieli <i>doſyć</i> cżáſu wróćić śię záś :
Ale oni lepƺey żądáją / to jeſt / niebieſkiey. Przetoż y ſam Bóg niewſtydźi śię názywáć Bogiem ich / bo im miáſto zgotował.
Wiárą ofiárował Abráhám Izááká będąc kuƺony : á ofiárował jednorodzonego / ten który był wźiął obietnicę ;
Do którego rzecżono : W Izááku tobie będźie názwano naśienie :
Uważájąc to / iż Bóg może y od umárłych wzbudźić : z kąd go też w podobieńſtwie <i>zmartwychwſtánia</i> przyjął.
Wiárą około przyƺłych rzecży błogoſłáwił Izáák Jákóbá y Eſawá.
Wiárą Jákób umierájąc káżdemu z ſynów Józefowych błogoſłáwił / y pokłonił śię podpárƺy śię ná wierzch laſki ſwojey.
Wiárą Józef umierájąc o wyśćiu ſynów Izráelſkich wzmiankę ucżynił / y zſtrony kośći ſwojich rozkazał.
Wiárą národźiwƺy śię Mojzeƺ / był ukryty przez trzy mieśiące od rodźiców ſwojich / przeto że widźieli nadobne dźiećiątko / y nie bali śię wyroku królewſkiego.
Wiárą Mojzeƺ będąc już doroſłym / zbraniał śię być zwány ſynem córki Fáráonowey :
Rácżey ſobie obrawƺy złe rzecży ćierpieć z ludem Bożym / niżeli docżeſną mieć z grzechu roſkoƺ :
Zá więkƺe pokładájąc bogáctwo nád ſkárby Egipſkie / urągánie CHryſtuſowe ; bo śię oglądał ná odpłátę.
Wiárą opuśćił Egipt / nie bojąc śię gniewu królewſkiego : bo jákoby widźiał niewidźiálnego / mężnie ſobie pocżynał.
Wiárą obchodźił Wielkąnoc y wylanie krwie ; áby ten który tráćił pierworodne / nie dotknął śię ich.
Wiárą przeƺli przez morze cżerwone / jáko po ſuƺy : o co kuśiwƺy śię Egipcżánie / potonęli.
Wiárą mury Jerychá upádły / gdy je obchodzono przez śiedm dni.
Wiárą Rácháb wƺetecżnicá nie zginęłá weſpół z niepoſłuƺnymi / przyjąwƺy ſpokojem ƺpiegi do goſpody.
A cóż więcey mam mówić? Boby mi cżáſu nie ſtáło / gdybym miał powiádáć o Gedeonie / y o Báráku / y o Sámſonie / y o Jefte / y o Dawidźie / y o Sámuelu / y o Prorokách.
Którzy przez wiárę zwalcżyli króleſtwá / cżynili ſpráwiedliwość / doſtępowáli obietnic / lwom páƺcżeki záwieráli :
Zágaƺáli moc ogniá / uchodźili oſtrzá miecżów / mocnymi śię ſtawáli z niemocnych / mężnymi bywáli ná wojnie / wojſká cudzoźiemców do ućiekánia przywodźili :
Niewiáſty odbieráły umárłe ſwoje wzbudzone ; á drudzy ſą ná próbách rozćiągnieni / nie przyjąwƺy wybáwienia / áby lepƺego doſtąpili zmartwychwſtánia.
Drudzy záśię pośmiewiſk y bicżowánia doświadcżyli ; nád to y związek y więźienia.
Byli kámionowáni / piłą przećieráni / kuƺeni / miecżem zábijáni / chodźili w owcżych y koźich ſkórách : byli w niedoſtátku / w ućiſku / w niewcżáśiech.
( Których nie był świát godźien ) Tułáli śię po puſtyniách / y po górách / y jáſkiniách / y jámách źiemie.
A ći wƺyſcy świádectwo otrzymawƺy przez wiárę / nie doſtąpili obietnice :
Przeto / że Bóg o nas coś lepƺego przejrzał / áby oni bez nas nieſtáli śię doſkonáłymi.
