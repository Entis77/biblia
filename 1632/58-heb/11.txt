A wiárá jeſt gruntem tych rzecży / których śię ſpodziewámy / y dowodem rzecży niewidziálnych.
Abowiem przez nią świádectwá doƺli przodkowie.
Wiárą rozumiemy / iż świát jeſt ſpráwiony ſłowem Bożem / ták iż rzecży / które widzimy / nie ſtáły śię z rzecży widziálnych / ále z nicżego.
Wiárą lepƺą ofiárę ofiárowáł Abel Bogu / niżeli Káin / przez którą świádectwo otrzymáł / że jeſt ſpráwiedliwy / jákoż ſám Bóg świádectwo dáł o dárách jego á przez te umárłƺy jeƺcże mówi.
Wiárą Enoch jeſt przenieśiony / áby nieoglądáł śmierći / y nie jeſt ználeziony / przeto / że go Bóg przenióſł ; ábowiem pierwej niż jeſt przenieśiony / miáł świádectwo / że śię podobáł Bogu.
A bez wiáry nie możná podobáć śię Bogu ; ábowiem ten / co przyſtępuje do Bogá / wierzyć muśi / że jeſt Bóg / á że nágrodę dáje tym / którzy go ƺukáją.
Wiárą obwieƺcżony będąc od Bogá Noe o tem / cżego jeƺcże nie było widzieć / ucżćiwość wyświádcżájąc / przygotowáł koráb ku záchowániu domu ſwego / przez który potępił świát / y ſtáł śię dziedzicem ſpráwiedliwośći tej / którá jeſt z wiáry.
Wiárą powołány będąc Abráhám / uſłucháł Bogá / áby poƺedł ná ono miejſce / które miáł wziąć zá dziedzictwo / y wyƺedł / nie wiedząc / dokąd idzie.
Wiárą mieƺkáł w ziemi obiecánej jáko w cudzej / mieƺkájąc w námiotách z Izáákiem y z Jákóbem / ſpólnymi dziedzicámi tejże obietnicy.
Abowiem ocżekiwáł miáſtá májącego grunty / którego ſpráwcą y budownikiem jeſt Bóg.
Wiárą tákże Sárá wzięłá moc ku przyjęćiu náśieniá / y mimo cżáſu wieku porodziłá / gdyż miáłá zá wiernego tego / który obiecáł.
A przetoż z jednego / y to obumárłego / rozpłodziło śię potomſtwo / jáko mnóſtwo gwiázd niebieſkich y jáko piáſek niezlicżony / który jeſt ná brzegu morſkim.
Według wiáry umárli ći wƺyſcy / nie wziąwƺy obietnic / ále z dáleká je upátrując / y ćieƺyli śię niemi / y witáli je y wyznáwáli / iż ſą gośćmi y przychodniámi ná ziemi.
Bo ći / którzy ták mówią / jáwnie okázują / iż ojcżyzny ƺukáją.
A wpráwdzie / gdyby byli ná onę pámiętáli / z której byli wyƺli / mieli doſyć cżáſu wróćić śię záś.
Ale oni lepƺej żądáją / to jeſt niebieſkiej ; przetoż y ſám Bóg nie wſtydzi śię názywáć Bogiem ich / bo im miáſto zgotowáł.
Wiárą ofiárowáł Abráhám Izááká / będąc kuƺony / á ofiárowáł jednorodzonego ten / który był wziął obietnicę.
Do którego rzecżono : W Izááku tobie będzie názwáne náśienie ;
Uwáżájąc to / iż Bóg może y od umárłych wzbudzić ; ſkąd go też w podobieńſtwie zmártwychwſtániá przyjął.
Wiárą około przyƺłych rzecży błogoſłáwił Izáák Jákóbá y Ezáwá.
Wiárą Jákób umierájąc / káżdemu z ſynów Józefowych błogoſłáwił y pokłonił śię podpárłƺy śię ná wierzch láſki ſwojej.
Wiárą Józef umierájąc / o wyjśćiu ſynów Izráelſkich wzmiánkę ucżynił / y z ſtrony kośći ſwoich rozkázáł.
Wiárą národziwƺy śię Mojżeƺ / był ukryty przez trzy mieśiące od rodziców ſwoich / przeto / że widzieli nádobne dziećiątko / y nie báli śię wyroku królewſkiego.
Wiárą Mojżeƺ / będąc już doroſłym / zbrániáł śię być zwány ſynem córki Fáráonowej /
Rácżej ſobie obráwƺy / złe rzecży ćierpieć z ludem Bożym / niżeli docżeſną mieć z grzechu rozkoƺ /
Zá więkƺe pokłádájąc bogáctwo nád ſkárby Egipſkie urągánie Chryſtuſowe ; bo śię oglądáł ná odpłátę.
Wiárą opuśćił Egipt / nie bojąc śię gniewu królewſkiego ; bo jákoby widziáł niewidziálnego / mężnie ſobie pocżynáł.
Wiárą obchodził wielkánoc y wylánie krwi / áby ten / który tráćił pierworodnych / nie dotknął śię ich.
Wiárą przeƺli przez morze Cżerwone / jáko po ſuƺy / o co kuśiwƺy śię Egipcżánie / potonęli.
Wiárą mury Jerychá upádły / gdy je obchodzono przez śiedm dni.
Wiárą Rácháb wƺetecżnicá nie zginęłá weſpół z niepoſłuƺnymi / przyjąwƺy z pokojem ƺpiegów do goſpody.
A cóż więcej mám mówić? Boby mi cżáſu nie ſtáło / gdybym miáł powiádáć o Giedeonie y o Báráku y o Sámſonie y o Jefćie y o Dawidzie y o Sámuelu y o prorokách.
Którzy przez wiárę zwálcżyli króleſtwá / cżynili ſpráwiedliwość / doſtępowáli obietnic / lwom páƺcżęki záwieráli ;
Zágáƺáli moc ogniá / uchodzili oſtrzá miecżów / mocnymi śię ſtáwáli z niemocnych / mężnymi bywáli ná wojnie / wojſká cudzoziemców do ućiekániá przywodzili.
Niewiáſty odbieráły umárłych ſwoich wzbudzonych ; á drudzy ſą ná próbách rozćiągnieni / nie przyjąwƺy wybáwieniá / áby lepƺego doſtąpili zmártwychwſtániá.
Drudzy záśię pośmiewiſk y bicżowániá doświádcżyli / nádto y związek y więzieniá.
Byli kámionowáni / piłą przećieráni / kuƺeni / miecżem zábijáni / chodzili w owcżych y kozich ſkórách ; byli w niedoſtátku / w ućiſku / w niewcżáſách ;
( Których nie był świát godzien ; ) tułáli śię po puſtyniách / y po górách / y jáſkiniách / y jámách ziemi.
A ći wƺyſcy świádectwo otrzymáwƺy przez wiárę / nie doſtąpili obietnicy.
Przeto / że Bóg o náſ coś lepƺego przejrzáł / áby oni bez náſ nie ſtáli śię doſkonáłymi.
