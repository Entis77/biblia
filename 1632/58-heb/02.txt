Przetoż muśimy tem pilniej przeſtrzegáć tego / cośmy ſłyƺeli / byśmy ſnáć nie przećiekli.
Bo poniewáż przez Anioły mówione ſłowo było pewne / á káżde przeſtępſtwo y niepoſłuƺeńſtwo wźięło ſpráwiedliwą zápłátę pomſty :
Jákoż my ućiecżemy / jeźli zániedbámy ták wielkiego zbáwieniá / które wźiąwƺy pocżątek opowiádániá przez ſámego Páná od tych / którzy go ſłyƺeli / nám jeſt potwierdzone?
Którym y Bóg świádectwo wydáwáł przez známioná y cudá y rozlicżne mocy / y przez udźielánie Duchá Świętego według woli ſwojej.
Abowiem nie Aniołom poddáł świát przyƺły / o którym mówimy.
A świádcżył ktoś ná niektórem miejſcu / mówiąc : Cóż jeſt cżłowiek / iż náń pámiętáƺ / álbo ſyn cżłowiecży / iż go náwiedzáƺ?
Ná máłą chwilę mniejƺym ucżyniłeś go od Aniołów / chwáłą y cżćią ukoronowáłeś go / y poſtánowiłeś go nád ucżynkámi rąk twoich /
Wƺyſtkoś poddáł pod nogi jego. A poddáwƺy mu wƺyſtko / nic nie zoſtáwił / coby mu poddánego nie było ; lecż teráz jeƺcże nie widźimy / áby mu wƺyſtko poddáne było.
Ale tego ( który máło co mniejƺym ſtáł śię niżeli Anjołowie ) JEzuſá widźimy / przez ućierpienie śmierći chwáłą y cżćią ukoronowánego / áby z łáſki Bożej zá wƺyſtkich śmierći ſkoƺtowáł.
Abowiem náleżáło ná tego / dlá którego jeſt wƺyſtko / y przez którego jeſt wƺyſtko / áby wiele ſynów do chwáły przywodząc / wodzá zbáwieniá ich przez ućierpienie doſkonáłym ucżynił.
Bo ten / który poświęcá / y ći / którzy bywáją poświęceni / z jednego ſą wƺyſcy / dlá której przycżyny nie wſtydźi śię / ich bráćmi názywáć /
Mówiąc : Opowiem imię twoje bráćiom moim / w pośrodku zgromádzeniá śpiewáć ći będę.
Y záśię : Já w nim ufáć będę ; á záśię : Oto já y dźieći / które mi dáł Bóg.
Poniewáż tedy dźieći ſpółecżność máją ćiáłá y krwi / y on tákże ſtáł śię ich ucżeſnikiem / áby przez śmierć zniƺcżył tego / który miáł włádzę śmierći / to jeſt dyjábłá /
A iżby wyſwobodźił tych / którzy dlá bojáźni śmierći po wƺyſtek cżáſ żywotá podlegli byli niewoli.
Bo záiſte nigdźie nie przyjął Aniołów / ále náśienie Abráhámowe przyjął.
Skąd miáł być we wƺyſtkiem podobny bráćiom / áby był miłośiernym y wiernym nájwyżƺym kápłánem w tem / co śię u Bogá ná ubłágánie zá grzechy ludzkie dźiáć miáło.
Abowiem że ſám ćierpiáł będąc kuƺony / może tych / którzy ſą w pokuſách / rátowáć.
