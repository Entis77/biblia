Przetoż muśimy tym pilniey przeſtrzegáć tego coſmy ſłyƺeli / byſmy ſnadź nieprzećiekli.
Bo ponieważ przez Anjoły mówione ſłowo było pewne / á káżde przeſtępſtwo y niepoſłuƺeńſtwo wźięło ſpráwiedliwą zapłátę pomſty :
Jákoż my ućiecżemy / jeſli zániedbamy ták wielkiego zbáwienia? Które wźiąwƺy pocżątek opowiádánia przez <i>ſámego</i> PAná / od tych którzy <i>go</i> ſłyƺeli / nam jeſt potwirdzone.
<i>Którym</i> y Bóg świádectwo wydawał / przez známioná y cudá / y rozlicżne mocy / y <i>przez</i> udźielánie Duchá Świętego / według woli ſwojey.
Abowiem nie Anjołom poddał świát przyƺły / o którym mówimy.
A świadcżył ktoś ná niektóreym miejſcu / mówiąc : Cóż jeſt cżłowiek iż nań pámiętaƺ : álbo ſyn cżłowiecży / iż go náwiedzaƺ?
Máło mniejƺym ucżyniłeś go od Anjołów / chwałą y cżćią ukoronowałeś go / y poſtánowiłeś go nád ucżynkámi rąk twojich.
Wƺyſtkoś poddał pod nogi jego. A poddawƺy mu wƺyſtko / nic nie zoſtáwił coby mu poddánego nie było : lecż teraz jeƺcże nie widźiemy / áby mu wƺyſtko poddano było.
Ale tego ( który máło co mniejƺym ſtał śię niżeli Anjołowie ) JEzuſá widźiemy / dla ućierpienia śmierći / chwałą y cżćią ukoronowánego ; áby z łáſki Bożey zá wƺyſtkie śmierći ſkoƺtował.
Abowiem należáło ná tego dla którego <i>jeſt</i> wƺyſtko y przez którego jeſt wƺyſtko / áby wiele ſynów do chwały przywodząc / wodzá zbáwienia ich przez ućierpienie doſkonáłym ucżynił.
Bo ten który poświąca / y ći którzy bywáją poświęceni / z jednego <i>ſą</i> wƺyſcy : dla którey przycżyny nie wſtyda śię ich bráćią názywáć /
Mówiąc : Opowiem Imię twoje bráći mojey / w pośrzodku zgromádzenia śpiewáćći będę.
Y záśię : Ja w nim dufáć będę. A záśię : Oto ja y dźieći które mi dał Bóg.
Ponieważ tedy dźieći ſpółecżność máją ćiáłá y krwie / y on tákże ſtał śię ich ucżeſnikiem / áby przez śmierć zniƺcżył tego który miał władzą śmierći / to jeſt Dyabłá :
A iżby wyſwobodźił te / którzy dla bojáźni śmierći po wƺyſtek cżás żywotá podlegli byli niewoli.
Bo zájiſte nigdźie nie przyjął Anjołów : ále naśienie Abráhámowe przyjął.
Z kąd miał być we wƺyſtkim podobny bráći / áby był miłośiernym y wiernym Nawyżƺym Kápłanem w tym / co <i>śię</i> u Bogá ku ocżyśćieniu grzechów ludzkich <i>dźiáć miáło</i>.
Abowiem że ſam ćierpiał będąc kuƺony / może tych którzy ſą w pokuſách rátowáć.
