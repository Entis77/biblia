Ale ſumá tego / co śię mówi / tá jeſt : Iż tákiego mámy nájwyżƺego kápłáná / który uśiádł ná práwicy ſtolicy wielmożnośći ná niebieśiech ;
Sługą będąc świątnicy / á práwdźiwego onego przybytku / który Pán zbudowáł / á nie cżłowiek.
Abowiem káżdy nájwyżƺy kápłán ku ofiárowániu dárów y ofiár bywá poſtánowiony / ſkąd potrzebá było / áby y ten miał / coby ofiárowáł.
Bo gdyby był ná źiemi / nie byłby kápłánem / pókiby zoſtáwáli oni kápłáni / którzy według zakonu dáry ofiárują /
Którzy ſłużą kƺtáłtowi y ćieniowi rzecży niebieſkich / jáko Mojżeƺowi od Bogá powiedźiáne było / gdy miał dokońcżyć przybytku : Pátrzájże / ( mówi / ) ábyś ucżynił wƺyſtko według kƺtáłtu / któryć jeſt okázány ná tey górze.
Ale teraz náƺ kápłán tem zácniejƺego urzędu doſtąpił / im jeſt pośrednikiem lepƺego przymierzá / które lepƺemi obietnicámi jeſt utwierdzone.
Bo gdyby ono pierwƺe było bez przygány / tedyćby wtóremu nie ƺukáno miejſcá.
Abowiem gániąc Żydów / mówi : Oto dni idą / mówi Pán / gdy ucżynię z domem Izráelſkim y z domem Judzkim przymierze nowe /
Nie według przymierzá / którem ucżynił z ojcámi ich w dźień / któregom ich ujął zá rękę ich / ábym ich wywiódł z źiemi Egipſkiej ; ábowiem oni nie zoſtáli w tem przymierzu mojem / á Jám ich zániedbáł / mówi Pán.
Przetoż toć jeſt przymierze / które poſtánowię z domem Izráelſkim po tych dniách / mówi Pán : Dám práwá moje w myśl ich / y ná ſercách ich nápiƺę je / y będę Bogiem ich / á oni będą ludem moim.
Y nie będźie ucżył żáden bliźniego ſwego / y żáden brátá ſwego / mówiąc : Poznáj Páná ; ábowiem wƺyſcy mię poznáją / od nájmniejƺego z nich áż do nájwiękƺego z nich.
Bo miłośćiw będę niepráwośćiom ich / á grzechów ich y niepráwośći ich nie wſpomnę więcey.
A gdy mówi : Nowe ; pierwƺe cżyni wiotchem ; á to / co wiotƺeje y zſtárzeje śię / bliſkie jeſt zniƺcżeniá.
