Abowiem ten Melchiſedek był król Sálem / kápłán Bogá nájwyżƺego / który záƺedł drogę Abráhámowi / gdy śię wrácáł od poráżki królów / y błogoſłáwił mu.
Któremu y dzieśięćiny ze wƺyſtkiego udzielił Abráhám ; który nájprzód wykłádá śię król ſpráwiedliwośći / potem też król Sálem / co jeſt / król pokoju.
Bez ojcá / bez mátki / bez rodu / áni pocżątku dni / áni końcá żywotá nie májąc / ále przypodobány będąc Synowi Bożemu / zoſtáje kápłánem ná wieki.
Obácżćież tedy / jáko wielki ten był / któremu też dzieśięćinę z łupów dáł Abráhám pátryjárchá.
A ći / którzy ſą z ſynów Lewiego / urząd kápłáńſki przyjmujący / rozkázánie máją / áby bráli dzieśięćinę od ludu według zákonu / to jeſt / od bráći ſwoich / choć wyƺli z biódr Abráhámowych.
Ale ten / którego ród nie jeſt pocżytány między nimi / dzieśięćinę wziął od Abráhámá / y temu / który miáł obietnicę / błogoſłáwił.
A bez wƺelkiego ſporu mniejƺy od więkƺego błogoſłáwieńſtwo bierze.
A tuć dzieśięćiny biorą ludzie / którzy umieráją ; tám záśię on / o którym świádcżono / iż żyje.
A iż ták rzekę y ſám Lewi / który dzieśięćiny bierze / dáł w Abráhámie dzieśięćinę.
Abowiem jeƺcże był w biodrách ojcowſkich / gdy wyƺedł przećiwko niemu Melchiſedek.
A przetoż byłáli doſkonáłość przez kápłáńſtwo lewitſkie / ( gdyż zá niego wydány jeſt zákon ludowi ) / jákáż tego jeƺcże byłá potrzebá / áby inƺy kápłán według porządku Melchiſedekowego powſtáł / á nie był według porządku Aáronowego miánowány?
A poniewáż kápłáńſtwo jeſt przenieśione / muśi też y zákon przenieśiony być.
Bo ten / o którym śię to mówi / inƺego jeſt pokoleniá / z którego żáden nie ſłużył ołtárzowi.
Abowiem jáwná jeſt / iż z pokoleniá Judowego poƺedł Pán náƺ / o którem pokoleniu nic z ſtrony kápłáńſtwá nie mówił Mojżeƺ.
Owƺem obfićie to jeƺcże y z tego jáwná jeſt / iż powſtáł inƺy kápłán według porządku Melchiſedekowego /
Który śię ſtáł nie według zákonu przykázániá ćieleſnego / ále według mocy żywotá nieſkázitelnego.
Abowiem ták świádcży : Tyś jeſt kápłánem ná wieki według porządku Melchiſedekowego.
Bo śię ſtáło znieśienie onego przyƺłego przykázániá dlá ſłábośći jego y niepożytku.
Bo nicżego do doſkonáłośći nie przywiódł zákon ; ále ná miejſce jego wprowádzoná jeſt lepƺá nádziejá / przez którą śię przybliżámy do Bogá.
A to y względem tego / że nie bez przyśięgi jeſt wprowádzoná.
Boć śię oni bez przyśięgi kápłánámi ſtáwáli / á ten z przyśięgą przez tego / który rzekł do niego : Przyśiągł Pán / á nie będzie tego żáłowáł : Tyś jeſt kápłánem ná wieki według porządku Melchiſedekowego /
Ták dálece lepƺego przymierzá ſtáł śię rękojmią Jezuſ.
Więc też onych wiele bywáło kápłánów dlátego / iż im śmierć nie dopuśćiłá / záwƺe trwáć.
Ale ten / iż ná wieki zoſtáje / wiecżne má kápłáńſtwo /
Przetoż y doſkonále zbáwić może tych / którzy przezeń przyſtępują do Bogá / záwƺe żyjąc / áby orędowáł zá nimi.
Tákiegoć záiſte przyſtáło nám mieć nájwyżƺego kápłáná / świętego / niewinnego / niepokálánego / odłącżonego od grzeƺników / y któryby śię ſtáł wyżƺy nád niebioſá :
Któryby nie potrzebowáł ná káżdy dzień / jáko oni nájwyżśi kápłáni / pierwej zá ſwoje grzechy włáſne ofiár ſpráwowáć / á potem zá ludzkie ; bo to ucżynił ráz ſámego śiebie ofiárowáwƺy.
Abowiem zákon ludzi podległych krewkośći ſtánowił zá nájwyżƺych kápłánów : ále ſłowo przyśięgi / które śię ſtáło po zákonie / poſtánowiło Syná Bożego doſkonáłego ná wieki.
