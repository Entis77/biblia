A miáłoć y pierwƺe ono przymierze uſtáwy około ſłużby Bożej y świątnicę świecką.
Abowiem ſpráwiony był przybytek pierwƺy / w którym był świecżnik / y ſtół / y pokłádne chleby / który zwáno świątnicą.
A zá drugą záſłoną był przybytek / który zwáno świątnicą nájświętƺą /
Májąc złotą kádźielnicę / y ſkrzynię przymierzá zewƺąd złotem powlecżoną / w którey było wiádro złote / májące w ſobie mánnę / y láſká Aáronowá / która byłá zákwitnęłá / y táblice przymierzá.
A nád ſkrzynią byli Cherubinowie chwáły / którzy záćieniáli ubłágálnię / o których rzecżách teraz nie potrzebá mówić o káżdej z oſobná.
A to gdy ták przygotowáno / do pierwƺego przybytku záwƺe wchodzą kápłáni / ſłużby Boże odpráwując ;
Ale do drugiego ráz w rok ſám nájwyżƺy kápłán / nie bez krwi / którą ofiáruje ſám zá śię y zá ludzkie niewiádomośći.
Przez co dáje znáć Duch Święty / iż jeƺcże nie byłá objáwioná drogá do świątnicy / póki jeƺcże trwáł pierwƺy przybytek /
Który był wzorem ná ten teráźniejƺy cżáſ / w którym dáry y ofiáry bywáją ofiárowáne / które nie mogą w ſumieniu doſkonáłym ucżynić tego / co ſłużbę Bożą odpráwuje ;
Tylko w pokármách y w nápojách y w różnych omywániách y w uſtáwách ćieleſnych áż do cżáſu nápráwieniá włożone były.
Ale CHryſtuſ przyƺedłƺy / nájwyżƺy kápłán przyƺłych dóbr / przez więkƺy y doſkonálƺy przybytek / nie ręką zbudowány / to jeſt / nie tego budynku ;
Ani przez krew kozłów y ćielców / ále przez włáſną krew ſwoję wƺedł ráz do świątnicy / ználázłƺy wiecżne odkupienie.
Abowiem jeſli krew wołów y kozłów y popiół jáłowicy pokrápiájący ſplugáwione poświęcá ku ocżyƺcżeniu ćiáłá :
Jákoż dáleko więcey krew CHryſtuſowá / który przez Duchá wiecżnego ſámego śiebie ofiárowáł nienágánionym Bogu / ocżyśći ſumienie wáƺe od ucżynków martwych ku ſłużeniu Bogu żywemu?
Y dlátego jeſt nowego teſtámentu pośrednikiem / áby / gdyby śmierć náſtąpiłá ná odkupienie onych wyſtępków / które były pod pierwƺym teſtámentem / ći / którzy ſą powołáni / wźięli obietnicę wiecżnego dźiedźictwá.
Abowiem gdźie jeſt teſtáment / potrzebá / áby śmierć náſtąpiłá tego / który cżyni teſtáment.
Bo teſtáment tych / którzy zmárli / mocny jeſt / gdyż jeƺcże nie jeſt wáżny / póki żyje ten / co teſtáment ucżynił.
Skąd áni on pierwƺy teſtáment bez krwi nie był poświęcony.
Abowiem gdy Mojżeƺ wƺyſtko przykázánie według zakonu opowiedźiáł wƺyſtkiemu ludowi / wźiąwƺy krew ćielców y kozłów z wodą y z wełną ƺárłátową / y z hizopem / y ſáme kśięgi y lud wƺyſtek pokropił /
Mówiąc : Táć jeſt krew przymierzá / które wám Bóg przykázáł.
Dotego y przybytek y wƺyſtko nácżynie do ſłużby Bożej náleżące krwią tákże pokropił.
A niemál wƺyſtko według zakonu krwią ocżyƺcżone bywá / á bez rozlániá krwi nie bywá odpuƺcżenie grzechów.
A ták potrzebá było / áby kƺtáłty onych rzecży / które ſą ná niebie / temi rzecżámi były ocżyƺcżone / á ſáme rzecży niebieſkie lepƺemi ofiárámi / niżeli te.
Abowiem CHryſtuſ nie wƺedł do świątnicy ręką ucżynioney / któráby byłá wizerunkiem práwdźiwej / ále do ſámego niebá / áby śię teraz okázywáł przed oblicżnośćią Bożą zá námi.
A nie iżby cżęſto ofiárowáł ſámego śiebie / jáko nájwyżƺy kápłán wchodźi do świątnicy co rok ze krwią cudzą ;
( Bo inácżej muśiáłby był cżęſtokroć ćierpieć od pocżątku świátá ) / lecż teraz przy ſkońcżeniu wieków ráz objáwiony jeſt ku zgłádzeniu grzechu przez ofiárowánie ſámego śiebie.
A jáko poſtánowiono ludźiom / ráz umrzeć / á potem będźie ſąd :
Ták y CHryſtuſ / ráz będąc ofiárowány ná zgłádzenie wielu grzechów / drugi ráz śię bez grzechu okáże tym / którzy go ocżekują ku zbáwieniu.
