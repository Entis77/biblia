A miáłoć y pierwƺe ono <i>przymierze</i> uſtáwy <i>około</i> ſłużby <i>Bożey,</i> y świątnicę świecką.
Abowiem ſpráwiony był przybytek pierwƺy / w którym był świecżnik / y ſtół / y pokłádne chleby / który zwano Świątnicą.
A zá drugą zaſłoną był przybytek / który zwano Świątnicą naświętƺą :
Májąc złotą kádźilnicę / y ſkrzynię przymierza / zewƺąd złotem powlecżoną ; w którey <i>było</i> wiádro złote / májące w ſobie Mánnę y laſkę Aáronowę która byłá zákwitnęłá / y Tablice Przymierza.
A nád Skrzynią byli Cherubinowie chwały / którzy záćieniáli Ubłagálnią : o których rzecżách teraz nie potrzebá mówić o káżdey z oſobná.
A to gdy ták przygotowano / do pierwƺego przybytku záwƺe wchodzą Kápłani / ſłużby <i>Boże</i> odpráwując.
Ale do drugiego raz w rok ſam Nawyżƺy Kápłan / nie bez krwie którą ofiáruje ſam zá śię y zá ludzkie niewiádomośći.
Przez co dawa znáć Duch Święty / iż jeƺcże nie byłá objáwiona drogá <i>do</i> Świątnice / póki jeƺcże trwał pierwƺy Przybytek.
Który był wzorem ná ten terázniejƺy cżás : w którym dáry y ofiáry bywáją ofiárowáne / które nie mogą w ſumnieniu doſkonáłym ucżynić tego / co ſłużbę Bożą odpráwuje :
Tylko w pokármách / y w napojách / y w różnych omywániách / y w uſtáwách ćieleſnych / áż do cżáſu nápráwienia włożone były.
Ale CHryſtus przyƺedƺy / Nawyżƺy Kápłan przyƺłych dóbr / przez więkƺy y doſkonálƺy przybytek nie ręką zbudowány / to jeſt / nie tego budynku ;
Ani przez krew kozłów y ćielców : ále przez właſną krew ſwoję wƺedł raz do Świątnice / ználazƺy wiecżne odkupienie.
Abowiem jeſli krew wołów y kozłów / y popiół jáłowice pokrapiájący / zplugáwione poświąca ku ocżyścieniu ćiáłá :
Jákoż dáleko więcey krew CHryſtuſowá / który przez Duchá wiecżnego ſámego śiebie ofiárował nienágánionym Bogu / ocżyśći ſumnienie wáƺe od ucżynków martwych ku ſłużeniu Bogu żywemu?
Y dla tego jeſt Nowego Teſtámentu Pośrzednikiem : áby / gdyby śmierć náſtąpiłá ná odkupienie onych wyſtępków które były pod pierwƺym Teſtámentem / ći którzy ſą powołáni / wźięli obietnicę wiecżnego dźiedźictwá.
Abowiem gdźie <i>jeſt</i> Teſtáment / potrzebá áby śmierć náſtąpiłá tego / który cżyni Teſtáment.
Bo Teſtáment tych którzy zmárli mocny jeſt : gdyż jeƺcże nie jeſt ważny póki żyw ten co Teſtáment ucżynił.
Z kąd áni on pierwƺy <i>Teſtáment</i> bez krwie nie był poświęcony.
Abowiem gdy Mojzeƺ wƺyſtko przykazánie według zakonu opowiedźiał wƺyſtkiemu ludowi / wźiąwƺy krew ćielców y kozłów z wodą y z wełną ƺárłatową / y z hizopem ; y ſáme kśięgi y lud wƺyſtek pokropił /
Mówiąc : Táć jeſt krew przymierza które wam Bóg przykazał.
Do tego / y przybytek y wƺyſtko nacżynie do ſłużby <i>Bożey</i> należące / krwią tákże pokropił.
A niemal wƺyſtko według zakonu krwią ocżyśćiono bywa : á bez rozlania krwie nie bywa odpuƺcżenie <i>grzechów</i>.
A ták potrzebá było / áby kƺtałty onych rzecży / które ſą ná niebie / tymi rzecżámi były ocżyśćione : á ſáme <i>rzecży</i> niebieſkie lepƺymi ofiárámi / niżeli te.
Abowiem CHryſtus nie wƺedł do świątnice ręką ucżynioney / któraby byłá wizerunkiem práwdźiwey / ále do ſámego niebá : áby śię teraz okázował przed oblicżnośćią Bożą zá námi.
A nie iżby cżęſto ofiárował ſámego śiebie / jáko Nawyżƺy Kápłan wchodźi do Świątnice co rok ze krwią cudzą :
( Bo inácżey muśiałby był cżęſtokroć ćierpieć od pocżątku świátá ) Lecż teraz przy ſkońcżeniu wieków / raz objáwiony jeſt ku zgłádzeniu grzechu przez ofiárowánie ſámego śiebie.
A jáko poſtánowiono ludźiom raz umrzeć / á potym <i>będźie ſąd</i> :
Ták y CHryſtus raz <i>będąc</i> ofiárowány ná zgłádzenie wielu grzechów ; drugi raz śię bez grzechu okaże tym / którzy go ocżekawają ku zbáwieniu.
