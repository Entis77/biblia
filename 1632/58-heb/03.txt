Przetoż bráćia święći / powołánia niebieſkie° ucżeſnicy / obacżćie Apoſtołá y Nawyżƺego Kápłaná wyznánia náƺego CHryſtuſá JEzuſá :
Wiernego temu / który go poſtánowił ; jáko y Mojżeƺ <i>był</i> we wƺyſtkim domu jego.
Abowiem tym więkƺey chwały ten nád Mojżeƺá godźien / im więkƺą cżeść ma budownik domu niżeli <i>ſam</i> dom.
Bo káżdy dom bywa budowány od kogo : ále który wƺyſtkie <i>rzecży</i> zbudował / Bóg jeſt.
A byłći Mojżeƺ wierny we wƺyſtkim domu jego jáko ſługá / ná świádectwo tego / co potym miáło być mówiono.
Ale CHryſtuſ jáko Syn pánuje nád domem ſwojim ; którego domem my jeſteſmy / jeſli tylko tę pewną ufność y tę chwałę nádźieje áż do końcá ſtátecżną záchowamy.
Przetoż jáko mówi Duch Święty : Dźiś / jeſlibyśćie głos jego uſłyƺeli /
Nie zátwardzajćież ſerc wáƺych / jáko w rozdraźnieniu w dźień onego pokuƺeniá ná puƺcży :
Kiedy mię kuśili ojcowie wáƺy / <i>y</i> doświadcżáli mię ; y widźieli ſpráwy moje przez cżterdźieśći lat.
Dla tegom śię rozgniewał ná ten naród / y rzekłem : Ci záwƺe błądzą ſercem / á nie poznawáją dróg mojich.
Przetoż / przyśiągłem w gniewie mojim / że nie wnijdą do odpocżynienia mojego.
Pátrzćież bráćia / by ſnadź nie było w którym z was ſerce złe <i>y</i> niewierne / któreby odſtępowáło od Bogá żywego :
Ale nápominajćie jedni drugie ná káżdy dźień / póki śię Dźiś názywa / áby kto z was nie był zátwárdzony oƺukániem grzechu.
Abowiem ſtáliſmy śię ucżeſnikámi CHryſtuſá / jeſliże tylko pocżątek tego gruntu áż do końcá ſtátecżny záchowamy.
<i>Przetoż</i> póki bywa rzecżono : Dźiś / jeſlibyśćie głos jego uſłyƺeli / nie zátwardzajćież ſerc ſwojich / jáko w onym rozdraźnieniu.
Abowiem niektórzy uſłyƺawƺy / rozdraźnili <i>PAná,</i> ále nie wƺyſcy którzy byli wyƺli z Egiptu przez Mojzeƺá.
A ná któreż śię gniewał przez cżterdźieśći lat? Izali nie ná te / którzy grzeƺyli / których ćiáłá poległy ná puƺcży?
A którymże przyśiągł / że nie mieli wniść do odpocżynienia jego? A za nie tym / którzy byli niepoſłuƺnymi?
Y widźiemy / yż <i>tam<i/> nie mogli wniść dla niedowiárſtwá.
