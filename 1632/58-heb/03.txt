Przetoż / bráćiá święći / powołániá niebieſkiego ucżeſnicy! obácżćie Apoſtołá y Nájwyżƺego Kápłáná wyznániá náƺego / Chryſtuſá Jezuſá /
Wiernego temu / który go poſtánowił / jáko y Mojżeƺ był we wƺyſtkim domu jego.
Abowiem tem więkƺej chwáły ten nád Mojżeƺá godzien / im więkƺą cżeść má budownik domu / niżeli ſám dom.
Bo káżdy dom bywá budowány od kogo ; ále który wƺyſtkie rzecży zbudowáł / Bóg jeſt.
A byłći Mojżeƺ wierny we wƺyſtkim domu jego / jáko ſługá / ná świádectwo tego / co potem miáło być mówione.
Ale Chryſtuſ jáko ſyn nád domem ſwoim pánuje / którego domem my jeſteśmy / jeźli tylko tę pewną ufność y tę chwáłę nádziei áż do końcá ſtátecżną záchowámy.
Przetoż jáko mówi Duch Święty : Dziś / jeźlibyśćie głoſ jego uſłyƺeli /
Nie zátwárdzájćież ſerc wáƺych / jáko w rozdráźnieniu / w dzień onego pokuƺeniá ná puƺcży.
Gdzie mię kuśili ojcowie wáśi / y doświádcżáli mię / y widzieli ſpráwy moje przez cżterdzieśći lát.
Dlátegom śię rozgniewáł ná ten náród y rzekłem : Ci záwƺe błądzą ſercem / á oni nie poznáwáją dróg moich.
Jákom przyśiągł w gniewie moim / że nie wnijdą do odpocżnieniá mojego.
Pátrzćież / bráćiá! by ſnáć nie było w którym z wáſ ſerce złe y niewierne / któreby odſtępowáło od Bogá żywego ;
Ale nápominájćie jedni drugich ná káżdy dzień / póki śię Dziś názywá / áby kto z wáſ nie był zátwárdzony oƺukániem grzechu.
Abowiem ſtáliśmy śię ucżeſnikámi Chryſtuſá / jeźliże tylko pocżątek tego gruntu áż do końcá ſtátecżny záchowámy.
Przetoż póki bywá rzecżone : Dziś / jeźlibyśćie głoſ jego uſłyƺeli / nie zátwárdzájćież ſerc wáƺych / jáko w onem rozdráżnieniu.
Abowiem niektórzy uſłyƺáwƺy / rozdráźnili Páná / ále nie wƺyſcy / którzy byli wyƺli z Egiptu przez Mojżeƺá.
A ná którychże śię gniewáł przez cżterdzieśći lát? Izáli nie ná tych / którzy grzeƺyli / których ćiáłá poległy ná puƺcży?
A którymże przyśiągł / że nie mieli wnijść do odpocżnieniá jego? Azáż nie tym / którzy byli niepoſłuƺnymi?
Y widzimy / iż tám nie mogli wnijść dlá niedowiárſtwá.
