Miłość bráterſká niech zoſtáje.
Nie zápominájćie ochoty ku gośćiom ; ábowiem przez tę niektórzy nie wiedząc / Anioły zá gośćie przyjmowáli.
Pámiętájćie ná więźniów / jákobyśćie ſpółwięźniámi byli ; ná utrápionych / jáko ći / którzy też w ćiele jeſteśćie.
Ucżćiwe jeſt máłżeńſtwo między wƺyſtkimi / y łoże niepokáláne ; ále wƺetecżników y cudzołożników Bóg będźie ſądźił.
Obcowánie wáƺe niech będźie bez łákomſtwá / przeſtwájąc ná tem / co máćie ; boć ſám powiedźiáł : Nie zániechám ćię / áni ćię opuƺcżę ;
Ták ábyśmy śmiele mówić mogli : Pán mi jeſt pomocnikiem / nie będę śię báł / áby mi co miał ucżynić cżłowiek.
Pámiętájćie ná wodzów wáƺych / którzy wám mówili ſłowo Boże / których obcowániá koniec upátrując náśládujćie wiáry ich.
JEzuſ CHryſtuſ wcżoráj y dźiś / tenże y ná wieki.
Zá náukámi rozmáitemi y obcemi nie unośćie śię ; ábowiem dobrá rzecż jeſt / áby łáſką było utwierdzone ſerce á nie pokármámi / które nie pomogły tym / co śię nimi báwili.
Mámy ołtárz / z którego nie máją wolnośći jeść ći / którzy przybytkowi ſłużą.
Abowiem bydląt / których krew bywá wnoƺoná zá grzech do świątnicy przez nájwyżƺego kápłáná / tych ćiáłá pálone bywáją zá obozem.
Dlátego y JEzuſ / áby poświęćił lud włáſną krwią ſwoją / zá brámą ućierpiáł.
Wynijdźmyż tedy do niego zá obóz / noƺąc urągánie jego.
Abowiem nie mámy tu miáſtá trwáłego / ále onego przyƺłego ƺukámy.
Przetoż przez niego ofiárujmy Bogu ofiárę chwáły uſtáwicżnie / to jeſt / owoce wárg wyznáwájących / imieniowi jego.
A dobrocżynnośći y udźielániá nie przepominájćie ; ábowiem śię Bóg w tákowych ofiárách kochá.
Bądźćie poſłuƺni wodzom wáƺym / y bądźćie im poddáni ; ábowiem oni cżują nád duƺámi wáƺemi / jáko ći / którzy licżbę oddáć máją ; áby to z rádośćią cżynili / á nie z wzdychániem ; boć wám to nie jeſt pożytecżne.
Módlćie śię zá námi ; ábowiem ufámy / iż mámy dobre ſumienie / jáko ći / którzy śię chcemy we wƺyſtkiem dobrze záchowáć.
A tem więcey proƺę was / ábyśćie to cżynili / ábym wám tem rychlej był przywrócony.
A Bóg pokoju / który wywiódł od umárłych we krwi przymierzá wiecżnego / onego wielkiego páſterzá owiec / Páná náƺego JEzuſá /
Niech was doſkonáłymi ucżyni w káżdym ucżynku dobrym ku cżynieniu woli ſwojey / ſpráwując w was to / co przyjemnego przed oblicżem jego / przez JEzuſá CHryſtuſá / któremu niech będźie chwałá ná wieki wieków. Amen.
A proƺę was / bráćia! znośćie ćierpliwie ſłowo nápominániá tego ; bomći do was krótko piſáł.
Wiedzćie o bráćie Tymoteuƺu / że jeſt wypuƺcżony / z którym ( jeſlibym rychło przyƺedł ) / oglądám was.
Pozdrówćie wƺyſtkich wodzów wáƺych / y wƺyſtkich świętych. Pozdráwiáją was bráćia z Włoch.
Łáſká niech będźie z wámi wƺyſtkimi. Amen.
