Abowiem káżdy Nawyżƺy Kápłan z ludźi wźięty / zá ludźie bywa poſtánowiony / w tych <i>rzecżách</i> które do Bogá należą : <i>to jeſt,</i> áby ofiárował dáry / y ofiáry zá grzechy.
Któryby mógł jáko przyſtoji użalić śię nieumiejętnych y błądzących / będący ſam obłożony krewkośćią.
A dla tey jeſt powinien / jáko zá lud / ták y ſam zá śię ofiárowáć zá grzechy.
A nikt ſobie tey cżći nie bierze / tylko <i>ten</i> który bywa powołány od Bogá / jáko y Aáron.
Ták y CHryſtus nie ſam ſobie tey cżći przywłaƺcżył / áby śię ſtał Nawyżƺym Kápłanem ; ále ten który mu rzekł : Syn mój jeſteś ty / Jam ćiebie dźiś zpłodźił.
Jáko y ná inƺym <i>miejſcu</i> mówi : Tyś jeſt Kápłanem ná wieki / według porządku Melchiſedekowego.
Który zá dni ćiáłá ſwego / modlitwy y uniżone proźby do tego / który go mógł záchowáć od śmierći / z wołániem wielkim y ze łzámi ofiárował : y wyſłuchány jeſt w tym ſtráchu.
A choć był Synem <i>Bożym,</i> wƺákże z tego co ćierpiał / náucżył śię poſłuƺeńſtwá.
A ták doſkonáłym będąc / ſtał śię wƺyſtkim ſobie poſłuƺnym / powodem zbáwienia wiecżnego.
Názwány będąc od Bogá Kápłanem Nawyżƺym / według porządku Melchiſedekowego.
O którym wieleby śię miáło mówić / y trudnych rzecży do wyłożenia : áleśćie śię wy ſtáli leniwi ku ſłuchániu.
Abowiem májąc bydź Náucżyćielámi względem cżáſu / záśię potrzebujećie / áby was ucżono / które ſą pierwƺe pocżątki mów Bożych : y ſtáliśćie śię <i>jako</i> mleká potrzebujący / á nie twárdego pokármu.
Bo káżdy co śię tylko mlekiem karmi / ten nie jeſt powiádomy mowy ſpráwiedliwośći : ( gdyż jeſt niemowiątkiem. )
Aleć doſkonáłym náleży twárdy pokárm : <i>to jeſt</i> tym / którzy przez przyzwycżájenie máją ſmyſły wyćwicżone / ku rozeznániu dobrego y złego.
