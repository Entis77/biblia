Abowiem káżdy nájwyżƺy kápłán z ludzi wzięty / zá ludzi bywá poſtánowiony w tych rzecżách / które do Bogá náleżą / to jeſt / áby ofiárowáł dáry / y ofiáry zá grzechy.
Któryby mógł / jáko przyſtoi / użálić śię nieumiejętnych y błądzących / będący ſám obłożony krewkośćią.
A dlá tej jeſt powinien / jáko zá lud / ták y ſám zá śię ofiárowáć zá grzechy.
A nikt ſobie tej cżći nie bierze / tylko ten / który bywá powołány od Bogá jáko y Aáron.
Ták y Chryſtuſ nie ſám ſobie tej cżći przywłáƺcżył / áby śię ſtáł nájwyżƺym kápłánem ; ále ten / który mu rzekł : Syn mój jeſteś ty / jám ćię dziś ſpłodził.
Jáko y ná inƺem miejſcu mówi : Tyś jeſt kápłánem ná wieki według porządku Melchiſedekowego.
Który zá dni ćiáłá ſwego modlitwy y uniżone prośby do tego / który go mógł záchowáć od śmierći / z wołániem wielkiem y ze łzámi ofiárowáł / y wyſłuchány jeſt dlá ucżćiwośći.
A choć był Synem Bożym / wƺákże z tego / co ćierpiáł / náucżył śię poſłuƺeńſtwá.
A ták doſkonáłym będąc / ſtáł śię wƺyſtkim ſobie poſłuƺnym przycżyną zbáwieniá wiecżnego /
Názwány będąc od Bogá kápłánem nájwyżƺym według porządku Melchiſedekowego.
O którym wieleby śię miáło mówić / y trudnych rzecży do wyłożeniá ; áleśćie śię wy ſtáli leniwi ku ſłuchániu.
Abowiem májąc być náucżyćielámi względem cżáſu / záśię potrzebujećie / áby wáſ ucżono / które ſą pierwƺe pocżątki mów Bożych / y ſtáliśćie śię jáko mleká potrzebujący / á nie twárdego pokármu.
Bo káżdy / co śię tylko mlekiem kármi / ten nie jeſt powiádomy mowy ſpráwiedliwośći : ( gdyż jeſt niemowlątkiem )
Aleć doſkonáłym náleży twárdy pokárm / to jeſt tym / którzy przez przyzwycżájenie máją zmyſły wyćwicżone ku rozeznániu dobrego y złego.
