Przetoż y my májąc ták wielki około śiebie leżący obłok świádków / złożywƺy wƺeláki ćiężar y grzech który nas ſnádnie obſtępuje : przez ćierpliwość bieżmy w zawodźie / który nam jeſt wyſtáwiony.
Pátrząc ná JEzuſá wodzá / y dokońcżyćielá wiáry / który dla wyſtáwioney ſobie rádośći / podjął krzyż wzgárdźiwƺy ſromotę / y uśiadł ná práwicy ſtolice Bożey.
Przetoż uważajćie jáki jeſt ten / który podejmował tákowe od grzeƺników przećiwko ſobie zprzećiwiánie ; ábyśćie oſłábiawƺy w umyſłách wáƺych / nie uſtawáli.
Jeƺcżeśćie śię áż do krwie nie zprzećiwili / walcżąc przećiwko grzechowi.
Cżyliśćie zápámiętáli nápominánia / które wam jáko ſynom mówi? Synu mój / nie lekce ſobie poważaj káźni PAńſkiey / á nie tráć ſercá gdy od niego bywaƺ karány.
Abowiem / kogo PAN miłuje / tego karze ; á ſmága káżdego / którego zá ſyná przyjmuje.
Jeſli znośićie karánie / Bóg śię wam ofiáruje jáko ſynom. Abowiem któryż jeſt ſyn / któregoby Oćiec nie karał?
A jeſli jeſteśćie bez karánia / którego wƺyſcy ſą ucżeſnikámi ; tedyśćie bękártámi / á nie ſynámi.
A nád to / ćieleſne Ojce náƺe mieliſmy którzy nas karáli / á baliſmy śię ich : Zaż dáleko więcey nie mamy być poddáni Ojcu duchów / ábyſmy żyli?
Abowiem oni ná máło dni jáko śię im zdáło nas káráli ; ále ten ku pożytkowi <i>náƺemu,</i> ná to / ábyſmy byli ucżeſnikámi świętobliwośći jego.
A wƺelkie karánie gdy przytomne jeſt / nie zda śię być weſołe / ále ſmętne : lecż potym owoc ſpráwiedliwośći ſpokojny przynośi / tym którzy ſą przez nie wyćwicżeni.
Przetoż opuƺcżone ręce / y zemdlone koláná wyproſtujćie ;
A cżyńćie koleje proſte nogámi wáƺymi : iżby to co jeſt chromego / z drogi nie uſtąpiło / ále rácżey uzdrowione było.
Pokoju náśládujćie ze wƺyſtkimi / y świętobliwośći : bez którey żaden nie ogląda PAná.
Upátrując żeby kto nie odpadł od łáſki Bożey ; á żeby który korzeń gorzkośći niepodróſł á nie przekáźił ; y przezeń áby śię ich wiele nie pokaláło.
Aby kto nie był wƺetecżnym álbo ſproſnym / jáko Eſaw : który zá potráwę jednę przedał pierworodztwo ſwoje.
Abowiem wiećie iż y potym / gdy chćiał odźiedźicżyć błogoſłáwieńſtwo / był odrzucony : bo nie ználazł miejſcá pokuty / choć jey z płácżem ƺukał.
Bośćie nieprzyſtąpili do góry która śię da dotknąć / y do ogniá gorájącego / y do wichru / y do ćiemnośći / y do burzy :
Y do dźwięku trąby / y do głoſu ſłów ; który ći co ſłyƺeli / prośili / áby więcey do nich nie mówiono :
( Abowiem niemogli znieść tego co im rozkázowano. Gdyby śię y bydlę góry dotknęło / będźie ukámionowáne / álbo poćiſkiem przebite ;
A ták ſtráƺne to było co widźieli / <i>że też</i> Mojzeƺ rzekł ; Ulękłem śię / y drżę. )
Aleśćie przyſtąpili do góry Syon / y do miáſtá Bogá żywego / <i>do</i> Jeruzalem niebieſkiego / y <i>do</i> niezlicżonych tyśięcy Anjołów :
Do wálnego zgromádzenia / y do zebránia pierworodnych / którzy ſą ſpiſáni w niebie ; y <i>do</i> Bogá Sędźiego wƺyſtkich / y <i>do</i> Duchów ſpráwiedliwych <i>y</i> doſkonáłych :
Y do Pośrzedniká nowego Teſtámentu JEzuſá / y do krwie pokropienia lepƺe rzecży mówiącey niż Ablowá.
Patrzajćież ábyśćie nie gárdźili tym który mówi. Abowiem jeſliż oni nie uƺli / którzy gárdźili tym który ná źiemi ná miejſcu Bożym mówił ; dáleko więcey my / jeſliże śię od tego który z niebá <i>jeſt</i> odwróćimy.
Którego głos ná on cżás poruƺył był źiemią / á teraz obiecał mówiąc : Jeƺcże Ja raz poruƺę / nie tylko źiemią / ále y niebem.
A to <i>że mówi</i> ; Jeƺcże raz : pokázuje znieśienie rzecży chwiejących śię / jáko tych które ſą ucżynione ; áby zoſtawáły te które śię nie chwieją.
Przetoż przyjmując króleſtwo nie chwiejące śię / miejmy łáſkę ; przez którą ſłużymy przyjemnie Bogu ze wſtydem y z bojáźnią.
Abowiem BÓG náƺ jeſt ogniem trawiącym.
