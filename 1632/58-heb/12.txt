Przetoż y my / májąc ták wielki około śiebie leżący obłok świádków / złożywƺy wƺeláki ćiężár y grzech / który náſ ſnádnie obſtępuje / przez ćierpliwość bieżmy w záwodźie / który nám jeſt wyſtáwiony ;
Pátrząc ná JEzuſá wodzá y dokońcżyćielá wiáry / który dlá wyſtáwionej ſobie rádośći / podjął krzyż / wzgárdźiwƺy ſromotę / y uśiádł ná práwicy ſtolicy Bożej.
Przetoż uwáżájćie / jáki jeſt ten / który podejmowáł tákowe od grzeƺników przećiwko ſobie ſprzećiwiánie / ábyśćie oſłábiwƺy w umyſłách wáƺych / nie uſtáwáli.
Jeƺcżeśćie śię áż do krwi nie ſprzećiwili / wálcżąc przećiwko grzechowi.
Cżyliśćie zápámiętáli nápominániá / które wám jáko ſynom mówi : Synu mój / nie lekce ſobie powáżáj káźni Páńſkiej / á nie tráć ſercá / gdy od niego bywáƺ kárány ;
Abowiem kogo Pán miłuje / tego kárze / á ſmágá káżdego / którego zá ſyná przyjmuje.
Jeźli znośićie káránie / Bóg śię wám ofiáruje jáko ſynom ; ábowiem któryż jeſt ſyn / któregoby ojćiec nie káráł?
A jeźli jeſteśćie bez kárániá / którego wƺyſcy ſą ucżeſnikámi / tedyśćie bękártámi á nie ſynámi.
A nádto ćieleſnych ojców náƺych mieliśmy / którzy náſ káráli / á báliśmy śię ich ; záż dáleko więcej nie mámy być poddáni Ojcu duchów / ábyśmy żyli?
Abowiem oni ná máło dni / jáko śię im zdáło / náſ káráli ; ále ten ku pożytkowi náƺemu ná to / ábyśmy byli ucżeſnikámi świętobliwośći jego.
A wƺelkie káránie / gdy przytomne jeſt / nie zdá śię być weſołe / ále ſmutne ; lecż potem owoc ſpráwiedliwośći ſpokojny przynośi tym / którzy ſą przez nie wyćwicżeni.
Przeto opuƺcżone ręce / y zemdlone koláná wyproſtujćie /
A cżyńćie koleje proſte nogámi wáƺemi / iżby to / co jeſt chromego / z drogi nie uſtąpiło / ále rácżej uzdrowione było.
Pokoju náśládujćie ze wƺyſtkimi y świętobliwośći / bez której żáden nie oglądá Páná ;
Upátrując / żeby kto nie odpádł od łáſki Bożej / á żeby który korzeń gorzkośći nie podróſł / á nie przekáźił / y przezeń áby śię ich wiele nie pokáláło ;
Aby kto nie był wƺetecżnym / álbo ſproſnym / jáko Ezáw / który zá potráwę jednę ſprzedáł pierworodztwo ſwoje.
Abowiem wiećie / iż y potem / gdy chćiáł odźiedźicżyć błogoſłáwieńſtwo / był odrzucony ; bo nie ználázł miejſcá pokuty / choć jej z płácżem ƺukáł.
Bośćie nie przyſtąpili do góry / którá śię dá dotknąć / y do ogniá gorejącego / y do wichru / y do ćiemnośći y do burzy /
Y do dźwięku trąby / y do głoſu ſłów / który ći / co ſłyƺeli / prośili / áby więcej do nich nie mówiono ;
( Abowiem nie mogli znieść tego / co im rozkázywáno : Gdyby śię y bydlę góry dotknęło / będźie ukámionowáne / álbo poćiſkiem przebite.
A ták ſtráƺne to było / co widźieli / że też Mojżeƺ rzekł : Uląkłem śię y drżę. )
Aleśćie przyſtąpili do góry Syon / y do miáſtá Bogá żywego / do Jeruzálemu niebieſkiego / y do niezlicżonych tyśięcy Aniołów ;
Do wálnego zgromádzeniá / y do zebrániá pierworodnych / którzy ſą ſpiſáni w niebie / y do Bogá / ſędźiego wƺyſtkich / y do duchów ſpráwiedliwych y doſkonáłych ;
Y do pośredniká nowego teſtámentu / JEzuſá / y do krwi pokropieniá / lepƺe rzecży mówiącej niż Ablowá.
Pátrzájćież / ábyśćie nie gárdźili tym / który mówi ; ábowiem jeźliż oni nie uƺli / którzy gárdźili tym / który ná źiemi ná miejſcu Bożem mówił / dáleko więcej my / jeźliże śię od tego / który z niebá jeſt / odwróćimy ;
Którego głoſ ná on cżáſ poruƺył był źiemią / á teráz obiecáł / mówiąc : Jeƺcże já ráz poruƺę nie tylko źiemią / ále y niebem.
A to też mówi : Jeƺcże ráz / pokázuje znieśienie rzecży chwiejących śię / jáko tych / które ſą ucżynione / áby zoſtáwáły te / które śię nie chwieją.
Przetoż przyjmując króleſtwo nie chwiejące śię / miejmy łáſkę / przez którą ſłużymy przyjemnie Bogu ze wſtydem y z ucżćiwośćią.
Abowiem Bóg náƺ jeſt ogniem tráwiącym.
