Mówię tedy : ( bráćiá! ) Pokąd dźiedźic jeſt dźiećięćiem / nic nie jeſt różny od ſługi / pánem będąc wƺyſtkiego ;
Ale jeſt pod opiekunámi y dozorcámi áż do cżáſu zámierzeniá ojcowſkiego.
Tákże y my / gdyśmy byli dźiećmi / pod żywioły świátá byliśmy zniewoleni.
Lecż gdy przyƺło wypełnienie cżáſu / poſłáł Bóg onego Syná ſwego / który śię urodźił z niewiáſty / który śię ſtáł pod zákonem /
Aby tych / którzy pod zákonem byli / wykupił / żebyśmy práwá przyſpoſobieniá zá ſynów doſtąpili.
A iżeśćie ſynowie / przetoż poſłáł Bóg Duchá Syná ſwego w ſercá wáƺe / wołájącego Abbá / to jeſt / Ojcże.
A ták już więcej nie jeſteś niewolnikiem / ále ſynem ; á poniewáż ſynem / tedy y dźiedźicem Bożym przez CHryſtuſá.
Aleć náoncżáſ nie znájąc Bogá / ſłużyliśćie tym / którzy z przyrodzeniá nie ſą bogowie.
A teráz poznáwƺy Bogá / owƺem y poznáni będąc od Bogá / jákoż śię záś názád wrácáćie ku żywiołom mdłym y mizernym / którym záśię znowu ſłużyć chcećie?
Przeſtrzegáćie dni y mieśiące / y cżáſy y látá.
Boję śię o wáſ / bym ſnáć dármo nie prácowáł około wáſ.
Bądźćie jáko já / gdyżem y já jeſt jáko wy / bráćiá! proƺę wáſ. W nicżemeśćie mnie nie ukrzywdźili.
Bo wiećie / żem w ſłábośći ćiáłá wám z przodku Ewángieliję opowiádáł.
A pokuƺeniá mego / które było w ćiele mojem / nie lekceśćie ſobie wáżyli / ániśćie nim gárdźili / áleśćie mię jáko Aniołá Bożego przyjęli / y jáko CHryſtuſá JEzuſá.
Jákież tedy było błogoſłáwieńſtwo wáƺe? ábowiem wám dáję świádectwo / iż / by byłá rzecż możná / dálibyśćie mi byli wyłupiwƺy ocży wáƺe.
Tákżem śię ſtáł nieprzyjáćielem wáƺym / práwdę wám mówiąc?
Páłáją ku wám miłośćią nie dobrze / owƺem chcą náſ odſtrychnąć / ábyśćie ich miłowáli.
A dobráć rzecż / páłáć miłośćią w dobrem záwƺe / á nie tylko / gdym jeſt obecnym u wáſ.
Dźiátki moje! ( które znowu z boleśćią rodzę / áżby CHryſtuſ był wykƺtáłtowány w wáſ / )
Chćiáłbym teráz być u wáſ / y odmienić głoſ mój / poniewáż wątpię o wáſ.
Powiedzćie mi / którzy pod zákonem chcećie być / nie ſłucháćież zákonu?
Abowiem nápiſáne / iż Abráhám miáł dwóch ſynów / jednego z niewolnicy / á drugiego z wolnej.
Lecż ten / który był z niewolnicy / według ćiáłá śię urodźił / á który z wolnej / według obietnicy.
Przez co znácżą śię inƺe rzecży ; ábowiem te ſą one dwá teſtámenty ; jeden z góry Synájſkiej / który rodźi w niewolę ; á ten jeſt jáko Agár.
Abowiem Agár jeſt górá Synáj w Arábii / á ſtoſuje śię do niej teráźniejƺe Jeruzálem ; bo jeſt w niewoli z dźiátkámi ſwojimi.
Lecż ono górne Jeruzálem wolne jeſt / które jeſt mátką wƺyſtkich náſ.
Abowiem nápiſáno : Rozweſel śię niepłodná / którá nie rodźiƺ ; porwij śię á záwołáj / którá nie prácujeƺ w porodzeniu ; bo tá opuƺcżoná wiele má dźiátek / więcej niż tá / którá má mężá.
My tedy / bráćiá! ták jáko Izáák / jeſteśmy dźiátkámi obietnicy.
Ale jáko ná on cżáſ ten / który śię był urodźił według ćiáłá / prześládowáł tego / który śię był urodźił według duchá / ták śię dźieje y teráz.
Ale co mówi Piſmo? Wyrzuć niewolnicę y ſyná jej ; ábowiem nie będźie dźiedźicżył ſyn niewolnicy z ſynem wolnej /
A ták / bráćiá! nie jeſteśmy dźiećmi niewolnicy / ále wolnej.
