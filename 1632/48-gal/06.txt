Bráćiá! jeźliby też cżłowiek záchwycony był w jákim upádku / wy duchowni : nápráwiájćie tákiego w duchu ćichośći / upátrując káżdy ſámego śiebie / ábyś y ty nie był kuƺony.
Jedni drugich brzemioná nośćie / á ták wypełniájćie zákon Chryſtuſowy.
Abowiem jeźli kto mniemá / żeby cżem był / nie będąc nicżem / tákiego záwodzi włáſny umyſł jego.
Ale káżdy niecháj włáſnego ſwego ucżynku doświádcżá / á tedy ſám w ſobie chwáłę mieć będzie / á nie w drugim.
Abowiem káżdy ſwoje włáſne brzemię ponieśie.
A niech udzielá ten / który bywá náucżány w ſłowie / temu / który go náucżá / ze wƺyſtkich dóbr.
Nie błądźćie ; Bóg śię nie dá z śiebie náśmiewáć ; ábowiem cobykolwiek śiáł cżłowiek / to też żąć będzie.
Bo kto śieje ćiáłu ſwemu / z ćiáłá żąć będzie ſkáżenie ; ále kto śieje duchowi / z duchá żąć będzie żywot wiecżny.
A dobrze cżyniąc nie ſłábiejmy ; ábowiem cżáſu ſwojego żąć będziemy nie uſtáwájąc.
Przeto tedy / póki cżáſ mámy / dobrze cżyńmy wƺyſtkim / á nájwięcej domownikom wiáry.
Widzićie / jákim długi liſt wám nápiſáł ręką moją.
Którzykolwiek chcą być pozorni według ćiáłá / ći wáſ przymuƺáją / ábyśćie śię obrzezáli / tylko áby dlá krzyżá Chryſtuſowego prześládowániá nie ćierpieli.
Abowiem y ći / którzy śię obrzezują / nie záchowywują zákonu ſámi / ále chcą / ábyśćie śię wy obrzezáli / żeby śię z ćiáłá wáƺego chlubili.
Ale já / nie dáj Boże / ábym śię miáł chlubić / tylko w krzyżu Páná náƺego Jezuſá Chryſtuſá / przez którego mi jeſt świát ukrzyżowány / á já świátu.
Abowiem w Chryſtuśie Jezuśie áni obrzezká nic nie wáży / áni nieobrzezká / ále nowe ſtworzenie.
A którzykolwiek według tego ƺnuru poſtępowáć będą / pokój ná nich przyjdzie y miłośierdzie / y ná lud Boży Izráelſki.
Ná oſtátek niecháj mi nikt trudnośći nie zádáje ; ábowiem já piątná Páná Jezuſowe noƺę ná ćiele mojem.
Łáſká Páná náƺego Jezuſá Chryſtuſá niech będzie z duchem wáƺym / bráćiá! Amen.
