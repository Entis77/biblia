Potem po cżternáſtu látách wſtąpiłem záśię do Jeruzálemu z Bárnábáƺem / wźiąwƺy z ſobą y Tytuſá.
A wſtąpiłem według objáwieniá / y przełożyłem im Ewángieliję / którą káżę między pogánámi á zwłáƺcżá zácniejƺym / bym ſnáć nádáremno nie bieżáł / álbo przedtem nie biegáł.
Ale áni Tytuſ / który zemną był / będąc Grekiem / nie był przymuƺony / obrzezáć śię /
A to dlá wprowádzonych fáłƺywych bráći / którzy śię wkrádli / áby wyƺpiegowáli wolność náƺę / którą mámy w CHryſtuśie JEzuśie / áby náſ w niewolę podbili.
Którymeśmy y ná chwilkę nie uſtąpili y nie poddáli śię / áby u wáſ práwdá Ewángielii zoſtáłá.
A od tych / którzy śię zdádzą być cżemś / ( ácż jákimi niekiedy byli / nic mi ná tem ; bo oſoby cżłowiecżej Bóg nie przyjmuje / ) ći mówię / którzy śię zdáli być cżemś / nic mi nie przydáli.
Owƺem / przećiwnym obycżájem / widząc / iż mi jeſt zwierzoná Ewángielijá między nieobrzezánymi / jáko Piotrowi między obrzezánymi /
( Abowiem ten / który był ſkutecżny przez Piotrá w ápoſtolſtwie między obrzezánymi / ſkutecżny był y we mnie między pogánámi. )
Y poznáwƺy łáſkę mnie dáną / Jákób y Kiefáſ y Ján / którzy śię zdádzą być filárámi / podáli práwicę mnie y Bárnábáƺowi ná towárzyſtwo / ábyśmy my między pogánámi / á oni między obrzezánymi kázáli.
Tylko upomnieli / ábyśmy ná ubogich pámiętáli / o com śię też pilnie ſtáráł / ábym to ucżynił.
A gdy przyƺedł Piotr do Antyjochii / ſprzećiwiłem śię mu w twárz ; y był godźien nágány.
Abowiem przedtem / niż przyƺli niektórzy od Jákóbá / weſpół z pogánámi jádáł ; á gdy ći przyƺli / ſchrániáł śię y odłącżáł / bojąc śię tych / którzy byli z obrzezániá.
A weſpół z nim obłudnie śię obchodźili y drudzy Żydźi / ták / że y Bárnábáƺ uwiedźiony był tą ich obłudą.
Ale gdym obácżył / iż nie proſto chodzą w práwdźie Ewángielii / rzekłem Piotrowi przed wƺyſtkimi : Poniewáż ty / będąc Żydem / po pogáńſku żyjeƺ á nie po żydowſku / cżemuż pogán przymuƺáƺ / po żydowſku żyć?
My / którzyśmy z przyrodzeniá Żydowie á nie z pogán grzeƺnicy /
Wiedząc / iż nie bywá uſpráwiedliwiony cżłowiek z ucżynków zákonu / ále przez wiárę w JEzuſá CHryſtuſá / y myśmy w JEzuſá CHryſtuſá uwierzyli / ábyśmy byli uſpráwiedliwieni z wiáry CHryſtuſowej / á nie z ucżynków zákonu / przeto / że nie będźie uſpráwiedliwione z ucżynków zákonu żádne ćiáło.
A jeźli my ƺukájąc / ábyśmy byli uſpráwiedliwieni w CHryſtuśie / znájdujemy śię też grzeƺnikámi / tedyć CHryſtuſ jeſt ſługą grzechu? Nie dáj tego Boże!
Abowiem jeźli to / com zburzył / znowu záśię buduję / przeſtępcą ſámego śiebie cżynię.
Bom já przez zákon zákonowi umárł / ábym żył Bogu.
Z CHryſtuſem jeſtem ukrzyżowány / á żyję już nie já / lecż żyje we mnie CHryſtuſ ; á to że teráz w ćiele żyję / w wierze Syná Bożego żyję / który mię umiłowáł / y wydáł ſámego śiebie zá mię.
Nie odrzucám tej łáſki Bożej ; bo jeźli przez zákon jeſt ſpráwiedliwość / tedyć CHryſtuſ próżno umárł.
