Tedy oblicżył Dawid lud / który miáł z ſobą / á poſtánowił nád nimi hetmány / y rotmiſtrze.
Y porucżył Dawid ludu trzećią cżęść pod rękę Joábowę / á trzećią cżęść pod rękę Abiſájá / ſyná Sárwii / brátá Joábowego / á trzećią cżęść pod rękę Itájá Gietejcżyká ; y rzekł król do ludu : Wynijdę y já tákże z wámi.
Ale lud rzekł : Nie wynijdzieƺ ; bo jeźlibyśmy my tył podáli / oni máło dbáć o náſ będą / choćby też náſ poległá połowá / máło dbáć o náſ będą ; ábowiemeś ty ſám jáko náſ dzieśięć tyśięcy. Przetoż teráz lepiej / ábyś nám był w mieśćie ná pomocy.
Y rzekł do nich król : Co śię wám zdá dobrego / to ucżynię. Tedy ſtáł król przy brámie / á wƺyſtek lud wychodził po ſtu y po tyśiącu.
Y rozkázáł król Joábowi / y Abiſájowi / y Itájowi / mówiąc : Łáſkáwie mi śię obchodźćie z ſynem moim Abſálomem. A wƺyſtek lud ſłyƺł / gdy przykázywáł król wƺyſtkim hetmánom o Abſálomie.
A ták wyćiągnął lud w pole przećiw Izráelowi / y zwiedli bitwę w leśie Efráim.
Támże poráżon jeſt lud Izráelſki od ſług Dawidowych ; y ſtáłá śię tám poráżká wielká dniá onego / á poległo ich dwádzieśćiá tyśięcy.
Bo gdy byłá bitwá rozproƺoná po wƺyſtkiej ziemi / więcej láſ pogubił ludu / niż ich miecż pożárł dniá onego.
Y nápádł Abſálom ná ſługi Dawidowe ; á Abſálom jecháł ná mule / y wbieżáł z nim muł pod gęſty á wielki dąb / y uwięzłá głowá jego ná dębie / y záwiſł między niebem y między ziemią ; ále muł / który był pod nim / wybiegł.
Co ujrzáwƺy mąż niektóry / oznájmił Joábowi / mówiąc : Otom widziáł Abſálomá wiƺącego ná dębie.
Tedy rzekł Joáb mężowi / który mu to oznájmił : Jeźliś widziáł / á cżmużeś go tám nie zábił y nie zrzućił ná ziemię? A jábym ći był powinien dáć dzieśięć ſrebrników / y jeden páſ rycerſki.
Y odpowiedziáł on mąż Joábowi : A já choćbym miáł odwáżonych ná rękách mych tyśiąc ſrebrników / nie podnióſłbym ręki mojej ná ſyná królewſkiego ; bośmy ſłyƺeli / gdy przykázáł król tobie y Abiſájowi y Itájowi / mówiąc : Ochrániájćie wƺyſcy ſyná mego Abſálomá.
Chybá / żebym chćiáł wdáć duƺę moję w niebezpiecżeńſtwo ; bo nie bywá nic zátájono przed królem ; y ty ſám byłbyś przećiwko mnie.
Tedy rzekł Joáb : Nie będęć śię já tu báwił z tobą ; przetoż wziąwƺy trzy drzewcá w rękę ſwoję / wráził je w ſerce Abſálomowe / gdy jeƺcże żyw był ná dębie.
A obſkocżywƺy Abſálomá dzieśięć ſług / którzy nośili broń Joábowę / bili / y zábili go.
Wtem zátrąbił Joáb w trąbę / y wróćił śię lud z pogoni zá Izráelem ; bo Joáb zátrzymáł lud.
A wziąwƺy Abſálomá / wrzućili go w tymże leśie w dół wielki / y nánośili náń bárdzo wielką kupę kámieniá. Ale wƺyſtek Izráel ućiekł / káżdy do námiotów ſwoich.
A Abſálom wziął był / y wyſtáwił ſobie zá żywotá ſwego ſłup / który jeſt w dolinie królewſkiej ; bo mówił : Niemám ſyná ; jednák zoſtáwię pámiątkę imieniá mego. Przetoż názwáł on ſłup imieniem ſwojim / który zowią miejſce Abſálomowe áż do dziśiejƺego dniá.
Tedy Achimááſ / ſyn Sádoká / rzekł : Proƺę niech idę á oznájmię królowi ućieƺną nowinę / iż go wybáwił Pán z ręki nieprzyjáćiół jego.
Ale mu rzekł Jáob : Nie byłbyś wdzięcżnym poſłem dziśiáj ; lecż to opowieƺ dniá drugiego / á dziś nie dáwáj o tem znáć / przeto iż ſyn królewſki zginął.
Potem Joáb rzekł do Chuſego : Idź / oznájmij królowi / coś widziáł. A ták ukłoniwƺy śię Chuſy Joábowi / bieżáł.
Y mówił powtóre Achimááſ / ſyn Sádoká / y rzekł do Joábá : Bądź co bądź / proƺę niech y já bieżę zá Chuſym. Y rzekł Joáb : Przecżbyś ty miáł bieżeć / ſynu mój / gdyż niemáƺ / cobyś dobrego zwiáſtowáł?
Y rzekł : Bądź co bądź / pobieżę. Y rzekł mu Joáb : Bieżże. A ták bieżáł Achimááſ proſtƺą drogą / y uprzedził Chuſego.
A Dawid śiedziáł między dwiemá brámámi. Y wyƺedł ſtróż ná dách brámy ná mur / á podnióſłƺy ocży ſwe / ujrzáł mężá jednego bieżącego.
Tedy záwołáwƺy ſtróż / opowiedziáł to królowi. Y rzekł król : Jeźliżeć ſám jeſt / dobre poſelſtwo w uſtách jego. A gdy ten ſpieƺnie ƺedł / y przybliżáł śię /
Ujrzáł ſtróż y drugiego mężá bieżącego / y záwołáł ſtróż ná wrotnego / mówiąc : Oto y drugi mąż bieży ſám. Y rzekł król : Y ten dobre poſelſtwo nieśie.
Nádto rzekł ſtróż : Zdá mi śię bieg pierwƺego / jáko bieg Achimááſá / ſyná Sádokowego. Y rzekł król : Mąż to dobry / y z dobrem poſelſtwem idzie.
Tedy záwołáł Achimááſ / y rzekł do królá : Pokój ; y ukłonił śię królowi twárzą ſwoją ku ziemi / y rzekł : Błogoſłáwiony Pán Bóg twój / któryć podáł te męże / co podnieśli ręce ſwe przećiw królowi / pánu memu.
Y rzekł król : Jákoli śię má ſyn mój Abſálom? Tedy Achimááſ odpowiedziáł : Widziáłem zámięƺánie wielkie / gdy poſyłáł ſługę królewſkiego Joáb / y mnie / ſługę twego ; ále nie wiem co było.
Potem rzekł król : Odſtąp / á ſtáń tám ; á on odſtąpiwƺy ſtánął.
A wtem Chuſy przyƺedł y rzekł : Opowiádá śię królowi / pánu memu / że ćię wybáwił Pán dziśiáj z ręki wƺyſtkich / którzy powſtáli przećiwko tobie.
Y rzekł król do Chuſego : A jákoli śię má ſyn mój Abſálom? Odpowiedziáł Chuſy : Bodáj ták byli nieprzyjáćiele królá / páná mego / y wƺyſcy / którzy powſtáwáją przećiw tobie ná złe / jáko ſyn twój!
Tedy śię záſmućił król / y wſtąpił ná ſálę onej brámy / á płákáł / y ták mówił idąc : Synu mój Abſálomie / ſynu mój! Synu mój Abſálomie! obym já był umárł miáſto ćiebie! Abſálomie / ſynu mój / ſynu mój!
