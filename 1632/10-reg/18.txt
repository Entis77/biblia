Tedy oblicżył Dawid lud / który miał z ſobą / á poſtánowił nád nimi hetmány / y rotmiſtrze.
Y porucżył Dawid ludu trzećią cżęść pod rękę Joábowę / á trzećią cżęść pod rękę Abiſájá / ſyná Sárwii / brátá Joábowego / á trzećią cżęść pod rękę Itájá Gietejcżyká ; y rzekł król do ludu : Wynijdę y ja tákże z wámi.
Ale lud rzekł : Nie wynijdźieƺ ; bo jeſlibyſmy my tył podáli / oni máło dbáć o nas będą / choćby też nas poległá połowá / máło dbáć o nas będą ; ábowiemeś ty ſám jáko nas dźieśięć tyśięcy. Przetoż teraz lepiej / ábyś nam był w mieśćie ná pomocy.
Y rzekł do nich król : Co śię wam zda dobrego / to ucżynię. Tedy ſtał król przy bramie / á wƺyſtek lud wychodźił po ſtu y po tyśiącu.
Y rozkázał król Joábowi / y Abiſájowi / y Itájowi / mówiąc : Łáſkáwie mi śię obchodźćie z ſynem moim Abſálomem. A wƺyſtek lud ſłyƺł / gdy przykázywał król wƺyſtkim hetmánom o Abſálomie.
A ták wyćiągnął lud w pole przećiw Izráelowi / y zwiedli bitwę w leśie Efráim.
Támże poráżon jeſt lud Izráelſki od ſług Dawidowych ; y ſtáłá śię tám poráżká wielka dniá onego / á poległo ich dwádźieśćiá tyśięcy.
Bo gdy byłá bitwá rozproƺoná po wƺyſtkiej źiemi / więcey lás pogubił ludu / niż ich miecż pożárł dniá onego.
Y nápádł Abſálom ná ſługi Dawidowe ; á Abſálom jechał ná mule / y wbieżał z nim muł pod gęſty á wielki dąb / y uwięzłá głowá jego ná dębie / y záwiſł miedzy niebem y miedzy źiemią ; ále muł / który był pod nim / wybiegł.
Co ujrzáwƺy mąż niektóry / oznájmił Joábowi / mówiąc : Otom widźiał Abſálomá wiƺącego ná dębie.
Tedy rzekł Joáb mężowi / który mu to oznájmił : Jeſliś widźiał / á cżmużeś go tám nie zábił y nie zrzućił ná źiemię? A jábym ći był powinien dáć dźieśięć ſrebrników / y jeden pás rycerſki.
Y odpowiedźiał on mąż Joábowi : A ja choćbym miał odwáżonych ná rękách mych tyśiąc ſrebrników / nie podnióſłbym ręki mojey ná ſyná królewſkiego ; bośmy ſłyƺeli / gdy przykázał król tobie y Abiſájowi y Itájowi / mówiąc : Ochrániájćie wƺyſcy ſyná mego Abſálomá.
Chybá / żebym chćiał wdáć duƺę moję w niebezpiecżeńſtwo ; bo nie bywa nic zátájono przed królem ; y ty ſám byłbyś przećiwko mnie.
Tedy rzekł Joáb : Nie będęć śię ja tu báwił z tobą ; przetoż wźiąwƺy trzy drzewcá w rękę ſwoję / wráźił je w ſerce Abſálomowe / gdy jeƺcże żyw był ná dębie.
A obſkocżywƺy Abſálomá dźieśięć ſług / którzy nośili broń Joábowę / bili / y zábili go.
Wtem zátrąbił Joáb w trąbę / y wróćił śię lud z pogoni zá Izráelem ; bo Joáb zátrzymał lud.
A wźiąwƺy Abſálomá / wrzućili go w tymże leśie w dół wielki / y nánośili nań bárzo wielką kupę kámieniá. Ale wƺyſtek Izráel ućiekł / káżdy do námiotów ſwojich.
A Abſálom wźiął był / y wyſtáwił ſobie zá żywotá ſwego ſłup / który jeſt w dolinie królewſkiej ; bo mówił : Niemám ſyná ; jednák zoſtáwię pámiątkę imienia mego. Przetoż názwał on ſłup imieniem ſwojim / który zowią miejſce Abſálomowe áż do dźiśiejƺego dniá.
Tedy Achimáás / ſyn Sádoká / rzekł : Proƺę niech idę á oznájmię królowi ućieƺną nowinę / iż go wybáwił Pán z ręki nieprzyjáćiół jego.
Ale mu rzekł Jáob : Nie byłbyś wdźięcżnym poſłem dźiśiáj ; lecż to opowieƺ dniá drugiego / á dźiś nie dáwáj o tym znáć / przeto iż ſyn królewſki zginął.
Potem Joáb rzekł do Chuſego : Idź / oznájmij królowi / coś widźiał. A ták ukłoniwƺy śię Chuſy Joábowi / bieżał.
Y mówił powtóre Achimáás / ſyn Sádoká / y rzekł do Joábá : Bądź co bądź / proƺę niech y ja bieżę zá Chuſym. Y rzekł Joáb : Przecżbyś ty miał bieżeć / ſynu mój / gdyż niemáƺ / cobyś dobrego zwiáſtował?
Y rzekł : Bądź co bądź / pobieżę. Y rzekł mu Joáb : Bieżże. A ták bieżał Achimáás proſtƺą drogą / y uprzedźił Chuſego.
A Dawid śiedźiał miedzy dwiemá bramámi. Y wyƺedł ſtróż ná dách bramy ná mur / á podnióſƺy ocży ſwe / ujrzał mężá jednego bieżącego.
Tedy záwołáwƺy ſtróż / opowiedźiał to królowi. Y rzekł król : Jeſliżeć ſám jeſt / dobre poſelſtwo w uſtách jego. A gdy ten ſpieƺnie ƺedł / y przybliżał śię /
Ujrzał ſtróż y drugiego mężá bieżącego / y záwołał ſtróż ná wrotnego / mówiąc : Oto y drugi mąż bieży ſám. Y rzekł król : Y ten dobre poſelſtwo nieśie.
Nádto rzekł ſtróż : Zda mi śię bieg pierwƺego / jáko bieg Achimááſá / ſyná Sádokowego. Y rzekł król : Mąż to dobry / y z dobrem poſelſtwem idźie.
Tedy záwołał Achimáás / y rzekł do królá : Pokój ; y ukłonił śię królowi twárzą ſwoją ku źiemi / y rzekł : Błogoſłáwiony Pán Bóg twój / któryć podał te męże / co podnieśli ręce ſwe przećiw królowi / pánu memu.
Y rzekł król : Jákoli śię ma ſyn mój Abſálom? Tedy Achimáás odpowiedźiał : Widźiałem zámięƺánie wielkie / gdy poſyłał ſługę królewſkiego Joáb / y mnie / ſługę twego ; ále nie wiem co było.
Potem rzekł król : Odſtąp / á ſtáń tám ; á on odſtąpiwƺy ſtánął.
A wtem Chuſy przyƺedł y rzekł : Opowiádá śię królowi / pánu memu / że ćię wybáwił Pán dźiśiáj z ręki wƺyſtkich / którzy powſtáli przećiwko tobie.
Y rzekł król do Chuſego : A jákoli śię ma ſyn mój Abſálom? Odpowiedźiał Chuſy : Bodáj ták byli nieprzyjáćiele królá / páná mego / y wƺyſcy / którzy powſtáwáją przećiw tobie ná złe / jáko ſyn twój!
Tedy śię záſmućił król / y wſtąpił ná ſálę oney bramy / á płákał / y ták mówił idąc : Synu mój Abſálomie / ſynu mój! Synu mój Abſálomie! obym ja był umárł miáſto ćiebie! Abſálomie / ſynu mój / ſynu mój!
