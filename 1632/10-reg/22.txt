Y mówił Dawid Pánu ſłowá tey pieśni w on dźień / gdy go wybáwił Pán z rąk wƺyſtkich nieprzyjáćiół jego y z ręki Saulowej ;
Y rzekł : Pán opoká mojá y twierdzá mojá / y wybáwićiel mój ze mną.
Bóg / ſkáłá mojá / w nim będę ufáł / tárcż mojá / róg zbáwieniá mego / podwyżƺenie moje / y ućiecżká mojá / zbáwićiel mój / który mię od gwáłtu wybáwiá.
Wzywáłem Páná chwáły godnego / á od nieprzyjáćiół moich byłem wybáwiony.
Abowiem ogárnęły mię były boleśći śmierći / potoki niezbożnych przeſtráƺyły mię.
Boleśći grobu ogárnęły mię / záchwyćiły mię śidłá śmierći.
W utrápieniu mojem wzywáłem Páná / á do Bogá mego wołáłem / y wyſłucháł z kośćiołá ſwego głoſ mój / á wołánie moje przyƺło do uƺów jego.
Tedy śię wzruƺyłá / á zádrżáłá źiemiá / á fundámenty niebá zátrząſnęły / y wzruƺyły śię dla gniewu jego.
Wyſtąpił dym z nózdrz jego / á ogień z uſt jego pożerájący ; węgle rozpáliły śię od niego.
Nákłonił niebioſ y zſtąpił / á ćiemność byłá pod nogámi jego.
Y jeźdźił ná Cherubinách / y látáł / y widźián jeſt ná ſkrzydłách wiátrowych.
Położył ćiemność około śiebie miáſto przybytku / zgromádzenie wód z obłoki niebieſkimi.
Od jáſnośći oblicżá jego rozpáliły śię węgle ogniſte.
Zágrzmiáł Pán z niebá / á nájwyżƺy wydáł głoſ ſwój.
Wypuśćił y ſtrzáły / á rozproƺył je / y błyſkáwicą potárł je.
Y okázáły śię głębokośći morſkie / á odkryły śię grunty świátá ná fukánie Páńſkie / ná tchnienie Duchá z nózdrz jego.
Poſłáwƺy z wyſokośći / przyjął mię / wyrwáł mię z wód wielkich.
Wybáwił mię od nieprzyjáćielá mego potężnego / od tych / którzy mię mieli w nienáwiśći / choć byli mocniejƺymi nád mię.
Uprzedźili mię w dźień utrápieniá mego ; ále Pán był podporą moją.
Y wywiódł mię ná przeſtrzeńſtwo ; wybáwił mię ; bo mię ſobie upodobáł.
Oddáł mi Pán według ſpráwiedliwośći mojey / według cżyſtośći rąk moich oddáł mi /
Gdyżem ſtrzegł dróg Páńſkich / ánim niezbożnie nie odſtáwáł od Bogá mego.
Abowiem wƺyſtkie ſądy jego ſą przed oblicżem mojem y uſtáwy jego / á nie odſtąpiłem od nich.
A będąc doſkonáły przed nim / wyſtrzegáłem śię niepráwośći mojey.
Przetoż oddáł mi Pán według ſpráwiedliwośći mojey / według cżyſtośći mojey przed oblicżnośćią ocżu ſwych.
Z miłośiernym miłośiernie poſtępujeƺ / z mężem doſkonáłym doſkonáłym jeſteś.
Z cżyſtym cżyſty jeſteś / á z przewrotnym ſurowie śię obchodźiƺ.
Ale wybáwiáƺ lud ubogi / á ocży twoje przed wynioſłymi opuƺcżáƺ.
Tyś záiſte pochodnią moją / o Pánie / á Pán oświeći ćiemnośći moje.
Bo w tobie przebiegłem wojſko / w Bogu moim przeſkocżyłem mur.
Drogá Bożá jeſt doſkonáłá / wyrok Páńſki náder cżyſty / tárcżą jeſt wƺyſtkim / którzy w nim ufáją.
Abowiem któż jeſt Bogiem oprócż Páná? á kto opoką oprócż Bogá náƺego?
Bóg jeſt mocą moją w wojſku / on cżyni doſkonáłą drogę moję.
Równá nogi moje z jeleniemi / ná wyſokich miejſcách moich ſtáwiá mię.
Ćwicży ręce me do boju / ták że kruƺę łuk miedźiány rámiony ſwemi.
Abowiem dáłeś mi tárcż zbáwieniá twego / á w ćichośći twojey rozmnożyłeś mię.
Rozƺerzyłeś kroki moje podemną / ták iż śię nie záchwiáły koſtki moje.
Goniłem nieprzyjáćioły moje / y wytráćiłem je / á nie wróćiłem śię / áżem je wyplenił.
Y wyniƺcżyłem je / y poprzebijáłem je / ták iż nie powſtáną : upádli pod nogámi mojemi.
Tyś mię przepáſáł mocą ku bitwie / á powáliłeś pod mię powſtájące przećiwko mnie.
Nádto podáłeś mi ƺyję nieprzyjáćiół moich / którzy mię mieli w nienáwiśći / y wykorzeniłem je.
Poglądáli / ále nie był wybáwićiel ; wołáli ná Páná / ále ich nie wyſłucháł.
Y potárłem je jáko proch źiemi / jáko błoto ná ulicách podeptáwƺy je / rozmiotáłem je.
Tyś mię od ſporu ludu mego wyrwáł ; záchowáłeś mię / ábym był głową národów ; lud / któregom nie znáł / ſłuży mi.
Synowie obcy kłámáli mną / á ſkoro uſłyƺeli / byli mi poſłuƺnymi.
Synowie obcy opádli / á drżeli y w zámknieniu ſwem.
Żyje Pán / y błogoſłáwioná ſkáłá mojá ; niechże będźie wywyżƺony Bóg / opoká zbáwieniá mego.
Bóg jeſt / który mi dáwá pomſty / á podbijá národy pod mię.
Który mię wywodźi od nieprzyjáćiół moich / á nád tymi / którzy powſtáwáją przećiwko mnie / wywyżƺáƺ mię / od cżłowieká niepobożnego wybáwiáƺ mię.
Przetoż będę ćię wyznáwáł Pánie między národámi ; á imieniowi twemu śpiewáć będę.
On jeſt wieżą zbáwieniá królá ſwego / á cżyniący miłośierdźie nád pomázáńcem ſwoim Dawidem / y nád náśieniem jego áż ná wieki.
