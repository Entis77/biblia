Y ſtáło śię potem / że ſobie náſpráwiáł Abſálom wozów / y koni / y pięćdźieśiąt mężów / którzy chodźili przed nim.
Y wſtáwájąc ráno Abſálom ſtáwáł podle drogi u brámy / á káżdego mężá / májącego ſpráwę á idącego do królá ná ſąd / przyzywáł Abſálom do śiebie / y mówił : Z któregożeś ty miáſtá? A gdy mu odpowiedźiáł : Z jednego pokoleniá Izráelſkiego jeſt ſługá twój.
Mówił mu Abſálom : Oto / ſpráwá twojá dobrá jeſt / y ſpráwiedliwá ; ále niemáƺ / ktoby ćię wyſłucháł u królá.
Nádto mówił Abſálom : O ktoby mię poſtánowił ſędźią w tey źiemi! áby do mnie chodźił káżdy / któryby miał ſpráwę u ſądu / dopomógłbym mu do ſpráwiedliwośći.
A gdy kto przyſtąpił / y ukłonił mu śię / śćiągáł rękę ſwą / á ująwƺy go / cáłowáł go.
A toć cżynił Abſálom wƺyſtkiemu Izráelowi / który przychodźił ná ſądy do królá / y ukrádáł Abſálom ſercá mężów Izráelſkich.
Y ſtáło śię po cżterdźieſtu látách / że rzekł Abſálom do królá : Niech idę proƺę / á oddám ślub mój w Hebronie / którym poślubił Pánu.
Abowiem ślub poślubił ſługá twój / kiedym mieƺkáł w Gieſſur Syryjſkim / mówiąc : Jeſliże mię záśię kiedy przywróći Pán do Jeruzálemu / tedy ſłużyć będę Pánu.
Y rzekł mu król : Idź w pokoju. A on wſtáwƺy poƺedł do Hebronu.
Tedy rozeſłáł Abſálom ƺpiegi między wƺyſtkie pokoleniá Izráelſkie / áby rzekli : Skoro uſłyƺyćie głoſ trąby / mówćież : Króluje Abſálom w Hebronie.
A z Abſálomem poƺło było dwieśćie mężów z Jeruzálemu záproƺonych / którzy ƺli w proſtośći ſwojey / niewiedząc o nicżem.
Poſłáł też Abſálom po Achitofelá Gilonicżyká / rádcę Dawidowego / áby przyƺedł z miáſtá ſwego Gilo / gdy miał ſpráwowáć ofiáry. Y ſtáło śię ſprzyśiężenie wielkie / á lud śię ſchodźił / y przybywáło go Abſálomowi.
Potem przyƺedł poſeł do Dawidá / mówiąc : Obróćiło śię ſerce mężów Izráelſkich zá Abſálomem.
Tedy rzekł Dawid do wƺyſtkich ſług ſwoich / którzy z nim byli w Jeruzálemie : Wſtáńćie / á ućiekájmy ; inácżej nieuƺlibyśmy przed twárzą Abſálomową. Spieƺćież śię / ázáż ujdźiem / by śię ſnáć nie poſpieƺył / á nie zájecháł nas / y nie obálił ná nas złego / y nie wyśiekł miáſtá oſtrzem miecżá.
Y rzekli ſłudzy królewſcy do królá : Wƺyſtko / cokolwiek ſobie upodobá król / pán náƺ / oto ſłudzy twoi.
A ták wyƺedł król / y wƺyſtek dom jego pieƺo ; tylko zoſtáwił król dźieśięć niewiáſt záłożnic / áby ſtrzegły domu.
A gdy wyƺedł król y wƺyſtek lud pieƺo / ſtánęli ná jednem miejſcu z dáleká.
Wƺyſcy też ſłudzy jego ƺli przy nim / y wƺyſcy Cheretcżycy / y wƺyſcy Feletcżycy / y wƺyſcy Gietejcżycy / ƺeść ſet mężów / którzy byli przyƺli pieƺo z Giet / ƺli przed twárzą królewſką.
Tedy rzekł król do Itájá Gietejcżyká : Cżemuż y ty z námi idźieƺ? Wróć śię / á zoſtáń przy królu ; boś ty cudzoźiemiec / á nie długo wróćiƺ śię do miejſcá twego.
Nie dáwnoś przyƺedł / á dźiśbym ćię ruƺyć miał / ábyś z námi ƺedł? Gdyż ja idę / ſám niewiem dokąd ; wróćże śię / á odprowádź bráćią ſwoję : niech będźie z tobą miłośierdźie y práwdá.
Ale odpowiedźiáł Itáj królowi / mówiąc : Jáko żywy Pán / jáko żywy też król pán mój / że ná któremkolwiek miejſcu będźie król / pán mój / choć w śmierći / choć w żywoćie / tám też bądźie ſługá twój.
Y rzekł Dawid do Itájá : Pójdźże / á przejdź. Y przeƺedł Itáj Gietejcżyk / y wƺyſcy mężowie jego / y wƺyſtkie dźiátki / które były z nim.
Tedy wƺyſtká źiemiá płákáłá głoſem wielkim / y wƺyſtek lud / który przechodźił. A ták król przeƺedł przez potok Cedron / á wƺyſtek lud przeƺedł przećiw drodze ku puƺcży.
A oto y Sádok y wƺyſcy Lewitowie byli z nim / nioſąc ſkrzynię przymierzá Bożego / y poſtáwili ſkrzynię Bożą ; ƺedł też Abijátár / áż wƺyſtek on lud przeƺedł z miáſtá.
Y rzekł król do Sádoká : Odnieś záśię ſkrzynię Bożą do miáſtá. Jeſlić znájdę łáſkę w ocżách Páńſkich / przywróći mię záśię / á ukáże mi ją / y przbytek ſwój.
Ale jeſliby ták rzekł : Nie podobáƺ mi śię ; otom ja / niech mi ucżyni / co dobrego jeſt w ocżách jego.
Nádto rzekł król do Sádoká kápłáná : Izáliś nie jeſt widzącym? Wróćże śię do miáſtá w pokoju / y Achimááſ / ſyn twój / y Jonátán / ſyn Abijátárá / dwáj ſynowie wáśi / z wámi.
Oto / ja pomieƺkám w równinách ná puƺcży / póki nie przyjdźie od was poſelſtwo dáwájące mi znáć.
A ták odprowádźili záśię Sádok y Abijátár ſkrzynię Bożą do Jeruzálemu / y zoſtáli tám.
Ale Dawid ƺedł ná górę oliwną / wſtępując y płácżąc / májąc głowę przykrytą / y idąc boſo ; wƺyſtek też lud / który z nim był / zákryli káżdy głowę ſwoję / á ƺli wſtępując y płácżąc.
Tedy dáno znáć Dawidowi / mówiąc : Achitofel jeſt z tymi / którzy śię zbuntowáli z Abſálomem. Y rzekł Dawid : O Pánie / proƺę / obróć w głupſtwo rádę Achitofelowę.
Y ſtáło śię / gdy Dawid przyƺedł áż ná wierzch góry / áby śię tám pomodlił Bogu / oto / ſpotkáł śię z nim Chuſáj Aráchitá / miáwƺy rozdárte ƺáty ſwe / á proch ná głowie ſwojey.
Y rzekł mu Dawid : Jeſli pójdźieƺ ze mną / będźieƺ mi ćiężárem ;
Ale jeſli śię do miáſtá wróćiƺ / á rzecżeƺ do Abſálomá : Królu / ſługą twoim będę / bom był ſługą ojcá twego zdáwná / ále teraz jám ſługą twoim : tedy mi obróćiƺ wniwecż rádę Achitofelową.
Azáż tám nie będźie z tobą Sádoká y Abijátárá / kápłánów? Przetoż cokolwiek uſłyƺyƺ z domu królewſkiego / oznájmiƺ Sádokowi y Abijátárowi / kápłánom.
Są też tám z nimi dwáj ſynowie ich / Achimááſ / ſyn Sádoká / y Jonátán / ſyn Abijátárá / przez które dáćie mi znáć o wƺyſtkiem / co jedno uſłuƺyćie.
Szedł tedy Chuſáj przyjáćiel Dawidá do miáſtá / á Abſálom też wjecháł do Jeruzálemu.
