A gdy Dawid zƺedł trochę z wierzchu góry / oto Sybá / ſługá Mefiboſetá / záƺedł mu drogę z párą oſłów ośiodłánych / ná których było dwieśćie chlebów / y ſto wiązánek rodzynków / y ſto wiązánek fig / y łágiew winá.
Tedy rzekł król do Syby : Ná cóż to? Y odpowiedźiał Sybá : Oſły te dla cżeládźi królewſkiej / áby ná nich jeźdźiłá / á chleb y figi / áby jedli ſłudzy / á wino / áby pił / ktoby uſtał ná puƺcży.
Y rzekł mu król : A gdźież jeſt ſyn páná twego? Y odpowiedźiał Sybá królowi : Oto zoſtał w Jeruzálemie ; ábowiem mówił : Dźiś mi przywróći dom Izráelſki króleſtwo ojcá mego.
Zátem rzekł król do Syby : Oto twoje jeſt wƺyſtko / cokolwiek miał Mefiboſet. Y rzekł Sybá / pokłon ucżyniwƺy : Niech znájdę łáſkę przed ocżymá twemi / królu / pánie mój.
Y przyƺedł król Dawid áż do Báhurym / á oto / z támtąd mąż wyƺedł z rodu domu Saulowego / á imię jego było Semej / ſyn Giery ; który wyƺedƺy / idąc złorzecżył.
A ćiſkał kámieńmi ná Dawidá / y ná wƺyſtkie ſługi królá Dawidá / choć wƺyſtek lud / y wƺyſtko rycerſtwo ƺło po práwej ſtronie jego / y po lewej ſtronie jego.
Y ták mówił Semej / złorzecżąc mu : Wynijdź / wynijdź mężu krwi / y mężu niezbożny.
Obróćił ná ćię Pán wƺyſtkę krew domu Saulowego / ná któregoś miejſcu królował / á podał Pán króleſtwo w ręce Abſálomá / ſyná twego ; á otoś ty we złem twojim / boś jeſt mężem krwi.
Y rzekł Abiſáj / ſyn Sárwii / do królá : Cżemuż złorzecży ten zdechły pies królowi / pánu memu? Niech idę proƺę / á utnę głowę jego.
Ale król rzekł : Cóż wam do tego / ſynowie Sárwii / że złorzecży? Ponieważ mu Pán rzekł : Złorzecż Dawidowi / y któżby śmiał rzec : Cżemu ták cżyniƺ?
Nádto rzekł Dawid do Abiſájego y do wƺyſtkich ſług ſwojich : Oto ſyn mój / który wyƺedł z żywotá mego / ƺuká duƺy mojey / jákoż dáleko więcey teraz ſyn Jemini? Zániechájćie go / niech złorzecży ; boć mu Pán rozkázał.
Snáć wejrzy Pán ná utrápienie moje / á oddá mi Pán dobrem zá złorzecżeniá jego dźiśiejƺe.
A ták ƺedł Dawid / y mężowie jego drogą / á Semej ƺedł ſtroną góry przećiwko niemu / á idąc złorzecżył / y ćiſkał kámieńmi przećiw niemu / y miotał prochem.
Y przyƺedł król ze wƺyſtkim ludem / który był przy nim ſprácowány / y támże odpocżął.
Lecż Abſálom y wƺyſtek lud mężów Izráelſkich / przyƺli do Jeruzálemu / tákże y Achitofel z nim.
A gdy przyƺedł Chuſáj Aráchitá / przyjáćiel Dawidá / do Abſálomá / rzekł Chuſáj do Abſálomá : Niech żyje król / niech żyje król!
Tedy rzekł Abſálom do Chuſájá : A tákáż to miłość twojá ku przyjáćielowi twemu? przecżżeś nie ƺedł z przyjáćielem twoim?
Odpowiedźiał Chuſáj Abſálomowi : Nie ; ále którego obrał Pán / y lud ten / y wƺyſcy mężowie Izráelſcy / tego będę / y z nim zoſtánę.
Do tego / komuż ja będę ſłużył? izali nie ſynowi jego? Jákom ſłużył ojcu twemu / ták będę y tobie.
Rzekł potem Abſálom do Achitofelá : Rádźćież / co mamy cżynić?
Odpowiedźiał Achitofel Abſálomowi : Wnijdź do záłożnic ojcá twego / które zoſtáwił / áby ſtrzegły domu ; á uſłyƺáwƺy wƺyſtek Izráel / żeś śię omierzył ojcu twemu / zmocnią śię ręce wƺyſtkich / którzy ſą z tobą.
Przetoż rozbili Abſálomowi námiot ná dáchu. Y wƺedł Abſálom do záłożnic ojcá ſwego przed ocżymá wƺyſtkiego Izráelá.
A rádá Achitofelowá / którą dáwał / byłá ná on cżás w tákiej wádze / jákoby śię kto rádźił Bogá. Tákowáć byłá wƺelká rádá Achitofelowá / jáko u Dawidá / ták u Abſálomá.
