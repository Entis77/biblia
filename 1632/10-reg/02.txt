Y ſtáło śię potem / że pytał Dawid Páná / mówiąc : Mámże iść do któregokolwiek miáſtá Judzkiego? Któremu Pán odpowiedźiał : Idź. Y rzekł Dawid : Dokądże pójdę? Y odpowiedźiał : Do Hebronu.
Tedy tám jechał Dawid / tákże y dwie żony jego / Achinoám Jezreelitká / y Abigáil / żoná przed tym Nábálowá z Kármelu.
Tákże męże ſwe / którzy z nim byli / zábrał Dawid / káżdego z domem jego / y mieƺkáli w miáſtách Hebrońſkich.
Przyƺli potem mężowie Judá / y pomázáli tám Dawidá zá królá nád domem Judá ; tedy opowiedźiáno Dawidowi / mówiąc : Mężowie z Jábes Gáláád ći pogrzebli Saulá.
Tedy wypráwił Dawid poſły do mężów z Jábes Gáláád / y rzekł do nich : Błogoſłáwieniśćie wy od Páná / którzyśćie ucżynili to miłośierdźie nád Pánem wáƺym Saulem / żeśćie go pogrzebli.
Przetoż teraz niech ucżyni Pán z wámi miłośierdźie / y práwdę / á ja też oddam wam to dobrodźiejſtwo / żeśćie ucżynili tę rzecż.
Teraz tedy niech śię zmácniáją ręce wáƺe / á bądźćie mężnymi ; bo choć umárł Saul / pán wáƺ / wƺákże mnie pomázał dom Judá zá królá nád ſobą.
A Abner / ſyn Nerá / hetmán nád wojſkiem Saulowem / wźiął Izboſetá ſyná Saulowego / y przyprowádźił go do Máchánáim /
Y ucżynił go królem nád Gáláádem / y nád Aſſury / y nád Jezreelem / y nád Efráimem / y nád Benjáminem / y nád wƺyſtkim Izráelem.
Cżterdźieśći lat miał Izboſet / ſyn Saulá / gdy pocżął królowáć nád Izráelem / á dwá látá królował ; tylko dom Judá ſtał przy Dawidźie.
Y byłá licżbá dni / których był Dawid królem w Hebronie nád domem Judzkim / śiedm lat y ƺeść mieśięcy.
Potem wƺedł Abner / ſyn Nerá / y ſłudzy Izboſetá / ſyná Saulowego / z Máchánáim do Gábáonu.
Joáb tákże / ſyn Sárwii / z ſługámi Dawidowymi wyƺli / y ſpotkáli śię z ſobą práwie u ſtáwu Gábáońſkiego / y zoſtáli jedni ná jedney ſtronie ſtáwu / á drudzy ná drugiej ſtronie ſtáwu.
Tedy rzekł Abner do Joábá : Niech teraz wſtáną młodźieńcy / á poigráją przed námi. Y rzekł Joáb : Niech wſtáną.
Wſtáli tedy ; y wyƺło w licżbie dwánaśćie z Benjámińcżyków ze ſtrony Izboſetá / ſyná Saulowego / á dwánaśćie z ſług Dawidowych.
Którzy uchwyćiwƺy śię / káżdy zá głowę przećiwniká ſwego / utopił miecż ſwój w boku jeden drugiego / y polegli poſpołu. Przetoż názwano miejſce ono Helkátáſſurym / które jeſt w Gábáonie.
Y byłá bitwá bárzo ſrogá dniá onego / á poráżon jeſt Abner y mężowie Izráelſcy od ſług Dawidowych.
Byli też tám trzej ſynowie Sárwii : Joáb / Abiſáj / y Aſáel ; ále Aſáel był prędkich nóg / jáko dźiká kozá.
Y gonił Aſáel Abnerá / á nie uſtąpił idąc áni ná práwo áni ná lewo / śćigájąc Abnerá.
Ale obejrzáwƺy śię Abner názád / rzekł : Tyżeś jeſt Aſáel? A on mu odpowiedźiał : Ja.
Tedy mu rzekł Abner : Uchyl śię ná práwą ſtronę twoję / álbo ná lewą ſtronę twoję / á pojmij ſobie jednego z młodźieńców / y weźmij ſobie łupy z niego ; ále Aſáel niechćiał od niego uſtąpić.
Tedy powtóre rzekł Abner do Aſáelá : Idź precż odemnie / bym ćię ſnadź nie przebił ku źiemi ; bo jákożbym śmiał podnieść twárz moję ná Joábá / brátá twego?
A gdy niechćiał uſtąpić / uderzył go Abner końcem włócżni pod piąte żebro / ták iż wyƺłá włócżniá ná drugą ſtronę. Támże pádł / y umárł ná onemże miejſcu / á wƺyſcy / którzy przychodźili do onego miejſcá / gdźie poległ Aſáel y umárł / záſtánáwiáli śię.
Wƺákże gonili Joáb y Abiſáj Abnerá ; y záchodźiło ſłońce / gdy przyƺli do págórká Ammá / który jeſt przećiw Giá ná drodze puſtyni Gábáońſkiej.
Tedy śię zebráli ſynowie Benjáminowi do Abnerá / ſkupiwƺy śię w jeden huf / y ſtánęli ná wierzchu jednego págórká.
Y záwołał Abner ná Joábá y rzekł : Izali śię ná wieki będźie ſrożył ten miecż? ázaż nie wieƺ / że ná oſtátku bywa gorzkość? y dokądże nie rzecżeƺ ludowi / áby śię wróćił od pogoni bráći ſwych?
Tedy rzekł Joáb : Jáko żywy Bóg / byś ty był nie wyzywał / zárázby śię był z poránku lud wróćił / káżdy od pogoni bráći ſwych.
A ták zátrąbił Joáb w trąbę y záſtánowił śię wƺyſtek lud / á nie gonili dálej Izráelá / áni śię więcey potykáli.
Ale Abner y mężowie jego ƺli polem cáłą onę noc / á przepráwiwƺy śię przez Jordán / przeƺli przez wƺyſtko Betoron / áż przyƺli do Máhánáim.
A Joáb wróćiwƺy śię z pogoni zá Abnerem / zebrał wƺyſtek lud ; y nie doſtáwáło mu z ſług Dawidowych dźiewiętnáſtu mężów / y Aſáelá.
Ale ſłudzy Dawidowi pobili z Benjámińcżyków / y z mężów Abnerowych trzy ſtá y ƺeśćdźieśiąt mężów / którzy tám pomárli.
A wźiąwƺy Aſáelá / pogrzebli go w grobie ojcá jego / który był w Betlehem ; potem ƺli cáłą noc Joáb y mężowie jego / á ná świtániu przyƺli do Hebronu.
