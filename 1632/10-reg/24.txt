Tedy śię znowu popędliwość Páńſká zápáliłá ná Izráelá / gdy pobudźił ƺátán Dawidá przećiwko nim / mówiąc : Idź / policż Izráelá y Judę.
Y rzekł król do Joábá / hetmáná wojſká ſwego : Przebież záráz wƺyſtkie pokoleniá Izráelſkie od Dán áż do Beerſebá / á policżćie lud / ábym wiedźiáł pocżet ludu.
Lecż Joáb rzekł do królá : Niech przymnoży Pán / Bóg twój / ludu / jáko teraz jeſt tyle ſto kroć / áby ná to ocży królá / páná mego / pátrzáły ; ále król / pán mój / przecżże śię tego nápierá?
Wƺákże przemogło ſłowo królewſkie Joábá y hetmány wojſká. Przetoż wyƺedł Joáb / y hetmáni wojſká od oblicżá królewſkiego / áby policżyli lud Izráelſki.
A przepráwiwƺy śię przez Jordán / położyli śię obozem przy Aroer / po práwej ſtronie miáſtá / które jeſt w pośród potoku Gád y przy Jázer.
Potem przyƺli do Gáláád / y do źiemi dolney Hádſy / á ſtámtąd przyƺli do Dán Jáán y w okół Sydonu.
Potem przyƺli ku twierdzy Tyrſkiej / y do wƺyſtkich miáſt Hewejſkich y Chánánejſkich / ſkąd wyƺli ná południe Judy do Beerſebá.
A obƺedłƺy wƺyſtkę źiemię / przyƺli po dźiewięćiu mieśiącách / y po dwudźieſtu dniách do Jeruzálemu.
Y oddáł Joáb pocżet oblicżonego ludu królowi. A było w Izráelu ośm kroć ſto tyśięcy mężów rycerſkich / godnych ku bojowi / á mężów Judá pięć kroć ſto tyśięcy mężów.
Potem uderzyło Dawidá ſerce jego / gdy oblicżył lud / y rzekł Dawid do Páná : Zgrzeƺyłem bárdzo / żem to ucżynił : ále teraz o Pánie! przenieś proƺę niepráwość ſługi twego / bomći bárdzo głupio ucżynił.
A gdy wſtáł Dawid ráno / oto ſłowo Páńſkie ſtáło śię do Gádá proroká / Widzącego Dawidowego / mówiąc :
Idź / á powiedz Dawidowi : Ták mówi Pán : Trzyć rzecży podáwám / obierz ſobie jednę z tych / ábym ći ucżynił.
A ták przyƺedł Gád do Dawidá / y oznájmił mu / á rzekł mu : Albo przyjdźie ná ćię głód przez śiedm lát w źiemi twojey / álbo przez trzy mieśiące będźieƺ ućiekáł przed nieprzyjáćioły twymi / á oni ćię gonić będą / álbo więc przez trzy dni będźie morowe powietrze w źiemi twojey ; rozmyślże śię prędko / á obácż / co mam odpowiedźieć temu / który mię poſłáł.
Y rzekł Dawid do Gádá : Jeſtem bárdzo śćiśniony. Niech proƺę rácżej wpádniemy w rękę Páńſką / gdyż wielkie ſą zlitowániá jego ; ále w rękę ludzką niech nie wpádám.
Tedy przepuśćił Pán powietrze morowe ná Izráelá od poránku áż do cżáſu náznácżonego / y umárło z ludu od Dán áż do Beerſebá śiedmdźieśiąt tyśięcy mężów.
A gdy wyćiągnął Anioł rękę ſwą ná Jeruzálem / áby je wytráćił / tedy śię użálił Pán onego złego / y rzekł do Aniołá / który tráćił lud : Doſyć teraz ; záwśćiągnij rękę twą. A Anioł Páńſki był podle bojewiſká Aráwny Jebuzejcżyká.
Y rzekł Dawid do Páná / gdy ujrzáł Aniołá bijącego lud / mówiąc : Otom ja zgrzeƺył / jám źle ucżynił ; ále te owce cóż ucżyniły? niech śię proƺę obróći ręká twojá ná mię y ná dom ojcá mego.
Tedy przyƺedł Gád do Dawidá onegoż dniá / y rzekł mu : Idź / á zbuduj ołtárz Pánu ná bojewiſku Aráwny Jebuzejcżyká.
Y ƺedł Dawid podług ſłowá Gádowego / jáko był rozkázáł Pán.
Tedy ſpojrzáwƺy Aráwná / ujrzáł królá / y ſługi jego / przychodzące do śiebie : y wyƺedł Aráwná / á pokłonił śię królowi twárzą ſwą ku źiemi.
Y rzekł Aráwná : Przecżże przyƺedł król / pán mój / do ſługi ſwego? Y odpowiedźiáł Dawid : Abym kupił u ćiebie to bojewiſko / y zbudowáł ná nim ołtárz Pánu / żeby záhámowáná byłá tá plágá między ludem.
Tedy rzekł Aráwná do Dawidá : Niech weźmie / á ofiáruje król / pán mój / co mu śię dobrego widźi : oto woły ná cáłopálenie / y wozy / y járzmá wołów ná drwá.
Wƺyſtko to dáwáł król Aráwná królowi Dawidowi. Y mówił Aráwná do królá : Pán / Bóg twój / niech ćię ſobie upodobá.
Lecż król rzekł do Aráwny : Nie ták / ále rácżej kupię u ćiebie y zápłácę ; áni będę ofiárowáł Pánu / Bogu memu / cáłopáleniá dármo dánego. A ták kupił Dawid ono bojewiſko y woły zá pięćdźieśiąt ſyklów ſrebrá.
Támże zbudowáł Dawid ołtárz Pánu / y ſpráwowáł cáłopáleniá / y ſpokojne ofiáry. Y ubłágány był Pán źiemi / á záhámowáná jeſt oná plágá od Izráelá.
