Y ſtáło śię po śmierći Saulowej / gdy śię Dawid wróćił od poráżki Amálekitów / że zámieƺkáł Dawid w Sycelegu przez dwá dni.
Tedy dniá trzećiego przybieżáł mąż z obozu Saulowego / á ƺáty jego rozdárte były / y proch ná głowie jego ; który przyƺedłƺy do Dawidá / upádł ná źiemię / y pokłonił śię.
Y rzekł do niego Dawid : Skąd idźieƺ? Y odpowiedźiáł mu : Z obozum Izráelſkiego uƺedł.
Rzekł mu znowu Dawid : Cóż śię ſtáło? proƺę powiedz mi. Y odpowiedźiáł : To / że ućiekł lud z bitwy / á do tego wiele poległo z ludu y pomárło / tákże y Saul / y Jonátán / ſyn jego / polegli.
Zátem rzekł Dawid do młodźieńcá / który mu to powiedźiáł : Jákoż wieƺ / iż umárł Saul y Jonátán / ſyn jego?
Odpowiedźiáł mu młodźieniec / który mu to oznájmił : Przyƺedłem z tráfunku ná górę Gielboe / á oto / Saul tkwiáł ná włócżni ſwojej / á wozy y jezdni dogániáli go.
Tedy obejrzáwƺy śię / obácżył mię / y záwołáł ná mię / y rzekłem : Owom já.
Y rzekł mi : Coś ty zácż? A jám mu odpowiedźiáł : Jeſtem Amálekitá.
Y rzekł mi : Stáń / proƺę / nádemną / á zábij mię : bo mię zdjęły ćiężkośći / gdyż jeƺcże wƺyſtká duƺá mojá we mnie jeſt.
Przetoż ſtánąwƺy nád nim / zábiłem go : bom wiedźieł / że nie będźie żyw po upádku ſwoim / y wźiąłem koronę / którá byłá ná głowie jego / y záwieƺenie / które było ná rámieniu jego / á przynioſłem je tu do páná mego.
Tedy Dawid pochwyćiwƺy ƺáty ſwoje / rozdárł je / tákże y wƺyſcy mężowie / którzy z nim byli.
A żáłując płákáli / y pośćili áż do wiecżorá dlá Saulá / y dlá Jonátáná / ſyná jego / y dlá ludu Páńſkiego / y dlá domu Izráelſkiego / że polegli od miecżá.
Y rzekł Dawid do młodźieńcá / który mu to oznájmił : Skądeś ty? Y odpowiedźiáł : Jeſtem ſynem mężá przychodniá Amálekity.
Zátem rzekł do niego Dawid : Jákożeś śię nie báł śćiągnąć ręki twej / ábyś zábił pomázáńcá Páńſkiego?
Záwołáł tedy Dawid jednego z ſług / y rzekł : Przyſtąp / á zábij go / á on go uderzył / że umárł.
Y rzekł do niego Dawid : Krew twojá ná głowę twoję : bo uſtá twoje świádcżyły przećiw tobie / mówiąc : Jám zábił pomázáńcá Páńſkiego.
Lámentowáł tedy Dawid lámentem tákowym nád Saulem / y nád Jonátánem / ſynem jego ;
( Rozkázáwƺy jednák / áby ucżono ſynów Judzkich z łuku ſtrzeláć / jáko nápiſáno w kśięgách Jáſár. )
O ozdobo Izráelſká! ná górách twoich zránionyś jeſt ; jákoż polegli mocárze twoi!
Nie powiádájćież w Get / áni rozgłáƺájćie po ulicách w Aƺkálonie / áby śię ſnáć nie weſeliły córki Filiſtyńſkie / by śię ſnáć nie rádowáły córki nieobrzezáńców.
O góry Gielboe! áni roſá / áni deƺcż niech nie upádá ná wáſ / y niech tám nie będą polá urodzájne ; ábowiem tám porzuconá jeſt tárcż mocárzów / tárcż Saulowá / jákoby nie był pomázány olejem.
Od krwi zábitych / y od ſádłá mocárzów ſtrzáłá łuku Jonátánowego nie wrácáłá śię ná wſtecż / á miecż Saulowy nie wrácáł śię próżno.
Saul y Jonátán miłośni y przyjemni zá żywotá ſwego / y w śmierći ſwojej nie ſą rozłącżeni / nád orły lekćiejśi / nád lwy mocniejśi byli.
Córki Izráelſkie płácżćie nád Saulem / który wáſ przyodźiewáł ƺárłátem rozkoƺnym / á który wáſ ubieráł w klejnoty złote ná ƺáty wáƺe.
Jákoż polegli mocárze w pośród bitwy! Jonátán ná górách twoich zábity jeſt.
Bárdzo mi ćię żál / bráćie mój Jonátánie / byłeś mi bárdzo wdźięcżnym ; więkƺá u mnie byłá miłość twojá / niż miłość niewieśćiá.
Jákoż polegli mocárze / á poginęłá broń wojenná!
