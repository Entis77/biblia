A porozumiáwƺy Joáb / ſyn Sárwii / że śię ſerce królewſkie obróćiło ku Abſálomowi /
Poſłáł Joáb do Tekuj / y wziął ſtámtąd niewiáſtę mądrą / y rzekł do niej : Proƺę zmyśl / jákobyś w żáłobie byłá / á oblecż śię proƺę w ƺáty żáłobne / y nie námázuj śię olejkiem / ále bądź jáko niewiáſtá / którá przez wiele dni w żáłobie chodzi po umárłym.
Y wnijdzieƺ do królá / á będzieƺ mówiłá do niego w ten ſpoſób ; y náucżył jej Joáb / jáko miáłá mówić.
Przetoż mówiłá oná niewiáſtá Tekuitſká do królá / upádłƺy oblicżem ſwem ná ziemię / á pokłoniwƺy śię rzekłá : Rátuj królu!
Y rzekł jej król : Cóż ći? A oná odpowiedziáłá : Záiſtem já niewiáſtá wdowá ; bo mi mąż mój umárł :
A ſłużebnicá twojá miáłá dwóch ſynów / którzy śię powádzili z ſobą ná polu ; á gdy nie był / ktoby je rozwádził / ránił jeden drugiego / y zábił go.
A oto / powſtáwƺy wƺyſtká rodziná przećiw ſłużebnicy twojej / mówią : Wydáj tego / który zábił brátá ſwego / że go zábijemy zá duƺę brátá jego / którego zámordowáł / owƺem zgłádzimy y dziedzicá ; á ták zágáƺą iſkierkę moję / którá pozoſtáłá / áby nie zoſtáwili mężowi memu imieniá y ƺcżątku ná ziemi.
Tedy rzekł król do niewiáſty : Idź do domu twego / á já ſkáżę zá tobą.
Y odpowiedziáłá niewiáſtá Tekuitſká królowi : Królu / pánie mój / niech będzie ná mnie tá niepráwość / y ná domu ojcá mego ; ále król y ſtolicá jego niech będzie niewinná.
Y rzekł król : Będzieli kto mówił przećiwko tobie / przywiedź go do mnie / á potem nie tknie śię ćiebie więcej.
Tedy oná rzekłá : Wſpomnij proƺę / królu / ná Páná Bogá twego / áby śię nie mnożyli mśćićiele krwi ná zgubę / á nie zgłádzili ſyná mego. Y odpowiedziáł : Jáko żywy Pán / że nie ſpádnie y nájmniejƺy włoſ ſyná twego ná ziemię.
Zátem rzekłá niewiáſtá : Niech przemówi proƺę ſłużebnicá twojá do królá / páná mego / ſłowo. A on rzekł : Mów.
Rzekłá tedy niewiáſtá : Y cżemużeś umyślił podobną rzecż przećiw ludowi Bożemu? ábowiem król mówi to ſłowo / jákoby był winny / poniewáż nie chceƺ przywróćić / królu / wygnáńcá ſwego.
Wƺyſcy umierámy / á jeſteśmy jáko wody rozláne po ziemi / które nie mogą być zebráne ; lecż jemu Bóg nie odjął żywotá / ále pewnie umyślił / áby nie wygániáł od śiebie wygnáńcá.
A teráz / żem przyƺłá mówić do królá / páná mego / te ſłowá / przycżyną jeſt / że mię przeſtráƺył lud ; przetoż rzekłá ſłużebnicá twojá : Będę teráz mówiłá do królá / ſnáć co ucżyni król ná prośbę ſłużebnicy ſwojej.
Abowiem uſłyƺy to król / y wybáwi ſłużebnicę ſwoję z rąk mężá / który wygłádzić chce mnie / y ſyná mego ſpołem / z dziedzictwá Bożego.
Rzekłá też ſłużebnicá twojá : Wżdyć mi będzie ſłowo królá / páná mego / ku poćieƺe ; ábowiem jáko Anioł Boży / ták jeſt król / pán mój / ſłuchájąc dobrego y złego / á Pán Bóg twój niech będzie z tobą.
A odpowiádájąc król rzekł do niewiáſty : Proƺę nie táj przedemną tego / o co śię ſpytám. Y rzekłá niewiáſtá : Mów proƺę / królu / pánie mój.
Tedy rzekł król : Izáli ty tego wƺyſtkiego nie cżyniƺ z nápráwy Joábowej? Y odpowiedziáłá niewiáſtá / y rzekłá : Jáko żyje duƺá twojá / królu / pánie mój / że nie możná uchylić śię áni ná práwo / áni ná lewo od wƺyſtkiego / co mówił król / pán mój ; ábowiem ſługá twój Joáb / on mi to rozkázáł / y on náucżył ſłużebnicy twojej tych wƺyſtkich ſłów.
Żem odmieniłá ſpoſób tej mowy / ſpráwił to ſługá twój Joáb ; lecż pán mój mądry jeſt / jáko jeſt mądry Anioł Boży / wiedząc wƺyſtko / co śię dzieje ná ziemi.
Przetoż rzekł król do Joábá : Otom teráz to ucżynił. Idźże á przywróć dziećię me Abſálomá.
Y upádł Joáb oblicżem ſwojim ná ziemię / á pokłoniwƺy śię błogoſłáwił królowi / y rzekł Joáb : Dziś poznáł ſługá twój / żem ználázł łáſkę w ocżách twoich / królu / pánie mój / poniewáż ucżynił król zádoſyć prośbie ſługi ſwego.
Wſtáł tedy Joáb / á ƺedł do Gieſſur / y przywiódł Abſálomá do Jeruzálemu.
Y rzekł król : Niech śię wróći do domu ſwego / ále oblicżá mego niech nie widzi. A ták wróćił śię Abſálom do domu ſwego / ále oblicżá królewſkiego nie widziáł.
A nie było mężá ták kráſnego / jáko Abſálom we wƺyſtkim Izráelu / coby miáł ták wielką chwáłę ; od ſtopy nogi jego áż do wierzchu głowy jego nie było ná nim zmázy.
A gdy ſtrzygł głowę ſwoję / ( á zwykł ją ná káżdy rok ſtrzydz ; bo mu ćiążyłá / przetoż ją ſtrzygł / )wáżyły włoſy głowy jego dwieśćie ſyklów wági królewſkiej.
Y urodzili śię Abſálomowi trzej ſynowie / y córká jedná / której imię było Támár / którá niewiáſtá byłá piękná ná wejrzeniu.
Y mieƺkáł Abſálom w Jeruzálemie dwá látá / á twárzy królewſkiej nie widziáł.
Przetoż poſłáł Abſálom do Joábá / chcąc go poſłáć do królá / ále on nie chćiáł przyjść do niego ; poſłáł potem powtóre / y nie chćiáł przyjść.
Tedy rzekł do ſług ſwoich : Przepátrzćie rolą Joábowę podle roli mojej / gdzie má jęcżmień ; idźćież / á ſpálćie go ogniem. Y zápálili ſłudzy Abſálomowi rolą onę ogniem.
Zátem wſtáwƺy Joáb / przyƺedł do Abſálomá w dom / y rzekł do niego : Cżemuż ſłudzy twoi ſpálili rolą moję ogniem?
Y odpowiedziáł Abſálom Joábowi : Otom poſłáł do ćiebie / mówiąc : Przyjdź ſám / á poślę ćię do królá / ábyś mówił : Ná cóżem przyƺedł z Gieſſur? Lepiej mi było tám jeƺcże zoſtáć ; przetoż teráz niech oglądám oblicże królewſkie ; wƺák jeźli jeſt przy mnie niepráwość / niech mię rozkáże zábić.
Tedy przyƺedł Joáb do królá / y oznájmił mu. Y przyzwáł Abſálomá / który przyƺedł do królá / y ukłonił śię twárzą ſwą ku ziemi przed królem ; y pocáłowáł król Abſálomá.
