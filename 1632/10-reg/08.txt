Y ſtáło śię potem / że poráził Dawid Filiſtyny / y poniżył je ; á wziął Dawid Meteg Ammá z rąk Filiſtyńſkich.
Poráził też y Moábity / które pomierzył ƺnurem / zrównáwƺy je z ziemią / y wymierzył ich dwá ƺnury ná zábićie / á cáły ƺnur ná záchowánie przy żywoćie ; y byli Moábitowie ſługámi Dawidowymi / przynoƺąc mu podátki.
Poráził też Dawid Hádádezerá / ſyná Rochobowego / królá Soby / gdy wyjecháł / áby rozprzeſtrzenił gránice ſwe nád rzeką Eufráteſ.
Y pojmáł z nich Dawid tyśiąc y śiedm ſet jezdnych / á dwádzieśćiá tyśięcy ludu pieƺego. Y poderznął Dawid żyły wƺyſtkim woźnikom / záchowáwƺy koni do ſtá wozów.
Przyćiągnął tedy Syryjcżyk z Dámáƺku ná pomoc Hádádezerowi / królowi Soby / y poráził Dawid Syryjcżyków dwádzieśćiá y dwá tyśiące mężów.
Y oſádził Dawid żołnierzem Syryją Dámáſką. A ták Syryjcżycy byli ſługámi Dawidowymi / przynoƺąc mu podátki ; y bronił Pán Dawidá wƺędzie / gdziekolwiek śię obróćił.
Pobráł też Dawid tárcże złote / które mieli ſłudzy Hádádezerowi / y wnióſł je do Jeruzálemu.
Przytem z Betáchu y z Berotu / miáſt Hádádezerowych / przynióſł król Dawid bárdzo wiele miedzi.
To uſłyƺáwƺy Tohy / król Emát / iż poráził Dawid wƺyſtko wojſko Hádádezerowe /
Poſłáł Tohy Jorámá / ſyná ſwego / do królá Dawidá / áby go pozdrowił w pokoju / y winƺowáł mu / przeto że zwálcżył Hádádezerá / y poráził go / ( ábowiem wálcżył z Tohym Hádádezer / )i przynióſł z ſobą nácżyniá ſrebrne / y nácżyniá złote / y nácżyniá miedziáne.
Które też rzecży poświęćił król Dawid Pánu z innem ſrebrem y złotem / które był poświęćił / pobráwƺy od wƺyſtkich národów / które ſobie podbił.
Jáko od Syryjcżyków / y od Moábcżyków / y od ſynów Ammonowych / y od Filiſtynów / y od Amálekitów / y z łupów Hádádezerá / ſyná Rochobowego / królá Soby.
A ták ucżynił ſobie Dawid imię / gdy śię wróćił / poráziwƺy Syryjcżyki w dolinie ſolnej / gdzie pobił ośmnáśćie tyśięcy ludu.
Poſtánowił też ſtráż w Edom / wƺyſtkę ziemię Edomſką oſádziwƺy żołnierzámi ; y byli wƺyſcy Edomcżycy ſługámi Dawidowymi / á bronił Pán Dawidá wƺędzie / gdzie śię obróćił.
Y królowáł Dawid nád wƺyſtkim Izráelem / y cżynił ſąd y ſpráwiedliwość wƺyſtkiemu ludowi ſwemu.
A Joáb / ſyn Sárwii / był nád wojſkiem / á Jozáfát ſyn Ahiludá / kánclerzem.
A Sádok / ſyn Achitobá / y Achimelech / ſyn Abijátárá / byli kápłánámi / á Sárájá piſárzem.
Bánájáſ też / ſyn Jojádá / nád Cheretcżykámi y Feletcżykámi / á ſynowie Dawidowi byli kśiążętámi.
