Zeƺły śię tedy wƺyſtkie pokoleniá Izráelſkie do Dawidá do Hebronu / y rzekły / mówiąc : Oto / my jeſteśmy kość twojá y ćiáło twoje.
A przeƺłych cżáſów / gdy Saul był królem nád námi / tyś wywodził y przywodził Izráelá. Nád to rzekł Pán do ćiebie : Ty będzieƺ páſł lud mój Izráelſki / á ty będzieƺ wodzem nád Izráelem.
A ták wƺyſcy ſtárśi Izráelſcy przyƺli do królá do Hebronu ; y ucżynił z nimi król Dawid przymierze w Hebronie przed Pánem ; y pomázáli Dawidá zá królá nád Izráelem.
Trzydzieśći lát było Dawidowi / gdy pocżął królowáć / á królowáł przez cżterdzieśći lát.
W Hebronie królowáł nád Judą przez śiedm lát y przez ƺeść mieśięcy / á w Jeruzálemie królowáł przez trzydzieśći y trzy látá nád wƺyſtkim Izráelem y nád Judą.
A ták poƺedł król y mężowie jego do Jeruzálemu przećiw Jebuzejcżykowi mieƺkájącmu w onej ziemi / który rzekł do Dawidá / mówiąc ; Nie wnijdzieƺ ſám / áż znieśieƺ ślepe y chrome / jákoby mówili : Nie wnijdzie tu Dawid.
Wƺákże wziął Dawid zámek Syońſki / á toć jeſt miáſto Dawidowe.
Bo rzekł był Dawid onego dniá : Ktobykolwiek zábił Jebuzejcżyká / á wƺedłby ná rynny / á pobił te ślepe y chrome / które má w nienáwiśći duƺá Dawidowá / poſtánowię go hetmánem. Dlá tegoż máwiáno : Ślepy y chromy nie wnijdzie do tego domu.
Y mieƺkáł Dawid ná onym zámku á przezwáł go miáſtem Dawidowem / y pobudowáł je Dawid wƺędy w koło od Mello / y wewnątrz.
A Dawid idąc poſtępowáł y róſł ; bo Pán Bóg záſtępów był z nim.
Tedy poſłáł Hyrám / król Tyrſki / poſły do Dawidá / y drzewá cedrowe / y ćieśle / y kámienniki / y murárze / którzy zbudowáli dom Dawidowi.
Y poznáł Dawid / iż go utwierdził Pán zá królá nád Izráelem / á iż wywyżƺył króleſtwo jego dlá ludu ſwego Izráelſkiego.
Y nápojmowáł ſobie Dawid jeƺcże więcej záłożnic y żon z Jeruzálemu / przyƺedłƺy z Hebronu / á národziło śię więcej Dawidowi ſynów y córek.
A teć ſą imioná tych / którzy mu śię urodzili w Jeruzálemie : Sámmá / y Sobáb / y Nátán / y Sálomon /
Y Ibchár / y Eliſuá / y Nefeg / y Jáfijá /
Y Eliſámá / y Elijádá / y Elifelet.
A uſłyƺáwƺy Filiſtynowie / że pomázáno Dawidá zá królá nád Izráelem / ruƺyli śię wƺyſcy Filiſtynowie / żeby ƺukáli Dawidá ; co gdy uſłyƺáł Dawid / uſtąpił ná zámek.
Tedy Filiſtynowie przyćiągnąwƺy / rozpoſtárli śię w dolinie Refáim.
Y rádził śię Dawid Páná / mówiąc : Mámli iść przećiwko Filiſtynom? podáƺli je w ręce moje? Odpowiedziáł Pán Dawidowi : Idź / bo pewnie podám Filiſtyny w ręce twoje.
A ták przyćiągnął Dawid do Báál Perázym / y poráził je tám Dawid y rzekł : Rozerwáł Pán nieprzyjáćioły moje przedemną / jáko śię rozrywáją wody. Przetoż názwáł imię miejſcá onego Báál Perázym.
Y zoſtáwili tám ryte ſwe obrázy / które popálił Dawid y mężowie jego.
Znowu jeƺcże przyćiągnęli Filiſtynowie / y rozpoſtárli śię w dolinie Refáim.
Y pytáł śię Dawid Páná / który odpowiedziáł : Nie pójdzieƺ przećiwko nim ; ále je obtocżywƺy z tyłu nátrzeƺ ná nie przećiwko morwom.
A gdy uſłyƺyƺ / iż záƺumią wierzchy morwów / tedy śię ruƺyƺ / gdyż ná ten cżáſ pójdzie Pán przed tobą / áby poráził wojſká Filiſtyńſkie.
Y ucżynił Dawid ták jáko mu rozkázáł Pán / á poráził Filiſtyny od Gábáá / áż kędy chodzą do Gázer.
