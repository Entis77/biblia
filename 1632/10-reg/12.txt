Przetoż poſłáł Pán Nátáná do Dawidá ; który przyƺedłƺy do niego / rzekł mu : Dwáj mężowie byli w jednem mieśćie / jeden bogáty á drugi ubogi.
Bogáty miáł owiec y wołów bárdzo wiele ;
A ubogi nie miáł jedno owiecżkę jednę máłą / którą był kupił / y chowáł ją / áż uroſłá przy nim / tákże y przy dźiátkách jego ; z bochná jego jádáłá / y z kubká jego pijáłá / y ná łonie jego ſypiáłá / á byłá mu jáko córká.
A gdy przyƺedł gość do onego mężá bogátego / żáłowáł wźiąć z owiec ſwoich álbo z wołów ſwoich / áby nágotowáł ucżtę gośćiowi / który był do niego przyƺedł : ále wźiął owiecżkę mężá onego ubogiego / y nágotowáł ją mężowi / który był do niego przyƺedł.
Tedy zápáliwƺy śię gniewem Dawid ná onegoż mężá bárdzo / rzekł do Nátáná : Jáko żywy Pán / że godźien śmierći jeſt mąż / który to ucżynił ;
Owcę tę nágrodźi cżworáko / przeto iż to ucżynił / á nie żáłowáł go.
Y rzekł Nátán do Dawidá : Tyś jeſt tym mężem. Tákći mówi Pán / Bóg Izráelſki : Jám ćię pomázáł / ábyś był królem nád Izráelem / y Jám ćię wyrwáł z rąk Saulowych ;
A podáłem ći dom páná twego / y żony páná twego ná łono twoje ; nádto oddáłem ći dom Izráelſki y Judzki / á byłoliby to máło / przydáłbym ći był dáleko więcej.
Cżemużeś zniewáżył ſłowo Páńſkie / cżyniąc to złe przed ocżymá jego? Uryjáƺá Hetejcżyká zábiłeś miecżem / á żonę jego wźiąłeś ſobie zá żonę / á ſámegoś zábił miecżem ſynów Ammonowych.
Przetoż teráz nie odejdźie miecż z domu twego áż ná wieki / dlá tego / iżeś mię zniewáżył / á wźiąłeś żonę Uryjáƺá Hetejcżyká / ábyć byłá zá żonę.
Ták mówi Pán : Oto / Já wzbudzę przećiwko tobie złe z domu twego / á pobráwƺy żony twe przed ocżymá twemi / dám je bliźniemu twemu / á będźie jáwnie ſpáł z żonámi twojimi.
A choćiáżeś to ty ucżynił potájemnie / Já jednák ucżynię to przed wƺyſtkim Izráelem / y przed ſłońcem.
Tedy rzekł Dawid do Nátáná : Zgrzeƺyłem Pánu. Záś rzekł Nátán do Dawidá : Pán też przenióſł grzech twój / nie umrzeƺ.
Wƺákże iżeś dáł przycżynę / áby urągáli nieprzyjáćiele Páńſcy dlá tej ſpráwy / przetoż ſyn / któryć śię urodźił / pewnie umrze.
Potem odƺedł Nátán do domu ſwego. Wtem záráźił Pán dźiećię / które byłá urodźiłá żoná Uryjáƺowá Dawidowi / y zwątpiono o nim.
Tedy śię modlił Dawid Bogu zá dźiećięćiem y pośćił / á wƺedłƺy do pokoju / leżáł przez noc ná źiemi.
Y przyƺli ſtárśi domu jego do niego / áby go podnieśli z źiemi ; ále niechćiáł / y nie jádł z nimi chlebá.
Y ſtáło śię dniá śiódmego / że umárło dźiećię. A obáwiáli śię ſłudzy Dawidowi / oznájmić mu / iż umárło dźiećię / bo mówili : Oto / póki jeƺcże dźiećię było żywę / mówiliśmy z nim / á nie ſłucháł głoſu náƺego ; cóż gdy mu powiemy : Umárło dźiećię / dopieroż śię będźie trápił.
A widząc Dawid / że ſłudzy jego ƺeptáli z ſobą / porozumiáł Dawid / iż umárło dźiećię. Y rzekł Dawid do ſług ſwoich : Albo umárło dźiećię? A oni odpowiedźieli : Umárło.
Tedy wſtáwƺy Dawid z źiemi / umył śię / y námázáł śię / y odmienił ƺáty ſwoje / á wƺedłƺy do domu Páńſkiego / modlił śię ; potem wróćiwƺy śię do domu ſwego / kázáł ſobie dáć jeść / y położono przedeń chleb / y jádł.
Y rzekli ſłudzy jego do niego : Cóż to jeſt / coś ucżynił? Dlá dźićięćiá / póki jeƺcże żyło / pośćiłeś y płákáłeś / á gdy umárło dźiećię / wſtáłeś y jádłeś chleb?
A on rzekł : Póki jeƺcże dźiećię żyło / pośćiłem y płákáłem ; bom mówił : Któż wie / nie zmiłujeli śię Pán nádemną / że będźie żywe dźiećię.
Ale teráz / gdy już umárło / przecżżebym miáł pośćić? Izáli je mogę jeƺcże názád wróćić? Já pójdę do niego / ále śię ono nie wróći do mnie.
Y ćieƺył Dawid Betſábeę / żonę ſwą / á wƺedłƺy do niej / ſpáł z nią. Y porodźiłá ſyná / y názwáł imię jego Sálomon / á Pán go miłowáł.
Przetoż poſłáł Nátáná proroká / y názwáł imię jego Jedydyjá / dlá Páná.
Potem wálcżył Joáb przećiw Rábbie ſynów Ammonowych / y wźiął miáſto królewſkie.
A poſłáwƺy Joáb poſły do Dawidá / rzekł : Wálcżyłem przećiw Rábbie y wźiąłem miáſto wód.
Przetoż teráz zbierz oſtátek ludu / á połóż śię obozem przećiwko miáſtu / y weźmij je / bym já ſnáć nie wźiął miáſtá tego / á przypiſánoby zwyćięſtwo imieniowi memu.
A ták zebráł Dawid wƺyſtek lud / y ćiągnął przećiw Rábbie / y dobywáł go / á wźiął je.
Wźiął też koronę królá ich z głowy jego / którá wáżyłá tálent złotá / á kámień drogi był ná niej ; y włożono ją ná głowę Dawidowę / á łupów z miáſtá wynióſł bárdzo wiele.
Lud też / który był w mieśćie / wywiódłƺy / podáł pod piły / y pod brony żelázne / y pod śiekiery żelázne / y wegnáł je w piec cegielny. A ták ucżynił wƺyſtkim miáſtom ſynów Ammonowych ; y wróćił śię Dawid / y wƺyſtek lud jego do Jeruzálem.
