Przetoż poſłał Pán Nátáná do Dawidá ; który przyƺedƺy do niego / rzekł mu : Dwáj mężowie byli w jednem mieśćie / jeden bogáty á drugi ubogi.
Bogáty miał owiec y wołów bárzo wiele ;
A ubogi nie miał jedno owiecżkę jednę máłą / którą był kupił / y chował ją / áż uroſłá przy nim / tákże y przy dźiátkách jego ; z bochná jego jádáłá / y z kubká jego pijáłá / y ná łonie jego ſypiáłá / á byłá mu jáko córká.
A gdy przyƺedł gość do onego mężá bogátego / żáłował wźiąć z owiec ſwojich álbo z wołów ſwojich / áby nágotował ucżtę gośćiowi / który był do niego przyƺedł : ále wźiął owiecżkę mężá onego ubogiego / y nágotował ją mężowi / który był do niego przyƺedł.
Tedy zápáliwƺy śię gniewem Dawid ná onegoż mężá bárzo / rzekł do Nátáná : Jáko żywy Pán / że godźien śmierći jeſt mąż / który to ucżynił ;
Owcę tę nágrodźi cżworáko / przeto iż to ucżynił / á nie żáłował go.
Y rzekł Nátán do Dawidá : Tyś jeſt tym mężem. Tákći mówi Pán / Bóg Izráelſki : Jam ćię pomázał / ábyś był królem nád Izráelem / y Jam ćię wyrwał z rąk Saulowych ;
A podáłem ći dom páná twego / y żony páná twego ná łono twoje ; nádto oddáłem ći dom Izráelſki y Judzki / á byłoliby to máło / przydáłbym ći był dáleko więcey.
Cżemużeś zniewáżył ſłowo Páńſkie / cżyniąc to złe przed ocżymá jego? Uryjáƺá Hetejcżyká zábiłeś miecżem / á żonę jego wźiąłeś ſobie zá żonę / á ſámegoś zábił miecżem ſynów Ammonowych.
Przetoż teraz nie odejdźie miecż z domu twego áż ná wieki / dla tego / iżeś mię zniewáżył / á wźiąłeś żonę Uryjáƺá Hetejcżyká / ábyć byłá zá żonę.
Ták mówi Pán : Oto / Ja wzbudzę przećiwko tobie złe z domu twego / á pobráwƺy żony twe przed ocżymá twemi / dam je bliźniemu twemu / á będźie jáwnie ſpał z żonámi twojimi.
A choćiáżeś to ty ucżynił potájemnie / Ja jednák ucżynię to przed wƺyſtkim Izráelem / y przed ſłońcem.
Tedy rzekł Dawid do Nátáná : Zgrzeƺyłem Pánu. Záś rzekł Nátán do Dawidá : Pán też przenióſł grzech twój / nie umrzeƺ.
Wƺákże iżeś dał przycżynę / áby urągáli nieprzyjáćiele Páńſcy dla tey ſpráwy / przetoż ſyn / któryć śię urodźił / pewnie umrze.
Potem odƺedł Nátán do domu ſwego. Wtem záráźił Pán dźiećię / które byłá urodźiłá żoná Uryjáƺowá Dawidowi / y zwątpiono o nim.
Tedy śię modlił Dawid Bogu zá dźiećięćiem y pośćił / á wƺedƺy do pokoju / leżał przez noc ná źiemi.
Y przyƺli ſtárśi domu jego do niego / áby go podnieśli z źiemi ; ále niechćiał / y nie jádł z nimi chlebá.
Y ſtáło śię dniá śiódmego / że umárło dźiećię. A obáwiáli śię ſłudzy Dawidowi / oznájmić mu / iż umárło dźiećię / bo mówili : Oto / póki jeƺcże dźiećię było żywe / mówiliśmy z nim / á nie ſłuchał głoſu náƺego ; cóż gdy mu powiemy : Umárło dźiećię / dopieroż śię będźie trápił.
A widząc Dawid / że ſłudzy jego ƺeptáli z ſobą / porozumiał Dawid / iż umárło dźiećię. Y rzekł Dawid do ſług ſwojich : Albo umárło dźiećię? A oni odpowiedźieli : Umárło.
Tedy wſtáwƺy Dawid z źiemi / umył śię / y námázał śię / y odmienił ƺáty ſwoje / á wƺedƺy do domu Páńſkiego / modlił śię ; potem wróćiwƺy śię do domu ſwego / kázał ſobie dáć jeść / y położono przedeń chleb / y jádł.
Y rzekli ſłudzy jego do niego : Cóż to jeſt / coś ucżynił? Dla dźiećięćiá / póki jeƺcże żyło / pośćiłeś y płákáłeś / á gdy umárło dźiećię / wſtáłeś y jádłeś chleb?
A on rzekł : Póki jeƺcże dźiećię żyło / pośćiłem y płákáłem ; bom mówił : Któż wie / nie zmiłujeli śię Pán nádemną / że będźie żywe dźiećię.
Ale teraz / gdy już umárło / przecżżebym miał pośćić? Izali je mogę jeƺcże názád wróćić? Ja pójdę do niego / ále śię ono nie wróći do mnie.
Y ćieƺył Dawid Betſábeę / żonę ſwą / á wƺedƺy do niej / ſpał z nią. Y porodźiłá ſyná / y názwał imię jego Sálomon / á Pán go miłował.
Przetoż poſłał Nátáná proroká / y názwał imię jego Jedydyjá / dla Páná.
Potem wálcżył Joáb przećiw Rábbie ſynów Ammonowych / y wźiął miáſto królewſkie.
A poſłáwƺy Joáb poſły do Dawidá / rzekł : Wálcżyłem przećiw Rábbie y wźiąłem miáſto wód.
Przetoż teraz zbierz oſtátek ludu / á połóż śię obozem przećiwko miáſtu / y weźmij je / bym ja ſnadź nie wźiął miáſtá tego / á przypiſánoby zwyćięſtwo imieniowi memu.
A ták zebrał Dawid wƺyſtek lud / y ćiągnął przećiw Rábbie / y dobywał go / á wźiął je.
Wźiął też koronę królá ich z głowy jego / która wáżyłá tálent złotá / á kámień drogi był ná niej ; y włożono ją ná głowę Dawidowę / á łupów z miáſtá wynióſł bárzo wiele.
Lud też / który był w mieśćie / wywiódƺy / podał pod piły / y pod brony żelázne / y pod śiekiery żelázne / y wegnał je w piec cegielny. A ták ucżynił wƺyſtkim miáſtom ſynów Ammonowych ; y wróćił śię Dawid / y wƺyſtek lud jego do Jeruzálem.
