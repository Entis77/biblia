Tedy rzekł Dawid : Jeſtże jeƺcże kto / coby pozoſtáł z domu Saulowego / ábym ucżynił nád nim miłośierdzie dlá Jonátáná?
Y był z domu Saulowego ſługá / którego zwáno Sybá : tego záwołáno do Dawidá. Y rzekł król do niego : Tyżeś jeſt Sybá? A on odpowiedziáł : Jám jeſt / ſługá twój.
Potem rzekł król : Jeſtże jeƺcże kto z domu Saulowego / ábym nád nim ucżynił miłośierdzie Boże? Odpowiedziáł Sybá królowi : Jeſt jeƺcże ſyn Jonátáná / chromy ná nogi.
Y rzekł do niego król : Gdzieli jeſt? A Sybá odpowiedziáł królowi : Oto / jeſt w domu Máchirá / ſyná Ammijelowego / w Lodebárze.
Przetoż poſłáł król Dawid / y wziął go z domu Máchirá / ſyná Ammijelowego z Lodebáru.
A gdy przyƺedł Mefiboſet / ſyn Jonátáná / ſyná Saulowego / do Dawidá / upádł ná oblicże ſwe / y pokłonił śię. Y rzekł Dawid : Mefiboſećie! Który odpowiedziáł : Oto / ſługá twój.
Y rzekł do nigo Dawid : Nie bój śię ; bo zápewne ucżynię z tobą miłośierdzie dlá Jonátáná / ojcá twego / y przywrócęć wƺyſtkę rolą Saulá / dziádá twego / á ty będzieƺ jádł chleb u ſtołu mego záwżdy.
Tedy ukłoniwƺy śię / rzekł : Coż jeſt ſługá twój / żeś śię obejrzáł ná pſá zdechłego / jákom já jeſt?
Zátem wezwáł król Syby / ſługi Saulowego / y rzekł mu : Cokolwiek miáł Saul / y wƺyſtek dom jego / dáłem ſynowi páná twego.
Będzieƺ tedy ſpráwowáł rolą jego / ty / ſynowie twoi / y ſłudzy twoi / á będzieƺ dodáwáł / áby miáł chleb ſyn páná twego / któryby jádł ; ále Mefiboſet / ſyn páná twego / będzie záwżdy jádáł chleb u ſtołu mego. A Sybá miáł piętnáśćie ſynów y dwádzieśćiá ſług.
Y odpowiedziáł Sybá królowi : Wƺyſtko / co rozkázáł król / pán mój / ſłudze ſwemu / ták ucżyni ſługá twój / ácżkolwiek Mefiboſet mógłby jádáć u ſtołu mego / jáko jeden z ſynów królewſkich.
Miáł też Mefiboſet ſyná máłego / imieniem Michá ; á wƺyſcy / którzy mieƺkáli w domu Sybowym / byli ſługámi Mefiboſetowymi.
A ták Mefiboſet mieƺkáł w Jeruzálemie / bo on u ſtołu królewſkiego záwżdy jádáł ; á był chromy ná obie nogi.
