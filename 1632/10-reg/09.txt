Tedy rzekł Dawid : Jeſtże jeƺcże kto / coby pozoſtał z domu Saulowego / ábym ucżynił nád nim miłośierdźie dla Jonátáná?
Y był z domu Saulowego ſługá / którego zwano Sybá : tego záwołáno do Dawidá. Y rzekł król do niego : Tyżeś jeſt Sybá? A on odpowiedźiał : Jam jeſt / ſługá twój.
Potem rzekł król : Jeſtże jeƺcże kto z domu Saulowego / ábym nád nim ucżynił miłośierdźie Boże? Odpowiedźiał Sybá królowi : Jeſt jeƺcże ſyn Jonátáná / chromy ná nogi.
Y rzekł do niego król : Gdźieli jeſt? A Sybá odpowiedźiał królowi : Oto / jeſt w domu Máchirá / ſyná Ammijelowego / w Lodebárze.
Przetoż poſłał król Dawid / y wźiął go z domu Máchirá / ſyná Ammijelowego z Lodebáru.
A gdy przyƺedł Mefiboſet / ſyn Jonátáná / ſyná Saulowego / do Dawidá / upádł ná oblicże ſwe / y pokłonił śię. Y rzekł Dawid : Mefiboſećie! Który odpowiedźiał : Oto / ſługá twój.
Y rzekł do nigo Dawid : Nie bój śię ; bo zápewne ucżynię z tobą miłośierdźie dla Jonátáná / ojcá twego / y przywrócęć wƺyſtkę rolą Saulá / dźiádá twego / á ty będźieƺ jádł chleb u ſtołu mego záwżdy.
Tedy ukłoniwƺy śię / rzekł : Coż jeſt ſługá twój / żeś śię obejrzał ná pſá zdechłego / jákom ja jeſt?
Zátem wezwał król Syby / ſługi Saulowego / y rzekł mu : Cokolwiek miał Saul / y wƺyſtek dom jego / dáłem ſynowi páná twego.
Będźieƺ tedy ſpráwował rolą jego / ty / ſynowie twoi / y ſłudzy twoi / á będźieƺ dodáwał / áby miał chleb ſyn páná twego / któryby jádł ; ále Mefiboſet / ſyn páná twego / będźie záwżdy jádał chleb u ſtołu mego. A Sybá miał piętnaśćie ſynów y dwádźieśćiá ſług.
Y odpowiedźiał Sybá królowi : Wƺyſtko / co rozkázał król / pán mój / ſłudze ſwemu / ták ucżyni ſługá twój / ácżkolwiek Mefiboſet mógłby jádáć u ſtołu mego / jáko jeden z ſynów królewſkich.
Miał też Mefiboſet ſyná máłego / imieniem Michá ; á wƺyſcy / którzy mieƺkáli w domu Sybowym / byli ſługámi Mefiboſetowymi.
A ták Mefiboſet mieƺkał w Jeruzálemie / bo on u ſtołu królewſkiego záwżdy jádał ; á był chromy ná obie nogi.
