A uſłyƺáwƺy Izboſet / ſyn Saulá / że poległ Abner w Hebronie / zemdláły ręce jego / y wƺyſtek Izráel był przeſtráƺony.
Miał też ſyn Saulá dwóch mężów hetmánów nád hufcámi / imię jednego Bááná / á imię drugiego Recháb / ſynowie Remmoná Berotcżyká z ſynów Benjáminowych ; bo też Berod policżon był w Benjáminie.
Ućiekli tedy Berotcżykowie do Gietáim / y byli tám przychodniámi áż do onego dniá.
A Jonátán / ſyn Saulá / miał jednego ſyná chromego ná nogi / ( bo gdy miał pięć lát / á przyƺłá wieść o śmierći Saulowej / y Jonátánowej z Jezreel / wźiąwƺy go mámká jego ućiekáłá / á gdy prędko ućiekáłá / upádł y ochromiáł / )á imię jego Mefiboſet.
Poƺli tedy ſynowie Remmoná Berotcżyká / Recháb y Bááná / y weƺli / gdy był nájgorętƺy dźień / do domu Izboſetá / który ſpáł ná łóżku w południe.
Ci tedy weƺli w dom jego / jákoby kupowáć zbożá ; támże go przebili pod piąte żebro Recháb y Bááná / brát jego / y ućiekli.
Bo gdy byli weƺli w dom / á on ſpáł ná łóżku ſwem w pokoju / kędy legáł / tedy go przebili / y zábili go / á ućiąwƺy głowę jego / wźięli ją / y poƺli drogą puƺcży przez cáłą onę noc.
Y przynieśli głowę Izboſetowę do Dawidá do Hebronu / y rzekli do królá ; Oto / głowá Izboſetá / ſyná Saulowego / nieprzyjáćielá twego / który ƺukáł duƺy twojey ; á dáł Pán królowi / pánu memu / pomſtę dźiśiáj nád Saulem y nád náśieniem jego.
Tedy odpowiedźiáł Dawid Rechábowi y Báánie / brátu jego / ſynom Remmoná Berotcżyká / y rzekł do nich : Jáko żywy Pán / który wybáwił duƺę moję ze wƺego ućiſku :
Jeſlim onego / który mi oznájmił / mówiąc : Oto umárł Saul / ( choć mu śię zdáło / że weſołą nowinę przynióſł / )pojmáwƺy zábił w Sycelegu / który rozumiáł / żem mu miał dáć zápłátę zá poſelſtwo jego :
Jáko dáleko więcey ludźie niepobożne / gdyż zábili mężá ſpráwiedliwego w domu jego / ná łożu jego? A teraz / izáli nie mam ƺukáć krwi jego z ręki wáƺej / y wygłádźić was z źiemi?
A ták rozkázáł Dawid ſługom / y zábili je / á obćiąwƺy ręce ich / y nogi ich / záwieśili je nád ſtáwem w Hebronie ; ále głowę Izboſetowę wźiąwƺy pogrzebli w grobie Abnerowym w Hebronie.
