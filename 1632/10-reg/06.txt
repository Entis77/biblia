Nádto zebrał jeƺcże Dawid wƺyſtkich przebránych z Izráelá trzydźieśći tyśięcy.
A ruƺywƺy śię / ƺedł Dawid y wƺyſtek lud / który był przy nim / z Báálá Judowego / áby przenieśli z támtąd ſkrzynię Bożą / przy którey wzywano imienia Páná zaſtępów / śiedzącego ná Cherubinách / którzy ſą ná niej.
Y wſtáwili ſkrzynię Bożą ná wóz nowy / wźiąwƺy ją z domu Abinádábowego / który jeſt w Gábáá ; lecż Ozá y Achyjo / ſynowie Abinádábowi / prowádźili on wóz nowy.
Y wźięli ją z domu Abinádábá / który był w Gábáá / á ƺli z ſkrzynią Bożą ; lecż Achyjo ƺedł przed ſkrzynią.
Dawid záśię y wƺyſtek Izráel gráli przed Pánem ná wƺelákich inſtrumentách z cedrowego drzewá / ná hárfách y ná ſkrzypcách / y ná bębnách / y ná piƺcżáłkách / y ná cymbáłách.
A gdy przyƺli do gumná Náchonowego / śćiągnął Ozá rękę ſwoję ku ſkrzyni Bożey / y zádźierżał ją ; bo woły były wyſtąpiły z drogi :
Przetoż rozgniewał śię bárzo Pán ná Ozę / y zábił go tám Bóg dla śmiáłośći / y támże umárł przy ſkrzyni Bożey.
Y záfráſował śię Dawid / że Pán ſrodze záráźił Ozę / y názwał miejſce ono Perezozá / áż do dniá tego.
A ták uląkł śię Dawid Páná dniá onego / y mówił : Jákoż wnijdźie do mnie ſkrzyniá Páńſka?
Przetoż niechćiał Dawid wprowádźić do śiebie ſkrzyni Páńſkiej do miáſtá ſwego / ále ją kázał wprowádźić w dom Obededomá Gietejcżyká.
Y zoſtáłá ſkrzyniá Páńſka w domu Obededomá Gietejcżyká przez trzy mieśiące / y błogoſłáwił Pán Obededomowi / y wƺyſtkiemu domowi jego.
Y oznájmiono królowi Dawidowi / mówiąc : Błogoſłáwi Pán domowi Obededomá / y wƺyſtkiemu / co ma / dla ſkrzyni Bożey. Tedy ƺedƺy Dawid / wźiął ſkrzynię Bożą z domu Obededomá do miáſtá Dawidowego z weſelem.
A gdy ći / którzy nieśli ſkrzynię Páńſką / poſtąpili ná ƺeść kroków / ofiárował wołu y báráná tłuſtego.
Y ſkákał Dawid ze wƺyſtkiej mocy przed Pánem / á był Dawid oblecżony w efod lniány.
A ták Dawid / y wƺyſtek dom Izráelſki prowádźili ſkrzynię Páńſką z weſelem / y z trąbieniem.
Y ſtáło śię / gdy ſkrzyniá Páńſka wchodźiłá do miáſtá Dawidowego / że Michol / córká Saulowá / wyglądájąc oknem / á widząc królá Dawidá ze wƺyſtkiej mocy ſkácżącego przed Pánem / wzgárdźiłá go w ſercu ſwojim.
A gdy przynieśli ſkrzynię Páńſką / poſtáwili ją ná miejſcu ſwem w pośrzodku namiotu / który był dla niej rozbił Dawid / y ofiárował Dawid przed Pánem cáłopalenia y ſpokojne ofiáry.
A gdy dokońcżył Dawid ofiárowáć cáłopalenia / y ofiár ſpokojnych / błogoſłáwił ludowi w imię Páná zaſtępów.
Y dał miedzy wƺyſtek lud / y miedzy wƺyſtko zgromádzenie Izráelſkie / od mężá áż do niewiáſty / káżdemu bochenek chlebá jeden / y jednę ƺtukę mięſá / y łágiew jednę winá. Y odƺedł wƺyſtek lud / káżdy do domu ſwego.
Potem wróćił śię Dawid / áby błogoſłáwił domowi ſwemu. Y wyƺłá Michol / córká Saulowá / przećiwko Dawidowi / á rzekłá : O jákoż chwálebny był dźiś król Izráelſki / który śię odkrywał dźiś przed ocżymá ſłużebnic ſług ſwojich / jáko śię zwykł odkrywáć jeden z ƺálonych!
Tedy rzekł Dawid do Michol : Przed Pánem ( który mnie rácżey obrał niż ojcá twego / y niżeli wƺyſtek dom jego / rozkázując mi / ábym był kśiążęćiem nád ludem Páńſkim / nád Izráelem / ) gráłem / y będę grał przed Pánem.
A im będę podlejƺym / niżlim śię ſtał / y uniżeńƺym w ocżách mojich / tym u tych ſłużebnic / o których mi powiádáƺ / będę chwálebniejƺy.
Przetoż Michol / córká Saulowá / niemiáłá dźiátek áż do dniá śmierći ſwojey.
