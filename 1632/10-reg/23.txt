A teć ſą oſtátecżne ſłowá Dawidowe. Rzekł Dawid / ſyn Iſájego / rzekł mówię mąż / który był zácnie wywyżƺony / pomázániec Bogá Jákóbowego / y wdzięcżny w pieśniách Izráelſkich :
Duch Páńſki mówił przez mię / á ſłowá jego pochodziły przez język mój.
Mówił Bóg Izráelſki do mnie / mówiłá ſkáłá Izráelſká : Ten / który pánowáć będzie nád ludem / będzie ſpráwiedliwy / pánowáć będzie w bojáźni Bożej.
Będzie jáko bywá świátłość poránná / gdy ſłońce ráno bez obłoków wſchodzi / á jáko od jáſnośći po deƺcżu wyráſtá ziele z ziemi.
A choć nie táki jeſt dom mój przed Bogiem / jednák przymierze wiecżne poſtánowił ze mną / utwierdzone we wƺyſtkiem y obwárowáne. A w temći jeſt wƺyſtko zbáwienie moje / y wƺyſtká ućiechá mojá / ácżkolwiek temu jeƺcże wzroſtu nie dáwá.
Ale nipobożni wƺyſcy będą jáko ćierń wyrwáni / którego rękomá nie biorą.
Lecż kto śię go jedno chce dotknąć / obwáruje śię żelázem y drzewem włócżni / álbo ogniem wypálá go do ƺcżętu ná miejſcu jego.
Teć ſą imioná mocárzów / które miáł Dawid : Jozeb Báſebet Táchmojcżyk / nájprzedniejƺy między trzemá ; który śię z ućiechą rzućił ná ośm ſet ludu z włócżnią / áby je zábił w jednej potrzebie.
A po nim był Eleázár / ſyn Dodoná / ſyná Ahohowego / między trzemá mocárzámi / którzy byli z Dawidem ; á ſromotnie lżyli Filiſtyny / którzy śię byli zebráli ku bitwie / gdy byli odćiągnęli mężowie Izráelſcy.
Ten powſtáwƺy bił Filiſtyny / ták iż uſtáłá ręká jego / y zdrętwiáłá ręká jego przy miecżu. Tedy ſpráwił Pán wielkie wybáwienie dniá onego / ták / że śię lud wróćił zá nim / tylko áby korzyśći zebráł.
A po nim był Semmá / ſyn Agi / Hárárcżyk ; ábowiem gdy śię byli Filiſtynowie zebráli do kupy / kędy byłá cżęść polá pełnego ſocżewicy / á lud inny był ućiekł przed Filiſtynámi :
Tedy ſtánąwƺy w pośród onej cżęśći polá / obronił go / y pobił Filiſtyny. A ták ſpráwił Pán wielkie wybáwienie.
Wyƺli też oni trzej z trzydzieſtu przedniejƺych / á przyƺli we żniwá do Dawidá / do jáſkini Odollám / gdy śię wojſko Filiſtyńſkie było obozem położyło w dolinie Refáim.
A Dawid ná ten cżáſ był ná miejſcu obronnem : ſtráż też Filiſtyńſká ná ten cżáſ byłá w Betlehem.
Tedy prágnął Dawid / y rzekł : O by mi śię kto dáł nápić wody z ſtudni Betlehemſkiej / którá jeſt u brámy!
Przetoż wpádli ći trzej mocárze do obozu Filiſtyńſkiego / y nácżerpáli wody z ſtudni Betlehemſkiej / którá byłá u brámy ; którą nieśli / y przynieśli do Dawidá. Ale jej on nie chćiáł pić / lecż ją wyláł przed Pánem.
Y rzekł : Nie dáj mi tego Pánie / ábym to miáł ucżynić. Izáli to nie krew mężów / którzy ƺli z niebezpiecżeńſtwem duƺ ſwoich? Y nie chćiáł jej pić. Toć ucżynili oni trzej mocárze.
Tákże Abiſáj brát Joábá / ſyn Sárwii / był przedniejƺym między trzemá. Ten podnióſł włócżnią ſwą przećiwko trzemá ſtom / y zábił je / y był ſłáwnym między trzemá.
Z tych trzech będąc nájſłáwniejƺym / był ich hetmánem ; wƺákże onych trzech pierwƺych nie doƺedł.
Bánájáſ też ſyn Jojády / ſyn mężá rycerſkiego / zácny w ſwych ſpráwách / z Kábſeel ; ten zábił dwóch mocárzów Moábſkich / tenże ƺedłƺy zábił lwá w pośrodku ſtudni / we dni śnieżne.
Tenże zábił mężá Egipcżániná / mężá ná podziw wielkiego / który Egipcżánin miáł w ręku włócżnią ; á on ƺedł ku niemu z kijem / á wydárłƺy włócżnią z ręki Egipcżániná / zábił go włócżnią jego.
Toć ucżynił Bánájáſ / ſyn Jojády / który też był ſłáwny między onymi trzemá mocárzámi.
Z tych trzydzieſtu był nájſłáwniejƺym ; wƺákże onych trzech nie doƺedł ; y poſtáwił go Dawid nád drábántámi ſwoimi.
Był też Aſáel / brát Joábá / między trzydzieſtomá. A ći ſą : Elkánán / ſyn Dodoná / Betlehemcżyk ;
Sámmá Hárodcżyk ; Eliká Hárodcżyk.
Heleſ Fáltycżyk ; Hyrá / ſyn Ikkieſá / Tekuitcżyk ;
Abijezer Anátotcżyk ; Mobonáj Huſátcżyk ;
Selmon Ahohytcżyk ; Máháráj Netofátcżyk ;
Heleb / ſyn Báány / Netofátcżyk ; Itáj / ſyn Rybájego / z Gábáát ſynów Benjáminowych ;
Bánájáſ Fárátońcżyk ; Háddáj od potoku Gááſ ;
Abijálbon Arbátcżyk ; Aſmewet Bárchomcżyk ;
Elijáchbá Sálábońcżyk ; z ſynów Jáſſonowych Jonátán ;
Semmá Hororcżyk ; Ahijám / ſyn Sárárá / Arárytcżyk ;
Elifelet / ſyn Acháſbájá / ſyná Máchátego ; Elijám / ſyn Achitofelá Gilońcżyká.
Hezráj Kármelcżyk ; Fáráj Arbitcżyk.
Igál / ſyn Nátáná z Soby ; Báni Gádcżyk.
Selek Ammonitcżyk ; Nácháráj Berotcżyk / który nośił broń Joábá / ſyná Sárwii ;
Hirá Jetrytcżyk ; Gáreb Jetrytcżyk.
Uryjáƺ Hetejcżyk. Owá wƺyſtkich trzydzieśći y śiedm.
