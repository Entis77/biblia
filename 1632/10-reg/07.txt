Y ſtáło śię / gdy śiedźiał król w domu ſwym / á Pán mu dał odpocżynienie zewƺąd od wƺyſtkich nieprzyjáćiół jego /
Że rzekł król do Nátáná proroká : Obácż proƺę / ja mieƺkám w domu cedrowym / á ſkrzyniá Boża mieƺká miedzy kortynámi.
Tedy rzekł Nátán do królá : Cokolwiek jeſt w ſercu twojim / idź / ucżyń ; bo Pán jeſt z tobą.
Potem onejże nocy ſtáło śię ſłowo Páńſkie do Nátáná / mówiąc :
Idź á mów do ſługi mego Dawidá : Ták mówi Pán : Izali mi ty zbudujeƺ dom ku mieƺkániu?
Poniewáżem nie mieƺkał w domu ode dniá / któregom wywiódł ſyny Izráelſkie z Egiptu / áż do dniá tego / álem chodźił w námioćie y w przybytku ;
Y wƺędźie / gdźiem chodźił ze wƺyſtkimi ſyny Izráelſkimi / izálim y ſłowo rzekł któremu z ſędźiów Izráelſkich / któremum rozkázał páść lud mój Izráelſki / mówiąc : Cżemużeśćie mi nie zbudowáli domu cedrowego?
Przetoż teraz to powiedz ſłudze memu Dawidowi : Ták mówi Pán zaſtępów : Jam ćiebie wźiął z owcżárni od owiec / ábyś był wodzem nád ludem moim / nád Izráelem ;
Y byłem z tobą wƺędy / gdźieśkolwiek chodźił / y wygłádźiłem wƺyſtkie nieprzyjáćioły twoje przed tobą / y ucżyniłem ći imię wielkie / jáko imię wielkich ludźi / którzy ſą ná źiemi.
Y poſtánowię miejſce ludowi memu Izráelſkiemu / y wƺcżepię go / iż będźie mieƺkał ná miejſcu ſwem / y nie będźie więcey poruƺony / áni go więcey ſynowie niepráwośći trápić będą / jáko przed tym.
Ode dniá / któregom poſtánowił ſędźie nád ludem moim Izráelſkim ; y dam ći odpocżynienie od wƺyſtkich nieprzyjáćiół twojich. Przetoż opowiádáć Pán / że on ſám tobie dom zbuduje.
Gdy śię wypełnią dniá twoje / y záśnieƺ z ojcy twoimi / wzbudzę naśienie twoje po tobie / które wynijdźie z żywotá twego / á umocnię króleſtwo jego ;
On zbuduje dom imieniowi memu / á Ja utwierdzę ſtolicę króleſtwá jego áż ná wieki.
Ja mu będę zá ojcá / á on mi będźie zá ſyná / który gdy wyſtąpi / ſkárżę go rózgą ludzką / y plágámi ſynów cżłowiecżych.
Lecż miłośierdźie moje nie będźie odjęte od niego / jákom je odjął od Saulá / któregom odrzućił przed twárzą twoją.
Y będźie utwierdzony dom twój / y króleſtwo twoje áż ná wieki przed tobą / á ſtolicá twojá będźie trwáłá áż ná wieki.
Według wƺyſtkich ſłów tych / y według wƺyſtkiego widzeniá tego / ták mówił Nátán do Dawidá.
Tedy wƺedƺy król Dawid / uśiádł przed oblicżem Páńſkim / y rzekł : Cóżem ja jeſt Pánie Boże / y co zá dom mój / żeś mię przywiódł áż dotąd?
Lecż y to máło było przed oblicżnośćią twoją / Pánie Boże ; áleś też obietnicę ucżynił o domie ſługi twego ná cżás dáleki / á to práwie obycżájem ludzkim / Pánie Boże!
Y cóż więcey ma mówić Dawid przed tobą? ábowiem ty znáƺ ſługę twego / o Pánie Boże.
Dla ſłowá twego / á według ſercá twego ucżyniłeś te wƺyſtkie wielkie rzecży / oznájmując je ſłudze twemu.
Przetoż wielmożnym jeſteś / Pánie Boże ; bo niemáƺ podobnego tobie / y niemáƺ Bogá oprócż ćiebie / według wƺyſtkiego / cośmy ſłyƺeli w uƺy náƺe.
Y gdźież jeſt táki lud ná źiemi / jáko Izráel? dla któregoby Bóg ƺedł / áby go ſobie odkupił zá lud / y ucżynił ſobie imię / y ſpráwił wam wielkie y ſtráƺne rzecży w źiemi twojey / przed oblicżem ludu twego / któryś ſobie wykupił z Egiptu / z pogáńſtwá y z bogów ich ;
Y zmocniłeś ſobie lud twój Izráelſki / ábyć był ludem áż ná wieki ; á ty Pánie ſtáłeś śię im zá Bogá.
Przetoż teraz / o Pánie Boże / ſłowo / któreś powiedźiał o ſłudze twoim y o domu jego / utwierdź áż ná wieki / á ucżyń ták jákoś mówił /
Aby uwielbione było imię twoje áż ná wieki / żeby mówiono : Pán zaſtępów Bogiem nád Izráelem / á dom ſługi twego Dawidá będźie umocniony przed twárzą twoją.
Abowiem ty Pánie zaſtępów / Boże Izráelſki / objáwiłeś ſłudze twemu / mówiąc : Dom zbuduję tobie. Przetoż zá ſłuƺną rzecż ználázł ſługá twój w ſercu ſwojim / áby śię modlił tobie tą modlitwą.
Teraz tedy / Pánie Boże / tyś ſám Bóg / á ſłowá twe ſą práwdą / y rzekłeś do ſługi twojego te dobre rzecży.
Rácżże już teraz błogoſłáwić domowi ſługi twego / áby trwał ná wieki przed tobą ; boś ty Pánie Boże rzekł : Że błogoſłáwieńſtwem twojim będźie ubłogoſłáwion dom ſługi twego ná wieki.
