Tedy śię tám pojáwił mąż niepobożny / którego zwáno Sebá / ſyn Bichry / mąż Jemini. Ten zátrąbił w trąbę / y rzekł : Nie mámy my dziáłu w Dawidzie / áni mámy dziedzictwá w ſynu Iſájego ; wróć śię káżdy do námiotów ſwoich / o Izráelu!
A ták odſtąpili wƺyſcy mężowie Izráelſcy od Dawidá zá Sebą / ſynem Bichry ; ále mężowie Judzcy trzymáli śię królá ſwego / od Jordánu áż do Jeruzálemu.
Y przyƺedł Dawid do domu ſwego w Jeruzálemie ; á wziąwƺy król dzieśięć niewiáſt záłożnic / które był zoſtáwił / áby ſtrzegły domu / oddáł je pod ſtráż / y żywił je / ále do nich nie wchodził ; y były pod ſtráżą áż do dniá śmierći ſwojej / we wdowim ſtánie.
Potem rzekł król do Amázy : Zbierz mi męże Judzkie zá trzy dni ; ty śię też tu ſtáw.
A ták poƺedł Amázá / áby zebráł lud Judzki ; lecż śię zábáwił nád cżáſ náznácżony / który mu był náznácżył.
Y rzekł Dawid do Abiſájego : Teráz gorzej nám ucżyni Sebá / ſyn Bichry / niż Abſálom ; przetoż ty weźmij ſługi páná twego / á goń go / by ſnáć nie ználázł ſobie miáſt obronnych / y nie uƺedł z ocżu náƺych.
Tedy wyƺli z nim mężowie Joábowi / y Chertcżycy y Feletcżycy / y wƺyſtko rycerſtwo / á wyƺli z Jeruzálemu w pogoń zá Sebą / ſynem Bichry.
A gdy byli u wielkiego kámieniá / który jeſt w Gábáon / tedy im Amázá zábieżáł. A Joáb miáł przepáſáną ƺátę ſwą / w której chodził / á ná niej páſ z miecżem przypáſány do biódr ſwoich w pochwách ſwych / którego ſnádnie mógł dobyć / y záś ſchowáć.
Y rzekł Joáb do Amázy : Jákoż śię máƺ / bráćie mój? Y ujął ręką práwą Joáb Amázę zá brodę / jákoby go cáłowáć miáł.
Ale Amázá nie poſtrzegł miecżá / który był w ręce Joábowej : y przebił go nim pod piąte żebro / y wyláł trzewá jego ná ziemię / á ták zá jádną ráną umárł. A Joáb y Abiſáj / brát jego / ƺli w pogoń zá Sebą / ſynem Bichry.
Tedy ſtánął jeden nád nim z ſług Joábowych / y rzekł : Ktokolwiek jeſt życżliwy Joábowi / á ktokolwiek trzymá z Dawidem / niech idzie zá Joábem.
Lecż Amázá wáláł śię w krwi w pośród drogi. A widząc on mąż / iż śię záſtánáwiáł wƺyſtek lud nád nim / zwlekł Amázę z drogi ná pole / y przyrzućił go ƺátą / gdyż widziáł / że ktokolwiek ƺedł mimo niego / záſtánáwiáł śię.
A gdy był zwlecżony z drogi / bieżáł káżdy mąż zá Joábem / goniąc Sebę / ſyná Bichry.
Który już był przeƺedł przez wƺyſtkie pokoleniá Izráelſkie / áż do Abel y Betmááchá / ze wƺyſtkimi Berymcżykámi / którzy śię też byli zebráli / á ƺli zá nim.
A gdy śię tám śćiągnęli / oblegli go w Abeli Betmááchá / y uſypáli ƺáńce przećiw miáſtu / ták iż ſtáli przed murem / á wƺyſtek lud / który był z Joábem / uśiłowáł obálić mury.
Wtem záwołáłá z miáſtá niektórá niewiáſtá mądrá : Słuchájćie / ſłuchájćie! rzecżćie proƺę do Joábá : Przyſtąp ſám / á rozmówię śię z tobą.
Który gdy do niej przyſtąpił / rzekłá mu oná niewiáſtá : Tyżeś jeſt Joáb? Y odpowiedziáł : Jeſtem. Tedy mu rzekłá : Słucháj ſłów ſłużebnicy twojej ; y odpowiedziáł : Słuchám.
Przetoż rzekłá / mówiąc : Powiádáno przedtem / mówiąc : Koniecżnie pytáć śię będą w Abelu / á ták śię wƺyſtko ſpráwi.
Jám jeſt jedno miáſto z ſpokojnych y wiernych w Izráelu / á ty ƺukáƺ / ábyś zátráćił miáſto y mátkę w Izráelu ; przecżże chceƺ zburzyć dziedzictwo Páńſkie?
Y odpowiedziáł jej Joáb / mówiąc : Niedáj / niedáj mi tego Boże / ábym miáł podwróćić y zburzyć je.
Nie tákći śię rzecż má. Ale mąż z góry Efráim / imieniem Sebá / ſyn Bichry / podnióſł rękę ſwą przećiw królowi Dawidowi ; wydájćież go ſámego / á odćiągnę od miáſtá. Zátem rzekłá niewiáſtá do Jáobá : Oto głowę jego zrzucą do ćiebie z muru.
A ták ſpráwiłá to oná niewiáſtá u wƺyſtkigo ludu mądrośćią ſwoją / że śćiąwƺy głowę Sebie / ſynowi Bichry / zrzućili ją do Jáoábá ; który zátrąbił w trąbę / y rozeƺli śię wƺyſcy od miáſtá / káżdy do námiotów ſwoich ; Joáb śię też wróćił do królá do Jeruzálemu.
Y był Joáb hetmánem nád wƺyſtkiem wojſkiem Izráelſkiem / á Bánájáſ / ſyn Jojády / nád Cheretcżykámi y nád Feletcżykámi.
Adorám był poborcą / á Jozáfát / ſyn Ahiludá / kánclerzem.
Sejá piſárzem / á Sádok y Abijátár byli kápłánámi.
Hirá tákże Jáirtcżyk był kśiążęćiem u Dawidá.
