Nádto rzekł Achitofel do Abſálomá : Niech proƺę wybiorę dwánaśćie tyśięcy mężów / á wſtáwƺy będę gonił Dawidá tey nocy ;
Y przypádnę nań / pokąd jeſt ſprácowány y zemdlonych rąk / á ſtrwożę go / y ućiecże wƺyſtek lud / który jeſt z nim / á zábiję królá ſámego.
A ták przywrócę wƺyſtek lud do ćiebie ; bo jákoby śię wƺyſcy ku tobie náwróćili / gdy zábiję tego mężá / którego ty ƺukáƺ / á wƺyſtek śię lud uſpokoi.
Y ſpodobáło śię to Abſálomowi / y wƺyſtkim ſtárƺym Izráelſkim.
Jednák rzekł Abſálom : Záwołáj rychło y Chuſájá Aráchity / ábyſmy uſłyƺeli / co on też powie.
A gdy przyƺedł Chuſáj do Abſálomá / rzekł Abſálom do niego / mówiąc : Ták powiedźiał Achitofel : Mámyli ucżynić według rády jego / cżyli nie? y ty powiedz.
Tedy odpowiedźiał Chuſáj Abſálomowi : Nie dobrá jeſt rádá / którą teraz dał Achitofel.
Nádto rzekł Chuſáj : Świádomyś ojcá twego y mężów jego / iż ſą mężni / y ſercá zájuƺonego / jáko niedźwiedźicá ośieroćiáłá w polu ; do tego oćiec twój jeſt mąż wálecżny / y nie będźie nocował z ludem.
A podobno y teraz śię kryje w jákiej jáſkini / álbo ná któremkolwiek miejſcu. Y ſtáłoby śię / jeſliżeby kto z twojich poległ ná tym pocżątku / żeby káżdy / ktoby o tym uſłyƺał / rzekł : Stáłá śię poráżká w ludźie / który ƺedł zá Abſálomem.
Tedy y nájmężniejƺy / którego ſerce jáko ſerce lwie / bárzo oſłábieje ; bo wie wƺyſtek Izráel / że mężnym jeſt oćiec twój / y mężni wƺyſcy / którzy ſą z nim.
Aleć rádzę / áby śię do ćiebie cále zebrał wƺyſtek Izráel od Dán áż do Beerſebá / jáko piáſek / który jeſt przy morzu w mnóſtwie / á ty żebyś oſobą ſwą ƺedł ná wojnę.
A ták poćiągniemy przećiwko niemu / ná któremkolwiek miejſcu ználeźiony będźie / y przypádniemy nań / jáko pádá roſá ná źiemię / y nie zoſtánie z niego / to jeſt / z tych wƺyſtkich mężów / którzy ſą z nim / áni jeden.
A jeſliżby do którego miáſtá uƺedł / tedy znieśie wƺyſtek Izráel do onego miáſtá powrozy / á poćiągniemy je áż do potoku / ták iż tám nie będźie ználeźion áni kámyk.
Tedy rzekł Abſálom y wƺyſcy mężowie Izráelſcy : Lepƺá jeſt rádá Chuſájego Aráchity / niż rádá Achitofelowá. Abowiem Pán był poſtánowił / áby rozerwáná byłá rádá Achitofelowá / która byłá dobrá / á ták áby przywiódł Pán złe ná Abſálomá.
Y oznájmił Chuſáj Sádokowi y Abijátárowi / kápłánom : Ták á ták rádźił Achitofel Abſálomowi / y ſtárƺym Izráelſkim ; álem ja ták á ták rádźił.
Teraz tedy poślijćie co rychlej / á oznájmijćie Dawidowi / mówiąc : Nie zoſtáwáj tey nocy w równinách puƺcży ; ále bez odwłoki przejdź / by ſnadź nie był pożárty król / y wƺyſtek lud / który jeſt z nim.
A Jonátán y Achimáás ſtáli u ſtudni Rogiel : y poƺłá dźiewká / á oznájmiłá im / áby poƺli / y donieśli to królowi Dawidowi ; bo śię nie śmieli ukázáć / áni wnijść do miáſtá.
Wƺákże obácżył je niektóry ſługá y powiedźiał Abſálomowi. Przetoż poƺedƺy obádwáj ſpieƺno / weƺli w dom niektórego mężá w Báhurym / który miał ſtudnią ná dworze ſwym / y ſpuśćili śię do niej.
A wźiąwƺy niewiáſtá płáchtę / rozćiągnęłá ją ná wierzchu ſtudni / y náſypáłá ná niej krup ; á ták śię tego nie dowiedźiáno.
Bo gdy przyƺli ſłudzy Abſálomowi do oney niewiáſty w dom / rzekli : Gdźie jeſt Achimáás y Jonátán? odpowiedźiáłá im niewiáſtá : Przeƺli przez rzekę ; á poƺukáwƺy ich / y nie ználázƺy / wróćili śię do Jeruzálemu.
A gdy oni odeƺli / tedy owi wyſtąpiwƺy z ſtudni poƺli / y oznájmili królowi Dawidowi / mówiąc do niego : Wſtáńćie / przepráwćie śię co rychlej przez wodę ; ábowiem ták rádźił przećiwko wam Achitofel.
Przetoż wſtáwƺy Dawid / y wƺyſtek lud / który był z nim / przepráwili śię przez Jordán / pierwej niż śię rozedniáło / á nie zoſtał y jeden / któryby śię nie przepráwił przez Jordán.
Tedy Achitofel widząc / iż śię nie ſtáło podług rády jego / ośiodłał oſłá / á wſtáwƺy jechał do domu ſwego / do miáſtá ſwego / á rozpráwiwƺy dom ſwój / powieśił śię / y umárł / á pogrzebion jeſt w grobie ojcá ſwego.
A Dawid już był przyƺedł do Máhánáim / gdy śię Abſálom przepráwił przez Jordán / on y wƺyſcy mężowie Izráelſcy z nim.
Y przełożył Abſálom Amázę / miáſto Joábá / nád wojſkiem. A ten Amázá był ſynem mężá / którego imię było Itrá / Izráelcżyk / który był wƺedł do Abigájli / córki Náháſowej / śioſtry Sárwii / mátki Joábowej.
Y położył śię obozem Izráel z Abſálomem w źiemi Gáláád.
Y ſtáło śię / gdy przyƺedł Dawid do Máhánáim / że Soby / ſyn Náháſowy z Rábby / ſynów Ammonowych / y Máchir / ſyn Ammijelowy z Lodebáru / y Bárſyláj Gáláádcżyk z Rogielim /
Pośćiel / y miednice / y nacżynia zduńſkie / y pƺenicę / y jęcżmień / y mąki / y krupy / y boby / y ſocżewice / y práżmá /
Y miodu / y máſłá / y owiec / y ſerów krowich przynieśli Dawidowi / y ludowi / który był z nim / áby jedli : bo mówili : Lud ten głodny jeſt / y ſprácowány / y prágnieniem zmorzony ná puƺcży.
