Y był głód zá dni Dawidowych przez trzy látá / jednego roku po drugim. Tedy ƺukał Dawid oblicża Páńſkiego / któremu rzekł Pán : Dla Saulá / y dla domu jego krwáwego / przeto iż pomordował Gábáonity.
Przyzwał tedy król Gábáonitów / y rzekł do nich : ( A ći Gábáonitowie nie byli z ſynów Izráelſkich / ále z oſtátków Amorejcżyków / którym ácż byli ſynowie Izráelſcy przyśięgli / wƺákże je uśiłował Saul wyplenić z gorliwośći ſwej dla ſynów Izráelſkich y Judzkich. )
Y rzekł Dawid do Gábáonitów : Cóż wam mam ucżynić? á cżym was ubłágáć / ábyśćie błogoſłáwili dźiedźictwu Páńſkimu?
Y odpowiedźieli mu Gábáonitowie : Nie idźie nam o ſrebro áni o złoto z Saulem / y z domem jego / áni o to / żebyſmy zábili kogo w Izráelu. A on rzekł : Cokolwiek rzecżećie / ucżynię wam.
Którzy rzekli do królá : Mężá / który nas wygubił / y ná tym był / áby nas do ƺcżętu wytráćił / żeby nas nic nie zoſtáło we wƺyſtkich gránicách Izráelſkich.
Wydájćie nam śiedmiu mężów z ſynów jego / á powieśimy je Pánu w Gábáá Saulá / niekiedy wybránego Páńſkiego. Tedy rzekł : Wydám.
Lecż ſfolgował król Mefiboſetowi / ſynowi Jonátáná / ſyná Saulowego / dla przyśięgi Páńſkiej / która byłá miedzy nimi / miedzy Dawidem y miedzy Jonátánem / ſynem Saulowym.
Ale wźiął król dwu ſynów Reſfy / córki Ai / które porodźiłá Saulowi / Armoniego y Mefiboſetá / y pięćiu ſynów śioſtry Micholi / córki Saulowej / które porodźiłá Adryjelowi / ſynowi Bárſylájá Meholátyckiego /
Y wydał je w ręce Gábáonitów / y powieśili je ná górze przed Pánem. Y umárli oni śiedmiu poſpołu / á pobići ſą w pierwƺe dni żniwá / ná pocżątku żniwá jęcżmiennego.
A wźiąwƺy Reſfá / córká Ai / wór / rozpoſtárłá go ná ſkále / ná pocżątku żniwá / áżby ná nie kropił deƺcż z niebá / y nie dopuƺcżáłá ptáſtwu powietrznemu / pádáć ná nie we dnie / áni zwierzowi polnemu w nocy.
Tedy oznájmiono Dawidowi / co ucżyniłá Reſfá / córká Ai / záłożnicá Saulowá.
Przetoż ƺedƺy Dawid wźiął kośći Saulowe / y kośći Jonátáná / ſyná jego od ſtárƺych Jábes Gáláádſkiego / którzy je byli ukrádli z ulicy Betſáńſkiej / kędy je byli záwieśili Filiſtynowie onegoż dniá / gdy poráźili Filiſtynowie Saulá w Gielboe.
A ták wźiął z támtąd kośći Saulowe y kośći Jonátáná / ſyná jego ; zebráno też kośći powieƺonych /
Y pogrzebli kośći Saulowe y Jonátáná / ſyná jego / w źiemi Benjáminá w Selá / w grobie Cyſá / ojcá jego / á ucżynili wƺyſtko / co był rozkázał król ; á ták potem ubłágány był Bóg źiemi.
Y byłá záśię wojná miedzy Filiſtynámi y Izráelem ; y ćiągnął Dawid y ſłudzy jego z nim / á wálcżyli przećiwko Filiſtynom / ták / że uſtał Dawid.
Tedy Jeſbibenob / który był z ſynów jednego olbrzymá / ( á grot drzewcá jego wáżył trzy ſtá ſyklów miedźi / á miał przepáſány miecż nowy ) umyślił był zábić Dawidá.
Ale go rátował Abiſáj / ſyn Sárwii / á rániwƺy Filiſtyná zábił go. Przetoż przyśięgli mężowie Dawidowi / mówiąc mu : Nie pójdźieƺ więcey z námi ná wojnę / ábyś nie zgáśił pochodni Izráelſkiej.
Y ſtáło śię potem / że byłá znowu wojná w Gob z Filiſtynámi / y zábił Sobocháj Huſátycki Sáfá / który był z ſynów tegoż olbrzymá.
Byłá też jeƺcże inná wojná w Gob z Filiſtynámi / kędy zábił Elháná / ſyn Jáárá Oregim / Betlehemcżyk / brátá Golijátowego z Giet / którego drzewce u włócżni było jáko náwój tkácki.
Nádto jeƺcże byłá wojná w Giet / kędy był mąż wielkiego wzroſtu / májąc po ƺeść pálców u rąk ſwojich / y po ƺeść pálców u nóg ſwojich / wƺyſtkich dwádźieśćiá y cżtery ; á ten też był ſynem tegoż olbrzymá.
Ten gdy urągał Izráelowi / zábił go Jonátán / ſyn Sámáá / brátá Dawidowego.
Ci cżterej byli ſynowie jednego olbrzymá z Giet / á ći polegli od ręki Dawidowej / y od ręki ſług jego.
