Y oznájmiono Joábowi : Oto król płácże y żáłuje Abſálomá.
Przetoż śię ono zwyćięſtwo dniá onego obróćiło w płácż wƺyſtkiemu ludowi ; ábowiem uſłyƺał lud dniá onego / że mówiono : Żáłośny jeſt król dla ſyná ſwego.
Zácżem wkrádł śię lud onego dniá wchodząc do miáſtá / jáko śię więc wkrádá lud / który śię wſtydźi / ućiekájąc z bitwy.
A król nákrywƺy oblicże ſwoje / wołał głoſem wielkim : Synu mój Abſálomie / Abſálomie / ſynu mój / ſynu mój!
Tedy wƺedł Joáb do królá w dom / y rzekł : Sháńbiłeś dźiś oblicże wƺyſtkich ſług twojich / którzy wybáwili duƺę twoję dźiśiáj / y duƺę ſynów twojich / y córek twojich / y duƺę żon twojich / y duƺę záłożnic twojich.
Miłując te / którzy ćię máją w nienáwiśći / á nienáwidząc tych / którzy ćię miłują. Abowiem pokázáłeś dźiś / że ſobie nie powáżáƺ hetmánów / y ſług twojich ; bom doznał tego dźiś / że gdyby Abſálom był żyw / choćbyſmy my wƺyſcy dźiś byli pobići / tedyćby śię to bárzo podobáło.
Przetoż teraz wſtáń / wynijdź / á mów łágodnie do ſług twojich. Boć przez Páná przyśięgám / jeſli ty nie wynijdźieƺ / że nie zoſtánie żáden z tobą tey nocy / á będźieć to gorzej / niżli wƺyſtko złe / którekolwiek ná ćię przychodźiło od młodośći twojey áż dotąd.
Wſtał tedy król / y śiádł w bramie ; y opowiedźiáno to wƺyſtkiemu ludowi / mówiąc : Oto król śiedźi w bramie. Y przyƺedł wƺyſtek lud przed oblicże królewſkie ; ále Izráelcżycy ućiekli byli / káżdy do namiotu ſwego.
Y ſtáło śię / że śię wƺyſtek lud ſprzecżał z ſobą we wƺyſtkich pokoleniách Izráelſkich / mówiąc : Król wyrwał nas z rąk nieprzyjáćiół náƺych / tenże nas też wyrwał z rąk Filiſtyńſkich / á teraz ućiekł z źiemi przed Abſálomem.
Lecż Abſálom / któregośmy byli pomázáli nád ſobą / zginął w bitwie ; á teraz przecżże wy zániedbywáćie przyprowádźić záśię królá?
Przetoż król Dawid poſłał do Sádoká y do Abijátárá / kápłanów / z temi ſłowy : Powiedzćie ſtárƺym Judzkim / mówiąc : Przecżże máćie być pośledniejƺymi w przyprowádzeniu záśię królá do domu jego? Abowiem ſłowá wƺyſtkiego Izráelá dochodźiły królá do domu jego.
Bráćiaśćie moi / kość mojá / á ćiáłośćie moje ; przecżże tedy máćie być pośledniejƺymi w przywróceniu królá?
Amáźie tákże powiedzćie : Izáliś ty nie jeſt kość mojá / y ćiáło moje? To niech mi ucżyni Bóg / y to niech przycżyni / jeſli hetmánem wojſká nie będźieƺ przedemną po wƺyſtkie dni / miáſto Joábá.
A ták nákłonił ſerce wƺyſtkich mężów Judzkich jáko mężá jednego / że poſłáli do królá mówiąc : Náwróć śię ty / y wƺyſcy ſłudzy twoi.
Wróćił śię tedy król / y przyƺedł áż do Jordánu ; á lud Judzki wyƺedł był do Gálgál / áby záƺedł w drodze królowi / á przeprowádźił królá przez Jordán.
Poſpieƺył śię tákże Semej / ſyn Giery / ſyná Jemini / który był z Báhurym / y wyƺedł z mężámi Judzkimi przećiwko królowi Dawidowi.
A było tyśiąc mężów z nim z Benjámitów ; Sybá tákże / ſługá domu Saulowego / y piętnaśćie ſynów jego / y dwádźieśćiá ſług jego z nim / y ƺcżęśliwie śię przepráwili zá Jordán do królá.
Przepráwili też prom / áby przewieźiono cżeládź królewſką / á iżby ucżyniono / coby mu śię nájlepiej podobáło ; á Semej / ſyn Giery / upádł przed królem / gdy śię przepráwił przez Jordán /
Y rzekł do królá : Nie przycżytáj mi / pánie mój / niepráwośći / áni wſpomináj / co lekkomyślnie ucżynił ſługá twój onegoż dniá / gdy wyƺedł król / pán mój / z Jeruzálemu / áby to miał przypuƺcżáć król do ſercá ſwego ;
Abowiem zná ſługá twój / żem zgrzeƺył ; á otom dźiś przyƺedł pierwej niż kto ze wƺyſtkiego domu Józefowego / ábym zájechał drogę królowi / pánu memu.
Tedy odpowiedźiał Abiſáj / ſyn Sárwii / y rzekł : Izáż dla tego nie ma być zábity Semej / że złorzecżył pomázáńcowi Páńſkimu?
Ale mu rzekł Dawid : Cóż wam do tego / ſynowie Sárwii / żeśćie mi dźiś przećiwnymi? Izali dźiś ma być zábity kto w Izráelu? Bo ázaż nie wiem / żem ja dźiś zoſtał królem nád Izráelem?
Y rzekł król do Semejá : Nie umrzeƺ ; y przyśiągł mu król.
Mefiboſet tákże / wnuk Saulá / wyjechał przećiw królowi ; który áni chędożył nóg ſwojich / áni cżeſał brody ſwojey / áni prał ƺát ſwojich / ode dniá / którego był wyƺedł król áż do dniá / którego śię wróćił w pokoju.
Y ſtáło śię / gdy zábieżał w Jeruzálemie królowi / rzekł mu król : Przecżżeś nie ƺedł ze mną / Mefiboſećie?
A on mu odpowiedźiał : Królu / pánie mój / zdrádźił mię ſługá mój ; bo rzekł był ſługá twój / ośiodłám ſobie oſłá / żebym wśiádƺy nań / jechał z królem / gdyż jeſt chromy ſługá twój.
Y oſkárżył ſługę twego przed królem / pánem moim ; ále król / pán mój / jeſt jáko Anjoł Boży ; przetoż ucżyń / co dobrego jeſt w ocżách twojich.
Abowiem wƺyſcy z domu ojcá mego byliśmy godni śmierći przed królem / pánem moim / á przećięś ty poſádźił ſługę twego miedzy tymi / którzy jádáją u ſtołu twego : Y cóż jeƺcże zá ſpráwiedliwość mojá / ábym śię miał więcey uſkárżáć ná królá?
Rzekł mu tedy król : Cóż maƺ więcey mówić w ſpráwie twojey? Jużem rzekł / ty y Sybá / podźielćie śię májętnośćią.
A Mefiboſet rzekł do królá : Y wƺyſtko niech weźmie / gdy śię tylko wróćił król / pán mój / w pokoju do domu ſwego.
Bárſyláj też Gáláádcżyk wyƺedƺy z Rogielim / przepráwił śię z królem przez Jordán / áby go prowádźił zá Jordán.
A Bárſyláj był bárzo ſtáry / májąc ośmdźieśiąt lat / który podejmował królá / póki mieƺkał w Máhánáim ; bo był cżłowiekiem bogátym bárzo.
Y rzekł król do Bárſylájego : Pójdź ze mną / á będę ćię chował przy ſobie w Jeruzálemie.
Ale Bárſyláj odpowiedźiał królowi : Wieleż jeſt dni lat żywotá mego / żebym miał iść z królem do Jeruzálemu?
Ośmdźieśiąt lat mi dźiśiáj ; izali mogę rozeznáć miedzy dobrem á złem? izali pocżuje ſmák ſługá twój w tym / cobym jádł / álbo cobym pił? izali ſłucháć mogę więcey głoſu śpiewáków y śpiewácżek? á przecżżeby miał być ſługá twój jeƺcże ćiężárem królowi / pánu memu?
Jeƺcże trochę pójdźie ſługá twój zá Jordán z królem : bo cżemużby mi miał dáwáć król tákową nágrodę?
Niech śię wróći proƺę ſługá twój / ábym umárł w mieśćie mojim / przy grobie ojcá mego y mátki mojey ; ále oto ſługá twój Chymhám pójdźie z królem / pánem moim : ucżyńże mu / co dobrego jeſt w ocżách twojich.
Y rzekł król : Niechże ze mną idźie Chymhám / á ja mu ucżynię / co dobrego będźie w ocżách twojich ; nádto / cokolwiek żądáć będźieƺ ode mnie / toć ucżynię.
A gdy śię przepráwił wƺyſtek lud przez Jordán / król śię też przepráwił. Tedy pocáłował król Bárſylájego / y błogoſłáwił mu ; który śię wróćił do miejſcá ſwego.
Potem przyƺedł król do Gálgál / przyƺedł też z nim Chymhám. Wƺyſtek też lud Judzki prowádźił królá / tákże y połowá ludu Izráelſkiego.
A oto / wƺyſcy mężowie Izráelſcy / zƺedƺy śię do królá / mówili do niego : Cżemuż ćię wykrádli bráćia náśi / mężowie Judzcy / y przeprowádźili królá y dom jego przez Jordán / y wƺyſtkie męże Dawidowe z nim?
Y odpowiedźieli wƺyſcy mężowie Judzcy mężom Izráelſkim : Przeto / iż nam powinny jeſt król. A przecżże śię gniewáć máćie o to? izali nam zá to jeść król dáwá / álbo nam jákie dáry rozdał?
Tedy odpowiedźieli mężowie Izráelſcy mężom Judzkim / y rzekli : Dźieśięć kroć więcey mamy do królá ; przetoż y Dawid więcey do nas náleży / niż do was. Przecżżeśćie nas lekce powáżyli? Azáżeśmy my o to pierwej nie mówili / ábyſmy przywróćili królá ſwego? Ale ſrożƺá byłá mowá mężów Judzkich / niż mowá mężów Izráelſkich.
