Y oznájmiono Joábowi : Oto król płácże y żáłuje Abſálomá.
Przetoż śię ono zwyćięſtwo dniá onego obróćiło w płácż wƺyſtkiemu ludowi ; ábowiem uſłyƺáł lud dniá onego / że mówiono : Żáłośny jeſt król dlá ſyná ſwego.
Zácżem wkrádł śię lud onego dniá wchodząc do miáſtá / jáko śię więc wkrádá lud / który śię wſtydźi / ućiekájąc z bitwy.
A król nákrywƺy oblicże ſwoje / wołáł głoſem wielkim : Synu mój Abſálomie / Abſálomie / ſynu mój / ſynu mój!
Tedy wƺedł Joáb do królá w dom / y rzekł : Sháńbiłeś dźiś oblicże wƺyſtkich ſług twoich / którzy wybáwili duƺę twoję dźiśiáj / y duƺę ſynów twoich / y córek twoich / y duƺę żon twoich / y duƺę záłożnic twoich.
Miłując te / którzy ćię máją w nienáwiśći / á nienáwidząc tych / którzy ćię miłują. Abowiem pokázáłeś dźiś / że ſobie nie powáżáƺ hetmánów / y ſług twoich ; bom doznáł tego dźiś / że gdyby Abſálom był żyw / choćbyśmy my wƺyſcy dźiś byli pobići / tedyćby śię to bárdzo podobáło.
Przetoż teráz wſtáń / wynijdź / á mów łágodnie do ſług twoich. Boć przez Páná przyśięgám / jeźli ty nie wynijdźieƺ / że nie zoſtánie żáden z tobą tej nocy / á będźieć to gorzej / niżli wƺyſtko złe / którekolwiek ná ćię przychodźiło od młodośći twojej áż dotąd.
Wſtáł tedy król / y śiádł w brámie ; y opowiedźiáno to wƺyſtkiemu ludowi / mówiąc : Oto król śiedźi w brámie. Y przyƺedł wƺyſtek lud przed oblicże królewſkie ; ále Izráelcżycy ućiekli byli / káżdy do námiotu ſwego.
Y ſtáło śię / że śię wƺyſtek lud ſprzecżáł z ſobą we wƺyſtkich pokoleniách Izráelſkich / mówiąc : Król wyrwáł náſ z rąk nieprzyjáćiół náƺych / tenże náſ też wyrwáł z rąk Filiſtyńſkich / á teráz ućiekł z źiemi przed Abſálomem.
Lecż Abſálom / któregośmy byli pomázáli nád ſobą / zginął w bitwie ; á teráz przecżże wy zániedbywáćie przyprowádźić záśię królá?
Przetoż król Dawid poſłáł do Sádoká y do Abijátárá / kápłánów / z temi ſłowy : Powiedzćie ſtárƺym Judzkim / mówiąc : Przecżże máćie być pośledniejƺymi w przyprowádzeniu záśię królá do domu jego? Abowiem ſłowá wƺyſtkiego Izráelá dochodźiły królá do domu jego.
Bráćiáśćie moi / kość mojá / á ćiáłośćie moje ; przecżże tedy máćie być pośledniejƺymi w przywróceniu królá?
Amáźie tákże powiedzćie : Izáliś ty nie jeſt kość mojá / y ćiáło moje? To niech mi ucżyni Bóg / y to niech przycżyni / jeźli hetmánem wojſká nie będźieƺ przedemną po wƺyſtkie dni / miáſto Joábá.
A ták nákłonił ſerce wƺyſtkich mężów Judzkich jáko mężá jednego / że poſłáli do królá mówiąc : Náwróć śię ty / y wƺyſcy ſłudzy twoi.
Wróćił śię tedy król / y przyƺedł áż do Jordánu ; á lud Judzki wyƺedł był do Gálgál / áby záƺedł w drodze królowi / á przeprowádźił królá przez Jordán.
Poſpieƺył śię tákże Semej / ſyn Giery / ſyná Jemini / który był z Báhurym / y wyƺedł z mężámi Judzkimi przećiwko królowi Dawidowi.
A było tyśiąc mężów z nim z Benjámitów ; Sybá tákże / ſługá domu Saulowego / y piętnáśćie ſynów jego / y dwádźieśćiá ſług jego z nim / y ƺcżęśliwie śię przepráwili zá Jordán do królá.
Przepráwili też prom / áby przewieźiono cżeládź królewſką / á iżby ucżyniono / coby mu śię nájlepiej podobáło ; á Semej / ſyn Giery / upádł przed królem / gdy śię przepráwił przez Jordán /
Y rzekł do królá : Nie przycżytáj mi / pánie mój / niepráwośći / áni wſpomináj / co lekkomyślnie ucżynił ſługá twój onegoż dniá / gdy wyƺedł król / pán mój / z Jeruzálemu / áby to miáł przypuƺcżáć król do ſercá ſwego ;
Abowiem zná ſługá twój / żem zgrzeƺył ; á otom dźiś przyƺedł pierwej niż kto ze wƺyſtkiego domu Józefowego / ábym zájecháł drogę królowi / pánu memu.
Tedy odpowiedźiáł Abiſáj / ſyn Sárwii / y rzekł : Izáż dlá tego nie má być zábity Semej / że złorzecżył pomázáńcowi Páńſkimu?
Ale mu rzekł Dawid : Cóż wám do tego / ſynowie Sárwii / żeśćie mi dźiś przećiwnymi? Izáli dźiś má być zábity kto w Izráelu? Bo ázáż nie wiem / żem já dźiś zoſtáł królem nád Izráelem?
Y rzekł król do Semejá : Nie umrzeƺ ; y przyśiągł mu król.
Mefiboſet tákże / wnuk Saulá / wyjecháł przećiw królowi ; który áni chędożył nóg ſwoich / áni cżeſáł brody ſwojej / áni práł ƺát ſwoich / ode dniá / którego był wyƺedł król áż do dniá / którego śię wróćił w pokoju.
Y ſtáło śię / gdy zábieżáł w Jeruzálemie królowi / rzekł mu król : Przecżżeś nie ƺedł ze mną / Mefiboſećie?
A on mu odpowiedźiáł : Królu / pánie mój / zdrádźił mię ſługá mój ; bo rzekł był ſługá twój / ośiodłám ſobie oſłá / żebym wśiádłƺy náń / jecháł z królem / gdyż jeſt chromy ſługá twój.
Y oſkárżył ſługę twego przed królem / pánem moim ; ále król / pán mój / jeſt jáko Anioł Boży ; przetoż ucżyń / co dobrego jeſt w ocżách twoich.
Abowiem wƺyſcy z domu ojcá mego byliśmy godni śmierći przed królem / pánem moim / á przećięś ty poſádźił ſługę twego między tymi / którzy jádáją u ſtołu twego : Y cóż jeƺcże zá ſpráwiedliwość mojá / ábym śię miáł więcej uſkárżáć ná królá?
Rzekł mu tedy król : Cóż máƺ więcej mówić w ſpráwie twojej? Jużem rzekł / ty y Sybá / podźielćie śię májętnośćią.
A Mefiboſet rzekł do królá : Y wƺyſtko niech weźmie / gdy śię tylko wróćił król / pán mój / w pokoju do domu ſwego.
Bárſyláj też Gáláádcżyk wyƺedłƺy z Rogielim / przepráwił śię z królem przez Jordán / áby go prowádźił zá Jordán.
A Bárſyláj był bárdzo ſtáry / májąc ośmdźieśiąt lát / który podejmowáł królá / póki mieƺkáł w Máhánáim ; bo był cżłowiekiem bogátym bárdzo.
Y rzekł król do Bárſylájego : Pójdź ze mną / á będę ćię chowáł przy ſobie w Jeruzálemie.
Ale Bárſyláj odpowiedźiáł królowi : Wieleż jeſt dni lát żywotá mego / żebym miáł iść z królem do Jeruzálemu?
Ośmdźieśiąt lát mi dźiśiáj ; izáli mogę rozeznáć między dobrem á złem? izáli pocżuje ſmák ſługá twój w tem / cobym jádł / álbo cobym pił? izáli ſłucháć mogę więcej głoſu śpiewáków y śpiewácżek? á przecżżeby miáł być ſługá twój jeƺcże ćiężárem królowi / pánu memu?
Jeƺcże trochę pójdźie ſługá twój zá Jordán z królem : bo cżemużby mi miáł dáwáć król tákową nágrodę?
Niech śię wróći proƺę ſługá twój / ábym umárł w mieśćie mojem / przy grobie ojcá mego y mátki mojej ; ále oto ſługá twój Chymhám pójdźie z królem / pánem moim : ucżyńże mu / co dobrego jeſt w ocżách twoich.
Y rzekł król : Niechże ze mną idźie Chymhám / á já mu ucżynię / co dobrego będźie w ocżách twoich ; nádto / cokolwiek żądáć będźieƺ ode mnie / toć ucżynię.
A gdy śię przepráwił wƺyſtek lud przez Jordán / król śię też przepráwił. Tedy pocáłowáł król Bárſylájego / y błogoſłáwił mu ; który śię wróćił do miejſcá ſwego.
Potem przyƺedł król do Gálgál / przyƺedł też z nim Chymhám. Wƺyſtek też lud Judzki prowádźił królá / tákże y połowá ludu Izráelſkiego.
A oto / wƺyſcy mężowie Izráelſcy / zƺedłƺy śię do królá / mówili do niego : Cżemuż ćię wykrádli bráćiá náśi / mężowie Judzcy / y przeprowádźili królá y dom jego przez Jordán / y wƺyſtkie męże Dawidowe z nim?
Y odpowiedźieli wƺyſcy mężowie Judzcy mężom Izráelſkim : Przeto / iż nám powinny jeſt król. A przecżże śię gniewáć máćie o to? izáli nám zá to jeść król dáwá / álbo nám jákie dáry rozdáł?
Tedy odpowiedźieli mężowie Izráelſcy mężom Judzkim / y rzekli : Dźieśięć kroć więcej mámy do królá ; przetoż y Dawid więcej do náſ náleży / niż do wáſ. Przecżżeśćie náſ lekce powáżyli? Azáżeśmy my o to pierwej nie mówili / ábyśmy przywróćili królá ſwego? Ale ſrożƺá byłá mowá mężów Judzkich / niż mowá mężów Izráelſkich.
