Y byłá długá wojná między domem Saulowym y między domem Dawidowym. Wƺákże Dawid poſtępowáł / y zmácniáł śię ; ále dom Saulów ſchodził y niƺcżáł.
Y národziło śię Dawidowi w Hebronie ſynów. A był pierworodny jego Amnon z Achinoámy Jezreelitki ;
Wtóry po nim był Helijáb z Abigáili / żony przedtem Nábálowej z Kármelu / á trzeći Abſálom / ſyn z Moáchy / córki Tolmájá królá Gieſſur ;
A cżwárty Adonijáƺ / ſyn Hágity / á piąty Sefátyjáƺ / ſyn Abitáli ;
A ƺóſty Jetráám z Egli / żony Dawidowej. Cić śię urodzili Dawidowi w Hebronie.
Y ſtáło śię / gdy byłá wojná między domem Saulowym y między domem Dawidowym / á Abner śię mężnie záſtáwiáł o dom Saulowy /
( A Saul miáł záłożnicę / której imię było Reſfá / córká Aje / )że rzekł Izboſet do Abnerá : Cżemuś wƺedł do záłożnicy ojcá mojego?
Y rozgniewáł śię Abner bárdzo o one ſłowá Izboſetowe / y rzekł : Izálim já pśiá głowá / którym przećiw Judzie dziś ucżynił miłośierdzie nád domem Saulá / ojcá twego / y nád bráćią jego / y nád przyjáćiołmi jego / y nie wydáłem ćię w rękę Dawidowę / á ty dziś ƺukáƺ ná mnie niepráwośći tej niewiáſty?
To niech ucżyni Bóg Abnerowi / y to niech mu przycżyni / jeźliże / jáko przyśiągł Pán Dawidowi / nie pomogę do tego /
Aby przenieśione było króleſtwo od domu Saulowego / á wyſtáwioná ſtolicá Dawidowá nád Izráelem / y nád Judą od Dán áż do Beerſebá.
Y nie mógł nic więcej odpowiedzieć Abnerowi / przeto że śię go báł.
A ták wypráwił Abner poſły do Dawidá od śiebie / mówiąc : Cżyjáż jeſt ziemiá? y żeby mówili : Ucżyń przymierze twoje ze mną / á oto ręká mojá będzie z tobą / áby obrócon był do ćiebie wƺyſtek Izráel.
Któremu odpowiedziáł : Dobrze / ucżynię z tobą przymierze. A wƺákże o jedno ćię proƺę / miánowićie / ábyś nie przychodził przed oblicże moje / áż mi pierwej przywiedzieƺ Michol / córkę Saulowę / gdy będzieƺ chćiáł przyjść / ábyś widziáł twárz moję.
Y wypráwił Dawid poſły do Izboſetá / ſyn Saulowego / mówiąc : Wydáj mi żonę moję Michol / którąm ſobie poślubił ſtem nieobrzezek Filiſtyńſkich.
Przetoż poſłáł Izboſet / y wziął ją od mężá / od Fáltyjelá / ſyná Láiſowego.
Tedy ƺedł z nią mąż jej / á idąc zá nią / płákáł jej / áż do Báchurym ; y rzekł do niego Abner : Idź / á wróć śię ; y wróćił śię.
Ucżynił potem Abner rzecż do ſtárƺych Izráelſkich / mówiąc : Przeƺłych cżáſów ƺukáliśćie Dawidá / áby był królem nád wámi.
Przetoż teráz ucżyńćie ták ; bo Pán rzekł o Dawidzie / mówiąc : Przez rękę Dawidá / ſługi mego / wybáwię lud mój Izráelſki z ręki Filiſtyńſkiej / y z ręki wƺyſtkich nieprzyjáćiół jego.
To też mówił Abner y do Benjámińcżyków. Potem odƺedł Abner / áby mówił z Dawidem w Hebronie wƺyſtko / co dobrego było w ocżách Izráelá / y w ocżách wƺyſtkiego domu Benjáminowego.
Gdy tedy przyƺedł Abner do Dawidá do Hebronu / á z nim dwádzieśćiá mężów / ſpráwił Dawid ná Abnerá / y ná męże / którzy z nim byli / ucżtę.
Y rzekł Abner do Dawidá : Wſtánę / á pójdę / ábym zebráł do królá / páná mego / wƺyſtkiego Izráelá / którzy z tobą ucżynią przymierze ; á będzieƺ królowáł nád wƺyſtkimi / jáko żądá duƺá twojá. A ták odpráwił Dawid Abnerá / który odƺedł w pokoju.
A oto / ſłudzy Dawidowi y Joáb wrácáli śię z wojny / korzyśći wielkie z ſobą prowádząc / ále Abnerá już nie było u Dawidá w Hebronie : bo go był odpráwił / y odƺedł był w pokoju.
Joáb tedy y wƺyſtko wojſko / które z nim było / przyƺli tám ; y dáno znáć Joábowi / mówiąc : Był tu Abner / ſyn Nerá / u królá ; ále go odpráwił / y odƺedł w pokoju.
Przetoż wƺedłƺy Joáb do królá / rzekł : Cóżeś ucżynił? Oto / przyƺedł był Abner do ćiebie ; przecżżeś go puśćił / áby záś odƺedł.
Znáƺ Abnerá / ſyná Nerowego / gdyż przyƺedł / áby ćię zdrádził / y żeby wiedziáł wyjśćie twoje / y wejśćie twoje / áby śię wywiedziáł o wƺyſtkiem / co ty cżyniƺ.
Tedy odƺedłƺy Joáb od Dawidá / wypráwił poſły zá Abnerem / którzy go wróćili od ſtudni Syrá / o cżem Dawid nie wiedziáł.
A gdy śię wróćił Abner do Hebronu / odwiódł go Joáb w pośród brámy / áby z nim po ćichu ( oſobno ) mówił / y przebił go tám pod piąte żebro / że umárł dlá krwi Aſáelá / brátá ſwego.
Co gdy potem uſłyƺáł Dawid / rzekł : Nie jeſtem winien / áni króleſtwo moje / przed Pánem áż ná wieki krwi Abnerá / ſyná Nerowego.
Niecháj przyjdzie ná głowę Joábowę / y ná wƺyſtek dom ojcá jego / y niech nie uſtáje z domu Joábowego płynienie náśieniá ćierpiący / y trędowáty / y o kiju chodzący / y od miecżá upádájący / y nie májący chlebá.
A ták Joáb y Abiſáj / brát jego / zábili Abnerá / przeto iż on też był zábił Aſáelá / brátá ich / w bitwie u Gábáonu.
Rzekł potem Dawid do Joábá y do wƺyſtkiego ludu / który był z nim : Porozdzierájćie odzieniá wáƺe / á opáƺćie śię w wory / y płácżćie nád Abnerem. A król Dawid ƺedł zá márámi.
A gdy pogrzebli Abnerá w Hebronie / podnióſł król głoſ ſwój / y płákáł nád grobem Abnerowym ; płákáł też wƺyſtek lud.
A ták lámentując król nád Abnerem / rzekł : Izáli ták miáł umrzeć Abner / jáko umierá nikcżemnik?
Ręce twoje nie były związáne / á nogi twoje nie były pętámi obćiążone ; poległeś jáko ten / który pádá przed ſynámi niezbożnymi. Tedy tem więcej wƺyſtek lud płákáł nád nim.
Potem przyƺedł wƺyſtek lud prośić Dawidá / áby jádł chleb / gdy jeƺcże był jáſny dzień : ále przyśiągł Dawid / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / jeźli przed zájśćiem ſłońcá ſkoƺtuję chlebá / álbo cżego innego.
Co gdy wƺyſtek lud obácżył / podobáło śię im to ; á wƺyſtko cokolwiek cżynił król / podobáło śię w ocżách wƺyſtkiego ludu.
Y poznáł wƺyſtek Izráel dniá onego / że nie z nápráwy królewſkiej zábity był Abner / ſyn Nerá.
Y rzekł król do ſług ſwoich : Azáż nie wiećie / że hetmán / á bárdzo wielki / poległ dziś w Izráelu?
A jám dziś jáko nowy / y dopiero pomázány król ; ći záśię mężowie / ſynowie Sárwii / ſrożśi ſą niżli já ; niechże oddá Pán cżyniącemu złe według złośći jego.
