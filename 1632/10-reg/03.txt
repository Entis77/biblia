Y byłá długá wojná miedzy domem Saulowym y miedzy domem Dawidowym. Wƺákże Dawid poſtępował / y zmácniał śię ; ále dom Saulów ſchodźił y niƺcżał.
Y národźiło śię Dawidowi w Hebronie ſynów. A był pierworodny jego Amnon z Achinoámy Jezreelitki ;
Wtóry po nim był Helijáb z Abigáili / żony przed tym Nábálowej z Kármelu / á trzeći Abſálom / ſyn z Moáchy / córki Tolmájá królá Gieſſur ;
A cżwárty Adonijáƺ / ſyn Hágity / á piąty Sefátyjáƺ / ſyn Abitáli ;
A ƺóſty Jetráám z Egli / żony Dawidowej. Cić śię urodźili Dawidowi w Hebronie.
Y ſtáło śię / gdy byłá wojná miedzy domem Saulowym y miedzy domem Dawidowym / á Abner śię mężnie záſtáwiał o dom Saulowy /
( A Saul miał záłożnicę / którey imię było Reſfá / córká Aje / )że rzekł Izboſet do Abnerá : Cżemuś wƺedł do záłożnicy ojcá mojego?
Y rozgniewał śię Abner bárzo o one ſłowá Izboſetowe / y rzekł : Izálim ja pśiá głowá / którym przećiw Judźie dźiś ucżynił miłośierdźie nád domem Saulá / ojcá twego / y nád bráćią jego / y nád przyjáćiołmi jego / y nie wydáłem ćię w rękę Dawidowę / á ty dźiś ƺukáƺ ná mnie niepráwośći tey niewiáſty?
To niech ucżyni Bóg Abnerowi / y to niech mu przycżyni / jeſliże / jáko przyśiągł Pán Dawidowi / nie pomogę do tego /
Aby przenieśione było króleſtwo od domu Saulowego / á wyſtáwioná ſtolicá Dawidowá nád Izráelem / y nád Judą od Dán áż do Beerſebá.
Y nie mógł nic więcey odpowiedźieć Abnerowi / przeto że śię go bał.
A ták wypráwił Abner poſły do Dawidá od śiebie / mówiąc : Cżyjáż jeſt źiemiá? y żeby mówili : Ucżyń przymierze twoje ze mną / á oto ręká mojá będźie z tobą / áby obrócon był do ćiebie wƺyſtek Izráel.
Któremu odpowiedźiał : Dobrze / ucżynię z tobą przymierze. A wƺákże o jedno ćię proƺę / miánowićie / ábyś nie przychodźił przed oblicże moje / áż mi pierwej przywiedźieƺ Michol / córkę Saulowę / gdy będźieƺ chćiał przyść / ábyś widźiał twárz moję.
Y wypráwił Dawid poſły do Izboſetá / ſyn Saulowego / mówiąc : Wydáj mi żonę moję Michol / którąm ſobie poślubił ſtem nieobrzezek Filiſtyńſkich.
Przetoż poſłał Izboſet / y wźiął ją od mężá / od Fáltyjelá / ſyná Láiſowego.
Tedy ƺedł z nią mąż jey / á idąc zá nią / płákał jey / áż do Báchurym ; y rzekł do niego Abner : Idź / á wróć śię ; y wróćił śię.
Ucżynił potem Abner rzecż do ſtárƺych Izráelſkich / mówiąc : Przeƺłych cżáſów ƺukáliśćie Dawidá / áby był królem nád wámi.
Przetoż teraz ucżyńćie ták ; bo Pán rzekł o Dawidźie / mówiąc : Przez rękę Dawidá / ſługi mego / wybáwię lud mój Izráelſki z ręki Filiſtyńſkiej / y z ręki wƺyſtkich nieprzyjáćiół jego.
To też mówił Abner y do Benjámińcżyków. Potem odƺedł Abner / áby mówił z Dawidem w Hebronie wƺyſtko / co dobrego było w ocżách Izráelá / y w ocżách wƺyſtkiego domu Benjáminowego.
Gdy tedy przyƺedł Abner do Dawidá do Hebronu / á z nim dwádźieśćiá mężów / ſpráwił Dawid ná Abnerá / y ná męże / którzy z nim byli / ucżtę.
Y rzekł Abner do Dawidá : Wſtánę / á pójdę / ábym zebrał do królá / páná mego / wƺyſtkiego Izráelá / którzy z tobą ucżynią przymierze ; á będźieƺ królował nád wƺyſtkimi / jáko żądá duƺá twojá. A ták odpráwił Dawid Abnerá / który odƺedł w pokoju.
A oto / ſłudzy Dawidowi y Joáb wrácáli śię z wojny / korzyśći wielkie z ſobą prowádząc / ále Abnerá już nie było u Dawidá w Hebronie : bo go był odpráwił / y odƺedł był w pokoju.
Joáb tedy y wƺyſtko wojſko / które z nim było / przyƺli tám ; y dáno znáć Joábowi / mówiąc : Był tu Abner / ſyn Nerá / u królá ; ále go odpráwił / y odƺedł w pokoju.
Przetoż wƺedƺy Joáb do królá / rzekł : Cóżeś ucżynił? Oto / przyƺedł był Abner do ćiebie ; przecżżeś go puśćił / áby záś odƺedł.
Znáƺ Abnerá / ſyná Nerowego / gdyż przyƺedł / áby ćię zdrádźił / y żeby wiedźiał wyjśćie twoje / y wejśćie twoje / áby śię wywiedźiał o wƺyſtkim / co ty cżyniƺ.
Tedy odƺedƺy Joáb od Dawidá / wypráwił poſły zá Abnerem / którzy go wróćili od ſtudni Syrá / o cżym Dawid nie wiedźiał.
A gdy śię wróćił Abner do Hebronu / odwiódł go Joáb w pośród bramy / áby z nim po ćichu ( oſobno ) mówił / y przebił go tám pod piąte żebro / że umárł dla krwi Aſáelá / brátá ſwego.
Co gdy potem uſłyƺał Dawid / rzekł : Nie jeſtem winien / áni króleſtwo moje / przed Pánem áż ná wieki krwi Abnerá / ſyná Nerowego.
Niecháj przydźie ná głowę Joábowę / y ná wƺyſtek dom ojcá jego / y niech nie uſtáje z domu Joábowego płynienie náśieniá ćierpiący / y trędowáty / y o kiju chodzący / y od miecżá upádájący / y nie májący chlebá.
A ták Joáb y Abiſáj / brát jego / zábili Abnerá / przeto iż on też był zábił Aſáelá / brátá ich / w bitwie u Gábáonu.
Rzekł potem Dawid do Joábá y do wƺyſtkiego ludu / który był z nim : Porozdźierájćie odźieniá wáƺe / á opáƺćie śię w wory / y płácżćie nád Abnerem. A król Dawid ƺedł zá márámi.
A gdy pogrzebli Abnerá w Hebronie / podnióſł król głos ſwój / y płákał nád grobem Abnerowym ; płákał też wƺyſtek lud.
A ták lámentując król nád Abnerem / rzekł : Izali ták miał umrzeć Abner / jáko umierá nikcżemnik?
Ręce twoje nie były związáne / á nogi twoje nie były pętámi obćiążone ; poległeś jáko ten / który pádá przed ſynámi niezbożnymi. Tedy tym więcey wƺyſtek lud płákał nád nim.
Potem przyƺedł wƺyſtek lud prośić Dawidá / áby jádł chleb / gdy jeƺcże był jáſny dźień : ále przyśiągł Dawid / mówiąc : To mi niech ucżyni Bóg / y to niech przycżyni / jeſli przed zájśćiem ſłońcá ſkoƺtuję chlebá / álbo cżego innego.
Co gdy wƺyſtek lud obácżył / podobáło śię im to ; á wƺyſtko cokolwiek cżynił król / podobáło śię w ocżách wƺyſtkiego ludu.
Y poznał wƺyſtek Izráel dniá onego / że nie z nápráwy królewſkiej zábity był Abner / ſyn Nerá.
Y rzekł król do ſług ſwojich : Azaż nie wiećie / że hetmán / á bárzo wielki / poległ dźiś w Izráelu?
A jam dźiś jáko nowy / y dopiero pomázány król ; ći záśię mężowie / ſynowie Sárwii / ſrożśi ſą niżli ja ; niechże oddá Pán cżyniącemu złe według złośći jego.
