Roku śiedmnáſtego Fácejáƺá / ſyná Romelijáƺowego / królowáł Acház / ſyn Joátámá / królá Judzkiego.
Dwádzieśćiá lát było Acházowi / gdy królowáć pocżął / á ƺeſnáśćie lát królowáł w Jeruzálemie ; ále nie cżynił / co dobrego jeſt w ocżách Páná / Bogá ſwego / jáko Dawid / ojćiec jego ;
Lecż chodził drogámi królów Izráelſkich. Nádto y ſyná ſwego dáł przewieść przez ogień według obrzydliwośći pogánów / które był Pán wygnáł przed oblicżem ſynów Izráeliſkich.
Ofiárowáł też y kádził ná wyżynách / y ná págórkách / y pod káżdem drzewem gáłęziſtem.
Tedy wyćiągnął Ráſyn / król Syryjſki / y Fácejáƺ / ſyn Romelijáƺá / król Izráelſki / przećiwko Jeruzálemowi ná wojnę / y oblegli Acházá ; wƺákże go dobyć nie mogli.
Tegoż cżáſu Ráſyn / król Syryjſki / przywróćił záśię Elát do Syryi / á wykorzenił Żydy z Elát ; ále Syryjcżycy przyƺedłƺy do Elát / mieƺkáli tám áż do dniá tego.
Y poſłáł Acház poſły do Teglát Fáláſerá królá Aſſyryjſkiego / mówiąc : Sługá twój y ſyn twój jeſtem. Przyćiągnij á wybáw mię z ręki królá Syryjſkiego / y z ręki królá Izráelſkiego / którzy powſtáli przećiwko mnie.
Tedy wziąwƺy Acház ſrebro y złoto / które śię ználázło w domu Páńſkim y w ſkárbách domu królewſkiego / poſłáł dár królowi Aſſyryjſkiemu.
Ná co mu przyzwolił król Aſſyryjſki ; á przyćiągnąwƺy król Aſſyryjſki pod Dámáƺek wziął go / y przenióſł obywátele jego do Chyr / á Ráſyná zábił.
Zátem jecháł król Acház przećiw Teglát Fáláſerowi / królowi Aſſyryjſkiemu / do Dámáƺku ; á ujrzáwƺy król Acház ołtárz w Dámáƺku / poſłáł do Uryjáƺá kápłáná wizerunek ołtárzá onego y kƺtáłt jego / według wƺyſtkiego / jáko był urobiony.
Y zbudowáł Uryjáƺ kápłán ołtárz według onego wƺyſtkiego / jáko był poſłáł król Acház z Dámáƺku ; ták ucżynił Uryjáƺ kápłán pierwej / niżeli śię wróćił król Acház z Dámáƺku.
A gdy śię wróćił król z Dámáƺku / ujrzáwƺy ołtárz / przyſtąpił do niego / y ſpráwowáł ofiáry ná nim.
Y zápálił cáłopálenie ſwoje / y ofiárę śniedną ſwoję / y ofiárowáł ofiárę mokrą ſwoję / y kropił krwią ofiár ſpokojnych ſwoich ná ołtárzu.
Ale ołtárz miedziány / który był przed Pánem / przenióſł z przedniej ſtrony domu / áby nie ſtáł między ołtárzem jego / y między domem Páńſkim ; á poſtáwił go po bok ołtárzá ku północy.
Y rozkázáł król Acház Uryjáƺowi kápłánowi / mówiąc : Ná tym więkƺym ołtárzu będzieƺ zápáláł cáłopálenie poránne y ofiárę śniedną wiecżorną / y cáłopálenie królewſkie / y ofiárę śniedną jego / y cáłopálenie wƺyſtkiego ludu ziemi / y ofiárę ich śniedną / y ofiáry mokre ich / y wƺelką krwią cáłopáleniá / y wƺelką krwią innych ofiár będzieƺ kropił ná nim ; ále ołtárz miedziány będzie mi ná rádzenie śię Bogá.
Y ucżynił Uryjáƺ kápłán według wƺyſtkiego / jáko był rozkázáł król Acház.
Nádto poodćináł król Acház liſtwy podſtáwków / y pozbieráł z nich wánny ; do tego morze zdjął z wołów miedziánych / które były pod nim / á położył je ná tle kámiennem.
Záſłonę tákże ſábátnią / którą było ſpráwiono w domu / y drzwi zewnętrzne / któremi król wchádzáł / odjął od domu Páńſkiego dlá bojáźni królá Aſſyryjſkiego.
A inne ſpráwy Acházowe / które cżynił / zápiſáne ſą w kronikách o królách Judzkich.
Y záſnął Acház z ojcámi ſwymi / y pogrzebiony jeſt z ojcámi ſwymi w mieśćie Dawidowem. A królowáł Ezechyjáƺ / ſyn jego / miáſto niego.
