Roku dwudzieſtego y trzećiego Joázá / ſyná Ochozyjáƺá / królá Judzkiego / królowáł Joácház / ſyn Jehu / nád Izráelem w Sámáryi śiedmnáśćie lát.
A cżynił złe przed ocżymá Páńſkimi ; bo náśládowáł grzechów Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá / y nie odchylił śię od nich.
Y zápálił śię gniew Páńſki przećiw Izráelowi / y podáł je w rękę Házáelá / królá Syryjſkiego / y w rękę Benádádá / ſyná Házáelowego / po wƺyſtkie dni.
Ale gdy śię modlił Joácház przed oblicżem Páńſkim / wyſłucháł go Pán ; bo widziáł śćiśnienie Izráelá / że go był ućiſnął król Syryjſki.
Przetoż dáł Pán Izráelowi wybáwićielá / á wyƺli z ręki Syryjcżyków / y mieƺkáli ſynowie Izráelſcy w przybytkách ſwych / jáko y przedtem.
Wƺákże nie odſtąpili od grzechów domu Jeroboámowego / który przywiódł do grzechu Izráelá / ále w nich chodzili ; do tego jeƺcże y gáj zoſtáł w Sámáryi.
Acżkolwiek nie zoſtáwił Joácházowi z ludu / jedno pięćdzieśiąt jezdnych / y dzieśięć wozów / y dzieśięć tyśięcy pieƺych / gdyż je był wytráćił król Syryjſki / y w proch je pomłóćił.
Ale inne ſpráwy Joácházowe / y wƺyſtko / co cżynił / y moc jego / ázáż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Joácház z ojcámi ſwymi / y pochowáno go w Sámáryi / á królowáł Joáz / ſyn jego / miáſto niego.
Roku trzydzieſtego y śiódmego Joázá / królá Judzkiego / królowáł Joáz / ſyn Joácházowy / nád Izráelem w Sámáryi ƺeſnáśćie lát ;
Y cżynił złe przed ocżymá Páńſkimi / nie uchylájąc śię od żádnych grzechów Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá ; ále w nich chodził.
A inne ſpráwy Joázowe / y wƺyſtko co cżynił / y moc jego / jáko wálcżył przećiw Amázyjáƺowi / królowi Judzkiemu / ázáż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Joáz z ojcámi ſwymi / á Jeroboám uśiádł ná ſtolicy jego. Y pogrzebion jeſt Joáz w Sámáryi z królámi Izráelſkimi.
A Elizeuƺ wpádł w ćiężką chorobę / w której też umárł. Y przyƺedł do niego Joáz / król Izráelſki / y płákáł nád nim / mówiąc : Ojcże mój / ojcże mój! wozie Izráelſki / y jázdo jego.
Tedy mu rzekł Elizeuƺ : Weźmij łuk y ſtrzáły ; á wziąwƺy przynióſł do niego łuk y ſtrzáły.
Y rzekł do królá Izráelſkiego : Weźmij w rękę twoję łuk ; y wziął go w rękę ſwoję ; włożył też Elizeuƺ ręce ſwe ná ręce królewſkie.
Y rzekł : Otwórz to okno ná wſchód ſłońcá. A gdy otworzył / rzekł Elizeuƺ : Strzelże! y ſtrzelił. Y rzekł : Strzáłá zbáwieniá Páńſkiego / á ſtrzáłá wybáwieniá przećiw Syryjcżykom ; ábowiem poráziƺ Syryjcżyki w Afeku áż do ƺcżętu.
Rzekł powtóre : Weźmij ſtrzáły! y wziął. Tedy rzekł do królá Izráelſkiego : Uderz w ziemię! y uderzył trzy kroć á potem przeſtáł.
Przetoż rozgniewáł śię náń mąż Boży / y rzekł : Miáłeś uderzyć pięć álbo ƺeść kroć / bobyś był poráził Syryjcżyki áż do ƺcżętu : á teráz tylko po trzy kroć poráziƺ Syryjcżyki.
Potem umárł Elizeuƺ / y pogrzebiono go. A kupy ſwáwolne Moábſkie wtárgnęły do ziemi roku drugiego.
Y ſtáło śię / gdy chowáno jednego cżłowieká / tedy ujrzáwƺy ſwáwolną kupę / wrzućili onego cżłowieká w grób Elizeuƺowy / który gdy był wrzucony / á dotknął śię kośći Elizeuƺowych / ożył y wſtáł ná nogi ſwoje.
A Házáel / król Syryjſki / trápił lud Izráelſki po wƺyſtkie dni Joácházowe.
Ale ulitowáwƺy śię ich Pán / zmiłowáł śię nád nimi / y náwróćił śię ku nim dlá przymierzá ſwego z Abráhámem / z Izáákiem / y z Jákóbem ; y nie chćiáł ich wytráćić / áni ich odrzućił od oblicżá ſwego / áż do tego cżáſu.
Y umárł Házáel / król Syryjſki / á królowáł Benádád / ſyn jego / miáſto niego.
Przetoż znowu Joáz / ſyn Joácházowy / odebráł miáſtá z ręki Benádádá / ſyná Házáelowego / które był wziął z rąk Joácházá / ojcá jego / przez wojnę ; bo po trzy kroć poráził go Joáz / y przywróćił miáſtá Izráelowi.
