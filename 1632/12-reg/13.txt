Roku dwudźieſtego y trzećiego Joázá / ſyná Ochozyjáƺá / królá Judzkiego / królował Joácház / ſyn Jehu / nád Izráelem w Sámáryi śiedmnáśćie lat.
A cżynił złe przed ocżymá Páńſkimi ; bo náśládował grzechów Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá / y nie odchylił śię od nich.
Y zápálił śię gniew Páńſki przećiw Izráelowi / y podał je w rękę Házáelá / królá Syryjſkiego / y w rękę Benádádá / ſyná Házáelowego / po wƺyſtkie dni.
Ale gdy śię modlił Joácház przed oblicżem Páńſkim / wyſłuchał go Pán ; bo widźiał śćiśnienie Izráelá / że go był ućiſnął król Syryjſki.
Przetoż dał Pán Izráelowi wybáwićielá / á wyƺli z ręki Syryjcżyków / y mieƺkáli ſynowie Izráelſcy w przybytkách ſwych / jáko y przedtem.
Wƺákże nie odſtąpili od grzechów domu Jeroboámowego / który przywiódł do grzechu Izráelá / ále w nich chodźili ; do tego jeƺcże y gáj zoſtał w Sámáryi.
Acżkolwiek nie zoſtáwił Joácházowi z ludu / jedno pięćdźieśiąt jezdnych / y dźieśięć wozów / y dźieśięć tyśięcy pieƺych / gdyż je był wytráćił król Syryjſki / y w proch je pomłóćił.
Ale inne ſpráwy Joácházowe / y wƺyſtko / co cżynił / y moc jego / ázaż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Joácház z ojcámi ſwymi / y pochowano go w Sámáryi / á królował Joáz / ſyn jego / miáſto niego.
Roku trzydźieſtego y śiódmego Joázá / królá Judzkiego / królował Joáz / ſyn Joácházowy / nád Izráelem w Sámáryi ƺeſnáśćie lat ;
Y cżynił złe przed ocżymá Páńſkimi / nie uchylájąc śię od żádnych grzechów Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá ; ále w nich chodźił.
A inne ſpráwy Joázowe / y wƺyſtko co cżynił / y moc jego / jáko wálcżył przećiw Amázyjáƺowi / królowi Judzkiemu / ázaż to nie jeſt nápiſáne w kronikách o królách Izráelſkich?
Y záſnął Joáz z ojcámi ſwymi / á Jeroboám uśiádł ná ſtolicy jego. Y pogrzebion jeſt Joáz w Sámáryi z królámi Izráelſkimi.
A Elizeuƺ wpádł w ćiężką chorobę / w którey też umárł. Y przyƺedł do niego Joáz / król Izráelſki / y płákał nád nim / mówiąc : Ojcże mój / ojcże mój! woźie Izráelſki / y jázdo jego.
Tedy mu rzekł Elizeuƺ : Weźmij łuk y ſtrzáły ; á wźiąwƺy przynióſł do niego łuk y ſtrzáły.
Y rzekł do królá Izráelſkiego : Weźmij w rękę twoję łuk ; y wźiął go w rękę ſwoję ; włożył też Elizeuƺ ręce ſwe ná ręce królewſkie.
Y rzekł : Otwórz to okno ná wſchód ſłońcá. A gdy otworzył / rzekł Elizeuƺ : Strzelże! y ſtrzelił. Y rzekł : Strzáłá zbáwieniá Páńſkiego / á ſtrzáłá wybáwieniá przećiw Syryjcżykom ; ábowiem poráźiƺ Syryjcżyki w Afeku áż do ƺcżętu.
Rzekł powtóre : Weźmij ſtrzáły! y wźiął. Tedy rzekł do królá Izráelſkiego : Uderz w źiemię! y uderzył trzy kroć á potem przeſtał.
Przetoż rozgniewał śię nań mąż Boży / y rzekł : Miáłeś uderzyć pięć álbo ƺeść kroć / bobyś był poráźił Syryjcżyki áż do ƺcżętu : á teraz tylko po trzy kroć poráźiƺ Syryjcżyki.
Potem umárł Elizeuƺ / y pogrzebiono go. A kupy ſwáwolne Moábſkie wtárgnęły do źiemi roku drugiego.
Y ſtáło śię / gdy chowano jednego cżłowieká / tedy ujrzáwƺy ſwáwolną kupę / wrzućili onego cżłowieká w grób Elizeuƺowy / który gdy był wrzucony / á dotknął śię kośći Elizeuƺowych / ożył y wſtał ná nogi ſwoje.
A Házáel / król Syryjſki / trápił lud Izráelſki po wƺyſtkie dni Joácházowe.
Ale ulitowáwƺy śię ich Pán / zmiłował śię nád nimi / y náwróćił śię ku nim dla przymierza ſwego z Abráhámem / z Izáákiem / y z Jákóbem ; y nie chćiał ich wytráćić / áni ich odrzućił od oblicżá ſwego / áż do tego cżáſu.
Y umárł Házáel / król Syryjſki / á królował Benádád / ſyn jego / miáſto niego.
Przetoż znowu Joáz / ſyn Joácházowy / odebrał miáſtá z ręki Benádádá / ſyná Házáelowego / które był wźiął z rąk Joácházá / ojcá jego / przez wojnę ; bo po trzy kroć poráźił go Joáz / y przywróćił miáſtá Izráelowi.
