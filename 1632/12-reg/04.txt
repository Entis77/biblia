A niewiáſtá jedná z żon ſynów prorockich wołáłá do Elizeuƺá / mówiąc : Sługá twój / mąż mój / umárł. A ty wieƺ / iż ſługá twój báł śię Páná. A teraz przyƺedł pożycżálnik / áby ſobie wźiął dwóch ſynów moich zá niewolniki.
Do którey rzekł Elizeuƺ : Cóż ći mam ucżynić? Powiedz mi / co máƺ w domu? A oná odpowiedźiáłá : Nie má ſłużebnicá twojá nic więcey w domu / jedno báńkę oliwy.
Y rzekł : Idźże / nápożycżáj ſobie nácżyniá z inąd u wƺyſtkich ſąśiádek twoich / nácżyniá próżnego nie máło.
A wƺedłƺy zámknij drzwi zá ſobą y zá ſynámi twymi / á nálej we wƺyſtkie te nácżyniá / á które będźie pełne / rozkáż odſtáwić.
A ták odƺedłƺy od niego / zámknęłá drzwi zá ſobą y zá ſynámi ſwymi.( Oni przynośili do niej / á oná nálewáłá. )
Y ſtáło śię / gdy nápełniłá one nácżyniá / rzekłá do ſyná ſwego : Przynieś mi jeƺcże nácżynie. A on jey odpowiedźiáł : Niemáƺ więcey nácżyniá. Y záſtánowiłá śię oliwá.
Potem oná przyƺedłƺy / oznájmiłá to mężowi Bożemu / który do niej rzekł : Idźże / ſprzedáj tę oliwę / á oddáj pożycżálnikowi twemu / á ty y ſynowie twoi żywćie śię oſtátkiem.
Stáło śię potem cżáſu niektórego / iż ƺedł Elizeuƺ przez Sunem / gdźie byłá niewiáſtá zácná / która go zátrzymáłá / áby jádł chleb ; á ták ile kroć támtędy chodźił / wſtępowáł do niej / áby jádł chleb.
Bo rzekłá byłá do mężá ſwego : Oto teraz wiem / że ten mąż Boży święty jeſt / który tędy przechodźi cżęſto.
Proƺę / ucżyńmy gmáƺek máły / á poſtáwmy mu tám łóżko y ſtół / y krzeſło y lichtárz / że kiedy przyjdźie do nas / ſkłoni śię tám.
A ták dniá jednego / gdy tám przyƺedł / ſkłonił śię do onego gmáƺku / y odpocżął tám.
Y rzekł do Giezego / ſługi ſwego : Záwołáj tey Sunámitki. Y záwołáł jey / á ſtánęłá przed nim.
Tedy mu rzekł : Powiedz jey : Oto piecżołujeƺ á ſtáráƺ śię o wƺyſtkie náƺe potrzeby ; cóż chceƺ / ábym ći ucżynił? Máƺże jáką potrzebę u królá / álbo u hetmáná wojſká? A oná rzekłá : W pośrodku ludu mego mieƺkám.
A on rzekł : Cóż wżdy mam ucżynić dla niej? Y odpowiedźiáł Giezy : Oto ſyná nie má / á mąż jey ſtáry.
Przetoż rzekł : Záwołájże jey. Y záwołáł jey / á oná ſtáłá u drzwi.
Y rzekł : O tym cżáśie po roku będźieƺ piáſtowáłá ſyná. A oná rzekłá : Nie omylájże / pánie mój / mężu Boży / nie omyláj ſłużebnicy twojey.
A ták pocżęłá niewiáſtá / y porodźiłá ſyná o onymże cżáśie po roku / jáko jey był powiedźiáł Elizeuƺ.
Y podroſło dźiećię. Y ſtáło śię dniá niektórego / że wyƺedłƺy do ojcá ſwego / do żeńców /
Rzekło do ojcá ſwego : Głowá mojá! Głowá mojá! A on rzekł ſłudze : Zánieś go do mátki jego.
Który wźiąwƺy go / zánióſł go do mátki jego ; y śiedźiáł ná łonie jey áż do południá y umárł.
Tedy oná ƺedłƺy położyłá go ná łóżku mężá Bożego / á zámknąwƺy drzwi / wyƺłá.
Potem przyzwáłá mężá ſwego / y rzekłá : Proƺę ćię / poślij ze mną jednego z ſług / y jednę oślicę / że pobieżę áż do mężá Bożego / y wrócę śię záś.
Który rzekł : Po cóż chceƺ jecháć do niego? Dźiś niemáƺ nowiu mieśiącá / áni ſábátu. Ale oná rzekłá : Dáj pokój.
A ták ośiodłáwƺy oślicę / rzekłá do ſługi ſwego : Pogániáj / á jedź / y nie mieƺkáj dla mnie w drodze / chybá żebym ći rozkázáłá.
Jecháłá tedy / y przyjecháłá do mężá Bożego ná górę Kármel. A gdy ją ujrzáł mąż Boży z dáleká / rzekł do Giezego ſługi ſwego : Oto oná Sunámitká.
Przetoż wynijdź przećiwko niej / á rzecż jey : A zdrowáś dobrze? zdrów y mąż twój? zdrów y ſyn?
A oná rzekłá : Zdrowi dobrze. A gdy przyƺłá do mężá Bożego ná górę / uchwyćiłá śię nóg jego ; y przyſtąpił Giezy / áby ją odepchnął. Ale mąż Boży rzekł : Zániecháj jey / boć w gorzkośći jeſt duƺá jey / á Pán zátáił przedemną y nie oznájmił mi.
A oná rzekłá : Azáżem páná mego prośiłá o ſyná? Izálim nie mówiłá : Nie omyláj mię?
Tedy on rzekł do Giezego : Przepáƺ biodrá twe / á weźmij láſkę moję w rękę twą / á idź ; jeſli kogo ſpotkáƺ / nie pozdráwiáj go ; á jeſliby ćię kto pozdrowił / nie odpowiádáj mu ; y połóż láſkę moję ná oblicże dźiećięćiá.
A mátká dźiećięćiá onego rzekłá : Jáko żywy Pán / y jáko żywá duƺá twojá / że śię ćiebie nie puƺcżę. A ták wſtáwƺy ƺedł zá nią.
A Giezy uprzedźił je / y położył láſkę ná oblicże dźiećięćiá ; lecż nie było głoſu / áni cżućiá. Przetoż śię wróćił przećiwko niemu y oznájmił mu / mówiąc : Nie ocućiło śię dźiećię.
Tedy wƺedł Elizeuƺ do domu / á oto dźiećię umárłe leżáło ná łóżku jego.
A gdy tám wƺedł / zámknął drzwi przed onymi obydwomá / y modlił śię Pánu.
Potem wſtąpiwƺy ná łoże / położył śię ná dźiećiątko / przyłożywƺy uſtá ſwe do uſt jego / á ocży ſwe do ocżów jego / y ręce ſwe do rąk jego / y rozpoſtárł śię ná nim / ták iż śię zágrzáło ćiáło dźiećięce.
A odwróćiwƺy śię od niego / przechádzáł śię po domu tám y ſám ; potem wſtąpił / á położył śię ná nim. Tedy kicháło dźiećię áż do śiódmego rázu / y otworzyło dźiećię ocży ſwoje.
Tedy záwołáł Giezego / y rzekł : Záwołáj tey Sunámitki. Y záwołáł jey / y przyƺłá do niego / y rzekł : Weźmij ſyná twego /
Która wƺedłƺy / upádłá u nóg jego / y kłániáłá śię áż do źiemi / á wźiąwƺy ſyná ſwego / wyƺłá.
Potem wróćił śię Elizeuƺ do Gálgál / á głód był w oney źiemi / y ſynowie proroccy mieƺkáli przy nim. Tedy rzekł do ſługi ſwego : Przyſtáw gárniec wielki / á uwárz káƺę ſynom prorockim.
Przetoż wyƺedł jeden ná pole / áby zbieráł źiołá / y ználázł máćicę polną / á názbieráł z niej owoców polnych pełen płáƺcż ſwój / á przyƺedłƺy nákrájáł ich w gárniec káƺy ; bo tego nie ználi.
Y wyláli mężom onym / áby jedli. A gdy jedli onę káƺę záwołáli / y rzekli : Śmierć w gárncu / mężu Boży! Y nie mogli jeść.
Y rzekł : Przynieśćie ſámi mąki ; á wſypáwƺy ją w gárniec rzekł : Nálej ludowi. Y jedli / y nie było nic więcey złego w gárncu.
Wtem mąż przyƺedł z Báálſáliſá / á przynióſł mężowi Bożemu chleby / z pierwoćin zbóż / dwádźieśćiá chlebów jęcżmiennych / y kłoſów pełnych świeżych nie wykruƺonych / y rzekł : Dáj ludowi / áby jedli.
Ale odpowiedźiáł ſługá jego : Cóż to mam dáć przed ſto mężów? Y rzekł : Dáj ludowi / áby jedli ; ábowiem ták mówi Pán : Będą jedli / y zbędźie.
A ták położył przed nie ; y jedli / á zbyło według ſłowá Páńſkiego.
