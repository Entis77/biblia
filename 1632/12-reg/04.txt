A niewiáſtá jedná z żon Synów Prorockich wołáłá do Elizeuƺá mówiąc : Sługá twój / mąż mój umárł : á ty wieƺ / iż ſługá twój bał śię PAná : A teraz przyƺedł pożycżálnik áby ſobie wźiął dwu Synów mojich zá niewolniki.
Do którey rzekł Elizeuƺ : Cóżći mam ucżynić? Powiedz mi co maƺ w domu? A oná odpowiedźiáłá : Nie ma ſłużebnicá twojá nic więcey w domu jedno báńkę oliwy.
Y rzekł : Idźże nápożycżaj ſobie nacżynia z inąd u wƺyſtkich ſąśiad twojich / nacżynia próznego niemáło.
A wƺedƺy zámkni drzwi zá ſobą / y zá Synámi twymi / á náliey we wƺyſtkie te nacżynia ; á które będźie pełne / rozkaż odſtáwić.
A ták odƺedƺy od niego / zámknęłá drzwi zá ſobą y zá Synámi ſwymi. ( Oni przynośili do niey / á oná nálewáłá. )
Y ſtáło śię gdy nápełniłá one nacżynia / rzekłá do Syná ſwego : Przynieś mi jeƺcże nacżynie : A on jey odpowiedźiał : Niemáƺ więcey nacżynia. Y záſtánowiłá śię Oliwá.
Potym oná przyƺedƺy oznájmiłá <i>to</i> mężowi Bożemu / który do niey rzekł : Idźże przedaj tę Oliwę / á oddaj Pożycżálnikowi twemu : á ty y Synowie twoji / żywćie śię oſtátkiem.
Stáło śię potym cżáſu niektórego / iż ƺedł Elizeuƺ przez Sunem / gdźie byłá niewiáſtá zacna / która go zátrzymáłá áby jadł chleb : á ták ile kroć tám tędy chodźił / wſtępował do niey / áby jadł chleb :
Bo rzekłá byłá do mężá ſwego : Oto teraz wiem / że ten mąż Boży święty jeſt / który tędy przechodźi cżęſto.
Proƺę ucżyńmy gmáƺek máły / á poſtawmy mu tám łóżko y ſtół / y krzeſło / y lichtarz ; że kiedy przydźie do nas / ſkłoni śię tám.
A ták dniá jednego / gdy tám przyƺedł / ſkłonił śię do onego gmáƺku / y odpocżynął tám.
Y rzekł do Gezego ſługi ſwego ; Záwołaj tey Sunámitki. Y záwołał jey / á ſtánęłá przed nim.
Tedy mu rzekł : Powiedz jey : Oto piecżołujeƺ á ſtáraƺ śię o wƺyſtkie náƺe potrzeby / cóż chceƺ ábymći ucżynił? Maƺże <i>jaką</i> potrzebę u Królá / álbo u Hetmáná wojſká? A oná rzekłá : Wpośrzodku ludu mego mieƺkam.
A on rzekł ; Cóż wżdy mam ucżynić dla niey? Y odpowiedźiał Gezy ; Oto Syná nie ma / á mąż jey ſtáry.
Przetoż rzekł : Záwołajże jey : y záwołał jey / á oná ſtałá u drzwi.
Y rzekł : O tym cżáśie po roku / będźieƺ piáſtowáłá Syná. A oná rzekłá : Nie omylajże Pánie mój / mężu Boży / nie omylaj ſłużebnice twojey.
A ták pocżęłá niewiáſtá / y porodźiłá Syná o onymże cżáśie po roku / jáko jey był powiedźiał Elizeuƺ.
Y podroſło dźiećię. Y ſtáło śię dniá niektórego / że wyƺedƺy do Ojcá ſwego / do żeńców :
Rzekło do Ojcá ſwego : Głowá mojá! Głowá mojá! A <i>on</i> rzekł ſłudze : Zánieś go do mátki jego.
Który wźiąwƺy go / zánióſł go do mátki jego : y śiedźiał ná łonie jey / áż do południá / y umárł.
Tedy oná ƺedƺy / położyłá go ná łóżku mężá Bożego : á zámknąwƺy drzwi / wyƺłá.
Potym przyzwáłá mężá ſwego / y rzekłá : Proƺę ćię pośli ze mną jednego z ſług / y jednę Oślicę / że pobieżę áż do mężá Bożego : y wrócę śię záś.
Który rzekł : Po cóż chceƺ jácháć do niego? Dźiś niemáƺ <i>nowiu</i> mieśiącá / áni Sábbátu. Ale <i>oná</i> rzekłá : <i>Daj</i> pokój.
A ták ośiodławƺy oślicę rzekłá do ſługi ſwego : Poganiaj / á jedź y nie mieƺkaj dla mnie w drodze / chybá żebymći rozkazáłá.
Jácháłá tedy / y przyjácháłá do mężá Bożego ná górę Kármel. A gdy ją ujrzał mąż Boży z dáleká / rzekł do Gezego ſługi ſwego : Oto oná Sunámitká.
Przetoż wynidź przećiwko niey / á rzecż jey : A zdrowáś dobrze / zdrów y mąż twój / zdrów y Syn?
A oná rzekłá ; Zdrowi dobrze. A gdy przyƺłá do mężá Bożego ná górę / uchwyćiłá śię nóg jego : y przyſtąpił Gezy áby ją odepchnął. Ale mąż Boży rzekł : Zániechaj jey / boć w gorzkośći jeſt duƺá jey / á Pan zátájił przede mną / y nie oznájmił mi.
A oná rzekłá : Azażem Páná mego prośiłá o Syná? Izalim nie mówiłá? Nie omylaj mię?
Tedy on rzekł do Gezego : Przepáƺ biodrá twe / á weźmi laſkę moję w rękę twą / á idź : jeſli kogo potkaƺ nie pozdrawiaj go. A jeſliby ćię kto pozdrowił / nie odpowiádaj mu : y połóż laſkę moję ná oblicże dźiećięćiá.
Ale mátká dźiećięćiá <i>onego</i> rzekłá : Żywie PAN / y żywie duƺá twojá / że śię ćiebie nie puƺcżę. A ták wſtawƺy ƺedł zá nią.
A Gezy uprzedźił je / y położył laſkę ná oblicże dźiećięćiá : Lecż nie było głoſu / áni cżućia. Przetoż śię wróćił przećiwko niemu / y oznájmił mu mówiąc : Nie ocućiło dźiećię.
Tedy wƺedł Elizeuƺ do domu / á oto dźiećię umárłe leżáło ná łóżku jego.
A gdy tám wƺedł / zámknął drzwi przed onymi obiemá / y modlił śię PANU.
Potym wſtąpiwƺy <i>ná łoże</i> położył śię ná dźiećiątko / przyłożywƺy uſtá ſwe do uſt jego / á ocży ſwe do ocżu jego / y ręce ſwe do ręku jego / y rozpoſtárł śię ná nim / ták iż śię zágrzało ćiáło dźiećięce.
A odwróćiwƺy śię od niego / przechadzał śię po domu tám y ſám : potym wſtąpił / á położył śię ná nim. Tedy kicháło dźiećię áż do śiódmego rázu : y otworzyło dźiećię ocży ſwoje.
Tedy záwołał Gezego / y rzekł : Záwołaj tey Sunámitki : y záwołał jey ; y przyƺłá do niego : y rzekł ; Weźmi Syná twego.
Która wƺedƺy upádłá u nóg jego / y kłaniáłá śię áż do źiemie : á wźiąwƺy Syná ſwego / wyƺłá.
Potym wróćił śię Elizeuƺ do Gálgál / á głód był w oney źiemi / y Synowie Proroccy mieƺkáli przy nim. Tedy rzekł do ſługi ſwego : Przyſtaw gárniec wielki / á uwarz káƺę Synom prorockim.
Przetoż wyƺedł jeden ná pole áby zbierał źiołá / y ználazł máćicę polną / á názbierał z niey owoców polnych pełen płaƺcż ſwój : á przyƺedƺy nákrájał ich w gárniec káƺe ; bo tego nie ználi.
Y wylali mężom onym áby jedli : A gdy jedli onę káƺę / záwołáli / y rzekli : Śmierć w gárncu / mężu Boży! y niemogli jeść.
Y rzekł : Przynieśćie ſám mąki : á wſypawƺy ją w gárniec / rzekł : Náliey ludowi / y jedli : y nie było nic więcey złego w gárncu.
Wtym mąż przyƺedł z Báálſáliſá / á przynióſł mężowi Bożemu chleby / z pierwoćin <i>zbóż,</i> dwádźieśćiá chlebów jęcżmiennych / y kłoſów pełnych świeżych nie wykruƺonych / y rzekł : Daj ludowi áby jedli.
Ale odpowiedźiał ſługá jego : Cóż to mam dáć przed ſto mężów? Y rzekł : Daj ludowi áby jedli : ábowiem ták mówi PAN ; Będą jedli / y zbędźie.
A ták położył przed nie : y jedli / á zbyło według ſłowá Páńſkiego.
