A niewiáſtá jedná z żon ſynów prorockich wołáłá do Elizeuƺá / mówiąc : Sługá twój / mąż mój / umárł. A ty wieƺ / iż ſługá twój báł śię Páná. A teráz przyƺedł pożycżálnik / áby ſobie wziął dwóch ſynów moich zá niewolniki.
Do której rzekł Elizeuƺ : Cóż ći mám ucżynić? Powiedz mi / co máƺ w domu? A oná odpowiedziáłá : Nie má ſłużebnicá twojá nic więcej w domu / jedno báńkę oliwy.
Y rzekł : Idźże / nápożycżáj ſobie nácżyniá z inąd u wƺyſtkich ſąśiádek twoich / nácżyniá próżnego nie máło.
A wƺedłƺy zámknij drzwi zá ſobą y zá ſynámi twymi / á nálej we wƺyſtkie te nácżyniá / á które będzie pełne / rozkáż odſtáwić.
A ták odƺedłƺy od niego / zámknęłá drzwi zá ſobą y zá ſynámi ſwymi.( Oni przynośili do niej / á oná nálewáłá. )
Y ſtáło śię / gdy nápełniłá one nácżyniá / rzekłá do ſyná ſwego : Przynieś mi jeƺcże nácżynie. A on jej odpowiedziáł : Niemáƺ więcej nácżyniá. Y záſtánowiłá śię oliwá.
Potem oná przyƺedłƺy / oznájmiłá to mężowi Bożemu / który do niej rzekł : Idźże / ſprzedáj tę oliwę / á oddáj pożycżálnikowi twemu / á ty y ſynowie twoi żywćie śię oſtátkiem.
Stáło śię potem cżáſu niektórego / iż ƺedł Elizeuƺ przez Sunem / gdzie byłá niewiáſtá zácná / którá go zátrzymáłá / áby jádł chleb ; á ták ile kroć támtędy chodził / wſtępowáł do niej / áby jádł chleb.
Bo rzekłá byłá do mężá ſwego : Oto teráz wiem / że ten mąż Boży święty jeſt / który tędy przechodzi cżęſto.
Proƺę / ucżyńmy gmáƺek máły / á poſtáwmy mu tám łóżko y ſtół / y krzeſło y lichtárz / że kiedy przyjdzie do náſ / ſkłoni śię tám.
A ták dniá jednego / gdy tám przyƺedł / ſkłonił śię do onego gmáƺku / y odpocżął tám.
Y rzekł do Giezego / ſługi ſwego : Záwołáj tej Sunámitki. Y záwołáł jej / á ſtánęłá przed nim.
Tedy mu rzekł : Powiedz jej : Oto piecżołujeƺ á ſtáráƺ śię o wƺyſtkie náƺe potrzeby ; cóż chceƺ / ábym ći ucżynił? Máƺże jáką potrzebę u królá / álbo u hetmáná wojſká? A oná rzekłá : W pośrodku ludu mego mieƺkám.
A on rzekł : Cóż wżdy mám ucżynić dlá niej? Y odpowiedziáł Giezy : Oto ſyná nie má / á mąż jej ſtáry.
Przetoż rzekł : Záwołájże jej. Y záwołáł jej / á oná ſtáłá u drzwi.
Y rzekł : O tym cżáśie po roku będzieƺ piáſtowáłá ſyná. A oná rzekłá : Nie omylájże / pánie mój / mężu Boży / nie omyláj ſłużebnicy twojej.
A ták pocżęłá niewiáſtá / y porodziłá ſyná o onymże cżáśie po roku / jáko jej był powiedziáł Elizeuƺ.
Y podroſło dziećię. Y ſtáło śię dniá niektórego / że wyƺedłƺy do ojcá ſwego / do żeńców /
Rzekło do ojcá ſwego : Głowá mojá! Głowá mojá! A on rzekł ſłudze : Zánieś go do mátki jego.
Który wziąwƺy go / zánióſł go do mátki jego ; y śiedziáł ná łonie jej áż do południá y umárł.
Tedy oná ƺedłƺy położyłá go ná łóżku mężá Bożego / á zámknąwƺy drzwi / wyƺłá.
Potem przyzwáłá mężá ſwego / y rzekłá : Proƺę ćię / poślij ze mną jednego z ſług / y jednę oślicę / że pobieżę áż do mężá Bożego / y wrócę śię záś.
Który rzekł : Po cóż chceƺ jecháć do niego? Dziś niemáƺ nowiu mieśiącá / áni ſábátu. Ale oná rzekłá : Dáj pokój.
A ták ośiodłáwƺy oślicę / rzekłá do ſługi ſwego : Pogániáj / á jedź / y nie mieƺkáj dlá mnie w drodze / chybá żebym ći rozkázáłá.
Jecháłá tedy / y przyjecháłá do mężá Bożego ná górę Kármel. A gdy ją ujrzáł mąż Boży z dáleká / rzekł do Giezego ſługi ſwego : Oto oná Sunámitká.
Przetoż wynijdź przećiwko niej / á rzecż jej : A zdrowáś dobrze? zdrów y mąż twój? zdrów y ſyn?
A oná rzekłá : Zdrowi dobrze. A gdy przyƺłá do mężá Bożego ná górę / uchwyćiłá śię nóg jego ; y przyſtąpił Giezy / áby ją odepchnął. Ale mąż Boży rzekł : Zániecháj jej / boć w gorzkośći jeſt duƺá jej / á Pán zátáił przedemną y nie oznájmił mi.
A oná rzekłá : Azáżem páná mego prośiłá o ſyná? Izálim nie mówiłá : Nie omyláj mię?
Tedy on rzekł do Giezego : Przepáƺ biodrá twe / á weźmij láſkę moję w rękę twą / á idź ; jeźli kogo ſpotkáƺ / nie pozdráwiáj go ; á jeźliby ćię kto pozdrowił / nie odpowiádáj mu ; y połóż láſkę moję ná oblicże dziećięćiá.
A mátká dziećięćiá onego rzekłá : Jáko żywy Pán / y jáko żywá duƺá twojá / że śię ćiebie nie puƺcżę. A ták wſtáwƺy ƺedł zá nią.
A Giezy uprzedził je / y położył láſkę ná oblicże dziećięćiá ; lecż nie było głoſu / áni cżućiá. Przetoż śię wróćił przećiwko niemu y oznájmił mu / mówiąc : Nie ocućiło śię dziećię.
Tedy wƺedł Elizeuƺ do domu / á oto dziećię umárłe leżáło ná łóżku jego.
A gdy tám wƺedł / zámknął drzwi przed onymi obydwomá / y modlił śię Pánu.
Potem wſtąpiwƺy ná łoże / położył śię ná dziećiątko / przyłożywƺy uſtá ſwe do uſt jego / á ocży ſwe do ocżów jego / y ręce ſwe do rąk jego / y rozpoſtárł śię ná nim / ták iż śię zágrzáło ćiáło dziećięce.
A odwróćiwƺy śię od niego / przechádzáł śię po domu tám y ſám ; potem wſtąpił / á położył śię ná nim. Tedy kicháło dziećię áż do śiódmego rázu / y otworzyło dziećię ocży ſwoje.
Tedy záwołáł Giezego / y rzekł : Záwołáj tej Sunámitki. Y záwołáł jej / y przyƺłá do niego / y rzekł : Weźmij ſyná twego /
Którá wƺedłƺy / upádłá u nóg jego / y kłániáłá śię áż do ziemi / á wziąwƺy ſyná ſwego / wyƺłá.
Potem wróćił śię Elizeuƺ do Gálgál / á głód był w onej ziemi / y ſynowie proroccy mieƺkáli przy nim. Tedy rzekł do ſługi ſwego : Przyſtáw gárniec wielki / á uwárz káƺę ſynom prorockim.
Przetoż wyƺedł jeden ná pole / áby zbieráł ziołá / y ználázł máćicę polną / á názbieráł z niej owoców polnych pełen płáƺcż ſwój / á przyƺedłƺy nákrájáł ich w gárniec káƺy ; bo tego nie ználi.
Y wyláli mężom onym / áby jedli. A gdy jedli onę káƺę záwołáli / y rzekli : Śmierć w gárncu / mężu Boży! Y nie mogli jeść.
Y rzekł : Przynieśćie ſámi mąki ; á wſypáwƺy ją w gárniec rzekł : Nálej ludowi. Y jedli / y nie było nic więcej złego w gárncu.
Wtem mąż przyƺedł z Báálſáliſá / á przynióſł mężowi Bożemu chleby / z pierwoćin zbóż / dwádzieśćiá chlebów jęcżmiennych / y kłoſów pełnych świeżych nie wykruƺonych / y rzekł : Dáj ludowi / áby jedli.
Ale odpowiedziáł ſługá jego : Cóż to mám dáć przed ſto mężów? Y rzekł : Dáj ludowi / áby jedli ; ábowiem ták mówi Pán : Będą jedli / y zbędzie.
A ták położył przed nie ; y jedli / á zbyło według ſłowá Páńſkiego.
