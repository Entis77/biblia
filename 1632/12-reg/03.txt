A Jorám / ſyn Achábá / pocżął królowáć nád Izráelem w Sámáryi roku ośmnáſtego Jozáfátá / królá Judzkiego / á królowáł dwánáśćie lát.
Y cżynił złe przed ocżymá Páńſkimi / ácż nie ták jáko ojćiec jego / y jáko mátká jego. Abowiem wyrzućił ſłupy Báálowe / których był nácżynił ojćiec jego.
Wƺákże w grzechách Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá / trwáł á nie odſtąpił od nich.
A Mezá / król Moábſki / miáł doſyć bydłá / á dáwáł królowi Izráelſkiemu ſto tyśięcy jágniąt / y ſto tyśięcy báránów z wełną ich.
Y ſtáło śię / gdy umárł Acháb / że odſtąpił król Moábſki od królá Izráelſkiego.
Wyćiągnął tedy król Jorám dniá onego z Sámáryi / y oblicżył wƺyſtkiego Izráelá.
A wyƺedłƺy poſłáł do Jozáfátá / królá Judzkiego / mówiąc : Król Moábſki odſtąpił odemnie ; poćiągnieƺże zemną przećiw Moábowi ná wojnę? Y odpowiedziáł : Poćiągnę. Jákom já / ták ty ; jáko lud mój / ták lud twój ; jáko konie moje / ták konie twoje.
Zátem rzekł : Którąż drogą poćiągniemy? Y odpowiedziáł : Drogą puƺcży Edomſkiej.
A ták wyćiągnął król Izráelſki y król Judzki / y król Edomſki. A gdy krążyli drogą śiedm dni / nie doſtáło wody wojſku / áni bydłu / które ƺło z nimi.
Y rzekł król Izráelſki : Ach / ách! Abowiem wezwáł Pán tych trzech królów / áby je podáł w ręce Moábſkie.
Ale Jozáfát rzekł : Niemáƺże tu proroká Páńſkiego / żebyśmy śię porádzili Páná przezeń? Y odpowiedziáł jeden z ſług królá Izráelſkiego / á rzekł : Jeſt tu Elizeuƺ / ſyn Sáfátá / który nálewáł wody ná ręce Elijáƺowe.
Tedy rzekł Jozáfát : U tegoć jeſt ſłowo Páńſkie. Y ƺli do niego król Izráelſki / y Jozáfát / y król Edomſki.
Y rzekł Elizeuƺ do królá Izráelſkiego : Co mnie y tobie? Idź do proroków ojcá twego / y do proroków mátki twej. Y rzekł mu król Izráelſki : Nie mów ták ; bo Pán powołáł tych trzech królów / áby je podáł w ręce Moábowe.
Y odpowiedziáł Elizeuƺ : Jáko żywy Pán záſtępów / przed którego oblicżem ſtoję / że gdybym śię nie oglądáł ná Jozáfátá / królá Judzkiego / nie dbáłbym ná ćię / ánibym ná ćię wejrzáł.
Przetoż teráz przywiedźćie mi ná hárfie grájącego. A gdy on grácż gráł / byłá nád nim ręká Páńſká.
Y rzekł : Ták mówi Pán : Pocżyńćie w tym potoku gęſte doły.
Abowiem ták mówi Pán : Nie ujrzyćie wiátru / áni ujrzyćie deƺcżu / wƺákże ten potok będzie pełen wody / ták / że pić będziećie wy / y ſtádá wáƺe / y bydłá wáƺe.
Aleć to jeƺcże máłá w ocżách Páńſkich ; ábowiem dá y Moábity w ręce wáƺe.
Y poburzyćie wƺyſtkie miáſtá obronne / y káżde miáſto wyborne / á wƺyſtkie drzewá dobre podrąbićie / y wƺyſtkie źródłá wód zátkáćie / y káżdą rolę dobrą kámieniem záwálićie.
Y ſtáło śię ráno / kiedy ofiárowáná bywá ofiárá śniedná / oto wody przychodziły drogą Edomſką / y nápełniłá śię ziemiá wodámi.
Tedy wƺyſcy Moábitowie uſłyƺáwƺy / że ćiągną królowie wálcżyć przećiwko nim / zwołáli wƺyſtkie / którzy ták ſtárzy byli / że páſ przypáſáć mogli / y wyżej ; á ſtánęli ná gránicách.
A wſtáwƺy ráno / gdy ſłońce weƺło nád temi wodámi / ujrzeli Moábcżycy náprzećiw ſobie wody cżerwone jáko krew /
Y rzekli : Krew jeſt ; pewnie śię pobili królowie / y pobići ſą jeden od drugiego. A ták teráz do łupu / o Moábcżycy!
A gdy przyƺli do obozu Izráelſkiego / powſtáli Izráelcżycy / y porázili Moábcżyki / którzy ućiekáli przed nimi / á oni je bili / y porázili Moábcżyki ;
Y miáſtá poburzyli / y ná káżde pole wyborne rzućił káżdy kámień ſwój / y zárzućili je / y wƺyſtkie żródłá wód pozátykáli / y wƺyſtkie drzewá dobre podrąbáli / ták / że tylko zániecháli kámieniá w murze Kicháreſet. A obległƺy je ći / co byli z procámi / dobywáli go.
Tedy widząc król Moábſki / że przemágáło przećiw niemu wojſko / wziął z ſobą śiedm ſet mężów dobywájących miecżá / żeby śię przebił przez wojſko królá Edomſkiego / ále nie mogli.
Przetoż pojmáwƺy ſyná jego pierworodnego / który miáł królowáć miáſto niego / ofiárowáł go cáłopáleniem ná murze. Y ſtáło śię zágniewánie wielkie / przećiw Izráelowi / y odćiągnęli od niego / á wróćili śię do ziemi ſwej.
