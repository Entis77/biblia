A Jorám Syn Achábów pocżął królowáć nád Izráelem w Sámáryjey / roku ośmnaſtego / Jozáfátá Królá Judſkiego : á królował dwánaśćie lat.
Y cżynił złe przed ocżymá Páńſkimi : ácż nie ták jáko Oćiec jego / y jáko mátká jego. Abowiem wyrzućił ſłupy Báálowe których był nácżynił Oćiec jego.
Wƺákże w grzechách Jeroboámá Syná Nábátowego / który przywiódł do grzechu Izráelá trwał / <i>á</i> nie odſtąpił od nich.
A Mezá Król Moábſki miał doſyć bydłá : á dawał Królowi Izráelſkiemu ſto tyśięcy jágniąt / y ſto tyśięcy báránów / z wełną ich.
Y ſtáło śię gdy umárł Acháb / że odſtąpił Król Moábſki od Królá Izráelſkiego.
Wyćiągnął tedy król Jorám dniá onego z Sámáryjey / y oblicżył wƺyſtkiego Izráelá.
A wyƺedƺy poſłał do Jozáfátá Królá Judſkiego / mówiąc : Król Moábſki odſtąpił ode mnie : Poćiągnieżże ze mną przećiw Moábowi ná wojnę? Y odpowiedźiał ; Poćiągnę. Jákom ja / ták ty ; jáko lud mój / ták lud twój ; jáko konie moje / ták konie twoje.
Zátym rzekł ; Którąż drogą poćiągniemy : Y odpowiedźiał : Drogą puƺcże Edomſkiey.
A ták wyćiągnął Król Izráelſki y Król Judſki / y Król Edomſki. A gdy krążyli drogą śiedm dni / nie doſtáło wody wojſku / áni bydłu które ƺło z nimi.
Y rzekł Król Izráelſki : Ach / ách! Abowiem wezwał PAN tych trzech Królów / áby je podał w ręce Moábſkie.
Ale Jozáfát rzekł : Niemáƺże tu Proroká Páńſkiego / żebyſmy śię porádźili PANA przezeń? Y odpowiedźiał jeden z ſług Królá Izráelſkiego / á rzekł : Jeſt tu Elizeuƺ / Syn Sáfátów / który nálewał wody ná ręce Eliaƺowe.
Tedy rzekł Jozáfát : U tegoć jeſt ſłowo PAŃſkie. Y ƺli do niego / Król Izráelſki / y Jozáfát / y Król Edomſki.
Y rzekł Elizeuƺ do Królá Izráelſkiego : Co mnie y tobie? idź do Proroków Ojcá twego / y do Proroków mátki twey : Y rzekł mu Król Izráelſki : Nie <i>mów</i> ták ; Bo PAN powołał tych trzech Królów / áby je podał w ręce Moábowe.
Y odpowiedźiał Elizeuƺ : Żywie Pan zaſtępów / przed którego oblicżem ſtoję / że / gdybym śię nie oglądał ná Jozáfátá Królá Judſkiego / nie dbałbym ná ćię / ánibym ná ćię wejrzał.
Przetoż teraz przywiedźćie mi ná hárfie grájącego : á gdy on gracż grał / byłá nád nim ręká Páńſka.
Y rzekł : Ták mówi PAN ; Pocżyńćie w tym potoku gęſte doły.
Abowiem ták mówi PAN : Nie ujrzyćie wiátru áni ujrzyćie deƺcżu / wƺákże ten potok będźie pełen wody / ták / że pić będźiećie wy / y ſtádá wáƺe / y bydłá wáƺe.
Aleć to jeƺcże máła w ocżách Páńſkich : ábowiem da y Moábity w ręce wáƺe.
Y poburzyćie wƺyſtkie miáſtá obronne / y káżde miáſto wyborne / á wƺyſtkie drzewá dobre podrąbićie : y wƺyſtkie źrzódłá wód zátkaćie : y káżdą rolą dobrą kámieniem záwálićie.
Y ſtáło śię ráno / kiedy ofiárowána bywa ofiárá śniedna / oto wody przychodźiły drogą Edomſką y nápełniłá śię źiemiá wodámi.
Tedy wƺyſcy Moábitowie uſłyƺawƺy że ćiągną Królowie walcżyć przećiwko nim / zwołáli wƺyſtkie którzy <i>ták ſtárzy byli że</i> pás przypáſáć mogli / y wyżey : á ſtánęli ná gránicách.
A wſtawƺy ráno / gdy ſłońce weƺło nád tymi wodámi / ujrzeli Moábcżycy náprzećiw ſobie wody cżerwone jáko krew.
Y rzekli ; Krew jeſt : Pewnie śię pobili Królowie / y pobići ſą jeden od drugiego : á ták teraz do łupu o Moábcżycy!
A gdy przyƺli do obozu Izráelſkiego / powſtáli Izráelcżycy / y poráźili Moábcżyki / którzy ućiekáli przed nimi / á oni je bili / y poráźili Moábcżyki :
Y miáſtá poburzyli / y ná káżde pole wyborne rzućił káżdy kámień ſwój : y zárzućili je / y wƺyſtkie źrzódłá wód pozátykáli : y wƺyſtkie drzewá dobre podrąbáli : ták że tylko zániecháli kámienia w murze Kirháreſet : á oblegƺy je ći co byli z procámi / dobywáli go.
Tedy widząc Król Moábſki / że przemagáło przećiw niemu wojſko / wźiął z ſobą śiedm ſet mężów dobywájących miecżá / żeby śię przebił przez <i>wojſko</i> Królá Edomſkiego : Ale nie mogli.
Przetoż pojimawƺy Syná jego pierworodnego który miał królowáć miáſto niego / ofiárował go cáłopaleniem ná murze. Y ſtáło śię zágniewánie wielkie przećiw Izráelowi / y odćiągnęli od niego / á wróćili śię do źiemie ſwey.
