A Jorám / ſyn Achábá / pocżął królowáć nád Izráelem w Sámáryi roku ośmnáſtego Jozáfátá / królá Judzkiego / á królował dwánáśćie lat.
Y cżynił złe przed ocżymá Páńſkimi / ácż nie ták jáko oćiec jego / y jáko mátká jego. Abowiem wyrzućił ſłupy Báálowe / których był nácżynił oćiec jego.
Wƺákże w grzechách Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá / trwał á nie odſtąpił od nich.
A Mezá / król Moábſki / miał doſyć bydłá / á dáwał królowi Izráelſkiemu ſto tyśięcy jágniąt / y ſto tyśięcy báránów z wełną ich.
Y ſtáło śię / gdy umárł Acháb / że odſtąpił król Moábſki od królá Izráelſkiego.
Wyćiągnął tedy król Jorám dniá onego z Sámáryi / y oblicżył wƺyſtkiego Izráelá.
A wyƺedƺy poſłał do Jozáfátá / królá Judzkiego / mówiąc : Król Moábſki odſtąpił odemnie ; poćiągnieƺże zemną przećiw Moábowi ná wojnę? Y odpowiedźiał : Poćiągnę. Jákom ja / ták ty ; jáko lud mój / ták lud twój ; jáko konie moje / ták konie twoje.
Zátem rzekł : Którąż drogą poćiągniemy? Y odpowiedźiał : Drogą puƺcży Edomſkiej.
A ták wyćiągnął król Izráelſki y król Judzki / y król Edomſki. A gdy krążyli drogą śiedm dni / nie doſtáło wody wojſku / áni bydłu / które ƺło z nimi.
Y rzekł król Izráelſki : Ach / ách! Abowiem wezwał Pán tych trzech królów / áby je podał w ręce Moábſkie.
Ale Jozáfát rzekł : Niemáƺże tu proroká Páńſkiego / żebyſmy śię porádźili Páná przezeń? Y odpowiedźiał jeden z ſług królá Izráelſkiego / á rzekł : Jeſt tu Elizeuƺ / ſyn Sáfátá / który nálewał wody ná ręce Elijáƺowe.
Tedy rzekł Jozáfát : U tegoć jeſt ſłowo Páńſkie. Y ƺli do niego król Izráelſki / y Jozáfát / y król Edomſki.
Y rzekł Elizeuƺ do królá Izráelſkiego : Co mnie y tobie? Idź do proroków ojcá twego / y do proroków mátki twej. Y rzekł mu król Izráelſki : Nie mów ták ; bo Pán powołał tych trzech królów / áby je podał w ręce Moábowe.
Y odpowiedźiał Elizeuƺ : Jáko żywy Pán zaſtępów / przed którego oblicżem ſtoję / że gdybym śię nie oglądał ná Jozáfátá / królá Judzkiego / nie dbáłbym ná ćię / ánibym ná ćię wejrzał.
Przetoż teraz przywiedźćie mi ná hárfie grájącego. A gdy on grácż grał / byłá nád nim ręká Páńſká.
Y rzekł : Ták mówi Pán : Pocżyńćie w tym potoku gęſte doły.
Abowiem ták mówi Pán : Nie ujrzyćie wiátru / áni ujrzyćie deƺcżu / wƺákże ten potok będźie pełen wody / ták / że pić będźiećie wy / y ſtádá wáƺe / y bydłá wáƺe.
Aleć to jeƺcże máłá w ocżách Páńſkich ; ábowiem da y Moábity w ręce wáƺe.
Y poburzyćie wƺyſtkie miáſtá obronne / y káżde miáſto wyborne / á wƺyſtkie drzewá dobre podrąbićie / y wƺyſtkie źródłá wód zátkáćie / y káżdą rolę dobrą kámieniem záwálićie.
Y ſtáło śię ráno / kiedy ofiárowáná bywá ofiárá śniedná / oto wody przychodźiły drogą Edomſką / y nápełniłá śię źiemiá wodámi.
Tedy wƺyſcy Moábitowie uſłyƺáwƺy / że ćiągną królowie wálcżyć przećiwko nim / zwołáli wƺyſtkie / którzy ták ſtárzy byli / że pás przypáſáć mogli / y wyżey ; á ſtánęli ná gránicách.
A wſtáwƺy ráno / gdy ſłońce weƺło nád temi wodámi / ujrzeli Moábcżycy náprzećiw ſobie wody cżerwone jáko krew /
Y rzekli : Krew jeſt ; pewnie śię pobili królowie / y pobići ſą jeden od drugiego. A ták teraz do łupu / o Moábcżycy!
A gdy przyƺli do obozu Izráelſkiego / powſtáli Izráelcżycy / y poráźili Moábcżyki / którzy ućiekáli przed nimi / á oni je bili / y poráźili Moábcżyki ;
Y miáſtá poburzyli / y ná káżde pole wyborne rzućił káżdy kámień ſwój / y zárzućili je / y wƺyſtkie żródłá wód pozátykáli / y wƺyſtkie drzewá dobre podrąbáli / ták / że tylko zániecháli kámieniá w murze Kicháreſet. A oblegƺy je ći / co byli z procámi / dobywáli go.
Tedy widząc król Moábſki / że przemágáło przećiw niemu wojſko / wźiął z ſobą śiedm ſet mężów dobywájących miecżá / żeby śię przebił przez wojſko królá Edomſkiego / ále nie mogli.
Przetoż pojmáwƺy ſyná jego pierworodnego / który miał królowáć miáſto niego / ofiárował go cáłopáleniem ná murze. Y ſtáło śię zágniewánie wielkie / przećiw Izráelowi / y odćiągnęli od niego / á wróćili śię do źiemi ſwej.
