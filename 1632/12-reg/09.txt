A Elizeuƺ prorok záwołał jednego z ſynów prorockich / y rzekł mu : Przepáƺ biodrá twoje / á weźmij tę báńkę olejku w rękę twą / á idź do Rámot Gáláádſkiego.
A gdy tám przyjdźieƺ / ujrzyƺ tám Jehu / ſyná Jozáfátowego / ſyná Námſy / á wƺedƺy tám / odwiedźieƺ go z pośrzodku bráći jego / y wprowádźiƺ go do gmáchu nájſkrytƺego.
A wźiąwƺy báńkę olejku / wylejeƺ ná głowę jego / y rzecżeƺ : Ták mówi Pán : Pomázáłem ćię zá królá nád Izráelem. A otworzywƺy drzwi ućiecżeƺ / y nie zábáwiƺ śię tám.
Tedy odƺedł on młodźieniec / ſługá prorocki / do Rámot Gáláádſkiego.
A gdy przyƺedł / oto hetmáni wojſk śiedźieli. Y rzekł : Hetmánie! mam nieco z tobą mówić. Y rzekł Jehu : Z którymże ze wƺyſtkich nas? Y odpowiedźiał : Z tobą / hetmánie!
Tedy wſtáwƺy wƺedł do gmáchu / á on wylał olejek ná głowę jego / y rzekł mu : Ták mówi Pán / Bóg Izráelſki : Pomázáłem ćię zá królá nád ludem Páńſkim / nád Izráelem.
Y wytráćiƺ dom Achábá / páná twego ; ábowiem pomƺcżę śię krwi ſług mojich proroków / y krwi wƺyſtkich ſług Páńſkich / z ręki Jezábeli.
A ták zginie wƺyſtek dom Achábowy ; y wykorzenię z domu Achábowego áż do nájmniejƺego ƺcżenięćiá / y więźniá / y opuƺcżonego w Izráelu.
Y ucżynię domowi Achábowemu / jáko domowi Jeroboámá / ſyná Nábátowego / y jáko domowi Báázy / ſyná Achyjáƺowego.
Jezábelę też zjedzą pſy ná polu Jezreelſkiem / á nie będźie / ktoby ją pogrzebał. To rzekƺy otworzył drzwi / y ućiekł.
A gdy Jehu wyƺedł do ſług páná ſwego / rzekł mu jeden : A dobrzeż wƺyſtko? Pocóż przychodźił ten ƺálony do ćiebie? A on im odpowiedźiał : Wy znáćie tego mężá / y mowę jego.
Tedy rzekli : Nie práwdá to ; proƺę powiedz nam. A on rzekł : Ták á ták rzekł do mnie / mówiąc : Ták mówi Pán : Pomázáłem ćię zá królá nád Izráelem.
Poſpieƺyli śię tedy / á wźiąwƺy káżdy ƺátę ſwą / kłádli je podeń ná nájwyżƺym ſtopniu / y zátrąbiwƺy w trąbę / mówili : Króluje Jehu!
Tedy śię ſprzyśiągł Jehu / ſyn Jozáfátá / ſyná Námſy / przećiw Jorámowi. ( A ná ten cżás Jorám ſtrzegł Rámot Gáláádſkiego / on y wƺyſtek Izráel / przed Házáelem / królem Syryjſkim.
Ale śię był wróćił król Jorám / áby śię lecżył w Jezreel ná rány / które mu byli zádáli Syryjcżycy / gdy wálcżył z Házáelem / królem Syryjſkim. ) Y rzekł Jehu : Jeſli śię wam zda / niech nie wychodźi nikt z miáſtá / żeby ƺedł co oznájmić w Jezreelu.
Y wśiádł ná wóz Jehu / y jechał do Jezreelá / bo tám Jorám leżał ; Ochozyjáƺ tákże / król Judzki / przyjechał był / áby náwiedźił Jorámá.
Wtem ſtróż / który ſtał ná wieży w Jezreelu / ujrzáwƺy pocżet Jehu przyjeżdżájący / rzekł : Pocżet jákiś widzę. Y rzekł Jorám : Weźmij jezdnego / á wyślij przećiwko nim / áby śię ſpytał / jeſli pokój.
A ták bieżał jezdny przećiwko niemu / y rzekł : Ták mówi król : A pokój? Y odpowiedźiał Jehu : Co tobie do pokoju? Obróć śię / jedź zá mną. Przetoż oznájmił ſtróż mówiąc : Dojecháłći poſeł do nich / ále śię nie wrácá.
Zátem poſłał drugiego jezdnego / który przyjecháwƺy do nich / rzekł : Ták mówi król : A pokój? Odpowiedźiał Jehu : Co tobie do pokoju? Obróć śię / á jedź zá mną.
Znowu oznájmił to ſtróż / mówiąc : Przyjecháłći do nich / ále śię nie wrácá. A przyjázd jego / jákoby przyjázd Jehu / ſyná Námſy ; bo ƺálenie jedźie.
Tedy rzekł Jorám : Záprzęgáj. Y záprzężono w wóz jego. Y wyjechał Jorám / król Izráelſki / y Ochozyjáƺ / król Judzki / káżdy ná woźie ſwym. A wyjecháwƺy przećiw Jehu / tráfili go ná polu Nábotá Jezreelſkiego.
A gdy ujrzał Jorám Jehu / rzekł : Jeſtże pokój Jehu? Y odpowiedźiał : Co zá pokój? ponieważ jeƺcże cudzołóſtwá Jezábeli / mátki twojey / y cżáry jey wielkie ſą.
Przetoż obróćiwƺy śię Jorám ućiekał / mówiąc do Ochozyjáƺá : Zdrádá / Ochozyjáƺu!
Tedy Jehu wźiąwƺy w ręce ſwoje łuk / poſtrzelił Jorámá miedzy rámioná jego / áż przeƺłá ſtrzáłá przez ſerce jego / ták / że pádł ná woźie ſwoim.
Potem rzekł Jehu do Bádákierá / hetmáná ſwego : Weźmij go / á porzuć ná polu Nábotá Jezreelſkiego ; ábowiem pámiętáƺ / gdyśmy ja y ty jecháli ſpołu zá Achábem / ojcem jego / że Pán wydał był przećiwko niemu tę pogróżkę.
Zájiſte krwi Nábotá / y krwi ſynów jego / którąm widźiał wcżoráj / rzekł Pán / pomƺcżę śię nád tobą ná tym polu. Pán to rzekł : przetoż teraz weźmij go / á porzuć go ná polu według ſłowá Páńſkiego.
Co Ochozyjáƺ / król Judzki / ujrzáwƺy / ućiekał drogą / do domu ogrodowego. Ale go gonił Jehu / y rzekł : Y tego zábijćie ná woźie jego. Y zránili go ná wſtąpie Guru / który jeſt podle Jebláám. A ućiekł do Mágieddy / y tám umárł.
Y kázáli go záwieść ſłudzy jego do Jeruzálemu / á pogrzebli go w grobie jego z ojcámi jego w mieśćie Dawidowem.
A roku jedenáſtego Jorámá / ſyná Achábowego / królował Ochozyjáƺ nád Judą.
Zátem przyƺedł Jehu do Jezreel. Co gdy Jezábelá uſłyƺáłá / ufárbowáłá twárz ſwoję / y ochędożyłá głowę ſwą á pátrzáłá z okná.
A gdy Jehu wjeżdżał w bramę / rzekłá : Jeſtże pokój / o Zymry / morderzu páná ſwego?
A on podnióſƺy twárz ſwoję ku oknu / rzekł : Któż ze mną trzymá / kto? Tedy wejrzeli nań dwáj álbo trzej komornicy jey.
Którym rzekł : Zrzuććie ją. Y zrzućili ją / y popryſkáłá śię śćiáná y konie krwią jey / y podeptał ją.
A gdy tám wƺedł / jádł y pił / y rzekł : Obácżćie proƺę onę przeklętą / á pogrzebćie ją ; boć córką królewſką jeſt.
Tedy ƺedƺy / áby ją pogrzebli / nie ználeſli z niej jedno cżáƺkę z głowy / y nogi / y dłoni rąk.
A wróćiwƺy śię / oznájmili mu to. Który rzekł : Wypełniło śię ſłowo Páńſkie / które powiedźiał przez ſługę ſwego Elijáƺá Teſbitę / mówiąc : Ná polu Jezreel zjedzą pſy ćiáło Jezábeli.
Niech będźie trup Jezábeli / jáko gnój ná roli / ná polu Jezreel / ták żeby nie mówiono. Táć jeſt Jezábelá.
