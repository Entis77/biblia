Roku wtórego Joázá / ſyná Joácházá / królá Izráelſkiego / pocżął królowáć Amázyjáƺ / ſyn Joázá / królá Judzkiego.
Dwádźieśćiá y pięć lát miáł / gdy królowáć pocżął / á dwádźieśćiá y dźiewięć lát królowáł w Jeruzálemie. Imię mátki jego było Joádáná z Jeruzálemu.
Ten cżynił / co dobrego jeſt przed ocżymá Páńſkimi / ácżkolwiek nie ták jáko Dawid / ojćiec jego ; według wƺyſtkiego / co cżynił Joáz / ojćiec jego / poſtępowáł.
Wƺákże wyżyny nie były znieśione ; jeƺcże lud ofiárowáł y kádźił po wyżynách.
A gdy zmocnione było króleſtwo w ręku jego / pobił ſługi ſwe / którzy byli zábili królá / ojcá jego.
Lecż ſynów onych morderców nie pobił / jáko nápiſáno w kśięgách zákonu Mojżeƺowego / gdźie rozkázáł Pán / mówiąc : Nie pomrą ojcowie zá ſynów / áni ſynowie pomrą zá ojców / ále káżdy zá grzech ſwój umrze.
Ten też poráźił Edomcżyków dźieśięć tyśięcy w dolinie ſolnej / y wźiął mocą Selę / á názwáł imię jej Jokteel / áż do tego cżáſu.
Tedy poſłáł Amázyjáƺ poſły do Joázá / ſyná Joácházá / ſyná Jehu / królá Izráelſkiego / mówiąc : Pójdź / wejrzymy ſobie w ocży.
Poſłáł záśię Joáz / król Izráelſki / do Amázyjáƺá / królá Judzkiego / mówiąc : Oſet / który jeſt ná Libánie / poſłáł do cedru Libáńſkiego / mówiąc : Dáj córkę twoję ſynowi memu zá żonę. Wtem przyƺedł zwierz polny / który jeſt ná Libánie / y podeptáł on oſet.
Żeś ty bárdzo poráźił Edomcżyki / dlátego śię podnioſło ſerce twoje. Chlubże śię / á śiedź w domu twoim ; y przecżże śię máƺ wdáwáć w to złe / ábyś upádł ty / y Judá z tobą?
Ale nie uſłucháł Amázyjáƺ. Przetoż wyćiągnął Joáz / król Izráelſki / á wejrzeli ſobie w ocży / on y Amázyjáƺ / król Judzki / w Betſemeſ / które jeſt w Judztwie.
Y poráżony jeſt Judá od ludu Izráelſkiego / á ućiekł káżdy do przybytku ſwego.
Lecż Amázyjáƺá / królá Judzkiego / ſyná Joázá / ſyná Ochozyjáƺowego / pojmáł Joáz / król Izráelſki / w Betſemeſ / á przyćiągnąwƺy do Jeruzálemu / zburzył mur Jeruzálemſki od brámy Efráim áż do brámy nárożnej / ná cżtery ſtá łokći.
Y zábráł wƺyſtko złoto y ſrebro y wƺyſtkie nácżyniá / które śię ználázły w domu Páńſkim / y w ſkárbách domu królewſkiego / y ludźie záſtáwne / y wróćił śię do Sámáryi.
A inne ſpráwy Joázowe / które cżynił / y moc jego / y jáko wálcżył z Amázyjáƺem / królem Judzkim / ázáż tego nie zápiſáno w kronikách o królách Izráelſkich?
Y záſnął Joáz z ojcámi ſwymi / á pogrzebiony jeſt w Sámáryi z królmi Izráelſkimi / á królowáł Jeroboám / ſyn jego / miáſto niego.
Y żył Amázyjáƺ / ſyn Joázowy / król Judzki / po śmierći Joázá / ſyná Joácházá / królá Izráelſkiego / piętnáśćie lát.
A inne ſpráwy Amázyjáƺowe / ázáż nie ſą opiſáne w kronikách o królách Judzkich?
Potem ſprzyśięgli śię przećiwko niemu niektórzy w Jeruzálemie ; ále ućiekł do Láchyſ. Przetoż poſłáwƺy zá nim do Láchyſ / zábili go tám.
Skąd przynieśli go ná koniách / y pogrzebiony jeſt w Jeruzálemie z ojcámi ſwymi / w mieśćie Dawidowem.
A ták wźiąwƺy wƺyſtek lud Judzki Azáryjáƺá / któremu było ƺeſnáśćie lát / poſtánowili go królem ná miejſcu ojcá jego Amázyjáƺá.
Ten pobudowáł Elát / y przywróćił je do Judy / gdy záſnął król z ojcámi ſwymi.
Roku piętnáſtego Amázyjáƺá / ſyná Joázá / królá Judzkiego / królowáł Jeroboám / ſyn Joázá / królá Izráelſkiego / w Sámáryi cżterdźieśći lát y rok.
A cżynił złe przed ocżymá Páńſkimi / nie uchylájąc śię od wƺyſtkich grzechów Jeroboámá / ſyná Nábátowego / który przywiódł do grzechu Izráelá.
Ten záśię przywróćił gránice Izráelſkie od wejśćiá do Emát áż do morzá puſtego / według ſłowá Páná / Bogá Izráelſkiego / które był wyrzekł przez ſługę ſwego Jonáƺá / ſyná Amáty / proroká ; który był z Gátefer.
Abowiem widźiáł Pán utrápienie Izráelſkie / im dálej tem więkƺe / ták / że y więźień / y opuƺcżony zniƺcżeni byli / á nie był / ktoby rátowáł Izráelá.
A nie rzekł był Pán / áby miáł wygłádźić imię Izráelá / żeby nie zoſtáło pod niebem : przetoż je wybáwił przez rękę Jeroboámá / ſyná Joázáowego.
A inne ſpráwy Jeroboámowe / y wƺyſtko co cżynił / y moc jego / którą wálcżył / y którą przywróćił Dámáƺek y Emát Judzkie Izráelowi / ázáż tego nie zápiſáno w kronikách o królách Izráelſkich?
Y záſnął Jeroboám z ojcámi ſwymi / z królmi Izráelſkimi / á królowáł Zácháryjáƺ / ſyn jego / miáſto niego.
