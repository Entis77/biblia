Tedy rzekł Elizeuƺ : Słuchájćie ſłowá Páńſkiego. Ták mówi Pán : O tym cżáśie jutro miárá mąki pƺennej będzie zá ſykiel / á dwie miáry jęcżmieniá zá ſykiel / w brámie Sámáryjſkiej.
Y odpowiedziáł kśiąże / ná którego śię ręce król wſpieráł / mężowi Bożemu / y rzekł : By też Pán pocżynił okná w niebie / izáliby to mogło być? Który mu rzekł : Oto ty ujrzyƺ ocżymá twemi ; ále tego jeść nie będzieƺ.
A byli cżterej mężowie trędowáći u wejśćiá brámy / którzy rzekli jeden do drugiego : Pocóż tu mieƺkámy / áżbyśmy pomárli?
Jeźli wnijdziemy do miáſtá / głód w mieśćie / y pomrzemy tám / á jeźli tu zoſtániemy / przećię pomrzemy. Teráz tedy pójdźćie / á zbieżmy do obozu Syryjſkiego ; jeźli náſ żywo zoſtáwią / będziemy żywi ; jeźli náſ też zábiją / pomrzemy.
Wſtáli tedy / gdy śię zmierzcháć pocżęło / áby ƺli do obozu Syryjſkiego ; á przyƺedłƺy ná koniec obozu Syryjſkiego / oto tám nie było nikogo.
Abowiem ſpráwił Pán / że ſłycháć było w obozie Syryjſkim grzmot wozów y tenten koni / y huk wojſká wielkiego / y rzekli jeden do drugiego : Oto nájął zá pieniądze przećiwko nám król Izráelſki króle Hetejſkie / y króle Egipſkie / áby przypádli ná náſ.
A ták wſtáwƺy ućiekli w zmierzch / zoſtáwiwƺy námioty ſwe / y konie ſwe y oſły ſwe / y obóz jáki był / á ućiekli / chcąc záchowáć duƺę ſwoję.
A gdy przyƺli oni trędowáći áż ná przodek obozu / weƺli do jednego námiotu / y jedli y pili / á nábráwƺy ſtámtąd ſrebrá y złotá / y ƺát / ƺli y ſkryli. Potem śię wróćili / y weƺli do drugiego námiotu / á nábráwƺy tákże ſtámtąd / odeƺli y pokryli.
Zátem rzekł jeden do drugiego : Nie dobrze cżynimy. Dzień ten jeſt dzień dobrej nowiny / á my milcżymy? Jeźli będziemy cżekáli áż do záránku / będziemy winni grzechu. Przetoż teráz pójdźćie / wnijdźmy / á opowiedzmy to domowi królewſkiemu.
A ták przyƺedłƺy záwołáli ná wrotnego miejſkiego / y powiedzieli im mówiąc : Przyƺliśmy do obozu Syryjſkiego / á oto nie było tám nikogo / áni głoſu ludzkiego / oprócż koni uwiązánych / y oſłów uwiązánych / y námiotów / jáko przedtem były.
Tedy on záwołáł ná inne wrotne / á ći opowiedzieli to w domu królewſkim.
Wſtáwƺy tedy król w nocy / rzekł do ſług ſwoich : Powiem já wám / co nám ucżynili Syryjcżycy ; wiedzą / żeśmy zgłodniáli / przetoż wyƺli z obozu / á pokryli śię w polu / mówiąc : Gdy wynijdą z miáſtá / pojmiemy je żywo / y miáſto ubieżemy.
Tedy odpowiedziáł jeden z ſług jego / y rzekł : Proƺę niech wezmą pięć koni pozoſtáłych / które zoſtáły w mieśćie ; ( oto one ſą jáko wƺyſtko mnóſtwo Izráelſkie / które zoſtáło w nim ; oto one ſą mówię jáko wƺyſtko mnóſtwo Izráelſkie / które ginie / )te wyślijmy á wywiedzmy śię.
A ták wziąwƺy dwá wozy z końmi / poſłáł król do obozu Syryjſkiego / mówiąc : Idźćie á obácżćie.
Y ƺli zá nimi áż do Jordánu / á oto po wƺyſtkiej drodze pełno było ƺát y nácżyniá / które porzucáli Syryjcżycy / kwápiąc śię. Tedy wróćiwƺy śię oni poſłowie / oznájmili to królowi.
Przeto wyƺedłƺy lud / rozchwyćił obóz Syryjſki ; á byłá miárá pƺennej mąki zá ſykiel / á dwie miáry jęcżmieniá zá ſykiel / według ſłowá Páńſkiego.
A król poſtánowił był onego kśiążęćiá / ná którego śię ręce wſpieráł / w brámie / którego lud podeptáł w brámie / áż umárł / jáko mu był powiedziáł mąż Boży / który o tem mówił / gdy był król przyƺedł do niego.
Y ſtáło śię według ſłowá / które był rzekł mąż Boży królowi / mówiąc : Dwie miáry jęcżmieniá zá ſykiel / á miárá pƺennej mąki będzie zá ſykiel / jutro o tym cżáśie w brámie Sámáryjſkiej.
Ná co był odpowiedziáł on kśiąże mężowi Bożemu / mówiąc : By też Pán ucżynił okná w niebie / izáli to będzie według ſłowá tego? A on mu rzekł : Oto ty ujrzyƺ ocżymá twemi / ále tego jeść nie będzieƺ.
Y ſtáło mu śię ták ; bo go podeptáł lud w brámie / áż umárł.
