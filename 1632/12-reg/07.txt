Tedy rzekł Elizeuƺ : Słuchájćie ſłowá Páńſkiego. Ták mówi Pán : O tym cżáśie jutro miárá mąki pƺenney będźie zá ſykiel / á dwie miáry jęcżmieniá zá ſykiel / w brámie Sámáryjſkiej.
Y odpowiedźiáł kśiąże / ná którego śię ręce król wſpieráł / mężowi Bożemu / y rzekł : By też Pán pocżynił okná w niebie / izáliby to mogło być? Który mu rzekł : Oto ty ujrzyƺ ocżymá twemi ; ále tego jeść nie będźieƺ.
A byli cżterej mężowie trędowáći u wejśćiá brámy / którzy rzekli jeden do drugiego : Pocóż tu mieƺkámy / áżbyśmy pomárli?
Jeſli wnijdźiemy do miáſtá / głód w mieśćie / y pomrzemy tám / á jeſli tu zoſtániemy / przećię pomrzemy. Teraz tedy pójdźćie / á zbieżmy do obozu Syryjſkiego ; jeſli nas żywo zoſtáwią / będźiemy żywi ; jeſli nas też zábiją / pomrzemy.
Wſtáli tedy / gdy śię zmierzcháć pocżęło / áby ƺli do obozu Syryjſkiego ; á przyƺedłƺy ná koniec obozu Syryjſkiego / oto tám nie było nikogo.
Abowiem ſpráwił Pán / że ſłycháć było w oboźie Syryjſkim grzmot wozów y tenten koni / y huk wojſká wielkiego / y rzekli jeden do drugiego : Oto nájął zá pieniądze przećiwko nám król Izráelſki króle Hetejſkie / y króle Egipſkie / áby przypádli ná nas.
A ták wſtáwƺy ućiekli w zmierzch / zoſtáwiwƺy námioty ſwe / y konie ſwe y oſły ſwe / y obóz jáki był / á ućiekli / chcąc záchowáć duƺę ſwoję.
A gdy przyƺli oni trędowáći áż ná przodek obozu / weƺli do jednego námiotu / y jedli y pili / á nábráwƺy ſtámtąd ſrebrá y złotá / y ƺát / ƺli y ſkryli. Potem śię wróćili / y weƺli do drugiego námiotu / á nábráwƺy tákże ſtámtąd / odeƺli y pokryli.
Zátem rzekł jeden do drugiego : Nie dobrze cżynimy. Dźień ten jeſt dźień dobrej nowiny / á my milcżymy? Jeſli będźiemy cżekáli áż do záránku / będźiemy winni grzechu. Przetoż teraz pójdźćie / wnijdźmy / á opowiedzmy to domowi królewſkiemu.
A ták przyƺedłƺy záwołáli ná wrotnego miejſkiego / y powiedźieli im mówiąc : Przyƺliśmy do obozu Syryjſkiego / á oto nie było tám nikogo / áni głoſu ludzkiego / oprócż koni uwiązánych / y oſłów uwiązánych / y námiotów / jáko przedtem były.
Tedy on záwołáł ná inne wrotne / á ći opowiedźieli to w domu królewſkim.
Wſtáwƺy tedy król w nocy / rzekł do ſług ſwoich : Powiem ja wám / co nám ucżynili Syryjcżycy ; wiedzą / żeśmy zgłodniáli / przetoż wyƺli z obozu / á pokryli śię w polu / mówiąc : Gdy wynijdą z miáſtá / pojmiemy je żywo / y miáſto ubieżemy.
Tedy odpowiedźiáł jeden z ſług jego / y rzekł : Proƺę niech wezmą pięć koni pozoſtáłych / które zoſtáły w mieśćie ; ( oto one ſą jáko wƺyſtko mnóſtwo Izráelſkie / które zoſtáło w nim ; oto one ſą mówię jáko wƺyſtko mnóſtwo Izráelſkie / które ginie / )te wyślijmy á wywiedzmy śię.
A ták wźiąwƺy dwá wozy z końmi / poſłáł król do obozu Syryjſkiego / mówiąc : Idźćie á obácżćie.
Y ƺli zá nimi áż do Jordánu / á oto po wƺyſtkiej drodze pełno było ƺát y nácżyniá / które porzucáli Syryjcżycy / kwápiąc śię. Tedy wróćiwƺy śię oni poſłowie / oznájmili to królowi.
Przeto wyƺedłƺy lud / rozchwyćił obóz Syryjſki ; á byłá miárá pƺenney mąki zá ſykiel / á dwie miáry jęcżmieniá zá ſykiel / według ſłowá Páńſkiego.
A król poſtánowił był onego kśiążęćiá / ná którego śię ręce wſpieráł / w brámie / którego lud podeptáł w brámie / áż umárł / jáko mu był powiedźiáł mąż Boży / który o tem mówił / gdy był król przyƺedł do niego.
Y ſtáło śię według ſłowá / które był rzekł mąż Boży królowi / mówiąc : Dwie miáry jęcżmieniá zá ſykiel / á miárá pƺenney mąki będźie zá ſykiel / jutro o tym cżáśie w brámie Sámáryjſkiej.
Ná co był odpowiedźiáł on kśiąże mężowi Bożemu / mówiąc : By też Pán ucżynił okná w niebie / izáli to będźie według ſłowá tego? A on mu rzekł : Oto ty ujrzyƺ ocżymá twemi / ále tego jeść nie będźieƺ.
Y ſtáło mu śię ták ; bo go podeptáł lud w brámie / áż umárł.
