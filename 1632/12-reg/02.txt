Y ſtáło śię / gdy miáł wziąć Pán Elijáƺá w wichrze do niebá / że wyƺedł Elijáƺ z Elizeuƺem z Gálgál.
Y rzekł Elijáƺ do Elizeuƺá : Proƺę śiedź tu ; bo mię Pán poſłáł áż do Betel. Y rzekł Elizeuƺ : Jáko żywy Pán / y jáko żywá duƺá twojá / że śię ćiebie nie puƺcżę. Y przyƺli do Betel.
Tedy wyƺli ſynowie proroccy / którzy byli w Betel / do Elizeuƺá / y rzekli do niego : Wieƺże / iż dziś Pán weźmie od ćiebie páná twego? A on rzekł : Wiemći ; milcżćie tylko.
Znowu rzekł mu Elijáƺ : Elizeuƺu / proƺę śiedź tu ; bo mię Pán poſłáł do Jerychá. A on odpowiedziáł : Jáko żywy Pán / y jáko żywá duƺá twojá / że śię ćiebie nie puƺcżę. A ták przyƺli do Jerychá.
Tedy przyſtąpiwƺy ſynowie proroccy / którzy byli w Jerychu / do Elizeuƺá / rzekli do niego : Wieƺże / że dziś Pán weźmie páná twego od ćiebie? A on rzekł : Wiemći ; milcżćie.
Jeƺcże mu rzekł Elijáƺ : Proƺę śiedź tu ; bo mię Pán poſłáł do Jordánu. Który odpowiedziáł : Jáko żywy Pán / jáko żywá y duƺá twojá / że śię ćiebie nie puƺcżę.
Y ƺli obádwáj. A pięćdzieśiąt mężów ſynów prorockich ƺli / y ſtánęli náprzećiwko z dáleká ; ále oni obáj ſtánęli nád Jordánem.
A wziąwƺy Elijáƺ płáƺcż ſwój / zwinął go / á uderzył nim wody / y rozdzieliły śię tám y ſám / ták iż przeƺli obáj po ſuƺy.
A gdy przeƺli / rzekł Elijáƺ do Elizeuƺá : Żądáj / cżego chceƺ / ábym ći ucżynił pierwej niż będę wzięty od ćiebie. Tedy rzekł Elizeuƺ : Proƺę niech będzie dwójnáſobny duch twój we mnie ;
Ale mu on odpowiedziáł : Trudnejś rzecży pożądáł ; wƺákże ujrzyƺli mię / gdy będę wzięty od ćiebie / ták ći śię ſtánie ; ále jeźli nie ujrzyƺ / nie ſtánieć śię.
Y ſtáło śię / gdy oni przećię ƺli rozmáwiájąc / oto wóz ogniſty / y konie ogniſte rozłącżyły obydwóch. Y wſtąpił Elijáƺ w wichrze do niebá.
Co Elizeuƺ widząc / wołáł : Ojcże mój / ojcże mój! Wozie Izráelſki y jázdo jego. Y nie widziáł go więcej. A pochwyćiwƺy ƺáty ſwe rozdárł je ná dwie cżęśći.
Y podnióſł płáƺcż Elijáƺowy / który był ſpádł z niego / á wróćiwƺy śię / ſtánął nád brzegiem Jordánu. A ták wziąwƺy płáƺcż Elijáƺowy / który był ſpádł z niego / uderzył nim wody / mówiąc : Gdzież jeſt Pán / Bóg Elijáƺowy?
A ták y on uderzył nim wody / á rozdzieliły śię tám y ſám / y przeƺedł Elizeuƺ.
Co widząc ſynowie proroccy / którzy byli w Jerycho / ſtojąc ná przećiwko / rzekli : Odpocżął duch Elijáƺowy nád Elizeuƺem ; á wyƺedłƺy przećiwko niemu / pokłonili mu śię áż do ziemi.
Y rzekli do niego : Oto teráz jeſt przy ſługách twych pięćdzieśiąt mężów mocnych. Proƺę niech idą / á niech ƺukáją Páná twego ; by go ſnáć nie zánióſł Duch Páńſki / á nie porzućił go ná której górze / álbo w której dolinie. Ale im on rzekł : Nie poſyłájćie.
A gdy náń nálegáli áż do uprzykrzeniá / rzekł : Poślijćież. A ták poſłáli onych pięćdzieśiąt mężów / którzy ƺukájąc przez trzy dni nie ználeźli go.
A gdy śię wróćili do niego / ( á on mieƺkáł w Jerycho / )rzekł do nich : Azáżem wám nie mówił : Nie chodźćie?
Rzekli też mężowie onego miáſtá do Elizeuƺá : Wej / oto mieƺkánie miáſtá tego jeſt dobre / jáko pánie mój widziƺ ; ále wody złe y ziemiá niepłodná.
Tedy rzekł : Przynieśćie mi báńkę nową / á włóżćie w nię ſoli. Y przynieśli mu.
A poƺedłƺy do źródłá wód / wrzućił tám ſoli / y rzekł : Ták mówi Pán : Uzdrowiłem te wody ; nie będzie więcej ſtámtąd śmierći / áni niepłodnośći.
A ták uzdrowione ſą one wody áż do dniá tego / według ſłowá Elizeuƺowego / które był powiedziáł.
Potem ƺedł ſtámtąd do Betel. A gdy ƺedł drogą / dzieći máłe wyƺły z miáſtá / y náśmiewáły śię z niego / y mówiły mu : Idźże łyſy / idźże łyſy!
Który obejrzáwƺy śię / ujrzáł je / y złorzecżył im w imieniu Páńſkim. Przetoż wyƺedłƺy dwie niedźwiedzice z láſu / rozdrápáły z nich cżterdzieśći y dwoje dzieći.
Y ƺedł ſtámtąd ná górę Kármel / á z onąd záśię wróćił śię do Sámáryi.
