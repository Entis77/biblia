Y ſtáło śię gdy miał wźiąć PAN Eliaƺá w wichrze do Niebá / że wyƺedł Eliaƺ z Elizeuƺem z Gálgál.
Y rzekł Eliaƺ Elizeuƺowi : Proƺę śiedź tu : bo mię PAN poſłał áż do Bethel. Y rzekł Elizeuƺ ; Żywie PAN / y żywie duƺá twojá / że śię ćiebie nie puƺcżę : y przyƺli do Bethel.
Tedy wyƺli Synowie Proroccy / którzy <i>byli</i> w Bethel / do Elizeuƺá / y rzekli do niego : Wieƺże iż dźiś PAN weźmie od ćiebie Páná twego? A on rzekł : Wiemći ; Milcżćie <i>tylko</i>.
Znowu rzekł mu Eliaƺ : Elizeuƺu / proƺę śiedź tu / bo mię PAN poſłał do Jerychá : A on odpowiedźiał : Żywie PAN / y żywie duƺá twojá / że śię ćiebie nie puƺcżę. A ták przyƺli do Jerychá.
Tedy przyſtąpiwƺy Synowie Proroccy którzy byli w Jerychu do Elizeuƺá / rzekli do niego : Wieƺże / że dźiś PAN weźmie Páná twego od ćiebie? A on rzekł ; Wiemći : Milcżćie.
Jeƺcże mu rzekł Eliaƺ : Proƺę śiedź tu / bo mię PAN poſłał do Jordanu : Który odpowiedźiał : Żywie PAN / żywie y duƺá twojá / że śię ćiebie nie puƺcżę.
Y ƺli obádwá. A pięćdźieśiąt mężów Synów Prorockich ƺli / y ſtánęli náprzećiwko z dáleká : Ale oni obá ſtánęli nád Jordanem.
A wźiąwƺy Eliaƺ płaƺcż ſwój / zwinął go / á uderzył nim wody / y rozdźieliły śię tám y ſám / ták iż przeƺli obá po ſuƺy.
A gdy przeƺli / rzekł Eliaƺ do Elizeuƺá : Żądaj cżego chceƺ / ábymći ucżynił pierwey niż będę wźięt od ćiebie. Tedy rzekł Elizeuƺ : Proƺę niech będźie dwójnaſobny Duch twój we mnie.
Ale mu on odpowiedźiał : Trudneyś rzecży pożądał : Wƺákże ujrzyƺli mię / gdy będę wźięt od ćiebie / tákći śię ſtánie : ále jeſli nie <i>ujrzyƺ,</i> nie ſtánieć śię.
Y ſtáło śię / gdy oni przećię ƺli rozmawiájąc / oto wóz ogniſty / y konie ogniſte rozłącżyły obudwu : y wſtąpił Eliaƺ w wichrze do niebá.
Co Elizeuƺ widząc / wołał : Ojcże mój / Ojcże mój! Woźie Izráelſki / y jázdo jego. Y nie widźiał go więcey. A pochwyćiwƺy ƺáty ſwe / rozdárł je ná dwie cżęśći.
Y podnióſł płaƺcż Eliaƺów który był ſpadł z niego : á wróćiwƺy śię / ſtánął nád brzegiem Jordanu. A ták wźiąwƺy płaƺcż Eliaƺów / który był ſpadł z niego / uderzył nim wody / mówiąc : Gdźież jeſt PAN / Bóg Eliaƺów?
A ták y on uderzył nim wody / á rozdźieliły śię tám y ſám / y przeƺedł Elizeuƺ.
Co widząc Synowie Proroccy / którzy <i>byli</i> w Jerycho / ſtojąc náprzećiwko / rzekli : Odpocżynął Duch Eliaƺów nád Elizeuƺem : á wyƺedƺy przećiwko niemu / pokłonili mu śię áż do źiemie.
Y rzekli do niego : Oto teraz jeſt przy ſługách twych pięćdźieśiąt mężów mocnych : proƺę niech idą / á niech ƺukáją Páná twego : by go ſnadź nie zánióſł Duch Páńſki / á nie porzućił go ná którey górze / álbo w którey dolinie. Ale im on rzekł : Nie poſyłajćie.
A gdy nań nálegáli áż do uprzykrzenia / rzekł : Poślićież. A ták poſłáli onych pięćdźieśiąt mężów / którzy ƺukájąc przez trzy dni / nie ználeźli go.
A gdy śię wróćili do niego ( á on mieƺkał w Jerychu ) rzekł do nich : Azam wam nie mówił / Nie chodźćie?
Rzekli też mężowie onego miáſtá do Elizeuƺá : Wej / oto mieƺkánie miáſtá tego jeſt dobre / jáko Pánie mój widźiƺ ; ále wody złe y źiemiá niepłodna.
Tedy rzekł : Przynieśćie mi báńkę nową / á włóżćie w nię ſoli : y przynieśli mu.
A poƺedƺy do źrzódłá wód / wrzućił tám ſoli / y rzekł : Ták mówi PAN ; Uzdrowiłem te wody : nie będźie więcey z támtąd śmierći / áni niepłodnośći :
A ták uzdrowione ſą one wody áż do dniá tego / według ſłowá Elizeuƺowego / które był powiedźiał.
Potym ƺedł z támtąd do Bethel. A gdy ƺedł drogą / dźieći máłe wyƺły z miáſtá / y náśmiewáły śię z niego / y mówiły mu : Idźże łyſy : idźże łyſy!
Który obejrzawƺy śię / ujrzał je / y złorzecżył im w Imieniu Páńſkim. Przetoż wyƺedƺy dwie niedźwiedźice z láſá / rozdrápáły z nich cżterdźieśći y dwoje dźieći.
Y ƺedł z támtąd ná górę Kármel / á z onąd <i>záśię</i> wróćił śię do Sámáryjey.
