W one dni záchorowáł Ezechyjáƺ áż ná śmierć / y przyƺedł do niego Izájáƺ prorok / ſyn Amoſowy / y rzekł mu : Ták mówi Pán : Rozpráw dom twój ; bo umrzeƺ / á nie będzieƺ żył.
Tedy obróćił Ezechyjáƺ twárz ſwoję do śćiány / y modlił śię Pánu / mówiąc :
Proƺę / o Pánie! wſpomnij teráz / żem chodził przed tobą w prwádzie / y w ſercu cáłem / cżyniąc to / co dobrego jeſt w ocżách twoich. Y płákáł Ezechyjáƺ płácżem wielkiem.
Ale jeƺcże Izájáƺ nie wyƺedł był do pół śieni / gdy śię ſłowo Páńſkie ſtáło do niego / mówiąc :
Wróć śię / á mów do Ezechyjáƺá / wodzá ludu mego : Ták mówi Pán / Bóg Dawidá / ojcá twego : Wyſłucháłem modlitwę twoję / á widziáłem łzy twoje ; oto Já uzdráwiám ćię / dniá trzećiego wnijdzieƺ do domu Páńſkiego ;
Y przydám do dni twoich piętnáśćie lát / á z ręki królá Aſſyryjſkiego wyrwę ćiebie / y to miáſto ; y bronić będę tego miáſtá dlá śiebie / y dlá Dawidá / ſługi mego.
Przytem rzekł Izájáƺ : Przynieśćie bryłę fig ſuchych. Którą przynióſłƺy włożyli ná wrzód / y zgoił śię.
Y rzekł Ezechyjáƺ do Izájáƺá : Jáki znák tego / że mię uzdrowi Pán / á iż pójdę dniá trzećiego do domu Páńſkiego?
Odpowiedziáł Izájáƺ : Toć będzie znákiem od Páná / iż ucżyni Pán tę rzecż / którąć obiecáł. Chceƺże / żeby ćień poſtąpił ná dzieśięć ſtopni / álbo żeby śię ná wſtecż wróćił ná dzieśięć ſtopni?
Y rzekł Ezechyjáƺ : Snádniej może ćień poſtąpić ná dół ná dzieśięć ſtopni / tego nie chcę ; ále niech śię wróći ćień ná wſtecż ná dzieśięć ſtopni.
Tedy wołáł Izájáƺ prorok do Páná ; y náwróćił ćień po onych ſtopniách / któremi był poſtąpił ná zegárze ſłonecżnym Acházowym / ná wſtecż ná dzieśięć ſtopni.
Onegoż cżáſu poſłáł Berodách Báládán / ſyn Báládánowy / król Bábilońſki / liſt y dáry do Ezechyjáƺá ; bo záſłyƺáł / że był zániemógł Ezechyjáƺ.
Y wyſłucháł ich Ezechyjáƺ y okázáł im wƺyſtkie ſkárbnice klejnotów ſwoich / ſrebro / y złoto / y rzecży wonne / y olejki nájwyborniejƺe / y dom rynƺtunków ſwoich / y wƺyſtko / co śię znájdowáło w ſkárbách jego ; nie było nic cżegoby im nie pokázáł Ezechyjáƺ w domu ſwym / y we wƺyſtkiem páńſtwie ſwojim.
Przetoż przyƺedł prorok Izájáƺ do królá Ezechyjáƺá / y rzekł mu : Coć powiedzieli ći mężowie / á ſkąd przyƺli do ćiebie? Y odpowiedziáł Ezechyjáƺ : Z ziemi dálekiej przyƺli / z Bábilonu.
Y rzekł : Cóż widzieli w domu twoim? Odpowiedziáł Ezechyjáƺ : Wƺyſtko / cokolwiek jeſt w domu moim / widzieli : nie było nic / cżegobym im nie pokázáł w ſkárbách moich.
Ale Izájáƺ rzekł do Ezechyjáƺá : Słucháj ſłowá Páńſkiego.
Oto przyjdą dni / w które zábiorą wƺyſtko do Bábilonu / cokolwiek jeſt w domu twoim / y cokolwiek ſchowáli ojcowie twoi áż do dniá tego ; nie zoſtánieć nic / mówi Pán.
Ale y ſyny twoje / którzy wynijdą z ćiebie / y które ſpłodziƺ / pobiorą / y będą komornikámi ná dworze królá Bábilońſkiego.
Tedy rzekł Ezechyjáƺ do Izájáƺá : Dobre jeſt ſłowo Páńſkie / któreś mówił. Nádto rzekł : Záiſte dobre / jeźli tylko pokój y práwdá będzie zá dni moich.
Ale inne ſpráwy Ezechyjáƺowe / y wƺyſtká moc jego / y jáko ucżynił ſádzáwkę / y rury / któremi przywiódł wodę do miáſtá / to zápiſáno w kronikách o królách Judzkich.
Y záſnął Ezechyjáƺ z ojcámi ſwymi / á królowáł Mánáſeſ / ſyn jego / miáſto niego.
