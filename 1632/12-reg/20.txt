W one dni záchorował Ezechyjáƺ áż ná śmierć / y przyƺedł do niego Izájáƺ prorok / ſyn Amoſowy / y rzekł mu : Ták mówi Pán : Rozpráw dom twój ; bo umrzeƺ / á nie będźieƺ żył.
Tedy obróćił Ezechyjáƺ twárz ſwoję do śćiány / y modlił śię Pánu / mówiąc :
Proƺę / o Pánie! wſpomnij teraz / żem chodźił przed tobą w prwádźie / y w ſercu cáłem / cżyniąc to / co dobrego jeſt w ocżách twojich. Y płákał Ezechyjáƺ płácżem wielkiem.
Ale jeƺcże Izájáƺ nie wyƺedł był do pół śieni / gdy śię ſłowo Páńſkie ſtáło do niego / mówiąc :
Wróć śię / á mów do Ezechyjáƺá / wodzá ludu mego : Ták mówi Pán / Bóg Dawidá / ojcá twego : Wyſłucháłem modlitwę twoję / á widźiałem łzy twoje ; oto Ja uzdráwiám ćię / dniá trzećiego wnijdźieƺ do domu Páńſkiego ;
Y przydám do dni twojich piętnaśćie lat / á z ręki królá Aſſyryjſkiego wyrwę ćiebie / y to miáſto ; y bronić będę tego miáſtá dla śiebie / y dla Dawidá / ſługi mego.
Przytem rzekł Izájáƺ : Przynieśćie bryłę fig ſuchych. Którą przynióſƺy włożyli ná wrzód / y zgoił śię.
Y rzekł Ezechyjáƺ do Izájáƺá : Jáki znák tego / że mię uzdrowi Pán / á iż pójdę dniá trzećiego do domu Páńſkiego?
Odpowiedźiał Izájáƺ : Toć będźie znákiem od Páná / iż ucżyni Pán tę rzecż / którąć obiecał. Chceƺże / żeby ćień poſtąpił ná dźieśięć ſtopni / álbo żeby śię ná wſtecż wróćił ná dźieśięć ſtopni?
Y rzekł Ezechyjáƺ : Snádniej może ćień poſtąpić ná dół ná dźieśięć ſtopni / tego nie chcę ; ále niech śię wróći ćień ná wſtecż ná dźieśięć ſtopni.
Tedy wołał Izájáƺ prorok do Páná ; y náwróćił ćień po onych ſtopniách / któremi był poſtąpił ná zegárze ſłonecżnym Acházowym / ná wſtecż ná dźieśięć ſtopni.
Onegoż cżáſu poſłał Berodách Báládán / ſyn Báládánowy / król Bábilońſki / liſt y dáry do Ezechyjáƺá ; bo záſłyƺał / że był zániemógł Ezechyjáƺ.
Y wyſłuchał ich Ezechyjáƺ y okázał im wƺyſtkie ſkárbnice klejnotów ſwojich / ſrebro / y złoto / y rzecży wonne / y olejki nájwyborniejƺe / y dom rynƺtunków ſwojich / y wƺyſtko / co śię znájdowáło w ſkárbách jego ; nie było nic cżegoby im nie pokázał Ezechyjáƺ w domu ſwym / y we wƺyſtkim páńſtwie ſwojim.
Przetoż przyƺedł prorok Izájáƺ do królá Ezechyjáƺá / y rzekł mu : Coć powiedźieli ći mężowie / á ſkąd przyƺli do ćiebie? Y odpowiedźiał Ezechyjáƺ : Z źiemi dálekiej przyƺli / z Bábilonu.
Y rzekł : Cóż widźieli w domu twoim? Odpowiedźiał Ezechyjáƺ : Wƺyſtko / cokolwiek jeſt w domu moim / widźieli : nie było nic / cżegobym im nie pokázał w ſkárbách mojich.
Ale Izájáƺ rzekł do Ezechyjáƺá : Słucháj ſłowá Páńſkiego.
Oto przyjdą dni / w które zábiorą wƺyſtko do Bábilonu / cokolwiek jeſt w domu twoim / y cokolwiek ſchowáli ojcowie twoi áż do dniá tego ; nie zoſtánieć nic / mówi Pán.
Ale y ſyny twoje / którzy wynijdą z ćiebie / y które ſpłodźiƺ / pobiorą / y będą komornikámi ná dworze królá Bábilońſkiego.
Tedy rzekł Ezechyjáƺ do Izájáƺá : Dobre jeſt ſłowo Páńſkie / któreś mówił. Nádto rzekł : Zájiſte dobre / jeſli tylko pokój y práwdá będźie zá dni mojich.
Ale inne ſpráwy Ezechyjáƺowe / y wƺyſtká moc jego / y jáko ucżynił ſádzáwkę / y rury / któremi przywiódł wodę do miáſtá / to zápiſáno w kronikách o królách Judzkich.
Y záſnął Ezechyjáƺ z ojcámi ſwymi / á królował Mánáſes / ſyn jego / miáſto niego.
