Y rzekli ſynowie proroccy do Elizeuƺá : Oto miejſce / ná którem mieƺkámy przed tobą / ćiáſne jeſt dla nas.
Niech idźiemy proƺę áż do Jordánu / á weźmiemy ſtámtąd káżdy po jednem drzewie / y zbudujemy tám ſobie miejſce ku mieƺkániu. Którym on rzekł : Idźćie.
Y rzekł jeden z nich : Pójdź proƺę y ty z ſługámi twoimi. A on rzekł : Y ja pójdę ; y ƺedł z nimi.
A przyƺedłƺy do Jordánu / rąbáli drzewo.
Y ſtáło śię / gdy jeden z nich obáláł drzewo / że mu śiekierá wpádłá w wodę ; y záwołáł / mówiąc : Ach / ách / pánie mój! y táć byłá pożycżoná.
Rzekł tedy mąż Boży : Gdźież upádłá? y ukázáł mu miejſce. A on ućiąwƺy drewno / wrzućił tám / y ſpráwił / że wypłynęłá oná śiekierá.
Y rzekł : Weźmij ją ſobie ; który śćiągnąwƺy rękę ſwą / wźiął ją.
A gdy król Syryjſki wálcżył z Izráelem / y nárádzáł śię z ſługámi ſwoimi / mówiąc : Ná tem á ná tem miejſcu położy śię wojſko moje :
Tedy poſłáł mąż Boży do królá Izráelſkiego / mówiąc : Strzeż śię / ábyś nie przechodźił przez ono miejſce ; bo tám Syryjcżycy ſą ná záſádzce.
Przetoż poſłáł król Izráelſki ná ono miejſce / o którem mu był powiedźiáł mąż Boży / y przeſtrzegł go / áby śię go chronił / nie ráz áni dwá.
A ták zátrwożyło śię ſerce królá Syryjſkiego dla tego. Przetoż zwołáwƺy ſług ſwoich / rzekł do nich : Cżemuż mi nie powiećie / kto wżdy z nas donośi to królowi Izráelſkiemu?
Y rzekł jeden z ſług jego : Nie ták / królu / pánie mój ; ále Elizeuƺ prorok / który jeſt w Izráelu / oznájmuje królowi Izráelſkiemu ſłowá / które mówiƺ w tájemnym pokoju twoim.
A on rzekł : Idźćie / á dowiedzćie śię / gdźie jeſt / ábym poſłáł y pojmáł go. Y powiedźiáno mu / mówiąc : Oto jeſt w Dotánie.
Przetoż poſłáł tám konie y wozy z wielkiem wojſkiem / którzy przyćiągnąwƺy w nocy / oblegli miáſto.
Tedy wſtáwƺy ráno ſługá mężá Bożego / wyƺedł / á oto wojſko otocżyło miáſto / y konie / y wozy. Y rzekł ſługá jego do niego : Ach pánie mój! Cóż mámy cżynić?
A on odpowiedźiáł : Nie bój śię ; bo więcey ich z námi / niż z nimi.
Modlił śię tedy Elizeuƺ / y rzekł : O Pánie / otwórz proƺę ocży jego / żeby widźiáł. Y otworzył Pán ocży ſługi onego / y ujrzáł / á oto górá pełná koni / y wozy ogniſte około Elizeuƺá.
A gdy Syryjcżycy ƺli do niego / modlił śię Elizeuƺ Pánu / mówiąc : Proƺę / záráź ten lud ślepotą. Y záráźił je Pán ślepotą według ſłowá Elizeuƺowego.
Wtem rzekł do nich Elizeuƺ : Nie táć to drogá / áni to miáſto. Pójdźćie zá mną / á záwiodę was do mężá / którego ƺukáćie. Y przywiódł je do Sámáryi /
A gdy weƺli do Sámáryi / rzekł Elizeuƺ : O Pánie / otwórz ocży tych / áby przejrzeli. Y otworzył Pán ocży ich / y widźieli / że byli w pośród Sámáryi.
Y rzekł król Izráelſki do Elizeuƺá / gdy je ujrzáł ;
Mámże je pobić / ojcże mój? Ale on rzekł : Nie bij. Azáżeś je wźiął przez miecż twój / álbo przez łuk twój / żebyś je miał pobić? Połóż chleb y wodę przed nie / áby jedli y pili / y wróćili śię do páná ſwego.
A ták przygotowáł dla nich doſtátek wielki / y jedli y pili ; y puśćił je / y odeƺli do páná ſwego. Y nie wáżyły śię więcey wojſká Syryjſkie wpádáć do źiemi Izráelſkiej.
Stáło śię potem / że zebráł Benádád / król Syryjſki / wƺyſtkie wojſká ſwe / á przyćiągnął y obległ Sámáryję.
Przetoż był głód wielki w Sámáryi ; ábowiem ją było oblężono / ták / iż głowę oślą ſprzedáwáno zá ośmdźieśiąt ſrebrników / á cżwártą cżęść miáry gnoju gołębiego zá pięć ſrebrników.
Y przydáło śię / gdy król Izráelſki przechádzáł śię po murze / że jedná niewiáſtá záwołáłá náń mówiąc : Rátuj mię królu / pánie mój!
Który rzekł : Nie rátujeli ćię Pán / ſkądże ja ćiebie porátuję? izáli z gumná / cżyli z práſy?
Nádto rzekł jey król : Cóż ći? A oná rzekłá : Tá niewiáſtá rzekłá do mnie : Dáj ſyná twego / żebyśmy go zjádły dźiśiáj / á jutro zjemy ſyná mego.
Y uwárzyłyśmy ſyná mego / y zjádłyśmy go. Potem rzekłám jey dniá drugiego : Dáj ſyná twego / ábyśmy go zjádły ; ále oná ſkryłá ſyná ſwego.
A gdy król uſłyƺáł ſłowá oney niewiáſty / rozdárł odźienie ſwe ; á gdy śię przechodźił po murze / widźiáł lud / że wór był ná ćiele jego od ſpodku.
Tedy rzekł król : To niecháj mi ucżyni Bóg / y to przycżyni / jeſli śię głowá Elizeuƺá / ſyná Sáfátowego / ná nim dźiśiáj oſtoi.
( A Elizeuƺ śiedźiáł w domu ſwoim / y ſtárcy śiedźieli z nim. ) Y poſłáł mężá z tych / którzy przed nim ſtáli ; á pierwej / niż on poſeł przyƺedł do niego / rzekł był do ſtárƺych : Nie wiećież / iż poſłáł ſyn tego mężobójcę / áby śćięto głowę moję? Pátrzćież / gdy przyjdźie ten poſeł / zámknijćież drzwi / á záhámujćie go przede drzwiámi ; boć tenten nóg páná jego jeſt już zá nim.
A gdy to jeƺcże mówił z nimi / oto poſeł przychodźił ku niemu / y rzekł : Oto to złe jeſt od Páná ; cżegoż mam więcey ocżekiwáć od Páná?
