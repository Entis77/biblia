Ośm lát było Jozyjáƺowi / gdy pocżął królowáć / á trzydźieśći jeden lát królowáł w Jeruzálemie ; á imię mátki jego było Jedydá / córká Adájá z Beſekátu.
Y cżynił / co było dobrego przed ocżymá Páńſkimi / chodząc wƺyſtkiemi drogámi Dawidá / ojcá ſwego / á nie uchyláł śię áni ná práwo áni ná lewo.
A ośmnáſtego roku królá Jozyjáƺá poſłáł król Sáfáná / ſyná Azálijáƺowego / ſyná Meſulámá / piſárzá / do domu Páńſkiego / mówiąc :
Idź do Helkijáƺá / kápłáná nájwyżƺego / áby zebráł pieniądze / które wnoƺono do domu Páńſkiego / które wybieráli ſtróżowie progu od ludu ;
A niech je dáwáją w ręce rzemieślników przełożonych nád robotą domu Páńſkiego / áby je dáwáli robotnikom / którzy robili w domu Páńſkim / nápráwiájąc ſkázę domu ;
To jeſt / budownikom y ćieślom / y murárzom / y ná zkupowánie drzewá / y kámieniá ćioſánego ku nápráwie domu.
Wƺákże niech nie cżynią licżby z pieniędzy / które dáwáją do rąk ich ; bo oni wiernie nimi ƺáfowáć będą.
Y rzekł Helkijáƺ / kápłán nájwyżƺy / do Sáfáná piſárzá : Kśięgi zakonu ználázłem w domu Páńſkim. Y dáł Helkijáƺ one kśięgi Sáfánowi / y cżytáł je ( Sáfán ).
Przyƺedłƺy tedy Sáfán piſárz do królá / odnióſł to królowi / y rzekł : Zebráli ſłudzy twoi pieniądze / które śię ználázły w domu Páńſkim / y oddáli je w ręce rzemieślników przełożonych nád robotą w domu Páńſkim.
Oznájmił też Sáfán piſárz królowi / mówiąc : Dáł mi kśięgę Helkijáƺ kápłán ; y cżytáł ją Sáfán przed królem.
A gdy uſłyƺáł król ſłowá kśiąg zakonu / rozdárł ƺáty ſwe.
Y rozkázáł król Helkijáƺowi kápłánowi / y Ahykámowi / ſynowi Sáfánowemu / y Achborowi / ſynowi Micheáƺowemu / y Sáfánowi piſárzowi / y Aſájáƺowi / ſłudze ſwemu / mówiąc :
Idźćie / porádźćie śię Páná o mię / y o lud / y o wƺyſtkiego Judę z ſtrony ſłów tych kśiąg / które ſą ználeźione ; bo wielki jeſt gniew Páńſki / który śię zápálił przećiwko nám / przeto iż nie poſłucháli ojcowie náśi ſłów tych kśiąg / żeby cżynili według wƺyſtkiego / co nám jeſt nápiſáne.
A ták poƺedł Helkijáƺ kápłán / y Ahykám / y Achbor / y Sáfán / y Azájáƺ / do Huldy prorokini / żony Sellumá / ſyná Tekui / ſyná Arááƺowego / który był ſtróżem ƺát ; á oná mieƺkáłá w Jeruzálemie ná drugiej ſtronie miáſtá ; y mówili z nią.
Która rzekłá do nich : Ták mówi Pán / Bóg Izráelſki : Powiedzćie mężowi / który was poſłáł do mnie :
Ták mówi Pán : Oto Ja przywiodę złe ná to miejſce y ná obywáteli jego według wƺyſtkich ſłów kśiąg tych / które cżytáł król Judzki ;
Przeto / że mię opuśćili / y kádźili bogom cudzym / áby mię dráźnili wƺyſtkiemi ſpráwámi rąk ſwoich / dla cżego rozpáliłá śię popędliwość mojá przećiwko miejſcu temu / y nie będźie ugáƺoná.
A królowi Judzkiemu / który was poſłáł o rádę do Páná / ták powiedzćie : Ták mówi Pán / Bóg Izráelſki / o ſłowách / któreś ſłyƺáł :
Poniewáż zmiękcżone jeſt ſerce twoje / á upokorzyłeś śię przed oblicżem Páńſkim / ſłyƺąc / com powiedźiáł przećiwko temu miejſcu / y przećiwko obywátelom jego / iż má przyjść w ſpuſtoƺenie y w przeklęſtwo ; y rozdárłeś ƺáty twe / á płákáłeś przedemną / y Jám ćię też wyſłucháł / mówi Pán.
Przetoż oto Ja ćię zbiorę do ojców twoich / á będźieƺ zebrány do grobu twego w pokoju / áby nie oglądáły ocży twoje wƺyſtkiego złego / które Ja przywiodę ná to miejſce. Y odnieśiono tę rzecż królowi.
