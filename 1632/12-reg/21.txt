We dwunáſtym roku był Mánáſes / gdy królowáć pocżął / á pięćdźieśiąt y pięć lat królował w Jeruzálemie ; á imię mátki jego było Hádſybá.
Y cżynił złe przed ocżymá Páńſkimi według obrzydłośći tych narodów / które wygnał Pán przed oblicżem ſynów Izráelſkich.
Abowiem znowu pobudował wyżyny / które był poburzył Ezechyjáƺ / oćiec jego / y wyſtáwił ołtárze Báálowi / y náſádźił gáj jáko był ucżynił Acháb / król Izráelſki / y kłániał śię wƺyſtkiemu wojſku niebieſkiemu / y ſłużył mu.
Pobudował też ołtárze w domu Páńſkim / o którym powiedźiał był Pán : W Jeruzálemie położę imię moje.
Nádto nábudował ołtárzy wƺyſtkiemu wojſku niebieſkiemu w obu śieniách domu Páńſkiego.
Syná tákże ſwego przewiódł przez ogień / y przeſtrzegał cżáſów / y báwił śię wieƺcżbą / y uſtáwił cżárnokśiężniki / y guślárze / á bárzo wiele złego cżynił przed ocżymá Páńſkimi / dráźniąc go.
Poſtáwił tákże báłwáná gájowego / którego był ucżynił w domu / o którym był rzekł Pán do Dawidá y do Sálomoná / ſyná jego : W domu tym y w Jeruzálemie / którem obrał ze wƺyſtkiego pokolenia Izráelſkiego / położę imię moje ná wieki :
A więcey śię nie dopuƺcżę ruƺyć nodze Izráelá z źiemi / którąm dał ojcom ich / by jedno ſkutecżnie ſtrzegli wƺyſtkiego / com im rozkázał / y wƺyſtkiego zakonu / który im przykázał ſługá mój Mojzeƺ.
Ale nie ſłucháli ; bo je zwiódł Mánáſes / ták iż śię gorzej ſpráwowáli niż narody / które wygłádźił Pán przed oblicżem ſynów Izráelſkich.
Acżkolwiek powiedźiał był Pán przez ſługi ſwoje proroki / mówiąc :
Przeto / że cżynił Mánáſes / król Judzki / te obrzydliwośći / cżyniąc gorƺe rzecży nád one wƺyſtkie / które cżynili Amorejcżycy / którzy byli przed nim / á że przywiódł w grzech y Judę przez brzydkie báłwány ſwoje ;
Przetoż ták mówi Pán / Bóg Izráelſki : Oto Ja przywiodę złe ná Jeruzálem y ná Judę / ták iż káżdemu / co to uſłyƺy / zábrzmi w obu uƺách jego.
Bo rozćiągnę nád Jeruzálem ƺnur Sámáryjſki / y wágę domu Achábowego / á wytrę Jeruzálem / jáko kto wyćierá miſę / á wytárƺy przewrácá ją dnem ku górze.
Y opuƺcżę oſtátki dźiedźictwá mego / á podám je w rękę nieprzyjáćiół ich ; y będą ná łup / y ná rozƺárpánie wƺyſtkim nieprzyjáćiołom ſwoim.
Przeto / iż śię dopuƺcżáli złego przed ocżymá memi / á dráźnili mię ode dniá / którego wyƺli ojcowie ich z Egiptu / áż do dźiśiejƺego dniá ;
Nádto y krwi niewinney Mánáſes wylał bárzo wiele / ták iż nią nápełnił Jeruzálem od końcá do końcá / oprócż grzechu ſwego / przez który przywiódł do grzechu Judę / áby cżynił złe przed ocżymá Páńſkimi.
A inne ſpráwy Mánáſeſowe / y wƺyſtko co cżynił / y grzech jego / którego śię dopuśćił / to zápiſáno w kronikách o królách Judzkich.
Y záſnął Mánáſes z ojcámi ſwymi / y pogrzebiony jeſt w ogrodźie domu ſwego / w ogrodźie Ozy ; á królował Amon / ſyn jego / miáſto niego.
Dwádźieśćiá y dwá látá miał Amon / gdy królowáć pocżął / á dwá látá królował w Jeruzálemie. A imię mátki jego było Meſállemet / córká Háruſá z Játeby.
Y cżynił złe przed ocżymá Páńſkimi / jáko cżynił Mánáſes / oćiec jego /
A chodźił wƺyſtkiemi drogámi / któremi chodźił oćiec jego / ſłużąc brzydkim báłwánom / którym ſłużył oćiec jego / y kłániał śię im ;
Y opuśćił Páná / Bogá ojców ſwojich á nie chodźił drogą Páńſką.
Ale śię ſprzyśięgli ſłudzy Amonowi przećiwko niemu / y zábili królá w domu jego.
Tedy pobił lud oney źiemi wƺyſtkie / którzy śię byli ſprzyśięgli przećiwko królowi Amonowi ; y poſtánowił lud oney źiemi królem Jozyjáƺá / ſyná jego / miáſto niego.
Ale inne ſpráwy Amonowe / które cżynił / opiſáne ſą w kronikách o królách Judzkich.
Y pochowano go w grobie jego w ogrodźie Ozy ; á królował Jozyjáƺ / ſyn jego / miáſto niego.
