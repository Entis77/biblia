We dwunáſtym roku był Mánáſeſ / gdy królowáć pocżął / á pięćdźieśiąt y pięć lát królowáł w Jeruzálemie ; á imię mátki jego było Hádſybá.
Y cżynił złe przed ocżymá Páńſkimi według obrzydłośći tych národów / które wygnáł Pán przed oblicżem ſynów Izráelſkich.
Abowiem znowu pobudowáł wyżyny / które był poburzył Ezechyjáƺ / ojćiec jego / y wyſtáwił ołtárze Báálowi / y náſádźił gáj jáko był ucżynił Acháb / król Izráelſki / y kłániáł śię wƺyſtkiemu wojſku niebieſkiemu / y ſłużył mu.
Pobudowáł też ołtárze w domu Páńſkim / o którym powiedźiáł był Pán : W Jeruzálemie położę imię moje.
Nádto nábudowáł ołtárzy wƺyſtkiemu wojſku niebieſkiemu w obu śieniách domu Páńſkiego.
Syná tákże ſwego przewiódł przez ogień / y przeſtrzegáł cżáſów / y báwił śię wieƺcżbą / y uſtáwił cżárnokśiężniki / y guślárze / á bárdzo wiele złego cżynił przed ocżymá Páńſkimi / dráźniąc go.
Poſtáwił tákże báłwáná gájowego / którego był ucżynił w domu / o którym był rzekł Pán do Dawidá y do Sálomoná / ſyná jego : W domu tym y w Jeruzálemie / którem obráł ze wƺyſtkiego pokoleniá Izráelſkiego / położę imię moje ná wieki :
A więcej śię nie dopuƺcżę ruƺyć nodze Izráelá z źiemi / którąm dáł ojcom ich / by jedno ſkutecżnie ſtrzegli wƺyſtkiego / com im rozkázáł / y wƺyſtkiego zákonu / który im przykázáł ſługá mój Mojżeƺ.
Ale nie ſłucháli ; bo je zwiódł Mánáſeſ / ták iż śię gorzej ſpráwowáli niż národy / które wygłádźił Pán przed oblicżem ſynów Izráelſkich.
Acżkolwiek powiedźiáł był Pán przez ſługi ſwoje proroki / mówiąc :
Przeto / że cżynił Mánáſeſ / król Judzki / te obrzydliwośći / cżyniąc gorƺe rzecży nád one wƺyſtkie / które cżynili Amorejcżycy / którzy byli przed nim / á że przywiódł w grzech y Judę przez brzydkie báłwány ſwoje ;
Przetoż ták mówi Pán / Bóg Izráelſki : Oto Já przywiodę złe ná Jeruzálem y ná Judę / ták iż káżdemu / co to uſłyƺy / zábrzmi w obu uƺách jego.
Bo rozćiągnę nád Jeruzálem ƺnur Sámáryjſki / y wágę domu Achábowego / á wytrę Jeruzálem / jáko kto wyćierá miſę / á wytárłƺy przewrácá ją dnem ku górze.
Y opuƺcżę oſtátki dźiedźictwá mego / á podám je w rękę nieprzyjáćiół ich ; y będą ná łup / y ná rozƺárpánie wƺyſtkim nieprzyjáćiołom ſwoim.
Przeto / iż śię dopuƺcżáli złego przed ocżymá memi / á dráźnili mię ode dniá / którego wyƺli ojcowie ich z Egiptu / áż do dźiśiejƺego dniá ;
Nádto y krwi niewinnej Mánáſeſ wyláł bárdzo wiele / ták iż nią nápełnił Jeruzálem od końcá do końcá / oprócż grzechu ſwego / przez który przywiódł do grzechu Judę / áby cżynił złe przed ocżymá Páńſkimi.
A inne ſpráwy Mánáſeſowe / y wƺyſtko co cżynił / y grzech jego / którego śię dopuśćił / to zápiſáno w kronikách o królách Judzkich.
Y záſnął Mánáſeſ z ojcámi ſwymi / y pogrzebiony jeſt w ogrodźie domu ſwego / w ogrodźie Ozy ; á królowáł Amon / ſyn jego / miáſto niego.
Dwádźieśćiá y dwá látá miáł Amon / gdy królowáć pocżął / á dwá látá królowáł w Jeruzálemie. A imię mátki jego było Meſállemet / córká Háruſá z Játeby.
Y cżynił złe przed ocżymá Páńſkimi / jáko cżynił Mánáſeſ / ojćiec jego /
A chodźił wƺyſtkiemi drogámi / któremi chodźił ojćiec jego / ſłużąc brzydkim báłwánom / którym ſłużył ojćiec jego / y kłániáł śię im ;
Y opuśćił Páná / Bogá ojców ſwoich á nie chodźił drogą Páńſką.
Ale śię ſprzyśięgli ſłudzy Amonowi przećiwko niemu / y zábili królá w domu jego.
Tedy pobił lud onej źiemi wƺyſtkie / którzy śię byli ſprzyśięgli przećiwko królowi Amonowi ; y poſtánowił lud onej źiemi królem Jozyjáƺá / ſyná jego / miáſto niego.
Ale inne ſpráwy Amonowe / które cżynił / opiſáne ſą w kronikách o królách Judzkich.
Y pochowáno go w grobie jego w ogrodźie Ozy ; á królowáł Jozyjáƺ / ſyn jego / miáſto niego.
