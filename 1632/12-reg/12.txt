Roku śiódmego Jehu pocżął królowáć Joáz / á cżterdźieśći lat królował w Jeruzálemie ; imię mátki jego było Sebijá z Beerſáby.
Y cżynił Joáz / co dobrego było w ocżách Páńſkich / po wƺyſtkie dni ſwoje / których go ucżył Jojádá kápłán.
Wƺákże wyżyny nie były znieśione ; jeƺcże lud ofiárował y kádźił ná onych wyżynách.
Y rzekł Joáz do kápłanów : Wƺyſtkie pieniądze poświęcone / które przychodzą do domu Páńſkiego / pieniądze tych / którzy idą w licżbę / pieniądze káżdego z oſobná według ƺácunku jego / y wƺyſtkie pieniądze / które ktokolwiek dobrowolnie znośi do domu Páńſkiego /
Te wezmą do śiebie kápłáni / káżdy od znájomego ſwego ; á oni nápráwią ſkázę domu Páńſkiego wƺędy / gdźieby śię ználázłá ſkázá.
Y ſtáło śię roku dwudźieſtego y trzećiego królá Joázá / gdy jeƺcże nie popráwili byli kápłáni ſkázy domu /
Że wezwał król Joáz Jojády kápłáná / y innych kápłanów / y mówił do nich : Przecż nie opráwujećie ſkázy domu? Przetoż teraz nie bierzćie pieniędzy od znájomych wáƺych / ále one ná popráwę ſkázy domu oddáwájćie.
Y zezwolili ná to kápłáni ; żeby nie bráli pieniędzy od ludu / y żeby nie popráwiáli ſkázy domu.
Przetoż wźiąwƺy Jojádá kápłán ſkrzynię jednę / ucżynił dźiurę w wieku jey / á poſtáwił ją przy ołtárzu po práwej ſtronie / kędy wchodzono do domu Páńſkiego. Y kłádli w nię kápłáni / którzy ſtrzegli progu / wƺyſtkie pieniądze / które wnoƺono do domu Páńſkiego.
A gdy widźieli / że było wiele pieniędzy w ſkrzyni / tedy przychodźił piſárz królewſki / y kápłán nájwyżƺy / którzy zlicżywƺy chowáli one pieniądze / które śię znájdowáły w domu Páńſkim.
Y dáwáli pieniądze gotowe w ręce rzemieślników / przełożonych nád robotą domu Páńſkiego ; á ći je wydáwáli ná ćieśle / y ná robotniki / którzy popráwiáli domu Páńſkiego ;
Y ná murárze / y ná te / co ćioſáli kámienie / y ná kupowánie drzewá / y ćioſánego kámieniá / ku popráwie ſkázy domu Páńſkiego / y ná wƺyſtek nákłád ku popráwie domu onego.
Wƺákże nie ſpráwowano do domu Páńſkiego kubków ſrebrnych / nacżynia do muzyki / miednic / y trąb / żádnego nacżynia złotego / y nacżynia ſrebrnego / z pieniędzy / które przynoƺono do domu Páńſkiego ;
Ale rzemieślnikom przełożonym nád robotą dáwáli je / y popráwiáli zá nie domu Páńſkiego.
A nie ſłucháno licżby tych ludźi / którym dáwano pieniądze w ręce ich / áby wydáwáli rzemieślnikom / ponieważ to oni wiernie odpráwowáli.
Ale pieniądze zá wyſtępek / y pieniądze zá grzechy / nie były wnoƺone do domu Páńſkiego ; kápłánom śię doſtáwáły.
Tedy wyćiągnął Házáel / król Syryjſki / á wálcżył przećiwko Giet / y wźiął je. Potem obróćił Házáel twárz ſwoję / áby ćiągnął przećiwko Jeruzálemowi.
Przetoż wźiął Joáz / król Judzki / wƺyſtkie rzecży poświęcone / które byli poświęćili Jozáfát y Jorám / y Ochozyjáƺ / ojcowie jego / królowie Judzcy / y to / co był ſám poświęćił / y wƺyſtko złoto / które śię ználázło w ſkárbách domu Páńſkiego / y domu królewſkiego / á poſłał to do Házáelá / królá Syryjſkiego / y odćiągnął od Jeruzálemu.
Ale inƺe ſpráwy Joázowe / y wƺyſtko co cżynił / ázaż to nie jeſt nápiſáne w kronikách o królách Judzkich?
Potem powſtáwƺy ſłudzy jego ſprzyśięgli śię miedzy ſobą y zábili Joázá w Betmello / którędy chodzą do Selli :
To jeſt / zábili go Joſáchár / ſyn Semáátowy / y Jozábád / ſyn Sommerowy ; ći ſłudzy jego zábili go / y umárł. A pochowáli go z ojcámi jego w mieśćie Dawidowem / y królował Amázyjáƺ / ſyn jego / miáſto niego.
