Roku śiódmego Jehu pocżął królowáć Joáz / á cżterdzieśći lát królowáł w Jeruzálemie ; imię mátki jego było Sebijá z Beerſáby.
Y cżynił Joáz / co dobrego było w ocżách Páńſkich / po wƺyſtkie dni ſwoje / których go ucżył Jojádá kápłán.
Wƺákże wyżyny nie były znieśione ; jeƺcże lud ofiárowáł y kádził ná onych wyżynách.
Y rzekł Joáz do kápłánów : Wƺyſtkie pieniądze poświęcone / które przychodzą do domu Páńſkiego / pieniądze tych / którzy idą w licżbę / pieniądze káżdego z oſobná według ƺácunku jego / y wƺyſtkie pieniądze / które ktokolwiek dobrowolnie znośi do domu Páńſkiego /
Te wezmą do śiebie kápłáni / káżdy od znájomego ſwego ; á oni nápráwią ſkázę domu Páńſkiego wƺędy / gdzieby śię ználázłá ſkázá.
Y ſtáło śię roku dwudzieſtego y trzećiego królá Joázá / gdy jeƺcże nie popráwili byli kápłáni ſkázy domu /
Że wezwáł król Joáz Jojády kápłáná / y innych kápłánów / y mówił do nich : Przecż nie opráwujećie ſkázy domu? Przetoż teráz nie bierzćie pieniędzy od znájomych wáƺych / ále one ná popráwę ſkázy domu oddáwájćie.
Y zezwolili ná to kápłáni ; żeby nie bráli pieniędzy od ludu / y żeby nie popráwiáli ſkázy domu.
Przetoż wziąwƺy Jojádá kápłán ſkrzynię jednę / ucżynił dziurę w wieku jej / á poſtáwił ją przy ołtárzu po práwej ſtronie / kędy wchodzono do domu Páńſkiego. Y kłádli w nię kápłáni / którzy ſtrzegli progu / wƺyſtkie pieniądze / które wnoƺono do domu Páńſkiego.
A gdy widzieli / że było wiele pieniędzy w ſkrzyni / tedy przychodził piſárz królewſki / y kápłán nájwyżƺy / którzy zlicżywƺy chowáli one pieniądze / które śię znájdowáły w domu Páńſkim.
Y dáwáli pieniądze gotowe w ręce rzemieślników / przełożonych nád robotą domu Páńſkiego ; á ći je wydáwáli ná ćieśle / y ná robotniki / którzy popráwiáli domu Páńſkiego ;
Y ná murárze / y ná te / co ćioſáli kámienie / y ná kupowánie drzewá / y ćioſánego kámieniá / ku popráwie ſkázy domu Páńſkiego / y ná wƺyſtek nákłád ku popráwie domu onego.
Wƺákże nie ſpráwowáno do domu Páńſkiego kubków ſrebrnych / nácżyniá do muzyki / miednic / y trąb / żádnego nácżyniá złotego / y nácżyniá ſrebrnego / z pieniędzy / które przynoƺono do domu Páńſkiego ;
Ale rzemieślnikom przełożonym nád robotą dáwáli je / y popráwiáli zá nie domu Páńſkiego.
A nie ſłucháno licżby tych ludzi / którym dáwáno pieniądze w ręce ich / áby wydáwáli rzemieślnikom / poniewáż to oni wiernie odpráwowáli.
Ale pieniądze zá wyſtępek / y pieniądze zá grzechy / nie były wnoƺone do domu Páńſkiego ; kápłánom śię doſtáwáły.
Tedy wyćiągnął Házáel / król Syryjſki / á wálcżył przećiwko Giet / y wziął je. Potem obróćił Házáel twárz ſwoję / áby ćiągnął przećiwko Jeruzálemowi.
Przetoż wziął Joáz / król Judzki / wƺyſtkie rzecży poświęcone / które byli poświęćili Jozáfát y Jorám / y Ochozyjáƺ / ojcowie jego / królowie Judzcy / y to / co był ſám poświęćił / y wƺyſtko złoto / które śię ználázło w ſkárbách domu Páńſkiego / y domu królewſkiego / á poſłáł to do Házáelá / królá Syryjſkiego / y odćiągnął od Jeruzálemu.
Ale inƺe ſpráwy Joázowe / y wƺyſtko co cżynił / ázáż to nie jeſt nápiſáne w kronikách o królách Judzkich?
Potem powſtáwƺy ſłudzy jego ſprzyśięgli śię między ſobą y zábili Joázá w Betmello / którędy chodzą do Selli :
To jeſt / zábili go Joſáchár / ſyn Semáátowy / y Jozábád / ſyn Sommerowy ; ći ſłudzy jego zábili go / y umárł. A pochowáli go z ojcámi jego w mieśćie Dawidowem / y królowáł Amázyjáƺ / ſyn jego / miáſto niego.
