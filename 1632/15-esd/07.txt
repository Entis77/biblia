A po tych ſpráwách zá królowániá Artákſerkſeſá / królá Perſkiego / Ezdráƺ / ſyn Sárájáƺá / ſyná Azáryjáƺowego / ſyná Helkijáƺowego /
Syná Sállumowego / ſyná Sádokowego / ſyná Achitobowego /
Syná Amáryjáƺowego / ſyná Azáryjáƺowego / ſyná Merájotowego /
Syná Zeráhyjáƺowego / ſyná Uzego / ſyná Bukkiego /
Syná Abiſujego / ſyná Fineeſowego / ſyná Eleázárowego / ſyná Aároná kápłáná nájwyżƺego :
Ten Ezdráƺ wyƺedł z Bábilonu / á był cżłowiekiem biegłym w zákonie Mojżeƺowym / który był dáł Pán / Bóg Izráelſki ; á pozwolił mu był król według ręki Páná Bogá jego nád nim / ná wƺyſtkę prośbę jego.
( Wyƺli też niektórzy z ſynów Izráelſkich / y z kápłánów / y z Lewitów / y śpiewáków / y z odźwiernych / y z Netynejcżyków / do Jeruzálemu roku śiódmego Artákſerkſeſá królá. )
Y przyƺedł do Jeruzálemu mieśiącá piątego ; tenći był rok śiódmy królá Dáryjuƺá.
Abowiem w pierwƺy dzień mieśiącá pierwƺego wyƺedł z Bábilonu / á dniá pierwƺego mieśiącá piątego przyƺedł do Jeruzálemu według łáſkáwego wſpomożeniá Bogá ſwego.
Bo Ezdráƺ przygotowáł był ſerce ſwe / áby ƺukáł zákonu Páńſkiego / y áby cżynił / y náucżáł w Izráelu uſtáw y ſądów.
A tenći jeſt przepiſ liſtu / który dáł król Artákſerkſeſ Ezdráƺowi / kápłánowi náucżonemu w zákonie / y biegłemu w tych rzecżách / które przykázáł Pán / y w uſtáwách jego w Izráelu.
Artákſerkſeſ / król nád królmi / Ezdráƺowi / kápłánowi náucżonemu w zákonie Bogá niebieſkiego / mężowi doſkonáłemu / y Cheenetcżykom.
Wydány odemnie jeſt dekret / iż ktobykolwiek dobrowolnie w króleſtwie mojem z ludu Izráelſkiego / y z kápłánów jego y z Lewitów chćiáł iść z tobą do Jeruzálemu / áby ƺedł.
Poniewáż od królá y od śiedmiu rádnych pánów jego jeſteś poſłány / ábyś dojrzáł Judy y Jeruzálemu według zákonu Bogá twego / który jeſt w rękách twoich.
A iżbyś odnióſł ſrebro y złoto / które król y rádni pánowie jego dobrowolnie ofiárowáli Bogu Izráelſkiemu / którego przybytek jeſt w Jeruzálemie ;
Do tego wƺyſtko ſrebro y złoto / któregobyś nábył we wƺyſtkiej kráinie Bábilońſkiej / z dobrowolnemi dárámi od ludu y od kápłánów / którzyby co dobrowolnie ofiárowáli ná dom Bogá ſwego / który jeſt w Jeruzálemie ;
Abyś prędko nákupił zá to ſrebro ćielców / báránów / báránków z śniednemi ofiárámi ich / y z mokremi ofiárámi ich / á ofiárowáł je ná ołtárzu domu Bogá wáƺego / który jeſt w Jeruzálemie ;
A cokolwiek śię tobie y bráćiom twoim będzie dobrego zdáło / z oſtátkiem ſrebrá y złotá ucżynić / według woli Bogá wáƺego ucżyńćie.
Nácżyniá też / któreć ſą oddáne do uſługi domu Bogá twego / oddáj przed Bogiem w Jeruzálemie ;
Tákże y inne rzecży / náleżące do domu Bogá twego / y coby potrzebá dáć / dáƺ z domu ſkárbów królewſkich.
A já / já król Artákſerkſeſ / rozkázáłem wƺyſtkim podſkárbim / którzyśćie zá rzeką / áby wƺyſtko / cżegobykolwiek żądáł od wáſ Ezdráƺ kápłán / náucżyćiel zákonu Bogá niebieſkiego / prędko śię ſtáło /
Aż do ſtá tálentów ſrebrá y áż do ſtá korcy pƺenicy / y áż do ſtá wiáder winá / y áż do ſtá báreł oliwy / á ſoli bez miáry.
Cobykolwiek było z rozkázániá Bogá niebieſkiego / niech będzie prędko dodáne do domu Bogá niebieſkiego ; bo przecż má być wzruƺony gniew jego przećiwko króleſtwu / królowi y ſynom jego?
Tákże oznájmujemy wám / áby ná żádnego z kápłánów / y z Lewitów / y z śpiewáków / y z odźwiernych / Netynejcżyków / y innych ſług domu Bogá tego / cłá / cżynƺów / y dáni dorocżnej żáden ſtároſtá nie wkłádáł.
A ty Ezdráƺu! według mądrośći Bogá twego / którá jeſt w tobie / poſtánowiƺ ſędziów / y w práwie biegłych / áby ſądzili wƺyſtek lud / który jeſt zá rzeką / ze wƺyſtkich / którzy ſą powiádomi zákonu Bogá twego ; á ktoby nie umiáł / ucżyć go będziećie.
A ktobykolwiek nie cżynił doſyć zákonowi Bogá twego / y práwu królewſkiemu / áby prędki dekret był o nim wyádány álbo ná śmierć / álbo ná wygnánie jego / álbo ná ſkáránie ná májętnośći / álbo ná więzienie. Y rzekł Ezdráƺ :
Błogoſłáwiony Pán / Bóg ojców náƺych / który to dáł w ſerce królewſkie / áby uwielbił dom Páńſki / który jeſt w Jeruzálemie ;
A ku mnie ſkłonił miłośierdzie przed królem y rádą jego / y wƺyſtkimi kśiążętámi królewſkimi możnymi. Przetoż já / będąc umocniony ręką Páná / Bogá mojego / którá jeſt nádemną / zgromádziłem z Izráelá przedniejƺych / którzy wyƺli zemną.
