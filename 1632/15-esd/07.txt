A po tych ſpráwách zá królowánia Artákſerkſeſá / królá Perſkiego / Ezdráƺ / ſyn Sárájáƺá / ſyná Azáryjáƺowego / ſyná Helkijáƺowego /
Syná Sállumowego / ſyná Sádokowego / ſyná Achitobowego /
Syná Amáryjáƺowego / ſyná Azáryjáƺowego / ſyná Merájotowego /
Syná Zeráhyjáƺowego / ſyná Uzego / ſyná Bukkiego /
Syná Abiſujego / ſyná Fineeſowego / ſyná Eleázárowego / ſyná Aároná kápłáná nájwyżƺego :
Ten Ezdráƺ wyƺedł z Bábilonu / á był cżłowiekiem biegłym w zakonie Mojżeƺowym / który był dał Pán / Bóg Izráelſki ; á pozwolił mu był król według ręki Páná Bogá jego nád nim / ná wƺyſtkę prośbę jego.
( Wyƺli też niektórzy z ſynów Izráelſkich / y z kápłánów / y z Lewitów / y śpiewáków / y z odźwiernych / y z Netynejcżyków / do Jeruzálemu roku śiódmego Artákſerkſeſá królá. )
Y przyƺedł do Jeruzálemu mieśiącá piątego ; tenći był rok śiódmy królá Dáryjuƺá.
Abowiem w pierwƺy dźień mieśiącá pierwƺego wyƺedł z Bábilonu / á dniá pierwƺego mieśiącá piątego przyƺedł do Jeruzálemu według łáſkáwego wſpomożeniá Bogá ſwego.
Bo Ezdráƺ przygotował był ſerce ſwe / áby ƺukał zakonu Páńſkiego / y áby cżynił / y náucżał w Izráelu uſtáw y ſądów.
A tenći jeſt przepis liſtu / który dał król Artákſerkſes Ezdráƺowi / kápłánowi náucżonemu w zakonie / y biegłemu w tych rzecżách / które przykázał Pán / y w uſtáwách jego w Izráelu.
Artákſerkſes / król nád królmi / Ezdráƺowi / kápłánowi náucżonemu w zakonie Bogá niebieſkiego / mężowi doſkonáłemu / y Cheenetcżykom.
Wydány odemnie jeſt dekret / iż ktobykolwiek dobrowolnie w króleſtwie mojim z ludu Izráelſkiego / y z kápłánów jego y z Lewitów chćiał iść z tobą do Jeruzálemu / áby ƺedł.
Poniewáż od królá y od śiedmiu rádnych pánów jego jeſteś poſłány / ábyś dojrzał Judy y Jeruzálemu według zakonu Bogá twego / który jeſt w rękách twojich.
A iżbyś odnióſł ſrebro y złoto / które król y rádni pánowie jego dobrowolnie ofiárowáli Bogu Izráelſkiemu / którego przybytek jeſt w Jeruzálemie ;
Do tego wƺyſtko ſrebro y złoto / któregobyś nábył we wƺyſtkiej kráinie Bábilońſkiej / z dobrowolnemi dárámi od ludu y od kápłánów / którzyby co dobrowolnie ofiárowáli ná dom Bogá ſwego / który jeſt w Jeruzálemie ;
Abyś prędko nákupił zá to ſrebro ćielców / báránów / báránków z śniednemi ofiárámi ich / y z mokremi ofiárámi ich / á ofiárował je ná ołtárzu domu Bogá wáƺego / który jeſt w Jeruzálemie ;
A cokolwiek śię tobie y bráćiom twoim będźie dobrego zdáło / z oſtátkiem ſrebrá y złotá ucżynić / według woli Bogá wáƺego ucżyńćie.
Nácżyniá też / któreć ſą oddáne do uſługi domu Bogá twego / oddáj przed Bogiem w Jeruzálemie ;
Tákże y inne rzecży / náleżące do domu Bogá twego / y coby potrzebá dáć / dáƺ z domu ſkárbów królewſkich.
A ja / ja król Artákſerkſes / rozkazałem wƺyſtkim podſkárbim / którzyśćie zá rzeką / áby wƺyſtko / cżegobykolwiek żądał od was Ezdráƺ kápłán / náucżyćiel zakonu Bogá niebieſkiego / prędko śię ſtáło /
Aż do ſtá tálentów ſrebrá y áż do ſtá korcy pƺenicy / y áż do ſtá wiáder winá / y áż do ſtá báreł oliwy / á ſoli bez miáry.
Cobykolwiek było z rozkázánia Bogá niebieſkiego / niech będźie prędko dodáne do domu Bogá niebieſkiego ; bo przecż ma być wzruƺony gniew jego przećiwko króleſtwu / królowi y ſynom jego?
Tákże oznájmujemy wam / áby ná żádnego z kápłánów / y z Lewitów / y z śpiewáków / y z odźwiernych / Netynejcżyków / y innych ſług domu Bogá tego / cłá / cżynƺów / y dáni dorocżney żáden ſtároſtá nie wkłádał.
A ty Ezdráƺu! według mądrośći Bogá twego / która jeſt w tobie / poſtánowiƺ ſędźiów / y w práwie biegłych / áby ſądźili wƺyſtek lud / który jeſt zá rzeką / ze wƺyſtkich / którzy ſą powiádomi zakonu Bogá twego ; á ktoby nie umiał / ucżyć go będźiećie.
A ktobykolwiek nie cżynił doſyć zakonowi Bogá twego / y práwu królewſkiemu / áby prędki dekret był o nim wyádány álbo ná śmierć / álbo ná wygnánie jego / álbo ná ſkáránie ná májętnośći / álbo ná więźienie. Y rzekł Ezdráƺ :
Błogoſłáwiony Pán / Bóg ojców náƺych / który to dał w ſerce królewſkie / áby uwielbił dom Páńſki / który jeſt w Jeruzálemie ;
A ku mnie ſkłonił miłośierdźie przed królem y rádą jego / y wƺyſtkimi kśiążętámi królewſkimi możnymi. Przetoż ja / będąc umocniony ręką Páná / Bogá mojego / która jeſt nádemną / zgromádźiłem z Izráelá przedniejƺych / którzy wyƺli zemną.
