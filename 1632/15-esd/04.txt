A gdy uſłyƺeli nieprzyjáćiele Judy y Benjáminá / iż lud / który przyƺedł z pojmániá / budowáli kośćiół Pánu / Bogu Izráelſkiemu ;
Tedy przyƺli do Zorobábelá y do przedniejƺych z domów ojcowſkich / mówiąc im : Będziemy budowáć z wámi / á jáko y wy będziemy ƺukáć Bogá wáƺego / gdyżeśmy mu ofiáry cżynili ode dni Aſárháddoná / królá Aſſyryjſkiego / który náſ tu przywiódł.
Ale im rzekł Zorobábel / y Jeſuá / y inni przedniejśi domów ojcowſkich z Izráelá : Nie wám / ále nám náleży budowáć dom Bogu náƺemu ; przetoż my ſámi budowáć będziemy Pánu / Bogu Izráelſkiemu / jáko nám rozkázáł Cyruſ / król Perſki.
A ták lud onej kráiny wątlił ręce ludu Judzkiego / y przeƺkádzáł im / áby nie budowáli.
Nádto przenájmowáli przećiwko nim rádców / áby rozrywáli rádę ich po wƺyſtkie dni Cyruſá / królá Perſkiego / áż do królowániá Dáryjuƺá / królá Perſkiego.
Bo gdy królowáł Aſweruſ / tedy ná pocżątku króleſtwá jego / nápiſáli ſkárgę przećiwko obywátelom Judzkim y Jeruzálemſkim /
Ták jáko zá dni Artákſerkſeſá piſáł Biſlán / Mitrydáteſ / Tábeel / y inni towárzyƺe jego do Artákſerkſeſá królá Perſkiego ; á piſmo liſtu tego nápiſáne było po ſyryjſku / y wyłożone też było po ſyryjſku.
Rechum kánclerz / y Symſáj piſárz nápiſáli liſt jeden przećiwko Jeruzálemowi do Artákſerkſeſá królá w ten ſpoſób :
To ucżynili nátencżáſ Rechum kánclerz / y Symſáj piſárz / y inni towárzyƺe ich / Dynájcżycy / y Afárſádchájcżycy / Tárpelájcżycy / Afárſájcżycy / Arkiewájcżycy / Bábilońcżycy / Suſánchájcżycy / Deháwejcżycy y Elmájcżycy ;
Y inne národy / które był przyprowádził Aſnápár wielki y ſłáwny / á oſádził nimi miáſtá Sámáryjſkie ; y inni / którzy byli zá rzeką / y Cheenetcżycy.
A tenći jeſt przepiſ liſtu / który poſłáli do Artákſerkſeſá królá :
Słudzy twoi / ludzie mieƺkájący zá rzeká / y Cheenetcżycy. Niech będzie wiádomo królowi / że Żydowie / którzy śię wróćili od ćiebie / przyƺedłƺy do náſ do Jeruzálemu / miáſto odporne y złe budują / y mury zákłádáją / á z gruntu je wywodzą.
Przetoż niech będzie wiádomo królowi / że będzieli to miáſto pobudowáne / y mury jego z gruntu wywiedzione / tedy cłá / cżynƺów / y dáni dorocżnej nie będą dáwáć / á ták dochodom królewſkim ujmá śię ſtánie.
Teráz tedy / poniewáż używámy dobrodziejſtw páłácu twego / ná ƺkodę królewſką nie godzi śię nám pátrzyć ; dlá tegośmy poſłáli / oznájmując to królowi /
Abyś dáł ƺukáć w kśięgách hiſtoryj ojców ſwoich / á znájdzieƺ w kśięgách hiſtoryj / y dowieƺ śię / iż to miáſto jeſt miáſto odporne y ƺkodliwe królom y kráinom / á iż śię w nim wƺcżynáły bunty od dáwnych dni / przez co to miáſto było zburzone.
Nádto wiádomo cżynimy królowi / że jeżli śię to miáſto pobuduje / y mury jego z gruntu wywiedzione będą / tedy już tá cżęść zá rzeką nie będzie twojá.
Tedy dáł odpowiedż król Rechumowi kánclerzowi / y Symſájemu piſárzowi / y innym towárzyƺom ich / którzy mieƺkáli w Sámáryi / tákże y innym zá rzeką w Selám y w Cheet :
Liſt / któryśćie poſłáli do náſ / jáwnie przedemną cżytáno.
Przetoż rozkázáłem / áby ƺukáno ; y ználeziono / że to miáſto z dáwná powſtáwáło przećiwko królom / á bunty y rozruchy bywáły w nim ;
Nádto królowie możni bywáli w Jeruzálemie / którzy pánowáli nád wƺyſtkiem / co jeſt zá rzeką / którym cłá / cżynƺe / y dáni dorocżne dáwáno.
Przetoż wydájćie wyrok / áby zábroniono onym mężom / áby to miáſto nie było budowáne / pókiby odemnie inƺy rozkáz nie wyƺedł.
Pátrzćież / ábyśćie śię w tem nie omylili. Przecżżeby urość miáło co złego ná ƺkodę królom?
A ták / gdy przepiſ liſtu Artákſerkſeſá był cżytány przed Rechumem / y Symſájem piſárzem / y przed towárzyƺámi ich / jecháli prędko do Jeruzálemu do Żydów / á zábronili im gwáłtem y mocą budowáć.
A ták uſtáłá robotá około domu Bożego / który był w Jeruzálemie / y zániecháno jej áż do wtórego roku króleſtwá Dáryjuƺá / królá Perſkiego.
