Tedy król Dáryjuƺ rozkázał / áby ƺukáno w biblijotece miedzy ſkárbámi támże złożonemi w Bábilonie.
Y ználeźiono w Achmećie ná zámku / który jeſt w źiemi Medſkiej / kśięgę jednę / á táká byłá zápiſáná w niej pámięć :
Roku pierwƺego Cyruſá królá / król Cyrus wydał wyrok o domu Bożym / który był w Jeruzálemie / áby był dom zbudowány dla miejſcá / gdźieby ofiáry ſpráwowano ; áby też grunty jego były wybudowáne / wyſokość jego ná ƺeśćdźieśiąt łokći / á ƺerokość jego ná ƺeśćdźieśiąt łokći.
Trzy rzędy z kámieniá wielkiego / á jeden rząd z drzewá nowego / á nákłád z domu królewſkiego dáwány będźie.
Nádto y nacżynia domu Bożego / złote y ſrebrne / które był zábrał Nábuchodonozor z kośćiołá / który jeſt w Jeruzálemie / á przenióſł do Bábilonu / niech wrócą / áby śię doſtáły do kośćiołá / który jeſt w Jeruzálemie / ná miejſce ſwe / y złożone były w domu Bożym.
Przetoż teraz Táttenáju / ſtároſto zá rzeką! z Setárbozenáimem / y z towárzyƺámi twymi / y Afárſechájcżycy / którzyśćie zá rzeką / uſtąpćie z támtąd.
Dopuśććie / żeby był budowány ten dom Boży od kśiążęćiá Żydowſkiego / y od ſtárƺych Żydowſkich / áby ten dom Boży zbudowáli ná miejſcu ſwem.
Odemnie też wyƺedł wyrok o tym / cobyśćie mieli cżynić z ſtárƺymi tych Żydów przy budowániu tego domu Bożego ; to jeſt / áby z májętnośći królewſkich / z dochodów / które ſą zá rzeką / dáwano bez omieƺkánia nákłád mężom tym / áby nie przeſtáwáli.
A ile potrzebá y wołów / y báránów / y báránków ná cáłopalenia Bogu niebieſkiemu / zbożá / ſoli / winá / y oliwy / ná rozkázánie kápłanów / którzy ſą w Jeruzálemie / áby im dáwano ná káżdy dźień / á to bez omieƺkánia ;
Aby mieli ſkąd ofiárowáć wonne kádzeniá Bogu niebieſkiemu / y áby śię modlili zá zdrowie królewſkie / y ſynów jego.
Nádto ucżyniony jeſt odemnie dekret : Ktobykolwiek wzruƺył to przykazánie / áby wyjęto drzewo z domu jego / y áby je podnieśiono / á ná nim go powieƺono / á dom jego áby był gnojowiſkiem dla tego.
A Bóg / który tám ucżynił mieƺkánie imieniowi ſwemu / niech zniƺcży káżdego królá y naród / któryby śćiągnął rękę ſwą ná odmiánę y ſkázę tego domu Bożego / który jeſt w Jeruzálemie. Ja Dáryjuƺ ucżyniłem ten dekret ; bez omieƺkánia niech będźie wykonány.
Tedy Táttenáj / ſtároſtá zá rzeką / y Setárbozenáj / y towárzyƺe ich według tego / jáko rozkázał król Dáryjuƺ / ták ucżynili bez omieƺkánia.
A ſtárśi Żydowſcy budowáli / y ƺcżęśćiło śię im według proroctwá Hággieuƺá proroká / y Zácháryjáƺá / ſyná Iddy ; y budowáli y dokonáli zá rozkázániem Bogá Izráelſkiego / y zá rozkázániem Cyruſá / y Dáryjuƺá / y Artákſerkſeſá / królów Perſkich.
Y dokońcżony jeſt on dom trzećiego dniá mieśiącá Adar / á ten był rok ƺóſty pánowánia Dáryjuƺá królá.
Tedy ſynowie Izráelſcy / kápłáni y Lewitowie / y inni z ludu / którzy przyƺli z więźieniá / poświęcáli on dom Boży z rádośćią.
A ofiárowáli przy poświęcániu onego domu Bożego / ćielców ſto / báránów dwieśćie / báránków cżtery ſtá / y kozłów z kóz ná ofiárę zá grzech zá wƺyſtkiego Izráelá / dwánaśćie / według licżby pokolenia Izráelſkiego.
Y poſtáwili kápłanów w rzędách ſwych / y Lewitów w przemiánách ſwojich / nád ſłużbą Bożą w Jeruzálemie / jáko nápiſáne w kśięgách Mojżeƺowych.
Obchodźili też ći / co przyƺli z niewoli / ſwięto przejśćiá cżternáſtego dniá mieśiącá pierwƺego.
Bo śię ocżyśćili kápłáni y Lewitowie jednoſtájnie / wƺyſcy byli ocżyƺcżeni ; przetoż ofiárowáli báránká świętá przejśćiá zá wƺyſtkich / którzy przyƺli z niewoli / y zá bráći ſwojich kápłanów / y zá śiebie ſámych.
A ták jedli ſynowie Izráelſcy / którzy śię wróćili z niewoli / y káżdy / który śię odłącżył od ſproſnośći narodów oney źiemi do nich / áby ƺukał Páná / Bogá Izráelſkiego.
Y obchodźili święto urocżyſte przáśników przez śiedm dni z rádośćią / przeto / że ich Pán był rozweſelił / á obróćił ſerce królá Aſſyryjſkiego do nich / áby zmocnił ręce ich w roboćie około domu Bożego / Bogá Izráelſkiego.
