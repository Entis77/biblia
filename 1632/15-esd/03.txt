A gdy náſtáł mieśiąc śiódmy / á ſynowie Izráelſcy byli w miáſtách / zgromádził śię lud jednomyślnie do Jeruzálemu.
Tedy wſtáwƺy Jeſuá / ſyn Jozedeká / y bráćiá jego kápłáni / y Zorobábel / ſyn Sálátyjelá / y bráćiá jego / zbudowáli ołtárz Bogá Izráelſkiego / áby ná nim ſpráwowáli cáłopáleniá / jáko nápiſáne w zákonie Mojżeƺá / mężá Bożego.
A gdy poſtáwili on ołtárz ná fundámenćie ſwym / choć śię báli národów poſtronnych / jednák ofiárowáli ná nim cáłopáleniá Pánu / cáłopáleniá ráno y w wiecżór.
Obchodzili też święto urocżyſte kucżek / jáko nápiſáne / ſpráwując cáłopáleniá ná káżdy dzień według licżby y według zwycżáju káżdą rzecż dniá ſwego ;
Potem cáłopálenie uſtáwicżne / y ná nowiu mieśiącá / y ná káżde urocżyſte święto Pánu poświęcone / y od káżdego dobrowolnie ofiárującego dobrowolną ofiárę Pánu.
Ode dniá pierwƺego mieśiącá śiódmego pocżęli ſpráwowáć cáłopáleniá Pánu / choć jeƺcże kośćiół Páńſki nie był záłożony.
Y oddáli pieniądze kámiennikom / y rzemieślnikom / tákże ſtráwę y nápój / y oliwę Sydońcżykom / y Tyryjcżykom / áby przywozili drzewo cedrowe z Libánu do morzá Joppy / jáko im pozwolił Cyruſ / król Perſki.
Potem roku wtórego po ich náwróceniu do domu Bożego w Jeruzálemie / mieśiącá wtórego / zácżęli Zorobábel / ſyn Sálátyjelá / y Jeſuá / ſyn Jozedeká / y inni bráćiá ich kápłáni / y Lewitowie / y wƺyſcy / którzy byli przyƺli z onej niewoli do Jeruzálemu / á poſtánowili Lewitów od dwudzieſtu lát y wyżej / áby byli dozorcámi nád robotą domu Páńſkiego.
Y ſtánął Jeſuá / ſynowie jego / y bráćiá jego ; Kádmiel też y ſynowie jego / ſynowie Judy ſpołu / áby przynágláli tym / którzy robili około domu Bożego ; ſynowie Chenádádowi / ſynowie ich / y bráćiá ich Lewitowie.
A gdy zákłádáli budownicy grunty kośćiołá Páńſkiego / poſtáwili kápłánów ubránych z trąbámi / y Lewitów / ſynów Aſáfowych z cymbáłámi / áby chwálili Páná według poſtánowieniá Dawidá / królá Izráelſkiego.
Y śpiewáli jedni po drugich chwáląc á wyſłáwiájąc Páná / że dobry / że ná wieki miłośierdzie jego nád Izráelem. Wƺyſtek tákże lud krzycżáł krzykiem wielkim / chwáląc Páná / przeto / iż był záłożony dom Páńſki.
A wiele ſtárców / z kápłánów / y z Lewitów / y z przedniejƺych domów ojcowſkich / którzy widzieli dom pierwƺy / gdy zákłádáno ten dom przed ocżymá ich / płákáli głoſem wielkim / á záśię wiele ich krzycżáło / z rádośćią wynoƺąc głoſy ;
Ták / iż lud nie mógł rozeznáć głoſu krzyku weſołego od głoſu płácżącego ludu ; ábowiem lud on krzycżáł głoſem wielkim / ták że głoſ było dáleká ſłyƺeć.
