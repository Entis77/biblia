A gdy náſtał mieśiąc śiódmy / á ſynowie Izráelſcy byli w miáſtách / zgromádźił śię lud jednomyślnie do Jeruzálemu.
Tedy wſtáwƺy Jeſuá / ſyn Jozedeká / y bráćia jego kápłáni / y Zorobábel / ſyn Sálátyjelá / y bráćia jego / zbudowáli ołtárz Bogá Izráelſkiego / áby ná nim ſpráwowáli cáłopalenia / jáko nápiſáne w zakonie Mojżeƺá / mężá Bożego.
A gdy poſtáwili on ołtárz ná fundámenćie ſwym / choć śię báli narodów poſtronnych / jednák ofiárowáli ná nim cáłopalenia Pánu / cáłopalenia ráno y w wiecżór.
Obchodźili też święto urocżyſte kucżek / jáko nápiſáne / ſpráwując cáłopalenia ná káżdy dźień według licżby y według zwycżáju káżdą rzecż dniá ſwego ;
Potem cáłopálenie uſtáwicżne / y ná nowiu mieśiącá / y ná káżde urocżyſte święto Pánu poświęcone / y od káżdego dobrowolnie ofiárującego dobrowolną ofiárę Pánu.
Ode dniá pierwƺego mieśiącá śiódmego pocżęli ſpráwowáć cáłopalenia Pánu / choć jeƺcże kośćiół Páńſki nie był záłożony.
Y oddáli pieniądze kámiennikom / y rzemieślnikom / tákże ſtráwę y nápój / y oliwę Sydońcżykom / y Tyryjcżykom / áby przywoźili drzewo cedrowe z Libánu do morzá Joppy / jáko im pozwolił Cyrus / król Perſki.
Potem roku wtórego po ich náwróceniu do domu Bożego w Jeruzálemie / mieśiącá wtórego / zácżęli Zorobábel / ſyn Sálátyjelá / y Jeſuá / ſyn Jozedeká / y inni bráćia ich kápłáni / y Lewitowie / y wƺyſcy / którzy byli przyƺli z oney niewoli do Jeruzálemu / á poſtánowili Lewitów od dwudźieſtu lat y wyżey / áby byli dozorcámi nád robotą domu Páńſkiego.
Y ſtánął Jeſuá / ſynowie jego / y bráćia jego ; Kádmiel też y ſynowie jego / ſynowie Judy ſpołu / áby przynágláli tym / którzy robili około domu Bożego ; ſynowie Chenádádowi / ſynowie ich / y bráćia ich Lewitowie.
A gdy zákłádáli budownicy grunty kośćiołá Páńſkiego / poſtáwili kápłanów ubránych z trąbámi / y Lewitów / ſynów Aſáfowych z cymbáłámi / áby chwálili Páná według poſtánowieniá Dawidá / królá Izráelſkiego.
Y śpiewáli jedni po drugich chwáląc á wyſłáwiájąc Páná / że dobry / że ná wieki miłośierdźie jego nád Izráelem. Wƺyſtek tákże lud krzycżał krzykiem wielkim / chwáląc Páná / przeto / iż był záłożony dom Páńſki.
A wiele ſtárców / z kápłanów / y z Lewitów / y z przedniejƺych domów ojcowſkich / którzy widźieli dom pierwƺy / gdy zákłádáno ten dom przed ocżymá ich / płákáli głoſem wielkim / á záśię wiele ich krzycżáło / z rádośćią wynoƺąc głoſy ;
Ták / iż lud nie mógł rozeznáć głoſu krzyku weſołego od głoſu płácżącego ludu ; ábowiem lud on krzycżał głoſem wielkim / ták że głos było dáleká ſłyƺeć.
