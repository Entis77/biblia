A ćić ſą przedniejśi z domów ſwych ojcowſkich / y ród tych / którzy wyƺli zemną z Bábilonu zá królowániá królá Artákſerkſeſá :
Z ſynów Fineáƺowych Gierſon ; z ſynów Itámárowych Dánijel ; z ſynów Dawidowych Háttuſ ;
Z ſynów Sechánijáƺowych / który był z ſynów Fáreſowych / Zácháryjáƺ / á z nim pocżet mężów ſto y pięćdzieśiąt ;
Z ſynów Páchátmoábowych Elijeoenáj / ſyn Zeráchyjáƺowy / á z nim dwieśćie mężów ;
Z ſynów Sechánijáƺowych ſyn Jáházyjelowy / á z nim trzy ſtá mężów ;
A z ſynów Adynowych Ebed / ſyn Jonátáná / á z nim pięćdzieśiąt mężów ;
A z ſynów Elámowych Iſájáƺ / ſyn Atálijáƺá / á z nim śiedmdzieśiąt mężów ;
A z ſynów Sefátyjáƺowych Zábádyjáƺ / ſyn Micháelowy / á z nim ośmdzieśiąt mężów ;
Z ſynów Joábowych Obádyjáƺ / ſyn Jechyjelowy / á z nim dwieśćie y ośmnáśćie mężów ;
A z ſynów Selomitowych / ſyn Joſyfijáƺowy / á z nim ſto y ƺeśćdzieśiąt mężów ;
A z ſynów Bebájowych Zácháryjáƺ / ſyn Bebájowy / á z nim dwádzieśćiá y ośm mężów ;
A z ſynów Azgádowych Johánán / ſyn Hákátánowy / á z nim ſto y dzieśięć mężów ;
A z ſynów Adonikámowych nájoſtátecżniejƺych / których te ſą imioná : Elifelet / Jehijel / y Semejáƺ / á z nimi ƺeśdzieśiąt mężów ;
A z ſynów Bigwájowych Utáj y Zábud / á z nimi śiedmdzieśiąt mężów ;
A ták zgromádziłem ich do rzeki / którá wpádá do Acháwy / y leżeliśmy tám obozem przez trzy dni : potem przeglądáłem lud y kápłánów / á z ſynów Lewiego nie ználázłem tám żádnego.
Przetoż poſłáłem Elijezerá / Aryjelá / Semejáƺá / y Elnátáná / y Járybá / y Elnátáná y Nátáná / y Zácháryjáƺá / y Meſullámá / przedniejƺych / y Jojárybá / y Elnátáná / mężów ucżonych ;
Y rozkázáłem im do Iddoná / przełożonego nád miejſcem Káſyfii / y włożyłem w uſtá ich ſłowá / które mieli mówić do Iddoná / Achywá y Netynejcżyków ná miejſcu Káſyfii / áby nám przywiedli ſług do domu Bogá náƺego.
Y przywiedli nám według ręki Bogá náƺego łáſkáwej nád námi / mężá náucżonego z ſynów Máheli / ſyná Lewiego / ſyná Izráelowego / y Serebijáƺá / y ſynów jego / y bráći jego ośmnáśćie ;
Y Háſábijáƺá / á z nim Jeſájáƺá z ſynów Merárego / bráći jego / y ſynów ich dwádzieśćiá ;
Nádto z Netynejcżyków / których był poſtánowił Dawid y przedniejśi ku poſłudze Lewitów / Netynejcżyków dwieśćie y dwádzieśćiá ; ći wƺyſcy z imieniá miánowáni byli.
Tedym tám zápowiedziáł poſt u rzeki Acháwy / ábyśmy śię dręcżyli przed Bogiem náƺym / á ƺukáli od niego drogi proſtej ſobie / y dziátkom náƺym / y wƺyſtkiej májętnośći náƺej.
Bom śię wſtydził prośić u królá o jáki pocżet jezdnych / áby nám byli ná pomocy przećiwko nieprzyjáćiołom w drodze ; bośmy byli powiedzieli królowi / mówiąc : Ręká Bogá náƺego jeſt nád wƺyſtkimi / którzy go ƺukáją uprzejmie / ále moc jego y popędliwość jego przećiwko wƺyſtkim / którzy go opuƺcżáją.
A gdyśmy pośćili / y prośiliśmy o to Bogá náƺego / wyſłucháł náſ.
Tedym odłącżył z przedniejƺych kápłánów dwánáśćie : Serebijáƺá / Háſábijáƺá / á z nimi bráći ich dzieśięć ;
Y odwáżyłem im ſrebro / y złoto / y nácżyniá ná ofiárę podnoƺeniá do domu Bogá náƺego / które ofiárowáli król y rádá jego / y kśiążętá jego / y wƺyſtek lud Izráelſki / ile śię go ználázło.
Odwáżyłem / mówię do rąk ich ſrebrá tálentów ƺeśćſet y pięćdzieśiąt / á nácżyniá ſrebrnego ſto tálentów / przytem złotá ſto tálentów.
Cżáƺ też złotych dwádzieśćiá / wáżących po tyśiąc łótów / á dwá nácżyniá z mośiądzu wybornego / ták piękne jáko złoto.
Potemem rzekł do nich : Wyśćie poświęceni Pánu / y nácżyniá tákże poświęcone / á to ſrebro y to złoto dobrowolnie ofiárowáne jeſt Pánu / Bogu ojców wáƺych.
Pilnujćież á ſtrzeżćie / áż to odwáżyćie przed kápłánámi przedniejƺymi / y Lewitámi / y kśiążętámi z domów ojcowſkich w Izráelu w Jeruzálemie / w gmáchách domu Páńſkiego.
A ták wzięli kápłáni y Lewitowie wágę onego ſrebrá / y złotá / y nácżyniá / áby je odnieśli do Jeruzálemu / do domu Bogá náƺego.
Zátem ruƺyliśmy śię od rzeki Acháwy dwunáſtego dniá / mieśiącá pierwƺego / ábyśmy ƺli do Jeruzálemu ; á ręká Bogá náƺego byłá nád námi / y wyrwáłá náſ z ręki nieprzyjáćielá y cżyhájącego ná náſ w drodze.
Y przyƺliśmy do Jeruzálemu / y zámieƺkáliśmy tám przez trzy dni.
A dniá cżwártego odwáżono ſrebro / y złoto / y nácżynie ono w domu Bogá náƺego do ręki Meremotá / ſyná Uryjáƺá kápłáná / z którym był Eleázár / ſyn Fineeſowy ; z nimi też byli Joſábád / ſyn Jeſuego / y Noádyjáƺ / ſyn Binnujego / Lewitowie ;
Pod licżbą y wágą wƺyſtko / y zápiſáno wágę tego wƺyſtkiego onegoż cżáſu.
Wróćiwƺy śię tedy z niewoli ći / którzy byli w pojmániu / ofiárowáli Bogu Izráelſkiemu ćielców dwánáśćie zá wƺyſtkiego Izráelá / báránów dziewięćdzieśiąt y ƺeść / báránków śiedmdzieśiąt y śiedm / kozłów zá grzech dwánáśćie / to wƺyſtko ná cáłopálenie Pánu.
Y oddáli wyroki królewſkie ſtároſtom królewſkim / y kśiążętom zá rzeką / á ći byli pomocą ludowi y domowi Bożemu.
