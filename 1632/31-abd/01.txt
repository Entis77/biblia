Widzenie Abdyjáƺowe. Ták mówi pánujący Pán o ziemi Edomſkiej : Słyƺeliśmy wieść od Páná y od poſłá wyſłánego do národów : Ruƺćie śię / á powſtáńmy przećiwko niemu ku bitwie /
Oto ćię málucżkim ucżynię między národámi / ty będzieƺ bárdzo wzgárdzony.
Pychá ſercá twego zdrádziłá ćię / o ty! który mieƺkáƺ w rozpádlinách ſkálnych / w wyſokiem mieƺkániu twojim / który mówiƺ w ſercu ſwojim : Któż mię ná ziemię śćiągnie?
Choćbyś śię wywyżƺył jáko orzeł / owƺem / choćbyś między gwiázdámi położył gniázdo twoje / y ſtámtąd ćię ſtárgnę / mówi Pán.
O jákożeś zniƺcżony! Izáliż złodzieje przyƺli ná ćię? Izáli zbójcy nocni? Izáliby krádli nád potrzebę ſwoję? Gdyby ći ná ćię przyƺli / co wino zbieráją / izáliby nie zoſtáwili którego groná?
Jákoż wyƺpiegowáne ſą ſkárby Ezáw / á wynálezione ſą ſkryte rzecży jego.
Aż do gránicy wypchną ćię wƺyſcy / z którymi máƺ przymierze ; zdrádzą ćię / moc wezmą nád tobą ći / z którymi máƺ pokój ; którzy chleb twój jedzą / ránęć zdrádliwie zádádzą / ták / iż śię nie obácżyƺ.
Izáli dniá onego / mówi Pán / nie wytrácę mędrców z Edom / á roztropnych z góry Ezáwá?
Y ulękną śię mocárze twoi / o Temánie! dlátego / że poráżeni będąc wygubieni będą wƺyſcy z góry Ezáwá.
Dlá bezpráwiá brátu twemu Jákóbowi ucżynionego háńbá ćię okryje / á wykorzeniony będzieƺ ná wieki.
Stáłeś dniá onego náprzećiwko / dniá onego / gdy cudzy imáli wojſko jego / y gdy cudzoziemcy wchodzili w brámy jego / á o Jeruzálem loſ miotáli / tyś też był jáko jeden z nich.
Nie pátrzże tedy ná dzień brátá ſwego / ná dzień pojmániá jego ; áni śię weſel nád ſynámi Judzkimi w dzień zginieniá ich / áni hárdzie mów uſty ſwemi w dzień ućiſku.
Nie wchodź w brámę ludu mego w dzień utrápieniá ich / áni pátrz ná złe jego w dzień doległośći jego / áni śćiągáj ręki ſwej ná májętność jego / w dzień ſkruƺeniá jego ;
Ani ſtój ná rozſtániu dróg / ábyś zátrácáł tych / którzy z nich uchodzą ; áni podáwáj nieprzyjáćielowi w moc tych / którzy z nich zoſtáli w dzień ućiſku.
Bo bliſki jeſt dzień Páńſki przećiwko tym wƺyſtkim národom ; jákoś ucżynił / ták ći śię ſtánie / nágrodá twojá obróći śię ná głowę twoję.
Bo poniewáż wy pić będziećie ná górze mojej świętej / ták pić będą wƺyſtkie národy ; uſtáwicżnie / mówię / pić y pożeráć będą / áż śię ſtáną / jákoby ich nie było.
A ná górze Syon będzie wybáwienie / á tá górá będzie świętá / y pośiędzie dom Jákóbowy ośiádłośći ſwe.
Y ſtánie śię dom Jákóbowy ogniem / á dom Józefowy płomieniem / dom záś Ezáwowy śćierniſkiem ; y rozpáli śię ná nich / y ſtráwi ich / á nikt nie zoſtánie z domu Ezáwowego ; bo Pán to mówił.
A ták odziedzicżą kráinę południową z górą Ezáwá / y równinę z Filiſtyńcżykámi ; pośiędą też kráinę Efráimową / y kráinę Sámáryi / y Benjáminową y Gáláádſką.
A záprowádzeni w niewolę tego wojſká ſynów Izráelſkich pośiędą to / co było Chánánejcżyków áż do Sárepty ; á záprowádzeni w niewolę Jeruzálemcżyków pośiędą to / co jeſt ná końcu páńſtwá / pośiędą z miáſtámi ná południe.
Y wſtąpią wybáwićiele ná górę Syon / áby ſądzili górę Ezáwá ; á ták będzie króleſtwo ſámego Páná.
