Izáli mądrość nie wołá / y roztropność nie wydáje głoſu ſwego?
Ná wierzchu wyſokich miejſc / przy drodze / y ná rozſtániu dróg ſtoi.
U brám / kędy śię chodźi do miáſtá / y w wejśćiu u drzwi wołá / mówiąc :
Ná was wołám / o mężowie! á głoſ mój obrácám do ſynów ludzkich.
Zrozumijćie proſtácy oſtrożność / á głupi zrozumijćie ſercem.
Słuchájćie ; bo o wielkich rzecżách będę mówił / á otworzenie wárg moich opowie ƺcżerość.
Záiſteć práwdę mówią uſtá moje / á niezbożność obrzydliwośćią jeſt wárgom moim.
Spráwiedliwe ſą wƺyſtkie ſłowá uſt moich ; niemáƺ w nich nic niepráwego áni przewrotnego.
Wƺyſtkie ſą práwe rozumnemu / á uprzejme tym / którzy znájdują umiejętność.
Przyjmijćież ćwicżenie moje / á nie ſrebro / á umiejętność rácżej / niż złoto wyborne.
Abowiem lepƺá jeſt mądrość niż perły / tákże wƺyſtkie pożądáne rzecży nie porównáją z nią.
Ja mądrość mieƺkám z roztropnośćią / y umiejętność oſtrożnośći wynájduję.
Bojáźń Páńſká jeſt / mieć w nienáwiśći złe. Ja nienáwidzę pychy / wyſokomyślnośći / y drogi złej / y uſt przewrotnych.
Przy mnie jeſt rádá / y práwdźiwá mądrość ; jám jeſt roztropność / á moc jeſt mojá.
Przez mię królowie królują / y kśiążętá ſtánowią ſpráwiedliwość.
Przez mię kśiążętá pánują / y wielmożnymi ſą wƺyſcy ſędźiowie źiemi.
Ja miłuję tych / którzy mię miłują ; á którzy mię ƺukáją ráno / znájdują mię.
Bogáctwo y ſłáwá przy mnie jeſt ; májętność trwáłá y ſpráwiedliwość.
Lepƺy jeſt owoc mój / niż złoto / y niż nájkoƺtowniejƺe złoto / á dochody moje lepƺe / niż ſrebro wyborne.
Prowádzę śćieƺką ſpráwiedliwośći / pośrodkiem śćieżek ſądu /
Abym tym / którzy mię miłują / dáłá w dźiedźictwo májętność wiecżną / y ſkárby ich nápełniłá.
Pán mię miał przy pocżątku drogi ſwej / przed ſpráwámi ſwemi / przed wƺyſtkiemi cżáſy.
Przed wieki jeſtem zrządzoná / przed pocżątkiem ; pierwej niż byłá źiemiá ;
Gdy jeƺcże nie było przepáśći / ſpłodzonám jeſt / gdy jeƺcże nie było źródeł opływájących wodámi.
Pierwej niż góry záłożone były / niż były págórki / ſpłodzonám jeſt.
Jeƺcże był nie ucżynił źiemi / y równin / áni pocżątku prochu okręgu źiemſkiego.
Gdy gotowáł niebioſá / támem byłá ; gdy rozmierzáł okrągłość nád przepáśćiámi ;
Gdy utwierdzáł obłoki w górze / y umácniáł źródłá przepáśći ;
Gdy zákłádáł morzu gránice jego / y wodom / áby nie przeſtępowáły rozkázániá jego ; gdy rozmierzáł grunty źiemi :
Tedym byłá u niego jáko wychowániec / y byłám ućiechą jego ná káżdy dźień / grájąc przed nim ná káżdy cżáſ.
Grám ná okręgu źiemi jego / á rozkoƺy moje / mieƺkáć z ſynámi ludzkimi.
Słuchájćież mię tedy teraz / ſynowie! ábowiem błogoſłáwieni / którzy ſtrzegą dróg moich.
Słuchájćie ćwicżeniá / nábądźćie rozumu / á nie cofájćie śię.
Błogoſłáwiony cżłowiek / który mię ſłuchá / cżując u wrót moich ná káżdy dźień / á ſtrzegąc podwoi drzwi moich.
Bo kto mię znájduje / znájduje żywot / á otrzymuje łáſkę od Páná.
Ale kto grzeƺy przećiwko mnie / krzywdę cżyni duƺy ſwojey ; wƺyſcy / którzy mię nienáwidzą / miłują śmierć.
