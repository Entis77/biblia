Synu mój! jeſlibyś ręcżył zá przyjáćielá twego / á dáłbyś obcemu rękę twoję :
Uśidliłeś śię ſłowy uſt twojich / pojmányś mowámi uſt twojich.
Przetoż ucżyń ták / ſynu mój! á wyzwól śię / gdyżeś wpádł w rękę przyjáćielá twego ; idźże / upokórz śię / á nálegáj ná przyjáćielá twego.
Nie dáwáj ſnu ocżom twoim / áni drzemánia powiekom twoim.
Wyrwij śię jáko łáni z rąk myśliwcá / y jáko pták z ręki ptáƺniká.
Idź do mrówki / leniwcże! obácż drogi jey / á nábądź mądrośći ;
Która / choć nie ma wodzá / áni przełożonego / áni páná /
Przećież w lećie gotuje pokárm ſwój / á zgromádzá w żniwá żywność ſwoję.
Leniwcże! dokądże leżeć będźieƺ? kiedyż wſtánieƺ ze ſnu ſwego?
Trochę śię prześpiƺ / trochę podrzemieƺ / trochę złożyƺ ręce / ábyś odpocżywał.
A wtem ubóſtwo twoje przydźie jáko podróżny / á niedoſtátek twój / jáko mąż zbrojny.
Cżłowiek niepobożny / mąż złośliwy chodźi w przewrotnośći uſt ;
Mrugá ocżymá ſwemi / mówi nogámi ſwemi / ukázuje pálcámi ſwemi ;
Przewrotnośći ſą w ſercu jego / myśli złe ná káżdy cżás / á zwády rozśiewá.
Przetoż prędko przydźie upádek jego ; nágle ſkruƺony będźie bez ulecżeniá.
Sześć jeſt rzecży / których nienáwidźi Pán / á śiódmá jeſt obrzydliwośćią duƺy jego :
Ocżów wynioſłych / języká kłámliwego / y rąk wylewájących krew niewinną ;
Sercá / które knuje myśli złe ; nóg / które śię kwápią bieżeć ku złemu ;
Świádká fáłƺywego / który mówi kłámſtwo / y tego / który śieje roſterki miedzy bráćmi.
Strzeżże / ſynu mój! przykázánia ojcá twego / á nie opuƺcżáj náuki mátki twojey.
Wiążże je záwżdy u ſercá twego / á wieƺáj je u ƺyi twojey.
Gdźiekolwiek pójdźieƺ / poprowádźi ćię ; gdy záśnieƺ / ſtrzedz ćię będźie / á gdy śię ocućiƺ / rozmáwiáć z tobą będźie /
( Bo przykazánie jeſt pochodnią / náuká świátłośćią / á drogą żywotá ſą kárnośći ćwicżeniá. )
Aby ćię ſtrzegły od niewiáſty złej / y od łágodnego języká niewiáſty obcey.
Nie pożądáj pięknośći jey w ſercu twojim / á niech ćię nie łowi powiekámi ſwemi.
Abowiem dla niewiáſty wƺetecżney zubożeje cżłowiek áż do kęſá chlebá ; owƺem żoná cudzołożná drogą duƺę łowi.
Izáż może kto bráć ogień do zánádrzy ſwojich / áby ƺáty jego nie zgorzáły?
Izáż może kto chodźić po rozpálonym węglu / áby śię nogi jego nie popárzyły?
Ták kto wchodźi do żony bliźniego ſwego / nie będźie bez winy / ktokolwiek śię jey dotknie.
Nie kłádą háńby ná złodźiejá / jeſliż co ukrádnie / chcąc náſyćić duƺę ſwoję / będąc głodnym ;
Ale gdy go záſtáną / nágrádzá śiedmioráko / álbo wƺyſtkę májętność domu ſwego dáje.
Lecż cudzołożący z niewiáſtą głupi jeſt / á kto chce zátráćić duƺę ſwoję / ten to cżyni.
Káránie y zelżywość odnieśie / á háńbá jego nie będźie zgłádzoná.
Bo záwiſná miłość jeſt zápálcżywośćią mężá / á nie ſfolguje w dźień pomſty.
Nie będźie miał względu ná żáden okup / áni przyjmie / choćiáżby mu nájwięcey dárów dáwano.
