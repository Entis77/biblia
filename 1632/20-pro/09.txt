Mądrość zbudowáłá dom ſwój / y wyćioſáłá śiedm ſłupów ſwoich ;
Pobiłá bydło ſwoje / roztworzyłá wino ſwoje / y ſtół ſwój przygotowáłá ;
A rozeſłáłá dźiewecżki ſwoje / wołá ná wierzchách nájwyżƺych miejſc w mieśćie / mówiąc :
Ktokolwiek jeſt proſtákiem / wſtąp ſám ; á do głupich mówi :
Pójdźćie / jedzćie chleb mój / y pijćie wino / którem roztworzyłá.
Opuśććie proſtotę / á będźiećie żyli / á chodźćie drogą roztropnośći.
Kto ſtrofuje náśmiewcę / odnośi háńbę ; á kto ſtrofuje niezbożniká / odnośi zelżywość.
Nie ſtrofuj náśmiewcy / áby ćię nie miáł w nienáwiśći ; ſtrofuj mądrego / á będźie ćię miłowáł.
Ucżyń to mądremu / á mędrƺym będźie ; náucż ſpráwiedliwego / á będźie umiejętniejƺym.
Pocżątek mądrośći jeſt bojáźń Páńſká / á umiejętność świętych jeſt rozum.
Bo przez mię rozmnożą śię dni twoje / y przedłużą śię látá żywotá.
Będźieƺli mądrym / ſobie będźieƺ mądrym ; á jeźli náśmiewcą / ty ſám ƺkodę odnieśieƺ.
Niewiáſtá głupiá świegotliwá jeſt / proſtácżká / y nic nieumiejącá ;
A śiedźi u drzwi domu ſwego ná ſtołku / ná miejſcách wyſokich w mieśćie /
Aby wołáłá ná idących drogą / którzy proſto idą śćieƺkámi ſwemi / mówiąc :
Ktokolwiek jeſt proſtákiem / wſtąp ſám ; á do głupiego mówi :
Wody krádźione ſłodƺe ſą / á chleb pokątny ſmácżniejƺy.
Ale proſták nie wie / że tám ſą umárli / á ći / których wezwáłá / ſą w głębokośćiách grobu.
