Lepƺy jeſt ubogi / który chodźi w uprzejmośći ſwej / niżeli przewrotny w wárgách ſwojich / który jeſt głupim.
Záiſte duƺy bez umiejętnośći nie dobrze / á kto jeſt prędkich nóg / potknie śię.
Głupſtwo cżłowiecże podwrácá drogę jego / á przećie przećiwko Pánu zápálá śię gniewem ſerce jego.
Bogáctwá przycżyniáją wiele przyjáćiół ; ále ubogi od przyjáćielá ſwego odłącżony bywá.
Fáłƺywy świádek nie będźie bez pomſty ; á kto mówi kłámſtwo / nie ujdźie.
Wielu śię ich uniżá przed kśięćiem / á káżdy jeſt przyjáćielem mężowi ƺcżodremu.
Wƺyſcy bráćia ubogiego nienáwidzą go ; dáleko więcey inni przyjáćiele jego oddáláją śię od niego ; wołá zá nimi / á niemáƺ ich.
Nábywá rozumu / kto miłuje duƺę ſwoję / á ſtrzeże roztropnośći / áby ználázł co dobrego.
Świádek fáłƺywy nie będźie bez pomſty ; á kto mówi kłámſtwo / zginie.
Nie przyſtoi głupiemu rozkoƺ / áni ſłudze pánowáć nád kśiążętámi.
Rozum cżłowiecży záwśćiągá gniew jego / á ozdobá jego jeſt mijáć przeſtępſtwo.
Zápálcżywość królewſká jeſt jáko ryk lwięćiá ; ále łáſká jego jeſt jáko roſá ná tráwie.
Syn głupi jeſt utrapieniem ojcu ſwemu / á żoná ſwárliwá jeſt jáko uſtáwicżne kápánie przez dách.
Dom y májętność dźiedźictwem przypádá po rodźicách ; ále żoná roztropná jeſt od Páná.
Leniſtwo przywodźi twárdy ſen / á duƺá gnuśná będźie łáknęłá.
Kto ſtrzeże przykázánia / ſtrzeże duƺy ſwojey ; ále kto gárdźi drogámi ſwemi / zginie.
Pánu pożycżá / kto ma litość nád ubogim / á on mu zá dobrodźiejſtwo jego oddá.
Kárz ſyná ſwego / póki o nim nádźiejá / á zábiegájąc zginieniu jego niech mu nie folguje duƺá twojá.
Wielki gniew okázuj / kiedy odpuƺcżáƺ káránie / grożąc mu / poniewáż odpuƺcżáƺ / że potem ſrożey káráć będźieƺ.
Słucháj rády / á przyjmuj kárność / ábyś kiedyżkolwiek był mądrym.
Wiele jeſt myśli w ſercu cżłowiecżem ; ále rádá Páńſká / tá śię oſtoi.
Pożądáná rzecż cżłowiekowi dobrocżynność jego / ále lepƺy jeſt ubogi / niż mąż kłámliwy.
Bojáżń Páńſká prowádźi do żywotá / á kto ją ma / w obfitośći mieƺká / y nie ſpotká go nieƺcżęśćie.
Leniwy kryje rękę ſwą pod páchę / y do uſt ſwych nie podnośi jey.
Bij náśmiewcę / żeby proſták był oſtrożniejƺym ; á roztropnego ſfukáj / żeby zrozumiał umiejętność.
Syn wſtyd y háńbę zádáwájący / ojcá gubi y mátkę wygánia.
Synu mój! przeſtáń ſłucháć náuki / któráby ćię odwodźiłá od mów rozumnych.
Świádek złośliwy pośmiewá śię z ſądu / á uſtá niezbożnych połykáją niepráwość.
Sądy ſą ná pośmiewców zgotowáne / á guzy ná grzbiet głupich.
