Nie náśláduj ludźi złych / áni żądáj przebywáć z nimi ;
Abowiem ſerce ich myśli o drápieſtwie / á wárgi ich mówią o ućiśnieniu.
Mądrośćią bywa dom zbudowány / á roztropnośćią umocniony.
Zájiſte przez umiejętność komory nápełnione bywáją wƺelákiemi bogáctwámi koƺtownemi y wdźięcżnemi.
Cżłowiek mądry mocny jeſt / á mąż umiejętny przydáje śiły.
Abowiem przez mądrą rádę zwiedźieƺ bitwę / á wybáwienie przez mnóſtwo rádców mieć będźieƺ.
Wyſokie ſą głupiemu mądrośći ; w bramie nie otworzy uſt ſwojich.
Kto myśli źle cżynić / tego złośliwym zwáć będą.
Złá myśl głupiego jeſt grzechem / á pośmiewcá jeſt obrzydliwośćią ludzką.
Jeſli będźieƺ gnuśnym / tedy w dźień ućiſku ſłábá będźie śiłá twojá.
Wybáwiáj pojmánych ná śmierć ; á od tych / którzy idą ná ſtrácenie / nie odwrácáj śię.
Jeſli rzecżeƺ : Otośmy o tym nie wiedźieli ; izali ten / który wáży ſercá / nie rozumie? á ten / który ſtrzeże duƺy twojey / nie rozezná? y nie oddá cżłowiekowi według ucżynków jego?
Jedz miód / ſynu mój! bo dobry / y pláſtr ſłodki podniebieniu twemu ;
Ták umiejętność mądrośći duƺy twojey / jeſliże ją znájdźieƺ ; onáć będźie nágrodą / á nádźiejá twojá nie będźie wyćiętá.
Nie cżyń záſádzki / niezbożniku! ná przybytek ſpráwiedliwego / á nie przeƺkádzáj odpocżnieniu jego.
Bo choć śiedm kroć upádá ſpráwiedliwy / przećie záś powſtáje ; ále niezbożni wpádną w nieƺcżęśćie.
Gdy upádnie nieprzyjáćiel twój / nie ćieƺ śię ; y gdy śię potknie / niech śię nie ráduje ſerce twoje ;
Aby ſnadź nie ujrzał Pán / á nie podobáłoby śię to w ocżách jego / y odwróćiłby od niego gniew ſwój ná ćię.
Nie gniewáj śię dla złośników / áni śię udáwáj zá niepobożnymi ;
Boć nie weźmie złośnik nágrody ; pochodniá niepobożnych zgáśnie.
Synu mój! bój śię Páná y królá / á z nieſtátecżnymi nie mięƺáj śię ;
Boć znágłá powſtánie zginienie ich / á upádek obydwóch któż wie?
Y toć też mądrym náleży : wzgląd mieć ná oſobę u ſądu / nie dobrá.
Tego / który mówi niepobożnemu ; Jeſteś ſpráwiedliwy / będą ludźie przeklináć / á narody śię nim brzydźić będą.
Ale którzy go kárzą / ƺcżęśliwi będą / á przydźie ná nich błogoſłáwieńſtwo káżdego dobrego.
Pocáłują wárgi tego / co mówi ſłowá práwdźiwe.
Rozrządź ná polu robotę twoję / á ſpráwuj pilnie rolę ſwoję ; á potem będźieƺ budował dom twój.
Nie bądź świádkiem lekkomyślnym przećiw bliźniemu ſwemu / áni cżyń łágodnych námów wárgámi ſwemi.
Nie mów : Jáko mi ucżynił / ták mu ucżynię ; oddam mężowi temu według ucżynku jego.
Szedłem przez pole mężá leniwego á przez winnicę cżłowieká głupiego ;
A oto poroſłá wƺędźie oſtem ; pokrzywy wƺyſtko pokryły / á płot kámienny jey rozwálił śię.
Co ja ujrzáwƺy złożyłem to do ſercá mego / á widząc to wźiąłem to ku przeſtrodze.
Trochę śię prześpiƺ / trochę podrzemieƺ / trochę złożyƺ ręce / ábyś odpocżywał ;
A wtem ubóſtwo twoje przydźie jáko podróżny / á niedoſtátek twój jáko mąż zbrojny.
