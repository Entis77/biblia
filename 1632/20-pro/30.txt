Te ſą ſłowá Agurá / ſyná Jákiego / y zebránie mów tegoż mężá do Ityjelá / do Ityjelá y Uchálá.
Záiſtem jeſt głupƺy nád innych / á rozumu cżłowiecżego nie mam.
Y nie náucżyłem śię mądrośći / á umiejętnośći świętych nie umiem.
Któż wſtąpił ná niebo / y záśię zſtąpił? któż zgromádźił wiátr do gárśći ſwych? Któż zágárnął wody do ƺáty ſwej? któż utwierdźił wƺyſtkie końcżyny źiemi? Cóż zá imię jego? y co zá imię ſyná jego? Wieƺże?
Wƺelká mowá Bożá jeſt cżyſtá ; on jeſt tárcżą tym / którzy ufáją w nim.
Nie przydáwáj do ſłów jego / áby ćię nie káráł / á byłbyś ználeźiony w kłámſtwie.
Dwóch rzecży żądám od ćiebie / nie odmáwiájże mi pierwej niż umrę.
Márność y ſłowo kłámliwe oddál odemnie ; ubóſtwá y bogáctwá nie dáwáj mi ; żyw mię tylko pokármem według potrzeby mojey ;
Abym ſnáć náſyconym będąc nie záprzáł śię ćiebie / y nie rzekł : Któż jeſt Pán? Albo zubożáwƺy żebym nie krádł / y nie bráł nádáremno imieniá Bogá mego.
Nie podwodź ná ſługę przed Pánem jego / być ſnáć nie złorzecżył / á ty ábyś nie zgrzeƺył.
Jeſt rodzáj / który ojcu ſwemu złorzecży / á mátce ſwojey nie błogoſłáwi.
Jeſt rodzáj / który śię zdá ſobie być cżyſty / choć od plugáſtwá ſwego nie jeſt omyty.
Jeſt rodzáj / którego ſą wynioſłe ocży / y powieki jego wywyżƺone ſą.
Jeſt rodzáj / którego zęby ſą jáko miecże / á trzonowe zęby jego jáko noże ná pożárćie ubogich ná źiemi / á nędzników między ludźmi.
Pijáwká má dwie córki / które mówią : Przynieś / przynieś. Trzy rzecży ſą / które nie bywáją náſycone / owƺem cżtery / które nie mówią : Doſyć.
Grób / y żywot niepłodny / źiemiá też nie bywá náſyconá wodą / á ogień nie mówi : Doſyć.
Oko / które śię náśmiewá z ojcá / y wzgárdzá poſłuƺeńſtwem máćierzyńſkiem / wykłują kruki u potoków / y orlętá je wyjedzą.
Te trzy rzecży ſą ukryte przedemną / owƺem cżtery / których nie wiem :
Drogi orlej ná powietrzu / drogi wężowej ná ſkále / drogi okrętowej w pośród morzá / y drogi mężowej z pánną.
Tákáć jeſt drogá niewiáſty cudzołożącey : je / á ućierá uſtá ſwoje / y mówi : Nie popełniłám złego ucżynku.
Dla trzech rzecży poruƺá śię źiemiá / owƺem dla cżterech / których znieść nie może :
Dla ſługi / kiedy pánuje / y dla głupiego / kiedy śię náſyći chlebá ;
Dla przemierzłej niewiáſty / kiedy zá mąż idźie / y dla dźiewki / kiedy dźiedźicży po páni ſwojey.
Teć ſą cżtery rzecży nájmniejƺe ná źiemi / wƺákże ſą mędrƺe nád mędrców :
Mrówki / huf ſłáby / które ſobie jednák w lećie gotują pokárm ſwój ;
Króliki / twór ſłáby / którzy jednák budują w ſkále dom ſwój ;
Száráńcże królá nie máją / á wƺákże wƺyſtkie hufámi wychodzą ;
Pájąk rękomá robi / á bywá w páłácách królewſkich.
Te trzy rzecży ſą / które wſpániále chodzą / owƺem cżtery / które zmężyle chodzą :
Lew nájmocniejƺy między zwierzętámi / który przed nikim nie uſtępuje :
Koń ná biodrách przepáſány / y koźieł / y król / przećiw któremu żáden nie powſtáje.
Jeſliś głupio ucżynił / gdyś śię wynośił / álbo jeſliś źle myślił / połóżże rękę ná uſtá.
Kto tłucże śmietánę / wybijá máſło ; á kto bárdzo noſ wyćierá / wyćiſká krew ; ták kto wzbudzá gniew / wƺcżyná zwádę.
