Teć też ſą przypowieśći Sálomonowe / które zebráli mężowie Ezechyjáƺá / królá Judzkiego.
Słáwá to Bożá / táić ſpráwę ; ále ſłáwá królów / wywiádywáć śię rzecży.
Wyſokość niebioſ / y głębokość źiemi / y ſerce królów nie ſą dośćignione.
Odejm zużelicę od ſrebrá / á wynijdźie odlewájącemu nácżynie koƺtowne.
Odejm niezbożniká od oblicżá królewſkiego / á umocni śię w ſpráwiedliwośći ſtolicá jego.
Nie udáwáj śię zá zácnego przed królem / á ná miejſcu wielmożnych nie ſtáwáj ;
Bo lepiej jeſt / iż ći rzeką : Poſtąp ſám : á niżeliby ćię zniżyć miáno przed kśięćiem ; co widują ocży twoje.
Nie pokwápiáj śię do ſwáru / byś ſnáć ná oſtátek nie wiedźiáł / co máƺ cżynić / gdyby ćię záwſtydźił bliźni twój.
Prowádź do końcá ſpráwę ſwoję z przyjáćielem twoim / á tájemnicy drugiego nie objáwiáj ;
By ćię ſnáć nie zelżył ten / co ćię ſłuchá / á nieſłáwá twojá zoſtáłáby ná tobie.
Jákie jeſt jábłko złote z wyrzezániem ſrebrnem / tákieć jeſt ſłowo do rzecży powiedźiáne.
Ten / który mądrze nápominá / jeſt u tego / co ſłuchá / jáko náuƺnicá złotá / y klejnot z ƺcżerego złotá.
Jáko źimno śnieżne cżáſu żniwá : ták poſeł wierny tym / którzy go poſyłáją ; bo duƺe pánów ſwych ochłádzá.
Cżłowiek / który śię chlubi dárem zmyślonym / jeſt jáko wiátr y obłoki bez deƺcżu.
Kśiążę ćierpliwośćią bywá zmiękcżony / á język łágodny kośći łámie.
Znájdźieƺli miód / jedzże / ileć potrzebá / by ſnáć objádłƺy śię go nie zwróćił.
Powśćiągnij nogę twoję od domu bliźniego twego / by ſnáć będąc ćiebie ſyt / nie miał ćię w nienáwiśći.
Káżdy / kto mówi fáłƺywe świádectwo przećiw bliźniemu ſwemu / jeſt jáko młot / y miecż / y ſtrzáłá oſtrá.
Ufność w cżłowieku przewrotnym jeſt w dźień ućiſku jáko ząb wyłámány y nogá wywinioná.
Jáko ten / który zewłocży odźienie cżáſu źimy / álbo leje ocet ná ſáletrę / táki jeſt ten / który śpiewá pieśni ſercu ſmutnemu.
Jeſliżeby łáknął ten / co ćię nienáwidźi / nákárm go chlebem ; á jeſliby prágnął / dáj mu śię nápić wody ;
Bo węgle rozpálone zgromádźiƺ ná głowę jego / á Pán ći nágrodźi.
Jáko wiátr północny deƺcż przynośi : ták przynośi twárz gniewliwą język uƺcżypliwy.
Lepiej mieƺkáć w kąćie pod dáchem / niżeli z żoną ſwárliwą w domu przeſtronnym.
Jáko wodá chłodná duƺy prágnącey : ták wieść dobrá z źiemi dálekiej.
Jáko źródło nogámi pomącone / álbo zdrój zepſuty : ták ſpráwiedliwy / który upádá przed niezbożnym.
Jáko jeść wiele miodu nie jeſt rzecż dobrá : ták ƺukánie włáſney ſłáwy jeſt nieſłáwne.
Mąż / który nie má mocy nád duchem ſwoim / jeſt jáko miáſto rozwálone bez muru.
