Gdy śiądźieƺ / ábyś jádł z pánem / uwáżáj pilnie / kto jeſt przed tobą ;
Inácżej wráźiłbyś nóż w gárdło ſwoje / jeſlibyś był chćiwy pokármu.
Nie prágnij łákoći jego ; bo ſą pokármem obłudnym.
Nie ſtáráj śię / ábyś śię zbogáćił ; owƺem / zániecháj opátrznośći twojey.
Y miáłżebyś obróćić ocży twoje ná bogáctwo / które prędko niƺcżeje? bo ſobie ucżyni ſkrzydłá podobne orlim / y uleći do niebá.
Nie jedz chlebá cżłowieká zázdroſnego / á nie żądáj łákoći jego.
Abowiem jáko on ćiebie wáży w myśli ſwej / ták ty wáż pokárm jego. Mówić : Jedz y pij / ále ſerce jego nie jeſt z tobą.
Sztucżkę twoję / którąś zjádł / zwróćiƺ / á utráćiƺ wdźięcżne ſłowá twoje.
Przed głupim nie mów ; ábowiem wzgárdźi roztropnośćią powieśći twoich.
Nie przenoś gránicy ſtárej / á ná rolę śierotek nie wchodź.
Bo obrońcá ich możny ; onći śię podejmuje ſpráwy ich przećiwko tobie.
Obróć do náuki ſerce twoje / á uƺy twoje do powieśći umiejętnośći.
Nie odejmuj od młodego kárnośći ; bo jeſli go ubijeƺ rózgą / nie umrze.
Ty go bij rózgą / á duƺę jego z piekłá wyrwieƺ.
Synu mój! będźieli mądre ſerce twoje / będźie śię weſeliło ſerce moje / ſerce moje we mnie ;
Y rozweſelą śię nerki moje / gdy będą mówiły wárgi twoje / co jeſt práwego.
Niech nie zájrzy ſerce twoje grzeƺnikom ; ále rácżej chodż w bojáźni Páńſkiej ná káżdy dźień ;
Bo iż jeſt zápłátá / przeto nádźiejá twojá nie będźie wykorzenioná.
Słucháj / ſynu mój! á bądź mądry / y náwiedź ná drogę ſerce twoje.
Nie bywáj między pijánicámi winá / áni między żárłokámi mięſá ;
Boć pijánicá y żárłok zubożeje / á oſpáły w łátách chodźić będźie.
Słucháj ojcá twego / który ćię ſpłodźił / á nie pogárdzáj mátką twoją / gdy śię zſtárzeje.
Kupuj práwdę / á nie ſprzedáwáj jey ; kupuj mądrość / umiejętność y rozum.
Bárdzo śię ráduje ojćiec ſpráwiedliwego / á kto ſpłodźił mądrego / weſeli śię z niego.
Niech śię tedy weſeli ojćiec twój / y mátká twojá ; y niech śię rozráduje rodźićielká twojá.
Synu mój! dáj mi ſerce twoje / á ocży twoje niecháj ſtrzegą dróg moich.
Bo nierządnicá jeſt dół głęboki / á cudzá żoná jeſt ſtudniá ćiáſná.
Oná też jáko zbojcá záſádzki cżyni / á zuchwálców między ludźmi rozmnáżá.
Komu biádá? Komu nieſtety? Komu zwády? Komu krzyk? Komu rány dáremne? Komu zápálenie ocżów?
Tym / którzy śiádáją ná winie ; tym / którzy chodzą / ƺukájąc przypráwnego winá.
Nie zápátruj śię ná wino / gdy śię rumieni / y gdy wydáje w kubku łunę ſwoję / á proſto wyſkákuje.
Bo ná koniec jáko wąż ukąśi / á jáko żmijá uƺcżknie ;
Ocży twoje pátrzyć będą ná cudze żony / á ſerce twe będźie mówiło przewrotnośći ;
Y będźieƺ jáko ten / który leży w pośród morzá / á jáko ten / który śpi ná wierzchu máƺtu ;
Rzecżeƺ : Ubito mię / á nie ſtękáłem / potłucżono mię / á nie cżułem. Gdy śię ocucę / udám śię záś do tego.
