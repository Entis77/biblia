Nie chlub śię ze dniá jutrzejƺego ; bo nie wieƺ / coć przynieśie dźień dźiśiejƺy.
Niecháj ćię kto inny chwáli / á nie uſtá twoje ; obcy / á nie wárgi twoje.
Ciężkić jeſt kámień / y piáſek wáżny ; ále gniew głupiego ćiężƺy / niż to oboje.
Okrutnyć jeſt gniew / y nágłá popędliwość ; ále przed zázdrośćią któż śię oſtoi?
Lepƺá jeſt przymówká jáwná / niżeli miłość ſkrytá.
Lepƺe ſą rány od przyjáćielá / niż łágodne cáłowánie cżłowieká nienáwidzącego.
Duƺá náſyconá y pláſtr miodu podepcże ; ále głodnej duƺy y gorzkość wƺeláká ſłodká.
Jáko pták odlátuje od gniázdá ſwojego : ták cżłowiek odchodźi od miejſcá ſwego.
Jáko máść y kádzenie uweſelá ſerce : ták ſłodkość przyjáćielá uweſelá więcej / niż włáſná rádá.
Przyjáćielá twego / y przyjáćielá ojcá twego nie opuƺcżáj / á do domu brátá twego nie wchodź w dźień utrápieniá twego ; bo lepƺy ſąśiád bliſki / niż brát dáleki.
Bądź mądrym / ſynu mój! á uweſeláj ſerce moje / ábym miáł co odpowiedźieć temu / któryby mi urągáł.
Oſtrożny / upátrując złe / ukrywá śię ; ále proſták wproſt idąc / w ƺkodę popádá.
Weźmij ƺátę tego / któryć ręcżył zá obcego ; á od tego / który ręcżył zá cudzoźiemkę / weźmij záſtáw jego.
Temu / który przyjáćielowi ſwemu wielkim głoſem ráno wſtáwájąc błogoſłáwi / pocżytáne to będźie zá przeklęſtwo.
Kápánie uſtáwicżne w dźień gwáłtownego deƺcżu / y żoná ſwárliwá ſą ſobie podobni ;
Kto ją kryje / kryje wiátr / á wonią wydá ; jáko olejek wonny w práwej ręce jego.
Żelázo żelázem bywá náoſtrzone ; ták mąż záoſtrzá oblicże przyjáćielá ſwego.
Kto ſtrzeże drzewá figowego / pożywá owocu jego ; tákże kto ſtrzeże páná ſwego / ucżcżony będźie.
Jáko śię w wodźie twárz przećiwko twárzy ukázuje : ták ſerce cżłowiecże przećiw cżłowiekowi.
Piekło y zátrácenie nie mogą być náſycone ; tákże y ocży ludzkie náſyćić śię nie mogą.
Tygiel ſrebrá á piec złotá doświádcżá / á cżłowieká wieść ſłáwy jego.
Choćbyś głupiego y w ſtępie między krupámi ſtąporem ſtłukł / nie odejdźie od niego głupſtwo jego.
Doglądáj pilnie dobytku twego / á miej piecżę o trzodách twoich.
Boć nie ná wieki trwá bogáctwo / áni koroná od národu do národu.
Gdy wzráſtá tráwá / á ukázują śię źiołá / tedy z gór śiáno zbieráją.
Owce będą ná ƺáty twoje / á nágrodą polną kozły.
Nádto doſtátek mleká koźiego ná pokárm twój / ná pokárm domu twego / y ná pożywienie dźiewek twoich.
