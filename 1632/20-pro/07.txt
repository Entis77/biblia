Synu mój! ſtrzeż ſłów moich / á przykázánie moje chowáj u śiebie.
Strzeż przykázáń moich / á żyć będzieƺ ; á náuki mojej / jáko źrenicy ocżów ſwych.
Uwiąż je ná pálcách twoich / nápiƺ je ná táblicy ſercá twego.
Mów mądrośći : Sioſtráś ty mojá / á roztropność przyjáćiółką názywáj /
Aby ćię ſtrzegły od żony cudzej / y od obcej / którá mówi łágodne ſłowá.
Bom oknem domu ſwego przez krátę moję wyglądáł ;
Y widziáłem między proſtákámi / obácżyłem między ſynámi młodzieńcá głupiego /
Który ƺedł ulicą przy rogu jej / drogą poſtępując ku domowi jej.
Ze zmierzkiem pod wiecżór / w ćiemnośći nocnej / y w mroku.
A oto niewiáſtá ſpotkáłá go / w ubiorze wƺetecżnicy / chytrego ſercá /
Świegotliwá y nieukróconá / á w domu włáſnym nie mogły śię oſtáć nogi jej ;
Ráz ná dworzu / ráz ná ulicách y po wƺyſtkich kątách záſádzki cżyniącá ;
Y uchwyćiłá go / y pocáłowáłá go / á złożywƺy wſtyd z twárzy ſwojej / rzekłá mu :
Ofiáry ſpokojne ſą u mnie ; dziśiájm oddáłá śluby moje.
Przetożem wyƺłá przećiw tobie / ábym pilnie ƺukáłá twárzy twojej / y ználázłám ćię.
Obiłám kobiercámi łoże moje / ozdobione rzezániem y prześćierádłámi egipſkiemi.
Potrząſnęłám pokój ſwój myrrą / áloeſem / y cynámonem.
Pójdźże / opójmy śię miłośćią áż do poránku / ućieƺmy śię miłośćią.
Boć mężá mego w domu niemáƺ ; pojecháł w drogę dáleką.
Worek pieniędzy wziął z ſobą ; dniá pewnego wróći śię do domu ſwego.
Y nákłoniłá go wielą ſłów ſwoich / á łágodnośćią wárg ſwoich zniewoliłá go.
Wnet poƺedł zá nią / jáko wół / gdy go ná rzeź wiodą / á jáko głupi do pętá / którem kárány bywá.
Y przebiłá ſtrzáłą wątrobę jego ; kwápił śię jáko pták do śidłá / nie wiedząc / iż je zgotowáno ná duƺę jego.
Przetoż teráz / ſynowie! ſłuchájćie mię / á bądźćie pilni powieśći uſt moich.
Niecháj śię nie uchylá zá drogámi jej ſerce twoje / áni śię tułáj po śćieƺkách jej.
Abowiem wielu zrániwƺy poráziłá / y mocárze wƺyſcy pozábijáni ſą od niej.
Dom jej jeſt jáko drogi piekielne / wiodące do gmáchów śmierći.
