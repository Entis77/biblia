Synu mój! ſtrzeż ſłów moich / á przykázánie moje chowáj u śiebie.
Strzeż przykázáń moich / á żyć będźieƺ ; á náuki mojey / jáko źrenicy ocżów ſwych.
Uwiąż je ná pálcách twoich / nápiƺ je ná táblicy ſercá twego.
Mów mądrośći : Sioſtráś ty mojá / á roztropność przyjáćiółką názywáj /
Aby ćię ſtrzegły od żony cudzej / y od obcey / która mówi łágodne ſłowá.
Bom oknem domu ſwego przez krátę moję wyglądáł ;
Y widźiáłem między proſtákámi / obácżyłem między ſynámi młodźieńcá głupiego /
Który ƺedł ulicą przy rogu jey / drogą poſtępując ku domowi jey.
Ze zmierzkiem pod wiecżór / w ćiemnośći nocney / y w mroku.
A oto niewiáſtá ſpotkáłá go / w ubiorze wƺetecżnicy / chytrego ſercá /
Świegotliwá y nieukróconá / á w domu włáſnym nie mogły śię oſtáć nogi jey ;
Ráz ná dworzu / ráz ná ulicách y po wƺyſtkich kątách záſádzki cżyniącá ;
Y uchwyćiłá go / y pocáłowáłá go / á złożywƺy wſtyd z twárzy ſwojey / rzekłá mu :
Ofiáry ſpokojne ſą u mnie ; dźiśiájm oddáłá śluby moje.
Przetożem wyƺłá przećiw tobie / ábym pilnie ƺukáłá twárzy twojey / y ználázłám ćię.
Obiłám kobiercámi łoże moje / ozdobione rzezániem y prześćierádłámi egipſkiemi.
Potrząſnęłám pokój ſwój myrrą / áloeſem / y cynámonem.
Pójdźże / opójmy śię miłośćią áż do poránku / ućieƺmy śię miłośćią.
Boć mężá mego w domu niemáƺ ; pojecháł w drogę dáleką.
Worek pieniędzy wźiął z ſobą ; dniá pewnego wróći śię do domu ſwego.
Y nákłoniłá go wielą ſłów ſwoich / á łágodnośćią wárg ſwoich zniewoliłá go.
Wnet poƺedł zá nią / jáko wół / gdy go ná rzeź wiodą / á jáko głupi do pętá / którem kárány bywá.
Y przebiłá ſtrzáłą wątrobę jego ; kwápił śię jáko pták do śidłá / nie wiedząc / iż je zgotowáno ná duƺę jego.
Przetoż teraz / ſynowie! ſłuchájćie mię / á bądźćie pilni powieśći uſt moich.
Niecháj śię nie uchylá zá drogámi jey ſerce twoje / áni śię tułáj po śćieƺkách jey.
Abowiem wielu zrániwƺy poráźiłá / y mocárze wƺyſcy pozábijáni ſą od niej.
Dom jey jeſt jáko drogi piekielne / wiodące do gmáchów śmierći.
