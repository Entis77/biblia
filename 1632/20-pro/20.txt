Wino cżyni pośmiewcę / á nápój mocny zwájcę ; przetoż káżdy / co śię w nim kochá / nie bywá mądrym.
Strách królewſki jeſt jáko ryk lwięćiá ; kto go rozgniewá / grzeƺy przećiwko duƺy ſwojej.
Ucżćiwá rzecż káżdemu / poprzeſtáć zwády ; ále głupim jeſt / co śię w nią wdáje.
Dlá źimná leniwy nie orze ; przetoż żebráć będźie we żniwá / ále nic nie otrzymá.
Rádá w ſercu mężá jeſt jáko wodá głęboká : jednák mąż rozumny nácżerpnie jej.
Więkƺá cżęść ludźi przechwálá śię ucżynnośćią ſwoją ; ále w ſámej rzecży / któż tákiego znájdźie?
Spráwiedliwy chodźi w uprzejmośći ſwojej ; błogoſłáwieni ſynowie jego po nim.
Król śiedząc ná ſtolicy ſądowej rozgániá ocżymá ſwemi wƺyſtko złe.
Któż rzecże : Ocżyśćiłem ſerce moje? cżyſtym jeſt od grzechu mego?
Dwojáki gwicht y dwojáká miárá / to oboje obrzydliwośćią jeſt Pánu.
Po zábáwách ſwych poznáne bywá y dźiećię / jeźli cżyſty y práwy ucżynek jego.
Ucho / które ſłyƺy / y oko / które widźi / Pán to oboje ucżynił.
Nie kocháj śię w ſpániu / byś ſnáć nie zubożáł / otwórz ocży ſwoje / á náſyćiƺ śię chlebem.
Złe to / złe to / mówi ten / co kupuje / á odƺedłƺy / áli śię chlubi.
Wárgi umiejętne ſą jáko złoto y obfitość pereł / y koƺtowne klejnoty.
Weźmij ƺátę tego / któryć ręcżył zá obcego ; á od tego / który ręcżył zá cudzoźiemkę / weźmij záſtáw jego.
Smácżny jeſt drugiemu chleb kłámſtwá ; ále potem piáſkiem nápełnione będą uſtá jego.
Myśli rádámi utwierdzáj / á wojnę prowádź opátrznie.
Kto objáwiá tájemnicę / zdrádliwie śię obchodźi ; przetoż z tymy / którzy pochlebiáją wárgámi ſwemi / nie miej towárzyſtwá.
Kto złorzecży ojcu ſwemu álbo mátce ſwojej / zgáśnie pochodniá jego w gęſtych ćiemnośćiách.
Dźiedźictwu prędko z pocżątku nábytemu náoſtátek błogoſłáwić nie będą.
Nie mów : Oddám złem. Ocżekuj ná Páná / á wybáwi ćię.
Obrzydliwośćią Pánu dwojáki gwicht / á ƺále fáłƺywe nie podobáją mu śię.
Od Páná bywáją ſpráwowáne drogi mężá ; ále cżłowiek jákoż zrozumie drogę jego?
Pożreć rzecż poświęconą / jeſt cżłowiekowi śidłem ; á poślubiwƺy co / tego záś ƺukáć / jákoby tego ujść.
Król mądry rozpráƺá niezbożnych / y przywodźi ná nich pomſtę.
Duƺá ludzká jeſt pochodnią Páńſką / którá doświádcżá wƺyſtkich ſkrytośći wnętrznych.
Miłośierdźie y práwdá królá ſtrzegą / á ſtolicá jego miłośierdźiem wſpártá bywá.
Ozdobá młodźieńców jeſt śiłá ich / á ſędźiwość pocżćiwośćią ſtárców.
Złemu ſą lekárſtwem śinośći rán / y rázy przenikájące do wnętrznośći żywotá jego.
