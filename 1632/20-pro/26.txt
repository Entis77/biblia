Jáko śnieg w lećie / y jáko deƺcż we żniwá ; ták głupiemu nie przyſtoi chwałá.
Jáko śię pták tám y ſám tułá / y jáko jáſkółká látá : ták przeklęſtwo niezáſłużone nie przydźie.
Bicż ná koniá / ogłów ná oſłá / á kij potrzebny jeſt ná grzbiet głupiego.
Nie odpowiádáj głupiemu według głupſtwá jego / ábyś mu y ty nie był podobny.
Odpowiedz głupiemu według głupſtwá jego / áby śię ſobie nie zdał być mądrym.
Jákoby nogi obćiął / ták śię bezpráwiá dopuƺcżá / kto śię głupiemu poſelſtwá powierzá.
Jáko nierówne ſą golenie u chromego : ták jeſt powieść w uſtách głupich.
Jáko kiedy kto przywiązuje kámień drogi do procy : ták cżyni ten / który ucżćiwość głupiemu wyrządzá.
Jáko ćiernie / gdy śię doſtáną w rękę pijánego : ták przypowieść jeſt w uſtách głupich.
Wielki Pán ſtworzył wƺyſtko / á dáje zápłátę głupiemu / dáje tákże zápłátę przeſtępcom.
Jáko pies wrácá śię do zwróceniá ſwego : ták głupi powtárzá głupſtwo ſwoje.
Ujrzyƺli cżłowieká / co śię ſobie zda być mądrym / nádźiejá o głupim lepƺá jeſt / niżeli o nim.
Leniwy mówi : Lew ná drodze / lew ná ulicách.
Jáko śię drzwi obrácáją ná záwiáſách ſwojich : ták leniwiec ná łóżku ſwojim.
Leniwiec rękę kryje do zánádrzy ſwych / á ćiężko mu jey podnośić do uſt ſwojich.
Leniwiec zda śię ſobie być mędrƺym / niżeli śiedm odpowiádájących z rozſądkiem.
Jákoby też pſá zá uƺy łápał / kto śię mimo idąc w cudzą zwádę wdáje.
Jáko ƺálony wypuƺcżá iſkry y ſtrzáły śmiertelne :
Ták jeſt káżdy / który podchodźi przyjáćielá ſwego / á mówi : Azám ja nie żártował?
Gdy nieſtáje drew / gáśnie ogień ; ták gdy nie będźie klátecżniká / ućichnie zwádá.
Jáko węgiel mártwy ſłuży do roznieceniá / y drwá do ogniá ; ták cżłowiek ſwárliwy do roznieceniá zwády.
Słowá obmówcy ſą jáko ſłowá zránionych ; á wƺákże przenikáją do wnętrznośći żywotá.
Wárgi nieprzyjáćielſkie y ſerce złe ſą jáko ſrebrná piáná / którą polewáją nacżynie gliniáne.
Ten / co kogo nienáwidźi / zá inƺego śię udáje wárgámi ſwemi ; ále w ſercu ſwojim myśli o zdrádźie.
Gdyć śię ochotnym mową ſwą ukázuje / nie wierz mu : bo śiedmioráká obrzydliwość jeſt w ſercu jego.
Nienáwiść zdrádliwie bywá pokrytá ; ále odkrytá bywá złość jey w zgromádzeniu.
Kto drugiemu dół kopie / wpádá weń ; á kto kámień tocży / ná niego śię obrácá.
Cżłowiek języká kłámliwego ma utrápienie w nienáwiśći / á uſtá łágodne przywodzą do upádku.
