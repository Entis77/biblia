Cżłowiek / który ná cżęſte káránie zátwárdzá kárk ſwój / nágle zniƺcżeje / y nie wſkórá.
Gdy śię rozmnáżáją ſpráwiedliwi / weſeli śię lud ; ále gdy pánuje bezbożnik / wzdychá lud.
Mąż / który miłuje mądrość / uweſelá ojcá ſwego ; ále kto chowá nierządnicę / tráći májętność.
Król ſądem źiemię utwierdzá ; ále mąż / który dáry bierze / podwrácá ją.
Cżłowiek / który pochlebiá przyjáćielowi ſwemu / rozćiągá śieć przed nogámi jego.
Wyſtępek złego jeſt mu śidłem ; ále ſpráwiedliwy śpiewá y weſeli śię.
Spráwiedliwy wyrozumiewá ſpráwę nędznych ; ále niezbożnik nie ma ná to rozumu y umiejętnośći.
Mężowie náśmiewcy záwodzą miáſto ; ále mądrzy odwrácáją gniew.
Mąż mądry / wiedźieli ſpór z mężem głupim / choćby śię gniewał / choćby śię też śmiał / nie będźie miał pokoju.
Mężowie krwáwi nienáwidzą uprzejmego ; ále uprzejmi ſtáránie wiodą o duƺę jego.
Wƺyſtkiego duchá ſwego wywierá głupi / ále mądry ná dálƺy cżás go záwśćiągá.
Páná / który rád ſłuchá ſłów kłámliwych / wƺyſcy ſłudzy jego ſą niepobożni.
Ubogi y zdźiercá ſpotkáli śię ; á wƺákże obydwóch ocży Pán oświecá.
Królá / który ſądźi ućiśnionych według prawdy / ſtolicá jego ná wieki utwierdzoná będźie.
Rózgá y kárność mądrość dáje ; ále dźiećię ſwáwolne záwſtydzá mátkę ſwoję.
Gdy śię rozmnáżáją niezbożni / rozmnáżá śię y przeſtępſtwo ; ále ſpráwiedliwi upádek ich oglądáją.
Kárz ſyná twego / á ſpráwić odpocżnienie / y ſpoſobi rozkoƺ duƺy twojey.
Gdy proroctwo uſtáje / lud bywá rozproƺony ; ále kto ſtrzeże zakonu / błogoſłáwiony jeſt.
Sługá nie bywá ſłowámi nápráwiony ; bo choć rozumie / jednák nie odpowiádá.
Ujrzyƺ cżłowieká ſkwápliwego w ſpráwách ſwojich ; ále lepƺá jeſt nádźiejá o głupim / niż o nim.
Kto w rozkoƺy chowá z dźiećińſtwá ſługę ſwego / ná oſtátek będźie chćiał być zá ſyná.
Cżłowiek gniewliwy wƺcżyná zwádę / á pierzchliwy wiele grzeƺy.
Pychá cżłowiecżá poniżá go ; ále pokorny w duchu ſłáwy doſtępuje.
Kto ſpółkuje ze złodźiejem / ma w nienáwiśći duƺę ſwoję ; tákże też kto przeklęſtwá ſłyƺy / á nie objáwiá go.
Strách cżłowiecży ſtáwiá ſobie śidło ; ále kto ma nádźieję w Pánu / wywyżƺony będźie.
Wiele tych / co ƺukáją twárzy pánów ; áleć od Páná jeſt ſąd káżdego.
Spráwiedliwym jeſt mąż niezbożny obrzydliwośćią ; á záśię kto w uprzejmośći chodźi / jeſt niezbożnym obrzydliwośćią.
