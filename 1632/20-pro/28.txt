Ućiekáją niezbożni / choć ich nikt nie goni : ále ſpráwiedliwi jáko lwię młode ſą bez bojáźni.
Dlá przeſtępſtwá ziemi wiele bywá kśiążąt jej ; ále dlá cżłowieká roztropnego y umiejętnego trwáłe bywá páńſtwo.
Mąż ubogi / który ućiſká nędznych / podobny jeſt dżdżowi gwáłtownemu / po którym chlebá nie bywá.
Którzy opuƺcżáją zákon / chwálą niezbożniká ; ále ći / którzy ſtrzegą zákonu / ſą im odpornymi.
Ludzie źli nie zrozumiewáją ſądu ; ále którzy Páná ƺukáją / rozumieją wƺyſtko.
Lepƺy jeſt ubogi / który chodzi w uprzejmośći ſwojej / niżeli przewrotny ná drogách ſwych / choćiáż jeſt bogáty.
Kto ſtrzeże zákonu / jeſt ſynem roztropnym ; ále kto kármi obżercę / cżyni zelżywość ojcu ſwemu.
Kto rozmnáżá májętność ſwoję z lichwy y z płátu / temu ją zbierá / który ubogiemu ƺcżodrze będzie dáwáł.
Kto odwrácá ucho ſwe / áby nie ſłucháł zákonu / y modlitwá jego jeſt obrzydliwośćią.
Kto záwodzi uprzejmych ná drogę złą / w dół ſwój ſám wpádnie ; ále uprzejmi odziedzicżą rzecży dobre.
Mąż bogáty zdá śię ſobie być mądrym ; ále ubogi roztropny dochodzi go.
Gdy śię rádują ſpráwiedliwi / wielká jeſt ſłáwá ; ále gdy powſtáwáją niepobożni / kryje śię cżłowiek.
Kto pokrywá przeſtępſtwá ſwe / nie poƺcżęśći mu śię ; ále kto je wyznáje y opuƺcżá / miłośierdziá doſtąpi.
Błogoſłáwiony cżłowiek / który śię záwƺe boi ; ále kto zátwárdzá ſerce ſwoje / wpádá w złe.
Pán niezbożny / pánujący nád ludem ubogim / jeſt jáko lew rycżący / y jáko niedźwiedź głodny.
Kśiążę bezrozumny wielkim jeſt drápieżcą : ále kto nienáwidzi łákomſtwá / przedłuży dni ſwoich.
Cżłowieká / który gwáłt cżyni krwi ludzkiej / choćby y do dołu ućiekáł / nikt nie zátrzymá.
Kto chodzi w uprzejmośći / záchowány będzie ; ále przewrotny ná drogách ſwoich oráz upádnie.
Kto ſpráwuje ziemię ſwoję / chlebem náſycony bywá ; ále kto náśláduje próżnujących / ubóſtwem náſycony bywá.
Mąż wierny przycżyni błogoſłáwieńſtwá ; ále kto śię prędko chce zbogáćić / nie bywá bez winy.
Mieć wzgląd ná oſobę / rzecż niedobrá ; bo nie jeden dlá kęſá chlebá ſtáje śię przewrotnym.
Prędko chce cżłowiek zázdrośćiwy zbogátnieć / á nie wie / iż náń niedoſtátek przyjdzie.
Kto ſtrofuje cżłowieká / więkƺą potem łáſkę znájduje / niż ten / co pochlebiá językiem.
Kto łupi ojcá ſwego / álbo mátkę ſwoję / á mówi / iż to nie grzech : towárzyƺem jeſt mężobójcy.
Wyſokomyślny wƺcżyná zwádę ; ále kto nádzieję má w Pánu / doſtátek mieć będzie.
Kto ufá w ſercu ſwem / głupi jeſt ; ále kto ſobie mądrze pocżyná / ten ujdzie nieƺcżęśćiá.
Kto dáje ubogiemu / nie będzie miáł niedoſtátku ; ále kto od niego odwrácá ocży ſwe / wielkie przeklęſtwá náń przyjdą.
Gdy niepobożni powſtáwáją / kryje śię cżłowiek ; ále gdy giną / ſpráwiedliwi śię rozmnáżáją.
