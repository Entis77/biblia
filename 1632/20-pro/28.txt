Ućiekáją niezbożni / choć ich nikt nie goni : ále ſpráwiedliwi jáko lwię młode ſą bez bojáźni.
Dla przeſtępſtwá źiemi wiele bywá kśiążąt jey ; ále dla cżłowieká roztropnego y umiejętnego trwáłe bywá páńſtwo.
Mąż ubogi / który ućiſká nędznych / podobny jeſt dżdżowi gwáłtownemu / po którym chlebá nie bywá.
Którzy opuƺcżáją zakon / chwálą niezbożniká ; ále ći / którzy ſtrzegą zakonu / ſą im odpornymi.
Ludźie źli nie zrozumiewáją ſądu ; ále którzy Páná ƺukáją / rozumieją wƺyſtko.
Lepƺy jeſt ubogi / który chodźi w uprzejmośći ſwojey / niżeli przewrotny ná drogách ſwych / choćiáż jeſt bogáty.
Kto ſtrzeże zakonu / jeſt ſynem roztropnym ; ále kto kármi obżercę / cżyni zelżywość ojcu ſwemu.
Kto rozmnáżá májętność ſwoję z lichwy y z płátu / temu ją zbierá / który ubogiemu ƺcżodrze będźie dáwał.
Kto odwrácá ucho ſwe / áby nie ſłuchał zakonu / y modlitwá jego jeſt obrzydliwośćią.
Kto záwodźi uprzejmych ná drogę złą / w dół ſwój ſám wpádnie ; ále uprzejmi odźiedźicżą rzecży dobre.
Mąż bogáty zda śię ſobie być mądrym ; ále ubogi roztropny dochodźi go.
Gdy śię rádują ſpráwiedliwi / wielká jeſt ſłáwá ; ále gdy powſtáwáją niepobożni / kryje śię cżłowiek.
Kto pokrywá przeſtępſtwá ſwe / nie poƺcżęśći mu śię ; ále kto je wyznáje y opuƺcżá / miłośierdźiá doſtąpi.
Błogoſłáwiony cżłowiek / który śię záwƺe boi ; ále kto zátwárdzá ſerce ſwoje / wpádá w złe.
Pán niezbożny / pánujący nád ludem ubogim / jeſt jáko lew rycżący / y jáko niedźwiedź głodny.
Kśiążę bezrozumny wielkim jeſt drápieżcą : ále kto nienáwidźi łákomſtwá / przedłuży dni ſwojich.
Cżłowieká / który gwáłt cżyni krwi ludzkiej / choćby y do dołu ućiekał / nikt nie zátrzymá.
Kto chodźi w uprzejmośći / záchowány będźie ; ále przewrotny ná drogách ſwojich oráz upádnie.
Kto ſpráwuje źiemię ſwoję / chlebem náſycony bywá ; ále kto náśláduje próżnujących / ubóſtwem náſycony bywá.
Mąż wierny przycżyni błogoſłáwieńſtwá ; ále kto śię prędko chce zbogáćić / nie bywá bez winy.
Mieć wzgląd ná oſobę / rzecż niedobrá ; bo nie jeden dla kęſá chlebá ſtáje śię przewrotnym.
Prędko chce cżłowiek zázdrośćiwy zbogátnieć / á nie wie / iż nań niedoſtátek przydźie.
Kto ſtrofuje cżłowieká / więkƺą potem łáſkę znájduje / niż ten / co pochlebiá językiem.
Kto łupi ojcá ſwego / álbo mátkę ſwoję / á mówi / iż to nie grzech : towárzyƺem jeſt mężobójcy.
Wyſokomyślny wƺcżyná zwádę ; ále kto nádźieję ma w Pánu / doſtátek mieć będźie.
Kto ufá w ſercu ſwem / głupi jeſt ; ále kto ſobie mądrze pocżyná / ten ujdźie nieƺcżęśćiá.
Kto dáje ubogiemu / nie będźie miał niedoſtátku ; ále kto od niego odwrácá ocży ſwe / wielkie przeklęſtwá nań przyjdą.
Gdy niepobożni powſtáwáją / kryje śię cżłowiek ; ále gdy giną / ſpráwiedliwi śię rozmnáżáją.
