Te ſą ſłowá Lemuelá królá / y zebránie mów / któremi go ćwicżyłá mátká jego.
Cóż rzekę / ſynu mój? cóż rzekę / ſynu żywotá mego? y cóż rzekę / ſynu ślubów moich?
Nie dáwáj niewiáſtom śiły twojey / áni dróg twoich tym / którzy do zginieniá królów przywodzą.
Nie królom / o Lemuelu! nie królom náleży pić wino / á nie pánom / báwić śię nápojem mocnym ;
By ſnáć pijąc nie zápomniáł ná uſtáwy / á nie odmienił ſpráw wƺyſtkich ludźi ućiśnionych.
Dájćie nápój mocny ginącemu / á wino tym / którzy ſą duchá ſfráſowánego.
Niech śię nápije / á zápomni ubóſtwá ſwego / á ná utrápienie ſwoje niech więcey nie wſpomni.
Otwórz uſtá ſwe zá niemym w ſpráwie wƺyſtkich oſądzonych ná śmierć.
Otwórz uſtá ſwe / ſądź ſpráwiedliwie / á podejmij śię ſpráwy ubogiego y nędznego.
Któż znájdźie niewiáſtę ſtátecżną / gdyż nád perły dáleko więkƺá jeſt cená jey?
Serce mężá jey ufá jey / á ná korzyśćiách ſchodźić mu nie będźie.
Dobrze mu cżyni / á nie źle / po wƺyſtkie dni żywotá ſwego.
Szuká wełny y lnu / á prácuje ochotnie rękomá ſwemi.
Podobná jeſt okrętom kupieckim ; z dáleká przywodźi żywność ſwoję.
Y wſtáje bárdzo ráno / á dáje pokárm cżeládźi ſwej / á obrok ſłuƺny dźiewkom ſwym.
Obmyślá rolę / y ujmuje ją ; z zárobku rąk ſwoich ƺcżepi winnice.
Przepáſuje mocą biodrá ſwe / á pośilá rámioná ſwoje.
Doświádcżá / że jeſt dobrá ſkrzętność jey / á nie gáśnie w nocy pochodniá jey.
Ręce ſwoje obrácá do kądźieli / á pálcámi ſwemi trzymá wrzećiono.
Rękę ſwą otwierá ubogiemu / á ręce ſwoje wyćiągá ku nędznemu.
Nie boi śię o cżeládź ſwoję cżáſu śniegu ; ábowiem wƺyſtká cżeládź jey obłocży śię w ƺátę dwoiſtą.
Kobierce ſobie robi ; płótno ſubtelne y ƺárłát jeſt odźieniem jey.
Znácżny jeſt w brámách mąż jey / gdy śiedźi między ſtárƺymi źiemi.
Płótno robi / y ſprzedáje ; tákże páſy ſprzedáje kupcowi.
Moc y przyſtojność jeſt odźieniem jey ; nie fráſuje śię o cżáſy przyƺłe.
Mądrze otwierá uſtá ſwe / á náuká miłośierdźiá jeſt ná języku jey.
Doglądá rządu w domu ſwym / á chlebá próżnując nie je.
Powſtáwƺy ſynowie jey błogoſłáwią jey ; tákże y mąż jey chwáli ją /
Mówiąc : Wiele niewiáſt grzecżnie ſobie pocżynáły ; ále je ty przechodźiƺ wƺyſtkie.
Omylná jeſt wdźięcżność / y márná piękność ; ále niewiáſtá / która śię Páná boi / tá pochwáły godná.
Dájćie jey z owocu ręku jey / á niecháj ją chwálą w brámách ucżynki jey.
