Te ſą ſłowá Lemuelá królá / y zebránie mów / któremi go ćwicżyłá mátká jego.
Cóż rzekę / ſynu mój? cóż rzekę / ſynu żywotá mego? y cóż rzekę / ſynu ślubów moich?
Nie dáwáj niewiáſtom śiły twojej / áni dróg twoich tym / którzy do zginieniá królów przywodzą.
Nie królom / o Lemuelu! nie królom náleży pić wino / á nie pánom / báwić śię nápojem mocnym ;
By ſnáć pijąc nie zápomniáł ná uſtáwy / á nie odmienił ſpráw wƺyſtkich ludzi ućiśnionych.
Dájćie nápój mocny ginącemu / á wino tym / którzy ſą duchá ſfráſowánego.
Niech śię nápije / á zápomni ubóſtwá ſwego / á ná utrápienie ſwoje niech więcej nie wſpomni.
Otwórz uſtá ſwe zá niemym w ſpráwie wƺyſtkich oſądzonych ná śmierć.
Otwórz uſtá ſwe / ſądź ſpráwiedliwie / á podejmij śię ſpráwy ubogiego y nędznego.
Któż znájdzie niewiáſtę ſtátecżną / gdyż nád perły dáleko więkƺá jeſt cená jej?
Serce mężá jej ufá jej / á ná korzyśćiách ſchodzić mu nie będzie.
Dobrze mu cżyni / á nie źle / po wƺyſtkie dni żywotá ſwego.
Szuká wełny y lnu / á prácuje ochotnie rękomá ſwemi.
Podobná jeſt okrętom kupieckim ; z dáleká przywodzi żywność ſwoję.
Y wſtáje bárdzo ráno / á dáje pokárm cżeládzi ſwej / á obrok ſłuƺny dziewkom ſwym.
Obmyślá rolę / y ujmuje ją ; z zárobku rąk ſwoich ƺcżepi winnice.
Przepáſuje mocą biodrá ſwe / á pośilá rámioná ſwoje.
Doświádcżá / że jeſt dobrá ſkrzętność jej / á nie gáśnie w nocy pochodniá jej.
Ręce ſwoje obrácá do kądzieli / á pálcámi ſwemi trzymá wrzećiono.
Rękę ſwą otwierá ubogiemu / á ręce ſwoje wyćiągá ku nędznemu.
Nie boi śię o cżeládź ſwoję cżáſu śniegu ; ábowiem wƺyſtká cżeládź jej obłocży śię w ƺátę dwoiſtą.
Kobierce ſobie robi ; płótno ſubtelne y ƺárłát jeſt odzieniem jej.
Znácżny jeſt w brámách mąż jej / gdy śiedzi między ſtárƺymi ziemi.
Płótno robi / y ſprzedáje ; tákże páſy ſprzedáje kupcowi.
Moc y przyſtojność jeſt odzieniem jej ; nie fráſuje śię o cżáſy przyƺłe.
Mądrze otwierá uſtá ſwe / á náuká miłośierdziá jeſt ná języku jej.
Doglądá rządu w domu ſwym / á chlebá próżnując nie je.
Powſtáwƺy ſynowie jej błogoſłáwią jej ; tákże y mąż jej chwáli ją /
Mówiąc : Wiele niewiáſt grzecżnie ſobie pocżynáły ; ále je ty przechodziƺ wƺyſtkie.
Omylná jeſt wdzięcżność / y márná piękność ; ále niewiáſtá / którá śię Páná boi / tá pochwáły godná.
Dájćie jej z owocu ręku jej / á niecháj ją chwálą w brámách ucżynki jej.
