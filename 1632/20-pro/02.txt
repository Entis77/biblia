Synu mój! jeſli przyjmieƺ ſłowá moje / á przykázánie moje záchowáƺ u śiebie ;
Nádſtáwiƺli mądrośći uchá twego / y nákłoniƺli ſercá twego do roztropnośći ;
Owƺem / jeſli ná rozum záwołáƺ / á roztropnośći wezwieƺli głoſem ſwoim ;
Jeſli jey ƺukáć będźieƺ jáko ſrebrá / á jáko ſkárbów ſkrytych pilnie ƺukáć będźieƺ :
Tedy zrozumieƺ bojáźń Páńſką / á znájomość Bożą znájdźieƺ.
Abowiem Pán dáje mądrość / z uſt jego pochodźi umiejętność y roztropność.
On záchowuje uprzejmym práwdźiwą mądrość ; on jeſt tárcżą chodzącym w ƺcżerośći /
Aby ſtrzegli śćieżek ſądu ; on drogi świętych ſwoich ſtrzeże.
Tedy wyrozumieƺ ſpráwiedliwość / y ſąd / y práwość / y wƺelką śćieƺkę dobrą.
Gdy wnijdźie mądrość w ſerce twoje / á umiejętność duƺy twojey wdźięcżná będźie :
Tedy ćię oſtrożność ſtrzedz będźie / á opátrzność záchowá ćię.
Wyrywájąc ćię od drogi złej / y od cżłowieká mówiącego przewrotnośći ;
Od tych / którzy opuƺcżáją śćieƺki proſte / udáwájąc śię drogámi ćiemnemi ;
Którzy śię rádują / gdy cżynią złe / á weſelą śię w złośliwych przewrotnośćiách ;
Których śćieƺki ſą krzywe / á ſámi ſą przewrotnymi ná drogách ſwoich ;
Wyrywájąc ćię od niewiáſty poſtronney y obcey / która pochlebiá łágodnemi ſłowy ;
Która opuƺcżá wodzá młodośći ſwojey / á przymierzá Bogá ſwojego zápominá.
Bo śię náchylá ku śmierći dom jey / á do umárłych śćieƺki jey.
Wƺyſcy / którzy do niej wchodzą / nie wrácáją śię / áni tráfiáją ná śćieƺkę żywotá.
A przetoż będźieƺ chodźił drogą dobrych / á śćieżek ſpráwiedliwych będźieƺ przeſtrzegáł.
Abowiem cnotliwi będą mieƺkáli ná źiemi / á ƺcżerzy trwáć będą ná niej ;
Ale niepobożni z źiemi wykorzenieni będą / á przewrotni będą z niej wygłádzeni.
