Przypowieśći Sálomoná / ſyná Dawidowego / królá Izráelſkiego /
Dla poznánia mądrośći y ćwicżeniá / ku wyrozumieniu powieśći roztropnych ;
Dla pojęćiá ćwicżeniá w rozumie / w ſpráwiedliwośći / w ſądźie y w práwośći ;
Dla podánia proſtákom oſtrożnośći / młodemu umiejętnośći / y opátrznośći.
Tych gdy mądry ſłucháć będźie / przybędźie mu náuki / á roztropny w rádách opátrzniejƺy będźie /
Aby zrozumiał przypowieśći / y wykłády ich / ſłowá mądrych y zágádki ich.
Bojáźń Páńſka jeſt pocżątkiem umiejętnośći ; ále głupi mądrośćią y ćwicżeniem gárdzą.
Słucháj / ſynu mój! ćwicżeniá ojcá twego / á nie opuƺcżáj náuki mátki twojey.
Bo to przydá wdźięcżnośći głowie twojey / y będźie łáńcuchem koƺtownym ƺyi twojey.
Synu mój! jeſliby ćię námáwiáli grzeƺnicy / nie przyzwáláj.
Jeſlićby rzekli : Pójdź z námi / cżyhájmy ná krew / záſádźmy śię ná niewinnego bez przycżyny ;
Pożremyż ich żywo / jáko grób / á cáłkiem / jáko zſtępujących w dół ;
Wƺelkiej májętnośći koƺtowney nábędźiemy / nápełnimy domy náƺe korzyśćią ;
Rzuć miedzy nas los twój ; mieƺek jeden wƺyſcy mieć będźiemy.
Synu mój! nie chodźże z nimi w drogę ; záwśćiągnij nogi twojey od śćieżek ich.
Abowiem nogi ich ku złemu bieżą / y ſpieƺą śię ná wylánie krwi.
Bo jáko prózno záſtáwiáją śieći przed ocżymá wƺelkiego ptáká ſkrzydláſtego :
Ták y ći ná krew ſwoję cżyháją / á záſádzáją śię ná duƺę ſwoję.
Tákieć ſą śćieƺki káżdego cżyhájącego ná zyſk / który duƺę páná ſwego odbierá.
Mądrość ná dworzu wołá / głos ſwój ná ulicách wydáje.
W nájwiękƺym zgiełku wołá / u wrót bram / w miáſtách powieśći ſwoje opowiádá / mówiąc :
Proſtácy! dokądże śię kocháć będźiećie w proſtoćie? á náśmiewcy pośmiewiſko miłowáć będźiećie? á głupi nienáwidźieć umiejętnośći będźiećie?
Náwróććież śię ná káránie moje ; oto wam wydám duchá mojego / á podám wam do znájomośći ſłowá moje.
Poniewáżem wołáłá / á nie chćieliśćie ; wyćiągáłám rękę moję / á nie był / ktoby uwáżał ;
Owƺem odrzućiliśćie wƺyſtkę rádę moję / á kárnośći mojey nie chćieliśćie przyjąć ;
Przetoż ja w zginieniu wáƺem śmiáć śię będę / będę z was ƺydźiłá / gdy przydźie / cżego śię ſtrácháćie.
Gdy przydźie jáko ſpuſtoƺenie / cżego śię ſtrácháćie / y gdy zginienie wáƺe przypádnie jáko wicher / gdy przydźie ná was ućiſk y utrápienie ;
Tedy mię wzywáć będą / á nie wyſłuchám ; ƺukáć mię będą z poránku / á nie znájdą mię.
Przeto / iż mieli w nienáwiśći umiejętność / á bojáźni Páńſkiej nie obráli ſobie /
Ani przeſtáwáli ná rádźie mojey / ále gárdźili wƺelką kárnośćią moją :
Przetoż będą używáć owocu dróg ſwojich / á rádámi ſwemi náſyceni będą.
Bo odwrócenie proſtáków pozábijá ich / á ƺcżęśćie głupich wytráći ich.
Ale kto mię ſłuchá / bezpiecżnie mieƺkáć będźie / á będźie wolny od ſtráchu złych rzecży.
