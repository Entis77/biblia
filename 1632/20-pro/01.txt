Przypowieśći Sálomoná / ſyná Dawidowego / królá Izráelſkiego /
Dlá poznániá mądrośći y ćwicżeniá / ku wyrozumieniu powieśći roztropnych ;
Dlá pojęćiá ćwicżeniá w rozumie / w ſpráwiedliwośći / w ſądzie y w práwośći ;
Dlá podániá proſtákom oſtrożnośći / młodemu umiejętnośći / y opátrznośći.
Tych gdy mądry ſłucháć będzie / przybędzie mu náuki / á roztropny w rádách opátrzniejƺy będzie /
Aby zrozumiáł przypowieśći / y wykłády ich / ſłowá mądrych y zágádki ich.
Bojáźń Páńſká jeſt pocżątkiem umiejętnośći ; ále głupi mądrośćią y ćwicżeniem gárdzą.
Słucháj / ſynu mój! ćwicżeniá ojcá twego / á nie opuƺcżáj náuki mátki twojej.
Bo to przydá wdzięcżnośći głowie twojej / y będzie łáńcuchem koƺtownym ƺyi twojej.
Synu mój! jeźliby ćię námáwiáli grzeƺnicy / nie przyzwáláj.
Jeźlićby rzekli : Pójdź z námi / cżyhájmy ná krew / záſádźmy śię ná niewinnego bez przycżyny ;
Pożremyż ich żywo / jáko grób / á cáłkiem / jáko zſtępujących w dół ;
Wƺelkiej májętnośći koƺtownej nábędziemy / nápełnimy domy náƺe korzyśćią ;
Rzuć między náſ loſ twój ; mieƺek jeden wƺyſcy mieć będziemy.
Synu mój! nie chodźże z nimi w drogę ; záwśćiągnij nogi twojej od śćieżek ich.
Abowiem nogi ich ku złemu bieżą / y ſpieƺą śię ná wylánie krwi.
Bo jáko próżno záſtáwiáją śieći przed ocżymá wƺelkiego ptáká ſkrzydláſtego :
Ták y ći ná krew ſwoję cżyháją / á záſádzáją śię ná duƺę ſwoję.
Tákieć ſą śćieƺki káżdego cżyhájącego ná zyſk / który duƺę páná ſwego odbierá.
Mądrość ná dworzu wołá / głoſ ſwój ná ulicách wydáje.
W nájwiękƺym zgiełku wołá / u wrót brám / w miáſtách powieśći ſwoje opowiádá / mówiąc :
Proſtácy! dokądże śię kocháć będziećie w proſtoćie? á náśmiewcy pośmiewiſko miłowáć będziećie? á głupi nienáwidzieć umiejętnośći będziećie?
Náwróććież śię ná káránie moje ; oto wám wydám duchá mojego / á podám wám do znájomośći ſłowá moje.
Poniewáżem wołáłá / á nie chćieliśćie ; wyćiągáłám rękę moję / á nie był / ktoby uwáżáł ;
Owƺem odrzućiliśćie wƺyſtkę rádę moję / á kárnośći mojej nie chćieliśćie przyjąć ;
Przetoż já w zginieniu wáƺem śmiáć śię będę / będę z wáſ ƺydziłá / gdy przyjdzie / cżego śię ſtrácháćie.
Gdy przyjdzie jáko ſpuſtoƺenie / cżego śię ſtrácháćie / y gdy zginienie wáƺe przypádnie jáko wicher / gdy przyjdzie ná wáſ ućiſk y utrápienie ;
Tedy mię wzywáć będą / á nie nie wyſłuchám ; ƺukáć mię będą z poránku / á nie znájdą mię.
Przeto / iż mieli w nienáwiśći umiejętność / á bojáźni Páńſkiej nie obráli ſobie /
Ani przeſtáwáli ná rádzie mijej / ále gárdzili wƺelką kárnośćią moją :
Przetoż będą używáć owocu dróg ſwoich / á rádámi ſwemi náſyceni będą.
Bo odwrócenie proſtáków pozábijá ich / á ƺcżęśćie głupich wytráći ich.
Ale kto mię ſłuchá / bezpiecżnie mieƺkáć będzie / á będzie wolny od ſtráchu złych rzecży.
