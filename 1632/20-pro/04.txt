Słuchájćie ſynowie! ćwicżeniá ojcowſkiego / á pilnujćie / ábyśćie umieli roztropność ;
Abowiem wám náukę dobrą dáję ; zakonu mego nie opuƺcżájćie.
Gdym był młodźiuchnym ſynem u ojcá mego / y jedynákiem u mátki mojey /
On mię ucżył / powiádájąc mi : Niech śię chwyći powieśći moich ſerce twoje / ſtrzeż przytkázáń moich / á będźieƺ żył.
Nábywáj mądrośći / nábywáj roztropnośći ; nie zápomináj / áni śię uchyláj od powieśći uſt moich.
Nie opuƺcżáj jey / á będźie ćię ſtrzegłá ; rozmiłuj śię jey / á záchowá ćię.
Pocżątkiem wƺyſtkiego jeſt mądrość / nábywájże mądrośći / á zá wzyſtkę májętność twoję nábywáj roztropnośći.
Wywyżƺáj ją / á wywyżƺy ćię / rozſłáwi ćię / gdy ją przyjmieƺ.
Przydá głowie twojey wdźięcżnośći / koroną ozdoby obdárzy ćię.
Słucháj / ſynu mój! á przyjmij powieśći moje / á rozmnożąć śię látá żywotá.
Drogi mądrośći náucżám ćię ; po śćieƺkách proſtych wiodę ćię ;
Któremi gdy pójdźieƺ / nie będźie śćiśniony chód twój ; á jeſli pobieżyƺ / nie potknieƺ śię.
Przyjmij ćwicżenie / nie puƺcżáj śię go / ſtrzeż go ; ábowiem ono jeſt żywotem twoim.
Śćieƺką niepobożnych nie chodź / á nie udáwáj śię drogą złośliwych.
Opuść ją / nie chodź po niej ; uchyl śię od niej / á omiń ją.
Boć oni nie záſną / áż co złego zbroją ; áni śię uſpokoją / áż kogo do upádku przywiodą ;
Abowiem jedzą chleb niezbożnośći / á wino drápieſtwá piją /
Ale śćieƺká ſpráwiedliwych jáko świátłość jáſná / która im dálej tem bárdźiej świeći / áż do dniá doſkonáłego.
Drogá záś niepobożnych jeſt jáko ćiemność ; nie wiedzą / o co śię otrąćić mogą.
Synu mój! ſłów moich pilnuj ; ku powieśćiom moim nákłoń uchá twojego.
Niech nie odchodzą od ocżów twoich / záchowáj je w pośród ſercá twego.
Abowiem żywotem ſą tym / którzy je znájdują / á wƺyſtkiemu ćiáłu ich lekárſtwem.
Nád wƺyſtko / cżego ludźie ſtrzegą / ſtrzeż ſercá twego ; bo z niego żywot pochodźi.
Oddál od śiebie przewrotność uſt / á złośliwe wárgi oddál od śiebie.
Ocży twoje niecháj ná dobre rzecży pátrzą / á powieki twoje niech drogę przed tobą proſtują.
Umiárkuj śćieƺkę nóg twoich / áby wƺyſtkie drogi twoje pewne były.
Nie uchyláj śię ná práwo áni ná lewo ; owƺem / odwróć nogę twoję od złego.
