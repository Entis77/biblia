Syn mądry przyjmuje ćwicżenie ojcowſkie ; ále náśmiewcá nie ſłuchá ſtrofowániá.
Káżdy będzie pożywáł dobrego z owocu uſt ſwoich ; ále duƺá przewrotnych krzywdy pożywáć będzie.
Kto ſtrzeże uſt ſwych / ſtrzeże duƺy ſwojej ; kto lekkomyślnie otwierá wárgi ſwe / będzie ſtárty.
Duƺá leniwego żądá / á nic nie má ; ále duƺá prácowitych zbogáći śię.
Słowá kłámliwego nienáwidzi ſpráwiedliwy ; ále niezbożny ſtáje śię obrzydliwym y ſháńbionym.
Spráwiedliwość ſtrzeże tego / który żyje bez zmázy ; ále niezbożność podwrácá grzeƺniká.
Znájduje śię táki co śię cżyni bogátym / á nie má nic ; y táki / co śię cżyni ubogim / choć má wiele bogáctw.
Okup żywotá cżłowiecżego jeſt bogáctwo jego ; ále ubogi nie ſłuchá łájániá.
Świátłość ſpráwiedliwych jáſná ; ále pochodniá bezbożnych zgáśnie.
Sámą tylko pychą cżłowiek zwády wƺcżyná / ále przy tych / co rádę przyjmują / jeſt mądrość.
Bogáctwá źle nábyte umniejƺą śię ; ále kto je zgromádzá ręką ſwą / przycżyniá ich.
Nádziejá długá wątli ſerce ; ále żądość wypełnioná jeſt drzewem żywotá.
Kto gárdzi ſłowem Bożem / ſám ſobie ƺkodzi ; ále kto śię boi przykázániá jego / odnieśie nágrodę.
Náuká mądrego jeſt źródłem żywotá ku ochronieniu śię śideł śmierći.
Rozum dobry dáje łáſkę ; ále drogá przewrotnych jeſt przykrá.
Káżdy oſtrożny umiejętnie ſobie pocżyná ; ále głupi rozpośćierá głupſtwo.
Poſeł niezbożny upádá we złe ; ále poſeł wierny jeſt lekárſtwem.
Ubóſtwo y zelżywość przyjdzie ná tego / który śię wyłámuje z kárnośći ; ále kto przeſtrzegá upominániá / wyſłáwiony będzie.
Żądność wypełnioná ſłodká jeſt duƺy ; ále odſtąpić od złego / głupim jeſt obrzydliwośćią.
Kto chodzi z mądrymi / mądrym będzie ; ále kto towárzyƺy z głupimi / ſtártym będzie.
Nieƺcżęśćie grzeƺników śćigá ; ále ſpráwiedliwym Bóg dobrem nágrodzi.
Dobry cżłowiek zoſtáwiá dziedzictwo ſynom ſynów ſwoich ; ále májętność grzeƺniká ſpráwiedliwemu záchowáná bywá.
Obfitá żywność ná roli ubogich / á drugi ginie przez nieroztropność.
Kto záwśćiągá rózgi ſwej / má w nienáwiśći ſyná ſwego ; ále kto go miłuje / wcżáſ go kárze.
Spráwiedliwy je / y náſycá duƺę ſwoję : ále żołądek niezbożnych niedoſtátek ćierpi.
