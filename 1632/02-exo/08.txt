Y rzekł PAN do Mojzeƺá : wnidź do Fáráoná / á mów do niego : Ták mówi PAN : Wypuść lud mój áby mi ſłużył.
Ale jeſli ty nie będźieƺ chćiał puśćić / Oto ja záráżę wƺyſtkie gránice twoje żábámi.
Y wyda rzeká żáby / które wylezą / y wnidą do domu twego / y do komory łożá twego / y ná pośćiel twoję / y do domu ſług twojich / y miedzy lud twój / y do pieców twojich / y w dźieże twoje.
Ták ná ćię jáko ná lud twój / y ná wƺyſtkie ſługi twoje polezą żáby.
Tedy rzekł PAN do Mojzeƺá : rzecz do Aároná : Wyćiągni rękę twoję z laſką twą ná rzeki / ná ſtrugi / y ná jeźiorá : á wywiedź żáby ná źiemię Egipſką :
Tedy wyćiągnął Aáron rękę ſwą ná wody Egipſkie : y wylázły żáby / które okryły źiemię Egipſką.
Y ucżynili tákże Cżárownicy cżárámi ſwymi : y wywiedli żáby ná źiemię Egipſką.
Zátym Fáráo wezwał Mojzeƺá y Aároná mówiąc : Módlćie śię Pánu / áby oddalił żáby odemnie / y od ludu mego : á wypuƺczę lud / áby ofiárowáli PANU.
Y rzekł Mojzeƺ do Fáráoná : Poczczę ćię tym / <i>á powiedz</i> kiedy śię modlić mam zá ćię / y zá ſługi twoje / y zá lud twój / áby wytrácone były żáby / od ćiebie / y z domów twojich / <i>á</i> tylko w rzece zoſtáły.
A <i>on</i> rzekł : Jutro. Tedy rzekł <i>Mojzeƺ : ucżynię</i> według ſłowá twego / ábyś wiedźiał / że niemaƺ / jáko PAN Bóg náƺ.
Y odejdą żáby od ćiebie / y od domów twojich / y od ſług twojich / y od ludu twego : tylko w rzece zoſtáną.
Wyƺedł tedy Mojzeƺ y Aáron od Fáráoná / y záwołał Mojzeƺ do PANA / áby odjął żáby / które był przepuśćił ná Fáráoná.
Y ucżynił PAN według ſłowá Mojzeƺowego / że wyzdycháły żáby z domów / ze wśi / y z pól.
Y zgromadzáli je ná kupy / y zśmierdłá śię źiemiá :
A widząc Fáráo / że miał wytchnienie / obćiążył ſerce ſwoje / y nie uſłuchał ich / jáko był powiedźiał PAN.
Y rzekł PAN do Mojzeƺá : Mów Aáronowi : Wyćiągni laſkę twoję / á uderz w proch źiemie / áby śię obróćił we wƺy / po wƺyſtkiey źiemi Egipſkiey.
Y ucżynili ták / bo wyćiągnął Aáron rękę ſwoję z laſką ſwoją / y uderzył w proch źiemie / y były wƺy ná ludźiách / y ná bydle : Wƺyſtek proch źiemie obróćił śię we wƺy / po wƺyſtkiey źiemi Egipſkiey.
Cżynili też tákże Cżárownicy przez cżáry ſwoje / áby wywiedli wƺy / ále nie mogli : y były wƺy ná ludźiách y ná bydle.
Tedy rzekli Cżárownicy do Fáráoná : Pálec <i>to</i> Boży jeſt : y zátwárdźiáło ſerce Fáráonowe / y nie uſłuchał ich / jáko powiedźiał PAN.
Y rzekł PAN do Mojzeƺá : Wſtań ráno / á ſtań przed Fáráonem : Oto wynidźie do wody / y rzecżeƺ do niego : Ták mówi PAN / wypuść lud mój / áby mi ſłużył :
Bo jesli ty nie wypuśćiƺ ludu mego / oto ja poſyłam ná ćię y ná ſługi twe / y ná lud twój / y ná domy twoje / rozmájite robáctwo : á będą nápełnione domy Egipſkie rozmájitym robáctwem / nád to y źiemiá ná którey oni <i>ſą</i>.
A oddźielę dniá onego źiemię Geſen / w którey lud mój mieƺká / áby tam nie było rozmájitego robáctwá / ábyś poznał żem <i>JA PAN</i> w pośrzodku źiemie.
Y położę <i>znák</i> odkupienia miedzy ludem mojim / y miedzy ludem twojim / jutro będźie znák ten.
Tedy ucżynił ták PAN / y przyƺło rozmájite robáctwo ćięƺkie ná dom Fáráonów / y ná dom ſług jego : y ná wƺyſtkę źiemię Egipſką : <i>y</i> pſowáłá śię źiemiá od rozmájitego robáctwá :
Zátym wezwał Fáráo Mojzeƺá y Aároná / y rzekł : Idźćie / ofiárujćie Bogu wáƺemu w tey źiemi.
Y odpowiedźiał Mojzeƺ. Nie godźi śię ták cżynić / bobyſmy obrzydliwość Egipſką ofiárowáli PANU Bogu náƺemu : á gdybyſmy ofiárowáli obrzydliwość Egipſką przed ocżymá ich / záżby nas nie ukámionowáli?
Drogę trzech dni pójdźiemy / ná puƺcżą : y ofiárowáć będźiemy PANU Bogu náƺemu / jáko rozkaże nam :
Y rzekł Fáráo : Jać wypuƺczę was / ábyśćie ofiárowáli PANU Bogu wáƺemu ná puƺcży / wƺákże dáleko nie odchodźćie / <i>y</i> módlćie śię zá mną.
Y odpowiedźiał Mojzeƺ : Ja wychodzę od ćiebie / y będę śię modlił Pánu / á odejdźie rozmájite robáctwo od Fáráoná / od ſług jego / y od ludu jego / jutro tylko niech więcey Fáráo niekłama / áby nie miał wypuśćić ludu dla ofiárowánia PANU.
Wyƺedƺy tedy Mojzeƺ od Fáráoná / modlił śię PANU.
Y ucżynił PAN według ſłowá Mojzeƺowego / y oddalił ono rozmájite robáctwo od Fáráoná / y od ſług jego / y od ludu jego : <i>á</i> nie zoſtáło y jednego.
Jednák Fáráo obćiążył ſerce ſwe / y tym rázem / á nie wypuśćił ludu.
