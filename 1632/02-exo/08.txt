Y rzekł Pán do Mojżeƺá : Wnijdź do Fáráoná / á mów do niego : Ták mówi Pán : Wypuść lud mój / áby mi ſłużył.
Ale jeſli ty nie będźieƺ chćiáł puśćić / oto / Ja záráżę wƺyſtkie gránice twoje żábámi.
Y wydá rzeká żáby / które wylezą y wnijdą do domu twego / y do komory łożá twego / y ná pośćiel twoję / y do domu ſług twoich / y między lud twój / y do pieców twoich / y w dźieże twoje.
Ták ná ćię / jáko ná lud twój / y ná wƺyſtkie ſługi twoje polezą żáby.
Tedy rzekł Pán do Mojżeƺá : Rzecż do Aároná : Wyćiągnij rękę twoję z láſką twą ná rzeki / ná ſtrugi / y ná jeźiorá / á wywiedź żáby ná źiemię Egipſką.
Tedy wyćiągnął Aáron rękę ſwą ná wody Egipſkie / y wylázły żáby / które okryły źiemię Egipſką.
Y ucżynili tákże cżárownicy cżárámi ſwemi / y wywiedli żáby ná źiemię Egipſką.
Zátem Fáráo wezwáł Mojżeƺá y Aároná / mówiąc : Módlćie śię Pánu / áby oddálił żáby ode mnie / y od ludu mego ; á wypuƺcżę lud / áby ofiárowáli Pánu.
Y rzekł Mojżeƺ do Fáráoná : Pocżcżę ćię tem / á powiedz / kiedy śię modlić mam zá ćię / y zá ſługi twoje / y zá lud twój / áby wytrácone były żáby od ćiebie / y z domów twoich / á tylko w rzece zoſtáły.
A on rzekł : Jutro. Tedy rzekł Mojżeƺ : Ucżynię według ſłowá twego / ábyś wiedźiáł / że niemáƺ / jáko Pán Bóg náƺ.
Y odejdą żáby od ćiebie / y od domów twoich / y od ſług twoich / y od ludu twego / tylko w rzece zoſtáną.
Wyƺedł tedy Mojżeƺ y Aáron od Fáráoná. Y záwołáł Mojżeƺ do Páná / áby odjął żáby / które był przepuśćił ná Fáráoná.
Y ucżynił Pán według ſłowá Mojżeƺowego / że wyzdycháły żáby z domów / ze wśi / y z pól.
Y zgromádzáli je ná kupy / y zśmierdłá śię źiemiá.
A widząc Fáráo / że miał wytchnienie / obćiążył ſerce ſwoje / y nie uſłucháł ich / jáko był powiedźiáł Pán.
Y rzekł Pán do Mojżeƺá : Mów do Aároná : Wyćiągnij láſkę twoję / á uderz w proch źiemi / áby śię obróćił we wƺy po wƺyſtkiej źiemi Egipſkiej.
Y ucżynili ták ; bo wyćiągnął Aáron rękę ſwoję z láſką ſwoją / y uderzył w proch źiemi ; y były wƺy ná ludźiách / y ná bydle ; wƺyſtek proch źiemi obróćił śię we wƺy po wƺyſtkiej źiemi Egipſkiej.
Cżynili też tákże cżárownicy / przez cżáry ſwoje / áby wywiedli wƺy / ále nie mogli ; y były wƺy ná ludźiách y ná bydle.
Tedy rzekli cżárownicy do Fáráoná : Pálec to Boży jeſt. Y zátwárdźiáło ſerce Fáráonowe / y nie uſłucháł ich / jáko powiedźiáł Pán.
Y rzekł Pán do Mojżeƺá : Wſtáń ráno / á ſtáń przed Fáráonem / ( oto / wynijdźie do wody / )i rzecżeƺ do niego : Ták mówi Pán : Wypuść lud mój / áby mi ſłużył ;
Bo jeſli ty nie wypuśćiƺ ludu mego / oto / Ja poſyłám ná ćię / y ná ſługi twe / y ná lud twój / y ná domy twoje rozmáite robáctwo ; á będą nápełnione domy Egipſkie rozmáitem robáctwem / nádto y źiemiá / ná którey oni ſą.
A oddźielę dniá onego źiemię Goſen / w którey lud mój mieƺká / áby tám nie było rozmáitego robáctwá / ábyś poznał / żem Ja Pán w pośrodku źiemi.
Y położę znák odkupieniá między ludem moim y między ludem twoim ; jutro będźie znák ten.
Tedy ucżynił ták Pán. Y przyƺło rozmáite robáctwo ćiężkie ná dom Fáráonów / y ná dom ſług jego / y ná wƺyſtkę źiemię Egipſką ; y pſowáłá śię źiemiá od rozmáitego robáctwá.
Zátem wezwáł Fáráo Mojżeƺá y Aároná / y rzekł : Idźćie / ofiárujćie Bogu wáƺemu w tey źiemi.
Y odpowiedźiáł Mojżeƺ : Nie godźi śię ták cżynić ; bobyśmy obrzydliwość Egipſką ofiárowáli Pánu Bogu náƺemu ; á gdybyśmy ofiárowáli obrzydliwość Egipſką przed ocżymá ich / záżby nas nie ukámionowáli?
Drogę trzech dni pójdźiemy ná puƺcżą / y ofiárowáć będźiemy Pánu Bogu náƺemu / jáko nám rozkáże.
Y rzekł Fáráo : Jáć wypuƺcżę was / ábyśćie ofiárowáli Pánu Bogu wáƺemu ná puƺcży / wƺákże dáleko nie odchodźćie / y módlćie śię zá mną.
Y odpowiedźiáł Mojżeƺ : Ja wychodzę od ćiebie / y będę śię modlił Pánu / á odejdźie rozmáite robáctwo od Fáráoná / od ſług jego / y od ludu jego jutro ; tylko niech więcey Fáráo nie kłámá / áby nie miał wypuśćić ludu dla ofiárowániá Pánu.
Wyƺedłƺy tedy Mojżeƺ od Fáráoná / modlił śię Pánu.
Y ucżynił Pán według ſłowá Mojżeƺowego / y oddálił ono rozmáite robáctwo od Fáráoná / y od ſług jego / y od ludu jego / á nie zoſtáło y jednego.
Jednák Fáráo obćiążył ſerce ſwe y tym rázem / á nie wypuśćił ludu.
