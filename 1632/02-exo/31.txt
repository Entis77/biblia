Potem rzekł Pán do Mojżeƺá / mówiąc :
Otom wezwał z imieniá Beſáleelá / ſyná Urowego / ſyná Churowego z pokolenia Judy.
Y nápełniłem go Duchem Bożym / mądrośćią / y rozumem / y umiejętnośćią we wƺelákiem rzemieśle ;
Ku dowćipnemu wymyślániu / cokolwiek może być urobione ze złotá / y z ſrebrá / y z miedźi.
Do rzezánia kámieniá ná oſádzenie / y ná wyrobienie drzewá ku wyſtáwieniu káżdej roboty.
A oto / Ja przydáłem mu Acholijábá / ſyná Achyſámechowego z pokolenia Dán ; á w ſerce káżdego dowćipnego dáłem mądrość / áby zrobili wƺyſtko / com ći przykázał.
Námiot zgromádzenia / y ſkrzynię świádectwá / y ubłágálnią / która ma być nád nią / y wƺyſtkie nácżyniá namiotu.
Stół tákże y nácżyniá jego / y świecżnik cżyſty ze wƺyſtkiem nácżyniem jego / y ołtárz do kádzeniá.
Tákże ołtárz do cáłopalenia ze wƺyſtkiem nácżyniem jego / y wánnę ze ſtolcem jey.
Tákże ƺáty do ſłużby / y ƺáty święte Aáronowi kápłánowi / y ƺáty ſynom jego ku ſpráwowániu kápłáńſtwá.
Y olejek pomázowánia / y kádzenie wonne do świątnicy ; według wƺyſtkiego / jákom ći rozkázał / ucżynią.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Ty też powiedz ſynom Izráelſkim / mówiąc : Przećię ſábbátów mojich przeſtrzegáć będźiećie ; bo ten znák jeſt miedzy mną y miedzy wámi w národźiech wáƺych / ábyśćie wiedźieli / żem Ja Pán / który was poświęcám.
Przetoż przeſtrzegájćie ſábbátu / święty bowiem jeſt wam. Ktoby go zgwáłćił / śmierćią umrze ; bo káżdy / coby weń robotę odpráwował / wytráconá będźie duƺá jego / z pośrzodku ludu ſwego.
Przez ƺeść dni odpráwowáná będźie robotá ; ále w dźień śiódmy ſábbát jeſt / odpocżynienie święte Pánu ; káżdy / ktoby robił robotę w dźień ſábbátu / śmierćią umrze.
Przetoż będą ſtrzedz ſynowie Izráelſcy ſábbátu / záchowując ſábbát w národźiech ſwych uſtáwą wiecżną.
Między mną y miedzy ſyny Izráelſkimi znákiem jeſt wiecżnym ; bo w ƺeśćiu dniách ucżynił Pán niebo y źiemię / á dniá śiódmego przeſtał y odpocżynął.
Y dał Pán Mojżeƺowi dokonáwƺy mowy z nim ná górze Synáj dwie táblice świádectwá / táblice kámienne / piſáne pálcem Bożym.
