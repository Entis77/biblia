Potem rzekł Pán do Mojżeƺá / mówiąc :
Otom wezwáł z imieniá Beſáleelá / ſyná Urowego / ſyná Churowego z pokoleniá Judy.
Y nápełniłem go Duchem Bożym / mądrośćią / y rozumem / y umiejętnośćią we wƺelákiem rzemieśle ;
Ku dowćipnemu wymyślániu / cokolwiek może być urobione ze złotá / y z ſrebrá / y z miedźi.
Do rzezániá kámieniá ná oſádzenie / y ná wyrobienie drzewá ku wyſtáwieniu káżdej roboty.
A oto / Ja przydáłem mu Acholijábá / ſyná Achyſámechowego z pokoleniá Dán ; á w ſerce káżdego dowćipnego dáłem mądrość / áby zrobili wƺyſtko / com ći przykázáł.
Námiot zgromádzeniá / y ſkrzynię świádectwá / y ubłágálnią / która má być nád nią / y wƺyſtkie nácżyniá námiotu.
Stół tákże y nácżyniá jego / y świecżnik cżyſty ze wƺyſtkiem nácżyniem jego / y ołtárz do kádzeniá.
Tákże ołtárz do cáłopáleniá ze wƺyſtkiem nácżyniem jego / y wánnę ze ſtolcem jey.
Tákże ƺáty do ſłużby / y ƺáty święte Aáronowi kápłánowi / y ƺáty ſynom jego ku ſpráwowániu kápłáńſtwá.
Y olejek pomázowániá / y kádzenie wonne do świątnicy ; według wƺyſtkiego / jákom ći rozkázáł / ucżynią.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Ty też powiedz ſynom Izráelſkim / mówiąc : Przećię ſábbátów moich przeſtrzegáć będźiećie ; bo ten znák jeſt między mną y między wámi w národźiech wáƺych / ábyśćie wiedźieli / żem Ja Pán / który was poświęcám.
Przetoż przeſtrzegájćie ſábbátu / święty bowiem jeſt wám. Ktoby go zgwáłćił / śmierćią umrze ; bo káżdy / coby weń robotę odpráwowáł / wytráconá będźie duƺá jego / z pośrodku ludu ſwego.
Przez ƺeść dni odpráwowáná będźie robotá ; ále w dźień śiódmy ſábbát jeſt / odpocżynienie święte Pánu ; káżdy / ktoby robił robotę w dźień ſábbátu / śmierćią umrze.
Przetoż będą ſtrzedz ſynowie Izráelſcy ſábbátu / záchowując ſábbát w národźiech ſwych uſtáwą wiecżną.
Między mną y między ſyny Izráelſkimi znákiem jeſt wiecżnym ; bo w ƺeśćiu dniách ucżynił Pán niebo y źiemię / á dniá śiódmego przeſtáł y odpocżynął.
Y dáł Pán Mojżeƺowi dokonáwƺy mowy z nim ná górze Synáj dwie táblice świádectwá / táblice kámienne / piſáne pálcem Bożym.
