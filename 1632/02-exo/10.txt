Rzekł zátym PAN do Mojzeƺá : Wnidź do Fáráoná / bom ja obćiążył ſerce jego / y ſerce ſług jego / ábym cżynił te znáki moje miedzy nimi.
A żebyś opowiádał w uƺách Synów twojich y wnuków twojich / com uczynił w Egipćie / y znáki moje którem pokazał ná nich : ábyśćie wiedźieli żem JA PAN.
Wƺedł tedy Mojzeƺ y Aáron do Fáráoná / y mówili mu : Ták mówi PAN / Bóg Hebrejcżyków : Dokądże nie chceƺ uniżyć śię przedemną? Wypuść lud mój áby mi ſłużył.
Bo jeſli nie będźieƺ chćiał wypuśćić ludu mego / oto ja przywiodę jutro ƺáráńczą ná gránicę twoję :
Która okryje wierzch źiemie / że jey widáć nie będźie : y poje oſtátek pozoſtáły / który wam zoſtał po grádźie : y pogryźie káżde drzewo rodzące ná polu :
Y nápełni domy twoje / y domy wƺyſtkich ſług twojich / y domy wƺyſtkiego Egiptu : Czego nie widźieli Ojcowie twoji / y Ojcowie Ojców twojich od pocżątku bytu ſwego ná źiemi / áż do tego dniá. A odwróćiwƺy śię / wyƺedł od Fáráoná.
Tedy rzekli ſłudzy Fáráonowi do niego : długoż będźie nam ten ku zgorƺeniu? Wypuść <i>te</i> męże áby ſłuzyli PANU Bogu ſwemu : Zaż jeƺcze nie wieƺ / że zniƺczał Egipt?
Y záwołano záś Mojzeƺá z Aáronem do Fáráoná / y rzekł do nich : Idźćie / ſłużćie PANU Bogu wáƺemu : Którzyż to ſą co pójdą?
Y odpowiedźiał Mojzeƺ : Z dźiećmi náƺymi / y z ſtárcámi náƺymi pójdźiemy / z Synámi náƺymi / y z córkámi náƺemi / z trzodámi náƺymi / y z bydłem náƺym pójdźiemy : bo święto PANU <i>obchodźić</i> mamy.
Tedy im on rzekł : Niech ták będźie PAN z wámi / jáko <i>ja</i> was puƺczę / y dźieći wáƺe : pátrzćie że <i>coś</i> złego przed ſobą maćie.
Nie ták : ále idźćie <i>ſámi</i> mężowie / á ſłużćie Pánu : ponieważ wy tego ƺukaćie. Y wygnał je od śiebie Fáráo.
Potym PAN rzekł do Mojzeƺá : Wyćiągni rękę twą ná źiemię Egipſką dla ƺáráńcży / áby przyƺłá ná źiemię Egipſką / á pożárłá wƺyſtko źiele ná źiemi / wƺyſtko co zoſtáło po grádźie.
Y wyćiągnął Mojzeƺ laſkę ſwoję ná źiemię Egipſką / á PAN przywiódł wiátr wſchodni ná źiemię przez cáły on dźień y przez cáłą noc : <i>á</i> gdy było ráno / wiátr wſchodni przynióſł ƺáráńczą.
Y przyƺłá ƺáráńczá ná wƺyſtkę źiemię Egipſką / y przypádłá ná wƺyſtkie gránice Egipſkie bárzo ćięƺka : przed tym nie było tey podobney ƺáráńczy / y po niej nie będźie tákowa.
Y okryłá wierzch wƺyſtkiey źiemie / ták yż źiemie znáć nie było / á pożárłá wƺyſtkę trawę źiemie / y wƺyſtek owoc drzewá / który zoſtał po grádźie : á nie zoſtáło żadney źielonośći ná drzewie / y ná trawie polney / we wƺyſtkiey źiemi Egipſkiej :
Przetoż co rychley Fáráo wezwał Mojzeƺá y Aároná / y rzekł : Zgrzeƺyłem <i>przećiwko</i> Pánu Bogu wáƺemu / y <i>przećiwko</i> wam.
A teraz odpuść proƺę grzech mój áby ten raz / á módlćie śię Pánu Bogu wáƺemu / áby oddalił odemnie tylko tę śmierć.
Y wyƺedƺy <i>Mojzeƺ</i> od Fáráoná / modlił śię PANU.
A obróćiwƺy PAN wiátr zachodni mocny bárzo / porwał ƺáráńczą / y wrzućił ją do Morzá czerwonego / <i>y</i> nie zoſtáłá áni jedná ƺáráńcza we wƺyſtkich gránicách Egipſkich.
Y zátwárdźił Pan ſerce Fáráonowe y nie wypuśćił Synów Izráelſkich.
Tedy rzekł PAN do Mojzeƺá : Wyćiągni rękę twą ku niebu / á będą ćiemnośći po źiemi Egipſkiey / y mácáć ich będą.
Y wyćiągnął Mojzeƺ rękę ſwą ku Niebu : y były ćiemnośći gęſte po wƺyſtkiey źiemi Egipſkiey przez trzy dni.
Nie widźiał jeden drugiego / áni śię kto ruƺył z miejſcá ſwego przez <i>one</i> trzy dni : lecz u wƺyſtkich Synów Izráelſkich byłá świátłość w mieƺkániách ich.
A wezwawƺy Fáráo Mojzeƺá / rzekł : Idźćie / ſłużćie PANU : tylko trzody wáƺe y bydłá wáƺe niech zoſtáną : y dźieći wáƺe niech idą z wámi.
Y odpowiedźiał Mojzeƺ : owƺem ty daƺ do rąk náƺych Ofiáry y cáłopalenia / które ofiárowáć będźiemy PANU Bogu náƺemu.
Przetoż y dobytek náƺ pójdźie z námi / á nie zoſtánie <i>y</i> kopyto : ábowiem z tego weźmiemi do ſłużby PANU / Bogu náƺemu : bo my nie wiemy / cżym ſłużyć mamy PANU / áż tám przyjdźiemy.
Y zátwárdźił PAN ſerce Fáráonowe / y nie chćiał ich puśćić.
Y rzekł Fáráo <i>Mojzeƺowie</i> : idź odemnie / á ſtrzeż śię / ábyś więcey nie widźiał oblicza mego / bo dniá / którego ujrżyƺ oblicze moje / umrzeƺ.
Y odpowiedźiał Mojzeƺ : Dobrześ powiedźiał / nie ujrzę więcey oblicza twego.
