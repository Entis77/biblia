Rzekł zátem Pán do Mojżeƺá : Wnijdź do Fáráoná ; bom Ja obćiążył ſerce jego / y ſerce ſług jego / ábym cżynił te znáki moje miedzy nimi ;
Ażebyś opowiádał w uƺách ſynów twojich / y wnuków twojich / com ucżynił w Egipćie / y znáki moje / którem pokázał ná nich / ábyśćie wiedźieli / żem Ja Pán.
Wƺedł tedy Mojzeƺ y Aáron do Fáráoná / y mówili mu : Ták mówi Pán / Bóg Hebrejcżyków : Dokądże nie chceƺ uniżyć śię przedemną? Wypuść lud mój / áby mi ſłużył.
Bo jeſli nie będźieƺ chćiał wypuśćić ludu mego / oto Ja przywiodę jutro ƺáráńcżą ná gránicę twoję /
Która okryje wierzch źiemi / że jey widáć nie będźie / y poje oſtátek pozoſtáły / który wam zoſtał po grádźie / y pogryźie káżde drzewo rodzące ná polu.
Y nápełni domy twoje / y domy wƺyſtkich ſług twojich / y domy wƺyſtkiego Egiptu / cżego nie widźieli ojcowie twoi / y ojcowie ojców twojich od pocżątku bytu ſwego ná źiemi áż do tego dniá. A odwróćiwƺy śię / wyƺedł od Fáráoná.
Tedy rzekli ſłudzy Fáráonowi do niego : Długoż będźie nam ten ku zgorƺeniu? Wypuść te męże / áby ſłużyli Pánu Bogu ſwemu ; záż jeƺcże nie wieƺ / że zniƺcżał Egipt?
Y záwołáno záś Mojżeƺá z Aáronem do Fáráoná / y rzekł do nich : Idźćie / ſłużćie Pánu Bogu wáƺemu ; którzyż to ſą / co pójdą?
Y odpowiedźiał Mojzeƺ : Z dźiećmi náƺemi y z ſtárcámi náƺymi pójdźiemy / z ſynámi náƺymi / y z córkámi náƺemi / z trzodámi náƺemi / y z bydłem náƺem pójdźiemy ; bo święto Pánu obchodźić mamy.
Tedy im on rzekł : Niech ták będźie Pán z wámi / jáko ja was puƺcżę / y dźieći wáƺe ; pátrzćie / że coś złego przed ſobą máćie.
Nie ták ; ále idźćie ſámi mężowie / á ſłużćie Pánu / ponieważ wy tego ƺukáćie. Y wygnał je od śiebie Fáráo.
Potem Pán rzekł do Mojżeƺá : Wyćiągnij rękę twą ná źiemię Egipſką dla ƺáráńcży / áby przyƺłá ná źiemię Egipſką / á pożárłá wƺyſtko źiele ná źiemi / wƺyſtko / co zoſtáło po grádźie.
Y wyćiągnął Mojzeƺ láſkę ſwoję ná źiemię Egipſką / á Pán przywiódł wiátr wſchodni ná źiemię przez cáły on dźień / y przez cáłą noc ; á gdy było ráno / wiátr wſchodni przynióſł ƺáráńcżą.
Y przyƺłá ƺáráńcżá ná wƺyſtkę źiemię Egipſką / y przypádłá ná wƺyſtkie gránice Egipſkie bárzo ćiężká ; przed tym nie było tey podobney ƺáráńcży / y po niej nie będźie tákowej.
Y okryłá wierzch wƺyſtkiej źiemi / ták / iż źiemi znáć nie było ; á pożárłá wƺyſtkę tráwę źiemi / y wƺyſtek owoc drzewá / który zoſtał po grádźie / á nie zoſtáło żádney źielonośći ná drzewie y ná tráwie polney we wƺyſtkiej źiemi Egipſkiej.
Przetoż co rychlej Fáráo wezwał Mojżeƺá y Aároná / y rzekł : Zgrzeƺyłem przećiwko Pánu Bogu wáƺemu / y przećiwko wam.
A teraz odpuść proƺę grzech mój áby ten raz / á módlćie śię Pánu Bogu wáƺemu / áby oddálił ode mnie tylko tę śmierć.
Y wyƺedƺy Mojzeƺ od Fáráoná / modlił śię Pánu.
A obróćiwƺy Pán wiátr záchodni mocny bárzo / porwał ƺáráńcżą / y wrzućił ją do morzá cżerwonego / y nie zoſtáłá áni jedná ƺáráńcżá we wƺyſtkich gránicách Egipſkich.
Y zátwárdźił Pán ſerce Fáráonowe / y nie wypuśćił ſynów Izráelſkich.
Tedy rzekł Pán do Mojżeƺá : Wyćiągnij rękę twą ku niebu / á będą ćiemnośći po źiemi Egipſkiej / y mácáć ich będą.
Y wyćiągnął Mojzeƺ rękę ſwą ku niebu / y były ćiemnośći gęſte po wƺyſtkiej źiemi Egipſkiej przez trzy dni.
Nie widźiał jeden drugiego / áni śię kto ruƺył z miejſcá ſwego przez one trzy dni ; lecż u wƺyſtkich ſynów Izráelſkich byłá świátłość w mieƺkániách ich.
A wezwáwƺy Fáráo Mojżeƺá / rzekł : Idźćie / ſłużćie Pánu ; tylko trzody wáƺe / y bydłá wáƺe niech zoſtáną / y dźieći wáƺe niech idą z wámi.
Y odpowiedźiał Mojzeƺ : Owƺem ty dáƺ do rąk náƺych ofiáry y cáłopalenia / które ofiárowáć będźiemy Pánu / Bogu náƺemu.
Przetoż y dobytek náƺ pójdźie z námi / á nie zoſtánie y kopyto ; ábowiem z tego weźmiemy do ſłużby Pánu / Bogu náƺemu ; bo my niewiemy / cżym ſłużyć mamy Pánu / áż tám przyjdźiemy.
Y zátwárdźił Pán ſerce Fáráonowe / y niechćiał ich puśćić.
Y rzekł Fáráo do Mojżeƺá : Idź ode mnie / á ſtrzeż śię / ábyś więcey nie widźiał oblicża mego ; bo dniá / którego ujrzyƺ oblicże moje / umrzeƺ.
Y odpowiedźiał Mojzeƺ : Dobrześ powiedźiał ; nie ujrzę więcey oblicża twego.
