Rzekł zátem Pán do Mojżeƺá : Wnijdź do Fáráoná ; bom Já obćiążył ſerce jego / y ſerce ſług jego / ábym cżynił te znáki moje między nimi ;
Ażebyś opowiádáł w uƺách ſynów twoich / y wnuków twoich / com ucżynił w Egipćie / y znáki moje / którem pokázáł ná nich / ábyśćie wiedzieli / żem Já Pán.
Wƺedł tedy Mojżeƺ y Aáron do Fáráoná / y mówili mu : Ták mówi Pán / Bóg Hebrejcżyków : Dokądże nie chceƺ uniżyć śię przedemną? Wypuść lud mój / áby mi ſłużył.
Bo jeźli nie będzieƺ chćiáł wypuśćić ludu mego / oto Já przywiodę jutro ƺáráńcżą ná gránicę twoję /
Którá okryje wierzch ziemi / że jej widáć nie będzie / y poje oſtátek pozoſtáły / który wám zoſtáł po grádzie / y pogryzie káżde drzewo rodzące ná polu.
Y nápełni domy twoje / y domy wƺyſtkich ſług twoich / y domy wƺyſtkiego Egiptu / cżego nie widzieli ojcowie twoi / y ojcowie ojców twoich od pocżątku bytu ſwego ná ziemi áż do tego dniá. A odwróćiwƺy śię / wyƺedł od Fáráoná.
Tedy rzekli ſłudzy Fáráonowi do niego : Długoż będzie nám ten ku zgorƺeniu? Wypuść te męże / áby ſłużyli Pánu Bogu ſwemu ; záż jeƺcże nie wieƺ / że zniƺcżáł Egipt?
Y záwołáno záś Mojżeƺá z Aáronem do Fáráoná / y rzekł do nich : Idźćie / ſłużćie Pánu Bogu wáƺemu ; którzyż to ſą / co pójdą?
Y odpowiedziáł Mojżeƺ : Z dziećmi náƺemi y z ſtárcámi náƺymi pójdziemy / z ſynámi náƺymi / y z córkámi náƺemi / z trzodámi náƺemi / y z bydłem náƺem pójdziemy ; bo święto Pánu obchodzić mámy.
Tedy im on rzekł : Niech ták będzie Pán z wámi / jáko já wáſ puƺcżę / y dzieći wáƺe ; pátrzćie / że coś złego przed ſobą máćie.
Nie ták ; ále idźćie ſámi mężowie / á ſłużćie Pánu / poniewáż wy tego ƺukáćie. Y wygnáł je od śiebie Fáráo.
Potem Pán rzekł do Mojżeƺá : Wyćiągnij rękę twą ná ziemię Egipſką dlá ƺáráńcży / áby przyƺłá ná ziemię Egipſką / á pożárłá wƺyſtko ziele ná ziemi / wƺyſtko / co zoſtáło po grádzie.
Y wyćiągnął Mojżeƺ láſkę ſwoję ná ziemię Egipſką / á Pán przywiódł wiátr wſchodni ná ziemię przez cáły on dzień / y przez cáłą noc ; á gdy było ráno / wiátr wſchodni przynióſł ƺáráńcżą.
Y przyƺłá ƺáráńcżá ná wƺyſtkę ziemię Egipſką / y przypádłá ná wƺyſtkie gránice Egipſkie bárdzo ćiężká ; przedtem nie było tej podobnej ƺáráńcży / y po niej nie będzie tákowej.
Y okryłá wierzch wƺyſtkiej ziemi / ták / iż ziemi znáć nie było ; á pożárłá wƺyſtkę tráwę ziemi / y wƺyſtek owoc drzewá / który zoſtáł po grádzie / á nie zoſtáło żádnej zielonośći ná drzewie y ná tráwie polnej we wƺyſtkiej ziemi Egipſkiej.
Przetoż co rychlej Fáráo wezwáł Mojżeƺá y Aároná / y rzekł : Zgrzeƺyłem przećiwko Pánu Bogu wáƺemu / y przećiwko wám.
A teráz odpuść proƺę grzech mój áby ten ráz / á módlćie śię Pánu Bogu wáƺemu / áby oddálił ode mnie tylko tę śmierć.
Y wyƺedłƺy Mojżeƺ od Fáráoná / modlił śię Pánu.
A obróćiwƺy Pán wiátr záchodni mocny bárdzo / porwáł ƺáráńcżą / y wrzućił ją do morzá cżerwonego / y nie zoſtáłá áni jedná ƺáráńcżá we wƺyſtkich gránicách Egipſkich.
Y zátwárdził Pán ſerce Fáráonowe / y nie wypuśćił ſynów Izráelſkich.
Tedy rzekł Pán do Mojżeƺá : Wyćiągnij rękę twą ku niebu / á będą ćiemnośći po ziemi Egipſkiej / y mácáć ich będą.
Y wyćiągnął Mojżeƺ rękę ſwą ku niebu / y były ćiemnośći gęſte po wƺyſtkiej ziemi Egipſkiej przez trzy dni.
Nie widziáł jeden drugiego / áni śię kto ruƺył z miejſcá ſwego przez one trzy dni ; lecż u wƺyſtkich ſynów Izráelſkich byłá świátłość w mieƺkániách ich.
A wezwáwƺy Fáráo Mojżeƺá / rzekł : Idźćie / ſłużćie Pánu ; tylko trzody wáƺe / y bydłá wáƺe niech zoſtáną / y dzieći wáƺe niech idą z wámi.
Y odpowiedziáł Mojżeƺ : Owƺem ty dáƺ do rąk náƺych ofiáry y cáłopáleniá / które ofiárowáć będziemy Pánu / Bogu náƺemu.
Przetoż y dobytek náƺ pójdzie z námi / á nie zoſtánie y kopyto ; ábowiem z tego weźmiemy do ſłużby Pánu / Bogu náƺemu ; bo my niewiemy / cżem ſłużyć mámy Pánu / áż tám przyjdziemy.
Y zátwárdził Pán ſerce Fáráonowe / y niechćiáł ich puśćić.
Y rzekł Fáráo do Mojżeƺá : Idź ode mnie / á ſtrzeż śię / ábyś więcej nie widziáł oblicżá mego ; bo dniá / którego ujrzyƺ oblicże moje / umrzeƺ.
Y odpowiedziáł Mojżeƺ : Dobrześ powiedziáł ; nie ujrzę więcej oblicżá twego.
