Potym rzekł Pan do Mojzeƺá : Wnidź do Fáráoná / á mów do niego : Ták mówi PAN / Bóg Hebrejcżyków : Wypuść lud mój / áby mi ſłużył :
Bo jeſli go ty nie będźieƺ chćiał wypuśćić / ále jeƺcze zátrzymawáć go będźieƺ /
Oto / ręká Páńſka będźie ná bydle twojim / które <i>jeſt</i> ná polu : ná koniách / ná oślech / ná wielbłądách / ná wołách y ná owcách / powietrze bárzo ćiężkie.
Y ucżyni PAN roźdźiał miedzy trzodámi Izráelſkimi / y miedzy trzodámi Egipſkimi : áby nic nie zdechło ze wƺyſtkiego co jeſt Synów Izráelſkich :
Y poſtánowił PAN cżás / mówiąc : Jutro ucżyni PAN tę rzecż ná źiemi.
Y ucżynił PAN tę rzecż názájutrz / że wyzdycháły wƺyſtkie bydłá Egipſkie : ále z bydłá ſynów Izráelſkich / nie zdechło áni jedno :
Y poſłał Fáráo / á oto nie zdechło z bydłá Izráelſkiego y jedno : ále oćiężáło ſerce Fáráonowe / y nie wypuśćił ludu.
Zátym rzekł PAN do Mojzeƺá y do Aároná : weźmićie pełne garśći wáƺe popiołu z piecá : á niech go rozrzući Mojzeƺ ku niebu / przed ocżymá Fáráonowemi :
Y obróći śię w proch po wƺyſtkiey źiemi Egipſkiey / y będźie ná ludźiach / y ná bydle wrzodem cżyniącym pryƺczele / po wƺyſtkiey źiemi Egipſkiey :
Wźięli tedy popiołu z piecá / y ſtánęli przed Fáráonem / y rozrzućił go Mojzeƺ ku Niebu : y ſtał śię wrzodem pryƺczele cżyniącym / ná ludźiách y ná bydle.
Y nie mogli cżárownicy ſtać przed Mojzeƺem dla wrzodu : bo był wrzód ná cżárownikách / y ná wƺyſtkich Egipcżánách.
Y zátwárdźił PAN ſerce Fáráonowe / y nie uſłuchał ich / jáko był powiedźiał PAN Mojzeƺowi.
Y rzekł Pan do Mojzeƺá : wſtań ráno / á ſtań przed Fáráonem / y mów do niego : Ták mówi PAN / Bóg Hebrejcżyków : Wypuść lud mój / áby mi ſłużył :
Bo tą rázą ja poſyłam wƺyſtkie plagi moje ná ſerce twoje / y ná ſługi twoje / y ná lud twój / ábyś wiedźiał że nie maƺ mnie podobnego po wƺyſtkiey źiemi.
Bo teraz śćiągnę rękę moję / á uderzę ćię y lud twój powietrzem / y wytrácony będźieƺ z źiemie.
A zájiſte dla tegom ćię záchował / ábym okazał ná tobie moc moję / y żeby opowiádáne było Imię moje / po wƺyſtkiey źiemi.
Jeƺcżeż śię ty wynośiƺ przećiw ludowi memu / nie chcąc go wypuśćić?
Oto / ja ſpuƺczę o tym czáśie jutro grad bárzo ćiężki / jákiemu nie było podobnego w Egipćie / ode dniá / którego jeſt záłożon / áż do tego cżáſu.
A ták pośli teraz / zgromadź bydło twoje / y wƺyſtko co maƺ ná polu : <i>bo</i> ná káżdego człowieká / y ná bydlę / które náleźione będźie ná polu / á nie będźie zegnáne w dom / ſpádnie ná nie grad / y pozdycháją.
Kto śię <i>tedy</i> ulękł ſłowá PAŃSKIEGO z ſług Fáráonowych / kazał ućiekáć ſługom ſwym / y z bydłem ſwojim do domu.
Ale kto nie przyłożył ſercá ſwego do ſłowá Páńſkiego / ten zoſtáwił ſługi ſwe / y bydło ſwe ná polu.
Y rzekł PAN do Mojzeƺá : Wyćiągni rękę twą ku niebu / że będźie grad po wƺyſtkiey źiemi Egipſkiey / ná ludźie / y ná bydło / y ná wƺelákie źiołá polne / w źiemi Egipſkiey.
A ták wyćiągnął Mojzeƺ laſkę ſwą ku Niebu / á PAN dał gromy y grády / y z ſtąpił ogień ná źiemię / y ſpuśćił PAN grad ná źiemię Egipſką.
Y był grad y ogień zmięƺány z grádem ćiężkim bárzo / jákiemu nie był podobny we wƺyſtkiey źiemi Egipſkiey / jáko w niey mieƺkáć poczęto.
Y potłukł on grad po wƺyſtkiey źiemi Egipſkiey / cokolwiek było ná polu / od człowieká y áż do bydlęćiá : y wƺyſtko źiele polne potłukł grad / y wƺyſtko drzewo polne połamał.
Tylko w źiemi Geſen / gdźie Synowie Izráelſcy mieƺkáli / nie było grádu.
Poſłał tedy Fáráo / á wezwał Mojzeƺá y Aároná / mówiąc do nich : Zgrzeƺyłem <i>y</i> tym rázem : PAN jeſt ſpráwiedliwy / ále ja y lud mój niezbożniſmy.
Módlćieƺ śię PANU ( boć doſyć <i>jeſt,</i> ) áby przeſtáły gromy Boże y grad / á wypuƺczę was : y nie będźiećie <i>tu</i> mieƺkáć dáliey.
Y rzekł Mojzeƺ do niego : gdy wynidę z miáſtá wyćiągnę ręce ſwe do PANA / á gromy uſtáną / y grad nie będźie więcey : ábyś wiedźiał że Pańſka jeſt źiemiá.
Ale ty y ſłudzy twoji wiem że śię jeƺcze nie bojićie oblicza PANA Bogá.
Len tedy y jęczmień potłuczony jeſt : bo jęczmień był niedoſtáły / á len podraſtał :
Pƺenicá jednák y żyto potłuczone nie były / bo późne były.
Wyƺedƺy tedy Mojzeƺ od Fáráoná z miáſtá / wyćiągnął ręce ſwe do PANA : y przeſtáły gromy y grády / á deƺcz nie pádał ná źiemię.
A widząc Fáráo / że przeſtał deƺcz / y grad / y gromy / przycżynił grzechu / á obćiążył ſerce ſwe ſam y ſłudzy jego.
Y zátwárdźiáło ſerce Fáráonowe / y nie wypuśćił Synów Izráelſkich / jáko był powiedźiał PAN przez Mojzeƺá.
