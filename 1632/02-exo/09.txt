Potem rzekł Pán do Mojżeƺá : Wnijdź do Fáráoná / á mów do niego : Ták mówi Pán / Bóg Hebrejcżyków : Wypuść lud mój / áby mi ſłużył ;
Bo jeźli go ty nie będzieƺ chćiáł wypuśćić / ále jeƺcże zátrzymywáć go będzieƺ :
Oto / ręká Páńſká będzie ná bydle twojim / które jeſt ná polu / ná koniách / ná oſłách / ná wielbłądách / ná wołách y ná owcách / powietrze bárdzo ćiężkie.
Y ucżyni Pán rozdziáł między trzodámi Izráelſkiemi / y między trzodámi Egipſkiemi / áby nic nie zdechło ze wƺyſtkiego / co jeſt ſynów Izráelſkich.
Y poſtánowił Pán cżáſ / mówiąc : Jutro ucżyni Pán tę rzecż ná ziemi.
Y ucżynił Pán tę rzecż názájutrz / że wyzdycháły wƺyſtkie bydłá Egipſkie ; ále z bydłá ſynów Izráelſkich nie zdechło áni jedno.
Y poſłáł Fáráo / á oto / nie zdechło z bydłá Izráelſkiego y jedno ; ále oćiężáło ſerce Fáráonowe / y nie wypuśćił ludu.
Zátem rzekł Pán do Mojżeƺá y do Aároná : Weźmijćie pełne gárśći wáƺe popiołu z piecá / á niech go rozrzući Mojżeƺ ku niebu przed ocżymá Fáráonowemi.
Y obróći śię w proch po wƺyſtkiej ziemi Egipſkiej / y będzie ná ludziách / y ná bydle wrzodem cżyniącym pryƺcżele / po wƺyſtkiej ziemi Egipſkiej.
Wzięli tedy popiołu z piecá / y ſtánęli przed Fáráonem / y rozrzućił go Mojżeƺ ku niebu ; y ſtáł śię wrzodem / pryƺcżele cżyniącym ná ludziách y ná bydle ;
Y nie mogli cżárownicy ſtáć przed Mojżeƺem dlá wrzodu ; bo był wrzód ná cżárownikách y ná wƺyſtkich Egipcżánách.
Y zátwárdził Pán ſerce Fáráonowe / y nie uſłucháł ich / jáko był powiedziáł Pán Mojżeƺowi.
Y rzekł Pán do Mojżeƺá : Wſtáń ráno / á ſtáń przed Fáráonem / y mów do niego : Ták mówi Pán / Bóg Hebrejcżyków : Wypuść lud mój / áby mi ſłużył ;
Bo tą rázą Já poſyłám wƺyſtkie plági moje ná ſerce twoje / y ná ſługi twoje / y ná lud twój / ábyś wiedziáł / że nie máƺ mnie podobnego po wƺyſtkiej ziemi.
Bo teráz śćiągnę rękę moję á uderzę ćię y lud twój powietrzem / y wytrácony będzieƺ z ziemi.
A záiſte / dlá tegom ćię záchowáł / ábym okázáł ná tobie moc moję / y żeby opowiádáne było imię moje po wƺyſtkiej ziemi.
Jeƺcżeż śię ty wynośiƺ przećiw ludowi memu / nie chcąc go wypuśćić?
Oto / Já ſpuƺcżę o tym cżáśie jutro grád bárdzo ćiężki / jákiemu nie było podobnego w Egipćie ode dniá / którego jeſt záłożon / áż do tego cżáſu.
A ták poślij teráz / zgromádź bydło twoje / y wƺyſtko / co máƺ ná polu ; bo ná káżdego cżłowieká / y ná bydlę / które ználezione będzie ná polu / á nie będzie zegnáne w dom / ſpádnie ná nie grád / y pozdycháją.
Kto śię tedy uląkł ſłowá Páńſkiego z ſług Fáráonowych / kázáł ućiekáć ſługom ſwym / y z bydłem ſwojim do domu ;
Ale kto nie przyłożył ſercá ſwego do ſłowá Páńſkiego / ten zoſtáwił ſługi ſwe y bydło ſwe ná polu.
Y rzekł Pán do Mojżeƺá : Wyćiągnij rękę twą ku niebu / że będzie grád po wƺyſtkiej ziemi Egipſkiej / ná ludzi / y ná bydło / y ná wƺelákie ziołá polne w ziemi Egipſkiej.
A ták wyćiągnął Mojżeƺ láſkę ſwą ku niebu / á Pán dáł gromy y grády / y zſtąpił ogień ná ziemię / y ſpuśćił Pán grád ná ziemię Egipſką.
Y był grád / y ogień zmieƺány z grádem ćiężkim bárdzo / jákiemu nie był podobny we wƺyſtkiej ziemi Egipſkiej / jáko w niej mieƺkáć pocżęto.
Y potłukł on grád po wƺyſtkiej ziemi Egipſkiej / cokolwiek było ná polu / od cżłowieká áż do bydlęćiá ; y wƺyſtko ziele polne potłukł grád / y wƺyſtko drzewo polne połámáł ;
Tylko w ziemi Goſen / gdzie ſynowie Izráelſcy mieƺkáli / nie było grádu.
Poſłáł tedy Fáráo / á wezwáł Mojżeƺá y Aároná / mówiąc do nich : Zgrzeƺyłem y tym rázem ; Pán jeſt ſpráwiedliwy / ále já y lud mój niezbożniśmy.
Módlćież śię Pánu / ( boć doſyć jeſt / )áby przeſtáły gromy Boże y grád ; á wypuƺcżę wáſ / y nie będziećie tu mieƺkáć dálej.
Y rzekł Mojżeƺ do niego : Gdy wynijdę z miáſtá / wyćiągnę ręce ſwe do Páná / á gromy uſtáną / y grád nie będzie więcej / ábyś wiedziáł / że Páńſká jeſt ziemiá ;
Ale ty y ſłudzy twoi / wiem / że śię jeƺcże nie boićie oblicżá Páná Bogá.
Len tedy y jęcżmień potłucżony jeſt ; bo jęcżmień był niedoſtáły / á len podráſtáł.
Pƺenicá jednák y żyto potłucżone nie były ; bo późne były.
Wyƺedłƺy tedy Mojżeƺ od Fáráoná z miáſtá / wyćiągnął ręce ſwe do Páná ; y przeſtáły gromy y grády / á deƺcż nie pádáł ná ziemię.
A widząc Fáráo / że przeſtáł deƺcż / y grád / y gromy / przycżynił grzechu / á obćiążył ſerce ſwe / ſám y ſłudzy jego.
Y zátwárdziáło ſerce Fáráonowe / y nie wypuśćił ſynów Izráelſkich / jáko był powiedziáł Pán przez Mojżeƺá.
