A teć ſą ſądy / które przełożyƺ przed oblicżem ich.
Jeſli kupiƺ niewolniká Hebrejcżyká / ƺeść lát ſłużyć ći będźie / á śiódmego wynijdźie wolny dármo.
Jeſliby ſám tylko przyƺedł / ſám odejdźie ; á jeſliby miał żonę / y żoná jego z nim wynijdźie.
Jeſli mu pán jego dáł żonę / á zrodźiłá mu ſyny álbo córki / żoná y dźieći jego będą páná jego / á on ſám tylko odejdźie.
A jeſliby mówiąc rzekł niewolnik : Miłuję páná mego / żonę moję / y ſyny moje / nie wynijdę wolnym :
Tedy przywiedźie go pán jego do ſędźiów / á poſtáwi go u drzwi álbo u podwoi ; y przekole mu pán jego ucho jego ƺydłem / y będźie mu niewolnikiem ná wieki.
Záś jeſliby kto záprzedáł córkę ſwoję / áby byłá niewolnicą / nie wynijdźie jáko wychodzą niewolnicy.
Jeſliby śię nie ſpodobáłá w ocżách páná ſwego / á nie poślubiłby jey ſobie / niech pozwoli / áby ją odkupiono ; obcemu ludowi nie będźie jey mógł ſprzedáć / poniewáż zgrzeƺył przećiwko niej.
A jeſliby ją ſynowi ſwemu poślubił / według práwá córek ucżyni jey.
Jeſliby też inną wźiął mu zá żonę / tedy pożywieniá jey / odźieniá jey / y práwá máłżeńſkiego nie umniejƺy jey.
A jeſli tych trzech rzecży nie ucżyni jey / tedy wynijdźie dármo bez okupu.
Ktoby uderzył cżłowieká / áżby umárł / śmierćią umrze ;
Lecż ktoby nie cżyháł ná kogo / áleby go Bóg podáł w rękę jego / náznácżęć miejſce / ná które będźie miał ućiec.
Ale jeſliby kto umyślnie przećiw bliźniemu ſwemu záſádźiwƺy śię zdrádą zábił go / y od ołtárzá mego weźmieƺ go / áby umárł.
Ktoby uderzył ojcá ſwego / álbo mátkę ſwoję / śmierćią umrze.
Ktoby ukrádł cżłowieká á ſprzedáłby go / á ználeźionby był w ręku jego / śmierćią umrze.
Ktoby złorzecżył ojcu ſwemu álbo mátce ſwojey / śmierćią umrze.
A gdyby śię poſwárzyli mężowie / y uderzyłby kto bliźniego ſwego kámieniem álbo pięśćią / á onby záráz nie umárł / áleby śię położył ná łoże ;
A wſtáwƺy chodźiłby po ulicy o láſce ſwej / nie będźie winien ten / który uderzył ; tylko omieƺkánie jego nágrodźi / á ná wylecżenie jego náłoży.
Jeſliby záś uderzył kto niewolniká ſwego / álbo niewolnicę ſwoję kijem / y umárliby w ręku jego / koniecżnie káránie odnieśie ;
Wƺákże / jeſliby dźień álbo dwá żyw zoſtáł / nie będźie kárány ; bo zá pieniądze jego on jeſt kupiony.
Jeſliby śię też powádźiwƺy mężowie / uderzył który z nich niewiáſtę brzemienną / ták żeby z niej płód wyƺedł / jednákby nie záƺłá śmierć / koniecżnie káránie odnieśie / jákie włoży náń mąż onejże niewiáſty / á dá wedle uznániá ſędźiów.
Ale gdźieby śmierć záƺłá / tedy dáƺ duƺę zá duƺę ;
Oko zá oko / ząb zá ząb / rękę zá rękę / nogę zá nogę /
Spárzelinę zá ſpárzelinę / ránę zá ránę / śiność zá śiność.
Jeſliby záś kto wybił oko niewolnikowi ſwemu / álbo oko niewolnicy ſwojey / áżby śię zepſowáło / wolno go puśći zá oko jego.
Jeſliby też kto ząb niewolnikowi ſwemu / álbo ząb niewolnicy ſwojey wybił / wolno go puśći zá ząb jego.
Jeſliby też cżyj wół ubódł mężá álbo niewiáſtę / á umárliby / koniecżnie ukámionowány będźie on wół / á nie będą jeść mięſá jego ; á pán wołu onego nie będźie winien.
Wƺákże / jeſliby wół bódł przedtem / á oſtrzegánoby w tym páná jego / y nie miáłby go pod ſtráżą / á zábiłby mężá álbo niewiáſtę / wół on ukámionowány będźie / nádto y pán jego umrze.
Jeſliby náń włożono / żeby śię odkupił / tedy dá okup zá duƺę ſwoję / jákikolwiek náń włożą.
Choćby ſyná ubódł / álbo córkę ubódł / podług tegoż ſądu poſtąpią z nim.
Jeſliby niewolniká ubódł wół / álbo niewolnicę / ſrebrá trzydźieśći ſyklów dá pánu jego / á wół on ukámionowány będźie.
Jeſliby kto otworzył ſtudnią / álbo jeſliby kto wykopáł ſtudnią / y nie nákryłby jey / á wpádłby w nią wół álbo ośieł :
Pán oney ſtudni oddá zápłátę / y nágrodźi pánu ich / á co zdechło / jego będźie.
Tákże / gdyby wół cżyj ubódł wołu ſąśiádá jego / áżby zdechł / tedy ſprzedádzą wołu żywego / y rozdźielą śię zápłátą jego / onym też zdechłym podźielą śię.
Ale jeſliby to było wiádomo / że on wół bódł przedtem / á nie miał go pod ſtráżą pán jego / koniecżnie oddá wołu zá wołu / á zdechłego ſobie weźmie.
