A teć ſą Sądy które przełożyƺ przed obliczem jich.
Jesli kupiƺ niewolniká Hebrejcżyká / ƺeść lat służyć <i>ći</i> będźie : á śiódmego wynidźie wolny dármo.
Jesliby ſam <i>tylko</i> przyƺedł / ſam odejdźie : á jesliby miał żonę / y żoná jego z nim wynidźie.
Jesli mu Pan jego dał żonę / á zrodźiłá mu Syny álbo córki : żoná y dźieći jego będą Páná jego / á on ſam tylko odejdźie.
A jesliby mówiąc rzekł niewolnik : Miłuję Páná mego / żonę moję / y Syny moje / nie wynidę wolnym.
Tedy przywiedźie go Pan jego do Sędźiów / á poſtáwi go u drzwi / álbo u podwojá / y przekole mu Pan jego ucho jego ƺydłem / y będźie mu niewolnikiem ná wieki.
Záś jesliby kto záprzedał córkę ſwoję / áby byłá niewolnicą : nie wynidźie jáko wychodzą niewolnicy.
Jesliby śię nie ſpodobáłá w ocżách Páná ſwego / á nie poślubiłby jey ſobie / niech pozwoli áby ją odkupiono : obcemu ludowi nie będźie jey mógł przedáć / ponieważ zgrzeƺył przećiwko niey.
A jesliby ją Synowi ſwemu poślubił / według práwá córek uczyni jey.
Jesliby też inną wźiął mu zá żonę / tedy pożywienia jey / odzienia jey / y práwá małżeńſkiego nie umniejƺy jey.
A jeſli tych trzech rzeczy nie uczyni jey / tedy wynidźie dármo bez okupu.
Ktoby uderzył cżłowieká áżby umárł / śmierćią umrze.
Lecż ktoby nie cżyhał ná kogo / áleby go Bóg podał w rękę jego / náznáczęć miejſce ná które będźie miał ućiec.
Ale jesliby kto umyślnie przećiw bliźniemu ſwemu / náſádźiwƺy śię zdrádą zábił go / <i>y</i> od Ołtarzá mego weźmieƺ go áby umárł :
Ktoby uderzył Ojcá ſwego / álbo mátkę ſwoję / śmierćią umrze.
Ktoby ukradł cżłowieká / á przedałby go / y ználeźionby był w ręku jego / śmierćią umrze.
Ktoby złorzeczył Ojcu ſwemu / álbo mátce ſwojey / śmierćią umrze.
A gdyby śię poſwárzyli mężowie / y uderzyłby kto bliźniego ſwego kámieniem álbo pięśćią / á onby <i>záraz</i> nie umárł / áleby śię położył ná łoże /
A wſtawƺy chodźiłby po ulicy / o laſce ſwey / nie będźie winien ten / który uderzył : tylko omięƺkánie jego nágrodźi / á ná wyleczenie jego náłoży.
Jesliby záś uderzył kto niewolniká ſwego / álbo niewolnicę ſwoję kijem / y umárliby w ręku jego / koniecznie karánie odnieśie.
Wƺákże jesliby dźień álbo dwá <i>żyw</i> zoſtał / nie będźie karány / bo zá pieniądze jego on jeſt <i>kupiony</i>.
Jesliby śię też powádźiwƺy mężowie / uderzył który z nich Niewiáſtę brzemienną / ták żeby z niey płód wyƺedł / jednákby nie záƺłá śmierć / koniecznie karánie odnieśie jákie włoży nań mąż onejże Niewiáſty / á da wedle <i>uznánia</i> Sędźiów.
Ale gdźieby śmierć záƺłá / tedy daƺ duƺę zá duƺę.
Oko zá oko / ząb zá ząb / rękę zá rękę / nogę zá nogę.
Spárzelinę zá ſpárzelinę / ránę zá ránę / śiność zá śiność.
Jesliby záś kto wybił oko niewolnikowi ſwemu / álbo oko niewolnicy ſwojey / áżby śię zepſowáło / wolno go puśći zá oko jego.
Jesliby też <i>kto</i> ząb niewolnikowi ſwemu / álbo ząb niewolnicy ſwojey wybił / wolno go puśći zá ząb jego.
Jesliby też czyj wół ubódł mężá álbo niewiáſtę / á umárliby / koniecznie ukámionowány będźie on wół / á nie będą jeść mięſá jego : á Pan wołu <i>onego</i> nie będźie winien.
Wƺákże jesliby wół bódł przed tem / á oſtrzeganoby w tym Páná jego / y niemiałby go pod ſtrażą / á zábiłby mężá álbo niewiáſtę / wół on ukámionowány będźie : nád to y Pan jego umrze :
Jesliby nań włożono żeby śię odkupił / tedy da okup zá duƺę ſwoję jákikolwiek nań włożą.
Choćby Syná ubódł / álbo córkę ubódł / podług tegoż ſądu poſtąpią z nim.
Jesliby niewolniká ubódł wół / álbo niewolnicę / śrebrá trzydźieśći ſyklów da Pánu jego / á wół on ukámionowány będźie.
Jesliby kto otworzył ſtudnią / álbo jesliby kto wykopał ſtudnią / y nie nákryłby jey / á wpadłby w nią wół / álbo ośieł.
Pan oney ſtudnie odda zapłátę / <i>y</i> nágrodźi Pánu ich / á co zdechło / jego będźie.
Tákże gdyby wół czyj ubódł wołu ſąśiádá jego / áżby zdechł / tedy przedádzą wołu żywego / y rozdźielą śię zapłátą jego. Onym też zdechłym podźielą śię.
Ale jesliby to było wiádomo / że on wół bódł przed tem / á nie miał go pod ſtrażą Pan jego / koniecznie odda wołu zá wołu / á zdechłego ſobie weźmie.
