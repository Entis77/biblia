Ruƺyli śię potem z Elimu / y przyƺło wƺyſtko mnóſtwo ſynów Izráelſkich ná puƺcżą Zyn / która leży między Elim y między Synáj / piętnáſtego dniá mieśiącá wtórego po wyjśćiu ich z źiemi Egipſkiej.
Y ƺemráło wƺyſtko zgromádzenie ſynów Izráelſkich przećiw Mojżeƺowi y przećiw Aáronowi ná puƺcży.
A mówili do nich ſynowie Izráelſcy : Obyśmy byli pomárli od ręki Páńſkiej w źiemi Egipſkiej / gdyśmy śiádáli nád gárncy mięſá / gdyśmy śię nájádáli chlebá do ſytośći ; bo teraz wywiedliśćie nas ná tę puƺcżą / ábyśćie pomorzyli to wƺyſtko mnóſtwo głodem.
Tedy rzekł Pán do Mojżeƺá : Oto / Ja / ſpuƺcżę wám / jáko deƺcż chleb z niebá / y będźie wychodźił lud / á będźie zbieráł / coby dość było ná káżdy dźień / ábym go doświádcżył / będźieli chodźił w zakonie moim / cżyli nie.
Ale dniá ƺóſtego nágotują to / co przynioſą / á będźie tyle dwoje niż co zbieráć zwykli ná káżdy dźień.
Y mówił Mojżeƺ y Aáron do wƺyſtkich ſynów Izráelſkich : W wiecżór poznáćie / iż Pán wywiódł was z źiemi Egipſkiej ;
A ráno oglądáćie chwáłę Páńſką ; bo uſłyƺáł ƺemrániá wáƺe przećiw Pánu. A my co jeſteśmy / iż ƺemrzećie przećiwko nám?
Y rzekł Mojżeƺ : Dá wám Pán / w wiecżór mięſo do jedzeniá / á chleb ráno do náſyceniá ; bo uſłyƺáł Pán ƺemrániá wáƺe / któremi ƺemrzećie przećiw jemu. A my co jeſteśmy? Nie przećiwko nám ſą ƺemrániá wáƺe / ále przećiwko Pánu.
Y rzekł Mojżeƺ do Aároná : Mów do wƺyſtkiego zgromádzeniá ſynów Izráelſkich : Przyſtąpćie przed oblicżność Páńſką ; bo uſłyƺáł ƺemránie wáƺe.
Y ſtáło śię / gdy mówił Aáron do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / że ſpojrzeli ku puƺcży / á oto / chwałá Páńſká ukázáłá śię w obłoku.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Uſłyƺáłem ƺemránie ſynów Izráelſkich ; rzecżże do nich / mówiąc : Między dwomá wiecżorámi będźiećie jeść mięſo / á ráno náſyceni będźiećie chlebem / y poznáćie / żem Ja Pán / Bóg wáƺ.
Stáło śię tedy wiecżór / że śię zlećiáły przepiórki / á okryły obóz / á ráno roſá leżáłá około obozu ;
A gdy przeſtáłá pádáć roſá / oto / ukázáło śię ná puƺcży coś drobnego / okrągłego drobnego jáko ƺron ná źiemi.
Co gdy ujrzeli ſynowie Izráelſcy / mówili jeden do drugiego : Mán hu? bo nie wiedźieli / co było. Y rzekł Mojżeƺ do nich : Tenći jeſt chleb / który wám dáł Pán ku jedzeniu.
Toć jeſt / co rozkázáł Pán : Zbierájćie z niego káżdy / ile trzebá ku jedzeniu / po mierze Gomer ná oſobę / według licżby duƺ wáƺych ; káżdy ná tych / którzy ſą w námioćie jego / zbierájćie.
Y ucżynili ták ſynowie Izráelſcy / y zbieráli jedni więcey / drudzy mniej.
Y mierzyli w Gomer / y nie zbywáło temu / co więcey názbieráł / áni nie doſtáwáło temu / co mniej ; káżdy według tego / co mógł zjeść / názbieráł.
Mówił też Mojżeƺ do nich : Żáden niech nie zoſtáwiá z niego áż do zárániá.
Jednák nie uſłucháli Mojżeƺá ; ále zoſtáwili z niego niektórzy áż do poránku / y obróćiło śię w robáki / y zśmiárdło śię ; y rozgniewáł śię ná nie Mojżeƺ.
A zbieráli to ná káżdy dźień ráno / káżdy według tego / co mógł zjeść ; á gdy śię zágrzáło ſłońce / tedy ono topniáło.
A gdy było dniá ƺóſtego / zbieráli chleb w dwójnáſób / po dwu Gomer ná káżdego. Y zeƺły śię wƺyſtkie kśiążętá zgromádzeniá / oznájmując to Mojżeƺowi.
Który im rzekł : Toć jeſt / co mówił Pán : Odpocżynienie ſábbátu świętego Pánu jutro będźie ; co máćie piec / piecżćie / á co máćie wárzyć / wárzćie / á cokolwiek zbędźie / zoſtáwćie ſobie / á záchowájćie do jutrá.
Zoſtáwiáli tedy ono ná jutro / jáko był rozkázáł Mojżeƺ ; á nie zśmiárdło śię / y robák nie był w nim.
Y mówił Mojżeƺ : Jedzćież to dźiś / bo dźiś ſábbát Pánu ; dźiś nie znájdźiećie tego ná polu.
Przez ƺeść dni zbieráć to będźiećie / á dniá śiódmego ſábbát ; nie będźie weń mánny.
Y ſtáło śię dniá śiódmego / wyƺli niektórzy z ludu / áby zbieráli ; ále nie ználeſli.
Tedy rzekł Pán do Mojżeƺá : Y pókiż nie będźiećie chćieli przeſtrzegáć przykázáń moich y zakonu mego?
Pátrzćie / iż wám Pán dáł ſábbát / dla tego w dźień ƺóſty dáwá wám chleb ná dwá dni ; zoſtáwájćie káżdy ná miejſcu ſwem / niech nie wychodźi żáden z miejſcá ſwego w dźień śiódmy.
Y odpocżywáł lud dniá śiódmego.
Y názwáł dom Izráelſki imię onego pokármu Mán / który był jáko náśienie koryjándrowe / biáły / á ſmák jego jáko plácki z miodem.
Mówił też Mojżeƺ : Ták rozkázáł Pán : Nápełnij Gomer z niego ná chowánie w národźiech wáƺych / áby widźieli chleb ten / którymem was kármił ná puƺcży / gdym was wywiódł z źiemi Egipſkiej.
Rzekł zátem Mojżeƺ do Aároná : Weźmij wiádro jedno / á náſyp w nie pełen Gomer mánny / á poſtáw je przed Pánem ná chowánie do národów wáƺych.
Jáko przykázáł Pán Mojżeƺowi / ták poſtáwił je Aáron przed świádectwem ná chowánie.
A ſynowie Izráelſcy jedli mánnę przez cżterdźieśći lát / áż przyƺli do źiemi mieƺkániá ; mánnę jedli / áż przyƺli do gránic źiemi Chánánejſkiej.
A Gomer jeſt dźieśiątá cżęść miáry Efá.
