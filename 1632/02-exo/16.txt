Ruƺyli śię potem z Elimu / y przyƺło wƺyſtko mnóſtwo ſynów Izráelſkich ná puƺcżą Zyn / którá leży między Elim y między Synáj / piętnáſtego dniá mieśiącá wtórego po wyjśćiu ich z ziemi Egipſkiej.
Y ƺemráło wƺyſtko zgromádzenie ſynów Izráelſkich przećiw Mojżeƺowi y przećiw Aáronowi ná puƺcży.
A mówili do nich ſynowie Izráelſcy : Obyśmy byli pomárli od ręki Páńſkiej w ziemi Egipſkiej / gdyśmy śiádáli nád gárncy mięſá / gdyśmy śię nájádáli chlebá do ſytośći ; bo teráz wywiedliśćie náſ ná tę puƺcżą / ábyśćie pomorzyli to wƺyſtko mnóſtwo głodem.
Tedy rzekł Pán do Mojżeƺá : Oto / Já / ſpuƺcżę wám / jáko deƺcż chleb z niebá / y będzie wychodził lud / á będzie zbieráł / coby dość było ná káżdy dzień / ábym go doświádcżył / będzieli chodził w zákonie moim / cżyli nie.
Ale dniá ƺóſtego nágotują to / co przynioſą / á będzie tyle dwoje niż co zbieráć zwykli ná káżdy dzień.
Y mówił Mojżeƺ y Aáron do wƺyſtkich ſynów Izráelſkich : W wiecżór poznáćie / iż Pán wywiódł wáſ z ziemi Egipſkiej ;
A ráno oglądáćie chwáłę Páńſką ; bo uſłyƺáł ƺemrániá wáƺe przećiw Pánu. A my co jeſteśmy / iż ƺemrzećie przećiwko nám?
Y rzekł Mojżeƺ : Dá wám Pán / w wiecżór mięſo do jedzeniá / á chleb ráno do náſyceniá ; bo uſłyƺáł Pán ƺemrániá wáƺe / któremi ƺemrzećie przećiw jemu. A my co jeſteśmy? Nie przećiwko nám ſą ƺemrániá wáƺe / ále przećiwko Pánu.
Y rzekł Mojżeƺ do Aároná : Mów do wƺyſtkiego zgromádzeniá ſynów Izráelſkich : Przyſtąpćie przed oblicżność Páńſką ; bo uſłyƺáł ƺemránie wáƺe.
Y ſtáło śię / gdy mówił Aáron do wƺyſtkiego zgromádzeniá ſynów Izráelſkich / że ſpojrzeli ku puƺcży / á oto / chwáłá Páńſká ukázáłá śię w obłoku.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Uſłyƺáłem ƺemránie ſynów Izráelſkich ; rzecżże do nich / mówiąc : Między dwomá wiecżorámi będziećie jeść mięſo / á ráno náſyceni będziećie chlebem / y poznáćie / żem Já Pán / Bóg wáƺ.
Stáło śię tedy wiecżór / że śię zlećiáły przepiórki / á okryły obóz / á ráno roſá leżáłá około obozu ;
A gdy przeſtáłá pádáć roſá / oto / ukázáło śię ná puƺcży coś drobnego / okrągłego drobnego jáko ƺron ná ziemi.
Co gdy ujrzeli ſynowie Izráelſcy / mówili jeden do drugiego : Mán hu? bo nie wiedzieli / co było. Y rzekł Mojżeƺ do nich : Tenći jeſt chleb / który wám dáł Pán ku jedzeniu.
Toć jeſt / co rozkázáł Pán : Zbierájćie z niego káżdy / ile trzebá ku jedzeniu / po mierze Gomer ná oſobę / według licżby duƺ wáƺych ; káżdy ná tych / którzy ſą w námioćie jego / zbierájćie.
Y ucżynili ták ſynowie Izráelſcy / y zbieráli jedni więcej / drudzy mniej.
Y mierzyli w Gomer / y nie zbywáło temu / co więcej názbieráł / áni nie doſtáwáło temu / co mniej ; káżdy według tego / co mógł zjeść / názbieráł.
Mówił też Mojżeƺ do nich : Żáden niech nie zoſtáwiá z niego áż do zárániá.
Jednák nie uſłucháli Mojżeƺá ; ále zoſtáwili z niego niektórzy áż do poránku / y obróćiło śię w robáki / y zśmiárdło śię ; y rozgniewáł śię ná nie Mojżeƺ.
A zbieráli to ná káżdy dzień ráno / káżdy według tego / co mógł zjeść ; á gdy śię zágrzáło ſłońce / tedy ono topniáło.
A gdy było dniá ƺóſtego / zbieráli chleb w dwójnáſób / po dwu Gomer ná káżdego. Y zeƺły śię wƺyſtkie kśiążętá zgromádzeniá / oznájmując to Mojżeƺowi.
Który im rzekł : Toć jeſt / co mówił Pán : Odpocżynienie ſábbátu świętego Pánu jutro będzie ; co máćie piec / piecżćie / á co máćie wárzyć / wárzćie / á cokolwiek zbędzie / zoſtáwćie ſobie / á záchowájćie do jutrá.
Zoſtáwiáli tedy ono ná jutro / jáko był rozkázáł Mojżeƺ ; á nie zśmiárdło śię / y robák nie był w nim.
Y mówił Mojżeƺ : Jedzćież to dziś / bo dziś ſábbát Pánu ; dziś nie znájdziećie tego ná polu.
Przez ƺeść dni zbieráć to będziećie / á dniá śiódmego ſábbát ; nie będzie weń mánny.
Y ſtáło śię dniá śiódmego / wyƺli niektórzy z ludu / áby zbieráli ; ále nie ználeźli.
Tedy rzekł Pán do Mojżeƺá : Y pókiż nie będziećie chćieli przeſtrzegáć przykázáń moich y zákonu mego?
Pátrzćie / iż wám Pán dáł ſábbát / dlá tego w dzień ƺóſty dáwá wám chleb ná dwá dni ; zoſtáwájćie káżdy ná miejſcu ſwem / niech nie wychodzi żáden z miejſcá ſwego w dzień śiódmy.
Y odpocżywáł lud dniá śiódmego.
Y názwáł dom Izráelſki imię onego pokármu Mán / który był jáko náśienie koryjándrowe / biáły / á ſmák jego jáko plácki z miodem.
Mówił też Mojżeƺ : Ták rozkázáł Pán : Nápełnij Gomer z niego ná chowánie w národziech wáƺych / áby widzieli chleb ten / którymem wáſ kármił ná puƺcży / gdym wáſ wywiódł z ziemi Egipſkiej.
Rzekł zátem Mojżeƺ do Aároná : Weźmij wiádro jedno / á náſyp w nie pełen Gomer mánny / á poſtáw je przed Pánem ná chowánie do národów wáƺych.
Jáko przykázáł Pán Mojżeƺowi / ták poſtáwił je Aáron przed świádectwem ná chowánie.
A ſynowie Izráelſcy jedli mánnę przez cżterdzieśći lát / áż przyƺli do ziemi mieƺkániá ; mánnę jedli / áż przyƺli do gránic ziemi Chánánejſkiej.
A Gomer jeſt dzieśiątá cżęść miáry Efá.
