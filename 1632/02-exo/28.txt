A ty weźmij do śiebie Aároná / brátá twego / y ſyny jego z nim / z pośrodku ſynów Izráelſkich / áby mi urząd kápłáńſki odpráwowáli / Aáron / Nádáb y Abiu / Eleázár / y Itámár / ſynowie Aáronowi.
A ſpráwiƺ ƺáty święte Aáronowi / brátu twemu / ná cżeść y ná ozdobę.
Ty śię też rozmówiƺ z káżdym umiejętnym rzemieślnikiem / któregom nápełnił Duchem mądrośći / áby urobili ƺáty Aáronowi ná poświęcenie jego / áby mi urząd kápłáńſki odpráwowáł.
A teć ſą ƺáty / które urobią : Nápierśnik / y nárámiennik / y płáƺcż / y ſukniá háftowáná / cżápká y páſ. Y urobią te ƺáty święte Aáronowi brátu twemu y ſynom jego / áby mi kápłáńſki urząd ſpráwowáli.
Y nábiorą złotá / y hijácyntu / y ƺárłátu / y kármázynu dwá kroć fárbowánego / y jedwábiu biáłego.
Y ucżynią nárámiennik ze złotá / y z hijácyntu / y z ƺárłátu / z kármázynu dwá kroć fárbowánego / y z jedwábiu biáłego kręconego / robotą háftárſką.
Dwá zwierzchne kráje zƺyte mieć będźie ná dwu końcách ſwych / á ták ſpołu ſpięte będą.
A przepáſánie nárámienniká tego / które ná nim będźie / podobne będźie roboćie jego ; będźie tákże ze złotá / z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z jedwábiu biáłego kręconego.
Y weźmieƺ dwá kámienie onychiny / y wyryjeƺ ná nich imioná ſynów Izráelſkich ;
Sześć imion ich ná jednym kámieniu / á imion ƺeść drugich ná drugim kámieniu / według národzeniá ich.
Robotą ſnycerzów / którzy kámienie rzezą / wyryjeƺ ná obu kámieniách imioná ſynów Izráelſkich / y oſádźiƺ je we złote oſádzeniá.
Y położyƺ te obádwá kámienie ná wierzchnich krájách nárámienniká / kámienie pámiątki dla ſynów Izráelſkich ; y nośić będźie Aáron imioná ich przed Pánem ná obu rámionách ſwych ná pámiątkę.
Ucżyniƺ też hácżyki złote.
Dwá też łáńcuƺki ze złotá ƺcżerego jednoſtájne ; ucżyniƺ je robotą plećioną / y záwieśiƺ te łáńcuƺki plećione ná hácżykách.
Ucżyniƺ też nápierśnik ſądu robotą háftárſką / według roboty nárámienniká urobiƺ go ; ze złotá / z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu kręconego ucżyniƺ go.
Cżworográniáſty będźie y dwoiſty / ná piędźi długość jego / y ná piędźi ƺerokość jego.
Y náſádźiƺ weń pełno kámieniá / cżtery rzędy kámieniá / tym porządkiem : ſárdyjuƺ / topázyjuƺ y ƺmárágd w pierwƺym rzędźie ;
W drugim záśię rzędźie : kárbunkuł / ſáfir / y jáſpiſ.
A w trzećim rzędźie : linkuryjuƺ / ácháteſ y ámetyſt.
A w cżwártym rzędźie : chryzolit / onychin y berył ; te będą wſádzone w złoto w rzędźiech ſwoich.
A tych kámieni z imionámi ſynów Izráelſkich będźie dwánáśćie według imion ich ; ták jáko rzezą piecżęći / káżdy według imieniá ſwego będą / dla dwunáſtu pokoleń.
Ucżyniƺ też do nápierśniká łáńcuƺki jednoſtájne robotą plećioną ze złotá ƺcżerego.
Ucżyniƺ też do nápierśniká dwá kolce złote / y przypráwiƺ te dwá kolce do obu krájów nápierśniká.
Y przewlecżeƺ dwá łáńcuƺki złote przez obá kolce u krájów nápierśniká.
Drugie záśię dwá końce dwu łáńcuƺków záwlecżeƺ ná dwá hácżyki / y przypráwiƺ do wierzchnich krájów nárámienniká ná przodku.
Ucżyniƺ też dwá kolce złote / które przypráwiƺ do dwu końców nápierśniká ná kráju jego / który jeſt od nárámienniká ze ſpodku.
Do tego ucżyniƺ dwá drugie kolce złote / które przypráwiƺ ná dwie ſtrony nárámienniká ze ſpodku ná przećiwko ſpojeniu jego / z wierzchu nád przepáſániem nárámienniká.
Ták zwiążą nápierśnik ten kolce jego z kolcámi nárámienniká ƺnurem hijácyntowym / áby był nád przepáſániem nárámienniká / żeby nie odewſtáwáł nápierśnik od nárámienniká.
Y będźie nośił Aáron imioná ſynów Izráelſkich ná nápierśniku ſądu / ná pierśiách ſwych / gdy będźie wchodźił do świątnicy / ná pámiątkę uſtáwicżną przed Pánem.
Położyƺ też ná nápierśniku ſądu Urim y Tummim / które będą ná pierśiách Aáronowych / gdy wchodźić będźie przed Páná ; y ponieśie Aáron ſąd ſynów Izráelſkich ná pierśiách przed Pánem uſtáwicżnie.
Ucżyniƺ też płáƺcż pod nárámiennik / wƺyſtek z hijácyntu.
A ná wierzchu w pośród jego będźie rozpór / który rozpór obwiedźieƺ brámą plećioną w páncerzowy wzór / áby śię nie rozdźieráł.
Ucżyniƺ też ná podołku jego jábłká gránátowe z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego ná podołku jego w około / á dzwonki złote między niemi w około.
Dzwonek złoty á jábłko gránátowe ; y záś dzwonek złoty y jábłko gránátowe u podołká płáƺcżá w około.
A będźie to miał ná ſobie Aáron przy poſługowániu / áby ſłyƺány był dźwięk jego / gdy będźie wchodźił do świątnicy przed Páná / y gdy záś wychodźić będźie / żeby nie umárł.
Ucżyniƺ też bláchę ze złotá ƺcżerego / á wyryjeƺ ná niej robotą tych / co piecżęći rzezą : Świętość Pánu.
Tę przywiążeƺ do ƺnuru hijácyntowego / y będźie ná cżápce ; ná przodku ná cżápce będźie.
A tá będźie nád cżołem Aáronowem / áby nośił Aáron niepráwość poświęconych rzecży / któreby poświęcáli ſynowie Izráelſcy przy wƺyſtkich dárách poświęconych rzecży ſwych ; á będźie nád cżołem jego uſtáwicżnie / áby im zjednáł łáſkę u Páná.
Spráwiƺ też ƺátę z biáłego jedwábiu dźiáną ; tákże ucżyniƺ cżápkę z jedwábiu biáłego / páſ też ucżyniƺ robotą háftárſką.
Synom tákże Aáronowym pocżyniƺ ƺáty ; y pocżyniƺ im páſy / y cżápki im pocżyniƺ ná cżeść y ná ozdobę.
A ubierzeƺ w nie Aároná / brátá twego / y ſyny jego z nim ; y pomáżeƺ je / á nápełniƺ ręce ich / y poświęćiƺ je / áby mi urząd kápłáńſki ſpráwowáli.
Urobiƺ im też ubiory lniáne / dla zákryćiá nágośći ćiáłá ; od biódr áż do udów będą.
A będą ná Aáronie y ná ſynách jego / gdy wchodźić będą do námiotu zgromádzeniá / álbo gdy będą przyſtępowáć do ołtárzá / áby ſłużyli w świątnicy / żeby nioſąc niepráwość / nie pomárli. Uſtáwá to wiecżná będźie jemu / y náśieniu jego po nim.
