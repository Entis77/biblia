A widząc lud / iż omieƺkiwáł Mojżeƺ zejść z góry / tedy zebráł śię lud przećiw Aáronowi / y mówili do niego : Wſtáń / ucżyń nám bogi / którzyby ƺli przed námi ; bo Mojżeƺowi / mężowi temu / który náſ wywiódł z ziemi Egipſkiej / nie wiemy co śię ſtáło.
Tedy im rzekł Aáron : Odejmijćie náuƺnice złote / które ſą ná uƺách żon wáƺych / ſynów wáƺych / y córek wáƺych / á przynieśćie do mnie.
Y poodrywáł wƺyſtek lud náuƺnice złote / które były ná uƺách ich / á przynieśli do Aároná.
Które gdy odebráł z ręku ich / wykƺtáłtowáł je rylcem y ucżynił z nich ćielcá odlewánego. Y rzekli : Ci ſą bogowie twoi / Izráelu / którzy ćię wywiedli z ziemi Egipſkiej.
Co ujrzáwƺy Aáron / zbudowáł ołtárz przed nim ; á wołájąc Aáron mówił : Święto Páńſkie jutro będzie.
A wſtáwƺy bárdzo ráno názájutrz / ofiárowáli cáłopáleniá / y przywiedli ofiáry ſpokojne ; y śiádł lud / áby jádł y pił / y wſtáli gráć.
Tedy rzekł Pán do Mojżeƺá : Idź / zſtąp ; bo śię popſowáł lud twój / któryś wywiódł z ziemi Egipſkiej.
Uſtąpili prędko z drogi / którąm im przykázáł ; ucżynili ſobie ćielcá odlewánego / y kłániáli śię mu / y ofiárowáli mu mówiąc : Ci ſą bogowie twoi / Izráelu / którzy ćię wywiedli z ziemi Egipſkiej.
Rzekł záśię Pán do Mojżeƺá : Widziáłem lud ten / á oto / jeſt lud twárdego kárku.
Przetoż teráz puść mię / że śię rozpáli popędliwość mojá ná nie y wygłádzę je ; á ćiebie ucżynię w náród wielki.
Y modlił śię Mojżeƺ Pánu Bogu ſwemu / á rzekł : Przecżże o Pánie / rozpálá śię popędliwość twojá przećiwko ludowi twemu / któryś wywiódł z ziemi Egipſkiej mocą wielką y ręką możną?
A przecżżeby Egipcżánie rzec mieli / mówiąc : Ná ich złe wywiódł je / áby je pobił ná górách / y áby je wygłádził z wierzchu ziemi? Odwróć śię od gniewu zápálcżywośći twojej / á ulituj śię nád złem ludu twego.
Wſpomnij ná Abráhámá / Izááká / y Izráelá / ſługi twoje / którymeś przyśiągł ſám przez śię y mówiłeś do nich : Rozmnożę náśienie wáƺe jáko gwiázdy niebieſkie / y wƺyſtkę tę ziemię / o którejm mówił. Dám ją náśieniu wáƺemu / y odziedzicżą ją ná wieki.
Y użálił śię Pán nád złem / które mówił / że ucżynić miáł ludowi ſwemu.
A obróćiwƺy śię Mojżeƺ zſtąpił z góry / dwie táblice świádectwá májąc w ręku ſwych / táblice piſáne po obu ſtronách ; y ná tej / y ná owej ſtronie były piſáne.
A one táblice robotą Bożą były ; piſmo tákże piſmo Boże było / wyryte ná táblicách.
A uſłyƺáwƺy Jozue głoſ ludu wołájącego / rzekł do Mojżeƺá : Głoſ bitwy w obozie.
Który odpowiedziáł : Nie jeſt to głoſ zwyćiężájących / áni głoſ poráżonych ; głoſ śpiewájących já ſłyƺę.
Y ſtáło śię / gdy śię przybliżył do obozu / że ujrzáł ćielcá y táńce ; á rozgniewáwƺy śię bárdzo Mojżeƺ porzućił z ręku ſwoich táblice / y ſtłukł je pod górą.
Wziął też ćielcá / którego byli ucżynili / y ſpálił go w ogniu / y ſkruƺył go áż ná proch / á wyſypáwƺy ná wodę / dáł pić ſynom Izráelſkim.
Y rzekł Mojżeƺ do Aároná : Cóż ći ten lud ucżynił / żeś wprowádził náń grzech wielki?
Odpowiedziáł Aáron : Niech śię nie rozpálá gniew páná mego ; ty znáƺ ten lud / jáko do złego ſkłonny jeſt.
Bo mi mówili : Ucżyń nám bogi / którzyby ƺli przed námi / gdyż Mojżeƺowi / mężowi temu / który náſ wywiódł z ziemi Egipſkiej / nie wiemy / co śię ſtáło.
Y odpowiedziáłem im : Kto má złoto / odrywájćie je z śiebie. Y dáli mi / y wrzućiłem je w ogień / y uláł śię ten ćielec.
Widząc tedy Mojżeƺ lud obnáżony / ( bo go był złupił Aáron ná zelżenie przed nieprzyjáćiołmi ich ).
Stánął Mojżeƺ w brámie obozu / y rzekł : Kto Páńſki / przyſtąp do mnie. Y zebráli śię do niego wƺyſcy ſynowie Lewiego.
Y rzekł do nich : Ták mówi Pán / Bóg Izráelſki : Przypáƺ káżdy miecż ſwój do biodry ſwojej ; przychodźćie á wrácájćie śię od brámy do brámy w obozie / á zábijájćie káżdy brátá ſwego / y káżdy przyjáćielá ſwego / y káżdy bliźniego ſwego.
Y ucżynili ſynowie Lewiego według ſłowá Mojżeƺowego ; y poległo z ludu dniá onego około trzech tyśięcy mężów.
Bo był rzekł Mojżeƺ : Poświęććie ręce ſwoje dziś Pánu / káżdy ná ſynu ſwym / y ná brátu ſwym / áby wám dáne było dziś błogoſłáwieńſtwo.
A gdy było názájutrz / mówił Mojżeƺ do ludu : Wyśćie zgrzeƺyli grzechem wielkim ; przetoż teráz wſtąpię do Páná / ázá go ubłágám zá grzech wáƺ.
Wróćiwƺy śię tedy Mojżeƺ do Páná / mówił : Proƺę / zgrzeƺył ten lud grzechem wielkim ; bo ſobie ucżynili bogi złote.
Teráz tedy / álbo odpuść grzech ich / álbo jeźli nie / wymáż mię proƺę z kśiąg twoich / któreś nápiſáł.
Y rzekł Pán do Mojżeƺá : Kto mi zgrzeƺył / tego wymáżę z kśiąg moich.
A teráz idź / prowádź ten lud / gdziem ći rozkázáł. Oto / Anioł mój pójdzie przed tobą ; ále w dzień náwiedzeniá mego náwiedzę też y ná nich ten grzech ich.
Skáráł tedy Pán lud / przeto / że ucżynili byli ćielcá / którego był ucżynił Aáron.
