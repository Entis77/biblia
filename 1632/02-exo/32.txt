A widząc lud iż omieƺkiwał Mojzeƺ zejść z góry / tedy zebrał śię lud przećiw Aáronowi / y mówili do niego : Wſtań / ucżyń nam Bogi / którzyby ƺli przed námi : bo Mojzeƺowi / mężowi temu / który nas wywiódł z źiemie Egipſkiey nie wiemy co śię ſtáło.
Tedy im rzekł Aáron : Odejmićie náuƺnice złote które ſą ná uƺách żon wáƺych / ſynów wáƺych / y córek wáƺych : á przynieśćie do mnie.
Y poodrywał wƺyſtek lud náuƺnice złote które <i>były</i> ná uƺách ich / á przynieśli do Aároná.
Które gdy odebrał z ręku ich / wykƺtałtował je rylcem y ucżynił z nich ćielcá odlewánego : Y rzekli / ći ſą Bogowie twoji Izráelu / którzy ćię wywiedli z źiemie Egipſkiey.
Co ujrzawƺy Aáron / zbudował Ołtarz przed nim : A wołájąc Aáron mówił : święto Páńſkie jutro <i>będźie</i>.
A wſtawƺy bárzo ráno názájutrz / ofiárowáli cáłopalenia / y przywiedli Ofiáry ſpokojne / y śiadł lud áby jadł y pił / y wſtáli gráć.
Tedy rzekł PAN do Mojzeƺá : Idź / zſtąp / bo śię popſował lud twój któryś wywiódł z źiemie Egipſkiey.
Uſtąpili prętko z drogi którąm im przykazał / ucżynili ſobie ćielcá odlewánego / y kłaniáli śię mu : y ofiárowáli mu / mówiąc : ci <i>ſą</i> Bogowie twoji Izráelu / którzy ćię wywiedli z źiemie Egipſkiey.
Rzekł záśię PAN do Mojzeƺá / Widźiałem lud ten / á oto <i>jeſt</i> lud twárdego kárku.
Przetoż teraz puść mię / że śię rozpali popędliwość mojá ná nie / y wygłádzę je : á ćiebie uczynię w Naród wielki.
Y modlił śię Mojzeƺ PANU Bogu ſwemu / á rzekł : Przecżże / o PAnie / rozpala śię popędliwość twojá przećiwko ludowi twemu / któryś wywiódł z źiemie Egipſkiey mocą wielką / y ręką możną?
A przecżżeby Egipcżánie rzec mieli / mówiąc : Ná ich złe wywiódł je / áby je pobił ná górách / y áby je wygłádźił z wierzchu źiemie : Odwróć śię od gniewu zápálcżywośći twojey / á ulituj śię nád złym ludu twego.
Wſpomni ná Abráhámá / Izááká / y Izráelá ſługi twoje / którymeś przyśiągł ſam przez śię / y mówiłeś do nich : Rozmnożę Naśienie wáƺe jáko gwiazdy Niebieſkie / y wƺyſtkę tę źiemię o któreym mówił / dam ją Naśieniu wáƺemu / y odźiedźiczą <i>ją</i> ná wieki.
Y użalił śię PAN nád złym które mówił / że uczynić miał ludowi ſwemu.
A obróćiwƺy śię Mojzeƺ zſtąpił z góry / dwie tablice świádectwá májąc w ręku ſwych : tablice piſáne po obu ſtronách / y ná tey y ná owey ſtronie <i>były</i> piſáne.
A one tablice robotą Bożą były : Piſmo tákże piſmo Boże było / wyryte ná tablicách.
A uſłyƺawƺy Jozue głos ludu wołájącego / rzekł do Mojzeƺá : Głos bitwy w Oboźie.
Który odpowiedźiał : Nie jeſt to głos zwyćiężájących / áni głos poráżonych / głos śpiewájących ja ſłyƺę.
Y ſtáło śię gdy śię przybliżył do Obozu / że ujrzał ćielcá y tańce : á rozgniewawƺy śię bárzo Mojzeƺ / porzućił z ręku ſwojich tablice / y ſtłukł je pod górą.
Wźiął też ćielcá / którego byli ucżynili / y ſpalił go w ogniu / y ſkruƺył go áż ná proch / á wyſypawƺy ná wodę / dał pić ſynom Izráelſkim.
Y rzekł Mojzeƺ do Aároná : Cóżći ten lud uczynił / żeś wprowádźił nań grzech wielki?
Odpowiedźiał Aáron : Niech śię nie rozpala gniew Páná mego : Ty znaƺ <i>ten</i> lud / jáko do złego <i>ſkłonny</i> jeſt.
Bo mi mówili : Ucżyń nam Bogi / którzyby ƺli przed námi : gdyż Mojzeƺowi mężowi temu / który nas wywiódł z źiemie Egipſkiey / nie wiemy co śię ſtáło.
Y odpowiedźiałem im / kto ma złoto / odrywajćie je z śiebie. Y dáli mi / y wrzućiłem je w ogień / y ulał śię ten ćielec.
Widząc tedy Mojzeƺ lud obnáżony ( bo go był złupił Aáron ná zelżenie przed Nieprzyjaćiołmi ich ; )
Stánął Mojzeƺ w bramie Obozu / y rzekł : kto Páńſki / <i>przyſtąp</i> do mnie. Y zebráli śię do niego wƺyſcy Synowie Lewiego.
Y rzekł do nich : Ták mówi PAN / Bóg Izráelſki : przypáƺ káżdy miecz ſwój do biodry ſwojey / przychodźćie á wracajćie śię / od bramy do bramy w Oboźie : á zábijajćie / káżdy brátá ſwego / y káżdy przyjaćielá ſwego / y káżdy bliźniego ſwego :
Y uczynili Synowie Lewiego / według ſłowá Mojzeƺowego : y poległo z ludu dniá onego / około trzech tyśięcy mężów.
Bo był rzekł Mojzeƺ : Poświęććie ręce ſwoje dźiś PANU / káżdy ná Synu ſwym / y ná brátu ſwym : áby wam dáne było dźiś błogoſłáwieńſtwo.
A gdy było názájutrz / mówił Mojzeƺ do ludu : Wyśćie zgrzeƺyli grzechem wielkim : przetoż teraz wſtąpię do PANA / áza go ubłagam zá grzech wáƺ.
Wróćiwƺy śię tedy Mojzeƺ do PANA / mówił : Proƺę / zgrzeƺył ten lud grzechem wielkim : bo ſobie uczynili Bogi złote.
Teraz tedy / álbo odpuść grzech ich / álbo jesli nie / wymaƺ mię proƺę z kśiąg twojich / któreś nápiſał.
Y rzekł PAN do Mojzeƺá : kto mi zgrzeƺył / tego wymáżę z kśiąg mojich.
A teraz idź / prowadź ten lud / gdźiem ći rozkazał / Oto Anjoł mój pójdźie przed tobą : ále w dźień náwiedzenia mego / náwiedzę też y ná nich <i>ten</i> grzech ich.
Skarał tedy PAN lud / przeto / że uczynili byli ćielcá / którego był uczynił Aáron.
