A widząc lud / iż omieƺkiwał Mojzeƺ zejść z góry / tedy zebrał śię lud przećiw Aáronowi / y mówili do niego : Wſtáń / ucżyń nam bogi / którzyby ƺli przed námi ; bo Mojżeƺowi / mężowi temu / który nas wywiódł z źiemi Egipſkiej / nie wiemy co śię ſtáło.
Tedy im rzekł Aáron : Odejmijćie náuƺnice złote / które ſą ná uƺách żon wáƺych / ſynów wáƺych / y córek wáƺych / á przynieśćie do mnie.
Y poodrywał wƺyſtek lud náuƺnice złote / które były ná uƺách ich / á przynieśli do Aároná.
Które gdy odebrał z ręku ich / wykƺtáłtował je rylcem y ucżynił z nich ćielcá odlewánego. Y rzekli : Ci ſą bogowie twoi / Izráelu / którzy ćię wywiedli z źiemi Egipſkiej.
Co ujrzáwƺy Aáron / zbudował ołtárz przed nim ; á wołájąc Aáron mówił : Święto Páńſkie jutro będźie.
A wſtáwƺy bárzo ráno názájutrz / ofiárowáli cáłopalenia / y przywiedli ofiáry ſpokojne ; y śiádł lud / áby jádł y pił / y wſtáli gráć.
Tedy rzekł Pán do Mojżeƺá : Idź / zſtąp ; bo śię popſował lud twój / któryś wywiódł z źiemi Egipſkiej.
Uſtąpili prędko z drogi / którąm im przykázał ; ucżynili ſobie ćielcá odlewánego / y kłániáli śię mu / y ofiárowáli mu mówiąc : Ci ſą bogowie twoi / Izráelu / którzy ćię wywiedli z źiemi Egipſkiej.
Rzekł záśię Pán do Mojżeƺá : Widźiałem lud ten / á oto / jeſt lud twárdego kárku.
Przetoż teraz puść mię / że śię rozpáli popędliwość mojá ná nie y wygłádzę je ; á ćiebie ucżynię w naród wielki.
Y modlił śię Mojzeƺ Pánu Bogu ſwemu / á rzekł : Przecżże o Pánie / rozpálá śię popędliwość twojá przećiwko ludowi twemu / któryś wywiódł z źiemi Egipſkiej mocą wielką y ręką możną?
A przecżżeby Egipcżánie rzec mieli / mówiąc : Ná ich złe wywiódł je / áby je pobił ná górách / y áby je wygłádźił z wierzchu źiemi? Odwróć śię od gniewu zápálcżywośći twojey / á ulituj śię nád złem ludu twego.
Wſpomnij ná Abráhámá / Izááká / y Izráelá / ſługi twoje / którymeś przyśiągł ſám przez śię y mówiłeś do nich : Rozmnożę naśienie wáƺe jáko gwiázdy niebieſkie / y wƺyſtkę tę źiemię / o którejm mówił. Dam ją náśieniu wáƺemu / y odźiedźicżą ją ná wieki.
Y użálił śię Pán nád złem / które mówił / że ucżynić miał ludowi ſwemu.
A obróćiwƺy śię Mojzeƺ zſtąpił z góry / dwie táblice świádectwá májąc w ręku ſwych / táblice piſáne po obu ſtronách ; y ná tey / y ná owej ſtronie były piſáne.
A one táblice robotą Bożą były ; piſmo tákże piſmo Boże było / wyryte ná táblicách.
A uſłyƺáwƺy Jozue głos ludu wołájącego / rzekł do Mojżeƺá : Głos bitwy w oboźie.
Który odpowiedźiał : Nie jeſt to głos zwyćiężájących / áni głos poráżonych ; głos śpiewájących ja ſłyƺę.
Y ſtáło śię / gdy śię przybliżył do obozu / że ujrzał ćielcá y táńce ; á rozgniewáwƺy śię bárzo Mojzeƺ porzućił z ręku ſwojich táblice / y ſtłukł je pod górą.
Wźiął też ćielcá / którego byli ucżynili / y ſpálił go w ogniu / y ſkruƺył go áż ná proch / á wyſypáwƺy ná wodę / dał pić ſynom Izráelſkim.
Y rzekł Mojzeƺ do Aároná : Cóż ći ten lud ucżynił / żeś wprowádźił nań grzech wielki?
Odpowiedźiał Aáron : Niech śię nie rozpálá gniew páná mego ; ty znáƺ ten lud / jáko do złego ſkłonny jeſt.
Bo mi mówili : Ucżyń nam bogi / którzyby ƺli przed námi / gdyż Mojżeƺowi / mężowi temu / który nas wywiódł z źiemi Egipſkiej / nie wiemy / co śię ſtáło.
Y odpowiedźiáłem im : Kto ma złoto / odrywájćie je z śiebie. Y dáli mi / y wrzućiłem je w ogień / y ulał śię ten ćielec.
Widząc tedy Mojzeƺ lud obnáżony / ( bo go był złupił Aáron ná zelżenie przed nieprzyjáćiołmi ich ).
Stánął Mojzeƺ w bramie obozu / y rzekł : Kto Páńſki / przyſtąp do mnie. Y zebráli śię do niego wƺyſcy ſynowie Lewiego.
Y rzekł do nich : Ták mówi Pán / Bóg Izráelſki : Przypáƺ káżdy miecż ſwój do biodry ſwojey ; przychodźćie á wrácájćie śię od bramy do bramy w oboźie / á zábijájćie káżdy brátá ſwego / y káżdy przyjáćielá ſwego / y káżdy bliźniego ſwego.
Y ucżynili ſynowie Lewiego według ſłowá Mojżeƺowego ; y poległo z ludu dniá onego około trzech tyśięcy mężów.
Bo był rzekł Mojzeƺ : Poświęććie ręce ſwoje dźiś Pánu / káżdy ná ſynu ſwym / y ná brátu ſwym / áby wam dáne było dźiś błogoſłáwieńſtwo.
A gdy było názájutrz / mówił Mojzeƺ do ludu : Wyśćie zgrzeƺyli grzechem wielkim ; przetoż teraz wſtąpię do Páná / ázá go ubłágám zá grzech wáƺ.
Wróćiwƺy śię tedy Mojzeƺ do Páná / mówił : Proƺę / zgrzeƺył ten lud grzechem wielkim ; bo ſobie ucżynili bogi złote.
Teraz tedy / álbo odpuść grzech ich / álbo jeſli nie / wymáż mię proƺę z kśiąg twojich / któreś nápiſał.
Y rzekł Pán do Mojżeƺá : Kto mi zgrzeƺył / tego wymáżę z kśiąg mojich.
A teraz idź / prowádź ten lud / gdźiem ći rozkázał. Oto / Anjoł mój pójdźie przed tobą ; ále w dźień náwiedzeniá mego náwiedzę też y ná nich ten grzech ich.
Skárał tedy Pán lud / przeto / że ucżynili byli ćielcá / którego był ucżynił Aáron.
