Rzekł jeƺcże Pán do Mojżeƺá y do Aároná w ziemi Egipſkiej / mówiąc :
Mieśiąc ten będzie wám pocżątkiem mieśiąców : pierwƺy wám będzie między mieśiącámi w roku.
Rzecżćie do wƺyſtkiego zgromádzeniá Izráelſkiego / mówiąc : Dzieśiątego dniá mieśiącá tego weźmie ſobie káżdy báránká według fámilii / báránká według domu.
A jeźliby mniejƺy był dom niżeliby zjeść mogli báránká / tedy przybierze y ſąśiádá ſwego / który jeſt nájbliżƺy domu jego / według licżby duƺ / nálicżywƺy tyle oſób / ileby ich zjeść mogło báránká.
Báránká zupełnego / ſámcá rocżnego / mieć będziećie ; z owiec álbo z kóz weźmiećie go.
Y będziećie go chowáli áż do cżternáſtego dniá mieśiącá tego ; á zábije go wƺyſtko zebránie zgromádzeniá Izráelſkiego między dwomá wiecżorámi.
Y wezmą ze krwi jego / y pokropią obádwá podwoje y nádprożnik u domu ; w którym go będą ſpożywáć.
Y będą jeść mięſo onej nocy piecżone przy ogniu / y przáśniki / z zioły gorzkiemi będą go jeść.
Nie jedzćie z niego nic ſurowego / áni wárzonego w wodzie / ále upiecżone przy ogniu. Głowę jego z nogámi jego / z wnętrznośćiámi jego.
A nie zoſtánie z niego nic do jutrá ; á jeźliby co z niego do jutrá zoſtáło / ogniem ſpálićie.
Ták go tedy pożywáć będziećie : Biodrá ſwe przepáƺećie / obuwie wáƺe będzie ná nogách wáƺych / á láſká wáƺá w ręce wáƺej / á jeść go będziećie ſpieƺno / ábowiem przejśćie jeſt Páńſkie.
Gdyż przejdę przez ziemię Egipſką tej nocy / y zábiję wƺelkie pierworodne w ziemi Egipſkiej / od cżłowieká áż do bydlęćiá / y nád wƺyſtkimi bogi Egipſkimi wykonám ſądy / Já Pán.
A będzie wám oná krew ná znák ná domách / w których będziećie ; bo ujrzáwƺy krew / minę wáſ / że nie będzie u wáſ plágá ku zátráceniu / gdy będę zábijáł w ziemi Egipſkiej.
A będzie wám ten dzień ná pámiątkę ; y będziećie go obchodzić zá święto Pánu w národziech wáƺych ; uſtáwą wiecżną obchodzić go będziećie.
Przez śiedm dni przáśniki jeść będziećie / á pierwƺego dniá záráz wypróżnićie kwáſ z domów wáƺych ; bo ktobykolwiek jádł co kwáƺonego od pierwƺego dniá áż do dniá śiódmego / wytráconá będzie duƺá oná z Izráelá.
W tenże dzień pierwƺy będzie zebránie święte / tákże dniá śiódmego zgromádzenie święte mieć będziećie ; żádnej roboty nie będziećie w nich cżynić / oprócż tego / cżego káżdy do jedzeniá używá / to ſámo gotowáć będziećie.
Y będziećie przeſtrzegáć przáśników ; ábowiem w ten dzień wywiodłem wojſká wáƺe z ziemi Egipſkiej ; przetoż przeſtrzegáć będziećie dniá tego w národziech wáƺych uſtáwą wiecżną.
Pierwƺego mieśiącá / cżternáſtego dniá tegoż mieśiącá / ná wiecżór jeść będziećie przáśniki áż do dniá dwudzieſtego pierwƺego tegoż mieśiącá ná wiecżór.
Przez śiedm dni kwáſ niech śię nie znájduje w domách wáƺych ; bo ktobykolwiek jádł co kwáƺonego / wytráconá będzie duƺá jego z zgromádzeniá Izráelſkiego / ták przychodzień / jáko y zrodzony w ziemi.
Nic kwáƺonego jeść nie będziećie ; we wƺyſtkich mieƺkániách wáƺych jeść będziećie przáśniki.
Wezwáł tedy Mojżeƺ wƺyſtkich ſtárƺych Izráelſkich / y rzekł do nich : Odłącżćie / á weźmijćie ſobie báránká według fámilii ſwych / á zárznijćie ná święto przejśćiá.
Weźmiećie też ſnopek hyſopu / y omocżyćie we krwi / którá będzie w miednicy / á pokropićie odrzwi / y obá podwoje oną krwią / którá będzie w miednicy ; á z wáſ nie wynijdzie żáden ze drzwi domu ſwego áż do poránku.
Bo przejdzie Pán zábijájąc Egipt ; á ujrzáwƺy krew ná odrzwiách y ná obu podwojách / przeſtąpi Pán drzwi / y niedopuśći morderzowi / wchodzić do domów wáƺych zábijáć wáſ.
Y przeſtrzegáć będziećie tego zá uſtáwę / tobie y ſynom twoim áż ná wieki.
A gdy wnijdziećie do ziemi / którą wám dá Pán / jáko obiecáł / tych obrzędów przeſtrzegáć będziećie.
A gdy wám rzeką ſynowie wáśi : Co to zá obrzędy wáƺe?
Tedy rzecżećie : Ofiárá to przejśćiá Páńſkiego / który przeſtępowáł domy ſynów Izráelſkich w Egipćie / gdy zábijáł Egipt / á domy náƺe wyzwáláł. Zátem ſchylił śię lud / y pokłonił śię.
Y poƺedłƺy ucżynili ſynowie Izráelſcy / jáko rozkázáł Pán Mojżeƺowi y Aáronowi / ták ucżynili.
Y ſtáło śię o północy / że Pán zábijáł wƺyſtkie pierworodztwá w ziemi Egipſkiej / od pierworodnego Fáráonowego / śiedzącego ná ſtolicy jego / áż do pierworodnego więźniá / który był w więzieniu / y wƺelkie pierworodne z bydląt.
Zátem wſtáł Fáráo onej nocy / y wƺyſcy ſłudzy jego / y wƺyſtek Egipt / y wƺcżął śię wielki krzyk w Egipćie ; bo nie było domu / w którymby nie był umárły.
A wezwáwƺy Fáráo Mojżeƺá y Aároná w nocy / rzekł : Wſtáńćie / wynijdźćie z pośrodku ludu mego / y wy y ſynowie Izráelſcy / á poƺedłƺy ſłużćie Pánu / jákośćie mówili.
Nádto trzody wáƺe / y bydłá wáƺe zábierzćie / jákośćie żądáli / á odchodząc / mnie też błogoſłáwćie.
Y przynágláli Egipcżánie ludowi / áby ich co rychlej wypráwili z ziemi ; bo mówili : Pomrzemy wƺyſcy.
Wziął tedy lud ćiáſtá ſwe / pierwej niż zákiśiáły ; á one ćiáſtá ſwe uwinąwƺy w ƺáty ſwe / kłádli ná rámioná ſwoje.
Tedy ſynowie Izráelſcy ucżynili według rozkázániá Mojżeƺowego / y wypożycżáli u Egipcżán nácżyniá ſrebrnego / y nácżyniá złotego / y ƺát.
A Pán dáł łáſkę ludowi w ocżách Egipcżánów / że im pożycżáli ; y złupili Egipt.
Ciągnęli tedy ſynowie Izráelſcy z Rámeſeſ do Suchotu / około ƺeść kroć ſto tyśięcy pieƺych mężów tylko / okrom dzieći.
Ale y ludu poſpolitego wiele ƺło z nimi / y owiec / y bydłá / dobytek bárdzo wielki.
Y popiekli z ćiáſtá zádziáłánego / które wynieśli z Egiptu / plácki przáśne ; bo nie było zákwáƺone / przeto że wygnáni byli z Egiptu / á nie mogli zmieƺkáć ; żywnośći też ſobie byli nie przygotowáli.
A cżáſu mieƺkániá ſynów Izráelſkich / którego mieƺkáli w Egipćie / było cżtery ſtá lát / y trzydzieśći lát.
Y ſtáło śię po cżterech ſet lát y trzydzieſtu lát / ſtáło śię onegoż dniá / wyƺły wƺyſtkie wojſká Páńſkie z ziemi Egipſkiej.
Noc tá obchodzoná má być Pánu / że je wywiódł ze ziemi Egipſkiej. Tá tedy noc Pánu obchodzoná má być od wƺyſtkich ſynów Izráelſkich w národziech ich.
Y rzekł Pán do Mojżeƺá y do Aároná : Tá jeſt uſtáwá świętá przejśćiá : Żáden obcy nie będzie jádł z niego.
Przetoż káżdego ſługę wáƺego / á zá pieniądze kupionego / obrzeżećie go / tedy będzie jádł z niego.
Przychodzień y nájemnik nie będzie jádł z niego.
W domu jednym będzie jedzony ; nie wynieśieƺ nic z domu z mięſá jego / á kośći nie złámiećie w nim.
Wƺyſtko zgromádzenie Izráelſkie ták ucżyni z nim.
A jeźliby kto z przychodniów był gośćiem u ćiebie / y chćiáłby obchodzić święto przejśćiá Pánu / pierwej obrzezány będzie káżdy mężcżyzná jego / á zátem przyſtąpi obchodzić je / y będzie jáko urodzony w ziemi. A ktobykolwiek nie był obrzezány / nie będzie jádł z niego.
Práwo jedno będzie w ziemi urodzonemu y przychodniowi / który jeſt gośćiem między wámi.
Ucżynili tedy wƺyſcy ſynowie Izráelſcy / jáko rozkázáł Pán Mojżeƺowi y Aáronowi / ták ucżynili.
Y ſtáło śię onegoż dniá / wywiódł Pán ſyny Izráelſkie z ziemi Egipſkiej z wojſki ich.
