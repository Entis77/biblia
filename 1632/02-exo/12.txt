Rzekł jeƺcże Pán do Mojżeƺá y do Aároná w źiemi Egipſkiej / mówiąc :
Mieśiąc ten będźie wám pocżątkiem mieśiąców : pierwƺy wám będźie między mieśiącámi w roku.
Rzecżćie do wƺyſtkiego zgromádzeniá Izráelſkiego / mówiąc : Dźieśiątego dniá mieśiącá tego weźmie ſobie káżdy báránká według fámilii / báránká według domu.
A jeźliby mniejƺy był dom niżeliby zjeść mogli báránká / tedy przybierze y ſąśiádá ſwego / który jeſt nájbliżƺy domu jego / według licżby duƺ / nálicżywƺy tyle oſób / ileby ich zjeść mogło báránká.
Báránká zupełnego / ſámcá rocżnego / mieć będźiećie ; z owiec álbo z kóz weźmiećie go.
Y będźiećie go chowáli áż do cżternáſtego dniá mieśiącá tego ; á zábije go wƺyſtko zebránie zgromádzeniá Izráelſkiego między dwomá wiecżorámi.
Y wezmą ze krwi jego / y pokropią obádwá podwoje y nádprożnik u domu ; w którym go będą ſpożywáć.
Y będą jeść mięſo onej nocy piecżone przy ogniu / y przáśniki / z źioły gorzkiemi będą go jeść.
Nie jedzćie z niego nic ſurowego / áni wárzonego w wodźie / ále upiecżone przy ogniu. Głowę jego z nogámi jego / z wnętrznośćiámi jego.
A nie zoſtánie z niego nic do jutrá ; á jeźliby co z niego do jutrá zoſtáło / ogniem ſpálićie.
Ták go tedy pożywáć będźiećie : Biodrá ſwe przepáƺećie / obuwie wáƺe będźie ná nogách wáƺych / á láſká wáƺá w ręce wáƺej / á jeść go będźiećie ſpieƺno / ábowiem przejśćie jeſt Páńſkie.
Gdyż przejdę przez źiemię Egipſką tej nocy / y zábiję wƺelkie pierworodne w źiemi Egipſkiej / od cżłowieká áż do bydlęćiá / y nád wƺyſtkimi bogi Egipſkimi wykonám ſądy / Já Pán.
A będźie wám oná krew ná znák ná domách / w których będźiećie ; bo ujrzáwƺy krew / minę wáſ / że nie będźie u wáſ plágá ku zátráceniu / gdy będę zábijáł w źiemi Egipſkiej.
A będźie wám ten dźień ná pámiątkę ; y będźiećie go obchodźić zá święto Pánu w národźiech wáƺych ; uſtáwą wiecżną obchodźić go będźiećie.
Przez śiedm dni przáśniki jeść będźiećie / á pierwƺego dniá záráz wypróżnićie kwáſ z domów wáƺych ; bo ktobykolwiek jádł co kwáƺonego od pierwƺego dniá áż do dniá śiódmego / wytráconá będźie duƺá oná z Izráelá.
W tenże dźień pierwƺy będźie zebránie święte / tákże dniá śiódmego zgromádzenie święte mieć będźiećie ; żádnej roboty nie będźiećie w nich cżynić / oprócż tego / cżego káżdy do jedzeniá używá / to ſámo gotowáć będźiećie.
Y będźiećie przeſtrzegáć przáśników ; ábowiem w ten dźień wywiodłem wojſká wáƺe z źiemi Egipſkiej ; przetoż przeſtrzegáć będźiećie dniá tego w národźiech wáƺych uſtáwą wiecżną.
Pierwƺego mieśiącá / cżternáſtego dniá tegoż mieśiącá / ná wiecżór jeść będźiećie przáśniki áż do dniá dwudźieſtego pierwƺego tegoż mieśiącá ná wiecżór.
Przez śiedm dni kwáſ niech śię nie znájduje w domách wáƺych ; bo ktobykolwiek jádł co kwáƺonego / wytráconá będźie duƺá jego z zgromádzeniá Izráelſkiego / ták przychodźień / jáko y zrodzony w źiemi.
Nic kwáƺonego jeść nie będźiećie ; we wƺyſtkich mieƺkániách wáƺych jeść będźiećie przáśniki.
Wezwáł tedy Mojżeƺ wƺyſtkich ſtárƺych Izráelſkich / y rzekł do nich : Odłącżćie / á weźmijćie ſobie báránká według fámilii ſwych / á zárznijćie ná święto przejśćiá.
Weźmiećie też ſnopek hyſopu / y omocżyćie we krwi / którá będźie w miednicy / á pokropićie odrzwi / y obá podwoje oną krwią / którá będźie w miednicy ; á z wáſ nie wynijdźie żáden ze drzwi domu ſwego áż do poránku.
Bo przejdźie Pán zábijájąc Egipt ; á ujrzáwƺy krew ná odrzwiách y ná obu podwojách / przeſtąpi Pán drzwi / y niedopuśći morderzowi / wchodźić do domów wáƺych zábijáć wáſ.
Y przeſtrzegáć będźiećie tego zá uſtáwę / tobie y ſynom twoim áż ná wieki.
A gdy wnijdźiećie do źiemi / którą wám dá Pán / jáko obiecáł / tych obrzędów przeſtrzegáć będźiećie.
A gdy wám rzeką ſynowie wáśi : Co to zá obrzędy wáƺe?
Tedy rzecżećie : Ofiárá to przejśćiá Páńſkiego / który przeſtępowáł domy ſynów Izráelſkich w Egipćie / gdy zábijáł Egipt / á domy náƺe wyzwáláł. Zátem ſchylił śię lud / y pokłonił śię.
Y poƺedłƺy ucżynili ſynowie Izráelſcy / jáko rozkázáł Pán Mojżeƺowi y Aáronowi / ták ucżynili.
Y ſtáło śię o północy / że Pán zábijáł wƺyſtkie pierworodztwá w źiemi Egipſkiej / od pierworodnego Fáráonowego / śiedzącego ná ſtolicy jego / áż do pierworodnego więźniá / który był w więźieniu / y wƺelkie pierworodne z bydląt.
Zátem wſtáł Fáráo onej nocy / y wƺyſcy ſłudzy jego / y wƺyſtek Egipt / y wƺcżął śię wielki krzyk w Egipćie ; bo nie było domu / w którymby nie był umárły.
A wezwáwƺy Fáráo Mojżeƺá y Aároná w nocy / rzekł : Wſtáńćie / wynijdźćie z pośrodku ludu mego / y wy y ſynowie Izráelſcy / á poƺedłƺy ſłużćie Pánu / jákośćie mówili.
Nádto trzody wáƺe / y bydłá wáƺe zábierzćie / jákośćie żądáli / á odchodząc / mnie też błogoſłáwćie.
Y przynágláli Egipcżánie ludowi / áby ich co rychlej wypráwili z źiemi ; bo mówili : Pomrzemy wƺyſcy.
Wźiął tedy lud ćiáſtá ſwe / pierwej niż zákiśiáły ; á one ćiáſtá ſwe uwinąwƺy w ƺáty ſwe / kłádli ná rámioná ſwoje.
Tedy ſynowie Izráelſcy ucżynili według rozkázániá Mojżeƺowego / y wypożycżáli u Egipcżán nácżyniá ſrebrnego / y nácżyniá złotego / y ƺát.
A Pán dáł łáſkę ludowi w ocżách Egipcżánów / że im pożycżáli ; y złupili Egipt.
Ciągnęli tedy ſynowie Izráelſcy z Rámeſeſ do Suchotu / około ƺeść kroć ſto tyśięcy pieƺych mężów tylko / okrom dźieći.
Ale y ludu poſpolitego wiele ƺło z nimi / y owiec / y bydłá / dobytek bárdzo wielki.
Y popiekli z ćiáſtá zádźiáłánego / które wynieśli z Egiptu / plácki przáśne ; bo nie było zákwáƺone / przeto że wygnáni byli z Egiptu / á nie mogli zmieƺkáć ; żywnośći też ſobie byli nie przygotowáli.
A cżáſu mieƺkániá ſynów Izráelſkich / którego mieƺkáli w Egipćie / było cżtery ſtá lát / y trzydźieśći lát.
Y ſtáło śię po cżterech ſet lát y trzydźieſtu lát / ſtáło śię onegoż dniá / wyƺły wƺyſtkie wojſká Páńſkie z źiemi Egipſkiej.
Noc tá obchodzoná má być Pánu / że je wywiódł ze źiemi Egipſkiej. Tá tedy noc Pánu obchodzoná má być od wƺyſtkich ſynów Izráelſkich w národźiech ich.
Y rzekł Pán do Mojżeƺá y do Aároná : Tá jeſt uſtáwá świętá przejśćiá : Żáden obcy nie będźie jádł z niego.
Przetoż káżdego ſługę wáƺego / á zá pieniądze kupionego / obrzeżećie go / tedy będźie jádł z niego.
Przychodźień y nájemnik nie będźie jádł z niego.
W domu jednym będźie jedzony ; nie wynieśieƺ nic z domu z mięſá jego / á kośći nie złámiećie w nim.
Wƺyſtko zgromádzenie Izráelſkie ták ucżyni z nim.
A jeźliby kto z przychodniów był gośćiem u ćiebie / y chćiáłby obchodźić święto przejśćiá Pánu / pierwej obrzezány będźie káżdy mężcżyzná jego / á zátem przyſtąpi obchodźić je / y będźie jáko urodzony w źiemi. A ktobykolwiek nie był obrzezány / nie będźie jádł z niego.
Práwo jedno będźie w źiemi urodzonemu y przychodniowi / który jeſt gośćiem między wámi.
Ucżynili tedy wƺyſcy ſynowie Izráelſcy / jáko rozkázáł Pán Mojżeƺowi y Aáronowi / ták ucżynili.
Y ſtáło śię onegoż dniá / wywiódł Pán ſyny Izráelſkie z źiemi Egipſkiej z wojſki ich.
