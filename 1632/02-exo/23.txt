Nie przyjmuj powieśći kłámliwej ; nie miej ſpółki z niepobożnym / ábyś miáł być świádkiem fáłƺywym.
Nie udáwáj śię zá wielkośćią do złego / y nie mów ták zá ſpráwą / cobyś śię nákłonił zá wielą ich ku podwróceniu ſądu.
Y nie ƺánuj ubogiego przy ſpráwie jego.
Gdybyś nátráfił wołu nieprzyjáćielá twego / álbo oſłá jego błądzącego / zwróćiƺ á dowiedzieƺ go do niego.
Jeźlibyś ujrzáł / że ośieł tego / który ćię má w nienáwiśći / leży pod brzemieniem ſwojim / ázáli zániecháƺ / ábyś mu pomóc nie miáł? owƺem porátujeƺ go poſpołu z nim.
Nie będzieƺ podwrácáł ſądu ubogiemu twemu w ſpráwie jego.
Od rzecży kłámliwej oddáliƺ śię / á niewinnego y ſpráwiedliwego nie zábijeƺ ; bo nieuſpráwiedliwię niezbożnego.
Dárów też bráć nie będzieƺ / poniewáż dár záślepiá mądre / y wywrácá ſłowá ſpráwiedliwych.
Przychodniá też nie ućiſkáj : bo ſámi wiećie / jáki jeſt żywot przychodniá / bośćie byli przychodniámi w ziemi Egipſkiej.
Przez ƺeść lát ośiewáć będzieƺ ziemię twoję / á będzieƺ zgromádzáł urodzáj jej ;
Ale śiódmego roku zániecháƺ jej ; że odpocżnie / áby jedli ubodzy ludu twego / á co zoſtánie po nich / poje zwierz polny. Tákże ucżyniƺ winnicy twojej / y oliwnicy twojej.
Przez ƺeść dni będzieƺ odpráwowáł roboty twoje ; ále dniá śiódmego odpocżnieƺ / áby ſobie wytchnął wół twój / ośieł twój / y żeby wytchnął ſyn niewolnicy twojej / y przychodzień.
A we wƺyſtkiem / com wám powiedziáł / oſtrożnymi bądźćie. Imieniá też cudzych bogów nie wſpominájćie / nie będzie ſłyƺáne z uſt twoich.
Trzy kroć święto obchodzić mi będziećie ná káżdy rok.
Świętá przáśników przeſtrzegáć będzieƺ ; śiedm dni jeść będzieƺ przáśniki / jákom ći rozkázáł / cżáſu mieśiącá Abibá ; boś weń wyƺedł z Egiptu / á nie ukáżećie śię przed twárz moję próżnymi.
Tákże święto żniwá pierwiáſtek prácy twojej / cośkolwiek śiáł ná polu ; święto też zbierániá ná ſchodzie roku / gdy zbierzeƺ práce twoje z polá.
Trzykroć do roku ukáże śię káżdy mężcżyzná twój przed oblicżem Pánującego Páná.
Nie będzieƺ ofiárowáł przy kwáśie krwi ofiáry mojej / áni zoſtánie przez noc tłuſtość ofiáry mojej áż do poránku.
Pierwiáſtki pierwƺych urodzájów ziemi twej przynieśieƺ w dom Páná Bogá twego ; nie będzieƺ wárzył koźlęćiá w mleku mátki jego.
Oto já poſyłám Aniołá przed tobą / áby ćię ſtrzegł w drodze / y wprowádził ćię ná miejſce / którem ći zgotowáł.
Oſtrożnym bądź przed oblicżnośćią jego ; á ſłucháj głoſu jego ; nie dráźnij go / boć nie przepuśći przeſtępſtwu wáƺemu / gdyż imię moje w nim jeſt.
Bo jeźli pilnie ſłucháć będzieƺ głoſu jego / y ucżyniƺ / cokolwiek rzekę / nieprzyjáćielem będę nieprzyjáćiół twych / y trápić będę tych / którzy ćię trápili.
Pójdzie bowiem Anioł mój przed tobą / y wprowádzi ćię do Amorejcżyká / y Hetejcżyká / y Ferezejcżyká / y Chánánejcżyká / Hewejcżyká / y Jebuzejcżyká / y wytrácę je.
Nie kłániájże śię bogom ich / áni im ſłuż / áni cżyń według ſpráw ich ; ále do gruntu popſujeƺ je / y wƺcżąt pokruƺyƺ obrázy ich.
Lecż ſłużyć będziećie Pánu Bogu wáƺemu / á on błogoſłáwić będzie chlebowi twemu / y wodom twoim ; y odejmę niemoc z pośrodku ćiebie.
Nie będzie poroniájącá áni niepłodná w ziemi twojej ; licżbę dni twoich dopełnię.
Strách mój puƺcżę przed tobą / y ſtrwożę wƺelki lud / przećiw któremu pójdzieƺ / y ucżynię / że wƺyſcy nieprzyjáćiele twoi podádzą tył przed tobą.
Poślę też ƺerƺenie przed tobą / które wypędzą Hewejcżyká / Chánánejcżyká / y Hetejcżyká przed oblicżnośćią twoją.
Nie wyrzucę go przed oblicżem twojim zá jeden rok / by śię ſnáć ziemiá w puſtynią nie obróćiłá / á nie námnożyło śię przećiwko tobie zwierzá dzikiego.
Po málucżku będę je wyrzucáł od oblicżá twego / áż śię rozmnożyƺ y ośiędzieƺ ziemię.
A położę gránice twoje od morzá cżerwonego áż do morzá Filiſtyńſkiego / á od puƺcży áż do rzeki ; bo podám w ręce wáƺe obywátele ziemi / y wypędziƺ je od oblicżá twego.
Nie poſtánowiƺ z nimi / áni z bogámi ich przymierzá.
Niech nie mieƺkáją w ziemi twej / by ćię ſnáć nie przywiedli do grzechu przećiwko mnie / gdybyś ſłużył bogom ich / coby tobie było śidłem.
