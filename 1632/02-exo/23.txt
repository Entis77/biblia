Nie przyjmuj powieśći kłamliwey : nie miej ſpółku z niepobożnym / ábyś miał być świádkiem fałƺywym.
Nie udawaj śię zá wielkośćią do złego / y nie mów ták zá ſpráwą / cobyś śię nákłonił zá wielą ich ku podwróceniu <i>ſądu</i>.
Y nieƺánuj ubogiego przy ſpráwie jego.
Gdybyś nátráfił wołu nieprzyjaćielá twego / álbo oſłá jego błądzącego / záwróćiƺ / á dowiedźieƺ go do niego.
Jeslibyś ujrzał / że ośieł tego / który ćię ma w nienawiśći / leży pod brzemieniē ſwojim / ázali zániechaƺ / ábyś mu pomóc nie miał? owƺem porátujeƺ go poſpołu z nim.
Nie będźieƺ podwracał ſądu ubogiemu twemu w ſpráwie jego.
Od rzeczy kłamliwey oddaliƺ śię / á niewinnego y ſpráwiedliwego nie zábijeƺ : bo nieuſpráwiedliwię niezbożnego.
Dárów też bráć nie będźieƺ / ponieważ dar záślepia mądre / y wywraca ſłowá ſpráwiedliwych.
Przychodniá też nie ućiſkaj : bo ſámi wiećie jáki jeſt żywot przychodniá / bośćie byli przychodniámi w źiemi Egipſkiey.
Przez ƺeść lat ośiewáć będźieƺ źiemię twoję / y będźieƺ zgromadzał urodzaj jey.
Ale śiódmego <i>roku</i> zániechaƺ jey że odpoczynie / áby jedli ubodzy ludu twego : á co zoſtánie po nich / poje zwierz polny. Tákże uczyniƺ winnicy twojey / y oliwnicy twojey.
Przez ƺeść dni będźieƺ odpráwował roboty twoje / ále dniá śiódmego odpoczynieƺ : áby ſobie wytchnął wół twój / ośieł twój / y <i>żeby</i> wytchnął Syn niewolnice twojey / y przychodźień.
A we wƺyſtkim com wam powiedźiał / oſtrożnemi bądźćie. Imienia też cudzych Bogów nie wſpominajćie / nie będźie ſłyƺáne z uſt twojich.
Trzy kroć święto obchodźić mi będźiećie ná káżdy rok.
Świętá Przáśników przeſtrzegáć będźieƺ : śiedm dni jeść będźieƺ przáśniki / jákom ći rozkazał / cżáſu mieśiącá Abibá : boś weń wyƺedł z Egiptu / á nie ukażećie śię przed twarz moję próżnymi.
Tákże święto żniwá pierwiaſtek prace twojey / cośkolwiek śiał ná polu : święto też zbieránia ná zchodźie roku / gdy zbierzeƺ prace twoje z polá.
Trzykroć do roku ukaże śię káżdy męƺczyzná twój przed obliczem PANUJĄCEGO PANA.
Nie będźieƺ ofiárował przy kwáśie krwie ofiáry mojey / áni zoſtánie przez noc tłuſtość ofiáry mojey / áż do poránku.
Pierwiaſtki pierwƺych urodzájów źiemie twey / przynieśieƺ w dom PANA Bogá twego : niebędźieƺ wárzył koźlęćiá w mleku mátki jego.
Oto ja poſyłam Anjołá przed tobą / áby ćię ſtrzegł w drodze / y wprowádźił ćię ná miejſce / którem <i>ći</i> zgotował.
Oſtrożnym bądź przed / oblicżnośćią jego / á ſłuchaj głoſu jego / nie draźni go / boć nie przepuśći przeſtępſtwu wáƺemu : gdyż Imię moje w nim jeſt.
Bo jeſli pilnie ſłucháć będźieƺ głoſu jego / y uczyniƺ cokolwiek rzekę / nieprzyjáćielem będę nieprzyjáćiół twych / y trapić będę tych / którzyby ćię trapili.
Pójdźie bowiem Anjoł mój przed tobą / y wprowádźi ćię do Amorejczyká / y Hettejczyká / y Ferezeyczyká / y Chánánejczyká / Hewejczyká / y Jebuzejczyká / y wytrácę je.
Nie kłániajże śię Bogom ich / áni im ſłuż / áni czyń według ſpraw ich : ále do gruntu popſujeƺ je / y wƺcząt pokruƺyƺ obrázy ich.
Lecz ſłużyć będźiećie PANU Bogu wáƺemu / á on błogoſłáwić będźie chlebowi twemu / y wodam twojim / y odejmę niemoc z pośrzodku ćiebie.
Nie będźie poroniájącá áni niepłodna w źiemi twojey : liczbę dni twojich dopełnię.
Strách mój puƺczę przed tobą / y ſtrwożę wƺelki lud / przećiw któremu pójdźieƺ / y uczynię że wƺyſcy nieprzyjaćiele twoji podádzą tył przed tobą.
Poślę też śierƺenie przed tobą które wypędzą Hewejczyká / Chánánejczyká / y Hettejczyká przed oblicznośćią twoją.
Nie wyrzucę go przed obliczem twojim zá jeden rok / by śię ſnadź źiemiá w puſtynią nie obróćiłá / á nie námnożyło śię przećiwko tobie zwierzá dźikiego.
Pomáluczku będę je wyrzucał / od oblicza twego / áż śię rozmnożyƺ / y ośiędźieƺ źiemię.
A położę gránice twoje od morzá cżerwonego / áż do morzá Filiſtyńſkiego / á od puƺczy / áż do rzeki : bo podam w ręce wáƺe obywátele źiemie / y wypędźiƺ je od oblicza twego.
Nie poſtánowiƺ z nimi / áni z Bogámi ich Przymierza.
Niech nie mieƺkáją w źiemi twey / by ćię ſnadź nie przywiedli do grzechu przećiwko mnie / gdybyś ſłużył Bogom ich / coby tobie było śidłem.
