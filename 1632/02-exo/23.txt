Nie przyjmuj powieśći kłámliwej ; nie miej ſpółki z niepobożnym / ábyś miał być świádkiem fáłƺywym.
Nie udáwáj śię zá wielkośćią do złego / y nie mów ták zá ſpráwą / cobyś śię nákłonił zá wielą ich ku podwróceniu ſądu.
Y nie ƺánuj ubogiego przy ſpráwie jego.
Gdybyś nátráfił wołu nieprzyjáćielá twego / álbo oſłá jego błądzącego / zwróćiƺ á dowiedźieƺ go do niego.
Jeſlibyś ujrzał / że ośieł tego / który ćię ma w nienáwiśći / leży pod brzemieniem ſwojim / ázáli zániecháƺ / ábyś mu pomóc nie miał? owƺem porátujeƺ go poſpołu z nim.
Nie będźieƺ podwrácał ſądu ubogiemu twemu w ſpráwie jego.
Od rzecży kłámliwej oddáliƺ śię / á niewinnego y ſpráwiedliwego nie zábijeƺ ; bo nieuſpráwiedliwię niezbożnego.
Dárów też bráć nie będźieƺ / poniewáż dár záślepiá mądre / y wywrácá ſłowá ſpráwiedliwych.
Przychodniá też nie ućiſkáj : bo ſámi wiećie / jáki jeſt żywot przychodniá / bośćie byli przychodniámi w źiemi Egipſkiej.
Przez ƺeść lat ośiewáć będźieƺ źiemię twoję / á będźieƺ zgromádzał urodzáj jey ;
Ale śiódmego roku zániecháƺ jey ; że odpocżnie / áby jedli ubodzy ludu twego / á co zoſtánie po nich / poje zwierz polny. Tákże ucżyniƺ winnicy twojey / y oliwnicy twojey.
Przez ƺeść dni będźieƺ odpráwował roboty twoje ; ále dniá śiódmego odpocżnieƺ / áby ſobie wytchnął wół twój / ośieł twój / y żeby wytchnął ſyn niewolnicy twojey / y przychodźień.
A we wƺyſtkiem / com wam powiedźiał / oſtrożnymi bądźćie. Imieniá też cudzych bogów nie wſpominájćie / nie będźie ſłyƺáne z uſt twojich.
Trzy kroć święto obchodźić mi będźiećie ná káżdy rok.
Świętá przáśników przeſtrzegáć będźieƺ ; śiedm dni jeść będźieƺ przáśniki / jákom ći rozkázał / cżáſu mieśiącá Abibá ; boś weń wyƺedł z Egiptu / á nie ukáżećie śię przed twárz moję próżnymi.
Tákże święto żniwá pierwiáſtek prácy twojey / cośkolwiek śiał ná polu ; święto też zbieránia ná ſchodźie roku / gdy zbierzeƺ práce twoje z polá.
Trzykroć do roku ukáże śię káżdy mężcżyzná twój przed oblicżem Pánującego Páná.
Nie będźieƺ ofiárował przy kwáśie krwi ofiáry mojey / áni zoſtánie przez noc tłuſtość ofiáry mojey áż do poránku.
Pierwiáſtki pierwƺych urodzájów źiemi twej przynieśieƺ w dom Páná Bogá twego ; nie będźieƺ wárzył koźlęćiá w mleku mátki jego.
Oto ja poſyłám Aniołá przed tobą / áby ćię ſtrzegł w drodze / y wprowádźił ćię ná miejſce / którem ći zgotował.
Oſtrożnym bądź przed oblicżnośćią jego ; á ſłucháj głoſu jego ; nie dráźnij go / boć nie przepuśći przeſtępſtwu wáƺemu / gdyż imię moje w nim jeſt.
Bo jeſli pilnie ſłucháć będźieƺ głoſu jego / y ucżyniƺ / cokolwiek rzekę / nieprzyjáćielem będę nieprzyjáćiół twych / y trápić będę tych / którzy ćię trápili.
Pójdźie bowiem Anioł mój przed tobą / y wprowádźi ćię do Amorejcżyká / y Hetejcżyká / y Ferezejcżyká / y Chánánejcżyká / Hewejcżyká / y Jebuzejcżyká / y wytrácę je.
Nie kłániájże śię bogom ich / áni im ſłuż / áni cżyń według ſpráw ich ; ále do gruntu popſujeƺ je / y wƺcżąt pokruƺyƺ obrázy ich.
Lecż ſłużyć będźiećie Pánu Bogu wáƺemu / á on błogoſłáwić będźie chlebowi twemu / y wodom twoim ; y odejmę niemoc z pośrzodku ćiebie.
Nie będźie poroniájącá áni niepłodná w źiemi twojey ; licżbę dni twojich dopełnię.
Strách mój puƺcżę przed tobą / y ſtrwożę wƺelki lud / przećiw któremu pójdźieƺ / y ucżynię / że wƺyſcy nieprzyjáćiele twoi podádzą tył przed tobą.
Poślę też ƺerƺenie przed tobą / które wypędzą Hewejcżyká / Chánánejcżyká / y Hetejcżyká przed oblicżnośćią twoją.
Nie wyrzucę go przed oblicżem twojim zá jeden rok / by śię ſnadź źiemiá w puſtynią nie obróćiłá / á nie námnożyło śię przećiwko tobie zwierzá dźikiego.
Po málucżku będę je wyrzucał od oblicżá twego / áż śię rozmnożyƺ y ośiędźieƺ źiemię.
A położę gránice twoje od morzá cżerwonego áż do morzá Filiſtyńſkiego / á od puƺcży áż do rzeki ; bo podám w ręce wáƺe obywátele źiemi / y wypędźiƺ je od oblicżá twego.
Nie poſtánowiƺ z nimi / áni z bogámi ich przymierza.
Niech nie mieƺkáją w źiemi twej / by ćię ſnadź nie przywiedli do grzechu przećiwko mnie / gdybyś ſłużył bogom ich / coby tobie było śidłem.
