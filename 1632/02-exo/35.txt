Potem zebrał Mojzeƺ wƺyſtko zgromádzenie ſynów Izráelſkich / y mówił do nich : Te ſą rzecży / które rozkázał Pán / ábyśćie je cżynili.
Przez ƺeść dni odpráwowáná będźie robotá ; ále dźień śiódmy będźie wam święty / ſábbát odpocżynieniá Páńſkiego ; ktoby weń robił robotę / umrze.
Nie rozniećićie ogniá we wƺyſtkich mieƺkániách wáƺych w dźień ſábbátu.
Rzekł też Mojzeƺ do wƺyſtkiego zgromádzenia ſynów Izráelſkich / mówiąc : Táć jeſt rzecż / którą przykázał Pán / mówiąc :
Złóżćie od śiebie podárek Pánu ; káżdy / kto jeſt ochotnego ſercá / przynieśie ten podárek Pánu / złoto / y ſrebro / y miedź.
Y hijácynt / y ƺárłát / y kármázyn dwá kroć fárbowány / y biáły jedwáb / y śierść koźią ;
Skóry też báránie cżerwono fárbowáne y ſkóry borſukowe / y drzewo ſytym ;
Y oliwę do świeceniá / y rzecży wonne ná olejek pomázowánia / y dla kádzeniá wonnego ;
Kámienie też onychiny / y kámienie do oſádzánia nárámienniká y nápierśniká.
A wƺyſcy dowćipnego ſercá miedzy wámi przyjdą / y robić będą / cokolwiek rozkázał Pán :
Przybytek / námiot jego / y przykryćie jego / hácżyki jego / y deſki jego / drągi jego / ſłupy jego / y podſtáwki jego ;
Skrzynie y drążki jey / ubłágálnią / y oponę do záſłony /
Stół y drążki jego / ze wƺyſtkiem nácżyniem jego / y chleby pokłádne ;
Y świecżnik do świeceniá z nácżyniem jego / y lámpy jego / y oliwę do świeceniá.
Ołtárz tákże do kádzeniá z drążkámi jego / y olejek pomázowánia / y kádzeniá wonne / y záſłonę do drzwi przybytku ;
Ołtárz do cáłopalenia / y krátę jego miedźiáną / drążki jego / y wƺyſtkie nácżyniá jego / wánnę z ſtolcem jey.
Opony do śieni / ſłupy jey / y podſtáwki jey / y záſłonę do drzwi u śieni.
Kołki do przybytku / y kołki do śieni z ƺnurámi jey.
Száty ſłużebne do uſługiwánia w świątnicy / ƺáty święte Aáronowi kápłánowi / y ƺáty ſynom jego / dla ſpráwowánia urzędu kápłáńſkiego.
Wyƺło tedy wƺyſtko zgromádzenie ſynów Izráelſkich od oblicżnośći Mojżeƺowej.
Y przyƺedł káżdy mąż / którego pobudźiło ſerce jego / y káżdy / w którym dobrowolny był duch jego / przynieśli podárek Pánu do robieniá namiotu zgromádzenia / y do wƺelkiej potrzeby jego / y ná ƺáty święte.
Przychodźili tedy mężowie z niewiáſtámi / káżdy / kto był ochotnego ſercá / przynośili zápony / y náuƺnice / y pierśćienie / y mánele / y wƺelákie nácżyniá złote / y ktokolwiek przynośił ofiárę złotą Pánu.
Káżdy też co miał hijácynt / y ƺárłát / y kármázyn dwá kroć fárbowány / y biáły jedwáb / y śierść koźią / y ſkóry báránie cżerwono fárbowáne / y ſkóry borſukowe / przynośili.
Ktokolwiek ofiárował podárek ſrebrá y miedźi / przynośili ná ofiárę Pánu / káżdy też / co miał drzewo ſytym / ná wƺeláką potrzebę ku uſłudze przynośili.
Y wƺyſtkie niewiáſty dowćipnego ſercá rękámi ſwemi przędły / á przynośiły co náprzędły / hijácynt / y ƺárłát / kármázyn dwá kroć fárbowány / y biáły jedwáb.
A wƺyſtkie niewiáſty / których pobudźiło ſerce ich umiejętne / przędły śierść koźią.
Przełożeni záśię przynośili kámienie onychiny / y kámienie do oſádzánia nárámienniká y nápierśniká.
Tákże rzecży wonne y oliwę do świeceniá / y ná olejek pomázowánia y ná wonne kádzeniá.
Káżdy mąż y niewiáſtá / w których ochotne ſerce było do ofiárowánia / ná káżdą robotę / którą rozkázał Pán cżynić przez Mojżeƺá / przynośili ſynowie Izráelſcy ofiárę dobrowolną Pánu.
Zátem rzekł Mojzeƺ do ſynów Izráelſkich : Oto / wezwał Pán z imieniá Beſáleelá / ſyná Urowego / ſyná Churowego / z pokolenia Judy ;
Y nápełnił go Duchem Bożym / mądrośćią / rozumem / y umiejętnośćią wƺelákiego rzemioſłá ;
Y ku dowćipnemu wymyślániu / cokolwiek może być urobione ze złotá / y z ſrebrá / y z miedźi ;
Do rzezánia kámieniá ku oſádzeniu / y ná wyrobienie drzewá / do cżynieniá wƺelákiej roboty zmyślney.
Dał nádto do ſercá jego / áby ucżyć mógł inƺych / on / y Acholijáb / ſyn Achyſámechów z pokolenia Dán.
Nápełnił je mądrośćią ſercá / áby robili wƺelákie rzemioſło ćieśielſkie / y háftárſkie / y tkáckie z hijácyntu / y z ƺárłátu / z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu tkácką robotą / áby robili káżdą robotę dowćipnie wymyślájąc.
