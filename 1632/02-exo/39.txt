Tákże z hijácyntu y z ƺárłátu / z kármázynu dwá kroć fárbowánego pocżynili ƺáty do uſługi / ku uſługowániu w świątnicy. Urobili też ƺáty święte Aáronowi / jáko był rozkázáł Pán Mojżeƺowi.
Y ucżynił nárámiennik ze złotá / z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu kręconego.
Náklepáli też bláƺek złotych / y náſtrzygli z nich nići do przetykániá hijácyntu / y do przetykániá ƺárłátu / y do przetykániá kármázynu dwá kroć fárbowánego / y do przetykániá biáłego jedwábiu / robotą háftárſką.
Nárámienniki przytem porobili ták / áby śię jeden z drugim ſpoić mógł ; ná dwu krájách ich ſpájáły śię.
Páſ też nárámienniká / który był ná nim / z tegoż był / y tąż robotą / ze złotá / z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu kręconego / jáko był Pán rozkázáł Mojżeƺowi.
Do tego wygotowáli kámienie onychiny / opráwione złotem oſádzeniem / rzezáne / jáko ryte bywáją piecżęći / z imiony ſynów Izráelſkich.
Y wpráwił je ná wierzchne kráje nárámienniká / áby były kámieńmi ná pámiątkę ſynom Izráelſkim / jáko był rozkázáł Pán Mojżeƺowi.
Ucżynił też nápierśnik robotą háftárſką według roboty nárámienniká / ze złotá / z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu kręconego.
Cżworográniáſty był ; dwoiſty ucżynili nápierśnik / ná piędźi długość jego / y ná piędźi ƺerokość jego / dwoiſty był.
Y náſádźili weń cżtery rzędy kámieniá tym porządkiem : ſárdyjuƺ / topázyjuƺ y ƺmárágd w rzędźie pierwƺym.
A w drugim rzędźie : kárbunkuł / ſáfir y jáſpiſ.
A w trzećim rzędźie : linkuryjuƺ / ácháteſ y ámetyſt.
A w cżwártym rzędźie : chryſolit / onychin / y berył / wƺyſtkie oſádzone w złoto w rzędźiech ſwych.
A tych kámieni z imionámi ſynów Izráelſkich dwánáśćie według imion ich było / ták / jáko rzezą piecżęći ; káżdy według imieniá ſwego podług dwunáſtu pokoleń.
Pocżynili też do nápierśniká łáńcuƺki jednoſtájne robotą plećioną ze złotá ƺcżerego.
Spráwili też dwá hácżyki złote / y dwá kolce złote / y przypráwili one dwá kolce do obu krájów nápierśniká.
A przewlekli one dwá łáńcuƺki złote przez obá kolce u krájów nápierśniká.
Drugie záś dwá kolce obu łáńcuƺków záwlekli do onych dwu hácżyków / y przypráwili do zwierzchnich krájów nárámienniká ná przodku.
Ucżynili tákże dwá kolce złote / które przypráwili do dwu końców nápierśniká ná kráju jego / który był po ſtronie nárámienniká ze ſpodku.
Ucżynili jeƺcże dwá kolce złote / które przypráwili ná dwu ſtronách nárámienniká ze ſpodku / ná przodku przećiwko ſpojeniu jego / które jeſt nád przepáſániem nárámienniká.
Y przywiązáli nápierśnik od kolców jego do kolców nárámienniká ƺnurem hijácyntowym / áby był nád przepáſániem nárámienniká / żeby nie odewſtáwáł nápierśnik od nárámienniká ; jáko był rozkázáł Pán Mojżeƺowi.
Urobił tákże płáƺcż pod nárámiennik robotą tkáną / wƺyſtek hijácyntowy ;
A rozpór płáƺcżá w pośród jego / jáko rozpór u páncerzá / y brámá około kráju jego / áby śię nie rozdźieráł.
Tákże u podołká płáƺcżá onego ucżynili jábłká gránátowe z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego / y z biáłego jedwábiu kręconego.
Pocżynili też dzwonki ze złotá ƺcżerego / y pozáwieƺáli one dzwonki między one jábłká gránátowe u podołká płáƺcżá w około / w pośród jábłek gránátowych ;
Dzwonek á jábłko gránátowe / y záś dzwonek y jábłko gránátowe / u podołká płáƺcżá w około ku poſługiwániu / jáko rozkázáł Pán Mojżeƺowi.
Porobili też ƺáty z biáłego jedwábiu robotą tkácką Aáronowi / y ſynom jego.
Cżápecżkę też z biáłego jedwábiu / y cżápki ozdobne z biáłego jedwábiu / y ubiory ćienkie z biáłego jedwábiu kręconego.
Páſ tákże z biáłego jedwábiu kręconego / y z hijácyntu / y z ƺárłátu / y z kármázynu dwá kroć fárbowánego robotą háftárſką / jáko Pán rozkázáł Mojżeƺowi.
Do tego ucżynili bláchę korony świętobliwośći ze złotá ƺcżerego / y wyryſowáli ná niej robotą rytą / jáko piecżęći rzezą : Świętość Pánu.
A przypráwili do niej ƺnur hijácyntowy / áby przywiązáná byłá do cżápki ná wierzchu / jáko był rozkázáł Pán Mojżeƺowi.
A ták ſkońcżyłá śię wƺyſtká robotá około przybytku y námiotu zgromádzeniá. Y ucżynili ſynowie Izráelſcy wƺyſtko / jáko był rozkázáł Pán Mojżeƺowi / ták ucżynili.
Y przynieśli ten przybytek do Mojżeƺá / námiot / y wƺyſtkie nácżyniá jego / háki jego / deſki jego / drągi jego / y ſłupy jego / y podſtáwki jego.
Przykryćie też z ſkór báránich cżerwono fárbowánych / y przykryćie z ſkór borſukowych / y oponę záſłony ;
Skrzynię świádectwá / y drążki jey / y ubłágálnią.
Stół / wƺyſtkie nácżyniá jego / y chleb pokłádny.
Świecżnik ochędożny / lámpy jego / lámpy ſporządzone / y wƺyſtkie nácżyniá jego / y oliwę ku świeceniu.
Ołtárz tákże złoty / y olejek pomázowániá / y kádźidło wonne / y záſłonę do drzwi námiotu ;
Ołtárz miedźiány / y krátę jego miedźiáną / drążki jego / y wƺyſtkie nácżyniá jego / wánnę y ſtolec jey.
Opony do śieni / y ſłupy ich z podſtáwkámi ich ; y záſłonę do brámy śienney / y ƺnury jey / y kołki jey / y wƺelákie nácżyniá ku ſłużbie przybytku y námiotu zgromádzeniá.
Száty ſłużebne do uſługowániá w świątnicy / ƺáty święte Aáronowi kápłánowi / y ƺáty ſynów jego do odpráwowániá urzędu kápłáńſkiego.
Według wƺyſtkiego / jáko był rozkázáł Pán Mojżeƺowi / ták ucżynili ſynowie Izráelſcy wƺyſtką tę robotę.
Y obejrzáł Mojżeƺ tę wƺyſtką robotę / á oto / ucżynili ją / jáko był rozkázáł Pán / ták ucżynili ; y błogoſłáwił im Mojżeƺ.
