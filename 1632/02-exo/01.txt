Teć ſą imioná Synów Izráelſkich którzy weƺli do Egiptu z Jákóbem : káżdy z domem ſwym weƺli.
Ruben / Symeon / Lewi / y Judás.
Iſáſchár / Zábulon / y Benjámin.
Dan / y Neftáli / Gád / y Aſſer.
A było wƺyſtkich duƺ które wyƺły z biódr Jákóbowych / śiedmdźieśiąt duƺ : á Józef był <i>przed tym</i> w Egipćie.
Y umárł Józef y wƺyſcy bráćia jego / y wƺyſtek on rodzaj.
A ſynowie Izráelſcy rozrodźili śię / y roſpłodźili śię / y rozmnożyli śię / y zmocnili śię bárzo wielce : á nápełnioná jeſt źiemiá nimi.
Miedzy tym powſtał Król nowy nád Egiptem / który nie znał Józefá.
Y rzekł do ludu ſwego : Oto lud ſynów Izráelſkich wielki / y możniejƺy nád nas.
Przetoż mądrze ſobie pocżnimy z nimi / by śię ſnadź nie rozmnożył : á jesliby przypádłá wojná / áby śię nie przyłączył y on do nieprzyjaćiół náƺych / y nie walczył przećiwko nam / y nie uƺedł z źiemie.
A ták uſtáwili nád nim poborce / áby go dręcżyli ćiężarámi ſwymi : y zbudował <i>lud Izráelſki</i> miáſtá ſkłádu Fáráonowi / Pitom / y Rámeſes.
Ale im więcey go trapili / tym więcey śię rozmnażał y tym więcey roſł : ták iż śćiśnieni byli dla ſynów Izráelſkich.
Y podbili Egipczánie Syny Izráelſkie w niewolą ćięƺką.
Y przykrzyli <i>im</i> żywot jich robotą ćięƺką / około gliny / y około cegieł / y około káżdey roboty ná polu / mimo wƺeláką robotę ſwą / do którey jich używáli bez litośći.
Y rozkazał Król Egipſki bábam Hebrejſkim / z których imię jedney Zeforá / á imię drugiey Fuá /
A rzekł : Gdy będźiećie bábić <i>niewiáſtam</i> Hebrejſkim / á ujrzyćie że rodzą / byłliby ſyn / zábijćież go : á jesli córká / niech żywa zoſtánie.
Ale bały śię one báby Bogá / y nie czyniły jáko im rozkazał Król Egipſki / ále żywo záchowywáły chłopiątká.
Záczym wezwawƺy Król Egipſki onych bab / mówił do nich : Czemuśćie to uczyniły / żeśćie żywo záchowáły chłopiątká?
Y odpowiedźiáły báby Fáráonowi : iż <i>nie ſą</i> jáko Niewiáſty Egipſkie / <i>niewiáſty</i> Hebrejſkie : bo ſą duże / pierwey niż przydźie do nich bábá / rodzą.
Y czynił dobrze Bóg onym bábam / y krźewił śię lud / y zmocnili śię bárzo.
Y ſtáło śię / przeto że śię bały one báby Bogá / pobudował im domy.
Tedy rozkazał Fáráo wƺyſtkiemu ludowi ſwemu / mówiąc : káżdego ſyná który śię urodźi / w rzekę go wrzuććie / á káżdą córkę żywo záchowajćie.
