Teć ſą imioná ſynów Izráelſkich / którzy weƺli do Egiptu z Jákóbem ; káżdy z domem ſwym weƺli :
Ruben / Symeon / Lewi / y Judáſ.
Iſáƺár / Zábulon / y Benjámin.
Dán / y Neftáli / Gád / y Aſer.
A było wƺyſtkich duƺ / które wyƺły z biódr Jákóbowych / śiedmdźieśiąt duƺ ; á Józef był przedtem w Egipćie.
Y umárł Józef / y wƺyſcy bráćia jego / y wƺyſtek on rodzáj.
A ſynowie Izráelſcy rozrodźili śię / y rozpłodźili śię / y rozmnożyli śię / y zmocnili śię bárdzo wielce / á nápełnioná jeſt źiemiá nimi.
Między tem powſtáł król nowy nád Egiptem / który nie znáł Józefá ;
Y rzekł do ludu ſwego : Oto lud ſynów Izráelſkich wielki / y możniejƺy nád nas.
Przetoż mądrze ſobie pocżnijmy z nimi / by śię ſnáć nie rozmnożył / á jeſliby przypádłá wojná / áby śię nie przyłącżył y on do nieprzyjáćiół náƺych / y nie wálcżył przećiwko nám / y nie uƺedł z źiemi.
A ták uſtáwili nád nim poborce / áby go dręcżyli ćiężárámi ſwemi ; y zbudowáł lud Izráelſki miáſtá ſkłádu Fáráonowi : Pytom y Rámeſeſ.
Ale im więcey go trápili / tem więcey śię rozmnáżáł / y tem więcey roſł / ták / iż śćiśnieni byli dla ſynów Izráelſkich.
Y podbili Egipcżánie ſyny Izráelſkie w niewolą ćiężką.
Y przykrzyli im żywot ich robotą ćiężką około gliny / y około cegieł / y około káżdej roboty ná polu / mimo wƺeláką robotę ſwą / do którey ich używáli bez litośći.
Y rozkázáł król Egipſki bábom Hebrejſkim / z których imię jedney Zeforá / á imię drugiej Fuá ;
A rzekł : Gdy będźiećie bábić niewiáſtom Hebrejſkim / á ujrzyćie że rodzą / byłliby ſyn / zábijćież go / á jeſli córká / niech żywá zoſtánie.
Ale báby one báły śię Bogá / y nie cżyniły / jáko im rozkázáł król Egipſki / ále żywo záchowywáły chłopiątká.
Zácżem wezwáwƺy król Egipſki onych báb / mówił do nich : Cżemuśćie to ucżyniły / żeśćie żywo záchowáły chłopiątká?
Y odpowiedźiáły báby Fáráonowi : Iż nie ſą jáko niewiáſty Egipſkie / niewiáſty Hebrejſkie ; bo ſą duże / pierwej niż przyjdźie do nich bábá / rodzą.
Y cżynił dobrze Bóg onym bábom ; y krzewił śię lud / y zmocnili śię bárdzo.
Y ſtáło śię / przeto że śię báły one báby Bogá / pobudowáł im domy.
Tedy rozkázáł Fáráo wƺyſtkiemu ludowi ſwemu / mówiąc : Káżdego ſyná / który śię urodźi / w rzekę go wrzuććie / á káżdą córkę żywo záchowájćie.
