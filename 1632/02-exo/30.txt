Uczyniƺ też Ołtarz dla kádzenia / z drzewá Syttym / uczyniƺ go.
Ná łokieć w dłuż / y ná łokieć w ƺerz / czworo grániſty będźie / á ná dwá łokćiá w zwyż : z niego <i>wychodźić będą</i> rogi jego.
A powleczeƺ go ƺczerym złotem / wierzch jego / y śćiány jego w około / y rogi jego : Uczyniƺ też Koronę złotą około niego.
Y dwie kolcá złote uczyniƺ też pod koroną we dwu kątách jego / po obu ſtronách jego / á przez nie przewleczeƺ drążki / áby noƺony był ná nich.
A uczyniƺ drążki one z drzewá Syttym / y powleczeƺ je złotem.
Y poſtáwiƺ go przed zaſłoną / zá którą jeſt Skrzyniá świádectwá / przed ubłagálnią / która jeſt nád świádectwem / gdźie śię z tobą zchodźić będę.
A będźie kádźił ná nim Aáron / kádzeniem z wonnych rzeczy ná káżdy poránek / przygotowawƺy lámpy / będźie kádźił.
Tákże gdy rozpali Aáron lámpy miedzy dwiemá wieczorámi / kádźić będźie kádzeniem uſtáwicznym przed PANEM / w narodźiech wáƺych.
Nie włożyćie nań kádźidłá obcego / áni Cáłopalenia / áni ofiáry ſuchey / áni ofiáry mokrey ofiárowáć będźiećie ná nim.
Tylko wykona oczyśćienie Aáron nád rogámi jego / raz w rok : przez krew ofiáry zá grzech / <i>w dźień</i> oczyśćiánia / raz w rok oczyśćienie odpráwi ná nim / w Narodźiech wáƺych : bo to rzecz Naświętƺa PANU.
Zátym rzekł PAN do Mojzeƺá / mówiąc :
Gdy zbierzeƺ główną ſummę Synów Izráelſkich / z tych / którzy máją iść w liczbę / da káżdy okup zá duƺę ſwą PANU / gdy je liczyć będźieƺ / áby nie przyƺłá ná nie plagá / gdy zliczeni będą.
To dawáć będą / Káżdy który idźie w liczbę / <i>da</i> puł ſyklá / według ſyklá świątnice / ( dwádźieśćia pieniędzy ſykiel <i>wáży,</i> ) puł ſyklá będźie podárek PANU.
Ktokolwiek idźie w liczbę ode dwudźieſtu lat y wyżey / odda podárek PANU.
Bogáty nie da więcey / á ubogi nie da mniey nád puł ſyklá / gdy będą dawáć ofiárę PAnu / dla oczyśćienia duƺ ſwojich :
A wybrawƺy pieniądze oczyśćienia od Synów Izráelſkich / daƺ je ná potrzeby Namiotu zgromádzenia : co będźie Synom Izráelſkim ná pámiątkę przed PANEM / ku oczyśćieniu duƺ wáƺych.
Potym rzekł PAN do Mojzeƺá mówiąc :
Uczyniƺ też Wánnę miedźiáną / y Stolec jey miedźiány / do umywánia : á poſtáwiƺ ją miedzy Namiotem zgromádzenia / y miedzy Ołtarzem / y nálejeƺ w nię wody.
Y umywáć będą Aáron y Synowie jego z niey ręce ſwoje / y nogi ſwoje.
Gdy wchodźić będą do Namiotu zgromádzenia / umywáć śię będą wodą / áby nie pomárli : tákże gdyby mieli przyſtępowáć do Ołtarzá / áby ſłużyli / y zápaláli Ofiárę ogniſtą PANU.
Y będą umywáli ręce ſwoje / y nogi ſwoje / áby nie pomárli : y będźie im to uſtáwą wieczną / jemu y naśieniu jego w rodzájách ich.
Rzekł jeƺcze PAN do Mojzeƺá / mówiąc :
Ty też weźmi ſobie wonnych rzeczy przednich : Mirry co naczyſtƺey / pięć ſet <i>łótów,</i> á Cinámonu wonnego / połowicę tego / <i>to jeſt</i> dwieśćie y pięćdźieśiąt <i>łótów</i> : y Tátárſkiego źiela dwieśćie y pięćdźieśiąt :
Kásſiey też pięć ſet <i>łótów</i> według Syklá Świątnice / y Oliwy z drzew Oliwnych / Hin :
Y uczyniƺ z tego Olejek pomázowánia ſwiętego / máść nawyborniejƺą / robotą Aptekárſką : Olejek <i>to</i> pomázowánia świętego będźie.
Y pomáżeƺ nim Namiot zgromádzenia / y Skrzynię świádectwá.
Tákże ſtół / y wƺyſtkie naczynia jego / y ſwiecznik / y naczynia jego / y Ołtarz ná którym kádzą :
Ołtarz też <i>do</i> Cáłopalenia ze wƺyſtkim naczyniem jego / y Wánnę z ſtolcem jey.
A poświęćiƺ je / áby naświętƺe były : cokolwiek śię ich dotknie / poświęcone będźie.
Aároná też y Syny jego pomáżeƺ / y poświęćiƺ je / áby mi ſpráwowáli urząd kápłáńſki.
A Synom Izráelſkim ták powieƺ / mówiąc : Olejek pomázowánia świętego mnie będźie <i>świętym</i> w narodźiech wáƺych.
Ciáło człowiecze / nie będźie nim mázáne / á według złożenia jego nie uczynićie <i>temu</i> podobnego <i>bo</i> święty jeſt / <i>y</i> święty wam będźie.
Ktobykolwiek uczynił tákową máść / á namázałby nią <i>kogo</i> obcego / wytrácony będźie z ludu ſwego.
Y rzekł PAN do Mojzeƺá : Weźmi ſobie rzeczy wonnych / Bálſámu / y Onychy / y Gálbánu wonnego / y kádźidłá czyſtego : wƺyſtkiego w równey wadze :
A uczyniƺ z tego kádzenie wonne / robotą Aptekárſką / to zmieƺánie czyſte / y święte <i>będźie</i> :
A utłukƺy to miáłko / kłáść będźieƺ z niego przed świádectwem w Namiećie zgromádzenia / gdźie śię z tobą zchodźić będę : Naświętƺe to będźie.
Kádzenia też którebyś czynił / według złożenia tego / nie uczynićie ſobie / toć będźie świętą rzeczą dla PANA.
Ktobykolwiek uczynił co podobnego / áby woniał z niego / wytrácony będźie z ludu ſwego.
