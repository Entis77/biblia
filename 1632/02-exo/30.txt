Ucżyniƺ też ołtárz dlá kádzeniá ; z drzewá ſytym ucżyniƺ go.
Ná łokieć w dłuż / y ná łokieć w ƺerz / cżworográniáſty będźie / á ná dwá łokćie w zwyż ; z niego wychodźić będą rogi jego.
A powlecżeƺ go ƺcżerem złotem / wierzch jego y śćiány jego w około / y rogi jego. Ucżyniƺ też koronę złotą około niego.
Y dwá kolce złote ucżyniƺ też pod koroną we dwu kątách jego / po obu ſtronách jego / á przez nie przewlecżeƺ drążki / áby noƺony był ná nich.
A ucżyniƺ drążki one z drzewá ſytym / y powlecżeƺ je złotem.
Y poſtáwiƺ go przed záſłoną / zá którą jeſt ſkrzyniá świádectwá przed ubłágálnią / którá jeſt nád świádectwem / gdźie śię z tobą ſchodźić będę.
A będźie kádźił ná nim Aáron kádzeniem z wonnych rzecży ná káżdy poránek ; przygotowáwƺy lámpy / będźie kádźił.
Tákże gdy rozpáli Aáron lámpy między dwomá wiecżorámi / kádźić będźie kádzeniem uſtáwicżnem przed Pánem w národźiech wáƺych.
Nie włożyćie náń kádźidłá obcego / áni cáłopáleniá / áni ofiáry ſuchej ; áni ofiáry mokrej ofiárowáć będźiećie ná nim.
Tylko wykoná ocżyƺcżenie Aáron nád rogámi jego ráz w rok ; przez krew ofiáry zá grzech / w dźień ocżyƺcżeniá / ráz w rok ocżyƺcżenie odpráwi ná nim w národźiech wáƺych ; bo to rzecż nájświętƺá Pánu.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Gdy zbierzeƺ główną ſummę ſynów Izráelſkich / z tych / którzy máją iść w licżbę / dá káżdy okup zá duƺę ſwą Pánu / gdy je licżyć będźieƺ / áby nie przyƺłá ná nie plágá / gdy zlicżeni będą.
To dáwáć będą : káżdy / który idźie w licżbę / dá pół ſyklá według ſyklá świątnicy ( dwádźieśćiá pieniędzy ſykiel wáży ; ) pół ſyklá będźie podárek Pánu.
Ktokolwiek idźie w licżbę ode dwudźieſtu lát y wyżej / oddá podárek Pánu.
Bogáty nie dá więcej / á ubogi nie dá mniej nád pół ſyklá / gdy będą dáwáć ofiárę Pánu / dlá ocżyƺcżeniá duƺ ſwoich.
A wybráwƺy pieniądze ocżyƺcżeniá od ſynów Izráelſkich / dáƺ je ná potrzeby námiotu zgromádzeniá / co będźie ſynom Izráelſkim ná pámiątkę przed Pánem / ku ocżyƺcżeniu duƺ wáƺych.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Ucżyniƺ też wánnę miedźiáną / y ſtolec jej miedźiány do umywániá / á poſtáwiƺ ją między námiotem zgromádzeniá / y między ołtárzem / y nálejeƺ w nią wody.
Y umywáć będą Aáron y ſynowie jego z niej ręce ſwoje y nogi ſwoje.
Gdy wchodźić będą do námiotu zgromádzeniá / umywáć śię będą wodą / áby nie pomárli ; tákże gdyby mieli przyſtępowáć do ołtárzá / áby ſłużyli / y zápálili ofiárę ogniſtą Pánu.
Y będą umywáli ręce ſwoje y nogi ſwoje / áby nie pomárli ; y będźie im to uſtáwą wiecżną / jemu y náśieniu jego / w rodzájách ich.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Ty też weźmij ſobie wonnych rzecży przednich : Myrry co nájcżyſtƺej pięćſet łótów / á cynámonu wonnego połowę tego / to jeſt / dwieśćie y pięćdźieśiąt łótów / y tátárſkiego źielá dwieśćie y pięćdźieśiąt ;
Káſyi też pięćſet łótów według ſyklá świątnicy / y oliwy z drzew oliwnych hyn.
Y ucżyniƺ z tego olejek pomázywániá świętego / máść nájwyborniejƺą / robotą áptekárſką : olejek to pomázowániá świętego będźie.
Y pomáżeƺ nim námiot zgromádzeniá / y ſkrzynię świádectwá.
Tákże ſtół y wƺyſtkie nácżyniá jego / y świecżnik / y nácżyniá jego / y ołtárz / ná którym kádzą ;
Ołtárz też do cáłopáleniá ze wƺyſtkiem nácżyniem jego / y wánnę z ſtolcem jej.
A poświęćiƺ je / áby nájświętƺe były ; cokolwiek śię ich dotknie / poświęcone będźie.
Aároná też / y ſyny jego pomáżeƺ / y poświęćiƺ je / áby mi ſpráwowáli urząd kápłáńſki.
A ſynom Izráelſkim ták powieƺ / mówiąc : Olejek pomázowániá świętego mnie będźie świętym w národźiech wáƺych ;
Ciáło cżłowiecże nie będźie nim mázáne / á według złożeniá jego nie ucżynićie temu podobnego : bo święty jeſt / y święty wám będźie.
Ktobykolwiek ucżynił tákową máść / á námázáłby nią kogo obcego / wytrácony będźie z ludu ſwego.
Y rzekł Pán do Mojżeƺá : Weźmij ſobie rzecży wonnych / bálſámu / y onychy / y gálbánu wonnego / y kádźidłá cżyſtego / wƺyſtkiego w równej wádze ;
A ucżyniƺ z tego kádzeniá wonne robotą áptekárſką ; to zmięƺánie cżyſte y święte będźie.
A utłukłƺy to miáłko / kłáść będźieƺ z niego przed świádectwem w námioćie zgromádzeniá / gdźie śię z tobą ſchodźić będę ; nájświętƺe to będźie.
Kádzeniá też / którebyś cżynił według złożeniá tego / nie ucżynićie ſobie ; toć będźie świętą rzecżą dlá Páná.
Ktobykolwiek ucżynił co podobnego / áby woniáł z niego / wytrácony będźie z ludu ſwego.
