Ucżyniƺ też ołtárz dlá kádzeniá ; z drzewá ſytym ucżyniƺ go.
Ná łokieć w dłuż / y ná łokieć w ƺerz / cżworográniáſty będzie / á ná dwá łokćie w zwyż ; z niego wychodzić będą rogi jego.
A powlecżeƺ go ƺcżerem złotem / wierzch jego y śćiány jego w około / y rogi jego. Ucżyniƺ też koronę złotą około niego.
Y dwá kolce złote ucżyniƺ też pod koroną we dwu kątách jego / po obu ſtronách jego / á przez nie przewlecżeƺ drążki / áby noƺony był ná nich.
A ucżyniƺ drążki one z drzewá ſytym / y powlecżeƺ je złotem.
Y poſtáwiƺ go przed záſłoną / zá którą jeſt ſkrzyniá świádectwá przed ubłágálnią / którá jeſt nád świádectwem / gdzie śię z tobą ſchodzić będę.
A będzie kádził ná nim Aáron kádzeniem z wonnych rzecży ná káżdy poránek ; przygotowáwƺy lámpy / będzie kádził.
Tákże gdy rozpáli Aáron lámpy między dwomá wiecżorámi / kádzić będzie kádzeniem uſtáwicżnem przed Pánem w národziech wáƺych.
Nie włożyćie náń kádzidłá obcego / áni cáłopáleniá / áni ofiáry ſuchej ; áni ofiáry mokrej ofiárowáć będziećie ná nim.
Tylko wykoná ocżyƺcżenie Aáron nád rogámi jego ráz w rok ; przez krew ofiáry zá grzech / w dzień ocżyƺcżeniá / ráz w rok ocżyƺcżenie odpráwi ná nim w národziech wáƺych ; bo to rzecż nájświętƺá Pánu.
Zátem rzekł Pán do Mojżeƺá / mówiąc :
Gdy zbierzeƺ główną ſummę ſynów Izráelſkich / z tych / którzy máją iść w licżbę / dá káżdy okup zá duƺę ſwą Pánu / gdy je licżyć będzieƺ / áby nie przyƺłá ná nie plágá / gdy zlicżeni będą.
To dáwáć będą : káżdy / który idzie w licżbę / dá pół ſyklá według ſyklá świątnicy ( dwádzieśćiá pieniędzy ſykiel wáży ; ) pół ſyklá będzie podárek Pánu.
Ktokolwiek idzie w licżbę ode dwudzieſtu lát y wyżej / oddá podárek Pánu.
Bogáty nie dá więcej / á ubogi nie dá mniej nád pół ſyklá / gdy będą dáwáć ofiárę Pánu / dlá ocżyƺcżeniá duƺ ſwoich.
A wybráwƺy pieniądze ocżyƺcżeniá od ſynów Izráelſkich / dáƺ je ná potrzeby námiotu zgromádzeniá / co będzie ſynom Izráelſkim ná pámiątkę przed Pánem / ku ocżyƺcżeniu duƺ wáƺych.
Potem rzekł Pán do Mojżeƺá / mówiąc :
Ucżyniƺ też wánnę miedziáną / y ſtolec jej miedziány do umywániá / á poſtáwiƺ ją między námiotem zgromádzeniá / y między ołtárzem / y nálejeƺ w nią wody.
Y umywáć będą Aáron y ſynowie jego z niej ręce ſwoje y nogi ſwoje.
Gdy wchodzić będą do námiotu zgromádzeniá / umywáć śię będą wodą / áby nie pomárli ; tákże gdyby mieli przyſtępowáć do ołtárzá / áby ſłużyli / y zápálili ofiárę ogniſtą Pánu.
Y będą umywáli ręce ſwoje y nogi ſwoje / áby nie pomárli ; y będzie im to uſtáwą wiecżną / jemu y náśieniu jego / w rodzájách ich.
Rzekł jeƺcże Pán do Mojżeƺá / mówiąc :
Ty też weźmij ſobie wonnych rzecży przednich : Myrry co nájcżyſtƺej pięćſet łótów / á cynámonu wonnego połowę tego / to jeſt / dwieśćie y pięćdzieśiąt łótów / y tátárſkiego zielá dwieśćie y pięćdzieśiąt ;
Káſyi też pięćſet łótów według ſyklá świątnicy / y oliwy z drzew oliwnych hyn.
Y ucżyniƺ z tego olejek pomázywániá świętego / máść nájwyborniejƺą / robotą áptekárſką : olejek to pomázowániá świętego będzie.
Y pomáżeƺ nim námiot zgromádzeniá / y ſkrzynię świádectwá.
Tákże ſtół y wƺyſtkie nácżyniá jego / y świecżnik / y nácżyniá jego / y ołtárz / ná którym kádzą ;
Ołtárz też do cáłopáleniá ze wƺyſtkiem nácżyniem jego / y wánnę z ſtolcem jej.
A poświęćiƺ je / áby nájświętƺe były ; cokolwiek śię ich dotknie / poświęcone będzie.
Aároná też / y ſyny jego pomáżeƺ / y poświęćiƺ je / áby mi ſpráwowáli urząd kápłáńſki.
A ſynom Izráelſkim ták powieƺ / mówiąc : Olejek pomázowániá świętego mnie będzie świętym w národziech wáƺych ;
Ciáło cżłowiecże nie będzie nim mázáne / á według złożeniá jego nie ucżynićie temu podobnego : bo święty jeſt / y święty wám będzie.
Ktobykolwiek ucżynił tákową máść / á námázáłby nią kogo obcego / wytrácony będzie z ludu ſwego.
Y rzekł Pán do Mojżeƺá : Weźmij ſobie rzecży wonnych / bálſámu / y onychy / y gálbánu wonnego / y kádzidłá cżyſtego / wƺyſtkiego w równej wádze ;
A ucżyniƺ z tego kádzeniá wonne robotą áptekárſką ; to zmięƺánie cżyſte y święte będzie.
A utłukłƺy to miáłko / kłáść będzieƺ z niego przed świádectwem w námioćie zgromádzeniá / gdzie śię z tobą ſchodzić będę ; nájświętƺe to będzie.
Kádzeniá też / którebyś cżynił według złożeniá tego / nie ucżynićie ſobie ; toć będzie świętą rzecżą dlá Páná.
Ktobykolwiek ucżynił co podobnego / áby woniáł z niego / wytrácony będzie z ludu ſwego.
