Potem tedy przyƺli Mojzeƺ y Aáron / y mówili do Fáráoná : Ták mówi Pán / Bóg Izráelſki : Puść lud mój / áby mi obchodźili święto ná puƺcży.
Y rzekł Fáráo : Któż jeſt Pán / żebym miał ſłucháć głoſu Jego / y puśćić Izráelá? Nie znam Páná / á Izráelá też nie puƺcżę.
Y odpowiedźieli : Bóg Hebrejcżyków zábieżał nam / pójdźiemy teraz drogą trzech dni ná puƺcżą / ábyſmy ofiárowáli Pánu Bogu náƺemu / by ſnadź nie przepuśćił ná nas moru álbo miecżá.
Y rzekł do nich król Egipſki : Przecż ty Mojżeƺu y Aáronie odrywáćie lud od robót ich? Idźćież do robót wáƺych.
Rzekł też Fáráo : Oto / wielki teraz jeſt ten lud w źiemi / á wy je odrywáćie od robót ich.
Rozkázał tedy Fáráo onegoż dniá przyſtáwom nád ludem / y urzędnikom jego / mówiąc :
Już więcey nie będźiećie dáwáć plew ludowi do cżynieniá cegły / jáko przed tym ; ſámi niech idą / y zbieráją ſobie plewy.
A tęż licżbę cegieł / którą cżynili przed tym / włożyćie im / nie umniejƺyćie z niej ; bo próżnują / dla tego oni wołáją / mówiąc : Pójdźiemy y będźiemy ofiárowáć Bogu náƺemu.
Niech śię przycżyni roboty mężom tym / á niech ją wykonywáją / áby nie ufáli ſłowom kłámliwym.
Wyƺedƺy tedy przyſtáwowie nád ludem y urzędnicy jego / rzekli do ludu / mówiąc : Ták mówi Fáráo : Ja nie będę wam dáwał plew.
Sámi idźćie / zbierájćie ſobie plewy / gdźie znájdźiećie ; bo śię nájmniej nie umniejƺy z roboty wáƺej.
Y rozbieżał śię lud po wƺyſtkiej źiemi Egipſkiej / áby zbierał śćierniſko miáſto plew.
A przyſtáwowie przynágláli / mówiąc : Wykonywájćie roboty wáƺe / zámiár káżdodźienny / jáko gdy wam dáwano plewy.
Tedy bito przyſtáwy ſynów Izráelſkich / które poſtánowili nád nimi urzędnicy Fáráonowi / mówiąc : Przecż nie wykonywáćie zámiáru ſwego w robieniu cegieł jáko pierwej / áni wcżoráj áni dźiś?
Y przyƺli przełożeni ſynów Izráelſkich y wołáli do Fáráoná / mówiąc : Cżemuż ták cżyniƺ ſługom twoim?
Plew nie dáją ſługom twoim / á mówią : Cegłę róbćie. Y oto / ſługi twe biją á lud twój grzeƺy.
Który rzekł : Próżnujećie / próżnujećie / dla tegoż mówićie : Pójdźiemy / ofiárowáć będźiemy Pánu.
Przetoż teraz idźćie / róbćie / á plew wam nie dádzą ; ále wy licżbę cegieł oddáwáć będźiećie.
A widząc przełożeni ſynów Izráelſkich / że źle z nimi / ponieważ mówiono : Nie umniejƺyćie z cegieł wáƺych zámiáru káżdodźiennego.
Tedy oni zábieżeli Mojżeƺowi y Aáronowi / którzy ſtáli / áby śię z nimi ſpotkáli / gdy wychodźili od Fáráoná.
Y rzekli do nich : Niech wejrzy Pán ná was á rozſądźi / żeśćie nas ohydźili w ocżách Fáráonowych / y w ocżách ſług jego / y dáliśćie miecż w rękę ich / áby nas zábili.
Y wróćił śię Mojzeƺ do Páná / á rzekł : Pánie / cżemuś to złe wprowádźił ná lud twój / cżemuś mię tu poſłał?
Bo od onego cżáſu / jákom wƺedł do Fáráoná / ábym mówił imieniem twojim / gorzej śię obchodźi z ludem tym ; á przećięś nie wybáwił ludu twego.
