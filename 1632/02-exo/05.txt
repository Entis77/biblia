Potem tedy przyƺli Mojżeƺ y Aáron / y mówili do Fáráoná : Ták mówi Pán / Bóg Izráelſki : Puść lud mój / áby mi obchodzili święto ná puƺcży.
Y rzekł Fáráo : Któż jeſt Pán / żebym miáł ſłucháć głoſu Jego / y puśćić Izráelá? Nie znám Páná / á Izráelá też nie puƺcżę.
Y odpowiedzieli : Bóg Hebrejcżyków zábieżáł nám / pójdziemy teráz drogą trzech dni ná puƺcżą / ábyśmy ofiárowáli Pánu Bogu náƺemu / by ſnáć nie przepuśćił ná náſ moru álbo miecżá.
Y rzekł do nich król Egipſki : Przecż ty Mojżeƺu y Aáronie odrywáćie lud od robót ich? Idźćież do robót wáƺych.
Rzekł też Fáráo : Oto / wielki teráz jeſt ten lud w ziemi / á wy je odrywáćie od robót ich.
Rozkázáł tedy Fáráo onegoż dniá przyſtáwom nád ludem / y urzędnikom jego / mówiąc :
Już więcej nie będziećie dáwáć plew ludowi do cżynieniá cegły / jáko przedtem ; ſámi niech idą / y zbieráją ſobie plewy.
A tęż licżbę cegieł / którą cżynili przedtem / włożyćie im / nie umniejƺyćie z niej ; bo próżnują / dlá tego oni wołáją / mówiąc : Pójdziemy y będziemy ofiárowáć Bogu náƺemu.
Niech śię przycżyni roboty mężom tym / á niech ją wykonywáją / áby nie ufáli ſłowom kłámliwym.
Wyƺedłƺy tedy przyſtáwowie nád ludem y urzędnicy jego / rzekli do ludu / mówiąc : Ták mówi Fáráo : Já nie będę wám dáwáł plew.
Sámi idźćie / zbierájćie ſobie plewy / gdzie znájdziećie ; bo śię nájmniej nie umniejƺy z roboty wáƺej.
Y rozbieżáł śię lud po wƺyſtkiej ziemi Egipſkiej / áby zbieráł śćierniſko miáſto plew.
A przyſtáwowie przynágláli / mówiąc : Wykonywájćie roboty wáƺe / zámiár káżdodzienny / jáko gdy wám dáwáno plewy.
Tedy bito przyſtáwy ſynów Izráelſkich / które poſtánowili nád nimi urzędnicy Fáráonowi / mówiąc : Przecż nie wykonywáćie zámiáru ſwego w robieniu cegieł jáko pierwej / áni wcżoráj áni dziś?
Y przyƺli przełożeni ſynów Izráelſkich y wołáli do Fáráoná / mówiąc : Cżemuż ták cżyniƺ ſługom twoim?
Plew nie dáją ſługom twoim / á mówią : Cegłę róbćie. Y oto / ſługi twe biją á lud twój grzeƺy.
Który rzekł : Próżnujećie / próżnujećie / dlá tegoż mówićie : Pójdziemy / ofiárowáć będziemy Pánu.
Przetoż teráz idźćie / róbćie / á plew wám nie dádzą ; ále wy licżbę cegieł oddáwáć będziećie.
A widząc przełożeni ſynów Izráelſkich / że źle z nimi / poniewáż mówiono : Nie umniejƺyćie z cegieł wáƺych zámiáru káżdodziennego.
Tedy oni zábieżeli Mojżeƺowi y Aáronowi / którzy ſtáli / áby śię z nimi ſpotkáli / gdy wychodzili od Fáráoná.
Y rzekli do nich : Niech wejrzy Pán ná wáſ á rozſądzi / żeśćie náſ ohydzili w ocżách Fáráonowych / y w ocżách ſług jego / y dáliśćie miecż w rękę ich / áby náſ zábili.
Y wróćił śię Mojżeƺ do Páná / á rzekł : Pánie / cżemuś to złe wprowádził ná lud twój / cżemuś mię tu poſłáł?
Bo od onego cżáſu / jákom wƺedł do Fáráoná / ábym mówił imieniem twojim / gorzej śię obchodzi z ludem tym ; á przećięś nie wybáwił ludu twego.
