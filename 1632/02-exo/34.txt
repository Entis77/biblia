Y rzekł PAN do Mojzeƺá : Wyćieƺ ſobie dwie tablice kámienne / podobne pierwƺym : á nápiƺę ná tychże tablicách ſłowá / które były ná tablicách pierwƺych / któreś ſtłukł.
A bądź gotów ráno / że wſtąpiƺ jutro ná górę Synáj / y ſtánieƺ przedemną / ná wierzchu tey góry.
Ale żaden niech nie wſtępuje z tobą : á nikt też niech nie będźie widźiány / po wƺyſtkiey górze : áni Owce / áni Woły / niech śię nie páſą / przećiwko tey górze :
Tedy wyćioſał Mojzeƺ dwie tablice kámienne / podobne pierwƺym / y wſtawƺy ráno / wſtąpił ná górę Synáj / jáko mu był rozkazał PAN / wźiąwƺy w ręce ſwe dwie tablice kámienne :
Y zſtąpił PAN w Obłoku / y ſtánął tám z nim / y záwołał imieniem PAN.
Bo przechodząc PAN przed twarzą jego / wołał : PAN / PAN : Bóg miłośierny y litośćiwy / nie rychły do gniewu : á obfity w miłośierdźiu / y w prawdźie.
Záchowywájący / miłośierdźie nád tyśiącmi : głádzący niepráwość y przeſtępſtwo / y grzech : nie uſpráwiedliwiájący winnego / náwiedzájąc niepráwość Ojcowſką w Synách / y w ſynách ſynów ich / do trzećiego y do cżwartego pokolenia.
Poſpieƺywƺy śię tedy Mojzeƺ / náchilił śię ku źiemi / y pokłonił śię.
Y rzekł : Jeslim teraz znalazł łáſkę w oczách twojich PAnie / niech idźie proƺę PAN w pośrzodku nas : lud bowiem ten / twárdego kárku jeſt : á odpuść Niepráwośći náƺe / y grzech náƺ / á miej nas zá dźiedźictwo.
Który odpowiedźiał : Oto ja poſtánowię przymierze / przed wƺyſtkim ludem twojim : Czynić będę cudá / które nie były czynione po wƺyſtkiey źiemi / y we wƺyſtkich narodźiech : y obacży wƺyſtek lud / miedzy którymeś ty / ſpráwę PAńſką : bo ſtráƺne będźie to / co ja uczynię z tobą.
Strzeżże tego / co ja dźiś roſkázuję tobie : Oto ja wypędzę przed obliczem twojim Amorejczyká / y Chánánejczyká / y Hetejczyká / y Ferezejczyká / y Hewejcżyká / y Jebuzejcżyká.
Strzeżże śię / ábyś ſnadź nie ſtánowił przymierza / z obywátelámi źiemie oney do którey ty wnidźieƺ / żebyć to nie było śidłem / w pośrzodku ćiebie :
Przetoż / Ołtarze ich zbórżyćie : báłwany ich połamiećie : y gáje ich <i>święcone</i> wyrąbićie.
Nie będźieƺ śię kłániał Bogu innemu / przeto że PAN <i>jeſt,</i> Zawiſtny imię jego / Bóg zawisny jeſt :
By ſnadź / uczyniwƺy przymierze z Obywátelem tey źiemie / gdyby oni cudzołożyli / z Bogi ſwymi : y ofiárowáli Bogom ſwym / ćiebie nie wezwáli / á jadłbyś z Ofiar ich :
Y brałbyś z Córek ich <i>żony</i> ſynom ſwym : y cudzołożyłyby córki ich z Bogi ſwymi : á przywiodłyby Syny twoje do wƺetecżeńſtwá z Bogi ſwymi.
Bogów odlewánych nie czyń ſobie.
Święto przáſników záchowywáć będźieƺ : przez śiedm dni jeść będźieƺ przaſniki / jákom ći rozkazał : czáſu mieśiącá Abib : ábowiem tegoż mieśiącá Abib wyƺedłeś z Egiptu.
Wƺyſtko co otwiera żywot / moje jeſt : y wƺyſtko z dobytku twego / cokolwiek ſámcem jeſt : pierworodne y z owiec y z wołów.
Ale pierworodne Oślę / odkupiƺ owcą : á jeslibyś go nie odkupił záłamieƺ mu ƺyję : káżdego pierworodnego z Synów twych odkupiƺ / y nie ukażą śię przed twarz moję prózni.
Sześć dni robić będźieƺ : á dniá śiódmego odpoczynieƺ : czáſu oránia / y czáſu żniwá odpoczynieƺ :
Święto Tegodniów uczyniƺ też ſobie : w pierwiaſtki żniwá pƺenicżnego : y ſwięto zbieránia / ná ſkończeniu roku :
Trzykroć do roku / ukaże śię káżdy męƺczyzná twój / przed Oblicże Pánującego PANA : Bogá Izráelſkiego.
Abowiem wypędzę Narody przed tobą : á rozƺerzę gránice twoje : y nie będźie pożądał nikt źiemie twojey / gdy pójdźieƺ / ábyś śię ukazał / przed obliczem PANA Bogá twego / trzykroć do roku.
Nie będźieƺ ofiárował przy kwáśie / krwie ofiáry mojey : y nie zoſtánie nic do jutrá / z ofiáry Obchodu ſwiętá Prześćia.
Pierwiaſtki / pierwƺych Urodzájów źiemie twey / przynieśieƺ w dom PANA / Bogá twego : Nie będźieƺ wárzył koźlęćiá / w mleku mátki jego.
Zátym rzekł PAN do Mojzeƺá : Nápiƺ ſobie te ſłowá / bo według ſłów tych poſtánowiłem z tobą przymierze / y z Izráelem.
Y był tám z PAnem czterdźieśći dni y czterdźieśći nocy : chlebá nie jadł / y wody nie pił : y nápiſał <i>PAN</i> ná tablicách ſłowá przymierza / Dźieśięć ſłów.
Y ſtáło śię / gdy zſtępował Mojzeƺ z góry Synáj / ( á dwie tablice świádectwá <i>miał</i> w ręku Mojzeƺ / gdy zſtępował z góry ) że nie wiedźiał Mojzeƺ iżby śię lſniłá ſkórá twarzy jego / gdy <i>PAN</i> mówił z nim.
Y ujrzeli / Aáron y wƺyſcy ſynowie Izráelſcy Mojzeƺá / á oto lſniłá śię ſkórá twárzy jego / y bali śię przyſtąpić do niego.
Ale záwołał ná nich Mojzeƺ : y náwróćili śię ku niemu Aáron / y wƺyſtkie kśiążętá zgromádzenia : y mówił Mojzeƺ do nich.
Potym też przyƺli wƺyſcy ſynowie Izráelſcy / którym przykazał wƺyſtko / co mówił PAN z nim / ná górze Synáj.
A póki Mojzeƺ mówił z nimi / miewał ná twarzy ſwojey zaſłonę :
Ale gdy wchadzał Mojzeƺ przed twarz PAńſką / áby rozmawiał z nim / odejmował zaſłonę / póki nie wyƺedł : á wyƺedƺy / mówił do Synów Izráelſkich co mu było rozkazano.
Widźieli tedy Synowie Izráelſcy twarz Mojzeƺowę że śię lſniłá ſkórá twarzy Mojzeƺowey : y kładł záś Mojzeƺ zaſłonę ná twarz ſwoję / póki nie wƺedł áby mówił z nim.
