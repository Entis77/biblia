Y rzekł Pán do Mojżeƺá : Wyćieƺ ſobie dwie táblice kámienne / podobne pierwƺym / á nápiƺę ná tychże táblicách ſłowá / które były ná táblicách pierwƺych / któreś ſtłukł.
A bądź gotów ráno / że wſtąpiƺ jutro ná górę Synáj / y ſtánieƺ przede mną ná wierzchu tey góry.
Ale żáden niech nie wſtępuje z tobą / á nikt też niech nie będźie widźiány po wƺyſtkiej górze ; áni owce / áni woły / niech śię nie páſą przećiwko tey górze.
Tedy wyćioſał Mojzeƺ dwie táblice kámienne / podobne pierwƺym ; y wſtáwƺy ráno / wſtąpił ná górę Synáj / jáko mu rozkázał Pán / wźiąwƺy w ręce ſwe dwie táblice kámienne.
Y zſtąpił Pán w obłoku / y ſtánął tám z nim / y záwołał imieniem Pán.
Bo przechodząc Pán przed twárzą jego / wołał : Pán / Pán / Bóg miłośierny y litośćiwy / nie rychły do gniewu / á obfity w miłośierdźiu y w práwdźie ;
Záchowujący miłośierdźie nád tyśiącámi / głádzący niepráwość y przeſtępſtwo y grzech / nie uſpráwiedliwiájący winnego / náwiedzájąc niepráwość ojcowſką w ſynách y w ſynách ſynów ich do trzećiego y do cżwártego pokolenia.
Poſpieƺywƺy śię tedy Mojzeƺ náchylił śię ku źiemi y pokłonił śię /
Y rzekł : Jeſlim teraz ználázł łáſkę w ocżách twojich / Pánie / niech idźie proƺę Pán w pośrzodku nas / lud bowiem ten twárdego kárku jeſt / á odpuść niepráwośći náƺe / y grzech náƺ / á miej nas zá dźiedźictwo.
Który odpowiedźiał : Oto / Ja poſtánowię przymierze ; przed wƺyſtkim ludem twoim cżynić będę cudá / które nie były cżynione po wƺyſtkiej źiemi y we wƺyſtkich národźiech ; y obácży wƺyſtek lud / miedzy którymeś ty / ſpráwę Páńſką ; bo ſtráƺne będźie to / co Ja ucżynię z tobą.
Strzeżże tego / co Ja dźiś rozkázuję tobie : Oto / Ja wypędzę przed oblicżem twojim Amorejcżyká / y Chánánejcżyká / y Hetejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká.
Strzeżże śię / ábyś ſnadź nie ſtánowił przymierza z obywátelámi źiemi oney / do którey ty wnijdźieƺ / żebyć to nie było śidłem w pośrzodku ćiebie.
Przetoż ołtárze ich zburzyćie / báłwány ich połámiećie / y gáje ich święcone wyrąbiećie.
Nie będźieƺ śię kłániał bogu innemu / przeto że Pán jeſt / záwiſny imię jego / Bóg záwiſny jeſt ;
By ſnadź / ucżyniwƺy przymierze z obywátelámi tey źiemi / gdyby oni cudzołożyli z bogámi ſwymi / y ofiárowáli bogom ſwym / ćiebie nie wezwáli / á jádłbyś z ofiár ich :
Y bráłbyś z córek ich żony ſynom ſwym / y cudzołożyłyby córki ich z bogi ſwymi / á przywiodłyby ſyny twoje do wƺetecżeńſtwá z bogi ſwymi.
Bogów odlewánych nie cżyń ſobie.
Święto przáśników záchowywáć będźieƺ ; przez śiedm dni jeść będźieƺ przáśniki / jákom ći rozkázał / cżáſu mieśiącá Abib ; ábowiem tegoż mieśiącá Abib wyƺedłeś z Egiptu.
Wƺyſtko / co otwierá żywot / moje jeſt ; y wƺyſtko z dobytku twego cokolwiek ſámcem jeſt / pierworodne y z owiec / y z wołów ;
Ale pierworodne oślę odkupiƺ owcą ; á jeſlibyś go nie odkupił / złámieƺ mu ƺyję. Káżdego pierworodnego z ſynów twych odkupiƺ / y nie ukáżą śię przed twárz moję próżni.
Sześć dni robić będźieƺ / á dniá śiódmego odpocżynieƺ ; cżáſu oránia y cżáſu żniwá odpocżynieƺ.
Święto Tygodni ucżyniƺ też ſobie / w pierwiáſtki żniwá pƺenicżnego / y święto zbieránia ná ſkońcżeniu roku.
Trzy kroć do roku ukáże śię káżdy mężcżyzná twój przed oblicżem Pánującego Páná / Bogá Izráelſkiego.
Abowiem wypędzę národy przed tobą / á rozƺerzę gránice twoje ; y nie będźie pożądał nikt źiemi twojey / gdy pójdźieƺ / ábyś śię ukázał przed oblicżem Páná Bogá twego trzy kroć do roku.
Nie będźieƺ ofiárował przy kwáśie krwi ofiáry mojey / y nie zoſtánie nic do jutrá z ofiáry obchodu świętá przejśćiá.
Pierwiáſtki pierwƺych urodzájów źiemi twej przynieśieƺ w dom Páná Bogá twego. Nie będźieƺ wárzył koźlęćiá w mleku mátki jego.
Zátem rzekł Pán do Mojżeƺá : Nápiƺ ſobie te ſłowá / bo według ſłów tych poſtánowiłem z tobą przymierze / y z Izráelem.
Y był tám z Pánem cżterdźieśći dni y cżterdźieśći nocy ; chlebá nie jádł / y wody nie pił ; y nápiſał Pán ná táblicách ſłowá przymierza / dźieśięć ſłów.
Y ſtáło śię / gdy zſtępował Mojzeƺ z góry Synáj / á dwie táblice świádectwá miał w ręku Mojzeƺ / gdy zſtępował z góry / że nie wiedźiał Mojzeƺ / iżby śię lśniłá ſkórá twárzy jego / gdy Pán mówił z nim.
Y ujrzeli Aáron / y wƺyſcy ſynowie Izráelſcy Mojżeƺá / á oto lśniłá śię ſkórá twárzy jego / y báli śię przyſtąpić do niego.
Ale záwołał ná nich Mojzeƺ / y náwróćili śię ku niemu Aáron / y wƺyſtkie kśiążętá zgromádzenia / y mówił Mojzeƺ do nich.
Potem też przyƺli wƺyſcy ſynowie Izráelſcy / którym przykázał wƺyſtko / co mówił Pán z nim ná górze Synáj.
A póki Mojzeƺ mówił z nimi / miewał ná twárzy ſwojey záſłonę ;
Ale gdy wchádzał Mojzeƺ przed twárz Páńſką / áby rozmáwiał z nim / odejmował záſłonę / póki nie wyƺedł ; á wyƺedƺy / mówił do ſynów Izráelſkich / co mu było rozkázáno.
Widźieli tedy ſynowie Izráelſcy twárz Mojżeƺową / że śię lśniłá ſkórá twárzy Mojżeƺowej ; y kłádł záś Mojzeƺ záſłonę ná twárz ſwoję / póki nie wƺedł / áby mówił z nim.
