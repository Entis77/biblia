Y rzekł Pán do Mojżeƺá : Wyćieƺ ſobie dwie táblice kámienne / podobne pierwƺym / á nápiƺę ná tychże táblicách ſłowá / które były ná táblicách pierwƺych / któreś ſtłukł.
A bądź gotów ráno / że wſtąpiƺ jutro ná górę Synáj / y ſtánieƺ przede mną ná wierzchu tej góry.
Ale żáden niech nie wſtępuje z tobą / á nikt też niech nie będzie widziány po wƺyſtkiej górze ; áni owce / áni woły / niech śię nie páſą przećiwko tej górze.
Tedy wyćioſáł Mojżeƺ dwie táblice kámienne / podobne pierwƺym ; y wſtáwƺy ráno / wſtąpił ná górę Synáj / jáko mu rozkázáł Pán / wziąwƺy w ręce ſwe dwie táblice kámienne.
Y zſtąpił Pán w obłoku / y ſtánął tám z nim / y záwołáł imieniem Pán.
Bo przechodząc Pán przed twárzą jego / wołáł : Pán / Pán / Bóg miłośierny y litośćiwy / nie rychły do gniewu / á obfity w miłośierdziu y w práwdzie ;
Záchowujący miłośierdzie nád tyśiącámi / głádzący niepráwość y przeſtępſtwo y grzech / nie uſpráwiedliwiájący winnego / náwiedzájąc niepráwość ojcowſką w ſynách y w ſynách ſynów ich do trzećiego y do cżwártego pokoleniá.
Poſpieƺywƺy śię tedy Mojżeƺ náchylił śię ku ziemi y pokłonił śię /
Y rzekł : Jeźlim teráz ználázł łáſkę w ocżách twoich / Pánie / niech idzie proƺę Pán w pośrodku náſ / lud bowiem ten twárdego kárku jeſt / á odpuść niepráwośći náƺe / y grzech náƺ / á miej náſ zá dziedzictwo.
Który odpowiedziáł : Oto / Já poſtánowię przymierze ; przed wƺyſtkim ludem twoim cżynić będę cudá / które nie były cżynione po wƺyſtkiej ziemi y we wƺyſtkich národziech ; y obácży wƺyſtek lud / między którymeś ty / ſpráwę Páńſką ; bo ſtráƺne będzie to / co Já ucżynię z tobą.
Strzeżże tego / co Já dziś rozkázuję tobie : Oto / Já wypędzę przed oblicżem twojim Amorejcżyká / y Chánánejcżyká / y Hetejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká.
Strzeżże śię / ábyś ſnáć nie ſtánowił przymierzá z obywátelámi ziemi onej / do której ty wnijdzieƺ / żebyć to nie było śidłem w pośrodku ćiebie.
Przetoż ołtárze ich zburzyćie / báłwány ich połámiećie / y gáje ich święcone wyrąbiećie.
Nie będzieƺ śię kłániáł bogu innemu / przeto że Pán jeſt / záwiſny imię jego / Bóg záwiſny jeſt ;
By ſnáć / ucżyniwƺy przymierze z obywátelámi tej ziemi / gdyby oni cudzołożyli z bogámi ſwymi / y ofiárowáli bogom ſwym / ćiebie nie wezwáli / á jádłbyś z ofiár ich :
Y bráłbyś z córek ich żony ſynom ſwym / y cudzołożyłyby córki ich z bogi ſwymi / á przywiodłyby ſyny twoje do wƺetecżeńſtwá z bogi ſwymi.
Bogów odlewánych nie cżyń ſobie.
Święto przáśników záchowywáć będzieƺ ; przez śiedm dni jeść będzieƺ przáśniki / jákom ći rozkázáł / cżáſu mieśiącá Abib ; ábowiem tegoż mieśiącá Abib wyƺedłeś z Egiptu.
Wƺyſtko / co otwierá żywot / moje jeſt ; y wƺyſtko z dobytku twego cokolwiek ſámcem jeſt / pierworodne y z owiec / y z wołów ;
Ale pierworodne oślę odkupiƺ owcą ; á jeźlibyś go nie odkupił / złámieƺ mu ƺyję. Káżdego pierworodnego z ſynów twych odkupiƺ / y nie ukáżą śię przed twárz moję próżni.
Sześć dni robić będzieƺ / á dniá śiódmego odpocżynieƺ ; cżáſu orániá y cżáſu żniwá odpocżynieƺ.
Święto Tygodni ucżyniƺ też ſobie / w pierwiáſtki żniwá pƺenicżnego / y święto zbierániá ná ſkońcżeniu roku.
Trzy kroć do roku ukáże śię káżdy mężcżyzná twój przed oblicżem Pánującego Páná / Bogá Izráelſkiego.
Abowiem wypędzę národy przed tobą / á rozƺerzę gránice twoje ; y nie będzie pożądáł nikt ziemi twojej / gdy pójdzieƺ / ábyś śię ukázáł przed oblicżem Páná Bogá twego trzy kroć do roku.
Nie będzieƺ ofiárowáł przy kwáśie krwi ofiáry mojej / y nie zoſtánie nic do jutrá z ofiáry obchodu świętá przejśćiá.
Pierwiáſtki pierwƺych urodzájów ziemi twej przynieśieƺ w dom Páná Bogá twego. Nie będzieƺ wárzył koźlęćiá w mleku mátki jego.
Zátem rzekł Pán do Mojżeƺá : Nápiƺ ſobie te ſłowá / bo według ſłów tych poſtánowiłem z tobą przymierze / y z Izráelem.
Y był tám z Pánem cżterdzieśći dni y cżterdzieśći nocy ; chlebá nie jádł / y wody nie pił ; y nápiſáł Pán ná táblicách ſłowá przymierzá / dzieśięć ſłów.
Y ſtáło śię / gdy zſtępowáł Mojżeƺ z góry Synáj / á dwie táblice świádectwá miáł w ręku Mojżeƺ / gdy zſtępowáł z góry / że nie wiedziáł Mojżeƺ / iżby śię lśniłá ſkórá twárzy jego / gdy Pán mówił z nim.
Y ujrzeli Aáron / y wƺyſcy ſynowie Izráelſcy Mojżeƺá / á oto lśniłá śię ſkórá twárzy jego / y báli śię przyſtąpić do niego.
Ale záwołáł ná nich Mojżeƺ / y náwróćili śię ku niemu Aáron / y wƺyſtkie kśiążętá zgromádzeniá / y mówił Mojżeƺ do nich.
Potem też przyƺli wƺyſcy ſynowie Izráelſcy / którym przykázáł wƺyſtko / co mówił Pán z nim ná górze Synáj.
A póki Mojżeƺ mówił z nimi / miewáł ná twárzy ſwojej záſłonę ;
Ale gdy wchádzáł Mojżeƺ przed twárz Páńſką / áby rozmáwiáł z nim / odejmowáł záſłonę / póki nie wyƺedł ; á wyƺedłƺy / mówił do ſynów Izráelſkich / co mu było rozkázáno.
Widzieli tedy ſynowie Izráelſcy twárz Mojżeƺową / że śię lśniłá ſkórá twárzy Mojżeƺowej ; y kłádł záś Mojżeƺ záſłonę ná twárz ſwoję / póki nie wƺedł / áby mówił z nim.
