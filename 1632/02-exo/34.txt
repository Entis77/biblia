Y rzekł Pán do Mojżeƺá : Wyćieƺ ſobie dwie táblice kámienne / podobne pierwƺym / á nápiƺę ná tychże táblicách ſłowá / które były ná táblicách pierwƺych / któreś ſtłukł.
A bądź gotów ráno / że wſtąpiƺ jutro ná górę Synáj / y ſtánieƺ przede mną ná wierzchu tey góry.
Ale żáden niech nie wſtępuje z tobą / á nikt też niech nie będźie widźiány po wƺyſtkiej górze ; áni owce / áni woły / niech śię nie páſą przećiwko tey górze.
Tedy wyćioſáł Mojżeƺ dwie táblice kámienne / podobne pierwƺym ; y wſtáwƺy ráno / wſtąpił ná górę Synáj / jáko mu rozkázáł Pán / wźiąwƺy w ręce ſwe dwie táblice kámienne.
Y zſtąpił Pán w obłoku / y ſtánął tám z nim / y záwołáł imieniem Pán.
Bo przechodząc Pán przed twárzą jego / wołáł : Pán / Pán / Bóg miłośierny y litośćiwy / nie rychły do gniewu / á obfity w miłośierdźiu y w práwdźie ;
Záchowujący miłośierdźie nád tyśiącámi / głádzący niepráwość y przeſtępſtwo y grzech / nie uſpráwiedliwiájący winnego / náwiedzájąc niepráwość ojcowſką w ſynách y w ſynách ſynów ich do trzećiego y do cżwártego pokoleniá.
Poſpieƺywƺy śię tedy Mojżeƺ náchylił śię ku źiemi y pokłonił śię /
Y rzekł : Jeſlim teraz ználázł łáſkę w ocżách twoich / Pánie / niech idźie proƺę Pán w pośrodku nas / lud bowiem ten twárdego kárku jeſt / á odpuść niepráwośći náƺe / y grzech náƺ / á miej nas zá dźiedźictwo.
Który odpowiedźiáł : Oto / Ja poſtánowię przymierze ; przed wƺyſtkim ludem twoim cżynić będę cudá / które nie były cżynione po wƺyſtkiej źiemi y we wƺyſtkich národźiech ; y obácży wƺyſtek lud / między którymeś ty / ſpráwę Páńſką ; bo ſtráƺne będźie to / co Ja ucżynię z tobą.
Strzeżże tego / co Ja dźiś rozkázuję tobie : Oto / Ja wypędzę przed oblicżem twojim Amorejcżyká / y Chánánejcżyká / y Hetejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká.
Strzeżże śię / ábyś ſnáć nie ſtánowił przymierzá z obywátelámi źiemi oney / do którey ty wnijdźieƺ / żebyć to nie było śidłem w pośrodku ćiebie.
Przetoż ołtárze ich zburzyćie / báłwány ich połámiećie / y gáje ich święcone wyrąbiećie.
Nie będźieƺ śię kłániáł bogu innemu / przeto że Pán jeſt / záwiſny imię jego / Bóg záwiſny jeſt ;
By ſnáć / ucżyniwƺy przymierze z obywátelámi tey źiemi / gdyby oni cudzołożyli z bogámi ſwymi / y ofiárowáli bogom ſwym / ćiebie nie wezwáli / á jádłbyś z ofiár ich :
Y bráłbyś z córek ich żony ſynom ſwym / y cudzołożyłyby córki ich z bogi ſwymi / á przywiodłyby ſyny twoje do wƺetecżeńſtwá z bogi ſwymi.
Bogów odlewánych nie cżyń ſobie.
Święto przáśników záchowywáć będźieƺ ; przez śiedm dni jeść będźieƺ przáśniki / jákom ći rozkázáł / cżáſu mieśiącá Abib ; ábowiem tegoż mieśiącá Abib wyƺedłeś z Egiptu.
Wƺyſtko / co otwierá żywot / moje jeſt ; y wƺyſtko z dobytku twego cokolwiek ſámcem jeſt / pierworodne y z owiec / y z wołów ;
Ale pierworodne oślę odkupiƺ owcą ; á jeſlibyś go nie odkupił / złámieƺ mu ƺyję. Káżdego pierworodnego z ſynów twych odkupiƺ / y nie ukáżą śię przed twárz moję próżni.
Sześć dni robić będźieƺ / á dniá śiódmego odpocżynieƺ ; cżáſu orániá y cżáſu żniwá odpocżynieƺ.
Święto Tygodni ucżyniƺ też ſobie / w pierwiáſtki żniwá pƺenicżnego / y święto zbierániá ná ſkońcżeniu roku.
Trzy kroć do roku ukáże śię káżdy mężcżyzná twój przed oblicżem Pánującego Páná / Bogá Izráelſkiego.
Abowiem wypędzę národy przed tobą / á rozƺerzę gránice twoje ; y nie będźie pożądáł nikt źiemi twojey / gdy pójdźieƺ / ábyś śię ukázáł przed oblicżem Páná Bogá twego trzy kroć do roku.
Nie będźieƺ ofiárowáł przy kwáśie krwi ofiáry mojey / y nie zoſtánie nic do jutrá z ofiáry obchodu świętá przejśćiá.
Pierwiáſtki pierwƺych urodzájów źiemi twej przynieśieƺ w dom Páná Bogá twego. Nie będźieƺ wárzył koźlęćiá w mleku mátki jego.
Zátem rzekł Pán do Mojżeƺá : Nápiƺ ſobie te ſłowá / bo według ſłów tych poſtánowiłem z tobą przymierze / y z Izráelem.
Y był tám z Pánem cżterdźieśći dni y cżterdźieśći nocy ; chlebá nie jádł / y wody nie pił ; y nápiſáł Pán ná táblicách ſłowá przymierzá / dźieśięć ſłów.
Y ſtáło śię / gdy zſtępowáł Mojżeƺ z góry Synáj / á dwie táblice świádectwá miał w ręku Mojżeƺ / gdy zſtępowáł z góry / że nie wiedźiáł Mojżeƺ / iżby śię lśniłá ſkórá twárzy jego / gdy Pán mówił z nim.
Y ujrzeli Aáron / y wƺyſcy ſynowie Izráelſcy Mojżeƺá / á oto lśniłá śię ſkórá twárzy jego / y báli śię przyſtąpić do niego.
Ale záwołáł ná nich Mojżeƺ / y náwróćili śię ku niemu Aáron / y wƺyſtkie kśiążętá zgromádzeniá / y mówił Mojżeƺ do nich.
Potem też przyƺli wƺyſcy ſynowie Izráelſcy / którym przykázáł wƺyſtko / co mówił Pán z nim ná górze Synáj.
A póki Mojżeƺ mówił z nimi / miewáł ná twárzy ſwojey záſłonę ;
Ale gdy wchádzáł Mojżeƺ przed twárz Páńſką / áby rozmáwiáł z nim / odejmowáł záſłonę / póki nie wyƺedł ; á wyƺedłƺy / mówił do ſynów Izráelſkich / co mu było rozkázáno.
Widźieli tedy ſynowie Izráelſcy twárz Mojżeƺową / że śię lśniłá ſkórá twárzy Mojżeƺowej ; y kłádł záś Mojżeƺ záſłonę ná twárz ſwoję / póki nie wƺedł / áby mówił z nim.
