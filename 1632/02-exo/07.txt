Y rzekł Pán do Mojżeƺá : Oto / poſtánowiłem ćię zá Bogá Fáráonowi / á Aáron / brát twój / będźie prorokiem twoim.
Ty powieƺ wƺyſtko / coć rozkáżę : ále Aáron / brát twój / będźie mówił do Fáráoná / áby wypuśćił ſyny Izráelſkie z źiemi ſwej.
A Já zátwárdzę ſerce Fáráonowe / y rozmnożę znáki moje y cudá moje w źiemi Egipſkiej.
Y nie uſłuchá wáſ Fáráo ; lecż Já włożę rękę moję ná Egipt / y wyprowádzę wojſká moje / lud mój / ſyny Izráelſkie / z źiemi Egipſkiej w ſądźiech wielkich.
A poznáją Egipcżánie / żem Já Pán / gdy wyćiągnę rękę moję ná Egipt / y wywiodę ſyny Izráelſkie z pośrodku ich.
Ucżynił tedy Mojżeƺ y Aáron / jáko im przykázáł Pán / ták ucżynili.
A Mojżeƺ miáł ośmdźieśiąt lát / á Aáron ośmdźieśiąt y trzy látá / gdy mówili do Fáráoná.
Rzekł tedy Pán do Mojżeƺá / y do Aároná / mówiąc :
Gdy wám rzecże Fáráo / mówiąc : Ucżyńćie jáki cud / tedy rzecżeƺ do Aároná : Weźmij láſkę twoję / á porzuć przed Fáráonem / á obróći śię w wężá.
Y przyƺedł Mojżeƺ z Aáronem do Fáráoná / y ucżynili ták / jáko rozkázáł Pán ; y porzućił Aáron láſkę ſwoję przed Fáráonem / y przed ſługámi jego / którá śię obróćiłá w wężá.
Wezwáł też Fáráo mędrców y cżárowników / y ucżynili y ći cżárownicy Egipſcy przez cżáry ſwe tákże.
Y porzućił káżdy láſkę ſwą / á obróćiły śię w węże ; ále pożárłá láſká Aáronowá láſki ich.
Y zátwárdźiáło ſerce Fáráonowe / y nie uſłucháł ich / jáko powiedźiáł Pán.
Zátem rzekł Pán do Mojżeƺá : Oćiężáło ſerce Fáráonowe ; nie chce puśćić ludu tego.
Idź do Fáráoná ráno / oto / wynijdźie nád wodę / tedy ſtánieƺ przećiwko niemu nád brzegiem rzeki / á láſkę / którá śię byłá obróćiłá w wężá / weźmieƺ w rękę twoję /
Y rzecżeƺ do niego : Pán Bóg Hebrejcżyków poſłáł mię do ćiebie / mówiąc : Wypuść lud mój / áby mi ſłużyli ná puƺcży / á oto / nie uſłucháłeś dotąd.
Przetoż ták mówi Pán : Po tem poznáƺ / żem já Pán ; oto / já uderzę láſką / którá jeſt w ręce mojej / ná wody / które ſą w rzece / á obrócą śię w krew.
A ryby / które ſą w rzece / pozdycháją / y zśmierdnie śię rzeká / y ſprácują śię Egipcżánie / ƺukájąc dlá nápoju wód z rzeki.
Tedy rzekł Pán do Mojżeƺá : Mów do Aároná : Weźmij láſkę twoję / á wyćiągnij rękę twą ná wody Egipſkie / ná rzeki ich / ná ſtrugi ich / y ná jeźiorá ich / y ná wƺelkie zgromádzenie wód ich ; y obrócą śię w krew / y będźie krew po wƺyſtkiej źiemi Egipſkiej / ták w nácżyniách drzewiánych / jáko w kámiennych.
Y ucżynili ták Mojżeƺ y Aáron / jáko rozkázáł Pán ; y podnióſłƺy láſkę uderzył wody / które były w rzece / przed ocżymá Fáráonowemi / y przed ocżymá ſług jego ; y obróćiły śię wƺyſtkie wody / które były w rzece / w krew.
A ryby / które były w rzece / pozdycháły / y zśmierdłá śię rzeká / że nie mogli Egipcżánie pić wody z rzeki ; y byłá krew po wƺyſtkiej źiemi Egipſkiej.
Y ucżynili tákże cżárownicy Egipſcy cżárámi ſwemi ; y zátwárdźiáło ſerce Fáráonowe / y nie uſłucháł ich / jáko był powiedźiáł Pán.
A odwróćiwƺy śię Fáráo / poƺedł do domu ſwego / á nie przyłożył ſercá ſwego y do tego.
Y kopáli wƺyſcy Egipcżánie około rzeki / ƺukájąc wody / áby pili ; bo nie mogli pić wody z rzeki.
Y wypełniło śię śiedm dni / jáko záráźił Pán rzekę.
