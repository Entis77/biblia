Záśpiewáł tedy Mojżeƺ y ſynowie Izráelſcy tę pieśń Pánu / á rzekli / mówiąc : Śpiewáć będę Pánu / iż wielmożnie wywyżƺon jeſt ; koniá y jezdnego jego wrzućił w morze.
Moc mojá y chwáłá mojá Pán / bo mi śię ſtáł zbáwieniem ; ten jeſt Bogiem moim / przetoż przybytek wyſtáwię mu ; Bóg ojcá mego / przetoż wywyżƺáć go będę.
Pán / mąż wálecżny / Pán imię jego.
Wozy Fáráonowe y wojſko jego wrzućił w morze / á wybráni wodzowie jego potopieni ſą w morzu cżerwonem.
Przepáśći okryły je ; poƺli w głębią jáko kámień.
Práwicá twojá / Pánie / uwielbioná jeſt w mocy / práwicá twojá / Pánie / potárłá nieprzyjáćielá.
A w wielkośći Májeſtátu twego podwróćiłeś przećiwniki twoje ; puśćiłeś gniew twój / który je pożárł jáko ſłomę.
A tchnieniem nozdrz twoich zebráły śię wody ; ſtánęły jáko kupá ćiekące wody / zśiádły śię otchłáni w pośród morzá.
Mówił nieprzyjáćiel : Będę gonił / dogonię ; będę dźielił łupy ; náſyći śię ich duƺá mojá / dobędę miecżá mego / wygłádźi je ręká mojá.
Wionąłeś wiátrem twym / okryło je morze ; połknieni ſą jáko ołów w wodách gwáłtownych.
Któż podobien tobie między bogámi / Pánie? któż jáko ty wielmożny w świętobliwośći / ſtráƺliwy w chwále / cżyniący cudá?
Wyćiągnąłeś práwicę twoję / pożárłá je źiemiá.
Prowádźiƺ w miłośierdźiu twojim ten lud / któryś odkupił ; poprowádźiƺ w możnośći twej do mieƺkániá świętobliwośći twojej.
Uſłyƺą národowie / zádrżą ; boleść zejmie obywátele Filiſtyńſkie.
Tedy śię polękáją kśiążętá Edomſkie / mocárze Moábſkie ſtrách zejmie ; ſtruchleją wƺyſcy obywátele Chánánejſcy.
Pádnie ná nie ſtrách y lękánie ; od wielkośći rámieniá twego umilkną jáko kámień / áż przejdźie lud twój Pánie / áż przejdźie lud ten / któregoś ſobie nábył.
Wprowádźiƺ je / y wƺcżepiƺ je ná górze dźiedźictwá twego / ná miejſcu / któreś ku mieƺkániu twemu ſpráwił / Pánie ; w świątnicy / Pánie / którą umocnią ręce twoje.
Pán królowáć będźie ná wieki wiecżne.
Bo weƺły konie Fáráonowe z wozámi jego / y z jezdnymi jego w morze / á obróćił Pán ná nie wody morſkie ; ále ſynowie Izráelſcy ƺli po ſuƺy środkiem morzá.
Tedy Máryjá / prorokini / śioſtrá Aáronowá / wźięłá bęben w rękę ſwoję / á wyƺły wƺyſtkie niewiáſty zá nią z bębnámi y z muzyką.
Y mówiłá do nich Máryjá : Śpiewájćie Pánu / ábowiem możnie wywyżƺon jeſt ; koniá y jezdnego jego wrzućił do morzá.
Potem ruƺył Mojżeƺ Izráelá od morzá cżerwonego / y weƺli w puƺcżą Sur ; á idąc trzy dni przez puƺcżą / nie ználeźli wody.
A gdy przyƺli do Márá / nie mogli pić wód z Márá / bo gorzkie były ; dlátegoż názwáno imię onego miejſcá Márá.
Tedy ƺemráł lud przećiw Mojżeƺowi / mówiąc : Cóż będźiemy pić?
Y wołáł ( Mojżeƺ ) do Páná ; á ukázáł mu Pán drzewo / które gdy wrzućił do wód / ſtáły śię ſłodkie wody. Tám mu uſtáwił práwá y ſądy / y tám go kuśił ;
Y rzekł : Będźieƺli pilnie ſłucháł głoſu Páná Bogá twego / á co dobrego w ocżách jego cżynić będźieƺ / y nákłoniƺ uƺy ku przykázániom jego / ſtrzegąc wƺyſtkich uſtáw jego / żádnej niemocy / którąm dopuśćił ná Egipt / nie dopuƺcżę ná ćię ; bom Já Pán / który ćię lecżę.
Y przyƺli do Elim / gdźie było dwánáśćie źródeł wód / y śiedmdźieśiąt pálm ; y położyli śię tám obozem nád wodámi.
