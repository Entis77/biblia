Y rzekł Pán do Mojżeƺá mówiąc :
Mów do ſynów Izráelſkich / niech śię wrócą y położą obozem przed Fiháhirot miedzy Migdoł / y miedzy morzem / przećiw Báálſefon / przećiw jemu położyćie obóz nád morzem.
Będźie bowiem Fáráo mówił o ſynách Izráelſkich : Strwożeni ſą w źiemi / záwárłá je puƺcżá.
Y zátwárdzę ſerce Fáráonowe / że je gonić będźie ; y uwielbiony będę w Fáráonie y we wƺyſtkim wojſku jego ; á poznáją Egipcżánie / żem Ja Pán ; y ucżynili ták.
Tedy dáno znáć królowi Egipſkiemu / że lud ućieká ; y odmieniło śię ſerce Fáráonowe y ſług jego przećiw ludowi / y rzekli : Cóżeśmy to ucżynili / żeśmy wypuśćili Izráelá / áby nam nie ſłużył?
Záprzągł tedy wóz ſwój / y lud ſwój wźiął z ſobą.
Wźiął też ƺeść ſet wozów wybornych / y wƺyſtkie wozy Egipſkie / y przełożone nád tym wƺyſtkim.
Y zátwárdźił Pán ſerce Fáráoná / królá Egipſkiego / y gonił ſyny Izráelſkie ; lecż ſynowie Izráelſcy wyƺli w ręce możney.
Y gonili je Egipcżánie / á dogonili je w oboźie nád morzem / wƺyſtkie konie / wozy Fáráonowe / y jezdne jego / y wojſká jego / nie dáleko Fiháhirot / przećiw Báálſefon.
A gdy Fáráo náſtępował / tedy podnieśli ſynowie Izráelſcy ocży ſwe / á oto / Egipcżánie ćiągną zá nimi ; y báli śię bárzo / y wołáli ſynowie Izráelſcy do Páná.
Y mówili do Mojżeƺá : Azaż nie było grobów w Egipćie? wywiodłeś nas / ábyſmy pomárli ná puƺcży ; cóżeś nam to ucżynił / żeś nas wywiódł z Egiptu?
Azaż nie to jeſt / cośmy do ćiebie mówili w Egipćie / mówiąc : Zániecháj nas / ábyſmy ſłużyli Egipcżánom? bo lepiej było nam ſłużyć Egipcżánom / niżeli pomrzeć ná puƺcży.
Y rzekł Mojzeƺ do ludu : Nie bójćie śię / ſtójćie / á pátrzájćie ná wybáwienie Páńſkie / które wam dźiś ucżyni ; bo Egipcżánów / których teraz widźićie / więcey nie oglądáćie ná wieki.
Pán będźie wálcżył zá was / á wy milcżeć będźiećie.
Y rzekł Pán do Mojżeƺá : Cóż wołáƺ do mnie? Mów do ſynów Izráelſkich / áby ćiągnęli ;
A ty podnieś láſkę twą / y wyćiągnij rękę twoję ná morze / y przedźiel je ; á niech idą ſynowie Izráelſcy środkiem morzá po ſuƺy.
A oto / Ja / Ja zátwárdzę ſerce Egipcżánów / że wnijdą zá nimi ; á będę uwielbiony w Fáráonie / y we wƺyſtkim wojſku jego / w woźiech jego / y w jezdnych jego.
Y dowiedzą śię Egipcżánie / żem Ja Pán / gdy uwielbiony będę w Fáráonie / w woźiech jego / y w jezdnych jego.
A ruƺywƺy śię Anjoł Boży / który chodźił przed obozem Izráelſkim / ƺedł pozád ich ; ruƺył śię też ſłup obłokowy / który ƺedł przed nimi / y ſtánął pozád im.
A przyƺedƺy miedzy obóz Egipſki / y miedzy obóz Izráelſki / był on obłok Egipcżánom ćiemny á Izráelcżykom oświecájący noc / ták / iż przyſtąpić nie mogli jedni do drugich przez cáłą noc.
Y wyćiągnął Mojzeƺ rękę ſwoję ná morze / á Pán rozpędźił morze wiátrem wſchodnim gwáłtownie wiejącym przez cáłą noc / y oſuƺył morze ; á rozſtąpiły śię wody.
Y ƺli ſynowie Izráelſcy środkiem morzá po ſuƺy ; á wody im były jáko mur / po práwej ſtronie ich / y po lewej ſtronie ich.
A goniąc Egipcżánie / weƺli zá nimi ; wƺyſtkie konie Fáráonowe / wozy jego / y jezdni jego / w pośrodek morzá.
Stáło śię tedy około ſtráży záránney / że wejrzał Pán ná obóz Egipſki z ſłupá ogniá y obłoku / y pomieƺał wojſko Egipſkie.
Y pozrucał kołá wozów ich / że je wlekli z ćiężkośćią ; zácżem rzekli Egipcżánie : Ućiekájmy przed Izráelem / bo Pán wálcży zá nimi przećiw Egipcżánom.
Y rzekł Pán do Mojżeƺá : Wyćiągnij rękę twoję ná morze / że śię wrócą wody ná Egipcżány / ná wozy ich / y ná jezdne ich.
Y wyćiągnął Mojzeƺ rękę ſwoję ná morze / y wróćiło śię morze záráz z ráná do mocy ſwojey ; á Egipcżánie ućiekáli przećiw jemu ; lecż Pán wráźił Egipcżány w pośród morzá.
Wróćiwƺy śię tedy wody / okryły wozy y jezdne / ze wƺyſtkim wojſkiem Fáráonowem / które weƺło zá nimi w morze / ták iż nie zoſtáło z nich y jednego.
Synowie záś Izráelſcy ƺli po ſuƺy środkiem morzá / á wody im były jáko mur / po práwej ſtronie ich / y po lewej ſtronie ich.
Y wybáwił Pán w on dźień Izráelá z ręki Egipcżánów ; y widźieli Izráelcżycy Egipcżány pomárłe ná brzegu morſkim.
Widźiał też Izráel onę moc wielką / którą ucżynił Pán nád Egipcżány ; á bał śię lud Páná / y uwierzyli Pánu / y Mojżeƺowi / ſłudze jego.
