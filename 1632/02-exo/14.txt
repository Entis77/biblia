Y rzekł Pán do Mojżeƺá mówiąc :
Mów do ſynów Izráelſkich / niech śię wrócą y położą obozem przed Fiháhirot między Migdoł / y między morzem / przećiw Báálſefon / przećiw jemu położyćie obóz nád morzem.
Będźie bowiem Fáráo mówił o ſynách Izráelſkich : Strwożeni ſą w źiemi / záwárłá je puƺcżá.
Y zátwárdzę ſerce Fáráonowe / że je gonić będźie ; y uwielbiony będę w Fáráonie y we wƺyſtkiem wojſku jego ; á poznáją Egipcżánie / żem Ja Pán ; y ucżynili ták.
Tedy dáno znáć królowi Egipſkiemu / że lud ućieká ; y odmieniło śię ſerce Fáráonowe y ſług jego przećiw ludowi / y rzekli : Cóżeśmy to ucżynili / żeśmy wypuśćili Izráelá / áby nám nie ſłużył?
Záprzągł tedy wóz ſwój / y lud ſwój wźiął z ſobą.
Wźiął też ƺeść ſet wozów wybornych / y wƺyſtkie wozy Egipſkie / y przełożone nád tem wƺyſtkiem.
Y zátwárdźił Pán ſerce Fáráoná / królá Egipſkiego / y gonił ſyny Izráelſkie ; lecż ſynowie Izráelſcy wyƺli w ręce możney.
Y gonili je Egipcżánie / á dogonili je w oboźie nád morzem / wƺyſtkie konie / wozy Fáráonowe / y jezdne jego / y wojſká jego / nie dáleko Fiháhirot / przećiw Báálſefon.
A gdy Fáráo náſtępowáł / tedy podnieśli ſynowie Izráelſcy ocży ſwe / á oto / Egipcżánie ćiągną zá nimi ; y báli śię bárdzo / y wołáli ſynowie Izráelſcy do Páná.
Y mówili do Mojżeƺá : Azáż nie było grobów w Egipćie? wywiodłeś nas / ábyśmy pomárli ná puƺcży ; cóżeś nám to ucżynił / żeś nas wywiódł z Egiptu?
Azáż nie to jeſt / cośmy do ćiebie mówili w Egipćie / mówiąc : Zániecháj nas / ábyśmy ſłużyli Egipcżánom? bo lepiej było nám ſłużyć Egipcżánom / niżeli pomrzeć ná puƺcży.
Y rzekł Mojżeƺ do ludu : Nie bójćie śię / ſtójćie / á pátrzájćie ná wybáwienie Páńſkie / które wám dźiś ucżyni ; bo Egipcżánów / których teraz widźićie / więcey nie oglądáćie ná wieki.
Pán będźie wálcżył zá was / á wy milcżeć będźiećie.
Y rzekł Pán do Mojżeƺá : Cóż wołáƺ do mnie? Mów do ſynów Izráelſkich / áby ćiągnęli ;
A ty podnieś láſkę twą / y wyćiągnij rękę twoję ná morze / y przedźiel je ; á niech idą ſynowie Izráelſcy środkiem morzá po ſuƺy.
A oto / Ja / Ja zátwárdzę ſerce Egipcżánów / że wnijdą zá nimi ; á będę uwielbiony w Fáráonie / y we wƺyſtkiem wojſku jego / w woźiech jego / y w jezdnych jego.
Y dowiedzą śię Egipcżánie / żem Ja Pán / gdy uwielbiony będę w Fáráonie / w woźiech jego / y w jezdnych jego.
A ruƺywƺy śię Anioł Boży / który chodźił przed obozem Izráelſkim / ƺedł pozád ich ; ruƺył śię też ſłup obłokowy / który ƺedł przed nimi / y ſtánął pozád im.
A przyƺedłƺy między obóz Egipſki / y między obóz Izráelſki / był on obłok Egipcżánom ćiemny á Izráelcżykom oświecájący noc / ták / iż przyſtąpić nie mogli jedni do drugich przez cáłą noc.
Y wyćiągnął Mojżeƺ rękę ſwoję ná morze / á Pán rozpędźił morze wiátrem wſchodnim gwáłtownie wiejącym przez cáłą noc / y oſuƺył morze ; á rozſtąpiły śię wody.
Y ƺli ſynowie Izráelſcy środkiem morzá po ſuƺy ; á wody im były jáko mur / po práwej ſtronie ich / y po lewej ſtronie ich.
A goniąc Egipcżánie / weƺli zá nimi ; wƺyſtkie konie Fáráonowe / wozy jego / y jezdni jego / w pośrodek morzá.
Stáło śię tedy około ſtráży záránney / że wejrzáł Pán ná obóz Egipſki z ſłupá ogniá y obłoku / y pomieƺáł wojſko Egipſkie.
Y pozrucáł kołá wozów ich / że je wlekli z ćiężkośćią ; zácżem rzekli Egipcżánie : Ućiekájmy przed Izráelem / bo Pán wálcży zá nimi przećiw Egipcżánom.
Y rzekł Pán do Mojżeƺá : Wyćiągnij rękę twoję ná morze / że śię wrócą wody ná Egipcżány / ná wozy ich / y ná jezdne ich.
Y wyćiągnął Mojżeƺ rękę ſwoję ná morze / y wróćiło śię morze záráz z ráná do mocy ſwojey ; á Egipcżánie ućiekáli przećiw jemu ; lecż Pán wráźił Egipcżány w pośród morzá.
Wróćiwƺy śię tedy wody / okryły wozy y jezdne / ze wƺyſtkiem wojſkiem Fáráonowem / które weƺło zá nimi w morze / ták iż nie zoſtáło z nich y jednego.
Synowie záś Izráelſcy ƺli po ſuƺy środkiem morzá / á wody im były jáko mur / po práwej ſtronie ich / y po lewej ſtronie ich.
Y wybáwił Pán w on dźień Izráelá z ręki Egipcżánów ; y widźieli Izráelcżycy Egipcżány pomárłe ná brzegu morſkim.
Widźiáł też Izráel onę moc wielką / którą ucżynił Pán nád Egipcżány ; á báł śię lud Páná / y uwierzyli Pánu / y Mojżeƺowi / ſłudze jego.
