Potem odpowiádájąc Mojżeƺ / rzekł : Ale oto / nie uwierzą mi / y nie uſłucháją głoſu mego / bo rzeką : Nie ukázáł śię tobie Pán.
Y rzekł mu Pán : Cóż jeſt w ręce twojej? y odpowiedziáł : Láſká.
Y rzekł : Porzuć ją ná ziemię ; y porzućił ją ná ziemię / á obróćiłá śię w wężá / y ućiekáł Mojżeƺ przed nim.
Y rzekł Pán do Mojżeƺá : Wyćiągnij rękę twoję / á ujmij go zá ogon ; y wyćiągnął rękę ſwoję / y ujął go / y obróćił śię w láſkę w ręce jego.
Abyć uwierzyli / iż ći śię ukázáł Pán / Bóg ojców ich / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów.
Y rzekł mu Pán jeƺcże : Włóż teráz rękę twoję w zánádrá twoje ; y włożył rękę ſwoję w zánádrá ſwoje ; y wyjął ją / á oto / ręká jego byłá trędowátá jáko śnieg.
Y rzekł : Włóż znowu rękę twoję w zánádrá twe ; y włożył znowu rękę ſwoję w zánádrá ſwe ; á gdy ją wyjął z zánádrzá ſwego / á oto / ſtáłá śię znowu jáko inne ćiáło jego.
Y ſtánie śię / jeźlić nie uwierzą y nie uſłucháją głoſu znáku pierwƺego / tedy uwierzą głoſowi znáku pośledniego.
Y ſtánie śię / jeźli nie uwierzą áni tym dwom znákom / y nie uſłucháją głoſu twego / weźmieƺ wody rzecżnej / y wylejeƺ ją ná ziemię ; tedy śię przemieni wodá oná / którą weźmieƺ z rzeki / á obróći śię w krew ná ziemi.
Y rzekł Mojżeƺ do Páná. Proƺę Pánie / nie jeſtem já mężem wymownym áni przedtem áni odtąd / jákoś mówił do ſługi twego : bom ćiężkich uſt y ćiężkiego języká.
A Pán mu rzekł : Któż ucżynił uſtá cżłowiekowi? álbo kto ucżynił niemego / álbo głuchego / álbo widzącego / álbo ślepego / izáż nie Já Pán?
Idźże teráz / á Já będę z uſty twojimi / y náucżę ćię / cobyś miáł mówić.
Y rzekł Mojżeƺ : Słucháj Pánie / poślij proƺę tego / kogo poſłáć máƺ.
Y zápálił śię gniew Páńſki ná Mojżeƺá / y rzekł : Azáż niewiem / iż Aáron / brát twój Lewitá wymownym jeſt? á oto y ten wynijdzie przećiwko tobie / y ujrzáwƺy ćię / uráduje śię w ſercu ſwojim.
Y będzieƺ mówił do niego / y włożyƺ ſłowá w uſtá jego / á Já będę z uſty twemi / y z uſty jego / y náucżę wáſ / cobyśćie mieli cżynić.
On będzie mówił zá ćię do ludu / y ſtánie śię / że on będzie tobie zá uſtá / á ty mu będzieƺ zá Bogá ;
Láſkę też tę weźmij w rękę twoję / którą będzieƺ cżynił znáki.
Odƺedł tedy Mojżeƺ / y wróćił śię do Jetrá / świekrá ſwego / y mówił do niego : Pójdę teráz / á wrócę śię do bráći mojej / którzy ſą w Egipćie / á obácżę / ſąli jeƺcże żywi. A Jetro rzekł do Mojżeƺá : Idź w pokoju.
Y rzekł Pán do Mojżeƺá w ziemi Mádyjáńſkiej : Idź / wróć śię do Egiptu ; pomárli bowiem wƺyſcy mężowie / którzy ƺukáli duƺy twojej.
Wziąwƺy tedy Mojżeƺ żonę ſwą / y ſyny ſwe / wſádził je ná oſłá / y wróćił śię do ziemi Egipſkiej ; wziął też Mojżeƺ láſkę Bożą w rękę ſwą.
Y rzekł Pán do Mojżeƺá : Gdy pójdzieƺ y wróćiƺ śię do Egiptu / pátrzájże / ábyś wƺyſtkie cudá / którem Já podáł w rękę twoję / cżynił przed Fáráonem / á Já zátwárdzę ſerce jego / áby nie wypuśćił lud.
Y rzecżeƺ do Fáráoná : Ták mówi Pán : Syn mój / pierworodny mój jeſt Izráel.
Y rzekłem do ćiebie : Wypuść ſyná mego / áby mi ſłużył ; á żeś go nie chćiáł wypuśćić / oto / Já zábiję ſyná twojego / pierworodnego twego.
Y ſtáło śię w drodze / w goſpodzie / że zábieżáł Pán Mojżeƺowi / y chćiáł go zábić.
Tedy wziąwƺy Zeforá krzemień oſtry / obrzezáłá nieobrzezkę ſyná ſwego / y porzućiłá przed nogi jego / y rzekłá : Zápráwdę oblubieńcem krwi jeſteś mi.
Y odƺedł od niego Pán. Tedy go názwáłá oblubieńcem krwi / dlá obrzezániá.
Y rzekł Pán do Aároná : Idź przećiwko Mojżeƺowi ná puƺcżą. Y ƺedł y záƺedł mu ná górze Bożej / y pocáłowáł go.
Tedy powiedziáł Mojżeƺ Aáronowi wƺyſtkie ſłowá Páná / który go poſłáł / y wƺyſtkie znáki / które mu rozkázáł.
Szedłƺy tedy Mojżeƺ z Aáronem / zgromádzili wƺyſtkie ſtárƺe ſynów Izráelſkich.
Y powiedziáł Aáron wƺyſtkie ſłowá / które mówił Pán do Mojżeƺá / á Mojżeƺ cżynił znáki przed ocżymá ludu.
Y uwierzył lud / y zrozumieli / że náwiedził Pán ſyny Izráelſkie / á iż wejrzáł ná utrápienie ich ; y ſchyliwƺy śię / pokłon ucżynili.
