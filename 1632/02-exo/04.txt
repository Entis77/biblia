Potem odpowiádájąc Mojzeƺ / rzekł : Ale oto / nie uwierzą mi / y nie uſłucháją głoſu mego / bo rzeką : Nie ukázał śię tobie Pán.
Y rzekł mu Pán : Cóż jeſt w ręce twojey? y odpowiedźiał : Láſká.
Y rzekł : Porzuć ją ná źiemię ; y porzućił ją ná źiemię / á obróćiłá śię w wężá / y ućiekał Mojzeƺ przed nim.
Y rzekł Pán do Mojżeƺá : Wyćiągnij rękę twoję / á ujmij go zá ogon ; y wyćiągnął rękę ſwoję / y ujął go / y obróćił śię w láſkę w ręce jego.
Abyć uwierzyli / iż ći śię ukázał Pán / Bóg ojców ich / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów.
Y rzekł mu Pán jeƺcże : Włóż teraz rękę twoję w zánádrá twoje ; y włożył rękę ſwoję w zánádrá ſwoje ; y wyjął ją / á oto / ręká jego byłá trędowátá jáko śnieg.
Y rzekł : Włóż znowu rękę twoję w zánádrá twe ; y włożył znowu rękę ſwoję w zánádrá ſwe ; á gdy ją wyjął z zánádrzá ſwego / á oto / ſtáłá śię znowu jáko inne ćiáło jego.
Y ſtánie śię / jeſlić nie uwierzą y nie uſłucháją głoſu znáku pierwƺego / tedy uwierzą głoſowi znáku pośledniego.
Y ſtánie śię / jeſli nie uwierzą áni tym dwom znákom / y nie uſłucháją głoſu twego / weźmieƺ wody rzecżney / y wylejeƺ ją ná źiemię ; tedy śię przemieni wodá oná / którą weźmieƺ z rzeki / á obróći śię w krew ná źiemi.
Y rzekł Mojzeƺ do Páná. Proƺę Pánie / nie jeſtem ja mężem wymownym áni przed tym áni odtąd / jákoś mówił do ſługi twego : bom ćiężkich uſt y ćiężkiego języká.
A Pán mu rzekł : Któż ucżynił uſtá cżłowiekowi? álbo kto ucżynił niemego / álbo głuchego / álbo widzącego / álbo ślepego / izáż nie Ja Pán?
Idźże teraz / á Ja będę z uſty twojimi / y náucżę ćię / cobyś miał mówić.
Y rzekł Mojzeƺ : Słucháj Pánie / poślij proƺę tego / kogo poſłáć maƺ.
Y zápálił śię gniew Páńſki ná Mojżeƺá / y rzekł : Azaż niewiem / iż Aáron / brát twój Lewitá wymownym jeſt? á oto y ten wynijdźie przećiwko tobie / y ujrzáwƺy ćię / uráduje śię w ſercu ſwojim.
Y będźieƺ mówił do niego / y włożyƺ ſłowá w uſtá jego / á Ja będę z uſty twemi / y z uſty jego / y náucżę was / cobyśćie mieli cżynić.
On będźie mówił zá ćię do ludu / y ſtánie śię / że on będźie tobie zá uſtá / á ty mu będźieƺ zá Bogá ;
Láſkę też tę weźmij w rękę twoję / którą będźieƺ cżynił znáki.
Odƺedł tedy Mojzeƺ / y wróćił śię do Jetrá / świekrá ſwego / y mówił do niego : Pójdę teraz / á wrócę śię do bráći mojey / którzy ſą w Egipćie / á obácżę / ſąli jeƺcże żywi. A Jetro rzekł do Mojżeƺá : Idź w pokoju.
Y rzekł Pán do Mojżeƺá w źiemi Mádyjáńſkiej : Idź / wróć śię do Egiptu ; pomárli bowiem wƺyſcy mężowie / którzy ƺukáli duƺy twojey.
Wźiąwƺy tedy Mojzeƺ żonę ſwą / y ſyny ſwe / wſádźił je ná oſłá / y wróćił śię do źiemi Egipſkiej ; wźiął też Mojzeƺ láſkę Bożą w rękę ſwą.
Y rzekł Pán do Mojżeƺá : Gdy pójdźieƺ y wróćiƺ śię do Egiptu / pátrzájże / ábyś wƺyſtkie cudá / którem Ja podał w rękę twoję / cżynił przed Fáráonem / á Ja zátwárdzę ſerce jego / áby nie wypuśćił lud.
Y rzecżeƺ do Fáráoná : Ták mówi Pán : Syn mój / pierworodny mój jeſt Izráel.
Y rzekłem do ćiebie : Wypuść ſyná mego / áby mi ſłużył ; á żeś go nie chćiał wypuśćić / oto / Ja zábiję ſyná twojego / pierworodnego twego.
Y ſtáło śię w drodze / w goſpodźie / że zábieżał Pán Mojżeƺowi / y chćiał go zábić.
Tedy wźiąwƺy Zeforá krzemień oſtry / obrzezáłá nieobrzezkę ſyná ſwego / y porzućiłá przed nogi jego / y rzekłá : Zápráwdę oblubieńcem krwi jeſteś mi.
Y odƺedł od niego Pán. Tedy go názwáłá oblubieńcem krwi / dla obrzezánia.
Y rzekł Pán do Aároná : Idź przećiwko Mojżeƺowi ná puƺcżą. Y ƺedł y záƺedł mu ná górze Bożey / y pocáłował go.
Tedy powiedźiał Mojzeƺ Aáronowi wƺyſtkie ſłowá Páná / który go poſłał / y wƺyſtkie znáki / które mu rozkázał.
Szedƺy tedy Mojzeƺ z Aáronem / zgromádźili wƺyſtkie ſtárƺe ſynów Izráelſkich.
Y powiedźiał Aáron wƺyſtkie ſłowá / które mówił Pán do Mojżeƺá / á Mojzeƺ cżynił znáki przed ocżymá ludu.
Y uwierzył lud / y zrozumieli / że náwiedźił Pán ſyny Izráelſkie / á iż wejrzał ná utrápienie ich ; y ſchyliwƺy śię / pokłon ucżynili.
