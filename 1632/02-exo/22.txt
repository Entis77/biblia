Jeſliby kto ukrádł wołu álbo owcę / y zábiłby je / álbo je ſprzedáł / pięć wołów wróći zá jednego wołu / á cżtery owce zá jedną owcę /
Jeſliby przy podkopywániu záſtány był złodźiej / á ubity będąc umárłby / kto zábił / nie będźie winien krwi ;
Jeſliby to po wejśćiu ſłońcá ucżynił / krwi winien będźie / koniecżnie wróći ; á jeſli nie má / ſprzedány będźie zá złodźiejſtwo ſwoje.
Jeſli ználeźioná będźie w ręku jego rzecż krádźioná / bądź wół / bądź ośieł / bądź owcá / jeƺcże żywe / we dwójnáſób wróći.
Gdyby wypáſł kto pole / álbo winnicę / y puśćiłby bydlę ſwoje / áby śię páſło ná polu cudzem : z nájlepƺego urodzáju polá ſwego / y z nájlepƺego urodzáju winnicy ſwej nágrodźi.
Jeſliby wyƺedł ogień / á tráfiłby ná ćiernie / y ſpáliłby ſtóg / álbo ſtojące zboże / álbo pole ſámo / koniecżnie nágrodźi ten / co zápálił / co zgorzáło.
Gdyby kto dáł bliźniemu ſwemu ſrebro / álbo nácżynie do ſchowániá / á to by ukrádźiono było z domu onego cżłowieká : jeſliby ználeźiony był złodźiej / wróći dwojáko.
Jeſliby nie był ználeźiony złodźiej / ſtáwi śię pán domu onego przed ſędźie / y przyśięże / że nie śćiągnął ręki ſwojey ná rzecż bliźniego ſwego.
O káżdą rzecż / o którąby był ſpór / o wołu / o oſłá / o owcę / o ƺátę / o káżdą rzecż zgubioną / gdyby kto rzekł / że to jeſt moje / przed ſędźie przyjdźie ſpráwá obudwu ; kogo winnym znájdą ſędźiowie / nágrodźi w dwójnáſób bliźniemu ſwemu.
Jeſliby kto dáł bliźniemu ſwemu oſłá / álbo wołu / álbo owcę / álbo inne bydlę ná chowánie / á zdechłoby álbo okálecżáło / álbo gwáłtem zájęte było / gdźieby nikt nie widźiáł :
Przyśięgá Páńſká będźie między obiemá / że nie śćiągnął ręki ſwej ná rzecż bliźniego ſwego : y przyjmie pán oney rzecży przyśięgę / á on nie będźie nágrádzáł.
A jeſliby mu to krádźieżą wźięto / nágrodźi pánu rzecży oney.
Jeſliby od zwierzá rozƺárpáne było / poſtáwi rozƺárpáne zá świádká / á rozƺárpánego nie nágrodźi.
Gdyby też kto pożycżył bydlęćiá od bliźniego ſwego / á okálecżáłoby / álbo zdechło w niebytnośći páná jego / koniecżnie nágrodźi.
Jeſliby pán jego był przy nim / nie będźie nágrádzáł ; á jeſliby nájęte było / nájem tylko zápłáći.
Jeſliby kto zwiódł pánnę / która nie jeſt poślubioná / y ſpáłby z nią / dá jey koniecżnie wiáno / y weźmie ją ſobie zá żonę.
Jeſliby żádną miárą ojćiec jey nie chćiáł mu jey dáć / odwáży ſrebrá według zwycżáju wiáná pánieńſkiego.
Cżárownicy żyć nie dopuśćiƺ.
Káżdy / ktoby śię złącżáł z bydlęćiem / śmierćią umrze.
Ktoby ofiárowáł bogom / oprócż ſámego Páná / wytrácony będźie.
Przychodniowi nie ucżyniƺ krzywdy / áni go ućiśnieƺ : bośćie byli przychodniámi w źiemi Egipſkiej.
Żádney wdowy áni śieroty trápić nie będźiećie.
Jeſlibyś je bez litośći trápił / á oneby wołáły do mnie / ſłyƺąc wyſłuchám wołánie ich.
Y rozgniewá śię zápálcżywość mojá / á pobiję was miecżem / y będą żony wáƺe wdowámi / á ſynowie wáśi śierotámi.
Jeſlibyś pieniędzy pożycżył ludowi memu ubogiemu / który mieƺká z tobą / nie bądźieƺ mu jáko lichwiárz / nie obćiążyćie go lichwą.
Jeſli w záſtáwie weźmieƺ ƺátę bliźniego twego / przed záchodem ſłońcá wróćiƺ mu ją ;
Bo to odźienie jego tylko to jeſt nákryćie ćiáłá jego / ná którem ſypiá ; będźieli do mnie wołáł / wyſłuchám go / bom Ja miłośierny.
Sędźiom nie będźieƺ złorzecżył / á przełożonego ludu twego nie będźieƺ przeklináł.
Z obfitośći zbożá / y ćiekących rzecży twych nie omieƺkáƺ pierwiáſtek ofiárowáć ; pierworodnego z ſynów twoich oddáƺ mi.
Toż ucżyniƺ z wołów twych y z owiec twoich ; śiedm dni będźie z mátką ſwoją / á óſmego dniá oddáƺ mi je.
Ludem świętym będźiećie mi / á mięſá ná polu rozƺárpánego jeść nie będźiećie ; pſom je wyrzućićie.
