A gdy uſłyƺáł Jetro / kápłán Mádyjáńſki / świekier Mojżeƺá / wƺyſtko / co ucżynił Bóg Mojżeƺowi / y Izráelowi / ludowi ſwemu / że wywiódł Pán Izráelá z Egiptu ;
Tedy wziął Jetro / świekier Mojżeƺá / Zeforę / żonę Mojżeƺowę / którą był odeſłáł.
Y dwu ſynów jej / z których imię jednemu Gerſon ; bo był powiedziáł Mojżeƺ : Byłem przychodniem w ziemi cudzej.
A imię drugiego Eliezer ; iż mówił : Bóg ojcá mojego był mi ku pomocy / y wyrwáł mię od miecżá Fáráonowego.
Y przyƺedł Jetro / świekier Mojżeƺá / z ſynámi jego y z żoną jego do Mojżeƺá ná puƺcżą / gdzie śię był obozem położył przy górze Bożej.
Y wſkázáł do Mojżeƺá : Já świekier twój Jetro idę do ćiebie / y żoná twojá / y jej dwá ſynowie z nią.
Zátem Mojżeƺ wyƺedł przećiwko świekrowi ſwemu / á ukłoniwƺy śię cáłowáł go ; y przywitáwƺy jeden drugiego / potem weƺli do námiotu.
Y rozpowiádáł Mojżeƺ świekrowi ſwemu wƺyſtko / co ucżynił Pán Fáráonowi y Egipcżánom zá przycżyną Izráelá ; y wƺyſtkę trudność / którá je ſpotkáłá w drodze / y jáko je Pán wybáwił.
Y rádowáł śię Jetro ze wƺyſtkiego dobrego / które ucżynił Pán Izráelowi / iż go wyrwáł z ręki Egipcżánów.
Y rzekł Jetro : Błogoſłáwiony Pán / który wáſ wyrwáł z ręki Egipcżánów y z ręki Fáráonowej / który wyrwáł lud z niewoli Egipſkiej.
Terázem doznáł / że więkƺy jeſt Pán nád wƺyſtkie bogi ; ábowiem cżem oni hárdzie powſtáwáli przećiwko niemu / tem poginęli.
Y wziął Jetro / świekier Mojżeƺá / cáłopálenie y ofiáry Bogu. Przyƺedł też Aáron y wƺyſcy ſtárśi Izráelſcy / áby jedli chleb z świekrem Mojżeƺowym przed Bogiem.
Y ſtáło śię ná zájutrz / że uśiádł Mojżeƺ / áby ſądził lud / y ſtáł lud przed Mojżeƺem od poránku áż do wiecżorá.
A widząc świekier Mojżeƺá wƺyſtko / co on cżynił z ludem / rzekł : Cóż to jeſt / co ty cżyniƺ z ludem? cżemuż ty ſám śiedziƺ / á lud wƺyſtek ſtoi przed tobą od poránku áż do wiecżorá?
Tedy Mojżeƺ odpowiedziáł świekrowi ſwemu : Iż przychodzi lud do mnie / áby śię rádził Bogá.
Gdy ſpráwę jáką máją / przychodzą do mnie / á rozſądzám między nimi / oznájmując uſtáwy Boże y práwá jego.
Zátem rzekł świekier Mojżeƺá do niego : Nie dobrá rzecż / którą ty cżyniƺ.
Pewnie uſtánieƺ / y ty y lud ten / który jeſt z tobą / bo ćiężƺá to rzecż nád śiły twoje ; nie będzieƺ jej mógł ty ſám podołáć.
Przetoż uſłucháj teráz głoſu mego / porádzęć / á będzie Bóg z tobą ; ſtój ty zá lud przed Bogiem / á odnoś ſpráwy do Bogá ;
A onych też náucżáj uſtáw y práw / oznájmując im drogę / którąby chodzić / y dzieło / któreby cżynić mieli.
Ty też upátrz ze wƺyſtkiego ludu męże ſtátecżne / bojące śię Bogá / męże práwdomówne / którzyby nienáwidzili łákomſtwá / á poſtánów z nich przełożone / tyśiącżniki / ſetniki / pięćdzieśiątniki y dzieśiątniki.
Którzy ná káżdy cżáſ lud ſądzić będą ; á gdy będzie rzecż wielká / odnioſą do ćiebie / á káżdą rzecż máłą ſądzić będą ſámi ; tedy ulżyƺ ſobie / gdy ponioſą ćiężár z tobą.
To jeźli ucżyniƺ / á rozkáżeć Bóg / oſtoiƺ śię / y ten wƺyſtek lud ná miejſcá ſwoje wrácáć śię będzie w pokoju.
Y uſłucháł Mojżeƺ rády świekrá ſwojego / á ucżynił wƺyſtko / jáko mu powiedziáł.
Y wybráł Mojżeƺ męże ſtátecżne ze wƺyſtkiego Izráelá / y poſtánowił je przełożonymi nád ludem / tyśiącżniki / ſetniki / pięćdzieśiątniki y dzieśiątniki.
Którzy ſądzili lud káżdego cżáſu ; trudne rzecży odnośili do Mojżeƺá / á káżdą rzecż mniejƺą ſámi ſądzili.
Zátem puśćił od śiebie Mojżeƺ świekrá ſwego / który odƺedł do ziemi ſwej.
