A gdy uſłyƺáł Jetro / kápłán Mádyjáńſki / świekier Mojżeƺá / wƺyſtko / co ucżynił Bóg Mojżeƺowi / y Izráelowi / ludowi ſwemu / że wywiódł Pán Izráelá z Egiptu ;
Tedy wźiął Jetro / świekier Mojżeƺá / Zeforę / żonę Mojżeƺowę / którą był odeſłáł.
Y dwu ſynów jey / z których imię jednemu Gerſon ; bo był powiedźiáł Mojżeƺ : Byłem przychodniem w źiemi cudzej.
A imię drugiego Eliezer ; iż mówił : Bóg ojcá mojego był mi ku pomocy / y wyrwáł mię od miecżá Fáráonowego.
Y przyƺedł Jetro / świekier Mojżeƺá / z ſynámi jego y z żoną jego do Mojżeƺá ná puƺcżą / gdźie śię był obozem położył przy górze Bożej.
Y wſkázáł do Mojżeƺá : Ja świekier twój Jetro idę do ćiebie / y żoná twojá / y jey dwá ſynowie z nią.
Zátem Mojżeƺ wyƺedł przećiwko świekrowi ſwemu / á ukłoniwƺy śię cáłowáł go ; y przywitáwƺy jeden drugiego / potem weƺli do námiotu.
Y rozpowiádáł Mojżeƺ świekrowi ſwemu wƺyſtko / co ucżynił Pán Fáráonowi y Egipcżánom zá przycżyną Izráelá ; y wƺyſtkę trudność / która je ſpotkáłá w drodze / y jáko je Pán wybáwił.
Y rádowáł śię Jetro ze wƺyſtkiego dobrego / które ucżynił Pán Izráelowi / iż go wyrwáł z ręki Egipcżánów.
Y rzekł Jetro : Błogoſłáwiony Pán / który was wyrwáł z ręki Egipcżánów y z ręki Fáráonowej / który wyrwáł lud z niewoli Egipſkiej.
Terazem doznał / że więkƺy jeſt Pán nád wƺyſtkie bogi ; ábowiem cżem oni hárdźie powſtáwáli przećiwko niemu / tem poginęli.
Y wźiął Jetro / świekier Mojżeƺá / cáłopálenie y ofiáry Bogu. Przyƺedł też Aáron y wƺyſcy ſtárśi Izráelſcy / áby jedli chleb z świekrem Mojżeƺowym przed Bogiem.
Y ſtáło śię ná zájutrz / że uśiádł Mojżeƺ / áby ſądźił lud / y ſtáł lud przed Mojżeƺem od poránku áż do wiecżorá.
A widząc świekier Mojżeƺá wƺyſtko / co on cżynił z ludem / rzekł : Cóż to jeſt / co ty cżyniƺ z ludem? cżemuż ty ſám śiedźiƺ / á lud wƺyſtek ſtoi przed tobą od poránku áż do wiecżorá?
Tedy Mojżeƺ odpowiedźiáł świekrowi ſwemu : Iż przychodźi lud do mnie / áby śię rádźił Bogá.
Gdy ſpráwę jáką máją / przychodzą do mnie / á rozſądzám między nimi / oznájmując uſtáwy Boże y práwá jego.
Zátem rzekł świekier Mojżeƺá do niego : Nie dobrá rzecż / którą ty cżyniƺ.
Pewnie uſtánieƺ / y ty y lud ten / który jeſt z tobą / bo ćiężƺá to rzecż nád śiły twoje ; nie będźieƺ jey mógł ty ſám podołáć.
Przetoż uſłucháj teraz głoſu mego / porádzęć / á będźie Bóg z tobą ; ſtój ty zá lud przed Bogiem / á odnoś ſpráwy do Bogá ;
A onych też náucżáj uſtáw y práw / oznájmując im drogę / którąby chodźić / y dźieło / któreby cżynić mieli.
Ty też upátrz ze wƺyſtkiego ludu męże ſtátecżne / bojące śię Bogá / męże práwdomówne / którzyby nienáwidźili łákomſtwá / á poſtánów z nich przełożone / tyśiącżniki / ſetniki / pięćdźieśiątniki y dźieśiątniki.
Którzy ná káżdy cżáſ lud ſądźić będą ; á gdy będźie rzecż wielká / odnioſą do ćiebie / á káżdą rzecż máłą ſądźić będą ſámi ; tedy ulżyƺ ſobie / gdy ponioſą ćiężár z tobą.
To jeſli ucżyniƺ / á rozkáżeć Bóg / oſtoiƺ śię / y ten wƺyſtek lud ná miejſcá ſwoje wrácáć śię będźie w pokoju.
Y uſłucháł Mojżeƺ rády świekrá ſwojego / á ucżynił wƺyſtko / jáko mu powiedźiáł.
Y wybráł Mojżeƺ męże ſtátecżne ze wƺyſtkiego Izráelá / y poſtánowił je przełożonymi nád ludem / tyśiącżniki / ſetniki / pięćdźieśiątniki y dźieśiątniki.
Którzy ſądźili lud káżdego cżáſu ; trudne rzecży odnośili do Mojżeƺá / á káżdą rzecż mniejƺą ſámi ſądźili.
Zátem puśćił od śiebie Mojżeƺ świekrá ſwego / który odƺedł do źiemi ſwej.
