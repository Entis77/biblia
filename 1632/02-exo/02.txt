Y wyƺedł mąż niektóry z domu Lewiego ; który pojął córkę z pokoleniá Lewiego.
Y pocżęłá oná niewiáſtá / á porodziłá ſyná ; á widząc go / że był nádobny / kryłá go przez trzy mieśiące.
Ale gdy go nie mogłá dłużej zátáić / wzięłá plećionkę z śitowiá y obmázáłá ją klejem y ſmołą ; á włożywƺy w nię ono dziećię / położyłá je między rogóż ná brzegu rzeki.
A ſtáłá śioſtrá jego z dáleká / áby wiedziáłá / co śię z nim dziáć będzie.
Wtem wyƺłá córká Fáráonowá / áby śię kąpáłá w rzece / á pánny jej przechádzáły śię po brzegu rzeki ; y ujrzáłá plećionkę w rogoży / y poſłáłá ſłużebnicę ſwą / áby ją wzięłá.
A otworzywƺy ujrzáłá dziećię / á ono chłopiątko płákáło ; á użáliwƺy śię go / rzekłá : Z dziátek Hebrejſkich jeſt ten.
Y rzekłá śioſtrá jego do córki Fáráonowej : Chceƺże / pójdę / y záwołám ći niewiáſty / mámki Hebrejſkiej / którá być odchowáłá to dziećię?
Y rzekłá jej córká Fáráonowá : Idź. Tedy poƺłá oná dziewecżká / y záwołáłá mátki onegoż dziećięćiá.
Do której rzekłá córká Fáráonowá : Weźmij to dziećię / y chowáj mi je / á já tobie dám zápłátę twoję ; y wziąwƺy niewiáſtá dziećię / chowáłá je.
A gdy podroſło ono dziećię / przywiodłá je do córki Fáráonowej / y było jej zá ſyná ; á názwáłá imię jego Mojżeƺ / bo mówiłá : Żem z wody wyćiągnęłá go.
Y ſtáło śię zá onych dni / gdy uróſł Mojżeƺ / że wyƺedł do bráći ſwej / y widziáł ćiężáry ich ; obácżył też mężá Egipcżániná / który bił jednego Hebrejcżyká z bráći jego.
A obejrzáwƺy śię tám y ſám / gdy widziáł / że nikogo nie máƺ / zábił Egipcżániná / y zágrzebł go w piáſek.
A wyƺedłƺy záś dniá wtórego / ujrzáł / á oto / dwáj mężowie Hebrejſcy wádzili śię ; y rzekł onemu / który krzywdę cżynił :
Cżemuż bijeƺ bliźniego ſwego? Ale mu odpowiedziáł : Któż ćię poſtánowił kśiążęćiem y ſędzią nád námi? álbo mię ty myśliƺ zábić / jákoś zábił Egipcżániná? y zląkł śię Mojżeƺ y rzekł :
Pewnie śię tá rzecż wyjáwiłá. Uſłyƺáł tedy Fáráo tę rzecż / y ƺukáł zábić Mojżeƺá. Lecż Mojżeƺ ućiekł od twárzy Fáráonowej / y mieƺkáł w ziemi Mádyjáńſkiej ; á przyƺedłƺy tám śiedziáł u ſtudni.
A kápłán Mádyjáńſki miáł śiedm córek / które wyƺedłƺy cżerpáły wodę / y nálewáły do koryt / áby nápoiły trzodę ojcá ſwego ;
A przyƺedłƺy páſterze odgániáli je. Tedy wſtáwƺy Mojżeƺ obronił ich / y nápoił bydło ich.
A gdy śię wróćiły do Ráguelá / ojcá ſwego / rzekł : Cżemuśćie dziś ták prędko przyƺły?
Y odpowiedziáły : Mąż Egipſki obronił náſ od ręki páſterzów ; nádto cżerpájąc nácżerpáł nám y wody y nápoił trzody.
Zátem rzekł do córek ſwych : A gdzież ten jeſt? Cżemuśćie opuśćiły cżłowieká tego? Záwołájćie go / áby jádł chleb.
Y przyzwolił Mojżeƺ mieƺkáć z onym mężem / który dáł Zeforę / córkę ſwoję / Mojżeƺowi.
Y urodziłá ſyná / á názwáł imię jego Gerſon / bo mówił : Byłem przychodniem w ziemi cudzej.
Y ſtáło śię po niemáłym cżáśie / że umárł król Egipſki ; y wzdycháli / y wołáli ſynowie Izráelſcy dlá niewoli ; á wſtąpiło wołánie ich do Bogá przed niewolą.
Y uſłyƺáł Bóg wołánie ich ; y wſpomniáł Bóg ná przymierze ſwoje z Abráhámem / z Izáákiem / y z Jákóbem.
Y wejrzáł Bóg ná ſyny Izráelſkie / y poznáł Bóg.
