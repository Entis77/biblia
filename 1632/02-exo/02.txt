Y wyƺedł mąż niektóry z domu Lewiego ; który pojął córkę z pokoleniá Lewiego.
Y pocżęłá oná niewiáſtá / á porodźiłá ſyná ; á widząc go / że był nádobny / kryłá go przez trzy mieśiące.
Ale gdy go nie mogłá dłużej zátáić / wźięłá plećionkę z śitowiá y obmázáłá ją klejem y ſmołą ; á włożywƺy w nię ono dźiećię / położyłá je między rogóż ná brzegu rzeki.
A ſtáłá śioſtrá jego z dáleká / áby wiedźiáłá / co śię z nim dźiáć będźie.
Wtem wyƺłá córká Fáráonowá / áby śię kąpáłá w rzece / á pánny jey przechádzáły śię po brzegu rzeki ; y ujrzáłá plećionkę w rogoży / y poſłáłá ſłużebnicę ſwą / áby ją wźięłá.
A otworzywƺy ujrzáłá dźiećię / á ono chłopiątko płákáło ; á użáliwƺy śię go / rzekłá : Z dźiátek Hebrejſkich jeſt ten.
Y rzekłá śioſtrá jego do córki Fáráonowej : Chceƺże / pójdę / y záwołám ći niewiáſty / mámki Hebrejſkiej / która być odchowáłá to dźiećię?
Y rzekłá jey córká Fáráonowá : Idź. Tedy poƺłá oná dźiewecżká / y záwołáłá mátki onegoż dźiećięćiá.
Do którey rzekłá córká Fáráonowá : Weźmij to dźiećię / y chowáj mi je / á ja tobie dám zápłátę twoję ; y wźiąwƺy niewiáſtá dźiećię / chowáłá je.
A gdy podroſło ono dźiećię / przywiodłá je do córki Fáráonowej / y było jey zá ſyná ; á názwáłá imię jego Mojżeƺ / bo mówiłá : Żem z wody wyćiągnęłá go.
Y ſtáło śię zá onych dni / gdy uróſł Mojżeƺ / że wyƺedł do bráći ſwej / y widźiáł ćiężáry ich ; obácżył też mężá Egipcżániná / który bił jednego Hebrejcżyká z bráći jego.
A obejrzáwƺy śię tám y ſám / gdy widźiáł / że nikogo nie máƺ / zábił Egipcżániná / y zágrzebł go w piáſek.
A wyƺedłƺy záś dniá wtórego / ujrzáł / á oto / dwáj mężowie Hebrejſcy wádźili śię ; y rzekł onemu / który krzywdę cżynił :
Cżemuż bijeƺ bliźniego ſwego? Ale mu odpowiedźiáł : Któż ćię poſtánowił kśiążęćiem y ſędźią nád námi? álbo mię ty myśliƺ zábić / jákoś zábił Egipcżániná? y zląkł śię Mojżeƺ y rzekł :
Pewnie śię tá rzecż wyjáwiłá. Uſłyƺáł tedy Fáráo tę rzecż / y ƺukáł zábić Mojżeƺá. Lecż Mojżeƺ ućiekł od twárzy Fáráonowej / y mieƺkáł w źiemi Mádyjáńſkiej ; á przyƺedłƺy tám śiedźiáł u ſtudni.
A kápłán Mádyjáńſki miał śiedm córek / które wyƺedłƺy cżerpáły wodę / y nálewáły do koryt / áby nápoiły trzodę ojcá ſwego ;
A przyƺedłƺy páſterze odgániáli je. Tedy wſtáwƺy Mojżeƺ obronił ich / y nápoił bydło ich.
A gdy śię wróćiły do Ráguelá / ojcá ſwego / rzekł : Cżemuśćie dźiś ták prędko przyƺły?
Y odpowiedźiáły : Mąż Egipſki obronił nas od ręki páſterzów ; nádto cżerpájąc nácżerpáł nám y wody y nápoił trzody.
Zátem rzekł do córek ſwych : A gdźież ten jeſt? Cżemuśćie opuśćiły cżłowieká tego? Záwołájćie go / áby jádł chleb.
Y przyzwolił Mojżeƺ mieƺkáć z onym mężem / który dáł Zeforę / córkę ſwoję / Mojżeƺowi.
Y urodźiłá ſyná / á názwáł imię jego Gerſon / bo mówił : Byłem przychodniem w źiemi cudzej.
Y ſtáło śię po niemáłym cżáśie / że umárł król Egipſki ; y wzdycháli / y wołáli ſynowie Izráelſcy dla niewoli ; á wſtąpiło wołánie ich do Bogá przed niewolą.
Y uſłyƺáł Bóg wołánie ich ; y wſpomniáł Bóg ná przymierze ſwoje z Abráhámem / z Izáákiem / y z Jákóbem.
Y wejrzáł Bóg ná ſyny Izráelſkie / y poznał Bóg.
