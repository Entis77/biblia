Potym mówił PAN do Mojzeƺá : Idź / ruƺ śię z tąd / ty y lud / któryś wywiódł z źiemie Egipſkiey do źiemie / o którąm przyśiągł Abráhámowi / Izáákowi / y Jákóbowi / mówiąc : Naśieniu twemu dam ją.
( Y poślę przed tobą Anjołá / y wyrzucę Chánánejcżyká / Amorrejcżyká / y Hettejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká. )
Do źiemie opływájącey mlekiem y miodem : lecz <i>ſam</i> nie pójdę z tobą / gdyżeś <i>jeſt</i> lud kárku twárdego / bym ćię ſnadź nie wytráćił w drodze.
A uſłyƺawƺy lud tę rzecz złą / záſmućił śię / y nie włożył żaden ochędóſtwá ſwego ná śię :
Abowiem rzekł PAN do Mojzeƺá : Powiedz Synom Izráelſkim / Wyśćie ludem twárdego kárku : przyjdę kiedy znagłá w pośrzód ćiebie / y wygłádzę ćię : Przetoż teraz złóż ochędóſtwo twoje z śiebie / á będę wiedźiał / coćbym uczynić miał.
Y złożyli Synowie Izráelſcy ochędóſtwo ſwoje / przy górze Horeb.
A Mojzeƺ wźiąwƺy Namiot rozbił <i>go</i> ſobie zá Obozem / opodal od Obozu : y názwał go Namiotem zgromádzenia. Tedy káżdy który chćiał o co pytáć PANA / wychodźił do Namiotu zgromádzenia / który był zá Obozem.
A gdy wychodźił Mojzeƺ do Namiotu / powſtawał wƺyſtek lud / y ſtał káżdy we drzwiách Namiotu ſwego / y pátrzáli zá Mojzeƺem / áż wƺedł do Namiotu.
Y bywáło to / że gdy wchadzał Mojzeƺ do Namiotu : z ſtępował ſłup Obłokowy / á ſtawał u drzwi Namiotu : y mawiał z Mojzeƺem.
A widząc wƺyſtek lud ſłup Obłokowy / ſtojący u drzwi Namiotu powſtawał wƺyſtek lud / y kłániał śię káżdy u drzwi Namiotu ſwego.
Y mawiał PAN do Mojzeƺá / twarzą w twarz / jáko mawia cżłowiek do przyjaćielá ſwego : potym wracał śię do Obozu / á ſługá jego Jozue Syn Nunów / młodźieńiec / nie odchodźił z pośrzodku Namiotu.
Tedy mówił Mojzeƺ do PANA : Wey / ty mi mówiƺ / prowadź lud ten : á tyś mi nie oznájmił / kogo pośleƺ ze mną? Nád to powiedźiałeś : Znam ćię z imienia / ználazłeś też łáſkę w oczách mojich /
Teraz tedy / jeslim ználazł łáſkę w oczách twojich / ukaż mi proƺę drogę twoję / żebym ćię poznał / y żebym ználazł łáſkę w oczách twojich / á obacz / że ludem twojim jeſt naród ten.
Y odpowiedźiał <i>PAN</i> : Oblicze moje pójdźie przed tobą / á damći odpoczynienie :
Y rzekł <i>Mojzeƺ</i> do niego : niepójdźieli oblicże twoje <i>z námi,</i> nie wywódź nas z tąd.
Abowiem / po czymże tu znáć będźie / żem ználazł łáſkę w ocżách twojich / ja y lud twój? izali <i>nie po tym</i> gdy pójdźieƺ z námi? bo ták oddźieleni będźiemy / ja y lud twój / od káżdego ludu / który jeſt ná źiemi.
Y rzekł PAN do Mojzeƺá : y tę rzecz / o którąś mówił / uczynię / boś ználazł łáſkę w ocżách mojich / y znam ćię z imienia.
Nád to rzekł <i>Mojzeƺ</i> : Ukaż mi / proƺę / chwałę twoję.
A <i>on</i> odpowiedźiał : Ja ſpráwię / że przejdźie wƺyſtko dobre moje przed twarzą twoją / y záwołam z imienia / PAN przed twarzą twoją : zmiłuję śię / nád kim śię zmiłuję / á zlituję śię / nád kim śię zlituję :
Y rzekł : Nie będźieƺ mógł widźieć oblicża mego / bo nie ujrzy mię cżłowiek / áby żyw zoſtał.
Y rzekł PAN : Oto miejſce u mnie / á ſtánieƺ ná opoce.
A gdy przechodźić będźie chwałá mojá / tedy ćię poſtáwię w roſpádlinie opoki / y zákryję ćię dłonią moją / póki nie przejdę.
Potym odejmę dłoń moję / y ujrzyƺ tył mój / ále twarz mojá nie będźie widźiána.
