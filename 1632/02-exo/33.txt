Potem mówił Pán do Mojżeƺá : Idź / ruƺ śię ſtąd / ty y lud / któryś wywiódł z źiemi Egipſkiej / do źiemi / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : Náśieniu twemu dam ją.
Y poślę przed tobą Aniołá / y wyrzucę Chánánejcżyká / Amorejcżyká / y Hetejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká.
Do źiemi opływájącey mlekiem y miodem ; lecż ſám nie pójdę z tobą / gdyżeś jeſt lud kárku twárdego / bym ćię ſnadź nie wytráćił w drodze.
A uſłyƺáwƺy lud tę rzecż złą / záſmućił śię / y nie włożył żáden ochędóſtwá ſwego ná śię.
Abowiem rzekł Pán do Mojżeƺá : Powiedz ſynom Izráelſkim : Wyśćie ludem twárdego kárku ; przydę kiedy z nágłá w pośród ćiebie / y wygłádzę ćię. Przetoż teraz złóż ochędóſtwo twoje z śiebie / á będę wiedźiał / coćbym ucżynić miał.
Y złożyli ſynowie Izráelſcy ochędóſtwo ſwoje przy górze Horeb.
A Mojzeƺ wźiąwƺy námiot / rozbił go ſobie zá obozem / opodál od obozu / y názwał go námiotem zgromádzenia. Tedy káżdy / który chćiał o co pytáć Páná / wychodźił do namiotu zgromádzenia / który był zá obozem.
A gdy wychodźił Mojzeƺ do namiotu / powſtáwał wƺyſtek lud / y ſtał káżdy we drzwiách namiotu ſwego ; y pátrzáli zá Mojżeƺem / áż wƺedł do namiotu.
Y bywáło to / że gdy wchádzał Mojzeƺ do namiotu / zſtępował ſłup obłokowy / á ſtáwał u drzwi namiotu / y máwiał Bóg z Mojżeƺem.
A widząc wƺyſtek lud ſłup obłokowy / ſtojący u drzwi namiotu / powſtáwał wƺyſtek lud y kłániał śię káżdy u drzwi namiotu ſwego.
Y máwiał Pán do Mojżeƺá twárzą w twárz / jáko máwiá cżłowiek do przyjáćielá ſwego ; potem wrácał śię do obozu / á ſługá jego Jozue / ſyn Nunów / młodźieniec / nie odchodźił z pośrzodku namiotu.
Tedy mówił Mojzeƺ do Páná : Wej / ty mi mówiƺ : Prowádź lud ten / á tyś mi nie oznájmił / kogo pośleƺ ze mną? Nád to powiedźiáłeś : Znam ćię z imienia / ználázłeś też łáſkę w ocżách mojich.
Teraz tedy / jeſlim ználázł łáſkę w ocżách twojich / ukáż mi proƺę drogę twoję / żebym ćię poznał / y żebym ználázł łáſkę w ocżách twojich / á obácż / że ludem twoim jeſt naród ten.
Y odpowiedźiał Pán : Oblicże moje pójdźie przed tobą / á dam ći odpocżynienie.
Y rzekł Mojzeƺ do niego : Nie pójdźieli oblicże twoje z námi / nie wywódź nas ſtąd.
Abowiem po cżemże tu znáć będźie / żem ználázł łáſkę w ocżách twojich / ja y lud twój? izali nie po tym / gdy pójdźieƺ z námi? bo ták oddźieleni będźiemy / ja y lud twój / od káżdego ludu / który jeſt ná źiemi.
Y rzekł Pán do Mojżeƺá : Y tę rzecż / o którąś mówił / ucżynię ; boś ználázł łáſkę w ocżách mojich / y znam ćię z imienia.
Nád to rzekł Mojzeƺ : Ukáż mi proƺę / chwałę twoję.
A on odpowiedźiał : Ja ſpráwię / że przejdźie wƺyſtko dobre moje przed twárzą twoją / y záwołám z imienia : Pán przed twárzą twoją ; zmiłuję śię / nád kim śię zmiłuję ; á zlituję śię / nád kim śię zlituję.
Y rzekł : Nie będźieƺ mógł widźieć oblicża mego ; bo nie ujrzy mię cżłowiek / áby żyw zoſtał.
Y rzekł Pán : Oto / miejſce u mnie / á ſtánieƺ ná opoce.
A gdy przechodźić będźie chwałá mojá / tedy ćię poſtáwię w rozpádlinie opoki / y zákryję ćię dłonią moją / póki nie przejdę.
Potem odejmę dłoń moję / y ujrzyƺ tył mój ; ále twárz mojá nie będźie widźiáná.
