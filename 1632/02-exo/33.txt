Potem mówił Pán do Mojżeƺá : Idź / ruƺ śię ſtąd / ty y lud / któryś wywiódł z źiemi Egipſkiej / do źiemi / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : Náśieniu twemu dám ją.
Y poślę przed tobą Aniołá / y wyrzucę Chánánejcżyká / Amorejcżyká / y Hetejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká.
Do źiemi opływájącey mlekiem y miodem ; lecż ſám nie pójdę z tobą / gdyżeś jeſt lud kárku twárdego / bym ćię ſnáć nie wytráćił w drodze.
A uſłyƺáwƺy lud tę rzecż złą / záſmućił śię / y nie włożył żáden ochędóſtwá ſwego ná śię.
Abowiem rzekł Pán do Mojżeƺá : Powiedz ſynom Izráelſkim : Wyśćie ludem twárdego kárku ; przyjdę kiedy z nágłá w pośród ćiebie / y wygłádzę ćię. Przetoż teraz złóż ochędóſtwo twoje z śiebie / á będę wiedźiáł / coćbym ucżynić miał.
Y złożyli ſynowie Izráelſcy ochędóſtwo ſwoje przy górze Horeb.
A Mojżeƺ wźiąwƺy námiot / rozbił go ſobie zá obozem / opodál od obozu / y názwáł go námiotem zgromádzeniá. Tedy káżdy / który chćiáł o co pytáć Páná / wychodźił do námiotu zgromádzeniá / który był zá obozem.
A gdy wychodźił Mojżeƺ do námiotu / powſtáwáł wƺyſtek lud / y ſtáł káżdy we drzwiách námiotu ſwego ; y pátrzáli zá Mojżeƺem / áż wƺedł do námiotu.
Y bywáło to / że gdy wchádzáł Mojżeƺ do námiotu / zſtępowáł ſłup obłokowy / á ſtáwáł u drzwi námiotu / y máwiáł Bóg z Mojżeƺem.
A widząc wƺyſtek lud ſłup obłokowy / ſtojący u drzwi námiotu / powſtáwáł wƺyſtek lud y kłániáł śię káżdy u drzwi námiotu ſwego.
Y máwiáł Pán do Mojżeƺá twárzą w twárz / jáko máwiá cżłowiek do przyjáćielá ſwego ; potem wrácáł śię do obozu / á ſługá jego Jozue / ſyn Nunów / młodźieniec / nie odchodźił z pośrodku námiotu.
Tedy mówił Mojżeƺ do Páná : Wej / ty mi mówiƺ : Prowádź lud ten / á tyś mi nie oznájmił / kogo pośleƺ ze mną? Nád to powiedźiáłeś : Znám ćię z imieniá / ználázłeś też łáſkę w ocżách moich.
Teraz tedy / jeſlim ználázł łáſkę w ocżách twoich / ukáż mi proƺę drogę twoję / żebym ćię poznał / y żebym ználázł łáſkę w ocżách twoich / á obácż / że ludem twoim jeſt náród ten.
Y odpowiedźiáł Pán : Oblicże moje pójdźie przed tobą / á dám ći odpocżynienie.
Y rzekł Mojżeƺ do niego : Nie pójdźieli oblicże twoje z námi / nie wywódź nas ſtąd.
Abowiem po cżemże tu znáć będźie / żem ználázł łáſkę w ocżách twoich / ja y lud twój? izáli nie po tem / gdy pójdźieƺ z námi? bo ták oddźieleni będźiemy / ja y lud twój / od káżdego ludu / który jeſt ná źiemi.
Y rzekł Pán do Mojżeƺá : Y tę rzecż / o którąś mówił / ucżynię ; boś ználázł łáſkę w ocżách moich / y znám ćię z imieniá.
Nád to rzekł Mojżeƺ : Ukáż mi proƺę / chwáłę twoję.
A on odpowiedźiáł : Ja ſpráwię / że przejdźie wƺyſtko dobre moje przed twárzą twoją / y záwołám z imieniá : Pán przed twárzą twoją ; zmiłuję śię / nád kim śię zmiłuję ; á zlituję śię / nád kim śię zlituję.
Y rzekł : Nie będźieƺ mógł widźieć oblicżá mego ; bo nie ujrzy mię cżłowiek / áby żyw zoſtáł.
Y rzekł Pán : Oto / miejſce u mnie / á ſtánieƺ ná opoce.
A gdy przechodźić będźie chwałá mojá / tedy ćię poſtáwię w rozpádlinie opoki / y zákryję ćię dłonią moją / póki nie przejdę.
Potem odejmę dłoń moję / y ujrzyƺ tył mój ; ále twárz mojá nie będźie widźiáná.
