Potem mówił Pán do Mojżeƺá : Idź / ruƺ śię ſtąd / ty y lud / któryś wywiódł z ziemi Egipſkiej / do ziemi / o którąm przyśiągł Abráhámowi / Izáákowi y Jákóbowi / mówiąc : Náśieniu twemu dám ją.
Y poślę przed tobą Aniołá / y wyrzucę Chánánejcżyká / Amorejcżyká / y Hetejcżyká / y Ferezejcżyká / Hewejcżyká / y Jebuzejcżyká.
Do ziemi opływájącej mlekiem y miodem ; lecż ſám nie pójdę z tobą / gdyżeś jeſt lud kárku twárdego / bym ćię ſnáć nie wytráćił w drodze.
A uſłyƺáwƺy lud tę rzecż złą / záſmućił śię / y nie włożył żáden ochędóſtwá ſwego ná śię.
Abowiem rzekł Pán do Mojżeƺá : Powiedz ſynom Izráelſkim : Wyśćie ludem twárdego kárku ; przyjdę kiedy z nágłá w pośród ćiebie / y wygłádzę ćię. Przetoż teráz złóż ochędóſtwo twoje z śiebie / á będę wiedziáł / coćbym ucżynić miáł.
Y złożyli ſynowie Izráelſcy ochędóſtwo ſwoje przy górze Horeb.
A Mojżeƺ wziąwƺy námiot / rozbił go ſobie zá obozem / opodál od obozu / y názwáł go námiotem zgromádzeniá. Tedy káżdy / który chćiáł o co pytáć Páná / wychodził do námiotu zgromádzeniá / który był zá obozem.
A gdy wychodził Mojżeƺ do námiotu / powſtáwáł wƺyſtek lud / y ſtáł káżdy we drzwiách námiotu ſwego ; y pátrzáli zá Mojżeƺem / áż wƺedł do námiotu.
Y bywáło to / że gdy wchádzáł Mojżeƺ do námiotu / zſtępowáł ſłup obłokowy / á ſtáwáł u drzwi námiotu / y máwiáł Bóg z Mojżeƺem.
A widząc wƺyſtek lud ſłup obłokowy / ſtojący u drzwi námiotu / powſtáwáł wƺyſtek lud y kłániáł śię káżdy u drzwi námiotu ſwego.
Y máwiáł Pán do Mojżeƺá twárzą w twárz / jáko máwiá cżłowiek do przyjáćielá ſwego ; potem wrácáł śię do obozu / á ſługá jego Jozue / ſyn Nunów / młodzieniec / nie odchodził z pośrodku námiotu.
Tedy mówił Mojżeƺ do Páná : Wej / ty mi mówiƺ : Prowádź lud ten / á tyś mi nie oznájmił / kogo pośleƺ ze mną? Nád to powiedziáłeś : Znám ćię z imieniá / ználázłeś też łáſkę w ocżách moich.
Teráz tedy / jeźlim ználázł łáſkę w ocżách twoich / ukáż mi proƺę drogę twoję / żebym ćię poznáł / y żebym ználázł łáſkę w ocżách twoich / á obácż / że ludem twoim jeſt náród ten.
Y odpowiedziáł Pán : Oblicże moje pójdzie przed tobą / á dám ći odpocżynienie.
Y rzekł Mojżeƺ do niego : Nie pójdzieli oblicże twoje z námi / nie wywódź náſ ſtąd.
Abowiem po cżemże tu znáć będzie / żem ználázł łáſkę w ocżách twoich / já y lud twój? izáli nie po tem / gdy pójdzieƺ z námi? bo ták oddzieleni będziemy / já y lud twój / od káżdego ludu / który jeſt ná ziemi.
Y rzekł Pán do Mojżeƺá : Y tę rzecż / o którąś mówił / ucżynię ; boś ználázł łáſkę w ocżách moich / y znám ćię z imieniá.
Nád to rzekł Mojżeƺ : Ukáż mi proƺę / chwáłę twoję.
A on odpowiedziáł : Já ſpráwię / że przejdzie wƺyſtko dobre moje przed twárzą twoją / y záwołám z imieniá : Pán przed twárzą twoją ; zmiłuję śię / nád kim śię zmiłuję ; á zlituję śię / nád kim śię zlituję.
Y rzekł : Nie będzieƺ mógł widzieć oblicżá mego ; bo nie ujrzy mię cżłowiek / áby żyw zoſtáł.
Y rzekł Pán : Oto / miejſce u mnie / á ſtánieƺ ná opoce.
A gdy przechodzić będzie chwáłá mojá / tedy ćię poſtáwię w rozpádlinie opoki / y zákryję ćię dłonią moją / póki nie przejdę.
Potem odejmę dłoń moję / y ujrzyƺ tył mój ; ále twárz mojá nie będzie widziáná.
