A Mojżeƺ páſł trzodę Jetrá / świekrá ſwego / kápłáná Mádyjáńſkiego ; y zágnáł trzodę ná puƺcżą / á przyƺedł do góry Bożej / do Horeb.
Y ukázáł mu śię Anioł Páńſki w płomieniu ogniſtym / z pośrodku krzá ; y widziáł / á oto / kierz gorzáł ogniem / á on kierz nie zgorzáł.
Tedy rzekł Mojżeƺ : Pójdę teráz / á oglądám to widzenie wielkie / cżemu nie zgore ten kierz.
A widząc Pán / iż ƺedł pátrzáć / záwołáł náń Bóg z pośrodku onego krzá / mówiąc : Mojżeƺu / Mojżeƺu! A on odpowiedziáł : Otom já.
Tedy rzekł : Nie przyſtępuj ſám ; zzuj bóty twe z nóg twoich ; ábowiem miejſce / ná którem ty ſtoiƺ / ziemiá świętá jeſt.
Zátem rzekł : Jám jeſt Bóg ojcá twego / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów ; y zákrył Mojżeƺ oblicże ſwe / bo śię báł pátrzáć ná Bogá.
Y rzekł Pán : Widząc widziáłem utrápienie ludu mojego / który jeſt w Egipćie / á wołánie ich ſłyƺáłem przed przyſtáwy ich ; bom doznáł boleśći jego.
Przetoż zſtąpiłem / ábym go wybáwił z ręki Egipſkiej y wywiódł go z ziemi tej do ziemi dobrej y przeſtronnej / do ziemi opływájącej mlekiem y miodem / ná miejſce Chánánejcżyká / y Hetejcżyká / y Amorejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká.
A teráz oto krzyk ſynów Izráelſkich przyƺedł przed mię : y widziáłem też ich ućiſk / którym je Egipcżánie ućiſkáją.
Przetoż teráz / pójdź / á poślę ćię do Fáráoná / ábyś wywiódł lud mój / ſyny Izráelſkie z Egiptu.
Y rzekł Mojżeƺ do Bogá : Któżem já / ábym ƺedł do Fáráoná / á wywiódł ſyny Izráelſkie z Egiptu?
Y rzekł Bóg : Oto / będę z tobą / á to miej ná znák / żem ćię já poſłáł : Gdy wywiedzieƺ ten lud z Egiptu / będziećie ſłużyć Bogu ná tej górze.
Y rzekł Mojżeƺ do Bogá : Oto / já pójdę do ſynów Izráelſkich / y rzekę im : Bóg ojców wáƺych poſłáł mię do wáſ : Jeźli mi rzeką : Które jeſt imię jego? Cóż im odpowiem?
Tedy rzekł Bóg do Mojżeƺá : Będę który Będę. Y rzekł : Ták powieƺ ſynom Izráelſkim : Będę poſłáł mię do wáſ.
Y mówił jeƺcże Bóg do Mojżeƺá : Ták rzecżeƺ do ſynów Izráelſkich : Pán / Bóg ojców wáƺych / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów poſłáł mię do wáſ ; toć imię moje ná wieki / y to pámiętne moje od národu do národu ;
Idźże / á zgromádź ſtárƺe Izráelſkie / y mów do nich : Pán / Bóg ojców wáƺych / ukázáł mi śię / Bóg Abráhámów / Izááków y Jákóbów / mówiąc : Wſpominájąc wſpomniáłem ná wáſ / y widziáłem / co śię wám dziáło w Egipćie.
Y rzekłem : Wyprowádzę wáſ z utrápieniá Egipſkiego do ziemi Chánánejcżyká / y Hetejcżyká / y Amorejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká / do ziemi opływájącej mlekiem y miodem.
Tedy uſłucháją głoſu twego / á pójdzieƺ ty y ſtárśi Izráelſcy do królá Egipſkiego / y rzecżećie do niego : Pán / Bóg Hebrejcżyków / zábieżáł nám ; przetoż teráz niech pójdziemy proƺę w drogę trzech dni ná puƺcżą / ábyśmy ofiárowáli Pánu / Bogu náƺemu.
Ale já wiem / że wám nie pozwoli król Egipſki odejść / jedno przez możną rękę.
A ták wyćiągnę rękę moję / y uderzę Egipt wƺyſtkiemi cudámi mojemi / które będę cżynił w pośrodku jego : á potem wypuśći wáſ.
Y dám łáſkę ludowi temu w ocżách Egipcżánów : y ſtánie śię / gdy wychodzić będziećie / że nie wynijdziećie próżni.
Ale wypożycży niewiáſtá u ſąśiádki ſwojej / y u goſpodyni domu ſwego / nácżyniá ſrebrnego / y nácżyniá złotego / y ƺát ; y włożyćie je ná ſyny wáƺe / y ná córki wáƺe / y złupićie Egipt.
