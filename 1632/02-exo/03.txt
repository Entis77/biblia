A Mojzeƺ páſł trzodę Jetrá / świekrá ſwego / kápłáná Mádyjáńſkiego ; y zágnał trzodę ná puƺcżą / á przyƺedł do góry Bożey / do Horeb.
Y ukázał mu śię Anioł Páńſki w płomieniu ogniſtym / z pośrzodku krzá ; y widźiał / á oto / kierz gorzał ogniem / á on kierz nie zgorzał.
Tedy rzekł Mojzeƺ : Pójdę teraz / á oglądám to widzenie wielkie / cżemu nie zgore ten kierz.
A widząc Pán / iż ƺedł pátrzáć / záwołał nań Bóg z pośrzodku onego krzá / mówiąc : Mojżeƺu / Mojżeƺu! A on odpowiedźiał : Otom ja.
Tedy rzekł : Nie przyſtępuj ſám ; zzuj bóty twe z nóg twojich ; ábowiem miejſce / ná którem ty ſtoiƺ / źiemiá świętá jeſt.
Zátem rzekł : Jam jeſt Bóg ojcá twego / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów ; y zákrył Mojzeƺ oblicże ſwe / bo śię bał pátrzáć ná Bogá.
Y rzekł Pán : Widząc widźiáłem utrápienie ludu mojego / który jeſt w Egipćie / á wołánie ich ſłyƺáłem przed przyſtáwy ich ; bom doznał boleśći jego.
Przetoż zſtąpiłem / ábym go wybáwił z ręki Egipſkiej y wywiódł go z źiemi tey do źiemi dobrej y przeſtronney / do źiemi opływájącey mlekiem y miodem / ná miejſce Chánánejcżyká / y Hetejcżyká / y Amorejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká.
A teraz oto krzyk ſynów Izráelſkich przyƺedł przed mię : y widźiáłem też ich ućiſk / którym je Egipcżánie ućiſkáją.
Przetoż teraz / pójdź / á poślę ćię do Fáráoná / ábyś wywiódł lud mój / ſyny Izráelſkie z Egiptu.
Y rzekł Mojzeƺ do Bogá : Któżem ja / ábym ƺedł do Fáráoná / á wywiódł ſyny Izráelſkie z Egiptu?
Y rzekł Bóg : Oto / będę z tobą / á to miej ná znák / żem ćię ja poſłał : Gdy wywiedźieƺ ten lud z Egiptu / będźiećie ſłużyć Bogu ná tey górze.
Y rzekł Mojzeƺ do Bogá : Oto / ja pójdę do ſynów Izráelſkich / y rzekę im : Bóg ojców wáƺych poſłał mię do was : Jeſli mi rzeką : Które jeſt imię jego? Cóż im odpowiem?
Tedy rzekł Bóg do Mojżeƺá : Będę który Będę. Y rzekł : Ták powieƺ ſynom Izráelſkim : Będę poſłał mię do was.
Y mówił jeƺcże Bóg do Mojżeƺá : Ták rzecżeƺ do ſynów Izráelſkich : Pán / Bóg ojców wáƺych / Bóg Abráhámów / Bóg Izááków / y Bóg Jákóbów poſłał mię do was ; toć imię moje ná wieki / y to pámiętne moje od národu do národu ;
Idźże / á zgromádź ſtárƺe Izráelſkie / y mów do nich : Pán / Bóg ojców wáƺych / ukázał mi śię / Bóg Abráhámów / Izááków y Jákóbów / mówiąc : Wſpominájąc wſpomniáłem ná was / y widźiáłem / co śię wam dźiáło w Egipćie.
Y rzekłem : Wyprowádzę was z utrapienia Egipſkiego do źiemi Chánánejcżyká / y Hetejcżyká / y Amorejcżyká / y Ferezejcżyká / y Hewejcżyká / y Jebuzejcżyká / do źiemi opływájącey mlekiem y miodem.
Tedy uſłucháją głoſu twego / á pójdźieƺ ty y ſtárśi Izráelſcy do królá Egipſkiego / y rzecżećie do niego : Pán / Bóg Hebrejcżyków / zábieżał nam ; przetoż teraz niech pójdźiemy proƺę w drogę trzech dni ná puƺcżą / ábyſmy ofiárowáli Pánu / Bogu náƺemu.
Ale ja wiem / że wam nie pozwoli król Egipſki odejść / jedno przez możną rękę.
A ták wyćiągnę rękę moję / y uderzę Egipt wƺyſtkiemi cudámi mojemi / które będę cżynił w pośrzodku jego : á potem wypuśći was.
Y dam łáſkę ludowi temu w ocżách Egipcżánów : y ſtánie śię / gdy wychodźić będźiećie / że nie wynijdźiećie próżni.
Ale wypożycży niewiáſtá u ſąśiádki ſwojey / y u goſpodyni domu ſwego / nácżyniá ſrebrnego / y nácżyniá złotego / y ƺát ; y włożyćie je ná ſyny wáƺe / y ná córki wáƺe / y złupićie Egipt.
