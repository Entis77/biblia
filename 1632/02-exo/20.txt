Y mówił Bóg wƺyſtkie te ſłowá á rzekł :
Jám jeſt Pán Bóg twój / którym ćię wywiódł z ziemi Egipſkiej / z domu niewoli.
Nie będzieƺ miáł bogów innych przedemną.
Nie cżyń ſobie obrázu rytego / áni żádnego podobieńſtwá rzecży tych / które ſą ná niebie wzgórę / y które ná ziemi niſko / y które ſą w wodách pod ziemią.
Nie będzieƺ śię im kłániáł / áni im będzieƺ ſłużył ; bom Já Pán Bóg twój / Bóg záwiſny w miłośći / náwiedzájący niepráwośći ojców nád ſyny w trzećiem y w cżwártem pokoleniu tych / którzy mię nienáwidzą ;
A cżyniący miłośierdzie nád tyśiącámi tych / którzy mię miłują / y ſtrzegą przykázániá mego.
Nie bierz imieniá Páná Bogá twego nádáremno ; bo śię Pán mśćić będzie nád tym / który imię jego nádáremno bierze.
Pámiętáj ná dzień odpocżynieniá / ábyś go święćił.
Sześć dni robić będzieƺ / y wykonáƺ wƺyſtkę robotę twoję.
Ale dniá śiódmego odpocżynienie jeſt Páná Bogá twego ; nie będzieƺ cżynił żádnej roboty weń / ty y ſyn twój / y córká twojá / ſługá twój / y ſłużebnicá twojá / bydlę twoje / y gość twój / który jeſt w brámách twoich ;
Bo przez ƺeść dni ſtworzył Pán niebo y ziemię / morze / y cokolwiek w nich jeſt / y odpocżął dniá śiódmego ; przetoż błogoſłáwił Pán dzień odpocżynieniá / y poświęćił go.
Cżćij ojcá twego y mátkę twoję / áby przedłużone były dni twoje ná ziemi / którą Pán Bóg twój dá tobie.
Nie będzieƺ zábijáł.
Nie będzieƺ cudzołożył.
Nie będzieƺ krádł.
Nie będzieƺ mówił przećiw bliźniemu twemu świádectwá fáłƺywego.
Nie będzieƺ pożądáł domu bliźniego twego / áni będzieƺ pożądáł żony bliźniego twego / áni ſługi jego / áni dziewki jego / áni wołu jego / áni oſłá jego / áni żádnej rzecży bliźniego twego.
Tedy wƺyſtek lud widząc gromy / y błyſkáwice / y głoſ trąby / y górę kurzącą śię / to widząc lud cofnęli śię / y ſtánęli z dáleká ;
Y mówili do Mojżeƺá : Mów ty z námi / á będziemy ſłucháć ; á niech nie mówi do náſ Bóg / byśmy ſnáć nie pomárli.
Y odpowiedziáł Mojżeƺ ludowi : Nie bójćie śię ; bo áby wáſ doświádcżył / przyƺedł Bóg / żeby bojáźń jego byłá przed oblicżem wáƺem / byśćie nie grzeƺyli.
Stáł tedy lud z dáleká ; á Mojżeƺ przyſtąpił do ćiemnej mgły / w której był Bóg.
Y rzekł Pán do Mojżeƺá : Ták powieƺ ſynom Izráelſkim : Wyśćie widzieli / żem z niebá mówił do wáſ :
Nie będziećie cżynić przy mnie bogów ſrebrnych / áni bogów złotych nie będziećie ſobie cżynić.
Ołtárz z ziemi ucżyniƺ mi / á ofiárowáć będzieƺ ná nim cáłopálenie twoje / y ſpokojne ofiáry twoje / owce twoje / y woły twoje ; ná któremkolwiek miejſcu pámiątkę ucżynię imieniá mego / przyjdę do ćiebie / y będęć błogoſłáwił.
A jeźli ołtárz kámienny ucżyniƺ mi / nie buduj go z ćioſánego kámieniá ; bo jeźlibyś żelázne nácżynie twoje podnióſł náń / ſplugáwiƺ go.
Nie będzieƺ wſtępowáł po ſtopniách do ołtárzá mojego / áby nie byłá odkrytá nágość twojá przy nim.
