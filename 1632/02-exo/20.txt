Y mówił Bóg wƺyſtkie te ſłowá á rzekł :
Jam jeſt Pán Bóg twój / którym ćię wywiódł z źiemi Egipſkiej / z domu niewoli.
Nie będźieƺ miał bogów innych przedemną.
Nie cżyń ſobie obrázu rytego / áni żádnego podobieńſtwá rzecży tych / które ſą ná niebie wzgórę / y które ná źiemi niſko / y które ſą w wodách pod źiemią.
Nie będźieƺ śię im kłániał / áni im będźieƺ ſłużył ; bom Ja Pán Bóg twój / Bóg záwiſny w miłośći / náwiedzájący niepráwośći ojców nád ſyny w trzećiem y w cżwártem pokoleniu tych / którzy mię nienáwidzą ;
A cżyniący miłośierdźie nád tyśiącámi tych / którzy mię miłują / y ſtrzegą przykázánia mego.
Nie bierz imieniá Páná Bogá twego nádáremno ; bo śię Pán mśćić będźie nád tym / który imię jego nádáremno bierze.
Pámiętáj ná dźień odpocżynieniá / ábyś go święćił.
Sześć dni robić będźieƺ / y wykonáƺ wƺyſtkę robotę twoję.
Ale dniá śiódmego odpocżynienie jeſt Páná Bogá twego ; nie będźieƺ cżynił żádney roboty weń / ty y ſyn twój / y córká twojá / ſługá twój / y ſłużebnicá twojá / bydlę twoje / y gość twój / który jeſt w brámách twojich ;
Bo przez ƺeść dni ſtworzył Pán niebo y źiemię / morze / y cokolwiek w nich jeſt / y odpocżął dniá śiódmego ; przetoż błogoſłáwił Pán dźień odpocżynieniá / y poświęćił go.
Cżćij ojcá twego y mátkę twoję / áby przedłużone były dni twoje ná źiemi / którą Pán Bóg twój da tobie.
Nie będźieƺ zábijał.
Nie będźieƺ cudzołożył.
Nie będźieƺ krádł.
Nie będźieƺ mówił przećiw bliźniemu twemu świádectwá fáłƺywego.
Nie będźieƺ pożądał domu bliźniego twego / áni będźieƺ pożądał żony bliźniego twego / áni ſługi jego / áni dźiewki jego / áni wołu jego / áni oſłá jego / áni żádney rzecży bliźniego twego.
Tedy wƺyſtek lud widząc gromy / y błyſkáwice / y głos trąby / y górę kurzącą śię / to widząc lud cofnęli śię / y ſtánęli z dáleká ;
Y mówili do Mojżeƺá : Mów ty z námi / á będźiemy ſłucháć ; á niech nie mówi do nas Bóg / byſmy ſnadź nie pomárli.
Y odpowiedźiał Mojzeƺ ludowi : Nie bójćie śię ; bo áby was doświádcżył / przyƺedł Bóg / żeby bojáźń jego byłá przed oblicżem wáƺem / byśćie nie grzeƺyli.
Stał tedy lud z dáleká ; á Mojzeƺ przyſtąpił do ćiemney mgły / w którey był Bóg.
Y rzekł Pán do Mojżeƺá : Ták powieƺ ſynom Izráelſkim : Wyśćie widźieli / żem z niebá mówił do was :
Nie będźiećie cżynić przy mnie bogów ſrebrnych / áni bogów złotych nie będźiećie ſobie cżynić.
Ołtárz z źiemi ucżyniƺ mi / á ofiárowáć będźieƺ ná nim cáłopálenie twoje / y ſpokojne ofiáry twoje / owce twoje / y woły twoje ; ná któremkolwiek miejſcu pámiątkę ucżynię imieniá mego / przydę do ćiebie / y będęć błogoſłáwił.
A jeſli ołtárz kámienny ucżyniƺ mi / nie buduj go z ćioſánego kámieniá ; bo jeſlibyś żelázne nacżynie twoje podnióſł nań / ſplugáwiƺ go.
Nie będźieƺ wſtępował po ſtopniách do ołtárzá mojego / áby nie byłá odkrytá nágość twojá przy nim.
