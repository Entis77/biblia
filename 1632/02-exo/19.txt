Mieśiącá trzećiego po wyjśćiu ſynów Izráelſkich z źiemi Egipſkiej / w tenże dźień przyƺli ná puƺcżą Synáj.
Bo ruƺywƺy śię z Ráfidym / y przyƺedłƺy áż ná puƺcżą Synáj / położyli śię obozem ná puƺcży / y rozbił tám Izráel námioty przećiw górze.
A Mojżeƺ wſtąpił do Bogá / y záwołáł náń Pán z góry / mówiąc : Ták powieƺ domowi Jákóbowemu / y oznájmiƺ ſynom Izráelſkim :
Wyśćie widźieli / com ucżynił Egipcżánom / y jákom wáſ nośił niby ná ſkrzydłách orłowych / y przywiodłem wáſ do śiebie.
Przetoż teráz jeźli ſłuchájąc poſłuƺni będźiećie głoſu memu / y ſtrzedz będźiećie przymierzá mego / będźiećie mi włáſnośćią nád wƺyſtkie národy ; choćiáż mojá jeſt wƺyſtká źiemiá.
A wy będźiećie mi króleſtwem kápłáńſkim / y národem świętym. Teć ſą ſłowá / które mówić będźieƺ do ſynów Izráelſkich.
Przyƺedłƺy tedy Mojżeƺ zwołáł ſtárƺych z ludu / y przełożył im wƺyſtkie te ſłowá / które mu rozkázáł Pán.
Y odpowiedźiáł wƺyſtek lud / ſpólnie mówiąc : Wƺyſtko / co Pán rzekł / ucżynimy. Y odnióſł Mojżeƺ ſłowá ludu do Páná.
Y rzekł Pán do Mojżeƺá : Oto / Já / przyjdę do ćiebie w gęſtym obłoku / áby ſłucháł lud / gdy będę mówił z tobą / áżeby też wierzyli tobie ná wieki ; ábowiem opowiedźiáł był Mojżeƺ ſłowá ludu onego Pánu.
Mówił záś Pán do Mojżeƺá : Idź do ludu / á poświęć je dźiś y jutro / á niech wypiorą ƺáty ſwoje.
Y niech będą gotowi ná dźień trzeći ; ábowiem trzećiego dniá zſtąpi Pán przed ocżymá wƺyſtkiego ludu ná górę Synáj.
Y zámierzyƺ gránice ludowi w około / mówiąc : Strzeżćie śię / ábyśćie nie wſtępowáli ná górę / áni śię dotykáli brzegu jej ; wƺelki / ktoby śię dotknął góry / śmierćią umrze.
Nie tknie śię go ręká / ále kámieniem ukámionują go ; álbo ſtrzelájąc záſtrzelą go ; bądź bydlę bądź cżłowiek / nie będźie żył. Gdy przewłocżnie trąbić będą / niech wſtąpią ná górę.
Zſtąpił tedy Mojżeƺ z góry do ludu / y poświęćił lud ; á upráli ƺáty ſwoje.
Y mówił do ludu : Bądźćie gotowi ná dźień trzeći / nie przyſtępujćie do żon.
Stáło śię tedy dniá trzećiego po ránu / że były grzmieniá / y błyſkáwice / y gęſty obłok nád górą / y głoſ trąby bárdzo potężny ; á báł śię wƺyſtek lud / który był w oboźie.
Y wywiódł Mojżeƺ lud ná przećiwko Bogu z obozu / á ſtánęli pod ſámą górą.
A górá Synáj kurzyłá śię wƺyſtká / przeto / iż zſtąpił ná nią Pán w ogniu ; y wyſtępowáł dym z niej / jáko dym z piecá / y trzęſłá śię wƺyſtká górá bárdzo.
A gdy śię głoſ trąby im dálej tem bárdźiej rozlegáł / Mojżeƺ mówił / á Bóg mu odpowiádáł głoſem.
Y zſtąpił Pán ná górę Synáj / ná wierzch góry / y wezwáł Pán Mojżeƺá ná wierzch góry / y wſtąpił tám Mojżeƺ.
Zácżem rzekł Pán do Mojżeƺá : Zſtąp / przeſtrzeż lud / by ſnáć nie przeſtąpili kreſu / chcąc Páná widźieć / áby ich nie pádło wiele :
Náwet y kápłáni / którzy przyſtępują do Páná / niech śię poświęcą / by ich ſnáć nie potráćił Pán.
Y rzekł Mojżeƺ do Páná : Nie będźie lud mógł wnijść ná górę Synáj / poniewáżeś ty náſ przeſtrzegł / mówiąć : Ogránicż górę / á poświęć ją.
Któremu Pán rzekł : Idź / zſtąp / á záś tu wſtąpiƺ / ty y Aáron z tobą : lecż kápłáni y lud niech nie przeſtępują kreſu / áby wſtąpili do Páná / by ich ſnáć nie potráćił.
Tedy zſtąpił Mojżeƺ do ludu y powiedźiáł im to.
