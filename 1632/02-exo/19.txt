Mieśiącá trzećiego po wyśćiu ſynów Izráelſkich z źiemi Egipſkiej / w tenże dźień przyƺli ná puƺcżą Synáj.
Bo ruƺywƺy śię z Ráfidym / y przyƺedƺy áż ná puƺcżą Synáj / położyli śię obozem ná puƺcży / y rozbił tám Izráel námioty przećiw górze.
A Mojzeƺ wſtąpił do Bogá / y záwołał nań Pán z góry / mówiąc : Ták powieƺ domowi Jákóbowemu / y oznájmiƺ ſynom Izráelſkim :
Wyśćie widźieli / com ucżynił Egipcżánom / y jákom was nośił niby ná ſkrzydłách orłowych / y przywiodłem was do śiebie.
Przetoż teraz jeſli ſłuchájąc poſłuƺni będźiećie głoſu memu / y ſtrzedz będźiećie przymierza mego / będźiećie mi włáſnośćią nád wƺyſtkie narody ; choćiáż mojá jeſt wƺyſtká źiemiá.
A wy będźiećie mi króleſtwem kápłáńſkim / y národem świętym. Teć ſą ſłowá / które mówić będźieƺ do ſynów Izráelſkich.
Przyƺedƺy tedy Mojzeƺ zwołał ſtárƺych z ludu / y przełożył im wƺyſtkie te ſłowá / które mu rozkázał Pán.
Y odpowiedźiał wƺyſtek lud / ſpólnie mówiąc : Wƺyſtko / co Pán rzekł / ucżynimy. Y odnióſł Mojzeƺ ſłowá ludu do Páná.
Y rzekł Pán do Mojżeƺá : Oto / Ja / przydę do ćiebie w gęſtym obłoku / áby ſłuchał lud / gdy będę mówił z tobą / áżeby też wierzyli tobie ná wieki ; ábowiem opowiedźiał był Mojzeƺ ſłowá ludu onego Pánu.
Mówił záś Pán do Mojżeƺá : Idź do ludu / á poświęć je dźiś y jutro / á niech wypiorą ƺáty ſwoje.
Y niech będą gotowi ná dźień trzeći ; ábowiem trzećiego dniá zſtąpi Pán przed ocżymá wƺyſtkiego ludu ná górę Synáj.
Y zámierzyƺ gránice ludowi w około / mówiąc : Strzeżćie śię / ábyśćie nie wſtępowáli ná górę / áni śię dotykáli brzegu jey ; wƺelki / ktoby śię dotknął góry / śmierćią umrze.
Nie tknie śię go ręká / ále kámieniem ukámionują go ; álbo ſtrzelájąc záſtrzelą go ; bądź bydlę bądź cżłowiek / nie będźie żył. Gdy przewłocżnie trąbić będą / niech wſtąpią ná górę.
Zſtąpił tedy Mojzeƺ z góry do ludu / y poświęćił lud ; á upráli ƺáty ſwoje.
Y mówił do ludu : Bądźćie gotowi ná dźień trzeći / nie przyſtępujćie do żon.
Stáło śię tedy dniá trzećiego po ránu / że były grzmieniá / y błyſkáwice / y gęſty obłok nád górą / y głos trąby bárzo potężny ; á bał śię wƺyſtek lud / który był w oboźie.
Y wywiódł Mojzeƺ lud ná przećiwko Bogu z obozu / á ſtánęli pod ſámą górą.
A górá Synáj kurzyłá śię wƺyſtká / przeto / iż zſtąpił ná nią Pán w ogniu ; y wyſtępował dym z niej / jáko dym z piecá / y trzęſłá śię wƺyſtká górá bárzo.
A gdy śię głos trąby im dálej tym bárdźiej rozlegał / Mojzeƺ mówił / á Bóg mu odpowiádał głoſem.
Y zſtąpił Pán ná górę Synáj / ná wierzch góry / y wezwał Pán Mojżeƺá ná wierzch góry / y wſtąpił tám Mojzeƺ.
Zácżem rzekł Pán do Mojżeƺá : Zſtąp / przeſtrzeż lud / by ſnadź nie przeſtąpili kreſu / chcąc Páná widźieć / áby ich nie pádło wiele :
Náwet y kápłáni / którzy przyſtępują do Páná / niech śię poświęcą / by ich ſnadź nie potráćił Pán.
Y rzekł Mojzeƺ do Páná : Nie będźie lud mógł wnijść ná górę Synáj / poniewáżeś ty nas przeſtrzegł / mówiąc : Ogránicż górę / á poświęć ją.
Któremu Pán rzekł : Idź / zſtąp / á záś tu wſtąpiƺ / ty y Aáron z tobą : lecż kápłáni y lud niech nie przeſtępują kreſu / áby wſtąpili do Páná / by ich ſnadź nie potráćił.
Tedy zſtąpił Mojzeƺ do ludu y powiedźiał im to.
