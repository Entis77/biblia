Proƺę wáſ tedy já więźień w Pánu / ábyśćie chodźili ták / jáko przyſtoi ná powołánie / którem jeſteśćie powołáni ;
Ze wƺeláką pokorą y ćichośćią y z nieſkwápliwośćią / znoƺąc jedni drugich w miłośći /
Stárájąc śię / ábyśćie záchowáli jedność duchá w zwiąſce pokoju.
Jedno jeſt ćiáło y jeden duch / jáko też jeſteśćie powołáni w jednej nádźiei powołániá wáƺego.
Jeden Pán / jedná wiárá / jeden chrzeſt ;
Jeden Bóg y Ojćiec wƺyſtkich / który jeſt náde wƺyſtko / y po wƺyſtkich y we wƺyſtkich wáſ.
Lecż káżdemu z náſ dáná jeſt łáſká według miáry dáru CHryſtuſowego.
Dlátego Piſmo mówi : Wſtąpiwƺy ná wyſokość / wiódł pojmánych więźni / y dáł dáry ludźiom.
Ale to / że wſtąpił / cóż jeſt / jedno iż pierwej był zſtąpił do nájniżƺych ſtron źiemi?
A który zſtąpił / ten jeſt / który y wſtąpił wyſoko nád wƺyſtkie niebioſá / áby nápełnił wƺyſtko.
Y tenże dáł niektóre Apoſtoły / á niektóre proroki / á drugie Ewángieliſty / drugie też páſterze y náucżyćiele.
Ku ſpojeniu świętych / ku prácy uſługiwániá / ku budowániu ćiáłá CHryſtuſowego ;
A iżbyśmy śię wƺyſcy zeƺli w jedność wiáry y znájomośći Syná Bożego / w mężá doſkonáłego / w miárę zupełnego wieku CHryſtuſowego /
Abyśmy więcej nie byli dźiećmi / chwiejącymi śię y unoƺącymi śię káżdym wiátrem náuki przez fortel ludzki / y przez chytrość podejśćiá błędem.
Ale ƺcżerymi będąc w miłośći rośnijmy w onego we wƺyſtkiem / który jeſt głową / to jeſt w CHryſtuſá /
Z którego wƺyſtko ćiáło przyſtojnie złożone y ſpojone we wƺyſtkich ſtáwách / przez które jeden cżłonek drugiemu pośiłku dodáje przez moc / którá jeſt w káżdym cżłonku / według miáry jego / wzroſt ćiáłu náleżący bierze ku budowániu ſámego śiebie w miłośći.
To tedy mówię / y oświádcżám śię przez Páná / ábyśćie już więcej nie poſtępowáli / jáko inśi pogánie poſtępują / w próżnośći umyſłu ſwego ;
Záćmiony májąc rozſądek / będąc oddáleni od żywotá Bożego dlá nieumiejętnośći / którá w nich jeſt z zátwárdzeniá ſercá ich.
Którzy zákámieniáwƺy / událi śię ná rozpuſtę ku popełniániu wƺelákiej niecżyſtośći z chćiwośćią /
Lecż wy nie tákeśćie śię náucżyli CHryſtuſá /
Jeźliśćie go tylko ſłucháli / y o nim wyucżeni byli / jáko jeſt ( tá ) práwdá w JEzuśie /
To jeſt / żebyśćie złożyli według pierwƺego obcowániá ſtárego cżłowieká / który śię pſuje przez pożądliwośći oƺukiwájące ;
Y odnowili śię duchem umyſłu wáƺego ;
Y oblekli śię w onego nowego cżłowieká / który według Bogá ſtworzony jeſt w ſpráwiedliwośći y w świętobliwośći práwdy.
Przetoż złożywƺy kłámſtwo mówćie práwdę / káżdy z bliźnim ſwoim ; bośćie cżłonkámi jedni drugich.
Gniewájćie śię á nie grzeƺćie ; ſłońce niech nie záchodźi ná rozgniewánie wáƺe.
Nie dáwájćie miejſcá dyjábłu.
Kto krádł / niech więcej nie krádnie / ále rácżej niech prácuje / robiąc rękomá / co jeſt dobrego / áby miáł ſkąd udźieláć potrzebującemu.
Żádná mowá plugáwá niech z uſt wáƺych nie pochodźi ; ále jeźli którá jeſt dobrá ku potrzebnemu zbudowániu / áby byłá przyjemná ſłuchájącym.
A nie záſmucájćie Duchá Świętego Bożego / którym zápiecżętowáni jeſteśćie ná dźień odkupieniá.
Wƺelká gorzkość / y zápálcżywość / y gniew / y wrzáſk / y bluźnierſtwo / niech będźie odjęte od wáſ / ze wƺeláką złośćią ;
A bądźćie jedni przećiwko drugim dobrotliwi / miłośierni / odpuƺcżájąc ſobie / jáko wám Bóg w CHryſtuśie odpuśćił.
