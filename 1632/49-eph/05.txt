Bądźćież tedy náśládowcámi Bożymi / jáko dźieći miłe ;
A chodźćie w miłośći / jáko y CHryſtuſ umiłowáł náſ / y wydáł ſámego śiebie ná ofiárę y ná zábićie Bogu ku wdźięcżnej wonnośći.
A wƺetecżeńſtwo y wƺelká niecżyſtość álbo łákomſtwo niecháj nie będźie áni miánowáne między wámi / jáko przyſtoi ná świętych.
Tákże ſproſność / y błázeńſkie mowy / y żárty / które nie przyſtoją / ále rácżej dźiękowánie.
Bo to wiećie / iż żáden wƺetecżnik / álbo niecżyſty / álbo łákomcá / ( który jeſt báłwochwálcą / ) nie má dźiedźictwá w króleſtwie CHryſtuſowem y Bożem.
Niecháj wáſ nikt nie zwodźi próżnemi mowámi ; ábowiem dlá tych rzecży przychodźi gniew Boży ná ſynów upornych ;
Nie bądźćież tedy ucżeſnikámi ich.
Abowiemeśćie byli niekiedy ćiemnośćią ; áleśćie teráz świátłośćią w Pánu ; chodźćież jáko dźiátki świátłośći /
( Bo owoc Duchá záleży we wƺelákiej dobrotliwośći y w ſpráwiedliwośći y w práwdźie. )
Obierájąc to / coby śię podobáło Pánu ;
A nie ſpółkujćie z ucżynkámi niepożytecżnemi ćiemnośći / ále je rácżej ſtrofujćie.
Abowiem co śię potájemnie od nich dźieje / ſromotá y mówić.
Lecż to wƺyſtko / gdy bywá od świátłośći ſtrofowáne / bywá objáwione ; ábowiem to wƺyſtko / co bywá objáwione / jeſt świátłośćią ;
Dlátego mówi Piſmo : Ocuć śię / który śpiƺ / y powſtáń od umárłych / á oświeći ćię CHryſtuſ.
Pátrzájćie tedy / jákobyśćie oſtrożnie chodźili / nie jáko niemądrzy / ále jáko mądrzy /
Cżáſ odkupując ; bo dni złe ſą.
Przetoż nie bądźćie nierozumnymi / ále zrozumiewájącymi / którá jeſt wolá Páńſká.
A nie upijájćie śię winem / w którym jeſt rozpuſtá ; ále bądźćie nápełnieni duchem /
Rozmáwiájąc z ſobą przez pſálmy / y hymny / y pieśni duchowne / śpiewájąc y grájąc w ſercu ſwojim Pánu /
Dźięki cżyniąc záwƺe zá wƺyſtko / w imieniu Páná náƺego JEzuſá CHryſtuſá / Bogu y Ojcu /
Będąc poddáni jedni drugim w bojáźni Bożej.
Żony! bądźćie poddáne mężom ſwoim / jáko Pánu :
Abowiem mąż jeſt głową żony / jáko y CHryſtuſ głową kośćiołá ; á on jeſt zbáwićielem ćiáłá.
Jáko tedy kośćiół poddány jeſt CHryſtuſowi / ták też żony mężom ſwoim we wƺyſtkiem.
Mężowie! miłujćie żony wáƺe / jáko y CHryſtuſ umiłowáł kośćiół / y wydáł ſámego śiebie záń /
Aby go poświęćił / ocżyśćiwƺy omyćiem wody przez ſłowo ;
Aby go ſobie wyſtáwił chwálebnym kośćiołem / niemájącym zmázy / álbo zmárƺcżku / álbo cżego tákiego / ále iżby był święty y bez nágány.
Ták powinni mężowie miłowáć żony ſwoje / jáko ſwoje włáſne ćiáłá ; kto miłuje żonę ſwoję / ſámego śiebie miłuje.
Abowiem żáden nigdy ćiáłá ſwego nie miáł w nienáwiśći / ále je żywi y ogrzewá / jáko y Pán kośćiół.
Gdyżeśmy cżłonkámi ćiáłá jego / z ćiáłá jego y z kośći jego.
Dlátego opuśći cżłowiek ojcá ſwego y mátkę / y przyłącży śię do żony ſwojej / y będą dwoje jednem ćiáłem.
Tájemnicá to wielká jeſt ; lecż já mówię o CHryſtuśie y o kośćiele.
A wƺákże y káżdy z wáſ z oſobná niecháj miłuje żonę ſwoję jáko śiebie ſámego / á żoná niech śię boi mężá ſwego.
