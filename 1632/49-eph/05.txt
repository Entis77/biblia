Bądźćież tedy náśládowcámi Bożymi / jáko dźieći miłe.
A chodźćie w miłośći / jáko y CHryſtus umiłował nas / y wydał ſámego śiebie ná ofiárę y ná zábićie Bogu / ku wdźięcżney wonnośći.
A wƺetecżeńſtwo y wƺelka niecżyſtość álbo łákomſtwo / niechaj nie będźie áni miánowáne miedzy wámi / jáko przyſtoji ná święte.
Tákże ſproſność / y błázeńſkie mowy / y żárty które nie przyſtoją : ále rácżey dźiękowánie.
Bo to wiećie / iż żaden wƺetecżnik / álbo niecżyſty / álbo łákomcá ( który jeſt báłwochwálcą ) nie ma dźiedźictwá w króleſtwie CHryſtuſowym y Bożym.
Niechaj was nikt nie zwodźi próżnymi mowámi : ábowiem dla tych rzecży przychodźi gniew Boży ná ſyny uporne.
Nie bądźćież tedy ucżeſnikámi ich.
Abowiemeśćie byli niekiedy ćiemnośćią : áleśćie teraz świátłośćią w PAnu. Chodźćież jáko dźiatki świátłośći.
Bo owoc Duchá <i>zależy</i> we wƺelákiey dobrotliwośći y w ſpráwiedliwośći / y w prawdźie.
Obierájąc to coby śię podobáło PAnu.
A nie ſpółkujćie z ucżynkámi niepożytecżnymi ćiemnośći / ále je rácżey ſtrofujćie.
Abowiem co śię potájemnie od nich dźieje / ſromotá y mówić.
Lecż to wƺyſtko gdy bywa od świátłośći ſtrofowano / bywa objáwiono. Abowiem to wƺyſtko / co bywa objáwiono / jeſt świátłośćią.
Dla tego mówi <i>piſmo</i> : Ocuć śię który ſpiƺ / y powſtań od umárłych ; á oświeći ćię CHryſtus.
Patrzajćież tedy jákobyśćie oſtrożnie chodźili ; nie jáko niemądrzy / ále jáko mądrzy.
Cżás odkupując ; bo dni złe ſą.
Przetoż nie bądźćie nierozumnymi ; ále zrozumiewájącymi która jeſt wola PAńſka.
A nie upijajćie śię winem / w którym jeſt rozpuſtá ; ále bądźćie nápełnieni Duchem :
Rozmawiájąc z ſobą przez Pſálmy y Hymny / y pieśni duchowne / śpiewájąc y grájąc w ſercu ſwojim PAnu.

Będąc poddáni jedni drugim w bojáźni Bożey.
Żony bądźćie poddáne mężom ſwojim / jáko PAnu.
Abowiem mąż jeſt głową żony / jáko y CHryſtus głową Kośćiołá : á on jeſt Zbáwićielem ćiáłá.
Jáko tedy Kośćiół poddány jeſt CHryſtuſowi / ták też żony mężom ſwojim / we wƺyſtkim.
Mężowie miłujćie żony wáƺe / jáko y CHryſtus umiłował Kośćiół / y wydał ſámego śiebie zań.
Aby go poświęćił / ocżyśćiwƺy omyćiem wody przez ſłowo.
Aby go ſobie wyſtáwił chwalebnym Kośćiołem / nie májącym zmázy / álbo zmarſku / álbo cżego tákiego : ále iżby był święty / y bez nágány.
Ták powinni mężowie miłowáć żony ſwoje / jáko ſwoje właſne ćiáłá. Kto miłuje żonę ſwoję / ſámego śiebie miłuje.
Abowiem żaden nigdy ćiáłá ſwego nie miał w nienawiśći ; ále je żywi y ogrzewa / jáko y PAN Kośćiół.
Gdyżeſmy cżłonkámi ćiáłá jego z ćiáłá jego / y z kośći jego.
Dla tego opuśći cżłowiek ojcá ſwego / y mátkę / y przyłącży śię do żony ſwojey : y będą dwoje jednym ćiáłem.
Tájemnicá to wielka jeſt : lecż ja mówię o CHryſtuśie y o Kośćiele.
A wƺákże y káżdy z was z oſobná / niechaj miłuje żonę ſwoję jáko śiebie ſámego ; á żoná niech śię boji mężá <i>ſwego</i>.
