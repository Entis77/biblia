Y nie podobáło śię to bárzo Jonáƺowi / y rozpálił śię gniew jego.
Przetoż śię modlił Pánu / y rzekł : Proƺę Pánie! ázáżem tego nie mówił / gdym jeƺcże był w źiemi mojey? Dlátegom śię poſpieƺył / ábym ućiekł do Tárſu / gdyżem wiedźiał / żeś ty Bóg łáſkáwy y litośćiwy / długo ćierpliwy y wielkiego miłośierdźiá / á który żáłujeƺ złego.
A teraz / o Pánie! proƺę / odbierz duƺę moję odemnie : bo mi lepiej umrzeć / niżeli żyć.
Y rzekł Pán : A dobrzeż to / że śię ták gniewáƺ?
Bo wyƺedł był Jonáƺ z miáſtá / y śiedźiał ná wſchód ſłońcá przećiwko miáſtu ; á ucżyniwƺy tám ſobie budę / uśiádł pod nią w ćieniu / áżby ujrzał / coby śię dźiáło z onem miáſtem.
A Pán Bóg był zgotował bánię / która wyroſłá nád Jonáƺem / áby záſłániáłá głowę jego / y záſtáwiáłá go od gorącá ; tedy śię Jonáƺ bárzo z oney báni rádował.
Wtem názájutrz ná świtániu nágotował Bóg robáká / który podgryzł onę bánię / ták / że uſchłá.
Y ſtáło śię / gdy weƺło ſłońce / wzbudźił Bóg wiátr ſuchy od wſchodu ſłońcá / y biło ſłońce ná głowę Jonáƺową / ták / iż omdlewał / y życżył ſobie śmierći / mówiąc : Lepiej mi umrzeć / niżeli żyć.
Y rzekł Bóg do Jonáƺá : Dobrzeż to / że śię ták gniewáƺ o tę bánię? Y rzekł : Dobrze / że śię gniewám áż ná śmierć.
Tedy mu rzekł Pán : Ty żáłujeƺ tey báni / około którejś nie prácował / ániś jey dał wzroſtu / która jedney nocy uroſłá / y jedney nocy zginęłá ;
A Jábym nie miał żáłowáć Niniwy / miáſtá ták wielkiego? w którym jeſt więcey niżeli ſto y dwádźieśćiá tyśięcy ludźi / którzy nie umieją rozeznáć miedzy práwicą ſwoją y lewicą ſwoją / y bydłá wiele.
