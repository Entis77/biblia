Y ſtáło śię ſłowo Páńſkie do Jonáƺá / ſyná Amáty / mówiąc :
Wſtáń / idź do Niniwy miáſtá tego wielkiego / á wołáj przećiwko niemu ; bo wſtąpiłá złość ich przed oblicże moje.
Ale Jonáƺ wſtáł / áby ućiekł do Tárſu od oblicżá Páńſkiego ; á przyƺedłƺy do Joppen / ználázł okręt / który miał iść do Tárſu / á zápłáćiwƺy od niego wſtąpił náń / áby płynął z nimi do Tárſu od oblicżá Páńſkiego.
Ale Pán wzruƺył wiátr wielki ná morzu / y powſtáł wicher wielki ná morzu ; y zdáło śię / jákoby śię okręt rozbić miał.
A żeglárze ulękłƺy śię wołáli káżdy do bogá ſwego / á wyrzucáli do morzá to / co mieli ná okręćie / áby tem lżejƺy był ; ále Jonáƺ zƺedł był ná ſtronę okrętu / á położywƺy śię ſpáł twárdo.
Tedy przyſtąpił do niego ſternik / y rzekł mu : Cóż cżyniƺ ty / oſpálcże? wſtáń / wołáj do Bogá ſwego / owá ſnáć wſpomni Bóg ná nas / ábyśmy nie zginęli.
Tedy rzekł jeden do drugiego : Chodźćie / rzućmy loſy / ábyśmy śię dowiedźieli / dla kogo to złe ná nas przyƺło ; rzućili tedy loſy / y pádł loſ ná Jonáƺá.
Y rzekli do niego : Powiedz nám proƺę / dla kogo to złe przyƺło ná nas? coś zá rzemioſłá? ſkąd idźieƺ? z którejś źiemi y z któregoś národu?
Y rzekł do nich : Jeſtem Hebrejcżyk / á boję śię Páná / Bogá niebieſkiego / który ſtworzył morze y źiemię.
Tedy śię zlękli mężowie ſtráchem wielkim ; á dowiedźiáwƺy śię mężowie oni / że od oblicżá Páńſkiego ućieká / ( bo im był oznájmił ) rzekli do niego : Cóżeś to ucżynił?
Nádto rzekli do niego : Cóż z tobą ucżynimy / áby śię morze uſpokoiło? Bo śię morze im dálej tem bárdźiej burzyło.
Tedy rzekł do nich : Weźmijćie mię / á wrzuććie mię w morze / á uſpokoi śię morze przed wámi / gdyż ja wiem / iż dla mnie to wzruƺenie wielkie ná was przyƺło.
Ale oni mężowie robili wioſłámi / chcąc śię do brzegu doſtáć / wƺákże nie mogli ; bo śię morze im dálej tem więcey burzyło przećiwko nim.
Wołáli tedy do Páná / mówiąc : O Pánie! prośimy / ábyśmy nie zginęli dla śmierći mężá tego / áni wkłádáj ná nas krwi niewinney ; bo ty / o Pánie! jáko chceƺ / ták cżyniƺ.
Zátem wźięli Jonáƺá y wrzućili go w morze ; y uſpokoiło śię morze od wzburzeniá ſwego.
Báli śię tedy mężowie ſtráchem wielkim Páná / y ofiárowáli ofiárę Pánu / y śluby cżynili.
