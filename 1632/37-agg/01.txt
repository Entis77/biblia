Roku wtórego Dáryjuƺá królá / mieśiącá ƺóſtego / dniá pierwƺego tegoż mieśiącá / ſtáło śię ſłowo Páńſkie przez Aggieuƺá proroká do Zorobábelá / ſyná Sálátyjelowego / kśiążęćiá Judzkiego / y do Jeſuego / ſyná Jozedekowego / kápłáná nájwyżƺego / mówiąc :
Ták powiádá Pán záſtępów / mówiąc : Ten lud mówi : Jeƺcże nie przyƺedł cżáſ / cżáſ budowániá domu Páńſkiego.
Przetoż śię ſtáło ſłowo Páńſkie przez Aggieuƺá proroká / mówiąc :
Izáli wám jeſt cżáſ / ábyśćie mieƺkáli w domách wáƺych liſtwowánych / á dom ten áby puſty ſtáł?
Teraz tedy ták mówi Pán záſtępów : Uwáżájćież / jáko śię wám powodźi ;
Siejećie wiele / á máło zbieráćie ; jećie / ále śię nie náſycáćie ; pijećie / ále nie ugáƺáćie prágnieniá ; obłócżyćie śię / ále śię nikt nie może zágrzáć / á ten / co ſobie zápłátę zgromádzá / zgromádzá ją do worká dźiuráwego.
Ták mówi Pán záſtępów : Uwáżájćie / jáko śię wám powodźi ;
Wſtąpćie ná tę górę / y zwoźćie drzewo ; budujćie ten dom / á zákochám śię w nim / y będę uwielbiony / mówi Pán.
Poglądáćie ná wiele / á oto máło doſtáwáćie / á co wnośićie do domu / to Ja rozdmuchywám ; dlácżego? mówi Pán záſtępów ; dlátego / że dom mój jeſt puſty / á międzytem śię káżdy z was ſtárá o dom ſwój.
Przetoż śię nád wámi niebo záwárło / áby roſy nie dáwáło ; źiemiá tákże záwárłá śię / áby nie wydáwáłá urodzáju ſwego.
A ták przyzwáłem ſuƺę ná tę źiemię / y ná te góry / y ná pƺenicę / y ná moƺcż / y ná oliwę / y ná to / coby miáłá wydáć źiemiá / y ná ludźi y ná bydło / y ná wƺyſtkę prácę ręcżną.
Tedy uſłucháł Zorobábel / ſyn Sálátyjelowy / y Jeſuá / ſyn Jozedekowy / kápłán nájwyżƺy / y wƺyſtkie oſtátki ludu / głoſu Páná / Bogá ſwego / y ſłów Aggieuƺá proroká / poniewáż go poſłáł Pán / Bóg ich ; bo śię lud báł oblicżá Páńſkiego.
Tedy Aggieuƺ / poſeł Páńſki / rzekł do ludu / będąc w poſelſtwie Páńſkim / mówiąc : Jám z wámi / mówi Pán.
Wtem wzbudźił Pán duchá Zorobábelá / ſyná Sálátyjelowego / kśiążęćiá Judzkiego / y duchá Jeſuego / ſyná Jozedekowego / kápłáná nájwyżƺego / y duchá oſtátków wƺyſtkiego ludu / że przyƺedłƺy robili około domu Páná záſtępów / Bogá ſwego.
