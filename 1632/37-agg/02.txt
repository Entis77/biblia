Dniá dwudźieſtego y cżwártego / mieśiącá ƺóſtego / roku wtórego Dáryjuƺá królá ;
Siódmego mieśiącá / dwudźieſtego y pierwƺego dniá tegoż mieśiącá ſtáło śię ſłowo Páńſkie przez Aggieuƺá proroká / mówiąc :
Mów teráz do Zorobábelá / ſyná Sálátyjelowego / kśiążęćiá Judzkiego / y do Jeſuego / ſyná Jozedekowego / kápłáná nájwyżƺego / y do oſtátku ludu / mówiąc :
Któż między wámi pozoſtáł / co widźiáł ten dom w ſłáwie pierwƺej jego? Y jáki wy teráz widźićie? Izáli nie jeſt przećiwko onemu jáko nic w ocżách wáƺych?
Jednák teráz zmocnij śię Zorobábelu! mówi Pán ; zmocnij śię / Jeſue / ſynu Jozedekowy / kápłánie nájwyżƺy! zmocnij śię też wƺyſtek ludu tej źiemi / mówi Pán / á róbćie ; bom Já z wámi / mówi Pán záſtępów /
Według ſłowá / któremem przymierze ucżynił z wámi / gdyśćie wychodźili z Egiptu ; duch tákże mój ſtánie w pośrodku wáſ / nie bójćież śię.
Bo ták mówi Pán záſtępów : Oto Já jeƺcże ráz / á to po máłym cżáśie / poruƺę niebem y źiemią / y morzem y ſuchą ;
Poruƺę / mówię / wƺyſtkie národy / y przyjdą do Pożądánego od wƺyſtkich národów ; y nápełnię ten dom chwáłą / mówi Pán záſtępów.
Mojeć jeſt ſrebro / y moje złoto / mówi Pán záſtępów.
Więkƺá będźie ſłáwá domu tego pośledniego / niż onego pierwƺego / mówi Pán záſtępów ; bo ná tem miejſcu dám pokój / mówi Pán záſtępów.
Dwudźieſtego y cżwártego dniá / dźiewiątego mieśiącá / roku wtórego Dáryjuƺá / ſtáło śię ſłowo Páńſkie przez Aggieuƺá proroká / mówiąc :
Ták mówi Pán záſtępów : Pytáj śię teráz kápłánów o zákon / mówiąc :
Gdyby kto nióſł mięſo poświęcone w podołku ſukni ſwej / álboby śię dotknął podołkiem ſwoim chlebá / álbo potráwy / álbo winá / álbo oliwy / álbo wƺelkiego pokármu / izáli śię poświęći? Y odpowiedźieli kápłáni á rzekli : Nie.
Tedy rzekł Aggieuƺ : Jeźli kto będąc niecżyſty od trupá / dotknąłby śię cżego z tych rzecży / będźieli niecżyſte? Y odpowiedźieli kápłáni y rzekli : Będźie niecżyſte.
Tedy odpowiádájąc Aggieuƺ rzekł : Tákći lud ten / ták y náród ten przed oblicżem mojem / mówi Pán / ták y wƺyſtká ſpráwá rąk ich / y cokolwiek tám ofiárowáli / niecżyſte było.
A ták uwáżájćież / proƺę / jáko śię wám powodźiło od tego dniá áż do onego / kiedy przeſtáno kłáść kámieniá ná kámieniu w kośćiele Páńſkim ;
Od tego cżáſu / gdy kto przyƺedł do gromády zbożá / od dwudźieſtu korcy ználázł dźieśięć ; gdy przyƺedł do práſy / áby nácżerpáł pięćdźieśiąt wiáder winá / ználázł tylko dwádźieśćiá ;
Káráłem wáſ zárázą zbóż y rdzą / y grádem wƺyſtkie práce rąk wáƺych ; wƺákże żáden z wáſ nie wróćił śię do mnie / mówi Pán.
Uwáżájćież teráz ode dniá tego áż do onego / ode dniá dwudźieſtego y cżwártego mieśiącá dźiewiątego áż do dniá / którego był záłożony kośćiół Páńſki / uwáżájćie / mówię :
Izáli jeƺcże jeſt náśienie w ƺpichlerzu? Y owƺem / áni máćicá winná / áni figowe áni gránátowe áni oliwne drzewo nie wydáły owocu ; lecż ode dniá tego będę błogoſłáwił.
Potem ſtáło śię ſłowo Páńſkie powtóre do Aggieuƺá dniá dwudźieſtego y cżwártego tegoż mieśiącá / mówiąc :
Mów do Zorobábelá / kśiążęćiá Judzkiego / á rzecż : Já poruƺę niebioſá y źiemię ;
Y podwrócę ſtolice króleſtw / y zepſuję moc króleſtw pogáńſkich ; podwrócę / mówię / wóz / y tych / którzy ná nim jeżdżą / y upádną konie y jeżdżący ná nich / káżdy od miecżá brátá ſwego.
W on dźień / mówi Pán záſtępów / wezmę ćię / Zorobábelu / ſynu Sálátyjelowy / ſługo mój! mówi Pán / y ucżynię ćię jáko ſygnet ; ábowiem wybráłem ćię / mówi Pán záſtępów.
