Nápominam tedy / áby przed wƺyſtkimi <i>rzecżámi</i> cżynione były proźby / modlitwy / przycżyny / y dźiękowánia zá wƺyſtkie ludźie :
Zá Króle y zá wƺyſtkie w przełożeńſtwie będące ; ábyſmy ćichy y ſpokojny żywot wiedli / we wƺelkiey pobożnośći y ucżćiwośći.
Abowiem to jeſt <i>rzecż</i> dobra y przyjemna przed Bogiem Zbáwićielem náƺym :
Który chce áby wƺyſcy ludźie byli zbáwieni / y ku znájomośći prawdy przyƺli.
Boć jeden jeſt Bóg / jeden tákże Pośrzednik między Bogiem y ludźmi / cżłowiek CHryſtus JEzus.
Który dał ſámego śiebie ná okup zá wƺyſtkie / <i>co jeſt</i> świádectwem cżáſów jego.
Ná com ja jeſt poſtánowiony zá Káznodźieję y Apoſtołá / ( prawdę mówię w CHryſtuśie / nie kłamam ) zá Náucżyćielá Pogánów / w wierze y w prawdźie.
Chcę tedy áby śię mężowie modlili ná káżdym miejſcu / podnoƺąc cżyſte ręce / bez gniewu y bez poſwarku.
Tákże y niewiáſty áby śię ubiorem przyſtojnym ze wſtydem y ſkromnośćią zdobiły ; nie z tráfionymi włoſámi / álbo złotem / álbo perłámi / álbo ƺátámi koƺtownymi :
Ale ( jáko przyſtoji niewiáſtam / które śię ozywáją do pobożnośći ) dobrymi ucżynkámi.
Niewiáſtá niech śię ucży w milcżeniu ze wƺelkim poddáńſtwem.
Bo niewieśćie nie pozwalam ucżyć / áni władze mieć nád mężem ; ále áby byłá w milcżeniu.
Bo Adam pierwƺy ſtworzony jeſt / potym Ewá.
Y Adam nie był zwiedźiony / ále niewiáſtá zwiedźiona będąc / przeſtępſtwá przycżyną byłá.
Lecż zbáwiona będźie przez rodzenie dźiatek ; jeſliby zoſtáłá w wierze / y w miłośći / y w świętobliwośći / z miernośćią.
