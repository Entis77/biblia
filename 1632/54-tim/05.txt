Stárƺemu nie łaj / ále jáko ojcá nápominaj ; młodƺe jáko bráćią :
Stárƺe niewiáſty jáko mátki / młodƺe jáko śioſtry / ze wƺeláką cżyſtośćią.
Wdowy miej w ucżćiwośći / które prawdźiwie <i>ſą</i> wdowámi.
A jeſli która wdowá dźieći álbo wnucżętá ma / niech śię ucżą pierwey przećiwko domowi ſwemu być pobożnymi / y wzajem oddawáć rodźicom. Abowiem to jeſt <i>rzecż</i> chwalebna y przyjemna przed oblicżem Bożym.
A która jeſt práwdźiwie wdowá y ośieroćiáła ; ma nádźieję w Bogu / y trwa w proźbách y w modlitwách w nocy y we dnie.
Ale która w roſkoƺách żyje / tá żyjąc / umárłą jeſt.
To tedy rozkázuj żeby były nienágánione.
A jeſli kto o ſwojich / á nawięcey o domowych ſtáránia nie ma / wiáry śię záprzał / y gorƺy jeſt niż niewierny.
Wdowá niech będźie obrána / któraby nie miáłá mniej ƺeśćidźieśiąt lat / która byłá żoną jednego mężá :
Májąca świádectwo w dobrych ucżynkách ; jeſli dźieći wychowáłá / jeſli gośćie przyjmowáłá / jeſli świętych nogi umywáłá / jeſli utrapionych wſpomagáłá / jeſli káżdego ucżynku dobrego náśládowáłá.
Wdów záśię młodƺych / chroń śię : Bo gdyby śię zbeſtwiły przećiw CHryſtuſowi / chcą zá mąż iść :
Májąc oſądzenie / iż pierwƺą wiárę odrzućiły :
Owƺem też próznując / ucżą śię chodźić od domu do domu : á nie tylko <i>ſą</i> próznujące / ále też świegotliwe / niepotrzebnymi rzecżámi śię báwiące / mówiąc co nieprzyſtoji.
Chcę tedy áby młodƺe ƺły zá mąż / dźieći rodźiły / goſpodyniámi były / przećiwnikowi żadney przycżyny nie dawáły ku obmowiſku.
Abowiem śię już niektóre obróćiły zá ƺátánem.
<i>Przetoż,</i> Jeſli który wierny álbo która wierna ma wdowy / niechże je opátruje : á niech Zbór nie będźie obćiążony / áby tym / które ſą prawdźiwie wdowy / doſtárcżyło.
Stárƺy / którzy śię w Przełożeńſtwie dobrze ſpráwują / niech będą miáni zá godne dwojákiey cżći : <i>á</i> zwłaƺcżá ći / którzy prácują w ſłowie y w náuce.
Abowiem piſmo mówi : Wołowi młócącemu nie záwiążeƺ gęby ; Y / Godźien jeſt robotnik zapłáty ſwojey.
Przećiwko Stárƺemu nie przyjmuj ſkárgi ; chybá zá dwiemá álbo trzemá świádkámi.
<i>A</i> te którzy grzeƺą / ſtrofuj przed wƺyſtkimi ; áby y drudzy bojaźń mieli.
Oświadcżam śię przed Bogiem y PAnem JEzuſem CHryſtuſem / y <i>przed</i> Anjoły wybránymi / ábyś tych rzecży przeſtrzegał / w oſobách nie brákując / nic nie cżyniąc z przychylnośći.
Rąku z prętká ná nikogo nie wkładaj / áni bądź ucżeſtnikiem cudzych grzechów. Sámego śiebie cżyſtym záchowaj.
<i>Sámey</i> wody więcey nie pijaj : ále używaj potroƺe winá dla żołądká twego y cżęſtych chorób twojich.
Grzechy niektórych ludźi przed tym ſą jáwne / y uprzedzáją ná ſąd : á zá niektórymi idą pozad.
Tákże też dobre ucżynki przed tym ſą jáwne : ále które ſą inákƺe / utájić śię niemogą.
