Stárƺemu nie łáj / ále jáko ojcá nápomináj / młodƺych jáko bráći /
Stárƺe niewiáſty jáko mátki / młodƺe jáko śioſtry / ze wƺeláką cżyſtośćią.
Wdowy miej w ucżćiwośći / które práwdźiwie ſą wdowámi.
A jeſli która wdowá dźieći álbo wnucżętá má / niech śię ucżą pierwej przećiwko domowi włáſnemu być pobożnemi / y wzájem oddáwáć rodźicom : ábowiem to jeſt rzecż chwálebná y przyjemná przed oblicżem Bożem.
A która jeſt práwdźiwie wdowá y ośieroćiáłá / má nádźieję w Bogu / y trwá w prośbách y w modlitwách w nocy y we dnie.
Ale która w rozkoƺách żyje / tá żyjąc umárłą jeſt.
To tedy rozkázuj / żeby były nienágánione.
A jeſli kto o ſwoich / á nájwięcey o domowych ſtárániá nie má / wiáry śię záprzáł / y gorƺy jeſt niż niewierny.
Wdowá niech będźie obráná / któráby nie miáłá mniej niż ƺeśćdźieśiąt lát / która byłá żoną jednego mężá /
Májącá świádectwo w dobrych ucżynkách / jeſli dźieći wychowáłá / jeſli gośći przyjmowáłá / jeſli świętych nogi umywáłá / jeſli utrápionych wſpomágáłá / jeſli káżdego ucżynku dobrego náśládowáłá.
Wdów záśię młodƺych chroń śię ; bo gdyby śię zbeſtwiły przećiw CHryſtuſowi / chcą zá mąż iść /
Májąc oſądzenie / iż pierwƺą wiárę odrzućiły ;
Owƺem też próżnując ucżą śię / chodźić od domu do domu ; á nie tylko ſą próżnujące / ále też świegotliwe / niepotrzebnemi rzecżámi śię báwiące / mówiąc / co nie przyſtoi.
Chcę tedy / áby młodƺe ƺły zá mąż / dźieći rodźiły / goſpodyniámi były ; przećiwnikowi żádney przycżyny nie dáwáły ku obmowiſku ;
Abowiem śię już niektóre obróćiły zá ƺátánem.
Przetoż / jeſli który wierny álbo która wierná má wdowy / niechże je opátruje / á niech zbór nie będźie obćiążony / áby tym / które ſą práwdźiwie wdowámi / ſtárcżyło.
Stárśi / którzy śię w przełożeńſtwie dobrze ſpráwują / niech będą mieni zá godnych dwojákiej cżći / á zwłáƺcżá ći / którzy prácują w ſłowie y w náuce.
Abowiem Piſmo mówi : Wołowi młócącemu nie záwiążeƺ gęby ; y : Godźien jeſt robotnik zápłáty ſwojey.
Przećiwko ſtárƺemu nie przyjmuj ſkárgi / chybá zá dwomá álbo trzemá świádkámi.
A tych / którzy grzeƺą / ſtrofuj przed wƺyſtkimi / áby y drudzy bojáźń mieli.
Oświádcżám śię przed Bogiem y Pánem JEzuſem CHryſtuſem / y przed Anioły wybránymi / ábyś tych rzecży przeſtrzegáł / w oſobách nie brákując / nic nie cżyniąc z przychylnośći.
Rąk z prędká ná nikogo nie wkłádáj / áni bądź ucżeſtnikiem cudzych grzechów : ſámego śiebie cżyſtym záchowáj.
Sámej wody więcey nie pijáj / ále używáj po troƺe winá dla żołądká twego y cżęſtych chorób twoich.
Grzechy niektórych ludźi przedtem ſą jáwne / y uprzedzáją ná ſąd / á zá niektórymi idą pozád.
Tákże też dobre ucżynki przedtem ſą jáwne ; ále które ſą inƺe / utáić śię nie mogą.
