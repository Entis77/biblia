Páweł Apoſtoł JEzuſá CHryſtuſá / podług rozrządzenia Bogá Zbáwićielá náƺego / y PAná JEzuſá CHryſtuſá / który jeſt nádźiejá náƺá /
Tymoteuƺowi / właſnemu ſynowi w wierze / <i>niech będźie</i> łáſká / miłośierdźie / pokój od Bogá Ojcá náƺego / y CHryſtuſá JEzuſá PAná náƺego.
Jákom ćię prośił / ábyś zoſtał w Efeźie / gdym ƺedł do Mácedoniey / <i>pátrzże</i> ábyś rozkazał niektórym / żeby inácżey nie ucżyli :
Y nie báwili śię baśniámi y wywodámi nieſkońcżonymi rodzáju / które więcey ſporów przynoƺą niż zbudowánia Bożego / które w wierze <i>zależy</i>.
Lecż koniec przykazánia jeſt miłość z cżyſtego ſercá / y z ſumnienia dobrego / y z wiáry nieobłudney.
Cżego niektórzy jáko celu uchybiwƺy / obróćili śię ku próznomównośći :
Chcąc być náucżyćielámi zakonu / nie rozumieją áni tego co mówią / áni co zá pewne twirdzą.
A wiemy że dobry jeſt zakon / jeſliby go kto przyſtojnie używał.
Wiedząc to / że ſpráwiedliwemu nie jeſt zakon poſtánowiony ; ále nieſpráwiedliwym / y niepoddánym / niepobożnym / y grzeƺnikom / złośliwym / y niecżyſtym / ojcomordercom / y mátkomordercom / mężobójcom :
Wƺetecżnikom / ſámcołożnikom / ludokrádcom / kłamcom / krzywoprzyśięſcom ; y jeſli co innego <i>jeſt</i> przećiwnego zdrowey náuce.
Według chwalebney Ewángeliey błogoſłáwionego Bogá / która mnie jeſt zwierzona.
Przetoż dźiękuję temu który mię umocnił / CHryſtuſowi JEzuſowi PAnu náƺemu ; iż mię zá wiernego oſądźił / ná uſługowánie poſtánowiwƺy <i>mię</i>.
Którym pierwey był bluźnierzem / y prześládowcą / y gwałtownikiem : Alem miłośierdźia doſtąpił / bom to z niewiádomośći cżynił <i>będąc</i> w niewierze.
Lecż náder obfitowáłá łáſká PAná náƺego z wiárą y z miłośćią / <i>która jeſt</i> w CHryſtuśie JEzuśie.
Wierna <i>jeſt</i> tá mowá / y wƺelkiego przyjęćia godna / iż CHryſtus JEzus przyƺedł ná świát áby grzeƺniki zbáwił / z których jam jeſt pierwƺy.
Alem dla tego miłośierdźia doſtąpił / áby ná mnie pierwƺym okazał JEzus CHryſtus wƺelką ćierpliwość / ná przykład tym którzy weń uwierzyć máją ku żywotowi wiecżnemu.
Przetoż Królowi wieków nieśmiertelnemu / niewidźiálnemu / ſámemu mądremu Bogu / <i>niech będźie</i> cżeść y chwałá ná wieki wieków / Amen.
Toć rozkazánie zlecam ſynu Tymoteuƺu / ábyś według proroctw / które uprzedźiły o tobie / bojował w nich on dobry bój :
Májąc wiárę y dobre ſumnienie / które niektórzy odrzućiwƺy / ƺkodę podjęli w wierze.
Z których jeſt Hymeneus / y Alexánder : którem oddał ƺátánowi / áby pokaráni będąc / náucżyli śię nie bluźnić.
