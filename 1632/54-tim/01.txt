Páweł / Apoſtoł JEzuſá CHryſtuſá podług rozrządzeniá Bogá / zbáwićielá náƺego / y Páná JEzuſá CHryſtuſá / który jeſt nádźiejá náƺá /
Tymoteuƺowi / włáſnemu ſynowi w wierze / niech będźie łáſká / miłośierdźie / pokój od Bogá / Ojcá náƺego / y CHryſtuſá JEzuſá / Páná náƺego.
Jákom ćię prośił / ábyś zoſtáł w Efeźie / gdym ƺedł do Mácedonii / pátrzże / ábyś rozkázáł niektórym / żeby inácżej nie ucżyli /
Y nie báwili śię báśniámi y wywodámi nieſkońcżonemi rodzáju / które więcey ſporów przynoƺą / niż zbudowániá Bożego / które w wierze záleży.
Lecż koniec przykázániá jeſt miłość z cżyſtego ſercá y z ſumieniá dobrego y z wiáry nieobłudney.
Cżego niektórzy jáko celu uchybiwƺy / obróćili śię ku próżnomównośći.
Chcąc być náucżyćielámi zakonu / nie rozumieją áni tego / co mówią / áni co zá pewne twierdzą.
A wiemy / że dobry jeſt zakon / jeſliby go kto przyſtojnie używáł /
Wiedząc to / że ſpráwiedliwemu nie jeſt zakon poſtánowiony / ále nieſpráwiedliwym y niepoddánym / niepobożnym y grzeƺnikom / złośliwym y niecżyſtym / ojcomordercom y mátkomordercom / mężobójcom /
Wƺetecżnikom / ſámcołożnikom / ludokrádcom / kłámcom / krzywoprzyśiężcom / y jeſli co innego jeſt przećiwnego zdrowej náuce.
Według chwálebney Ewángielii błogoſłáwionego Bogá / która mi jeſt zwierzoná.
Przetoż dźiękuję temu / który mię umocnił / CHryſtuſowi JEzuſowi / Pánu náƺemu / iż mię zá wiernego oſądźił / ná uſługiwánie poſtánowiwƺy mię.
Którym pierwej był bluźniercą y prześládowcą y gwáłtownikiem ; álem miłośierdźiá doſtąpił / bom to z niewiádomośći cżynił / będąc w niewierze.
Lecż náder obfitowáłá łáſká Páná náƺego z wiárą y z miłośćią / która jeſt w CHryſtuśie JEzuśie.
Wierná jeſt tá mowá y wƺelkiego przyjęćiá godná / iż CHryſtuſ JEzuſ przyƺedł ná świát / áby grzeƺników zbáwił / z których jám jeſt pierwƺy.
Alem dlátego miłośierdźiá doſtąpił / áby ná mnie pierwƺym okázáł JEzuſ CHryſtuſ wƺelką ćierpliwość ná przykłád tym / którzy weń uwierzyć máją ku żywotowi wiecżnemu.
Przetoż królowi wieków nieśmiertelnemu / niewidźiálnemu ſámemu mądremu Bogu / niech będźie cżeść y chwałá ná wieki wieków. Amen.
Toć rozkázánie zálecám / ſynu Tymoteuƺu! ábyś według proroctw / które uprzedźiły o tobie / bojowáł w nich on dobry bój /
Májąc wiárę y dobre ſumienie / które niektórzy odrzućiwƺy / ƺkodę podjęli w wierze ;
Z których jeſt Hymeneuſ y Alekſánder / którychem oddáł ƺátánowi / áby pokáráni będąc / náucżyli śię nie bluźnić.
