Páweł / Apoſtoł Jezuſá Chryſtuſá podług rozrządzeniá Bogá / zbáwićielá náƺego / y Páná Jezuſá Chryſtuſá / który jeſt nádziejá náƺá /
Tymoteuƺowi / włáſnemu ſynowi w wierze / niech będzie łáſká / miłośierdzie / pokój od Bogá / Ojcá náƺego / y Chryſtuſá Jezuſá / Páná náƺego.
Jákom ćię prośił / ábyś zoſtáł w Efezie / gdym ƺedł do Mácedonii / pátrzże / ábyś rozkázáł niektórym / żeby inácżej nie ucżyli /
Y nie báwili śię báśniámi y wywodámi nieſkońcżonemi rodzáju / które więcej ſporów przynoƺą / niż zbudowániá Bożego / które w wierze záleży.
Lecż koniec przykázániá jeſt miłość z cżyſtego ſercá y z ſumieniá dobrego y z wiáry nieobłudnej.
Cżego niektórzy jáko celu uchybiwƺy / obróćili śię ku próżnomównośći.
Chcąc być náucżyćielámi zákonu / nie rozumieją áni tego / co mówią / áni co zá pewne twierdzą.
A wiemy / że dobry jeſt zákon / jeźliby go kto przyſtojnie używáł /
Wiedząc to / że ſpráwiedliwemu nie jeſt zákon poſtánowiony / ále nieſpráwiedliwym y niepoddánym / niepobożnym y grzeƺnikom / złośliwym y niecżyſtym / ojcomordercom y mátkomordercom / mężobójcom /
Wƺetecżnikom / ſámcołożnikom / ludokrádcom / kłámcom / krzywoprzyśiężcom / y jeźli co innego jeſt przećiwnego zdrowej náuce.
Według chwálebnej Ewángielii błogoſłáwionego Bogá / którá mi jeſt zwierzoná.
Przetoż dziękuję temu / który mię umocnił / Chryſtuſowi Jezuſowi / Pánu náƺemu / iż mię zá wiernego oſądził / ná uſługiwánie poſtánowiwƺy mię.
Którym pierwej był bluźniercą y prześládowcą y gwáłtownikiem ; álem miłośierdziá doſtąpił / bom to z niewiádomośći cżynił / będąc w niewierze.
Lecż náder obfitowáłá łáſká Páná náƺego z wiárą y z miłośćią / którá jeſt w Chryſtuśie Jezuśie.
Wierná jeſt tá mowá y wƺelkiego przyjęćiá godná / iż Chryſtuſ Jezuſ przyƺedł ná świát / áby grzeƺników zbáwił / z których jám jeſt pierwƺy.
Alem dlátego miłośierdziá doſtąpił / áby ná mnie pierwƺym okázáł Jezuſ Chryſtuſ wƺelką ćierpliwość ná przykłád tym / którzy weń uwierzyć máją ku żywotowi wiecżnemu.
Przetoż królowi wieków nieśmiertelnemu / niewidziálnemu ſámemu mądremu Bogu / niech będzie cżeść y chwáłá ná wieki wieków. Amen.
Toć rozkázánie zálecám / ſynu Tymoteuƺu! ábyś według proroctw / które uprzedziły o tobie / bojowáł w nich on dobry bój /
Májąc wiárę y dobre ſumienie / które niektórzy odrzućiwƺy / ƺkodę podjęli w wierze ;
Z których jeſt Hymeneuſ y Alekſánder / którychem oddáł ƺátánowi / áby pokáráni będąc / náucżyli śię nie bluźnić.
