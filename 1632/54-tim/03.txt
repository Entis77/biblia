Wierna jeſt <i>tá</i> mowá : Jeſli kto Biſkupſtwá żąda / dobrey prace żąda.
Ale Biſkup ma bydź nienágániony / mąż jedney żony / cżuły / trzeźwi / poważny / gośćinny / ku náucżániu ſpoſobny ;
Nie pijánicá winá / nie bitny / nie ſproſnego zyſku chćiwy : ále ſłuƺność miłujący / nie ſwarliwy / nie łákomy :
Któryby dom ſwój dobrze rządźił / któryby dźiatki miał w poddáńſtwie / ze wƺeláką ucżćiwośćią :
( Bo jeſliby kto nie umiał ſwego właſnego domu rządźić / jákoż piecżą będźie miał o Kośćiele Bożym? )
Nie nowotny ; áby będąc nádęty / nie wpadł w ſąd potwarce.
Muśi też mieć świádectwo dobre od obcych / áby nie wpadł w háńbę y w śidło potwarce.
Dyákonowie tákże <i>máją być</i> poważni / nie dwojiſtego ſłowá / nie pijánicy wiela winá / nie chćiwi ſproſnego zyſku :
Májący tájemnicę wiáry w cżyſtym ſumnieniu.
A ći też niech będą pierwey doświadcżeni ; zátym niech ſłużą jeſli ſą bez nágány.
Żony tákże <i>niech máją</i> poważne / nie potwarliwe / trzeźwie / wierne we wƺyſtkim.
Dyákonowie niech będą mężámi jedney żony / którzyby dźiatki dobrze rządźili / y właſne domy.
Abowiem którzyby dobrze ſłużyli / ſtopień ſobie dobry zjednáją / y wielkie beſpiecżeńſtwo w wierze / która jeſt w CHryſtuśie JEzuśie.
Toć tobie piƺę / májąc nádźieję że w rychle przydę do ćiebie.
A jeſlibym omięƺkał / ábyś wiedźiał jáko śię maƺ w domu Bożym ſpráwowáć / który jeſt Kośćiołem Bogá żywego / filarem y utwirdzeniem prawdy.
A záprawdę wielka jeſt tájemnicá pobożnośći / <i>że</i> Bóg objáwiony jeſt w ćiele / uſpráwiedliwiony jeſt w Duchu / widźiány jeſt od Anjołów / kazány jeſt Pogánom / uwierzono mu ná świećie / wźięty jeſt wzgórę do chwały.
