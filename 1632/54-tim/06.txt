Którzykolwiek ſłudzy ſą pod járzmem / niech rozumieją pánów ſwych godnych być wƺelákiej cżći / áby imię Boże y náuká nie byłá bluźnioná.
A którzy máją pánów wiernych / niech nimi nie gárdzą / dlátego / iż ſą bráćmi / ále tem rácżej niech ſłużą / iż ſą wierni y mili / dobrodziejſtwá Bożego ucżeſnicy. Tego náucżáj y do tego upomináj.
Jeźli kto inácżej ucży / á nie przyſtępuje do zdrowych mów Páná náƺego Jezuſá Chryſtuſá / y do tej náuki / którá jeſt według pobożnośći /
Táki nádęty jeſt y nic nie umie / ále ƺáleje około gádek y ſporów o ſłowá / z których pochodzi zázdrość / ſwár / złorzecżeniá / złe podejrzeniá /
Przewrotne ćwicżeniá ludzi umyſłu ſkáżonego / y którzy pozbáwieni ſą práwdy / którzy rozumieją / że pobożność jeſt zyſkiem ćieleſnym ; odſtąpże od tákich.
A jeſtći wielki zyſk pobożność z przeſtáwániem ná ſwem ;
Abowiem niceśmy nie przynieśli ná ten świát / bez pochyby że też wynieść nic nie możemy ;
Ale májąc żywność y odzienie / ná tem przeſtáwáć mámy.
Bo którzy chcą bogátymi być / wpádáją w pokuƺenie y w śidło y w wiele głupich y ƺkodliwych pożądliwośći / które pogrążáją ludzi ná zátrácenie y zginienie.
Abowiem korzeń wƺyſtkiego złego jeſt miłość pieniędzy / których niektórzy prágnąc / pobłądzili od wiáry y poprzebijáli śię wielomá boleśćiámi.
Ale ty / cżłowiecże Boży! chroń śię tákich rzecży / á náśláduj ſpráwiedliwośći / pobożnośći / wiáry / miłośći / ćierpliwośći / ćichośći.
Bojuj on dobry bój wiáry / chwyć śię żywotá wiecżnego / do któregoś też powołány / y wyznáłeś dobre wyznánie przed wielomá świádkámi.
Rozkázujęć przed Bogiem / który wƺyſtko ożywiá / y przed Chryſtuſem Jezuſem / który oświádcżył przed Ponckim Piłátem dobre wyznánie /
Abyś záchowáł to przykázánie / będąc bez zmázy / bez nágány / áż do objáwieniá Páná náƺego Jezuſá Chryſtuſá /
Które cżáſów ſwoich okáże on błogoſłáwiony / y ſám możny król królujących / y Pán pánujących ;
Który ſám má nieśmiertelność / y mieƺká w świátłośći nieprzyſtępnej / którego nie widziáł żáden z ludzi / áni widzieć może ; któremu niech będzie cżeść y moc wiecżná. Amen.
Bogácżom w tym teráźniejƺym wieku rozkáż / áby nie byli wyſokomyślnymi / áni nádziei pokłádáli w bogáctwie niepewnem / ále w Bogu żywym / który nám wƺyſtkiego obfićie ku używániu dodáje :
Aby innym dobrze cżynili / w ucżynki dobre bogátymi byli / rádzi dáwáli / á rádzi udzieláli /
Skárbiąc ſámi ſobie grunt dobry ná przyƺły cżáſ / áby otrzymáli żywot wiecżny.
O Tymoteuƺu! ſtrzeż tego / cżegoć śię powierzono / á brzydź śię świecką próżnomównośćią y ſprzecżániem około fáłƺywie názwánej umiejętnośći /
Którą śię niektórzy ƺcżycąc z ſtrony wiáry / celu uchybili. Łáſká niech będzie z tobą. Amen.
