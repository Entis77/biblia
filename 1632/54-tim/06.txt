Którzykolwiek ſłudzy ſą pod járzmem / niech rozumieją pánów ſwych godnych być wƺelákiej cżći / áby imię Boże y náuká nie byłá bluźnioná.
A którzy máją pánów wiernych / niech nimi nie gárdzą / dlátego / iż ſą bráćmi / ále tem rácżej niech ſłużą / iż ſą wierni y mili / dobrodźiejſtwá Bożego ucżeſnicy. Tego náucżáj y do tego upomináj.
Jeſli kto inácżej ucży / á nie przyſtępuje do zdrowych mów Páná náƺego JEzuſá CHryſtuſá / y do tey náuki / która jeſt według pobożnośći /
Táki nádęty jeſt y nic nie umie / ále ƺáleje około gádek y ſporów o ſłowá / z których pochodźi zázdrość / ſwár / złorzecżeniá / złe podejrzeniá /
Przewrotne ćwicżeniá ludźi umyſłu ſkáżonego / y którzy pozbáwieni ſą práwdy / którzy rozumieją / że pobożność jeſt zyſkiem ćieleſnym ; odſtąpże od tákich.
A jeſtći wielki zyſk pobożność z przeſtáwániem ná ſwem ;
Abowiem niceśmy nie przynieśli ná ten świát / bez pochyby że też wynieść nic nie możemy ;
Ale májąc żywność y odźienie / ná tem przeſtáwáć mámy.
Bo którzy chcą bogátymi być / wpádáją w pokuƺenie y w śidło y w wiele głupich y ƺkodliwych pożądliwośći / które pogrążáją ludźi ná zátrácenie y zginienie.
Abowiem korzeń wƺyſtkiego złego jeſt miłość pieniędzy / których niektórzy prágnąc / pobłądźili od wiáry y poprzebijáli śię wielomá boleśćiámi.
Ale ty / cżłowiecże Boży! chroń śię tákich rzecży / á náśláduj ſpráwiedliwośći / pobożnośći / wiáry / miłośći / ćierpliwośći / ćichośći.
Bojuj on dobry bój wiáry / chwyć śię żywotá wiecżnego / do któregoś też powołány / y wyznáłeś dobre wyznánie przed wielomá świádkámi.
Rozkázujęć przed Bogiem / który wƺyſtko ożywiá / y przed CHryſtuſem JEzuſem / który oświádcżył przed Ponckim Piłátem dobre wyznánie /
Abyś záchowáł to przykázánie / będąc bez zmázy / bez nágány / áż do objáwieniá Páná náƺego JEzuſá CHryſtuſá /
Które cżáſów ſwoich okáże on błogoſłáwiony / y ſám możny król królujących / y Pán pánujących ;
Który ſám má nieśmiertelność / y mieƺká w świátłośći nieprzyſtępney / którego nie widźiáł żáden z ludźi / áni widźieć może ; któremu niech będźie cżeść y moc wiecżná. Amen.
Bogácżom w tym teráźniejƺym wieku rozkáż / áby nie byli wyſokomyślnymi / áni nádźiei pokłádáli w bogáctwie niepewnem / ále w Bogu żywym / który nám wƺyſtkiego obfićie ku używániu dodáje :
Aby innym dobrze cżynili / w ucżynki dobre bogátymi byli / rádźi dáwáli / á rádźi udźieláli /
Skárbiąc ſámi ſobie grunt dobry ná przyƺły cżáſ / áby otrzymáli żywot wiecżny.
O Tymoteuƺu! ſtrzeż tego / cżegoć śię powierzono / á brzydź śię świecką próżnomównośćią y ſprzecżániem około fáłƺywie názwáney umiejętnośći /
Którą śię niektórzy ƺcżycąc z ſtrony wiáry / celu uchybili. Łáſká niech będźie z tobą. Amen.
