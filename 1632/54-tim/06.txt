Którzykolwiek ſłudzy ſą pod járzmem / niech rozumieją Pány ſwe godne <i>być</i> wƺelákiey cżći : áby Imię Boże y náuká nie byłá bluźniona.
A którzy máją Pány wierne / niech <i>nimi</i> nie gárdzą / dla tego iż ſą bráćią : ále tym rácżey niech ſłużą iż ſą wierni y mili / dobrodźiejſtwá <i>Bożego</i> ucżeſnicy. Tego náucżaj / y <i>do tego</i> upominaj.
Jeſli kto inácżey ucży / á nie przyſtępuje do zdrowych mów PAná náƺego JEzuſá CHryſtuſá / y do tey náuki która jeſt według pobożnośći :
<i>Táki</i> nádęty jeſt y nic nie umie / ále ƺáleje około gadek y ſporów o ſłowá : z których pochodźi zazdrość / ſwar / złorzecżenia / złe podejrzenia :
Przewrotne ćwicżenia ludźi umyſłu ſkáżonego / y <i>którzy</i> pozbáwieni ſą prawdy ; którzy rozumieją / że pobożność jeſt zyſkiem <i>ćieleſnym</i>. Odſtąpże od tákich.
A jeſtći wielki zyſk pobożność / z przeſtawániem ná ſwym.
Abowiem niceſmy nie przynieśli ná ten świát : bez pochyby że też wynieść nic nie możemy.
Ale májąc żywność y odźienie / ná tym przeſtawáć mamy.
Bo którzy chcą bogátymi bydź / wpadáją w pokuƺenie / y w śidło / y w wiele głupich y ƺkodliwych pożądliwośći / które pogrążáją ludźie ná zátrácenie y zginienie.
Abowiem korzeń wƺyſtkiego złego / jeſt miłość pieniędzy : których niektórzy prágnąc / pobłądźili od wiáry / y poprzebijáli śię wielą boleśći.
Ale ty cżłowiecże Boży / chroń śię tákich rzecży / á náśláduj ſpráwiedliwośći / pobożnośći / wiáry / miłośći / ćierpliwośći / ćichośći.
Bojuj on dobry bój wiáry / chwyć śię żywotá wiecżnego ; do któregoś też powołány / y wyznałeś dobre wyznánie przed wielą świádków.
Rozkázujęć przed Bogiem / który wƺyſtko obżywiá / y przed CHryſtuſem JEzuſem / który oświadcżył przed Pontſkim Piłatem dobre wyznánie :
Abyś záchował to Przykazánie / <i>będąc</i> bez zmázy / bez nágány / áż do objáwienia PAná náƺego JEzuſá CHryſtuſá.
Które cżáſów ſwojich okaże on błogoſłáwiony / y ſam możny Król królujących y PAn pánujących :
Który ſam ma nieśmiertelność / <i>y</i> mieƺká w świátłośći nieprzyſtępney : którego nie widźiał żáden z ludźi / áni widźieć może : Któremu <i>niech będźie</i> cżęść y moc wiecżna / Amen.
Bogacżom w tym teráźniejƺym wieku rozkaż / áby nie byli wyſokomyślnymi / áni nádźieje pokładáli w bogáctwie niepewnym / ále w Bogu żywym / który nam wƺyſtkiego obfićie ku używániu dodawa.
Aby <i>innym</i> dobrze cżynili / w ucżynki dobre bogátymi byli / rádźi dawáli / y <i>rádźi</i> udźieláli.
Skárbiąc ſámi ſobie grunt dobry ná przyƺły <i>cżás,</i> áby otrzymáli żywot wiecżny.
O Tymoteuƺu! ſtrzeż <i>tego,</i> cżegoć śię powierzono / á brzydź śię świecką próznomównośćią y zprzecżániem <i>około</i> fałƺywie názwáney umiejętnośći.
Którą śię niektórzy ƺcżycąc zſtrony wiáry / celu uchybili. Łáſká <i>niech będźie</i> z tobą / Amen.
