Zmocnił śię tedy Sálomon / ſyn Dawidowy / w króleſtwie ſwem / á Pán / Bóg jego był z nim / y uwielbił go wyſoce.
Y przykázał Sálomon wƺyſtkiemu Izráelowi / półkownikom / rotmiſtrzom / y ſędźiom / tákże wƺyſtkim przełożonym nád wƺyſtkim Izráelem / y przedniejƺym domów ojcowſkich.
Y ƺedł Sálomon y wƺyſtko zgromádzenie z nim ná wyżynę / która byłá w Gábáonie ; ábowiem tám był námiot zgromádzenia Bożego / który ſpráwił Mojzeƺ ſługá Páńſki / ná puƺcży.
( Ale ſkrzynię Bożą przenióſł był Dawid z Káryjátyjárym / nágotowáwƺy jey miejſce ; bo jey był námiot rozbił w Jeruzálemie. )
Ołtárz też miedźiány / który był urobił Beſáleel / ſyn Urowy / ſyná Hurowego / był tám przed przybytkiem Páńſkim / gdźie Páná ƺukał Sálomon / y wƺyſtko zgromádzenie.
Y przyſtąpił tám Sálomon do ołtárzá miedźiánego / który był przed námiotem zgromádzenia y ofiárował ná nim ofiár pálonych tyśiąc.
Onejże nocy ukázał śię Bóg Sálomonowi / y rzekł do niego : Proś cżego chceƺ / á dam ći.
Tedy rzekł Sálomon do Bogá : Tyś ucżynił z Dawidem / ojcem moim / miłośierdźie wielkie / y poſtánowiłeś mię królem miáſto niego.
A teraz / o Pánie Boże! niech będźie ſtáłe ſłowo twoje / któreś mówił do Dawidá / ojcá mego ; boś ty mię ucżynił królem nád ludem wielkim / który jeſt jáko proch źiemi.
Przetoż dáj mi mądrość y umiejętność / ábym wychodźił y wchodźił przed tym ludem : ábowiem któż jeſt / coby mógł ſądźić ten lud twój ták wielki.
Tedy rzekł Bóg do Sálomoná : Dla tego / iżeś to miał w ſercu ſwem / á nie prośiłeś o bogáctwá / o májętnośći / y o ſłáwę / ániś prośił o wytrácenie tych / co ćię nienáwidzą / ániś też prośił o długie żyćie / áleś ſobie prośił o mądrość y umiejętność / ábyś ſądźił lud mój / nád którymem ćię poſtánowił królem :
Mądrość y umiejętność dáná jeſt tobie ; do tego dam ći bogáctwá / y májętność / y ſłáwę / ták / że żáden nie był tobie równy z królów / którzy byli przed tobą / y po tobie nie będźie tobie równy.
Y wróćił śię Sálomon od oney wyżyny / która byłá w Gábáonie / do Jeruzálemu / od namiotu zgromádzenia / y królował nád Izráelem.
A názbierał Sálomon wozów y jezdnych / y miał tyśiąc y cżtery ſtá wozów / y dwánaśćie tyśięcy jezdnych / których rozſádźił po miáſtách wozów / y przy ſobie w Jeruzálemie.
Y złożył król w Jeruzálemie złotá y ſrebrá / jáko kámieniá / á cedrów złożył jáko ſykomorów / których ná polu rośćie bárzo wiele.
Y przywodzono konie Sálomonowi z Egiptu / y rozlicżne towáry ; bo kupcy królewſcy bráli towáry rozlicżne zá pewne pieniądze.
A wychodźili y przywodźili z Egiptu cug woźników zá ƺeść ſet ſrebrników / á koniá zá ſto y zá pięćdźieśiąt. A ták wƺyſcy królowie Hetejſcy / y królowie Syryjſcy z rąk ich koni doſtáwáli.
