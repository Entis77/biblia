Zmocnił śię tedy Sálomon / ſyn Dawidowy / w króleſtwie ſwem / á Pán / Bóg jego był z nim / y uwielbił go wyſoce.
Y przykázáł Sálomon wƺyſtkiemu Izráelowi / półkownikom / rotmiſtrzom / y ſędźiom / tákże wƺyſtkim przełożonym nád wƺyſtkim Izráelem / y przedniejƺym domów ojcowſkich.
Y ƺedł Sálomon y wƺyſtko zgromádzenie z nim ná wyżynę / którá byłá w Gábáonie ; ábowiem tám był námiot zgromádzeniá Bożego / który ſpráwił Mojżeƺ ſługá Páńſki / ná puƺcży.
( Ale ſkrzynię Bożą przenióſł był Dawid z Káryjátyjárym / nágotowáwƺy jej miejſce ; bo jej był námiot rozbił w Jeruzálemie. )
Ołtárz też miedźiány / który był urobił Beſáleel / ſyn Urowy / ſyná Hurowego / był tám przed przybytkiem Páńſkim / gdźie Páná ƺukáł Sálomon / y wƺyſtko zgromádzenie.
Y przyſtąpił tám Sálomon do ołtárzá miedźiánego / który był przed námiotem zgromádzeniá y ofiárowáł ná nim ofiár pálonych tyśiąc.
Onejże nocy ukázáł śię Bóg Sálomonowi / y rzekł do niego : Proś cżego chceƺ / á dám ći.
Tedy rzekł Sálomon do Bogá : Tyś ucżynił z Dawidem / ojcem moim / miłośierdźie wielkie / y poſtánowiłeś mię królem miáſto niego.
A teráz / o Pánie Boże! niech będźie ſtáłe ſłowo twoje / któreś mówił do Dawidá / ojcá mego ; boś ty mię ucżynił królem nád ludem wielkim / który jeſt jáko proch źiemi.
Przetoż dáj mi mądrość y umiejętność / ábym wychodźił y wchodźił przed tym ludem : ábowiem któż jeſt / coby mógł ſądźić ten lud twój ták wielki.
Tedy rzekł Bóg do Sálomoná : Dlátego / iżeś to miáł w ſercu ſwem / á nie prośiłeś o bogáctwá / o májętnośći / y o ſłáwę / ániś prośił o wytrácenie tych / co ćię nienáwidzą / ániś też prośił o długie żyćie / áleś ſobie prośił o mądrość y umiejętność / ábyś ſądźił lud mój / nád którymem ćię poſtánowił królem :
Mądrość y umiejętność dáná jeſt tobie ; do tego dám ći bogáctwá / y májętność / y ſłáwę / ták / że żáden nie był tobie równy z królów / którzy byli przed tobą / y po tobie nie będźie tobie równy.
Y wróćił śię Sálomon od onej wyżyny / którá byłá w Gábáonie / do Jeruzálemu / od námiotu zgromádzeniá / y królowáł nád Izráelem.
A názbieráł Sálomon wozów y jezdnych / y miáł tyśiąc y cżtery ſtá wozów / y dwánáśćie tyśięcy jezdnych / których rozſádźił po miáſtách wozów / y przy ſobie w Jeruzálemie.
Y złożył król w Jeruzálemie złotá y ſrebrá / jáko kámieniá / á cedrów złożył jáko ſykomorów / których ná polu rośnie bárdzo wiele.
Y przywodzono konie Sálomonowi z Egiptu / y rozlicżne towáry ; bo kupcy królewſcy bráli towáry rozlicżne zá pewne pieniądze.
A wychodźili y przywodźili z Egiptu cug woźników zá ƺeść ſet ſrebrników / á koniá zá ſto y zá pięćdźieśiąt. A ták wƺyſcy królowie Hetejſcy / y królowie Syryjſcy z rąk ich koni doſtáwáli.
