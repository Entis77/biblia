Tedy poſtánowili królem obywátele Jeruzálemſcy Ochozyjáƺá / ſyná jego nájmłodƺego / miáſto niego ; bo wƺyſtkich ſtárƺych pobiłá byłá kupá ſwáwolná / która byłá przyƺłá z Arábcżykámi do obozu / á ták królował Ochozyjáƺ / ſyn Jorámá / królá Judzkiego.
Cżterdźieśći y dwá látá było Ochozyjáƺowi / gdy pocżął królowáć / á jeden rok królował w Jeruzálemie ; á imię mátki jego było Atálijá / córká Amrego.
Ten też chodźił drogámi domu Achábowego ; bo mátká jego rádźiłá mu / áby śię niepobożnie ſpráwował.
Przetoż cżynił złe przed ocżymá Páńſkimi / jáko dom Achábowy ; tych bowiem miał zá rádców ſwojich po śmierći ojcá ſwego / ná zginienie ſwoje.
Bo według rády ich chodźił / y jechał z Jorámem / ſynem Achábowym / królem Izráelſkim / ná wojnę przećiw Házáelowi / królowi Syryjſkiemu / do Rámot Gáláádſkiego / kędy zránili Syryjcżycy Jorámá.
A gdy śię wróćił / áby śię lecżył w Jezreelu / ( ábowiem miał rány / które mu zádáno w Rámie / gdy śię potykał z Házáelem / królem Syryjſkim ) tedy Azáryjáƺ / ſyn Jorámá / królá Judzkiego / przyjechał do Jezreelá / náwiedzáć niemocnego Jorámá / ſyná Achábowego / bo był chory.
A było to od Bogá ná upádek Ochozyjáƺowi / że przyjechał do Jorámá. Abowiem przyjecháwƺy wyjechał z Jorámem przećiw Jehu / ſynowi Námſy / którego był pomázał Pán / áby wytráćił dom Achábowy.
A gdy ſąd wykonywał Jehu nád domem Achábowym / ználázł niektórych kśiążąt Judzkich / y ſynów bráći Ochozyjáƺowych / którzy ſłużyli Ochozyjáƺowi / y pobił ich.
Potem ƺukał Ochozyjáƺá / y pojmáli go / gdy śię krył w Sámáryi / á przywiódƺy go do Jehu / zábili go / y pogrzebli go / bo mówili : Synći to jeſt Jozáfátá / który ƺukał Páná cáłem ſercem ſwojim. A ták nie było nikogo w domu Ochozyjáƺowym / któryby mógł otrzymáć króleſtwo.
Przetoż Atálijá / mátká Ochozyjáƺowá / widząc / że umárł ſyn jey / wſtáwƺy wytráćiłá wƺyſtko naśienie królewſkie z domu Judy.
Ale Joſábet / córká królewſká / wźięłá Joázá / ſyná Ochozyjáƺowego / y wykrádłá go z pośród ſynów królewſkich / których zábijáno / y ſchowáłá go y mámkę jego do gmáchu / gdźie były łożá. Y ſkryłá go Joſábet / córká królá Jorámá / żoná Jojády kápłáná / ( bo oná byłá śioſtrą Ochozyjáƺową ) przed Atáliją / áby go nie zábiłá.
Y był z nimi w domu Bożym / będąc ſkrytym przez ƺeść lat / których Atálijá królowáłá nád tą źiemią.
