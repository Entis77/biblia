We dwunáſtym roku był Mánáſes / gdy królowáć pocżął ; á pięćdźieśiąt y pięć lat królował w Jeruzálemie.
Ten cżynił złe przed ocżymá Páńſkimi według obrzydłośći onych národów / które wygnał Pán przed oblicżem ſynów Izráelſkich.
Abowiem znowu pobudował wyżyny / które był poburzył Ezechyjáƺ / oćiec jego ; wyſtáwił też y ołtárze Báálom / á náſádźił gájów / y kłániał śię wƺyſtkiemu wojſku niebieſkiemu / á ſłużył mu /
Pobudował też ołtárze w domu Páńſkim / o którym powiedźiał był Pán : W Jeruzálemie będźie imię moje ná wieki.
Nádto nábudował ołtárze wƺyſtkiemu wojſku niebieſkiemu we dwu śieniách domu Páńſkiego.
Y przewodźił ſynów ſwych przez ogień w dolinie ſynów Hennomowych ; nádto cżáſów przeſtrzegał / y báwił śię wieƺcżbą y cżárnokśięſtwem / á uſtáwił cżárnokśiężników y guślárzy / y bárzo wiele złego cżynił przed ocżymá Páńſkimi / dráźniąc go.
Poſtáwił tákże báłwáná rytego / którego był ucżynił w domu Bożym / o którym był rzekł Bóg do Dawidá y do Sálomoná / ſyná jego : W domu tym y w Jeruzálemie / którem obrał ze wƺyſtkich pokoleń Izráelſkich / położę imię moje ná wieki ;
A nie dopuƺcżę śię więcey ruƺyć nodze Izráelá z źiemi / którąm náznácżył ojcom wáƺym / by jedno ſtrzegli y ſpráwowáli śię według wƺyſtkiego / com im rozkázał / według wƺyſtkiego zakonu / y uſtáw / y ſądów wydánych przez Mojżeƺá.
Ale Mánáſes zwiódł Judę y obywáteli Jeruzálemſkich / ták iż śię gorzej ſpráwowáli niż národy / które Pán wygłádźił przed oblicżem ſynów Izráelſkich.
Bo choć mówił Pán do Mánáſeſá / y do ludu jego / przećież oni nie ſłucháli.
Przetoż Pán náwiódł ná nich hetmánów wojſká królá Aſſyryjſkiego / którzy pojmáwƺy Mánáſeſá w ćierniu / y związáwƺy go dwomá łáńcuchámi / záwiedli go do Bábilonu.
Który będąc ućiśniony / modlił śię Pánu / Bogu ſwemu / y upokorzył śię bárzo przed oblicżem Bogá ojców ſwojich /
Y prośił go ; á dał mu śię uprośić / y wyſłuchał modlitwę jego / á przywróćił go do Jeruzálemu ná króleſtwo jego. Tedy poznał Mánáſes / iż ſám Pán jeſt Bogiem.
Potem budował mur około miáſtá Dawidowego ku záchodniej ſtronie Gichonu potoku áż do wejśćiá do brámy rybney / y otocżył murem Ofel / y wywiódł go bárzo wyſoko ; poſtánowił też hetmánów po wƺyſtkich miáſtách obronnych w Judźie.
Znióſł też bogów cudzych / y báłwáná z domu Páńſkiego / y wƺyſtkie ołtárze / które był pobudował ná górze domu Páńſkiego / y w Jeruzálemie / y wyrzućił zá miáſto.
Zátem nápráwił ołtárz Páńſki / y ſpráwował ná nim ofiáry ſpokojne / y dźiękcżynieniá / á przykázał Judźie / áby ſłużyli Pánu / Bogu Izráelſkiemu.
Wƺákże jeƺcże lud ofiárował ná wyżynách / lecż tylko Pánu / Bogu ſwemu.
Ale inne ſpráwy Mánáſeſowe / y modlitwá jego do Bogá jego / y ſłowá widzących / którzy máwiáli do niego w imię Páná / Bogá Izráelſkiego / ſą w kśięgách ſpráw królów Izráelſkich.
Modlitwá záś jego / y jáko jeſt wyſłuchány / y káżdy grzech jego / y przeſtępſtwo jego / y miejſcá / ná których był pobudował wyżyny / y wyſtáwił gáje święcone / y báłwány / przedtem niż śię był upokorzył / zápiſáne w kśięgách Chozájá.
Potem záſnął Mánáſes z ojcámi ſwymi / y pochowáli go w domu jego ; á Amon / ſyn jego / królował miáſto niego.
We dwudźieſtu y dwu látách był Amon / gdy królowáć pocżął / á dwá látá królował w Jeruzálemie.
Y cżynił złe przed ocżymá Páńſkimi / jáko cżynił Mánáſes / oćiec jego ; ábowiem wƺyſtkim báłwánom / których był nácżynił Mánáſes / oćiec jego / ofiárował Amon / y ſłużył im.
A nie upokorzył śię przed oblicżem Páńſkim / jáko śię upokorzył Mánáſes / oćiec jego ; owƺem ten Amon dáleko więcey grzeƺył.
Y ſprzyśięgli śię przećiw niemu ſłudzy jego / y zábili go w domu jego.
Ale lud oney źiemi pobił wƺyſtkich / co śię byli ſprzyśięgli przećiw królowi Amonowi ; á poſtánowił lud źiemi królem Jozyjáƺá / ſyná jego / miáſto niego.
