We dwunáſtym roku był Mánáſeſ / gdy królowáć pocżął ; á pięćdźieśiąt y pięć lát królowáł w Jeruzálemie.
Ten cżynił złe przed ocżymá Páńſkimi według obrzydłośći onych národów / które wygnáł Pán przed oblicżem ſynów Izráelſkich.
Abowiem znowu pobudowáł wyżyny / które był poburzył Ezechyjáƺ / ojćiec jego ; wyſtáwił też y ołtárze Báálom / á náſádźił gájów / y kłániáł śię wƺyſtkiemu wojſku niebieſkiemu / á ſłużył mu /
Pobudowáł też ołtárze w domu Páńſkim / o którym powiedźiáł był Pán : W Jeruzálemie będźie imię moje ná wieki.
Nádto nábudowáł ołtárze wƺyſtkiemu wojſku niebieſkiemu we dwóch śieniách domu Páńſkiego.
Y przewodźił ſynów ſwych przez ogień w dolinie ſynów Hennomowych ; nádto cżáſów przeſtrzegáł / y báwił śię wieƺcżbą y cżárnokśięſtwem / á uſtáwił cżárnokśiężników y guślárzy / y bárdzo wiele złego cżynił przed ocżymá Páńſkimi / dráźniąc go.
Poſtáwił tákże báłwáná rytego / którego był ucżynił w domu Bożym / o którym był rzekł Bóg do Dawidá y do Sálomoná / ſyná jego : W domu tym y w Jeruzálemie / którem obráł ze wƺyſtkich pokoleń Izráelſkich / położę imię moje ná wieki ;
A nie dopuƺcżę śię więcey ruƺyć nodze Izráelá z źiemi / którąm náznácżył ojcom wáƺym / by jedno ſtrzegli y ſpráwowáli śię według wƺyſtkiego / com im rozkázáł / według wƺyſtkiego zakonu / y uſtáw / y ſądów wydánych przez Mojżeƺá.
Ale Mánáſeſ zwiódł Judę y obywáteli Jeruzálemſkich / ták iż śię gorzej ſpráwowáli niż národy / które Pán wygłádźił przed oblicżem ſynów Izráelſkich.
Bo choć mówił Pán do Mánáſeſá / y do ludu jego / przećież oni nie ſłucháli.
Przetoż Pán náwiódł ná nich hetmánów wojſká królá Aſſyryjſkiego / którzy pojmáwƺy Mánáſeſá w ćierniu / y związáwƺy go dwomá łáńcuchámi / záwiedli go do Bábilonu.
Który będąc ućiśniony / modlił śię Pánu / Bogu ſwemu / y upokorzył śię bárdzo przed oblicżem Bogá ojców ſwoich /
Y prośił go ; á dáł mu śię uprośić / y wyſłucháł modlitwę jego / á przywróćił go do Jeruzálemu ná króleſtwo jego. Tedy poznał Mánáſeſ / iż ſám Pán jeſt Bogiem.
Potem budowáł mur około miáſtá Dawidowego ku záchodniej ſtronie Gichonu potoku áż do wejśćiá do brámy rybney / y otocżył murem Ofel / y wywiódł go bárdzo wyſoko ; poſtánowił też hetmánów po wƺyſtkich miáſtách obronnych w Judźie.
Znióſł też bogów cudzych / y báłwáná z domu Páńſkiego / y wƺyſtkie ołtárze / które był pobudowáł ná górze domu Páńſkiego / y w Jeruzálemie / y wyrzućił zá miáſto.
Zátem nápráwił ołtárz Páńſki / y ſpráwowáł ná nim ofiáry ſpokojne / y dźiękcżynieniá / á przykázáł Judźie / áby ſłużyli Pánu / Bogu Izráelſkiemu.
Wƺákże jeƺcże lud ofiárowáł ná wyżynách / lecż tylko Pánu / Bogu ſwemu.
Ale inne ſpráwy Mánáſeſowe / y modlitwá jego do Bogá jego / y ſłowá widzących / którzy máwiáli do niego w imię Páná / Bogá Izráelſkiego / ſą w kśięgách ſpráw królów Izráelſkich.
Modlitwá záś jego / y jáko jeſt wyſłuchány / y káżdy grzech jego / y przeſtępſtwo jego / y miejſcá / ná których był pobudowáł wyżyny / y wyſtáwił gáje święcone / y báłwány / przedtem niż śię był upokorzył / zápiſáne w kśięgách Chozájá.
Potem záſnął Mánáſeſ z ojcámi ſwymi / y pochowáli go w domu jego ; á Amon / ſyn jego / królowáł miáſto niego.
We dwudźieſtu y dwóch látách był Amon / gdy królowáć pocżął / á dwá látá królowáł w Jeruzálemie.
Y cżynił złe przed ocżymá Páńſkimi / jáko cżynił Mánáſeſ / ojćiec jego ; ábowiem wƺyſtkim báłwánom / których był nácżynił Mánáſeſ / ojćiec jego / ofiárowáł Amon / y ſłużył im.
A nie upokorzył śię przed oblicżem Páńſkim / jáko śię upokorzył Mánáſeſ / ojćiec jego ; owƺem ten Amon dáleko więcey grzeƺył.
Y ſprzyśięgli śię przećiw niemu ſłudzy jego / y zábili go w domu jego.
Ale lud oney źiemi pobił wƺyſtkich / co śię byli ſprzyśięgli przećiw królowi Amonowi ; á poſtánowił lud źiemi królem Jozyjáƺá / ſyná jego / miáſto niego.
