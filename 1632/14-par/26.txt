Tedy wƺyſtek lud Judzki wźięli Uzyjáƺá / który miáł ƺeſnáśćie lát / y poſtánowili go królem miáſto ojcá jego Amázyjáƺá.
Ten pobudowáł Elát / á przywróćił je do Judy / gdy záſnął król z ojcámi ſwymi.
Szeſnáśćie lát było Uzyjáƺowi / gdy królowáć pocżął / á pięćdźieśiąt y dwá látá królowáł w Jeruzálemie ; á imię mátki jego Jechálijá z Jeruzálemu.
Ten cżynił / co było dobrego w ocżách Páńſkich według wƺyſtkiego / jáko cżynił Amázyjáƺ / ojćiec jego.
Y ƺukáł Bogá zá dniá Zácháryjáƺá / rozumiejącego widzeniá Boże ; á po wƺyſtkie one dni / których ƺukáł Páná / ƺcżęśćił go Bóg.
Bo ruƺywƺy śię / wálcżył z Filiſtynámi / y poburzył mury w Get / y mury w Jábnie / y mury w Azoćie / á pobudowáł miáſtá w Azoćie y w źiemi Filiſtyńſkiej.
Abowiem pomágáł mu Bóg przećiw Filiſtynom / y przećiw Arábcżykom / którzy mieƺkáli w Gurbáálu / y przećiw Máhunitom.
Y dáwáli Ammonitowie Uzyjáƺowi dáry / á roznioſło śię imię jego áż do ſámego Egiptu ; bo śię był zmocnił náder bárdzo.
Y budowáł Uzyjáƺ wieże w Jeruzálemie nád brámą nárożną / y nád brámą doliny / y nád Mikzoá / y umocnił je.
Pobudowáł też wieże ná puƺcży / y pokopáł wiele ſtudźien ; bo miáł bárdzo wiele ſtád / ták w dolinách / jáko y w równinách / y rolników / y winiárzy po górách / y ná Kármelu : ábowiem śię kocháł w upráwiániu ról.
Miáł też Uzyjáƺ wojſko gotowe do bitwy / które wychodźiło ná wojnę w pocżtách ſwych według licżby / jáko byli oblicżeni przez Jechijelá kánclerzá / y Mááſájáƺá hetmáná / pod ſpráwą Hánánijáƺá / kśiążęćiá królewſkiego.
Wƺyſtká licżbá przedniejƺych z domów ojcowſkich / ludźi rycerſkich / dwá tyśiące y ƺeść ſet.
A pod ſpráwą ich ludu wálecżnego trzy kroć ſto tyśięcy / y śiedm tyśięcy y pięć ſet / ludu ſpoſobnego do wojny / ná pomoc królowi przećiw nieprzyjáćielowi.
Y zgotowáł Uzyjáƺ wƺyſtkiemu onemu wojſku tárcże y drzewce / y przyłbice / y páncerze / y łuki / y kámienie do proc.
Nácżynił też w Jeruzálemie ƺtuk wojennych bárdzo miſternych / áby były ná wieżách / y ná węgłách ku wypuƺcżániu ſtrzáł / y kámieniá wielkiego ; y roznioſło śię imię jego dáleko / przeto / że miáł dźiwną pomoc / áż śię zmocnił.
Ale gdy śię zmocnił / podnioſło śię ſerce jego ku zginieniu jego / y wyſtąpił przećiw Pánu / Bogu ſwemu ; bo wƺedł do kośćiołá Páńſkiego / áby kádźił ná ołtárzu kádzeniá.
Y wƺedł zá nim Azáryjáƺ kápłán / á z nim kápłánów Páńſkich ośmdźieśiąt / mężów dużych.
Y ſtánęli przećiw Uzyjáƺowi królowi / á mówili mu : Nie twojá to rzecż Uzyjáƺu! kádźić Pánu / ále kápłánów / ſynów Aáronowych / którzy ſą poświęceni / áby kádźili. Wynijdźże z świątnicy ; ábowiemeś wyſtąpił / á nie będźieć to ku ſłáwie przed Pánem Bogiem.
Przetoż śię rozgniewáł Uzyjáƺ / májąc w rękách ſwych kádźielnicę / áby kádźił. A gdy śię ſrożył przećiwko kápłánom / trąd wyſtąpił ná cżoło jego przed kápłánámi w domu Páńſkim u ołtárzá kádzeniá.
A wejrzáwƺy náń Azáryjáƺ / kápłán nájwyżƺy / y wƺyſcy kápłáni / á oto był trędowátym ná cżole ſwojim ; przetoż prędko wygnáli go ſtámtąd ; owƺem y ſám poſpieƺáł wynijść / przeto / że go záráźił Pán.
A ták był król Uzyjáƺ trędowátym áż do dniá śmierći ſwej / y mieƺkáł w domu oſobnym / będąc trędowátym ; ábowiem był wyłącżon z domu Páńſkiego ; międzytem Joátám / ſyn jego / był nád domem królewſkim / ſądząc lud źiemi.
A inne ſpráwy Uzyjáƺowe / pierwƺe y poślednie / opiſáł prorok Izájáƺ / ſyn Amoſowy.
Y záſnął Uzyjáƺ z ojcámi ſwymi / á pochowáno go z ojcámi jego ná polu grobów królewſkich ; bo mówili : Trędowáty jeſt. A królowáł Joátám / ſyn jego / miáſto niego.
