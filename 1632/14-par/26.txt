Tedy wƺyſtek lud Judzki wźięli Uzyjáƺá / który miał ƺeſnaśćie lat / y poſtánowili go królem miáſto ojcá jego Amázyjáƺá.
Ten pobudował Elát / á przywróćił je do Judy / gdy záſnął król z ojcámi ſwymi.
Szeſnaśćie lat było Uzyjáƺowi / gdy królowáć pocżął / á pięćdźieśiąt y dwá látá królował w Jeruzálemie ; á imię mátki jego Jechálijá z Jeruzálemu.
Ten cżynił / co było dobrego w ocżách Páńſkich według wƺyſtkiego / jáko cżynił Amázyjáƺ / oćiec jego.
Y ƺukał Bogá zá dniá Zácháryjáƺá / rozumiejącego widzeniá Boże ; á po wƺyſtkie one dni / których ƺukał Páná / ƺcżęśćił go Bóg.
Bo ruƺywƺy śię / wálcżył z Filiſtynámi / y poburzył mury w Get / y mury w Jábnie / y mury w Azoćie / á pobudował miáſtá w Azoćie y w źiemi Filiſtyńſkiej.
Abowiem pomágał mu Bóg przećiw Filiſtynom / y przećiw Arábcżykom / którzy mieƺkáli w Gurbáálu / y przećiw Máhunitom.
Y dáwáli Ammonitowie Uzyjáƺowi dáry / á roznioſło śię imię jego áż do ſámego Egiptu ; bo śię był zmocnił náder bárzo.
Y budował Uzyjáƺ wieże w Jeruzálemie nád bramą nárożną / y nád bramą doliny / y nád Mikzoá / y umocnił je.
Pobudował też wieże ná puƺcży / y pokopał wiele ſtudźien ; bo miał bárzo wiele ſtád / ták w dolinách / jáko y w równinách / y rolników / y winiárzy po górách / y ná Kármelu : ábowiem śię kochał w upráwiániu ról.
Miał też Uzyjáƺ wojſko gotowe do bitwy / które wychodźiło ná wojnę w pocżtách ſwych według licżby / jáko byli oblicżeni przez Jechijelá kánclerzá / y Mááſájáƺá hetmáná / pod ſpráwą Hánánijáƺá / kśiążęćiá królewſkiego.
Wƺyſtká licżbá przedniejƺych z domów ojcowſkich / ludźi rycerſkich / dwá tyśiące y ƺeść ſet.
A pod ſpráwą ich ludu wálecżnego trzy kroć ſto tyśięcy / y śiedm tyśięcy y pięć ſet / ludu ſpoſobnego do wojny / ná pomoc królowi przećiw nieprzyjáćielowi.
Y zgotował Uzyjáƺ wƺyſtkiemu onemu wojſku tárcże y drzewce / y przyłbice / y páncerze / y łuki / y kámienie do proc.
Nácżynił też w Jeruzálemie ƺtuk wojennych bárzo miſternych / áby były ná wieżách / y ná węgłách ku wypuƺcżániu ſtrzał / y kámieniá wielkiego ; y roznioſło śię imię jego dáleko / przeto / że miał dźiwną pomoc / áż śię zmocnił.
Ale gdy śię zmocnił / podnioſło śię ſerce jego ku zginieniu jego / y wyſtąpił przećiw Pánu / Bogu ſwemu ; bo wƺedł do kośćiołá Páńſkiego / áby kádźił ná ołtárzu kádzeniá.
Y wƺedł zá nim Azáryjáƺ kápłán / á z nim kápłanów Páńſkich ośmdźieśiąt / mężów dużych.
Y ſtánęli przećiw Uzyjáƺowi królowi / á mówili mu : Nie twojá to rzecż Uzyjáƺu! kádźić Pánu / ále kápłanów / ſynów Aáronowych / którzy ſą poświęceni / áby kádźili. Wynijdźże z świątnicy ; ábowiemeś wyſtąpił / á nie będźieć to ku ſłáwie przed Pánem Bogiem.
Przetoż śię rozgniewał Uzyjáƺ / májąc w rękách ſwych kádźielnicę / áby kádźił. A gdy śię ſrożył przećiwko kápłánom / trąd wyſtąpił ná cżoło jego przed kápłánámi w domu Páńſkim u ołtárzá kádzeniá.
A wejrzáwƺy nań Azáryjáƺ / kápłán nájwyżƺy / y wƺyſcy kápłáni / á oto był trędowátym ná cżole ſwojim ; przetoż prędko wygnáli go z támtąd ; owƺem y ſám poſpieƺał wynijść / przeto / że go záráźił Pán.
A ták był król Uzyjáƺ trędowátym áż do dniá śmierći ſwej / y mieƺkał w domu oſobnym / będąc trędowátym ; ábowiem był wyłącżon z domu Páńſkiego ; międzytem Joátám / ſyn jego / był nád domem królewſkim / ſądząc lud źiemi.
A inne ſpráwy Uzyjáƺowe / pierwƺe y poślednie / opiſał prorok Izájáƺ / ſyn Amoſowy.
Y záſnął Uzyjáƺ z ojcámi ſwymi / á pochowano go z ojcámi jego ná polu grobów królewſkich ; bo mówili : Trędowáty jeſt. A królował Joátám / ſyn jego / miáſto niego.
