Po tych ſpráwách y pewnem ich poſtánowieniu przyćiągnął Sennácheryb / król Aſſyryjſki / á wtárgnąwƺy do Judzkiej źiemi / położył śię obozem przećiw miáſtom obronnym / á umyślił je ſobie dobyć.
A widząc Ezechyjáƺ / że przyćiągnął Sennácheryb / á iż twárz ſwoję obróćił / áby wálcżył przećiw Jeruzálemowi :
Tedy wƺedł w rádę z kśiążętámi ſwymi y z rycerſtwem ſwem / áby zátkáli źródłá wód / które były zá miáſtem ; y pomogli mu.
Bo zebráwƺy śię lud wielki zátkáli wƺyſtkie źródłá / y potok / który płynął przez pośrodek źiemi / mówiąc : Cżemużby przyƺedłƺy królowie Aſſyryjſcy mieli ználeść ták wiele wód?
A pokrzepiwƺy śię / pobudowáł wƺyſtkie mury obálone / y nábudowáł wież / przytem zewnątrz drugi mur ; y zmocnił Mello w mieśćie Dawidowem / y pocżynił broni bárdzo wiele / y tárcży.
Poſtánowił też hetmánów wojennych nád ludem / których zgromádźił do śiebie ná ulicę brámy miejſkiej / y mówił łáſkáwie do nich / á rzekł :
Zmácniájćie śię / y mężnie ſobie pocżynájćie ; nie bójćie śię / áni śię lękájćie twárzy królá Aſſyryjſkiego / áni twárzy wƺyſtkiego mnóſtwá / które jeſt z nim ; bo więkƺy jeſt z námi niżeli z nim.
Z nimi jeſt rámię ćieleſne ; ále z námi jeſt Pán / Bóg náƺ / áby nas rátowáł y odpráwiáł wojny náƺe. Tedy ſpoległ lud ná ſłowách Ezechyjáƺá / królá Judzkiego.
Potem poſłáł Sennácheryb / król Aſſyryjſki / ſługi ſwe do Jeruzálemu / ( á ſám dobywáł Láchiſ / á wƺyſtká moc jego byłá z nim / )do Ezechyjáƺá / królá Judzkiego / y do wƺyſtkich z Judy / którzy byli w Jeruzálemie / mówiąc :
Ták mówi Sennácheryb / król Aſſyryjſki : W cżemże wżdy ufáćie / że śiedźićie w murách Jeruzálemſkich?
Izáli Ezechyjáƺ nie zwodźi was / áby was pomorzył głodem y prágnieniem / mówiąc : Pán / Bóg náƺ / wyrwie nas z ręki królá Aſſyryjſkiego?
Izáli nie ten Ezechyjáƺ znióſł wyżyny jego / y ołtárze jego / y rozkázáł Judźie y Jeruzálemcżykom / mówiąc : Przy jednym tylko ołtárzu kłániáć śię będźiećie / y ná nim kádźić?
Izáli nie wiećie / com ucżynił ja y ojcowie moi wƺyſtkim národom źiemſkim? Azáż jákim ſpoſobem mogli bogowie národów źiemſkich wyrwáć źiemię ſwoję z ręki mojey?
Któż był ze wƺyſtkich bogów onych národów / które wytráćili ojcowie moi / coby mógł wybáwić lud ſwój z ręki mojey / áby też mógł Bóg wáƺ wyrwáć was z ręki mojey?
Przetoż teraz niech was nie zwodźi Ezechyjáƺ / á niech was ná to nie námáwiá / áni mu wierzćie. Jeſlić nie mógł żáden bóg wƺyſtkich národów y króleſtw wyrwáć ludu ſwego z ręki mojey / y z ręki ojców moich / pogotowiu Bóg wáƺ nie wyrwie was z ręki mojey.
Nádto jeƺcże mówili ſłudzy jego przećiw Pánu Bogu / y przećiwko Ezechyjáƺowi ſłudze jego.
Liſty też piſáł / urągájąc Pánu / Bogu Izráelſkiemu / á mówiąc przećiwko niemu temi ſłowy : Jáko bogowie národów źiemſkich nie wyrwáli ludu ſwego z ręki mojey / ták nie wyrwie Bóg Ezechyjáƺowy ludu ſwego z ręki mojey.
Y wołáli głoſem wielkim po żydowſku przećiwko ludowi Jeruzálemſkiemu / który był ná murách / ſtráƺąc go y trwożąc go / áby ták miáſto wźięli.
A mówili przećiw Bogu Jeruzálemſkiemu / jáko przećiw bogom národów źiemſkich / którzy ſą robotą rąk ludzkich.
Tedy śię modlił Ezechyjáƺ król / y Izájáƺ prorok / ſyn Amoſá / dla tego / y krzycżeli ku niebu.
Y poſłáł Pán Aniołá / który wytráćił káżdego mocárzá w wojſku / y wodzá / y hetmáná w oboźie królá Aſſyryjſkiego. Y wróćił śię z poháńbieniem twárzy do źiemi ſwojey. A gdy wƺedł do domu bogá ſwego / ći / którzy wyƺli z żywotá jego / tám go zábili miecżem.
A ták wybáwił Pán Ezechyjáƺá y obywáteli Jeruzálemſkich z rąk Sennácherybá / królá Aſſyryjſkiego / y z rąk wƺyſtkich / á ſpráwił im pokój zewƺąd.
Tedy wiele ich przynośiło Pánu dáry do Jeruzálemu y upominki koƺtowne Ezechyjáƺowi / królowi Judzkiemu ; y wywyżƺony jeſt w ocżách wƺyſtkich národów potem.
W one dni záchorowáł Ezechyjáƺ áż ná śmierć / y modlił śię Pánu ; który mówił do niego / á dáł mu znák.
Ale nie według dobrodźiejſtw ſobie ucżynionych ſpráwowáł śię Ezechyjáƺ / bo śię wynioſło ſerce jego ; przetoż powſtáł przećiw niemu gniew / y przećiw Judźie / y przećiw Jeruzálemowi.
Ale gdy śię upokorzył Ezechyjáƺ ( bo śię było wynioſło ſerce jego ) on y obywátele Jeruzálemſcy / nie przyƺedł ná nich gniew Páńſki zá dni Ezechyjáƺowych.
A miał Ezechyjáƺ bogáctwá y ſłáwę bárdzo wielką ; bo ſobie zebráł ſkárby ſrebrá y złotá / y kámieni drogich / y rzecży wonnych / y rynƺtunków / y wƺelákiego nácżyniá koƺtownego.
Miał też ƺpichlerze dla urodzájów zbożá y winá / y oliwy / y obory dla bydeł / y zwierzyniec dla rozmáitych zwierząt.
Miáſtá też ſobie pobudowáł / y miał ſtádá owiec / y wołów mnóſtwo ; ábowiem mu dáł Bóg májętność bárdzo wielką.
Ten też Ezechyjáƺ zátkáł źródło wód w Gichonie wyżƺe / á przywiódł je dołem ná záchód ſłońcá ku miáſtu Dawidowemu ; y ƺcżęśćiło śię Ezechyjáƺowi we wƺyſtkich ſpráwách jego.
A wƺákże dla poſłów kśiążąt Bábilońſkich / którzy byli poſłáni do niego / áby go pytáli o znák / który śię był ſtáł w źiemi / opuśćił go Bóg / áby go kuśił / á iżby wiedźiáno wƺyſtko / co było w ſercu jego.
Ale inne ſpráwy Ezechyjáƺowe / y dobrocżynnośći jego / nápiſáne ſą w widzeniu Izájáƺá proroká / ſyná Amoſowego / y w kśięgách królów Judzkich y Izráelſkich.
Y záſnął Ezechyjáƺ z ojcámi ſwymi / y pogrzebiony jeſt nád grobámi ſynów Dawidowych. Y wyrządźili mu wƺyſtek Judá y obywátele Jeruzálemſcy ucżćiwość przy śmierći jego. A królowáł Mánáſeſ / ſyn jego / miáſto niego.
