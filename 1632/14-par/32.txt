Po tych ſpráwách y pewnem ich poſtánowieniu przyćiągnął Sennácheryb / król Aſſyryjſki / á wtárgnąwƺy do Judzkiej źiemi / położył śię obozem przećiw miáſtom obronnym / á umyślił je ſobie dobyć.
A widząc Ezechyjáƺ / że przyćiągnął Sennácheryb / á iż twárz ſwoję obróćił / áby wálcżył przećiw Jeruzálemowi :
Tedy wƺedł w rádę z kśiążętámi ſwymi y z rycerſtwem ſwem / áby zátkáli źródłá wód / które były zá miáſtem ; y pomogli mu.
Bo zebráwƺy śię lud wielki zátkáli wƺyſtkie źródłá / y potok / który płynął przez pośrodek źiemi / mówiąc : Cżemużby przyƺedƺy królowie Aſſyryjſcy mieli ználeść ták wiele wód?
A pokrzepiwƺy śię / pobudował wƺyſtkie mury obálone / y nábudował wież / przytem zewnątrz drugi mur ; y zmocnił Mello w mieśćie Dawidowem / y pocżynił broni bárzo wiele / y tárcży.
Poſtánowił też hetmánów wojennych nád ludem / których zgromádźił do śiebie ná ulicę bramy miejſkiej / y mówił łáſkáwie do nich / á rzekł :
Zmácniájćie śię / y mężnie ſobie pocżynájćie ; nie bójćie śię / áni śię lękájćie twárzy królá Aſſyryjſkiego / áni twárzy wƺyſtkiego mnóſtwá / które jeſt z nim ; bo więkƺy jeſt z námi niżeli z nim.
Z nimi jeſt rámię ćieleſne ; ále z námi jeſt Pán / Bóg náƺ / áby nas rátował y odpráwiał wojny náƺe. Tedy ſpoległ lud ná ſłowách Ezechyjáƺá / królá Judzkiego.
Potem poſłał Sennácheryb / król Aſſyryjſki / ſługi ſwe do Jeruzálemu / ( á ſám dobywał Láchis / á wƺyſtká moc jego byłá z nim / )do Ezechyjáƺá / królá Judzkiego / y do wƺyſtkich z Judy / którzy byli w Jeruzálemie / mówiąc :
Ták mówi Sennácheryb / król Aſſyryjſki : W cżemże wżdy ufáćie / że śiedźićie w murách Jeruzálemſkich?
Izali Ezechyjáƺ nie zwodźi was / áby was pomorzył głodem y prágnieniem / mówiąc : Pán / Bóg náƺ / wyrwie nas z ręki królá Aſſyryjſkiego?
Izali nie ten Ezechyjáƺ znióſł wyżyny jego / y ołtárze jego / y rozkázał Judźie y Jeruzálemcżykom / mówiąc : Przy jednym tylko ołtárzu kłániáć śię będźiećie / y ná nim kádźić?
Izali nie wiećie / com ucżynił ja y ojcowie moi wƺyſtkim národom źiemſkim? Azaż jákim ſpoſobem mogli bogowie narodów źiemſkich wyrwáć źiemię ſwoję z ręki mojey?
Któż był ze wƺyſtkich bogów onych narodów / które wytráćili ojcowie moi / coby mógł wybáwić lud ſwój z ręki mojey / áby też mógł Bóg wáƺ wyrwáć was z ręki mojey?
Przetoż teraz niech was nie zwodźi Ezechyjáƺ / á niech was ná to nie námáwiá / áni mu wierzćie. Jeſlić nie mógł żáden bóg wƺyſtkich narodów y króleſtw wyrwáć ludu ſwego z ręki mojey / y z ręki ojców mojich / pogotowiu Bóg wáƺ nie wyrwie was z ręki mojey.
Nádto jeƺcże mówili ſłudzy jego przećiw Pánu Bogu / y przećiwko Ezechyjáƺowi ſłudze jego.
Liſty też piſał / urągájąc Pánu / Bogu Izráelſkiemu / á mówiąc przećiwko niemu temi ſłowy : Jáko bogowie narodów źiemſkich nie wyrwáli ludu ſwego z ręki mojey / ták nie wyrwie Bóg Ezechyjáƺowy ludu ſwego z ręki mojey.
Y wołáli głoſem wielkim po żydowſku przećiwko ludowi Jeruzálemſkiemu / który był ná murách / ſtráƺąc go y trwożąc go / áby ták miáſto wźięli.
A mówili przećiw Bogu Jeruzálemſkiemu / jáko przećiw bogom narodów źiemſkich / którzy ſą robotą rąk ludzkich.
Tedy śię modlił Ezechyjáƺ król / y Izájáƺ prorok / ſyn Amoſá / dla tego / y krzycżeli ku niebu.
Y poſłał Pán Aniołá / który wytráćił káżdego mocárzá w wojſku / y wodzá / y hetmáná w oboźie królá Aſſyryjſkiego. Y wróćił śię z poháńbieniem twárzy do źiemi ſwojey. A gdy wƺedł do domu bogá ſwego / ći / którzy wyƺli z żywotá jego / tám go zábili miecżem.
A ták wybáwił Pán Ezechyjáƺá y obywáteli Jeruzálemſkich z rąk Sennácherybá / królá Aſſyryjſkiego / y z rąk wƺyſtkich / á ſpráwił im pokój zewƺąd.
Tedy wiele ich przynośiło Pánu dáry do Jeruzálemu y upominki koƺtowne Ezechyjáƺowi / królowi Judzkiemu ; y wywyżƺony jeſt w ocżách wƺyſtkich narodów potem.
W one dni záchorował Ezechyjáƺ áż ná śmierć / y modlił śię Pánu ; który mówił do niego / á dał mu znák.
Ale nie według dobrodźiejſtw ſobie ucżynionych ſpráwował śię Ezechyjáƺ / bo śię wynioſło ſerce jego ; przetoż powſtał przećiw niemu gniew / y przećiw Judźie / y przećiw Jeruzálemowi.
Ale gdy śię upokorzył Ezechyjáƺ ( bo śię było wynioſło ſerce jego ) on y obywátele Jeruzálemſcy / nie przyƺedł ná nich gniew Páńſki zá dni Ezechyjáƺowych.
A miał Ezechyjáƺ bogáctwá y ſłáwę bárzo wielką ; bo ſobie zebrał ſkárby ſrebrá y złotá / y kámieni drogich / y rzecży wonnych / y rynƺtunków / y wƺelákiego nacżynia koƺtownego.
Miał też ƺpichlerze dla urodzájów zbożá y winá / y oliwy / y obory dla bydeł / y zwierzyniec dla rozmáitych zwierząt.
Miáſtá też ſobie pobudował / y miał ſtádá owiec / y wołów mnóſtwo ; ábowiem mu dał Bóg májętność bárzo wielką.
Ten też Ezechyjáƺ zátkał źródło wód w Gichonie wyżƺe / á przywiódł je dołem ná záchód ſłońcá ku miáſtu Dawidowemu ; y ƺcżęśćiło śię Ezechyjáƺowi we wƺyſtkich ſpráwách jego.
A wƺákże dla poſłów kśiążąt Bábilońſkich / którzy byli poſłáni do niego / áby go pytáli o znák / który śię był ſtał w źiemi / opuśćił go Bóg / áby go kuśił / á iżby wiedźiáno wƺyſtko / co było w ſercu jego.
Ale inne ſpráwy Ezechyjáƺowe / y dobrocżynnośći jego / nápiſáne ſą w widzeniu Izájáƺá proroká / ſyná Amoſowego / y w kśięgách królów Judzkich y Izráelſkich.
Y záſnął Ezechyjáƺ z ojcámi ſwymi / y pogrzebiony jeſt nád grobámi ſynów Dawidowych. Y wyrządźili mu wƺyſtek Judá y obywátele Jeruzálemſcy ucżćiwość przy śmierći jego. A królował Mánáſes / ſyn jego / miáſto niego.
