Pocżął tedy Sálomon budowáć dom Páńſki w Jeruzálemie ná górze Moryjá / którá byłá ukázáná Dawidowi / ojcu jego / ná miejſcu / które zgotowáł Dawid ná bojewiſku Ornáná Jebuzejcżyká.
A pocżął go budowáć mieśiącá wtórego / dniá wtórego / roku cżwártego króleſtwá ſwego.
Tenći jeſt pomiár Sálomonowy / według którego budowáł dom Boży / wdłuż łokći ƺeśćdzieśiąt / łokći podług miáry pierwƺej / wƺerz łokći dwádzieśćiá.
Przyśionek záśię / który był przed oną długośćią y przed ƺerokośćią domu / był ná dwádzieśćiá łokći / á ná wzwyż ſto y dwádzieśćiá łokći ; który obłożył wewnątrz ƺcżerem złotem.
A dom wielki okrył drzewem jodłowem / który też obił ƺcżerem złotem / y dáł po wierzchu nácżynić pálm y łáńcuƺków.
Nákrył też dom kámieniem drogim ozdobnie / á złoto było złoto párwáimſkie.
Nádto powlekł dom / bálki / podwoje / y śćiány jego / y drzwi jego złotem / á wyrył Cherubiny ná śćiánách.
Spráwił też dom świątnicy nájświętƺej / którego długość byłá według ƺerokośći domu ná dwádzieśćiá łokći / á ƺerokość jego ná dwádzieśćiá łokći / y powlekł go złotá ƺcżerego ƺeśćią ſet tálentów ;
Gwoździe też wáżyły pięćdzieśiąt ſyklów złotá / tákże y ſále powlekł złotem.
Spráwił też w domu świątnicy nájświętƺej dwá Cherubiny robotą miſterną / y opráwił je złotem.
A ſkrzydłá Cherubinów były wdłuż ná dwádzieśćiá łokći ; ſkrzydło jedno ná pięć łokći / á dośięgáło śćiány domu ; drugie tákże ſkrzydło ná pięć łokći dośięgáło ſkrzydłá Cherubiná drugiego.
Tákże ſkrzydło Cherubiná drugiego ná pięć łokći dośięgáło śćiány domu / á ſkrzydło drugie ná pięć łokći ſkrzydłá Cherubiná drugiego.
A ták ſkrzydłá onych Cherubinów rozƺerzone były ná dwádzieśćiá łokći ; á oni ſtáli ná nogách ſwych / á twárze ich były w dom obrócone.
Spráwił też záſłonę z hijácyntu / y z ƺárłátu / y z jedwábiu / y z ſubtelnego lnu / y dáł wyháftowáć ná niej Cherubiny.
Ucżynił też przed domem dwá ſłupy ná trzydzieśći y ná pięć łokći wzwyż / á gáłká / którá byłá ná wierzchu ich / káżdá byłá ná pięć łokći.
Spráwił też łáńcuƺki / jáko w świątnicy / á przypráwił je ná wierzch onych ſłupów ; ſpráwił też ſto jábłek gránátowych / które wpráwił między one łáńcuƺki.
Y poſtáwił one ſłupy przed kośćiołem / jeden po práwej á drugi po lewej ſtronie ; y názwáł imię tego / co był ná práwej ſtronie / Jáchyn / á imię tego / co był ná lewej ſtronie / Boáz.
