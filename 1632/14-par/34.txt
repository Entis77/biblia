Ośm lat było Jozyjáƺowi / gdy królowáć pocżął / á trzydźieśći y jeden lat królował w Jeruzálemie.
Ten cżynił / co było dobrego przed ocżymá Páńſkimi / chodząc drogámi Dawidá / ojcá ſwego / y nie uchylał śię áni ná práwo áni ná lewo.
Bo óſmego roku królowánia ſwego / będąc jeƺcże dźiećięćiem / pocżął ƺukáć Bogá Dawidá / ojcá ſwego / á dwunáſtego roku pocżął ocżyƺcżáć Judę y Jeruzálem od wyżyn y od gájów święconych / y od báłwánów / y od rytych obrázów.
Abowiem przed ocżymá jego pokáżono ołtárze Báálów / y báłwány ſłonecżne / które były w górze ná nich / podćiął : tákże y gáje święcone / y obrázy ryte / y obrázy odláne pokruƺył y potárł / á rozmiotał je po grobách tych / którzy im ofiárowáli.
Kośći też kápłanów popálił ná ołtárzách ich / y ocżyśćił Judę y Jeruzálem ;
Tákże y miáſtá Mánáſeſowe y Efráimowe / y Symeonowe / áż do Neftálimá / y puſtynie ich okolicżne.
A ták poburzył ołtárze / y gáje święcone / y báłwány pokruƺył w ƺtuki / y wƺyſtkie obrázy powyćinał we wƺyſtkiej źiemi Izráelſkiej ; potem śię wróćił do Jeruzálemu.
A roku ośmnáſtego królowánia ſwego / gdy ocżyśćił źiemię y dom Páńſki / poſłał Sáfáná / ſyná Azálijáƺowego / y Mááſájáƺá / przełożonego miáſtá / y Joáchá / ſyná Joácházowego / kánclerzá / áby nápráwiono dom Páná / Bogá jego.
Którzy przyƺedƺy do Helkijáƺá / kápłáná nájwyżƺego / oddáli pieniądze znieśione do domu Bożego / które byli zebráli Lewitowie / ſtróżowie progu / od ſynów Mánáſeſowych y Efráimowych / y od wƺyſtkich oſtátków Izráelſkich / y od wƺyſtkiego Judy y Benjáminá / á wróćili śię do Jeruzálemu.
Y oddáli je w ręce rzemieślników / przełożonych nád robotą domu Páńſkiego / á oni je wydáwáli ná robotników / którzy robili w domu Páńſkim / nápráwiájąc y utwierdzájąc dom.
A dáwáli je ćieślom y murárzom zá ſkupowánie kámieniá ćioſánego / y drzewá ná ſpájánie y ná piętrá domów / które byli popſuli królowie Judzcy.
A mężowie oni byli wiernymi w tey prácy : á nád nimi byli przełożonymi Jáchát / y Abdyjáƺ / Lewitowie / z ſynów Merárego / y Zácháryjáƺ y Meſullám z ſynów Káátowych / którzy przynágláli roboćie ; á káżdy z Lewitów umiał gráć ná inſtrumentách muzycżnych.
Nád tymi też / którzy nośili brzemioná y przynágláli robotnikom przy káżdej roboćie / byli z Lewitów piſárze / y przyſtáwowie / y odźwierni.
A gdy wynáƺáli pieniądze znieśione do domu Páńſkiego / ználázł Helkijáƺ kápłán kśięgi zakonu Páńſkiego / podánego przez Mojżeƺá.
Tedy odpowiedźiał Helkijáƺ y rzekł do Sáfáná piſárzá : Ználázłem kśięgi zakonu w domu Páńſkim. Y oddał Helkijáƺ kśięgę Sáfánowi.
A Sáfán przynióſł onę kśięgę do królá ; á przytem oznájmił to królowi / mówiąc : Wƺyſtko / coś porucżył w rękę ſług twojich / wykonywáją :
Bo zebráwƺy pieniądze / które śię ználázły w domu Páńſkim / oddáli je w ręce przyſtáwów y w ręce robotników.
Nádto oznájmił Sáfán piſárz królowi mówiąc : Kśięgę mi też dał Helkijáƺ kápłán ; y cżytał ją Sáfán przed królem.
A gdy ſłyƺał król ſłowá zakonu / rozdárł ƺáty ſwoje.
Y rozkázał król Helkijáƺowi / y Achikámowi / ſynowi Sáfánowemu / y Abdonowi / ſynowi Micháſowemu / y Sáfánowi / piſárzowi / y Aſájáƺowi / ſłudze królewſkiemu / mówiąc :
Idźćie / rádźćie śię Páná o mię / y o oſtátek ludu w Izráelu y w Judźie około ſłów tych kśiąg / które ſą ználeźione ; bo wielka jeſt popędliwość Páńſka / która jeſt wyláná ná nas / przeto / że nie ſtrzegli ojcowie náśi ſłowá Páńſkiego / áby cżynili według wƺyſtkiego / co jeſt nápiſáne w tych kśięgách.
A ták poƺedł Helkijáƺ / y którzy byli przy królu / do Huldy prorokini / żony Sellumá / ſyná Tekui / ſyná Háſrowego / ſtróżá ƺát ; á oná mieƺkáłá w Jeruzálemie ná drugiej ſtronie miáſtá ; y mówili z nią o tym.
Która rzekłá do nich : Ták mówi Pán / Bóg Izráelſki : Powiedzćie mężowi / który was poſłał do mnie :
Ták mówi Pán : Oto / Ja przywiodę złe ná to miejſce / y ná obywáteli jego / wƺyſtkie przeklęſtwá nápiſáne w tych kśięgách / które cżytáno przed królem Judzkim.
Przeto / że mię opuśćili / y kádźili bogom cudzym / áby mię dráźnili wƺyſtkiemi ſpráwámi rąk ſwojich ; dlácżego wyleje śię popędliwość mojá ná to miejſce / y nie będźie ugáƺoná.
A królowi Judzkiemu / który was poſłał o rádę do Páná / ták powiedzćie : Ták mówi Pán / Bóg Izráelſki / o ſłowách / któreś ſłyƺał :
Gdyż ſerce twoje zmiękcżone jeſt / y upokorzyłeś śię przed oblicżem Bożem / ſłyƺąc ſłowá jego przećiwko temu miejſcu / y przećiwko obywátelom jego / á upokorzywƺy śię przedemną rozdárłeś ƺáty ſwe / y płákáłeś przedemną / przetożem ćię wyſłuchał / mówi Pán ;
Oto ja ćię zbiorę do ojców twojich / á będźieƺ włożony do grobu twego w pokoju / áby nie oglądáły ocży twoje wƺyſtkiego złego / które Ja przywiodę ná to miejſce / y ná obywáteli jego. Y odnieśli tę rzecż królowi.
Tedy poſłáwƺy król zgromádźił wƺyſtkich ſtárƺych z Judy y z Jeruzálemu.
Y wſtąpił król do domu Páńſkiego / y wƺyſcy mężowie Judzcy / y obywátele Jeruzálemſcy / y kápłáni y Lewitowie / y wƺyſtek lud / od wielkiego áż do máłego / y cżytał gdy wƺyſcy ſłyƺeli / wƺyſtkie ſłowá kśiąg przymierza / które było ználeźione w domu Páńſkim.
Potem ſtojąc król ná miejſcu ſwem / ucżynił przymierze przed Pánem / że chce chodźić zá Pánem / y ſtrzedz przykázáń jego / y świádectw jego / y uſtáw jego / ze wƺyſtkiego ſercá ſwego y ze wƺyſtkiej duƺy ſwojey / y pełnić ſłowá przymierza tego / które były w onych kśięgách nápiſáne.
Y rozkázał ſtáć w tym przymierzu wƺyſtkim / którzy ználeźieni byli w Jeruzálemie y w Benjáminie ; y cżynili obywátele Jeruzálemſcy według przymierza Bogá / Bogá ojców ſwojich.
Tedy uprzątnął Jozyjáƺ wƺyſtkie obrzydłośći ze wƺyſtkich kráin ſynów Izráelſkich / á przywiódł do tego wƺyſtkich / którzy śię znájdowáli w Izráelu / áby ſłużyli Pánu / Bogu ſwemu. Po wƺyſtkie dni jego nie odſtąpili od náśládowánia Páná / Bogá ojców ſwojich.
