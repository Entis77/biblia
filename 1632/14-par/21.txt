Potem záſnął Jozáfát z ojcámi ſwymi / y pochowány jeſt z ojcámi ſwymi w mieśćie Dawidowem / á królowáł Jorám / ſyn jego / miáſto niego.
Który miáł bráći / ſynów Jozáfátowych : Azáryjáƺá / y Jehijelá / y Zácháryjáƺá / y Azáryjáƺá / y Micháelá / y Sefátyjáƺá ; wƺyſcy ći byli ſynowie Jozáfátá / królá Izráelſkiego.
Którym był dáł ojćiec ich upominków wiele / ſrebrá y złotá / y rzecży koƺtownych z miáſtámi obronnemi w Judzie ; ále króleſtwo oddáł Jorámowi / poniewáż on był pierworodnym.
Náſtąpił tedy Jorám ná króleſtwo ojcá ſwego / á zmocniwƺy śię pozábijáł wƺyſtkich bráći ſwoich miecżem / tákże y niektórych z przedniejƺych w Izráelu.
Trzydzieśći y dwá látá miáł Jorám / gdy pocżął królowáć / á ośm lát królowáł w Jeruzálemie.
Y chodził drogámi królów Izráelſkich / jáko cżynił dom Achábowy / bo córká Achábowá byłá żoną jego ; y cżynił złe przed ocżymá Páńſkimi.
Ale nie chćiáł Pán wytráćić domu Dawidowego / dlá przymierzá / które był ucżynił z Dawidem / á iż był przyrzekł dáć mu pochodnię y ſynom jego po wƺyſtkie dni.
Zá jego dni odſtąpili Edomcżycy / áby nie byli poddánymi Judzie / á poſtánowili nád ſobą królá.
Przetoż śię ruƺył Jorám z hetmánámi ſwymi / y ze wƺyſtkiemi wozámi ſwemi / á wſtáwƺy w nocy / poráził Edomcżyków / którzy go byli otocżyli y hetmánów wozów jego.
Ale przećież odſtąpili Edomcżycy / áby nie byli pod mocą Judy / áż do dniá tego ; odſtąpiłá też y Lebná tegoż cżáſu / áby nie byłá pod mocą jego / przeto / iż był Jorám opuśćił Páná / Bogá ojców ſwoich.
Nádto nábudowáł wyżyn po górách Judzkich / y przywiódł w cudzołóſtwo obywáteli Jeruzálemſkich / y przymuƺáł jáko y Judę.
Tedy przyƺło do niego piſánie od Elijáƺá proroká w ten ſpoſób : Ták mówi Pán / Bóg Dawidá / ojcá twego : Przeto / iżeś nie chodził drogámi Jozáfátá / ojcá twego / y drogámi Azy / królá Judzkiego ;
Aleś chodził drogą królów Izráelſkich / á przywiodłeś w cudzołóſtwo Judę / y obywáteli Jeruzálemſkich / ták jáko cudzołożył dom Achábowy ; nádto y bráći twoich / dom ojcá twego / lepƺych nád ćię / pomordowáłeś :
Otoż Pán uderzy plágą wielką lud twój / y ſynów twoich / y żony twoje / y wƺyſtkę májętność twoję ;
Ná ćię też przyjdą niemocy wielkie / boleść wnętrzá twego / áż wypłyną trzewá twoje dlá boleśći dzień ode dniá ćiężƺej.
A ták pobudził Pán przećiwko Jorámowi duchá Filiſtyńcżyków / y Arábcżyków / którzy byli pogránicżni Etyjopcżykom ;
Którzy wtárgnąwƺy do ziemi Judzkiej / ſplundrowáli ją y pobráli wƺyſtkę májętność / którá śię ználázłá w domu królewſkim ; do tego y ſynów jego / y żony jego / ták / iż mu nie zoſtáł ſyn / jedno Joácház / nájmłodƺy z ſynów jego.
A nád to wƺyſtko záráził go Pán ná wnętrzu jego niemocą nieulecżoną.
A gdy dzień po dniu náſtępowáł / á cżáſ dwóch lát wychodził / wypłynęły wnętrznośći jego z boleśćią / y umárł w niemocách ćiężkich ; á nie ucżynił mu lud jego przy pogrzebie zápáłu / jáko cżynili zápáł ojcom jego.
Trzydzieśći y dwá látá miáł / gdy królowáć pocżął / á ośm lát królowáł w Jeruzálemie / á zƺedł ták / iż go nikt nie żáłowáł ; y pogrzebiony jeſt w mieśćie Dawidowem / wƺákże nie w grobách królewſkich.
