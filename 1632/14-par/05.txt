A ták dokońcżoná jeſt wƺyſtká robotá / którą ſpráwił Sálomon do domu Páńſkiego / y wnióſł tám Sálomon rzecży / które był poświęćił Dawid ojćiec jego : ſrebro / y złoto / y wƺyſtkie nácżyniá włożył do ſkárbów domu Bożego.
Tedy zebráł Sálomon ſtárƺych z Izráelá / y wƺyſtkich przedniejƺych z káżdego pokoleniá / y przedniejƺych z ojców ſynów Izráelſkich / do Jeruzálemu / áby przenieśli ſkrzynię przymierzá Páńſkiego z miáſtá Dawidowego / które jeſt Syon.
Y zebráli śię do królá wƺyſcy mężowie Izráelſcy w święto urocżyſte / które bywá mieśiącá śiódmego.
A gdy śię zeƺli wƺyſcy ſtárśi z Izráelá / wźięli Lewitowie ſkrzynię ;
Y nieśli ſkrzynię / y námiot zgromádzeniá / y wƺyſtkie nácżyniá święte / które były w námioćie / przenieśli je kápłáni y Lewitowie.
Zátem król Sálomon / y wƺyſtko zgromádzenie Izráelſkie / co śię byli zeƺli do niego przed ſkrzynię / ofiárowáli owce y woły / których nie możná oblicżyć / áni obráchowáć przez mnóſtwo.
Wnieśli tedy kápłáni ſkrzynię przymierzá Páńſkiego ná miejſce jej / do wnętrznego domu / to jeſt do świątnicy nájświętƺej pod ſkrzydłá Cherubinów ;
Abowiem Cherubinowie mieli rozćiągnione ſkrzydłá nád miejſcem ſkrzyni / y okrywáli Cherubinowie ſkrzynię / y drążki jej z wierzchu.
Y powyćiągáli one drążki / ták że końce ich było widáć z ſkrzyni ná przodku świątnicy ; ále ich nie widáć było zewnątrz / y támże zoſtáły áż do dniá tego.
A nic nie było w ſkrzyni / tylko dwie táblice / które tám był włożył Mojżeƺ ná Horebie / gdy ſtánowił przymierze Pán z ſynámi Izráelſkimi po wyjśćiu ich z Egiptu.
A gdy wychodźili kápłáni z świątnicy / ( bo wƺyſcy kápłáni / ile ich było / poświęćili śię byli / á nie przeſtrzegáli porządków. )
Stáli Lewitowie śpiewácy / y wƺyſcy / którzy byli przy Aſáfie / Hemmánie / y Jedytunie / y ſynowie ich / y bráćiá ich / oblecżeni będąc w ƺáty biśiorowe / z cymbáłámi y z hárfámi y z cytrámi / ſtáli mówię ná wſchodniej ſtronie ołtárzá / á przy nich kápłánów ſto y dwádźieśćiá trąbiących w trąby.
Y ſtáło śię / gdy jednoſtájnie trąbili / y śpiewáli / y wydáwáli jednáki głoſ / chwáląc y ſłáwiąc Páná : y gdy podnośili głoſ ná trąbách / y ná cymbáłách / y ná innych inſtrumentách muzycżnych / y chwálili Páná / że dobry / że ná wieki miłośierdźie jego / tedy dom on nápełniony jeſt obłokiem / to jeſt dom Páńſki.
Ták iż śię nie mogli kápłáni oſtáć / y ſłużyć dlá onego obłoku ; ábowiem nápełniłá byłá chwáłá Páńſká dom Boży.
