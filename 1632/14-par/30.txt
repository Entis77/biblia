Potem rozeſłáł Ezechyjáƺ do wƺyſtkiego Izráelá y do Judy ; tákże też liſty nápiſáł do Efráimá y do Mánáſeſá / áby przyƺli do domu Páńſkiego do Jeruzálemu y obchodzili święto przejśćiá Pánu / Bogu Izráelſkiemu.
Bo urádził król y kśiążętá jego / y wƺyſtko zgromádzenie w Jeruzálemie / áby obchodzili święto przejśćiá mieśiącá wtórego ;
Gdyż nie mogli go obchodzić cżáſu ſwego / przeto / iż kápłánów poświęconych nie było / ile ich było potrzebá / y lud nie był zgromádzony do Jeruzálemu.
A podobáłá śię tá rzecż królowi y wƺyſtkiemu zgromádzeniu.
Y poſtánowili / áby obwołáno po wƺyſtkim Izráelu / od Beerſáby áż do Dán / żeby śię zeƺli ná obchód świętá przejśćiá Pánu / Bogu Izráelſkiemu / do Jeruzálemu ; ábowiem już go dáwno nie obchodzili / jáko było nápiſáne.
Przetoż poſłowie ƺli z liſtámi od królá y od kśiążąt jego po wƺyſtkim Izráelu y Judzie z rozkázániem królewſkiem / mówiąc : Synowie Izráelſcy! náwróććie śię do Páná / Bogá Abráhámowego / Izáákowego / y Izráelowego / á on śię náwróći do oſtátków / które z wáſ uƺły z rąk królów Aſſyryjſkich.
Y nie bądźćie jáko ojcowie wáśi / y jáko bráćiá wáśi / którzy wyſtąpili przećiwko Pánu / Bogu ojców ſwoich : y podáł ich w ſpuſtoƺenie / jáko ſámi widzićie.
Teráz tedy nie zátwárdzájćie kárku wáƺego / jáko ojcowie wáśi : dájćie rękę Pánu / á pójdźćie do świątnicy jego / którą poświęćił ná wieki / y ſłużćie Pánu / Bogu wáƺemu / á odwróći śię od wáſ gniew popędliwośći jego.
Abowiem jeźli śię náwróćićie do Páná / bráćiá wáśi / y ſynowie wáśi miłośierdzie otrzymáją u tych / którzy ich záwiedli w niewolę / ták / iż śię wrócą do tej ziemi ; miłośierny bowiem / y dobrotliwy jeſt Pán / Bóg wáƺ / á nie odwróći od wáſ oblicżá ſwego / jeźli śię náwróćićie do niego.
A gdy oni poſłowie chodzili od miáſtá do miáſtá przez ziemię Efráimową y Mánáſeſową áż do Zábulon / náśmiewáli śię z nich / y ƺydzili z nich.
Wƺákże niektórzy mężowie z Aſer / y z Mánáſe / y z Zábulon / ukorzywƺy śię przyƺli do Jeruzálemu.
W Judzie też już byłá ręká Bożá / gdy im dáł ſerce jedno / áby cżynili rozkázánie królewſkie y kśiążąt / według ſłowá Páńſkiego.
Y zebráło śię do Jeruzálemu wiele ludu / áby obchodzili święto urocżyſte przáśników mieśiącá wtórego ; á było zgromádzenie bárdzo wielkie.
Tedy powſtáwƺy znieśli ołtárze / które były w Jeruzálemie / wƺyſtkie też ołtárze / ná których kádzono / porozwáláli / á wrzućili do potoku Cedron.
Potem ofiárowáli báránká wielkánocnego / dniá cżternáſtego / mieśiącá wtórego ; á kápłáni y Lewitowie záwſtydziwƺy śię / poświęcáli śię / á przywodzili cáłopáleniá do domu Páńſkiego.
Y ſtáli w porządku ſwym według zwycżáju ſwego / y według zákonu Mojżeƺá / mężá Bożego ; á kápłáni kropili krwią / wziąwƺy ją z ręki Lewitów.
A iż tákich było wiele w zgromádzeniu / którzy śię byli nie poświęćili / przetoż Lewitowie ofiárowáli ofiáry świętá przejśćiá zá káżdego niecżyſtego / áby był poświęcony Pánu.
Bo wielká licżbá ludu tego / to jeſt / wiele ich z Efráimá / y z Mánáſeſá / y z Iſáſchárá / y z Zábulonu nie byli ocżyƺcżeni / á przećież jedli báránká wielkánocnego / inácżej niż nápiſáno ; ále śię Ezechyjáƺ modlił zá nich / mówiąc : Dobrotliwy Pán niech ocżyśći káżdego /
Którykolwiek zgotowáł wƺyſtko ſerce ſwe / áby ƺukáł Bogá / Páná Bogá ojców ſwoich / choćby ocżyƺcżony nie był według ocżyƺcżeniá świątnicy.
Y wyſłucháł Pán Ezechyjáƺá / y záchowáł lud.
A ták obchodzili ſynowie Izráelſcy / którzy byli w Jeruzálemie / urocżyſte święto przáśników przez śiedm dni z weſelem wielkiem : y chwálili Páná. Lewitowie ná káżdy dzień / á kápłáni ná inſtrumentách ſłáwili moc Páńſką.
Y mówił Ezechyjáƺ łáſkáwie do wƺyſtkich Lewitów / którzy mieli dobre rozumienie o Pánu. Y jedli przez śiedm dni onego świętá / ſpráwując ofiáry ſpokojne / á wyſłáwiájąc Páná / Bogá ojców ſwoich.
Tedy urádziło wƺyſtko zgromádzenie / áby to jeƺcże cżynili przez drugie śiedm dni ; á ták obchodzili znowu śiedm dni z weſelem.
Abowiem Ezechyjáƺ / król Judzki / dáł był zgromádzeniu tyśiąc ćielców / y śiedm tyśięcy owiec : kśiążętá też dáli zgromádzeniu tyśiąc ćielców / y owiec dzieśięć tyśięcy. Y poświęćiło śię kápłánów bárdzo wiele.
A ták weſeliło śię wƺyſtko zgromádzenie Judzkie / y kápłáni / y Lewitowie / y wƺyſtko zgromádzenie / które było przyƺło z Izráelá / y przychodniowie / którzy przyƺli z ziemi Izráelſkiej / y mieƺkájący w Judzie.
Y było wielkie weſele w Jeruzálemie ; bo ode dni Sálomoná / ſyná Dawidowego / królá Izráelſkiego / nic tákiego nie było w Jeruzálemie.
Potem powſtáli kápłáni y Lewitowie / y błogoſłáwili ludowi ; á wyſłuchány jeſt głoſ ich / y przyƺłá modlitwá ich do mieƺkániá świętobliwośći Páńſkiej / do niebá.
