W śiedmiu látách był / Joáz / gdy królowáć pocżął / á cżterdźieśći lat królował w Jeruzálemie. Imię mátki jego było Sebijá z Berſáby.
Y cżynił Joáz / co było dobrego przed ocżymá Páńſkimi / po wƺyſtkie dni Jojády kápłáná.
A Jojádá dał mu dwie żony ; y płodźił ſynów y córki.
Y ſtáło śię potem / że umyślił w ſercu ſwojim Joáz odnowić dom Páńſki.
Przetoż zebráwƺy kápłanów y Lewitów rzekł do nich : Wynijdźćie do miaſt Judzkich / y wybierájćie od wƺyſtkiego Izráelá pieniądze ná popráwę domu Bogá wáƺego ná káżdy rok / á wy śię z tym poſpieƺćie ; ále śię nie ſpieƺyli Lewitowie.
Tedy wezwał król Jojády / przedniejƺego kápłáná / y rzekł mu : Przecżże śię nie upomináƺ u Lewitów / áby znośili z Judy y z Jeruzálemu podárki poſtánowione przez Mojżeƺá / ſługę Páńſkiego / zgromádzeniu Izráelſkiemu / ná námiot zgromádzenia?
Bo Atálijá niezbożná y ſynowie jey wyłupili dom Boży / á wƺyſtkie rzecży poświęcone z domu Páńſkiego obróćili ná báłwány.
Przetoż rozkázał król / áby ucżyniono ſkrzynię jednę / á poſtáwiono ją przed bramą domu Páńſkiego.
Y obwołáno w Judźie y w Jeruzálemie / áby znoƺono Pánu podárek poſtánowiony przez Mojżeƺá / ſługę Bożego / ná Izráelá ná puƺcży.
Y weſelili śię wƺyſcy kśiążętá / y wƺyſtek lud / á przynoƺąc / rzucáli do oney ſkrzyni / áż ją nápełnili.
A gdy przynośili ſkrzynię ná rozkáz królewſki przez ręce Lewitów / ( gdy widźieli / że było wiele pieniędzy ) przychodźił piſárz królewſki / y przyſtáw kápłáná nájwyżƺego / y wypróżniáli ſkrzynię ; potem ją odnośili / y ſtáwiáli ją ná miejſcu ſwem. Ták cżynili ná káżdy dźień / y zebráli pieniędzy bárzo wiele.
Które oddáwał król y Jojádá przełożonym nád robotą około domu Páńſkiego ; á ći nájmowáli kámienników y ćieśli do popráwy domu Páńſkiego / tákże y kowáli robiących żelázem y miedźią / ku zmocnieniu domu Páńſkiego.
A ták robili robotnicy / y bráło nápráwę ono dźieło przez ręce ich ; y przywiedli dom Boży do cáłośći ſwojey / y zmocnili go.
A gdy dokońcżyli / przynieśli przed królá y przed Jojádę oſtátek pieniędzy / á nárobiono z nich nacżynia do domu Páńſkiego / nacżynia ku poſługiwániu / y moźdźierzy y cżáƺ / y innego nacżynia złotego y ſrebrnego / á ofiárowáli cáłopalenia w domu Páńſkim uſtáwicżnie po wƺyſtkie dni Jojády.
Potem zſtárzał śię Jojádá / á będąc pełen dni / umárł ; ſto y trzydźieśći lat miał / gdy umárł.
Y pochowano go w mieśćie Dawidowem z królmi / przeto / że cżynił dobrze w Izráelu / y Bogu / y domowi jego.
A gdy umárł Jojádá / przyƺli kśiążętá Judzcy / y pokłonili śię królowi ; tedy ich uſłuchał król.
Skąd opuśćiwƺy dom Páná / Bogá ojców ſwych / ſłużyli gájom y báłwánom ; przetoż przyƺedł gniew ná Judę y ná Jeruzálem dla tego wyſtępku ich.
Y poſyłał do nich proroków / żeby ich náwróćili do Páná ; którzy choć śię oświádcżáli przećiw nim / ále ich przećież nie ſłucháli.
Owƺem / gdy Duch Boży wzbudźił Zácháryjáƺá / ſyná Jojády kápłáná / ( który ſtánąwƺy przed ludem / rzekł im : Ták mówi Bóg : Przecżże przeſtępujećie przykázánia Páńſkie? Nie poƺcżęśći śię wam ; ábowiem iżeśćie wy opuśćili Páná / on was też opuśći. )
Tedy śię ſprzyśięgli przećiwko niemu / y ukámionowáli go zá rozkázániem królewſkim w śieni domu Páńſkiego.
Y nie pámiętał król Joáz ná miłośierdźie / które był ucżynił z nim Jojádá / oćiec jego / ále zábił ſyná jego ; który gdy umierał / mówił : Niech to obácży Pán / á zemśći śię.
Y ſtáło śię po roku / przyćiągnęło przećiwko niemu wojſko Syryjſkie / á przyƺło do Judy y do Jeruzálemu / y wygłádźili z ludu wƺyſtkich kśiążąt ich / á wƺyſtkie łupy ich poſłáli królowi w Dámáƺku.
Bo w máłym pocżćie ludu przyćiągnęło było wojſko Syryjſkie ; á wżdy Pán podał w ręce ich bárzo wielkie wojſko / przeto / iż opuśćili Páná / Bogá ojców ſwojich. A ták nád Joázem wykonáli ſądy.
A gdy odćiągnęli od niego / zoſtáwiwƺy go w wielkich niemocách / ſprzyśięgli śię przećiwko niemu ſłudzy jego dla krwi ſynów Jojády kápłáná / y zábili go ná łożu jego. Y ták umárł / á pochowano go w mieśćie Dawidowem ; ále go nie pochowano w grobách królewſkich.
A ćić ſą / którzy śię byli ſprzyśięgli przećiw niemu : Zábát / ſyn Semááty Ammonitki / y Jozábát / ſyn Semáryty Moábitki.
Lecż o ſynách jego / y o wielkim podátku nań włożonym / y o nápráwie domu Bożego / to wƺyſtko nápiſáne w kśięgách królewſkich ; y królował Amázyjáƺ / ſyn jego / miáſto niego.
