W śiedmiu látách był / Joáz / gdy królowáć pocżął / á cżterdźieśći lát królowáł w Jeruzálemie. Imię mátki jego było Sebijá z Berſáby.
Y cżynił Joáz / co było dobrego przed ocżymá Páńſkimi / po wƺyſtkie dni Jojády kápłáná.
A Jojádá dáł mu dwie żony ; y płodźił ſynów y córki.
Y ſtáło śię potem / że umyślił w ſercu ſwojim Joáz odnowić dom Páńſki.
Przetoż zebráwƺy kápłánów y Lewitów rzekł do nich : Wynijdźćie do miáſt Judzkich / y wybierájćie od wƺyſtkiego Izráelá pieniądze ná popráwę domu Bogá wáƺego ná káżdy rok / á wy śię z tem poſpieƺćie ; ále śię nie ſpieƺyli Lewitowie.
Tedy wezwáł król Jojády / przedniejƺego kápłáná / y rzekł mu : Przecżże śię nie upomináƺ u Lewitów / áby znośili z Judy y z Jeruzálemu podárki poſtánowione przez Mojżeƺá / ſługę Páńſkiego / zgromádzeniu Izráelſkiemu / ná námiot zgromádzeniá?
Bo Atálijá niezbożná y ſynowie jey wyłupili dom Boży / á wƺyſtkie rzecży poświęcone z domu Páńſkiego obróćili ná báłwány.
Przetoż rozkázáł król / áby ucżyniono ſkrzynię jednę / á poſtáwiono ją przed brámą domu Páńſkiego.
Y obwołáno w Judźie y w Jeruzálemie / áby znoƺono Pánu podárek poſtánowiony przez Mojżeƺá / ſługę Bożego / ná Izráelá ná puƺcży.
Y weſelili śię wƺyſcy kśiążętá / y wƺyſtek lud / á przynoƺąc / rzucáli do oney ſkrzyni / áż ją nápełnili.
A gdy przynośili ſkrzynię ná rozkáz królewſki przez ręce Lewitów / ( gdy widźieli / że było wiele pieniędzy ) przychodźił piſárz królewſki / y przyſtáw kápłáná nájwyżƺego / y wypróżniáli ſkrzynię ; potem ją odnośili / y ſtáwiáli ją ná miejſcu ſwem. Ták cżynili ná káżdy dźień / y zebráli pieniędzy bárdzo wiele.
Które oddáwáł król y Jojádá przełożonym nád robotą około domu Páńſkiego ; á ći nájmowáli kámienników y ćieśli do popráwy domu Páńſkiego / tákże y kowáli robiących żelázem y miedźią / ku zmocnieniu domu Páńſkiego.
A ták robili robotnicy / y bráło nápráwę ono dźieło przez ręce ich ; y przywiedli dom Boży do cáłośći ſwojey / y zmocnili go.
A gdy dokońcżyli / przynieśli przed królá y przed Jojádę oſtátek pieniędzy / á nárobiono z nich nácżyniá do domu Páńſkiego / nácżyniá ku poſługiwániu / y moźdźierzy y cżáƺ / y innego nácżyniá złotego y ſrebrnego / á ofiárowáli cáłopáleniá w domu Páńſkim uſtáwicżnie po wƺyſtkie dni Jojády.
Potem zſtárzáł śię Jojádá / á będąc pełen dni / umárł ; ſto y trzydźieśći lát miał / gdy umárł.
Y pochowáno go w mieśćie Dawidowem z królmi / przeto / że cżynił dobrze w Izráelu / y Bogu / y domowi jego.
A gdy umárł Jojádá / przyƺli kśiążętá Judzcy / y pokłonili śię królowi ; tedy ich uſłucháł król.
Skąd opuśćiwƺy dom Páná / Bogá ojców ſwych / ſłużyli gájom y báłwánom ; przetoż przyƺedł gniew ná Judę y ná Jeruzálem dla tego wyſtępku ich.
Y poſyłáł do nich proroków / żeby ich náwróćili do Páná ; którzy choć śię oświádcżáli przećiw nim / ále ich przećież nie ſłucháli.
Owƺem / gdy Duch Boży wzbudźił Zácháryjáƺá / ſyná Jojády kápłáná / ( który ſtánąwƺy przed ludem / rzekł im : Ták mówi Bóg : Przecżże przeſtępujećie przykázániá Páńſkie? Nie poƺcżęśći śię wám ; ábowiem iżeśćie wy opuśćili Páná / on was też opuśći. )
Tedy śię ſprzyśięgli przećiwko niemu / y ukámionowáli go zá rozkázániem królewſkim w śieni domu Páńſkiego.
Y nie pámiętáł król Joáz ná miłośierdźie / które był ucżynił z nim Jojádá / ojćiec jego / ále zábił ſyná jego ; który gdy umieráł / mówił : Niech to obácży Pán / á zemśći śię.
Y ſtáło śię po roku / przyćiągnęło przećiwko niemu wojſko Syryjſkie / á przyƺło do Judy y do Jeruzálemu / y wygłádźili z ludu wƺyſtkich kśiążąt ich / á wƺyſtkie łupy ich poſłáli królowi w Dámáƺku.
Bo w máłym pocżćie ludu przyćiągnęło było wojſko Syryjſkie ; á wżdy Pán podáł w ręce ich bárdzo wielkie wojſko / przeto / iż opuśćili Páná / Bogá ojców ſwoich. A ták nád Joázem wykonáli ſądy.
A gdy odćiągnęli od niego / zoſtáwiwƺy go w wielkich niemocách / ſprzyśięgli śię przećiwko niemu ſłudzy jego dla krwi ſynów Jojády kápłáná / y zábili go ná łożu jego. Y ták umárł / á pochowáno go w mieśćie Dawidowem ; ále go nie pochowáno w grobách królewſkich.
A ćić ſą / którzy śię byli ſprzyśięgli przećiw niemu : Zábát / ſyn Semááty Ammonitki / y Jozábát / ſyn Semáryty Moábitki.
Lecż o ſynách jego / y o wielkim podátku náń włożonym / y o nápráwie domu Bożego / to wƺyſtko nápiſáne w kśięgách królewſkich ; y królowáł Amázyjáƺ / ſyn jego / miáſto niego.
