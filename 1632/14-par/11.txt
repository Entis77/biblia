Przyjecháwƺy tedy Roboám do Jeruzálemu / zebrał dom Judowy y pokolenie Benjáminowe / ſto y ośmdźieśiąt tyśięcy mężów przebránych do boju / áby wálcżyli z Izráelem / á żeby przywrócone było króleſtwo do Roboámá.
Y ſtáło śię ſłowo Páńſkie do Semejáƺá / mężá Bożego / mówiąc :
Powiedz Roboámowi / ſynowi Sálomonowemu / królowi Judzkiemu / y wƺyſtkiemu Izráelowi w Judźie y w pokoleniu Benjáminowem / mówiąc :
Ták mówi Pán : Nie wychodźćie / áni wálcżćie z bráćmi wáƺymi / wróććie śię káżdy do domu ſwego ; ábowiem odemnie ſtáłá śię tá rzecż. Y uſłucháli ſłowá Páńſkiego / y wróćili śię / á nie ćiągnęli przećiw Jeroboámowi.
Y mieƺkał Roboám w Jeruzálemie / á pobudował miáſtá obronne w Judźie.
Y zbudował Betlehem / y Etám y Tekue ;
Y Betſur / y Soko / y Adullám :
Y Get / y Máreſá / y Zyf :
Y Adoráim / y Láchis / y Aſeká ;
Y Sorá / y Ajálon / y Hebron / które były w Judźie y w pokoleniu Benjáminowem miáſtá obronne.
A gdy zmocnił one twierdze / poſtánowił w nich ſtároſtów / y wyſtáwił ƺpichlerze dla zbożá / y dla oliwy / y dla winá.
A w káżdym mieśćie złożył tárcże y włócżnie / á opátrzył je bárzo mocno : á ták pánował nád Judą y Benjáminem.
Kápłáni też y Lewitowie / którzy byli we wƺyſtkim Izráelu / zebráli śię do niego ze wƺyſtkich gránic ſwojich.
Bo opuśćiwƺy Lewitowie przedmieśćiá ſwoje / y ośiádłośći ſwoje / ƺli do Judy y do Jeruzálemu ; ( gdyż ich był wyrzućił Jeroboám y ſynowie jego / áby nie odpráwowáli urzędu kápłáńſkiego Pánu.
Y poſtánowił ſobie kápłánów po wyżynách / y dyjábłom / y ćielcom / których był nácżynił. )
A zá nimi ze wƺyſtkich pokoleń Izráelſkich / którzy obróćili ſercá ſwe ku ƺukániu Páná / Bogá Izráelſkiego / przyƺli do Jeruzálemu / áby ofiárowáli Pánu / Bogu ojców ſwojich.
A ták umocnili króleſtwo Judzkie / y utwierdźili Roboámá / ſyná Sálomonowego / przez trzy látá ; ábowiem chodźili drogą Dawidá y Sálomoná przez one trzy látá.
Potem pojął ſobie Roboám zá żonę Máhálátę córkę Jerymotá / ſyná Dawidowego / y Abiháilę / córkę Elijábá / ſyná Iſájego.
Która mu urodźiłá ſynów : Jehuſá / y Semáryjáƺá / y Záámá.
A po niej pojął Mááchę / córkę Abſálomową / która mu urodźiłá Abijáƺá / y Etájá / y Syſę / y Sálomitę.
Y miłował Roboám Mááchę / córkę Abſálomową / nád wƺyſtkie żony ſwoje / y nád záłożnice ſwoje. Abowiem pojął był żon ośmnáśćie / á záłożnic ƺeśćdźieśiąt / y ſpłodźił dwádźieśćiá y ośm ſynów y ƺeśćdźieśiąt córek.
Y poſtánowił Roboám Abijáƺá / ſyná Mááchy / zá kśięćiá / zá hetmáná miedzy bráćmi jego ; ábowiem zámyślał go ucżynić królem.
A roztropnie ſobie poſtępując / rozſádźił wƺyſtkich innych ſynów ſwych po wƺyſtkich kráinách Judzkich y Benjáminowych / po wƺyſtkich miáſtách obronnych / y opátrzył ich doſtátkiem żywnośći / y nádał im wiele żon.
