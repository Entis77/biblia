A Ezechyjáƺ gdy pocżął królowáć / miáł dwádzieśćiá y pięć lát ; á dwádzieśćiá y dziewięć lát królowáł w Jeruzálemie. Imię mátki jego Abi / córká Zácháryjáƺowá.
A cżynił / co było dobrego przed ocżymá Páńſkimi / według wƺyſtkiego / jáko cżynił Dawid ojćiec jego.
Ten roku pierwƺego królowániá ſwego / mieśiącá pierwƺego / otworzył drzwi domu Páńſkiego / y popráwił je.
Y przywiódł kápłánów y Lewitów / á zgromádził ich ná ulicę wſchodnią.
Y rzekł do nich : Słuchájćie mię Lewitowie : Teráz śię poświęććie : poświęććie też y dom Páná / Bogá ojców wáƺych / y wyrzuććie plugáſtwá z świątnicy :
Abowiem zgrzeƺyli ojcowie náśi / y cżynili złe przed ocżymá Páná / Bogá náƺego / opuƺcżájąc go / y odwrácájąc oblicżá ſwoje od przybytku Páńſkiego / á obrácájąc śię tyłem do niego.
Zámknęli też drzwi u przyśionká / y pogáśili lámpy / á kádzidłem nie kádzili / áni cáłopáleniá ofiárowáli w świątnicy Bogu Izráelſkiemu.
Przetoż był gniew Páńſki nád Judą / y nád Jeruzálemem / á podáł ich ná rozproƺenie / ná ſpuſtoƺenie / y ná pośmiech / jáko ſámi widzićie ocżymá wáƺemi.
Bo oto polegli ojcowie náśi od miecżá / á ſynowie náśi / y córki náƺe / y żony náƺe záwiedzione ſą w niewolę dlá tego.
Teráz tedy umyśliłem ucżynić przymierze z Pánem / Bogiem Izráelſkim / áby odwróćił od náſ gniew popędliwośći ſwojej.
Synowie moi! nie bądźćież już niedbáłymi ; bo wáſ Pán obráł / ábyśćie ſtojąc przed nim ſłużyli mu / á byli ſługámi jego / y kádzili mu.
Tedy powſtáli Lewitowie : Máchát / ſyn Amáſájowy / y Joel / ſyn Azáryjáƺowy / z ſynów Káátowych ; á z ſynów Merárego : Cyſ / ſyn Abdy / y Azáryjáƺ / ſyn Jeháleelowy ; á z Gierſońcżyków : Joách / ſyn Zámy / y Eden / ſyn Joáchowy :
A z ſynów Eliſáfánowych : Symry y Jehijel : á z ſynów Aſáfowych : Zácháryjáƺ y Mátánijáƺ ;
A z ſynów Hemánowych : Jehijel y Symchy ; á z ſynów Jedytunowych : Semejáƺ y Uzyjel.
Y zgromádzili bráći ſwych / którzy poświęćiwƺy śię przyƺli według rozkázániá królewſkiego / y rozkázániá Páńſkiego / áby wycżyśćili dom Páńſki.
A wƺedłƺy kápłáni do domu Páńſkiego / áby go ocżyśćili / wynieśli wƺyſtkie plugáſtwá / które ználeźli w kośćiele Páńſkim / do śieni domu Páńſkiego ; á Lewitowie zábráwƺy to wynieśli precż do potoku Cedron.
Y pocżęli pierwƺego dniá mieśiącá pierwƺego poświęcáć / á dniá óſmego tegoż mieśiącá weƺli do przyśionku Páńſkiego y poświęcáli dom Páńſki przez ośm dni / á dniá ƺeſnáſtego / mieśiącá pierwƺego / dokońcżyli.
Potem weƺli do królá Ezechyjáƺá / y rzekli : Ocżyśćiliśmy wƺyſtek dom Páńſki / y ołtárz cáłopáleniá / y wƺyſtkie nácżyniá jego / y ſtół pokłádny y wƺyſtkie nácżyniá jego ;
Tákże wƺyſtko nácżynie / które był odrzućił król Acház zá królowániá ſwego / gdy grzeƺył / zgotowáliśmy y poświęćili ; á oto ſą przed ołtárzem Páńſkim.
A ták wſtáwƺy ráno król Ezechyjáƺ zgromádził przedniejƺych miáſtá / y ƺedł do domu Páńſkiego.
Y przywiedziono mu ćielców śiedm / y báránów śiedm / y báránków śiedm / y kozłów śiedm / ná ofiárę zá grzech / zá króleſtwo / y zá świątnicę / y zá Judę / á rozkázáł ſynom Aáronowym / kápłánom / áby ofiárowáli ná ołtárzu Páńſkim.
A ták pobili one woły / á kápłáni wziąwƺy krew ich kropili po ołtárzu ; pobili też y bárány / á kropili krwią ich po ołtárzu ; pobili też y báránki / á kropili krwią ich po ołtárzu.
Przywiedli też kozły ná ofiárę zá grzech przed królá y przed zgromádzenie / którzy włożyli ręce ſwoje ná nie.
Y pobili je kápłáni / y ocżyśćili krwią ich ołtárz ná ocżyƺcżenie wƺyſtkiego Izráelá ; ábowiem zá wƺyſtkiego Izráelá rozkázáł król ofiárowáć cáłopálenie y ofiárę zá grzech.
Poſtánowił też y Lewitów w domu Páńſkim z cymbáłámi / y z cytrámi / y z hárfámi / według rozkázániá Dawidowego / y Gádá / widzącego królewſkiego / y Nátáná proroká ; bo to było rozkázánie Páńſkie przez proroków jego.
A ták ſtáli Lewitowie z inſtrumentámi Dawidowemi / y kápłáni z trąbámi.
Y rozkázáł Ezechyjáƺ / áby ofiárowáli cáłopálenie ná ołtárzu ; á gdy śię zácżęło cáłopálenie / pocżęło śię śpiewánie Pánu / y trąbienie / y gránie ná inſtrumentách Dawidá / królá Izráelſkiego.
Tedy wƺyſtko zgromádzenie kłániáło śię / á śpiewácy śpiewáli / y trębácże trąbili ; co wƺyſtko trwáło / póki śię nie dokońcżyło cáłopálenie.
A gdy śię ſkońcżyło cáłopálenie / poklęknęli król y wƺyſcy / którzy z nim byli / y modlili śię.
Tedy rozkázáł król Ezechyjáƺ y kśiążętá Lewitom / áby chwálili Páná ſłowy Dawidowemi / y Aſáfá / widzącego ; y chwálili z weſelem wielkiem / á kłániájąc śię modlili śię.
Zátem rzekł Ezechyjáƺ / mówiąc : Teráześćie poświęćili ręce wáƺe Pánu ; przyſtąpćież / á przywiedźćie ofiáry ſpokojne / y ofiáry chwáły do domu Páńſkiego. Przetoż przywiodło ono zgromádzenie ofiáry ſpokojne y ofiáry chwáły / y káżdy ochotnego ſercá przywiódł ofiáry ná cáłopálenie.
Y byłá licżbá ofiár ná cáłopálenie / które przywiodło zgromádzenie / wołów śiedmdzieśiąt / báránów ſto / báránków dwieśćie / wƺyſtko to ná cáłopálenie Pánu.
Innych też rzecży poświęconych było : wołów ƺeść ſet / y owiec trzy tyśiące.
Lecż kápłánów máło było / ták / iż nie mogli nádążyć odzieráć ze ſkóry wƺyſtkich ofiár ná cáłopálenie : przetoż im pomágáli Lewitowie / bráćiá ich / póki nie dokońcżyli onej prácy / y póki śię nie poświęćili drudzy kápłáni ; ábowiem Lewitowie byli ochotniejśi / áby śię poświęćili / niż kápłáni.
Nádto y cáłopáleniá było bárdzo wiele z tłuſtośćiámi ſpokojnych ofiár / y z mokremi ofiárámi ná cáłopálenie. A ták wygotowáná byłá ſłużbá domu Páńſkiego.
Y weſelił śię Ezechyjáƺ y wƺyſtek lud z tego / co Bóg ludowi przygotowáł ; bo śię tá rzecż byłá z prędká ſtáłá.
