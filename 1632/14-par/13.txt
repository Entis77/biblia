Roku ośmnáſtego królá Jeroboámá / królował Abijáƺ nád Judą.
Trzy látá królował w Jeruzálemie / á imię mátki jego było Michájá / córká Uryjelowá z Gábáá. Y byłá wojná miedzy Abijáƺem y miedzy Jeroboámem.
Przetoż Abijáƺ uƺykował wojſko ludźi bárzo wálecżnych cżtery kroć ſto tyśięcy mężów przebránych ; Jeroboám tákże uƺykował śię przećiwko niemu / májąc ośm kroć ſto tyśięcy mężów przebránych bárzo wálecżnych.
Tedy ſtánął Abijáƺ ná wierzchu góry Semeron / która byłá miedzy górámi Efráimſkiemi / y rzekł : Słuchájćie mię / Jeroboámie y wƺyſtek Izráelu!
Izali wam nie náleży wiedźieć / że Pán / Bóg Izráelſki / dał króleſtwo Dawidowi nád Izráelem ná wieki / onemu y ſynom jego przymierzem trwáłem?
Lecż powſtał Jeroboám / ſyn Nábátowy / ſługá Sálomoná / ſyná Dawidowego / y ſtał śię odpornym pánu ſwemu.
Y zebráli śię do niego mężowie lekkomyślni / á ludźie niepobożni / y zmocnili śię przećiw Roboámowi / ſynowi Sálomonowemu ; á Roboám będąc dźiećięćiem / y ſercá lękliwego / nie mógł śię im mężnie oprzeć :
Zácżem wy śię teraz myślićie zmocnić przećiw króleſtwu Páńſkimu / które jeſt w rękách ſynów Dawidowych / á jeſt was kupá wielka / y máćie z ſobą ćielce złote / których wam nácżynił Jeroboám zá bogów.
Izáżeśćie nie zárzućili kápłanów Páńſkich / ſynów Aáronowych y Lewitów / á pocżyniliśćie ſobie kápłanów / jáko inni národowie źiemſcy? Ktokolwiek przychodźi / áby poświęcone były ręce jego / z ćielcem młodym y z śiedmią báránów / ſtáje śię kápłánem tych / którzy nie ſą bogowie.
Ale my jeſteśmy Páná / Bogá náƺego / y nie opuśćiliśmy go ; á kápłáni / którzy ſłużą Pánu / ſą ſynowie Aáronowi / y Lewitowie / którzy pilnują urzędu ſwego.
Y ofiárują Pánu cáłopalenia ná káżdy záránek / y ná káżdy wiecżór / y kádzą rzecżámi wonnemi / y pokłádáją chleby ná ſtole cżyſtym ; tákże świecżnik złoty / y lámpy jego ſporządzáją / áby gorzáły ná káżdy wiecżór. A ták my ſtrzeżemy rozkázánia Páná / Bogá náƺego / á wyśćie go opuśćili.
Przetoż oto / z námi jeſt ná cżele Bóg y kápłáni jego / y trąby głośne / áby brzmiáły przećiwko wam. Synowie Izráelſcy! nie wálcżćież z Pánem / Bogiem ojców wáƺych ; bo śię wam nie powiedźie.
Międzytem obwiódł záſádzkę Jeroboám / áby ná nich przypádli z tyłu ; y byli jedni w ocży Judźie / á drudzy ná záſádzce z tyłu ich.
Tedy ujrzáwƺy ſynowie Judzcy / że ná nich náſtępowáłá bitwá z przodku y z tyłu / wołáli do Páná / á kápłáni trąbili w trąby.
Ucżynili też okrzyk mężowie Judy. Y ſtáło śię w onym okrzyku mężów Judzkich / że Bóg poráźił Jeroboámá / y wƺyſtkiego Izráelá przed Abijáƺem y Judą.
Y ućiekáli ſynowie Izráelſcy przed Judą ; ále ich podał Bóg w ręce ich.
Y poráźili ich Abijáƺ y lud jego poráżką wielką / ták iż legło pobitych z Izráelá pięć kroć ſto tyśięcy mężów ná wybór.
Przetoż poniżeni ſą ſynowie Izráelſcy náoncżás : á zmocnili śię ſynowie Judzcy / ponieważ ſpolegáli ná Pánu / Bogu ojców ſwych.
Y gonił Abijáƺ Jeroboámá / á odjął mu miáſtá Betel y wśi jego / y Jeſáná y wśi jego / y Efron y wśi jego.
A nie mógł śię więcey pokrzepić Jeroboám zá dni Abijáƺowych / y uderzył go Pán / że umárł.
Ale Abijáƺ zmocnił śię / y pojął ſobie żon cżternaśćie / y ſpłodźił dwádźieśćiá y dwu ſynów / y ƺeſnaśćie córek.
Ale inne ſpráwy Abijáƺowe / y poſtępki jego / y ſłowá jego / zápiſáne ſą w kśiędze proroká Jáddy.
