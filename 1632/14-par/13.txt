Roku ośmnáſtego królá Jeroboámá / królowáł Abijáƺ nád Judą.
Trzy látá królowáł w Jeruzálemie / á imię mátki jego było Michájá / córká Uryjelowá z Gábáá. Y byłá wojná między Abijáƺem y między Jeroboámem.
Przetoż Abijáƺ uƺykowáł wojſko ludźi bárdzo wálecżnych cżtery kroć ſto tyśięcy mężów przebránych ; Jeroboám tákże uƺykowáł śię przećiwko niemu / májąc ośm kroć ſto tyśięcy mężów przebránych bárdzo wálecżnych.
Tedy ſtánął Abijáƺ ná wierzchu góry Semeron / którá byłá między górámi Efráimſkiemi / y rzekł : Słuchájćie mię / Jeroboámie y wƺyſtek Izráelu!
Izáli wám nie náleży wiedźieć / że Pán / Bóg Izráelſki / dáł króleſtwo Dawidowi nád Izráelem ná wieki / onemu y ſynom jego przymierzem trwáłem?
Lecż powſtáł Jeroboám / ſyn Nábátowy / ſługá Sálomoná / ſyná Dawidowego / y ſtáł śię odpornym pánu ſwemu.
Y zebráli śię do niego mężowie lekkomyślni / á ludźie niepobożni / y zmocnili śię przećiw Roboámowi / ſynowi Sálomonowemu ; á Roboám będąc dźiećięćiem / y ſercá lękliwego / nie mógł śię im mężnie oprzeć :
Zácżem wy śię teráz myślićie zmocnić przećiw króleſtwu Páńſkimu / które jeſt w rękách ſynów Dawidowych / á jeſt wáſ kupá wielká / y máćie z ſobą ćielce złote / których wám nácżynił Jeroboám zá bogów.
Izáżeśćie nie zárzućili kápłánów Páńſkich / ſynów Aáronowych y Lewitów / á pocżyniliśćie ſobie kápłánów / jáko inni národowie źiemſcy? Ktokolwiek przychodźi / áby poświęcone były ręce jego / z ćielcem młodym y z śiedmią báránów / ſtáje śię kápłánem tych / którzy nie ſą bogowie.
Ale my jeſteśmy Páná / Bogá náƺego / y nie opuśćiliśmy go ; á kápłáni / którzy ſłużą Pánu / ſą ſynowie Aáronowi / y Lewitowie / którzy pilnują urzędu ſwego.
Y ofiárują Pánu cáłopáleniá ná káżdy záránek / y ná káżdy wiecżór / y kádzą rzecżámi wonnemi / y pokłádáją chleby ná ſtole cżyſtym ; tákże świecżnik złoty / y lámpy jego ſporządzájá / áby gorzáły ná káżdy wiecżór. A ták my ſtrzeżemy rozkázániá Páná / Bogá náƺego / á wyśćie go opuśćili.
Przetoż oto / z námi jeſt ná cżele Bóg y kápłáni jego / y trąby głośne / áby brzmiáły przećiwko wám. Synowie Izráelſcy! nie wálcżćież z Pánem / Bogiem ojców wáƺych ; bo śię wám nie powiedźie.
Międzytem obwiódł záſádzkę Jeroboám / áby ná nich przypádli z tyłu ; y byli jedni w ocży Judźie / á drudzy ná záſádzce z tyłu ich.
Tedy ujrzáwƺy ſynowie Judzcy / że ná nich náſtępowáłá bitwá z przodku y z tyłu / wołáli do Páná / á kápłáni trąbili w trąby.
Ucżynili też okrzyk mężowie Judy. Y ſtáło śię w onym okrzyku mężów Judzkich / że Bóg poráźił Jeroboámá / y wƺyſtkiego Izráelá przed Abijáƺem y Judą.
Y ućiekáli ſynowie Izráelſcy przed Judą ; ále ich podáł Bóg w ręce ich.
Y poráźili ich Abijáƺ y lud jego poráżką wielką / ták iż legło pobitych z Izráelá pięć kroć ſto tyśięcy mężów ná wybór.
Przetoż poniżeni ſą ſynowie Izráelſcy náoncżáſ : á zmocnili śię ſynowie Judzcy / poniewáż ſpolegáli ná Pánu / Bogu ojców ſwych.
Y gonił Abijáƺ Jeroboámá / á odjął mu miáſtá Betel y wśi jego / y Jeſáná y wśi jego / y Efron y wśi jego.
A nie mógł śię więcej pokrzepić Jeroboám zá dni Abijáƺowych / y uderzył go Pán / że umárł.
Ale Abijáƺ zmocnił śię / y pojął ſobie żon cżternáśćie / y ſpłodźił dwádźieśćiá y dwóch ſynów / y ƺeſnáśćie córek.
Ale inne ſpráwy Abijáƺowe / y poſtępki jego / y ſłowá jego / zápiſáne ſą w kśiędze proroká Jáddy.
