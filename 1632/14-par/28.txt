Dwádźieśćiá lat miał Acház / gdy królowáć pocżął / á ƺeſnaśćie lat królował w Jeruzálemie / y nie cżynił / co było dobrego przed ocżymá Páńſkimi / jáko Dawid / oćiec jego ;
Ale chodźił drogámi królów Izráelſkich ; nádto ulał y ſłupy báłwochwálſkie.
Tákże y ſám kádźił w dolinie Benhennon / y pálił ſynów ſwych ogniem według obrzydliwośći pogán / których wygnał Pán przed ſynámi Izráelſkimi.
Ofiárował też y kádźił ná wyżynách / y ná págórkách / y pod káżdym drzewem gáłęźiſtem.
Przetoż dał go Pán / Bóg jego / w rękę królá Syryjſkiego / którzy poráźiwƺy go / pojmáli z ludu jego więźniów wiele / á przywiedli ich do Dámáƺku. Nádto y w rękę królá Izráelſkiego podány jeſt / który go poráźił poráżką wielka.
Abowiem Fácejáƺ / ſyn Romelijáƺowy / pobił w Judźie ſto y dwádźieśćiá tyśięcy dniá jednego / wƺyſtko mężów wálecżnych / przeto / iż opuśćili Páná / Bogá ojców ſwojich.
Zychry tákże / mocárz Efráimſki / zábił Mááſájáƺá / ſyná królewſkiego / y Aſrykámá / przełożonego domu jego / y Elkáná / wtórego po królu.
Nádto pojmáli ſynowie Izráelſcy z bráći ſwych dwá kroć ſto tyśięcy niewiáſt / ſynów / y córek / y bárzo wiele łupów pobráli od nich / y záprowádźili korzyść do Sámáryi.
Y był tám prorok Páńſki / imieniem Obed / który záƺedƺy onemu wojſku idącemu do Sámáryi / rzekł im : Oto / rozgniewáwƺy śię Pán / Bóg ojców wáƺych / ná Judę / podał ich w rękę wáƺę / á wyśćie ich pomordowáli w popędliwośći / która áż do niebá przyƺłá.
A jeƺcże lud z Judy y z Jeruzálemu chcećie ſobie podbić zá niewolników y zá niewolnice ; ázaż y przy was ſámych nie maƺ wyſtępku przećiw Pánu / Bogu wáƺemu?
Przetoż teraz mię ſłuchájćie / á odwiedźćie więźniów / którycheśćie pojmáli z bráći wáƺych ; bo pewnie gniew popędliwośći Páńſkiej wiśi nád wámi.
Tedy powſtáli mężowie z kśiążąt ſynów Efráimowych : Azáryjáƺ / ſyn Johánánowy / Báráchyjáƺ / ſyn Meſyllemotowy / y Ezechyjáƺ / ſyn Sállumowy / y Amáſá / ſyn Hádlájowy przećiwko tym / którzy śię wrácáli z wojny ;
Y rzekli do nich : Nie wodźćie tu tych więźniów ; bo grzech przećiwko Pánu ná nas myślićie przywieść / przycżyniájąc do grzechów náƺych / y do wyſtępków náƺych : bo wielki jeſt grzech náƺ / y gniew popędliwośći nád Izráelem.
Przetoż ono wojſko zoſtáwiło y więźniów y łupy ſwe przed kśiążętámi / y przed wƺyſtkim zgromádzeniem.
A powſtáwƺy mężowie / którzy ſą z imienia miánowáni / wźięli onych więźniów / á wƺyſtkich obnáżonych miedzy nimi przyodźiáli z onych korzyśći / á przyodźiáwƺy ich y dáwƺy im obuwie / nákármili ich / y nápoili ich / y pomázáli ich / y odprowádźili ná oſłách káżdego ſłábego / á przyprowádźili ich do Jerychá / miáſtá pálm / do bráći ich : á potem śię wróćili do Sámáryi.
Náoncżás poſłał król Acház do królów Aſſyryjſkich / áby mu pomoc dáli :
Bo jeƺcże byli przyćiągnęli y Edomcżycy / y poráźili Judę / á nábráli więźniów.
Nádto y Filiſtynowie wtárgnęli do miaſt w równinách / y ná południe do Judy / á wźięli Betſemes / y Ajálon / y Gáderot / y Sokot y wśi jego / y Támne y wśi jego / y Gimzo y wśi jego / á mieƺkáli w nich.
Pán bowiem poniżał Judę dla Acházá / królá Izráelſkiego / przeto / iż odwróćił Judę / áby śię przewrotnie obchodźił z Pánem.
Y przyćiągnął do niego Tyglát Filneſer / król Aſſyryjſki / który go bárdźiej ućiſnął / ániżeli mu pomógł.
A choćiáż pobrał Acház ſkárby z domu Páńſkiego / y z domu królewſkiego / y od kśiążąt / á dał królowi Aſſyryjſkiemu / przećież go nie rátował.
A cżáſu nájwiękƺego ſwego ućiſku przycżyniał grzechów przećiwko Pánu. Tákić był król Acház.
Abowiem ofiárował bogom z Dámáƺku / od których był poráżony / y mówił : Ponieważ bogowie królów Syryjſkich pomágáją im / będę im ofiárował / áby mię rátowáli ; ále mu oni byli ná upádek / y wƺyſtkiemu Izráelowi.
Przetoz pobrał Acház nacżynia domu Bożego / y pokruƺył one nacżynia domu Bożego / á zámknął drzwi domu Páńſkiego / y pobudował ſobie ołtárze po wƺyſtkich kątách w Jeruzálemie.
Tákże y w káżdym mieśćie Judzkiem pocżynił wyżyny / áby kádźił bogom cudzym / y wzruƺył ku gniewu Páná / Bogá ojców ſwojich /
Ale inne ſpráwy jego / y wƺyſtkie poſtępki jego / pierwƺe y poślednie / zápiſáne ſą w kśięgách o królách Judzkich y Izráelſkich.
Y záſnął Acház z ojcámi ſwymi / y pochowáli go w mieśćie Jeruzálemie ; bo go nie wprowádźili do grobów królów Izráelſkich ; á Ezechyjáƺ / ſyn jego / królował miáſto niego.
