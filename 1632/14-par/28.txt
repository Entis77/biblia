Dwádźieśćiá lát miał Acház / gdy królowáć pocżął / á ƺeſnáśćie lát królowáł w Jeruzálemie / y nie cżynił / co było dobrego przed ocżymá Páńſkimi / jáko Dawid / ojćiec jego ;
Ale chodźił drogámi królów Izráelſkich ; nádto uláł y ſłupy báłwochwálſkie.
Tákże y ſám kádźił w dolinie Benhennon / y pálił ſynów ſwych ogniem według obrzydliwośći pogán / których wygnáł Pán przed ſynámi Izráelſkimi.
Ofiárowáł też y kádźił ná wyżynách / y ná págórkách / y pod káżdem drzewem gáłęźiſtem.
Przetoż dáł go Pán / Bóg jego / w rękę królá Syryjſkiego / którzy poráźiwƺy go / pojmáli z ludu jego więźniów wiele / á przywiedli ich do Dámáƺku. Nádto y w rękę królá Izráelſkiego podány jeſt / który go poráźił poráżką wielká.
Abowiem Fácejáƺ / ſyn Romelijáƺowy / pobił w Judźie ſto y dwádźieśćiá tyśięcy dniá jednego / wƺyſtko mężów wálecżnych / przeto / iż opuśćili Páná / Bogá ojców ſwoich.
Zychry tákże / mocárz Efráimſki / zábił Mááſájáƺá / ſyná królewſkiego / y Aſrykámá / przełożonego domu jego / y Elkáná / wtórego po królu.
Nádto pojmáli ſynowie Izráelſcy z bráći ſwych dwá kroć ſto tyśięcy niewiáſt / ſynów / y córek / y bárdzo wiele łupów pobráli od nich / y záprowádźili korzyść do Sámáryi.
Y był tám prorok Páńſki / imieniem Obed / który záƺedłƺy onemu wojſku idącemu do Sámáryi / rzekł im : Oto / rozgniewáwƺy śię Pán / Bóg ojców wáƺych / ná Judę / podáł ich w rękę wáƺę / á wyśćie ich pomordowáli w popędliwośći / która áż do niebá przyƺłá.
A jeƺcże lud z Judy y z Jeruzálemu chcećie ſobie podbić zá niewolników y zá niewolnice ; ázáż y przy was ſámych nie máƺ wyſtępku przećiw Pánu / Bogu wáƺemu?
Przetoż teraz mię ſłuchájćie / á odwiedźćie więźniów / którycheśćie pojmáli z bráći wáƺych ; bo pewnie gniew popędliwośći Páńſkiej wiśi nád wámi.
Tedy powſtáli mężowie z kśiążąt ſynów Efráimowych : Azáryjáƺ / ſyn Johánánowy / Báráchyjáƺ / ſyn Meſyllemotowy / y Ezechyjáƺ / ſyn Sállumowy / y Amáſá / ſyn Hádlájowy przećiwko tym / którzy śię wrácáli z wojny ;
Y rzekli do nich : Nie wodźćie tu tych więźniów ; bo grzech przećiwko Pánu ná nas myślićie przywieść / przycżyniájąc do grzechów náƺych / y do wyſtępków náƺych : bo wielki jeſt grzech náƺ / y gniew popędliwośći nád Izráelem.
Przetoż ono wojſko zoſtáwiło y więźniów y łupy ſwe przed kśiążętámi / y przed wƺyſtkiem zgromádzeniem.
A powſtáwƺy mężowie / którzy ſą z imieniá miánowáni / wźięli onych więźniów / á wƺyſtkich obnáżonych między nimi przyodźiáli z onych korzyśći / á przyodźiáwƺy ich y dáwƺy im obuwie / nákármili ich / y nápoili ich / y pomázáli ich / y odprowádźili ná oſłách káżdego ſłábego / á przyprowádźili ich do Jerychá / miáſtá pálm / do bráći ich : á potem śię wróćili do Sámáryi.
Náoncżáſ poſłáł król Acház do królów Aſſyryjſkich / áby mu pomoc dáli :
Bo jeƺcże byli przyćiągnęli y Edomcżycy / y poráźili Judę / á nábráli więźniów.
Nádto y Filiſtynowie wtárgnęli do miáſt w równinách / y ná południe do Judy / á wźięli Betſemeſ / y Ajálon / y Gáderot / y Sokot y wśi jego / y Támne y wśi jego / y Gimzo y wśi jego / á mieƺkáli w nich.
Pán bowiem poniżáł Judę dla Acházá / królá Izráelſkiego / przeto / iż odwróćił Judę / áby śię przewrotnie obchodźił z Pánem.
Y przyćiągnął do niego Tyglát Filneſer / król Aſſyryjſki / który go bárdźiej ućiſnął / ániżeli mu pomógł.
A choćiáż pobráł Acház ſkárby z domu Páńſkiego / y z domu królewſkiego / y od kśiążąt / á dáł królowi Aſſyryjſkiemu / przećież go nie rátowáł.
A cżáſu nájwiękƺego ſwego ućiſku przycżyniáł grzechów przećiwko Pánu. Tákić był król Acház.
Abowiem ofiárowáł bogom z Dámáƺku / od których był poráżony / y mówił : Poniewáż bogowie królów Syryjſkich pomágáją im / będę im ofiárowáł / áby mię rátowáli ; ále mu oni byli ná upádek / y wƺyſtkiemu Izráelowi.
Przetoz pobráł Acház nácżyniá domu Bożego / y pokruƺył one nácżyniá domu Bożego / á zámknął drzwi domu Páńſkiego / y pobudowáł ſobie ołtárze po wƺyſtkich kątách w Jeruzálemie.
Tákże y w káżdem mieśćie Judzkiem pocżynił wyżyny / áby kádźił bogom cudzym / y wzruƺył ku gniewu Páná / Bogá ojców ſwoich /
Ale inne ſpráwy jego / y wƺyſtkie poſtępki jego / pierwƺe y poślednie / zápiſáne ſą w kśięgách o królách Judzkich y Izráelſkich.
Y záſnął Acház z ojcámi ſwymi / y pochowáli go w mieśćie Jeruzálemie ; bo go nie wprowádźili do grobów królów Izráelſkich ; á Ezechyjáƺ / ſyn jego / królowáł miáſto niego.
