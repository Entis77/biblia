Tedy wźiął lud źiemi Joácházá / ſyná Jozyjáƺowego / á poſtánowili go zá królá ná miejſcu ojcá jego w Jeruzálemie.
Dwádźieśćiá y trzy látá było Joácházowi / gdy pocżął królowáć / á trzy mieśiące królował w Jeruzálemie.
Bo go złożył król Egipſki w Jeruzálemie / y náłożył winę ná onę źiemię ſto tálentów ſrebrá / y tálent złotá.
Y poſtánowił królem król Egipſki Elijákimá / brátá jego / nád Judą y nád Jeruzálemem / y odmienił imię jego / á názwał go Joákim ; á Joácházá / brátá jego / wźiąwƺy Necho / záwiódł go do Egiptu.
Dwádźieśćiá y pięć lat miał Joákim / gdy królowáć pocżął / á jedenaśćie lat królował w Jeruzálemie ; y cżynił złe przed ocżymá Páná Bogá ſwego.
Przećiw któremu wyćiągnął Nábuchodonozor / król Bábilońſki / y związał go dwomá łáńcuchámi miedźiánemi / áby go záwiódł do Bábilonu.
Nacżynia też domu Páńſkiego znióſł Nábuchodonozor do Bábilonu / y oddał je do kośćiołá ſwego w Bábilonie.
A oſtátek ſpráw Joákimowych / y obrzydłośći jego / które cżynił / y cokolwiek śię znájdowáło przy nim / to zápiſáne w kśięgách królów Izráelſkich y Judzkich. A królował Joáchyn / ſyn jego / miáſto niego.
Ośm lat miał Joáchyn / gdy królowáć pocżął / á trzy mieśiące y dźieśięć dni królował w Jeruzálemie ; y cżynił złe przed ocżymá Páńſkimi ;
Potem po roku poſłał król Nábuchodonozor / y kázał go przywieść do Bábilonu / y z nácżyniem koƺtownem domu Páńſkiego / á poſtánowił królem Sedekijáƺá / brátá jego / nád Judą y nád Jeruzálemem.
Dwádźieśćiá y jeden lat miał Sedekijáƺ / gdy królowáć pocżął / á jedenaśćie lat królował w Jeruzálemie.
Y cżynił złe przed ocżymá Páná / Bogá ſwego / á nie upokorzył śię przed Jeremijáƺem prorokiem / który mówił z uſt Páńſkich.
Owƺem y przećiwko królowi Nábuchodonozorowi powſtał / który go był przyśięgą záwiązał przez Bogá ; á zátwárdźiwƺy kárk ſwój upárł śię w ſercu ſwojim / áby śię nie náwróćił do Páná / Bogá Izráelſkiego.
Wƺyſcy też przedniejśi kápłáni / y lud / wielce rozmnożyli niepráwośći według wƺyſtkich obrzydliwośći pogáńſkich ; y ſplugáwili dom Páńſki / który był poświęćił w Jeruzálemie.
A Pán / Bóg ojców ich / poſyłał do nich poſłów ſwojich / á poſyłał ráno wſtáwájąc ; bo folgował ludowi ſwemu / y mieƺkániu ſwemu.
Ale oni ƺydźili z poſłów Bożych / y lekce ſobie powáżyli ſłów jego / á náśmiewáli śię z proroków jego / áż przyƺłá popędliwość Páńſka ná lud jego / ták / iż żádnego ulecżeniá nie było.
Bo przywiódł ná nich królá Cháldejſkiego / który pomordował młodźieńców ich miecżem w domu świątnicy ich / á nie przepuśćił áni młodźieńcowi áni pánnie / ſtárcowi y zgrzybiáłemu ; wƺyſtkich podał w ręce jego.
Nádto wƺyſtkie nacżynia domu Bożego / wielkie y máłe / y ſkárby domu Páńſkiego / y ſkárby królewſkie y kśiążąt jego / wƺyſtko przenióſł do Bábilonu.
Y ſpálili dom Boży / á zburzyli mury Jeruzálemſkie / y wƺyſtkie páłáce jego popálili ogniem / y wƺyſtkie jego nacżynia koƺtowne popſuli.
A tych / którzy uƺli miecżá / przenióſł do Bábilonu ; y byli niewolnikámi jego y ſynów jego áż do królowánia królá Perſkiego ;
Aby śię wypełniło ſłowo Páńſkie powiedźiáne przez uſtá Jeremijáƺowe / áżby odpráwiłá źiemiá ſábáty ſwoje ; bo po wƺyſtkie dni ſpuſtoƺeniá ſwego odpocżywáłá / áż śię wypełniło śiedmdźieśiąt lat.
Potem roku pierwƺego Cyruſá / królá Perſkiego / áby śię wypełniło ſłowo Páńſkie powiedźiáne przez uſtá Jeremijáƺowe / wzbudźił Pán duchá Cyruſá / królá Perſkiego / że kázał obwołáć y rozpiſáć po wƺyſtkim króleſtwie ſwojim / mówiąc :
Ták mówi Cyrus / król Perſki : Wƺyſtkie króleſtwá źiemi dał mi Pán / Bóg niebieſki ; y ten mi rozkázał / ábym mu zbudował dom w Jeruzálemie / które jeſt w Judztwie. Kto tedy jeſt miedzy wámi ze wƺyſtkiego ludu jego / który budowáć chce / z tym niech będźie Pán / Bóg jego / á ten niecháj idźie.
