Tedy wziął lud ziemi Joácházá / ſyná Jozyjáƺowego / á poſtánowili go zá królá ná miejſcu ojcá jego w Jeruzálemie.
Dwádzieśćiá y trzy látá było Joácházowi / gdy pocżął królowáć / á trzy mieśiące królowáł w Jeruzálemie.
Bo go złożył król Egipſki w Jeruzálemie / y náłożył winę ná onę ziemię ſto tálentów ſrebrá / y tálent złotá.
Y poſtánowił królem król Egipſki Elijákimá / brátá jego / nád Judą y nád Jeruzálemem / y odmienił imię jego / á názwáł go Joákim ; á Joácházá / brátá jego / wziąwƺy Necho / záwiódł go do Egiptu.
Dwádzieśćiá y pięć lát miáł Joákim / gdy królowáć pocżął / á jedenáśćie lát królowáł w Jeruzálemie ; y cżynił złe przed ocżymá Páná Bogá ſwego.
Przećiw któremu wyćiągnął Nábuchodonozor / król Bábilońſki / y związáł go dwomá łáńcuchámi miedziánemi / áby go záwiódł do Bábilonu.
Nácżyniá też domu Páńſkiego znióſł Nábuchodonozor do Bábilonu / y oddáł je do kośćiołá ſwego w Bábilonie.
A oſtátek ſpráw Joákimowych / y obrzydłośći jego / które cżynił / y cokolwiek śię znájdowáło przy nim / to zápiſáne w kśięgách królów Izráelſkich y Judzkich. A królowáł Joáchyn / ſyn jego / miáſto niego.
Ośm lát miáł Joáchyn / gdy królowáć pocżął / á trzy mieśiące y dzieśięć dni królowáł w Jeruzálemie ; y cżynił złe przed ocżymá Páńſkimi ;
Potem po roku poſłáł król Nábuchodonozor / y kázáł go przywieść do Bábilonu / y z nácżyniem koƺtownem domu Páńſkiego / á poſtánowił królem Sedekijáƺá / brátá jego / nád Judą y nád Jeruzálemem.
Dwádzieśćiá y jeden lát miáł Sedekijáƺ / gdy królowáć pocżął / á jedenáśćie lát królowáł w Jeruzálemie.
Y cżynił złe przed ocżymá Páná / Bogá ſwego / á nie upokorzył śię przed Jeremijáƺem prorokiem / który mówił z uſt Páńſkich.
Owƺem y przećiwko królowi Nábuchodonozorowi powſtáł / który go był przyśięgą záwiązáł przez Bogá ; á zátwárdziwƺy kárk ſwój upárł śię w ſercu ſwojim / áby śię nie náwróćił do Páná / Bogá Izráelſkiego.
Wƺyſcy też przedniejśi kápłáni / y lud / wielce rozmnożyli niepráwośći według wƺyſtkich obrzydliwośći pogáńſkich ; y ſplugáwili dom Páńſki / który był poświęćił w Jeruzálemie.
A Pán / Bóg ojców ich / poſyłáł do nich poſłów ſwoich / á poſyłáł ráno wſtáwájąc ; bo folgowáł ludowi ſwemu / y mieƺkániu ſwemu.
Ale oni ƺydzili z poſłów Bożych / y lekce ſobie powáżyli ſłów jego / á náśmiewáli śię z proroków jego / áż przyƺłá popędliwość Páńſká ná lud jego / ták / iż żádnego ulecżeniá nie było.
Bo przywiódł ná nich królá Cháldejſkiego / który pomordowáł młodzieńców ich miecżem w domu świątnicy ich / á nie przepuśćił áni młodzieńcowi áni pánnie / ſtárcowi y zgrzybiáłemu ; wƺyſtkich podáł w ręce jego.
Nádto wƺyſtkie nácżyniá domu Bożego / wielkie y máłe / y ſkárby domu Páńſkiego / y ſkárby królewſkie y kśiążąt jego / wƺyſtko przenióſł do Bábilonu.
Y ſpálili dom Boży / á zburzyli mury Jeruzálemſkie / y wƺyſtkie páłáce jego popálili ogniem / y wƺyſtkie jego nácżyniá koƺtowne popſuli.
A tych / którzy uƺli miecżá / przenióſł do Bábilonu ; y byli niewolnikámi jego y ſynów jego áż do królowániá królá Perſkiego ;
Aby śię wypełniło ſłowo Páńſkie powiedziáne przez uſtá Jeremijáƺowe / áżby odpráwiłá ziemiá ſábáty ſwoje ; bo po wƺyſtkie dni ſpuſtoƺeniá ſwego odpocżywáłá / áż śię wypełniło śiedmdzieśiąt lát.
Potem roku pierwƺego Cyruſá / królá Perſkiego / áby śię wypełniło ſłowo Páńſkie powiedziáne przez uſtá Jeremijáƺowe / wzbudził Pán duchá Cyruſá / królá Perſkiego / że kázáł obwołáć y rozpiſáć po wƺyſtkiem króleſtwie ſwojim / mówiąc :
Ták mówi Cyruſ / król Perſki : Wƺyſtkie króleſtwá ziemi dáł mi Pán / Bóg niebieſki ; y ten mi rozkázáł / ábym mu zbudowáł dom w Jeruzálemie / które jeſt w Judztwie. Kto tedy jeſt między wámi ze wƺyſtkiego ludu jego / który budowáć chce / z tym niech będzie Pán / Bóg jego / á ten niecháj idzie.
