A gdy dokońcżył Sálomon modlitwy / tedy ogień zſtąpił z niebá / y pożárł cáłopálenie y inne ofiáry / á chwałá Páńſká nápełniłá on dom.
Y nie mogli kápłáni wnijść do domu Páńſkiego / przeto / że chwałá Páńſká nápełniłá dom Páńſki.
Y wƺyſcy ſynowie Izráelſcy / widząc / gdy zſtępowáł ogień / y chwałá Páńſká ná dom / upádli twárzą ſwą ná źiemię / ná tło / á kłániájąc śię chwálili Páná / że dobry / że ná wieki miłośierdźie jego.
A król y wƺyſtek lud ſpráwowáli ofiáry przed Pánem.
Tedy nábił król Sálomon ná ofiáry wołów dwádźieśćiá y dwá tyśiące / á owiec ſto y dwádźieśćiá tyśięcy / gdy poświęcáli dom Boży król y wƺyſtek lud.
Ale kápłáni ſtáli w rzędách ſwych : Lewitowie tákże z inſtrumentámi muzyki Páńſkiej / które był ſpráwił Dawid król ku chwáleniu Páná / ( że ná wieki miłośierdźie jego ) pieśnią Dawidową / którą im podáł. Inni też kápłáni trąbili przećiwko nim / á wƺyſcy Izráelcżycy ſtáli.
Nádto poświęćił Sálomon pośrodek śieni / która byłá przed domem Páńſkim ; bo tám ofiárowáł cáłopáleniá / y tłuſtośći ſpokojnych ofiár / przeto / że ná ołtárzu miedźiánym / który był ſpráwił Sálomon / nie mogły śię zmieśćić cáłopáleniá / y ofiáry śniedne y tłuſtośći.
Y obchodźił Sálomon onego cżáſu święto urocżyſte przez śiedm dni / y wƺyſtek Izráel z nim / zgromádzenie bárdzo wielkie / od wejśćiá do Emát áż do rzeki Egipſkiej.
Potem ucżynili dniá óſmego święto ; ábowiem poświęcenie ołtárzá ſpráwowáli przez śiedm dni / y święto urocżyſte obchodźili przez śiedm dni.
A dniá dwudźieſtego y trzećiego / mieśiącá śiódmego / rozpuśćił lud do przybytków ſwoich / weſelący śię y ćieƺący śię w ſercu ſwem z dobrodźiejſtwá / które ucżynił Pán Dawidowi y Sálomonowi y Izráelowi / ludowi ſwemu.
A ták dokońcżył Sálomon domu Páńſkiego / y domu królewſkiego / á wƺyſtko / co był umyślił w ſercu ſwem / ucżynić w domu Páńſkim y w domu ſwym / wykonáł ƺcżęśliwie.
Potem ukázáł śię Pán Sálomonowi w nocy / y rzekł do niego : Wyſłucháłem modlitwę twoję / y obráłem to miejſce ſobie zá dom do ofiáry.
Jeſli zámknę niebo / żeby nie było deƺcżu / y jeſli rozkáżę ƺáráńcży / áby pożárłá źiemię ; jeſli też poślę powietrze ná lud mój ;
A jeſliby śię upokorzył lud mój / nád którym wzywáno imieniá mego / á modliłby śię / y ƺukáłby twárzy mojey / odwróćiwƺy śię od dróg ſwoich złych : tedy Ja wyſłuchám z niebá / y odpuƺcżę grzech ich / á uzdrowię źiemię ich.
Ocży też moje otworzone będą / á uƺy moje nákłonione ku modlitwie ucżynioney ná tem miejſcu.
Bom teraz obráł y poświęćił ten dom / áby tu przebywáło imię moje áż ná wieki ; y będą tu ocży moje / y ſerce moje po wƺyſtkie dni.
A ty będźieƺli chodźił przede mną / jáko chodźił Dawid / ojćiec twój / á będźieƺ śię ſpráwowáł według wƺyſtkiego / com ći przykázáł / ſtrzegąc uſtáw moich y ſądów moich :
Tedy utwierdzę ſtolicę króleſtwá twego / jákom ucżynił umowę z Dawidem / ojcem twoim / mówiąc : Nie będźie odjęty z národu twego mąż pánujący nád Izráelem.
Ale jeſli śię wy odwróćićie / á opuśćićie uſtáwy moje / y przykázániá moje / którem wám podáł / á odƺedłƺy będźiećie ſłużyli bogom cudzym / y będźiećie śię im kłániáli :
Tedy ich wykorzenię z źiemi mojey / którąm im dáł ; á ten dom / którym poświęćił imieniowi memu / odrzucę od oblicżá mego / y podám go ná przypowieść / y ná báśń między wƺyſtkie národy.
Nádto y ten dom / który był ſłáwny / káżdemu mimo idącemu będźie ná podźiw / y rzecże : Przecżże ták ucżynił Pán tey źiemi y temu domowi?
Tedy odpowiedzą : Przeto / iż opuśćili Páná / Bogá ojców ſwoich / który ich wywiódł z źiemi Egipſkiej / á chwyćili śię bogów cudzych / y kłániáli śię im / y ſłużyli im / dlátegoż przywiódł ná nich to wƺyſtko złe.
