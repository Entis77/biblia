Y miał Jozáfát bogáctw y ſłáwy bárzo wiele / á ſpowinowáćił śię z Achábem.
Y przyjechał po kilku látách do Achábá do Sámáryi ; y nábił Acháb owiec y wołów wiele dla niego / y dla ludu / który był z nim / y námáwiał go / áby ćiągnął do Rámot Gáláád.
Y rzekł Acháb / król Izráelſki / do Jozáfátá / królá Judzkiego : Poćiągnijże ze mną do Rámot Gáláád? A on mu odpowiedźiał : Jáko ja / ták y ty / á jáko lud twój / ták lud mój / y będźiemy z tobą ná wojnie.
Nádto rzekł Jozáfát do królá Izráelſkiego : Proƺę / pytáj śię dźiś ſłowá Páńſkiego.
A ták zebrał król Izráelſki proroków cżtery ſtá mężów / y rzekł do nich : Mámże ćiągnąć do Rámot Gáláád ná wojnę / cżyli zániecháć? A oni odpowiedźieli : Ciągnij ; bo je da Bóg w ręce królewſkie.
Ale Jozáfát rzekł : Niemáƺże tu jeƺcże którego proroká Páńſkiego / żebyſmy śię go pytáli?
Y rzekł król Izráelſki do Jozáfátá : Jeſt jeƺcże mąż jeden / przez któregobyſmy śię mogli rádźić Páná / ále go ja nienáwidzę ; bo mi nie prorokuje nic dobrego / ále záwżdy złe ; á tenći jeſt Micheáƺ / ſyn Jemlowy. Y rzekł Jozáfát : Niech ták nie mówi król.
Tedy záwołał król Izráelſki niektórego komorniká / y rzekł : Przywiedź tu rychło Micheáƺá / ſyná Jemlowego.
Międzytem król Izráelſki / y Jozáfát / król Judzki / śiedźieli káżdy z nich ná ſtolicy ſwojey / ubráni w ƺáty królewſkie / á śiedźieli ná plácu u wrót bramy Sámáryjſkiej / á wƺyſcy prorocy prorokowáli przed nimi.
A Sedechyjáƺ / ſyn Chánáánowy / ſpráwił ſobie rogi żelázne / y rzekł : Ták mówi Pán : Temi będźieƺ bódł Syryjcżyków / áż ich wyniƺcżyƺ.
Toż wƺyſcy prorocy prorokowáli / mówiąc : Ciągnij do Rámot Gáláád / á będźieć śię ƺcżęśćiło ; ábowiem je podá Pán / w ręce królewſkie.
Tedy poſeł / który chodźił / áby przyzwał Micheáƺá / rzekł do niego / mówiąc : Oto ſłowá proroków jednemi uſty dobrze tuƺą królowi ; niechże będźie / proƺę / ſłowo twoje jáko jednego z nich / á mów dobre rzecży.
Y rzekł Micheáƺ : Jáko żyje Pán / że co mi kolwiek rozkáże Bóg mój / to mówić będę.
A gdy przyƺedł do królá / rzekł król do niego : Micheáƺu! mámyż ćiągnąć ná wojnę przećiw Rámot Gáláád / cżyli zániecháć? A on odpowiedźiał : Ciągnijćie / á poƺcżęśći śię wam / y będą podáni w ręce wáƺe.
Y rzekł król do niego : A wieleż ćię rázy mam przyśięgą zobowiązywáć / ábyś mi nie mówił jedno práwdę w imieniu Páńſkim?
Przetoż rzekł : Widźiałem wƺyſtek lud Izráelſki rozproƺony po górách jáko owce / które nie máją páſterzá ; bo Pán rzekł : Nie májąći Páná ; niech śię wróći káżdy do domu ſwego w pokoju.
Y rzekł król Izráelſki do Jozáfátá : Izáżem ći nie powiádał / że mi nic dobrego prorokowáć nie miał / ále złe?
Ale on rzekł : Słuchájćież tedy ſłowá Páńſkiego : Widźiałem Páná śiedzącego ná ſtolicy jego / y wƺyſtko wojſko niebieſkie ſtojące po práwicy jego y po lewicy jego.
Y rzekł Pán : Kto zwiedźie Achábá / królá Izráelſkiego / áby ƺedł / á poległ w Rámot Gáláád? A gdy mówił jeden ták / á drugi mówił inácżey /
Wyſtąpił duch / y ſtánął przed Pánem / y rzekł : Ja go zwiodę. A Pán mu rzekł : Przez cóż?
Y odpowiedźiał : Wynijdę / á będę kłámliwym duchem w uſtách wƺyſtkich proroków jego. Y rzekł Pán : Zwiedźieƺ / y pewnie przemożeƺ : Idźże / á ucżyń ták.
Przetoż teraz / oto dał Pán duchá kłámliwego w uſtá tych proroków twojich / gdyż Pán wyrzekł przećiwko tobie złe.
Tedy przyſtąpiwƺy Sedechijáƺ / ſyn Chánáánowy / uderzył Micheáƺá w policżek / mówiąc : A którąż drogą odƺedł duch Páńſki odemnie / áby mówił z tobą?
Y odpowiedźiał Micheáƺ : Oto ty ujrzyƺ dniá onego / kiedy wnijdźieƺ do nájſkrytƺej komory / ábyś śię ſkrył.
Y rzekł król Izráelſki : Weźmijćie Micheáƺá / á odwiedźćie go do Amoná / ſtároſty miejſkiego / y do Joázá / ſyná królewſkiego.
Y rzecżećie im : Ták mówi król : Wſádźćie tego do więźieniá / á dáwájćie mu jeść chleb utrapienia / y wodę ućiſku / áż śię wrócę w pokoju.
Ale odpowiedźiał Micheáƺ : Jeſliże śię wróćiƺ w pokoju / tedy nie mówił Pán przez mię. Nádto rzekł : Słuchájćież wƺyſcy ludźie.
A ták ćiągnął król Izráelſki / y Jozáfát / król Judzki / do Rámot Gáláád.
Y rzekł król Izráelſki do Jozáfátá : Odmienię śię / á pójdę do bitwy : ále ty ubierzeƺ śię w ƺáty ſwe. Y odmienił śię król Izráelſki / á ƺli do bitwy.
A król Syryjſki rozkázał był hetmánom / którzy byli nád wozámi jego / mówiąc : Nie potykájćie śię áni z máłym áni z wielkim / tylko z ſámym królem Izráelſkim.
A gdy ujrzeli Jozáfátá hetmáni / co byli nád wozámi / rzekli : Król Izráelſki jeſt. Y obróćili śię przećiw niemu / áby śię z nim potykáli ; ále záwołał Jozáfát / á Pán go rátował ; y odwróćił ich Bóg od niego.
Bo obácżywƺy hetmáni / co byli nád wozámi / że nie ten był król Izráelſki / odwróćili śię od niego.
Lecż niektóry mąż ſtrzelił ná niepewne z łuku / y poſtrzelił królá Izráelſkiego / miedzy nity y miedzy páncerz ; który rzekł woźnicy ſwemu : Náwróć / á wywieź mię z wojſká ; bom jeſt rániony.
Y wzmogłá śię bitwá dniá onego / á król Izráelſki ſtał ná woźie przećiw Syryjcżykom áż do wiecżorá : y umárł / gdy záchodźiło ſłońce.
