A śiódmego roku zmocniwƺy śię Jojádá / záćiągnął rotmiſtrzów / Azáryjáƺá / ſyná Jerohámowego / y Iſmáelá / ſyná Johánánowego / y Azáryjáƺá / ſyná Obedowego / y Mááſájáƺá / ſyná Adájáƺowego / y Elizáfátá / ſyná Zychry / z ſobą w przymierze.
Którzy obchodząc Judzką źiemię zebráli Lewitów ze wƺyſtkich miáſt Judzkich / y przedniejƺych z domów ojcowſkich w Izráelu / á przyƺli do Jeruzálemu.
Y ucżyniło wƺyſtko zgromádzenie przymierze w domu Bożym z królem ; bo im był rzekł Jojádá : Oto ſyn królewſki będźie królowáł / jáko powiedźiáł Pán o ſynách Dawidowych.
Toć jeſt co ucżynićie : Trzećiá cżęść z was / którzy przychodźićie w ſábát z kápłánów y z Lewitów / będźie odźwiernymi w brámách.
A trzećiá cżęść będźie w domu królewſkim / á trzećiá cżęść w brámie fundámentu ; ále wƺyſtek lud zoſtánie w śieniách domu Páńſkiego.
A niecháj nikt nie wchodźi w dom Páńſki / tylko kápłáni / á uſługujący Lewitowie ; ći niecháj wchodzą / ábowiem ſą poświęceni ; ále wƺyſtek inny lud niech trzymá ſtráż Páńſką.
Y obſtąpią Lewitowie królá zewƺąd / májąc káżdy broń ſwą w ręce ſwej ; á ktobykolwiek wƺedł w dom / niech będźie zábity ; á bądźćie przy królu / gdy będźie wchodźił / y gdy będźie wychodźił.
Y ucżynili Lewitowie / y wƺyſtek lud Judzki / według wƺyſtkiego / co był rozkázáł Jojádá kápłán ; y wźiął káżdy mężów ſwych / którzy przychodźili w ſábát / y którzy odchodźili w ſábát / bo był nie rozpuśćił Jojádá kápłán pocżtów ich.
Y rozdáł Jojádá kápłán rotmiſtrzom włócżnie / y tárcże / y puklerze / które były królá Dawidá / które były w domu Bożym.
Poſtáwił też wƺyſtek lud ; á káżdy miał broń w ręce ſwej / od práwej ſtrony domu / áż do lewej ſtrony domu przećiwko ołtárzowi / y domowi / około królá zewƺąd.
Zátem wywiedli ſyná królewſkiego / y włożyli náń koronę y świádectwo / á poſtánowili go królem ; y pomázáli go Jojádá y ſynowie jego / y mówili : Niech żyje król!
Wtem uſłyƺáwƺy Atálijá krzyk zbiegájącego śię ludu / y chwálącego królá / weƺłá do ludu do domu Páńſkiego.
A gdy ujrzáłá / że król ſtáł ná májeſtáćie ſwoim w wejśćiu / y kśiążętá y trąby około królá / y wƺyſtek lud oney źiemi weſelący śię / y trąbiący w trąby / y śpiewáki z inſtrumentámi muzycżnemi / y tych / którzy zácżynáli śpiewánie / tedy rozdárłá Atálijá ƺáty ſwoje / mówiąc : Sprzyśiężenie! ſprzyśiężenie!
Przetoż rozkázáł wynijść Jojádá kápłán rotmiſtrzom y hetmánom wojſká / y rzekł do nich : Wywiedźćie ją z zágrodzeniá kośćiołá / á ktoby zá nią ƺedł / niech będźie zábity miecżem ; bo był kápłán rzekł : Nie zábijájćie jey w domu Páńſkim.
Y ucżynili jey plác. A gdy przyƺłá ku wejśćiu brámy / którą wodzono konie do domu królewſkiego / támże ją zábili.
Tedy ucżynił Jojádá przymierze między Pánem / y między wƺyſtkim ludem / y między królem / áby byli ludem Páńſkim.
Potem wƺedł wƺyſtek lud do domu Báálowego / y zburzyli go / y ołtárze jego / y báłwány jego połámáli / Mátáná tákże kápłáná Báálowego zábili przed ołtárzámi.
Y poſtánowił znowu Jojádá przełożonych nád domem Páńſkim pod rządem kápłánów y Lewitów / których był rozrządźił Dawid w domu Páńſkim / áby ofiárowáli cáłopáleniá Pánu / jáko nápiſáno w zakonie Mojżeƺowym / z weſelem / y z pieśniámi / według rozrządzeniá Dawidowego.
Poſtáwił też odźwiernych u brám domu Páńſkiego / áby tám nie wchodźił niecżyſty dla jákiejkolwiek rzecży.
Potem wźiąwƺy rotmiſtrzów y przedniejƺych / y tych / którzy pánowáli nád ludem / y wƺyſtek lud oney źiemi / prowádźili królá z domu Páńſkiego ; y przyƺli brámą wyżƺą do domu królewſkiego / á poſádźili królá ná ſtolicy króleſtwá.
Y weſelił śię wƺyſtek lud oney źiemi / y uſpokoiło śię miáſto / gdy Atáliję zábili miecżem.
