Dwádźieśćiá y pięć lát miáł Amázyjáƺ / gdy królowáć pocżął / á dwádźieśćiá y dźiewięć lát królowáł w Jeruzálemie. Imię mátki jego Joádáná / z Jeruzálemu.
Y cżynił co było dobrego przed ocżymá Páńſkimi / wƺákże nie doſkonáłem ſercem.
Y ſtáło śię / gdy było utwierdzone króleſtwo jego / że pomordowáł ſługi ſwe / którzy zábili królá / ojcá jego.
Wƺákże ſynów ich niepobił : ále ucżynił / jáko nápiſáno w zákonie / w kśięgách Mojżeƺowych / gdźie przykázáł Pán / mówiąc : Nie umrą ojcowie zá ſynów / áni ſynowie umrą zá ojców / ále káżdy zá grzech ſwój umrze.
Tedy zgromádźił Amázyjáƺ lud Judzki / y poſtánowił ich według domów ojcowſkich zá półkowników y zá rotmiſtrzów po wƺyſtkiem pokoleniu Judowem / y Benjáminowem / á policżywƺy ich od dwudźieſtu lát y wyżej / ználázł ich trzy kroć ſto tyśięcy ná wybór / gotowych do boju / noƺących drzewce y tárcżę.
Nájął tákże zá pieniądze z Izráelá ſto tyśięcy mężów dużych zá ſto tálentów ſrebrá.
Lecż mąż Boży przyƺedł do niego mówiąc : Królu! niech nie wychodźi z tobą wojſko Izráelſkie ; bo nie máƺ Páná z Izráelem y ze wƺyſtkimi ſynámi Efráimowymi.
Ale jeźli mi nie wierzyƺ / idź / y zmocnij śię ku bitwie / á poráźi ćię Bóg przed nieprzyjáćielem ; bo w mocy Bożej jeſt rátowáć / y do upádku przywieść.
Tedy rzekł Amázyjáƺ mężowi Bożemu : A cóż mám cżynić z tem ſtem tálentów / którem dáł wojſku Izráelſkiemu? Odpowiedźiáł mąż Boży : Má Pán / ſkąd ći może dáć dáleko więcej nádto.
Przetoż oddźielił Amázyjáƺ to wojſko / które było przyƺło do niego z Efráimá / áby ƺło ná miejſce ſwe ; y rozgniewáli śię bárdzo ná Judę / y wróćili śię do miejſcá ſwego z wielkim gniewem.
Lecż Amázyjáƺ zmocniwƺy śię / wywiódł lud ſwój / y ćiągnął ná dolinę Soli / y poráźił ſynów Seir dźieśięć tyśięcy.
Dźieśięć tákże tyśięcy żywo pojmáli ſynowie Judzcy / y przywiedli ich ná wierzch ſkáły / y zrzućili ich z wierzchu ſkáły / áż śię wƺyſcy porozpukáli.
Ono záśię wojſko / które rozpuśćił Amázyjáƺ / áby nie ƺło z nim ná wojnę / wtárgnęło do miáſt Judzkich / od Sámáryi áż do Betoronu / á poráźiwƺy w nich trzy tyśiące ludu / zábráli korzyść wielką.
A gdy śię Amázyjáƺ wróćił od poráżki Idumejcżyków / przynióſł z ſobą bogów ſynów Seir / y wyſtáwił ich ſobie zá bogów / á kłániáł śię przed nimi / y kádźił im.
Przetoż rozgniewáł śię Pán bárdzo ná Amázyjáƺá / y poſłáł do niego proroká / który mu rzekł : Przecżże ƺukáƺ bogów ludu tego / którzy nie wyrwáli ludu ſwego z ręki twej?
A gdy on do niego mówił / rzekł mu król : Izáli ćię zá rádcę królewſkiego obráno? Przeſtáń tego / áby ćię nie zábito. A ták przeſtáł prorok ; wƺákże rzekł : Wiem / że ćię umyślił Bóg zátráćić / gdyżeś to ucżynił / á nie uſłucháłeś rády mojej.
Tedy nárádźiwƺy śię Amázyjáƺ / król Judzki / poſłáł do Joázá / ſyná Joácházá / ſyná Jehu / królá Izráelſkiego / mówiąc : Pójdź / á wejrzymy ſobie w ocży.
Y poſłáł Joáz / król Izráelſki / do Amázyjáƺá / królá Judzkiego / mówiąc : Oſet / który jeſt ná Libánie / poſłáł do cedru Libáńſkiego / mówiąc : Dáj córkę twoję ſynowi memu zá żonę. Wtem idąc tędy zwierz polny / który był ná Libánie / podeptáł on oſet.
Myśliłeś : Otom poráźił Edomcżyków ; przetoż wynioſło ćię ſerce twoje / ábyś śię tem chlubił. Siedźże tedy w domu twym ; przecż śię máƺ wdáwáć w to złe / ábyś upádł / ty y Judá z tobą?
Ale nie uſłucháł Amázyjáƺ ; bo to było od Bogá / áby ich podáł w ręce nieprzyjáćielſkie / przeto / że ƺukáli bogów Idumejſkich :
Wyćiągnął tedy Joáz / król Izráelſki / y wejrzeli ſobie w ocży / on y Amázyjáƺ / król Judzki / w Betſemeſ / które jeſt w Judźie.
Y poráżony jeſt Judá przed Izráelem / á poućiekáli káżdy do námiotów ſwoich.
Lecż Amázyjáƺá / królá Judzkiego / ſyná Joázowego / ſyná Joácházá / pojmáł król Izráelſki w Betſemeſ / y przywiódł go do Jeruzálemu / á obálił mury Jeruzálemſkie od brámy Efráimſkiej áż do brámy nárożnej / ná cżteryſtá łokći.
Y zábráł wƺyſtko złoto y ſrebro / y wƺyſtkie nácżyniá / które śię ználázły w domu Bożym u Obededomá y w ſkárbách domu królewſkiego / y ludźi / záſtáwnych / á wróćił śię do Sámáryi.
Y żył Amázyjáƺ / ſyn Joázowy / król Judzki / po śmierći Joázá / ſyná Joácházá / królá Izráelſkiego / piętnáśćie lát.
A inne ſpráwy Amázyjáƺowe / pierwƺe y poślednie / izáli nie ſą zápiſáne w kśięgi królów Judzkich y Izráelſkich?
A od onego cżáſu / jáko odpádł Amázyjáƺ od Páná / ucżyniono przećiwko niemu ſprzyśiężenie w Jeruzálemie. Lecż on ućiekł do Láchiſ ; ále poſłáno zá nim do Láchiſ / y zábito go tám.
A przynióſłƺy go ná koniách / pochowáli go z ojcámi jego w mieśćie Judzkiem.
