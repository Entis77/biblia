Tedy jecháł Roboám do Sychem ; bo w Sychem zebráł śię był wƺyſtek Izráel / áby go poſtánowili królem.
A gdy to uſłyƺáł Jeroboám / ſyn Nábátowy / który był w Egipćie / gdzie był ućiekł przed królem Sálomonem / wróćił śię Jeroboám z Egiptu ;
Bo poſłáli y wezwáli go. Tedy przyƺedł Jeroboám y wƺyſtek Izráel / y rzekli do Roboámá / mówiąc :
Ojćiec twój włożył ná náſ járzmo ćiężkie ; ále ty teráz ulżyj nám niewoli ſrogiej ojcá twego / y járzmá ćiężkiego / które włożył ná náſ / á będziemyć ſłużyli.
Który im rzekł : Po trzech dniách wróććie śię do mnie. Y odƺedł lud.
Międzytem wƺedł król Roboám w rádę z ſtárƺymi / którzy ſtáwáli przed Sálomonem / ojcem jego / zá żywotá jego / mówiąc : Co wy rádzićie? Jákąbym miáł dáć odpowiedź ludowi temu?
Którzy mu odpowiedzieli / mówiąc : Jeźli dziś powolny będzieƺ ludowi temu / á ucżyniƺ im kwoli / y będzieƺ mówił do nich ſłowá łágodne / będą ſługámi twymi po wƺyſtkie dni.
Ale on opuśćiwƺy rádę ſtárƺych / którą mu podáli / wƺedł w rádę z młodzieńcámi / którzy z nim wzrośli y ſtáwáli przed nim.
Y rzekł do nich : Cóż wy rádzićie / ábyśmy odpowiedzieli ludowi temu / którzy rzekli do mnie / mówiąc : Ulżyj tego járzmá / które włożył ojćiec twój ná náſ?
Tedy mu odpowiedzieli oni młodzieńcy / którzy z nim wzrośli / mówiąc : Ták odpowieƺ temu ludowi / którzy do ćiebie rzekli / mówiąc : Ojćiec twój włożył ná náſ ćiężkie járzmo / ále ty ulżyj go nám. Ták rzecżeƺ do nich : Nájmniejƺy pálec mój miężƺy niż biodrá ojcá mego.
Przetoż teráz / ojćiec mój kłádł ná wáſ járzmo ćiężkie / ále já przydám do járzmá wáƺego ; ojćiec mój káráł wáſ bicżykámi / á já wáſ będę káráł korbácżámi.
Przyƺedł tedy Jeroboám y wƺyſtek lud do Roboámá dniá trzećiego / jáko im był rozkázáł król / mówiąc : Wróććie śię do mnie dniá trzećiego.
Y odpowiedziáł im król ſurowie / bo opuśćił król Roboám rádę ſtárców.
A rzekł do nich według rády młodzieńców / mówiąc : Ojćiec mój obćiążył wáſ járzmem ćiężkiem / ále já przydám do niego ; ojćiec mój káráł wáſ bicżykámi / ále já wáſ będę káráł korbácżámi.
Y nie uſłucháł król ludu : ( bo byłá przycżyná od Bogá / áby doſyć ucżynił Pán ſłowu ſwemu / które był powiedziáł przez Achyjáƺá Sylonitcżyká do Jeroboámá / ſyná Nábátowego. )
Przetoż widząc wƺyſtek Izráel / że ich król nie uſłucháł / odpowiedziáł lud królowi / mówiąc : Cóż mámy zá dziáł w Dawidzie? á co zá dziedzictwo w ſynu Iſájowym? Káżdy idź do námiotów ſwych / o Izráelu! á ty Dawidzie! opátrz teráz dom ſwój. Y rozeƺli śię wƺyſcy Izráelcżycy do námiotów ſwoich.
A ták tylko nád ſynámi Izráelſkimi / którzy mieƺkáli w miáſtách Judzkich / królowáł Roboám.
Y poſłáł król Roboám Adorámá / który był poborcą / y ukámionowáli go ſynowie Izráelſcy / że umárł ; przetoż król Roboám wśiádłƺy co rychlej ná wóz / ućiekł do Jeruzálemu.
A ták odſtąpili Izráelcżycy od domu Dawidowego / áż do dniá tego.
