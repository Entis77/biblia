Tedy jechał Roboám do Sychem ; bo w Sychem zebrał śię był wƺyſtek Izráel / áby go poſtánowili królem.
A gdy to uſłyƺał Jeroboám / ſyn Nábátowy / który był w Egipćie / gdźie był ućiekł przed królem Sálomonem / wróćił śię Jeroboám z Egiptu ;
Bo poſłáli y wezwáli go. Tedy przyƺedł Jeroboám y wƺyſtek Izráel / y rzekli do Roboámá / mówiąc :
Oćiec twój włożył ná nas járzmo ćiężkie ; ále ty teraz ulżyj nam niewoli ſrogiej ojcá twego / y járzmá ćiężkiego / które włożył ná nas / á będźiemyć ſłużyli.
Który im rzekł : Po trzech dniách wróććie śię do mnie. Y odƺedł lud.
Międzytem wƺedł król Roboám w rádę z ſtárƺymi / którzy ſtáwáli przed Sálomonem / ojcem jego / zá żywotá jego / mówiąc : Co wy rádźićie? Jákąbym miał dáć odpowiedź ludowi temu?
Którzy mu odpowiedźieli / mówiąc : Jeſli dźiś powolny będźieƺ ludowi temu / á ucżyniƺ im kwoli / y będźieƺ mówił do nich ſłowá łágodne / będą ſługámi twymi po wƺyſtkie dni.
Ale on opuśćiwƺy rádę ſtárƺych / którą mu podáli / wƺedł w rádę z młodźieńcámi / którzy z nim wzrośli y ſtáwáli przed nim.
Y rzekł do nich : Cóż wy rádźićie / ábyſmy odpowiedźieli ludowi temu / którzy rzekli do mnie / mówiąc : Ulżyj tego járzmá / które włożył oćiec twój ná nas?
Tedy mu odpowiedźieli oni młodźieńcy / którzy z nim wzrośli / mówiąc : Ták odpowieƺ temu ludowi / którzy do ćiebie rzekli / mówiąc : Oćiec twój włożył ná nas ćiężkie járzmo / ále ty ulżyj go nam. Ták rzecżeƺ do nich : Nájmniejƺy pálec mój miężƺy niż biodrá ojcá mego.
Przetoż teraz / oćiec mój kłádł ná was járzmo ćiężkie / ále ja przydám do járzmá wáƺego ; oćiec mój kárał was bicżykámi / á ja was będę kárał korbácżámi.
Przyƺedł tedy Jeroboám y wƺyſtek lud do Roboámá dniá trzećiego / jáko im był rozkázał król / mówiąc : Wróććie śię do mnie dniá trzećiego.
Y odpowiedźiał im król ſurowie / bo opuśćił król Roboám rádę ſtárców.
A rzekł do nich według rády młodźieńców / mówiąc : Oćiec mój obćiążył was járzmem ćiężkiem / ále ja przydám do niego ; oćiec mój kárał was bicżykámi / ále ja was będę kárał korbácżámi.
Y nie uſłuchał król ludu : ( bo byłá przycżyná od Bogá / áby doſyć ucżynił Pán ſłowu ſwemu / które był powiedźiał przez Achyjáƺá Sylonitcżyká do Jeroboámá / ſyná Nábátowego. )
Przetoż widząc wƺyſtek Izráel / że ich król nie uſłuchał / odpowiedźiał lud królowi / mówiąc : Cóż mamy zá dźiał w Dawidźie? á co zá dźiedźictwo w ſynu Iſájowym? Káżdy idź do námiotów ſwych / o Izráelu! á ty Dawidźie! opátrz teraz dom ſwój. Y rozeƺli śię wƺyſcy Izráelcżycy do námiotów ſwojich.
A ták tylko nád ſynámi Izráelſkimi / którzy mieƺkáli w miáſtách Judzkich / królował Roboám.
Y poſłał król Roboám Adorámá / który był poborcą / y ukámionowáli go ſynowie Izráelſcy / że umárł ; przetoż król Roboám wśiádƺy co rychlej ná wóz / ućiekł do Jeruzálemu.
A ták odſtąpili Izráelcżycy od domu Dawidowego / áż do dniá tego.
