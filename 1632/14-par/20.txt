Y ſtáło śię potem / że przyćiągnęli ſynowie Moábowi / y ſynowie Ammonowi / á z nimi niektórzy mieƺkájący z Ammonitámi / przećiwko Jozáfátowi ná wojnę.
Y przyƺło / á opowiedźiáno Jozáfátowi / mówiąc : Przyćiągnęło przećiwko tobie wojſko wielkie z zámorzá / z Syryi / á oto ſą w Cháſeſontámár / które jeſt Engáddy.
Y uląkł śię / á obróćił Jozáfát twárz ſwoję / áby ƺukał Páná / y zápowiedźiał poſt wƺyſtkiemu ludowi Judzkiemu.
Tedy śię zgromádźił lud Judzki / áby ƺukáli Páná ; tákże ze wƺyſtkich miaſt Judzkich zeƺli śię ƺukáć Páná.
A ták ſtánął Jozáfát w zgromádzeniu Judzkiem y Jeruzálemſkiem w domu Páńſkim przed śienią nową.
Y rzekł : Pánie / Boże ojców náƺych : Izáliś nie ty ſám Bogiem ná niebie? á nie ty pánujeƺ nád wƺyſtkiemi króleſtwámi národów? ázaż nie w rękách twojich jeſt moc y śiłá? á nie máƺ / ktoby śię mógł oſtáć przed tobą.
Izáliś nie ty / Boże náƺ! wypędźił obywáteli tey źiemi przed twárzą ludu twego Izráelſkiego / y podáłeś ją náśieniu Abráhámá przyjáćielá twego ná wieki?
Y mieƺkáli w niej / y zbudowáli tobie w niej świątnicę dla imieniá twego / mówiąc :
Jeſliby ná nas przyƺło złe / miecż pomſty / álbo powietrze / álbo głód / á ſtániemy przed tym domem / y przed oblicżem twojim / gdyż imię twoje jeſt w domu tym / á záwołámy do ćiebie w ućiſkách náƺych / tedy wyſłucháƺ y wybáwiƺ.
Teraz tedy / oto ſynowie Ammonowi / y Moábowi / y górá Seir / przez którycheś ty nie dopuśćił przejść Izráelowi / gdy ƺli z źiemi Egipſkiej / ále ich minęli / á nie wytráćili ich.
Oto teraz nam oni oddáwáją / gdyż przyƺli / áby nas wyrzućili z dźiedźictwá twego / któreś nam dał práwem dźiedźicżnem.
O Boże náƺ! izali ich nie będźieƺ ſądźił? W náſći záiſte nie máƺ żádney mocy przećiw mnóſtwu ták wielkiemu / które przyƺło ná nas / y nie wiemy / co cżynić mamy / tylko ku tobie obrácámy ocży náƺe.
Wƺyſtek też lud Judzki ſtał przed Pánem / y dźiátki ich / żony ich / y ſynowie ich.
Ale Jeházyjel / ſyn Zácháryjáƺá / ſyná Benájáƺowego / ſyná Jehyjelowego / ſyná Mátánijáƺowego / Lewitá z ſynów Aſáfowych / ná którego przyƺedł Duch Páńſki w pośród onego zgromádzenia /
Rzekł : Słuchájćie wƺyſcy z Judy / y obywátele Jeruzálemſcy / y ty królu Jozáfáćie! Ták wam powiádá Pán : Nie bójćie śię wy / áni śię lękájćie tego mnóſtwá ták wielkiego ; nie wáƺá jeſt wálká / ále Bożá.
Jutro śię ruƺćie przećiwko nim ; oto oni pójdą ſtroną góry Sys / y znájdźiećie ich ná końcu potoku przećiw puƺcży Jeruel.
Nie wy śię potykáć będźiećie w tey bitwie ; ſtáwćie śię / y ſtójćie / á oglądájćie wybáwienie Páńſkie nád wámi / o Judo / y Jeruzálemie! Nie bójćież śię / áni śię lękájćie ; jutro wynijdźiećie przećiwko nim / á Pán będźie z wámi.
Y pokłonił śię Jozáfát twárzą ku źiemi / á wƺyſtek lud Judzki / y obywátele Jeruzálemſcy pádli przed oblicżem Páńſkim / kłániájąc śię Pánu.
Wſtáli też Lewitowie z ſynów Káátowych / y z ſynów Korego / y chwálili Páná / Bogá Izráelſkiego / głoſem wielkim y wynioſłym.
Potem wſtáwƺy ráno ćiągnęli ná puƺcżę Tekuá ; á gdy wychodźili / ſtánął Jozáfát / y rzekł : Słuchájćie mię Judo / y obywátele Jeruzálemſcy! Wierzćie Pánu / Bogu wáƺemu / á będźiećie bezpiecżni ; wierzćież prorokom jego / á będźie śię wam ƺcżęśćiło.
A wƺedƺy w rádę z ludem / poſtánowił śpiewáków Pánu / którzyby go chwálili w ozdobie świętobliwośći / idąc przed uƺykowánymi do bitwy / y mówiąc : Wyſłáwiájćie Páná / ábowiem ná wieki miłośierdźie jego.
A wtencżás / gdy oni zácżęli śpiewánie y chwały / obróćił Pán záſádzkę ná ſynów Ammonowych y Moábowych / y ná obywáteli góry Seir / która byłá przyƺłá przećiw Judźie / y bili śię ſámi.
Bo powſtáli ſynowie Ammonowi y Moábowi przećiwko obywátelom góry Seir / áby ich pobili y wygłádźili. A gdy już do końcá poráźili onych / co mieƺkáli w Seir / oburzył śię jeden przećiw drugiemu / áż śię ſpólnie wybili.
A wtem lud Judzki przyƺedł do Máſfá / bliſko puƺcży ; y ſpojrzáwƺy ná ono mnóſtwo widźieli / á oto trupy leżáły po źiemi / ták iż nikt nie uƺedł.
Przetoż przyƺedł Jozáfát y lud jego / áby rozchwyćili łupy ich ; y ználeſli przy nich bárzo wiele bogáctw y ná trupách klejnotów koƺtownych / których rozchwyćili miedzy śię ták wiele / że ich zánieść nie mogli : przez trzy dni bráli one łupy / ábowiem ich było bárzo wiele.
Potem dniá cżwártego zebráli śię w dolinie Beráchá ; bo iż tám błogoſłáwili Páná / przetoż názwáli imię miejſcá onego doliną Beráchá / áż do dźiśiejƺego dniá.
Zátem obróćili śię wƺyſcy mężowie Judzcy y Jeruzálemſcy / y Jozáfát przed nimi / áby śię wróćili do Jeruzálemu z rádośćią ; ábowiem ućieƺył ich był Pán nád nieprzyjáćiołámi ich.
Y wjecháli do Jeruzálemu z hárfámi / y z cytrámi / y z trąbámi / do domu Páńſkiego.
Tedy przypádł ſtrách Boży ná wƺyſtkie króleſtwá źiemi / gdy uſłyƺáły / iż Pán wálcżył przećiw nieprzyjáćiołom ludu Izráelſkiego.
A ták uſpokoiło śię króleſtwo Jozáfátowe / bo mu dał odpocżnienie Bóg jego zewƺąd.
Y królował Jozáfát nád Judą / á miał trzydźieśći y pięć lat / gdy królowáć pocżął / á dwádźieśćiá y pięć lat królował w Jeruzálemie ; á imię mátki jego było Azubá / córká Sálájowá.
A chodźił drogą ojcá ſwego Azy / y nie uchylał śię od niej / cżyniąc co było dobrego w ocżách Páńſkich.
Wƺákże wyżyny nie były znieśione ; bo jeƺcże lud nie zgotował był ſercá ſwego ku Bogu ojców ſwojich.
A oſtátek ſpráw Jozáfátowych pierwƺych y poślednich jeſt zápiſány w kśiędze Jehu / ſyná Hánániego / któremu było rozkázáne / áby to włożył w kśięgi królów Izráelſkich.
Potem ſtowárzyƺył śię Jozáfát / król Judzki / z Ochozyjáƺem / królem Izráelſkim / którego ſpráwy były bárzo niepobożne.
A ſtowárzyƺył śię z nim ná to / áby nábudowáli okrętów / któreby chodźiły do Tárſys ; y budowáli one okręty w Aſyongáber.
Przetoż prorokował Eliezer / ſyn Dodáwáhowy z Máreſy / przećiw Jozáfátowi mówiąc : Iżeś śię ſtowárzyƺył z Ochozyjáƺem / rozerwał Pán ſpráwy twoje / y porozbijáły śię okręty / y nie mogły iść do Tárſys.
