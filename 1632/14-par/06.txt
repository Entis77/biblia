Tedy rzekł Sálomon : Pán powiedźiał / iż mieƺkáć miał we mgle.
A jam zbudował dom ná mieƺkánie tobie / o Pánie! y miejſce / gdźiebyś miał mieƺkáć ná wieki.
A obróćiwƺy król oblicże ſwe błogoſłáwił wƺyſtkiemu zgromádzeniu Izráelſkiemu / ( á wƺyſtko zgromádzenie Izráelſkie ſtáło. )
Y rzekł : Błogoſłáwiony Pán / Bóg Izráelſki / który mówił uſty ſwemi do Dawidá / ojcá mego / y wypełnił to ſkutecżnie / mówiąc :
Ode dniá / któregom wywiódł lud mój z źiemi Egipſkiej / nie obráłem miáſtá ze wƺyſtkich pokoleń Izráelſkich ku zbudowániu domu / gdźieby przebywáło imię moje / ánim obrał mężá / któryby był wodzem nád ludem moim Izráelſkim.
Alem obrał Jeruzálem / áby tám przebywáło imię moje ; obráłem też y Dawidá / áby był nád ludem moim Izráelſkim.
Poſtánowił był Dawid oćiec mój / w ſercu ſwem / zbudowáć dom imieniowi Páná / Bogá Izráelſkiego.
Ale rzekł Pán do Dawidá / ojcá mego : Acżkolwiekeś był poſtánowił w ſercu ſwem / zbudowáć dom imieniowi memu / y dobrześ ucżynił / żeś to umyślił w ſercu ſwem :
Wƺákże ty nie będźieƺ budował tego domu ; ále ſyn twój / który wynijdźie z biódr twych / ten zbuduje dom imieniowi memu.
A ták utwierdźił Pán ſłowo ſwoje / które był powiedźiał ; bom ja powſtał miáſto Dawidá / ojcá mego / á uśiádłem ná ſtolicy Izráelſkiej / jáko był powiedźiał Pán / y zbudowáłem ten dom imieniowi Páná / Bogá Izráelſkiego.
Támżem też poſtáwił ſkrzynię oną / w którey jeſt przymierze Páńſkie / które ucżynił z ſynámi Izráelſkimi.
Tedy ſtánął Sálomon przed ołtárzem Páńſkim / przed wƺyſtkim zgromádzeniem Izráelſkiem / y wyćiągnął ręce ſwe.
Abowiem był ucżynił Sálomon ſtolec miedźiány / który poſtáwił w pośród śieni / ná pięć łokći wdłuż / á ná pięć łokći wƺerz / á ná trzy łokćie wzwyż ; y wſtąpił nań / á poklęknąwƺy ná koláná ſwoje przed wƺyſtkim zgromádzeniem Izráelſkiem / wyćiągnął ręce ſwe ku niebu /
Y rzekł : Pánie / Boże Izráelſki! nie maƺ tobie podobnego Bogá ná niebie y ná źiemi / który chowáƺ umowę y miłośierdźie nád ſługámi twymi / którzy chodzą przed tobą cáłem ſercem ſwem ;
Któryś ſpełnił ſłudze twemu Dawidowi / ojcu memu / coś powiedźiał ; y coś mówił uſty twemi / toś ſkutecżnie wypełnił / jáko śię to dźiś pokázuje.
Przetoż teraz / o Pánie / Boże Izráelſki! ſpełń ſłudze twemu Dawidowi / ojcu memu / coś mu był powiedźiał / mówiąc : Nie będźie odjęty mąż z narodu twego przed twárzą moją / áby nie miał śiedźieć ná ſtolicy Izráelſkiej / jeſli tylko przeſtrzegáć będą ſynowie twoi drogi ſwej / chodząc w zakonie moim / jákoś ty chodźił przed oblicżnośćią moją.
Przetoż teraz / o Pánie / Boże Izráelſki! niech będźie utwierdzone ſłowo twoje / któreś mówił do ſługi twego Dawidá.
( Acżći wpráwdźie / izali Bóg będźie mieƺkał z cżłowiekiem ná źiemi? Oto niebioſá / y niebá niebios nie mogą ćię ogárnąć / jákoż dáleko mniej ten dom / którym zbudował? )
A wejrzyj ná modlitwę ſługi twego / y ná prośbę jego / o Pánie / Boże mój! wyſłucháj wołánie y modlitwę / którą śię modli ſługá twój przed tobą ;
Aby były ocży twoje otworzone nád tym domem we dnie y w nocy ; nád tym miejſcem / o któremeś powiedźiał / że tu przebywáć będźie imię twoje : ábyś wyſłuchiwał modlitwę / którą śię będźie modlił ſługá twój ná tym miejſcu.
Wyſłuchájże tedy prośbę ſługi twego / y ludu twego Izráelſkiego / którąć śię modlić będą ná tym miejſcu ; ty wyſłucháj z miejſcá mieƺkánia twego / z niebá / á wyſłucháwƺy bądź miłośćiw.
Gdyby cżłowiek zgrzeƺył przećiwko bliźniemu ſwemu / á przywiódłby go do przyśięgi / ták żeby przyśięgáć muśiał / á przyƺłáby przyśięgá oná przed ołtárz twój w tym domu :
Ty wyſłucháj z niebá / á rozeznáj y rozſądź ſług twojich / potępiájąc niezbożnego / á obrácájąc drogę jego ná głowę jego / y uſpráwiedliwiájąc ſpráwiedliwego / á oddáwájąc mu według ſpráwiedliwośći jego.
Gdyby też był poráżony lud twój Izráelſki od nieprzyjáćielá / przeto / iż zgrzeƺyli przećiwko tobie / á náwróćiliby śię / wyznáwájąc imię twoje / y modląc śię przepráƺáliby ćię w tym domu :
Ty wyſłucháj z niebá / á odpuść grzech ludowi twemu Izráelſkiemu / á przywróć ich záśię do źiemi / którąś im dał y ojcom ich.
Tákże gdyby záwárte było niebo / y nie byłoby deƺcżu / przeto / że zgrzeƺyli tobie / á modliliby śię ná tym miejſcu / wyznáwájąc imię twoje / y od grzechu ſwego odwróćiliby śię / gdybyś ich utrápił :
Ty wyſłucháj z niebá / á odpuść grzech ſług twojich / y ludu twego Izráelſkiego / náucżywƺy ich drogi práwej / po którey chodźić máją / á dáj deƺcż ná źiemię twoję / którąś dał ludowi twemu w dźiedźictwo.
Byłliby głód ná źiemi / byłliby mór / ſuƺá / y rdzá / ƺáráńcżá y chrząƺcże ; jeſliby go śćiſnął nieprzyjáćiel jego w źiemi mieƺkánia jego / álbo jákákolwiek plágá / álbo jákákolwiek niemoc :
Wƺelką modlitwę / y wƺelką prośbę / którąby cżynił którykolwiek cżłowiek / álbo wƺyſtek lud twój Izráelſki / poznáwƺy káżdy z nich káránie ſwoje / y boleść ſwoję / y podnióſłby ręce ſwe w tym domu :
Ty wyſłucháj z niebá / z miejſcá mieƺkánia twego / á odpuść y oddáj káżdemu według wƺyſtkich dróg jego / które znáƺ w ſercu jego / ( bo ty / ty ſám znáƺ ſercá ſynów ludzkich. )
Aby śię ćiebie báli / y chodźili drogámi twemi po wƺyſkie dni / których żyć będą ná źiemi / którąś dał ojcom náƺym.
Nádto y cudzoźiemiec / który nie jeſt z ludu twego Iráelſkiego / y ten przyjdźieli z źiemi dálekiej dla imienia twego wielkiego / y ręki twojey możney / y dla rámieniá twego wyćiągnionego ; przyjdąli á modlić śię będą w tym domu :
Ty wyſłucháj z niebá / z miejſcá mieƺkánia twego / y ucżyń to wƺyſtko / o co záwołá do ćiebie on cudzoźiemiec / áby poználi wƺyſcy národowie źiemſcy imię twoje / y báli śię ćiebie jáko lud twój Izráelſki / y wiedźieli / że imię twoje wzywáne jeſt nád domem tym / którym zbudował.
Gdyby wyƺedł lud twój ná wojnę przećiwko nieprzyjáćiołom ſwoim / drogą / którąbyś ich poſłał / y modlilićby śię / obróćiwƺy śię ku temu miáſtu / któreś obrał / y ku domowi / którym zbudował imieniowi twemu :
Wyſłuchájże z niebá modlitwy ich / y prośbę ich / á wykonáj ſąd ich.
Gdyby zgrzeƺyli przećiwko tobie / ( bo nie maƺ cżłowieká / coby nie grzeƺył ) á rozgniewáwƺy śię ná nich / podáłbyś ich pod moc nieprzyjáćielowi / któryby ich pojmáwƺy záwiódł ich w niewolę do źiemi dálekiej / álbo bliſkiej ;
A upámiętáliby śię w ſercu ſwojim w oney źiemi / do którey ſą záprowádzeni w niewolę / á náwróćiwƺy śię / modliliby śię w źiemi niewoli ſwojey / mówiąc : Zgrzeƺyliśmy źleśmy ucżynili / y niepobożnieśmy śię ſpráwowáli ;
A náwróćiliby śię do ćiebie z cáłego ſercá ſwego / y z cáłej duƺy ſwej w źiemi niewoli ſwojey / do którey będąc pojmáni záprowádzeni byli / á modliliby śię / obróćiwƺy śię ku źiemi ſwej / którąś dał ojcom ich / y ku miáſtu / któreś obrał / y ku domowi / którym zbudował imieniowi twemu :
Wyſłuchájże z niebá / z miejſcá mieƺkánia twego / modlitwę ich y prośbę ich / á wykonáj ſąd ich / y odpuść ludowi twemu / który tobie zgrzeƺył.
A ták teraz / o Boże! proƺę / niech będą ocży twoje otworzone / y uƺy twoje nákłonione ku modlitwie / ucżynioney ná tym miejſcu.
Teraz tedy powſtáń / o Pánie Boże! ku odpocżnieniu twemu / ty y ſkrzyniá mocy twojey ; kápłáni twoi / o Pánie Boże! niech będą oblecżeni zbáwieniem / á święći twoi niecháj śię w dobrách rádują.
O Pánie Boże! nie odwrácáj oblicża od pomázáńcá twego ; pámiętáj ná miłośierdźie obiecáne Dawidowi / ſłudze twemu.
