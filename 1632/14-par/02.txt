Umyślił tedy Sálomon budowáć dom imieniowi Páńſkimu / y páłác królewſki dlá śiebie.
Y nálicżył Sálomon śiedmdzieśiąt tyśięcy mężów / co nośili ćiężáry / á ośmdzieśiąt tyśięcy mężów / którzy rąbáli drzewo ná górze á do nich przyſtáwów trzy tyśiące y ƺeść ſet.
Wypráwił też Sálomon do Hirámá / królá Tyrſkiego / mówiąc : Jákoś śię obchodził z Dawidem / ojcem moim / poſyłájąc mu drzewo cedrowe / áby zbudowáł ſobie dom do mieƺkániá / ták śię obejdź ze mną.
Oto já chcę budowáć dom imieniowi Páná / Bogá mego / ábym mu go poświęćił / á iżbym kádził przed nim rzecżámi wonnemi / y dlá uſtáwicżnego pokłádániá chlebá / y dlá cáłopáleniá poránnego / y wiecżornego w ſábáty / y ná nowiu mieśiąców / y w świętá urocżyſte Páná / Bogá náƺego / co má być ná wieki w Izráelu.
A dom / który budowáć mám / wielki będzie ; ábowiem więkƺy jeſt Bóg náƺ nád wƺyſtkich bogów.
Acż któż ták wiele przemoże / áby mu mógł dom zbudowáć? poniewáż go niebioſá / y niebá niebioſ ogárnąć nie mogą ; á já cóżem jeſt / żebym mu dom budowáć miáł? chybá tylko dlá kádzeniá przed nim.
Przetoż teráz poślij mi mężá umiejętnego / coby umiáł robić złotem / y ſrebrem / y miedzią / y żelázem / y ƺárłátem / y kármázynem / y hijácyntem / á coby umiáł ryſowáć y rzezáć z innymi umiejętnymi / którzy przy mnie ſą w Judzie y w Jeruzálemie / których ſporządził Dawid / ojćiec mój.
Poślij mi też drzewá cedrowego / jodłowego / y álmugimowego z Libánu ; bo já wiem / iż ſłudzy twoi umieją wyrąbywáć drzewo ná Libánie ; á oto ſłudzy moi będą z ſługámi twoimi /
Aby mi wygotowáli drzewá co nájwięcej ; bo dom / który já budowáć chcę / wielki má być ná podziw.
A oto robotnikom ſługom twoim / którzy máją wyćináć drzewo / dám pƺenicy mełtej korcy dwádzieśćiá tyśięcy / y jęcżmieniá korcy dwádzieśćiá tyśięcy / y winá wiáder dwádzieśćiá tyśięcy / y oliwy báreł dwádzieśćiá tyśięcy.
Tedy odpowiedziáł Hirám / król Tyrſki / przez piſánie / które poſłáł do Sálomoná : Iż umiłowáł Pán lud ſwój / poſtánowił ćię nád nim królem.
Y przydáł Hirám mówiąc : Błogoſłáwiony Pán / Bóg Izráelſki / który ucżynił niebo y ziemię / który dáł Dawidowi królowi ſyná mądrego / y umiejętnego / rozumnego / y roztropnego / áby budowáł dom Pánu / y páłác królewſki dlá śiebie.
Poſłáłem ći tedy mężá mądrego / y umiejętnego / y roztropnego / Chirámá Abijego.
Syná niewiáſty z córek Dánowych / którego ojćiec był obywátel Tyrſki ; który umiáł robić złotem / ſrebrem / miedzią / żelázem / kámieniem / y drzewem / y ƺárłátem / hijácyntem ze lnu ſubtelnego / y jedwábiu kármázynowego ; który umie rzezáć wƺelkie rzezánie / y wymyślić rozmáitą miſterną robotę / którą mu zádádzą z mądrymi twymi / y z mądrymi páná mego Dawidá / ojcá twego.
Pƺenicę tylko / y jęcżmień / oliwę / y wino / co obiecáł pán mój / niech przyśle ſługom ſwym.
A my náwyćinámy drzewá z Libánu / ileć go będzie potrzebá / y ſpuśćimyć je ná trátwách po morzu do Joppy / á ty je káżeƺ zwieść do Jeruzálemu.
Oblicżył tedy Sálomon wƺyſtkich cudzoziemców / którzy byli w ziemi Izráelſkiej po onem policżeniu / którem ich policżył Dawid / ojćiec jego / y ználázło śię ich ſto y pięćdzieśiąt tyśięcy / y trzy tyśiące y ƺeść ſet.
A poſtánowił z nich śiedmdzieśiąt tyśięcy / co ćiężáry nośili / á ośmdzieśiąt tyśięcy tych / co wyrąbywáli ná górze / á trzy tyśiące y ƺeść ſet przyſtáwów nád robotámi ludu.
