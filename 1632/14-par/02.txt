Umyślił tedy Sálomon budowáć dom imieniowi Páńſkimu / y páłác królewſki dla śiebie.
Y nálicżył Sálomon śiedmdźieśiąt tyśięcy mężów / co nośili ćiężáry / á ośmdźieśiąt tyśięcy mężów / którzy rąbáli drzewo ná górze á do nich przyſtáwów trzy tyśiące y ƺeść ſet.
Wypráwił też Sálomon do Hirámá / królá Tyrſkiego / mówiąc : Jákoś śię obchodźił z Dawidem / ojcem moim / poſyłájąc mu drzewo cedrowe / áby zbudował ſobie dom do mieƺkánia / ták śię obejdź ze mną.
Oto ja chcę budowáć dom imieniowi Páná / Bogá mego / ábym mu go poświęćił / á iżbym kádźił przed nim rzecżámi wonnemi / y dla uſtáwicżnego pokłádánia chlebá / y dla cáłopalenia poránnego / y wiecżornego w ſábáty / y ná nowiu mieśiąców / y w świętá urocżyſte Páná / Bogá náƺego / co ma być ná wieki w Izráelu.
A dom / który budowáć mam / wielki będźie ; ábowiem więkƺy jeſt Bóg náƺ nád wƺyſtkich bogów.
Acż któż ták wiele przemoże / áby mu mógł dom zbudowáć? poniewáż go niebioſá / y niebá niebios ogárnąć nie mogą ; á ja cóżem jeſt / żebym mu dom budowáć miał? chybá tylko dla kádzeniá przed nim.
Przetoż teraz poślij mi mężá umiejętnego / coby umiał robić złotem / y ſrebrem / y miedźią / y żelázem / y ƺárłátem / y kármázynem / y hijácyntem / á coby umiał ryſowáć y rzezáć z innymi umiejętnymi / którzy przy mnie ſą w Judźie y w Jeruzálemie / których ſporządźił Dawid / oćiec mój.
Poślij mi też drzewá cedrowego / jodłowego / y álmugimowego z Libánu ; bo ja wiem / iż ſłudzy twoi umieją wyrąbywáć drzewo ná Libánie ; á oto ſłudzy moi będą z ſługámi twoimi /
Aby mi wygotowáli drzewá co nájwięcey ; bo dom / który ja budowáć chcę / wielki ma być ná podźiw.
A oto robotnikom ſługom twoim / którzy máją wyćináć drzewo / dam pƺenicy mełtey korcy dwádźieśćiá tyśięcy / y jęcżmieniá korcy dwádźieśćiá tyśięcy / y winá wiáder dwádźieśćiá tyśięcy / y oliwy báreł dwádźieśćiá tyśięcy.
Tedy odpowiedźiał Hirám / król Tyrſki / przez piſánie / które poſłał do Sálomoná : Iż umiłował Pán lud ſwój / poſtánowił ćię nád nim królem.
Y przydał Hirám mówiąc : Błogoſłáwiony Pán / Bóg Izráelſki / który ucżynił niebo y źiemię / który dał Dawidowi królowi ſyná mądrego / y umiejętnego / rozumnego / y roztropnego / áby budował dom Pánu / y páłác królewſki dla śiebie.
Poſłáłem ći tedy mężá mądrego / y umiejętnego / y roztropnego / Chirámá Abijego.
Syná niewiáſty z córek Dánowych / którego oćiec był obywátel Tyrſki ; który umiał robić złotem / ſrebrem / miedźią / żelázem / kámieniem / y drzewem / y ƺárłátem / hijácyntem ze lnu ſubtelnego / y jedwábiu kármázynowego ; który umie rzezáć wƺelkie rzezánie / y wymyślić rozmáitą miſterną robotę / którą mu zádádzą z mądrymi twymi / y z mądrymi páná mego Dawidá / ojcá twego.
Pƺenicę tylko / y jęcżmień / oliwę / y wino / co obiecał pán mój / niech przyśle ſługom ſwym.
A my náwyćinámy drzewá z Libánu / ileć go będźie potrzebá / y ſpuśćimyć je ná trátwách po morzu do Joppy / á ty je káżeƺ zwieść do Jeruzálemu.
Oblicżył tedy Sálomon wƺyſtkich cudzoźiemców / którzy byli w źiemi Izráelſkiej po onem policżeniu / którem ich policżył Dawid / oćiec jego / y ználázło śię ich ſto y pięćdźieśiąt tyśięcy / y trzy tyśiące y ƺeść ſet.
A poſtánowił z nich śiedmdźieśiąt tyśięcy / co ćiężáry nośili / á ośmdźieśiąt tyśięcy tych / co wyrąbywáli ná górze / á trzy tyśiące y ƺeść ſet przyſtáwów nád robotámi ludu.
