Pſálm ſámego Dawidá. O miłośierdźiu y o ſądźie śpiewáć będę ; tobie / o Pánie! śpiewáć będę.
Oſtrożnym będę ná drodze uprzejmej / kiedy przyjdźieƺ do mnie ; będę chodźił uſtáwicżnie w ƺcżerośći ſercá mego / w domu moim.
Nie położę przed ocży moje złej rzecży ; káżdą ſpráwę wyſtępników mam w nienáwiśći / á nie chwyći śię mnie.
Serce przewrotne odſtąpi odemnie / á o złe nie będę dbał.
Tego / który potájemnie obmáwiá bliźniego ſwego / wytnę ; ocżów wynioſłych / y ſercá nádętego nie będę mógł ćierpieć.
Ocży moje obrócone będą ná práwdomównych w źiemi / áby śiádáli zemną ; kto chodźi drogą uprzejmą / ten mi ſłużyć będźie.
Nie będźie mieƺkał w domu moim zdrájcá ; ten / który mówi kłámſtwo / nie oſtoi śię przed ocżymá memi.
Co poránek tráćić będę wƺyſtkich niezbożnych ná źiemi / ábym ták wykorzenił z miáſtá Páńſkiego wƺyſtkich / którzy cżynią niepráwość.
