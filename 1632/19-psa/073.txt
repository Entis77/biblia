Pſálm Aſáfowy. Záiſteć dobry jeſt Bóg Izráelowi / tym / którzy ſą cżyſtego ſercá.
Ale nogi moje máło śię były nie potknęły / á bliſko tego było / że máło nie ƺwánkowáły kroki moje /
Gdym był záwiśćią poruƺon przećiwko ƺálonym / widząc ƺcżęśćie niepobożnych.
Bo nie máją związków áż do śmierći / ále w cáłośći zoſtáje śiłá ich.
W prácy ludzkiej nie ſą / á káźni / jáko inni ludźie / nie doznáwáją.
Przetoż otocżeni ſą pychą / jáko łáńcuchem złotym / á przyodźiáni okrutnośćią / jáko ƺátą ozdobną.
Wyſtąpiły od tłuſtośći ocży ich / á więcej máją nád pomyślenie ſercá.
Rozpuśćili śię / y mówią złośliwie / o ućiśnieniu bárdzo hárdźie mówią.
Wyſtáwiáją przećiwko niebu uſtá ſwe / á język ich krąży po źiemi.
A przetoż ná to przychodźi lud jego / gdy śię im wody już wierzchem leją /
Że mówią : Jákoż má Bóg o tem wiedźieć? álbo máli o tem wiádomość Nájwyżƺy?
Abowiem / oto ći niezbożnymi będąc / máją pokój ná świećie / y nábywáją bogáctw.
Próżno tedy w cżyſtośći chowám ſerce moje / á w niewinnośći ręce moje omywám.
Poniewáż mię cáły dźień biją / á káránie ćierpię ná káżdy poránek.
Rzekęli : Będę też ták o tem mówił / tedy rodzáj ſynów twoich rzecże / żem im niepráw.
Chćiáłemći tego rozumem dośćignąć / ále mi śię tu trudno zdáło ;
Ażem wƺedł do świątnicy Bożej / á tum porozumiáł dokońcżenie ich.
Zpráwdęś ich ná miejſcách śliſkich poſtáwił / á podájeƺ ich ná ſpuſtoƺenie.
Oto jákoć przychodzą ná ſpuſtoƺenie! niemál w okámgnieniu niƺcżeją y giną od ſtráchu.
Są jáko ſen temu / co ocućił ; Pánie! gdy ich obudźiƺ obráz ich lekce powáżyƺ.
Gdy zgorzkło ſerce moje / á nerki moje ćierpiáły kłućie :
Zgłupiáłem był / á nicem nie rozumiáł / byłem przed tobą jáko bydlę.
A wƺákże záwżdy byłem z tobą ; boś mię trzymáł zá práwą rękę moję.
Według rády ſwej prowádź mię / á potem do chwáły przyjmieƺ mię.
Kogożbym innego miáł ná niebie? Y ná źiemi oprócż ćiebie w nikim innym upodobániá nie mám.
Choć ćiáło moje / y ſerce moje uſtánie / jednák Bóg jeſt ſkáłą ſercá mego / y dźiáłem moim ná wieki.
Gdyż oto ći / którzy śię oddáláją od ćiebie / zginą ; wytrácáƺ tych / którzy cudzołożą odſtępowániem od ćiebie.
Aleć mnie nájlepƺá jeſt trzymáć śię Bogá ; przetoż pokłádám w Pánu pánującym nádźieję moję / ábym opowiádáł wƺyſtkie ſpráwy jego.
