W tobie / Pánie! nádźieję mam : niech ná wieki poháńbiony nie będę.
Według ſpráwiedliwośći twej wybáw mię / y wyrwij mię ; nákłoń ku mnie uchá twego / y záchowáj mię.
Bądź mi ſkáłą mieƺkánia / gdźiebym záwżdy uchodźił ; przykázáłeś / áby mię ſtrzeżono ; boś ty ſkáłą moją y twierdzą moją.
Boże mój! wyrwij mię z ręki niezbożniká / z ręki przewrotnego y gwáłtowniká ;
Abowiemeś ty ocżekiwániem mojim / Pánie! Pánie! nádźiejo mojá od młodośći mojey.
Ná tobiem ſpoległ záráz z żywotá ; tyś mię wywiódł z żywotá mátki mojey ; w tobie chwałá mojá záwżdy.
Jáko dźiwowiſko byłem u wielu ; wƺákże tyś jeſt mocną nádźieją moją.
Niecháj będą nápełnione uſtá moje chwáłą twoją / przez cáły dźień ſłáwą twoją.
Nie odrzucájże mię w ſtárośći mojey ; gdy uſtánie śiłá mojá / nie opuƺcżáj mię.
Abowiem mówili nieprzyjáćiele moi przećiwko mnie / á ći / którzy cżyháli ná duƺę moję / rádę ucżynili ſpołem /
Mówiąc : Bóg go opuśćił / gońćież go / á pojmijćie go ; boć nie ma / ktoby go wyrwał.
Boże! nie oddálájże śię odemnie ; o Boże mój! poſpieƺże śię ná rátunek mój.
Niechże będą záwſtydzeni / niech zginą przećiwnicy duƺy mojey ; niech będą okryći zelżywośćią y wſtydem / którzy mi ƺukáją złego.
Aleć ja záwżdy ocżekiwáć będę / á tym więcey rozƺerzę chwałę twoję.
Uſtá moje opowiádáć będą ſpráwiedliwość twoję / cáły dźień zbáwienie twoje / ácżkolwiek licżby jego nie wiem.
Przyſtąpię do wyſłáwiánia wƺelákiej mocy Páná pánującego / będę wſpominał właſną ſpráwiedliwość twoję.
Boże! ucżyłeś mię od młodośći mojey / y opowiádám áż po dźiś dźień dźiwne ſpráwy twoje.
A przetoż áż do ſtárośći y ſędźiwośći nie opuƺcżáj mię / Boże! áż opowiem rámię twoje temu národowi / y wƺyſtkim potomkom moc twoję.
Bo ſpráwiedliwość twojá / Boże! wywyżƺoná jeſt / cżyniƺ zájiſte rzecży wielkie. Boże! któż jeſt podobien tobie?
Który / ácżeś przypuśćił ná mię wielkie y ćiężkie ućiſki / wƺákże záśię do żywotá przywrácáƺ mię / á z przepáśći źiemſkich záśię wywodźiƺ mię.
Rozmnożyƺ doſtojność moję á záśię ućieƺyƺ mię.
A ja też wyſłáwiáć ćię będę ná inſtrumentách muzycżnych / y práwdę twoję / Boże mój! będęć śpiewał przy hárfie / o Święty Izráelſki!
Rozrádują śię wárgi moje / gdyć będę śpiewał / y duƺá mojá / którąś wykupił.
Nádto y język mój będźie opowiádał przez cáły dźień ſpráwiedliwość twoję ; bo śię záwſtydźić / y háńbę odnieść muśieli ći / którzy ƺukáli nieƺcżęśćiá mego.
