W tobie / Pánie! nádzieję mám : niech ná wieki poháńbiony nie będę.
Według ſpráwiedliwośći twej wybáw mię / y wyrwij mię ; nákłoń ku mnie uchá twego / y záchowáj mię.
Bądź mi ſkáłą mieƺkániá / gdziebym záwżdy uchodził ; przykázáłeś / áby mię ſtrzeżono ; boś ty ſkáłą moją y twierdzą moją.
Boże mój! wyrwij mię z ręki niezbożniká / z ręki przewrotnego y gwáłtowniká ;
Abowiemeś ty ocżekiwániem mojem / Pánie! Pánie! nádziejo mojá od młodośći mojej.
Ná tobiem ſpoległ záráz z żywotá ; tyś mię wywiódł z żywotá mátki mojej ; w tobie chwáłá mojá záwżdy.
Jáko dziwowiſko byłem u wielu ; wƺákże tyś jeſt mocną nádzieją moją.
Niecháj będą nápełnione uſtá moje chwáłą twoją / przez cáły dzień ſłáwą twoją.
Nie odrzucájże mię w ſtárośći mojej ; gdy uſtánie śiłá mojá / nie opuƺcżáj mię.
Abowiem mówili nieprzyjáćiele moi przećiwko mnie / á ći / którzy cżyháli ná duƺę moję / rádę ucżynili ſpołem /
Mówiąc : Bóg go opuśćił / gońćież go / á pojmijćie go ; boć nie má / ktoby go wyrwáł.
Boże! nie oddálájże śię odemnie ; o Boże mój! poſpieƺże śię ná rátunek mój.
Niechże będą záwſtydzeni / niech zginą przećiwnicy duƺy mojej ; niech będą okryći zelżywośćią y wſtydem / którzy mi ƺukáją złego.
Aleć já záwżdy ocżekiwáć będę / á tem więcej rozƺerzę chwáłę twoję.
Uſtá moje opowiádáć będą ſpráwiedliwość twoję / cáły dzień zbáwienie twoje / ácżkolwiek licżby jego nie wiem.
Przyſtąpię do wyſłáwiániá wƺelákiej mocy Páná pánującego / będę wſpomináł włáſną ſpráwiedliwość twoję.
Boże! ucżyłeś mię od młodośći mojej / y opowiádám áż po dziś dzień dziwne ſpráwy twoje.
A przetoż áż do ſtárośći y ſędziwośći nie opuƺcżáj mię / Boże! áż opowiem rámię twoje temu národowi / y wƺyſtkim potomkom moc twoję.
Bo ſpráwiedliwość twojá / Boże! wywyżƺoná jeſt / cżyniƺ záiſte rzecży wielkie. Boże! któż jeſt podobien tobie?
Który / ácżeś przypuśćił ná mię wielkie y ćiężkie ućiſki / wƺákże záśię do żywotá przywrácáƺ mię / á z przepáśći ziemſkich záśię wywodziƺ mię.
Rozmnożyƺ doſtojność moję á záśię ućieƺyƺ mię.
A já też wyſłáwiáć ćię będę ná inſtrumentách muzycżnych / y práwdę twoję / Boże mój! będęć śpiewáł przy hárfie / o Święty Izráelſki!
Rozrádują śię wárgi moje / gdyć będę śpiewáł / y duƺá mojá / którąś wykupił.
Nádto y język mój będzie opowiádáł przez cáły dzień ſpráwiedliwość twoję ; bo śię záwſtydzić / y háńbę odnieść muśieli ći / którzy ƺukáli nieƺcżęśćiá mego.
