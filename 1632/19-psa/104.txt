Błogoſłáw / duƺo mojá! Pánu. Pánie / Boże mój! wielceś jeſt wielmożnym ; chwałę y ozdobę przyoblokłeś.
Przyodźiáłeś śię świátłośćią jáko ƺátą ; rozćiągnąłeś niebioſá jáko oponę.
Któryś záſklepił ná wodách páłáce ſwoje ; który używáƺ obłoków miáſto wozów ; który chodźiƺ ná ſkrzydłách wiátrowych ;
Który cżyniƺ duchy poſłámi ſwymi ; ty cżyniƺ ſługi ſwe ogniem páłájącym.
Ugruntowáłeś źiemię ná ſłupách jey / ták / że śię nie poruƺy ná wieki wiecżne.
Przepáśćią jáko ƺátą przyodźiáłeś ją był / ták / że wody ſtáły nád górámi.
Ná zgromienie twoje rozbiegły śię / á ná głos pogromu twego prędko zućiekáły.
Wſtąpiły góry / zniżyły śię doliny ná miejſce / któreś im záłożył.
Zámierzyłeś im kres / áby go nie przeſtępowáły / áni śię wrácáły ná okryćie źiemi.
Który wypuƺcżáƺ źródłá po dolinách / áby płynęły miedzy górámi /
A nápój dáwáły wƺyſtkiemu zwierzowi polnemu ; á z nich gáƺą leśne oſły prágnienie ſwoje.
Przy nich mieƺká ptáſtwo niebieſkie / á z pośród gáłązek głos wydáje.
Który pokrápiáƺ góry z páłáców ſwojich / áby śię z owoców ſpráw twojich náſycáłá źiemiá.
Zá twoją ſpráwą rośćie tráwá dla bydłá / á źiołá ná pożytek cżłowiecży ; ty wywodźiƺ chleb z źiemi :
Y wino / które uweſelá ſerce cżłowiecże / od którego śię lśni twárz jáko od oleju ; y chleb / który zátrzymuje żywot ludzki.
Náſycone bywáją y drzewá Páńſkie / y cedry Libánu / których náſádźił ;
Ná których ptáki gniázdá ſwe máją / y boćián ná jedlinách ma dom ſwój.
Góry wyſokie dźikim kozom / á ſkáły ſą ućiecżką królikom.
Ucżynił mieśiąc dla pewnych cżáſów / á ſłońce zná záchód ſwój.
Przywodźiƺ ćiemność / y bywá noc / w którą wychodzą wƺyſtkie zwierzętá leśne.
Lwiętá rycżą do łupu / y ƺukáją od Bogá pokármu ſwego.
Lecż gdy ſłońce wznijdźie / záś śię zgromádzáją / y w jámách ſwojich kłádą śię.
Tedy wychodźi cżłowiek do roboty ſwojey / y do prácy ſwojey áż do wiecżorá.
O jákoż wielkie ſą ſpráwy twoje / Pánie! te wƺyſtkie mądrześ ucżynił / á nápełnioná jeſt źiemiá bogáctwem twojim.
W morzu záś wielkiem y bárzo ƺerokiem / tám ſą płázy / którym nie máƺ licżby / y zwierzętá máłe y wielkie.
Po nim okręty przechodzą / y wieloryb / któregoś ty ſtworzył / áby w nim igrał.
Wƺyſtko to ná ćię ocżekuje / ábyś im dał pokárm cżáſu ſwego.
Gdy im dájeƺ / zbieráją ; gdy otwieráƺ rękę twoję / náſycone bywáją dobremi rzecżámi.
Lecż gdy ukrywáƺ oblicże twoje / trwożą ſobą ; gdy odbieráƺ duchá ich / giną / y w proch śię ſwój obrácáją.
Gdy wyſyłáƺ duchá twego / ſtworzone bywáją / y odnáwiáƺ oblicże źiemi.
Niechájże będźie chwałá Páńſká ná wieki ; niech śię rozweſeli Pán w ſpráwách ſwojich.
On gdy wejrzy ná źiemię / zádrży ; dotknie śię gór / á zákurzą śię.
Będę śpiewał Pánu zá żywotá mego ; będę śpiewał Bogu memu / póki mię ſtáje.
O nim będźie wdźięcżná mowá mojá / á ja śię rozweſelę w Pánu.
Oby byli wytráceni grzeƺnicy z źiemi / á niezbożnych áby już nie było! Błogoſłáw / duƺo mojá! Pánu. Hállelujá.
