Błogoſłáw / duƺo mojá! Pánu. Pánie / Boże mój! wielceś jeſt wielmożnym ; chwáłę y ozdobę przyoblokłeś.
Przyodziáłeś śię świátłośćią jáko ƺátą ; rozćiągnąłeś niebioſá jáko oponę.
Któryś záſklepił ná wodách páłáce ſwoje ; który używáƺ obłoków miáſto wozów ; który chodziƺ ná ſkrzydłách wiátrowych ;
Który cżyniƺ duchy poſłámi ſwymi ; ty cżyniƺ ſługi ſwe ogniem páłájącym.
Ugruntowáłeś ziemię ná ſłupách jej / ták / że śię nie poruƺy ná wieki wiecżne.
Przepáśćią jáko ƺátą przyodziáłeś ją był / ták / że wody ſtáły nád górámi.
Ná zgromienie twoje rozbiegły śię / á ná głoſ pogromu twego prędko zućiekáły.
Wſtąpiły góry / zniżyły śię doliny ná miejſce / któreś im záłożył.
Zámierzyłeś im kreſ / áby go nie przeſtępowáły / áni śię wrácáły ná okryćie ziemi.
Który wypuƺcżáƺ źródłá po dolinách / áby płynęły między górámi /
A nápój dáwáły wƺyſtkiemu zwierzowi polnemu ; á z nich gáƺą leśne oſły prágnienie ſwoje.
Przy nich mieƺká ptáſtwo niebieſkie / á z pośród gáłązek głoſ wydáje.
Który pokrápiáƺ góry z páłáców ſwoich / áby śię z owoców ſpráw twoich náſycáłá ziemiá.
Zá twoją ſpráwą rośnie tráwá dlá bydłá / á ziołá ná pożytek cżłowiecży ; ty wywodziƺ chleb z ziemi :
Y wino / które uweſelá ſerce cżłowiecże / od którego śię lśni twárz jáko od oleju ; y chleb / który zátrzymuje żywot ludzki.
Náſycone bywáją y drzewá Páńſkie / y cedry Libánu / których náſádził ;
Ná których ptáki gniázdá ſwe máją / y boćián ná jedlinách má dom ſwój.
Góry wyſokie dzikim kozom / á ſkáły ſą ućiecżką królikom.
Ucżynił mieśiąc dlá pewnych cżáſów / á ſłońce zná záchód ſwój.
Przywodziƺ ćiemność / y bywá noc / w którą wychodzą wƺyſtkie zwierzętá leśne.
Lwiętá rycżą do łupu / y ƺukáją od Bogá pokármu ſwego.
Lecż gdy ſłońce wznijdzie / záś śię zgromádzáją / y w jámách ſwoich kłádą śię.
Tedy wychodzi cżłowiek do roboty ſwojej / y do prácy ſwojej áż do wiecżorá.
O jákoż wielkie ſą ſpráwy twoje / Pánie! te wƺyſtkie mądrześ ucżynił / á nápełnioná jeſt ziemiá bogáctwem twojim.
W morzu záś wielkiem y bárdzo ƺerokiem / tám ſą płázy / którym nie máƺ licżby / y zwierzętá máłe y wielkie.
Po nim okręty przechodzą / y wieloryb / któregoś ty ſtworzył / áby w nim igráł.
Wƺyſtko to ná ćię ocżekuje / ábyś im dáł pokárm cżáſu ſwego.
Gdy im dájeƺ / zbieráją ; gdy otwieráƺ rękę twoję / náſycone bywáją dobremi rzecżámi.
Lecż gdy ukrywáƺ oblicże twoje / trwożą ſobą ; gdy odbieráƺ duchá ich / giną / y w proch śię ſwój obrácáją.
Gdy wyſyłáƺ duchá twego / ſtworzone bywáją / y odnáwiáƺ oblicże ziemi.
Niechájże będzie chwáłá Páńſká ná wieki ; niech śię rozweſeli Pán w ſpráwách ſwoich.
On gdy wejrzy ná ziemię / zádrży ; dotknie śię gór / á zákurzą śię.
Będę śpiewáł Pánu zá żywotá mego ; będę śpiewáł Bogu memu / póki mię ſtáje.
O nim będzie wdzięcżná mowá mojá / á já śię rozweſelę w Pánu.
Oby byli wytráceni grzeƺnicy z ziemi / á niezbożnych áby już nie było! Błogoſłáw / duƺo mojá! Pánu. Hállelujá.
