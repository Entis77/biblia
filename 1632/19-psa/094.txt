Boże pomſt! Pánie Boże pomſt! rozjáśnij śię!
Podnieś śię / o Sędźio wƺyſtkiej źiemi! á dáj zápłátę pyƺnym.
Dokądże niepobożni / Pánie! dokądże niepobożni rádowáć śię będą?
Długoż będą świegotáć y hárdźie mówić / chlubiąc śię wƺyſcy / którzy cżynią niepráwość?
Lud twój / Pánie! trzeć / á dźiedźictwo twoje trápić?
Wdowy y przychodniów mordowáć? á śierotki zábijáć?
Mówiąc : Nie widźi tego Pán / áni tego rozumie Bóg Jákóbowy.
Zrozumićież / o wy bydlęcy między ludźmi! á wy ƺáleni kiedyż zrozumiećie?
Izáli ten / który ƺcżepił ucho / nie ſłyƺy? y który ukƺtáłtowáł oko / izáli nie widźi?
Izáli ten / który ćwicży národy / nie będźie káráł? który ucży cżłowieká umiejętnośći.
Pán zná myśli ludzkie / iż ſą ƺcżerą márnośćią.
Błogoſłáwiony jeſt mąż / którego ty ćwicżyƺ / Pánie! á zakonu twego ucżyƺ go.
Abyś mu ſpráwił pokój od złych dni / áżby był wykopány dół niezbożnikowi.
Abowiem / nie opuśći Pán ludu ſwego / á dźiedźictwá ſwego nie zániechá.
Ale áż ku ſpráwiedliwośći obróći śię ſąd / á zá nim wƺyſcy ſercá uprzejmego.
Któżby śię był záſtáwił zá mną przećiwko złośnikom? ktoby śię był ujął o mnie przećiwko tym / którzy cżynią niepráwość?
By mi był Pán nie przybył ná pomoc / máłoby byłá nie mieƺkáłá duƺá mojá w milcżeniu.
Jużem był rzekł : Záchwiáłá śię nogá mojá ; ále miłośierdźie twoje / o Pánie! zátrzymáło mię.
W wielkośći utrápieniá mego / we wnętrznośćiách moich / poćiechy twoje rozweſeláły duƺę moję.
Izáli z tobą towárzyƺy ſtolicá niepráwośći tych / którzy ſtánowią krzywdę miáſto práwá?
Którzy śię zbieráją przećiwko duƺy ſpráwiedliwego / á krew niewinną potępiáją?
Ale Pán jeſt twierdzą moją / á Bóg mój ſkáłą ufnośći mojey.
Onći obróći ná nich niepráwość ich / á dla złośći ich wytráći ich ; wytráći ich Pán / Bóg náƺ.
