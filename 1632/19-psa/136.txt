Wyſłáwiájćież Páná / ábowiem jeſt dobry ; ábowiem ná wieki miłośierdźie jego.
Wyſłáwiájćież Bogá nád bogámi ; ábowiem ná wieki miłośierdźie jego.
Wyſłáwiájćież Páná nád pánámi ; ábowiem ná wieki miłośierdźie jego ;
Tego / który ſám cżyni cudá wielkie ; ábowiem ná wieki miłośierdźie jego.
Który mądrze niebioſá ucżynił ; ábowiem ná wieki miłośierdźie jego ;
Który rozćiągnął źiemię ná wodách ; ábowiem ná wieki miłośierdźie jego ;
Który ucżynił świátłá wielkie ; ábowiem ná wieki miłośierdźie jego ;
Słońce / áby pánowáło we dnie ; ábowiem ná wieki miłośierdźie jego ;
Mieśiąc y gwiázdy / áby pánowáły w nocy ; ábowiem ná wieki miłośierdźie jego.
Który poráźił Egipcżán ná pierworodnych ich ; ábowiem ná wieki miłośierdźie jego.
Który wywiódł Izráelá z pośrodku ich ; ábowiem ná wieki miłośierdźie jego ;
W ręce mocney y w rámieniu wyćiągnionem ; ábowiem ná wieki miłośierdźie jego.
Który rozdźielił morze Cżerwone ná rozdźiáły ; ábowiem ná wieki miłośierdźie jego ;
Y przeprowádźił lud Izráelſki pośrodkiem jego ; ábowiem ná wieki miłośierdźie jego.
Y wrzućił Fáráoná z wojſkiem jego w morze Cżerwone ; ábowiem ná wieki miłośierdźie jego.
Który prowádźił lud ſwój przez puƺcżę ; ábowiem ná wieki miłośierdźie jego.
Który poráźił królów wielkich ; ábowiem ná wieki miłośierdźie jego ;
Y pobił królów możnych ; ábowiem ná wieki miłośierdźie jego ;
Sehoná / królá Amorejſkiego ; ábowiem ná wieki miłośierdźie jego ;
Y Ogá / królá Báſáńſkiego ; ábowiem ná wieki miłośierdźie jego.
Y dáł źiemię ich w dźiedźictwo ; ábowiem ná wieki miłośierdźie jego ;
W dźiedźictwo Izráelowi / ſłudze ſwemu ; ábowiem ná wieki miłośierdźie jego.
Który w uniżeniu náƺem pámiętá ná nas ; ábowiem ná wieki miłośierdźie jego.
Y wybáwił nas od nieprzyjáćiół náƺych ; ábowiem ná wieki miłośierdźie jego.
Który dáje pokárm wƺelkiemu ćiáłu ; ábowiem ná wieki miłośierdźie jego.
Wyſłáwiájćież Bogá niebioſ ; ábowiem ná wieki miłośierdźie jego.
