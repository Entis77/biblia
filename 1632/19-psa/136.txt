Wyſłáwiájćież Páná / ábowiem jeſt dobry ; ábowiem ná wieki miłośierdzie jego.
Wyſłáwiájćież Bogá nád bogámi ; ábowiem ná wieki miłośierdzie jego.
Wyſłáwiájćież Páná nád pánámi ; ábowiem ná wieki miłośierdzie jego ;
Tego / który ſám cżyni cudá wielkie ; ábowiem ná wieki miłośierdzie jego.
Który mądrze niebioſá ucżynił ; ábowiem ná wieki miłośierdzie jego ;
Który rozćiągnął ziemię ná wodách ; ábowiem ná wieki miłośierdzie jego ;
Który ucżynił świátłá wielkie ; ábowiem ná wieki miłośierdzie jego ;
Słońce / áby pánowáło we dnie ; ábowiem ná wieki miłośierdzie jego ;
Mieśiąc y gwiázdy / áby pánowáły w nocy ; ábowiem ná wieki miłośierdzie jego.
Który poráził Egipcżán ná pierworodnych ich ; ábowiem ná wieki miłośierdzie jego.
Który wywiódł Izráelá z pośrodku ich ; ábowiem ná wieki miłośierdzie jego ;
W ręce mocnej y w rámieniu wyćiągnionem ; ábowiem ná wieki miłośierdzie jego.
Który rozdzielił morze Cżerwone ná rozdziáły ; ábowiem ná wieki miłośierdzie jego ;
Y przeprowádził lud Izráelſki pośrodkiem jego ; ábowiem ná wieki miłośierdzie jego.
Y wrzućił Fáráoná z wojſkiem jego w morze Cżerwone ; ábowiem ná wieki miłośierdzie jego.
Który prowádził lud ſwój przez puƺcżę ; ábowiem ná wieki miłośierdzie jego.
Który poráził królów wielkich ; ábowiem ná wieki miłośierdzie jego ;
Y pobił królów możnych ; ábowiem ná wieki miłośierdzie jego ;
Sehoná / królá Amorejſkiego ; ábowiem ná wieki miłośierdzie jego ;
Y Ogá / królá Báſáńſkiego ; ábowiem ná wieki miłośierdzie jego.
Y dáł ziemię ich w dziedzictwo ; ábowiem ná wieki miłośierdzie jego ;
W dziedzictwo Izráelowi / ſłudze ſwemu ; ábowiem ná wieki miłośierdzie jego.
Który w uniżeniu náƺem pámiętá ná náſ ; ábowiem ná wieki miłośierdzie jego.
Y wybáwił náſ od nieprzyjáćiół náƺych ; ábowiem ná wieki miłośierdzie jego.
Który dáje pokárm wƺelkiemu ćiáłu ; ábowiem ná wieki miłośierdzie jego.
Wyſłáwiájćież Bogá niebioſ ; ábowiem ná wieki miłośierdzie jego.
