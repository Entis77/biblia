Modlitwá Dawidowá. Nákłoń / Pánie! uchá twego / á wyſłucháj mię ; bomći nędzny y ubogi.
Strzeżże duƺy mojey / bom jeſt ten / którego ty miłujeƺ ; záchowáj ſługę twego / Boże mój! który ma nádźieję w tobie.
Zmiłuj śię nádemną / Pánie / ábowiem do ćiebie ná káżdy dźień wołám.
Rozweſel duƺę ſługi twego ; bo do ćiebie / o Pánie! duƺę ſwą podnoƺę.
Boś ty / Pánie! dobry y litośćiwy / y wielce miłośierny wƺyſtkim / którzy ćię wzywáją.
Wyſłuchájże / Pánie! modlitwę moję / á poſłucháj pilnie głoſu prośby mojey.
Wzywám ćię w dźień ućiſku mego ; bo mię ty wyſłucháƺ.
Nie maƺ żádnego podobnego tobie miedzy bogámi / o Pánie! y nie maƺ tákowych ſpráw / jáko ſą twoje.
Wƺyſtkie narody / któreś ty ſtworzył / przychodząc kłániáć śię będą przed oblicżem twojim / Pánie! y wielbić będą imię twoje.
Boś ty jeſt wielki / á cżyniƺ cudá ; tyś ſám jeſt Bogiem.
Náucż mię / Pánie / drogi twojey / ábym chodźił w práwdźie twojey / á uſtánów ſerce moje w bojáźni imienia twego ;
A będę ćię chwálił / Pánie / Boże mój! ze wƺyſtkiego ſercá mego / y będę wielbił imię twoje ná wieki /
Ponieważ miłośierdźie twoje wielkie jeſt nádemną / á tyś wyrwał duƺę moję z dołu głębokiego.
O Boże! powſtáli hárdźi przećiwko mnie / á rotá okrutników ƺukáłá duƺy mojey / ći / którzy ćię przed ocżymá nie máją.
Ale ty / Pánie / Boże miłośierny y litośćiwy / y nierychły ku gniewu / y wielce miłośierny / y práwdźiwy!
Wejrzyj ná mię / á zmiłuj śię nádemną / dájże moc twoję ſłudze twemu / á záchowáj ſyná ſłużebnicy twojey.
Okáż mi znák dobroći twojey / áby to widząc ći / którzy mię máją w nienáwiśći / záwſtydzeni byli / żeś mię ty / Pánie! porátował / y poćieƺyłeś mię.
