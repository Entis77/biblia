Modlitwá Dawidowá. Nákłoń / Pánie! uchá twego / á wyſłucháj mię ; bomći nędzny y ubogi.
Strzeżże duƺy mojej / bom jeſt ten / którego ty miłujeƺ ; záchowáj ſługę twego / Boże mój! który má nádzieję w tobie.
Zmiłuj śię nádemną / Pánie / ábowiem do ćiebie ná káżdy dzień wołám.
Rozweſel duƺę ſługi twego ; bo do ćiebie / o Pánie! duƺę ſwą podnoƺę.
Boś ty / Pánie! dobry y litośćiwy / y wielce miłośierny wƺyſtkim / którzy ćię wzywáją.
Wyſłuchájże / Pánie! modlitwę moję / á poſłucháj pilnie głoſu prośby mojej.
Wzywám ćię w dzień ućiſku mego ; bo mię ty wyſłucháƺ.
Nie máƺ żádnego podobnego tobie między bogámi / o Pánie! y nie máƺ tákowych ſpráw / jáko ſą twoje.
Wƺyſtkie národy / któreś ty ſtworzył / przychodząc kłániáć śię będą przed oblicżem twojim / Pánie! y wielbić będą imię twoje.
Boś ty jeſt wielki / á cżyniƺ cudá ; tyś ſám jeſt Bogiem.
Náucż mię / Pánie / drogi twojej / ábym chodził w práwdzie twojej / á uſtánów ſerce moje w bojáźni imieniá twego ;
A będę ćię chwálił / Pánie / Boże mój! ze wƺyſtkiego ſercá mego / y będę wielbił imię twoje ná wieki /
Poniewáż miłośierdzie twoje wielkie jeſt nádemną / á tyś wyrwáł duƺę moję z dołu głębokiego.
O Boże! powſtáli hárdzi przećiwko mnie / á rotá okrutników ƺukáłá duƺy mojej / ći / którzy ćię przed ocżymá nie máją.
Ale ty / Pánie / Boże miłośierny y litośćiwy / y nierychły ku gniewu / y wielce miłośierny / y práwdziwy!
Wejrzyj ná mię / á zmiłuj śię nádemną / dájże moc twoję ſłudze twemu / á záchowáj ſyná ſłużebnicy twojej.
Okáż mi znák dobroći twojej / áby to widząc ći / którzy mię máją w nienáwiśći / záwſtydzeni byli / żeś mię ty / Pánie! porátowáł / y poćieƺyłeś mię.
