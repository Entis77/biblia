Przedniejƺemu śpiewákowi pſálm Dawidowy.
Z żądośćią ocżekiwáłem Páná ; á ſkłonił śię ku mnie / y wyſłucháł wołánie moje ;
Y wyćiągnął miecż z dołu ƺumiącego y z błotá lgnącego / á poſtáwił ná ſkále nogi moje / y utwierdził kroki moje ;
A włożył w uſtá moje pieśń nową / chwáłę náleżącą Bogu náƺemu / co gdy wiele ich oglądá / ulękną śię / á będą mieć nádzieję w Pánu.
Błogoſłáwiony cżłowiek / który pokłádá w Pánu nádzieję ſwoję / á nie oglądá śię ná hárdych / áni ná tych / którzy śię unoƺą zá kłámſtwem.
Wieleś ucżynił / Pánie / Boże mój! cudów twoich / á myśli twoich o náſ nikt porządnie wylicżyć nie może przed tobą ; chćiáłlibym je wypowiedzieć y wymówić / dáleko ich więcej / niżby wypowiedziáne być mogły.
Ofiáry y obiáty nie chćiáłeś / áleś mi przekłół uƺy ; cáłopáleniá y ofiáry zá grzech nie żądáłeś.
Tedym rzekł : Oto idę ; w kśięgách nápiſáno o mnie ;
Abym cżynił wolę twoję / Boże mój! prágnę / ábowiem zákon twój jeſt w pośrodku wnętrznośći moich.
Opowiádáłem ſpráwiedliwość twoję w zgromádzeniu wielkiem ; oto wárg moich nie záwśćiągnąłem / ty wieƺ / Pánie!
Spráwiedliwośći twojej nie ukryłem w pośród ſercá mego / práwdę twoję y zbáwienie twoje opowiádáłem ; nie táiłem miłośierdziá twego y práwdy twojej w zgromádzeniu wielkiem.
Przetoż ty / Pánie! nie záwśćiągáj odemnie litośći twoich ; miłośierdzie twoje y práwdá twojá niech mię záwżdy ſtrzegą.
Abowiem ogárnęły mię nieƺcżęśćiá / którym niemáƺ licżby ; dośćignęły mię niepráwośći moje / ták / że przejrzeć nie mogę ; rozmnożyły śię nád włoſy głowy mojej / á ſerce moje opuśćiło mię.
Rácżże mię / Pánie! wyrwáć ; o Pánie! ná rátunek mój poſpieƺ.
Niech będą poháńbieni / ( á niech śię záwſtydzą wƺyſcy / ) którzy ƺukáją duƺy mojej / áby ją zátráćili ; niechájże śię ná wſtecż cofną / á niech śię záwſtydzą / którzy mi życżą złego.
Niech będą ſpuſtoƺeni zá to / ze mię ſháńbić uśiłują / mówiąc mi : Ehej! ehej!
Ale niech śię rozrádują y rozweſelą w tobie wƺyſcy / którzy ćię ƺukáją / y miłują zbáwienie twoje ; niech mówią záwżdy : Niecháj będzie Pán uwielbiony.
Jámći wpráwdzie ubogi y nędzny ; áleć Pán myśli o mnie. Tyś jeſt pomocnikiem moim y wybáwićielem moim ; Boże mój! nie omieƺkujże.
