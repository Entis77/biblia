Przedniejƺemu śpiewákowi pſálm Dawidowy.
Z żądośćią ocżekiwáłem Páná ; á ſkłonił śię ku mnie / y wyſłucháł wołánie moje ;
Y wyćiągnął miecż z dołu ƺumiącego y z błotá lgnącego / á poſtáwił ná ſkále nogi moje / y utwierdźił kroki moje ;
A włożył w uſtá moje pieśń nową / chwáłę náleżącą Bogu náƺemu / co gdy wiele ich oglądá / ulękną śię / á będą mieć nádźieję w Pánu.
Błogoſłáwiony cżłowiek / który pokłádá w Pánu nádźieję ſwoję / á nie oglądá śię ná hárdych / áni ná tych / którzy śię unoƺą zá kłámſtwem.
Wieleś ucżynił / Pánie / Boże mój! cudów twoich / á myśli twoich o nas nikt porządnie wylicżyć nie może przed tobą ; chćiáłlibym je wypowiedźieć y wymówić / dáleko ich więcey / niżby wypowiedźiáne być mogły.
Ofiáry y obiáty nie chćiáłeś / áleś mi przekłół uƺy ; cáłopáleniá y ofiáry zá grzech nie żądáłeś.
Tedym rzekł : Oto idę ; w kśięgách nápiſáno o mnie ;
Abym cżynił wolę twoję / Boże mój! prágnę / ábowiem zakon twój jeſt w pośrodku wnętrznośći moich.
Opowiádáłem ſpráwiedliwość twoję w zgromádzeniu wielkiem ; oto wárg moich nie záwśćiągnąłem / ty wieƺ / Pánie!
Spráwiedliwośći twojey nie ukryłem w pośród ſercá mego / práwdę twoję y zbáwienie twoje opowiádáłem ; nie táiłem miłośierdźiá twego y práwdy twojey w zgromádzeniu wielkiem.
Przetoż ty / Pánie! nie záwśćiągáj odemnie litośći twoich ; miłośierdźie twoje y práwdá twojá niech mię záwżdy ſtrzegą.
Abowiem ogárnęły mię nieƺcżęśćiá / którym niemáƺ licżby ; dośćignęły mię niepráwośći moje / ták / że przejrzeć nie mogę ; rozmnożyły śię nád włoſy głowy mojey / á ſerce moje opuśćiło mię.
Rácżże mię / Pánie! wyrwáć ; o Pánie! ná rátunek mój poſpieƺ.
Niech będą poháńbieni / ( á niech śię záwſtydzą wƺyſcy / ) którzy ƺukáją duƺy mojey / áby ją zátráćili ; niechájże śię ná wſtecż cofną / á niech śię záwſtydzą / którzy mi życżą złego.
Niech będą ſpuſtoƺeni zá to / ze mię ſháńbić uśiłują / mówiąc mi : Ehej! ehej!
Ale niech śię rozrádują y rozweſelą w tobie wƺyſcy / którzy ćię ƺukáją / y miłują zbáwienie twoje ; niech mówią záwżdy : Niecháj będźie Pán uwielbiony.
Jámći wpráwdźie ubogi y nędzny ; áleć Pán myśli o mnie. Tyś jeſt pomocnikiem moim y wybáwićielem moim ; Boże mój! nie omieƺkujże.
