Przedniejƺemu śpiewákowi ná Soſánnim pſálm świádectwá Aſáfowi.
O Páſterzu Izráelſki! poſłucháj / który prowádźiƺ Józefá jáko ſtádo owiec ; który śiedźiƺ ná Cherubinách / rozjáśnij śię.
Wzbudź moc ſwoję przed Efráimem / y Benjáminem / y Mánáſeſem / á przybądź ná wybáwienie náƺe.
O Boże! przywróć náſ / á rozjáśnij nád námi oblicże twoje / á będźiemy zbáwieni.
Pánie / Boże záſtępów! dokądże śię będźieƺ gniewáł ná modlitwę ludu ſwego?
Nákármiłeś ich chlebem płácżu / y nápoiłeś ich łzámi miárą wielką.
Wyſtáwiłeś náſ ná zwádę ſąśiádom náƺym ; á nieprzyjáćiołom náƺym / áby ſobie z náſ śmiech ſtroili.
O Boże záſtępów ; przywróć náſ / á rozjáśnij nád námi oblicże twoje / á będźiemy zbáwieni.
Tyś máćicę winną z Egiptu przenióſł ; wyrzućiłeś pogán / á wſádźiłeś ją.
Uprzątnąłeś dlá niej / y ſpráwiłeś / że śię rozkorzeniłá y nápełniłá źiemię.
Okryte ſą góry ćieniem jej / á gáłęźie jej jáko nájwyżƺe cedry.
Rozpuśćiłá látorośle ſwe áż do morzá / y áż do rzeki gáłązki ſwe.
Przecżżeś tedy rozwálił płot winnicy / ták / że ją ƺárpáją wƺyſcy / którzy mimo drogą idą?
Zniƺcżył ją wieprz dźiki / á zwierz polny ſpáſł ją.
O Boże záſtępów! náwróć śię proƺę / ſpojrzyj z niebá / y obácż / á náwiedź tę winną máćicę ;
Tę winnicę / którą ƺcżepiłá práwicá twojá / y látoroſłki / któreś ſobie zmocnił.
Spáloná jeſt ogniem / y wyrąbáná ; ginie od zápálcżywośći oblicżá twego.
Niech będźie ręká twojá nád mężem práwicy twojej / nád ſynem cżłowiecżym / któregoś ſobie zmocnił.
A nie odſtąpimy od ćiebie ; záchowáj náſ przy żywoćie / á imieniá twego wzywáć będźiemy.
O Pánie / Boże záſtępów! náwróćże náſ záśię ; rozjáśnij nád námi oblicże twoje / á będźiemy zbáwieni.
