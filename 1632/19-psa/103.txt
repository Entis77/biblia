Pſálm Dawidowy. Błogoſłáw duƺo mojá Pánu / y wƺyſtkie wnętrznośći moje imieniowi jego świętemu.
Błogoſłáwże duƺo mojá Pánu / á nie zápomináj wƺyſtkich dobrodziejſtw jego.
Który odpuƺcżá wƺyſtkie niepráwośći twoje ; który uzdráwiá wƺyſtkie choroby twoje ;
Który wybáwiá od śmierći żywot twój ; który ćię koronuje miłośierdziem y wielką litośćią :
Który náſycá dobrem uſtá twoje / á odnáwiá jáko orłá młodość twoję.
Pán cżyni / co ſpráwiedliwego jeſt / y ſądy wƺyſtkim ućiśnionym.
Oznájmił drogi ſwe Mojżeƺowi / á ſynom Izráelſkim ſpráwy ſwoje.
Miłośierny y litośćiwy jeſt Pán / nierychły do gniewu / y wielkiego miłośierdziá.
Nie będzie śię ná wieki wádził / á gniewu wiecżnie chowáł.
Nie według grzechów náƺych obchodzi śię z námi / áni według niepráwośći náƺych odpłácá nám.
Abowiem jáko ſą niebioſá wyſokie nád ziemią / ták jeſt utwierdzone miłośierdzie jego nád tymi / którzy śię go boją ;
A jáko dáleko jeſt wſchód od záchodu / ták dáleko oddálił od náſ przeſtępſtwá náƺe.
Jáko má litość ojćiec nád dziátkámi / ták má litość Pán nád tymi / którzy śię go boją.
Onći záiſte zná / cośmy zá ulepienie ; pámiętá / żeśmy prochem.
Dni cżłowiecże ſą jáko tráwá / á jáko kwiát polny / ták kwitnie ;
Gdy náń wiátr powienie / áliśći go niemáƺ / áni go więcej pozná miejſce jego.
Ale miłośierdzie Páńſkie od wieków áż ná wieki nád tymi / którzy śię go boją / á ſpráwiedliwość jego nád ſynámi ſynów /
Którzy ſtrzegą przymierzá jego / y pámiętáją ná przykázánie jego / áby je cżynili.
Pán ná niebioſách utwierdził ſtolicę ; á króleſtwo jego nád wƺyſtkimi pánuje.
Błogoſłáwćież Pánu Aniołowie jego mocni w śile / którzy cżynićie rozkázániá jego / poſłuƺnymi będąc głoſowi ſłowá jego.
Błogoſłáwćie Pánu wƺyſtkie wojſká jego / ſłudzy jego / którzy cżynićie wolę jego.
Błogoſłáwćie Pánu wƺyſtkie ſpráwy jego / ná wƺyſtkich miejſcách pánowániá jego. Błogoſłáw / duƺo mojá! Pánu.
