Pſálm Dawidowy. Błogoſłáw duƺo mojá Pánu / y wƺyſtkie wnętrznośći moje imieniowi jego świętemu.
Błogoſłáwże duƺo mojá Pánu / á nie zápomináj wƺyſtkich dobrodźiejſtw jego.
Który odpuƺcżá wƺyſtkie niepráwośći twoje ; który uzdráwiá wƺyſtkie choroby twoje ;
Który wybáwiá od śmierći żywot twój ; który ćię koronuje miłośierdźiem y wielką litośćią :
Który náſycá dobrem uſtá twoje / á odnáwiá jáko orłá młodość twoję.
Pán cżyni / co ſpráwiedliwego jeſt / y ſądy wƺyſtkim ućiśnionym.
Oznájmił drogi ſwe Mojżeƺowi / á ſynom Izráelſkim ſpráwy ſwoje.
Miłośierny y litośćiwy jeſt Pán / nierychły do gniewu / y wielkiego miłośierdźiá.
Nie będźie śię ná wieki wádźił / á gniewu wiecżnie chował.
Nie według grzechów náƺych obchodźi śię z námi / áni według niepráwośći náƺych odpłácá nam.
Abowiem jáko ſą niebioſá wyſokie nád źiemią / ták jeſt utwierdzone miłośierdźie jego nád tymi / którzy śię go boją ;
A jáko dáleko jeſt wſchód od záchodu / ták dáleko oddálił od nas przeſtępſtwá náƺe.
Jáko ma litość oćiec nád dźiátkámi / ták ma litość Pán nád tymi / którzy śię go boją.
Onći záiſte zná / cośmy zá ulepienie ; pámiętá / żeśmy prochem.
Dni cżłowiecże ſą jáko tráwá / á jáko kwiát polny / ták kwitnie ;
Gdy nań wiátr powienie / áliśći go niemáƺ / áni go więcey pozná miejſce jego.
Ale miłośierdźie Páńſkie od wieków áż ná wieki nád tymi / którzy śię go boją / á ſpráwiedliwość jego nád ſynámi ſynów /
Którzy ſtrzegą przymierza jego / y pámiętáją ná przykazánie jego / áby je cżynili.
Pán ná niebioſách utwierdźił ſtolicę ; á króleſtwo jego nád wƺyſtkimi pánuje.
Błogoſłáwćież Pánu Aniołowie jego mocni w śile / którzy cżynićie rozkázánia jego / poſłuƺnymi będąc głoſowi ſłowá jego.
Błogoſłáwćie Pánu wƺyſtkie wojſká jego / ſłudzy jego / którzy cżynićie wolę jego.
Błogoſłáwćie Pánu wƺyſtkie ſpráwy jego / ná wƺyſtkich miejſcách pánowánia jego. Błogoſłáw / duƺo mojá! Pánu.
