Pieśń Dawidowá. Błogoſłáwiony Pán / ſkáłá mojá / który ćwicży ręce moje do bitwy / á pálce moje do wojny.
Miłośierdziem mojem / y twierdzą moją / ućiecżką moją / wybáwićielem moim / y tárcżą moją on mi jeſt / przetoż w nim ufám ; onći podbijá pod mię lud mój.
Pánie! cóż jeſt cżłowiek / że náń máƺ bácżenie? á ſyn cżłowiecży / że go ſobie powáżáƺ?
Cżłowiek márnośći jeſt podobny ; dni jego jáko ćień pomijájący.
Pánie! nákłoń niebioſ twoich / á zſtąp ; dotknij śię gór / á zákurzą śię.
Zábłyśnij błyſkáwicą / á rozproƺ ich ; puść ſtrzáły twoje / á poráź ich.
Śćiągnij rękę ſwą z wyſokośći ; wybáw mię / á wyrwij mię z wód wielkich / z ręki cudzoziemców.
Których uſtá kłámſtwo mówią / á práwicá ich / práwicá omylná.
Boże! pieśń nową tobie záśpiewám ; ná lutni / y ná inſtrumenćie o dzieśięćiu ſtronách śpiewáć ći będę.
Bóg dáje zwyćięſtwo królom / á Dawidá / ſługę ſwego / wybáwiá od miecżá ſrogiego.
Wybáwże mię / á wyrwij mię z ręki cudzoziemców / których uſtá mówią kłámſtwo / á práwicá ich práwicá omylná ;
Aby ſynowie náśi byli jáko ƺcżepy roſnące w młodośći ſwojej / á córki náƺe / jáko kámienie węgielne / wyćioſáne w budynku kośćielnym.
Szpiżárnie náƺe pełne niech wydáwáją wƺelákie potrzeby ; trzody náƺe niech rodzą tyśiące / niech rodzą dzieśięć tyśięcy w oborách náƺych.
Woły náƺe niech będą tłuſte ; niech nie będzie wtárgnieniá / áni zájęćiá / áni nárzekániá po ulicách náƺych.
Błogoſłáwiony lud / któremu śię ták dzieje. Błogoſłáwiony lud / którego Bogiem jeſt Pán.
