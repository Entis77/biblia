Pſálm Dawidowy. Pánie! wyſłucháj modlitwę moję / á przyjmij w uƺy prośby moje ; dlá práwdy twojej wyſłucháj mię y dlá ſpráwiedliwośći twojej.
A nie wchodź w ſąd z ſługą twoim ; ábowiem nie będzie uſpráwiedliwiony przed oblicżem twojim żáden żyjący.
Gdyż prześláduje nieprzyjáćiel duƺę moję / potárł równo z ziemią żywot mój ; ſpráwił to / że muƺę mieƺkáć w ćiemnośćiách / jáko ći / którzy zdáwná pomárli.
Y śćiśniony jeſt we mnie duch mój / á we wnętrznośćiách moich niƺcżeje ſerce moje.
Wſpominám ſobie dni dáwne / y rozmyślám o wƺyſtkich ſpráwách twoich / y ucżynki rąk twoich rozbierám.
Wyćiągám ręce moje ku tobie ; duƺá mojá / jáko ſuchá ziemiá / ćiebie prágnie. Selá.
Pośpieƺ śię / á wyſłucháj mię / Pánie! uſtáje duch mój ; nie ukrywájże oblicżá twego przedemną ; bomći podobny zſtępującym do grobu.
Spráw / ábym ráno ſłyƺáł miłośierdzie twoje / bo w tobie ufám ; oznájmij mi drogę / którąbym miáł chodzić ; bo do ćiebie podnoƺę duƺę moję.
Wyrwij mię od nieprzyjáćiół moich / Pánie! do ćiebie śię ućiekám.
Náucż mię cżynić wolę twoję / ábowiemeś ty Bóg mój ; duch twój dobry niech mię prowádzi po ziemi práwej.
Dlá imieniá twego / Pánie! ożyw mię ; dlá ſpráwiedliwośći twojej wywiedź z utrápieniá duƺę moję.
Y dlá miłośierdziá twego wytráć nieprzyjáćiół moich / á wygłádź wƺyſtkich przećiwników duƺy mojej ; bom já ſługá twój.
