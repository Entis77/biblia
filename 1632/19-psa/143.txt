Pſálm Dawidowy. Pánie! wyſłucháj modlitwę moję / á przyjmij w uƺy prośby moje ; dla práwdy twojey wyſłucháj mię y dla ſpráwiedliwośći twojey.
A nie wchodź w ſąd z ſługą twoim ; ábowiem nie będźie uſpráwiedliwiony przed oblicżem twojim żáden żyjący.
Gdyż prześláduje nieprzyjáćiel duƺę moję / potárł równo z źiemią żywot mój ; ſpráwił to / że muƺę mieƺkáć w ćiemnośćiách / jáko ći / którzy zdáwná pomárli.
Y śćiśniony jeſt we mnie duch mój / á we wnętrznośćiách moich niƺcżeje ſerce moje.
Wſpominám ſobie dni dáwne / y rozmyślám o wƺyſtkich ſpráwách twoich / y ucżynki rąk twoich rozbierám.
Wyćiągám ręce moje ku tobie ; duƺá mojá / jáko ſuchá źiemiá / ćiebie prágnie. Selá.
Pośpieƺ śię / á wyſłucháj mię / Pánie! uſtáje duch mój ; nie ukrywájże oblicżá twego przedemną ; bomći podobny zſtępującym do grobu.
Spráw / ábym ráno ſłyƺáł miłośierdźie twoje / bo w tobie ufám ; oznájmij mi drogę / którąbym miał chodźić ; bo do ćiebie podnoƺę duƺę moję.
Wyrwij mię od nieprzyjáćiół moich / Pánie! do ćiebie śię ućiekám.
Náucż mię cżynić wolę twoję / ábowiemeś ty Bóg mój ; duch twój dobry niech mię prowádźi po źiemi práwej.
Dla imieniá twego / Pánie! ożyw mię ; dla ſpráwiedliwośći twojey wywiedź z utrápieniá duƺę moję.
Y dla miłośierdźiá twego wytráć nieprzyjáćiół moich / á wygłádź wƺyſtkich przećiwników duƺy mojey ; bom ja ſługá twój.
