Przedniejƺemu śpiewákowi dla Jedytuná pſálm Aſáfowy.
Głos mój podnoƺę do Bogá / kiedy wołám ; głos mój podnoƺę do Bogá / áby mię wyſłuchał.
W dźień utrapienia mego ƺukáłem Páná : wyćiągáłem w nocy ręce moje bez przeſtánia / á nie dáłá śię ućieƺyć duƺá mojá.
Wſpomináłem ná Bogá / á trwożyłem ſobą ; rozmyśláłem / á utrapieniem śćiśniony był duch mój. Selá.
Zátrzymywáłeś ocży moje / áby cżuły ; potártym był / áżem nie mógł mówić.
Przychodźiły mi ná pámięć dni przeƺłe y látá dáwne.
Wſpomináłem ſobie ná śpiewánie moje ; w nocym w ſercu ſwem rozmyślał / y wywiádywał śię o tym duch mój / mówiąc :
Izali mię ná wieki odrzući Pán / á więcey mi już łáſki nie ukáże?
Izali do końcá uſtánie miłośierdźie jego / y koniec weźmie ſłowo od rodzáju áż do rodzáju? Izali zápomniał Bóg zmiłowáć śię?
Izali zátrzymał w gniewie litośći ſwoje? Selá.
Y rzekłem : Toć jeſt śmierć mojá ; wƺákże práwicá Nájwyżƺego ucżyni odmiánę.
Wſpomináć ſobie będę ná ſpráwy Páńſkie / á przypomináć ſobie będę dźiwne ſpráwy twoje / zdáwná ucżynione.
Y będę rozmyślał o wƺelkiem dźiele twojim / y o ucżynkách twojich będę mówił :
Boże! świętá jeſt drogá twojá. Któryż Bóg jeſt ták wielki / jáko Bóg náƺ?
Tyś jeſt Bóg / który cżyniƺ cudá ; podáłeś do znájomośći miedzy narody moc twoję.
Odkupiłeś rámieniem twojim lud ſwój / ſyny Jákóbowe y Józefowe. Selá.
Widźiáły ćię wody / o Boże! widźiáły ćię wody / y ulękły śię / y wzruƺyły śię przepáśći.
Obłoki wydáły powodźi ; niebioſá wydáły gromy / á ſtrzáły twoje tám y ſám biegáły.
Hucżáło grzmienie twoje po obłokách / błyſkáwice oświećiły okrąg źiemi / źiemiá śię wzruƺyłá y zátrzęſłá.
Przez morze byłá drogá twojá / á śćieƺki twoje przez wody wielkie / wƺákże śládów twojich nie było.
Prowádźiłeś lud twój / jáko ſtádo owiec / przez rękę Mojżeƺá y Aároná.
