Wyſłáwiájćie Páná ; ogłáƺájćie imię jego ; opowiádájćie między národámi ſpráwy jego.
Śpiewájćie mu / śpiewájćie mu pſálmy / rozmáwiájćie o wƺyſtkich cudách jego.
Chlubćie śię imieniem świętym jego ; niech śię weſeli ſerce ƺukájących Páná.
Szukájćież Páná y mocy jego ; ƺukájćie oblicżá jego záwƺe.
Przypominájćie ſobie dźiwy jego / które cżynił / cudá jego y ſądy uſt jego.
Wy náśienie Abráhámá / ſługi jego! Wy ſynowie Jákóbowi / wybráni jego!
Onći jeſt Pán / Bóg náƺ / po wƺyſtkiej źiemi ſądy jego.
Pámiętá wiecżnie ná przymierze ſwoje : ná ſłowo / które przykázáł áż do tyśiącżnego pokoleniá ;
Które poſtánowił z Abráhámem / y ná przyśięgę ſwą ucżynioną Izáákowi.
Bo je poſtánowił Jákóbowi zá uſtáwę / á Izráelowi zá umowę wiecżną.
Mówiąc : Tobie dám źiemię Chánánejſką zá ƺnur dźiedźictwá wáƺego ;
Kiedy ich był máły pocżet / práwie máły pocżet / á jeƺcże w niej byli przychodniámi.
Przechodźili záiſte od národu do národu / á z króleſtwá innego ludu ;
Nie dopuƺcżáł nikomu / áby im miał krzywdę cżynić ; náwet káráł dla nich y królów / mówiąc :
Nie tykájćie pomázáńców moich / á prorokom moim nie cżyńćie nic złego.
Gdy przywołáwƺy głód ná źiemię / wƺyſtkę podporę chlebá pokruƺył.
Poſłáł przed nimi mężá / który był zá niewolniká ſprzedány / to jeſt Józefá ;
Którego nogi pętámi trápili / á żelázo śćiſnęło ćiáło jego /
Aż do onego cżáſu / gdy śię o nim wzmiánká ſtáłá ; mowá Páńſká doświádcżáłá go.
Poſłáwƺy król kázáł go puśćić ; ten / który pánowáł nád národámi / wolnym go ucżynił.
Poſtánowił go pánem domu ſwego / y kśiążęćiem nád wƺyſtką dźierżáwą ſwoją /
Aby włádáł y kśiążętámi jego według zdániá duƺy ſwojey / y ſtárców jego mądrośći náucżáł.
Potem wƺedł Izráel do Egiptu / á Jákób był gośćiem w źiemi Chámowej ;
Gdźie rozmnożył Bóg lud ſwój bárdzo / y ucżynił go możniejƺym nád nieprzyjáćiół jego.
Odmienił ſerce ich / iż mieli w nienáwiśći lud jego / á zmyśláli zdrády przećiw ſługom jego.
Poſłáł Mojżeƺá / ſługę ſwego y Aároná / którego obráł ;
Którzy im przedłożyli ſłowá znáków jego / y cudá w źiemi Chámowej.
Poſłáł ćiemnośći / y záćmiło śię / á nie byli odpornymi ſłowu jego.
Obróćił wody ich w krew / á pomorzył ryby w nich.
Wydáłá źiemiá ich mnóſtwo żáb / y były w páłácách królów ich.
Rzekł / á przyƺłá rozmáitá muchá / y mƺyce we wƺyſtkich gránicách ich.
Dáł grád miáſto deƺcżu / ogień pálący ná źiemię ich.
Tákże potłukł winnice ich / y figi ich / á pokruƺył drzewá w gránicách ich.
Rzekł / á przyƺłá ƺáráńcżá / y chrząƺcżów niezlicżone mnóſtwo ;
Y pożárły wƺelkie źiele w źiemi ich / á pojádły urodzáje źiemi ich.
Náwet pobił wƺyſtko pierworodztwo w źiemi ich / pocżątek wƺyſtkiej śiły ich.
Tedy ich wywiódł ze ſrebrem y ze złotem / á nie był nikt ſłáby między pokoleniem ich.
Rádowáł śię Egipt / gdy oni wychodźili ; ábowiem był przypádł ná nich ſtrách ich.
Rozpoſtárł obłok ná okryćie ich / á ogień ná oświecánie nocy.
Ná żądánie ich przywiódł przepiórki / á chlebem niebieſkim náſyćił ich.
Otworzył ſkáłę y wypłynęły wody / á płynęły po ſuchych miejſcách jáko rzeká.
Abowiem wſpomniáł ná ſłowo świętobliwośći ſwojey / które rzekł do Abráhámá / ſługi ſwego.
Przetoż wywiódł lud ſwój z weſelem / á z śpiewániem wybránych ſwoich.
Y podáł im źiemię pogán / á pośiedli práce národów.
Aby záchowáli uſtáwy jego / á práwá jego przeſtrzegáli. Hállelujá.
