Przedniejƺemu śpiewákowi z ſynów Korego / ná Alámot pieśń.
Bóg jeſt ućiecżką y śiłą náƺą / rátunkiem we wƺelkim ućiſku nájpewniejƺym.
Przetoż śię báć nie będźiemy / choćby śię poruƺyłá źiemiá / choćby śię przenioſły góry w pośród morzá ;
Choćby záƺumiáły / á wzburzyły śię wody jego / y zátrzęſły śię góry od náwáłnośći jego. Selá.
Strumienie rzeki jego rozweſeláją miáſto Boże / nájświętƺe z przybytków nájwyżƺego.
Bóg jeſt w pośrzodku jego / nie będźie poruƺone ; porátuje go Bóg záráz z poránku.
Gdy śię wzburzyły narody / á zátrząſnęły śię króleſtwá / Pán wydał głos ſwój / y rozpłynęłá śię źiemiá.
Pán zaſtępów jeſt z námi ; twierdzą wyſoką jeſt nam Bóg Jákóbowy. Selá.
Pójdźćie / oglądájćie ſpráwy Páńſkie / jákie ucżynił ſpuſtoƺenie ná źiemi ;
Który uśmierzá wojny áż do końcżyn źiemi / łuk kruƺy / y oręże łámie / á wozy ogniem páli /
Mówiąc : Uſpokójćie śię / á wiedzćie / żem Ja Bóg ; będę wywyżƺony miedzy národámi / będę wywyżƺony ná źiemi.
Pán zaſtępów z námi ; twierdzą wyſoką jeſt nam Bóg Jákóbowy. Selá.
