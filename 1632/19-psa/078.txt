Pieśń wyucżájącá / podáná Aſáfowi. Słucháj / ludu mój! zákonu mego ; nákłońćież uƺów ſwych do ſłów uſt moich.
Otworzę w podobieńſtwie uſtá moje / á będę opowiádáł przypowiáſtki ſtárodáwne.
Cośmy ſłyƺeli / y poználi / y co nám ojcowie náśi opowiádáli.
Nie zátáimy tego przed ſynámi ich / którzy przyƺłym potomkom ſwoim opowiádáć będą chwáły Páńſkie / y moc jego / y cudá jego / które ucżynił.
Bo wzbudźił świádectwo w Jákóbie / á zákon wydáł w Izráelu ; przykázáł ojcom náƺym / áby to do wiádomośći podáwáli ſynom ſwoim /
Aby poznáł wiek potomny / ſynowie / którzy śię národźić mieli / á oni záś powſtáwƺy / áby to opowiádáli ſynom ſwoim ;
Aby pokłádáli w Bogu nádźieję ſwoję / á nie zápomináli ná ſpráwy Boże / ále ſtrzegli przykázáń jego ;
Aby śię nie ſtáli jáko ojcowie ich národem odpornym y niepoſłuƺnym / národem / który nie wygotowáł ſercá ſwego / áby był wierny Bogu duch jego.
Albo jáko ſynowie Efráimowi zbrojni / którzy / choć umieli z łuku ſtrzeláć / wƺákże w dźień wojny tył podáli.
Bo nie przeſtrzegáli przymierzá Bożego / á według zákonu jego zbrániáli śię chodźić.
Zápomnieli ná ſpráwy jego / y ná dźiwne dźiełá jego / które im pokázywáł.
Przed ojcámi ich cżynił cudá w źiemi Egipſkiej / ná polu Soán.
Rozdźielił morze / y przeprowádźił ich / y ſpráwił / że ſtánęły wody jáko kupá.
Prowádźił ich w obłoku we dnie / á káżdej nocy w jáſnym ogniu.
Rozƺcżepił ſkáły ná puƺcży / á nápoił ich / jáko z przepáśći wielkich.
Wywiódł ſtrumienie ze ſkáły / á ucżynił / że wody ćiekły jáko rzeki.
A wƺákże oni przycżynili grzechów przećiwko niemu / á wzruƺyli Nájwyżƺego ná puƺcży do gniewu ;
Y kuśili Bogá w ſercu ſwem / żądájąc pokármu według lubośći ſwojej.
A mówili przećiwko Bogu temi ſłowy : Izáli może Bóg zgotowáć ſtół ná tej puƺcży?
Oto uderzył w ſkáłę / á wypłynęły wody / y rzeki wezbráły ; izáli też będźie mógł dáć chleb? Izáli nágotuje mięſá ludowi ſwemu?
Przetoż uſłyƺáwƺy to Pán / rozgniewáł śię / á ogień śię zápálił przećiw Jákóbowi / tákże y popędliwość powſtáłá przećiw Izráelowi ;
Przeto / iż nie wierzyli Bogu / á nie mieli nádźiei w zbáwieniu jego.
Choć był rozkázáł obłokom z góry / y forty niebieſkie otworzył.
Y ſpuśćił im jáko deƺcż mánnę ku pokármowi / á pƺenicę niebieſką dáł im.
Chleb mocárzów jádł cżłowiek / á zeſłáł im pokármów do ſytośći.
Obróćił wiátr ze wſchodu ná powietrzu / á przywiódł mocą ſwą wiátr z południá ;
Y ſpuśćił ná nich mięſo jáko proch / y ptáſtwo ſkrzydláſte jáko piáſek morſki ;
Spuśćił je w pośród obozu ich / wƺędy około námiotów ich.
Y jedli / á náſyceni byli hojnie / y dáł im / cżego żądáli.
A gdy jeƺcże nie wypełnili żądośći ſwej / gdy jeƺcże pokárm był w uſtách ich :
Tedy zápálcżywość Bożá przypádłá ná nich / y pobił tłuſtych ich / á przedniejƺych z Izráelá poráźił.
Ale w tem wƺyſtkiem jeƺcże grzeƺyli / y nie wierzyli cudom jego ;
Przetoż ſpráwił / że márnie dokonáli dni ſwoich / y lát ſwoich w ſtráchu.
Gdy ich tráćił / jeźliże go ƺukáli / y náwrácáli śię / á ƺukáli z ráná Bogá /
Przypominájąc ſobie / iż Bóg był ſkáłą ich / á Bóg nájwyżƺy odkupićielem ich :
( Acżkolwiek pochlebiáli mu uſty ſwemi / y językiem ſwoim kłámáli mu ;
A ſerce ich nie było ƺcżere przed nim / áni wiernymi byli w przymierzu jego. )
On jednák będąc miłośierny odpuƺcżáł niepráwośći ich / á nie zátrácáł ich / ále cżęſtokroć odwrácáł gniew ſwój / á nie pobudzáł wƺyſtkiego gniewu ſwego ;
Bo pámiętáł / że ſą ćiáłem / wiátrem / który odchodźi / á nie wrácá śię záś.
Jáko go cżęſto dráźnili ná puƺcży / y do boleśći przywodźili ná puſtyniách?
Bo coráz kuśili Bogá / á Świętemu Izráelſkiemu gránice zámierzáli.
Nie pámiętáli ná rękę jego / y ná on dźień / w który ich wybáwił z utrápieniá ;
Gdy cżynił w Egipćie znáki ſwoje / á cudá ſwe ná polu Soán ;
Gdy obróćił w krew rzeki ich / y ſtrumienie ich / ták / że z nich pić nie mogli.
Przepuśćił ná nich rozmáite muchy / áby ich kąſáły / y żáby / áby ich gubiły :
Y dáł chrząƺcżom urodzáje ich / á práce ich ƺáráńcży.
Potłukł grádem ƺcżepy ich / á drzewá leśnych fig ich grádem lodowym.
Y podáł grádowi bydło ich / á májętność ich węglu ogniſtemu.
Poſłáł ná nich gniew zápálcżywośći ſwojej / popędliwość / y rozgniewánie / y ućiśnienie / przypuśćiwƺy ná nich ániołów złych.
Wyproſtowáł śćieżkę gniewowi ſwemu / nie záchowáł od śmierći duƺy ich / y ná bydło ich powietrze dopuśćił ;
Y pobił wƺyſtko pierworodztwo w Egipćie / pierwiáſtki mocy ich w przybytkách Chámowych ;
Ale jáko owce wyprowádźił lud ſwój / á wodźił ich jáko ſtádá po puƺcży.
Wodźił ich w bezpiecżeńſtwie / ták / że śię nie lękáli / ( á nieprzyjáćiół ich okryło morze / )
Aż ich przywiódł do świętej gránicy ſwojej / ná onę górę / której nábyłá práwicá jego.
Y wyrzućił przed twárzą ich národy / y ſpráwił / że im przyƺły ná ƺnur dźiedźictwá ich / áżeby mieƺkáły w przybytkách ich pokoleniá Izráelſkie.
A wƺákże przećież kuśili y dráźnili Bogá nájwyżƺego / á świádectwá jego nie ſtrzegli.
Ale śię odwróćili / y przewrotnie śię obchodźili / jáko y ojcowie ich ; wywróćili śię jáko łuk omylny.
Bo go wzruƺyli do gniewu wyżynámi ſwemi / á rytemi báłwánámi ſwemi pobudźili go do zápálcżywośći.
Co ſłyƺąc Bóg rozgniewáł śię / y zbrzydźił ſobie bárdzo Izráelá /
Ták / że opuśćiwƺy przybytek w Sylo / námiot / który poſtáwił między ludźmi /
Podáł w niewolę moc ſwoję / y ſłáwę ſwoję w ręce nieprzyjáćielſkie.
Dáł pod miecż lud ſwój / á ná dźiedźictwo ſwoje rozgniewáł śię.
Młodźieńców jego ogień pożárł / á pánienki jego nie były ucżcżone.
Kápłáni jego od miecżá polegli / á wdowy jego nie płákáły.
Lecż potem ocućił śię Pán jáko ze ſnu / jáko mocárz wykrzykájący od winá.
Y záráźił nieprzyjáćiół ſwoich ná pośládkách / á ná wiecżną háńbę podáł ich.
Ale choć wzgárdźił námiotem Józefowym / á pokoleniá Efráimowego nie obráł /
Wƺákże obráł pokolenie Judowe / y górę Syon / którą umiłowáł.
Y wyſtáwił ſobie jáko páłác wyſoki świątnicę ſwoję / jáko źiemię / którą ugruntowáł ná wieki.
Y obráł Dawidá ſługę ſwego / wźiąwƺy go z obór owcżych ;
Gdy chodźił zá owcámi kotnemi / przyprowádźił go / áby páſł Jákóbá / lud jego / y Izráelá / dźiedźictwo jego ;
Który ich páſł w ƺcżerośći ſercá ſwego / á w roztropnośći rąk ſwoich prowádźił ich.
