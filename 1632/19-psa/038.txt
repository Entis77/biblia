Pſálm Dawidowy ku przypominániu.
Pánie! w popędliwośći twojey nie náćieráj ná mię / á w gniewie twoim nie kárz mię.
Abowiem ſtrzáły twoje utknęły we mnie / á ręká twojá dolegá mię.
Niemáƺ nic cáłego w ćiele mojim dla rozgniewánia twego ; niemáƺ odpocżynku kośćiom moim dla grzechu mojego.
Bo niepráwośći moje przyćiſnęły głowę moję ; jáko brzemię ćiężkie obćiążyły mię.
Zjątrzyły śię / y pogniły rány moje / dla głupſtwá mojego.
Skurcżyłem śię / y ſkrzywiłem śię bárzo / ná káżdy dźień w żáłobie chodzę.
Abowiem wnętrznośći moje pełne ſą brzydkośći / á nie maƺ nic cáłego w ćiele mojim.
Zemdláłem / y ſtártym jeſt bárzo / rycżę dla trwogi ſercá mego.
Pánie! przed tobą jeſt wƺyſtká żądość mojá / á wzdychánie moje przed tobą nie jeſt ſkryte.
Serce moje ſkácże ; opuśćiłá mię śiłá mojá / á jáſnośći ocżów mojich nie maƺ przy mnie.
Którzy mię miłują / y przyjáćiele moi / ſtronią od rán mojich / á powinowáći moi z dáleká ſtoją.
Y záſtáwili śidłá ći / którzy ƺukáją duƺy mojey ; á którzy mi ƺukáją złego / mówili przewrotnie / y zdrády przez cáły dźień zmyśláli.
Alem ja niby głuchy nie ſłyƺał / á jáko niemy / który uſt ſwojich nie otwierá.
Y ſtałem śię jáko cżłowiek / który nic nie ſłyƺy / y niemá odporu w uſtách ſwojich.
Abowiem ná ćię / Pánie! ocżekuję ; ty zá mię odpowieƺ / Pánie / Boże mój!
Bom rzekł : Niecháj śię nie ćieƺą ze mnie ; gdyby ƺwánkowáłá nogá mojá / niecháj śię hárdźie nie podnoƺą przećiwko mnie.
Bom ja upádku bliſki / á boleść mojá záwżdy jeſt przedemną.
Owƺem / niepráwość moję wyznáję / á fráſuję śię dla grzechu mojego.
Ale nieprzyjáćiele moi weſelą śię / zmácniáją śię / y rozmnáżáją śię ći / którzy mię nienáwidzą bez przycżyny :
A oddáwájąc mi złem zá dobre ſprzećiwiáją mi śię / przeto / że náśláduję tego / co jeſt dobrego.
Nie opuƺcżájże mię / Pánie / Boże mój! nie oddálájże śię odemnie.
Poſpieƺ ná rátunek mój / Pánie zbáwieniá mego!
