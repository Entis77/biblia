Pſálm Dawidowy ku przypominániu.
Pánie! w popędliwośći twojej nie náćieráj ná mię / á w gniewie twoim nie kárz mię.
Abowiem ſtrzáły twoje utknęły we mnie / á ręká twojá dolegá mię.
Niemáƺ nic cáłego w ćiele mojem dlá rozgniewániá twego ; niemáƺ odpocżynku kośćiom moim dlá grzechu mojego.
Bo niepráwośći moje przyćiſnęły głowę moję ; jáko brzemię ćiężkie obćiążyły mię.
Zjątrzyły śię / y pogniły rány moje / dlá głupſtwá mojego.
Skurcżyłem śię / y ſkrzywiłem śię bárdzo / ná káżdy dźień w żáłobie chodzę.
Abowiem wnętrznośći moje pełne ſą brzydkośći / á nie máƺ nic cáłego w ćiele mojem.
Zemdláłem / y ſtártym jeſt bárdzo / rycżę dlá trwogi ſercá mego.
Pánie! przed tobą jeſt wƺyſtká żądość mojá / á wzdychánie moje przed tobą nie jeſt ſkryte.
Serce moje ſkácże ; opuśćiłá mię śiłá mojá / á jáſnośći ocżów moich nie máƺ przy mnie.
Którzy mię miłują / y przyjáćiele moi / ſtronią od rán moich / á powinowáći moi z dáleká ſtoją.
Y záſtáwili śidłá ći / którzy ƺukáją duƺy mojej ; á którzy mi ƺukáją złego / mówili przewrotnie / y zdrády przez cáły dźień zmyśláli.
Alem já niby głuchy nie ſłyƺáł / á jáko niemy / który uſt ſwoich nie otwierá.
Y ſtáłem śię jáko cżłowiek / który nic nie ſłyƺy / y niemá odporu w uſtách ſwoich.
Abowiem ná ćię / Pánie! ocżekuję ; ty zá mię odpowieƺ / Pánie / Boże mój!
Bom rzekł : Niecháj śię nie ćieƺą ze mnie ; gdyby ƺwánkowáłá nogá mojá / niecháj śię hárdźie nie podnoƺą przećiwko mnie.
Bom já upádku bliſki / á boleść mojá záwżdy jeſt przedemną.
Owƺem / niepráwość moję wyznáję / á fráſuję śię dlá grzechu mojego.
Ale nieprzyjáćiele moi weſelą śię / zmácniáją śię / y rozmnáżáją śię ći / którzy mię nienáwidzą bez przycżyny :
A oddáwájąc mi złem zá dobre ſprzećiwiáją mi śię / przeto / że náśláduję tego / co jeſt dobrego.
Nie opuƺcżájże mię / Pánie / Boże mój! nie oddálájże śię odemnie.
Pośpieƺ ná rátunek mój / Pánie zbáwieniá mego!
