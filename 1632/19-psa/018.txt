Przedniejƺemu śpiewákowi pieśń Dawidá / ſługi Páńſkiego / który mówił do Páná ſłowá tey pieśni onego dniá / gdy go Pán wyrwał z rąk wƺyſtkich nieprzyjáćiół jego / y z ręki Saulowej ; y rzekł :
Rozmiłuję śię ćiebie / Pánie / mocy mojá!
Pán opoką moją / twierdzą moją / y wybáwićielem moim ; Bóg mój ſkáłá mojá / w nim będę ufał ; tárcżá mojá / y róg zbáwieniá mego / ućiecżká mojá.
Wzywáłem Páná chwálebnego / á od nieprzyjáćiół mojich byłem wybáwiony.
Ogárnęły mię były boleśći śmierći / á potoki niepobożnych zátrwożyły mię.
Boleśći grobu ogárnęły mię były / záchwyćiły mię śidłá śmierći.
W utrápieniu mojim wzywáłem Páná / y wołáłem do Bogá mego ; wyſłuchał z kośćiołá ſwego głos mój / á wołánie moje przed oblicżnośćią jego przyƺło do uƺu jego.
Tedy śię źiemiá wzruƺyłá y zádrżáłá / á fundámenty gór zátrząſnęły śię / y wzruƺyły śię od gniewu jego.
Wyſtępował dym z nozdrzy jego / węgle śię rozpáliły od niego.
Nákłonił niebios / y zſtąpił / á ćiemność byłá pod nogámi jego.
A wśiádƺy ná Cherubá / látał ; látał ná ſkrzydłách wiátrowych.
Ucżynił ſobie z ćiemnośći ukryćie / około śiebie námiot ſwój z ćiemnych wód / y z gęſtych obłoków.
Od bláſku przed nim rozeƺły śię obłoki jego / grád y węgle ogniſte.
Y zágrzmiał ná niebie Pán / á Nájwyżƺy wydał głos ſwój / grád y węgle ogniſte.
Wypuśćił ſtrzáły ſwe / y rozproƺył ich / á błyſkáwicámi gęſtemi rozgromił ich.
Y okázáły śię głębokośći wód / á odkryte ſą grunty świátá ná fukánie twoje / Pánie! y ná tchnienie wiátru nozdrzy twojich.
Poſłáwƺy z wyſokośći záchwyćił mię ; wyćiągnął mię z wód wielkich.
Wyrwał mię od mocnego nieprzyjáćielá mego / y od tych / którzy mię mieli w nienáwiśći / choć byli mocniejƺymi nád mię.
Uprzedźili mię byli w dźień utrapienia mego ; ále Pán był podporą moją.
Wywiódł mię ná przeſtrzeńſtwo ; wyrwał mię / iż mię umiłował.
Nágrodźił mi Pán według ſpráwiedliwośći mojey ; według cżyſtośći rąk mojich oddał mi.
Bom ſtrzegł dróg Páńſkich / ánim odſtąpił niezbożnie od Bogá mego.
Bom miał wƺyſtkie ſądy jego przed ocżymá memi / á uſtáw jego nie odrzucáłem od śiebie.
Owƺem / byłem ƺcżerym przed nim / á ſtrzegłem śię od niepráwośći mojey.
Przetoż oddał mi Pán według ſpráwiedliwośći mojey / według cżyſtośći rąk mojich / która byłá przed ocżymá jego.
Ty / Pánie! z miłośiernym miłośiernie śię obejdźieƺ / á z mężem ƺcżerym ƺcżerze ſobie poſtąpiƺ.
Z uprzejmym uprzejmie śię obejdźieƺ / á z przewrotnym przewrotnie ſobie poſtąpiƺ ;
Abowiem ty lud utrápiony wybáwiƺ / á ocży wynioſłe poniżyƺ.
Ty zájiſte rozświećiƺ pochodnię moję ; Pán / Bóg mój / oświeći ćiemnośći moje.
Gdyż z tobą przebiłem śię przez wojſko / á z Bogiem moim przeſkocżyłem mur.
Drogá Boża doſkonáłá jeſt ; ſłowo Páńſkie jeſt ogniem wypłáwione. Tárcżą jeſt wƺyſtkich / którzy w nim ufáją.
Bo któż jeſt Bóg / oprócż Páná? á kto opoką / oprócż Bogá náƺego?
On jeſt Bogiem / który mię opáſuje mocą / á cżyni proſtą drogę moję.
Krzepi nogi moje jáko jelenie / á ná wyſokich miejſcách mojich ſtáwiá mię.
Ćwicży ręce moje do boju / ták / iż kruƺę łuk miedźiány rámionámi ſwemi.
Dáłeś mi też tárcż zbáwieniá twego / á práwicá twojá podpieráłá mię / y dobrotliwość twojá uwielmożyłá mię.
Rozƺerzyłeś kroki moje podemną / ták / że śię nie záchwiáły golenie moje.
Goniłem nieprzyjáćiół mojich / á dośćignąłem ich : y nie wróćiłem śię / áżem ich wytráćił.
Poráźiłem ich ták / iż nie mogli powſtáć ; upádli pod nogi moje.
Tyś mię opáſał mocą ku bitwie ; powſtáwájących przećiwko mnie obáliłeś pod mię.
Podáłeś mi tył nieprzyjáćiół mojich / ábym tych / którzy mię nienáwidzą / wykorzenił.
Wołálić / ále nie był / ktoby ich wybáwił ; do Páná / ále ich nie wyſłuchał.
Y potárłem ich / jáko proch od wiátru ; jáko błoto ná ulicách podeptáłem ich.
Wyrwáłeś mię od zwád ludzkich / á poſtáwiłeś mię głową národom ; lud / któregom nie znał / ſłużył mi.
Skoro uſłyƺeli / byli mi poſłuƺni ; cudzoźiemcy obłudnie mi śię poddáwáli.
Cudzoźiemcy opádli / á drżeli w zámknieniách ſwojich.
Żyje Pán / błogoſłáwioná opoká mojá ; przetoż niech będźie wywyżƺony Bóg zbáwieniá mego.
Bóg jeſt / który mi zlecá pomſty / y podbijá mi narody.
Tyś wybáwićiel mój od nieprzyjáćiół mojich ; tyś mię nád powſtáwájących przećiwko mnie wywyżƺył ; od mężá drápieżnego wyrwáłeś mię.
Przetoż ćię / Pánie! będę wyznáwał miedzy národámi / á będę śpiewał imieniowi twemu.
Boś zácnie wybáwił królá ſwego / á cżyniƺ miłośierdźie pomázáńcowi ſwemu Dawidowi / y náśieniu jego / áż ná wieki.
