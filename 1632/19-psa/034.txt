Pſálm Dawidowy / gdy ſobie odmienił poſtáwę przed Abimelechem / od którego będąc wygnány / odƺedł.
Będę błogoſłáwił Páná ná káżdy cżáſ ; záwżdy będźie chwáłá jego w uſtách moich.
W Pánu śię chlubić będźie duƺá mojá / co uſłyƺáwƺy pokorni rozweſelą śię.
Wielbijćie Páná ze mną / á wywyżƺájmy imię jego ſpołecżnie.
Bom ƺukáł Páná / y wyſłucháł mię / á ze wƺyſtkich ſtráchów moich wyrwáł mię.
Którzy náń ſpoglądáją / á zbiegáją śię do niego / oblicżá ich nie będą záwſtydzone.
Ten chudźiná wołáł / á Pán wyſłucháł / y ze wƺyſtkich ućiſków jego wybáwił go.
Zátácżá obóz Anioł Páńſki około tych / którzy śię go boją / y wyrywá ich.
Skoƺtujćież / á obácżćie / jáko jeſt dobry Pán : błogoſłáwiony cżłowiek / który w nim ufá.
Bójćie śię Páná święći jego ; bo niemáƺ niedoſtátku bojącym śię go.
Lwiętá niedoſtátek ćierpią y głód ; lecż ƺukájącym Páná nie będźie ſchodźiło ná wƺelkiem dobrem.
Pójdźćież ſynowie / ſłuchájćie mię ; bojáźni Páńſkiej wáſ náucżę.
Któż jeſt / co chce długo żyć / á miłuje dni / áby widźiáł dobrá?
Strzeż języká twego od złego / á wárg twoich / áby nie mówiły zdrády.
Odwróć śię od złego / á cżyń dobrze ; ƺukáj pokoju / á śćigáj go.
Ocży Páńſkie otworzone ſą ná ſpráwiedliwych / á uƺy jego ná wołánie ich ;
Ale oblicże Páńſkie przećiwko tym / którzy broją złośći / áby wykorzenił z źiemi pámiątkę ich.
Wołáją ſpráwiedliwi / á Pán ich wyſłuchiwá / y ze wƺyſtkich trudnośći ich wybáwiá ich.
Bliſki jeſt Pán tym / którzy ſą ſkruƺonego ſercá / á utrápionych w duchu záchowuje.
Wiele złego przychodźi ná ſpráwiedliwego ; ále z tego wƺyſtkiego wyrywá go Pán.
On ſtrzeże wƺyſtkich kośći jego / ták / iż y jedná z nich nie ſkruƺy śię.
Zábije złość niepobożnego / á którzy w nienáwiśći máją ſpráwiedliwego / będą ſpuſtoƺeni ;
Ale Pán odkupi duƺę ſług ſwoich / á nie będą ſpuſtoƺeni wƺyſcy / którzy w nim ufáją.
