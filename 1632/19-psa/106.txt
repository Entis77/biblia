Hállelujá. Wyſłáwiájćie Páná ; ábowiem dobry / ábowiem ná wieki miłośierdźie jego.
Któż wyſłowi niezmierną moc Páńſką / á wypowie wƺyſtkę chwáłę jego?
Błogoſłáwieni / którzy ſtrzegą ſądu / á cżynią ſpráwiedliwość ná káżdy cżáſ.
Pámiętáj ná mię / Pánie! dla miłośći ku ludowi ſwemu ; náwiedźże mię zbáwieniem ſwojim /
Abym używáł dobrego z wybránymi twoimi / á weſelił śię w rádośći národu twego / y chlubił śię weſpół z dźiedźictwem twojim.
Zgrzeƺyliśmy z ojcámi ſwymi ; nieſpráwiedliwieśmy cżynili / y niepráwość popełniáli.
Ojcowie náśi w Egipćie nie zrozumieli cudów twoich / áni pámiętáli ná wielkość miłośierdźiá twego ; ále odpornymi byli przy morzu Cżerwonem.
A wƺákże ich wyſwobodźił dla imieniá ſwego / áby oznájmił moc ſwoję.
Bo zgromił morze Cżerwone / y wyſchło / á przewiódł ich przez przepáśći / jáko przez puƺcżę.
A ták záchowáł ich od ręki tego / który ich miał w nienáwiśći / á wykupił ich z ręki nieprzyjáćielſkiej.
W tem okryły wody tych / którzy ich ćiążyli ; nie zoſtáł áni jeden z nich.
A choć uwierzyli ſłowom jego / y wyſłáwiáli chwáłę jego :
Przećież prędko zápomnieli ná ſpráwy jego / y nie cżekáli ná rády jego.
Ale zdjęći będąc chćiwośćią ná puƺcży / kuśili Bogá ná puſtyniách.
Y dáł im / cżego żądáli / á wƺákże przepuśćił ſuchoty ná nich.
Zátem gdy śię wzruƺyli záwiśćią przećiw Mojżeƺowi w oboźie / y przećiw Aáronowi / świętemu Páńſkimu :
Otworzyłá śię źiemiá / y pożárłá Dátáná / y okryłá rotę Abironową /
Y zápálił śię ogień ná zebránie ich ; płomień ſpálił niepobożnych.
Spráwili y ćielcá ná Horebie / y kłániáli śię báłwánowi litemu /
Y odmienili chwáłę ſwą w podobieńſtwo wołu / jedzącego tráwę.
Zápomnieli ná Bogá / wybáwićielá ſwego / który cżynił wielkie rzecży w Egipćie ;
Rzecży dźiwne w źiemi Chámowej / rzecży ſtráƺne przy morzu Cżerwonem.
Przetoż rzekł / że ich chćiáł wytráćić / gdyby śię był Mojżeƺ / wybrány jego / nie ſtáwił w onem rozerwániu przed nim / á nie odwróćił popędliwośći jego / áby ich nie tráćił.
Wzgárdźili też źiemią pożądáną / nie wierząc ſłowu jego.
Y ƺemrząc w námiotách ſwoich / nie byli poſłuƺni głoſowi Páńſkimu.
Przetoż podnióſł rękę ſwoję przećiwko nim / áby ich pobił ná puƺcży ;
A żeby rozrzućił náśienie ich między pogán / y rozproƺył ich po źiemiách.
Sprzęgli śię też byli z báłwánem Báálfegorem / á jedli ofiáry umárłych.
A ták dráźnili Bogá ſpráwámi ſwemi / że śię ná nich oborzyłá plágá ;
Aż śię záſtáwił Fineeſ / á pomſtę ucżynił / y rozerwáná jeſt oná plágá ;
Co mu pocżytáno ku ſpráwiedliwośći od národu do národu / áż ná wieki.
Znowu go byli wzruƺyli do gniewu u wód Merybá / ták / iż śię źle dźiáło y z Mojżeƺem dla nich.
Abowiem rozdráźnili duchá jego / że wyrzekł co nieſłuƺne uſty ſwemi.
Nádto nie wytráćili onych národów / o których im był Pán powiedźiáł.
Ale pomięƺáwƺy śię z onemi národámi / náucżyli śię ſpráw ich :
Y ſłużyli báłwánom ich / które im były śidłem.
Abowiem dyjábłom ofiárowáli ſynów ſwoich / y córki ſwoje /
Y wylewáli krew niewinną / krew ſynów ſwoich / y córek ſwoich / których ofiárowáli báłwánom rytym Chánánejſkim / ták / że ſplugáwioná byłá źiemiá onem krwi rozlániem.
Y zmázáli śię ſpráwámi ſwemi / á cudzołożyli wynálázkámi ſwemi.
Przetoż zápáliwƺy śię Pán w popędliwośći przećiw ludowi ſwemu / obrzydźił ſobie dźiedźictwo ſwoje /
Y podáł ich w ręce pogánom ; á pánowáli nád nimi / którzy ich mieli w nienáwiśći ;
Y ućiſkáli ich nieprzyjáćiele ich / ták / że poniżeni byli pod ręką ich.
Cżęſtokroć ich wybáwiáł ; wƺákże go oni wzruƺáli do gniewu rádámi ſwemi / zácżem poniżeni byli dla niepráwośći ſwoich.
A wƺákże wejrzáł ná ućiſk ich / y uſłyƺáł wołánie ich.
Bo ſobie wſpomniáł ná przymierze ſwoje z nimi / á żáłowáł tego według wielkiej litośći ſwojey.
Ták / że im zjednáł miłośierdźie przed ocżymá wƺyſtkich / którzy ich byli pojmáli.
Wybáwże nas / Pánie / Boże náƺ! á zgromádź nas z tych pogán / ábyśmy wyſłáwiáli imię świętobliwośći twojey / á chlubili śię w chwále twojey.
Błogoſłáwiony Pán / Bóg Izráelſki / od wieków áż ná wieki ; ná co niech rzecże wƺyſtek lud : Amen / Hállelujá.
