Pieśń ſtopni dlá Sálomoná. Jeźli Pán domu nie zbuduje / próżno prácują ći / którzy go budują ; jeźli Pán nie będzie ſtrzegł miáſtá / próżno cżuje ten / który go ſtrzeże.
Próżno máćie ráno wſtáwáć / długo śiádáć / y jeść chleb boleśći / poniewáż Pán umiłowánemu ſwemu ſen dáje.
Oto dziátki ſą dziedzictwem od Páná / á płód żywotá nágrodą.
Jáko ſtrzáły w ręku mocárzá / ták ſą dziátki / które śię dárzą.
Błogoſłáwiony mąż / który niemi nápełnił ſájdák ſwój ; nie będą záwſtydzeni / gdy śię w brámie rozpieráć będą z nieprzyjáćiołmi ſwymi.
