Przedniejƺemu śpiewákowi / jáko : Nie zátrácáj / pieśń złotá Dawidowá.
O zgromádzenie! Izáli popráwdźie ſpráwiedliwość mówićie? A uprzejmież ſądźićie / wy ſynowie ludzcy?
Owƺem / rádniej w ſercu niepráwośći knujećie / á gwáłty rąk wáƺych ná źiemi odwáżáćie.
Odłącżyli śię niezbożnicy záráz od národzeniá ; pobłądźili záráz z żywotá mátki ſwej / mówiąc kłámſtwo.
Jád máją w ſobie / jáko wężowy / jáko jád żmii głuchej / która zátulá ucho ſwoje /
Aby nie ſłyƺáłá głoſu záklinácżá / áni cżárowniká w cżárách biegłego.
O Boże! pokruƺże zęby ich w uſtách ich ; połám / Pánie! lwiąt trzonowe zęby.
Niech śię rozpłyną jáko wodá / niech śię wniwecż obrócą ; niech będą jáko ten / który náćiągá łuk / wƺákże śię ſtrzáły jego łámią.
Jáko ślimák / który ſchodźi y niƺcżeje ; jáko mártwy płód niewieśći niech nie oglądáją ſłońcá.
Ciernie wáƺe pierwej niż wypuƺcżą tárny ſwoje / zá źieloná w gniewie Bożym jáko wichrem porwáne będą.
Y będźie śię weſelił ſpráwiedliwy / gdy ujrzy pomſtę ; nogi ſwoje umyje we krwi niepobożnego.
Y rzecże káżdy : Zápráwdęć ſpráwiedliwy odnieśie pożytek z ſpráwiedliwośći ſwojey ; záiſteć jeſt Bóg / który ſądźi ná źiemi.
