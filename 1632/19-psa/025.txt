Pſálm Dawidowy. Do ćiebie / Pánie! duƺę moję podnoƺę.
Boże mój! w tobie ufám ; niech nie będę záwſtydzony / niech śię nie weſelą nieprzyjáćiele moi ze mnie.
A ták wƺyſcy / którzy ocżekują ćiebie / nie będą záwſtydzeni ; záwſtydzeni będą bez przycżyny niepráwość cżyniący.
Pánie! dáj mi poznáć drogi twe / śćieżek twojich náucż mię.
Dáj / ábym chodźił w práwdźie twojey / y náucż mię ; boś ty jeſt Bóg zbáwieniá mego ; ćiebie ocżekuję dniá káżdego.
Wſpomnij ná litośći twoje / Pánie! y ná miłośierdźiá twoje / które ſą od wieku.
Grzechów młodośći mojey / y przeſtępſtw mojich nie rácż pámiętáć ; według miłośierdźiá twego wſpomnij ná mię / dla dobroći twojey / Pánie!
Dobry y práwy jeſt Pán ; przetoż drogi náucżá grzeƺników.
Poprowádźi ćichych w ſądźie / á náucży pokornych drogi ſwojey.
Wƺyſtkie śćieżki Páńſkie ſą miłośierdźie y práwdá tym / którzy ſtrzegą przymierza jego / y świádectwá jego.
Pánie! dla imieniá twego odpuść niepráwość moję / bo wielká jeſt.
Jeſtże cżłowiek / co śię boi Páná? Náucży go drogi / którąby miał obráć.
Duƺá jego w dobrem przemieƺkiwáć będźie / á náśienie jego odźiedźicży źiemię.
Tájemnicá Páńſká objáwioná jeſt tym / którzy śię go boją / á przymierze ſwoje oznájmuje im.
Ocży moje uſtáwicżnie pátrzą ná Páná ; ábowiem on wywodźi z śieći nogi moje.
Wejrzyjże ná mię / á zmiłuj śię nádemną ; bom jeſt nędzny y opuƺcżony.
Utrapienia ſercá mego rozmnożyły śię ; z ućiſków mojich wywiedź mię.
Obácż udręcżenie moje / y prácę moję / á odpuść wƺyſtkie grzechy moje.
Obácż nieprzyjáćiół mojich / jáko śię rozmnożyli / á máją mię nieſłuƺnie w nienáwiśći.
Strzeż duƺy mojey / á wyrwij mię / ábym nie był poháńbiony ; bo w tobie nádźieję mam.
Niewinność y ƺcżerość niech mię ſtrzegą ; bom ná ćię ocżekiwał.
O Boże! wybáwże Izráelá ze wƺyſtkich ućiſków jego.
