Przedniejƺemu śpiewákowi ná Soſánnim pſálm Dawidowy.
Wybáw mię / o Boże! boć przyƺły wody áż do duƺy mojey.
Pogrążony jeſtem w głębokiem błoćie / gdźie dná niemáƺ ; przyƺedłem w głębokośći wód / á náwáłność ich porwáłá mię.
Sprácowáłem śię wołájąc / wyſchło gárdło moje ; uſtáły ocży moje / gdym ocżekiwał Bogá mojego.
Więcey jeſt tych / którzy mię máją w nienáwiśći bez przycżyny / niż włoſów ná głowie mojey ; zmocnili śię ći / którzy mię wygubić uśiłują / á ſą nieprzyjáćiołmi mymi nieſłuƺnie ; cżegom nie wydárł / muśiáłem nágrádzáć.
Boże! ty znáƺ głupſtwo moje / á wyſtępki moje nie ſą tájne przed tobą.
Niechájże nie będą záwſtydzeni dla mnie ći / którzy ná ćię ocżekują / Pánie / Pánie zaſtępów! niech nie przychodzą dla mnie do háńby ći / którzy ćię ƺukáją / o Boże Izráelſki!
Bo dla ćiebie ponoƺę urągánie / á zelżywość okryłá oblicże moje.
Stáłem śię obcym bráćiom moim / á cudzoźiemcem ſynom mátki mojey /
Przeto / że gorliwość domu twego zżárłá mię / á urągánie urągájących tobie przypádło ná mię.
Gdym płákał y trápił poſtem duƺę moję / ſtáło mi śię to poháńbienie.
Gdym wźiął ná śię wór miáſto ƺáty / byłem u nich przypowieśćią.
Mówili o mnie ći / którzy śiedźieli w brámie / á byłem pioſnką u tych / którzy pili mocny nápój.
Ale ja obrácám modlitwę moję do ćiebie / Pánie! cżás jeſt upodobánia twego ; o Boże! według wielkośći miłośierdźiá twego wyſłuchájże mię / dla prawdy zbáwieniá twego.
Wyrwij mię z błotá / ábym nie był pogrążony ; niech będę wyrwány od tych / którzy mię nienáwidzą / jáko z głębokośći wód ;
Aby mię nie zátopiły ſtrumienie wód / y nie pożárłá głębiá y nie záwárłá nádemną ſtudniá wierzchu ſwego.
Wyſłuchájże mię / Pánie! boć dobre jeſt miłośierdźie twoje ; według wielkiej litośći twojey wejrzyj ná mię.
Nie zákrywájże oblicżá twego od ſługi ſwego / bom jeſt w utrápieniu ; pośpieƺże śię / wyſłucháj mię.
Przybliż śię do duƺy mojey / á wybáw ją ; dla nieprzyjáćiół mojich odkup mię.
Ty znáƺ poháńbienie moje / y zelżywość moję / y wſtyd mój : przed tobąć ſą wƺyſcy nieprzyjáćiele moi.
Poháńbienie pokruƺyło ſerce moje / z cżegom był żáłośny ; ocżekiwáłem / ázáliby śię mię kto użálił / ále nikt nie był ; ázáliby mię kto poćieƺył / álem nie ználázł.
Owƺem / miáſto pokármu podáli mi żółć / á w prágnieniu mojim nápoili mię octem.
Niechájże im będźie ſtół ich przed nimi śidłem / á ƺcżęśćie ich ná upádek.
Niech śię záćmią ocży ich / áby nie widźieli / á biodrá ich niech śię záwżdy chwieją.
Wylij ná nich rozgniewánie ſwoje / á popędliwość gniewu twego niech ich ogárnie.
Niech będźie mieƺkánie ich puſte / w námiotách ich niech nikt nie mieƺká.
Bo tego / któregoś ty ubił / prześládują / á o boleśći poránionych twojich rozmáwiáją.
Przydájże niepráwość ku niepráwośći ich / á niech nie przychodzą do ſpráwiedliwośći twojey.
Niech będą wymázáni z kśiąg żyjących / á z ſpráwiedliwymi niech nie będą zápiſáni.
Jámći utrápiony / y zboláły ; lecż zbáwienie twoje / Boże! ná miejſcu bezpiecżnem poſtáwi mię.
Tedy będę chwálił imię Boże pieśnią / á będę je wielbił z dźiękcżynieniem.
A będźie to przyjemniejƺe Pánu / niżeli wół álbo ćielec rogáty z rozdźielonemi kopytámi.
To widząc pokorni rozrádują śię / ƺukájąc Bogá / á ożyje ſerce ich ;
Iż wyſłuchiwá Pán ubogich / á więźniámi ſwymi nie gárdźi.
Niech go chwálą niebioſá y źiemiá / morze y wƺyſtko / co śię w nich ruchá.
Bógći záiſte záchowá Syon / y pobuduje miáſtá Judzkie ; y będą tám mieƺkáć / á źiemię tę dźiedźicżnie otrzymáją.
Tákże y náśienie ſług jego dźiedźicżnie ją otrzymá / á którzy miłują imię jego / będą w niej mieƺkáć.
