Przedniejƺemu śpiewákowi ná Soſánnim pſálm Dawidowy.
Wybáw mię / o Boże! boć przyƺły wody áż do duƺy mojej.
Pogrążony jeſtem w głębokiem błoćie / gdzie dná niemáƺ ; przyƺedłem w głębokośći wód / á náwáłność ich porwáłá mię.
Sprácowáłem śię wołájąc / wyſchło gárdło moje ; uſtáły ocży moje / gdym ocżekiwáł Bogá mojego.
Więcej jeſt tych / którzy mię máją w nienáwiśći bez przycżyny / niż włoſów ná głowie mojej ; zmocnili śię ći / którzy mię wygubić uśiłują / á ſą nieprzyjáćiołmi mymi nieſłuƺnie ; cżegom nie wydárł / muśiáłem nágrádzáć.
Boże! ty znáƺ głupſtwo moje / á wyſtępki moje nie ſą tájne przed tobą.
Niechájże nie będą záwſtydzeni dlá mnie ći / którzy ná ćię ocżekują / Pánie / Pánie záſtępów! niech nie przychodzą dlá mnie do háńby ći / którzy ćię ƺukáją / o Boże Izráelſki!
Bo dlá ćiebie ponoƺę urągánie / á zelżywość okryłá oblicże moje.
Stáłem śię obcym bráćiom moim / á cudzoziemcem ſynom mátki mojej /
Przeto / że gorliwość domu twego zżárłá mię / á urągánie urągájących tobie przypádło ná mię.
Gdym płákáł y trápił poſtem duƺę moję / ſtáło mi śię to poháńbienie.
Gdym wziął ná śię wór miáſto ƺáty / byłem u nich przypowieśćią.
Mówili o mnie ći / którzy śiedzieli w brámie / á byłem pioſnką u tych / którzy pili mocny nápój.
Ale já obrácám modlitwę moję do ćiebie / Pánie! cżáſ jeſt upodobániá twego ; o Boże! według wielkośći miłośierdziá twego wyſłuchájże mię / dlá práwdy zbáwieniá twego.
Wyrwij mię z błotá / ábym nie był pogrążony ; niech będę wyrwány od tych / którzy mię nienáwidzą / jáko z głębokośći wód ;
Aby mię nie zátopiły ſtrumienie wód / y nie pożárłá głębiá y nie záwárłá nádemną ſtudniá wierzchu ſwego.
Wyſłuchájże mię / Pánie! boć dobre jeſt miłośierdzie twoje ; według wielkiej litośći twojej wejrzyj ná mię.
Nie zákrywájże oblicżá twego od ſługi ſwego / bom jeſt w utrápieniu ; pośpieƺże śię / wyſłucháj mię.
Przybliż śię do duƺy mojej / á wybáw ją ; dlá nieprzyjáćiół moich odkup mię.
Ty znáƺ poháńbienie moje / y zelżywość moję / y wſtyd mój : przed tobąć ſą wƺyſcy nieprzyjáćiele moi.
Poháńbienie pokruƺyło ſerce moje / z cżegom był żáłośny ; ocżekiwáłem / ázáliby śię mię kto użálił / ále nikt nie był ; ázáliby mię kto poćieƺył / álem nie ználázł.
Owƺem / miáſto pokármu podáli mi żółć / á w prágnieniu mojem nápoili mię octem.
Niechájże im będzie ſtół ich przed nimi śidłem / á ƺcżęśćie ich ná upádek.
Niech śię záćmią ocży ich / áby nie widzieli / á biodrá ich niech śię záwżdy chwieją.
Wylij ná nich rozgniewánie ſwoje / á popędliwość gniewu twego niech ich ogárnie.
Niech będzie mieƺkánie ich puſte / w námiotách ich niech nikt nie mieƺká.
Bo tego / któregoś ty ubił / prześládują / á o boleśći poránionych twoich rozmáwiáją.
Przydájże niepráwość ku niepráwośći ich / á niech nie przychodzą do ſpráwiedliwośći twojej.
Niech będą wymázáni z kśiąg żyjących / á z ſpráwiedliwymi niech nie będą zápiſáni.
Jámći utrápiony / y zboláły ; lecż zbáwienie twoje / Boże! ná miejſcu bezpiecżnem poſtáwi mię.
Tedy będę chwálił imię Boże pieśnią / á będę je wielbił z dziękcżynieniem.
A będzie to przyjemniejƺe Pánu / niżeli wół álbo ćielec rogáty z rozdzielonemi kopytámi.
To widząc pokorni rozrádują śię / ƺukájąc Bogá / á ożyje ſerce ich ;
Iż wyſłuchiwá Pán ubogich / á więźniámi ſwymi nie gárdzi.
Niech go chwálą niebioſá y ziemiá / morze y wƺyſtko / co śię w nich ruchá.
Bógći záiſte záchowá Syon / y pobuduje miáſtá Judzkie ; y będą tám mieƺkáć / á ziemię tę dziedzicżnie otrzymáją.
Tákże y náśienie ſług jego dziedzicżnie ją otrzymá / á którzy miłują imię jego / będą w niej mieƺkáć.
