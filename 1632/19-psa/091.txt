Ten / który mieƺká w ochronie Nájwyżƺego / y w ćieniu Wƺechmocnego przebywáć będźie ;
Rzecże Pánu : Nádźiejá mojá y zámek mój / Bóg mój / w nim nádźieję mieć będę.
Onći záiſte wybáwi ćię z śidłá łowcżego / y z powietrzá nájjádowitƺego.
Pierzem ſwem okryje ćię / á pod ſkrzydłámi jego bezpiecżnym będźieƺ ; práwdá jego tárcżą y puklerzem.
Nie ulęknieƺ śię ſtráchu nocnego / áni ſtrzáły látájącey we dnie ;
Ani zárázy morowej / która przechodźi w ćiemnośći / áni powietrzá morowego / które zátrácá w południe.
Pádnie po boku twym tyśiąc / á dźieśięć tyśięcy po práwej ſtronie twojey ; ále śię do ćiebie nie przybliży.
Tylko to ocżymá twemi obácżyƺ / á nágrodę niepobożnych oglądáƺ.
Poniewáżeś ty Páná / który jeſt nádźieją moją / y Nájwyżƺego / zá przybytek ſwój położył :
Nie ſpotká ćię nic złego / áni jáká plágá przybliży śię do námiotu twego.
Abowiem Aniołom ſwoim przykázáł o tobie / áby ćię ſtrzegli ná wƺyſtkich drogách twoich.
Ná rękách nośić ćię będą / byś ſnáć nie obráźił o kámień nogi twojey.
Po lwie / y po bázyliƺku deptáć będźieƺ / lwię y ſmoká podepcżeƺ.
Iż śię we mnie / mówi Pán / rozkocháł / wyrwę go / y wywyżƺę go / przeto / iż poznał imię moje.
Będźie mię wzywáł / á wyſłuchám go ; Ja z nim będę w utrápieniu / wyrwę go / y uwielbię go.
Długośćią dni náſycę go / y okáżę mu zbáwienie moje.
