Przedniejƺemu śpiewákowi pſálm y pieśń Dawidowá.
Tobie przynáleży / o Boże! chwáłá ná Syonie / á tobie ślub má być oddány.
Ty wyſłuchiwáƺ modlitwy ; przetoż do ćiebie przychodźi wƺelkie ćiáło.
Wielkie niepráwośći / które wźięły górę nád námi / y przeſtępſtwá náƺe ty ocżyƺcżáƺ.
Błogoſłáwiony / kogo ty obieráƺ á przyjmujeƺ / áby mieƺkáł w śieniách twoich ; będźiemy náſyceni dobrámi domu twego / w świątnicy kośćiołá twego.
Przedźiwne rzecży podług ſpráwiedliwośći mówiƺ do náſ / Boże zbáwieniá náƺego / nádźiejo wƺyſtkich krájów źiemi / y morzá dálekiego!
Który utwierdzáƺ góry mocą ſwoją / śiłą przepáſány będąc ;
Który uśmierzáƺ ƺum morſki / ƺum náwáłnośći jego / y wzruƺenie národów /
Ták / że śię báć muƺą cudów twoich / którzy mieƺkáją ná krájách źiemi ; których náſtáwániem poránku y wiecżorá do weſelá pobudzáƺ.
Náwiedzáƺ źiemię / y odwilżáƺ ją ; obfićie ją ubogácáƺ ſtrumieniem Bożym / nápełnionym wodámi / y gotujeƺ zboże ich / gdy ją ták przypráwiáƺ.
Zágony jej nápáwáƺ / brózdy jej zniżáƺ / dżdżámi ją odmiękcżáƺ / á urodzájom jej błogoſłáwiƺ.
Koronujeƺ rok dobroćią twą / á śćieżki twoje ſkrápiáƺ tłuſtośćią.
Skrápiáƺ páſtwiſká ná puſtyniách ; ták / że y págórki rádośćią przepáſáne bywáją.
Przyodźiewáją śię polá ſtádámi owiec / á doliny okrywáją śię zbożem ; ták / że wykrzykáją y śpiewáją.
