Pieśń wyucżájącá / podáná Aſáfowi. Przecżżeś náſ / o Boże! do końcá odrzućił? Przecżże śię rozpáliłá zápálcżywość twojá przećiwko owcom páſtwiſká twego?
Wſpomnij ná zgromádzenie twoje / któreś ſobie zdáwná nábył y odkupił / ná pręt dziedzictwá twego / ná tę górę Syon / ná której mieƺkáƺ.
Poſpieƺże śię ná ſrogie popuſtoƺenie ; á jáko wƺyſtko poburzył nieprzyjáćiel w świątnicy!
Rycżeli nieprzyjáćiele twoi w pośrodku zgromádzeniá twego / á ná znák tego zoſtáwili wiele chorągwi ſwoich.
Zá rycerzá miáno tego / który śię z wyſoká z śiekierą zánośił / rąbiąc drzewo wiązániá jego.
A teráz już y rzezániá jego ná porząd śiekierámi y młotámi tłuką.
Záłożyli ogień w świątnicy twojej / á obáliwƺy ná ziemię / ſplugáwili przybytek imieniá twego.
Mówili w ſercu ſwojim : Zburzmy je poſpołu ; popálili wƺyſtkie przybytki Boże w ziemi.
Znáków náƺych nie widzimy : już niemáƺ proroká / y niemáƺ między námi / któryby wiedziáł / póki to má trwáć.
Dokądże / o Boże! przećiwnik będzie urągáć? izáli nieprzyjáćiel będzie bluźnił imię twoje áż ná wieki?
Przecżże zſtrzymujeƺ rękę twoję ; á práwicy ſwej z zánádrzá ſwego cále nie dobędzieƺ?
Wƺákeś ty / Boże! zdáwná królem moim ; ty ſpráwujeƺ hojne zbáwienie w pośród ziemi.
Tyś mocą twoją rozdzielił morze / á potárłeś głowy wielorybów w wodách.
Tyś ſkruƺył głowę Lewiátáná / dáłeś go zá pokárm ludowi ná puƺcży.
Tyś przerwáł źródłá y potoki ; tyś oſuƺył rzeki byſtre.
Twójći jeſt dzień / twojá też y noc ; tyś ucżynił świátło y ſłońce.
Tyś záłożył wƺyſtkie gránice ziemi ; láto y zimę tyś ſpráwił.
Wſpomnijże ná to / że nieprzyjáćiel zelżył Páná / á lud ƺálony jáko urągá imieniowi twemu.
Nie podáwájże tej kupie duƺy ſynogárlicy twojej ; ná ſtádko ubogich twoich nie zápomináj ná wieki.
Obejrzyj śię ná przymierze twoje ; ábowiem y nájćiemniejƺe kąty ziemi pełne jáſkiń drápieſtwá.
Niechájże nędznik nie odchodzi z háńbą ; ubogi y żebrák niecháj chwáli imię twoje.
Powſtáńże / o Boże! ujmij śię o ſpráwę twoję ; wſpomnij ná poháńbienie twoje / które śię dzieje od ƺálonych ná káżdy dzień.
Nie zápominájże wykrzykániá nieprzyjáćiół twoich / y huku tych / co przećiwko tobie powſtáwáją / który śię uſtáwicżnie śili.
