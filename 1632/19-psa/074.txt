Pieśń wyucżájącá / podáná Aſáfowi. Przecżżeś náſ / o Boże! do końcá odrzućił? Przecżże śię rozpáliłá zápálcżywość twojá przećiwko owcom páſtwiſká twego?
Wſpomnij ná zgromádzenie twoje / któreś ſobie zdáwná nábył y odkupił / ná pręt dźiedźictwá twego / ná tę górę Syon / ná której mieƺkáƺ.
Poſpieƺże śię ná ſrogie popuſtoƺenie ; á jáko wƺyſtko poburzył nieprzyjáćiel w świątnicy!
Rycżeli nieprzyjáćiele twoi w pośrodku zgromádzeniá twego / á ná znák tego zoſtáwili wiele chorągwi ſwoich.
Zá rycerzá miáno tego / który śię z wyſoká z śiekierą zánośił / rąbiąc drzewo wiązániá jego.
A teráz już y rzezániá jego ná porząd śiekierámi y młotámi tłuką.
Záłożyli ogień w świątnicy twojej / á obáliwƺy ná źiemię / ſplugáwili przybytek imieniá twego.
Mówili w ſercu ſwojim : Zburzmy je poſpołu ; popálili wƺyſtkie przybytki Boże w źiemi.
Znáków náƺych nie widźimy : już niemáƺ proroká / y niemáƺ między námi / któryby wiedźiáł / póki to má trwáć.
Dokądże / o Boże! przećiwnik będźie urągáć? izáli nieprzyjáćiel będźie bluźnił imię twoje áż ná wieki?
Przecżże zſtrzymujeƺ rękę twoję ; á práwicy ſwej z zánádrzá ſwego cále nie dobędźieƺ?
Wƺákeś ty / Boże! zdáwná królem moim ; ty ſpráwujeƺ hojne zbáwienie w pośród źiemi.
Tyś mocą twoją rozdźielił morze / á potárłeś głowy wielorybów w wodách.
Tyś ſkruƺył głowę Lewiátáná / dáłeś go zá pokárm ludowi ná puƺcży.
Tyś przerwáł źródłá y potoki ; tyś oſuƺył rzeki byſtre.
Twójći jeſt dźień / twojá też y noc ; tyś ucżynił świátło y ſłońce.
Tyś záłożył wƺyſtkie gránice źiemi ; láto y źimę tyś ſpráwił.
Wſpomnijże ná to / że nieprzyjáćiel zelżył Páná / á lud ƺálony jáko urągá imieniowi twemu.
Nie podáwájże tej kupie duƺy ſynogárlicy twojej ; ná ſtádko ubogich twoich nie zápomináj ná wieki.
Obejrzyj śię ná przymierze twoje ; ábowiem y nájćiemniejƺe kąty źiemi pełne jáſkiń drápieſtwá.
Niechájże nędznik nie odchodźi z háńbą ; ubogi y żebrák niecháj chwáli imię twoje.
Powſtáńże / o Boże! ujmij śię o ſpráwę twoję ; wſpomnij ná poháńbienie twoje / które śię dźieje od ƺálonych ná káżdy dźień.
Nie zápominájże wykrzykániá nieprzyjáćiół twoich / y huku tych / co przećiwko tobie powſtáwáją / który śię uſtáwicżnie śili.
