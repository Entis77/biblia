Pieśń y pſálm Aſáfowy.
O Boże! nie milcżże / nie bądź jáko ten / co nie ſłyƺy / y nie chćiej śię uſpokoić / o Boże!
Bo śię oto nieprzyjáćiele twoi burzą / á ći / którzy ćię w nienáwiśći máją / podnoƺą głowę.
Przećiwko ludowi twemu wymyślili chytrą rádę / á ſpiknęli śię przećiw tym / których ty ochrániáƺ ;
Mówiąc : Pójdźćie / á wytráćmy ich / niech nie będą národem / ták / żeby y nie wſpomináno więcey imieniá Izráelſkiego.
Abowiem ſpiknęli śię jednomyślnie / przymierze przećiwko tobie ucżynili :
Námioty Edomcżyków / y Iſmáelcżyków / Moábcżyków / y Agáreńcżyków /
Giebálcżyków / y Ammonitcżyków / y Amálekitcżyków / tákże Filiſtyńcżyków z tymi / którzy mieƺkáją w Tyrze ;
Więc y Aſſyryjcżycy złącżyli śię z nimi / będąc rámieniem ſynom Lotowym. Selá.
Ucżyńże im ták jáko Mádyjáncżykom / jáko Syſárze / jáko Jábinowi u potoku Cyſon.
Którzy ſą wygłádzeni w Endor ; ſtáli śię jáko gnój ná źiemi.
Obchodźże śię z nimi / y z ich hetmánámi / jáko z Orebem / y jáko z Zebą / y jáko z Zebeem / y jáko z Sálmánem / ze wƺyſtkimi kśiążętámi ich ;
Bo rzekli : Pośiądźmy dźiedźicżnie przybytki Boże.
Boże mój! ucżyńże ich jáko koło / y jáko źdźbło przed wiátrem.
Jáko ogień / który lás páli / y jáko płomień / który zápálá góry.
Ták ich ty wichrem twoim śćigáj / á burzą twą zátrwóż ich.
Nápełnij twárze ich poháńbieniem / áby ƺukáli imieniá twego / Pánie!
Niech będą záwſtydzeni y uſtráƺeni áż ná wieki / á będąc poháńbieni niech záginą.
A ták niech poznáją / żeś ty / którego imię jeſt Pán / tyś ſám Nájwyżƺym nád wƺyſtką źiemią.
