Pſálm Aſáfowi podány. Bóg nád Bogámi / Pán mówił y przyzwał źiemię od wſchodu ſłońcá áż do záchodu jego.
Objáśnił śię Bóg z Syonu w doſkonáłej ozdobie.
Przydźie Bóg náƺ / á nie będźie milcżał ; ogień przed twárzą jego będźie pożerał / á około niego powſtánie wicher gwáłtowny.
Przyzwie z góry niebioſá y źiemię / áby ſądźił lud ſwój /
Mówiąc : Zgromádźćie mi świętych mojich / którzy ze mną ucżynili przymierze przy ofierze.
Tedy niebioſá opowiedzą ſpráwiedliwość jego ; ábowiem ſám Bóg jeſt ſędźią. Selá.
Słucháj / ludu mój! á będę mówił ; ſłucháj / Izráelu! á oświádcżę śię przed tobą : Jam Bóg / Bóg twój Jam jeſt.
Nie będę ćię z ofiár twojich winił / áni z cáłopalenia twego / które ſą záwƺe przedemną.
Nie wezmę z domu twojego ćielcá / áni z okołu twego kozłów.
Abowiem mój jeſt wƺelki zwierz leśny / y tyśiące bydłá po górách.
Znam wƺyſtko ptáſtwo po górách / y zwierz polny jeſt przedemną.
Będęli łáknął / nie rzekęć o to ; bo mój jeſt okrąg źiemi / y nápełnienie jego.
Izali jádám mięſo wołowe? álbo pijám krew kozłową?
Ofiáruj Bogu chwałę / y oddáj Nájwyżƺemu śluby twoje ;
A wzywáj mię w dźień utrapienia : tedy ćię wyrwę / á ty mię uwielbiƺ.
Lecż niezbożnemu rzekł Bóg : Cóżći do tego / że opowiádáƺ uſtáwy moje / á bierzeƺ przymierze moje w uſtá twoje?
Ponieważ maƺ w nienáwiśći kárność / y zárzućiłeś ſłowá moje zá śię.
Widźiƺli złodźiejá / bieżyƺ z nim / á z cudzołożnikámi maƺ ſkłád twój.
Uſtá twoje rozpuƺcżáƺ ná złe / á język twój ſkłádá zdrády.
Záśiádƺy mówiƺ przećiwko brátu twemu / á lżyƺ ſyná mátki twojey.
Toś cżynił / á Jam milcżał ; dlátegoś mniemał / żem ja tobie podobny / ále będę ćię kárał / y ſtáwięć to przed ocży twoje.
Zrozumiejćież to wżdy teraz / którzy zápomináćie Bogá / bym was ſnadź nie porwał / á nie będźie ktoby was wyrwał.
Kto mi ofiáruje chwałę / ucżći mię ; á temu / który nápráwiá drogę ſwą / ukáżę zbáwienie Boże.
