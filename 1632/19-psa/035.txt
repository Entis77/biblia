Pſálm Dawidowy. Rozpieráj śię / Pánie! z tymi / którzy śię ze mną ſpieráją ; á wálcż przećiwko tym / którzy wálcżą przećiwko mnie.
Porwij pukierz y tárcżę / á powſtáń ná rátunek mój.
Dobądź włócżni / á ſtáw śię ná drodze przećiwko tym / którzy mię prześládują. Rzecżże duƺy mojej : Jám jeſt zbáwieniem twojim.
Niech będą poháńbieni y záwſtydzeni / którzy ƺukáją duƺy mojej ; niech tył podádzą / y niech będą záwſtydzeni / którzy mi źle myślą.
Niech będą jáko plewy przed wiátrem / á Anioł Páńſki niecháj ich rozproƺy.
Niech będzie drogá ich ćiemná y śliſká / Anioł Páńſki niech ich goni.
Abowiem bez przycżyny záſtáwili ná mię w dole śieći ſwoje / y bez przycżyny ukopáli dół duƺy mojej.
Niecháj ná nich przyjdzie ſpuſtoƺenie / którego śię nie ſpodziewáli ; á śieć ich / którą záſtáwili / niech ich ułowi ná zginienie / á niech w nią wpádną.
Ale duƺá mojá niech śię rozráduje w Pánu / niech śię rozweſeli w zbáwieniu jego.
Tedy wƺyſtkie kośći moje rzeką : Pánie! któż podobny tobie? który wyrywáƺ utrápionego od mocniejƺego nádeń / á nędznego y ubogiego od drápieżcy jego.
Powſtáwáją świádkowie fáłƺywi / á o cżem nie wiem / pytáją mię.
Oddáwáją mi złem zá dobre / chcąc mię pozbáwić duƺy mojej /
Choćiáżem śię já w wór obłócżył / gdy oni chorowáli ; trápiłem poſtem duƺę moję / y modliłem śię cżęſto ſám u śiebie zá nimi.
Jáko do przyjáćielá / jáko do brátá mego / uſtáwicżniem chádzáł ; poniżáłem śię jáko ten / który śię ſmući / chodząc po mátce w żáłobie.
Lecż oni / gdym já chorowáł / weſelili śię / y zbieráli śię ; zbieráli śię przećiwko mnie / jákoby byli dlá mnie utrápieni / cżegom já nie ſpoſtrzegł ; ƺcżypáli mię / á nie milcżeli.
Z obłudnikámi / z náśmiewcámi / z pochlebcámi zgrzytáli ná mię zębámi ſwemi.
Pánie! długoż ná to pátrzeć będzieƺ? wyrwijże duƺę moję od zguby ich / od lwiąt jedynácżkę moję.
Będę ćię wyſłáwiáł w zgromádzeniu wielkiem ; między ludem wielkim będę ćię chwálił.
Niech śię nie weſelą ze mnie / którzy mi ſą nieprzyjáćiołmi bez przycżyny ; którzy mię máją w nienáwiśći nieſłuƺnie / niech nie mrugáją okiem.
Abowiem nie mówią o pokoju ; ále przećiwko ſpokojnym ná ziemi zdrádliwe ſłowá zmyśláją.
Owƺem / rozdzieráją ná mię gębę ſwą / mówiąc : Ehej! ehej! widzić to oko náƺe.
Widziƺ to / Pánie! nie milcżże Pánie! nie oddáláj śię odemnie.
Obudźże śię / á ocuć dlá ſądu mego / Boże mój y Pánie mój! dlá ſpráwy mojej.
Sądź mnie według ſpráwiedliwośći twojej / Pánie Boże mój! á niech śię nie weſelą nádemną.
Niech nie mówią w ſercu ſwojim : Ehej / duƺo máƺá! niech nie mówią : Pożárliśmy go.
Niechájże będą poháńbieni / y záwſtydzeni wƺyſcy weſelący śię ze złego mego ; niech będą oblecżeni w háńbę / y w ſromotę / którzy śię chlubią przećiwko mnie.
Ale ći / którzy śię kocháją w ſpráwiedliwośći mojej / niech śpiewáją y rádują śię / á niech mówią uſtáwicżnie : Niech będzie uwielbiony Pán / który życży pokoju ſłudze ſwemu.
A język mój będzie opowiádáł ſpráwiedliwość twoję / y ná káżdy dzień chwáłę twoję.
