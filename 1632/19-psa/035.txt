Pſálm Dawidowy. Rozpieráj śię / Pánie! z tymi / którzy śię ze mną ſpieráją ; á wálcż przećiwko tym / którzy wálcżą przećiwko mnie.
Porwij pukierz y tárcżę / á powſtáń ná rátunek mój.
Dobądź włócżni / á ſtáw śię ná drodze przećiwko tym / którzy mię prześládują. Rzecżże duƺy mojey : Jám jeſt zbáwieniem twojim.
Niech będą poháńbieni y záwſtydzeni / którzy ƺukáją duƺy mojey ; niech tył podádzą / y niech będą záwſtydzeni / którzy mi źle myślą.
Niech będą jáko plewy przed wiátrem / á Anioł Páńſki niecháj ich rozproƺy.
Niech będźie drogá ich ćiemná y śliſká / Anioł Páńſki niech ich goni.
Abowiem bez przycżyny záſtáwili ná mię w dole śieći ſwoje / y bez przycżyny ukopáli dół duƺy mojey.
Niecháj ná nich przyjdźie ſpuſtoƺenie / którego śię nie ſpodźiewáli ; á śieć ich / którą záſtáwili / niech ich ułowi ná zginienie / á niech w nią wpádną.
Ale duƺá mojá niech śię rozráduje w Pánu / niech śię rozweſeli w zbáwieniu jego.
Tedy wƺyſtkie kośći moje rzeką : Pánie! któż podobny tobie? który wyrywáƺ utrápionego od mocniejƺego nádeń / á nędznego y ubogiego od drápieżcy jego.
Powſtáwáją świádkowie fáłƺywi / á o cżem nie wiem / pytáją mię.
Oddáwáją mi złem zá dobre / chcąc mię pozbáwić duƺy mojey /
Choćiáżem śię ja w wór obłócżył / gdy oni chorowáli ; trápiłem poſtem duƺę moję / y modliłem śię cżęſto ſám u śiebie zá nimi.
Jáko do przyjáćielá / jáko do brátá mego / uſtáwicżniem chádzáł ; poniżáłem śię jáko ten / który śię ſmući / chodząc po mátce w żáłobie.
Lecż oni / gdym ja chorowáł / weſelili śię / y zbieráli śię ; zbieráli śię przećiwko mnie / jákoby byli dla mnie utrápieni / cżegom ja nie ſpoſtrzegł ; ƺcżypáli mię / á nie milcżeli.
Z obłudnikámi / z náśmiewcámi / z pochlebcámi zgrzytáli ná mię zębámi ſwemi.
Pánie! długoż ná to pátrzeć będźieƺ? wyrwijże duƺę moję od zguby ich / od lwiąt jedynácżkę moję.
Będę ćię wyſłáwiáł w zgromádzeniu wielkiem ; między ludem wielkim będę ćię chwálił.
Niech śię nie weſelą ze mnie / którzy mi ſą nieprzyjáćiołmi bez przycżyny ; którzy mię máją w nienáwiśći nieſłuƺnie / niech nie mrugáją okiem.
Abowiem nie mówią o pokoju ; ále przećiwko ſpokojnym ná źiemi zdrádliwe ſłowá zmyśláją.
Owƺem / rozdźieráją ná mię gębę ſwą / mówiąc : Ehej! ehej! widźić to oko náƺe.
Widźiƺ to / Pánie! nie milcżże Pánie! nie oddáláj śię odemnie.
Obudźże śię / á ocuć dla ſądu mego / Boże mój y Pánie mój! dla ſpráwy mojey.
Sądź mnie według ſpráwiedliwośći twojey / Pánie Boże mój! á niech śię nie weſelą nádemną.
Niech nie mówią w ſercu ſwojim : Ehej / duƺo máƺá! niech nie mówią : Pożárliśmy go.
Niechájże będą poháńbieni / y záwſtydzeni wƺyſcy weſelący śię ze złego mego ; niech będą oblecżeni w háńbę / y w ſromotę / którzy śię chlubią przećiwko mnie.
Ale ći / którzy śię kocháją w ſpráwiedliwośći mojey / niech śpiewáją y rádują śię / á niech mówią uſtáwicżnie : Niech będźie uwielbiony Pán / który życży pokoju ſłudze ſwemu.
A język mój będźie opowiádáł ſpráwiedliwość twoję / y ná káżdy dźień chwáłę twoję.
