Modlitwá utrápionego / gdy będąc w ućiſku / przed Pánem wylewá żądość ſwoje.
Pánie! wyſłucháj modlitwę moję / á wołánie moje niecháj przyjdźie do ćiebie.
Nie ukrywáj oblicżá twego przedemną ; w dźień ućiſku mego nákłoń ku mnie uchá twego ; w dźień którego ćię wzywám / prędko mię wyſłucháj.
Abowiem niƺcżeją jáko dym dni moje / á kośći moje jáko ogniſko wypálone ſą.
Poráżone jeſt jáko tráwá / y uwiędło ſerce moje / ták / żem zápomniáł jeść chlebá ſwego.
Od głoſu wzdychániá mego przylgnęły kośći moje do ćiáłá mego.
Stáłem śię podobnym pelikánowi ná puƺcży ; jeſtem jáko puhácż ná puſtyniách.
Cżuję / á jeſtem jáko wróbel ſámotny ná dáchu.
Przez cáły dźień urągáją mi nieprzyjáćiele moi / á náśmiewcy moi przeklináją mię.
Bo jádám popiół jáko chleb / á nápój mój mięƺám ze łzámi /
Dla rozgniewániá twego / y dla zápálcżywośći gniewu twego ; ábowiem podnióſłƺy mię porzućiłeś mię.
Dni moje ſą jáko ćień náchylony / á jám jáko tráwá uwiądł ;
Ale ty / Pánie! trwáƺ ná wieki / á pámiątká twojá od národu do národu.
Ty powſtáwƺy zmiłujeƺ śię nád Syonem ; boć cżáſ / żebyś śię zlitowáł nád nim / gdyż przyƺedł cżáſ náznácżony.
Abowiem upodobáły śię ſługom twoim kámienie jego / y nád prochem jego zmiłują śię ;
Aby śię báli pogánie imieniá Páńſkiego / á wƺyſcy królowie źiemſcy chwáły twojey ;
Gdy pobuduje Pán Syon / y okáże śię w chwále ſwojey ;
Gdy wejrzy ná modlitwę poniżonych / nie gárdząc modlitwą ich.
To zápiƺą dla národu potomnego / á lud / który má być ſtworzony / chwálić będźie Páná /
Że wejrzáł z wyſokośći świątnicy ſwojey / że z niebá ná źiemię ſpojrzáł ;
Aby wyſłucháł wzdychániá więźniów / y rozwiązáł ná śmierć ſkázánych ;
Aby opowiádáli ná Syonie imię Páńſkie / á chwáłę jego w Jeruzálemie /
Gdy śię poſpołu zgromádzą národy y króleſtwá / áby ſłużyły Pánu.
Utrápił w drodze śiłę moję / ukróćił dni moich ;
Ażem rzekł ; Boże mój! nie bierz mię w połowie dni moich ; od národu bowiem áż do národu trwáją látá twoje /
Y pierwej niżeliś záłożył źiemię / y niebioſá / dźieło rąk twoich.
One pominą / ále ty zoſtájeƺ ; wƺyſtkie te rzecży jáko ƺátá zwiotƺeją / jáko odźienie odmieniƺ je / y odmienione będą.
Ale ty tenżeś záwżdy jeſt / á látá twoje nigdy nie uſtáną.
Synowie ſług twoich u ćiebie mieƺkáć będą / á náśienie ich zmocni śię przed tobą.
