Modlitwá utrápionego / gdy będąc w ućiſku / przed Pánem wylewá żądość ſwoje.
Pánie! wyſłucháj modlitwę moję / á wołánie moje niecháj przydźie do ćiebie.
Nie ukrywáj oblicża twego przedemną ; w dźień ućiſku mego nákłoń ku mnie uchá twego ; w dźień którego ćię wzywám / prędko mię wyſłucháj.
Abowiem niƺcżeją jáko dym dni moje / á kośći moje jáko ogniſko wypálone ſą.
Poráżone jeſt jáko tráwá / y uwiędło ſerce moje / ták / żem zápomniał jeść chlebá ſwego.
Od głoſu wzdychánia mego przylgnęły kośći moje do ćiáłá mego.
Stáłem śię podobnym pelikánowi ná puƺcży ; jeſtem jáko puhácż ná puſtyniách.
Cżuję / á jeſtem jáko wróbel ſámotny ná dáchu.
Przez cáły dźień urągáją mi nieprzyjáćiele moi / á náśmiewcy moi przeklináją mię.
Bo jádám popiół jáko chleb / á nápój mój mięƺám ze łzámi /
Dla rozgniewánia twego / y dla zápálcżywośći gniewu twego ; ábowiem podnióſƺy mię porzućiłeś mię.
Dni moje ſą jáko ćień náchylony / á jam jáko tráwá uwiądł ;
Ale ty / Pánie! trwáƺ ná wieki / á pámiątká twojá od narodu do narodu.
Ty powſtáwƺy zmiłujeƺ śię nád Syonem ; boć cżás / żebyś śię zlitował nád nim / gdyż przyƺedł cżás náznácżony.
Abowiem upodobáły śię ſługom twoim kámienie jego / y nád prochem jego zmiłują śię ;
Aby śię báli pogánie imienia Páńſkiego / á wƺyſcy królowie źiemſcy chwały twojey ;
Gdy pobuduje Pán Syon / y okaże śię w chwále ſwojey ;
Gdy wejrzy ná modlitwę poniżonych / nie gárdząc modlitwą ich.
To zápiƺą dla narodu potomnego / á lud / który ma być ſtworzony / chwálić będźie Páná /
Że wejrzał z wyſokośći świątnicy ſwojey / że z niebá ná źiemię ſpojrzał ;
Aby wyſłuchał wzdychánia więźniów / y rozwiązał ná śmierć ſkázánych ;
Aby opowiádáli ná Syonie imię Páńſkie / á chwałę jego w Jeruzálemie /
Gdy śię poſpołu zgromádzą narody y króleſtwá / áby ſłużyły Pánu.
Utrápił w drodze śiłę moję / ukróćił dni mojich ;
Ażem rzekł ; Boże mój! nie bierz mię w połowie dni mojich ; od narodu bowiem áż do narodu trwáją látá twoje /
Y pierwej niżeliś záłożył źiemię / y niebioſá / dźieło rąk twojich.
One pominą / ále ty zoſtájeƺ ; wƺyſtkie te rzecży jáko ƺátá zwiotƺeją / jáko odźienie odmieniƺ je / y odmienione będą.
Ale ty tenżeś záwżdy jeſt / á látá twoje nigdy nie uſtáną.
Synowie ſług twojich u ćiebie mieƺkáć będą / á naśienie ich zmocni śię przed tobą.
