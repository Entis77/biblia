Wyſłáwiájćie Páná / ábowiem dobry ; ábowiem ná wieki trwá miłośierdzie jego ;
Rzecż teráz / Izráelu! że ná wieki miłośierdzie jego.
Rzecż teráz / domie Aáronowy! że ná wieki miłośierdzie jego.
Rzecżćież teráz / którzy śię boićie Páná / że ná wieki miłośierdzie jego.
W ućiſku wzywáłem Páná ; wyſłucháł mię / y ná przeſtrzeńſtwie poſtáwił mię Pán.
Pán jeſt zemną / nie będę śię báł / żeby mi co ucżynił cżłowiek.
Pán jeſt zemną między pomocnikámi mymi ; przetoż já oglądám pomſtę nád tymi / którzy mię máją w nienáwiśći.
Lepiej mieć nádzieję w Pánu / niżeli ufáć w cżłowieku.
Lepiej mieć nádzieję w Pánu / niżeli ufáć w kśiążętách.
Wƺyſtkie národy ogárnęły mię ; ále w imieniu Páńſkim wygubiłem ich.
Cżęſtokroć mię ogárnęły ; ále w imieniu Páńſkim wygubiłem ich.
Ogárnęły mię jáko pƺcżoły / ále zgáſły jáko ogień z ćierniá ; bo w imieniu Páńſkim wytráćiłem ich.
Bárdzoś potężnie ná mię náćieráł / ábym upádł ; ále Pán porátowáł mię.
Pán jeſt mocą moją / y pieśnią moją ; on był moim wybáwićielem.
Głoſ wykrzykániá y zbáwieniá w przybytkách ſpráwiedliwych / práwicá Páńſká dokázáłá mocy ;
Práwicá Páńſká wywyżƺyłá śię ; práwicá Páńſká dokázáłá mocy.
Nie umrę / ále będę żył / ábym opowiádáł ſpráwy Páńſkie.
Pokáráłći mię Pán ſrodze ; ále mię ná śmierć nie podáł.
Otwórzćie mi brámy ſpráwiedliwośći / á wƺedłƺy w nie będę wyſłáwiáł Páná.
Táć jeſt brámá Páńſká / którą ſpráwiedliwi wchodzą.
Tuć já ćiebie wyſłáwiáć będę ; boś mię wyſłucháł / y byłeś wybáwićielem moim.
Kámień / którzy odrzućili budujący / ucżyniony jeſt głową węgielną.
Od Páná śię to ſtáło / á jeſt dziwno w ocżách náƺych.
Tenći to dzień / który ucżynił Pán ; rozweſelmyż śię / á rozrádujmy śię weń.
Proƺę / Pánie! záchowájże teráz ; proƺę Pánie! zdárz teráz.
Błogoſłáwiony / który przychodzi w imię Páńſkie ; błogoſłáwimy wám z domu Páńſkiego.
Bógći Pánem / onći náſ oświećił ; przywiążćie báránki powrozámi ku ofierze áż do rogów ołtárzá.
Tyś jeſt Bóg mój ; przetoż ćię wyſłáwiáć będę / Boże mój! wywyżƺáć ćię będę.
Wyſłáwiájćież Páná / ábowiem jeſt dobry : ábowiem ná wieki miłośierdzie jego.
