Hállelujá. Chwálćie Páná ná niebioſách ; chwálćież go ná wyſokośćiách.
Chwálćie go wƺyſcy Aniołowie jego ; chwálćie go wƺyſtkie wojſká jego.
Chwálćie go ſłońce y mieśiącu ; chwálćie go wƺyſtkie jáſne gwiázdy.
Chwálćie go niebioſá nád niebioſámi / y wody / które ſą nád niebem.
Chwálćie imię Páńſkie ; ábowiem on rozkázáł / á ſtworzone ſą.
Y wyſtáwił je ná wieki wiecżne ; záłożył im kreſ / którego nie przeſtępują.
Chwálćie Páná ná źiemi / ſmoki y wƺyſtkie przepáśći.
Ogień y grád / śnieg y párá / wiátr gwáłtowny / wykonywájący rozkáz jego ;
Góry / y wƺyſtkie págórki / drzewá rodzájne / y wƺyſtkie cedry ;
Zwierzętá / y wƺyſtko bydło / gádźiny / y ptáſtwo ſkrzydláſte.
Królowie źiemſcy / y wƺyſtkie národy ; kśiążętá y wƺyſcy ſędźiowie źiemi ;
Młodźieńcy / tákże y pánny / ſtárzy y młodźi /
Chwálćie imię Páńſkie ; ábowiem wywyżƺone jeſt imię jego ſámego / á chwáłá jego nád źiemią y niebem.
Y wywyżƺył róg ludu ſwego / chwáłę wƺyſtkich świętych jego / miánowićie ſynów Izráelſkich / ludu jemu nájbliżƺego. Hállelujá.
