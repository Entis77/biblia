Przedniejƺemu śpiewákowi Jedytunowi pſálm Dawidowy.
Rzekłem : Będę ſtrzegł dróg mojich / ábym nie zgrzeƺył językiem ſwym ; włożę munƺtuk w uſtá moje / póki niepobożny będźie przedemną.
Zániemiáłem milcżąc ; zámilknąłem y w dobrej ſpráwie ; ále boleść mojá bárdźiej śię wzmágáłá.
Rozpáliło śię ſerce moje we wnętrznośćiách mojich ; w rozmyślániu mojim rozżárzył śię ogień / áżem ták rzekł językiem ſwoim :
Dáj mi poznáć / Pánie! dokońcżenie moje / y wymiár dni mojich jáki jeſt / ábym wiedźiał / ják długo trwáć będę.
Otoś ná dłoni wymierzył dni moje / á wiek mój jeſt jáko nic przed tobą ; zápráwdę ƺcżerą márnośćią jeſt wƺelki cżłowiek / choć nájdużƺy. Selá.
Zápráwdę pomijá cżłowiek jáko ćień ; zápráwdę prózno śię kłopocże / zgromádzá / á nie wie / kto to pobierze.
A teraz ná cóż ocżekuję / Pánie? Tyś jeſt ſám ocżekiwániem mojim.
Przetoż od wƺyſtkich przeſtępſtw mojich wybáw mię ; ná pośmiech głupiemu nie dáwáj mię.
Zániemiáłem / y nie otworzyłem uſt mojich / przeto / żeś to ty ucżynił.
Odejmij odemnie káránie twoje ; bom od ſmágánia ręki twojey uſtał.
Gdy ty gromiąc kárzeƺ cżłowieká dla niepráwośći / wnet niƺcżyƺ jáko mól grzecżność jego ; záiſteć márnośćią jeſt wƺelki cżłowiek. Selá.
Wyſłuchájże modlitwę moję / Pánie! á wołánie moje przyjmij w uƺy ſwoje / nie milcż ná łzy moje ; bomći ja przychodniem u ćiebie / y komornikiem / jáko wƺyſcy ojcowie moi.
Sfolguj mi / ábym śię pośilił / pierwej niżeli odejdę / á nie będźie mię.
