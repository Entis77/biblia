Przedniejƺemu śpiewákowi Jedytunowi pſálm Dawidowy.
Rzekłem : Będę ſtrzegł dróg moich / ábym nie zgrzeƺył językiem ſwym ; włożę munƺtuk w uſtá moje / póki niepobożny będzie przedemną.
Zániemiáłem milcżąc ; zámilknąłem y w dobrej ſpráwie ; ále boleść mojá bárdziej śię wzmágáłá.
Rozpáliło śię ſerce moje we wnętrznośćiách moich ; w rozmyślániu mojem rozżárzył śię ogień / áżem ták rzekł językiem ſwoim :
Dáj mi poznáć / Pánie! dokońcżenie moje / y wymiár dni moich jáki jeſt / ábym wiedziáł / ják długo trwáć będę.
Otoś ná dłoni wymierzył dni moje / á wiek mój jeſt jáko nic przed tobą ; zápráwdę ƺcżerą márnośćią jeſt wƺelki cżłowiek / choć nájdużƺy. Selá.
Zápráwdę pomijá cżłowiek jáko ćień ; zápráwdę próżno śię kłopocże / zgromádzá / á nie wie / kto to pobierze.
A teráz ná cóż ocżekuję / Pánie? Tyś jeſt ſám ocżekiwániem mojem.
Przetoż od wƺyſtkich przeſtępſtw moich wybáw mię ; ná pośmiech głupiemu nie dáwáj mię.
Zániemiáłem / y nie otworzyłem uſt moich / przeto / żeś to ty ucżynił.
Odejmij odemnie káránie twoje ; bom od ſmágániá ręki twojej uſtáł.
Gdy ty gromiąc kárzeƺ cżłowieká dlá niepráwośći / wnet niƺcżyƺ jáko mól grzecżność jego ; záiſteć márnośćią jeſt wƺelki cżłowiek. Selá.
Wyſłuchájże modlitwę moję / Pánie! á wołánie moje przyjmij w uƺy ſwoje / nie milcż ná łzy moje ; bomći já przychodniem u ćiebie / y komornikiem / jáko wƺyſcy ojcowie moi.
Sfolguj mi / ábym śię pośilił / pierwej niżeli odejdę / á nie będzie mię.
