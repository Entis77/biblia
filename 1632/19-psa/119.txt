Błogoſłáwieni / którzy żyją bez nágány / którzy chodzą w zákonie Páńſkim /
Błogoſłáwieni / którzy ſtrzegą świádectw jego / y którzy go ze wƺyſtkiego ſercá ƺukáją ;
Y którzy nie cżynią niepráwośći / ále chodzą drogámi jego.
Tyś przykázáł / áby pilnie ſtrzeżono rozkázáń twoich.
Oby wyproſtowáne były drogi moje ku przeſtrzegániu práw twoich!
Tedy nie będę záwſtydzony / gdy śię będę oglądáł ná wƺyſtkie rozkázániá twoje.
Będę ćię wyſłáwiáł w ƺcżerośći ſercá / gdy śię náucżę práw ſpráwiedliwośći twojej.
Uſtáw twoich z pilnośćią ſtrzedz będę ; tylko mię nie opuƺcżáj.
Jákim ſpoſobem ocżyśći młodzieniec śćieƺkę ſwoję? Gdy śię záchowá według ſłowá twego.
Ze wƺyſtkiego ſercá mego ƺukám ćię ; nie dopuƺcżájże mi błądzić od rozkázáń twoich.
W ſercu mojem ſkłádám wyroki twoje / ábym nie zgrzeƺył przećiwko tobie.
Błogoſłáwionyś ty / Pánie! náucżże mię uſtáw twoich.
Wárgámi mojemi opowiádám wƺyſtkie ſądy uſt twoich.
W drodze świádectw twoich kochám śię więcej / niż we wƺyſtkich bogáctwách.
O przykázániách twoich rozmyślám / y przypátruję śię drogom twoim.
W uſtáwách twoich kochám śię / y nie zápominám ſłów twoich.
Dáruj to ſłudze twemu / ábym żył / á przeſtrzegáł ſłów twoich.
Odſłoń ocży moje / ábym śię przypátrzył dziwom z zákonu twego.
Jeſtem gośćiem ná ziemi ; nie ukrywáj przedemną rozkázáń twoich.
Omdlewá duƺá mojá / prágnąc ſądów twoich ná káżdy cżáſ.
Wytráćiłeś pyƺnych ; przeklęći ſą ći / którzy błądzą od rozkázáń twoich.
Oddál odemnie poháńbienie y wzgárdę / gdyż ſtrzegę świádectw twoich.
Y kśiążętá záśiádáją / á mówią przećiwko mnie ; wƺákże ſługá twój rozmyślá w uſtáwách twoich.
Świádectwá twoje záiſte ſą mojem kochániem / y rádcámi mymi.
Przylgnęłá do prochu duƺá mojá ; ożywże mię według ſłowá twego.
Drogi moje rozpowiedziáłem / á wyſłucháłeś mię ; náucż mię uſtáw twoich.
Dáj / ábym zrozumiáł drogę rozkázáń twoich / áżebym rozmyśláł o dziwnych ſpráwách twoich.
Rozpływá śię od ſmutku duƺá mojá ; utwierdźże mię według ſłowá twego.
Drogę kłámliwą oddál odemnie / á zákonem twoim udáruj mię.
Obráłem drogę práwdy / á ſądy twoje przekłádám ſobie.
Przyſtáłem do świádectw twoich ; Pánie! nie záwſtydzájże mię.
Drogą przykázáń twoich pobieżę / gdy rozƺerzyƺ ſerce moje.
Náucż mię / Pánie! drogi uſtáw twoich / á będę jej ſtrzegł áż do końcá.
Dáj mi rozum / ábym ſtrzegł zákonu twego / áżebym go przeſtrzegáł ze wƺyſtkiego ſercá.
Dáj / ábym chodził śćieƺką przykázáń twoich / gdyż w tem jeſt upodobánie moje.
Nákłoń ſerce moje do świádectw twoich / á nie do łákomſtwá.
Odwróć ocży moje / áby nie pátrzáły ná márność ; ná drodze twojej ożyw mię.
Utwierdź wyrok twój ſłudze twemu / który śię oddáł bojáźni twojej.
Oddál odemnie poháńbienie moje / którego śię boję ; bo ſądy twoje dobre.
Oto prágnę rozkázáń twoich ; w ſpráwiedliwośći twojej ożyw mię.
Niech ná mię przyjdą litośći twoje / Pánie! y zbáwienie twoje według wyroku twego.
Ták ábym odpowiedź mógł dáć ſámą rzecżą temu / który mi urągá / gdyż ufám w ſłowie twojim.
A nie wyjmuj z uſt moich ſłowá nájpráwdziwƺego ; ábowiem ſądów twoich ocżekuję.
Y będę ſtrzegł zákonu twego záwƺe / áż ná wieki wiecżne.
A uſtáwicżnie będę chodził ná przeſtrzeńſtwie / bom śię dopytáł rozkázáń twoich.
Owƺem / będę mówił o świádectwách twoich przed królmi / á nie będę záwſtydzony.
Bom śię rozkocháł w przykázániách twoich / którem umiłowáł.
Przyłożę y ręce moje do rozkázáń twoich / które miłuję / á będę rozmyśláł o uſtáwách twoich.
Wſpomnij ná ſłowo wyrzecżone do ſługi twego / któremeś mię ubezpiecżył.
Toć poćiechá mojá w utrápieniu mojem / że mię wyrok twój ożywiá.
Pyƺni bárdzo śię ze mnie náśmiewáją ; wƺákże śię od zákonu twego nie uchylám.
Bo pámiętám ná ſądy twoje wiecżne / Pánie! któremi śię ćieƺę.
Strách mię ogárnął nád niezbożnymi / którzy opuƺcżáją zákon twój.
Są mi uſtáwy twoje pieśniámi w domu pielgrzymſtwá mego.
Wſpominám ſobie y w nocy ná imię twoje / Pánie! y ſtrzegę zákonu twego.
Toć mám z tego / że przeſtrzegám przykázáń twoich.
Rzekłem : Pánie! to jeſt cżąſtká mojá / przeſtrzegáć ſłów twoich.
Modlę śię przed oblicżem twojim ze wƺyſtkiego ſercá ; zmiłujże śię nádemną według ſłowá twego.
Uwáżyłem w myślách drogi moje / á obróćiłem nogi moje ku świádectwom twoim.
Spieƺę śię / á nie omieƺkuję przeſtrzegáć rozkázáń twoich.
Hufy niepobożnych złupiły mię ; ále ná zákon twój nie zápominám.
O północy wſtáję / ábym ćię wyſłáwiáł w ſądách ſpráwiedliwośći twojej.
Jeſtem towárzyƺem wƺyſtkich / którzy śię ćiebie boją / y tych / którzy przeſtrzegáją przykázáń twoich.
Pánie! pełná jeſt ziemiá miłośierdziá twego ; náucżże mię uſtáw twoich.
Łáſkáwieś poſtąpił ze ſługą twoim / Pánie! według ſłowá twego.
Dobrego rozumu y umiejętnośći náucż mię ; bom przykázániom twoim uwierzył.
Pierwej niżem śię był uniżył / błądziłem ; ále teráz wyroku twego przeſtrzegám.
Dobryś ty y dobrotliwy ; náucżże mię uſtáw twoich.
Uknowáli hárdzi kłámſtwo przećiwko mnie ; ále já ze wƺyſtkiego ſercá ſtrzegę przykázáń twoich.
Serce ich zátyło jáko ſádło ; ále śię já zákonem twoim ćieƺę.
Jeſt mi to ku dobremu / żem był utrápiony / ábym śię náucżył uſtáw twoich.
Lepƺy mi jeſt zákon uſt twoich / niżeli tyśiące złotá y ſrebrá.
Ręce twoje ucżyniły mię / y wykƺtáłtowáły mię ; dájże mi rozum / ábym śię náucżył przykázáń twoich ;
Aby śię rádowáli bojący śię ćiebie / ujrzáwƺy mię / że ná ſłowo twoje ocżekuję.
Znám / Pánie! iż ſą ſpráwiedliwe ſądy twoje / á iżeś mię ſłuƺnie utrápił.
Niechájże mię / proƺę / ućieƺy miłośierdzie twoje według wyroku twego / któryś ucżynił ſłudze twemu.
Niechże ná mię przyjdą litośći twoje / ábym żył ; bo zákon twój jeſt kochániem mojem.
Niech będą záwſtydzeni pyƺni / przeto / że mię chytrze podwróćić chćieli ; ále já rozmyśláć będę w przykázániách twoich.
Niech śię obrócą do mnie / którzy śię ćiebie boją / y którzy znáją świádectwá twoje.
Niech będzie ſerce moje uprzejme przy uſtáwách twoich / ábym nie był záwſtydzony.
Tęſkni duƺá mojá po zbáwieniu twojim / ocżekuję ná ſłowo twoje.
Uſtáły ocży moje / cżekájąc wyroku twego / gdy mówię : Kiedyż mię poćieƺyƺ?
Choćiáżem jeſt jáko nácżynie ſkórzáne w dymie / wƺákżem uſtáw twoich nie zápomniáł.
Wieleż będzie dni ſługi twego? kiedyż ſąd wykonáƺ nád tymi / którzy mię prześládują?
Pyƺni pokopáli mi doły / co nie jeſt według zákonu twojego.
Wƺyſtkie przykázániá twoje ſą práwdą ; bez przycżyny mię prześládują ; rátujże mię.
Bez máłá mię już wniwecż nie obróćili ná ziemi ; á wƺákżem já nie opuśćił przykázáń twoich.
Według miłośierdziá twego ożyw mię / ábym ſtrzegł świádectwá uſt twoich.
O Pánie! ſłowo twoje trwá ná wieki ná niebie.
Od národu do národu práwdá twojá ; ugruntowáłeś ziemię / y ſtoi.
Według rozrządzeniá twego trwá to wƺyſtko áż do dniá tego ; wƺyſtko to záiſte jeſt ku ſłużbie twojej.
By był zákon twój nie był kochániem mojem / dáwnobym był zginął w utrápieniu mojem.
Ná wieki nie zápomnę ná przykázániá twoje / gdyżeś mię w nich ożywił.
Twójćim já / záchowájże mię ; bo przykázáń twoich ƺukám.
Cżekáją ná mię niezbożnicy / áby mię zátráćili ; ále já świádectwá twoje uwáżám.
Wƺelkiej rzecży koniec widzę ; ále przykázánie twoje bárdzo ƺerokie.
O jákom śię rozmiłowáł zákonu twego! ták / iż káżdego dniá jeſt rozmyślániem mojem.
Nád nieprzyjáćiół moich mędrƺym mię cżyniƺ przykázániem twojim ; bo je mám uſtáwicżnie przed ſobą.
Nád wƺyſtkich náucżyćieli moich ſtáłem śię rozumniejƺym ; bo świádectwá twoje ſą rozmyślániem mojem.
Nád ſtárców jeſtem roztropniejƺy ; bo przykázáń twoich przeſtrzegám.
Od wƺelkiej złej drogi záwśćiągám nogi ſwoje / ábym ſtrzegł ſłowá twego.
Od ſądów twoich nie odſtępuję / przeto / że ich ty mnie ucżyƺ.
O ják ſą ſłodkie ſłowá twoje podniebieniu memu! nád miód ſą ſłodƺe uſtom moim.
Z przykázáń twoich nábyłem rozumu : przetoż mám w nienáwiśći wƺelką śćieƺkę obłędliwą.
Słowo twe jeſt pochodnią nogą moim / á świátłośćią śćieƺce mojej.
Przyśięgłem y ucżynię temu doſyć / że będę ſtrzegł ſądów ſpráwiedliwośći twojej.
Jeſtem bárdzo utrápiony ; o Pánie! ożyw mię według ſłowá twego.
Pánie! dobrowolne śluby uſt moich przyjmij proƺę zá wdzięcżne / á ſądów twoich náucż mię.
Duƺá mojá jeſt w uſtáwicżnem niebezpiecżeńſtwie ; wƺákże ná zákon twój nie zápominám.
Sidło ná mię niezbożnicy záſtáwili ; lecż já śię od przykázáń twoich nie obłądzę.
Zá dziedzictwo wiecżne wziąłem świádectwá twoje ; bo ſą rádośćią ſercá mego.
Nákłoniłem ſercá mego ku wykonywániu uſtáw twoich uſtáwicżnie / y áż do końcá ( żywotá ).
Wymyſły mám w nienáwiśći / á zákon twój miłuję.
Tyś jeſt ućiecżką moją / y tárcżą moją ; ná ſłowo twoje ocżekuję.
Odſtąpćież odemnie złośnicy / ábym ſtrzegł rozkázániá Bogá mojego.
Utwierdźże mię według ſłowá twego / ábym żył / á nie záwſtydzáj mię w ocżekiwániu mojem.
Podpieráj mię / ábym był záchowány / y rozmyśláł w uſtáwách twoich uſtáwicżnie.
Podeptáłeś wƺyſtkich / którzy śię obłądzili od uſtáw twoich ; ábowiem jeſt kłámliwá zdrádá ich.
Odrzucáƺ jáko zużelicę wƺyſtkich niezbożników ziemi ; dlá tego miłuję świádectwá twoje.
Drży od ſtráchu przed tobą ćiáło moje ; bo śię ſądów twoich lękám.
Cżynię ſądy y ſpráwiedliwość : nie podáwájże mię tym / którzy mi gwáłt cżynią.
Záſtąp ſám ſługę twego ku dobremu / áby mię hárdzi nie potłocżyli.
Ocży moje uſtáły / cżekájąc ná zbáwienie twoje / y ná wyrok ſpráwiedliwośći twojej.
Obchodź śię z ſługą twoim według miłośierdziá twego / á uſtáw twoich náucż mię.
Sługámći já twój / dájże mi zrozumienie ; ábym umiáł świádectwá twoje.
Cżáſći już / ábyś cżynił Pánie! ábowiem wzruƺono zákon twój.
Dlátego umiłowáłem rozkázániá twoje nád złoto / á nád złoto nájwyborniejƺe.
Przeto / że wƺyſtkie przykázániá twoje / wƺyſtkie práwdziwe być uznáję / á wƺelkie śćieƺki obłędliwe mám w nienáwiśći.
Dziwne ſą świádectwá twoje ; przetoż ich ſtrzeże duƺá mojá.
Pocżątek ſłów twoich oświecá y dáje rozum proſtákom.
Uſtá moje otwierám y dyƺę ; ábowiemem przykázáń twoich prágnął.
Wejrzyjże ná mię / á zmiłuj śię nádemną według práwá tych / którzy miłują imię twoje.
Drogi moje utwierdź w ſłowie twojim / á niech nádemną nie pánuje żádná niepráwość.
Wybáw mię od ućiśnieniá ludzkiego / ábym ſtrzegł rozkázáń twoich.
Rozświeć nád ſługą twoim oblicże twoje / á náucż mię uſtáw twoich.
Strumienie wód płyną z ocżów moich dlá tych / którzy nie ſtrzegli zákonu twego.
Spráwiedliwyś ty / Pánie! y práwdziwy w ſądách twoich.
Przykázáłeś ſpráwiedliwe świádectwá twoje / y wielce práwdziwe.
Zniƺcżyłá mię gorliwość mojá / iż zápomináją ná ſłowo twoje nieprzyjáćiele moi.
Doſkonále ſą doświádcżone ſłowá twoje ; dlátego śię ſługá twój w nich rozkocháł.
Jám málucżki y wzgárdzony ; wƺákże przykázáń twoich nie zápominám.
Spráwiedliwość twojá ſpráwiedliwość wiecżná / á zákon twój práwdá.
Ućiſk y utrápienie przyƺło ná mię ; przykázániá twoje ſą kochániem mojem.
Spráwiedliwość świádectw twoich trwá ná wieki ; dáj mi rozum / á żyć będę.
Wołám ze wƺyſtkiego ſercá / wyſłuchájże mię / o Pánie! á będę ſtrzegł uſtáw twoich.
Wołám do ćiebie / záchowájże mię / á będę pilen świádectw twoich.
Uprzedzám ćię ná świtániu y wołám / ná ſłowo twoje ocżekując.
Uprzedzáją ſtráż nocną ocży moje / przeto / ábym rozmyśláł o wyrokách twoich.
Pánie! głoſ mój uſłyƺ według miłośierdziá twego ; według ſądu twego ożyw mię.
Przybliżáją śię / którzy náśládują złośći / ći / którzy śię od zákonu twego oddálili.
Bliſkoś ty jeſt / Pánie! á wƺyſtkie przykázániá twoje ſą práwdą.
Dáwno to wiem o świádectwách twoich / żeś je ná wieki ugruntowáł.
Obácż utrápienie moje / á wyrwij mię ; bom ná zákon twój nie zápomniáł.
Stáń przy ſpráwie mojej / á obroń mię ; dlá ſłowá twego ożyw mię.
Dálekoć jeſt od niezbożników zbáwienie ; bo śię nie bádáją o uſtáwách twoich.
Wielkie ſą litośći twoje / Pánie! według ſądów twoich ożyw mię.
Wieleć jeſt prześládowców moich y nieprzyjáćiół moich ; wƺákże od świádectw twoich nie uchylám śię.
Widziáłem przeſtępców / y mierziáło mię to / że wyroku twego nie przeſtrzegáli.
Obácżże Pánie! iż rozkázániá twoje miłuję ; według miłośierdziá twego ożyw mię.
Nájprzedniejƺá rzecż ſłowá twego jeſt práwdá / á ná wieki trwá wƺelki ſąd ſpráwiedliwośći twojej.
Kśiążętá mię prześládują bez przycżyny ; wƺákże ſłów twoich boi śię ſerce moje.
Já śię weſelę z wyroku twego / ták jáko ten / który znájduje wielkie korzyśći.
Ale nienáwidzę kłámſtwá / y brzydzę śię nim ; ále zákon twój miłuję.
Chwálę ćię śiedm kroć przez dzień / dlá ſądów twoich ſpráwiedliwych.
Pokój wielki dájeƺ tym / którzy miłują zákon twój / á nie doznáwáją żádnego obráżeniá.
Pánie! ocżekuję zbáwieniá twego ; á przykázániá twoje wykonywám.
Przeſtrzegá duƺá mojá świádectw twoich ; ábowiem je bárdzo miłuję.
Przeſtrzegám przykázáń twoich y świádectw twoich ; ábowiem wƺyſtkie drogi moje ſą przed tobą.
Pánie! niech śię przybliży wołánie moje przed oblicże twoje ; według ſłowá twego dáj mi zrozumienie.
Niech przyjdzie prośbá mojá przed twárz twoję / á według obietnicy twojej wyrwij mię.
Chwáłę wydádzą wárgi moje / gdy mię náucżyƺ uſtáw twoich.
Opowiádáć będzie język mój wyroki twoje ; bo wƺyſtkie przykázániá twoje ſą ſpráwiedliwość.
Niech mi będzie ná pomocy ręká twojá / gdyżem ſobie obráł przykázániá twoje.
Pánie! zbáwieniá twego prágnę / á zákon twój jeſt kochániem mojem.
Żyć będzie duƺá mojá / y będzie ćię chwáliłá / á ſądy twoje będą mi ná pomocy.
Błądzę jáko owcá zgubioná / ƺukájże ſługi twego ; boć przykázáń twoich nie zápominám.
