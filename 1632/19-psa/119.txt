Błogoſłáwieni / którzy żyją bez nágány / którzy chodzą w zakonie Páńſkim /
Błogoſłáwieni / którzy ſtrzegą świádectw jego / y którzy go ze wƺyſtkiego ſercá ƺukáją ;
Y którzy nie cżynią niepráwośći / ále chodzą drogámi jego.
Tyś przykázał / áby pilnie ſtrzeżono rozkázáń twojich.
Oby wyproſtowáne były drogi moje ku przeſtrzegániu práw twojich!
Tedy nie będę záwſtydzony / gdy śię będę oglądał ná wƺyſtkie rozkázánia twoje.
Będę ćię wyſłáwiał w ƺcżerośći ſercá / gdy śię náucżę práw ſpráwiedliwośći twojey.
Uſtáw twojich z pilnośćią ſtrzedz będę ; tylko mię nie opuƺcżáj.
Jákim ſpoſobem ocżyśći młodźieniec śćieƺkę ſwoję? Gdy śię záchowá według ſłowá twego.
Ze wƺyſtkiego ſercá mego ƺukám ćię ; nie dopuƺcżájże mi błądźić od rozkázáń twojich.
W ſercu mojim ſkłádám wyroki twoje / ábym nie zgrzeƺył przećiwko tobie.
Błogoſłáwionyś ty / Pánie! náucżże mię uſtáw twojich.
Wárgámi mojemi opowiádám wƺyſtkie ſądy uſt twojich.
W drodze świádectw twojich kochám śię więcey / niż we wƺyſtkich bogáctwách.
O przykázániách twojich rozmyślám / y przypátruję śię drogom twoim.
W uſtáwách twojich kochám śię / y nie zápominam ſłów twojich.
Dáruj to ſłudze twemu / ábym żył / á przeſtrzegał ſłów twojich.
Odſłoń ocży moje / ábym śię przypátrzył dźiwom z zakonu twego.
Jeſtem gośćiem ná źiemi ; nie ukrywáj przedemną rozkázáń twojich.
Omdlewá duƺá mojá / prágnąc ſądów twojich ná káżdy cżás.
Wytráćiłeś pyƺnych ; przeklęći ſą ći / którzy błądzą od rozkázáń twojich.
Oddál odemnie poháńbienie y wzgárdę / gdyż ſtrzegę świádectw twojich.
Y kśiążętá záśiádáją / á mówią przećiwko mnie ; wƺákże ſługá twój rozmyślá w uſtáwách twojich.
Świádectwá twoje záiſte ſą mojim kochániem / y rádcámi mymi.
Przylgnęłá do prochu duƺá mojá ; ożywże mię według ſłowá twego.
Drogi moje rozpowiedźiáłem / á wyſłucháłeś mię ; náucż mię uſtáw twojich.
Dáj / ábym zrozumiał drogę rozkázáń twojich / áżebym rozmyślał o dźiwnych ſpráwách twojich.
Rozpływá śię od ſmutku duƺá mojá ; utwierdźże mię według ſłowá twego.
Drogę kłámliwą oddál odemnie / á zakonem twoim udáruj mię.
Obráłem drogę prawdy / á ſądy twoje przekłádám ſobie.
Przyſtáłem do świádectw twojich ; Pánie! nie záwſtydzájże mię.
Drogą przykázáń twojich pobieżę / gdy rozƺerzyƺ ſerce moje.
Náucż mię / Pánie! drogi uſtáw twojich / á będę jey ſtrzegł áż do końcá.
Dáj mi rozum / ábym ſtrzegł zakonu twego / áżebym go przeſtrzegał ze wƺyſtkiego ſercá.
Dáj / ábym chodźił śćieƺką przykázáń twojich / gdyż w tym jeſt upodobánie moje.
Nákłoń ſerce moje do świádectw twojich / á nie do łákomſtwá.
Odwróć ocży moje / áby nie pátrzáły ná márność ; ná drodze twojey ożyw mię.
Utwierdź wyrok twój ſłudze twemu / który śię oddał bojáźni twojey.
Oddál odemnie poháńbienie moje / którego śię boję ; bo ſądy twoje dobre.
Oto prágnę rozkázáń twojich ; w ſpráwiedliwośći twojey ożyw mię.
Niech ná mię przyjdą litośći twoje / Pánie! y zbáwienie twoje według wyroku twego.
Ták ábym odpowiedź mógł dáć ſámą rzecżą temu / który mi urągá / gdyż ufám w ſłowie twojim.
A nie wyjmuj z uſt mojich ſłowá nájpráwdźiwƺego ; ábowiem ſądów twojich ocżekuję.
Y będę ſtrzegł zakonu twego záwƺe / áż ná wieki wiecżne.
A uſtáwicżnie będę chodźił ná przeſtrzeńſtwie / bom śię dopytał rozkázáń twojich.
Owƺem / będę mówił o świádectwách twojich przed królmi / á nie będę záwſtydzony.
Bom śię rozkochał w przykázániách twojich / którem umiłował.
Przyłożę y ręce moje do rozkázáń twojich / które miłuję / á będę rozmyślał o uſtáwách twojich.
Wſpomnij ná ſłowo wyrzecżone do ſługi twego / któremeś mię ubezpiecżył.
Toć poćiechá mojá w utrápieniu mojim / że mię wyrok twój ożywiá.
Pyƺni bárzo śię ze mnie náśmiewáją ; wƺákże śię od zakonu twego nie uchylám.
Bo pámiętám ná ſądy twoje wiecżne / Pánie! któremi śię ćieƺę.
Strách mię ogárnął nád niezbożnymi / którzy opuƺcżáją zakon twój.
Są mi uſtáwy twoje pieśniámi w domu pielgrzymſtwá mego.
Wſpominam ſobie y w nocy ná imię twoje / Pánie! y ſtrzegę zakonu twego.
Toć mam z tego / że przeſtrzegám przykázáń twojich.
Rzekłem : Pánie! to jeſt cżąſtká mojá / przeſtrzegáć ſłów twojich.
Modlę śię przed oblicżem twojim ze wƺyſtkiego ſercá ; zmiłujże śię nádemną według ſłowá twego.
Uwáżyłem w myślách drogi moje / á obróćiłem nogi moje ku świádectwom twoim.
Spieƺę śię / á nie omieƺkuję przeſtrzegáć rozkázáń twojich.
Hufy niepobożnych złupiły mię ; ále ná zakon twój nie zápominam.
O północy wſtáję / ábym ćię wyſłáwiał w ſądách ſpráwiedliwośći twojey.
Jeſtem towárzyƺem wƺyſtkich / którzy śię ćiebie boją / y tych / którzy przeſtrzegáją przykázáń twojich.
Pánie! pełna jeſt źiemiá miłośierdźiá twego ; náucżże mię uſtáw twojich.
Łáſkáwieś poſtąpił ze ſługą twoim / Pánie! według ſłowá twego.
Dobrego rozumu y umiejętnośći náucż mię ; bom przykázániom twoim uwierzył.
Pierwej niżem śię był uniżył / błądźiłem ; ále teraz wyroku twego przeſtrzegám.
Dobryś ty y dobrotliwy ; náucżże mię uſtáw twojich.
Uknowáli hárdźi kłámſtwo przećiwko mnie ; ále ja ze wƺyſtkiego ſercá ſtrzegę przykázáń twojich.
Serce ich zátyło jáko ſádło ; ále śię ja zakonem twoim ćieƺę.
Jeſt mi to ku dobremu / żem był utrápiony / ábym śię náucżył uſtáw twojich.
Lepƺy mi jeſt zakon uſt twojich / niżeli tyśiące złotá y ſrebrá.
Ręce twoje ucżyniły mię / y wykƺtáłtowáły mię ; dájże mi rozum / ábym śię náucżył przykázáń twojich ;
Aby śię rádowáli bojący śię ćiebie / ujrzáwƺy mię / że ná ſłowo twoje ocżekuję.
Znam / Pánie! iż ſą ſpráwiedliwe ſądy twoje / á iżeś mię ſłuƺnie utrápił.
Niechájże mię / proƺę / ućieƺy miłośierdźie twoje według wyroku twego / któryś ucżynił ſłudze twemu.
Niechże ná mię przyjdą litośći twoje / ábym żył ; bo zakon twój jeſt kochániem mojim.
Niech będą záwſtydzeni pyƺni / przeto / że mię chytrze podwróćić chćieli ; ále ja rozmyśláć będę w przykázániách twojich.
Niech śię obrócą do mnie / którzy śię ćiebie boją / y którzy znáją świádectwá twoje.
Niech będźie ſerce moje uprzejme przy uſtáwách twojich / ábym nie był záwſtydzony.
Tęſkni duƺá mojá po zbáwieniu twojim / ocżekuję ná ſłowo twoje.
Uſtáły ocży moje / cżekájąc wyroku twego / gdy mówię : Kiedyż mię poćieƺyƺ?
Choćiáżem jeſt jáko nacżynie ſkórzáne w dymie / wƺákżem uſtáw twojich nie zápomniał.
Wieleż będźie dni ſługi twego? kiedyż ſąd wykonáƺ nád tymi / którzy mię prześládują?
Pyƺni pokopáli mi doły / co nie jeſt według zakonu twojego.
Wƺyſtkie przykázánia twoje ſą práwdą ; bez przycżyny mię prześládują ; rátujże mię.
Bez máłá mię już wniwecż nie obróćili ná źiemi ; á wƺákżem ja nie opuśćił przykázáń twojich.
Według miłośierdźiá twego ożyw mię / ábym ſtrzegł świádectwá uſt twojich.
O Pánie! ſłowo twoje trwá ná wieki ná niebie.
Od národu do národu práwdá twojá ; ugruntowáłeś źiemię / y ſtoi.
Według rozrządzeniá twego trwá to wƺyſtko áż do dniá tego ; wƺyſtko to záiſte jeſt ku ſłużbie twojey.
By był zakon twój nie był kochániem mojim / dáwnobym był zginął w utrápieniu mojim.
Ná wieki nie zápomnę ná przykázánia twoje / gdyżeś mię w nich ożywił.
Twójćim ja / záchowájże mię ; bo przykázáń twojich ƺukám.
Cżekáją ná mię niezbożnicy / áby mię zátráćili ; ále ja świádectwá twoje uwáżám.
Wƺelkiej rzecży koniec widzę ; ále przykazánie twoje bárzo ƺerokie.
O jákom śię rozmiłował zakonu twego! ták / iż káżdego dniá jeſt rozmyślániem mojim.
Nád nieprzyjáćiół mojich mędrƺym mię cżyniƺ przykázániem twojim ; bo je mam uſtáwicżnie przed ſobą.
Nád wƺyſtkich náucżyćieli mojich ſtałem śię rozumniejƺym ; bo świádectwá twoje ſą rozmyślániem mojim.
Nád ſtárców jeſtem roztropniejƺy ; bo przykázáń twojich przeſtrzegám.
Od wƺelkiej złej drogi záwśćiągám nogi ſwoje / ábym ſtrzegł ſłowá twego.
Od ſądów twojich nie odſtępuję / przeto / że ich ty mnie ucżyƺ.
O ják ſą ſłodkie ſłowá twoje podniebieniu memu! nád miód ſą ſłodƺe uſtom moim.
Z przykázáń twojich nábyłem rozumu : przetoż mam w nienáwiśći wƺelką śćieƺkę obłędliwą.
Słowo twe jeſt pochodnią nogą moim / á świátłośćią śćieƺce mojey.
Przyśięgłem y ucżynię temu doſyć / że będę ſtrzegł ſądów ſpráwiedliwośći twojey.
Jeſtem bárzo utrápiony ; o Pánie! ożyw mię według ſłowá twego.
Pánie! dobrowolne śluby uſt mojich przyjmij proƺę zá wdźięcżne / á ſądów twojich náucż mię.
Duƺá mojá jeſt w uſtáwicżnem niebezpiecżeńſtwie ; wƺákże ná zakon twój nie zápominam.
Sidło ná mię niezbożnicy záſtáwili ; lecż ja śię od przykázáń twojich nie obłądzę.
Zá dźiedźictwo wiecżne wźiąłem świádectwá twoje ; bo ſą rádośćią ſercá mego.
Nákłoniłem ſercá mego ku wykonywániu uſtáw twojich uſtáwicżnie / y áż do końcá ( żywotá ).
Wymyſły mam w nienáwiśći / á zakon twój miłuję.
Tyś jeſt ućiecżką moją / y tárcżą moją ; ná ſłowo twoje ocżekuję.
Odſtąpćież odemnie złośnicy / ábym ſtrzegł rozkázánia Bogá mojego.
Utwierdźże mię według ſłowá twego / ábym żył / á nie záwſtydzáj mię w ocżekiwániu mojim.
Podpieráj mię / ábym był záchowány / y rozmyślał w uſtáwách twojich uſtáwicżnie.
Podeptáłeś wƺyſtkich / którzy śię obłądźili od uſtáw twojich ; ábowiem jeſt kłámliwá zdrádá ich.
Odrzucáƺ jáko zużelicę wƺyſtkich niezbożników źiemi ; dla tego miłuję świádectwá twoje.
Drży od ſtráchu przed tobą ćiáło moje ; bo śię ſądów twojich lękám.
Cżynię ſądy y ſpráwiedliwość : nie podáwájże mię tym / którzy mi gwáłt cżynią.
Záſtąp ſám ſługę twego ku dobremu / áby mię hárdźi nie potłocżyli.
Ocży moje uſtáły / cżekájąc ná zbáwienie twoje / y ná wyrok ſpráwiedliwośći twojey.
Obchodź śię z ſługą twoim według miłośierdźiá twego / á uſtáw twojich náucż mię.
Sługámći ja twój / dájże mi zrozumienie ; ábym umiał świádectwá twoje.
Cżáſći już / ábyś cżynił Pánie! ábowiem wzruƺono zakon twój.
Dla tego umiłowáłem rozkázánia twoje nád złoto / á nád złoto nájwyborniejƺe.
Przeto / że wƺyſtkie przykázánia twoje / wƺyſtkie práwdźiwe być uznáję / á wƺelkie śćieƺki obłędliwe mam w nienáwiśći.
Dźiwne ſą świádectwá twoje ; przetoż ich ſtrzeże duƺá mojá.
Pocżątek ſłów twojich oświecá y dáje rozum proſtákom.
Uſtá moje otwierám y dyƺę ; ábowiemem przykázáń twojich prágnął.
Wejrzyjże ná mię / á zmiłuj śię nádemną według práwá tych / którzy miłują imię twoje.
Drogi moje utwierdź w ſłowie twojim / á niech nádemną nie pánuje żádná niepráwość.
Wybáw mię od ućiśnieniá ludzkiego / ábym ſtrzegł rozkázáń twojich.
Rozświeć nád ſługą twoim oblicże twoje / á náucż mię uſtáw twojich.
Strumienie wód płyną z ocżów mojich dla tych / którzy nie ſtrzegli zakonu twego.
Spráwiedliwyś ty / Pánie! y práwdźiwy w ſądách twojich.
Przykázáłeś ſpráwiedliwe świádectwá twoje / y wielce práwdźiwe.
Zniƺcżyłá mię gorliwość mojá / iż zápomináją ná ſłowo twoje nieprzyjáćiele moi.
Doſkonále ſą doświádcżone ſłowá twoje ; dla tego śię ſługá twój w nich rozkochał.
Jam málucżki y wzgárdzony ; wƺákże przykázáń twojich nie zápominam.
Spráwiedliwość twojá ſpráwiedliwość wiecżná / á zakon twój práwdá.
Ućiſk y utrápienie przyƺło ná mię ; przykázánia twoje ſą kochániem mojim.
Spráwiedliwość świádectw twojich trwá ná wieki ; dáj mi rozum / á żyć będę.
Wołám ze wƺyſtkiego ſercá / wyſłuchájże mię / o Pánie! á będę ſtrzegł uſtáw twojich.
Wołám do ćiebie / záchowájże mię / á będę pilen świádectw twojich.
Uprzedzám ćię ná świtániu y wołám / ná ſłowo twoje ocżekując.
Uprzedzáją ſtráż nocną ocży moje / przeto / ábym rozmyślał o wyrokách twojich.
Pánie! głos mój uſłyƺ według miłośierdźiá twego ; według ſądu twego ożyw mię.
Przybliżáją śię / którzy náśládują złośći / ći / którzy śię od zakonu twego oddálili.
Bliſkoś ty jeſt / Pánie! á wƺyſtkie przykázánia twoje ſą práwdą.
Dáwno to wiem o świádectwách twojich / żeś je ná wieki ugruntował.
Obácż utrápienie moje / á wyrwij mię ; bom ná zakon twój nie zápomniał.
Stáń przy ſpráwie mojey / á obroń mię ; dla ſłowá twego ożyw mię.
Dálekoć jeſt od niezbożników zbáwienie ; bo śię nie bádáją o uſtáwách twojich.
Wielkie ſą litośći twoje / Pánie! według ſądów twojich ożyw mię.
Wieleć jeſt prześládowców mojich y nieprzyjáćiół mojich ; wƺákże od świádectw twojich nie uchylám śię.
Widźiáłem przeſtępców / y mierźiáło mię to / że wyroku twego nie przeſtrzegáli.
Obácżże Pánie! iż rozkázánia twoje miłuję ; według miłośierdźiá twego ożyw mię.
Nájprzedniejƺá rzecż ſłowá twego jeſt práwdá / á ná wieki trwá wƺelki ſąd ſpráwiedliwośći twojey.
Kśiążętá mię prześládują bez przycżyny ; wƺákże ſłów twojich boi śię ſerce moje.
Ja śię weſelę z wyroku twego / ták jáko ten / który znájduje wielkie korzyśći.
Ale nienáwidzę kłámſtwá / y brzydzę śię nim ; ále zakon twój miłuję.
Chwálę ćię śiedm kroć przez dźień / dla ſądów twojich ſpráwiedliwych.
Pokój wielki dájeƺ tym / którzy miłują zakon twój / á nie doznáwáją żádnego obráżeniá.
Pánie! ocżekuję zbáwieniá twego ; á przykázánia twoje wykonywám.
Przeſtrzegá duƺá mojá świádectw twojich ; ábowiem je bárzo miłuję.
Przeſtrzegám przykázáń twojich y świádectw twojich ; ábowiem wƺyſtkie drogi moje ſą przed tobą.
Pánie! niech śię przybliży wołánie moje przed oblicże twoje ; według ſłowá twego dáj mi zrozumienie.
Niech przydźie prośbá mojá przed twárz twoję / á według obietnicy twojey wyrwij mię.
Chwałę wydádzą wárgi moje / gdy mię náucżyƺ uſtáw twojich.
Opowiádáć będźie język mój wyroki twoje ; bo wƺyſtkie przykázánia twoje ſą ſpráwiedliwość.
Niech mi będźie ná pomocy ręká twojá / gdyżem ſobie obrał przykázánia twoje.
Pánie! zbáwieniá twego prágnę / á zakon twój jeſt kochániem mojim.
Żyć będźie duƺá mojá / y będźie ćię chwáliłá / á ſądy twoje będą mi ná pomocy.
Błądzę jáko owcá zgubioná / ƺukájże ſługi twego ; boć przykázáń twojich nie zápominam.
