Przedniejƺemu śpiewákowi ná cżás poránny pſálm Dawidowy.
Boże mój! Boże mój! cżemuś mię opuśćił? oddáliłeś śię od wybáwieniá mego / od ſłów ryku mego.
Boże mój! wołám we dnie / á nie ozywáƺ mi śię ; y w nocy / á nie mogę śię uſpokoić.
Aleś ty Święty / mieƺkájący w chwałách Izráelſkich.
W tobie nádźieję mieli ojcowie náśi ; nádźieję mieli / á wybáwiłeś ich.
Do ćiebie wołáli / á wybáwieni ſą ; w tobie nádźieję mieli / á nie byli poháńbieni.
Alem ja robák / á nie cżłowiek : pośmiewiſko ludzkie / y wzgárdá poſpólſtwá.
Wƺyſcy / którzy mię widzą / ƺydzą ze mnie ; wykrzywiáją gębę / chwieją głową / mówiąc :
Spuśćił śię ná Páná / niechże go wyrwie ; niech go wybáwi / poniewáż śię w nim kocha.
Aleś ty jeſt / któryś mię wywiódł z żywotá / cżyniąc mi dobrą nádźieję jeƺcże u pierśi mátki mojey.
Ná tobie ſpolegám od národzeniá ſwego ; z żywotá mátki mojey tyś Bogiem moim.
Nie oddálájże śię odemnie ; ábowiem utrápienie bliſkie jeſt / á niemáƺ / ktoby rátował.
Obtocżyło mię mnóſtwo ćielców ; byki z Báſán obległy mię.
Otworzyły ná mię gębę ſwą jáko lew ƺárpájący y rycżący.
Rozpłynąłem śię jáko wodá / á rozſtąpiły śię wƺyſtkie kośći moje ; ſtáło śię ſerce moje jáko woſk / zſtopniáło w pośród wnętrznośći mojich.
Wyſchłá jáko ſkorupá moc mojá / á język mój przyſechł do podniebieniá mego ; náwet w prochu śmierći położyłeś mię.
Abowiem pſy mię obſkocżyły / gromádá złośników obległá mię ; przebodli ręce moje y nogi moje.
Zlicżyłbym wƺyſtkie kośći moje ; lecż oni ná mię pátrząc / przypátrują mi śię.
Rozdźielili odźienie moje miedzy śię / á o ƺáty moje los miotáli.
Ale ty / Pánie! nie oddáláj śię : mocy mojá! ná rátunek mój poſpieƺ.
Wyrwij od miecżá duƺę moję / z mocy pśiej jedynácżkę moję.
Wybáw mię z páƺcżęki lwiej / á od rogów jednorożcowych wyzwól mię.
Tedy opowiem imię twoje bráćiom mym ; w pośród zgromádzenia chwálić ćię będę.
Mówiąc : Którzy śię boićie Páná / chwálćie go ; wƺyſtko potomſtwo Jákóbowe wyſłáwiájćie go / á niech śię go boi wƺyſtko náśienie Izráelſkie.
Abowiem nie wzgárdźił / áni śię odwróćił od utrapienia ubogiego / áni ſkrył od niego oblicżá ſwego ; owƺem / gdy do niego wołał / wyſłuchał go.
O tobie chwałá mojá w zgromádzeniu wielkiem ; śluby moje oddam przed tymi / którzy śię ćiebie boją.
Będą jeść ubodzy / y náſycą śię ; chwálić będą Páná / którzy go ƺukáją ; ſerce wáƺe żyć będźie ná wieki.
Wſpomną y náwrócą śię do Páná wƺyſtkie gránice źiemi / y kłániáć śię będą przed oblicżem twojim wƺyſtkie pokolenia národów.
Abowiem Páńſkie jeſt króleſtwo / á on pánuje nád národámi.
Wƺyſcy bogáći źiemi będą jeść / y upádáć przed nim / przed oblicżnośćią jego kłániáć śię będą wƺyſcy zſtępujący w proch / y którzy duƺy ſwej żywo záchowáć nie mogą.
Náśienie ich ſłużyć mu będźie / á będźie przywłáƺcżáne Pánu w káżdym wieku.
Zbieżą śię / á będą opowiádáli ſpráwiedliwość jego národowi / który z nich wynijdźie / iż ją on wykonał.
