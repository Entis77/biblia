Nuż teráz / bogácże! płácżćie / nárzekájąc nád nędzámi wáƺemi / które przyjdą.
Bogáctwo wáƺe zgniło / á ƺáty wáƺe mole zgryzły.
Złoto wáƺe y ſrebro wáƺe pordzewiáło / á rdzá ich będźie ná świádectwo przećiwko wám / y pożre ćiáłá wáƺe jáko ogień ; zgromádźiliśćie ſkárb ná oſtátnie dni.
Oto / zápłátá robotników / którzy żęli kráiny wáƺe / od wáſ zátrzymáná wołá / á wołániá żeńców weƺły do uƺów Páná záſtępów.
Żyliśćie w rozkoƺách ná źiemi y bujáliśćie ; wytucżyliśćie ſercá wáƺe / jáko ná dźień zábijániá ofiár.
Potępiliśćie / zámordowáliśćie ſpráwiedliwego / á nie ſprzećiwiá śię wám.
Przetoż / bráćiá! bądźćie ćierpliwymi áż do przyjśćiá Páńſkiego. Oto / orácż ocżekuje drogiego pożytku źiemi / ćierpliwie go ocżekując / áżby otrzymáł deƺcż ránny y wiecżorny.
Bądźćież y wy ćierpliwymi / á utwierdzájćie ſercá wáƺe ; ábowiem śię przybliżá przyjśćie Páńſkie.
Nie wzdychájćie jedni przećiwko drugim / bráćiá! ábyśćie nie byli oſądzeni. Oto / ſędźiá już przede drzwiámi ſtoi.
Bierzćie zá przykłád / bráćiá moi! utrápieniá y ćierpliwośći proroków / którzy mówili w imieniu Páńſkim.
Oto / zá błogoſłáwionych mámy tych / którzy ćierpieli. O ćierpliwośći Ijobowej ſłyƺeliśćie / y koniec Páńſki widźieliśćie / iż wielce miłośierny jeſt Pán y litośćiwy.
A przed wƺyſtkiemi rzecżámi / bráćiá moi! nie przyśięgájćie áni przez niebo / áni przez źiemię / áni żádną inƺą przyśięgą ; ále niech będźie mowá wáƺá : Ták / ták ; y Nie / nie ; ábyśćie w obłudę nie wpádli.
Jeſt kto utrápiony między wámi / niechże śię modli ; jeſt kto dobrej myśli / niechájże śpiewá.
Choruje kto między wámi / niechże záwołá ſtárƺych zborowych / á niech śię modlą zá nim / pomázując go olejkiem w imieniu Páńſkim ;
A modlitwá wiáry uzdrowi chorego / y podnieśie go Pán ; á jeźliby śię grzechu dopuśćił / będźie mu odpuƺcżone.
Wyznáwájćie jedni przed drugimi upádki / á módlćie śię jedni zá drugimi ; ábyśćie byli uzdrowieni. Wiele może uprzejmá modlitwá ſpráwiedliwego.
Elijáƺ był cżłowiek tymże biedom poddány / jáko y my / á modlitwą modlił śię / żeby deƺcż nie pádáł ; y nie pádáł deƺcż ná źiemię trzy látá y ƺeść mieśięcy /
Y záś śię modlił / á wydáło niebo deƺcż / y źiemiá zrodźiłá owoce ſwoje.
Bráćiá! jeźliby śię kto z wáſ obłądźił od práwdy / á náwróćiłby go kto /
Niechże wie / że ktoby odwróćił grzeƺniká od błędnej drogi jego / záchowá duƺę od śmierći / y zákryje mnóſtwo grzechów.
