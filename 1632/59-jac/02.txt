Bráćiá moi! nie miejćie z brákowániem oſób wiáry Páná náƺego Jezuſá Chryſtuſá / który chwálebny jeſt.
Abowiem gdyby wƺedł do zgromádzeniá wáƺego mąż / májąc pierśćień złoty w ƺáćie świetnej / á wƺedłby też y ubogi w podłem odzieniu :
Y wejrzelibyśćie ná tego / co má świetną ƺátę / á rzeklibyśćie mu : Ty! śiądź ſám pocżćiwie! á ubogiemubyśćie rzekli : Ty! tám ſtój / álbo śiądź tu pod podnóżkiem moim!
Azáżeśćie już nie ucżynili różnośći między ſobą / y nie ſtáliśćie śię ſędziámi myśli złych?
Słuchájćie / bráćiá moi mili! ázáż Bóg nie obráł ubogich ná tym świećie / áby byli bogátymi w wierze / y dziedzicámi króleſtwá / które obiecáł tym / którzy go miłują?
Aleśćie wy zniewáżyli ubogiego. Azáż bogácże gwáłtem wáſ nie ućiſkáją / y do ſądów wáſ nie poćiągáją?
Azáż oni nie bluźnią onego zácnego imieniá / które jeſt wzywáne nád wámi?
A jeźliże pełnićie zákon królewſki według Piſmá : Będzieƺ miłowáł bliźniego twego jáko ſámego śiebie / dobrze cżynićie.
Lecż jeźli oſobámi brákujećie / grzech popełniáćie / y bywáćie przekonáni od zákonu / jáko przeſtępcy.
Abowiem ktobykolwiek záchowáł wƺyſtek zákon / á w jednymby upádł / ſtáł śię winien wƺyſtkich przykázáń.
Bo który rzekł : Nie będzieƺ cudzołożył / ten też rzekł : Nie będzieƺ zábijáł ; á jeźlibyś nie cudzołożył / álebyś zábił / ſtáłeś śię przeſtępcą zákonu.
Ták mówćie y ták cżyńćie / jáko ći / którzy według zákonu wolnośći máćie być ſądzeni.
Abowiem ſąd bez miłośierdziá będzie temu / co nie cżynił miłośierdziá ; ále miłośierdzie chlubi śię przećiwko ſądowi.
Cóż pomoże / bráćiá moi! jeźliby kto rzekł / iż má wiárę / á ucżynkówby nie miáł? izáli go oná wiárá może zbáwić?
A gdyby brát álbo śioſtrá byli nieodziáni / y ſchodziłoby im ná powƺedniej żywnośći /
Y rzekłby im kto z wáſ : Idźćie w pokoju / ugrzejćie śię / y nájedzćie śię / á nie dálibyśćie im potrzeb ćiáłu náleżących / cóż to pomoże?
Tákże y wiárá / nie máli ucżynków / mártwá jeſt ſámá w ſobie.
Ale rzecże kto : Ty máƺ wiárę / á já mám ucżynki ; ukáż mi wiárę twoję bez ucżynków twoich / á já tobie ukáżę wiárę moję z ucżynków moich.
Ty wierzyƺ / iż jeden jeſt Bóg / dobrze cżyniƺ ; y dyjábli temu wierzą / wƺákże drżą /
Ale chceƺli wiedzieć / o cżłowiecże márny! iż wiárá bez ucżynków mártwá jeſt?
Abráhám / ojćiec náƺ / izáli nie z ucżynków uſpráwiedliwiony jeſt / gdy ofiárowáł Izááká / ſyná ſwego / ná ołtárzu?
Widziƺ / iż wiárá ſpólnie robiłá z ucżynkámi jego / á z ucżynków wiárá doſkonáłą śię ſtáłá.
A ták wypełniło śię Piſmo / które mówi : Y uwierzył Abráhám Bogu / y przycżytáno mu to ku ſpráwiedliwośći / y przyjáćielem Bożym názwány jeſt.
A widzićież / iż z ucżynków uſpráwiedliwiony bywá cżłowiek / á nie z wiáry tylko.
Tákże też y Rácháb / wƺetecżnicá / izáli nie z ucżynków jeſt uſpráwiedliwioná / gdy przyjęłá onych poſłów y inƺą drogą wypuśćiłá?
Abowiem jáko ćiáło bez duƺy jeſt mártwe / ták y wiárá bez ucżynków mártwá jeſt.
