Jákób / ſługá Boży y Páná Jezuſá Chryſtuſá / dwunáſtu pokoleniom / które ſą w rozproƺeniu / zdrowiá życży.
Zá nájwiękƺą rádość miejćie / bráćiá moi! gdy w rozmáite pokuſy wpádáćie /
Wiedząc / iż doświádcżenie wiáry wáƺej ſpráwuje ćierpliwość ;
A ćierpliwość niech má doſkonáły ucżynek / żebyśćie byli doſkonáli y zupełni / którymby ná nicżem nie ſchodziło.
A jeźli komu z wáſ ſchodzi ná mądrośći / niech prośi u Bogá / który ją ƺcżerze wƺyſtkim dáje / á nie wymáwiá ; y będzie mu dáná.
Ale niech prośi z wiárą / nic nie wątpiąc ; ábowiem kto wątpi / jeſt podobny wáłowi morſkiemu / który od wiátru pędzony y miotány bywá.
Bo niecháj nie mniemá ten cżłowiek / áby co miáł wziąć od Páná.
Mąż umyſłu dwoiſtego jeſt nieſtátecżny we wƺyſtkich drogách ſwoich.
A niech śię chlubi brát niſkiego ſtánu w wywyżƺeniu ſwojim /
A bogáty w poniżeniu ſwojim ; bo jáko kwiát tráwy przeminie.
Abowiem jáko ſłońce / kiedy weƺło z gorącośćią / uſuƺyło tráwę / á kwiát jej opádł / y zginęłá oná ślicżność kƺtáłtu jego / ták y bogáty w drogách ſwoich uwiędnie.
Błogoſłáwiony mąż / który znośi pokuƺenie ; bo gdy będzie doświádcżony / weźmie koronę żywotá / którą obiecáł Pán tym / którzy go miłują.
Żáden / gdy bywá kuƺony / niech nie mówi : Od Bogá kuƺony bywám ; bo Bóg nie może kuƺony być we złem / á ſám nikogo nie kuśi.
Ale káżdy bywá kuƺony / gdy od ſwoich włáſnych pożądliwośći bywá poćiągniony y przynęcony.
Zátem pożądliwość pocżąwƺy / rodzi grzech / á grzech będąc wykonány / rodzi śmierć.
Nie błądźćież / bráćiá moi mili!
Wƺelki dátek dobry / y wƺeláki dár doſkonáły z góry jeſt / zſtępujący od Ojcá świátłośći / u którego niemáƺ odmiány / áni záćmieniá ná wſtecż śię wrácájącego.
Który / przeto że chćiáł / porodził náſ ſłowem práwdy ku temu / żebyśmy byli niejákiemi pierwiáſtkámi ſtworzeniá jego.
A ták / bráćiá moi mili! niech będzie káżdy cżłowiek prędki ku ſłuchániu / ále nierychły ku mówieniu / y nierychły ku gniewowi.
Bo gniew mężá nie ſpráwuje ſpráwiedliwośći Bożej.
A ták odrzućiwƺy wƺelákie plugáſtwo y zbytek złośći / z ćichośćią przyjmijćie ſłowo wƺcżepione / które może zbáwić duƺe wáƺe.
A bądźćie cżynićielámi ſłowá / á nie ſłuchácżámi tylko oƺukiwájącymi ſámych śiebie.
Abowiem jeźli kto jeſt ſłuchácżem ſłowá á nie cżynićielem / ten podobny jeſt mężowi przypátrującemu śię oblicżu národzeniá ſwego we zwierćiádle ;
Bo ſámego śiebie obejrzáł y odƺedł / á wnet zápomniáł / jákim był.
Ale ktoby wejrzáł w on doſkonáły zákon wolnośći / y zoſtáwáłby w nim / ten nie będąc ſłuchácżem zápámiętliwym / ále cżynićielem ucżynku / ten błogoſłáwionym będzie w ucżynku ſwoim.
Jeźli kto między wámi zdá śię być nábożnym / nie kiełznájąc języká ſwego / ále zwodząc ſerce ſwe / tego nábożeńſtwo próżne jeſt.
Nábożeńſtwo cżyſte y niepokáláne u Bogá y Ojcá to jeſt : Náwiedzáć śieroty y wdowy w ućiſku ich / y záchowáć ſámego śiebie niepokálánym od świátá.
