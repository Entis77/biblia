Skądże ſą wálki y zwády między wámi? Izáli nie ſtąd / to jeſt z lubośći wáƺych / które wálcżą w cżłonkách wáƺych?
Pożądáćie / á nie máćie / zájrzyćie y záwidzićie / á nie możećie doſtáć ; wádzićie śię y wálcżyćie / wƺákże nie otrzymujećie / przeto iż nie prośićie.
Prośićie á nie bierzećie / przeto / iż źle prośićie / ábyśćie to ná rozkoƺe wáƺe obrácáli.
Cudzołożnicy y cudzołożnice! nie wiećież / iż przyjáźń świátá jeſt nieprzyjáźnią Bożą? Przetoż / ktobykolwiek chćiáł być przyjáćielem tego świátá / ſtáje śię nieprzyjáćielem Bożym.
Albo mniemáćie / iż próżno Piſmo mówi : Izáli ku zázdrośći pożądá duch / który w náſ mieƺká?
Owƺem / hojniejƺą dáje łáſkę ; bo mówi : Bóg śię pyƺnym ſprzećiwiá / ále pokornym łáſkę dáje.
Poddájćież śię tedy Bogu / á dájćie odpór dyjábłu / á ućiecże od wáſ.
Przybliżćie śię ku Bogu / á przybliży śię ku wám. Ochędóżćie ręce grzeƺnicy / y ocżyśććie ſercá / którzyśćie umyſłu dwoiſtego /
Bądźćie utrápieni / y żáłujćie y płácżćie ; śmiech wáƺ niech śię obróći w żáłość / á rádość w ſmutek.
Uniżájćie śię przed oblicżem Páńſkim / á wywyżƺy wáſ.
Nie obmáwiájćie jedni drugich / bráćiá! Kto obmáwiá brátá / y potępiá brátá ſwego / obmáwiá zákon / y potępiá zákon ; á jeźli potępiƺ zákon / nie jeſteś cżynićielem zákonu / ále ſędzią.
Jeden jeſt zákonodáwcá / który może zbáwić y zátráćić. Ale ty ktoś jeſt / co potępiáƺ drugiego?
Nuż teráz wy / co mówićie : Dziś / álbo jutro pójdziemy do tego miáſtá / y zámieƺkámy tám przez jeden rok / á będziemy kupcżyć / y zyſk ſobie przywiedziemy ;
( Którzy nie wiećie / co jutro będzie ; bo cóż jeſt żywot wáƺ? Párá záiſte jeſt / którá śię ná máły cżáſ pokázuje / á potem niƺcżeje. )
Zámiáſt tego / cobyśćie mieli mówić : Będzieli Pán chćiáł / á będziemyli żywi / ucżynimy to álbo owo.
Ale teráz chlubićie śię w pyƺe wáƺej ; wƺelká chlubá tákowá złá jeſt.
Przetoż / kto umie dobrze cżynić / á nie cżyni / grzech má.
