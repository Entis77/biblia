Skądże <i>ſą</i> walki y zwády miedzy wámi? Izali nie z tąd? <i>to jeſt, z</i> lubośći wáƺych / które walcżą w cżłonkách wáƺych.
Pożądaćie / á nie maćie : zájrzyćie y zawidźićie / á nie możećie doſtáć : wádźićie śię y walcżyćie / wƺákże nie otrzymawaćie / przeto iż nie prośićie.
Prośićie / á nie bierzećie ; przeto iż źle prośićie : ábyśćie to ná roſkoƺy ſwoje obracáli.
Cudzołożnicy y cudzołożnice / nie wiećież iż przyjaźń świátá / jeſt nieprzyjáźnią Bożą? Przetoż / ktobykolwiek chćiał być przyjaćielem tego świátá / ſtawa śię nieprzyjaćielem Bożym.
Albo mniemaćie / iż prózno piſmo mówi : <i>Izali</i> ku zazdrośći pożąda duch który w nas mieƺka?
Owƺem hojniejƺą dawa łáſkę : bo mówi ; Bóg śię pyƺnym zprzećiwia / ále pokornym łáſkę dawa.
Poddajćież śię tedy Bogu / á dajćie odpór dyabłu / á ućiecże od was.
Przybliżćie śię ku Bogu / á przybliży śię ku wam. Ochędóżćie ręce grzeƺnicy / y ocżyśććie ſercá / którzyśćie umyſłu dwojiſtego.
Bądźćie utrapieni / y żáłujćie / y płácżćie : śmiech wáƺ niech śię obróći w żáłość / á rádość w ſmętek.
Uniżajćie śię przed oblicżem PAńſkim / á wywyżƺy was.
Nie obmawiajćie jedni drugich / bráćia : Kto obmawia brátá / y potępia brátá ſwego / obmawia zakon / y potępia zakon : á jeſli potępiƺ zakon / nie jeſteś cżynićielem zakonu / ále ſędźią.
Jeden jeſt Zakonodawcá / który może zbáwić y zátráćić. Ale ty ktoś jeſt co potępiaƺ drugiego?
Nuż teraz wy co mówićie : Dźiś / álbo jutro pójdźiemy do tego miáſtá / y zmieƺkamy tám przez jeden rok / á będźiemy kupcżyć / y zyſk ſobie przywiedźiemy :
( Którzy nie wiećie co jutro <i>będźie</i> : Bo cóż jeſt żywot wáƺ? Párá zájiſte jeſt / która śię ná máły <i>cżás</i> pokázuje / á potym niƺcżeje. )
Miáſto tego cobyśćie mieli mówić ; Będźieli PAn chćiał / á będźiemyli żywi / ucżynimy to álbo owo.
Ale teraz chlubićie śię w pyƺe ſwojey. Wƺelka chlubá tákowa / zła jeſt.
Przetoż / kto umie dobrze cżynić / á niecżyni / grzech ma.
