Byli też y fáłƺywi prorocy między ludem / jáko y między wámi będą fáłƺywi náucżyćiele / którzy z ſobą wprowádzą kácerſtwá zátráceniá ; y Páná / który ich kupił / záprzą śię / ſámi ná śię przywodząc prędkie zginienie.
A wiele ich náśládowáć będą zginieniá ich / przez których drogá práwdy będzie bluźnioná.
Y przez łákomſtwo zmyślonemi ſłowámi kupcżyć będą / którym ſąd z dáwná nie omieƺkuje / y zátrácenie ich nie drzemie.
Abowiem jeźli Bóg Aniołom / którzy byli zgrzeƺyli / nie przepuśćił / ále ſtrąćiwƺy ich do piekłá / podáł łáńcuchom ćiemnośći / áby byli záchowáni ná ſąd :
Tákże y pierwƺemu świátu nie przepuśćił / ále Noego ſámoóſmego / káznodzieję ſpráwiedliwośći / záchowáł / przywiódłƺy potop ná świát niepobożnych ;
Y miáſtá Sodomcżyków y Gomorry w popiół obróćiwƺy podwróceniem potępił / wyſtáwiwƺy je ná przykłád tym / którzyby niepobożnie żyli ;
A ſpráwiedliwego Lotá / onych niezbożników rozpuſtnem obcowániem ſtrápionego / wyrwáł.
Abowiem widzeniem y ſłyƺeniem on ſpráwiedliwy mieƺkájąc między nimi / dzień po dniu duƺę ſpráwiedliwą ucżynkámi ich niezbożnemi trápił.
Umie Pán pobożnych z pokuƺeniá wyrywáć / á nieſpráwiedliwych ná dzień ſądu ku kárániu chowáć ;
A nájwięcej tych / którzy zá ćiáłem w pożądliwośći plugáſtwá chodzą / á zwierzchnośćią pogárdzáją / śmieli / y ſobie śię podobájący / nie wzdrygáją śię bluźnić przełożeńſtw.
Choćiáż Aniołowie będąc więkƺymi śiłą y mocą / nie przynoƺą przećiwko nim przed Páná bluźniercżego ſądu.
Ale ći / jáko bydło bezrozumne / które zá przyrodzeniem idzie / ſpráwione ná ułowienie y ſkázę / bluźniąc to / cżego nie wiedzą / w tej ſkázie ſwojej záginą.
Y odnioſą zápłátę nieſpráwiedliwośći / jáko y ći / którzy máją zá rozkoƺ káżdodzienne lubośći / będąc plugáſtwem y zmázą / rozkoƺ máją w zdrádách ſwoich / z wámi bánkietując :
Ocży máją pełne cudzołóſtwá / y bez przeſtániá grzeƺące / przyłudzájąc duƺe nieſtátecżne / májąc ſerce wyćwicżone w łákomſtwie / ſynowie przeklęſtwá /
Którzy opuśćiwƺy proſtą drogę / zbłądzili / náśládując drogi Báláámá / ſyná Boſorowego / który zápłátę nieſpráwiedliwośći umiłowáł ;
Ale miáł káránie zá ſwój wyſtępek / poniewáż járzmu niemá oślicá poddáná / cżłowiecżym głoſem przemówiwƺy / záhámowáłá ƺáleńſtwo proroká.
Ci ſą ſtudniámi bez wody / obłoki od wichru pędzone / którym chmurá ćiemnośći ná wieki jeſt záchowáná.
Abowiem nádętą próżność mówiąc / przyłudzáją przez pożądliwość ćiáłá y rozpuſty tych / którzy byli práwdziwie ućiekli od obcujących w błędzie /
Wolność im obiecując / á ſámi będąc niewolnikámi ſkázy. Abowiem kto jeſt od kogo przezwyćiężony / temu też jeſt zniewolony.
Bo poniewáż oni uƺli plugáſtw świátá przez poznánie Páná y zbáwićielá Jezuſá Chryſtuſá / á znowu śię záś niemi uwikłáwƺy / zwyćiężeni bywáją ; ſtáły śię oſtátecżne rzecży ich gorƺe niż pierwƺe.
Boby im było lepiej / nie uznáć drogi ſpráwiedliwośći / niżeli poznáwƺy ją / odwróćić śię od podánego im rozkázániá świętego.
Aleć śię im przydáło według onej práwdziwej przypowieśći : Pieſ wróćił śię do zwrácániá ſwego / á świniá umytá do wálániá śię w błoćie.
