Byli tez y fałƺywi Prorocy miedzy ludem / jáko y miedzy wámi będą fałƺywi náucżyćiele ; którzy z ſobą wprowádzą kácerſtwá zátrácenia ; y PAná / który je kupił / záprzą śię ſámi / ná śię przywodząc prętkie zginienie.
A wiele ich náśládowáć będą zginienia ich / przez które drogá prawdy będźie bluźniona :
Y przez łákomſtwo zmyślonymi ſłowy wámi kupcżyć będą / którym ſąd z dawná nie omieƺkawa / y zátrácenie ich nie drżemie.
Abowiem jeſli Bóg Anjołom którzy byli zgrzeƺyli nie przepuśćił / ále zſtrąćiwƺy je do piekłá / podał łáńcuchom ćiemnośći / áby byli záchowáni ná ſąd :
Tákże y pierwƺemu świátu nie przepuśćił ; ále Noego ſámo-óſmego / káznodźieję ſpráwiedliwośći záchował / przywiódƺy potop ná świát niepobożnych :
Y miáſtá Sodomcżyków y Gomorry w popiół obróćiwƺy podwróceniem potępił ; wyſtáwiwƺy je ná przykład tym / którzyby niepobożnie żyli :
A ſpráwiedliwego Lotá / onych niezbożników roſpuſtnym obcowániem ztrapionego / wyrwał.
Abowiem widzeniem y ſłyƺeniem on ſpráwiedliwy mieƺkájąc miedzy nimi / dźień po dniu duƺę ſpráwiedliwą ucżynkámi <i>ich</i> niezbożnymi trapił.
Umie PAn pobożnych z pokuƺenia wyrywáć / á nieſpráwiedliwych ná dźień ſądu ku karániu chowáć.
A nawięcey tych / którzy zá ćiáłem w pożądliwośći plugáſtwá chodzą / á zwierzchnośćią pogardzáją / śmieli / y ſobie śię podobájący / nie wzdrygáją śię bluźnić przełożeńſtw.
Choćiaż Anjołowie będąc więkƺymi śiłą y mocą / nie przynoƺą przećiwko nim przed PAná bluźnierſkiego ſądu.
Ale ći jáko bydło bezrozumne / które zá przyrodzeniem idźie / ſpráwione ná ułowienie y ſkázę / bluźniąc to cżego nie wiedzą / w tey ſkáźie ſwojey záginą :
Y odnioſą zapłátę nieſpráwiedliwośći / jáko ći którzy máją zá roſkoƺ káżdodźienne lubośći : <i>będąc</i> plugáſtwem y zmázą / roſkoƺ máją w zdrádách ſwojich / z wámi bánkietując.
Ocży májąc pełne cudzołóſtwá / y bez przeſtánia grzeƺące. Przyłudzájąc duƺe nieſtátecżne / májąc ſerce wyćwicżone w łákomſtwie / ſynowie przeklęctwá :
<i>Którzy</i> opuśćiwƺy proſtą drogę / zbłądźili ; náśládując drogi Báláámá <i>ſyná</i> Boſorowego / który zapłátę nieſpráwiedliwośći umiłował /
Ale miał karánie zá ſwój wyſtępek : <i>ponieważ</i> járzmu niema <i>oślicá</i> poddána / cżłowiecżym głoſem przemówiwƺy / záhámowáłá ƺaleńſtwo Proroká.
Ci ſą ſtudniámi bez wody / obłoki od wichru pędzone / którym chmurá ćiemnośći ná wieki jeſt záchowána.
Abowiem nádętą prózność mówiąc / przyłudzáją przez pożądliwość ćiáłá y roſpuſty tych / którzy byli prawdźiwie ućiekli od obcujących w błędźie :
Wolność im obiecując / á ſámi będąc niewolnikámi ſkázy. Abowiem kto jeſt od kogo przewyćiężony / temu też jeſt zniewolony.
Bo poniewáż <i>oni</i> uƺli plugaſtw świátá / przez poznánie PAná y Zbáwićielá JEzuſá CHryſtuſá / á znowu śię záś nimi uwikławƺy / zwyćiężeni bywáją ; ſtáły śię oſtátecżne <i>rzecży</i> ich gorƺe niż pierwƺe.
Boby im było lepiey nie uznáć drogi ſpráwiedliwośći / niżeli poznawƺy <i>ją,</i> odwróćić śię od podánego im rozkazánia świętego.
Aleć śię im przydáło <i>według</i> oney prawdźiwey przypowieśći : Pies wróćił śię do zwracánia ſwego ; á świniá umyta do walánia śię w błoćie.
