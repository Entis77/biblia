Potem ſtáło śię ſłowo Páná záſtępów / mówiąc :
Ták mówi Pán záſtępów : Zápáliłem śię nád Syonem gorliwośćią wielką / owƺem / rozgniewániem wielkiem zápáliłem śię.
Ták mówi Pán : Náwróćiłem śię do Syonu / y mieƺkám w pośród Jeruzálemu / áby Jeruzálem zwáno miáſtem wiernem / á górę Páná záſtępów / górą świętobliwośći.
Ták mówi Pán záſtępów : Jeƺcże śiádáć będą ſtárcy y báby ná ulicách Jeruzálemſkich / májąc káżdy z nich láſkę w ręce ſwej dlá zeƺłośći wieku.
Ulice tákże miáſtá pełne będą chłopiąt y dźiewcżąt grájących ná ulicách jego.
Ták mówi Pán záſtępów : Izáli / że śię to niepodobná widźi przed ocżymá oſtátków ludu tego tych dni / będźie też to niepodobná przed ocżymá mojemi? mówi Pán záſtępów.
Ták mówi Pán záſtępów : Oto Já wybáwię lud mój z źiemi ná wſchód / y z źiemi ná záchód ſłońcá.
Y przywiodę ich záś ; á będą mieƺkáć w pośród Jeruzálemu / y będą ludem moim / á Já będę Bogiem ich w práwdźie y w ſpráwiedliwośći.
Ták mówi Pán záſtępów : Niech śię zmocnią ręce wáƺe / którzyśćie ſłucháli w tych dniách ſłów tych z uſt proroków / którzy byli ode dniá / którego záłożony jeſt dom Páná záſtępów / że kośćiół má być dobudowány.
Bo śię przed temi dniámi prácá ludzká y prácá bydląt nie nágrádzáłá / náwet wychodzącemu y wchodzącemu nie było pokoju dlá nieprzyjáćielá ; bom Já ſpuśćił wƺyſtkich ludźi jednego z drugim.
Lecż teráz nie ták jáko zá dni przeƺłych cżynię oſtátkowi ludu tego / mówi Pán záſtępów :
Ale śiewy máćie ſpokojne ; winná máćicá wydáje owoc ſwój / y źiemiá wydáje urodzáj ſwój / niebioſá tákże wydáwáją roſę twoję / á to wƺyſtko dáję w ośiádłość oſtátkom ludu tego.
Y ſtánie śię / że jákośćie byli przeklęſtwem między pogánámi / o domie Judzki y domie Izráelſki! ták wáſ záś będę ochrániáł / y będźiećie błogoſłáwieńſtwem ; nie bójćie śię / niech śię zmácniáją ręce wáƺe.
Bo ták mówi Pán záſtępów : Jákom wám był umyślił źle ucżynić / gdy mię do gniewu pobudzáli ojcowie wáśi / mówi Pán záſtępów / á nie żáłowáłem tego /
Ták náwróćiwƺy śię umyśliłem w te dni dobrze cżynić Jeruzálemowi y domowi Judzkiemu ; nie bójćież śię.
Teć ſą rzecży / które cżynić będźiećie : Práwdę mówćie káżdy z bliźnim ſwoim / práwy y ſpokojny ſąd cżyńćie w brámách wáƺych ;
A jeden drugiemu nic złego nie myślćie w ſercách wáƺych / á w krzywoprzyśięſtwie śię nie kochájćie ; bo to wƺyſtko jeſt / cżego nienáwidzę / mówi Pán.
Y ſtáło śię ſłowo Páná záſtępów do mnie / mówiąc :
Ták mówi Pán záſtępów : Poſt cżwártego / y poſt piątego / y poſt śiódmego / y poſt dźieśiątego mieśiącá obróći śię domowi Judzkiemu w rádość y weſele / y w rozkoƺne urocżyſte świętá ; ále práwdę y pokój miłujćie.
Ták mówi Pán záſtępów : Jeƺcżeć będą przychodźić národy y obywátele wielu miáſt ;
Przychodźić / mówię / będą obywátele jednego miáſtá do drugiego / mówiąc : Pójdźmy ochotnie błágáć oblicże Páńſkie / á ƺukáć Páná záſtępów ; y rzecże káżdy : Pójdę y já.
A ták wiele ludu y národów niezlicżonych przyjdźie ƺukáć Páná záſtępów w Jeruzálemie / y błágáć oblicże Páńſkie.
Ták mówi Pán záſtępów : W one dni uchwycą śię dźieśięć mężów ze wƺyſtkich języków onych národów ; uchwycą śię / mówię / podołká jednego Żydá / mówiąc : Pójdźiemy z wámi / bo ſłyƺymy / że Bóg jeſt z wámi.
