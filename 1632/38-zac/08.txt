Potem ſtáło śię ſłowo Páná zaſtępów / mówiąc :
Ták mówi Pán zaſtępów : Zápáliłem śię nád Syonem gorliwośćią wielką / owƺem / rozgniewániem wielkiem zápáliłem śię.
Ták mówi Pán : Náwróćiłem śię do Syonu / y mieƺkám w pośród Jeruzálemu / áby Jeruzálem zwano miáſtem wiernem / á górę Páná zaſtępów / górą świętobliwośći.
Ták mówi Pán zaſtępów : Jeƺcże śiádáć będą ſtárcy y báby ná ulicách Jeruzálemſkich / májąc káżdy z nich láſkę w ręce ſwej dla zeƺłośći wieku.
Ulice tákże miáſtá pełne będą chłopiąt y dźiewcżąt grájących ná ulicách jego.
Ták mówi Pán zaſtępów : Izali / że śię to niepodobná widźi przed ocżymá oſtátków ludu tego tych dni / będźie też to niepodobná przed ocżymá mojemi? mówi Pán zaſtępów.
Ták mówi Pán zaſtępów : Oto Ja wybáwię lud mój z źiemi ná wſchód / y z źiemi ná záchód ſłońcá.
Y przywiodę ich záś ; á będą mieƺkáć w pośród Jeruzálemu / y będą ludem moim / á Ja będę Bogiem ich w práwdźie y w ſpráwiedliwośći.
Ták mówi Pán zaſtępów : Niech śię zmocnią ręce wáƺe / którzyśćie ſłucháli w tych dniách ſłów tych z uſt proroków / którzy byli ode dniá / którego záłożony jeſt dom Páná zaſtępów / że kośćiół ma być dobudowány.
Bo śię przed temi dniámi prácá ludzká y prácá bydląt nie nágrádzáłá / náwet wychodzącemu y wchodzącemu nie było pokoju dla nieprzyjáćielá ; bom Ja ſpuśćił wƺyſtkich ludźi jednego z drugim.
Lecż teraz nie ták jáko zá dni przeƺłych cżynię oſtátkowi ludu tego / mówi Pán zaſtępów :
Ale śiewy máćie ſpokojne ; winná máćicá wydáje owoc ſwój / y źiemiá wydáje urodzáj ſwój / niebioſá tákże wydáwáją roſę twoję / á to wƺyſtko dáję w ośiádłość oſtátkom ludu tego.
Y ſtánie śię / że jákośćie byli przeklęſtwem miedzy pogánámi / o domie Judzki y domie Izráelſki! ták was záś będę ochrániał / y będźiećie błogoſłáwieńſtwem ; nie bójćie śię / niech śię zmácniáją ręce wáƺe.
Bo ták mówi Pán zaſtępów : Jákom wam był umyślił źle ucżynić / gdy mię do gniewu pobudzáli ojcowie wáśi / mówi Pán zaſtępów / á nie żáłowáłem tego /
Ták náwróćiwƺy śię umyśliłem w te dni dobrze cżynić Jeruzálemowi y domowi Judzkiemu ; nie bójćież śię.
Teć ſą rzecży / które cżynić będźiećie : Práwdę mówćie káżdy z bliźnim ſwoim / práwy y ſpokojny ſąd cżyńćie w brámách wáƺych ;
A jeden drugiemu nic złego nie myślćie w ſercách wáƺych / á w krzywoprzyśięſtwie śię nie kochájćie ; bo to wƺyſtko jeſt / cżego nienáwidzę / mówi Pán.
Y ſtáło śię ſłowo Páná zaſtępów do mnie / mówiąc :
Ták mówi Pán zaſtępów : Poſt cżwártego / y poſt piątego / y poſt śiódmego / y poſt dźieśiątego mieśiącá obróći śię domowi Judzkiemu w rádość y weſele / y w rozkoƺne urocżyſte świętá ; ále práwdę y pokój miłujćie.
Ták mówi Pán zaſtępów : Jeƺcżeć będą przychodźić národy y obywátele wielu miaſt ;
Przychodźić / mówię / będą obywátele jednego miáſtá do drugiego / mówiąc : Pójdźmy ochotnie błágáć oblicże Páńſkie / á ƺukáć Páná zaſtępów ; y rzecże káżdy : Pójdę y ja.
A ták wiele ludu y národów niezlicżonych przydźie ƺukáć Páná zaſtępów w Jeruzálemie / y błágáć oblicże Páńſkie.
Ták mówi Pán zaſtępów : W one dni uchwycą śię dźieśięć mężów ze wƺyſtkich języków onych národów ; uchwycą śię / mówię / podołká jednego Żydá / mówiąc : Pójdźiemy z wámi / bo ſłyƺymy / że Bóg jeſt z wámi.
