Potem náwróćił śię Anioł / który mówił zemną / y obudźił mię / jáko gdy kto budzony bywá ze ſnu ſwego ;
Y rzekł do mnie : Cóż widźiƺ? Y rzekłem : Widzę / á oto świecżnik wƺyſtek złoty / á cżáƺá ná wierzchu jego / y śiedm lámp jego ná nim / śiedm też nálewek do onych śiedmiu lámp / które ſą ná wierzchu jego ;
Dwie też oliwy przytem / jedná po práwej ſtronie cżáƺy / á drugá po lewej ſtronie jey.
Tedy odpowiádájąc rzekłem do Aniołá / który mówił zemną / mówiąc : Cóż to jeſt / pánie mój?
Y odpowiedźiał Anioł / który mówił zemną / y rzekł mi : Izáż nie wieƺ / co to jeſt? Y rzekłem : Nie wiem / pánie mój!
Tedy odpowiádájąc rzekł do mnie / mówiąc : Toć jeſt ſłowo Páńſkie do Zorobábelá mówiące : Nie wojſkiem áni śiłą ſtánie śię to / ále duchem moim / mówi Pán zaſtępów.
Cóżeś ty jeſt / o góro wielká! przećiwko Zorobábelowi? Równiná ; bo on wywiedźie kámień główny z głośnym okrzykiem : Łáſká / łáſká nád nim.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Ręce Zorobábelowe záłożyły dom ten / y ręce jego dokonáją go ; á dowieƺ śię / że Pán zaſtępów poſłał mię do was.
Bo któżby wzgárdźił dniem máłych pocżątków? poniewáż śię weſelą / pátrząc ná ten kámień / to jeſt / ná práwidło w ręce Zorobábelowej / ná te śiedm ocżów Páńſkich przechodzących wƺyſtkę źiemię.
Tedy odpowiádájąc rzekłem mu : Cóż ſą te dwie oliwy po práwej ſtronie tego świecżniká / y po lewej ſtronie jego?
Znowu odpowiádájąc rzekłem mu : Cóż ſą te dwie oliwki / które ſą miedzy dwomá rurkámi złotemi / które z śiebie złoto wylewáją?
Tedy rzekł do mnie / mówiąc : Izali nie wieƺ / co to jeſt? Rzekłem : Nie wiem / Pánie mój.
Y rzekł : Teć ſą one dwie oliwy / które ſą u Pánującego ná wƺyſtkiej źiemi.
