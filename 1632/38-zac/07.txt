Potem ſtáło śię roku cżwártego Dáryjuƺá królá / ſtáło śię ſłowo Páńſkie do Zácháryjáƺá dniá cżwártego / mieśiącá dźiewiątego / który jeſt Kiſlef ;
Gdy poſłał lud do domu Bożego Sáráſſárá y Regiemmelechá / y mężów jego / áby śię modlili przed oblicżem Páńſkim ;
Y áby mówili do kápłanów / którzy byli w domu Páná zaſtępów / tákże y do proroków / mówiąc : Izali jeƺcże płákáć będę mieśiącá piątego / wyłącżywƺy śię ták / jákom już cżynił przez kilká lat?
Y ſtáło śię ſłowo Páná zaſtępów do mnie / mówiąc :
Rzecż do wƺyſtkiego ludu tey źiemi / y do kápłanów / mówiąc : Gdyśćie pośćili y płákáli piątego y śiódmego mieśiącá przez te śiedmdźieśiąt lat / izáżeśćie mnie / mnie / mówię / poſt pośćili?
A gdy jećie álbo pijećie / izali nie ſobie jećie y nie ſobie pijećie?
Izáliśćie nie ták cżynić mieli według ſłowá / które przepowiedźiał Pán przez proroków przeƺłych / gdy jeƺcże Jeruzálem bezpiecżeńſtwá y pokoju używáło / y miáſtá jego około niego / y lud w ſtronie południowej y po polách mieƺkał ( w pokoju? )
Y ſtáło śię ſłowo Páńſkie do Zácháryjáƺá / mówiąc :
Ták powiedźiał Pán zaſtępów / mówiąc : Spráwiedliwie ſądźćie / á miłośierdźie y litość pokázujćie káżdy nád bliźnim ſwoim ;
A wdowy y śieroty / y przychodniá / y ubogiego nie ućiſkájćie / y złego jeden przećiwko drugiemu nie myślćie w ſercu ſwojim.
Ale nie chćieli dbáć ; y obróćili śię tyłem / á uƺy ſwe zátulili / áby nie ſłucháli.
Sercá też ſwe zátwárdźili jáko dyjáment / áby nie ſłucháli zakonu tego y ſłów / które poſyłał Pán zaſtępów duchem ſwoim przez proroków przeƺłych / ſkąd przyƺedł wielki gniew od Páná zaſtępów.
Bo jáko oni / gdy ich wołáno / nie ſłucháli / ták też / gdy oni wołáli / nie wyſłucháłem / mówi Pán zaſtępów.
Y rozproƺyłem ich jáko wicher miedzy wƺyſtkie narody / które nie ználi / y tá źiemiá ſpuſtoƺáłá po nich / ták / że nie był przechodzący y wrácájący śię / á ták źiemię pożądáną w ſpuſtoƺenie obróćili.
