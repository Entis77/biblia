Mieśiącá óſmego roku wtórego Dáryjuƺá ſtáło śię ſłowo Páńſkie do mnie Zácháryjáƺá / ſyná Báráchyjáƺowego / ſyná Iddowego / proroká / mówiąc :
Rozgniewáł śię Pán ná ojców wáƺych bárdzo.
Przetoż rzecż do nich : Ták mówi Pán záſtępów : Náwróććie śię do mnie / mówi Pán záſtępów / á náwrócę śię do wáſ / mówi Pán záſtępów.
Nie bądźćież jáko ojcowie wáśi / ná których wołáli prorocy pierwśi / mówiąc : Ták mówi Pán záſtępów : Náwróććież śię teráz od złych dróg wáƺych / y od złych ſpráw wáƺych ; ále nie uſłucháli / áni dbáli ná mię / mówi Pán.
Gdźież ſą ojcowie wáśi / y prorocy? Izáli ná wieki żyć będą?
Wƺákże izáli ſłowá moje y uſtáwy moje / którem rozkázáł prorokom / ſługom moim / nie záśięgły ojców wáƺych? ták / że náwróćiwƺy śię rzekli : Jáko nám Pán záſtępów ucżynić umyślił według dróg náƺych y według ucżynków náƺych / ták nám ucżynił.
Dniá dwudźieſtego y cżwártego jedenáſtego mieśiącá ( ten jeſt mieśiąc Sebát ) roku wtórego Dáryjuƺowego ſtáło śię ſłowo Páńſkie do Zácháryjáƺá / ſyná Báráchyjáƺowego / ſyná Iddowego / proroká / mówiąc :
Widźiáłem w nocy / á oto mąż jecháł ná koniu rydzym / który ſtáł między mirtámi / które były w niźinie / á zá nim konie rydze / cżárne y biáłe.
Tedym rzekł : Co zácż ſą ći / pánie mój? Y rzekł do mnie Anioł rozmáwiájący zemną : Já tobie okáżę / co zácż ſą.
Y odpowiedźiáł mąż / który ſtáł między mirtámi / y rzekł : Ci ſą / których poſłáł Pán / áby przeƺli źiemię.
Y odpowiedźieli Aniołowi Páńſkimu ſtojącemu między mirtámi / y rzekli : Przeƺliśmy źiemię / á oto wƺyſtká źiemiá bezpiecżeńſtwá y pokoju używá.
Tedy odpowiedźiáł Anioł Páńſki / y rzekł : O Pánie záſtępów! y dokądże śię nie zmiłujeƺ nád Jeruzálemem / y nád miáſtámi Judzkiemi / ná któreś śię gniewáł już śiedmdźieśiąt lát?
Y odpowiedźiáł Pán Aniołowi onemu / który mówił zemną / ſłowy dobremi / ſłowy poćieƺnemi.
Y rzekł do mnie Anioł / który mówił zemną : Wołáj á rzecż : Ták mówi Pán záſtępów : Zápáliłem śię zá Jeruzálemem y zá Syonem gorliwośćią wielką.
A gniewám śię bárdzo ná te národy / które używáją pokoju ; bo gdym śię Já trochę zágniewáł / tedy one pomágáły do złego.
Przetoż ták mówi Pán : Náwróćiłem śię do Jeruzálemu w miłośierdźiu / dom mój zbudowány będźie w nim / mówi Pán záſtępów / y ƺnur rozćiągniony będźie ná Jeruzálem.
Jeƺcże wołáj / mówiąc : Ták mówi Pán záſtępów : Jeƺcże śię oſádzą miáſtá moje dlá obfitośći dobrego ; bo jeƺcże Pán Syon poćieƺy / y obierze jeƺcże Jeruzálem.
Tedym podnióſł ocży ſwe y ujrzáłem / á oto cżtery rogi.
Y rzekłem do Aniołá / który mówił zemną : Cóż to jeſt? Y rzekł do mnie : Teć ſą rogi / które rozrzućiły Judę y Izráelá y Jeruzálem.
Ukázáł mi też Pán cżterech kowáli.
Y rzekłem : Cóż ći idą cżynić? Y odpowiedźiáł / mówiąc : Teć ſą rogi / które rozrzućiły Judę / ták / iż żáden nie mógł podnieść głowy ſwojej ; przetoż ći przyƺli / áby je przeſtráƺyli / y ſtrąćili rogi tych národów / które podnioſły róg przećiwko źiemi Judzkiej / áby ją rozrzućiły.
