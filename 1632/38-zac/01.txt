Mieśiącá óſmego roku wtórego Dáryjuƺá ſtáło śię ſłowo Páńſkie do mnie Zácháryjáƺá / ſyná Báráchyjáƺowego / ſyná Iddowego / proroká / mówiąc :
Rozgniewał śię Pán ná ojców wáƺych bárzo.
Przetoż rzecż do nich : Ták mówi Pán zaſtępów : Náwróććie śię do mnie / mówi Pán zaſtępów / á náwrócę śię do was / mówi Pán zaſtępów.
Nie bądźćież jáko ojcowie wáśi / ná których wołáli prorocy pierwśi / mówiąc : Ták mówi Pán zaſtępów : Náwróććież śię teraz od złych dróg wáƺych / y od złych ſpráw wáƺych ; ále nie uſłucháli / áni dbáli ná mię / mówi Pán.
Gdźież ſą ojcowie wáśi / y prorocy? Izali ná wieki żyć będą?
Wƺákże izali ſłowá moje y uſtáwy moje / którem rozkázał prorokom / ſługom moim / nie záśięgły ojców wáƺych? ták / że náwróćiwƺy śię rzekli : Jáko nam Pán zaſtępów ucżynić umyślił według dróg náƺych y według ucżynków náƺych / ták nam ucżynił.
Dniá dwudźieſtego y cżwártego jedenáſtego mieśiącá ( ten jeſt mieśiąc Sebát ) roku wtórego Dáryjuƺowego ſtáło śię ſłowo Páńſkie do Zácháryjáƺá / ſyná Báráchyjáƺowego / ſyná Iddowego / proroká / mówiąc :
Widźiałem w nocy / á oto mąż jechał ná koniu rydzym / który ſtał miedzy mirtámi / które były w niźinie / á zá nim konie rydze / cżárne y biáłe.
Tedym rzekł : Co zácż ſą ći / pánie mój? Y rzekł do mnie Anjoł rozmáwiájący zemną : Ja tobie okáżę / co zácż ſą.
Y odpowiedźiał mąż / który ſtał miedzy mirtámi / y rzekł : Ci ſą / których poſłał Pán / áby przeƺli źiemię.
Y odpowiedźieli Aniołowi Páńſkimu ſtojącemu miedzy mirtámi / y rzekli : Przeƺliśmy źiemię / á oto wƺyſtká źiemiá bezpiecżeńſtwá y pokoju używá.
Tedy odpowiedźiał Anjoł Páńſki / y rzekł : O Pánie zaſtępów! y dokądże śię nie zmiłujeƺ nád Jeruzálemem / y nád miáſtámi Judzkiemi / ná któreś śię gniewał już śiedmdźieśiąt lat?
Y odpowiedźiał Pán Aniołowi onemu / który mówił zemną / ſłowy dobremi / ſłowy poćieƺnemi.
Y rzekł do mnie Anjoł / który mówił zemną : Wołáj á rzecż : Ták mówi Pán zaſtępów : Zápáliłem śię zá Jeruzálemem y zá Syonem gorliwośćią wielką.
A gniewám śię bárzo ná te narody / które używáją pokoju ; bo gdym śię Ja trochę zágniewał / tedy one pomágáły do złego.
Przetoż ták mówi Pán : Náwróćiłem śię do Jeruzálemu w miłośierdźiu / dom mój zbudowány będźie w nim / mówi Pán zaſtępów / y ƺnur rozćiągniony będźie ná Jeruzálem.
Jeƺcże wołáj / mówiąc : Ták mówi Pán zaſtępów : Jeƺcże śię oſádzą miáſtá moje dla obfitośći dobrego ; bo jeƺcże Pán Syon poćieƺy / y obierze jeƺcże Jeruzálem.
Tedym podnióſł ocży ſwe y ujrzáłem / á oto cżtery rogi.
Y rzekłem do Aniołá / który mówił zemną : Cóż to jeſt? Y rzekł do mnie : Teć ſą rogi / które rozrzućiły Judę y Izráelá y Jeruzálem.
Ukázał mi też Pán cżterech kowáli.
Y rzekłem : Cóż ći idą cżynić? Y odpowiedźiał / mówiąc : Teć ſą rogi / które rozrzućiły Judę / ták / iż żáden nie mógł podnieść głowy ſwojey ; przetoż ći przyƺli / áby je przeſtráƺyli / y ſtrąćili rogi tych narodów / które podnioſły róg przećiwko źiemi Judzkiej / áby ją rozrzućiły.
