Otwórz / Libánie! wrotá ſwe / niech pożre ogień cedry twoje.
Rozkwil śię jodło! bo upádł cedr / bo wielmożni ſpuſtoƺeni ſą ; kwilćie dęby Báſáńſkie / bo wyćięty jeſt lás ogrodzony.
Głos nárzekánia páſterzy ſłyƺány jeſt / iż zburzoná jeſt wielmożność ich ; głos ryku lwiąt / iż zburzoná jeſt pychá Jordánu.
Ták mówi Pán / Bóg mój : Páś owce ná rzeź zgotowáne ;
Które dźierżáwcy ich zábijáją / á nie bywáją obwinieni / y owƺem / ſprzedáwájący je mówią : Błogoſłáwiony Pán / żeśmy śię zbogáćili / á którzy je páſą / nie máją litośći nád niemi.
Przetoż nie ſfolguję więcey obywátelom tey źiemi / mówi Pán ; bo oto Ja podám tych ludźi káżdego w rękę bliźniego jego / y w rękę królá ich / y potrą źiemię / á nie wyrwie jey z rąk ich.
Bom páſł owce zgotowáne ná rzeź / was / mówię / o nędzne owce! y wźiąwƺy ſobie dwie láſki / jednęm názwał Ućiechą / á drugąm názwał Związujących / á páſłem one owce.
Y zgłádźiłem trzech páſterzy w jednym mieśiącu ; ále utęſkniłá ſobie duƺá mojá z nimi / przeto / że duƺá ich brzydźiłá śię mną.
Rzekłem tedy : Nie będęć was páſł ; co zdychá / niech zdechnie / á co ma być wygłádzone / niech będźie wygłádzone / á które pozoſtáną / niech pożerá mięſo jedná drugiej.
Przetoż wźiąwƺy láſkę moję Ućiechy / porąbáłem ją / wzruƺywƺy przymierze moje / którem poſtánowił z tym wƺyſtkim ludem.
A dniá onego / gdy wzruƺone było / pewnie poználi nędzni z trzody / którzy śię ná mię oglądáli / że to ſłowo Páńſkie.
Bom rzekł do nich : Jeſli to jeſt dobre w ocżách wáƺych / dájćie zápłátę moję / á jeſli nie / zániechájćież ; tedy odwáżyli zápłátę moję trzydźieśći ſrebrników.
Zátem rzekł Pán do mnie : Porzuć je przed gárncárzá ; zácnáż to zápłátá / którąm jeſt od nich ták drogo oƺácowány! Wźiąłem tedy trzydźieśći ſrebrników / á porzućiłem je w domu Páńſkim przed gárncárzá.
Potem porąbáłem láſkę moję drugą Związujących / wzruƺywƺy bráterſtwo miedzy Judą y miedzy Izráelem.
Y rzekł Pán do mnie : Weźmij ſobie jeƺcże oręż páſterzá głupiego.
Bo oto Ja wzbudzę páſterzá w tey źiemi / który nie będźie obłąkánych náwiedzał / áni będźie jágniątek ƺukał / złámánego też lecżyć / y tego / co uſtánie / nośić nie będźie ; ále mięſo tłuſtych jeść będźie / á kopytá ich poſtrącá.
Biádá páſterzowi niepożytecżnemu / który opuƺcżá trzodę! miecż nád rámieniem jego y nád práwem okiem jego ; rámię jego cále uſchnie / á práwe oko jego cále záćmione będźie.
