Otwórz / Libánie! wrotá ſwe / niech pożre ogień cedry twoje.
Rozkwil śię jodło! bo upádł cedr / bo wielmożni ſpuſtoƺeni ſą ; kwilćie dęby Báſáńſkie / bo wyćięty jeſt láſ ogrodzony.
Głoſ nárzekániá páſterzy ſłyƺány jeſt / iż zburzoná jeſt wielmożność ich ; głoſ ryku lwiąt / iż zburzoná jeſt pychá Jordánu.
Ták mówi Pán / Bóg mój : Páś owce ná rzeź zgotowáne ;
Które dzierżáwcy ich zábijáją / á nie bywáją obwinieni / y owƺem / ſprzedáwájący je mówią : Błogoſłáwiony Pán / żeśmy śię zbogáćili / á którzy je páſą / nie máją litośći nád niemi.
Przetoż nie ſfolguję więcej obywátelom tej ziemi / mówi Pán ; bo oto Já podám tych ludzi káżdego w rękę bliźniego jego / y w rękę królá ich / y potrą ziemię / á nie wyrwie jej z rąk ich.
Bom páſł owce zgotowáne ná rzeź / wáſ / mówię / o nędzne owce! y wziąwƺy ſobie dwie láſki / jednęm názwáł Ućiechą / á drugąm názwáł Związujących / á páſłem one owce.
Y zgłádziłem trzech páſterzy w jednym mieśiącu ; ále utęſkniłá ſobie duƺá mojá z nimi / przeto / że duƺá ich brzydziłá śię mną.
Rzekłem tedy : Nie będęć wáſ páſł ; co zdychá / niech zdechnie / á co má być wygłádzone / niech będzie wygłádzone / á które pozoſtáną / niech pożerá mięſo jedná drugiej.
Przetoż wziąwƺy láſkę moję Ućiechy / porąbáłem ją / wzruƺywƺy przymierze moje / którem poſtánowił z tym wƺyſtkim ludem.
A dniá onego / gdy wzruƺone było / pewnie poználi nędzni z trzody / którzy śię ná mię oglądáli / że to ſłowo Páńſkie.
Bom rzekł do nich : Jeźli to jeſt dobre w ocżách wáƺych / dájćie zápłátę moję / á jeźli nie / zániechájćież ; tedy odwáżyli zápłátę moję trzydzieśći ſrebrników.
Zátem rzekł Pán do mnie : Porzuć je przed gárncárzá ; zácnáż to zápłátá / którąm jeſt od nich ták drogo oƺácowány! Wziąłem tedy trzydzieśći ſrebrników / á porzućiłem je w domu Páńſkim przed gárncárzá.
Potem porąbáłem láſkę moję drugą Związujących / wzruƺywƺy bráterſtwo między Judą y między Izráelem.
Y rzekł Pán do mnie : Weźmij ſobie jeƺcże oręż páſterzá głupiego.
Bo oto Já wzbudzę páſterzá w tej ziemi / który nie będzie obłąkánych náwiedzáł / áni będzie jágniątek ƺukáł / złámánego też lecżyć / y tego / co uſtánie / nośić nie będzie ; ále mięſo tłuſtych jeść będzie / á kopytá ich poſtrącá.
Biádá páſterzowi niepożytecżnemu / który opuƺcżá trzodę! miecż nád rámieniem jego y nád práwem okiem jego ; rámię jego cále uſchnie / á práwe oko jego cále záćmione będzie.
