Potem obróćiwƺy śię podnioſłem ocżu ſwych y ujrzáłem / á oto cżtery wozy wychodziły z pośrodku dwóch gór / á góry one były góry miedziáne.
W pierwƺym wozie były konie rydze / á w drugim wozie konie wrone ( káre ) ;
W trzećim wozie konie biáłe / á w cżwártym wozie były konie ſtrokáte / á wƺyſtkie mocne.
Tedy odpowiádájąc rzekłem do Aniołá / który mówił zemną : Co to jeſt / Pánie mój?
Y odpowiedziáł Anioł á rzekł do mnie : Te ſą cżtery wiátry niebieſkie / wychodzące z miejſcá / gdzie ſtáły / przed pánującym nád wƺyſtką ziemią.
Konie wrone záprzężone wychodzą do ziemi północnej / á biáłe wychodzą zá nimi / ſtrokáte záś wychodzą do ziemi południowej.
Te tedy mocne konie wyƺedłƺy chćiáły iść / áby obeƺły ziemię ; tedy rzekł : Idźćie / á obejdźćie ziemię! Y obeƺły ziemię.
A záwołáwƺy mię rzekł do mnie / mówiąc : Oto te / które wyƺły do ziemi północnej / uſpokoiły duchá mego w ziemi północnej.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Weźmij od tych / co byli pojmáni od Cheldájego y od Tobijáƺá / y od Jedájáƺá ; ( á ty przyjdzieƺ tegoż dniá / y wnijdzieƺ do domu Joſyjáƺá / ſyná Sofonijáƺowego ) którzy idą z Bábilonu ;
Weźmij / mówię / ſrebro y złoto / á ucżyń korony / á włóż je ná głowę Jeſuego / ſyná Jozedekowego / kápłáná nájwyżƺego.
Y rzecż do niego / mówiąc : Ták powiádá Pán záſtępów / mówiąc : Oto mąż / którego imię jeſt Látorośl / który z miejſcá ſwego wyrośnie / ten wyſtáwi kośćiół Pánu.
Bo ten má wyſtáwić kośćiół Pánu / ten záś przynieśie ſłáwę / y śiedzieć y pánowáć będzie ná ſtolicy ſwojej / y będzie kápłánem ná ſtolicy ſwojej / á rádá pokoju będzie między nimi obiemá.
A te korony zoſtáną Chelemowi / y Tobijáƺowi / y Jedájáƺowi / y Chenowi / ſynowi Sofonijáƺowemu / ná pámiątkę w kośćiele Páńſkim.
Bo dálecy przyjdą / á będą budowáć kośćiół Páńſki ; y dowiećie śię / że Pán záſtępów poſłáł mię do wáſ ; á to śię ſtánie / jeźli pilnie ſłucháć będziećie głoſu Páná / Bogá ſwego.
