Potem obróćiwƺy śię podnioſłem ocżu ſwych y ujrzáłem / á oto cżtery wozy wychodźiły z pośrodku dwóch gór / á góry one były góry miedźiáne.
W pierwƺym woźie były konie rydze / á w drugim woźie konie wrone ( káre ) ;
W trzećim woźie konie biáłe / á w cżwártym woźie były konie ſtrokáte / á wƺyſtkie mocne.
Tedy odpowiádájąc rzekłem do Aniołá / który mówił zemną : Co to jeſt / Pánie mój?
Y odpowiedźiáł Anioł á rzekł do mnie : Te ſą cżtery wiátry niebieſkie / wychodzące z miejſcá / gdźie ſtáły / przed pánującym nád wƺyſtką źiemią.
Konie wrone záprzężone wychodzą do źiemi północney / á biáłe wychodzą zá nimi / ſtrokáte záś wychodzą do źiemi południowej.
Te tedy mocne konie wyƺedłƺy chćiáły iść / áby obeƺły źiemię ; tedy rzekł : Idźćie / á obejdźćie źiemię! Y obeƺły źiemię.
A záwołáwƺy mię rzekł do mnie / mówiąc : Oto te / które wyƺły do źiemi północney / uſpokoiły duchá mego w źiemi północney.
Y ſtáło śię ſłowo Páńſkie do mnie / mówiąc :
Weźmij od tych / co byli pojmáni od Cheldájego y od Tobijáƺá / y od Jedájáƺá ; ( á ty przyjdźieƺ tegoż dniá / y wnijdźieƺ do domu Joſyjáƺá / ſyná Sofonijáƺowego ) którzy idą z Bábilonu ;
Weźmij / mówię / ſrebro y złoto / á ucżyń korony / á włóż je ná głowę Jeſuego / ſyná Jozedekowego / kápłáná nájwyżƺego.
Y rzecż do niego / mówiąc : Ták powiádá Pán záſtępów / mówiąc : Oto mąż / którego imię jeſt Látorośl / który z miejſcá ſwego wyrośnie / ten wyſtáwi kośćiół Pánu.
Bo ten má wyſtáwić kośćiół Pánu / ten záś przynieśie ſłáwę / y śiedźieć y pánowáć będźie ná ſtolicy ſwojey / y będźie kápłánem ná ſtolicy ſwojey / á rádá pokoju będźie między nimi obiemá.
A te korony zoſtáną Chelemowi / y Tobijáƺowi / y Jedájáƺowi / y Chenowi / ſynowi Sofonijáƺowemu / ná pámiątkę w kośćiele Páńſkim.
Bo dálecy przyjdą / á będą budowáć kośćiół Páńſki ; y dowiećie śię / że Pán záſtępów poſłáł mię do was ; á to śię ſtánie / jeſli pilnie ſłucháć będźiećie głoſu Páná / Bogá ſwego.
