Brzemię ſłowá Páńſkiego nád Izráelem. Ták mówi Pán / który rozpoſtárł niebioſá / á ugruntował źiemię / który tworzy duchá cżłowiecżego we wnętrznośćiách jego :
Oto Ja poſtáwię Jeruzálem kubkiem opojeniá wƺyſtkim národom okolicżnym / którzy będą przećiwko Judźie ná oblężenie / y przećiwko Jeruzálemowi.
Owƺem / ſtánie śię dniá onego / że ucżynię Jeruzálem kámieniem ćiężkim wƺyſtkim národom ; wƺyſcy / którzy go dźwigáć będą / bárzo śię uráżą / choćby śię zgromádźiły przećiwko niemu wƺyſtkie narody źiemi.
Dniá onego / mówi Pán / záráżę káżdego koniá zdrętwieniem y jeźdźcá jego ƺáleńſtwem ; ále nád domem Judá otworzę ocży moje / á káżdego koniá narodów záráżę ślepotą.
Y rzekną kśiążętá Judzcy w ſercu ſwem : Mamy śiłę y obywátele Jeruzálemſcy w Pánu zaſtępów / Bogu ſwoim.
Dniá onego położę kśiążąt Judzkich jáko węgle ogniſte miedzy drwy / á jáko pochodnię gorejącą miedzy ſnopy ; y pożrą ná práwą y ná lewą ſtronę wƺyſtkie narody okolicżne / y zoſtánie jeƺcże Jeruzálem ná miejſcu ſwem w Jeruzálemie.
Záchowá Pán y námioty Judzkie pierwej / áby śię nie wywyżƺáłá chwałá domu Dawidowego / y chwałá obywáteli Jeruzálemſkich przećiwko Judźie.
Dniá onego Pán będźie bronił obywáteli Jeruzálemſkich / á któryby był miedzy nimi nájſłábƺy / ſtánie śię dniá onego podobny Dawidowi / á dom Dawidowy podobny bogom / podobny Aniołowi Páńſkimu przed nimi.
Bo śię ſtánie dniá onego / że ƺukáć będę wƺyſtkie narody / które przyćiągną przećiwko Jeruzálemowi / ábym je wytráćił.
Y wyleję ná dom Dawidowy / y ná obywáteli Jeruzálemſkich Duchá łáſki y modlitw / á pátrzyć będą ná mię / którego przebodli ; y płákáć będą nád nim płácżem / jáko nád jednorodzonym ; gorzko / mówię / płákáć będą nád nim / jáko gorzko płácżą nád pierworodnym.
Dniá onego będźie wielkie kwilenie w Jeruzálemie / jáko kwilenie w Adádrymon ná polu Mágieddon ;
Bo źiemiá kwilić będźie / káżde pokolenie oſobno / pokolenie domu Dawidowego oſobno / y niewiáſty ich oſobno ; pokolenie domu Nátánowego oſobno / y niewiáſty ich oſobno ;
Pokolenie domu Lewiego oſobno / y niewiáſty ich oſobno ; pokolenie Semejego oſobno / y niewiáſty ich oſobno ;
Wƺyſtkie inƺe pokolenia / káżde pokolenie oſobno / y niewiáſty ich oſobno.
