Oto ja poſyłám Aniołá mego / który zgotuje drogę przed oblicżem mojem ; á zárázem przyjdźie do kośćiołá ſwego Pánujący / którego wy ƺukáćie / y Anioł przymierzá / którego wy żądáćie ; oto / przyjdźie / mówi Pán záſtępów.
Lecż któż będźie mógł znieść dźień przyjśćiá jego? Y kto śię oſtoi / gdy on śię okáże? Bo on jeſt jáko ogień roztápiájący / y jáko mydło blechárzów.
Y będźie śiedźiáł roztápiájąc y wycżyƺcżájąc ſrebro / y ocżyśći ſyny Lewiego / y przepłáwi je jáko złoto y jáko ſrebro / y będą ofiárowáć Pánu dár w ſpráwiedliwośći.
Tedy Pánu będźie wdźięcżná ofiárá od Judy y od Jeruzálemu / jáko zá dni pierwƺych y jáko zá lát ſtárodáwnych.
Ale śię do was przybliżę z ſądem / y będę świádkiem prędkim przećiwko cżárownikom / y przećiwko cudzołożnikom / y przećiwko krzywoprzyśiężcom / y przećiwko tym / którzy krzywdę cżyniąc zátrzymywáją zápłátę nájemnikowi / wdowie y śieroćie y cudzoźiemcowi krzywdę cżynią / á nie boją śię mnie / mówi Pán záſtępów.
Gdyż ja Pán nie odmieniám śię / przetoż wy / ſynowie Jákóbowi! nie jeſteśćie zniƺcżeni.
Záráz ode dni ojców wáƺych odſtąpiliśćie od uſtáw moich / á nie ſtrzegliśćie ich ; náwróććież śię do mnie / á náwrócę śię do was / mówi Pán záſtępów. Ale mówićie : W cżemże śię náwróćimy?
Izáli cżłowiek má Bogá złupić / że wy mnie łupićie? Wƺákże mówićie : W cżemże ćię łupimy? W dźieśięćinách y w ofiárách.
Zgołáśćie przeklęći / iż mię ták łupićie / wy y wƺyſtek náród wáƺ.
Znieśćie wƺyſtkę dźieśięćinę do ƺpichleżá / áby byłá żywność w domu moim / á doświádcżćie mię teraz w tem / mówi Pán záſtępów ; jeſli wám nie otworzę okien niebieſkich / á nie wyleję ná was błogoſłáwieńſtwá / ták że go nie będźiećie mieli gdźie podźiáć ;
Y zgromię dla was pożerájącego / á nie popſuję wám urodzáju źiemſkiego / y nie pochybi winná máćicá ná polu / mówi Pán záſtępów.
Y będą was błogoſłáwić wƺyſcy národowie ; bo wy będźiećie źiemią rozkoƺną / mówi Pán záſtępów.
Zmocniły śię przećiwko mnie ſłowá wáƺe / mówi Pán ; wƺákże mówićie : Cóżeśmy mówili przećiwko tobie?
Mówiliśćie : Próżná to Bogu ſłużyć / á cóż zá pożytek / choć będźiemy ſtrzedz rozkázániá jego / y będźiemy ſmętnie chodźić / bojąc śię Páná záſtępów?
Owƺem / pyƺne mámy zá błogoſłáwione / poniewáż śię ći budują / którzy cżynią niezbożność / á którzy kuƺą Bogá / záchowáni bywáją.
Tedy rozmáwiáli o tem ći / którzy śię boją Páná / káżdy z bliźnim ſwoim. Y obácżył Pán / á uſłyƺáł / y nápiſáno kśięgę pámiątki przed oblicżem jego dla bojących śię Páná y myślących o imieniu jego.
Cić mi będą / mówi Pán záſtępów / w dźień / który Ja ucżynię / włáſnośćią ; y zmiłuję śię nád nimi / jáko śię zmiłowywá ojćiec nád ſynem ſwoim / który mu ſłuży.
Tedy śię náwróćićie / á obácżyćie różność między ſpráwiedliwym y niezbożnym / między tym / który ſłuży Bogu / y między tym / który mu nie ſłuży.
