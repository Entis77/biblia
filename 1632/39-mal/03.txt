Oto ja poſyłám Aniołá mego / który zgotuje drogę przed oblicżem mojim ; á zárázem przydźie do kośćiołá ſwego Pánujący / którego wy ƺukáćie / y Anjoł przymierza / którego wy żądáćie ; oto / przydźie / mówi Pán zaſtępów.
Lecż któż będźie mógł znieść dźień przyjśćiá jego? Y kto śię oſtoi / gdy on śię okaże? Bo on jeſt jáko ogień roztápiájący / y jáko mydło blechárzów.
Y będźie śiedźiał roztápiájąc y wycżyƺcżájąc ſrebro / y ocżyśći ſyny Lewiego / y przepłáwi je jáko złoto y jáko ſrebro / y będą ofiárowáć Pánu dar w ſpráwiedliwośći.
Tedy Pánu będźie wdźięcżná ofiárá od Judy y od Jeruzálemu / jáko zá dni pierwƺych y jáko zá lat ſtárodáwnych.
Ale śię do was przybliżę z ſądem / y będę świádkiem prędkim przećiwko cżárownikom / y przećiwko cudzołożnikom / y przećiwko krzywoprzyśiężcom / y przećiwko tym / którzy krzywdę cżyniąc zátrzymywáją zápłátę nájemnikowi / wdowie y śieroćie y cudzoźiemcowi krzywdę cżynią / á nie boją śię mnie / mówi Pán zaſtępów.
Gdyż ja Pán nie odmieniám śię / przetoż wy / ſynowie Jákóbowi! nie jeſteśćie zniƺcżeni.
Záráz ode dni ojców wáƺych odſtąpiliśćie od uſtáw mojich / á nie ſtrzegliśćie ich ; náwróććież śię do mnie / á náwrócę śię do was / mówi Pán zaſtępów. Ale mówićie : W cżemże śię náwróćimy?
Izali cżłowiek ma Bogá złupić / że wy mnie łupićie? Wƺákże mówićie : W cżemże ćię łupimy? W dźieśięćinách y w ofiárách.
Zgołáśćie przeklęći / iż mię ták łupićie / wy y wƺyſtek naród wáƺ.
Znieśćie wƺyſtkę dźieśięćinę do ƺpichleżá / áby byłá żywność w domu moim / á doświádcżćie mię teraz w tym / mówi Pán zaſtępów ; jeſli wam nie otworzę okien niebieſkich / á nie wyleję ná was błogoſłáwieńſtwá / ták że go nie będźiećie mieli gdźie podźiáć ;
Y zgromię dla was pożerájącego / á nie popſuję wam urodzáju źiemſkiego / y nie pochybi winná máćicá ná polu / mówi Pán zaſtępów.
Y będą was błogoſłáwić wƺyſcy národowie ; bo wy będźiećie źiemią rozkoƺną / mówi Pán zaſtępów.
Zmocniły śię przećiwko mnie ſłowá wáƺe / mówi Pán ; wƺákże mówićie : Cóżeśmy mówili przećiwko tobie?
Mówiliśćie : Próżná to Bogu ſłużyć / á cóż zá pożytek / choć będźiemy ſtrzedz rozkázánia jego / y będźiemy ſmętnie chodźić / bojąc śię Páná zaſtępów?
Owƺem / pyƺne mamy zá błogoſłáwione / ponieważ śię ći budują / którzy cżynią niezbożność / á którzy kuƺą Bogá / záchowáni bywáją.
Tedy rozmáwiáli o tym ći / którzy śię boją Páná / káżdy z bliźnim ſwoim. Y obácżył Pán / á uſłyƺał / y nápiſano kśięgę pámiątki przed oblicżem jego dla bojących śię Páná y myślących o imieniu jego.
Cić mi będą / mówi Pán zaſtępów / w dźień / który Ja ucżynię / włáſnośćią ; y zmiłuję śię nád nimi / jáko śię zmiłowywá oćiec nád ſynem ſwoim / który mu ſłuży.
Tedy śię náwróćićie / á obácżyćie różność miedzy ſpráwiedliwym y niezbożnym / miedzy tym / który ſłuży Bogu / y miedzy tym / który mu nie ſłuży.
