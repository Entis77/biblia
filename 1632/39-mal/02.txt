A ták teráz do wáſ mówię to rozkázánie / o kápłáni!
Jeźli nie uſłucháćie / y jeźli nie złożyćie do ſercá / ábyśćie dáli chwáłę imieniowi memu / mówi Pán záſtępów / tedy poślę ná wáſ przeklęſtwo / á przeklináć będę błogoſłáwieńſtwá wáƺe ; owƺem jużem je przeklął : bośćie tego nie złożyli do ſercá.
Oto / Já wám popſuję śiewy wáƺe / á wrzucę łájno ná twárzy wáƺe / łájno / mówię / ofiár wáƺych / ták / że wáſ przytárgnie do śiebie.
Y dowiećie śię / iżem poſłáł do wáſ to rozkázánie / áby było przymierze moje z Lewim / mówi Pán záſtępów.
Przymierze moje było z nim żywotá y pokoju / á dáłem mu je dlá bojáźni ; bo śię mnie báł / y dlá imieniá mojego ſkruƺony był.
Zákon práwdy był w uſtách jego / á niepráwość nie byłá ználeźioná w wárgách jego ; chodźił zemną w pokoju y w práwośći / y wielu odwróćił od niepráwośći.
Bo wárgi kápłáńſkie umiejętnośći ſtrzedz máją / á pytáć śię będą ná zákon z uſt jego ; Aniołem záiſte jeſt Páná záſtępów.
Aleśćie wy zſtąpili z drogi / y byliśćie zgorƺeniem wielom w zákonie / zepſowáliśćie przymierze Lewiego / mówi Pán záſtępów.
Przetoż y Jám wáſ ucżynił wzgárdzone y poniżone u wƺyſtkich ludźi / ták jákośćie wy nie ſtrzegli dróg moich / á mieliśćie wzgląd ná oſobę w zákonie.
Izáli nie jeden ojćiec jeſt náſ wƺyſtkich? Izáli nie jáden Bóg ſtworzył náſ? Cżemuż tedy brát zdrádzá brátá ſwego / gwáłcąc przymierze ojców náƺych?
Zdrádliwie śię ſpráwuje Judá / á obrzydliwość śię dźieje w Izráelu y w Jeruzálemie ; bo Judá ſplugáwił świętobliwość Páńſką / w której śię kocháć miáł / pojąwƺy zá żone córkę bogá obcego.
Pán mężá / który to cżyni / wykorzeni z námiotów Jákóbowych / ták cżującego / jáko y odpowiádájącego y ofiárującego dár Pánu záſtępów.
A tośćie drugi ráz ucżynili / okrywájąc łzámi ołtárz Páńſki / płácżem y wołániem / ták / że więcej nie pátrzy ná dár / y nie przyjmuje wdźięcżnej ofiáry z ręki wáƺej.
Wƺákże mówićie : Dlácżegoż to? Dlá tego / że Pán jeſt świádkiem między tobą y między żoną młodośći twojej / przećiw którejś ty wyſtąpił / gdyż oná jeſt towárzyƺką twoją / y żoną przymierzá twego.
Bo ázáż nie ucżynił jednym / choć mu jeƺcże duchá zbywáło? A cżemuż jednym? áby ƺukáli náśieniá Bożego ; á ták ſtrzeżćie duchá ſwego / á z żoną młodośći ſwojej śię zdrádliwie nie obchodźćie ;
Bo on má w nienáwiśći opuƺcżenie żony / mówi Pán / Bóg Izráelſki / przeto że táki okrywá okrućieńſtwo płáƺcżem jego / mówi Pán záſtępów ; á ták ſtrzeżćie duchá wáƺego / á nie obchodźćie śię zdrádliwie.
Prácęśćie zádáli Pánu ſłowy ſwojimi / á przećię mówićie : W cżemżeśmy mu prácę zádáli? W tem / gdy mówićie : Wƺelki / który złość cżyni / podobá śię Pánu / y w tych má kochánie ; álbo : Gdźież jeſt Bóg ſądu?
