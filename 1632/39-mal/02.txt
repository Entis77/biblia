A ták teraz do was mówię to rozkázánie / o kápłáni!
Jeſli nie uſłucháćie / y jeſli nie złożyćie do ſercá / ábyśćie dáli chwáłę imieniowi memu / mówi Pán záſtępów / tedy poślę ná was przeklęſtwo / á przeklináć będę błogoſłáwieńſtwá wáƺe ; owƺem jużem je przeklął : bośćie tego nie złożyli do ſercá.
Oto / Ja wám popſuję śiewy wáƺe / á wrzucę łájno ná twárzy wáƺe / łájno / mówię / ofiár wáƺych / ták / że was przytárgnie do śiebie.
Y dowiećie śię / iżem poſłáł do was to rozkázánie / áby było przymierze moje z Lewim / mówi Pán záſtępów.
Przymierze moje było z nim żywotá y pokoju / á dáłem mu je dla bojáźni ; bo śię mnie báł / y dla imieniá mojego ſkruƺony był.
Zakon práwdy był w uſtách jego / á niepráwość nie byłá ználeźioná w wárgách jego ; chodźił zemną w pokoju y w práwośći / y wielu odwróćił od niepráwośći.
Bo wárgi kápłáńſkie umiejętnośći ſtrzedz máją / á pytáć śię będą ná zakon z uſt jego ; Aniołem záiſte jeſt Páná záſtępów.
Aleśćie wy zſtąpili z drogi / y byliśćie zgorƺeniem wielom w zakonie / zepſowáliśćie przymierze Lewiego / mówi Pán záſtępów.
Przetoż y Jám was ucżynił wzgárdzone y poniżone u wƺyſtkich ludźi / ták jákośćie wy nie ſtrzegli dróg moich / á mieliśćie wzgląd ná oſobę w zakonie.
Izáli nie jeden ojćiec jeſt nas wƺyſtkich? Izáli nie jáden Bóg ſtworzył nas? Cżemuż tedy brát zdrádzá brátá ſwego / gwáłcąc przymierze ojców náƺych?
Zdrádliwie śię ſpráwuje Judá / á obrzydliwość śię dźieje w Izráelu y w Jeruzálemie ; bo Judá ſplugáwił świętobliwość Páńſką / w którey śię kocháć miał / pojąwƺy zá żone córkę bogá obcego.
Pán mężá / który to cżyni / wykorzeni z námiotów Jákóbowych / ták cżującego / jáko y odpowiádájącego y ofiárującego dár Pánu záſtępów.
A tośćie drugi ráz ucżynili / okrywájąc łzámi ołtárz Páńſki / płácżem y wołániem / ták / że więcey nie pátrzy ná dár / y nie przyjmuje wdźięcżney ofiáry z ręki wáƺej.
Wƺákże mówićie : Dlácżegoż to? Dla tego / że Pán jeſt świádkiem między tobą y między żoną młodośći twojey / przećiw którejś ty wyſtąpił / gdyż oná jeſt towárzyƺką twoją / y żoną przymierzá twego.
Bo ázáż nie ucżynił jednym / choć mu jeƺcże duchá zbywáło? A cżemuż jednym? áby ƺukáli náśieniá Bożego ; á ták ſtrzeżćie duchá ſwego / á z żoną młodośći ſwojey śię zdrádliwie nie obchodźćie ;
Bo on má w nienáwiśći opuƺcżenie żony / mówi Pán / Bóg Izráelſki / przeto że táki okrywá okrućieńſtwo płáƺcżem jego / mówi Pán záſtępów ; á ták ſtrzeżćie duchá wáƺego / á nie obchodźćie śię zdrádliwie.
Prácęśćie zádáli Pánu ſłowy ſwojimi / á przećię mówićie : W cżemżeśmy mu prácę zádáli? W tem / gdy mówićie : Wƺelki / który złość cżyni / podobá śię Pánu / y w tych má kochánie ; álbo : Gdźież jeſt Bóg ſądu?
