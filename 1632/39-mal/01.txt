Brzemię ſłowá Páńſkiego przećiwko Izráelowi / przez Máláchyjáƺá.
Umiłowáłem was / mówi Pán / á wy mówićie : W cżemżeś nas umiłował? Izali Ezáw nie był brátem Jákóbowi? mówi Pán ; wƺákżem umiłował Jákóbá /
A Ezáwá miáłem w nienáwiśći / y podáłem góry jego ná ſpuſtoƺenie / á dźiedźictwo jego ſmokom ná puſtyni.
Jeſliby rzekłá źiemiá Edomſká : Stáliśmy śię ubogimi / ále wróćimy śię / y pobudujemy miejſcá popuſtoƺone / tedy ták mówi Pán zaſtępów : Niecháj oni budują / á Ja rozwálę ; y názwię ich gránicą niepobożnośći y ludem / który ſobie zbrzydźił Pán áż ná wieki.
To oglądáją ocży wáƺe / á wy rzecżećie : Niech będźie Pán uwielbiony w gránicách Izráelſkich.
Syn ma w ucżćiwośći ojcá / á ſługá páná ſwego ; jáźlim tedy Ja ojcem / gdźież jeſt cżeść mojá? y jeſliżem Ja Pánem / gdźież jeſt bojáźń mojá? mówi Pán zaſtępów wam / o kápłáni! którzy lekce powáżáćie imię moje / á wƺákże mówićie : W cżemże lekce powáżámy imię twoje?
Którzy przynoƺąc ná ołtárz mój ofiárę ſplugáwioną / mówićie : Cżemżeśmy ćię ſplugáwili? Tym / że mówićie : Stół Páńſki ſplugáwiony jeſt.
Bo gdy przywodźićie / co ślepego jeſt / ná ofiárę / izali to nie złá rzecż? Y gdy przywodźićie chrome y chore / izali to nie złá rzecż? Ofiáruj to jedno kśiążęćiu twemu / obácżyƺ / jeżeli mu śię to podobáć będźie / á przyjmieli twárz twoję / mówi Pán zaſtępów.
Przetoż teraz błágájćie / proƺę / oblicże Boże / áby śię zmiłował nád námi ; ále póki śię to dźieje z ręki wáƺej / izali przyjmie oblicże którego z was? mówi Pán zaſtępów.
Owƺem / kto jeſt miedzy wámi / áby záwárł drzwi / álbo dármo zápálił ná ołtárzu moim? Nie mam chęći do was / mówi Pán zaſtępów / y ofiáry nie przyjmię z ręki wáƺej.
Abowiem od wſchodu ſłońcá áż do záchodu jego wielkie jeſt imię moje miedzy národámi / á ná wƺelkiem miejſcu przynieśione będźie kádzenie imieniowi memu y ofiárá cżyſtá ; wielkie zájiſte imię moje będźie miedzy národámi / mówi Pán zaſtępów.
Lecż wy plugáwićie je / gdy mówićie : Stół Páńſki ſplugáwiony jeſt ; á co nań kłádą / jeſt wzgárdzonym pokármem.
Mówićie też : O jákáż to prácá! choćbyśćie to zdmuchnąć mogli / mówi Pán zaſtępów ; bo przynośićie to / co jeſt wydártego / y chromego y ſchorzáłego / á przynośićie to ná ofiárę : izali to przyjmę z ręki wáƺej? mówi Pán.
Y owƺem przeklęty jeſt zdrádliwy / który májąc w trzodźie ſwej ſámcá / á przećię cżyniąc śluby ofiáruje Pánu to / co jeſt ułomnego ; bom Ja królem wielkim / mówi Pán zaſtępów / á imię moje ſtráƺne jeſt miedzy narody.
