Brzemię ſłowá Páńſkiego przećiwko Izráelowi / przez Máláchyjáƺá.
Umiłowáłem wáſ / mówi Pán / á wy mówićie : W cżemżeś náſ umiłowáł? Izáli Ezáw nie był brátem Jákóbowi? mówi Pán ; wƺákżem umiłowáł Jákóbá /
A Ezáwá miáłem w nienáwiśći / y podáłem góry jego ná ſpuſtoƺenie / á dziedzictwo jego ſmokom ná puſtyni.
Jeźliby rzekłá ziemiá Edomſká : Stáliśmy śię ubogimi / ále wróćimy śię / y pobudujemy miejſcá popuſtoƺone / tedy ták mówi Pán záſtępów : Niecháj oni budują / á Já rozwálę ; y názwię ich gránicą niepobożnośći y ludem / który ſobie zbrzydził Pán áż ná wieki.
To oglądáją ocży wáƺe / á wy rzecżećie : Niech będzie Pán uwielbiony w gránicách Izráelſkich.
Syn má w ucżćiwośći ojcá / á ſługá páná ſwego ; jáźlim tedy Já ojcem / gdzież jeſt cżeść mojá? y jeźliżem Já Pánem / gdzież jeſt bojáźń mojá? mówi Pán záſtępów wám / o kápłáni! którzy lekce powáżáćie imię moje / á wƺákże mówićie : W cżemże lekce powáżámy imię twoje?
Którzy przynoƺąc ná ołtárz mój ofiárę ſplugáwioną / mówićie : Cżemżeśmy ćię ſplugáwili? Tem / że mówićie : Stół Páńſki ſplugáwiony jeſt.
Bo gdy przywodzićie / co ślepego jeſt / ná ofiárę / izáli to nie złá rzecż? Y gdy przywodzićie chrome y chore / izáli to nie złá rzecż? Ofiáruj to jedno kśiążęćiu twemu / obácżyƺ / jeżeli mu śię to podobáć będzie / á przyjmieli twárz twoję / mówi Pán záſtępów.
Przetoż teráz błágájćie / proƺę / oblicże Boże / áby śię zmiłowáł nád námi ; ále póki śię to dzieje z ręki wáƺej / izáli przyjmie oblicże którego z wáſ? mówi Pán záſtępów.
Owƺem / kto jeſt między wámi / áby záwárł drzwi / álbo dármo zápálił ná ołtárzu moim? Nie mám chęći do wáſ / mówi Pán záſtępów / y ofiáry nie przyjmię z ręki wáƺej.
Abowiem od wſchodu ſłońcá áż do záchodu jego wielkie jeſt imię moje między národámi / á ná wƺelkiem miejſcu przynieśione będzie kádzenie imieniowi memu y ofiárá cżyſtá ; wielkie záiſte imię moje będzie między národámi / mówi Pán záſtępów.
Lecż wy plugáwićie je / gdy mówićie : Stół Páńſki ſplugáwiony jeſt ; á co náń kłádą / jeſt wzgárdzonym pokármem.
Mówićie też : O jákáż to prácá! choćbyśćie to zdmuchnąć mogli / mówi Pán záſtępów ; bo przynośićie to / co jeſt wydártego / y chromego y ſchorzáłego / á przynośićie to ná ofiárę : izáli to przyjmę z ręki wáƺej? mówi Pán.
Y owƺem przeklęty jeſt zdrádliwy / który májąc w trzodzie ſwej ſámcá / á przećię cżyniąc śluby ofiáruje Pánu to / co jeſt ułomnego ; bom Já królem wielkim / mówi Pán záſtępów / á imię moje ſtráƺne jeſt między národy.
